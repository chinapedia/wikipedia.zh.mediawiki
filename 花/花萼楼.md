'''花萼楼'''位于中国[[广东省|广东省]][[大埔县|大埔县]][[大东镇|大东镇]]联丰村，建于明[[万历|万历]]三十六年（1608年），是一座林氏家族居住的圆形[[土楼|土楼]]。2002年被列为[[广东省文物保护单位|广东省文物保护单位]]。

== 名称来源 ==
花萼楼楼名出自 “花萼相辉”，源于《[[诗经|诗经]]·常棣》“常棣之华（花），鄂（萼）不韡韡{{注释|1=韡韡（wěiwěi），意为光明华美的样子。}}。凡今之人，莫如兄弟。”花萼为兄弟和睦之意<ref name=":0">{{Cite web|url = http://www.meizhou.cn/news/1105/21/11052100046.html|title = 寂寞花萼楼|accessdate = |author = |date = 2011-05-21|publisher = 梅州网}}</ref>。

== 布局 ==
花萼楼占地面积近2300平方米，楼高11.9米，底层宽2米。顶层宽1.3米，墙体用[[夯土|夯土]]筑成，楼顶为木梁灰瓦，是土木结构的圆形建筑。花萼楼共有三环楼层，内环一层30个间，二环二层60间，外环三层120间，共210个房间。第一层没有窗户，第二、三层墙上设有内小外大呈三角的枪眼，只有一个大门，门框用厚而宽的[[花岗岩|花岗岩]]石板组成，大门钉上坚厚的铁皮，这是为防止外族人[[土客冲突|侵扰]]而特意设计的。楼内有面积为283.4平方米的圆形[[天井_(建筑)|天井]]，铺有[[鹅卵石|鹅卵石]]，中心有一个直径3米的[[方孔钱|古钱币]]图案，寓意着人们祈求丰衣足食的心愿。天井一侧有一口古井，楼内还有[[砻|砻]]、[[碓|碓]]等生活设施。<ref name=":0">{{Cite web|url = http://www.meizhou.cn/news/1105/21/11052100046.html|title = 寂寞花萼楼|accessdate = |author = |date = 2011-05-21|publisher = 梅州网}}</ref><ref>{{Cite web|url =http://www.gd.xinhuanet.com/rensheng/2008-07/17/content_13845424.htm|title =花萼楼|accessdate =|author =|date =2008-07-17|publisher =新华网|deadurl =yes|archiveurl =https://web.archive.org/web/20160224164748/http://www.gd.xinhuanet.com/rensheng/2008-07/17/content_13845424.htm|archivedate =2016-02-24}}</ref>
