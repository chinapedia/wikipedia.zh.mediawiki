《'''花'''》，是[[日本|日本]]樂團[[橘子新樂園|橘子新樂園]]的第8張[[單曲|單曲]]。2004年10月20日發行。是他們迄今銷量最高的單曲和代表作之一。

==簡介==
是同年由[[中村獅童_(第2代)|中村獅童]]和[[竹內結子|竹內結子]]的[[東寶株式會社|東寶系]][[電影|電影]]《[[現在，很想見你|現在，很想見你]]》的[[主題曲|主題曲]]，電影取得很大的成功，票房高達48億日元。很多人觀影之後都直接去唱片店購買主題曲《花》的單曲CD。<ref>[http://ent.163.com/ent/editor/music/041124/041124_345653.html 大冢爱登专辑榜冠军 《花》连续四周夺冠]</ref>

單曲發行後連續四個星期取得[[Oricon公信榜|Oricon公信榜]]單曲週榜冠軍，連續六週的週間銷量都超過10萬。

是2004年日本單曲銷量的第4位<ref>{{Cite web |url=http://www.musictvprogram.com/corner-ranking-2004.html |title={{lang|ja|2004年間シングルランキング TOP100}} |access-date=2010-10-12 |archive-url=https://web.archive.org/web/20120206234708/http://www.musictvprogram.com/corner-ranking-2004.html |archive-date=2012-02-06 |dead-url=yes }}</ref>，2005年的第21位<ref>{{Cite web |url=http://www.musictvprogram.com/corner-ranking-2005.html |title={{lang|ja|2005年間シングルランキング TOP100}} |access-date=2010-10-12 |archive-url=https://web.archive.org/web/20101005060538/http://www.musictvprogram.com/corner-ranking-2005.html |archive-date=2010-10-05 |dead-url=yes }}</ref>。出貨量超過百萬，獲得[[日本唱片協會|日本唱片協會]]的百萬唱片認證。Oricon總銷量高達999,789張，累計銷量比2004年單曲年榜的前三位單曲都高。也是橘子新樂園自身迄今最高銷量的單曲作品。

在Oricon公信榜的統計中距離百萬銷量僅差211張，是目前和Oricon百萬銷量距離最近的單曲。

榮獲2005年度[[MTV日本音樂錄影帶大獎|MTV日本音樂錄影帶大獎]]的年度最佳音樂錄影帶獎、[[JASRAC賞|JASRAC賞]]金賞等獎項。

在2005年底的[[第56回NHK紅白歌合戰|第56回NHK紅白歌合戰]]舉行之前的觀眾期待歌曲調查中，是白組第二最期待演出的歌曲<ref>[http://paper.wenweipo.com/2005/10/08/EN0510080012.htm 票選SMAP晨女呼聲最高]</ref>，但是橘子新樂園以在家鄉[[沖繩縣|沖繩]]進行[[演唱會|演唱會]]優先為由辭退了演出邀請。

是2005年度Oricon[[卡拉OK|卡拉OK]]榜冠軍歌曲。
