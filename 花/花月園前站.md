'''花月園前站'''（{{jpn|j=花月園前駅|hg=かげつえんまええき|rm=Kagetsuenmae eki|f=y}}）是一個位於[[神奈川縣|神奈川縣]][[橫濱市|橫濱市]][[鶴見區_(橫濱市)|鶴見區]]{{link-ja|生麥|生麦}}五丁目，屬於[[京濱急行電鐵|京濱急行電鐵]][[京急本線|本線]]的[[鐵路車站|鐵路車站]]。[[車站編號|車站編號]]是'''KK30'''。

過去在花月園競輪開賽日，傍晚的急行會臨時停靠本站，[[新逗子站|新逗子站]]始發至[[神奈川新町站|神奈川新町站]]的急行延長至[[京急川崎站|京急川崎站]]，增開神奈川新町站始發的上行臨時急行<ref>但受限於本站月台長度，包括臨時停車的急行在內全部列車都為6輛編組</ref>，直到1999年（平成11年）7月31日改點為止。

==年表==
* 1914年（[[大正|大正]]3年）4月12日 - 車站啟用。
* 1946年 - 花月園閉園。站名維持不變。
* 1971年（[[昭和|昭和]]46年）12月 - 改建為跨站式站房。
* 1999年（平成11年）7月31日 - 時刻表修改，[[京急蒲田站|京急蒲田站]] - 新逗子站間的急行廢止，本站不再有急行臨時停車。
* 2020年3月 - 更改站名為'''花月總持寺車站'''（{{lang|ja|花月総持寺駅}}），為便利旅客識別，加註[[副站名|副站名]]「舊站名 花月園前」（{{lang|ja|旧駅名　花月園前}}）<ref>{{cite web|url=https://www.keikyu.co.jp/company/news/2018/20190125HP_18229TS.html|title=沿線地域の活性化を目的に2020年3月 4駅の駅名を変更します|publisher=京浜急行電鉄株式会社|date=2019-01-25|language=ja}}</ref>。


== 車站構造 ==
本站為[[側式月台|側式月台]]2面2線[[地面車站|地面車站]]。[[跨站式站房|跨站式站房]]連結一座東西向的行人天橋。付費區大廳與各月台之間設有電梯。過去{{link-ja|花月園競輪場|花月園競輪場|花月園競輪}}開賽日人潮擁擠，因此在當日會開設特別售票處因應，但隨著2010年3月結束比賽後，現在已不再使用。

== 站名由來 ==
站名來自1914年開園的{{link-ja|花月園遊園地|花月園遊園地}}（1946年閉園，之後改為花月園競輪場）。

== 花月園站前平交道 ==
[[File:Kagetu660802.JPG|thumb]]
站前北側的平交道，其長度僅次於已廢止的{{link-ja|總持寺平交道|総持寺踏切}}，平交道從西側起分別為[[橫須賀線|橫須賀線]]、[[京濱東北線|京濱東北線]]、[[東海道線_(JR東日本)|東海道本線]]、{{link-ja|東海道貨物線|東海道貨物線}}、京急本線的線路，是著名的{{link-ja|打不開的平交道|開かずの踏切}}。京急的平交道之所以與JR分離，是因為在地面車站時代，本站的上行月台站房就在當時國鐵與京急的平交道之間所導致。本站興建跨站式站房時已另外興建行人天橋供民眾使用，但平交道至今仍未廢除。
