'''花叶海棠'''（[[学名|学名]]：{{lang|la|''Malus transitoria''}}）是[[蔷薇科|蔷薇科]][[苹果属|苹果属]]的植物，是中国的特有植物。分布于[[中国大陆|中国大陆]]的[[甘肃|甘肃]]、[[四川|四川]]、[[内蒙古|内蒙古]]、[[青海|青海]]、[[陕西|陕西]]等地，生长于海拔1,500米至3,900米的地区，常生于生山坡丛林中及黄丘陵上，目前已由人工引种栽培。

== 描述 ==

[[File:A_rich_table_for_birds_-_close_view_of_fruit_-_geograph.org.uk_-_607135.jpg|thumb]]
*灌木至小乔木，高可达8米；小枝细长，圆柱形，嫩时密被绒毛，老枝暗紫色或紫褐色；冬芽小，卵形，先端钝，密被绒毛，暗紫色，<br />有数枚外露鳞片。
*叶片卵形至广卵形，长2.5-5厘米，宽2-4.5厘米，先端急尖，基部圆形至宽楔形，边缘有不整齐锯齿，通常3-5不规则深裂，稀不<br />裂，裂片长卵形至长椭圆形，先端急尖，上面被绒毛或近于无毛，下面密被绒毛；叶柄长1.5-3.5厘米，有窄叶翼，密被绒毛；托叶叶<br />质，卵状披针形，先端急尖，全缘，被绒毛。
*花序近伞形，具花3-6朵，花梗长1.5-2厘米，密被绒毛；苞片膜质，线状披针形，具毛，早落；花直径1-2厘米；萼筒钟状，密被<br />绒毛；萼片三角卵形，先端圆钝或微尖，全缘，长约3毫米，内外两面均密被绒毛，比萼筒稍短；花瓣卵形，长8-10毫米，宽5-7毫米，<br />基部有短爪，白色；雄蕊20-25。花丝长短不等，比花瓣稍短；花柱3-5，基部无毛，比雄蕊稍长或近等长。果实近球形，直径6-8毫米，<br />萼片脱落，萼洼下陷；果梗长1.5-2厘米，外被绒毛。[[花期|花期]]5月，果期9月。

== 别名 ==
花叶杜梨（出现陕北土名），马杜梨、小白石枣、涩枣子（甘肃土名），细弱海棠（华北经济植物志）
