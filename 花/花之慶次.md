《'''花之慶次'''》（日文原名：'''花の慶次 -雲のかなたに-'''）是[[日本|日本]]歷史[[漫畫|漫畫]]，改編自[[隆慶一郎|隆慶一郎]]原作之歷史小說「[[一夢庵風流記|一夢庵風流記]]」，[[原哲夫|原哲夫]]漫畫。描寫[[戰國_(日本)|戰國]]之世，[[傾奇者|傾奇者]]・[[前田利益|前田慶次]]的故事。

== 主要登場人物 ==
; [[前田利益|前田慶次]]
: 天下第一之傾奇者，瀧川一益從弟・益氏之次男、前田利家兄・利久之養子。愛馬'''松風'''。
; 捨丸
: 原[[四井主馬|四井主馬]]手下之忍者，後成為慶次之隨侍。
; 岩兵衛
: 漫畫版原創人物，慶次之隨侍。
; 芙杏 
: 漫畫版原創人物。
; [[芳春院|阿松]]  
: 利家[[正室|正室]]。
; [[前田利家|前田利家]] 
: 前田家當主、戰國大名。

== 技 ==
=== 流派 ===
; 穀藏院一刀流（こくぞういん・いっとうりゅう）
: 前田慶次（前田利益）開創之[[劍術|劍術]]流派。
; 手
: 體術。
; 甲陽流忍術
: 體術。
; 天下一夢想流
: 荒井願鬼坊之流派。
; 二刀流
: 龍嶽之技。

=== 奧義 ===
; 靭猿（うつぼざる）
: 加賀忍軍之秘術。
; 火術不知火
: 加賀忍軍之技。
; 影象青攪
: 毛虎親方五人眾之技。
; 水蓮波
: 毛虎親方五人眾之技。
