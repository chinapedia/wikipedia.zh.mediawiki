《'''花樣年華 pt.2'''》是[[韓國|韓國]]男子組合[[防彈少年團|防彈少年團]]第四張韓語迷你專輯，為「青春二部曲」系列專輯的第二部曲。2015年11月30日由[[Big_Hit娛樂|Big Hit娛樂]]製作，[[LOEN娛樂|LOEN娛樂]]發行<ref name=release>{{cite web |url=http://kpopn.com/2015/10/28/311341/ |title=BTS 防彈少年團11/30發行新專輯|work=kpopn |date=2015-10-28 |accessdate=2015-12-09}}</ref>，主打曲目為《RUN》。

==簡介==
* 2015年9月8日，團體在Naver V App與tvcast發布《花樣年華 on stage : prologue》影片，當時預告《花樣年華 pt.2》將會在10月推出<ref>{{cite web |url=http://kpopn.com/2015/09/08/297889/ |title=BTS《花樣年華 on stage》影片 |publisher=kpopn |date=2015-09-08 |accessdate=2015-12-06 }}</ref>，隨後10月28日[[Big_Hit娛樂|Big Hit娛樂]]敲定11月30日發行新專輯，並預先在11月27到29日舉行的《花樣年華 on stage》演唱會表演新曲<ref name=release/>。<br />11月18日，團體在官方[[Youtube|Youtube]]頻道上傳Intro曲〈Never Mind〉的動畫音樂錄影帶<ref>{{cite web |url=http://kpopn.com/2015/11/18/317470/ |title=CNBLUE 9/14 攜《2gether》回歸 |publisher=kpopn |date=2015-11-18 |accessdate=2015-12-09 }}</ref>，緊接在11月23日及24日發放兩批專輯概念照<ref>{{cite web |url=http://kpopn.com/2015/11/23/318989/ |title=BTS《花樣年華 pt.2》概念照公開|work=kpopn |date=2015-11-23 |accessdate=2015-12-06}}</ref><ref>{{cite web |url=http://www.koreastardaily.com/tc/photo/70966 |title=防彈少年團《花樣年華pt.2》二版宣傳照 徜徉夢境與蝶共舞|work=韓星網 |date=2015-11-24 |accessdate=2015-12-09}}</ref>，並於11月26日公開完整專輯曲目名單<ref>{{cite web |url=http://kpopn.com/2015/11/26/319875/ |title=BTS 防彈少年團新輯曲目、試聽 |publisher=kpopn |date=2015-11-26 |accessdate=2015-12-09 }}</ref>。
* 此張專輯於發行6週後攀上[[告示牌_(雜誌)|美國告示牌]]世界專輯榜冠軍寶座，取得K-POP史上在美國告示牌世界專輯榜的最初逆行。接著蟬聯4週冠軍，之後連續15週TOP 10，讓防彈少年團成為同一專輯最長時間位居第一名和最長期間TOP 10紀錄保持人<ref>{{cite web |url=http://m.star.naver.com/bts/news/end?id=7396363 |title=防彈少年團新紀錄，會到何時停止?|work=Naver |date=2016-03-17}}</ref>。
