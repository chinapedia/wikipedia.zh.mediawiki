'''花坮郡'''（{{kor|諺=화대군|漢=花坮郡|羅=Hwadae gun|f=y}}）是[[朝鲜|朝鲜]]一個位於[[咸鏡北道|咸鏡北道]]南部海岸的郡，當地的[[舞水端里|舞水端里]]是[[大浦洞1號|大浦洞1號]]的發射場，用來為朝鲜作[[導彈|導彈]]試驗及發射[[人造衛星|人造衛星]]之用。

== 名稱 ==

由於「花坮郡」裡的「坮」字並非一般常用的中文及日文系統的常用字，大多數傳媒都會把名稱寫作「'''华台郡'''」、「'''花台郡'''」或「'''花臺郡'''」，以便能夠正確顯示或輸入。

== 地理 ==

花坮郡的東側和南側向[[東朝鮮海|東朝鮮海]]([[日本海|日本海]])。北部都是海拔1,000m以上的山地，但海岸地方卻是[[平野|平野]]。

在花坮郡與[[金策市|金策市]]劃分邊界的[[南大川_(咸鏡北道)|南大川]]，是花坮郡的主要河川，從它還分支出[[花坮川|花坮川]]和[[龍山川|龍山川]]等河川。這裡的海岸線跟南北韓在東朝鮮海的其他海岸同樣單調。山林佔了全郡面積的83%。由於有和暖的海風調節，當地冬天的氣溫比較溫和。

== 氣候 ==

年平均溫度為 7.6℃，1月時約為 -7℃，8月時約為 22.2℃。年平均降水量為 700mm。

==历史==
花坮郡是二战后朝鲜民主主义人民共和国新设的郡，原分属[[明川郡|明川郡]]和[[吉州郡|吉州郡]]。1952年12月朝鲜朝鲜行政区划改编，将吉州郡[[东海面|东海面]]、明川郡[[下古面|下古面]]、[[下加面|下加面]]全体，以及[[上古面|上古面]]一部分（10个-{里}-）、上加面一部分（1个-{里}-）合并新设花坮郡。新设[[花坮邑|花坮邑]]，即原下加面花坮-{里}-。新设的花坮郡总共1邑22面。

== 產業 ==
農業和水產業是當地的主要產業。

花坮郡的面積有13％是耕地，主要農產有：[[大米|大米]]、[[玉米|玉米]]、[[大豆|大豆]]、[[蔬菜|蔬菜]]、[[煙草|煙草]]等。家畜有：[[牛|牛]]、[[羊|羊]]、[[豬|豬]]、[[雞|雞]]等，當中以被稱為[[花坮羊|花坮羊]]的優質羊品種尤為特產。在荷坪里的[[養蠶業|養蠶業]]很發達。沿近海有[[鱈科|鱈魚]]、[[鯖魚|{{僻字]]、[[鱒魚|鱒魚]]、[[沙丁魚|沙丁魚]]、[[太平洋玉筋魚|玉筋魚]]等豐富的水產資源，是咸鏡北道的主要水產基地。當中的{{僻字|𩸽|魚花}}魚，在舞水端里附近的近海域是日本海最多漁獲的海域。此外，在當地有[[舞水端淺海養殖事業所|舞水端淺海養殖事業所]]和[[泗浦水產事業所|泗浦水產事業所]]，所生產的[[昆布|昆布]]和[[海苔|海苔]]都很優質。郡內還有醋、家具、農機具、食料工場等產業。

== 交通 ==
由於沒有鐵道，當地只有靠公路及海路與其他地方聯繫。

道路方面，與[[吉州郡|吉州郡]]及[[清津市|清津市]]有公路連結；海路方面，以[[泗浦港|泗浦港]]為中心與郡外的清津及[[羅津|羅津]]、郡內的舞水端里及木津里連結。
