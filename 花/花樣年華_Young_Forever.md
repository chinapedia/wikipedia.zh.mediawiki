《'''花樣年華 Young Forever'''》是[[韓國|韓國]][[男子團體|男子團體]][[防彈少年團|防彈少年團]]的韓語特別專輯，「青春二部曲」系列專輯的收尾作品。2016年5月2日由[[Big_Hit娛樂|Big Hit娛樂]]製作，[[LOEN娛樂|LOEN娛樂]]發行<ref name=release>{{cite web |url=http://kpopn.niusnews.com/=K0m3ml48 |title=防彈少年團五月回歸、辦演唱會|work=kpopn |date=2016-03-21 |accessdate=2016-05-03}}</ref>，主打曲目為〈불타오르네（FIRE）〉。

==簡介==
2016年3月21日，[[Big_Hit娛樂|Big Hit娛樂]]公布團體將推出特別專輯，同時在專輯推出後舉行宣傳演唱會《{{lang|ko|2016 BTS LIVE 花樣年華 on stage : epilogue}}》<ref name=release/>。到4月18日，官方敲定專輯會拍攝三支音樂錄影帶<ref>{{cite web |url=http://www.koreastardaily.com/tc/news/76936 |title=防彈少年團新專輯《花樣年華Young Forever》五月發布|work=韓星網 |date=2016-04-18 |accessdate=2016-05-03}}</ref>，並在兩天後發放先行曲〈EPILOGUE : Young Forever〉的音樂錄影帶<ref>{{cite web |url=http://kpopn.niusnews.com/=K1coccd4 |title=BTS 新曲《Young Forever》MV |publisher=kpopn |date=2015-04-20 |accessdate=2016-05-03}}</ref>；專輯概念照則緊接在4月22日、25日及26日分三批發放<ref>{{cite web |url=http://www.koreastardaily.com/tc/photo/77156 |title=防彈少年團《花樣年華Young Forever》首波概念照公開 |publisher=韓星網 |date=2016-04-22 |accessdate=2016-05-03}}</ref><ref>{{cite web |url=http://kpopn.niusnews.com/=K1ovrml0 |title=BTS《花樣年華 Young Forever》概念照2|work=kpopn |date=2016-04-25 |accessdate=2016-05-03}}</ref><ref>{{cite web |url=http://kpopn.niusnews.com/=K32fk2w4 |title=BTS《花樣年華 Young Forever》概念照3|work=kpopn |date=2016-04-26 |accessdate=2016-05-03}}</ref>。

4月27日，專輯曲目名單公開<ref>{{cite web |url=http://kpopn.niusnews.com/=K20iee89 |title=BTS《花樣年華 Young Forever》曲目表 |publisher=kpopn |date=2016-04-27 |accessdate=2016-05-03}}</ref>，次日（28日），報導指出專輯已獲超過30萬張的訂單<ref>{{cite web |url=http://www.koreastardaily.com/tc/news/77437 |title=防彈少年團新專輯預售量突破30萬張　人氣爆棚 |publisher=韓星網 |date=2016-04-28 |accessdate=2016-05-03}}</ref>。主打曲〈불타오르네（FIRE）〉之音樂錄影帶在5月2日隨專輯音源公開<ref>{{cite web |url=http://kpopn.niusnews.com/=K1ycbyy7 |title=BTS 防彈少年團《Fire》MV |publisher=kpopn |date=2016-05-02 |accessdate=2016-05-03 }}</ref>。

5月2日，主打歌〈燃燒起來（FIRE）〉發行數小時後取得韓國八大數位音樂榜即時榜第一名，達成「All-Kill」<ref>{{Cite web |url =http://www.allkpop.com/article/2016/05/bts-score-an-all-kill-with-fire|title =BTS score an all-kill with 'Fire'!|language =en}}</ref>。《花樣年華 Young Forever》登上iTunes綜合專輯榜第九名，並蟬聯三日iTunes MV綜合排行榜首。5月5日韓國時間凌晨三點，〈燃燒起來（FIRE）〉MV點擊數突破1000萬次<ref>[http://www.vlovekpop.com/20160506-bts-2/ 75小時點擊率破1000萬！防彈少年團《FIRE》再創新紀錄！] {{Wayback|url=http://www.vlovekpop.com/20160506-bts-2/ |date=20161021040327 }}vlovekpop{{zh}}</ref>。5月10日，《花樣年華 Young Forever》以第107名打進美國告示牌200，專輯10首歌曲進入世界數位歌曲榜，主打歌、收錄曲〈Save Me〉和〈Epilogue: Young Forever〉成為該排行榜前三名<ref>{{Cite web |url =http://www.billboard.com/articles/columns/k-town/7358129/bts-young-forever-album-billboard-200-world-charts|title =BTS' 'Young Forever' Compilation Album Sends K-Pop Act to New Heights on Billboard 200|date =5/10/2016|language =en}}</ref>。6月9日，Youtube5月份全球K-POP MV觀看次數，〈FIRE〉和〈Save ME〉分別奪下冠亞軍<ref>{{Cite web|url =http://www.allthatstar.com/tw/view.php?action=5205&postID=249757|title =Billboard評觀看最多KPOP MV BTS新歌佔據前兩名|date =6/11/2016|language =中文|access-date =2016-10-26|archive-url =https://web.archive.org/web/20160817092302/http://www.allthatstar.com/tw/view.php?action=5205&postID=249757|archive-date =2016-08-17|dead-url =yes}}</ref>。

9月7日，[[蘋果公司|蘋果公司]]舉行[[iPhone|iPhone]]7發表會並且介紹Apple Music新服務，《花樣年華 Young Forever》成為展示牆上七十張專輯當中唯一的韓語專輯。<ref>{{cite journal|title='아이폰7' 발표 행사에 등장한 한국 아이돌 앨범 재킷|journal=YTN|date=2016-09-09|url=http://www.ytn.co.kr/_sn/0117_201609091425298902|accessdate=2016-08-16|language=ko}}</ref>
