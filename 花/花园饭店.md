'''花园饭店'''（{{lang-en|'''Okura Garden Hotel Shanghai'''}}）位于[[上海|上海]][[茂名南路|茂名南路]]58号，毗邻繁华的[[淮海中路|淮海中路]]，与[[锦江饭店|锦江饭店]]隔路相望，由原法国总会改建而成，是一个有古典特色的[[五星级|五星级]]宾馆。現由[[日本|日本]][[大仓饭店|大仓饭店管理集团]]管理。酒店于1990年3月20日开业，现共有478间客房，5间餐厅和3间酒吧。

== 历史 ==
今[[长乐路_(上海)|长乐路]]以南，[[茂名南路_(上海)|茂名南路]]以西的地块，原隶属于建于1903年的[[德国乡村俱乐部|德国乡村俱乐部]]。1914年，[[上海法租界|上海法租界]]第三次扩张，使得原本处于上海西郊的德国乡村俱乐部被划入法租界内。1918年，德国在一战中战败，[[法租界公董局|法租界公董局]]将德国乡村俱乐部没收。对此，中国政府提出抗议，后来经协调，地皮拍卖，由法租界公董局购下<ref name="a">{{cite web|title =凡尔登花园|publisher =杨浦区图书馆|date =|url =http://www.yplib.org.cn/structure/jdsh/bnsz/jz_73250_1.htm|accessdate =2014-01-25|language =zh|deadurl =yes|archiveurl =https://web.archive.org/web/20140203082719/http://www.yplib.org.cn/structure/jdsh/bnsz/jz_73250_1.htm|archivedate =2014-02-03}}</ref>。其中位于[[长乐路_(上海)|长乐路]]、[[陕西南路_(上海)|陕西南路]]转角处的一部分地块用于建造新式里弄[[长乐村_(上海)|凡尔登花园]]。另一部分则于1924年开始建造新的法国总会（原总会位于南昌路，今[[法国学堂|科学会堂]]），建筑由[[赉安|法商赉安洋行]]设计，于1926年建成<ref name="b">{{cite web|title =法国总会|publisher =杨浦区图书馆|date =|url =http://www.yplib.org.cn/structure/jdsh/bnsz/dg_76362_1.htm|accessdate =2014-01-25|language =zh|deadurl =yes|archiveurl =https://web.archive.org/web/20140203082605/http://www.yplib.org.cn/structure/jdsh/bnsz/dg_76362_1.htm|archivedate =2014-02-03}}</ref> <ref name="p">{{cite book |title = Shanghai Style: Art and Design Between the Wars |author = 潘玲 |publisher = Long river press |ISBN = 9781592650781 |pages = 221-222}}</ref>。[[太平洋战争|太平洋战争]]中法国总会一度被日军占用，抗战胜利后成为美国海军俱乐部<ref>{{cite book |title = 永不飘散的风情 |author = 惜珍 |publisher = 东方出版中心 |pages = 147-149 |ISBN = 9787547300169}}</ref>

1954年被[[上海市人民委员会|上海市人民委员会]]（[[上海市政府|市政府]]）接管，作为市文化俱乐部。1960年归[[锦江饭店|锦江饭店]]管理，成为上海锦江俱乐部，又名58号俱乐部<ref name="c">{{cite web |title =上海黄浦 |publisher = 上海市黄浦区人民政府 |date =  |url = http://www.huangpuqu.sh.cn/shhp/infodetail/?infoid=93303715-8e9f-492a-879a-ef5b307b9656&categoryNum=002015001005 |accessdate = 2014-01-26 |language = zh}}</ref>。1985年起，由日本[[野村证券|野村证券]]投资改建为花园饭店，1989年竣工，1990年开业。1994年作为法国总会俱乐部旧址，列入第二批[[上海市优秀历史建筑|上海市优秀历史建筑]]。

法国总会曾是法租界重要的社交场所之一，改建后花园饭店亦接待过诸多外宾。法国前总理、总统[[希拉克|希拉克]]多次下榻花园饭店<ref name="c" />。

== 建筑特色 ==
花园饭店占地3.7[[公顷|公顷]]，花园面积2.8公顷，总建筑面积6.5万平方米。

=== 外部设计 ===
花园饭店由裙楼与新建主楼組成。裙楼高两层，为原法国总会俱乐部建筑，钢混结构。建筑总体属巴洛克风格，但局部带有装饰艺术特征。建筑南立面为白色，中间和两边均突出，呈弧形。底层为仿粗石拱门券，窗，2层楼两侧则有爱奥尼式[[柱廊|柱廊]]<ref name="p" />。其上曾有两个圆拱顶的亭子，1949年为减缓建筑沉降而拆除，后来仿建于花园中<ref name="c" />。新建主楼高33层，[[钢筋混凝土|钢筋混凝土]]结构，位于裙楼北侧。
