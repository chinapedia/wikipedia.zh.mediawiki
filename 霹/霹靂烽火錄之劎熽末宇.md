《'''霹靂天命之仙魔鏖鋒'''》，霹靂國際多媒體股份有限公司於2016年10月7日發行之[[霹靂布袋戲|霹靂布袋戲]]，為霹靂國際多媒體的2016年新劇集。該劇集延續上一部《[[霹靂狼煙之古原爭霸|霹靂狼煙之古原爭霸]]》之劇情。

== 詩序 ==

== 劇集概述 ==
霹靂天命之仙魔鏖鋒 夸幻之父開啟山海奇觀，引發八大高手進行一場攸關武林未來的古原爭霸。圓公子主持奇寶之爭，卻在過程中愛上魚美人，兩人珠胎暗結，圓公子不願坐以待斃，進而與夸幻之父反目成仇，主動揭露這場通天陰謀，也讓古原爭霸進入了尾聲。 
玉梁皇、樂尋遠、任平生等高手憤而轉向，夸幻之父成為眾矢之的，正當四面楚歌之際，有生之蓮解鋒鏑挺身而出，主動協助夸幻之父，形成雙方對立的局面，究竟解鋒鏑真正的盤算為何？蛻變而出的夸幻之父，踏上藏晦居，脫去靈體的外貌，將對武林產生怎樣的改變？ 
狩宇之內，皇暘耿日取得曙光之源，喚醒狩宇期盼的希望，暘神逆神暘，這名傳說精靈天下最頂尖的高手，再度甦醒，武林將陷入一場未知的腥風血雨。天魔繭夔禺疆得到九五之盒，裡面究竟會埋藏怎樣的祕密？聖母九嬰能順利復活嗎？魔流劍風之痕的命運又是如何？道劍劍非道又會遭遇怎樣的陰謀？ 
昔日精幽大戰，幽界魔君對上天織主、錻鍠寒武紀的世紀之戰，幽界與精靈天下埋藏的過往，真相究竟如何？月下孤星淚，百步銀邪槍，又會對紅塵雪與練習生產生怎樣的衝擊？墨傾池終於盡得三派單鋒劍式，他真正的目的為何？又會牽連出何種淵源過往？ 
窈窈之冥，天地人三光齊現，神秘的天跡、人覺、地冥，這三人為何開啟末日預言？究竟人中之龍代表為何？梵天一頁書的希望又在何處？謎中謎，局中局，撲朔迷離，一切分曉，請繼續收看霹靂天命之仙魔鏖鋒。
== 登場角色 ==

* [[解鋒鏑|解鋒鏑]] (素還真)
:「有生之蓮解鋒鏑」意指有生之年，皆心繫於弭平烽火狼煙。在素還真亡心後，被埋入葬神之野九丈地底，開啟九輪天的玄天魔能，卻意外激盪素還真的芬陀利華清淨不垢體作用，因而修補傷體，讓素還真重生成為解鋒鏑。解鋒鏑雖失去了素還真的記憶，卻因天命使然，開始介入九輪天與苦境之戰端，從中以智慧斡旋各大勢力，守護整個武林。
因闖泥婆暗界需要生字卷，而讓赦天琴箕協助其恢復記憶，之後又一連串奔波，雙鬢已白，再過不久即會恢復素還真身分
* [[圓公子|圓公子]] (黿無極)
:八面玲瓏之主，具神秘感以及翩翩貴氣的美男子。玉樹臨風中，帶有冷利的眼神，看似為人隨和海派，卻又會瞬間變臉、反覆無常。家產雄厚、富甲一方，表面耽溺於宴樂歌舞享受，心裡卻是最深沉難解，故圓公子底下之美人與奴僕對其格外敬畏，絲毫不敢怠慢。
與患天常一戰受傷嚴重功體盡廢，夸幻的咒術也被擊破，變回原本面貌。

* [[夸幻之父|夸幻之父]]
:位於神秘洞窟中的龐然怪獸，外表奇異、巨大，總是據地而坐。看似愚鈍，實際卻陰險狡詐、冷酷殘忍，喜愛命令手下誘捕女性至洞內，供其選親繁殖，若有不從便即刻將其殺害。過去曾與縱橫子以棋相賭，並以雨霖鈴和自身幻甲作為籌碼，因而也讓縱橫子兄妹間產生嫌隙。頭上有著精靈三角，四肢各別有著三支幻甲，分別被雨霖鈴製成格天三鋒、帝弓十二虹與虎尾春冰。後敗於練習生，墜入煉火深淵，卻在多年後又開始進行新的計畫。
於16章遭暘神擊斃，蛻變成 禪劍一如 寄曇說

在圓公子反叛之後，被百寒之氣凍為人體，被解鋒鏑所救，之後解鋒鏑故意透漏行跡於匆匆，邪天子與玉梁皇隨即前往刺殺夸幻之父，在被白衣劍少救出後，解鋒鏑令他服下一頁書復活靈丹。因和暘神一戰，使用大量真力，而讓聖珠啟動，待有朝一日，一頁書便會重生。
* [[亂世狂刀|亂世狂刀]]
:性情剛烈霸氣、豪邁奔放，為愛執著不悔的血性男兒。刀法精純、氣勢驚天，以一口獅頭寶刀、一套狂龍八斬法揚名天下；用情至深，為愛妻慕容嬋可堪任何犧牲、甚至一日三千斬亦在所不惜。因緣際會下得道尊相助，成為道教傳人，在歷經諸多波折和淬鍊下，心性也更為成熟、圓融，體悟出無心無我的刀鋒境界，蛻變再出的狂刀，其修為、造詣更上一層樓。

在他救出芙蓉之後，被金蛾人(20多年前被狂刀重傷但未死的人)盯上，展開一連串報復。

擁有單鋒劍境，實力高超，運使單鋒時，寶刀會變成藍色。

不動城解散之後，狂刀在墓前悼念劍君，感嘆著自己的刀法無人能證，葉小釵便拔劍與狂刀一決。對戰中狂刀與葉小釵皆看見了對手化為劍君的幻覺，彷彿是昔日三人刀劍共舞之景。
* [[葉小釵|葉小釵]]
:耿直專一、守信重諾的俠客，雖為侍童出身，但不屈不撓的個性，豐富的人生閱歷，讓其成為中原武林不可或缺的一員，享有「刀狂劍痴」之盛名，披肩的白髮和臉上的英雄疤，搭配沉默無言的特色，營造出該角色獨特的風範。

不動城解散後，暗中保護黿無極，但黿無極一心平凡，葉小釵便答應他不再關注他們的行動。之後前往劍君之墓，拔劍與狂刀一決。對戰中狂刀與葉小釵皆看見了對手化為劍君的幻覺，彷彿是昔日三人刀劍共舞之景。
* [[白衣劍少|白衣劍少]]
:魔流劍風之痕之徒弟，個性冷靜穩重、重情重義，特異的體質不畏一般劇毒。被魔劍道之主誅天所收養，成為魔劍道少主，有著複雜的身世，另一身分為西疆皇朝太子。師承風之痕劍法，其劍法出神入化，於風中快意的冷靜，置身在劍速之極意當中，已達自我巔峰。
誰知師尊魔流劍風之痕，在天魔繭惡意玩弄之下只能與不動城一戰戰死，白衣千辛萬苦復活魔流劍之後，竟中天魔繭詭計，風之痕魂識遭天魔繭改造，而失去自我。
喚回風之痕的關鍵一戰遭穿心一劍重傷瀕死，由緋羽怨姬治療，而暫時退出檯面。
* [[生命練習生|生命練習生]]
:當代用槍之高手，人稱槍神，乃為人正直、豪放不羈、直率敢言的熱血男兒。看似驕傲，卻從不自視甚高，自有一套探索生命的人生哲理。曾在万堺論衡結束後，與劍非刀以武比試，兩人相互競亮中同也肯定彼此，並結交為友。堅持以生命練習槍法，盼以萬千次的練習來達到武道之最，並成為真正的槍者巔峰。真實身分為精幽大戰的魔君
* [[紅塵雪|紅塵雪]] (天子槍)
:倚晴江山樓之主，亦是隱流左單鋒之主。文秀儒雅、貌美如花，善於琴棋書畫，學識淵博又天賦異稟，能將知識傳授給靈獸。看似武林少見才女，實有劍中洛神之美譽，其武學脫胎於父親天劍老人的極單鋒，練就自成一格的「翩單鋒」，宛若遊龍，點化為驚鴻七式。
第二身分為現今的【天子槍】，武器為  :帝詔‧闋九重
* [[劍非道|劍非道]]
:乃万堺紀年的一代遊俠「劍非刀」在死後復生的新身分。清荷元年時期，劍非道已成為道門太上府的首徒，手持玄笛，身揹道劍相忘，風姿清卓、一派偉岸，珞珞出群的表現，因而被委以重任看守道鎮伏魔崖。在看似沉穩的性格中，卻有著對邪惡不容肆虐的正義之心，因而在幽都、魔流劍、歎希奇等勢力巔亂武林時挺身對抗。

在天魔繭用計之下，遭取出雙眼
* [[巧天工|巧天工]]
:巧天工，人稱芙蓉鑄客，愜意居於青梗山。總自稱本芙女，性情刁蠻古怪、伶牙俐齒，最喜閱耽美書籍。為四大名手之一，鑄劍功力堪稱一流，更對劍的特性瞭若指掌，頭上的髮釵拔下後可化為單鋒劍。亂世狂刀與沽命師都曾請求其鑄造單鋒劍，巧天工表面看似刁難狂刀，實則有深意寄託其中，是一介以補天裂鑄術聞名的女高手。

已於古原爭霸中以詐死之法退隱
* [[夔禺疆|夔禺疆]] (天魔繭)
:天地雙繭之天魔繭，張狂自信、威風凜然，為幽界之梟雄領導，為大局著想而不擇手段，視幽界霸業為畢生首要目標。出身於繭族，由金絲魔繭脫胎而出，故在天繭周身有肉眼難視的金絲繭線護身，能以此防禦、攻擊，展現不世的魔主霸威。

吸收了聖母，控制風之痕，但不知遺體可使其復活
* [[無限|無限]] (地繭)
:來自幽界至高無上天地雙繭的「地繭」，邪魅冷峻、孤高優雅，平時待在神亡異空內，並坐在羽毛繭上，持鬼面弓弦來拉奏陽魔琴，將來犯之人殺退。雖是守護幽界無法離開，但地繭卻能分化出六色各不相同的殺魂，六殺體各有著地繭身上的其一特徵，並代替地繭完成一切幽界的任務。
* [[任平生|任平生]]
:性格淡泊的江湖行者，以一派閒情度日，十足愜意。手持玉竹杖，喜愛踏青，更有許多自己發現的私房景點，所以常帶著朋友一同遊山玩水，也替人導覽風景名勝。任平生以誠待人，故朋友與客戶來自四面八方，有武林高手，更有市井小民。但任平生真實身份，乃單鋒劍隱流右單鋒之主，武學高深莫測，與之隨和性格相比，更顯得其鋒芒內斂。
為搶奪硫炎靈帖，而心性墮落，走入歧途，因知道當年與天劍老人的爭執是一場誤會，痛改前非。
* [[墨傾池|墨傾池]]
:儒門高層聖司，創立現今的「文詣經緯」。清冷肅氣、威德兼存，在文士風骨之下卻也略有傲氣。一身儒風的雪白衣裳，顯示出忠於儒學、崇尚古賢，內外清心並遵於正派的處世之道。雖隱居於幽靜之地，實則對天下之事瞭若指掌。墨傾池名字代表「清池染墨仍清，是表即便遭受染污，仍維持清心，心堅意定，不移本心。」
* [[玄武魄|玄武魄]]
:魔吞十二宮「玄武宮」代表，臉戴玄武異譜，烏黑長髮束起後隨風飄揚，有著超凡神力，出招時總颳起強大颶風捲碎敵人。神龍見首不見尾，平時亦少與夥伴們聯繫，卻被安排了祕密的任務獨立執行，並適時在逆三教、九輪天最關鍵一戰中給予夥伴最強大的後盾支持。
* [[弒君士|弒君士]]
:出身啟示國度的高手，聖禦八士之一，原名傲軍士，圓武院之主，因親手斬殺佔王而以弒君士的稱號聞名。以圓武院強大的軍事力量統御應許月灣。過去曾受瀕死重創，全身重傷，體無完膚。雖身心受到極大痛苦，但也激發出超越以往的修為。
* [[玉粱皇|玉粱皇]]
:武都玉嵎之主，乃「槍中之皇」，素有千面修羅槍之稱。不可一世、城府深沉，時而霸氣、時而冷酷，在過去北辰皇朝覆滅後，玉梁皇以兵征伐，一統北域四族，成為北嵎新主，建立槍之國度御宇皇朝。
* [[邪天子|邪天子]]
:六道神兵府之主，出身草莽綠林，瘋癲過人，過去被稱為「瘋王」，行事總是不按牌理出牌，瘋瘋癲癲，讓人難以抓摸，不時還會拿出鏡子照自己。邪天子與玉梁皇、應笑我乃舊識，故六道神兵府除針對天子槍之外，更與玉梁皇合作共謀山海奇觀，積極在暗中運作。
* [[應笑我|應笑我]] (單鋒劍魔)
:魔單鋒創派之主，雖名劍魔，但非妖魔之身，乃修練成魔。性格冷邪，專注自我武道，追求「滅我之境」，在經歷過七七四十九次自我毀滅後，再入江湖，意欲殺掉天下間的單鋒劍者，讓魔派成為單鋒之首。
* [[皇暘耿日|皇暘耿日]]
:參加曠世窮武的八大高手之一，八種武學中“奇”之領域的代表。昴宿兒之兄，為長耳獸族「狩宇」之主。個性沉穩，行事看似慢條斯理而迂迴，實則作風明快，斷事果決。曾拯救整個狩宇走出瀕臨滅絕的危機，令族人心悅臣服。「狩宇」一族崇拜宇宙間的日月星辰，以日為君，而暘帝即為像徵太陽一般光明偉大的存在。
* [[別見黃花落|別見黃花落]] (箕子臬)
:來自異國，自稱穿夢旅行者的奇人，因具有夢牒之能，因而與具有字牒的盜天下有所交集，也因此涉入怪販妖市之事務。個性古靈精怪，無法忍受別人對自己的忽視，是故在龍戩對別見黃花落下達永不錄用妖市之臣的命令時，反而激起別見黃花落的鬥志，只要是與龍戩、盜天下，甚至是整個怪販妖市有關的事物，定要從中插手，證明自己的能力不容忽視。
* [[夢不覺|夢不覺]]
:神秘的浮動山城主事，地位崇高的大極宮「司天」，氣度高雅淡定，擅長夢占之術，修為高深莫測，為夢牒的最先持有人，與素還真為舊識。昔日箕子臬彈響優律天琴，本當入主大極宮，但卻因占卜算得不詳之果，讓夢不覺設下諸多考驗試煉，不料出現穆天教派作手、以及既濟此人之變數，讓事情發展逐漸脫離掌控，甚至衍生夢魔橫空出世之危……。
* [[既濟|既濟]]
:浮動山城的珠寶商人，肩負寶石布袋，為別見黃花落夢中出現的神秘人物，雖是市井商販，但交遊廣闊，儼然有一方民間勢力之氣候，深藏不露。於夢境中與箕子臬相遇，解大極宮通緝之危，而後結伴而行，然其目的不明，後更意外讓箕子臬為地母之靈所咒。既濟之能，就連大極宮司天也難以掌控，且似與穆天教派有所牽連。
* [[琥珀|琥珀]]
:精靈之女，原始型態為一顆琥珀寶石，被生命練習生以熱血溫醒後，三天迅速長成少女，性情就如貓一般，會喵喵叫，喜歡玩逗貓棒。個性天真無邪，對世事充滿好奇，隨著逐漸成長，會有如貓一般，帶有內斂、優雅、自負、忠於自己的感覺。
真實身分為禁城罌粟‧天織主
* [[嘯日猋|嘯日猋]]
:魔吞十二宮「赤龍宮」代表，臉戴赤龍異譜，透露著驚人的血紅之光，霸氣凌人下，作風更是果決，為不動城極為神秘的隱藏勢力。手持神刀龍鱗，承認自己會承接赤龍異譜，乃五龍之影永存心中，並以此來緬懷兄弟之間的情義，卸下異譜，則是『不動城』任務完結之時。
* [[棄神類·虩|棄神類·虩]]
:棄神谷中的最強者，令人驚駭恐怖的存在，一開始不存理智，只會發出吼聲，對實力高強的活物有極高的興趣，一旦打敗對手後，就會失去興趣。在聽到琥珀的歌聲之後，開始有了別的堅持，一心要琥珀待在自己身邊，唱歌給自己聽。虩具有特殊的灼世烈眼，棄神谷中的迷霧與潛藏的危機，在他眼中形同虛設。
雙眼為特殊的灼世烈眼 可發出雷射光，克制精靈，劍非道因被射擊到背門，舊傷爆發。
* [[棄神類·鴀|棄神類·鴀]]
:不存理智、只存殺戮之心的嗜血獸人，不會講話，生毛長爪、身形似鷹，在棄神谷中攻擊聖君士與紅塵雪。實際上鴀是聖君士的教父，當年於棄神谷中重創未亡，後來異化為棄神類。
* [[若葉汝嬰|若葉汝嬰]]
:出身於黑海森獄望族——"若葉家族"，天性率真可愛。但卻因若葉家在一次事件中觸怒閻王，整個若葉家族被除籍，永世貶為平民，若葉家血脈因此凋零四散。若葉家主以異術強行把他變回小孩狀態，託付給千玉屑。因赬手奎章被屬於千玉屑的感情所影響，因此也費盡心神保護若葉家族所剩唯一的血脈若葉汝嬰。
* [[聖母|聖母]] (九嬰)
:幽界之中，以黑暗之母—九嬰為眾人尊崇的聖母，疑似幽界主宰。
* [[夕華沉|夕華沉]]
:浮動山城之東極暠主，後隱居優律山城。
* [[夢魔|夢魔]]
:浮動山城之夢魔。
