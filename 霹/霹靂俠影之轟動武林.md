《'''霹靂俠影之轟動武林'''》（{{lang-nan-poj|Phek-le̍k Kiap-iáⁿ chi Hong-tōng Bú-lîm}}）是[[台灣|台灣]][[大霹靂文創整合行銷股份有限公司|大霹靂文創整合行銷股份有限公司]]發行[[霹靂布袋戲|霹靂布袋戲]]於2013年1月12日發行，霹靂國際多媒體於2013年的開春鉅獻的新劇集。該劇集延續上一部《[[霹靂驚鴻之刀劍春秋|霹靂驚鴻之刀劍春秋]]》之劇情。

== 主要登場角色 ==
; 齊煙九點‧[[天踦爵|天踦爵]]
: 真實身分為素還真與時間城所締約的第一段時間化身。

; 三餘‧[[無夢生|無夢生]]
: 因天踦爵的時間將用盡，故在前往時間城在求延時（刀劍春秋第十章），三餘無夢生就是素還真訂下另一時間契約所化出。為第二段時間。
: 刻意以不同身分誘敵。一登場的神祕舉止，竟令屈世途等人驚奇不已，面對接下來的武林變局，三餘無夢生會有什麼乾坤妙計？

; 丹華抱一‧[[鷇音子|鷇音子]]
: 身揹天榜，手持時計，居於羅浮山的神秘奇人，擅煉玄丹，偏向道家色彩。真實身分為素還真的化身之一，在無夢生前往相借非關時間的考驗中，途中歷經殊離山魔考一時失魂出神之下，使素還真第三魂體化為飛鷹離去，剛好在聖魔元史正在尋找宿主之時意外發現此飛鷹，而追上並點中宿主，並向時間樹緹命借時而出現的第三段時間化身。
: 未現身前以未來的形影解救了地獄變，一登場便影響了無夢生，雖表明立場中立，但卻以烽火天榜揭開未來武林最具影響力的風雲人物十二人，目的是為了攪亂武林。而三餘無夢生（或者說是素還真）是最大的阻礙，所以聖魔元史的目標才會瞄準了素還真，打算以素制素。
: 因是意外而締下契約的化身，不能與同為契約下的無夢生主動相見，否則就會像王見王一樣——死棋。而無夢生亦是如此。但就在引無夢生來相見時，被無夢生以搏命的方式，巧妙的利用逆時計化掉了此障礙。因此王見王，再也不會有死棋相剋之相。

; 百世經綸‧[[一頁書|一頁書]]
: 梵天再度復出後，因被天之厲以及魔皇夾殺，而導致半身不遂。後因天踦爵與眾人的努力下，得到流火陽鐵以治癒所重創之龍骨以及步香塵的治療之下終究痊癒。後得知陽鐵乃烽火關鍵所在，後梵天接下重任，成為烽火關鍵人。

; 塵外孤标‧[[意琦行|意琦行]]
:意琦行爲内七修之首，更是首位通悟内七修武學之人。意琦行素以絕代剑宿自詡，武學特色自然以劍爲主，其劍法雖以武道七修為基礎，然其造詣，實以遠超七修門法。红炉点雪為其成名招式，然此招背後，非單純的警告取命，實有禪機寓於其中。意琦行個性孤傲，但他十分重視武道七修的完整性，所以在淵頂之戰后，曾將七修武學傳與有緣人。
:意琦行回到戰雲界后，恢復戰雲界第一戰士「絕代天驕」的身份，帶領戰雲雙擘對決妖界。但同時體内的天之厲意識也開始覺醒，進而佔據意琦行原本意識，幸三餘、鳳座察覺有異設下防護措施，最後在與聖嬰主交易下，以意識之刃斬殺天厲，恢復自我。
:意琦行當年入苦境尹宴荒城，武脈混亂險亡，遇見百岫嶙峋以一殺一救原則用刀氣封第三武脈所救，造成意琦行武者身心上的隱憂，后在沐靈山開化解封下，重新悟得神識之劍。身為人三脈的意琦行，也背負起淨化天地人叁脈的天命，協助破除塵世暗夜。

: 在重遇雨鐘三千樓後裔雲滄海、無故事的人等江湖事，重拾佩刀，對上以報盛華年之子被殺的「葬刀會」、報雲滄海之死的「天佛原鄉」、報兄弟之死的「烈武壇」，最後在超軼主奔走阻止烈武壇復仇及告知痕江月陰謀， 殺了葬刀會痕江月，報了君舍魄、星狼弓、寄天風、憑風一刀之仇，卻引出了「北狗」最光陰找上綺羅生，將之帶入時間之城。

; 儒門龍首‧[[疏樓龍宿|疏樓龍宿]]

; 北狗‧[[最光陰|最光陰]]
: 名列五大傳奇之一「北狗」，又稱老狗，臉帶狗頭面具，創立蝕劍刀法的冷酷刀客。
: 因長時與犬獒為伴，其行為模式與常人不同，因自覺長相與狗不同，而戴著狗頭面具遮掩自己外貌，曾參與第過一次凋亡禁決，殺盡無始暗界上千妖族，僅存活三名殘存者：酆都三千里、葬雲霄、釋閻摩，但凋亡禁決結束多年來始終下落不明。
: 其真實出身來自時間城日晷集日精而成的無魄之人，受魄冠大禮，掌掠時任務，後在離開時間城修行時遇到刀神九千勝，與之交手未落敗，故而結交莫逆，九千勝並贈與一隻雪獒，埋下日後在金獅帝國收養小蜜桃之契機。後九千勝帶其參加琅華宴，向召集人文熙載推薦其為「元字第座」，埋下與烈劍宗少主暴雨心奴不滿，於是凌辱殺死文熙載之女，陷害最光陰，再以地獄十八陣殺死九千勝。最光陰為替好友九千勝求得活命之機回到時間城，捨棄魄冠自願降級，將其心轉給九千勝，依靠著逆時計而活(綺羅生有雙心由來)，並與時間城訂下契約，顧守時間樹。依逆時計作用，每到他二十九歲的生命終數時就會回朔十年，而這回朔的十年間一切記憶都會消失，就在漫長的時間消磨下，讓其遺忘了初衷目的，也逐漸產生厭倦之心。於是他決定去做一些不一樣的事而離開時間城，自己丟棄逆時計並且開始行走江湖，因接受金獅窟受害者的請託，才因此參加了凋亡禁決。
: 常年伴陪身邊的小蜜桃於一次戰役中走失，老狗就此銷聲匿跡尋找其蹤，後因熟悉之感而找上綺羅生。

: 久遠前與七修創者一戰後重創而銷聲匿跡，得策夢侯以八品神通治癒。對流火陽鐵失落一事極為關切，言明此乃烽火之關鍵。
:受戚太祖計謀而逐漸與暮成雪、判死生等人產生代溝，在畸光族被不明人士以自身武學滅族後更是百口莫辯。後遭被天之厲意識占據的絕代天驕重創後，陷入葬刀會殺網，更遭屬下吾不留背叛的致命一擊，最後於一往情徑的無言花海中逝世。

=== 丘山百妖路 ===
; 聖嬰主‧[[焱無上|焱無上]]
: 怪樂地的真正領導者，自稱本爺，天生擁有意識燄能，其性狂傲好戰，能運使古傳六訣，被傳為妖界創者「鬥神血泣」之轉世。昔日巨魔神一役後，又意外因「逆．時計」的影響而逐漸退化成嬰兒之身，怪樂地一夕失去焱無上、妖天師兩大高手，所幸怪樂地依靠妖界的平衡而未被併吞，由妖剎剎接續領導，並詐稱嬰兒為妖剎剎之子。
: 後怪樂地被滅於鬼荒地獄變後，化為嬰兒的焱無上流轉至步香塵、娛笑顛之手，後為擺脫嬰兒之身，答應無夢生條件，以意識之刃殺除天之厲意識，讓意琦行擺脫天之厲的影響，不料焱無上運功將「逆．時計」回歸到無夢生身上，趁機欲一併除去意琦行意識(為綺羅生所救)。接續聯合封世末、獄天玄皇殲滅欲界第三天勢力，並在妖界三雄爭霸中取得勝利，其個性看起來散漫不拘小節，卻十分清楚目前妖界狀況，唯有解放妖界深層增加妖界實力，故在交戰當下詢問末世、獄天玄皇誰願意犧牲自我，就在獄天玄皇自願時，轉而將封世末殺死，其妖能皆回歸於獄天玄皇之身而回復完整的妖能，並將封末世身上所取出的初末雙心賜給釋閻摩提升妖能，其手段突顯聖嬰主對封世末所想，認為妖界當以團結為首要。

; 黑獄之主‧[[獄天玄皇|獄天玄皇]]
: 乃墮神闕與初代鬼荒共同擁戴的強者，手腕柔軟，能屈能伸。與暗界穹魁在巨魔神一役後，被佛鄉所禁，本應遭暗界之主穹魁封世末吸收妖氣而亡，卻因事先在煉池妖脈中寄入一點靈識，埋下日後復生契機，墮神闕生前將自己一半妖氣和雙佛靈氣注入妖脈之中，使獄天玄皇得以重生且再度重掌黑獄。
:性格沉穩，以妖界未來為大局考量，並希望世人能夠破除妖界並非殘忍好殺之偏見，對無始暗界殲滅黑獄、壓低自己地位之行徑採取退讓態度，並與三餘無夢生十分交好，多次協助正道、佛鄉對抗欲界，故與封世末主張對外侵略、暗界為尊的理想截然不同，是以聖嬰主考量犧牲妖界領導換取烈雲洞府妖脈一事上，選擇犧牲封世末，並歸還了原被封世末吸取的妖能皆回自身，最終回復完整的妖能，其妖能恢復之強大，連凝聚洗罪三尊之功於一體的赤慧慈航走不下一招而被殺。而後協同佛鄉在青蒙山防堵涯十滅復生、殲滅欲界勢力一戰，迷達意外現身，並施掌重創獄天玄皇，玄皇臨死之前，將其妖力、黑白雙玉佛交付給無夢生，變消散於妖界之中。
		
; 暗界之主‧[[穹魁封世末|穹魁封世末]]
: 暗界最高領首領，以「末」自號，斷絕塵寰始章。
: 無始暗界之主與獄天玄皇在巨魔神一役後，遭天佛原鄉所禁，藉著吸收獄天玄皇的妖能而存活下來。生存意念強烈，認為妖界的一切，皆是為了成就自己而存在，自己則是為了造就妖界輝煌盛世而生。最終因自負以及自己的理念而被聖嬰主所殺，其心為聖嬰主賜給釋閻摩。

; 犍羅金剛 
: 欲界初天主事者，為人自負驕傲，如同早前八方壇主的位階。執掌前哨先鋒統籌，負責整頓新舊兩部成員(新部由鳴中孚招募，舊部由巫六奇統合)，由洗罪三尊得知玉海九輪盤內開天之路為金獅壁窟後，便率眾前往金獅壁窟，合洗罪三尊擊退絕代天驕，一舉攻佔開天之路。豈料因佔領金獅壁窟這個舉動惹惱了北狗，被攔腰斬殺而亡，屍體由部屬部洲神將、部洲天護帶往第二天，本身靈氣回歸魔佛波旬，加強波旬破封靈力。

: 欲界第二天主事，駐紮青蒙山。老練而有威儀，在犍羅金剛遭北狗所殺後，接替統整欲界勢力的責任，甫出場便一解無戒悲航之禁錮，將無夢生視為必除對象，命令座下武將佛戰對付無夢生。並積極與洗罪三尊維持聯繫，關注各方勢力，得知妖界百妖路折損不少戰力，在新仇舊怨合併清算下，進攻百妖路，不料反中甕中捉鱉之計，須彌帝釋、佛戰、部洲神將、部洲天護相繼戰死，初天、第二天勢力瓦解。

; 雙世同生‧[[律懲王|律懲王]] 
: 欲界第三天主事者，根據地為律天。在欲界第二天遭妖界所滅之後，出面整頓欲界勢力，得知佛骨凶兵能破欲界特有金剛不壞護體，在輔座忘塵緣安排下，順利除去荒初禁赦，斬斷妖界一臂，並迅速下達進攻妖界的命令，而後為盡快開通「開天之路」放出魔佛及天佛原鄉內欲界部眾，積極追殺金獅幣持有者，卻仍一無所獲，最後妖界新主焱無上為報復欲界，領獄天玄皇、封世末合力殺上律天，律懲王、善法衛、無戒悲航盡死於一役。

; 諸境如想‧[[涯十滅|涯十滅]]
: 欲界第五天主事者，職為欲界掌命，教下有七行宮之主鳴中孚、缺天赤、仁王凡七夜，為欲界各天中實力最強、權威最大之領導者，連地位如「輔座」忘塵緣在其面前仍需恭敬數分。本身實力不凡，在玉海九輪盤中赫然現身，一出場統整欲界初、二、三天游散信眾人馬，並成功以金獅幣通啟開天之路，迎接魔佛波旬後，聯合妖界，攻上天佛原鄉，不料聖嬰主倒戈刺中涯十滅，使之爆體身亡，因魔佛加持緣故，選擇青蒙山復生，在忘塵緣故意顯露欲界行蹤、暗中找回迷達，佛鄉、妖界聯手欲除根絕惡反中其計，損失慘重，獄天玄皇、龍吟法問、琶語僧律均喪身此役，涯十滅得以重生。
: 而後得知魔佛閻達為天佛原鄉所擒，錯判大宗師已殺死佛鄉高手，佛鄉目前勢單力薄，遂大軍出動，命缺天赤留守教內，忘塵緣、佛劍守住後路防止變數，本是一面倒擊敗佛鄉，不料大宗師助裳瓔珞一掌，巨魔神被牽制，佛鄉高手譴彌勒、臨江悟北、無惑渡迷出現助拳，使情勢轉而僵持不下，兼之佛劍分說恢復原狀，殺死忘塵緣後衝入戰場，鳴中孚被殺、欲界立即潰不成軍，涯十滅與迷達退走回到總壇，不料正逢北狗最光陰為報廉鴻彧被殺之仇，教內缺天赤等人已亡於刀下，佛劍分說、裳瓔珞阻斷退路，至此涯十滅選擇死於北狗最光陰刀下，讓靈氣回歸魔佛迷達身上，助其脫身。

; 聖行者‧[[佛劍分說|佛劍分說]]
: 三教先天之一、有著「聖行者」之稱，與疏樓龍宿、劍子仙跡並列三教頂峰，亦被世人稱為 [[三先天|三先天]]，其個性剛直不阿、正氣凜然，曾一度亡於黑獄的佛劍分說，其屍骨遭到妖皇鍛鍊成佛骨凶兵。
: 後在忘塵緣與疏樓龍宿的奔波後復生，不幸為身為欲界暗樁的忘塵緣從中做手後意識遭到波旬迷達以密咒影響，成為欲界強大戰力之一，與五天之主涯十滅平起平坐。

=== 戰雲界 ===
; 鳳座‧[[朝天驕|朝天驕]]
: 戰雲三驕之首，耿直中帶有高傲的氣息，為人自信，凡事說一是一，溫和平穩中帶有王者的冷酷風範，遇事則保持一貫的冷漠淡定。由於當初巨魔神攻打妖界時失落陰元，致使朝天驕在內的四大奇觀之主受困封印；在陰元回歸解封後，正式帶領戰雲界攻打百妖路，不料此戰竟折損戰雲雙擘，使戰雲界元氣大傷，轉向奔走意琦行被天之厲意識困擾一事，雖成功解除胞弟之苦，戰雲界卻也因主事者不在的情況下，由煙都偷走巨魔神，覆滅戰雲界。鳳座雖與冰樓合作推敲兇手與煙都有關，卻無直接證據。

: 後大宗師古陵逝煙忌憚朝天驕可操控巨魔神，兼之要奪朝天驕體內「雲元」，命令徒弟宮無后一劍取下朝天驕首級，卻因宮無后未以一劍風徽劍法嫁禍，使冰樓之主玄冥氏在一字鑄骨證實朝天驕屍身上劍痕後，確認煙都為幕後元兇。

;[[御宇天驕|御宇天驕]]
: 戰雲界三大驕傲之一御宇天驕，部屬稱其為「御主」，英挺雄拔、強傲自信的戰雲界強者，渾身具有不怒自威的王霸之氣，下轄四凶戰士，曾敗天之厲，為戰雲懸圃之領導者，後成為追亡狩之一。費盡心思讓意琦行 (絕代天驕)回歸戰雲界，因多次維護身為辟命敵的意琦行，最後為鬼荒地獄變偷襲身亡。

; 塵外孤標‧[[意琦行|意琦行]]
: 人稱絕代劍宿，乃武道七修第一人，其真實身分乃戰雲界三大驕傲之一「絕代天驕」，其性情冷傲脫俗，超然獨處，思想行為不同凡俗的絕代劍宿，在七修中有著「武始通修」之美譽，亦狂亦俠亦超塵。近日與同修好友太羽驚鴻‧[[一留衣|一留衣]]， 好友. 白衣沽酒‧[[綺羅生|綺羅生]] 挑戰鬼荒‧[[地獄變|地獄變]]，最後慘遭敗戰，好友一留衣遭重傷死亡，意琦行決定回歸戰雲界，重拾「絕代天驕」身份。

; 裁令主[[寰無疆|寰無疆]]
: 戰雲雙擘之一，高傲自大，語帶刻薄，雖與御宇天驕、絕代天驕為同儕，卻是百般針對、多番刁難，只服膺於鳳座朝天驕。在鳳座領導戰雲界與妖界一役中，裁令主對上荒初禁赦，大意輕敵反命喪佛骨凶兵刀下。

; 雲師[[造烽煙|造烽煙]]
: 與裁令主同為戰雲雙擘，但有別於裁令主的挖苦，語言風格較成熟穩重。對戰雲界忠心耿耿，負責管理戰雲界戰俘訓練，麾下馭有冰火饕餮等戰力。心知裁令主對意琦行有所偏見，故言語間對絕代天驕較為偏袒，在鳳座領導戰雲界與妖界一役中，造烽煙對上酆都三千里，被其所殺，使鳳座此番出征損兵折將，埋下日後戰雲界戰力大損遠因。

; 胤蒼狼
: 為御宇天驕座下謀士，在御宇天驕身亡後，積極主導意琦行回歸戰雲界，待鳳座破除封印後，為其所用，不料巨魔神失竊，導致戰雲界隨之落地雲爆，胤蒼狼也亡身在此。

; 曦眼蒼鶻 
: 為一忠心勇武的驕傲戰士，眼睛具有能找尋渾淪陰元的特殊能力(金曦眼)，由胤蒼狼指派尋找渾淪陰元，雖成功找上漂血孤島的老狗與綺羅生，結果碰到老狗找不到小蜜桃發洩怒意，曦眼蒼鶻尚未及反應便中刀身亡。

=== 冰樓 ===
; 冰王‧玄冥氏
: 冰樓之主，專長打造神兵，尤擅神弓，與朝天驕、古陵逝煙、杜舞雩並列四大奇觀領導者，儒雅明理，作風穩健，與檯面上的霸主有所區隔。自陰元回歸後，便積極調查當年質子回歸的血案真相，並與戰雲界鳳座結盟，協助調查巨魔神被盜一案。

; 霜旒玥珂
: 冰樓公主，昔時冰樓遭封，自己也陷入沉眠狀態，因身為王脈一員，致使冰封過深，無法自行解封，後因疏樓龍宿之助而重見天日。習有冰樓絕技「雕霜沐雪奪形訣」，能為佛劍分說塑造重生之軀。性情嬌縱，興趣是收集鑑賞天下奇珍，也因此為龍宿的華麗風采所傾慕。

; 鏤冰氏
: 冰樓副樓主，疏樓龍宿破解冰封後，進入冰樓所見的第一人，負責協助總理冰樓內務，謹慎細微，在得知龍宿目的後，領其破除被冰封的公主霜旒玥珂，得以重塑佛劍分說生機。

=== 煙都 ===
; 大宗師‧古陵逝煙
: 煙都眾人崇仰的大宗師，高深莫測，武學造詣有「一式留神」之美稱。性格冷言疏淡，看似退居幕後，卻凡事盡在其眼中，運籌帷幄掌控煙都行事方針，心機深沉難以臆測，干係著冰樓血案與巨魔神失落之謎團。

=== 天葬十三刀 ===
: 組織內成員非為刀客之流，與葬刀會專門埋葬刀者的組織有所不同，其存在的目的為找出名曰「天葬十三刀」地名，後因聚集有志一同的人，故轉變成一個組織的稱呼。如果成員有意退出組織，需由組織內成員共同表達同意，才可以離開，一旦內部成員有人死亡，則由召集人黃羽客尋找適合人選遞補。另外天葬十三刀與葬刀會創辦者步武東皇有相當關係，步武東皇故能調遣痕千古、別黃昏兩人。

: 天葬十三刀由黃羽客出面召集各方能人異士、武功高手，目前所知奇技者為鬼盜(盜賊之最，盜術臻智能竊人靈魂)、喫神(佳餚能始食用者如臨仙境)、畫聖上官圓缺，武功代表者有痕千古、別黃昏。

;夜歌殘賦‧[[痕千古|痕千古]]
: 痕江月雙胞胎兄弟，出自煙都，舊名為「痕千宮」，喜愛在雨中聆聽劍律小酌，其招如輕煙、式如詩賦，武學總稱為五更賦。在大宗師與步武東皇共謀後，易名千古，並加入天葬十三刀。性格冷情殘酷，操縱慾強，曾將遭烈武壇驅逐而投靠葬刀會的痕江月，調教成如同自己影子般的存在，教之天崩六朋劍，並賜化影神銳。
: 過去參與過初次凋亡禁決，認識釋閻摩。也參與屠殺金獅帝國的鷹戰行動，逼殺癲不亂，並將忠於先帝烈顏真的護衛姜回殺死，卻遇到北狗，敗其手上，唇上留有一痕，埋下日後獨孤毒為報殺夫之仇，開出要殺人兇手人頭的條件，故戚太祖為與獨孤毒換劇毒一眼留魂的解藥，將他斷首。

;塔鈴獨語‧[[別黃昏|別黃昏]]

: 個性冷沈孤僻，不太與人接近，少見情緒波動。習慣佇立於黃沙荒塔之下，專注仰望塔尖上的銅鈴，聆聽隨風迴響。殺人時，會先出現鈴聲迴盪，然後把對方拉入荒塔剪影幻境中擊殺。

;[[黃羽客|黃羽客]]
