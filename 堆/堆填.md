'''堆填'''是一種[[廢物|廢物]]{{link-en|廢棄物處理|Waste treatment|處理方法}}，一般用於處理[[垃圾|垃圾]]、{{link-en|建築廢料|Construction waste}}或[[淤土|淤土]]等{{link-en|都市固體廢棄物|Municipal solid waste|固體廢物}}。用作堆填的場地則稱為'''垃圾堆填区'''，是集中放置垃圾的地方。

== 選址 ==
選址有以下要點：
* 遠離[[民居|民居]]
* 附近沒有過高的山，以免令堆填工作困難
* 石質堅實；若堆填近於海，則應做好防止[[海水倒灌|海水的滲入]]
* 水陸[[交通|交通]]方便
* 有防{{link-en|垃圾滲出水|Leachate|污水滲出}}的功能

部分國家經完善規劃不致影響民居；但在[[最不发达国家|貧窮國家]]，由於資金不足，往往將垃圾隨便棄置。

== 管理 ==
有機物若沒有妥善處理，將自然分解，會釋出{{link-en|掩埋沼氣|Landfill gas}}與{{link-en|垃圾滲出水|Leachate}}。
* 滲出水：要有[[污水|污水]]收集管，舖放{{link-en|防滲層墊層|Landfill liner}}，興建化油污池。
* [[沼氣|沼氣]]：可能引致[[氣爆|氣爆]]、[[火災|火災]]（例如[[美國|美國]]每年有約8300宗火災因此而起<ref>{{cite web |url=http://www.usfa.fema.gov/downloads/pdf/tfrs/v1i18.pdf |title=存档副本 |accessdate=2006-06-03 |0=2004-11-08 |deadurl=yes |archiveurl=https://web.archive.org/web/20041108083302/http://www.usfa.fema.gov/downloads/pdf/tfrs/v1i18.pdf |archivedate=2004-11-08 }}</ref>）
亦有地方善用{{link-en|垃圾場沼氣發電|Landfill gas utilization|沼氣發電}}，例如[[台灣|台灣]][[山水綠生態公園|山豬窟]]垃圾衛生掩埋場。

== 垃圾掩埋場區實作 ==
[[File:Jakarta_slumlife71.JPG|thumb]]]]

[[File:Chureca3.JPG|thumb]]

[[File:Vyskytná_nad_Jihlavou-skládka.JPG|thumb]]

目前被認同的垃圾掩埋場區是以密封式設計，一般會以多層功能密封底和面。以香港其中一個垃圾掩埋場區為例：從底層（原地面）開始，往上有人工排水層、聚乙烯薄片層、膨漲潤土墊、聚乙烯薄片層、不織布層、污水過濾層、土工織網過濾層、底層覆蓋土，然後堆置垃圾、覆蓋土，再重複堆置垃圾、覆蓋土直至滿載，然後其上再加土工織網層、膠膜、排水網層、土工織網過濾層、覆蓋土，最後才是一般地面工程。 

其配備包括： 
*垃圾污水收集管道 
*污水抽取井 
*污水處理廠 
*垃圾掩埋場氣體排氣坑 
*垃圾掩埋場氣體收集管道 
*垃圾掩埋場氣體抽取井 
*垃圾掩埋場氣體處理廠

完成後的垃圾掩埋場區一般可供康樂或公園用途
