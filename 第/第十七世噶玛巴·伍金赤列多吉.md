'''第十七世噶瑪巴·鄔金欽列多傑'''（{{bo|z=རྒྱལ་བ་ཀརྨ་པ་ཨོ་རྒྱན་འཕྲིན་ལས་རྡོ་རྗེ།|w=O-rgyan 'Phrin-las Rdo-rje|t=Ogyen Trinley Dorje}}，{{bd|1985年|6月26日或6月19日||}}），又譯為'''伍金赤列多吉'''或'''烏金聽列多傑'''，生於[[昌都县|昌都县]][[拉多乡_(卡若区)|拉多乡]]，第十七世[[噶玛巴|噶玛巴]][[大寶法王|大寶法王]]，为[[藏傳佛教|藏传佛教]][[噶玛噶举派|噶玛噶举派]]最高宗教領袖，[[楚布寺|楚布寺]][[住持|住持]]。

他得到了[[中国政府|中国政府]]和[[第十四世达赖喇嘛|达赖喇嘛]]的共同认可。（另有一位由[[第十四世夏瑪巴·米龐確吉羅佐|第十四世夏瑪巴]]认定的十七世噶玛巴——[[第十七世噶玛巴·赤列塔耶多吉|赤列塔耶多吉]]，见[[第十七世噶瑪巴爭議|第十七世噶瑪巴爭議]]）

== 转世 ==
1981年，[[噶玛巴·让琼日必多吉|第十六世噶玛巴]]在[[美国|美国]][[芝加哥|芝加哥]]逝世。噶玛巴·鄔金欽列多傑是严格按照第十六世噶玛巴遗嘱及转世法则产生的灵童，得到了[[中国政府|中国政府]]和[[十四世達賴喇嘛|达赖喇嘛]]的共同认可。噶玛巴·鄔金欽列多傑出生于中国[[西藏自治区|西藏自治区]][[昌都县|昌都县]][[拉多乡_(昌都县)|拉多乡]]（Lhatok）巴果（Bakor）。他的父母是牧民，有九个儿女，伍金赤列多吉在三个男孩中排行第二，他幼年在[[拉多乡_(卡若区)|拉多乡]]的[[嘎来寺|嘎来寺]]学习[[藏文|藏文]]和[[佛学|佛学]]基础知识。1992年8月2日，他在[[拉萨|拉萨]][[大昭寺|大昭寺]]剃度 ，得法名为“白恰达让琼乌金杰瓦牛古卓堆赤列多吉则恰列南巴尔杰瓦德”；9月27日，经[[国务院宗教事务局|国务院宗教事务局]]批准，正式在楚布寺[[坐床|坐床]]。第十七世噶玛巴是[[中华人民共和国中央政府|中华人民共和国中央政府]]批准认定的第一位藏传佛教转世活佛。坐床后，他曾多次到中国内地参观，并受到中国国家领导人的接见。<ref>{{Cite web|url=http://news.sina.com.cn/richtalk/news/china/9901/012810.html|title=李瑞环接见十七世噶玛巴活佛|accessdate=2017-10-25|author=|date=1999-01-28|work=|publisher=新浪}}</ref>

== 离藏 ==
1999年12月28日夜，十四岁的他离开楚布寺出走[[印度|印度]]。2000年1月7日，中国国务院国务院新闻办发言人就十七世噶玛巴出走一事发表谈话，指出噶玛巴在留给寺庙的信中说，此行是为了去国外取历世噶玛巴活佛的黑帽和法器，“不是背叛国家和民族，背叛寺庙和领导”<ref>{{Cite web|url=http://news.sina.com.cn/china/2000-1-7/49914.html|title=国务院新闻办发言人就17世噶玛巴出走一事发表谈话|accessdate=2017-10-25|author=|date=2000-01-07|work=|publisher=新浪网}}</ref>。1月13日，中国官方否认另册立噶玛巴活佛。1月21日，中国官方指噶玛巴终会返藏。2001年4月27日他在临时驻地印度[[达兰萨拉|达兰萨拉]]附近的[[上密院|上密院]]会见记者并发表声明：“1999年12月28日，在黑夜籠罩下，我和幾位貼身侍從，離開了我在西藏的寺院出走到印度。離開家鄉、寺院、僧眾、父母、家庭和西藏人民的決定，完全是我個人的決定。沒有人告訴我要離開，也沒有人叫我來此。我離開我的家鄉是為了弘揚佛法，另一方面是為了接受[[噶瑪噶舉派|噶瑪噶舉]]傳承深奧的[[灌頂|灌頂]]、口傳和教授，這些我只能從上一世噶瑪巴的弟子得到，他們是[[第十二世大司徒巴·贝玛东由宁杰旺波|大司徒仁波切]]和[[第十二世嘉察巴·扎巴丹巴雅培|國師嘉察仁波切]]。他們兩位都曾被預言是我的老師，目前都住在印度。”2004年5月30日，[[新华社|新华社]]发了一篇名为“楚布寺众僧俗期盼第十七世噶玛巴早日归寺”的通讯，[[喇嘛|喇嘛]]嘎玛·洛追桑布说道：「大家都猜测他现在暂时无法回到楚布寺是因为受到别人的恫吓」，藉以暗喻法王遲不返回西藏是因為外力的脅迫。

== 性指控 ==

2019年1月24日，一家台湾报纸《镜週刋》刊登了关于噶玛巴有女朋友的指控，女信众<ref>https://www.facebook.com/mandarava.oct</ref>声称噶玛巴和她自己不仅自2013年以来已经有5年的关系，而且还发生过性行为。 《镜週刋》说这不可想象，而且它违反了佛教规则。 <ref>https://www.mirrormedia.mg/story/20190123soc001/</ref><ref>https://www.youtube.com/watch?v=jratcdnfZPE</ref> 女性众提供了Skype录音对话，媒体机构将录音发送给瓦器声纹鉴识实验室，<ref>http://www.voice-forensics.com/</ref>，实验室表示录音是真实的。 <ref>https://m.mirrormedia.mg/story/20190122soc006/</ref><ref>http://www.voice-forensics.com/-announcement/0125</ref>这位女性还透露了她与[[宗薩蔣揚欽哲仁波切|宗薩蔣揚欽哲仁波切]]的谈话，在询问他是否是一名皮条客时，仁波切确认他是皮条客。<ref>https://m.mirrormedia.mg/story/20190122soc009/</ref>。仁波切发表声明并证实女受害人是他的信众，他们也有关于噶玛巴的谈话。<ref>http://blog.sina.cn/dpool/blog/s/blog_483d94f40102yuyt.html</ref>女性受害者说，她不是唯一的受害者，噶玛巴与众多女性信众发生性关系。<ref>https://tw.appledaily.com/new/realtime/20190123/1505599/</ref>
