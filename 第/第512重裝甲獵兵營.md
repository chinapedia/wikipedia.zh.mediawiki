'''德意志國防軍第512重裝甲獵兵營'''（[[德語|德語]]：'''Schwere Panzerjäger-Abteilung 512.'''）是[[第二次世界大戰|第二次世界大戰]]結束前數個月[[納粹德國|納粹德國]]的一支反戰車部隊。它是[[德意志國防軍|德意志國防軍]]內唯二獲得[[獵虎驅逐戰車|獵虎驅逐戰車]]的單位，同時也參與過[[西方战线_(第二次世界大战)|西部戰線的戰鬥]]。

第512重裝甲獵兵營於1945年2月11日在[[奧地利|奧地利]]多勒斯海姆（Döllersheim）倉促成軍，成員多半是[[第501重戰車營|第424重戰車營]]的老兵。<ref name="LDW-512">{{Template:Cite web|title = schwere Panzerjäger-Abteilung 512|url = http://www.lexikon-der-wehrmacht.de/Gliederungen/schwPzJAbt/schwPzJAbt512.htm|work = Lexikon der Wehrmacht|accessdate = 27 April 2013}}</ref>第424重戰車營原先為第501重戰車營，是[[納粹德國|德國]]最早成立的[[重坦克營|重戰車營]]之一，並參與過[[北非戰場|北非]]及[[苏德战争|東部戰線]]的戰事。<ref name="LDW-501">{{Template:Cite web|title = s.Pz.Abt. 501|url = http://www.lexikon-der-wehrmacht.de/Gliederungen/PanzerAbt/PanzerAbt501-R.htm|work = Lexikon der Wehrmacht|accessdate = 27 April 2013}}</ref>第512獵兵營於1945年2月16日接收第一輛[[獵虎驅逐戰車|獵虎驅逐戰車]]；到了3月13日時，該營已經擁有兩個連、共20輛[[獵虎式驅逐戰車|獵虎]]，以及第三個以[[第502重戰車營|第511重戰車營]]成員為基礎編成的連。<ref name="LDW-512"/>[[獵虎驅逐戰車|獵虎驅逐戰車]]是各參戰國在戰爭期間所製造過最重的裝甲戰鬥車輛之一，光底盤就重達72噸；搭載的[[PaK_44反坦克炮|128毫米主炮]]更是威力驚人。然而，它卻非常緩慢，其所使用的引擎原先是設計給57噸重的[[虎I戰車|虎I戰車]]使用的，不過這具引擎光是在[[虎I戰車|虎I戰車]]上都顯得馬力不足，更別提用來推動重達72噸的[[獵虎式驅逐戰車|獵虎]]了。它的產量不多－大約僅製造了88輛－而且僅分配給兩個單位使用：第512與[[第653重裝甲獵兵營|第653重裝甲獵兵營]]。

== 雷根馬戰役 ==
1945年3月21日，第512重裝甲獵兵營被重編至第三軍旗下，並參與了[[雷根馬戰役|雷根馬戰役]]。由於通訊中斷及[[盟軍|盟軍]]的空中攻勢等原因，第512獵兵營花了整整10天才將隸屬於第二連的5輛[[獵虎驅逐戰車|獵虎驅逐戰車]]運送至前線。第一連則損失了七輛[[獵虎式驅逐戰車|獵虎]]，其中四輛是在衛戍後方的作戰中損失，餘下三輛則是因為機械故障而被乘員自行廢棄。當他們終於在赫爾博恩附近遭遇美軍裝甲部隊時，獵虎驅逐戰車發揮了最大功效，並從至少3公里的距離外擊毀了30輛美軍戰車。到3月底時，第512獵兵營僅剩下13輛獵虎驅逐戰車。4月間，第一及第二連於[[魯爾包圍戰|魯爾包圍戰]]中被殲滅，第三連則於[[哈茨山|哈茨山]]區的作戰行動中全軍覆沒。<ref name="LDW-512"/>

第二連的連長是[[奥托·卡利乌斯|奧托·卡利烏斯]]中尉，他同時也是戰爭期間最傑出的戰車指揮官之一。
