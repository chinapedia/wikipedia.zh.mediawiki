'''第9屆立法院'''為當前[[中華民國|中華民國]]最高立法機關[[立法院|立法院]]的屆別，開會地點位於[[臺北市|臺北市]]。其中在2016年1月16日選出的[[2016年中華民國立法委員選舉|立法委員]]，立法委員任期從2016年2月1日至2020年1月31日，涵蓋[[中華民國總統|中華民國總統]][[馬英九|馬英九]]與[[蔡英文|蔡英文]]的任期。這屆立法院席次總計113席，包括73席單一[[選區|選區]]選出的區域立法委員、6席平地和山地[[臺灣原住民族|原住民選舉區]]選出的原住民立法委員，及34席由[[比例代表制|比例代表制]]選出的全國不分區及[[海外華人|僑居]]國外國民立法委員<ref name = "王呈嘉">{{Cite web |url = http://www.setn.com/News.aspx?NewsID=118869 |title = 【開票結果】第14任總統副總統、第9屆立委當選一覽表 |author = 王呈嘉 |date = 2016年1月16日 |publisher = [[三立新聞台|三立新聞台]] |language = zh-hant |accessdate = 2016年4月29日 |quote =  }}</ref>。[[民主進步黨|民主進步黨]]這屆席次自40席增長至68席<ref>{{Cite web |url = http://www.cna.com.tw/news/aipl/201601170016-1.aspx |title = 第9屆立委當選名單 |author =  |date = 2016年1月17日 |publisher = [[中央通訊社|中央通訊社]] |language = zh-hant |accessdate = 2016年4月29日 |quote =  }}</ref>，[[中國國民黨|中國國民黨]]則從上屆64席掉至35席，[[時代力量|時代力量]]和[[親民黨|親民黨]]分別獲得5席和3席，[[無黨團結聯盟|無黨團結聯盟]]和[[無黨籍|無黨籍]]各有1席<ref name = "王呈嘉" />。相對地，[[台灣團結聯盟|台灣團結聯盟]]在這屆立法院則沒有任何席次<ref name = "中央通訊社">{{Cite web |url = http://www.cna.com.tw/news/firstnews/201601165045-1.aspx |title = 立委選舉結果出爐 選後版圖分析總整理 |author =  |date = 2016年1月16日 |publisher = [[中央通訊社|中央通訊社]] |language = zh-hant |accessdate = 2016年4月29日 |quote =  }}</ref>。

這屆立法院有47名新任立法委員就任<ref name = "林倖妃">{{Cite web |url = http://www.cw.com.tw/article/article.action?id=5074011 |title = 國會由藍轉綠　民進黨如何啟動改革？ |author = 林倖妃 |date = 2016年1月19日 |publisher = 《[[天下雜誌|天下雜誌]]》 |language = zh-hant |accessdate = 2016年4月29日 |quote =  }}</ref>，男性和女性分別為71人與42人（男性63%、女性37%），女性立法委員較上屆[[第8屆中華民國立法委員名單|多5個席次]]<ref name = "鄭閔聲">{{Cite web |url = http://www.businesstoday.com.tw/article-content-80392-138216 |title = 最年輕的立法院 新勢力全面啟動改革 |author = 鄭閔聲 |date = 2016年1月21日 |publisher = 《[[今周刊|今周刊]]》 |language = zh-hant |accessdate = 2016年4月29日 |quote =  }}</ref>。由於許多中國國民黨籍資深立法委員競選連任失利，使得這屆立法委員平均當選年齡49.94歲，是2008年[[立委席次減半運動|席次減半]]後最為年輕的一屆<ref name = "鄭閔聲"/>。這屆立法院也是中華民國自1948年行憲選出首屆立法委員以來，民主進步黨首次單獨獲得過半的席次<ref name = "鄭閔聲"/>取得完全執政<ref name = "王呈嘉" />，終結了中國國民黨及泛藍勢力超過一甲子的優勢。新創立的時代力量成為第3大黨<ref name = "林倖妃"/>，和親民黨同樣可依法組成黨團，具提案權和參與黨團協商的權利<ref name = "中央通訊社"/>，這讓立法院繼續維持由「兩大兩小」4個黨團共同協商的狀況<ref name = "鄭閔聲"/>。
