'''第47屆金馬獎'''是2010年[[台灣電影|台灣]]與[[華語電影|華語電影]]業界的年度盛事之一，表揚2010年度傑出華語電影與電影人。[[金馬獎頒獎典禮|頒獎典禮]]於台灣時間11月20日晚上7時在[[桃園市|桃園縣]][[桃園區|桃園市]]的[[桃園展演中心|多功能展演中心]]舉行，由[[蔡康永|蔡康永]]、[[徐熙娣|小S]]擔任主持人。該屆頒獎典禮由[[臺灣電視公司|臺灣電視公司]]執行製播、[[台視主頻|台視頻道]]全程[[現場直播|實況轉播]]，本屆金馬獎也是首次在中國大陸電視台首播。

== 參賽資格 ==

== 舉辦過程 ==
*本屆金馬獎入圍名單於10月1日傍晚揭曉。
*10月18日桃園縣長[[吳志揚|吳志揚]]宣佈下正式啟動。
*本屆金馬影展年度形象廣告由導演[[鈕承澤|鈕承澤]]、[[柯一正|柯一正]]、[[吳念真|吳念真]]、[[侯孝賢|侯孝賢]]與[[林郁智|納豆]]共同擔綱演出。
*本屆主持人[[蔡康永|蔡康永]]和[[徐熙娣|小S]]（打扮成[[納美人|納美人]]），以及星光大道主持人[[林暐恆|阿Ken]]和[[林郁智|納豆]]拍攝的宣傳短片分別自11月6日起播出。
*本屆金馬獎表演嘉賓由[[蕭亞軒|蕭亞軒]]、（[[譚詠麟|譚詠麟]]+[[Mr.|Mr.]]）（演唱多首歷年電影主題曲）共同擔綱演出。
== 評選過程 ==

=== 入選過程 ===
第47屆金馬獎公布入圍名單 ，金馬影展執行委員會執行長聞天祥今天點出今年遺珠，他說，「[[艋舺_(電影)|艋舺」]]男主角趙又廷是新人獎落選第一順位；「[[决战刹马镇|決戰剎馬鎮]]」扮醜的[[林志玲|林志玲]]，差一票入圍。談到今年的金馬遺珠，聞天祥表示，趙又廷在最佳新演員獎中，是落選的第一順位；另外，則是以「決戰剎馬鎮」角逐最佳女主角提名的林志玲。
聞天祥說，林志玲在「決戰剎馬鎮」不僅是造型變化，口條跟肢體都表現優異，可惜片中戲分被大堆頭的演員埋沒，有評審建議將林志玲轉提名最佳女配角，可惜以一票之差落榜。「決戰剎馬鎮」男主角[[孙红雷|孫紅雷]]也在最佳男主角最後一輪失利；此外，主演「[[得閒炒飯|得閒炒飯]]」的港星[[吳君如|吳君如]]也在最佳女主角，止於最後一輪投票。在「最佳新演員」獎人氣旺的[[郭采潔|郭采潔]]，也沒有出現在入圍名單。對此聞天祥表示，郭采潔在新人獎最終投票，得票數排在同樣落選的趙又廷之後。

=== 決選過程 ===
有[[王学圻|王學圻]]參與的表演獎項爭奪，他都會被視為大熱門，金馬影帝頒出之前，儘管他缺席頒獎禮，但仍是最被看好的一位。但在評審過程中，普遍被認為演技好的王學圻在首輪只得兩票陪末座，阮經天完全不是爆冷的角色。並不是說王學圻比其他人演得差，評審更認同的是演員個人的突破，評審團主席黃建業評價王學圻演技爐火純青，但戲分未能盡情發揮，倪大紅因作品風格讓表現受限，[[秦昊|秦昊]]散發出超越外表的微妙吸引力。黃建業認為通過《艋舺》的演出，阮經天證明他不只是一個偶像，“片末被偷襲、剎那間的回眸神情複雜度被詮釋得淋漓盡致，更在容易流於誇張、木訥的情緒中，交出了不落俗套且極有深度的表情，相較于片中同輩演員更令人驚艷。小天贏在他的表現、對角色成熟的理解度。結果阮經天以7票贏過秦昊的4票、倪大紅的4票排在首位。次輪評選，王學圻的兩票無懸念地轉投阮經天，使他得以票過半數而當選影帝。

[[吕丽萍|呂麗萍]]憑《玩酷青春》入圍最佳女主角，封後同樣是當晚一大冷門。《[[唐山大地震_(电影)|唐山大地震]]》徐帆在媒體預測的呼聲最高，不過她跟王學圻一樣很早出局。評審焦點落在呂麗萍與張艾嘉身上，在勢均力敵的態勢下呂麗萍以一票險勝。雖然[[徐帆|徐帆]]在《唐山大地震》中哭到肝腸寸斷，但未能打動金馬評審的心，因而迅速被排除在考慮範圍之外。《[[月滿軒尼詩|月滿軒尼詩]]》的湯唯表現不俗，但也許因為影片是愛情小品的關係，在評審心中未夠分量，二輪後也退出了爭奪。

金馬獎最佳導演頒給《第四張畫》鐘孟宏時，不少人為張作驥導演的《當愛來的時候》在爭奪最佳影片上捏一把汗。電影頒獎禮上最佳電影和最佳導演由不同電影獲得，這種情況並不多見，也可以看成是評審在兩難抉擇下的平衡之舉。陳德森的《十月圍城》瑕瑜互現，在最佳影片評審中一早出局並不令人意外。

郝蕾以《第四張畫》拿下最佳女配角，她在片中戲份雖不多，但作為配角，她出現的每場戲都表現亮眼，獲得評審團的多數人認同。

吳朋奉在《父後七日》裡的演出深入角色精髓，又以通俗的表現方式貼近觀眾，因而獲獎。謝霆鋒在《十月圍城》的表現雖受肯定，但只能嘆對手錶現更優異。

最佳劇情片為《第四張畫》與《當愛來的時候》之爭。評審團從劇本開始仔細審查各環節，評審認為《第》片挑戰敏感話題的勇氣及呈現出來的旺盛生命力十分值得鼓勵；《當》片展現了堅毅及奔放力，同時精準呈現出家族情感與倫理關係的複雜度。最終兩片各獲最佳導演及最佳劇情片，實力差距極小。 《十月圍城》在最佳劇情片投票時，也並非第一輪就出局。 ”

== 頒獎典禮 ==
1.小S除了猛吃男星豆腐，她與曾寶儀在頒獎典禮的「大和解」，為典禮掀起意外高潮。
2.頒獎人林志玲被小S譏諷美腿是修圖結果，索性撩起長裙證明美腿真假，再掀高潮。
3.但公認最感人的橋段是李安、張艾嘉頒「終身成就獎」給製片人徐立功，以及張小燕、陶大偉頒發「特殊貢獻獎」給孫越，無論頒獎引言或得獎致詞都讓人動容。

== 新聞爭議 ==
*金馬獎才結束，爭議陸續出現，先是傳出得獎名單在當晚典禮進行時即流出，接著影帝入圍者大陸演員[[王學圻|王學圻]]的[[經紀人|經紀人]][[穆曉光|穆曉光]]，又因未即時告知大會王學圻沒來、自己卻帶助理住進大會特別安排給王學圻的房間及消費問題而和[[金馬執委會|金馬執委會]]起爭執。
*關於名單外洩一事，傳言有噗友早了數小時在噗浪上公布幾個名單，[[金馬執委會|金馬執委會]]上噗浪和[[PTT|PTT]]查詢，遍尋不著，有可能是噗友Po文後自行刪除或是誤傳。[[金馬執委會|金馬執委會]]對此展開檢討，強調頒獎典禮現場的反應看不出有入圍者或觀禮者先得知結果，而且唯一率先知道名單的只有評審及四大報的記者代表，下屆將對得獎名單有更嚴格的保護。
*歌手[[彭佳慧|彭佳慧]]擔任金馬獎表演嘉賓，演唱[[英文老歌|英文老歌]]，為已故電影人獻唱致意，沒想到整段演出，導播只給了她一秒鐘的畫面，電視轉播只能看到追思逝世電影人的紀念影片。[[彭佳慧|彭佳慧]]事後得知感到相當失落。[[台視|台視]]發言人[[周法勛|周法勛]]坦承11月19日彩排時即得知會有這種狀況，[[金馬執委會|金馬執委會]]表示每位逝世影人的片段只有五秒左右，採排時有要求影片必須完整放映，但如何在保持影片完整性之餘還能顧及表演者的畫面安排，則是製播單位的權責，當時台視總監亦表示交給他們負責即可，執委會也對結果表示遺憾。

== 收視率 ==
*本屆金馬獎頒獎典禮收視率已於11月22日晚間出爐，平均收視率高達3.5%，總收視人口數共有448萬2千人，突破金馬獎近5年新高。
*金馬獎近兩年極力改善「終身成就獎」與「特別貢獻獎」被當作「冷場」的慣例，不僅為得獎者製作影片介紹，獻獎人也特別設計，果然這屆最打動人心的莫過於榮獲「終身成就獎」的[[徐立功|徐立功]]先生，以及「特別貢獻獎」的[[孫越|孫越]]先生，在溫馨的場面下，除了[[徐熙娣|小S]]深受感動，眼眶泛著淚水外，就連在場嘉賓及網友也紛紛感動落淚，此段收視率更是穩定保持在4.56%。
