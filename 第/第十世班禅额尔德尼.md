'''确吉坚赞'''，法名'''罗桑赤列伦珠确吉坚赞'''（{{bo|z=བློ་བཟང་འཕྲིན་ལས་ལྷུན་གྲུབ་ཆོས་ཀྱི་རྒྱལ་མཚན་|w=blo bzang 'phrin las lhun grub chos kyi rgyal mtshan|p=Losang Chinlä Lhünchub Qögyi Gyäcän}}，{{bd|1938年|2月3日|1989年|1月28日|catIdx=Qögyi Gyäcän}}），原俗名'''贡布慈丹'''（又译'''宫保慈丹'''、'''贡布才旦'''，{{bo|z=མགོན་པོ་ཚེ་བརྟན་|w=mgon po tshe brtan|p=Gönbo Cêdän}}），[[中国|中国]][[青海省|青海省]][[循化撒拉族自治县|循化县]]人。[[藏传佛教|藏传佛教]][[格鲁派|格鲁派]]第十世[[班禅额尔德尼|班禅额尔德尼]]。

[[九世班禅|九世班禅]][[圆寂|圆寂]]后，班禅堪布会议厅依据[[乃琼护法|乃琼护法]][[占卜|占卜]]得到的“[[神谕|神谕]]”，于[[藏区|藏区]]东部找到了包括贡布慈丹内的9名灵童。经过长时间生活在九世班禅身边僧侣的选择，确定贡布慈丹为九世班禅转世。由于当时日喀则政府与拉萨政府的竞争关系，堪布会议致电[[中华民国政府|中华民国政府]]，要求依据清末惯例免予[[金瓶掣签|金瓶掣签]]。中華民國政府代总统[[李宗仁|李宗仁]]1949年6月3日于广州签发电令，免予金瓶掣签，承認贡布慈丹为十世班禅额尔德尼。贡布慈丹于8月10日[[坐床|坐床]]典礼后正式继任。<ref>{{cite web | url=http://www.mfa.gov.cn/chn//pds/ziliao/zt/ywzt/wzzt/2108/2136/2146/2151/t8660.htm | title=藏传佛教 | website=[[中华人民共和国外交部|中华人民共和国外交部]]}}</ref>同年9月，[[中国人民解放军|中国人民解放军]]占领[[西宁|西宁]]后，十世班禅主动与[[中华人民共和国中央人民政府_(1949–1954)|中华人民共和国中央人民政府]]联系，表示拥护中央人民政府。此后，十世班禅历任[[全国人大常委会|全国人大常委会]]委员、[[全国政协副主席|全国政协副主席]]、[[西藏自治区筹备委员会|西藏自治区筹备委员会]]副主任等职务。

[[1959年藏区骚乱|1959年藏区骚乱]]中，[[第十四世達賴喇嘛|第十四世達賴喇嘛]]出走，十世班禅选择留在[[西藏|西藏]]，接受了中央人民政府任命的[[西藏自治区筹备委员会|西藏自治区筹备委员会]]代主任的职务。出于对中华人民共和国在藏区实施改革中产生问题的不满，十世班禅于1962年编写了《[[七万言书|七万言书]]》，对[[中華人民共和國治藏歷史|中华人民共和国的西藏政策]]进行了严厉的挞伐。该报告与国防部长[[彭德怀|彭德怀]]《[[八万言书|八万言书]]》一起，被[[中国共产党中央委员会主席|中国共产党中央委员会主席]][[毛泽东|毛泽东]]定性为“[[反动派|反动派]]的疯狂反扑”，导致班禅与[[共產黨|共產黨]]关系恶化。1964年，十世班禅出人意外地于一次万余藏人参与的集会中，表示达赖喇嘛是西藏最高领袖，宣扬西藏独立，此后立即被逮捕，被指控为“反人民、反社会主义、蓄谋叛乱”、“最大的反动[[农奴|农奴]]主之一”被剥夺了一切职务。1966年[[文化大革命|文化大革命]]爆发后，十世班禅被批判为“班禅集团”受到迫害入狱。直到1978年平反，在得到时任[[中国共产党中央委员会副主席|中国共产党中央委员会副主席]][[邓小平|邓小平]]的允许后，十世班禅得以出狱。出狱后，十世班禅被恢复[[党和国家领导人|党和国家领导人]]职务，并在西藏实施了一系列涉及[[藏族|藏族]][[文化|文化]]、[[宗教|宗教]]、[[语言|语言]]、[[习俗|习俗]]的重建与改革，得到了许多[[藏族|藏族]]人的爱戴<ref>{{Cite web|title= “最好的爱国者”：十世班禅大师的悲欢人生 |author= 冯都 |url=http://dangshi.people.com.cn/GB/85038/9338130.html|work=中国共产党新闻网 |date=| accessdate=2018-01-16}}</ref><ref name=yhz>{{cite book|title=《班禅额尔德尼传》 第八章 十世班禅确吉建筑 |author= [[牙含章|牙含章]]|publisher=[[华文出版社|华文出版社]] |page=292-342 |ISBN=9787507509144 |date=2000-1 |accessdate=2012-07-11}}</ref>。

十世班禅与太太[[李洁|李洁]]的女儿叫[[仁吉旺姆|仁吉旺姆]]，班禅去世的时候，她才五岁半。小学毕业后，李洁将仁吉旺姆送到美国上学。此后，仁吉旺姆一直在美国求学。<ref>{{cite web |title=十世班禅妻子和女儿的生活 |url =http://www.huanzhongnet.com/index.php?m=content&c=index&a=show&catid=101&id=6588 |publisher=环中网}}</ref>

1944年在[[拉卜楞寺|拉卜楞寺]][[拉科活佛|拉科活佛]][[拉科·晋美赤烈嘉措|拉科·晋美赤烈嘉措]]（同年成为贡布慈丹的正经师<ref>{{cite web|url= http://people.tibetcul.com/ls/gsdd/200708/6506.html |title=拉科仁波切 |publisher= 藏人文化网 |date=2007-08-04 |accessdate=2012-07-16}}</ref>）主持下，于[[宗喀巴|宗喀巴]]诞生处圣迹前举行仪式，从10名灵童中确认贡布慈丹为第九世班禪额尔德尼的[[轉世灵童|轉世灵童]]，授法名“确吉坚赞”，迎请至[[塔尔寺|塔尔寺]]供养，接受严格的经学教育，潜心研习《釋量論》、《[[现观庄严论|现观庄严论]]》、《[[入中論|入中論]]》、《戒律本论》、《[[俱舍论|俱舍论]]》（合称[[五部大论|五部大论]]）等佛教经典<ref name=yhz/>。

经过诸多过程，最后在1949年6月3日，由处于[[广州|广州]]的[[国民政府|国民政府]]代总统[[李宗仁|李宗仁]]颁布文书:“青海灵童宫保慈丹，慧性澄圆，灵异夙著，查系第九世班禅额尔德尼转世，应即免于[[掣籤|掣籤]]，特准继任为第十世班禅额尔德尼。”8月10日，国民政府专使[[关吉玉|关吉玉]]（时任[[蒙藏委员会|蒙藏委员会]]委员长）和青海省政府主席[[马步芳|马步芳]]的代表[[马继融|马继融]]参加了在[[塔尔寺|塔尔寺]]举行的坐床大典，并颁发了汉、藏文合璧的“西藏班禅行辕堪布会议厅”印鉴。班禅堪布会议厅邀请[[拉卜楞寺|拉卜楞寺]][[拉科活佛|拉科活佛]][[拉科·晋美赤烈嘉措|拉科·晋美赤烈嘉措]]为贡布慈丹剪发，取法名“罗桑赤烈伦珠确吉坚赞”，简称“确吉坚赞”，并受[[沙弥戒|沙弥戒]]<ref name=qhxww>{{Cite web |title=《西海都市报》十世班禅圆寂20周年纪念特刊 |url= http://news.sina.com.cn/o/2009-07-17/154015970980s.shtml |publisher=新浪新闻|work=青海新闻网 |author= |date=2009-07-17 |accessdate=2012-06-17}}</ref><ref name=qhrmzf>{{Cite web|title=十世班禅大师|url=http://www.qh.gov.cn/system/2010/10/20/006069250.shtml|publisher=[[青海省政府网|青海省政府网]]|author=|date=2010-10-20|accessdate=2012-06-17|deadurl=yes|archiveurl=https://archive.is/20121203152841/http://www.qh.gov.cn/system/2010/10/20/006069250.shtml|archivedate=2012-12-03}}</ref><ref>{{Cite web|title=九世班禅转世灵童寻访和认定纪略|url=http://www.studa.net/lishi/060115/15560248-4.html|publisher=中国论文下载中心|author=黄维忠|date=2006-01-15|accessdate=2012-07-16|deadurl=yes|archiveurl=https://web.archive.org/web/20140517225517/http://www.studa.net/lishi/060115/15560248-4.html|archivedate=2014-05-17}}</ref>。8月26日，中国人民解放军占领[[兰州|兰州]]，[[青海|青海]]局势混乱。班禅堪布会议厅带着十世班禅避居六世班禅的驻锡地[[香日德|香日德]]。临行前，十世班禅向[[嘉雅·洛桑丹白坚赞|嘉雅经师]]交代任务，让他在[[塔尔寺|塔尔寺]]挑选一些可靠的喇嘛，到西宁观察[[中国人民解放军|解放军]]的举动，及对待藏民和宗教的态度，在弄清楚后向嘉雅经师汇报。同日，国民政府专使蒙藏委员会向堪布会议厅表示，希望可以在合适的时候接班禅及少数随行人乘飞机前往[[台湾|台湾]]<ref name="fhls">{{cite web |url = http://news.ifeng.com/history/phtv/wdzgx/detail_2011_11/15/10667600_0.shtml |title = 谁被邓小平称为“一个最好的爱国者”？ |publisher=凤凰网 |work=凤凰网历史 |date=2011-11-15 |accessdate=2012-06-25 }}</ref><ref name="yhz" /><ref>{{cite web|url=http://www.qh.xinhuanet.com/misc/2012-03/21/content_24930838.htm|title=十世班禅避居香日德|author=辛光武|publisher=都兰县人民政府|date=2012-03-21|accessdate=2012-07-16|deadurl=yes|archiveurl=https://web.archive.org/web/20140517225924/http://www.qh.xinhuanet.com/misc/2012-03/21/content_24930838.htm|archivedate=2014-05-17}}</ref>。

=== 中国人民解放军进入西藏 ===
1949年9月，[[中国人民解放军|中国人民解放军]]占领[[西宁|西宁]]。9月10日，[[嘉雅·洛桑丹白坚赞|嘉雅活佛]]派出的喇嘛携带解放军在西宁到处张贴的《约法八章》返回，十世班禅与身边人员讨论后决定按照旧有习惯，用[[糌粑|糌粑丸]]打卦，决定与共产党合作<ref name="fhls" />。十世班禅随即派人同[[中国共产党|中国共产党]]进行联系。10月1日[[中华人民共和国|中华人民共和国]]成立，[[班禅堪布会议厅|班禅堪布会议厅]]在[[西寧|西寧]]發電報給中共中央主席、中央人民政府主席[[毛澤東|毛澤東]]、[[中国人民解放军|中国人民解放军]]總司令[[朱德|朱德]]，表示拥护[[中华人民共和国中央人民政府_(1949–1954)|中华人民共和国中央人民政府]]<ref name="qhxww" />。

{{Quote|“北京中央人民政府毛主席、中国人民解放军朱德司令钧鉴：钧座以大智大勇之略，成救国救民之业，义师所至，全国欢腾，班禅世受国恩，备荷优崇。二十余年来，为了西藏领土主权之完整，呼吁奔走，未尝稍懈。第以未获结果，良用疚心。刻下羁留青海，待命返藏。兹幸在钧座领导之下，西北已获解放，中央人民政府成立，凡有血气，同声鼓舞。今后人民之康乐可期，国家之复兴有望。西藏解放，指日可待。班禅谨代表全藏人民，向钧座致崇高无上之敬意，并矢拥护爱戴之忱。”—十世班禅致中华人民共和国中央人民政府电报<ref name="qhxww" /><ref name="clx">{{cite journal zh |author = 陈立旭 |year=1996 |month= |title=毛泽东与十世班禅 |journal=青海民族学院报 |volume= |issue= |pages=44-47 |id= |url= |accessdate=2012-07-11 |quote= }}</ref>}}

1951年春季，[[第十四世达赖喇嘛|第十四世达赖喇嘛]]亲政，[[中华人民共和国中央人民政府_(1949–1954)|中华人民共和国中央人民政府]]和[[噶厦政府|噶厦政府]]商定将于[[北京|北京]]举行谈判。中华人民共和国政府邀请十世班禅进京。4月中旬，十世班禅在前往北京途中于[[西安|西安]]与[[中共中央西北局|中共中央西北局]]第二书记[[习仲勋|习仲勋]]会谈时谈到：“我们是专程去北京向毛主席致敬的，我们把藏族人民对中央人民政府和毛主席的良好祝愿亲自传达给毛主席。”“要坚决拥护中央人民政府的正确领导，决心与西藏各界爱国人士一道，为西藏的解放和藏族人民的团结而努力奋斗。”4月25日，班禅等45人到达北京。当天晚上，政务院总理[[周恩来|周恩来]]为其举办宴会。宴会前，十世班禅与周恩来进行了长时间的交谈。[[班禅|班禅]]回藏是中华人民共和国中央人民政府与西藏[[噶厦|噶厦]]谈判的条件之一。5月1日，毛泽东在天安门城楼上接见了十世班禅<ref name=qhrmzf/>。

[[File:195105_1951年劳动节班禅向毛泽东进哈达.png|thumb]]。]]
5月23日，中华人民共和国中央人民政府与西藏地方政府举行了《关于和平解放西藏办法的协议》的签字仪式。次日下午，毛泽东在[[中南海|中南海]][[怀仁堂|怀仁堂]]与西藏地方政府和谈代表和十世班禅会面。十世班禅向毛泽东献了[[哈达|哈达]]，赠送了锦旗、金盾、长寿铜像、银曼札、藏香及其它贵重礼品。当晚，毛泽东举办了盛大宴会，十世班禅在宴会上讲到：“多少年来没有解决的中国的民族问题—[[西藏问题|西藏问题]]，在毛主席领导下胜利地解决了。和平解放西藏是中国各民族大家庭的一大喜事。中华人民共和国中央人民政府、达赖和班禅三方面的团结，只有在中国共产党和人民政府领导下才能实现。”5月28日，班禅堪布会议厅声明表示“为了西藏民族的彻底解放和发展，为了巩固和发展中国人民的胜利，今后坚决拥护毛主席的领导，拥护中华人民共和国中央人民政府和中国共产党的领导，为正确执行全部协议，为西藏民族与中国各民族的团结和西藏民族内部的团结而奋斗”。6月2日，十世班禅赴[[天津|天津]]、[[上海|上海]]、[[杭州|杭州]]等地参观。6月26日返抵西宁塔尔寺<ref name=qhrmzf/><ref name=qhxww/>。12月19日，十世班禅一行人从塔尔寺启程返回西藏<ref name=mzdybc>{{cite web |url = http://book.163.com/09/1009/16/5L6RFCVF00923INP.html |title = 毛泽东尊重宗教领袖 与班禅促膝长谈 |publisher=网易 |work=网易读书 |date=2009-10-09 |accessdate=2012-07-01 }}</ref>。

1952年4月28日，经过四个多月的艰苦跋涉，十世班禅到达[[拉萨|拉萨]]。当天下午，十世班禅前往[[布达拉宫|布达拉宫]]，在[[日光殿|日光殿]]与十四世达赖喇嘛会面。此次会面是这是自从[[九世班禅|九世班禅]]与[[十三世达赖喇嘛|十三世达赖喇嘛]]失和后，27年以来班禅与达赖第一次相聚。十世班禅在拉萨期间，堪布会议厅与[[噶厦政府|噶厦政府]]讨论并解决了恢复班禅固有地位和职权的问题。6月9日，十世班禅从拉萨启程<ref name=mzdybc/>。6月23日十世班禅回到了位于西藏日喀则的扎什伦布寺<ref name=qhxww/><ref name=yhz/>。

1954年9月，十世班禅同达赖喇嘛一同前往北京，出席了[[第一届全国人民代表大会第一次会议|第一届全国人民代表大会第一次会议]]，27日当选为[[全国人民代表大会常务委员会|全国人民代表大会常务委员会]]委员，12月25日当选为[[全国政协副主席|全国政协副主席]]。在此期间，十世班禅多次与毛泽东、[[刘少奇|刘少奇]]、[[周恩来|周恩来]]等领导人会面<ref name=qhxww/><ref name=ws/><ref name=clx/>。

1955年2月23日[[藏历|藏历]]木羊年正月初一，十四世达赖和十世班禅前往中南海向毛泽东、刘少奇、周恩来等领导人拜年。24日下午，十四世达赖和十世班禅在中南海举行宴会，毛泽东、刘少奇、周恩来等领导人应邀出席。会上，十世班禅致辞“深深地认识到这只有在中国共产党和毛主席领导下和祖国温暖大家庭中，才能如此尊重少数民族的风俗习惯，才能如此关怀备至。这是毛主席伟大民族政策的光辉。”<ref name=mzdybc/>

1955年3月9日，国务院总理[[周恩来|周恩来]]主持召开了第七次[[国务院全体会议|国务院全体会议]]，专门讨论西藏工作。会议通过了《国务院关于成立西藏自治区筹备委员会的决定》，决定由第十四世达赖喇嘛任[[西藏自治区筹备委员会|西藏自治区筹备委员会]]主任委员，第十世班禅额尔德尼任第一副主任委员，[[张国华|张国华]]为第二副主任委员。此外，会议还通过了《国务院对西藏地方政府和班禅堪布会议厅委员会之间关于历史悬案问题的谈判达成的协议的批复》。3月12日，十世班禅结束了对内地的视察访问，离开北京返会西藏，参与西藏自治区的筹备<ref name=ws/>。当天，毛泽东与[[汪锋|汪锋]]、[[计晋美|计晋美]]等到畅观楼与十世班禅交谈，毛泽东对十世班禅说：“每个民族都应有自己的民族领袖，西藏有达赖和班禅这样的领袖是很好的。”对此，十世班禅感到十分惊讶，毛泽东于是重复了一遍，十世班禅于是回答到：“我不是领袖，只有毛主席是各族人民的伟大领袖。”毛泽东再次回应说：“您就是领袖嘛！我看在西藏，不能只喊‘毛主席[[万岁|万岁]]！’、‘朱总司令万岁！’，还要喊‘达赖喇嘛万岁！’、‘班禅喇嘛万岁！’”<ref name=mzdybc/>。离开北京返回西藏途中，十世班禅开始学习[[汉语|汉语]]，三年后已掌握二到三千个[[汉字|汉字]]，可以阅读一般的汉文文件和书报。同时，十世班禅还学习了[[中国历史|中国历史]]、[[中國共產黨歷史|中国共产党党史]]等。在[[佛学|佛学]]方面，在[[恩久·洛桑群培|恩久经师]]的指导下，十世班禅仔细阅读了[[五部大论|五部大论]]、[[彌勒#彌勒五論|慈氏五论]]和[[四续部|四续部]]<ref name=zrwhw/>。

=== 藏区骚乱 ===
{{See also|1959年藏区骚乱}}

从1955年下半年开始，中华人民共和国政府在内地推行[[社会主义改造|社会主义改造]]和[[农业合作化运动|农业合作化运动]]，并在[[金沙江|金沙江]]以东包括川、滇、甘、青的藏区开始实行[[西藏土地改革運動|土地改革]]，随即藏区发生了民變，且规模日益扩大，中共称此为“[[康巴叛乱|康巴叛乱]]”。在对[[四川|四川]]、[[青海|青海]]等地藏区农牧业强行进行社会主义改革时，大量摧毁寺院，强迫僧人还俗，许多普通群众被打成“叛匪”，结果造成人民生活陷入困苦<ref name=ws/>。

1956年4月22日，[[西藏自治区筹备委员会|西藏自治区筹备委员会]]成立，十世班禅任筹委会第一副主任委员。在成立大会上，十世班禅积极主张土地改革，还提出要在自己所属的[[春堆庄园|春堆庄园]]首先进行试点，为“如何改革封建农奴制度，解放农奴和奴隶”摸索经验<ref name=xhs>{{cite web| url=http://cpc.people.com.cn/GB/64093/64094/8723211.html | title=爱国主义的典范 维护祖国统一的楷模—纪念第十世班禅额尔德尼·确吉坚赞圆寂20周年|author= 杜青林 |publisher=新华网|work=人民日报 |date=2009-01-27 |accessdate=2012-06-25}}</ref>，时任国务院副总理[[陈毅|陈毅]]率领中央代表团前往西藏做了大量工作<ref name=mzdybc/>。在土地改革开始后，十世班禅的父亲为顺应时势，主动从拉萨前往日喀则为自己曾是农奴主阶级而向群众道歉，还表示要自我改造。然而，在之后的群众斗争中，十世班禅的父亲依然遭到了殴打<ref name=tz>{{cite book |title=《天葬：西藏的命运》 |author=[[王力雄|王力雄]] |date=1998 |accessdate=2012-07-07 }}</ref>。

11月，应[[印度政府|印度政府]]邀请，十世班禅与达赖喇嘛一同前往[[印度|印度]]参加[[释迦牟尼|释迦牟尼]]涅槃2500周年纪念活动，并在印度各地朝拜和参观访问。在印度期间，获印度[[婆罗奈斯佛教大学|婆罗奈斯佛教大学]]荣誉[[佛学|佛学]]博士学位。当时西藏獨立勢力在印度活躍，周恩来正在印度访问，并与十世班禅和达赖喇嘛进行了会谈，十世班禅表示绝不受外界影响，并强调要遵守《[[十七条协议|十七条协议]]》。1957年1月29日，十世班禅先行回国<ref>{{cite web| url=http://data.book.163.com/book/section/0000FPTJ/0000FPTJ36.html | title=《毛泽东与佛教》第二节 对达赖 耐心帮(5) |author= 王兴国 |work=[[中共党史出版社|中共党史出版社]] |publisher=网易读书|date= |accessdate=2012-07-01}}</ref><ref name=qhxww/><ref name=clx/>。

1958年，十世班禅在西藏日喀则扎什伦布寺辩经大法会上考取了“[[噶钦|噶钦]]”[[学位|学位]]{{NoteTag|1=噶钦（{{bo|z=དཀའ་ཆེན|w=dka' chen}}），是后藏扎什伦布寺的最高学位，与前藏[[拉萨三大寺|拉萨三大寺]]的[[拉让巴格西|拉让巴格西]]学位相当。}}。自1958年起，十世班禅在青海[[塔尔寺|塔尔寺]]和扎什伦布寺开讲《[[菩提道次第广论|菩提道次第广论]]》、《时轮金刚灌顶法》等显密佛教经典，并精心钻研[[宗喀巴|宗喀巴]]的著作<ref name=qhxww/>。

1959年3月10日，西藏拉萨人民发动反抗运动，随后与中国人民解放军发生激战。17日十四世达赖喇嘛率人撤出拉萨。28日，总理周恩来下令将[[西藏|西藏]][[噶厦|噶厦]]政府解散，并由[[西藏自治区筹备委员会|西藏自治区筹备委员会]]行使西藏政府职权，任命十世班禅为筹备委员会代理主任委员。29日，十世班禅向北京发电报，表示支持国务院的决定，并从日喀则赶往拉萨，行使代理主任委员职权。4月1日，达赖喇嘛抵达印度，在印度组织[[西藏流亡政府|西藏流亡政府]]，致力于“[[西藏独立运动|西藏独立运动]]”<ref name=fx/>。

同年4月14日，十世班禅率领西藏代表团抵达北京，准备参加了第二届全国人大第一次会议<ref name=mzdybc/>。当晚，周恩来在[[中南海|中南海]]为其举行宴会。15日，毛泽东召集第十六次扩大的[[最高国务会议|最高国务会议]]，重点谈论西藏问题。十世班禅大师出席会议并在人大全体会议上作了长篇发言，阐述了“西藏在平息叛乱和改革中取得的伟大胜利，严厉谴责一小撮反动分子背叛祖国、妄图阻挡西藏社会进步的罪行。同时，真诚希望达赖喇嘛回到祖国怀抱，共同建设新西藏”<ref name=ws/><ref name=clx/>。6月28日，十世班禅主持召开了[[西藏自治区筹备委员会|西藏自治区筹备委员会]]第二次全体会议，讨论了平叛与改革等政策和措施。会议通过了《关于进行民主改革的决定》。十世班禅在这次会议上提出“宪法进庙”，主张对寺庙实行民主管理<ref name=xhs/><ref>{{cite book|title=《解放西藏史》|url=http://tibet.news.cn/szts/2009-05/21/content_16596220.htm|chapter=第八章 第一节 中央关于西藏民主改革的方针、政策|chapterurl=http://tibet.news.cn/szts/2009-05/20/content_16578449.htm|author=|work=中共党史出版社|publisher=[[新华网|新华网]]|ISBN=9787801999030|date=2008-03|accessdate=2014-05-17|deadurl=yes|archiveurl=https://web.archive.org/web/20111120071657/http://tibet.news.cn/szts/2009-05/21/content_16596220.htm|archivedate=2011-11-20}}</ref><ref name=qhxww/>。8月7日，西藏工委发布了《关于班禅堪厅地区民主改革的指示》，成立“双反委员会”，任命[[刘鹏|刘鹏]]（日喀则专署专员）为主任，[[纳旺金巴|纳旺金巴]]（班禅堪厅副主任）、[[土丹尼玛|土丹尼玛]]为副主任，带领扎什伦布寺3500余名僧人学习并进行“反封建特权、反封建剥削和算政治迫害账、等级压迫账、经济剥削账”的“双反三算”运动。运动进行7个月后，依照班禅的意见，保留1968名僧人并组成了民主管理委员会<ref name=jfxzs84/>。

1959年9月，十世班禅前往北京参加建国十周年庆典，在欢迎会谈话上谴责[[美国|美国]]操纵[[联合国大会|联合国大会]]把西藏问题列入联大议程。但在十世班禅的一生当中，从未斥责或批评过[[丹增嘉措|达赖喇嘛]]<ref name=fx/>，并多次拒绝谴责达赖喇嘛<ref name=nyt1/>。10月至12月，在中共中央的决定下，十世班禅在[[李维汉|李维汉]]、[[陈叔通|陈叔通]]、[[汪锋|汪锋]]等的陪同下，前往[[重庆|重庆]]、[[南昌|南昌]]、[[上海|上海]]等城市参观访问、疗养，并开了一个小型的“神仙会”，会后形成了《李维汉同志和班禅副委员长谈话纪要》<ref name=mzdybc/><ref name=rm1>{{Cite web |title=伟大的爱国主义者班禅大师永垂不朽 |url=http://xz.people.com.cn/GB/138902/8689888.html |publisher=[[人民日报|人民日报]] |author=[[杨静仁|杨静仁]]、汪锋、[[平杰三|平杰三]]、[[阎明复|阎明复]]、[[江平|江平]] |date=1989-02-15 |accessdate=2012-05-08 |deadurl=yes |archiveurl=https://web.archive.org/web/20120708063335/http://xz.people.com.cn/GB/138902/8689888.html |archivedate=2012-07-08 }}</ref>。

1961年1月23日，十世班禅与毛泽东会面，毛泽东表示“反右必出‘左’，日喀则有些群众抄了班禅父母、[[计晋美|计晋美]]和[[拉敏·益喜楚臣|拉敏]]等人的家，这种行为是不对的”，并对十世班禅说：“过去有十一万喇嘛，现在留下几千人专门学经，有近十万人从事生产，这对发展经济和人口有很大好处<ref name=jfxzs84>{{Cite book |title=《解放西藏史》 |url=http://tibet.news.cn/szts/2009-05/21/content_16596220.htm |chapter=第八章·第四节 寺庙的民主改革 |chapterurl=http://tibet.news.cn/szts/2009-05/20/content_16577622.htm |ISBN=9787801999030 |author= |publisher=中共党史出版社 |website=[[新华网|新华网]] |date=2008-03 |accessdate=2014-05-17 |deadurl=yes |archiveurl=https://web.archive.org/web/20111120071657/http://tibet.news.cn/szts/2009-05/21/content_16596220.htm |archivedate=2011-11-20 }}</ref>。”“你们是爱国进步人士，我们从香日德起就合作”，“我们一起搞了这样久，从1959年以来，在平叛、改革等大政方针上，我们都是一致的，始终一致合作”，“我告诉你一个办法，你有什么不满，有什么意见和问题，就向他们讲。听说你有这个作风，很好。你所想的他们不知道不好。工委给我们汇报说他们满意，觉得你讲真话，有时你同他们开个座谈会，把你的牢骚也发出来，双方一讲就通了。”会上，毛泽东指出，西藏现在要纠正“左”的偏向<ref name=mzdybc/>。1月24日，十世班禅与[[周恩来|周恩来]]会面，周恩来对十世班禅说：“你们谈得好。我们相处十一年了，见面什么话都可以谈。”“只要不背叛祖国，不压迫群众，保守一点，慢一点不要紧。今后我们之间多谈心，成为‘随便、自由、不客气’的朋友。”十世班禅虽然仍心有疑虑但仍深受鼓舞<ref name=rm1/><ref name=zgxw/>。同年4月29日，十世班禅主动提出了结束[[班禅堪布会议厅委员会|班禅堪布会议厅委员会]]的报告，并在7月9日得到国务院的同意。随后，十世班禅指示下属将印信文书档案交给西藏自治区委员会<ref name=zgzx>{{Cite journal |title = 第十世班禅额尔德尼·确吉坚赞评传（续） |url = http://www.cqvip.com/read/read.aspx?id=12444565  |journal = 《中国藏学》| pages=23-41|author1 = [[江平|江平]]  |author2=李佐民 |author3=宋盈亭 |author4=辛文波 |author5=廖祖桂|year =1997年 |issue=第2期|accessdate=2018-01-16}}</ref>。

=== 《七万言书》 ===
{{Main|七万言书}}

==== 质疑 ====
1961年9月，十世班禅前往北京参加国庆活动。随后半年，十世班禅曾到[[西藏|西藏]]、[[四川|四川]]、[[青海|青海]]、[[云南|云南]]诸省藏区访问考查，在发现人民公社的问题及执行民族、宗教、统战政策的问题后，质问中国共产党四川干部“共产党的宗旨是全心全意为人民服务，你们为什么不为老百姓说话？为什么不向中央反映真实情况？为什么在人民的苦难面前闭上眼睛？”十世班禅在四川[[甘孜藏族自治州|甘孜藏族自治州]]视察时，州负责人向他汇报甘孜、[[阿坝藏族羌族自治州|阿坝]]两个自治州在平叛、改革后广大劳动人民获得了翻身解放，生产发展了，生活改善了。他随即打断其谈话说：“甘孜、阿坝是开展平叛、改革最早的地区，平叛五六年还平息不下去，你们这里平叛扩大化现象非常严重，匪民不分，把大批劳动人民、基本群众打成‘叛匪’。在改革时，大批寺院被毁坏，喇嘛被强迫还俗……许多老百姓吃不饱肚子，甚至有饿死人的现象。”“有人批评我，老讲民族、宗教问题。我是活佛，是藏族，谈民族问题、宗教问题是很自然的呀。你们在座的都是共产党员吧？你们为什么不为老百姓说话？为什么不向中央反映真实情况？”<ref name=ws/>

==== 劝阻 ====
对此，十世班禅认为自己有必要有责任向中央反映问题，并把此想法告诉了身边的人，但遭到了他们的反对。十世班禅的经师[[恩久·洛桑群培|恩久活佛]]（曾为[[九世班禅|九世班禅]]的[[行辕|行辕]]參謀之一，与九世班禅一起到青海，并主持过寻访九世班禅的[[轉世靈童|轉世靈童]]<ref name=xdfs>{{Cite web|title=心道法师与法幢宗|url=http://www.fxwz.net/lynews/about.asp?type=4|author=理净|publisher=陇原佛学网站|date=2005-08-25|accessdate=2012-05-31|deadurl=yes|archiveurl=https://web.archive.org/web/20120708165717/http://www.fxwz.net/lynews/about.asp?type=4|archivedate=2012-07-08}}</ref>）更极力劝阻，并特意提醒他1957年[[反右运动|反右运动]]和1959年[[庐山会议_(1959年)|庐山会议]]后的反右倾记忆犹新，因此不可轻举妄动。恩久活佛对十世班禅说到：“达赖活佛已经走了，现在只剩下你一个，要是你有不祥，不只是你，也不只是我们[[扎什伦布|扎什伦布]]和[[后藏|后藏]]的事，整个藏区的政教事务都会受到影响，几百万藏民会感到没有依靠。” 但十世班禅认为自己的目的是为了搞好西藏的工作且没有私心并回答到：“只要民族振兴，人民幸福、佛法宏扬，我个人暂受一点冤屈，也无需愧悔”<ref name=ws/><ref>{{cite book|author=降边嘉措 |title=《班禅大师》 }}</ref><ref name=mlh>{{Cite web|title=《七萬言書》|url=http://xizang-zhiye.org/xizang-books/七萬言書5.pdf|author=十世班禪　|publisher=[[西藏流亡政府|西藏流亡政府]]|date=1962-05-18|accessdate=2015-03-26|deadurl=yes|archiveurl=https://web.archive.org/web/20150611092636/http://xizang-zhiye.org/xizang-books/%E4%B8%83%E8%90%AC%E8%A8%80%E6%9B%B85.pdf|archivedate=2015-06-11}}</ref>。于是他力排众议在当年年底，由于[[汉语|汉语]]不够好，开始用[[藏文|藏文]]起草《[[七万言书|七万言书]]》，耗时五个月<ref name=rm1/>。

==== 发布 ====
1962年4月21日至5月29日在北京召开的民族工作会议上，时任[[中共中央统战部|中共中央统战部]]部长[[李维汉|李维汉]]检讨了“平叛扩大化”时所犯的错误，并要求加紧“甄别纠正”，当时的气氛表面上有所解冻。藏传佛学高僧[[中国佛教协会|中国佛教协会]]会长[[喜饶嘉措|喜饶嘉措]]在会上对中国共产党领导进行了尖锐的批判：“我今天要说句真心话，你们有些做法太失人心，[[蒋介石|蒋介石]]、[[马步芳|马步芳]]没有做过的事，你们做了……你们老爱回避实质问题而搞数字游戏⋯⋯我也向您学习，用几个数字，讲讲你们这几年的毛病：一说假话，二不认错，三乱整人，四无佛心，不讲人道……”。对此进行翻译的人员一度不敢进行翻译工作<ref name=tz/>。

5月初，十世班禅基本写完了藏文稿，后请人翻译成[[汉文|汉文]]，以《关于西藏总的情况和具体情况以及西藏为主的[[藏族|藏族]]各地区的甘苦和今后希望要求的报告》为题递交给国务院。在该文中，十世班禅提出“七个认识”，认为中国共产党“平叛”和“民主改革”中出现的错误和问题极其严重：一、对自治权利的认识；二、对中央和地方干部过问西藏问题的认识；三、对是否要消灭藏族问题的认识；四、对是否要消灭宗教问题的认识；五、对叛乱原因的认识；六、对地方错误的认识；七、对国家前途问题的认识<ref name=jl>{{Cite web|title=十世班禅上“七万言书”被批判　险些死于狱中 |url = http://news.163.com/05/0825/17/1S1366V000011247.html |publisher=网易新闻 |date=2005-08-25 | accessdate=2012-05-08}}</ref><ref name=fx/>。在此意见书中，十世班禅认为“西藏的叛乱是具有反党、反祖国、反人民和反民主、反社会主义和反革命的性质，罪恶很大。党采取平叛的政策是非常正确、非常必要的，而且是适时的。”但他也指出“改革后佛教遭受巨大的衰败而濒于灭亡，我们藏人于心不忍。”实行专政的过程中“很多在押犯悲惨地死去了，几年来，藏族人口有很大减少，这对藏族来说，是个很危险的问题。”平叛扩大化时，[[青|青]]、[[甘肃省|甘]]、[[四川|川]]、[[滇|滇]]藏区有的地方“除老幼妇女等不能打仗的外，其余青壮年男子以及通情达理的人，大部分被逮捕关押了。”十世班禅通过列举大量事实，说明了在平叛、改革中出现错误和问题十分严重，还强调：这些问题和错误若不认真加以纠正，藏族将面临灭族灭教的严重危险<ref name=ws/>。

针对西藏实行民主改革中的“[[无产阶级专政|无产阶级专政]]”的场面，十世班禅表示：
{{Quote|在民改斗争的地方刮起了两大风，如果想进行斗争，即如被斗者虽然没有特别严重的过错，也要捏造许多严重的罪行，并予夸大，随心所欲、颠倒是非，不仅毫无根据，一个比一个更尖锐激烈、更粗暴、更狂妄、更矜夸过火地姿意进行诬陷，以致冤枉了许多好人；而且对於那些疯狂的人，反予以奖励表扬，而对真假不加调查等，没有进行应有的掌握，此其狂风之一。……斗争一开始，大喊、怒吼几下，同时拔发揪须，拳打脚踢，拧肉掐肩，推来推去，有些人还用大钥匙和棍棒加以毒打，致使被斗者七窍流血，失去知觉而昏倒，四肢断折等，严重受伤，有的甚至在斗争时当场丧命，此其狂风之二。|《七万言书》<ref name=mlh/>}}

针对民主改革中对待[[宗教|宗教]]的问题，十世班禅表示：
{{Quote|掀起了一个消灭[[佛像|佛像]]、[[佛经|佛经]]、[[佛塔|佛塔]]等的滔天浪潮，把无数佛像、佛经、佛塔烧毁，抛入水中，扔到地上，拆毁和熔化，对寺庙、佛堂、玛尼墙、佛塔姿意进行了疯象闯入般破坏……盗走了许多佛像饰品和佛塔体内的宝贵物品。……而公然无忌地侮辱宗教，把‘大藏经’用於沤肥的原料，专门把许多佛画和经书用於制鞋原料等，毫无任何道理。|《七万言书》<ref name=mlh/>}}

针对[[解放军|解放军]]进入西藏后宣传的旧西藏的黑暗与落後与[[共产党|共产党]]带来的光明等，十世班禅表示:
{{Quote|在西藏一些地方出现了个别人饿死的情况，这是非常不应该，是恶劣的、严重的。过去西藏虽是被黑暗、野蛮的封建统治的社会，但是粮食并没有那样缺，特别是由於佛教传播极广，不论贵贱任何人，都有[[布施|济贫舍施]]的好习惯；讨饭也完全可以维生，不会发生饿死人的情况。我们也从来没有听说过饿死人的情况……勿使众生饥饿，勿使佛教灭亡！勿使我雪域之人灭绝！为祝为祷！|《七万言书》<ref name=mlh/>}}

==== 反响 ====
5月下旬，国务院总理[[周恩来|周恩来]]前往东北前向中央有关领导及中央统战部、中央民委、西藏工委交待，嘱咐与十世班禅协商、谈心。《七万言书》发布后，十世班禅将其送达中央领导和相关部门，引发了高度重视。依照周恩来的指示，李维汉与[[张经武|张经武]]、[[张国华|张国华]]、[[王其梅|王其梅]]、[[夏辅仁|夏辅仁]]、[[阿沛·阿旺晋美|阿沛·阿旺晋美]]、[[帕巴拉·格列朗杰|帕巴拉·格列朗杰]]及[[平杰三|平杰三]]、[[刘春|刘春]]等人一同召开参加了会议，与十世班禅一同研究相关提问题，并商量解决办法。7月19日，会议形成了四个重要文件<ref name=dsbl>{{cite web |url = http://www.tibet3.com/lishi/2008-12/24/content_7274.htm |title=十世班禅与"七万言书" |publisher=中国藏族网通 |work=党史博览 |author=杨明伟 |date=2008-12-24 |accessdate=2012-06-27 |deadurl=yes |archiveurl = https://web.archive.org/web/20140714222305/http://www.tibet3.com/lishi/2008-12/24/content_7274.htm |archivedate=2014-07-14 }}</ref>，分别为：《加强自治区筹委会工作，改进合作共事关系》；《关于继续贯彻执行宗教信仰自由政策的几项规定》；《继续贯彻执行处理反、叛分子规定的意见》；《培养和教育干部的具体办法》<ref>{{cite web|url=http://lz.book.sohu.com/chapter-14892-111867909.html|title=《周恩来生平全纪录》 第一章 殚精竭虑护精华（34）|website=搜狐读书|publisher=中央文献出版社|author=江明武|date=|accessdate=2012-07-01|deadurl=yes|archiveurl=https://web.archive.org/web/20140714202732/http://lz.book.sohu.com/chapter-14892-111867909.html|archivedate=2014-07-14}}</ref>。

周恩来回到北京后再次详细研究了十世班禅的《七万言书》及四个文件。7月24日下午，周恩来与十世班禅再次会面，回答了《七万言书》中提出的问题，在《七万言书》中“问题摆出来了，就必须要解决。但并不等于说你提的问题都对，有对的，有不对的。因为有这两方面的情况，所以要慎重研究。对的就接受过来，不对的正面给你提出意见，提出批评。你有意见可以保留，也是许可的。”<ref name=dsbl/>

8月初，十世班禅回到西藏。8月13日到9月2日，[[谭冠三|谭冠三]]主持召开了西藏工委第六次扩大会议并传达了周恩来与[[李维汉|李维汉]]等对西藏工作的指示，十世班禅对此感到很高兴。但随后形势突变，[[北戴河会议|北戴河会议]]上，毛泽东提出了“千万不要忘记阶级斗争”、“阶级斗争要年年讲、月月讲、天天讲”的号召，并批评了李维汉，认为李维汉“统战部不抓阶级斗争，搞投降主义。”<ref name=ws/>在看到《七万言书》后，毛泽东将此定性为“无产阶级敌人的反攻倒算”<ref name=tz/>。

==== 批判 ====
1962年9月召开的中共八届十中全会期间，国务院副总理[[习仲勋|习仲勋]]被打成“[[习仲勋反党集团|习仲勋反党集团]]”的首要分子。不久，李维汉被免去[[中共中央统战部|中共中央统战部]]部长职务。习李二人的免职直接影响了十世班禅。9月底，国家主席[[刘少奇|刘少奇]]办公厅主任[[张经武|张经武]]飞往[[拉萨|拉萨]]，传达北戴河会议的内容。西藏相关领导认为，李维汉向班禅投降，对班禅“向党猖狂进攻”一味“迁就、妥协”，助长了班禅的“反动气焰”。十世班禅对此非常震惊，并在此之后与西藏工委主要负责人关系日趋紧张，分歧日益加深。工委不断对他“批评教育”，而他的西藏自治区筹委会的工作也被停止。会后，有人批判《七万言书》为“农奴主夺无产阶级专政的权”、“反党反社会主义的反动纲领”，并与[[彭德怀|彭德怀]]在[[庐山会议_(1959年)|庐山会议]]上提出的“[[八万言书|八万言书]]”呼应，“一个在党内，一个在党外，共同向党进攻”。还有人认为十世班禅的意见与[[中印边界问题|中印边界问题]]有关，“具有国际背景的阶级斗争”。对此，十世班禅表示无法忍受，并说：“我这个人看到‘左’的错误，看到不合理现象，看到老百姓吃苦受难，从心底感到气愤，根本坐不住。要我不讲话，或者讲好听的假话，或者闭上眼睛，我根本办不到，这大概叫‘江山易改，本性难移’吧！”<ref name=zgxw/><ref name=ws/>。

1963年全年到1964年初，十世班禅被[[禁錮|禁錮]]在一棟小楼内，十分消沉痛苦。他想見[[毛泽东|毛泽东]]、[[周恩来|周恩来]]但未得到回应。此后，十世班禅开始求助神佛，请占卜师[[卜卦|卜卦]]，向[[佛|佛]]、[[菩薩|菩薩]]祈求[[托梦|托梦]]，并希望神灵可以为其辨明是非<ref name=ws>{{cite journal zh|url=http://news.ifeng.com/history/zhongguoxiandaishi/detail_2013_05/02/24868763_0.shtml |title=十世班禅蒙难始末 |journal=《文史精华》 |author=[[麦群忠|麦群忠]] |issue=第6期 |accessdate=2012-05-30|date=2005年6月}}</ref>。

1964年9月18日至11月4日，在[[西藏自治区筹备委员会|西藏自治区筹备委员会]]第七次扩大会议期间，十世班禅受到严厉批判，被指控“反人民、反社会主义、蓄谋叛乱”。会上，十世班禅据理力争毫不妥协，甚至大吵大闹，拍桌子摔茶杯，但发言被多次打断，为此，十世班禅还获罪“态度恶劣”、“抗拒改造”。会上，“翻身农奴代表”声泪俱下地指控十世班禅残酷压迫剥削农奴，犯有滔天罪行，并批判他是“最大的反动农奴主之一”。随后，会议批判了“班禅叛国集团”，指控十世班禅阴谋“背叛祖国”，搞“西藏独立”，指控原[[班禅堪布会议厅|堪布厅]]的主要成员是“班禅集团”的“黑干将”、“黑高参”。此外，会议还指控了十世班禅在[[日喀则|日喀则]]和拉萨的“班禅集团反党叛国罪行展”，并发动“百万翻身农奴大张旗鼓地揭发批判，愤怒声讨”。在会议期间，十世班禅的家被抄，查出大批卦辞，并被认为是“向党进攻”、“诅咒共产党、诅咒毛主席”的罪证。十世班禅一度成了西藏阶级斗争的主要对象，被认为是西藏各族人民“最危险的敌人”<ref name="zgxw" /><ref name="xmx" /><ref name="ws" />。当月，由[[周恩来|周恩来]]安排，班禅举家搬到北京已故副委员长[[沈钧儒|沈钧儒]]的寓所居住，仍受到副委员长的待遇<ref name="jl" />。期间，十世班禅参加了[[国家民委|国家民委]]办的学习班，还被安排到低压电器厂接受劳动改造<ref name="mgzw" />。12月17日，国务院决定撤销十世班禅的西藏自治区筹委会副主任委员及代理主任委员职务，仅保留委员职务。<ref>{{cite book|title=《中国共产党组织史资料》（附卷一：中华人民共和国政权组织）|publisher=中共党史出版社|year=2000}}</ref>在此后举行的[[第三届全国人大第一次会议|第三届全国人大第一次会议]]和第四届[[全国政协|全国政协]]第一次会议上，十世班禅还被撤销了[[全国人大常委会副委员长|全国人大常委会副委员长]]和[[全国政协副主席|全国政协副主席]]的职务，只保留了[[全国政协常务委员|全国政协常务委员]]一职<ref name="zhw">{{cite news|url= http://news.china.com/history/all/11025807/20120612/17252890_2.html |title=十世班禅：“文革”中没死掉是周总理的恩德 |publisher=中华网 |work=人民网|author= |date=2012-06-12 |accessdate=2012-06-17}}</ref>。原[[班禅堪布会议厅|班禅堪布会议厅]]的主要成员及十世班禅的身边人员，如[[詹东·计晋美|詹东·计晋美]]、[[恩久·洛桑群培|恩久·洛桑群培]]、[[顿吉·索朗多吉|顿吉·索朗多吉]]、[[嘉雅·洛桑丹白坚赞|嘉雅·洛桑丹白坚赞]]等人也在1964年受到批判及处理。<ref>为詹东·计晋美、恩久·洛桑群培、顿吉·索朗多吉平反昭雪　 西藏三位上层爱国人士追悼会在拉萨举行，人民日报1980年7月12日</ref>

=== 文革入狱 ===
[[文化大革命|文化大革命]]爆发后，部分斗争矛头指向了[[丹增嘉措|达赖喇嘛]]和十世班禅。[[周恩来|周恩来]]对红卫兵表示对[[乌兰夫|乌兰夫]]、十世班禅等少数民族领袖人物要加以保护，如果他们有问题可以写材料揭发，也可以背对背批判，但不能揪出批斗。因此，十世班禅在初期生活相当平静<ref name=ws/><ref name=xmx/>。

1966年8月末，[[中央民族学院|中央民族学院]]的一些人强行进入十世班禅的住处，将十世班禅押送到学院，召开了大规模的批斗会并成立了“揪鬥班禅联络站”、“批判班禅指挥部”等组织，争相批斗班禅。其中，批斗罪名包括虐待殴打家中保姆，喜好男色等<ref name=xmx>{{cite web |url = http://www.tangben.com/Himalaya/hm83enlarge.htm |author=徐明旭 |title= 《陰謀與虔誠﹕西藏騷亂的來龍去脈》 |publisher= |date= |accessdate=2012-05-14 }}</ref>。在周恩来的协调下，十世班禅由[[北京卫戍区|北京卫戍区]]转移到[[北京政法学院|北京政法学院]]接受“监护”，与被“北航红旗”揪回的[[彭德怀|彭德怀]]关押在同一地点。 两个多月后，得到周恩来指示，十世班禅被卫戍区送回了寓所<ref name=ws/><ref name=jl/>。

1968年2月，十世班禅被再次带走，并以隔离监护的名义囚禁在[[秦城监狱|秦城监狱]]，与外界失去联系。在狱中，十世班禅受尽折磨<ref name=fx>{{cite web |url=http://www.aaa110.com/showcontract.asp?id=37612&fid=13 |title=十世班禅-确吉坚赞 |publisher=中国佛学网 |date=2007-04-28 |accessdate=2012-05-07 }}{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。在监狱中，十世班禅十分愤怒，经常猛烈地敲打铁门并呼唤：“打开铁门、把我拉出去，交给群众批斗。”“你们为什么不批判我，不提审我，我要见人，我要说话！”有时，十世班禅故意将饭碗打翻，或在放风时故意用脚踹[[警衛|警衛]]，以激怒他们和他们吵架<ref name=zrwhw/>。由于监狱管理员都为[[汉人|汉人]]，十世班禅没有机会使用[[藏语|藏语]]。在狱期间，警卫每天送饭前都会递一张藏文纸条，询问十世班禅想要甚麼餐點。但是，那时十世班禅还不懂[[汉文|汉文]]，因此受到很大的刺激。于是，他给监狱长提意见，要了一本《[[新华字典|新华字典]]》，并将其“从头到尾反复背诵”<ref name=xgblcs>{{cite web|url=http://hk.plm.org.cn/gnews/2009217/2009217106042.html|title=第十世班禅额尔德尼：大师、大爱、大智慧|publisher=香港宝莲禅寺|work=中国民族报|author=田建明|date=|accessdate=2012-06-25|deadurl=yes|archiveurl=https://web.archive.org/web/20140513100506/http://hk.plm.org.cn/gnews/2009217/2009217106042.html|archivedate=2014-05-13}}</ref>。同时，他主动和监狱管理人员谈话，以学习[[普通話|普通話]]，并艰难地阅读了《[[毛語錄|毛語錄]]》、毛詩詞等毛泽东著作和中文报刊，将一本《新华字典》翻烂。出狱后，十世班禅曾说：“这本小字典，是我最好的老师，我要把它作为珍贵文物好好保存。”通过数年狱中的刻苦自学，出狱时，十世班禅已能流利地说汉语，汉文水平与社会知识进步较大<ref name=ws/>。十世班禅的牢房与原[[北京市|北京市]]副市长[[万里|万里]]较近，两人关系因此较为亲密<ref name=mgzw/>。

因受十世班禅1962年《七万言书》一案的牵连，八十岁的[[喜饶嘉措|喜饶嘉措]]在文革中受到批斗被被打断一条腿，并在受尽折磨后于1968年死去<ref>{{cite web|url=http://www.observechina.net/info/artshow.asp?ID=38977 |title=班禅喇嘛文革黑狱十年 |publisher= |author=徐冰 |work=《观察》 |date=2006-4 |accessdate=2012-06-27 }}</ref>。1969年，一度有报道称十世班禅已经在监狱内死亡<ref name=nyt1/>。

在文革期间，西藏本土兴起的[[红卫兵|红卫兵]]狂热地参与了“[[破四旧|破四旧]]”，在文化方面造成巨大破坏，藏民大多篤信佛法，因而對於[[共產黨|共產黨]]深為反感。文革结束后，[[中华人民共和国政府|中华人民共和国政府]]在藏区的政治影响力受到巨大挑战，而重新获得自由的十世班禅则享有了极大的声望。在文革的运动中，一批积极参加文革运动的藏区本土干部被十世班禅视为“投机钻营分子、地痞、流氓无产者”。与此同时，文革时期一度熄灭的宗教热情在藏区恢复。当时，虽然遭到共产党长期严厉批判，当中华人民共和国政府与[[十四世达赖|十四世达赖]]接触达赖代表团回国时，其代表团仍然出人意料地得到了万人空巷的欢迎。进入八十年代后，西藏局势的不稳定随之而来<ref>{{cite web|url=http://www.chinatibetnews.com/yuanzang/2012-02/15/content_882891.htm |title=援藏干部三年轮换 主要任务为招商引资 |publisher=中国西藏新闻网 |author=王辉 |work=援藏网 |date=2012-02-15 |accessdate=2012-07-02 |deadurl=yes |archiveurl=https://web.archive.org/web/20120525193529/http://chinatibetnews.com/yuanzang/2012-02/15/content_882891.htm |archivedate=2012-05-25 }}</ref>。

=== 出狱 ===
==== 普通人的生活 ====
1977年10月，十世班禅获释，但被软禁于北京直至1982年。出狱后，十世班禅过上了普通人的生活，每天他很早起床，念完晨经后外出跑步，从[[北京站|北京站]]一直跑到[[天安门广场|天安门广场]]。还与其他北京人一同作[[體操|體操]]，与见面的陌生人问声好。跑步后，吃过早点看书学习。有时到胡同裏散步，有时自己买菜和日用品<ref name=zrwhw>{{cite web|url= http://people.tibetcul.com/ls/gsdd/200411/1265_3.html |title=十世班禅大师 |author=  |publisher=藏人文化网|date=2004-11-07|accessdate=2012-06-25}}</ref>。与此同时，十世班禅开始对自己的“七万言书”反省，并认为自己在文革中的遭遇并不委屈，并表示：“过去我确实犯了错误，说了一些错话，做了一些错事，今后我准备好好总结一下，再给中央写个检讨报告。”<ref name=jl/>在一次谈话中，十世班禅谈到“过去我被人扶在宝座上，接受众人拜见，就连我自己的父母来见我，都要给我磕头。那时我不知道天高地厚，但是现在我知道了，那样做是错误的。一个不知道天高地厚的人，什么事情也办不成，现在越想越可笑。”<ref name=xgblcs/>

==== 结婚 ====
1978年，十世班禅与其他民主人士组成代表团到外地考察，原69军军长[[董其武|董其武]]是考察团副团长。考察之中，十世班禅和与其年龄相近的董其武的警卫关系较近。于是，十世班禅委托该警卫员给自己介绍结婚对象，得到该警卫员答应。回北京后，该警卫员介绍了董其武的外孙女[[李洁|李洁]]。19岁的李洁出于好奇带着自己的五姨见了班禅，而十世班禅一见倾情直接向其求婚<ref name=fh>{{cite web|url=http://city.ifeng.com/special/lasa/20110513/88544.shtml|title=仁吉旺姆：父亲是我一个永远的目标|author=李菁（[[三联生活周刊|三联生活周刊]]）|publisher=[[凤凰网|凤凰网]]|date=2011-05-13|accessdate=2012-05-10|deadurl=yes|archiveurl=https://web.archive.org/web/20120321103011/http://city.ifeng.com/special/lasa/20110513/88544.shtml|archivedate=2012-03-21}}</ref>。李洁被十世班禅的气质所吸引，愿意放弃[[第四军医大学|第四军医大学]]的学业与十世班禅结婚，但遇到舆论阻力和女方家人的反对，李洁的外祖父董其武也不同意。对此，十世班禅说：“喇嘛结婚并不违反教规。达赖喇嘛的哥哥[[嘉乐顿珠|嘉乐顿珠]]结了婚，达赖的弟弟也结了婚，他们都是活佛，可见一定程度是可以结婚的。”{{NoteTag|1=此說不符合事实，因达赖喇嘛的哥哥嘉乐顿珠并不是活佛。}}为此，他们二人向中共中央发出报告，申请结婚。1978年6月，在得到[[邓小平|邓小平]]批准同意后，二人领取了结婚证书并举行了隆重的结婚典礼。[[李先念|李先念]]、[[张爱萍|张爱萍]]、[[杨得志|杨得志]]、[[阿沛·阿旺晋美|阿沛·阿旺晋美]]、[[赵朴初|赵朴初]]等出席了婚礼。婚后，李洁成为他的妻子兼秘书，并育有一女<ref name=zgxw>{{cite web|url=http://xz.people.com.cn/GB/139893/9341882.html |title=“最好的爱国者”：十世班禅大师的悲欢人生 |author=冯都 |website = 中国共产党新闻网 |date=2009-05-21 |accessdate=2012-05-09 |deadurl=yes |archiveurl=https://web.archive.org/web/20151117021356/http://xz.people.com.cn/GB/139893/9341882.html |archivedate=2015-11-17 }}</ref>。

==== 半平反 ====
1979年，对于[[西藏独立运动|西藏独立运动]]，十世班禅表示：“我维护[[中国统一|祖国统一]]的立场是坚定不移的，对于分裂祖国的行径，我过去反对，现在反对，将来也反对，我愿为维护祖国统一的伟大事业作出最大牺牲。”<ref name=zgxw/>7月2日，十世班禅在政协五届二次会议上被增选为[[全国政协副主席|全国政协副主席]]<ref name=clx/>。1980年9月，当选[[全国人大常委会副委员长|全国人大常委会副委员长]]，但是「反人民、反社會主義、蓄謀叛亂」的三頂帽子並沒有摘掉，算是「半[[平反|平反]]」<ref>{{Cite web |url = http://vot.org/cn/%E3%80%8A%E6%9D%80%E4%BD%9B%E3%80%8B%E4%BD%9C%E8%80%85%E5%AE%89%E4%B9%90%E4%B8%9A%E8%B0%88%E4%B8%AD%E5%85%B1%E5%AF%B9%E4%B8%A4%E4%B8%96%E7%8F%AD%E7%A6%85%E5%96%87%E5%98%9B%E7%9A%84%E8%BF%AB%E5%AE%B3/ |title=《殺佛》作者安樂業談中共對兩世班禪喇嘛的迫害 |publisher=西藏之聲 |date=2015年5月25日 }}</ref>。之后不久，十世班禅前往藏区视察。出发前，[[全国政协主席|全国政协主席]][[邓小平|邓小平]]和十世班禅谈话，称赞他是“我们国家一个最好的爱国者，是拥护国家统一的。” 视察中“你自己的活动，可以放手，不要顾虑。”“你在这次下去的时候可以说，文化革命中那些事情都错了。你不要怕，就说这是中央领导同志讲的。”<ref name=zrwhw/><ref name=fx/>

此后，十世班禅七次去西藏，三次去青海，两次去四川藏区视察。在此期间，十世班禅也曾率领代表团前往的[[澳大利亚|澳大利亚]]、[[新西兰|新西兰]]、[[尼泊尔|尼泊尔]]及[[南美|南美]][[玻利维亚|玻利维亚]]数国进行访问<ref name=zrwhw/><ref name=>{{cite web|url= http://www.tibet.cn/09zcfj/lsyg/201105/t20110520_1046204.htm|title=第十世班禅额尔德尼·确吉坚赞 |author=  |publisher=中国西藏网综合  |date=2011-05-20 |accessdate=2012-06-27}}</ref>。出国访问期间，十世班禅曾收到来信劝其趁出国访问的机会到自由国家避难，十世班禅对此回答到“我和达赖喇嘛是教友，我尊重他，但我们走的路不一样。”<ref name=zcjs/>十世班禅任[[中华人民共和国宪法修改委员会|中华人民共和国宪法修改委员会]]委员，[[援助西藏发展基金会|援助西藏发展基金会]]主任委员，[[中国佛教协会|中国佛教协会]]名誉会长，[[中国藏语系高级佛学院|中国藏语系高级佛学院]]院长。

=== 返藏重建与改革 ===
1982年，离开西藏18年之久的十世班禅回到西藏，受到了数千藏人的欢迎。一周后，他重开了[[扎什伦布寺|扎什伦布寺]]的佛塔，当地三万居民参加了相关仪式，该寺曾存放了五世至九世班禅的遗骨，文革时期被[[红卫兵|红卫兵]]肢解抛弃，后被藏人找回<ref name=nyt1/><ref name=fxbz>{{cite web |url = http://www.beita.org/html/chuanchengcang/jinxiandaidade/200812/05-1522.html |title = 十世班禅大师 |publisher=佛学宝藏 |author=游子 |date=2008-12-05  |accessdate=2012-05-31 }}</ref>。从当年起，十世班禅在以国家领导人和宗教领袖的双重身份下访前往藏区各地访问，在发表谈话中，十世班禅号召信徒群众不要再奉献钱物和布施，认为这些原本只是送给活佛举行象征意义的见面礼，但是由于有人把它误认为是祈福赎罪的代价，并认为奉献的越多就越虔诚，导致了一些群众倾家荡产，为此与教义不符且加重了生活负担，应该予以改革，并表示：“这种对奉献的误解行之已久，积累成习，需要由宗教界自己去改革，看来需要我带个头，从教义的角度向信教群众讲清道理，劝导他们逐步改变这种习惯。”此后，十世班禅在每次访问藏区期间都声明不收奉献和布施。但仍旧有许多信众坚持该习惯。十世班禅有时把收到的现金交予当地政府用于教育文化事业，有时则留给寺庙，或者直接捐给当地的学校和幼儿园。根据不完全统计，1982年到1988年间，十世班禅先后在当地捐款20万元<ref name=zgzx/>。

1983年6月，女儿[[堯西·班·仁吉旺姆|堯西·班·仁吉旺姆]]于北京降生<ref name=mgzw>{{Cite web|title= 在美国长大的十世班禅漂亮女儿 |url = http://www.sinovision.net/portal.php?mod=view&aid=43016|publisher=美国中文网 |date=2008-04-02 | accessdate=2018-01-16}}</ref>。

文革期间，《[[格萨尔|格萨尔]]》被批判为“大毒草”，《格萨尔》的各种抄本、刻本、译本、文物、画卷等在藏区各地遭到焚毁。文革结束后，《格萨尔》得到平反，1980年社科院少文所和民研会为抢救《格萨尔》成立了《格萨尔》工作组。1986年5月，十世班禅与中国《格萨尔》工作组成员会面并留影<ref>{{Cite web|title=《格萨尔》事典  |url= http://iel.cass.cn/news_show.asp?newsid=8405 |publisher=中国社会科学院民族文学研究所|author=德吉 |date=2010-04-19| accessdate=2012-07-12}}</ref>。

当年11月，十世班禅率领全国人大代表团访问[[尼泊尔|尼泊尔]]，向居住在尼泊尔的藏族人讲经摸顶，收到奉献布施4万余盾尼泊尔卢比。访问后，十世班禅将所得大部分捐给了释迦摩尼诞生地的[[藍毗尼园|藍毗尼]]建设委员会<ref name=zgzx/>。12月9日，十世班禅与[[赵朴初|赵朴初]]、[[赵沨|赵沨]]和[[吕骥|吕骥]]等前在出访欧洲前前往[[中央音乐学院|中央音乐学院]]参加了“[[北京佛教音乐团|北京佛教音乐团]]”举办的音乐会<ref name=ess>{{Cite web|title=十世班禅二三事 |url=http://news.163.com/10/0130/12/5U9DFL89000120GR.html  |publisher=网易新闻 |work=北京晚报(北京) |date=2010-01-30 | accessdate=2012-07-01}}</ref>。

1987年3月28日，十世班禅在全国人大小组会上发言时谈到：“1958年我在青海听到党内文件上说‘要挑起叛乱、压出叛乱，然后在平叛过程中，彻底解决宗教和民族问题’”<ref>[http://www.xizang-zhiye.org/gb/arch/writings/banchan/banchan1.html 班禅喇嘛就 "平叛" 等问题在 "人代＂讨论会上的发言] {{webarchive|url=https://web.archive.org/web/20090903134719/http://www.xizang-zhiye.org/gb/arch/writings/banchan/banchan1.html |date=2009-09-03 }}</ref>。7月17日，十世班禅表示：“我们今天在这里欢聚一堂的时候，自然想到达赖喇嘛和国外的藏族同胞。他们远离故土、流落异国，脱离祖国，离别同胞，这是任何的爱国之心、有爱民族之心的人所不堪的。作为同一民族的同胞兄弟，我对他们的这种处境历来十分关切。特别是对达赖喇嘛本人，由于他和我同是佛门弟子，同是历史上形成的黄教教主宗喀巴衣钵的传承人，是很好的教友，我对他更为关切。”<ref name=zcj/>9月1日，十世班禅在北京的[[西黄寺|西黄寺]]主持创办了[[中国藏语系高级佛学院|中国藏语系高级佛学院]]<ref name=fxbz/><ref>{{Cite news|title=一面班禅 |url=http://money.163.com/11/1224/00/7M0GTCO800253B0H.html |publisher=网易财经 |work=中国经营报(北京) |date=2011-12-24 | accessdate=2012-07-01}}</ref>。

1988年2月8日、9日，十世班禅表示1987年10月，警察向围攻警察局的人群开火警告，造成人员死伤，与中国官方宣称没有人员死亡的说法冲突<ref name=nyt2>{{Cite news|title= China's Account of Riot in Tibet Is Disputed |url= http://www.nytimes.com/1988/02/16/world/china-s-account-of-riot-in-tibet-is-disputed.html |publisher=纽约时报 |date=1988-02-16 | accessdate=2012-05-08}}</ref>。同年，十世班禅与阿沛·阿旺晋美在西藏自治区大会上一同倡议并通过了《关于学习、使用、发展藏语文的若干规定》的决议，以推动对整个藏区民族文化、语言、文字的教育<ref name=fxbz/>。4月，在全国人大七届一次会议中外记者招待会上，十世班禅回顾自己的历史时表示：“我在监狱里没有死掉，主要是周恩来先生的恩情。”<ref name=dsbl/>5月，[[全国人大常委会|全国人大常委会]]决定对十世班禅进行平反<ref name=zrwhw/>。

由于文革期间，中国当时仅存的一套官刻《[[乾隆大藏經|乾隆版大藏经]]》的经板7万多块经板中约有1万块遭到毁坏。1987年，[[中国佛教协会|中国佛教协会]]、[[北京市文物事业管理局|北京市文物事业管理局]]、文物出版社决定重印《乾隆版大藏经》。1988年11月24日，十世班禅前往[[人民大会堂|人民大会堂]]西藏厅参加了《乾隆版大藏经》发行仪式，并为重印的[[大藏经|大藏经]]题词：“戒恶行善，克己自律，乃佛法也。弘扬佛学，利益众生，愿一切虔诚的信徒康福长寿。遍行善事乃众生幸福之源。”此外，他还说到：“‘文革’期间有很多珍贵的文物被毁，西藏地区也有许多珍藏的藏传佛教珍贵文物也都被毁掉了。没关系，会有大护法补刻上毁掉的经板。”<ref name=ess/>

=== 圆寂 ===
1989年1月9日，十世班禅乘专机从北京动身前往西藏[[日喀则|日喀则]][[扎什伦布寺|扎什伦布寺]]，主持班禅东陵札什南捷开光典礼（五世至九世班禅大师遗体合葬灵塔祀殿）。全国人大常委会副委员长[[习仲勋|习仲勋]]送行时说：“这个季节西藏缺氧严重，你要注意自己的身体，不要性急，要劳逸结合。”十世班禅说：“开光典礼办完，便遂了我最大的心愿，就是死了也瞑目。”习仲勋说：“[[佛祖|佛祖]]不要你走，[[马克思|马克思]]也不要你走。”陪同班禅一同出行的包括李佐民、任仁等人外，还有班禅的父母、经师及习仲勋送的[[德國牧羊犬|黑背狼犬]]<ref name=fhls/>。

1月10日，十世班禅参拜了[[大昭寺|大昭寺]]朝拜[[释迦牟尼|释迦牟尼]][[佛像|佛像]]，诵经祈祷。1月11日上午，[[中共|中共]][[中央委员|中央委员]]、西藏自治区党委书记[[热地|热地]]，向十世班禅归还60年代从班禅家中没收的档案材料，其中包括了《[[七万言书|七万言书]]》手稿和照片、梦兆录、卦辞等。十世班禅对此说到：“你们看，这意见书，全是我亲手书写，亲自修改的，为了它，我花了不少心血。批判时，说是这个人写的，那个人写的，冤枉了不少人。”<ref name=zrwhw/>

1月14日乘车前往300公里外的日喀则<ref name=zgxw/>。22日上午，十世班禅主持了在扎什伦布寺[[班禅东陵|班禅东陵]][[扎什南捷|扎什南捷]]的开光典礼。参加此次典礼的人还包括[[国务院宗教事务局|国务院宗教事务局]]的领导，西藏、[[四川|四川]]、[[甘肃|甘肃]]、[[青海|青海]]、[[贵州|贵州]]、[[宁夏|宁夏]]等省区领导及当地党政军领导和僧侣代表。会上，十世班禅与中央代表和时任[[中共西藏自治区党委|中共西藏自治区党委]]书记[[胡锦涛|胡锦涛]]先后发表了讲话。开光典礼达5个小时，后宣布：“我的任务完成了！”<ref>{{Cite web|title= 十世班禅大师的最后20天 |url= http://www.people.com.cn/GB/14576/42169/3059213.html |author=王世发 |publisher=人民网 |date=1991-02-20 | accessdate=2012-06-27}}</ref><ref>{{Cite web|title= The Panchen Lama's Last Speech |url= http://www.columbia.edu/itc/ealac/barnett/pdfs/link5-Panchenspeech.pdf |publisher=哥伦比亚大学 |date=1991-02-20 | accessdate=2012-05-08}}</ref>

1月23日，十世班禅给各省代表召开座谈会，听取对西藏工作的意见。十世班禅发表讲话对[[中华人民共和国政府|中华人民共和国政府]]的西藏政策进行了激烈的批评，认为[[中华人民共和国治藏历史|中华人民共和国治理西藏30年]]来付出的代价超过了发展带来的好处，并针对当时对待藏人示威的严厉处置表示有些官员反复犯错<ref name=nyt1>{{Cite news |title= The Panchen Lama Is Dead at 50; Key Figure in China's Tibet Policy |url = http://www.nytimes.com/1989/01/30/obituaries/the-panchen-lama-is-dead-at-50-key-figure-in-china-s-tibet-policy.html?pagewanted=2 |publisher=纽约时报 |date=1989-01-30 | accessdate=2012-05-08}}</ref><ref name="Laird355">{{cite book |title=The Story of Tibet: Conversations with the Dalai Lama|publisher=Grove Press, New York |ISBN=978-0-8021-1827-1 |page = 355 }}</ref><ref name="BBC">{{cite news| title = Panchen Lama Poisoned arrow |url = http://www.bbc.co.uk/dna/h2g2/A644320 | publisher = BBC h2g2 - an encyclopaedic project contributed to by people from all over the world| date = 2001-10-14 | accessdate = 2007-04-29}}</ref>。24日，为宗教界代表召开座谈会，听取对藏传佛教发展的意见。25日与26日，为五万余信众逐一[[灌頂|摩顶]]，因此劳累过度<ref name=zgxw/>。27日晚，宴请日喀则军政领导<ref name=fhls/>。同日，十世班禅讲到：“扎什南捷开光典礼大会，我再加这样一句话就好了：我和达赖喇嘛都是[[宗喀巴|宗喀巴]]弟子。在这个隆重的开光大典上，我很想念我的教友达赖喇嘛。”<ref name=fxbz/>

1月28日凌晨，十世班禅在日喀则在主持历代[[班禅|班禅]][[灵塔|灵塔]]落成仪式中[[心臟病|心臟病]]发作，心电图显示为急性下壁和广泛前壁[[心肌梗塞|心肌梗塞]]<ref name=zcjs/>。时任中共中央总书记[[赵紫阳|赵紫阳]]得知后立即派遣一队心脏病专家乘专机前往日喀则，时任[[中共中央办公厅|中共中央办公厅]]主任[[温家宝|温家宝]]任抢救小组组长。温家宝到达拉萨后，乘[[直升机|直升机]]前往日喀则的班禅新宫[[德庆格桑颇章|德庆格桑颇章]]。抢救进行两个小时后，守候在班禅身边的[[嘉雅·洛桑丹白堅贊|嘉雅活佛]]对救助人员说到：“你们已经尽了最大的努力，现在请把大师交给我们吧，让我们按照藏传佛教的仪轨处理”。随后，[[全国人大常委会|全国人大常委会]]开始编写讣告并准备在十世班禅去世后发布，讣告中称“伟大的爱国主义者，著名的国务活动家，中国共产党的忠诚朋友，中国藏传佛教的杰出领袖，十世班禅额尔德尼确吉坚赞在日喀则新宫[[德庆格桑颇章|德庆格桑颇章]]逝世，享年51岁。”<ref name=fhls/>29日晚20时16分，十世班禅因抢救无效而圆寂<ref>{{cite web|url= http://www.bbc.co.uk/dna/h2g2/A644320 | title= Panchen Lama |author= |publisher=h2g2 |date=2009-03-26 |accessdate=2012-07-07}}</ref><ref name=nyt1/><ref>{{Cite news |title= 十亲历抢救胡耀邦与十世班禅不为人知的细节 |url=http://history.news.163.com/08/0718/08/4H4DS5K700011247.html|author=|work=新华网(北京) |publisher=网易新闻 |date=2008-07-18 | accessdate=2012-07-01}}</ref>。

=== 身后 ===
1989年1月29日，[[温家宝|温家宝]]等人在日喀则参加十世班禅的遗体告别仪式。按照仪轨，十世班禅的遗体经脱水整形处理后永久保存。2月5日，在[[人民大会堂|人民大会堂]]举行了十世班禅的追悼会<ref name="fhls" />。在得到长期保存处理并在扎什伦布寺的[[则甲大厅|则甲大厅]]供奉4年零8个月接受信众瞻仰朝拜后，1993年8月30日，十世班禅的遗体被安放在耗资6406万元修建的黄金皮灵塔祀殿[[释颂南捷|释颂南捷]]。十世班禅成为1970年代以来，除毛泽东以外唯一一位[[遗体|遗体]]得以保存的[[党和国家领导人|党和国家领导人]]<ref>{{cite web |author = |url = http://news.china.com/zh_cn/history/all/11025807/20051220/12962122.html |title = 奉安四年零八个月 十世班禅法体安放揭秘 |date=2005-12-20 |publisher=中华网新闻 |accessdate=2012-07-02}}</ref><ref name="zcjs" />。

1989年十世班禅去世后，时任西藏日喀则[[扎什伦布寺|扎什伦布寺]][[堪布|堪布]][[恰扎·强巴赤列|恰扎·强巴赤列]]被[[中华人民共和国国务院|中华人民共和国国务院]]任命为十世班禅[[转世灵童|转世灵童]]寻访小组组长，主持寻找十世班禅的转世灵童<ref>{{cite news |url = http://politics.dwnews.com/news/2008-06-02/4064524-2.html |title = 谁迫使中国佛教协会副会长出走 |publisher = 多维新闻 |deadurl = yes |archiveurl = https://archive.is/20120719133711/http://politics.dwnews.com/news/2008-06-02/4064524-2.html |archivedate = 2012-07-19 }}</ref><ref name="zcjs" />，对此，国务院拨款600万元，让强巴赤列全权负责。平时，强巴赤列活佛对北京表现得百依百顺，并因此获得[[全国政协|全国政协]]常务委员、西藏自治区政协副主席的地位，以致被部分藏人认为是“汉人的走狗”。在转世灵童寻访期间，小组同时与[[西藏流亡政府|西藏流亡政府]]保持着联系，强巴赤列暗中将寻访灵童的每一步的情况向[[第十四世达赖喇嘛|第十四世达赖喇嘛]]汇报，并听取了达赖喇嘛的指示<ref name="tz" /><ref name="BBC2" />。

1993年8月，强巴赤列托人给达赖喇嘛送信，在信中强巴赤列表示“虽然班禅大师去世时的言行和[[观湖|观湖]]等迹象显示，转世灵童降生在“[[西藏|西藏]]”区域内，但还须达赖打卦，兆示灵童的特征和诞生方位。”同年年底，强巴赤列再次给达赖喇嘛送信通报寻访灵童将在下一年开始。在三次寻访之后，选出了七名候选男童和五名待查男童。[[更登確吉尼瑪|更登確吉尼瑪]]排在五名待查男童之内<ref name="zcj4.5">{{cite web |url = http://www.people.com.cn/item/jplz/banchan/4-5.html |title = 《十一世班禅坐床记》：五　灵童认定受干扰 |author=刘伟 |publisher = [[内蒙古人民出版社|内蒙古人民出版社]] |date=1999-06-01 |accessdate=2012-07-07 }}</ref><ref name="zcjs" />。

1995年，强巴赤列将寻访结果通信告知了十四世达赖喇嘛<ref name="BBC2">{{cite news |url = http://news.bbc.co.uk/chinese/trad/hi/newsid_6590000/newsid_6590400/6590423.stm |publisher=[[BBC|BBC]] |title = 達賴指定班禪轉世靈童年滿18歲 |date=2007-04-25 |accessdate=2009-05-13}}</ref>。5月13日深夜，达赖喇嘛驻[[香港|香港]]代表、达赖喇嘛二哥[[嘉乐顿珠|嘉乐顿珠]]通过电话告知在北京和西藏负责转世灵童工作的人员，第二天达赖喇嘛将正式宣布十世班禅转世灵童的结果。5月14日，达赖喇嘛在[[印度|印度]][[德里|德里]]通过广播宣布生于西藏[[嘉黎县|嘉黎县]]的[[更登確吉尼瑪|更登確吉尼瑪]]为十世班禅的[[转世灵童|转世灵童]]<ref name="zcj">{{cite web |url = http://news.163.com/05/0831/10/1SFRIUSK00011JR5.html |title = 十一世班禅坐床记(五) |author= |work=人民日报-人民网 |publisher=网易 |date=2005-08-31 |accessdate=2012-07-03 }}</ref><ref name="tz" />。当月，强巴赤列被秘密[[逮捕|逮捕]]，并在此后以[[分裂国家罪|分裂国家罪]]和[[故意泄露国家秘密罪|故意泄露国家秘密罪]]被[[西藏自治区日喀则市中级人民法院|西藏自治区日喀则市中级人民法院]]判处[[有期徒刑|有期徒刑]]6年、[[剥夺政治权利|剥夺政治权利]]3年<ref name="zgrq">{{cite web|url=http://www.hrichina.org/cn/content/4533 |title=恰扎．强巴赤列 （Chadrel Rinpoche） |publisher=[[中国人权|中国人权]] |deadurl=yes |archiveurl=https://web.archive.org/web/20120719214611/http://hrichina.org/cn/content/4533 |archivedate=2012-07-19 }}</ref>。强巴赤列对此表示不后悔，并解释到“我是受了[[比丘戒|比丘戒]]，并由[[达赖喇嘛|达赖喇嘛]][[灌顶|灌顶]]的。我必须服从灌顶[[上师|上师]]的意志，否则，是上不了[[天堂|天堂]]的。”<ref name="tz" />同年11月29日，中国政府在[[大昭寺|大昭寺]]经[[金瓶挚签|金瓶挚签]]仪式认定[[确吉杰布|确吉杰布]]是十世班禅的[[转世灵童|转世灵童]]，并于12月决定确吉杰布为第十一世班禅额尔德尼，12月8日在扎什伦布寺举行[[坐床|坐床]]大典<ref name="zcj1.1">{{cite web |url = http://www.people.com.cn/item/jplz/banchan/1-1.html |title = 十一世班禅坐床记：一、1995年11月29日大昭寺的早晨 |author = 刘伟 |publisher=蒙古人民出版社 |date=1999-06-01 |accessdate=2012-07-07 }}</ref><ref name="zcjs">{{cite book |title = 《活佛从圆寂到转生 十一世班禅活佛认定·册立·坐床纪实》 |author=廖东凡、张晓明 |publisher = 华文出版社 |ISBN = 9787507505856 |date=1997-1 |accessdate=2012-07-11 }}</ref>。
