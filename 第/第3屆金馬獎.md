'''第3屆[[金馬獎|金馬獎]]'''，又稱'''五十四年度國語影片展覽及優良國片頒獎典禮'''，是由[[中華民國|中華民國]]官方舉辦的國產電影評選活動，為1965年[[台灣|台灣]]與[[華語電影|華語電影]]業界的年度盛事之一。

== 活動安排 ==
[[行政院新聞局|行政院新聞局]]為了表揚1965年度傑出電影與電影工作者，舉辦第3屆金馬獎。得獎者會受邀參加頒獎典禮，並上台接受金馬獎獎座（或獎狀、獎牌），並且致詞發表感言。優良國語影片展覽於1965年10月27日至10月30日一連四天放映<ref name="聯合報一">本報訊：〈金馬獎頒獎禮 卅日上午舉行〉《聯合報》，1965年10月27日，第8版。</ref>。頒獎典禮於[[台灣時間|台灣時間]]1965年10月30日上午十時在[[台北市中山堂|台北市中山堂]]舉行，由新聞局局長[[沈劍虹|沈劍虹]]主持並致詞，[[行政院院長|行政院院長]][[嚴家淦|嚴家淦]]擔任頒奬人並致詞<ref>本報訊：〈優良國語片 今行頒獎禮〉《聯合報》，1965年10月30日，第8版。</ref>，[[中國小姐|中國小姐]][[劉秀嫚|劉秀嫚]]主持「訪問得獎人」節目<ref name="聯合報一" /><ref>本報訊：〈金獎輝煌 銀星耀目 優良國語片 頒授金馬獎〉《聯合報》，1965年10月31日，第6版。</ref>。第3屆金馬獎影后[[李麗華|李麗華]]未出席，由《[[養鴨人家|養鴨人家]]》稱帝的[[葛香亭|葛香亭]]與[[邵氏兄弟|邵氏兄弟]]花旦[[李菁|李菁]]代表致謝詞<ref name="金馬獎的沿革">{{cite web|url=http://movie.cca.gov.tw/files/15-1000-276,c102-1.php|title=金馬獎的沿革|author=宇業熒|date=2001年11月9日|work=台灣電影筆記|publisher=金馬四十專刊|accessdate=2010年12月10日|language=zh|deadurl=yes|archiveurl=https://archive.is/20091116162659/http://movie.cca.gov.tw/files/15-1000-276,c102-1.php|archivedate=2009年11月16日}}</ref>。新聞局於9月16日發布該年度度優良國語影片及最佳個人技術得獎名單<ref>趙堡：〈中影養鴨人家榮獲 最佳劇情片金馬獎〉《聯合報》，1965年9月17日，第8版。</ref>。

== 獎勵辦法與停辦原因 ==
1964年3月25日，新聞局公佈《五十三年獎勵國語影片辦法》：最佳美術設計由一名增二名，分彩色片最佳設計及黑白片最佳設計；最佳音樂獎也增為二名，分歌劇(舞)片非歌劇(舞)片各一名。新辦法中另一新規定是，凡申請獎勵的影片及個人技衛評審認為確具特殊成就，而其性質類別為辦法中未訂明者，得酌給特別獎。<ref>本報訊：〈獎勵國語影片 獎額獎金增加〉《聯合報》，1964年3月26日，第8版。</ref>因新聞局在台北剛主辦完[[亞洲影展|亞洲影展]]，以及發生[[民航公司106號班機空難|民航空難]]，該年度的金馬獎宣布停辦一次{{notetag|1964年6月20日，[[民航空運公司|民航空運公司]][[民航公司106號班機空難|CT-106號班次]]（出廠編號為B908號）班機因[[劫機|劫機]]事件在[[台中縣|台中縣]][[神岡區|神岡鄉]]上空爆炸，隨後墜毀，機上57人全部死亡，受難者當中有許多當時到台灣參加[[亞洲影展|亞洲影展]]的貴賓，其中包括香港[[國際電影懋業|國際電影懋業]]創辦人[[陸運濤|陸運濤]]夫婦。事發後造成港台兩地極大的震撼，成為金馬獎停辦的原因之一。}}，故於隔年第3屆的金馬獎正式實施該辦法<ref name="金馬獎的沿革" />。

== 養鴨人家與葛香亭==
《[[養鴨人家|養鴨人家]]》是[[中央電影公司|中央電影公司]]首次獲金馬獎最佳劇情片、最佳導演、最佳男主角、最佳彩色攝影等四大獎，一改過去由[[香港|香港]][[邵氏兄弟|邵氏兄弟]]與[[電懋|電懋]]壟斷各獎項的局面，為[[台灣電影史|台灣電影史]]上樹立起一個重要的里程碑，此後由台灣得獎的項目相較過去有明顯地增加。本屆的金馬影帝[[葛香亭|葛香亭]]，又於1970年以《高山青》再度獲得金馬獎最佳男主角獎，2005年獲得金馬獎終身成就獎。
