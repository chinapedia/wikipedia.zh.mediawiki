'''第四次俄土战争'''（1735—1739年）起因於[[俄羅斯帝國|俄羅斯帝國]]與[[土耳其|土耳其]]之間在1733年—1735年[[波蘭王位繼承戰爭|波蘭王位繼承戰爭]]后不斷激化的矛盾以及[[克裡米亞|克裡米亞]][[韃靼人|韃靼人]]對俄國國土無休止的襲擊。這場戰爭同時展現了[[俄羅斯帝國|俄羅斯帝國]]持續向[[黑海|黑海]]地區擴張的決心。

== 戰前的俄國外交政策 ==
1735年戰爭爆發前，俄羅斯帝國與[[波斯|波斯]]在1732年—1735年間簽訂了一系列有利的條約，并退還了所有在俄國—波斯戰爭中所征服的土地；同年俄奧聯軍趕走了法國人支持的[[斯坦尼斯瓦夫一世|斯坦尼斯瓦夫一世]]，扶持[[薩克森|薩克森]][[選帝侯|選帝侯]][[奥古斯特三世_(波兰)|奧古斯特三世]]繼承其父的波蘭王位。自1726年起，[[奧地利|奧地利]]就成為了俄國的盟友。

== 戰爭進程 ==
1735年底克裡米亞韃靼人襲擊[[烏克蘭|烏克蘭]]和[[高加索|高加索]]成為了戰爭的藉口。在1736年，俄國的指揮官們期望能奪取[[亞速|亞速]]和[[克里米亞|克里米亞]]半島。5月20日，由[[陸軍元帥|陸軍元帥]][[伯克哈德克·裏斯托夫·馮·馬尼克|伯克哈德克·裏斯托夫·馮·馬尼克]]率領的[[第聶伯河軍團|第聶伯河軍團]]六萬兩千人迅速攻佔了[[彼列科普|彼列科普]]，并在6月17日佔領[[克里米亞汗國|克里米亞汗國]]首都[[巴赫切薩拉伊|巴赫切薩拉伊]]。但是，由於缺乏補給以及[[流行病|流行病]]的爆發，使得馬尼克不得不率軍撤回[[烏克蘭|烏克蘭]]。6月19日，由[[彼得·彼得羅維奇·拉西|彼得·拉西]]指揮的俄國[[頓河軍團|頓河軍團]]（兩萬八千人）在海軍中將彼得·布烈達（Peter Bredahl）指揮的同區艦隊支援下俄軍攻佔了亞速要塞。1737年7月，馬尼克的軍隊佔領[[奧恰科夫|奧恰科夫]]。[[彼得·彼得羅維奇·拉西|拉西]]將軍的軍團（將近四萬人）在同月進入克裡米亞半島，多次擊敗[[克裡米亞汗國|克裡米亞汗國]]的軍隊并進佔別[[別洛戈爾斯克_(烏克蘭)|卡拉薩巴沙]]（Karasubazar）。儘管如此，拉西和他的軍團依然如上次戰局一樣因缺乏補給而不得不撤退。

1737年7月，奧地利加入戰爭并攻打土耳其，然而卻在諸如[[巴尼亞盧卡|巴尼亞盧卡]]等等戰役中多次敗北。8月，俄國、奧地利與土耳其在[[涅米羅夫|涅米羅夫]]開始了沒有成果的和談。1738年中並無重要的軍事行動，爲了避免瘟疫爆發俄軍只得撤出奧恰科夫和金布恩。

== 最後階段 ==
1739年，馬尼克元帥的軍隊穿越第聶伯河，在Stavuchany擊敗土軍并佔領Khotin（8月19日）和[[雅西|雅西]]。然而，[[神聖羅馬帝國|神聖羅馬帝國]]的軍隊卻在[[格羅茨卡|格羅茨卡]]被土耳其擊敗并于8月21日同土耳其單獨簽訂了[[貝爾格萊德合約|貝爾格萊德合約]]。這件事以及即將到來的瑞典入侵，迫使俄國在9月18日與土耳其簽訂[[尼什合約|尼什合約]]結束戰爭。
