'''第六次俄土战争'''是發生在1787——1792年，[[俄罗斯帝国|俄國]]與[[奥斯曼帝国|土耳其]]之間的一場戰爭。作為[[俄土战争|俄土戰爭]]之一環。在此次戰爭中，[[土耳其|土耳其]]战败，被迫签定《[[雅西和约|雅西和约]]》。割让[[德涅斯特河|德涅斯特河]]以东的广阔领土。

== 背景 ==
1787年5月，俄罗斯的[[叶卡捷琳娜二世|叶卡捷琳娜女皇]]与神圣罗马帝国皇帝[[约瑟夫二世_(神圣罗马帝国)|约瑟夫二世]]巡幸了[[克里米亚|克里米亚半岛]]和[[新俄羅斯|新俄罗斯]]地区。{{sfn|Stone|1994|page=134}}有关叶卡捷琳娜的[[希腊复国计划|希腊复国计划]]（拜占庭复国计划）的流言甚嚣尘上{{sfn|Dowling|2015|page=744}}，君士坦丁堡方面的舆论开始骚动了起来。俄土两国互相抱怨对方不能谨守[[庫楚克開納吉和約|库楚克开纳吉和约]]，而英、法则竭力支持奥斯曼帝国。
== 战争 ==
1787年，奥斯曼帝国要求俄罗斯帝国放弃克里米亚以及[[黑海|黑海]]沿岸的驻防。1787年8月19日，俄方借此向奥斯曼帝国宣战。{{sfn|Cunningham|1993|page=2}}土军先向南乌克兰的金伯恩发起了两波海上攻势，但都被[[亚历山大·瓦西里耶维奇·苏沃洛夫|苏沃洛夫]]将军击退了<ref name="tuker">Tuker.2011.863-964</ref>，这也保障了俄军在克里米亚的优势。<ref name="tuker"/>{{sfn|Dowling|2014|page=841}}随后俄军于[[摩尔达维亚公国|摩尔达维亚]]占领了[[雅西|雅西]]与[[霍京|霍京]]两座城市。1788年12月6日，俄军的[[格里戈里·亚历山德罗维奇·波将金|波将金]]大公与苏沃洛夫在围城六个月后攻克了[[第聂伯河|第聂伯河]]河口的[[奧恰基夫|沃奇亚]]。<ref name="tuker"/>{{sfn|Dowling|2014|page=841}}波将金下令将该城屠城。<ref name="tuker"/>

1789年8月1日，苏沃洛夫于[[福克沙尼战役|福克沙尼战役]]击败了土军，9月22日，俄军又在[[勒姆尼克战役|勒姆尼克战役]]取胜。奥地利则在此时通过防守反击取下了[[贝尔格莱德|贝尔格莱德]]。<ref name="tuker"/>1790年12月，奥斯曼帝国已与奥地利帝国停战，俄军仍继续进攻并拿下了坚固的[[伊斯梅尔_(乌克兰)|伊斯梅尔]]城。苏沃洛夫甚至进军至君士坦丁堡城下以示威吓。{{sfn|Dowling|2014|page=841}}1791年7月9日，土军在马钦的战役中再度败北。{{sfn|Sicker|2001|page=82}}{{sfn|Dowling|2014|page=841}}由于俄方开始担心[[普魯士|普鲁士]]会干预战争，俄土于1791年7月31日签订了的停战协议。{{sfn|Sicker|2001|page=82}}

== 结果 ==
根据1792年1月9日的雅西合约，奥斯曼帝国承认俄罗斯帝国对其于1783年吞并[[克里米亞汗國|克里米亚汗国]]以及[[葉迪山|叶迪山]]地区的主权。俄罗斯欧洲部分的边界推进至第聂伯河。{{sfn|Sicker|2001|page=82}}奥斯曼帝国尝试夺回克里米亚的企图宣告失败。若非法国发生[[法国大革命|大革命]]，其处境或许还会变得更糟。{{sfn|Sicker|2001|page=82}}
