'''德意志國防軍第653重裝甲獵兵營'''（[[德語|德語]]：'''Schwere Panzerjäger-Abteilung 653.'''）是[[第二次世界大戰|第二次世界大戰]]期間[[納粹德國|納粹德國]]的一支反戰車部隊，裝備了[[象式重驅逐戰車|象式驅逐戰車]]以及後來的[[獵虎驅逐戰車|獵虎驅逐戰車]]等重型武器。該單位參與過[[苏德战争|東線]]及[[西方战线_(第二次世界大战)|西線戰事]]，並分別於1943年及1945年參與過[[義大利|義大利]]前線的戰事。

== 東方戰線與義大利戰事 ==
[[File:Bundesarchiv_Bild_101I-313-1004-25,_Italien,_Panzer_"Elefant".jpg|right]]，可能隸屬於第653重裝甲獵兵營。]]
第653重裝甲獵兵營成立於1943年3月31日，前身為第197突擊炮營（[[德語|德語]]：197th Sturmgeschütz Battalion）。<ref name="LDW-653">{{Template:Cite web|title = schwere Panzerjäger-Abteilung 653|url = http://www.lexikon-der-wehrmacht.de/Gliederungen/schwPzJAbt/schwPzJAbt653.htm|work = Lexikon der Wehrmacht|accessdate = 27 April 2013}}</ref>第197[[突擊炮|突擊炮]]營本身則成立於1940年，稍後亦參與過[[入侵南斯拉夫|入侵南斯拉夫]]及[[苏德战争|東方戰線]]的戰事。<ref name="LDW-197">{{Template:Cite web|title = Sturmgeschütz-Abteilung 197|url = http://www.lexikon-der-wehrmacht.de/Gliederungen/Sturmgeschutzeinheiten/Sturmgeschutzabteilung197.htm|work = Lexikon der Wehrmacht|accessdate = 27 April 2013}}</ref>第653營原先僅裝備[[象式重驅逐戰車|斐迪南驅逐戰車]]：<ref name="LDW-653"/>一款以[[VK4501(P)|VK4501(P)]]底盤為基礎的驅逐戰車，配備了威力強大的[[Pak_43反坦克炮|88毫米反戰車炮]]。

第653重裝甲獵兵營隸屬於第41裝甲軍，並於1943年8月參與了[[庫斯克會戰|庫斯克會戰]]，同時也參與了反制[[尼科波尔_(乌克兰)|尼科波爾]]及[[下第聶伯河攻勢|下第聶伯河攻勢]]的戰鬥。在[[烏克蘭|烏克蘭]]遭受大量損失後，第653營被撤往[[維也納|維也納]]進行整補。在那裡，第653營第一連接收了11輛新型的象式驅逐戰車，並於1944年2月被派往[[義大利|義大利]]參與[[鵝卵石行動|安濟奧戰役]]。第二及第三連則於同年4月間接收了30輛戰鬥車輛，並隨後被指派隨第21裝甲軍前往東部戰線。<ref name="LDW-653"/>

到了8月，第二及第三連總計已剩下12輛驅逐戰車；他們隨後被撤往[[克拉科夫|克拉科夫]]，並被重新整編為第二連。該連自此不再隸屬第653營，而是成為第17集團軍的一部份；在被重新命名為'''第614重裝甲獵兵連'''後，該單位被重新派遣至東部戰場上。第614連直到戰爭結束時都待在東部戰線上，並於1945年5月[[柏林戰役|柏林戰役]]結束後有兩輛象式驅逐戰車倖存。<ref>{{Template:Cite book|last = Münch|first = Karlheinz|title = The Combat History Of German Heavy Anti-tank Unit 653 In World War II|year = 2005|publisher = Stackpole Books|isbn = 0811732428|url = http://books.google.co.uk/books?id=eONpj6gjD1UC}}</ref>

== 西線戰事 ==
[[File:Destroyedjagdtiger.jpg|right]]被擊毀於[[法國|法國]][[洛林|洛林]]。]]
第三連則回到西部戰線上與第一連一同作戰；第一連稍早前由於戰損過於嚴重（僅存4輛象式驅逐戰車）而被撤往[[維也納|維也納]]。1944年9月，第一及第三連皆獲得了新型的[[獵虎驅逐戰車|獵虎驅逐戰車]]；<ref name="LDW-653"/>該型驅逐戰車配有威力強大的[[PaK_44反坦克炮|128毫米反戰車炮]]，且僅被發配給兩個單位使用，分別為第653及[[第512重裝甲獵兵營|第512重裝甲獵兵營]]。

在重新整補後，第653營又再次被拆散：第一連被重編至第15軍旗下，並前往北部參與[[守望萊茵河作戰|守望萊茵河作戰]]；第三連則被編至武裝親衛隊第17「[[蓋茲·馮·貝爾力希傑|蓋茲·馮·貝爾力希傑]]」裝甲擲彈兵師（[[德語|德語]]：17. SS-Panzergrenadier-Division „Götz von Berlichingen“）旗下，並於次年1月參與[[北風行動|北風行動]]。2月時，第一及第三連重新於[[兰道|蘭道]]會合，並獲得更多[[獵虎驅逐戰車|獵虎驅逐戰車]]，總數來到41輛。4月，第653營撤退至奧地利，並在那裡獲得更多驅逐戰車。第653營後來被分派給靠近[[林茲|林茲]]的[[厄斯特马克集团军群|厄斯特馬克集團軍]]，直到戰爭結束。<ref name="LDW-653"/>
