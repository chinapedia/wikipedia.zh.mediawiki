{{vgname|第二國度|ja=二ノ国|cn=二之国|tw=第二國度}}是一個由[[日本|日本]][[LEVEL-5|LEVEL-5]]製作發行的[[电子角色扮演游戏|电子角色扮演游戏]]系列。遊戲邀請了[[吉卜力工作室|吉卜力工作室]]協助遊戲製作，音樂由[[久石讓|久石讓]]負責編寫。

== 作品 ==

* '''[[第二國度_漆黑的魔導士|第二國度 漆黑的魔導士]]''' － 在2010年12月發售的[[任天堂DS|任天堂DS]]專用遊戲軟體。遊戲包含了一本實體的書籍，書中提供了稱為「ROON」的紋章圖案，玩者可在DS觸控螢幕上使用觸控筆進行描繪，使用各種不同的魔法力量。
* '''第二國度 Hotroit Stories''' － 以[[行動電話|行動電話]]為發行平台的遊戲。在日本透過[[NTT_DoCoMo|NTT DoCoMo]]的[[i-mode|i-mode]]配信發行。內容是以奧利佛的家鄉汽車工業城Hotroit（ホットロイト）為舞台的角色扮演遊戲。在2010年12月9日推出「第一章～奧利佛和馬克」（第1章〜オリバーとマーク）。
* '''[[第二國度_白色聖灰的女王|第二國度 白色聖灰的女王]]''' － 於2011年11月在日本發售的[[PlayStation_3|PlayStation 3]]專用遊戲軟體。最大的特色是利用即時運算的3D畫面表現出吉卜力工作室的動畫風格，動畫影片則是沿用自《漆黑的魔導士》。
* '''[[第二國度II_王國再臨|二之国II 亡灵王国]]''' － 将于PlayStation 4和Windows上推出的角色扮演游戏。于2015年12月的PSX会议上首次公布。

== 製作 ==

《第二國度》系列是由製作過《[[勇者鬥惡龍IX_星空的守護者|勇者鬥惡龍IX]]》、《[[雷頓教授系列|雷頓教授系列]]》和《[[閃電十一人|閃電十一人系列]]》等遊戲作品的[[LEVEL-5|LEVEL-5]]企劃、製作，公司的社長[[日野晃博|日野晃博]]擔任場景設計、總監督的工作。動畫部分由[[吉卜力工作室|吉卜力工作室]]製作，[[百瀬義行|百瀬義行]]擔任動畫導演。音樂則是由[[久石讓|久石讓]]譜寫，並請來日本歷史最長的[[東京愛樂交響樂團|東京愛樂交響樂團]]（東京フィルハーモニー交響楽団）演奏。

任天堂DS和PS3版本的《第二國度》也起用了多位日本的知名[[演員|演員]]來為遊戲角色配音，包括[[多部未華子|多部未華子]]（主角奧利佛）、[[長澤雅美|長澤雅美]]（瑪露）、[[古田新太|古田新太]]（小滴）、[[大泉洋|大泉洋]]（賈羅）、[[八嶋智人|八嶋智人]]（喵達爾14世）等。PS3版本則較DS版多了[[蘆田愛菜|蘆田愛菜]]、[[比嘉愛未|比嘉愛未]]以及雙人諧星組合「美國螯蝦」加入配音陣容。<ref name="voice">{{cite web |url=http://gnn.gamer.com.tw/4/57004.html|title=《第二國度 白色聖灰的女王》當紅童星蘆田愛菜等新聲優陣容公布望|author= Sam|date=Augest 25, 2011 |publisher=巴哈姆特電玩資訊站 |accessdate=27 Nov 2011}}</ref>另外，溝端淳平也參與配音（在2011年11月的宣傳片上公佈）

遊戲的主題曲是《心のかけら》（心的碎片），由久石讓作曲、[[鈴木麻實子|鈴木麻實子]]作詞，久石讓的女兒[[麻衣|麻衣]]演唱。
