'''削减战略武器条约'''（{{lang|en|'''St'''rategic '''A'''rms '''R'''eduction '''T'''reaty, '''START'''}}）是由[[美国|美国]]和[[苏联|苏联]]之间签署的旨在削减并限制战略性进攻武器的一份条约。由于美国与苏联继任体[[俄罗斯联邦|俄罗斯联邦]]在1993年又签署了[[第二阶段削减战略武器条约|第二阶段条约]]，所以此条约其后也被称为'''第一阶段削减战略武器条约'''。这一条约是历史上最大和最复杂的武器限制条约。在2001年底销毁计划完成时，美俄双方已拆除了当时存在的战略核武器的约80％。
{{核武器}}
== 计划 ==
该削减计划的建议最初是由[[美国总统|美国总统]][[里根|里根]]于1982年6月29日在[[日内瓦|日内瓦]]提出的<ref>[http://www.time.com/time/magazine/article/0,9171,921207,00.html 里根说：到时间开始了 - 时代]{{En}}</ref>。里根在第三阶段[[战略武器限制谈判|战略武器限制谈判]]的基础上，提出了分两个阶段减少战略武器的计划。

===限制条件===
第一阶段将双方所有类型的弹头减少至6000枚（轰炸机及其导弹按一定标准折算），并将洲际弹道导弹弹头数量限制在2500枚以下，洲际导弹运载工具削减至1600件，如[[SS-18|SS-18]]之类的重型武器则限制在110件以下。

===武器换算===
*只可携带炸弹或短程攻击导弹（SRAM）的重型轰炸机按一个弹头计算
*美国前150架重型轰炸机可仅计算为10枚ALCM，前提是每架轰炸机最多仅可携带20发远程空中发射巡航导弹（ALCM）
*相应地，苏联前180架重型轰炸机可仅计算为8枚ALCM，前提为每架轰炸机最多仅可携带16发远程空中发射巡航导弹
*可从导弹上“卸载”下来不超过1250枚的弹头，这些弹头并不按现存多弹头弹道导弹计数

== 签署 ==
条约1991年7月31日由[[美国总统|美国总统]][[老布什|老布什]]与[[苏联总统|苏联总统]][[戈尔巴乔夫|戈尔巴乔夫]]于[[莫斯科|莫斯科]]签署，有效期15年。但五个月后，苏联解体，条约的生效随之推迟。其后在1992年5月23日美国与独立后的[[俄罗斯|俄罗斯]]、[[白俄罗斯|白俄罗斯]]、[[哈萨克斯坦|哈萨克斯坦]]和[[乌克兰|乌克兰]]四个武器继承国家在[[里斯本|里斯本]]签署了附加条约“里斯本议定书”。后三国其后将其核军备送至俄罗斯处理。在1994年12月5日，五个缔约国在布达佩斯交换批准书协议后条约正式生效。

== 销毁及效果 ==
由于协议通过在运输工具数量上加以限制了弹头的部署。依据条约内容，美俄双方的众多导弹以及飞机被拆除。

武器的销毁核查采用了一套严格的核查制度，如现场检查，定期交流信息，以及包括[[人造卫星|人造卫星]]在内的国家技术手段。

美国曾将解体后的轰炸机放在原地长达三个月之久，即为了使俄罗斯卫星得到足够信息<ref>[http://www.cnn.com/SPECIALS/cold.war/experience/the.bomb/route/06.amarc/ CNN. Special: COLD WAR. ''Uncle Sam's salvage yard: A Cold War icon heads for the scrap heap'' By Andy Walton, CNN Interactive] {{webarchive|url=https://web.archive.org/web/20080323112510/http://www.cnn.com/SPECIALS/cold.war/experience/the.bomb/route/06.amarc/ |date=2008-03-23 }}</ref>。

美国和俄罗斯目前的战略核弹头部署数量分别为3696和4237，包括战略导弹和战术导弹在内的弹头数量则分别为10000左右和15000至17000左右<ref>[http://www.armscontrol.org/factsheets/ Arms Control Association: Fact Sheets]</ref>。美国的[[B-52|B-52]]轰炸机主要被送往位于[[戴维斯·蒙山空军基地|戴维斯·蒙山空军基地]]的{{tsl|en|Aerospace Maintenance and Regeneration Center|航天维修与改造中心}}进行拆除。

== 到期与延期 ==
根据1997年3月在赫尔辛基首脑会议上作出的承诺，双方原则上同意谈判一项协议，使裁武条约的持续时间无限。但严格依据条约内容，该条约于2009年12月5日到期。俄美双方曾在2009年协商延期条约，俄罗斯外长[[拉夫罗夫|拉夫罗夫]]称已做好了削减核武器的准备，并试图着手与美国举行会谈，[[美国国务卿|美国国务卿]][[希拉里·克林顿|希拉里]]也表示，将尽快重启削减战略核武器条约谈判<ref>[http://www.chinanews.com.cn/gj/oz/news/2009/02-08/1554136.shtml 俄外长称俄罗斯已做好削减核武器数量准备——中新网]</ref>。如今这一条约已在2010年被全新的裁减军备协议——[[新削减战略武器条约|新削减战略武器条约]]所取代。
