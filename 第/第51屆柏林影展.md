'''第51屆[[柏林影展|柏林影展]]'''（{{lang-de|'''''Internationale Filmfestspiele Berlin 2001'''''}}）於2001年2月7日至2月18日於[[德國|德國]][[柏林|柏林]]舉辦<ref name="Berlinale 2000">{{cite web |url=http://www.berlinale.de/en/archiv/jahresarchive/2001/01_jahresblatt_2001/01_Jahresblatt_2001.html|title=51st Berlin International Film Festival |accessdate=2012-10-21 |work=berlinale.de}}</ref>，開幕片為法國導演[[让-雅克·阿诺|尚-賈克·阿諾]]執導的《[[大敵當前|大敵當前]]》<ref>{{cite web |url=http://www.screendaily.com/berlin-confirms-enemy-at-the-gates-as-opener/404665.article |title=Berlin confirms Enemy At The Gates as opener |accessdate=1 September 2014 |work=screendaily.com |archiveurl=https://web.archive.org/web/20140902093539/http://www.screendaily.com/berlin-confirms-enemy-at-the-gates-as-opener/404665.article |archivedate=2014年9月2日 |deadurl=no }}</ref><ref>{{cite web |url=http://abcnews.go.com/Entertainment/story?id=109818 |title=Enemy at the Gates Opens Berlin Film Festival |accessdate=1 September 2014 |work=abcnews.go.com |archiveurl=https://web.archive.org/web/20140502002002/http://abcnews.go.com/Entertainment/story?id=109818 |archivedate=2014年5月2日 |deadurl=no }}</ref>，閉幕片則是美國導演[[史丹利·庫柏力克|史丹利·庫柏力克]]於1968年的科幻經典《[[2001太空漫遊_(電影)|2001太空漫遊]]》70 mm修復版本。<ref>{{cite web |url=http://www.nitrateonline.com/2001/fberlin01.html |title=Berlin Gets Intimate |accessdate=1 September 2014 |work=nitrateonline.com |archiveurl=https://web.archive.org/web/20080720134153/http://www.nitrateonline.com/2001/fberlin01.html |archivedate=2008年7月20日 |deadurl=no }}</ref><ref>{{cite web |url=http://www.backstage.com/news/screen-version-of-wit-to-premeire-at-berlin-film-festival/ |title=Screen Version of 'Wit' to Premeire at Berlin Film Festival |accessdate=1 September 2014 |work=backstage.com |archiveurl=https://web.archive.org/web/20140902092901/http://www.backstage.com/news/screen-version-of-wit-to-premeire-at-berlin-film-festival/ |archivedate=2014年9月2日 |deadurl=no }}</ref>

本屆最高榮譽[[金熊獎|金熊獎]]頒給了法國導演[[巴提斯·謝侯|巴提斯·謝侯]]執導的英-法合作片《{{link-en|親密關係 (2001年電影)|Intimacy (film)|親密關係}}》。<ref name="PRIZES">{{cite web |url=http://www.berlinale.de/en/archiv/jahresarchive/2001/03_preistr_ger_2001/03_Preistraeger_2001.html |title=PRIZES & HONOURS 2001 |accessdate=9 June 2014 |work=berlinale.de |archiveurl=https://web.archive.org/web/20131015074930/http://www.berlinale.de/en/archiv/jahresarchive/2001/03_preistr_ger_2001/03_Preistraeger_2001.html |archivedate=2013年10月15日 |deadurl=no }}</ref>
