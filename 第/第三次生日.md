{{Vgname|第三次生日|ja=ザ・サード バースデイ|en=The 3rd Birthday}}是[[史克威尔艾尼克斯|史克威尔艾尼克斯]]和HexaDrive开发，史克威尔艾尼克斯发行的第三人称[[動作角色扮演遊戲|動作角色扮演]]射击游戏<ref name="IGN" /><ref name="Joystiq">{{cite web|title=The 3rd Birthday review: New year's Eve|author=David Wolinsky|date=April 7, 2011|publisher=[[Joystiq|Joystiq]]|url=http://www.joystiq.com/2011/04/07/the-3rd-birthday-review/|accessdate=2011-04-09}}</ref>。游戏对应[[PlayStation_Portable|PlayStation Portable]]平台，是[[寄生前夜系列|寄生前夜系列]]的第三部游戏。游戏于2010年12月22日在日本发行<ref name="Release2010">{{cite web |url=http://www.vg247.com/2010/08/25/kitase-confirms-winter-2010-japanese-release-for-3rd-birthday-2011-for-europe/ |title=Kitase confirms winter 2010 Japanese release for 3rd Birthday, 2011 for Europe |author=Johnny Cullen |date=August 25, 2010 |work= |publisher=VG247 |accessdate=August 24, 2010}}</ref>，2011年3月29日在北美发行<ref name="Q12010">{{cite web |url=http://www.vg247.com/2010/10/20/the-3rd-birthday-slated-for-q1-2011-in-west/ |title=The 3rd Birthday slated for Q1 2011 in west |author=Johnny Cullen |date=October 20, 2010 |work= |publisher=VG247 |accessdate=October 20, 2010}}</ref>，2011年4月1日在欧洲发行<ref name="April 1, 2011">{{cite web |url=http://www.rpgsite.net/news/1002/the-3rd-birthday-dated-for-europe-english-language-trailer-released.html/ |title=European Release Date for The 3rd Birthday |deadurl=yes |archiveurl=https://web.archive.org/web/20110227213619/http://www.rpgsite.net/news/1002/the-3rd-birthday-dated-for-europe-english-language-trailer-released.html |archivedate=2011-02-27 }}</ref>。

游戏设定于[[纽约城|纽约城]][[曼哈顿|曼哈顿]]，情节于《[[寄生前夜II|寄生前夜II]]》十年之后。玩家扮演[[阿雅·布雷亚|阿雅·布雷亚]]，和威胁城市的神秘的“扭曲者”生物战斗。

== 玩法 ==

史克威尔艾尼克斯将游戏称为“电影[[动作角色扮演游戏|动作RPG]]”<ref name="square-enix.co.jp">{{cite web|url=http://www.square-enix.co.jp/3rd_birthday/ |title=The 3rd Birthday |publisher=Square Enix |date= |accessdate=2011-03-12}}</ref>。总监田畑端称，游戏性和剧情因新创能力“增驱(Over Dive)”缠绕在一起，这一能力可让主角的精神进入其他人，如切换到队伍任何一员的身体，或者控制友好的[[非玩家角色|非玩家角色]]（NPC）。可以命令队员攻击特定敌人或在战斗中自我防守。玩家可以即时改变所控角色及其战斗行动<ref name="Tabata">{{cite web | author=IGN staff | date=August 20, 2008 | title=Parasite Eve Sequel Detailed | url=http://psp.ign.com/articles/900/900565p1.html | work=IGN.com | accessdate=2009-02-02}}</ref>。游戏采用《[[生化危机4|生化危机4]]》、《[[死亡空间|死亡空间]]》和[[质量效应系列|质量效应系列]]那样的越肩视角。阿雅在游戏中可以选择不同的服装搭配，但这必须先完成游戏各种困难模式解锁。在战斗中衣服会被划破，阿雅的皮肤会不同程度的露出来。
而本次作品將粒線體用在技能呈現上，稱之為DNA技能樹、以3X3表格來放置戰鬥中所獲得的DNA能力球，並依照排列會出現不同變化可能會造成變異增加或減少能力。

== 情节 ==

[[粒線體|粒線體]]的變異開端起源於1997年12月的[[曼哈頓島|曼哈頓島]]封鎖事件、2000年9月的N.M.C.（Neo-Mitochondria-Creature/新粒線體創造物）事件後Aya・Brea的活躍事跡告一段落。
在這之後10年的時光流逝2010年12月24日、在紐約被發現新品種異形的存在。由外形來看似扭曲的被稱呼為扭曲者(Twisted)以捕食人類為生、並且擁有爆炸性的繁殖能力。變為破滅狀態的紐約出現了扭曲者的巢穴[[巴別塔|巴別塔]]「Babel」。美利堅合眾國(USA)組織了對抗扭曲者的反扭曲團隊（Counter Twisted Investigation/簡稱CTI）。CTI是開發透過改變過去、改變現在的稱為「Over Dive・System」的新裝置、藉此來對抗扭曲者。Aya在失去過往記憶之後受到CTI組織保護、她是唯一的OVERDIVE系統適用者。
惡夢的Babel出現已經過了3年2013年12月24日。Aya・Brea的OVER DIVE對抗扭曲者的戰鬥即將展開。

== 开发 ==

期限宣布中从称游戏为手机作品。但在2008年8月的非公开聚会DKΣ3713中，宣布称《第三次生日》将改为[[PlayStation_Portable|PlayStation Portable]]作品，<ref>{{cite web|url=http://m.kotaku.com/5032271/final-fantasy-agito-xiii-parasite-eve-sequel-are-psp-bound |title=PSP announcement |publisher=M.kotaku.com |date=2008-08-01 |accessdate=2011-03-12}}</ref>，没有提到先前所称的手机版。在[[东京电玩展|东京电玩展]]宣传资料袋上，PlayStation Portable独占的消息的得到确认。

剧本总监[[鸟山求|鸟山求]]构思了雪天纽约设定、血迹斑斑的婚纱礼服景象等故事概念。开发过程中他为面向成人数次调整剧情，并加入了曲折的叙事<ref>{{cite web |author=Toriyama, Motomu |date=2010-11-22 |url=http://blog.jp.square-enix.com/3rd_birthday/2010/11/-vol3.html |title=クリエイターズ メッセージ vol.3 |publisher=[[Square_Enix|Square Enix]] |accessdate=2013-06-01}}</ref>。游戏剧本由动画剧本师[[竹内利光|竹内利光]]编写，他曾参与过连续剧《[[Keroro軍曹|Keroro軍曹]]》、《[[工作狂|工作狂]]》和[[原创动画录影带|原创动画录影带]]《[[网球王子|网球王子]]》的创作<ref>{{cite web |date=2008-12-15 |url=http://www.gametrailers.com/videos/g7wadg/the-3rd-birthday-dks3713-trailer |title=The 3rd Birthday Video - DKS3713 Trailer |publisher=[[Square_Enix|Square Enix]] |accessdate=2013-06-08}}</ref>。在6月5日《[[Fami通|Fami通]]》一篇野村哲野的采访中，他称游戏头几个阶段已经完成，已经可玩<ref>{{cite web |url=http://www.psphyper.com/psp/square-birth-by-sleep-3rd-birthday-development-status/#comments |title=Square: Birth by Sleep & 3rd Birthday Development Status |publisher=PSPHyper |date=2009-06-08 |accessdate=2011-03-12 |deadurl=yes |archiveurl=https://web.archive.org/web/20110715125556/http://www.psphyper.com/psp/square-birth-by-sleep-3rd-birthday-development-status/#comments |archivedate=2011年7月15日 |df= }}</ref>。日本唱片艺术家[[Superfly|Superfly]]乐团演唱了游戏主题曲《[[Eyes_on_Me_(Superfly歌曲)|Eyes on Me]]》<ref>{{cite web|url=http://superfly-web.com/news/2010/10/20102eyes-on-me.php|title=Superfly {{!}} News {{!}}2010年第2弾シングル！「Eyes On Me」発売！|date=2010-10-26|accessdate=2010-10-27|deadurl=yes|archiveurl=https://web.archive.org/web/20101030025208/http://superfly-web.com/news/2010/10/20102eyes-on-me.php|archivedate=2010-10-30}}</ref>。

史克威尔艾尼克斯在欧洲独占发行了扭曲者版。该版本使用红色游戏盒，附有48页的原画和概念插图手册、两张版画，以及独有《[[最终幻想_纷争012|最终幻想 纷争012]]》[[雷霆_(最终幻想)|雷霆]]服装的代码。在北美则是所有首批游戏有雷霆服装解锁码。
