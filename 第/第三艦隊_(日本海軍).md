'''第三艦隊'''是[[大日本帝國海軍|大日本帝國海軍]]的部隊之一。跟常設的[[第1艦隊_(日本海軍)|第一艦隊]]・[[第2艦隊_(日本海軍)|第二艦隊]]不同，只在有需要時編制・解散的特設艦隊，[[日俄战争|日俄戰争]]至[[太平洋战争|太平洋戰争]]間共出現過六次。

== 初代（1903年12月28日新編～1905年12月20日解散） ==
1903年12月的臨戰編制，[[大本營_(大日本帝國)|大本營]]直轄用以警備・哨戒的老朽艦集合。1904年3月編入[[聯合艦隊|聯合艦隊]]。支援陸軍部隊的[[金州_(消歧义)|金州]]上陸，[[波羅的海艦隊|波羅的海艦隊]]的探知，[[对马海峡海战|對馬海峡海戰]]當日的敵情報告等，為日本勝利作出很大貢獻。支援[[库页岛之战|樺太作戰]]上陸時再編。1905年12月聯合艦隊解散式同時解散。

== 二代（1908年12月24日從南清艦隊改名～1915年12月25日解散） ==
1905年12月開始，本土常設第一・第二艦隊，中国大陸方面則以邦人保護為名目派遣「[[南清舰队|南清艦隊]]」。這個南清艦隊改名後為第二代的第三艦隊。因為是海外派遣部隊故沒有司令部。[[第一次世界大战|第一次世界大戰]]勃發時，因[[中華民國|中華民國]]政府宣佈中立，[[巡洋舰|巡洋艦]]退去，河川砲艦於[[上海市|上海]]集合解除武裝、第三艦隊解散。

== 三代（1915年12月25日新編～1922年12月1日解散） ==
對應[[第一次世界大战|第一次世界大戰]]的臨戰編制，作為進出南方的第二艦隊的留守部隊。1917年[[1917年俄国革命|俄国革命，]]成為制圧[[苏联|苏联]]沿海州方面的主力部隊。陸軍[[西伯利亚干涉|西伯利亞出兵]]時作船團護衛。[[尼港事件|尼港事件]]勃發時救援部隊派遣失敗。第一次世界大戰終結，西伯利亞出兵中止，[[华盛顿会议|华盛顿軍縮會議]]後因舊式戰艦廢棄而解散。

== 四代（1932年2月2日新編～1939年11月15日改名第一遣支艦隊） ==
1932年1月28日[[一·二八事變|第一次上海事變]]勃発，向中國駐留的[[第一遣外艦隊|第一遣外艦隊]]・{{tsl|ja|第二遣外艦隊|第二遣外艦隊}}派遣增援部隊，此3支部隊統括成第四代第三艦隊。增援部隊包括「上海特別陸戰隊」。以河川砲艦為主體對[[长江|揚子江]]流域進行監視。1937年7月[[中国抗日战争|日華事變]]勃発，再增援大量部隊，10月增援部隊新編成{{tsl|ja|第四艦隊 (日本海軍)|第四艦隊 (日本海軍)|第四艦隊}}，和第三艦隊合併統率而成{{tsl|ja|支那方面艦隊|中國方面艦隊}}（支那方面艦隊）。第三艦隊司令部兼任艦隊司令部。1939年11月15日，3個艦隊分別改名「～遣支艦隊」，第三艦隊改名為「第一遣支艦隊」，司令部兼任解除。另外，第一遣支艦隊於1943年8月20日降格為揚子江方面特別根據地隊。新編至改名間7年9個月，是經歴代第三艦隊中最長。

== 五代（1941年4月10日新編～1942年3月10日改名第二[[南遣舰队|南遣艦隊]]） ==
作為攻略[[菲律宾|菲律賓]]部隊而編制。支援陸軍上陸，[[黎牙實比市|黎牙實比市]]攻略作戰等，菲律賓占領後轉向蘭印。根據陸海軍協定，海軍負責占領，陸軍負責駐留。蘭印占領1年後改名第二南遣艦隊。

== 六代（1942年7月14日新編～1944年11月15日解散） ==
作為[[中途岛海战|中途岛海战]]壞滅的[[第1航空艦隊_(日本海軍)|第一航空艦隊]]的後繼部隊，以沒參加中途岛海战的[[翔鹤号航空母舰|翔鶴]]・[[瑞鶴號航空母艦|瑞鶴]]為中心再建的機動部隊。

壞滅的第一航空艦隊長官[[南云忠一|南雲忠一]]中將和参謀長草鹿龍之介少將任第三艦隊指揮官。第三艦隊予定為空母六隻（翔鶴、瑞鶴、瑞鳳和飛鷹、隼鷹、龍驤２組）和戰艦2隻、巡洋艦4隻、長良為旗艦的驅逐艦16隻共計29隻的大艦隊<ref>星亮一『南雲忠一 空母機動部隊を率いた悲劇の提督』ＰＨＰ文庫p313</ref>。[[菲律賓海海戰|菲律賓海海戰]]時失去旗艦[[大鳳號航空母艦|大鳳]]跟大量艦上機飛行員，其後練成的航空機亦於[[台灣空戰|台灣空戰]]中消耗，[[雷伊泰灣海戰|雷伊泰灣海戰]]時基本沒有艦上機，作為誘敵艦隊而参加。雷伊泰灣海戰後、殘存的空母集合至第1航空戰隊，附屬聯合艦隊(後編入第二艦隊)，第4航空戰隊的航空戰艦[[伊勢號戰艦|伊勢]]・[[日向號戰艦|日向]]編入第二艦隊，第三艦隊廢止。其後、第4航空戰隊完成了{{tsl|ja|北号作戦|北号作戰}}。
