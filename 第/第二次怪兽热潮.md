'''第二次怪獣ブーム ''' （だいにじかいじゅうブーム）是指由1971年（昭和46年）至1974年（昭和49年）{{refnest|group = "注"|『超人画報』では、1973年までとしている<ref name = "画報">{{Cite book |和書 |editor=竹書房/イオン編 |date=1995-11-30 |title=超人画報 国産架空ヒーロー40年の歩み |publisher=[[竹書房|竹書房]] |pages=85|chapter=BonusColumn「変身ブーム到来!!」|id=C0076 |isbn=4-88475-874-9}}</ref>。}}的特攝劇為主，所引起的社會現象。別称、「'''変身热潮'''」<ref name = "画報" />{{refnest|group = "注"|ただし『[[全怪獣怪人|全怪獣怪人]]（上下巻）』のように「第二次怪獣ブーム」と「変身ブーム」を別個のものとして扱っているものもある<ref name = "全怪獣上">{{Harvnb|全怪獣怪人 上|1990|pp=144 - 147}}</ref>{{Sfn|全怪獣怪人 下|1990|pp=34、130 - 133}}。}}。當時在该熱潮下横多書籍和玩具制作集中在巨大怪獸的主題上。電視台和電影製作公司也常拍攝此類作品。
