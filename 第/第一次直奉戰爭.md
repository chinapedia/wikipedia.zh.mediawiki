'''第一次直奉戰爭'''，發生於1922年4月28日至5月5日，以[[直系|直]]勝[[奉系|奉]]敗收場。

== 起因 ==
起因于[[直皖战争|直皖战争]]胜利果实分赃不均和直系对亲日亲奉的[[梁士诒|梁士诒]]内阁不满。

1921年12月，梁士诒组阁。1922年1月吴佩孚联合六省军阀，通电攻击梁士诒内阁媚日卖国，迫梁1月15日请假离京，直奉矛盾日趋激化。[[張作霖|張作霖]]、[[段祺瑞|段祺瑞]]和[[孫中山|孫中山]]聯盟對抗直系[[曹锟|曹锟]]和[[吳佩孚|吳佩孚]]。張作霖自任總司令，[[孫烈臣|孫烈臣]]任副總司令，[[楊宇霆|楊宇霆]]任參謀長。

== 經過 ==
4月，奉軍12萬人入[[山海關|山海關]]，4月29日張作霖到天津[[軍糧城|軍糧城]]自任鎮威軍總司令，即日下達奉軍的總攻击令，第一次直奉戰爭爆發。奉軍西路軍由張作霖當總司令，分三個梯隊：第一梯隊[[張作相|張作相]]，第二梯隊[[張學良|張學良]]，第三梯隊[[李景林|李景林]]。<ref name="學良口述">[[唐德剛|唐德剛]]著、張學良口述：《張學良口述歷史》，[[台北|台北]]：[[遠流出版|遠流出版]]，2009年3月1日初版</ref>{{rp|140}}直系方面吳佩孚為總司令指揮7個師、5個旅約10萬人，進駐[[涿州|涿州]]。西路为[[琉璃河|琉璃河]][[吴佩孚|吴佩孚]]第三师，[[固安|固安]][[王承斌|王承斌]]第二十三师，东路为[[马厂|马厂]][[张国溶_(北洋政府将军)|张国溶]]第二十六师和[[张福来|张福来]]第二十四师。4月29日兩軍在[[馬廠|馬廠]]（右翼）、[[长辛店镇|長辛店]]（左翼）、[[霸縣|霸縣]]（中路）展開[[固安戰鬥|固安戰鬥]]、[[永清戰鬥|永清戰鬥]]、[[長辛店戰鬥|長辛店戰鬥]]、[[霸縣戰鬥|霸縣戰鬥]]等戰鬥，激戰中兩軍互有勝負，吳佩孚出奇兵繞道攻擊奉軍後方，奉軍腹背受敵；再加上奉軍[[邹芬|邹芬]]第十六師臨陣倒戈，全陣崩潰。5月4日，直軍佔領長辛店、[[永清縣|永清]]、[[靜海縣|靜海]]。与此同时，[[冯玉祥|冯玉祥]][[西北军|西北军]]出[[潼关|潼关]]，击败[[赵倜|赵倜]]，占领[[河南|河南]]。

== 結果 ==
最後直系獲勝，張作霖退出[[山海關|山海關]]，在[[秦皇島|秦皇島]]遭到直系海軍的炮火攻擊。但直軍在進攻山海關過程中，遭到[[張學良|張學良]]、[[郭松齡|郭松齡]]部隊的頑強抵抗，進攻被阻止，又經外國[[傳教士|傳教士]]調停。5月5日，第一次直奉戰争結束，后雙方在秦皇島英艦簽定和約停戰。

1922年7月，奉軍回到奉天，成立東三省陸軍整理處，統監[[孫烈臣|孫烈臣]]，副監張作相（名義）、[[姜登選|姜登選]]（行政），[[參謀長|參謀長]]張學良。<ref name="學良口述"/>{{rp|141,197}}6月，張作霖自任東北保安總司令，宣布“自治”，苦心經營海軍和空軍，訓練軍隊。吳佩孚恢复法统，重开[[国会|国会]]，逼迫[[徐世昌|徐世昌]]下野，[[黎元洪|黎元洪]]復任大总统。广东[[陈炯明|陈炯明]]认为法统恢復，应停止北伐，与[[孫中山|孫中山]]产生分歧。孙從北伐途中回[[廣州|廣州]]，被陳炯明困於[[永豐艦|永豐艦]]，最后不得不离开广州。1923年10月，[[曹錕|曹錕]]以賄選當上大總統。

在[[长辛店战斗|长辛店战斗]]中，直系出动[[空军|空军]]进行轰炸，是取得胜利的关键因素。
