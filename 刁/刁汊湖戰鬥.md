'''刁汊湖戰鬥'''是1932年5月2日－30日發生於[[湖南省_(中華民國)|湖北]]中部[[刁汊湖|刁汊湖]]湖区的一系列[[战斗|战斗]]，其中5月23日至30日的[[天門戰鬥|天門戰鬥]]，中共方面将称之为张家场战斗<ref name="a2"/>（张家场地点在今[[天门市|天门市]]<ref name="a1"/>）。該系列戰鬥是[[第一次国共内战|第一次国共内战]]主要戰鬥之一，交戰一方為中華民國國民革命軍，另一方則為中國共產黨所領導之中國工農紅軍。是国军对[[洪湖苏区|洪湖苏区]]的红军、游击队、[[赤卫队|赤卫队]]的清剿。

当时，刁汊湖湖区及附近地区是洪湖苏区中中共政权活跃地带。中共方面资料，称因国军的军事行动“人民群众受害甚大，湖中散洲尽成焦土，沿湖农村大批被烧，居民被杀死、溺死者达五千余人。”，张家场战斗亦被认为是在[[夏曦|夏曦]]主导下，红三军“得不偿失的消耗战”<ref name="a2">{{Cite web |url=http://military.china.com/history4/62/20140805/18685551_1.html |title=红三军保卫洪湖苏区：瓦庙集战斗后红军只剩2000人(2) |accessdate=2016-07-07 |author= |date=2014-08-05 |work= |publisher=中华网 |language=Zh-hans |deadurl=yes |archiveurl=https://web.archive.org/web/20160809101713/http://military.china.com/history4/62/20140805/18685551_1.html |archivedate=2016-08-09 }}</ref>。同年，在国军的重重围剿之下，中共政权最终失去洪湖苏区。

== 张家场战斗 ==

[[张家场战斗|张家场战斗]]或称九真张家场战斗<ref name="a3">{{Cite web |url=http://www.tmswb.gov.cn/dstd/jdxd/201206/t20120612_211102_8.html |title=《中国共产党天门历史》（第一卷）》第八章“左”倾错误的危害与天门苏区的丧失|accessdate=2016-07-12 |author= |date=2012-06-12 |work= |publisher=中共天门市委办公室网站，来源：中共天门市委党史研究室|language=Zh-hans }}</ref>。5月23日，[[贺龙|贺龙]]和代政委[[宋盘铭|宋盘铭]]率红三军主力，对皂市附近[[九真镇|九真镇]]张家场<ref name="a3"/>一带的国军第44师132旅及补充团发起袭击。战斗至30日，因国军援军达到，红三军撤出战斗。国军131旅旅长于兆龙以下八百人伤亡<ref>{{Cite web |url=http://cpc.people.com.cn/GB/69112/78233/78687/78977/5442457.html |title=《贺龙年谱》1932年　三十六岁 |accessdate=2016-07-07 |author= |date= |work= |publisher=中国共产党新闻网，来源：《贺龙年谱》|language=Zh-hans }}</ref>。31日，国军出击至柳河，红三军主力已撤退<ref name="a2"/>。

张家场战斗遗址现已列入天门市[[爱国主义教育基地|爱国主义教育基地]]<ref name="a1">{{Cite web |url=http://news.cnhubei.com/xw/2013zt/2013qhxj/qhdl/tianmen/zjdh/cs_zjdh/201502/t3185422.shtml |title=张家场战斗遗址|accessdate=2016-07-07 |author=编辑：刘晓洁  |date=2015-02-15 |work= |publisher=荆楚网|language=Zh-hans }}</ref>。
