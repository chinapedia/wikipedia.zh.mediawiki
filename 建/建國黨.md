'''建國黨''' ，是[[臺灣|臺灣]]的[[中華民國政黨列表|政黨]]之一，1996年10月6日成立，黨員主要是從[[民主進步黨|民主進步黨]]分裂出來的[[台灣獨立運動|台灣獨立運動]]成員<ref name="TaiwanIndependence">{{Cite web|url=http://www.drnh.gov.tw/ImagesPost/e3bc66dd-0cd2-49a1-a1f6-4129f4ccfd3f/db0cc1c7-5c47-4770-9463-8acf964c5f0e_ALLFILES.pdf|author=吳明時|language=zh-tw|title=1990年代的基層臺獨運動－一個社會運動的觀點|publisher=《[[國史館|國史館]]學術集刊》第四期|date=2004-09|accessdate=2013-07-11|deadurl=yes|archiveurl=https://web.archive.org/web/20140201190321/http://www.drnh.gov.tw/ImagesPost/e3bc66dd-0cd2-49a1-a1f6-4129f4ccfd3f/db0cc1c7-5c47-4770-9463-8acf964c5f0e_ALLFILES.pdf|archivedate=2014-02-01}}</ref>。

== 歷史 ==
=== 建黨 ===
建國黨一直是[[臺灣獨立|臺灣獨立]]的急先鋒，標榜「以建立新而獨立的[[臺灣共和國|臺灣共和國]]，維護臺灣國民及其世代子孫追求[[民主|民主]]、[[自由|自由]]、[[安全|安全]]、[[公平|公平]]、[[幸福|幸福]]的[[權利|權利]]為永不改變的最高宗旨」，主張立刻建立臺灣共和國，認爲[[中華民國|中華民國]]已不存在、[[中國國民黨|中國國民黨]]在台灣的統治爲非法。

[[1996年中華民國總統選舉|1996年中華民國總統選舉]]，[[民主進步黨|民主進步黨]]提名的正副總統候選人[[彭明敏|彭明敏]]、[[謝長廷|謝長廷]]的得票率只有21.1%，創下民進黨歷年得票率最低紀錄。隨後，民進黨[[美麗島系|美麗島系]]公開批判彭明敏的[[激進|激進]]台獨路線。彭明敏退出民進黨，並與[[台灣獨立建國聯盟|台灣獨立建國聯盟]]一部分勢力另組建國黨。

當時對民進黨中央先後提出「[[大膽西進|大膽西進]]」、「[[大聯合政府|大聯合]]」、「[[大和解|大和解]]」等主張不滿的台獨人士233人，鑑於1997年[[香港主權移交|香港主權移交]]的時限將屆，認為[[中華人民共和國政府|中華人民共和國政府]]更將以「[[一個中國原則|一個中國原則]]」壓縮台灣的國際空間，因此有組黨的行動。組黨的理由，一方面對內為了對抗民進黨將[[台獨黨綱|臺獨黨綱]]「文件化」（意即架空臺獨黨綱的計劃），另一方面則對外宣示抗拒中華人民共和國政府意圖以「[[一國兩制|一國兩制]]」或「武力解放」的方式取得台灣的企圖。

1996年10月6日，建國黨在[[臺北市|臺北市]][[士林區|士林區]]創黨，通過[[李鎮源|李鎮源]]為第一屆[[黨主席|黨主席]]、[[林山田|林山田]]為第一屆副主席、[[李勝雄|李勝雄]]為第一屆秘書長，選出「決策委員」李鎮源、李勝雄、林山田、[[李永熾|李永熾]]、[[廖宜恩|廖宜恩]]、[[鄭欽仁|鄭欽仁]]、[[陳茂雄|陳茂雄]]、[[李慶雄|李慶雄]]、[[郭倍宏|郭倍宏]]、[[李喬|李喬]]、[[徐馨生|徐馨生]]。李鎮源在接任建國黨主席時私下的一段豪邁磅礡的表白：「我想以有生之年貢獻給台灣這片土地，希望眼睛尚未闔上之前能夠見到台灣獨立建國成功，這是我的夢！」<ref>{{Cite web|url=http://www.hi-on.org.tw/bulletins.jsp?b_ID=47306|author=鄒景雯|language=zh-tw|title=台灣獨立建國 是他未圓的夢|publisher=[[自由時報|自由時報]]|date=2001-11-02|accessdate=2014-11-14}}</ref>

在建國黨的發起人名單中，[[企業|企業]]界的[[中產階級|中產階級]]75人，[[教育|教育]]界人士67人，[[醫學|醫學]]界人士26人，[[律師|律師]]及[[法學|法學]]界人士8人，[[文化|文化]][[藝術|藝術]]工作者30人，其他各地基層工作者27人。企業界的中產階級包括[[游善伯|游善伯]]（當時大倫氣球副總經理，長期關心[[黨外運動|黨外運動]]）、[[江茂彰|江茂彰]]（當時[[富邦人壽|富邦人壽]]中部區部經理）、[[辜寬敏|辜寬敏]]（[[鹿港辜家|鹿港辜家]]成員，長期支持彭明敏）、[[蔡仲伯|蔡仲伯]]（東錦企業創辦人）等。教育界人士包括[[教授|教授]]42人與[[教師|教師]]25人：教授主要以[[台灣教授協會|台灣教授協會]]會員為主，像是林山田、[[李永熾|李永熾]]、[[許慶雄|許慶雄]]、[[張炎憲|張炎憲]]、[[李筱峰|李筱峰]]等；教師主要以[[台灣教師聯盟|台灣教師聯盟]]成員為主，例如前台灣教師聯盟執行委員[[林央敏|林央敏]]；因此，建國黨創黨時，有些民進黨人士私下以「五點半建國黨」一語[[諷刺|諷刺]]建國黨這些教授是在各[[學校|學校]]於[[台灣時間|台灣時間]]每日17時30分全部下課之後搞「[[業餘|業餘]]革命」。

1997年10月5日，建國黨舉行第一屆全國黨員代表大會，辜寬敏、李鴻禧、林山田等人連番痛批民進黨。辜寬敏抨擊，民進黨自從提出「大和解」、「大聯合」以來已完全變節，許多地方基層與他一樣對民進黨失望且反感，建國黨的成立是因為民進黨的變節；但建國黨的任務並非要與民進黨作好朋友，而是要取代民進黨；而他尚未退出民進黨，是因為他在等待彭明敏一起行動，否則他早就[[退黨|退黨]]了。李鴻禧則說，最近台灣獨立運動的路很坎坷，過去被台獨人士寄望「結束外來政權」的人如今已經變節；他也嘲諷民進黨「辣妹助選團」：「他們（民進黨）以為：辣妹搖兩下之後，大家就會趕快入黨。」林山田則以民進黨候選人包[[尿布|尿布]]比賽諷刺，面對[[中國威脅|中國威脅]]，民進黨竟然還在「包尿布」，建國黨不做這種[[媚俗|媚俗]]的政黨。大會隨後並發表宣言，再度抨擊：台灣內部有一股肯定中華民國體制的[[維持現狀|維持現狀]]勢力，盤據在許多反對運動人士的腦海中，並於修憲時放棄台灣這個領域內「主權人」應有自己憲法的崇高意旨。<ref>{{Cite web|url=http://forums.chinatimes.com/special/independent/86dd1006.htm|author=張瑞昌|language=zh-tw|title=獨派大老痛批民進黨媚俗變節|publisher=[[中國時報|中國時報]]|date=1997-10-06|accessdate=2013-07-11}}</ref>

1997年10月6日，建國黨一周年黨慶，民進黨婦女部主任[[李永萍|李永萍]]帶一包尿布前往「慶祝」，實際上是諷刺林山田的言論，與建國黨方面言語不合，雙方不歡而散<ref>{{Cite web|url=http://nrch.cca.gov.tw/ccahome/search/search_meta.jsp?xml_id=0000785333&dofile=cca220001-hp-cdn0076173-0001-i.jpg|author=方賓照|language=zh-tw|title=民進黨婦女部主任李永萍(右二)昨日特地帶一包尿布前往建國黨，慶祝建國黨一周年黨慶，卻與建國黨方面言語不合，雙方不歡而散。|publisher=[[中央日報|中央日報]]|date=1997-10-07|accessdate=2013-07-11}}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

===黨旗===
建國黨黨旗由郭倍宏之妻[[張舜華|張舜華]]設計，旗上僅有三線：先於距上端五分之二處畫一橫線，再於橫線中點連兩線至左下、右下兩頂點，即告完成。黨旗含意：人文方面，一可看作先民漂洋過海來到臺灣，於海岸某點登陸後逐漸擴散，遍及全島；二可看作今日臺灣人民從四面八方來，同心協力匯聚出歷史的主流大道，勇往直前邁向獨立建國。旗面分為4個區塊，代表[[臺灣四大族群|臺灣四大族群]]互相融合。黨旗由藍色、綠色、金黃色組成：自然方面，可解為「藍色海洋、青色草原及金色山脈」或「蔚藍天空下、青青草原上，金黃山脈高聳入雲」，皆顯現臺灣獨特的地理景觀；又可看成是一張環保的面孔，淺藍額頭象徵清新空氣，墨綠眼鏡象徵鬱茂森林，金黃鼻頭象徵成熟豐收的大地；也可看作在蔚藍天空與靛青海洋水天一線的美麗景色中，象徵獨立建國力量的金黃[[斗笠|斗笠]]正在向上成長，即將從平靜無波的水面突破而出、掀起滔天巨浪。整幅圖案亦可視為「-{台}-」字的變體。

===發展===
李鎮源及其以後的建國黨主席[[許世楷|許世楷]]、[[鄭邦鎮|鄭邦鎮]]均為[[學者|學者]]出身，堪稱典型的「書生論政」風格；建國黨黨員[[陳文輝|陳文輝]]、[[陳光復|陳光復]]、李慶雄等人，亦分別曾任第三屆及第四屆[[立法委員|立法委員]]；但由於與時代主流意識及[[價值觀|價值觀]]有所落差等諸多因素，建國黨許多有志難伸的創黨黨員如李永熾、鄭欽仁、陳茂雄、郭倍宏、李喬、辜寬敏、許慶雄、張炎憲、李筱峰、林央敏等精英逐漸淡出。<ref>{{Cite web|url=http://pnn.pts.org.tw/main/2011/10/06/%E3%80%90%E5%8F%B0%E7%81%A3%E6%AD%B7%E5%8F%B2%E9%9A%A7%E9%81%93%E3%80%9119961006%E5%BB%BA%E5%9C%8B%E9%BB%A8%E5%BB%BA%E7%AB%8B/|author=蔡漢勳|language=zh-tw|title=【台灣歷史隧道】1996年10月6日建國黨建立|publisher=[[公視新聞|公視新聞]]議題中心|date=2011-10-06|accessdate=2013-07-11|archive-url=https://web.archive.org/web/20141128010421/http://pnn.pts.org.tw/main/2011/10/06/%E3%80%90%E5%8F%B0%E7%81%A3%E6%AD%B7%E5%8F%B2%E9%9A%A7%E9%81%93%E3%80%9119961006%E5%BB%BA%E5%9C%8B%E9%BB%A8%E5%BB%BA%E7%AB%8B/|archive-date=2014年11月28日|dead-url=yes}}</ref>

1997年10月19日，李鎮源致函《[[新新聞|新新聞]]》，駁斥李筱峰在《新新聞》對建國黨的批評：1.建國黨並不排斥公職人員，故無「反對公職掛帥」；2.建國黨「缺乏民主」之說，是以訛傳訛所導致的誤解；3.建國黨黨職人員多具[[法律|法律]]與政治專才，也有從事社會運動及組織的人才，也有多方面的政策委員及顧問，也有連任多年的立法委員及其他公職人員，故並非「缺乏政治專業人才」；4.建國黨決策委員僅決定政策，公職人員、總部、各地辦公處及分部執行政策，「決策委員把持黨務」是言過其實；5.建國黨本屆決策委員是由建國黨創黨籌備委員會選出，並非由黨主席指定；6.建國黨主席僅在決策委員會中，指定林山田兼任副主席、李勝雄兼任秘書長<ref>{{cite news|url=|title=來函|author=李鎮源|date=1997-10-19|accessdate=2013-09-24|publisher=《新新聞》第554期|page=6}}</ref>。

== 政治理念 ==

=== 黨綱 ===
建國黨的「建國綱領」有10點：
# 以台灣全體國民為主體，以[[台灣本島|台灣本島]]及其附屬[[島嶼|島嶼]]為[[領土|領土]]範圍，形成命運共同體，制定台灣共和國[[憲法|憲法]]。
# 深入台灣[[社會|社會]]，組織群眾，聯合社會各階層，清除[[黑金|黑金]][[腐敗|腐敗]]，粉碎中華人民共和國的[[侵略|侵略]]，完成獨立建國目標。
# 保障國民[[基本人權|基本人權]]，作為國家與社會應遵守的基本原則。
# 以[[政治|政治]]民主為基礎，建立[[國民主權|國民主權]]的政治體制。
# 以[[主權國家|主權國家]]的地位加入[[國際組織|國際組織]]，與世界各國建立[[外交|外交]]關係及[[經濟|經濟]][[貿易|貿易]]的合作。
# 堅持以台灣為主體的教育與文化，建立以互相尊重的[[個人主義|個人主義]]為基礎的[[社區|社區]]主義與公共文化。
# 推行[[男女平等|兩性平權]]政策，保障社會弱勢者的權利。
# 尊重各[[族群|族群]][[語言|語言]]與文化的自主，保障[[原住民族|原住民族]]設置[[自治區|自治區]]的權利。
# 厚植台灣自主性的經濟實力，結合全球經濟體系，並實現經濟公平的理想。
# 以綠色和平及永續發展為指標，從事國土規劃與[[環境|環境]][[生態|生態]]的保護，並促進台灣與[[地球村|地球村]]的和諧共存。

=== 政治活動 ===
1997年6月5日，建國黨批評，在[[國家認同|國家認同]]上，台灣人本非中國人，不應站在「中國一部分」的角度編寫台灣歷史；《[[認識台灣|認識台灣]]》教科書中，社會篇最具正義性與符合課程原則，地理篇寫作品質中等，歷史篇「作者具有健康的歷史知識，卻無全盤托出歷史真相的膽識，學生將無法正確理解台灣歷史傳承的脈絡」；建國黨主張應重新撰寫《認識台灣》歷史篇，以台灣主體之立場，從五萬年前[[長濱文化|長濱文化]]開始落筆，將歷史真實一一公正呈現，使台灣新的一代有能力在心靈上做台灣的主人<ref>{{Cite web|url=|author=台北訊|language=zh-tw|title=建國黨：未堅持台灣主體性立場 應重編|publisher=中國時報|date=1997-06-06|accessdate=2015-07-04}}</ref>。

1999年10月14日，建國黨決策會議決定發行「台灣錢」及「建國<nowiki>彩券</nowiki>」，以籌募建國黨長期運作經費及參加[[2000年中華民國總統大選|2000年中華民國總統大選]]之費用。「台灣錢」是建國黨以台灣共和國名義發行的[[貨幣|貨幣]]，「建國<nowiki>彩券</nowiki>」是建國黨以台灣共和國名義發行的[[刮刮樂|刮刮樂]]型[[彩券|彩券]]，兩者性質均為[[有價證券|有價證券]]，建國黨將兩者統稱為「建國黨券鈔」。

1999年11月5日，鄭邦鎮親自在[[中華民國中央銀行|中華民國中央銀行]]前主持「台元券發行[[記者會|記者會]]」，宣佈正式發行「台灣錢」，並發表台灣錢的[[鈔票|鈔票]]「台元券」。鄭邦鎮說，建國黨之所以選擇在中華民國中央銀行發表台灣錢，是為了「揭穿中華民國所發行的[[新台幣|新台幣]]是國民黨把[[台灣銀行|台灣銀行]]代理發行的新台幣作為地方貨幣，而不是一個[[國家|國家]]的[[國幣|國幣]]」。台元券票-{面}-金額分為100元、500元、1000元與5000元。建國黨規定，台灣錢為不可兌換券，但可用來購買建國黨發售、義賣之貨品。建國黨同時公布台灣錢[[匯率|匯率]]如下：台灣錢1元兌換新台幣1元，台灣錢30元兌換1[[美元|美元]]，台灣錢1元兌換4[[日圓|日圓]]。

2000年1月6日，由於鄭邦鎮的連署人數未能跨過連署門檻，建國黨召開記者會宣布鄭邦鎮退出2000年中華民國總統大選，鄭邦鎮帶領建國黨決策委員向支持者鞠躬致歉；鄭邦鎮批評，現行體制既未實施[[兩票制|兩票制]]，因此依據立法委員選舉得票率來規範正副總統參選門檻似有[[違憲|違憲]]之嫌，他們將尋求[[司法院大法官|司法院大法官]][[釋憲|釋憲]]來導正制度<ref>{{Cite web|url=http://forums.chinatimes.com.tw/report/vote2000/chen/89010714.htm|author=吳典蓉|language=zh-tw|title=建國黨提名人鄭邦鎮連署未過關 將聲請釋憲|publisher=中國時報|date=2000-01-07|accessdate=2014-11-17|deadurl=yes|archiveurl=https://web.archive.org/web/20141129020209/http://forums.chinatimes.com.tw/report/vote2000/chen/89010714.htm|archivedate=2014-11-29}}</ref>；同日，[[中華民國中央選舉委員會|中華民國中央選舉委員會]]回應，大法官會議已於[[司法院釋字第468號解釋|司法院釋字第468號解釋]]指出，連署制度並不違憲<ref>{{Cite web|url=http://forums.chinatimes.com.tw/report/vote2000/chen/89010715.htm|author=董孟郎|language=zh-tw|title=中選會：大法官已解釋過 不違憲|publisher=中國時報|date=2000-01-07|accessdate=2014-11-01|deadurl=yes|archiveurl=https://web.archive.org/web/20141129020214/http://forums.chinatimes.com.tw/report/vote2000/chen/89010715.htm|archivedate=2014-11-29}}</ref>。
