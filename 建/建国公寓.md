'''建国公寓'''是位于[[上海市|上海市]][[徐汇区|徐汇区]][[建国西路_(上海)|建国西路]]394号的一幢高层公寓住宅，处于建国西路北侧，[[岳阳路_(上海)|岳阳路]]、[[太原路_(上海)|太原路]]之间。公寓原名'''法国太子公寓'''或'''道斐南公寓'''（{{lang-en|'''Dauphine Apartments'''}}），得名于{{tsl|en|Dauphine_of_France|法国王太子妃}}。

==历史==
20世纪20至30年代，随上海地价飙升，大量高层公寓被建造。<ref name=gc>[http://www.shtong.gov.cn/newsite/node2/node2245/node75091/node75095/node75115/node75127/userobject1ai90867.html 专业志 >> 上海住宅建设志 >> 第一篇旧有住宅 >> 第三章花园住宅、高层公寓 >> 第二节　高层公寓]上海地方志办公室</ref>建国公寓建于1935年前后，由[[赉安|法商赉安洋行]]设计，安记营造厂承建。<ref>[http://www.shtong.gov.cn/newsite/node2/node2245/node69543/node69552/node69640/node69644/userobject1ai67938.html 	专业志 >> 上海建筑施工志 >> 人物篇 >> 第一章人物传略 >> 姚长安]上海地方志办公室</ref>公寓建成之初原由太子洋行代理老板与高级职员居住，20世纪50年代起其产权曾归[[上海市公安局|上海市公安局]]所有，由公安局经济犯罪检查总队使用。<ref name=wh>[http://whj.xh.sh.cn/xhwh_website/HTML/xhwh/xhwh_wwbh_wbdw/Info/Detail_25738.htm 首页 > 文物保护 > 文保单位 >道斐南公寓]徐汇文化</ref><ref name=s>[http://www.docin.com/p-353333613.html 赉安洋行在上海的建筑作品研究]</ref>2005年，经济犯罪检查总队迁出，公寓为上海市机关事务局所有，作为高干住宅公寓使用。<ref name=tp>[http://tianping.xh.sh.cn/html/xh_tpwhzx/Flash_yffb/you/you-6.swf 道斐南公寓]天平街道不可移动文物介绍</ref>公寓分别于1994年和2003年，列入第二批[[上海市优秀历史建筑|上海市优秀历史建筑]]和[[徐汇区登记不可移动文物列表|徐汇区登记不可移动文物]]名单。

==建筑特色==
建国公寓占地面积2500平方米，是一幢九层钢筋混凝土结构的建筑。公寓坐北朝南，建筑平面为“一字形”。这种结构的建筑往往沿主干道布置（例如[[淮中大楼|淮中大楼]]）以节省用地。<ref name=gc></ref>但是建国公寓并不临街，而是处于其所属地块的北侧。公寓南侧设有绿化等景观。<ref name=s></ref>
===建筑外观===
建国公寓是一幢现代风格的建筑，但是仍然带有装饰艺术的特征。<ref name=s></ref>其立面材质为水泥刷面。建筑立面结构对称，竖向采用三段式构图，中部凸出，拥有超出层面的竖向线条。建筑底层入口处设有较大的雨棚；每一层两侧均出挑转角为弧形的水泥阳台。建筑中部凸起部分两侧设置转角窗，两旁的窗户为圆形或方形。除此之外，建筑立面无其他明显的装饰，具有现代建筑简洁的特点。<ref name=wh></ref><ref name=tp></ref>
