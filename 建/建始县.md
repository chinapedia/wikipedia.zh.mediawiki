'''建始县'''是[[中华人民共和国|中国]][[湖北省|湖北省]][[恩施土家族苗族自治州|恩施土家族苗族自治州]]辖下的一个[[县|县]]，原属[[四川省|四川省]]。总面积为2659平方公里，2004年人口为51万人。

== 历史 ==
* [[三国|三国]]·[[吴|吴]][[永安_(孫休)|永安]]三年（260年）置县，隶属[[荆州|荆州]]，是为建始建置之始。
* [[西晋|西晋]][[太康|太康]]五年（265年）省后复置，属荆州[[建平郡|建平郡]]。
* [[北周|北周]][[建德|建德]]三年（574年）改置[[业州|业州]]，[[隋朝|隋]][[大业|大业]]初年（605年）废州置县，属[[清江郡|清江郡]]；[[义宁|义宁]]二年（618年）复置业州；
* [[唐朝|唐]][[贞观|贞观]]八年（634年）再废州，以县属[[施州|施州]]。
* [[宋朝|宋]]属[[夔州路|夔州路]][[施州|施州]]
* [[元朝|元]]属[[四川行省|四川行省]][[夔州路|夔州路]][[施州|施州]]。
* [[明朝|明]][[洪武|洪武]]四年（1371年）裁[[施州|施州]]，直屬[[四川省|四川省]][[夔州府|夔州府]]<ref>《明史》卷43，頁1029一頁1030、卷44，頁109：</ref>；
* 洪武九年（1376年）四月，夔州府改為散州，建始縣往屬[[四川省|四川省]][[重慶府|重慶府]]<ref>《明太祖實錄》卷105</ref>。
* 洪武十年(1377年)五月，夔州升為直隸州，建始縣屬[[四川省|四川省]][[夔州直隸州|夔州直隸州]]<ref>《明太祖實錄》卷112</ref>。
* 洪武十三年(1380年)十一月，升夔州直隸州為府，屬[[四川省|四川省]][[夔州府|夔州府]]<ref>《明太祖實錄》卷134</ref>。
* 洪武十四年(1381年)五月，復置施州，分建始縣隸之，屬[[四川省|四川省]][[夔州府|夔州府]][[施州|施州]]<ref>《明太祖實錄》卷137</ref>。
* 洪武二十三年(1390年)，再裁施州，建始縣還屬[[四川省|四川省]][[夔州府|夔州府]]<ref>《大明一統志》卷66</ref>。
* 雍正十三年（1735年），改建始縣往屬湖北省[[施南府|施南府]]。
* 民国元年（1912年）废府存县，直属湖北省；民国25年（1936）属湖北省第七行政督察区。
* 1949年11月5日后属恩施专区（1970年改恩施地区，1983年改鄂西土家族苗族自治州，1993年改[[恩施土家族苗族自治州|恩施土家族苗族自治州]]）。<ref name="人口区划">{{cite web|title=人口区划|url=http://www.hbjs.gov.cn/col/col13/index.html|publisher=中国建始网|deadurl=yes|archiveurl=https://web.archive.org/web/20101216215455/http://hbjs.gov.cn/col/col13/index.html|archivedate=2010-12-16}}</ref>

==行政区划==
建始县辖6个镇：业州、红岩、高坪、景阳、官店、花坪，4个乡：长梁、茅田、龙坪、三里。

== 地理 ==
建始县位于鄂西南山区北部，东连[[巴东县|巴东县]]，以[[野三河|野三河]]为界；西接[[恩施市|恩施市]]，以[[太阳河|太阳河]]为界；南邻[[鹤峰县|鹤峰县]]，以[[长河|长河]]、[[茶寮河|茶寮河]]为界；北与重庆市[[巫山县|巫山县]]毗连；西北与重庆市[[奉节县|奉节]]、巫山两县接壤。南北长89公里，东西最宽64.5公里，最窄处12.5公里，总面积2666.55平方公里。全县最高点北部[[太平峰|太平峰]]海拔2090.1米，最低点东南部野三口（汇入清江处）海拔213米，平均海拔1152米。<ref>{{cite web|title=建始概要|url=http://www.hbjs.gov.cn/col/col2/index.html|publisher=中国建始网|deadurl=yes|archiveurl=https://web.archive.org/web/20101216203104/http://hbjs.gov.cn/col/col2/index.html|archivedate=2010-12-16}}</ref>

北部属[[巫山|巫山]]，南部属[[武陵山脉|武陵山脉]]。<ref name="湖北省地图册">{{cite book |title=湖北省地图册 |year=2008 |publisher=星球地图出版社 |isbn=978-7-80212-174-4 }}</ref>

主要河流有[[清江|清江]]干流景阳河段、[[野三河|野三河]]、[[马水河|马水河]]、[[太阳河|太阳河]]、[[东龙河|东龙河]]、[[闸木水河|闸木水河]]、[[刀龙河|刀龙河]]、[[伍家河|伍家河]]、[[猪耳河|猪耳河]]等。至2001年底，全县建有[[四十二坝水库|四十二坝]]、[[小溪口水库|小溪口]]二座中型水库及小(Ⅰ)型水库5座、小(Ⅱ)型水库8座。<ref>{{cite web |title=自然地理|url=http://www.jianshiren.com |publisher=中国建始人网 }}</ref>另有正在建设中的[[野三河水电站|野三河水电站]]。<ref>{{cite web|title=湖北省人民政府关于禁止在建始县野三河水电站工程占地和水库淹没影响区新增建设项目及迁入人口的通告|url=http://www.hubei.gov.cn/zwgk/zfxxgk/zfgw/zfgb2009/gg0906/200905/t20090505_96587.shtml|publisher=湖北省政府门户网站}}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

== 人口 ==
2000年全县有[[汉族|汉]]、[[土家族|土家]]、[[苗族|苗]]、[[回族|回]]、[[满族|满]]、[[蒙古族|蒙古]]、[[侗族|侗]]、[[畲族|畲]]、[[白族|白]]、[[彝族|彝]]、[[维吾尔族|维吾尔]]、[[朝鲜族|朝鲜]]、[[壮族|壮]]、[[藏族|藏]]、[[瑶族|瑶]]等15个民族，其中[[汉族|汉族]]占总人口的63.7%。<ref name="人口区划"/>

== 教育 ==
2007年全县有[[中学|中学]]25所，[[小学|小学]]222所，[[中等专业学校|中等专业学校]]3所，[[特殊教育学校|特殊教育学校]]1所。全县共有专任教师3983人，在校学生67464人。<ref name="2007年"/>

== 医疗卫生 ==
2007年全县有[[医院|医院]]2所、[[卫生院|卫生院]]10所，拥有床位973张。<ref name="2007年"/>
