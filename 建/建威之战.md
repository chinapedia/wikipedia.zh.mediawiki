'''建威之战'''，是中国[[三国|三国]]时期敌国[[蜀汉|蜀汉]]和[[曹魏|曹魏]]之间的一场战斗，也是蜀汉[[丞相|丞相]][[诸葛亮|诸葛亮]]发起第三次北伐的一场战役。

此战以蜀汉夺取今[[甘肃|甘肃]][[陇南|陇南]]一带的[[武都|武都]]、[[阴平|阴平]]两郡告终，成為漢中西北屏障。

== 战事 ==
229年，诸葛亮令宿将[[陈式|陈式]]（《[[资治通鉴|资治通鉴]]》作陈戒）率军欲取武都、阴平二郡，诸葛亮亲自留在[[阳平关|阳平关]]集结地为后援。

不久得知魏國[[雍州|雍州]][[刺史|刺史]][[郭淮|郭淮]]已调兵后，诸葛亮立即率本部进军武都郡西北角的[[建威|建威]]，寻求与魏军决战以歼灭其雍州边防主力軍，起初郭淮想攻打陈式军以防武都、阴平陷落，但在得知诸葛亮到达建威之后，郭淮决定不攻打陈式，而是令军队救建威，建立防线以防蜀汉军进军。

诸葛亮到建威后没有继续进军，下令部隊就地紮營，而同時部署军队迎战可能出现的魏國援军。

魏國郭淮深知武都、阴平二郡的魏军战斗力难以抗衡蜀汉军，亦知诸葛亮試圖與郭淮军决战並围城打援，因此便放弃对武都、阴平的防守。魏國郭淮率領雍州边防主力軍撤退後，蜀汉军取得武都、阴平二郡。

同年冬季诸葛亮徙府营於[[終南山|南山]]下原上，築漢城於[[沔陽|沔陽]]，築樂城於[[城固|城固]]，养兵练武、屯田養息，准备第四次北伐。

== 后果 ==
蜀汉军在此战中未能歼灭魏军主力，亦未能获得大量物资，但武都、陰平兩郡在地理上具有一定的战略地位<ref>后來[[邓艾|邓艾]]即从阴平山险小路避开[[姜维|姜维]]重兵设防的剑阁，偷袭[[江油|江油]]、[[成都|成都]]得手。</ref>，从而巩固了蜀汉的西北边防，此後蜀汉与[[氐人|氐人]]、[[羌人|羌人]]建立了一定程度的联系。

诸葛亮在夺得武都、阴平后，修筑汉、乐二城，增强[[汉中|汉中]]的军备、养兵练武，准备往后的北伐（参见231年[[诸葛亮北伐|诸葛亮北伐]]中的第四次北伐）。

武都、陰平兩郡被蜀漢佔據之后，魏军试图攻击汉中，但遭有所准备的蜀漢軍隊在汉中诸道击败，又因山路與山中棧道難行，不久魏军撤退。

== 評價 ==
早在[[漢中之戰|漢中之戰]]時[[曹操|曹操]]便將武都地區的居民與糧食大半撤離，而且武都、陰平兩郡地瘠人稀，因此曹魏在建威之戰後並未積極收復這兩郡。

然而武都、陰平兩郡可形成漢中地區西北屏障，'''武都郡'''可作為進攻曹魏領地的重要戰略據點，'''陰平郡'''的陰平道則能繞過重兵把守的劍閣直取蜀漢腹地成都，對蜀漢仍有一定的戰略價值。

先前诸葛亮因[[街亭之战|街亭之战]]的敗績而辞去[[丞相|丞相]]，之后蜀汉皇帝[[刘禅|刘禅]]下诏褒扬诸葛亮在[[陈仓之围|第二次北伐]]中击败[[王雙_(曹魏)|王双]]及此次北伐夺取武都、阴平之功，复诸葛亮为丞相。
