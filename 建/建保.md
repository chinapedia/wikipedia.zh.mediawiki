'''建保'''（1214年十二月六日至1219年四月十二日）是[[日本|日本]]的[[年號|年號]]之一。[[後鳥羽上皇|後鳥羽上皇]]實施[[院政|院政]]。這個時代的[[天皇|天皇]]是[[順德天皇|順德天皇]]。[[鎌倉幕府|鎌倉幕府]]之[[征夷大將軍|征夷大將軍]]為[[源實朝|源實朝]]；[[執權|執權]]為[[北條義時|北條義時]]。

== 改元 ==
*建曆三年十二月六日（1214年1月18日）改元建保
*建保七年四月十二日（1219年5月27日）改元承久

== 出處 ==
《[[書經|書經]]·多士》：「自[[成湯|成湯]]至于[[帝乙|帝乙]]，罔不明德恤祀；亦惟天丕'''建'''，'''保'''乂有殷；殷王亦罔敢失帝，罔不配天，其澤。」

== 逝世 ==
*建保七年正月：[[源實朝|源實朝]]，鎌倉幕府第三代征夷大將軍。

== 大事記 ==
*建保七年正月：[[源賴家|源賴家]]之二子[[源公曉|源公曉]]在[[鶴岡八幡宮|鶴岡八幡宮]]暗殺[[源實朝|源實朝]]。
