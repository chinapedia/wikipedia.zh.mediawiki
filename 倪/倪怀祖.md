'''倪怀祖'''（{{bd|1905年||1991年|}}），中国化学家、药学家。

倪怀祖是中国福建省[[福州|福州]]人，他是家中的次子，父亲倪文修，母亲[[倪林和平|倪林和平]]，胞兄倪述祖（[[倪柝声|倪柝声]]）日后成为著名教会领袖。1928年，倪怀祖毕业于[[上海圣约翰大学|上海圣约翰大学]]化学系，1930年获[[硕士|硕士]]学位，留校任教<ref>[http://www.shtong.gov.cn/newsite/node2/node2245/node65059/node65069/node65117/node65123/userobject1ai60092.html 上海化学工业志 >> 第六篇人物 >> 第一章人物传略]</ref>。1937年<ref>上海医药志 >> 第一编行业 >> 第二章化学制药行业 >> 第二节　企业</ref>，其堂兄、商业巨子倪树祖（“Peter倪”）投资创办上海[[中国生化制药厂|中国生化制药厂]]，倪怀祖起初担任厂长，后来专门负责研制药品，任化学师、技术经理。他陆续发明、研制成功多种药品。1939年，胞兄[[倪柝声|倪柝声]]出任厂长。

1950年，中国政府下令将生化制药厂并入沈阳东北制药总厂，倪怀祖担任技术总顾问，试制成功维生素C和DDT等。1952年回到上海，任教会所办翠华染料厂总工程师。上海市地方工业局接管翠华染料厂<ref>[http://www.shtong.gov.cn/newsite/node2/node2245/node65059/node65065/node65100/node65157/userobject1ai59854.html 上海化学工业志 >> 第二篇行业企业 >> 第二章企业 >> 第三节　中型企业]</ref>后，1954年倪怀祖改任永华樟脑厂（1960年并入桃浦化工厂）工程师。1972年发明金属阳极隔膜电介槽，1978年因而获得全国科学大会重大贡献奖。  

倪怀祖是一位虔诚的基督徒。抗战时期曾在澳门发起传福音聚会<ref>敞开的门第十六期</ref>。文革期间因遭到残酷批斗而公开宣布放弃信仰，日后又恢复信仰<ref>陈终道：倪柝声，张品蕙为信基督而死</ref>。 

== 家庭 ==
倪怀祖的妻子[[徐奉先|徐奉先]]，是中国安徽省一个中国内地会牧师的次女。1989年在高安路家中接待[[张锡康|张锡康]]所在的擘饼聚会<ref>张锡康回忆录—上海地方教会六十年来的回顾</ref>。2003年6月17日曾在上海家中接受采访，驳斥[[梁家麟|梁家麟]]的《倪柝声的荣辱升黜》一书。妻姐[[徐凤翔|徐凤翔]]（1900年-2001年），是[[地方教会|地方教会]]中的一位女同工，曾长期在[[上海|上海]]、[[台北|台北]]和[[西雅图|西雅图]]侍奉；妻妹徐恩秀嫁给倪柝声的五弟倪兴祖，在美国[[加利福尼亞州|加利福尼亞州]][[帕萨迪纳_(加利福尼亚州)|帕萨迪纳]]定居。内侄徐祥生（徐腓利）则是殉道者[[张愚之|张愚之]]的三女儿张向晨的丈夫。倪怀祖夫妇和[[倪柝声|倪柝声]]夫妇一同安葬在江苏省[[苏州市|苏州市]]香山公墓。
