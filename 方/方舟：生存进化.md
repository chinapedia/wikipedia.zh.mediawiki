{{vgname|方舟：生存进化|en=Ark: Survival Evolved}}是一款由Studio Wildcard制作与发行的生存类[[沙盒|沙盒]][[动作冒险游戏|动作冒险]][[视频游戏|遊戲]]，使用[[虚幻引擎|虚幻4]]引擎打造，于2017年8月29日在[[Linux|Linux]]，[[Microsoft_Windows|Microsoft Windows]]，[[OS_X|OS X]]，[[PlayStation_4|PlayStation 4]]和[[Xbox_One|Xbox One]]平台发行，并于2018年6月14日在[[iOS|iOS]]和[[Android|Android]]平台发行；开发者也计划[[任天堂Switch|任天堂Switch]]平台发行。游戏舞台設定在一座充滿[[恐龍|恐龍]]的神秘島嶼「方舟」，玩家必須運用技巧與智慧在危机四伏的世界中生存<ref>{{Cite web|url = http://gnn.gamer.com.tw/9/118889.html|title = 開放世界恐龍生存遊戲《方舟：生存進化》釋出 Xbox One 版宣傳影片|accessdate = 2015-08-13|author = |date = 2015-08-05|publisher = [[巴哈姆特|巴哈姆特]]|language = zh-tw}}</ref>。

游戏以[[第一人称|第一人称]]或[[第三人称|第三人称]]视角进行。在游戏世界中，玩家可以步行或者骑恐龙，可以使用[[枪械|枪械]]或{{link-en|简易武器|Improvised weapon}}来对抗敌对人类和生物，建立基地用以防御。玩家也可以利用从游戏中搜刮的资源打造自己的武器。游戏可以选择[[单人电子游戏|单人游戏]]或[[多人电子游戏|多人游戏]]，其中多人游戏可以允许多名玩家同时在线参与。

游戏于2014年10月开始制作，开发团队为了在游戏玩法上有所创新，对恐龙的外貌进行了大量研究。

==游戏设计==
《方舟：生存进化》是一款{{link-en|生存类|Survival mode}}动作冒险游戏，采用[[第一人称视角游戏|第一人称视角]]，在某些情况下可以切换[[第三人称视角游戏|第三人称视角]]，玩家必须通过打猎、种植、科研等多种方式生存下去，此外玩家还需要建立基地来保护自己<ref name="Wire"/>。游戏的地图被称为“方舟”，面积约为48平方公里<ref>{{harvnb|Rapczak|2015|loc=20:20}}</ref>，其中四分之三为陆地，四分之一为海洋<ref name="FAQ"/>。

要建立基地，玩家必须获得基本部件比如地板、门窗，这是随着玩家的进度和收集水平的提高逐渐获得的。只要玩家有资源，就可以制造任何组件。当建筑的支柱和基础受到破坏时，建筑物的结构完整性也会受到损害<ref name="Wire"/>。玩家还可以在游戏中制作道具，比如武器，这可以通过收集制作所需的资源和技术来实现<ref>{{harvnb|Rapczak|2015|loc=31:31}}</ref>。此外，玩家可以制作附加组件并把它们添加到武器上，比如为手枪或机枪添加[[瞄准镜|瞄准镜]]或[[手电筒|手电筒]]<ref>{{harvnb|Rapczak|2015|loc=31:51}}</ref>。如果玩家受伤后食用了必要的食物，其生命值会逐渐恢复<ref>{{harvnb|Rapczak|2015|loc=41:02}}</ref>。

游戏中已超过60种恐龙<ref name="Press1">{{cite press release |url=http://www.gamasutra.com/view/pressreleases/243167/ARK_SURVIVAL_EVOLVED_ndash_A_NEW_BREED_OF_OPENWORLDDINOSAUR_ADVENTURE_IS_COMING.php |title=Ark: Survival Evolved – A New Breed of Open-World Dinosaur Adventure is Coming |type=Press release |location=Seattle, Washington |publisher=Studio Wildcard |date=2015-05-11 |archiveurl=https://www.webcitation.org/6YbwjBXfI?url=http://www.gamasutra.com/view/pressreleases/243167/ARK_SURVIVAL_EVOLVED_ndash_A_NEW_BREED_OF_OPENWORLDDINOSAUR_ADVENTURE_IS_COMING.php |archivedate=2015-05-18 |deadurl=no |language=en }}</ref>，游戏中的恐龙可以通过喂食来将其驯服，一旦恐龙被驯服，玩家将可以使用恐龙的能力，如飞行或快速水下移动，以及它的统计数据如速度，健康和伤害等<ref name="Wire"/>，使用这些能力时，玩家将不能控制他们的个人武器。当所骑的恐龙较大时，其他玩家可以骑在恐龙身上并且控制他们的武器<ref>{{harvnb|Rapczak|2015|loc=36:27}}</ref>。恐龙也可以用来携带物品，玩家可以给它们下达攻防命令，例如：一群[[犹他盗龙|犹他盗龙]]可以被用来防御基地，一群[[三角龙|三角龙]]可以被用来攻击敌军基地<ref name="Wire">{{cite web |url=http://news.xbox.com/2015/05/games-fight-for-life-in-studio-wildcards-ark-survival-evolved |title=Fight for Life in Studio Wildcard's ARK: Survival Evolved |work=Xbox Wire |publisher=[[Xbox|Xbox]]. [[Microsoft|Microsoft]] |date=2015-05-15 |accessdate=2015-08-14 |archiveurl=https://www.webcitation.org/6YbL3yoJZ?url=http://news.xbox.com/2015/05/games-fight-for-life-in-studio-wildcards-ark-survival-evolved |archivedate=2015-05-17 |deadurl=no |language=en }}</ref>。游戏中还有各种其他史前物种，如[[渡渡鸟|渡渡鸟]]、[[剑齿虎|剑齿虎]]、[[真猛玛象|真猛玛象]]和巨大的昆虫如[[巨脉蜻蜓|巨脉蜻蜓]]和[[巨蟻|巨蟻]]。游戏中的每一个生物都生活在一定的生态系统和捕食者结构中<ref name="PS announcement"/>。

== 开发 ==
《方舟：生存进化》的前期制作开始于2014年10月<ref>{{harvnb|Rapczak|2015|loc=7:38}}</ref>，位于[[西雅图|西雅图]]的游戏开发商Studio Wildcard选择了埃及开发商Instinct Games作为其合作伙伴共同开发游戏<ref name="Mac-Lin"/>。在研究游戏的史前物种信息时，开发团队阅读了“普通书籍”和网上的文章，并寻求在生物科学领域有研究的朋友的帮助<ref name="FAQ"/>。当创建物种和世界时，尽管有些物种在其对应的历史时期中早已进化了，但开发团队设计的游戏玩法仍然很有新意<ref name="FAQ">{{cite web |url=http://steamcommunity.com/app/346110/discussions/0/613957600545122958/ |title=Frequently Asked Questions |last=Karunakaran |first=Jathiesh |publisher=[[Steam|Steam]]、[[维尔福|维尔福]] |date=2015-05-12 }}</ref>。游戏最初通过{{link-en|Steam抢先体验|Early access}}于2015年6月2日在[[Microsoft_Windows|Microsoft Windows]]上发布<ref name="Press1"/>，7月1日在[[Linux|Linux]]和[[OS_X|OS X]]发布<ref name="Mac-Lin"/>，12月16日通过Xbox游戏预览计划在[[Xbox_One|Xbox One]]发布<ref>{{cite web |url=http://www.gamespot.com/articles/xbox-one-getting-dino-survival-game-ark-this-winte/1100-6429445/ |title=Xbox One }}</ref>。游戏最终版于2017年在Linux，OS X，Microsoft Windows，[[PlayStation_4|PlayStation 4]]和[[Xbox_One|Xbox One]]发布<ref name="Mac-Lin"/>。游戏将支持[[Oculus_Rift|Oculus Rift]]和[[PS_VR|PS VR]]<ref name="Press1"/>，Xbox One版本则通过{{le|ID@Xbox|ID@Xbox}}计划发布<ref name="Mac-Lin">{{cite press release |archivedate=2015年7月2日 |deadurl=yes |language=en |df= }}</ref>。游戏之后于2018年6月14日在[[iOS|iOS]]和[[Android|Android]]发布，并将于2018年秋季在[[任天堂Switch|任天堂Switch]]发布。

开发团队指出，游戏将吸引所有玩家，而不是只喜欢生存模式的玩家<ref>{{harvnb|Rapczak|2015|loc=38:59}}</ref>。团队也为玩家增加了结局，因为他们认为大多数生存游戏缺乏一个最终目标，创意总监傑西·韋帕札克（Jesse Rapczak）说他们想“提供一个深度和广度，让这个世界不止意味着结局，而是一个值得探索的地方。”<ref name="Wire"/>

游戏搭载[[虚幻引擎|虚幻4]]引擎，韋帕札克称其包含“数以万计”的{{le|人工智能 (电子游戏)|Artificial intelligence (video games)|人工智能}}实体<ref name="PS announcement"/>。游戏还支持[[虚拟现实|虚拟现实]]，韋帕札克拥有近三年[[头戴式显示器|头戴式显示器]]经验，他声称游戏一开始就基于虚拟现实设计<ref name="PS announcement">{{cite web |url=http://blog.eu.playstation.com/2015/05/11/ride-dinosaurs-in-ark-survival-evolved-coming-soon-to-ps4/ |title=Ride dinosaurs in ARK: Survival Evolved, coming soon to PS4 |last=Rapczak |first=Jesse |work={{le|PlayStation Blog|PlayStation Blog}} |publisher=[[Sony_Computer_Entertainment|Sony Computer Entertainment]] |date=2015-05-11 |accessdate=2015-08-14 |archiveurl=https://www.webcitation.org/6Z0U5Mo7R?url=http://blog.eu.playstation.com/2015/05/11/ride-dinosaurs-in-ark-survival-evolved-coming-soon-to-ps4/ |archivedate=2015-06-03 |deadurl=no |language=en }}</ref>。
