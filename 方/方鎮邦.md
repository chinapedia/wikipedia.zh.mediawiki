'''方鎮邦'''（'''Stephen Fong Chun-bong'''），[[香港|香港]]前政治人物，前民主黨及自由黨成員，第二屆[[沙田區議會|沙田區議會]]頌安選區民選[[區議員|區議員]]。方鎮邦因涉桃色糾紛而退黨轉投[[自由黨_(香港)|自由黨]]及一度失蹤。<ref>[http://the-sun.on.cc/channels/news/20060515/20060515023541_0000.html 前民主黨區議員方鎮邦涉桃色失蹤]</ref>方鎮邦因涉嫌騙取區議會營運開支津貼，被[[廉政公署|廉政公署]]拘捕及起訴。<ref>[http://hk.news.yahoo.com/070704/12/2asew.html 沙田區議員方鎮邦涉騙津貼]</ref>

== 簡歷 ==
方鎮邦原本任職銀行外匯部，2000年因法團事務認識民主黨區議員，後來加入民主黨，任地區幹事。<ref name="s20070705">[http://the-sun.on.cc/channels/news/20070705/20070705033622_0000.html 《涉騙薪津 轉投自由黨遭開除 前民主黨區議員方鎮邦被捕》]，香港《太陽報》，2007年7月5日。</ref>

在[[2003年香港區議會選舉|2003年香港區議會選舉]]，方在沙田頌安選區以59票之差擊敗競逐連任的[[民建聯|民建聯]][[陳克勤|陳克勤]]<ref>{{cite web |url=http://www.elections.gov.hk/elections/dc2003/tc_chi/result/result_st.html |title=存档副本 |accessdate=2007-04-03 |deadurl=yes |archiveurl=https://web.archive.org/web/20070930160242/http://www.elections.gov.hk/elections/dc2003/tc_chi/result/result_st.html |archivedate=2007-09-30 }}</ref>，成為民選區議員。<ref>[http://www.info.gov.hk/gia/general/200311/24/1124076.htm ４００名當選區議員名單]</ref>

方其後因涉及桃色事件，於2006年1月被要求退出民主黨，隨即轉投[[自由黨|自由黨]]。<ref name="s20060515">[http://the-sun.on.cc/channels/news/20060515/20060515023541_0000.html 《前民主黨區議員 上月起缺席會議 方鎮邦涉桃色失蹤》]，香港《太陽報》，2006年5月15日。</ref>

== 是非 ==
方鎮邦的議員辦事處設於[[頌安邨|頌安邨]]頌智樓地下。方在2006年期間曾經失蹤，缺席多次區議會會議。<ref name="s20060515"/>其議員辦事處後來也因欠租，被房屋署收回。<ref name="s20070705"/>

方鎮邦被自由黨開除黨籍。[[廉政公署_(香港)|廉政公署]]於2007年7月4日拘捕及起訴方，指他涉嫌騙取區議會營運開支津貼合共14,000元，訛稱用作支付辦公室助理薪金。<ref name="s20070705"/>方於[[沙田裁判法院|沙田裁判法院]]被判兩項欺詐罪名成立，11月26日被判即時監禁4個月，並褫奪議席。<ref>[http://www.takungpao.com/news/07/11/27/GW-828995.htm 《區議員呃津囚四月》] {{webarchive|url=https://web.archive.org/web/20071205125120/http://www.takungpao.com/news/07/11/27/GW-828995.htm |date=2007-12-05 }}，香港《大公報》，2007年11月27日。</ref>
