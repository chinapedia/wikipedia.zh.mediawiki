'''方濟各-伯多祿·特巴茲-凡·埃爾斯特'''（{{lang-de|Franz-Peter Tebartz-van Elst}}；{{bd|1959年|11月22日|catIdx=Franz-Peter Tebartz-van Elst}}），是[[德國|德國]]籍[[天主教|天主教]][[主教|主教]]，也是[[天主教林堡教區|林堡教區榮休主教]]。

==生平==
保理加布生於1959年11月22日，[[西德|西德]][[北萊茵-威斯特法倫州|北萊茵-威斯特法倫州]]西北部的[[克萊韋縣|克萊韋縣]]。他於1985年5月26日晉鐸。

2004年1月18日晉牧，並被時任[[教宗若望·保祿二世|教宗若望·保祿二世]]任命為[[天主教明斯特教區|明斯特教區輔理主教]]。

1998年3月24日，他接替成為[[天主教林堡教區|林堡教區正㩲主教]]。但他上任後因奢華的消費習慣和其他財務上違規的行為，而受到外界抨擊。2013年10月20日，埃爾斯特根據教會法典向時任[[教宗方濟各|教宗方濟各]]提出辭呈林堡教區正㩲主教之職責，並在2014年3月26日獲得教宗准許。<ref name=>{{cite web |url=http://saltandlighttv.org/chinese/%E6%95%99%E5%AE%97%E6%96%B9%E6%BF%9F%E5%90%84%E6%8E%A5%E5%8F%97%E6%9E%97%E5%A0%A1%E6%95%99%E5%8D%80%E4%B8%BB%E6%95%99%E8%BE%AD%E8%81%B7/|title=教宗方濟各接受林堡教區主教辭職 |publisher= 鹽與光 |accessdate=2014年3月25日}}</ref><ref>{{cite web |url=http://china.ucanews.com/2016/05/06/%E6%95%99%E5%AE%97%E7%82%BA%E5%8F%A4%E5%B7%B4%E4%BB%BB%E5%91%BD%E6%96%B0%E7%B8%BD%E4%B8%BB%E6%95%99%EF%BC%8C%E6%98%AF%E4%B8%80%E7%B3%BB%E5%88%97%E9%87%8D%E8%A6%81%E4%BB%BB%E5%91%BD%E7%9A%84%E7%AC%AC/ |title=教宗為古巴任命新總主教，是一系列重要任命的第一步 |publisher= [[天主教亞洲通訊社|天主教亞洲通訊社]] |accessdate=2016年5月6日}}</ref>教區主教一直懸空至2016年7月，最後由{{link-en|喬治·貝青|Georg Bätzing}}繼任。<ref>{{cite news|url=https://www.zeit.de/gesellschaft/zeitgeschehen/2016-07/bistum-limburg-papst-franziskus-vatikan-bischof-georg-baetzing|title=Georg Bätzing folgt auf Tebartz-van Elst|publisher=Zeit Online|language=de|date=2016-07-01|accessdate=2018-12-02|archivedate=2018-12-02|archiveurl=https://web.archive.org/web/20181202062313/https://www.zeit.de/gesellschaft/zeitgeschehen/2016-07/bistum-limburg-papst-franziskus-vatikan-bischof-georg-baetzing|}}</ref>
