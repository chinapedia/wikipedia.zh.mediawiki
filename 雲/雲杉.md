'''雲杉'''（學名：{{lang|la|''Picea asperata''}}）为[[松科|松科]][[雲杉屬|雲杉屬]]的一种，是原產於[[中國西部|中國西部]]的特有樹種，主要分布於[[青海|青海]]東部、[[甘肅|甘肅]]南部、[[陝西|陝西]]西南與南部、到[[四川|四川]]西部一帶<ref name=Conifers/>。

== 生態 ==
雲杉為多年生長綠[[喬木|喬木]]，棲息在[[海拔|海拔]]{{convert|2000|m|ft}}到{{convert|3800|m|ft}}的[[山脈|山區]]與[[河谷|河谷]]，[[抗寒區|抗寒區]]（{{lang|en|Hardiness Zones}}）為 6 至 7 級區，需有足夠日照，溼潤環境，排水良好、中性至酸性的土壤，能在貧瘠或強酸土質生長，但不能適應空氣污染環境，生長緩慢，自然環境中常聚生成廣大的純林，遠望有如白雲，故稱雲杉<ref name=NCU/><ref name=EOT/><ref name=PFAF/>。

== 形態 ==
*[[樹幹|樹幹]]：高可達{{convert|45|m|ft}}、寬可達{{convert|100|cm|in}}，表面覆蓋著不規則的[[鱗片|鱗片狀]]木皮，緊密而粗糙，呈[[灰褐色|灰褐色]]<ref name=Conifers/>。
*[[樹枝|樹枝]]：[[棕黃色|棕黃色]]或[[棕紅色|棕紅色]]，變成[[褐色|褐色]]或灰褐色<ref name=Conifers/>。
*[[葉|葉]]：[[針|針狀]]葉片沿小枝往上向前層疊生長，下沿葉片逐漸橫向展開，葉片表面可能覆有白霜、微彎，截面呈四角型，長約{{convert|10|mm|in|2}}至{{convert|20|mm|in|2}}，寬約{{convert|1|mm|in|2}}至{{convert|2|mm|in|2}}，每邊有 4 到 8 個氣孔，尾端尖銳<ref name=Conifers/>。
*[[花|花]]：每年4月開花，雌雄異花同株，籍風授粉<ref name=PFAF/>
*[[孢子叶球|孢子叶球]]：初果[[綠色|綠色]]、成熟時[[淺棕色|淺棕色]]或[[紅褐色|紅褐色]]，[[圓柱體|圓柱型]]毬果，長約{{convert|5|cm|in}}至{{convert|16|cm|in}}，直徑約{{convert|2.5|cm|in|1}}至{{convert|3.5|cm|in|1}}，由帶有種子的種鱗組成，種鱗[[倒卵形|倒卵形]]，長約{{convert|2|cm|in|2}}，寬約{{convert|1.5|cm|in|2}}，邊緣半整或鋸齒狀<ref name=Conifers/>。
*種子：呈倒卵形，長約{{convert|4|mm|in|2}}，翼寬{{convert|11|mm|in|2}}，每年9至10月種子成熟<ref name=Conifers/>。

== 分類 ==
依[[DNA|DNA]]分析，雲杉為[[雲杉屬|雲杉屬]]第五[[演化支|演化支]]物種，目前已知的[[變種|變種]]有<ref name=Ran/><ref name=Sigurgeirsson/>：
*雲杉（{{lang|la|''Picea asperata'' var. ''asperata''}}）：毬果長約{{convert|6|cm|in}}至{{convert|12|cm|in}}，尾端呈[[圓錐形|圓錐形]]。
*[[橙色雲杉|橙色雲杉]]（{{lang|la|''Picea asperata'' var. ''aurantiaca''}}，即{{lang|la|''P. aurantiaca''}}）：枝芽呈[[橙色|橙色]]。
*[[異鱗雲杉|異鱗雲杉]]（又名茂縣雲杉<ref name=Wang/>，{{lang|la|''Picea asperata'' var. ''heterolepis''}}，即{{lang|la|''P. heterolepis''}}）：枝芽無毛，毬果尾端呈[[菱形|菱形]]。
*[[大果雲杉|大果雲杉]]（{{lang|la|''Picea asperata'' var. ''ponderosa''}}）：毬果較大，長約{{convert|12|cm|in}}至{{convert|15|cm|in}}。
*[[卷雲杉|卷雲杉]]（又名箭爐雲杉<ref name=Wang/>，{{lang|la|''Picea asperata'' var. ''retroflexa''}}，即{{lang|la|''P. retroflexa''}}）：枝芽帶[[黃|黃]]。

== 保護 ==
雲杉雖沒有被列為受威脅的[[物種|物種]]，惟由於[[伐木|伐木業]]毁林濫採，造成其數量不斷下降<ref name=iucn/>。

== 用途 ==
雲杉是木結他面板中最常見，木紋筆直細密，能提供清晰響亮的音色，
高韌度帶來良好的共嗚和延音，偏白的色澤也是其賣點之一。
