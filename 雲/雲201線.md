'''雲201線 東和－水碓'''，是位於[[雲林縣|雲林縣]]的一條鄉道，北起[[雲林縣|雲林縣]][[古坑鄉|古坑鄉]]東和，南至[[雲林縣|雲林縣]][[古坑鄉|古坑鄉]]水碓，全長3.647公里。{{r|thb}}有一支線雲201-1線。

== 路線說明 ==

;雲林縣
*（早期路線）斗六市：'''斗六'''（舊起點，[[File:TW_PHW3.svg|23px]][[台3線|台3線]]舊線岔路）→ 中華路 → → 永安路 → 成功路（雲214線岔路） → '''五里亭'''（雲215線岔路） →鄉界
*古坑鄉：鄉界 → 文化路 → '''東和'''文化路 0.0k（起點，[[File:TW_CHW149a.svg|30px]][[縣道149號#.E7.94.B2.E7.B7.9A|149甲線]]岔路）→ 嘉興路路口（[[File:TW_THWyl198.png|23px]]雲198線岔路） → 廣濟路路口1.0（[[File:TW_THWyl213.png|23px]]雲213線岔路）右轉 → 長安路路口（[[File:TW_THWyl198-1.png|23px]]雲198-1線岔路） → 廣濟路 → '''高林'''1.9（[[File:TW_THWyl201-1.png|23px]]雲201-1線岔路） →水  3.4 [[File:TW_THWyl200.png|23px]]雲200線岔路 → '''水碓'''3.647k（終點，[[File:TW_CHW154b.svg|30px]][[縣道154號#.E4.B9.99.E7.B7.9A|154乙線]]岔路）

'''雲201-1線 高林－荷苞'''，是位於[[雲林縣|雲林縣]]的一條鄉道，東起[[雲林縣|雲林縣]][[古坑鄉|古坑鄉]]高林村高林（林子頭），西至[[雲林縣|雲林縣]][[古坑鄉|古坑鄉]]荷苞村荷苞，全長1.329公里。{{r|thb}}為雲201線的支線。
