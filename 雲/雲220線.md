'''雲220線 龍頭－石壁'''，是位於[[中華民國|中華民國]]（[[臺灣|臺灣]]）[[雲林縣|雲林縣]]的一條鄉道，東起[[雲林縣|雲林縣]][[古坑鄉|古坑鄉]]龍頭，西至[[雲林縣|雲林縣]][[古坑鄉|古坑鄉]]石壁，全長4.919公里。{{r|thb}}有一支線雲220-1線。

== 路線說明 ==

本線編號為目前已知鄉道中編號最大者（另有尚未確認之雲221線～雲223線），若[[File:TW_CHW149a.svg|30px]][[縣道149號#.E7.94.B2.E7.B7.9A|149甲線]]因天災中段可與[[File:TW_THWyl220-1.png|23px]]雲220-1線一同作為替代道路。

;雲林縣

*古坑鄉： '''東和'''文化路 0.0k（起點，[[File:TW_CHW149a.svg|30px]][[縣道149號#.E7.94.B2.E7.B7.9A|149甲線]]岔路）→ 嘉興路路口（[[File:TW_THWyl198.png|23px]]雲198線岔路） → 廣濟路路口（[[File:TW_THWyl213.png|23px]]雲213線岔路）右轉 → 長安路路口（[[File:TW_THWyl198-1.png|23px]]雲198-1線岔路） → 廣濟路 → '''高林'''（[[File:TW_THWyl201-1.png|23px]]雲201-1線岔路） → [[File:TW_THWyl200.png|23px]]雲200線岔路 → '''水碓'''3.647k（終點，[[File:TW_CHW154b.svg|30px]][[縣道154號#.E4.B9.99.E7.B7.9A|154乙線]]岔路）

== 沿革 == 

本線編號至1986年皆為空號{{r|gazette1966}}{{r|gazette1976}}，其路段前段則於1961年即已編入[[File:TW_THWyl208.png|23px]][[雲208線|雲208線]]{{r|gazette1966}}，於1983年可能改併入[[File:TW_CHW149a.svg|30px]][[縣道149號#.E7.94.B2.E7.B7.9A|149甲線]]，而後於1997年因草嶺隧道貫通，本路線及通往石壁之道路改編為本編號所有。

'''雲220-1線 草嶺－龍頭'''，是位於[[雲林縣|雲林縣]]的一條鄉道，南起[[雲林縣|雲林縣]][[古坑鄉|古坑鄉]]草嶺，北至[[雲林縣|雲林縣]][[古坑鄉|古坑鄉]]龍頭，全長5.655公里。{{r|thb}}為雲220線的支線。

=== 歷史沿革 ===
本線編號至1986年皆為空號{{r|gazette1966}}{{r|gazette1976}}，其路段前段則可能於1983年改編入[[File:TW_CHW149a.svg|30px]][[縣道149號#.E7.94.B2.E7.B7.9A|149甲線]]，而後於1997年因草嶺隧道貫通，本路線改編為本編號所有。
