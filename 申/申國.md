'''申国'''是[[周代|周代]]的一个[[诸侯|诸侯]]国，[[姜|姜]]姓<ref>宋衷 注，秦嘉謨等 辑，《世本八种》：“許、州、向、申、姜姓也，炎帝後。”，商务印书馆</ref>，封爵为[[伯爵|伯爵]]。申国為[[申伯|申伯]]在[[謝國|謝國]]舊地所建。[[春秋|春秋]]初期为[[楚文王|楚文王]]吞并，其地被[[楚国|楚国]]设置为[[县|县]]。

== 疆域 ==
申国在[[陈|陈]]（今[[河南|河南]][[淮阳|淮阳]]）、[[郑|郑]]（今[[河南|河南]][[新郑|新郑]]）的南方<ref>郑玄《诗笺》：“平王母家申國，在陳、鄭之南，迫近彊楚。”</ref>。據《汉书·地理志》记载，申国的都城在[[南阳|南阳]][[宛县|宛县]]（今河南[[南阳市|南阳市]]），在冥厄（冥厄关，今[[湖北|湖北]][[广水市|广水市]]和[[河南|河南]][[信阳县|信阳县]]交界处）之北，淮水（[[淮河|淮河]]）之南<ref>《汉书·地理志》：“申在南阳宛县，申伯之国，吕太公所封，……”，“申之疆域，在冥厄之北，淮水之南”。</ref>。

== 历史 ==
申国的肇始是周王朝的申伯，申伯是[[周天子|周天子]]的[[舅父|舅父]]。[[周宣王|周宣王]]時，天子命[[召穆公|召穆公]]滅了[[謝國|謝國]]（今[[河南省|河南省]][[南阳市|南阳市]]），为镇抚[[楚国|楚国]]等南方[[诸侯|诸侯]]，於是命[[申伯|申伯]]在謝國舊地建立[[申国|申国]]，作为镇抚南方的军事重镇，因申伯國於謝，從此又稱[[謝國|謝國]]。

宣王还亲自前往郿（今[[陕西省|陕西省]][[眉县|眉县]]东北）为申伯践行，赐予车马及玉圭，申伯成为申国的开国君主，此事迹记载于《[[诗经|诗经]]·大雅·嵩高》中。<ref>《诗经·大雅·崧高》:「王命召伯，定申伯宅。……式是南邦，因是謝人，以作爾庸。」</ref>

申国的封地是周王朝控制南方的门户重镇。宣王还派[[召伯虎|召伯虎]]去为申伯经营，彻取村社土地的一部分作为公田，还派遣近臣[[傅御|傅御]]把申伯的原有的臣屬、農民迁去，以便他迅速建立统治地位<ref>《诗经·大雅·崧高》:「王命召伯，彻申伯土田，王命傅御，迁其私人。……申伯番番，既入于谢，徒御啴啴，周邦咸喜，戎有良翰。不显申伯，王之元舅，文武是宪。」</ref>。

[[周幽王|周幽王]]时，王后是申国申侯的女儿。幽王因[[褒姒|褒姒]]的缘故废了王后以及太子[[宜臼|宜臼]]而改立褒姒的儿子[[伯服|伯服]]为太子，使得申后的父亲[[申侯|申侯]]震怒。公元前771年，申侯联合[[缯国|缯国]]和[[犬戎|犬戎]]一起进攻幽王。幽王用烽火召集诸侯，而无人响应，最后幽王被杀于[[骊山|骊山]]下。申侯、[[魯孝公|鲁侯]]和[[许文公|许文公]]在申国另立了周平王，也就是幽王的太子[[宜臼|宜臼]]<ref>《史记·周本纪》：“（幽王）废申后，去太子也。申侯怒，与缯、西夷犬戎攻幽王。幽王举烽火征兵，兵莫至。遂杀幽王骊山下，虏褒姒，尽取周赂而去。”</ref><ref>《竹书纪年》：“幽王八年，立褒姒之子伯服为太子。”、“太子与幽王俱死于戏。先是，申侯、鲁候及许文公立平王于申，以本太子故称天王。幽王既死，而虢公翰又立王子余臣于携，周二王并立。”</ref>。

《[[左传|左传]]·隐公元年》记载，[[郑武公|郑武公]]夫人娶自申国，称为“武姜”。

[[楚国|楚国]]于春秋初年开始扩张，[[楚文王|楚文王]]于公元前688年（[[鲁庄公|鲁庄公]]六年）讨伐申国，据《[[左传|左传]]》记载，讨伐途中经过[[邓国|邓国]]，[[邓祁侯|邓祁侯]]说：“楚文王是我的外甥”，于是让他停在邓国，设宴款待他。騅甥，聃甥和養甥请求邓祁侯杀掉楚王，邓祁侯不许，于是三人说：“亡邓国者，必是此人。等到他灭了申国，再来灭邓国，就好像咬噬您的肚脐一样，再也来不及图谋应付了。要杀他，只有乘这个时候。”楚文王这次讨伐申国后陷入与[[邓国|邓国]]和[[巴国|巴国]]的战争，但这次伐申的结果不明。<ref>《左传·庄公六年》，又见《左传·庄公十八年》</ref> 

《左传·哀公十七年》追溯了[[楚文王|楚文王]]“实县申息”的功业。[[息国|息国]]的灭亡至晚在公元前680年<ref>《左传·庄公十四年》</ref>。因此，申国的灭亡在楚文王伐申之后，约在[[息国|息国]]灭亡时间左右，大致在公元前688年至公元前680年。

申国被并入楚国后，成为楚国北方重要的县。[[城濮之战|城濮之战]]中，楚国令尹[[子玉|子玉]]并未率楚国全部主力出征，而是以申息两县的兵力为主。因此子玉战败，楚成王有言：“你若回国，申息两县的父老怎么办？<ref>《左传·僖公二十八年》：“既败，王使谓之曰：大夫若入，其若申、息之老何？”</ref>”

公元前594年，[[楚庄王|楚庄王]]同意以申和吕两地部分土地作为[[子重|子重]]的赏田。而当时楚国申县的长官[[巫臣|巫臣]]劝谏，说申和吕是楚国北方边境重要的军事重地和兵源地，如果以申吕为赏田而不是楚王直辖，那么就丧失了这两地的军事功能，[[晋国|晋国]]和[[郑国|郑国]]就必然突破边境而攻击到楚国的腹地[[汉江|汉江]]流域<ref>《左传·成公七年》：“楚圍宋之役，師還，子重請取於申呂，以為賞田，王許之，申公巫臣曰，不可，此申呂所以邑也，是以為賦，以御北方，若取之，是無申呂也，晉鄭必至于漢。”</ref>。

公元前585年(鲁成公六年)，晋国讨伐[[蔡国|蔡国]]，楚国以申息的兵力救蔡。晋国将领认为，此战胜利则仅仅是击败楚国的两县而已，失败更为耻辱，因此主动撤退<ref>《左传·成公六年》</ref>。[[顾颉刚|顾颉刚]]指出，以申息两县的兵力即足以和霸主之国晋国的军队周旋，可见申息两县的富庶。<ref>顾颉刚：《春秋时代的县》，《禹贡》第7卷第6、7期合期，1937年6月。转引自陈苏镇、张帆编；《中国古代史读本（上册）》，北京大学出版社2006年1月第1版，第112页。</ref>

公元前529年，[[楚灵王|楚灵王]]死，[[楚平王|楚平王]]立。当年楚灵王灭蔡国后，灵王将[[许国|许国]]、[[胡国|胡国]]、[[沈国|沈国]]、[[道国|道国]]、[[房国|房国]]和申国迁徙至楚国境内，楚平王即位后使陈国和蔡国复国，而这些被迁徙的小国，也都復國。申国正在复国之列<ref>《左传·昭公十三年》：“楚之滅蔡也，靈王遷許，胡，沈，道，房，申，於荊焉，平王即位，既封陳蔡，而皆復之。”</ref>。

== 青铜器铭文记载 ==

1980年代，河南省南阳市出土一批申国[[青铜器|青铜器]]，其中包括仲爯父簋，其铭文经过[[李学勤|李学勤]]释读有“南申伯”，并认为铭文在“申”前冠以“南”字可能是为了与“西申”相别，或在两周之际可能还有除申伯之国另外的申国存在，即《古本竹书纪年》所载之“西申”（“申侯之国”）。李学勤进一步推论，这位南申伯仲爯父就是《[[诗经|诗经]]·大雅·荡·崧高》中的“申伯”。<ref>李学勤：《论仲爯父簋与申国》，《中原文物》1984年04期。转引自熊逸：《春秋大义2隐公元年》，广西师范大学出版社2009年1月第1版，第330页。</ref>{{R|春秋变迁|page1=43—45}}
