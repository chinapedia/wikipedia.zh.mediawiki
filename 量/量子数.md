'''量子數'''描述[[量子力學|量子系統]]中動力學上各守恒數的值。它們通常按性質描述[[原子|原子]]中[[電子|電子]]的各[[能量|能量]]，但也會描述其他物理量（如[[角動量|角動量]]、[[自旋|自旋]]等）。由於任何量子系統都能有一個或以上的量子數，列出所有可能的量子數是件沒有意義的工作。

== 有多少個量子數？ ==
「要多少個量子數才能描述任何已知系統？」這道問題並沒有一致的答案，儘管要解決每一個系統都必需要對系統進行全面分析。任何系統的動力學都由一量子[[哈密頓算符|哈密頓算符]]，'''H'''，所描述。系統中有一量子數對應能量，即哈密頓算符的[[特徵值|特徵值]]。對每一個算符'''O'''而言，還有一個量子數可與哈密頓算符交換（即滿足'''OH = HO'''這條關係式）。這些是一個系統中所能有的所有量子數。注意定義量子數的算符'''O'''應互相獨立。很多時候，能有好幾種選擇一組互相獨立算符的方法。故此，在不同的條件下，可使用不同的量子數組來描述同一個系統。

== 原子內的單個電子 ==
{{本節並非該問題的完整描述。詳見[[類氫原子|類氫原子]]、[[波耳原子|波耳原子]]、[[薛定諤方程|薛定諤方程]]及[[狄拉克方程|狄拉克方程]]。}}

最被廣為研究的量子數組是用於一[[原子|原子]]的單個[[電子|電子]]：不只是因為它在[[化學|化學]]中有用（它是[[週期表|週期表]]、[[化合價|化合價]]及其他一系列特性的基本概念），還因為它是一個可解的真實問題，故廣為教科書所採用。

在非相對論性[[量子力學|量子力學]]中，這個系統的哈密頓算符由電子的[[動能|動能]]及[[勢能|勢能]]（由電子及原子核間的[[庫侖定律|庫侖力]]所產生）。動能可被分成，有環繞[[原子核|原子核]]的電子[[角動量|角動量]]，'''J'''的一份，及餘下的一份。由於勢能是球狀對稱的關係，其完整的哈密頓算符能與'''J'''<sup>2</sup>交換。而'''J'''<sup>2</sup>本身能與角動量的任一分量（按慣例使用'''J'''<sub>z</sub>）交換。由於這是本題中唯一的一組可交換算符，所以會有三個量子數。

依慣例，它們被稱為：
* [[主量子數|主量子數]]（n=1，2，3，4 …）代表除掉'''J'''<sup>2</sup>以後'''H'''的特徵值。這個數因此會視電子與原子核間的距離（即半徑座標'''r'''）而定。平均距離會隨着'''n'''增大，因此不同量子數的量子態會被說成屬於不同的電子層。
* [[角量子數|角量子數]]（l=0，1 … n-1）（又稱'''方位角量子數'''或'''軌道量子數'''）通過關係式<math>L^2 = \hbar^2 l(l+1)</math>來代表軌道角動量。在化學中，這個量子數是非常重要的，因為它表明了一軌道的形狀，並對[[化學鍵|化學鍵]]及[[鍵角|鍵角]]有重大影響。有些時候，不同角量子數的軌域有不同代號，'''l=0'''的軌域叫s軌域，'''l=1'''的叫p軌域，'''l=2'''的叫d軌域，而'''l=3'''的則叫f軌域。
* [[磁量子數|磁量子數]]（m<sub>l</sub>= -l，-l+1 … 0 … l-1，l）代表[[特徵值|特徵值]]，<math>L_z = m_l \hbar </math><ref>{{cite book|author=Arthur Beiser, Kok Wai Cheah|title=''Concepts of modern physics''|year=2015|publisher=[[麥格羅-希爾集團|麥格羅-希爾集團]]|isbn=9789814595261|pages=第229頁}}</ref>。這是軌道[[角動量|角動量]]沿某指定軸的投影。

從[[光譜學|光譜學]]中所得的結果指出一個軌道最多可容納兩個電子。然而兩個電子絕不能擁有完全相同的量子態（[[泡利不相容原理|泡利不相容原理]]），故也絕不能擁有同一組量子數。所以為此特別提出一個假設來解決這問題，就是設存在一個有兩個可能值的第四個量子數。這假設以後能被相對論性量子力學所解釋。

*[[自旋量子數|自旋量子數]]（m<sub>s</sub>= -1/2 或 +1/2）代表電子的固有[[角動量|角動量]]。這是[[自旋|自旋]] s=1/2沿某指定軸的射影。
作為摘要，一電子的量子態視下列各量子數而定：

例：用於描述[[氟|氟]]（F）原子最外層電子（即[[價電子|價電子]]，位於[[原子軌道|原子軌道]]2p）的各量子數值為：n=2，l=1，m<sub>l</sub>=1或0或-1，m<sub>s</sub>=-1/2或1/2。

注意[[分子軌道|分子軌道]]需要使用完全不同的量子數組，因為其[[哈密頓算符|哈密頓算符]]及對稱跟上述相當不同。

== 適用於自旋－軌道交互作用的量子數 ==
{{詳見[[克里布希－戈登係數|克里布希－戈登係數]]}}

當考慮到[[自旋-軌道作用|自旋-軌道作用]]時，l、m及s就再不能與[[哈密頓算符|哈密頓算符]][[交換律|交換]]，因而它們的值會隨時間改變。故應該使用另一組[[量子數|量子數]]。這組包括了
*[[總角動量量子數|總角動量量子數]]（j=1/2，3/2 … n-1/2）通過關係式<math>J^2 = \hbar^2 j(j+1)</math>代表着總[[角動量|角動量]]。
*[[次要總角動量量子數|總角動量沿某指定軸的投影]]（m<sub>j</sub>=-j，-j+1 … j-1，j），此數與m類似，且滿足關係式<math>m_j = m_l + m_s</math>。
*[[宇稱|宇稱]]。它是經反射所得的[[特徵值|特徵值]]，當態之l為偶數時其值為正（即+1），奇數時其值為負（即-1）。前者亦被稱為'''偶宇稱'''，後者則為'''奇宇稱'''。

例：考慮以下八個態，定義它們的量子數：
# l = 1，m<sub>l</sub> = 1，m<sub>s</sub> = +1/2
# l = 1，m<sub>l</sub> = 1，m<sub>s</sub> = -1/2
# l = 1，m<sub>l</sub> = 0，m<sub>s</sub> = +1/2
# l = 1，m<sub>l</sub> = 0，m<sub>s</sub> = -1/2
# l = 1，m<sub>l</sub> = -1，m<sub>s</sub> = +1/2
# l = 1，m<sub>l</sub> = -1，m<sub>s</sub> = -1/2
# l = 0，m<sub>l</sub> = 0，m<sub>s</sub> = +1/2
# l = 0，m<sub>l</sub> = 0，m<sub>s</sub> = -1/2

系統的[[量子態|量子態]]能被這八個態的線性組合所描述。但由於[[自旋-軌道作用|自旋-軌道作用]]的關係，如欲使用八個由[[哈密頓算符|哈密頓算符]]的[[特徵向量|特徵向量]]（即每一個代表一個態且不會因時間而跟其他態混合）所組成的態來描述同一個系統，應考慮以下這八個態：
# j = 3/2， m<sub>j</sub> = 3/2，奇宇稱 （從上態1得）
# j = 3/2， m<sub>j</sub> = 1/2，奇宇稱 （從上態2及3得）
# j = 3/2， m<sub>j</sub> = -1/2，奇宇稱 （從上態4及5得）
# j = 3/2， m<sub>j</sub> = -3/2，奇宇稱 （從上態6得）
# j = 1/2， m<sub>j</sub> = 1/2，奇宇稱 （從上態2及3得）
# j = 1/2， m<sub>j</sub> = -1/2，奇宇稱 （從上態4及5得）
# j = 1/2， m<sub>j</sub> = 1/2，偶宇稱 （從上態7得）
# j = 1/2， m<sub>j</sub> = -1/2，偶宇稱 （從上態8得）

== 基本粒子 ==
:<small> 如欲見更完整的基本粒子量子態敍述，可參閱[[標準模型理論|標準模型理論]]及[[味_(粒子物理學)|味 (粒子物理學)]]</small>
基本粒子包含不少量子數，一般來說它們都是粒子本身的。但需要明白的是，基本粒子是[[粒子物理學|粒子物理學]]上[[標準模型理論|標準模型]]的量子態，所以這些粒子量子數間的關係跟模型的哈密頓算符一樣，就像[[波耳原子|波耳原子]]量子數及其哈密頓算符的關係那樣。亦即是說，每一個量子數代表問題的一個對稱性。這在[[量子場論|場論]]中有着更大的用處，被用於識別[[時空|時空]]及[[內|內]]對稱。

一般跟[[時空對稱|時空對稱]]有關係的量子數有[[自旋|自旋]]（跟旋轉對稱有關）、[[宇稱|宇稱]]、[[C-宇稱|C-宇稱]]、[[T-宇稱|T-宇稱]]（跟時空上的[[龐加萊對稱|龐加萊對稱]]有關係）。一般的內對稱有[[輕子數|輕子數]]、[[重子數|重子數]]及[[電荷|電荷數]]。條目[[味_(粒子物理學)|味]]有這些量子數的更詳細列表。

值得一提的是較次要但常被混淆的一點。大部分守恒量子數都是可相加的。故此，在一基本粒子反應中，反應前後的量子數總和應相等。然而，某些量子數（一般被稱為宇稱）是可相乘的；即它們的積是守恒的。所以可相乘的量子數都屬於一種對稱（像守恒那樣），而在這種對稱中使用兩次對稱變換式跟沒用過是一樣的。它們都屬於一個叫'''Z'''<sub>2</sub>的抽象[[群|群]]。
