'''量子引力'''，是對[[引力場|引力場]]進行[[量子化|量子化]]描述的理論，屬於[[萬有理論|萬有理論]]之一。研究方向主要嘗試結合[[廣義相對論|廣義相對論]]與[[量子力學|量子力學]]，是當前[[物理學|物理學]]尚未解决的問題。當前主流嘗試理論有：[[超弦理論|超弦理論]]、[[迴圈量子重力理論|迴圈量子重力理論]]。[[引力波|引力波]]的发现，为量子引力理论提供了新的[[証據|佐证]]。

== 背景 ==
{{unsolved|物理學|如何將[[量子力學|量子力學]]與[[廣義相對論|廣義相對論]]、[[引力|引力現象]]合併在一起，並且在[[微觀|微觀]]長度等級下維持正確性？任何候選的量子引力理論能提供什麼樣可證實的[[預測|預測]]呢？}}

经典描述下的[[引力|引力]]，是由[[愛因斯坦|愛因斯坦]]於1916年建立的[[廣義相對論|廣義相對論]]成功描述的。该理论透過[[質量|質量]]對於[[時空|時空]][[曲率|曲率]]的影響（[[愛因斯坦方程式|愛因斯坦方程式]]）而對[[水星近日點歲差偏移|水星近日點歲差偏移]]、[[引力紅移|引力場下光線紅移]]、[[光線彎折|光線彎折]]等三種問題提出了完滿的解釋，並且至今為止在[[天文學|天文學]]的觀測上，實驗數據與廣義相對論預測值的相符程度遠高於其他競爭理論。因此，廣義相對論描述古典引力的正確性很少有人懷疑。

另一方面，量子力學從[[狄拉克|狄拉克]]建立了[[相對論性量子力學|相對論性量子力學]]的[[狄拉克方程式|狄拉克方程式]]開始，擴充成[[量子場論|量子場論]]的各種形式。其中包括了[[量子電動力學|量子電動力學]]與[[量子色動力學|量子色動力學]]，成功地解釋了[[基本交互作用|四大基本力]]中的三者－[[電磁力|電磁力]]、原子核的[[強力|強力]]與[[弱力|弱力]]的量子行為，僅剩下引力的量子性尚未能用量子力學來描述。

除了未能達成對於引力量子（[[引力子|引力子]]）的描述之外，兩個成功的理論在根本架構上也有衝突之處：量子場論是建構在[[廣義相對論|廣義相對論]]的平坦[[時空|時空]]下基本力的粒子場上。如果要透過這種相同模式來對引力場進行[[量子化|量子化]]，則主要問題是在廣義相對論的彎曲時空架構，無法一如以往透過[[重整化|重整化]]的數學技巧來達成量子化描述，亦即[[引力子|引力子]]會互相吸引，而當把所有反應加總常會得到許許多多的[[無限大|無限大值]]，沒辦法用數學技巧得到有意義的有限值；相對地，例如[[量子電動力學|量子電動力學]]中對於[[光子|光子]]的描述，雖然仍會出現一些無限大值，但為數較少可以透過[[重整化|重整化]]方法可以將之消除，而得到實驗上可量到的、具有意義的有限值。

至於實驗方面，很遺憾，量子引力所探討的能量與尺度乃是目前實驗室條件下無法觀測得到的，有些學者提出一些觀點可能可以透過[[天文學|天文學]]上的觀測來檢驗，但仍屬少數特例。因此希望從實驗觀測得到一些關於量子引力理論發展上的提示，現階段仍屬不可行。

推導量子引力理論的一般方法是假設這個理論是簡單優雅的，然後回頭看看現前的理論，找尋[[對稱性|對稱性]]及提示以想辦法優雅地合併它們成為一個更加普適的理論。這方法的一項問題是沒人可以肯定量子引力是否會是一個簡單優雅的理論。

这个理论的意义在于了解一些涉及龐大[[質量|質量]]或[[能量|能量]]以及很小尺度的空間的問題，例如[[黑洞|黑洞]]的行為，以及[[宇宙|宇宙]]的起源。

== 歷史觀點 ==
歷史上，對於量子理論與要求[[背景獨立|背景獨立]]的廣義相對論兩者明顯的矛盾曾出現過兩種反應。 

第一種是廣義相對論所採的[[幾何|幾何]]詮釋並非究竟，而只是一個未知的[[背景相依|背景相依]]理論的近似表現。舉例來說，這在[[史蒂文·溫伯格|史蒂文·溫伯格]]的經典教科書《引力與宇宙學》裡面被明白表示過。

另外相抗衡的觀點是背景獨立是基礎性質，而量子力學需要被一般化，改寫成一個沒有預設特定時間的理論。這樣的幾何觀點在[[查爾斯·米斯納|米斯納]]（Charles W. Misner）、[[約翰·惠勒|約翰·惠勒]]與[[基普·索恩|基普·索恩]]三人合寫的經典著作《引力論》中詳述過。

[[理論物理|理論物理]]巨擘對於引力意義採相反看法的兩本書，很有趣地幾乎同時發表於1970年代早期。出現了這樣的僵局使得[[理查·費曼|理查·費曼]]在1960年代早期給太太的一封信中，絕望地寫道：「提醒我不要再參加任何一個引力會議。」理查·費曼對於使量子引力曾做過重要的嘗試。

這兩種論點分別發展出[[弦論|弦論]]與[[迴圈量子引力理論|迴圈量子引力理論]]。

== 量子力學與廣義相對論的不相容 ==
[[File:Calabi-Yau.png|thumb]]]]

理論物理上最深奧的問題之一是調和[[廣義相對論|廣義相對論]]以及量子力學。廣義相對論描述引力並且適用在大尺度結構（[[恆星|恆星]]、[[行星|行星]]、[[銀河|銀河]]），而量子力學描述其他三種作用在微觀尺度的[[基本力|基本力]]。

廣義相對論教導我們沒有固定的時空背景，而在[[牛頓力學|牛頓力學]]與[[狹義相對論|狹義相對論]]則有出現；時空幾何是[[動態|動態]]的。雖然在原則上容易掌握，這卻是廣義相對論中最難了解的概念，而且它所帶來的結果是相當深遠的，也沒完全地探索完，即使僅就古典層級而言。就某種程度而言，廣義相對論可以視作是一種[[關係理論|關係理論]]，在這樣的理論中，物理上唯一要緊的訊息是時空中不同事件彼此間的關係。

[[File:Gravity_Probe_B.jpg|thumb]]（GP-B）：已经测量了地球附近[[时空曲率|时空曲率]]进行[[验证|验证]]爱因斯坦的广义相对论的相关[[模型|模型]]的应用。]]

另一方面，由于量子力学最开始是從固定背景(非動態的)結構开始研究的，它依赖于固定的时空背景。在量子力學中，[[時間|時間]]是開始就給定而且非動態的，恰如牛頓的古典力學一般。在相對論性量子場論中，一如在古典場論中，[[閔可夫斯基時空|閔可夫斯基時空]]是理論的固定背景。[[弦論|弦論]]從擴充量子場論出發，其中點粒子代之以弦樣物體，在固定時空背景中做傳遞。雖然弦論的起源是在[[夸克侷束|夸克侷束]]（quark confinement）研究方面而不是在[[量子引力|量子引力]]方面，很快就發現弦的頻譜包括了[[引力子|引力子]]，而且弦的幾種特定振動模式的「凝聚」等價於對原始背景的修改。

[[File:Point&string.png|right]]中的[[弦_(物理學)|弦]]]]

處在彎曲（非閔可夫斯基式）背景下的量子場論，雖然並非引力的量子理論，亦顯示了量子場論中的一些假設無法被延伸到彎曲時空中，完善的量子引力理論就更不用提了。特別地說，真空—當它存在時—被指出和觀察者所經過的時空路徑有相依性(見[[盎魯效應|盎魯效應]])。此外，場概念看起來比粒子概念還要來得基本(粒子概念被認為是描述局域交互作用的方便法)。後者觀點是有爭議性的，和[[史蒂文·溫伯格|史蒂文·溫伯格]]的著作《量子場論》在閔可夫斯基空間中所發展出的量子場論相矛盾。

[[迴圈量子引力|迴圈量子引力]]是建構背景獨立量子理論的努力成果。[[拓撲量子場論|拓撲量子場論]]提供了背景獨立量子場論的一例，但其沒有局域的自由度而僅有有限個全域自由度。如此要描述3+1維的引力則顯得不足；按照廣義相對論，即使在真空，引力也有局域自由度。然而在2+1維，引力就可以是拓撲場論，而其也被成功地透過多種方法進行量子化，包括[[自旋網路|自旋網路]]的方法。

此外尚有三處量子力學與廣義相對論的拉鋸戰。

*首先，廣義相對論預言了自己在[[奇點|奇點]]會失效，而量子力學在奇點附近則會和廣義相對論格格不入。
*其次，在量子力學的[[海森堡不確定原理|海森堡不確定原理]]下，粒子的[[位置|位置]]與[[速度|速度]]無法同時確知，尚不清楚如何决定一个粒子的引力场。
*最後一處的拉鋸戰並非邏輯上的矛盾，它涉及了「量子力學造成[[貝爾不等式|貝爾不等式]]的違反」（暗示[[超光速|超光速]]影響）與「相對論中[[光速|光速]]作為速度上限」這兩者間的困境。前兩點的解決之道可能出自對於廣義相對論有更好的了解<ref>[http://arxiv.org/abs/astro-ph/0506506]</ref>。

量子化引力的「直接」方法有多項選擇。
* 是否要如同[[霍金|霍金]]一樣，採用對[[威克轉動|威克轉動]]過的[[黎曼度規|黎曼度規]]做[[泛函積分|泛函積分]]？參見[[歐幾里得式路徑積分|歐幾里得式路徑積分]]方法。
* 我們有用[[協變|協變]]Peierls bracket嗎？
* 我們有用[[BRST|BRST]]/Batalin-Vilkovisky形式，或[[規範固定|規範固定]]，或[[規範分解|規範分解]]嗎？
* 如果我們選擇了[[正則量子化|正則量子化]]，我們有用[[愛因斯坦-希爾伯特作用量|愛因斯坦-希爾伯特作用量]]將度規僅當作是動態量，以得到[[惠勒-得衛特方程式|惠勒-得衛特方程式]]嗎？ 
**抑或我們將度規與[[仿射聯絡|仿射聯絡]]各自處理？
* 抑或我們是否擁有整個[[龐加萊群|龐加萊群]]以作為[[規範群|規範群]]，並以[[愛因斯坦-嘉當理論|愛因斯坦-嘉當理論]]作為起點？
* 抑或我們有用[[活動標架|活動標架]]的[[嘉當方法|嘉當方法]]以及[[帕拉丁尼作用量|帕拉丁尼作用量]]，以得到[[第二類約束|第二類約束]]? 
* 我們有否消除掉第二類約束，利用[[阿希提卡變數|阿希提卡變數]]來得到[[迴圈量子引力|迴圈量子引力]]，或者我們要做其他方案？ 
: [[旋量場|旋量場]]的存在可能迫使我們要從事[[嘉當形式|嘉當形式]]或其他相當者的研究。
* 又或許我們我們應該關注[[微分同胚群表象|微分同胚群表象]]，一如[[韋格納|韋格納]]關注[[龐加萊群表象|龐加萊群表象]]一樣。

== 溫伯格-維騰定理 ==
在[[量子場論|量子場論]]中有則[[溫伯格-維騰定理|溫伯格-維騰定理]]，對於[[複合引力|複合引力]]/[[突現引力|突現引力]]方面的理論施加了一些約束條件。
