'''声母特点'''
# 古微母独立，音值为 ʋ。如 微/ʋei/≠围/ui/、万/ʋã/≠腕/uã/、网/ʋɔ/≠往/uɔ/、袜/ʋɛʔ/≠握/uɛʔ/。
# 古娘母二三等读如 ɳ，与泥母 n 形成对立。是晋语汾州一代特征。如 挠/ɳɑu/≠脑/nɑu/、扭/ɳou/≠努/nou/、拧/ɳə̃/≠能/nə̃/、女/ɳɤu/≠弩/nɤu/、喃/ɳã/≠南/nã/、躡/ɳəʔ/≠那/nəʔ/、攮/ɳuɔ/≠齉/nuɔ/。
# 古晓母匣母与模韵相拼时，混入 f 声母。如 胡=扶=/fv/、虎=斧=/fv/、互=父=/fv/。
# 全浊声母清化方式在汾阳城关口音中体现为平声送气，仄声不送气，与官话基本一致。与周边典型的吕梁（平仄皆送气）、并州（平仄皆不送气）清化方式迥异。
# 翘舌组声母 tʂ tʂʰ ʂ ʐ ɳ，除与 ʅ 相拼的音节外，新派有全部平舌化成为 ts tsʰ s z nz 的变化趋势。
# 鼻音声母 m n ŋ ɳ 后分别带有同部位的塞音声母，实际音值为 mb nd ŋg ɳɖ，老派口音较为明显。
# k kʰ ŋ 声母可与细音 i 相拼，音节来自于见组山摄一等字白读以及见组果摄一等开口字白读。如 干/ki/ 看/kʰi/ 安/ŋi/ 哥/ki/ 我/ŋi/ 等。

'''韵母特点'''
# i u y 发生高化舌尖化裂化，分别形成 /ɿ/、/ɯʉ • ʉ/、/ʮ/ 三个特殊的韵母，其中 /ɯʉ/ 和 /ʉ/ 属于同音位的不同变体，在唇音声母后读 /ʉ/，其余声母后读 /ɯʉ/。如 闭/pɿ/、第/tɿ/、米/mɿ/、离/lɿ/、猪/tʂɯʉ/、初/tʂʰɯʉ/、苦/kʰɯʉ/、布/pʉ/、醉/tsʮ/、岁/sʮ/、鱼/ʮ/、吕/lʮ/ 等。
# 梗摄二等舒声白读同假摄，为 /a/ /ia/ /ya/ 韵母。如 生/ʂa/、争/tʂa/、棚/pʰia/、耕/tɕia/、横/ɕya/、犷/tɕya/ 等。
# 梗摄三四等与曾摄三等白读脱鼻，韵母为 /ʅ/ /ɿ/。如 正/tʂʅ/、城/ʂʅ/、镜/tsɿ/、青/tsʰɿ/、星/sɿ/、影/ɿ/ 等。
# 宕江摄不分文白一律脱鼻，不分开合口。如 钢/kuɔ/=光/kuɔ/、张/tʂuɔ/=庄/tʂuɔ/、行/xuɔ/=黄/xuɔ/ 等。
# 效摄白读分二等和三四等。如 交/tɕiɑu/≠浇/tɕiɯ/、敲/tɕʰiɑu/≠悄/tɕʰiɯ/、效/ɕiɑu/≠笑/ɕiɯ/ 等。
# 蟹摄白读开口一二等有别，一等为 /ei/ 韵母，二等为 /ai/ 韵母。如 孩/xei/≠鞋/xai/、害/xei/≠澥/xai/、爱/ŋei/≠隘/ŋai/ 等。
# 果摄分开合口，开口呼为 /ɯ/ 韵母，合口呼为 /u/ 韵母。如 何/xɯ/≠和/xu/、柯/kʰɯ/≠科/kʰu/、多/tɯ/≠躲/tu/、拖/tʰɯ/≠妥/tʰu/ 等。
# 咸山摄白读二等与三四等有别。二等保留主元音和鼻化韵尾，为 /ã/ /iã/ /uã/ 韵母，三四等主元音和鼻韵尾脱落，只保留介音 /i/ /u/ /y/。如 间/tɕiã/≠尖/tɕi/、关/kuã/≠官/ku/、绢/tɕy/ 等。其中山摄一等合口帮组保留介音 /u/，如 半/pu/、盘/pu/、满/mu/ 等。
# 咸山摄一等见系声母白读韵母为 /i/，其中在晓母匣母后颚化。如 干/ki/、看/kʰi/、安/ŋi/、旱/ɕi/ 等。
# 止摄蟹摄合口三等白读为 /ʮ/ 韵母（支微入虞）。如 醉/tsʮ/、嘴/tsʮ/、岁/sʮ/、穗/sʮ/、遂/sʮ/、随/sʮ/、泪/lʮ/、垒/lʮ/ 等。
# 有两组八个入声韵，主元音分别为 /ɛ/ 和 /ə/。收喉塞尾 /ʔ/。

=== 儿化韵 ===
汾阳话有着丰富的儿化现象，一些儿化对元音造成了很大音变，如路儿/lou→lɯ̯˞/，桃儿/tʰau→tʰuə˞/ 等。有的字词甚至只有儿化用法，如很多动物名称 /kɯ̯<sup>ʉ˞</sup>/咕儿<sub>老鼠，</sub>/tsɯ̯˞/鸡儿，/ma̯˞/马儿 等。
