'''雙唇鼻音'''是[[輔音|輔音]]的一種，用於一些[[口語|口語]]中。雙唇鼻音在[[國際音標|國際音標]]的符號是{{IPA|{{尖括号|m}}}}，[[X-SAMPA|X-SAMPA音標]]的符號則是 <tt>m</tt>。世界上大部分[[語言|語言]]均有使用雙唇鼻音。[[漢語|漢語]]中“摸”、“麼”等字均是以雙唇鼻音開首，漢語拼音中以[[m|m]]表示。

== 特徵 ==
雙唇鼻音特點:

*發音方法是閉塞，也就是說通過阻礙空氣在聲腔流動來發音。

== 該音見於漢語中 ==
* 在[[現代標準漢語|現代標準漢語]]裡，雙唇鼻音見於''媽'' ({{lang|la|mā}})、''摸'' ({{lang|la|mō}})、''麼'' ({{lang|la|me}})的音首。雙唇鼻音韻尾皆化為n。
* 於[[粵語|粵語]][[廣州話|廣州話]]，此音可以在音首和音末中出現，如''無'' (mou4)、''甘'' (gam1)；亦可單獨出現，如''唔'' (m4)。
* 在[[闽南语|闽南语]]裡，此音可在音首和音末出現，如''麵'' (mi)、''甘'' (gamf)；亦可單獨出現，如''姆'' (mr)。(發音採[[MLT|MLT]])

== 該音見於英語中 ==
[[英語|英語]]中雙唇鼻音以字母m表示。包括[[美式英語|美式英語]]在內的某些英語口音，雙唇鼻音可以單獨成一音節，如''bottom''。
在英語中唇齒鼻音是雙唇鼻音的[[同位異音|同位異音]]。見條目[[唇齒鼻音|唇齒鼻音]]。
