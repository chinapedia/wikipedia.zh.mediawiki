'''夫妻在家看黄碟事件'''，也称“黄碟案”<ref group=註>“黄”于此处意为“色情的”，“碟”意为“碟片”，即“色情录像”。</ref>，是发生在[[中国|中国]][[陕西省|陕西省]][[延安市|延安市]]警方在2002年8月18日进入民宅查获淫秽录像的事件。此事件在中国引起了公众对[[隐私权|隐私权]]、警察权力、公民[[私权|私权]]与国家[[公权|公权]]边界等问题的广泛关注。<ref>{{Cite web |url= http://news.sina.com.cn/s/2003-01-05/0921862639.shtml |title= 夫妻家中看黄碟事件：公民权利终于得到保护 |author=  |date= 2003年1月5日 |publisher= 新浪网 |language= zh-hans |accessdate= 2013年10月10日 |quote=  }}</ref>

==事件经过==
===查处===
2002年8月18日23时，陕西省延安市万花山派出所接到举报，一居民家正播放“黄碟”，派出所四名民警前往调查。民警从窗口发现房间内的[[电视机|电视机]]正在播放淫秽录像，找藉口进入该居民家中，表明身份，并要求两人拿出“黄碟”。<ref name=first>{{Cite web |url= http://hsb.hsw.cn/gb/newsdzb/2002-08/20/2002-08-20-04shxw5.htm |title= 家中看黄碟民警上门查 |author=  |date= 2002-08-20 |publisher= 华商网 |language= zh-hans |accessdate= 2013年10月10日 |quote=  }}</ref>张某为阻止民警的调查，抡起木棍砸向一名民警，致使该民警手被打肿，另一名民警的衣服被抓破；派出所以[[妨害公务|妨害公务]]罪名将张某带回派出所<ref name=hsw1025>{{Cite web |url= http://hsb.hsw.cn/gb/newsdzb/2002-10/25/2002-10-25-26fzzk1.htm |title= 家中看黄碟 丈夫被刑拘 |author= 斌摄 |date= 2002-10-25 |publisher= 华商报 |language= zh-hans |accessdate= 2013年10月10日 |quote=  }}</ref>，并将从现场搜到的3张光碟连同电视机、[[影碟机|影碟机]]作为证据带回。<ref name=first/>被[[刑事拘留|刑事拘留]]后，张某曾向采访的记者表示，四名将其带回派出所的民警在他住所内时没有戴警帽，也没有佩带警号和警徽，仅身着警服。<ref name=hsw1025/>

===拘留===
张某在交了1000元“暂扣金”之后，张某领回了被拿走的电视机、影碟机。张某的妻子告诉采访记者原本要求缴纳3000元，经协商才降到1000元。<ref name=hsw1025/>

2002年10月21日，延安宝塔区公安分局治安大队将张某以[[妨害公务罪|妨害公务罪]][[刑事拘留|刑事拘留]]。<ref name=hsw1025/>同年11月5日，张某[[取保候审|取保候审]]。一个月后，张某被解除取保候审，宝塔分局撤销此案。<ref name=29000yuan>{{Cite web |url= http://hsb.hsw.cn/gb/newsdzb/2003-01/01/2003-01-01-01yw4.htm |title= “看黄碟”夫妇获赔两万九 |author= 小斌 |date= 2003-01-01 |publisher=华商网  |language= zh-hans |accessdate= 2013年10月10日 |quote=  }}</ref>

===事件处理===
2002年12月31日，陕西省延安市宝塔区公安分局向当事人赔礼道歉，一次性补偿当事人[[人民幣|人民幣]]29137元（含医疗费、误工费等）。<ref name=29000yuan/>

次年，陕西省延安市宝塔区公安分局免除主要责任人万花山派出所所长贺宏亮职务，调任其他岗位。<ref>{{Cite web|url=http://law.eastday.com/epublish/gb/paper4/5/class000400001/hwz598224.htm|title="夫妻看黄碟"最终结案:当地派出所所长被免职|author=|date=2003-01-15|publisher=东方法治|language=zh-hans|accessdate=2013年10月10日|quote=|deadurl=yes|archiveurl=https://web.archive.org/web/20131013052725/http://law.eastday.com/epublish/gb/paper4/5/class000400001/hwz598224.htm|archivedate=2013年10月13日}}</ref>其他两名协警也被公安机关辞退。<ref name=after>{{Cite web |url= http://www.hsw.cn/news/2006-12/17/content_5976918.htm |title= 延安小夫妻在家看黄碟被抓续：生活几陷绝境(图) |author=  |date= 2006年12月17日 |publisher= 华商网 |language= zh-hans |accessdate= 2013年10月12日 |quote=  }}</ref>

===影响===
因为“黄碟案”的影响，张某夫妻无法再维持宁静的生活。2003年6月，张某夫妻前往[[西安|西安]]谋生，依旧为“黄碟案”的影响所困扰。<ref name=after/>

==评论==
[[北京大学法学院|北京大学法学院]]教授[[贺卫方|贺卫方]]认为“这则案例，鼓舞市民应该为私人权利受到尊重而进行勇敢斗争”。<ref>{{Cite web |url= http://news.sina.com.cn/c/2007-11-13/011112889894s.shtml |title= 在家看黄碟是私人权利 |author=  |date= 2007年11月13日 |publisher= 新浪网/南方都市报 |language= zh-hans |accessdate= 2013年10月12日 |quote=  }}</ref>

北京大学法学院教授[[朱苏力|苏力]]认为因为“有人举报”，说明“看黄碟”并非是私密的行为，警方如果不出警可能会面对不负责任的指责。对此案的评论的法律人忽视司法与执法的实际。<ref>苏力. 当代中国法理的知识谱系及其缺陷——从“黄碟案”透视. 北京:中外法学,2003(15):304.</ref>
