《'''酒佬日記'''》（{{lang-en|'''Sideways'''}}）是一齣2004年的美國電影。由Jim Taylor和[[阿歷山大·佩恩|阿歷山大·佩恩]](Alexander Payne)編劇，[[阿歷山大·佩恩|阿歷山大·佩恩]]導演。劇本改編自Rex Pickett在2004年出版的同名小說。主要演員為[[保罗·吉亚玛提|保罗·吉亚玛提]]、[[托马斯·哈登·丘奇|托马斯·哈登·丘奇]]、Virginia Madsen和[[吳珊卓|吳珊卓]]。酒佬日記贏得了奧斯卡金像獎最佳改編劇本獎，並獲提名四個其他獎項。

== 劇情 ==
Miles Raymond (保罗·吉亚玛提)是一個不成功的作家、酗酒的人和中學英文教師。他住在[[聖迭戈|聖迭戈]]。他和他那個快要結婚的好朋友Jack Cole (托马斯·哈登·丘奇)打算前往Santa Ynez谷暢遊。Miles對前妻念念不忘，今次一心想去放鬆一下，不過Jack卻只是想去與別的女人胡混。

在Santa Ynez，他們遇上了在餐廳The Hitching Post當侍應的Maya (Virginia Madsen)和她的朋友，在酒莊工作的Stephanie (吳珊卓)。四人一起約會晚餐，不過Miles和Jack沒有告訴兩位女子Jack快要結婚。飯後Jack跟Stephanie回家上床，而Miles則與Maya一起喝紅酒，Miles更給Maya看自己寫的小說。

Maya後來從Miles口中發現原來Jack快要結婚，告訴了Stephanie。她怒打Jack，打斷了他的鼻樑。兩女跟兩男分手。之後Miles在一個試酒會中喝醉了，之後Jack又搭上了另一個女人，卻遺下了自己的錢包在她的家中，要Jack為他取回。為了要和他的未婚妻解釋鼻樑的傷，Jack將Miles的車撞上了一棵樹。

在婚禮上，Miles重遇他的前妻，她已再婚及懷孕，令Miles終於明白二人已經無可能復合。他獨自在一間路邊的咖啡店，以一個塑膠杯喝下了他了的珍藏，本想與他的前妻一起分享的一支1961年[[白馬酒莊|白馬酒莊]](Château Cheval Blanc)紅酒。

後來，他收到了Maya的訊息。她說她真的很喜歡他的作品，邀請他來探望她。最後一幕，Miles回到Santa Ynez，敲上了Maya的家門。

== 評價 ==
美國著名影評家Roger Ebert寫道：「在那七天所發生的事，加起來成為了全年最好的人性喜劇。它是喜劇，因為它可笑；它也具人性，因為它意外地感人。」另外他又寫：「Virginia Madsen在片中對Miles的回答既簡單又真實，如果這個世界有公義的話，她將會獲得奧斯卡提名。」<ref>{{cite web|url=http://rogerebert.suntimes.com/apps/pbcs.dll/article?AID=/20041028/REVIEWS/40922017/1023|title=SIDEWAYS (R)|author=Roger Ebert|publisher=''Chicago Sun-Times''|date=2004年10月29日}}</ref>

== 影響 ==
[[File:Hitching_Post.jpg|right]]
這齣電影為美國加州Santa Ynez超過20個消費場所製造商機，當地旅遊局特別印刷10,000幅《酒佬日記》地圖，方便遊客抵達後容易找到拍攝地點。其中一個拍攝場地Hitching Post餐廳，成為很多影迷光顧的食肆。旅遊局發言人布魯克斯表示：「我們真是無法預測《酒佬日記》效應如此巨大，當初一般認為這不是一部主流電影，應當不會受到廣泛注意。但出乎意料之外，《酒佬日記》上映後成為一部受歡迎的電影，直接令到Santa Ynez旅遊業大受裨益。」<ref>{{cite newspaper|url=http://paper.wenweipo.com/2005/03/03/OT0503030003.htm|title=造就旅遊勝地《酒佬日記》|newspaper=''[[文匯報|文匯報]]''|author=李明|date=2005年3月3日}}</ref>

由於男主角Miles偏愛[[黑皮諾|黑皮諾]]，因此《酒佬日記》帶紅了較冷門的[[黑皮諾|黑皮諾]]紅酒，引起好些迴響。<ref>{{cite newspaper|url=http://www.hkheadline.com/living/living_content.asp?contid=62122&srctype=g|title=葡萄酒莊深度遊|author=張芷靈|newspaper=''[[頭條日報|頭條日報]]''|accessdate=2012年3月15日}}</ref>

== 日本重拍版本 ==
《酒佬日記》獲日本重拍，日本版由[[菊地凛子|菊地凛子]]、[[鈴木京香|鈴木京香]]、[[小日向文世|小日向文世]]及[[生瀬勝久|生瀬勝久]]主演，全片在美國加州取景，2009年秋天在日本上畫。<ref>{{cite newspaper|url=http://hk.apple.nextmedia.com/realtime/art_main.php?iss_id=20081107&sec_id=7018882&art_id=11820808|title=日本重拍 《酒佬日記》|newspaper=''[[蘋果日報|蘋果日報]]''|date=2008年11月7日}}</ref>
