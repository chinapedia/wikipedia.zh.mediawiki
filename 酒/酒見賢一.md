'''酒見賢一'''（{{lang-ja|酒見賢一}}，{{bd|1963年|11月26日||}}）是日本[[福岡縣|福岡縣]][[久留米市|久留米市]]出生的男性小說家，作品採用的題材以改編[[中國歷史|中國歷史]]上的人事物居多。

== 經歷 ==
酒見賢一在就讀[[愛知大學|愛知大學]]畢業後至[[名古屋|名古屋]]當地一間負責住屋設備的公司就職，並在勤務的空檔裡嘗試撰寫文章。

之後於1989年發表了個人第一篇作品《[[後宮小說|後宮小說]]》，此小說則獲得當年第一屆的日本奇幻小說大獎。<ref>{{Cite web |url=http://www.shinchosha.co.jp/prizes/fantasy/1/ |script-title=ja:第一回日本ファンタジーノベル大賞 |work=[[新潮社|新潮社]] |accessdate=2013年5月22日 }}</ref>後續在1992年酒見賢一以《墨攻》、《在陋巷》、《畢達哥拉斯之旅》三篇作品得到中島敦紀念獎，<ref>{{Cite web |url=http://www.city.yokohama.lg.jp/kyoiku/library/chosa/mokuroku/yokohamabungaku/nakajima.html |script-title=ja:歿後五十年中島敦記念賞 |work=橫濱市立圖書館 |accessdate=2013年5月22日 }}</ref>而2000年推出的《周公旦》則拿下第19屆的新田次郎文學獎。<ref>{{Cite web |url=http://homepage1.nifty.com/naokiaward/kenkyu/furok_NITTAaward.htm |script-title=ja:新田次郎文学賞受賞作一覧1-32回 |work=[[直木獎|直木獎]] |accessdate=2013年5月22日 |title=存档副本 |archive-url=https://web.archive.org/web/20130719173253/http://homepage1.nifty.com/naokiaward/kenkyu/furok_NITTAaward.htm |archive-date=2013年7月19日 |dead-url=yes }}</ref>
