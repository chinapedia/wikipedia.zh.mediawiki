'''伏尔加格勒'''（{{lang-ru|Волгогра́д}}，[[羅馬化|羅馬化]]：{{lang|en|Volgograd}}）是[[俄羅斯|俄羅斯]]南部[[伏爾加格勒州|伏爾加格勒州]]的[[都市|都市]]，也是伏爾加格勒州的[[首府|首府]]。該城始建于1589年，原名'''察里津'''（{{lang|ru|Цари́цын}}），1925年4月10日改称'''斯大林格勒'''（{{lang|ru|Сталингра́д}}），1961年改為現名。2013年1月31日，伏尔加格勒市[[杜马|杜马]]通过决议，在每年的8月23日（[[苏德互不侵犯条约|苏德互不侵犯条约]]签署日）、2月2日（[[斯大林格勒保卫战|斯大林格勒保卫战]]胜利日）、5月9日（[[德国投降|纳粹德国投降日]]和[[歐戰勝利紀念日|歐戰勝利紀念日]]）、9月2日（[[日本投降|日本投降日]]）、6月22日（苏德战争的[[巴巴罗萨作战|巴巴罗萨作战]]开始日）和11月19日（斯大林格勒保卫战的[[天王星作战|天王星作战]]开始日）这六个纪念日里将市名改为斯大林格勒。

== 历史 ==
[[File:Volgograd_de_insulo.jpg|thumb]]{{clear|left}}
这里是[[乌兹别克汗|乌兹别克汗]]的首都。1589年，[[沙皇俄国|沙皇俄国]]在[[察里津河|察里津河]]和[[伏尔加河|伏尔加河]]的交汇处的一个岛上建造了察里津要塞。察里津是[[鞑靼语|鞑靼语]]的名称。在1670年和1774年察里津两次分别被{{link-en|斯捷潘·拉辛|Stepan Razin}}和[[普加乔夫起义|普加乔夫起义]]军攻陷。到19世纪，这里已经是伏尔加河上的重要港口和商业中心：全俄罗斯第三大[[铁路|铁路]]枢纽，[[木材|木材]]、[[盐|盐]]、[[石油|石油]]的集散地，伏尔加河下游最大的商业和工业城市。

在1917年11月7日—1922年10月的[[俄国内战|俄国内战]]时期，这里发生了[[察里津战役|察里津战役]]（1918年7月—1919年2月，又称[[察里津保卫战|察里津保卫战]]）。[[斯大林|斯大林]]（本姓朱加什维利）时任北高加索军区军事委员会主席。斯大林、[[伏罗希洛夫|伏罗希洛夫]]等人在此指挥[[苏联红军|苏联红军]]成功地抵抗了[[邓尼金|邓尼金]]的[[白军|白军]]的进攻。1919年6月30日白军最终占领了这座城市。1920年1月又被红军收回，并在1925年被命名为斯大林格勒。在斯大林执政时期，这里成了苏联重要的[[重工业|重工业]]中心，[[粮食|粮食]]、[[石油|石油]]和[[煤炭|煤炭]]的主要产区，和[[铁道|铁道]]、伏尔加河运集散中心。北高加索的粮食，[[巴库|巴库]]的石油，[[中亚|中亚]]的[[棉花|棉花]]，都要通过这一工业铁路枢纽和伏尔加河的港口，才能提供给苏联的中心地区。

[[苏德战争|苏德战争]]（[[蓝色行动|蓝色行动]]）期间，这里是著名[[蓝色行动|蓝色行动]]（1942年6月28日—11月18日）以及包括[[天王星行动|天王星行动]]（1942年11月19日—11月22日）、[[土星行动|土星行动]]（1942年12月12日—1943年2月19日）（总共是1942年6月28日—1943年2月19日）的[[斯大林格勒战役|斯大林格勒保卫战]]的战场。[[纳粹德国|纳粹德国]]在1941年7月7日—9月26日的[[基辅战役|基辅防御战役]]之后占领左岸[[乌克兰|乌克兰]]，之后俄罗斯南部的斯大林格勒成了苏联中央地区通向南方重要经济区域的咽喉，战略地位十分重要，加上这座城市是以当时苏联的最高领袖命名的，因此斯大林格勒的得失对战局和人心都会产生巨大的影响。战争中24万余纳粹德国德意志国防军陆军及其罗马尼亚王国盟友的士兵和47.87万人苏联红军阵亡，平民伤亡不计其数。斯大林格勒全市90%的建筑被毁，但纳粹德国国防军陆军只是占领了这里二个月。1943年3月之后苏联迅速重建斯大林格勒。1945年斯大林格勒和其他三座城市（[[列宁格勒|列宁格勒]]、[[塞瓦斯托波尔|塞瓦斯托波尔]]和[[敖德萨|敖德萨]]）成为第一批“英雄城市”。

1961年因[[赫鲁晓夫|赫鲁晓夫]]“反对斯大林个人崇拜”这里被更名为“伏尔加格勒”。此后至今不时有人提议把它的名字改回斯大林格勒。在[[康斯坦丁·乌斯季诺维奇·契尔年科|契尔年科]]执政的1985年改名几乎成功。2005年4月16日[[俄罗斯联邦共产党|俄罗斯联邦共产党]]中央委员会主席[[久加诺夫|久加诺夫]]说“我们应该积极支持前线老兵的呼吁，将伏尔加格勒改回原名斯大林格勒。”

2013年1月31日，伏尔加格勒市杜马通过一项决议，在每年的8月23日（苏德互不侵犯条约签署日）、2月2日（斯大林格勒保卫战胜利日）、5月9日（纳粹德国无条件投降日和二战欧洲战场胜利日）、9月2日（对日作战胜利纪念日）、6月22日（苏德战争的巴巴罗萨作战开始日）和11月19日（斯大林格勒保卫战的天王星作战开始日）这六个纪念日裡将市名改为斯大林格勒。<ref>https://www.telegraph.co.uk/history/world-war-two/9839666/Russia-revives-Stalingrad-city-name.html</ref>

== 地理 ==
[[File:Volgograd_oficeja_konstruajho.jpg|thumb]]

伏尔加格勒位于俄罗斯联邦西南部的[[北高加索|北高加索]]地区，[[伏尔加河|伏尔加河]]下游，[[顿河|顿河]]大弯曲部以东约60公里处。城区沿河岸绵延约100公里。伏尔加格勒面积565[[平方公里|平方公里]]。

伏尔加格勒是一个俄国南方城市，平均气温[[夏季|夏季]]26度，[[冬季|冬季]]零下9度。气候属[[温带大陆性湿润气候|温带大陆性湿润气候]]，风景秀丽，气候宜人。

== 人口 ==

截至2010年，统计人口1,021,215，其中近90%是[[俄罗斯|俄罗斯]]人，其他人口较多的少数民族有：[[乌克兰|乌克兰]]人、[[喀山|喀山]]人、[[日耳曼|日耳曼]]人、[[鞑靼|鞑靼]]人等。

== 交通经济 ==
[[File:Volgograd_trolebuso_3.jpg|thumb]]
伏尔加格勒位于连接俄国南方两大水系伏尔加河和[[顿河_(俄羅斯)|顿河]]的[[伏爾加-頓河運河|伏爾加-頓河運河]]的东端，有铁路和[[莫斯科|莫斯科]]（1073公里。）、[[乌克兰|乌克兰]]、[[高加索|高加索]]、[[西伯利亚|西伯利亚]]等地区相连。今天的伏尔加格勒的主要工业包括：[[造船|造船]]、[[石化|石化]]、[[钢铁|钢铁]]、[[铝材|铝材]]、[[机械|机械]]、[[汽车|汽车]]和[[化工|化工]]。北面不远有一座大型的[[水力发电站|水力发电站]]。
