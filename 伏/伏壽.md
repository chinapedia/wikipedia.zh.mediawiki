'''伏壽'''（{{bd|2世纪||214年|catIdx=Fu,伏}}），是[[汉献帝|汉献帝]]皇后。[[东汉|东汉]]末年[[徐州刺史部|徐州]][[琅邪郡|琅琊]][[诸城市|東武]]（今[[山东省|山东]][[诸城市|诸城]]）人。伏皇后是西漢大司徒[[伏湛|伏湛]]的八世孫，父親是學者[[伏完|伏完]]，母是[[盈|盈]]。

== 生平 ==
[[初平|初平]]元年（190年），[[董卓|董卓]]帶漢獻帝西遷[[长安|长安]]，伏壽跟隨並入[[掖庭|掖庭]]為[[貴人|貴人]]。[[兴平_(东汉)|興平]]二年（195年），伏貴人被立為[[皇后|皇后]]。同年漢獻帝東歸，但被[[李傕|李傕]]、[[郭汜|郭汜]]等人追擊，在[[東澗|東澗]]發生戰事，守護獻帝的軍隊戰敗，多名官員被殺；獻帝亦乘夜渡河逃走，伏皇后和六宮都徒步跟隨。當時伏皇后手持[[縑布|縑布]]數匹，被[[董承|董承]]派來的符節令[[孫徽|孫徽]]以刀刃脅逼奪去，更殺害伏皇后側的侍從，鮮血濺在伏皇后的衣服上。

[[建安_(东汉)|建安]]元年（196年），回到已成廢墟的[[洛阳市|洛陽]]，但因各州郡首長擁兵自重，無人接濟，官員都十分飢餓，部份更餓死。及後[[曹操|曹操]]迎獻帝到[[许昌市|許昌]]。[[曹操|曹操]]“奉天子以令不臣”，東漢皇室已经丧失实权，宮中的守衛和侍從其實都是曹操的人。對內外官員的殺戮亦很常見，例如[[議郎|議郎]][[趙彥_(東漢)|趙彥]]曾經向獻帝進言分析局勢和對策，因而被曹操殺害。建安五年（200年），董承與[[王子服|王子服]]、[[种辑|种輯]]等圖謀誅除曹操失敗被殺，曹操更多次請獻帝殺死身為[[贵人|貴人]]的[[董承|董承]]女兒，獻帝虽以[[董貴人|董贵人]]有懷身孕而不答應，但董贵人仍被杀害。

伏皇后見官員被殺和曹操屢逼獻帝誅殺後宮，因而更為恐懼，於是寫信給父親伏完陳說被曹操勢力威壓的情形，要求父親密謀政變，但伏完不敢。209年伏完去世，不料[[建安_(东汉)|建安]]十九年（214年），伏皇后當年的圖謀泄露，曹操知道後大怒，逼獻帝廢后，更先替獻帝寫好詔書，命[[郗慮|郗慮]]持節接收皇后印綬，又命尚書令[[华歆|华歆]]領兵入宮捉拿伏皇后。

伏皇后关门躲在牆壁之中，華歆破坏宫门和墙壁将她牵出。經過外殿时，伏皇后披发赤脚，見獻帝与御史大夫[[郗慮|郗虑]]同坐，抓着献帝的手哭著說：「不能復相活邪？（不能再救救我嗎？）」獻帝答：「我亦不知命在何時。（我亦不知我何時會死。）」又對身旁的郗慮說：「郗公，天下寧有是邪？（郗公，天下間有這樣的事嗎？）」最後伏皇后被[[幽闭|幽闭]]而死（《[[曹瞒传|曹瞒传]]》称当场被杀），所生的兩位皇子亦以毒酒毒殺，伏氏[[宗族|宗族]]有百多人亦被處死，伏皇后母親盈（可能即樊普姐）等十九人都被流放到[[涿郡|涿郡]]。《后汉书》作“曹操杀皇后伏氏”。

== 雜記 ==
据《献帝春秋》，伏完还将信给[[荀彧|荀彧]]和妻弟樊普看，樊普将其封存呈给曹操，荀彧担心祸及自身，还劝曹操进女于献帝，曹操和荀彧议及此事时，荀彧有“伏皇后无子”之语。不過[[裴松之|裴松之]]作注时质疑这段记载中荀彧的邏輯有問題。由於整段記事的可信度有問題，伏皇后最後確定有兩子被毒酒毒死，這兩個小孩的出生時間也不見得要在伏完密謀之後。

== 争议 ==
民国史学家[[呂思勉|呂思勉]]对伏皇后被诛杀一案存有不同看法，认为以[[曹操|曹操]]的地位和气魄不会简单因为伏皇后给其父亲的一封信有诋毁之言而加害，而且[[汉献帝|汉献帝]]也不会因为[[董承|董承]]被害而迁怒[[曹操|曹操]]（[[董承|董承]]本身也不是靠得住的忠臣），认为这个案件背后“一定另有原因”，而且背后的事情很大、不便于记载到正史之上，吕思勉推测认为伏皇后有一个通盘的政治计划而且事关大局，因此[[曹操|曹操]]不得不动手<ref>吕思勉，《三国史话·替魏武帝辩诬》：“所以我说伏皇后的被杀，是一定另有政治上的阴谋的，不过其真相不传于后罢了。假定伏皇后的被杀，是别有阴谋，则魏武帝一身，既然关系大局的安危，自不得不为大局之故而将他扑灭。这正和带兵的不能因军中有一群人反对他而即去职，或自杀，置军队的安危于不顾一样。”</ref>。

== 评价 ==
*[[曹操|曹操]]：「皇后寿，得由卑贱，登显尊极，自处椒房，二纪于兹。既无任、姒徽音之美，又乏谨身养己之福，而阴怀妒害，苞藏祸心，弗可以承天命，奉祖宗。」
*[[王嘉_(东晋)|王嘉]]：「献帝伏皇后，聪惠仁明，有闻于内则。」
*[[范晔_(刘宋)|范晔]]：「坤惟厚载，阴正乎内《诗》美好逑，《易》称归妹。祁祁皇孋，言观贞淑。媚兹良哲，承我天禄。班政兰闺，宣礼椒屋。既云德升，亦曰幸进。身当隆极，族渐河润。视景争晖，方山并峻。乘刚多阻，行地必顺。咎集骄满，福协贞信。庆延自己，祸成谁衅。」
*[[蔡东藩|蔡东藩]]：「诛奸无力反招灾，巾帼拚生剧可哀；前有董妃后伏后，魂兮可向许宫来！」

== 文学形象 ==
[[罗贯中|罗贯中]]历史小说《[[三国演义|三国演义]]》作伏-{后}-被曹操下令左右乱棒打死。民国小说家[[周大荒|周大荒]]创作《[[反三国演义|反三国演义]]》时，因认为曹操无权对皇帝、皇后加罪及杀害，故没有沿用，将情节改为曹操以此为契机称帝，废献帝为山阳公，华歆不候曹操旨意即暗遣勇士随献帝夫妇到山阳乘间刺杀，地方官吏上奏称二人急病而死。
