'''伏見區'''（{{jpn|j=伏見区|hg=ふしみく|rm=Fushimi ku|f=y}}）是[[京都市|京都市]]11区之一，位於京都市南部，區內主要為住宅區，為京都市內人口最多的區。<ref>{{Cite web| url=http://www.city.kyoto.lg.jp/fushimi/page/0000013282.html| title=伏見区の概要 : 伏見区の沿革・特長・文化| publisher=京都市伏見區公所| date=2012-11-12| accessdate=2017-06-06| language=ja}}</ref>

== 历史 ==
現在的伏見區範圍在令制國時期分屬[[山城國|山城國]][[紀伊郡|紀伊郡]]、[[乙訓郡|乙訓郡]]、[[宇治郡|宇治郡]]、[[久世郡|久世郡]]、[[綴喜郡|綴喜郡]]。[[平安時代|平安時代]]此地多為京都貴族的別莊，16世紀末[[豐臣秀吉|豐臣秀吉]]晚年在此建立了[[伏見城|伏見城]]，此後開始以城下町的形勢發展，因此出現大量的武士住宅<ref>{{Cite web| url=http://www.city.kyoto.lg.jp/fushimi/page/0000013318.html| title=安土桃山時代　秀吉が開いた城下町| publisher=京都市伏見區公所| date=2008-09-12| accessdate=2017-06-06| language=ja}}</ref>，雖然一度遭遇{{tsl|ja|慶長伏見地震}}損毀，但很快地有完成重建，而[[豐臣秀吉|豐臣秀吉]]最終過世於伏見城內。此後伏見城成為德川家康監視位於大阪的[[豐臣秀賴|豐臣秀賴]]的據點，1600年[[關原之戰|關原之戰]]前發生的[[伏見城之戰|伏見城之戰]]，西軍先進攻了德川家康所擁有的伏見城，造成伏見城再次燒毀，但德川家康為了繼續監視豐臣家，再次迅速的完成重建；最終德川將柯在此就任[[征夷大將軍|征夷大將軍]]，此後建立了江戶幕府。

在[[江戶時代|江戶時代]]初期伏見城遭到廢除，改建{{tsl|ja|淀城}}，並成為{{tsl|ja|淀藩}}的領地，由於位於京都與大阪之間，此地藉由[[淀川|淀川]]的水上運輸成為港口{{tsl|ja|伏見港}}及宿場町{{Tsl|ja|伏見宿 (京街道)|伏見宿 (京街道)|伏見宿}}繼續發展。

由於鄰近京都，在19世紀江戶時代末期的動亂中，著名的兩次[[寺田屋事件|寺田屋事件]]即發生於伏見，決定德川幕府結束的關鍵戰爭[[鳥羽伏見之戰|鳥羽伏見之戰]]也發生於此，此戰也造成伏見半數的民宅遭到焚毀。<Ref>{{Cite web| url=http://www.city.kyoto.lg.jp/fushimi/page/0000013321.html| title=伏見区の歴史 : 江戸時代～幕末　港湾商業都市の繁栄| publisher=京都市伏見區公所| date=2008-09-12| accessdate=2017-06-06| language=ja}}</ref>

1889年，日本實施[[町村制|町村制]]，現在的伏見區範圍在當時分屬：伏見町、深草村、向島村、堀內村、竹田村、下鳥羽村、横大路村、納所村、醍醐村、久我村、羽束師村、淀村、淀町、美豆村。

在[[1929年|1929年]]，伏見町改制為{{Tsl|ja|伏見市}}，但由於市區逐漸與京都市相連，在1931年與周邊的深草村、向島村、堀內村、竹田村、下鳥羽村、横大路村、納所村、醍醐村一同併入京都市，並成為新設立的伏見區；1950年至1957年期間，久我村、羽束師村、淀町也先後被併入京都市，並劃入伏見區，形成現在的伏見區範圍。<ref>{{Cite web| url=http://www.city.kyoto.lg.jp/fushimi/page/0000013327.html| title=伏見区の歴史 : 近代産業への飛躍／明治・大正・昭和へ| publisher=京都市伏見區公所| date=2008-09-12| accessdate=2017-06-06| language=ja}}</ref>
