'''棕硬尾鴨'''（[[學名|學名]]：''{{lang|la|Oxyura jamaicensis}}''）是一種[[硬尾鴨|硬尾鴨]]。

== 特徵 ==
成年雄鴨的身體呈銹紅色，面部白色，冠黑色。成年雌鴨的身體呈灰褐色，面部灰色，喙深色，兩頰有斑紋。南方[[亞種|亞種]]的[[火地島棕硬尾鴨|火地島棕硬尾鴨]]有時被認為是另一[[物種|物種]]，其面部黑色，體型較大。[[安第斯棕硬尾鴨|安第斯棕硬尾鴨]]的黑色羽毛各有不同，面部白色，有可能是指名亞種及火地島棕硬尾鴨的[[混種|混種]]。由於安第斯棕硬尾鴨的數量變得較為稀少，故有需要確定其分類狀況來進行保育工作。

== 習性 ==
棕硬尾鴨棲息在[[北美洲|北美洲]]的[[沼澤|沼澤]]湖及池中，在[[南美洲|南美洲]]的[[安地斯山脈|安地斯山脈]]也有分布。牠們會在茂密的沼澤林中築巢，以較高的植物掩護避免被掠食者發現，典型的巢中有5-15隻小鴨<ref name ="LPZRuddyDuck">[http://www.lpzoo.org/animals/factsheet/ruddy-duck "Ruddy Duck Fact Sheet, Lincoln Park Zoo"]</ref>。每年都會進行[[繁殖|繁殖]]。棕硬尾鴨是[[候鳥|候鳥]]，會遷徙到海灣及沒有結冰的湖中過冬。

牠們能夠潛入水中。牠們主要吃水生[[植物|植物]]的[[種子|種子]]及[[根|根]]、水生[[昆蟲|昆蟲]]及[[甲殼類|甲殼類]]。
==入侵物种==
棕硬尾鴨在1948年被引入[[英國|英國]]，后来向[[歐洲|歐洲]]蔓延。牠們急進的求偶行為，會與其他物種（如[[白頭硬尾鴨|白頭硬尾鴨]]）[[混種|混種]]引起了關注。故此，欧洲各国开始有計劃地將牠們消滅<ref>{{cite web|url=http://news.bbc.co.uk/1/hi/uk/2814293.stm|title=R.I.P. Ruddy duck|publisher=BBC NEWS}}</ref>。到2014年初其数量已从2000年巅峰时期的5000只降至不足100只<ref name=Telegraph2014/>。
