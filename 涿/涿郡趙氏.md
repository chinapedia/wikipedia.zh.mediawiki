'''涿郡趙氏'''是一个以[[涿郡|涿郡]]为[[郡望|郡望]]的[[士族_(中国)|士族]]。

[[趙姓|趙姓]]是中國的大姓。根據史書的記載，趙姓的血緣祖先是上古時期的[[五帝|五帝]]之一的[[少昊|少昊]]，傳說，[[颛顼|颛顼]]的孙女[[女修|女修]]因吞了[[玄鳥|玄鳥]]卵而生了[[大业|大业]]。[[大业|大业]]的后裔[[造父|造父]]因善於車馬，深得[[周穆王|周穆王]]的賞識，又兼他在平定[[徐偃王|徐偃王]]叛亂中及時駕車帶周穆王回京，周穆王就賜造父[[趙城|趙城]] (今[[山西|山西]][[洪洞縣|洪洞縣]][[趙城鎮|趙城鎮]]) 作為[[封邑|封邑]]。造父的後代子孫以封邑趙城作為趙姓。 到了[[春秋|春秋]]時期，趙、魏、韓三家共為[[晉國|晉國]]的權臣。晉國末年，趙、魏、韓[[三家分晉|三家分晉]]，並得到[[天子|天子]][[周威烈王|周威烈王]]認可與冊封，分別為[[趙國|趙國]]、[[魏國|魏國]]和[[韓國|韓國]]。趙國為[[戰國|戰國]][[七雄|七雄]]之一。後為[[秦國|秦國]]所滅。

自從趙國亡國後，趙姓後人分為兩派：一系是代王[[趙嘉|趙嘉]]後人，被秦始皇封往[[天水郡|天水郡]]的[[天水趙氏|天水趙氏]]，另一系是幽繆王[[趙遷|趙遷]]的後人，被秦始皇流徙至房陵的涿郡趙氏。幽繆王趙遷被流放到房陵（今湖北房縣）後，終日鬱鬱寡歡，數年後病逝於此。伴隨遷來此處的趙國宗室及大臣和侍從都安頓於此。最後形成'''涿郡趙氏'''。[[西漢|西漢]]時，涿郡趙氏自房陵遷居涿郡中分立河間郡，蠡吾屬河間，涿郡趙氏故又稱[[河間趙氏|河間趙氏]]。<ref>百家姓 - 趙，http://defate.com/name/100/008.htm </ref>

==宋朝皇室==
[[漢宣帝|漢宣帝]]時的名臣、[[京兆尹|京兆尹]][[趙廣漢|趙廣漢]]就出自涿郡趙氏。趙廣漢亦是[[宋朝皇族|宋朝皇族]]之始祖。趙廣漢後人[[趙朓|趙朓]]，曾任唐朝的永清、文安、幽都縣令，趙朓獨子[[趙珽|趙珽]]累官至御史中丞。趙珽的兒子[[趙敬|趙敬]]在[[後梁|後梁]]時任營、薊、涿三州刺史。後梁被[[後唐|後唐]]滅，趙敬殉難，他的兒子[[趙弘殷|趙弘殷]]─[[宋太祖|宋太祖]][[趙匡胤|趙匡胤]]之父，為避父禍，逃難到今[[洛陽|洛陽]]郊外的夾馬營，被鄉紳[[杜爽|杜爽]]招為[[贅婿|贅婿]]，定居於此。<ref>中華頌 -> 百家姓 -> 名姓望族，{{cite web |url=http://big5.ccnt.com.cn/china/surname/tribe/cw-zhao-2.htm |title=存档副本 |accessdate=2011-03-05 |deadurl=yes |archiveurl=https://web.archive.org/web/20110518231307/http://big5.ccnt.com.cn/china/surname/tribe/cw-zhao-2.htm |archivedate=2011-05-18 }} </ref> 後[[趙匡胤|趙匡胤]]稱帝，建[[宋|宋]]。
