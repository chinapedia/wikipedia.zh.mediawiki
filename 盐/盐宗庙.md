'''盐宗庙'''亦称'''曾公祠'''，位于[[中国|中国]][[江苏省|江苏省]][[扬州市|扬州市]][[广陵区|广陵区]][[扬州城池|古城]]东南隅、徐凝门内南河下康山街20号，西邻[[卢氏盐商住宅|盐商卢绍绪宅第]]，由两淮盐商出资、两淮盐运使[[方浚颐|方浚颐]]主持兴建于清[[同治|同治]]十二年（1873），大殿内供奉盐业生产、营销和盐政的三位始祖，即最早煮海盐的[[夙沙氏|夙沙氏]]、最早的盐商[[胶鬲|胶鬲]]和最早的盐官[[管仲|管仲]]，三宗同祀的形式国内仅存<ref>{{cite web|url=http://www.jscin.gov.cn/web/showinfo/Showinfo.aspx?infoid=6347635f-54c0-4b31-b13f-dd2008a6fd59&siteid=1|title=《扬州市城建档案馆为盐宗庙修复开放做好服务工作》|accessdate=2013-12-07|work=|publisher=江苏省住房和城乡建设厅|archiveurl=https://web.archive.org/web/20131211162354/http://www.jscin.gov.cn/web/showinfo/Showinfo.aspx?infoid=6347635f-54c0-4b31-b13f-dd2008a6fd59&siteid=1|archivedate=2013-12-11|deadurl=yes}}</ref>。后因[[曾国藩|曾国藩]]一度担任两江总督兼两淮盐政，任内开通长江盐路，促进扬州盐业复兴，故一年后改庙宇为其专祠<ref>{{cite web |url=http://www.yznews.com.cn/news/2007-03/20/content_479473.htm|title=《扬州盐宗庙为何改名曾公祠》|accessdate=2013-12-07|work=陈跃|publisher=扬州网}}</ref>。建筑原有五进，今存大门，二门和正殿前三进，均为面阔三间的硬山顶建筑，正殿两侧有廊庑，其后的后楼、戏台等已不存<ref>{{cite web |url=http://www.yznews.com.cn/yzwb/html/2007-03/30/content_2282067.htm|title=《盐宗庙的前世与今生》|accessdate=2013-12-07|work=孙诚兵|publisher=扬州晚报}}</ref>。大门外侧墙面做工精细，门上砖嵌额枋内雕有海棠式框景，内为拐子锦和阳刻灵芝一枚（神灵之意）<ref>{{cite web |url=http://www.yznews.com.cn/yzwb/html/2008-03/13/content_3690378.htm|title=《盐宗庙》|accessdate=2013-12-07|work=|publisher=扬州晚报}}</ref>。2006年全面修缮，2007年4月外对开放。
