'''墙角枪'''（CornerShot）是一种应用于[[城鎮戰|巷战]]的特种[[武器|武器]]，由[[以色列|以色列]]牆角射擊公司（Corner Shot Holdings, LLC）設計。

== 歷史 ==
其實早於[[第一次世界大战|一戰]]、[[第二次世界大战|二戰]]時期已經有墙角枪的意念。一戰時的[[澳大利亚|澳洲]]軍、二戰時[[納粹德國|納粹德國]]有製作了類似的組件，當中納粹德國製作的組件為[[StG44突击步枪|StG44突擊步槍]]專用。

== 設計 ==
墙角枪可以在墙角能够在不暴露自己身体的任何部分的情况下攻击另一边，但嚴格來說，墙角枪只是一個槍支的支架。

墙角枪分前后两部分，中间用活页相连，前面的部分包括[[手枪|手枪]]和安装在原刺刀位置的彩色[[摄像头|摄像头]]，可以左右转向63度，后部主要是枪托、扳机、2.5英寸[[液晶屏|液晶屏]]监视器、[[戰術導軌|戰術導軌]]和电池（可以提供枪支150分钟的工作时间）；前部的手枪可以旋转63度攻击墙角另一边的敌人，前部的摄像头将信号传递到后半部的液晶屏幕，枪手可以进行观察和瞄准，另外摄像头下瑞亦有[[手電筒|電筒]]以應付黑暗環境。

墙角枪的标准型配置的是一般而不同口徑的[[半自动手枪|半自动手枪]]（如貝瑞塔系列、格洛克系列或92式、CZ系列、Five-seveN系列、SIG Sauer系列等），也可以裝配[[轉輪手枪|轉輪手枪]]（目前只限中國的216兵工廠型號）。改装型可以用来发射40毫米[[榴弹發射器|榴弹發射器]]及短[[突击步枪|突击步枪]]或[[卡宾枪|卡賓槍]]（如[[AK-47突击步枪|AK系列]]、[[M16突擊步槍|M16系列]]等），甚至[[反坦克榴弹发射器|Panzerfaust]]火箭筒。

墙角枪手枪型的有效射程有100米，若使用5.7×28毫米口徑手槍的話則可以達到200米。

== 型號 ==
*'''手枪型'''（CSM）—[[半自動手槍|半自动手枪]]
*'''步槍型'''（APR）－短[[突击步枪|突击步枪]]或[[卡宾枪|卡賓槍]]

*'''反坦克火箭型'''（CSP）

== 類似的武器 ==
[[中國|中國]]、[[巴基斯坦|巴基斯坦]]、[[伊朗|伊朗]]、[[印度|印度]]及[[大韩民国|南韓]]有製造類似墙角枪的武器，牆角射擊公司也有和[[法国|法國]]的Omega公司合作研發OMEGA-Cornershot，中國的216兵工廠更製造了裝配[[左轮手枪|轉輪手枪]]的墙角枪。

*[[中國|中國]]：HD-66、CF-06、CS/LW9型拐弯枪
*[[法国|法國]]： OMEGA-Cornershot
*[[巴基斯坦|巴基斯坦]]：{{link-en| POF Eye| POF Eye}}

== 採購 ==
原廠製造的墙角枪由於因為過高的價格，目前僅有少部分國家使用，主要用於反[[恐怖主义|恐]]及營救[[人质|人质]]；而大多數的國家所採用的墙角枪，皆為類似但自主研發的拐角射擊裝置。
