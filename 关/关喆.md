'''关喆'''（{{bd|1981年|4月23日|catIdx=G关}}），[[满族|满族]]，[[中國|中國]][[歌手|歌手]]，畢業於[[燕山大学|燕山大学]]，2005年获得东方卫视《[[我型我秀|我型我秀]]》歌唱比赛全国第12名出道发行首張個人專輯《關愛》，由于发展不顺又参加各种选秀比赛，2011年曾赴台灣參加中視[[華人星光大道_(第一屆)|第一屆《華人星光大道》]]，但因工作緣故晉級18強後退賽，于2012年参加浙江卫视《[[中国好声音|中国好声音]]》获得年度八强后才再次走红。

==主要经历==
*1981年 出生於吉林省吉林市一个普通工人家庭；
*2002年 毕业於燕山大学计算机专业；
*2005年 获得东方卫视《我型我秀》歌唱比赛全国第12名出道<ref>[http://ent.sina.com.cn/y/o/2005-08-10/1144805987.html 东方卫视莱卡我型我秀之老大关喆走了]</ref>，并签约[[艾回唱片|艾回唱片]]；
*2006年 在北京现代音乐学院担任流行演唱系特别讲师，并参加CCTV[[青歌赛|青歌赛]]获得个人通俗组优秀奖<ref>[http://yule.sohu.com/20060719/n244345233.shtml 青歌赛通俗唱法决赛选手：关喆]</ref>；
*2007年 担任国内歌手孙悦、毛宁、那英、潘阳等的专辑音乐制作人，并与毛宁同居<ref>{{Cite web |url=http://sports.21cn.com/integrate/other/2012/09/20/13039338.shtml |title=关喆承认与毛宁同居疑似同性恋 |access-date=2014-03-06 |archive-url=https://web.archive.org/web/20121014162130/http://sports.21cn.com/integrate/other/2012/09/20/13039338.shtml |archive-date=2012-10-14 |dead-url=yes }}</ref>；
*2008年 与日本著名音乐制作人菊地圭介共同创作并演唱的[[北京奥运会|北京奥运会]]歌曲《北京 北京 我爱北京》荣获第四届奥运歌曲征集世界第二名，成为了2008年奥运会闭幕式的主题曲<ref>{{Cite web |url=http://news.bjcma.com/sdbd/5201.html |title=关喆作品《北京北京我爱北京》奥运闭幕式大放光彩 |access-date=2014-03-06 |archive-url=https://web.archive.org/web/20140306131147/http://news.bjcma.com/sdbd/5201.html |archive-date=2014-03-06 |dead-url=yes }}</ref>，同年4月发行首张个人专辑《关爱》<ref>[http://ent.zjol.com.cn/05ent/system/2008/04/18/009432368.shtml 关喆新专辑《关爱》发布 信给“克隆师弟”捧场]</ref>；
*2009年 发行第二张张个人专辑《永远的永远》<ref>[http://ent.sina.com.cn/y/2009-07-16/22262613817.shtml 关喆上海宣传全新创作大碟《永远的永远》]</ref>；
*2010年 参加世博会倒计时200天《微笑上海》群星唱世博大型演唱会和[[2010快乐男声|2010快乐男声]]总决赛、复活赛以及吉林赛区评委；
*2011年 参加台湾中视《[[华人星光大道|华人星光大道]]》第一季晋级18强后退赛<ref>[http://video.sina.com.cn/p/ent/y/2011-08-08/115061437431.html 华人星光大道 关喆《日不落》]</ref>；
*2012年 参加浙江卫视《[[中国好声音|中国好声音]]》获得年度八强<ref>[http://ent.163.com/photoview/28NN0003/522597.html#p=9M1CHBEQ28NN0003 关喆：“后好声音时代”不好混]</ref>，并发行第三张个人专辑《身边的故事》<ref>{{Cite web |url=http://fashion.ce.cn/mlc/201203/01/t20120301_23120645.shtml# |title=关喆新专辑《身边的故事》发行 北京办音乐会 |access-date=2014-03-06 |archive-url=https://web.archive.org/web/20140306130508/http://fashion.ce.cn/mlc/201203/01/t20120301_23120645.shtml# |archive-date=2014-03-06 |dead-url=yes }}</ref>；
*2013年 签约台湾[[华纳唱片公司|华纳唱片公司]]<ref>[http://www.chinanews.com/yl/2013/12-13/5616864.shtml 关喆签新东家 林俊杰萧敬腾等同门师兄送祝福]</ref>。
*2014年 第一屆QQ音樂年度盛典，以創作單曲《想定下來》榮獲年度熱歌金曲獎
*2015年 以熱門單曲《想你的夜》榮獲Red Box & Green Box Karaoke 2014年最高點播率Ｋ歌20強
*2016年 參加[[湖南衛視|湖南衛視]]《[[我是歌手第四季|我是歌手第四季]]》，在第二集因搶位失敗而淘汰
