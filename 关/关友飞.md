'''关友飞'''（{{bd|1957年|||}}），[[广西|广西]][[梧州|梧州]]人。[[中国人民解放军海军|中国人民解放军海军]]少将。

==生平==
曾长期在[[中国人民解放军海军|中国人民解放军海军]]舰艇部队服役，并曾在[[中国人民解放军海军大连舰艇学院|中国人民解放军海军大连舰艇学院]]学习。后任[[中央军事委员会国际军事合作办公室|中华人民共和国国防部外事办公室]]综合局局长。2008年，任[[中华人民共和国国防部|中华人民共和国国防部]]外事办公室副主任。2013年，任国防部外事办公室主任，[[中国人民解放军总参谋部|中国人民解放军总参谋部]][[中国人民解放军总参谋部外事局|外事局]]局长。2016年1月，任新组建的[[中央军委国际军事合作办公室|中央军委国际军事合作办公室]]主任<ref>{{Cite web |url = http://news.takungpao.com/paper/q/2016/0113/3267192.html |title =  五湖四海选将促军种融合|author =  |date = 2016-01-13 |publisher =大公网  |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://news.ifeng.com/a/20160115/47091082_0.shtml |title = 关友飞少将出任军委国际军事合作办公室主任 |author =  |date = 2016-01-15 |publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref>。
