《'''貧窮姊妹物語'''》是[[日本漫畫家|日本漫畫家]]{{link-ja|一十泉|かずといずみ}}<ref>《貧窮姊妹物語》青文出版社中文版單行本（Sake譯）譯為「一十泉」，{{lang|ja|かずといずみ}}短篇漫畫集《小行星小故事》（{{lang|ja|小さな惑星の小さなお話}}）[[東立出版社|東立出版社]]中文版（莊湘萍譯，東立出版社2009年2月15日第1刷，ISBN 978-986-10-2916-0）譯為「[[和人泉|和人泉]]」。《貧窮姊妹物語》與《小行星小故事》的後記漫畫中的作者[[自畫像|自畫像]]完全相同，由此可證「一十泉」與「和人泉」是同一人。</ref> 所創作的[[漫畫|漫畫]]<ref>{{cite web|url=http://acg.gamer.com.tw/acgDetail.php?s=21481 | title=【漫畫】貧窮姊妹物語|publisher=巴哈姆特電玩資訊站|accessdate=2014-01-21 |language=zh-tw}}</ref>，以及依此改編的[[電視動畫|電視動畫]]<ref>{{cite web|url=http://acg.gamer.com.tw/acgDetail.php?s=21662 | title=【動畫】貧窮姊妹物語|publisher=巴哈姆特電玩資訊站|accessdate=2014-01-21 |language=zh-tw}}</ref>與[[廣播劇|廣播劇]]。

== 概要 ==
本作品在[[小學館|小學館]]發行的《[[月刊Sunday_GENE-X|月刊Sunday GENE-X]]》雜誌2004年5月號（2004年4月）開始連載，於2006年11月號（2006年10月）連載完畢。[[台灣|台灣]]與[[香港|香港]]地區的中文版皆由[[青文出版社|青文出版社]]代理，分別發行台灣版與香港版。動畫版於2006年6月起至9月止於日本地區播映，播映期間共3個月，一共十集。

== 故事簡介 ==
'''山田今日'''與'''山田明日'''是一對姊妹，她們的母親在明日出生的那一年過世，父親欠下龐大[[債務|債務]]後下落不明。<ref>《貧窮姊妹物語》台灣中文版單行本第1卷（青文出版社2006年5月30日初版一刷，ISBN 986-156-549-3）第11頁。</ref> 兩人住在[[東京都|東京都]][[下町|下町]]<ref name="ReferenceA">《貧窮姊妹物語》台灣中文版單行本第1卷（青文出版社2006年5月30日初版一刷，ISBN 986-156-549-3）第17頁。</ref> 屋齡40年的[[公寓|公寓]]，大小只有1[[坪|坪]]，格局只有一房，沒有浴室，房租一個月兩萬六千[[日圓|日圓]]。<ref>《貧窮姊妹物語》台灣中文版單行本第1卷（青文出版社2006年5月30日初版一刷，ISBN 986-156-549-3）第2與8頁。</ref> 姊姊今日是[[中學生|中學生]]，一邊求學一邊[[打工|打工]]（送[[報紙|報紙]]等）維持生計；妹妹明日雖然是[[小學生|小學生]]，但是會幫忙做家事。雖然她們過著[[貧窮|貧窮]]的生活，周圍的人們也支持著她們。《貧窮姊妹物語》描寫的就是山田姐妹的生活。

在這部作品的設定中，幾年前日本《{{link-ja|勞動基準法 (日本)|労働基準法|勞動基準法}}》修正使中學生與[[成年人|成年人]]擁有同等的[[工作權|工作權]]（原作第2話設定是20XX年）<ref name="ReferenceA"/>；現在日本的法律認定，早晨送報紙或送牛奶是例外<ref>總務之森：《中學年齡層的少年送報紙，是否違反勞動基準法？》（{{lang|ja|総務の森「中学生くらいの少年が新聞配達、労働基準法違反ではないのか?」}}）2005年4月5日，http://www.soumunomori.com/column/article/atc-799/{{Ja icon}}</ref>。

== 登場人物 ==
=== 山田家 ===
; {{nihongo|山田今日|山田 きょう<ref>《貧窮姊妹物語》[[同人誌|同人誌]]版改為「山田今日」。</ref>、やまだ きょう||[[配音員|配音員]]：[[坂本真綾|坂本真綾]](動畫版)、[[田中理惠_(聲優)|田中理惠]]([[廣播劇CD|廣播劇CD]]版)}}
: 就讀中學三年級的姊姊（15歲）。她每天早上跟下午送報紙來維持家裡的生計，在第13話與第45話中從事其他種類的打工（試吃推銷員、[[便利商店|便利商店]]店員、餐廳服務生等）。上課時總是打瞌睡（在第23話中稱之為「睡眠學習」），但是成績不錯<ref>《貧窮姊妹物語》台灣中文版單行本第2卷（青文出版社2006年7月30日初版一刷，ISBN 986-156-572-8）第100頁。</ref>。總是思念著妹妹，雖然她害怕打雷，但是她不會停止照顧她的妹妹。她答應她的母親會好好的照顧她的妹妹明日，而實現對她的母親的諾言。
; {{nihongo|山田明日|山田 あす<ref>《貧窮姊妹物語》同人誌版改為「山田明日」。</ref>、やまだ あす||配音員：[[金田朋子|金田朋子]](動畫版)、[[釘宮理惠|釘宮理惠]](廣播劇CD版)}}
: 就讀小學三年級的妹妹（9歲）。她總是為家裡購物、做菜跟打掃。明日喜歡她的姊姊，而想嘗試讓她的生活得更好，因為她總是為了她努力工作賺錢。她擅長打[[麻將|麻將]]，是整個商店街的知名人物<ref>《貧窮姊妹物語》台灣中文版單行本第1卷（青文出版社2006年5月30日初版一刷，ISBN 986-156-549-3）第66頁。</ref>。明日在班上擔任學級委員，當作業不會寫的時候，同班同學都會請教她。她的音感不好，但是她請教正男教她如何唱歌。奇怪的地方是，她有對她的母親一點點記憶。當她還在她母親的子宮裡的時候，她的母親要求今日照顧她。
; {{nihongo|山田佳子|山田 佳子、やまだ かこ||配音員：[[久川綾|久川綾]](動畫版)}}
: 山田姊妹的母親，在今日6歲的時候（即明日出生的那一年）過世，年齡、享年不明。
; {{nihongo|山田週|山田 週、やまだ しゅう}}
: 山田姊妹的父親，年齡不明。因為背負著龐大的債務，拋棄了他的女兒，下落不明；在最終回中，他欠債的原因被證實是作友人的連帶保證人，該友人無力清償債務三千萬日圓，他才將原住處賣掉以幫該友人償還三千萬日圓<ref>《貧窮姊妹物語》台灣中文版單行本第4卷（青文出版社2007年2月1日初版一刷，ISBN 978-986-156-819-5）第149頁。</ref>。作者原本未曾想到要畫他的臉，所以沒畫他的臉；但是作者看過《貧窮姊妹物語》電視動畫版[[角色設定|角色設定]]資料之後，取得設定者[[高村和宏|高村和宏]]同意使用於漫畫中。<ref>《貧窮姊妹物語》台灣中文版單行本第4卷（青文出版社2007年2月1日初版一刷，ISBN 978-986-156-819-5）第160頁。</ref>

=== 山田姐妹身邊的人們 ===
; {{nihongo|林源三|林 源三、はやし げんぞう||配音員：[[麥人|麥人]](動畫版、廣播劇CD版)}}
: 山田姊妹居住的公寓的房東（78歲），去催繳房租的的時候被山田姊妹叫作「恐怖的大魔王」。雖然他個性非常頑固，其實心地非常善良，總是關心著山田姊妹。
; {{nihongo|三枝蘭子|三枝 蘭子、さえぐさ らんこ||配音員：[[平松晶子|平松晶子]](動畫版、廣播劇CD版)}}
: 住在山田姊妹隔壁的小說家，年齡不明，一天到晚戴著[[太陽眼鏡|太陽眼鏡]]。使用[[筆名|筆名]]「森蘭子」執筆並獲得大木獎。
; {{nihongo|一之倉正男|一ノ倉 正男、いちのくら まさお||配音員：[[岸尾大輔|岸尾大輔]](動畫版)}}
: 住在山田姊妹隔壁的鄰居（29歳），夢想是成為[[歌手|歌手]]；即使靠打工勉強度日、屢次參加選拔賽都落選，依舊不改其志。他曾在[[livehouse|livehouse]]駐唱，擁有許多女性歌迷。
; {{nihongo|越後屋金子|越後屋 金子、えちごや きんこ||配音員：[[進藤尚美|進藤尚美]](動畫版、廣播劇CD版)}}
: 就讀中學三年級（15歲），大企業「越後屋集團」的後代，興趣是節儉，曾向山田姊妹請教節儉之道。跟今日不同中學的學生，並且擔任[[學生會|學生會]]會長。
; {{nihongo|越後屋銀子|越後屋 銀子、えちごや ぎんこ||配音員：[[小櫻悅子|小櫻悅子]](動畫版)、[[齋藤千和|齋藤千和]](廣播劇CD版)}}
: 金子的妹妹，就讀小學三年級（9歲）。明日隔壁班的同學。非常尊敬金子，羨慕即使貧窮也可以過生活的山田姊妹。

=== 山田家的親戚 ===
動畫版中未登場。
; {{nihongo|高島未來|高島 未来、たかしま みらい}}
: 山田姊妹的阿姨（33歲），職業是[[建築師|建築師]]，生活富裕。幼時因家貧而被送給住在[[仙台|仙台]]的高島家收養，故能體會貧窮生活的滋味。由於工作的關係（必須赴建築工地監工等），她很少與母親同住。她推薦今日轉學到私立中學就讀，並承諾負擔今日轉學後的生活費，讓今日專心求學、不必打工；但今日不願與明日分隔兩地，故予以婉拒。
; {{nihongo|高島時|高島 トキ、たかしま とき}}
: 高島未來的養母（原是未來的阿姨）（70歲），也是山田姊妹的姨婆，住在仙台。對未來非常頑固，但是對山田姊妹非常溫柔。
