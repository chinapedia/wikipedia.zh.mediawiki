{{japanese|japanese=寬徳|kana=かんとく|romaji=Kantoku}}

'''寬德'''（1044年十一月二十四日至1046年四月十四日）是[[日本|日本]]的[[年號|年號]]之一。此時的[[天皇|天皇]]是[[後朱雀天皇|後朱雀天皇]]與[[後冷泉天皇|後冷泉天皇]]。

== 改元 ==
*長久五年十一月二十四日（1044年12月16日） 改元寬德
*寬德三年四月十四日（1046年5月22日） 改元永承

== 出處 ==
《[[後漢書|後漢書]]卷二十七·宣張二王杜郭吳承鄭趙列傳第十七·杜林傳》：「……蠲除苛政，更立疏網，海內歡欣，人懷'''寬德'''……」
