'''寬政'''是[[日本|日本]]的[[年號|年號]]之一，在[[天明_(日本)|天明]]之後、[[享和|享和]]之前，指的是1789年到1801年之間。此時代的[[天皇|天皇]]是[[光格天皇|光格天皇]]，[[江戶幕府|江戶幕府]]的將軍是[[德川家齊|德川家齊]]。

== 改元 ==
*天明9年1月25日（公元1789年2月19日） 因内裏炎上等災異而改元。
*寬政13年2月5日（公元1801年3月19日） 改元為享和。

== 出典 ==
出自《[[左傳|左傳]]》的「施之以'''寬'''、寬以濟猛、猛以濟寬、'''政'''是以和」。

== 出生 ==

* 6年：[[水野忠邦|水野忠邦]]（江戶幕府老中首座、[[遠江國|遠州]]{{link-ja|濱松藩|浜松藩}}藩主）
