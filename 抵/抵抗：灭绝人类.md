{{vgname|抵抗：灭绝人类|en=Resistance: Fall of Man|cn=抵抗：灭绝人类|tw=全面對抗：人類沒落之日}}是一款由Insomniac开发的[[第一人称射击游戏|主视角射击游戏]]，适用于[[PlayStation_3|PlayStation 3]]平台，2006年11月发售。游戏场景设定在[[十月革命|十月革命]]、[[第二次世界大战|第二次世界大战]]均未发生过的20世纪中叶。玩家在游戏中扮演一名美国士兵内森·黑尔（Nathan Hale），与试图灭绝人类的合成兽Chimera战斗。这种怪兽由俄国人研发，已踏平亚洲和欧洲大部。玩家使用的是真实世界中不存在的自动武器和能量武器。

本作的多人游戏模式可以支持最多40人进行联机。<ref>{{Cite web |url = http://ps3.tgbus.com/zixun/200611/20061123155639_4.shtml |title = PS3大作《抵抗：灭绝人类》GameSpot玩后感全文翻译 |author = onami |date = 2006-11-23 |publisher = 电玩巴士 |language = zh-cn |accessdate = 2015-09-28 }}</ref>2014年3月28日，索尼关闭了《抵抗：灭绝人类》、《[[抵抗2|抵抗2]]》、《[[抵抗3|抵抗3]]》的线上服务。<ref>{{Cite web |url = http://www.ali213.net/news/html/2013-12/93314.html |title = 今后只能玩单机！索尼关闭多款PS3游戏服务器 |author = 乐乐 |date = 2013-12-27 |publisher = 游侠网 |language = zh-cn |accessdate = 2019-02-13 }}</ref><ref>{{Cite web |url = http://www.ali213.net/news/html/2014-1/94084.html |title = SONY宣布旗下《抵抗》1-3系列服务器将停运 |author =  |date = 2014-01-07 |publisher = 游侠网 |language = zh-cn |accessdate = 2019-02-13 }}</ref><ref>{{cite web |url=http://www.eurogamer.net/articles/2014-01-06-sony-turning-off-gran-turismo-5-resistance-servers |title=Sony turning off Gran Turismo 5 and Resistance servers |author=Robert Purchese |date=2014-01-06 |website=[[Eurogamer|Eurogamer]] |publisher=Gamer Network |language=en|accessdate=2019-02-13}}</ref>
