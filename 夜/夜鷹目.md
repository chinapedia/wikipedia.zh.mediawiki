'''夜鹰目'''（[[学名|学名]]：{{lang|la|''Caprimulgiformes''}}）在[[动物分类学|动物分类学]]上是[[鸟纲|鸟纲]]中的一个[[目_(生物)|目]]。这一目的鸟分布广泛，除了[[南極洲|南極洲]]外各大洲都有分布。

== 學名和俗名 ==
夜鷹目的學名 Caprimulgiformes 來自[[夜鷹屬|夜鷹屬]]的屬名 ''Caprimulgus''，在[[拉丁文|拉丁文]]中的意思是「飲[[山羊|山羊]]乳者」（{{lang|la|''capra-mulgere''}}），這是因為以前在西方的坊間相傳夜鷹會盜取山羊的[[羊乳|羊乳]]，而生物學之父[[林奈|林奈]]卻以此命名，也就一直沿用至今，當然此說法並沒有科學根據，畢竟也有部分觀點指出夜鷹可能只是為了要吃山羊附近的蟲子罷了。

在中文裡，「夜鹰」这个名字来自于[[日語|日語]]的「{{lang|ja|夜鷹}}（{{lang|ja|ヨタカ}}）」（「夜鷹」現今在日語中是指[[普通夜鷹|普通夜鷹]]）。[[中国|中国]]古代称其为「蚊母」，也叫蟁母，早在[[晋代|晋代]]和[[唐代|唐代]]就有所记述。这一目的鸟有一个显著的共同特征，即嘴短且嘴裂阔，因其在飞翔时张口食蚊，而古人误认为是吐出蚊子，所以给了它「蚊母」、「吐蚊鸟」等名字。夜鹰目的鸟白天休息于地上，或將身体平贴于树干上，在中国[[华北|华北]]地区被称为「贴树皮」。而在台灣，則有「石磯仔」、「石磯鳥」等俗稱。

夜鷹在分類上一直有爭議，以下是目前採信度較高的分類方式：

* [[林鸱科|林鸱科]] (Nyctibiidae) 1属：7种
* [[耳夜鹰科|耳夜鹰科]] (Eurostopodidae) 2属：7种；有时被放在夜鹰科之下的[[耳夜鹰亚科|耳夜鹰亚科]] (Eurostopodinae) 
* [[夜鹰科|夜鹰科]] (Caprimulgidae)

== 分布 ==
除北美洲北部、南美洲南部及某些大洋中的岛屿外，几乎遍布世界。[[油鸱科|油鸱科]]分布于[[南美|南美]][[热带|热带]]地区，[[蟆口鸱科|蟆口鸱科]]分布于[[东洋界|东洋界]]和[[澳洲界|澳洲界]]，[[林鸱科|林鸱科]]分布于[[中南美洲|中南美洲]]熱帶地區。

== 习性 ==
夜鹰目的鸟为[[夜行性|夜行性]]。白天大都蹲伏在草地里或树枝上，有的也在洞穴中。食物主要是[[昆虫|昆虫]]，在飞行中捕食。夜鹰在地面或岩石上营巢，雏鸟属晚成雏。

有的夜鹰还有[[冬眠|冬眠]]蛰伏的习性。

== 外形特征 ==
夜鹰目的鸟嘴短，嘴裂阔，口须长，[[眼睛|眼睛]]较大。体色多呈斑杂状，可以很好的伪装在地面、岩石和树皮中。雌雄无差别。体长14-55厘米。在生殖季节一些夜鹰尾部出现长羽，翘内侧出现白色斑纹，这些特征于进行空中求偶炫耀时更加明显。
