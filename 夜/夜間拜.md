{{伊斯蘭教法學}}'''夜間拜'''（{{Lang-en|tahajjud}}，{{lang-ar|تهجد}}）又稱'''泰罕朱德'''、'''禮夜間拜、深夜禮拜'''<ref>{{Cite web|url = http://www.zhenchen.org/article-1060-1.html|title = 禮拜|accessdate = 2015-05-23|author = 握不住的沙|date = 2015-01-02|publisher = 穆斯林百科|deadurl = yes|archiveurl = https://web.archive.org/web/20160305002414/http://www.zhenchen.org/article-1060-1.html|archivedate = 2016-03-05}}</ref>，是[[伊斯兰教|伊斯蘭教]]之中自願性的祈禱，屬於[[懿行拜|懿行拜]]的一種。這種禮拜是在早上凌晨時段，太陽還未升起之時進行。這並非要求所有[[穆斯林|穆斯林]]的每日必須進行的[[五功|五功]]之內，然而[[伊斯蘭教的先知|伊斯蘭先知]][[穆罕默德|穆罕默德]]曾被記載他時常進行夜間拜<ref>{{Cite web|url = http://www.islam.net.cn/html/yisilanjiao/jiaoyigangyao/20110305/198.html|title = 善行(15)礼夜间拜(Tahajjud/Qiyam Layl)|accessdate = 2015-05-23|author = 阿伊莎|date = 2011-01-15|publisher = |deadurl = yes|archiveurl = https://web.archive.org/web/20160304091113/http://www.islam.net.cn/html/yisilanjiao/jiaoyigangyao/20110305/198.html|archivedate = 2016-03-04}}</ref><ref>{{Cite web|url =http://ch.islamtoday.net/node/357|title =夜间拜|accessdate =2015-05-23|author =|date =|publisher =}}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，同時也鼓勵同伴進行這項動作，將這段時間奉獻真主是有許多好處的，這是種潔敬心靈的方法，讓人能夠更接近真主的國度。
