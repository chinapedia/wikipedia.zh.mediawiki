《'''夜魔俠'''》（{{lang-en|''Daredevil''}}）是一部於[[2003年電影|2003年]]上映的美國[[超級英雄電影|超級英雄]]電影，由{{link-en|馬克·史蒂芬·強森|Mark Steven Johnson}}執導及編劇，劇情圍繞著[[漫威漫畫|漫威漫畫]]旗下角色[[夜魔俠|夜魔俠]]展開。該片的主演包含[[班·艾佛列克|班·艾佛列克]]、[[珍妮佛·嘉納|珍妮佛·嘉納]]、[[麥可·克拉克·鄧肯|麥可·克拉克·鄧肯]]、[[柯林·法洛|柯林·法洛]]、{{link-en|喬·潘托里亞諾|Joe Pantoliano}}、[[強·法夫洛|-{zh-tw:強·法夫洛;zh-hk:莊·法來奧;zh-cn:乔恩·法夫罗;}-]]、[[大衛·凱斯|大衛·凱斯]]、{{link-en|利藍·歐爾瑟|Leland Orser}}與[[艾蓮·朋佩歐|艾蓮·朋佩歐]]等人，劇情圍繞著白天當[[律師|律師]]，晚上化身成「夜魔俠」的[[夜魔俠|麥特·墨鐸]]展開。

電影的發展作業於1997年開始，由[[二十世紀福斯|二十世紀福斯]]與[[哥倫比亞影業|哥倫比亞影業]]主導。2000年，{{link-en|攝政娛樂|Regency Enterprises}}取得了版權。《夜魔俠》中的場景主要在[[洛杉磯市中心|洛杉磯市中心]]取景。[[節奏特效工作室|節奏特效工作室]]受聘負責該片的CGI部分。{{link-en|葛瑞米·李維|Graeme Revell}}譜寫了電影的配樂，其原聲帶於2003年3月發行。

《夜魔俠》於2003年2月14日在美國發行，其動作場面、演員的演技、配樂、劇情及視覺風格皆獲得了讚美，但許多影評人都批評了該片的深度不夠。儘管所獲評價多為負面，該片仍在票房方面取得成功，以7,800萬美元的預算成功進帳了1.792億美元的成功票房。該片的分拆電影《[[幻影殺手_(電影)|幻影殺手]]》於2005年上映，在票房及評價方面都極其失敗。

2004年，《夜魔俠》的R級導演剪輯版發布，比原版多出了30分鐘。該剪輯版與原版比起評價較佳。製片方原本打算出該片的續集，但因《幻影殺手》的失敗而胎死腹中。2015年，[[Netflix|Netflix]]與[[漫威電視|漫威電視]]合作出品的[[夜魔俠_(電視劇)|同名電視劇]]（背景設在與該片不同的[[漫威電影宇宙|另一個宇宙]]）首播，並廣受好評。

== 劇情 ==

[[夜魔俠|麥特·墨鐸]]是一位出身於紐約市[[地獄廚房_(曼哈頓)|地獄廚房]]的盲人律師，與好友{{link-en|佛吉·尼爾森|Foggy Nelson|法蘭克林·「佛吉」·尼爾森}}一同經營一間事務所。麥特因幼時誤觸放射性廢料，導致雙目失明，卻增強了其餘感官的能力，身手敏捷俐落。在父親傑克遭歹徒襲擊而橫死街頭後，麥特誓言要為無辜受害的人討回公道。白天是律師的他，晚上便化身成「夜魔俠」，伸張正義，將法律無法制裁的惡徒[[就地正法|就地正法]]。

一天，麥特在咖啡店裡與希臘船運大亨——尼可拉斯·納崔斯（{{lang|en|Nikolas Natchios}}）之女[[幻影殺手|艾麗嘉]]邂逅。艾麗嘉之父尼可拉斯其實有另一個身份——[[黑手黨|黑手黨]]的一份子，他的老大是綽號「[[金霸王_(漫威漫畫)|金霸王]]」的威爾森·菲斯克（{{lang|en|Wilson Fisk}}）。尼可拉斯表面上是個生意人，暗地裡專門做些見不得光的買賣。尼可拉斯想要脫離黑手黨重新做人，引起菲斯克不滿，他派遣一名殺手「[[靶眼_(漫威漫畫)|靶眼]]」去殺害尼可拉斯。麥特得知消息後，便出手相助。然而，尼可拉斯仍然被靶眼所殺，且靶眼殺尼可拉斯所用的武器正是夜魔俠的手杖，讓艾麗嘉誤以為夜魔俠就是殺父仇人，精通武術及格鬥的她，恨不得手刃夜魔俠。與此同時，記者{{link-en|班·尤立克|Ben Urich}}發現了麥特身為夜魔俠的秘密。艾麗嘉為了報殺父之仇而找上了麥特，在雙方的打鬥中，麥特因為出手有顧忌而受了重傷，艾麗嘉趁機揭開對方的面具，才知道原來夜魔俠就是麥特，此時靶眼也來了，和艾麗嘉展開決鬥，結果艾麗嘉被靶眼殺害，而麥特則趁機逃到教堂裡去，靶眼在後面緊追不捨。

在雙方的打鬥中，靶眼發現噪音是麥特的弱點，便製造噪音，欲殺了麥特。此時，麥特聽到遠處有人要開槍射擊，便巧施妙計，讓靶眼中彈，接著麥特把靶眼丟出窗外，靶眼因此被警方逮捕。而麥特也因此得知幕後主使者是菲斯克，而菲斯克正是十多年前殺死傑克的人，於是麥特決定要去找菲斯克算帳。經過一段時間的纏鬥後，麥特終於打敗菲斯克，不過他並沒有殺菲斯克，而是把菲斯克交給警方，而菲斯克在被警方帶走前撂下狠話，說自己出獄以後會來找麥特的麻煩。事情結束後，尤立克寫了一篇有關夜魔俠真實身分的報導，但隨後決定將其刪掉。在[[片尾彩蛋|片尾彩蛋]]片段中，受重傷送醫的靶眼用身旁的針筒精準的射死了蒼蠅，說道：「靶眼」。

== 角色 ==

* [[班·艾佛列克|班·艾佛列克]]飾演[[夜魔俠|麥特·墨鐸 / 夜魔俠]]，來自[[地獄廚房_(曼哈頓)|地獄廚房]]的盲人律師，幼時誤觸[[放射性廢料|放射性廢料]]，導致雙目失明，卻增強了其餘感官的能力，他也因此成為了維護正義的夜魔俠。艾佛列克於2001年10月簽約參演。該角原定由[[馮·迪索|馮·迪索]]飾演，但迪索選擇出演《[[玩命關頭|玩命關頭]]》（2001年）而婉拒<ref name="AffleckCast"/>。在2011年2月的一次採訪中，[[蓋·皮爾斯|蓋·皮爾斯]]表示，他曾被《夜魔俠》的劇組選為飾演夜魔俠的人選，但他拒絕了，因為「漫畫類的東西實在不是我喜歡的類型，真的」<ref name="Pearce interview"/>。艾佛列克的好友[[麥特·戴蒙|麥特·戴蒙]]透露，自己也曾是人選之一。戴蒙表示自己和班「都很喜歡這本漫畫書，但我當時不大信任劇本或導演」<ref name="Damon"/>。[[柯林·法洛|柯林·法洛]]也被外界認為是夜魔俠的演員人選之一，直到艾佛列克簽下合約飾演夜魔俠<ref name="Farrell"/>。身為一名粉絲，艾佛列克讀了《夜魔俠》的每一集，以將其原汁原味地呈現在大螢幕上。{{link-en|喬·克薩達|Joe Quesada}}認為其「賺到了」，因為身為主角的艾佛列克在克薩達與[[法蘭克·米勒|法蘭克·米勒]]一同創作《{{link-en|惡魔捍衛者|Guardian Devil}}》時就被拿來當做夜魔俠形象的參考<ref name="Comic-con"/>。艾佛列克表示，《夜魔俠》是他幼年時最喜歡的漫畫<ref name="MTV (2002-06-24)"/>，並解釋他為什麼選擇飾演這個角色：「每個人童年都有一樣東西，人們記著它並與它堅守在一起。這個故事正是給我的」<ref name="JoBlo.com (2002-06-20)"/>。此外，他還表示，「我不想讓別人來演夜魔俠，我深怕他們演得與漫畫不一樣，搞砸了它」<ref name="MTV feature"/>。{{link-en|史考特·泰羅|Scott Terra}}在片中飾演年幼時的麥特·墨鐸。泰羅於2002年3月簽約參演<ref name="IGN tidbits"/>。

* [[珍妮佛·嘉納|珍妮佛·嘉納]]飾演[[幻影殺手|艾麗嘉·納崔斯]]，億萬富翁尼可拉斯·納崔斯之女，精通武術，與麥特之間有情愫。許多女演員都曾是艾麗嘉的演員人選，如[[潘妮洛普·克魯茲|潘妮洛普·克魯茲]]、[[莎瑪·海耶克|莎瑪·海耶克]]、[[娜塔莉·波曼|娜塔莉·波曼]]、[[劉玉玲|劉玉玲]]、[[潔西卡·艾芭|潔西卡·艾芭]]與[[凱蒂·荷姆斯|凱蒂·荷姆斯]]等人<ref name="IGN (2002-01-27)"/>。最後，劇組列出了一張人選名單，上頭包含珍妮佛·嘉納、{{link-en|茱蓮娜·布蕾羅克|Jolene Blalock}}、{{link-en|米婭·梅斯特羅|Mía Maestro}}及[[蘿娜·米莎|蘿娜·米莎]]<ref name="Daredevil parade"/>。劇組最終選擇嘉納來飾演艾麗嘉<ref name="IGN (2002-01-27)"/>。嘉納形容艾麗嘉「堅強且沉著，美麗且聰明，她是一個優質的角色模範」<ref name="MTV (2002-06-03)"/>。她指出，該角在片中的服裝將會與漫畫中的不一樣。艾麗嘉在漫畫中穿著紅緞布，而在電影中，艾麗嘉穿的則是黑皮革製的衣服。嘉納解釋道，「紅色遮掩不了鋼絲，我知道這聽起來很荒謬，但是你多少得保護一下你的皮膚。他們把我甩上屋頂，我破皮了，所以想像一下如果我什麼保護措施都沒有會怎樣」<ref name="MTV feature"/>。

* [[麥可·克拉克·鄧肯|麥可·克拉克·鄧肯]]飾演[[金霸王_(漫威漫畫)|威爾森·菲斯克 / 金霸王]]，掌控全紐約市的集團首腦。鄧肯於2002年1月簽約飾演菲斯克<ref name="DuncanCast"/>。鄧肯入選時，體重290磅。為了飾演菲斯克，鄧肯需增重40磅。為此，鄧肯每天舉重30分鐘<ref name="UGO"/>。

* [[柯林·法洛|柯林·法洛]]飾演[[靶眼_(漫威漫畫)|萊斯特 / 靶眼]]，額頭上有個標靶印記的冷血神射殺手。

* [[強·法夫洛|-{zh-tw:強·法夫洛;zh-hk:莊·法來奧;zh-cn:乔恩·法夫罗;}-]]飾演{{link-en|佛吉·尼爾森|Foggy Nelson|法蘭克林·「佛吉」·尼爾森}}，麥特的律師好友。

* {{link-en|喬·潘托里亞諾|Joe Pantoliano}}飾演{{link-en|班·尤立克|Ben Urich}}，編寫「城市傳說」專欄的紐約記者，相信夜魔俠的存在。
 
* [[大衛·凱斯|大衛·凱斯]]飾演{{link-en|傑克·墨鐸 (漫畫)|Jack Murdock (comics)|傑克·墨鐸}}，麥特的父親，曾是擁有「惡魔」之名的拳擊好手。

* {{link-en|利藍·歐爾瑟|Leland Orser}}飾演威斯利·歐文·威爾奇（{{lang|en|Wesley Owen Welch}}），菲斯克的隨從秘書。

* {{link-en|藍尼·洛夫汀|Lennie Loftin}}飾演尼克·馬諾里斯（{{lang|en|Nick Manolis}}），紐約警局局長，亦為偵探。

* {{link-en|艾利克·阿瓦里|Erick Avari}}飾演尼可拉斯·納崔斯，艾麗嘉的父親，同時也是希臘船運大亨。

* [[艾蓮·朋佩歐|艾蓮·朋佩歐]]飾演[[凱倫·佩吉|凱倫·佩吉]]，律師辦公室的總機小姐。

* {{link-en|德瑞克·歐康納|Derrick O'Connor}}飾演伊夫瑞特神父（{{lang|en|Father Everett}}），知道麥特一切的神父。

此外，《夜魔俠》漫畫原著的作者[[史丹·李|史丹·李]]、[[法蘭克·米勒|法蘭克·米勒]]與[[凱文·史密斯|凱文·史密斯]]在片中客串出演<ref name="MTV feature"/>。{{link-en|馬克·馬戈利斯|Mark Margolis}}在片中客串飾演法隆（未掛名）<ref name="IMDb-Cast"/>，一名曾雇用菲斯克擔任殺手的歹徒。

== 發行 ==
本片由[[二十世紀福斯|二十世紀福斯]]發行，上映日期為2003年2月14日。

== 評價與票房 ==
《夜魔俠》上映後的第一個週末有4千萬美元的票房。最後在北美的票房有1億2百萬美元，全球的票房有1億7千9百萬美元。

而在評價方面，《夜魔俠》在電影評論網站[[爛番茄|爛番茄]]221篇評論中，有44%的影評家給予正面的評價。而另一個評論網站[[Metacritic|Metacritic]]統計35位評論家對此片的觀點後，得到的平均得分為42分。
