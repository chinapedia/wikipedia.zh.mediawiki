'''萨克森-迈宁根'''（'''Sachsen-Meiningen'''）是[[韦廷王朝|韦廷家族]]在[[图林根|图林根]]的一个[[韋廷家族恩斯廷系諸邦國|邦国]]。1680年由[[萨克森-哥达|萨克森-哥达]]分裂产生。1871年加入[[德意志帝国|德意志帝国]]。1918年德国君主制覆亡，1920年成为[[魏玛共和国|魏玛共和国]][[图林根州|图林根州]]的一部分。

== 成立背景 ==
萨克森-哥达公爵兼[[萨克森-阿尔滕堡|萨克森-阿尔滕堡]]公爵[[恩斯特一世_(萨克森-哥达)|恩斯特一世]]有12个儿子，其中有七人活到成年。1675年，恩斯特一世去世，他活着的最年长的儿子[[弗里德里希一世_(萨克森-哥达-阿尔滕堡)|弗里德里希]]继承了他的领地。但遭到其他儿子的反对。1680年，弗里德里希一世与六个弟弟通过协议瓜分了父亲的领土。其中恩斯特一世的第六子[[博恩哈德一世_(萨克森-迈宁根)|博恩哈德]]得到了迈宁根，并于次年称萨克森-迈宁根公爵。

== 简史 ==
随着1699年[[萨克森-科堡|萨克森-科堡]]和1710年[[萨克森-勒姆希尔德|萨克森-勒姆希尔德]]的绝嗣，萨克森-迈宁根扩大了国土。1735年，[[萨克森-科堡|萨克森-科堡]]的[[松讷贝格|松讷贝格]]归于萨克森-迈宁根，1753年，勒姆希尔德的三分之二归于萨克森-迈宁根。

19世纪初，萨克森-迈宁根先后加入了[[莱茵邦联|莱茵邦联]]和[[德意志邦联|德意志邦联]]。

1825年，[[萨克森-哥达-阿尔滕堡|萨克森-哥达-阿尔滕堡]]绝嗣，归于[[萨克森-科堡-萨尔费尔德|萨克森-科堡-萨尔费尔德]]和[[萨克森-希尔德布尔格豪森|萨克森-希尔德布尔格豪森]]。1826年，恩斯廷系诸邦国领地重新划分。萨克森-迈宁根得到了希尔德布尔格豪森、萨尔费尔德、[[坎堡|坎堡]]、克拉尼希费尔德等领地，领土大大增加。

1829年，萨克森-迈宁根公国实施[[宪法|宪法]]。1833年，图林根诸邦加入[[德意志关税同盟|德意志关税同盟]]。

1866年的[[普奥战争|德意志内战]]中，萨克森-迈宁根与[[奥地利帝国|奥地利帝国]]和[[巴伐利亚王国|巴伐利亚王国]]结盟，与[[普鲁士王国|普鲁士王国]]作战。结果是普鲁士王国获胜，萨克森-迈宁根公爵[[博恩哈德二世_(萨克森-迈宁根)|博恩哈德二世]]被迫逊位于其子[[格奥尔格二世_(萨克森-迈宁根)|格奥尔格二世]]。萨克森-迈宁根成为[[北德意志邦联|北德意志邦联]]的一个邦。

1871年，萨克森-迈宁根加入[[德意志帝国|德意志帝国]]。1918年[[德国革命|德国革命]]推翻了君主制，萨克森-迈宁根公国成为萨克森-迈宁根自由州，并于1920年并入[[图林根州|图林根自由州]]。

== 行政区划 ==
1826年后的萨克森-迈宁根公国分为四个县：
* [[迈宁根县|迈宁根县]]：面积749km²，1910年人口70933；
* [[希尔德堡豪森县|希尔德堡豪森县]]：面积777km²，1910年人口60409；
* [[松讷贝格县|松讷贝格县]]：面积344km²，1910年人口68033；
* [[萨尔费尔德县|萨尔费尔德县]]：面积598km²，1910年人口69541。
