'''萨克森-安哈尔特'''（{{lang-de|Sachsen-Anhalt}}）是[[德意志联邦共和国|德意志联邦共和国]]的一个[[州|州]]，州府在[[马格德堡|马格德堡]]。它与[[下萨克森州|下萨克森州]]、[[勃兰登堡州|勃兰登堡州]]、[[萨克森州|萨克森州]]和[[图林根州|图林根州]]相邻。

== 地理 ==
萨克森-安哈尔特的北部是平原，这里的城市与北德的城市有着共同的历史关系，它们中有些是[[汉萨同盟|汉萨同盟]]的成员。中部是肥沃的马格德堡平原。西南部是[[哈茨山|哈茨山]]。南部靠近萨克森的边界上以[[哈雷_(德国)|哈雷]]为中心是一个重要的工业区（有“化学三角”之称）。

== 经济与交通 ==
萨克森-安哈尔特有两个重要的工业区，在南部从[[哈雷_(德国)|哈雷]]开始一直到萨克森州的[[莱比锡|莱比锡]]是德国历史上传统的重工业区，尤其以[[化工|化工]]为主，其他工业如光学工业也是很著名的。中部以[[马格德堡|马格德堡]]为中心是另一个重要的工业区。[[马格德堡|马格德堡]]的优点是它交通方便。这座城市位于中欧东西交通要道之上，从[[巴黎|巴黎]]通向[[华沙|华沙]]，甚至[[莫斯科|莫斯科]]的[[高速公路|高速公路]]和[[铁路|铁路]]经过这里，从德国西部通向首都[[柏林|柏林]]的[[德國高速列車|德國高速列車]]也通过此處。贯通中欧的[[中欧运河|中欧运河]]和从[[捷克|捷克]]既可以航行的[[易北河|易北河]]在这里相交。

哈茨山区是德国的一个重要的国内旅游区。

=== 就业 ===
在[[兩德統一|兩德統一]]的过程中，原[[東德|東德]]的萨克森-安哈尔特州受到很大的经济冲击，一些过去东德重要的国营企业（如鲁纳石油提炼厂、罗伯特隆电子工业等）关闭，使萨克森-安哈尔特州许多人失业。萨克森-安哈尔特州是今天德国失业率比较高（2005年达20%）、负债也比较高的一个州，在德国[[德国行政区划|联邦州]]中属于相对贫困州。

== 历史 ==
萨克森-安哈尔特是一个非常年轻的州，[[第二次世界大战|第二次世界大战]]后它才成立。从历史上来看它的主要部分是[[普鲁士|普鲁士]]的萨克森省，加上二战以前的安哈尔特共和国和不伦瑞克公国的一部分。

1947年7月21日萨克森-安哈尔特州正式成立，但1952年7月23日就被解散了（东德当时废弃了州作为行政区，而直接国内分地区和县，以加强中央对地区的控制），1990年10月28日两德合併后再度成立。

== 政治 ==
从1990年萨克森-安哈尔特州再次成立后，该州的政府多次更换，原因往往是原州政府被控贪污或原政府成员曾为[[东德国家安全局|东德国家安全局]]工作（按德国法律原东德国家安全局的正式或非正式工作人员不得进入国家机构）。

在1998年的州政府选举中极右的[[德意志人民联盟|德意志人民联盟]]首次参加选举立刻获得了近13%的选票，全国为之大哗。

在2002年的一次大选中[[德國基督教民主聯盟|基督教民主联盟]]获胜，[[德国民主社会主义党|德国民主社会主义党]]获第二位，[[德国社会民主党|德国社会民主党]]第三，[[德国自由民主党|德国自由民主党]]第四，其它党派未能获得5%的选票落选。德意志人民联盟因内部分裂落选。

=== 最新选举 ===
在2011年3月20日的州议会选举中，结果是:德国基督教民主联盟(CDU)41，左翼党(Linke)29，德国社会民主党(SPD)26，联盟90/绿党(B90/G)9，基民盟和社民党组成了新一届大联合政府。

现在的州长是基督教民主联盟的[[Reiner_Haseloff|Reiner Haseloff]]。

== 行政区划 ==
[[File:LandkreiseSachsenAnhalt2007.png|right]]
