'''薩克森戰爭'''（{{lang-de|Sachsenkriege}}、{{lang-en|Saxon Wars}}）指西元772年開始至西元804年結束，由[[查理大帝|查理大帝]]統治的[[法兰克王国|法蘭克王國]]與[[薩克森人|薩克森人]]之間持續了三十年以上的戰爭<ref name="試論查理曼對薩克森人的策略分析〉">[http://www.dylw.net/lishi/104918.html 试论查理曼对萨克森人的策略分析]</ref>。

== 薩克森人的社會 ==
薩克森人在著於前1世紀的『[[高盧戰記|高盧戰記]]』（[[尤利烏斯·凱撒|蓋烏斯·尤利烏斯·凱撒]]）以及著於1世紀的『[[日耳曼尼亞志|日耳曼尼亞志]]』（[[塔西陀|普布利烏斯·科爾奈利烏斯·塔西陀]]）都有相關記錄，但直到2世紀中期的史書上才開始出現他們的記載。原先僅在德國北部的[[霍爾斯坦|霍爾斯坦]]地方西南部一帶居住，在2世紀到4世紀之間，其居住範圍開始擴大，到了4世紀後半至5世紀時，一部份薩克森人、[[盎格魯人|盎格魯人]]與[[朱特人|朱特人]]登陸[[大不列顛島|大不列顛島]]，成為今日英國人（[[盎格魯-撒克遜人|盎格魯-撒克遜人]]）的祖先。在6世紀初期，薩克森人的勢力範圍達到了[[萊茵河|萊茵河]]一帶。在7世紀末期吸納了大量部族後，其勢力範圍更是擴張至包含[[易北河|易北河]]到[[埃姆斯河|埃姆斯河]]及之間的廣泛區域。

薩克森人的社會由[[貴族|貴族]]、自由民及[[解放奴隷|解放奴隷]]所組成，與其他的[[日耳曼人|日耳曼部落]]不同，並不禁止貴族與其他身分的人通婚。6世紀後半，因薩克森人[[法蘭克人|法蘭克人]]之間的對抗開始激化，而導致薩克森人在政治上的進一步統合，與全體[[部族|部族]]有關的議題在[[威悉河|威悉河]]河畔的馬爾庫洛召開。在[[宗教|宗教]]方面，與[[法蘭克人|法蘭克人]]及[[哥特人|哥特人]]不同，他們並不接受[[基督教|基督教]]，而持續傳統信仰。

== 薩克森戰爭的開始與過程 ==
[[法兰克王国|法蘭克王國]]對薩克森人的衝突始於[[查理大帝|查理大帝]]的父親[[矮子丕平|矮子丕平]]。他於西元738年，在[[墨洛溫王朝|墨洛溫王朝]]時代擔任[[宮相|宮相]]時，便因領土擴張與基督教的傳教，而與不願改宗的薩克森人有著持續的衝突。[[法兰克王国|法蘭克王國]]的法蘭克人沿著萊茵河支流[[美茵河|美茵河]]殖民德意志地區，這也是日後的[[東法蘭克王國|東法蘭克王國]]的雛形

西元772年，查理大帝率軍出征薩克森人，並破壞了薩克森人的聖樹依爾米斯爾，漫長血腥的薩克森戰爭由此爆發。查理大帝先後對薩克森人發起了八次遠征，因法蘭克王國佔有軍事上的優勢，因此陸續有薩克森貴族投入旗下。西元777年，由法蘭克人與薩克森人雙方的貴族共同召開聯合會議，這也是薩克森王{{link-en|維杜金德|Widukind}}初次被記載在文獻上。维杜金德逃亡至當時[[丹麥人|丹麥人]]的國家，並獲得丹麥王的庇護以對抗法蘭克王國。

西元778年，查理大帝轉戰[[西班牙|西班牙]]的[[加泰隆尼亞|加泰隆尼亞]]。同年，维杜金德回到薩克森，率領薩克森部族突襲法蘭克王國的領土並大肆掠奪，但查理大帝回國後，情況急轉直下。每當查理大帝與维杜金德交戰後，便有大量的薩克森人遭到處刑<ref name="試論查理曼對薩克森人的策略分析〉"/>。西元780年，查理大帝率軍進攻易北河流域，並建造[[馬德堡|馬德堡]]與[[哈雷_(德國)|哈雷]]兩座城寨。西元782年，查理大帝在威悉河支流[[阿勒爾河|阿勒爾河]]河畔的[[費爾登|費爾登]]，屠殺4,500名薩克森[[戰俘|俘虜]]<ref>日本歷史學者坂井榮八郎,指出當時被稱為「國家」的法蘭克王國，在現代的理解看來更像是「教會」，也因為如此，薩克森戰爭的意義不同於其他戰爭，乃是基督徒對異教徒的「聖戰」。坂井（2003）</ref>，此被查理大帝邀請來擔任宮廷教師的英格蘭學者[[阿尔琴|阿爾琴]]所記錄。

西元785年，查理大帝以和平手段收服持續進行游擊戰來抵抗基督教的维杜金德，並以[[投降|投降]]為名改信基督教。同年，维杜金德率領他的部族投降，並在阿蒂尼宮接受[[洗禮|洗禮]]。查理大帝冊封维杜金德為[[薩克森公爵|薩克森公爵]]。在這之後维杜金德便不再參與薩克森戰爭，而在[[修道院|修道院]]過著隱居生活。西元797年，查理大帝將不尊重基督教秩序者既為[[死罪|死罪]]的法令改為繳納罰金。西元804年，薩克森人完全臣服於法蘭克王國，薩克森戰爭因而結束。查理大帝征服了大半的[[日耳曼尼亚|日耳曼尼亞]]，以移居法蘭克人到該地區來壓制薩克森人的反抗，並任命薩克森部族中的實權者為[[伯|伯]]（Graf）來統治薩克森人。

== 薩克森戰爭的結果與影響 ==
薩克森戰爭的結果，最直接的部分在於薩克森被併入法蘭克王國的版圖中，同時也決定了基督教在歐洲的地位。另外这场持续了三十三年的戰爭，在不斷的反叛與再征服中，給薩克森帶來無比的血腥，造成了四分之一的萨克森人死亡<ref name="試論查理曼對薩克森人的策略分析〉"/><ref name="历史巅峰处的永恒——影响世界的100帝王排行榜〉">[http://www.tqxz.com/book2/book_2_090.htm 历史巅峰处的永恒——影响世界的100帝王排行榜]</ref>。

這場戰爭後，萨克森人被迫改信基督教，而不愿信仰基督教以及后来又改信异教的人均被判处死刑，[[日耳曼尼亚|日耳曼尼亞]]也融入了基督文化区，而後[[日耳曼尼亚|日耳曼尼亞]]以單獨的政治實體誕生也脫離不了這場戰爭的影響<ref name="試論查理曼對薩克森人的策略分析〉"/>。
