'''福本清三'''（ふくもと せいぞう，1943年2月3日 - ）[[日本|日本]][[演員|演員]]。[[兵庫縣|兵庫縣]][[城崎郡|城崎郡]][[香住町|香住町]]（現・[[美方郡|美方郡]][[香美町|香美町]]香住区）出身。[[血型|血型]][[B型|B型]]。本名為'''橋本清三'''（はしもと せいぞう）。隸屬[[東映京都撮影所|東映京都撮影所]]。

資歷長達50年以上，在[[時代劇|時代劇]]或[[現代劇|現代劇]]中專門扮演「被殺的角色」，因此擁有「''被殺五萬次的男人''」的綽號<ref name="ctdy20140809">{{Cite web |date=2014-08-09 |url=http://www.cinematoday.jp/page/N0065318|title=“被殺五萬次的男人”福本清三得到主演男優獎！日本人初の快挙！|publisher=シネマトゥデイ|accessdate=2014-08-09}}</ref>。

== 来歷 ==
為6兄弟中的三男。中学畢業後，雖然在親戚開的米店工作，但卻羞於說出像「歡迎光臨」等招待客人的話，因此在1959年的15歲時，一起與經營房仲業的叔叔進入[[東映京都撮影所|東映京都撮影所]]，並成為「大部屋演員」。

自20歲後開始演出被殺的角色，並且在東映京都及關聯公司製作的全部 [[時代劇|時代劇]]中演出。当時因為隸屬京都撮影所，所以很少在[[東映東京撮影所|東映東京撮影所]]製作的[[電影|電影]]・[[電視劇|電視劇]]演出。在[[動作電影|動作電影]]及[[黑道電影|黑道電影]]中也多次演出，在電影『[[無仁義之戰_廣島死鬥篇|無仁義之戰 廣島死鬥篇]]』中，有一段被擊中的瞬間從坐姿立刻轉變成跳起來而死亡的場景，[[導演|導演]]・[[深作欣二|深作欣二]]本來預計讓福本穿著[[橡膠|橡膠]]衣演出，卻被他以[[殺陣|殺陣師]]的身分說「我的身體裝有發條，所以不用」而阻止，並單純依照自己身体能力演出這段場景。

1978年，在電視時代劇『[[柳生一族の陰謀|柳生一族の陰謀]]』中，演出裏柳生衆之一・キタノ而成為準主演成員，1982年上映的角川電影『[[伊賀忍法帖|伊賀忍法帖]]』，被拔擢為演出敵方・魔界五人衆之一・虚空坊的角色。這時經常與[[千葉真一|千葉真一]]率領的[[ジャパンアクションクラブ|ジャパンアクションクラブ]] (''JAC'')成員們共同演出，也受到真田廣之，志穗美悦子等人的深厚信賴。

定年直前的2002年，演出[[好萊塢電影|好萊塢電影]]『[[末代武士|末代武士]]』，扮演一位好幾次主角與其對話都不會回答的沉默武士。在最後一戰戰死的場景裡，發揮出過往被殺的演技而演出充滿迫力的「死法」。同作品的英文官方網站中，他也被註記成「kirareyaku（日語「斬られ役」，被殺的角色）」的身分。

2003年在東映已經到屆退之年，但之後仍以[[嘱託|嘱託]]的形式，設籍在東映並持續演員的事業，同時也擴展他的活動範圍。自『末代武士』與[[湯姆·克魯斯|湯姆·克魯斯]]一起演出開始，即表示「就算是屆退也不會退下來，一定要進軍[[好萊塢|好萊塢]]」。2006年，接受[[大河劇|NHK大河劇]]『[[功名的十字路口|功名的十字路口]]』的演出邀請，並在劇中發揮被殺的演技。

2010年，在特攝電視劇『[[假面騎士W|假面騎士W]]』中的[[劇中劇|劇中劇]]『風之左平次』中，以壞人角色登場，並在劇中劇中演出真正的時代劇。隔年的2011年，在假面騎士誕生40周年記念電影『[[オーズ・電王・オールライダー_レッツゴー仮面ライダー|オーズ・電王・オールライダー レッツゴー仮面ライダー]]』中，演出[[修卡|修卡]]的幹部‧黑將軍。

在著作中，稱自己「沒有代表作」。

在2014年6月14日上映的電影『[[太秦ライムライト|太秦ライムライト]]』中，初次以「專門演出被殺角色的老手演員」一角成為主角，並且在第18回[[ファンタジア国際電影祭|ファンタジア国際電影祭]]中，以第一位日本人的身分獲得影帝獎，同時也刷新了歴代最年長的演員得獎記錄<ref name="ctdy20140809"/>。
