'''福州黑猪'''是已经[[灭绝|灭绝]]的[[家猪|家猪]]品种，原产于[[中华人民共和国|中国]][[福建省|福建省]][[福州市|福州市]]郊区。

福州黑猪被毛为黑色、耳朵下垂且较大、体型较大，猪的额头有很深的菱形皱纹。母猪头胎产仔10～11头，经产仔12.2头<ref>{{Cite web |url=http://www.dadchina.net/info/zyread.asp?id=40 |title=http://www.dadchina.net/info/zyread.asp?id=40 |access-date=2009-06-29 |archive-url=https://web.archive.org/web/20160304094718/http://www.dadchina.net/info/zyread.asp?id=40 |archive-date=2016-03-04 |dead-url=yes }}</ref>。

1973年，福建省政府曾经成立福州黑猪育种协作组并建立三个[[品系|品系]]。但由于生产速度慢且[[瘦肉率|瘦肉率]]低，福州黑猪逐渐退出市场。据《福建省志·畜牧志》，1990年年末福州黑猪为3000头<ref>[http://www.fjsq.gov.cn/ShowText.asp?ToBook=67&index=20& 畜牧志 一、地方品种]</ref>。1997年时，福州市就已经看不到福州黑猪<ref>
{{Cite web |url=http://www.shac.gov.cn/scfx/hqfx/xu/t20010417_3212.htm |title=福建福州黑猪无觅处（2001.04.17) |access-date=2009-06-29 |archive-url=https://web.archive.org/web/20040910101644/http://www.shac.gov.cn/scfx/hqfx/xu/t20010417_3212.htm |archive-date=2004-09-10 |dead-url=yes }} {{webarchive|url=https://web.archive.org/web/20040910101644/http://www.shac.gov.cn/scfx/hqfx/xu/t20010417_3212.htm |date=2004-09-10 }}
</ref>。2004年，有报道称福州黑猪濒临灭绝<ref>
{{Cite web |url=http://www.fj.xinhuanet.com/news/2004-11/11/content_3202142.htm |title=部分特种畜禽濒临灭绝 福建启动保护规划 |access-date=2009-06-29 |archive-url=https://web.archive.org/web/20041214162405/http://www.fj.xinhuanet.com/news/2004-11/11/content_3202142.htm |archive-date=2004-12-14 |dead-url=yes }} {{webarchive|url=https://web.archive.org/web/20041214162405/http://www.fj.xinhuanet.com/news/2004-11/11/content_3202142.htm |date=2004-12-14 }}
</ref>。2007年，福建省畜牧总站对全省畜禽遗传资源进行调查，发现福州黑猪已经消失<ref>
