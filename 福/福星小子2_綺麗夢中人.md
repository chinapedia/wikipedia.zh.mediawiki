《'''福星小子2 綺麗夢中人'''》（{{lang-ja|うる星やつら2　ビューティフル・ドリーマー}}）是於1984年上映的[[日本動畫|日本動畫]]，由[[押井守|押井守]]執導，以基於漫畫家[[高橋留美子|高橋留美子]]的作品《[[福星小子|福星小子]]》為背景製作的劇場動畫。

== 劇情簡介 ==
《福星小子》主角諸星當一行人為了在他們的學校友引高中準備校祭而忙碌、並在過程不時打打鬧鬧，而他們的班導溫泉老師卻一臉疲累地來到學校。

之後校內的護士櫻花老師前去觀看溫泉老師的異樣時，溫泉老師表示著他突然發現自己好像一直困在「校祭舉行的前一天」的時空裡。隨後不久諸星當的其他朋友們在晚上回家時，也發現似乎被某種奇異的因素、給拘限在無法離開友引高中一帶範圍的不可思議現象。

== 製作內容 ==
此動畫劇情主軸有受到思想家[[莊子|莊子]]提過的故事《[[莊周夢蝶|莊周夢蝶]]》所影響，並因此刻意在動畫中喫茶店裡安排一小段穿插蝴蝶出現的畫面。<ref name="DVD_Audio_Commentary">出自日版DVD中作品講解音軌。</ref>另外義大利導演[[費德里柯·費里尼|費德里柯·費里尼]]、法國導演[[尚盧·高達|尚盧·高達]]的電影作品，以及善於以非現實物體架構為畫作題材的荷蘭畫家[[莫里茨·科內利斯·埃舍爾|莫里茨·科內利斯·埃舍爾]]，也是此動畫內容、風格靈感來源。<ref name="DVD_Audio_Commentary"/>而動畫中的城鎮背景則是以當時押井守為了製作動畫；在沿著鐵道支線[[西武新宿線|西武新宿線]]一帶所暫居的住所週遭為範本。<ref name="DVD_Audio_Commentary"/>

== 原作者評價 ==
原作者高橋留美子雖曾在連載《福星小子》的漫畫雜誌《[[週刊少年Sunday|週刊少年Sunday]]》上對於回覆讀著詢問的專欄上提過「押井守是天才，這部動畫是押井守的傑作、觀眾可以好好地去享受」的內容。不過在由小說家[[平井和正|平井和正]]所編寫的採訪集上，有對於這部動畫的看法為「這是屬於押井守的《福星小子》」。<ref>{{cite book |script-title=ja:高橋留美子の優しい世界―女神の“おもしろユートピア” |date=1988年7月 |publisher=[[德間書店|德間書店]] |author=平井和正 |isbn=978-419153-711-8 |accessdate=2014年12月6日 }}</ref>押井守另在2003[[東京國際影展|東京國際影展]]上表示著有聽到高橋留美子反應過「在角色人性上表現不一樣」的意見。<ref>{{Cite web |url=http://www.kyo-kan.net/oshii-ig/report/200301106.html |script-title=ja:日本アニメーション特集 月刊ニュータイプ Selections Animation Odyssey 2003 ～検証！ 監督たちの劇場デビュー作～ |work={{lang|ja|野良犬の塒}} |accessdate=2014年12月6日 }}</ref>而此動畫在近二十年後重製發行的DVD版本裡有關解說作品的音訊裡頭，有提及「高橋留美子最喜愛的《福星小子》劇場版是第一作《唯有你》；最討厭的則是本作《綺麗夢中人》」、和「此動畫對於高橋留美子來說就像是『逆鱗』的行為」。<ref name="DVD_Audio_Commentary"/>

== 後續 ==
押井守多年後在[[朝日新聞|朝日新聞]]在訪談上表示著；當時在《綺麗夢中人》獲得多數正面回響後，於是次回執導的動畫《[[天使之卵|天使之卵]]》裡便抱持著「純粹以自己喜好的角度來製作應該也不會出太大問題」的想法。而《天使之卵》在上映後反而獲得較多負面意見，讓他過了將近三年低潮的時間。<ref>{{Cite web |url=http://www.asakyu.com/column/?id=824 |script-title=ja:ひとりよがりの罠（わな）がある |publisher=[[朝日新聞|朝日新聞]] |accessdate=2014年12月6日 }}</ref>
