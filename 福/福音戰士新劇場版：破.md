《'''福音戰士新劇場版：破'''》（{{lang-ja|ヱヴァンゲリヲン新劇場版:破　EVANGELION:2.0 YOU CAN (NOT) ADVANCE.}}）是2009年公映的[[日本|日本]][[動畫電影|動畫電影]]。是[[福音戰士新劇場版|福音戰士新劇場版]]系列的第二部。

== 概要 ==
[[福音戰士新劇場版|福音戰士新劇場版]]系列是重新制作的《[[新世纪福音战士|新世纪福音战士]]》的再构筑作品 (REBUILD)。本作是系列(全4部予定)第二部，于2009年6月27日在日本公映。2007年公映的《[[福音戰士新劇場版：序|福音戰士新劇場版：序]]》的续作。标题-{『破』}-取自[[日本雅樂|日本雅樂]]用語-{「序破急」}-。2010年5月26日[[Blu-ray|Blu-ray]]/[[DVD|DVD]]版《EVANGELION 2.22 YOU CAN (NOT) ADVANCE.》开始发售。

本作延续前作《序》的制作班底：[[庵野秀明|庵野秀明]]总监督，[[鶴卷和哉|鶴卷和哉]]、[[摩砂雪|摩砂雪]]监督。《序》讲了原作TV版前6话的情节，而《破》涵盖从第8话<ref group="注">明日香、来日</ref>到第19话<ref group="注">男子汉的战斗</ref>的内容。这个范围非常广，为了把它做得更像一部电影，没有用太多TV版的内容，有相当的一部分都是新作的<ref name=24FrameV57>鶴卷和哉专访，刊载于杂志《24格》第57期</ref>。有新角色（[[真希波·真理·伊拉絲多莉亞斯|真希波·真理·伊拉絲多莉亞斯]]）、新EVA机体的登场；设定、剧情等都进行了大幅的修改；以及[[惣流·明日香·蘭格雷|明日香]]的姓氏「惣流」改成「式波」。同時投入更多已趨成熟的CG技術，實際呈現出全新的作品風格<ref name=info>[https://web.archive.org/web/20110907194152/http://www.prowaremedia.com.tw/eva_newmov02_info.htm 作品解說]. 普威爾國際</ref>。

== 故事內容 ==
在旧北极的NERV基地，被解剖了的'''第3使徒'''逃脱封印，随后由'''真希波'''駕駛'''EVA暫設5號機'''迎擊。受到重创的5号機破坏使徒核心后，插入拴脫離、机体自爆。
鏡頭轉到第3新東京市，真嗣與源堂到唯的墓碑前掃墓。回途中'''第7使徒'''来袭，'''明日香'''驾驶'''EVA2號機'''登場迎擊、將其殲滅。之後明日香与真嗣一样搬到了美里家，並轉學到了同一所學校。

加持邀请真嗣、綾波、冬二、劍介以及明日香參觀將紅色海水還原的機構，真嗣在加持的話語中知道了美里加入NERV的原因，也就是第二次衝擊。
衛星軌道上出現的'''第8使徒'''发动重力攻击，美里制定三架EVA共同作戰計畫。在使徒快要墜落於第3新東京市時，初號機（真嗣）用A.T.力場承接，核心則由零號機（綾波）固定，2號機（明日香）将其擊破。兩架EVA都受了相當嚴重的損傷。修理期間真嗣等人度過了一段快樂的校園時光。

NERV北米第2支部进行的EVA4号机试验导致支部被毁灭。之后3号机移交日本，并安排于松代进行启动实验。礙於「梵諦岡條約」規定一國只能有三架EVA，2號機遭到封印，明日香转而担任3號機启动实验的测试驾驶员。实验当天，明日香搭乘的3號機被使徒給入侵并引起爆炸。之后认定为'''第9使徒'''，源堂命令初號機出擊，真嗣因為知道明日香在插入拴内而無法出手。之后在源堂的命令下初號機轉由傀儡系統操控，最终以暴力的戰鬥方式將3號機分屍并咬碎3號機插入拴（使徒核心）。奇蹟生還的明日香因可能遭受使徒的精神污染而被隔離。自责的真嗣决定不再駕駛EVA并離家出走。

最强的拒绝型'''第10使徒'''來襲，秘密入境的真希波駕駛2號機在地下空间迎击使徒。第10使徒的A.T.力場非常強，即使2号机变身'''兽化第2形态(The Beast)'''仍然不敵。之后零號機手持N2航空诱导弹零距離攻擊使徒核心也失败了。結果零號機被使徒捕食，吸収了零号機和綾波的第10使徒变身人型。目睹使徒造成的惨状及零號機被捕食后，真嗣又决定駕駛初號機出擊。
一番战斗后，初号機電源耗尽、活动停止。在真嗣喊出「把綾波......還給我！」之后，初號機覺醒成「接近神的存在」（疑似神化第1觉醒形态），拘束具脫落。依靠这力量完全压制第10使徒，并從使徒中救出綾波。最终变身「光之巨人」（疑似神化第2形态），引发第三次衝擊。

渚薰駕駛'''6號機'''（Mark.06）赶到，以卡西烏斯之槍贯穿初號機，第三次衝擊中止。

=== 次回予告 ===
綾波与真嗣一同被困在冻结了的EVA初号机。被废弃的要塞都市，被幽禁的NERV人员。被投进终极教条区的EVA6号机，还有蠢蠢欲动的EVA8号机及其驾驶员。这群被命运操纵着的孩子，终于集合起来了。到底，这群渴望生存下去的人，最终会走向何方？

次回：'''[[福音戰士新劇場版：Q|福音戰士新劇場版：Q]]'''

=== EVANGELION ===
{{see|Evangelion}}
以对抗「使徒」为目的而建造的人型兵器。由[[NERV|NERV]]负责开发、持有和使用。驾驶员坐进圆筒形驾驶舱「插入栓」中，主要是通过神经连接对機体进行操作。由脐带电缆提供电力，内置电力最大可支持5分钟的活动时间。與使徒一样拥有A.T.力场，可以与使徒的A.T.立场相中和、侵食或破坏。

* 初號機（EVANGELION試驗初號機　EVANGELION TEST TYPE-01）
: 双眼、头上有一长角、紫色涂装嵌有萤光绿色。最终超越人型，覺醒成疑似神化形态。
* 零號機(改)（EVANGELION試作零號機　EVANGELION PROTO TYPE-00'）
: 单眼、山吹色涂装的機体。最终被第10使徒吞食。
* 2號機（EVANGELION正規實用型 2號機(先行量産機)　EVANGELION PRODUCTION MODEL-02）
: 四眼、红色機体。与TV版的「贰号机」相比外型有所改变，特别是头部有一对角。对战第10使徒时启用「The Beast」变形成'''獸化第2形態（第1種）'''。
* 3號機（EVANGELION正規實用型 3號機　EVANGELION PRODUCTION MODEL-03）

=== 使徒 ===
{{see also|使徒 (新世纪福音战士)}}
向第3新東京市袭来的未知敌性生命体。形状和能力都各不相同，但必定会有叫做「核心」的部位（通常为红色球体）、此处被破坏之后就会停止活动，全身变作血一样的液体（形象崩坏）。

* 第2使徒 Lilith
: 於前作《序》曾登場，一直封印在NERV最底層。本作中仅以一个画面出现，并无更多信息。
* 第3使徒，新登場。造形设定：[[鬼头莫宏|鬼头莫宏]]
: 从永久冻土发掘出来，在NERV欧洲支部北極基地解剖、封印的使徒。骨骸外殼的龍形體，武器是光线。最终被暫設5號機破坏核心、歼灭。
* 第7使徒，新登場。造形设定：[[コヤマシゲト|コヤマシゲト]]、小松田大全
: 出现于相模湾、[[饮水鸟|饮水鸟]]造型的使徒，能使海面结冰在其上行走。被从空中袭击的2號機破坏伪装核心后，身体一度崩坏，之后复活、变形。真正的核心被破坏后形象崩壊。
* 第8使徒，参考原TV版「空天使Sahaquiel」之重新設計。造形设定：总体[[前田真宏|前田真宏]]、人型部分[[本田雄|本田雄]]
: 从空中落下的使徒。初始因扭曲光线呈现黑色球體，之后身体展開、变形。与初号機接触后，人型的本体出现。高速运动的核心被零號機固定后，由2號機擊破。
* 第9使徒，参考原TV版「霰天使Bardiel」之重新設計。造形设定：[[山下育人|山下育人]]
: 侵蝕並寄生在EVA3號機之內。
* 第10使徒，参考原TV版「力天使Zeruel」之重新設計。造形设定：[[淺利義遠_(漫畫家)|淺利義遠]]
: 最强的拒绝型。带状的触手可自由伸缩进行攻击。捕食零号機(改)后变身人型。

===原声带===
电影配乐依然由[[鷺巢詩郎|鷺巢詩郎]]负责。本作有很多日常的场景，庵野导演在考虑这些场景要用什么样的音乐时，偶然听到了自己导演、鷺巢詩郎配乐的电视动画《[[男女蹺蹺板|男女蹺蹺板]]》里的曲子。当时就觉得可以拿来用，然后向鷺巢詩郎提出加入《男女蹺蹺板》里的曲子，并重新编曲。
电影[[原声音乐|原声音乐]]专辑于2009年7月8日发售。发行商为[[STARCHILD|STARCHILD]]。分为收录有电影版BGM时长、完整时长2两种的2碟装专辑《SPECIAL EDITION》与仅收录由Disc 1（电影版BGM时长）的通常版两个版本发售。电影主题曲《[[Beautiful_World/Kiss_&_Cry|Beautiful World -PLANiTb Acoustica Mix-]]》作为限定下载曲目并未收录。在此专辑发行之前的6月27日，《Les Bêtes》的混音版本《Les Bêtes-Royal Mirrorball Remix》在[[ITunes|ITunes]]上架，此曲也没有收录在本专辑中。由扮演[[綾波零|綾波零]]的配音演员[[林原惠|林原惠]]演唱的电影插曲《今日の日はさようなら》和《[[給我一雙翅膀|給我一雙翅膀]]》有收录在专辑中。

== 动画制作 ==
本作与前作《序》几乎是同时开发的，企画初期只是打算弄成总集篇，并未想过要把原作破坏。《序》上映后，备受好评，次回予告也获得好评。[[庵野秀明|庵野秀明]]导演[[杀必死|杀必死]]精神大开，于是打算让新角色真希波多点出场、更接近故事中枢，这正是破（坏）的起点。除了真希波外，明日香的登场由于TV版第8话的原画遗失了，于是重新设计了使徒（第7使徒）、明日香的出场镜头也全部改掉了（2号机的设计也有所改变）。以及明日香驾驶EVA3号机（[[贞本义行|贞本义行]]的提案）,此提案也历经反复。[[鶴卷和哉|鶴卷和哉]]甚至建议把【在“重做一次”的过程中，想要改变但又不想改变】嵌入故事中（虽然被否决了，但鶴卷表示现状反而与此靠拢）。在故事的高潮（对应TV版第19话）甚至加入第23话的要素（綾波死掉）。<ref name=pamphlet>福音戰士新劇場版：破 场刊　鶴卷和哉专访</ref><ref name=docume>《福音戰士新劇場版：破　全記録全集》</ref>

经过《序》的试水后，本作积极地使用[[CG|CG]]。在片头部分，暂设5号机与第3使徒的战斗场面就是以全CG为前提进行设计制作的。<ref name=docume />
