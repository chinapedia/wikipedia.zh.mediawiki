福克蘭群島的領土總面積為12,000平方公里（4,700平方英哩），略小於[[康乃狄克州|康乃狄克州]]或[[北愛爾蘭|北愛爾蘭]]。它的總[[海岸線|海岸線]]長約為1,300公里（800英哩）<ref name="autogenerated1">{{cite web |url=https://www.cia.gov/library/publications/the-world-factbook/geos/fk.html |title=Falkland Islands (Islas Malvinas) |work=[[CIA|CIA]] - [[The_World_Factbook|The World Factbook]]}}</ref>。

== 地貌 ==
除了2个主岛外，其余岛屿还有[[塞瓦尔德斯群岛|塞瓦尔德斯群岛]]、[[森德尔斯岛|森德尔斯岛]]、[[佩布尔岛|佩布尔岛]]、[[莱夫利岛|莱夫利岛]]、[[布利克岛|布利克岛]]、[[斯皮德韦尔岛|斯皮德韦尔岛]]、[[纽岛|纽岛]]、[[比弗岛|比弗岛]]、[[威德尔岛|威德尔岛]]等。

[[东福克兰岛|东福克兰岛]]的[[尤斯伯恩山|尤斯伯恩山]](Mount Usborne)海拔达705米，为福克兰群岛的最高点。

== 气候 ==
马岛四周被南大西洋环绕，导致常年温度变化不大。月平均温度9℃，7月平均温度2℃。年降雨量平均573.6cm。
