'''福山雅治'''（{{jpn|j=福山 雅治|hg=ふくやま まさはる|rm=Fukuyama Masaharu}}，{{bd|1969年|2月6日|||catIdx=Fukuyama Masaharu}}），日本的[[創作歌手|創作歌手]]、[[演員|演員]]、[[電台|電台]][[DJ|DJ]]、[[攝影師|攝影師]]，出生於[[日本|日本]][[長崎縣|長崎]]，並擔任長崎故鄉大使。隸屬於[[AMUSE|AMUSE]]株式會社，所屬唱片公司為[[環球音樂|環球音樂]]，身高181cm<ref>{{cite web |url=http://www.vip-times.co.jp/?talent_id=M93-2807 |title=日本タレント名鑑 福山雅治のプロフィール |work=VIPタイムズ社| accessdate=2018-10-04}} </ref>，血型[[O型|O型]]。

除了早期剛出道時的少數作品，福山雅治的音樂詞曲都由他本人創作，根據日本[[Oricon|Oricon]]統計，截至2017年9月，福山雅治的音樂作品總銷量達到2410萬張，為日本歷代男性獨唱歌手的銷量冠軍<ref>{{cite web |url=http://www.oricon.co.jp/news/2097437/full/ |title=【オリコン】福山雅治、シングル18作目首位 歴代1位記録3部門を自ら更新 |work=ORICON STYLE |date=2017-09-19 |accessdate=2018-10-4}}</ref>。

除了音樂之外，福山雅治也參與電影及電視劇的主演，並多次擔任日本[[朝日電視台|朝日電視台]]的[[奧林匹克運動會|奧林匹克運動會]]專任攝影師<ref>{{cite web |url=http://www.asahi.com/olympics/columns/fukuyamaangle/prologue.html  |title=福山雅治さん、五輪を撮る|work=朝日新聞 |date=2012-07-23 |accessdate=2018-10-4}}</ref>。2015年與[[吹石一惠|吹石一惠]]結婚<ref>[http://www.cna.com.tw/news/firstnews/201509285010-1.aspx 福山雅治與吹石一惠閃婚]，中央社，2015年9月28日</ref>。

== 生平 ==
*1969年生於[[長崎縣|長崎縣]][[長崎市|長崎市]]，一路從宝珠幼稚園、長崎市立稻佐小學校、長崎市立淵中學校、長崎縣立長崎工業高級中學，幼稚園開始到高中畢業的求學生涯都在家鄉長崎度過。 
*1987年高中畢業後在長崎的電力公司就職。由於嚮往到[[東京|東京]]從事音樂工作，4個月之後就辭去上班族的工作，獨自[[上京|上京]]，在東京從事了披薩店店員等多項打工工作。
*1988年，福山雅治在[[橫田飛行場|橫田飛行場]]附近的木材店工作期間，參加經紀公司[[Amuse|Amuse]]的「10部電影主角選拔」（{{lang|ja|アミューズ・10ムービーズオーディション}}）甄選活動獲選之後，於電影《{{lang|ja|ほんの5g}}》以演員身份出道。
*1990年3月21日，發行第一張單曲《{{lang|ja|追憶の雨の中}}》，正式以歌手身份出道。
*1991年4月，於[[全國FM聯合|FM横浜]]的節目《ROCK AGE》首次擔任廣播節目主持工作。同年10月於[[TBS電視台|TBS電視台]]首次演出電視劇《愛有明天》（{{lang|ja|あしたがあるから}}），飾演若村良一。
*1993年於第44回[[NHK紅白歌合戰|NHK紅白歌合戰]]首次出場，演出曲目為《MELODY》。
*1996年至1997年暫時休止音樂活動，期間除了繼續擔任[[日本放送|日本放送]]的電台節目主持工作之外，只參加了Amuse的音樂會義演，及客串[[富士電視台|富士電視台]]電視劇《[[一個屋簷下|一個屋簷下2]]》。
*1990年代後半起，也開始以攝影家的身份活動，多次受邀擔任[[朝日電視台|朝日電視台]]的[[奧林匹克運動會|奧林匹克運動會]]專任攝影師，2006年起也陸續出版了數本攝影作品。在音樂、戲劇之外，也參與廣告、綜藝節目、運動節目、教育節目的演出，演藝活動範圍廣泛。
*2008年擔任「長崎故鄉大使」。
*2014年應[[台灣觀光局|台灣觀光局]]邀請，擔任2014年「台灣觀光親善大使」。
*2015年9月28日，與2012年曾被週刊報導交往消息的吹石一惠結婚<ref>[http://www.daily.co.jp/newsflash/gossip/2015/09/28/0008437861.shtml 福山雅治、吹石一恵と結婚発表]</ref><ref>[http://www.oricon.co.jp/news/2059877/full/ 福山雅治、吹石一恵と結婚「ささやかながらも明るい家庭を」【コメント全文】]</ref>。
*2016年12月，宣佈第一個孩子誕生，未公佈孩子性別。<ref>{{Cite news |url=http://hk.apple.nextmedia.com/enews/realtime/20161222/56081433 |title=【聖誕節前出世】福山雅治做老竇 全程陪生B 開心到喊 |date=2016-12-22}}</ref>

== 電視劇 ==
*1991年10月，[[TBS電視台|TBS電視台]]《愛有明天》（{{lang|ja|あしたがあるから}}）飾演若村良一。 
*1992年4~6月，TBS電視台《愛又怎麼樣》（{{lang|ja|愛はどうだ}}）飾演矢澤誠；福山雅治為此劇創作演唱的插曲《Good Night》銷售超過30萬張，成為他第一首打入[[Oricon公信榜|Oricon公信榜]]前10名的作品。
*1992年10~12月，TBS電視台《親密愛人》（{{lang|ja|ホームワーク}}）飾演瀧本周二。
*1993年4~6月，[[富士電視台|富士電視台]]《[[一個屋簷下|一個屋簷下]]》（{{lang|ja|ひとつ屋根の下}}）飾演柏木雅也；此劇平均收視率為28.4%，最高收視率達37.8%，建立了富士電視台「[[月九|月九]]」的地位。
*1995年7~9月，富士電視台《他日再相逢》（{{lang|ja|いつかまた逢える}}）飾演紺野伸一；此劇為福山雅治首次以主角身份演出電視劇。
*1997年4~6月，富士電視台《一個屋簷下2》（{{lang|ja|ひとつ屋根の下}}パート2）飾演柏木雅也。
*1998年4~6月，TBS電視台《邂逅》（{{lang|ja|めぐり逢い}}）飾演中田修二，女主角為[[常盤貴子|常盤貴子]]，講述兩人從邂逅以來經歷波折仍無法阻隔彼此思慕的愛情故事。
*1999年6月1日，富士電視台《[[古畑任三郎|古畑任三郎]]》第35集，飾演精英犯罪者堀井岳。
*1999年7~9月，富士電視台《無暇的愛》（{{lang|ja|パーフェクトラブ!}}）飾演牙醫楠武人。
*2003年1月2日，[[日本電視台|日本電視台]]特別劇《天國的大輔》（{{lang|ja|天國のダイスケへ~箱根驛伝が結んだ絆}}），飾演採訪佐藤大輔（[[小栗旬|小栗旬]] 飾）的記者飯田洋平。
*2003年1~3月，富士電視台《[[美女或野獸|美女或野獸]]》（{{lang|ja|美女か野獣}}），飾演玩世不恭的新聞製作人永瀨洋海，女主角為[[松嶋菜菜子|松嶋菜菜子]]。
*2007年10~12月，富士電視台《[[破案天才伽利略|破案天才伽利略]]》（{{lang|ja|ガリレオ(テレビドラマ)}}），飾演天才物理學家[[湯川學|湯川學]]副教授；協助[[柴咲幸|柴咲幸]]飾演的女警[[內海薰|內海薰]]偵破與科學相關的離奇案件，本作品的初回收視率達24.7%，並以平均21.91%的收視率成為當季收視冠軍。
*2010年1~11月，[[NHK|NHK]][[大河劇|大河劇]]第49作《[[龍馬傳|龍馬傳]]》（{{lang|ja|龍馬伝}}）飾演[[坂本龍馬|坂本龍馬]]；福山雅治為此作品首度將頭髮留長，並以此劇獲得「第48回ギャラクシー賞」電視部門個人賞。
*2013年4~6月，富士電視台《[[破案天才伽利略|破案天才伽利略]]2》（{{lang|ja|ガリレオ(テレビドラマ)}}第2シーズン）飾演湯川學副教授；女配角岸谷美砂由[[吉高由里子|吉高由里子]]飾演。本劇4月15日起在日本播出，[[台灣|台灣]][[緯來日本台|緯來日本台]]於日本播出後的同一週週六同期播出。
*2014年6月，台灣偶像劇《[[你照亮我星球|你照亮我星球]]》飾演福山雅治本人；演出內容於第1集、第6集、第7集播出。
*2016年4~6月，富士電視台《[[Love_Song_(電視劇)|Love Song]]》（{{lang|ja|ラヴソング(テレビドラマ}}）飾演神代廣平。
*2019年4~6月，TBS電視台《集團左遷!!》（{{lang|ja|集団左遷!!}}）飾演片岡洋。

== 人物 ==
*福山雅治為日本已故攝影大師{{Link-ja|植田正治|植田正治}}的弟子。
*2000年，擔任朝日電視台[[雪梨奧運會|雪梨奧運會]]官方攝影師。
*2001年，於[[東京巨蛋|東京巨蛋]]及[[大阪巨蛋|大阪巨蛋]]舉行演唱會。
*2004年，擔任朝日電視台[[雅典奧運會|雅典奧運會]]官方攝影師。
*2006年，於[[東京鐵塔|東京鐵塔]]舉辦個人小型回顧展 。
*2008年，擔任「長崎故鄉大使」、朝日電視台[[北京奧運會|北京奧運會]]官方攝影師。
*2009年12月31日，以《{{lang|ja|はつ恋}}》，由長崎連線第2次參加NHK紅白歌合戰，這也是自1993年{{Link-ja|第44回NHK紅白歌合戦|第44回NHK紅白歌合戰}}以來，福山雅治時隔16年重返紅白舞台。　　
*2010年，主演NHK大河劇《龍馬傳》，獲得「第2回観光廳長官表揚」。於[[第61回NHK紅白歌合戰|第61回NHK紅白歌合戰]]以連線方式，第3次出場紅白。
*2010年9月17日，《[[哆啦A夢|哆啦A夢]]》卡通裡出現以福山雅治為原型所設計的新人物——[[源静香|源静香]]崇拜的超级巨星福山雅秋；對此角色安排完全不知情的福山雅治，在主持的「魂ラジオ」廣播節目裡收到聽眾的郵件通知此事，隨即於節目裡公開表示非常開心與榮幸；並表示即使當時在拍攝NHK大河劇《龍馬傳》行程忙碌，也願意親自為福山雅秋配音。由於此一契機，福山雅治隨即受邀即為2011年2月11日播出的《哆啦A夢3小時SP》以及2011年公開的劇場版《[[大雄與鐵人兵團|大雄與鐵人兵團]]》裡的{{Link-ja|福山雅秋|福山雅秋}}一角配音，這也是福山雅治第一次擔任[[聲優|聲優]]工作。
*2011年，獲得第48回電視銀河獎個人賞受獎。這一年，由於發生了[[東日本大震災|311東日本大震災]]，福山雅治的原訂的日本巡迴演唱會暫時中斷巡演；在震災之後，福山雅治由於體會到「家人的羈絆」而提筆創作出《{{Link-ja|成為一家人吧|家族になろうよ}}》。
*2011年1~6月，NHK《{{Link-ja|hot spots 最後的樂園|ホット・スポット 最後の楽園}}》第一季開始播出。
*2011年2~11月，《THE LIVE BANG!!》演唱會於日本18個城市巡迴演出54場，表演時間總計125小時。
*2011年12月，於[[第62回NHK紅白歌合戰|第62回NHK紅白歌合戰]]以連線方式，第4次出場於紅白。
*2012年，繼3月28日的《{{Link-ja|生きてる生きてく|生きてる生きてく}}》之後，10月10日發行的第29支單曲《{{Link-ja|Beautiful life／GAME|Beautiful life／GAME}}》獲得[[Oricon公信榜|Oricon公信榜]]冠軍。
*2012年3月20日，福山雅治出道第22年前夕，官方影迷俱樂部《BROS.TV》網路直播開始，福山雅治親自擔任局長演出，於每月播出1次，每次約1小時（《BROS.TV》於2014年3月暫時中止，2015年3月重新開始放送）。擔任朝日電視台[[倫敦奧運會|倫敦奧運會]]官方攝影師。2012年底，於[[第63回NHK紅白歌合戰|第63回NHK紅白歌合戰]]以連線方式，第5次出場紅白。
*2013年4月，單曲《{{Link-ja|誕生日には真白な百合を|誕生日には真白な百合を}}》獲得Oricon公信榜冠軍。
*2013年12月31日，福山雅治於日本[[橫濱|橫濱]]舉辦跨年演唱會，並依照慣例在他的跨年演唱會上，以連線方式出場NHK紅白歌合戰。由於福山雅治這一年的跨年演唱會首度對海外（台灣、香港）的電影院開放衛星實況轉播權，因此福山雅治在他第6次出場的[[第64回NHK紅白歌合戰|第64回NHK紅白歌合戰]]裡，由[[橫濱國際平和會議場|橫濱國際平和會議場]]的跨年演唱會現場，牽線促成東京[[澀谷|澀谷]]的[[NHK大廳|NHK紅白歌唱大賽現場]]與[[台北101跨年煙火表演|台北101跨年晚會]]現場的三地衛星連線機會，這也是NHK紅白歌合戰首度對海外進行三地連線。
*2014年2月25日，應台灣觀光局邀請，擔任2014年「台灣觀光親善大使」。福山雅治除了到台北市[[迪化街|迪化街]]為台灣拍攝旅遊宣傳短片，在他從4月5日由東京巨蛋開始的2014年日本全國六大[[巨蛋|巨蛋]]巡迴演唱會中，一路由大阪巨蛋、[[名古屋巨蛋|名古屋巨蛋]]、[[札幌巨蛋|札幌巨蛋]]、[[福岡巨蛋|福岡巨蛋]]、到[[埼玉超級競技場|埼玉超級競技場]]，6個巨蛋會場、14場演出的每回開場前，都會播出這支推廣台灣旅遊的短片，並且在每位觀眾入場時，隨同演唱會紀念品，發送推廣台灣旅遊的宣傳廣告及明信片。
*2014年3月，福山雅治發行《{{Link-ja|HUMAN|HUMAN}}》專輯。即使走紅以來不乏有單曲蟬聯冠軍榜的經驗，但這是他首度以專輯在[[Oricon公信榜|Oricon公信榜]]內蟬聯雙週冠軍。福山雅治在這張專輯的宣傳活動裡，不只一次提到這是他第一次不以市場為取向，而是以一位邁入45歲的創作歌手身份，以自己的喜好為主而寫出的關於自己的體驗與思想的歌。
*2014年6月1日，完整公開受邀為台灣偶像劇「[[你照亮我星球|你照亮我星球]]」量身打造的主題曲《破曉》中文版，這是福山雅治首次為日本之外的電視劇創作主題曲，也是首度演唱中文歌曲。
*2014年3~5月，福山雅治於日本舉行全國六大巨蛋巡迴演唱會。
*2014年6月7日，福山雅治於[[台北小巨蛋|台北小巨蛋]]舉行出道24年以來，首次的海外演唱會<ref>[http://www.appledaily.com.tw/realtimenews/article/new/20140607/412208/ 福山雅治High翻小巨蛋 撂中文大秀親和力]</ref>，並首度公開演唱中文歌曲《破曉》。
*2014年6月13、14日，福山雅治於香港[[灣仔會議展覽中心|灣仔會議展覽中心]]Hall 5BC，舉辦了2場香港演唱會。
*2014年10月，NHK《hot spots 最後的樂園》第二季開始播出<ref>[http://www.nhk.or.jp/hotspot/index.html NHK 最後の楽園]</ref>。
*2015年3月28日，福山雅治主動結束於[[日本放送|日本放送]]主持23年的電台節目《{{Link-ja|福山雅治的All Night Nippon Saturary Special 魂的Radio|福山雅治のオールナイトニッポンサタデースペシャル 魂のラジオ}}》<ref>[http://www.oricon.co.jp/news/2050763/full/ 福山雅治、オールナイトニッポン23年の歴史に幕]</ref>，當天[[東京|東京]][[有樂町|有樂町]]的日本放送樓下，從下午開始有來自日本各地的聽眾聚集，當福山雅治進入錄音室前，也在日本放送大樓玄關前，對聚集的聽眾揮手致意。節目從日本時間晚上十一點半開始，在深夜一點節目結束之前，聚集在日本放送樓下人行道上的聽眾已接近3000人。福山雅治於深夜十二點半最後一集的節目結束之後，下樓以擴音器表示對於聽眾多年支持的感動與謝意，並叮囑大家小心回家<ref>[http://headlines.yahoo.co.jp/hl?a=20150329-00000006-mantan-ent 魂ラジ最終回にファン3000人 ANN足かけ23年「また、いつか…」と惜別]</ref><ref>[http://headlines.yahoo.co.jp/hl?a=20150328-00000102-dal-ent 福山雅治ANN重大発表なし 魂ラジ最終回にファン3千人]</ref>。
