'''福岡高等裁判所那霸支部'''（ふくおかこうとうさいばんしょなはしぶ）是[[福岡高等裁判所|福岡高等裁判所]]設置於[[沖繩縣|沖繩縣]][[那霸市|那霸市]]的支部。
== 所在地 ==
*[[沖繩縣|沖繩縣]][[那霸市|那霸市]]樋川1丁目14番1号

== 沿革 ==
*[[1950年|1950年]][[8月1日|8月1日]] - 在「琉球民裁判所制」（[[琉球列島美國民政府|美國民政府]]布告第12號）內設立「琉球上訴裁判所」。
*[[1968年|1968年]][[1月1日|1月1日]] - 《裁判所法》（1967年立法第125號）實施後，正式改名為「琉球高等裁判所」。
*[[1972年|1972年]][[5月15日|5月15日]] - 沖繩復歸日本後，改名為「福岡高等裁判所那霸支部」。

== 管轄區域 ==
在[[福岡高等裁判所|福岡高等裁判所]]的管轄範圍內，下列區域的訴訟案件由那霸支部管轄：
*[[沖繩縣|沖繩縣]]全境
