'''福知山市'''（{{jpn|j=福知山市|hg=ふくちやまし|rm=Fukuchiyama shi|f=y}}）為位於[[近畿地方|近畿地方]]北部，[[京都府|京都府]]西北部的行政區劃。是京都府內第三個成立的城市行政區劃，亦是京都府北部的主要都市。

轄區位於福知山盆地及周邊山區，其中約76%為山林地，市區位於盆地中{{tsl|ja|由良川}}沿岸。

== 歷史 ==
在令制國時代屬於[[丹波國|丹波國]][[天田郡|天田郡]]，另有少部分區域屬於[[與謝郡|與謝郡]]、[[何鹿郡|何鹿郡]]和[[丹後國|丹後國]][[加佐郡|加佐郡]]；戰國時代時{{tsl|ja|塩見頼勝|鹽見頼勝}}先在福知山盆地中央的丘陵建了橫山城，後來[[織田信長|織田信長]]家臣[[明智光秀|明智光秀]]攻下此地並負責統領，將橫山城整建並更名為[[福知山城|福知山城]]，此後此區以[[城下町|城下町]]形式開始發展<Ref name="Brief">{{Cite web| url=http://www.city.fukuchiyama.kyoto.jp/about/fukuchiyama/outline/| title=概要| publisher=福知山市政府| accessdate=2017-06-08| language=ja}}</ref>；[[江戶時代|江戶時代]]後，成為[[福知山藩|福知山藩]]領地，1669年後由{{Tsl|ja|朽木氏}}治理直到幕府時代結束。此外，現在轄區的東北部部分地區在江戶時代屬於{{Tsl|ja|丹後田辺藩|丹後田邊藩}}（後短暫更名為舞鶴藩）和{{tsl|ja|宮津藩}}。

1871年[[廢藩置縣|廢藩置縣]]後，福知山藩改設為福知山縣，舞鶴藩和宮津藩也改設為舞鶴縣宮津縣，同年這三縣旋即被整併到[[豐岡縣|豐岡縣]]；但在1876年豐岡縣被廢除，原豐岡縣轄區被分割分別併入[[兵庫縣|兵庫縣]]和[[京都府|京都府]]，而現屬福知山市的區域在當時都被劃入京都府。

1889年日本實施[[町村制|町村制]]，正式設立近代行政區劃'''福知山町'''，現在的福知山市範圍還包括當時天田郡下的其他20個村及與謝郡雲原村和何鹿郡佐賀村。1918年至1936年期間，曾我井村、庵我村、雀部村、下豐富村被併入福知山町，隨著規模的變大，福知山町也在1937年改制為'''福知山市'''。

此後在1950年代的[[市町村合併|昭和大合併]]和2000年代的平成大合併期間，先後再與上豐富村、下川口村、西中筋村、上六人部、中六人部、下六人部村、上川口村、金谷村、金山村、雲原村、{{tsl|ja|三岳村 (京都府)|三岳村 (京都府)|三岳村}}、佐賀村、{{tsl|ja|大江町 (京都府)|大江町 (京都府)|大江町}}、[[三和町_(京都府)|三和町]]、{{tsl|ja|夜久野町}}合併，形成現在的福知山市範圍。<ref>{{Cite web| url=http://www.city.fukuchiyama.kyoto.jp/about/fukuchiyama/history/| title=福知山市のあゆみ| publisher=福知山市政府| accessdate=2017-06-08| language=ja}}</ref>

== 交通 ==
福知山過去就位於[[京都|京都]]通往[[山陰地方|山陰地方]]的[[山陰道_(日本)|山陰道]]上，現在繼承的山陰道的公路[[國道9號_(日本)|國道9號]]也一樣通過了福知山市的主要市區。

1896年興建連結軍港{{tsl|ja|舞鶴港}}及大阪的{{tsl|ja|阪鶴鉄道|阪鶴鐵道}}通過了福知山地區，後來也成為現在的[[西日本旅客鐵道|西日本旅客鐵道]][[福知山線|福知山線]]，現在的[[福知山車站|福知山車站]]是福知山線與[[山陰本線|山陰本線]]的交會站，西日本旅客鐵道在此也設有{{tsl|ja|西日本旅客鉄道福知山支社|西日本旅客鐵道福知山支社|福知山支社}}及車輛基地。

而1923年開始經營的{{Tsl|ja|北丹鉄道|北丹鐵道}}也以福知山車站作為起點，但由於長期虧損此條路線在1971年停止營運，而後由[[北近畿丹後鐵道|北近畿丹後鐵道]]重整路線，開始經營通往[[宮津市|宮津市]]的[[宮福線|宮福線]]，但依舊由於虧損問題在2014年轉由[[WILLER_TRAINS|WILLER TRAINS]]經營。

福知山車站也為這三條鐵路交會，成為[[北近畿|北近畿]]鐵道網的中心車站。

== 觀光資源 ==

[[File:Fukuchiyama_castle01_2816.jpg|thumb]]

福知山市主要的觀光主題為[[明智光秀|明智光秀]]相關的歷史景點及{{tsl|ja|大江山}}[[酒呑童子|酒呑童子]]的傳說，因此主要景點包括1986年重建的[[福知山城|福知山城]]及以當地鬼傳說的{{tsl|ja|日本の鬼の交流博物館|日本的鬼之交流博物館}}。

== 教育 ==
轄內唯一的大學是{{tsl|ja|福知山公立大学|福知山公立大學}}，福知山公立大學是[[北近畿|北近畿]]唯一的四年制大學，過去原本為私立學校，成立時間可以追朔至1871年，2000年更名為京都創成大學，2010年再更名為成美大學，但此後由於經營不佳遭評鑑不合格，面臨被迫關閉，在2016年起改由福知山市政府接手，改組為市立的福知山公立大學。
