'''福康'''[[隕石|隕石]]是於2000年在[[中國|中國]][[新疆维吾尔自治区|新疆维吾尔自治区]]的[[阜康市|福康]]附近山中發現的。[[橄欖隕鐵|橄欖隕鐵]]是有著[[橄欖石|橄欖石]]結晶的美麗[[石鐵隕石|石鐵隕石]]。

== 歷史 ==
2000年，一位中國的批發商，柯作楷(Zuokai Ke)，在新疆靠近[[阜康市|阜康市]]的地方買了一塊1003公斤的隕石。在2005年2月，他從主體上裁切了20公斤，將這小塊帶去參加[[土桑寶石和礦物展|土桑寶石和礦物展]]，並遇見了[[亞利桑那大學|亞利桑那大學]]的D.S. Lauretta。事後，D.S. Lauretta、D. Hill、M. Killgore、D. Della-Giustina和Y. Goreva（分別服務於土桑的[[亞利桑那大學|亞利桑那大學]]、月球與行星研究室、和西南隕石中心）研究了這顆隕石。但近期柯作楷在接受《21世纪经济报道》采访时否认自己参与了该陨石的买卖。http://money.163.com/11/0528/01/753SDHQ300253B0H_2.html<ref name=metbull>[http://tin.er.usgs.gov/meteor/metbull.php?sea=Fukang&sfor=names&ants=%20&falls=%20&stype=contains&lrec=50&map=ge&browse=%20&country=All&srt=name&categ=All&mblist=All&phot=%20&snew=0&pnt=no&code=34491 Meteoritical Bulletin Database: Fukang]</ref>。

== 分類和成分 ==
福康橄欖隕鐵在[[鎳|鎳]]-[[鐵|鐵]]基中包含大量寶石等級的橄欖石或貴橄欖石。 
橄欖石有圓形到角質等許多不同的形狀，許多都有範圍從5毫米到數釐米不等的裂縫。主體的有些部分有最大直徑達到11釐米，有著薄金屬脈的大塊橄欖石群聚。
Fo<sub>86.4</sub>和克分子Fe/Mg = 0.1367、Fe/Mn = 40.37，和[[鎳|Ni]] = 0.03 wt%。金屬基主要是[[錐紋石|錐紋石]]，[[鎳|鎳]]的平均含量是6.98 wt%，含硫的蟲蛀形的[[磁黃鐵礦|隕硫鐵]]含有一些橄欖石<br/>氧的同位素：δ<sup>18O</sup> 2.569 ‰, δ<sup>17O</sup> 1.179 ‰, ∆1 7O = −0.157 ‰<ref>[http://tin.er.usgs.gov/meteor/docs/mb90.pdf Meteoritical Bulletin, no. 90, MAPS 41, 1383-1418 (2006)]</ref>。

== 標本 ==
總共有31公斤的標本存放在[[亞利桑那大學|亞利桑那大學]]，M. Killgore持有31公斤隕石，一位匿名者擁有隕石的主要部分<ref name=metbull />。在2008年4月，英國第二大的拍買公司邦漢思提出這顆隕石的主體在曼哈頓的拍賣場上拍賣。邦漢思的底價是200萬美金，但是未能賣出。一個19 x 36 英吋的窗口被切開並拋光，以提供觀看隕石內部的寶石區域<ref> [http://www.farlang.com/art/gemstone-meteorite Fukang Gemstone Meteorite] </ref>。
