'''福建省'''是[[中華人民共和國|中國]]经济发展最快的[[中国一级行政区|省份]]之一，2010年经济总量位居[[中华人民共和国省级行政区地区生产总值列表|全国第十二位]]。1979-2007年29年按绝对数比较，2007年[[福建省地区生产总值|GDP总量]]为1978年的138.02倍；按不变价格推算（以同期全国平均物价水平为基准），GDP年平均增长率12.4％，增长速度次于[[广东省地区生产总值|广东]]、[[浙江省地区生产总值|浙江]]等二[[中国一级行政区|省份]]居[[中华人民共和国省级行政区地区生产总值列表|全国第三位]]，2010年GDP增长13.8%，仅次于[[天津市|天津市]]、[[重庆市|重庆市]]、[[四川省|四川省]]，增幅居全国第四位。2016年GDP總量2.59萬億人民幣<ref>{{Cite web |url=http://www.chinatimes.com/realtimenews/20160210000828-260409 |title=大陸各省市GDP地圖 已有5省超越台灣 |accessdate=2018-04-30 |author=盧伯華、周凡鈺 |date=2016-02-10 |work= |publisher=中時電子報|language=Zh-hant }}</ref>。同年，福建省为[[中华人民共和国中央政府|中央政府]]提供了644亿元的财政盈余，在六个能为中央政府提供[[财政|财政]]盈余的[[省级行政区|省级行政区]]中排名第六<ref>{{Cite web |url=http://cj.sina.com.cn/article/detail/1010236564/349470?column=realty&ch=9 |title=重磅：神奇的财政平衡术，31省财政转移支付全景图 |accessdate=2018-04-30 |author= |date=2017-08-07 |work= |publisher=新浪网|language=Zh-hans }}</ref>。

== 历史 ==

1980年代以前，福建由于处于潜在的[[台海战争|台海战争]]前线，[[中华人民共和国中央政府|中央政府]]对福建的投资极少，经济发展缓慢。1978-2008年，有21年GDP年增长率高于10％，GDP增幅最高的年份为1993年，年增长率达到22.6％。[[福建省地区生产总值|福建省GDP总量]]在全国的位次从1978年的第22位，至2007年上升至第10位；福建省1981年GDP总量首次突破100亿元，1993年GDP突破1千亿元，2008年突破1万亿元。据初步核算，2010年福建省地区生产总值为14357.12亿元，比上年增长13.8%，人均地区生产总值39432元，比上年增长13.0%，三次产业比例为9.5∶51.3∶39.2<ref>

== 各类产业 ==

=== 工业 ===
1980年代以后，执行改革开放政策，合理利用[[侨资|侨资]]和[[外资|外资]]，引进先进技术和设备，先后建立厦门经济特区、[[马尾经济技术开发区|马尾经济技术开发区]]、沿海开放地区和台商投资区。2005年两岸开始互访，台海战争的可能性大大减少，以及中央政府提出加快海西发展，福建获得中央难得的投资。虽然经济迅速增长，但是福建至今还是[[中国|中国]]东南沿海省市人均GDP较低的省份。[[福州|福州]]、[[厦门|厦门]]、[[泉州|泉州]]所在的闽东南地区，经济较为发达。[[乡镇企业|乡镇企业]]稳步发展，农村[[外向型经济|外向型经济]]发展迅速。工业以轻型为主、门类齐全，形成以福州、厦门、泉州、漳州、三明、莆田、南平、邵武、永安、龙岩、漳平等为中心的工业区，以[[轻工|轻工]]、[[电子|电子]]、[[食品|食品]]、[[水产|水产]]加工为骨干的沿海工业和以[[原材料|原材料]]、[[纺织|纺织]]、[[森工|森工]]、[[化工|化工]]为骨干的内地铁路沿线工业配置的格局。

=== 农业 ===
福建多山，耕地缺乏。稻米是主要农作物。全省有17个[[粮食|粮食]]生产基地县。[[水田|水田]]面积约占耕地面积的81％，闽北、闽西北水田多为浸冬田型，宜种双季稻与单季晚稻；闽东南大面积[[水田|水田]]多为一年两熟或一年三熟的稻-稻-麦水旱轮作制。福建是中国发展[[热带|热带]]、[[亚热带|亚热带]][[经济作物|经济作物]]的重要基地之一。[[甘蔗|甘蔗]]亩产与[[含糖率|含糖率]]均居全国之冠。2010年福建省国民经济和社会发展统计公报显示全省粮食种植面积1848.45万亩，粮食产量661.89万吨，其中稻谷507.94万吨。面積
*稻谷面积1282.24万亩
*烟叶种植面积97.28万亩
*油菜种植面积167.49万亩
*蔬菜种植面积 1000.40万亩
