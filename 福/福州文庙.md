'''福州文庙'''（[[福州話|福州話]][[平話字|平話字]]：{{lang|cdo|Hók-ciŭ Ùng-miêu}}）是位于[[福州市|福州市]][[鼓楼区_(福州市)|鼓楼区]]圣庙路的一座[[文庙|文庙]]，又称“先师庙”，俗称“圣人庙”（{{lang|cdo|Séng-ìng-miêu}}）、“圣人殿”，现存建筑建于清末。1961年被福州市人民委员会公布为第一批市级文物保护单位；1996年被福建省人民政府公布为[[福建省文物保护单位|福建省文物保护单位]]，2006年被[[中华人民共和国国务院|中华人民共和国国务院]]公布为[[第六批全国重点文物保护单位|第六批全国重点文物保护单位]]。<ref name=":0">{{Cite news|url =http://www.sd.xinhuanet.com/news/2005-11/24/content_5666128.htm|title =福州文庙|last =|first =|date =2005-11-24|work =新华网山东频道|accessdate =|publisher =新华社|deadurl =yes|archiveurl =https://web.archive.org/web/20070912051405/http://www.sd.xinhuanet.com/news/2005-11/24/content_5666128.htm|archivedate =2007-09-12}}</ref><ref name=":1">{{Cite book|title = 福建省志·文物志|last = |first = |publisher = 方志出版社|year = 2002|isbn = 9787801226389|location = 北京|pages = |url = https://books.google.com.hk/books/about/%E7%A6%8F%E5%BB%BA%E7%9C%81%E5%BF%97.html?id=2XWiPwAACAAJ|accessdate = 2016-02-23}}</ref><ref name="国务院关于核定并公布第六批全国重点文物保护单位的通知">[http://www.gov.cn/zwgk/2006-06/02/content_297818.htm 国务院关于核定并公布第六批全国重点文物保护单位的通知]</ref>

==历史==
[[File:Foochow_confucian_temple_1926.jpeg|thumb]]。]]
福州文庙初为培育生徒的学宫，兼祀[[孔子|孔子]]，位于城西北。[[唐|唐]][[大历|大历]]七年（772年），[[福建观察使|福建观察使]][[李椅|李椅]]迁至现址。[[五代|五代]][[闽国|闽国]]时期，闽王[[王审知|王审知]]于921年将其改设为[[四门学|四门学]]，后来废止。到了[[北宋|北宋]][[太平兴国|太平兴国]]中期，转运使杨克让开始将其作为[[孔庙|孔庙]]。[[明|明]][[洪武|洪武]]七年（1374年）建大成殿与明伦堂，[[永乐|永乐]]四年（1406年）以学厅为乡贤祠，[[成化|成化]]十三年（1477年）大修庙学，凿[[泮池|泮池]]，并于池上修桥，[[弘治|弘治]]初年，改作[[棂星门|棂星门]]，以石料代替木材，[[嘉靖|嘉靖]]十一年（1532年）称先师庙。[[清|清]][[康熙|康熙]]十一年（1672年）大修庙学、棂星门。历史上文庙三次遭到火焚。[[咸丰|咸丰]]元年（1851年）八月初三，福州文庙遭火灾被毁后，于是年十二月至次年六月重建为现今的文庙。<ref name=":0" /><ref name=":1" />[[中国抗日战争|抗战]]时期[[日本军|日军]]进占福州后，曾闯入文庙，在大殿操练<ref name=":2">{{Cite news|title = 在福州孔庙读小学|author = 祥季|date = 2016-02-23|work = 海峡都市报|location = 福州|page = A21}}</ref>。1949年，文庙被改为福州市大根第一中心小学<ref>{{Cite book|title = 鼓楼区志|publisher = 方志出版社|year = 2001|isbn = 7801226267|location = 北京|chapter=学区|chapterurl = http://www.fjsq.gov.cn/ShowText.asp?Tobook=3163&index=1026&Query=1&}}</ref>，将两厢之走廊分隔成一间间教室。1950年代初学校搬至[[于山|于山]]南麓。<ref name=":2" />后来文庙又改建为红卫商场、少年宫等场所。21世纪初，少年宫搬出文庙，文物部门随之对文庙进行了修缮。<ref name=":0" /><ref name=":1" />

==建筑==
福州文庙现有建筑重修于1852年（清咸丰二年），占地7552平方米，建筑面积4000平方米，坐北朝南，按中轴线自南至北依次为外门埕、棂星门，泮池，仪门厅、大成殿，中轴线上的建筑依次为棂星门、仪门、大成殿；两侧则是廊庑、官厅、乡贤祠等。大成殿面阔七间，进深四间，重檐[[歇山顶|歇山顶]]，高19.6米，殿内四根石柱用材硕大，为文庙建筑所罕见，殿后檐间有“仰之弥高”匾额。<ref name=":0" /><ref name=":1" /><ref>{{Cite book |author=国家文物局 编 |title=全国重点文物保护单位（第六批）·第Ⅴ卷 |publisher=文物出版社 |year=2008 |isbn=978-7-5010-2446-9 |pages=400 }}</ref>
