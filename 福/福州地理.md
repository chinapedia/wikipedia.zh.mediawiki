[[福州|福州]]位于[[中国|中国]]大陆东南部，[[福建|福建]]东部的[[闽江|闽江]]口地区。其位置在东经118°08-{’}-～120°31-{’}-，北纬25°15-{’}-～26°29-{’}-之间。东部隔[[台湾海峡|台湾海峡]]与[[台湾|台湾]][[台北市|台北市]]相望，南部与[[泉州市|泉州市]]和[[莆田市|莆田市]]，西部与[[三明市|三明市]]，北部与[[南平市|南平市]]和[[宁德市|宁德市]]分别接壤。

==地形==
福州西部有鹫峰山—[[戴云山脉|戴云山脉]]，東面则是[[台湾海峡|台湾海峡]]，境内地势自西向東依次下降。地貌类型自西向東依次由[[中山_(地貌學)|中山]]而至[[低山|低山]]而较高的丘陵而至台地平原最后则是大海。全市总面积12,154平方公里，其中山地丘陵面积占福州总面积的72.68％,在闽江、[[鳌江|鳌江]]和龙江下游及山区盆地有多片小平原。[[闽江|闽江]]自西北向东南方向横切山脉，造成福州境内峡谷幽深，江面狭窄而且水流湍急，自安仁溪口以下，河谷逐渐开阔，水流平缓东流，最后至福州盆地形成福州境内最大的[[福州平原|福州平原]]。在福州平原周围，分别东有[[鼓山|鼓山]]、西有[[旗山國家森林公園|旗山]]、南有[[五虎山|五虎山]]、北有[[莲花峰|莲花峰]]环绕，其海拔多在600～1000米之间，因此地貌上也是一个典型的河口[[盆地|盆地]]。另外在[[鳌江|鳌江]]和龙江下游以及山区盆地也有较多小平原<ref name="搜狐体育"> {{cite web|url=http://2008.sohu.com/20070423/n249638254.shtml |title=福建省福州市(中国)-搜狐2008奥运 |accessdate=2008-07-26 |publisher=搜狐体育 |language=zh-hans }}</ref>。

福州[[海岸线|海岸线]]漫长曲折，北起罗源县与宁德市交界处的井水，南至福清市与莆田市交界处的[[江口镇|江口镇]]，多属[[基岩海岸|基岩海岸]]，由北至南跨越了[[罗源湾|罗源湾]]、闽江口、[[福清湾|福清湾]]和[[兴化湾|兴化湾]]等四个港湾<ref> {{cite web |url=http://www.fzmsa.gov.cn/InfoView.aspx?FCCD=101&CCD=10103 |title=辖区概况 |accessdate=2008-07-26 |publisher=福州海事局 |language=zh-hans |deadurl=yes |archiveurl=https://web.archive.org/web/20120419111920/http://www.fzmsa.gov.cn/InfoView.aspx?FCCD=101&CCD=10103 |archivedate=2012-04-19 }}</ref>，其中大陆海岸线长920公里，海岛海岸线长390公里<ref> {{cite web|url=http://news.fznews.com.cn/fuzhou/2008-2-28/20082285ZK+ZM5aWV111916.shtml |title=福建公布海岸线长度　福州大陆海岸线９２０千米
|accessdate=2008-07-26 |publisher=福州新闻网 |language=zh-hans }}</ref>。

==自然资源==
福州的矿产主要以[[叶腊石|叶腊石]]、[[花冈石|花冈石]]、[[硅砂|硅砂]]等非金属矿为主，福州也是[[寿山石|寿山石]]主要出产地。全市[[森林|森林]]覆盖率达54.8%左右，是中国三大林业基地之一。<ref name="福州市自然资源">
{{Cite web |url=http://www.fuzhou.gov.cn/html/20070425/929967.html |title=福州市自然资源 |access-date=2009-08-11 |archive-url=https://web.archive.org/web/20070831072236/http://www.fuzhou.gov.cn/html/20070425/929967.html |archive-date=2007-08-31 |dead-url=yes }} {{webarchive|url=https://web.archive.org/web/20070831072236/http://www.fuzhou.gov.cn/html/20070425/929967.html |date=2007-08-31 }}
</ref>福州有着丰富的[[温泉|温泉]]资源，全市市区地热总储量达9800立方米，是中国最大的市区温泉和三大温泉区之一，市区的温泉带位于城区东北部，北起树兜南至王庄，西起五一路东至六一路的温泉带，南北长5公里，东西宽1公里，面积达5平方公里，温泉出露带面积约占福州市面积的7%，而且水质优良，邻区还有桂湖、洪塘、宏屿、浦口等温泉分布。

==海洋==
福州海域总面积11.09万平方公里，[[海岸线|海岸线]]总长1137公里，占福建省岸线总长的三分之一。沿海多[[天然良港|天然良港]]，可兴建1～20万吨[[深水泊位|深水泊位]]的[[天然良港|天然良港]]有100多个：福清湾、罗源湾、兴化湾是中国少有的[[深水港湾|深水港湾]]。东部沿海的[[海坛岛|海坛岛]]面积251.4平方公里，是福建第一大岛，中国第五大岛。岛上的[[三十六脚湖|三十六脚湖]]，湖面积达2.1平方公里，是福建第一大天然[[淡水湖|淡水湖]]。其它较大的岛屿有[[江阴岛|江阴岛]]（已经连陆为[[江阴半岛|江阴半岛]]）、[[琅歧岛|琅歧岛]]、[[粗芦岛|粗芦岛]]、[[川石岛|川石岛]]、[[大练岛|大练岛]]等。有海洋鱼类500多种。
