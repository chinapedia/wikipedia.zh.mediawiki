'''福州火车南站站'''位于[[福州市|福州市]][[仓山区|仓山区]]胪雷路国铁[[福州南站|福州南站]]正下方，是[[福州地铁|福州地铁]][[福州地铁1号线|1号线]]一期与未来规划[[福州地铁5号线|5号线]]的终点站，该站于2016年5月18日投入使用。<ref name=":1">{{Cite web|url=http://finance.ifeng.com/a/20160518/14393601_0.shtml|title=福州地铁1号线南段试运营 最高票价3元|accessdate=2016-07-08|work=finance.ifeng.com}}</ref>

== 车站结构 ==
福州火车南站站结构与国铁福州南站共建，呈东西走向，主体结构为双柱三跨地下二层车站，地下一层为站厅层；地下二层为站台层，岛式站台。车站全长约500米，宽约31~56米，深14.6米。<ref name=":2">{{Cite web|url=http://fj.sina.com.cn/news/m/2010-12-30/073886934.html|title=福州火车南站地铁站300米延伸段拟进场开挖|accessdate=2019-01-02|author=|date=|work=fj.sina.com.cn|publisher=}}</ref><ref name=":3">

=== 站厅 ===
{{ multiple image | align = right | direction = vertical | width = 220
| image1 =20171118 Concurse of Fuzhou South Railway Station (Fuzhou Metro).jpg| caption1 = 站厅付费区（2017年11月）
}}
福州火车南站站站厅位于国铁福州南站地下一层。车站站厅采用蓝色波浪型吊顶与白色搪瓷板装修，设客服中心2处，分布于站厅付费区东西两端，服务设施设有智能导乘机、自动售货机、自动柜员机与自助拍照机，站厅东侧规划设有便利店，车站D出入口通道设有洗手间。<ref>{{Cite web|url=http://fzmtr.com/html/fzdt/2018-08-17/644429903.html|title=福州火车南站站_站点信息|accessdate=2019-01-02|author=|date=|work=fzmtr.com|publisher=}}</ref>

==== 车站艺术 ====
{{ multiple image | align = right | direction = vertical | width = 220
| image1 =Mural at Metro Fuzhou South Railway Station (20170128165237).jpg| caption1 = 《五虎雄姿》漆画（2017年1月）
}}
福州火车南站站吊顶以“海丝蓝”为空间主题色，利用舒展的线条造型体现海纳百川的福州城市精神与海上丝绸之路的灿烂文化。车站站厅付费区中侧设有一副漆画艺术墙，名称为《五虎雄姿》。漆画长14米，宽2米，采用漆画髹饰技艺制作。漆画运用独特的髹饰展示了闽江口日出与沿岸五虎山一带瑰丽、壮美的景色，亦展现了福州深厚的漆文化底蕴。<ref>{{Cite web|url=http://www.mnw.cn/news/fj/1512978.html|title=揭开福州地铁1号线全线的神秘面纱：地下福州城“福满多”-闽南网|accessdate=2019-01-02|work=www.mnw.cn}}</ref>

=== 周边 ===
福州火车南站站位于国铁福州南站正下方，所处位置是一座交通换乘枢纽。铁路售票处可由D1口出向南步行约50米到达；铁路进站可由B/C/E/F口出通过换乘层进站口直接进站，亦可由D1口出向南步行约50米通过楼梯由2楼出发层进站口进站；铁路出站口可由D1/D2口出站向东步行约20米到达；火车南站公交枢纽站与机场巴士站可由D2口出向北步行约200米到达；陈绍宽故居可由D1口出向西南步行约250米到达。

== 未来发展 ==
福州火车南站站为福州地铁5号线在建车站，5号线车站位于既有福州南站东广场，为地下三层车站，沿广场南北布置，5号线与1号线未来将通过长通道实现换乘，5号线开通后福州火车南站站也将实现对[[仓山区|仓山区]]螺洲、义序、浦上、金山与[[闽侯县|闽侯县]]永丰片区的直达。<ref>{{cite news|title=福州地铁5号线9月底前开建4号线年底前开建走向一览|url=http://www.fj.chinanews.com/news/2017/2017-02-09/373018.html|accessdate=2019-01-03|date=2017-02-08}}</ref>

== 历史 ==
* 2008年11月，车站主体开始建设。<ref name=":3" />
* 2010年11月，车站主体封顶。<ref>[http://news.163.com/10/1122/10/6M3C20BQ00014AED.html 福州地铁一号线火车南站封顶]</ref>
* 2010年12月，车站延伸段开始建设<ref name=":2" />
* 2012年4月，车站西延伸段主体结构完工。<ref>[http://www.fjsen.com/d/2012-02/23/content_7880943.htm 福州地铁火车南站站西段主体4月完工 15站点已围挡]</ref>
* 2014年6月15日，福州火车南站站-[[清凉山停车场|清凉山停车场]]区间开始盾构。<ref name=":0">
[http://mag.fznews.com.cn/shtml/fzwb/20140616/97670.shtml 清凉山停车场 盾构机开挖]{{Dead link|date=2019年2月 |bot=InternetArchiveBot |fix-attempted=yes }}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}
