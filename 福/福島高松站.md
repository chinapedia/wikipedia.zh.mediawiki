'''福島高松站'''（{{jpn|j=福島高松駅|hg=ふくしまたかまつえき|rm=Fukushima-takamatsu eki|f=y}}）是一個位於[[宮崎縣|宮崎縣]][[串間市|串間市]]大學高松的[[九州旅客鐵道|九州旅客鐵道]]（JR九州）[[日南線|日南線]]車站。

== 車站構造 ==
*單面月台1面1線的地上車站。
*[[無人站|無人站]]。

== 利用狀況 ==
* 在2006年，平均1日有4人上下車。

== 历史 ==
* 1949年（[[昭和|昭和]]24年）9月15日 - '''福島高松臨時停車站'''啟用。
* 1950年（昭和25年）1月10日 - 升級為'''福島高松車站'''。
* 1962年（昭和37年）8月1日 - 車站無人化。
* 1987年（昭和62年）4月1日 - 由九州旅客鐵道繼承車站。
