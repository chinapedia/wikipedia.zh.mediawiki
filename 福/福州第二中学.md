'''福州第二中学'''位於[[中国|中国]][[福建省|福建省]][[福州市|福州市]][[鼓楼区_(福州市)|鼓楼区]]，在[[乌山_(福州)|乌山]]麓，[[白马河_(闽江)|白马河]]畔<ref>{{Cite journal|title=求真务实创特色|author=邹黎华|url=http://www.cnki.com.cn/Article/CJFD2003-FJXB200309023.htm|journal=福建教育学院学报|issue=9|doi=|others=|year=2003|volume=|page=|pmid=}}</ref>，临[[光禄坊|光禄坊]]<ref name=":1" />[[光禄坊|{{Rp]]，是福建省一级达标[[高级中学|高级中学]]<ref name=":2">{{Cite web|url=http://www.gl.gov.cn/archive/2012/7/23/330105.html|title=福州二中|accessdate=2017-07-10|author=|date=2012-07-23|first=|publisher=数字福建鼓楼示范区建设领导小组办公室|archiveurl=https://web.archive.org/web/20170710142641/http://www.gl.gov.cn/archive/2012/7/23/330105.html|archivedate=2017-07-10}}</ref>，福州市属省重点中学<ref name=":1">{{Cite book|title=福州市教育志|last=|first=|publisher=|year=1995|isbn=|location=福州|pages=|chapter=普通中学教育}}</ref>{{Rp|145}}，也是福州市唯一一所[[中国人民武装警察部队|武警]][[国防生|国防生]]生源基地<ref name="中招专栏" />。简称'''福州二中'''。這所學校的歷史，可以追溯到西元1911年創建的福建法政专门学校附属中学堂。該校佔地面積3.45万[[平方米|平方米]]。<ref name=":1" />{{Rp|145}}及至2012年，有近2000名学生在校，教职工凡153人<ref name=":2" />。該校具備包括「科技節」<ref name=":1" />{{Rp|145}}、學生社團<ref name="中招专栏" />之類的學園文化，校训为“求知、求真、求实”<ref name=":3" />。詩人蒲風、[[雷石榆|雷石榆]]和畫家林子白等曾在此任教<ref name=":0" />{{Rp|45-46}}，前福建省省长[[黄小晶|黄小晶]]、[[中国工程院|中国工程院]]院士张文海等人曾在此就讀<ref name=":4" />。

== 校史沿革 ==
福州二中的校史，最早可追溯到福建法政专门学校附属中学堂。它於西元1911年创办，当时在福州白水井。1929年，改名私立福建学院附属中学。次年搬入光禄坊。[[中国抗日战争|抗日战争期间]]，迁到[[闽清县|闽清县]]、[[蒲城县|蒲城县]]。1945年迁回。1951年，私立厚美中学并入该校。<ref name=":1"/>{{Rp|145}}同年9月，福建师范学院附属中学并入<ref name=":1" />{{Rp|108}}，改名福建师范学院附属中学<ref name=":1" />{{Rp|145}}。翌年10月<ref name=":1" />{{Rp|145}}（一说9月<ref name=":1" />{{Rp|108}}），改名福州第二中学<ref name=":1" />{{Rp|145}}。

二中原有3年制初中部和3年制高中部。初中部於2000年停止招生。

1953年时，二中所在，为大根区{{Notetag|1=1956年，大根区并入[[鼓楼区_(福州市)|鼓楼区]]<ref name="民国共和国">{{cite book | title=鼓楼区志|chapter=第二节 境域 | publisher=方志出版社 | author=福建省福州市鼓楼区地方志编纂委员会 | year=2001 | location=北京 | isbn=7-80122-626-7 | quote=民国35年（1946年）元旦……西南至白马河北桥与台江区交界。}}</ref>。|name=大根區}}学院路<ref name=":1" />{{Rp|108}}。後校址记为光禄坊38号<ref name=":1" />{{Rp|145}}<ref name=":0">{{Cite journal|title=福州二中的闽台情缘|author=唐岱蒙|url=http://www.fjnet.cn/yhzh/2014-06/19/content_14321439.htm|last2=|journal=炎黄纵横|issue=6|doi=|others=唐嘉历|year=2014|volume=|page=45-46|pmid=}}</ref>{{Rp|45}}。而後又改为光禄坊100号<ref name=":0" />{{Rp|45}}。

== 布局施设 ==
[[File:福州二中教学楼.jpg|缩略图]]
校园佔地3.45万[[平方米|平方米]]，建筑面积2.07万平方米。<ref name=":1" />{{Rp|145}}学校教学楼有36间教室，每间教室65[[平方米|平方米]]，配有[[液晶显示器|液晶]][[投影機|投影仪]]、实物展示台、[[電子計算機|电脑]]等多媒体设备。普通教室以外，有[[物理学|物理]]、[[化學|化学]]、[[生物|生物]][[实验室|实验室]]11间、多媒体大教室3间、多媒体阶梯教室1间。此外有教师多媒体电子备课室、学生电子阅览室、理化生多媒体实验室、探究室、多媒体网络教室、多媒体学术报告厅。<ref name=":2" />

该校图书馆是省级示范图书馆<ref name=":2" />。1990年代初，该校图书馆藏书2.9万册<ref name=":1" />{{Rp|145}}。2012年时，藏书9万余册，各种期刊300余种，各类视频资料2000多盘。数字图书馆方面，在2012年有数字图书2万册，知识视频库800集，影视200部，音乐1200首，“知识大百科”5万条。<ref name=":2" />
[[File:福州二中操场一瞥.jpg|缩略图]]
校内有露天[[游泳池|游泳池]]和300[[米_(单位)|米]][[塑料|塑胶]][[跑道_(体育)|跑道]]、4个[[籃球場|篮球场]]、3个[[排球場|排球场]]、2个[[羽毛球場|羽毛球场]]、1个[[足球場|小足球场]]<ref name=":2" />。2011年，该校新体艺馆落成<ref name=":0" />{{Rp|46}}，建筑面积3700平方米，其中设[[体操|室内体操房]]、[[舞蹈|室内舞蹈室]]、室内篮球场馆<ref name=":2" />。

心理健康教育方面，校内设“育心园”，面积凡400多平方米。在园内，有专用教室、音乐放松室、多功能活动室、箱庭游戏室、个体辅导室、办公接待室。<ref name=":2" />此园亦为[[福建师范大学|福建师范大学]]和[[漳州师范大学|漳州师范大学]]心理学专业实践基地<ref>{{Cite journal|title=育人育心:阳光心理奠基幸福人生——小记福州二中特色心理健康教育|author=罗亚|url=http://www.cnki.com.cn/Article/CJFDTotal-PAXY201406006.htm|journal=平安校园|issue=6|doi=|others=|year=2014|volume=|page=|pmid=}}</ref>。

校园内曾经有过两栋青砖楼，名为文虎楼，由[[海外華人|华侨]][[胡文虎|胡文虎]]在1935年捐建。<ref name=":1" />{{Rp|145}}

== 教育情况 ==
1989年，第二中学有教职工169人，其中127人专任[[教师|教师]]，3人为[[特级教师|特级教师]]，28人为[[高级教师|高级教师]]。当年[[班級|教学班]]37个，1864名[[学生|学生]]在校。<ref name=":1" />{{Rp|145}}及於2012年，学校有近2000名学生在校，教职工153人。教职工中，特级教师2人，高级教师63人，[[一级教师|一级教师]]37人。从级别与学历上看，有2位国家级骨干教师，5位省级骨干教师，10馀省市优秀青年教师及市级骨干教师，10位[[硕士研究生|硕士研究生]]，12位[[教育硕士|教育硕士]]。<ref name=":2" />

== 学园文化 ==
福州二中自1984年起，每年秋季举办科技节<ref name=":1" />{{Rp|145}}。
在学生社团这一方面，该校有模拟联合国社、山神话剧社、风骨文社、足球社、街舞社、动漫社、器乐社、摄影社、机器人社、跆拳道社、魔术社等。其中，山神话剧社成立于1988年，其[[中国话剧|话剧]]作品《项链》、《荆轲刺秦王》曾在省、市内获奖。校足球队成立于1998年10月，成员来自各年段，球队吉祥物为“二哈”——[[西伯利亞雪橇犬|西伯利亚雪橇犬]]，亦谓[[哈士奇|哈士奇]]。<ref name="中招专栏" />
该校是福建省传统项目（[[田径|田径]]）学校，1982年至1989年八年蝉联福州市中学生田径运动会团体总分冠军<ref name=":1" />{{Rp|145}}。

该校设定校风为“严、勤、实、恒”，校训为“求知、求真、求实”，办学宗旨为“依法治校、以德立校、科研兴校、文化强校”<ref name=":3">{{cite web|url=http://www.fjcl.gov.cn/html/eduhighmsg/20130917/1658812.html|title=福州第二中学学校信息|work=长乐市人民政府|date=2013-09-17|accessdate=2017-07-10|archivedate=2017-07-10|archiveurl=https://web.archive.org/web/20170710161907/http://www.fjcl.gov.cn/html/eduhighmsg/20130917/1658812.html}}</ref>。
== 校外交流 ==
2012年暑期，福州二中邀请[[臺灣|台湾]]台东中学学生篮球队赴该校友谊比赛。2013年暑期两岸青年夏令营中，[[嘉義縣|嘉义]][[嘉義縣立永慶高級中學|永庆高中]]师生在二中联欢，并与此校人员“结对子”。<ref name=":0" />{{Rp|46}}

== 部分师生 ==
1936年，[[蒲风|蒲风]]在福建学院附中担任[[语文|国文]]教员。蒲氏是[[诗人|诗人]]，[[新四军|新四军]]的一员。<ref name=":0" />{{Rp|45}}次年1月，蒲氏将教职让给挚友[[雷石榆|雷石榆]]。雷氏是诗人，曾参与[[臺灣文藝聯盟|台湾文艺联盟]]，此时也在附中担任国文教员。[[林子白|林子白]]曾在1930年代担任此校美术教员。林氏是[[畫家|画家]]，美术教育家。<ref name=":0" />{{Rp|46}}

至于曾在第二中学就读的校友，在政治界的有前福建省省长[[黄小晶|黄小晶]]、省人大副主任林强。在科学界，有[[中国工程院|中国工程院]]院士张文海，他是[[冶金|冶金专家]]。在教育领域，郑捷曾毕业于该校，他是全国十佳教师，也是全国人大代表。曾乃梁亦然，他是[[武术|武术]]教练。<ref name=":4">{{Cite web|url=http://www.fzez.cn/erzhong/?p=2080|title=福州二中简介|accessdate=2017-07-10|author=|date=2013-04-27|publisher=福州市第二中学|archiveurl=https://web.archive.org/web/20170710154249/http://www.fzez.cn/erzhong/?p=2080|archivedate=2017-07-10}}</ref>
