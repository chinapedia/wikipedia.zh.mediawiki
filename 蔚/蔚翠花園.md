'''蔚翠花園'''，是[[香港|香港]][[北區_(香港)|北區]]的一系列大樓型住宅，由[[新鴻基地產|新鴻基地產]]所建，位於[[上水|上水]][[吉祥街|吉祥街]]9號。

== 簡介 ==
蔚翠花園位於上水吉祥街9號，在上水及粉嶺邊沿地段，毗鄰欣翠花園及維也納花園。居民要轉乘村巴往火車站，而步行前往則要15分鐘。蔚翠花園所處為純住宅地段，四周種值了不少植物。居民日常購物可乘車往上水火車站及新都廣場一帶。

1999年及2000年，蔚翠花園分別接受香港電台電視部及無線新聞部採訪，並製作新聞專題，如新聞透視，講述有關平台漏水的情況。

蔚翠花園呎價基於2007-2011年本身大型改善計劃、平台漏水工程,、2010-2012年路政處隔音屏工程、2012年路政處行人路改善工程等而上升。這幾項工程令屋苑居住環境提升，{{Fact|平均呎價與同區頂級住宅差不多|time=2018-05-15}}。

== 會所 ==
會所設施包括：有游泳池、桑拿室、健身室、兒童遊樂場、空調閱讀室及自修室；並會定期舉辦活動，如水上嘉年華。

== 環保設計 ==
{{Unreferenced section|time=2016-03-04T13:31:53+00:00}}
蔚翠花園設有環保設計，並獲環保署嘉許，以下是相關資料：
*規劃對象：道路交通噪音及九廣鐵路列車噪音
*保護目標：伸延的鰭及隔音屏障
*設計資料：隔音屏障：4-10分貝（A）
*建築設計：伸延的鰭：2-4分貝（A）
 
== 2008-2013年蔚翠花園大型改善計劃 ==
{{Unreferenced section|time=2016-03-04T13:31:53+00:00}}
2008-2013年蔚翠花園大型改善計劃是一個大型工程，支出近1億，包括平台漏水工程、園景設計工程、遊樂場設計工程、更換蔚翠花園全車隊的車為歐盟五型排氣標準的村巴、會所翻新等過百個項目。

由於因平台漏水而進行工程，工程分4期進行，由2008年5月12日至2010年1月30日，但施工時間卻嚴重超出預期，現時至少要到2013年才完成，並需要封閉平台進行工程，工程費用8,600萬元。平台基本設施由新昌工程負責，園景設計由琦?園景公司負責，遊樂場設計由顯飛康樂公司負責，總工程監工和顧問由譚天瑞顧問公司，遊樂場石屎台由新昌工程負責，總電力工程由譚天瑞顧問公司和新昌工程負責。

{{Fact|工程進行期間，新昌的一名泥水主管請了三名黑工。在1月10日，警方在蔚翠花園拘捕了三名黑工和泥水主管，事件亦被報章所報道。|time=2016-03-04T13:31:53+00:00}}

工程後的特色如下：
*加設花園射燈
*加設流水瀑布
*加設平台中央廣場
*游泳池加入海底元素和圖案

== 管理公司 ==
蔚翠花園曾經將保安工作外判予[[康業管理公司|康業管理公司]]及[[威格斯管理公司|威格斯管理公司]]，但兩者均有管理不善的問題，導致居民不滿<ref>第4屆第2次管理委員會會議記錄</ref>。

=== 康業事件 ===
2001年，[[康業管理公司|康業管理公司]]因管理不善，出現了多項失誤，包括：
*[[平台|平台]]的[[保安|保安]]問題，在它執業期間，出現了多宗偷竊案。
*未能解決[[泳池|泳池]]、[[噴水池|噴水池]]的[[漏水|漏水]]問題。
*多宗投訴遭到忽視<ref>第2屆第8次管理委員會會議記錄</ref>。

=== 威格斯事件 ===
在[[威格斯管理公司|威格斯管理公司]]的管理下出現了多項重大管理失當事件：
*欠下[[陽光巴士|陽光巴士]]村巴的服務費共80萬元，須由業主立案法團出資。
*仍未能解決泳池、噴水池的漏水問題。
*出現了差別服務，如在工作時間幫關係較好的業主搬運，此舉令到其它業主覺得不公<ref>第4屆第3次管理委員會會議記錄</ref>。

=== 置邦管理公司 ===
2006年，[[置邦管理公司|置邦管理公司]]接手管理蔚翠花園，不能解決多項問題：
*不能解決[[泳池|泳池]]、[[噴水池|噴水池]]的[[漏水|漏水]]問題。
*服務差<ref>第6屆第2次管理委員會會議記錄</ref>。

===穿梭巴士===
NR14蔚翠花園和中環的巴士服務（免費但不需持住戶証上車），引來欣翠花園居民乘搭，引來不滿。
*可乘坐屋苑穿梭巴士5分鐘前往[[上水站|上水站]]，約8分鐘一班，由陽光巴士及蔚翠花園業主立案法團負責營運。
*為蔚翠花園居民提供來往蔚翠花園和[[上水站|上水鐵路站]]
*{{Fact|蔚翠花園在2010年1月增加了一輛歐盟五型排氣標準的27座位村巴，這部車是全港首批歐盟五型排氣標準的車，可說是為環保盡一分力。|time=2016-03-04T13:31:53+00:00}}
*{{Fact|蔚翠花園已於2010年2月完成更換蔚翠花園全車隊的車為歐盟五型排氣標準的村巴，約支出250萬。|time=2016-03-04T13:31:53+00:00}}
*{{Fact|更換的歐盟五型排氣標準村巴有約50%出現故嫜,有村巴甚至在路中死火。|time=2016-03-04T13:31:53+00:00}}


NR16蔚翠花園和中環的巴士服務（收費18.5）
*NR16（蔚翠花園至上環）穿梭巴士基於星期日及公眾假期的班次使用率偏低（只得上午7時25分一個班次，使用人數約3.39人次），有需要停止有關服務以減低其營運成本。法團管委會於第六屆管理委員會第十九次會議討論後同意有關安排。有關停止服務的安排將於2010年3月7日開始，而NR16於星期一至六的所有班次則不受影響。
*蔚翠花園已於2010年2月完成更換蔚翠花園NR16車隊的車為歐盟五型排氣標準的村巴
*NR16於星期六的班次每班60座位的巴士亦只錄得平均約4.5人次,蔚翠花園正考減少在星期六的班次
*{{Fact|茲因近日鄰近屋苑取消穿梭巴士服務，接獲業戶反映有其他屋苑居民使用本苑穿梭巴士服務。為保障蔚翠花園業戶利益，蔚翠花園客户服務中心懇請各業戶及住戶使用穿梭巴士服務時出示有效住戶證或乘車證，訪客應出示訪客乘車證以茲識別。蔚翠花園客户服務中心亦會安排職員不定時作出抽查，如發現有非本苑業戶使用穿梭巴士服務，蔚翠花園客户服務中心職員有權拒絕未持有乘車証之人士登車及報警處理。|time=2016-03-04T13:31:53+00:00}}
