'''狹間站'''（{{jpn|j=狭間駅|hg=はざまえき|rm=Hazama eki|f=y}}）是一個位於[[東京都|東京都]][[八王子市|八王子市]]{{link-ja|東淺川町|東浅川町}}，屬於[[京王電鐵|京王電鐵]][[高尾線|高尾線]]的[[鐵路車站|鐵路車站]]。[[車站編號|車站編號]]是'''KO51'''。

== 年表 ==
* 1967年10月1日，作為京王帝都電鐵的車站啟用。
* 2013年2月22日，準特急開始停靠此站，同日以KO51作為車站編號。

== 車站構造 ==
本站是[[相對式月台|相對式月台]]2面2線[[地面車站|地面車站]]。站房位於下行月台側，各月台間有[[高架橋|高架橋]]連通，高架上設有電梯。

廁所位於下行月台側的付費區內。

往[[高尾站_(東京都)|高尾]]方向有條分岐的維護用線路。

== 站名由來 ==
「'''狹間'''」原是初澤山麓的聚落名，之後成為周邊地區的町名。本站站名取自於此。
