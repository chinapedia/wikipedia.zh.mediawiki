'''單一制企業'''（{{lang-ru|Унитарное предприятие}}，{{lang-en|Unitary enterprise}}），是[[獨立國家聯合體|獨立國協]]非公司[[法人|法人]]的[[獨資企業|獨資企業]]，即不能採持股比例分割、也不能分配於企業勞工中而具單一性，未經所有權者同意，經營或業務管理權者不得處分資產。通常法人不對所有權者債務負責，所有權者也不承擔法人債務，除非企業財產無法清償債務。<ref>{{cite web|coauthors=|author=崔熳|date=|title=俄罗斯的民事主体制度|url=http://www.uibe.edu.cn/upload/up_wyxy/lunwen-cuiman.htm|archiveurl=https://web.archive.org/web/20040423053953/http://www.uibe.edu.cn/upload/up_wyxy/lunwen-cuiman.htm|website=[[对外经济贸易大学|对外经济贸易大学]]|archivedate=2004-04-23|accessdate=|deadurl=yes}}</ref><ref>{{cite journal|coauthors=|author=张建文|date=2008-09|year=|title=俄罗斯单一制企业研究|journal=经济法论丛|volume=15|issue=|isbn=|pages=356-379|id=|url=http://qiduojun.csu.edu.cn/jingjifaluncong/335.html|archiveurl=https://web.archive.org/web/20170202122948/http://qiduojun.csu.edu.cn/jingjifaluncong/335.html|publisher=中国方正出版社|archivedate=2017-02-02|accessdate=|deadurl=yes}}</ref><ref>{{cite journal |coauthors= |author=张力 |date=2010 |year= |title=俄罗斯民法中单一制企业的主体地位及其过渡性——转型社会中“商业公营造物”的实证分析 |journal=法律科学 |volume= |issue=5  |isbn= |pages= 164 |id= |url= |publisher=[[西北政法大学|西北政法大学]]学报 |archivedate=|accessdate = |deadurl=no}}</ref><ref>{{cite book |author=张力 |title=俄罗斯非公司制国家企业法研究——历史、现实与镜鉴 |url=http://www.megbook.com.tw/mall/detail.jsp?proID=2856604 |format= |accessdate= |edition= |series= |date= 2016-07-01  |publisher=[[华中科技大学|华中科技大学]]出版社  |language= |isbn=978-7-5680-1565-3  |pages= |script-chapter= |chapterurl= |unified= |quote= }}</ref>

== 由來 ==
[[苏联解体|苏联解体]]後[[獨立國家聯合體|獨立國協]]採[[市场经济|自由經濟]]，在[[苏联|前蘇聯]][[計劃經濟|計劃經濟]]的國有廠企機構[[私有化|私有化]]改革中，所出現的商業組織形式，發軔於1994年度聯邦法律第51號《[[s:ru:Гражданский_кодекс_РФ|俄羅斯聯邦民法典]]》（{{lang|ru|№ 51-ФЗ «Гражданский кодекс Российской Федерации»}}，{{lang|en|No. 51-FZ "Civil Code of the Russian Federation"}}）後其他成員紛紛跟進。儘管目標[[私有化|私有化]]<ref>{{cite web|url=http://by.mofcom.gov.cn/article/jmxw/201401/20140100465298.shtml |title=2011-2013年白改造国家单一制企业为开放式股份公司的计划如期完成 |website=[[中华人民共和国商务部|中华人民共和国商务部]] |author=驻白俄罗斯经商参处 |language= |date=2014-01-20 |accessdate=}}</ref><ref>{{cite web |url=http://report.nat.gov.tw/ReportFront/report_detail.jspx?sysId=C10303263 |title=「第8屆首爾國際競爭論壇」及「第18屆競爭政策國際研討會」 |website=公務出國報告資訊網 |author= |language= |date=2014-09-30 |accessdate= |archive-url=https://web.archive.org/web/20170202023442/http://report.nat.gov.tw/ReportFront/report_detail.jspx?sysId=C10303263 |archive-date=2017-02-02 |dead-url=yes }}</ref>，但[[國防工業|國防工業]]、背負政策任務及壟斷性質等公有事業仍維持單一制企業運作。

== 俄羅斯 ==
根據2002年度聯邦法律第161號《關於國家和市有單一制企業》（{{lang|ru|№ 161-ФЗ «О государственных и муниципальных унитарных предприятиях»}}，{{lang|en|No. 161-FZ "On state and municipal unitary enterprises"}}）規定公有單一制企業如下：
#聯邦國有單一制企業（{{lang|ru|Федеральное государственное унитарное предприятие - ФГУП, FGUP}}，{{lang|en|Federal state unitary enterprise - FSUE}}）：聯邦所屬企業。
#國有單一制企業（{{lang|ru|Государственное унитарное предприятие - ГУП, GUP}}，{{lang|en|State unitary enterprise - SUE}}）：[[俄罗斯联邦主体|聯邦主體]]所屬企業。
#市有單一制企業（{{lang|ru|Муниципальное унитарное предприятие - МУП, MUP}}，{{lang|en|Municipal unitary enterprise - MUE}}）：自治城市所屬企業。

== 其他國家 ==
大部分[[獨立國家聯合體|獨立國協]]國家，單一制企業除國（市）營事業外，還衍生非公司私營企業或自營作業者等的「私有單一制企業」（{{Slavonic|Частное унитарное предприятие - ЧУП}}<ref>[http://china.mfa.gov.by/zh/investment1/economics/corporate/ 白俄罗斯企业实体 - 白俄罗斯共和国驻华大使馆]</ref>，{{lang|en|Private unitary enterprise - PUE}}），外商也可利用這種方式保持資產所有權<ref>[http://www.belarus.by/cn/business/companies 在白俄罗斯创办公司 - Belarus.by]</ref>（相當於[[外商独资企业|外商独资企业]]）。

== 表記方式 ==
[[獨立國家聯合體|獨立國協]]單一制企業因各國規定不同，企業特取名稱也有所差異，例如：

;[[俄罗斯|俄羅斯]]
:依[[俄罗斯联邦主体|聯邦主體]]管轄分別冠上「共和國」（{{lang|ru|республиканское-, РГУП}}）、「邊疆區」（{{lang|ru|краевое-, КГУП}}）、「州（區）」（{{lang|ru|областное-, ОГУП}}）等。

;[[白俄罗斯|白俄羅斯]]
:私有單一制企業係由個別投資，若其另有企業體成立，不使用「私有」字眼而僅表示單一制企業（{{lang|by|УП}}）。公有單一制企業則略去「國有」（{{lang|by|государственное}}）直接用「共和國有」（{{lang|be|республиканское-, РУП}}）、「市有」（{{lang|be|коммунальное-, КУП}}）<ref>{{cite web|url=http://pravnik.by/chup-v-minske-belarus.html |script-title=be:Унитарное предприятие, ЧУП: информация для учредителя в Беларуси |website= {{lang|be|Адвокат в Минске}} |author={{lang|be|Владимир Пунько}} |language=be |date=2014-08-09 |accessdate=}}</ref>。

;[[乌克兰|烏克蘭]]
:因《烏克蘭商法典》（{{lang|ua|Господарський кодекс України}}，{{lang|en|Commercial Code of Ukraine}}）已規定單一制企業適用範圍，並不特取該名稱。像是「國有企業『烏克蘭港務局』」（{{lang|ua|Державне підприємство «Адміністрація морських портів України»}}）<ref>USPA: [http://www.uspa.gov.ua/en/about-us {{lang|en|About Us}}] {{en}} {{uk}} {{ru}}</ref>。
