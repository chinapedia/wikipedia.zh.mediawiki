'''單線鐵路'''是一種鐵路線的形式，使兩個方向的列車只可使用同一條路軌前進。單線鐵路多出現於一些較少客量的鐵路線上，也可能會在一些鐵路[[支線|支線]]中出現，甚或會在列車班次較少的鐵路線或沒有太多資金或缺乏空間去建設[[雙線鐵路|雙線鐵路]]上出現。

== 優點 ==
*相比[[雙線鐵路|雙線鐵路]]及[[四線鐵路|四線鐵路]]，單線鐵路建築成本較低。
*相比雙線鐵路及四線鐵路，可以較方便地在複雜地形上建設。

== 缺點 ==
*在運作一條鐵路線上，當一段單線路段中，列車需時15分鐘通過，則該線每小時每方向只有2列列車通過。但雙線鐵路則可以每小時每方向有高達15列列車通過（當列車每四分鐘經過一個[[鐵路信號機|鐵路信號機]]），因此有些重要幹線的單線鐵路為增加容量，會在全部路段加鋪鐵軌成為雙線鐵路，例如日本的[[奧羽本線|奧羽本線]]、台灣的[[海岸線_(臺鐵)|海線]]及[[台東線|台東線]]等。
*當一架列車在單線路段中出意外，那段鐵路線便不能讓其他列車通行，例如當[[黃埔站_(香港)|黃埔站]]出現故障時，[[觀塘綫|觀塘綫]]下行列車只能以[[何文田站|何文田站]]作為臨時的終點站。但雙線鐵路則可以以單線雙程方式，有限度地繼續讓列車通行。

==产生的原因==
*铁路发展期
*铁路衰退期
*复线化准备
*灾害

== 運行方式 ==
{{seealso|閉塞 (鐵路)}}
當單線鐵路中，需要在同一時間上使用多於一列的列車，那必需要建設一個[[會讓站|會讓站]]（一段短的雙線鐵路），使一方向的列車進入及通過車站後，另一方向列車在另一路軌等待。在一些情況（例如英國的Abbey線、台鐵[[林口線|林口線]]）中，由於同時間只有一組列車運作，所以便不需要建設一個會讓站，不需避開另一方向列車。

在單線鐵路上的會讓站，有一些措施必需辦妥，例如[[轉轍器|轉轍器]]必需扳至正確的位置，使一方列車不會誤入另一方等待列車的軌道上，造成兩車正面相撞的意外。因此，在會讓站內或附近地帶需要設置一些鐵路信號機。在傳統英國做法（一此國家也依此做法），在每一段會讓站與會讓站之間的單線鐵路中，都有獨特的電氣路籤（牌），列車需要通過那路段時，需要向會讓站站員索取電氣路籤（牌），使列車可以得到那路段的行駛權，而其他列車因無法得到電氣路籤（牌），而防止進入那路段，造成正面相撞的意外。現時只有少許鐵路線使用那種系統，而[[香港|香港]][[港鐵|港鐵]]（當時為[[九廣鐵路|九廣鐵路]]）[[東鐵線|東鐵線]]的[[大學站_(香港)|大學站]]至[[羅湖站|羅湖站]]之間，在未鋪設雙線及電氣化前，便使用該系統<ref>[http://www.youtube.com/watch?v=w04wssAvkxI&feature=relmfu HONGKONG TRAIN ７、８０年代柴油火車 PT2]，[[YouTube|YouTube]]</ref>。在早期的北美洲，在一段有單線鐵路的鐵路線上，都會設有簡單的時間表，列出列車到達會讓站的時間，使得單線鐵路上不会有兩列列車以不同方向運行，可是這個做法會不靈活且沒有效率。所以便發明了電報系統，使列車与车站間可以進行通訊。
