'''單峰駱駝'''（[[學名|學名]]：''{{lang|la|Camelus dromedarius}}''）是一種大型的[[偶蹄目|偶蹄目]]動物，產於[[非洲|非洲]]北部、[[亞洲|亞洲]]西部與南部<ref name="阿爾金山大自然">[http://www.chinagev.org/nature/aejinshan.htm 阿爾金山大自然]{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }} - 漫漫「驼生」，作者為汪永晨（人民網亦有收錄該文，位於[http://past.people.com.cn/bike/viewnews.btml?id=387858 此處]{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}，文章名稱為「大漠苦行者——野骆驼」，作者汪永晨）</ref>，亦有部分是來自[[非洲之角|非洲之角]]、[[蘇丹共和國|蘇丹共和國]]、[[衣索比亞|衣索比亞]]和[[索馬利亞|索馬利亞]]。然而，現今的野生駱駝僅有[[澳大利亞|澳大利亞]]的[[澳洲野生駱駝|澳洲野生駱駝]]及[[南美洲|南美洲]]的[[美洲駝|美洲駝]]。<ref name="阿爾金山大自然"/>。

單峰駱駝是[[駱駝科|駱駝科]]中最為人所知的成員，而駱駝科中的其他成員有南美洲的[[骆马属|骆马属]]和[[羊驼属|羊驼属]]的几种[[美洲駝|美洲駝]]。單峰駱駝的背上僅有一個駱峰，跟背上有兩個駱峰的[[雙峰駱駝|雙峰駱駝]]有著明顯的不同。有時單峰駱駝亦會被稱為“阿拉伯駱駝”。一些人仍舊認為單峰駱駝的英語「dromedary」只適用於賽跑的駱駝（此名來自希臘語，意為「奔跑」）。在[[奧克蘭動物園|奧克蘭動物園]]（Oakland Zoo）的網站上有這麼一句話：''「『dromedary』只適用於[[阿拉伯|阿拉伯]]的賽跑駱駝，即軍用駱駝隊中的駱駝。<ref>英語為「''The name "Dromedary" is properly reserved for the [[阿拉伯|Arabian]] racing camel such as those used in the various military camel corps.''」</ref><ref name=oaklandzoo>{{cite web | title = 動物 A-Z 阿拉伯駱駝 | work = 奧克蘭動物園 | url = http://www.oaklandzoo.org/atoz/azarabcamel.html | accessdate = 2006年3月1日 | deadurl = yes | archiveurl = https://web.archive.org/web/20070416081648/http://www.oaklandzoo.org/atoz/azarabcamel.html | archivedate = 2007年4月16日 }}</ref>」''

== 馴養 ==
單峰駱駝本是來自[[亞洲|亞洲]]西部及[[東非|東非]]，在數千年前已開始在[[阿拉伯|阿拉伯]]中部或南部被馴養。專家表示，一些人認為單峰駱駝早在公元前4000年已被馴養，而其他大部分人則認為是公元前1400年。現時所知全球約有一千三百萬頭單峰駱駝已被馴養，大多是在[[西印度|西印度]]到[[巴基斯坦|巴基斯坦]]，再到[[伊朗|伊朗]]至北非一帶。一些單峰駱駝後來傳入澳大利亞，在澳大利亞對駱駝的需求減少後，這些駱駝被放入野外野化，現在單是在[[澳洲野生駱駝|澳大利亞的單峰駱駝]]數量最少也有500,000頭。<ref name=agric>{{cite web|title=Farmnote 122/2000：野生駱駝（澳大利亞西部） |url=http://www.agric.wa.gov.au/pls/portal30/docs/FOLDER/IKMP/PW/VP/FER/F12200.PDF |accessdate=2005年12月5日 |deadurl=yes |archiveurl=https://web.archive.org/web/20050319035659/http://www.agric.wa.gov.au/pls/portal30/docs/FOLDER/IKMP/PW/VP/FER/F12200.PDF |archivedate=2005年3月19日 }}</ref>約於[[前2千年|前2千年]]，單峰駱駝傳入[[埃及|埃及]]和[[北非|北非]]。

[[File:Menare.jpg|thumb]]的駱駝商隊]]
據估計，[[雙峰駱駝|雙峰駱駝]]約於公元前2500年在亞洲馴養。雙峰駱駝比單峰駱駝更健壯更結實，也更能吃苦耐勞，從[[伊朗|伊朗]]到[[西藏|西藏]]的地區都能存活下來。<ref name=abc>{{cite web | title = 生物的特色 - 駱駝 | url = http://www.abc.net.au/creaturefeatures/facts/camels.htm | accessdate = 2005年12月5日 | archive-url = https://web.archive.org/web/20060112023308/http://www.abc.net.au/creaturefeatures/facts/camels.htm | archive-date = 2006-01-12 | dead-url = yes }}</ref>雖是這樣，單峰駱駝比雙峰駱駝要更高，也跑得更快：若是有人駕馭的話，可一直維持著時速13至14公里的速度持續18小時，而雙峰駱駝在長時間揹著貨物的情況下行進的速度只為時速4公里，每日可行50公里。

但牠们不耐寒，所以在土耳其以雙峰駱駝为主。

== 歷史 ==
約於[[前2千年|前2千年]]，單峰駱駝逐漸在[[撒哈拉沙漠|撒哈拉沙漠]]地區居住，但是在前900年左右又再次消失於撒哈拉沙漠。牠們大多是被人類捕獵的。後來[[埃及|埃及]]入侵[[波斯|波斯]]時，[[岡比西斯二世|岡比西斯二世]]把已經被馴養的單峰駱駝傳入波斯地區。被馴養的單峰駱駝在北非被廣泛使用，而直到後來，[[羅馬帝國|羅馬帝國]]仍然使用駱駝隊帶著戰士到沙漠邊緣巡邏。可是波斯駱駝並不適合用來穿越[[撒哈拉沙漠|撒哈拉沙漠]]；這種穿越大沙漠的長途旅行通常是靠戰車達成的。

在第4世紀，更強壯和耐久力更強的[[雙峰駱駝|雙峰駱駝]]首度傳入非洲。牠們傳入非洲後，開始有愈來愈多的人使用牠們，因為這種駱駝較適合作穿越大沙漠的長途旅行之用，且可以裝運更多更重的貨物。這時，[[跨撒哈拉貿易|跨撒哈拉貿易]]終於得以進行。

== 解剖學與繁殖 ==
[[File:Camel_Skeleton_-_Richard_Owen_-_On_the_Anatomy_of_Vertebrates_(1866).jpg|thumb]]的[[骨骼|骨骼]]結構]]
雄性單峰駱駝的[[上顎|上顎]]較軟，使牠們可以生出一個粉紅色的袋子，這個袋子在[[阿拉伯文|阿拉伯文]]中稱為「''doula''」。在交配季節期間，這個袋子會吊在雄性單峰駱駝嘴的兩旁，以吸引異性。單峰駱駝的[[睫毛|睫毛]]很濃密，耳朵小而多毛。

單峰駱駝的妊娠期長約12個月。通常牠們一次只生一隻小駱駝，成年駱駝會一直親自照顧小駱駝，直到小駱駝18個月為止。雌性單峰駱駝在3到4歲後就性成熟，而雄性單峰駱駝在5到6歲後才性成熟。牠們的壽命一般為25年，而最長可達50年。

成年的單峰駱駝可長達10尺，高達6到7尺。牠們一般重1000至1500磅。

== 價值 ==
單峰駱駝的壞脾氣和倔強是出了名的，每一隻駱駝都會用跺腳和奔跑來表示不滿，但如果馴養得宜，牠們也是很親切、有耐性和聰明的。現在被馴養了的單峰駱駝可為人類提供[[駱駝奶|駱駝奶]]和肉，也可用來裝載貨物或乘客。不像馬那樣，牠們要裝載時得跪下，讓貨物或乘客送上牠們的背上。在[[埃及|埃及]]，人們可見很多[[警察|警察]]騎著駱駝四處巡邏。
