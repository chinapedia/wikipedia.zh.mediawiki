'''單飛雪'''，[[台灣|台灣]][[愛情小說|言情小說]][[作家|作家]]。[[臺北市私立華岡藝術學校|華岡藝術學校]]畢業生。1998年11月首次使用本筆名於台灣[[林白出版社|林白出版社]]（更名為狗屋/果樹出版社）出版第一部作品。

2002年9月起，[[香港|香港]][[天地圖書|天地圖書]]旗下的小小書坊及星河出版社開始持續出版單飛雪的[[香港|香港]]版書籍。截至2008年6月共有35部作品除台灣版外另有發行[[香港|香港]]版。

2008年，單飛雪的作品《欲罷不能》及《噓！別說我愛老大》由[[泰國|泰國]]Jamsai出版社[https://web.archive.org/web/20070401203528/http://www.jamsai.com/Default.aspx แจ่มใส]<ref>{{cite web |url= http://www.jamsai.com/popup/promote/more_than_love/advise/index.asp |title= สำนักพิมพ์แจ่มใส |accessdate= 2008年3月10日 |language= th |deadurl= yes |archiveurl= https://web.archive.org/web/20080326172423/http://www.jamsai.com/popup/promote/more_than_love/advise/index.asp |archivedate= 2008年3月26日 }}</ref>翻譯成[[泰文|泰文]]於當地發行。此為台灣本土言情小說獲翻譯為外文的先例之一。

2014年於皇冠60週年特刊<ref> 皇冠60週年特刊（720期2014/02/248頁-253頁）</ref> 首次邀請單飛雪作家撰寫散文《感謝的力量》，並於文後介紹作家本人為張阿捨<ref>{{Cite web |url = https://www.crown.com.tw/MagIndex.aspx?pid=720 |title = 皇冠60週年特刊 |author = 張阿捨 |date =2014年2月 |publisher = 鑫濤出版事業有限公司 |language = zh-hant |accessdate = 2014年2月24日 |quote = 感謝的力量 }}</ref>，介紹內容為：「張阿捨，曾以『單飛雪』為名，撰寫愛情故事。閒以『張阿捨』為名，撰繪家常事。愛世間無心機之可人兒。勵志吃爽，愛透，活夠帥。一生憑自由意志，江湖打工。家財萬貫或身無分文，皆不能奪其自樂之志。常感生如夢，渾沌迷離，變幻莫測。但求活出，當下精彩」。

單飛雪閒以張阿捨為名於痞客邦部落格[http://asir9.pixnet.net/blog 「吉羊居」]撰繪家常事。
