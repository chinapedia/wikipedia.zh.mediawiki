'''唐奕'''（{{bd|1993年|1月8日|catIdx=Tang Yi}}），[[上海|上海]]人，[[中国|中国]]女子[[游泳|游泳]]运动员。

== 生平 ==
2008年，唐奕成為[[2008年夏季奥林匹克运动会|北京奥运会]]中国体育代表团的成员，主攻自由泳。

2010年，唐奕代表[[2010年亚洲运动会中国代表团|中国]]參加[[2010年亞洲運動會|廣州亞運會]][[2010年亞洲運動會游泳比賽|游泳比賽]]的女子50米自由泳、100米自由泳、200米自由泳、4x100米混合泳、4x100米自由泳接力及4x200米自由泳接力六個項目，奪得四金二銀的佳績。

2012年，唐奕代表[[2012年夏季奥林匹克运动会中国代表团|中国]]出戰[[英國|英國]][[倫敦|倫敦]]舉行的[[2012年夏季奥林匹克运动会|奥林匹克运动会]][[2012年夏季奧林匹克運動會游泳比賽|游泳比賽]][[2012年夏季奧林匹克運動會女子100米自由泳比賽|女子100米自由泳]]、[[2012年夏季奧林匹克運動會女子4×100米自由泳接力比賽|4×100米自由泳接力]]和[[2012年夏季奧林匹克運動會女子4×100米混合泳接力比賽|4×100米混合泳接力]]賽事，其中在100米自由泳取得銅牌。

2016年8月，唐奕代表[[2016年夏季奥林匹克运动会中国代表团|中国]]出战[[巴西|巴西]][[里约热内卢|里约热内卢]]举行的[[2016年夏季奥林匹克运动会|奥林匹克运动会]][[2016年夏季奥林匹克运动会游泳比赛|游泳比赛]][[2016年夏季奥林匹克运动会游泳比赛－女子4×100米自由泳接力|女子4×100米自由泳接力]]赛事。在女子4×100米自由泳接力预赛中，[[朱梦惠|朱梦惠]]、[[孙美辰|孙美辰]]、唐奕和[[沈铎|沈铎]]组成的中国队以3分37.25秒的成绩名列第9，未能进入决赛<ref>{{cite news|accessdate=2016-08-06|url=http://2016.163.com/16/0807/06/BTRLN8BA00050I18.html|title=唐奕发文称里约之行结束:感谢支持者 祝中国加油|publisher=网易体育}}</ref>。

== 教练 ==
* 1999年 上海市长宁区体校 教练何纲
* 2004年 上海体育运动技术学院 教练谢军
* 2005年 上海游泳队 教练钱敏
* 2006年 国家游泳队 教练潘佳章

== 主要成绩 ==
* 2006年 世界青年游泳锦标赛 200米自由泳 第二名
* 2006年 [[2006年亚洲运动会|亚洲运动会]] 4×200米自由泳接力 第一名
* 2006年 全国冬季游泳锦标赛 200米自由泳 第一名
* 2007年 悉尼青年节游泳赛 200米自由泳 第一名
* 2007年 日本国际邀请赛 4×100米自由泳接力 第二名
* 2008年 全国游泳锦标赛暨奥运会选拔赛 100米自由泳 第四名
* 2010年 [[2010年夏季青年奧林匹克運動會|2010年夏季青年奧林匹克運動會]] 4×100米自由泳接力 第一名
* 2010年 2010年夏季青年奧林匹克運動會 100米自由泳 第一名
