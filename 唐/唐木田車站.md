'''唐木田站'''（{{jpn|j=唐木田駅|hg=からきだえき|rm=Karakida eki|f=y}}）是一個位於[[東京都|東京都]][[多摩市|多摩市]]{{link-ja|唐木田|唐木田}}一丁目，屬於[[小田急電鐵|小田急電鐵]][[多摩線|多摩線]]的[[鐵路車站|鐵路車站]]。是多摩線的[[終點站|終點站]]。[[車站編號|車站編號]]是'''OT 07'''。

== 歷史 ==
本站位於多摩線小田急多摩中心站往相模原方向的延伸路線上，在地方居民的請求下設站。小田急電鐵也為了確保車庫用地而同意設站。
* 1990年（[[平成|平成]]2年）3月27日 - 伴隨多摩線延伸而設站，為小田急電鐵第69座車站。為各站停車停靠站。
* 2000年（平成12年）12月2日 - [[浪漫特快|浪漫特快]]「Homeway」與急行開始在多摩線運行，本站為停靠站。
* 2002年（平成14年）3月23日 - 新增多摩急行班次，為其停靠站。
* 2004年（平成16年）12月11日 - 新增區間準急班次，為其停靠站。
* 2008年（平成20年）3月20日 - 浪漫特快「Metro Homeway」開始運行，為其終點站。
* 2016年（平成28年）3月26日 - 廢止停靠本站浪漫特快「Homeway」、「Metro Homeway」與區間準急。

== 站名由來 ==
取自車站所在地西南方的地名。月台大部分屬於[[多摩市|多摩市]]{{link-ja|中澤 (多摩市)|中沢 (多摩市)|中澤}}地區，站務室與站房商店用地則屬於{{link-ja|唐木田|唐木田}}地區。

== 車站構造 ==
本站是[[側式月台|側式月台]]1面1線（1號月台）與[[島式月台|島式月台]]1面2線（2、3號月台），共計2面3線的[[地面車站|地面車站]]，為[[跨站式站房|跨站式站房]]。月台南端有樓梯通往剪票口。因為此地有喜多見檢車區唐木田出張所，因此有許多回送列車。喜多見檢車區唐木田出張所可不掉頭直接至2、3號月台。

夜晚站內會停放三列6輛編組列車、四列8輛編組列車、九列10輛編組列車。

== 相模原延伸 ==
未來在[[駐日美軍|駐日美軍]]返還部分[[相模補給總庫|相模補給總庫]]用地後，小田急計畫將本線往[[町田市|町田市]]小山田、[[神奈川縣|神奈川縣]][[相模原市|相模原市]]方向延伸。2014年5月26日，在與[[町田市|町田市]]及[[相模原市|相模原市]]的會談中決議推動多摩線延伸，並目標在[[中央新幹線|中央新幹線]]預定通車的2027年以前完成<ref name="response20140527">[http://response.jp/article/2014/05/27/224080.html 町田・相模原両市長、小田急多摩線延伸で覚書締結] - レスポンス、2014年5月27日</ref>。
