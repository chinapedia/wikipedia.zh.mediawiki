'''唐隆之变'''，或稱'''唐隆政變'''，是710年7月21日（[[唐中宗|唐中宗]][[景龙|景龙]]四年，[[唐少帝|唐少帝]][[唐隆|唐隆]]元年），由当时皇叔相王[[李旦|李旦]]第三子临淄王[[李隆基|李隆基]]和[[太平公主|太平公主]]于唐朝首都[[长安城|长安城]]共同发起的一场[[宫廷政变|宫廷政变]]。李隆基以[[禁軍|禁軍]]杀了[[韋后|韋后]]、[[安乐公主|安乐公主]]，并彻底剿灭韦氏集团。这次政变的后果是：即位不足一个月的[[李重茂|李重茂]]退位，李旦[[復辟|復辟]]为[[唐睿宗|唐睿宗]]，李隆基被立为[[皇太子|皇太子]]，掌握实权，太平公主的权势更加强大。

== 背景 ==
{{main|神龍革命|重俊之變}}
[[唐中宗|唐中宗]]以[[神龍革命|神龍革命]][[復辟|復辟]]后，对与其共患难的妻子[[韦后|韦后]]十分纵容，导致朝政大权被韦后外戚掌握，[[武则天|武则天]]侄[[武三思|武三思]]在韦后和[[昭容|昭容]][[上官婉儿|上官婉儿]]的帮助下，位列[[宰相|宰相]]，在朝堂中形成一个以韦氏为首的'''武、韦专政集团'''。他们大肆打压[[张柬之|张柬之]]、[[崔玄暐|崔玄暐]]、[[敬晖|敬晖]]等[[神龍革命|神龍革命]]的功臣势力，激化了其他李唐皇族之间的矛盾。

首当其冲的就是[[太子|太子]][[李重俊|李重俊]]，他并非韦后亲生，[[安乐公主|安乐公主]]与其夫[[武崇训|武崇训]]（武三思子）经常侮辱他并想夺了他的太子之位。707年7月，李重俊发动[[重俊之變|重俊之變]]，杀死[[武三思|武三思]]父子，但因皇帝對起事的[[禁军|禁军]]喊話，最後士兵們倒戈，重俊兵败逃亡，途中被左右所杀。

武三思虽死但韦后的势力更加强大，而当时天下并不安定，多有水旱之灾，而韦后等人只热衷于打压反对势力，培植亲信，朝政混乱。此时的朝堂，韦后及其集团虽然大权在握，但对其痛恨者甚多。

== 经过 ==
710年6月，中宗暴死，传聞韦后和安乐公主下毒暗殺中宗，朝野上下人心惶惶。韦后欲临朝[[摄政|摄政]]，扶中宗庶子李重茂登基，[[改元|改元]][[唐隆|唐隆]]，并将领南北衙军队交与韦家子弟统领，其独揽大权之心昭然若揭，原親近韋氏的[[兵部侍郎|兵部侍郎]][[崔日用|崔日用]]暗向李隆基告密，勸其立即發動兵變。於是，李隆基與太平公主的兒子[[薛崇簡|薛崇簡]]、苑總監[[鍾紹京|鍾紹京]]等，密謀策劃，欲先發制人。

有人建議，把發動政變的事先向李隆基之父相王李旦報告，李隆基胸有成竹地說：「我為了拯救[[社稷|社稷]]的危難，為君主、父親救急，成功了福祉歸於[[朝廷|宗社]]，失敗了我因忠孝而死，不連累相王。怎可以報告，讓相王擔心呢！現在報告，相王若贊成，就是害他參與了危險的起事；若他不贊成，我計謀就失敗了。」<ref>《旧唐书》·玄宗纪「我拯社稷之危，赴君父之急，事成福歸於宗社，不成身死於忠孝，安可先請，憂怖大王乎！若請而從，是王與危事；請而不從，則吾計失矣。」</ref>於是，決定背著相王，立即行動。

[[唐隆|唐隆]]元年（710年）六月庚子日申時，李隆基等人穿便服，進入禁苑，到苑總監[[鍾紹京|鍾紹京]]住處。這時，鍾紹京反悔，拒絕參加這次政變。但其妻許氏卻堅定地說：「忘身殉國，神必助之。既然參與同謀，即使不參加實質的兵變，勢難免罪。」鍾紹京於是前往拜謁李隆基。入夜後，萬騎果毅[[李仙鳧|李仙鳧]]、[[葛福順|葛福順]]、[[陈玄礼|陈玄礼]]等軍官都先後來到，請李隆基發佈命令，終於定于七月廿一夜发动政变。

當夜，[[葛福顺|葛福顺]]突袭羽林营，杀分领左右[[羽林军|羽林军]]的韦后族弟卫尉卿[[韦璿|韦璿]]、韦后堂侄长安令[[韦播|韦播]]、韦后堂甥郎将[[高嵩|高嵩]]，策反了羽林军，并攻入[[玄德门|玄德门]]，[[李仙凫|李仙凫]]亦引兵攻入[[白兽门|白兽门]]，于[[三更|三更]]会师于[[凌烟阁|凌烟阁]]。李隆基引兵入称：“韦氏毒死先帝，陰谋危害社稷，今晚應該要一起诛殺诸韦”，守卫内宫的[[武士|武士]]纷纷倒戈响应，韦后逃入飞骑营反被[[斬首|斬首]]，[[宗楚客|宗楚客]]、[[安乐公主|安乐公主]]、[[武延秀|武延秀]]、[[上官婉儿|上官婉儿]]、[[韦温_(宰相)|韦温]]、内将军贺娄氏（女）、[[赵履温|赵履温]]等人也陆续被殺，并于全城搜捕韦氏集团人员，声言凡身高高於马鞭的男性皆處死，但崔日用率兵杀诸韦时却“襁褓儿无免者”（然而韦氏家族仍然有后人，并未灭绝）。韦后党羽[[宰相|宰相]][[尚书令|尚书令]][[韦巨源|韦巨源]]认为身为大臣必须赴难，出行到都街，被乱兵杀死。韦后面首马秦客、杨均、叶静能等被枭首。韦后被暴尸于市。

== 后续 ==
政变集团又分别在[[华州|华州]]、[[怀州|怀州]]捕杀在外的宰相[[纪处讷|纪处讷]]、[[张嘉福|张嘉福]]。710年7月24日，太平公主逼[[李重茂|李重茂]]退位，李旦登上帝位，是为[[唐睿宗|唐睿宗]]，改元[[景云|景云]]。李旦嫡长子[[李成器|李成器]]鉴于“国治以序，国乱以功”，主动提出将太子之位让给李隆基。三天后，李隆基被立为[[太子|太子]]。此次政变虽结束了[[韦后|韦皇后]]集团控制朝堂的状况，但代之以太子李隆基与太平公主的对抗，而睿宗如同高宗及中宗一样，无法处理后妃干政的局面。

712年7月睿宗[[禪讓|禪讓]]，號稱[[上皇|上皇]]。李隆基即位，是为唐玄宗。但太上皇仍然掌握三品以上人事权，且七名宰相五个出自太平公主门下。

713年7月[[先天之變|先天之變]]，玄宗誅滅了太平公主，结束了自中宗朝以来，朝局多年混乱，以及[[武则天|武则天]]以来后妃专权干政的局面。
