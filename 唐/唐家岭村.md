[[北京市|北京市]][[海淀区|海淀区]]唐家岭村，位于北京市区西北五环外的[[西北旺镇|西北旺镇]]
== 地理位置 ==

唐家岭村东邻[[昌平区|昌平区]]的[[回龙观|回龙观]]，西邻[[土井村|土井村]]，南邻[[后厂村|后厂村]]、[[杨庄子|杨庄子]]、[[东北旺|东北旺]]，北邻[[辛店村|辛店村]]和昌平区的[[二拨子|二拨子]]村，包含靠近[[航天城|航天城]]的[[邓庄子|邓庄子]]，属于比较典型的城乡结合部。
近年来南面的东北旺村土地被[[中关村软件园|中关村软件园]]征用，造成该村直接面对城市规划的局面。
== 人文历史 ==

据传说，该村现名源于原称[[挡儿岭|挡儿岭]]。民间附会为[[宋|宋]]代[[佘太君|佘太君]]在[[望儿山|望儿山]]向北望儿，因该处遮挡儿子身影，因而称此处为“挡儿岭”，后转演成今名。
== 人口 ==
唐家岭村户籍人口2600人，[[北漂|外来流动人口]]数万人。
近年来，随着上地附近的[[树村|树村]]、[[马连洼|马连洼]]、[[东北旺|东北旺]]等[[城中村|城中村]]的改造和拆迁，唐家岭村成了[[中关村软件园|中关村软件园]]及[[上地信息产业基地|上地信息产业基地]]附近的最后一个城中村。
因为周边社区的租房价格较高，有许多在附近工作的人在这里租房，2006年以来，该村很多村民都在新建和扩建新房，以出租给更多的人赚取房租，但该村并不属于[[貧民窟|貧民窟]]。唐家岭村拆迁后，多数租客转而在昌平区[[史各庄街道|史各庄]]租房。<ref>{{cite news|author=叶晓彦|url=http://bjwb.bjd.com.cn/html/2014-06/04/content_185102.htm|title=小白领聚集的北四村|publisher=[[北京晚报|北京晚报]]|date=2016-06-04|accessdate=2016-11-10|deadurl=yes|archiveurl=https://web.archive.org/web/20161110174542/http://bjwb.bjd.com.cn/html/2014-06/04/content_185102.htm|archivedate=2016-11-10}}</ref>
