'''唐亮'''{{BD|1910年|6月13日|1986年|11月20日|catIdx=T}}，原名'''唐昌贤'''，别名'''唐昌明'''，[[湖南|湖南]][[浏阳|浏阳]]人，[[中国人民解放军|中国人民解放军]]开国上将。

唐亮1930年参加[[中国工农红军|中国工农红军]]，同年加入[[中国共产党|中国共产党]]，历任[[红三军团|红三军团]]第二师六团连政治委员，七团总支部书记，军团直属队政治处主任，第六团政治委员，[[红一军团|红一军团]]第二师政治部组织科科长，第二师政治部副主任、主任。参加[[长征|长征]]。[[抗日战争|抗日战争]]时期，任[[八路军|八路军]]第[[115师|115师]]政治部组织科科长，教导大队政治委员，第[[344旅|344旅]]政治部副主任，[[八路军第2纵队|八路军第2纵队]]政治部副主任，冀鲁豫军区政治部主任、山东滨海军区政治委员。[[第二次国共内战|第二次国共内战]]时期，任[[新四军|新四军]]兼山东军区政治部副主任，[[山东野战军|山东野战军]]政治部主任，华东军区政治部副主任，[[华东野战军|华东野战军]]政治部主任兼第三兵团政治委员，[[第三野战军|第三野战军]]政治部主任。曾建议将解放军中的[[士兵委员会|士兵委员会]]改为[[革命军人委员会|革命军人委员会]]。[[中华人民共和国|中华人民共和国]]成立后，任华东军区政治部副主任、主任、副政治委员，[[南京军区|南京军区]]政治委员，[[中国人民解放军军政大学|中国人民解放军军政大学]]政治委员，政治学院院长、政治委员等职。

== 生平 ==
=== 早期经历 ===
生于湖南浏阳[[永和市|永和市]][[火石岭|火石岭]]。11岁时到浏阳县城一家纸爆作坊当学徒。1926年参加浏阳县青年工人俱乐部，后又参加当地工人纠察队、农民赤卫队<ref name="欧阳青"/>。[[马日事变|马日事变]]后，返回家乡参加农民协会，曾任农民赤卫队小队长。1929年，任浏阳县[[诚嘉乡|诚嘉乡]]苏维埃政府宣传文化委员，参加过攻打永和市和浏阳城的战斗。1930年8月参加[[中国工农红军|中国工农红军]]，同年加入[[中国共产党|中国共产党]]，任[[红三军团|红三军团]][[中国工农红军第八军|八军]]第二师党委秘书。1931年起任红二师六团一连政治委员、七团党总支书记、师直属政治处主任、六团政治委员等职，参加了[[中央苏区|中央苏区]]历次反围剿战争，在战斗中多次负伤<ref name="gf">{{Cite web|url = http://cpc.people.com.cn/GB/64162/126778/127857/7538776.html|title = 唐亮|accessdate = 2015-06-26|publisher=中国共产党新闻网}}</ref>。

1934年10月，唐亮任红三军团随营学校党总支书记，随部参加[[长征|长征]]。到陕北后，先后任[[红一军团|红一军团]]第二师政治部组织科科长，政治部副主任、主任，参加[[东征战役|东征战役]]、[[西征战役|西征战役]]和[[山城堡战役|山城堡战役]]等<ref>{{cite book |title=中国二十世纪纪事本末 |volume = 附卷·人物 |editor = 周鸿、朱汉国 主编 |location = 济南 |publisher=山东人民出版社 |year=2000 |page = 621 |ISBN = 7-209-02403-4}}</ref>。

=== 抗日战争时期 ===
抗战爆发后，任八路军第[[115师|115师]]政治部组织科科长，参加了[[平型关战斗|平型关战斗]]。1937年底，任第115师教导大队政治委员。1939年起任第344旅政治部副主任、第2纵队政治部副主任，随部队开辟冀鲁豫根据地。1939年6月，任[[冀鲁豫军区|冀鲁豫军区]]政治部主任<ref>{{cite book |author=《中国人民解放军军史》编写组 |title=《中国人民解放军军史》 |volume = 第二卷 |year=2010 |publisher=军事科学出版社|page = 137 |ISBN = 978-7-80237-381-5}}</ref><ref>{{cite book |author=张明金、刘立勤 |title=中国人民解放军历史上的200个军区 |year=2010 |publisher=解放军文艺出版社 |location=北京 |page = 112 |ISBN = 978-7-5033-2252-5}}</ref>。1942年8月，唐亮调任115师教导第4旅兼湖西军分区政治委员、中共湖西地委书记，领导湖西区反扫荡战争<ref>{{cite book |author=王辅 |title=日军侵华战争 Volume3|location = 沈阳 |publisher=辽宁人民出版社 |year=2015 |page = 1454 |ISBN = 978-7-205-08145-4 }}</ref>。1943年初，因劳累过度，被批准到[[山东军区|山东军区]]医院治疗<ref name="jlz"/>。

1944年秋，唐亮接替阵亡的[[符竹庭|符竹庭]]，任八路军滨海军区政治委员兼中共滨海区委书记<ref name="jlz">{{cite book|author=《中国人民解放军高级将领传编审委员会》，中国中共党史人物研究会编|title=中国人民解放军高级将领传|volume=第11卷|year=2007|location=北京|publisher=解放军出版社|pages=275-328|isbn=7-5065-5287-6}}</ref>。在[[莒县战斗|莒县战斗]]中，曾成功争取[[莫正民|莫正民]]部投奔八路军，尔后里应外合攻占县城，使滨海、鲁中两根据地连成一片<ref name="gf"/>。此后，又同司令员[[陈士榘|陈士榘]]一起指挥[[诸城战役|诸城战役]]、[[郯马战役|郯马战役]]等。

=== 第二次国共内战时期 ===
[[File:HDYZJ.png|thumb]]、[[丁秋生|丁秋生]]、[[韦国清|韦国清]]、[[邓子恢|邓子恢]]、[[陈毅|陈毅]]、唐亮、[[粟裕|粟裕]]、[[陈士榘|陈士榘]]、[[谭震林|谭震林]]。]]
[[中日战争|中日战争]]结束后，中共部队战略调整，新四军主力进入[[山东军区|山东]]，与八路军山东军区合并。1946年1月，唐亮任[[新四军|新四军]]兼山东军区政治部副主任，[[山东野战军|山东野战军]]政治部主任<ref>{{cite book|title=国之栋梁|author=刘清芳|location=保定|publisher=河北大学出版社|year=2012|pages=204|isbn=978-7-5666-0110-0}}</ref>。1947年1月2日，唐亮主持制定了《山东野战军关于处理俘虏官兵的决定》，阐述补充[[国军|国军]]俘虏入解放军的政策<ref name="欧阳青">{{cite book |author=欧阳青 |title=大授衔：1955共和国将帅授衔档案|year=2011|publisher=长城出版社 |location=北京|pages=389-391|isbn=978-7-5483-0058-8}}</ref>。1947年1月23日，任华东军区政治部副主任[[华东野战军|华东野战军]]政治部主任<ref>{{cite book|author=南京军区《第三野战军战史》编辑室|title=中国人民解放军第三野战军战史|location=北京|publisher=解放军出版社|year=1999|pages=89|isbn=978-7-5065-5397-1}}</ref>。

1947年7月，唐亮任华东野战军西线兵团政治委员，与[[陈士榘|陈士榘]]率部进军鲁南<ref name="jlz"/>。9月，唐亮参与指挥[[沙土集战役|沙土集战役]]，全歼国军整编第57师。1948年3月，唐亮与陈士榘建议将[[士兵委员会|士兵委员会]]改为[[革命军人委员会|革命军人委员会]]，这个建议得到中共中央的赞同，并得以在全军实行。1948年3月，与陈士榘指挥4个纵队发起[[洛阳战役|洛阳战役]]<ref>{{Cite book|author=汪朝光|title=中国近代通史第十卷：中国命运的决战（1945-1949）|year=2013|publisher=江苏人民出版社|pages=356|isbn=978-7-2140-8098-1}}</ref>。6月参与指挥[[豫东战役|开封战役]]，并部署接管工作。在[[济南战役|济南战役]]时，提出“打到济南府，活捉[[王耀武|王耀武]]”的口号；在[[淮海战役|淮海战役]]时，提出“勇猛坚决，干脆彻底地消灭敌人，不让敌人逃到江南去”的口号<ref name="欧阳青"/>。

1949年1月，唐亮任[[第三野战军|第三野战军]]政治部主任，参与指挥[[渡江战役|渡江战役]]<ref>{{cite book|author=《中国人民解放军军史》编写组|title=中国人民解放军军史：第三卷|year=2010|publisher=军事科学出版社|pages=325|isbn=978-7-80237-381-5}}</ref>。1949年9月起任[[南京市|南京市]]军事管制委员会副主任、[[中共南京市委|中共南京市委]]第一副书记、书记<ref name="gf"/>。

=== 中华人民共和国成立后 ===
1950年8月，唐亮任中共华东军区党委第三书记，主持军区日常工作<ref name="bkqs">{{cite book |editor = 宋时轮 |title=《[[中国大百科全书|中国大百科全书]]》|volume=军事II|pages=1005|year=1989|location=北京 |publisher = 中国大百科全书出版社 |ISBN = 978-7-5000-0247-5}}</ref>。1952年10月起，任华东军区政治部主任兼干部管理部部长，军区副政治委员、党委第二书记。1955年4月，唐亮任[[南京军区|南京军区]]政治委员（1958年9月起为第二政委）<ref>{{cite book|author=《中国人民解放军军史》编写组|title=中国人民解放军军史：第五卷|year=2011|publisher=军事科学出版社|pages=21|isbn=978-7-80237-427-0}}</ref>。同年9月被授予[[中国人民解放军上将|上将]]军衔，获二级[[八一勋章_(1955年)|八一勋章]]、一级[[独立自由勋章|独立自由勋章]]、一级[[解放勋章|解放勋章]]。1958年5月，唐亮在[[中共八大二次会议|中共八大二次会议]]上增选为中央候补委员<ref>{{Cite web|url = http://dangshi.people.com.cn/GB/170835/178594/178595/10696406.html|title =“把我从中央委员候选名单里划去”--记唐亮上将|publisher=中国共产党新闻网|accessdate=2015-06-26}}</ref>。1964年1月，唐亮因病离职休养。1972年，唐亮出任[[中国人民解放军军政大学|中国人民解放军军政大学]]政治委员<ref>{{cite book|author=《中国人民解放军军史》编写组|title=中国人民解放军军史：第六卷|year=2011|publisher=军事科学出版社|pages=249|isbn=978-7-80237-427-0}}</ref>。1977年任解放军政治学院院长，后改任政治委员<ref name="jlz"/>。

唐亮是第一、二、三届[[国防委员会|国防委员会]]委员，中国共产党第八、九、十、十一届候补中央委员。在[[中国共产党第十二次全国代表大会|中国共产党第十二次全国代表大会]]上被选为[[中央顾问委员会|中央顾问委员会]]委员。1986年11月20日，病逝于[[北京|北京]]<ref>{{cite book| language = zh-hans|author= 星火燎原编辑部 |title=中国人民解放军将帅名录|volume=第一卷|year=2007|publisher=解放军出版社 |location=北京|isbn=7-5065-5031-8| pages = 134}}</ref>。他被官方评价为“久经考验的忠诚的共产主义战士、杰出的政治工作领导者、无产阶级革命家”<ref>{{cite web|url=http://dangshi.people.com.cn/GB/170835/178594/178595/10687665.html|title=久经考验的忠诚共产主义战士、杰出的政治工作领导者唐亮同志在京逝世 党和国家领导人及驻京部队指战员向他的遗体告别|publisher=人民日报|date=1986-12-03|accessdate=2015-06-25}}</ref>。
