'''唐莊宗李存-{勗}-'''，[[山西|山西]][[应县|应县]]人，[[沙陀|沙陀]]族，本姓'''朱邪'''，因其父是[[河東郡_(中國)|河東]][[節度使|節度使]][[李克用|李克用]]受[[唐懿宗|唐懿宗]]赐以李姓，而改姓李，諱'''存-{勗}-'''{{noteTag|1=-{勗}-，亦作「勖」，國語拼音：xù ，注音：ㄒㄩˋ，粵語拼音juk1<ref>[https://humanum.arts.cuhk.edu.hk/Lexis/lexi-can/search.php?q=%D4m 勖 - 粵語審音配詞字庫]</ref>。}}（{{bd|885年|12月2日|926年|5月15日|catIdx=L李}}），唐光启元年正月（885年12月）生于[[山西|山西]][[应县|应县]]，[[五代|五代]]時期[[后唐|后唐]]开国皇帝。小名“亚子”，藝名“'''李天下'''”，以勇猛闻名。

923年5月13日在[[魏州|魏州]]（[[河北|河北]][[大名府|大名府]]）称帝，[[国号|国号]]唐，史称[[后唐|后唐]]。後因義兄[[李嗣源|李嗣源]]被軍士擁戴[[造反|造反]]，揮軍直取[[洛陽|洛陽]]。宮中指揮使[[郭从谦|郭从谦]]為報仇，趁機發動兵變——[[興教門之變|興教門之變]]，將存勗殺害。

《旧五代史》记载：庄宗光圣神闵孝皇帝，讳存勖，武皇帝之长子也。母曰贞简皇后曹氏，以唐光启元年岁在乙巳，冬十月二十二日癸亥，生帝于晋阳宫。
<br />
《册府元龟》记载：后唐庄宗以光启元年十月癸亥生于晋阳宫。
<br />
《旧唐书》记载：十二月辛亥朔。
<br />
李存勖以唐光启元年十月二十二日（885年12月2日）生于晋阳宫，而十月二十二日不是癸亥是癸酉，十月十二日才是癸亥。同光元年至三年的万寿节（李存勖生日）皆在十月二十二日，李存勖的生日就是十月二十二日。光启元年十月壬子朔，癸亥是十二日，癸酉是二十二日，李存勖若生于十二日何以过二十二日生日，此当是编纂者失察误抄所致。

== 生平 ==

===少年時代===
[[李存勗|李存勗]]是[[李克用|李克用]]與[[貞簡皇后|貞簡皇后]]曹氏的長子。他自幼擅長[[騎馬|騎馬]][[射箭|射箭]]，膽力過人，為[[李克用|李克用]]所寵愛。少年時隨父作戰，11歲就與父親到長安向[[唐朝|唐朝]]朝廷報功，得到[[唐昭宗|唐昭宗]]的賞賜和誇獎。

[[李存勗|李存勗]]成年後狀貌雄偉瑰麗，得習《春秋》，豁達而且通大義，並勇敢善戰，熟知戰略要術。他又喜愛音樂、歌舞、[[俳優|俳優]]之戲，旁人多有異談。當時，軍閥割據混戰、佔據河東的[[李克用|李克用]]常被控制河南的[[朱溫|朱溫]]牽制圍困，兵力不足，地盤狹小，非常悲觀。[[李存勗|李存勗]]勸說其父：“[[朱全忠|朱全忠]]恃其武力，吞滅四鄰，想篡奪帝位，這是自取滅亡。我們千萬不可灰心喪氣，要積蓄力量，等待時機”。[[李克用|李克用]]聽後大為高興，重新振作起來，與[[朱全忠|朱全忠]]對抗。

===即位晉王===
後梁[[開平|開平]]二年(908年)正月，李克用病死，李存勗於同月襲晉王位。但是當時的兵馬大權歸於其叔父[[李克寧_(五代)|李克寧]]，軍民之事皆由李克寧決定，權柄既重，令眾人皆攀附李克寧。當辦完喪事後，李存勗與[[張承業|張承業]]、[[李存璋|李存璋]]設計，要除去勢力龐大的叔父[[李克寧|李克寧]]。同年二月二十日，當諸將於府第時，乃伏兵於府中，置酒大會，李克寧既至，於席間擒下[[李存顥|李存顥]]、李克寧二人，李存勗哭著責備李克寧：「姪兒一開始就打算把軍隊、政權都讓給叔父，叔父不願意背棄我父親的遺命，怎麼現在又把我跟我母親丟給豺狼虎豹？叔父怎麼忍心？」李克寧泣對：「這是讒言啊，我還能說甚麼？」當日，李克寧與李存顥俱伏法。

其後，李存勗認為潞州（今山西上黨）是河東屏障，沒有潞州對河東不利，所以他立即率軍從晉陽出發，直取上黨，乘大霧突襲圍潞州的梁軍，大獲全勝。李存勗的用兵之奇使梁太祖朱溫大驚，他說：「生兒子就要生李存勗一樣的兒子，[[李克用|李克用]]不會滅亡了啊！至於我的兒子，豬狗之輩而已！」{{noteTag|1=原文：「生子當如李亞子，克用為不亡矣！至如吾兒，豚犬耳！」（《[[資治通鑒|資治通鑒]]》卷266）}}

===建立後唐===
當潞州之圍解決後，河東威振，控制[[鎮州_(唐朝)|鎮州]]的[[王鎔|王鎔]]和控制[[定州|定州]]的[[王處直|王處直]]見形勢驟變，也動搖了依附後梁的信心，竟然和李存勗結成聯盟共同對付後梁。後梁為了保護河北之地，不惜一切，出兵再戰，於是雙方在[[柏鄉|柏鄉]]又展開了一場血戰。[[柏鄉之戰|柏鄉之戰]]中，晉軍有[[周德威|周德威]]等三千騎兵和鎮州、定州兵；對方梁軍有[[王景仁|王景仁]]率領的禁軍和魏博兵八萬。梁軍守衛柏鄉、以逸待勞，在地形、兵力、裝備幾方面處於優勢；而晉軍是騎兵，機動性和進攻能力大，對梁軍構成威脅。戰役開始，李存勗採用周德威建議，引誘梁兵出城，聚而殲之，晉軍主動後撤。梁軍主將王景仁果然上當，傾巢而出。晉軍抓住機會，以騎兵猛烈突擊梁軍，周德威攻右翼，李嗣源攻左翼，鼓譟而進。這時晉軍李存璋率領的騎兵大隊也趕上，梁軍丟盔棄甲，死傷殆盡。這一仗，使梁軍喪失了對河北的控制權，之後，朱溫一聽晉軍就談虎色變。而李存勗卻進一步安定了河東局勢，他息兵行賞，任用賢才，懲治貪官惡吏，寬刑減賦，一時河東大治。

李克用臨死時，交給李存勗三支箭，囑咐他要完成三件大事：一是討伐[[劉仁恭|劉仁恭]]，攻克[[幽州|幽州]]；二是征討[[契丹|契丹]]，解除北方邊境的威脅；第三件大事就是要消滅世敵朱溫。他將三支箭供奉在家廟裡，每臨出征就派人取來，放在精製的絲套裡，帶著上陣，打了勝仗，又送回家廟，表示完成了任務。其後李存勗達成李克用遺志，打敗契丹，攻破[[燕|燕]]地，並且攻滅[[劉守光|劉守光]]與[[劉仁恭|劉仁恭]]父子割據的[[桀燕|桀燕]]政權，並且於923年，在[[魏州|魏州]]（[[河北|河北]][[大名縣|大名縣]]西）稱帝，國號為唐，史稱[[後唐|後唐]]，其後攻滅[[後梁|後梁]]，統一北方。李存勗還收降了[[李茂貞|李茂貞]]建立的[[岐|岐]]，並攻滅[[王建_(前蜀)|王建]]所建立的[[前蜀|前蜀]]。

李存勗以唐朝赐姓为李的合法继承人身份，打起中兴[[唐朝|唐朝]]的旗号，并为[[唐朝|唐朝]][[皇帝|皇帝]]立庙。又以诛灭唐朝逆臣之名，[[族灭|族灭]]了[[后梁|后梁]][[宰相|宰相]][[敬翔|敬翔]]、[[李振|李振]]等人，将帮助朱溫篡唐的旧臣11人贬官。

但李存勗到了晚年自認為已經拚命一生，應該好好享樂，遂荒廢朝政。李存勗自幼喜歡看戲、演戲，常粉墨登場，並自號[[藝名|藝名]]“李天下”。[[伶人|伶人]]大受[[皇帝|皇帝]]寵幸，以至于伶人[[景进|景进]]干预朝政。士大夫皆气愤，又不敢出气。李存勗又派伶人、[[宦官|宦官]]搶民女入宮，強擄[[魏博|魏博]][[士卒|士卒]]們妻女千餘人，怨聲四起。同光二年，李存勗恢复旧唐宦官的势力，本来已经消失的监军又凌驾于藩镇之上，导致诸将更大的不满。<ref>《资治通鉴·卷二百七十三》</ref>[[同光|同光]]三年（925年），李存勗派遣兒子魏王[[李继岌|李继岌]]、侍中[[郭崇韬|郭崇韬]]，攻滅前蜀。但是其後继岌、崇韬互相猜疑。郭崇韬又得罪宦官，李存勗於是对崇韬起疑，下命[[孟知祥|孟知祥]]入蜀，见机行事<ref>《资治通鉴·卷二百七十四》及孟知祥将行，帝语之曰：“闻郭崇韬有异志，卿到，为朕诛之。”知祥曰：“崇韬，国之勋旧，不宜有此。俟臣至蜀察之，苟无他志则遣还。”帝许之。</ref>。翌年，李存勗被宦官的谗言所迷惑，诛杀了[[朱友谦|朱友谦]]、[[李存乂|李存乂]]。后唐朝廷人心惶惶。

後唐同光四年（926年），[[魏博|魏博]]士兵[[皇甫暉|皇甫暉]]在[[鄴城|鄴城]]叛亂，是為[[鄴城之亂|鄴城之亂]]，李存勗命[[李绍荣|李绍荣]]前往討伐，久不能下，无奈命[[李嗣源|李嗣源]]攻[[鄴城|鄴城]]，李嗣源命其女婿[[石敬瑭|石敬瑭]]同征。兵進[[魏州|魏州]]時，李嗣源卻被叛軍擁戴，恭迎入城，李嗣源百口莫辯，石敬瑭表示就算不造反也無法免責，李嗣源因而擁兵自立，與魏博的叛軍合兵造反。李嗣源占據[[汴州|汴州]]（今[[河南|河南]][[開封|開封]]），進軍[[洛邑|洛邑]]，先鋒[[石敬瑭|石敬瑭]]則帶兵逼進汜水關（河南[[滎陽|滎陽]]汜水鎮），李存勗決定親-{征}-反擊。

===[[興教門之變|興教門之變]]===
這時擔任指揮使的伶人[[郭从谦|郭从谦]]不知[[李存乂|李存乂]]已被莊宗殺死，欲奉李存乂之名作乱，火燒[[興教門|興教門]]。蕃汉马步使[[朱守殷|朱守殷]]见危不救。李存勗當時僅有[[符彥卿|符彥卿]]及[[王全斌|王全斌]]等少數將領效忠他。[[郭从谦|郭从谦]]率兵攻入皇城。李存勗被流箭射中。[[王全斌|王全斌]]將其扶至[[絳霄殿|絳霄殿]]。李存勗失血過多，渴懑求飲，經宦官奉進[[酪漿|酪漿]]，喝完一杯，遽爾殞命。<ref>《[[资治通鉴|资治通鉴]]》卷二七五，“亂兵焚興敎門，緣城而入，近臣宿將皆釋甲潛遁，獨散員都指揮使李彥卿及宿衞軍校何福進、王全斌等十餘人力戰。俄而帝為流矢所中，鷹坊人善友扶帝自門樓下，至絳霄殿廡下，抽矢，渴懣求水，皇后不自省視，遣宦者進酪，須臾，帝殂。”，[[胡三省|胡三省]]註：“酪，歷各翻，乳漿也。凡中矢刃傷血悶者，得水尚可活，飲酪是速死也。”</ref>王全斌大慟而去。一名伶人揀丟棄的樂器放在李存勗屍體上，點火焚屍。<ref>《[[新五代史|新五代史]]》卷十四，庄宗敬皇后刘氏传：“郭从谦反，庄宗中流矢，伤甚，卧绛霄殿廓下，渴欲得饮，后令宦官进飧酪，不自省视。庄宗崩，后与李存渥等焚嘉庆殿，拥百骑出师子门。后于马上以囊盛金器宝带，欲于太原造寺为尼。在道与存渥奸，及至太原，乃削发为尼。明宗入立，遣人赐后死。晋天福五年，追谥曰神闵敬皇后。”</ref>史稱[[興教門之變|興教門之變]]。[[李嗣源|李嗣源]]入[[洛陽|洛陽]]杀尽叛臣，葬李存勗屍骨于[[雍陵|雍陵]]，進廟號莊宗，李嗣源在[[汴州|汴州]]稱帝，是為[[後唐明宗|後唐明宗]]。

== 评价 ==
李存勗[[稱帝|稱帝]][[即位|即位]]之前，和[[后梁|后梁]]血战十餘年，大小百餘战，作战英勇异常。但打了天下，却不懂得治天下，宠幸伶人，重用[[宦官|宦官]]，又吝於[[銀錢|銀錢]]，不抚恤[[士卒|士卒]]，三年後因[[兵变|兵变]]被杀，失败之速，亦是罕见。<br />

北宋[[歐陽修|歐陽修]]寫《[[新五代史|新五代史]]·伶官傳序》便是討論李存勗沉溺逸樂、寵信樂官而致亡國的史實，叹惜李存勗“方其盛也，举天下之豪杰复能与之争；及其衰也，数十伶人困之，而身死国灭，为天下笑。”，說明“憂勞可以興國，逸豫可以亡身”的歷史規律{{noteTag|1=《[[新五代史|新五代史]]》：“其战于胡柳也，嬖伶周匝为梁人所得。其后灭梁入汴，周匝谒于马前，莊宗得之喜甚，赐以金帛，劳其良苦。周匝对曰：‘身陷仇人，而得不死以生者，教坊使陈俊，内园栽接使储德源之力也。愿乞二州以报此二人。’莊宗皆许以为刺史。郭崇韬谏曰：‘陛下所与共取天下者，皆英豪忠勇之士。今大功始就，封赏未及于一人，而先以伶人为刺史，恐失天下心。不可！’因格其命。逾年，而伶人屡以为言，莊宗谓崇韬曰：‘吾已许周匝矣，使吾惭见此三人，公言虽正，然当为我屈意行之。’卒以俊为景州刺史，德源为宪州刺史。” }} 。<br />

《[[旧五代史|旧五代史]]》则称赞李存勗是“中兴之主”，是唐朝的合法继承者，但語鋒一轉，隨即批評他“忘櫛沐之艱難，徇色禽之荒樂”、“伶人亂政、靳吝貨財、大臣無罪以獲誅、眾口吞聲而避禍”{{noteTag|1=《[[旧五代史|旧五代史]]·[[唐书|唐书]]·庄宗纪》 }} 。

[[朱温|朱温]]评价李存勗说“生子当如李亚子”，[[中國共產黨中央委員會主席|中國共產黨中央委員會主席]][[毛澤東|毛澤東]]也同意這個看法。<ref name=" 毛泽东点评历代帝王 纣王能文能武很有本事">{{cite web |url=http://news.ifeng.com/history/3/200711/1115_337_298656_1.shtml|title=《毛泽东点评历代帝王 纣王能文能武很有本事》|accessdate=2007年11月15日|work=|publisher=凤凰网|language=zh-hans}}</ref><ref name="毛泽东妙语评点历代帝王 ">{{cite web |url=http://www.china.com.cn/culture/txt/2007-08/22/content_8729040.htm|title=《毛泽东妙语评点历代帝王》|accessdate=2007-08-22|work=|publisher=中国网|language=zh-hans}}</ref><ref name="毛泽东对历代帝王的评说 ">{{cite web |url=http://news.xinhuanet.com/politics/2007-11/15/content_7077554.htm|title=《毛泽东对历代帝王的评说》|accessdate=2007年11月15日|work=|publisher= 新华网|language=zh-hans}}</ref>
