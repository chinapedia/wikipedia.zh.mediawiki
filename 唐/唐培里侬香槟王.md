'''唐培里侬香槟王'''（[[法語|法語]]：'''Dom Pérignon'''），是[[法國|法國]]的[[香檳|香檳]]品牌，創建於1668年，目前隶属于[[酩悅·軒尼詩－路易·威登集團|酩悅·軒尼詩－路易·威登集團]]（LVMH）旗下。

==历史==
唐培里侬香槟王是由位于Hautvillers[[本笃会|本笃会]]修道院[[修士|修士]][[唐·皮耶尔·培里侬|唐·皮耶尔·培里侬]]（Dom Pierre Pérignon，1638-1715）創立，本是在該修道院管理酒窖。他是最早通过调配不同种类[[葡萄|葡萄]]来改善[[葡萄酒|葡萄酒]]品质的人；1670年，他最先引用[[软木塞|软木塞]]，以保持[[葡萄酒|葡萄酒]]新鲜，同时采用更厚玻璃酒瓶，以面对当时酒瓶很易爆炸的问题。

在十八世紀起，唐培里侬香槟王已深受法國王室認可，并在國王[[路易十四|路易十四]]宫廷上享用。
<ref>{{cite web|url=http://www.hislife.com.cn/article_2099.html|title=唐培里侬香槟王2000年份粉红香槟|accessdate=2010-03-31|deadurl=yes|archiveurl=https://web.archive.org/web/20111120062934/http://www.hislife.com.cn/article_2099.html|archivedate=2011-11-20}}</ref>
随后又在[[英国|英国]]稳固了其地位。

==特色==
唐培里侬一直遵循以下十条特色：<ref>{{cite web |url=http://www.neeu.com/news/2010-01-29/14066_1.html|title=浪漫的注解 唐培里侬香槟王|accessdate=2010-01-29}}</ref>

* 所有旗下香槟酒都是年份[[葡萄酒|葡萄酒]]。
* 经过独特调配制成。
* 将[[白香槟|白香槟]]与[[粉红香槟|粉红香槟]]进行调配，使黑、白葡萄达到完美平衡。
* 原料选用[[法国|法国]]香槟区葡萄。
* 每个流程都严格遵循传统，尊重[[葡萄|葡萄]]生长的土壤与气候。
* 张力来自于精密度。
* 独特口感：口中触感浑然天成，新鲜、成熟、浓厚、轻柔，有着爱抚丝绒般的质感。
* 保证长达7年的酝酿过程。
* 独特醇香源自其矿物性质。
* 个性优雅、微妙而独特。

==产品==
唐培里侬香槟王只在[[葡萄|葡萄]]质量俱佳的年份酿造陈酿的年份香槟。唐培里侬香槟王的三大产品系列包括年份香槟(Vintage)、年份粉红香槟(Rosé Vintage)，以及珍藏年份香槟 (Œnothèque)。目前，最新的香槟年份包括2002年份香槟、2000年份粉红香槟、1996年珍藏年份香槟和1990年珍藏年份粉红香槟。

===唐培里侬1990年珍藏年份粉红香槟===
该香槟由于采用夏多内葡萄与黑皮诺葡萄作为原料，口感细腻，并有丰富香气；白葡萄与黑葡萄搭配产生口感味觉的平衡。<ref>{{cite web |url=http://fcwchenping.blog.163.com/blog/static/10808719220110701328192/|title=唐培里侬香槟王1990年珍藏年份粉红香槟|accessdate=2011-01-07}}</ref>

===唐培里侬香槟王1996年珍藏年份香槟===
该香槟采用醇厚的黑皮诺葡萄酿造，口感与气味都较为强烈。<ref>{{cite web |url=http://www.gq.com.cn/34/20100927/2504.html|title=挑战时间 唐培里侬香槟王1996年份珍藏香槟|accessdate=2010-09-27}}</ref>

===唐培里侬香槟王2000年份粉红香槟===
渐变的琥珀色，黄铜色和金黄色组成了结构缜密的2000年年份粉红香槟。其气味浓郁，混合了花香，黑樱桃，柑橘果香以及可可的烟熏味。该香槟气泡丰富，口感从浓郁渐变成柔滑细腻，并带有浓缩的奶味和葡萄柚的涩味。<ref>{{cite web |url=http://fashion.ifeng.com/news/detail_2010_07/20/1803927_0.shtml|title=Dom Pérignon 2000年份粉红香槟|accessdate=2010-07-20}}</ref>

===唐培里侬香槟王2002年份香槟===
该年份香槟兼有新鲜杏仁、麦收时节的气味，柠檬、干果、烟熏和烤面包的气味与浓香。口感醇厚且轻盈，并略带矿物质感。<ref>{{cite web |url=http://lux.hexun.com/2010-08-16/124606229.html|title=唐培里侬香槟王2002年份香槟发布|accessdate=2010-08-16}}</ref>
