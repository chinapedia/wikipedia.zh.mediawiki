== 故事簡介 ==
迪爾在某次委託下進入深邃之森，在那裡遇見了一名骨瘦如柴的魔族少女。那名身上帶有罪人烙印的少女，拉提娜讓迪爾不忍將她棄置在森林內，猜想這或許是某種緣分的迪爾，於是決定擔任少女的監護人，可是－－「拉提娜太可愛了，我不想去工作！」「你又發作啦？」－－迪爾竟在不知不覺之間，完全變成了溺愛女兒的傻瓜父親!?超強冒險者青年與神祕魔族少女的溫馨奇幻故事就此揭幕！！

== 登場角色 ==
※聲優：第六卷ドラマCD／電視動畫
; {{nihongo|迪爾·雷齊|デイル・レキ|Dale Leki，聲優：[[小林裕介|小林裕介]]／[[岡本信彦|岡本信彦]]}}
:'''''別稱：「橙之神（科莫賽）」的高階神官、白金勇者、傻爸爸(老公)、殘念勇者。'''''
: 「提斯洛」的民族的一員，受到「橙之神（科莫賽）」的寵愛，擅長製作魔道具。族長家的長子，從小就以提斯洛繼承人為前提下，受到族人的各種訓練，後來也因為擁有象徵「勇者」的複數加護存在，為了確保族人與周遭領地的外交關係，被族人賦予職務名「雷齊」(意思為「在外戰鬥的戰士」)，十五歲便離開村落，做為一名冒險者協助拉邦德國討伐魔族，靠著數年來立下的成績，樹立起不會讓自己因為年紀而被輕視的名聲。而族長繼承人則由弟弟「約爾克」接任。
:在某次討伐魔獸的任務中，在「庫羅茲」的西南方森林撿到年幼的拉提娜，順勢成為拉提娜監護人。雖然是實力非凡的冒險者，但在面對拉提娜時就會完全變成溺愛女兒的傻瓜父親！對接近拉提娜的任何男性都抱有著敵意。
:一直將拉提娜當作可愛的女兒看待，但在得知拉提娜的感情後，經過肯尼斯的開導，逐漸將拉提娜視為一名女性看待。與拉提娜成為戀人後總是不自重的在眾人面前放閃，也時常在「傻爸爸」與「傻老公」間切換。
:從未告知拉提娜自己是勇者的事，在被拉提娜變為眷族後，成為擁有魔族與勇者之力的「超規格勇者」，除了實力大幅提升，睡眠與進食也只需最低限度而已，甚至殘念的用在欣賞拉提娜一整晚的睡臉。
:在拉提娜消失後，帶著哈格爾以驚人的行動力討伐各魔王，並以「白金勇者」廣為所知，除了偶爾穿上提斯洛製作的白金盔甲，其他大多數的時候總是披著黑色的魔獸大衣，絲毫沒有白金的元素。實際上主要的由來是，為了「白金」而戰。
:在後傳的奇幻夢境裡，在面對拉提娜的父親拉古覺得比跟魔王對峙還緊張，也不會對拉提娜像拉古撒嬌的樣子吃醋。
; {{nihongo|拉提娜|ラティナ|Latina，聲優：[[日高里菜|日高里菜]]／[[高尾奏音|高尾奏音]]}}
:'''''年齡變化：7歲→8歲→10歲→14歲→16歲'''''
: 在7歲時遭到故鄉「凡斯略」流放，後被迪爾所撿到的魔族少女，坦率善良又聰明，學習能力異常的強大，擁有「天」和「冥」屬性魔法。從小便展顯出驚人的魔法操控力，但也有意外頑固的一面。被迪爾發現時，左邊身為魔人族象徵的角已經斷裂，在森林中的男性魔人族骨骸是拉提娜的父親兼導師「拉古」。
:8歲時在黃之殿學習，受到新老師的言語攻擊事件後，自己用魔法折斷了右邊僅存的角。角被魯道夫做成飾品掛在脖子上。
: 從小在虎貓亭向師父肯尼斯學習各種料理，在食物的話題上與肯尼斯有莫名的共識。對來到虎貓亭凶神惡煞冒險者不會感到害怕，也因拉提娜可愛的樣子，深受許多有地位的冒險者與憲兵喜愛，成為虎貓亭的看板娘，店裡的來客數會隨著拉提娜是否在店裡而有驚人的起伏。對迪爾的感情也被店裡的常客廣為所知，也受到大力的支持。
: 在10歲的時候一同回到迪爾的故鄉「提斯洛」，乖巧勤奮的個性深受迪爾家人的喜愛，在離開前被迪爾的祖母「溫蒂嘉德」賦予職務名「木托」(意思為「守護家庭的女性」)，讓提斯洛成為拉提娜的另一個回歸之處。居住在提斯洛期間除了受到「黃之神（阿斯法爾）神官，柯爾奈略•卡卡伽」指導，瞭解了不少魔人族的知識，也習得了刷毛技巧，讓不少小狗屈服於拉提娜的手下，被溫蒂嘉德帶到提斯洛的禁地兼幻獸「天翔狼」的聖地，不靠「央」的魔法便讓天翔狼完全服從。
:第三卷末14歲時向迪爾告白，說出了沒把迪爾當爸爸的話(從小便將提爾當作一名異性看待)，引起了一陣風波。
:第四卷關係開始從父女向情人轉變，並且在16歲時被迪爾求婚推倒。因為無法忍耐魔人族與凡人族壽命上的差異，選擇成為第八位魔王「理外的魔王」，並把迪爾變為眷屬(=魔族)，將父親的名字「斯馬拉古蒂」作為迪爾的魔族名。
:為了保護身邊的人不被魔王殺害，自願被七位魔王封印。在提爾的面前突然消失，導致迪爾動身討伐魔王。
:原名普拉蒂娜，被稱為白金公主，其意義為——月亮，在大神官艾佩羅琪的神諭得知「月光將引領王者走向破滅。」，經過身為紫之神官的母親「莫芙」再次預言後得知，拉提娜可能在未來會被二之魔王當人質，為了保護姊妹倆，由父親拉古帶著遭受斷角流放的拉提娜離開凡斯略。有個姊姊芙莉索斯，成為了有黃金之名(太陽)的新一之魔王。
:從小便習得各種廚藝、裁縫與多樣的知識，在音樂與協調性意外地沒有天分。
; {{nihongo|肯尼斯·古倫格|ケニス・クリューゲル|Kenneth Klügel，聲優：[[田丸篤志|田丸篤志]]／[[小山剛志|小山剛志]]}}
: 迪爾所下榻的旅館「躍動的虎貓亭」的老闆，與妻子一同照料迪爾與拉提娜的生活。
:曾與迪爾一同冒險，算是迪爾的老大哥也是拉緹娜尊敬老師。是一名以研究各種地方料理及食材為人生目的之人。甚至就連他身為優秀冒險者且是頂尖重裝戰士的經歷，也都只是他為達成目標的手段而已。在找到自己最大目標的罕見食材——某種棲息在海中的龍種——之後，便入贅虎貓亭，從冒險者生活退休以料理人的身分來度過剩下的人生。
:與莉塔育有兩個孩子。
; {{nihongo|莉塔·古倫格|リタ・クリューゲル|Rita Klügel，聲優：[[松井惠理子|松井惠理子]]／[[沼倉愛美|沼倉愛美]]}}

:當虎貓亭的會計跟處理綠之神的傳言版，強悍的個性有時連店裡兇悍魯忙的冒險者都退避三舍。
:與肯尼斯育有兩個孩子。
; {{nihongo|芙莉索斯|フリソス|Furisosu}}
:拉提娜的雙胞胎姐姐。外表與拉提娜神似，不同於拉提娜的灰色眼睛，眼睛是富含魔力象徵的金黃色。與拉提娜一樣，擁有「天」和「冥」屬性魔法，兩人皆不能使用其他屬性的魔法。
:從小便被父母拉古與莫芙照顧在凡斯略神殿深處，與拉提娜感情十分融洽，由大神官艾佩羅琪的神諭中得知是「照亮吾等未來的是太陽之光。」但也因此被迫與拉提娜分離。
:為了讓找回拉提娜並保護她，成為了有黃金之名(太陽)的新一之魔王，在王之座中提出了封印理外之王的拉提娜，阻止了當場想直接動手的其他魔王。
:在其他魔王遭到迪爾殺害，拉提娜的封印逐漸弱化後，向拉邦德國提出建交。後來與拉提娜一起拜祭亡父拉古後，更提出將母親莫芙的遺體移葬於此，並建立廟宇紀念他們。除了藉此增加兩族交流之外，也希望父母能在兩個國度之間守望着拉提娜的幸福。

=== 魔王 ===
; {{Nihongo|一之魔王|}}
:象徵:權杖
:唯一一個治理魔人族國家的魔王，數百年前的一之魔王和繼承人慘遭二之魔王殺害，一直以來的空缺由芙莉索斯填補。
; {{Nihongo|二之魔王|}}
:象徵:刀刃
:別稱「殺戮魔王」。以制造死亡而感到愉悅的駭人魔王，親自用雙手取走無數生命。
; {{Nihongo|三之魔王|}}
:象徵:水瓶
:別名「海之魔王」。住在與水鱗族共存生活的大規模集落。在成為魔王之前，就已經活過相當長的時間，用從容的態度面對前來討伐自己的迪爾，感覺的出迪爾與八之魔王的關係，為了保護人民甘願被殺，並希望迪爾討伐會威脅聚落的七之魔王。
; {{Nihongo|四之魔王|}}
:象徵:樹幹
:罹患致死疾病，落入絕望深淵的存在，是她在成為「四之魔王」之前的模樣。對生的渴望，引誘她讓自己成為「病魔」，借著讓自己成為過去所懷抱的絕望，讓她成為了「四之魔王」。將濃密的魔素朝四周圍釋放，任何生物都難以逃過一劫。
; {{Nihongo|五之魔王|}}
:象徵:書卷
:別稱「塔之魔王」。為了滿足自己追求知識的願望而成為魔王，獲得「名為魔王的力量」不僅不會衰老，甚至已經超越了壽命的框架，耗費漫長歲月收集藏書。
:在得知理之魔王的出現後，像其他魔王提出剷除理之魔王的提議。
; {{Nihongo|六之魔王|}}
:象徵:巨劍
:別稱「巨人之王」。有著領袖魅力的王者風範，帶領族人四處流浪，由於是一名用心照顧自己民族的傑出領袖，迪爾對此抱著敬意，尊重他身為戰士的自尊，提出一對一的單挑。
; {{Nihongo|七之魔王|}}
:象徵:旗幟
:別稱「戰亂魔王」。熱愛戰爭，喜好用強勢武力蹂躪對手的存在。率領大軍讓無數人遭到蹂躪與毀滅，僅僅為了追求戰亂，掀起戰爭不需要任何理由，也不在乎結果。
