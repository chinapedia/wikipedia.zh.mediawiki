'''医学遗传学国家重点实验室'''是一个[[中华人民共和国国家重点实验室列表|国家重点实验室]]，[[科技部|科技部]]是实验室的宏观管理部门，[[教育部|教育部]]是实验室的行政主管部门，[[中南大学|中南大学]]是实验室的依托单位。[[夏昆|夏昆]]教授为实验室现任主任，[[张学敏|张学敏]]教授为学术委员会主任。

依照中国科技部发布的2016年生物和医学领域国家重点实验室评估结果，医学遗传学国家重点实验室未通过评估，不再列入国家重点实验室序列。<ref>{{cite news|author1=谭嘉|title=医学遗传学国家级实验室被摘牌_健康_央视网(cctv.com)|url=http://jiankang.cctv.com/2017/06/30/ARTInvZTW42Zu9Dh1XPGnXLw170630.shtml|accessdate=2018-04-22|work=健康报|date=2017-06-30}}</ref>

==附设机构==
*遗传学国家重点学科
*医学细胞遗传学国家培训中心
*中国遗传资源保藏中心
*中国基因诊断与治疗研究中心
*中南大学基因药物中心
*[[中南大学生命科学学院|中南大学生命科学学院]]遗传学系。
*国家生命科学与技术人才培养基地——基因科学与技术产业化点<ref>{{cite web |url=http://www.sklmg.edu.cn/articles_18.shtml |language= |title=实验室管理 |publisher=医学遗传学国家重点实验室 |date= |accessdate=2015年6月 |deadurl=yes |archiveurl=https://web.archive.org/web/20160412165023/http://www.sklmg.edu.cn/articles_18.shtml |archivedate=2016-04-12 }}</ref>

==研究方向==
医学遗传学国家重点实验室的研究聚焦于鉴定神经变性疾病、精神疾病和智力障碍等人类多发、严重疾病的遗传学和表观遗传学基础，建立相应的细胞和动物模型、探索其病理机制、开发新的诊断和治疗方法。<ref>{{cite web |url=http://www.sklmg.edu.cn/articles_21.shtml |language= |title=研究方向 |publisher=医学遗传学国家重点实验室 |date= |accessdate=2015年6月 |deadurl=yes |archiveurl=https://web.archive.org/web/20150527222239/http://www.sklmg.edu.cn/articles_21.shtml |archivedate=2015-05-27 }}</ref>
