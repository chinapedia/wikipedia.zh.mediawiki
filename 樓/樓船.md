'''樓船'''，[[中國|中國]]古代[[戰船|戰船]]，因[[船|船]]高[[首|首]]寬，外觀似[[樓|樓]]，而得名。因其船大樓高，遠攻近戰皆合宜，故為古代[[水戰|水戰]]之主力。但亦因船只過高，常致重心不稳，不適遠航，故多只在[[内河|内河]]及沿海的水戰中擔任主力。

== 簡介 ==
作戰用的樓船一般分三層，第一層為[[庐|庐]]；第二層為[[飛庐|飛庐]]；最上層為[[爵室|爵室]]<ref>《[[释名|释名]]》：「其上屋曰庐，像庐舍也。其上重屋曰飞庐，在上故曰飞也。又在其上曰爵（雀）室，于中侯望之如鸟爵之警视也。」 </ref>。每層都設有防護[[女牆|女牆]]，用以防禦敵方射來之[[弓箭|弓箭]]、[[矢石|矢石]]<ref>《[[太白阴经|太白阴经]]》：「船上建楼三重，列女墙、战格......忽遇暴风，人力不能制，不便于事。然为水军，不可不设，以张形势」</ref>。女牆上開有[[箭眼|箭眼]]，用以發射[[弓弩|弓弩]]。為防禦敵方[[火攻|火攻]]，船上蒙有[[皮革|皮革]]，以隔熱。樓船上常遍插[[旗幡|旗幡]]和[[刀槍|刀槍]]，以壯聲勢，又使用[[纖繩|纖繩]]、[[櫓|櫓]]、[[帆|帆]]和[[楫|楫]]等多種設備，以提升其機能。

== 歷史 ==
樓船之名始出現於[[春秋戰國時期|春秋戰國時期]]之[[越國|越國]]，其設有[[樓船軍|樓船軍]]<ref>[[席龍飛|席龍飛]]《中國造船史》，[[湖北教育出版社|湖北教育出版社]]出版</ref>。而真正於大戰爭中出現則是於吳國，[[吳王僚|吳王僚]]二年（前525年），於[[吴楚长岸之战|吴楚长岸之战]]中，吳國即以大型樓船「[[余皇|余皇]]」，作為[[指揮艦|指揮艦]]。

[[西漢|西漢]]时期，樓船開始成為主力戰艦，為主帥所乘，船上可容兵員數十至數百名，因此漢代[[將軍|將軍]]中有統帥水軍的[[樓船將軍|樓船將軍]]，水軍亦稱[[樓船士|樓船士]]。在[[漢武帝|漢武帝]]發兵滅[[南越|南越]]、[[衛氏朝鮮|朝鮮]]等戰爭中，樓船都發揮了重要的作用。
[[File:Gulangyu-ferry.jpg|缩略图]]
[[三國|三國]]至[[南北朝|南北朝]]時期，樓船仍普遍運用在水戰中，其中[[東吳|東吳]]所造的樓船如：「[[飛雲|飛雲]]」、「[[蓋海|蓋海]]」等，皆有五層，可載3,000名士兵。[[西晉|西晉]][[泰始|泰始]]八年（272年）[[晉武帝|晉武帝]]升[[王濬|王濬]]為[[益州|益州]][[刺史|刺史]]，並密命其於[[四川|四川]]組建樓船，以滅東吳，其所造之船，最大的可載2,000多人，且能在船上馳馬往來<ref>《[[晉書|晉書]]·王濬傳》：「方百二十步，受二千人。以木为城，起楼橹，开四出门，其上皆得驰马来往。」</ref>。[[五代十國|五代十國]]時期，南方諸國，設有[[樓船指揮使|樓船指揮使]]一職，閩王[[王延鈞|王延鈞]][[從子|從子]][[王仁達|王仁達]]即曾任此職。至宋代，樓船仍為軍事裝備，但重要性已大不如前。

== 用途 ==
除戰爭外，樓船亦被當作一種高級之[[遊船|遊船]]。[[西漢|西漢]][[漢武帝|漢武帝]]執政之時建有[[豫章樓船|豫章樓船]]，可以乘载万人，船上起宫室<ref>《[[太平御覽|太平御覽]]·舟部三》</ref>。[[新朝|新朝]]末[[東漢|東漢]]初，[[公孫述|公孫述]]盤據[[漢中|漢中]]之時，即曾經建造用[[絲帛|絲帛]]裝飾的[[十層赤楼帛兰船|十層赤楼帛兰船]]<ref>《[[後漢書|後漢書]]·[[隗囂|隗囂]][[公孫述|公孫述]]列傳》：「遂大作營壘，陳車騎，肄習戰射，會聚兵甲數十萬人，積糧漢中，築宮南鄭。又造十層赤樓帛蘭船。多刻天下牧守印章，備置公卿百官。」</ref>。
