== 歷史 ==
[[欧盟|欧盟]]是[[日本|日本]]的第三大贸易合作伙伴。欧盟在[[亞洲|亞洲]]的第二大貿易夥伴是日本（第一大貿易夥伴為[[中國|中國]]），歐盟企業每年出口到日本貿易量為580億歐元，約2兆新台幣，其中78%是中小企業；日本每年出口到歐盟貿易量為280億歐元，約新台幣1兆元；協議全面實施後日本將取消97%、價值約10億歐元的歐盟進口貨物關稅，<ref name = "歐盟赴日簽署貿易協議 共組抗美陣線_2018" /><ref name = 中華經濟研究院_史惠慈> {{ cite web | title = None | url = http://www.cier.edu.tw/site/cier/public/data/173-17%E5%8D%80%E5%9F%9F%E7%B6%93%E6%BF%9F%E8%AB%96%E5%A3%87.pdf | publisher = 中華經濟研究院 | author = 史惠慈 | language = zh-tw | accessdate = 2018-07-26 }} </ref>

* 2011年5月日本與歐盟開始準備經濟夥伴協定。<ref name = 中華經濟研究院_史惠慈 />
* 2012年11月，歐盟理事會同意與日本展開談判。<ref name = 中華經濟研究院_史惠慈 />
* 2013年4月日欧经济伙伴关系协定（EPA）正式启动首次談判。<ref name = 中華經濟研究院_史惠慈 />
* 2017年，目前双方的矛盾焦点在于，欧洲要求开放日本[[农产品|农产品]]市场，而日本则希望欧洲降低[[汽车|汽车]][[关税|关税]]。这一次在会议结束前日本方面就放出消息，将下调对欧进口[[猪肉|猪肉]]的关税，无疑是希望加快谈判进程，释放善意。同时日本希望在[[英国|英国]]完成[[英國脫離歐盟|脱欧]]程序前，正式签订与欧洲的经济伙伴关系协定，以免欧洲政治安全局势动荡带来影响。目前日本市场上的猪肉有5成来自进口。其中欧洲产猪肉占到整体的17%。
* 2017年7月6日，歐盟發佈與日本談判後的經濟夥伴關係協定內容。<ref name = 中華經濟研究院_史惠慈 />
* 2017年12月8日，歐盟貿易官員Cecilia Malmström與日本外務省宣布討論達成最後協議。<ref name = "#EUTrade news_2017"> {{ cite web | title = #EUTrade news | url = http://trade.ec.europa.eu/doclib/press/index.cfm?id=1767 | publisher = Trade - European Commission | language = en | date = 2017-12-08 }} </ref>
* 2018年7月17日，歐盟官員執委會主席容克（Jean-Claude Juncker）與理事會主席圖斯克（Donald Tusk）飛抵日本與[[日本首相|日本首相]][[安倍晉三|安倍晉三]]簽訂經濟夥伴協定，仍待日本國會與歐盟議會通過後，才會正式實施。<ref name = "歐盟赴日簽署貿易協議 共組抗美陣線_2018" /><ref name = "「全球最大的自由貿易協定」將成型！歐盟、日本正式簽署經濟夥伴關係協定_2018" />
* 2018年12月8日，日本參院在全體會議上批准了與歐盟的經濟夥伴關係協定（EPA），日方完成國會程序。
* 2018年12月12日，歐洲議會表決通過名為“經濟夥伴關係協定”的歐盟與日本貿易協定。
* 2019年2月1日，日歐經濟合作協定正式生效。
