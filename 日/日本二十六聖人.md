'''日本二十六聖人'''是1597年2月5日（[[日本|日本]][[慶長|慶長]]2年12月19日）[[豐臣秀吉|豐臣秀吉]]下令在[[長崎市|長崎]]處決26名[[天主教徒|天主教徒]]，此事件後來被稱作「二十六人的[[殉教|殉教]]」，這4位[[西班牙|西班牙]]籍[[傳教士|傳教士]]、1位[[墨西哥|墨西哥]]籍（當時稱為「[[新西班牙總督轄區|新西班牙]]」）[[修士|修士]]、1位[[葡萄牙|葡萄牙]]籍修士、20位日本籍[[信徒|信徒]]（含3名未成年者）被追封為[[聖人|聖人]]，遂以「日本二十六聖人」稱之。

== 事件始末 ==
十六世紀末葉相繼發生信奉[[天主教|天主教]]的[[吉利支丹大名|吉利支丹大名]]與[[吉利支丹|吉利支丹]]燒燬本地[[寺廟|寺廟]]和[[神社|神社]]並迫害僧侶的事件；相對的，各地信奉[[佛教|佛教]]或[[神道|神道]]的[[大名|大名]]迫害吉利支丹的事情也時有所聞。此外，[[葡萄牙|葡萄牙]]商人將日本人當作[[奴隸|奴隸]]販賣到海外的事情爆發後，豐臣秀吉遂於[[天正_(日本)|天正]]15年（1587年）頒佈《[[伴天連追放令|伴天連追放令]]》，定天主教為[[邪教|邪教]]。不過，著眼於[[南蠻貿易|南蠻貿易]]的實際利益，並未大規模地迫害該教。因此天主教仍在暗地裡[[布道|布道]]傳教，[[洗禮|受洗]]成為吉利支丹的日本人也不會被公家強迫還俗。不過，[[文禄|文禄]]5年（1596年）10月發生西班牙船艦「聖菲利浦號」漂流至[[土佐國|土佐國]]的{{link-ja|聖菲利浦號事件|サン＝フェリペ号事件}}，豐臣秀吉再度於12月8日頒布禁教令。另外，繼[[耶穌會|耶穌會]]後進入日本傳教的[[方濟各會|方濟各會]]非常活躍，不理會禁教令繼續佈教，故[[京都|京都]][[奉行|奉行]][[石田三成|石田三成]]下令逮捕京都內的方濟各會的傳教士與[[基督徒|基督徒]]，並予以處刑。這些方濟各會信徒都追隨西班牙[[阿爾坎塔拉|阿爾坎塔拉]]出身的神父伯多祿·包蒂斯塔（Pedro Bautista），故被稱作「阿爾坎塔拉派」。於是在[[大阪|大阪]]和京都逮捕了方濟各會成員7名、信徒14名，以及耶穌會成員3名，合計共24名。原本石田三成打算排除包括{{link-ja|保祿·三木|パウロ三木}}在內的3名耶穌會成員，結果因故作罷。

24人被帶到京都[[堀川通|堀川通]]的橋邊削去左耳（豐臣秀吉命令削去[[耳朵|耳朵]]與[[鼻子|鼻子]]），並在城內遊街示眾。[[慶長|慶長]]2年（1597年）11月24日豐臣秀吉下令將囚犯押往長崎行刑。詎料受託照顧耶穌會成員的伯多祿·助四郎（{{lang|ja|ペトロ助四郎}}）及照料方濟各會成員的伊勢的方濟各（{{lang|ja|伊勢のフランシスコ}}）也被捉拿，這二人身為基督徒，為了[[信仰|信仰]]而犧牲生命。一行人在寒冷的冬季裡徒步走向長崎，負責帶隊押人的寺澤半三郎見到其中12歲少年路易斯·茨木（{{lang|ja|ルドビコ茨木}}）咬牙苦撐的樣子，曾勸他放棄信仰，但卻被路易斯鄭重其事地拒絕。迪亞戈·喜齋（{{lang|ja|ディエゴ喜斎}}）和五島的若翰·草庵（{{lang|ja|五島のヨハネ草庵}}）則在向耶穌會神父[[巴範濟|巴範濟]]（{{lang|it|Francesco Pasio}}）[[和好聖事|告解懺悔]]並立下誓願後，獲准入籍耶穌會。

行刑日期在[[慶長|慶長]]2年12月19日（1597年2月5日），地點並非選在一般的刑場，這26人被帶往長崎西坂的山丘上。行刑前為了避免市區內的混亂而發出禁止外出令，但是西坂山丘上仍聚集了超過4千人。臨死之前，保祿·三木仍持續向圍觀群眾宣揚自己的信仰；一行人在當日上午10點左右遭到[[劊子手|劊子手]]以長槍刺穿兩腋的方式身亡殉教。

由於天主教教會具有保留[[聖髑|聖髑]]的傳統，這些日本最初的殉教者遺骸被分送至世界各地，並受到尊崇保存。因為當時有位葡萄牙籍傳教士[[路易斯·弗洛伊斯|路易斯·弗洛伊斯]]紀錄下了報告書，日本二十六聖人的殉教事蹟反而在[[歐洲|歐洲]]流傳開來。1862年6月8日[[教宗|羅馬教宗]][[庇護九世|庇護九世]]宣布將這26人[[基督教聖人|列聖]]；1962年適逢列聖100周年紀念，由日本建築名家[[今井兼次|今井兼次]]設計的[[日本二十六聖人紀念館|日本二十六聖人紀念館]]與雕刻家[[舟越保武|舟越保武]]設計的紀念碑在長崎市西坂的山丘上成立，天主教會將這26位聖人殉難的2月5日列為節日。

== 二十六聖人之姓名 ==
以下姓名{{NoteTag|所列之翻譯採天主教譯名：方濟各（法蘭西斯）、伯多祿（彼得）、若望（約翰）、馬爾定（馬丁）、多默（多馬）、良（李奧）、保祿（保羅）、彌額爾（米迦勒）、嘉俾額爾（加百列）、味增爵（文森特）、葛斯默（科司梅）。}}順序依二十六聖人紀念碑右側為先：
# '''伊勢的方濟各'''（{{lang-ja|伊勢のフランシスコ}}）：日籍[[木匠|木匠]]，為了照料[[方濟各會|方濟各會]]成員而同行，不料途中被捕。
# '''葛斯默·竹屋'''（{{lang-ja|コスメ竹屋}}）：日籍、38歲，於[[大阪市|大阪]]被捕。
# '''伯多祿·助四郎'''（{{lang-ja|ペトロ助四郎}}），另稱「佩多羅·助四郎」（{{lang-ja|ペドロ助四郎}}）{{NoteTag|差異在第二字[[清濁音|清濁音]]的不同。}}）：日籍，為了照顧[[耶穌會|耶穌會]]成員而同行，但途中遭逮捕。
# '''彌額爾·小崎'''（{{lang-ja|ミゲル小崎}}），另稱「米卡爾·小崎」（{{lang-ja|ミカエル小崎}}）：日籍、46歲，多默·小崎之父，於[[京都市|京都]]被捕。
# '''雅各伯·喜齋'''（{{lang-ja|ディエゴ喜斎}}）：日籍、64歲，為耶穌會成員，於大阪被捕。
# {{link-ja|'''保祿·三木'''|パウロ三木}}（{{lang-ja|パウロ三木}}）：日籍、33歲，為耶穌會成員，於大阪被捕。
# '''保祿·茨木'''（{{lang-ja|パウロ茨木}}）：日籍、54歲，良·烏丸之兄，於京都被捕。
# '''五島的若望·草庵'''（{{lang-ja|五島のヨハネ草庵}}），另稱「若望·五島」({{lang-ja|ヨハネ五島}}）：日籍、19歲，為耶穌會成員，於大阪被捕。
# '''類思·茨木'''（{{lang-ja|ルドビコ茨木}}{{NoteTag|「ルドビコ」來自[[世界語|世界語]]「Ludoviko」，即為“Louis”。}}）：日籍、12歲（26人中最年幼），保祿·茨木、良·烏丸之[[甥|甥]]，於京都被捕。
# '''長崎的安多尼'''（{{lang-ja|長崎のアントニオ}}）：日籍、13歲，父親為[[中國人|中國人]]，鄭姓，母親為日本人，於京都被捕。
# '''伯多祿·包蒂斯塔'''（{{lang-es|Pedro Bautista}}）：[[西班牙|西班牙]]籍方濟各會[[祭司|神父]]、48歲，他一直是保祿·三木的同伴，於京都被捕<ref>{{CathEncy|wstitle=Sts. Peter Baptist and Twenty-Five Companions}}</ref>。尚未到日本之前，他在[[菲律賓|菲律賓]]傳教莫約十年。
# '''阿吉雷的馬爾定·羅伊納茲'''（{{lang-es|Martin Loynaz de Aguirre}}）：[[西班牙|西班牙]]籍方濟各會[[祭司|神父]]、30歲，於大阪被捕<ref>[http://saints.sqpn.com/saintm5m.htm Saint Martin of the Ascension] {{webarchive|url=https://web.archive.org/web/20120101193146/http://saints.sqpn.com/saintm5m.htm |date=2012-01-01 }}</ref>。
# '''耶穌的斐理伯'''（{{lang-es|Felipe de Jesús}}）：[[墨西哥|墨西哥]]籍（當時稱為「[[新西班牙總督轄區|新西班牙]]」）、24歲，方濟各會修士，於京都被捕。他殉教後成為墨西哥第一位被追封的[[聖人|聖人]]，同時也是[[墨西哥市|墨西哥市]]的[[主保聖人|主保聖人]]。
# '''貢薩洛·賈西亞'''（{{lang-es|Gonzalo Garcia}}）：[[葡萄牙|葡萄牙]]籍、40歲，方濟各會修士，於京都被捕。
# '''方濟各·布蘭科'''（{{lang-es|Francisco Branco}}）：西班牙籍、28歲，方濟各會神父，於京都被捕。
# '''聖邁克爾的方濟各'''（{{lang-es|San Francisco de San Miguel}}）：西班牙籍、53歲，方濟各會修士，於京都被捕。
# '''宮古的瑪第亞'''（{{lang-ja|マチアス}}）：日籍，於京都被捕。原本並未列在逮捕名單上，但因[[洗禮|洗禮]]名相同遭到拘捕。
# '''良·烏丸'''（{{lang-ja|レオ烏丸}}）：日籍、48歲，保祿·茨木之弟、類思·茨木之[[叔父|叔父]]。
# '''宮古的味增爵'''（{{lang-ja|ボナベントゥラ}}；{{lang-la|Bonaventura}}）：日籍，於京都被捕。
# '''多默·小崎'''（{{lang-ja|トマス小崎}}）：日籍、14歲，彌額爾·小崎之子。
# '''若亞敬·榊原'''（{{lang-ja|ホアキン榊原}}），另稱「尤阿金·榊原」（{{lang-ja|ヨアキム榊原}}）：日籍、40歲，於大阪被捕。
# '''方濟各醫生'''（{{lang-ja|医者のフランシスコ}}）：日籍、46歲，於京都被捕。
# '''[[布道|講道]]者多瑪斯'''（{{lang-ja|トマス談義者}}）：日籍、36歲，於京都被捕。
# '''絹屋的若望'''（{{lang-ja|絹屋のヨハネ}}）：日籍、28歲，於京都被捕。
# '''嘉俾額爾'''（{{lang-ja|ガブリエル}}）：日籍、19歲，於京都被捕。
# '''保祿·鈴木'''（{{lang-ja|パウロ鈴木}}）：日籍、49歲，於京都被捕。

== 奉二十六聖人為主保的教堂==
* 日本殉道聖人堂（Chiesa dei Sainti Martiri Giapponesi）：位於[[義大利|義大利]][[拉齊奧大區|拉齊奧大區]][[羅馬省|羅馬省]][[奇維塔韋基亞|奇維塔韋基亞]]。
* 二十六聖殉道者君王堂（Igreja Cristo Rei dos 26 Mártires）：位於[[巴西|巴西]][[聖保羅州|聖保羅州]]，由[[福岡縣|福岡縣]][[日裔巴西人|移民]]於1935年成立。
* [[大浦天主堂|大浦天主堂]]：位於[[長崎縣|長崎縣]][[長崎市|長崎市]]。
* 堂崎教堂：位於長崎縣[[五島市|五島市]]。
* 聖斐理伯堂（西坂教會）：位於長崎縣長崎市，該教會會所亦即[[日本二十六聖人紀念館|日本二十六聖人紀念館]]，由[[今井兼次|今井兼次]]設計。
* 本所教堂：位於[[東京都|東京都]][[墨田區|墨田區]]。
* 美濃加茂天主教堂：位於[[岐阜縣|岐阜縣]][[美濃加茂市|美濃加茂市]]。
