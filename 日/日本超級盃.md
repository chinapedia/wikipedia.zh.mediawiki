'''日本超級盃'''或'''富士全錄超級盃'''(富士ゼロックス スーパーカップ)是一年一度由[[日本足球協會|日本足球協會]]與[[J聯賽|J聯賽]]賽會於[[日本|日本]]舉行的[[足球|足球]]比賽，比賽於一季的開始舉行並由上季[[J聯賽|J聯賽]]冠軍與[[天皇盃|天皇盃]]冠軍角逐。由1994年起，[[富士全錄|富士全錄]]便贊助本項賽事。

== 參賽球會 ==
在正常的情況下，參賽的球會是:
* 上年度[[J聯賽|J聯賽]]冠軍
* 上年度[[天皇盃|天皇盃]]冠軍

可是，若有同一支球隊奪得兩個冠軍時，就會由[[天皇盃|天皇盃]]的亞軍頂替席位。

== 比賽規則 ==
* 一場90分鐘的賽事(兩節45分鐘的賽事)
* 若法定時間未能分勝負，不設加時，立刻以互射[[點球|點球]]方式得出冠軍

== 以前的超級盃 ==
