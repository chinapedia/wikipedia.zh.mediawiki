'''日本郵船株式會社'''（[[日語|日語]]：{{lang|ja|日本郵船株式会社}}），是[[日本|日本]]代表性的大型[[船运|海運]][[公司|公司]]之一，與[[三菱商事|三菱商事]]同為[[三菱财阀|三菱財閥]]（今 [[三菱集团|三菱集團]]）的源流企業。為了使海空運能達到一條龍的服務，因此下設相關企業如郵船通運公司等，在主要海外各分支機構負責進出口承攬業務、報關業務、自主保稅倉庫、發貨中心、汽車貨運配送及庫存管理，使服務脈絡更廣闊。

== 沿革 ==
*1870年　九十九商會（之後改名為三川商會、三菱商會）設立。
*1875年　在三菱財閥創立者[[岩崎彌太郎|岩崎彌太郎]]的主導之下，三菱商會合併國有企業「日本國郵便蒸氣船會社」，並易名為「郵便汽船三菱會社」。
*1885年　郵便汽船三菱會社與另一家大型航運企業「[[共同運輸會社|共同運輸會社]]」合併，日本郵船會社成立。
*1893年　改制為[[株式會社|株式會社]]，日本郵船株式會社誕生。
*1926年　與第二東洋汽船株式會社合併。
*1945年　[[第二次世界大战|二戰]]末期，所屬船舶「[[阿波丸_(日本郵船)|阿波丸]]」在[[台灣海峽|台灣海峽]]遭遇[[美國海軍|美國海軍]][[潛水艇|潛水艇]]襲擊，並遭到撃沈（即[[阿波丸事件|阿波丸事件]]）。
*1949年　在東京、大阪、名古屋等證券交易所上市。
*1950年　於札幌證券交易所上市。　
*1960年　二戰少數免於遭受戰火波及的日本商船、也是日本郵船重要資產的「[[冰川丸|冰川丸]]」除役，日本郵船退出客輪事業。
*1964年　合併三菱海運株式會社（前三菱汽船、極東海運：原三菱商事船舶部分拆獨立而成）。
*1969年　近海、國內航線部門分割，並委託近海郵船株式會社經營。
*1973年　所屬船舶「[[山城丸|山城丸]]」在[[敘利亞|敘利亞]][[拉塔基亞|拉塔基亞]]（Latakia）停泊時，適逢[[贖罪日戰爭|第四次中東戰爭]]爆發，船隻遭受戰火波及而損毀。
*1989年　日本水晶郵輪株式會社（Crystal Cruises Japan；日後改名為郵船郵輪株式會社）設立。
*1998年　合併日本舊六大海運会社之一的昭和海運株式會社。
*2003年　日本郵船歷史博物館開館。
*2005年　收購[[全日本空輸|全日空]]持有的[[日本貨物航空|日本貨物航空（NCA）]]股份，使其成為日本郵船的子公司。
*2006年　與大型陸運業者[[大和運輸|大和運輸]]的[[母公司|母公司]][[大和控股|大和控股]]（{{lang|ja|ヤマトホールディングス}}）發表合作關係。
*2016年  日本郵船決定與[[商船三井|商船三井]]、{{link-ja|川崎汽船|川崎汽船}}等兩家航運公司合作，將彼此旗下的的[[貨櫃|貨櫃]]業務整併成一家貨櫃航運公司Ocean Network Express(ONE) 預計從2017年7月1日著手合併，自2018年4月1日即以新的聯盟運作，总部设在新加坡。将ONE收归旗下的持股公司Ocean Network ExpresHoldings(ONEHD)的总部位于东京，日本邮船对其出资比率为38%，商船三井与川崎汽船分别为31%。
<ref>[http://www.chinatimes.com/realtimenews/20161101003082-260410 日3海運貨櫃整併 業界：對市場是好事]，中時電子報，2016年11月01日</ref>。
