'''小<ruby class="zy center"><rb>碓</rb><rp>（</rp><rt>ㄉㄨㄟˋ</rt><rp>）</rp></ruby>尊'''（{{jpn|j = 小碓尊|hg = おうす の みこと|rm = Woutsu no mikoto|kyu = |hgt = をうす の みこと}}），號'''日本武尊'''（{{jpn|j = 日本武尊|hg = やまとたけ の みこと|rm = Yamatotake no mikoto|kyu = |hgt = }}<ref>{{Cite book|author = 舍人親王|authorlink = 舍人親王|title = 日本書紀|location = |publisher = |date = 1669年|accessdate = 2019年2月7日|url = https://kotenseki.nijl.ac.jp/biblio/200005037/viewer/182}}</ref><ref>{{Cite book|author = 太安萬侶|authorlink = 太安萬侶|title = 古事記|location = |publisher = [[前川茂右衞門|前川茂右衞門]]|date = 1644年|accessdate = 2019年2月7日|url = http://dl.ndl.go.jp/info:ndljp/pid/2538184/94}}</ref>{{Refnest|group="註"|一說<ref>{{Cite book|author = {{link-ja|伴信友}}|others = |title = 比古婆衣（《伴信友全集. 第4》所収）|location = [[東京|東京]]|publisher = {{link-ja|國書刊行會 (1905-1922)|国書刊行会 (1905-1922)|國書刊行會}}|date = 1909年|accessdate = 2019年4月15日|url = http://dl.ndl.go.jp/info:ndljp/pid/991315/33}}</ref>爲「{{jpn|j = |hg = やまとたける の みこと|rm = Yamatotakelu no mikoto|kyu = |hgt = |link = no}}」。此一說非也<ref>{{Cite book|author = 敷田年治|authorlink = 敷田年治|others = |title = 中臣宮処氏本系帳考証. 下巻|location = [[東京|東京]]|publisher = [[玄同会|玄同会]]|date = 1895年12月|accessdate = 2019年4月15日|url = http://dl.ndl.go.jp/info:ndljp/pid/780487/7}}</ref>。}}，倭建命<ref>《[[古事記|古事記]]》中的稱呼。</ref>，{{bd|72年||113年|}}），是[[日本|日本]][[古墳時代|古墳時代]]的人物。《[[常陸國風土記|常陸國風土記]]》作'''倭武天皇'''。傳說他力大無窮，善用智謀，於[[景行天皇|景行天皇]]期間東征西討，為[[倭王權|倭王權]]開疆擴土。最後雖英年早逝，而無繼承皇位，但子嗣為今日[[日本天皇|天皇]]之直系祖先。

==生平與傳說==
[[File:Emperor_family_tree8-15.png|right]]
[[File:Yamato_Takeru(bronze_statue,Osaka)01.jpg|200px]][[堺市|堺市]][[大鳥大社|大鳥大社]]的日本武尊像]]
根據《[[日本書紀|日本書紀]]》記載，[[景行天皇|景行天皇]]與皇后[[稻日大郎姬|稻日大郎姬]]生有一對雙胞胎，兄名[[大碓皇子|大碓皇子]]（《[[古事記|古事記]]》作大碓命），弟名小碓尊（《[[古事記|古事記]]》作小碓命），其中小碓尊身材魁梧，能獨立扛起一座鼎。

據《[[古事記|古事記]]》記載，小碓尊十六歲時，有次景行天皇問他為何其兄不與他共餐，並命他勸大碓皇子。然而過了五天，大碓皇子仍未出現。於是景行天皇再度詢問小碓尊其兄情況如何，小碓尊答說他已經勸說過了，天皇再問他如何勸說大碓皇子，小碓尊則說他趁皇子清早起床如廁時，將他抓住並折斷其頭部與四肢，然後用草蓆裹住屍身丟棄他處<sup id="fn_1_back">[[#fn_1|1]]</sup>。景行天皇聽完小碓尊一番話後，對於他的暴行大為驚恐，便令他遠離皇宮，前往西方討伐熊襲的川上梟帥。

===討伐[[熊襲|熊襲]]<sup id="fn_2_back">[[#fn_2|2]]</sup>===
小碓尊受命後，打聽到[[美濃國|美濃國]]有神射手[[弟彥公|弟彥公]]，於是派人找他，並將劍藏入懷中後出發執行詔命。

到了川上梟帥家，發現他家外剛建好三層房舍，並準備大開筵席慶祝。於是小碓尊趁人多混雜時混到其旁邊，並在筵席當天假扮成少女，隨其他侍女進入兩兄弟家中。入室後，川上梟帥看上扮成少女的小碓尊，便將其帶到房間，陪他宴樂。當川上梟帥快醉倒時，小碓尊趁勢取出懷裡的短劍，朝他胸口刺入。

這時瀕死的川上梟帥請求小碓尊讓他說完遺言，小碓尊同意後，川上梟帥便問誰令他是誰，小碓尊答說：「我是大足彥天皇的兒子，叫做日本童男。」熊建聽完後，嘆說西國無人能比兩兄弟，但[[日本|大倭國]]居然有人比我們兄弟強壯，於是贈與小碓尊「'''日本武皇子'''」（《[[古事記|古事記]]》作'''倭建御子'''）。從此，小碓尊被稱作'''日本武尊'''（《[[古事記|古事記]]》作'''倭建命'''）。

日本武尊殺死川上梟帥後，命弟彥公斬殺餘黨，即返回京城。途中，山神、河神與穴戶神等神祇與他同行拜見天皇。

===擊殺出雲建<sup id="fn_3_back">[[#fn_3|3]]</sup>===
根據《[[古事記|古事記]]》記載，日本武尊返京前經過[[出雲國|出雲國]]，打算殺掉出雲建。於是，他蓄意與後者結交，並將身上配置的換成假刀，然後利用兩人在肥河洗頭時，向出雲建提議換刀。之後，日本武尊故意提出比劍的建議，於是兩人便用對手的刀進行比試。由於出雲建佩帶的是對手的假刀，因此比試時拔不出來，這時日本武尊拔出出雲建的刀，一刀砍死了對手。殺了出雲建後，日本武尊唱歌悲嘆他的愚蠢，並於回京後將一切事蹟稟報天皇。

===東征===
[[景行天皇|景行天皇]]四十年時，東國有[[蝦夷|蝦夷]]騷動，天皇便令日本武尊前去討伐，並命[[吉備武彥|吉備武彥]]與大伴武日連兩人與之隨行<sup id="fn_4_back">[[#fn_4|4]]</sup>。於是當年[[十月|十月]]，日本武尊動身往東，並先前往[[伊勢神宮|伊勢神宮]]參拜，期間，倭姬命將[[天叢雲劍|天叢雲劍]]交付給他<sup id="fn_5_back">[[#fn_5|5]]</sup>。之後，他行經[[尾張國|尾張國]]，借住宮簀媛（《[[古事記|古事記]]》作'''美夜受姬'''）家，兩人日久生情，決定返國時結婚。

隔年，日本武尊抵達[[駿河國|駿河國]]<sup id="fn_6_back">[[#fn_6|6]]</sup>，而當地有人騙他說此地生有許多麋鹿，而日本武尊也聽信他的話，跑到原野上尋找野獸。於是，這人便放火焚燒原野，打算趁機殺掉日本武尊。日本武尊發現情勢不妙，連忙拔出[[天叢雲劍|天叢雲劍]]，一邊以劍砍草開路一邊逃跑，最終才得以倖免<sup id="fn_7_back">[[#fn_7|7]]</sup>。從此，[[天叢雲劍|天叢雲劍]]又被稱作'''草薙劍'''。等到逃出後，日本武尊找到欺騙他的人之躲藏處，將其全部殺光。

之後，日本武尊來到[[相模國|相模國]]，並搭船前往[[上總國|上總國]]。當船行經半路時，突然刮起暴風，使得船隻飄蕩而無法前行。這時武尊之妻[[弟橘媛|弟橘媛]]說這是因為海神想打翻武尊搭乘船，而她願意跳入海中為他犧牲。於是[[弟橘媛|弟橘媛]]跳入海中，暴風立刻停止，船隻才得以靠岸。而後，日本武尊從[[上總國|上總國]]進入[[陸奧國|陸奧國]]，並將大鏡子掛在船上，由海路進入[[蝦夷|蝦夷]]境內。這時[[蝦夷|蝦夷]]的領袖島津神、[[國神|國津神]]從海岸上眺望日本武尊的船隻，被他的氣勢震懾到，於是拋棄武器前去拜見他，並問他是誰，日本武尊便說：「我是當今人神的兒子！」於是蝦夷人全部投降，日本武尊則赦免他們的罪行。

平定蝦夷後，日本武尊輾轉來到[[甲斐國|甲斐國]]，在此宴樂時，因舉蠟燭的侍者答應妥帖，便封他為東國國造。之後又北上至[[上野國|上野國]]，並西行至碓日阪。在此，他與吉備武彥分道，吉備武彥前去越地，以考察當地地理環境與人民生活，日本武尊則前往信濃。信濃國群山遍佈，日本武尊只能穿梭其中，靠打獵果腹。某天，山神化身成白鹿出現在日本武尊眼前，他立即獵捕該鹿，這時他忽然迷路，無法走出山林<sup id="fn_8_back">[[#fn_8|8]]</sup>。此時出現一頭白狗帶路，才得以走出山間，與吉備武彥在[[美濃國|美濃國]]會合。之後在[[尾張國|尾張國]]停留一個多月，與宮簀媛結婚。

而後，日本武尊聽說[[近江國|近江國]]的五十葺山有荒神，於是再度徒步離家前去挑戰。途中遇到一條大蛇<sup id="fn_9_back">[[#fn_9|9]]</sup>，日本武尊以為是主神的使者，便跨過大蛇繼續前進，然而這條大蛇實際上是主神的化身。主神見他如此輕視，便召集天空雲朵匯集，降下冰雨，使他難以前行。日本武尊在大雨中勉強逃離山中，在山下飲泉水後才恢復神智，但這時他的身體已十分虛弱了。最後他回到[[尾張國|尾張國]]，轉而到[[伊勢國|伊勢國]]，將蝦夷俘虜獻入[[伊勢神宮|伊勢神宮]]，不久便於褒野去世，得年三十歲。
