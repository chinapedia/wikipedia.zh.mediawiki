'''日本滑雪技术锦标赛'''，或'''全日本滑雪技术锦标赛'''，由[[日本滑雪协会|日本滑雪协会]]（SAJ）主办的全日本[[滑雪|滑雪]]技术比赛，简称为'''技选赛'''。

== 概述 ==
此赛事，凡获得SAJ1级的滑雪选手，可以报名参加，按地区代表队方式报名。与高山滑雪比赛不同，此赛事不以速度作为竞赛的标准，也不以难度系数标准来评定，而是在指定的雪道上，对参赛选手所滑出的弯型，如何使用与发挥雪板的性能等方面进行评估，然后打分<ref>{{Cite web |url=http://www.sohu.com/a/211283045_654928 |title=日本SAJ双板考级纪实 |access-date=2017-12-18  }}</ref>，将这些评估点合计作为比赛得分。虽然比赛不以速度作为评判标准，但为了滑出优美的曲线，选手通常都以极高的速度滑行，以展现自身的技术和进攻性。

此赛事从1964年开始，截至2019年3月，合计举办了56届。<ref>[http://www.ski-japan.or.jp/education/23021/ 第56届日本滑雪技术锦标赛报道]</ref>

== 赛制 ==
* 地形包括中级道，高级道，复合型坡道，机压雪道，不平整雪道，自然形成的蘑菇道。
* 选手要在以上雪道内按要求滑出大弯 小弯，以及变节奏滑行，选手们比的是技术实力的发挥。
* 每次滑行，由5位裁判给出得分，去掉最高分和最低分，剩下3名裁判的得分相加即为最后得分。<ref>[http://54gisen.info/junior/index_jr.html 日本滑雪技术锦标赛规则] {{Wayback|url=http://54gisen.info/junior/index_jr.html |date=20170817080039 }}</ref>
* 每个裁判初始成绩都是100分，每一次技术失误扣除至少1分，直至滑行结束，得到最后得分。
* 参加决赛的男选手限定为70名，女选手30名，5天的比赛中，在3种不同地形的雪道展开比赛，最终总分最高者胜出。

== 引入中国 ==
2016年3月，中国首届大众滑雪技术大奖赛在[[崇礼区|崇礼区]][[万龙滑雪场|万龙滑雪场]]举办，这次技术大赛赛制借鉴了[[韩国|韩国]]、[[日本|日本]]滑雪技术大赛安排，大赛裁判员由日韩为主体的专业人员和国内本土资深的滑雪人士担当。<ref>[http://sports.163.com/16/1122/15/C6G20D5H00050J9O.html 中国大众滑雪技术大赛为滑雪的普及注入活力]</ref>
