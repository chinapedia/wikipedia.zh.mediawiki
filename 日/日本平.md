'''日本平'''（{{jpn|j=日本平|hg=にほんだいら|rm=Nihondaira|f=y}}）是一個位于[[靜岡縣|靜岡縣]][[靜岡市|靜岡市]][[清水區_(日本)|清水區]]與[[駿河區|駿河區]]之間的的觀光景點。名稱由來於[[日本武尊|日本武尊]]（{{lang|ja|やまとたけるのみこと}}）傳說。

== 概要 ==
[[File:Mount_Fuji_and_Port_of_Shimizu.JPG|thumb]]及[[清水港|清水港]]]]
[[File:Shizuoka_Plain_Relief_Map,_SRTM-1.jpg|thumb]]
日本平是指[[駿河灣|駿河灣]]沿岸附近的有度山（{{lang|ja|うどやま}}）山頂及附近一帶，可在此越過駿河灣看見[[富士山|富士山]]及[[伊豆半島|伊豆半島]]，向北可望見[[赤石山脈|赤石山脈]]（南阿爾卑斯山）<ref name="sudo">{{cite book |author={{lang|ja|須藤英一}}|script-title=ja:《新・日本百名道》 |year=2013 |publisher={{lang|ja|大泉書店}} |location=東京都 |isbn=978-4-278-04113-2|language=ja}}</ref>{{rp|111}}。在山頂可一覽清水區的市區和[[清水港|清水港]]，也是夜景觀賞勝地之一。1950年，在毎日新聞社主辦的觀光地百選中，日本平獲得平原部門的首位。1959年，日本平被日本政府指定為名勝<ref>{{cite web |url=http://kunishitei.bunka.go.jp/bsys/maindetails.asp |script-title=ja:日本平 |accessdate=2017-07-15|publisher={{lang|ja|国指定文化財等データベース}} |language=ja}}</ref>。在1980年的日本觀光地百選競賽中亦獲得第一<ref>{{cite web |url=http://www.city.shizuoka.jp/000097808.pdf |script-title=ja:日本平公園基本計画概要版（2章） |accessdate=2017-07-15|publisher={{lang|ja|静岡市}} |language=ja}}</ref>。2016年，日本平被認定為「日本夜景遺產」<ref>{{cite web |url=http://www.at-s.com/news/article/topics/shizuoka/267505.html |script-title=ja:静岡・日本平「夜景遺産」に　霊峰望むパノラマ評価 |accessdate=2017-07-15 |publisher={{lang|ja|静岡新聞}} |language=ja |deadurl=yes |archiveurl=https://web.archive.org/web/20160812063300/http://www.at-s.com/news/article/topics/shizuoka/267505.html |archivedate=2016-08-12 }}</ref>。

日本平最高地點有度山山頂標高307米，山麓多闊葉樹林，廣佈[[茶|茶]]園及[[蜜柑|蜜柑]]園。以有度山為中心的丘陵是通過[[彎曲_(力學)|彎曲]]而形成，在過去10萬年期間隆起了300米，在日本是頗為劇烈的地殼變動<ref>{{cite web |url=http://sk01.ed.shizuoka.ac.jp/koyama/public_html/etc/opinion/jihyo6.html |script-title=ja:静岡平野の地下構造探査を |accessdate=2017-07-15|publisher={{lang|ja|静岡大学}} |language=ja}}</ref>。丘陵的沿海一側因海浪和沿岸流的侵蝕地形險峻。流出的土砂隨沿岸流移動，是形成{{link-ja|三保半島|三保半島}}的重要原因。丘陵西側至北緣有活斷層<ref>{{cite web |url=https://gbank.gsj.jp/activefault/cgi-bin/segment_param.cgi?SearchTYPE=&fval_type1=298-01&segment_id=298-01&topic_list=2&search_mode=0 |script-title=ja:活動セグメントの概要とパラメータ代表値 |accessdate=2017-07-15|publisher={{lang|ja|産業技術総合研究所 活断層データベース}} |language=ja}}</ref>。有度山這一地名取自這裡曾設有[[駿河國|駿河國]]{{link-ja|有度郡|有度郡}}。

== 設施、古蹟 ==
丘陵南部的久能山（標高216米）山麓有[[德川家康|德川家康]]的靈廟[[久能山東照宮|久能山東照宮]]，其社殿是[[江戶時代|江戶時代]]初期建築的代表，被選入[[日本國寶|日本國寶]]<ref>{{cite web |url=https://www.toshogu.or.jp/about/shinto_shrine.php |script-title=ja:国宝御社殿 |accessdate=2017-07-15|publisher={{lang|ja|久能山東照宮}} |language=ja}}</ref>。{{link-ja|日本平纜車|日本平ロープウェイ}}連接日本平山頂和東照宮兩地。另外，由於日本童話{{link-ja|紅鞋子|赤い靴}}主角女生和其母親出身於清水區宮加三（舊不二見村），山頂有其母子像。日本平山頂的「山頂吟望台」和「東展望台」則是一覽富士山和清水港等地的瞭望地。山頂附近有{{link-ja|日本平酒店|日本平ホテル}}<ref>{{cite web |url=http://www.ndhl.jp/outline |script-title=ja:施設概要 |accessdate=2017-07-15|publisher={{lang|ja|日本平ホテル}} |language=ja}}</ref>、日本平高爾夫俱樂部、梅園等設施，並且還有靜岡縣內各電視台共同建設的{{link-ja|日本平數位塔|日本平デジタルタワー}}<ref>{{cite web |url=https://www.pref.shizuoka.jp/bunka/bk-210/shisetu/documents/4kihonkeikaku.pdf |script-title=ja:日本平公園（名勝日本平）修正基本計画及び基本設計  |accessdate=2017-07-15|publisher={{lang|ja|静岡県}} |language=ja}}</ref>。山麓的駿河區一側有{{link-ja|日本平動物園|日本平動物園}}、靜岡縣舞台藝術公園、遊木之森、清水區一側有[[清水心跳|清水心跳]]的主場球場{{link-ja|靜岡市清水日本平運動公園球場|静岡市清水日本平運動公園球技場}}<ref>{{cite web |url=http://www.city.shizuoka.jp/000_007081.html |script-title=ja:清水日本平運動公園球技場・庭球場 |accessdate=2017-07-15|publisher={{lang|ja|静岡市}} |language=ja}}</ref>。

== 靜岡市道池田日本平線 ==
靜岡縣道路公社為發展觀光業，在日本平修建了東西方向的收費公路日本平公園公路（{{lang|ja|日本平パークウェイ}}），於1964年3月1日全線通車。該公路分為自靜岡一側的丘陵至日本平山頂的日本平公園公路，以及自清水至日本平山頂的「清水日本平公園公路」兩個部分<ref name="sudo"></ref>{{rp|111}}。公園一帶被指定為日本平・三保松原縣立自然公園<ref>{{cite book |author={{lang|ja|徳久球雄}}|script-title=ja:《コンサイス日本山名辞典》 |year=1992 |publisher={{lang|ja|三省堂}} |location=東京都 |isbn=4-385-15403-1|language=ja}}</ref>{{rp|398}}，是著名的瞭望茶園及富士山的地點<ref name="sudo"></ref>{{rp|111}}。2004年4月1日，日本平公園公路成為免費公路，改為靜岡市道池田日本平線<ref>{{cite web |url=https://www.pref.shizuoka.jp/soumu/so-030a/documents/07shiryou7-2.pdf |script-title=ja:資料 7-2 |accessdate=2017-07-15|publisher={{lang|ja|静岡県道路公社}} |language=ja}}</ref>。違法競走型暴走族於夜間聚集在日本平公園公路並進行[[漂移|漂移]]是當地的一大問題。靜岡縣警察亦對這一違法行為進行取締。另外，據當時的道路運送法，日本平公園公路在海上收費公路時禁止排量125cc以下的摩托車、自行車、步行者通行。在改為免費公路後，依據靜岡縣公安委員會的方針，日本平公園公路仍禁止自行車和步行者通行。
