《'''日本最漫長的一天'''》（{{jpn|j=日本のいちばん長い日|hg=にほんのいちばんながいひ|rm=Nippon no ichibon nagai hi|f=h}}）是[[第二次世界大戰|戰後]]七十年之際，由導演原田眞人改編自[[半藤一利|半藤一利]]原著《日本最漫長的一天》之[[電影|電影]]作品，於2015年8月8日在[[日本|日本]]上映。本片是繼1967年由[[岡本喜八|岡本喜八]]拍攝《[[日本最長的一日|日本最長的一日]]》後，再次以[[宮城事件|宮城事件]]為背景製作的同名電影作品。

==故事概要==

1945年[[第二次世界大戰|第二次世界大戰]]末期[[軸心國|軸心國]]成員之一的[[大日本帝國|大日本帝國]]在海外勢力節節敗退，在日本本土也面臨著[[東京大空襲|東京大空襲]]、[[長崎市原子彈爆炸|長崎]]和[[廣島市原子彈爆炸|廣島]]遭受[[美國|美國]][[廣島與長崎原子彈爆炸|毀滅性轟炸]]。[[內閣總理大臣|內閣總理大臣]][[鈴木貫太郎|鈴木貫太郎]]和政府內部軍隊商討如何終止戰爭，希望能夠儘早結束災難的[[昭和天皇|昭和天皇]]在[[御前會議|御前會議]]表明接受[[波茨坦宣言|波茨坦條約]]<ref>{{Cite web |url = http://www.plurk.com/p/kw7pjl |title = 日本最長的一日 |author = 許哲寧 |date = 2015年4月14日 |publisher = Plurk |language = zh−tw |accessdate = 2016年1月19日 |quote = 究竟日本是如何果斷做出決定，在二次大戰之末，無條件接受「波茨坦宣言」投降？八月即將上映的電影《日本のいちばん長い日》故事描述的正是這個不為人知的背景：1945年8月15日的前一天「日本最長的一日」，天皇、內閣、與將領們的激烈拉鋸。 }}</ref>，並請[[日本放送協會|日本放送協會]]製作昭和天皇親口錄製的[[終戰詔書|終戰詔書]]全文的[[黑膠唱片|黑膠唱片]]，打算在1945年8月15日正午播放。不料15日凌晨，為阻止這段[[玉音放送|玉音放送]]的陸軍軍官，發布軍事動員令控制[[宮城事件|皇居]]；叛亂最終被平息，受降的錄音準時播送至日本各地，正式宣告[[日本投降|結束戰爭]]。所謂「最漫長的一日」，就是在昭和天皇宣稱將要無條件投降的那一刻起、到[[宮城事件|宮城事件]]叛軍被鎮壓之後的24小時<ref>{{Cite web |url = http://thehousenewsbloggers.net/2015/08/24/【終戰70週年系列】《日本最長的一天》：窮途末路/ |title = 《日本最長的一天》：窮途末路的人性掙扎 |author = 林兆彬 |date = 2015年8月24日 |publisher = 主場博客 |language = zh-hk |accessdate = 2016年1月19日 |quote = 故事改編自昭和史研究學者半藤一利的小說，是繼1967年之後，第二次被搬上大銀幕。講述在太平洋戰爭末期（1945年），英、美、中三國所發起的波茨坦宣言要求日本無條件投降，並表示如果日本不肯投降的話，盟軍將進攻日本。所謂的「最長的一日」，是指昭和天皇向鈴木貫太郎內閣宣告日本要無條件投降，引起日本陸軍少壯派軍官的不滿。在1945年8月14至15日，他們企圖發動政變，奪走天皇宣告無條件投降的錄音唱片「天皇玉音」，阻止日本投降。 }}</ref>。
