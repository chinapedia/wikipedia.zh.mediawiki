'''日本皇室'''（{{jpn|j=皇室|hg=こうしつ|rm=Kōshitsu|f=h}}）包括[[日本天皇|天皇]]及[[皇族|皇族]]{{NoteTag|依現行《[[皇室典範|皇室典範]]》，「皇族」一詞指天皇的親族，而不包含天皇本人。}}，具體的構成包括天皇與{{link-ja|內廷皇族|内廷皇族}}組成的「內廷」、與其他男性皇族及其[[家庭|家庭]]組成的數個[[宮家|宮家]]。皇室成員没有[[姓氏|姓氏]]，無[[选举权|选举权]]，不受日本的[[户籍|户籍]][[法律|法律]]管理，而有专门的《[[皇室典範|皇室典範]]》規範皇室的運作。

== 概述 ==
{{日本政治}}
===姓名、称号===
为了维持日本皇室自立國以來便綿延不斷的形象，天皇與皇族均只有[[名字|名字]]而沒有[[姓氏|姓氏]]。过去几个世纪以来，男子的名字以“仁”（{{lang|ja|ひと}}，hito）结尾，女子的名字以“子”（{{lang|ja|こ}}，ko）结尾。历史上有少量例外。

皇室成員皆有稱號。一般來說，直系三代以內皇子、皇孙子称“[[亲王|亲王]]”，皇女称“[[內亲王|內亲王]]”；直系三代之外皇孙称“[[王_(日本)|王]]”，皇孙女称“[[女王_(日本)|女王]]”。在皇室財力較弱的時例外，曾經有過只有天皇嫡子女才有親王、內親王之位號，庶子女則封為王和女王。从前“女王”的称号只可到直系第六代皇孙女，[[明治天皇|明治天皇]]之后这一限制取消。另外还有“皇太子”、“皇太孙”、“皇后”、“皇太后”、“太皇太后”、“王妃”等称呼，意义与[[中文|中文]]的相同。

天皇的子嗣誕生後，在正式命名之前會先有一個「御稱號」（即[[幼名|幼名]]），如[[昭和天皇|昭和天皇]]从前称「迪宫」、[[日本|日本]][[今上天皇|今上天皇]][[明仁|明仁]]从前称「继宫」。除了皇太子之外，其他皇子在成年或成婚后，会經[[宮內廳|宮內廳]]批准建立一个[[宮家|宮家]]，並由天皇下賜一個「[[宮號|宮號]]」，大多以「○○宮」/「○宮」形式命名。

明治維新後，天皇、皇后、太上天皇、皇太后、太皇太后敬称“陛下”，其餘皇族敬称“殿下”。天皇去世为“崩御”（[[駕崩|駕崩]]），其余的皇族去世为“[[薨|薨去]]”。

===身份===
所有皇室身份都是男系继承的，即皇族的子女、孙子女是皇族，而外孙子女不是。明治維新後規定，女性皇室成员（內亲王、女王）下嫁平民後即脫離皇籍，成为平民。女性皇族成员必須嫁给男性皇族成员，她的后代才会是皇族。

取得皇室身份的途径包括在天皇、亲王、王的家庭中出生，或者嫁给天皇、亲王、王。至於脫離皇籍，在古代由天皇決定[[臣籍降下|臣籍降下]]，[[明治維新|明治維新]]後脫離皇籍的途径有成年后自愿脱离、女性成员和非皇族结婚、和皇族成员离婚等。

===繼承權===
日本现行的《皇室典範》规定，女性皇族無法繼承皇位，成年後也不像男性皇族能創設[[宮家|宮家]]，而降嫁後便自動脫離皇籍成為平民。

但在現行的《皇室典範》實施前，女性皇族雖然無法創設宮家，但降嫁後不需脫離皇籍（但若嫁與平民，其後代不為皇族），也能在無適任男性皇族繼位的情況下，由女性皇族繼承皇位成為天皇，甚至是天皇的遗孀也可以成为天皇，但她必须自己本身就是皇族，而且不曾和天皇以外的人生下子女。简言之，女性天皇只可以是男性天皇繼位之间的过渡。有史可查的10任8位女天皇，自身都是皇族，並在继位之后没有生育，且传位於男性皇族，其中僅奈良時代的[[元明天皇|元明天皇]]除外，她在位8年後讓位於其女冰高皇女，是為[[元正天皇|元正天皇]]。

由于日本皇室从1965年到2006年间没有男子诞生，天皇皇位在延續幾代之後面將临後繼無人的可能，因此日本国会一度考虑修改皇室典範，允许女性皇族繼位，並容許其和平民结婚传位给后代，或者恢复二战之后废除的宫家成員之后代的皇族身份。但在天皇次子[[秋篠宮文仁親王|秋篠宮文仁親王]]的妻子[[秋篠宮妃紀子|秋篠宮妃紀子]]於2006年9月6日生下幼子[[悠仁親王|悠仁親王]]之后，修改《皇室典範》的程序被搁置。

== 成員 ==
现在的所有皇室成员都为[[大正天皇|大正天皇]]和貞明皇后的后代或后代之女性配偶。

== 日常生活 ==
===宫內厅===
日本皇室的管理机构是[[宫內厅|宫內厅]]。宫內厅有一千多名职员，其中很多高官出身[[華族_(日本)|华族]]。天皇和皇太子及其家族成員的生活费被称为“內廷费”，近年是大约三亿[[日圓|日圓]]。其他皇室成员的生活费被称作“皇族费”，每名成年男性成员每年获得皇族费2700万日圓，而秋篠宮文仁親王一家每年能得到皇族费4600万日圓，得子之后会更多。皇室的收入来源主要是[[股票|股票]]投资。天皇的[[皇居|皇居]]中有160名佣人。
