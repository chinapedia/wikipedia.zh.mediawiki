'''日本獼猴'''（[[學名|學名]]：{{lang|la|'''''Macaca fuscata'''''}}），也叫'''雪猴'''，是生活在[[日本|日本]]北部的一種[[獼猴|獼猴]]。它們是世界上生活地區最北（最北達到[[本州|本州]][[青森縣|青森縣]]北部[[下北半島|下北半島]]<ref name="fn1"/><ref name="fn5"/>）的非人類[[靈長目|靈長目]]動物。日本獼猴的毛色為灰褐色，臉部、四肢和下腹部為紅色，尾巴很短。共有兩個亞種：
* 指名亞種 ''Macaca fuscata fuscata''
* 屋久島亞種 ''Macaca fuscata yakui''

== 形態 ==
日本獼猴的[[体長|体長]]有{{convert|47|-|60|cm}}<ref name="fn5"/>。公猴尾長{{convert|7|-|11|cm}}、雌猴尾長{{convert|6|-|11|cm}}<ref name="fn5"/>。[[体重|体重]]雄猴{{convert|6|-|18|kg}}、雌猴{{convert|6|-|14|kg}}<ref name="fn5"/>。在本州東北地方及中部地方的山岳部有大型的個体群<ref name="fn5"/>。短尾<ref name="fn5"/>，雖然在寒冷地區生長，身體沒有長密的体毛，反而傾向於温暖地的短薄体毛<ref name="fn1"/>。不過，幼獸的体毛相對較為密，成長後體毛密度會減低<ref name="fn1"/>。背部毛色赤褐色或褐色，腹面毛色灰色<ref name="fn1"/>。種小名''fuscata''的意思就是「有暗色」的意思<ref name="fn3"/>。

臉部和屁股等裸露的皮膚紅色<ref name="fn2"/><ref name="fn5"/>。

公猴犬齒發達<ref name="fn5"/>。屋久島亞種的体毛長<ref name="fn1"/>，毛色暗褐<ref name="fn2"/><ref name="fn4"/>。頭蓋屬小型<ref name="fn1"/>。眼窩縱長，間幅狹窄<ref name="fn1"/>。

== 分佈範圍和食性 ==
日本獼猴在白天活動，大多數時間都在森林中，範圍包括落葉林、闊葉林和常綠林。它們以種子、根莖、花蕾、水果、[[無脊椎動物|無脊椎動物]]、漿果、樹葉、鳥蛋、蘑菇和穀物為食。

== 習性與繁殖 ==
日本獼猴一般組成一個規模為20至100頭的群體，分成幾個家庭，一般而言，母猴與公猴的比例約為3比1。母猴的地位有明顯等級制，公猴在種群中則比較游離。

日本獼猴可以在{{convert|-15|°C}}的嚴酷環境中生存，這在靈長目動物中是非常少見的。为了能在这样低温的严酷环境中生存，日本猕猴进化出了集体泡温泉的习性，但只有最高等級的雌性和幼崽才能入內。[[探索频道|探索频道]]的《[[地球变种|地球变种]]》纪录片摄制组拍摄到部分日本猕猴为了拾取温泉底部的食物，会憋气潜水1分钟左右的新习性。

在交配季節，母猴會與10頭左右的公猴進行[[交配|交配]]的嘗試。日本獼猴的孕期為173天，每胎只產一仔，幼子出生時的重量約為{{convert|500|g}}。平均壽命為30年。

拉斐拉·S·C·竹下(Rafaela S.C. Takeshita)和她的京都大學同事在《靈長目動物》期刊上發表了成果。正如所料，在猴子泡澡期間，壓力水平下降。另一個能表明泡澡對獼猴重要性的地方是，地位較高的母猴在池水裡的時間更長。<ref>{{Cite news|url=https://cn.nytimes.com/science/20180412/japan-monkeys-hot-springs-stress/zh-hant/|title=日本雪猴如何減壓？泡溫泉|date=2018-04-12|newspaper=紐約時報中文網|accessdate=2018-08-30|language=zh}}</ref>

==日本純化 ==
