'''行事曆'''（Calander，前稱：iCal）是[[苹果电脑公司|苹果电脑公司]]出品的一个[[日程管理|日程管理]][[应用程序|应用程序]]，运行在[[Mac_OS_X|Mac OS X]] [[操作系统|操作系统]]。 行事曆是第一个提供多日历支持、并可以在[[WebDAV|WebDAV]]服务器上发布／订阅日历的日程管理程序。

在2002年9月10日发布当初是一个[[Mac_OS_X_v10.2|Mac OS X v10.2]]系统的免费下载程序。之后被捆绑在[[Mac_OS_X_v10.3|Mac OS X v10.3]]操作系统中。iCal 2 也是作为[[Mac_OS_X_v10.4|Mac OS X v10.4]]的一部分发布的。

== 功能 ==
*跟踪排程、生日、提醒事項、活动和[[约会|约会]]，支持多日历浏览方式（如 "家用", "工作", "孩子"用的日历），不同的日历用不同的色彩标识，可以快速区别冲突和富余的时间。
*与[[MobileMe|MobileMe]]（即早期的[[.Mac|.Mac]]）和[[iCloud|iCloud]]集成，用户可以通过[[Internet|Internet]]共享日历。用户也可以通过[[WebDAV|WebDAV]]协议共享，如与Google calendar实现同步。
*用户可以[[定制|定制]]其他日历用户，以保持与朋友、同事的同步，协调运动时间和电视节目时间等等。
*可以通过屏幕、电子邮件、短消息等进行活动预告和提醒。还有叫[http://www.benkazez.com/icalevents/ iCal Events]的第三方[[Dashboard|Dashboard]] widget让用户在Dashboard中进行时间提醒。
*可以与[[Things|Things]]等待办事项管理软件进行同步，进行待办事项管理。

== 逸事 ==
行事曆启动后，[[图标|图标]]显示当前日期；而OS X启动时候图标默认显示的是7月17日——2002年的这一天行事曆在[[纽约|纽约]]的[[Macworld_Expo|Macworld Expo]]面市。因此一些Mac用户称7月17日为'iCal日'。
