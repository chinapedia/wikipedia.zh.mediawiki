日語從其他語言中吸收了大量辭彙，來自[[古代漢語|古代漢語]]的稱為「[[漢字詞|漢語]]」，來自其他語言的稱為「外來語」。關於來自[[法語|法語]]、[[德語|德語]]、[[荷蘭語|荷蘭語]]、[[葡萄牙語|葡萄牙語]]的辭彙，參見本篇最後所列的條目。本篇敘述日語中的[[義大利語|義大利語]]、[[俄語|俄語]]、[[阿拉伯語|阿拉伯語]]和[[西班牙語|西班牙語]]借詞。

== 意大利語 ==
[[意大利語|意大利語]]為[[日語|日語]]借用的辭彙主要有[[音階|音階名]]與[[演奏記號|演奏記號]]（{{lang|ja|フォルティッシモ}}、{{lang|ja|エスプレッシーヴォ}}、{{lang|ja|カンタービレ}}等）等音樂用語，及一些[[意大利餐|意大利餐]]相關的詞語（如{{lang|ja|スパゲッティ}}、{{lang|ja|ティラ・ミ・ス}})。此外尚有部分汽車名、建築物名、[[時裝|時裝]]、[[足球|足球]]等相關的詞語。

== 俄語 ==
日語中的俄語借詞主要有[[十八世紀|十八世紀]]至[[十九世紀|十九世紀]]兩國間的交流所帶來的俄國文化相關的辭彙及1917年[[十月革命|十月革命]]後與[[社會主義|社會主義]]思想一起進入日本的辭彙。部分前一類辭彙在日常生活中使用（如{{lang|ja|イクラ}}等）。後一類多數僅作為描述蘇聯/俄羅斯特有的組織和現象的專有名詞使用（{{lang|ja|コルホーズ}}、{{lang|ja|ペレストロイカ}}等），也有少數詞語進入了一般使用，如{{lang|ja|コンビナート}}、{{lang|ja|ノルマ}}等。

{{lang|ru|в}}(/v/音) 舊譯「{{lang|ja|ウィ・ウ・ウェ・ウォ}}」，較新譯法為「{{lang|ja|ヴィ・ヴ・ヴェ・ヴォ}}」。但"{{lang|ru|ва}}"一般習慣性的譯為「{{lang|ja|ワ}}」:{{lang|ja|マリア・シャラポワ}}。

== 阿拉伯語 ==
阿拉伯語起源的辭彙多是通過其他語言如荷蘭語、英語等進入日語的。

如以下例詞：{{lang|ja|トタン}}（鍍鋅鐵皮，白鐵皮）、{{lang|ja|じょうろ}}（{{lang|ja|如雨露}}，噴壺）、{{lang|ja|コーヒー}}（咖啡）、{{lang|ja|ラケット}}（球拍）、{{lang|ja|シロップ}}（糖漿、果子露）、{{lang|ja|アルコール}}（酒精、乙醇）、{{lang|ja|アルカリ}}（堿、強堿）、{{lang|ja|ソーダ}}（蘇打、碳酸鈉、鈉鹽）、{{lang|ja|シャーベット}}（冰糕、果子露冰激凌）、{{lang|ja|チェス}}（國際象棋、西洋象棋）、{{lang|ja|チェック}}（最初原意是指方格花紋、格子、花格，而後因為英語借詞才有“支票”等含義）、{{lang|ja|ソファー}}（沙發）、{{lang|ja|モンスーン}}（季風）、{{lang|ja|アベレージ}}（平均、平均值）、{{lang|ja|ジャケット}}（夾克、短外套、短上衣）、{{lang|ja|ゼロ}}（零）、{{lang|ja|タリフ}}（関税、関税表）、{{lang|ja|リスク}}（風險）、{{lang|ja|アルゴリズム}}（算法、處理方式）、{{lang|ja|アドベ}}（砂、砂質粘土）、{{lang|ja|タマリンド}}（羅望子）、{{lang|ja|コットン}}（棉花、棉紗、棉布）、{{lang|ja|シェリフ}}（貴族、出身高貴）、{{lang|ja|サフラン}}（番紅花、藏紅花）、{{lang|ja|アドミラル}}（海軍上將、提督）、{{lang|ja|センナ}}（番瀉樹）、{{lang|ja|シュガー}}（砂糖、白糖）等。

科學術語亦有相當數量阿拉伯語起源的辭彙，特別是[[化學|化學]]、[[數學|數學]]、[[天文學|天文學]]等。[[中世紀|中世紀]]時[[阿拉伯帝國|阿拉伯帝國]]的科學較興盛，向[[歐洲|歐洲]]輸出了知識及辭彙。
