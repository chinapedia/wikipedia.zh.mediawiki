'''日落隧道'''（{{lang-en|Sunset Tunnel}}）是[[美國|美國]][[加州|加州]][[舊金山|舊金山]]一條長度4,232呎（1,290米）之[[輕軌|輕軌]]隧道。此隧道經過{{link-en|舊金山麗景公園|Buena Vista Park}}的陡峭丘陵，由[[舊金山輕軌|舊金山輕軌]][[舊金山輕軌N線|N線]]使用。

此隧道開通於1928年10月21日，由當時舊金山市長James Rolph主持通車典禮。<ref name="sfgenealogy">{{cite web
 |url = http://www.sfgenealogy.com/sf/history/hgtun.htm
 |title = San Francisco History - City's Tunnels
 |accessdate = July 14, 2008
 |last = Wallace
 |first = Kevin
 |date = March 27, 1949
 |publisher = ''[[San_Francisco_Chronicle|San Francisco Chronicle]]''
 }}</ref> 隧道的東側入口位在杜伯斯大街（Duboce Avenue）與諾伊街（Noe street）路口，鄰近{{link-en|杜伯斯公園|Duboce Park}}；西側入口則在卡爾街（Carl Street）與柯爾街（Cole Street）路口。

隧道的使用僅限於[[舊金山輕軌|舊金山輕軌]]列車，然而隧道並未完整地隔離保護，[[塗鴉|塗鴉]]等[[破壞|破壞]]行為時常可見。此外汽車駕駛將車輛開入隧道的事件也時有耳聞。<ref>{{cite web |url=http://www.njudahchronicles.com/2008/04/no_really_it_happened_again_car_enters_m.html |title=存档副本 |accessdate=2015-02-22 |deadurl=yes |archiveurl=https://web.archive.org/web/20130908234743/http://www.njudahchronicles.com/2008/04/no_really_it_happened_again_car_enters_m.html |archivedate=2013-09-08 }}</ref>
