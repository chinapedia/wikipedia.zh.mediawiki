'''日本漫畫家協會獎'''是社團法人{{tsl|ja|日本漫画家協会|日本漫畫家協會}}為了提升日本漫畫界的發展及表彰優秀作品，而於1972年創立的漫畫獎<ref name="官網01">{{Cite web |url = http://www.nihonmangakakyokai.or.jp/?tbl=about&id=14 |title = 日本漫画家協会賞とは |author = 日本漫畫家協會 |date = 2014-11-13 |language = ja |accessdate = 2016-05-23}}</ref><ref name="kuboc">{{Cite web |url = http://kubocchannel.blog.jp/archives/59757489.html |title = 第４５回日本漫画家協会賞は「鼻紙写楽」が大賞！だったお話♪ |date = 2016-05-10 |language = ja |accessdate = 2016-05-23}}</ref><ref name="巴哈01">{{Cite web |url = http://home.gamer.com.tw/creationDetail.php?sn=1605239 |title = 日本漫畫家協會賞各獎項結果發表 |date = 2012-05-14 |publisher = 巴哈姆特電玩資訊站 |accessdate = 2016-05-23}}</ref>。

這個獎項的特點是不論專業、業餘、國籍、年齡、性別，純粹以使用不同手法所創作並發表的各類型漫畫作品為對象，並經由10至12位評選委員嚴格考核，從而選出在一年中的優秀作品<ref name="官網01"/>。評選委員包括漫畫家、漫畫評論家及漫畫關係者<ref name="kuboc"/><ref name="巴哈01"/>。

起初，此獎項只設立了大獎、優秀獎及特別獎<ref name="官網01"/>。其後，在1990年協會應[[文部省|文部省]]（現時為[[文部科學省|文部科學省]]）的請求，增設文部大臣獎（現時為文部科學大臣獎）並直至現時<ref name="官網01"/>。此外，獲獎者所得的獎金是從協會會員的會費中撥出的<ref name="kuboc"/>。
