'''日本国际广播电台'''（{{lang-ja|NHKワールド ラジオ日本}}；{{lang-en|NHK World Radio Japan}}）是[[日本|日本]]的一個[[廣播電台|廣播電台]]，屬[[日本广播协会|日本广播协会]]（NHK）对日本國外[[广播|广播]]业务（[[NHK日本国际传媒|NHK日本国际传媒]]）的一部分。

== 概要 ==
1935年（[[昭和|昭和]]10年）6月1日日本[[国际广播|国际广播]]开播，当时称为“日本海外广播”，使用[[英语|英语]]和[[日语|日语]]向[[北美|北美]]西部和[[夏威夷|夏威夷]]地区播出了1小时的节目。现在的[[日本广播协会|日本广播协会]]的前身——社团法人日本广播协会——在海外广播刚开始时，使用的呼号为「東京廣播電台」（{{lang|ja|ラジオ・トウキョウ}}；Radio Tokyo），当时并没有定为正式台呼。直到1941年（昭和16年）才将其确定为正式的台呼。

社团法人日本广播协会的对外国际广播，在[[二战|二战]]结束后的1945年（昭和20年）9月根据[[驻日盟军总司令|驻日盟军总司令部]]的命令而停止运作。1950年（昭和25年），《[[廣播法_(日本)|广播法]]》（{{lang|ja|放送法}}）实施，社团法人日本广播协会遭解散，其业务被新成立的特殊法人「日本广播协会」（NHK）所继承。

1952年2月（昭和27年）2月，开始以「日本廣播電台」（{{lang|ja|ラジオ日本}}；Radio Japan）为台呼重新开始对外進行国际广播。1996年（[[平成|平成]]8年）NHK进行机构改革，其海外广播业务被划分由「NHK环球广播网 日本国际广播电台」（{{lang|ja|NHKワールド・ラジオ日本}}）負責，以18种不同的[[语言|语言]]进行对外广播。

== 现况 ==
NHK环球广播网的广播业务分为兩類，除了使用[[日语|日语]]和[[英语|英语]]对全球进行综合广播外，也在不同地区进行相应、针对性的区域广播，其对外日语广播节目大多与[[NHK广播第1频率|NHK广播第1频率]]同步<ref>{{lang|ja|[http://www3.nhk.or.jp/nhkworld/resources/brochure/pdf/rj_programs_ja.pdf 平成２８年度 ＮＨＫワールド・ラジオ日本　放送番組時刻表]}}</ref>。NHK环球广播网同时通过[[因特网|因特网]]发布多语种的[[新闻|新闻]]。

NHK的国际广播节目，由于[[短波|短波]]信号反射的缘故，在日本国内大部地区反而有收听上的困难，但在部分地区仍可接收到信号。部分听众是通过接收日本国外的中继站发射信号来收听，故在日本国内也拥有忠实听众。由於主要是對日本國外進行播放工作，日本國際廣播電台並不像其他日本國內的廣播電台般向国内听众發行寄送[[受信確認證|受信確認證]]（{{lang|ja|ベリカード}}，可視為是廣播電台業的[[QSL卡|QSL卡]]），不过日本国际广播电台外语组却向日本国内听众发放QSL卡（这一点与[[中国国际广播电台|中国国际广播电台]]类似）。

针对[[发达国家|发达国家]]和地区的[[卫星电视|卫星电视]]、[[有线电视|有线电视]]及[[互联网电视|互联网电视]]迅速发展普及这一状况，NHK更制订新计划，加强NHK节目在发达国家及地区的互联网及卫星电视之覆盖率。自2007年10月1日开始，NHK环球广播网停止对[[欧洲|欧洲]]（俄罗斯除外）、[[北美|北美]]與[[夏威夷|夏威夷]]地区的日语广播，逐漸消減对发达国家及地区的广播。

日本国际广播电台现有18种语言对外广播，[[调谐信号|节目开始曲]]为日本民歌《[[樱花_(民谣)|樱花]]》，在节目正式播放之前会播放《{{lang|ja|数え歌}}》的钟声。<ref>[http://pan.baidu.com/s/1o7Xc91S 日本国际广播电台中文广播开台录音，由中国大陆网友Radiofun-Hong录制]，2016-08-30</ref><ref>[http://www.intervalsignals.net/sounds/jap-z-radio_japan_cmk_c1977.mp3 日本国际广播电台英语广播开台录音，由新西兰网友Chris Mackerell录制]，Interval Signals Online，1977</ref>

== 沿革 ==
* 1930年10月27日 与其他国际电台进行节目交换播出
* 1934年4月29日 名崎发射台启用
* 1934年6月1日 开始向[[满洲地区|满洲]]及[[台湾|台湾]]用[[短波|短波]]频率定时转播日本国内广播节目
* 1935年6月1日 开始向[[北美|北美]]西部、[[夏威夷|夏威夷]]地区进行海外广播（当时社团法人[[日本广播协会|日本广播协会]]的[[英文|英文]]全称为The Broadcasting Corporation Of Japan）
* 1937年4月1日 开始对外进行[[法语|法语]]广播（同年4月2日[[德语|德语]]，8月3日[[西班牙语|西班牙语]]、8月23日[[汉语|汉语]]），最初对外广播时使用的呼号为东京广播电台（Radio Tokyo）
* 1938年1月3日 开始对外进行[[葡萄牙语|葡萄牙语]]广播 （同年8月15日[[荷兰语|荷兰语]]）
* 1940年6月1日 对外开始[[印地语|印地语]]广播（同年6月3日[[缅甸语|缅甸语]]、[[泰语|泰语]]，8月1日[[广东话|广东话]]）
* 1940年11月1日 八俣发射站启用
* 1941年1月1日 对外开始[[阿拉伯语|阿拉伯语]]、[[意大利语|意大利语]]、[[福建话|福建话]]、[[马来语|马来语]]广播（同年12月18日[[他加禄语|他加禄语]]）
* 1941年1月正式将“东京广播电台”确定为对外广播呼号。
* 1942年4月1日 对外使用[[泰米尔语|泰米尔语]]进行广播（同年8月16日[[土耳其语|土耳其语]]、10月11日[[波斯语|波斯语]]、12月3日[[乌尔都语|乌尔都语]]、12月5日[[俄语|俄语]]）
* 1943年7月2日  对外使用[[孟加拉语|孟加拉语]]广播（同年8月1日[[古吉拉特语|古吉拉特语]]）
* 1945年8月15日 对海外进行[[玉音放送|玉音放送]]
* 1945年9月4日 遵照[[驻日盟军总司令|驻日盟军总司令]]部的命令，停止对外的外国语广播。
* 1945年9月10日 遵照驻日盟军总司令部的命令，停止对外的日语广播。
* 1950年6月1日 广播法实施，解散社团法人日本广播协会。新成立[[日本广播协会|日本广播协会]]，将其作为特殊法人继承社团法人日本广播协会一切权利和义务。
* 1952年2月1日 重新开始对外广播，使用呼号为「Radio Japan」（{{lang|ja|ラジオニッポン}}）。

== 发射台及在日本國外的播送 ==
日本国际广播电台在日本国内的播放基地，是位於[[茨城县|茨城县]][[古河市|古河市]]东山田的[[八俣发射台|八俣发射台]]，這個由NHK向[[KDDI|KDDI]]租借設施運作的發射台設有300千瓦发射机7部，100千瓦发射机4部，主要是播放日本国际广播电台及英国[[BBC|BBC]]世界台的內容（中文广播已于2011年3月25日停止转播），除此之外也有[[法国国际广播电台|法国国际广播电台]]、[[加拿大国际广播电台|加拿大国际广播电台]]（从2012年6月25日起停止短波广播）、对[[朝鲜|朝鲜]]播出的[[潮风电台|潮风电台]]（{{lang|ja|しおかぜ}}，針對日本人疑似遭[[北韓政府綁架事件|北韓政府綁架事件]]而由民間團體在2005年時開播的特殊電台）、“[[故乡的风|故乡的风]]”（日语节目台呼，韩语节目台呼为“日本的风”）等广播电台的节目。目前由于英国BBC加拿大国际广播电台和法国国际广播电台的中文短波广播相继结束，该发射台只负责发射NHK和潮风电台、故乡的风电台的广播。

在日本國外的播放方面，日本國際廣播電台的節目是以當地電台轉播的方式，在[[新加坡|新加坡]]、[[英国|英国]]、[[法国|法国]]、[[德国|德国]]與[[加拿大|加拿大]]等國播放。在[[俄罗斯|俄罗斯]][[莫斯科|莫斯科]]（[[中波|中波]]738千赫），[[亚美尼亚|亚美尼亚]]（中波1350千赫），[[中国|中国]][[香港|香港]]（[[调频|调频]]99.7兆赫），[[印尼|印尼]][[雅加达|雅加达]]（[[调频|调频]]89.2兆赫），[[孟加拉國|孟加拉]][[达卡|达卡]]（调频97.6兆赫），[[巴勒斯坦|巴勒斯坦]][[杰宁|杰宁]]（调频89.3兆赫）、[[拉马拉|拉马拉]]（调频107.2兆赫），[[阿富汗|阿富汗]][[喀布尔|喀布尔]]與[[赫拉特|赫拉特]]（调频88.0兆赫），[[塔吉克斯坦|塔吉克斯坦]]（中波900千赫）等地區或城市有直接廣播。除此之外，也在[[中美洲|中美洲]][[加勒比海|加勒比海]]的[[波内赫岛|波内赫岛]]上租用[[短波|短波]]中继站，对中美洲及[[南美洲|南美洲]]进行广播，发射功率为250千瓦。<ref>{{cite web |url=http://www3.nhk.or.jp/nhkworld/japanese/radio/shortwave/southamerica.html |language=ja |title=日本國際廣播電台放送周波数表（日本語短波放送）-南米 周波数表 |publisher=NHK Online |date= |accessdate=2011-03-31 |deadurl=yes |archiveurl=https://web.archive.org/web/20100408103418/http://www.nhk.or.jp/nhkworld/japanese/radio/shortwave/southamerica.html |archivedate=2010-04-08 }}</ref>。

此外，NHK的國內二台（[[NHK廣播第2頻率|NHK廣播第2頻率]]）也会播出由日本国际广播电台制作的外语节目，其中在东京时间每天下午會播出10-15分钟長的華語新闻，每周六、日下午播出华语节目《日本百科》（{{lang-ja|日本ジャーナル}}）。
