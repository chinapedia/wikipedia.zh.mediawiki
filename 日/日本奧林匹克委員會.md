'''日本奧林匹克委員會'''（{{jpn|j=日本オリンピック委員会|hg=ニッポンオリンピックイインカイ|rm=Nippon orinpikku iinkai|f=h}}，簡稱'''日本奧委會'''）是[[國際奧林匹克委員會|國際奧林匹克委員會]]承認的[[國家奧林匹克委員會|國家奧林匹克委員會]]，其總部位於[[日本|日本]][[東京都|東京都]][[澀谷區|澀谷區]]的[[岸紀念體育館|岸紀念體育館]]。日本奧委會是在1911年成立的一個非盈利的日本體育組織，負責日本國內及國際運動及比賽、加強與國際奧委會、其他國家奧委會聯繫、以及維護[[奧林匹克|奧林匹克]]之名稱、標誌及會徽的唯一機構。日本奧委會曾主辦[[1964年夏季奧林匹克運動會|東京奧運]]、[[1972年冬季奧林匹克運動會|札幌冬奧會]]及[[1972年冬季奧林匹克運動會|長野冬奧會]]等綜合性運動會。

==歷史==
1909年5月10日，[[柔道|柔道]]創始人[[嘉納治五郎|嘉納治五郎]]於[[柏林|柏林]]進行的[[國際奧林匹克委員會全體委員大會|第10屆國際奧委會全體委員大會]]會議上當選為[[國際奧林匹克委員會委員|國際奧林匹克委員會委員]]，成為首位成為國際奧委會委員的[[日本人|日本人]]；及後他在1911年7月在日本成立'''日本業餘體育協會'''（{{lang-ja|'''大日本体育協会'''}}），翌年被[[國際奧委會|國際奧委會]]承認為日本的[[國家奧林匹克委員會|國家奧林匹克委員會]]<ref>{{cite web |language=en | title=Japan |publisher=國際奧委會  |accessdate=2011年12月12日 |url=http://www.olympic.org/japan }}</ref>。1912年，以嘉納治五郎為團長的日本隊參與[[1912年夏季奧林匹克運動會|斯德哥爾摩奧運]]，除了是日本首次參與奧運外，也是首個參與奧運的亞洲國家。日本於1913年參與首屆[[遠東運動會|遠東運動會]]；1917年的第三屆遠東運動會在日本東京舉辦，而日本業餘體育協會也於同年正式加入遠東體育協會，惟於兩年後的3月17日宣布退出該協會<ref>{{cite web |language=ja | title=JOC年表 1909 - 1919 |publisher=日本奧委會 |accessdate=2011年12月12日 |url=http://www.joc.or.jp/about/history/1909.html }}</ref>。

日本奧委會於1936年7月為[[東京|東京]]在第36屆國際奧委會全體委員大會上成功申辦[[1940年夏季奧林匹克運動會|第12屆奧運]]主辦權，但及後國際奧委會宣布因[[第二次世界大戰|第二次世界大戰]]關係而取消。日本業餘體育協會於1945年進行改組為團體組織，翌年12月4日正式改組並易名為'''日本奧林匹克委員會'''（{{lang-ja|'''日本オリンピック委員会'''}}）<ref>{{cite web |language=ja | title=JOC年表 1920-1946 |publisher=日本奧委會 |accessdate=2011年12月12日 |url=http://www.joc.or.jp/about/history/1920.html }}</ref>。二戰結束後，日本奧委會不獲邀請參與同在1948年舉辦的[[1948年夏季奧林匹克運動會|倫敦奧運]]以及[[1948年冬季奧林匹克運動會|聖莫里茨冬奧會]]；而日本奧委會於1952年起出席[[亞洲運動會|亞洲運動會]]。

日本奧委會在國際奧委會主席[[艾弗里·布倫戴奇|艾弗里·布倫戴奇]]於1955年訪日時提出東京申辦[[1964年夏季奧林匹克運動會|第18屆奧運]]主辦權的建議後提出申辦，並於1958年5月13日成功申辦該運動會主辦權<ref>{{cite web |language=ja | title=JOC年表 1947-1961 |publisher=日本奧委會 |accessdate=2011年12月12日 |url=http://www.joc.or.jp/about/history/1947.html }}</ref>；而在1966年的第64屆國際奧委會全體委員大會上宣布[[札幌|札幌]]成功申辦[[1972年冬季奧林匹克運動會|1972年冬奧會]]主辦權，令日本成為首個主辦夏季及冬季奧運的亞洲國家；另一方面，在東京以及札幌舉辦奧運期間，國際奧委會分別在兩地召開第63屆及第72屆國際奧委會全體委員大會<ref>{{cite web |language=ja | title=JOC年表 1962年至1976年 |publisher=日本奧委會 |accessdate=2011年12月12日 |url=http://www.joc.or.jp/about/history/1962.html }}</ref>。國際奧委會執行委員會於1979年10月23日至25日在[[名古屋|名古屋]]舉行，會議焦點是解決[[兩岸|兩岸]]奧運席位問題的[[名古屋決議|名古屋決議]]。日本奧委會於1986年及1990年成功在札幌主辦首兩屆[[亞洲冬季運動會|亞洲冬季運動會]]；1989年8月7日，[[文部省|日本文部大臣]]批准日本奧委會成為財團法人組織（{{lang|ja|財団法人}}）；1990年9月在東京主辦第96屆國際奧委會全體委員大會<ref>{{cite web |language=ja | title=JOC年表 1977-1990 |publisher=日本奧委會 |accessdate=2011年12月12日 |url=http://www.joc.or.jp/about/history/1977.html }}</ref>。

日本奧委會於1991年6月成功為[[長野|長野]]奪得[[1998年冬季奧林匹克運動會|第18屆冬奧會]]。同年9月15日，日本奧委會在東京召開的[[東亞運動會總會|東亞奧林匹克委員會]]第一屆理事會上提出舉辦[[東亞運動會|東亞運動會]]的構想<ref>{{cite web |language=ja | title=東アジア競技大会 / 沿革 |publisher=日本奧委會 |accessdate=2011年12月12日 |url=http://www.joc.or.jp/int_games/east_asia/enkaku.html }}</ref>。翌年1月1日，日本奧委會設立「日本奧委會體育獎（{{lang-ja|JOCスポーツ賞}}）」；奧委會官方會刊《奧林匹克》於同年2月14日正式創刊<ref>{{cite web |language=ja | title=JOC年表 1991-1995 |publisher=日本奧委會 |accessdate=2011年12月12日 |url=http://www.joc.or.jp/about/history/1991.html }}</ref>。

2000年8月28日，日本奧委會決定由[[大阪|大阪]]申辦[[2008年夏季奧林匹克運動會|第29屆奧運]]主辦權，但最終不敵[[北京|北京]]<ref>{{cite web |language=ja | title=JOC年表 1996-2000 |publisher=日本奧委會 |accessdate=2011年12月12日 |url=http://www.joc.or.jp/about/history/1996.html }}</ref>。2001年及2003年，大阪及[[青森|青森]]分別成功主辦[[2001年東亞運動會|第三屆東亞運]]、[[2003年亞洲冬季運動會|第五屆亞冬運]]。2006年8月31日，日本奧委會決定由東京申辦[[2016年夏季奧林匹克運動會|第31屆奧運]]主辦權，但最終由[[里約熱內盧|里約熱內盧]]奪取<ref>{{cite web |language=ja | title=JOC年表 2011- |publisher=日本奧委會 |accessdate=2011年12月12日 |url=http://www.joc.or.jp/about/history/2011.html }}</ref>。由於日本政府對財團法人制度進行改革，日本奧委會於2011年4月1日被[[財務省|日本財務省]]增進為公益財團法人組織（{{lang|ja|公益社団法人}}）<ref>{{cite web |language=ja |title=特定公益増進法人（旧第２号－イ～ヌ） |publisher=[[財務省|財務省]] |date=2011年4月1日 |accessdate=2011年12月14日 |url=http://www.mof.go.jp/tax_policy/reference/tokutei_koueki/ichiran/ko03.htm#07 |deadurl=yes |archiveurl=https://web.archive.org/web/20111203154013/http://www.mof.go.jp/tax_policy/reference/tokutei_koueki/ichiran/ko03.htm#07 |archivedate=2011年12月3日 }}</ref>；日本奧委會於同年7月16日在該會成立100週年慶祝大會時正式宣布由東京角逐[[2020年夏季奧林匹克運動會|第32屆奧運]]主辦權<ref>{{cite web |language=ja |title=招致スケジュール |publisher=日本奧委會 |accessdate=2011年12月12日 |url=http://www.tokyo2020.jp/schedule/index.html |deadurl=yes |archiveurl=https://web.archive.org/web/20120120062517/http://tokyo2020.jp/schedule/index.html |archivedate=2012年1月20日 }}</ref><ref>{{cite web |language=ja | title=創立100周年祝賀式典を開催 |publisher=日本奧委會 |date=2011年7月16日 |accessdate=2011年12月12日 |url=http://www.joc.or.jp/photonews/2011/07/100-9306.html }}</ref>。並於2013年9月申辦成功。
