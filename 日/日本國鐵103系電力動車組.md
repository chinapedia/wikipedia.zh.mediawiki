'''103系'''是[[日本國有鐵道|日本國有鐵道]]所開發和使用的一款[[直流|直流]][[通勤型電聯車|通勤型電聯車]]。103系以[[日本國鐵101系電力動車組|101系]]為基礎，鑒於101系製造成本高昂，考慮當時日本國鐵的財政狀況而設計的。從1963年3月到1984年1月間，共製造出3,447節103系車廂，是日本歷代製造數目最多的鐵路車輛。

在[[國鐵分割民營化|國鐵分割民營化]]之後，103系仍然活躍於由日本國鐵拆分而出的[[東日本旅客鐵道|東日本旅客鐵道]]（JR東日本）、[[東海旅客鐵道|東海旅客鐵道]]（JR東海）、[[西日本旅客鐵道|西日本旅客鐵道]]（JR西日本）等鐵路公司的路線上。雖然從首批車輛製造至今已接近50年，但103系尚未完全退役，較後期製造的車輛現時仍於西日本地區及九州的[[筑肥線|筑肥線]]服務。

== 概要 ==
1958年時，日本國鐵[[中央線|中央線]]101系投入服務，但是由於乘車率經常在100%以上的非常狀況，催生了103系的誕生，所以103系在1964年以後就大量製造，成為戰後日本國鐵擁有數量最多的電車。1970~1980年代，[[東京|東京]]、[[大阪|大阪]]及日本的大都市普遍都是使用103系作通勤輸送目的。而全車均是使用普通鋼製造的。

跟103系有關連的系列有：與[[東京地下鐵東西線|東京地下鐵東西線]]作直通運轉用的鋁製電車[[日本國鐵301系電力動車組|301系]]、地方電化路線用的[[日本國鐵105系電力動車組|105系]]、[[飯田線|飯田線]]用的[[日本國鐵119系電力動車組|119系]]和[[日本國鐵|日本國鐵]]末期在[[四國地方|四國地方]]投入的[[日本國鐵121系電力動車組|121系]]。

== 基本 ==
103系的車體是建基於101系。以下是103系和101系不同的地方：

*車輪直徑擴大（860mm→910mm），而車廂地板也隨之而上升。
*運轉台窗口縮小
*運轉台下部加設入風口
*[[方向幕|方向幕]]擴大
*新增了重開車門按鈕

但是103系最為人垢病的地方就是噪音，經調查後是由它的MT55型電動機發出。例如[[埼京線|埼京線]]及[[京葉線|京葉線]]、一些站和站間距離較長的路線，因為它們普遍的行車速度（在這些路線）甚高，產生出非常大的噪音及震盪，所以經常被沿線居民投訴。

在整個103系生產年期中，新造車有3,447部，20部由[[日本國鐵72系電力動車組|72系]]改裝，36部由101系改裝，總數3,503部。不過65部在[[奈良線|奈良線]]、[[櫻井線|櫻井線]]及[[和歌山線|和歌山線]]中的103系0及1000番台就改裝成105系。

== 番台分類 ==
基本上來說，103系的製造年期長達21年。由於要切合服務需求，因此做成大量的「番台」，即是分支版本。以下是103系各番台的介紹：

=== 900番台 ===
是103系的先行試製車，1963年在[[JR山手線|山手線]]試車。和普通103系的主要分別就是在[[轉向架|轉向架]]方面。先行試作車當時是使用0番台這個名字來作稱呼。八輛試作車是使用了以下的編號作標記：クハ103-901~904，モハ103-901、902、102-901、902。而這批試製車在它們晩年的時候是和量產車一起在[[JR山手線|山手線]]服務，直到1992年，因為全數的試驗車壽命已差不多至極限而和一些的103系一起被淘汰。

=== 0番台 ===
由1964年至1984年都還在生產，是103系最基本型號。而在不同年期生產的版本都是有分別的。

==== 1963年~1970年製造型號 ====
*103系的量產車第一條服務的路線是[[山手線|山手線]]，而使上面提到的試作車編號都改為'''{{lang|ja|サハ}}103型'''。兩年後，即是1965年，[[京濱東北線|京濱東北線]]也投入了103系，不過當年的[[列車製造廠|列車製造廠]]並未能配合到103系的長度，所以出現過3+7（甚至是2+6）的編組，在這時候出現了'''{{lang|ja|クモハ}}103'''及Mc編組的偶數編號的[[駕駛車|駕駛車]]變成了'''{{lang|ja|クハ}}103-500番台'''。
*在1967年投入的{{lang|ja|クハ}}103（[[常磐快速線|常磐快速線]]用），[[轉向架|轉向架]]車輪的制軔方式改用了碟片剎車。
*被喻為「103系之首」即1963年投入營運的{{lang|ja|クハ}}103-1，直至2011年3月才從[[JR西日本|JR西日本]]的[[阪和線|阪和線]]退役，退役時為[[JR集團|JR集團]]車齡最高的[[直流電車|直流電車]]，使用達48年。退役後於2015年3月從吹田工場迴送至[[京都鐵道博物館|京都鐵道博物館]]，於2016年3月開館起展示。

==== 冷氣試作車（1970年製造）型號 ====
*在當時、京王帝都電鐵（現在的[[京王電鐵|京王電鐵]]）的[[京王5000系電力動車組|5000系]]及部分的私鐵都在通勤電車上加設了冷氣系統，所以日本國鐵就在1個103系編組上試驗冷氣系統，這車也是投入到山手線去。為了比較各種的空調系統，每一個車箱的冷氣都是不同型號的。而在冷氣量產型號上的冷氣電機試驗車所使用的是有不同的。
在1978年的量產型クハ103的設計稍作修改，它的不同地方有：車窗排列、電動方向幕、前照燈改成1盞[[白熱燈|白熱燈]]、運轉席下的通風口仍保留原來的設計。
*クハ103-178及179、モハ103-279~281及102-434~436、サハ103-306及307，全數車輛由JR東日本接收。

==== 1971年-1972年製造型號 ====
*承上文，這一批的103系就是上面所說的量產型，車頭的照明燈由1盞白熱燈變作2盞燈泡，俗稱「1次改良車」。
*クハ103-180~212（這188個車廂當初是配屬[[松戶電車區|松戶電車區]]作[[常磐快速線|常磐快速線]]用，後來，再由[[中央·總武緩行線|中央·總武緩行線]]轉到[[京葉線|京葉線]]去了。也有部份是屬於[[JR西日本|JR西日本]]明石電車區（モハ103-282~330、102-437~486，サハ103-308~323），任務是取代在[[京阪神緩行線|京阪神緩行線]]的舊型電車。上面提到的（其中一列是クハ103-188）及餘下的車輛就再重新編配到[[松戶電車區|松戶電車區]]，好讓[[常磐快速線|常磐快速線]]的編組增長（由之前的8節變作10節）。

==== 1973年製造型號 ====
*在1973年，由於日本國鐵的財政預算不足，原本預計在[[京濱東北線|京濱東北線]]投入服務的冷氣車組（モハ103-373~382、102-529~538）被迫變作「預備安裝冷氣」車組，此外，座位也有擴闊。
*クハ103-213~268，モハ103-331~413、102-487~569，サハ103-324~359（モハ103-373~382及102-529~538除外）這批的クハ103型大部分屬於[[京阪神緩行線|京阪神緩行線]]（[[高槻電車區|高槻電車區]]）。所以導致[[JR西日本|JR西日本]]後來接收到的103系也是中期生產的103系。
*餘下的駕駛車，就和其他車廂組合在[[中央快速線|中央快速線]]服務，這批的電車就是日後在[[中央、總武緩行線|中央、總武緩行線]]（[[習志野電車區|習志野電車區]]）黃色車身的103系，還有一部分的103系編入了[[陸前原之町電車區|陸前原之町電車區]]（現在的宮城野電車區）。RT235編組（クハ103-235共四列）日後將被205系取替。

==== 1974年-1982年製造型號 ====
*為提防[[踏切事故|踏切事故]]（平交道事故）駕駛車使用了「高駕駛室」的設計。在駕駛車的窗口下加上了不鏽鋼飾帶，為視覺上帶來新的感覺。在這時候[[山手線|山手線]]及[[京濱東北線|京濱東北線]]正式使用[[列車自動控制系統|ATC]]系統，駕駛室後面部分放置了ATC的儀器。後來[[中央快速線|中央快速線]]及[[福知山線|福知山線]]也採用ATC系統（797、799~808、810~815、817~844、846、848、850）。
*經過[[北陸隧道列車火災事故|北陸隧道列車火災事故]]後、和地下鐵作直通運轉的1000番台符合了[[A-A基準|A-A基準]]，也設有貫通門。

==== 1983年~1984年製造 ====
*這批的103系是最後製造的、車廂編號是モハ103-787-793及，モハ102-2044-2050
*當時[[日本國鐵201系電力動車組|201系]]已開始製造了，所以這批的103系都混合了一些201系的元素。
:*車頂設計不同

=== 地下鐵直通車輛 ===
[[File:Jnr_103-1200.jpg|thumb]]
現主力的[[東京地下鐵|東京地下鐵]]車輛主要都是209系或其私鐵車輛。而日本國鐵時代的車輛103系1200番台及301系大多都已經被引退了。

==== 1000番台 ====
1976年起10節16編組、一共160輛製造並配置在松戶電車區，在[[常磐緩行線|常磐緩行線]]至[[東京地下鐵千代田線|帝都高速度交通營團千代田線]]直通服務、後來調配地上線服務。也有改造了[[日本國鐵105系電力動車組|105系電車]]。

==== 1200番台 ====
是日本國鐵時代東京地下鐵的主力電車，以藍帶作為車身的顏色，車頭前有通過門可與301系併結行走。 103系1200番台圖樣是右邊。

==== 1500番台 ====
屬於[[九州旅客鐵道|JR九州]]的唐津運輸中心，使用[[筑肥線|筑肥線]]至[[福岡市交通局|福岡市營地下鐵]][[空港線|空港線]]直通運轉。

== 改造、改修番台 ==
=== 750番代（由101系改編） ===
1972年起由{{lang|ja|サハ}}101型改造{{lang|ja|サハ}}103型，差異有車體全高和轉向架。改造內容是增加冷氣、[[方向幕|方向幕]]為布牌顯示等。

=== 2000·2050番代 ===
為了解決[[關西本線|關西本線]]（大和路線）、[[阪和線|阪和線]]駕駛車輛的不足，由101系電車改造後編入。2000番代屬于奈良電車區使用關西本線，2050番代屬于日根野電車區使用阪和線。

=== 3000番代 ===
1985年在大井工場改造。是由原來的[[日本國鐵72系電力動車組|72系電車]]之中，用翻新車體的{{lang|ja|モハ}}72型970番代和{{lang|ja|クハ}}79型600番代，更新103系的機器而來的。配置在川越電車區。

由於當地[[鐵道電化規格|鐵道電化規格]]與日本JR/私鐵所採用的[[高架電纜|架空電車線方式]][[直流|直流]]1500V相同，而路軌[[軌距|軌距]]規格同樣是1067mm，故只需加裝當地鐵道的信號保安裝置便可行走。

後來因日本私鐵有不少[[不鏽鋼|不鏽鋼]]車身的通勤列車退役，而[[印尼|印尼]]方面亦考慮到[[不鏽鋼|不鏽鋼]]車身在保養方面較有成本效益，故後來相繼引入[[東急8000系電力動車組|東急8000系]]、[[東急8500系電力動車組|東急8500系]]、[[東京地下鐵5000系電力動車組|東京地下鐵5000系電力動車組]]及[[東葉高速鐵道|東葉高速鐵道]][[東葉高速鐵道1000系電力動車組|1000系]]，而103系就再沒有被引入。印尼所有的103系已於2016年1月1日全數退役。
