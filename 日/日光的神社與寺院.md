'''日光的神社與寺院'''<ref>[http://twh.hach.gov.tw/WorldHeritageContent.action?id=731 世界遺產內容]{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>（[[日文|日文]]：'''{{lang|ja|日光の社寺}}'''），是指存在於[[日本|日本]][[栃木縣|栃木縣]][[日光市|日光市]]的[[神社|神社]]與[[寺院|寺院]]的總稱。1999年12月2日被[[聯合國教科文組織|聯合國教科文組織]]指定為[[世界遺產|世界遺產]]。

== 簡介 ==
於1616年，[[德川家康|德川家康]]的靈廟，即[[日光東照宮|日光東照宮]]開始興建。此後，日光一地就成為了德川家族及幕府的聖地。之後，在1636年，東照宮又進行了大規模的翻修，才形成了現在的規模。接著還興建了德川幕府第三代將軍，也就是[[德川家光|德川家光]]的靈廟，即今[[輪王寺|大猷院]]。

此外，自8世紀以來，日光是以[[男體山|男體山]]為中心的山嶽信仰聖地。在男體山的山麓及附近的[[中禪寺湖|中禪寺湖]]有許多的神社及寺院存在。其中，建築最為壯觀華麗的是[[二荒山神社|二荒山神社]]及[[輪王寺|輪王寺]]，由於幕府的大力經營，建築物上的雕刻非常細緻。

== 登錄範圍 ==
* [[日光東照宮|日光東照宮]]
* [[日光二荒山神社|日光二荒山神社]]
* [[輪王寺|日光山輪王寺]]
共計103棟建築。由於包含範圍總共有兩座神社及一座寺院，也被當地的人暱稱為「二社一寺」。

== 登錄基準 ==
日光的神社與寺院因符合下列的標準而被登錄為世界文化遺產：
* (i) 代表了人類創造精神的傑作。
* (iv) 突出地代表了某一類建築或技術的，並且展示了人類歷史上的某一段或幾段非常重要的時期。
* (vi) 直接或明確地同某些具有突出的、普世的價值的事件、現實的傳統、思想、信仰、文化作品或文藝作品相聯繫
