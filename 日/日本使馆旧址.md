'''日本使馆旧址'''，位于[[北京市|北京市]][[东城区_(北京市)|东城区]][[正义路|正义路]]2号（[[北京市人民政府|北京市人民政府]]院内），是[[日本驻华大使馆|日本驻华大使馆]]旧址<ref name=dhly>{{Cite web |url = http://www.bjdclib.com/subdb/laneculture/historybuilding/200906/t20090610_1829.html |title =  日本使馆旧址|author =  |date =  |publisher =  2009-06-10|language = 东华流韵 |accessdate =  |quote =  }}</ref>。

==历史==
[[清朝|清朝]][[光绪|光绪]]二十六年（1900年）之前位于[[东交民巷|江米巷]]北侧的[[日本公使馆旧址|日本公使馆]]，在1900年[[义和团|义和团]]围攻使馆区时并未被破坏。但是在《[[辛丑条约|辛丑条约]]》签订后，日本借口公使馆原址狭小，不敷使用，迫使清政府将西边的[[詹事府|詹事府]]、[[肃亲王府|肃亲王府]]及部分民宅划为新的用地。该用地南部新建日本使馆，北部新建日本兵营。新址的北部和东部与[[法国使馆旧址|法国使馆]]、[[意大利使馆旧址|意大利使馆]]相接，南部与[[西班牙使馆旧址|西班牙使馆]]、[[横滨正金银行北京支行|横滨正金银行]]相邻，正门开在御河路（今正义路），和[[淳亲王府|英国使馆]]隔御河相对<ref name=dhly/>。

该日本使馆的设计师是日本建筑师[[真水英夫|真水英夫]]。使馆兴建时保留了肃亲王府花园内的部分树木、山石、围墙<ref name=dhly/>。

[[中华人民共和国|中华人民共和国]]时期，日本使馆旧址成为[[北京市人民政府|北京市人民政府]]办公区。2001年，日本使馆旧址作为东交民巷使馆建筑群的组成部分，被列为[[第五批全国重点文物保护单位|第五批全国重点文物保护单位]]<ref name=dhly/>。

2012年，北京市启动了该市自中华人民共和国成立以来规模最大的“名城标志性历史建筑恢复工程”和“‘百项’文物保护修缮工程”，投入10亿元人民币用于保护。日本使馆旧址被列入“‘百项’文物保护修缮工程”。2012年，北京市测绘设计研究院开始为日本使馆旧址主楼（北京市人民政府6号楼）采集三维激光扫描数据，以便为今后修缮做准备<ref>{{Cite web |url =  http://culture.people.com.cn/n/2012/0716/c172318-18525915.html|title =北京市文保日本使馆旧址修缮 进入扫描数据阶段  |author =  |date = 2012-07-16 |publisher =人民网  |language =  |accessdate =  |quote =  }}</ref>。

==建筑==
日本使馆旧址现存建筑有使馆大门、主楼等<ref name=dhly/>。

使馆大门楼坐东朝西，面阔三间，进深一间，总长度为14.74米，中央门洞净宽为4.56米。大门楼两侧带有弧形墙，从而形成了长24米、深3.34米的大门前空地。大门楼是二层建筑，两侧门房里均有小楼梯可通到二层。风格为西方近代[[折衷主义建筑|折衷主义建筑]]风格。立面分为三部分，两侧是灰砖碉楼式墩体，三段式，檐部为[[古典主义建筑|古典主义建筑]]风格，女儿墙饰有[[巴洛克建筑|巴洛克]]式涡卷，上方是方锥形铁皮屋顶，总高9.35米。两楼间的大拱形门洞中部腰线之上是石墙，拱券之上套有山花<ref name=dhly/>。

使馆主楼是地上二层建筑，砖木结构，东北局部有地下室。主楼平面呈不规则矩形，南立面约长46米，南北进深约为34米，四面均有出入口，其中南面为主要出入口。立面重点偏东。入口大门微前凸，门前台阶上有一对[[爱奥尼柱式|爱奥尼式矮柱]]，矮柱上支撑一半圆筒拱做成门罩，两侧设有扶壁式厚侧墙。二层中央的门前有牛腿挑出阳台，门两侧有壁柱，上层的墙上带有盾徽装饰。门套上饰有花形，檐部壁柱头有巴洛克式涡卷，古典主义檐口上，立有方锥顶，正面开老虎窗。入口西翼的主体为六开间，底层是连续的罗马券柱式，中间用瓶式栏杆相连接，二层是双圆式柱廊，上用简化的檐部结束，双柱间有[[新艺术运动|新艺术运动]]风格铁花栏杆。西翼末端是个约7米见方的碉楼式凸出体，上部后退，四角的细柱支撑着一个四锥顶。中央入口的东翼设有两开间实墙开窗，屋顶檐口上有栏杆式女儿墙。东翼末端底层是半圆形凸出体，四根扁平[[塔司干柱式|塔司干式壁柱]]之间开有长窗，窗套是立在小塔司干壁柱上的半圆券。檐部上有弧形大挑檐，其上凸起成弧形护墙。二层墙面开有并联的两窗夹一门，门窗套是完整的罗马柱式，其上饰有半圆石券，顶部有高出屋檐的三角形山墙。该建筑西立面对着花园，门前有平台，平台长约25米、宽约4至8米。中段的立面是与正面西翼类似的通廊，但是二层是爱奥尼式双柱。北段末端向前凸出，装饰复杂。南段末端在南立面西翼的方形凸出体北侧，加有一座[[中世纪|中世纪]]风格的圆形碉楼<ref name=dhly/>。

日本使馆旧址的大门及主楼造型融合了西方古典主义建筑、巴洛克建筑、中世纪建筑风格，不讲求古典比例，而是讲求多要素融合。这是西欧近代集仿主义风格，更是日本近代模仿欧式风格的作品。该院内还有一幢二层楼以及一组平房，也都是近代建筑<ref name=dhly/>。
