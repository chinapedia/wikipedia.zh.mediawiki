== 日本劍術的起源 ==

{{see|日本刀}}

[[鑄造|鑄造]]術於4世紀由[[中國|中國]][[唐代|唐代]]及[[韓國|韓國]][[百濟|百濟]]傳到[[奈良時代|奈良時代]]的日本，弧形的[[日本刀|日本刀]]在[[平安時代|平安時代]]開始出現。日本劍術起源可以追溯到刀剑大革新的[[室町时代|室町时代]]，日本鑄劍名匠[[岡崎五郎入道正宗|岡崎五郎入道正宗]]研究出锻铁制法，相對的劍術亦應運而生。刀身剛性的改善孕育出全身力量集於刀刃之上、招招狠劈、攻擊為主的力量型劍術。

== 居合道 ==

{{see|居合道}}

日本劍術有其獨特的拔刀、揮劍和收劍方式，[[居合道|居合道]]則以此為基礎，發展出一連串極度規範的基本技法，講究「一擊必殺」。

== 二刀流劍法 ==

{{see|二天一流}}

使用雙刀的劍術，歷史上會使雙刀的劍客不多，[[宮本武藏|宮本武藏]]和[[柳生十兵衛|柳生十兵衛]]是當中的佼佼者。

== 劍術與現代劍道 ==

{{see|劍道}}

日本劍術是日本刀的使用技法，而[[劍道|劍道]]廣義來說是一種精神或態度，1945年以前的[[劍道|劍道]]曾是[[中學生|中學生]]必修的課程，使用的是竹劍，1950年代以來的劍道是一項純粹的體育運動。
