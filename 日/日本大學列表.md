== 國立大學 ==
在日本，所谓國立大學就是國家直接設置的大學和國立大學法人設置的大學。國立大學法人是國家全額出資的團體。但自從2004年4月1日的新學年開始，所有國立大學都要根據國立大學法成立獨立的法人組織。

== 公立大學 ==
