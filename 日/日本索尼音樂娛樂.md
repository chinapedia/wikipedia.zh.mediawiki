'''日本索尼音樂娛樂'''（{{lang|ja|ソニー・ミュージックエンタテインメント}}，簡稱索尼音樂（'''''{{Color|#000000|Sony}}{{Color|#EE0000|M}}{{Color|#808080|usic}}'''''）或'''SME'''、'''SMEJ'''）是[[索尼|索尼]]專責日本國內[[音樂|音樂]]事業部門的經營統合及[[控股公司|控股公司]]。1968年創立，之後曾經歷多次企業[[合併|合併]]與分割。現在的公司法人成立於2003年4月1日，為索尼全資[[子公司|子公司]]，總部位於[[東京|東京]][[麴町|麴町]]。以其為中心組成的日本索尼音樂集團（Sony Music Group）獨立於全球的[[索尼音樂娛樂|索尼音樂娛樂]]體系之外，後者由索尼在美國的事業體主導經營，兩者僅在品牌與公司名稱上共用「Sony Music」名稱。

== 歷史 ==
1968年3月1日，索尼與美國[[哥倫比亞廣播公司|哥倫比亞廣播公司]]（CBS）合資成立[[CBS/Sony|CBS索尼]]（CBS/Sony；成立時名為「CBS索尼唱片」）。此後索尼在日本國內又陸續成立[[史诗唱片日本|日本史詩唱片]]、CBS索尼唱片等音樂關聯企業。1988年1月，在索尼[[董事長|董事長]][[盛田昭夫|盛田昭夫]]的主導下，索尼跨海併購CBS旗下的[[哥倫比亞電影公司|電影]]與音樂公司，並在同年將旗下日本國內的音樂關聯企業全部併入CBS/Sony。索尼在美國所併購的CBS音樂部門（CBS唱片公司），於1991年更名為[[索尼音樂娛樂|索尼音樂娛樂]]（Sony Music），負責經營索尼在日本以外的全球音樂事業。

1991年4月，CBS/Sony改用現名（索尼音樂娛樂，下稱SME），並於同年11月在[[東京證券交易所|東證]]二部（中小企業板）上市。2000年，SME與索尼交換股權，成為後者的全資子公司，並自東證二部下市。2001年，SME將旗下音樂品牌與音樂製作部門分拆為獨立公司，改以[[控股公司|控股公司]]形式營運。2003年4月1日，SME進行企業重組，公司更名為「株式會社SMEJ」，並同時成立索尼音樂娛樂（SME新公司法人）、索尼文化娛樂（簡稱SCU）兩家子公司；株式會社SMEJ則於同年7月1日併入索尼母公司，使上述兩家公司成為索尼的「親公司」，擁有完整的控制權。2006年12月，索尼文化娛樂併入索尼音樂娛樂。

[[File:SME_Nogizaka_Building.jpg|thumb]]
由於股權分開的緣故，索尼音樂的全球體系在2000年後的一系列企業整併，與日本索尼音樂集團沒有直接關係。索尼音樂的全球體系與[[贝塔斯曼音乐集团|博德曼音樂]]（BMG）在2004年11月合併為[[索尼博德曼|索尼博德曼]]時，BMG的日本子公司「BMG Fun House」直屬於全球的索尼博德曼體系，隔年更名為「日本博德曼音樂」（BMG JAPAN）。2008年8月5日，索尼宣布收購[[博德曼|博德曼]]所持有的索尼博德曼50%股權，並於10月1日正式生效，索尼博德曼成為美國索尼的全資子公司，並更名回「索尼音樂娛樂」；日本博德曼音樂撥歸日本索尼音樂集團擁有及管理<ref>http://www.sony.co.jp/SonyInfo/IR/news/qfhh7c00000jbz2m-att/081002_SMEJ.pdf</ref>，之後因公司部門分割重組而在2009年10月解散。

2017年10月，日本索尼音乐宣布成立游戏发行公司“[[Unties|Unties]]”，并以此品牌在[[PlayStation_4|PlayStation 4]]、[[PlayStation_VR|PlayStation VR]]，甚至[[任天堂Switch|任天堂Switch]]和PC上发行游戏，发行的作品以独立游戏为主<ref>{{Cite web|title=索尼新发行公司Unties将在Switch上卖游戏|url=http://play.163.com/17/1017/18/D0VIP16K00318QLL.html|work=网易爱玩网|accessdate=2019-01-29|date=2017-10-17}}</ref>。

2014年4月1日，索尼音樂娛樂旗下8家音樂廠牌整合並重組為索尼音樂品牌公司，以取代原來的索尼音樂唱片公司（{{lang|en|Sony Music Record}}）。

===== 旗下廠牌 =====
; [[Sony_Music_Records|Sony Music Records]]
: 前身為1968年3月由索尼與CBS合併而成的“CBS/Sony公司”。2001年10月被索尼音樂娛樂所成立的同名製作公司合併。

; [[SME_Records|SME Records]]
: 1998年成立，業務涵蓋J-Pop的偶像派及實力派歌手。原為索尼音樂唱片旗下子廠牌，2003年4月成為獨立廠牌。

; [[SACRA_MUSIC|SACRA MUSIC]]
: 2017年成立。主要針對動畫主題曲，以及那些不僅活躍在日本同時也活躍在海外藝人的唱片。

; {{Link-ja|Ariola Japan|アリオラジャパン}}
: 2009年成立。由原[[BMG_JAPAN|博德曼日本]]在日本國內業務改組而成，主要承擔J-Pop中的個性派藝人作品的發行。

; [[史詩唱片日本|史詩唱片日本]]（EPIC Records Japan）
: 1978年成立。前身為史詩索尼唱片。主要承擔J-Pop中的實力派藝人作品的發行。

; 史詩JYP（Epic JYP）
: 主要承擔[[JYP娛樂|JYP娛樂]]旗下藝人作品的發行。

; {{Link-ja|Ki/oon Music|キューンミュージック}}
: 1992年成立。前身為Ki/oon Sony，主要承擔獨立音樂及搖滾音樂藝人的作品發行。

; [[Sony_Music_Associated_Records|Sony Music Associated Records]]
: 1998年成立，業務涵蓋[[J-Pop|J-Pop]]的偶像派及實力派歌手。同時也涉及跨界藝人的作品。

; 日本索尼音樂國際（Sony Music Japan International）
: 2001年成立。主要負責在日本發行海外音樂。

; {{link-ja|索尼音樂行銷|ソニー・ミュージックマーケティング}}（Sony Music Marketing Inc.）
: 2001年成立。負責娛樂軟件的發行與銷售。
