蹈火節（{{lang-ta|தீமிதி}}<ref name="theemidhi">{{cite web |url=http://www.heb.gov.sg/mainhindufestivals.html |title=存档副本 |accessdate=2007-11-09 |0=2007-10-31 |deadurl=yes |archiveurl=https://web.archive.org/web/20071031070055/http://www.heb.gov.sg/mainhindufestivals.html |archivedate=2007-10-31 }} </ref><ref name="theemithi">{{cite web |url=http://infopedia.nlb.gov.sg/articles/SIP_762_2004-12-23.html |title=存档副本 |accessdate=2007-11-09 |0=2007-10-29 |deadurl=yes |archiveurl=https://web.archive.org/web/20071029123341/http://infopedia.nlb.gov.sg/articles/SIP_762_2004-12-23.html |archivedate=2007-10-29 }}  </ref> '''Kundam'''）<ref>{{cite web|url=http://www.hindu.com/2005/03/26/stories/2005032602370300.htm |title=Tamil Nadu / Erode News : `Kundam' festival on Tuesday |publisher=The Hindu |date=2005-03-26 |accessdate=2014-06-12}}</ref>，是[[印度教|印度教]]的傳統節日，起源自[[印度南部|印度南部]][[泰米尔纳德邦|泰米尔纳德邦]]，並在[[泰米爾曆|泰米爾曆]]的Aipasi月份慶祝（大約為[[公曆|公曆]]的10月至11月）。節日裏，地上會鋪着炙熱的木材，而信徒們則會步過火路。
