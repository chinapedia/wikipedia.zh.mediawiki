'''济宁路浸信会教堂'''为[[中国|中国]][[青岛市|青岛市]]的一座教堂，建于1928年，由[[美南浸信会|美南浸信会]]差会开辟，座落于[[市北区|市北区]]济宁路31号（禹城路口）的山坡，是一座宏伟的[[新古典主义建筑|新古典主义风格]]建筑。

== 历史 ==
济宁路[[浸信会|浸信会]]由美籍牧师崔怡美开辟于1923年。1928年，信徒于墨林出资兴建了目前的[[新古典主义建筑|新古典主义风格]]教堂，建筑面积1693平方米。济宁路浸信会是青岛市区的8座浸信会教堂中规模最大的一座，1948年青岛市浸信会联合会在此成立。

1951年，美南浸信会的美国传教士全部撤离青岛。1958年秋，青岛市基督教实行联合礼拜，56处堂点合并为10处，该堂得以保留，乐陵路、松山路、平阴路等处浸信会均合并于此。1965年，济宁路浸信会被迫关闭，献给政府，改为工厂。 <ref>{{cite web|title =青岛市志18：民族宗教志 第二篇 宗教：基督教|url =http://qpinet.qingdao.gov.cn/shizhi.nsf/0/8749a5b427f034fb482566e1000c0268?OpenDocument|author =|date =|language =zh|publisher =|pages =}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

济宁路浸信会教堂旧址现为凯越国际青年旅馆。2005年，济宁路浸信会浸信会礼拜堂旧址被列为青岛市第七批文物保护单位 <ref>{{cite web|title = 　关于公布青岛市第七批市级文物保护单位的通知(二○○五年二月五日)|url = http://www.qingdao.gov.cn/n172/n68422/n1527/n1662668/n1662669/161049.html|author = |date = |language = zh|publisher = |pages = |access-date = 2012-08-05|archive-url = https://web.archive.org/web/20081121013115/http://www.qingdao.gov.cn/n172/n68422/n1527/n1662668/n1662669/161049.html|archive-date = 2008-11-21|dead-url = yes}}</ref>。
