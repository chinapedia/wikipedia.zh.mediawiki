{{山东历史}}
[[济南|济南]]拥有悠久的历史，1986年12月被[[国务院|国务院]]公布为[[国家历史文化名城|国家历史文化名城]]。济南因地处[[济水|济水]]之南而得名，早在八九千年之前的新石器时代早期，济南地区就已有了人类活动的踪迹。[[北辛文化|北辛文化]]、[[大汶口文化|大汶口文化]]、[[龙山文化|龙山文化]]等时期的文化遗址在济南地区均有发现。商代，[[城子崖|城子崖]]一带，建立了[[谭国|谭国]]。西周，行[[分封制|分封制]]，济南地区属于[[齐国|齐国]]。秦朝实行[[郡县制|郡县制]]，今济南市区地属[[济北郡|济北郡]]，称[[历下邑|历下邑]]。汉初，设立'''[[济南郡|济南郡]]'''，此为“'''济南'''”一名出现之始，当时郡治设在[[东平陵|东平陵]]（今[[章丘县|章丘县]][[平陵城|平陵城]]），东汉至魏晋，济南多次在[[济南国|济南国]]及[[济南郡|济南郡]]间变换。晋[[永嘉|永嘉]]年间，济南郡治由东平陵移至[[历城|历城]]。北魏，在济南郡之地置'''[[齐州|齐州]]'''，治所仍在历城。隋唐，或为齐郡，或为齐州。1116年，宋徽宗升州为府，齐州改称'''[[济南府|济南府]]'''。金代，济南府属[[山东东路|山东东路]]。元代，改称'''[[济南路|济南路]]'''，直隶于[[中书省|中书省]]。明初，曾置山东行省，以济南府为[[山东布政使司|山东布政使司]]、[[山东都指挥使司|山东都指挥使司]]及[[山东按察使司|山东按察使司]]驻地，济南始为山东首府，从此济南成为山东地区的政治中心。清代，济南仍为山东省治。中华民国成立后，改府为道，济南初属[[岱北道|岱北道]]，1914年改称[[济南道|济南道]]。1929年设立[[济南市|济南市]]。1948年9月24日，[[人民解放军|人民解放军]]攻克济南。


== 史前時代 ==

发源于济南的[[龙山文化|龙山文化]]存在于距今5000-4100年前左右，分布于黄河的中下游。龙山文化是中国[[青铜器文化|青铜器文化]]的形成期，当时已有了较为发达的畜牧业，且[[巫术|巫术]]活动较多。这是一个父权社会文明，私有财产出现，并开始有了[[阶级|阶级]]社会。其黑陶制品成为了著名的艺术品。

== 古代 ==

{{main|濟南郡|济南府}}
相传[[舜|舜]]曾“渔于雷泽，躬耕于[[历山|历山]]”。[[历山|历山]]即济南市区南部的[[千佛山|千佛山]]。所以市内至今还散遺各個以[[舜|舜]]命名的地名，如“[[舜井|舜井]]”、“[[舜耕路|舜耕路]]”、“[[舜耕山|舜耕山]]”等。

商代末期[[帝乙|帝乙]]、[[帝辛|帝辛]]（[[纣|纣]]）克东夷时[[甲骨文|甲骨文]]卜辞中的“泺(音洛，四声)”字即今日的[[趵突泉|趵突泉]]，从而把济南泉水有文字记载的历史上溯至3544年前（即公元前1542年）。春秋时期著名的[[齐|齐]][[晋|晋]]“崤”之战发生在市北的马鞍山一带。

中国首部诗歌总集《[[诗经|诗经]]》中的一首名叫《大东》的诗词，是现存最早的一篇有关济南的文献。

[[汉代|汉代]]公元前164年设立济南国，公元前154年又废国改郡。[[汉灵帝|汉灵帝]]时，[[曹操|曹操]]曾任济南相。[[西晋|西晋]]永嘉(307～312年)年间，济南郡治由东平陵移至[[历城|历城]]。[[隋朝|隋朝]]开皇三年(583年)，改济南郡为齐州。这段时期济南的佛教比较兴盛，留有[[千佛山|千佛山]]、[[灵岩寺|灵岩寺]]、千佛崖石窟造像、[[四门塔|四门塔]]（中国现存最古老的石塔）等佛教遗迹。

到了[[宋代|宋代]]济南属京东路，沿称齐州。[[宋徽宗|宋徽宗]]政和六年（公元1116年），齐州升为济南府。[[曾巩|曾巩]]任职于济南时曾各种赋诗撰文，促进了济南文学的发展。[[南宋|南宋]]年间，济南的文学更加繁荣，出现了[[李清照|李清照]]、[[辛弃疾|辛弃疾]]等中国历史上的著名宋词作家，两人被称作“济南二安”（李清照的号为“易安”）。

中国广告史上最早的商业印刷广告，出现在宋代的济南。[[上海博物馆|上海博物馆]]藏的[[北宋|北宋]]济南刘家针铺广告铜版，是现存最早的工商业印刷广告。在这则高12.5厘米，宽13厘米的广告铜板上，中心位置绘有商标——白兔捣药图，广告的标题是商店名称“济南刘家功夫针铺”，告诉人们商店的地址是“认门前白兔儿为记”，商店经营项目、质量要求和经营方针是：“收买上等钢条，造功夫细针，不误宅院使用；可转为贩，别有加饶。请记白”。这是目前世界上发现的最早印刷广告文物，比西方的印刷广告早三百多年。

[[元朝|元朝]]济南的文化依旧兴盛，产生过[[杜仁杰|杜仁杰]]、[[张养浩|张养浩]]等著名文学家和散曲文人。

明初，曾置山东行省，济南始为山东首府，清代济南仍为山东省治。[[康熙|康熙]]年间，济南的商业开始逐渐发展。

== 近代 ==

1904年，济南开设了商埠，工商业有了进一步的发展。1911年津浦铁路开通并路过济南。1912年[[清政府|清政府]]被推翻，[[中华民国|中华民国]]成立后，济南改作济南市。民国时期济南的工商业更加繁荣，[[瑞蚨祥|瑞蚨祥]]、[[大观园|大观园]]商业城和济南[[嘉禾|嘉禾]]卷烟厂开张，商埠的面积不断扩大。

1928年，进行[[北伐战争|北伐]]的[[国民革命军|国民革命军]]占领济南。而[[日本|日本]]却借口以“北伐军杀害济南日本侨民”为由，于5月3日出兵济南报复，将[[国民政府|国民政府]]外交部特派山东交涉员[[蔡公时|蔡公时]]及署内职员17人虐杀，随后对城中百姓进行大规模屠杀，超过6000中国居民被杀害，史称“[[五三惨案|五三惨案]]”。直至次年3月，南京国民政府与日本政府签订《中日济案协定》之后，日军才退出济南。

1937年，日本[[侵华战争|侵华战争]]爆发。同年12月，汉奸[[馬良_(民國)|马良]]出城迎接欲侵略济南的日军进城，济南陷落。1945年8月日军战败，[[国民政府|国民政府]]于同年收复济南。

1948年[[国共内战|国共内战]]时期，战乱导致全国物价飞涨、纸币贬值，济南经济全面崩溃，商埠逐渐消失。同年9月24日，[[中国人民解放军|中国人民解放军]][[济南战役|占领济南]]，并成立济南特别市和济南市人民政府。

== 现代 ==

[[中华人民共和国|中华人民共和国]]成立后，济南进行工商业[[社会主义|社会主义]]改造，各商铺以及工厂改由公私合营办理。

[[文化大革命|文化大革命]]时期，济南再次陷入混乱局面。学校和单位以开批斗大会为由停止工作，全市各部门瘫痪，[[红卫兵|红卫兵]]上街抄家、武斗。

1978年[[改革开放|改革开放]]后，济南进入复苏阶段。有[[中国轻骑集团|轻骑]]、[[小鸭集团|小鸭]]、[[浪潮集团|浪潮]]和[[鲁能集团|鲁能]]等著名企业出现，居民的生活水平有所提高，商业、旅游业、体育业也有所发展。

2004年济南成为[[2004年亚洲杯足球赛|亚洲杯足球赛]]的主赛场之一，2009年济南承办了[[中华人民共和国第十一届全运会|中华人民共和国第十一届全运会]]。
