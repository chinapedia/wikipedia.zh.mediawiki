'''千年鷹号'''（英文：'''{{lang|en|Millennium Falcon}}'''，经常简称为'''{{lang|en|Falcon}}'''）是《[[星球大战|星球大战]]》系列作品中一艘虚构的[[宇宙飞船|宇宙飞船]]，它的主要驾驶员是走私犯兼船长[[韩·索罗|韩·索罗]]（[[哈里森·福特|哈里森·福特]]飾演）和他的副驾驶员[[丘巴卡|丘巴卡]]（{{link-en|彼得·馬修|Peter Mayhew}}飾演）。在电影设定中，千年鷹是一艘经过韩·索罗大幅改装的{{lang|en|YT-1300}}型货船，其特性能够很好地服务于韩的走私业务。千年鷹是星战电影及其他形式文艺作品中出场次数最频繁的飞船之一，除了在经典三部曲中具有举足轻重的地位外，它还在前传三部曲中的《[[星球大战三部曲：西斯大帝的復仇|星球大战三部曲：西斯大帝的復仇]]》中客串出场，而在星战为数众多的小说及游戏等作品中也处处可见它的行踪。[[英国广播公司|英国广播公司]]在一篇新闻稿中评论道，按道理说千年鷹号可以算是这个宇宙中最著名的宇宙飞船了<ref name="bbc"/>。

==起源和设计==

根据星战产业的缔造者[[乔治·卢卡斯|乔治·卢卡斯]]介绍，最初创造千年鷹的灵感来源于一个[[汉堡包|汉堡包]]的外形，而驾驶舱仿如置于汉堡包旁的一个[[橄榄|橄榄]]<ref name="SW.comBTS">{{cite web en |url=http://www.starwars.com/databank/starship/millenniumfalcon/?id=bts|title=Star Wars: Databank: Millennium Falcon (Behind the Scenes)|publisher=starwars.com|accessdate=2007-06-27}}</ref>。在原先的设计中千年鷹具有更长的外形，但由于这个设计恰好和英国的科幻电视剧《太空：1999》中的飞船有过分相似之处，卢卡斯最终决定放弃这个设计方案<ref name="SW.comBTS" />。而原先制作好的模型经过改造并重新规划尺寸后被用在了[[莉亚公主|莉亚公主]]的飞船[[坦特维四号飞船|坦特维四号]]上<ref name=SculptingAG>{{cite book |last=Peterson |first=Lorne |title=Sculpting A Galaxy - Inside the Star Wars Model Shop |edition=1st ed. |year=2006 |publisher=Insight Editions |location=San Rafael, California |isbn=1-933784-03-2 |pages=pp 2-3}}</ref>。

在《[[星球大战五部曲：帝国大反擊|星球大战五部曲：帝国大反擊]]》的拍摄中，剧组人员在西[[威尔士|威尔士]]的[[彭布罗克|彭布罗克]]市的一个机库里制作了千年鷹的原尺寸模型<ref name="bbc">{{cite web en|url=http://news.bbc.co.uk/2/hi/uk_news/wales/south_west/4555455.stm|title=Towns secret Star Wars history|publisher=[[BBC|BBC]]}}</ref>。这个模型直径超过23米，重达23吨，利用压缩空气气垫装置可以在其位置周围漂浮在十六分之一英尺（约1.6毫米）的高度。不过这个庞然大物在三年后的《[[星球大战六部曲：绝地大反攻|星球大战六部曲：绝地大反攻]]》中拍摄的所有镜头最后都被剪掉，这主要包括影片中赫特族泽巴的驳船爆炸后千年鷹出现在[[塔图因|塔图因]]沙海中的一些镜头。而在这部电影中出现千年鷹的其他镜头都来自背景彩绘（{{lang|en|matte painting}}）和另外制作的小尺寸模型。也大约是在同一时间那个千年鷹的原尺寸模型被丢弃。

==描述==
千年鷹是韩·索罗通过赌博从他的好友[[藍道·卡瑞森|藍道·卡瑞森]]那里赢得的<ref name="SW.comEU">{{cite web en|url=http://www.starwars.com/databank/starship/millenniumfalcon/?id=eu|title=Star Wars: Databank: Millennium Falcon (Expanded Universe)|publisher=starwars.com|accessdate=2007-06-27}}</ref>。在《[[星際大戰四部曲：曙光乍現|星球大战IV：新希望]]》中[[欧比旺·克诺比|欧比旺·克诺比]]和[[卢克·天行者|卢克·天行者]]搭乘了千年鷹，在韩和丘巴卡的带领下运载着[[C-3PO|C-3PO]]和[[R2-D2|R2-D2]]以及盗取的[[银河帝国|帝国]][[死星|死星]]计划图纸准备前往[[奧德蘭_(星際大戰)|奧德兰]]。卢克第一次见到千年鷹时惊呼其外观“真是一块废物！”，然而韩却声称说千年鷹“可以达到零点五倍超空间。她可能是其貌不扬，但她在关键时刻从未掉过链子，小子。我亲自给她做过很多改装。”其后千年鷹被死星截获，千年鷹的船员们都躲到为走私而专门设计的地板下的一个秘密隔间内。在叛军攻击死星的关键时刻，韩驾驶千年鷹突然出现在[[达斯·维达|达斯·维达]]的[[钛战机|钛战机]]上空并使其偏离航道，从而帮助卢克成功击中死星主反应堆。

千年鷹在续作《[[星際大戰五部曲：帝國大反擊|星球大战V：帝国反击战]]》中出场贯穿始终，而千年鷹本身的“废物”特质造成她故障频频（主要为超空间引擎无法启动），在这一部电影中不停地给韩制造麻烦。韩和莉亚等人为了躲避帝国暴风军的追击，在无法启动超空间引擎的情况下飞入[[小行星带|小行星带]]。其后一行人自以为摆脱了帝国的追捕（实则被帝国雇佣的赏金猎人[[保巴·菲特|保巴·菲特]]跟踪），飞向[[贝斯平|贝斯平]]的云城寻找旧友藍道·卡瑞森的帮助。尽管最后韩被前来的达斯·维达抓住并被冻在结晶碳中，藍道和莉亚等人得以成功营救了前来帮助的卢克，并依靠{{lang|en|R2-D2}}修好的超空间引擎驾驶千年鷹逃离贝斯平。

在后来的《[[星際大戰六部曲：絕地大反攻|星球大战VI：绝地归来]]》中，在叛军与帝国决战的高潮阶段，藍道驾驶千年鷹领导战斗机小队摧毁了第二死星。

千年鷹最常经过的一条航线是卡塞尔线，这是一条走私犯常用的从卡塞尔星出发途径{{lang|en|Maw}}[[黑洞|黑洞]][[星团|星团]]的走私航线<ref name="SWKessel">{{cite web en|url=http://www.starwars.com/databank/location/kessel/?id=eu |title=Star Wars: Databank: Kessel (Expanded Universe) |publisher=starwars.com |accessdate=2007-06-27 |deadurl=yes |archiveurl=https://web.archive.org/web/20071013212136/http://starwars.com/databank/location/kessel/?id=eu |archivedate=2007-10-13 }}</ref>。韩在夸耀千年鷹的能力时曾说：

{{quote|
韩：快船？难道你从未听说过千年鷹？<br />
欧比旺：为什么就应该听说过呢？<br />
韩：她只需要飞12[[秒差距|秒差距]]就可以走完卡塞尔线！
}}

这里他是指千年鷹在飞行过程中可以比一般的飞船更接近{{lang|en|Maw}}黑洞星团，从而大幅缩短航程<ref name="SWKessel" />。在《星球大战IV：新希望》出版的[[DVD|DVD]]的评论音轨中卢卡斯解释说，在星战的世界中超空间飞行需要经过仔细的导航，以避免途径[[恒星|恒星]]、[[行星|行星]]及[[小行星带|小行星带]]等障碍物<ref name="ANHcommentary">Commentary track on ''A New Hope'' DVD</ref>。由于较长航程的航线往往不可能是一条直线，一般意义上的最快航程也就是尽可能避开障碍的最短航程<ref name="ANHcommentary" />。科幻作家{{lang|en|A.C.Crispin}}在星战小说《韩·索罗三部曲之反抗的黎明》中详细描述了韩·索罗引以得意的12秒差距卡塞尔线<ref>{{cite book|title=Rebel Dawn|last=Cripsin|first=A. C.|series=Han Solo Trilogy|publisher=[[Bantam|Bantam]]|isbn=0553574175}}</ref>。

==文化影响==

电视剧作家[[乔斯·惠登|乔斯·惠登]]称千年鷹是他创作短篇科幻电视剧《[[萤火虫_(电视剧)|萤火虫]]》的两大灵感之一<ref name="whedon">{{cite video|people=[[乔斯·惠登|乔斯·惠登]]|title=What's in a Firefly|year=2005}}</ref>。具有千年鷹特别的外形的飞船在《[[星际旅行VIII：第一类接触|星际旅行VIII：第一类接触]]》（也是由[[工业光魔|工业光魔]]制作特效）、《[[银翼杀手|银翼杀手]]》<ref>{{cite web en|url=http://www.brmovie.com/Locations/Falcon.htm|title=www.brmovie.com/Locations/Falcon.htm}}</ref>、《[[星河战队_(电影)|星河战队]]》以及《星球大战》的恶搞作品《{{link-en|Spaceball|Spaceball}}》中都有出现，而千年鷹本身在《[[辛普森一家|辛普森一家]]》<ref>{{cite web en|url=http://www.snpp.com/episodes/AABF06|title=www.snpp.com/episodes/AABF06|deadurl=yes|archiveurl=https://web.archive.org/web/20080509133242/http://www.snpp.com/episodes/AABF06|archivedate=2008-05-09}}</ref>、《[[厄夜怪客|厄夜怪客]]》[[OVA|OVA]]、《[[阿特米斯奇幻历险|阿特米斯奇幻历险]]》之《永恒的密码》都有提及。在电视剧《[[迷失|迷失]]》第四季的《{{link-en|回家的感觉真好|Something nice back home}}》一集中，可以看到一个千年鷹的模型玩具。

有很多玩具模型都以千年鷹作为题材，其中还包括一个[[变形金刚|变形金刚]]的版本<ref>{{cite web en|url=http://www.starwars.com/collecting/news/hasbro/news20060302.html|title=Star Wars: Collecting: Till All Are One ''Millennium Falcon''|publisher=starwars.com|accessdate=2007-06-28|date=2006-03-02}}</ref>。[[乐高|乐高]]发行的一款千年鷹模型由超过5000块积木构成，是当时发行时最大的乐高模型<ref>{{cite web en|url=http://gear.ign.com/articles/763/763948p1.html|title=Lego Ultimate Collector's Millennium Falcon Announced|last=Block|first=Gerry|publisher=ign.com||date=2007-02-12|accessdate=2007-06-28}}</ref>，此外发行过千年鷹玩具的还有{{lang|en|Kenner}}、[[孩之宝|孩之宝]]和{{lang|en|Micro Machines}}等公司。
