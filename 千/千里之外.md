《'''千里之外'''》是[[周杰伦|周杰伦]]专辑《[[依然范特西|依然范特西]]》的主打歌，和[[费玉清|费玉清]]合唱，由[[方文山|方文山]]作词、周杰伦作曲、[[林迈可|林迈可]]編曲。此歌获得[[香港|香港]]第29届[[十大中文金曲颁奖音乐会|十大中文金曲颁奖音乐会]]全国最受欢迎中文歌曲奖、[[中國大陆|中國大陆]]的2006年度[[音乐风云榜|音乐风云榜]]最佳歌曲、並且入圍[[台湾|台湾]]的[[第18届金曲奖|第18届金曲奖]]最佳年度歌曲奖。

== 翻唱与恶搞 ==
中國大陆歌手[[张杰_(大陆歌手)|张杰]]在参加《[[快乐男声|快乐男声]]》时演唱这首歌。[[央视|央视]]《[[新闻联播|新闻联播]]》著名已故主播[[罗京|罗京]]也曾在综艺节目《[[欢乐中国行|欢乐中国行]]》中翻唱这首歌<ref>{{cite web|url= http://www.ln.xinhuanet.com/xwzx/2008-01/26/content_12327883.htm|title= 千里之外》起风波 罗京人气赶超费玉清|accessdate= 2009-07-17|work= 新华网|date= 2008-01-26|deadurl= yes|archiveurl= https://web.archive.org/web/20080225145330/http://www.ln.xinhuanet.com/xwzx/2008-01/26/content_12327883.htm|archivedate= 2008-02-25}}</ref>。2007年[[东方歌舞团|东方歌舞团]]在[[重庆|重庆]]举办的歌舞晚会《争奇斗艳》上，歌手李星演唱这首歌的民歌版<ref>{{cite web |url= http://ent.sina.com.cn/y/2007-02-06/10081440454.html|title=民歌版《千里之外》战周杰伦 晚会“争奇斗艳”|accessdate=2009-07-17 |work= [[新浪|新浪]]|date= 2007年2月6日}}</ref>。此外，这首歌也遭网友[[恶搞|恶搞]]，变成[[流氓|流氓]]追债的歌曲《欠我十块》<ref>{{cite web|url= http://news.cnnb.com.cn/system/2007/11/05/005391010.shtml|title= 周董再遭整蛊 《千里之外》变《欠我十块》|accessdate= 2009-07-17|work= 中国宁波网|date= 2007-11-05|archive-url= https://web.archive.org/web/20180513152705/http://news.cnnb.com.cn/system/2007/11/05/005391010.shtml|archive-date= 2018-05-13|dead-url= yes}}</ref>。

== 同名歌曲 ==
[[马来西亚|马来西亚]]歌手[[易桀齐|易桀齐]]在2007年的专辑《一整片天空》的主打歌名为《千里之外》，与周杰伦的《千里之外》撞车，遭不少网民的非议与质疑，认为他借周杰伦[[炒作|炒作]]；但实际上，易桀齐的《千里之外》创作于2003年，两首歌在风格和歌词上没有一点雷同<ref>{{cite web |url= http://www.chinanews.com.cn/yl/ytcf/news/2007/04-19/919785.shtml|title= 新歌取名《千里之外》 易桀齐被指借周杰伦炒作|accessdate=2009-07-17 |work= 中国新闻网|date= 2007-04-19}}</ref><ref>{{cite web|url=http://fzwb.ynet.com/article.jsp?oid=18521531|title=易桀齐《千里之外》 与周杰伦歌名撞车|accessdate=2009-07-17|work=|publisher=|date=2007-01-16|deadurl=yes|archiveurl=https://web.archive.org/web/20110721172302/http://fzwb.ynet.com/article.jsp?oid=18521531|archivedate=2011-07-21}}</ref>。
