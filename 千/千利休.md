'''千利休'''（{{bd|1522年||1591年|4月21日|catIdx=S}}）是[[日本|日本]][[战国_(日本)|戰國時代]][[安土桃山時代|安土桃山時代]]著名的[[茶道|茶道]]宗師，日本人稱[[茶聖|茶聖]]。本名[[田中氏|田中]]與四郎，[[家紋|家紋]]是「利休梅」。時人把他與[[今井宗久|今井宗久]]、[[津田宗及|津田宗及]]合稱為「天下三宗匠」。

== 生平 ==
千利休的祖父為千阿彌， [[千家|以千為姓]]，為室町幕府[[足利義政|足利義政]]同朋眾一員，[[应仁之乱|应仁之乱]]時因被懷疑通敵逃亡至[[堺|堺]]，後復出服侍[[足利義尚|足利義尚]]直到義尚於近江征伐六角氏時戰死，因病回到堺養病，父親田中與兵衛(法名一忠了専)自稱新田-{里}-見田中氏一族，但無據可考，在[[和泉國|和泉國]]的堺經營魚問屋(批發)，名稱為「魚屋（ととや）」，為堺會合眾納屋十人眾其中一人。出身於「時宗」<ref>時眾，淨土宗的一個派別，語源為臨命終「時」（臨終），阿彌在戰時隨君主赴戰場負責出使敵陣及戰亡將士喪葬事務的助唸團體，平常則負責主君們的各項藝能，亦稱茶坊主或坊主。豐臣秀吉的繼父竹阿彌以及被前田利家殺害的信長兄弟拾阿彌，則為織田信秀的同朋眾。</ref>的僧侶階層但較為富裕的家庭。幼名叫与四郎（與四郎），法名則為宗易（そうえき）、居士齋號為抛筌齋（ほうせんさい）。他從小就愛好茶道，17歲拜{{tsl|ja|北向道陳}}為師，不久向{{tsl|ja|武野紹鷗}}學習「寂茶」。1540年19歲時父親與兵衛過世，同年改名為宗易，號抛筌齋。廣為世人熟知的“利休”之名，乃因1585年[[豐臣秀吉|豐臣秀吉]]在皇宮開設茶會，向[[正親町天皇|正親町天皇]]獻茶，並準備於此時就任[[關白|關白]]、[[太政大臣|太政大臣]]以及受天皇御賜「豐臣」之姓氏。因此天皇賜給「利休」之居士號，在此之前，他對外一直用千宗易的本名。

他因擅長[[茶湯|茶湯]]曾幫[[松永久秀|松永久秀]]於永祿八年舉辦多聞山茶會，三年後隨久秀成為[[織田信長|織田信長]]的茶頭。信長死後，他轉而侍奉[[豐臣秀吉|豐臣秀吉]]。1587年起他主辦秀吉發起的北野大茶湯， 成為天下第一的茶匠，聲名達到頂點，並獲三千石知行地。之後發生「黃金茶室」事件發生爭執，[[小田原之戰|小田原之戰]]時在陣城其弟子成為秀吉茶頭的[[山上宗二|山上宗二]]，於談話中觸怒秀吉而被流放，之後投靠[[後北條氏|後北條氏]]，利休被處死的真實原因不明，通說是因為作為資助[[大德寺|大德寺]]三門重造的千利休在[[山門|山門]]金毛閣上放置自己的木雕像<ref>山門之制形如闕，開三門，故亦曰三門。三門閣上必安置[[十六羅漢|十六羅漢]]之像，中安寶冠之釋迦，以月蓋長者[[善財童子|善財童子]]挾侍之（有為釋迦或觀音者），又有置[[五百羅漢|五百羅漢]]者。大德寺的閣門毀於應仁之亂，閣門上層結構為千利休捐助興建，大德寺的[[總見院|總見院]]是秀吉葬織田信長之處，故秀吉曾於千利休木像底下通過，這對武士是一種無禮的行為；[[臨濟宗|臨濟宗]]大德寺派曾出了一位[[住持|住持]]奇僧[[一休|一休]]宗純，此寺即為該宗派大本山，金毛閣的下層結構為一休和尚勸募興建，大德寺的[[茶道|茶道]]文化亦為一休引進。</ref>，二年後被[[石田三成|石田三成]]等人彈劾，千利休曾找[[細川忠興|細川忠興]]與[[德川家康|德川家康]]、[[前田利家|前田利家]]向秀吉說情，但沒有結果，被秀吉敕令回堺閉門思過（[[蟄居|蟄居]]，自我幽禁），尚未返回堺市，即被豐臣秀吉召回京都，木像被處以腰斬，利休居士被則被命令切腹，切腹時遺留辭世之句為：

}}

== 死罪的原因 ==
利休觸怒秀吉，造成死罪的原因，通說是「大德寺三門（金毛閣）改修之際的增上慢，在樓門的二階設置自身的木像，讓秀吉從下方通過」，但是也有以下各種說法，原因仍不明。另外，也有秀吉只命令蟄居，無意判處死罪，但是利休完全不做解釋、謝罪，更加怒觸怒秀吉，所以被命令切腹之說。

* 低價[[茶器|茶器]]類以高額賣出，有中飽私囊（賣僧行為）之嫌。
* 擅自攜出[[二條天皇|二條天皇]]陵之石，用作手水鉢、庭石等。
* 和秀吉在茶道的看法上對立。
* 秀吉希望納利休女兒為妾，但是利休以「不希望被認為是靠女兒出世」而拒絕，秀吉因此事懷恨。
* 秀吉企圖獨占交易，利休想保護堺的權益，因此被厭惡。
* 被捲入[[豐臣秀長|豐臣秀長]]死後豐臣政權內的政治鬥爭。
* 利休對秀吉入侵[[朝鮮|朝鮮]]的決定始終堅持以反對的態度來表示自己的不滿。
* 利休所修行的南宗寺和德川家康有密切的關係，秀吉懷疑家康派間諜在茶中下毒圖謀毒害他。
