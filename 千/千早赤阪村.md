'''千早赤阪村'''（{{jpn|j=千早赤阪村|hg=ちはやあかさかむら|rm=Chihayaakasaka mura|f=y}}）是位於[[日本|日本]][[大阪府|大阪府]]東南部的行政區劃，現在是大阪府內人口最少，也是唯一的村級行政區劃。<ref name="Brief">{{Cite web| url=http://www.vill.chihayaakasaka.osaka.jp/menu-s-murashoukai.html| title=村的介紹| publisher=千早赤阪村公所| accessdate=2018-06-13| language=ja}}</ref>

大阪府最高峰，也是近畿地區的著名登山景點[[金剛山_(金剛山地)|金剛山]]位於其東南部，周邊的山林風光，加上與[[楠木正成|楠木正成]]相關的景點，使得觀光業成為本村的主要產業<ref name="Brief"/>；此外村內的主要農產為花卉及[[香菇|香菇]]栽種，而在江戶時代至1950年代期間，這裡也曾是著名的{{Tsl|ja|高野豆腐|凍豆腐}}產地。

== 歷史 ==

在令制國時代屬於[[河內國|河內國]][[石川郡_(大阪府)|石川郡]]，被稱為武神的[[南北朝时代_(日本)|南北朝时代]]著名武將[[楠木正成|楠木正成]]於13世紀末出生於此，此處後來也成為楠木正成支持[[後醍醐天皇|後醍醐天皇]]對抗北朝的據點。

在[[江戶時代|江戶時代]]，現在的千早赤阪村大部分區域為{{tsl|ja|下館藩}}的領地，另有部分區域為幕府的直屬領；19世紀[[明治維新|明治維新]]後，這些區域曾先後隸屬大坂裁判所司農局、大阪府司農局、大阪府南司農局、[[河內縣_(日本)|河內縣]]、[[堺縣|堺縣]]、[[五條縣|五條縣]]、下館縣、[[茨城縣|茨城縣]]，1872年第一次府縣整併後，全數區域被併入堺縣，但到了1881年堺縣又被併入大阪府，此後此區成為大阪府的轄區。

1889年日本實施[[町村制|町村制]]，現在的千早赤阪村轄區在當時北半部屬於{{tsl|ja|千早村}}，南半部屬於{{Tsl|ja|赤阪村}}；1956年兩個行政區劃合併為'''千早赤坂村'''。

2002年後，千早赤坂村曾先後討論過與[[富田林市|富田林市]]或[[河內長野市|河內長野市]]的合併案，但由於未能獲得村議會的通過，最終未加入任何合併案。但由於人口持續外流，影響村的財政<ref>{{Cite web| url=http://www.vill.chihayaakasaka.osaka.jp/soumu-gyouzaiseikaikaku.html| title=行政改革の推進| publisher=千早赤阪村公所| accessdate=2018-06-13| language=ja}}</ref>，於2014年被日本政府列為{{tsl|ja|過疎地域|過疏地區}}<ref>{{Cite web| url=http://www.soumu.go.jp/main_content/000476769.pdf#search=%27%E5%88%A9%E6%A0%B9%E7%94%BA+%E9%81%8E%E7%96%8E%E5%9C%B0%E5%9F%9F+%E6%8C%87%E5%AE%9A%27| title=平成２９年３月３１日付けで以下の市町村が公示されました（４月１日施行）。| publisher=總務省| accessdate=2018-06-12| language=ja| archive-url=https://web.archive.org/web/20170829080246/http://www.soumu.go.jp/main_content/000476769.pdf#search=%27%E5%88%A9%E6%A0%B9%E7%94%BA+%E9%81%8E%E7%96%8E%E5%9C%B0%E5%9F%9F+%E6%8C%87%E5%AE%9A%27| archive-date=2017-08-29| dead-url=yes}}</ref>，成為大阪府裡唯一的過疏地區。

== 交通 ==

村内無鐵路通過，對外交通需仰賴{{tsl|ja|金剛自動車}}或{{tsl|ja|南海バス|南海巴士}}的客運前往位於[[河內長野市|河內長野市]]的[[南海電氣鐵道|南海電氣鐵道]][[高野線|高野線]][[河內長野車站|河內長野車站]]或位於[[富田林市|富田林市]]的[[近畿日本鐵道|近畿日本鐵道]][[長野線_(近畿日本鐵道)|長野線]][[富田林車站|富田林車站]]，再轉乘鐵路。

過去在1927年曾有業者規劃興建自當時的{{Tsl|ja|川上 (河内長野市)|川上 (河內長野市)|川上村}}（現已併入河內長野市）至千早的「金剛登山鐵道」，雖已取得政府核准，但最終並未完成此條鐵路路線。

== 觀光資源 ==

由於日本14世紀[[南北朝时代_(日本)|南北朝时代]]著名武將[[楠木正成|楠木正成]]出生於此，此地有許多與其有關的地點發展為觀光景點，包括楠木正成對抗北朝時建立的防守據點{{tsl|ja|楠木七城}}的[[千早城|千早城]]遺跡、{{Tsl|ja|上赤坂城}}遺跡、{{Tsl|ja|下赤坂城}}遺跡，建於千早城遺址祭祀楠木正成的{{tsl|ja|千早神社}}。

另外因為[[金剛山_(金剛山地)|金剛山]]擁有數量不遜於富士山的登山者，千早赤阪村興建了{{tsl|ja|金剛山ロープウェイ|金剛山纜車}}，現在也是全日本唯一村營的纜車，同時在其山上的金剛山站旁也有村營旅館－香楠莊。

== 教育 ==
村內無高等教育學校及高中，僅有{{tsl|ja|千早赤阪村立中学校|千早赤阪村立中學校}}一間中學及{{Tsl|ja|千早赤阪村立赤阪小学校|千早赤阪村立赤阪小學校}}和{{tsl|ja|千早赤阪村立千早小吹台小学校|千早赤阪村立千早小吹台小學校}}兩間小學。
