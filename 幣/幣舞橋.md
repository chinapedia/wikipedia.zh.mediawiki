'''幣舞橋'''是位於[[日本|日本]][[北海道|北海道]][[釧路市|釧路市]]跨越[[釧路川|釧路川]]的[[橋梁|橋梁]]，位於市區[[釧路車站|釧路車站]]前的主要道路北大通上，也是釧路川最下游的橋梁。橋南側的「幣舞圓環」則是釧路國道（[[國道38號|國道38號]]）的終點和根室國道（[[國道44號|國道44號]]）的起點。

「幣舞」的名稱來自本地的[[阿伊努語|阿伊努語]]舊地名「nusa-oma-i」，意思是祭祀神的地方。<ref name="釧路開建">{{cite web|url=http://www.ks.hkd.mlit.go.jp/road/etc/alacarte/alacarte3.html|title=幣舞橋アラカルト|publisher=釧路開發建設部|accessdate=2016-02-09|language=ja|deadurl=yes|archiveurl=https://web.archive.org/web/20160216074507/http://www.ks.hkd.mlit.go.jp/road/etc/alacarte/alacarte3.html|archivedate=2016-02-16}}</ref>。

== 歴史 ==
現在幣舞橋的位置最早在1889年曾建了一座名為「愛北橋」的木橋，為來自[[名古屋|名古屋]]的公司「愛北物產」所興建，長度207公尺，路面寬3.6公尺，在當時是北海道最長的木橋；由於是私人興建，當時要通過橋梁需要收費。<ref name="釧路開建"/>

1898年愛北橋損毀，在1900年由政府在此興建了新的木橋，長度203公尺，路面寬4.2公尺，新的橋梁以釧路川南岸的地名「幣舞」來命名為「幣舞橋」；但是由於釧路川中有上游漂下來的浮木，且冬季河水結冰後飄下來的浮冰，橋梁在1909年損壞。<ref name="釧路開建"/>

1909年立刻又重建了新的木橋，長度仍然是全長203公尺，寬度略為增加為4.5公尺，但是僅支撐六年，在1915年再度因為流木和流冰的衝擊而損壞。<ref name="釧路開建"/>

重建的木橋也立刻在1915年完成，長度201公尺，寬度增加為7.2公尺；但由於先前多次的損壞，同時也開始討論應興建一座不用擔心被流木和流冰衝擊的永久性橋梁。<ref name="釧路開建"/>

採用[[鋼筋混凝土|鋼筋混凝土]]興建的橋梁於1924年開始興建，1928年完工並啟用，為北海道第一個採用鋼筋混凝土建的橋梁，全長113公尺，寬度18.3公尺，為雙向四車道並規劃了人行道。由於採用歐式的拱型設計，並在橋梁兩側立了四個大理石柱，讓橋樑的外型受到好評，當時和[[札幌市|札幌市]]的{{tsl|ja|豊平橋|豐平橋}}、[[旭川市|旭川市]]的{{tsl|ja|旭橋 (旭川市)|旭橋 (旭川市)|旭橋}}被並列為「北海道三大名橋」。<ref name="釧路開建"/><ref>{{cite web |url=http://pucchi.net/hokkaido/closeup/bridge.php |title=你知道北海道三大名橋? 豐平橋、旭橋、幣舞橋 |date=2008-03-13 |work={{tsl|ja|北海道ファンマガジン|北海道粉絲雜誌}} |publisher=PNG Office |accessdate=2016-02-09 |language=ja}}</ref>

由於橋梁老化且車道數不足影響交通，1976年開始再次進行改建，改建後車道增加為雙向共六線，同時加入了四件雕刻作品「道東四季之像」。<ref name="釧路開建"/>

=== 年表 ===
* 1889年：由愛北物産興建的收費橋梁「愛北橋」完成。
* 1898年：愛北橋損毀。
* 1900年12月：由政府興建的第一代「幣舞橋」（木橋）完成。
* 1909年：幣舞橋損毀，同年第二代「幣舞橋」（木橋）完成。
* 1915年：第二代幣舞橋損毀，同年第三代「幣舞橋」（木橋）完成。
* 1924年：第四代橋梁新建工程開始。
* 1928年11月3日：第四代「幣舞橋」（鋼筋混凝土）完成。
* 1975年7月1日：為橋梁新建工程，啟用替代用臨時橋。<ref>{{cite web|url=http://thesis.ceri.go.jp/db/files/GR0002200138.pdf|title=關於幣舞橋架換伴隨交通處理|author=關谷強|coauthors=煤賀清隆、高橋毅|format=PDF|publisher={{tsl|ja|土木研究所}} 寒地土木研究所|date=1977-02-25|language=ja|accessdate=2016-02-09}}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。
* 1976年11月26日：現在的第五代「幣舞橋」完成。
* 1977年5月3日：「道東四季之像」完成並揭幕。<ref name="アートマップ">{{cite web |url=http://www.kushiro-artmu.jp/pamphlet/09artmap2.pdf |title=釧路藝術地圖 |format=PDF |publisher={{tsl|ja|北海道立釧路芸術館|北海道立釧路藝術館}} |accessdate=2016-02-09| language=ja}}</ref><ref name="北の交差点">{{cite journal |url=http://www.rmec.or.jp/kousaten/10pdf/103436.pdf |title=詩歌文學旅次登場的霧都・釧路的象徵「幣舞橋」 |format=PDF |journal=北之交差点 |volume=10 AUTUMN-WINTER |year=2001 |publisher=北海道道路管理技術中心 |page=34-36 |author=三浦宏 |accessdate=2016-02-09 |language=ja |deadurl=yes |archiveurl=https://web.archive.org/web/20160304071610/http://www.rmec.or.jp/kousaten/10pdf/103436.pdf |archivedate=2016-03-04 }}</ref>。

== 道東四季之像 ==

橋上的四件雕刻作品「道東四季之像」分別表現出春夏秋冬<ref name="アートマップ"/><ref name="観光コンベンション協会">{{cite web|url=http://www.kushiro-kankou.or.jp/tourism/nusamai-bridge.html|title=幣舞橋|publisher=釧路觀光會議協會|accessdate=2016-02-09|language=ja|deadurl=yes|archiveurl=https://web.archive.org/web/20150214003226/http://www.kushiro-kankou.or.jp/tourism/nusamai-bridge.html|archivedate=2015年2月14日|df=}}</ref>，為市民發起並集資完成的作品。<ref name="北の交差点"/><ref>{{cite web |url=http://kenmane.kensetsu-plaza.com/bookpdf/135/at_01.pdf |title=釧路的象徵 幣舞橋 |format=PDF |work=土木紀行 |publisher=建設管理技術 |accessdate=2016-02-09| language=ja}}</ref>。

* 春之像 作者：[[舟越保武|舟越保武]]　主題：「若葉が萌えいずる雪解けの季節」
* 夏之像 作者：{{tsl|ja|佐藤忠良}}　主題：「さわやかな風を受けて羽ばたく若々しさ」
* 秋之像 作者：{{tsl|ja|柳原義達}}　主題：「迫りくる厳しい冬に立ち向かう精神と緊張感」
* 冬之像 作者：{{tsl|ja|本郷新|本鄉新}}　主題：「寒さと冬をはねのけて春を待ち望む心」

== 周邊 ==
[[File:Nusamai_Bridge_Fisherman_Wharf_Kushiro.JPG|thumb]]
位處釧路市的臨海區，西北側為[[釧路漁人碼頭MOO|釧路漁人碼頭MOO]]，南側為幣舞圓環，與[[北海道道25號釧路港線|北海道道25號釧路港線]]和{{tsl|ja|北海道道113号釧路環状線|北海道道113號釧路環狀線}}相接，圓環南側也是進入幣舞公園的入口，公園內可以俯瞰幣舞橋和[[釧路港|釧路港]]。
