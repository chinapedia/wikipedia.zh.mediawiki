'''蔗糖廠'''（sugar cane mill），是用[[甘蔗|甘蔗]]製造[[砂糖|砂糖]]或[[白糖|白糖]]的[[工廠|工廠]]。現代化的蔗糖廠內部擁有一連串的機器設備，用來壓榨甘蔗，並以榨出的蔗汁製糖。

== 製程 ==
[[File:Sugarcanemill-2.JPG|thumb]]
用甘蔗製造砂糖有一連串的程序：
# 收取甘蔗並缷貨
# 甘蔗準備（切成條狀）
# 榨汁
# 除去雜質
# 煮汁
# 結晶
# 分離結晶糖與蔗汁（使用離心機）
# 烘乾
# 包裝出貨
