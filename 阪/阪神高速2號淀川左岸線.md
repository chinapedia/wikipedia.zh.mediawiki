== 歷史 ==
* 1994年4月28日 : 北港系統交流道 - 島屋出入口間通車。
* 2001年 2月26日 : 環球影城出口通車。
* 2001年（平成13年）3月30日 : [[島屋東入口|島屋東入口]]通車。
* 2011年（平成23年）9月29日 : 工程中[[正蓮寺川|正蓮寺川]]水流入隧道內<ref>{{cite web | url=http://kenplatz.nikkeibp.co.jp/article/const/news/20111012/553317/ | title= 淀川左岸線之開挖工程擋土牆崩壊 | publisher= 日經BP | date=2011年9月30日 | accessdate=December 4, 2011}}</ref><ref>{{cite web | url=http://www.hanshin-exp.co.jp/company/topics/2011-1011.html | title= 淀川左岸線事故之狀況等 | publisher=[[阪神高速道路|阪神高速道路]] | date=2011年10月12日 | accessdate=December 4, 2011}}</ref>。
* 2011年（平成23年）12月1日 : 島屋東入口'''廢止'''。
* 2013年（平成25年）2月21日 : 島屋出入口 - 海老江系統交流道間通車日時及交流道名稱發表（北港東→淀川左岸舞洲、正蓮寺川與大開未變更）。
* 2013年（平成25年）5月25日 : 島屋出入口 - 海老江系統交流道間通車。

== 淀川左岸線延伸部 ==
淀川左岸線延伸部是指從大阪市北區豐崎至[[門真市|門真市]]薭島（{{lang|ja|ひえじま}}）（[[近畿自動車道|近畿自動車道]]・[[第二京阪道路|第二京阪道路]] [[門真系統交流道|門真系統交流道]]），總長大約10[[公里|公里]]的[[快速道路|快速道路]]建設[[計畫|計畫]]<ref name="pamphlet">{{Cite web|date=2013年1月 |url=http://www.kkr.mlit.go.jp/naniwa/14/pdf/01.pdf |format=PDF |title=大阪都市再生環状道路 淀川左岸線延伸部|publisher=[[国土交通省|国土交通省]]・大阪府・大阪市 |accessdate=2013-11-01}}</ref>。本項建設未來將可以與已經通車的[[近畿自動車道|近畿自動車道]]、[[阪神高速4號灣岸線|灣岸線]]、事業中的[[阪神高速6號大和川線|大和川線]]、'''淀川左岸線'''等道路共同組成總長約60公里的[[大阪都市再生環狀道路|大阪都市再生環狀道路]]，期待可以解除大阪都心部的慢性[[交通堵塞|交通堵塞]]問題<ref name="pamphlet"/>。淀川左岸線延伸部被以'''大阪門真線'''的名義指定為[[地域高規格道路|地域高規格道路]]候補路線。

這個計畫以公眾參與的方式進行，以下的4案可供比較檢討<!--リンク切れ:<ref name="teigen">{{Cite web|date=2006年12月15日 |url=http://www.kkr.mlit.go.jp/road/kansen/kanjo/04/pdf/a/a_teigen.pdf |format=PDF |title=淀川左岸線延伸部有識者委員会 提言|publisher=淀川左岸線延伸部有識者委員会 |accessdate=2013-11-1}}</ref>--><ref>{{Cite web |url=http://www.kkr.mlit.go.jp/road/kansen/kanjo/04/a.htm |title=大阪都市再生環状道路 |publisher=国土交通省[[近畿地方整備局|近畿地方整備局]] |accessdate=2013-11-01}}</ref>。
; A案
: 大半利用[[都市計畫道路|都市計畫道路]]用地建設[[隧道|隧道]]。
; B案
: 與A案同樣的路線，近豐崎側的部分採用隧道，近薭島側的部分則是[[高架道路|高架]]。
; C案
: 興建深度超過40公尺的隧道，不必用地補償。路線距離可縮減。
; D案
: 豐崎側利用[[阪神高速12号守口線|阪神高速12号守口線]]與森小路線，森小路出口以東則以隧道延伸。

以上的4案在納入沿線居民的意見作為參考並且檢討之後，淀川左岸線延伸部有識者委員會將A案與C案列為優先方案。[[2006年|2006年]][[12月15日|12月15日]]提出了以隧道為主體構造，並且可以活用大深度地下的建言。隧道對於沿道環境的影響較小，且大深度地下不需要用地補償，對於工事期間的縮短有幫助。另外，沿線便利性相當高的中間（大約是在與[[国道479号線|国道479号線（大阪内環状線）]]交叉的附近）正在努力爭取設置[[交流道|交流道]]。<!--リンク切れ:<ref name="teigen"/>-->

2013年1月，[[環境影響評估|環境影響評估]]的手續經啟動<ref>{{Cite web|url=http://www.pref.osaka.jp/sokei/yodogawasagan/|title=（暫稱）淀川左岸線延伸部|publisher=大阪府都市整備部綜合計畫課設施計畫集團|accessdate=2013-11-01}} {{Cite web|url=http://www.city.osaka.lg.jp/toshikeikaku/page/0000198452.html|title=（暫稱）淀川左岸線延伸部　環境影響評価方法書及要約書縦覧，説明会及意見書的領受|publisher=大阪市都市計画局計画部都市計画課|accessdate=2013-11-01|deadurl=yes|archiveurl=https://web.archive.org/web/20131101191724/http://www.city.osaka.lg.jp/toshikeikaku/page/0000198452.html|archivedate=2013-11-01}}</ref>。
