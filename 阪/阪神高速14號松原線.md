'''阪神高速14號松原線'''（{{lang|ja|阪神高速14号松原線、はんしんこうそく14ごうまつばらせん、Route 14 Matsubara Line}}）是[[日本|日本]][[阪神高速道路|阪神高速道路]]路線，[[大阪府|大阪府]][[大阪市|大阪市]]的環狀線分歧（夕陽丘附近）至大阪府[[松原市_(日本)|松原市]][[松原系統交流道|松原系統交流道]]。急彎加上跨越鐵道造成道路線形不佳，是事故發生率高的路線。另外，與阪和道和西名阪道連接也使的本道路交通量相當大，在平日的早晨與黃昏的時間帶時，上行線經常發生起於環狀線合流處的[[交通堵塞|交通堵塞]]。

== 路線名 ==
* 大阪府道高速大阪池田線（環狀線至[[阿倍野入口|阿倍野入口]]附近）
* 大阪府道高速大阪松原線（阿倍野入口至松原系統交流道）

== 歷史 ==
* 1970年3月13日 : 公園北分歧（環狀線）- 阿倍野入口間通車。
* 1980年3月1日 : 阿倍野入口 - 西名阪道松原交流道間通車，'''全線通車'''（但是當時的松原交流道僅阪神高速與西名阪道直通，與一般道連接則須利用三宅出入口）。
* 1980年4月17日 : 喜連瓜破出入口啟用。
* 1981年10月22日 : 駒川出入口啟用。
* 1986年3月31日 : 文之里出口啟用。
* 1986年12月25日 : 平野出入口啟用。
* 1987年5月11日 : 文之里入口啟用。
* 1989年3月29日 : 松原系統交流道內與阪和道連接的匝道啟用。
* 1992年2月26日 : 大堀出入口啟用。
* 2012年9月10日 : 三宅迷你停車區廢止。
* 2013年3月21日 : 三宅系統交流道，與6號大和川線相連。三宅系統交流道 - 松原系統交流道間拓寬為雙向6車道。
* 2015年3月29日 : 松原系統交流道內至近畿道之聯絡道通車<ref>[http://www.hanshin-exp.co.jp/topics2/1423802244F.pdf 松原系統交流道（北西橫渡線）與4號灣岸線三寶入口（關西機場方向）通車] 阪神高速道路㈱　2015年2月15日閲覧</ref>。
