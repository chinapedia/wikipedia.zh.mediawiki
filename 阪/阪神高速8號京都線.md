'''阪神高速8號京都線'''（{{lang|ja|阪神高速8号京都線}}、{{lang|ja|はんしんこうそく8ごうきょうとせん}}、{{lang|en|Route 8 Kyoto Line}}）是[[日本|日本]][[阪神高速道路|阪神高速道路]]路線，通過[[京都府|京都府]][[京都市|京都市]]。

== 概要 ==
本道路是京都市第一條'''[[都市高速道路|都市高速道路]]'''，2011年3月時全線通車也是京都市唯一的都市高速道路。本道路被指定為[[地域高規格道路|地域高規格道路]]的計畫路線「[[京都高速道路|京都高速道路]]」。通車後，行駛南北走向的國道1號經過[[南區_(京都市)|南區]]以及[[伏見區|伏見區]]時，周邊塞車狀況已大幅度好轉<ref>{{PDFlink|[http://www.hanshin-exp.co.jp/topics2/1209515676F.pdf 阪神高速８号京都線・第二京阪道路上鳥羽出入口～巨椋池インターチェンジ開通後の交通状況について]}} 阪神高速株式会社</ref>。

=== 路線名 ===
本路線正式名稱為京都市道高速道路1號線及和京都市道高速道路2號線<ref>[http://web.gis.survey.ne.jp/Kyoto/ninteirosen/ 京都市認定路線網図提供システム] - 京都市（2012年9月8日閲覧）</ref>，[[道路法|道路法]]上屬於[[市町村道|市道]]。路線稱呼、隧道、各出入口名稱在2007年8月28日決定。

; 京都市道高速道路1號線
: 京都市[[山科區|山科區]]西野山櫻之馬場町（[[山科出入口|山科出入口]]）至同市[[伏見區|伏見區]]深草西川原町（[[鴨川東出入口|鴨川東出入口]]）總長為2.8公里的路段。基本計畫的路線名為新十條通。

; 京都市道高速道路2號線
: 京都市伏見區深草西川原町（鴨川東出入口）至同區向島大黒（[[第二京阪道路|第二京阪道路]]連接部）總長7.3公里的路段。基本計畫的路線名為油小路線。
此外，2008年上述兩路段通車時連接此兩路段之鴨川東出入口 - 上鳥羽出入口總長1.9公里的油小路線斜久世橋路段仍在施工。

=== 獨立通行費設定 ===
本路線不直接連接其他阪神高速道路路線，因此通行費設定（收費圏）也獨立於其他地方。因此，往來兩地區可經由一般國道或者利用其他收費道路(需另繳通行費，如第二京阪道路、[[近畿自動車道|近畿自動車道]]、京滋繞道、名神高速道路）。

== 道路規格 ==
* 道路等級 : 第2種第2級
* 設計速度 : 60km/h
** 出入口: 40km/h
* 車道數 : 4車道（鴨川東出入口-上鳥羽出入口間為暫定2車道）

== 沿革 ==
2008年（[[平成|平成]]20年）1月19日[[上鳥羽出入口|上鳥羽出入口]]～[[第二京阪道路|第二京阪道路]]連接部區間（油小路線）通車、同年6月1日[[山科出入口|山科出入口]]～[[鴨川東出入口|鴨川東出入口]]區間（新十条通、[[稻荷山隧道_(京都府)|稻荷山隧道]]）通車。2011年3月27日鴨川東出入口～上鳥羽出入口區間（油小路線斜久世橋區間）通車、全線正式完工通車。
