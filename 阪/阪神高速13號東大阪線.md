'''阪神高速13號東大阪線'''（{{lang|ja|阪神高速13号東大阪線}}／{{lang|ja|はんしんこうそく13ごう ひがしおおさかせん}}，{{lang|en|Route 13 Higashi-Osaka Line}}）是[[日本|日本]][[阪神高速道路|阪神高速道路]]路線，從[[大阪府|大阪府]][[大阪市|大阪市]]至[[東大阪市|東大阪市]][[第二阪奈收費道路|第二阪奈收費道路]]。

== 概要 ==
幾乎全線都在中央大通上方。[[阪神高速1號環狀線|1号環狀線]]内側至船場付近（[[信濃橋出入口|信濃橋出入口]]、[[本町出口|本町出口]]間）則是建設在船場中心上方空間。

整條道路幾乎都是直線但多有起伏。

出口多從左側車道岔出，但是東行線中野出入口和水走出入口的出口引道卻被設在右側車道。因此這兩處常常因車流交織而塞車。

== 路線名 ==
（與阪神高速16號大阪港線合併）大阪府道高速大阪東大阪線

== 歷史 ==
* 1970年3月8日 : 西横堀～法圓坂通車。
* 1974年11月2日 : 森之宮～長田通車。
* 1978年3月25日 : 法圓坂～森之宮通車（因為挖掘到難波宮遺跡而延後通車。此外為保護遺跡本路段為平面道路）<ref>[https://www.hanshin-exp.co.jp/company/torikumi/renewal/koshin/koshin02.html 阪神高速的大規模更新 13號東大阪線 法圓坂付近 ～1978年（昭和53年）通車～]</ref>。
* 1983年12月7日 : 長田～東大阪系統交流道通車，連接近畿自動車道。
* 1987年3月4日 : 東大阪系統交流道～水走通車。
* 1997年4月23日 : 水走～西石切通車，同時'''全線通車'''。連接第二阪奈收費道路。
