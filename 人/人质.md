'''人質'''，即是被俘虜者擄走的人。這詞最早的用意是指在戰爭裡，某一方將敵人的人捉走，然後向敵方提出條件，以方便威脅對方。但到了近代，人質一詞意即被犯罪者[[綁票|綁架]]，接著強求人質的[[親屬|親屬]]，[[雇主|雇主]]或甚至該[[政府|政府]]做出某些事情，或控制其舉動，尤其是在威脅的態度下進行。在特別的情況下，例如綁架者提出的限期已到（如發出[[最後通牒|最後通牒]]來通知），而對方沒有回應或拒絕其要求，綁架者將會伤害人質。

== 政治 ==
{{main|質子 (東亞政治)}}
在[[中國|中國]][[春秋戰國|春秋戰國]]時代，[[諸侯|諸侯]]之間為了互相取信，會互相交換人質，稱為'''質子'''。通常是以諸侯的[[兒子|兒子]]作為質子質押於別國，[[秦異人|秦異人]]就是一例子。除了以平等方式互相交換人質外，亦有當弱國有求於強國時，就出現單方面送出質子的情況。

互換質子的行為在[[日本|日本]]亦有出現。

== 非法地綁架人質 ==
綁架或擄走人質的行為，已经在今日被公眾視為是一種[[刑事|刑事]][[犯罪|罪行]]，或是[[恐怖主義|恐怖主義]]的行為。「abductee（被綁架者）」一詞出現於1970年代。人質通常會被武裝的恐怖份子或者罪犯用绳子进行捆绑约束，绑住手脚。有时候会用胶带封住嘴巴，用布带蒙住眼睛，并且关押在一座大廈或一輛車子裡，我們稱這類情況為「hostage crisis（[[人質|人質]][[危機|危機]]）」。通常在綁架人質的事件裡，綁架者往往會提出[[贖金|贖金]]或者其他要求作为交換條件。

== 拯救人質 ==
各國對于如何拯救人質，均有一套相關的原則和方案。首先得确定人质和劫匪的大概位置，在人質的安全獲得保障下，大多以談判方式和劫持者談判，以達到雙方目的。警方目的多為䆁放人質，而劫匪多為要求大額金錢和提供相關車輛以逃離警方追捕，也有的是出于某些[[政治|政治]]目的。

在人質安全有威脅下、劫持者情緒失控或談判破裂時，警方多會派出接受過[[室內近身作戰|室內近身作戰]]和人質拯救訓練的[[特種警察部隊|特種警察部隊]]進行攻堅，拘捕劫持者和拯救人質。於必要時，例如於特警隊員和人質安全受到严重威脅時，會選擇开枪击伤或者撃斃劫持者。
