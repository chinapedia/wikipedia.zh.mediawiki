'''人民法院信息技术服务中心'''（最高人民法院信息中心），位于[[北京市|北京市]][[东城区_(北京市)|东城区]][[东交民巷|东交民巷]]27号，是[[中华人民共和国最高人民法院|中华人民共和国最高人民法院]]直属[[事业单位|事业单位]]<ref name=zhaolu>{{Cite web |url =http://www.court.gov.cn/zixun-xiangqing-38412.html  |title = 人民法院信息技术服务中心2017年应届毕业生招录公告 |author =  |date = 2017-03-23 |publisher = 中华人民共和国最高人民法院 |language =  |accessdate =  |quote =  }}</ref>。

==简介==
2013年10月15日，人民法院信息技术服务中心成立，最高人民法院常务副院长[[沈德咏|沈德咏]]出席成立仪式并讲话<ref>{{Cite web |url = http://www.law-lib.com/fzdt/newshtml/yjdt/20131016090012.htm |title =  最高人民法院成立信息技术服务中心|author =  |date = 2013-10-16 |publisher = 法律图书馆 |language =  |accessdate =  |quote =  }}</ref>。人民法院信息技术服务中心主要负责全国法院“天平工程”项目建设；全国法院信息化建议需求管理、顶层设计、规划计划、考核评价、信息化宣传；全国法院信息系统研制建设；全国法院数据资源建设和管理；全国法院信息系统安全保障设施和机制建设；全国法院信息系统运行、维护、管理保障；全国法院信息网络统筹设计和管理维护等工作<ref name=zhaolu/><ref>{{Cite web |url = http://www.gjsy.gov.cn/cxzl/ |title = 人民法院信息技术服务中心 |author =  |date =  |publisher =事业单位在线 |language =  |accessdate =2017-04-05  |quote =  }}</ref>。
