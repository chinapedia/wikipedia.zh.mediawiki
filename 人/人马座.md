'''人马座'''（{{lang-la|'''Sagittarius'''}}，[[天文符号|天文符号]]：♐），又稱'''射手座'''，是一个[[天球|南天]][[黄道带|黄道带]][[星座|星座]]，面积867.43平方度，占全天面积的2.103%<ref>{{cite web |title=人马座是什么星座 星座详情大揭秘 |url=http://www.meilele.com/article/zhuangxiu-fengshui/36310.html |website=美樂樂}}</ref>，在全天88个星座中，[[星座面积列表|面积排行]]第十五。人马座中亮于5.5等的恒星有65颗，最亮星为[[箕宿三|箕宿三]]（人马座ε），视星等为1.85。每年7月7日子夜人马座中心经过[[中天_(天文学)|上中天]]。

==名称==
人马座又称射手座。“射手座”是从[[拉丁语|拉丁语]]Sagittarius意译而来，sagittarius一词[[拉丁语|拉丁语]]中意为“持箭者”，箭的[[拉丁语|拉丁语]]是sagitta，[[天箭座|天箭座]]即Sagitta；相似情况也可以对比[[拉丁语|拉丁语]]中的水aqua与持水者aquarius，[[宝瓶座|宝瓶座]]/[[水瓶座|水瓶座]]即Aquarius。人马座原为[[巴比伦天文学|巴比伦天文学]]中黄道星座之一，其原型为[[苏美尔|苏美尔]]神祇[[内尔伽勒|内尔伽勒]]（Nergal），形态为拉弓射箭的带翼人马。<ref name="Rogers" /> 黄道星座随后被希腊的星座系统沿用，但射手座在[[希腊神话|希腊神话]]代表[[半人马|人马]][[喀戎|喀戎]]，传说其发明了射箭。

== 特点 ==
人马座是[[夏季|夏季]]夜空中最大、最显著的[[星座|星座]]之一，它西接[[天蝎座|天蝎座]]、東連[[摩羯座|摩羯座]]，北面是[[蛇夫座|蛇夫座]]、[[盾牌座|盾牌座]]和[[巨蛇座|巨蛇座]]，南边则是一系列小型星座，如[[望远镜座|望远镜座]]、[[显微镜座|显微镜座]]、[[南冕座|南冕座]]等。

[[File:Sagittarius_constellation_detail_long_exposure.jpg|left]]

人马座并不难认，因为它主要的星排列得像一个茶壶：[[箕宿二|箕宿二]]（δ）、[[箕宿三|箕宿三]]（ε）、[[斗宿六|斗宿六]]（ζ）及[[斗宿三|斗宿三]]（φ）组成壶身；[[斗宿二|斗宿二]]（λ）为壶盖；[[箕宿一|箕宿一]]（γ）为壶嘴；[[斗宿四|斗宿四]]（σ）与[[斗宿五|斗宿五]]（τ）为壶柄。另外其中六顆星排列相-{斗}-杓：[[斗宿一|斗宿一]]（μ）、斗宿二（λ）、斗宿三（φ）、斗宿四（σ）、斗宿五（τ）和斗宿六（ζ），在古代中国稱為[[南斗六星|南斗六星]]，也就是[[斗宿|斗宿]]名稱的來源。 

在人马座天区，[[银河|银河]]达到最大的亮度和宽度，银河系的中心位于射电源人马座A*附近。

== 神话与历史 ==
在[[希腊神话|希腊神话]]中，射手座的箭对着位于它前方的[[天蝎座|天蝎座]]的胸，射手座在天上是为了给去取[[金羊毛|金羊毛]]的[[伊阿宋|伊阿宋]]指路，[[古希腊人|古希腊人]]将射手座与[[萨杜恩|萨杜恩]]（半人半[[羊|羊]]）以及[[克洛托斯|克洛托斯]]联系起来。[[罗马|罗马]]人却将他认为是优雅而智慧的[[喀戎|喀戎]]，这导致人们常常把它与南天的[[半人马座|半人马座]]混淆，然而这两种[[生物|生物]]有一个明显的区别，射手座是一名猎手，这可以追溯到[[美索不达米亚|美索不达米亚]]的箭神涅伽尔（Nergal），而他与暴躁的[[战神|战神]]和[[火神|火神]]伊拉（Irra）有关。

== [[深空天体|深空天体]] ==
從[[地球|地球]]看來，[[银河系|本银河系]]的中心位于人马座，雖然银心被人马臂上的[[星云|星云]]和尘埃带所遮挡，但是人马座的银河仍是非常浓密，中间还有很多明亮的[[星团|星团]]。

这个星座中的天体主要是银河深处的[[宇宙|宇宙]]天体，包括[[发射星云|发射星云]]和[[暗星云|暗星云]]，[[疏散星团|疏散星团]]和[[球状星团|球状星团]]以及[[行星状星云|行星状星云]]。人马座有多达15个[[梅西耶天体|梅西耶天体]]——这是所有星座中最多的。其中很多用[[双筒望远镜|双筒望远镜]]就可以观测到。

与银河系中心有关的[[人马座A|人马座A]]是一个复杂的[[无线电|无线电源]]，[[天文学家|天文学家]]相信它或许包含了一个超大[[质量|质量]]的[[黑洞|黑洞]]。

=== [[梅西耶天体|梅西耶天体]] ===
[[File:M8HunterWilson.jpg|thumb]]
* [[礁湖星云|礁湖星云]]（M8）：位于斗宿二附近，这个明亮的发射星云被一条遮掩物质形成的暗带切开了。用双筒望远镜容易看到，这个星云中包含了一个稀疏的疏散星团，即[[NGC_6530|NGC 6530]]。
* [[奥米加星云|奥米加星云]]（M17）：接近[[盾牌座|盾牌座]]边界的M17是天空中最明亮的发射星云之一，它亦称天鹅星云、马蹄星云，用双筒望远镜就很容易看见它。这是因为其中最明亮的一部分让人想起在湖面游动的一只[[天鹅|天鹅]]的身体。用一架8英寸（20厘米）或更大的望远镜装上[[星云滤光器|星云滤光器]]可以去除几乎充满这个区域的较暗的星云状物质。
* [[M18|M18]]：这个小的疏散星团有十几个星等+9到+10的天体。
* [[三裂星云|三裂星云]]（M20）：亦稱三叶星云，是发散和发射混和型星云，包含了不少年轻[[恒星|恒星]]。它的彩色照片非常美丽，可以看见它桃红色和亮蓝色的部分。用小型望远镜看卻難以分辨——由遮掩物质形成的暗沟把这个星云分为三个区域，所以它被称为三叶星云。在星云的中心还有一颗耀眼的[[三合星|三合星]]。M20相距地球五千[[光年|光年]]，其中炽热年轻的恒星被尘埃和气体所包围。
[[File:trifid.nebula.arp.750pix.jpg|thumb]]
* [[M21|M21]]：这是一个明亮但相当小的[[疏散星团|疏散星团]]，用小型望远镜很容易看见，总星等為+5.9。
* [[M22_(星團)|M22]]：它的直径约18[[弧分|弧分]]，是个结构松散的球状星团，用8英寸（20厘米）望远镜可以很好的观测，甚至，能分辨出它中心部分很多单个恒星。M22是相距地球[[较近|较近]]的星团，只有一万光年。总星等为+5.1。
* [[M23|M23]]：这是个大而壮观的疏散星团，约100颗星等在+9或更暗的恒星。它的星等是+5.5。
* [[小人马恒星云|小人马恒星云]]（M24）：它的直径超过1度，并不是真正的星团，而是[[银河|银河]]中的一个明亮部分。裸眼可见。但用双筒望远镜看的更清晰。用小型望远镜可以看见数不清的成片恒星。
* [[M25|M25]]：这是一个明亮稀疏的疏散星团，有大约50颗+6星等的恒星，还有更多更暗的。
* [[M28|M28]]：这是一个小的中等亮度但很致密的球状星团。需要约口径12英寸（30厘米）的望远镜才能分辨出单个恒星。其星等是+6.9。
* [[M54|M54]]：这是一个中等亮度但小而致密的小球状星团，需要使用大望远镜才能分辨出它，它的星等是+7.7，距离地球68000光年。
* [[M55|M55]]：在箕宿二西面约7.5°的M55。这是一个明亮结构松散的球状星团，用8英寸（20厘米）望远镜容易分辨。其星等是+7.0，距离地球20000光年。
* [[M69|M69]]：这是个又小又圆的球状星团，相当致密。使用8英寸（20厘米）的望远镜的高倍放大可以分辨出一些処于边缘的星，星等+7.7。
* [[M75|M75]]：也稱為'''NGC 6864'''，是一個[[球狀星團|球狀星團]]，於1780年被[[皮埃尔·梅尚|皮埃尔·梅尚]]發現，並且在同年被[[查尔斯·梅西耶|梅西爾]]發現並登錄在他的疑似[[彗星|彗星]]目錄中。
