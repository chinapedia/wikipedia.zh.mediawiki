==简介==
人民法院新闻传媒总社的前身是人民法院报社。人民法院报社是最高人民法院直属[[事业单位|事业单位]]，主办《人民法院报》、《[[人民司法|人民司法]]》杂志、[[中国法院网|中国法院网]]、最高人民法院影视中心以及内部文化刊物《天平》等媒体<ref>{{Cite web |url =  http://www.chinacourt.org/article/detail/2013/07/id/1023439.shtml|title = 人民法院报社公开招聘工作人员公告 |author =  |date =2013-07-09  |publisher = 中国法院网 |language =  |accessdate =  |quote =  }}</ref>。此外还负责管理最高人民法院官网、官微、官博，以及旗下各种[[微信|微信]][[微博|微博]]app<ref name=gengming/>。人民法院报社拥有东花市（北京市[[东城区_(北京市)|东城区]]东花市北里西区22号）、[[永定门|永定门]]（北京市[[丰台区|丰台区]][[右安门|右安门]]东庄幸福二巷18号）两处办公区<ref name=guanyu>{{Cite web |url =http://www.chinacourt.org/article/detail/2012/10/id/669614.shtml  |title = 关于人民法院报 |author =  |date =  2012-10-23 |publisher = 中国法院网 |language =  |accessdate =  |quote =  }}</ref>。2016年，经[[中央机构编制委员会办公室|中央机构编制委员会办公室]]批准，“人民法院报社”更名为“人民法院新闻传媒总社”<ref name=gengming>{{Cite web |url = http://news.ifeng.com/a/20160816/49785820_0.shtml |title = 人民法院报社更名为人民法院新闻传媒总社 |author =  |date = 2016-08-16 |publisher =凤凰网  |language =  |accessdate =  |quote =  }}</ref>。

===最高人民法院影视中心===
最高人民法院影视中心于1998年1月成立，由最高人民法院主管，人民法院报社（后改为人民法院新闻传媒总社）主办，2004年取得[[国家广电总局|国家广电总局]]颁发的“广播电视节目制作经营许可证”。最高人民法院影视中心负责组织法治题材的影视剧、专题片的选题策划和拍摄工作<ref>{{Cite web |url = http://www.chinacourt.org/article/detail/2012/10/id/669638.shtml |title =影视中心  |author =  |date = 2012-10-23 |publisher = 中国法院网 |language =  |accessdate =  |quote =  }}</ref>。
