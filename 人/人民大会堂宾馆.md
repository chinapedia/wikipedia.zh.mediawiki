'''人民大会堂宾馆'''，位于[[北京市|北京市]][[西城区|西城区]][[西交民巷|西交民巷]]甲19号，是[[全国人民代表大会常务委员会办公厅|全国人民代表大会常务委员会办公厅]][[人民大会堂管理局|人民大会堂管理局]]下属[[国有企业|国有企业]]。

==简介==
人民大会堂宾馆作为企业于1994年8月23日成立，为全民所有制企业。经营范围涉及住宿，制售中餐，销售饮料、酒、定型包装食品，零售卷烟，会议服务，物业管理，经济信息咨询，健身服务，打字、复印、传真，销售工艺品、日用品，零售字画，汽车租赁。税务登记日期为1996年11月1日<ref>{{Cite web |url =  http://qyxy.baic.gov.cn|title = 人民大会堂宾馆 |author =  |date =  |publisher = 北京市企业信用信息网 |language =  |accessdate =2017-03-28  |quote =  }}</ref>。2012年1月，人民大会堂宾馆重新装修后开始对外营业<ref name=qq>{{Cite web |url = http://bj.jjj.qq.com/a/20150507/060646.htm |title =北京人民大会堂宾馆  |author =  |date = 2015-05-07 |publisher = 腾讯 |language =  |accessdate =  |quote =  }}</ref>。

人民大会堂宾馆内设上百套客房。五层设有多功能厅。宾馆还设有三个会议室、七个宴会包房，以及餐厅、咖啡厅、美发室、茶社、健身房、台球、游泳池、酒吧、卡拉OK等设施<ref name=qq/>。

人民大会堂宾馆立足于为[[全国人大|全国人大]]、国务院各部委、各省、自治区、直辖市政府提供接待服务<ref name=qq/>。人民大会堂宾馆经常接待每年“全国[[两会|两会]]”代表团<ref>{{Cite web |url =http://news.163.com/15/0228/20/AJIODKH000014SEH.html  |title = 今年全国人大代表政协委员开会时住哪儿 |author =  |date = 2015-02-28 |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref>。
