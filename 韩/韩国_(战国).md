'''韓國'''是[[中國|中國]][[战国七雄|战国七雄]]之一，也作'''乾国'''，出自[[晉國|晉國]][[宗室|宗室]]，[[姬姓|姬姓]]。後世[[历史學家|历史學家]]將韩、[[魏国_(战国)|魏]]、[[赵国|赵]]、[[秦国|秦]]、[[楚国|楚]]、[[燕国|燕]]与[[齐国|齐]]合称[[战国七雄|战国七雄]]。韩国国土主要包括今[[山西|山西]]南部及[[河南|河南]]北部，初都[[平陽_(战国韩都)|平陽]]（今山西[[临汾|临汾]]），灭[[鄭國_(諸侯國)|郑国]]后则迁[[新郑|新郑]]（今[[河南|河南]][[郑州|郑州]][[新郑|新郑]]）。

== 起源 ==
{{main|晋国六卿|下宮之難|三家分晉}}
'''韓'''起源于[[晉國|晉國]]的[[公族|公族]]，[[姬姓|姬姓]]，可上推[[晉穆侯|晉穆侯]]。
[[File:HanZhuan2.jpg|90px]]
[[曲沃桓叔|曲沃桓叔]][[庶子|庶子]][[韓萬|韓萬]]，因協助[[曲沃武公|曲沃武公]]由[[曲沃|曲沃]]克[[晉|晉]]，被武公封於[[韓|韓]]地（韓原，山西稷山，其東鄰襄汾西鄰河津。襄汾为[[陶寺遗址|陶寺遗址]]所在），繁衍[[氏族|氏族]]，在[[晉|晉]]成為[[門閥|門閥]]名[[卿|卿]]，[[韓獻子|韓獻子]][[韓厥|韓厥]]在“[[下宮之難|下宮之難]]”后帮助[[赵武|赵武]]复兴赵氏。

傳至[[韓康子|韓康子]]，[[韩康子|韩康子]]與[[魏桓子|魏桓子]]奉执政[[正卿|正卿]][[智伯|智伯]]之命，討伐[[趙襄子|趙襄子]]。最後韩、魏倒戈，與趙國合作，[[三家滅智|三家滅智]]，瓜分了[[智伯|智伯]]所有[[食邑|食邑]]。從此[[韓國_(戰國)|韓]]、[[趙國|趙]]、[[魏國|魏]]三[[卿|卿]][[世族|世族]]獨霸。然后三卿再将[[晋|晋]]的领地瓜分，经过[[三晉伐齊之戰|三晉伐齊之戰]]后，在[[周天子|周天子]]的同意下，成为三个[[诸侯国|诸侯国]]，是為[[三家分晋|三家分晋]]，故[[中國历史|中國历史]]將韩、[[魏国_(战国)|魏]]、赵合称为[[三晋|三晋]]。

== 强盛 ==
韩国地居[[中原|中原]]，北临魏、赵，东有齐，南有楚，西有秦，四面受敵。幸而韩国以其著名的兵器－－[[弩|弩]]，为各国所畏惧。所谓“天下之強弓勁弩皆從韓出”，韩国的弩能射800米之外，“遠者括蔽洞胸，近者鏑弇心”<ref name=sj69>《史記卷69·蘇秦列傳·第九》</ref>。除此以外，韩国的劍也异常锋利，皆“陸斷牛馬，水截鵠鴈”，“當敵則斬堅甲鐵幕”<ref name=sj69/>。前375年，韓便曾以其厉兵灭了中原传统古国[[鄭國_(諸侯國)|郑国]]。而韓国国势最强要数[[韩昭侯|韩-{釐}-侯]]在位之时。他用[[法家|法家]]的[[申不害|申不害]]为[[宰相|宰相]]，内政修明，国成[[小康|小康]]之治。

== 衰微 ==
由於地處[[黄河|黄河]]中游地区，韓国東部和北部都被魏國包圍、西有秦國、南有楚國、以及有當時已很薄弱的[[東周|東周]]（洛陽），完全沒有發展的空间，国土在七国之中最小，屡遭列强欺凌。早年已经为魏齐之间的争霸资本，于前341年的[[馬陵之戰|馬陵之戰]]是围魏救韩的结果。秦楚争霸时，秦又要挟韩魏共同伐楚<ref>周慎靚王三年（己酉，前312年）；慎靚王十三年（己未，前302年）</ref>。末期的韩国成了[[秦国|秦国]]和[[山东六国|山东六国]]中其它國家爭霸的缓冲地，苟延残存；前262年，秦国大举进攻韩国[[上党|上党]]，上党不願被秦佔有，降于赵国，引发了[[长平之战|长平之战]]。两场决定霸主局势之战都由韩国而起，充分体现了韩国被列强围欺的困境。前230年，韩国军队屡战屡败，成为[[山东六国|山东六国]]中第一个被秦所灭的诸侯国。

== 大事记 ==
* 前403年：韓联同趙魏[[三家灭知|三家灭知]]。周初命晉大夫[[韓虔|韓虔]]、[[魏斯|魏斯]]、[[趙籍|趙籍]]為諸侯。
* 前400年：韓联同魏趙伐楚，至桑丘。鄭圍韓[[陽翟|陽翟]]。
* 前394年：韓救魯于齊。
* 前391年：秦伐韓[[宜陽|宜陽]]，取六邑。
* 前385年：韓伐鄭，取[[陽城|陽城]]；伐宋，執宋公。
* 前380年：齊伐燕，取[[桑丘|桑丘]]。韓联同魏、趙伐齊，至桑丘。
* 前378年：韓联同魏、趙伐齊，至[[靈丘|靈丘]]。
* 前376年：韓联同魏、趙三家廢[[晉靜公|晉靜公]]，分其地。
* 前375年：韓滅鄭，徙都新鄭。
* 前366年：秦敗韓師、魏師於[[洛陽|洛陽]]。
* 前362年：魏敗韓師、趙師於[[澮|澮]]。
* 前358年：秦敗韓師於[[西山|西山]]。
* 前353年：韓伐東周，取[[陵觀|陵觀]]、[[廩丘|廩丘]]。
* 前351年：[[韓昭侯|韓昭侯]]以[[申不害|申不害]]為相。
* 前341年：韓被魏伐。齐救韩，[[马陵之战|马陵之战]]。
* 前337年：韓申不害卒。
* 前335年：秦伐韓，拔宜陽。
* 前333年：[[合縱|合縱]]。
* 前323年：韓、燕皆稱王。
* 前318年：楚、趙、魏、韓、燕同伐秦，[[函谷關之戰_(前318年)|攻函谷關]]。秦人出兵逆之，五國之師皆敗走。
* 前317年：秦敗韓師於[[脩魚|脩魚]]，斬首八萬級，虜其將[[麿|麿]]、[[申差|申差]]於[[觀澤|觀澤]]。
* 前316年：侵秦，败北。
* 前314年：被秦败於岸門，韓太子倉入質於秦以和。
* 前312年：秦、韓、魏南襲楚，至鄧。
* 前308年－307年：秦将甘茂[[宜陽之戰|破韩国宜陽]]，斬首六萬。
* 前306年：秦以[[武遂|武遂]]復歸之韓。楚王與齊、韓合縱。
* 前303年：秦復取韓[[武遂|武遂]]。齊、韓、魏以楚負其縱親，合兵伐楚，秦救楚，三國引兵去。
* 前301年：秦會韓、魏、齊兵伐楚，敗楚。
* 前296年：齊、韓、魏、趙、宋同擊秦，至[[鹽氏|鹽氏]]而還。
* 前293年：[[伊闕之戰|韓、魏伐秦]]。秦白起擊敗魏師、韓師，斬首二十四萬級，拔五城。
* 前291年：秦伐韓，拔宛。
* 前290年：韓入武遂地二百里於秦。
* 前286年：秦敗韓師於[[夏山|夏山]]。
* 前284年：樂毅領燕、秦、魏、韓、趙[[濟西之戰|联合伐齊]]。
* 前275年：秦相國穰侯伐魏。韓[[暴鳶|暴鳶]]救魏，[[穰侯|穰侯]]大破之，斬首四萬。
* 前273年：[[華陽之戰|魏趙联合伐韓華陽]]。秦救韓，杀魏趙兵十四萬。秦王欲令韓、魏與秦一起伐楚，未行。
* 前265年：齐趙联合伐韓，取注人。
* 前264年：[[陘城之戰|秦伐韓]]，拔九城，斬首五萬。
* 前263年：秦武安君伐韓，取南陽；攻太行道，絕之。
* 前262年：秦伐韓，拔野王。上黨路絕，[[長平之戰|韩献上黨与赵]]。
* 前256年：秦伐韓，取[[陽城|陽城]]、[[負黍|負黍]]，斬首四萬。
* 前254年：韓王入朝于秦。
* 前249年：秦伐韓，取[[成皋|成皋]]、[[滎陽|滎陽]]，成为秦国的[[三川郡|三川郡]]。
* 前246年：韓使水工[[鄭國_(水工)|鄭國]]為間於秦。
* 前244年：[[蒙驁|蒙驁]]伐韓，取十二城。
* 前241年：楚、趙、魏、韓、衛[[函谷關之戰_(前241年)|合從以伐秦]]，楚王為從長，[[春申君|春申君]]用事，取[[壽陵|壽陵]]。至函谷，秦師出，五國之師皆敗走。
* 前233年：韓王向秦納地效璽，請為藩臣，使[[韓非|韓非]]聘秦。
* 前231年：韓向秦獻南陽地。
* 前230年：[[秦滅韓之戰|秦滅韓]]，虜韓王安，以其地置潁川郡。韓亡。<ref>《資治通鑒》卷1至6</ref>

==== 宰相 ====
* [[申不害|申不害]]是初期的著名人物，[[法家|法家]]中“术”的代表人物。《[[史記|史記]]》在《[[s:史记/卷063|老子韓非列傳]]》後面寫他是「故[[鄭國|鄭]]之賤臣。學術以干[[韓昭侯|韓昭侯]]，昭侯用為相。內脩政教，外應諸侯，十五年。終申子之身，國治兵彊，無侵韓者」。
* [[張開地|張開地]]是張平的父親，楚漢時期[[張良|張良]]的祖父，為[[韓昭侯|韓昭侯]]、[[韓宣惠王|韓宣惠王]]與[[韓襄王|韓襄王]]的宰相。
* [[張平|張平]]是張開地之子，楚漢時期張良的父親，為[[韓釐王|韓釐王]]與[[韓桓惠王|韓桓惠王]]的宰相，與父親張開地共同擔任戰國時代韓國五任韓王的宰相，此稱為[[五代相國|五代相國]]。
* [[俠累|俠累]]，[[韓烈侯|韓烈侯]]在位初期，俠累任韓國的相國，與濮陽的嚴遂（字仲子）爭權，嚴仲子失敗，出走他國。後來在齊國，有人向嚴仲子推薦聶政。 嚴仲子雇聶政殺俠累，於威烈王五年（甲申，公元前397年），俠累被殺。

==== 其他 ====
* [[郑国_(战国人)|郑国]]是晚期的著名人物，他是韩国派去[[秦国|秦国]]的[[水利|水利]][[工程师|工程师]]，他修筑了[[郑国渠|郑国渠]]，韓國派他去，本來是為了耗費秦國的[[財政|財政]]，卻幫助了[[秦國|秦國]][[農業|農業]]躍進，[[稅|稅]]收大增。
* [[韩非|韩非]]是晚期的著名人物，他是[[韓國|韓國]][[宗室|宗室]][[貴族|貴族]]，是[[法家|法家]]代表人物，曾自學[[老子|老子]]思想，又拜[[荀卿|荀卿]]為師，兼採[[商鞅|商鞅]]重法、[[申不害|申不害]]重術、[[慎到|慎到]]重勢等三家學說之長，發揚法家，集[[法家|法家]]思想的大成。後被迫侍奉[[秦始皇|秦始皇]]，遭[[李斯|李斯]]害死。
