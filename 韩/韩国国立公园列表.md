'''韩国国立公园'''（[[韩语|韩语]]：한국국립공원），是[[大韩民国政府|大韩民国政府]]为了保护和保存[[自然生态|自然生态]]系统、自然以及文化景观以实现[[可持续发展|可持续发展]]而指定并加以管理的保留区，分为“山岳公园”、“海上•海岸公园”和“历史遗迹形态公园”三类。迄今韩国政府在其境内总共划设了21座[[国立公园|国立公园]]，包括16个山岳公园、4个海上•海岸和1个历史遗迹形态公园，約佔韩国國土總面積的3.9%。<ref name=main>{{cite web |url = http://chinese.knps.or.kr/Knp/AboutKnp.aspx?MenuNum=1&Submenu=01 |title = 韩国国立公园简介 |publisher = 国立公园管理公团 |accessdate = 2014-12-27 |deadurl = no |archiveurl = https://web.archive.org/web/20141226183355/http://chinese.knps.or.kr/Knp/AboutKnp.aspx?MenuNum=1&Submenu=01 |archivedate = 2014-12-26 |df =  }}</ref>

受到[[美国国家公园列表|美国国家公园]]{{NoteTag|美国国家公园制度开始于1872年<ref name="公团">{{cite journal |title=韩国国立公园管理公团  |author=（韩）韩相一  |journal=《中国园林》 |date=2005年第7期}}</ref>}}和[[日本国立公园|日本国立公园]]制度的影响{{NoteTag|1930年，美国国家公园制度传入日本{{r|公团}}。}}，韩国1967年颁布《公园法》，建立了韩国首个，也是最大的山岳国立公园，[[智异山国立公园|智异山国立公园]]<ref name=yg>{{cite name=xl>{{cite 2012年年末成立的[[无等山国立公园|无等山国立公园]]是韩国最近成立的国立公园<ref name=wd>{{cite 韩国国立公园在《公园法》颁布初期是由韩国建设部委托地方政府管理的。由于有些国立公园横跨多个[[市|市]]或[[道_(行政区划)|道]]引起[[行政管理|行政管理]]上的问题，韩国依照《自然公园》于1987年7月1日成立了专门管理国立公园的机构国立公园管理公团。1991年4月23日，国立公园管理公团被编入韩国内务部，1998年2月开始被规划到[[韩国环境部|韩国环境部]]<ref name="环境">{{cite journal |title=韩国的国立公园 |author=王英斌 |journal=《环境》 |year = 1994年 |issue = 第12期 }}</ref>{{r|公团|yg}}。目前，除[[汉拏山国立公园|汉拏山国立公园]]由[[济州特别自治道|济州特别自治道]]地方政府管理外，韩国国立公园都由国立公园管理公团统一管理{{NoteTag|庆州国立公园在成立之初由庆州市管理，2008年起改由国立公园管理公团管理{{r|lis}}。}}<ref name=gl>{{cite web|url=http://chinese.knps.or.kr/Introduction/Introduce.aspx?MenuNum=4&Submenu=12|title=韩国国立公园管理公团简介|author=|date=|work=|publisher=国立公园管理公团|accessdate=24 August 2012|deadurl=no|archiveurl=https://web.archive.org/web/20120830082741/http://chinese.knps.or.kr/Introduction/Introduce.aspx?MenuNum=4&Submenu=12|archivedate=2012年8月30日|df=}}</ref>。 

韩国国立公园成立初期对游客实行[[免费|免费]]政策。1970年，[[俗离山国立公园|俗离山国立公园]]率先开始实行[[门票|门票]]收费制度。到1974年，韩国的大部分国立公园都取消了免门票的政策。目前只有[[庆州国立公园|庆州国立公园]]是免费的。1993年，韩国国立公园的门票收入占到了公园总收入的65%。随着游客数量的逐年增加，韩国政府为有效保护公园的自然环境，对游客的数量和活动进行了限制，通过预约制控制游客数量，游客野炊和[[宿营|宿营]]只能在制定区域进行，严格禁止夜间登山。除此之外，韩国还对国立公园具有高保护价值的部分地区实行自然休息年制，以避免人为破坏{{r|环境}}<ref>{{cite web| title =特别保护区指南| url =http://chinese.knps.or.kr/Knp/Special.aspx?MenuNum=1&Submenu=03| publisher =国立公园管理公团| accessdate =2015-07-29| deadurl =no| archiveurl =https://web.archive.org/web/20150801035453/http://chinese.knps.or.kr/Knp/Special.aspx?MenuNum=1&Submenu=03| archivedate =2015-08-01| df =}}</ref><ref>{{cite journal |title = 韩国国立公园概况  |author=（韩）韩相壹 |coauthors = |journal=《中国园林》 |date=2002年02期}}</ref>。
