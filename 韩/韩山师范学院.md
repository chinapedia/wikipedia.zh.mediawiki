'''韩山师范学院'''是一所[[广东省|广东省]]属本科师范院校。坐落于享有“中国瓷都”、“中国著名婚纱晚礼服城市”、“中国潮菜之乡”等美誉的国家历史文化名城——[[潮州市|潮州市]]。是[[粤东|粤东]]地区唯一的一所本科师范院校。

韩山师范学院占地面积86万平方米，建筑面积50万平方米；现有在职教职工1210人，其中专任教师950名；全日制在校生21000多人；设有三个校区（校本部、韩东校区、潮州师范分院），18个二级学院，55个本科专业，13个专科专业。

== 历史 ==
其前身最初可以追溯至宋[[元祐|元祐]]五年设立的韩山书院，其后多番兴废。清[[康熙|康熙]]二十七年，韩山上又建立起了一座书院，当时叫做[[昌黎书院|昌黎书院]]，[[雍正|雍正]]十年又改叫[[韩山书院|韩山书院]]。此后，韩山书院一直办学。光绪二十九年，正是清政府推行新式学堂的年头，韩山书院根据《钦定师范学堂章程》报请上头批准，改名“[[惠潮嘉师范学堂|惠潮嘉师范学堂]]”，成了一所新式师范学堂，它是广东省第一所师范学校。韩师的校史一般是从这一年算起的。

中华民国建立以后，惠潮嘉师范学堂又改名为“广东省惠潮梅师范学校”，后又改称“广东省立第二师范学校”、“广东省立韩山师范学校”。

* 1939年，日本侵占潮州。韩师多次迁移。
* 1946年，日本投降。韩师的师生们回归校园。
* 1958年，韩师从中等师专升级成了高等师专，校名同时改称“汕头韩山师范专科学校”。但不久后，有关部门宣布停办韩师师专，转成教师进修学校。
* 1978年，经国务院同意，复办韩师师专，定名“韩山师范专科学校”。1993年，韩山师范专科学校升格成本科师范学院，定名“韩山师范学院”。
* 2001年，接收原潮州市师范学校，并将其易名为“韩山师范学院潮州师范分院”。

== 校区 ==
韩山师范学院一共有三个校区：校本部（桥东街道）、韩东校区（官塘镇）、潮州师范分院（西荣路）。此外还在枫溪区设置一个中专学校——广东省陶瓷职业技术学校。

== 研究机构 ==
潮学研究院、环境化学应用技术研究所、教育研究所、资源植物研究所、科学思维研究所、规划设计研究所、数学研究所、计算机应用研究所、无机化学研究所、催化科学和技术研究所、材料科学研究所、八股文研究所、体育文化研究中心等。

== 交通 ==
潮州粤运公交101、104、105、108、110、112、113、201、203、204、207、602线路有途径韩山师范学院校本部。109、115、205线路则途径潮州师范分院（西荣路）。108线路既途径校本部也途径韩东校区（即该路线的起终点站）。
