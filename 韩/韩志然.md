'''韩志然'''（{{bd|1952年|2月||catIdx=Han韩志然}}），[[内蒙古|内蒙古]][[奈曼旗|奈曼旗]]人，[[蒙古族|蒙古族]]，曾任[[巴彦淖尔盟|巴彦淖尔盟]]盟长、[[内蒙古自治区|内蒙古自治区]]政府秘书长、[[包头市|包头市]]市长、[[乌兰察布盟|乌兰察布盟]]委书记（市委书记）、[[呼和浩特市|呼和浩特市]]委书记。

2015年6月17日，第十二届全国政协常委员会第十一次会议免去韩志然第十二届全国政协常委、委员资格。2015年6月23日，政协内蒙古自治区第十一届委员会常务委员会第十一次会议决定，免去韩志然自治区十一届政协副主席职务、撤销其自治区十一届政协委员资格<ref>{{cite web|title=内蒙古政协副主席韩志然涉严重违纪被免职|url=http://news.ifeng.com/a/20150623/44026675_0.shtml|publisher=凤凰网|accessdate=2015-06-23}}</ref>。据媒体报道，韩志然的落马源于他卷入被称为内蒙古“黄金大盗”的宋文代案。宋文代举报了韩志然<ref>{{cite web|url=http://news.sina.com.cn/c/2015-06-24/143831983145.shtml|title=内蒙政协原副主席韩志然涉“黄金大盗”案被查|publisher=新浪网|accessdate=2015-06-24}}</ref>。2016年1月29日，中纪委通报，韩志然因严重违纪，受到留党察看二年处分，降为副厅级非领导职务<ref name="重大职务调整">{{cite web |url = http://www.ccdi.gov.cn/xwtt/201601/t20160129_73665.html |title = 中央纪委2015年给予10名中管干部党纪重处分并作出重大职务调整 |accessdate = 2016年1月29日 |publisher = 中央纪委监察部网站}}</ref>。

== 爭議 ==
任呼和浩特市委書記期間，韩志然认为，低房价制约了城市发展。为此，呼和浩特市在2004年底开始，在专业公司的帮助下，出让土地8000多亩，土地价格从2004年底的每亩30万元提升到50万元以上，最高达到300万元。市政府的土地净收益超过4亿元。地方政府“有形之手”是主要推动力量，政府直接运营土地，大幅拉高土地价格，借机“以地生财”。商品住房平均销售价格也从2004年的每平方米1590元陡然升到2005年的2400多元，引起民眾不滿。<ref>[http://news.mzyfz.com/times/b/20071015/164902.shtml 呼和浩特市委书记该如何当？]{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>並且多次宣稱呼和浩特房价不存在泡沫。<ref>[http://www.chinanews.com.cn/news/2006/2006-06-19/8/745560.shtml 呼和浩特市委书记否认房价泡沫 市民称房价太高]</ref>

== 管理理念 ==

房地产市场的快速发展是经济快速增长的必然结果，是工业化、城市化进程加快的必然结果。

增建大批高檔住宅提高呼和浩特市人均住房面积。
