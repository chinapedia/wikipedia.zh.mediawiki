'''[[韩国|韩国]][[围棋|围棋]]联赛'''

== 2002年 ==
地區性圍棋對抗賽

== 2003年 ==
改名韓國夢幻聯賽，只有6支隊伍，總規模為3.6億[[韓元|韓元]]

== 2004年 ==
改名韓國圍棋聯賽，發展到了有8家企業俱樂部參加，總預算達到10億韓元。每隊4人共32人中，2003年度獎金排名前16名的棋手自動獲得參賽權，其他16個名額由其餘的180名棋手競爭獲得。
*團體獎金：第1名為1億韓元，墊底的第8名則只有1000萬韓元。
*個人獎金：每場勝利主將獎金為200萬韓元、第二台150萬韓元、第三台120萬韓元、第四台100萬韓元。輸的一方則不管是第幾台，統統只支付50萬韓元的對局費。
此外還設立了1000萬韓元的最佳棋手獎和300萬韓元的多勝獎。

== 2005年 ==
分为季前赛、常规赛和季后赛三个阶段。奖金总规模15亿韩元，从4月28日起开始正式比赛。
*代表队：韩国游戏、帕克兰德、泛洋建营、宝海、新盛建设、第一火灾、网石和血网。
*参赛棋手：[[李昌镐|李昌镐]]九段、[[李世石|李世石]]九段、[[崔哲瀚|崔哲瀚]]九段、[[刘昌赫|刘昌赫]]九段、[[曹薰铉|曹薰铉]]九段等24名棋手被列入种子选手。<ref>[http://weiqi.sports.tom.com/news/12467.html 向伟大的韩国围棋挑战！——韩国围棋联赛声势逼人_TOM棋圣]</ref>
*冠军队伍和[[中国围棋甲级联赛|中国围棋甲级联赛]]冠军进行“韩中围棋联赛冠军对抗战”。
