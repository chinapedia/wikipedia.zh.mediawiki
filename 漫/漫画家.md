'''漫画家'''指创作[[漫画|漫画]]的人，漫畫有不同的類型，有單幅漫畫、四格漫畫、連環漫畫等等。連載強度較低的，可能只由一個漫畫家做完整個創作。連載強度較大的漫畫，往往會由一個工作室團隊來製作，像美國的[[漫威漫畫|漫威]]各英雄角色都有對應的團隊，日本的漫畫也都有工作室，香港的[[黃玉郎|黃玉郎]]與所轄主筆也是如此。一般來講，漫畫原作與主要畫家都可以稱為漫畫家。其它參與作畫但沒參與創作的，通常只能稱為漫畫助手。

== 獨立作業的漫畫家 ==
像日本[[長谷川町子|長谷川町子]]以黑白四格漫畫的型式連載[[海螺小姐|海螺小姐]]，線條簡要，沒有繁複背景，沒有上色問題，創作的壓力主要在於劇情。這樣就可能獨立作業。

== 帶領工作室的漫畫家 ==
在日本很常見，日本的漫畫周刊一周要畫近20頁的連載，還有其它上色稿件等工作，連載壓力強度大。通常會找幾個助手協助。比較制度化的是像[[秋本治|秋本治]]創作漫畫[[烏龍派出所|烏龍派出所]]，在1990年代成立製作公司[[有限会社アトリエびーだま|有限会社アトリエびーだま]]以企業化方式雇用助手。

== 漫畫原作 ==
{{see also|漫畫原作者|漫畫原作}}
通常會撰寫劇本，有的會參與製作分鏡。譬如漫威蜘蛛人的劇本創作是[[Stan_Lee|Stan Lee]]，日本漫畫[[北斗之拳|北斗之拳]]的原作是[[武論尊|武論尊]]。

== 主要畫家 ==
會執行人物的美術設計，並主持作畫創作的部份。譬如漫威蜘蛛人的主要畫家是[[Jack_Kirby|Jack Kirby]]，日本漫畫[[北斗之拳|北斗之拳]]的作畫是[[原哲夫|原哲夫]]。

== 漫畫助手 ==
執行一些技術上的細節，譬如描背景、畫特效線、塗黑、貼網點、等等。通常不參與創作，但有些會幫漫畫家提供故事點子。

==職業病==
由於長時間久坐，缺乏運動，不規律的睡眠與工作，所以容易導致[[職業病|職業病]]。例如[[腰痛|腰痛]]、[[腱鞘炎|腱鞘炎]]，或是[[猝死|猝死]]。

==戲稱==
;一部漫畫家
泛指只有一部成名作，而無後續其他作品的漫畫家。
