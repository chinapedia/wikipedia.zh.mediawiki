《'''漫画家與助手'''》（{{lang-ja|マンガ家さんとアシスタントさんと}}）是[[ヒロユキ|ヒロユキ]]所作的[[日本漫画|日本漫画]]。从2008年开始连载于《[[YOUNG_GANGAN|YOUNG GANGAN]]》和《[[月刊少年GANGAN|月刊少年GANGAN]]》。其中《月刊少年GANGAN》上连载的作品是[[四格漫画|四格漫画]]的形式。另外自2010年6月24日起在《[[GANGAN_ONLINE|GANGAN ONLINE]]》上連載。本作是描写漫画家'''愛徒勇氣'''和助手'''足須沙穗都'''的日常生活的喜剧作品。漫畫由[[東立出版社|東立出版社]]代理。

2013年12月發表動畫化消息<ref>{{Cite web|publisher=Natalie|title=ヒロユキ「漫畫家與助手」動畫化|url=http://natalie.mu/comic/news/105104|accessdate=2013-12-06}}</ref>，於2014年4月7日於[[TOKYO_MX|TOKYO MX]]等日本電視台播出<ref name=animeanime>{{Cite web|publisher=アニメ！アニメ！|title=「漫畫家與助手」14年4月放送開始 GANGAN發的人氣作品動畫化|url=http://animeanime.jp/article/2013/12/29/16898.html|accessdate=2013-12-28}}</ref>。

== 登場人物 ==
※配音演員為廣播劇CD版／電視動畫版。
=== 《羞羞拿铁咖啡》相關 ===
; {{nihongo|愛徒 勇氣|愛徒 勇気（あいと ゆうき）|[[配音員|配音員]]：[[福山潤|福山潤]]／[[松岡禎丞|松岡禎丞]]}}
: 身高175CM，生日1月1日，22歲。
: 本作的主人公，长相平常、喜愛[[女用內褲|女用內褲]]的漫画家。喜欢钻牛角尖，有妄想癖，自稱「愛與勇氣的漫畫家」（{{lang|ja|愛と勇気の漫画家}}），梦想是「死前被女生告白一次」（{{lang|ja|死ぬまでに一度でいいから女の子に告白されたい}}）。在少年漫畫雜誌《月刊少年GONGON》连载漫畫《羞羞拿铁咖啡》（{{lang|ja|はじらってカフェラッテ}}），簡稱《羞鐵》（{{lang|ja|はじカフェ}}），在杂志上人气投票排名（总共20部作品中的）15位。经常因为好色被打，不过因为是M，所以没什么收敛。對女用內褲的喜愛近乎偏執，甚至可以看到人就分析他人對內褲的喜好。

; {{nihongo|足須 沙穗都|足須 沙穂都（あしす さほと）|配音員：[[能登麻美子|能登麻美子]]／[[早見沙織|早見沙織]]}}
: 身高163CM，生日3月27日，19歲。
: 愛徒的助手，是个温和的黑髮美女。根据見張的介紹，从1年前开始帮忙，同時自己也正在努力成為一名漫畫家。夢想是「當一個不給編輯添麻煩的漫畫家」，不過上交的作品大多沒被採用。
: 曾經以一部女主角欺負男性[[廢人|廢人]]上司的漫畫《懲罰！》獲得《月刊少年GONGON》新人獎最大的獎項，但是因為內容太激烈而無法刊登。
: 在以她與愛徒之間的關係為參考創作的漫畫《漫畫助手》出道成為職業漫畫家後，雖然辭去了愛徒助手的工作，卻搬家到了愛徒的公寓，繼續著與愛徒的故事。

; {{nihongo|音砂 見張|音砂 みはり（おとすな みはり）|配音員：[[戶松遙|戶松遙]]／[[能登有沙|能登有沙]]}}
: 身高155CM，生日7月22日，22歲。
: 愛徒和貫的[[责任编辑|责任编辑]]。经常以[[西装|西装]]登场，金髮[[双马尾|双马尾]]。性格很严厉，经常毫不留情以一拳制裁暴走的愛徒，但是也有「和不錯的人结婚，拥有幸福的家庭」这样可爱的梦想。
: 高中時與愛徒是同班同學，自己本想成為漫畫家但畫工太爛。無意間發現了愛徒的繪畫才能。非常在意自己AA[[罩杯|罩杯]]的[[貧乳|貧乳]]，當愛徒特別提起時就會反擊。
: 高中畢業後，赴《月刊少年GONGON》編輯部應徵责任编辑，在[[面試|面試]]時忽然拿自己AA罩杯的貧乳開玩笑，卻因此被皆野纏錄取。曾被纏[[爆料|爆料]]初戀對象是愛徒。
: 在高中同學會結束的歸途，曾半開玩笑地和愛徒說，如果多年後雙方都沒交往對象的話，就交往看看。

; {{nihongo|風羽 玲奈|風羽 りんな（ふわ りんな）|配音員：[[豐崎愛生|豐崎愛生]]／[[井口裕香|井口裕香]]}}
: 身高158CM，生日10月8日，18歲。
: 愛徒的狂熱[[爱好者|粉絲]]，虽然不知道怎么画漫画，不过因为可爱而被录用为助手。是愛徒助手中唯一的[[巨乳|巨乳]]。性格[[天然呆|天然呆]]，對所有人都笑臉相迎。

; {{nihongo|黑井 瀨奈|黒井 せな（くろい せな）|配音員：[[釘宮理惠|釘宮理惠]]／同左}}
: 身高140CM，生日12月10日，19歲。
: 号称“不管是怎樣的原稿都能以完美的速度和品質趕上，連大腕漫畫家都要向她低頭”的超級助手，因為不會想故事，所以當不成漫畫家。
: 性格[[傲嬌|傲嬌]]，私下非常喜歡可愛的東西，但因不夠坦率而無法在別人面前表現。非常在意個子像小孩一般的嬌小。
: 對愛徒有好感。

; {{nihongo|波田 學|波田 まなび（なみだ まなび）}}
: 想成為漫畫家的女中學生，因被《月刊少年GONGON》編輯部退稿而與愛徒碰面，隨後成為愛徒的徒弟，時常因愛徒與助手的互動而混亂。

; {{nihongo|清崎 純|清崎 純（きよさき じゅん）}}
: 沙穗都的助手，有G罩杯的巨乳，對愛徒和他的漫畫抱持偏見。懂得[[防身術|防身術]]。初登場即被一華與纏揉胸部。

; {{nihongo|胸罩貓|ブラニャー|配音員：無／未公佈}}
: 愛徒撿到的貓，長得跟愛徒為了討沙穗都開心所畫的[[吉祥物|吉祥物]]一模一樣。雖然不會講話，卻會寫字，所以能與人類溝通。有畫漫畫的資質，隨後在《月刊少年GONGON》登載新的作品。原本一開始只是將愛徒當成奴隸看待，但故事後發展出了友情。

=== 《月刊少年GONGON》連載組 ===
; {{nihongo|貫 一真|貫 一真（つらぬき かずま）|配音員：{{tsl|ja|宮下栄治|宮下榮治}}}}
: 熱血漫画家，认为给見張添了麻烦的愛徒是敌人。在「月刊少年GONGON」上连载《熱血戰士》（{{lang|ja|熱血（ファイアー）ファイター}}），在杂志上人气排名（总共20部作品中的）14位。愛徒认为他是个闷骚的人。後來《熱血戰士》由於單行本銷量不佳而被[[腰斬|腰斬]]。
: 原本的責任編輯是見張，在經歷一段時間失業後，由種持葵擔任他的責任編輯。暗恋見張，但葵的出現讓他陷入迷惘。
; {{nihongo|兔美都 冥聖|兎美都 冥聖（とみと めいせい）|配音員：[[綠川光|綠川光]]}}
: 作品畅销的漫画家，性格有问题，计划将沙穗都收为自己的助手。因為他的性格關係，常常趕走或嚇跑助手。没有朋友，本人也非常在意這點。在《月刊少年GONGON》上连载《J・O・D黑暗審判》（J・O・D{{lang|ja|ジャッジメント・オブ・ダークネス}}），人气是（20部作品中的）第2位。
; {{nihongo|元木 玉子|元木 たまこ（もとき たまこ）}} 
: 《幹勁1000%》（{{lang|ja|テンション1000%}}）的作者，新人漫畫家。元氣十足的少女。是《羞羞拿鐵咖啡》的粉絲，非常尊敬愛徒。
; {{nihongo|今野 一華|今野 一華（こんの いちか）}} 
: 漫畫家，年近三十的[[眼鏡娘|眼鏡娘]]，《月刊少年GONGON》人氣排名第一的作品《冒險世界》（{{lang|ja|アドベンチャーワールド}}）的作者。喜歡捉弄人。與愛徒初次見面后覺得他很有趣，便成為了他的鄰居。非常在意年齡這點，當有人刻意談起她的年紀時會不高興。

=== 《月刊少年GONGON》編輯部 ===
; {{nihongo|皆野 纏|皆野 まとめ（みなの まとめ）|配音員：無／[[田中真奈美|田中真奈美]]}}
: 虽然外表是少女，却是《GONGON》的編輯長（主编）。喜欢吃零食，总是在吃巧克力或者[[曲奇|曲奇]]。曾经给作品人气下滑到最下位的愛徒建议，不过愛徒没有意识到她是主编。性格上[[無口|很少有豐富的表情變化]]，但其實有點好色。
; {{nihongo|富都 剛健|富都 剛健（とみと ごうけん）|配音員：無／[[安元洋貴|安元洋貴]]}}
: 兔美都的责任编辑，也是他的哥哥。《GONGON》副主编。表情冷酷，身材高大，不过很讲礼貌。
; {{nihongo|種持 葵|種持 葵（たねもち あおい）}}
: 新人編輯，看起來是可愛的美少女，但其實是男性。與玲奈一樣也是愛徒的狂熱爱好者。後來成為貫一真新連載作品的責任編輯，而他的性別與少女相貌的反差總是讓愛徒非常困惑。

=== 其他角色 ===
; {{nihongo|住野 守|住野 まもり（すみの まもり）}}
: 愛徒住的公寓的管理人，天然呆的眼镜娘。
; {{nihongo|住野 美麗|住野 美麗（すみの みれい）}} 
: 守的姐姐，對於愛徒時常不繳房租而困擾。
; {{nihongo|足須 沙穗乃|足須 沙穂乃（あしす さほの）|配音員：無／[[伊藤美来|伊藤美来]]}}
: 沙穗都的妹妹，13歲的中學生，非常支持愛徒追求姊姊，有點好色。
; {{nihongo|榎下 紗紗繪|榎下 紗々絵（えのした ささえ）}}
: 一華的助手主任兼經紀人，對一華非常嚴苛。勸一華放棄獲得幸福（找對象結婚）的想法，為幾百萬讀者的幸福而努力。
