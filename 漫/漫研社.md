《'''漫研社'''》（{{lang-ja|あいまいみー}}）是由{{tsl|ja|ちょぼらうにょぽみ|Choborau-nyopomi}}創作的[[日本|日本]]四格{{tsl|ja|ギャグ漫画|搞笑漫畫}}作品，自2009年6月在[[竹書房|竹書房]]的[[網路漫畫|網路漫畫]]發布站《{{tsl|ja|まんがライフWIN|漫畫人生WIN}}》連載。

== 簡介 ==
此[[四格漫畫|四格漫畫]]作品描寫的是倉持南高中漫畫研究社四位[[女高中生|女高中生]]「不知所謂的[[空氣系|日常生活]]」。如同單行本第1冊的介紹「不被常識束縛的四格漫畫」<ref group="竹" name="vol1">{{Cite web|url=http://www.takeshobo.co.jp/book_d/shohin/2036001|script-title=ja:あいまいみー（1）|publisher=竹書房|accessdate=2016-11-09|language=ja}}</ref>所言，作品中經常出現脫離常識、異想天開、荒誕無稽，有時甚至是瘋狂、病態的[[超现实主义|超現實]]劇情。熟悉四格漫畫的藝人{{tsl|ja|向清太朗}}曾在2011年評論此作品為「對這個時代來說還太早的漫畫」<ref>{{Cite web|url=http://www.cyzo.com/2011/05/post_7433_entry.html|script-title=ja:松本人志も食い付いた!?　壮絶カオス美少女萌え4コマ『あいまいみー』とは？|date=2011-05-26|publisher=日刊サイゾー|accessdate=2016-11-09|language=ja}}</ref>。

== 登場人物 ==
; {{Nihongo|蛯原愛|蛯原 愛（えびはら あい）|聲：[[大坪由佳|大坪由佳]]}}
: 暱稱為「愛」，倉持南高中漫畫研究社（漫研）中唯一認真創作漫畫的正常人，負責[[吐槽|吐槽]]。經常被麻衣等人騷擾，不曾在打架中輸給她們。
: 有個叫作夏菜子（聲：[[仲谷明香|仲谷明香]]）的妹妹。

; {{Nihongo|麻衣|麻衣（まい）|聲：[[內田彩|內田彩]]}}
: 天真無邪的漫研社員，容易受到他人影響，不時與咪一同裝傻。
: 曾因為在[[河童|河童]]居住的河川中倒入水泥而被河童毆打。

; {{Nihongo|咪妮亞·梅妮希|ミィニャ・メーニッヒ|聲：[[內田真禮|內田真禮]]}}
: 暱稱為「咪」，老是喜歡裝傻的漫研社員。
: 有時會因為過於亂七八糟的裝傻而受到愛猛烈的還擊。

; {{Nihongo|波乃香前輩|ぽのか先輩|聲：[[茅野愛衣|茅野愛衣]]}}
: 上述三人於漫研社的前輩，特徵是Half-up造型的紫色長髮與充滿謎團的壺罐。
: 登場頻率較同年級的三位後輩低，而登場時大多會發生匪夷所思的事情。

; {{Nihongo|六筆協會|六筆協会（ろくふできょうかい）}}
: 以[[意大利|-{zh-tw:義;zh-hk:意;zh-cn:意;}-大利]][[佛羅倫斯|佛羅倫斯]]北部為據點進行活動的神秘組織。由亞-{里}-莎·倫敦（聲：茅野愛衣）、佐藤鈴木（聲：天津向）、田中（聲：天津向）、彈嫩太郎（聲：茅野愛衣）、國木田麻理子（聲：{{tsl|ja|桜 稲垣早希|櫻 稻垣早希}}）、唐納文協長（聲：{{tsl|ja|R藤本}}）六位成員組成，其中唐納文協長為實質領導人。
: 上述成員中，佐藤鈴木由於在宴會一武鬥會敗給愛而失去消息，彈嫩太郎以繼承老家[[農業|農業]]為由退出協會，亞-{里}-莎則被波乃香殺害。

; {{Nihongo|Choborau-nyopomi|ちょぼらうにょぽみ|聲：{{tsl|ja|ちょぼらうにょぽみ|Choborau-nyopomi}}（第一季）；[[NEEKO|NEEKO]]（第二季以後）}}
: 此漫畫的作者，偶爾會莫名其妙地登場。

; {{Nihongo|野乃本索梅拉、野乃本庫庫露、天童雫、松嶋愛|野乃本ソメラ、野乃本ククル、天童雫、松嶋あい}}
: 來自同一位作者的漫畫《[[不思議美眉|不思議美眉]]》中的角色，其中索梅拉、庫庫露、松嶋以背景角色的形式登場，而天童雫（聲：茅野愛衣）則以網路偶像「TS」名義參加宴會一武鬥會，並與愛展開了壯烈的對決。

; {{Nihongo|柴犬子|しばいぬ子さん|聲：[[佐佐木未來|佐佐木未來]]}}

== 電視動畫 ==
第一季以《Choborau-nyopomi劇場 漫研社》<ref group="註">寫有「建議精神年齡18歲以上」字樣。</ref>為題，於2013年1月至3月播出<ref>{{Cite web|url=http://natalie.mu/comic/news/79381|script-title=ja:ちょぼらうにょぽみ「あいまいみー」TVアニメが新春放送|date=2012-11-04|publisher=コミックナタリー|accessdate=2016-11-09|language=ja}}</ref>，為每話5分鐘的[[泡面番|短篇動畫]]，由[[漫才|漫才師]]兼[[搞笑藝人|搞笑藝人]]的{{tsl|ja|向清太朗||天津向}}擔任旁白。

2014年4月1日宣佈製作第二季，標題定為《Choborau-nyopomi劇場第二幕 漫研社 -妄想災難-》<ref>{{Cite web|url=https://twitter.com/anime_aimaimi/status/451024118826037248|script-title=ja:【お知らせ】（続き）|date=2014-04-01|publisher=アニメあいまいみー公式Twitter|accessdate=2016-11-09|language=ja}}</ref>，於同年7月至9月播出。

2016年11月宣佈製作第三季，並在2017年1月播放<ref>{{Cite web|url=http://natalie.mu/comic/news/208214|script-title=ja:アニメ「あいまいみー」第3期が決定！2017年1月から放送スタート|date=2016-11-07|publisher=コミックナタリー|accessdate=2016-11-09|language=ja}}</ref>，標題為《Choborau-nyopomi劇場第三幕 漫研社～surgical friends～》。
