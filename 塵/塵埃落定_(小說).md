《'''塵埃落定'''》是一部1998年发表的基於[[藏族|藏族]]民風傳統的[[長篇小說|長篇小說]]，共计十二章有48.1万字。其作者[[阿來|阿來]]為[[藏族|藏族]]人士，為雜誌《[[科幻世界|科幻世界]]》編輯。該小說在2000年獲得第五届[[茅盾文學獎|茅盾文學獎]]<ref >{{cite web |url = http://www.china.com.cn/culture/txt/2000-10/19/content_5008247.htm | title= 茅盾文学奖“尘埃落定” |website = 中国网 |date=2000-10-19 |accessdate= }}</ref>。

==梗概==
故事讲述了[[20世纪20年代|20世纪20年代]]至[[中华人民共和国|中华人民共和国]]成立之前，[[四川|四川]][[阿坝|阿坝]]一位藏族[[土司|土司]]和他家人的命运。书中大致介绍了阿坝地区的经济发展，土司之间的领土争端以及土司继承权的斗争。

== 评价 ==
