'''门洛帕克'''（[[英語|英語]]：'''Menlo Park'''），又譯'''-{zh-hans:门洛公园;zh-hk:門洛公園;zh-tw:門洛帕克;}-'''，[[美国|美国]][[加利福尼亞州|加利福尼亞州]][[聖馬刁縣|聖馬刁縣]]東南部的一座[[城市|城市]]。根据[[美国人口调查局|美国人口调查局]]2000年的[[人口普查|人口普查]]，該市共有30,787人。值得一提的是，「加利福尼亞州門洛帕克」的名稱早見於（約1850年）<ref>Stanger, 1963. pg. 109</ref> [[湯瑪斯·愛迪生|湯瑪斯·愛迪生]]（約1876年）在[[新澤西州|新澤西州]][[門洛帕克_(新澤西州)|門洛帕克]]所創的發明品。

== 地理 ==
門洛帕克位於[[北緯|北緯]]37°27'，[[東經|東經]]122°10'。（37.454188 -122.178579）{{GR|1}}。

根據[[美國人口調查局|美國人口調查局]]，門洛帕克佔的總面積有45.1[[平方公里|平方公里]]（17.4 [[平方英里|平方英里]]），當中陸地佔26.2平方公里（10.1平方英里），水域佔18.9平方公里（7.3平方英里），即41.88%。

[[File:California_live_oak.jpg|thumb]]（[[學名|學名]]：''Quercus agrifolia''）是門洛公園的市標]]
Santa Cruz大街是門洛帕克[[中心商務區|中心商務區]]的[[主街|主街]]，而門洛中心正位處該大街與[[國王大道_(加利福尼亞州)|國王大道]]（El Camino Real）的交界。門洛帕克[[文娛中心|文娛中心]]被Ravenswood大街、Alma街、Laurel街和Burgess車道圍繞，那裏設有[[市議會|市議會]]辦公室、[[圖書館|圖書館]]、[[警署|警署]]和伯吉斯公園（Burgess Park）（設有各項康樂設施）。

== 人口 ==
[[File:Cafe_borrone.jpg|thumb]]，是門洛帕克鬧市著名的午餐聚腳點]]

根據2000年[[人口普查|人口普查]]{{GR|2}}，門洛帕克有30,785人、12,387戶，當中7,122個家庭在市內居住。[[人口密度|人口密度]]是1,173.4人/平方公里（3,040.1人/平方英里）。該市有12,714個住屋單位，平均密度達484.6人/平方公里（1,255.5人/平方英里）。人口中，[[白人|白人]]佔72.35%，[[亞洲人|亞洲人]]占7.15%，[[非裔美國人|非裔美國人]]佔7.03%，[[美國原住民|美國原住民]]佔0.44%，[[太平洋島國人|太平洋島國人]]占1.26%，其他[[種族|種族]]佔8.56%，而3.21%是[[混血兒|混血兒]]。各種族中的[[西班牙人|西班牙人]]和[[拉丁美洲人|拉丁美洲人]]共佔人口的15.60%。

在12,387戶中，有26.6%與未滿18歲孩童同住，43.2%是同住[[婚姻|夫婦]]，8.5%是未婚獨居女性，42.5%是非原生家庭。戶籍中，32.1%是個體戶，11.4%是65歲或以上獨居長者。戶籍平均大小是2.41人，而家庭平均大小是3.12人。

人口分佈方面，未滿18歲人士佔21.9%, 18-24歲佔6.2%，25-44歲佔34.7%，45-64歲佔21.4%，65歲或以上佔而15.9%。年齡中位數是37歲，男女比例為每100個女性相對有94.0個男性，每100個18歲或以上女性相對有90.6個男性。

門洛帕克的戶籍收入中位數是84,609[[美元|美元]]，而家庭收入中位數是105,550美元。男性收入中位數是79,766美元，而女性收入中位數是51,101美元。該市的[[人均收入|人均收入]]是53,341美元，約有4.2%的家庭與6.9%的人口處於[[貧窮線|貧窮線]]下，當中8.8%未滿18歲，7.3%年屆64歲或以上。

== 特色 ==
這裡是許多部門和著名大公司的所在地，包括：
* [[美國地質調查局|美國地質調查局]]西部地區地理辦公室的所在地
* [[沙丘路|沙丘路]]（Sand Hill Road）是[[矽谷|矽谷]]不少[[風險投資|風險投資]]公司的選址
* [[SRI國際|SRI國際]]（SRI International，前身為[[史丹福研究所|史丹福研究所]]）的所在地
* [[SLAC國家加速器實驗室|史丹福線性加速器中心]]的所在地
* 《[[夕陽雜誌|夕陽雜誌]]》（''Sunset Magazine''）的公司辦公室和展覽花園
* [[Google公司|Google公司]]於1998年9月7日在當地創立（後來遷至[[山景城_(加利福尼亞州)|山景城]]）
* 毗鄰[[史丹福大學|史丹福大學]]和[[史丹福購物中心|史丹福購物中心]]
* [[聖派屈克神學院_(門洛帕克)|聖派屈克神學院暨大學]]的所在地
* [[福樂北加州|福樂北加州]]（Fuller Northern California，[[福樂神學院|福樂神學院]]的分校）的所在地
* [[Geron公司|Geron公司]]（Geron Corporation）的所在地。該公司是一家[[生化科技|生化科技]]公司，專注於[[端粒|端粒]]（telomere）和[[幹細胞|幹細胞]]的研究
* [[圓桌武士比薩|圓桌武士比薩]]（Round Table Pizza）第一家連鎖店的所在地
* 著名社交網站[[facebook|facebook]]的總公司所在地

== 交通 ==
由於門洛帕克鄰近[[史丹福大學|史丹福大學]]，加上缺乏幹線或高速幹道接駁[[280號州際公路_(加利福尼亞州)|280號州際公路]]與[[101號美國國道|101號美國國道]]，該市在高峰時間和上學時間會飽受嚴重的[[交通擠塞|交通擠塞]]。該市的幹線網絡大部分是由雙線道路組成。

門洛帕克制定了針對上午2時-5時在住宅街道停車的政策<ref>[http://www.ci.menlo-park.ca.us/police/parking.html MPPD Parking Permits<!-- Bot generated title -->] {{webarchive|url=https://web.archive.org/web/20070314002852/http://www.ci.menlo-park.ca.us/police/parking.html |date=2007-03-14 }}</ref>，由門洛帕克警察部在每晚執行。車輛初犯時會受到一次警告，若然再犯，就會被處罰35[[美元|美元]]的告票。居住在市內某些公寓大樓的居民，由於大樓缺乏充足的居民停車位，他們須購買全年停車許可證。每6個月，居民須為賓客購買50張停車許可證（2005年4月，賓客許可證收費10美元，每包有5張）。殘疾人士車輛可獲豁免。
