《'''爆笑吸血鬼'''》（{{lang|ja|マスターモスキートン}}）是1996年11月21日－1997年4月19日發售，由[[赤堀悟|赤堀悟]]、[[根岸弘|根岸弘]]擔任原作的[[日本|日本]][[OVA|OVA]]，全6話。據赤堀悟所言，本作的女主角伊娜荷是以他的老婆[[北川美幸|北川美幸]]為原型<ref>小黑祐一郎著『この人に話を聞きたい アニメプロフェッショナルの仕事 1998-2001』 [[飛鳥新社|飛鳥新社]]、2006年</ref>。

== 故事大綱 ==
為了得到據說能使持有者擁有不老不死之力的歐帕斯，菲特曼波雷家的大小姐伊娜荷與吸血鬼摩斯基頓訂立了讓他復活的「血之契約」。與伊娜荷一起生活的摩斯基頓老是被任性的她耍得團團轉，一直不停遇上困境。為了找尋歐帕斯，摩斯基頓與伊娜荷一起展開冒險。

== 電視播放 ==
1997年1月，由[[京都放送|KBS京都]]將OVA改成電視版播放。開頭還有播放赤堀悟所演出的作品解説，另外播放結束後也進行送禮物的通知。

== 漫画化 ==
《爆笑吸血鬼》有被改編成漫畫，由[[赤堀悟|赤堀悟]]、[[根岸弘|根岸弘]]擔任原作，[[磯俣務|磯俣務]]作画，[[角川書店|角川書店]]發行日文版，[[東立出版社|東立出版社]]以《吸血鬼軍團》為名發行台灣中文版，全4集。
