{{engname|《颐和园》|''Summer Palace''}}是一部2006年中国大陆电影，由[[娄烨|娄烨]]执导，[[郭晓冬|郭晓冬]]、[[郝蕾|郝蕾]]、[[胡伶|胡伶]]和[[张献民|张献民]]主演。影片讲述了两名青年男女跨越十几年的的感情纠纷，其中穿插着[[六四事件|六四事件]]等社会背景。本片也是中国大陆首部男女主角正面全裸出镜的电影<ref>[http://variety.com/2006/film/reviews/summer-palace-1200516187/ Review: ‘Summer Palace’]. 本片于2004年9月开机，2005年5月关机，拍摄时长九个月。取景地点包括[[北京|北京]]、[[重庆|重庆]]、[[武汉|武汉]]、[[柏林|柏林]]、[[北戴河区|北戴河]]、[[图们市|图们]]。《颐和园》于2006年5月18日在[[第59届戛纳电影节|戛纳电影节]]首映。电影中的政治色彩与大量的性爱场景在中国大陆受到关注<ref>{{Cite web|url=http://ent.qq.com/a/20120416/000341.htm|title=专访娄烨：我想更加自由地拍电影|accessdate=2016-02-27|author=|date=2012-04-16|publisher=腾讯网}}</ref>，因未经中国[[国家广播电影电视总局|国家广播电影电视总局]]批准而擅自参加[[第59届戛纳电影节|戛纳电影节]]，本片在中国大陆被封禁，导演与制片人受到了处罚。

==剧情==
1987年，中朝边境[[图们市|图们]]。邮递员晓军去邮局拿信件，之后来到余红父亲开的杂货铺。余红打开信件，发现自己被北京的“北清大学”录取。余红在一个篮球场与晓军见面，晓军与几个打篮球的人发生争执，被打得鼻青脸肿。晚上在草丛中，余红与晓军发生了性行为。

余红乘火车来到了北清大学，开始了大学生活。一天，余红结识了隔壁宿舍的女生李缇，聊天中她告诉余红她男朋友若古在柏林留学。李缇和余红与回国后的若古见面，并与在同一大学的周伟结识。余红逐渐与周伟走近并与其发生性关系。两人在[[颐和园|颐和园]][[昆明湖|昆明湖]]泛舟，在夕阳下相依。

余红去周伟宿舍，发现他正与一个女生一起吃饭，余红见状离开。周伟去台球厅找余红，余想挣脱周伟的怀抱，但最后顺从。在一次欢愉后，余红要周伟去结扎，并称那样就不疼了。周询问缘由，余说是心理学老师告诉她的，并且她和他上过床。北清大学的学生开始去天安门，余红几人也一同跟随。李缇与周伟在宿舍进行性行为时，被学校人员发现。余红见到了周伟的一名舍友，他告诉她教务处已经知道了周李的事情，让她不要再去见周伟。

余红遇到了来北京探望她的晓军，两人和衣而卧，半夜晓军悄然离去。余红发现晓军不见，急忙到操场寻找。在操场示威过程中枪声响起，学生们开始四散。余红的舍友冬冬找到周伟，告诉她余红不见了。周伟急忙带领冬冬寻找余红，但未果而返。冬冬再次找到周伟，告诉他余红已离开。若古离开北京返回柏林，余红与晓军则返回图们。当时恰为1989年。

1991年，余红离开图们去往深圳。1994年，周伟与李缇到达柏林。次年，余红与深圳好友王波到达武汉。余红在一间卡拉OK唱歌，之后与一个中年男性去了一间餐厅。下雨天，余红来到中年男人家中，两人拥吻缠绵。余红接受了其单位同事吴刚的爱意，并与之发生了性关系。余红去医院打胎，与吴刚提出分手，并决定去重庆。周伟也打算去重庆，在柏林的一次聚会中，李缇从高楼落下自杀身亡。周伟回到重庆后，在偶然的机会下联系上了余红，而此时余红已为人妻。两人见面后余红借口去买酒而离开，周伟等了一晚上后驾车离开，而余红在路上看到了驾车途中的周伟。

== 主题 ==
导演娄烨称这部影片探讨的是爱情。他认为余红和周伟最美好的瞬间是在颐和园湖面上泛舟的场景，这是片名《颐和园》的含义。<ref>{{Cite web|url=http://news.qq.com/a/20060520/000484.htm|title=《颐和园》亮相戛纳　娄烨　我是“限制级导演”|accessdate=2016-02-27|author=|date=2006-05月-20|publisher=腾讯新闻}}</ref>主演郝蕾认为“《颐和园》是一个少女的成人礼，也是娄烨他们整整一代人的青春记忆”<ref name=":0" />。也有评论认为《頤和園》是一部愛情電影，卻也是一部政治電影，只是愛情濃烈明朗，政治隱晦迂迴，用愛情來掩護政治<ref>{{Cite web|url=http://app2.atmovies.com.tw/eweekly/EB1793431343/|title=婁燁頤和園：政治巨靈|accessdate=2016-02-27|author=|date=|publisher=開眼電影網}}</ref>。

== 制作 ==
2004年9月，《颐和园》在北京开机<ref>{{Cite web|url=http://yule.sohu.com/20041004/n222343091.shtml|title=娄烨执导电影《颐和园》秘密开机 郭小冬主演|accessdate=2016-02-27|author=|date=2004-10-04|publisher=搜狐}}</ref>。2005年5月影片在柏林关机，拍摄历时长九个月。剧组先后辗转北京、重庆、武汉、柏林、北戴河、图们六个城市，导演娄烨从900多本拷贝中剪出了9个小时的素材，最终制作出140分钟的“征求意见版”。<ref>{{Cite web|url=http://www.ycwb.com/gb/content/2006-04/22/content_1112725.htm|title=戛纳电影节完全名单公布：《颐和园》将竞争金棕榈|accessdate=2016-02-27|author=|date=2006-04-22|publisher=金羊网|archive-url=https://web.archive.org/web/20170305002346/http://www.ycwb.com/gb/content/2006-04/22/content_1112725.htm|archive-date=2017-03-05|dead-url=yes}}</ref>

== 发行 ==
2006年5月18日，《颐和园》于法国[[坎城影展|坎城影展]]首映<ref name="Ocean">{{cite web|url=http://www.ocean-films.com/unejeunessechinoise/|title=Une Jeunesse Chinoise (Summer Palace)|accessdate=2008-01-20|publisher=Océan Films|archiveurl=https://web.archive.org/web/20071217185326/http://www.ocean-films.com/unejeunessechinoise/|archivedate=2007-12-17|deadurl=yes}}</ref>。其后陆续在加拿大、韩国、香港、新加坡、美国、日本、台湾等地上映。

=== 家庭媒体 ===
《颐和园》于2008年1月28日在法国释出二区版DVD。单碟版包括原声普通话配法文字幕的影片、制作特辑纪录片、一个有关审查的特辑、娄烨的电影笔记和剧组的传记。<ref name="Ocean"/>一区版DVD于2008年3月11日在美国释出<ref>{{cite web|url=http://movies.yahoo.com/movie/1809423283/dvdinfo|title= Summer Palace (2008) - DVD/Video| publisher= Yahoo Movies |accessdate=2008-01-25}}</ref>。

== 评价 ==
《颐和园》在多个国际电影节上映，最引人注目的是[[戛纳电影节|戛纳电影节]]<ref name="festival-cannes.com">{{cite web |url=http://www.festival-cannes.com/en/archives/ficheFilm/id/4352475/year/2006.html |title=Festival de Cannes: Summer Palace |accessdate=2009-12-13|work=festival-cannes.com}}</ref>，它是当年唯一参加角逐的亚洲电影<ref>{{cite web|url=http://www.hollywoodreporter.com/hr/search/article_display.jsp?vnu_content_id=1002384372|title = 'Palace' producers challenge Beijing censors|author =Landreth, Jonathon |publisher = ''[[The_Hollywood_Reporter|The Hollywood Reporter]]''|date =2006-04-22|accessdate=2008-01-20}}</ref>，但未获得任何奖项<ref name="festival-cannes.com" />。

评论人士的评价总体积极，他们表扬了电影的野心及广度，但许多人抱怨140分钟显得太长了。《[[綜藝_(雜誌)|综艺]]》杂志的德里克·埃利（Derek Elley）称电影“有半个小时是多余的”<ref>{{cite web | url =http://www.variety.com/index.asp?layout=features2006&content=jump&jump=review&dept=cannes&nav=RCannes&articleid=VE1117930547 | title = Summer Palace| accessdate = 2007-08-13| author = Elley, Derek |date= 2006-05-18 | publisher = ''[[Variety_(magazine)|Variety]]''}}</ref>，《[[每日电讯报|每日电讯报]]》也有相同的说法，但表示尽管如此影片仍是“一部未打磨和混乱的新作品”<ref>{{cite web | url =

=== 禁令 ===
2006年5月15日，《颐和园》送到中国[[国家广播电影电视总局|国家广播电影电视总局]]接受审查。次日电影局转述了审片委员会的审片意见，称“画面太暗，看不清，声音也听不清楚”，从而拒绝审查<ref>{{Cite web|url=http://news.sina.com.cn/o/2006-05-17/02568943085s.shtml|title=戛纳电影节18日开幕 《颐和园》被拒审|accessdate=2016-02-27|author=|date=2006-05-17|publisher=新浪}}</ref>。17日再度送审，同样被拒绝。但娄烨坚持以私人名义带领主演郭晓冬、郝蕾等出席电影记者会及首映礼<ref>{{Cite web|url=http://news.boxun.com/news/gb/china/2006/05/200605190912.shtml|title=电影《颐和园》涉六四内容遭北京封杀|accessdate=2016-02-27|author=|date=2006-05-19|publisher=博讯新闻网}}</ref>。9月1日，导演娄烨因违规参赛，和该片的制片人耐安一起，被广电总局处罚5年内不得拍片<ref>[http://ent.sina.com.cn/m/c/2006-09-04/09151229884.html 娄烨违规参赛戛纳电影节 五年内不准拍片(图)]</ref>。但在被禁的5年内，娄烨仍参与了两部电影的拍摄<ref>{{Cite web|url=http://fj.qq.com/a/20120417/000120.htm|title=流放者娄烨：曾以为《颐和园》是最后一部电影|accessdate=2016-02-27|author=|date=2012-04-17|publisher=腾讯福建频道}}</ref>。
