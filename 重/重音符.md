'''重音符''' 或 '''抑音符'''（grave accent；  '''` '''；{{IPA|◌̀}}、{{IPAc-en|us|ɡ|r|eɪ|v}} or {{IPAc-en|uk|ɡ|r|ɑː|v}}）<ref>[http://www.oxforddictionaries.com/view/entry/m_en_gb0347890#m_en_gb0347890 Oxford Dictionaries]</ref>是一個[[變音符號|變音符號]]，用於[[希臘語|希臘語]]（1982年前）、[[法語|法語]]、[[加泰羅尼亞語|加泰羅尼亞語]]、[[威爾士語|威爾士語]]、[[義大利語|義大利語]]、[[越南語|越南語]]、[[蘇格蘭蓋爾語|蘇格蘭蓋爾語]]、[[挪威語|挪威語]]、[[葡萄牙語|葡萄牙語]]和其他語言內。

== 高度 ==
重音符在一些的[[羅曼語族|羅曼語族]]內用來標記母音 ''e'' 和 ''o'' 的[[構音高度|構音高度]]。在[[法語|法語]]、[[義大利語|義大利語]]和[[加泰羅尼亞語|加泰羅尼亞語]]裡，有這標記的母音表示其為[[半閉母音|開的]]。

在[[加泰羅尼亞語|加泰羅尼亞語]]裡，重音符被用來標記[[重音|重音]]及某些重音母音的不同音量，如 ''è'' {{IPA|[ɛ]}}對''é'' {{IPA|[e]}}，或''ò'' {{IPA|[ɔ]}}對''ó'' {{IPA|[o]}}。而其子母 ''a'' 只能取重音符而不能取[[銳音符|銳音符]]，''i'' 和 ''u'' 則相反。

在[[義大利語|義大利語]]裡，重音符除了用來標記重音（當放在 ''a''、''i''、''u'' 以上時）之外，亦用來標記[[半開母音|開放]]發音：''è'' = {{IPA|[ɛ]}} (對應於''é'' = {{IPA|[e]}}); ''ò'' = {{IPA|[ɔ]}} (對應於''ó'' = {{IPA|[o]}})。例子和更多的訊息請見[[銳音符|銳音符]]。

在[[法語|法語]]裡，重音符有兩種用途。放在字母''e''上時，它標記母音的不同音量：''è'' {{IPA|[ɛ]}}和''e'' {{IPA|[ə]}}。放在字母''a''和''u''上時，它對發音沒有作用，而只是用來分別拼寫相同的同音異義字而已。

在[[越南語|越南語]]裡，重音符用來標記聲調中的'''玄聲'''（[[陽平|陽平]]）。

===編程用途===
重音符在編程方面有獨立（非和其他字元結合）的用途，在此情況又可稱為「反引號（''backquote''）或 ''backtick''。

==== {{tsl|en|command substitution|指令替代}} ====

大部份[[Unix_shell|Unix shell]]以及程式語言如[[Perl|Perl]]、[[PHP|PHP]]以及[[Ruby_(programming_language)|Ruby]]等都以成對的重音符作指令替代，意思是以某一個指令的輸出結果作為另一個指令的輸入項。例如以下指令：

 <tt>echo It is now `date`</tt> 

在替代指令後會等如：

 <tt>echo It is now {{#time:D M j H:i:s "GMT" Y}}</tt>

在正式執行時會產生以下輸出結果：

 <tt>It is now {{#time:D M j H:i:s "GMT" Y}}</tt>

==== 資料庫 ====
在SQL語句中，重音符可用以標示表格或欄位……等的名稱。
