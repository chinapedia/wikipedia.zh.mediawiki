{{Otheruses|重生}}
在[[基督教|基督教]]中，'''重生'''指[[人|人]]“'''属灵的再生'''”，与每个人都经历的身体的第一次出生相对。“重生”一词来源于{{Bible|约翰福音|3|3}}：“耶稣回答说，我实实在在的告诉你，人若不'''重生'''{{NoteTag|另一种翻译为“从上面生”。}}，就不能见[[神的国|神的国]]。”<ref>{{CUV|约翰福音|3|3}}</ref>

这是一个与[[基督教救赎论|基督教救赎论]]有关的术语。在大部分[[基督教历史|基督教历史]]中，“重生”被理解为通过[[洗礼|洗礼]]这种[[圣礼|圣礼]]达到的精神再生，[[罗马天主教会|罗马天主教会]]和[[东正教教会|东正教教会]]至今依然是如此理解。

== 起源 ==
《圣经》中第一次提到“重生”是在{{Bible|约翰福音|3}}中，出现在耶稣对称为[[法利赛人|法利赛人]]的犹太教拉比[[尼哥底母|尼哥底母]]的教导当中。

《[[圣经|圣经]]》中与“重生”相关的术语包括新的出生、复活、新生活、新造、心灵的更新、向罪死和向义活、从黑暗转入光中等<ref name="Hodge1">{{cite web |last = Hodge |first = Charles.|title = "Regeneration." ''Systematic Theology-Volume III.''|url = http://www.ccel.org/ccel/hodge/theology3.iii.i.i.html |language = en}}</ref>。

== 公共立场 ==
在当代，“重生”一词与自1960年代后期以来从美国开始，后来遍及世界各地的福音复兴运动有着广泛的联系。“重生”被用来指一种激烈转变的经历，日益成为一个术语，用来识别虔诚的信徒。

1976年，水门事件的同谋者科尔森的书《重生》获得了国际上的注意。他被时代杂志评为“美国最具影响力的25位[[福音派|福音派]]之一。”<ref>{{cite web |url = http://www.time.com/time/covers/1101050207/photoessay/5.html |title = The 25 Most Influential Evangelicals in America |language = en}}</ref>。这个词由于这一年的总统候选人[[吉米·卡特|吉米·卡特]]将自己形容为“重生”而更加普及，特别是在花花公子杂志第一次采访美国总统候选人时。现代音乐家小理查德彭尼曼，<ref name="whitep83">{{cite book |last = White |first = Charles |year = 2003 |page = 83 (see text under photo on opposite page) |title=''The Life and Times of Little Richard: The Authorised Biography''|publisher = Omnibus Press}}</ref>、[[鲍勃·迪伦|鲍勃·迪伦]],<ref>{{cite book |editor = Cott |title = ''Dylan on Dylan: The Essential Interviews'' |pages = 279–285}}</ref>、[[戴夫·马斯泰恩|戴夫·马斯泰恩]]、[[约翰尼·卡什|约翰尼·卡什]]、[[奇里夫·李察|奇里夫·李察]]<ref>{{cite web |url = http://www.cbn.com/media/player/index.aspx?s=/vod/AL24v2_WS |title = Lou Gramm Knows What Love Is }}</ref>，都是对现代文化产生了影响的重生的艺术家，其他还有摔跤明星[[尚恩·麥可|尚恩·麥可]]、[[克里斯·傑利可|克里斯·傑利可]]、[[杰西麦卡尼|杰西麦卡尼]]，也被提及是重生的。前任[[阿拉巴马州|阿拉巴马州]]州长和美国总统候选人[[乔治·华莱士|乔治·华莱士]]在1970年代末经历了重生，他也为自己早些时候的[[种族隔离|种族隔离]]观点表示道歉。
