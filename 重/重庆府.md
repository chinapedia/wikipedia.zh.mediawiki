'''重庆府'''，[[南宋|南宋]]设置的一个[[府_(行政区划)|府]]。

[[淳熙|淳熙]]十六年（1189年），[[宋孝宗|宋孝宗]][[內禪|內禪]][[宋光宗|宋光宗]]，光宗升[[恭州_(北宋)|恭州]]置，[[治所|治所]]在[[巴县_(北周)|巴县]]（今[[重庆市|重庆市]]）。属[[夔州路|夔州路]]。[[元朝|元朝]][[至元_(忽必烈)|至元]]十六年（1279年）改为[[重庆路|重庆路]]。辖境扩大。[[明代|明代]][[洪武|洪武]]初复改为府。民國二年（1913年）废。

== 轄境 ==
重慶府轄境在不同時期，各有損益，其曾所轄範圍，大致如下：
* 現[[重慶直轄市|重慶直轄市]]所轄各區縣，不含原屬[[潼川府|潼川府]]的[[潼南縣|潼南縣]]。
* 四川省[[武勝縣|武勝縣]]（原屬重慶府，即定遠縣）、[[瀘州市|瀘州市]]、[[江安縣|江安縣]]、[[納溪縣|納溪縣]]、[[合江縣|合江縣]]（重慶府瀘州，含所轄三縣）、[[開江縣|開江縣]]（原屬夔州府、重慶府，即新寧縣）、[[達州市|達州市]]（原屬夔州府、重慶府，即達縣）、[[宣漢縣|宣漢縣]]（原屬夔州府、重慶府，即東鄉縣）、[[萬源縣|萬源縣]]（原屬夔州府、重慶府，即太平縣）。
* 湖北省[[建始縣|建始縣]]（原屬夔州府、重慶府）

== 元代的重慶路<ref>《大元一統志》卷五</ref> ==
至元十六年(1279年)在重庆府设置四川南道宣慰司和重庆路，立重庆路总管府，除录事司领巴县、江津、南川三县外，领泸、忠、合、涪等州及石柱宣慰司。
**[[巴县_(北周)|巴縣]]、[[江津縣|江津縣]]、[[南川縣|南川縣]]
**[[合州|合州]]（領縣三：[[石照縣|石照縣]]、[[銅梁縣|銅梁縣]]、[[定遠縣_(四川省)|定遠縣]]）、大足縣？
**[[忠州|忠州]]（領縣三：[[臨江縣|臨江縣]]、[[豐都縣|豐都縣]]、[[南賓縣_(忠州)|南賓縣]]）、墊江縣？
**[[涪州|涪州]]（領縣一：[[武龍縣|武龍縣]]）
**[[瀘州|瀘州]]（領縣三：[[江安縣|江安縣]]、[[納溪縣|納溪縣]]、[[合江縣|合江縣]]）
**石柱宣慰司

== 明代的重慶府<ref>《明史》卷四十三 志第十九</ref> ==
* 洪武四年，為府，領縣、州，裁[[石照縣|石照縣]]、[[臨江縣|臨江縣]]、[[南賓縣_(忠州)|南賓縣]]
**[[巴县_(北周)|巴縣]]、[[江津縣|江津縣]]、[[昌寧縣|昌寧縣]]、[[大足縣|大足縣]]（分合州屬大足縣屬府）、[[綦江縣|綦江縣]]（[[播州宣慰司|播州宣慰司]]來屬）、[[南賓縣_(忠州)|南賓縣]]（洪武四年省），夔州府之[[新寧縣_(四川省)|新寧縣]]（現開江縣）來屬。
**[[合州|合州]]（領縣二：[[銅梁縣|銅梁縣]]、[[定遠縣_(四川省)|定遠縣]]）
**[[忠州|忠州]]（領縣二：[[豐都縣|豐都縣]]、[[墊江縣|墊江縣]]）
**[[涪州|涪州]]（領縣二：[[武龍縣|武龍縣]]、[[彭水縣|彭水縣]]）
**[[瀘州|瀘州]]（領縣三：[[江安縣|江安縣]]、[[納溪縣|納溪縣]]、[[合江縣|合江縣]]）
* 洪武五年十二月庚辰，裁黔江縣入彭水縣<ref>《明太祖實錄》卷之七十七洪武五年十二月甲戌朔</ref>。
* 洪武六年，[[瀘州|瀘州]]改直隸州，屬四川行省。
* 洪武六年九月己亥，置[[長壽縣|長壽縣]]，屬[[涪州|涪州]]，尋改屬府。<ref>「洪武六年九月己亥朔置重慶府涪州長壽縣」《明太祖實錄》卷之八十五洪武六年九月己亥朔。《明史稿》作“洪武四午置此縣，屬濟州，六年改屬”</ref>
* 洪武六年十二月甲寅，析巴縣地置[[永川縣|永川縣]]、[[榮昌縣|榮昌縣]]<ref>《明太祖實錄》卷之八十六洪武六年十一月戊戌朔，《明史稿》作洪武四年。</ref>
* 洪武七年，省[[昌寧縣|昌寧縣]]入[[榮昌縣|榮昌縣]]。
* 洪武中，省[[南賓縣_(忠州)|南賓縣]]入[[酆都縣|酆都縣]]
* 洪武九年四月甲午，[[夔州府|夔州府]]降為散州，其原領[[巫山縣|巫山縣]]、[[大寧縣_(四川省)|大寧縣]]（現巫溪縣）、[[雲陽縣|雲陽縣]]、[[萬縣|萬縣]]、[[開縣|開縣]]、[[建始縣|建始縣]]、[[梁山縣_(四川省)|梁山縣]]（現梁平縣）、[[達縣|達縣]]八縣並屬重慶府。
* 洪武十年五月戌寅，夔州升直隸州，其原領[[巫山縣|巫山縣]]、[[大寧縣_(四川省)|大寧縣]]（現巫溪縣）、[[雲陽縣|雲陽縣]]、[[萬縣|萬縣]]、[[開縣|開縣]]、[[建始縣|建始縣]]、[[達縣|達縣]]七縣屬之，[[新寧縣_(四川省)|新寧縣]]（現開江縣）裁入[[梁山縣_(四川省)|梁山縣]]（現梁平縣），屬忠州。
省[[南川縣|南川縣]]入[[綦江縣|綦江縣]]、省[[豐都縣|豐都縣]]入[[涪州|涪州]]，[[武龍縣|武龍縣]]入[[彭水縣|彭水縣]]，[[彭水縣|彭水縣]]往屬[[涪州|涪州]]<ref>《明太祖實錄》卷112</ref>。
* 洪武十三年十一月庚戌，復置[[南川縣|南川縣]]；復置[[豐都縣|豐都縣]]，改名[[酆都縣|酆都縣]]；復置[[武龍縣|武龍縣]]，改名[[武龍縣|武龍縣]]；分忠州屬[[梁山縣_(四川省)|梁山縣]]（現梁平縣）還屬夔州府<ref>《明太祖實錄》卷134</ref>。
* 洪武十四年六月癸未，復置[[黔江縣|黔江縣]]，屬重慶府<ref>《明太祖實錄》卷137</ref>.
* 成化十七年九月庚寅，析[[銅梁縣|銅梁縣]]、[[遂寧縣|遂寧縣]]地置[[安居县_(明朝)|安居縣]]<ref>《明憲宗實錄》卷219：“其地乃商賈聚集之處．去縣遠，數遭賊寇焚劫，因分銅梁縣四里，遂寧縣一里為安居縣”</ref>。
* 成化十九年三月丙辰，分[[巴縣|巴縣]]地復置[[璧山縣|璧山縣]]。<ref>《明憲宗實錄》238：“復設四川壁山縣。壁山故有縣，後併入重慶府之巴縣，裏民訴稱其地去巴縣三百餘裏，在萬山之中，乞仍設縣。鎮守等官，勘奏以為宜，從之。</ref>

== 清代的重慶府 ==
* 順治初，因明制，領州三，縣十七。
* 康熙元年（1662年），省[[銅梁縣|銅梁縣]]、[[安居县_(明朝)|安居縣]]入合州，省[[璧山縣|璧山縣]]入[[永川縣|永川縣]]，省[[武隆縣|武隆縣]]入[[涪州|涪州]]。
* 康熙八年（1669年），省[[定遠縣_(四川省)|定遠縣]]入[[合州|合州]]。
* 康熙六十年（1721年），復置[[銅梁縣|銅梁縣]]，以[[安居縣|安居縣]]併入。
* 雍正六年（1728年），復置[[大足縣|大足縣]]、[[璧山縣|璧山縣]]、[[定遠縣_(四川省)|定遠縣]]三縣。
* 雍正十一年（1733年）冬十月乙卯，改重慶府屬之[[黔江縣|黔江縣]]、隸重慶府分駐黔江同知管轄。兼轄附近[[彭水縣|彭水縣]]。升府屬之[[忠州|忠州]]為[[忠州直隸州|忠州直隸州]]。兼轄[[墊江縣|墊江縣]]、[[酆都縣|酆都縣]]、及[[夔州府|夔州府]]屬之[[梁山縣|梁山縣]]三縣<ref>《大清世宗憲皇帝實錄卷》之一百三十六雍正十一年癸丑冬十月己酉朔</ref>。
* 雍正十二年（1734年），改[[黔江縣|黔江縣]]、[[彭水縣|彭水縣]]二縣置[[黔彭直隸廳|黔彭直隸廳]]。
* 雍正十三年（1735年），又改平茶長官司為[[秀山縣|秀山縣]]，屬[[黔彭直隸廳|黔彭直隸廳]]。
* 乾隆元年（1736年），改[[黔彭直隸廳|黔彭直隸廳]]為[[酉陽直隸州|酉陽直隸州]]，以黔、彭、秀 三縣來隸。
* 乾隆二十九年（1662年），以巴縣江北鎮置[[江北廳|江北廳]]。
