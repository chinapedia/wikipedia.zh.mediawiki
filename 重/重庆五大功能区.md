'''重庆五大功能区'''是[[重庆市|重庆市]]在2013年9月提出的关于重庆市功能区域划分的一项[[政策|政策]]。它将重庆市划分为都市功能核心区、都市功能拓展区、城市发展新区、渝东北生态涵养发展区和渝东南生态保护发展区等五个功能区域<ref>{{cite web |url=http://www.cq.gov.cn/today/news/2013/9/15/1056332.shtml |title=中共重庆市委四届三次全会召开：建设五大功能区等 |date=2013-09-15 |accessdate=2016-04-30 |publisher=重庆市人民政府 }}</ref>。

==都市功能核心区 ==
包括：重庆市[[渝中区|渝中区]]全域和大渡口区、[[江北区|江北区]]、[[沙坪坝区|沙坪坝区]]、九龙坡区、[[南岸区|南岸区]]处于内环以内的区域，面积294平方公里<ref>
{{cite web |url=http://www.bing.com/knows/search?q=%E9%83%BD%E5%B8%82%E5%8A%9F%E8%83%BD%E6%A0%B8%E5%BF%83%E5%8C%BA&mkt=zh-cn |title=重庆都市功能核心区 |accessdate=2016-04-30 |publisher=Bing Knows }}</ref>。

== 都市功能拓展区 ==
除都市功能核心区以外的重庆主城区。
*重庆主城区：渝中区、[[大渡口区|大渡口区]]、江北区、沙坪坝区、九龙坡区、南岸区、[[北碚区|北碚区]]、渝北区、[[巴南区|巴南区]]。
*都市功能拓展区：大渡口区、江北区、沙坪坝区、[[九龙坡区|九龙坡区]]、南岸区部分区域，及北碚区、[[渝北区|渝北区]]、巴南区。
