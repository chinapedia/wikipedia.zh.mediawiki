'''重庆工商大学派斯学院'''是经[[中华人民共和国教育部|中华人民共和国教育部]]批准设立的[[全日制|全日制]][[本科|本科]]普通高等学校<ref>教发函[2003]541号</ref>，位于[[重庆市|重庆市]][[合川区|合川区]]。

== 校史更迭 ==

*1937年，成立四川省国立第二中学
*1950年，在四川省国立第二中学基础上设立合川师范学校
*2002年，并入重庆工商大学，更名[[重庆工商大学|重庆工商大学]]派斯学院<ref>{{cite web| url=http://www.paisi.edu.cn/index.asp |title=历史沿革 |accessdate=2016-01-05 |publisher=重庆工商大学派斯学院 |  }}</ref>

== 教学单位 ==

该校共有6个二级学院，分别是：

#会计学院
#管理学院
#金融学院
#文学与传媒学院
#软件工程学院
#外国语学院

== 教育情况 ==

*图书馆：拥有馆藏纸质图书116万册，报刊600余种<ref>{{cite web |url=http://lib.paisi.edu.cn/shownews.asp?BigClass=271&smallclass=&id=7847 |title=图书馆简介 |accessdate=2015-01-05 |publisher=重庆工商大学派斯学院图书馆 }}</ref>
*学生：学院有全日制在校学生约10000人
*教师：该校有专任教师600余人，正教授46人，具有高级职称的教师178人

== 学校文化 ==
*[[校训|校训]]：厚德博学、求是创新
*办学理念：质量立校、特色兴校、人才强校
