{{Japanese-horizontal|《'''謎解音'''》|ナゾトキネ}}是於2016年10月至12月播放的短篇[[日本電視動畫|日本電視動畫]]，每話10分鐘。

類型為解謎逃脫動畫。其動畫工作室位於[[立川市|立川市]]{{tsl|ja|曙町 (立川市)||曙町}}，動畫中亦以立川市為舞台<ref>{{Cite web|url=http://tachikawa.keizai.biz/headline/2270/|script-title=ja:立川が舞台の謎解き脱出アニメ「ナゾトキネ」放映開始へ|date=2016-10-04|accessdate=2016-10-06|publisher=立川経済新聞|language=ja}}</ref>。

片尾曲畫面中設置了迷你謎題，解謎時需要在正篇動畫中尋找提示，可以到官方網站測試答案是否正確。

== 故事簡介 ==
解音原本正為了即將來訪的客戶打掃會議室，沒想到周遭突然被光芒圍繞而無法從謎之空間「Quiz'n」出去！想要出去的話，心須按照奇怪生物的指示將謎題解開。解開謎題後可以暫時回到現實世界，然而解音不時會重新進入Quiz'n。

== 登場人物 ==
=== 主要人物 ===
; {{Nihongo|網野解音|網野 解音（あみの ときね）|聲：{{tsl|ja|原奈津子}}}}
: 廣告代理商「TEN2」的社長秘書。實際上是隱性動畫和特攝宅，處於{{Nihongo|Quiz'n|クイズン}}中時會被強制Cosplay。

; {{Nihongo|哈琴|ハッチン|聲：[[德井青空|德井青空]]}}
: 下半身處在帽子裡的謎樣生命體，位於謎之空間「Quiz'n」內。被解音叫作{{Nihongo|「豚八」|ぶたはち}}，帶有「{{lang|ja|けども}}」的語尾。人類靈光乍現時結晶化的{{Nihongo|Q Stone|Qストーン}}正是牠所追求之物。

; {{Nihongo|水上鄉華|水上 郷華（みなかみ きょうか）|聲：[[松井惠理子|松井惠理子]]}}
: 「TEN2」的接待人員，與解音同期。喜歡整理整頓。

; {{Nihongo|山墨芳英|山墨 芳英（やまずみ よしえ）|聲：[[八島莎拉拉|八島莎拉拉]]}}
: 「TEN2」的接待人員，與解音同期。能有效率地完成工作。

; {{Nihongo|柳澤夏子|柳澤 なつこ（やなぎさわ なつこ）|聲：[[佐佐木未來|佐佐木未來]]}}
