'''漲落定理'''是[[統計力學|統計力學]]中的一個定理，用來處理遠離[[熱力學平衡|熱力學平衡]]（熵最大值）之下，系統的[[熵|熵]]會在某一定時間中增加或減少的相對機率。[[熱力學第二定律|熱力學第二定律]]預測一獨立系統的熵應該趨向增加，直到其達到平衡為止，但在統計力學被發現之後，物理學家了解到第二定律只是統計上的一種行為，因此應該總是有一些機率會使得獨立系統的熵會自發性地''減少''；漲落定理準確地量化了此機率。

== 定理概述和实例 ==

波动耗散定理说，当存在着消耗能量，将其转化为热能（例如，摩擦）的方法，存在相关的逆过程[[Thermal_fluctuation|的热波动]]。通过考虑一些例子可以最好地理解这一点：
* ''[[Drag_(physics)|拖动]]和[[Brownian_motion|布朗运动]]''
:: 如果一个对象是通过流体移动，它就会有[[Drag_(physics)|阻力]]（空气阻力或流体阻力）。阻力消耗动能，将其转化为热量。相应的波动[[Brownian_motion|布朗运动]]。流体中的物体不静止，而是随着小的且快速变化的速度移动，因为流体中的分子碰撞到其中。布朗运动将热能转换成动能 - 与阻力相反。
* ''[[Electrical_resistance_and_conductance|电阻]]和[[Johnson_noise|约翰逊噪声]]''
:: 如果电流通过导线环带运行的[[Resistor|电阻]]，因为阻力电流会迅速变为零。电阻消耗电能，把它变成热量（[[Joule_heating|焦耳热]]）。相应的波动就是[[Johnson_noise|约翰逊噪声]]。其中具有电阻器的导线回路实际上不具有零电流，其具有由电阻器中的电子和原子的热波动引起的小且快速波动的电流。约翰逊噪声将热能转换为电能 - 与电阻相反。
* ''[[Absorption_(electromagnetic_radiation)|光吸收]]和[[Thermal_radiation|热辐射]]''
:: 当光照射物体时，光的一部分被吸收，使得物体更热。这样，光吸收将光能转换成热。相应的波动是[[Thermal_radiation|热辐射]]（例如，“红热”对象的发光）。热辐射将热能转换为光能 - 光吸收的相反。事实上，[[Kirchhoff's_law_of_thermal_radiation|热辐射的基尔霍夫定律]]证实了更有效的物体吸收光，其就会放射更多的热辐射。
:: 

== 具体的例子 ==
: 波动耗散定理是一个[[Statistical_thermodynamics|统计热力学]]量化之间波动的系统中的关系[[Thermal_equilibrium|热平衡]]，并且系统的施加扰动的响应的一般的结果。
: 因此，该模型允许例如：使用分子模型在线性响应理论中来预测材料性质。该定理假设应用扰动，如机械力或电场，足够弱以至于rates of [[Relaxation|Relaxation]]保持不变。
* 布朗运动
: 例如，[[爱因斯坦|爱因斯坦]]在他1905年论文上指出[[布朗运动|布朗运动]]是相同的随机的力导致在布朗运动的粒子的不稳定的运动也将导致拖如果颗粒是通过流体拉动。换句话说，如果试图在特定方向上干扰系统，则静止时粒子的波动具有与必须消除的摩擦力相同的原点。  根据该观察爱因斯坦能够利用[[统计力学|统计力学]]推导出[[爱因斯坦关系|爱因斯坦-Marian Smoluchowski關係]]:<math> D =  {\mu \, k_B T} </math>
:: 其连接[[菲克定律|扩散常数]]<math>D</math>和颗粒迁移率<math>{\mu \,}</math>，粒子的终端漂移速度的所施加的力的比率。<math>k_B</math>是[[Boltzmann_constant|玻耳兹曼常数]]，并且<math>T</math>是[[绝对温度|绝对温度]]。
