'''军机大臣列表'''，旨在列出[[中国|中国]][[清朝|清朝]]自[[雍正|雍正]]七年（1729年）至[[宣统|宣统]]三年（1911年）历任[[军机大臣|军机大臣]]。军机大臣雅称为“枢密”，属于任务性质，而非定制官衔，无品阶可言，担任者亦无俸禄。

[[雍正帝|雍正帝]]初为指挥青海战事之需要，于[[雍正|雍正]]七年（1729年）设立军需房；后形成“办理军机处”，沿为定制，负责辅佐皇帝处理军国大政。雍正时首任军机大臣为[[张廷玉|张廷玉]]与[[鄂尔泰|鄂尔泰]]，恩遇最隆。自[[军机处|军机处]]设立，军机大臣取代[[内阁大学士|内阁大学士]]成为清朝事实上的宰相，但与宰相职务不同的是此为没有任何品阶的非定制职衔；直到清末官制改定，理论上其功能为新内阁所取代，然而清末[[宣统|宣统]]三年（1911年）首任[[内阁总理大臣_(清朝)|内阁总理大臣]][[庆亲王|庆亲王]][[奕劻|奕劻]]，原亦为领班军机王大臣。

军机大臣由[[亲王|亲王]]、[[大学士|大学士]]、[[尚书|尚书]]、[[侍郎|侍郎]]或京堂充任，无定员，一般六、七人。其中资历最深者为“首席军机大臣”，或称“领班军机大臣”，但没有正式的名义；入值军机时按资历深浅的分别称“军机处行走”、“军机大臣上行走”、“军机大臣上学习行走”三种；“军机处行走”名目从[[嘉庆|嘉庆]]后期起即极少使用，仅[[恭亲王|恭亲王]][[奕䜣|奕䜣]]以议政王入值军机时使用过一次。

== 雍正年间 ==
[[雍正|雍正]]七年（1729年），清軍大舉征伐西北兩路，六月初十（7月5日）正式設立'''軍需房'''於[[隆宗門|隆宗門]]內北侧<ref>[[梁章鉅|梁章鉅]]《樞垣記略》卷廿七「（雍正七年），以内阁在[[太和门|太和门]]外，儤直者多虑漏泄事机，始设军机房于隆宗门内。」</ref>，选内阁中谨慎縝密者入值，以处理紧急军务。旋即改稱“辦理軍需處”或“軍需處”，再改稱“辦理軍務處”<ref>[[莊吉發|莊吉發]]《清代奏摺制度》:「自從雍正七年設立軍需房後，先改稱軍需處或辦理軍需處，旋稱辦理軍務處。」</ref>、“軍機房”<ref>《歷史檔案》1990年4期載趙志強〈軍機處成立時間考訂〉:「至七年六月始改為軍機房。」</ref>。雍正八年（1730年），始稱“軍機處”<ref>《光绪会典事例》載乾隆帝上谕:「第自雍正八年设立军机处以来五十余年，所有谕旨、批奏事件，未经发钞者尚多。」</ref>，以張廷玉、蔣廷錫、[[馬爾賽|馬爾賽]]入值辦理一切事務<ref>錢實甫 編《清代職官年表》一冊，頁一三六</ref>，成為定制。雍正十年（1732年）三月，正式定名“辦理軍機處”，简称“軍機處”。

雍正十三年（1735年）八月，[[乾隆帝|乾隆帝]]即位後，以西北軍事底定，撤除軍機處，改設“總理事務處”兼理軍機事務，以[[莊親王|莊親王]][[允祿|允祿]]、[[果親王|果親王]][[允禮|允禮]]、[[保和殿|保和殿]][[大學士|大學士]][[鄂爾泰|鄂爾泰]]、張廷玉為總理事務王大臣，原軍機大臣[[訥親|訥親]]、[[徐本|徐本]]、[[班第|班第]]、[[索柱_(康熙乙未進士)|索柱]]、[[豐盛額|豐盛額]]、[[海望|海望]]、[[莽鵠立|莽鵠立]]、[[納延泰|納延泰]]改在總理處「協辦總理事務」或「差委辦事」。

== 乾隆年间 ==
乾隆帝即位后，以“宽猛相济”理念施政，先后平定[[新疆|新疆]]、[[蒙古|蒙古]]，并大兴[[文字狱|文字狱]]、编写《[[四库全书|四库全书]]》。还使[[四川|四川]]、[[贵州|贵州]]等地继续[[改土归流|改土归流]]，人口不断增加，在乾隆末年时突破了三亿大关，约占当时世界人口的三分之一，统治期间与[[康熙|康熙]]、[[雍正|雍正]]三朝合称“[[康雍乾盛世|康雍乾盛世]]”。乾隆在位后期，宠信[[和珅|和珅]]等奸臣，朝政日益贪污腐败，各地发生民变。

乾隆六十年（1795年），乾隆帝[[禅位|禅位]]于子颙琰，为[[太上皇|太上皇]]，但依然“[[训政|训政]]”，在宫内仍然沿用乾隆年号，仍为实际上的最高统治者，直至嘉庆四年（1799年）初驾崩。

[[乾隆|乾隆]]二年十一月廿八日（1738年1月17日），准總理事務王大臣解職，復設“辦理軍機處”，以鄂爾泰、張廷玉、訥親、海望、納延泰、班第入值。

== 嘉庆年间 ==
嘉庆是[[清仁宗|清仁宗]]颙琰在位年号。嘉庆帝在位前四年并无实权，乾隆帝死后才独掌大权。嘉庆帝掌权后，肃清吏治，惩治贪官[[和珅|和珅]]等人。然而在位后期，贪污问题更加严重。嘉庆帝在位期间，清朝发生了[[白莲教之乱|白莲教之乱]]和[[天理教之乱|天理教之乱]]，八旗生计、河道漕运等问题也日益凸显。嘉庆帝与其子[[道光帝|道光帝]]统治期间，清朝政治危机日益严重，史称“[[嘉道中衰|嘉道中衰]]”。嘉庆二十五年七月二十五日（1820年9月2日），嘉庆帝死于[[承德避暑山庄|承德避暑山庄]]，遗诏传位于次子绵宁。

== 道光年间 ==
[[道光|道光]]是[[清宣宗|清宣宗]]旻宁在位年号。道光帝即位后，本人力行节俭，勤于政务，并进行整顿吏治，整厘盐政，通海运，平定[[张格尔之乱|张格尔之乱]]，严禁鸦片，起到了一定积极作用。然而清朝已经积重难反，国势进一步衰落。道光十九年（1839年），[[林则徐|林则徐]]进行[[虎门销烟|虎门销烟]]，次年[[第一次鴉片戰爭|第一次鴉片戰爭]]爆發，由于道光帝战守无策，時和時戰，再加上武器装备上的差距，清朝战败于[[英国|英国]]，并与英人簽訂近代中国的首條[[不平等條約|不平等條約]]《[[南京條約|南京條約]]》，割讓[[香港岛|香港岛]]及開放[[五口通商|五口通商]]，被视为[[中国近代史|中国近代史]]的开始。道光三十年（1850年）正月，道光帝在[[圆明园|圆明园]]去世，遗诏传位于四子奕詝。

== 咸丰年间 ==
[[咸丰|咸丰]]是[[清文宗|清文宗]]奕詝在位年号。咸丰帝即位后勤于政事，对朝政进行大幅度改革。然而此时，清朝在内发生[[太平天国运动|太平天国运动]]，在外发生[[第二次鸦片战争|第二次鸦片战争]]，最后英法联军攻入北京，火烧[[圆明园|圆明园]]，清朝以签定一系列不平等条约收场。随后为扭转内外交困的局面，清朝开始[[洋务运动|洋务运动]]。

咸丰十一年七月十七日（1861年8月22日），咸丰帝崩于[[承德|承德]][[避暑山庄|避暑山庄]]，遗诏以皇子载淳即位。命怡亲王[[载垣|载垣]]、郑亲王[[端华|端华]]、御前大臣[[景寿|景寿]]、协办大学士[[肃顺|肃顺]]，军机大臣[[穆荫|穆荫]]、[[匡源|匡源]]、[[杜翰|杜翰]]、[[焦佑瀛|焦佑瀛]]等八人为[[赞襄政务八大臣|赞襄政务八大臣]]。此后，[[慈禧太后|慈禧太后]]、[[慈安太后|慈安太后]]联合恭亲王[[奕䜣|奕䜣]]发动[[祺祥政变|祺祥政变]]，开启了[[慈禧太后|慈禧太后]]统治清朝的历史。

== 同治年间 ==
[[同治|同治]]是[[清穆宗|清穆宗]]载淳在位年号，其在位初期由[[慈禧太后|慈禧太后]]、[[慈安太后|慈安太后]]垂帘听政，随后大权逐步滑入慈禧太后之手。在此期间，清朝镇压[[太平天国|太平天国]]、[[捻军|捻军]]等各地民变，并大力开展[[洋务运动|洋务运动]]，清朝一时出现“[[同治中兴|同治中兴]]”的局面。同治十二年（1873年）同治帝亲政，次年十二月初五（公历1875年1月12日）崩于紫禁城养心殿。由于同治帝无子，醇亲王[[奕譞|奕譞]]之子[[载湉|载湉]]即位，两宫皇太后继续垂帘听政。

== 光绪年间 ==
[[光绪|光绪]]是[[清德宗|清德宗]]载湉在位年号。 因[[同治帝|同治帝]]无子，被两宫皇太后立为帝，起初由慈安、慈禧两宫太后垂帘听政。光绪七年（1881年）[[慈安太后|慈安太后]]崩逝后由[[慈禧太后|慈禧太后]]一人垂帘。在此期间，发生[[中法战争|中法战争]]，前期由于清军战败，慈禧太后将以恭親王[[奕䜣|奕䜣]]为首的军机大臣全部斥退，史称“甲申易枢”。光绪十五年（1889年），光绪帝亲政，实际上大权仍掌握在慈禧太后手中。

光绪二十年（1894年），发生[[中日甲午战争|中日甲午战争]]，以清朝战败，签订《[[马关条约|马关条约]]》告终。随后，列强纷纷在中国划定势力范围。光绪二十四年（1898年），光绪帝实行[[戊戌变法|戊戌变法]]，但却受到以慈禧太后为首的保守派镇压，从此被慈禧太后幽禁在[[中南海|中南海]][[瀛台|瀛台]]。此后又发生[[八国联军侵华|八国联军侵华]]，清朝被迫签订《[[辛丑条约|辛丑条约]]》，内外危机进一步加重。为缓解危机，慈禧太后不得不开展[[清末新政|清末新政]]、[[预备立宪|预备立宪]]等措施。光绪三十四年十月二十一日（1908年11月14日），光绪帝遭毒杀，次日慈禧太后亦崩。

== 宣统年间 ==
宣统为中国清朝第十二个皇帝、末代皇帝[[溥仪|溥仪]]的在位年号，溥仪为[[清宣宗|清宣宗]]曾孙、醇亲王[[载沣|载沣]]之子。宣统帝即位时年仅三岁，其父载沣为[[监国|监国]][[摄政王|摄政王]]。在此期间，清朝[[立宪运动|立宪运动]]声势愈发浩荡，[[孙中山|孙中山]]领导的[[中国同盟会|中国同盟会]]也在国内频频发动暴动。宣统三年四月初十（1911年5月8日），清政府宣布废除军机处，实行[[责任内阁|责任内阁]]制，史称“[[皇族内阁|皇族内阁]]”。同年八月十九日（10月10日），[[武昌起义|武昌起义]]爆发，随后全国各地纷纷响应，[[中华民国|中华民国]]建立，清朝统治土崩瓦解；十二月二十五日（1912年2月12日），宣统帝下诏退位，清朝灭亡。
