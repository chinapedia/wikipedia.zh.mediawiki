'''辽宁路'''是[[中国|中国]][[天津市|天津市]][[和平区_(天津市)|和平区]]的一条街道，修筑于1919年，原来跨[[天津法租界|天津法租界]]和[[天津日租界|天津日租界]]2个[[天津租界|租界]]。辽宁路西到佳木斯道，东到[[营口道|营口道]]，长1148米。目前，辽宁路地区是[[天津市历史风貌建筑|天津市历史风貌建筑]]的聚集地之一，大量的名人故居坐落于此。<ref>[http://www.tjhp.gov.cn/servlet/ContentServer?cid=1105523261039&pagename=tjhp%2FPage%2Fsub-t46&lv=2&lv2=1100820937330&lv1=1087380370033&c=Page 和平区志]{{dead link|date=十月 2017 |bot=InternetArchiveBot }}</ref>

== 历史 ==
辽宁路原来跨[[天津法租界|天津法租界]]和[[天津日租界|天津日租界]]2个[[天津租界|租界]]。[[天津法租界|天津法租界]]部分分为两个部分：从圣路易路（Rue Saint Louis）（今[[营口道|营口道]]）到霞飞路（Rue Cours Joffre）（今[[花园路_(天津)|花园路]]）原名为李泽河路（Rue de L'yser），从霞飞路（Rue Cours Joffre）（今[[花园路_(天津)|花园路]]）到[[天津日租界|天津日租界]]的秋山街（今锦州道）段原名为梅大夫路（Rue Mesny）。[[天津日租界|天津日租界]]部分从秋山街（今锦州道）到吾妻街（今佳木斯道），原名常磐街。1946年，[[天津市政府_(中华民国)|天津市政府]]光复后，改名为辽宁路至今。

== 现况与发展 ==
辽宁路目前西到佳木斯道，东到[[营口道|营口道]]，全长1148米，宽7米，两侧人行道均宽5米，为沥青路面。沿途建有有[[百货大楼_(天津)|天津百货大楼]]、[[章瑞庭旧宅|章瑞庭旧宅]]、[[张公撝旧居|张公撝旧居]]、[[原久大精盐公司大楼|久大精盐公司驻津办事处]]、[[乔铁汉故居|乔铁汉故居]]等建筑。

== 辽宁路小吃街 ==
{{天津主要商业街}}
[[中华人民共和国|中华人民共和国]]成立初期，辽宁路由于靠近[[天津劝业场|天津劝业场]]便成为当时的商铺聚集之地，多以小吃为主，其中较为有名的是小豆粥和牛肉烧饼等小吃。20世纪80年代末，由于[[天津劝业场|天津劝业场]]的改造，辽宁路小吃街曾一度消失。20世纪90年代，辽宁路又出现多家小吃夜市。<ref>[http://news.enorth.com.cn/system/2009/04/04/003948613_01.shtml 辽宁路小吃街“盛装”迎食客 16日正式开街]</ref>2009年，[[天津市人民政府|天津市人民政府]]对辽宁路小吃街进行整修，有多家天津小吃[[老字号|老字号]]如：[[耳朵眼炸糕|耳朵眼炸糕]]、[[石头门坎素包|石头门坎素包]]、天宝楼酱货、曹记驴肉、认一民牛肉烧饼、天津茶汤李等入驻辽宁路小吃街。此外，辽宁路上的正阳春、三合益、宴宾楼等餐厅也是天津著名的老字号酒店。<ref>[http://shop.tj.soufun.com/2009-4-24/2537829_1.htm 天津辽宁路小吃街开街 50余种小吃穿街品尝]</ref>
