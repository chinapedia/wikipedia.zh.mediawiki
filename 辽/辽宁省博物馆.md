'''辽宁省博物馆'''，是[[中華人民共和國|中國]][[辽宁省|辽宁省]]内规模最大的综合性博物馆，[[国家一级博物馆|国家一级博物馆]]、[[中央地方共建国家级博物馆|中央地方共建国家级博物馆]]。

== 历史沿革 ==
* 1933年，[[满洲国|伪满洲国]]在原奉系军阀[[汤玉麟|汤玉麟]]官邸建立'''满洲国立中央博物馆奉天分馆'''。
* 1946年，[[中国抗日战争|中国抗日战争]]胜利后由[[国民政府|国民政府]]接管，改为'''国立沈阳博物院古物馆'''，系当时国内三大博物院之一。<ref name="辽博的10个全国第一">{{cite web| url=http://www.lnmuseum.com.cn/lbgl.do?method=gl| title=辽博的10个全国第一| publisher=新华网| accessdate=2004-11-12| language=zh-hans| deadurl=yes| archiveurl=https://web.archive.org/web/20130306082313/http://www.lnmuseum.com.cn/lbgl.do?method=gl| archivedate=2013-03-06}}</ref>
* 1948年11月，[[东北人民政府|东北人民政府]]文物管理委员会接收该馆。
* 1949年，利用馆内原有基建，扩充馆藏后正式建立了'''东北博物馆'''，这成为[[中共建政|中共建政]]后的第一座博物馆。
* 1958年1月5日，改为'''辽宁省博物馆'''。
* 1983年2月21日，馆址西侧扩建展览楼4000平方米，扩建文物库房1000平方米。
* 1985年10月16日，[[博物馆之友|博物馆之友]]召开第一次会议，首批发展博物馆之友31人，同期馆工会成立。
* 1997年3月6日，辽宁省博物馆新馆（市府广场）立项，新建地址位于沈阳市市府广场东侧。
* 2004年11月12日，筹建六年的辽宁省博物馆新馆（市府广场）开馆，新馆展厅总面积8531平方米，设12个展厅。
* 2008年5月18日，被[[国家文物局|国家文物局]]评定为首批[[国家一级博物馆|国家一级博物馆]]。
* 2009年，被国家文物局和[[中华人民共和国财政部|中华人民共和国财政部]]认定为首批[[中央地方共建国家级博物馆|中央地方共建国家级博物馆]]。
* 2011年8月，因应辽宁省举办[[中华人民共和国第十二届运动会|第十二届全国运动会]]，辽宁省博物馆新馆在沈阳市浑南区正式破土动工。
* 2015年5月16日，辽宁省博物馆新馆正式向公众开放。新馆占地面积达8.32万平方米，是目前全国面积最大的省级博物馆。运营后的新馆首展1417件文物珍品，其中馆藏文物1281件。

== 展览陈列 ==
常规展览有历史陈列“古代辽宁”；“明清瓷器展”、“明清玉器展”、“中国古代碑志展”、“辽代陶瓷”等专题陈列，另有若干临时特展。

== 主要馆藏 ==
[[File:Northern_Yan_duck-shaped_glass.jpg|thumb]]]]
博物馆馆内现有馆藏文物近120000件（套），收藏范围以辽宁地区考古出土文物和传世的历史艺术类文物为主体，藏品包含珍贵文物数万件，分考古、书法、绘画、丝绣、青铜、陶瓷等17个门类。

其中曾被宫廷收藏的传世文物，如晋唐宋元书画、宋元明清[[缂丝|缂丝]][[刺绣|刺绣]]、历代碑志、古地图、历代货币等影响有别于其他珍贵文物。法书中的《曹娥诔辞》、《仲尼梦奠帖》、《古诗四帖》；绘画中的《[[簪花仕女图|簪花仕女图]]》、《[[虢国夫人游春图|虢国夫人游春图]]》、《[[瑞鹤图|瑞鹤图]]》；拓片中的《熹平石经》和《正始石经》都是并世无双的名作。这一批清宫旧藏奠定了辽宁省博物馆在历史艺术领域的专业地位。

辽宁地区考古出土文物构成馆藏的另一部分。[[红山文化|红山文化玉器]]、[[青銅器|商周时期窖藏青铜器]]、辽代瓷器、明清版画除了反映出各自的时代特征之外，还富于东北的地方以及民族特色。
