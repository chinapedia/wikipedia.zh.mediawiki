'''迭戈·巴勃罗·西蒙尼·冈萨雷斯'''（{{lang-es|'''Diego Pablo Simeone González'''}}），绰号'''匪徒'''（{{lang-es|'''El Cholo'''}}），出生於[[阿根廷|阿根廷]][[布宜諾斯艾利斯|布宜諾斯艾利斯]]，是一名[[阿根廷|阿根廷]][[足球|足球]]領隊及前[[足球|足球]]運動員，世界足壇最佳阿根廷巨星之一。目前正執教[[西甲|西甲]]球隊[[馬德里競技俱樂部|馬德里競技]]。

他擅長踢防守中場的位置，球風鐵血剛硬，是1990年代[[阿根廷國家足球隊|阿根廷國家足球隊]]的隊長及靈魂人物，曾是國家隊上陣次數紀錄最高保持者''（2007年2月被[[罗伯托·阿亚拉|-{zh-hans:罗伯托·阿亚拉;zh-hk:羅拔圖艾耶拉;zh-tw:羅貝托·阿亞拉;}-]]打破紀錄）''，由1988年至2002年曾經替國家隊上陣106場。

西蒙尼的教練生涯頗受好評，將長年成績低迷的馬競調教為一流強隊，不論聯賽或歐冠都具奪冠實力。風格延續球員時期的激情鐵血，善於鼓動球員及球迷氣氛，與現時利物浦教練[[尤爾根·克洛普|高普]]並稱兩大激情派教練。

== 球會生涯 ==
西蒙尼在1987年開始職業球員生涯，在[[沙士菲|沙士菲]]渡過三赛季後，出國遠征到[[意大利甲組聯賽|意大利甲組聯賽]]球會[[比薩足球會|比薩]]發展，不幸地是，施蒙尼效力第一季球隊便面臨降级，但他仍然跟隨球隊降落乙組聯賽。1992年轉會至[[西維爾足球會|西維爾]]，球風強悍的他在球隊上获得一個重要的位置，無論防守或進攻都有出色的水準。直至1995年加盟[[馬德里體育會|馬德里體育會]]令足球生涯開始平步青雲，施蒙尼在1995/96年球季不單在聯賽射入多達12球，更協助球隊同時奪得該季聯賽及杯赛冠軍，施蒙尼的作用顯得他有如球隊的發電機，而且擁有充足的體力，時常奔跑全場去努力搶球，需要的時候可以擔當攻擊中場上前作出致命的助攻或射門。翌年獲得豪门球會[[國際米蘭|國際米蘭]]邀請加盟，他在球隊中擔任防守中場的角色，加盟首季立即成為隊中的正選，並帶領國際米蘭晉身[[欧洲联盟杯|欧洲联盟杯]]決賽，最後憑[[伊万·萨莫拉诺|森莫蘭奴]]、[[哈维尔·萨内蒂|查維·辛尼迪]]及[[罗纳尔多|朗拿度]]的入球，3比0擊敗[[拉齐奥足球俱乐部|拉齐奥]]奪得冠軍。隨後1999年轉會拉齐奥，除了協助球隊贏得26年來首個聯賽錦標外，並且擊敗旧主國際米蘭奪得[[意大利盃|意大利盃]]冠軍成就雙料冠軍。

2003年施蒙尼重返馬德里體育會，但當時年屆33歲的他已經不復當年勇，翌年決定返回祖國[[阿根廷|阿根廷]]加盟[[阿維拉達競賽會|阿維拉達競賽會]]，2006年2月17日在一場對[[拉普拉塔大學生|拉普拉塔大學生]]中最後一次為球會上陣後，施蒙尼宣佈退休，正式結束了19年的球員生涯。退休後他改任足球領隊，競賽會由於為了挽留施蒙尼，決定選擇他出任執教競賽會。

== 國際隊生涯 ==
施蒙尼在1988年7月14日首次對國家隊上陣，該場賽事阿根廷大敗於澳洲1比4，其後協助國家奪得1991年及1993年[[美洲國家盃|美洲國家盃]]冠軍，亦曾經代表阿根廷出戰1996年奧運，並先後擊敗西班牙及葡萄牙殺入決賽面對尼日利亞，可惜當時後備入替的他在對手臨完場的入球以2比3飲恨。施蒙尼曾經帶領阿根廷出席過[[1994年世界盃足球賽|1994年]]、[[1998年世界盃足球賽|1998年]]及[[2002年世界盃足球賽|2002年]]三屆[[世界盃足球賽|世界盃]]，其中1998年世界盃十六強賽事迎戰[[英格蘭國家足球隊|英格蘭]]，下半場初段对對方球員[[碧咸|碧咸]]犯规，其後他被對方報復性以「虎尾腳」勾跌，當時這連串行為被旁邊的球證看得清清楚楚，後施蒙尼被罰黃牌而將碧咸則紅牌被逐，間接地令阿根廷在該場賽事擊敗英格蘭，往後此比賽一直成為爭議。
2002年12月20日對[[墨西哥國家足球隊|墨西哥]]時打破了前國家隊後衛{{tsl|en|Oscar Ruggeri|雷基利}}的97場紀錄成為阿根廷上陣次數的最高紀錄保持者。[[2002年世界盃足球賽|2002年世界盃]]分組賽對[[英格蘭國家足球隊|英格蘭]]，除了施蒙尼及碧咸兩大「仇人」再度碰頭成為賽事焦點外，這場比賽亦成為施蒙尼最後一次為阿根廷國家足球隊上陣，當時已為國家隊隊長的碧咸主射十二碼得手，令英格蘭憑此球以1比0擊敗阿根廷得以復仇成功，隨後阿根廷被[[瑞典國家足球隊|瑞典]]1比1迫和而無緣出線。

2002年世界盃後，施蒙尼曾經說：「1978年的那批球員是我們一生的偶像，雖然1982年的他們沒有帶回勝利，可我們永遠把他們視為英雄。我和我的隊友們已經竭盡全力了，三屆世界盃的遺憾是永遠的。我被刺傷了，很痛，很痛……」

== 戰術風格 ==
後腰出身的西蒙尼非常重視防守，也是防守反擊的簇擁者，打法剛硬鐵血，善用體力戰進行高位搶逼對付控球打法的球隊，不過面對同樣龜縮防守反擊的球隊時西蒙尼則毫無應對之法，所以常發生遇強則強、遇弱則弱的情形，主動進攻能力不佳是西蒙尼跟其他著名防守反擊派教練最大的差別。
