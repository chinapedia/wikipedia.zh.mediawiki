'''师胜杰'''（{{bd|1953年|4月3日<ref name="hljwht">{{cite web |title=文化厅>文化龙江>文化名人>师胜杰 |url=http://www.hljwht.gov.cn/index.php/home/whlj/detail/id/5542.html |website=黑龙江省文化厅}}</ref>|2018年|9月28日<ref>{{cite news |title=著名相声表演艺术家师胜杰因病逝世 享年66岁 |url=http://ent.sina.com.cn/s/m/2018-09-28/doc-ihkmwytp7734672.shtml |accessdate=2018-09-28 |work=新浪娱乐}}</ref>|}}），出生于天津，中国[[相声|相声]]演员，[[国家一级演员|国家一级演员]]。

==生平==
1953年4月，师胜杰出生于[[天津市|天津]][[和平区_(天津市)|和平区]]南市<ref>{{Cite web|url=http://tj.sina.com.cn/cul/news/2013-10-25/1007809.html|title=师胜杰：想回老家天津买房 直言春晚毁相声|accessdate=2017-05-24|author=|date=2013-10-25|work=|publisher=新浪|archive-url=https://web.archive.org/web/20160408065652/http://tj.sina.com.cn/cul/news/2013-10-25/1007809.html|archive-date=2016-04-08|dead-url=yes}}</ref><ref>{{Cite web|url=http://ent.cntv.cn/mingxing/shishengjie/|title=师胜杰-明星库|accessdate=2017-05-24|author=|date=|work=|publisher=中国网络电视}}</ref>，祖籍[[河北省|河北]][[武清|武清]]（今属[[天津市|天津市]]）[[杨村街道|杨村]]，父亲师世元、母亲高秀琴均是[[相声|相声]]演员<ref name="ent163">{{cite web |title=师胜杰被侯宝林收为关门弟子 曾公开支持郭德纲 |url=http://ent.163.com/18/0929/07/DSRTO1AH00038FO9.html |website=网易娱乐 |date=2018-09-29}}</ref>。6岁半时，原本随祖母在天津生活的师胜杰到[[黑龙江省|黑龙江省]][[哈尔滨|哈尔滨]]投奔之前已在那里安家工作的父亲<ref name="tjrb"/>，居住在曲艺表演丰富的[[道外|道外]][[北市场|北市场]]。7岁时，师胜杰首次登台，与父亲合作表演了相声《[[捉放曹|捉放曹]]》。后来，师胜杰拜天津相声演员[[朱相臣|朱相臣]]为师。师胜杰被选入哈尔滨市少年宫艺术团，表演歌剧、快板和相声，曾经为国家主席[[刘少奇|刘少奇]]、国务院总理[[周恩来|周恩来]]及外宾[[西哈努克亲王|西哈努克亲王]]等演出<ref name="ent163"/>。

1969年8月，16岁的师胜杰到[[黑龙江生产建设兵团|黑龙江生产建设兵团]][[上山下乡|下乡]]当[[知青|知青]]，因[[政治审查|政治审查]]不合格，被安排到八团三十四连进行劳动改造。1975年，师胜杰被抽调到兵团文艺宣传队，不久后与[[姜昆|姜昆]]结对表演相声。1976年，师胜杰和姜昆合作的相声《林海红英》在的黑龙江省文艺调演中脱颖而出，两人受到了一些专业文艺团体的关注，调演结束后，姜昆被[[中国广播艺术团|中国广播艺术团]]说唱团团长[[马季|马季]]收于麾下，后来成为马季的第一个徒弟，而师胜杰因为政审问题，只能黯然返回兵团。1976年底，师胜杰被调到黑龙江省龙江剧院曲艺队，才终于成为专业相声演员<ref name="ent163"/><ref name="sohu">{{cite web |title=《男人世界》：相声背后的师胜杰 |url=http://yule.sohu.com/20070522/n250155195.shtml |website=搜狐 |date=2007-05-22}}</ref>。

1984年，师胜杰到[[山东省|山东省]][[青岛市|青岛市]]参加全国相声新作演出评比活动，并凭相声《肝胆相照》获得一等奖；而且他的出色表演受到了活动艺术顾问、相声宗师[[侯宝林|侯宝林]]的赞赏，被侯宝林收为关门弟子，并在青岛友谊宾馆举行了拜师仪式。1985年底，师胜杰被评为中国“[[十大笑星|十大笑星]]”之一，是入选者中唯一的非北京籍演员。1987年，他与[[原建邦|原建邦]]创作，和[[于浮生|于浮生]]表演的相声《小鞋匠的奇遇》在[[中国艺术节|中国首届艺术节]]荣获了金奖。1989年，当选为黑龙江曲艺家协会主席<ref name="hljwht" />。

师胜杰曾多次参加[[中国中央电视台春节联欢晚会|中国中央电视台春节联欢晚会]]，央视春晚作品有1989年小品《招聘》，1997年相声《打传呼》，1998年相声《同桌的你》，2002年群口相声《马年说马》等<ref name="hljwht" />。并在《CCTV全国相声大赛》、《星光大道》等综艺节目中担任过评委<ref name="ent163"/>。

2018年9月28日，因癌症去世<ref>[https://news.sina.cn/2018-09-29/detail-ifxeuwwr9315412.d.html?from=wap 姜昆悼念诗透露师胜杰罹患癌症].新浪网.</ref>。

== 家庭 ==
师胜杰的父亲师世元是杨村人，最早学织洋袜子，后来因为喜欢曲艺成为一名票友，再后来拜熙醒生为师成为相声艺人，1940、1950年代在[[北京|北京]]、天津一带颇有名气。师胜杰的母亲高秀琴是孤儿，从小学老旦，后因嗓子坏了，就给师世元作[[捧哏|捧哏]]并结为夫妻<ref name="tjrb">{{cite news |author1=郑长宁 |author2=丁佳文 |title=师胜杰 抖响现实主义包袱儿 |url=http://news.163.com/14/0918/08/A6DLR03L00014AED.html |work=天津日报 |date=2014-09-18}}</ref>。1959年，师世元将家迁到位于[[中国东北地区|中国东北地区]]的[[黑龙江省|黑龙江省]][[哈尔滨|哈尔滨]]，出任哈尔滨民间艺术团相声队队长<ref name="ent163"/>，是东三省相声的开拓者之一<ref name="tjrb"/>。师胜杰在家中排行老二，有一个哥哥<ref name="ent163"/>。1965年底，因受政治运动影响，父亲师世元自杀，哥哥被划为[[现行反革命|现行反革命]]。1978年，父亲和哥哥得到[[平反冤假错案|平反]]<ref name="sohu"/>。1981年6月，师胜杰与妻子宋艳结婚，两人婚后育有一女，名为师艺文。
