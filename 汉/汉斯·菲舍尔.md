'''汉斯·菲舍尔'''（{{lang-de|'''Hans Fischer'''}}，{{bd|1881年|7月27日|1945年|3月31日|catIdx=F}}），[[德国|德国]][[有机化学|有机化学]]家，1930年[[诺贝尔化学奖|诺贝尔化学奖]]获得者。<ref name="nobel_1930">{{cite web |url=http://nobelprize.org/nobel_prizes/chemistry/laureates/1930/index.html |title=The Nobel Prize in Chemistry 1930 |publisher=[[诺贝尔基金会|诺贝尔基金会]] |date= |accessdate=2008年12月23日 |language=en }}</ref>

== 早年经历 ==
汉斯·菲舍尔1881年7月27日出生于德国[[法兰克福|法兰克福]]。他先后在[[斯图加特|斯图加特]]和[[威斯巴登|威斯巴登]]念[[小学|小学]]和[[中学|中学]]。1899年从中学毕业后，菲舍尔先后在[[瑞士|瑞士]][[洛桑大学|洛桑大学]]和德国[[马尔堡大学|马尔堡大学]]同时修读[[化学|化学]]和[[医学|医学]]两个专业。1904年，他获得马尔堡大学化学学位，其后在1906年和1908年分别获得医学执照和[[医学博士|医学博士]]学位。<ref name="nobel_bio">{{cite web |url=http://nobelprize.org/nobel_prizes/chemistry/laureates/1930/fischer-bio.html |title=Biography |publisher=[[诺贝尔基金会|诺贝尔基金会]] |date= |accessdate=2008年12月23日 |language=en }}</ref>

== 学术经历 ==
汉斯·菲舍尔毕业后到[[慕尼黑|慕尼黑]]当[[内科医生|内科医生]]，后来进入[[柏林|柏林]]第一化学研究所，在[[赫尔曼·埃米尔·菲舍尔|赫尔曼·埃米尔·菲舍尔]]的指导下进行科学研究。1911年，他回到慕尼黑，次年获得[[内科医学|内科医学]]讲师资格。1913年，成为慕尼黑心理学研究所的[[心理学|心理学]]讲师。1916年，成为[[因斯布鲁克大学|因斯布鲁克大学]]医学化学教授，1918年转到[[维也纳大学|维也纳大学]]。从1921年起，直到去世，他一直在[[慕尼黑工业大学|慕尼黑工业大学]]担任有机化学教授。<ref name="nobel_bio"/>

菲舍尔的科研工作主要集中于研究[[血液|血液]]和[[胆汁|胆汁]]中的[[色素|色素]]、绿色[[植物|植物]]中的[[叶绿素|叶绿素]]以及[[衍生物|衍生]]出这些色素的[[吡咯|吡咯]]。其中最重要的工作是[[胆红素|胆红素]]与[[血红素|血红素]]的合成。<ref name="nobel_bio"/>他合成了超过130种[[卟啉|卟啉]]，其中在1929年合成血红素。<ref name="msu">{{cite web |url=http://www2.chemistry.msu.edu/Portraits/PortraitsHH_Detail.asp?HH_LName=FischerH |title=Hans Fischer |publisher=[[密歇根州立大学|密歇根州立大学]]化学系 |date=2003年5月16日 |accessdate=2008年12月24日 |language=en }}</ref>1930-40年代，他对叶绿素的研究形成了100多篇论文，阐明了叶绿素的结构，重点论证叶绿素分子中含有[[二氢卟吩|二氢卟吩]]环系，且中心有1个与二氢卟吩环形成配位的[[镁|镁]]原子。这些研究成果为后来[[罗伯特·伯恩斯·伍德沃德|罗伯特·伯恩斯·伍德沃德]]合成叶绿素奠定了基础。<ref name="dlkp">{{cite web |url=http://www.dlkp.gov.cn/kejijingying/nuobeier/huaxue/1930-1.htm |title=汉斯·菲舍尔 |publisher=大连科普网 |date= |accessdate=2008年12月24日 |language=zh |archive-url=https://web.archive.org/web/20041229094259/http://www.dlkp.gov.cn/kejijingying/nuobeier/huaxue/1930-1.htm |archive-date=2004-12-29 |dead-url=yes }}</ref>

菲舍尔的研究受到广泛认可，并因此获得多项荣誉。其中在1930年，“由于对血红素和叶绿素结构的研究，以及血红素的合成”，汉斯·菲舍尔获得诺贝尔化学奖。<ref name="nobel_1930"/><ref name="xinhua">{{cite web |url=http://news.xinhuanet.com/ziliao/2008-01/09/content_7323037.htm |title=德国柏林洪堡大学 |publisher=[[新华网|新华网]] |date= |accessdate=2008年12月23日 |language=zh }}</ref>此外，[[月球|月球]]上的“菲舍尔[[环形山|环形山]]”以他和赫尔曼·埃米尔·菲舍尔的名字命名。

== 个人生活 ==
汉斯·菲舍尔1935年与维尔特鲁德·豪费结婚。<ref name="nobel_bio"/>1945年，[[第二次世界大战|第二次世界大战]]结束前不久，由于自己的实验室和研究成果在轰炸中几乎全部被毁，菲舍尔过度沮丧，在慕尼黑[[自杀|自杀]]去世。<ref name="msu"/>
