'''汉斯-威廉·穆勒-沃尔法特'''（{{lang-de|'''Hans-Wilhelm Müller-Wohlfahrt'''}}，{{bd|1942年|8月12日|catIdx=Müller-Wohlfahrt, Hans-Wilhelm}}）是一名[[德国|德国]][[骨科|骨科]][[医生|医生]]及[[运动医学|运动医学]]专家，曾供职于[[拜仁慕尼黑|拜仁慕尼黑]]，在治疗运动损伤方面处于世界领先地位<ref>{{zh}}{{Cite web |url=http://sports.163.com/12/0812/00/88LRC4P100051C97.html |title=拜仁神医迎70大寿 妙手回春曾助博尔特重返赛场 |work=网易 | date=2012-08-12|accessdate=2013-07-01}}</ref>。此外，他也是一位[[作家|作家]]及[[膳食补充剂|膳食补充剂]]企业“穆勒-沃尔法特程式保健股份公司（{{lang|de|formula Müller-Wohlfahrt Health & Fitness AG}}）”的[[董事长|董事长]]。

==生平==
沃尔法特出生于[[下萨克森州|下萨克森州]]列尔哈费（{{lang|de|Leerhafe}}，今属[[维特蒙德|维特蒙德]]）的一个[[牧师|牧师]]家庭。早年他曾是一位[[田径运动员|田径运动员]]，并在1963年于[[耶费尔|耶费尔]]玛丽恩高中（{{lang|de|Mariengymnasium Jever}}）毕业后开始进修医学课程。其后，沃尔法特供职于[[柏林|柏林]]鲁道夫·菲尔绍医院（{{lang|de|Rudolf-Virchow-Krankenhaus}}）的骨科诊所，逐渐发展为一名骨科专家，更在1971年取得[[博士|博士]]学位。

沃尔法特已婚，他与[[妻子|妻子]]目前居住于[[慕尼黑|慕尼黑]]。在1999年至2001年期间，沃尔法特的[[女儿|女儿]]玛莲·穆勒-沃尔法特（{{lang|de|Maren Müller-Wohlfahrt}}）曾与[[洛塔尔·马特乌斯|洛塔尔·马特乌斯]]相恋，遭到沃尔法特的干涉<ref>{{zh}}{{Cite web |url=http://sports.titan24.com/other/2010-01-06/58888.html |title=马特乌斯曾与沃尔法特女儿拍拖 穆里尼奥质疑医术 |work=体坛网 | date=2010-01-06|accessdate=2013-07-01}}</ref>。此外，他还有一个[[儿子|儿子]]。

==职业生涯==
沃尔法特在1975年至1977年间曾在[[德甲|德甲]]球队[[柏林赫塔|柏林赫塔]]担任医生<ref>{{zh}}{{Cite web |url=http://sports.titan24.com/other/2010-01-06/58889.html |title=沃尔法特曾任赫塔和拜仁队医 为博尔特等大腕治伤 |work=体坛网 | date=2010-01-06|accessdate=2013-07-01}}</ref>。自1977年4月起，他来到慕尼黑，成为拜仁慕尼黑的俱乐部医生。直至1996年沃尔夫冈·彼得接任前，沃尔法特一直是拜仁的随队医生，出现在球队每场比赛的替补席当中。此后，他仍会以“医务主管”的身份现场参与球队的部分比赛。他担任这一职位直至2008年11月，因与主教练[[尤尔根·克林斯曼|尤尔根·克林斯曼]]意见不合而辞职<ref name="cctv">{{zh}}{{Cite web |url=http://sports.cctv.com/20081105/116211.shtml |title=拜仁首席队医突然辞职 媒体疑其遭克林斯曼排挤 |work=中国中央电视台 | date=2008-11-05|accessdate=2013-07-01}}</ref>。在后者于2009年4月被解雇后，沃尔法特再度作为队医重回拜仁替补席，并且得到了另两位队医卢茨·汉赛尔（{{lang|de|Lutz Hänsel}}）和彼得·于布拉克（{{lang|de|Peter Ueblacker}}）的支持。自1995年以来，他还需要为[[德国国家足球队|德国国家足球队]]提供医疗服务<ref>{{zh}}{{Cite web |url=http://soccer.titan24.com/08-11-05/130364.html |title=神医31年后告别拜仁 巴拉克为他曾与切尔西翻脸 |work=体坛网 | date=2008-11-05|accessdate=2013-07-01}}</ref>。

2008年5月，沃尔法特在慕尼黑市中心的{{link-de|阿尔特霍夫|Alter Hof}}开设了一家面积达1600平方米的骨科医疗中心。而该中心的投资人则是时任另一支德甲球队[[霍芬海姆|霍芬海姆]]主席的{{link-de|迪特玛·霍普|Dietmar Hopp}}<ref name="cctv"/>。

沃尔法特的一些非正统治疗方法存在争议，例如使用牛血、[[蜂蜜|蜂蜜]]或公鸡[[鸡冠|鸡冠]]上的提取物或进行[[肌肉注射|肌肉注射]]<ref>{{en}}{{Cite web |url=http://espn.go.com/espn/otl/story/_/id/7324261/germany-dr-hans-wilhelm-muller-wohlfahrta-great-healer-quack-hyperactive-syringe |title=The doctor they call 'Healing Hans' |work=ESPN | date=2011-12-15|accessdate=2013-07-01}}</ref>。而他对运动员使用的[[顺势疗法|顺势疗法]]药物也遭到部分德国同行的质疑<ref>{{de}}{{Cite web|url=http://www.zeit.de/2003/02/M_9fller-Wohlfahrt?page=all|title=Der Arzt, seine Haut und die Pille|work=Die Zeit|number=2|year=2003|accessdate=2010-05-31|Author=Katrin Wilkens}}</ref>。尽管他的[[替代疗法|替代疗法]]和顺势疗法经常遭到批评，但他仍是国际公认的运动损伤专家，尤其是在[[膝盖|膝盖]]及[[肌腱|肌腱]]伤病领域拥有极高的威望<ref>{{en}}{{Cite web|url=http://www.dailyrecord.co.uk/sport/football/st-johnstone-star-peter-macdonald-1035543|title=St Johnstone star Peter MacDonald gets goat blood injections to cure hamstring problem|work=Daily Record|date=2009-09-02|accessdate=2013-07-01|Author=Neil Cameron}}</ref>。

2012年，沃尔法特曾帮助受肌肉伤势困扰的田径运动员[[尤塞恩·博尔特|尤塞恩·博尔特]]，并使他可以赶在[[伦敦奥运会|伦敦奥运会]]比赛前伤愈。后者在[[2012年夏季奧林匹克運動會田徑男子100公尺比賽|100米赛跑]]取得[[金牌|金牌]]后公开感谢了这位德国骨科医生，并声称沃尔法特是自己可以在奥运会蝉联冠军的一大关键人物<ref>{{zh}}{{Cite web |url=http://sports.163.com/12/1219/10/8J32LKJK00051C97.html |title=博尔特年年拜访拜仁神医 小猪带飞人游览慕尼黑 |work=网易 | date=2012-12-19|accessdate=2013-07-01}}</ref>。
