'''汉台区'''是[[中国|中国]][[陕西省|陕西省]][[汉中市|汉中市]]所辖的一个[[市辖区|市辖区]]。总面积为556平方公里，2002年人口为20万人。

== 行政区划 ==

2011年，汉中市人民政府向[[陕西省人民政府|陕西省人民政府]]申请，调整行政区划。汉台区由7个镇、2个乡、8个街道办事处变为7个镇、8个街道办事处<ref>{{Cite web |url=http://www.hanzhong.gov.cn/xxgk/gkml/zfwj/zfjzfbwj/hzzwj/201410/t20141018_21197.html |title=汉中市人民政府关于审定汉中市乡镇机构改革行政区划调整方案的请示|accessdate=2019-04-08 |author= |date=2011-04-25 |work= |publisher=汉中市人民政府网站|language=Zh-hans }}</ref>。
