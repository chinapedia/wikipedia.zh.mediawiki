'''汉魏许都故城'''又称'''张潘故城'''，位于[[河南省|河南省]][[许昌市|许昌市]]区东南18公里处的[[许昌县|许昌县]]古城村旁。2013年5月，被[[中华人民共和国国务院|中国国务院]]列入第七批[[全国重点文物保护单位|全国重点文物保护单位]]名单中<ref name="16处">{{cite web |url=http://www.21xc.com/Article/ShowArticle.asp?ArticleID=238243 |language= |title=许昌16处文物古迹成“国保” |publisher=许昌网 |date=2013-05-06 |accessdate= }}{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

== 简介 ==
故城分为[[外城|外城]]、[[内城|内城]]，外城遗址目前依稀可见，宛如游龙，外城的范围包括许昌县[[张潘镇|张潘镇]]的营王村、任庄村、古城村、门道张村、吴庄、刘庄、焦庄、董庄等；内城在外城东南方向，呈正方形，内城包括盆李南村、北村等，面积约1.44平方公里，高出地面3米，轮廓较为分明。内城的土城墙有一小部分尚存，大部分已夷为耕地。<ref name= "发生"/>

内城的西南方有一处著名的高台——[[毓秀台|毓秀台]]，是[[汉献帝|汉献帝]]率百官祭祀天地的地方，台上有天爷殿、天王殿、东西厢房、山门等建筑。在故城遗址内很多建筑已不复存在，但毓秀台仍耸立着，为珍贵的遗迹<ref name="发生">{{cite web |url=http://www.21xc.com/Article/ShowArticle.asp?ArticleID=182666 |language= |title=汉魏许都故城：重大历史事件的发生地 |publisher=许昌网 |date=2010-11-18 |accessdate= }}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

1980年，故城遗址被许昌县政府列为县级文物保护单位；1986年又被[[河南省|河南省]]政府列为[[河南省文物保护单位|省级文物保护单位]]。<ref name= "发生"/>

== 参观 ==
景区免费参观，主要看点有汉末皇城遗址，汉献帝祭天的毓秀台，汉砖汉瓦、汉井、汉磨等古文物。自驾车可沿[[许昌市东城区|许昌市东城区]]学院路，行至与S237交叉口左转，再沿S237向东16公里路南即到。<ref>{{cite web |url=http://www.xuchang.gov.cn/sitegroup/root/html/4aef14082d7203a5012d7ba8b2f10e8d/20111223103429626.html |language= |title=汉魏许都故城遗址公园 |publisher=许昌市人民政府 |date=2011-04-21 |accessdate= }}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>
