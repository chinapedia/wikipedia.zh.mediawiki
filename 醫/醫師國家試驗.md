'''医師国家試験 '''(いしこっかしけん)，是日本[[厚生勞動省|厚生勞動省]]醫政局根據該國《醫師法》第9條至第16條規定舉辦的醫師資格國家考試，於每年2月中旬進行。日本[[醫學院|医学院]]入学考试的竞争率高，但是医師国家試験通过率在大多数医学院中在90%左右，相比于[[律師資格考試|律师考试]]和其他国家资格考试要更高。这是因为医師国家試験将"修完医学正规课程并毕业"作为受试的前提条件。也就是说，医学院入学考试合格之后，晋级到最终学年（第6学年），并且毕业考试合格并从医学院毕业，到此为止的所有一贯的课程都为必须，其结果是只有那些掌握了能基本通过医師国家試験知识的人能够参加考试。因此，并不是说考试本身就是容易的。

而且，因为医師国家試験的通过率与大学评估的存在关系，所以近年来，教授医師国家試験予備校的授業或者模擬試験的医学部也不在少数。

== 沿革 ==
* 在[[1946年|1946年]]，基于医師実地修練制度，进行了第一回医師国家試験。
* [[1984年|1984年]]之前，[[春季|春]][[秋季|秋]]季一年两次在[[1985年|1985年]]改为与春季一年一次。
* [[1993年|1993年]]起不再指定出题科目，变为以全科为出题科目的综合问题形式。
* [[2001年|2001年]]起，出题数变为550题（后为530题），9个 block（其中有50题（后为30题）的试行问题），試験日程为3日。
* [[2004年|2004年]]之前是每年的三月举行，从[[2005年|2005年]]开始，伴随着 临床研修义务化考试于每年2月举行。
* [[2007年|2007年]]，取消试行问题，题量变为500题，8个板块。
* [[2008年|2008年]]，就必修的基本事项，医学总论，医学各论的每一个领域，一般性问题与临床实地问题(包括长文形式)在同一个板块内出题，3个领域×3个板块=9个板块的出题形式。
* [[2009年|在2009年]]，在提前预告的新的出题形式(多项选择题・计算问题・不定项选择题)中，采用了多项选择题以及计算问题。另一方面，不会出现不定项选择题，今后也不会采用 <ref>{{PDFlink|[http://www.mhlw.go.jp/stf/shingi/2r9852000001f1cf-att/2r9852000001f1dw.pdf 医師国家試験改善検討部会 報告書]}}([//ja.wikipedia.org/wiki/Portable_Document_Format PDF])<span> </span></ref>.开始出现英语问题。
* [[2010年|2010年]]，104F17的问题取自[[芥川龙之介|芥川龙之介]]的小说"[[齒輪_(小說)|齿轮]]"，第一次出现包含病迹学要素的考题。
* [[2012年|2012年]]，时隔四年厚生劳动省发表修订版新指南 <ref name="ishikokushi2012">[http://www.mhlw.go.jp/topics/2012/05/tp0510-01.html 厚生労働省公式サイト - 平成２５年版医師国家試験出題基準について]</ref>.作为医师国家考试的全体的目的，追加了「确认临床实习中的学习成果」.
* [[2015年|2015年]]，出现了英语试题。

== 受验资格 ==
于医師法第11、12规定。
* 在基于学校教育法[[大學|大学]]， 修了[[医学|医学]]的正规课程([[醫學院|医学院]]医学科・6年制)者。
* [[防衛醫科大學校|防卫医科大学]]毕业生 (防卫省设置法第17条)。
* 医師国家試験预备实验合格者，于合格后经过了1年以上的诊疗及[[公共卫生|公共卫生]]相关的实践培训。
* 外国医学院毕业或已取得外国医师资格者，或[[厚生劳动大臣|厚生劳动大臣]]所认定的具有上述与上述两者同等以上学力及技能并且适当者。
* 根据关于伴随者冲绳复归的厚生省关系法令的適用的特別措置等的政令第17条第1款，冲绳复归之前根据琉球政府的医師法(1955立教号法律第74) 之规定取得医师资格者，并厚生劳动大臣所认定者。
另外，医師法中虽未直接记载，在考试实施当年的三月中旬前预计可以从大学的医学正规课程毕业，并基于厚生劳动省的[[布告|告示]]可以取得受验资格 <ref>[http://www.mhlw.go.jp/kouseiroudoushou/shikaku_shiken/ishi/index.html 医師国家試験の施行について｜厚生労働省]厚生労働省公式サイト。</ref>.

== 考试内容 ==

=== 出题基准 ===
以下仅为根据[[厚生勞動省|厚生劳动省]]所公示的考试内容。
* 关于临床上必要的[[医学|医学]]及[[公共卫生|公共卫生]]问题，作为医生所须具备的知识及技能。
* 试行问题不计分([[2007年|2007年]]起试行问题已废止)。
考试内容仅是根据上述的，不同于[[司法試験|司法考试]]一样限定出题科目，而是同[[美國醫師執照考試|USMLE]]一样不分阶段，[[基础医学|基础医学]]、[[临床医学|临床医学]]、[[社会医学|社会医学]]等所有的医学相关科目皆为出题范围。另外,不分科目，而是以混杂了所有的科目的全科的综合问题来测试。

由从各自的专业领域遴选出来的“医師国家試験委员会”设计出题。4年一次公布的“国家医师考试出题基准”,大致以“基准”中所列举的项目、疾病、症状等作为出题的基本。

=== 考试构成 ===
每回考试由下列内容所构成的合计500题的选择题，分为A~I 板块，于三天的日程内实施。
* 必修基本事项・一般问题
* 必修基本事项・临床实地问题（包含长文形式）
* 医学总论・一般问题
* 医学总论・临床实地问题（包含长文形式）
* 医学各论・一般问题 
* 医学各论・临床实地问题（包含长文形式）
试题册全部板块皆为问题文与别册分开，别册中包含了问题文参照的检查画面、图片、图表等。另外，解答用纸里存在纵式与横式的填写栏。一般问题计1分，临床实地问题计3分，经过删除不切当的问题后，全部满足下述的合格基准后即为合格。此外，关于每回问题与正确答案范例，合格发表后的每年4月左右于厚生劳动省首页中登载。

== 合格基准和合格率 ==

=== 合格基准 ===
以下皆满足者为合格（一般问题、临床实际问题的基准于合格发表时登载）。
* 一般问题（总论＋各论）：相对基准（根据往年65%左右浮动）
* 临床实地问题（总论＋各论）：相对基准（根据往年60%后半浮动）
* 必修问题（一般＋临床实地）：8成（绝对基准）
* 禁忌肢の选择数：2问以下（绝对基准）
必修问题中评分除外等进行调整时，评分对象的问题须有8成以上的得分方为合格（从2006年开始，只有答错被列为得分除外题的情况下，分数才有所更改。因此考生根据不同必修问题的满分也会不同)。另外，更变为禁忌肢的选择数在3道题以下。

=== 合格率 ===
参看医師国家試験的合格率，从1-100回的平均为84.2% <ref>{{PDFlink|[http://www.wam.go.jp/wamappl/bb10examinfo.nsf/0/e049ac1105b8c5e849257140001f46de/$FILE/100-isi.pdf 第100回医師国家試験の合格発表について]}}([//ja.wikipedia.org/wiki/Portable_Document_Format PDF])<span> </span>（独立行政法人福祉医療機構）</ref>，在最近几年，在80%上下浮动。

== 試験地点 ==
北海道、宮城県、東京都、新潟県、愛知県、石川県、大阪府、広島県、香川県、福岡県、熊本県、沖縄県之12都道府県举行。东京都因为历年3成以上的受验者集中于此，因为设置两个受验会场。

== 关联项目 ==
* 医学教育
* 医術開業試験
* [[美國醫師執照考試|USMLE]] - [[美国|アメリカ合衆国]]の医師国家試験。出題される問題は基礎、臨床、総合の3段階に分かれている。2004年からは基本的な臨床技能に関する実技試験も行われる。
