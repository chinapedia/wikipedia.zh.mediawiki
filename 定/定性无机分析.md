'''定性无机分析'''是[[分析化学|分析化学]]中的一种用来寻找无机化合物中元素组成的分析方法。它主要来鉴定水溶液中的[[离子|离子]]，因此，一些其他形式的物质在用标准方法分析之前，先要转化为水溶液中的离子形式。之后，溶液会加入多种试剂，通过化学反应来检测针对特定离子的可能出现的颜色变化、沉淀或其他可观现象。<ref name=QAES>E. J. King "Qualitative Analysis and Electrolytic Solutions" 1959, Harcourt, Brace, and World, New York.</ref>

==阳离子H<sub>2</sub>S系统分析法==
本分析法利用24种常见阳离子的硫化物溶解度差异，将它们分为4组进行分析。<ref name=ac />
===第一组阳离子的分析===
本组的阳离子有Ag<sup>+</sup>、Hg<sub>2</sub><sup>2+</sup>和Pb<sup>2+</sup>三种，组试剂是[[盐酸|盐酸]]。这三种阳离子的氯化物难溶于水，借此和其他阳离子分离：
: Ag<sup>+</sup> + Cl<sup>−</sup> → AgCl↓
: Hg<sub>2</sub><sup>2+</sup> + 2 Cl<sup>-</sup> → Hg<sub>2</sub>Cl<sub>2</sub>↓
: Pb<sup>2+</sup> + 2 Cl<sup>−</sup> → PbCl<sub>2</sub>↓
注意事项：
*沉淀本组离子时，Cl<sup>-</sup>稍过量，并控制其浓度为0.5mol/L。如果用浓盐酸，则[[氯化银|AgCl]]和[[氯化铅|PbCl<sub>2</sub>]]会形成配合物[AgCl<sub>2</sub>]<sup>-</sup>和[PbCl<sub>4</sub>]<sup>2-</sup>而溶解。
*第二组阳离子中，Sb<sup>3+</sup>和Bi<sup>3+</sup>的水解倾向较强，当溶液pH较高时，会生成白色的碱式盐沉淀而混入第一组阳离子中。因此，应补充适量[[硝酸|HNO<sub>3</sub>]]，使H<sup>+</sup>的浓度在2.0~2.4mol/L之间。
*由于PbCl<sub>2</sub>溶解度不是特别小，所以当Pb<sup>2+</sup>浓度较低时，可能不会检出。这时，Pb<sup>2+</sup>应在第二组阳离子的分析中检验。
本组离子的系统分析：
*Pb<sup>2+</sup>的鉴定：
:PbCl<sub>2</sub>和其他两种氯化物不同的是，它可溶于热水。因此，将所沉淀的氯化物过滤后加水，并加热，使PbCl<sub>2</sub>溶解，趁热分离。先用[[乙酸|乙酸]]酸化离心液，再加入K<sub>2</sub>CrO<sub>4</sub>或KI，如果出现黄色沉淀，则说明存在Pb<sup>2+</sup>。
::Pb<sup>2+</sup> + CrO<sub>4</sub><sup>2-</sup> → PbCrO<sub>4</sub>↓（黄色沉淀）
:: Pb<sup>2+</sup> + 2I<sup>−</sup> → PbI<sub>2</sub>↓（黄色沉淀）
*Ag<sup>+</sup>的鉴定
:分出PbCl<sub>2</sub>之后，沉淀用热水洗涤，之后加入[[氨水|氨水]]。离心分离。离心液加入HNO<sub>3</sub>酸化，如果出现白色沉淀，则说明有Ag<sup>+</sup>的存在。
:: AgCl + 2 NH<sub>3</sub> → [Ag(NH<sub>3</sub>)<sub>2</sub>]Cl（无色溶液）
::: [Ag(NH<sub>3</sub>)<sub>2</sub>]Cl + 2 HNO<sub>3</sub> → 2 NH<sub>4</sub>NO<sub>3</sub> + AgCl↓（白色沉淀）
*Hg<sub>2</sub><sup>2+</sup>的鉴定
:上一步鉴定Ag<sup>+</sup>加入氨水时，如果沉淀变为灰黑色，则说明有Hg<sub>2</sub><sup>2+</sup>。
::Hg<sub>2</sub>Cl<sub>2</sub> + 2 NH<sub>3</sub> → HgNH<sub>2</sub>Cl↓（白） + Hg↓（黑） + NH<sub>4</sub>Cl

===第二组阳离子的分析===
本组离子包括Pb<sup>2+</sup>、Bi<sup>3+</sup>、Cu<sup>2+</sup>、Cd<sup>2+</sup>、Hg<sup>2+</sup>、As<sup>3+</sup>、As<sup>5+</sup>、Sb<sup>3+</sup>、Sb<sup>5+</sup>、Sn<sup>2+</sup>和Sn<sup>4+</sup>组成，其中As、Sb和Sn可能以含氧酸根的形式出现。这组阳离子不被Cl<sup>-</sup>沉淀，但在0.3mol/L [[盐酸|HCl]]溶液中，可以被H<sub>2</sub>S沉淀，借此和第三、四组阳离子分离。

注意事项：
*As有+3和+5两种价态，其中As(V)与H<sub>2</sub>S反应速率慢，应先用[[碘化铵|碘化铵]]将其还原为As(III)。
*Sn有+2和+4两种价态，为使操作简便，需要用[[过氧化氢|过氧化氢]]溶液将Sn(II)氧化为Sn(IV)。
*Sb有+3和+5两种价态，检验Sb的存在时，所用的[[罗丹明B|罗丹明B]]只对+5价的Sb有效，因此溶液中的Sb(III)也需要事先氧化。
*在沉淀本组阳离子时，酸度应控制在0.3mol/LHCl，如果pH过低，CdS、SnS和PbS将沉淀不完全或不沉淀而混入第三组阳离子之中，又不利于As、Sb和Sn的硫化物的生成，如果pH较高，第三组中Zn2+将会以ZnS的形式沉淀而混入本组。
*加热溶液有利于As(V)被NH<sub>4</sub>I还原，但降低了H<sub>2</sub>S的溶解度，故需冷却至室温。

基于以上问题考虑，[[硫乙醯胺|硫乙酰胺]]（TAA）是比H<sub>2</sub>S更理想的沉淀剂，理由如下：
*酸性、氨性和碱性环境下，TAA水解产生H<sub>2</sub>S、HS<sup>−</sup>和S<sup>2−</sup>，分别可以替代H<sub>2</sub>S、(NH<sub>4</sub>)<sub>2</sub>S和Na<sub>2</sub>S使用；
*TAA会产生均相沉淀，得到良好的晶形，有利于沉淀的后期处理；
*TAA在90℃的酸性溶液中，可以将+5价的As和Sb还原为+3价的产物，省去了NH<sub>4</sub>I；
*TAA在碱性环境中加热，会生成少量的多硫化物，可将Sn(II)氧化为Sn(IV)，省去了H<sub>2</sub>O<sub>2</sub>。

但需要注意，TAA提供的S<sup>2-</sup>较少，在分离第二、三组阳离子时，需调节酸度至0.6mol/LH<sup>+</sup>，使As沉淀完全，之后溶液稀释一倍，再进行操作。

本组离子在以上步骤沉淀完之后，离心并洗涤，用TAA的碱性溶液（需加热）或Na<sub>2</sub>S处理沉淀，Hg、As、Sb和Sn的硫化物溶解，而剩下Cu、Cd、Pb和Bi的硫化物：
: HgS + S<sup>2−</sup> → [HgS<sub>2</sub>]<sup>2-</sup>
: As<sub>2</sub>S<sub>3</sub> + 3 S<sup>2−</sup> → 2 [AsS<sub>3</sub>]<sup>3-</sup>
: Sb<sub>2</sub>S<sub>3</sub> + 3 S<sup>2−</sup> → 2 [SbS<sub>3</sub>]<sup>3-</sup>
: SnS<sub>2</sub> + S<sup>2−</sup> → [SnS<sub>3</sub>]<sup>2-</sup>

将沉淀用含有氯化铵的水洗涤，加入6mol/L[[硝酸|HNO<sub>3</sub>]]溶解。
*Cd<sup>2+</sup>的鉴定：
:在以上硝酸溶解液中加入1:1[[丙三醇|丙三醇]]（甘油）和过量的浓氢氧化钠溶液。Cu、Pb和Bi形成配合物溶解，留下Cd(OH)<sub>2</sub>。用盐酸溶解[[氢氧化镉|Cd(OH)<sub>2</sub>]]，并稀释至H<sup>+</sup>约为0.3mol/L，加入TAA并加热，若产生黄色沉淀，则表明存在Cd<sup>2+</sup>：
::Cd<sup>2+</sup> + S<sup>2-</sup> = CdS↓（黄色沉淀）
*Cu<sup>2+</sup>、Pb<sup>2+</sup>和Bi<sup>3+</sup>的鉴定：
:将甘油配合物离心液分成三份，乙酸酸化后加入[[亚铁氰化钾|K<sub>4</sub>[Fe(CN)<sub>6</sub>]]]，如出现红棕色沉淀，表明有Cu<sup>2+</sup>；6mol/L乙酸酸化后加入K<sub>2</sub>CrO<sub>4</sub>，如出现黄色沉淀，则说明有Pb<sup>2+</sup>；将离心液加入至新制Na<sub>2</sub>SnO<sub>2</sub>溶液，如有黑色沉淀，则说明有Bi<sup>3+</sup>。
: 2Cu<sup>2+</sup> + [Fe(CN)<sub>6</sub>]<sup>4-</sup> → Cu<sub>2</sub>[Fe(CN)<sub>6</sub>]↓（红棕色沉淀）
: Pb<sup>2+</sup> + CrO<sub>4</sub><sup>2−</sup>→ PbCrO<sub>4</sub>↓（黄色沉淀）
: 2Bi<sup>3+</sup> + 3 SnO<sub>2</sub><sup>2-</sup> + 6OH<sup>-</sup> → 2Bi↓（黑色沉淀） + 3 SnO<sub>3</sub><sup>2-</sup> + 3H<sub>2</sub>O

将之前得到的Hg、As、Sb和Sn的硫代酸盐用3mol/L[[盐酸|盐酸]]处理，硫代酸盐分解，析出硫化物。
*Sb、Sn的鉴定：
:将用3mol/L盐酸处理得到的沉淀洗涤后加入8mol/L盐酸，Sb和Sn溶解，存在形式为[SbCl<sub>6</sub>]<sup>3−</sup>和[SnCl<sub>6</sub>]<sup>2−</sup>。
:将溶液分为两份，一份检验Sn：
:将还原铁粉加入溶液，还原Sn<sup>4+</sup>为Sn<sup>2+</sup>，之后加入[[氯化汞|HgCl<sub>2</sub>]]溶液，如果有灰色或黑色沉淀，则表明Sn存在：
:: [SnCl<sub>4</sub>]<sup>2−</sup> + 2 HgCl<sub>2</sub> → Hg<sub>2</sub>Cl<sub>2</sub>↓（白色） + [SnCl<sub>6</sub>]<sup>2−</sup>
:: [SnCl<sub>4</sub>]<sup>2−</sup> + Hg<sub>2</sub>Cl<sub>2</sub> → 2 Hg（黑色） + [SnCl<sub>6</sub>]<sup>2−</sup>
:另一份检验Sb:
:在溶液中加入[[亚硝酸钠|亚硝酸钠]]固体，将Sb<sup>3+</sup>氧化为Sb<sup>5+</sup>，然后加入红色的[[罗丹明B|罗丹明B]]，如有蓝色或紫色沉淀产生，则表明Sb存在。该沉淀可以被苯萃取，得到紫红色的苯溶液。
:Sn(IV)的存在不影响此鉴定。
*As的鉴定：
:将鉴定Sb和Sn过程中用8mol/L盐酸未溶解的固体洗涤，加入12%(NH<sub>4</sub>)<sub>2</sub>CO<sub>3</sub>加热，使As<sub>2</sub>S<sub>3</sub>溶解：
::As<sub>2</sub>S<sub>3</sub> + 3CO<sub>3</sub><sup>2-</sup> → AsS<sub>3</sub><sup>3-</sup> + AsO<sub>3</sub><sup>3-</sup> + 3CO<sub>2</sub>↑
:离心分离，溶液中含As，剩余沉淀含Hg。
:取出溶液，加入3mol/LHCl，如果As存在，将产生黄色的[[三硫化二砷|As<sub>2</sub>S<sub>3</sub>]]沉淀。
*Hg<sup>2+</sup>的鉴定：
:将上一步所得沉淀洗涤之后用[[王水|王水]]处理，将其溶解。溶解之后通过煮沸去除王水。
:: 3HgS + 2NO<sub>3</sub><sup>−</sup> + 12Cl<sup>−</sup> + 8H<sup>+</sup> → 3HgCl<sub>4</sub><sup>2−</sup> + 3S + 2NO + 4H<sub>2</sub>O
:之后加入[[氯化亚锡|SnCl<sub>2</sub>]]溶液，如果产生由白至黑的沉淀，则说明存在Hg<sup>2+</sup>。

===第三组阳离子的分析===
本组离子包括Al<sup>3+</sup>、Cr<sup>3+</sup>、Fe<sup>3+</sup>、Fe<sup>2+</sup>、Mn<sup>2+</sup>、Zn<sup>2+</sup>、Co<sup>2+</sup>和Ni<sup>2+</sup>共8种（实为7种元素），它们在鉴定时不需组内特别分离。本组的组试剂为[[氨|NH<sub>3</sub>]]-[[氯化铵|NH<sub>4</sub>Cl]]存在下的(NH<sub>4</sub>)<sub>2</sub>S（或TAA，加热），铁、锰、锌、钴、镍均以硫化物沉淀，铝、铬以氢氧化物的形式沉淀。借此与第四组阳离子分离。

注意事项：
*本组需要用[[氨|NH<sub>3</sub>]]-[[氯化铵|NH<sub>4</sub>Cl]][[缓冲溶液|缓冲溶液]]调节溶液pH在9.0左右，确保Al<sup>3+</sup>和Cr<sup>3+</sup>沉淀完全，防止第四组的Mg<sup>2+</sup>产生沉淀。
*为防止硫化物形成胶体，需要加入NH<sub>4</sub>Cl并加热。
*产生的沉淀不宜久置，否则[[硫化钴|CoS]]和[[硫化镍|NiS]]会因为构型改变而难溶。

上一组分析留下酸性溶液，溶液用氨水中和至碱性，并加入NH<sub>4</sub>Cl，再加入(NH<sub>4</sub>)<sub>2</sub>S或TAA并加热（约10min）。将所得沉淀用热的稀[[硝酸|硝酸]]溶解，之后逐一鉴定组内离子。
*Fe<sup>3+</sup>的鉴定
: 溶液中的Fe<sup>2+</sup>全部被氧化成Fe<sup>3+</sup>，如需确定铁的价态，取原液鉴定。Fe<sup>2+</sup>可与[[铁氰化钾|铁氰化钾]]形成蓝色的KFe[Fe(CN)<sub>6</sub>]沉淀，或者与[[邻二氮菲|邻二氮菲]]在弱酸性溶液中形成橙红色可溶络合物。其他阳离子均不干扰这两个反应。
: 针对Fe3+，可以用NH4SCN或[[亚铁氰化钾|K4[Fe(CN)6]]]鉴别，前者产生可溶性血红色络合物，后者产生[[普鲁士蓝|普鲁士蓝]]沉淀。其他阳离子不干扰。[[File:Eisen(III)-Ionen_und_Thiocyanat.JPG|right]]
: K<sup>+</sup> + Fe<sup>3+</sup> + [Fe(CN)<sub>6</sub>]<sup>4-</sup> = KFe[Fe(CN)<sub>6</sub>]↓

*Mn<sup>2+</sup>的鉴定：
:在强酸性条件下，Mn<sup>2+</sup>可以被[[铋酸钠|NaBiO<sub>3</sub>]]、[[二氧化铅|PbO<sub>2</sub>]]或[[过硫酸铵|(NH<sub>4</sub>)<sub>2</sub>S<sub>2</sub>O<sub>8</sub>]]（Ag<sup>+</sup>离子催化）氧化为MnO<sub>4</sub><sup>−</sup>，产生淡紫红色的溶液。
: 2 Mn<sup>2+</sup> + 5 NaBiO<sub>3</sub> + H<sup>+</sup> = 2 MnO<sub>4</sub><sup>−</sup> + 5 Bi<sup>3+</sup> + 5 Na<sup>+</sup> + 7 H<sub>2</sub>O

* Cr<sup>3+</sup>的鉴定：
:在溶液中加入稍过量的NaOH使Cr<sup>3+</sup>转化为[Cr(OH)<sub>4</sub>]<sup>−</sup>。之后，用[[过氧化氢|H<sub>2</sub>O<sub>2</sub>]]将其氧化为CrO<sub>4</sub><sup>2-</sup>。用H<sub>2</sub>SO<sub>4</sub>酸化后加入[[乙醚|乙醚]]（或[[正戊醇|正戊醇]]），再加入H<sub>2</sub>O<sub>2</sub>，如果有机层有蓝色的[[过氧化铬|过氧化铬]]（CrO<sub>5</sub>）产生，则说明Cr<sup>3+</sup>存在。
* Ni<sup>2+</sup>的鉴定：
:在溶液中加入H<sub>2</sub>O<sub>2</sub>，将二价亚铁氧化为三价铁，排除亚铁的干扰。加入氨水将溶液调为碱性，并用[[酒石酸|酒石酸]]或[[柠檬酸|柠檬酸]]将Fe<sup>3+</sup>和Mn<sup>2+</sup>等阳离子掩蔽，之后加入丁二酮肟，如产生桃红色沉淀，则说明存在Ni<sup>2+</sup>。
:注：镍和丁二酮肟的络合物在酸性条件下分解。
* Co<sup>2+</sup>的鉴定：
: 在溶液中加入NH<sub>4</sub>F将Fe<sup>3+</sup>掩蔽，再加入丙酮和饱和NH<sub>4</sub>SCN溶液。如溶液出现蓝色，则说明有Co<sup>2+</sup>。
* Zn<sup>2+</sup>的鉴定：
: 在溶液中加入NH<sub>4</sub>F将Fe<sup>3+</sup>掩蔽，再加入[[四硫氰合汞化铵|(NH<sub>4</sub>)<sub>2</sub>[Hg(SCN)<sub>4</sub>]]]溶液，如果产生白色晶体沉淀，则说明有Zn<sup>2+</sup>。如果溶液中有微量Co<sup>2+</sup>，将迅速产生天蓝色的沉淀（混晶）。（如果没有，滴加几滴Co<sup>2+</sup>）
:: Zn<sup>2+</sup> + [Hg(SCN)<sub>4</sub>]<sup>2-</sup> = Zn[Hg(SCN)<sub>4</sub>]
:: Co<sup>2+</sup> + [Hg(SCN)<sub>4</sub>]<sup>2-</sup> = Co[Hg(SCN)<sub>4</sub>]
: 注：如果沉淀产生缓慢（超过2min），则不能说明存在Zn<sup>2+</sup>。
* Al<sup>3+</sup>的鉴定：
:在溶液中加入HAc-NaAc缓冲溶液，再加入[[铝试剂|铝试剂]]，如出现鲜红色沉淀，则存在Al<sup>3+</sup>。
:如果存在干扰离子，试液用Na<sub>2</sub>CO<sub>3</sub>-Na<sub>2</sub>O<sub>2</sub>处理。

===第四组阳离子的分析===
本组离子包括Ca<sup>2+</sup>、Mg<sup>2+</sup>、Ba<sup>2+</sup>、Sr<sup>2+</sup>、K<sup>+</sup>、Na<sup>+</sup>和NH<sub>4</sub><sup>+</sup>，共7种离子。本组无组试剂。

注意事项：
* 第三组阳离子沉淀后的离心液不可久置，以防溶液中的S<sup>2−</sup>被氧化为SO<sub>4</sub><sup>2−</sup>，导致Sr<sup>2+</sup>和Ba<sup>2+</sup>沉淀而丢失。
* NH<sub>4</sub><sup>+</sup>应取未加入任何组试剂的原试液检验。

将离心液蒸干并灼烧，除去铵盐。冷却之后加入适量稀盐酸溶解。
* NH<sub>4</sub><sup>+</sup>的鉴定：
:将原试液取样，加入浓NaOH并加热，使NH<sub>4</sub><sup>+</sup>转化为NH<sub>3</sub>检验。NH<sub>3</sub>遇到湿润的红色石蕊试纸，试纸变蓝。
:此外，NH<sub>4</sub><sup>+</sup>还能使碱性的[[碘化汞钾|碘化汞钾]]产生红棕色沉淀。
*Ba<sup>2+</sup>和Sr<sup>2+</sup>的鉴定：
:取以上用稀盐酸溶解的试液，加入过量NaAc溶液，使pH在4～5之间。加入K<sub>2</sub>CrO<sub>4</sub>，如有黄色沉淀，则说明有Ba<sup>2+</sup>。滤液调节pH，使酸性减弱，如有黄色沉淀，再加入HAc，如黄色沉淀溶解，则说明有Sr<sup>2+</sup>。
:注：BaCrO<sub>4</sub>不溶于乙酸，而SrCrO<sub>4</sub>溶于乙酸。它们都溶于强酸且不溶于氢氧化钠和氨水。另外，在溶液中加入玫瑰红酸钠，Ba<sup>2+</sup>与之形成鲜红色沉淀。
*Ca<sup>2+</sup>的鉴定：
:在试液中加入饱和(NH<sub>4</sub>)<sub>2</sub>SO<sub>4</sub>，离心分离，取溶液。
:: CaSO<sub>4</sub>(S) + (NH<sub>4</sub>)<sub>2</sub>SO<sub>4</sub>(aq) = (NH<sub>4</sub>)<sub>2</sub>[Ca(SO<sub>4</sub>)<sub>2</sub>](aq)
:离心液中加入(NH<sub>4</sub>)<sub>2</sub>C<sub>2</sub>O<sub>4</sub>，如有白色沉淀，则表示Ca<sup>2+</sup>存在。
*Mg<sup>2+</sup>的鉴定：
:将溶液调为碱性，加入镁试剂I，如果产生天蓝色沉淀，则示有Mg<sup>2+</sup>。
*K<sup>+</sup>的鉴定：
: 将溶液调为中性后加入HAc酸化，再加入Na<sub>3</sub>Co(NO<sub>2</sub>)<sub>6</sub>，如出现黄色沉淀，则表明存在K<sup>+</sup>。
:: 2 K<sup>+</sup> + Na<sup>+</sup> + Co(NO<sub>2</sub>)<sub>6</sub><sup>3−</sup> = K<sub>2</sub>Na[Co(NO<sub>2</sub>)<sub>6</sub>]↓
: 或者将溶液调为中性后加入[[四苯硼钠|四苯硼钠]]，如出现白色沉淀，则说明存在K<sup>+</sup>。
:: K<sup>+</sup> + [B(C<sub>6</sub>H<sub>5</sub>)<sub>4</sub>]<sup>−</sup> = K[B(C<sub>6</sub>H<sub>5</sub>)<sub>4</sub>]↓
*Na<sup>+</sup>的鉴定：
: 在溶液中加入饱和Ba(OH)<sub>2</sub>至碱性，加入过量(NH<sub>4</sub>)<sub>2</sub>CO<sub>3</sub>除去Ba<sup>2+</sup>，再将溶液蒸干灼烧至恒重。取少量水溶解，加入少许醋酸和几滴乙醇。加入过量[[乙酸铀酰锌|乙酸铀酰锌]]或[[乙酸铀酰镁|乙酸铀酰镁]]，如有柠檬黄色结晶，则示有Na<sup>+</sup>存在。
:: Na<sup>+</sup> + Zn<sup>2+</sup> + 3 UO<sub>2</sub><sup>2+</sup> + 9 CH<sub>3</sub>COO<sup>−</sup> + 9 H<sub>2</sub>O = NaZn(UO<sub>2</sub>)<sub>3</sub>(CH<sub>3</sub>COO)<sub>9</sub>·9H<sub>2</sub>O

此外，碱金属和碱土金属（镁除外）有时候还可以通过特征的[[焰色反应|焰色反应]]来鉴别。

==两酸两碱系统分析法==
本法先检出NH<sub>4</sub><sup>+</sup>、Na<sup>+</sup>、Fe<sup>2+</sup>和Fe<sup>3+</sup>。再进行以下操作：<ref name=ac />
*加入HCl沉淀出Ag<sup>+</sup>、Hg<sub>2</sub><sup>2+</sup>、Pb<sup>2+</sup>，分别检验；
*取上一步的溶液加入H<sub>2</sub>SO<sub>4</sub>和[[乙醇|乙醇]]，沉淀出Pb<sup>2+</sup>、Ba<sup>2+</sup>和Ca<sup>2+</sup>，分别检验；
*取上一步的溶液加入NH<sub>3</sub>-NH<sub>4</sub>Cl和H<sub>2</sub>O<sub>2</sub>，沉淀出Hg<sup>2+</sup>、Al<sup>3+</sup>、Bi<sup>3+</sup>、Cr<sup>3+</sup>、Sn<sup>4+</sup>、Fe<sup>3+</sup>、Sb<sup>3+</sup>、Sb<sup>5+</sup>和Mn<sup>2+</sup>，分别检验；
*取上一步的溶液加入过量NaOH，沉淀出Cu<sup>2+</sup>、Ni<sup>2+</sup>、Cd<sup>2+</sup>、Mg<sup>2+</sup>和Co<sup>2+</sup>，再分别检验；
*剩余As(III)、As(V)、Na<sup>+</sup>、Zn<sup>2+</sup>、NH<sub>4</sub><sup>+</sup>和K<sup>+</sup>。

==阴离子分析 ==
常见的阴离子按照其性质可分为三组。
===第一组阴离子的分析===
第一组阴离子包含CO<sub>3</sub><sup>2-</sup>、HCO<sub>3</sub><sup>-</sup>、CH<sub>3</sub>COO<sup>-</sup>、S<sup>2-</sup>、SO<sub>3</sub><sup>2-</sup>、S<sub>2</sub>O<sub>3</sub><sup>2-</sup>和NO<sub>2</sub><sup>-</sup>，组试剂是稀[[硫酸|硫酸]]。
*CO<sub>3</sub><sup>2-</sup>和HCO<sub>3</sub><sup>-</sup>遇到稀硫酸放出CO<sub>2</sub>，通入澄清的石灰水，石灰水先变浑浊，后澄清。
*CH<sub>3</sub>COO<sup>-</sup>和稀硫酸作用，生成CH<sub>3</sub>COOH，用黄色的FeCl<sub>3</sub>检验，如果变血红色，则示有CH<sub>3</sub>COO<sup>−</sup>。
**除此之外，在含CH<sub>3</sub>COO<sup>-</sup>中加入浓硫酸和戊醇，加热反应，生成[[乙酸戊酯|乙酸戊酯]]，有特殊的水果香味。另一种方法是利用[[硝酸镧|La(NO<sub>3</sub>)<sub>3</sub>]]和[[碘|I<sub>2</sub>]]在氨性溶液中生成暗蓝色沉淀来鉴别。<ref name=ac>《分析化学》（上册）. 高等教育出版社. 华中师范大学 等编. P<sub>33~61</sub>. ISBN 978-7-04-031145-7</ref>
*S<sup>2-</sup>遇到稀硫酸放出[[硫化氢|H<sub>2</sub>S]]，使[[醋酸铅|醋酸铅]]试纸变黑，或使[[硝普钠|硝普钠]]溶液变紫。
*SO<sub>3</sub><sup>2-</sup>和S<sub>2</sub>O<sub>3</sub><sup>2-</sup>被酸化时均会放出SO<sub>2</sub>，可以使酸化的KMnO<sub>4</sub>褪色，或使酸化的K<sub>2</sub>Cr<sub>2</sub>O<sub>7</sub>由橙变绿，也可让碱性品红褪色。但不同的是S<sub>2</sub>O<sub>3</sub><sup>2-</sup>会产生乳黄色的[[硫|硫]]单质沉淀。
*NO<sub>2</sub><sup>-</sup>遇到稀硫酸，在冰水中得到[[亚硝酸|浅蓝色溶液]]，如果是室温，则放出红棕色有刺激性臭味的气体（[[二氧化氮|NO<sub>2</sub>]]），另外，这种气体能使淀粉-碘化钾试纸变蓝。
**另外，NO<sub>2</sub><sup>-</sup>的一个特效反应是，将HAc加入至含有NO<sub>2</sub><sup>-</sup>的溶液中，再加入[[对氨基苯磺酸|对氨基苯磺酸]]，发生重氮化反应，与[[1-甲萘胺|α-萘胺]]生成红色的偶氮化合物。<ref name=ac />

===第二组阴离子的分析===
第二组阴离子包含Cl<sup>−</sup>、Br<sup>−</sup>、I<sup>−</sup>、NO<sub>3</sub><sup>−</sup>和C<sub>2</sub>O<sub>4</sub><sup>2-</sup>，组试剂是浓[[硫酸|硫酸]]。
*Cl<sup>-</sup>可以通过[[铬酰氯|铬酰氯]]试验鉴定。当Cl<sup>−</sup>和K<sub>2</sub>Cr<sub>2</sub>O<sub>7</sub>与浓H<sub>2</sub>SO<sub>4</sub>加热时，会产生CrO<sub>2</sub>Cl<sub>2</sub>蒸气，蒸气通入NaOH，产生Na<sub>2</sub>CrO<sub>4</sub>，CrO<sub>4</sub><sup>2−</sup>与Pb(CH<sub>3</sub>COO)<sub>2</sub>作用，产生黄色沉淀。
*Br<sup>−</sup>和I<sup>−</sup>可以通过其单质形态在CHCl<sub>3</sub>、CCl<sub>4</sub>、[[苯|C<sub>6</sub>H<sub>6</sub>]]或CS<sub>2</sub>中的显色得到。含Br<sub>2</sub>的显深红色，含I<sub>2</sub>的显紫色。
**如果用氯氧化，则需注意过量的氯会将I<sup>-</sup>氧化为IO<sub>3</sub><sup>-</sup>而褪色。对于溴，可能出现Br<sub>2</sub>的深红色，也可能出现BrCl的黄色。<ref name=ac />
*NO<sub>3</sub><sup>−</sup>可以通过[[棕色环试验|棕色环试验]]或用[[硫酸|硫酸]]-[[二苯胺|二苯胺]]来鉴别。前者产生棕色环，后者产生深蓝色环。
**如果溶液中同时含有NO<sub>2</sub><sup>-</sup>，需要事先除去。方法是在酸性环境中，加入尿素，加热使其分解。
**另外，NO<sub>3</sub><sup>-</sup>也可以在HAc环境中，用[[锌|锌]]使其还原为NO<sub>2</sub><sup>-</sup>，再检验NO<sub>2</sub><sup>-</sup>。
*C<sub>2</sub>O<sub>4</sub><sup>2-</sup>在被浓硫酸处理时，会放出CO<sub>2</sub>和CO，其中CO<sub>2</sub>可以使澄清的石灰水变浑浊。C<sub>2</sub>O<sub>4</sub><sup>2-</sup>本身可以使酸化的KMnO<sub>4</sub>褪色，并与CaCl<sub>2</sub>反应产生沉淀。

===第三组阴离子的分析===
第三组阴离子包含SO<sub>4</sub><sup>2−</sup>、PO<sub>4</sub><sup>3−</sup>和BO<sub>3</sub><sup>3−</sup>，它们在硫酸中几乎无作用。
*SO<sub>4</sub><sup>2−</sup>可用BaCl<sub>2</sub>鉴定，反应产生BaSO<sub>4</sub>不溶于任何酸碱。
*PO<sub>4</sub><sup>3−</sup>则在HNO<sub>3</sub>下与[[钼酸铵|钼酸铵]]反应，产生黄色沉淀。此沉淀溶于氨水或碱；在酸性环境下，可以氧化联苯胺，得到蓝色产物。[[File:Phosphatnachweis.jpg|right]]
**注：溶液中的AsO<sub>4</sub><sup>3-</sup>和SiO<sub>3</sub><sup>2-</sup>需要加酒石酸来消除干扰。玻璃器皿可能产生微量的SiO<sub>3</sub><sup>2-</sup>。
*BO<sub>3</sub><sup>3−</sup>和无水[[乙醇|乙醇]]或[[甲醇|甲醇]]，在浓[[硫酸|硫酸]]的催化下，产生硼酸酯，加热点燃有绿色火焰<ref>《无机化学》（第二版）下册. 庞锡涛 主编. 高等教育出版社. 4-3 硼的含氧化合物. P<sub>182</sub></ref>。

==不常见元素的定性分析==
*锗
:向含有锗的溶液中，加入含0.2mol/L左右的HNO<sub>3</sub>的5%钼酸铵溶液，生成柠檬黄色溶液（H<sub>4</sub>(GeMo<sub>12</sub>O<sub>40</sub>)）。<ref name=w3>《无机化学丛书》.第三卷 碳 硅 锗分族. 科学出版社. P<sub>267</sub>. 3.定性分析</ref>
:或者取极少量的[[苏木因|苏木因]]溶于5ml3mol/LH<sub>2</sub>SO<sub>4</sub>，再慢慢滴入95ml乙醇。向酸性含锗的溶液中滴加6滴苏木因的醇溶液和15滴四氯化碳，震荡15至30秒，如出现紫色配合物且溶于四氯化碳，则检出Ge。
::注：Mo和Sb也能形成紫色配合物，但不溶于四氯化碳。
*硒
:向试样中加入[[氢氧化钠|氢氧化钠]]和[[盐酸羟胺|盐酸羟胺]]除去重金属和贵金属元素，再向试样的盐酸溶液中通入[[二氧化硫|二氧化硫]]，如果出现红色的沉淀，则说明含有硒。如果同时含有碲，可用[[盐酸羟胺|盐酸羟胺]]选择性还原：<ref name=w5>《无机化学丛书》. 第五卷 氧 硫 硒分族. 科学出版社. P<sub>257</sub>. 1.5.2 定性检出</ref>
:2 NH<sub>2</sub>OH + SeO<sub>3</sub><sup>2-</sup> + 2 H<sup>+</sup> → N<sub>2</sub>O + Se + 4 H<sub>2</sub>O
*碲
:向试样中加入[[氢氧化钠|氢氧化钠]]和[[盐酸羟胺|盐酸羟胺]]除去重金属和贵金属元素，再向试样的热的盐酸溶液中通入[[二氧化硫|二氧化硫]]，如果出现黑色沉淀，则说明含有碲。用[[氯化亚锡|氯化亚锡]]也能将Te(IV)还原。所产生的黑色沉淀溶于少量浓硫酸，产生Te<sub>4</sub><sup>2+</sup>多聚阳离子，为鲜红色的溶液，稀释后析出黑色沉淀。<ref name=w5 />
:: TeO<sub>3</sub><sup>2−</sup> + 2 Sn<sup>2+</sup> + 6 H<sup>+</sup> → Te↓ + 2 Sn<sup>4+</sup> + 3 H<sub>2</sub>O
*钍
:Th<sup>4+</sup>可以在pH=2的条件下被钍试剂I检出，或在2~9mol/LHCl条件下被[[偶氮胂III|偶氮胂III]]（铀试剂）检出。
*铀
:UO<sub>2</sub><sup>2+</sup>可以和[[铁氰酸钾|铁氰酸钾]]作用，产生红棕色沉淀，该方法能检出1ppm的铀。Fe<sup>3+</sup>、Cu<sup>2+</sup>对此反应有干扰，需预先除去。<ref name=w10>《无机化学丛书》.第十卷.锕系 锕系后元素. 科学出版社. P<sub>147</sub>. 7.6.5 铀的主要分析方法</ref>
::UO<sub>2</sub><sup>2+</sup> + 2 K<sup>+</sup> + [Fe(CN)<sub>6</sub>]<sup>4-</sup> = K<sub>2</sub>UO<sub>2</sub>[Fe(CN)<sub>6</sub>]↓（红棕色沉淀）
:另外，在pH=0.5~3.5介质中，UO<sub>2</sub><sup>2+</sup>能和H<sub>2</sub>O<sub>2</sub>反应，生成浅黄色沉淀（如UO<sub>2</sub>(O<sub>2</sub>)·4H<sub>2</sub>O等）。仅Th和Zr、Hf有类似反应。
