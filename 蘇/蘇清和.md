'''蘇清和'''<ref>參見-{[[環球音樂|環球音樂]]}- · 校園快樂幫No.1 校園歌喉戰冠軍合輯簡介{{cite web |url=http://www.umusic.com.tw/Microsite/prospects/index.html |title=存档副本 |accessdate=2009-03-04 |deadurl=yes |archiveurl=https://web.archive.org/web/20081104072958/http://www.umusic.com.tw/Microsite/prospects/index.html |archivedate=2008-11-04 }}</ref>（{{bd|1985年|2月24日}}），[[臺灣|臺灣]][[屏東縣|屏東縣]]人，為[[排灣族|排灣族]]原住民，於2007年參加[[臺灣|臺灣]][[華視|華視]]《[[快樂星期天|快樂星期天]]》的校園歌喉戰單元所主辦的第一年度歌唱選秀比賽，並於年度冠軍賽中獲得第五名，曾參與[[環球音樂|環球音樂]]所製作的《[[校園快樂幫|校園快樂幫]]No.1校園歌喉戰冠軍合輯》<ref>參見-{藝是娛樂}- · 蘇清和官網介紹http://ispr.pixnet.net/blog/post/15507939</ref>，演唱《愛一直閃亮》<ref>參見-{[[環球音樂|環球音樂]]}- · 合輯介紹{{cite web |url=http://www.umusic.com.tw/Microsite/prospects/index1.html |title=存档副本 |accessdate=2008年12月16日 |deadurl=yes |archiveurl=https://web.archive.org/web/20081104074124/http://www.umusic.com.tw/Microsite/prospects/index1.html |archivedate=2008年11月4日 |df= }}</ref>，現合約已到期，為[[國立勤益科技大學|國立勤益科技大學]]機械系學生。
