'''蘇瀑支線機場'''<ref name="Airport">[http://www.sfairport.com/ Sioux Falls Regional Airport], official website</ref> <span contenteditable="false">(</span>IATA<span contenteditable="false">: </span>'''FSD<ref><cite class="citation web">[http://www.iata.org/publications/Pages/code-search.aspx "IATA Airport Code Search (FSD: Sioux Falls Regional / Jo Foss Fld)"]. </cite></ref>'''<span contenteditable="false">,</span> [[國際民航組織機場代碼|ICAO]]<span contenteditable="false">: </span>'''KFSD'''<span contenteditable="false">,</span> [[美国联邦航空管理局|FAA]]<span contenteditable="false"> </span>LID<span contenteditable="false">: </span>'''FSD'''<span contenteditable="false">)</span>，也稱為 '''Joe Foss Field，'''<ref name="FAA">[http://www.gcr1.com/5010web/airport.cfm?Site=FSD FAA Airport Master Record for FSD] (<span class="PDFlink">[http://www.gcr1.com/5010web/Rpt_5010.asp?au=PU&o=PU&faasite=22789.*A&fn=FSD Form 5010]{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</span> <small>[[便携式文档格式|PDF]]</small>). </ref> 是一個由蘇瀑支線機場管理局管理的軍民共用機場。它位於[[蘇瀑|蘇瀑]]市中心西北方約6公里。<ref name="FAA">[http://www.gcr1.com/5010web/airport.cfm?Site=FSD FAA Airport Master Record for FSD] (<span class="PDFlink">[http://www.gcr1.com/5010web/Rpt_5010.asp?au=PU&o=PU&faasite=22789.*A&fn=FSD Form 5010]{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</span> <small>[[便携式文档格式|PDF]]</small>). </ref>它服務了泛蘇瀑地區，南達科塔州的東部，明尼蘇達州的西南部，以及愛荷華州的西北部。

該機場被包括在2011-2015年的綜合機場系統的國家計畫， 並因為它每年均有超過10,000個乘客而被分類為一個''主要的商業服務''的機場。在[[美國聯邦航空總署|美國聯邦航空總署]]的每一個紀錄裡，這個機場於2011年有423,288位乘客，比2010年的355,939位乘客增長18.92%。<span class="mw-ref" rel="dc:references" data-sourceid="cite_ref-5">[[#cite_note-5|<span class="mw-reflink-text"><nowiki>[5]</nowiki></span>]]</span>

FSD提供由Midco所提供的公眾[[WiFi|WiFi]]。

== 設施及飛機 ==
該機場佔地面積635[[公頃|公頃]] (1,570 [[英畝|英畝]]) ，海拔為436米（1430英尺）。機場擁有三條[[混凝土|混凝土]]跑道： 3/21為8,999 x 150英尺 (2,743 x 46米),  15/33是8,000 x 150英尺 (2,438 x 46米)，9/27是3,151x 75 英尺 (960 x 23米)。它也有一個有[[瀝青|瀝青]]表面的H1[[直升机停机坪|直升機停機坪]]，面積為50 x 50英尺(15 x 15 米)。<ref name="FAA">[http://www.gcr1.com/5010web/airport.cfm?Site=FSD FAA Airport Master Record for FSD] (<span class="PDFlink">[http://www.gcr1.com/5010web/Rpt_5010.asp?au=PU&o=PU&faasite=22789.*A&fn=FSD Form 5010]{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</span> <small>[[便携式文档格式|PDF]]</small>). </ref>

== 事故 ==
