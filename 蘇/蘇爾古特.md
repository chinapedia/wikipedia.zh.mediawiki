'''蘇爾古特''' （{{lang|ru|Сургу́т}}）是[[俄羅斯|俄羅斯]][[漢特-曼西自治區|漢特-曼西自治區]]最大的城市、[[秋明州|秋明州]]第二大城市，位於[[鄂畢河|鄂畢河]]畔。2010年人口為306,703 人。

始建於1594年，1965年6月12日建市。
== 历史 ==
苏尔古特于1594年在[[沙皇|沙皇]][[费奥多尔·伊万诺维奇|费奥多尔一世]]在位期间建立，是西伯利亚最古老的城市之一，苏尔古特一词来源于[[汉特语|汉特语]]单词 "sur" （鱼） "gut" （洞）的组合。20世纪60年代，该市成为[[秋明州|秋明州]]石油天然气工业中心，1965年6月25日建市，2000年通车的[[苏尔古特大桥|苏尔古特大桥]]是世界上最长的独塔斜拉桥。

== 气候 ==
苏尔古特属于亚寒带气候（[[柯本气候分类法|柯本气候分类法]]属于''Dfc''型），冬季苦寒，夏季稍事温暖，降水量中等。

== 经济 ==
石油资源和天然气加工是苏尔古特的支柱产业，该市享有“俄罗斯石油之都”的美誉，该市最知名的企业是Surgutneftegaz和Surgutgazprom（[[俄罗斯天然气工业股份公司|俄罗斯天然气工业股份公司]]的子公司）。苏尔古特电厂GRES-1和GRES-2产生超过7200 MW的廉价电力来供应该地区的生产生活需要。

== 交通运输 ==
[[苏尔古特国际机场|苏尔古特国际机场]]距离苏尔古特市区10公里，有通往[[莫斯科|莫斯科]]、[[圣彼得堡|圣彼得堡]]、[[迪拜|迪拜]]、[[伊尔库茨克|伊尔库茨克]]等地的定期航班。

秋明-新乌连戈伊铁路传过该市，通往[[秋明|秋明]]方向，苏尔古特的下一站是撒雷母（Salym），[[荷兰皇家壳牌石油公司|荷兰皇家壳牌石油公司]]在该地拥有油田设施。

P-404公路可以从苏尔古特通往秋明。

该市有[[鄂毕河|鄂毕河]]港口。
