'''蘇悉地羯羅經'''（[[梵語|梵語]]：Susiddhikara-mahātantra-sādhanopāyika-paṭala），收入[[大正藏|大正藏]]密教部。本經廣說有關佛部、蓮花部、金剛部等三部{{tsl|en|Siddhi|悉地}}成就之儀則，與《[[大日經|大日經]]》、《[[金剛頂經|金剛頂經]]》同為密教三部大經之一，內容包括持誦、[[灌頂|灌頂]]、祈請、[[護摩|護摩]]、{{tsl|en|Sādhanā|成就}}、時分等。

日本[[台密|台密]]認為，大日經、金剛頂經二經為「胎（藏）、金（剛）而二」之經；主張本經為詮釋金剛、胎藏「兩部不二」深旨之經<ref name="susiddhi">{{cite web|title=一行佛學辭典搜尋|url=http://buddhaspace.org/dict/index.php?keyword=%E8%98%87%E6%82%89%E5%9C%B0%E7%B6%93}}</ref>。

== 題解 ==
蘇悉地羯羅，意譯為「妙成就作業」，亦即成就一切世出世間作業的妙法<ref>{{cite web|title=蘇悉地羯羅|url=https://www.buddhistdoor.org/tc/dictionary/details/%E8%98%87%E6%82%89%E5%9C%B0%E7%BE%AF%E7%BE%85}}</ref>。

== 傳譯 ==
唐開元十四年（726）輸波迦羅（即[[善無畏|善無畏]]三藏）譯。大正藏第十八冊中所收之本經共有三種版本，皆為三卷，即麗本三十七品、宋本三十八品、和本三十四品。和本乃[[唐順宗|唐順宗]]永貞元年（805，即日本延歷二十四年），由日本天台宗開祖[[最澄|最澄]]請回日本；其後，[[圓仁|圓仁]]又請回梵本<ref name="susiddhi"/>。

== 內容 ==
全書敘說相應於息災、增益、降伏等作法之[[真言|真言]]及其持誦法、持誦者之人品、供養法、灌頂法、三種護摩等密教儀軌，以及根據此等密教儀軌而來的種種{{tsl|en|Sādhanā|成就}}法。

由於本經係闡明「真言行者」之威儀法則等，因此古來被稱為咒毗奈耶。據《[[開元釋教錄|開元釋教錄]]》，本經與《[[蘇婆呼童子請問經|蘇婆呼童子請問經]]》並為密教律部之戒經。密教以為，真言行者行住坐臥之行事，若能常依此二經，則悉地之事業較易成就。其行法乃依[[圓仁|圓仁]]之《蘇悉地妙心大》，及[[最澄|最澄]]之《藥師儀軌》的次第<ref name="susiddhi"/>。

== 地位 ==
[[東密|東密]]以大日經、金剛頂經兩部經典為最極祕經，空海雖由惠果傳此法，其所著《真言付法傳》僅記載兩部大法，而未付囑此法。[[台密|台密]]則以蘇悉地經說兩部不二之深旨，尤勝前二經，故特重蘇悉地灌頂<ref>{{cite web|title=蘇悉地灌頂|url=https://www.buddhistdoor.org/tc/dictionary/details/%E8%98%87%E6%82%89%E5%9C%B0%E7%81%8C%E9%A0%82}}</ref>。
