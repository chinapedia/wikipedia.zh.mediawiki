'''蘇格蘭足球聯賽'''（'''Scottish Football League'''）是蘇格蘭球會分級別以循環對賽形式的比賽，包含三個組別：'''[[蘇格蘭足球甲組聯賽|甲組聯賽]]'''或簡稱'''蘇甲'''（Scottish Division One），'''[[蘇格蘭足球乙級聯賽|乙組聯賽]]'''或簡稱'''蘇乙'''（Scottish Division Two）及'''[[蘇格蘭足球丙級聯賽|丙組聯賽]]'''或簡稱'''蘇丙'''（Scottish Division Three）。蘇格蘭足球聯賽現時由三十支球隊組成<small><ref>[http://www.scottishfootballleague.com/scottish_football.cfm?curpageid=946 S.F.L. Clubs] {{Wayback|url=http://www.scottishfootballleague.com/scottish_football.cfm?curpageid=946 |date=20070806114350 }}</ref></small>，為會員提供循環式聯賽、[[蘇格蘭聯賽杯|聯賽杯]]及[[蘇格蘭聯賽挑戰杯|聯賽挑戰杯]]等[[足球|足球]]競賽。日常會務由管理委員會處理，委員會由八位經選舉的球隊代表組成，包括會長及副會長各一，甲組、乙組及丙組代表各兩名。

每季的甲組聯賽冠軍有權升級到[[蘇格蘭足球超級聯賽|超級聯賽]]，但需符合其球場規格條例，按2006年3月的球場規格包括擁有或優先租用的主場球場容量不少於6,000座及球場草坪需有地下發熱系統。[[法尔科克足球俱乐部|福爾柯克]]在2002/03年賽季因球場規格不符而被拒升級，當時的容量規定為10,000座，福爾柯克為此新建全座席可容6,000人的球場，而超級聯賽亦順應將規定減為6,000，在2004/05年再獲甲組聯賽冠軍的福爾柯克才可順利升級。

近年來部份聯賽球隊發生財務困難而被接管，[[Airdrieonians_F.C.|艾迪爾人]]（Airdrieonians）更在2001/02賽季末遭[[清盤|清盤]]，聯賽會籍被停止，[[英格蘭|英格蘭]][[UniBond_League|北超聯]]（UniBond League）的[[格雷纳足球俱乐部|基特拿]]在2002年填補其空缺，而基特拿亦於2008年破產退出，由[[安南竞技足球俱乐部|安南竞技]]（Annan Athletic）加入替補。

[[流浪者足球俱乐部|格拉斯哥流浪]]是歷年來獲得最多聯賽冠軍的球隊，至1998年共获得47次苏格兰足球聯賽冠軍。

== 第一次單級別時期（1890-1893） == 
蘇格蘭足球聯賽始於1890年以提供有秩序的賽程給日漸增加的足球隊。國內十四支球隊獲邀請出席1890年3月20日在[[格拉斯哥|格拉斯哥]][[Holton's_Commercial_Hotel|豪頓商業酒店]]舉行的籌備全國性聯賽會議。有十二支球隊出席會議，缺席的其中一隊是當時最強勁及具影響力的[[女王公园足球俱乐部|格拉斯哥昆士柏]]，昆士柏一直抗拒加入聯賽，最後迫於形勢才在聯賽成立十年後的1900年加入，但其在蘇格蘭足球壇的地位亦大不如前。

== 第一次兩級別時期（1893-1915） == 
聯賽的推行證實成功，在1893年增加以[[Scottish_Football_Alliance|蘇格蘭足球同盟]]成員為骨幹的乙組聯賽，包括十支球隊：[[邓迪足球俱乐部|邓迪]]、[[希伯尼安足球俱乐部|-{zh-hans:希伯尼安; zh-hant:喜百年;}-]]、[[格里诺克慕顿足球俱乐部|-{zh-hans:慕頓; zh-hant:摩頓;}-]]、[[马瑟韦尔足球俱乐部|馬瑟韋爾]]、Northern、[[帕尔蒂克足球俱乐部|-{zh-hans:帕尔蒂克; zh-hk:巴特里;}-]]、[[Port_Glasgow|格拉斯哥港]]、[[St_Bernard's_F.C.|聖伯納德]]、Thistle及重新加入聯賽的Cowlairs（在1891/92賽季退出）。[[希伯尼安足球俱乐部|-{zh-hans:希伯尼安; zh-hant:喜百年;}-]]獲得29分成為首屆乙組聯賽冠軍。當時的升級或降級由球會投票決定，自動升降制度直到1922年才訂立。

== 第二次兩級別時期（1921-1923） ==

== 第二次三級別時期（1946-1955） ==

== 第三次三級別時期（1975-1994） == 
聯賽改制分為三級，更名為超級聯賽（10隊）、甲組（14隊）及乙組（14隊）。由於觀眾急跌而希望改制後增加強隊的對陣次數以刺激票房。三級制一直維持到1994年，其間只因政治原因而改變各級別的球隊數目分配：1986/87至1987/88賽季改為12:12:14，1988/89至1990/91賽季回復10:14:14及1991/92至1993/94賽季再回復12:12:14。

== 第四次三級別時期（1998-2008） ==
1998年超級聯賽的10支球隊脫離蘇格蘭聯賽組成[[蘇格蘭足球超級聯賽|蘇格蘭足球超級聯賽]]（蘇超）以獲取更大電視轉播收益。餘下的三個級別維持原有的名稱甲組、乙組及丙組，每級各10隊。2000年蘇格蘭聯賽再從[[Highland_Football_League|高地聯賽]]增添兩名成員，分別為[[Elgin_City_F.C.|埃尔金]]及[[Peterhead_F.C.|彼得黑德]]，使包括蘇超12隊（新加入[[邓弗姆林足球俱乐部|鄧弗姆林]]及[[圣米伦足球俱乐部|聖美倫]]）的聯賽成員增至42隊。自2000/01球季開始，蘇超與甲組為一升一降，其餘各級之間為兩升兩降。
