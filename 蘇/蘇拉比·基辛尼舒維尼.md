'''蘇拉比·基辛尼舒維尼'''（'''{{lang-ka|ზურაბ ხიზანიშვილი}}'''，{{bd|1981年|10月6日|catIdx=ხიზანიშვილი ,ზურაბ}}）是一名[[格魯吉亞|格魯吉亞]]足球員，現效力{{link-en|国际巴库|Inter Baku PIK}}，司職[[後衛_(足球)|後衛]]<ref>{{en}}[http://www.rovers.premiumtv.co.uk/page/ProfilesDetail/0,,10303~10070,00.html Official profile]{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>。

== 生涯 ==
;早年
基辛尼舒維尼的足球生涯在[[第比利斯戴拿模|第比利斯戴拿模]]開始，效力球會一季後，他轉投{{Link-en|第比利斯諾爾基戴拿模|FC Norchi Dinamo Tbilisi}}以爭取正選上陣的機會。

1999年，他加盟前球會的同市競爭對手[[第比利斯火車頭|第比利斯火車頭]]。他在第比利斯火車頭的表現雖然引起了部分[[歐洲|歐洲]]頂級球會的注意，不過他在後期卻備受傷患困擾，職業生涯蒙上陰影。最終，他被球會終止合約。

;蘇格蘭時期
他在[[阿仙奴足球會|阿仙奴]]、[[富咸足球會|富咸]]和[[韋斯咸足球會|韋斯咸]]試腳均失敗<ref>{{en}}{{cite web

他迅速在{{Link-en|丹斯公園球場|Dens Park}}顯示出他的影響力，並引起了[[蘇格蘭足球超級聯賽|蘇超]]球會[[格拉斯哥流浪足球會|格拉斯哥流浪]]的注意。2003年6月，他通過[[波士文裁決|波士文裁決]]以交換球員的形式加盟格拉斯哥流浪，{{Link-en|羅倫索·亞武羅素|Lorenzo Amoruso}}則加盟登地聯。

儘管他在[[埃布羅克斯球場|埃布羅克斯球場]]上陣了35場賽事，不過他卻逐漸失寵。2005年8月31日，他被球會允許外借至[[英格蘭足球超級聯賽|英超]]球會[[布力般流浪足球會|布力般流浪]]直至當季結束<ref>{{en}}{{cite web

|url=http://news.bbc.co.uk/sport2/hi/football/teams/b/blackburn_rovers/4896390.stm

他在[[保羅·恩斯|保羅·恩斯]]和[[森姆·艾拿戴斯|森姆·艾拿戴斯]]的麾下失去了正選地位。2009年9月17日，他被球會外借至[[英格蘭足球冠軍聯賽|英冠]]球會[[紐卡素足球會|紐卡素]]，為期三個月<ref>{{en}}{{cite web

== 國際賽 ==
基辛尼舒維尼是[[格魯吉亞國家足球隊|格魯吉亞]]的常規球員。

2009年6月6日，他在對[[摩爾多瓦國家足球隊|摩爾多瓦]]的賽事中攻入其唯一一球的國際賽入球<ref>{{ka icon}}{{cite news|url=http://www.gff.ge/index.php?lang_id=ENG&sec_id=114&info_id=3146|title=Match Report|publisher=GFF|accessdate=2009-06-11}}</ref>。

== 個人生活 ==
基辛尼舒維尼與格魯吉亞著名模特兒莎樂美·格菲娜舒娃，並育有三名孩子。他是前[[蘇聯國家足球隊|蘇聯]]後衛{{Link-en|盧達·基辛尼舒維尼|Nodar Khizanishvili}}的兒子。
