'''苏四十三'''（{{bd|1729年||1781年||}}），人称'''尕苏阿訇'''。[[撒拉族|撒拉族]]。原籍[[甘肃|甘肃]][[河州|河州]]（今[[临夏|临夏]]），祖迁[[循化厅|循化厅]]撒拉街子工，父亲苏那黑置庄田于查加工古节烈庄，遂定居。自幼习[[阿拉伯文|阿拉伯文]]，入学街子工清真寺，后为阿訇。

== 早年经历 ==
[[乾隆|乾隆]]二十六年(1761年)，[[马明心|马明心]]到循化传[[苏菲主义|苏菲主义]][[哲合忍耶|哲合忍耶]]学理，人称“新教”，苏四十三成为其最早的门生之一。乾隆三十四年(1769年)，循化哲合忍耶派首领贺麻路乎与韩哈吉教争，被官府发配[[乌鲁木齐|乌鲁木齐]]为奴，苏四十三成为循化哲合忍耶派首领，在循化传播新教，提出革除[[门宦|门宦]]、消灭土司，与“老教”[[虎夫耶|虎夫耶]]华寺门宦教争，清政府施行“帮扶旧教”、“尽洗新教”、“赦一剿一”的政策，打压新教，苏四十三对此不满。

== 反叛 ==
乾隆四十六年（1781年）旧历三月，循化[[伊斯兰教|伊斯兰教]]新老教派械斗。虎夫耶教长韩三十八被杀，其子上告[[兰州|兰州]]省城衙门，要求惩处苏四十三、韩二个。[[陕甘总督|陕甘总督]][[勒尔谨|勒尔谨]]派兰州[[知府|知府]][[扬士玑|扬士玑]]会同河州副将[[新柱|新柱]]处理。因衙门支持老教，苏四十三率一千余人起兵反叛，冲击衙门，杀死新柱和刘汉时、扬士玑、守备徐前彦登和土司韩成玉等。又联合部分虎夫耶派教民、[[汉族|汉族]]、[[藏族|藏族]]等二千余人，攻克河州，杀知州，释放犯人。当地[[回族|回族]]和[[东乡族|东乡族]]也都参与叛军，并贡献粮食。叛军过[[洮河|洮河]]，陕甘总督勒尔谨在[[临狄道|临狄道]](今[[临洮市|临洮市]])坐镇指挥军队镇压。[[甘肃布政使|甘肃布政使]][[王廷赞|王廷赞]]命令把在安定(今[[定西|定西]])关川的苏四十三师傅马明心诱捕到兰州。三月廿一日，苏四十三为营救道祖马明心，率众三千攻打兰州。[[乾隆帝|乾隆帝]]得讯，调陕西、四川和[[新疆|新疆]]的军队出击，令户部拨款一百八十万两军饷，命王廷赞“督率坚守”、“毋乃过涉张惶，若稍有疏虞，王廷赞不能当其咎。”<ref>《清实录》卷1127</ref>廿六日，清军到兰州，次日，叛军从西关攻城，要求释放马明心，王廷赞带马明心上城，要求马明心劝起义军后撒，被拒绝，王廷赞处决马明心，用大炮轰退起义军。

== 战殁 ==
在两万多清军的攻击下，匪首苏四十三带叛军逃到龙尾、华林二山。四月二十日，[[大学士|大学士]]、[[都统|都统]][[阿桂|阿桂]]到兰州，以“坐失时机”、“办理谬误”罪名，将勒尔谨革职问罪，放出因贪污下狱的[[云贵总督|云贵总督]][[李侍尧|李侍尧]]，代勒之职。阿桂设下了“以退为进”、“佯败诱之出壕” <ref>《再征叛回》</ref>之计，毙伤叛军军二百多人。又令“绝汲道，湮井沟”。六月十五日，大雨，清军炮火失效，叛军反击，击退清军。午后天明，清军火攻，焚毁叛军帐房。匪首苏四十三持大刀与阿訇马明德、海朝宗等指挥反击，阵亡。叛军到七月六日下午被清军全部歼灭。
