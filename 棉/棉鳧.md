'''棉鳧'''（''Nettapus coromandelianus''）<ref name=ali-daniel>{{cite book en|last = Ali|first = Salim|coauthors = J C Daniel|title = The book of Indian Birds, Twelfth Centenary edition|year = 1983|publisher=Bombay Natural History Society/Oxford University Press|location = New Delhi}}</ref>又名'''-{zh-cn:棉鸭;zh-tw:棉鳧;}-'''，是[[印度|印度]]、[[巴基斯坦|巴基斯坦]]、[[東南亞|東南亞]]及[[澳洲|澳洲]]的細小[[棲鴨|棲鴨]]。

== 特征 ==
[[File:Cotton_Pgymy_Goose_I2-_Kolkata_IMG_4808.jpg|thumb]][[加爾各答|加爾各答]]的雄性棉鳧。]]
[[File:Cotton_Pygmy_Goose_I_IMG_9241.jpg|thumb]]的雌性棉鳧。]]
[[File:Whiskered_Tern_with_Cotton_Pygmy_Geese_I_IMG_9385.jpg|thumb]]一起休息的雌性棉鳧。]]

棉鳧可算是[[世界|世界]]上最細小的水鳥，只重160克及長26厘米。牠們的[[羽毛|羽毛]]主要呈白色。鳥喙短而像[[鵝|鵝]]的喙，底部較深。雄性棉鳧繁殖時毛色有黑綠色光澤，頭部、頸部及下身主要呈白色，有明顯的黑色頸圈及白色的覆尾羽。頭圓及腳短。飛行時，雄鳥雙翼呈綠色及有白帶，在大群[[樹鴨|樹鴨]]中亦易於確認。雌鳥較淡色，沒有黑色頸圈，只有窄小的白色覆尾羽。在非繁殖期間，雄鳥的羽毛與雌性的相似。

== 分佈 ==
棉鳧一般都不會遷徙，但在潮濕的季節會分散開來。生活在[[中國|中國]]的棉鳧則會向南方遷徙。牠們在樹穴中築巢，每次產8-15顆蛋。

棉鳧在[[亞洲|亞洲]]很是普遍，但在[[澳洲|澳洲]]的數量則似乎在逐步下降。

棉鳧一般生活在[[湖泊|湖泊]]、雨水溝等。由於棉鳧較為溫馴，所以很多時都會生活在[[人類|人類]]附近。

== 食物 ==
棉鳧主要吃[[種子|種子]]及[[蔬菜|蔬菜]]，尤其[[睡蓮科|睡蓮科]]，並[[昆蟲|昆蟲]]、[[甲殼類|甲殼類]]等。

== 築巢 ==
棉鳧會於7月至9月在近水的空樹幹中築巢，有時會以草、垃圾及羽毛作為物料。
