《'''棉花俱樂部'''》（{{lang-en|'''''The Cotton Club'''''}}）是一部1984年的[[犯罪電影|犯罪電影]]，以1930年代[[哈林區|哈林區]]的{{link-en|棉花俱樂部 (夜店)|Cotton Club|棉花俱樂部}}為主題。

本片由[[法蘭斯·哥普拉|法蘭斯·哥普拉]]執導及編劇（與{{link-en|威廉·肯尼迪|William Kennedy (author)}}共同編劇），主演是[[李察·基爾|李察·基爾]]、[[格雷戈里·海因斯|格雷戈里·海因斯]]、[[黛安·蓮恩|黛安·蓮恩]]和{{link-en|隆內特·麥基|Lonette McKee}}，其他演員包括[[鮑勃·霍斯金斯|鮑勃·霍斯金斯]]、[[詹姆士·雷瑪|詹姆士·雷瑪]]、[[尼古拉斯·基治|尼古拉斯·基治]]、{{link-en|阿倫·加菲爾德|Allen Garfield}}、[[羅蘭士·費斯賓|羅蘭士·費斯賓]]、{{link-en|格溫·沃頓|Gwen Verdon}}及{{link-en|弗雷德·格溫|Fred Gwynne}}。

儘管電影票房欠佳，卻獲得幾個獎項提名，包括[[金球獎|金球獎]]最佳導演和最佳電影，[[奧斯卡金像獎|奧斯卡金像獎]][[奧斯卡最佳藝術指導獎|最佳藝術指導]]和[[奥斯卡最佳影片剪辑奖|最佳影片剪接]]<ref>{{Cite web |url = http://www.boxofficemojo.com/oscar/movies/?id=cottonclub.htm |title = The Cotton Club |publisher = Box Office Mojo |language = en}}</ref>。然而，黛安·蓮恩得到[[金草莓獎|金草莓獎]]最差女配角提名。

== 劇情 ==
在1920至1930年間，「棉花俱樂部」是紐約哈林區一家著名的夜總會，Dixie Dwyer則是在俱樂部裡表演的喇叭手，在那個黑幫橫行的年代，這處銷金窟自然也屬於黑手黨所管轄，Dixie無意中拯救老大Dutch Schultz免遭槍殺，於是Schultz就將青梅竹的女人Vera Cicero託付給Dixie照顧，之後Vera及Dixie萌生愛意，而黑幫糾葛日益紛擾，棉花俱樂部的愛恨情仇一觸即發。
