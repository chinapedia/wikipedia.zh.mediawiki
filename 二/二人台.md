'''二人台'''，又名'''二人班'''、'''打玩意儿'''，是[[中国|中国]]地方[[戏曲|戏曲]]剧种之一，[[中国国家级非物质文化遗产|中国国家级非物质文化遗产]]之一。因使用[[晋语|晋语]]表演，其流行地区主要是[[山西省|山西省]]、[[河套地区|河套地区]]及其周边晋语使用地区。

== 历史 ==
二人台起源于[[山西|山西]]，成长于[[内蒙古|内蒙古]]，流行于山西北部、内蒙古西部、[[陕西|陕西]]北部、[[河北|河北]][[张家口|张家口]]地区等四省区，在[[宁夏|宁夏]]部分地区也有表演。

[[清朝|清朝]]后期，山西北部的农民为了逃荒西出[[塞外|塞外]]，远赴内蒙古中西部乃至更遥远的地区垦荒、挖煤、拉骆驼、做小生意。这种行为被称为“[[走西口|走西口]]”。这些人把山西的[[秧歌|秧歌]]、[[高跷|高跷]]、[[旱船|旱船]]、[[道情|道情]]、[[打坐腔|打坐腔]]、[[转火龙|转火龙]]等民间艺术形式带到塞外，加上内蒙古民歌的音乐和戏剧的化妆，形成了二人台这个艺术形式。也有说法认为二人台源自[[宋代|宋代]]初年的“舞鞭”<ref>[http://news.folkw.com/www/dongtaizixun/084617401.html 山西河曲“二人台”和“河曲”地名之源]2008年6月19日造访。</ref>，源自清[[康熙|康熙]]年间流入张家口地区的垦荒移民带来的民歌<ref>[http://www.hebei.com.cn/node2/hbgl/whys/dfxq/userobject1ai218791.html 坝上二人台] {{webarchive|url=https://web.archive.org/web/20050923065259/http://www.hebei.com.cn/node2/hbgl/whys/dfxq/userobject1ai218791.html |date=2005-09-23 }}河北省官方网站，2008年6月19日造访。</ref>，或者[[明朝|明朝]]末年就已经形成于内蒙古[[土默川|土默川]]地区<ref name="二"/>。

[[民国|民国]]初年，[[蒙古族|蒙古族]]民间艺人[[荣双羊|荣双羊]]改变“丝弦坐腔”形式，开始分角色化妆演唱，把二人台艺术带进了河套地区。1931年后，[[计子玉|计子玉]]、[[樊六|樊六]]的二人台小班在河套地区表演。1949年[[中华人民共和国|中华人民共和国]]成立前，[[巴彦淖尔盟|巴彦淖尔盟]]地区共有二人台职业小班20至30个<ref name="二">[http://www.nmg.gov.cn/pages/2005720/20057202310533.htm 二人台] {{webarchive|url=https://web.archive.org/web/20060719004306/http://nmg.gov.cn/pages/2005720/20057202310533.htm |date=2006-07-19 }}中国内蒙古官方网站二人台介绍，2005-07-20，2008年6月19日造访。</ref>。

和源于山西[[忻州市|忻州市]][[河曲县|河曲县]]流行于[[呼和浩特|呼和浩特]]以西的“西路二人台”交相辉映的是源于河北张家口地区[[尚义县|尚义县]]，以[[康保县|康保县]]、[[张北县|张北县]]等地为流传中心的“东路二人台”，又称“坝上二人台”、“张家口二人台”、“河北二人台”。东路二人台唱念用张家口及坝上地区方言，平声不分阴阳，有入声字，这是与西路二人台的重要区别<ref>[http://www.jiaozhou.gov.cn/ygjnews/jzyg/ygtp/tpjj/200804/481.html 河北二人台]{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}[[胶州|胶州]]官方网站，2008-4-29，2008年6月19日造访。</ref>。

== 艺术特点 ==
二人台故名思意，就是两个人一台戏，表演以两个人说唱为主，有一[[生角|生]]一[[旦角|旦]]、一[[丑角|丑]]一旦、两小旦等，但都是两个人出场。内容多为[[喜剧|喜剧]]。

二人台使用的语言是[[晋语|晋语]]。有些戏如《什可拉奔花》、《海莲花》等采用[[蒙语|蒙]]汉两种语言演唱<ref name="二"/>。

二人台服装和化妆仿效[[晋剧|晋剧]]，道具包括长绸、[[扇子|扇子]]、手帕、[[霸王鞭|霸王鞭]]等。二人台伴奏乐器有[[笛子|笛子]]、[[扬琴|扬琴]]、[[四胡|四胡]]、“四块瓦”（两对竹板）等。

== 现状 ==
二人台和众多民间艺术形式一样，在当代日益失去影响力。内蒙古、山西、陕西、河北当地的政府试图对二人台进行扶植。

2004年5月四省联合举办了“二人台”大赛。同年四省拍摄了12集大型电视文化纪实片《魂牵梦绕二人台》<ref>[http://news.xinhuanet.com/society/2006-02/28/content_4236655.htm “二人台”与“二人转”：一字之差天壤之别]新华网，2006年2月28日，2008年6月18日造访。</ref>。

== 传统曲目 ==
二人台的传统曲目中有不少以“打”字开头的曲目，故而得名“打玩意儿”。二人台的传统剧目有200多个。

*《[[走西口|走西口]]》
*《打金钱》
*《打樱桃》
*《打后套》
*《五哥放羊》
*《珍珠倒卷帘》
*《打连城》

1949年之后，有大量新曲目出现，但传统曲目仍然占据主要市场。
