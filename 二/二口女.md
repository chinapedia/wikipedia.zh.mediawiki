''' 二口女'''（{{lang|ja|ふたくちおんな}}）是[[日本妖怪|日本妖怪]]中的一種後腦勺也長了一張[[口腔|口]]的[[女性|女性]]妖怪，她可以將自己的頭髮當做[[觸手|觸手]]使用，以向頭後面的口餵食。在日本的[[民間傳說|民間傳說]]中經常出現，其正體多是深山中的農村中出現的[[山姥姥|山姥姥]]。

== 民間傳說「不用吃飯的新娘」 ==
''傳說的版本可能由於地區因素而稍有不同，其中最有名的是福島縣的說法''：

某村住了一個非常吝嗇的男子，希望能娶到一位不用吃飯的女子作為妻子。後來真的有一位女子表示她不用吃飯，男子遂娶其為妻。婚後妻子在吃飯的時候果然沒有吃任何東西，男子感到非常滿意。不過不久就發現家裡儲存的糧食卻不斷減少。

於是某日男子假裝出外辦事，卻躲在家中偷看其妻，結果看到妻子解開頭後面的頭髮，赫然露出第二張嘴，並用頭髮將[[飯糰|飯糰]]送入其中，臉上的嘴還說道「不錯，飯糰握的很好」。男子嚇得想要逃走，卻被恢復了[[山姥姥|山姥姥]]原形的妻子裝入浴盆準備帶回深山中。男子在山道上成功逃脫，並藏身於生長了香味極濃的[[菖蒲|菖蒲]]的濕原，山姥姥跟踪其足跡到此，結果最後被菖蒲的靈力融化而死。

== 對民間傳說的分析 ==
二口女的民間傳說不僅讓吝嗇、愛好虐待妻子的男人引以為戒，同時也表示了對外表看似貞淑的女性可能具有恐怖本性的憂慮。

首先，男子的嫁娶條件，「什麼都不用吃的女性」當然是現實中不存在的，山里住的妖怪為男子的貪慾所吸引嫁到男子家本身就包含了「除了人類以外」或「不人道的待遇」的涵義。其次，在男子為了滿足自己的貪慾而省下糧食的同時，卻同時背負上了生命危險，反映了人類的貪慾甚至可以左右生死。

這個民間傳說在日本全國都有，但是[[東日本|東日本]]和[[西日本|西日本]]在內容上是有很大區別的。前述的二口女是山姥姥這一類的[[鬼女|鬼女]]，男子靠菖蒲之力得以逃脫的說法主要來自東日本，山姥姥襲擊男子的日子為5月5日，也是日本的端午節家家都插菖蒲以趕鬼的風俗的由來。<ref name="naya">{{Cite book<!--|和書-->|author=那谷光代|editor=稲田浩二他編|title=日本昔話事典|year=1977|publisher=[[弘文堂|弘文堂]]|isbn=978- 4-335-95002-5|pages=704-706頁|chapter=食わず女房}}</ref><ref name="hanabe">{{Cite book<!--|和書-->|author=花部英雄|editor =乾克己他編|title=日本伝奇伝説大事典|year=1986|publisher=[[角川書店|角川書店]]|isbn=978-4-04-031300-9|pages=338頁|chapter=食わず女房}}</ref>在西日本則多認為二口女的正體是[[蜘蛛|蜘蛛]]。二口女知道男子想要逃跑，和同伴說道「今天晚上會讓蜘蛛殺了他」，卻被男子偷聽到，於是夜裡用圍爐裡的火將家裡出現的蜘蛛全殺了。這也是日本諺語「夜のクモは親に似ていても殺せ」（夜裡出現的蜘蛛即使像親人一樣也要把它殺了）的由來。<ref name="naya" /><ref name="hanabe" />據說蜘蛛來的那一晚正好是[[除夕|除夕]]，因此有了除夕之夜家中的爐火不可以熄滅的習俗。 <ref name="hanabe" />

另外，男子為了節省糧食而不讓妻子吃飯最終失敗的故事常常被長輩當做[[笑話|笑話]]講給小孩子聽。<ref name="hanabe" />

== 《繪本百物語》中的二口女 ==
[[江戶時代|江戶時代]]的怪談集《[[繪本百物語|繪本百物語]]》中對「二口女」有另一種解釋：

[[下總國|下總國]]（現在的[[千葉縣|千葉縣]]）的一個男子在妻子去世後娶了後妻，他和去世的妻子有一個女兒。當後妻自己生了女兒後，就開始虐待前妻的女兒，不給她吃飯，最後女孩子被餓死了。

這樣過了四十九天后（即七七之後），男子在砍柴的時候不小心砍到了妻子的後腦勺。當時出了很多血，但是沒有危及到生命，傷口好了之後卻長成了人的嘴唇的形狀，頭蓋骨突出一部分變成了牙齒，頭上的肉也有部分變成了舌頭。這個傷口時時刻刻都在疼，只有給這張嘴餵食才可以減輕痛苦。
之後，傷口處還不斷傳來細小的聲音。將耳朵靠近後可以聽出小孩子的聲音在不斷說道「這是對違心殺害前妻之女的報應」。

同本書中還提到，江戶時代的古書中有記載一種叫做「[[人面瘡|人面瘡]]」的奇病，作惡者會長出人面瘡，二口女可能也是因為其惡行而長出人面瘡的女性。 <ref name="ehon100">{{Cite book<!--|和書-->|author=桃山人|editor=[[多田克己|多田克己]]編|title=竹原春泉絵本百物語-桃山人夜話-|year=1997|publisher= [[國書刊行會|國書刊行會]]|isbn=978-4-336-03948-4|pages=51-52頁}}</ref>《繪本百物語》即是利用怪談宣揚[[道德|人道]]，告誡世人。 <ref>{{Cite book<!--|和書-->|title=竹原春泉絵本百物語|pages=139頁}}</ref>

《繪本百物語》的二口女是人類作惡而成，前面提到的民間傳說「不用吃飯的新娘」的二口女是妖怪山姥姥的化身，兩者是對同一種妖怪的不同解釋。 <ref>{{Cite book<!--|和書-->|author=[[村上健司|村上健司]]編著|title=日本妖怪大事典|year=2005|publisher=[[角川書店|角川書店]]|series=Kwai books|isbn=978-4 -04-883926-6|pages=287-289頁}}</ref>
