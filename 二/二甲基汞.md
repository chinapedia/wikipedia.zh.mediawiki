'''二甲基汞'''，化學式(CH<sub>3</sub>)<sub>2</sub>Hg，是一種含[[汞|汞]][[有機化合物|有機化合物]]。易揮發，易燃，劇毒。<ref name="化学百科">{{zh}}[http://www.chinabaike.com/article/316/hx/2007/20071109627095.html 化学百科 二甲基汞]</ref>亦是已知最危險的有機汞化合物，对胎兒的[[神經系統|神經系統]]、[[智商|智商]]和[[記憶|記憶]]等有危害<ref>{{zh}}[http://www.51hp.net/Article/hbcs/201010/207177.html 無憂環評 二甲基汞與智商]{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，數微升即可致死。

二甲基汞能滲過[[乳膠|乳膠]]，溶解[[橡胶|橡胶]]和[[生胶|生胶]]。美國[[達特茅斯學院|達特茅斯學院]]的毒物化學教授{{le|卡伦·韦特豪恩|Karen Wetterhahn}}，因在實驗中只不慎將“兩滴”二甲基汞灑在戴手套的手上，二甲基汞滲過她戴著的乳膠[[手套|手套]]後接觸[[皮膚|皮膚]]，最終不治身亡。<ref>{{zh}}[http://ejournal.stpi.narl.org.tw/NSC_INDEX/Journal/EJ0001/9804/9804-09.pdf 徐統 清華大學材料科學工程系 汞]</ref>

== 性質 ==
=== 物理性質 ===
常溫常壓為無色液體。能與[[乙醇|乙醇]]和[[乙醚|乙醚]]互溶，不能與[[水|水]]互溶<ref name="化学百科" />。

=== 化學性質 ===
二甲基汞與[[氯化汞|氯化汞]]反應，生成[[氯化甲基汞|氯化甲基汞]]：
: <math>\rm \ (CH_3)_2Hg  +  HgCl_2 \rightarrow 2CH_3HgCl</math>

它也容易被光解，产生[[甲烷|甲烷]]、[[乙烷|乙烷]]和[[汞|汞]]。<ref name=env />

== 製備 ==
由[[甲基鋰|甲基鋰]]與[[氯化汞|氯化汞]]在[[乙醚|乙醚]]的環境下反應生成<ref name="化学百科" />。

由[[钠汞齐|钠汞齐]]和[[碘甲烷|碘甲烷]]在[[二甲苯|二甲苯]]环境下反應，生成二甲基汞和[[碘化鈉|碘化鈉]]。

: <math>\rm \ Hg + 2Na +  2CH_3 I \rightarrow (CH_3)_2Hg + 2 NaI</math>

實驗室可由過量[[格氏試劑|格氏試劑]]與氯化汞作用制得。

== 用途 ==
* 製備[[三甲基镓|三甲基镓]]<ref name="化学百科" />和[[氯化甲基汞|氯化甲基汞]]

== 危險 ==
* 遇明火、高溫或強[[氧化劑|氧化劑]]燃燒。
* 能透過[[聚氯乙烯|聚氯乙烯]]和[[氯丁橡膠|氯丁橡膠]]，接觸時必須穿戴特別手套，[[皮膚|皮膚]]接觸數微升即可致死。
* 危害環境，對水生生物具長期危害。
