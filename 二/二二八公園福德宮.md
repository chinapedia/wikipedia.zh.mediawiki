'''二二八公園福德宮'''，位於[[臺北市|臺北市]][[二二八公園|二二八公園]][[急公好義坊|急公好義坊]]之側，地址[[臺北市|臺北市]][[中正區_(臺北市)|中正區]]懷寧街109號，俗稱'''新公園土地廟'''，主祀[[福德正神|福德正神]]（土地公），是位於[[博愛特區|博愛特區]]的知名[[土地廟|土地廟]]，是[[北臺灣|北臺灣]]最有名的[[土地廟|土地廟]]之一。

== 祀神 ==
*[[福德正神|福德正神]]、[[地母|地母]]娘娘、[[關聖帝君|關聖帝君]]、[[中壇元帥|中壇元帥]]等。

== 簡史 ==
相傳清代[[嘉慶|嘉慶]]年間，[[鄉民|鄉民]]在附近偶見巨石，燒香奉敬，稱作「石頭公」，祈求出入平安。

[[日本|日本]][[昭和|昭和]]六年（1931年）建了[[臺北天滿宮社|臺北天滿宮社]]，祭祀[[神道|神道]]學問之神[[菅原道真|菅原道真]]。

1945年，[[日本投降|日本投降]]，[[神社|神社]]遭到毀棄，改奉石頭公，而後又認為是土地公[[化身|化身]]，改稱[[土地廟|土地廟]]，每逢[[作牙|作牙]]日，皆虔誠敬奉。

1980年代，臺北市政府公園路燈管理處員工鳩資重建。

2000年代，信眾成立管理委員會，[[臺北市政府|臺北市政府]]同意立案。

== 特色 ==
福德宮「管區」包含[[臺灣總統府|總統府]]等政府機關，可說是「最有權勢」的土地公之一，政界參拜者眾，不分[[泛藍|藍]][[泛綠|綠]]，[[臺灣總統|總統]][[馬英九|馬英九]]、前[[行政院院長|行政院院長]][[蘇貞昌|蘇貞昌]]、[[立法院長|立法院長]][[王金平|王金平]]、前[[監察院長|監察院長]][[陳履安|陳履安]]、前[[總統府秘書長|總統府秘書長]][[葉菊蘭|葉菊蘭]]等政治人物也都曾親臨參拜。逢年過節，附近居民、行人也會準備祭品來祈求平安。
