[[有机化学|有机化学]]中，'''二硫化物'''指含有[[二硫键|二硫键]]（-S-S-）的[[有机硫化合物|有机硫化合物]]，具有通式R-S-S-R'。其中的[[硫|硫]][[氧化态|氧化态]]为-1，与[[过氧化物|过氧化物]]中的氧类似。

R与R'都为氢（H<sub>2</sub>S<sub>2</sub>），或有一个为氢（H-S-S-R）时，得到的二硫化物都是不稳定的。但相对于同族多具爆炸性的[[有机过氧化物|有机过氧化物]]，二硫化物稳定性更强，有些[[多硫化物|多硫化物]]也是稳定的。

==合成==
对称的二硫化物一般由两分子[[硫醇|硫醇]]（R-SH）氧化缩合制取，氧化剂可以是[[双氧水|双氧水]]或[[碘|碘]]。用双氧水作氧化剂时，需严格控制反应物的量，氧化剂过量则会导致二硫化物被进一步氧化为[[亚磺酸|亚磺酸]]，以及[[磺酸|磺酸]]。

:<math>\rm 2R-SH \longrightarrow R-S-S-R</math>

此外，用一比一的[[硫化钠|硫化钠]]与[[硫|硫]]配置Na<sub>2</sub>S<sub>2</sub>（[[多硫化物|多硫化钠]]），再使其与两倍量的[[烷化剂|烷化剂]]（R-X）反应，也可以得到二硫化物：

:<math>\rm Na_2S + S \longrightarrow Na_2S_2</math>

:<math>\rm 2R-X + Na_2S_2 \longrightarrow R-S-S-R + 2NaX</math>

不对称的二硫化物可通过[[硫醇|硫醇]]负离子与R-S-X类型的化合物于低温反应制备（-X为-[[卤素|卤素]]、-SO<sub>3</sub>Na、-CN）。反应条件应当严格控制，过量的硫醇盐会使此类二硫化物发生交换反应，从而降低产率。

:<math>\rm R-S-X + R'-S^- \longrightarrow R-S-S-R' + X^-</math>

==反应==
放置或还原剂[[还原|还原]]时，二硫化物会分解为相应的硫醇。[[亲核试剂|亲核试剂]]的进攻也会打断二硫键，形成两个新的硫醇/[[硫醚|硫醚]]：

:<math>\rm R-S-S-R' + Nu^- \longrightarrow R-S^- + Nu-S-R'</math>

此类型的氧化还原反应在生物体内及医药上都有很重要的用途。
