'''二俁川站'''（{{jpn|j=二俣川駅|hg=ふたまたがわえき|rm=Futamatagawa eki|f=y}}）位於[[日本|日本]][[神奈川縣|神奈川縣]][[橫濱市|橫濱市]][[旭區_(橫濱市)|旭區]]{{link-ja|二俁川|二俣川}}二丁目，是[[相模鐵道|相模鐵道]]的[[鐵路車站|鐵路車站]]。[[車站編號|車站編號]]是'''SO10'''。{{link-ja|電報略號 (鐵道)|電報略号 (鉄道)|電報略號}}是「{{lang|ja|フタ}}」。2017年3月18日改點後現在相鐵所有的定期列車都停靠此站。

作為二俁川管區管理此站－[[希望丘站|希望丘站]]。

== 車站構造 ==
[[島式月台|島式月台]]2面4線的[[橋上車站|橋上車站]]。基本上外側2線為相鐵本線、中央2線為泉野線。月台東側與内側有2條引上線，供不停本站時使用。

== 历史 ==
*1926年（[[大正|大正]]15年）5月12日 開業。當時為島式月台2面3線。
*1976年（[[昭和|昭和]]51年）4月8日 泉野線二俁川 - 泉野間開業。
*1989年（[[平成|平成]]元年）5月28日 改良為島式月台2面4線。增設引上線。車站構內流過的二俁川[[暗渠|暗渠]]化。

== 站名的由来 ==
車站所在地的地名為「'''二俁川'''」。
