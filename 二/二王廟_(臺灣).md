'''二王廟'''位於臺灣[[臺南市|臺南市]][[永康區|永康區]]，主祀鄭府二王爺，目前多認為該神祇即是[[鄭經|鄭經]]，是全臺灣唯一主祀鄭經的廟宇{{r|祀神故事}}{{r|神明傳說}}{{r|石}}{{r|戴}}。其廟名有時會冠上所在地地名，而稱'''永康二王廟'''、'''網寮二王廟'''{{NoteTag|該地原屬「網寮里」，1994年分出「二王里」{{r|戴}}。}}、'''甲內二王廟'''{{NoteTag|當地舊稱「甲內」{{r|戴}}。}}等等{{r|石}}{{r|戴}}。該廟原本在「二王崙」上{{NoteTag|舊址在今永康中山南路（舊稱「南新公路」）砲校對面南側，二王里活動中心廣場內{{r|石}}{{r|神明傳說}}{{r|戴}}。《臺南州祠廟名鑑》記載的所在地是「永康庄網寮780」{{r|相}}。}}，在清中葉是[[永康上中里|永康上中里]]的境主廟，但隨著各庄頭各自建廟或廢庄，變成單純的庄廟，於二次大戰後遷到今址{{r|神明傳說}}{{r|石}}{{r|戴}}。

== 沿革 ==
[[File:重修二王廟碑記_01.jpg|thumb]]
[[File:重修二王廟碑記_02.jpg|thumb]]
=== 立廟由來 ===
據說二王廟前身為一小祠，於鄭經於二王崙墜馬逝世後建立{{r|石}}{{r|神明傳說}}。康熙四十一年（1702年）{{NoteTag|《[[臺南州祠廟名鑑|臺南州祠廟名鑑]]》也記載二王廟創立於康熙四十一年{{r|相}}。}}，當地仕紳李文奇等人捐資蓋廟，於該年十二月廿六日（1703年2月11日）開工，次年三月建成（1703年4、5月）{{r|石}}{{r|神明傳說}}。

二王廟之由來除了紀念鄭經的說法之外，還有紀念[[鄭成功|鄭成功]]之弟的說法{{r|相}}{{r|戴}}。此外除了永康二王崙有二王廟之外，根據[[蔣毓英|蔣毓英]]《臺灣府志》（1685年）的記載，在府城[[東安坊|東安坊]]也有二王廟{{NoteTag|《臺灣府志‧卷六》：「二王廟：在東安坊，-{云}-神乃代天巡狩之神，威靈顯赫，土人祀之，內有寧靖王書匾「代天府」三字。」 學者石萬壽〈王爺信仰與延平王君臣關係之探討〉一文認為該廟乃[[鄭克塽|鄭克塽]]紀念鄭經而建於孔廟東北，[[原臺南警察署|原臺南警察署]]處{{r|戴}}。}}{{r|戴}}。

=== 發展 ===
[[File:1933年前的永康二王廟.jpg|thumb]]
據《欽定平定臺灣紀略》記載，[[林爽文事件|林爽文事件]]時，閩浙總督常青駐節在[[後甲關帝廳|後甲關帝廳]]，而乾隆五十二年（1787年）有林爽文勢力從南潭、蔦松等處前來，清軍乃前往「二王宮廟」與之交戰{{r|石}}{{r|神明傳說}}{{r|戴}}。據說為避免二王廟被發現是崇祀鄭經，乃在該年（1787年）聽從土虱堀王志遠的建議，增祀[[關聖帝君|關聖帝君]]（據說分靈自後甲關帝廳）{{r|石}}{{r|神明傳說}}{{r|戴}}。咸豐二年（1852年），董事陳邦基等人重修，咸豐七年（1857年）落成，立有〈重修二王廟碑記〉二石碑為記{{r|石}}{{r|戴}}。

到了日治時期，二王廟在明治卅七年（1904年）於董願、張媽喜、謝義、沈海、陳可、戴東等信徒募款下再次重修{{r|相}}{{r|石}}{{r|神明傳說}}{{r|戴}}。大正五年（1916年），林番薯、李弄、陳港、林裕、孫喜、董丁進、陳標基等人再次修廟{{r|石}}{{r|神明傳說}}{{r|戴}}。在[[皇民化運動|皇民化運動]]期間，因故荒廢{{NoteTag|皇民化運動期間，日本人曾在二王廟舉行「諸神昇天」的儀式焚燒神像，其中包括[[鹽行天后宮|鹽行天后宮]]的大媽<ref>{{cite web|url=http://www.yongkangcity.gov.tw/?menu=travel3_2&notice_id=11 |title=鹽行天后宮簡介 |publisher=臺南市永康區公所 |accessdate=2016-06-13 }}</ref>。}}{{r|石}}{{r|神明傳說}}{{r|戴}}。

二次大戰後，甲內人李案子{{NoteTag|李案子為李文奇（康熙年間倡建者）之後代{{r|神明傳說}}，曾任永康庄警防團長、永康鄉民代表、網寮村村長{{r|戴}}。}}倡修，將二王廟從二王崙遷到甲內聚落內重建{{r|石}}{{r|神明傳說}}{{r|戴}}。工程於民國卅八年（1949年）9月24日{{NoteTag|農曆八月三日{{r|戴}}。}}動土，11月18日{{NoteTag|農曆九月廿八日{{r|戴}}。}}上樑{{r|戴}}。民國四十三年（1954年）完工後，民國四十六年（1957年）舉行祈安建醮大典{{r|戴}}{{r|石}}。之後在民國五十七年（1968年）、七十七年（1988年）又有重修{{r|祀神故事}}。

== 祀神 ==
二王廟除了供奉鄭府二王爺（鄭府二千歲）與關聖帝君外，還供奉有開臺聖王（大王）、鄭府三千歲（三王）{{NoteTag|其身分眾說紛紜，《南瀛祀神故事誌》寫說是二王爺的結拜兄弟{{r|戴}}{{r|祀神故事}}。}}、清水祖師、普庵祖師、三彭祖師、天上聖母、觀音菩薩、田都元帥、中壇元帥、關平太子、周倉將軍、註生娘娘、福德正神、謝范二將軍、八家將大神與虎爺{{r|戴}}。

=== 二王 ===
根據《臺灣省臺南縣市寺廟大觀》（1963年）的記載，二王爺的身分有三種說法{{r|戴}}。一是鄭經，據說他在1681年騎馬到甲內附近時，突然暈厥墜馬，後來病逝，後人遂建祠奉祀稱二王爺{{r|戴}}。二是鄭成功之弟，傳說他隨鄭成功來臺時，因嘉義地區有賊匪生事，乃前往討伐，追擊到二王崙時落馬而死，後人乃建廟供奉{{r|相}}{{r|戴}}。三是認為二王爺其實是兩個人，分別是鄭成功四子鄭睿與十子鄭發，兩人來臺後未娶妻就去世（現存其墓「[[藩府二鄭公子墓|藩府二鄭公子墓]]」），之後顯靈保佑民眾，遂建廟奉祀{{r|戴}}。今多採鄭經之說{{NoteTag|相對鄭經的「二王」之稱，石萬壽〈王爺信仰與延平王君臣關係之探討〉一文提到[[鄭成功祖廟|鄭成功祖廟]]原稱「先王廟」，有了鄭經的二王廟後，乃改稱「大王廟」{{r|戴}}。但戴文鋒《永康的歷史遺跡與民間信仰文化》指出此說未見於文獻，乃石萬壽教授於1970年代採風遺老所得{{r|戴}}。}}。

此外也有說法認為「二王」其實是指武王鄭成功與文王鄭經的合稱{{NoteTag|[[夏琳|夏琳]]《[[閩海紀要|閩海紀要]]》一書提到鄭克塽在永曆卅五年（1681年）拜表請追謚，「謚先藩為文王，追謚賜姓延平王為武王」。}}，蔡相煇〈二王廟與鄭成功父子陵寢〉（1984年）一文便提到鄭經死後與鄭成功合葬，故稱二王{{r|戴}}。戴文鋒《永康的歷史遺跡與民間信仰文化》從此說，認為二王廟原本是主祀「文、武二王」之廟，後人誤以為「二王」是指排序，所以主祀神變成只剩鄭經{{r|戴}}。

== 宗教活動 ==
二王廟的關聖帝君壽誕是在農曆五月十三日，二王爺壽誕則是在五月十五日{{r|戴}}。在二王爺壽誕前，依二王爺旨意繪在壽誕三天前舉行「三日遊」或「一日遊」的夜巡遶境，且因兩尊主神壽誕接近，通常會一起出轎遶境{{r|戴}}。1980年代之後改成三年一巡（日巡），2010年才依神明旨意恢復夜巡{{r|戴}}。

== 傳說 ==
傳說在清朝末年時，二王廟所在的二王崙周邊都是大榕樹，要到甲內只能從二王廟旁的小徑（相當於今之永二街）進出{{r|神明傳說}}。當時有一群盜賊橫行，且會故意放出預告給要搶劫的聚落{{r|神明傳說}}。甲內的居民某日得知三天後將遭盜賊侵襲，因治安敗壞無法寄望於官府，只得向二王爺祈求保佑{{r|神明傳說}}。而就在盜賊預告的當天，這群盜賊到了二王崙之後卻找不到路，傳說這是二王爺顯靈，讓周遭的榕樹枝條擋住小徑，也包圍住甲內，使盜賊鎩羽而歸{{r|神明傳說}}。

== 其他 ==
過去二王廟舊址旁邊有「菜豆市」，市場邊有個小販賣的粽子很有名{{r|周}}。據說二次大戰時要到南洋當軍伕的人會在菜豆市集合，出發前每人發給一顆肉粽{{r|周}}。當地有諺語「送啊送，送去二王吃肉粽」，一說指的是被拉去從軍的事{{r|周}}，另一說是指送去墳場<ref>{{cite web|url=http://www.yongkangcity.gov.tw/?menu=travel2_2&notice_id=20 |title=風俗民情 |publisher=永康區公所 |accessdate=2016-06-11 }}</ref>。
