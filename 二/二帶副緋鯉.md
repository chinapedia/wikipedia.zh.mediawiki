'''二帶副緋鯉'''（[[学名|学名]]：{{lang|la|''Parupeneus bifasciatus''}}），又稱'''三帶副緋鯉'''，俗名'''秋姑、鬚哥'''，為[[輻鰭魚綱|輻鰭魚綱]][[鱸形目|鱸形目]][[鬚鯛科|鬚鯛科]]的一個[[種|種]]。

== 分布 ==
本魚分布於[[印度|印度]][[太平洋|太平洋]]區，包括[[東非|東非]]、[[紅海|紅海]]、[[波斯灣|波斯灣]]、[[亞丁灣|亞丁灣]]、[[馬爾地夫|馬爾地夫]]、[[葛摩|葛摩]]、[[模里西斯|模里西斯]]、[[塞席爾群島|塞席爾群島]]、[[印度|印度]]、[[斯里蘭卡|斯里蘭卡]]、[[馬來西亞|馬來西亞]]、[[安達曼海|安達曼海]]、[[泰國|泰國]]、[[菲律賓|菲律賓]]、[[印尼|印尼]]、[[中國|中國]]的[[南海|南海]]、[[東海|東海]]和[[台灣|台灣]]、[[日本|日本]]、[[越南|越南]]、[[新幾內亞|新幾內亞]]、[[所羅門群島|所羅門群島]]、[[澳洲|澳洲]]、[[馬里亞納群島|馬里亞納群島]]、[[馬紹爾群島|馬紹爾群島]]、[[密克羅尼西亞|密克羅尼西亞]]、[[帛琉|帛琉]]、[[諾魯|諾魯]]、[[斐濟群島|斐濟群島]]、[[東加|東加]]、[[吉里巴斯|吉里巴斯]]、[[吐瓦魯|吐瓦魯]]、[[萬納杜|萬納杜]]、[[法屬波里尼西亞|法屬波里尼西亞]]、[[墨西哥|墨西哥]]、[[加拉巴哥群島|加拉巴哥群島]]、[[厄瓜多|厄瓜多]]等海域。属于热带和亚热带近岸底层鱼类。<ref name='id7576'/>

== 深度 ==
水深1~80公尺。

== 特徵 ==
本魚體為黃色，腹部顏色較淡，體側具3條黑色寬橫帶，橫帶皆只達體側中央，它們分別在第一背鰭、第二背鰭下方及尾柄處。另其眼眶四周顏色較深，觸鬚黑褐色，尾鰭分叉，上下葉寬，背鰭兩個。背鰭硬棘共8枚、軟條共9枚；臀鰭硬棘1枚、軟條7枚。體長可達35公分。

== 生態 ==
本魚常可在礁湖或向海礁坡發現單獨或三五成群地活動。幼魚較常在礁台上游走，而成魚則在礁石外緣活動，屬肉食性，以[[多毛類|多毛類]]、[[甲殼類|甲殼類]]為食。

== 經濟利用 ==
食用魚，適合煎食或煮[[薑|薑]]絲[[清湯|清湯]]，小個體可做為觀賞魚。
