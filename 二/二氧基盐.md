'''二氧基盐'''指含有'''二氧基阳离子'''（O<sub>2</sub><sup>+</sup>）的一类不多见的化合物，氧的[[氧化态|氧化态]]为+1/2。该离子由[[氧气|氧气]]失电子得到：
:O<sub>2</sub> → O<sub>2</sub><sup>+</sup> + e<sup>−</sup>
以上的反应需要很大的能量，约为1165kJ/mol，即氧气分子的[[电离能|电离能]]。

== 结构 ==
O<sub>2</sub><sup>+</sup>离子呈[[顺磁性|顺磁性]]，与[[一氧化氮|一氧化氮]]的价电子数相等，其中O-O键[[键级|键级]]为2.5，键长为112.3pm，键能为625.1kJ/mol<sup>1</sup>，伸缩频率为1858[[波数|cm<sup>−1</sup>]]。

== 制备 ==
六氟合铂(V)酸二氧基（O<sub>2</sub><sup>+</sup>PtF<sub>6</sub><sup>−</sup>）是第一个制得二氧基盐，是用强氧化性的[[六氟化铂|六氟化铂]]（PtF<sub>6</sub>）氧化氧气得到的：
:O<sub>2</sub> + PtF<sub>6</sub> → O<sub>2</sub>[PtF<sub>6</sub>]

O<sub>2</sub><sup>+</sup>PtF<sub>6</sub><sup>−</sup>的制得直接推动了[[稀有气体化合物|稀有气体化合物]]的发展。鉴于Xe与O<sub>2</sub>第一电离能相近，因此科学家尝试用六氟化铂氧化氙，并从此制得了第一个稀有气体化合物[[六氟合铂酸氙|六氟合铂酸氙]]。

二氧基盐中的阴离子还可以是MF<sub>6</sub><sup>−</sup>（M = P、As、Sb）、SnF<sub>6</sub><sup>2−</sup>和[[氟硼酸盐|BF<sub>4</sub><sup>−</sup>]]。制备方法还有：
*高温或日光下用氧和氟的混合物处理，如：
:2O<sub>2</sub> + F<sub>2</sub> + 2 [[五氟化砷|AsF<sub>5</sub>]] → 2 O<sub>2</sub><sup>+</sup>AsF<sub>6</sub><sup>−</sup>
*高于423K用[[二氟化氧|二氟化氧]]处理：
:4 OF<sub>2</sub> + 2 AsF<sub>5</sub> → 2 O<sub>2</sub><sup>+</sup>AsF<sub>6</sub><sup>−</sup> + 3 F<sub>2</sub>
*约83K用[[二氟化二氧|二氟化二氧]]处理：
:2 O<sub>2</sub>F<sub>2</sub> + 2 [[三氟化硼|BF<sub>3</sub>]] → 2 O<sub>2</sub><sup>+</sup>BF<sub>4</sub><sup>−</sup> + F<sub>2</sub>
:2 O<sub>2</sub>F<sub>2</sub> + 2 [[五氟化磷|PF<sub>5</sub>]] → 2 O<sub>2</sub>PF<sub>6</sub> + F<sub>2</sub>

这些物质在室温下迅速分解：
:2 O<sub>2</sub>BF<sub>4</sub> → 2 O<sub>2</sub> + F<sub>2</sub> + 2 BF<sub>3</sub>
:2 O<sub>2</sub>PF<sub>6</sub> → 2 O<sub>2</sub> + F<sub>2</sub> + 2 PF<sub>5</sub>
