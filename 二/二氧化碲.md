'''二氧化碲'''化学式TeO<sub>2</sub>，是一种白色固体，加热变黄。其有两种晶型，无色四面体结晶的副黄碲矿是[[四方晶系|四方晶系]]的α-TeO<sub>2</sub>，黄色的单斜矿石黄碲矿是β-TeO<sub>2</sub>。<ref name = "Greenwood">{{Greenwood&Earnshaw1st|page=911}}</ref>对于二氧化碲性质的研究多基于α-TeO<sub>2</sub>。<ref name = "McWhinnie"> W.R.McWhinnie (1995) ''Tellurium - Inorganic chemistry''  Encyclopedia of Inorganic Chemistry Ed. R. Bruce King (1994) John Wiley & Sons ISBN 978-0-471-93620-6 </ref>

==制备==
二氧化碲可由元素直接化合得到：<ref name = "Greenwood"/>

:Te + O<sub>2</sub> → TeO<sub>2</sub>

或由[[亚碲酸|亚碲酸]]H<sub>2</sub>TeO<sub>3</sub>热分解，或碱性[[硝酸盐|硝酸盐]]Te<sub>2</sub>O<sub>4</sub>·HNO<sub>3</sub>在400°C以上热分解制得。<ref name = "Greenwood"/>

==性质==
二氧化碲于732 °C时熔化，形成红色液体。

二氧化碲因其[[两性|两性]]溶于酸碱，微溶于水，在pH=4.0时[[溶解度|溶解度]]最小。<ref name = "Greenwood"/>TeO<sub>2</sub>可被强氧化剂氧化为碲酸或碲酸盐。
==应用==
二氧化碲应用于声光器件。由于α-TeO<sub>2</sub>具有双折射性和旋光特性，旋光度为87°/mm，沿＜110＞方向传播的声速仅为616m/s，所以是一种高品质的声光材料。二氧化碲用于制造特殊性能的玻璃。碲玻璃具有高折射率，能穿透中红外波段的电磁波。<ref>
{{cite journal |author=Stegeman R, Jankovic L, Kim H, Rivero C, Stegeman G, Richardson K, Delfyett P, Guo Y, Schulte A, Cardinal T |title=Tellurite glasses with peak absolute Raman gain coefficients up to 30 times that of fused silica |journal=Optics Letters |volume=28 |issue=13 |pages=1126–8 |year=2003 |pmid=12879929 |doi=10.1364/OL.28.001126 |url=}}</ref>
