'''二字頭'''（{{lang-en|'''The 20s'''}}）是一個由[[香港|香港]]年輕人發行、非牟利的[[雜誌|雜誌]]，為一[[月刊|月刊]]並於每月首日出版，其目標為幫助[[80後|80後]]發聲，創刊於2011年7月1日（星期五）並於2011年[[七一遊行|七一遊行]]首次派發。

== 簡介 ==
二字頭為一本由一些[[香港高登|香港高登]]的參與者發起的雜誌，每月首日出版一期，全書十多頁不等。值得一提的是，該雜誌不供出售，目前亦無法在[[便利店|便利店]]或各[[報社|報社]]中取得，只有兩個定點派發的地方<ref>{{cite web |url=http://20s.hk/distribute.php |title=存档副本 |accessdate=2011-07-10 |deadurl=yes |archiveurl=https://web.archive.org/web/20110701125900/http://20s.hk/distribute.php |archivedate=2011-07-01 }}</ref>，以及會在全港各地不定點不定時招募[[義工|義工]]派發<ref>http://www.facebook.com/the20s</ref>；未能取得該雜誌的人士，可以在其官方網站中閱讀或下載[[PDF|PDF]]檔案，或於其官方網頁中訂閱[[電子報|電子報]]。創刊當日此雜誌獲香港傳媒[[蘋果日報|蘋果日報]]報導。

該雜誌的編採隊伍有[[社民連|社民連]]的幹事，也有[[民建聯|民建聯]]的支持者，該雜誌希望能夠平衡各方面的意見<ref>{{cite news|url=http://www1.hk.apple.nextmedia.com/template/apple/art_main.php?iss_id=20110701&sec_id=4104&art_id=15392411|title=隔牆有耳：去遊行有雜誌攞|newspaper=蘋果日報|date=2011-07-01|accessdate=2011-07-10}}</ref>。

目前該雜誌接受任何人士投稿<ref>{{cite web |url=http://20s.hk/guideline.php |title=存档副本 |accessdate=2011-07-10 |deadurl=yes |archiveurl=https://web.archive.org/web/20110701112420/http://20s.hk/guideline.php |archivedate=2011-07-01 }}</ref>，投稿包括圖片或文字。
=== 創刊號 ===
首期創刊號於2011年7月1日開始派發，該期共有16頁，9篇文章，印刷了約5000份，尾頁刊登了發起此[[雜誌|雜誌]]之地方（[[香港高登|香港高登]]討論區）的全頁[[廣告|廣告]]<ref>{{cite book|title=二字頭|pages=16|url=http://20s.hk/publications/1/2011-07-01.pdf|accessdate=2011-07-10|language=zh-hk|date=2011-07-01}}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

首期主要向[[七一遊行|七一遊行]]參與者、各大[[行人專用區|行人專用區]]及[[屯門區|屯門區]]的青年派發，詳細派發地如下：
* [[銅鑼灣|銅鑼灣]]（七一遊行隊伍）
* [[旺角|旺角]]行人專用區
* [[銅鑼灣|銅鑼灣]]行人專用區
* [[市中心站|市中心站]]（附近[[行人天橋|行人天橋]]）
