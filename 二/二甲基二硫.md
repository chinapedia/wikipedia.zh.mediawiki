'''二甲基二硫'''是一种有机[[硫|硫]][[化合物|化合物]]。

== 存在 ==
是白鬼笔气味组分之一<ref>[http://www.pilzepilze.de/tstijve1.html Geruchs- und Farbstoffe der Rutenpilze]</ref>，也是牛奶见光变质时散发气味的成分之一（甲硫醛的降解产物）。<ref>[http://www.chemiedidaktik.uni-wuppertal.de/alte_seite_du/material/milch/milcheiweiss/lichtgeschmack.pdf Entstehung des Lichtgeschmacks in Milch] {{webarchive|url=https://web.archive.org/web/20090617083719/http://www.chemiedidaktik.uni-wuppertal.de/alte_seite_du/material/milch/milcheiweiss/lichtgeschmack.pdf |date=2009-06-17 }}</ref>

== 制取 ==
可由甲硫醇与硫磺反应而得。
::<math>\mathrm{2 \ CH_3SH + S \rightarrow CH_3S{-}SCH_3 + H_2S}</math>

== 性质 ==
可燃、无色至黄色液体，有恶臭。不溶于水，可与乙醇、乙醚、醋酸混溶。嗅觉阈值6ppm。<ref>[http://www.enius.de/schadstoffe/dimethyldisulfid.html Dimethyldisulfid (Enius)]</ref>蒸气与空气形成爆炸性混合物。

== 应用 ==
用作溶剂、农药中间体、催化剂的钝化剂、结焦抑制剂等。
