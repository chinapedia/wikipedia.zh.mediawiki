'''二胡'''是一种[[拉弦樂器|拉弦樂器]]。有兩根[[弦|弦]]，現代二胡以[[純五度|純五度]][[定弦|定弦]]，在[[胡琴|胡琴]]之中屬於中高[[音域|音域]]樂器。现在各地方根據需要的條件，獨立發展出不同形状的[[弓弦樂器|弓弦樂器]]，如[[高胡|高胡]]、[[京胡|京胡]]、[[板胡|板胡]]、[[二弦|二弦]]、[[喇叭弦|喇叭弦]]等。為滿足[[中樂交響化|中樂交響化]]的需要，根據二胡的制式，也另外發明[[中胡|中胡]]等。

製琴師通常以木材製作二胡，如[[薔薇木|紅木]]、[[小葉紫檀|小葉紫檀]]、[[黑檀|烏木]]、[[杞梓木|雞翅木]]、[[玫瑰木|血檀]]、[[黑黃檀|大葉紫檀]]、[[雙雄蘇木|非洲紅花梨]]、[[雙雄蘇木|非洲紫檀]]等，也有製琴師以[[象牙|象牙]]、玉石、塑膠等材料製作二胡，因為他們有[[聲學|聲學]]以外的考量。琴筒一端蒙以蛇皮，這是二胡獨特音色的來源，一些高級的二胡则采用[[蟒蛇|蟒蛇]]皮。不論是來自野生或被圈養的蛇，收集蛇皮的成本日益增高。而[[瀕危野生動植物種國際貿易公約|瀕危野生動植物種國際貿易公約]]限制其參與團體對蠎蛇皮的進出口，近年有人便開發蛇皮的替代材料，以期模仿蛇皮的音色。通常使用鋼弦、尼龍弦、絲弦，[[羊腸弦|羊腸弦]]已極少人使用。总结来说，这导致了用原材料制作的二胡也随着减少了，取而代之的是替代材料。

[[中国|中国]]為二胡的主要生產地，其中[[上海市|上海]]、[[北京市|北京]]及[[苏州市|蘇州]]最为频繁。不同的地方生产的二胡音色各有不同及差异：傳統上，上海生產的二胡，音色較柔美；北京出產的胡，音色較高亢；蘇州出產的二胡，音色则揉合上海與北京的特色。[[File:江河水.ogg|File:江河水.ogg]]

== 歷史 ==
[[唐朝|唐朝]]時稱二胡為“奚琴”，是居住[[中国东北地区|中国东北地区]]的[[奚族|奚人]]乐器。

[[宋朝|宋代]]音乐理论家[[陈旸|陈旸]]于《乐书》中记载：“奚琴本胡乐也，出于弦鼗而形亦类焉，奚部所好之乐也。盖其制，两弦间以竹片轧之，至今民间用焉。” 奚琴发源于中国东北，和来自西北的[[胡琴|胡琴]]有很大区别， 它只有两条[[弦_(樂器)|琴弦]]，轧时竹片在两弦之间，不在两弦之上。

欧阳修也在《试院闻奚琴作》写到 “奚琴本出胡人乐，奚奴弹之双泪落”。此时奚琴尚未成为主流演奏乐器。

在[[元朝|元代]]，奚琴开始在宴乐中用于独奏或合奏，并出现在用于军队的演奏活动中。《马可·波罗游记》中曾记载：“鞑靼人又有一种风俗。当他们队伍排好，等待打仗的时候，他们唱歌合奏他们的二弦琴，极其好听。”这二弦琴描述的就是二胡。

[[明朝|明代]]时，二胡开始变成了一种成熟的弓弦乐器，《元史·礼乐志》记载 “胡琴，制如火不思，卷颈，龙首，二弦，用弓捩之弓之弦以马尾”。此时的二胡在[[奚琴|奚琴]] 基础上加上了围定弦长的千斤，与今形制已大致相同。

现朝鲜半岛仍称二胡为[[奚琴|奚琴]]。

[[五四運動|五四運動]]時期，[[劉天華|劉天華]]融合歐洲的音樂理論、演奏技巧，根據中國[[民間音樂|民間音樂]]特性及借鏡[[小提琴|小提琴]]，將二胡定弦為d、a，並對二胡的演奏技巧進行革新，擴展二胡的表現力，並且為二胡創作多首獨奏曲。

1920年代，[[周少梅|周少梅]]、[[周榮根|周榮根]]、[[陶洪茂|陶洪茂]]合作改良二胡：加長琴杆至90公分、加大琴筒、改用莾蛇皮、改用[[軟弓|軟弓]]、改用更粗的[[琴弦|琴弦]]，使[[音量|音量]]、[[音色|音色]]有所改善。

1930年代，[[蔣風之|蔣風之]]設計出「六角啞鈴式」二胡，又被稱為「蔣氏南胡」。[[周榮庭|周榮庭]]與蔣風之、[[陸修堂|陸修堂]]等共同設計出「洋山柄頭」二胡和「迴紋頭」二胡。

1950年代至1960年代間，[[上海民族樂團|上海民族樂團]]、[[中國廣播民族樂團|中國廣播民族樂團]]、[[中央民族樂團|中央民族樂團]]等模仿[[交響樂團|交響樂團]]的職業「[[民族樂團|民族樂團]]」出現，增加對二胡專業的需求。

1963年，首屆全國二胡比賽於第4屆[[上海之春音樂節|上海之春音樂節]]舉辦。

1970年代，[[湯良德|湯良德]]研製出前八角後圓（方圓）二胡和可調節的半月形音窗。1975年，[[王國潼|王國潼]]與[[滿瑞興|滿瑞興]]制訂出「扁八」二胡的尺寸為：琴筒長133毫米，前口縱向直徑86毫米，横向直徑92毫米，後口内徑84毫米；琴筒中加裝竹筒以改良高音音色。

1970年代，王國潼與滿瑞興開始研發[[二泉琴|二泉琴]]，1980年代[[閔惠芬|閔惠芬]]與[[王瑞泉|王瑞泉]]、[[沈德慶|沈德慶]]研製成功。

現代二胡作品出現，有《[[豫北敘事曲|豫北敘事曲]]》(1959年)、《[[三門峽暢想曲|三門峽暢想曲]]》(1960年)、二胡協奏曲《[[長城隨想|長城隨想]]》(1982年)等。

1980年代中後期，已出版的[[二胡獨奏曲|二胡獨奏曲]]達數百首，並開始發展[[敘事曲|敘事曲]]、[[協奏曲|協奏曲]]等大型二胡曲。

[[臺北市民族器樂大賽|臺北市民族器樂大賽]]、[[中央電視台民樂大賽|中央電視台民樂大賽]]等專業比賽相繼舉行。

== 結構 ==
二胡包括[[琴身|琴身]]（包括琴桿、琴筒、琴托、皮膜、音窗）、[[弦_(樂器)|琴弦]]系统（包括琴弦、弦軸、千斤、琴碼、音垫）和[[琴弓|弓]]（包括弓魚、弓桿、弓毛）。

=== 琴身 ===

[[File:Erhu_snake_skin.png|thumb]]
[[File:Erhu-8-sided-front.png|thumb]]
* 琴桿
*: 立於二胡正中，穿過琴筒、琴托。橫切面從前為圓型，後因使用拉力更大的金屬弦，易令圓形琴桿彎曲變形，遂發展出較抗拉力的淚滴形琴桿。琴桿最上端為'''琴頭'''，較普遍的式樣為流線型向音窗方向彎曲，末端張濶，飾以[[象牙|象牙]]片或骨片，也有俗稱「[[雲頭|雲頭]]」的迴紋形、俗稱「龍頭」的[[囚牛|囚牛]]型及其他造型，某些琴頭的形狀可反映二胡的生產地。琴頭以下為弦軸孔，開弦軸孔的一段琴桿為方柱體。琴桿下端為一方形榫頭，比琴桿細，下接琴托榫孔。
* 琴筒
*: 為二胡的共鳴體，由多塊木片以[[花膠|花膠]]膠合 併而成，有多種外形，其外形通常與內腔相同或相似，可影響音色及音量：
*:* 圓形
*:* 六角形
*:* 八角形
*:* 前八角后圆（皮膜為八角形，出音口為圓形，又稱方圓）
*:* 扁圓形，[[陳耀星|陳耀星]]設計
*:* 扁前八角後圓（又稱扁八）
* 琴托
*: 位於琴筒下方。
* 皮膜
*: 二胡琴筒前口部分，又叫琴皮，或称琴膜，是二胡重要的发声装置组成部分。中、高档二胡琴多采用[[蟒蛇皮|蟒皮]]。传统上采用中国南方和东南亚一带的金花蟒的蟒皮，現代有多種替代材料。
* 音窗
*: 位於琴筒背面，以木片雕刻而成，南方二胡多為全封閉音窗，北方二胡多為半封閉音窗，<ref>{{cite book |author=趙寒陽; 劉愛明 |title=《二胡音樂詞典》 |pages=第342頁 |publisher=藍天出版社 |location=北京 |isbn=9787509404348|date=2010年9月}}</ref>音窗覆蓋的面積比率可影響音量和音色。

=== 琴弦系統 ===
* [[弦_(樂器)|弦]]
*: 張兩弦，早期為絲或[[羊腸弦|羊腸弦]]等，近代引入金屬弦後，最初多用銅弦，現代以鋼絲包尼龍弦為主流，當代開始使用銀、鎳等鋼合金為外層，合成纖維為芯。
* 弦軸
*: 分木軸和機械軸兩種
* 重千斤
*: 二胡的上切音點，分固定千斤和活動千斤兩大類
* 琴碼
*: 將弦的震動傳導至皮膜，以楓木，烏木，紫檀，紅木，松節，竹，紙等製作，形狀上尖下寬，有橢圓形底，圓形底，方底、橋式等，物料及形狀皆影響[[音色|音色]]及[[音質|音質]]。
* 音塾
*: 在琴码下方、被夾於弦與皮膜之間作為[[阻尼|阻尼]]，取柔軟和有彈性的物料，如布料、毛皮、乳膠等，可稳定音质和减少[[狼音|狼音]]。而阻慢蛇皮音色的熟成，不雅觀等，為不使用音塾的原因。

=== 琴弓 ===
[[File:Erhu-bow-hair-between-strings.png|thumb]]
二胡弓的標準長度為816mm，其等級以竹材的直徑區分，高級弓的直徑為7±0.5mm，中級弓直徑為7±1mm，普級弓的直徑為6±2mm。<ref>中華人民共和國[[國家輕工業部|國家輕工業部]]，1965年</ref>規範簡單明瞭卻無法作為製造和選擇的依據，現代的製弓師會製作更長更粗的弓以配合現代人的演奏習慣，也有依照演奏者的要求去製作。

弓的重量、彈性與硬度，會對演奏產生不同效果，[[二胡演奏家|二胡演奏家]]常依照樂曲選擇不同的弓。

* 弓毛
*: 弓毛用馬尾製成，以白馬尾為佳。
* 弓桿
*: 由富彈性的竹所製
* 弓魚
*: 馬尾庫，於弓根（手握琴弓的一端）固定、調節弓毛的裝置，因北方琴弓上的馬尾庫形似一尾魚而得名，甚至有製弓師雕刻出眼、鰓、鱗為飾，而南方琴弓上的馬尾庫根本不像魚也被稱為弓魚，由白玉、象牙、木、塑膠所製。

== 蟒皮 ==

1988年，中國就[[聯合國|聯合國]][[瀕危野生動植物種國際貿易公約|瀕危野生動植物種國際貿易公約]]（CITES）通過保護[[瀕危物種|瀕危物種]]的法例後，使無牌使用和貿易蟒蛇成為非法活動。<ref>{{cite web |url=http://www.taipeitimes.com/News/feat/archives/2005/09/13/2003271531 |title=Erhus sing as snakes gently weep |author=AFP, Beijing |date=13 September 2005 |work= |publisher=Taipeitimes.com |accessdate=14 September 2010}}</ref>為規範使用蟒蛇皮，中國的[[國家林業局|國家林業局]]引入[[東南亞|東南亞]]的蟒蛇皮賣家和中國的樂器製造商之間的認證計劃。 2005年1月1日，新規定要求二胡有來自國家林業局發出的證明書，以證明二胡的蟒蛇皮不是來自野生蟒蛇，而是來自農場飼養的蟒蛇。並容許個人因需要可帶2支二胡出中國，商業買家需要額外的出口證書。

中國以外，二胡製造商可自行發出經各自國家政府批准的CITES許可證。這樣的出口是合法的，因為他們已經從合法的來源取得蠎皮。

[[香港中樂團|香港中樂團]]研究替代蟒蛇皮並於2005年設計了「生態胡琴系列」，以[[聚對苯二甲酸乙二酯|PET]][[聚酯|聚酯]]膜替代蟒蛇皮。香港中國樂團聲稱生態胡琴系列不僅大大降低對野生蟒蛇的壓力，同時也增強了儀器的耐用性和音色。{{Citation needed}}

== 定弦 ==
[[胡琴|胡琴]]作為傳統戲曲之伴奏樂器時，有[[純四度|純四度]]定弦，如[[二弦|二弦]]，亦有[[純五度|純五度]]定弦，如[[板胡|板胡]]。而二胡有「主音胡琴」和「托音胡琴」之分，主音胡琴定內弦為d、外弦為a，[[周少梅|周少梅]]和[[劉天華|劉天華]]均使用主音二胡；托音胡琴定內弦為G、外弦為d，被採用於傳統[[江南絲竹|江南絲竹]]，[[華彥鈞|華彥鈞]]三首傳世二胡曲為托音胡琴定弦。

== 演奏技法 ==

左手持琴按弦，右手拉弓，弓毛被置於雙弦之中拉奏。演奏時多採坐姿，於巡遊或應演出需要，可加裝掛鈎，掛於腰間站奏。
