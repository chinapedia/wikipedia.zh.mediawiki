'''普意雅'''（{{bd|1862年||1930年|9月}}）[[法国|法国]]人，曾任中国[[平汉铁路|平汉铁路]]全路总工程师，也是摄影家。

==生平==
普意雅1862年生于法国，是法国国立中央工艺学院工程师。1898年受[[清朝|清朝]]政府聘请，来中国测绘中国铁路沿线地图，后任平汉铁路北段总工程师。1906年升任平汉铁路全路总工程师，任至1927年卸任<ref name=wy>{{Cite web |url =http://money.163.com/14/0923/16/A6REAMQI00254TI5_all.html  |title =档案解密：北京朝内81号身世之谜  |author =  |date =2014-09-23  |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref><ref name=liu>{{Cite web |url = http://www.nlc.gov.cn/hxjy2008/2009/0303/article_67.htm |title =普意雅先生及其收藏的近代北京照片  |author = 刘忠民 |date =  |publisher =  北京纪事2002年第5期|language =  |accessdate =  |quote =  }}</ref>。

1917年5月，作为中国政府顾问，普意雅在给北京都市营造局呈拟的意见书中，阐述了京都市政待办的大工程，其中一些建议获得采纳。1917年7月汛期，普意雅在勘测京汉铁路沿线期间，撰写了《水灾善后问题》，由同事[[华南圭|华南圭]]译为中文在报刊上发表<ref name=wy/>。

普意雅曾编绘《中华民国国有铁路沿线地图》，其中有多幅北京地图、北京四郊地图。普意雅还绘制过北京[[东交民巷|东交民巷]]使馆分布图、[[京师华商电灯股份有限公司|京师华商电灯股份有限公司]]拟从[[西便门|西便门]]到[[石景山|石景山]]新厂接修线杆图<ref name=wy/><ref name=liu/>。1927年辞去平汉铁路全路总工程师职务后，又在1928年绘制出[[京兆地方|京兆地方]]分县图、新测实用北平都市全图、北平附近地图等等<ref name=wy/>。

普意雅收藏有同时期以北京为主、以平汉铁路沿线地区为次的地方社会生活照片。这批照片大部分拍摄于1900年至1910年间，少部分为此后至1930年代以前拍摄<ref name=wy/><ref name=liu/>。

1930年9月，普意雅在[[北平|北平]]（今北京）病逝，享年68岁<ref name=liu/><ref name=wy/>。

普意雅逝世后，其遗孀朱德蓉于1932年10月将其拍摄和收藏的上述照片资料及其他文图资料捐赠给国立北平图书馆（即今[[中国国家图书馆|中国国家图书馆]]）。国立北平图书馆为此举办接受赠书仪式，[[袁同礼|袁同礼]]馆长代表国立北平图书馆接受赠书。1933年3月，国立北平图书馆举办普意雅所赠图书展示会，袁同礼馆长及[[胡适|胡适]]、[[翁文灏|翁文灏]]、朱德蓉、法国驻华公使等三百多人出席<ref name=liu/><ref name=wy/>。
