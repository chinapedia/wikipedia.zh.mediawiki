'''普利亚'''（{{lang-it|Puglia}}，{{lang-la|Apulia}}），也译作'''阿普利亚'''<ref>[http://it.mofcom.gov.cn/article/ztdy/201605/20160501327915.shtml 普利亚大区介绍--中华人民共和国驻意大利共和国大使馆经济商务参赞处]</ref>，是[[意大利|意大利]]南部的一个大区，东邻[[亚得里亚海|亚得里亚海]]，东南面临[[爱奥尼亚海|爱奥尼亚海]]，南面则邻近奥特朗托海峡及塔兰托湾。该区南部知名的[[萨伦托|萨伦托]]半岛，组成了意大利「皮靴」的一部分。大区面积19,366km²，人口4,023,957人（1991年）。

该区还与其它意大利大区连接，北邻[[莫利塞|莫利塞]]大区，西临[[坎帕尼亚|坎帕尼亚]]大区，及西南面的[[巴斯利卡塔|巴斯利卡塔]]大区。大区隔着[[亚得里亚海|亚得里亚海]]及[[爱奥尼亚海|爱奥尼亚海]]与[[希腊|希腊]]和[[阿尔巴尼亚|阿尔巴尼亚]]相邻。大区扩展至北面远处的莫特—格加奴地区，是[[第二次布匿战争|第二次布匿战争]]中最后一次围城所发生的地方。

== 历史 ==
在古代只有该区北部被称为普利亚；南面则称作[[卡拉布里亚|卡拉布里亚]]，被称呼为是意大利“皮靴”的脚尖。于公元前4世纪被[[古罗马|罗马人]]征服之前，该区是[[古意大利人|古意大利人]]及[[希腊|希腊]]殖民者定居之地方。公元五世纪西罗马帝国灭亡之后，普利亚依次被[[东哥特人|东哥特人]]、[[伦巴第人|伦巴第人]]及[[拜占庭帝国|拜占庭帝国]]占据。

于11世纪，普利亚被[[诺曼人|诺曼人]]征服；[[罗拔·古伊斯格|罗拔·古伊斯格]]（Robert Guiscard）于1059年建立了普利亚公国。于11世纪末诺曼人征服了[[西西里|西西里]]之后，[[巴勒莫|巴勒莫]]取代了美菲（Melfi，现时普利亚西部）成为了诺曼统治权力的中心，之后普利亚仅仅只是起初的[[西西里王国|西西里王国]]，和接着的[[那不勒斯王国|那不勒斯王国]]的一个省分。由12世纪未到13世纪早期，普利亚是都是[[霍亨斯陶芬王朝|霍亨斯陶芬王朝]]的皇帝喜爱的一个居住地点，特别是[[腓特烈二世_(神圣罗马帝国)|腓特烈二世]]。海岸地区后来被[[奥斯曼帝国|奥斯曼帝国]]及[[威尼斯人|威尼斯人]]占领。于1861年，该区加入了[[意大利|意大利]]。

封建制度长期盛行于普利亚的农村地区；社会和农业的革命于19世纪进展缓慢，到20世纪中期进展的步伐才加快。

普利亚地区于11世纪－13世纪时的建筑特色受到[[古希腊|古希腊]]、[[阿拉伯|阿拉伯]]、[[诺曼人|诺曼人]]及[[比萨|比萨]]建筑风格的影响。于大区內，巴里及莱切市都設有大学。

== 地理 ==
[[巴里|巴里]]是普利亚大区的首府，大区分成了数个省分（亦同时是那些省分的首府），包括[[巴里省|巴里省]]、[[布林迪西省|布林迪西省]]、[[福贾|福贾]]省、[[莱切|莱切]]省及[[塔兰托|塔兰托]]省。在2005年决定组成一个新省分巴列塔—安德里亚—特拉尼省（Barletta-Andria-Trani）。大区内主要是平原地带，大区内的低洼海岸则被北方格加奴山区阻断，区内的中北部都有山脉。

== 经济 ==
普利亞在過去很長時間，都主要以農業為主，是全意大利發展比較遲緩地區之一。但自千禧年起，亦開始有一些工業園的設立，包括紡織、鋼鐵、機械、石油等等。意大利石油公司[[埃尼|埃尼]]、汽車配件商Magneti Marelli和航天公司[[芬梅卡尼卡|芬梅卡尼卡]]都有在普利亞設廠。
