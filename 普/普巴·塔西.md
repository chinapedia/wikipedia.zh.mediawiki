'''普巴·塔西·雪巴·曼德瓦'''（{{lang-en|'''Phurba Tashi Sherpa Mendewa'''}}，{{bd|1971年||catIdx=Tashi, Phurba}}），[[雪巴人|雪巴人]]，一位知名的[[聖母峰|聖母峰]][[登山家|登山家]]、同時也是隸屬[[紐西蘭|紐西蘭]]籍登山家[[羅素·布萊斯|羅素·布萊斯]]（[[:en:Russell_Brice|Russell Brice]]）旗下遠征隊[[:en:Himex|Himex]]的領隊（Sirdar）兼嚮導（Guide）<ref name="himex">{{en}}{{cite web |url=http://www.himex.com/english/team.htm |title=Team - climbing sherpas - Sirdar Phurba Tashi |publisher=www.himex.com |accessdate=2011-03-05 |deadurl=yes |archiveurl=https://web.archive.org/web/20100301090412/http://www.himex.com/english/team.htm |archivedate=2010-03-01 }}</ref>。

== 概要 ==
普巴在高海拔強健的體能狀況，以及具備後勤支援的知識、專業的登山技能，因而被號稱「聖母峰[[氂牛|氂牛]]（Everest Yak）」<ref name="himex" />。他曾於2006年、2007年參與[[探索頻道|探索頻道]]《聖母峰：攀越極限》（[[:en:Everest:_Beyond_the_Limit|Everest: Beyond the Limit]]）的拍攝過程<ref>{{en}}{{cite web |url=http://www.yourdiscovery.com/everest/about/team_sherpa.shtml |title=sherpa phurba tashi（sirdar）　|publisher=www.yourdiscovery.com |accessdate=2011-03-05}}</ref>，以及創下17次登頂聖母峰的記錄（更新至2010年為止，單2007年這一季他就登上了3次）<ref name="himex" />，緊追在[[車旺·尼瑪·謝爾帕|車旺·尼瑪·謝爾帕]]（已故）與目前[[金氏世界紀錄|金氏世界紀錄]]保持人[[阿帕·謝爾巴|阿帕·謝爾巴]]之後<ref>{{en}}{{cite web |url=http://www.guinnessworldrecords.com/Search/Details/Most-conquests-of-Mt.-Everest/44791.htm |title=Most conquests of Mt. Everest |publisher=www.guinnessworldrecords.com |accessdate=2011-03-05}}</ref><ref>{{en}}{{cite web |url=http://www.himex.com/english/news/2010-05-07Everest.htm |title=Everest 2010 - Newsletter 25（May 08 2010） |publisher=www.himex.com |accessdate=2011-03-07 |deadurl=yes |archiveurl=https://web.archive.org/web/20100513015902/http://www.himex.com/english/news/2010-05-07Everest.htm |archivedate=2010-05-13 }}</ref>。其他記錄分別為[[卓奧友峰|卓奧友峰]]5次、[[馬納斯盧峰|馬納斯盧峰]]2次、[[阿瑪達布蘭峰|阿瑪達布蘭峰]]與[[希夏邦馬峰|希夏邦馬峰]]1次<ref name="himex" />。

== 聖母峰：攀越極限 ==
=== 2006年 ===
在探索頻道第一季的節目中，普巴曾將雙腿裝上義肢，並成功登頂的的紐西蘭籍登山家[[馬克·英格斯|馬克·英格斯]]（[[:en:Mark_Inglis|Mark Inglis]]）從6553公尺（21,500英尺）的高度背下山去，並安全抵達在6400公尺（21,000英尺）前進基地營（Advanced Base Camp）。馬克起初對普巴要一個人背他下山感到訝異，因為他有54公斤（120磅）重，而且路徑又結冰難走，但是普巴還是辦到了<ref>{{zh}}探索頻道《聖母峰：攀越極限》第一季第6集〈最終代價（The Final Cost）〉（0:31:20 - 0:33:30）</ref>。馬克是世界上第一位登上聖母峰的雙腿截肢者，他曾於2007年受邀來[[台灣|台灣]][[台北市|台北市]][[誠品|誠品]]信義店，分享他的一些故事與感想<ref>{{zh}}{{cite news |url=http://www.nownews.com/2007/05/18/10845-2098576.htm |title=沒有腿照樣成功攀越聖母峰！馬克來台分享他的故事 |publisher=今日新聞網 |date=2007-05-18 |accessdate=2011-03-05}}</ref>。
=== 2007年 ===
在第二季的節目中，遠征隊隊長羅素說服了英籍登山家大衛·塔特（[[:en:David_Tait|David Tait]]）與普巴結伴同行，協助大衛完成世界上首次「南北雙側橫切聖母峰（double traverse of Everest）」的任務。其路線是，先從北側的路線登頂，再從南側下山，休息3天後，再從原路返回。這項任務難度之高在於，兩人皆不曾去過聖母峰南側，南側也尚未重新架設安全索（[[:en:fixed_rope|fixed rope]]），最後還必須穿越危險的[[昆布冰瀑|昆布冰瀑]]（[[:en:Khumbu_Icefall|Khumbu Icefall]]，聖母峰遇難人數最多的地形）<ref>{{zh}}探索頻道《聖母峰：攀越極限》第二季第3集〈汰弱留強（Judgment Day）〉</ref>。當大衛抵達南側基地營後，疲憊的他決定放棄任務。改變他信念的原因是，他認為普巴的能力遠在他之上，如果繼續這樣下去，普巴會將所有的榮譽讓給他（指停下腳步，讓大衛先登頂，雪巴人通常會這麼做）<ref>{{zh}}探索頻道《聖母峰：攀越極限》第二季第4集〈波折連連（World Record）〉</ref><ref>{{en}}{{cite web |url=http://www.discoverychannel.co.uk/everest/about/team_david.shtml |title=david tait（climber） |publisher=www.yourdiscovery.com |accessdate=2011-03-05}}</ref>。

=== 2009年 ===
在第三季的節目中，節目製作小組入駐聖母峰南側的基地營。2009年5月5日，大衛完成了他人生第3次的登頂，採取南側路線，再次與普巴結伴同行（這是他第14次的登頂）<ref>{{en}}{{cite web |url=http://www.everest1953.co.uk/News.php |title=3rd January: 2011 Mission |publisher=www.everest1953.co.uk |accessdate=2011-03-05 |deadurl=yes |archiveurl=https://web.archive.org/web/20110227001739/http://www.everest1953.co.uk/News.php |archivedate=2011-02-27 }}</ref>。

1999年5月27日，普巴完成他人生第1次登頂聖母峰的記錄，當年他才28歲，正為羅素工作。登頂路線是行經北坳的北側路線。與他登頂的有另外兩名雪巴人是Karsang Sherpa、Lobsang Temba，其他三名登頂的客戶是Geoffrey Robb（澳籍男子）、[[:de:Helga_Hengge|Ms. Helga Hengge]]（德美雙重國籍女子）、Kazuhiko Kozuka（日籍男子）<ref>{{en}}{{cite book |editor=Walt Unsworth |title=Everest: the mountaineering history |publisher=The Mountaineers Books |year=2000 |page=頁696 |ISBN=9780898866704}}</ref>。

普巴每年會帶領隊上幾名雪巴人在聖母峰上架設約6公里長的安全索，這項例行任務極度危險，但是全由Himex遠征隊包辦下來<ref>{{zh}}探索頻道《聖母峰：攀越極限》第一季第2集〈守護先鋒（The Gatekeeper）〉</ref>。其他遠征隊則必須以隊上的雪巴協作一人算一百[[美元|美元]]的代價，來支付羅素繩索與人事上的費用<ref>{{en}}{{cite web |url=http://www.alanarnette.com/alan/everest2005.php |title=May 26 2005 - And there off! |publisher=www.alanarnette.com |accessdate=2011-03-07 |deadurl=yes |archiveurl=https://web.archive.org/web/20110314163408/http://www.alanarnette.com/alan/everest2005.php |archivedate=2011-03-14 }}</ref>。普巴與手下完成這項艱鉅的任務後，會順便進行登頂。因此當季若無其他特殊狀況，每年至少會累積1次登頂記錄，之後則必須陪同客戶一起登頂。至於能否取得第2、3次登頂記錄，則必須視羅素的判斷、客戶的狀況與天候因素等<ref>{{en}}{{cite book |editor=Heil, Nick/ McKinley, John (ILT) |title=Dark Summit: The True Story of Everest's Most Controversial Season |publisher=Henry Holt & Co |date=2009-02-03 |ISBN=9780805089912}}</ref>。

== 早期經歷與家庭成員 ==
普巴年輕時，在別的遠征隊當[[廚師|廚師]]，當時羅素已注意到他<ref name="himex" />。隨後他轉任[[挑夫|挑夫]]的工作，等他展現神速般的登山技術後，開始漸漸被旁人注意到<ref name="stuckintherockies">{{en}}{{cite web |url=http://www.stuckintherockies.com/2008/10/dingboche-phurba-tashi-sherpa/ |title=Dingboche & Phurba Tashi Sherpa |publisher=www.stuckintherockies.com |accessdate=2011-03-05}}</ref>。普巴育有三子（其中兩個是雙胞胎）一女，家庭成員還有配偶、母親等<ref>{{en}}{{cite web |url=http://www.himex.com/english/news/2010-05-13Everest.htm |title=Everest 2010 - Newsletter 28（May 13 2010） |publisher=www.himex.com |accessdate=2011-03-05 |deadurl=yes |archiveurl=https://web.archive.org/web/20100519033925/http://www.himex.com/english/news/2010-05-13Everest.htm |archivedate=2010-05-19 }}</ref><ref name="stuckintherockies" />。
