'''普普沙'''是一个塞满了萨尔瓦多咸馅菜的厚玉米饼。 它通常和辣白菜沙拉和[[番茄沙司|番茄沙司]]一起食用。 普普沙在[[洪都拉斯|洪都拉斯]]（尤其是南部地区）是一个较为热门的菜。<ref>{{cite web |title=Pupusa, Wikipedia English |url=https://en.wikipedia.org/wiki/Pupusa}}</ref>

普普沙的玉米饼通常是由玉米粉手工制作的，在铁板上填塞和制作。 常见的馅料包括多种奶酪，猪肉和油豆。
[[File:Pupusas.jpg|缩略图]]

== 词源 ==
普普沙可能是西班牙语中“popotlax”的翻译，这是皮皮语或者[[纳瓦特尔语|纳瓦特尔语]]言中的合成单词，意思是大的填充的tlaxkalli（意为“玉米饼”）。 这个词也有可能来自皮皮语中的“pupusawa”。

== 历史 ==
几个世纪以前，普普沙由居住在现在被称为[[萨尔瓦多|萨尔瓦多]]领土的皮皮族部落首先创建。他们的厨具在“萨尔瓦多的庞贝城”的Joya deCerén村庄中被挖掘了出来，这个村庄曾经被火山爆发带来的灰烬掩埋。这里出土了大约2000年前烹饪好的食物，这些食物保存完好，看起来就跟当时刚制作完成时一模一样。在萨尔瓦多的其他考古遗址也可以找到皮皮族人的厨具。

[[前哥伦布时期|前哥伦布时期]]的普普沙通常是素陷的，并且形状通常呈半月形。它们的陷里通常会加[[西葫蘆|栉瓜]]的花瓣和蓓蕾，还有比如卡西林和莫拉或菌类和盐这样的草药。正如方济会修道士伯纳迪诺·德·萨哈贡（Bernardino De Sahagun）所提及的，直到1570年，肉类才开始被作为普普沙的一种陷料加入到普普沙里面。
在20世纪40年代后期，普普沙的传播大多局限在乡镇中心，如克萨尔特佩克。

这个国家的一些城市。而没有在萨尔瓦多大量传播。
随着[[六十年代|六十年代]]克萨尔特佩克人口开始迁徙到其他地区，各种各样的普普沙开始在全国各地以及如洪都拉斯和危地马拉的邻近地大量传播。在七十年代的危地马拉，普普沙的形状通常是半月形。 然而在查拉特南戈地区，这种半月形的普普沙则会被认为是吃完了剩的，在此地区， 鱼肉陷的普普沙会比较少见。另外，莱波河以东的普普沙通常会更大一些。

20世纪80年代，萨尔瓦多内战迫使当地居民移民到其他国家，他们中大多数都去了美国。因此，在国外萨尔瓦多人聚集的地方，总能看见普普沙的身影。迁移过来的人们已经把这道菜带到了美国的大部分地区。在加拿大的许多地区也会有普普沙的存在。近年来，甚至在[[阿得雷德|阿德莱德]]，澳大利亚的墨尔本和悉尼的一些拉丁美洲餐馆也可以找到这种小吃。
[[File:Typical_Pupuseria_setup_in_El_Salvador.jpg|缩略图]]
2005年4月1日，萨尔瓦多立法议会宣布普普沙为国菜，并且11月的第二个星期日为全国普普沙日。节日当天会在首都和国内几个大城市开展集市贸易活动。 2007年11月10日，为庆祝全国普普沙日，文化部长在首都公园组织了一场博览会，在那里他们制作了世界上最大的普普沙。 这个普普沙直径3.15米，用200磅的马萨，40磅的芝士和40磅的猪肚制成的，最后被大约5000人左右瓜分了。五年后，这个纪录再次被打破，新的普普沙直径为4.25米。记入[[金氏世界紀錄|吉尼斯世界纪录]]的最大普普沙是2015年11月8日在萨尔瓦多奧洛奎爾塔创建的4.5米普普沙。

在2011年，“卫报”（The Guardian）将普普沙选为纽约最佳街头美食。

在国内外，传统的普普沙一般都和库尔蒂多（类似于德国酸菜的辣味版和韩国泡菜的淡味版）还有番茄酱一起食用，并且一般都不用餐具，直接用手抓着吃。

== 变种 ==
萨尔瓦多和洪都拉斯都曾被认为是普普沙的出生地。萨尔瓦多考古学家罗伯托·奥多涅斯（Roberto Ordonez）认为普普沙是由皮乌尔人创造的，这是由于在皮乌尔语中普普沙这个名字具有含义（意为“膨胀”），以及在乔伊亚·德·切伦（Joya de Ceren）中发现了用来做早期普普沙的材料和工具的手工艺品。洪都拉斯的词源学家表示，由于皮乌尔语与纳瓦特语非常接近，洪都拉斯纳瓦族部落也有可能创造了这种食物。在中国 - [[东盟自由贸易区|东盟自由贸易区]]谈判期间，也讨论了普普沙的来源。萨尔瓦多和洪都拉斯两国都都想让普普沙成为其国独一无二的出口商品。但是两天后，洪都拉斯代表团最终将这项出口权利让给了萨尔瓦多。
[[File:Typical_serving_of_Pupusas_in_El_Salvador.jpg|缩略图]]
在萨尔瓦多，当地吃一种普普沙的变种，叫做Pupusa de arroz（米饭普普沙）。当地人使用米粉来制作面饼，然后在里面填入切碎的猪肉，奶酪，豆类，西葫芦和其他蔬菜。这种食物发源自萨尔瓦多东部的Olocuilta镇，但现在已经扩散到全国各地。在阿莱格里亚发现的另一个普普沙的地区版本则叫做Pupusa de banano（香蕉普普沙），顾名思义，这种版本的普普沙会在馅料中加入香蕉。

在美国制造的普普沙通常用玉米面粉 -和马萨混合物制成，而不是新鲜的马萨肉。美国的一些相对高端的普普沙则使用米粉和小麦面粉制作。

在美国新墨西哥州的圣塔菲，当地人发明了包括使用菠菜，意大利辣香肠，奶酪和青椒的各种变种。

同时，普普沙也存在于临近的中美洲国家。 洪都拉斯的普普沙通常使用当地quesillo类型的奶酪来填充。 在哥斯达黎加，普普沙分为两种，分别叫做“萨尔瓦多普普沙”和“普普沙”，而后者是哥斯达黎加的本地版本。 在那里，普普沙是地区嘉年华节日的主要食物。

类似的墨西哥菜被称为gordita（字面意思是“小脂肪”），不同的是gorditas通常只在一边开口。 在[[委内瑞拉|委内瑞拉]]，他们用制作面包的方式制作普普沙（首先将面团煮熟，然后切成两半，最后像汉堡一样将馅料塞进去）。 哥伦比亚永远有自己的面包式普普沙，但与委内瑞拉不同的是，哥伦比亚的普普沙通常在没有填充任何馅料的情况下食用，或者在烹饪前就将填充物放入面团内。

尽管普普沙的市场价格非常便宜，但它却是萨尔瓦多的经济占比很重的一部分。 除普普沙本身之外，它的原材料也被出口到其他国家。在2005年，仅仅是出口美国的普普沙原材料loroco就价值604,408美元。普普沙在美国可以在许多西班牙裔超市和国际超市找到，尤其是在华盛顿特区或者纽约长岛这种萨尔瓦多人人口密集的地方。

普普沙在萨尔瓦多经济中有着很大的分量。 从萨尔瓦多经济部2001年至2003年的统计数据来看，普普沙为萨尔瓦多带来了超过2千2百万美元的收入，而loroco等普普沙原材料的出口也有助于刺激萨尔瓦多的经济。截至2005年前，有超过30万人以生产制作普普沙为生，而其中大多都是妇女。
