'''普拉查特叻·差亚卡叻'''（{{lang-th|'''พระเจ้าบรมวงศ์เธอ พระองค์เจ้าบุรฉัตรไชยากร กรมพระกำแพงเพ็ชรอัครโยธิน'''}}，[[皇家轉寫|皇家轉寫]]：'''Burachat Chaiyakon'''；{{bd|1881年|1月23日|1936年|9月14日|catIdx=P}}），[[却克里王朝|却克里王朝]]第五代国王[[拉玛五世|拉玛五世]]第18子。即为甘烹碧亲王。 

== 留学生涯 ==
1894年，在读完{{link-en|玫瑰园中学|Suankularb Wittayalai School}}后，普拉查特叻被拉玛五世送到了[[英国|英国]]著名的学校[[哈罗公学|哈罗公学]]，不久又转到[[剑桥大学三一学院|剑桥大学三一学院]]学习工程学，后在[[查塔姆|查塔姆]]地区參與工程。其后又到[[法国|法国]]、[[荷兰|荷兰]]等国的工程學校续修[[水壩|水壩]]與[[運河|運河]]工程。毕业以后，他回到英国成为公共工程局的職員。

== 回国之后 ==  
1904年，22岁的普拉查特叻回到[[泰国|泰国]]，成为陆军军队的一名工程师，也就是[[泰国皇家陆军|泰国皇家陆军]]公务员，并在此职务上工作17年。他发展了泰国政府的工程部队，并教授他在西方学来的知识。

== 退休和去世 ==
1933年，他退休并移居[[新加坡|新加坡]]直到他1936年去世，享年55歲。

== 国家工程 ==
普拉查特叻被称为泰国[[铁路|铁路]]之父，1910年在哥哥[[拉玛六世|拉玛六世]]即位后，他成为泰国铁路北方局局长。1917年拉玛六世将泰国铁路南北分局合并，成立[[泰国皇家铁路局|泰国皇家铁路局]]，于是普拉查特叻成为第一任局长，在他指挥下，修建了从[[呵叻府|呵叻府]]至[[乌汶府|乌汶府]]的東北地區铁路，后又修建了[[北柳府|北柳府]]至[[沙缴府|沙缴府]]的东线铁路，并使用[[柴油机车|柴油机车]]运行，这使泰国成为亚洲第一个使用柴油机车的国家。

他还主持修建了很多桥梁，就包括泰国第一座铁路桥[[Kasatsuk桥|Kasatsuk桥]]等。他还成立了泰国第一座广播电台[[Phayathai|Phayathai]]，而此后，他又使泰国成为亚洲第一个在广播电视方面有经验的国家，所以他也被称为泰国广电之父。1926年4月1日，他成为[[泰国商务部|泰国商务部]]和[[泰国交通部|泰国交通部]]大臣。
