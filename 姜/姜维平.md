==在中國的職業生涯==
姜维平於[[遼寧大學|遼寧大學]]畢業，並獲得歷史學位。<ref name=RSF />在1980年代，他成為了[[新华社|新华社]]的一名記者。於1984年，他寫了他的第一個關於[[薄熙来|薄熙来]]的文章。當時，薄熙来只是一個小區域的首長<ref name=TS/>。在1990年代初，姜维平成為了香港《[[文汇报_(香港)|文汇报]]》的[[中国东北地区|中国东北地区]]分社總編輯。<ref name=CECC>{{cite web |url=http://www.cecc.gov/pages/virtualAcad/index.phpd?showsingle=36155 |title=Anti-Corruption Reporter Jiang Weiping Released from Prison One Year Early |publisher=Congressional-Executive Commission on China |deadurl=no |archivedate=2012-09-12 |archiveurl=https://www.webcitation.org/6AbzYXnD2?url=http://www.cecc.gov/pages/virtualAcad/index.phpd?showsingle=36155 |accessdate=September 11, 2012 }}</ref>

在1999年中期，姜维平在《前哨》發表了一系列報道。<ref name="articles"/>該系列共包含八則報道，且全部都與中國政府官員涉嫌貪污有關。<ref name=TS/>他並沒有使用真名發佈報道，而是用筆名。<ref name=CECC />姜维平在報道中指出時任[[大連|大連]]市長的薄熙來掩蓋了他的妻子[[谷開來|谷開來]]的貪污罪行。<ref name=TS/>他亦指出薄熙來將國家公帑花費在汽車、公寓和情婦上。<ref name=CECC />他也在報道中指出[[瀋陽|瀋陽]]副市長[[馬向東|馬向東]]使用國家公帑在澳門睹博，已導致國家失去了近300萬美元的公帑。<ref name=CPJ>{{cite web |url=http://cpj.org/awards/2001/jiang.php |title=Jiang Weiping: Awardee 2001 |publisher=Committee to Protect Journalists |archivedate=2012-09-12 |deadurl=no |archiveurl=https://www.webcitation.org/6Ac0H5Uxu?url=http://cpj.org/awards/2001/jiang.php |accessdate=September 11, 2012 }}</ref>馬向東於2001年因此被處決。<ref>{{cite web |url=http://news.bbc.co.uk/2/hi/asia-pacific/1719089.stm |title=China executes corrupt officials |date=December 19, 2001 |publisher=BBC News |archivedate=2012-09-12 |deadurl=no |archiveurl=https://www.webcitation.org/6AcgWAjvp?url=http://news.bbc.co.uk/2/hi/asia-pacific/1719089.stm |accessdate=September 12, 2012 }}</ref>

== 逮捕和監禁 ==
儘管以筆名發表報道，但姜维平仍然被中國政府發現了其真實身份。他於2000年12月被大连[[国安局|国安局]]逮捕，並被控泄露国家秘密罪。<ref name=GM />根據姜维平指出，其辯護律師亦因願意為他辯護而在一日後被逮捕。<ref name=RSF2/>於2001年9月5日，他在第二次審訊中被判有罪<ref name=CPJ/>，並被判8年有期徒刑。<ref name=GM>{{cite web |url=http://www.theglobeandmail.com/commentary/editorials/book-about-fallen-chinese-official-a-testament-to-journalistic-courage/article4353365/ |title=Book about fallen Chinese official a testament to journalistic courage |date=June 19, 2012 |work=The Globe and Mail |archivedate=2012-09-12 |deadurl=no |archiveurl=https://www.webcitation.org/6Ac2YWRtn?url=http://www.theglobeandmail.com/commentary/editorials/book-about-fallen-chinese-official-a-testament-to-journalistic-courage/article4353365/ |accessdate=September 11, 2012 }}</ref>后來，判刑被遼寧省高級人民法院減低至6年。<ref name="闯祸"/><ref>{{cite web |url=http://www.ifex.org/china/2006/01/04/journalist_jiang_weiping_released/ |title=Journalist Jiang Weiping released from prison after five years |date=January 4, 2006 |publisher=Committee to Protect Journalists via IFEX |deadurl=no |archivedate=2012-09-12 |archiveurl=https://www.webcitation.org/6Ac54PwiI?url=http://www.ifex.org/china/2006/01/04/journalist_jiang_weiping_released/ |accessdate=September 11, 2012 }}</ref>

中國法院對姜维平的判刑引來世界各地的記者組織抗議。<ref name=PENC/>[[無國界記者|無國界記者]]將姜维平描述為「一個見義勇為，在兩度批評中國共產黨高層腐敗時，從沒想過會有危險的模範記者。」<ref name=RSF>{{cite web |url=http://en.rsf.org/china-my-experience-as-a-journalist-part-10-08-2009,34141 |title=My Experience as a Journalist (Part 1) |author=Jiang Weiping |date=August 10, 2009 |publisher=Reporters Without Borders |archivedate=2012-09-12 |deadurl=yes |archiveurl=https://www.webcitation.org/6Ac3IJ6Qe?url=http://en.rsf.org/china-my-experience-as-a-journalist-part-10-08-2009,34141 |accessdate=September 11, 2012 }}</ref>。[[喬治·W·布什|喬治·W·布什]]政府的行政管理部門亦抗議中國政府監禁姜维平，並呼籲中國當局將他釋放。<ref>{{cite web |url=http://www.highbeam.com/doc/1P3-1190466161.html |title=U.S. Welcomes Early Release of Chinese Journalist |author=David Gollust |date=January 4, 2007 |publisher=Voice of America news |accessdate=September 12, 2012 |archive-url=https://web.archive.org/web/20131105214518/http://www.highbeam.com/doc/1P3-1190466161.html |archive-date=2013-11-05 |dead-url=yes }}</ref><ref>{{cite web |url=http://www.highbeam.com/doc/1P1-72643008.html |title=Activist: China discloses court decisions favoring labor leader, detained journalist |author=Joe McDonald |date=March 18, 2003 |publisher=Associated Press |accessdate=September 12, 2012 |archive-url=https://web.archive.org/web/20131105214524/http://www.highbeam.com/doc/1P1-72643008.html |archive-date=2013-11-05 |dead-url=yes }}</ref>

姜维平後來寫道，他被警察強迫作虛假供述，並因此多番被警察折磨。他指出自己在審問期間曾多次失去了知覺，其中一次更需要住院治療。後來，一位獄警同意將姜维平的信送交其妻子。在信中，姜维平指出他獲得的待遇已有改善。<ref name=RSF2 />於2003年，姜维平被轉移至大連[[瓦房店市|瓦房店市]]監獄，並進行[[強迫勞動|強迫勞動]]。<ref name=RSF2 />

== 出獄後 ==
於2006年1月3日，姜维平獲釋，並被軟禁在家。<ref name=RSF2>{{cite web |url=http://en.rsf.org/china-my-life-in-prison-part-2-17-08-2009,34202 |title=My Life in Prison (Part 2) |author=Jiang Weiping |date=August 17, 2009 |publisher=Reporters Without Borders |deadurl=yes |archivedate=2012-09-12 |archiveurl=https://www.webcitation.org/6Ac3mIFlF?url=http://en.rsf.org/china-my-life-in-prison-part-2-17-08-2009,34202 |accessdate=September 11, 2012 }}</ref>於2009年，他獲准移民到加拿大。<ref name=GM />

他於[[多倫多|多倫多]]定居，並在{{link-en|梅西大學 (多倫多)|Massey College, Toronto|梅西大學}}擔任學者。於2012年6月，{{link-en|阿爾弗雷德·A·克諾夫|Alfred A. Knopf|克諾夫}}宣佈他們會出版姜维平寫的薄熙來傳。當時，薄熙來之妻谷開來因涉嫌謀殺英國商人[[尼尔·海伍德|尼尔·海伍德]]而轟動全球。<ref name=GM/>

==奖项==
於2001年，姜维平获[[保护记者委员会|保护记者委员会]]颁发[[国际新闻自由奖|国际新闻自由奖]]<ref name="CPJ award"/>。該獎項旨在表彰「面對攻擊、威脅或監禁時，仍然願意捍衞新聞自由的人」。姜维平延至2009年才能親身接受獎項。<ref>{{cite web|url=http://cpj.org/awards/2009/jiang-weiping-china.php|title=Jiang Weiping, China|publisher=Committee to Protect Journalists|archivedate=2012-08-11|deadurl=yes|archiveurl=https://www.webcitation.org/69qKULDJl?url=http://cpj.org/awards/2009/jiang-weiping-china.php|accessdate=August 11, 2012}}</ref>

于2006年，他获加拿大笔会（{{lang|en|PEN Canada}}）颁发單一人道奖，並獲得5,000加元獎金。<ref name=PENC>{{cite web |url=http://www.ifex.org/canada/2006/09/29/chinese_journalist_jiang_weiping/ |title=Chinese journalist Jiang Weiping wins inaugural PEN Canada One Humanity Award |date=September 29, 2006 |publisher=PEN Canada via IFEX |deadurl=no |archivedate=2012-09-12 |archiveurl=https://www.webcitation.org/6Ac0jMRaZ?url=http://www.ifex.org/canada/2006/09/29/chinese_journalist_jiang_weiping/ |accessdate=September 11, 2012 }}</ref>

== 個人生活 ==
姜维平的妻子是李楊凌，而李楊凌也曾因抗議姜维平被監禁而被短暫拘留。二人育有一名女兒。<ref>{{cite web |url=http://www.nytimes.com/2004/12/01/opinion/01kristof.html?pagewanted=print&position= |title=China's Donkey Droppings |author=Nicholas D. Kristof |date=December 1, 2004 |work=The New York Times |deadurl=no |archivedate=2012-09-12 |archiveurl=https://www.webcitation.org/6Ac1YoVca?url=http://www.nytimes.com/2004/12/01/opinion/01kristof.html?_r=1 |accessdate=September 11, 2012 }}</ref>於2004年，在姜维平獲釋前2年，加拿大笔会擔保其妻子和女兒均能移居加拿大。<ref name=GM/>
