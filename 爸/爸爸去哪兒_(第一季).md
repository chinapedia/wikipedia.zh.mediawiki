《'''[[爸爸去哪兒|爸爸去哪兒]]'''》第一季（[[英語|英語]]：Where Are We Going? Dad (Season 1)），是[[中華人民共和國|中國]]的[[湖南衛視|湖南衛視]]2013年度第四季度推出的週五晚22點档[[親子|親子]]互動[[真人秀|真人秀]]節目，節目[[版權|版權]]引自[[大韓民國|南韓]][[MBC|MBC]]電視臺，概念參考自南韓MBC電視臺節目《[[爸爸_你去哪兒|爸爸 你去哪兒]]》。節目於8月開始錄製，並於2013年10月11日接档選秀節目《[[快樂男聲2013|快樂男聲]]》，同年12月27日，節目最後一集第12集播完，第一季也宣告完結。這是湖南衛視繼之前《[[變形計|變形計]]》之後又一檔真人秀親子互動節目<ref>[http://www.apdnews.com/news/41830.html 亲子节目“爸爸去哪儿”] {{webarchive|url=https://web.archive.org/web/20131102030914/http://www.apdnews.com/news/41830.html |date=2013-11-02 }}，亚太日报，2013年10月31日</ref>，並由《變形計》製作人謝滌葵及其團隊和《[[我是歌手|我是歌手]]》製作人洪濤及其團隊聯合操刀打造。

== 原版與引進 ==
《爸爸去哪兒》，也叫“爸爸，你去哪兒”或“爸爸，我們去哪兒”，原為南韓MBC電視臺於週日下午播出的綜藝節目，節目於2013年1月播出。同年，[[湖南衛視|湖南衛視]]引進該節目版權並著手製作與推出。這是湖南衛視繼《[[我是歌手|我是歌手]]》之後第二度引進MBC的節目製作版權。

原版本的節目形態為五位明星父親與自己的子女一起到偏僻的村莊或是條件較為惡劣的環境下生存數日，或是過夜旅行體驗過程。湖南衛視版本在尊重原版的情況下，增加了側重考驗明星帶孩子的能力，並向觀眾傳遞正能量，讓更多的人更加重視親子之間交流與互動<ref>[http://zixun.hunantv.com/hntv/20130916/1727042224.html   《爸爸去哪儿》激发家庭正能量 亲子互动呼唤真情回归]金鷹網  2013-09-16</ref>。

== 首映禮 ==
2013年10月8日，製作單位聯合[[萬達集團|萬達影城]]為節目舉行了首映禮，這也是繼《[[我是歌手|我是歌手]]》之後，再度有節目聯合院線將電視節目搬上大螢幕。現場也集結了來自全國80餘家媒體參加首映禮<ref>[http://ent.qq.com/a/20131010/013312.htm 《爸爸去哪儿》首映受好评 导演记录感动笑点  ] 腾讯娱乐 2013-10-10</ref><ref>[http://ent.sina.com.cn/v/m/2013-10-10/10414020983.shtml 《爸爸去哪儿》首映李湘女儿偷亲小小志 ]新浪娱乐 2013年10月10日</ref>。

=== 路線圖 ===

製作單位所選取的六個錄製地點涵蓋了[[中華人民共和國|中國]][[華北地區|華北]]、[[中國西北地區|西北]]、[[中國西南地區|西南]]、[[華東地區|華東]]、[[中南地區|中南]]、[[中國東北地區|東北]]等中國[[中國地理|傳統六大地理大區]]，地理形態從[[沙漠|沙漠]]跨越到[[島嶼|島嶼]]，氣候特點從[[亞熱帶|亞熱帶]]跨越到[[亞寒帶|亞寒帶]]。氣候的多變和唯美的景色，也為節目增色不少，同時也起到宣傳在地觀光的作用。一度傳出有多個地方的旅遊局和景區願意出資跟準備好維安工作歡迎製作單位到當地取景，但都遭製作單位一一回絕<ref>[http://www.chinanews.com/yl/2013/11-11/5488892.shtml 《爸爸去哪儿》否认收景区钱：都要自个给钱的。]中國新聞網 2013年11月11日</ref><ref>[http://caijing.chinadaily.com.cn/xfly/2013-11-11/content_10548543.html 《爸爸去哪儿》否认收景区钱。] {{webarchive|url=https://web.archive.org/web/20131212033620/http://caijing.chinadaily.com.cn/xfly/2013-11-11/content_10548543.html |date=2013-12-12 }}中國日報網 2013年11月11日</ref>。

== 宣傳 ==
與以往[[湖南衛視|湖南衛視]]過往節目的宣傳及營銷手法一樣，除了首映禮是與《[[我是歌手|我是歌手]]》獨有以外，節目開播之前就在社群網站大規模貼出預告片和花絮，以《爸爸去哪兒》的官方微博為例，光節目官方海報就先後出了好幾彈，同時聘請為《我是歌手》藝人量身定做的漫畫海報團隊，再次為《爸爸去哪兒》團隊設計漫畫圖<ref>[http://weibo.com/babaqunaer 《爸爸去哪兒》官方微博]新浪微博 2013年10月21日</ref>。另外據湖南衛視官方微博的消息，目前為止從未在戶外播過的節目內容，宣傳片也於[[北京|北京]][[首都機場|首都機場]][[T2|T2]][[航站樓|航站樓]]全天滾動播出<ref>[http://weibo.com/1638629382/Af3j4e6U9 新浪微博湖南衛視官方帳號]新浪微博 2013年10月21日</ref>。
== 廣告置入 ==
節目開播前期，原本的冠名[[贊助商|贊助商]]為[[美的集團|美的集團]]的“美的IH智能煲”，但開播當晚卻換成了[[華潤醫藥|華潤醫藥]]的品牌“999感冒靈和小兒感冒藥”。有一說法指出，由於美的集團看過節目的片花以後，對節目是否能引起話題和獲得良好收視存在疑慮，因而臨時撤下廣告贊助，但隨著節目首期收視飆紅跟話題不斷，該廠商也錯失了一次營銷產品的好機會<ref name="廣告商撤資"/>。而華潤三九則在廣告商和電視博弈之間撿漏上位，並在節目播出之後引起強烈迴響和超高收視而收穫出其意料的結果<ref name="廣告商撤資"/><ref>[http://www.chinanews.com/yl/2013/10-15/5380647.shtml 《爸爸去哪儿》零差评，曾遇赞助商取消冠名窘况。]中國新聞網 2013年10月15日</ref>。隨著節目在[[社群網站|社群網站]]爆紅，收視也由原來的1.4到最最高5.3的收視，廣告的置入越來越多，除卻冠名贊助，中間插入的聯合贊助也多了如[[思念食品|思念食品]]的“[[金牌灌湯水饺|金牌灌湯水饺]]”和“[[英菲尼迪|英菲尼迪]]汽車”廠商加入廣告陣營<ref name="廣告商撤資">[http://news.sina.com.cn/o/2013-11-03/190328608886.shtml 《爸爸去哪儿》掀起收视高潮，广告商暗中角力。]新浪新聞中心 2013年11月03日</ref>。由於此，首期節目播到23點35分便結束，到最新一期播到隔日凌晨0點才結束，除了跟節目剪輯增加畫面有關以外，也跟廣告的時長變長有直接關係。

== 迴響 ==
=== 社交網絡爆紅 ===
節目由五位來自[[演藝界|演藝界]]、[[體育|體育界]]、[[時尚|時尚界]]的人及其子女組成，因此具備了一定的話題，尤其是有跳水奧運冠軍[[田亮|田亮]]和人稱“小旋風”的[[林志穎|林志穎]]的加盟，節目播出之後佔據著[[新浪微博|新浪微博]]熱門話題榜榜單，五位參加人員的微博粉絲多則增加将近两千万，少則幾百萬。而官方帳號則從開播之初幾萬直接近乎夸张地飆漲到三百一十多万{{備註|1=以上所有數據截止到2013年12月7日為止。}}。

此外，王詩齡母親[[李湘|李湘]]在2013年10月底為女兒開通新浪微博，每條微博轉發都上千條，評價更是有接近上萬條，至2013年11月17日粉絲數量便已經突破41萬<ref>[http://news.yesky.com/hot/384/35422884.shtml 《爸爸》走红 王诗龄开微博人气直逼老李湘] {{webarchive|url=https://web.archive.org/web/20131214063641/http://news.yesky.com/hot/384/35422884.shtml |date=2013-12-14 }} 2013年11月17日</ref>；其母李湘的微博粉絲也超千萬<ref>[http://www.shcaoan.com/wy/Jsa/85165.html 李湘沾光王诗龄微博粉丝破千万 爸爸去哪儿移师威海鸡鸣岛(图)] 2013年10月31日</ref>。

=== 收視一路上揚 ===
首集收視1.4的成績和[[浙江衛視|浙江衛視]]節目《[[中國夢想秀|中國夢想秀]]》幾乎打平{{备注|1=不同時段播出，詳情請閱讀“收視率一節”。}}，但從第二期開始直接飆漲到2.5，第三、四期直接破3，成為所有電視頻道於22點檔開播的節目中，第一個收視率達到3%以上的節目{{备注|1=《[[中國好聲音|中國好聲音]]》最高破6，但其節目從晚間9點15分開始；《[[非誠勿擾_(節目)|非誠勿擾]]》曾經最高破4，但其節目開始時間仍然為晚間9點15分開始，因此歸類為21點檔節目，不算真正意義上的22點檔節目，以上數據為CSM46城數據。}}，到最新一期的CSM48城市網達到5.3的收視，一舉打破《[[中國好聲音|中國好聲音]]》保持的5.2的最高收視，成為近年來收視最高的綜藝節目，也是22點檔第一個破5的綜藝節目。節目不僅自身收視高，還拉提了[[湖南衛視|湖南衛視]]其他節目，在11月2日播出的《[[快樂大本營|快樂大本營]]》《爸爸去哪兒》專場的節目中，原本常態收視平均為2%的收視率爆衝到4.2%，創近幾年新高<ref>[http://weibo.com/2635260410/Ah9agCbD1 2013年11月2日收視排行榜]CSM46城市網2013年11月2日收視排行榜 2013年11月02日</ref>，而在10月18日中播出的《[[天天向上|天天向上]]》《爸爸去哪兒》專場中，原本常態節目收視率為1%到1.5%之間瞬間爆衝到1.9%<ref>[http://weibo.com/2635260410/Af1c6EmCp 2013年11月2日收視排行榜]CSM46城市網2013年10月18日收視排行榜 2013年10月18日</ref>，足見節目的高話題性與影響力。

=== 親子節目紛出現 ===
自從本季的《爸爸去哪兒》成功後，2013年下旬開始中國大陸各衛視出現了很多親子節目上檔，粗略統計，2014年便大約有十餘檔親子節目上線，其中有7檔主打“爸爸”牌，如[[浙江衛視|浙江衛視]]的《人生第一次》、《[[爸爸回來了|爸爸回來了]]》，還有搭載“爸爸”順風車的《好爸爸壞爸爸》、《爸爸請回答》、《爸爸我來了》等<ref>{{cite web|url=http://yule.sohu.com/20140428/n398893417.shtml | language= | title="国产综艺猛“拼爹” “爸爸”已快被玩坏 |publisher=搜狐娱乐 |date=2014-04-28  }}</ref>。

== 評價 ==
=== 正面 ===
根據《[[北京青年報|北京青年報]]》援引媒體調查的數據，節目之所以走紅，被吸引的點包括：排第一的是“孩子純真的童心”，佔44%；排在第二位的是“重新發現了父愛”，佔11%；而“作為養育孩子的父母，有許多共鳴”占到7%，觀眾對節目的看法多停留在積極一面<ref name="北京青年報"/> 。由於暑假期間各大電視頻道扎堆播出音樂節目，《爸爸去哪兒》從眾多音樂節目中脫穎而出，網友直呼電視屏幕多了新鮮感。除了被五位明星父親吸引之外，更多的是被孩子們天真無邪的動作舉止和笑臉吸引。 《北京青年報》社論用“紅”與“黑”的觀點點評節目，認為節目將爸爸在親子教育中容易被忽略的角色展現在大眾面前，讓人重新思考在親子教育中爸爸的角色，同時節目中爸爸引導孩子們解決生活問題，並通過與孩子的溝通來拉近彼此的瞭解<ref name="北京青年報評論"/>。
=== 爭議 ===
==== 節目性質 ====
然而《北京青年報》的社論同時也批評製作單位拿孩子的童趣當娛樂，拿孩子來作秀，用孩子們的天真來博取娛樂的噱頭，並批評製作單位雖賺足了笑點、掙夠了收視率，卻在無形中卻傷了孩子<ref name="北京青年報">[http://ent.sina.com.cn/r/m/2013-11-01/11544034896.shtml 北京青年報：親子節目別拿童真當娛樂。 ]新浪娛樂引用《北京青年報》社論2013年11月1日</ref>。但此番言論仍然只是一家之言，並未能代表觀眾的想法。然而《北京青年報》連續發表社論，連番點評節目，認為節目內容設計完全與小孩教育無關，該觀點認為，節目中爸爸做飯及帶孩子的表現完全只能當笑料和娛樂，除了這方面再找不出關於親子教育的東西，將孩子帶到村莊和沙漠完全是折騰孩子<ref name="北京青年報評論">[http://ent.sina.com.cn/r/m/2013-11-01/11594034899.shtml 评论：剖析《爸爸去哪儿》的红与黑。]新浪娛樂引用《北京青年報》評論 2013年11月1日</ref>。

==== 節目設計 ====
於第四期播出的內容中，有一段“保護雞蛋”環節，由孩子保護雞蛋不被打破，爸爸們則想辦法打破雞蛋。此番設計遭來一些網友非議，認為節目浪費食物不說，還將大人聯合起來欺騙小孩，並且美化成培養孩子們的責任感<ref>[http://v.qq.com/cover/9/95u33brar7iucf7.html?vid=e001373t0l5&pgv_ref=aio2012&ptlang=2052 《爸爸》护蛋环节引争议,天天感动网友。]騰訊娛樂 2013-11-04</ref>。

==== 版權引進與國內節目創新 ====
《爸爸去哪兒》節目版權引進自[[大韓民國|南韓]][[文化廣播_(韓國)|文化廣播公司]]，也是繼《[[中國好聲音|中國好聲音]]》和《[[我是歌手|我是歌手]]》後第三個火爆[[華人圈|華人圈]]的綜藝節目，縱觀這些節目都有一個特點就是全部引進國外[[版權|版權]]{{备注|1=《中國好聲音》引自[[荷蘭|荷蘭]]，《我是歌手》、《爸爸去哪兒》引自韓國。}}，而中國境內原創節目市場則日趨蕭條與不景氣，國內眾多電視臺一窩蜂的模仿與引進，進一步打壓了原創節目的空間。隨著本節目火爆螢屏，[[中共|中共]]官方媒體《[[人民日報|人民日報]]》針對這一現象，在社論裏面批評[[中華人民共和國電視臺列表|中國電視臺]]，並列出了中國電視現狀三大罪狀：'''補課'''：買模式“逼迫”中國電視從作坊式走向工業化流水線；'''惰性'''：“拿來主義”暫時可保收視率，卻抹殺原創動力和能力；'''求索'''：買“點子”做本土化移植，學習如何把創意落實為模式<ref>[http://enjoy.eastday.com/e/20131030/u1a7743309.html 《人民日报》关注《爸爸去哪儿》:中国电视期待原创]東方網引用《人民日報》社論 2013年10月30日</ref>。
