'''蓬图瓦兹'''（{{lang-fr|Pontoise}}），法国北部城市，[[法兰西岛|法兰西岛大区]]（Région Île de France）[[瓦茲河谷省|瓦勒德瓦兹省]]（Val d'Oise，95省）的一个市镇和该省的一个副省会，距离[[巴黎|巴黎]]市区大约34公里。蓬图瓦兹和与其相邻的[[塞尔吉|塞日]]（Cergy）共同组建形成了瓦勒德瓦兹省的行政中心。蓬图瓦兹历史悠久，罗马人开辟出的巴黎—鲁昂通道就在这一地区跨越了[[瓦兹河|瓦兹河]]（L'Oise），“蓬图瓦兹”在法语中即为“瓦兹河大桥”之意。依靠其水陆交通之便，蓬图瓦兹长期以来为这一地区的商业重镇，英法百年战争之后开始衰败。二战后，随着行政区划的调整和大巴黎城市规划方案的实施，蓬图瓦兹逐渐成为了巴黎西北方向的重要卫星城，人口数量和城市规模均大幅增加。蓬图瓦兹也是法国历史文化名城（Ville d'art et d'histoire）之一，法国著名画家[[卡米耶·毕沙罗|卡米耶·毕沙罗]]和[[保罗·塞尚|保罗·塞尚]]曾长期居住于此，并以这一地区的风光为背景，创作了大量的印象派油画作品。

== 行政区划 ==
蓬图瓦兹是瓦勒德瓦兹省的首府。蓬图瓦兹属于巴黎都市区和巴黎城市圈。同时，蓬图瓦兹和与其相邻的12个市镇共同组建了塞日-蓬图瓦兹城市社区（Communauté d'agglomération de Cergy-Pontoise），旨在协调这一地区之间的发展。

蓬图瓦兹是瓦勒德瓦兹省的省会，也是蓬图瓦兹地区（Arrondissement de Pontoise）的行政中心（Chef-lieu），[[即副省会|即副省会]]（Sous-préfecture）。

蓬图瓦兹（北纬49.05度，东经2.09度），位于法国北部，法兰西岛大区的西北部和瓦勒德瓦兹省的西部。

== 历史 ==
和巴黎周边的许多卫星城不同，蓬图瓦兹历史悠久，是维克桑（Vexin）地区的中心城市。

=== 起源 ===
蓬图瓦兹最早形成于高卢-罗马时期（注：尚有争议），罗马人在这一时期修建了巴黎—鲁昂大道，即“凯撒之路”（Chaussée Jules César）。该大道即在蓬图瓦兹附近跨越了瓦兹河。

=== 发展 ===
在[[中世纪|中世纪]]早期，蓬图瓦兹是瓦兹河上唯一一座可以通过的桥梁，蓬图瓦兹也因此长期成为了这一地区的一个交通枢纽和军事要塞，商业贸易也由此产生，并推动了城市的发展。

公元9世纪末，诺曼人入侵维克桑地区，并一度驱散当地的居民。

公元911年，西法兰克王国国王查理三世（Charles III le Simple）与诺曼人首领罗龙（Rollon）签订条约，诺曼人暂时退出了这一地区。鉴于蓬图瓦兹重要的地理位置，查理三世在这一地区大量修建防御工事和城墙，以保护巴黎。

公元1188年，法兰西王国国王腓力二世（Philippe Auguste）将蓬图瓦兹封为“商业自由市”（charte communale），并赋予了大量行政和司法的自治权限。蓬图瓦兹因此迅速发展。法国著名诗人维永（François Villon）在描述巴黎时甚至写到：“巴黎，蓬图瓦兹旁的城市（Paris, près Pontoise）”，由此可见当时蓬图瓦兹的城市地位。

=== 衰退 ===
在[[百年战争|英法百年战争]]（Guerre de Cent Ans）期间，蓬图瓦兹由于地理位置重要，经常成为军事攻击的目标。故而这一地区商业出现严重的衰退，城市人口和规模也大幅下降。到了宗教战争（Guerre de Réligion）时期，新教教徒和天主教徒之间的斗争在此限制了这座城市的发展。

=== 复兴 ===
法国大革命结束后，蓬图瓦兹及整个维克桑地区被归入了[[塞纳-瓦兹省|塞纳-瓦兹省]]（Seine-et-Oise），编号为78。

蓬图瓦兹城市的再次大规模发展，出现于工业革命期间，即公元19世纪。经过蓬图瓦兹的{{link-fr巴黎—迪耶普（Dieppe）|Ligne de Saint-Denis à Dieppe}}铁路于1863年通车，该线路大大缩短了巴黎与蓬图瓦兹之间的旅行时间。

1964年，法国政府实施大巴黎地区的行政区划改革方案，原有的塞纳-瓦兹省被一分为二，南部改为伊夫林省，编号仍为78；蓬图瓦兹及原78省的北部地区则重新组建了一个新的省份——瓦勒德瓦兹省（Val d'Oise），编号为95，省会即蓬图瓦兹。

1970年，随着塞日行政中心的建成，瓦勒德瓦兹省的省政府迁离蓬图瓦兹，入驻2公里外的市镇——塞日（Cergy）。但蓬图瓦兹仍保留“省会”的名号。与此同时，法国政府推行“为大城市减压（Décentralisation）”计划，在巴黎周边大量修建卫星城和城市新区。其中塞日-蓬图瓦兹（Cergy-Pontoise）是巴黎附近最大的两个“新城”之一（另一个是马恩拉瓦莱（Marne-la-Vallée），又称马恩河谷）。

2000年，巴黎郊区地铁C线（RER C）连接蓬图瓦兹的一个分支建成运营。

2004年，巴黎郊区地铁J线（Transilien J）建成通车，使得巴黎—蓬图瓦兹之间的列车平均发车间距缩短至10分钟以内。

2006年，蓬图瓦兹被评为法国国家历史文化名城。

== 市长 ==
现任蓬图瓦兹市长是菲利普·武永（Philippe Houillon）先生，他是法国最大的右翼政党人民运动联盟（UMP）的成员。该政党已于2015年5月正式更名为法国共和党（Les Républicains，简称LR）。

== 经济收入 ==
蓬图瓦兹为巴黎附近的新兴城市，第三产业发达，特别是商业和公共服务业。

作为中心城市，蓬图瓦兹市中心集中了大量的离退休人员（22.9%）和普通工人（15.1%），而中产以上阶级家庭主要分布在周边市镇。故而从单个的市镇来看，蓬图瓦兹的人均收入水平并不高。

2011年，蓬图瓦兹的人均工资（Moyenne des salariés）为2398欧元，家庭月平均可支配收入（Revenu par ménage）的中位数为2487欧元，后者在法国36717个市镇中排名第16149位，低于瓦勒德瓦兹省平均水平（2922欧元）和法兰西岛大区平均水平（2909欧元），高于法国本土的平均水平（2410欧元）。

2011年，蓬图瓦兹的家庭月平均消费水平（Niveau de Vie）为1593欧元，在法国36717个市镇中排名第13361位，低于瓦勒德瓦兹省平均水平（1696欧元）、法兰西岛大区的平均水平（1854欧元）和法国本土的平均水平（1602欧元）<ref>{{Cite web|url = http://www.salairemoyen.com/salaire-ville-95500-PONTOISE.html|title = 法国人均工资（salaire moyen）查询网站（法语）|accessdate = 2016-02-17|author = |date = |publisher = }}</ref>。

== 交通 ==
蓬图瓦兹位于巴黎西北，法国国家A15号高速公路和多条省道经过该地。以下为以蓬图瓦兹为中心，前往主要城市的公路里程：
* 距离[[巴黎|巴黎]]（Paris）32公里（高速公路）
* 距离[[里昂|里昂]]（Lyon）503公里（高速公路）
* 距离[[马赛|马赛]]（Marseille）812公里（高速公路）
* 距离[[亞眠|亚眠]]（Amiens）114公里（省道22公里，其余为高速公路）
* 距离[[鲁昂|鲁昂]]（Rouen）92公里（高速公路）
* 距离凡尔赛（Versailles）33公里（省道和国道）
* 距离[[楠泰尔|楠泰尔]]（Nanterre）31公里（高速公路）
* 距离[[博比尼|博比尼]]（Bobigny）34公里（高速公路）
* 距离[[博韦|博韦]]（Beauvais）50公里（省道22公里，其余为高速公路）
* 距离[[塞尔吉|塞日]]（Cergy）3公里（市区道路）
* 距离萨尔塞勒（Sarcelles）30公里（国道+高速公路）
* 距离阿尔让特伊（Argenteuil）23公里（高速公路）
[[蓬图瓦兹站|蓬图瓦兹站]]（Gare de Pontoise）位于蓬图瓦兹市区南部，是巴黎郊区地铁C线、H线和J线的交汇点，可分别直达前往巴黎奥斯特里茨站、巴黎北站和巴黎圣拉扎尔车站，同时还可以前往外省的吉索尔（Gisors）、克雷伊（Creil）等地。

蓬图瓦兹附近有一个小型停机坪，一般不对民众开放；蓬图瓦兹距离巴黎戴高乐机场大约41公里。

蓬图瓦兹市区的公交线路主要由两个公交公司运营，分别为瓦勒德瓦兹公共交通公司（Busval d'Oise）和瓦勒德瓦兹城际公交公司（Stivo）。共有数十条公交线路，其线路网覆盖了蓬图瓦兹及周边主要地区。此外，巴黎公交N150路（夜间公交）也经过了蓬图瓦兹。

== 教育 ==
蓬图瓦兹属于凡尔赛学区（Academie Versailles）。

=== 中学 ===
截止2015年11月，蓬图瓦兹市镇范围内共有3所初中和一所大学校预备高中（STS/CPGE）。

=== 大学 ===
蓬图瓦兹境内没有高等教育机构。

注：塞吉-蓬图瓦兹大学（Université de Cergy-Pontoise）所在的塞日（Cergy）距离蓬图瓦兹市区只有三公里，但其校园所在地不在蓬图瓦兹的市镇范围内。

== 文化节日 ==
圣马丁集市（Foire Saint-Martin），起源于中世纪，已有800多年的历史，每年11月11日~18日举行。2015年，由于巴黎11.13枪击案的发生，圣马丁集市提前结束。

== 艺术 ==
法国著名画家卡米耶·毕沙罗（Camille Pissarro）和其好友保罗·塞尚（Paul Cézanne）曾被蓬图瓦兹附近的风景所吸引，并以此为主题创作了多幅写实作品。
