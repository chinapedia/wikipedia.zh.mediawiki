'''凤凰光学集团有限公司'''是一家以生产[[镜头|镜头]]、[[顯微鏡|顯微鏡]]为主的公司，销售和技术开发中心位于[[中国|中国]][[上海市|上海市]]，生产制造总部位于[[江西省|江西省]][[上饶市|上饶市]]西郊，在上海、广东、江西建立了三大生产基地<ref>{{cite web |url=http://www.phenixoptics.com.cn/shownews.asp?id=141 |title=央视焦点访谈聚焦凤凰光学 |accessdate=2009-10-25 |language=zh-hans |deadurl=yes |archiveurl=https://web.archive.org/web/20081225073727/http://www.phenixoptics.com.cn/shownews.asp?id=141 |archivedate=2008-12-25 }}</ref>。前身为'''江西光学仪器总厂'''（简称江光、江光总厂）。

近年来，凤凰光学主要生产[[数码相机|数码相机]]、手机、扫描仪、复印机、监视器材的镜片镜头等，以中小球面镜片加工为主，月产量可达300万片。<ref>{{cite web |url=http://www.phenixoptics.com.cn/shownews.asp?id=64 |title=40年技术经验沉积的光学加工基地 |accessdate=2009-10-25 |language=zh-hans |deadurl=yes |archiveurl=https://web.archive.org/web/20081223223131/http://www.phenixoptics.com.cn/shownews.asp?id=64 |archivedate=2008-12-23 }}</ref>现任董事长任捷。<ref name="光学第一股"/>

== 历史 ==
1965年，根据[[三线建设|三线建设]]的战略部署，中央调集[[上海照相机二厂|上海照相机二厂]]、[[上海电影机械厂|上海电影机械厂]]、南京[[江南光学仪器厂|江南光学仪器厂]]、[[南京测绘仪器厂|南京测绘仪器厂]]、[[南京模具厂|南京模具厂]]等单位的相关人才和设备在江西[[德兴|德兴]]大茅山区组建了一个综合性的光学仪器厂，即“江西光学仪器总厂”。1970年建成投产。205型相机生产之初仍沿用上海照相机二厂的“海鸥”商标。1981年7月申请注册“凤凰”商标，1983年5月10日获得批准。<ref name="回味历史三十载">{{cite web |url=http://www.phenixoptics.com.cn/shownews.asp?id=164 |title=回味历史三十载 凤凰相机纵览(一) |accessdate=2009-10-25 |language=zh-hans }}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

1988年江光总厂从德兴搬迁到了上饶市。<ref name="光学第一股">{{cite web |url=http://www.phenixoptics.com.cn/shownews.asp?id=148 |title=“光学第一股”掌门人任捷：凤凰再生 |accessdate=2009-10-25 |language=zh-hans }}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>1997年5月，“'''凤凰光学'''”（{{sse|600071}}）股票在[[上海证券交易所|上海证券交易所]]上市，被誉为“中国光学行业第一股”。这一年，销售总公司迁至上海，外贸总部则迁至[[深圳|深圳]]。1998年开发中心迁至上海。<ref name="光学第一股"/>

该公司至今只生产35mm画幅的相机。<ref name="回味历史三十载"/>

2003年开始在[[上饶县|上饶县]]旭日工业园区内新建凤凰光学城区，面积约465亩。<ref>{{cite web |url=http://www.phenixoptics.com.cn/shownews.asp?id=24 |title=凤凰光学与上饶县政府签订共同兴建凤凰光学城区协议 |accessdate=2009-10-25 |language=zh-hans }}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

2004年，日本[[富士能|富士能]]和凤凰合资组建江西凤凰富士能有限公司。<ref>{{cite web |url=http://www.phenixoptics.com.cn/shownews.asp?id=479 |title=凤凰改制是大势所趋 |accessdate=2009-10-25 |language=zh-hans |deadurl=yes |archiveurl=https://web.archive.org/web/20081228024536/http://www.phenixoptics.com.cn/shownews.asp?id=479 |archivedate=2008-12-28 }}</ref>
