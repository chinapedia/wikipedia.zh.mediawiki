'''凤凰城影评人协会奖十佳影片'''是由美国凤凰城影评人协会每年年末评定的“十佳影片”名单，其中，其十佳影片名单的影片几乎年年得到了[[奥斯卡最佳影片奖|奥斯卡最佳影片奖]]。<ref>{{cite web |url=http://ent.sina.com.cn/m/c/2012-12-25/14273820106.shtml |title=凤凰城影评人协会奖《少年派》斩获三奖 |accessdate=2012 |publisher=新浪网 |work= |language=zh-hans}}</ref><ref>{{cite web |url=http://ent.163.com/11/1230/15/7MHJI9LU00034O55.html |title=凤凰城影评人协会奖揭晓 《艺术家》狂揽八奖 |accessdate=2011 |publisher=网易 |work= |language=zh-hans}}</ref><ref>{{cite web |url=http://ent.sina.com.cn/m/f/2010-12-29/15143192100.shtml |title=凤凰城影评人协会奖揭晓 《演讲》力挫《网战》 |accessdate=2010 |publisher=新浪网 |work= |language=zh-hans}}</ref><ref>{{cite web |url=http://ent.sina.com.cn/m/f/2009-12-25/18272824274.shtml |title=《无良杂牌军》获凤凰城影评人协会四项大奖 |accessdate= 2009|publisher=新浪网 |work= |language=zh-hans}}</ref><ref>{{cite web |url=http://ent.sina.com.cn/m/2008-12-17/00572298996.shtml |title= 凤凰城影评人协会奖揭晓 《贫民富翁》风卷残云|accessdate=2008 |publisher=新浪网 |work= |language=zh-hans}}</ref><ref>{{cite web |url=http://ent.sina.com.cn/m/f/2007-12-19/11081841262.shtml |title=凤凰城影评人协会奖：《老无所依》风卷残云 |accessdate=2007 |publisher=新浪网 |work= |language=zh-hans}}</ref>
