'''宦溪镇'''是[[中华人民共和国|中国]][[福建省|福建省]][[福州市|福州市]][[晋安区|晋安区]]下辖的一个[[镇|镇]]<ref>{{PRC admin/ref sgc|{{PRC admin/parent|35/01/11/103/000}}}}</ref>，位于晋安区东北部。

宦溪镇主要民族为[[汉族|汉族]]和[[畲族|畲族]]。

== 历史沿革 ==
[[唐朝|唐朝]]，宦溪属闽县八座乡稷下里。[[宋朝|宋]][[太平兴国|太平兴国]]六年(981年)，八座乡改隶怀安县。[[元朝|元朝]]，里改为都，宦溪属怀安县八座乡六都。[[明朝|明]][[万历|万历]]八年(1508 年)，怀安县并入侯官县，宦溪属八座乡42都。清朝，归侯官县大北岭区。

[[中华民国|中华民国]]期间，今宦溪分别属宦溪、捷后两个乡，属闽侯县新店区(5区)。[[中华人民共和国|中华人民共和国]]成立初，宦溪属闽侯县第8区(石牌区)。1962年归[[晋安区|福州市郊区]]管辖。1970年，改属福州市北峰区。1975年，北峰区撤销，宦溪重新归福州市郊区。1992年，宦溪乡撤乡改镇。

2004年，[[鼓岭乡|鼓岭乡]]并入宦溪镇。

== 行政区划 ==
宦溪镇政府驻宦溪村，共辖24个[[行政村|行政村]]，分别是：<ref>{{PRC admin/ref sgc|35/01/11/103/000}}</ref>

宦溪村、黄田村、黄土岗村、湖中村、牛项村、湖山村、垄头村、山溪村、峨嵋村、亥由村、板桥村、降虎村、弥高村、创新村、中心村、民义村、建立村、增楼村、胜利村、洲洋村、鹅鼻村、宜夏村、过仑村、南洋村。

== 旅游 ==
宦溪镇的[[鼓岭|鼓岭]]是著名的避暑胜地。宦溪当地的芙蓉石为[[寿山石|寿山石]]的主要石种之一。
