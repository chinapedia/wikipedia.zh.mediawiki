'''盈江县'''是[[中华人民共和国|中华人民共和国]][[云南省|云南省]][[德宏傣族景颇族自治州|德宏傣族景颇族自治州]]西北部的一个县，西、西北、西南面与[[缅甸|缅甸]]接壤。面积4429平方公里，2004年人口27万人。县政府驻：平原镇永胜路花园巷6号。

== 概況<ref>中共雲南省委政策研究室 雲南省志編纂委員會主編.雲南地州市縣概況 德宏傣族景頗族自治州分冊.雲南人民出版社,1987年08月第1版.第83頁</ref> ==

盈江是古「西南絲道」的交通要道，歷代滇緬的重大邊事，多與此相聯繫。境內有[[明代|明代]]所建的銅壁、神護、萬仞、巨石四關（鐵壁關、虎踞關、漢龍關和天馬關四關淪緬），和「虎跳石峽谷」、曼勐町塔等風景名勝。自然資源豐富，有中國種植最早的橡膠母樹，及野生稻、美登木、鹿角蕨等珍稀植物，有鹿血酒、糯米香酒、藤器等土特名產，是糧蔗主要產區之一。

盈江縣地處西南邊陲，位於東經97°31′-98°15′、北緯24°24′-25°20′之間。東與[[騰衝|騰衝]]交界，東南與[[梁河|梁河]]毗連，南與[[隴川|隴川]]接壞，西面和西北、西南與[[緬甸|緬甸]]為鄰。國境線從北部的野牛坡至南部的朝陽山，全長214.6公里。縣境東西最大橫距54公里，南北最大縱距114公里，全縣總面積4311.74平方公里，其中水域面積34.52平方公里，山區面積佔總面積的85.2％，壩區面積佔14.8％。縣城平原鎮東至省會[[昆明市|昆明市]]864公里，南至州府芒市257公里。

== 沿革 ==

* [[西漢|西漢]]時期屬古[[滇越乘象國|滇越乘象國]]。
* [[東漢|東漢]]、[[蜀漢|蜀漢]]、[[晉|晉]]時期屬[[永昌郡|永昌郡]][[哀牢縣|哀牢縣]]。
* [[南朝|南朝]]是[[永昌郡|永昌郡]]的[[西城縣|西城縣]]。
* [[唐代|唐代]][[南詔|南詔]]時期為永昌節度軟化府轄的押西城。
* [[宋代|宋代]][[大理國|大理國]]時期為[[騰衝府|騰衝府]]的乞蘭部。
* [[元代|元代]]設[[鎮西路|鎮西路]]軍民總管府。
* [[明代|明代]]永樂元年(1403年)改為[[干崖土司|干崖長官司]]，正統九年(1444年)升宣撫司，正統十四年(1449年)析干崖地置盞達副長官司。
* [[清代|清代]]順治十六年(1659年)，干崖仍授宣撫司，盞達升為副宣撫司。
* 民國元年(1912年)，分設干崖、盞達彈壓委員，民國五年(1916年)改設行政委員，隸[[騰越道|騰越道]]。民國二十年(1932年)，改設[[盈江設治局|盈江設治局]]、[[蓮山設治局|蓮山設治局]]，為第一殖邊督辦所轄。
* 1950年5月，[[中國人民解放軍|中國人民解放軍]]進駐盈江、蓮山，6月分別成立盈江、蓮山民族行政委員會，1951年底成立盈江，蓮山兩縣民族民主聯合政府。1958年10月，蓮山、盈江兩縣合併為盈江縣(盞西地區原屬[[騰衝縣|騰衝縣]]，1955年劃入)。

==行政區劃==

全县辖8个镇、6个乡、1个民族乡：

*鎮：[[平原镇_(盈江县)|平原镇]]、[[旧城镇_(盈江县)|旧城镇]]、[[那邦镇|那邦镇]]、[[弄璋镇|弄璋镇]]、[[盏西镇|盏西镇]]、[[卡场镇|卡场镇]]、[[昔马镇|昔马镇]]、[[太平镇_(盈江县)|太平镇]]。

*鄉：[[新城乡_(盈江县)|新城乡]]、[[油松岭乡|油松岭乡]]、[[芒章乡|芒章乡]]、[[支那乡|支那乡]]、[[勐弄乡|勐弄乡]]、[[铜壁关乡|铜壁关乡]]。

*民族鄉：[[苏典傈僳族乡|苏典傈僳族乡]]。

*農場：[[盈江农场|盈江农场]]。

==2011年云南盈江地震==

当地时间2011年3月10日12时58分，[[云南省|云南省]]盈江县发生5.8级地震，此后10分钟之内又相继发生了3次余震，震级分别为4.7、4.5和3.6级<ref>{{cite web | language = zh-cn | publisher = 人民网 | title =云南盈江5.8级地震  陈厚群：附近水电站未受影响| url = http://scitech.people.com.cn/GB/14112836.html| date = 2011-03-10 | accessdate = 2011-03-10 }}</ref>。截止17时39分，地震至少造成了25人死亡，207人受伤<ref>{{cite web | language = zh-tw | publisher = Yahoo！奇摩 | title = 雲南規模5.8地震　24死207傷 | url = http://tw.news.yahoo.com/article/url/d/a/110310/8/2nsm5.html | date = 2011-03-10 | accessdate = 2011-03-10 | deadurl = yes | archiveurl = https://web.archive.org/web/20110313003500/http://tw.news.yahoo.com/article/url/d/a/110310/8/2nsm5.html | archivedate = 2011-03-13 }}</ref>。
