'''阳山水蜜桃'''，为[[中国|中国]][[江苏省|江苏省]][[无锡市|无锡市]]著名[[土特产|土特产]]，因种植于[[惠山区|惠山区]][[阳山镇|阳山镇]]而得名，以其形美色艳、汁多甘厚、入口即化等特点而驰名中外，至今已有70余年历史。

== 简介 ==

旧时无锡多产脆桃，而大量栽培水蜜桃的历史，始自1930年前后，藕塘丁村上三六桃园由[[浙江|浙江]][[奉化|奉化]]引进的“白芒蟠桃”、“玉露水蜜桃”等品种，加之1920年，从日本引进的“白凤水蜜桃”，在阳山多年培育后此時業已大获成功。几种果品逐渐改良，形成阳山水蜜桃独特之风味，遂成为无锡名产，名闻江南。<ref name="资料">徐桥猛、王力行编，《无锡菜典》（中国轻工业出版社，2007年），页175。</ref>又查奉化桃种，实源自[[上海|上海]]黄泥墙，故阳山水蜜桃、奉化玉露桃和可说与上海水密桃一脉相承。<ref>朱海平，〈沪上水蜜桃的历史变迁〉，《上海档案》2013年12期，页38。</ref>

阳山水蜜桃之成功，源自其栽培地“阳山”原为1.4亿年前形成的古火山，这也是华东地区唯一的古火山。火山喷发后，留下的火山灰土壤里，含有多种保健矿物质和微量元素，爲阳山水蜜桃独特的口感和营养之秘诀提供了基礎。<ref name="简介">张惠明、俞显峰，〈七月桃乡都是甜——“中国水蜜桃之乡”无锡阳山镇〉，《长三角》2007年08期，页29-30。</ref>

如今，无锡阳山镇水蜜桃不仅并列与[[山东|山东]][[肥城|肥城]]、[[河北|河北]][[深州|深州]]、[[浙江|浙江]][[奉化|奉化]]是公认的中国四大传统产桃区，<ref name="简介"></ref>更获得不少赞誉，《华尔街日报》就曾赞美其为“世界上最美味的桃子”。<ref>[http://chinese.wsj.com/gb/20090828/trv085104.asp Stan Sesser，〈世上最美味的桃子〉，《华尔街日报》中文网（2009年8月28日）。]</ref>

== 品种 ==

阳山水蜜桃每年6月中旬至8月下旬先后陆续上市，最迟熟的到9月上旬上市，共计有20多个品种，早熟品种主要有春蕾、早花露、雨花露、银花露；中熟品种主要有白凤、朝晖、阳山84-2；晚熟品种主要有湖景蜜露、阳山蜜露、白花、阳山84-1/3、迟圆蜜等。其中早熟的“雨花露”、中熟的“白凤”、晚熟的“白花”三个品种质量最高、销售最畅。<ref name="资料"></ref>

== 辨别 ==

无锡阳山水蜜桃因产量关系，每年桃子上市后就会出现不少仿冒者。正宗阳山水蜜桃的辨别，可总结为一看、二闻、三掰、四尝。

正宗的阳山水蜜桃成熟后并不如常见桃子般带有大量红晕，而是呈乳白色或乳黄色带少量红晕，同时阳山水蜜桃相较仿冒桃小，且外形无仿冒桃漂亮。阳山水蜜桃香味浓郁，更是远超他处，十分明显。此外，到桃子成熟期后，仿冒桃可用力一掰两半，阳山水蜜桃则极难掰开。若加以品尝，则更见区别，即仿冒桃果肉偏硬、带有酸味。<ref>[http://www.js.xinhuanet.com/2013-08/01/c_116775873.htm 〈阳山水蜜桃减产“李鬼”钻起空子 辨别有秘籍〉，《江南晚报》（2013年8月1日）。] {{webarchive|url=https://web.archive.org/web/20140904204107/http://www.js.xinhuanet.com/2013-08/01/c_116775873.htm |date=2014年9月4日 }}</ref>

一般而言，只有在无锡阳山镇桃园及无锡市各大型超市内，方可购买到正宗阳山水蜜桃。

== 轶事 ==

* 关于无锡水蜜桃之起源，民间有传说谓是[[孙悟空|孙悟空]]偷了王母娘娘蟠桃园中的仙桃，返回花果山途中，火眼金晴见到了阳山风景秀丽，不免多看了两眼，怀中却有只仙桃滚落下来，被阳山一位老农捡到，吃了桃肉，随手将桃核种入土中。老农登仙而去，仙桃桃核破土发芽，长成桃树，结成了水蜜桃。<ref>王立人主编，《无锡山水》（凤凰出版社，2009年），页170。</ref>

* 虽然阳山水蜜桃年年仿冒者甚多，但由于名气实在太大、供不应求，阳山桃农一点也不担心仿冒桃带来的危害，甚至有桃农表示买到假的就算“运气不好了”，真可谓是“皇帝的女儿不愁嫁”。<ref>[http://www.wxrb.com/node/kandian/2012-7-27/2H8KH0210331522.html 〈阳山水蜜桃做假手法形形色色 地方名牌保护存漏洞，〉，《江南晚报》（2013年7月27日）]</ref>
