'''差春騷府'''<ref name="zhou">{{cite book|title=[[世界地名翻译大辞典|世界地名翻译大辞典]]|author=周定国|publisher=中国对外翻译出版公司|year=2008|isbn=978-7-5001-0753-8|page=1233}}</ref>（{{thai|t=จังหวัดฉะเชิงเทรา|r=Changwat Chachoengsao}}，{{IPA-th|t͡ɕāŋ.wàt t͡ɕʰàʔ.t͡ɕʰɤ̄ːŋ.sāw}}）一譯'''彩瓊騷府'''，是[[泰王國|泰王國]]東部的一個府。華人又稱其為'''北柳府'''（[[潮州話|潮州話]]：{{IPA|[p˭ǎ li̯ù]}}）。其相鄰的府有[[巴真府|巴真府]]、[[沙繳府|沙繳府]]、[[春武里府|春武里府]]、[[莊他武里府|莊他武里府]]、[[曼谷|曼谷]]、[[巴吞他尼府|巴吞他尼府]]、[[那空那育府|那空那育府]]，西南隅瀕[[泰國灣|泰國灣]]。如今擁有人口65萬<!--，[[府尹|府尹]]為安濃·蓬南若（Anon Phromnarot）-->。

== 地理 ==
* 差春騷府距[[首都|首都]][[曼谷|曼谷]]約為100公里。其疆域北連[[巴真府|巴真府]]和[[那空那育府|那空那育府]]；南毗[[春武里府|春武里府]]和[[莊他武里府|莊他武里府]]；東瀕[[沙繳府|沙繳府]]，西臨[[曼谷|曼谷]]和[[沙沒巴干府|沙沒巴干府]]。
* 差春騷府的大部份為低窪平原，土地肥沃，山林較少，港汊縱橫交錯，大都流入'''挽巴功河'''，是一著名的魚米之鄉。
* 差春騷府的西部為[[挽芭空河|挽芭空河]]（另一[[中文|中文]]譯名為「'''挽巴功河'''」，'''Bang Pa Kong River'''）及低地平原，用來廣種[[稻米|稻米]]。東部為低[[丘陵|丘陵地帶]]，平均海拔高度約為100公尺。

== 行政劃分 ==

== 風景名勝 ==

* 爍石山（Khao Hin Sorn）
* 順通-沃漢寺佛祖紀念碑（Phraya Sri Sunthorn Voharn Monument）
* 哇銷湯窩拉喃窩拉威韓寺：位於北柳府署的西南方約2公里處，內供奉一尊被認為是最顯赫的「銷湯佛像」，每年的[[春節|春節]]、5月、12月，北柳府及附近民眾均在此舉行隆重的禮佛盛會。
* 考欣桑山：位於拍農沙拉堪縣，由石頭疊成的小山，佈滿天然的麟駒怪石和樹木，一片大自然美景。
* 哇波挽卡寺：位於挽卡縣，寺內供有佛像，附近有棵大樹，吸引成千上萬隻[[蝙蝠|蝙蝠]]棲居在此，以水果作食品，吸引無數遊客前往觀賞。
* 龍福寺：位於北柳直轄縣，為[[中國|中國]]式建築藝術風格，是[[大乘佛教|大乘佛教]]場所，內有精美雕刻。為[[拉瑪五世|拉瑪五世]]的敕封佛寺。
* 哇烏提拍咖喃寺：位於「龍福寺」附近，且形狀類似上述寺廟，但規格較小， 同為[[大乘佛教|大乘佛教]]的佛寺。
* 差春騷城堡：位於差春騷直轄縣的市區內，建於[[拉瑪三世|拉瑪三世]]時代的1834年，為一古代城堡，現已改為監牢。
* 萱巴拉咖廷森林公園：位於沙南猜曲縣，面積達297平方公里，是研究大自然森林的好場地。
* 遊「挽巴功河」：可在北柳市場前租船出遊，沿途欣賞兩岸風景，如有果園、寺廟、古跡等等，終點站為挽卡縣的「哇波挽卡寺」，寺內有棵大樹，棲居著數十萬隻[[蝙蝠|蝙蝠]]。
