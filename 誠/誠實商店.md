'''誠實商店'''是[[中華民國|中華民國]]一種特殊的小型[[商店|商店]]<ref>[http://www.epochtimes.com/b5/12/11/7/n3724405.htm%E5%90%B3%E9%B3%B3%E8%AA%A0%E5%AF%A6%E5%95%86%E5%BA%97%E9%96%8B%E5%BC%B5-%E9%9D%92%E8%BC%94%E6%9C%83%E4%B8%BB%E5%A7%94%E4%B8%BB%E6%8C%81%E5%89%AA%E7%B6%B5.html 吳鳳誠實商店開張 行政院青輔會主委主持剪綵]</ref>，店內沒有店員，顧客取貨後，自行於投錢箱給錢找錢，考驗著人性與商家顧客互信買賣交易。多數業主因不敷成本而歇店，少數不以營利為目的之商家反倒遠近馳名，這類商家評價良好，但利潤不多有虧損風險，是個叫好不叫座之商家。而這類商店在[[中華民國|中華民國]]、[[中國大陸|中國大陸]]（無人售貨商店）、[[日本|日本]]（無人商店）、[[菲律賓|菲律賓]]（誠信商店）都有開設，雖名稱不同，但都有異曲同功之妙。

== 特性 ==
誠實商店與傳統的商店差異很大。首先，顧客於誠實商店取貨後，自行投錢找錢，使購物過程更便利，也因無店員之緣故，其規模也相對變小。另外，誠實商店主張信賴顧客原則，因此不設監視系統。誠實商店亦無店員站崗，也不擾客做商品介紹，而為方便讓顧客了解價錢與投錢找錢，貨物多用標價，少用條碼識別，所以，誠實商店不設[[收銀機|收銀機]]。

誠實商店並不會售賣高檔價商品，其售賣的貨品包括零嘴小吃、飲料、冰品、童玩、飾件、書籍、二手物、環保藝品等平價商品。誠實商店的其中一個特處就是免費茶水供應與提供書報閱讀休憩。

== 型態 ==
誠實商店的型態主要分為三種：回饋性、公益性和觀光景點性。回饋性誠實商店是以機動性攤車設置，多在公園等戶外休閒場所販賣冰品、飲料、零嘴，盈餘較少，甚而虧損，但業主多不太計較。而公益性誠實商店多由學校兼設，老師與學生捐物贊助，盈餘較多，所得用於協助清寒學生。<ref>[http://www.cna.com.tw/News/aEDU/201211070218-1.aspx 吳鳳科大誠實商店助清寒學生]</ref><ref>{{Cite web |url=http://www.dk101.com/index.php/viewnews-44659.html |title=亞洲大學誠實商店開幕　誠實傳愛心助清寒學生 |access-date=2012-12-24 |archive-url=https://web.archive.org/web/20160304062933/http://www.dk101.com/index.php/viewnews-44659.html |archive-date=2016-03-04 |dead-url=yes }}</ref><ref>[http://blog.xuite.net/canmag2001/blog/51926876-%E9%8A%98%E5%82%B3%E8%AA%A0%E5%AF%A6%E5%95%86%E5%BA%97+%E4%B8%80%E5%B9%B4%E6%A5%AD%E7%B8%BE%E4%BA%94%E8%90%AC!　銘傳誠實商店　一年業績五萬]</ref><ref>[http://sites.cjcu.edu.tw/slc/page_B0404001.html　長榮有品誠實商店]</ref><ref>[http://www.uonline.nccu.edu.tw/index_content.asp?sn=3&an=6613　誠實商店　付錢由心證-聖約翰科大]{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref><ref>[http://www.libertytimes.com.tw/2012/new/apr/10/today-center13.htm　五權社區大　開誠實商店　義賣助溫飽弱勢生]</ref>另觀光景點性誠實商店，則是主售懷舊、人文、環保、藝飾物品，加以景點吸引人潮，盈餘穩定，兼能推展觀光。<ref>[http://www.ettoday.net/news/20121121/129927.htm 老外遺忘相機在台東誠實商店　七天仍原地沒人動-台東鹿野76商店]</ref><ref>[http://blog.xuite.net/h8305/happylife/37749587-%E5%8F%B0%E4%B8%AD%E6%A5%93%E6%A8%B9%E7%A4%BE%E5%8D%80...%E8%AA%A0%E5%AF%A6%E5%95%86%E5%BA%97(%E6%A6%AE%E5%88%A9%E5%95%86%E5%BA%97 楓樹社區誠實商店-台中市南屯榮利商店]</ref><ref>[http://enews.ccu.edu.tw/modules/news/article.php?storyid=6313　販賣誠實與良心　無人收前錢商店-嘉義竹崎明海良心站]{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>

這類商店在中國大陸、日本、菲律賓也有開設。如中國大陸在九八抗洪時，江西九江永修縣有一商戶開設了無人售貨商店，供抗洪官民購物用<ref>[http://jx.people.com.cn/BIG5/190289/12342279.html 中國大陸-無人售貨商店]</ref>。在日本群馬縣伊香保神社有專賣護身符的無人商店<ref>[http://translate.google.com/translate?hl=zh-TW&langpair=en%7Czh-TW&u=http://www.dannychoo.com/post/en/26044/&ei=ZULaUMjuEMTJkgWutYCwAQ 日本-無人商店]</ref>。而菲律賓也有開在Navotas市的兩家誠信商店<ref>[http://translate.google.com/translate?hl=zh-TW&langpair=en%7Czh-TW&u=http://nel-ceo.blogspot.com/2011/06/honesty-store-store-without-keeper.html&ei=3lXaUN7pFqT5mAX88ICQBg　菲律賓-誠信商店]</ref>，和設在丹巴群島的誠信咖啡店<ref>[http://translate.google.com/translate?hl=zh-TW&langpair=en%7Czh-TW&u=http://www.flickr.com/photos/theworldismycanvas/2411955835/　菲律賓-誠信咖啡店]</ref>，這些商店雖名稱不同，但都有異曲同工之妙。

== 優缺點 ==
誠實商店的優點推廣誠實美德，去討價與節帳時間。而其缺點則為商品易碰損、遭竊，顧客如結帳不實，收支不易平衡。

== 評價 ==
誠實商店評價良好，但利潤不多有虧損風險，是個叫好不叫座之商家，即贊同此商業行為人多，想要附合開店之人少。中興大學教授李宗儒認為誠實商店的商品應多融入地方產品特色，或是從歷史上相關誠實的名人或故事開發周邊產品，比較容易吸引客群，如再輔以藝文、環保、公益之推廣來淨化人心，成效更佳。<ref>[http://www.nchu.edu.tw/news-detail.php?id=18880　只看不買　誠實商店撐不住]</ref>
