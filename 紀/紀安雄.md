'''紀安雄'''（{{jpn|j=紀安雄|hg=き の やすお|rm=Ki no Yasuo}}，{{bd|822年||886年|7月3日|catIdx=Ki no Yasuo}}）是[[日本|日本]][[平安時代|平安時代]]前期貴族和[[儒學者|儒學者]]，{{Link-ja|氏姓制度|氏姓制度|氏姓}}最初是苅田[[姓_(日本政治結構)|直]]，後來改為{{Link-ja|紀氏|紀氏|紀}}{{Link-ja|朝臣 (日本)|朝臣|朝臣}}，父親是{{Link-ja|苅田種繼|苅田種継}}。官位為[[從五位|從五位]]上[[周防國|周防守]]。

== 出自 ==
苅田氏（苅田首）祖籍為[[讚岐國|讚岐國]][[豐田郡_(香川縣)|苅田郡]]（現[[香川縣|香川縣]][[觀音寺市|觀音寺市]]和[[三豐市|三豐市]]一帶），是以[[武内宿禰|武内宿禰]]之孫田鳥宿禰為祖先，自稱為{{Link-ja|紀氏|紀氏}}一族的[[豪族|豪族]]<ref>大田亮『姓氏家系大辞典』[[角川書店|角川書店]]、1963年</ref>。安雄的父親是讚岐的[[國人|國人]]，擔任從五位下{{Link-ja|明經博士|明経博士|助教}}的苅田種繼。

== 經歷 ==
安雄最初是{{Link-ja|明經道|明経道|明經得業生}}，後來在[[天安_(文德天皇)|天安]]2年（858年）於明經試及第，升至[[從六位|從六位]]下，出任{{Link-ja|大學寮|大学寮|直講}}。天安3年（859年），安雄接替辭任的廣宗安人，出任存問兼領渤海客使，負責接待[[渤海使|渤海使]]的事宜。[[貞觀_(清和天皇)|貞觀]]4年（862年），安雄與一族的氏雄和今雄等人，將根據地從苅田郡移至左京（[[京都府|京都府]][[京都市|京都市]][[左京區|左京區]]）<ref>《[[日本三代實錄》貞観4年5月13日条</ref>。貞觀5年（863年），安雄升至{{Link-ja|外位]]下，並且與其他有學識的[[公卿|公卿]]大夫等一同編撰{{Link-ja|三代格式|三代格式|貞觀格式}}。

期間，作為儒學者的安雄在[[釋奠禮|釋奠禮]]上，分別講解了《[[毛詩|毛詩]]》（貞觀2年（860年））<ref>『日本三代実録』貞観2年2月6日条</ref>、《{{Link-ja|御注孝經|御注孝経}}》（貞觀4年（862年））<ref>《日本三代實錄》貞観4年8月11日条</ref>和《[[周易|周易]]》（貞觀8年（866年））<ref>『日本三代実録』貞観8年2月1日条</ref>。

貞觀9年（867年）正月，安雄升至{{Link-ja|內位|内位}}的[[從五位|從五位]]下，並且在同年11月上奏自己為[[武内宿禰|武内宿禰]]後裔，並且改姓{{Link-ja|紀氏|紀氏|紀}}{{Link-ja|朝臣 (日本)|朝臣|朝臣}}。貞觀11年（869年），他獲任命為[[勘解由使|勘解由次官]]兼[[下野國|下野介]]。同年4月，安雄與[[大納言|大納言]]{{Link-ja|藤原氏宗|藤原氏宗}}等一同完成格式編寫，向[[清和天皇|清和天皇]]呈上{{Link-ja|貞觀格|貞観格}}<ref>《日本三代實錄》貞観11年4月13日</ref>，及後於貞觀13年（871年）再呈上{{Link-ja|貞觀式|貞観式}}<ref>《日本三代實錄》貞観13年8月25日</ref>。貞觀16年（874年），他升至從五位上。

其後，安雄出任{{Link-ja|主計寮|主計寮|主計頭}}後在貞觀19年（877年）轉為擔任地方官，就任{{Link-ja|武藏國司|武蔵国司|武藏守}}。在其執政下，凡事從簡，並且重視施予恩惠，官人和民眾均感到非常滿足<ref name="a">《日本三代實錄》仁和2年5月28日条</ref>。武藏守任期結束後，安雄重返[[平安京|平安京]]，並且在[[元慶|元慶]]6年（882年）出任{{Link-ja|鑄錢司|鋳銭司|鑄錢長官}}兼[[周防國|周防守]]，但再未能如擔任武藏守時一樣獲得好評<ref name="a" />。[[仁和_(光孝天皇)|仁和]]2年5月28日（886年7月3日），安雄死去，享年65歲。最終官位是前周防守從五位上。

== 人物 ==
安雄自幼學業成績便備受好評。性格優哉悠哉，對凡事也不會有什麼意見。他精通[[經學|經學]]的同時，修飾美化[[詩歌|詩歌]]和文章的字詞亦駕輕就熟，[[重陽|重陽]]時曾經作為[[文人|文人]]而獲召見<ref name="a" />。
