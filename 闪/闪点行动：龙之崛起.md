{{vgname|闪点行动：龙之崛起|en=Operation Flashpoint: Dragon Rising|cn=闪点行动：龙之崛起|tw=閃擊點行動：龍之崛起}}是一款第一人称军事模拟游戏，是著名的电脑游戏《[[閃擊點行動：冷戰危機|閃擊點行動：冷戰危機]]》的续作。此游戏由[[EGO_Engine|EGO Engine]]引擎開發，是由本身用來製作[[赛车游戏|赛车游戏]]的封閉式引擎（使用[[EGO_Engine|EGO Engine]]引擎開發的赛车游戏，也就是玩家們熟悉的、以《[[科林麥克雷拉力賽：塵埃|科林麥克雷拉力賽：塵埃]]》、《[[超级房车赛：起点|超级房车赛：起点]]》等系列游戏），來打造可以在虛擬世界中能現實所有環境大自然上支持的游戏。开发及出版的工作全部由[[Codemaster|Codemaster]]公司推行，续作确定日期将于2009年10月发售于[[PC|PC]]、[[PlayStation_3|PlayStation 3]]和[[Xbox_360|Xbox 360]]三個平台<ref>{{cite web |url=http://www.codemasters.com/flashpoint/ |title=存档副本 |accessdate=2006-01-15 |deadurl=yes |archiveurl=https://web.archive.org/web/20060115060535/http://www.codemasters.com/flashpoint/ |archivedate=2006-01-15 }}</ref>。

== 玩法 ==

=== 多人遊戲 ===

今作閃點行動續作中，首次加入[[多人合作模式|多人合作模式]]，允许四名玩名在游戏任務中合作進行，支持[[局域連线|局域連线]]模式，能进行[[局域連线|局域連线]]的合作模式，也支持[[互聯網連线|互聯網連线]]模式以及[[互聯網連线|互聯網連线]]的[[合作游戏模式|合作模式]]。

=== 任务编辑器 ===

闪点行动：龙之崛起是自帶任务编辑器，创建、编辑、保存以及修改战斗不同任务，编辑不同武器、載具、单位类型、战术、活跃程度以及也可以配置电脑[[AI|AI]]。[[Xbox_360|Xbox 360]]版以及[[PS3|PS3]]版版本遊戲不會自帶任务编辑器。

== 游戏故事 ==

今作故事将是位於在[[库页岛|库页岛]]的海域，[[日本|日本]]北边的一座游戏虚拟岛屿Skira，描述[[俄罗斯|俄罗斯]]与[[中国|中国]]处于一触即发的紧张局势。主张在[[天然资源|天然资源]]所有权的[[中国|中国]]，在某一天[[中国|中国]]突然派出[[人民解放军|人民解放军]]登上了岛屿，並开始了登陆占领。得知此事之后[[俄罗斯|俄罗斯]]要求[[中国|中方]]撤出Skira岛屿，而[[中国|中方]]则重申应岛屿原为其领土。最后两方谈判破裂，[[俄羅斯|俄羅斯]]便向[[美國|美國]]請求援助，[[美国|美国]]派出驻扎在[[硫磺岛|硫磺岛]]的[[两栖攻击舰|两栖攻击舰]]前往冲突地区，战争開始爆发，並在這裏爆发一連串的激烈战争。

從這時起，玩家将扮演[[海豹部队|美国特种部队]]领导人，並阻止企图在天然資源上争夺问题而演变成国际战争。

《閃擊點行動：龍之崛起》中包含[[美國海軍陸戰隊|美國海軍陸戰隊]]和[[中国人民解放軍|中国人民解放軍]]2支迥異的武裝力量，他們有各自的車輛和武器。

== 开发 ==

[[閃擊點行動：冷戰危機|閃擊點行動：冷戰危機]](Operation Flashpoint: Cold War Crisis)由[[波希米亚互动工作室|波希米亚互动工作室]]开发(Real Virtuality引擎所有者)，并由[[Codemasters|Codemasters]]([[閃擊點行動：冷戰危機|閃擊點行動：冷戰危機]]品牌所有人)发行。闪点行动：龙之崛起不是[[閃擊點行動：冷戰危機|閃擊點行動：冷戰危機]]真正意义上的续作。

由于与原作《[[閃擊點行動：冷戰危機|閃擊點行動：冷戰危機]]》開發的[[波希米亚互动工作室|波希米亚互动工作室]]，因上次的法律纠纷，今次续作《闪点行动：龙之崛起》是由出版商[[Codemaster|Codemaster]]負責開發游戏，而波希米亚互动工作室将不会参与续作任何开发行列或相關工作。

闪点行动正统续作为[[武装突袭系列|武装突袭系列]]。
