'''莫匹罗星'''（英文：{{lang|en|'''Mupirocin'''}}）是一种外用[[抗生素|抗生素]]类[[抗感染药|抗感染药]]。

== 生物来源 ==
莫匹罗星又称假单胞菌酸A，由[[萤光假单胞菌|荧光假单胞菌]](''Pseudomonas fluorescens'')分离出。

== 药理学 ==
莫匹罗星作用在[[细菌|细菌]]体内的[[异亮氨酸—tRNA合成酶|异亮氨酸—tRNA合成酶]]与[[异亮氨酸|异亮氨酸]]的结合点上，从而阻碍[[氨基酸|氨基酸]]的合成，也消耗了细胞内的[[tRNA|tRNA]]，使[[RNA|RNA]]的合成及[[蛋白质|蛋白质]]的合成过程終止。

== 毒理学 ==
莫匹罗星对[[哺乳类|哺乳类]]动物[[异亮氨酸—tRNA合成酶|异亮氨酸—tRNA合成酶]]的亲和力很低，对人的毒性很小。

== 药代动力学 ==
== 适应症 ==
[[革兰氏阳性菌|革兰氏阳性]][[球菌|球菌]]引起的皮肤感染
* 原发性皮肤感染：[[脓包病|脓包病]]、[[毛囊炎|毛囊炎]]、[[疖肿|疖肿]]
* 继发性皮肤感染：[[湿疹|湿疹]]、[[溃疡|溃疡]]、[[皮炎|皮炎]]、[[创伤合并感染|创伤合并感染]]
有显著治疗效果

== 用法用量 ==
每日3次涂于患处，5天为一[[疗程|疗程]]

== 不良反应 ==
偶见灼烧感、刺痛、瘙痒

== 禁忌 ==
对莫匹罗星过敏者

== 注意事项 ==
* 肾功能损害者、孕妇慎用
* 不宜用于粘膜部位及伤口

== 特殊群体用药 ==
* 孕妇慎用
