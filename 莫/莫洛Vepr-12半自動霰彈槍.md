'''莫洛Vepr-12'''（{{Lang|en|Molot Vepr-12}}；{{Lang-ru|Молот Вепрь-12}}；Вепрь，中文意思：[[野豬|野豬]]）是一系列由[[俄罗斯|俄罗斯]]槍械製造商{{Link-ru|莫洛機械廠股份公司|Вятско-Полянский машиностроительный завод|莫洛武器有限公司}}（Molot Oruzhie Ltd.）所研製及生產的多用途[[半自動霰彈槍|半自動]][[霰彈槍|霰彈槍]]（[[戰鬥霰彈槍|戰鬥霰彈槍]]），為以原來的{{Link-en|卡拉什尼科夫步槍|Kalashnikov rifle}}為藍本並進行構圖，再運用比較沉重的[[RPK輕機槍|RPK]][[轻机枪|轻机枪]]的加強型[[機匣|機匣]]為基礎製造，發射3英吋麥格農[[霰彈|12鉛徑霰彈]]。

該槍可用作狩獵、運動射擊比賽（通常於{{Link-en|實戰射擊|International Practical Shooting Confederation}}比賽的公開課中使用）、自我防衛以及軍警制式武器。該槍可說是其「表親」[[Saiga-12半自動霰彈槍|Saiga-12半自動霰彈槍]]的直接的競爭對手。

== 歷史 ==
Vepr-12的早期版本是由{{Link-en|卡拉什尼科夫集團|Kalashnikov Concern|伊茨瑪希工廠}}（現稱：卡拉什尼科夫集團）生產，其後生產轉移動到{{Link-ru|莫洛機械廠股份公司|Вятско-Полянский машиностроительный завод|莫洛武器有限公司}}所在的霰彈槍進行了重新設計和大量生產。Vepr-12的設計目的是作為早已經由伊茨瑪希工廠大量生產並大受歡迎的[[Saiga-12半自動霰彈槍|Saiga-12]][[半自動霰彈槍|半自動]][[霰彈槍|霰彈槍]]的一個直接競爭對手。與Saiga-12一樣，Vepr-12也被設計為一個多功能的武器平台，並且備受使用它的[[獵人|獵人]]和專業射手的一致好評。出於考慮到這些因素，莫洛公司推出了獨特的功能，例如雙手靈巧操作的保險控制桿和槍機[[空槍掛機|保持開放]]的機構。

== 功能 ==
由於[[霰彈|12鉛徑]]霰彈藥筒與AK-47採用的[[7.62×39mm|7.62×39毫米]][[苏联|苏联]][[步枪|步枪]][[子彈|子彈]]兩種[[彈藥|彈藥]]之間尺寸的差異較大，因此拋殼口有所延長，從而使霰彈槍順利彈出彈殼而避免{{Link-en|槍械故障|Firearm malfunction|彈殼未能拋出}}的故障風險。

與莫洛公司命名為Vepr的所有步槍的設計模式很像的是，Vepr-12的[[機匣|機匣]]是由比較沉重的[[RPK輕機槍|RPK]][[轻机枪|轻机枪]]的加強型機匣為基礎所製造的。獲加強以後的機匣構造比標準型{{Link-en|卡拉什尼科夫步槍|Kalashnikov rifle}}的機匣要更為厚重。

Vepr-12還採用了獨特的{{Link-en|保險扣|Safety (firearms)}}和槍機釋放機構。雖然目前大多數型號都是使用向側摺疊式[[槍托|槍托]]，但其實Vepr-12半自動霰彈槍提供固定式槍托型號。與標準型卡拉什尼科夫步槍所採用的大型保險槓桿不同的是，Vepr-12採用的是設在霰彈槍兩側、讓雙手皆可靈巧操作的保險控制桿。

Vepr-12在卡拉什尼科夫步槍槍族之中不同尋常的另一特徵，就是其裝有的、能夠[[空槍掛機|保持槍機開放]]的機構。也就是說，發射並拋出槍內最後一發霰彈藥筒以後，槍機鎖會將後退的槍機鎖在機匣後部並且保持開放狀態。為了方便霰彈槍的快速重新裝填，槍機釋放桿位於[[扳機護瑻|扳機護圈]]的內部，從而讓使用者在抓握把手無需離開{{Link-en|手槍握把|Pistol grip}}以下釋放槍機。<ref>[http://world.guns.ru/shotgun/rus/vepr-12-e.html Modern Firearms—Vepr12 tactical / practical shotgun]</ref>

此外對於這種步槍類型而言還有的一個不尋常的特點，就是其新增了的一個[[直向插槽|彈匣插座]]，可讓彈匣插入的方式如[[AR-15自動步槍|AR-15]]般單手直接插入，而非標準型AK槍族上發現的「搖動與鎖緊」式彈匣系統（先用彈匣前端插入彈匣插槽的前卡位，再將彈匣後端完全插入彈匣插槽，並以彈匣卡筍固定），而這樣也避免了彈匣插入及連接時會出現的困難。

[[电镀|鍍]][[铬|铬]]內襯也遍及Vepr-12的內部，包括導氣箍、[[槍管|槍管]]和[[膛室|膛室]]，給予該霰彈槍優良的耐[[腐蚀|腐蚀]]性。

與其表親Saiga-12不同的是，Vepr-12裝上的[[氣動式_(槍械)|氣動式]]系統為自我調節式，可讓Vepr-12發射一般市面發售的[[霰彈|12鉛徑]]霰彈藥筒並且完成武器每發之間的射擊循環，而不用擔心會對霰彈槍造成損毀。

Vepr-12亦取消了標準型卡拉什尼科夫步槍所採用、機匣左側裝上的{{Link-en|華沙條約導軌|Warsaw Pact rail}}，並改為採用在機匣大型{{Link-en|防塵蓋|Dust cover (firearms)}}頂部所設有的[[皮卡汀尼導軌|MIL-STD-1913戰術導軌]]，用以安裝[[瞄准镜|光學瞄準鏡]]、[[紅點鏡|紅點鏡／反射式瞄準鏡]]、[[全像武器照準器|全息瞄準鏡]]、[[夜視鏡|夜視鏡]]和／或[[熱成像儀|熱成像儀]]。機匣大型防塵蓋藉由[[铰链|铰链]]連接，從而讓使用者打開和關閉防塵蓋的同時，不會影響到安裝於戰術導軌以上的光學瞄準具的[[歸零|歸零]]。

== 用途 ==
Vepr-12同時面向民用和軍用市場，應用範圍涵蓋狩獵、自我防衛和比賽用途。特別是，霰彈槍型之中又劃分出了為各個Vepr-12的目標客戶對象之一的{{Link-en|實戰射擊|International Practical Shooting Confederation}}（IPSC）等級競賽射手之間的利基型小眾的型號。憑藉其自我調節式系[[氣動式_(槍械)|氣動式]]系統，Vepr-12可以有效地發射眾多的[[霰彈|12鉛徑]]霰彈藥筒並且完成武器每發之間的射擊循環，這樣使它成為一個具有高度靈活性的[[半自動霰彈槍|半自動霰彈槍]]系武器平台。與其表親[[Saiga-12半自動霰彈槍|Saiga-12]]非常相似的是，Vepr-12非常適合裝上合適的光學系統以後於{{Link-en|飛碟射擊|Clay pigeon shooting}}比賽中使用。

另外，俄羅斯武器製造商Almaz-Antey亦以Vepr-12用作其研製的武裝[[無人機|無人機]]的內藏武器，以12發彈匣供彈，用以攔截無人機、氣球等低空飛行器。<ref>[https://dronesplayer.com/uav-drone/%E5%8D%8A%E8%87%AA%E5%8B%95%E9%9C%B0%E5%BD%88%E6%A7%8D%E6%88%90%E3%80%8C%E9%A3%9B%E8%A1%8C%E7%A5%9E%E6%A7%8D%E3%80%8D%E3%80%80%E4%BF%84%E6%96%B0%E6%AC%BE%E7%84%A1%E4%BA%BA%E6%A9%9F%E5%B0%8D%E6%8A%97/ DronesPlayer.com—半自動霰彈槍成「飛行神槍」　俄新款無人機對抗低空威脅]</ref>

== 衍生型 ==
Vepr-12亦生產出多種的衍生型，每一個型號都具有獨特的標記。

Vepr-12可於作出些許修改以後使用與Saiga-12相同的[[彈匣|彈匣]]供彈，但Saiga-12卻不能使用Vepr-12的彈匣和彈鼓。

=== ВПО-205（VPO-205） ===
VPO-205系列適用於12鉛徑霰彈「18.5×70毫米R」和12鉛徑麥格農「18.5×76毫米R」霰彈藥筒。它裝有向側面折疊型雙支撐架型折疊式[[槍托|槍托]]，配有槍托尾墊和可翻轉的托腮板。

* '''VPO-205-00'''（{{Lang|ru|ВПО-205-00}}） — 基本型號，配備了一根430毫米槍管，裝有可拆卸型[[炮口制動器|槍口制退器]]，以及在槍托折疊以下將扳機鎖定的機構的斷路保險以防止在[[槍托|槍托]]折疊以下射擊。其重量為8.59磅（3.9千克）。
* '''VPO-205-01'''（{{Lang|ru|ВПО-205-01}}） — 運動型號，配備了一根520毫米槍管，裝有永久地固定型槍口制退器，可在槍托折疊以下射擊。其重量為8.81磅（4.0千克）。
* '''VPO-205-02'''（{{Lang|ru|ВПО-205-02}}） — 競賽型號，配備了一根680毫米槍管和可互換的[[喉缩|收束器]]，統一該槍械系列的“狙擊”和“狙擊—自動”能力，可在槍托折疊以下射擊。其重量為9.25磅（4.2千克）。
* '''VPO-205-03'''（{{Lang|ru|ВПО-205-03}}） — 緊湊的軍用型號，這款型號具有[[自動霰彈槍|全自動射擊]]的能力，配備了一根305毫米槍管，包括可拆卸型槍口制退器。固定槍托時的整體長度為34.13英寸（867毫米），而槍托可折疊以後的總長度為23.66英寸（601毫米）。它在前端的兩側和底部的下方都設有短[[皮卡汀尼導軌|皮卡汀尼導軌]]，[[機匣|機匣]]頂部和前端底部則是長皮卡汀尼導軌。其重量為9.25磅（4.2千克）。
* '''VPO-205-04'''（{{Lang|ru|ВПО-205-04}}） — 配備了一根483毫米槍管和木製槍托。
* '''VPO-205C'''（{{Lang|ru|ВПО-205С}}） — 制式武器型，[[膛室|膛室]]固定為12/70，2005至2006年之間研製，旨在為[[私营保安公司|私营保安公司]]所採用。
* '''VPO-205-00-SP'''（{{Lang|ru|ВПО-205-00-СП}}） — 實戰射擊運動型號。
* '''Vepr-12 IPSC型''' — 運動型號，<ref>Михаил ТДегтярёв. [http://www.kalashnikov.ru/upload/medialibrary/23a/iwa-2011.pdf Тяга к перемене мест. Впечатления с выставки IWA-2011] // журнал «Калашников. Оружие, боеприпасы, снаряжение», № 4, 2011. стр.8-17</ref>按照實戰射擊聯合會的規定，並考慮到了射手與運動員的願望而進行修改。在設計中導入其他元素（可倒轉安裝的保險裝置、左側的{{Link-en|拉機柄|Cocking handle}}、有沒有正規的瞄準具等等），這使其操作更為方便。

=== ВПО-206（VPO-206） ===
VPO-206系列適用於12鉛徑霰彈「18.5×70毫米R」和12鉛徑麥格農「18.5×76毫米R」霰彈藥筒。它裝有可伸縮的[[塞缪尔·柯尔特|柯爾特]][[M4卡賓槍|M4]]型[[馬格普工業公司|馬格普]]滑動式[[槍托|槍托]]。

* '''VPO-206-00'''（{{Lang|ru|ВПО-206-00}}） — 基本型號，為VPO-205-00的12/70型[[膛室|膛室]]修改型。配備了一根430毫米槍管，裝有可拆卸型[[炮口制動器|槍口制退器]]。其重量為9.47磅（4.3千克）。
* '''VPO-206-01'''（{{Lang|ru|ВПО-206-01}}） — 運動型號，配備了一根520毫米槍管，裝有永久地固定型槍口制退器。其重量為9.7磅（4.4千克）。
* '''VPO-206-02'''（{{Lang|ru|ВПО-206-02}}） — 競賽型號，配備了一根680毫米槍管和可互換的[[喉缩|收束器]]。其重量為10磅（4.55千克）。

== 出口及法律地位 ==
Vepr-12廣泛地出口世界。

在俄羅斯，要取得Vepr-12霰彈槍可說是非常簡單，只需要取得一個“[[滑膛槍炮|滑膛槍]]牌”就行（霰彈槍在俄羅斯是比較容易獲得，而一個“[[步枪|步枪]]牌”就需要擁有一枝滑膛槍5年和一張[[狩獵|狩獵]][[许可证|许可证]]）。<ref>[http://www.zakonrf.info/zoboruzhii/13/ 俄羅斯聯邦槍械管制法—第13條—俄羅斯聯邦公民獲取武器的權利]</ref>但亦由於俄羅斯法律，國內型和許多出口型Vepr-12霰彈槍版本都裝上了斷路保險，當該槍的[[槍托|槍托]]折疊以下無法開火。（這是由於俄羅斯槍械管制法列明，除了[[手枪|手枪]]以外，任何人擁有全長度小於800毫米的[[長管槍械|長管]][[枪械|枪械]]並且射擊即屬違法）。<ref>[http://www.zakonrf.info/zoboruzhii/6/ 俄羅斯聯邦槍械管制法—第6條—銷售受到實施規例所限制的民用及軍用武器]</ref>

該霰彈槍也獲准進口到[[美国|美国]]。然而，Vepr-12在美国卻相對地困難得多。[[美國菸酒槍炮及爆裂物管理局|美國菸酒槍炮及爆裂物管理局]]進口管理法規規定的附送的彈匣容量不得超過5發，以及折疊式槍托（適用的衍生型）需要在槍托展開以下的位置進行[[焊接|焊接]]以保持展開。尤其是在[[纽约|纽约]]和[[加利福尼亚州|加利福尼亚州]]，因為它是一枝能夠使用可拆式[[彈匣|彈匣]]供彈的[[半自動霰彈槍|半自動]][[霰彈槍|霰彈槍]]，在原廠配置以下的Vepr-12被列為是一款“攻擊武器”（{{Lang-en|Assault Weapons}}），並因而被禁止在這些地區出售。
