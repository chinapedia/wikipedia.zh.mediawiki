'''莫桂蘭'''（{{bd|1892年|10月15日|1982年|11月3日}}）是[[嶺南|嶺南]]武術宗師[[黃飛鴻|黃飛鴻]]的第四任[[妻子|妻子]]，[[廣東|廣東]][[高州|高州]]人，童年时父母双逝，被广州开武馆的叔父收养和接养，自幼练[[南拳|南拳]][[五大门派|五大门派]]之一：[[莫家拳|莫家拳]]。

1911年，黃飛鴻在端午節龍舟賽後跟弟子去校場表演梅花樁舞獅。據聞當時19歲的莫桂蘭慕名到場觀摩，卻因為黃飛鴻一時失誤，被他脫出的鞋子擊中。莫桂蘭大怒，躍上擂台展莫家拳狂攻黃飛鴻，一打結緣。最終在莫桂蘭的叔父作主下，莫桂蘭十九歲下嫁花甲黃飛鴻，年齡相差約四十歲。<ref name="martialab170514">{{cite web |url=https://martialab.hk01.com/%E6%96%87%E5%8C%96/90942/%E5%90%91%E6%AF%8D%E8%A6%AA%E8%87%B4%E6%95%AC-%E8%A8%98%E5%8D%97%E5%8C%97%E5%85%A9%E6%AF%8D%E8%A6%AA%E6%AD%A6%E8%A1%93%E5%AE%B6 |title=向母親致敬——記南北兩母親武術家 |date=2017-05-14 }}</ref><ref>{{cite news|url=https://hk.lifestyle.appledaily.com/lifestyle/columnist/15069223/daily/article/20161204/19853998 |title=莫桂蘭怒打黃飛鴻|publisher=蘋果日報 |date=2016-12-04 }}</ref>

由於黃飛鴻當前三任妻子都是早逝，黃飛鴻自認[[剋妻|剋妻]]，且黃曾發誓不再[[續弦|續弦]]，就以莫桂蘭為[[側室|側室]]，然而實際上莫桂蘭是黃飛鴻當時唯一的[[配偶|配偶]]。<ref>{{Cite web |url=http://www.gdmm.chinanews.com.cn/article/2007/0123/article_67.html |title=黃飛鴻與高州莫桂蘭 |deadurl=yes |archiveurl=https://web.archive.org/web/20100121162326/http://www.gdmm.chinanews.com.cn/article/2007/0123/article_67.html |archivedate=2010-01-21 }}</ref><ref name="ming110421">{{cite news|url=http://dailynews.sina.com/bg/chn/chnpolitics/mingpao/20110420/14212386440.html|title=黃飛鴻第四任妻子 於灣仔開設館授徒|publisher=明報|date=2011-04-21}}</ref>

由於莫桂蘭有莫家拳的武術底子，黃飛鴻便將自己的各式洪家拳如虎鶴雙形、鐵綫拳等傳授她。莫桂蘭亦掌握了黃飛鴻的獅藝訣竅，並籌組了一個女子舞獅團，開獅藝先河。<ref name="martialab170514" />

1924年，寶芝林醫館在武裝暴亂中被焚毀，之後黃飛鴻過身，莫桂蘭生活不如意，但仍堅持傳授黃飛鴻的武藝。<ref name="martialab170514" />莫桂蘭於1936年移居[[香港|香港]]，以行醫和授拳維生，開設[[武館|武館]]授徒。1944年至1969年期間，她於[[香港島|香港島]][[灣仔|灣仔]][[告士打道|告士打道]]116號開設「黃飛鴻國術社」教授[[武術|武術]]，之後到[[筲箕灣|筲箕灣]]「黃飛鴻健身學院」任教，及後因租金上升和重建等原因，莫桂蘭多次搬遷武館，至1980年代初因習武風潮日漸衰退而結束武館，莫桂蘭於1982年11月3日安詳離世，享年90歲。<ref>{{Cite web |url=http://podcast.rthk.hk/podcast/item_epi.php?pid=716&lang=zh-CN&id=44897 |title=香港電台：香港故事-洪拳雜憶}}</ref><ref name="ming110421" />
