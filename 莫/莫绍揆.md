'''莫绍揆'''（{{bd|1917年|8月19日| 2011年|10月14日| catIdx=Mo莫}}），生于广西桂平，卒于江苏南京。数理逻辑学家，中国[[数理逻辑|数理逻辑]]教育和研究的开拓者之一，[[南京大学|南京大学]]数学系教授。

==生平==
莫绍揆，字非斯，1917年8月13日生于广西省[[桂平市|桂平县]]油麻乡大中村，三岁丧慈母。1930年入广州广雅中学，1935年考入中央大学数学系。1939年毕业于[[中央大学|中央大学]]理学院数学系，随后在中央大学留校任助教两年，1941年转[[中山大学|中山大学]]做数学系讲师，1944年两广沦陷时曾到玉林中学任英语讲师，1945年抗战胜利回中央大学做讲师。 1947年考取公费赴瑞士[[洛桑大学|洛桑大学]]，第二年转入瑞士国立高等工业学校，后法国[[巴黎大学|巴黎大学]]留学深造，研究[[数理逻辑|数理逻辑]]，师从数理逻辑大师{{tsl|en|Paul Bernays|保羅·伯奈斯}}(Paul Bernays)，后者是二十世纪数学领袖[[大卫·希尔伯特|大卫·希尔伯特]]学生、助理和亲密合作者。<ref>{{cite web |url=http://www.moszq.com/v-575-1.html |title=全国著名教授绍揆公 |publisher=莫氏宗亲网 |date=2011-02-04 |accessdate=2015-01-08 |author=莫经山 }}</ref> 

中华人民共和国成立后，1950年回国，到南京大学任教，先后担任数学系副教授、教授、博士生导师，1958年在[[孙钟秀|孙钟秀]]等协助下于创建中国第一个数理逻辑专业<ref>{{cite web |url=http://www.techcn.com.cn/index.php?doc-view-26855.html |title=孙钟秀院士 人物百科 |publisher=科技中国|date= |accessdate=2015-01-08 |author=}}</ref><ref>{{cite web|url=http://museum.nju.edu.cn/yc/renwu/szx.HTML|title=我国计算机软件的开拓者 —计算机科学家 孙钟秀|publisher=南京大学|date=|accessdate=2015-01-08|author=朱煊|deadurl=yes|archiveurl=https://web.archive.org/web/20050315063042/http://museum.nju.edu.cn/yc/renwu/szx.HTML|archivedate=2005-03-15}}</ref>，任数理逻辑教研室主任。<ref>{{cite web |url=http://d.wanfangdata.com.cn/periodical_njdxxb-sx200702001.aspx |title=莫绍揆先生简介 |publisher=南京大学学报(数学半年刊) |date=2008年5月22日 |accessdate=2015-01-08 |author=丁德成 }}</ref>

作为中国[[数理逻辑|数理逻辑]]教育和研究的开拓者之一，是中国第一批博士生导师，他在1965年出版的《数理逻辑导论》、1980年《数理逻辑初步》和1982年《数理逻辑教程》是中国最早的数理逻辑教材，既是他的讲义，也是自学青年的读本，既影响了从事数理逻辑的研究工作，也对计算机专业从事计算模型、计算理论和程序设计语言理论研究产生了很大影响。<ref>{{cite web |url=http://www.cnki.net/KCMS/detail/detail.aspx?QueryID=0&CurRec=1&recid=&filename=ZLJX200910002014&dbname=CPFDLAST2014&dbcode=CPFD&pr=&urlid=&yx=&uid=WEEvREcwSlJHSldSdnQ0SWZaL1NFOUd5Mk1wN3A0QmVsVXlub2hTTVBHUWdvd3pVNll3Vk53RVVVc0x2cXQ5Rm1nPT0=$9A4hF_YAuvQ5obgVAqNKPCYcEjKensW4IQMovwHtwkF4VYPoHbKxJw!!&v=MDI5OTV2Rnluc1VMM05KbDhVUHlIQmRyRzRIdGpOcjQ5Rlp1c09DQk5LdWhkaG5qOThUbmpxcXhkRWVNT1VLcmlmWnVO |title=纪念恩师莫绍揆先生 |publisher=中国逻辑学会 |date=2009-10-26 |accessdate=2015-01-08 |author=刘定一 }}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

业余，莫绍揆爱好桥牌和围棋，曾参加校系比赛。

莫绍揆，2011年10月14日1时40分逝世，享年94岁。遗体告别仪式于10月20日上午在南京石子岗殡仪馆举行。<ref>{{cite web |url=http://news.chinaunix.net/blog/2011/1017/1707411.shtml |title=沉痛悼念恩师莫绍揆先生 |publisher=ChinaUnix |date=2011-10-17 |accessdate=2015-01-08 |author=袁萌 }}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

==学术==
===数理逻辑===
莫绍揆在1965年出版的《数理逻辑导论》、1980年《数理逻辑初步》和1982年《数理逻辑教程》是中国最早的数理逻辑教材。1979年，中国逻辑学会成立，莫绍揆负责数理逻辑分支学科，并担任常务理事；常务理事还有[[金岳霖|金岳霖]]并任会长。
莫绍揆在1954年指出多值逻辑也不能无条件地使用概括原理，在1955年改进了[[大卫·希尔伯特|大卫·希尔伯特]]和{{tsl|en|Paul Bernays|保羅·伯奈斯}} (Paul Bernays)的命题演算公理系统。<ref>[http://app.nanjing.gov.cn/njnj/sz8_12/9/html/09Noname049.html 南京市政府网 基础科学 数理逻辑与数学基础]{{Dead link|date=2018年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

===理论计算机科学===
莫绍揆在1987年出版了《递归函数论》，1987年与[[王元_(院士)|王元]]合著《可计算性理论》，还著有《算法论》，这几本书偏向理论计算机科学，即计算模型和计算理论。
曾与[[胡世华|胡世华]]在[[递归论|递归论]]问题上有所交流和讨论。<ref>{{cite web|url=http://news.ustc.edu.cn/kdr/200611/t20061127_109941.html |title=胡世华：中国数学逻辑奠基人生 |publisher=中国科大新闻网 |date=2006-11-27 |accessdate=2015-01-08 |author=光明网}}</ref>

==奖项==
===莫绍揆奖学金===
美国宾州斯特劳兹堡州立大学（East stroudsburs university of Pennsylvania）数学系教授史念东感恩莫绍揆师恩，在[[南京大学|南京大学]]设立以莫绍揆命名的奖学金，每年捐资1000美元，奖励南京大学数学系博士研究生1-2名，每人奖励500美元，数理逻辑专业品学兼优的优先，2014年一位博士获奖。<ref>{{cite web|url=http://news.ustc.edu.cn/kdr/200611/t20061127_109941.html |title= 一位工人数学家的感恩之旅——莫绍揆奖学金在南京大学设立背后的故事 |publisher=南京大学 |date=2014-06-19  |accessdate=2015-01-08 |author=南京大学教育发展基金会}}</ref>

==家庭==
* 儿子莫宅愉，武汉华中理工大学数学系研究生，后分配在南京航空学院工作，1988年考取加拿大卡勒噶里大学，1994年获博士学位。妻子陶进，和儿子莫与非均迁居加拿大。
* 长女莫寄怡，南京大学数学科讲师，年芳34早逝。
* 次女莫宴情，1982年毕业于长春地质学院，一年后考入南京大学读研，获硕士学位，分配河海大学，1987年留学美国，获地质学硕士学位。丈夫和女儿均留居美国。
* 孙子莫与非。
