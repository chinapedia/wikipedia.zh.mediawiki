'''莫哈末哈欣胡先'''將軍（馬來語：Md Hashim bin Hussein，1946年6月2日- ）是[[柔佛|柔佛]]新山人，[[马来西亚皇家陸軍|马来西亚皇家陸軍]]第十八任總司令，上将軍銜。曾担任馬来西亞驻巴基斯坦大使，国家化学武器監管局主席等職務。

== 生平 ==
哈欣胡先出生于馬来西亞柔佛州的一個鄉下。基於家庭在于1963年考入馬来西亞皇家學院（Royal Malay College)當學員。

== 陆军学习生涯 ==
* 1964年得最高元首委任為皇家陆軍少尉。
* 1971年特優成績赴澳洲情報局受訓
* 1975年赴美国步兵學校受訓
* 1979年少校軍銜，得馬来西亚高級指揮系深造，學習一年在于1980年英國步兵軍校受訓。不断提升個人作戰指揮，軍事知識精忠於國家
* 1989年升級为皇家陆軍中校。並在馬来西亚國防大學深造同年获得英国国王学院“King's College”成為“战争學研究生”，两年學習获得“战争學”硕士學位。

== 陆軍生涯 ==
哈欣胡先 任职如下：
* 联邦森林战术指導專员 任职中尉
* 皇家軍事學院指導專员 任职上尉
* 第11皇家馬来步兵营指揮 任职少校 
* 馬来西亞武装部隊學院 任职少校
* 馬来西亞于澳洲陆軍軍官學員互換 任职中校
* 馬来西亞皇家陆軍基礎訓練學校院長 任职上校
* 联合國波斯尼亚战區維和部隊(UNPROFOR)首任指揮官 任职准将
* 協助創立馬来西亚皇家第“10旅”特种伞兵部队旅遊 首任指揮官（伞兵之父稱號） 任职准将
*馬來西亞皇家陸軍 第4步兵师师长 任职少将
* 馬来西亞國防陆軍總部 第一參謀長 任职中将
* 馬来西亞國防陸軍總部 軍官人事部 副總參謀長 任职中将
* 馬来西亞皇家陆軍第18任總司令 上将軍銜

== 軍旅生涯的逗號 ==
哈欣胡先在于2002年卸任。服役34年軍旅生活，并未賦閒。彪炳的战绩已及過往對國家的贡献，獲馬來西亞最高元首委任為“馬来西亞”驻“巴基斯坦”大使。任馬来西亞大使三年，鞏固了馬来西亞與巴基斯坦两國深交精神。

2005年回國交旨卸任大使職責。

同時馬來西亞外交部特委任將軍為國家化學武器监管局主席。在2008年哈欣胡先選擇卸甲歸田。
