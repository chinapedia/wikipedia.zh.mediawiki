'''莫斯科中央陸軍職業足球會'''（{{lang-ru|Профессиональный футбольный клуб ЦСКА Москва}}）是一家位於[[俄罗斯|俄罗斯]][[莫斯科|莫斯科]]的体育运动俱乐部[[莫斯科中央陆军体育俱乐部|莫斯科中央陆军体育俱乐部]]的一个组成部分。

== 球會歷史 ==
正式而言，莫斯科中央陸軍並不是軍方[[莫斯科中央陸軍體育俱樂部|中央陸軍體育會]]的一部份，但俄羅斯國防部作為中央陸軍的大股東之一，中央體育會也會將球會定入會中名下(參見[http://www.cska.ru/ CSKA Moscow])。

它曾经赢得了7次[[苏联|苏联]]联赛冠军（1946年、1947年、1948年、1950年、1951年、1970年、1991年），5次苏联杯赛冠军（1945年、1948年、1951年、1955年、1991年），赢得俄罗斯杯赛冠军于2002年、2005年及2006年，赢得[[俄羅斯足球超級聯賽|俄羅斯足球超級聯賽]]冠军于2003年、2005年及2006年，亚军于2002年和2004年，还有就是2004年、2006年、2007获得了[[俄罗斯超级杯|俄罗斯超级杯]]。

最近幾年，俱乐部的主要财政来源便是俄罗斯首富翁，亿万富翁[[罗曼·阿布拉莫维奇|罗曼·阿布拉莫维奇]]所掌控的[[西伯利亚石油公司|西伯利亚石油公司]]所支付的赞助费，在2005年度，甚至一度达到了1000万美元一年，这样的赞助费即使放在欧洲豪门俱乐部也能排在前几位，但他并不是中央陆军队的直接拥有者，因为同时，他还拥有曾获[[英格兰超级联赛|英格兰超级联赛]]冠军的[[切尔西足球俱乐部|-{zh-hans:切尔西; zh-hant:車路士; zh-tw:切爾西;}-]]，而根据[[欧洲足联|欧洲足联]]的相关规定，当同一个老板所拥有的多支不同的队伍，仅允许其中一支参加欧洲锦杯赛事。

莫斯科中央陆军最重要的辉煌战绩是在2005年-{zh-hans:欧洲联盟杯; zh-hant:歐洲足協盃;}-决赛赛场上一举击败[[葡萄牙足球超级联赛|葡萄牙]]的[[里斯本竞技|-{zh-hans:里斯本竞技;zh-hk:士砵亭;zh-tw:里斯本運動;}-]]从而捧得了[[歐洲足協盃|-{zh-hans:欧洲联盟杯; zh-hant:歐洲足協盃;}-]]冠军，这也是俄罗斯足球俱乐部历史上第一次获得欧洲赛事的冠军。

== 近况 ==
因为获得了06赛季[[俄罗斯足球超级联赛|俄罗斯足球超级联赛]]冠军，从而直接入围了[[2007年至2008年歐洲聯賽冠軍盃|07-08赛季欧洲冠军杯]]正赛阶段小组赛，同组的对手是[[国际米兰|国际米兰]]、[[埃因霍温|埃因霍温]]和[[费内巴切|费内巴切]]。

莫斯科中央陸軍於2010年已從[[盧日尼基體育場|盧日尼基體育場]]移師至它們的[[莫斯科中央陸軍球場|新球場]]。

在2010年3月16日，中央陸軍在加時賽憑日本球員[[本田圭佑|本田圭佑]]以3比2擊敗[[西維爾|西維爾]]後晉身該屆[[2009年至2010年歐洲聯賽冠軍盃|歐聯]]的半準決賽。但欠運地在八強賽中遇着上季意大利盟主[[國際米蘭|國際米蘭]]，以總回合2比1淘汰出局。

2013年10月23日，因為球迷[[種族歧視|種族歧視]][[亞亞·圖雷|亞亞·圖雷]]，當時他向裁判抗議但裁判不理，最終該俱樂部被罰5萬歐元及關閉部分主場。

2014年10月，歐洲足總（[[UEFA|UEFA]]）下令該隊踢2場[[無觀眾比賽|無觀眾比賽]]，原因是該隊球迷以種族歧視語彙謾罵對手的[[黑人|黑人]]球員，間接使該隊在歐冠小組賽就出局。

== 暱稱 ==
起初，中央陸軍被稱為'''[[馬廐|馬房]]'''，這大概是因為他們首個訓練基地的建築物設置於從前是[[蘇波夫宮|蘇波夫親王]]的馬房當中。<ref>[http://www.sovsport.ru/news/default.asp?id=257334 Interview] {{webarchive|url=https://web.archive.org/web/20080315224701/http://www.sovsport.ru/news/default.asp?id=257334 |date=2008-03-15 }} with [[Vladimir_Fedotov|Vladimir Fedotov]] in [[Soviet_Sport|Soviet Sport]], 2007-04-24</ref> 此稱被認為具冒犯性，後來中央陸軍被轉稱為'''群馬'''，時至現在此稱號已成為了球員和球迷廣泛使用的名稱，隨之也出現了如'''男子軍'''({{lang-ru|армейцы}})及'''紅藍軍團'''({{lang-ru|красно-синие }})等昵稱。
