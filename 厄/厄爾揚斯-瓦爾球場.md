'''厄爾揚斯-瓦爾球場'''（{{lang-sv|'''Örjans Vall'''}}）是位於[[瑞典|瑞典]][[哈爾姆斯塔德|哈爾姆斯塔德]]的足球場，建於1922年。

== 歷史 ==
厄爾揚斯-瓦爾球場於1922年7月30日完工，當時稱為「哈爾姆斯塔德運動場」（''Halmstads Idrottsplats''），由當時尚末登基的[[古斯塔夫六世·阿道夫|古斯塔夫六世]]主持揭幕，只有12,000人獲准進場觀禮，數以千計的群眾在場外聚集。同年8月球場更名厄爾揚斯-瓦爾球場用以紀念當地一所名為「聖厄爾揚斯」（''S:t Örjans''）的[[中世紀|中世紀]][[醫院|醫院]]。9月3日舉行首場[[英式足球|足球]]比賽。

厄爾揚斯-瓦爾球場除用作足球比賽外，歷年來亦曾舉辦其他運動項目。直到1980年代[[田徑|田徑]]跑道環繞球場草坪，是當地田徑會主要的比賽及訓練場地。1972年球場進行重修，建成主看台及計分牌取代原來的木構建築物。

== 國際競賽 ==
厄爾揚斯-瓦爾球場於[[1958年世界盃足球賽|1958年世界盃]]主辦兩場第一組分組初賽，分別為[[北愛爾蘭足球代表隊|北愛爾蘭]]1-0擊敗[[捷克斯洛伐克國家足球隊|捷克斯洛伐克]]及[[阿根廷國家足球隊|阿根廷]]3-1擊敗北愛爾蘭。

在[[瑞典|瑞典]]舉行的[[2009年歐洲U-21足球錦標賽|2009年歐洲U-21足球錦標賽]]，厄爾揚斯-瓦爾球場原本不在主辦球場之列，但[[布羅斯|布羅斯]]的[[萊耶華倫球場|萊耶華倫球場]]（Borås Arena）內有一所「麥斯漢堡飽店」（Max Hamburgers），但比賽卻由[[麦当劳|麦当劳]][[贊助|贊助]]而衍生[[法律|法律]]問題<small><ref>{{cite web |language=sv |url=http://www.svd.se/nyheter/inrikes/artikel_1080393.svd |title=Uefa vill stänga Max - har McDonald's som sponsor |publisher=svd.se |date=2008-05-01 |accessdate=2009-06-26 }}</ref></small>，當時希望「麥斯」在比賽期間暫時關閉以解決問題<small><ref>{{cite web |language=sv |url=http://www.svd.se/sportspel/nyheter/artikel_1331191.svd |title=Max viker sig för Uefas krav |publisher=svd.se |date=2008-06-28 |accessdate=2009-06-26 }}</ref></small>，惟最後被「麥斯」拒絕<small><ref>{{cite web |language=sv |url=https://di.se/Nyheter/?page=/Avdelningar/Artikel.aspx%3FarticleId%3D2008%255C07%255C21%255C293114%26SectionID%3Dettan%26menusection%3Dstartsidanhuvudnyheter |title=Hamburgerbråk stoppar U21-EM-spel i Borås |publisher=di.se |date=2008-07-21|accessdate=2009-06-26 }}</ref></small>，因此比賽移師到厄爾揚斯-瓦爾球場舉行<small><ref>{{cite web |language=sv |url=http://www.bt.se/sport/halmstad-tar-u21-em-fran-boras(829613).gm |title=Halmstad tar U21-EM från Borås |publisher=bt.se |date=2008-09-03 |accessdate=2009-06-26 }}</ref></small>。有三場比賽安排在這裡舉行，全部是B組的分組初賽，包括揭幕戰[[英格蘭21歲以下足球代表隊|英格蘭U21]]以2-1擊敗[[芬蘭21歲以下國家足球隊|芬蘭U21]]<small><ref>{{cite web |language=en |url=http://www.uefa.com/competitions/under21/fixturesresults/round=15086/match=304488/index.html |title=Richards gets England off the mark |publisher=[[UEFA|''uefa.com'']] |date=2009-06-15 |accessdate=2009-06-27 }}</ref></small>、[[德國21歲以下國家足球隊|德國U21]]以2-0淨勝芬蘭U21<small><ref>{{cite web |language=en |url=http://www.uefa.com/competitions/under21/fixturesresults/round=15086/match=304492/index.html |title=Clinical Germany end Finland hopes |publisher=[[UEFA|''uefa.com'']] |date=2009-06-18 |accessdate=2009-06-27 }}</ref></small>及德國U21與英格蘭U21以1-1賽和攜手出線的比賽<small><ref>{{cite web |language=en |url=http://www.uefa.com/competitions/under21/fixturesresults/round=15086/match=304496/index.html |title=Germany join England in last four |publisher=[[UEFA|''uefa.com'']] |date=2009-06-22 |accessdate=2009-06-27 }}</ref></small>，而小組另一支球隊[[西班牙21歲以下國家足球隊|西班牙U21]]則全程留在[[哥德堡|哥德堡]]的[[舊烏利維球場_(2008年)|舊烏利維球場]]作賽。
