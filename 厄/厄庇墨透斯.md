'''艾比米修斯'''（[[希腊语|希腊语]]：{{lang|el|Επιμηθέας}}，即“後見之明”）[[希腊神话|希腊神话]]中的一个[[提坦|提坦]]。
 
厄庇墨透斯是第一代提坦神[[伊阿珀托斯|伊阿珀托斯]]的儿子，[[普罗米修斯|普罗米修斯]]（即“先見之明”）的兄弟。在传说里他与普罗米修斯一起用泥土创造人类，然而古代这两个神常用做人类的象征（厄庇墨透斯代表人类的愚昧，而普罗米修斯则代表人类的聪明）。
 

== 创造万物 ==
原来在创造动物和人类的时候，厄庇墨透斯负责赋予每种动物以良好的本能，他就把勇敢赐给[[狮子|狮子]]，把快跑的能力给了[[兔子|兔子]]，把敏锐的眼力给了[[老鹰|老鹰]]，这样一个一个把他所有好的才能给动物，但他本来只是后见之明，没有考虑到他所有的创造物，到了人类的时候，什么也没剩下来给他们作为生存的依靠。所以人类既不是[[动物界|动物界]]最勇敢，又不是最快，也不是最强最凶猛，普罗米修斯看了人类很可怜，就偷偷上了[[奥林匹斯山|奥林匹斯山]]把[[奧林匹斯十二主神|十二位诸神]]的[[圣火|圣火]]拿回来给人类。后来普罗米修斯在诸神的背后教会了人类聪明和智慧，为的就是来弥补他的弟弟厄庇墨透斯的疏忽和不顾。
 
== 后代子孙 ==
厄庇墨透斯的妻子是[[潘多拉|潘多拉]]，二人育有一女[[皮拉|皮拉]]（{{lang|el|Πύρρα}}），而皮拉后来嫁與[[杜卡利翁|杜卡利翁]]（{{lang|el|Δευκαλίων}}）。在惩罚人类的[[大洪水|大洪水]]裡，唯有皮拉和杜卡利翁两个人生存下来。
 
== 科学专有名词 ==
西方[[天文学|天文学]]用他的名字来称呼距离[[土星|土星]]第五远的[[土星的天然卫星|卫星]]。
