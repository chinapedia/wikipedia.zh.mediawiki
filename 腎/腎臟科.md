{{醫學}}{{medical}}
'''腎臟科'''是[[內科學|內科學]]的一個分枝，主要處理的是[[腎|腎臟]]及[[泌尿系統|泌尿系統]]的相關問題及疾病。

== 腎臟科的範疇 ==

== 誰需要看腎臟科 ==
*常見可能與腎臟疾病相關的症狀：[[水腫|水腫]]、尿液多泡沫、[[高血壓|高血壓]]、[[貧血|貧血]]、倦怠等。
*常見須看腎臟科的問題：[[蛋白尿|蛋白尿]]、[[血尿|血尿]]、[[腎功能異常|腎功能異常]]等。
*常見須看腎臟科的疾病：[[腎衰竭|腎衰竭]]、[[泌尿道感染|泌尿道感染]]、[[電解質異常|電解質異常]]、藥物或毒物中毒、[[高血壓|高血壓]]等。

== 腎臟科常處理的疾病 ==
*[[急性腎衰竭|急性腎衰竭]]
*[[慢性腎衰竭|慢性腎衰竭]]
*[[腎病症候群|腎病症候群]]
*[[急性腎炎|急性腎炎]]
*[[泌尿道感染|泌尿道感染]]
*各種[[電解質異常|電解質異常]]
*藥物或毒物中毒的處置
*[[高血壓|高血壓]]
*[[腎臟移植|腎臟移植]]的術後照顧<ref>記者陳麗婷,"排斥抗體分析技術 突破器官移植障礙"{{cite web |url=http://www.cna.com.tw/ShowNews/Detail.aspx?pNewsID=201002020055&pType1=MD&pType0=aALL |title=存档副本 |accessdate=2010-02-02 |deadurl=yes |archiveurl=https://web.archive.org/web/20160304124737/http://www.cna.com.tw/ShowNews/Detail.aspx?pNewsID=201002020055&pType1=MD&pType0=aALL |archivedate=2016-03-04 }},[[中央通訊社|中央通訊社]],[[台北市|台北市]],20100202 12:07:57.</ref>
*[[血液透析|血液透析]]及[[腹膜透析|腹膜透析]]病患的照顧

== 腎臟科常用的診斷工具 ==
*尿液試紙檢驗及沉渣鏡檢
*尿液生化分析（尿中的蛋白定量、[[肌酸酐|肌酸酐]]、[[尿素氮|尿素氮]]及各種[[電解質|電解質]]）及血液生化分析（血中的[[肌酸酐|肌酸酐]]、[[尿素氮|尿素氮]]及各種[[電解質|電解質]]）
*影像學檢查:腎臟[[超音波|超音波]]、[[電腦斷層掃描|電腦斷層掃描]]、[[核磁共振造影|核磁共振造影]]；腎盂顯影攝影（靜脈顯影腎盂造影、前行性腎盂造影、逆行性腎盂造影）
