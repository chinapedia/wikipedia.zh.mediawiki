'''α'''（'''alpha'''），是[[Sony|Sony公司]]的[[數位單眼相機|數位單眼相機]]（DSLR）品牌。

Sony自1997年第一台[[CyberShot|CyberShot]] F1問世至今，配合自身在硬體領域上的優異技術與卓越的[[CCD|CCD]]感光元件，另外搭載[[德國|德國]][[卡爾·蔡司|卡爾·蔡司]]（Carl Zeiss）镜头，目前在全球消費性數位相機市场取得占有率第二，並和[[尼康|尼康]]、[[佳能|佳能]]、[[富士胶片|富士]]、[[柯達|柯達]]並列數位相機五大[[品牌|品牌]]{{請求來源}}。

Sony擁有優異的影像感光元件技術，卻缺乏鏡頭技術；雖然與卡爾·蔡司合作製造高階消費性數位相機，但無法與高階級的DSLR抗衡。為此，2005年7月19日，Sony發表與[[柯尼卡美能達|柯尼卡美能達]]（Konica Minolta）合作加入DSLR市場，以因應消費型數位相機市場的飽和。

2006年1月19日，Sony發佈自2006年4月1日起全面接管柯尼卡美能達影像事業部門。2006年4月20日，Sony發表DSLR品牌α（alpha）；該年6月7日，Sony正式發表其首部DSLR——α100。

== α涵義 ==
# 使用α做為名稱及商標，以沿續過去α系統的品牌印象。α（alpha）是希臘文首個字母，有著開啟及精粹的意涵存在，也象徵SONY DSLR數位影像新紀元的開始。
# SONY α Mount是[[Minolta_AF|Minolta AF]] Mount的相同設計，因此可兼容柯尼卡美能達Alpha / Maxxum / Dynax，也持續發展更高品質的α Mount鏡頭系統。

== α技術 ==
*搭載Sony自行開發「BIONZ」影像處理器
*搭載「D-Range Optimizer」動態範圍最佳化技術
*搭載「Super Steady Shot」光學防振技術
*搭載具備防塵鍍膜與防塵振動的「Anti-Dust」自動除塵系統
*2007年起，全面支援高畫質傳輸的HD 1080i技術
*2010年PMA展發表使用Exmor HD感測器的α機種，正式支援動畫拍攝能力
*2010年PMA展發表無反光鏡機種[[索尼NEX系列|NEX]]，正式跨入EVIL相機界
*2010年8月發表新系列[[SLT|SLT]]，使用固定式半透明反光鏡，實現了在動畫拍攝時也能實行相位對焦
*2013年11月發表全新A7、A7R全片幅E-Mount接口無反相機，同時也宣布將NEX整合至α系統當中，從此α系列並存著A-Mount及E-Mount。
*2014年11月發表A7 II全片幅E-Mount接口無反相機，首次在E-Mount機身導入五軸防手震技術。
*2015年6月發表A7R II全片幅E-Mount接口無反相機，將有效像素提升至4240萬像素，內置5軸防抖系統，具備399個相位檢測對焦點，並且實現了機內的4K視頻錄製功能。

== α鏡頭 ==
{{main|索尼A卡口镜头列表|索尼E卡口镜头列表}}
α鏡頭系列最初来源于[[美能达|美能达]]于1985年发表世界上第一款自动对焦的AF Mount系统。因此美能达的所有自动對焦鏡頭，都能在SONY的數位單眼相機上正常使用，同时SONY的鏡頭也在美能達的底片機和數位單眼相機上使用。

2006年内α系统的发布会上，索尼发布了19款镜头和2款增距镜，其中大部分来自于原先柯尼卡美能达的镜头。在2010年，索尼公司推出了[[索尼NEX系列|NEX]]机型，也宣告了E卡口镜头的诞生；自此，A口与E口共同构成了α镜头镜头群。
