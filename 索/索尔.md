'''索爾'''（{{lang|en|Thor}}，[[古諾爾斯語|古諾爾斯語]]：{{lang|non|Þórr}}，也譯為'''托-{}-爾'''、'''霍爾'''），[[日耳曼地區|日耳曼地區]]稱'''多納爾'''（{{lang|en|Donar}}），是[[北歐神话|北歐神话]]中負責掌管[[戰爭|戰爭]]與[[農業|農業]]的神，職責是保護諸神國度的安全與在人間巡視農作。[[北歐|北歐人]]相傳每當雷雨交加時就是索爾乘坐馬車出來巡視，因此稱呼索爾為“雷神”。另外，索爾的勇敢善戰在諸神與巨人間是非常有名的。索爾的力量相當巨大，在神話中甚至可以獨自挑戰[[巨人|巨人群]]，每當諸神被巨人們欺負或者攻擊時，只要索爾一站出來立刻就讓巨人族知難而退。除了索爾本身力量強大之外，「[[妙爾尼爾|妙爾尼爾]]」（雷神之鎚）更是讓索爾所向無敵的寶貴武器，因此與諸神敵對的巨人族們相當畏懼索爾：只要索爾在諸神的一天，巨人們幾乎不敢對諸神們輕舉妄動。

== 索爾的身世 ==
索爾的母親是[[嬌德|嬌德]]，其妻子是[[希芙|希芙]]。生父則眾說紛紜，大多數的人認為索爾是[[奧丁|奧丁]]之子，但其實這樣的觀點，也是很讓人存疑的。

大約西元一世紀時，索爾的地位比奧丁還高，在北歐的第一聖地魯薩拉神殿中，索爾的雕像就立在神殿正中央，兩邊則是[[奧丁|奧丁]]（{{lang|en|Odin}}）和[[弗雷|弗雷]]（{{lang|en|Freyr}}）。有時候，索爾也被尊為[[希臘神話|希臘神話]]中的[[宙斯|宙斯]]。而在[[日耳曼人|日耳曼人]]的習俗中，索爾的日子是一星期最重要的星期四，日耳曼人都在星期四舉行會議，所以索爾同時也是會議和契約的守護神。[[星期四|Thursday]]一詞就是由Thor得命的。

而同為戰爭之神的[[索爾|索爾]]和[[奧丁|奧丁]]，是什麼時候互換了地位？可以這麼說，索爾是古老農業社會的神，他是古老戰神，同時也掌管和農業息息相關的氣候，手中的[[雷神之鎚|雷神之鎚]]有祝福婚姻、生產、復活和安撫亡靈等力量，而索爾本人帶給人的印象是強大的、正直講信義的。這是農民的形象，是早期農耕社會中的表現。

而奧丁則是貴族的戰神，他是參謀、詩人、知識份子。同時他也是一個為達目地不擇手段的神。再看北歐的歷史，自給自足的農業社會崩潰後，人們離開土地航向海洋，追求權力、財富與冒險。隨著海盜的時代來臨，奧丁的地位漸漸的抬高了。而《[[新艾達經|新艾達經]]》的作者[[斯諾里·斯圖魯松|史洛里]]更是在作品中大力吹捧奧丁，這可能和奧丁同時也是詩歌之神的身份有關。於是，索爾的地位下滑，而奧丁從中竄起。

== 索爾的家庭 ==
索爾一共結了兩次婚，第一次是和女巨人[[雅恩莎撒|雅恩莎撒]]（Jarnsaxa），他們生下的孩子是[[曼尼與摩迪|曼尼]]（{{lang|en|Magni}}）。他是力量的象徵，是眾神中力量最大的一位。

後來他又和[[希芙|希芙]]（{{lang|en|Sif}}）女神結婚，一般認為他們有兩個孩子，兒子[[曼尼與摩迪|摩迪]]（{{lang|en|Modi}}）和女兒[[斯露德|斯露德]]（{{lang|en|Thrud}}）。摩迪是憤怒的人格化，[[斯露德|斯露德]]則是女武神之一。

除此之外，希芙另外還有一個孩子，冬神——[[烏勒爾|烏勒爾]]（{{lang|en|Ullr}}）。

== 索爾的武器與裝備 ==
[[File:Thor_in_his_chariot.jpg|thumb]]
[[File:Offering_by_Lund.jpg|thumb]]
* [[妙爾尼爾|妙爾尼爾]]（{{lang|en|Mjollnir}}）：其名帶有粉碎之意。俗稱「[[雷神之鎚|雷神之鎚]]」。

* [[雅恩格利佩爾|雅恩格利佩爾]]（Iarngreiper）/[[鐵手套|鐵手套]]（{{lang|en|iron gauntlets}}）：根據北歐神話的記載，索爾在得到妙爾尼爾之後和侏儒額外訂做的手套，此手套專門拿來搭配使用妙爾尼爾，讓索爾無論怎麼褌舞槌子都不會感到疲倦。這個寶物並沒有出現在《[[舊艾達經|舊艾達經]]》中，所以一般相信，這是《[[新艾達經|新艾達經]]》的作者[[斯諾里·斯圖魯松|史洛里]]從對現今來說已經遺失的資料中所引用之物，或是史洛里自己的創造之物。

* [[梅金吉奧德|梅金吉奧德]]（Megingjord）/力量腰帶（{{lang|en|girdle of might}}）：索爾本身擁有的寶物，只要將此腰帶繫上即可湧現比平時更加強大的力量。這個寶物同樣的也沒有出現在《[[舊艾達經|舊艾達經]]》中，所以一般相信，這是《[[新艾達經|新艾達經]]》的作者[[斯諾里·斯圖魯松|史洛里]]從遺失的資料中所引用之物，或是史洛里自己的創造之物。

== 索爾的居所 ==

索爾的居所是[[畢爾斯基爾尼爾|畢爾斯基爾尼爾]]（{{lang|en|Bilskirnir}}），又稱為[[閃電宮|閃電宮]]。《[[詩體埃達|詩體埃達]]》中記載此宮座落在Thrudheim，意即「力量的世界」（{{lang|en|World of strength}}），而《[[散文埃達|散文埃達]]》中則是紀載位於[[斯羅德萬|斯羅德萬]]（{{lang|en|Thrudvang}}）平原上，意即'''力量的原野'''（Fields of strength）。

== 索爾的宿敵 ==
索爾一生中幾乎無人能敵，但是他仍有一個終生的宿敵，就是大蛇[[耶夢加得|耶夢加得]]（{{lang|non|Jörmungandr}}），[[耶夢加得|耶夢加得]]為[[洛基|洛基]]的兒子，擁有與地球同等長的巨大身軀與無比的力量，索爾曾經和牠衝突多次，最終在[[諸神的黃昏|諸神的黃昏]]（{{lang|non|Ragnarok}}）中索爾力戰耶夢加得，結果雙雙戰亡。
