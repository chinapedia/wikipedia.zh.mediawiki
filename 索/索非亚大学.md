'''索非亚大学'''（{{lang-bg|Софийски университет}}），全称“'''索非亚圣克莱门特奥赫里德大学'''”（{{lang|bg|Софийски университет „Св. Климент Охридски“}}），是[[保加利亚|保加利亚]]最古老的[[高等教育|高等学府]]，创办于1888年10月1日。学校现有1598名教学人员和25368名学生，其中[[国际学生|国际学生]]约1500人。<ref name="mission"/>[[校长|校长]]为伊万·伊尔切夫[[教授|教授]]。<ref name="rector"/>

== 历史 ==
[[File:Sof.University~1935.jpg|thumb]]
1880年，保加利亚教育部向国民议会提交了一项议案，提出创办保加利亚的第一所高等院校，并覆盖[[法学|法学]]、[[哲学|哲学]]、[[医学|医学]]、[[自然科学|自然科学]]与[[工程|工程]]等学科。1887年，教育部长颁布法令，开办高等[[教育学|教育学]]课程。<ref name="founding">{{cite web |url=http://portal.uni-sofia.bg/index.php/eng/the_university/history/founding |title=Founding |publisher=索非亚大学 |date= |accessdate=2009年11月27日 |language=en |deadurl=yes |archiveurl=https://web.archive.org/web/20100214161809/http://portal.uni-sofia.bg/index.php/eng/the_university/history/founding |archivedate=2010年2月14日 }}</ref>由于这项课程进展顺利，次年末，国民议会通过议案，在此课程的基础上创办高等院校<ref name="founding"/>，教学工作于1888年10月1日正式开始<ref name="opening">{{cite web |url=http://portal.uni-sofia.bg/index.php/eng/the_university/history/opening |title=Opening |publisher=索非亚大学 |date= |accessdate=2009年11月27日 |language=en |deadurl=yes |archiveurl=https://web.archive.org/web/20100214162016/http://portal.uni-sofia.bg/index.php/eng/the_university/history/opening |archivedate=2010年2月14日 }}</ref>。最早创办的[[学院_(大学)|学院]]是[[历史学|历史]]与[[语言学|语言学]]学院，首批学生43人，均为男性，学习的课程为历史学、[[斯拉夫语族|斯拉夫语言学]]和哲学与教育学。<ref name="opening"/>第二年，[[物理学|物理]]与[[数学|数学]]学院成立。<ref name="academic_history">{{cite web |url=http://portal.uni-sofia.bg/index.php/eng/the_university/history/academic_history |title=Academic History |publisher=索非亚大学 |date= |accessdate=2009年11月27日 |language=en |deadurl=yes |archiveurl=https://web.archive.org/web/20100214162152/http://portal.uni-sofia.bg/index.php/eng/the_university/history/academic_history |archivedate=2010年2月14日 }}</ref>

1894年通过的新大学法案正式将这所高等学校承认为大学。同年，法学院成立。20世纪初，医学、[[农学|农学]]、[[林学|林学]]、[[神学|神学]]和[[经济学|经济学]]等学科也相继创办。1905年，大学被重新命名为“索非亚圣克莱门特奥赫里德大学”。1938年，建校50周年之际，索非亚大学拥有7个学院、52个研究所、42826名全日制学生。<ref name="academic_history"/>

[[第二次世界大战|第二次世界大战]]结束后，保加利亚的高等教育体制发生了巨大变化。索非亚大学的一些学科纷纷独立出来，成为新的高等院校，包括经济学高等学院、医学院、农学院等，也有一些学科则整合到[[保加利亚科学院|保加利亚科学院]]的体系中。近年来，这些学科又重新并入索非亚大学。<ref name="academic_history"/>

== 教学机构 ==
[[File:Sofia_University_-_Chem_2.jpg|thumb]]
索非亚大学现有16个[[学院_(大学)|学院]]和3个独立的系，分别是：历史学院、斯拉夫研究学院、经典与现代语言学学院、哲学学院、法学院、教育学院、[[学前教育|学前教育]]与[[初等教育|初等教育]]学院、神学院、[[新闻学|新闻学]]与[[大众传媒|大众传媒]]学院、经济与[[工商管理|工商管理]]学院、数学与信息学院、物理学院、[[化学|化学]]学院、[[生物|生物]]学院、[[地质|地质]]与[[地理|地理]]学院、医学院，以及语言学习系、信息与教师在职培训系、[[体育|体育]]系。<ref name="faculties">{{cite web |url=http://portal.uni-sofia.bg/index.php/eng/the_university/faculties_and_departments |title=Faculties and Departments |publisher=索非亚大学 |date= |accessdate=2009年11月27日 |language=en |deadurl=yes |archiveurl=https://web.archive.org/web/20100213144714/http://portal.uni-sofia.bg/index.php/eng/the_university/faculties_and_departments |archivedate=2010年2月13日 }}</ref>

== 国际交流 ==
索非亚大学是欧洲大学协会（EUA）和欧洲首都大学网络（UNICA）的成员学校。<ref name="unica">{{cite web |url=http://www.ulb.ac.be/unica/members.html |title=Members |publisher=欧洲首都大学网络 |date= |accessdate=2009年11月27日 |language=en |deadurl=yes |archiveurl=https://web.archive.org/web/20090413141132/http://www.ulb.ac.be/unica/members.html |archivedate=2009年4月13日 }}</ref>除了通过[[欧盟|欧盟]]的伊拉斯谟（Erasmus）等项目与欧洲范围内的大学进行交流，索非亚大学还与世界上60多个国家的大学签署了合作协议。<ref name="agreement">{{cite web |url=http://portal.uni-sofia.bg/index.php/eng/the_university/international_relations/partners/agreements_for_international_cooperation |title=Agreements for International Cooperation |publisher=索非亚大学 |date= |accessdate=2009年11月27日 |language=en |deadurl=yes |archiveurl=https://web.archive.org/web/20090527004640/http://portal.uni-sofia.bg/index.php/eng/the_university/international_relations/partners/agreements_for_international_cooperation |archivedate=2009年5月27日 }}</ref>
