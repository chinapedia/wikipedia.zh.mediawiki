'''彥五瀨命'''（{{jpn|j = 彦五瀬命|hg = ひこいつせ の みこと|rm = Pikoitutse no mikoto|kyu = 彥五瀨命|hgt = }}），一般稱作'''五瀨命'''（{{jpn|j = 五瀬命|hg = いつせ の みこと|rm = Itutse no mikoto|kyu = 五瀨命|hgt = }}）是[[神武天皇|神武天皇]]（日本第1代[[天皇|天皇]]）的皇兄。

== 事績 ==
彥五瀨命跟隨弟弟[[神武東征|神武天皇東征]]，在[[河内國|河内國]]的孔舍衛坂同[[長髓彥|長髓彥]]交戰中，長髓彥用箭射中了祂。彥五瀨命認為自己是[[太陽神|太陽神]]之子，不該面對著[[太陽|太陽]]（向東）作戰，回頭（向西）卻又背離太陽，因此向南撤退。到達[[雄水門|雄水門]]時箭傷惡化，大聲咒罵敵人；最終由於傷勢過重而在[[紀國|紀國]]竈山逝世。

=== 父母 ===
* 父：[[彥波瀲武鸕鶿草葺不合尊|彥波瀲武鸕鶿草葺不合尊]]（彥波瀲尊、彥澰尊） - [[彥火火出見尊|彥火火出見尊]]之子。[[天照大神|天照大神]]之四世孫。
* 母：[[玉依姬|玉依姬]] - [[海童|海童]]之小女。彥波瀲武鸕鶿草葺不合尊之[[姨|姨]]・[[乳母|乳母]]。

彥五瀨命，是彥波瀲武鸕鶿草葺不合尊的第一子也。

=== 同母兄弟姉妹 ===
* 弟：[[稻飯命|稻飯命]] - 爲妣国而沒海爲[[鋤持神|鋤持神]]。爲[[新羅|新良國]]國主。[[新羅|新羅]]國王之祖。
* 弟：[[三毛入野命|三毛入野命]]（三毛野命） - 渡于[[常世鄕|常世鄕]]。
* 弟：[[稚三毛野尊|稚三毛野尊]]（神日本磐余彥尊、神倭天皇） - 神武天皇（日本第1代[[天皇|天皇]]）。生母[[豐玉姬|豐玉姬]]（入[[海|海]]而歸[[龍宮|龍宮]]）。

== 解說 ==
[[本居宣長|本居宣長]]在《古事記傳》裡稱「五瀨」乃嚴稻，應解釋為穀物或食物之神。

== 信仰 ==
目前主祭彥五瀨命的[[神社|神社]]主要有下列：
* {{link-ja|竈山神社}}（[[和歌山縣|和歌山縣]][[和歌山市|和歌山市]]和田）：其北面是彥五瀨命之墓。
* {{link-ja|安仁神社}}（[[岡山縣|岡山縣]][[岡山市|岡山市]]）：與稻飯命、三毛入野命共同祭祀。
