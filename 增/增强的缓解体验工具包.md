'''增强的缓解体验工具包'''（{{lang-en|'''Enhanced Mitigation Experience Toolkit'''}}，简称'''EMET'''）是[[微软|微软]]提供的一个适用于[[微软Windows|微软Windows]]的[[免費軟體|免费]]的安全工具包。它提供了一个统一的接口来开启和微调Windows的众多安全特性。它可以作为防御[[恶意软件|恶意软件]]攻击的额外一层，位于[[防火墙|防火墙]]之后，[[杀毒软件|杀毒软件]]之前。

EMET主要面向[[系统管理员|系统管理员]]，不过最新版本也支持任何运行[[Windows_10|Windows 10]]、[[Windows_7|Windows 7]]、[[Windows_8.1|Windows 8.1]]、[[Windows_Server_2008|Windows Server 2008]]、[[Windows_Server_2012|Windows Server 2012]]、Windows Server 2012 R2以及[[Windows_Vista|Windows Vista]]上的用户。安装要求包括已安装[[微软|Microsoft]] [[.NET框架|.NET framework]] 4.5。<ref>[https://www.microsoft.com/en-us/download/details.aspx?id=50766 Enhanced Mitigation Experience Toolkit System Requirements]. Microsoft.com. Retrieved 20 September 2016.</ref>较旧版本还可在[[Windows_XP|Windows XP]]上使用，但部分特性不可用。<ref>[http://krebsonsecurity.com/2013/06/windows-security-101-emet-4-0/ Windows Security 101: EMET 4.0. Krebs on Security]</ref>

该产品的[[产品生命周期|生命周期]]原定于2017年1月27日结束，但在收到一些反馈后，微软已决定延长至2018年7月31日。<ref>[https://support.microsoft.com/en-au/kb/2458544 The Enhanced Mitigation Experience Toolkit Knowledgebase Article]. Microsoft.com. Retrieved 25 December 2016</ref><ref>{{cite web|title=Microsoft EMET gets end-of-life reprieve|url=http://www.infoworld.com/article/3138967/security/microsoft-emet-gets-end-of-life-reprieve.html|website=infoworld|accessdate=2017-01-26}}</ref><ref>{{cite web|title=Windows 10 still needs EMET exploit protection, US CERT tells Microsoft|url=https://nakedsecurity.sophos.com/2016/11/25/windows-10-still-needs-emet-exploit-protection-us-cert-tells-microsoft/|website=Naked Security|accessdate=2017-01-26}}</ref><ref>{{cite web|title=Microsoft extends support for EMET security tool|url=http://www.theregister.co.uk/2016/11/04/happy_hack_hardening_emet_helmet_to_guard_users_past_win_8_death/|website=The Register|accessdate=2017-01-26}}</ref>

微软于2017年10月发布的秋季创意者更新（1709）版本包含有“Exploit Protection”功能，涵盖了EMET包含的许多功能，并提供迁移路径<ref>[https://docs.microsoft.com/windows/threat-protection/windows-defender-exploit-guard/exploit-protection-exploit-guard 应用迁移以帮助防止通过漏洞进行攻击 | Microsoft Docs]</ref>。
