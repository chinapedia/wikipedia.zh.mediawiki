'''鄭顯三'''是[[臺北市立麗山高級中學|臺北市立麗山高級中學]]籌備處籌備主任及麗山高級中學第一任退休校長，於1993年就任籌備主任，2000就任第一屆校長，於2005年自麗山退休<ref name=":0">{{Cite web|url =http://www.lssh.tp.edu.tw/lssh0424.htm|title =單元：麗山高中介紹|accessdate =2015-08-29|author =|date =|publisher =麗山高中|deadurl =yes|archiveurl =https://web.archive.org/web/20160304130424/http://www.lssh.tp.edu.tw/lssh0424.htm|archivedate =2016-03-04}}</ref>。[[國立臺南大學|台南師範大學]]（現為台南大學）48級普師科畢業、[[師範大學|師大]]教育系畢業、[[國立政治大學|政治大學]]教育研究所結業。曾任國小、初中教師；國中教務主任；台北市新民、北投、明德、士林國中校長等職<ref>{{Cite web|url = http://www.cloud.taipei.gov.tw/web_lct_getContent?AreaId=112&OrganizationId=260|title = 市政雲服務－北投區 : 各級學校 ─ 臺北市立明德國民中學|accessdate = 2015-09-09|author = |date = |publisher = }}</ref>。

1993年奉派籌備麗山高中，歷經都市計畫變更、地形地物礦坑調查與地質鑽探、校舍配置安全評估、環境影響評估及差異分析報告、水土保持計畫審查、都市設計審議等前置作業。

2000年校舍完成，擔任創校首任校長，致力發展數理科學教育，三年有成：台北市學生科展團體成績第四名、第一名及第三名；全國科展兩件第一名、一件第三名；台灣區國際科展一件第一名、兩件第二名；旺宏青年科學獎銀牌；第二屆全國高中生高溫超導創意賽銅牌獎等；全國高中生網際網路程式設計賽第五名；而體育方面的手球和射箭隊，也有多次獲獎奪牌紀錄。<ref name=":0" /><ref>{{Cite web|url = http://www2.nutn.edu.tw/gac500/webservice/96/outstanding/016/016.htm|title = 台南大學第十六屆傑出校友|accessdate = 2015-08-28|author = |date = |publisher = }}</ref><ref>{{Cite news|url = http://www.lihpao.com/?action-viewnews-itemid-56453|title = 提升學生國際競賽能力|last = 陳|first = 恆光|date = 2001-04-24|work = |accessdate = 2015-09-09}}</ref>

麗山高中校歌歌詞亦出於其手筆<ref>{{Cite web|url =http://www.lssh.tp.edu.tw/song0424.htm|title =麗山高中校歌|accessdate =2015-08-29|author =|date =|publisher =|deadurl =yes|archiveurl =https://web.archive.org/web/20160611051143/http://www.lssh.tp.edu.tw/song0424.htm|archivedate =2016-06-11}}</ref>。

2005年自麗山退休後於每年固定回校參加麗山高中校慶。

鄭顯三於退休後受聘為麗山高中校務發展委員會顧問，繼續為麗山高中服務。
