'''鄭韶婕'''（{{lang|en|'''Cheng Shao-Chieh'''}}，{{bd|1986年|1月4日||}}），出生於[[台灣|台灣]][[台北|台北]]，為[[中華民國|中華民國]]女子[[羽球|羽球]]選手<ref name="gz2010">{{cite web |url=http://www.gz2010.cn/info/ENG_ENG/ZB/ZBB101A_BD@@@@@@@@@@@@@@@ENG_number=289327.html |title=CHENG Shao Chieh - Biography |work=gz2010.cn |publisher=Guangzhou Asian Games Organizing Committee |accessdate=25 June 2011 |deadurl=yes |archiveurl=https://web.archive.org/web/20170805060316/http://www.gz2010.cn/info/ENG_ENG/ZB/ZBB101A_BD@@@@@@@@@@@@@@@ENG_number=289327.html |archivedate=2017年8月5日 }}</ref>，有「-{羽球}-精靈」的外號<ref>{{zh-tw}}{{cite news|title=全人教育體育週 「羽球精靈」鄭韶婕分享心路歷程|url=http://enews.cycu.edu.tw/Paper2/Paper.srv?cmd=detailPaper&sn=5363|newspaper=中原大學新聞網}}</ref>。2004年尚為[[臺北市私立金甌女子高級中學|金甌女中]]學生時即代表中華台北參加[[2004年夏季奥林匹克运动会羽毛球比赛|雅典奧運]]，2008年及2012年分別再度代表中華台北參加[[2008年夏季奧林匹克運動會羽毛球女子單打比賽|北京奧運]]及[[2012年夏季奧林匹克運動會羽毛球女子單打比賽|倫敦奧運]]的羽球項目。

== 簡歷 ==
鄭韶婕生於運動世家，父親鄭清輝是[[臺北市士林區社子國民小學|臺北市社子國小]]的老師和羽球教練<ref>{{zh-tw}}{{cite web|url=http://web.stps.tp.edu.tw:8080/w700/badminton/a1.htm|title=鄭清輝老師 - 臺北市社子國小羽球隊|accessdate=2017-12-31}}</ref>，家中三姊弟（姊姊[[鄭筱澐|鄭筱澐]]、弟弟[[鄭浩宇|鄭浩宇]]）自小已跟著父親學習打羽球，三人長大後都成了羽毛球選手<ref>{{zh-tw}}{{cite news|title=老爸看小婕 從小就是冒險王|url=http://www.epochtimes.com/b5/5/8/22/n1026387.htm|newspaper=自由時報|date=2005-08-22}}</ref>。

由臺北市社子國小、[[臺北市立成淵高級中學|成淵國中]]、[[臺北市私立金甌女子高級中學|金甌女中]]，至後來進入[[國立臺灣師範大學|國立臺灣師範大學]]，鄭韶婕一直也在傳統的羽球強校就讀；在此氛圍下，她在國小五年級時已贏得了人生第一個女單冠軍<ref>{{zh-tw}}{{cite news|title=鄭韶婕 像打不死的蟑螂|url=http://pr.ntnu.edu.tw/newspaper/index.php?mode=data&id=1654|newspaper=聯合晚報|date=2009-08-31}}</ref>。

2011年，在[[英國|英國]][[倫敦|倫敦]]舉辦的[[2011年世界羽毛球錦標賽|世界羽毛球錦標賽]]，在四強賽中擊敗當時世界排名第10的[[德國|德國]]選手[[朱利安·申克|朱利安·申克]]挺進決賽，而在冠軍賽中不敵當時世界排名第2的[[中國|中國]]選手[[王仪涵|王仪涵]]，獲得[[銀牌|銀牌]]，不僅寫下他個人在世羽賽的最佳成績，也與1999年男子單打銀牌的[[陳鋒|陳鋒]]並列成為台灣羽球代表隊在世羽賽的最佳成績。<ref>{{zh-tw}}{{Cite web |url=http://www.cna.com.tw/news/firstnews/201108150013-1.aspx |title=世羽奪銀 鄭韶婕：享受比賽  |accessdate=2011-08-20 |work= |publisher=中央社 |date=2011-08-15 |author=黃貞貞 }}</ref><ref>{{zh-tw}}{{Cite web |url=http://www.libertytimes.com.tw/2011/new/aug/15/today-t2.htm |title=世羽賽 鄭韶婕奪銀  |accessdate=2011-08-20 |work= |publisher=自由時報 |date=2011-08-15 |author=陳筱琳 |archiveurl=https://web.archive.org/web/20110815022610/http://www.libertytimes.com.tw/2011/new/aug/15/today-t2.htm|archivedate=2011-08-15}}</ref><ref>{{zh-tw}}{{Cite web |url=http://www.tvbs.com.tw/NEWS/NEWS_LIST.asp?no=sunkiss20110814224606 |title=羽球／〈獨家〉台羽球女將鄭韶婕　世錦奪銀第1人  |accessdate=2011-08-20 |work= |publisher=TVBS |date=2011-08-15 |author=吳亭儀 |archiveurl=https://web.archive.org/web/20110909113004/http://www.tvbs.com.tw/NEWS/NEWS_LIST.asp?no=sunkiss20110814224606|archivedate=2011-09-09}}</ref>

===倫敦奧運===
2012年7月，鄭韶婕代表[[2012年夏季奧林匹克運動會中華台北代表團|中華台北]]出戰[[英國|英國]][[倫敦|倫敦]]舉行的[[2012年夏季奥林匹克运动会|奥林匹克运动会]][[2012年夏季奧林匹克運動會羽毛球比賽|羽毛球比賽]][[2012年夏季奧林匹克運動會羽毛球女子單打比賽|女子單打]]項目，在小組賽階段先後擊敗[[2012年夏季奧林匹克運動會奧地利代表團|奧地利]]的[[西蒙娜·普鲁茨|西蒙娜·普鲁茨]]<ref>{{zh-tw}}{{cite news|title=奧運》鄭韶婕輕取奧地利　小組賽開紅盤|url=http://iservice.libertytimes.com.tw/liveNews/news.php?no=673576&type=%E9%AB%94%E8%82%B2|newspaper=自由時報電子報|date=2012-07-30|archiveurl=https://archive.is/20140305165555/http://iservice.libertytimes.com.tw/liveNews/news.php?no=673576&type=%E9%AB%94%E8%82%B2|archivedate=2014-03-05|deadurl=yes}}</ref>和[[2012年夏季奧林匹克運動會土耳其代表團|土耳其]]的[[内斯里汉·伊吉特|内斯里汉·伊吉特]]<ref>{{zh-tw}}{{cite news|title=鄭韶婕2連勝　奪女單16強門票|url=http://iservice.libertytimes.com.tw/2012/specials/olympics/news.php?no=674091&type=1&rno=1|newspaper=自由時報電子報|date=2012-07-31}}</ref>，順利晉級淘汰賽。首輪淘汰賽，她又以2比0（21-18、21-10）擊敗[[2012年夏季奧林匹克運動會新加坡代表團|新加坡]]的[[顾娟|顾娟]]晉級<ref>{{zh-tw}}{{cite news|title=羽球／羽球女單傳捷報　鄭韶婕拍落顧娟晉級前8強|url=http://www.nownews.com/n/2012/08/01/164257|newspaper=今日新聞|date=2012-08-01}}</ref>。及後，鄭韶婕對戰頭號種子、[[2012年夏季奧林匹克運動會中國代表團|中國]]的[[王仪涵|王仪涵]]，最終以0比2（14-21、11-21）落敗出局，只能追平雅典奧運八強的最佳戰績<ref>{{zh-tw}}{{cite news|title=倫敦奧運／鄭韶婕未能扳倒世界球后　於8強止步|url=http://www.ettoday.net/news/20120802/82570.htm|newspaper=ETtoday|date=2012-08-02}}</ref>。

===因傷淡出===
2012年下半年，鄭韶婕一直受[[鼠蹊|鼠蹊]]傷患影響，經檢查後證實為[[疝氣|疝氣]]。在倫敦奧運後，她隨即回台開刀，休息超過三個月後才復出參加韓國公開賽與國內的排名賽<ref>{{zh-tw}}{{cite news|title=羽球／拋開搞怪髮型　一姐鄭韶婕「頓悟」變乖乖牌|url=http://www.ettoday.net/news/20130129/158669.htm|newspaper=ETtoday|date=2013-01-29}}</ref>。

之後鄭韶婕因膝傷始終無法重回賽場，在接受[[膝蓋軟骨|膝蓋軟骨]]手術後一直處於復健狀態；直到2015年，她更因為右膝傷再度復發，要再一次開刀，只能暫時封拍。她表示，雖還想再打球，但自己已29歲，再拚國際賽難度較高，日後或許只能參加國內排名賽和業餘賽事<ref>{{zh-tw}}{{cite news|title=羽球／右膝傷復發 「羽球精靈」鄭韶婕暫封拍|url=http://news.tvbs.com.tw/sports/news-603805/|agency=TVBS|date=2015-06-18}}</ref>。
