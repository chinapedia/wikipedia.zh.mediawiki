'''鄭經嗣位之爭'''，又稱'''鄭經克臺'''或'''鄭經靖難'''，是[[臺灣|臺灣]][[明鄭王朝|明鄭王朝]]的歷史事件。西元1662年（[[永曆_(南明)|永曆]]十六年）5月至11月，首代[[延平王|延平王]][[鄭成功|鄭成功]]病薨後，鄭成功之子[[鄭經|鄭經]]與鄭成功之弟[[鄭襲_(明鄭)|鄭襲]]為了爭奪王位，展開長達約半年的內部政治、軍事鬥爭。鄭經由[[思明|思明]]（今[[福建|福建]][[廈門|廈門]]）親自[[東征|東征]][[承天府|東都承天府]]（今[[臺灣|臺灣]][[臺南|臺南]]）獲勝，嗣位[[登基|登基]]。<ref>《臺灣外紀》卷12</ref>

== 鄭經醜聞 ==
西元1661年，[[永曆帝|永曆帝]]於[[緬甸|緬甸]]遭[[吳三桂|吳三桂]]擒拿，並被[[絞刑|絞死]]於[[雲南|雲南]]，[[南明|南明]]滅亡。[[延平王|延平王]][[鄭成功|鄭成功]]人在[[臺灣|臺灣]]，聞訊後，下令[[世子|王世子]][[鄭經|鄭經]]留守[[思明州|思明州]]。然而，世子與幼弟的[[乳母|乳母]][[陳昭娘|陳昭娘]][[私通|私通]]產子（[[鄭克𡒉|鄭克𡒉]]）。此消息傳回[[東都明京|東都承天府]]（今臺灣臺南）並成為當時[[士大夫|士大夫]]間傳講的醜聞。

[[世子嬪|世子夫人]][[唐王妃_(東寧)|唐氏]]之祖父[[唐顯悅|唐顯悅]]氣憤地[[奏折|上奏]][[鄭成功|鄭成功]]世子亂倫之事<ref>《[[臺灣外紀|臺灣外紀]]》卷12：「三父八母，乳母亦居其一。令郎狎而生子，不聞飭責，反加齎賞。此治家不正，安治國乎？」</ref>。鄭成功聽聞後，十分憤怒，下令[[洪旭|洪旭]]等駐廈將領處死[[董王妃_(東寧)|董王妃]]、[[世子|王世子]][[鄭經|鄭經]]、[[陳昭娘|陳昭娘]]與所出之子。然而，在[[廈門|廈門]]的[[宗室|宗室]][[鄭泰|鄭泰]]認為不妥，於是沒有遵照鄭成功之命，諸將僅將陳昭娘處死，唯一欲遵從王命的[[周全斌|周全斌]]亦遭[[洪旭|洪旭]]捉拿。諸將抗命之事隨後傳回東都，也開始了[[臺海|臺海]]兩岸內部緊張對立的局面。<ref>{{wikisource-inline|清史稿/卷224#鄭成功|《清史稿》卷224}}</ref>

== 鄭襲自立 ==
西元1662年6月，[[鄭成功|鄭成功]]染[[傷寒|傷寒]]並發狂而薨<ref>《臺灣外紀》卷12：「成功偶感風寒……嘆曰：『吾有何面目見先帝於地下也！』以兩手抓其面而逝。」</ref>，其弟[[鄭襲_(明鄭)|鄭襲]]為其治喪，通知[[思明州|思明州]]（今[[福建|福建]][[廈門|廈門]]）的世子[[鄭經|鄭經]]，在台諸將推舉[[鄭襲_(明鄭)|鄭襲]]為「[[代理|護理]]」，暫時安撫[[東都|東都]]民心。鄭襲成為護理後，其心腹[[蔡雲|蔡雲]]、[[李應清|李應清]]、[[曹從龍|曹從龍]]、[[張驥|張驥]]等四人密謀以「世子亂倫，情急於勢，黨眾拒父。」為理由擁立鄭襲承繼先王成為新君。鄭襲同意後，四人並開始拉攏[[黃昭|黃昭]]及[[蕭拱宸|蕭拱宸]]等駐台將領，黃蕭二人一同意之，並擁鄭襲為「東都主」。

此時，[[賞勳司|賞勳司]][[蔡政|蔡政]]將先王的[[翼善冠|翼善冠]]以及[[紅色|緋色]][[袞龍袍|袞龍袍]]送至[[思明州|思明州]]。[[洪旭|洪旭]]、[[黃廷|黃廷]]、[[王秀奇|王秀奇]]等文武大臣請[[王世子|王世子]][[鄭經|鄭經]]於[[思明|思明]]宣布登基嗣位，號「世藩」。[[鄭經|鄭經]]隨後命令[[周全斌|周全斌]]為五軍[[都督|都督]]，以[[陳永華|陳永華]]為[[諮議參軍|諮議參軍]]、[[馮錫范|馮錫范]]為[[侍衛鎮|侍衛鎮]]，開始準備東征靖難事宜。

== 鄭經東征 ==
同年11月，世子軍先至[[澎湖|澎湖]]，與[[黃昭|黃昭]]、[[蕭拱宸|蕭拱宸]]等人協商不成後，率軍東征。[[鄭經|鄭經]]親自率軍於[[鹿耳門|鹿耳門]]登陸，駐軍[[承天府|東都]]的[[右虎衛|右虎衛]][[黃安|黃安]]亦率軍前來與世子會師。會合後，世子派人於[[承天府|承天府]]佈告譴責黃蕭二人，以自己為王室正統自居。<ref>《臺灣外紀》卷12：「叔姪至親，並無間言。因黃、蕭二賊陰謀不軌，乘先王賓天，遂從中構釁，假造遺言，離間骨肉，煽惑軍心。爾諸將士悉受先王數十年拳養，豈有相從作此背逆？明係脅逼之故，余自當相諒。亟宜悔過倒戈，生擒二賊！共扶王室，名垂竹帛。」</ref>[[黃昭|黃昭]]聽聞消息後，率軍至潦港與世子軍對戰，起先雖佔優勢，但[[黃昭|黃昭]]本人中流箭身亡，全軍敗退。[[黃昭|黃昭]]死後，諸將皆降。世子入主[[安平古堡|安平王城]]後，[[鄭襲_(明鄭)|鄭襲]]為保全性命，將責任全推給幾位心腹身上。<ref>《臺灣外紀》卷12：「幾為奸人離間。」</ref>隨後[[蔡雲|蔡雲]][[自縊|自縊]]，[[蕭拱宸|蕭拱宸]]、[[張驥|張驥]]、[[李應清|李應清]]以及[[曹從龍|曹從龍]]等人於[[承天府|承天府]]遭[[斬首|斬首]]示眾，族人流放。鄭襲遭軟禁，內亂盡平。

== 鄭襲降清 ==
1663年，鄭襲畏懼被殺，半夜率領[[官吏|官吏]]二百二十四人，[[士兵|士卒]]一百二十人及[[銀錢|銀錢]]、[[船隻|船隻]]、[[槍械|槍械]]等物，投奔[[清朝|清朝]]。<ref>{{cite book|author=臺灣銀行經濟研究室編輯|title=《鄭氏關係文書》|url=https://books.google.com/books?id=LwcHuAAACAAJ|year=1995年|publisher=台灣銀省文献委员会|isbn=978-957-00-5796-6|pages=1-8頁}} 以及{{cite book|author=鄭維中|title=《製作福爾摩沙: 追尋西洋古書中的台灣身影》| url=https://books.google.com/books?id=p4rTAAAAMAAJ|year=2006年| publisher=如果出版社| isbn=978-986-82416-2-6|page=202頁}}</ref>
