'''鄭丹瑞'''（'''Lawrence Cheng Tan Shui'''，{{bd|1954年|11月27日||}}），人称“'''旦哥'''”或“'''阿旦'''”，[[香港|香港]]著名的跨媒体文化人，涉猎电影、电视、电台、舞台、报纸[[专栏|专栏]]，是[[香港娱乐圈|香港娱乐圈]]、文化界的多面手及前[[無綫電視|無綫電視]]基本藝人合約藝員兼[[藝員訓練班|藝員訓練班]]導師。鄭丹瑞是由[[電台|電台]][[DJ|DJ]]出道，曾是香港最受欢迎的男DJ，并历任[[香港電台|香港電台]]節目總監、[[香港商业電台|香港商业電台]]營運總裁，其编撰的《[[小男人周记|小男人周记]]》是华语影坛至今唯一一部[[广播剧|广播剧]]，並创下多项纪录，收听率高居榜首，也是迄今为止首部原创的广播剧剧本能发展为[[小说|小说]]专栏、电视剧、电影及[[舞台剧|舞台剧]]作品。鄭丹瑞也曾制作参与过多部经典的广播剧、[[香港電視劇|电视剧]]、[[香港電影|电影]]，主持过上百台经典综艺晚会和大型节目，有“[[金牌司仪|金牌司仪]]”的美誉。

== 簡介及早年經歷 ==
鄭丹瑞1974年於[[聖保羅書院|聖保羅書院]]畢業後入讀香港浸會學院（今[[香港浸會大學|香港浸會大學]]前身）[[傳理系|傳理系]]，同時兼職為[[商業電台|商業電台]]寫[[廣播劇|廣播劇]]，是當時中文節目部的職員<ref>{{cite web|url=https://www.instagram.com/p/BcZuXragMmp/?taken-by=happyeggba|title=鄭丹瑞instagram圖片|quote=咁就四十年...|language=zh-yue}}</ref>。鄭丹瑞給人的印象是口才了得、博览群书、才華橫溢和學識豐富，早年本想當[[记者|记者]]和新闻[[主播|主播]]工作，但最後进入了[[娱乐圈|娱乐界]]。1978年大學畢業後加入[[麗的電視|麗的電視]]任職編劇，一年後升任編審編寫《[[驟雨中的陽光|驟雨中的陽光]]》等[[青春片|青春]][[愛情片|愛情劇]]。1983年加入[[香港電台|香港電台]]，初時有自己的早晨節目《音樂蛋》，他回想當年確實做得不好，後來被高層以經費不足為由，調去深宵節目《[[輕談淺唱不夜天|輕談淺唱不夜天]]》，更是半放棄狀態。後來他收到一封信件，寫着：「親愛的節目主持人：如果你不想做這個節目，請你離開，我是一個需要音樂去陪伴我通宵工作的聽眾。」他當頭棒喝，深知自己不足。於是開始寫些短劇，也不只播歌，每個小時也花點不同心思。後來，鄭丹瑞等到同一字迹的聽眾來信點歌，他就知道自己對得住大氣電波及所有倚靠電波的人。及後他也獲得廣播處長[[張敏儀|張敏儀]]的讚許，之後被調到《花喱花碌星期日》，並給他[[何嘉麗_(歌手)|何嘉麗]]和[[林姍姍|林姍姍]]做拍檔，成就之後的「三個小神仙」。<ref>{{cite news|url=http://www.metrodaily.hk/metro_news/%E6%96%B0%E8%81%9E%E5%B0%88%E9%A1%8C-%E6%97%A6%E5%93%A5%E9%81%87%E7%8F%8A%E7%8F%8A-%E5%9B%9E%E6%9C%9Bdj%E5%A4%A2/|date=2018年8月24日|title=新聞專題 旦哥遇珊珊 回望DJ夢|newspaper=''[[都市日報|都市日報]] ''}}</ref>之後他也製作與主持《[[青春交響曲|青春交響曲]]》等節目，1984年升任[[監製|監製]]。另曾担任[[香港演艺学院|香港演艺学院]]的导师以及[[無綫電視藝員訓練班|無綫電視藝員訓練班]]的校长。

== 演藝界經歷 ==
在广播界，鄭丹瑞更联同[[林姗姗|林姗姗]]、[[何嘉麗|何嘉麗]]等组成了香港首屈一指电台主持组合“三个小神仙”（也是该节目名字）。在电台时期，鄭丹瑞提携了多位日后香港首屈一指的广播界、主持界名人DJ，如[[林海峰|林海峰]]、[[卓韵芝|卓韵芝]]、[[葛民辉|葛民辉]]等等，鄭丹瑞可說是香港广播界中打造新锐的“金牌圣手”。

在电视界，鄭丹瑞有香港“[[金牌司仪|金牌司仪]]”的美誉，曾主持过百台大型节目和[[综艺节目|综艺节目]]，如《[[欢乐今宵|欢乐今宵]]》、《[[劲歌金曲|劲歌金曲]]》、[[香港电影金像奖|香港电影金像奖]]頒獎典禮、[[香港小姐|香港小姐]]競選等等，精通中英文。此外，鄭丹瑞也参演过多部脍炙人口的电视剧演出和[[编剧|编剧]]工作，如鄭丹瑞与香港“桥王”[[黎文卓|黎文卓]]合作自编自演的《[[摩登小男人|摩登小男人]]》系列[[单元剧|单元剧]]、《[[金牙大状|金牙大状]]》、《[[香港人在广州|香港人在广州]]》、《[[高朋满座|高朋满座]]》等经典电视剧。《[[摩登小男人|摩登小男人]]》更运用创新的方式，将综艺节目、情感节目、两性节目及戏剧节目等元素并融一起，创造了星光熠熠的经典。他亦擔綱影視旁白，包括電視節目以及廣告。

[[主持人|主持]]方面，鄭丹瑞独树一帜，亦庄亦谐，除在香港主持多台晚会外，更曾在[[台湾|台湾]]与[[张小燕|张小燕]]合作主持[[金马奖|金马奖]]，在[[中国内地|中国内地]]主持选美节目《[[美在花城|美在花城]]》。在编剧和策划工作上，鄭丹瑞曾创造出香港[[丽的电视|丽的电视]]同时编撰三部戏剧的纪录，在半年时间内，从编剧跃升为[[编审|编审]]。

在电影界，鄭丹瑞集“编、导、演、监制、发行、策划”于一身，是香港和[[华语电影|华语电影]]圈的著名[[監製|監製]]、[[導演|導演]]、編劇、演員和作家。鄭丹瑞曾导演过40多部电影，包括首部执导的中产喜剧《[[吴三桂与陈圆圆|吴三桂与陈圆圆]]》。鄭丹瑞最擅长观察都市中的男女关系，编撰的作品多是中产都市的恋爱故事。包括《[[小男人周记|小男人周记]]》、《[[喷火女郎|喷火女郎]]》等。鄭丹瑞也尝试过编导和监制人性阴暗题材的作品，如《[[黄蜂尾后针|黄蜂尾后针]]》、《[[阿嫂|阿嫂]]》；以及青春唯美的偶像类型电影，如《[[六楼后座|六楼后座]]》、《[[分手说爱你|分手说爱你]]》等。

幕前演出經驗方面，鄭丹瑞曾主演或[[客串|客串]]过超过60部电影，其中凭借《[[风尘三侠|风尘三侠]]》提名[[香港电影金像奖|香港电影金像奖]]和台湾[[金马奖|金马奖]]的最佳男配角。鄭丹瑞以高挑的外形、奇怪荒谬的表情和喜感的演繹，成为香港电影中，标记性的小人物演员，其出演[[漫画|漫画]]改编的电影《[[风云雄霸天下|风云雄霸天下]]》[[文丑丑|文丑丑]]一角，更成为作者[[马荣成|马荣成]]及大众心目中最佳的演绎者。

鄭丹瑞因為主演[[廣播劇|廣播劇]]《[[小男人周記|小男人周記]]》（1986年）中的男主角阿寬而成為雅痞一族的代表。這本廣播劇[[小說|小說]]搬上電影院[[銀幕|銀幕]]時，他成為電影版的男主角，並在港式都市喜劇中建立了一個在感情上游移不定的典型小男人形象。劇情描述一名廣告公司小職員跟他的兩名死黨好友在[[愛情|愛情]]、[[婚姻|婚姻]]、[[事業|事業]]與[[友情|友情]]等各方面的遭遇，主線是他嫌[[妻子|妻子]]太悶，想出去風流瀟灑找情人，結果發現不外如是。電影版當時仍是新導演的[[陳嘉上|陳嘉上]]展現了清新而感性的敘事手法，加上[[鍾楚紅|鍾楚紅]]、[[鄭裕玲|鄭裕玲]]、[[李美鳳|李美鳳]]、[[胡慧中|胡慧中]]等一群巨星影后级美女合演，頗具活色生香的娛樂趣味。  

郑丹瑞在年輕時有過人生得意的階段，在半年间從電視編劇升為編審，後來也遇到過劇本不賣座的時候。他希望成為電台DJ，結果進入[[香港電台|香港電台]]做了DJ，也曾經做過[[黃金時段|黃金時段]]的節目，但後來卻被調至主持一個凌晨2至6時的通宵節目，陷入人生低谷。鄭丹瑞於1987年跳槽到[[香港商業電台|香港商業電台]]任[[商業二台|商業二台]]DJ，遇上另外一位也在1989年跳槽到商台的同事[[林珊珊|林珊珊]]和[[何嘉麗|何嘉麗]]，成為電台節目經典<ref>[http://www.rthk.org.hk/classicschannel/main_radio_80_s3.htm 經典重溫頻道 - 經典電台頻道Classics Radio]</ref>。鄭曾任香港商業電台營運總裁。

== 其他工作 ==
鄭丹瑞也是[[香港电影导演会|香港电影导演会]]的会员、[[香港编剧家协会|香港编剧家协会]]的中坚、[[香港演艺人协会|香港演艺人协会]]的副会长，为香港和两岸电影电视产业立下汗马功劳。他也在《[[香港經濟日報|香港經濟日報]]》寫專欄，每隔幾年也會出書。

鄭丹瑞也乐于提携后辈，发掘演艺界的新星。於2012年，鄭丹瑞重返[[無綫電視|無綫電視]]並擔任第26期[[無綫電視藝員訓練班|藝員訓練班]]總司令。

== 家庭 ==
鄭丹瑞為家中為孻子，有一兄一姊，父親曾任[[銀行|銀行]][[經理|經理]]。鄭現今已婚，育有两个女儿。大女兒鄭瑤（Zaneta），幼女兒鄭珉（Amanda）。

在1993年，鄭出版過一張唱片《給最愛女兒的說話》，當中的歌曲就是他送給他的兩位女兒的禮物和告誡，如《給最愛女兒的說話》、《給女兒的信》等等；而《那夜我替女兒「R」背脊》{{NoteTag|「R」與「抓」字的粵語發音接近，「『R』背脊」就是「抓背部的癢」的意思。}}將他的慈父形象表露無遺，更憑《那夜我替女兒「R」背脊》奪得1993年[[兒歌金曲頒獎典禮|兒歌金曲]]獎項。
