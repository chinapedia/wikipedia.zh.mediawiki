'''浒山街道'''是[[中国|中国]][[浙江省|浙江省]][[慈溪市|慈溪市]]一个[[街道_(行政区划)|街道]]。西与[[余姚市|余姚市]]接壤，北与[[宗汉街道|宗汉街道]]、[[古塘街道|古塘街道]]相连，南与[[横河镇_(慈溪市)|横河镇]]相接，东与[[白沙路街道|白沙路街道]]相连。<ref name="adjust">{{cite web|url=http://www.cixi.gov.cn/art/2008/7/9/art_14354_219784.html|title=我市部分行政区划调整了|date=2008-07-09|accessdate=2010-09-25}}</ref>辖区面积9.6[[平方公里|平方公里]]，常住人口7万余人，流动人口4万余人<ref>{{cite web|url=http://cxnews.cnnb.com.cn/gb/node2/node3/node11/userobject1ai134596.html|title=浒山强化街面防控织就天网保平安|publisher=慈溪新闻网|date=2010-05-14|accessdate=2010-09-25}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，[[街道办事处|街道办事处]]驻寺山路65号。<ref name="main">{{cite web|url=http://hushan.cixi.gov.cn/col/col673/index.html|title=浒山街道人文历史|publisher=浒山街道办事处|accessdate=2010-09-25}}</ref>

== 沿革 ==
浒山原为[[明代|明]][[洪武|洪武]]二十年（1387年）为防御倭寇修筑的[[三山所城|三山所城]]所在。最初，由于所城位于浒山、乌山、眉山附近，因而称三山所城，后迁至浒山附近，改名浒山所城。民国十九年（1930年），置浒山镇。1954年县域调整后，浒山镇成为[[慈谿县|慈谿县]][[县城|县城]]所在。<ref name="main" />2008年，以三北大街、新城大道为界，划分浒山街道为浒山街道、古塘街道、白沙路街道三部分。<ref name="adjust" />

== 下辖 ==
浒山街道下辖25个社区，1个村：
* 下辖社区：楼家社区、古塘社区、新江路社区、浒西社区、东海社区、寺山路社区、金南社区、金东社区、鸣山社区、湾底社区、剑山社区、光辉社区、孙塘社区、东门社区、三碰桥社区、水南社区、虞家路社区、阳明社区、金山社区、上叶家社区、南孙塘社区、眉山社区、施山社区、东山社区、虞波社区
* 下辖村：天香桥村

== 经济 ==
浒山街道乡镇经济发达，轻纺针织与密封材料行业在中国大陆拥有重要地位。此外，石墨垫片、机电、五金、家电、塑料制品行业也是浒山街道重要的工业部门。<ref>{{cite web|url=http://www.cixi.gov.cn/art/2008/8/15/art_14523_2.html|title= 机构职能 - 各镇（街道） - 浒山街道|accessdate=2010-09-25}}</ref>
