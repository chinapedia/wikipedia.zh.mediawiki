'''何中华'''（{{bd||3月7日|}}），中国大陆男艺人<ref>{{Cite web|url=http://star.tvyugao.com/star/hezhonghua/|title=演员何中华个人资料|author=|date=|publisher=tv电视节目|language=|accessdate=|quote=|deadurl=yes|archiveurl=https://web.archive.org/web/20150110172038/http://star.tvyugao.com/star/hezhonghua/|archivedate=2015-01-10}} </ref>
，武打明星代表<ref>{{Cite web |url= http://ent.sina.com.cn/s/m/2003-05-30/1756152317.html |title= “功夫大侠”何中华：曾是歌舞厅中的卖艺人(图) |author=  |date=2003-05-30 |publisher= 新浪娱乐 |language=  |accessdate=  |quote=  }} </ref>，2009年获得“功夫—中国影视走向世界的桥梁杰出新人”荣誉。

==表演經歷==
何中华出生于吉林省磐石县，1980年考入吉林省戏剧学院修[[戏剧|戏剧]]，并利用课余时间学音乐，涉猎颇广。1987年毕业后曾在吉林省曲艺团担任键盘手，不时有机会上台唱几句。在毕业的前三个月，拍摄了第一部电视剧《[[康德第一保镖|康德第一保镖]]》，后经人介绍，又参加了电影《[[命夺黄金图|命夺黄金图]]》、《[[毒草霉|毒草霉]]》的拍摄。

1992年，电视剧《[[多情剑客无情剑|多情剑客无情剑]]》开拍，何中华受邀出任该剧的武术指导，并出演了剧中角色。以后，何中华又在《[[七种兵器|七种兵器]]》等多部武打电视剧中出任了武术指导，并招集了不少吉林省戏校的同班同学担任剧中的武术替身。1995年，金德顺导演筹拍一部20集的农村题材电视连续剧《[[苍凉后土|苍凉后土]]》，一位朋友把何中华介绍给了金导，金导决定让何中华在剧中出演一个重要角色。紧接着，金德顺导演的20集都市题材电视连续剧《[[冬去春来|冬去春来]]》，再次让何中华扮演了重要角色，而在这部剧拍完后，何中华的表演就驾轻自如多了。

1996年，香港艺人[[刘德华|刘德华]]来到长影拍摄电影《[[烽火佳人|烽火佳人]]》，恰巧何中华也在剧中出演一个飞行员的角色。如果说《苍凉后土》、《冬去春来》两部剧让何中华明白了该怎样表演外，在《烽火佳人》中，让何中华总结出了“做好人演好戏的”人生信条。刘德华的敬业精神，深深打动了何中华，这也是何中华第一次与港台明星合作，也正是这次的合作，让何中华在今后与港台明星的合作中学到了更多的做人的道理。

1998年初出茅庐的他因在《[[马永贞_(1998年电视剧)|马永贞]]》中的出色表演一炮走红<ref>{{Cite web |url= http://ent.qq.com/a/20120206/000257.htm |title= 何中华领衔《马永贞》 14年后再续英雄侠义缘 |author=  |date= |publisher= 腾讯娱乐 |language=  |accessdate=  |quote=  }} </ref>，陆续参加了《[[少年苏灿|少年苏灿]]》、《[[中华英豪|中华英豪]]》、《[[戲說乾隆_(中視)|戏说乾隆]]》、《[[香帅传奇|香帅传奇]]》、《[[短刀行|短刀行]]》、《[[回旋刀|回旋刀]]》、《[[燕子盗|燕子盗]]》、《[[机灵小不懂|机灵小不懂]]》、《[[鏡花緣傳奇|镜花缘传奇]]》、《[[少年包青天|少年包青天]]》、《[[三少爺的劍_(電影)|三少爷的剑]]》、《[[流星蝴蝶劍_(2010年電視劇)|流星蝴蝶剑]]》、《[[少年黄飞鸿|少年黄飞鸿]]》、《[[缉私要案组|缉私要案组]]》等电视剧的拍摄。在何中华的演艺历程中，有一部他认为是极具里程碑意义的电视剧——《[[三少爷的剑|三少爷的剑]]》<ref>{{Cite web |url= http://ent.sina.com.cn/v/2002-09-24/0949102651.html|title= “三少爷”何中华：要与李冰冰唱情歌(附图) |author=  |date= |publisher= 新浪娱乐 |language=  |accessdate=  |quote=  }} </ref>，他在片中挑大梁，饰演男主角三少爷谢晓峰。《三少爷的剑》是取材于[[古龙|古龙]]的武侠小说，所以何中华坦言，古龙的剑可不好耍，因为在原著中，写实的招数太少，拍摄中多靠现场演绎，相比而言，他更喜欢[[金庸|金庸]]的作品<ref>{{Cite web |url= http://ent.sina.com.cn/s/m/2003-05-30/1756152317.html |title= “功夫大侠”何中华：曾是歌舞厅中的卖艺人(图) |author=  |date=2003-05-30 |publisher= 新浪娱乐 |language=  |accessdate=  |quote=  }} </ref>。
