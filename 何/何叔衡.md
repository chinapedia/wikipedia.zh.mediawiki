'''何叔衡'''（{{bd|1876年|5月27日|1935年|2月24日}}）<ref>[http://www.zgdsw.org.cn/GB/222988/223508/14821471.html 《何叔衡》“何叔衡1876年5月27日出生于湖南省宁乡县杓子冲一个农民家庭。”]</ref>，[[表字|字]]'''玉衡'''，[[号|号]]'''琥璜'''，[[湖南|湖南]][[宁乡|宁乡]]人。[[中国共产党|中国共产党]]创始人之一。

==生平==
[[File:The_He_Shuheng_Statue.jpg|thumb]]

1913年，何叔衡就读于[[湖南省立第一师范学校|湖南省立第一师范学校]]，后在长沙[[楚怡小学堂|楚怡小学堂]]和第一师范附小任教。1918年，与[[毛泽东|毛泽东]]、[[萧瑜|萧瑜]]、[[蔡和森|蔡和森]]等发起成立[[新民学会|新民学会]]，曾任执行委员长。并于2年后，参加长沙共产主义小组、被湖南省教委派为省通俗教育馆馆长，主办《湖南通俗报》。

1921年7月，与[[毛泽东|毛泽东]]作為[[長沙|長沙]]代表参加在上海召開的[[中国共产党第一次全国代表大会|中国共产党第一次全国代表大会]]（在大會召開之前，幾位主要代表認為何不應出席大會，以某項工作緊急為理由，請何先行返湘處理，未實際出席大會。詳張國燾《我的回憶》），时年45岁。1921年8月，与毛泽东创办[[湖南自修大学|湖南自修大学]]。第一次国共合作时期，任[[国民党湖南省党部|国民党湖南省党部]]执行委员和监察委员、湖南省法院陪审员、惩治土豪劣绅特别法庭成员、《湖南民报》馆馆长等职。1927年5月，长沙“[[马日事变|马日事变]]”后，赴上海转入地下工作，与[[谢觉哉|谢觉哉]]、[[徐特立|徐特立]]等创办中共印刷机构“[[聚成印刷公司|聚成印刷公司]]”，任经理。1928年6月，赴[[苏联|苏联]][[莫斯科|莫斯科]]，参加在那里举行的[[中國共產黨第六次全國代表大會|中國共產黨第六次全國代表大會]]。9月入[[莫斯科中山大学|莫斯科中山大学]]，与徐特立、[[吴玉章|吴玉章]]、[[董必武|董必武]]、[[林伯渠|林伯渠]]等编在特别班学习。1930年7月回上海，任[[共产国际救济总会|共产国际救济总会]]和全国[[互济会|互济会]]主要负责人。

1931年秋，何叔衡进入[[江西|江西]][[中央苏区|中央苏区]]，当选为[[中华苏维埃共和国中央执行委员会|中华苏维埃共和国中央执行委员会]]委员、临时中央工作检察人民委员、内务人民委员部代理部长、临时最高法庭主席、工农监察部部长等职。1934年[[中央红军|中央红军]]主力突围[[长征|长征]]后，留守赣闽根据地坚持游击斗争。1935年2月24日，何叔衡在福建长汀水口被国民党部队包围，于突围中牺牲{{r|bio|page=16}}，终年59岁。

== 家庭 ==
何叔衡和妻子袁娥没有子女。何新九是何叔衡弟兄过继的。<ref name="fish" >{{cite web |url= http://dangshi.people.com.cn/n/2013/0609/c85037-21804434.html|title="一大"代表何叔衡：后人三代皆务农 |accessdate= 2013年6月9日|publisher=中国共产党新闻网 |work= |language=zh-hans}}</ref><ref name="bull" >{{cite web |url= http://www.crt.com.cn/news2007/News/lyrw/1369212655H4GGIB355IFJ45K7IIHH_2.html|title=“一大”代表何叔衡：后人三代皆务农 |accessdate=2013-06-09 |publisher=红色旅游网 |work= |language=zh-hans}}</ref><ref name="bee" >{{cite web |url=http://news.ifeng.com/gundong/detail_2013_06/09/26286176_0.shtml |title="一大"代表何叔衡：后人三代皆务农 |accessdate=2013年6月9日 |publisher=凤凰网 |work= |language=zh-hans}}</ref>

何新九一生育有4子3女，二儿子名叫何霞飞，何霞飞的大儿子名字不详，大儿媳名叫李小良，是一名乡村教师，何霞飞的二儿子名叫何光华，二儿媳名叫李国辉，何叔衡故居由李国辉负责看管，沙田乡人民政府每月支付给她800元，何盛明则是何叔衡的曾孙，何叔衡是[[中国共产党|中国共产党]]创始人之一，作为中国共产党创党元老，由于他1935年2月24日在福建长汀跳下悬崖牺牲，其后人五代并没有享受到父荫，仍然居住在宁乡县沙田乡，曾孙何盛明曾是[[煤矿|煤矿]]和[[钢铁|钢厂]]的矿工，年纪大了之后回乡种田，何盛明的2个儿子也是靠卖苦力挣钱，大儿子是海尔空调的安装工人，二儿子是[[中联重科|中联重科]][[机床|机床]]工人。<ref name="fish" /><ref name="bull" /><ref name="bee" />

1928年，何叔衡远赴[[苏联|苏联]]留学深造，其中1929年给儿子何新九写的家书写到：“我的人生观，绝不是想安居乡里以求善终，绝对不能为一家一身升官发财以愚懦子孙。此数言请你注意。”后来这句话成了何家的[[家训|家训]]。<ref name="fish" /><ref name="bull" /><ref name="bee" />
