'''何重建'''（{{bd|890年代||10世纪|catIdx=H}}），[[五代十国|五代十国]]时期军事人物。

== 家世 ==
何重建祖上是世居云、朔间的[[回鹘|回鹘]]别部人，祖何庆、父何怀福都是唐末[[河东节度使|河东节度使]][[李克用|李克用]]的小校，在河东军部[[太原|太原]]安家。

何重建年轻时谨厚，效力[[石敬瑭|石敬瑭]]帐下，管马厩，为奉德马军都指挥使。<ref name=HFD94>[[:s:旧五代史/卷94|《旧五代史》卷九十四]]</ref><ref name=h9k>《九国志》</ref>

== 效力后晋 ==
石敬瑭建立[[后晋|后晋]]为高祖皇帝后，何重建屡次典禁军，历任遥领驩、睦二州[[刺史|刺史]]、实领[[曹州|曹州]]刺史。<ref name=HFD94/><ref name=h9k/>

[[天福_(后晋)|天福]]七年（942年）正月，[[彰武军节度使|彰武军节度使]][[丁审琪|丁审琪]]贪暴，胡人为其所苦，部下军校贺行政与胡人勾结作乱，攻军部[[延州_(延安)|延州]]，何重建时任曹州防御使，奉命率兵救之，同、鄜二州援兵也来到，延州得救。何重建以所部兵攻延州，丁审琪遁去。二月，[[后晋高祖|后晋高祖]]以何重建暂代彰武军兵马留后，召丁审琪归朝。<ref>[[:s:资治通鉴/卷283|《资治通鉴》卷二百八十三]]</ref>何重建下车告以威福，边民安心。闰三月，被任为彰武军节度使。<ref>[[:s:旧五代史/卷80|《旧五代史》卷八十]]</ref>八年（943年）十一月，被任为[[威胜节度使|威胜节度使]]。后因避[[后晋出帝|后晋出帝]]讳，改名'''何建'''。[[开运_(后晋)|开运]]元年（944年）二月，何重建守杨刘镇，正逢[[平卢军节度使|平卢军节度使]][[杨光远|杨光远]]勾结[[契丹|契丹]]作乱，何重建被任为东南面马步都部署，率兵屯[[郓州|郓州]]、[[汶阳|汶阳]]。<ref>[[:s:资治通鉴/卷284|《资治通鉴》卷二百八十四]]</ref>五月，改任[[贝州|贝州]]永清军节度使。<ref>[[:s:旧五代史/卷82|《旧五代史》卷八十二]]</ref>十一月，改任为[[镇宁军节度使|镇宁军节度使]]兼北面行营马军右厢排阵使。<ref>[[:s:旧五代史/卷83|《旧五代史》卷八十三]]</ref>二年（945年），改[[河阳节度使|河阳节度使]]。十月，改[[泾原节度使|泾原节度使]]。在历次任上，都廉洁宽简。累次加官至检校[[太傅|太傅]]。三年（946年）正月，改[[雄武军节度使|雄武军节度使]]。<ref name=HFD94/><ref name=h9k/><ref>[[:s:旧五代史/卷84|《旧五代史》卷八十四]]</ref><ref name=SGCQ53>《十国春秋》卷五十三</ref>

== 举地归蜀 ==
[[辽朝|辽]]灭后晋，[[会同|会同]]十年（947年）正月，派使者持诏书赐后晋各藩镇，各节度使争相上表称臣，但何重建愤然对将吏们说：“我侍奉石氏二主，累次当节度使，作为人臣的荣耀也到了极点了。今日不能率兵赴难，岂可受制于契丹！”杀辽使者，遣[[秦州|秦州]]（雄武军军部）押衙[[吴崇恽|吴崇恽]]奉表，举秦、阶、成三州降于[[后蜀后主|后蜀后主]]。<ref name=HFD94/><ref name=h9k/><ref>[[:s:旧五代史/卷99|《旧五代史》卷九十九]]</ref><ref name=SS274>[[:s:宋史/卷274|《宋史》卷二百七十四]]</ref>泾原节度使[[史匡威|史匡威]]也据本镇不受辽之命。因何、史不附辽，辽势头受阻。<ref name=ZZTJ286>[[:s:资治通鉴/卷286|《资治通鉴》卷二百八十六]]</ref>[[义州|义州]][[刺史|刺史]][[王钦祚|王钦祚]]子刑部郎中[[王瑜_(五代)|王瑜]]劝父入蜀以免为辽所虏，王钦祚不肯，王瑜胁迫他动身，但因陇东屯兵阻拦未能入蜀，后皆被杀。<ref>[[:s:旧五代史/卷96|《旧五代史》卷九十六]]</ref>

二月，后晋[[河东节度使|河东节度使]]、北平王[[刘知远|刘知远]]闻何重建归蜀，叹道：“中原无主，令藩镇外附，吾为方伯，良可愧也！”<ref name=SGCQ53/>当月，称帝建立[[后汉|后汉]]。<ref name=ZZTJ286/>

后主厚待何重建，遣右千牛卫上将军[[李继勋|李继勋]]骑马慰谕，赏赐甚厚，在何重建经略下，蜀军夺取固镇，何重建请求蜀军与阶、成二州兵共扼[[散关|散关]]，以取[[凤州|凤州]]。后主发山南西道兵三千七百前去。何重建派宫苑使崔延琛进攻凤州，未果，退保固镇。当月，被加[[特进|特进]]、[[同平章事|同平章事]]。<ref name=SGCQ53/>四月，在何重建经略下，凤州防御使[[石奉頵|石奉頵]]也归蜀，<ref name=HFD94/><ref name=h9k/><ref name=ZZTJ286/><ref>[[:s:旧五代史/卷115|《旧五代史》卷一百一十五]]</ref><ref>[[:s:宋史/卷255|《宋史》卷二百五十五]]</ref><ref>[[:s:宋史/卷479|《宋史》卷四百七十九]]</ref>何重建又奉命攻克[[成州_(西魏)|成州]]。<ref>《[[蜀梼杌|蜀梼杌]]》</ref>于是后蜀达到了之前[[前蜀|前蜀]]的疆域。<ref name=NHFD94>[[:s:新五代史/卷64|《新五代史》卷六十四]]</ref>

十二月，因辽朝所任[[凤翔节度使|凤翔节度使]][[侯益|侯益]]、[[护国节度使|护国节度使]][[赵匡赞_(五代)|赵匡赞]]新降求援，[[山南西道节度使|山南西道节度使]][[张虔钊|张虔钊]]被任为北面行营招讨安抚使，出兵凤翔，何重建被任为招讨安抚副使，张虔钊出散关，何重建出[[陇州|陇州]]，共率兵五万，从成都出发，旌旗数十里，军容颇盛，意在接应和威胁侯益，但[[后汉|后汉]]右卫大将军[[王景崇_(五代)|王景崇]]纠合凤翔、晋昌、邠宁、泾原军队大破蜀军，<ref>[[:s:旧五代史/卷101|《旧五代史》卷一百零一]]</ref>侯益害怕，也归顺后汉、坚壁抵御蜀军，<ref>[[:s:宋史/卷254|《宋史》卷二百五十四]]</ref>蜀军无功，次年二月都虞候[[韩保贞|韩保贞]]等从陇州引兵回师，会合何重建西归。<ref name=HFD94/><ref name=h9k/><ref name=SGCQ53/><ref>[[:s:旧五代史/卷74|《旧五代史》卷七十四]]</ref><ref>[[:s:资治通鉴/卷287|《资治通鉴》卷二百八十七]]</ref><ref>《新五代史·后蜀世家》误作何重建等的接应对象为后来叛汉的[[永兴节度使|永兴节度使]][[赵思绾|赵思绾]]和占据凤翔的王景崇。</ref>后主以[[高彦俦|高彦俦]]代领雄武军。<ref name=SS274/>

何重建坚请入朝，后主不许。<ref name=h9k/>次年，何重建改任[[保宁军节度使|保宁军节度使]]。十三年（950年），[[安思谦|安思谦]]被任为保宁军节度使，可见当时何重建已不在任。何重建后累次加官至[[中书令|中书令]]。<ref name=HFD94/>

[[广政|广政]]二十年（957年）四月，[[后周世宗|后周世宗]]收复秦、成、阶等州，在《[[:s:平秦成阶等州德音|平秦成阶等州德音]]》中称“自从何重建等归蜀后，秦、成、阶等州连年饥荒，百姓和军人被倍加劳役、敛税”，故下令革除前弊。七月，后蜀后主在[[:s:与周世宗书|给后周世宗的求和书信]]中也提及了何重建归蜀之事。

二十一年（958年），何重建被加兼[[侍中|侍中]]。六十九岁去世于后蜀都城[[成都|成都]]。<ref name=h9k/><ref name=SGCQ53/>

== 评价 ==
*《旧五代史》史臣曰：何建举秦、陇之封，附巴、邛之俗，守方之寄，其若是乎！<ref name=HFD94/>
*《十国春秋》：何重建终始不渝……其人均有足取者。<ref name=SGCQ53/>
