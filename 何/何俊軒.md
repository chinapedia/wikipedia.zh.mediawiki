'''何俊軒'''（{{lang|en|'''Ho Chun Hin'''}}，{{bd|1975年|2月27日|catIdx=H何}}<ref>[https://archive.is/20140415073543/http://artiste.tvb.com/main/artistedetail/ctiho 何俊軒 Ho Chun Hin - TVB 藝人資料 - tvb.com]</ref>），原名'''何仲偉'''<ref>{{web reference |URL=https://www.hk01.com/%E5%A8%9B%E6%A8%82/39976/-%E5%9F%8E%E5%AF%A8%E8%8B%B1%E9%9B%84-%E5%9C%B0%E6%94%AF%E5%85%88%E7%94%9F%E4%BD%95%E4%BF%8A%E8%BB%92%E6%8B%8D%E5%8A%87%E9%9A%AA%E7%9B%B2-%E6%9C%89%E7%99%BC%E6%8F%AE%E7%9A%84%E8%A7%92%E8%89%B2-%E4%B8%80%E5%AE%9A%E8%A6%81%E5%81%9A |title=【城寨英雄】地支先生何俊軒拍劇險盲：有發揮的角色，一定要做 |work=《[[香港01|香港01]]》 |year=2016年 |date=8月28日}}</ref>，香港男演員及廣播主持人，為前[[無綫電視|無綫電視]]藝員。

== 簡歷 ==
何俊軒早年畢業於[[彩雲聖若瑟小學|彩雲聖若瑟小學]]和[[德仁書院|德仁書院]]（[[美孚|美孚]]分校）<ref name="CTI Ho's alma mater, print screened on 2018-6-9">[https://drive.google.com/file/d/1EpbPuuQ2_7ia0VzyFQm20BntZ1kVTOGb/view?usp=sharing CTI Ho's alma mater, print screened on 2018-6-9]</ref>。兒時曾以加入香港田徑代表隊為夢想，就讀中小學期間經常參與田徑運動<ref name=CTI>{{web reference |URL=http://tw.weibo.com/HoChunHinCTI/4145374296448546 |title=兒時志願是入香港田徑隊 代表香港參加奧運 |work=《新浪微博 - 何俊軒》 |year=2017年 |date=8月27日}}</ref>，而涉足娛樂圈前是個日本料理師傅{{noteTag|資料出自2012年無綫電視節目《芝人口面不知心》第17集〈公仔箱的小角色〉。}}。2001年，他畢業於[[無綫電視藝員訓練班|第15期無綫電視藝員訓練班]]後加入[[無綫電視|無綫電視]]，同屆畢業的藝人還有[[寧進|寧進]]、[[章志文|章志文]]、[[廖碧兒|廖碧兒]]等。

何俊軒多年來曾參演逾四十部無綫電視劇集，但通常擔演黑社會成員、警務人員或殺手等角色。2016年12月，他因拍攝該台劇集《[[踩過界|踩過界]]》時十字韌帶斷裂而決定離開無綫電視，並與妻子移民到[[加拿大|加拿大]][[多倫多|多倫多]]休養<ref>{{web reference |URL=https://www.hk01.com/%E5%A8%9B%E6%A8%82/114009/-%E5%90%8C%E7%9B%9F-%E5%BE%A1%E7%94%A8%E9%A0%AD%E9%A6%AC%E4%BD%95%E4%BF%8A%E8%BB%92%E6%8B%8D%E5%8A%87%E6%96%B7%E5%8D%81%E5%AD%97%E9%9F%8C%E5%B8%B6-%E5%BF%83%E6%B7%A1%E8%BE%AD%E8%81%B7-%E4%BF%82%E6%88%91%E6%9C%80%E5%BE%8C%E6%BC%94%E5%87%BA |title=【同盟】御用頭馬何俊軒拍劇斷十字韌帶　心淡辭職：係我最後演出 |work=《香港01》 |year=2017年 |date=8月23日}}</ref><ref>{{web reference |URL=https://news.mingpao.com/pns/dailynews/web_tc/article/20170824/s00016/1503510375476 |title=何俊軒加國養傷告別無綫 《同盟》監製點名讚專業 |work=《香港01》 |year=2017年 |date=8月24日}}</ref>。2018年，他加盟[[加拿大中文電台|加拿大中文電台]]，跟前無綫電視藝員[[蘇敏聰|蘇敏聰]]一同主持節目《[[愛上星期六|愛上星期六]]》<ref>{{web reference |URL=http://am1430.com/hot_topics_detail.php?i=1789 |title=歡迎 《何俊軒》 加盟加拿大中文電台 |work=《加拿大中文電台 - 娛樂追蹤》 |year=2018年 |date=2月27日}}</ref>。另外，他經常在大型節目中擔任禮儀先生，護送嘉賓上台。

== 家庭 ==
何俊軒家裡有三個姊姊，其外甥是香港男子田徑運動員[[鄧亦峻|鄧亦峻]]<ref name=CTI/>。

== 感情生活 ==
