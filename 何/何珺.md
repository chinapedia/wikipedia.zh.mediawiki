'''何珺'''{{BD|1984年|10月18日|||catIdx= H何珺}}“珺”音“郡”，[[江苏|江苏]][[无锡|无锡]]人，[[北京电影学院|北京电影学院]]04级表演系本科毕业，[[中国大陆|中国大陆]]女[[演员|演员]]。

== 简介 ==
何珺早年毕业于[[北京电影学院|北京电影学院]]，曾出演[[电视剧|电视剧]]《[[小洋楼|小洋楼]]》。何珺曾被选为[[张艺谋|张艺谋]]《[[金陵十三钗|金陵十三钗]]》演员之一，但是后来遭到弃用，并且在其[[新浪微博|新浪微博]]中发博文说自己和其余三位演员被弃用是遭人算计，并爆料[[张艺谋|张艺谋]]已经有了三个孩子。<ref>{{cite web |url= http://ent.sina.com.cn/s/m/2012-03-13/22203579695.shtml|title=何珺称遭十三钗开除有内情 付璐璐原是副导演 |accessdate= 2012年|work= |publisher=新浪网 |language =zh-cn}}</ref><ref>{{cite web |url=http://entertainment.dbw.cn/system/2012/03/13/053742532.shtml |title=女演员何珺爆料：张艺谋已再婚并育有三子女 |accessdate=2012年 |work= |publisher=东北网 |language=zh-cn |archive-url=https://web.archive.org/web/20120316225510/http://entertainment.dbw.cn/system/2012/03/13/053742532.shtml |archive-date=2012-03-16 |dead-url=yes }}</ref><ref>{{cite web |url=http://media.people.com.cn/GB/40606/17365361.html |title=女演员何珺实名爆料：张艺谋已再婚并育有三子女 |accessdate= 2012年|work= |publisher=人民网 |language =zh-cn}}</ref><ref>{{cite web |url= http://ent.ifeng.com/idolnews/mainland/detail_2012_03/13/13164824_0.shtml|title=连线张艺谋再婚新闻爆料人：好多人知道但不敢讲 |accessdate= 2012年|work= |publisher=凤凰网 |language =zh-cn}}</ref>
