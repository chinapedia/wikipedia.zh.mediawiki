'''何基灃'''（{{bd|1898年||1980年|1月20日|catIdx=H何}}），[[表字|字]]芑蓀，[[直隸|直隸]]（今[[河北|河北]]）[[藁城|藁城]]人。原[[中華民國國軍|国军]]将领，中华人民共和国政府官员。

== 早年經歷 ==
何在1923年於在[[保定陸軍軍官學校|保定陸軍軍官學校]]畢業后，參加[[馮玉祥|馮玉祥]]部隊（即[[西北軍|西北軍]]）。

== 中年經歷 ==
1930年，西北軍於[[中原大戰|中原大戰]]失敗後解體，其一小部分退入[[山西|山西]]南部，被[[國民政府|國民政府]]改編為第二十九軍；1931年，何被派任第二十九軍第一〇九旅副旅長。

1933年春，赴[[喜峰口|喜峰口]]抗擊日軍，以[[喜峰口战斗_(抗日战争)|戰功]]升為第一一〇旅旅長。

1937年[[盧溝橋事變|盧溝橋事變]]前夕，率部駐守[[盧溝橋|盧溝橋]]一帶，多次挫敗日軍。盧溝橋事變發生后，曾率部在盧溝橋奮起抗擊日軍侵略。<ref>[http://cpc.people.com.cn/GB/64162/64165/74856/74961/5137386.html 1948年12月10日 毛澤東、朱德電賀何基灃、張克俠起義]</ref>8月初，何升任第七十七軍第一七九師師長。11月上旬，何基灃率部退守[[大名府|大名府]]，遭遇日軍圍攻，終因彈盡援絕而失守。何與部下撤退到[[南樂縣|南樂縣]]城後，拔槍自戕，幸得部屬及時搶救方得脫險。

1938年，何經[[中國共產黨|中國共產黨]]地下黨員[[连玉岗|连玉岗]]介紹，對中共及[[八路軍|八路軍]]有較深刻認識。後在[[武漢|武漢]]經朋友的介紹，他會見了西北軍故舊，當時已是中國共產黨員的[[賴亞力|賴亞力]]，並通過賴亞力的協助於八路軍辦事處見到[[周恩來|周恩來]]，然後由周安排於同年2月秘密到達[[延安|延安]]。在延安的期間，何與當時中共領導人[[毛澤東|毛澤東]]、[[劉少奇|劉少奇]]、[[朱德|朱德]]等人面談後申请加入共產黨，並於1939年1月被豫鄂边区特委批准為中國共產黨特别黨員<ref>{{cite book |author=陈立人、林晚 |title=百战将星——何基沣 |chapter=秘密入党 |location=北京 ||publisher=解放军文艺出版社 |pages=216-258 |date=1988年6月}}</ref>。已成為共產黨員的他仍在部隊工作，同[[新四軍|新四軍]]四師、五師建立了聯繫，為中共派遣打入國軍從事[[兵運|兵運]]的[[臥底|臥底]]，在新四軍向[[湖北|鄂]]、[[豫|豫]]、[[皖|皖]]等地的發展和建立[[大別山|大別山]]根據地過程中起了重要作用。

1948年[[淮海戰役|淮海戰役]]，時任[[中華民國陸軍|中華民國陸軍]]第三綏靖區副司令官的何基灃，和[[張克俠|張克俠]]一起率第五十九軍全部，第七十七軍大部共兩萬多人於11月8日清晨在[[賈汪|賈汪]]、[[台兒庄|台兒庄]]地區临阵發動[[贾汪起义|贾汪事變]]，背叛國軍投共產黨，直接导致国军[[黄百韬|黄百韬]]所部在奉命西撤途中遭到解放军[[粟裕|粟裕]]所部阻截和合围，正式揭开淮海战役(徐蚌会战)帷幕。此后，先后擔任[[中國人民解放軍|中國人民解放軍]]第三十四軍軍長、[[南京警備司令部|南京警備司令部]][[司令員|司令員]]等職。

== 中华人民共和国成立后 ==
[[File:First-Class_Liberation_Medal_Awarded_to_He_Jifeng_by_the_People's_Republic_of_China_in_1955.jpg|left]]
历任[[中华人民共和国水利部|水利部]]副部長、[[中华人民共和国国务院|國務院]][[水土保持委員會|水土保持委員會]]副主任兼秘書長、[[中华人民共和国农业部|農業部]]副部長等職。曾被選爲[[人大代表|人大代表]]，被邀爲[[全國政協|全國政協]]委員、常委。

1980年1月20日病逝于[[北京|北京]]。他的骨灰則按照遺願，一半灑在[[盧溝橋|盧溝橋]]畔，一半灑在當年的淮海戰場上。

從1939年算起，何秘密加入中國共產黨到1949年中共建国時黨齡本已达十年。但据悉当时[[周恩來|周恩來]]曾對何說過一句話：「基灃同志，過去的事，就讓它作為黨的一個秘密吧！」，所以何直至去世也没有暴露其臥底身份。
