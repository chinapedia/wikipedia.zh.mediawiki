'''何氏細鬚魮'''（[[学名|学名]]：''Leptobarbus hoevenii''），俗稱'''何氏細鬚䰾'''、'''紅尾金絲'''、'''蘇丹魚'''、栗色鯊，為[[輻鰭魚綱|輻鰭魚綱]][[鯉形目|鯉形目]][[鯉科|鯉科]]的一個[[種|種]]。

== 分布 ==
本魚分布於[[泰國|泰國]]、[[柬埔寨|柬埔寨]]、[[寮國|寮國]]、[[馬來西亞|馬來西亞]]、[[印尼|印尼]][[蘇門答臘|蘇門答臘]]、[[婆羅洲|婆羅洲]]的溪流。

== 特徵 ==
本魚銀河色的腹側在側線以上有黃綠色的色澤。幼魚從頭到尾有一條黑色斑紋，會隨年齡的增長而褪去。腹鰭和尾鰭呈紅色，紅色尾鰭叉面有黑邊，背鰭基本上是無色的。背鰭軟條9枚；臀鰭軟條8枚，體長可達100公分。

== 生態 ==
本魚棲息在流動河川與溪流了與季節性地的洪泛區上。具洄游性，每年1、2月向河川上游洄游；而6、7月向下游的遷移時成群巡遊，屬雜食性，性情溫和。

== 經濟利用 ==
在[[柬埔寨|柬埔寨]]及[[越南|越南]]為重要的食用魚，並予以養殖，另可作為觀賞魚。
