'''何履光'''（{{bd|？||8世纪|catIdx=H}}），[[唐朝|唐朝]]军事人物，曾多次对[[南诏|南诏]]作战，官至[[岭南节度使|岭南节度使]]，[[安史之乱|安史之乱]]时期奉诏出兵勤王。他也是第一位可考的海南籍官员。<ref name=ZQG>周泉根《扦难御侮 谋赞体国—第一位琼籍官吏何履光考议》[A].《新东方》[J].2008年第5期</ref>

== 家世 ==
何履光是[[珠崖|珠崖]]人，<ref name=ZQG/><ref name=GYJ>《[[广异记|广异记]]》</ref>一作[[邕管|邕管]][[贵州_(广西)|桂州]]人。<ref name=MS>《[[蛮书|蛮书]]》卷七</ref>

何履光在《[[旧唐书|旧唐书]]》《[[新唐书|新唐书]]》中均未列传。

== 征讨南诏 ==
[[唐玄宗|唐玄宗]][[天宝_(唐朝)|天宝]]年间，曾任[[安南都护|安南都护]]、[[容管经略使|容管经略使]]。曾三次奏报本地异象。<ref name=GYJ/>八载（749年）十月，时任[[特进|特进]]的何履光奉命统领十道兵马从[[安南|安南]]进军讨伐[[云南|云南]]蛮国，<ref>[[:s:新唐书/卷5|《新唐书》卷五]]</ref>十载（751年）收复安宁城，复立[[马援|马援]]铜柱，回师。<ref>[[:s:新唐书/卷222上|《新唐书》二百二十二上]]</ref>十二载（752年）五月，在左武卫大将军任上率岭南五府兵击南诏。<ref>[[:s:资治通鉴/卷216|《资治通鉴》卷二百一十六]]</ref>都督将军贾颧全军覆没被擒，何履光仅以身免。十三载（753年），在[[岭南节度使|岭南节度使]]任上奉命与前云南郡都督兼[[侍御史|侍御史]][[李宓|李宓]]、中使萨道悬逊统秦陇、安南士兵修舟船，<ref name=MS/>水陆并进讨[[南诏|南诏]]，大败，李宓阵亡，<ref>[[:s:南诏德化碑|郑回《南诏德化碑》]]</ref>何履光遁去。<ref>《[[南诏野史|南诏野史]]》</ref>先前[[鲜于仲通|鲜于仲通]]兵败于南诏的战事，何履光可能也有参与。<ref name=ZQG/>

== 勤王救难 ==
十五载（756年），何履光正收复蛮王的大和城时，因[[安禄山|安禄山]]叛乱，奉玄宗诏率兵去西川，收复之事搁置。<ref name=MS/>五月，叛军围攻[[南阳|南阳]]，[[夷陵|夷陵]]太守虢王[[李巨|李巨]]受任为陈留、谯郡太守、[[河南节度使|河南节度使]]，统领何履光、[[黔中节度使|黔中节度使]][[赵国珍|赵国珍]]、[[南阳节度使|南阳节度使]][[鲁炅|鲁炅]]。<ref>[[:s:资治通鉴/卷218|《资治通鉴》卷二百一十八]]</ref>何履光、赵国珍及[[襄阳|襄阳]]防御使[[徐浩_(唐朝)|徐浩]]还未到南阳，鲁炅已被叛军所败，来自岭南、黔中、荆襄的裨将的亲属一半在军队，多带着金银为资粮，逃跑时把军资器械都丢弃在路上，堆积如山，叛军因此暴富。<ref>[[:s:旧唐书/卷114|《旧唐书》卷一百一十四]]</ref><ref>[[:s:新唐书/卷147|《新唐书》一百四十七]]</ref>李巨赶到南阳，<ref>[[:s:新唐书/卷79|《新唐书》卷七十九]]</ref>催促何履光、赵国珍同到。<ref>[[:s:旧唐书/卷112|《旧唐书》卷一百一十二]]</ref>[[监察御史|监察御史]][[高适|高适]]认为三位节度使都持节，数名监军也主事，南阳各军才屡战屡败。<ref name=ZQG/><ref>[[:s:陈潼关败亡形势疏|:s:陈潼关败亡形势疏]]</ref>唐玄宗嘉其言。<ref>[[:s:旧唐书/卷111|《旧唐书》卷一百一十一]]</ref><ref>[[:s:新唐书/卷143|《新唐书》一百四十三]]</ref>

[[崔国辅|崔国辅]]有《上何都督履光书》。

== 归隐山林 ==
同年十月（《[[唐会要|唐会要]]》作次年正月），贺兰进明被任为岭南节度使，未详何履光因何去职。

何履光最后归隐[[贵县|贵县]]。《[[一统志|一统志]]》误将曾任特进的何履光记作“何特进”“何履光”二人，《[[风土记|风土记]]》则误作“何履”“何光”二人。《[[徐霞客游记|徐霞客游记]]》载贵县有两座山峰，传说为“二何”所化。《西事珥》及[[朱惠荣|朱惠荣]]校注有辨证。<ref name=ZQG/>

当代学者[[易闻晓|易闻晓]]曾作《海口赋》赞何履光：
{{quote|噫吁彼苍连海，郁乎斯地有灵，半钟琼台毓秀，咸过海口鸿声。至于何公节度，兵定南诏，功佐天宝。}}
