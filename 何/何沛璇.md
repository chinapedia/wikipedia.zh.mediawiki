'''何沛璇'''  1995年5月15日出生于马来西亚吉隆坡。在吉隆坡接受了三年的华小教育之后，2005年随父母移居印尼雅加达，就读于当地的一所国际学校。
2007年，参与了雅加达一场圣诞节的电视节目演出。2008年，词曲创作了第一首歌上传至YouTube。2009年，随家人迁回马来西亚。2011年，在一项马来西亚全国大型的歌唱比赛《Malaysia Music Festival》中，自弹自唱自己的原创英语歌曲，成功晋级九强，从而有机会与曹格、品冠、张智成、方炯缤等马来西亚知名歌手同台演出，当晚也与著名马来女歌手Yuna合唱。2012年，去美国加州留学，就读于当地的一所学院。2014年八月中旬，考进美国旧金山大学（University of San Francisco) 攻读为期两年的传媒与电影科系，2016年5月以優異成績取的學士學位。

== 发片经历 ==

 
2013年，何沛璇于暑假期间回到马来西亚，她的父亲找来了老友音乐人陈温法帮她制作自己的创作，打算送一张迷你专辑给她当作18岁的生日礼物。制作人听
到了沛璇的创作，建议她的父母将她的创作推出市场，于是协助沛璇录制创作专辑。沛璇的父亲何文庆博士，当年在大学就读本科时是马来西亚创作团体“激荡工作坊”创作小组“风信子”的成员之一，因此深知流行歌曲可以给人带来正能量，欣然同意让女儿出来锻炼和吸取经验，当上“发片歌手”。

== 作品 ==

2014年7月--推出首张个人创作EP《对的时刻》，收录何沛璇的四首原创歌曲。
2014年12月--推出出道迷你音乐会的现场录音CD。
2015年5月--推出第二张个人创作专辑《First Chapter》，是一张中英语的双语专辑。

== 荣誉 == 

Eunice何沛璇堪称马来西亚乐坛史上第一位出道即办音乐会的新人，在2014年8月4日，除了迷你音乐会，当天也是她的首张专辑《对的时刻》发片记者会以及微电影首播的史无前例三合一活动。[2-3]出道以来，所获奖项包括：
《Neway本地K歌榜颁奖典礼2014》：
（一）10大热门本地K歌奖（《对的时刻》）、（二）我最喜爱杰出新人奖（银奖）（Eunice Hoo）和（三）Neway推荐新人创作歌手奖（Eunice Hoo）。
《2015年娱协奖》：
（一）十大原创歌曲奖《对的时刻》、（二）新人推荐奖（铜奖）（Eunice Hoo）。
《Neway本地K歌榜颁奖典礼2015》：
（一）10大热门本地K歌奖（《Andersen也不明白》）、（二）推崇唱作歌手奖（Eunice Hoo）。

== 评价 ==

《对的时刻》制作人陈温法表示，Eunice的创作与大马现有的创作不一样，他说：“我把这张专辑当作老外的专辑来做，她的唱腔也很洋派。”。
而到发片记者会现场支持的另一位音乐人伍家辉也赞说：“我刚听以为这些歌是从版权公司买回来的作品，后来知道是她自己的创作，真的很惊喜，
声音像是蔡健雅加Olivia Ong的混合体。”。这张专辑也找来马来西亚著名影视艺人陈凯旋合作拍MV，也将专辑中4首歌拍摄成微电影。[4]
2014年8月4日，何沛璇《对的时刻》迷你演唱会开场，反应热烈，音乐会满座!座位供不应求。[5] 
发片时只有19岁的Eunice Hoo因为《对的时刻》这首歌让她少有知名度，发片后迅速举办音乐会的她，隔天8月5日即搭机到美国继续自己的学业。[6]
