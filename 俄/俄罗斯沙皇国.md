'''俄罗斯沙皇国'''或'''沙皇俄国''' ({{lang-ru|'''Царство Русское''', ''Tsarstvo Russkoye''}})，简称'''沙俄'''或'''俄国'''，是1547年至1721年間，[[伊凡四世|伊凡四世]]加冕[[沙皇|沙皇]]後，[[彼得大帝|彼得大帝]]改国号为[[俄羅斯帝國|俄羅斯帝國]]之前的[[俄羅斯|俄羅斯]]国号。

== 繼承拜占庭的傳統 ==
{{See also|沙皇|第三羅馬|拜占庭帝国|凯撒 (头衔)}}

[[File:Ivans_ivory_throne.jpg|thumb]]的象牙皇座]] 
16世纪時，[[俄国|俄国]]统治者使用了一個彰顯力量的、独裁的頭銜——[[沙皇|沙皇]]（Tsar），由[[拉丁文|拉丁文]]Caesar（[[凯撒_(头衔)|凱撒]]）的[[西里尔字母|西里尔]]轉寫（類似德語的[[凱撒_(德國)|Kaiser]]），一個原先用于[[羅馬帝國|羅馬帝國]]皇帝的稱呼。[[莫斯科大公|莫斯科大公]]试图强调他是一個与[[羅馬皇帝列表|羅馬]]或[[拜占庭皇帝列表|拜占庭皇帝]]拥有同等地位的統治者。事实上，在[[伊凡三世|伊凡三世]]与最后一个[[拜占廷|拜占廷]]公主[[索菲娅·帕列奥罗格|索菲娅·帕列奥罗格]]成婚后，[[莫斯科|莫斯科]]朝廷經常以[[第三罗马|第三罗马]]自矜，大肆引入[[拜占廷|拜占廷]]的词汇、礼仪、官员头衔和标志（如[[双头鹰|双头鹰]]），以致最后[[双头鹰|拜占庭双头鹰]]竟變成了[[俄羅斯聯邦國徽|俄罗斯国徽]]。

起初，[[凯撒_(头衔)|凱撒]]頭銜仅為統治者冗長頭銜的一部分，但在[[伊凡四世|伊凡四世]]统治时期（1533－1584）就搖身一變，变成至高无上的[[君主專制|專制君主]]的意思。被加冕沙皇因此至少被[[俄罗斯正教会|俄罗斯正教会]]承认为[[皇帝|皇帝]]。東正教[[牧首|牧首]]{{link-en|普斯科夫的斐洛菲斯|Philotheus of Pskov}}便声明「从1453年[[君士坦丁堡的陷落|君士坦丁堡陷落]]的那一刻起，[[俄罗斯沙皇|俄罗斯沙皇]]是[[罗马帝国|罗马]]与[[拜占庭帝国|拜占庭帝国]]唯一的合法继承者，[[莫斯科|莫斯科]]是“[[第三羅馬|第三个罗马]]”。」這个信念成为了随后几个世纪[[俄罗斯人|俄罗斯人]]的共识。

== 早期的[[伊凡四世|伊凡四世]]统治 ==
[[沙皇|沙皇]]独裁权力的发展在[[伊凡四世|伊凡四世]]在位时期达到了顛峰。伊凡四世也以雷帝（他的俄语称号''groznyi''，意为“令人惧怕”）的称号广为人知。伊凡将[[沙皇|沙皇]]的地位强化到一种空前的程度，也展示出了没有约束的权力在一个精神不稳定的人手里有多大的风险。虽然伊凡很聪慧而精力充沛，他却饱受妄想和抑郁症发作的困扰，而且在统治期间也常有极端暴力的举动。

1533年，伊凡四世成為莫斯科三大諸侯中最有權力的一個。

== 动乱时代 ==
{{主条目|俄罗斯混乱时期}}
1591年，俄罗斯[[沙皇|沙皇]][[费奥多尔·伊万诺维奇|费奥多尔一世]]的同父异母兄弟、有王位继承权的季米特里被[[鲍里斯·戈东诺夫|鲍里斯·戈东诺夫]]秘密杀害。1604年，一个自称是季米特里的人出现，并且得到了[[波兰|波兰]]国王[[西吉斯蒙德三世|西吉斯蒙德三世]]的支持。1605年，戈东诺夫的儿子[[费奥多尔·戈东诺夫|费奥多尔二世]]被杀，季米特里即位，史称'''“[[伪德米特里一世|伪沙皇季米特里]]”'''或“'''伪季米特里'''”。

由于采取亲波兰的政策，伪季米特里触怒了俄罗斯大贵族。在大贵族瓦西里·舒伊斯基的率领下，季米特里被杀，舒伊斯基即位，号称[[瓦西里四世·伊万诺维奇·叔伊斯基|瓦西里四世]]（1606—1610）。

1607年，另一个伪季米特里出现了，再次得到波兰的支持。1610年，波兰军队攻占[[莫斯科|莫斯科]]，舒伊斯基被杀。这一次，西吉斯蒙德想让自己的儿子[[弗瓦迪斯拉夫|弗瓦迪斯拉夫]]继承俄罗斯皇位，后来又改变主意想自己兼任俄罗斯沙皇。但是波兰入侵者在1612年被俄罗斯贵族们召集的军队驱逐。

1613年，[[全俄罗斯缙绅会议|全俄罗斯缙绅会议]]（zemsky sobor）宣布立17岁的[[米哈伊尔·费奥多罗维奇·罗曼诺夫|米哈伊尔·罗曼诺夫]]为沙皇。[[罗曼诺夫王朝|罗曼诺夫王朝]]开始。

== 占领烏克蘭 ==
[[File:Strorus_gorod.jpg|thumb]]
17世紀，[[俄羅斯|俄羅斯]]向[[烏克蘭|烏克蘭]]東部進攻，國界開始與[[波蘭立陶宛|波蘭立陶宛聯邦]]接鄰。

1654年，[[烏克蘭|烏克蘭]]領袖[[博格丹·赫梅利尼茨基|博格丹·赫梅利尼茨基]]希望烏克蘭可以受到[[俄國|俄國]][[阿列克谢·米哈伊洛维奇·罗曼诺夫|阿列克谢一世]]保護，阿列克谢接受，並簽署[[佩列亚斯拉夫条约|佩列亚斯拉夫条约]]，而後在[[俄波战争_(1654年－1667年)|俄波战争]]中擊敗[[波蘭|波蘭]]。

== 文化與宗教傳播 ==
[[俄羅斯|俄羅斯]]在向西南擴張的途中，與[[羅馬|羅馬]][[天主教|天主教]]以及[[波蘭|波蘭]]反宗教組織有密切聯繫，亦帶來了西方學術。大多數[[烏克蘭人|烏克蘭人]]信奉[[東正教|東正教]]，因此透過[[基辅大学|基輔學院]]，[[俄羅斯|俄羅斯]]可以與[[波蘭|波蘭]]以及[[中歐|中歐]]地區與其他[[東正教|東正教]]國家聯繫。

[[俄羅斯|俄羅斯]][[東正教|東正教]][[牧首|牧首]][[尼孔|尼孔]]決定以[[希臘語|希臘語]]為藍本，發明屬於俄羅斯的文字，即[[西里爾字母|西里爾字母]]。

== 征服西伯利亞 ==
{{See|俄羅斯征服西伯利亞}}
[[File:Surikov_Pokoreniye_Sibiri_Yermakom.jpg|thumb]]]]

[[俄羅斯|俄羅斯]]在東征途中只遇到了小規模的反抗。1581年，商人[[斯特羅加諾夫家族|斯特羅加諾夫家族]]進行羽絨貿易，招聘[[哥薩克|哥薩克]]領袖[[葉爾馬克·齊莫菲葉維奇|葉爾馬克·齊莫菲葉維奇]]，開始向[[西伯利亞|西伯利亞]]進攻。葉爾馬克打敗了[[西伯利亞汗國|西伯利亞汗國]]，將國界伸延至[[鄂畢河|鄂畢河]]及[[额尔齐斯河|额尔齐斯河]]，後俄軍繼續東進，吞併了衰退中的[[瓦剌|瓦剌汗國]]，此汗國為上世紀[[明朝|明朝]]的重要边患。

東征隊伍以[[Mangazeya|Mangazeya]]為根據地，商人、貿易家及探險家由[[鄂畢河|鄂畢河]]延伸至[[葉尼塞河|葉尼塞河]]，再向東至[[太平洋|太平洋]]的[[勒拿河|勒拿河]]。1648年，[[哥薩克人|哥薩克人]][[謝苗·伊凡諾維奇·迭日涅夫|謝苗·伊凡諾維奇·迭日涅夫]]率先橫渡[[太平洋|太平洋]]，前往[[美洲大陸|美洲大陸]]。17世紀中期，[[俄羅斯|俄羅斯]]已在[[黑龍江|黑龍江]]駐紮，並與[[中國|中國]]接觸。

1689年與[[清朝|清朝]]的衝突後，與[[中國|中國]]簽署和約。在[[尼布楚條約|尼布楚條約]]，中國收復了[[黑龍江|黑龍江]]以南的領土，而[[贝加尔湖|贝加尔湖]]以東為[[俄羅斯|俄羅斯]]所有，與中國貿易的路線也借道此區。在這一世紀以來[[俄國|俄國]]與[[清朝|清朝]]維持和睦關係。
