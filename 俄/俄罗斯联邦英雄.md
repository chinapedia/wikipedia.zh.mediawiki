'''俄罗斯联邦英雄''' （{{lang-ru|Герой Российской Федерации}}，[[俄语罗马化|俄语罗马化]]：{{transl|ru|Geroy Rossiyskoy Federatsii}}）是一种[[俄罗斯联邦|俄罗斯联邦]]勋章和奖项，也是俄罗斯國民可以获得最高级别的荣誉头衔。[[俄罗斯总统|俄罗斯总统]]拥有头衔的授予权力，授予那些为国家服务而做出突出和勇敢行动或事蹟的國民。自创立以来，它已经被授予1051次，主要的授予对象是[[宇航员|宇航员]]以及参与对[[车臣共和国|车臣]]之军事行动的军人。几位艺术家、政治家、经济学家和运动员也曾获授予此头衔。

== 创始 ==
这个头衔在1992年3月20日[[俄罗斯联邦总统|俄罗斯联邦总统]][[鲍里斯·叶利钦|鲍里斯·叶利钦]]签署的一项法定中被创立，以取代[[苏联|苏联]]时代的[[苏联英雄|苏联英雄]]和[[社会主义劳动英雄|社会主义劳动英雄]]。2553-I号法令写有头衔的授予准则（见下方），以及伴随着这个荣誉的[[金星勋章|金星勋章]]。[[俄罗斯宪法|俄罗斯宪法]]第71条规定政府可以授予头衔、荣誉和勋章，而第89条授予总统创立新国家奖章的权力<ref name="ruconst">{{citeweb|url=http://award.adm.gov.ru/kons/kons_1.htm |title=Государственные награды России: Из Конституции Российской Федерации |accessdate=2005-09-30 |language=Russian |deadurl=yes |archiveurl=https://web.archive.org/web/20111102231511/http://award.adm.gov.ru/kons/kons_1.htm |archivedate=2011-11-02 }}</ref>。这是俄罗斯总统可以向所有人授予的最高级别荣誉。有别于苏联英雄的头衔，俄罗斯英雄头衔没有其他勋章和称号。

== 授予标准 ==
2553-I号法令指明此头衔可以颁发给一个为国家和人民服务是做出英勇事迹的人士，因此普通國民和军事人员皆有机会获得此头衔。如果授勋者的英勇行为使他/她付出了生命，可以向他/她追颁俄罗斯联邦英雄的头衔。总统曾经大约463次追颁勋章，主要追颁的对象是在[[第一次车臣战争|第一次]]和[[第二次车臣战争|第二次]]对[[车臣共和国|车臣共和国]]的军事行动中牺牲的军人。最近的一次是总统[[弗拉基米尔·弗拉基米罗维奇·普京|普京]]向宇航员[[阿列克谢·尼古拉耶维奇·奥维切尼|阿列克谢·尼古拉耶维奇·奥维切尼]]颁发以表彰其在国际空间站执行长期太空任务中表现出的勇气和英雄气概。

== 设计与展示 ==
[[金星勋章|金星勋章]]的设计在2553-I法令中已经被详细地说明<ref name="ruconst"/>，设计与[[苏联英雄|苏联英雄]]的勋章十分相似。色带是一个长方形的悬挂装置，高19.5mm，宽15mm，颜色是象征着[[俄罗斯国旗|俄罗斯国旗]]的[[红|红]]、[[白|白]]和[[蓝|蓝]]三色。色带下面吊着的五角[[星|星]]直径为15mm，表面并没有任何设计。在反面，有2x4mm的凸字体'''俄罗斯英雄'''（{{lang-ru|Герой России}}），在顶部有一个1mm大的[[序号|序号]]。序号显示了这个奖项被颁发的次数（例如SN164说明该勋章是第164被颁发的）。在色带的背面，有一个由针和钩组成的扣紧装置，目的是把勋章系在衣服上。勋章由[[金|金]]制造，重大约21.5[[克|克]]。<ref name="ruconst"/>

当这个勋章在公众场所被穿戴的时候，它会被戴在西装外套左边所有俄罗斯和苏联勋章上面的位置。有些以往也曾获颁苏联勋章（譬如[[苏联英雄|苏联英雄]]和[[社会主义劳动英雄|社会主义劳动英雄]]）的人士，会在佩戴此勋章的同时也佩戴苏联勋章。

== 获授者 ==
[[File:Dmitry_Medvedev_with_Oleg_Mutovin.jpg|thumb]]向奥列格·穆托芬（Oleg Mutovin）授予俄罗斯联邦英雄的头衔]]
{{Main|俄罗斯联邦英雄列表}}

此头衔的授予对象主要有两个类别:[[车臣冲突|车臣冲突]]的参与者和[[宇航员|宇航员]]。每一个进入[[太空|太空]]的宇航员会自动获授此头衔；[[车臣|车臣]]冲突的参与者则需要有英勇的战斗行为或领导亲[[俄罗斯政府|莫斯科]]政府才能获授此头衔。在有些情况下，执行任务过程中被杀的人会获授此头衔，包括在战斗中被杀的军人和[[暗杀|暗杀]]事件中被杀的官员。这种获授者的例子有前[[车臣共和国总统|车臣共和国总统]][[艾哈迈德·卡德罗夫|艾哈迈德·卡德罗夫]]。这名亲莫斯科的领导人于2004年5月9日车臣首府[[格罗兹尼|格罗兹尼]]的[[胜利纪念日|胜利纪念日]]游行中被车臣武装份子以炮弹改造的地雷刺杀身亡。艾哈迈德去世之后的几天，总统[[弗拉基米尔·弗拉基米罗维奇·普京|弗拉基米尔·弗拉基米罗维奇·普京]]就向他追授这个头衔。一阵子过后，普京向他的儿子[[拉姆赞·卡德罗夫|拉姆赞·卡德罗夫]]授予同样的头衔，以表扬他在车臣的工作。<ref name="kadyrov">{{citeweb |url=http://www.themoscowtimes.com/stories/2004/12/30/015.html |title=Kadyrov Named Hero of Russia |accessdate=2005-10-01}}</ref>

所有俄罗斯宇航员会在他们进入太空的旅程后获授此头衔；有些宇航员在进入太空已经获授了此头衔，譬如长时期为国家服务的试飞员。宇航员同时也会获授[[俄罗斯联邦飞行员-宇航员|俄罗斯联邦飞行员-宇航员]]的头衔。有些头衔的获授者，例如[[谢尔盖·克里卡廖夫|谢尔盖·克里卡廖夫]]，也获授了苏联英雄和[[列宁勋章|列宁勋章]]。<ref name="krikalev">{{citeweb |url=http://www.jsc.nasa.gov/Bios/htmlbios/krikalev.html |title=Cosmonaut Bio: Sergei K. Krikalev (10/2005) |accessdate=2005-10-05}}</ref>大多数宇航员同时获得苏联英雄和俄罗斯联邦英雄，是因为他们“成功体现了飞行的勇气和[[英雄主义|英雄主义]]的表现。”<ref name="cosmonauts">{{citeweb |url=http://www.warheroes.ru/hero/hero.asp?Hero_id=871 |title=Герой Советского Союза, Герой России Поляков Валерий Владимирович |accessdate=2005-10-05 |language=Russian}}</ref>
