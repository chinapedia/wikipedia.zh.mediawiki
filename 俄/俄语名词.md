在[[俄语|俄语]]中，[[名词|名词]]用来表示事物的名称。'''俄语名词'''可以分为普通名词和专有名词，动物名词和非动物名词。他们还有[[性_(語法)|性]]、[[數_(语法)|数]]、[[格_(语法)|格]]的区别。

==专有名词==

俄语中的的[[专有名词|专有名词]]包括人名、地名以及书报杂志等各种[[出版物|出版物]]的名称。专有名词的第一个[[字母|字母]]需要大写。人名、地名以及书报杂志等各种[[出版物|出版物]]的名称还要标上[[书名号|书名号]]。

==动物名词==

俄语中的[[动物名词|动物名词]]是用来表示[[人|人]]和[[动物|动物]]的名词，在句中一般回答{{lang|ru|кто}}（谁）的问题。例如：

:{{lang|ru|Кто это?}} // 这是谁？
:{{lang|ru|Это '''учитель'''.}} // 这是老师。

其他不是表示人和动物的名词则称为[[非动物名词|非动物名词]]，在句中一般回答{{lang|ru|что}}（什么）的问题。例如：

:{{lang|ru|Что это?}} // 这是什么？
:{{lang|ru|Это '''книга'''.}} // 这是书。

俄语名词有性的区别。根据词尾不同分为阳性、阴性、中性。

*俄语中还有10个以'''-мя'''结尾的名词是中性名词，例如{{lang|ru|время}}（时间）。

==数==

俄语名词分为单数和复数。单数名词表示一个事物；复数名词表示两个以上的事物。

复数用结尾的变化来表示。共有四种：ы，и，а，я。

*有些名词复数的结尾是'''-а'''（'''-я'''），则[[重音|重音]]在词尾上。
*某些阳性和中性名词的结尾是'''-ья'''。
*一些名词变成复数时，重音会移动。
*以г，к，х，ж，ш，ч，щ结尾的阳性名词，复数加'''и'''，而不是加 '''ы'''。
*以га，ка，ха，жа，ша，ча，ща结尾的阴性名词，复数结尾将'''-а'''改为'''-и'''，而不是'''-ы'''。

===变格时词身-о，-е的隐现规则===

以'''-а，-я'''结尾的阴性名词和'''-о'''结尾中性名词，如果词尾前有两个并列辅音字母，复数第二格在去掉'''-а，-я，-о'''同时，在两个并列辅音字母之间加入'''-о'''或'''-е'''。其中：
*两个辅音字母中有一个是'''-ж，-ш，-щ，-ч'''则中间加上'''-е-'''。
*当两个辅音字母中有一个是'''-г， -к，-х'''，而另外一个字母不是'''-ж，-ш，-щ，-ч'''，则加上'''-о-'''。
*如果两个字母都不是上述这些字母，则加上'''-е-'''。

以'''-а，-я'''结尾的阴性名词和'''-о'''结尾中性名词，如果词尾前是'''-й'''与辅音字母，或'''-ь'''与辅音字母，复数第二格在去掉'''-а，-я，-о'''同时，须要将'''-й'''，'''-ь'''变成'''-е'''。

以'''-ец'''结尾的阳性名词变格，如果结尾前为'''-л-'''，则'''-ец'''变为'''-ьц'''；如果结尾前为元音字母，则'''-ец'''变为'''-йц'''。

还有一些名词变格时，结尾前的'''-о'''，'''-е-'''会消失。
