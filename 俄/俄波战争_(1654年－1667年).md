'''1654年至1667年间的俄波战争'''（{{lang-ru|Русско-польская война (1654—1667)}}、{{lang-pl|Wojna polsko-rosyjska (1654–1667)}}））亦称为'''十三年战争'''<ref name=Frost13/>或'''[[第一次北方战争|第一次北方战争]]'''<ref name=Frost13/>，这时[[俄羅斯沙皇国|沙皇俄國]]与[[波兰立陶宛联邦|波兰立陶宛联邦]]之间爆发的最后一场大规模冲突。1655年至1660年间，[[第二次北方战争|第二次北方战争]]也在波兰立陶宛联邦爆发，因此这几场战争在波兰并称为[[大洪水时代|洪水之战]]。这场战争以俄罗斯的重大领土扩张结束，标志着俄国崛起的开始。

== 背景 ==
乌克兰哥萨克反抗[[波兰立陶宛联邦|波兰立陶宛联邦]]的[[赫梅利尼茨基起义|赫梅利尼茨基起义]]是这起冲突的导火索。[[哥萨克人|哥萨克人]]领导人[[博格丹·赫梅利尼茨基|博格丹·赫梅利尼茨基]]主要从[[阿列克谢一世|阿列克谢一世]]获得外国支持，并答应给他的盟友报酬。但因1651年的{{link-en|土地会议|Zemsky Sobor}}在将[[哥萨克|哥萨克]]放入[[莫斯科|莫斯科]]的[[势力范围|势力范围]]，及在反抗波兰的战争中加入哥萨克一方的问题上还犹豫不决，直到1653年新的人民会议最终批准将[[乌克兰|乌克兰]]并入[[沙皇俄国|沙皇俄国]]，沙皇才加入战争。在哥萨克于[[佩列亚斯拉夫会议|佩列亚斯拉夫会议]]签认协议后，俄波战争就变得不可避免了。

== 入侵联邦 ==
1654年7月，兵力达41000人的俄军（表面上由沙皇统帅，但实际上由亲王[[亚科夫·切尔卡斯基|亚科夫·切尔卡斯基]]、亲王[[尼基塔·奥多埃夫斯基|尼基塔·奥多埃夫斯基]]和亲王[[安德烈·霍万斯基|安德烈·霍万斯基]]指挥）占领了边境要塞[[别雷|别雷]]和[[多罗戈布日|多罗戈布日]]，并开始围攻[[斯摩棱斯克|斯摩棱斯克]]。
 
因为[[波兰立陶宛联邦指挥官|立陶宛大指挥官]][[亚努什·拉齐维乌_(1612年-1655年)|亚努什·拉齐维乌]]带着1000兵马抵达较[[奥尔沙|奥尔沙]]偏西处，俄罗斯在斯摩棱斯克情况危急。切尔卡斯基在[[什科洛夫|什科洛夫]]击败拉齐维乌。在3个月的围攻之后，斯摩棱斯克这一[[斯摩棱斯克战争|先前的俄波冲突]]中的主要目标在9月落入俄军手中。

同时，亲王[[亚列克谢·特鲁贝茨科伊|亚列克谢·特鲁贝茨科伊]]率领俄军南翼从[[布良斯克|布良斯克]]到达乌克兰。俄军很快占领了[[第聂伯河|第聂伯河]]和[[别列津纳河|别列津纳河]]之间的领土，其中特鲁贝茨科伊占领了[[姆斯季斯拉夫尔|姆斯季斯拉夫尔]]和[[罗斯拉夫尔|罗斯拉夫尔]]，而他的乌克兰盟友占领了[[霍梅尔|霍梅尔]]。而[[费奥多尔·舍列梅捷夫|费奥多尔·舍列梅捷夫]]指挥北翼军从[[普斯科夫|普斯科夫]]出征，占领了立陶宛城市[[内维尔|内维尔]]（7月1日）、[[波洛茨克|波洛茨克]]（7月17日）和[[维捷布斯克|维捷布斯克]]（11月17日）。 

因此波属[[利沃尼亚|利沃尼亚]]布满了沙皇的部队，并在[[卢扎|卢扎]]和[[雷泽克内|雷泽克内]]稳固地扎下根来。与此同时，赫梅利尼茨基与莫斯科贵族[[布图尔林|布图尔林]]联军攻打[[沃利尼亚|沃利尼亚]]。尽管指挥官间内讧颇多，他们还是在年末攻占了[[奥斯特罗格|奥斯特罗格]]和[[罗夫诺|罗夫诺]]。

== 1655年战役 ==
1655年冬春两季，亲王拉齐维乌在[[白俄罗斯|白俄罗斯]]发起反攻，重夺[[奥尔沙|奥尔沙]]，并围攻[[莫吉尔约夫|莫吉尔约夫]]。此次围攻持续了3个月，但战果未卜。1月，舍列梅捷夫和赫梅利尼茨基在[[阿赫马托夫|阿赫马托夫]]牵着波军鼻子走，这时另一支波军（与[[鞑靼人|鞑靼人]]联盟）在拉什科夫击溃了俄乌分遣队。

这些不利情况让沙皇担心受怕，他从莫斯科迅速赶来，在他的催促下，指挥官开始发动一波大规模进攻。立陶宛军队很少发动有效的反抗，并在7月3日放弃了[[明斯克|明斯克]]，将这座城市留给了哥萨克和切尔卡斯基部。[[立陶宛大公国|立陶宛大公国]]首都[[维尔纽斯|维尔纽斯]]在7月31日被俄军攻破。接着在8月，[[考纳斯|考纳斯]]和[[赫罗德诺|赫罗德诺]]也被占领。 

另一方面，亲王[[沃尔孔斯基|沃尔孔斯基]]从[[基辅|基辅]]出发，沿着第聂伯河和[[普里皮亚季河|普里皮亚季河]]的上游游去，牵制立陶宛，并在途中占领[[平斯克|平斯克]]。特鲁贝茨科伊部攻占[[斯洛尼姆|斯洛尼姆]]和[[克莱茨克|克莱茨克]]，这时舍列梅捷夫在6月17日略微希望向前攻占[[韦利日|韦利日]]。一支立陶宛卫戍部队依然在[[斯塔雷-比霍夫|斯塔雷-比霍夫]]顽抗，这时赫梅利尼茨基和布图尔林已经活跃于[[加利西亚_(中欧)|加利西亚]]。他们在9月发起对[[利沃夫|利沃夫]]的进攻，并在[[萨皮耶哈|萨皮耶哈]]于[[布列斯特|布列斯特]]附近战败后进入[[卢布林|卢布林]]。

== 停战 ==
俄罗斯对联邦的入侵也使得瑞典国王[[卡尔十世_(瑞典)|卡尔十世]]在1655年[[第二次北方战争|入侵波兰]]。

[[阿法纳西·奥尔丁-纳什乔金|阿法纳西·奥尔丁-纳什乔金]]在11月2日与波兰人谈判停战。随后，在[[第二次北方战争|第二次北方战争]]中的[[俄瑞战争_(1656年-1658年)|1656年至1658年间的俄瑞战争]]中，俄军开始向瑞属利沃尼亚进发，并围攻[[里加|里加]]。 

虽然警惕于波兰的鬼鬼祟祟，赫梅利尼茨基还是不反对这份临时休战协议，支持沙皇。<ref>[http://vostlit.narod.ru/Texts/Dokumenty/Russ/XVII/1640-1660/AlexejI/Gramoty_Bogdan_1656/text.htm Грамоты из переписки царя Алексея Михайловича с Богданом Хмельницким в 1656 г.]</ref>

== 与维霍夫斯基作战 ==
1657年，赫梅利尼茨基去世，他的继位者是[[伊凡·维霍夫斯基|伊凡·维霍夫斯基]]。维霍夫斯基[[哈迪亚赫条约|与波兰人结盟]]，波兰人随后击败并击退了瑞典人与他们的入侵。在这种情况下，沙皇与瑞典签署了对他有利的[[瓦列尔萨里条约|瓦列尔萨里条约]]，这份条约允许他重新与波兰人交战，1658年10月，沙皇重新与波兰人交战。在两年的和平状态下，白俄罗斯贵族（与众多哥萨克领导人）开始改变阵营，在白俄罗斯帮助波兰人发起对俄罗斯的冬季攻势。

在北方，萨皮耶哈封锁维尔纽斯的尝试被亲王[[尤里·多尔戈鲁科夫|尤里·多尔戈鲁科夫]]击退（10月1日）。在南方，乌克兰的维霍夫斯基未能让[[基辅|基辅]]摆脱舍列梅捷夫的控制，重夺此地。但在1659年7月，维霍夫斯基与他的[[克里米亚鞑靼人|克里米亚鞑靼人]][[科诺托普战役|痛击]]特鲁贝茨科伊的军队，随后围攻[[科诺托普|科诺托普]]。 

舍列梅捷夫的军队在8月从基辅出发，解除了维霍夫斯基对俄罗斯对乌克兰的征服的威胁。在维霍夫斯基于[[切黑伦|切黑伦]]被击败并逃到波兰时，哥萨克罢黜了他，并选举赫梅利尼茨基之子[[尤里·赫梅利尼茨基|尤里]]为乌克兰新任酋长。

== 战争结束 ==
[[File:Rzeczpospolita.png|thumb]]中割让]]
1660年，形势变得对波兰有利。波兰国王[[约翰二世_(波兰)|约翰二世]]在与瑞典签署[[奥利瓦条约|奥利瓦条约]]，[[第二次北方战争|第二次北方战争]]结束后，就能将他的所有军队投入到东部边境了。 

波军发动了一波较大的攻势，并到1660年年底前将俄军逐出白俄罗斯。萨皮耶哈击败了霍万斯基，舍列梅捷夫被迫在[[楚德尼夫|楚德尼夫]][[楚德努夫战役|投降]]。这时代最卓越的波军将领是[[斯特凡·恰尔涅茨基|斯特凡·恰尔涅茨基]]：他在[[波隆卡战役|波隆卡战役]]中牵制多尔戈鲁科夫军，并在1661年重夺维尔纽斯。联邦军队一个接着一个地收复立陶宛大公国的其他城镇。形势的逆转迫使沙皇接受[[卡尔迪斯条约|卡尔迪斯条约]]，以避免重新与瑞典作战。

直到1663年年末，波兰国王渡过第聂伯河，侵入[[左岸乌克兰|左岸乌克兰]]。在他征途中的大多数城镇在无抵抗的情况下投降，但是他1月[[格鲁霍夫攻城战|对格鲁霍夫的围攻]]却是惨败，并在[[诺夫哥罗德-塞维尔斯基|诺夫哥罗德-塞维尔斯基]]进一步遭受挫折。但是，俄军在1664年夏于维捷布斯克的战败是战争中的最后一件知名事件。 

停战谈判从1664年一直拖延至1667年1月，这时[[耶日·塞巴斯蒂安·卢博米尔斯基|耶日·塞巴斯蒂安·卢博米尔斯基]]的[[卢博米尔斯基起义|起义]]迫使波兰签署[[安德鲁索沃协议|安德鲁索沃协议]]，在那份协议中波兰将斯摩棱斯克和左岸乌克兰（包括[[基辅|基辅]]）割让给俄国。
