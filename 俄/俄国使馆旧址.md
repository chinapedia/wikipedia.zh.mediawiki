'''俄国使馆旧址'''，位于今[[北京市|北京市]][[东城区_(北京市)|东城区]][[东交民巷|东交民巷]]27号（[[中华人民共和国最高人民法院|中华人民共和国最高人民法院]]），先后是[[俄罗斯帝国|俄罗斯帝国]]及[[苏联|苏联]]驻华使馆旧址，现已无存。

俄国使馆旧址以西为俄国兵营旧址，位于今东交民巷29-35号。遗留至今的部分建筑以“原俄国使馆武官处”之名被列为[[北京市东城区普查登记文物|北京市东城区普查登记文物]]。

==简介==
1689年起，根据《[[尼布楚条约|尼布楚条约]]》，[[俄罗斯帝国|俄罗斯帝国]]要求派神父和学生来[[北京|北京]]学习。1684年，清廷将北京东江米巷（今[[东交民巷|东交民巷]]）南会同馆改为俄罗斯馆，供俄罗斯帝国的商人、神父、学生居住。[[雍正|雍正]]五年（1727年），中俄签订《[[恰克图界约|恰克图界约]]》，其中第五条准许[[东正教|东正教]]在北京东江米巷的俄罗斯馆建教堂：“在京之俄馆，嗣后仅止来京之俄人居住。俄使请造庙宇，中国办理俄事大臣等帮助于俄馆盖庙，现在驻京喇嘛一人，复议补遣三人，于此庙居住，俄人照伊规矩，礼佛念经，不得阻止。”（此条中的“喇嘛”指东正教神父，“庙”指东正教堂。）按照该条规定，雍正十年（1732年），在东江米巷的俄罗斯馆创建了一座“罗刹庙”（即[[东正教|东正教]]堂），称为奉献节教堂（圣玛利亚教堂），人称“南馆”（以与“[[北馆|北馆]]”区别）。奉献节教堂建成后，[[东正教北京传道团|东正教北京传道团]]随之从[[东直门|东直门]]内北官厅一带的胡家圈胡同的罗刹庙（[[北馆|北馆]]）迁入东江米巷（东交民巷）的南馆。<ref>{{Cite web |url = http://news.ifeng.com/a/20140910/41915540_0.shtml |title = 老北京有座“罗刹庙” |author =  |date = 2014-09-10 |publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref><ref name=tx>{{Cite web|url =http://www.mzb.com.cn/html/Home/report/345883-20001.htm|title =试论东正教在北京的传播|author =佟洵|date =|publisher =北京联合大学学报1999(2)|language =|accessdate =|quote =}}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

[[第二次鸦片战争|第二次鸦片战争]]后，俄国援引1860年中俄《[[中俄北京条约|北京条约]]》有关条款，于1861年直接由俄国政府外交部派公使进驻北京，无须再以东正教北京传道团充任公使之职能，并且将东正教北京传道团划归俄罗斯东正教最高宗务会议直接领导，并将东正教北京传道团所驻的东江米巷南馆改成了俄国驻华公使馆，令东正教北京传道团迁回北馆居住。从1861年到1889年，东正教北京传道团管辖的机构，包括[[北馆|圣索菲亚教堂]]（北馆），奉献节教堂（南馆），以及[[安定门|安定门]]外东安定村的[[圣母堂_(北京)|圣伊尔库茨基·英诺肯提乙教堂]]。同时还辅有修道院，以及教会办的男女学校等。<ref name=tx/>

1861年原南馆改建为俄国驻华公使馆后，原有建筑除奉献节教堂以外，全部拆除。俄国公使馆与[[英国使馆旧址|英国公使馆]]相邻，在《[[辛丑条约|辛丑条约]]》签订后，两国公使馆面积都有所扩大，平分了由东交民巷、[[东长安街|东长安街]]、西河沿、户部街围合的街区。俄国公使馆圈占了原[[鸿胪寺|鸿胪寺]]南半部分及原[[钦天监|钦天监]]，俄国兵营占据了原[[太医院|太医院]]，练兵场占据了原[[礼部|礼部]]和[[工部|工部]]的一部分。1917年以前，俄国都从[[圣彼得堡大学|圣彼得堡大学]]东方语言学院教师中以及传教士中选拔俄国驻华公使馆人员。

1924年，[[中华民国|中华民国]]（[[北洋政府|北洋政府]]）同[[苏联|苏联]]正式建交后，原俄国驻华公使馆改为苏联驻华公使馆，原俄国兵营的一部分仍交由苏联驻华公使馆使用，苏联驻华公使馆也成为[[中国共产党|中国共产党]]、[[中国国民党|中国国民党]]在北京开展活动的场所，两党在北京的领导人[[李大钊|李大钊]]曾和家人居住在苏联驻华公使馆内的原俄国兵营里。1927年，[[奉系|奉系]][[张作霖|张作霖]]策动北京军警突袭苏联驻华公使馆，逮捕了李大钊。4月28日，李大钊被处决。

1928年[[北伐战争|北伐战争]]结束后，苏联驻华公使馆也随之迁往首都[[南京|南京]]，在东交民巷仅留下一个武官留守处。1949年[[北平和平解放|北平和平解放]]后，[[中国人民解放军|中国人民解放军]]开入北平城，在列队通过东交民巷时，其他国家使领馆均关闭大门，仅有苏联外交人员热烈欢迎。此后由于中苏友好关系，1950年1月[[中国人民解放军北京市军事管制委员会|中国人民解放军北京市军事管制委员会]]宣布征用东交民巷外国兵营时，仅宣布征用[[美国|美国]]、[[英国|英国]]、[[法国|法国]]、[[荷兰|荷兰]]四国兵营，并未征用原俄国兵营，也未在苏联代理处外张贴布告。<ref>{{Cite web |url =http://news.ifeng.com/a/20150626/44051191_0.shtml  |title = 新中国收回东交民巷外国兵营始末 |author =  |date =2015-06-26  |publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref>1950年7月，苏联将原俄国兵营正式交还中方，苏联驻华大使馆外交人员仍在原处居住、办公，改由苏联向中国租用。1950年代，苏联驻华大使馆新馆（位于原北馆处）建成后，苏联驻华大使馆正式从东交民巷迁出。后来，在东交民巷原俄国使馆旧址上，建设了最高人民法院办公大楼，奉献节教堂也在1992年被拆除。原俄国使馆旧址已无任何遗迹。

==建筑==
===俄国使馆旧址===
俄国使馆旧址位于东交民巷27号，现为最高人民法院第一办公区。俄国使馆旧址已无存。

2000年代，在文物部门建议下，最高人民法院依照历史上俄国使馆的建筑形制和风格，在东交民巷北侧建设了沿街建筑。这些沿街建筑新建完成后，基本恢复了东交民巷该路段的统一风貌。其中在今最高人民法院大门以东和以西，按照原俄国使馆大门的原貌分别复建了一座大门，作为点景之用。今最高人民法院大门和原俄国使馆建筑则没有任何关系。<ref>{{Cite web|url =http://www.bj.xinhuanet.com/bjpd_sdwm/2007-11/13/content_11659855.htm|title =东交民巷使馆区有望整体保护|author =|date =2007-11-13|publisher =新华网|language =|accessdate =|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20170104235843/http://www.bj.xinhuanet.com/bjpd_sdwm/2007-11/13/content_11659855.htm|archivedate =2017-01-04}}</ref>

===俄国兵营旧址===
俄国兵营旧址位于今东交民巷29号及35号。今东交民巷29号原为甲29号、29号、甲31号、31号、35号（部分），其中旧门牌的东交民巷29号曾经是[[中华人民共和国外交部|中华人民共和国外交部]]的信使队等部门的集体宿舍。<ref>{{Cite web |url =http://news.hebei.com.cn/system/2015/05/15/015530349.shtml  |title = 关宗山：为国家机密护航16年 |author =  |date =2015-05-15 |publisher =长城网  |language =  |accessdate =  |quote =  }}</ref>2011年，东交民巷甲29号、29号、甲31号、31号、35号（部分）拆迁新建，成为新的东交民巷29号。<ref>{{Cite web|url =http://www.bjjs.gov.cn/publish/portal0/tab662/info64765.htm|title =东城区东交民巷29号、31号院项目拆迁单位招标公告|author =|date =2011-07-29|publisher =北京市住房和城乡建设委员会|language =|accessdate =|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20170105083719/http://www.bjjs.gov.cn/publish/portal0/tab662/info64765.htm|archivedate =2017-01-05}}</ref>2012年，旧门牌的东交民巷29号、31号、甲31号作为“原俄国使馆武官处”被列为[[北京市东城区普查登记文物|北京市东城区普查登记文物]]。2014年，对新的东交民巷29号中的历史建筑即“原俄国使馆武官处”的修缮基本完成。<ref>{{Cite web |url = http://www.chinadaily.com.cn/hqgj/jryw/2014-01-09/content_10993692.html |title = 前门铛铛车延至永定门正研究 已委托专业部门做设计 |author =  |date = 2014-01-09 |publisher =  中国日报网 |language =  |accessdate =  |quote =  }}</ref>新的东交民巷29号隶属[[北京市人民政府宽沟招待所|北京市人民政府宽沟招待所]]。<ref>{{Cite web|url =http://www.huashangyuanda.com/yewut_detail.html|title =工程业绩|author =|date =|publisher =华商远大|language =|accessdate =2017-01-05|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20170104234223/http://www.huashangyuanda.com/yewut_detail.html|archivedate =2017-01-04}}</ref><ref>{{Cite web |url = http://www.bidcenter.com.cn/newscontent-11524928-4.html |title = 东交民巷29号工程消防工程中标结果 |author =  |date = 2012-12-04 |publisher = 中国采招网 |language =  |accessdate =  |quote =  }}</ref> 今东交民巷35号为[[中国人民武装警察部队北京市总队|中国人民武装警察部队北京市总队]]第十支队，正门开在[[新大陆_(胡同)|新大陆]]东侧，挂“东交民巷35号”门牌。
