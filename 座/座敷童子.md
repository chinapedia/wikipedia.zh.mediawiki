'''座敷童子'''主要指[[岩手縣|岩手縣]]的傳說中的一種[[精靈|精靈]]，是住在家宅和倉庫裡的神<ref name="nomura">{{Cite book<!--|和書-->|author=渡辺公一|editor=野村純一他編|title=昔話・伝説小事典|year=1987|publisher=みずうみ書房|isbn=978- 4-8380-3108-5|pages=121頁|chapter=ザシキワラシ}}</ref>。按照傳說，座敷童子常常戲弄家裡的人，會為見到它的人帶來幸運，有座敷童子在的家庭會很富足。 [[柳田國男|柳田國男]]的『[[遠野物語|遠野物語]]』對座敷童子記述道「這個神寄宿的家庭會富貴自在穩定」「它的名字叫做座敷童子」。

== 概要 ==
座敷童子的傳言主要來自[[岩手縣|岩手縣]]中心地區，[[青森縣|青森縣]]、[[宮城縣|宮城縣]]、[[秋田縣|秋田縣]]和[[東北地方|東北地方]]各縣也有分佈。

一般對祂的表述為紅臉垂髪的五，六歲小孩子，不過根據住家地區不同，其外表的年齡範圍可以下到三歲，上到十五歲左右<ref name="dis05m">{{Cite book<!--|和書-->|author=[[村上健司|村上健司]]|editor=講談社コミッククリエイト編|title=DISCOVER妖怪日本妖怪大百科|year=2008|publisher=[[講談社|講談社]]|series=KODANSHA Officisil File Magazine|volume=VOL.05|isbn=978-4-06-370035-0|pages=10-11頁|chapter=特集座敷わらしとは何か?}}</ref>。一般留[[御河童|御河童]]髪型或平頭。性別有男有女，男孩子多穿帶條紋的黑色絣織和服，女孩子則多著紅色的[[小袖|小袖]]和服，有的時候會穿[[振袖|振袖]]<ref name="dis05m" />。也有分不出其性別的時候，有時一個家庭中也有可能寄宿了多個座敷童子<ref name="dis05m" />，甚至同時有複數的男女童子都存在的情況。也有傳說其可能會以黑色的動物或武士的形態出現<ref name="dis05m" />。

座敷童子非常喜歡惡作劇，有時會在家裡留下用煤灰或白粉做的小腳印，或是在夜晚故意不停搞出[[紡車|紡車]]的聲音，讓起居室裡充滿歌舞聲。有人獨自在家縫紉時，聽到隔壁的房間有紙張沙沙作響的聲音，間或夾著噴嚏聲，明明沒有人的空屋卻傳出談話聲。座敷童子還會在半夜裡故意在客人的床上或枕邊來回奔跑，或大力壓在人身上，讓人噩夢連連<ref name="nomura" />。

座敷童子很喜歡和小孩子一起玩耍<ref name="nomura" />。很多其它地方的人會到岩手的早池峰神社參拜，有的時候座敷童子會出現在小孩子中間，並教他們岩手的童謠<ref>{{Cite book<!--|和書-->|author=怪談レストラン編集委員會・[[松谷みよ子|松谷みよ子]]責任編集|title=座敷わらしレストラン|year=2007|publisher=[[童心社|童心社]]|series=[[怪談レストラン|怪談レストラン]]|isbn=978-4-494-01373-9|pages=17-25頁}}</ref>。

家裡的小孩子一般可以看得見座敷童子，而成年人看不見它，所以成年人常常會看到小孩子對著空氣說話或嬉笑。有的時候在數小孩子的數目時，會發現多了一個誰也不認識的小孩。文學上常常以此為題材<ref name="tada">{{Cite book<!--|和書-->|author=[[多田克己|多田克己]]|title=幻想世界の住人たち|volume=IV|year=1990|publisher= [[新紀元社|新紀元社]]|series =Truth in fantasy|isbn=978-4-915146-44-2|pages=256-261頁}}</ref>。

== 和家庭興衰之間的關聯性 ==
其在[[民間信仰|民間信仰]]中最大的特徵是，座敷童子在則家庭興旺，座敷童子去則家庭敗落。即座敷童子和[[惠比寿|福神]]很相似，被認為是掌管家庭興衰的守護靈<ref name="tada" /><ref name="sakurai">{{Cite book<!--|和書-->|author=[[桜井徳太郎|桜井徳太郎]]編|title=民間信仰辭典|year=1980|publisher=[[東京堂出版|東京堂出版]]|isbn=978-4-490-10137-9|pages=134-135頁}}</ref>。

『遠野物語』有記載座敷童子離開後整個家族因為食物中毒而全部滅絕的傳說：
[[岩手縣|岩手縣]]土淵村（現在的[[遠野市|遠野市]]）大字飯豊的資産家的座敷童子被小孩子射箭射到，座敷童子便離開了，其家運頃刻之間即消失，很快就敗落了<ref name="tono">{{Cite book<!--|和書-->|author=[[柳田國男|柳田國男]]|title=[[遠野物語|遠野物語]]・山の人生|year=1976|publisher=[[岩波書店|岩波書店]]| series=[[岩波文庫|岩波文庫]]|isbn=978-4-00-331381-7|pages=23-25頁}}</ref>。有人認為座敷童子去則家庭敗落的說法，和[[貧乏神|貧乏神]]去則家庭重新富足的說法是相關聯的<ref name="shinwadensetsu" />。

也有見到白色的童子是吉兆，看到紅色的童子（有紅的臉，穿著紅的衣服，手中提著紅色小桶的童子）從家裡出走是噩兆的說法。前面提到的整個家族中毒死的事例中就有人看到了紅童子<ref name="kowaihanasi">{{Cite book<!--|和書-->|author=杉山茂勲|editor=早川和樹編|title=こわい話- あなたの知らないニッポンの“恐怖”|year=2008|publisher={{tsl|ja|ミリオン出版|}}|isbn=978-4-8130-2076-9|pages=100-103頁|chapter=座敷童子の部屋}}</ref>。

== 風俗 ==
一些日本豪族認為其舊宅里有座敷童子，不可以在家裡說不相信座敷童子這樣的話，甚至會每日進行供奉。據說座敷童子喜愛小豆飯，所以他們每天都會向其供奉小豆飯，如果供奉的飯沒有減少則代表家將衰敗<ref name="tada" />。雖然座敷童子和[[犬神|犬神]]的情況比較類似，但和犬神相反的是，人們認為座敷童子是一種保護神，對其充滿敬畏，也希望其到自己的家中落腳。

[[二戸市|二戸市]]一帶的居民會在自己的家中供奉童子的畫像，並在祭壇上擺上和菓子或是玩具，這個風俗殘留至今。有些風俗則是專門在家中為座敷童子建一間小屋，希望能夠留住它，以保持家族興旺<ref name="kowaihanasi" />。

『遠野物語』中提到，土淵村的富豪家中有一個叫做「座頭部屋」的小房間，舊時家裡開宴會時，都會專門派人到該房間門口呼喚座頭並等待片刻，以示邀請座敷童子<ref>{{Cite book<!--|和書-->|title=遠野物語・山の人生|pages=48-52頁}}</ref>。文學研究者[[三浦佑之|三浦佑之]]認為座敷童子是家中的守護靈<ref name="shinwadensetsu" />。

[[青森縣|青森縣]][[五戸町|五戸町]]的居民在新修房屋的時候，會在地板下面埋一個金球，希望座敷童子來拍球，給家族帶來幸福<ref>{{Cite book<!--|和書-->|title=幻想世界の住人たち|volume=IV|pages=275頁}}</ref>。

== 各種各樣的座敷童子 ==
座敷童子根據所處的地區，還被稱為「座敷童」、「座敷童眾」、「座敷少爺」、「禦蔵ボッコ」、「座敷小僧」、「カラコワラシ」。「座敷少爺」的稱呼主要用在岩手內陸部，[[宮沢賢治|宮沢賢治]]的著作中有提到這個名字<ref>{{Cite book<!--|和書-->|author=[[高橋貞子|高橋貞子]]|title=DISCOVER妖怪日本妖怪大百科|volume=VOL.05|pages=7頁|chapter=座敷わらしの棲む家}}</ref>。

『[[十方庵遊歴雑記|十方庵遊歴雑記]]』中有關於在岩手縣[[江刺市|江刺市]]（現在的[[奧州市|奧州市]]江刺區）稲瀬的座敷童子的記錄：
家裡的土間中出現了座敷童子，不斷重複「コメツキワラシ」，「ノタバリコ」和「ウスツキコ」這幾句話，其中長的最白的穿的最華麗的座敷童子被稱為「チョウピラコ」<ref name="sasaki">{{Cite book<!--|和書-->|author=[[佐佐木喜善|佐佐木喜善]]|title=遠野のザシキワラシとオシラサマ|year=2007|publisher=[[中央公論新社|中央公論新社]]|series= [[中公文庫|中公文庫]]|volume=|isbn=978-4-12-204892-8|pages=15-55頁}}</ref>。且不論其是否和家族興衰有關，家中總是出現這種怪聲，還伴隨惡臭，讓人不免對其產生厭惡<ref name="dis05m" />。

據說如果座敷童子伸出像藤蔓一般的細手向人招手的話，則代表將有洪水或海嘯，因此也有人用「細手（ほそで）」或「細手長手（ほそでながて）」稱呼它們<ref name="sasaki" />。 「クラワラシ」，「[[倉少爺|倉少爺]]」則是用來稱呼生活在倉庫中的座敷童子。

儘管東北地方有很多關於座敷童子的傳說，[[秋田縣|秋田縣]]卻出奇的少，這是因為秋田[[三吉鬼|三吉鬼]]的存在，使當地人認為座敷童子是下等妖怪<ref name="tada" />。

== 起源 ==
[[File:Akinari_Zeni-no-sei.jpg|right]]『[[雨月物語|雨月物語]]』的「貧富論」中的銭的靈魂]]
民俗學者[[佐佐木喜善|佐佐木喜善]]認為座敷童子是在家裡被殺害並埋在地板下的小孩子的靈<ref>{{Cite book<!--|和書-->|author=志村有弘監修|title=図説地図とあらすじで読む日本の妖怪伝説|year=2008|publisher=[[青春出版社|青春出版社]]|isbn=978-4- 413-00965-2|pages=71 頁}}</ref>。東北地方曾出現過「臼殺」，是為了減輕家裡的生活負擔，將多出來的小孩用口趕到石臼下面壓死，並將其埋在土間的廚房下面的習俗。可能由於大雨或是地震使小孩的殘骸暴露出來（甚至是小孩的靈魂出現）被外人看到，家人為了掩蓋事實即說是座敷童子出現<ref name="kowaihanasi" />。因此前面提到的「ノタバリコ」或「ウスツキワラシ」是座敷童子中最下等的，前者意指從土間裡面爬出來，後者則是使用石臼的聲音還伴隨惡臭<ref name="sasaki" />，這些都代表了座敷童子和被「臼殺」並被埋在土間的小孩有很大關聯<ref name="dis05m" />。

除了前面提到的關聯，也有人將座敷童子和從村外來的六部（巡禮僧）殺害老屋原本的住戶造成家族沒落的傳說相結合，認為座敷童子是古舊的村落中黑暗歷史的象徴<ref name="shinwadensetsu" />。

[[高橋貞子|高橋貞子]]在其著作『座敷わらしを見た人びと』有其他的看法，認為座敷童子是製作榻榻米工匠在對這家人的怨氣，或者是其為了進行詛咒在施工時放入柱子或者樑柱的用薄木片削成的人形幻化而成的<ref name="dis05m" />。

也存在座敷童子的正體其實是[[河童|河童]]的說法，認為其是住宅附近的湖里住的河童進入住家惡作劇或是定居的結果<ref>{{Cite book<!--|和書-->|title=遠野のザシキワラシとオシラサマ|pages=65-67頁}}</ref>。

另外，座敷童子作為小孩子的姿態，可能是由[[佛教|佛教]]的[[護法童子|護法童子]]的姿態和[[民間信仰|民間信仰]]相結合的結果<ref name="sakurai" /><ref>{{Cite book<!--|和書-->|author=[[岩井宏實|岩井宏實]]監修|title=日本の妖怪百科|year=2000|publisher=[[河出書房新社|河出書房新社]]|volume=4|isbn=978-4-309-61384-0|pages=30頁}}</ref><ref name= "shinwadensetsu">{{Cite book<!--|和書-->|author=三浦佑之|editor=吉成勇編|title=日本「神話・伝説」総覧|year=1992|publisher= [[新人物往來社|新人物往來社]]|series=歴史読本特別増刊・事典シリーズ|isbn=978-4-4040-2011-6|pages=256-257 頁|chapter=ザシキワラシ伝説}}</ref>。

柳田國男認為座敷童子信仰可能是對護法童子的信仰，對民間巫女的守護霊的信仰，若葉地區對清新的魂的信仰和認為神降人間成為家庭守護霊的信仰的結合體。柳田在[[民俗學|民俗學]]方面的論點則認為座敷童子和如竜宮童子一類的由異界來到人間的童子有很大聯繫<ref name="shinwadensetsu" />。

此外，[[民俗學|民俗學]]者[[小松和彥|小松和彥]]則提出他的[[文化人類學|文化人類學]]觀點：度量村落的興衰程度的一個最明顯的標誌即是兒童，富裕的家庭則可以有多個孩子，而貧困的家庭則可能造成小孩的夭折，可以從座敷童子的存在獲得村落貧富差距的變化信息<ref name="shinwadensetsu" />。

[[上田秋成|上田秋成]]在[[江戶時代|江戶]]後期出版的讀本『[[雨月物語|雨月物語]]』中的「貧富論」裡描述了在[[陸奧國|陸奧國]]（現在青森縣）的武士家中出現了代表銭的靈魂名叫「黃金的精霊」的小老頭，和武士進行了「關於錢的愉快的對話」，文學博士阿部正路認為這就是座敷童子的原型<ref>{{Cite book<!--|和書-->|author=阿部正路|title=日本の妖怪たち|year=1981|publisher=[[東京書籍|東京書籍]]|isbn=978-4-487-72164-1|pages=117 -120頁}}</ref>。

== 故事背景 ==
在镰仓幕府时期，东北岩手县乡下的一间破屋子里，住着一个小女孩和她母亲，因为父亲去世，而母亲又生重病，所以变成母女两的生活靠女儿天天上山采药来维持。一天清晨，小女孩老样子去采药，可突然没回来，但采的药还是有出现，母亲问附近的人都不知道是怎么一回事。之后，母亲决定埋伏，但却发现居然是自己女儿的身影，母亲大喊一声，衝了出去，女孩闻声就逃跑，母亲紧跟着，却怎么也追不上，两人一直跑到一座悬崖，然后女儿便眼泪汪汪的望着母亲然后跳进雾中消失不见，然后母亲向前走去，发现-{只}-有一个药篮与一双鞋子。原来女儿意外失足身亡，但因为放不下母亲的执念所以又回来，也因为生前的不幸，不希望其他人也不幸，所以变成座敷童子。而且特别喜欢帮助穷困的人家。
