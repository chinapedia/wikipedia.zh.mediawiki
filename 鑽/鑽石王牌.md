《'''鑽石王牌'''》（{{lang-ja|ダイヤのA}}）是[[寺嶋裕二|寺嶋裕二]]創作的[[棒球漫畫|棒球漫畫]]，在《[[週刊少年Magazine|週刊少年Magazine]]》2006年第24號開始連載。而漫畫預計分兩部，第一部在2015年第7號完結（2015年1月14日），之後也預告將在10、11號出番外。至於第二部則在2015年第38號繼續連載。

== 概要 ==
此部作品採用「[[棒球留學|棒球留學]]」的強校背後的努力，與其他作品凸顯主角球隊為「弱小的球隊」不同。作為主角舞台的青道高中，因校舍中寫著「人間是宝」與「今日學習（{{lang|ja|今日学べ}}）」的匾額，推測是以埼玉榮高校做為基礎來描寫的。本作第53回（平成19年度）[[小學館漫畫賞|小學館漫畫賞]]少年漫畫部門得主（同時也是[[週刊少年Magazine|週刊少年Magazine]]連載作品中頭一位獲得此獎賞之作品），也於第34回（平成22年度）講談社漫畫賞少年部門受賞。2013年10月6日，電視動畫開播。<br/>
第1季1-13話片尾曲「Seek Diamonds」同時是原作411話、KPC版23卷及動畫第2季51話的標題，象徵戰勝藥師後取得甲子園參賽資格（也是actII的分水嶺），但作為片尾曲並不包含此意義。

== 故事簡介 ==
; 第1部『鑽石王牌』
: 於[[初級中學|-{zh-cn:初中;zh-hk:初中;zh-tw:國中;}-]]時期最後的大賽，統合即將被廢止的赤城中學參賽，卻在第一回戰中投出暴投，而輸掉比賽的[[投手|投手]]'''澤村榮純'''，本來決定要與-{zh-cn:初中;zh-hk:初中;zh-tw:國中;}-時期的朋友一同考入故鄉的高中，卻被[[東京|東京]]的棒球名校青道高中挖角，並在那裡遇見天才[[捕手|捕手]]'''御幸一也'''，就此改變他的人生…。

; 第2部『鑽石王牌 act2』
:「我還能繼續，還能更上一層樓，向更前方，只要和這個人一起。」

== 電視動畫 ==
