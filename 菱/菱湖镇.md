'''菱湖镇'''，是[[中国|中国]][[浙江省|浙江省]][[湖州市|湖州市]][[南浔区|南浔区]]下属的一个[[镇|镇]]，总面积108平方公里，辖28个[[行政村|行政村]]和3个[[居委会|居委会]]，常住人口10万人<ref>[http://lh.chinananxun.gov.cn:96/81/200531885707.htm 菱湖概况] {{webarchive|url=https://web.archive.org/web/20090715231133/http://lh.chinananxun.gov.cn:96/81/200531885707.htm |date=2009-07-15 }}</ref> 。该镇地处[[杭嘉湖平原|杭嘉湖平原]]中部，湖州市区以南27公里，杭州以北90公里，水网密布，拥有大片的桑基鱼荡，是著名的丝绸、鱼虾产地，该镇也是[[沈尹默|沈尹默]]、[[沈兼士|沈兼士]]、[[沈迈士|沈迈士]]等众多名人的家乡。

== 歷史 ==
菱湖镇是一个历史悠久的[[江南|江南]]水乡古镇。古代神話[[羿射九日|羿射九日]]和[[嫦娥奔月|嫦娥奔月]]的傳說就發生在菱湖鎮[[射中村|射中村]]。5000至3000年前就已有先進的[[良渚文化|良渚文化]]，4700年前的[[防風氏|防風氏]]，又是古代著名的抗洪英雄，菱湖鎮屬於古[[防風國|防風國]]轄區。

古名秀溪，又名淩波塘。原是被用來種菱的大湖，早于[[唐|唐]][[宝历|宝历]]元年(公元825年)，[[刺史|刺史]][[崔元亮|崔元亮]]修建凌波塘(印塘路)及秀溪桥(即务桥)。遂于塘与桥之间集市成镇。贸易繁盛，时有“水市一之称（见《[[唐书|唐书]]》）。周围湖泊，处处种菱，唐诗僧[[皎然|皎然]]诗中有「秋色起菱湖」及「路入菱湖深」等句，每年秋季“叢叢菱葉隨波起，朵朵菱花背日開”。因「其地宜菱，以旁菱湖而名。」於是，淩波、秀溪、龍湖則成了菱湖鎮的別名。

[[北宋|北宋]]初，罷鎮將、設巡檢寨，屬[[歸安縣|歸安縣]]。[[南宋|南宋]]時興市厘，建橋修路，設務關掌茶酒稅務，設四柵，初具市鎮規模。[[明朝|明朝]][[洪武|洪武]]十五年（1382年）設菱湖鎮，置稅庫局務官，設予備倉；成化年間設湖秀巡檢署，後設思溪河泊所。至[[明朝|明朝]][[嘉靖|嘉靖]][[萬曆|萬曆]]年間（1522年-1620年）經濟繁榮，蠶絲發達，池蕩人工養魚遍布，成為「東南巨都」、「歸安雄鎮」；[[清朝|清朝]][[乾隆|乾隆]]年間以後，歸安縣主簿署移駐菱湖，設厘稅局，並進入相當繁榮時期，「弟宅雲連，蔚為冠蓋」。[[光緒|光緒]]二年（1876年）人口達6656人，成為[[湖州府|湖州府]]第一大鎮。1912年民國廢[[歸安|歸安]]、[[烏程|烏程]]二縣，合並為[[吳興縣_(中華民國)|吳興縣]]，菱湖為縣屬鎮。

== 沿革 ==
[[File:Lin_Gheu_Old_Map.JPG|thumb]]
*[[唐|唐]]屬[[烏程縣|烏程縣]]；
*[[宋|宋]]屬[[歸安縣|歸安縣]]；
*[[民國元年|民國元年]]（1912年）屬[[吳興縣_(中華民國)|吳興縣]]；
*1949年4月28日，菱湖和平解放，為[[吳興縣|吳興縣]]直屬鎮。1949年5月至1954年10月，[[吳興縣|吳興縣]]委縣政府機關設在菱湖鎮。
*1958年屬菱湖公社。
*1959年恢複菱湖鎮，鎮人民政府設在廣場橋北堍，系原菱湖區委、溪西公社所在地。
*1986年11月，溪西鄉撤銷，並入菱湖鎮。
*1993年隸屬[[菱湖區|菱湖區]]。
*1999年10月，下昂鎮、新溪鄉並入菱湖鎮。
*2003年行政區劃改革撤銷[[菱湖區|菱湖區]]，改為[[南潯區|南潯區]]所轄。

== 行政區劃 ==
菱湖鎮下轄6個[[社區|社區]]、28個[[行政村|行政村]]（574個村民小組）（2007年数据），鎮政府駐地為菱湖鎮東柵路8號。

* '''社區'''：西栅社区、东栅社区、南栅社区、北栅社区、下昂社区、建豐社区。

* '''行政村'''（括注[[村民小组|村民小组]]数）：東河(14)、南浜(14)、勤勞(14)、王介墩(22)、陳邑(16)、楊介巷(25)、費介埭(21)、勤儉(18)、水產(6)、沈介埭(6)、南雙林(36)、盧介莊(33)、永福(17)、菱東(19)、建豐(17)、永豐(16)、達民(24)、思溪(27)、下昂(23)、新廟里(33)、竹墩(22)、六堡里(23)、射中(38)、千豐(28)、三溪(20)、山塘(14)、許聯(16)、楊港(12)。
