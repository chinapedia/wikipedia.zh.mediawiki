'''树兜站'''位于[[福州市|福州市]][[鼓楼区_(福州市)|鼓楼区]]五四路与华林路交叉路口，是[[福州地铁|福州地铁]][[福州地铁1号线|1号线]]的一座中间站，亦是距离福州五四路CBD最近的车站，2017年1月6日开通<ref>{{cite news|author=原浩|url=http://news.fznews.com.cn/dsxw/20170104/586c38fd119eb.shtml|title=福州地铁1号线（一期）全线1月6日正式开通试运营|publisher=福州新闻网|date=2017-01-04|accessdate=2017-01-05}}</ref>。

== 站名来源 ==
《闽都别记》中记载，树兜所在位置曾是瓮城，呈现衣兜状，起到保护福州城的作用。福州人形象地把瓮城所在地叫“兜”。又由于这里树木较多，就被称为“{{僻字|𣗬|木臭}}兜”，后改寫为“树兜”。<ref>
{{Cite web |url=http://digi.dnkb.com.cn/dnkb/html/2013-04/21/content_260201.htm |title=斗门附近山头曾葬着抗战士兵 旧时要进福州得先过树兜城门 |access-date=2017-01-01 |archive-url=https://web.archive.org/web/20130511020004/http://digi.dnkb.com.cn/dnkb/html/2013-04/21/content_260201.htm |archive-date=2013-05-11 |dead-url=yes }} {{Wayback|url=http://digi.dnkb.com.cn/dnkb/html/2013-04/21/content_260201.htm |date=20130511020004 }}
</ref>

=== 站厅 ===
{{ multiple image | align = right | direction = vertical | width = 220
| image1 = 20170101 Concourse of Shudou Station.jpg | caption1 = 站厅闸机

树兜站站厅位于华林路与五四路交叉路口下方。车站站厅采用白色直线型吊顶与白色搪瓷板装修，设客服中心2处，分布于站厅东西两侧，服务设施规划设置智能导乘机2处、自动售货机2处、ATM柜员机3处，C出入口通道旁规划设有[[便利店|便利店]]，D出入口通道设有洗手间。

=== 周边 ===
树兜站位于福州五四路CBD北端，周边以商务用地与住宅用地为主。五四路商务区可由C、D出入口出向南步行抵达。福建中医药大学屏山校区、福州市第二人民医院、福建经贸会展中心可由B口出步行抵达，福州大戏院、福州屏东中学、福州市鼓楼实验小学可由C口出步行抵达，温泉公园、福州市行政服务中心、鼓楼区行政服务中心、温泉小学、银河花园大饭店可由D口出步行抵达。

== 事故 ==
* 2015年8月10日，受强台风“[[台风苏迪罗_(2015年)|苏迪罗]]”的大雨影响，树兜站盾构井进水，导致井内9名工人被困，此后9名工人全部获救。<ref>[http://news.fznews.com.cn/fuzhou/20150811/55c9adeadb06b.shtml 福州地铁树兜站盾构井进水 9名被困工人全部获救]</ref>
