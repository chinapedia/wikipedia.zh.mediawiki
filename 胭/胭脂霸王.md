{{engname|《胭脂霸王》|''King Rouge''}}，2013年上映于中国大陆的民初剧，导演王重光，主演杨紫、赵显宰、李修贤、姜超、王今铎、晨梓妍。<ref>{{cite web |url= http://yule.sohu.com/20130222/n366753903.shtml|title= 《胭脂霸王》湖南开播 姜超大反派成最大亮点|accessdate=2013 |publisher=搜狐网 |work= |language=zh-hans}}</ref><ref>{{cite web |url= http://ent.163.com/13/0226/13/8OL46A1000031GVS.html|title=《胭脂霸王》湖南卫视热播 人物关系剧照曝光 |accessdate= 2013|publisher=网易 |work= |language=zh-hans}}</ref><ref>{{cite web |url=http://ent.sina.com.cn/v/m/2013-02-24/09473862764.shtml |title=《胭脂霸王》湖南热播 传递正能量 |accessdate=2013 |publisher=新浪网 |work= |language=zh-hans}}</ref>2013年2月23日[[湖南卫视|湖南卫视]](上午)首播。

==剧情==
该剧通过“雷儿”这个女镖师霸王的塑造讲述了近代镖局的变迁历程。
