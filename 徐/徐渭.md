'''徐渭'''（{{bd|1521年|3月12日|1593年||Xu徐}}），字文長，號青藤老人、青藤道士、天池生、天池山人、天池漁隱、金壘、 金回山人、山陰布衣、白鷳山人、鵝鼻山儂、田丹水、田水月，中国[[明代|明代]]文学家、书画家、軍事家，[[浙江|浙江]][[山阴县|山阴县]]（今属[[浙江|浙江]][[绍兴市|绍兴市]]）人<ref name="rmrb1961">{{Cite news|title=纪念我国古代十大画家|author=左海|publisher=《[[人民日报|人民日报]]》|date=1961-03-04|page=8}}</ref>。

== 生平 ==
[[正德|正德]]十六年（1521年），二月初四日出生於浙江[[绍兴|绍兴]]府山阴城大云坊的官僚世家。父[[徐鏓|徐鏓]]官至四川夔州府同知，妻童夫人，生长子徐淮，次子徐潞。童夫人亡，續娶苗夫人，未生育，納其苗氏侍女為妾，生徐渭。不久徐鏓死於紹興，苗夫人視徐渭為己出。

徐渭少年时天才超逸，入徐氏私塾读书，“六歲受《大學》，日誦千餘言”，“书一授数百字，不再目，立诵师听。”<ref>《畸谱》</ref>，十歲仿[[揚雄|揚雄]]《解嘲》作了一篇《釋毀》，性格豪放，“指掌之间，万言可就。”<ref>《上提学副使张公书》</ref>，二十岁时成为生员。[[嘉靖|嘉靖]]二十年（1541年）入赘同县潘克敬家，接下來八次应试不中，“再试有司，皆以不合规寸，摈斥于时。”<ref>《上提学副使张公书》</ref>，妻子十四歲與徐渭结婚，夫妻十分恩愛，惜僅十九歲即病故。<ref>《亡妻潘墓志铭》稱：“生则短而死则长，女其待我于松柏之阳。”</ref>嘉靖二十四年，長兄徐淮服食丹藥而死，因已入贅潘家，無法繼承遺產。<ref>《赠妇翁潘公序》云：“其后乙巳，某以卜居为豪无赖所诖误，家殆尽。”</ref>嘉靖二十六年（1547年）在[[山阴|山阴]]城东赁房设馆授徒，與紹興文人[[蕭勉|蕭勉]]、[[陳鶴|陳鶴]]、[[楊珂|楊珂]]、[[朱公節|朱公節]]、[[沈鍊|沈鍊]]、[[錢楩|錢楩]]、[[柳文|柳文]]、[[諸大綬|諸大綬]]、[[呂光升|呂光升]]等號“越中十子”。<ref>《紹興府志·卷六九·徐渭傳》：“渭與蕭柱山勉，陳海樵鶴，楊秘圖珂，朱東武公節，沈青霞錬，錢八山楩，柳少明文，及諸龍泉、呂對明稱越中十子。”</ref>。

后来为[[浙江|浙]][[福建|闽]]总督作幕僚，曾入[[胡宗宪|胡宗宪]]幕府，一切疏計，皆出其手，又出奇计大破[[徐海_(明朝)|徐海]]等[[倭寇|倭寇]]。

一日胡宗憲於[[舟山|舟山]]捕獲白鹿，徐渭撰《進白鹿表》、《再进白鹿表》、《再进白鹿赐一品俸谢表》等並獻於朝中，視為祥物，“上又留心文字，凡俪语奇丽处，皆以御笔点出，别令小臣录为一册”<ref>《[[万历野获编|万历野获编]]》卷十四</ref>，學士[[董汾|董汾]]等對徐渭文章大為賞識。又代作《贺严公生日启》，吹捧奸相[[严嵩|严嵩]]。

嘉靖四十三年（1564年）[[胡宗憲|胡宗憲]]以“党[[严嵩|严嵩]]及奸欺贪淫十大罪”被捕，獄中[[自杀|自杀]]，徐渭作《十白赋》哀之。[[李春芳|李春芳]]严查胡宗宪案，徐渭一度因此发狂，作《自为墓志铭》，以至多次自杀，“走拔壁柱钉可三寸许，贯左耳窍中，颠于地。”，被游方郎中华氏救活，又“引巨錐刺耳，深數寸；又以椎碎腎囊，皆不死。”<ref>《明史·文苑傳》</ref>，都失败，嘉靖四十五年（1566年）懷疑继妻张氏不貞<ref>《徐文长逸稿》卷一一，《上鬱心齋》：“抑不知河间奇节，卒成掩鼻之羞，贾宅重严，乃有窃香之狡。”</ref>，仲秋之夜以鈍器擊殺張氏<ref>[[顧景星|顧景星]]《白茅堂集》卷四十三：“文长之椎杀继妻也，雪天有童跼灶下。妇怜之，假以亵服。文长大詈，妇亦詈。时操欋收冰，怒掷妇，误中。”</ref>，張氏當場致死<ref>[[沈德符|沈德符]]《徐文長傳》：“徐此後遂患狂易，疑其繼室有外遇，無故殺之。”[[陶望齡|陶望齡]]《徐文長傳》中說：“渭為人猜而妬，妻死後有所娶，輒以嫌棄，至是又擊殺其後婦，遂坐法，繫獄中。”徐渭《上鬱心齋書》自言：“頃罹內變，紛受浮言：‘出於忍則入於狂，出於疑則入於矯！’但如以為‘狂’，何不概施于行道之人？如以為‘忍’，何不漫加於先棄之婦？如以為‘多疑’而妄動，則殺人伏法，豈是輕犯之科！如以為‘過矯’而好奇，則蹀血同衾，又豈流芳之事！凡此大凡，雖至愚亦知所避……事難概料，大約如斯。伏望明公曲諒隱衷，力扶公道，勿泥前說，賜挽後評。倘能出萬死於一生，即是垂三綱於九鼎。不勝懇竦，實倍叫號！”徐渭在《畸谱》中又自云：“易（癔）复，杀张下狱。”</ref>，被革去生员资格，下狱七年。<ref>《畸譜》：“易復，殺張下獄。隆慶元年丁卯。”</ref>入狱其间，生母去世。狱中完成《[[周易参同契|周易参同契]]》注释，揣摩书画艺术。

[[萬曆|萬曆]]元年（1573年）除夕，大赦天下，為状元[[張元忭|張元忭]]等所救出獄。<ref>《畸譜》：“除，釋謀歸，飲於吳。明日元旦，拜張座。”</ref>从此潦倒，痛恨达官贵人，浪遊[[金陵|金陵]]、[[宣化|宣]][[蓟辽|遼]]、[[北京|北京]]，又过[[居庸关|居庸关]]赴塞外[[宣化|宣化]]府等地。在遼東總兵[[李成梁|李成梁]]的府中教授其子[[李如松|李如松]]、[[李如柏|李如柏]]兵法。在[[塞外|塞外]]结识蒙古首领[[俺答|俺答]]夫人[[三娘子|三娘子]]。万历五年（1577年）回绍兴，注释[[郭璞|郭璞]]《[[葬书|葬书]]》。晚年以卖画为生，但从不为当政官僚作画，“有书数千卷，后斥卖殆尽。畴莞破弊，不能再易，至借稿寝。”<ref>[[陶望龄|陶望龄]]：《徐文长传》</ref>。
萬曆二十一年（1593年），在貧困中結束一生，年七十三，葬于紹興城南木柵山。

== 傳說 ==
[[浙江|浙江]]民间现在仍然普遍流传他的故事传说，关于他年轻时超凡入聖，后来如何捉弄官宦等。

== 才華 ==
他自己认为自己“书第一，诗二，文三，画四。”，他的草书相当著名，发泄了愤世疾俗之气，被称为“明之草书，以天池生为始。”[[诗歌|诗歌]]得“[[李贺|李贺]]之奇，[[苏轼|苏轼]]之辩”，他的传世[[著作|著作]]有《徐文长全集》，[[剧本|剧本]]《四声猿》，《歌代啸》，嘉靖三十八年撰成[[戏曲|戏曲]]理论著作《[[南词叙录|南词叙录]]》，总结宋、元南戏艺术。{{Fact|另外目前學界亦興起一種說法，認為徐渭就是《[[金瓶梅|金瓶梅]]》的作者[[蘭陵笑笑生|蘭陵笑笑生]]。|time=2018-09-18T21:39:19+00:00}}<ref>{{Cite journal|title=论徐渭和《金瓶梅》|author=胡令毅|url=http://www.nssd.org/articles/article_read.aspx?id=25959051|journal=国家哲学社会科学学术期刊数据库|issue=|doi=|others=|year=2007-06|volume=|page=|pmid=}}</ref>

徐渭的[[绘画|绘画]]新颖奇特，打破了花鸟画、山水画、人物画的题材界限，水墨大写意花鸟笔势狂逸，墨汁淋漓，是寫意花鳥畫成熟的標誌。與陳淳並稱為“青藤”、“白陽”。徐渭在《書謝時臣淵明卷為葛公旦》中指出：“……畫病，不病在墨輕與重，在生動與不生動耳。”他对后来[[清代|清代]][[八大山人|八大山人]]、[[扬州八怪|扬州八怪]]都有很大影响。例如，[[郑燮|郑燮]]自稱“青藤门下一走狗”。近代畫家[[齊白石|齊白石]]曾說：“青藤、雪個、大滌子之畫，能橫塗縱抹，-{余}-心極服之，恨不生前二百年，為諸君磨墨理紙。諸君不納，余于門之外，餓而不去，亦快事故。”[[吳昌碩|吳昌碩]]說：“青藤畫中聖，書法逾魯公。”
