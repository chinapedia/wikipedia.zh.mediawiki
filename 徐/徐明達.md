'''徐明達'''（'''Ming-Ta Hsu'''），[[台灣|台灣]][[生物化學|生化]][[學者|學者]]，[[國立台灣大學|台灣大學]]化學系畢，[[加州理工學院|加州理工學院]]化學生物博士。專長領域為[[分子生物學|分子生物學]]、[[腫瘤基因|腫瘤基因]]、[[電子顯微鏡|電子顯微鏡]]，他是前[[國立陽明大學|陽明大學]]副校長兼生化所教授。<ref>{{Cite web |url = https://www.nctu.edu.tw/component/k2/item/899-2015-05-29-03-09-25 |title = 從有到無再到有的創造力 徐明達教授傳授大學學什麼 |author =  |date = 2015-05-29 |publisher = [[國立交通大學|國立交通大學]] |language = zh-tw |accessdate = 2019-04-21 |quote = 徐明達教授專長為生物化學，臺大化學所取得碩士學位，赴美加州理工學院攻讀生物化學博士學位，於史丹佛大學醫學中心完成生物化學博士後研究。目前在陽明大學擔任教職，是國科會C1高通量基因體分析核心設施主持人及NRPB生技醫藥國家型科技計畫資源中心整合性基因體分析委員會召集人。為普及科學教育，教學研究之餘持續編寫科普書籍，《病毒的故事》、《細菌的世界》等，致力提升大眾生命科學素養，希望消除科學與人文的鴻溝、建立理性與感性的連結。}}</ref>
