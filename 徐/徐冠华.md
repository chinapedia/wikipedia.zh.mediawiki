'''徐冠华'''（{{bd|1941年|12月16日|catIdx=X}}），出生于[[上海|上海]]；[[北京林业大学|北京林业学院]]毕业，遥感应用学专家，[[中科院|中科院]]院士，[[香港城市大学|香港城市大学]]荣誉博士。1984年9月加入[[中国共产党|中国共产党]]，是[[中共|中共]]第十六届中央委员。现任第十一届[[全国政协|全国政协]]常委、教科文卫体委员会主任。曾任[[中华人民共和国科学技术部|中华人民共和国科学技术部]]部长，[[中国科学院|中国科学院]]副院长。

== 早年经历 ==
徐冠华1959年进入[[北京林业大学|北京林业学院]]学习。毕业后，他就被分配到中国林业科学院工作，历任研究实习员、教师、助理研究员、研究员、资源信息所所长等职。1979年至1981年间，徐冠华还曾被派往[[瑞典|瑞典]][[斯德哥尔摩大学|斯德哥尔摩大学]]，从事遥感数字图象处理的研究工作。1992年，他当选为[[中科院|中科院]]地学部学部委员；次年，被任命为[[中科院|中科院]]遥感应用研究所所长；又是一年后，他晋升为[[中科院|中科院]]副院长。

== 进入政界 ==
1995年，徐冠华开始进入政府部门工作，先后出任[[中华人民共和国科学技术部|国家科学技术委员会]]副主任，党组成员、党组副书记，[[科学技术部|科学技术部]]副部长、党组副书记等职。2001年2月，他接替因任职年龄到届的[[朱丽兰|朱丽兰]]出任[[中华人民共和国科学技术部|科技部]]部长；并在十届[[全国人大|全国人大]]一次会议上产生的[[温家宝|温家宝]]“内阁”中，继续留任[[中华人民共和国科学技术部|科技部]]部长。2007年4月，徐冠华也因任职年龄到届而离职。

在2008年3月的十一届[[全国政协|全国政协]]一次会议上，徐冠华当选[[全国政协|全国政协]]常委、教科文卫体委员会主任。

== 学术成就 ==
徐冠华是[[中华人民共和国|中华人民共和国]]遥感应用科学领域的专家，曾多次获得“有突出贡献的中青年专家”、“有突出贡献的留学回国人员”、“有突出贡献的科技人员”等荣誉称号。[[1991年|1991年]]，他还被[[人事部|人事部]]批准享受政府特殊津贴。以表彰他的学术成就及对社会的贡献，2006年11月15日，[[香港城市大学|香港城市大学]]颁授徐冠华荣誉博士。[[2009年6月|2009年6月]]，[[香港中文大学|香港中文大学]]委任徐冠华为该校太空与地球信息科学研究所伟伦太空与地球科学研究客座教授。[[2010年12月|2010年12月]]，徐冠华被香港中文大学授予荣誉博士学位。
