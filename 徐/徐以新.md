'''徐以新'''（{{bd|1911年|11月14日|1994年|12月30日|catIdx=X}}），曾用名'''徐一新'''，出生于[[浙江省|浙江省]][[衢县|衢县]]岭头<ref name="a2"/>（今浙江省[[衢州市|衢州市]]境），[[中华人民共和国|中华人民共和国]]外交官、曾任[[中华人民共和国外交部|中华人民共和国外交部]]副部长。[[第六届全国政协常务委员|第六]]、[[第七届全国政协常务委员|七届全国政协常委]]。

== 生平 ==

生于[[辛亥革命|辛亥革命]]成功之时，其父为他取名“徐一新”寓有“天地一新”之意<ref>{{Cite web |url=http://view.news.qq.com/a/20060508/000038.htm |title=徐以新：历史上著名的半个布尔什维克 |accessdate=2015-04-21 |author= |date=2006-05-08|work= |publisher=腾讯网，来源：党史博览|language=Zh-hans }}</ref>。 1927年1月追随北伐军参加[[北伐战争|北伐战争]]，1927年3月参加[[上海工人第三次武装起义|上海工人第三次武装起义]]。4月，在武汉[[中共中央军委|中共中央军委]]任[[技术秘书|技术秘书]]的徐以新，由[[陈赓|陈赓]]等人介绍加入[[中国共产主义青年团|中国共产主义青年团]]<ref name="a2"/>。其后参与[[南昌暴动|南昌暴动]]，后送入苏联[[莫斯科中山大学|莫斯科中山大学]]，为“[[二十八个布尔什维克|二十八个半布尔什维克]]”的“半个”，早年支持[[王明|王明]]<ref>{{cite news|author=汪云生、尤国珍|title=徐以新 被称为“半个布尔什维克”|url=http://www.28-99.com/index.php?option=com_k2&view=item&layout=item&id=1415&ptype=zgjb&date=2011-01-19&node=03&Itemid=7|accessdate=2012-02-13|newspaper=人物春秋}}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

1930年，由中国共产主义青年团转入[[中国共产党|中国共产党]]。此后回国奔赴[[鄂豫皖苏区|鄂豫皖苏区]]担任[[红四方面军|红四方面军]]总部参谋，随军[[长征|长征]]。后担任延安[[鲁迅艺术学院|鲁迅艺术学院]][[政治部主任|政治部主任]]。1942年5月起，任[[中共中央整风学习委员会|中共中央整风学习委员会]]秘书主任。1943年至1945年，任[[中共中央直属机关|中共中央直属机关]]党委书记兼[[中共中央管理局|中央管理局]]政治部主任。1948年11月至1949年6月，任[[沈阳军区|沈阳军区]]政治部主任。

中华人民共和国成立后，进入[[中华人民共和国外交部|中国外交部]]。1950年5月至1951年1月，任[[中华人民共和国外交部|外交部]]苏联东欧司副司长。1951年1月至1954年8月外交部苏联东欧司司长。9月起，担任首任[[中国驻阿尔巴尼亚大使列表|中华人民共和国驻阿尔巴尼亚大使]]。1958年，担任[[中国驻挪威大使列表|中华人民共和国驻挪威王国大使]]。1962年，担任[[中国驻叙利亚大使列表|中华人民共和国驻叙利亚大使]]。1966年，升任[[中华人民共和国外交部|中华人民共和国外交部]]副部长。1979年，担任[[中国驻巴基斯坦大使列表|中华人民共和国驻巴基斯坦大使]]。

== 家庭 ==

妻子：陆红。徐以新逝世后，一半骨灰安葬于出生地[[衢州市|衢州市]]的[[岭洋乡|岭洋乡]]岭头村<ref name="a1">{{Cite web |url=http://qjwb.zjol.com.cn/html/2011-05/31/content_870528.htm?div=-1 |title=墓碑上为何无名 村中为何埋着徐以新一半骨灰|accessdate=2015-04-21 |author= |date=2014-07-03|work= |publisher=浙江在线，来源：钱江晚报，2011年5月31日，A0006版|language=Zh-hans }}</ref>。1995年，立有徐以新纪念碑。2001年，陆红逝世后，一半骨灰[[合葬|合葬]]于此。<ref name="a2">{{Cite web |url=http://news.qz828.com/system/2014/07/03/010836031.shtml |title=【三衢名人堂】陆放：专注学术的“红二代”|accessdate=2015-04-21 |author=责任编辑：吾献红 |date=2014-07-03|work= |publisher=衢州新闻网，来源：衢州晚报|language=Zh-hans }}</ref>

次子：陆放，随母姓。
