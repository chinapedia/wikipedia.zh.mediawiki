'''徐德恩'''（{{bd|1987年|1月1日|catIdx=X}}），[[中国|中国]][[足球|足球]]运动员，司职中场。

== 俱乐部生涯 ==
徐德恩出道于[[广州医药|广州]]青年队，曾经代表香雪制药参加[[香港甲组足球联赛|香港甲组足球联赛]]。2006年7月，徐德恩在青年队和一线队的几场热身赛中表现出色，被主教练[[戚务生|戚务生]]提拔到一队<ref>[http://news.titan24.com/titan/51/07-12-25/12284.html “百米王”曾不会颠球 徐德恩六年级才接触足球] {{webarchive|url=https://web.archive.org/web/20110506000804/http://news.titan24.com/titan/51/07-12-25/12284.html |date=2011-05-06 }}</ref>。8月5日，他在广州医药客场0比0战平[[南昌八一|南昌八一]]的比赛中替补[[黄志毅|黄志毅]]出场，第一次参加职业联赛<ref>{{Cite web |url=http://www.ycwb.com/gb/content/2006-08/06/content_1179599.htm |title=中甲联赛：穗军又平弱旅 冲超前景逐渐黯淡  |access-date=2010-01-07 |archive-url=https://web.archive.org/web/20160307193246/http://www.ycwb.com/gb/content/2006-08/06/content_1179599.htm |archive-date=2016-03-07 |dead-url=yes }}</ref>。

2007年广州队主场对呼和浩特的比赛中，徐德恩首次在[[中甲联赛|中甲联赛]]中进球。

2008年广州队首次参加中超联赛，但他的出场次数寥寥，在[[2008年中国足球超级联赛|2008赛季]]只替补出场3次，而在[[2009年中国足球超级联赛|2009赛季]]只出场1次。[[2010年中国足球甲级联赛|2010赛季]]广州队因被揭发打假球而降入甲级，徐德恩在赛季初曾获得几次上场机会，但表现糟糕，之后就再也没有进入球队参加比赛的大名单。

2011年，徐德恩离开广州队后曾到同城兄弟[[广东日之泉|广东日之泉]]试训，不过试训失败。随后，徐德恩自由转会加盟中甲球队[[湖南湘涛|湖南湘涛]]，合约为三年。

== 国家队生涯 ==
虽然徐德恩在[[2006年中国足球甲级联赛|2006赛季]]代表广州队出场次数不多，但他的表现却受到国青队教练组的注意，并将他征调入[[中国国青队|中国国青队]]参加2006年亚青赛。他在10月30日中国队的小组赛第一场比赛中替补出场，并打进一球，帮助国青队1比0击败澳大利亚国青队。
