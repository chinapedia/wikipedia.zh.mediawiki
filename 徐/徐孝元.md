'''徐孝元'''<ref>{{cite web |url=http://cfile224.uf.daum.net/attach/24080734540E79FC2DCE86|title=仁川亞運韓國國家代表隊名單}}</ref>（'''Seo Hyowon'''，{{bd|1987年|5月10日||}}）是[[韓國|韓國]]女子[[乒乓球|乒乓球]]運動員<ref>{{cite web|url=http://www.ittf.com/ittf_stats/stats_per_player.asp?player_id=108470|title=Stats Records - ITTF Statistics|deadurl=yes|archiveurl=https://web.archive.org/web/20120812221425/http://www.ittf.com/ittf_stats/stats_per_player.asp?player_id=108470|archivedate=2012-08-12}}</ref><ref>{{cite web|url=http://www.ittf.com/ittf_ranking/world_ranking_per_name.asp?Siniors=1&Player_ID=108470|title=World ranking Record for SEO Hyowon (KOR)|deadurl=yes|archiveurl=https://web.archive.org/web/20140322225629/http://www.ittf.com/ittf_ranking/world_ranking_per_name.asp?Siniors=1&Player_ID=108470|archivedate=2014-03-22}}</ref><ref>{{cite web |url=http://www.ittfranking.com/cgi-bin/ttplayer.cgi?number=108470&lang=0&format=detailed|title=SEO Hyowon (KOR) - ITTF Rankings}}</ref>。她曾獲得[[國際桌總職業巡迴賽|國際桌總職業巡迴賽]]韓國公開賽、波蘭公開賽的女子單打冠軍，是繼[[金暻娥|金暻娥]]、[[朴美英|朴美英]]之後又一位達到世界頂級水準的韓國女子削球手，她亦被球迷譽為「韓國乒壇第一美女」。
