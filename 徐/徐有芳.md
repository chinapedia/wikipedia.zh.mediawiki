'''徐有芳'''（{{bd|1939年|12月|||catIdx=}}），[[安徽|安徽]][[广德|广德]]人，[[中国共产党|中国共产党]]及[[中华人民共和国|中华人民共和国]]官员，曾任[[中华人民共和国林业部|中华人民共和国林业部]]部长<ref name=rm>{{Cite web |url =http://news.sohu.com/84/51/news200655184.shtml  |title =人事任免：徐有芳当选为中共黑龙江省委书记  |author =  |date = 2002-04-29 |publisher =  |language =  |accessdate =  |quote =  }}</ref>。

==生平==
徐有芳是大学文化，1963年8月参加工作，1973年4月加入[[中国共产党|中国共产党]]，[[中国科学院|中国科学院]]名誉研究员。1963年毕业于[[安徽农学院|安徽农学院]]林学专业。1963年8月到1973年10月，任[[吉林省|吉林省]]八家子林业局营林处技术员、计划科计划员、政治部宣传科科员、党办秘书。1973年10月到1981年11月，任[[吉林省林业厅|吉林省林业厅]]计划处工程师、副处长。1981年11月到1983年4月，任吉林省露水河林业局常务副局长、党委常委。1983年4月到1985年1月，任吉林省林业厅副厅长、[[党组|党组]]成员，吉林省林业工业联合公司经理、分党组书记。1985年1月到1986年3月，任[[中华人民共和国林业部|林业部]]林业工业局局长。1986年3月到1993年3月，任林业部副部长、党组成员。1987年5月任[[大兴安岭火灾|大兴安岭森林火灾]]扑火指挥部副总指挥。1993年3月到1997年7月，任林业部部长、党组书记。1993年到1998年，任[[全国绿化委员会|全国绿化委员会]]副主任、办公室主任。1994年任中国防治沙漠化协调小组组长。1996年6月到1997年6月，兼任[[国务院环境保护委员会|国务院环境保护委员会]]副主任。1997年2月，任[[首都绿化委员会|首都绿化委员会]]副主任委员<ref name=rm/>。

1997年7月，被[[中共中央|中共中央]]任命为[[中共黑龙江省委|中共黑龙江省委]]书记。1998年4月，在中共黑龙江省第八届委员会第一次全体会议上当选为中共黑龙江省委书记。1999年2月黑龙江省九届人大二次会议上，补选为[[黑龙江省人大常委会|黑龙江省第九届人大常委会]]主任。2002年4月29日，中共黑龙江省第九届委员会第一次全体会议选举徐有芳为中共黑龙江省委书记<ref name=rm/>。2003年1月，当选黑龙江省第十届人大常委会主任。2003年4月，黑龙江省十届人大常委会二次会议上，徐有芳辞去主任职务，会议补选[[宋法棠|宋法棠]]为主任<ref>{{Cite web |url =http://www.npc.gov.cn/npc/zt/qt/dfrd30year/2011-03/23/content_1648569.htm  |title = 黑龙江省五至十一届人大常委会组成人员名单 |author =  |date = 2011-03-23 |publisher = 中国人大网 |language =  |accessdate =  |quote =  }}</ref>。

2003年4月至2007年7月，任[[中央农村工作领导小组|中央农村工作领导小组]]副组长（正部长级）。他是中共第十五、十六届中央委员。
