'''徐邦达'''（{{bd|1911年|7月7日|2012年|2月23日|catIdx=Xu徐邦达}}），字'''孚尹'''，号'''李庵'''、'''心远生'''，晚年号'''蠖叟'''。著名书画鉴定专家。曾任北京[[故宫博物院|故宫博物院]]研究员、中央文物鉴定委员会常务理事。有“徐半尺”（意指其鉴画，于画轴展开半尺已知真伪）和“华夏辨画第一人”之誉。<ref name ="mlx" />

== 生平 ==
徐邦达生于上海一个[[海宁|海宁]]富商家庭。父徐尧臣，绸缎商人，但喜[[收藏|收藏]]，故家中收藏赡富，徐邦达自幼即深受薰陶。少年时先后从[[娄东派|娄东派]]画家[[李醉石|李醉石]](名涛)习山水，著名画家[[赵时棢|赵时棢]]习鉴赏，又入书画大师[[吴湖帆|吴湖帆]]门下多年，并曾师从[[冯超然|冯超然]]、[[陈定山|陈定山]]等。

1937年夏，应[[上海博物馆|上海博物馆]]董事长[[叶恭绰|叶恭绰]]之邀协助“上海市文献展览”的古书画征选工作。此后又受叶之聘，撰写了《上海市文献展览古书画提要目录》一书。

1945年，[[抗战|抗战]]胜利后，当选[[中国美术家协会|中国美术家协会]]理事。次年，受聘[[上海美术馆|上海美术馆]]筹备处顾问。

1949年，中华人民共和国成立后，受聘[[上海市文物管理委员会|上海市文物管理委员会]][[顾问|顾问]]。

1950年，经时任国家文物局局长的[[郑振铎|郑振铎]]推荐，调任文物局文物处业务秘书。

1953年秋，调[[故宫博物院|故宫博物院]]参与绘画馆的筹建，从此未曾离开。

1978年，与[[启功|启功]]、[[谢稚柳|谢稚柳]]、[[刘九庵|刘九庵]]等组成全国书画巡回鉴定专家组，历时八年，著成多卷本《中国古代书画图目》。

2012年2月23日8时38分，在北京逝世，享年101岁。<ref>[http://culture.cntv.cn/20120224/114297.shtml 书画鉴定泰斗徐邦达辞世] {{Wayback|url=http://culture.cntv.cn/20120224/114297.shtml |date=20160304092011 }} 中国网络电视台</ref><ref>[http://www.dpm.org.cn/shtml/400/@/118499.html 故宫博物院成立徐邦达先生治丧工作组]{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }} 故宫博物院官方网站</ref>

==著名书画保护和鉴定案例==
===[[赵孟頫|赵孟頫]]《水村图》===
1953年以8000元购自东北通化市郊县大栗子沟，藏[[故宫博物院|故宫博物院]]。

===[[郑洞国|郑洞国]]皮箱===
1950年代，追回郑洞国所购溥仪取自故宫的多件珍宝。其中珍品包括五代杰出[[书法家|书法家]][[杨凝式|杨凝式]]的《夏热帖》，元代[[王蒙|王蒙]]的《太白山图》卷等。

===文革后的综合考察===

== 评价 ==
[[故宫博物院|故宫博物院]]的[[讣告|讣告]]评价他：“六十年来，徐邦达完成的600万字巨著《[[徐邦达集|徐邦达集]]》是其研究古书画的心血所成，铸就了影响海内外艺术史界的经典之作和宗师之论，他的逝世，是中国博物馆界的巨大损失，也是我国艺术界特别是书画鉴定界不可弥补的重大损失。”<ref name ="mlx">{{cite web |url=http://culture.ifeng.com/gundong/detail_2012_02/26/12786862_0.shtml?_from_ralated  |title=书画鉴定大家徐邦达去世 曾获业界雅号“徐半尺” |accessdate= 2012年|work=|publisher=凤凰网 |language = zh-hans}}</ref>

徐邦达的弟子[[杨臣彬|杨臣彬]]：“他的脑子就像是一部记录书画的活辞典、一架拍摄书画的录影机。在我的眼里，先生就是为中国书画而生的，先生无论是在书画鉴定方面，抑或在自己的创作上，都像古人说的那样：‘眼界尽古人神髓’”。<ref>{{cite web |url= http://culture.ifeng.com/whrd/detail_2012_03/01/12886759_0.shtml |title=“鉴定泰斗”徐邦达追悼会举行 故宫新老院长送别 |accessdate= 2012年|work=|publisher=凤凰网 |language = zh-hans}}</ref>
