'''徐诚之'''（{{bd|1921年||2017年|1月21日}}）原名'''徐崇信'''，[[山东省|山东省]][[招远县|招远县]]人，[[中国人民解放军铁道兵|中国人民解放军铁道兵]]党委常委、政治部主任<ref name=ifeng>{{Cite web |url =http://news.ifeng.com/a/20170205/50651067_0.shtml  |title =原铁道兵党委常委、政治部主任徐诚之逝世，享年96岁  |author =  |date = 2017-02-05 |publisher =凤凰网  |language =  |accessdate =  |quote =  }}</ref><ref name=gmw>{{Cite web |url = http://www.gmw.cn/02lrtd/content/2007-05/28/content_613855.htm |title =徐诚之：关于“战祸的记忆”  |author =  |date = 2007-05-28 |publisher =光明网  |language =  |accessdate =  |quote =  }}</ref>。

==生平==
1931年[[九·一八事变|九·一八事变]]发生后，徐诚之所在的学校里有多位老师宣传抗日救亡。1934年，[[冯玉祥|冯玉祥]]曾到该学校训话，勉励学生读书，不当亡国奴，并题词：“还我河山，一片精忠报国。驱尔异类，百年奇耻不共天。”<ref name=gmw/>

1937年[[抗日战争|抗日战争]]全面爆发。1938年10月，徐诚之参加[[八路军|八路军]]。1939年5月加入[[中国共产党|中国共产党]]。后在[[中国人民抗日军政大学|中国人民抗日军政大学]]一分校三支校任政治教员。[[抗日战争|抗日战争]]后期，1944年11月调任由[[刘公岛|刘公岛]]起义的[[汪精卫政权|汪精卫政权]]海军改编的[[胶东军区海军支队|胶东军区海军支队]]的政治处俱乐部主任，授少校<ref name=ifeng/><ref name=gmw/>。

1945年10月，胶东军区海军支队参与攻克[[即墨县|即墨县]]城后，奉命开往[[中国东北|中国东北]]，并改编为辽东人民自卫军第二纵队第二支队，徐诚之任第二支队政治部宣传股股长。第二支队从今[[吉林省|吉林省]]开进，先后攻克白旗屯、[[乌拉街|乌拉街]]、[[榆树县|榆树县]]、[[舒兰县|舒兰县]]、[[五常县|五常县]]，最终抵达[[牡丹江市|牡丹江市]]。其中第二支队在攻克五常县城后，徐诚之曾在1946年1月短暂出任五常县县长。此后，徐诚之参加了[[辽沈战役|辽沈战役]]、[[平津战役|平津战役]]<ref name=ifeng/>。

[[中华人民共和国|中华人民共和国]]成立后，徐诚之任铁道兵团宣传部副部长，1950年参加[[抗美援朝战争|抗美援朝战争]]，担任中朝联合前方铁道运输政治部宣传部部长。回国以后，徐诚之历任[[中国人民解放军铁道兵|中国人民解放军铁道兵]]宣传部部长、师政委，[[大兴安岭|大兴安岭]]会战指挥部政治部主任，铁道兵学校副政委，铁道兵政治部主任，铁道兵党委常委兼政治部主任<ref name=ifeng/>。

1960年徐诚之晋升大校军衔。徐诚之曾获[[中华人民共和国|中华人民共和国]]三级独立自由勋章、二级解放勋章，[[朝鲜民主主义人民共和国|朝鲜民主主义人民共和国]]二级共和国勋章、二级国旗勋章。1988年，获独立功勋荣誉章。徐诚之是[[第五届全国人民代表大会|第五届全国人民代表大会]]代表<ref name=ifeng/>。

2017年1月21日，徐诚之在北京病逝，享年96岁<ref name=ifeng/>。
