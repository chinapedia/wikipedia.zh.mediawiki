'''徐用儀'''（{{bd|1826年||1900年|8月11日|catIdx=X}}），字'''吉甫'''，別字'''筱雲'''，[[浙江|浙江]][[海鹽|海鹽]]人。[[清朝|清朝]]政治人物，「[[庚子被禍五大臣|庚子被禍五大臣]]」<ref>另外四位為[[袁昶|袁昶]]、[[許景澄|許景澄]]、[[聯元|聯元]]與[[立山_(清朝官员)|立山]]</ref>之一。

== 從政經歷 ==
[[File:总理衙门大臣合影.jpg|thumb]]、[[徐用仪|徐用仪]]、庆亲王[[奕劻|奕劻]]、[[许庚身|许庚身]]、[[廖寿恒|廖寿恒]]、[[张荫桓|张荫桓]]。]]
徐用儀出生於[[道光|道光]]六年（1826年），[[咸豐_(年號)|咸豐]]九年（1859年）[[順天|順天]][[鄉試|鄉試]]舉人。[[同治|同治]]初年，擔任[[軍機處|軍機]][[章京|章京]]，兼職於[[總理各國事務衙門|總理各國事務衙門]]。後升[[鴻臚寺|鴻臚寺]][[少卿|少卿]]。[[光緒|光緒]]三年（1877年），擔任[[太僕寺|太僕寺]][[少卿|少卿]]，並升職為[[大理寺|大理寺]][[卿|卿]]，跟過去一樣掌管[[軍機處|軍機處]]。後來升職為[[工部侍郎|工部侍郎]]，並擔任[[總理各國事務衙門|總理各國事務衙門]][[大臣|大臣]]，並歷任[[兵部侍郎|兵部侍郎]]與[[吏部侍郎|吏部侍郎]]等職，最後被任命為[[軍機大臣|軍機大臣]]。[[光緒|光緒]]二十年（1894年）正月，加封為[[太子少保|太子少保]]。

== 甲午戰爭 ==
[[光緒|光緒]]二十年（1894年），當時在[[甲午戰爭|甲午戰爭]]前夕，整個朝廷都在爭論要和還是要戰，主和派樞臣[[孫毓汶|孫毓汶]]被彈劾罷免之後，由[[翁同龢|翁同龢]]接任，[[翁同龢|翁同龢]]是主戰派，與主和派的徐用儀在討論政事時起了衝突，徐用儀被趕出樞廷，並解除所有官職。[[光緒|光緒]]二十四年（1898年）[[戊戌政變|戊戌政變]]後，[[慈禧太后|慈禧太后]]再度親政，徐用儀復職，並推薦[[太常寺|太常寺]][[卿|卿]][[袁昶|袁昶]]，剛好[[許景澄|許景澄]]回到京城，兩人都被命令與徐用儀一同任職於[[總理各國事務衙門|總理各國事務衙門]]。[[光緒|光緒]]二十五年（1899年），升任[[都察院|都察院]]左都[[御史|御史]]，並為[[兵部尚書|兵部尚書]]。

== 庚子事件 ==
[[File:徐用仪.jpg|210px]]》之徐用仪像]]
[[光緒|光緒]]二十六年（1900年），[[義和團運動|義和團運動]]興起。最早是[[光緒帝|光緒帝]]推動的[[百日維新|百日維新]]新政讓中外都贊同，而[[王儲|王儲]]卻一直沒有人選（[[光緒帝|光緒帝]]無子女）。端王[[載漪|載漪]]掌權之後想要廢掉[[光緒帝|光緒帝]]，可是害怕外國人反對，聽聞[[義和團|義和團]]「拳民」有神奇的力氣，又仇恨西教，想要以此來協助壓抑外國人，並召他們進入[[北京|北京]]，而[[義和團|義和團]]因此受到官方的保護而更加壯大。徐用儀提議希望嚴格禁止[[義和團|義和團]]的行動，但是不被接受，而在[[德意志帝國|德意志帝國]]的[[公使|公使]][[克林德|克林德]]被殺害後，徐用儀擔憂地說：「禍始此矣！」，並勸告慶親王[[奕劻|奕劻]]，厚葬[[克林德|克林德]]。

後來，各國兵艦到達[[津|津]][[沽|沽]]，朝廷召集廷臣一起討論是和或戰。徐用儀、[[許景澄|許景澄]]、[[袁昶|袁昶]]及[[尚書|尚書]][[立山_(清朝官员)|立山]]、[[內閣學士|內閣學士]][[聯元|聯元]]等五人一起上書說：「奸民不可縱，外釁不可啟。」認為要嚴逞肇事的義和拳民，並與[[八國聯軍|八國聯軍]]議和。但是[[載漪|載漪]]等人主戰甚力，朝廷大臣都接受了主張。徐用儀奉[[慈禧太后|慈禧太后]]的命令到使館討論緩兵的事宜，被主戰派認為是奸邪。

7月28日，[[許景澄|許景澄]]與[[袁昶|袁昶]]先被殺害，徐用儀自知难以幸免，但仍然意氣自如。8月11日，徐用儀被逮捕，並送到莊王[[載勛|載勛]]府邸。徐用儀並沒有辯駁，只說：「天降奇禍，死固分耳！」，與[[立山_(清朝官员)|立山]]、[[聯元|聯元]]一同於[[菜市口|菜市口]]被處死。過了三天，[[八國聯軍|聯軍]]就進入[[北京|北京]]，而[[慈禧太后|慈禧太后]]與[[光緒帝|光緒帝]]往西離京。

同年12月，[[光緒帝|光緒帝]]下詔為徐用儀等人平反，恢復舊有的官稱。[[宣統|宣統]]元年（1909年），追諡「'''忠愍'''」。[[浙江|浙江]]人為他設立祠堂於[[西湖|西湖]]，與[[許景澄|許景澄]]、[[袁昶|袁昶]]並稱「三忠」。
