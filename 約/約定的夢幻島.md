《'''約定的夢幻島'''》（{{lang-ja|約束のネバーランド}}）是日本《[[週刊少年Jump|週刊少年Jump》]]於2016年第35期開始連載的[[少年漫画|少年漫畫]]，原作為[[白井海芋|白井カイウ]]，作畫為[[出水ぽすか|出水ぽすか]]。故事開始的時間點在2045年，主角艾瑪、諾曼和雷原本快樂地生活在孤兒院中，直到發現孤兒院隱藏不為人知的祕密。

本作2017年榮獲[[小學館漫畫賞|小學館漫畫賞]]少年向部門賞<ref>[https://www.shogakukan.co.jp/mangasho/kako/ 小学館漫画賞：歴代受賞者]</ref>，2018年5月28日宣布动画化<ref>{{cite news |title=約束のネバーランド」TVアニメ化決定　2019年1月からフジテレビ“ノイタミナ”で放送 |url=https://news.biglobe.ne.jp/entertainment/0528/oks_180528_7456176965.html |accessdate=2019-01-14 |work=biglobe}}</ref>，於2019年1月播放[[電視動畫|電視動畫]],在第一期完結後官方釋出第二期消息並於2020年開播.<ref>{{cite news |title=「約束のネバーランド」19年1月TVアニメ化！ キャラボイスが聴けるPVも公開 |url=https://animeanime.jp/article/2018/05/28/37908.html |accessdate=2019-01-14 |work=animeanime}}</ref>。

單行本發行量截至第6卷累計達330萬冊，截至第8卷累計突破500萬冊。2018年11月25日，系列全球销量累计突破870万册。<ref>{{Cite news|url=https://www.animatetimes.com/news/details.php?id=1543115320|title=『約束のネバーランド』甲斐田裕子さん・藤田奈央さんら追加声優10名解禁！ 　ジャンプスペシャルアニメフェスタで第1話を世界最速上映|author=|date=|work=|accessdate=}}</ref>

== 故事簡介 ==
我們所約定的永恆之地啊，究竟存在何處?

美味的飯菜、潔白的制服、規律的作息、溫柔的媽媽……一切的一切，都像是一場夢一樣。

2045年，入秋時分，作為孤兒院「Grace Field House」最年長的三位十一歲孩童︰諾曼、雷與艾瑪，每天過著與孤兒院大夥遊戲的日子，誰也都沒有懷疑過圍繞著孤兒院之外的森林有著什麼，也從來沒有人知悉孤兒院大門以外的世界。有的只有孤兒院中的圖書室，靠著書中盛載的文字與插畫，外頭的風景就像在眼前一樣。

而這裡的生活也不是永遠的，只要滿了六歲，隨時都有可能幸運地被人領養，離開孤兒院，最晚也不會超過十二歲。

某個孤兒院小孩離開的晚上，諾曼和艾瑪發現她離去時，忘了帶走最心愛的娃娃，在他們追到了大門之後，才發現那些隱藏在他們生活之中詭異的細節，掩飾了他們怎麼也想不到的驚人事實……

=== 農場 ===
; {{nihongo|農場|農園}}
: 農場是指，專門飼養供鬼食用的人類的場所。已知農場目前分為三種，量產型農場、高級農場、特殊實驗農場。
:; 量產型農場
:: 走工業化模式，其生產模式是大規模集中機械式培育低品質的人類，培養方式僅止於將人養活養肥，時間到了就出貨。這類農場在鬼界有數百個（50話）。
:; 高級農場
:: 雖然少量生產，但以精緻栽培高品質人類為導向，此處的食用兒成長至一定年齡就會被「出貨」宰殺。各農場的出貨最大年齡有所不同，已知GF為十二歲，GB為十五歲。目前高級農場只有四處，分別為GF（Grace Field）、GB（Glory Bell）、GV（Grand Valley）、GR（Goodwill Ridge）四個農場。每座農場底下配置五個「孤兒院」（農場），如艾瑪他們所成長的「孤兒院」就稱為「第三農場」。
:; 特殊實驗農場
:: 目前只有一處，就是「Λ7214 」，烙印位置在左胸（亞當、諾曼）。在74話時，畫面是鏡子中的諾曼，故位置看似相反。

; 秘密房間
: GF農場內有養育者「媽媽」伊薩貝拉專門與鬼總部通訊的秘密房間，它隱藏在盥洗室及伊薩貝拉的書房間，是個大多數孩子都不知道的小房間。
: 一開始只有從幼時就明白真相的雷知道，真相曝光後，一心想確認的唐、吉爾妲曾闖進去。進入方式是從伊薩貝拉書房中的書櫃打開暗門，前方亦有一道上鎖的門，可以用GF的萬用鑰匙打開。房間裡有伊薩貝拉用來聯絡總部的通訊機、歷年的成績紀錄本，以及歷屆已出貨孩子當時帶走的的玩具和私人用品（均為伊薩貝拉在孩子出貨後私自保留），其中也包括柯妮的兔子玩偶LB。就此之後，唐和吉爾妲開始相信艾瑪他們所說的真相。

==== 養育者 ====
; {{nihongo|飼育監／媽媽|飼育監／ママ}}

: 養育者及助手，幼時也都曾是所屬高級農場的食用兒，她們在達到一定條件（例如滿分、頂級）及養育者推薦下，在出貨當日可以選擇接受成為養育者或助手的訓練，或是乾脆出貨被吃掉。

==== 食用兒 ====
; {{nihongo|食用兒|食用児}}
: 食用兒是指，一群專供鬼食用的畜牧人類，他們一出生就在農場生活，也在農場出貨被宰殺。食用兒最好吃的部位是大腦，智商的優劣能決定食用品質和美味等級。絕大多數的食用兒終其一生像家畜般生活及被宰殺，但只有極少部分的女性食用兒，能被原本管理農場的「媽媽」推薦為農場管理人員。依據不同農場間的性質，食用兒的狀態各有不同。

; 量產型農場食用兒
: 毫無智力可言，他們沒有意識、沒有語言能力、無法跟人對話，也沒有自己的名字。他們的培養方式完全像被以牲畜的方式對待，因此他們沒有獨立思考能力和行動自由，智商非常低，連虛假的幸福感都沒有。這類農場的食用兒在身體固定的部位上，會被烙上該農場的圖案（50話與56話）。

; 高級農場食用兒
: 雖擁有高等智商，但終其一生會以為自己在「孤兒院」長大，並以「出養」的名義進入「出貨」程序。大多數此種農場的食用兒，只有在出貨被宰殺的那天，才會知道世界的真相。
:; 培育
:: 高級農場食用兒從小在「愛」的環境中被培育長大，有完善的飲食、穩定的作息、開闊的活動空間、同儕們彼此的刺激、友情、戀愛等充分的情感刺激，以及鬼刻意設置的日復一日的智力測驗，使得高級農場培植出的食用兒口感較佳，其中又以GF農場第三養育場的伊薩貝拉媽媽所培育出來的食用兒品質最佳。
:; 等級
:: 高級農場生產的食用兒，依據品質還可細分為下級、中級、優級、高級、頂級等五個等級。頂級的食用兒（如艾瑪、諾曼、雷），稀有到只有鬼帝王能享用，連鬼貴族拜雍都不能染指。
:; 識別編號（認識番号）
:: 因高級農場食用兒在數量上相當稀少，每一個食用兒身上都有各自的識別編號，並依據所屬農場的不同，編號烙印的位置、格式、排列與字體也不一樣。

; 特殊實驗農場食用兒
: 目前漫畫劇情透漏不多。該農場出身及被轉送至此的食用兒（如亞當與諾曼）身上有被烙上Λ7214農場所屬的圖案（諾曼脖子上的GF編號22194仍然留存）。已知亞當具有怪力，能搬運房屋作為武器，連列烏維斯都能被拋飛，被列烏維斯攻擊也沒有受到太大的損傷。另外，諾曼被GF農場出貨後，被他的「新爸爸」皮塔·拉托里轉調到「Λ7214 」，進行各種智力測驗。

; 食用兒編號
: 四個高級農場的識別編號有所不同，同一所屬農場的食用兒也都有各自的編號,因此若有食用兒逃出農場，光憑編號就能知道食用兒的身分和名字，已知會被烙上識別編號的，只有四個高級農場出身的食用兒：
:* GF農場：編號位於脖子左側，是五位阿拉伯數字，如艾瑪的編號是63194。
:* GB農場：編號在腹部，共有四個英文大寫字母和兩個阿拉伯數字，如盧卡斯的編號為KGX2A7。
:* GV農場：編號在胸口上方，組成是（1）一個英文大寫字母（2）一個羅馬數字（3）中間被一道橫槓分開的六個數字。例如：瓦歐蕾特的編號為DIV332-198。
:* GR農場：目前識別編號不明。

=== 生物·植物 ===
; {{nihongo|鬼|鬼（おに）}}
: 之間有分野生無法跟人類對話的「下等種」，以及有語言能力、會穿衣服、有社會文化階級的「上等種」。上等種的鬼，有鬼帝王、鬼貴族，以及鬼士兵等不同階級，在高級農場所出產的特上等食用兒（如艾瑪、諾曼、雷），只有鬼帝王能享用，連拜雍之類的鬼貴族都吃不到。
: 目前食用兒的栽培，是由具有智力的上等種鬼所壟斷。食用兒農場可能是由鬼貴族所獨佔或投資，他們會把食用兒作為祭典上的饗宴來享用。除了吃人肉，鬼之間也會彼此自相殘殺、吞食。鬼在生態上，似乎是隨森林和水源移動，在生理上自然排斥荒漠。
: 艾瑪等人在前往黃金池塘的路上，被野生鬼追殺，他們看到被陷阱困住的野生鬼彼此互相吞食，作為逃離陷阱的手段。這種吞食的生物模式在上等鬼也不例外，例如在黃金池塘戰役裡，諾爾瑪被槍殺後，諾斯把她吃掉，似乎是吸收她的能力或意識，變得更強大。
: 鬼的姿態外觀會演變，繆西卡有次跟艾瑪聊天，聊到鬼與鬼之間外觀差異的話題時，她用鬼語自答：「這一千年來我們的姿態演變得相當不規律呢。」
: 諾曼向艾瑪等人透露鬼最初的姿態是像細菌般的存在，後來鬼的基因發生突變形成「基因水平繁殖」，當它們進食不同物種就會繼承該物種的基因和特點而獲得不同的姿態，因此才有各種各樣的鬼。直到鬼首次進食人類後它們獲得人類的語言和智慧並呈現出人類的姿態。但由於鬼的基因進度進化，它們必需保持進食才能維持遺傳的特徵，例如透過進食人類而獲得智慧的鬼六個月沒有進食人類的話就會失去人形和智慧退化成為沒有智慧的野生鬼。因此，五大貴族的鬼為了維持智慧才會建立農場培育食用兒童。

; {{nihongo|韦达|ヴィダ}}
: 是鬼的世界中一种随处可见的吸血植物。被鬼用在屠杀猎物的仪式上，科尼與克洛涅就是被鬼用韦达杀害。在猎物未死亡时用韦达刺向猎物的心脏。意为向神献上粮食。要是神接受的话，韦达上的花就会开放。摘下韦达后即可食用猎物。
: 此仪式兼备放血的作用，这样的话猎物的肉就能长久保存（猎物死亡后再刺的话就无法吸光猎物体内的血液）。<ref>漫画第49话</ref>

; 吸血樹
: GF農場外有吸血樹群，他們會讓獵物掉入陷阱中，把獵物引誘到根部，再進行捕捉，吸乾獵物的血液。
: 艾瑪等人脫離GF農場後的第一個挑戰，就是一群人掉入吸血樹的陷阱中。在危急中，艾瑪聯想到吸血樹就是威廉·密涅瓦撰寫的冒險故事「烏戈冒險記」第三章中的「阿爾巴比涅拉之蛇」，故事關於洞穴、寶石、蛇的記載，宛如吸血樹內部的生態（40話）。
: 吸血樹探知獵物的方式是靠觸覺，這種植物沒有嗅覺、視覺、聽覺。一旦獵物碰觸到根的前端，吸血樹就會展開攻擊。逃離吸血樹的作法，就是在地底的天井開個孔，讓外部冰冷的空氣灌入洞中，吸血樹就會失去活力。

: 伊薩貝拉隨身攜帶的懷錶，外觀只是一般的懷錶，事實上具有追蹤食用兒動向的追蹤器。但懷錶無法具體顯示特定食用兒的名字，它的螢幕只會在食用兒所處位置上顯示小圓點。

; 萬用鑰匙
: 一支可以打開GF所有房間的萬用鑰匙（包括秘密房間），伊薩貝拉只持有一支，平時放在她制服的右側口袋。唐曾偷偷拿走萬用鑰匙，放在地上，再由3歲的艾根撿到，轉交給伊薩貝拉。

; {{nihongo|測驗|テスト}}
: 鬼為了提升高級農場食用兒的大腦品質，每天對農場內食用兒進行智力測驗。為隱藏真相，養育者伊薩貝拉對孩子們灌輸，這是為了孩子們的未來與他們自身著想，而做的具有替代學校功能的測驗。
: GF農場的測驗難度較高，滿分為300分。每位食用兒每日的成績會被紀錄，按照年齡大小排列，並使用識別編號來紀錄（而非食用兒的名字）。食用兒在同齡人中成績越低，則越早被出貨，成績紀錄本以年份作為區分（例如2039年一本、2040年一本，以此類推）。GF農場的成績記錄本存放在伊薩貝拉所使用的秘密房間，已經被出貨的孩子的那一列會被劃掉，總部的鬼也有食用兒成績的資料。

; {{nihongo|鋼筆|ペン}}
: 威廉·密涅瓦（詹姆斯·拉托里）特製的鋼筆是推動劇情的重要道具，在筆尾有「W.M.」的標籤。這隻鋼筆在外觀上跟一般鋼筆沒什麼不同，但要寫的時候沒墨水。當使用者發現沒墨水時，為了換墨水，會打開鋼筆，此時會發現筆管貼著「B-06-32」的標籤。當筆完全拉開，會開啟鋼筆的投影功能，在空間中投射出可觸控的投影畫面。鋼筆的功能相當多元，除了能顯示所處位置，他還記載許多資料，惟為避免鋼筆機密資料被鬼道竊，詹姆斯·拉托里在設計上刻意從食用兒的角度出發，若要搜索鋼筆更機密的情報，必須配合威廉·密涅瓦的書籍內容（沒有摩斯密碼那本），依據鋼筆指示，找到書中關鍵字輸入鋼筆，才能看到更多的指示。
: 鋼筆的筆帽可以替換。艾瑪依據詹姆斯·拉托里的指示，在黃金池塘的地底電梯前的電話抽屜的夾板後方，找到替換鋼筆的筆帽（記憶芯片）。新的筆帽的資料，包括來往兩個世界道路的情報、支持者的聯絡方式、GP（黃金池塘）的設計圖、七面牆壁的情報，以及「Λ7214 」食用兒實驗農場的情報。
: 關於鋼筆的所有權，中間轉了好幾手。起初，克洛涅修女在能跟鬼平起平坐的人類身邊，撿到那男人掉落的鋼筆。後來修女被轉調到GF農場擔任伊薩貝拉媽媽的助手，她在離開GF農場（被滅口）前，於諾曼的抽屜遺留這隻鋼筆。諾曼在被出貨前，又將此鋼筆轉交給艾瑪。

; {{nihongo|烏戈冒險記|ウーゴ冒険記}}
: 由威廉·密涅瓦所撰寫的冒險故事，敘述主人公「烏戈」與浣熊夥伴「馬文」在世界密境旅行的冒險故事<ref>漫畫第40話</ref>。這本書有威廉·密涅瓦（即詹姆斯·拉托里）蓋上的[[摩尔斯电码|摩斯密碼]]「約定」。
: 在艾瑪等人逃出GF農場，被困在吸血樹底時，艾瑪想起這本書其實有提供脫困的線索，才意識到這是一本教導食用兒如何在野生鬼族領地求生的故事書。<ref>漫畫第40話、第41話</ref>

=== 其他 ===
; {{nihongo|約定|約束（やくそく）}}
: 指一千年以前，為了保護人族跟鬼族不彼此獵殺，兩者「約定」互不侵犯彼此的世界。而鬼族為了食用人肉，人族在鬼的世界遺留一些人類，供鬼拿去繁殖，作為專供食用的人肉。
