'''約翰·斯特拉特，第三代瑞利男爵'''，[[功績勳章_(英國)|OM]]，[[皇家學會院士|FRS]]（{{lang-en|'''John Strutt, 3rd Baron Rayleigh'''}}，{{bd|1842年|11月12日|1919年|6月30日|catIdx=R}}），[[英國|英國]][[物理學家|物理學家]]。他与[[威廉·拉姆齐|威廉·拉姆齐]]合作发现[[氩|氩]][[元素|元素]]，并因此获得1904年[[诺贝尔物理学奖|诺贝尔物理学奖]]。<ref name="nobel_1904">{{cite web |url=http://nobelprize.org/nobel_prizes/physics/laureates/1904/index.html |title=The Nobel Prize in Physics 1904 |publisher=[[诺贝尔基金会|诺贝尔基金会]] |date= |accessdate=2008年12月18日 |language=en }}</ref>他还发现了[[瑞利散射|瑞利散射]]，预测了[[面波|面波]]的存在。

== 生平 ==
瑞利1842年出生于英国[[埃塞克斯郡|埃塞克斯郡]]的[[莫尔登|莫尔登]]，年幼时身体虚弱。他毕业于[[哈罗公学|哈罗公学]]；1861年进入[[剑桥大学|剑桥大学]][[三一学院_(剑桥)|三一学院]]学习[[数学|数学]]，先后于1865年和1868年获得[[学士|学士]]和[[硕士|硕士]]学位。<ref name="nobel_bio">{{cite web |url=http://nobelprize.org/nobel_prizes/physics/laureates/1904/strutt-bio.html |title=Lord Rayleigh - Biography |publisher=[[诺贝尔基金会|诺贝尔基金会]] |date= |accessdate=2008年12月18日 |language=en }}</ref>

瑞利在1871年与{{link-en|詹姆斯·梅特兰·鲍尔弗|James Maitland Balfour}}的女儿伊夫琳·鲍尔弗（Evelyn Balfour）结婚，两人育有三个儿子。其中长子后来成为[[帝国理工学院|帝国理工学院]]物理教授。1873年，他的父亲[[約翰·詹姆斯·斯特拉特，第二代瑞利男爵|約翰·詹姆斯·斯特拉特，第二代瑞利男爵]]去世，他作为继承人成为第三代瑞利男爵。<ref name="nobel_bio"/>

瑞利1873年当选英国[[皇家学会|皇家学会]]院士。1879年被剑桥大学任命，接替[[詹姆斯·克拉克·麦克斯韦|詹姆斯·克拉克·麦克斯韦]]担任实验物理教授及[[卡文迪许实验室|卡文迪许实验室]]主任。1884年，瑞利离开剑桥，到自己在埃塞克斯郡的别墅继续实验研究。1887年至1905年，他在英国[[皇家研究所|皇家研究所]]担任自然哲学教授。1905年至1908年担任皇家学会会长。<ref name="nobel_bio"/>1908年直到逝世任剑桥大学校长。<ref name="ncu">{{cite web|url=http://wlsyzx.ncu.edu.cn/source/special/web/rl.htm|title=国外著名物理学家：瑞利|publisher=[[南昌大学|南昌大学]]基础物理实验中心|date=|accessdate=2008年12月18日|language=}}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

1919年6月30日，瑞利在埃塞克斯郡[[威特姆|威特姆]]去世。<ref name="nobel_bio"/>

== 研究 ==
瑞利于1883年最早描述了[[海鸟|海鸟]]的{{link-en|动力翱翔|Dynamic soaring}}，发表于英国的《[[自然_(期刊)|自然]]》杂志。1896年，瑞利提出“[[双耳效应|双耳效应]]”理论，解释了人为什么能够分辨声音的方向。<ref name="hqu">{{cite web|url=http://zwx.hqu.edu.cn/%BD%CC%D1%A7%D7%CA%D4%B4.files/gdjs/02.%C9%F9%D2%F4.doc|title=广播电视技术基础：第二章 声音|author=洪强，黄志浩|publisher=[[华侨大学|华侨大学]]中文系广电教研室|date=|accessdate=2008年12月18日|format=DOC|language=}}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>1877年至1878年间，他出版了两卷《声学理论》。他还参与《[[大英百科全书|大英百科全书]]》的编写。<ref name="nobel_bio"/>

瑞利长期致力于气体[[密度|密度]]的研究，他在研究中发现从液态空气中分馏出来的氮气，跟从[[亚硝酸铵|亚硝酸铵]]分解得到的氮气，密度存在超过实验误差范围的差异。<ref name="physica">{{cite web |url=http://www.physica.cn/about/004.htm |title=瑞利 |publisher=physica.cn |date= |accessdate=2008年12月18日 |language= |archive-url=https://web.archive.org/web/20110221194856/http://www.physica.cn/about/004.htm |archive-date=2011年2月21日 |dead-url=yes }}</ref>后来他遇到[[威廉·拉姆齐|威廉·拉姆齐]]，两人决定合作查明这一问题的原因。1894年8月13日，瑞利与拉姆齐宣布他们发现一种新的气体元素[[氩|氩]]。<ref>{{En icon}}{{cite journal en |title=Argon, a New Constituent of the Atmosphere |author=Lord Rayleigh; William Ramsay |journal=Proceedings of the Royal Society of London |volume=57 |issue=1 |pages=265–287 |year=1894–1895 |url=http://links.jstor.org/sici?sici=0370-1662%281894%2F1895%2957%3C265%3AAANCOT%3E2.0.CO%3B2-X |doi=10.1098/rspl.1894.0149 }}</ref>之后，在瑞利的协助下，拉姆齐又相继发现了几种新的[[稀有氣體|惰性气体]]元素。<ref name="mpn">{{cite web|url=http://www.xjbzedu.gov.cn/bzjys/fyja/kxj/for/006.htm|title=诺贝尔物理学奖百年回顾|author=厉光烈，李龙|publisher=《现代物理知识》|date=|accessdate=2008年12月18日|language=}}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

瑞利提出的分子散射公式解释了“天空为什么是蓝的”，被称为[[瑞利散射|瑞利散射]]定律。他还提出了[[瑞利准则|瑞利准则]]。

== 荣誉与奖项 ==
瑞利先后获得过[[皇家奖章|皇家奖章]]（1882年）<ref>{{cite web |url=http://royalsociety.org/page.asp?id=1753 |title=Royal archive winners Prior to 1900 |publisher=[[皇家学会|皇家学会]] |date= |accessdate=2008年12月18日 |language=en }}</ref>、[[马泰乌奇奖章|马泰乌奇奖章]]（1894年）<ref>{{cite web |url=http://www.accademiaxl.it/finalita_prscientifici_med_matteucci_eng.php |title=Matteucci Medal |publisher=意大利科学院 |date= |accessdate=2008年12月18日 |language=en |deadurl=yes |archiveurl=https://web.archive.org/web/20080401080739/http://www.accademiaxl.it/finalita_prscientifici_med_matteucci_eng.php |archivedate=2008年4月1日 }}</ref>和[[科普利奖章|科普利奖章]]（1899年）<ref>{{cite web |url=http://royalsociety.org/page.asp?id=1743 |title=Copley archive winners 1899 - 1800 |publisher=[[皇家学会|皇家学会]] |date= |accessdate=2008年12月18日 |language=en }}</ref>。1904年，因“研究气体密度，并从中发现氩”，瑞利被授予[[诺贝尔物理学奖|诺贝尔物理学奖]]。<ref name="nobel_1904"/>1920年，他被追授[[拉姆福德奖章|拉姆福德奖章]]。<ref>{{cite web |url=http://royalsociety.org/page.asp?id=1747 |title=Rumford archive winners 1988 - 1900 |publisher=[[皇家学会|皇家学会]] |date= |accessdate=2008年12月18日 |language=en }}</ref>

此外，[[火星|火星]]和[[月球|月球]]上有[[环形山|环形山]]以瑞利的名字命名。{{link-en|小行星22740|22740 Rayleigh}}也被命名为“瑞利星”。<ref name="nasa">{{cite web |url=http://ssd.jpl.nasa.gov/sbdb.cgi?sstr=22740 |title=22740 Rayleigh (1998 SX146) |publisher=[[NASA|NASA]] |date= |accessdate=2008年12月18日 |language=en }}</ref>
