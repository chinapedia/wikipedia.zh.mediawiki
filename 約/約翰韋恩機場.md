'''約翰韋恩橘郡機場''' （{{Airport codes|SNA<ref>{{cite web | url = http://www.iata.org/publications/Pages/code-search.aspx | title = IATA Airport Code Search (SNA: Santa Ana / John Wayne) | publisher = [[International_Air_Transport_Association|International Air Transport Association]] | accessdate = December 27, 2012}}</ref>|KSNA|SNA}}） 是一個位於美國[[加利福尼亚州|加利福尼加州]][[橙縣|橘郡]]的機場。機場位於[[歐文_(加利福尼亞州)|爾灣]]，[[科斯塔梅薩|科斯塔梅薩]]及[[纽波特比奇_(加利福尼亚州)|纽波特比奇]]三市交叉口。機場IATA代碼SNA表示的城市聖安娜位於機場北側，並未與該機場接壤。約翰韋恩機場原名為[[橘郡機場|橘郡機場]]，後於1979年改名為約翰韋恩機場以紀念原居住在紐波特比奇並於該年去世的演員[[約翰·韋恩|John Wayne]]。

美國國家機場系統整合計畫（NPIAS, The National Plan of Integrated Airport Systems）於2011-2015年將約翰韋恩機場歸類為主要用於民航運輸機場。<ref>{{Cite web|url = http://www.faa.gov/airports/planning_capacity/npias/reports/media/2011/npias_2011_appA.pdf|title = 2011–2015 NPIAS Report, Appendix A|format = PDF, 2.03 MB|work = National Plan of Integrated Airport Systems|publisher = Federal Aviation Administration|date = October 4, 2010}}</ref> 根據美国联邦航空管理局（FAA）紀錄，2014年共有4,584,147名旅客登機，較於2013年的4,450,628名旅客增長。<ref>{{Cite web|url = http://www.faa.gov/airports/planning_capacity/passenger_allcargo_stats/passenger/media/cy14-commercial-service-enplanements.pdf|title = Enplanements for CY 2011|format = PDF, 1.7 MB|work = CY 2014 Passenger Boardings at Commercial Service Airports|publisher = Federal Aviation Administration|date = September 22, 2015}}</ref> 約翰韋恩機場鄰近的通用航空機場包括富樂頓市機場，民用機場包括[[长滩机场|長灘機場]]，[[洛杉磯國際機場|洛杉磯國際機場]]以及[[安大略国际机场|安大略國際機場]]。 2014年，約翰韋恩機場成為[[大洛杉矶地区|大洛杉磯地區]]以使用人數計算第二繁忙的機場。<ref name="ACI-NA">{{Cite web|url = http://www.aci-na.org/content/airport-traffic-reports|title = 2014 North American (ACI-NA) Top 50 airports (includes Passenger, Cargo and Movements) Retrieved December 18, 2015|format = XLS, 147KB|publisher = Airport Council International}}</ref>

約翰韋恩機場的主要運營公司包括[[西南航空|西南航空]]，[[聯合航空|美國聯合航空]]，[[美國航空|美國航空]]，[[阿拉斯加航空|阿拉斯加航空]]及[[達美航空|達美航空]]等。 <ref>{{Cite web|url = http://www.rita.dot.gov/bts/data_and_statistics/by_mode/airline_and_airports/airline_passengers.html|title = DOT BTS Passenger Data from SNA|work = 1/15 to 9/15 Date search of SNA by each Airline (WN: 1,569,780, AA(+US) 528,435, UA 483,764, AS 384,842, DL, 283,186|publisher = DOT BTS|date = December 18, 2015}}</ref>

機場的主要跑道長約1,738米，短於其他美國同級別主要機場。由於跑道長度限制，在約翰韋恩機場可以起降的最大客機為[[波音757|波音757]]。較大型的貨機亦可在約翰韋恩機場運營，例如[[聯邦快遞|聯邦快遞]]的[[空中客车A300|空巴300]]/[[空中客车A310|310]]型貨機。機場亦有少數登機門可以停靠[[波音767|波音767]]及同級別的航機，但這些航機需要減載或限制燃油量運營。目前仍未有[[廣體飛機|廣體客機]]計畫於約翰韋恩機場運營。

約翰韋恩機場距離位於[[阿納海姆_(加利福尼亞州)|安納海姆]]的[[迪士尼樂園|迪士尼樂園]]約23公里。

== 歷史 ==
[[File:Orange_County_Airport,_1950s.jpg|thumb]]
[[File:Orange_County_Airport_terminal,_circa_1967.jpg|thumb]]1923年，为了在尔湾公司拥有的土地上经营一所飞行学校，埃迪·马丁与詹姆斯欧文签订了5年的租赁合同，建立了第一个简易的机场。  1939年橙县政府通过土地互换买下了机场，直到现在该机场依然归橙县政府管理与所有。

1926年，马丁增加了第一个机库。 1935年在埃迪·马丁机场，[[霍华德·休斯|霍华德·休斯]]上演他的创造世界速度纪录的飞行。

随着圣安娜陆军航空基地于1942年启用，紧邻的马丁机场被暂时关闭。

作为二战期间的一个军事基地后，圣安娜陆军机场被联邦政府归还到橙县政府，然而机场仍然保持开放给所有类型的飞行使用。

除了继续服务于航空，该领域成为一个重要的飙车中心。1950年到1959年，CJ“帕皮”哈特和克雷顿亨特举办圣安娜短程高速赛车赛 ，他们被认为是世界上第一个商业短程高速赛车比赛。 每个星期天，为了举办赛车，机场跑道关闭，不进行飞机起降。

最初的单一跑道很长，有两个210度（21号）和30度（3号）的跑道。1964年机场被重建，两条平行跑道面向190度和10度的配置维持到现在。重建后，较长的跑道19R（现为20R）只比旧的21号跑道要长，但长度足以符合规定容纳喷气式客机正常起降。一个完整的仪表着陆系统（ILS）也被安装到机场，服务于商业运营。

20世纪50年代，机场只有Bonanza航空运营的[[洛杉磯國際機場|洛杉矶]]经停[[聖地牙哥國際機場|圣迭戈]]飞往[[菲尼克斯天港国际机场|菲尼克斯]]之间的几个航班。1963年，Bonanza开通了到菲尼克斯的直飞航班，机型为F27飞机，1965年又开通了前往[[拉斯维加斯|拉斯维加斯]]的航班。1967年，加州航空开启了直飞旧金山的航线，一周48班，[[洛克希德·马丁|洛克希德马丁公司]]的伊莱克特拉型飞机执飞。同年晚些时候，bonanza航空使用[[道格拉斯DC-9|DC-9]]运营了该机场第一班喷气式飞机航班。

1967年， 埃迪·马丁航站楼被修建，可以容纳每年40万人次的乘客。机场于1974年增加了两个乘客等待区，1980年和1982年又新建了行李领取区和候机楼衔接区。

1976-77年，机场开通了飞往[[盐湖城国际机场|盐湖城]]的航班（[[Hughes_Airwest|Hughes Airwest]] ，[[道格拉斯DC-9|DC9]]系列），随后开通了一系列直飞航线：1982年开通直飞[[丹佛國際機場|丹佛]]（Frontier，[[麥道MD-80|MD80]]系列），1983年[[达拉斯-沃思堡国际机场|达拉斯-沃思堡]]（[[美國航空|美国航空]]， MD80系列），1986年[[奥黑尔国际机场|芝加哥]]（加州航空 [[波音737-300型|波音737-300]]系列）以及1991年开通纽约肯尼迪的航线（ 美国西部航空，[[波音757|波音757]]）。

橘郡机场于1979年改名为约翰·韦恩机场后，[[約翰·韋恩|约翰·韦恩]]协会委托雕塑家罗伯特·萨默斯在德克萨斯州都柏林Hoka Hey Foundry打造铜像“the duke”，并在1982年11月4日将铜像捐献给橘郡。如今，铜像坐落在托马斯F·莱利航站楼的到达层。

1990年，托马斯F·莱利航站楼开放，陈旧的埃迪·马丁航站楼被更现代化的新航站楼替换。新设施包括14个装卸桥，4个行李转盘，更开阔的空间和独特的靠近道路的抵港及出发层。在1994年，被弃用的埃迪·马丁航站楼被拆除。

在90年代末和21世纪初，托洛海军陆战队航空站的关闭引发了关于建设一个新的，更大的机场的提案 。然而，经过一系列的政治斗争，与来自埃尔托罗附近居民的强烈反对，该提案被驳回，新机场并未能够建成。

2011年，作为一个5.43亿美元的扩建项目的一部分，机场增加了更多的航站楼空间和修缮了现有航站楼。 计划还在c航站楼新建了六个登机口，通勤区域，以及在a航站楼建设了新的通勤设施。

机场先前有[[阿罗哈航空|阿罗哈航空]]（2001-2008），[[維珍美國航空|维珍美国]] （2009- 2010年）和[[加拿大航空|加拿大航空]]（2010）进驻。[[File:Orange_County_Airport_terminal,_1971.jpg|thumb]]

== 航廈 ==
主要客运航站楼，托马斯F.莱利航站楼，以在80年代游说扩建机场的已故县主管的名字命名。托马斯F.莱利航站楼被划分成三个候机区域，A，B,C。
所有三个航站楼，A，B和C，都是同一个托马斯F·莱利楼，安检后的乘客可以在所有的候机区之间移动。安检通道在所有三个航站楼都存在。所有安检区域也有为头等舱和精英常旅客“准备的快速安检通道。进入安检之前，室内转换航站楼亦可，值机区域在所有三个航站楼之间区域​​的也有相互连接的通道。在所有三个航站楼都有免费的Wi-Fi。
