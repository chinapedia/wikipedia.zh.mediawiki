'''約翰·克萊頓·梅爾'''（{{lang-en|John Clayton Mayer}}，{{bd|1977年|10月16日|catIdx=M}}），[[美國|美國]][[創作歌手|創作歌手]]。梅爾生於[[康涅狄格州|康涅狄格州]]，在1997年搬家至[[喬治亞州|喬治亞州]][[亞特蘭大|亞特蘭大]]之前，於[[伯克利音樂學院|伯克利音樂學院]]進修音樂、並獲得了一些支持者。他的首二張專輯《[[發聲中心|發聲中心]]》（''Room for Squares''）及《[[甜蜜負荷|甜蜜負荷]]》（''Heavier Things''）表現出色，為梅爾贏得了多張[[白金認證|白金認證]]。4次獲得[[葛萊美獎|葛萊美獎]]最佳流行男歌手。

梅爾初期的曲風以[[搖滾|搖滾]]及[[流行|流行]]為主。直到2005年與著名的[[藍調歌手|藍調歌手]]如[[B·B·金|比·比·金]]及[[埃里克·克莱普顿|艾瑞克·克萊普頓]]合作，而逐漸轉型為藍調歌手，並組成約翰梅爾三重奏（''John Mayer Trio''）推廣[[藍調|藍調]]。藍調的曲風可以從2006年推出的專輯《[[聲聲不息|聲聲不息]]》（''Continuum''）中明顯感覺到。2007年，梅爾分別以專輯《聲聲不息》及歌曲〈''Waiting on the World to Change''〉贏得第四十九屆葛萊美獎最佳流行專輯及最佳流行男歌手。梅爾的事業後來擴展至[[棟篤笑|棟篤笑]]、設計及寫作；他並曾為雜誌寫了一些文章，大部份刊登於[[時尚先生雜誌|時尚先生雜誌]]（''Esquire''）。

== 早期生活 ==

約翰·梅爾出生於[[康涅狄格州|康涅狄格州]][[布里奇波特_(康涅狄格州)|布里奇波特]]，母親瑪格莉特是英文老師，父親理查是高中校長，梅爾在家裡的三個兒子中排行第二<small><ref name=ELLE>Ruth Shaut (''[[ELLE|ELLE]]'')(2006).[http://jmeyecandy.org/thumbnails.php?album=480 "Blues Brother"] {{webarchive|url=https://web.archive.org/web/20081024084257/http://jmeyecandy.org/thumbnails.php?album=480 |date=2008-10-24 }} J-mayer.org. 2006-08-03查閱.</ref></small>。他在[[費爾菲爾德_(康乃狄克州)|費爾菲爾德]]附近長大，小時候還認識了現在的名[[網球|網球]]選手[[詹姆斯·布雷克|詹姆斯·布雷克]]<small><ref>Pratt, Devin (2006). [http://www.fhmus.com/articles-2064.asp "Tennis's Nice Guy Breaks Down the Season"] {{webarchive|url=https://web.archive.org/web/20061017100508/http://www.fhmus.com/articles-2064.asp |date=2006-10-17 }} FHMUs.com. 2007-05-30查閱.</ref></small>。梅爾從小就喜歡聽流行歌，因為看了電影[[回到未來|回到未來]]中[[馬蒂·麥夫萊|馬蒂·麥夫萊]]（[[迈克尔·J·福克斯|迈克尔·J·福克斯]]飾）的吉他表演，從此對吉他深深著迷。十三歲時，他父親為梅爾租了一把吉他，鄰居也給了他一些[[史蒂維·雷·沃恩|史蒂維·雷·沃恩]]的卡帶，自此梅爾便愛上了藍調。儘管父母對他學習吉他有所保留，梅爾仍然著迷於彈奏吉他，經過兩年的練習後，還是高中生的梅爾便開始在藍調酒吧等地演奏。梅爾也和他的朋友提姆·普洛卡錫尼、喬·貝勒茲涅及瑞奇·沃夫，組了一個樂團「維蘭諾華」（''Villanova''）<small><ref name=TW>Wallace, William (2005). [http://www.tweedmag.com/News/Joe_Beleznay_wants_to_be_the_ball.html "Joe Beleznay wants to be the ball"] TweedMag.com. 2006-10-30查閱.</ref></small>。

十七歲時，梅爾因罹患心律失常而住院一個禮拜。「那就是我身體裡的創作歌手誕生的瞬間，」梅爾對此說道，他出院後當晚寫下他的第一首自創曲<small><ref name=RS06>Hiatt, Brian (September 21, 2006), [http://www.rollingstone.com/news/story/11515443/john_mayer_speaks_listen_to_his_hilarious_takes_on_paris_hilton_brad__angelina_living_in_ny "My Big Mouth Strikes Again"]. ''Rolling Stone''. (1009): 66-70</ref></small>。不久後，梅爾得了嚴重的恐慌症，甚至到現在梅爾仍須隨身攜帶一種名為[[阿普唑侖|阿普唑侖]]的抗焦慮藥物<small><ref name=RS06/><ref name=MandS>[http://a1135.g.akamai.net/f/1135/18227/1h/cchannel.download.akamai.com/18227/podcast/RICHMOND-VA/WRVQ-FM/johnmayer2.mp3 "John Mayer"] {{webarchive|url=https://web.archive.org/web/20120127141057/http://a1135.g.akamai.net/f/1135/18227/1h/cchannel.download.akamai.com/18227/podcast/RICHMOND-VA/WRVQ-FM/johnmayer2.mp3 |date=2012-01-27 }}. ''Melissa and Sid''. 2008-03-3].</ref></small>。畢業後，梅爾在一間加油站工作十五個月，直到存了足夠的錢後，買了一把1996年、有史蒂維·雷·沃恩簽名的''Stratocaster''—他第一把專屬吉他<small><ref name="AskMen">No byline (2007). [http://www.people.com/people/john_mayer "John Mayer: Five Fun Facts"] People.com 2007-11-28查閱</ref></small>。

== 音樂生涯 ==

=== 早期 ===
十九歲那年，約翰·梅爾進入[[麻州|麻州]][[波士頓|波士頓]]的[[柏克理音樂學院|柏克理音樂學院]]就讀<small><ref name=ELLE/></small>。兩個學期後，梅爾決定休學，並在他的朋友克雷·庫克（''Clay Cook''）的要求下，一起搬到了[[喬治亞州|喬治亞州]]的[[亞特蘭大|亞特蘭大]]<small><ref name=Berklee>Small, Mark (2005). [http://www.berklee.edu/bt/172/coverstory.html "John Mayer '98: Running with the Big Dogs"] Berklee.edu. 2007-4-23查閱.</ref></small>。在為兩人合組的樂團取了團名「洛飛大師」（''LoFi Masters''）後，梅爾與庫克開始了他們認真的在當地咖啡廳、俱樂部（像是[[艾迪的閣樓|艾迪的閣樓]]（''Eddie's Attic''）等地）演出的一段時光<small><ref name="AskMen" /></small>。然而，梅爾的願望是朝較流行的曲風前進<small><ref name="MOA">Guthrie, Blake (2003). [http://atlanta.creativeloafing.com/gyrobase/Content?oid=oid%3A14103 "Mayer of Atlanta: John Mayer plays Philips Arena, and all I got was this lousy cover story"] {{webarchive|url=https://web.archive.org/web/20080110213456/http://atlanta.creativeloafing.com/gyrobase/Content?oid=oid%3A14103 |date=2008-01-10 }} CreativeLoafing.com. 2007-2-17查閱.</ref></small>，在音樂理念不同的情況下，梅爾與庫克拆夥，梅爾便開始了單飛生涯<small><ref name=Berklee/></small>。

在當地的製作人兼工程師葛倫·馬圖洛（''Glenn Matullo''）的幫助下，梅爾錄製了一張獨立EP《[[表達自我|表達自我]]》（''Inside Wants Out''）。庫克也在EP中擔任許多歌曲的共同作曲人，特別是在梅爾的第一張單曲〈''No Such Thing''〉中<small><ref name="MOA" /></small>。這張EP只收錄了八首歌，除了一首〈''Comfortable''〉只負責歌唱之外，每首都是由梅爾身兼主唱與吉他手。EP的開頭曲〈''Back To You''〉並找了整個樂團協力演出，包括EP的共同製作人[[大衛·迪拉·勒布魯伊爾|大衛·迪拉·勒布魯伊爾]]（''David "DeLa" LaBruyere''）擔任貝斯手<small><ref name=MO>Alter, Gaby (2007). [http://mixonline.com/live/tourprofiles/audio_john_mayer_2/ "Tour Profile: John Mayer"] {{webarchive|url=https://web.archive.org/web/20140904114813/http://mixonline.com/live/tourprofiles/audio_john_mayer_2/ |date=2014-09-04 }} MixOline.com. 2007-4-23查閱.</ref></small>。梅爾並與勒布魯伊爾開始在整個喬治亞州以及周圍的省分進行巡迴演唱。

== 私人生活 ==

梅爾身上有許多刺青，包括分別在左、右手臂後方的「Home」和「Life」（歌曲名）、左胸前的「77」（他的出生年份）以及右肩上像[[錦鯉|錦鯉]]的魚。他的左手臂則覆滿了於2008年4月完成的[[袖刺青|袖刺青]]，裡面包括了肩上的「SRV」（梅爾的偶像史帝維·雷·華漢（''Stevie Ray Vaughan''））、二頭肌上裝飾用的長方形、手臂內側像龍的插畫，以及許多花的插圖。2003年，梅爾在右手前臂上刺了三個長方形<small><ref name=blender>Collis, Clark (2003). [http://www.blender.com/guide/articles.aspx?id=722 "Dear Superstar: John Mayer"] {{webarchive|url=https://web.archive.org/web/20090213135059/http://www.blender.com/guide/articles.aspx?id=722 |date=2009-02-13 }} Blender.com. 2006-11-2查閱.</ref></small>，並於2009年漸漸將其中兩個填滿了圖案<small><ref>No byline (2007).
