'''約翰·雷西格'''（{{lang-en|John Resig}}，{{bd|1984年|5月8日|||John Resig}}）是美國男性[[軟體工程師|軟體工程師]]，為用於簡化網頁程式語言[[JavaScript|JavaScript]]編寫流程的[[JavaScript函式庫|JavaScript函式庫]]「[[jQuery|jQuery]]」之創建者。<ref name="StackOverFlow">{{Cite web |url=http://stackoverflow.com/users/6524/john-resig |title=John Resig less info |work=StackOverFlow .Com |accessdate=2014年10月24日 }}</ref>

== 經歷 ==
在就讀[[羅徹斯特理工學院|羅徹斯特理工學院]]期間，雷西格主修有關[[電腦科學|電腦科學]]的領域課程，並曾實作過關於網路數據的[[資料探勘|資料探勘]]、或是有關即時線上協作方面的研究。<ref name="RIT">{{Cite web |url=http://www.rit.edu/alumni/ihf/inductee.php?inductee=10 |title=John Resig |work=[[羅徹斯特理工學院|羅徹斯特理工學院]] |accessdate=2014年10月24日 }}</ref><ref>{{Cite web |url=http://scholar.google.com/citations?user=zk6oZ8YAAAAJ |title=John Resig Rochester Institute of Technology |work=Google 學術搜尋 |accessdate=2014年10月24日 }}</ref>

2006年雷西格則設計出可簡化[[JavaScript|JavaScript]]開發流程方法的應用函式庫「[[jQuery|jQuery]]」，此成果後續讓雷西格在2010年4月間被母校羅徹斯特理工學院入選為對於資訊技術革新有功的人物。<ref name="RIT"/>而在jQuery方面的設計以外，雷西格也陸續投入{{le|Processing.js}}<ref>{{Cite web |url=http://processingjs.org/ |title=About Processing.js |work=Process.js |accessdate=2014年10月24日 }}</ref>、[[EnvJS|EnvJS]]<ref>{{Cite web |url=http://www.envjs.com/ |title=Envjs |work=Envjs.Com |accessdate=2014年10月24日 |archive-url=https://web.archive.org/web/20091019142240/http://www.envjs.com/ |archive-date=2009-10-19 |dead-url=yes }}</ref>、[[TestSwarm|TestSwarm]]<ref>{{Cite web |url=https://github.com/jquery/testswarm/wiki |title=home |work=Git Hub |accessdate=2014年10月24日 }}</ref>、[[Sizzle|Sizzle]]、[[FUEL|FUEL]]等與JavaScript應用相關的專案。

雷西格除先後曾於[[Mozilla公司|Mozilla公司]]擔任JavaScript方面使用工具的開發、以及在[[可汗學院|可汗學院]]負責應用程式設計外，<ref name="StackOverFlow"/>亦多次出席擔任網際網路公司[[Google|Google]]或[[雅虎|雅虎]]所舉辦活動的演講者、和參與[[西南偏南|西南偏南]]等網路會議相關活動，並有-{訪問}-學者的身份在日本[[立命館大學|立命館大學]]涉獵有關[[浮世繪|浮世繪]]方面事物。<ref>{{Cite web |url=http://ejohn.org/about/ |title=About |work=ejohn.org |author=約翰·雷西格 |accessdate=2014年10月24日 }}</ref>
