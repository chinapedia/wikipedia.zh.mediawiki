{{日本作品|約會～戀愛為何物～|デート〜恋とはどんなものかしら〜|Date: What's it like to be in love?}}，為日本[[富士電視台|富士電視台]]於[[富士電視台週一晚間九點連續劇|月九]]時段（週一21:00 - 21:54，[[日本標準時|JST]]）播出的[[日本電視劇|電視連續劇]]，由[[杏_(演員)|杏]]主演，為其初次主演月九劇<ref>{{cite web|url =http://thetv.jp/news_detail/52824/|title =杏&長谷川博己が“月9”でデート! しかし恋愛力はゼロ!?|publisher=webザテレビジョン|date=2014-11-28|accessdate=2014-12-01}}</ref><ref>{{cite web|url =http://mantan-web.jp/2014/11/28/20141128dog00m200024000c.html|title =杏：“月9”初主演で前髪バッサリ 長谷川博己と“恋愛不適合者”のラブコメディー|publisher=毎日新聞デジタル|date=2014-11-28|accessdate=2014-12-01}}</ref>。此劇亦是執筆《[[永遠的三丁目的夕陽|永遠的三丁目的夕陽]]》系列及《[[Legal_high|Legal high]]》等作品的知名編劇[[古澤良太|古澤良太]]初次編寫戀愛題材。特別篇{{日本作品|約會大作戰 2015夏 祕湯|デート〜恋とはどんなものかしら〜 2015夏 祕湯}}於2015年9月28日於富士電視台播出（9:00 - 11:18，[[日本標準時|JST]]）。

== 製作概要 ==
[[富士電視台|富士電視台]]黃金時段「[[月九|月九]]」以往以「戀愛劇」為人所知，如《[[悠長假期|長假]]》、《[[戀愛世代|戀愛世代]]》、《[[大和拜金女|大和拜金女]]》等，其後，題材趨於廣泛；此劇是[[月九|月九]]在睽違數檔之後重回原創戀愛劇之路線。[[杏_(演員)|杏]]初次主演月九劇，[[長谷川博己|長谷川博己]]亦是初次演出月九。<ref>[http://www.cinemacafe.net/article/2014/11/28/27658.html 杏＆長谷川博己が初“月9”ドラマ！　超理系女子×ニート男子で「結婚」目指す]</ref>

此劇背景為[[橫濱市|橫濱市]]，描述主攻[[總體經濟學|總體經濟學]]的超頑強型29歲國家[[公務員|公務員]]藪下依子（[[杏_(演員)|杏]] 飾）與熱愛[[文學|文學]][[藝術|藝術]]的[[尼特族|尼特族]]谷口巧（[[長谷川博己|長谷川博己]] 飾）之間的故事。兩人戀愛力為「零」，藉著不斷約會以累積戀愛經驗，試圖達成結婚的心願。<ref>[http://www.holyshare.com.tw/enter/index/articleid/29209/ 冬季日劇《約會》杏搭檔長谷川博己出演戀愛雙菜鳥]</ref>

1月12日，此劇召開正式記者會。亦是新婚後的杏與國仲涼子首次出現在公開媒體場合。<ref>[http://www.oricon.co.jp/news/2047086/full/ 国仲涼子、結婚後初公の場 杏と“新婚女優”揃い踏み]</ref><ref>[http://www.cinemacafe.net/article/2015/01/12/28621.html “新妻”杏＆国仲涼子が揃い踏み！ 理想のデートを語る]</ref><ref>{{cite news|title=新手人妻夾擊 光棍凍未條啦 |url=http://www.appledaily.com.tw/realtimenews/article/new/20150112/540555/ |language=zh-hant |author=吳惠菁／綜合報導 |publisher=蘋果日報 |date=2015-01-12 }}</ref>

== 劇情大綱 ==
在[[東京大學|東京大學]]數理研究所研究[[千禧年大獎難題|千禧問題]]的藪下依子（[[杏_(演員)|杏]] 飾），畢業後以[[總體經濟學|總體經濟學]]的分析專長進入[[內閣府|內閣府]]經濟總合研究所，獨自一人住在國家提供的公務員宿舍。生性古板、擁有數千萬存款的藪下，戀愛能力卻為零，因年近30歲，加上父親的關心，因而開始煩惱著結婚一事。藪下登錄了結婚網站欲尋得真命天子。

谷口巧（[[長谷川博己|長谷川博己]] 飾），自稱「[[高等遊民|高等遊民]]」，雖已35歲但仍無正職，因而投靠母親。谷口鎮日沉迷在文學的世界中，為不折不扣的[[啃老族|啃老族]]，靠著開設美術教室的母親過活。因母親身體漸漸不適而感到不安的谷口，決定找個「富婆」希望能少奮鬥20年。谷口以任職「出版社」的假資料登錄結婚網站......。

身為戀愛白癡的藪下和谷口以結婚為目標開始約會。這是一場「笨拙約會」的愛情故事。<ref>[http://japan.videoland.com.tw/channel/20150601/introduction.html 緯來日本台－約會大作戰]</ref>

== 主要角色 ==

:29歲，生日1985年3月22日、血型AB型的國家[[公務員|公務員]]。性格認真、有條不紊、一本正經、邏輯清晰、語速過人、工作效率一人抵五人的菁英理科女子，但討厭被稱呼為「理科女」。在國家配給的宿舍一人獨居。
:對[[質數|質數]]有特別愛好，有驚人的數字記憶能力，能背誦圓周率至23000位的數字。物品的放置、每天的服裝和食物都有固定的依循準則。
:以30歲之前結婚為目標，因此透過交友網站尋找對象，進而認識谷口巧。
*谷口巧（[[長谷川博己|長谷川博己]]／小學時期：{{link-ja|山崎龍太郎|山崎竜太郎}}）：
:35歲，生日1979年7月23日、血型A型的[[尼特族|尼特族]]。熱愛文學與藝術，自稱[[高等遊民|高等遊民]]。13年來大門不出二門不邁、資訊嚴重滯後、茶來伸手飯來張口的[[家裡蹲|家裡蹲]]。
:心目中的理想女性是「[[原節子|原節子]]、[[峰不二子|峰不二子]]、[[梅黛兒|梅黛兒]]、[[奧黛麗·赫本|奧黛麗·赫本]]加起來除以四」的女子。

:35歲，谷口的老友，經營島田工程所。因疑似搞婚外情而氣走老婆。
*島田佳織（[[國仲涼子|國仲涼子]]／小學時期：高野友那）：
:32歲，前不良少女。島田宗太郎的妹妹，在島田工程所工作，擅長畫畫，有時會幫忙谷口留美的美術教室，十分關心谷口。
*鷲尾豐（[[中島裕翔|中島裕翔]]）：
:26歲，運動用品公司職員。因喜好[[劍道|劍道]]而和藪下俊雄相識的年輕上班族，喜歡藪下依子。
*藪下小夜子（[[和久井映見|和久井映見]]）：
:女主角依子的母親，數學家，39歲因胃癌過世。時常以[[幽靈|幽靈]]姿態出現在藪下依子身邊與其對話，經常被女兒依子趕走。
*谷口留美（[[風吹純|風吹純]]）：
:61歲，谷口巧的母親。開設美術教室，與35歲的無業兒子相依為命。接受了兒子是[[尼特族|尼特族]]的事實。
*藪下俊雄（[[松重豐|松重豐]]）：
:56歲，藪下依子的父親。擔心依子的終身大事。
