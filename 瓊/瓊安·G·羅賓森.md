'''瓊安·蓋爾·羅賓森'''（{{lang-en|Joan Gale Robinson}}，{{bd|1910年||1988年||catIdx=Joan G Robinson}}），婚後姓'''湯瑪斯'''（Thomas），是二十世紀出生於英國[[白金漢郡|白金漢郡]]{{link-en|杰羅茲克羅斯|Gerrards Cross}}的女性作家、插畫家。

== 生平 ==
早年羅賓森曾在切爾西插畫工作室（Chelsea Illustrators Studio）學習美術技法，<ref name="usm"/>之後1939起羅賓森開始從事作家生涯，並為自己的著作提供插圖。<ref>{{Cite web |url=http://www.carolinesheldon.co.uk/pdf/dbpdf.php?dbID=96 |title=joan G. Robinson Agent:Caroline Sheldon |work=Caroline Sheldon |format=[[pdf|pdf]] |page=第1頁 |accessdate=2014年10月4日 |deadurl=yes |archiveurl=https://web.archive.org/web/20141006100943/http://www.carolinesheldon.co.uk/pdf/dbpdf.php?dbID=96 |archivedate=2014年10月6日 }}</ref>在創作期間時除多以瓊安·G·羅賓森為筆名外、也曾以本姓湯瑪斯的名義來發表作品。<ref name="usm"/>

羅賓森生平中的著作包含《瑪麗瑪麗》（Mary-Mary）系列、和以女兒黛博拉（Deborah）喜愛的[[泰迪熊|泰迪熊]]為題材的《泰迪羅賓森》（Teddy Robinson）系列，<ref>{{Cite web |url=https://www.hotkeybooks.com/authors/detail/joan-g.-robinson |title=Joan G. Robinson |work=HotKey Books |accessdate=2014年10月4日 |deadurl=yes |archiveurl=https://web.archive.org/web/20141006161520/https://www.hotkeybooks.com/authors/detail/joan-g.-robinson |archivedate=2014年10月6日 }}</ref>以及往後2014年有被日本動畫室[[吉卜力工作室|吉卜力工作室]]改編成[[回憶中的瑪妮|同名動畫]]的《[[回憶中的瑪妮_(小說)|回憶中的瑪妮]]》等等。<ref>{{Cite web |url=http://eiga.com/news/20140416/2/ |script-title=ja:米林宏昌監督「思い出のマーニー」にかける並々ならぬ思い |publisher=Eiga.Com inc |date=2014年4月16日 |accessdate=2014年10月4日 }}</ref>

於1988年，羅賓森在進行度假期間裡離世。<ref name="usm"/><ref>{{cite book |title=[[回憶中的瑪妮_(小說)|回憶中的瑪妮]]：後記 |date=2014年11月1日 |publisher=[[台灣東販|台灣東販]] |isbn=978-986-331-549-0 |page=228頁 }}</ref>
