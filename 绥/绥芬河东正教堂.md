'''绥芬河东正教堂'''又称'''协达亚·尼古拉东正教堂''' ({{lang-ru|Свято-Николаевская церковь}})，俗称'''喇嘛台'''。位于[[黑龙江省|黑龙江省]][[绥芬河市|绥芬河市]]。

1908年，在铁路中東鐵路[[绥芬河站|绥芬河站]]附近的木造東正教教堂被焚 (建於1898年)<ref>{{Cite web |url=http://www.orthodox.cn/localchurch/harbin/suifenhe_cn.htm|title=绥芬河协达亚·尼古拉堂|accessdate=2017-06-28  |language=Zh-hans }}</ref>。1913年，俄僑在车站正东建成砖结构的协达亚·尼古拉教堂<ref>{{Cite web|url=http://hlj.sina.com.cn/travel/line/2013-06-19/063857147_4.html|title=绥芬河东正教堂旧址|accessdate=2013-07-07|author=|date=2013年6月19日|work=|publisher=新浪网|language=Zh-hans}}{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>。教堂在文化大革命中被破坏，恢复后[[三自爱国教会|三自爱国教会]]被中国政府允许使用教堂。1999年2月1日，[[黑龙江省人民政府|黑龙江省人民政府]]公布为[[第四批黑龙江省文物保护单位|第四批黑龙江省文物保护单位]]。2007年修复塔楼，今天为基督新教礼拜堂-福音堂使用。<ref>{{Cite web |url=http://www.orthodox.cn/contemporary/harbin/suifenhe_cn.htm|title=绥芬河教堂旧址|accessdate=2017-06-28  |language=Zh-hans }}</ref>
