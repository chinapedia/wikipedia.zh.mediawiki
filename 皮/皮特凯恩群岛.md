'''皮特凯恩群岛'''（{{lang-en|Pitcairn Islands}}、[[諾福克語|諾福克語]]：{{lang|pih|Pitkern Ailen}}），正式名称为'''皮特凯恩、[[亨德森岛|亨德森]]、[[迪西島|迪西]]和[[奧埃諾島|奧埃諾]]群岛'''，是由4座岛屿组成的[[南太平洋|南太平洋]][[群岛|群岛]]，其中只有第二大岛嶼'''皮特凯恩岛'''有人定居。该群岛也是[[英国|英国]]在[[太平洋|太平洋]]地区剩下的最后一块[[英国海外领土|海外领地]]。

小岛之所以出名，是因为岛上居民都是英国皇家海軍{{link-en|邦蒂号|HMS Bounty}}上的[[邦蒂號叛變|叛变]]船员和大溪地人的后裔，这段具有传奇性的历史已经被写成小说，并被拍成《[[叛艦喋血記|叛艦喋血記]]》等多部电影。此外该岛也可能是世界上居住人口最少的地區，大约只有48人（9個家庭）还居住在此。皮特凯恩還是全球人口最少的具有管轄權的地區（尽管它還不是一個[[主權國家|主權國家]]）。聯合国非自治領土列表里面包括皮特凯恩群岛。2002年，岛上的7名男性居民被[[新西兰|新西兰]]当局以[[性侵害|性侵害]]和强暴猥亵女童罪的罪名起诉，再度引起了[[媒体|媒体]]的關注。

== 历史 ==
[[File:Mutiny_HMS_Bounty.jpg|thumb]]船员叛变，1789年4月29号]]
{{Main|皮特凯恩群岛历史}}

[[考古学家|考古学家]]认为，[[波利尼西亚人|波利尼西亚人]]是皮特凯恩群岛的最早居民，他们在岛上的踪迹最晚也可追溯到公元15世纪。但是当[[葡萄牙|葡萄牙]]探险家[[奎洛斯|奎洛斯]]发现该群岛时，岛上已经无人居住。1767年英国人皮特凯恩重新发现了该群岛，并以自己的名字为群岛命名。

1790年1月15日，[[邦蒂号|邦蒂号]]上的9名叛变船员和他们的[[大溪地|大溪地]]同伴逃亡到该岛。他们包括了叛变的首领、船上的大副弗莱彻·克里斯蒂安（{{lang|en|Fletcher Christian}}）和其他8名船员、6名大溪地男人和12名大溪地女人。他们将邦蒂号上所有的物资都搬下船，然后放火把整艘船都烧了。不过今天人们还是可以在海底看到邦蒂号的残骸。

在岛上，船员很快就发生了内乱。加上疾病侵袭，到1800年時，叛變的船員中，只剩下约翰·亚当斯一人幸存。1814年[[英国海军|英国海军]]再度发现了该岛，但却被岛民的历史所感动，因此决定不将最后剩下的叛变者逮捕，认为这样做是“极不人道与残酷的”。

1838年该岛正式成为英国[[殖民地|殖民地]]。到1850年中岛上的居民人口多到小岛无法承受的地步，岛民们于是向[[维多利亚女王|维多利亚女王]]求援。女王批准岛民们迁徙至[[诺福克岛|诺福克岛]]（Norfolk Island）。于是在1856年5月3日，全部193名岛上的居民开始迁徙。但是18个月后，17名岛民又重新回到了皮特凯恩群岛，5年后又有27人返回家园。

1937年岛上的人口达到历史性的高峰233人，之后又开始了大规模的迁徙活动，大部分是迁往邻近的[[新西兰|新西兰]]，使得今天岛上的人口保持在50人左右。1997年，一名英国女记者到达该岛采访，并首次披露了岛上长达200年的对未成年女童性侵害的历史，之后又有许多以前岛上的女性居民向新西兰警方申诉，自己从12岁起就被岛上其他有血缘关系的男性居民性侵害或[[强奸|强奸]]甚至[[轮奸|轮奸]]。2002年新西兰当局决定以强暴、猥亵女童等罪名起诉岛上的7名男性居民。目前审讯已經完結。

== 政治 ==
{{Main|皮特凯恩群岛政治}}

皮特凯恩群岛的政治基于一个[[议会民主制|议会民主制]]的框架，[[英女王|英女王]]是[[国家元首|国家元首]]。[[市长|市长]]是[[政府首脑|政府首脑]]。该岛的[[宪法|宪法]]制订于1964年。皮特凯恩群岛是世界上最小的[[民主政体|民主政体]]。皮特凯恩的[[总督|总督]]由英国驻新西兰[[最高专员|最高专员]]担任，因此目前该群岛的总督是[[喬納森·辛克萊|喬納森·辛克萊]]（2014年8月15日上任），特派员是[[喬納森·辛克萊|喬納森·辛克萊]]（2014年8月15日上任）。岛上的日常事务由一名民选的市长担任，前任市长史蒂夫·克里斯蒂安已经被新西兰政府以强暴等罪名起诉，现任市长为[[肖恩·克里斯坦_(市长)|肖恩·克里斯坦]]（2014年1月1日上任）。皮特凯恩群岛的立法机关为[[岛屿委员会_(皮特凯恩群岛)|岛屿委员会]]（其中总督和市长都是[[官守議員|当然议员]]，4个议席由[[普选|普选]]产生，1个议席由议长任命，2个议席由总督任命）。市长任期是3年，特派员任期不定，[[议员|议员]]任期是1年。

== 军事 ==
皮特凯恩群岛属于[[英国海外领土|英国海外领土]]，这也意味着防务由英国国防部和英国军队负责。

== 地理 ==

皮特凯恩群岛由4个岛组成，分别是皮特凯恩岛、[[迪西島|迪西島]]、[[亨德森岛|亨德森岛]]、[[奧埃諾島|奧埃諾島]]。

唯一始终有人居住的岛是皮特凯恩岛，只能通过小船从邦蒂湾登陆。

[[亨德森岛|亨德森岛]]大概占了群岛总面积的67％，在内部人迹罕至的地方一些独有的动植物品种，虽然这个岛也可以容纳少部分人定居，但很难登陆，它的海岸由尖利陡峭的[[珊瑚礁|珊瑚礁]]组成。

皮特凯恩岛是由被称为皮特凯恩[[热点_(地质学)|热点]]的上升岩浆形成的。

其它几个岛的距离都超过了100公里且无人定居。

== 经济 ==
皮特凯恩的肥沃的山谷一直出产多种[[蔬菜|蔬菜]]和[[水果|水果]]，其中包括[[柑橘|柑橘]]、[[甘蔗|甘蔗]]、[[水果|水果]]、[[香蕉|香蕉]]、[[薯蓣|薯蓣]]和[[豆|豆]]类植物。不过岛上居民的主要生活来源倒不是[[农业|农业]]，而是[[渔业|渔业]]，易货贸易依然是岛上的主要贸易形式。此外[[邮票|邮票]]发行也是小岛的主要收入之一，[[蜂蜜|蜂蜜]]和[[工艺品|工艺品]]也帮助岛民增加收入，他们的主要买家都是途经皮特开恩前往[[巴拿马运河|巴拿马运河]]的英国和新西兰船只。主要的劳动力是岛上的12名男性居民。

== 人口 ==
皮特凯恩岛居民主要由[[邦蒂号|邦蒂号]]上的叛变船员和[[大溪地|大溪地]]人的后裔组成，大部分居民都是[[基督复临安息日会|基督复临安息日会]]的信徒。[[皮特肯語|皮特肯語]]是一种主要为18世纪[[英语|英语]]并混合了一部分[[大溪地語|大溪地語]]的[[克里奥尔语|克里奥尔语]]，它被当地居民作为第一语言并在岛上唯一一所学校和英语一起做为教学语言。[[皮特肯語|皮特肯語]]和另外一种克里奥尔语[[诺福克语|诺福克语]]有密切关系，因为皮特凯恩人19世纪中曾在[[诺福克岛|诺福克岛]]上生活过。

2003年9月，事隔17年岛上再次有小孩出生。2007年3月3日，另外一个孩子亞德里安娜·翠西·克里斯坦（Adrianna Tracey Christian）也在岛上出生。

== 交通 ==
岛上的居民都是乘船而来，其中最著名的是邦蒂号，该船后来发生叛变并且在邦蒂湾被烧毁。

皮特凯恩岛没有机场或海港，岛上居民依靠大艇通过邦蒂湾，在汽船和陆地之间运输人和货物。如果要去皮特凯恩岛必须先飞到塔希提，然后到[[甘比爾群島|曼加雷瓦]]，再坐超过30小时的船，每几个月只有一艘船航行。另外一个选择是坐从新西兰出发的货船，那样要七天的时间。离岛就要碰运气了，并不是什么时候想走就有船的。

岛上共有6.4公里长的路，没有铁路，因此长时间步行是去往岛上其它地方的唯一办法。

上个世纪70年代初，为了方便老年人，人们将一部[[Mini_moke|Mini moke]]车带到岛上来，但崎岖的地形和暴雨很快就损坏了这辆微型车，然后第二辆，第三辆[[Mini_moke|Mini moke]]车不得不送到岛上来替代它，最近几年更多适应该岛地形的车辆出现了。

== 法律与犯罪 ==
[[File:Adamstown1.jpg|缩略图]]
岛上禁止[[舞蹈|舞蹈]]、[[酒精|酒精]]和公开示爱。但是2003年开始，对酒精的禁令似乎有所松动，多个家庭已经开始饮酒。

目前岛上7名男性居民和已经离开群岛的另外6名居民面临着对未成年少女性侵害的罪名。2002年新西兰通过一项特别法案，特别允许在新西兰领土上对案件进行审理；但是2004年13名被告成功地上诉，迫使审判必须在皮特开恩群岛上举行。3名法官、7名原告双方律师以及其他法庭工作人员和6名记者已经在9月底抵达了皮特凯恩群岛，并且审判也已经展开。预计审判将持续7周，居住在海外的证人将通过[[卫星|卫星]]视讯作证。

起诉的罪名包括21起强奸案、41起性骚扰案和2起不正当性接触，受害者全部是14岁以下女童。皮特开恩的男性居民指责警方诱迫当事的女性成员作证，而多位女性成员也正在努力试图收回自己的证词。他们认为与未成年女子性交是岛上的传统，特别是在人口如此少的岛屿上，两性之间不可避免地会发生正常社会中无法接受的非正当关系。

审判其间岛上所有的居民被勒令交出20把手枪，避免可能伤害到岛上参与审判的外国人。
