'''塘沽区'''是原[[中华人民共和国|中华人民共和国]][[天津市|天津市]]下辖的一个[[市辖区|市辖区]]，现为[[滨海新区|滨海新区]]的核心区。原塘沽区面积为790.2[[平方千米|平方千米]]，人口为46.2万。

== 历史 ==
[[File:塘沽地區的貝殼堤實土採樣.jpg|thumb]]
宋庆历八年（公元1048年），[[黄河|黄河]]改道北移，夺[[海河|海河]]河道，在[[天津|天津]]附近入海。黄河有惊人的造陆能力，至南宋建炎二年（公元1128年），塘沽陆地逐步形成<ref>滨海新区历史建置沿革 {{cite web |url=http://www.bh.gov.cn/html/BHXQZWW/BHSZ22361/2012-02-29/Detail_526028.htm |title=存档副本 |accessdate=2015-08-13 |deadurl=yes |archiveurl=https://web.archive.org/web/20150923184333/http://www.bh.gov.cn/html/BHXQZWW/BHSZ22361/2012-02-29/Detail_526028.htm |archivedate=2015-09-23 }}</ref>。20世纪80年代后，塘沽区先后建立了[[天津经济技术开发区|天津经济技术开发区]]、天津港保税区和国家海洋高新技术开发区。2009年11月9日，塘沽区和[[汉沽区|汉沽区]]、[[大港区|大港区]]合并成为新的市辖区[[滨海新区|滨海新区]]<ref>[http://news.xinhuanet.com/fortune/2009-11/09/content_12417762.htm 国务院正式批复同意天津市调整滨海新区行政区划]，新华网，2009年11月9日。</ref>。
== 地理 ==
原塘沽区地理坐标为北纬38°44′～39°13′，东经117°30′～117°46，面积为790.2[[平方千米|平方千米]]，人口为46.2万。塘沽地热资源丰富，储量达434.9亿吨，出水温度为65-78℃，“海水”与“温泉”的结合在沿海大城市非常少见。塘沽地区水体资源丰富，有91.16千米长的海岸线，55公里海滩，境内海河长度38千米，水域面积63平方千米。因此，其水产品资源丰富。塘沽滩涂广阔延缓，高潮带盛产蟹类，中潮带是鱼虾蟹贝类自然繁殖区，著名的[[渤海|渤海]]对虾、梭子蟹享誉国内外，海水养殖业和以海鲜为特色的餐饮业发达。<ref>[http://www.tanggu.gov.cn/tggl/index.html 塘沽区人民政府] {{webarchive|url=https://web.archive.org/web/20090412055533/http://www.tanggu.gov.cn/tggl/index.html |date=2009-04-12 }} 塘沽概览</ref>

== 经济 ==
原塘沽区的[[天津港|天津港]]是中国北方第一大港口，世界第五大港口，也是中国最大的人工港，年吞吐量2.5亿吨。塘沽区聚集了[[化工|化工]]、制盐、海洋石油及造船基地等大企业，原塘沽区还有中国最大的盐业基地[[天津碱厂|天津碱厂]]，还有建于1880年的[[北洋水师|北洋水师]]大沽船坞遗址和中国化学工业的摇篮“[[黄海化学工业研究社旧址|黄海学社]]”遗址。

== 文化 ==
原塘沽区拥有[[全国重点文物保护单位|全国重点文物保护单位]]、津门十景之一的[[大沽口炮台|大沽口炮台]]遗址和始建于[[明朝|明朝]][[永乐|永乐]]年间的海门古刹塘沽潮音寺。

== 交通 ==
原塘沽区拥有[[海河大桥|海河大桥]]，该桥是世界第三的独塔斜拉桥。2004年3月28日，连接[[天津中心城区|天津市中心]]和塘沽的[[天津地铁九号线|天津地铁九号线]]通车试运营。

== 行政区划 ==
原塘沽区辖11个街道、1个镇：[[新村街道|新村街道]]、解放路街道、三槐路街道、新港街道、向阳街道、杭州道街道、新河街道、大沽街道、北塘街道、渤海石油街道、胡家园街道、新城镇。
