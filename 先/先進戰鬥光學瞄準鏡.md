'''先进战斗光学瞄準鏡'''（{{Lang-en|'''A'''dvanced '''C'''ombat '''O'''ptical '''G'''unsight}}，簡稱：'''ACOG'''）是一種由[[Trijicon公司|Trijicon公司]]研製及生產的[[光學|光学]][[瞄准镜|瞄准镜]]系統。ACOG光學瞄準鏡裝置在設計上是用在[[M16突擊步槍|M16系列步槍]]（即是[[M4卡賓槍|M4卡賓槍]]也可裝上），但Trijicon公司製造的ACOG配件亦可協助安裝於其他的武器。不同型號的ACOG可提供1.5—6倍的[[望远镜放大倍数|放大倍率]]。<ref>{{cite web |url=http://www.trijicon.com/na_en/products/product1.php?id=ACOG |title=Official Listing of ACOG models |accessdate=2011-02-11}}</ref>

與許多反射式瞄准镜（例如[[Aimpoint_Comp_M2紅點鏡|Aimpoint Comp M2]]）相反，ACOG內部的[[瞄准標記|疊加式分劃標記]]是由內置式[[荧光|螢光粉]]作夜間照明。有些型號更會在瞄准鏡的外部頂部端加上一條採光用途的被動外置式[[光導纖維|光纖]][[導光管|導光管]]系统，以便在白天時間吸收自然光（光纖有多種花紋和顏色，而目前最常見的是[[红色|红色]]），然後在自然光不足時使用安裝於ACOG內置式低[[放射性|放射性]]的[[氢|氢]][[同位素|同位素]][[氚|氚]][[燈|燈]]提供光源。由於氚的[[半衰期|半衰期]]約為10—15年，在這段時間亮度會因為放射性衰變而在一般的時間內減少了一半並且失去光澤，這時便需要更換內部的氚燈。

== 設計 ==
ACOG是專為M16系列步槍使用而製作。安裝時，它具有一個“夾片式”設計的基座，使視線望向瞄準點後使用。

ACOG提供了多種不同型號，包括製造商配置不同距離用的{{Link-en|分劃板|Reticle}}、照明系統和其他功能。大部份的ACOG型號都無需使用[[电池|电池]]，<ref name="2011/10/28">[http://www.thefirearmblog.com/blog/2011/10/28/trijicon-4x32-battery-acog/ The Firearm Blog—Trijicon 4×32 Battery ACOG]</ref>並被設計成使用內置式[[放射性|放射性衰變]][[氚|氚]][[荧光|螢光粉]]，可在任何情況以下提供照明。氚光照明的使用壽命為10—15年。<ref>氚的半衰期為12.5年。</ref>有些型號更會在瞄准鏡的外部頂部端加上一條採光用途的被動外置式[[光導纖維|光纖]][[導光管|導光管]]系统，這在通常情況下能夠使分劃標記的亮度能夠與視野配合，因為它收集來自瞄準鏡以外周圍的環境光。但是在某些情況，例如陽光直接射向導光管以下，分劃標記的亮度未能夠與視野配合。<ref>[http://jerkingthetrigger.com/blog/2010/03/16/tactical-handyman-acog-fiber-optic-fix/ actical Handyman – ACOG Fiber Optic Fix]</ref>分劃標記並具有其他功能，例如[[彈著補助|彈著補助]]（{{Lang-en|'''B'''ullet '''D'''rop '''C'''ompensation}}，簡稱：BDC）和其他不同的分劃標記形狀（圓形、倒V形、[[三角形|三角形]]或圈內點形）。

TA01NSN（[[北約庫藏編號|國家庫存編號]]：1240-01-412-6608）具有[[望远镜放大倍数|放大]]4倍、十字式瞄準分劃線和子彈[[弹道学|彈道]]補償（{{Lang-en|Bullet Drop Compensator}}，簡稱：BDC）。因此，就算沒有經過任何調整，都能夠令使用者攻擊距離在600公尺（656.17[[码|码]]）的目標，其掩膜也會在黑暗環境之中發出琥珀色的光<ref>[http://www.trijicon.com/na_en/products/product3.php?pid=TA01NSN TA01NSN: Trijicon ACOG 4x32 Scope with Amber Center Illumination for M4A1 – includes Flat Top Adapter, Backup Iron Sights and Dust Cover]</ref>。另外，有些型號會按照客戶要求而在頂部裝上整合式的三角形柱狀的[[機械瞄具#鬼環|鬼環]]式[[照門|照門]]及片狀[[準星|準星]]作為後備[[機械瞄具|機械瞄具]]，攻擊距離在50[[米_(单位)|公尺]]（54.68[[码|码]]，164.04[[英尺|英尺]]）以內。

TA31RCO亦具有[[望远镜放大倍数|放大]]4倍能力，不過沒有後備[[機械瞄具|機械瞄具]]，並使用[[红色|红色]]「Λ」字式瞄準分劃線，可自動調整的光線條件，並具有子彈[[弹道学|彈道]]補償，能夠令使用者攻擊距離在800公尺（874.89码，1,968.5英尺）的目標<ref>[http://www.trijicon.com/na_en/products/product3.php?pid=TA31RCO-A4CP TA31RCO-A4CP: Trijicon ACOG 4x32 Scope with BAC USMC Rifle Combat Optic (RCO) for A4 (20" barrel)]</ref>。

M150 RCO更增加了翻蓋子，[[防水|防水]]深度到了11公尺（12.03码，36.09英尺）。<ref>{{cite web |url=http://www.trijicon.com/user/parts/products1.cfm?PartID=637&back_row=4&categoryID=3 |title=TA31RCO-M150CP: 4x32 Trijicon ACOG Army Rifle Combat Optic (RCO) |publisher=Trijicon.com |date= |accessdate=2010-05-23 |archiveurl=https://web.archive.org/web/20100608012944/http://www.trijicon.com/user/parts/products1.cfm?PartID=637&back_row=4&categoryID=3 |archivedate=2010年6月8日 |deadurl=yes }}</ref>有些型號甚至有30公尺（32.81码，98.43英尺）的防水深度。

== 賓登式瞄準概念 ==
有多款型號的ACOG的設計使用了“賓登式瞄準概念”（{{Lang-en|Bindon Aiming Concept}}，簡稱：BAC）的自動變倍功能，這種瞄準技術概念是由[[Trijicon公司|Trijicon公司]][[創辦人|創辦人]]和光學設計師格林·賓登（{{Lang-en|Glyn Bindon}}）所開發的。具有賓登式瞄準概念的瞄準鏡在武器快速運動過程之中只能[[望远镜放大倍数|放大]]1倍；而武器停止運動或緩慢運動時則會自動恢復到原有的放大倍率。

該技術主要是利用因高亮度[[光導纖維|光纖]][[導光管|導光管]]照明系統而發出[[氚|氚]]光的瞄準鏡{{Link-en|分劃板|Reticle}}和瞄準鏡後方的新型聚焦[[目镜|目镜]]和[[稜鏡|稜鏡]]所組成作為{{Link-en|準直式瞄準鏡|Collimator sight|準直式}}[[瞄準鏡|瞄準鏡]]的正像系統這兩個部分。<ref>Jane's international defense review: IDR., Volume 34, Issues 1-6</ref>正如任何其他的準直式瞄準鏡型號，使用者瞄準時其實並不需要完全專注要通過瞄準鏡瞄準，而是以保持（[[无穷|無限]]）平行的影像瞄準，其中一隻眼睛（作為瞄準眼，也就是通過瞄準鏡進行觀察的眼睛）的[[焦點|焦點]]就在瞄準鏡分劃板上發出氚光部分以取得其清晰影像，而另一隻眼睛（作為裸眼）則是保持觀察整個[[視野|視野]]以取得的清晰全景影像去搜索其他目標。另外一部分技術就是在進行更精確的射擊時，在瞄準眼和裸眼的清晰影像以保持著平行的模式以下同時被傳遞到[[大腦|大腦]]，從而避免了因只集中於一個合焦距離傳來的清晰影像造成了人腦強制性選擇對觀察／瞄準所取得的影像而對反應速度造成的影響。在觀察和瞄準都是同時進行以下，使得瞄準眼和裸眼能夠同時工作。這解決了所有傳統的光學瞄準鏡對於觀察瞄準鏡{{Link-en|分劃板|Reticle}}中心附近的目標或是快速橫向移動的目標時，必須先在兩隻眼之間進行切換以取得清晰影像才能尋找目標、進行瞄準和射擊的相同問題。只有某些型號的ACOG的設計在光纖導光管照明系統有著足夠亮度照著分劃板以下才會使用這種技術。

[[美国陆军|美国陆军]]和[[美国海军陆战队|海军陆战队]]兩個武装力量都分別為其[[M4卡賓槍|M4卡賓槍]]和[[M16突擊步槍#M16A4|M16A4步槍]]購入TA31RCO ACOG，一個4倍[[望远镜放大倍数|放大倍率]]以及32毫米光學[[出射瞳|出射瞳]]（4×32），並且裝上專用設計的[[弹道学|彈道]]補償分劃標記、光纖和氚光照明系統的光學瞄準鏡。<ref>[http://combatopticsreviews.com/trijicon-acog-ta31rco-m4cp/ Combat Optics Reviews  Trijicon ACOG TA31RCO-M4CP (USMC AN/PVQ-31B), Posted on December 17, 2010]</ref>在美国陆军服役的將其命名為'''M150步槍戰鬥光學瞄準鏡'''，而在美国海军陆战队服役的則將其命名為'''AN/PVQ-31步槍戰鬥光學瞄準鏡'''。<ref>{{cite web |url=http://www.trijicon.com/whats_new.cfm |title=Trijicon website |publisher=Trijicon.com |date= |accessdate=2010-05-23 |deadurl=yes |archiveurl=https://web.archive.org/web/20100521053355/http://www.trijicon.com/Whats_new.cfm |archivedate=2010-05-21 }}</ref><ref>{{cite web |url=http://www.pica.army.mil/picatinnypublic/news/archive/2008/12-19-08.asp |title=Picatinny hosts 2008 Type Classification and Materiel Release Awards Ceremony |publisher=Pica.army.mil |date= |accessdate=2010-05-23 |deadurl=yes |archiveurl=https://web.archive.org/web/20100401234102/http://www.pica.army.mil/PicatinnyPublic/news/archive/2008/12-19-08.asp |archivedate=2010-04-01 }}</ref>美國陸軍的M150在2008年12月開始才分類的，雖然美國陸軍在之前一直在使用其它的型號。在2005年10月的評估完成以後，美国海军陆战队計劃擁有115,000套ACOG存貨，讓在海軍陸戰隊以內的每一枝步槍和每一枝卡賓槍的存貨都可以配備一個可使用的ACOG。<ref name="Quantico.usmc.mil">{{cite web |url=http://www.quantico.usmc.mil/Sentry/StoryView.aspx?SID=63 |title=Quantico Sentry - Marines test combat optics curriculum |publisher=Quantico.usmc.mil |date=2005-10-27 |accessdate=2010-05-23 |deadurl=yes |archiveurl=https://web.archive.org/web/20110725012445/http://www.quantico.usmc.mil/Sentry/StoryView.aspx?SID=63 |archivedate=2011-07-25 }}</ref><ref name="Quantico.usmc.mil"/>而根據彭德爾頓的[[新聞|新聞]]報導，表示ACOG將會在美国海军陆战队的“最”前線部隊內發配。<ref>[http://www.cpp.usmc.mil/scout/Articles/dec%2023/cmc.asp CMC: Corps may need fewer augmentees in Iraq next time] {{webarchive|url=https://web.archive.org/web/20110629104010/http://www.cpp.usmc.mil/scout/Articles/dec%2023/cmc.asp |date=2011-06-29 }}</ref>

而ACOG TA01NSN，一個只配備氚夜間照明和後備機械瞄具的4×32瞄準鏡，則是[[特種部隊|特种作战]][[美国特种作战司令部|司令部]]的[[M4卡賓槍|M4A1卡賓槍]]改進型（{{Link-en|SOPMOD|SOPMOD}}）套件的一部分。使用它的特種部隊成員可以按照個人喜好和任務要求以配置他們的武器，包括將其裝上槍上。另外，亦有其他型號的Trijicon ACOG可以發現，應該是個別部隊的士兵自行購入。
