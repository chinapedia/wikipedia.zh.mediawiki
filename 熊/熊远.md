'''熊远'''（{{bd|3世纪||320年代|catIdx=X}}），字'''孝文'''，[[东晋|东晋]]大臣。<ref name=BJ91>[[:s:晋书/卷91|《晋书》卷九十一]]</ref>

== 家世 ==
熊远是[[豫章|豫章]][[南昌|南昌]]人。祖父熊翘曾为[[石崇|石崇]]苍头（仆人），性廉直，有士风。黄门郎[[潘岳|潘岳]]见了称赞他，劝石崇让他自由，于是熊翘回到了乡里。<ref name=BJ91/>

== 西晋年间 ==
熊远有志向，本县召他为功曹，不应，县里强行给他衣帻扶他谒见。十多天后，荐于本郡，于是辟为文学掾。熊远说：“辞大不辞小。”坚请留县。太守举熊远为[[孝廉|孝廉]]，正逢太守讨伐氐羌，熊远没有成行，送他到陇右才回来。后来的太守[[会稽|会稽]]人[[夏静|夏静]]辟他为功曹。夏静去职后，熊远送他到会稽才回来。本州辟熊远主簿、别驾，举[[秀才|秀才]]，除监军[[华轶|华轶]]司马，领[[武昌|武昌]]太守、宁远护军。<ref name=BJ91/>

琅琊王[[司马睿|司马睿]]为[[丞相|丞相]]，[[建兴_(西晋)|建兴]]元年（313年）八月，引熊远为主簿。熊远上书认为眼下律令不行，朝令夕改，不是为政之道，请求上言议论应该引用律令、经传，不得仅出自无依据的人情，以免有违旧典，权宜之计是皇帝所用，不是臣子应该用的。<ref>[[:s:奏请议狱皆准律令|:s:奏请议狱皆准律令]]</ref>当时传言[[晋朝|晋朝]]帝陵[[北陵|北陵]]被发掘，司马睿将要举哀，熊远上疏认为应该先派使者摄河南尹前去证实传言再举哀，再命将领去[[洛阳|洛阳]]修复园陵讨伐叛逆，天下自然响应服从，并且举了[[楚庄王|楚庄王]]伐[[宋国|宋]]为[[申无畏|申无畏]]报仇得到楚军响应、[[项羽|项羽]]杀[[义帝|义帝]]而[[刘邦|刘邦]]哭义帝的事论证大义的重要性。<ref>[[:s:闻北陵被发上疏|:s:闻北陵被发上疏]]</ref>当时正逢[[杜弢|杜弢]]作乱，天下多事，司马睿不能从其言。<ref name=BJ91/><ref>[[:s:晋书/卷30|《晋书》卷三十]]</ref><ref>[[:s:资治通鉴/卷088|《资治通鉴》卷八十八]]</ref>

江东农桑弛废，熊远建议立春时皇帝祈谷于上帝，选择时辰耕种，亲率三公、九卿、诸侯、大夫亲耕帝田以劝农，并举了《[[诗经|诗经]]》“弗躬弗亲，庶人不信”之句。<ref>[[:s:建议劝农桑|:s:建议劝农桑]]</ref>为时议称美。<ref name=BJ91/>

当年十二月，春节要奏乐，熊远向司马睿上谏指出[[晋怀帝|晋怀帝]]梓宫未还，举[[齐桓公|齐桓公]]忧虑中国时诸侯不召而至、骄傲自大后九国叛离的故事，认为设饮食赐给群臣就可以了，<ref>[[:s:怀帝梓宫未返正会不宜作乐议|:s:怀帝梓宫未返正会不宜作乐议]]</ref>司马睿采纳。<ref name=BJ91/>二年（314年）[[太尉|太尉]][[荀组|荀组]]被任为[[尚书令|尚书令]]、[[豫州|豫州]]牧，熊远上疏反对，认为没有宰相出领州牧的，何况荀组素有朝望又熟悉旧典，不应该外放。<ref>[[:s:谏以尚书令荀组领豫州牧启|:s:谏以尚书令荀组领豫州牧启]]</ref>

熊远转丞相参军。琅邪国侍郎[[王鉴_(晋朝)|王鉴]]劝司马睿亲征杜弢，熊远又上疏指出[[商朝|商朝]][[武丁|武丁]]讨伐[[鬼方国|鬼方国]]也用了三年，用兵难不是稀奇事，只有确保有足够的人才钱粮器械才应该亲征；并建议像以前一样派五千人与水军同进讨，如齐用[[穰苴|穰苴]]、秦用[[王翦|王翦]]那样选择有才的督护，就不用担心叛军。<ref>[[:s:上疏谏亲征杜弢|:s:上疏谏亲征杜弢]]</ref>杜弢被平定后，熊远转从事中郎，累迁太子中庶子、尚书左丞、[[散骑常侍|散骑常侍]]。司马睿常叹熊远公忠体国，说：“卿在朝正色，不茹柔吐刚，忠亮至到，可为王臣也。吾所欣赖，卿其勉之！”<ref name=BJ91/>

== 东晋年间 ==
司马睿建立东晋为[[晋元帝|晋元帝]]后，想赐投刺劝进的官吏加位一等，百姓赐为司徒吏，共计二十余万。熊远认为应该按[[汉朝|汉朝]]例，赐天下人爵位，才是广施恩泽不偏颇。元帝不从。<ref name=BJ91/><ref name=ZZTJ90>[[:s:资治通鉴/卷090|《资治通鉴》卷九十]]</ref>

熊远转御史中丞。时尚书[[刁协|刁协]]主政，众人都怕他。尚书郎[[卢綝|卢綝]]入内当值时在大司马门外遇到刁协，刁协醉了要卢綝避让，卢綝不肯，刁协令随从揪住卢綝使其堕马，拖到刁协车前才放开。熊远奏免刁协官。<ref name=BJ91/>元帝姨母广昌乡君去世还没下葬，熊远上表认为应该按古礼大夫死办理，废除冬至小会，<ref>[[:s:广昌乡君丧宜废冬至小会表|:s:广昌乡君丧宜废冬至小会表]]</ref>元帝下诏给[[太常卿|太常卿]][[贺循|贺循]]看熊远表文，贺循答熊远所答合乎古义。<ref>[[:s:晋书/卷20|《晋书》卷二十]]</ref>

[[大兴_(东晋)|大兴]]元年（318年）冬季有雷电、大雨，元帝下诏罪己，又下诏群臣问得失。十一月，熊远又上疏说自己不懂天道，议论人事，以未报先帝之仇、群臣无忧国之心无哀容、选官用人失当为三失，举[[晋武帝|晋武帝]]擢上书人[[郭翼|郭翼]]为屯留令又置谏官的例子。<ref name=BJ91/><ref>[[:s:因灾异上疏|:s:因灾异上疏]]</ref>

熊远累迁[[侍中|侍中]]，约[[大兴_(东晋)|大兴]]四年（321年）出补会稽内史。[[江州|江州]]刺史[[王敦|王敦]]作乱，部下龙骧将军[[沈充|沈充]]举兵响应，加熊远为将军，熊远拒而不受，不给沈充军资，保境安民。王敦到石头城，说服朝廷征召熊远，于是拜熊远太常卿（约[[永昌_(东晋)|永昌]]元年（322年）），加散骑常侍。王敦深深害怕熊远正直有谋，引他为长史。数月后熊远病卒。<ref name=BJ91/>

== 家庭 ==
熊远弟[[熊缙|熊缙]]，名亚于熊远，为王敦主簿，官终[[鄱阳|鄱阳]]太守。<ref name=BJ91/>

== 评价 ==
*《晋书》
**史臣曰：'''熊远'''、王鉴有毗济（[[辛毗|辛毗]]、[[蒋济|蒋济]]）之道，比之大厦，其榱桷之佐乎（是如同大厦的屋椽一样的辅佐者啊）！
**赞曰：'''孝文'''忠謇（忠诚正直），嘉言斯践（美好的言论得到践行）。<ref name=BJ91/>
