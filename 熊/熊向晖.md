'''熊向晖'''{{BD|1919年|4月|2005年|9月9日|catIdx=Xiong熊}}，原名'''熊汇荃'''，[[安徽|安徽]][[鳳陽|鳳陽]]人，生于[[山东省|山东省]]掖县（今[[莱州市|莱州市]]），中华人民共和国外交官、政治人物。

[[清华大学|清华大学]]中文系毕业，[[中国共产党|中国共产党]]情报工作“[[后三杰|后三杰]]”之首（[[申健|申健]]和[[陳忠經|陳忠經]]是另二傑）。出身於官宦家庭，父亲在[[中华民国|中华民国]][[湖北|湖北]]高等法院任庭長。1949年之前潛伏在[[中國国民革命军|中國国民革命军]][[胡宗南|胡宗南]]部进行[[间谍|间谍]]工作，1949年后在[[中华人民共和国|中华人民共和国]]外交界工作。

==生平==
熊家有八個小孩，熊向暉排行第五。1936年9月，熊向暉十七歲考入[[清華大學|清華大學]]中文系。在清華期間，他在[[中华民族解放先锋队|中华民族解放先锋队]]任队委兼秘书，積極參加抗日救亡活動。1936年12月8日，经[[蒋南翔|蒋南翔]]、[[杨学诚|杨学诚]]介绍，秘密加入[[中国共产党|中国共产党]]。

时中共领袖[[周恩来|周恩来]]着眼于情报工作的长期性，委托蒋南翔在清华大学寻找“出身名门望族或官宦之家，年纪较轻，儀表不俗，公开的政治面目不左不右，言谈举止有爱国进步青年的气质，知识面广，记忆力较强，看过一些介绍马列主义基本原理的书籍和孙中山的著作，肯动脑子，比较细心，能随机应变”的人员，潛伏入[[中国国民党|中国国民党]]内部。因此，熊向晖于1937年11月，奉命趕赴[[長沙|長沙]]，报名参加胡宗南的第一軍中由[[李芳兰|李芳兰]]任团长的湖南青年第三戰地服務團。

1938年5月初，胡宗南對熊向暉愛護有加，送他到[[黄埔军校|陸軍軍官學校]]第七分校學習。熊向暉成為黃埔第十五期生，熊向暉代表全體畢業生致辭。從學校學習期滿，熊向暉便被委派為胡宗南的[[機要秘書|機要秘書]]。<ref>{{Cite web |url = http://news.ltn.com.tw/news/opinion/paper/807526 |title = 〈自由廣場〉熊向暉與張顯耀 |author = 陳俊吉 |date = 2014-08-25 |publisher = 自由時報 |language = zh-tw |accessdate = 2015-01-27 |quote =  }}</ref>[[蔣介石|蔣介石]]給胡宗南的密令都由熊向暉簽收，胡宗南下達命令也都由熊向暉起草。但熊向暉將所有內容一概記下，將它偷偷送到[[毛澤東|毛澤東]]、[[周恩來|周恩來]]手中。<ref>{{Cite web |url = http://cpc.people.com.cn/GB/85037/8541897.html |title = 胡宗南委任我当他的机要秘书 |author = 熊向晖　摘《我的情报与外交生涯》 |date =  |publisher = 中國共產黨新聞網 |language = zh-cn |accessdate = 2015-01-27 |quote = 1939年3月，我在军校学习期满，胡宗南指定我在有西安党政军各界领导人参加的毕业典礼上代表毕业生致辞，讲词由我自拟。我讲得激昂慷慨。礼成后，胡找我谈话，表示满意。他说：现在的局面比过去大得多，他需要一个既懂军事又懂政治的助手。他在“黄埔大家庭”里选，选定了我，委派我担任他的侍从副官、机要秘书，原侍从副官唐西园另有任用。我表示不敢从命。他问：为什么？我说：胡先生认识我还不到一年半，不少人知道我曾是清华“民先”队的负责人，有“左”倾嫌疑，在胡先生身边担当这样重要的工作，我不能胜任，也不好办事，别人可能说闲话，对胡先生不利。胡宗南说：你的情况，我完全了解。我一向是疑人不用，用人不疑。我做出的决定，谁敢说不？ }}</ref>

1943年[[抗日战争_(中国)|抗日战争]]期间，蒋介石命令第八战区副司令长官胡宗南以50万军队闪击仅有中共[[359旅|359旅]]驻守的[[延安|延安]]，经熊向晖的洩密，共产党通知驻[[重庆|重庆]]的[[董必武|董必武]]向舆论界洩露了蒋的命令，[[闪击延安计划|闪击延安计划]]未遂。

抗戰勝利後不久，胡宗南又保薦熊向暉去[[美國|美國]]留學。未及成行，1947年3月10日晚上十時，胡宗南在[[洛川|洛川]]「前線指揮所」召集整一軍和整二十九軍的軍、師、旅長開會，具體部署二次闪击延安的方略。熊向暉再度洩密急忙偷偷將情報，派胡宗南的機要通信員送西安[[潘裕然|潘裕然]]，請潘將轉給[[王石堅|王石堅]]。3月19日，根據[[毛澤東|毛澤東]]、[[周恩來|周恩來]]的指示，共產黨中央撤出延安，胡宗南進攻延安時，只剩一座空城。胡宗南到毛泽东住的窑洞里，见桌上压着纸条：“胡宗南到延安，势成骑虎，进又不能进，退又退不得，奈何、奈何！”熊向晖后于1947年5月20日离开胡宗南，9月到美国[[西保留地大学|西保留地大学]]研究院学习。1947年9月，中共數十名諜報人員被捕，<ref>1947年9月28日上海《時代日報》上刊登新聞：“合眾社北平27日電：河北孫連仲部下政治部主任餘興欽與人事組主任謝子延在27日黎明前被此間中央政府當局逮捕。”《周恩來年譜》1947年9月28日載：“從合眾社消息中得知余心清等2人27日在北平被捕後，和任弼時電告楊尚昆、李克農。9月底，和任弼時召集戴鏡元、李質忠、童小鵬、羅青長等開會，研究通訊、機要、情報工作，為確保秘密，決定將現有的機要處分為情報、統戰、軍隊三部分。並研究通訊機要工作中口令編制、使用、保管等問題。”</ref>其中包括王石堅。<ref>熊向暉《地下十二年與周恩來》註釋：“熊文中一關鍵人物即王石堅，熊之一切情報均經王傳送，其重要性可想而知。王石堅後為當局偵破將之拘捕，王乃投降隨國民政府來臺，恢復原名趙耀斌，曾任‘國防部’情報局專門委員，並在臺結婚，現已病故。”（台灣《傳記文學》1991年二月號和三月號）</ref>軍統在調查過程中終於發現熊向暉等三人的真實身份。但此時，熊向晖人已在美國，胡宗南為求自保極力將此事壓下。<ref>沈醉《軍統內幕》：“1947年秋冬间，我曾去西安见过胡，保密局行动处处长叶翔之正在西安搜捕中共地下党组织……发现胡宗南的秘书和他的西北通讯社的负责人当中有中共党员，已经活动了多年……胡对此的确大吃一惊。这个死要面子的人，听说自己的亲信中居然有了共产党，脸都气得发青。他立刻决定将所有涉及他部下的几个人都由他自行处理，要叶翔之不必过问，连向蒋介石报告时也应当把这几个人另外列出来，千万不能让蒋介石知道。”《熊向暉回憶錄》中描述了周恩來的分析：“王石堅的事，是下面保密局系統的人搞的，蔣介石不會馬上知道。從戴笠起，沿下來的鄭介民、毛人鳳，都同胡宗南有很深的關係，他們唇齒相依。胡宗南一定會壓住。王石堅沒有供出你們三人的真實身份，那當然好。其實供不供無關重要。……即便王石堅招供，胡宗南也會讓保密局保他的駕，……我估計胡宗南心裏有數，必然壓著頂著，不敢聲張，還會否認。對這樣的事，我有點準頭。”</ref>1948年9月获[[社会科学|社会科学]][[硕士|硕士]]学位后回国。

1949年11月6日中午，熊向暉在[[中南海|中南海]]勤政殿遇到[[张治中|张治中]]、[[邵力子|邵力子]]、[[刘斐|刘斐]]等中國国民党原和谈代表，[[張治中|張治中]]十分驚訝：“這不是熊老弟麼？你也起義了？”周恩來說：“他不是起義，是歸隊。”張治中說：“我早知道蔣介石在軍事上、政治上都遠遠不是共產黨的對手，今天才知道，在情報上他也遠遠不是共產黨的對手。”毛澤東曾稱讚熊向暉的地下情報工作是中共情報工作最成功、最模範的事例：“一个人能顶几个师”。<ref>《後“三傑”之首熊向暉一人可頂幾個師（下）》，《遼沈晚報》2009年4月20日第7版</ref>

1949年起，熊向暉先後任外交部新聞司副司長、辦公廳副主任。1962年接替[[宦乡|宦乡]]担任駐英代辦直到1967年1月。

1969年，毛泽东、周恩來要求，由[[陈毅|陈毅]]、[[叶剑英|叶剑英]]、[[徐向前|徐向前]]、[[聂荣臻|聂荣臻]]四位元帅一起召開“国际问题研究小组”，從6月7日至7月1日就如何应对严峻的国际局势进行了六次讨论。熊向晖記錄、並起草《对战争形势的初步估计》，得出“在可以预见的时期内，美帝、苏修单独或联合发动大规模侵华战争的可能性都不大”的结论。

1971年，擔任周恩來的助理，參加了中美重大外交活動。1972年，中華人民共和國和[[墨西哥|墨西哥]]建交，熊向暉任[[中国驻墨西哥大使列表|第一任大使]]。<ref>{{cite web|author=中华人民共和国外交部|title=中国驻墨西哥历任大使|url=http://www.mfa.gov.cn/chn/pds/ziliao/wjrw/2167/2174/t9289.htm|publisher=中华人民共和国外交部|accessdate=2012-03-09}}</ref>1973年末，任[[中共中央调查部|中共中央调查部]]副部长，1978年后，兼任[[中共中央统战部|中共中央统战部]]副部长兼[[中国人民外交学会|中国人民外交学会]]副会长。1982年4月，被免去中央调查部副部长。1982年10月，遵照中共中央的决定，接替[[刘宁一|刘宁一]]，担任以[[荣毅仁|荣毅仁]]为董事长的[[中国国际信托投资公司|中国国际信托投资公司]]党组书记并兼副董事长至1987年。1980年代，任第五、七届[[全国政协|全国政协]]常委、第六届全国政协委员，[[欧美同学会|欧美同学会]]副会长、名誉会长。

1991年，熊向暉親撰回憶錄《地下十二年与周恩来》由[[中共中央文献研究室|中央文献研究室]]审定出版，才逐漸為人所知。

2005年9月9日在北京逝世，享年86歲。

==家庭和亲属==
妻子[[谌筱华|谌筱华]]，1947年1月在[[南京|南京]]成婚。
