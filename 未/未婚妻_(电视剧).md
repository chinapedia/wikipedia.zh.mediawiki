{{engname|《未婚妻》|Fiancee}}，导演[[陈慧翎|陈慧翎]]，主演丹尼斯·吴、马思纯、陈彦妃、袁成杰等。该剧根据网络热门言情小说[[蓝小汐|蓝小汐]]的《成全了自己的碧海蓝天》改编而成，2011年12月杀青。

== 剧情 ==
在[[南京市|宁]]工作的蓝小依放弃工作去追求爱情，来[[上海市|沪]]寻找热恋男友田飞，情敌小露爱上了田飞，并且生了他的孩子，田飞向蓝小依许诺2年后再见，2年后，蓝小依通过自己的打拼在上海买了房子，并且在一次面试上认识了未来的男友猴子。一次，蓝小依发现自己怀孕了，结果是田飞的孩子。田飞碰到一次车祸变成了植物人，蓝小依带着女儿看望他并且唤醒了他的记忆，最后，蓝小依和猴子结了婚，而田飞也和小露没有离婚。<ref>{{cite web |url=http://et.21cn.com/gundong/etscroll/2012/05/29/12003815.shtml |title=资料：电视剧《未婚妻》故事梗概 |accessdate=2012 |publisher=21CN |work= |language=zh-cn |deadurl=yes |archiveurl=https://web.archive.org/web/20140215011358/http://et.21cn.com/gundong/etscroll/2012/05/29/12003815.shtml |archivedate=2014年2月15日 |df= }}</ref>

== 花絮 ==
该剧由[[台湾|台湾]][[金钟奖|金钟奖]]导演[[陈慧翎|陈慧翎]]执导。<ref>{{cite web |url=http://ent.sina.com.cn/v/m/2011-11-25/18043491177.shtml |title=都市偶像剧《未婚妻》张华芝升级演外婆 |accessdate=2011 |publisher=新浪网 |work= |language=zh-cn}}</ref>

该剧由杭州南广影视制作有限公司、[[湖南广播电视台|湖南广播电视台]]、湖南广播电视台卫视频道一同打造。<ref>{{cite web |url=http://et.21cn.com/gundong/etscroll/2012/05/29/12004329.shtml |title=资料：电视剧《未婚妻》主创人员 |accessdate=2012 |publisher=21CN |work= |language=zh-cn |deadurl=yes |archiveurl=https://web.archive.org/web/20160304120620/http://et.21cn.com/gundong/etscroll/2012/05/29/12004329.shtml |archivedate=2016年3月4日 |df= }}</ref>

该剧在2011年12月27日成功杀青。<ref>{{cite web|url=http://et.21cn.com/gundong/etscroll/2011/12/27/10272118.shtml|title=《未婚妻》成功杀青 袁成杰戚薇携手做披萨|accessdate=2011|publisher=21CN|work=|language=zh-cn|deadurl=yes|archiveurl=https://web.archive.org/web/20160304185618/http://et.21cn.com/gundong/etscroll/2011/12/27/10272118.shtml|archivedate=2016年3月4日|df=}}</ref>
