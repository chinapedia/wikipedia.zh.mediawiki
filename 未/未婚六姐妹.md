「未婚六姉妹」（日語：{{lang|ja|みこん六姉妹}}）為[[中部日本放送|中部日本放送]]（CBC）製作，於2006年10月2日～11月24日於[[TBSテレビ|TBS]]系「30戲劇」（週一至週五下午1:30〜2:00）播出的帶狀連續劇。全劇共有40集。由[[橋野惠美|橋野惠美]]、[[田中美奈子|田中美奈子]]及[[大和田伸也|大和田伸也]]主演。第2部於2008年3月3日～5月2日播出，全劇共有45集。台灣近期由國興衛視引進播出。

== 劇情概要 ==
[[明治時代|明治時代]]創業日式點心老店「壽屋」的6姊妹都未婚。店主松太郎的妻子16年前過世後，獨自撫育6個女兒長大。除了得照顧店內，還得時時關注女兒們的最終歸宿，搞的松太郎一個頭八個大…

== 登場人物 ==
=== 壽家 ===
* 壽二葉：[[橋野惠美|橋野惠美]]（第1～2部）<ref>[http://hicbc.com/tv/drama30/mikon2/syunsairokka/index.htm 公式サイトの「春彩六華まんじゅう」ページ]</ref><ref>橋野惠美與當時合演的演員綱島剛太郎於2009年1月7日宣布[[結婚|結婚]]。</ref>
: 壽家的二女兒。在「壽屋」的櫃檯幫忙收銀（第2部）。
* 壽一美：[[田中美奈子|田中美奈子]]（第1～2部）
: 壽家的大女兒。受幸子姨的幫助，成為料理店「新一幸」的老闆娘（第2部）。
* 壽三笑：[[廣澤草|廣澤草]]（第1部） → [[太田有美|太田有美]]（第2部）
: 壽家的三女兒。漫畫家（第1部） → 占卜師（第2部）
* 壽四麻子：[[遊井亮子|遊井亮子]]（第1～2部）
: 壽家的四女兒。美容師。
* 壽五真子：[[濱丘麻矢|濱丘麻矢]]（第1～2部）
: 壽家的五女兒。在「壽屋」擔任學徒（第2部）。
* 壽睦：[[加賀美早紀|加賀美早紀]]（第1～2部）
: 壽家最小的女兒。大學生，在出版社打工（第2部）。
* 壽美海（マリン）：-{[[吉田里琴|吉田里琴]]}- （第1～2部）
: 一美的獨生女兒。
* 壽松太郎：[[大和田伸也|大和田伸也]]（第1～2部）
: 六姊妹的父親，「壽屋」的老闆。
* 壽玉緒
: 松太郎已過世的妻子、六姊妹的母親。

: 小睦的男朋友，在兩人正式結婚前，被發現與他人有外遇，小睦斷然宣布婚事中止，兩人正式分手。
