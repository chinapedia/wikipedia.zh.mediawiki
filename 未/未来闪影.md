《'''未来闪影'''》（{{lang-en|'''''FlashForward'''''}}）是一部美国[[电视剧|电视剧]]，由[[布兰农·布拉加|布兰农·布拉加]]和[[大衛·S·高耶|大衛·S·高耶]]搬上电视屏幕，并于2009年9月24日至2010年5月27日期間在[[美國廣播公司|ABC]]上播放第一季。本剧改编自[[加拿大|加拿大]][[科幻小说|科幻小说]]作家[[罗伯特·J·索耶|罗伯特·J·索耶]]的1999年小说《''{{link-en|Flashforward (novel)|Flashforward (novel)|Flashforward}}''》<ref>[http://www.guokr.com/article/66632/ 12年前科幻小说准确预言今年诺奖得主] 果壳网 2011-10-09 18:00 2013-9-24 15:55:06</ref>。

2010年5月，ABC宣布不再预订《未来闪影》的第二季。<ref>{{cite web |url= http://www.denofgeek.com/tv/flashforward/20438/flashforward-cancelled|title= FlashForward cancelled |author = Simon Brew |date = 2010-05-14 |publisher = Den Of Geek |language = en}}</ref>

== 背景 ==
神秘事件导致地球上几乎每一个人同时丧失意识，或者说昏迷2分钟17秒，在这2分钟17秒中，人们看到了他们生活在未来，2010年4月29日晚上8点的影像-也就是全球性的“未来闪影”，同时由Stanford Wedeck率领，Mark Benford，以及他的伙伴Demetri Noh为主要角色的一队[[联邦调查局|联邦调查局]]特工开始调查在在这一昏迷事件中究竟发生了什么，为什么会发生，以及未来是否会再次发生。Benford则在调查中首先提出了他自己独特的預見影像：在他的預見影像中，他看见自己正在观察六个月以来的调查成果，这成为了他和他的团队重建整个调查的线索。

在他们的调查中，他们发现了一连串与預見未来影像相关的事件，包括“零号嫌疑犯”（Simon Campos），在此次事件中仍然保持清醒，邪恶的 "D. Gibbons" （Dyson Frost），以及1991年发生在[[索马里|索马里]]的类似昏迷事件。同时，私人关系也在主角的生活中占据了很大篇幅。Mark Benford看见了已经戒酒的他又开始酗酒，他的妻子，Olivia发现自己在和另一个男人（Lloyd Simcoe）同床，Demetri发现自己没有預見影像，意味着他可能无法活到預見影像发生的那一天。其他的角色也都在奋力和他们預見影像中发生的始料不及的关系做抗争。

== 演員和角色 ==
=== 主要演員 ===
* [[約瑟夫·費因斯|約瑟夫·費因斯]] 飾演 联邦调查局特工Mark Benford（香港配音：[[楊耀泰|楊耀泰]]）
:Mark的預見影像指引了10月6日后对事件的整个调查。作为Olivia的丈夫和Charlie的父亲，他在故事发生前已经戒掉酗酒。但是在他的預見影像中，他看见自己正在板上做些什么，这时有武装并戴着面具的人闯入了他的办公室。并且他当时在喝酒，之后他一直试图对Olivia隐瞒这一点。
* [[約翰·趙|約翰·趙]] 飾演 联邦调查局特工Demetri Noh（香港配音：[[陳健豪|陳健豪]]）
:Mark在联邦调查局的工作伙伴。他没有預見影像，因此他很害怕自己会活不到4月29日，尤其是在接到一个电话，告诉他自己会被谋杀之后。在昏迷发生之前他已经和律师Zoey订婚，她一直坚信她的預見影像是关于他和她在沙滩上的婚礼，但后来她明白过来这很可能是他的葬礼。即便如此，3月15日Mark还是在千钧一发之际救了他。
* [[傑克·達文波特|傑克·達文波特]] 飾演 Llyod Simcoe博士（香港配音：[[何偉誠|何偉誠]]）
:[[斯坦福大学|斯坦福大学]]的学者。已经和他分居的妻子在昏迷中遭遇车祸而去世，他的儿子Dylan患有自闭症，Dylan在預見影像中也看到了Olivia，Charlie也看到了Dylan。他的研究伙伴是Simon Campus，Llyod相信他们应当为昏迷事件负责，但是根据中间人，Simon的叔叔Teddy所说，他们的直线粒子对撞机只是增强了导致預見影像的碰撞效应，使得这一碰撞的能量达到了影响全球的水平。
* [[扎克瑞·奈頓|扎克瑞·奈頓]] 飾演 Bryce Varley医生（香港配音：[[李家傑_(配音員)|李家傑]]）
:Olivia的实习医生，由于被查出发展到第四阶段的[[肾细胞癌|肾细胞癌]]，在昏迷发生之前他险些自杀。在預見影像中他看到了自己和一个手臂上有“信”字样的女孩在一起，因此在昏迷之后他重新产生了生的希望。此外，他十分擅长绘画。在''Course Correction''，也就是第十九集中，他得知自己的癌症已处于康复阶段。
* [[多米尼克·莫纳汉|多米尼克·莫纳汉]] 飾演 Simon Campos博士（香港配音：[[羅偉民|羅偉民]]）
:[[量子物理|量子物理]]学家,在斯坦福大学是Llyod Simcoe的研究伙伴，但他并不相信自己应当为昏迷负责。对别人提到他的預見影像时他总是说看见自己在和一个人搏斗并最后杀了这个人，但这其实是一个谎言。在''Revelation Zero (Part 2)''中观众发现他其实就是零号嫌疑犯，因为Teddy叔叔给了他一个指环（名为"Q.E.D", 即“量子缠绕装置”）使得他在昏迷发生时仍然保持清醒。而Teddy其实是交易的中间人，并和希望控制昏迷发生的人绑架了Simon的妹妹。
* {{link-en|布萊恩·F·奧博恩|Brían F. O'Byrne}} 飾演 Aaron Stark（香港配音：[[李家輝|李家輝]]）
:正在戒酒的酗酒者，Mark在[[戒酒无名会|戒酒无名会]]的保证人以及他的密友。他原本以为女儿Tracy在[[阿富汗|阿富汗]]执行任务的时候被杀，但在他自己的預見影像中，他看见自己和女儿在一起。后来在''The Gift''中，他发现自己的女儿出现在客厅。之后Tracy遭到Jericho这个在阿富汗执行外包保安任务的公司绑架并带回阿富汗拷打。Wedeck作为联邦调查局分局局长为Aaron提供了武器以及其他协助，让他得以前往阿富汗以找到Tracy。
* {{link-en|寇特妮·B·萬斯|Courtney B. Vance}} 飾演 Stanford Wedeck（香港配音：[[陳德_(配音員)|陳德]]）
:联邦调查局洛杉矶分局局长。他领导Mark的团队以及对整个昏迷事件的调查。他在华盛顿的人脉十分丰富，并在调查中多次起到关键作用。在他的預見影像中，他正在卫生间中读一张报纸。
* {{link-en|鬆雅·瓦格|Sonya Walger}} 飾演 Olivia Benford医生（香港配音：[[鄧潔麗|鄧潔麗]]）
:Mark的妻子，职业生涯十分成功的外科医生。她是Bryce的上司。在她的預見影像中，她正在家中和Llyod博士发生暧昧关系，虽然她之前从未遇到过他。但是他们最后还是在她治疗Dylan的时候相遇了。
* {{link-en|克莉斯汀·伍茲|Christine Woods}} 飾演 联邦调查局特工Janis Hawk（香港配音：[[王慧珠|王慧珠]]）
:在联邦调查局洛杉矶分局作为Mark团队的医院。在她的預見影像中她怀孕了并正在做[[超声波|超声波]]检查，这对她来说实在是太难以置信了，因为她仍然单身而且是一个女同性恋者。后来她负责监管和照看Simon。在''Queen Sacrifice''一集中，她间谍的身份被揭开。但是在''Goodbye Yellow Brick Road''一集中，她实际是在特工Vogel的要求下卧底在联邦调查局作为[[中央情报局|中央情报局]]的双重间谍

== 制作 ==
第一集，也就是试播集是由David S. Goyer（另外他也导演了本集）和Brannon Braga所撰写的。剧本由[[罗伯特·J·索耶|罗伯特·J·索耶]]同名小说改编，Goyer和Braga作为[[执行制片人|执行制片人]]，另外还有Jessica Boesiczky Goyer, Vince Gerardis以及Ralph Vicinanza.<ref>{{cite web| url=http://www.comingsoon.net/news/tvnews.php?id=49023|title=Flash Forward New Companion for Lost| publisher=www.comingsoon.net| accessdate=2008-08-09}}</ref> 

''未来闪影''原本计划在[[HBO|HBO]]播出，但后来它认为广播网络会更适合这部电视剧，因此卖掉了播出权。2009年5月[[美国广播公司|ABC]]安排了试映之后预定了十三集。<ref>{{cite web|url=http://www.hollywoodreporter.com/hr/content_display/news/e3i4b5caa365ad73b3a562218379433fee4 |title=The Hollywood Reporter |publisher=The Hollywood Reporter |date=2009-05-08 |accessdate=2009-09-25}}</ref>2009年10月12日，[[美国广播公司|ABC]]将第一季的集数增加到22集。当日晚些时候又宣布会追加三集的订阅，增加到25集，继而又调整到了24集，而实际播出仍然是22集。<ref name="24 episodes">{{cite web|url=http://www.thefutoncritic.com/news.aspx?id=8416 |title=ABC Confirms Episode Orders |publisher=The Futon Critic |date=2009-12-08|accessdate=2009-12-04}}</ref><ref>{{cite web |url=http://www.variety.com/article/VR1118009832.html?categoryId=14&cs=1 |title=ABC picks up more 'FlashForward' |publisher=''[[綜藝|綜藝]]'' |first=Stuart |last=Levine |date=2009-10-12 |accessdate=2009-10-14}}</ref><ref>{{cite web |url=http://ausiellofiles.ew.com/2009/10/12/this-just-in-flash-forward-gets-full-season/ |title=This just in: 'FlashForward' gets full season |publisher=''[[娱乐周刊|娱乐周刊]]'' |author=[[Michael_Ausiello|Michael Ausiello]] |work=The Ausiello Files |date=2009-10-12 |accessdate=2009-10-12}}</ref><ref>{{cite web |url=http://artsbeat.blogs.nytimes.com/2009/10/12/abc-orders-full-season-of-flashforward/ |title=ABC Orders Full Season of 'FlashForward' |publisher=''[[纽约时报|纽约时报]]'' |work=ArtsBeat: The Culture at Large |last=Itzkoff |first=Dave |date=2009-10-12 |accessdate=2009-10-12}}</ref>

未来闪影每一集开头显示''FlashForward''标志的时候都会在字母间包括一张隐蔽的图片，大约表示出这一集的主要内容。在DVD/下载版上，可以通过暂停来看清这些图片。

2009年10月21日，执行制片人[[:en:Marc_Guggenheim|Marc Guggenheim]]宣布将会离开''未来闪影''剧组。联合制作人和另一位执行制片人David S. Goyer则代替他成为本片的负责人。<ref>{{cite news | author=Joyce Eng | title=FlashForward Changes Showrunners| url=http://www.tvguide.com/News/FlashForward-Changes-Showrunners-1011084.aspx| work=TVGuide.com}}</ref>2010年2月5日Goyer宣布将不再担任负责人以便将精力花在重点的电影导演上。不过他仍然会参与这部电视剧。Goyer的位置由他的妻子[[:en:Jessika_Goyer|Jessika]]替代，此外还有[[:en:Lisa_Zewrling|Lisa Zewrling]]以及[[:en:Timothy_J._Lea|Timothy J. Lea]].

2010年5月13日有报道称''未来闪影''未能获得下一季的预订。<ref>{{cite web|url=http://ausiellofiles.ew.com/2010/05/13/abc-cancels-romantically-challenged-scrubs-ted-flashforward |title=THIS JUST IN: ABC Cancels "Romantically Challenged", "Scrubs", "Better Off Ted" and "FlashForward"] |work=[[娱乐周刊|娱乐周刊]] |date=2010-05-13 |deadurl=yes |archiveurl=https://web.archive.org/web/20100515045621/http://ausiellofiles.ew.com/2010/05/13/abc-cancels-romantically-challenged-scrubs-ted-flashforward/ |archivedate=2010-05-15 }}</ref> ABC第二天确认了这个消息。<ref>{{cite web |url=http://www.denofgeek.com/television/484717/flashforward_cancelled.html |title=FlashForward cancelled |work=Don of Geek |date=2010-05-14}}</ref><ref>{{cite web |url=http://www.sevensidedcube.net/entertainment/2010/heroes-and-flashforward-cancelled/ |title=“Heroes” and “FlashForward” Cancelled |work=Seven Sided Cube |date=2010-05-15}}</ref> David S. Soyer在接受在[[时代周刊|时代周刊]]的采访中他说自己从未预料到这一结局

传言称[[Starz|Starz]]电视台会考虑购买未来闪影的版权并有可能更新第二季。<ref>{{cite web|url=http://www.tvsquad.com/2010/07/23/is-flashforward-heading-to-starz/ |title=Is 'FlashForward' Heading to Starz? |work=TV Squad |date=2010-07-23}}</ref><ref>{{cite web |url=http://scifimafia.com/2010/07/flashforward-might-be-saved-by-starz/ |title=存档副本 |accessdate=2010-09-06 |deadurl=yes |archiveurl=https://web.archive.org/web/20100807001720/http://scifimafia.com/2010/07/flashforward-might-be-saved-by-starz/ |archivedate=2010-08-07 }}</ref>

=== 收视率 ===
尽管未来闪影的第一集在美国取得了收视人数1247万的好成绩，第一季结束时的收视率仍然迅速降到了三分之一的水平。<ref>{{cite web |url=http://tvbythenumbers.com/2010/03/28/flashforwards-ratings-blame-the-show-not-the-scheduling/46408 |title=Updated: "FlashForward's"; Ratings: Blame the Show, Not the Scheduling |last1=Seidman |first1=Robert |date=March 28, 2010 |work=tvbythenumbers.com |publisher=TVbytheNumbers |accessdate=June 5, 2010}}</ref> 未来闪影在美国的播放时间由于太晚而受到一些人的诟病。但在其他国家，例如英国，即便是黄金时间播放，收视率仍然大大下降。<ref>{{cite web|url=http://www.digitalspy.co.uk/tv/s56/flashforward/news/a210358/flashforward-makes-poor-uk-return.html|title=FlashForward makes poor ratings return|date=March 24, 2010|work=[[Digital_Spy|Digital Spy]]|accessdate=May 5, 2010}}</ref>
