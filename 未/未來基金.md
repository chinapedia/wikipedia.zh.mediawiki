'''未來基金'''（[[英文|英文]]：'''Future Fund'''）由前任[[財政司司長|財政司司長]][[曾俊華|曾俊華]]於2015年2月25日透過《[[2015年至2016年度香港政府財政預算案|2015年至2016年度香港政府財政預算案]]》宣佈將會設立，作為長線[[投資|投資]]用途，用以爭取高回報的[[利益|利益]]，並且增強[[香港政府|香港政府]]的[[財政|財政]]的[[可持續性|可持續性]]。當香港政府財政儲備降至不足半年的公共開支時，就會啟動[[預警|預警]]，香港政府需要就此進行[[債務|債務]][[融資|融資]]或者[[資產證券化|資產證券化]]，最後才考慮動用未來基金，由[[立法會|立法會]]批准，甚至中斷未來基金為期至少10年的存放期，以解公共支出的燃眉之急。12月18日，政府宣佈未來基金將於2016年1月1日設立。

為了避免引起外界不必要的憂慮，誤以為香港政府的財政實力及[[宏觀經濟|宏觀經濟]]的基礎轉弱，而影響香港的[[信貸評級|信貸評級]]，因此建議中的未來基金將仍然被列為香港政府財政儲備的一部分。

== 資金來源 ==
[[長遠財政計畫工作小組|長遠財政計畫工作小組]]建議將2,197億港元[[土地基金|土地基金]]結餘撥予未來基金，作為其首筆資金，往後每年則將25%至33%（因應每年財政狀況及社會需要而可以再行決定。）[[香港政府|香港政府]]財政盈餘注資入未來基金。

== 投資安排 ==
[[長遠財政計畫工作小組|長遠財政計畫工作小組]]建議由[[金融管理局|金融管理局]]或者聘請[[投資管理|投資管理]]經理，將未來基金的一半以上用作比較高風險及比較高回報的長期增長組合，例如[[私募股權|私募股權]]和[[房地產|房地產]]，另外一半則用為[[債券|債券]]、[[股票|股票]]或及[[保本基金|保本基金]]投資組合，為期至少10年。長遠財政計畫工作小組同時建議，未來基金應該繼續存放在外匯基金下，以受惠於外匯基金既具規模的投資基礎設施及相關的專門知識，此安排既可以降低成本，亦可以平衡投資的風險與回報，並且增強外匯基金的財政資源，有助於維持香港金融及[[香港貨幣|貨幣]]的穩定。而金融管理局日後每年需要至少一次就未來基金的投資策略諮詢[[財政司司長|財政司司長]]和[[財經事務及庫務局局長|財經事務及庫務局局長]]<ref>[http://www.hket.com/eti/article/0cd84603-3d75-4c3a-ba61-69daf36cfbe8-011772?section=001 年撥至少1/4盈餘 注未來基金] {{Wayback|url=http://www.hket.com/eti/article/0cd84603-3d75-4c3a-ba61-69daf36cfbe8-011772?section=001 |date=20150405050350 }} 《經濟日報》 2015年3月3日</ref>。

== 歷史 ==
2014年3月，[[長遠財政計畫工作小組|長遠財政計畫工作小組]]公布第一階段報告，闡述[[香港人口|香港人口]][[人口老化|老化]]及[[經濟增長|經濟增長]]逐步放緩將為香港公共財政帶來沉重壓力，因而建議必須及早採取果斷行動，應付挑戰，而避免未來香港政府財政儲備出現結構性赤字。<ref>[http://www.nanzao.com/tc/opinion/14c3156c706302d/monitor-wei-lai-ji-jin-jian-yi-bu-qie-shi-ji Monitor: 「未來基金」建議不切實際]</ref>

2015年2月25日，時任[[財政司司長|財政司司長]][[曾俊華|曾俊華]]透過《[[2015年至2016年度香港政府財政預算案|2015年至2016年度香港政府財政預算案]]》宣佈設立未來基金，作為長線投資用途，用以爭取高回報的利益。當[[香港政府|香港政府]]財政儲備降至不足半年的公共開支時，才可以啟動提款作為應急用途。他又表示，已經要求[[財經事務及庫務局局長|財經事務及庫務局局長]][[陳家強|陳家強]]和[[香港金融管理局總裁|香港金融管理局總裁]]陳德霖商量訂定未來基金的具體管理及投資機制，期望於同年落實此儲蓄計劃。翌月2日，長遠財政計畫工作小組公布於未來成立未來基金的第二階段報告，身兼[[財經事務及庫務局|財經事務及庫務局]]常任秘書長（庫務）的長遠財政計畫工作小組主席[[謝曼怡|謝曼怡]]表示，香港[[人口老化|人口老化]]加劇，公共醫療及安老服務都需要增加開支，縱然香港政府財政儲備於未來數年仍然會錄得盈餘，不過按照現有政策及開支增長，以及[[香港經濟|香港]]經濟增長逐步放緩的趨勢，預計於10年內將會浮現結構性赤字，故此需要未雨綢繆，建議採用最具效率及最符合成本效益的行政途徑，盡快設立未來基金，作為高回報投資，藉此減輕香港政府的財政壓力<ref>[http://www.info.gov.hk/gia/general/201503/02/P201503020796.htm 長遠財政計劃工作小組發表第二階段報告（附圖／短片）] 《新聞公報》 2015年3月2日</ref>。長遠財政計畫工作小亦組表示，決定以行政途徑成立未來基金，在香港政府帳目中開設一個名義帳目，而不用立法會通過；提款安排就沿用土地基金決議案和《公共財政條例》，由立法會批准<ref>[http://www.news.gov.hk/tc/categories/finance/html/2015/03/20150302_140142.shtml?pickList=topstories 未來基金增強財政可持續性] 政府新聞網 2015年3月2日</ref><ref>[http://www.singpao.com/xw/yw/201503/t20150303_550079.html 未來基金今年啟動 2200億打底 財政盈餘注資] {{webarchive|url=https://web.archive.org/web/20150709021226/http://www.singpao.com/xw/yw/201503/t20150303_550079.html |date=2015-07-09 }} 《成報》 2015年3月3日</ref>。

== 評價 ==
[[香港中文大學|香港中文大學]]經濟學系教授宋恩榮指出，[[澳大利亞|澳大利亞]]及[[新西蘭|新西蘭]]等國家均早就成立類似於「未來基金」的儲蓄計劃，而且具有相當成效。他解釋，未來基金的首筆資金來源為土地基金，而土地基金乃由[[香港金融管理局|香港金融管理局]]所管理，現時被併入[[外匯基金|外匯基金]]內。而外匯基金的作用為支持[[香港聯繫匯率制度|香港聯繫匯率制度]]，故此其流動性相當重要，投資方面亦相對地保守，回報率亦因而比較低。目前[[港元|港元]][[貶值|貶值]]的壓力相當低，外匯基金的儲蓄亦相當充裕，故此設立未來基金可以作為一個長線投資，從而賺取比較高回報的利益<ref>[http://www.singpao.com/xw/yw/201502/t20150226_549636.html 「未來基金」方案 下周一公布] {{webarchive|url=https://web.archive.org/web/20150402110804/http://www.singpao.com/xw/yw/201502/t20150226_549636.html |date=2015-04-02 }} 《成報》 2015年2月26日</ref>。

[[香港中文大學|香港中文大學]]全球經濟及金融研究所研究教授廖柏偉表示，香港政府財政盈餘依靠土地收益，沒有長期保證，香港面社會對人口老化，終有一日入不敷支。他以同樣面臨人口老化的[[日本|日本]]為例子，表示日本於1990年因為財政赤字開始而發行[[國債|國債]]，國債至2015年佔[[國民生產總值|國民生產總值]]250%，20年間，其赤字累積佔國民生產總值180%。

[[羅兵咸永道會計師事務所|羅兵咸永道會計師事務所]]高級顧問王銳強指出，日本每年支付還債利息相當於收入25%；香港政府現時仍有盈餘，不過於10年後出現結構性財政赤字亦並非不可能。[[畢馬威會計師事務所|畢馬威會計師事務所]]前合夥人王尹巧儀認為，香港需要居安思危，不希望步日本後塵<ref>[https://hk.news.yahoo.com/%E6%9C%AA%E4%BE%86%E5%9F%BA%E9%87%91%E5%84%B2%E6%87%89%E6%80%A5%E9%8C%A2-%E5%8D%8A%E4%BD%9C%E9%AB%98%E9%A2%A8%E9%9A%AA%E6%8A%95%E8%B3%87-215621601.html 未來基金儲應急錢 半作高風險投資] 《星島日報》 2015年3月3日</ref>。
