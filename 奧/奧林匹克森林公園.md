{{北京中轴线}}
'''奥林匹克森林公园'''，位于[[北京市|北京市]][[朝阳区_(北京市)|朝阳区]]，地处[[奥林匹克公园_(北京)|奥林匹克公园]]北部，占地680公顷，建成时是北京规划建设的最大的城市公园<ref>{{cite news |url = http://news.xinhuanet.com/environment/2009-08/05/content_11832030.htm |title = 北京奥林匹克森林公园国庆节将免费对外开放 |date = 2009-08-05 |accessdate = 2012-04-28 }}</ref>，是[[2008年夏季奥林匹克运动会|2008年奥运会]]配套工程之一，但在奥运会和残奥会期间并未开放，而是在残奥会结束的2008年10月开放了南园，2009年开放了北园。作为[[北京中轴线|北京城市中轴线]]的北端终点<ref name="mlbjzzx">{{cite book |author = 李建平 |authorlink = 李建平 |title = 魅力北京中轴线 |publisher = 文化艺术出版社 |year = 2008年 |ISBN = 978-7-5039-3519-0}}</ref>，森林公园提出“通向自然的轴线”的概念，轴线逐渐融入自然，并消失在森林中。[[北京五环路|北五环路]]穿公园而过，将公园分为南北两园，南园以人工景观、休闲娱乐为主，北园则以生态园林、自然野趣为主，两园通过中轴线上跨越五环路的一条生态廊道相连。

== 历史 ==
{{See also|奥林匹克公园 (北京)}}
[[File:Rockery_in_the_entrance_of_Olympic_Forest_Park.jpg|奥林匹克森林公园入口处的假山]]
北京获得[[2008年夏季奥林匹克运动会|2008年奥运会]]主办权后，在2002年3月至7月间，[[北京市规划委员会|北京市规划委员会]]面向全球对[[奥林匹克公园_(北京)|奥林匹克公园]]的规划设计方案进行了国际征集，美国[[佐佐木建筑师事务所|佐佐木建筑师事务所]]<ref>{{cite book |editor = 安纪国际出版有限公司 |location = 昆明 |publisher = 云南科技出版社 |title = 佐佐木建筑师事务所 |ISBN = 7-5216-2074-2 |pages = 6-13 |date = 2004年10月 }}</ref>和中国天津华汇工程建筑设计有限公司的联合设计方案获得一等奖。这个方案中，沿着中轴线建立了中华文明五千年纪念大道，向北消失与森林公园中，而森林公园的规划为：以[[北京五环路|五环]]为界，南园以水为主，设计大面积的“龙湖”作为龙形水系的“龙头”，隐喻[[东海|东海]]，并在中央设置一岛寓意[[蓬莱仙岛|蓬莱仙岛]]；北园以山为主，主峰高50至60米，寓意[[昆仑山脉|昆仑山脉]]；南北园间设置3座绿色廊道。同年11月的奥林匹克森林公园和中心区景观设计方案征集中，佐佐木建筑师事务所和[[北京清华同衡规划设计研究院有限公司|北京清华城市规划设计研究院]]的联合方案中标，这个设计延续了前述设计的主要特征<ref name="whlypart1" />。

随后，由清华城市规划设计研究院在联合方案的基础上，对方案进行深化，并做出了最终方案，于2005年10月27日由[[中共北京市委|中共北京市委]]常委原则性批准。方案的主要改动包括：取消了五千年纪念大道的设置，同时将森林公园和奥林匹克公园中心区分开，分别交由清华城市规划设计研究院和[[北京市建筑设计研究院|北京市建筑设计研究院]]设计；主山由五环北侧移到南侧，相应的压缩南园主湖面积，主山借原有地名命名“仰山”，主湖借“奥林匹克”的“奥”字命名“奥海”。整个园区的中轴线设计为“通向自然的轴线”，从城市逐步走入自然，最终消失于森林公园中<ref name="whlypart1">{{cite book |title = 五环绿苑——奥林匹克公园 |editor = 北京清华城市规划设计研究院本卷主编 |location = 北京 |publisher = [[中国建筑工业出版社|中国建筑工业出版社]] |year = 2009年 |ISBN = 978-7-112-09878-1 |pages = 1–12 }}</ref>。

森林公园南区占地380公顷，定位为“生态森林公园”，以人工景区为主，主要提供市民休闲娱乐的场所；而北区占地300公顷，定位更靠近自然野趣，主打[[生态|生态]]牌，通过减少设施和游人数目来创造适宜动植物生长繁衍的环境，是源生树种种源基地和鸟类保护区。南北两园间通过一个生态廊道相连<ref>{{cite news |url = http://travel.people.com.cn/GB/9795202.html |title = 奥林匹克森林公园北区十一免费开放 主打生态牌 |date = 2009-08-05 |accessdate = 2012-04-28 }}</ref><ref name="whlypart1" />。

森林公园于2005年6月30日开工<ref>{{cite news |url = http://news.xinhuanet.com/sports/2005-06/30/content_3158773.htm |title = 奥林匹克森林公园开工建设 预计2007年底竣工 |date = 2005-06-30 |accessdate = 2012-04-28 }}</ref>，2008年7月3日落成，但在奥运期间未曾开放，直到奥运结束的10月26日才向游人开放南园<ref>{{Cite web |url = http://news.sina.com.cn/c/2008-10-30/032616552367.shtml |title = 北京奥林匹克森林公园南园免费开放 |date = 2008-10-30 |accessdate = 2010-08-24 }}</ref>。而北园至2009年10月才向游客开放<ref>{{cite web |url = http://news.sohu.com/20090806/n265747011.shtml |title = 奥林匹克森林公园北园十一起免费游 以树木为主 |date = 2009-08-06 |accessdate = 2010-08-24 }}</ref>。

== 水系 ==
[[File:Olympic_Forest_Park2.jpg|thumb]]
[[File:Olympic_Forest_Park3.jpg|thumb]]
森林公园水面面积为67.7公顷，其中主湖20.3公顷，各处[[湿地|湿地]]5.71公顷，市政河道25公顷（包括清河导流渠和仰山大沟等），其他水面16.69公顷。清河导流渠和仰山大沟承担为[[北小河|北小河]]提供水源的任务。南园利用原有洼里公园、碧玉公园水域，挖湖堆山形成主湖<ref name="whlygardening" />，作为龙形水系的头部；北园水系则引自清河导流渠，蜿蜒曲折，营造山林野趣<ref name="whlywater" />。

水循环方面，非汛期，北园通过市政[[中水|中水]]补水，南园通过泵站实现水循环，利用中水和暗管补充蒸发量；汛期，清河导流渠和仰山大沟用于奥林匹克中心区排洪功能，水可以得到及时补充但水质不能保证，为此只能采用局部小循环的模式。园区的湿地可以为园区水循环提供净水能力，净化[[清河_(北京)|清河]]上游来的经污水处理的再生水。当发生特大暴雨时，园区通过清河和北小河两路退水<ref name="whlywater">{{cite book |title = 五环绿苑——奥林匹克公园 |editor = 北京清华城市规划设计研究院本卷主编 |location = 北京 |publisher = [[中国建筑工业出版社|中国建筑工业出版社]] |year = 2009 |ISBN = 978-7-112-09878-1 |pages = 84–101 }}</ref>。

== 植被设计 ==
设计师基于对北京本底[[群落|生物群落]]、公园场地本底的调查，确定设计原则。整个设计以[[生态学|生态学]]为指导，保持[[生物多样性|生物多样性]]，同时体现“通向自然的轴线”的理念，南园以休闲娱乐为主，北园以生态保护和恢复功能为主。乔木配植主要采取混交林的形式，包括乔木混交林和乔灌混交林，以达到四季景色不同的特点<ref name="whlygardening">{{cite book |title = 五环绿苑——奥林匹克公园 |editor = 北京清华城市规划设计研究院本卷主编 |location = 北京 |publisher = [[中国建筑工业出版社|中国建筑工业出版社]] |year = 2009 |ISBN = 978-7-112-09878-1 |pages = 102–124 }}</ref>。

== 文物古迹 ==
奥林匹克森林公园内，有多处文物古迹，包括龙王庙一座，还有现存的14通螭首龟趺石碑以及2组（4根）墓表。
* [[龙王庙_(奥林匹克森林公园)|龙王庙]]：位于奥林匹克森林公园南园西南部，[[科荟路|科荟路]]北侧。
* [[卫武家族墓|卫武家族墓]]：位于龙王庙以东，[[科荟路|科荟路]]北侧。现存3通石碑，1组（2根）墓表。
* [[图海家族墓|图海家族墓]]：位于龙王庙以北。现存5通石碑。
* [[海望家族墓|海望家族墓]]：位于卫武家族墓以北，现存4通石碑。
* [[吴努春墓|吴努春墓]]：位于海望家族墓以东，现存1通石碑。
* [[兆惠墓|兆惠墓]]：位于奥林匹克森林公园[[奥林匹克森林公园曲棍球场|曲棍球场]]。现存1通石碑，1组（2根）墓表。

此外，兆惠墓东北原有[[乾隆帝|乾隆帝]]第九女[[和硕和恪公主|和硕和恪公主]]与[[兆惠|兆惠]]之子[[札兰泰|札兰泰]]合葬墓，和硕和恪公主于乾隆三十七年（1772年）下嫁札兰泰。乾隆四十五年（1780年）公主病逝。乾隆五十三年（1788年）札兰泰病逝，与公主合葬此处。1930年代，该墓被盗毁。

== 交通 ==
参照[[北京植物园|北京植物园]]和[[天坛公园|天坛公园]]所做的客流预测显示，南园年游客接待量在450万人次，北园在80万人次；南园高峰日接待量3.4万人次，北园6600人次。对外交通方面，[[北京地铁8号线|地铁8号线]][[森林公园南门站|森林公园南门站]]预测可以承担每日2万人次的客流量；地面公交系统则预计可以承担6000人次，这样公共交通系统可以提供高峰日75%的游客量，游客自备车需满足25%的游客量，因此公园内建设了3000个停车位。公园内部道路也按高峰日游客不会感到不适、极端日不能保证全程舒适但不会造成拥挤进行设计<ref name="whlytransport" />。

公园四边道路分别是：东侧[[安立路|安立路]]、南侧[[科荟路|科荟路]]、西侧[[林萃路|林萃路]]、北侧跨[[清河|清河]]北侧的[[清河北路|清河北路]]、东北角的[[洼里三路|洼里三路]]。其中科荟路中段封闭改造为花坛<ref>{{cite news |url = http://bjyouth.ynet.com/article.jsp?oid=46265897 |title = 科荟路封闭“奥园”南北将连通 |date = 2008-11-28 |accessdate = 2012-04-28 |deadurl = yes |archiveurl = https://archive.is/20120707032415/http://bjyouth.ynet.com/article.jsp?oid=46265897 |archivedate = 2012-07-07 }}</ref><ref>{{cite news |title = 巩汉林炮轰北京科荟路封闭 呼吁重大决策听证 |url = http://news.qq.com/a/20100310/003916.htm |date = 2010-03-10 |accessdate = 2012-04-28 }}</ref>，远期可能实施为地下隧道<ref>{{cite news |url = http://news.163.com/09/1018/04/5LSMODSG000120GR.html |title = 科荟路中段变身绿地花坛 |date = 2009-10-18 |accessdate = 2012-04-28 }}</ref>。[[北京五环路|北五环]]东西向穿公园而过；[[奥林东路|奥林东路]]、[[奥林西路|奥林西路]]分别从东部和西部南北向穿过。公园在周边道路和五环路上规划11个游览出入口。其中南门为主入口，此外在北五环设置有2个一级入口分别通向南园和北园，供游客自备车使用<ref name="whlytransport" />。

公园主要道路全部和城市道路分离，一级园路路宽8米，二级园路路宽4.5至6米，三级园路路宽1.5至3米。沿主干道设置[[电瓶车|电瓶车]]游览路线，在主要景点和路口设置站点<ref name="whlytransport">{{cite book |title = 五环绿苑——奥林匹克公园 |editor = 北京清华城市规划设计研究院本卷主编 |location = 北京 |publisher = [[中国建筑工业出版社|中国建筑工业出版社]] |year = 2009 |ISBN = 978-7-112-09878-1 |pages = 125–134 }}</ref>。
