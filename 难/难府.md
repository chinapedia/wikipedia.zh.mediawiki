'''難府'''<ref name="zhou">{{cite book|title=[[世界地名翻译大辞典|世界地名翻译大辞典]]|author=周定国|publisher=中国对外翻译出版公司|year=2008|isbn=978-7-5001-0753-8|page=1234}}</ref>（{{thai|t=จังหวัดน่าน|r=Changwat Nan}}，{{IPA-th|t͡ɕāŋ.wàt nâːn}}），一譯'''楠府'''，是[[泰國北部|泰國北部]]的一個[[泰國的府|府]]。鄰近府分別為（由南方起順時針方向）[[程逸府|程逸府]]、[[帕府|帕府]]和[[帕堯府|帕堯府]]。東北方則與[[寮國|寮國]][[赛宋奔省|赛宋奔省]]交界。

== 地理環境 ==
難府位於[[難河|難河]]偏遠河谷，周圍是群山森林。最高山脈''Phu Khe''高2079公尺，位於東北方，接近寮國邊境。

== 歷史 ==
難府長久以來因為地理位置偏遠，與其他王國少有往來，故始終保持獨立。13世紀末期，第一個王朝建立於Mueang Pua（又名Varanagara）附近。統治者是Phukha王朝，與[[永珍|永珍]]建國者有親戚關係。然而因為往南發展比往東或往西容易一點，所以他開始和[[素可泰王朝|素可泰王朝]]來往。在[[十四世紀|十四世紀]]他遷都到今天的[[難府城|難府城]]。

到[[十五世紀|十五世紀]]，素可泰王朝日漸衰落，難地則對[[蘭納|蘭納國]]稱臣。1443年[[孔濤國王|孔濤國王]]（''King Kaen Thao''）圖謀佔據鄰國帕堯的土地，他請求蘭納國王[[提羅嘉叻|提羅嘉叻]]（''Tilokaraj''）幫助他抵禦[[越南|越南]]的入侵，即便越南並無作此威脅。他趁機殺掉了帕堯國王，但是隨即提羅嘉叻王不滿而興兵攻打難國，並於1449年將她消滅。

在[[蘭納|蘭納]]被[[緬甸|緬甸]]統治期間，難地在人民多次起義，試圖獲得自立，但未能成功。1714年正式納入緬甸的版圖下，直到1783年緬甸被暹羅擊敗，難地選擇歸附暹羅。但是1893年[[河口事件|河口事件]]發生之後，暹羅被迫將東難大部交予[[法屬印度支那|法屬印度支那]]管轄。 

然而難地對[[暹羅|暹羅王朝]]保持一定程度上的獨立，直到1931年才完全被納入泰王國的版圖並成為一個府份。 

[[二十世紀|二十世紀]][[八十年代|八十年代]]以前，本地匪徒橫行，所謂[[泰國人民解放軍|泰國人民解放軍]]（''People's Liberation Army of Thailand''──''PLAT''）也經常對社會安全造成威脅，公路時常被游擊隊在一夜之間破壞，今天局勢雖然趨向緩和，但是建設速度依然緩慢，鄉間比較荒蕪。

== 人口 ==
約10.5%人口是[[山地民族|山地民族]]。

== 行政及管理 ==

本府行政上被劃分為15縣（''ampoe''）<!--和1支縣（''king ampoe''）-->，又進一步被劃分為99區（''tambon''）及848村（''muban''）。
