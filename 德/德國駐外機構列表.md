'''德國駐外機構列表'''列出[[德意志聯邦共和國|德意志聯邦共和國]]在世界各地設置的[[外交代表機構|外交代表機構]]，由[[大使館|大使館]]（{{lang|de|Botschaft}}）、[[總領事館|總領事館]]（{{lang|de|Generalkonsulate}}）、[[領事館|領事館]]（{{lang|de|Konsulate}}）和常駐代表處（{{lang|de|Ständige Vertretung}}）組成。這些駐外機構接受[[德國外交部|德國外交部]]管理，負責維護德國在外利益、服務海外{{Link-en|德國國籍法|German nationality law|德國公民}}，也與地主國（駐在國）政府交流，以促進德國與外國在政治、經濟、文化及科學領域的關係。德國駐外館處的其他任務還包括資訊蒐集、向德國聯邦政府各部門匯報相關議題、協助德國企業在外活動。與使館駐在國人民相關的業務，則有受理、核發其赴德[[簽證|簽證]]、向其宣傳德國概況及對外政策等<ref name="enmission">{{lang|en|{{Cite web |title=The German Missions Abroad |url=http://www.auswaertiges-amt.de/EN/AAmt/Auslandsvertretungen/Uebersicht_node.html |work=[[德國外交部|Federal Foreign Office]] |accessdate=2014-11-02}}}}{{en}}</ref><ref name="demission">{{lang|de|{{Cite web |title=Auslandsvertretungen |url=http://www.auswaertiges-amt.de/DE/AAmt/Auslandsvertretungen/Uebersicht_node.html |work=[[德國外交部|Auswärtiges Amt]] |date=2014-01-14 |accessdate=2014-11-02}}}}{{de}}</ref>。

德國大使館的館務涵蓋了{{Link-en|雙邊關係|Bilateral relations}}中的各個領域，除了政經文、外交、德商支援和公民服務外，還會參與國際研究與技術合作活動，或是社會發展及農業政策，因此館內常短期地附派有德國政府其他部會的外派官員或專家。在與德國有進行防務合作、或是攸關德國國防政策的國家，當地的德國大使館中亦會駐有[[德國聯邦國防軍|德軍]][[武官|武官]]。[[德国驻外大使列表|大使]]通常為大使館位階最高的人員，由於德國並非在每個邦交國都設有大使館，所以該館的大使可能{{Link-en|多重委派|Dual accreditation|兼任德國對多國派遣的大使}}<ref name="enembassy">{{lang|en|{{Cite web |title=The Embassies |url=http://www.auswaertiges-amt.de/sid_C2A116B5DC395FA3FC83EEDF312CEE23/EN/AAmt/Auslandsvertretungen/Botschaften_node.html |work=[[德國外交部|Federal Foreign Office]] |date=2011-08-17 |accessdate=2014-11-02}}}}{{en}}</ref><ref name="deembassy">{{lang|de|{{Cite web |title=Botschaften |url=http://www.auswaertiges-amt.de/DE/AAmt/Auslandsvertretungen/Botschaften_node.html |work=[[德國外交部|Auswärtiges Amt]] |date=2013-10-23 |accessdate=2014-11-02}}}}{{de}}</ref>，而使館的服務對象也會因此同時含括駐在國、兼轄國在內。相較之下，德國領事館或總領事館的服務範圍僅限於一國之內的特定地區，職責也不包括大使館處理的政治事務，而是著重於領務服務、貿易、文化及{{Link-en|公眾外交|Public diplomacy}}<ref name="enconsulate">{{lang|en|{{Cite web |title=Consulates-General and Consulates |url=http://www.auswaertiges-amt.de/EN/AAmt/Auslandsvertretungen/Konsulate_node.html |work=[[德國外交部|Federal Foreign Office]] |accessdate=2014-11-02}}}}{{en}}</ref><ref name="deconsulate">{{lang|de|{{Cite web |title=Generalkonsulate und Konsulate |url=http://www.auswaertiges-amt.de/DE/AAmt/Auslandsvertretungen/Konsulate_node.html |work=[[德國外交部|Auswärtiges Amt]] |accessdate=2014-11-02}}}}{{de}}</ref>。

德國常駐代表處的性質也有別於使領館，派駐對象並非單一國家，而是諸如[[聯合國|聯合國]]、[[歐洲聯盟|歐洲聯盟]]、[[北大西洋公約組織|北大西洋公約組織]]、[[經濟合作暨發展組織|經濟合作暨發展組織]]、[[教科文組織|教科文組織]]和[[歐洲委員會|歐洲委員會]]等[[國際組織|國際組織]]，代表德國在這些組織中的利益。另一項相異之處，在於德國常駐代表處並不設有領務部門，也不辦理簽證、[[德国护照|護照]]相關的領事服務<ref name="en常駐代表機構">{{lang|en|{{Cite web |title=The Permanent Missions |url=http://www.auswaertiges-amt.de/EN/AAmt/Auslandsvertretungen/STV_node.html |work=[[德國外交部|Federal Foreign Office]] |accessdate=2014-11-02}}}}{{en}}</ref><ref name="de常駐代表機構">{{lang|de|{{Cite web |title=Ständige Vertretungen |url=http://www.auswaertiges-amt.de/DE/AAmt/Auslandsvertretungen/STV_node.html |work=[[德國外交部|Auswärtiges Amt]] |accessdate=2014-11-02}}}}{{de}}</ref>。

截至2014年1月，德國在全球各地總共設有220多座駐外代表機構，當中大使館約150個、領事館及總領事館約60個、常駐國際組織代表處約10個<ref name="enmission"></ref><ref name="demission"></ref>。
