'''德国商用机场列表'''（{{lang-de|'''Liste der Verkehrsflughäfen in Deutschland'''}}）包含当前由[[德国|德国]]航空管理局主管、符合德国民用航空法分类的所有现役德国[[商用机场|商用机场]]。这些商用机场首先应用于商用航空、通用航空及特殊航空。本表将调用全部39座机场各自的[[IATA机场代码|IATA代码]]、[[ICAO机场代码|ICAO代码]]、类别、2010年的运输数据，以及[[跑道|跑道]]和容量信息。

==列表说明==
; 机场名
: 机场的称谓。
; IATA代码
: 由三个[[拉丁字母|拉丁字母]]组成，用于识别各机场的唯一代码。IATA代码由[[国际航空运输协会|国际航空运输协会]]分配。
; IACO代码
: IACO代码由[[国际民航组织|国际民航组织]]分配，是由四个拉丁字母组成的机场唯一代码。其中以ED开头的用以表示德国机场，以ET开头的为军用机场及前[[东德|东德]]机场代码。
; 类别
: 区域和国际机场的区别取决于授予的许可证。这种划分基于德国的民用航空法。符合国际机场地位的机场以灰色框显示。此外在德国的民用机场中，未获得官方认可，但在实际运营中提供国际航班的机场有：多特蒙德、法兰克福－哈恩、腓特烈港、卡尔斯鲁厄/巴登-巴登、吕贝克、帕德博恩/利普施塔特和韦策<ref>[http://www.adv.aero/mitglieder.html Mitgliederliste ADV] {{webarchive|url=https://web.archive.org/web/20110218093458/http://www.adv.aero/mitglieder.html |date=2011-02-18 }}</ref>。
; 开放年份
: 机场于当前地点正式运营的年份。
; 年客运量
: 机场的年度旅客发送总量，括号内为具体年份。当中包括定期航班、包价旅行、商务航班、商业航班和通用航空<ref name="adv_2009">adv.aero: ''[http://www.adv.aero/statistiken2009.html Statstiken IVF 2009] {{webarchive|url=https://web.archive.org/web/20100810145654/http://www.adv.aero/statistiken2009.html |date=2010-08-10 }}'' Abgerufen am 19. Januar 2010</ref>。
; 年起降量
: 机场的年度飞机起降总量，括号内为具体年份。当中包括定期航班、包价旅行、商务航班、商业航班和通用航空。
; 年货运量
: 机场的年度航空货物处理总量，括号内为具体年份。当中包括[[航空邮件|航空邮件]]的吨数。
; 跑道
: 德国航空管理局所核准的[[跑道|起降跑道]]长度及宽度。实际的数据如再加上开放空间及停机道可能会有所不同。
; 方向
: 起降跑道所处的方向。其特点是有两个数字分别位于跑道的两端。這些跑道會根據它們的磁方位角而被命名，其方位角同时指明了该跑道的使用方向，即使用跑道时航空器的运动方向。命名的原則是取跑道磁方位角的前兩位數。機場有超過一條方向相同的跑道的，便會在數字之後加以「L」、「C」、「R」來區別，分別代表左、中和右。
; 表面
: 起降跑道的构造成分。他们可能是[[沥青|沥青]]（标记为A）、[[混凝土|混凝土]]（标记为C）或[[草地|草地]]（标记为G）。
; 容量
: 机场年均可承受旅客发送总量的上限。
; 注
: 未知的数据以N/A（不可用）表示。
