'''德庆县'''是[[中国|中国]][[广东省|广东省]][[肇庆市|肇庆市]]下辖的一个县。面积2,258平方千米，人口35万。邮政编码526600。

== 行政区划 ==
下辖1个街道、14个镇：
* 街道办事处：德城街道。
* 镇：新圩镇、官圩镇、马圩镇、高良镇、莫村镇、永丰镇、武垄镇、播植镇、凤村镇、九市镇、悦城镇、回龙镇、沙旁镇、古有镇。

县人民政府驻德城街道。

== 歷史 ==
[[漢武帝|漢武帝]][[元鼎|元鼎]]六年（公元前111年）建縣。唐時劃歸[[端州|端州]]贞观元年（627年）又歸[[南康州|南康州]]，十二年改为康州。
[[南宋|南宋]]紹興元年（1131年）升为德慶府，以示「因德致慶」。[[民國|民國]]元年（1912年）改為德慶縣。

== 地理 ==
德慶縣屬低山[[丘陵|丘陵]]區，東北西三面環山，[[西江|西江]]從南部縣境流過。

== 氣候 ==
1月均溫12.6℃，7月均溫28.6℃，年均降水量1,151毫米。

== 自然資源 ==
礦藏：[[石灰石|石灰石]]、[[花崗石|花崗石]]、[[鉭|鉭]]、[[鈮|鈮]]、[[錫|錫]]、[[金|金]]等。

野生動物：[[穿山甲|穿山甲]]、[[黃猄|黃猄]]、[[果子狸|果子狸]]、[[芒花狸|芒花狸]]、[[白筍狸|白筍狸]]、[[貓狸|貓狸]]、[[箭豬|箭豬]]、[[貓頭鷹|貓頭鷹]]、[[石蛤|石蛤]]、[[北鵑|北鵑]]、[[畫眉|畫眉]]等。

林木：[[紅椎|紅椎]]、[[毛椎|毛椎]]、[[椆木|椆木]]、[[格木|格木]]、[[黃杞|黃杞]]、[[大水松|大水松]]。

== 經濟 ==
[[農業|農業]] - 主種[[稻|稻]]、[[木薯|木薯]]、[[甘蔗|甘蔗]]等，建立了[[松脂|松脂]]、[[水果|水果]]、[[南藥|南藥]]、[[肉桂|肉桂]]、[[蠶桑|蠶桑]]等五大基地，有「綠色銀行」之稱。德庆是全国柑桔产业十强县和中国果蔬无公害十强县。

[[工業|工業]] - 林產[[化工|化工]]、[[稀土|稀土]]冶煉、建材石料、輕紡服裝、食品飲料、機械電子等。

2002年全縣國內生產總值36億2千萬[[人民幣|人民幣]]。

== 风景名胜 ==
{{see also|德庆县各级文物保护单位列表}}
* [[德庆学宫|德庆学宫]]，位于城内朝阳西路，始建于宋，重建于元，大成殿是广东唯一一座元代木构建筑。1996年被列为第四批全国重点文物保护单位。
* [[悦城龙母祖庙|悦城龙母祖庙]]，位于悦城镇，是祀奉龙母蒲媪的庙宇，现存建筑基本为清代所建，其砖、木、石雕工艺精湛，是岭南古建筑的典范。2001年被列为第五批全国重点文物保护单位。
* [[三元塔_(德庆)|三元塔]]，位于城东郊[[德庆西江大桥|德庆西江大桥]]北岸东侧的白沙山上，是一座明代楼阁式砖塔，平面八角，外观9层，内分17层，高58.13米，1989年被列为广东省文物保护单位。
* [[三洲岩摩崖石刻|三洲岩摩崖石刻]]，位于九市镇三洲村，现存宋至清代摩崖石刻91刻，具有较高的历史、文物价值，2002年被列为广东省文物保护单位。
* [[华表石|华表石]]，又名锦石山、锦裹石，位于[[回龙镇|回龙镇]]建丰村的[[西江|西江]]北岸。海拔173米。属[[丹霞地貌|丹霞地貌]]。[[华表石摩崖石刻|华表石摩崖石刻]]为广东古代最大的摩崖石刻，2002年被列为广东省文物保护单位。<ref>{{cite web|url=http://www.zqdcn.gov.cn/tour/ShowArticle.aspx?ArticleID=916|title=德庆华表石摩崖石刻|accessdate=2009-12-13|author=|publisher=肇庆数字文化网|language=zh-hans|deadurl=yes|archiveurl=https://web.archive.org/web/20131219132825/http://www.zqdcn.gov.cn/tour/ShowArticle.aspx?ArticleID=916|archivedate=2013-12-19}}</ref>
* [[龙山宫|龙山宫]]，位于官圩镇安良村社咀山，明代建筑，1989年被列为广东省文物保护单位。
* [[盤龍峽瀑布|盤龍峽瀑布]]堪稱是目前廣東省內最優美的瀑布。
