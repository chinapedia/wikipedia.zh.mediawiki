[[德國聯邦國防軍|德國聯邦國防軍]]'''海軍'''（[[德语|德语]]：'''{{lang|de|Deutsche Marine}}'''）是[[德国|德意志联邦共和国]]的[[海军|海军]]部队。

德国海军的历史可追溯至[[德意志统一|德意志统一]]前。最早作为独立战斗单位出现的是普鲁士海军，随后成为北德意志邦联海军（{{lang|de|Norddeutsche Bundesmarine}}）。1872年德国统一后，其建立了[[德意志帝国海军|德意志帝国海军]]。随着工业的发展，帝国海军的力量大大加强，一度发展为仅次于[[英國皇家海军|英國皇家海军]]的世界上第二强大的海军部队。直到[[第一次世界大战|第一次世界大战]]结束后，[[魏玛共和国|魏玛共和国]]成立，海军失去了绝大多数舰船，改称「[[國家海軍|國家海軍]]」。到了[[纳粹德国|纳粹德国]]时期，再度更名为「[[纳粹德国海军|战争海军]]」。

1956年，西德加入[[北约组织|北约组织]]，联邦海军（{{lang|de|Bundesmarine}}）随之建立。1990年[[两德统一|两德统一]]后，联邦海军同东德的人民海军（{{lang|de|Volksmarine}}）合并，统一称为德国海军。

== 使命 ==
德国海军作为联邦国防军的一部分，在北约联盟中也扮演重要角色。其使命包括维持和平以及维护德国和其盟国领土主权的完整。

== 作战行动 ==
德国海军长久以来一直是北约海上力量的重要组成部分。此外，德国海军也曾参加国际反恐行动，比如[[持久自由行动|持久自由行动]]和[[积极奋进行动|积极奋进行动]]等。

目前，德国海军已派出2艘护卫舰、4艘攻击快艇和2艘辅助军舰参加联合国驻黎巴嫩过渡部队，使其成为了德国海军目前所参加的最大型的作战行动。

== 组织机构 ==
德国海军由{{Link-de|德國海軍總監|Inspekteur der Marine}}指揮，海軍總部位于[[波恩|波恩]]。主要的指挥部包括位于[[格吕克斯堡|格吕克斯堡]]的'''舰队指挥部'''和位于[[罗斯托克|罗斯托克]]的'''海军办公室'''。所有的作战部队均由舰队总司令指挥，而其它单位，如海军学校、海军基地和试验设施等则由海军办公室管制。目前，约有19,000名官兵在德国海军中服役，此外还有约6,000名海军人员在联邦国防军的各个核心部门工作。

== 军舰及武器系统 ==
德國在二次大戰後，由於為戰敗國的身分，[[同盟國|同盟國]]占領當局禁止德國海軍使用[[巡洋艦|巡洋艦]]、[[航空母艦|航空母艦]]、[[核子動力潛艇|核子動力潛艇]]等侵略性艦艇，與裝備在其上的[[彈道飛彈|彈道飛彈]]、[[巡弋飛彈|巡弋飛彈]]和[[核子武器|核子武器]]。故德國海軍現階段的編制以[[驅逐艦|驅逐艦]]、[[巡防艦|巡防艦]]、柴電動力潛艦為主，跟曾為二戰戰敗國的[[義大利海軍|義大利海軍]]和[[日本海上自衛隊|日本海上自衛隊]]相似，並以近海防禦為建軍重心。
