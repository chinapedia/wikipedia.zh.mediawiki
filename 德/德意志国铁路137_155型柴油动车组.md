'''DRG Class SVT 137''' 是一種屬於[[流線型|流線型]]的[[柴聯車|柴聯車]]，它開始被使用於[[德意志国铁路|德意志国铁路]] 時期，並在二次大戰以後分別由屬於西德的 [[德国联邦铁路|德国联邦铁路]]（後來DB將型號改為VT 04）與屬於東德的 [[德国国营铁路|德国国营铁路]] 所繼續使用。在當時，它創造了最高時速 160 km/h（100 mph）與平均時速達到 132 km/h的[[軌道列車速度紀錄|世界紀錄]]。因此它可以算是當時中歐的第一列[[HST|柴油高速火車]]。

== 各種型式與其相關資料 ==
'''SVT 137''' 是參考原型車 [[飛行的漢堡人|DRG Class SVT 877]] 而加以改進，其總共可區分為四種不同的通用型式與一種實驗類型式：
* 「Bauart Hamburg」（[[汉堡市|漢堡]]型式），兩節車廂組成，採用[[關節式轉向架|關節式轉向架]]，604 kW，2'Bo'2'，No 137 149 — 152
* 「Bauart Leipzig」（[[萊比錫|萊比錫]]型式），三節車廂組成，採用關節式轉向架，884 kW，2'Bo'Bo'2'，No 137 153 – 154 與 137 233 – 234，DR 183 251
* 「Bauart Köln"（[[科隆|科隆]]型式），三節車廂組成，共建造14組，採用傳統[[格爾利茨|格爾利茨]]型式（"Bauart Görlitz"）的[[轉向架|轉向架]]，882 kW / 956 kW， 	2'Bo'+2'2'+Bo'2'，160 t，座位數 132，No 137 273 — 278 與 137 851 — 858。
* 「Bauart Berlin」（[[柏林|柏林]]型式），四節車廂組成，其中包含一節動力／郵件車廂，使用單一慢速柴油引擎：970 kW，2'Bo'2'2'+'2'2'+Bo'2'，210.6 t，座位數126，No 137 901 — 902
* SVT 137 155（Kruckenberg實驗型式），由當初於1934－1938年設計建造[[Schienenzeppelin|Schienenzeppelin]]的[[Franz_Kruckenberg|Franz Kruckenberg]]所設計。三節車廂組成，採用了[[柴油液體變速驅動|柴油液體變速驅動]]（diesel—hydraulic），其中只有一端的車頭附有柴油引擎。因其為實驗車型，所以並未投入正式的客運服務。但在1939年的6月23日於[[漢堡市|漢堡]]—[[柏林|柏林]]（Hamburg—Berlin）之間創下了時速高達 215 km/h的[[軌道列車速度紀錄|世界紀錄]]。此車型的設計也大大影響了之後西德DB的 [[DB_Class_VT_10.5|DB Class VT 10.5]]，[[DB_Class_VT_11.5|DB Class VT 11.5]] 與東德DR的 [[DR_Class_VT_18.16|DR Class VT 18.16]] 相關設計與生產。

大部分的SVT 137都是屬於 [[柴油電力驅動|柴油電力驅動]]（diesel—electric），只有兩組「Bauart Leipzig」（萊比錫形式）與Kruckenberg實驗型使用了[[柴油液體變速驅動|柴油液體變速驅動]]（diesel—hydraulic）。

== 歷史 ==
由於 [[DRG_SVT_877|DRG Class SVT 877]]「飛行的漢堡人」（Flying Hamburger）的成功，DRG便開始計畫一套由[[柴聯車|柴聯車]]所組成，從首都[[柏林|柏林]]出發至德國各個主要城市的的高速鐵路網。而首先在1934/1935年間所生產的「Bauart Hamburg」（漢堡型式），便展現了與原型車SVT 877有所不同的設計：座位數由原先的99減少為77，最主要的不同便是由原先一排 3+1 的座位改為 2+1。「Bauart Hamburg」（漢堡型式）全長也比SVT 877長了2.3公尺。

1935年的7月1日，北部的主要幹線 [[柏林|柏林]]—[[漢諾威|漢諾威]]—[[科隆|科隆]] 的SVT高速客運正式開始營運；而另一條貫穿中部的主要幹線 [[柏林|柏林]]—[[萊比錫|萊比錫]]—[[埃爾福特|埃爾福特]]—[[法蘭克福|法蘭克福]] 則緊接著在1935年的8月15日開始服務載客。而自1936年開始，通往南部主要城市的 [[柏林|柏林]]—[[紐倫堡|紐倫堡]]—[[慕尼黑|慕尼黑]]／[[司圖加特|司圖加特]] 路線也加入了整個高速鐵路網展開營運。

在1936年的2月17日，「Bauart Leipzig」（萊比錫形式）締造了時速高達 205 km/h 的[[軌道列車速度紀錄|世界紀錄]] 而接著從1936年的5月開始，這列火車並在 [[柏林|柏林]]—[[弗羅茨瓦夫|佈雷斯勞]]—[[比托姆|比托姆]]（通往現在波蘭）的路線當中，以4小時17分跑完全程，也就是說它的平均表定時速達到了驚人的 109 km/h。

隨著[[第二次世界大战|第二次世界大战]]的開始，SVT的高速客運服務從1939年的8月開始暫停服務。且在[[第二次世界大战|第二次世界大战]]中，有部分的車廂被[[德意志國防軍|德軍]]用在軍事用途的運輸當中。

在二次世界大戰以後，有部分的 SVT 137 被送交到其他國家的手上。其中 SVT 137 274一開始被使用於美國佔領區，但它後來被轉送到[[美國|美國]][[弗吉尼亚州|弗吉尼亚州]]的 [[Fort_Eustis|Fort Eustis]]，被當作「有趣的科技物品」（Technical interesting object）所保存。而 SVT 137 852 則在1949年 開始服務於 CSD（捷克斯洛伐克國鐵），並被用在 [[布拉格|布拉格]]—[[布拉提斯拉瓦|布拉提斯拉瓦]] 的線路上。另外還有一列 SVT 137 855 被運用為 Trainset DP—14 班次，服務於[[苏联|苏联]]的 SZD。

其他剩下在德國的 SVT 137 則是被大戰之後西德的 [[德國鐵路|德意志聯邦鐵路（DB）]] 將型號改為VT 04或VT 06與東德的 [[德国国营铁路|德意志國家鐵路（DR）]] 改為VT 137，（之後又改為 VT 183）繼續使用。

DRG SVT 137 225 「Bauart Hamburg」（如右上圖所示），曾經被前[[德意志民主共和国|東德]]政府一直使用到1975年為止。在1991年，此列火車被重新漆上原來的塗裝，並且保存展示於現今的[[萊比錫|萊比錫]]中
