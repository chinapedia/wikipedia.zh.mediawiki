'''德鲁伊'''是[[大型多人在線角色扮演遊戲|大型多人在線角色扮演遊戲]][[魔兽世界|魔兽世界]]中十一大职业中的一个，可以变成多种动物，是一个混合职业。

== 背景 ==

在[[暴雪|暴雪]]的[[魔兽争霸III|魔兽争霸III]]中，就出现了利爪德鲁伊和猛禽德鲁伊的兵种。但在当时的背景故事中，只有男性的[[夜精灵|夜精灵]]才可能成为德鲁伊。在魔兽世界[[網路遊戲|網路遊戲]]中，德鲁伊作为大地自然教义的传播者，非常容易得过渡到了[[牛头人|牛头人]]和[[夜精灵|夜精灵]]之中，并且每一位德鲁伊都可以随意变形成熊，猎豹，花斑豹，海豹，甚至在新的资料片[[燃烧的远征|燃烧的远征]]中，可以变化成枭兽和老鹰以及生命之树。到了新的[[浩劫与重生|浩劫与重生]]中，自然教义又传播到了[[食人妖|食人妖]]和[[狼人|狼人]]之中，他们成为了最新的可以使用德鲁伊魔法的种族。

== 缩写 ==

德鲁伊的英文是Druid，简写为Dr。

== 职业特点 ==

德鲁伊可以穿皮甲、布甲，可以使用法杖、双手锤、单手锤、匕首、長柄武器。德鲁伊可以选择变成为[[熊|熊]]、[[猎豹|猎豹]]、[[海豹|海豹]]、[[枭兽|枭兽]]，在资料片[[魔兽世界：燃烧的远征|魔兽世界：燃烧的远征]]中，还可以变成[[飛行型態|飛行型態]]、[[树|树]]<ref name="魔兽1">{{cite web|url=http://www.wowchina.com/newplayer/classes/druid/shapeshifting.shtml|title=德鲁伊-游戏-魔兽世界|publisher=网之易|accessdate=2012|deadurl=yes|archiveurl=https://web.archive.org/web/20090603165223/http://www.wowchina.com/newplayer/classes/druid/shapeshifting.shtml|archivedate=2009-06-03}}</ref>。

=== 变形能力 ===

普遍认为，德鲁伊最吸引玩家的地方在于他的变形能力，可以变成各种各样的动物<ref name="魔兽1">{{cite web|url=http://www.wowchina.com/newplayer/classes/druid/shapeshifting.shtml|title=德鲁伊-游戏-魔兽世界|publisher=网之易|accessdate=2012|deadurl=yes|archiveurl=https://web.archive.org/web/20090603165223/http://www.wowchina.com/newplayer/classes/druid/shapeshifting.shtml|archivedate=2009-06-03}}</ref>。

在魔兽世界中，德鲁伊有以下形态：

* 熊形态：增加护甲和生命。其操作类似[[战士_(魔兽世界)|战士]]，带有怒气槽。外表是一只熊。
* 猎豹形态：增加伤害输出。其操作类似[[盗贼_(魔兽世界)|盗贼]]，有能量槽。外表是一只猎豹，联盟为黑色，部落为黄色，被玩家戏称为“没翅膀的[[双足飞龙|双足飞龙]]”。
* 水栖形态：增加游泳速度，只能在水中使用。外表是一头海豹。
* 旅行形态：增加移动速度，只能在室外的陆地上使用。外表是一只鹿，也可以通过雕文变成一只豹。
* 枭兽形态：增强法力回复速度和护甲，常用于法术攻击。其操作类似[[法师_(魔兽世界)|法师]]。需要平衡系的天赋支持才可以变成这种形态。外表是一隻枭兽

在资料片[[魔兽世界：燃烧的远征|魔兽世界：燃烧的远征]]中，德鲁伊增加了两种形态：

* 飞行形态：可以飞行。只能在非战斗时启用。外表是一只鳥。食人妖的德魯伊外表是一只蝙蝠。
可以通过德鲁伊訓練師學習变成迅捷飞行形态，飞行速度会更快。
* 生命之树形态：增加治疗效果，降低治疗法术的法力消耗。其操作类似[[牧师_(魔兽世界)|牧师]]。需要恢复专精并且选择化身天赋才可以变成这种形态。外表是一颗树。
*树人形态：与人形态时拥有相同的能力。需要树人雕纹才可以变成这种形态。

有趣的是，牛头人在变形后仍会保留头上的牛角，所以部落阵营的德鲁伊看起来常常是长着角的熊、狮子。相应的夜精灵变成的猎豹则长有长长的尖耳朵。

=== 专精 ===

平衡专精主要增强德鲁伊的法术攻击。

守护专精主要增强德鲁伊的防护能力。

野性专精主要增强德鲁伊的物理攻击。

恢复专精主要增加德鲁伊的治疗法术。

=== 优势和劣势 ===

德鲁伊的灵活度很高，可以随时切换自己的形态适应不同的情况。但在不同的形态下，操作方法也不相同。所以德鲁伊玩家常常要了解担任坦克、物理攻击、法术攻击、治疗这些不同角色时应该如何操作。

德鲁伊可以使用水生和旅行形态、飞行形态进行更快的旅行，部分德鲁伊玩家喜欢到处探索地图，采集资源等等，这些形态对他们很有帮助。

德魯伊擁有能在戰鬥中復活的技能，但一次冷卻10分钟。

== 队伍和团队 ==

一般在五人队伍中，因为德鲁伊的变形能力，所以可以根据队伍的其他成员组成，而选择担当坦克、治疗、伤害输出的角色，灵活度比较高。

在大型的团队[[副本_(魔兽世界)|副本]]中，德鲁伊往往会变形成树或者熊，以便担当治疗或者坦克的角色。

== 历史和现状 ==

在魔兽世界70年代前，德鲁伊曾经因为练级速度非常慢，以至于普遍被认为有耐心的玩家才能忍受的一个职业。很多玩家因此放弃了这个职业，导致大型的团队副本中德鲁伊的数量稀少。

在1.8补丁和资料片燃烧的远征中，德鲁伊的野性战斗天赋得到增强，练级速度得到了大幅提升，且80年代時平衡得到增強，吸引了很多新玩家。
