'''德华大学'''（{{lang-de|Deutsch-Chinesische Hochschule}}），又译作'''德华高等学堂'''、'''青岛特别高等专门学堂'''、'''青岛特别高等学堂'''、'''青岛特别高等专门学校'''等，是20世纪初[[德国|德国]]和[[清朝|中国]]在[[青岛|青岛]]合办的一所大学，也是第一所设立在中国，并得到政府承认的德国大学，存在时间为1909年到1914年。 因其办学地点位于[[台西_(青岛市)|台西]]海滨[[台西_(青岛市)#大黑栏和小黑栏|黑澜]]兵营旧址，所以又俗称'''黑澜大学'''<ref>{{cite web |url=http://www.qingdaonews.com/content/2007-01/18/content_7716680.htm|title=《“黑澜”与“迎宾馆”》|accessdate=2009-11-14 |work=|publisher=青岛新闻网}}</ref>。

== 办学历史 ==
1905年，德国开始执行将青岛建成模范租借地的计划，投入巨资兴建各类基础设施。1907年，德国计划在青岛租借地创办一所高等学堂。1908年，中德两国代表，学部尚书[[张之洞|张之洞]]和[[汉堡大学|汉堡大学]]汉学家[[福兰阁|福兰阁]]（Otto Franke）通过谈判，决定在青岛联合办学，其开办经费由德国政府投资60万马克，中国政府投资4万马克<ref>{{cite web|url=http://wanbao.qingdaonews.com/html/2007-09/16/content_83994.htm|title=《创办德华大学》|accessdate=2011-01-11|work=宗缵|publisher=青岛晚报|archive-url=https://web.archive.org/web/20160423171347/http://wanbao.qingdaonews.com/html/2007-09/16/content_83994.htm|archive-date=2016-04-23|dead-url=yes}}</ref>。总管全校事务的学堂监督（即校长）由德国人担任，中国方面可委派一位学堂总稽察，行使监管中国师生以及与校长共同签署学生毕业证书的职能<ref name="multiple1">{{cite web|url=http://www2.sdnews.com.cn/news/wenwei/2006-11/10_380184.html|title=《一所久已尘封于历史档案的大学——德华大学旧事》|accessdate=2008-08-10|work=|publisher=山东新闻网|deadurl=yes|archiveurl=https://web.archive.org/web/20081119020742/http://www2.sdnews.com.cn/news/wenwei/2006-11/10_380184.html|archivedate=2008-11-19}}</ref>。

1909年9月12日学校开学（开学典礼在10月25日补行），第一任校长为德国地质学家、海军部官员[[格奥尔格·凯贝尔|格奥尔格·凯贝尔]]（Georg Keiper），总稽察为学部员外郎[[蒋楷|蒋楷]]。1912年时该校有德国教师26人，多数拥有博士、硕士学位，如数学教授{{link-en|康拉德·克诺普|Konrad Knopp|康拉德·克诺普}}、植物学教授威廉·瓦格纳、物理学教授{{link-de|卡尔·艾利希·胡普卡|Karl Erich Hupka|卡尔·艾利希·胡普卡}}等，也有6位既有中国科举考试功名，又有新式学堂学历的中国教师，包括中国文学及伦理学教授[[于濂芳|于濂芳]]等。学生来自全国和海外，1912年在校学生有340人<ref name="multiple1" />，1914年达400余人。

1914年，德华大学因[[青岛战役|日德战争]]爆发停办，至此共毕业本科生200多名<ref>{{cite web |url=http://www.qingdaonews.com/epaper/qdzb/html/2007-05/25/content_816645.htm|title=《青岛特别高等学堂》|accessdate=2011-01-11 |work=鲁海|publisher=青岛早报}}</ref>。在校学生中有43人迁至[[上海|上海]][[同济大学|同济医工学校]]，同济医工学校因此增设土木科。

== 学制与课程 ==
该校学制为二级高等教育建制，包括预科、本科及中文科。其中预科传授基础知识（包括德语、历史、地理、数学、逻辑学、生物学、物理学、化学和绘画课这些西方课程以及古籍、历史、地理、伦理和文学这些中国传统课程），学制6年，相当于正规中学。本科修学专门学科，学制4年到6年，最初设法政、工、医、农林四科，本科亦开设德文和中文课。另外亦有中文科，毕业考试通过者可以进入[[国子监|国子监]]以取得文学学衔<ref name="multiple2">{{cite web |url=http://www.qdda.gov.cn/expertLibrary.do?method=viewingArticle&subject_id=12259376220001761001&articleId=12488346092291833001&resumeId=12486772863101562001|title=《德华高等学堂》|accessdate=2009-11-14 |work=李明|publisher=青岛档案信息网}}</ref>。在课程设置上该校更为注重应用技术，而非涉及意识形态的内容<ref>{{cite web |url=http://www.qdda.gov.cn/front/zizhengcankao/preview.jsp?ArticleID=12615339092651028001&subjectid=12259375799211842001|title=《德华大学农林科简介》|accessdate=2011-01-11 |work=聂惠哲|publisher=青岛档案信息网}}</ref>。

== 学校旧址 ==
德华大学的旧址地处铁路[[青岛站|青岛站]]以西的[[台西_(青岛市)|台西片区]]，北起单县路、西至汶水路，南至贵州路海滨（二十世纪八十年代[[市南区|市南区]]台西片区南部填海，使得贵州路不再滨海），东至鱼台路<ref name="multiple3">{{cite web |url=http://www.qdda.gov.cn/front/qingdaojingguan/preview.jsp?subjectid=12259376569844725001&ID=1806391|title=《朝城路》|accessdate=2009-11-14 |work=鲁海|publisher=青岛档案信息网}}</ref>。学校旧址建立在建于1899年的德国海军野战炮兵兵营（Feldartilleriekaserne，俗称黑澜兵营）的原址上，1909年开学的同时，也在进行教学楼的建设工程。新的楼房均于1910年之后竣工。主教学楼位于南部，高两层，四坡斜屋顶，其后侧为大礼堂和图书馆。此外校区还包括北部的两栋学生宿舍，以及一座服务楼与一座住宅<ref name="multiple2" />。西侧有一栋双层长条形建筑为野战炮兵营的遗存，该楼至今仍存，为青岛市区最老的兵营建筑。直至1914年，校方仍在筹划校舍的扩建工程，比如向北扩建一座新的宿舍楼，将教学大楼向西扩建等。一些扩建计划已经开始实施，甚至已经开始铺设地基，但战争和学校的停办打断了一切计划。

学校停止办学后，校舍相继被改做他用。其中原办公楼与教学楼成为胶济铁路局兼日军青岛军政署铁道部办公楼，自此长期为铁路部门所用，1949年后为青岛铁路分局（今[[济南铁路局|济南铁路局]]青岛办事处）沿用至今。1916年4月15日，日本高等女校开学，校址设在原野战炮兵营的长营房内，位于今黄台路的[[青岛日本高等女学校|新校舍]]于1918年建成，原营房被铁路部门一并使用。原学生宿舍后来成为青岛铁路医院（今青岛市第九人民医院），原学生宿舍以北地段改为铁路青岛工务段、电气段所属。汶上路以西部分30年代成为[[青岛学院商业学校|青岛学院商业学校]]，现为[[青岛第一中学|青岛一中]]。朝城路以东部分地段后来分别成为1930年开办的青岛市民众教育馆<ref name="multiple3" />、青岛朝城路小学及青岛第十二中学（后改为青岛外事服务学校）。
