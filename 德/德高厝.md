'''德高厝'''是[[中華民國|中華民國]][[臺灣|臺灣]][[臺南市|臺南市]][[東區_(臺南市)|東區]]的一處地名，舊稱'''下灣'''、'''竹篙厝'''{{r|地}}。廣義的德高厝在被稱為「竹篙厝」的時期相當於今東區[[東門路_(臺南市)|東門路]]以南各里的範圍{{NoteTag|1=日治時期的竹篙厝庄、竹篙厝大字，後來變成竹篙厝段。}}，狹義的德高厝則指位於今[[德高里_(臺南市)|德高里]]的德高厝聚落{{r|地}}。目前該地名的使用可見於德高里的德高厝上帝廟與崇成里的德高厝郵局。

== 地名演變 ==
[[File:德高厝上帝廟.jpg|thumb]]
[[File:臺南德高厝郵局.jpg|250px]]
[[File:臺南德高厝老屋01.jpg|thumb]]
[[File:臺南德高厝老屋02.JPG|thumb]]

據學者盧嘉興的考據，德高厝聚落舊稱下灣{{NoteTag|1=盧嘉興〈由竹篙里仁和宮探討舊籍所記仁和里下灣之上帝廟〉一文指出，舊有方-{志}-所記載位在[[仁和里_(堡里)|仁和里]]下灣的上帝廟，應該就是今天的[[德高厝上帝廟|德高厝上帝廟]]，是以可知德高厝舊稱下灣{{r|盧}}。}}，乾隆卅一年（1766年）[[臺灣縣_(1684年-1887年)|臺灣縣]]知縣[[趙愛|趙愛]]頒「齒德可風」匾給成為鄉飲大賓的當地大戶陳家家主陳玉瓊{{NoteTag|1=陳玉瓊（1708年2月10日－1782年2月24日），名「寧」，是陳家開基祖陳元享三子{{r|盧}}。}}，因當地陳家有位「德高望重」的長者，是以大家尊稱此地為「德高厝」{{r|盧}}。而從[[德高厝上帝廟|德高厝上帝廟]]所保有的乾隆四十六年（1781年）石製香爐來看，可知當時已使用德高厝作為地名，而嘉慶年間修纂的《[[續修臺灣縣志|續修臺灣縣志]]》仍將[[仁和里_(堡里)|仁和里]]的上帝廟記在下灣，盧嘉興認為這是修志者因襲舊志，未實地訪查並更改所致{{r|盧}}。

然而根據當地陳家神主牌位的寫法來看，道光十七年（1837年）時地名已經被寫成「竹高厝」，這可能是因為後人已不知地名典故，又因當時村莊周圍大多種有竹子禦敵，而將「德」改為臺語音近的「竹」字，後來又覺得「竹高」無任何意義，又進一步改寫成「竹篙」，而後「竹篙厝」的地名便沿用下來，使得地名由雅而俗{{r|盧}}。

日治時期，原先的仁和里竹篙厝庄在大正九年（1920年）改行五州二廳制後變成[[臺南市_(州轄市)|臺南市]]底下的竹篙厝[[大字|大字]]，二次大戰後沿用地名設有「竹篙里」，經盧嘉興考證後，民國六十六年（1977年）1月依據「德高厝」之名改成「德高里」{{r|地}}{{r|東}}。

== 行政區劃演變 ==
德高厝地區在明鄭時期屬於[[萬年州|萬年州]][[仁和里_(堡里)|仁和里]]，清朝初年屬於[[臺灣縣_(1684年-1887年)|臺灣縣]]仁和里{{r|地}}。日治時期「竹篙厝」地區改為仁和里竹篙厝庄，先後隸屬於臺南民政支部、[[臺南縣_(日治時期)|臺南縣]]（臺南辨務署<ref> 《臺灣總督府府報》〈府令第二十一號〉，1897年6月10日</ref>）、[[臺南廳|臺南廳]]（永仁區{{NoteTag|1=區長役場最初即設於竹篙厝庄<ref name="告"/>，後來在1911年6月7日改到永康下里後甲庄<ref> 《臺灣總督府府報》〈告示第九十號〉，1911年6月7日</ref>。}}<ref name="告">《臺灣總督府報》〈告示第三號〉，1910年1月18日</ref>），[[1920年臺灣|1920年]]地方制度改制後，「竹篙厝庄」改成「竹篙厝」[[大字|大字]]，隸屬於[[臺南州|臺南州]][[臺南市_(州轄市)|臺南市]]<ref>《臺灣總督府府報》〈府令第四十七號〉，1920年8月10日</ref>{{r|地}}。

二次大戰後，竹篙厝大字被分成竹篙里、大明-{里}-、府宅里、路東里、泉南里、龍山-{里}-，竹篙厝聚落即位在竹篙里{{r|地}}，而後竹篙里在1974年7月1日分出仁和里{{r|仁}}與大智里{{r|大}}。名稱改成德高里之後，1982年8月1日德高里分出崇學里{{r|學}}與崇善里{{r|善}}，而形成當前德高里的規模{{r|地}}。

== 聚落 ==
[[File:竹篙厝庄聚落.svg|thumb]]
根據臺灣堡圖的記載，日治初期德高厝地區（竹篙厝庄）只有竹篙厝與大路東兩個聚落。竹篙厝聚落最初原稱下灣，在明鄭時期便已形成且有上帝廟，後來在清乾隆年間改成「德高厝」，之後演變成「竹篙厝」，後來在1977年方改回德高厝{{r|地}}{{r|東}}。大路東則位於現在的崇明-{里}-境內，日治時期因為昭和七年（1932年）成立的[[臺灣糖業試驗所|糖業試驗所]]開闢農場，被迫遷村到現在大同路以西的地區居住{{r|地}}。
