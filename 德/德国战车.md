'''雷姆斯汀'''（Rammstein）是一個在1994年成立的[[德國|德國]]樂團，由[[蒂尔·林德曼|蒂尔·林德曼]]（主唱）、[[理查德·克魯斯帕|理查德·克魯斯帕]]（主奏吉他）、[[保罗·兰德斯|保罗·兰德斯]]（节奏吉他）、[[奥利·里德尔|奥利·里德尔]]（貝斯）、[[克里斯多夫·施耐德|克里斯多夫·施耐德]]（鼓手）、[[克里斯汀·洛倫茨|克里斯汀·洛倫茨]]（鍵盤）六人組成。他們的音樂風格包含了[[重金屬音樂|重金屬]]/[[工業金屬|工業金屬]]和[[電子音樂|電子音樂]]的元素，在德國當地，該團體亦被視為[[克羅埃西亞獨立國|NDH]](新德意志酷派搖滾浪潮)的一員。他們的歌曲絕大部分都以德文演唱，專輯並且在全球已銷售超過千萬。雷姆斯汀在德國是家喻戶曉的第一天團。

雷姆斯汀名字来源于1988年在德国[[拉姆斯泰因航展事故|拉姆斯泰因镇上发生的空难]]，多加的一个"m"使名字的涵意变为撞城车。

== 歷史 ==
=== 成立與早期（1994年以前） ===
乐团是由吉他手 Richard Z. Kruspe 所成立。1989年，他逃至[[西柏林|西柏林]]，並組成樂團 Orgasm Death Gimmicks。這段期間，他受到美國音樂的影響，尤其是 [[吻樂團|Kiss]]。在[[柏林圍牆|柏林圍牆]]倒塌之後，他回到[[施威林|施威林]]，當時的 Till Lindemann 是當地的編籃工，並在 First Arsch 樂團擔任鼓手。

當時，Richard 與 [[:en:The_Inchtabokatable|The Inchtabokatables]] 樂團的 Oliver Riedel、[[:en:Die_Firma|Die Firma]] 的 Christoph "Doom" Schneider 住在一起。Richard 意識到之前的音樂並不適合他，他想要一種結合機器和吉他的音樂。於是三人開始一同創作，但 Richard 發現同時創作音樂和歌詞非常困難，因此說服 Till Lindemann 加入雷姆斯汀。Richard 第一次發現到 Till 是某次無意中聽到他在工作時唱歌。

1994年的柏林舉辦一場業餘樂團的競賽，獲勝者可以到專業錄音室錄製一張四首歌曲的 Demo。他們贏得了這場比賽，並引起 Paul H. Landers 的注意，他在聽完 Demo 後希望能加入樂團。為了讓音樂更為完整。雷姆斯汀試著邀請先前與 Paul H. Landers 同為 [[:en:Feeling_B|Feeling B]] 團員的 Christian "Flake" Lorenz 加入。一開始 Lorenz 感到猶豫，但最終還是同意加入樂團。

=== 《Herzeleid》（1995–1997） ===
1995年3月，雷姆斯汀開始錄製第一張專輯《Herzeleid》，由[[:en:Jacob_Hellner|Jacob Hellner]]擔任製作人。<ref>{{cite web| url = http://www.rammstein.com/_Voelkerball/Band/Timeline/T1995/| title = Rammstein :: Band :: Timeline| accessdate = 2008-06-21| publisher = Rammstein.com| deadurl = yes| archiveurl = https://web.archive.org/web/20080613205805/http://www.rammstein.com/_Voelkerball/Band/Timeline/T1995/| archivedate = 2008年6月13日| df = }}</ref> 8月17日發行第一張單曲《Du Riechst So Gut》，9月24日推出專輯。同年，樂團與  [[:en:Clawfinger|Clawfinger]] 一同在[[華沙|華沙]]與[[布拉格|布拉格]]舉行巡迴演唱。12月2日至22日，雷姆斯汀在德國舉行17場的巡迴演唱，成功的提升樂團知名度。1996年初，他們繼續舉行數場演唱，並在1月8日發行第二張單曲《Seemann》。

3月27日，雷姆斯汀前往[[倫敦|倫敦]]演出 [[音樂電視網|MTV]] 的節目《Hanging Out》，是他們第一次在[[英國|英國]]演出。樂團首度在德國之外獲得主流市場的注目，是[[九寸釘|九吋釘]]靈魂人物、音樂指導 [[特倫特·雷澤諾|Trent Reznor]] 將他們的《Heirate Mich》與《Rammstein》兩首歌曲選進[[大衛·林區|大衛·林區]]的影片《驚狂》（Lost Highway）之中。電影原聲帶於1996年秋天在美國發行，並在稍後於1997年4月在歐洲發行。

1996年9月至10月，雷姆斯汀在[[德國|德國]]、[[奧地利|奧地利]]、[[瑞士|瑞士]]舉行巡迴演唱會，並在9月27日舉行周年演唱會「100 years of Rammstein」。演唱會的來賓包括了[[魔比|魔比]]（Moby）、[[:en:Christiane_Herbold|Bobo]] 和 Berlin Session Orchestra，柏林藝術家 Gert Hof 則負責燈光秀。

=== 《Sehnsucht》與《柏林現場》（1996年下旬–2000） ===
1996年11月，雷姆斯汀前往[[馬爾他|馬爾他]]的 Temple Studios 錄製專輯《Sehnsucht》。<ref>{{cite web| url = http://www.rammstein.com/_Voelkerball/Band/Timeline/T1996/| title = Rammstein :: Band :: Timeline| accessdate = 2008-06-21| publisher = Rammstein.com| deadurl = yes| archiveurl = https://web.archive.org/web/20080613205810/http://www.rammstein.com/_Voelkerball/Band/Timeline/T1996/| archivedate = 2008年6月13日| df = }}</ref> 這次一樣由 Jacob Hellner 擔任製作人。專輯中的第一首單曲《Engel》在1997年4月1日推出，5月23日在德國認證為金唱片。隨後這張單曲發行歌迷板《Engel Fan Edition》，收錄兩首未發表的歌曲《Feuerräder》、《Wilder Wein》。專輯的第二首單曲是《Du Hast》，在1997年8月攻占德國單曲榜第五名。

之後樂團持續舉行巡迴，1997年8月22日正式推出《Sehnsucht》專輯。專輯在發行兩周後登上排行榜冠軍。此時，首張專輯《Herzeleid》與《Sehnsucht》中的兩支單曲（《Engel》和《Du Hast》）都在排行榜前二十名。1997年9月到11月，雷姆斯汀在歐洲各地舉行場場滿座的演唱會，隨後發行翻唱自[[發電廠樂團|發電廠樂團]]的歌曲《Das Modell》。這張單曲還收錄了另一首未發行的歌曲《Kokain》。1997年12月5日，雷姆斯汀與 [[:en:KMFDM|KMFDM]] 舉行樂團在美國的首次巡迴。

1998年8月22日至23日，雷姆斯汀在柏林 Wuhlheide 演唱，吸引了超過17,000名歌迷進場，是他們當時最大的一場演唱會。暖場樂團和歌手包括了 [[:en:Danzig_(band)|Danzig]]、[[:en:Nina_Hagen|Nina Hagen]]、[[:en:Joachim_Witt|Joachim Witt]] 以及 [[:en:Alaska_(singer)|Alaska]]。這場演唱會將發行現場DVD《柏林現場》。

1998年9月至10月，雷姆斯汀與[[崆樂團|崆樂團]]、[[:en:Ice_Cube|Ice Cube]]、[[:en:Orgy_(band)|Orgy]]、[[林普巴茲提特|林普巴茲提特]]一同參加「Family Values tour」。11月2日，《Sehnsucht》在美國認證為金唱片。

雷姆斯汀被當年的[[MTV歐洲音樂大獎|MTV歐洲音樂大獎]]提名為「最佳搖滾藝人」，並在11月12日的典禮中演唱《Du Hast》。

1999年2月，雷姆斯汀被第42屆[[葛萊美獎|葛萊美獎]]提名為「最佳金屬演出」。8月30日，發行《柏林現場》的CD版，限量版則是雙CD版本。兩周後，《柏林現場》登上德國專輯榜冠軍。9月13日，發行《柏林現場》錄影帶版，11月26日發行DVD版。

== 成員 ==
雷姆斯汀自從1994年組建成立之日起一直保持著同樣的樂團陣容，這在人員更迭頻繁的重金屬樂團中可謂鳳毛麟角。

樂團成員宣稱如果有任一人脫團，那將意味著雷姆斯汀的解散。洛倫茲經常說自己並非樂團的正式成員，而是跟樂團“協作”。

理查德·Z·克鲁斯帕（Richard Z. Kruspe）：主音吉他手、和聲

保羅·H·蘭德斯（Paul H. Landers）：節奏吉他手、和聲

提爾·林德曼（Till Lindemann）：主唱、口琴

克里斯蒂安·洛倫茲（Christian "Flake" Lorenz ）：鍵盤、採樣、合成器、喇叭

奧利弗·“奧利”·里德爾（Oliver "Ollie" Riedel）：低音吉他

克里斯托弗·施耐德（Christoph Schneider）：鼓、打擊樂器
