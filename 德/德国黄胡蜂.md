'''德國黃胡蜂'''（''Vespula germanica''）是在大部份[[北半球|北半球]]地區都能見到的[[黃蜂|黃蜂]]，原住於[[歐洲|歐洲]]、[[北非|北非]]及[[亞洲|亞洲]]的[[溫帶|溫帶]]。牠們被引入其他地方，如[[北美洲|北美洲]]、[[澳洲|澳洲]]及[[新西蘭|新西蘭]]。德國黃胡蜂屬於[[胡蜂科|胡蜂科]]，並有時被誤指是[[胡蜂屬|胡蜂屬]]的一種。

== 特徵 ==
[[File:Vespula_germanica01.jpg|thumb]]
德國黃胡蜂約有13毫米長，呈典型的黃黑色。牠們的外形很像[[普通黃胡蜂|普通黃胡蜂]]，但從[[頭部|頭部]]觀看，可見牠們的臉上有三個細小的黑點。德國黃胡蜂在[[腹部|腹部]]亦有黑點，但普通黃胡蜂有類似的斑紋並與黑環連接，形成另一種圖樣。

== 蜂巢 ==
德國黃胡蜂的蜂巢是以咀嚼後的[[植物|植物]]纖維，加上[[唾液|唾液]]組成。蜂巢一般會築在接近地面，而非灌木及樹上。它的巢房是開放式的，並有巢柄連接蜂巢及底部。德國黃胡蜂會在巢柄周圍分泌一種可以抗拒[[螞蟻|螞蟻]]的[[化學物質|化學物質]]，以避免牠們的侵襲。

德國黃胡蜂的蜂后會獨自開始建立蜂巢，在第一次產卵前會完成20-30個巢房。蜂后會視乎氣候的情況，一般在[[春天|春天]]，留在一根葉柄上，並開始建立一個蜂房。接著在此蜂房上另外再建立6個蜂房，形成典型的[[六角形|六角形]]。

當幼蟲長成[[工蜂|工蜂]]後，牠們會肩負尋找食物、照顧幼蟲／卵及蜂巢修理的工作。完成的蜂巢可能闊20-30厘米，當中有達3000隻德國黃胡蜂。

每一個德國黃胡蜂的蜂群中有一隻蜂后及多隻工蜂。蜂群一般只會存在一年，除了蜂后外，其他的工蜂都會在[[冬天|冬天]]前[[死亡|死亡]]。但是在較溫和的地方，如[[新西蘭|新西蘭]]，到了冬天仍會有約10%的德國黃胡蜂生存。新的蜂后及雄蜂會在[[夏天|夏天]]末出生，[[交配|交配]]過後，蜂后會在石隙或其他有蓋地方過冬。

德國黃胡蜂會捕捉其他[[昆蟲|昆蟲]]及[[毛蟲|毛蟲]]來餵養其幼蟲，所以牠們一般而言是益蟲。成蟲吃[[花蜜|花蜜]]及甜的[[果實|果實]]，且會吃[[人類|人類]]的食物及渣滓，尤其是[[肉類|肉類]]。

[[蜂鷹|蜂鷹]]會侵襲及挖開蜂巢，吃當中的幼蟲。[[蜂蚜蠅屬|蜂蚜蠅屬]]的''Volucella pellucens''及其親屬會產卵在蜂巢內，讓其幼蟲出生時吃德國黃胡蜂的幼蟲。

== 害蟲 ==
德國黃胡蜂與[[普通黃胡蜂|普通黃胡蜂]]及[[馬蜂屬|馬蜂屬]]的兩個[[物種|物種]]在[[新西蘭|新西蘭]]都被認為是害蟲。德國黃胡蜂可能是於19世紀末引進新西蘭，於約1940年前大量減少。<ref>[http://www.ebop.govt.nz/land/media/pdf/Fact_Sheet_PA03.pdf Pest Animal Control] {{webarchive|url=https://web.archive.org/web/20081015070614/http://www.ebop.govt.nz/land/media/pdf/Fact_Sheet_PA03.pdf |date=2008-10-15 }} Bay of Plenty environment report. Retrieved 7 January 2007</ref>牠們在[[櫸木|櫸木]]林中很普遍，因為牠們吃樹皮上昆蟲所分泌的[[蜜露|蜜露]]。這使[[森林|森林]]的蜜露大量減少，令原住的[[鳥類|鳥類]]失去食糧，影響當中的[[生態|生態]]。

德國黃胡蜂的蜂巢在室內亦十分之大，甚至整個閣樓都是其蜂巢。這可能與新西蘭較溫暖的冬季有關，與一般歐洲的環境有很大的分別。被蛰后皮肤会起红包但没有毒。
