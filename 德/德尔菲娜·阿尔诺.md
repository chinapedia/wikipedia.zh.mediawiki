'''德尔菲娜·阿尔诺'''（{{lang-fr|Delphine Arnault}}；{{bd|1975年|4月4日}}），是一名法国商人，担任[[路易·威登|路易·威登]]集团副总裁<ref>http://womany.net/read/article/3086</ref>。

== 学历与职业初期 ==
阿尔诺在[[塞纳河畔讷伊|塞纳河畔讷伊]]出生，毕业于[[里尔|里尔]][[北方高等商学院|北方高等商学院]]和[[倫敦政治經濟學院|倫敦政治經濟學院]]，德尔菲娜·阿尔诺首先在[[麦肯锡|麦肯锡]]战略咨询部开始了她的职业生涯。
2000年，她开始为[[酩悅·軒尼詩－路易·威登集團|酩悅·軒尼詩－路易·威登集團]]效力。她是LVMH集团总裁[[贝尔纳·阿尔诺|贝尔纳·阿尔诺]]的长女。

== LVMH ==

自2003年起，德尔菲娜·阿尔诺便成为[[酩悅·軒尼詩－路易·威登集團|酩悅·軒尼詩－路易·威登集團]]集团（LVMH）管理层成员。她是管理层最年轻的，同时也是首个女性成员<ref>http://www.businessoffashion.com/community/people/delphine-arnault</ref>。她同时也是多个奢侈品牌的行政人员：高级成衣品牌Céline<ref>http://www.purepeople.com/article/fashion-week-marie-ange-casta-et-kanye-west-fans-de-l-elegance-francaise_a116649/37</ref>,西班牙皮具与时装品牌[[罗威|罗威]]，意大利时装品牌Emilio Pucci，酒类品牌酩悅·軒尼詩，[[法國電視六台|法国电视六台]]以及其他在LVMH集团旗下的公司。她同时还是一家资产管理公司的联合负责人。最近她又成为了[[二十世紀福斯|二十世紀福斯]]与[[哈瓦斯通讯社|哈瓦斯通讯社]]的董事会成员。

2008年，她成为[[克里斯汀·迪奧_(品牌)|迪奥]]的副总裁。2013年，她改任[[路易·威登|路易·威登]]副总裁。

2014年，德尔菲娜·阿尔诺提议创办LVMH Prize大赛<ref>http://www.lvmh.com/group/lvmh-commitments/art-culture/lvmh-prize-for-young-fashion-designers/</ref>。该大赛在全球范围内选拔优秀的时尚设计师。她希望LVMH集团通过这个比赛来选拔并帮助有才华和有创意的设计师<ref>http://fashion.qq.com/a/20150530/007141.htm</ref>，并帮助他们实现梦想。

== 个人生活 ==

德尔菲娜·阿尔诺是法国商人[[贝尔纳·阿尔诺|贝尔纳·阿尔诺]]与其第一任妻子Anne Dewavrin所生的的长女。
