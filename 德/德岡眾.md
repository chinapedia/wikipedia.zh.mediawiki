'''德岡眾'''（とくおかしゅう），效力於[[周防國|周防國]]大名[[大內家|大內家]]及[[安藝國|安藝國]]大名[[毛利元就|毛利元就]]的[[忍者|忍者]]集團，活躍於[[山口縣|山口縣]]、[[廣島縣|廣島縣]]一帶。。
== 起源 ==
德岡眾與大內家同樣起源於[[百濟|百濟]]<ref>{{cite web |url=http://hiroshima.cool.ne.jp/kotodama-era/ninja_tokuoka.htm |title=存档副本 |accessdate=2010年7月29日 |deadurl=yes |archiveurl=https://web.archive.org/web/20080907075123/http://hiroshima.cool.ne.jp/kotodama-era/ninja_tokuoka.htm |archivedate=2008年9月7日 |df= }}，徳岡家の歴史</ref>，得意忍術也是從百濟學來的遁甲術，自承祖先是[[推古天皇|推古天皇]]時代百濟聖明王的第三子琳聖太子，在東渡日本後改稱多多良濱氏，一直服膺於控制周防國的大內家。
== 歷史 ==
在[[毛利元就|毛利元就]]於[[嚴島之戰|嚴島之戰]]擊敗[[陶晴賢|陶晴賢]]並將周防、長門兩國納入掌中後，德岡眾便轉為投效毛利家，配屬於毛利家部將[[杉原盛重|杉原盛重]]，協助[[吉川元春|吉川元春]]在[[山陰道|山陰道]]的戰事。德岡眾當時是採用二元統治的方式，分別是退隱的祖父德岡利兵衛與率領忍役百人組的組頭德岡久兵衛。

1578年，毛利家進攻尼子氏再興的基地上月城時，德岡眾由德岡久兵衛率領二十多名[[忍者|忍者]]助戰在上月城周邊收集情報和施放謠言，但是德岡久兵衛也和尼子家的忍者[[鉢屋眾|鉢屋眾]]交手時戰死<ref>[http://books.google.co.jp/books?id=GErt8bU4F1MC&printsec=frontcover&dq=%E3%82%B9%E3%83%BC%E3%83%91%E3%83%BC%E5%BF%8D%E8%80%85%E5%88%97%E4%BC%9D&hl=ja&ei=fhxRTNj2J4WevQOk-5GuBw&sa=X&oi=book_result&ct=result&resnum=1&ved=0CCsQ6AEwAA#v=onepage&q&f=false]，スーパー忍者列伝</ref>。
