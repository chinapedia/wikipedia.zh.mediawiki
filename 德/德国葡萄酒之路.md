'''德国葡萄酒之路'''（{{lang-de|Deutsche Weinstraße}}）是[[德国|德国]]第一条旅游路线，1935年开放，位于[[莱茵兰-普法尔茨州|莱茵兰-普法尔茨州]]，全长85公里。这条路线的标志是黄色背景上的一串十个葡萄和路线的名称。

== 历史 ==

为了促进旅游业，德国在1930年代开始设立一些旅游路线（{{lang|de|Touristenstraßen}}）。德国葡萄酒之路是在1935年10月19日正式开放的。为了配合这条线路的建立，当局在沿途的一些道路原来的名字后面加上了“{{lang|de|an der Weinstraße}}”字样，即“在葡萄酒之路上”。

== 地理 ==

[[File:Deutsches-Weintor.jpg|thumb]]
这条路线穿越普法尔茨葡萄酒产区。该地区是德国最温暖的一个区域，属于[[地中海式气候|地中海式气候]]，适合种植[[無花果|無花果]]，[[柠檬|柠檬]]，[[奇異果|奇異果]]这类在德国其他地区没有的植物。本地區所種植的[[雷司令|雷司令]]葡萄品種最負盛名。

位于德法边界[[施崴根-雷西腾巴赫|施崴根-雷西腾巴赫]]的{{link-de|德国葡萄酒之门|Deutsches Weintor}}标志着这条路线的起点。这座建筑于1936年使用[[砂岩|砂岩]]建造。路线一直向北延伸，经过[[巴特贝格匝本|巴特贝格匝本]]、[[兰道|兰道]]、[[埃登科本|埃登科本]]、[[葡萄酒之路上的新城|葡萄酒之路上的新城]]、[[戴德士海姆|戴德士海姆]]、[[巴特迪克海姆|巴特迪克海姆]]和[[格陵施塔特|格陵施塔特]]，到达{{link-de|博肯海姆|Bockenheim}}。

== 葡萄酒节 ==
[[File:Dubbeglas.jpg|thumb]]
每年的三月到十月，德国葡萄酒之路举办众多的露天葡萄酒节，成为旅游的一个亮点。世界上最大的葡萄酒节每年九月在巴特迪克海姆的香肠集市（{{lang|de|Wurstmarkt}}）举行。这里有能装下170万升葡萄酒的世界上最大的[[葡萄酒桶|葡萄酒桶]]，吸引了超过60万名旅客。其他重要的葡萄酒节还有十月的{{lang|de|Deutsches Weinlesefest}}（{{lang|de|Neustadt an der Weinstraße}}），七月的{{lang|de|Stadtmauerfest}}（{{lang|de|Freinsheim}}），八月的{{lang|de|Deidesheimer Weinkerwe}}（{{lang|de|Deidesheim}}），和每年最早举办的{{lang|de|Mandelblütenfest}}（{{lang|de|Gimmeldingen}}）。

每逢八月的最后一个星期天，是德国葡萄酒之路体验日（{{lang|de|Erlebnistag Deutsche Weinstraße}}），整条葡萄酒之路都禁止汽车通行，众多葡萄酒生产商和小酒吧为成千上万的[[徒步|徒步]]的、骑[[自行车|自行车]]的、溜[[轮滑|轮滑]]的漫游者服务。

与德国其他产酒区不同，这里的葡萄酒使用一种50[[毫升|毫升]]的玻璃杯销售，而不是通常的25毫升。这种名为{{lang|de|Dubbeglas}}的玻璃杯具有特殊的造型，酒杯上宽下窄，杯壁上有酒窝（{{lang|de|Dubbe}}）状的纹饰。
