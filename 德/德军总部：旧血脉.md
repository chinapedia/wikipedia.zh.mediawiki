{{vgname|德军总部：旧血脉|en=Wolfenstein: The Old Blood}}是一款由[[Machine_Games|Machine Games]]制作、[[贝塞斯达软件|贝塞斯达软件]]（''Bethesda Softworks''）发行的[[第一人称射击游戏|第一人称射击游戏]]。《德军总部：旧血脉》是《[[德军总部：新秩序|德军总部：新秩序]]》的独立资料片与前传性作品。讲述了原版之前的的故事，游戏设定[[纳粹|纳粹]]即将赢得[[二战|二战]]的1946年，为了转变[[盟军|盟军]]的士气，{{Link-en|B.J. Blazkowicz|William "B.J." Blazkowicz}}必须探访[[巴伐利亚|巴伐利亚]]执行重要任务，使用像手动步枪、榴弹枪等全新的武器武装自己，与纳粹的战争机器决一死战，利用全新的攀墙能力来终结纳粹敌人。<ref>{{Cite web |url = http://www.sonkwo.com/products/1660 |title = 德军总部：旧血脉 |author = 杉果游戏 |date = |publisher = 北京中电博亚科技有限公司 |language = |accessdate = 2015-08-18 |quote = }}</ref><ref>{{Cite web |url = http://ku.gamersky.com/2015/Wolfenstein-The-Old-Blood/ |title = 德军总部：旧血脉 |author = 游民星空 |date = |publisher = 石家庄友曼网络科技有限公司 |language = |accessdate = 2015-08-18 |quote = }}</ref>
