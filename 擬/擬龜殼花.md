'''擬龜殼花'''（[[學名|學名]]：''Macropisthodon rudis''），又名'''偽蝮蛇'''（[[英文|英文]]：Red Keelback<ref name="Keelback">{{Cite web |url=http://archive.zo.ntu.edu.tw/rept_index.asp?rept_id=R0060 |title=存档副本 |access-date=2011-08-25 |archive-url=https://archive.is/20121221211436/http://archive.zo.ntu.edu.tw/rept_index.asp?rept_id=R0060 |archive-date=2012-12-21 |dead-url=yes }}</ref>、False Viper<ref name="FalseViper">http://digimuse.nmns.edu.tw/DigiMuse/NewModule.aspx?ObjectId=0b00000181b7dbbe&ParentID=0b00000181b7db63&Type=&Part=</ref>、或False Habu<ref name="FalseHabu">http://www.snakesoftaiwan.com/Macrophistodon%20rudis%20rudis/species_macrophistodon_rudis.htm</ref>）、'''稜鱗蛇'''、'''假龜殼花'''，是[[蛇亞目|蛇亞目]]、[[游蛇科|游蛇科]]、[[頸稜蛇屬|頸稜蛇屬]]之下的一[[種_(生物)|種]]胎生無毒蛇類，主要分佈在[[中國|中國]]中、南部和[[台灣|台灣]]1500公尺以下地區，其外貌與[[龜殼花|龜殼花]]十分相似，藉以達到[[擬態|擬態]]、躲避[[天敵|天敵]]的效果。<ref name="firstchapter">{{cite book |author=杜銘章  |title=蛇類大驚奇 |origyear=2004|publisher= 遠流 |language=zh-hant |isbn=9573253275 }}</ref>

== 特徵 ==
中型[[蛇類|蛇類]]，體形粗胖，頭頂扁平，[[瞳孔|瞳孔]]圓形，尾短，體背灰褐色，上有兩列[[黑褐色|黑褐色]]的似橢圓形班塊，且夾雜較小的不規則狀斑紋。外貌與[[龜殼花|龜殼花]]類似，但是擬龜殼花的頭部比龜殼花還要圓一些，身長約80-120公分<ref name="FalseViper" />。

== 習性 ==
主要棲息在低海拔山區或溪澗的[[闊葉林|闊葉林]]中，以[[鼠|鼠類]]、[[蛙|蛙類]]為主食、亦補食[[蜥蜴|蜥蜴]]和[[蚯蚓|蚯蚓]]。受干擾時會把頭壓低，使其更趨於扁平、三角形狀，做出類似[[腹蛇科|腹蛇科]]的毒蛇姿態以嚇阻天敵，但其實攻擊性不強。多於夏秋時[[生產|生產]]。<ref name="snakeeeeee">{{cite book |author=林俊聰 |others= 向高世、杜銘章、林俊聰、陳王時、陳添喜（攝影），呂光洋（審訂），石瑞德（主編）|title=台灣96種爬動物圖鑑 |year=2007年1月 |publisher=社團法人台北市野鳥學會 |location=台北市 |language=zh-hant |isbn=9789868204539 }}</ref>

==  與龜殼花的差異 ==
從生活環境來說，擬龜殼花多分布於1500米以下地區，而龜殼花的生存海拔範圍為82至2200米<ref name='id4782'>{{cite web| url =http://www.bioinfo.cn/db05/BjdwSpecies.php?action=view&id=4782| title =原矛头蝮| work =《中国动物物种编目数据库》| publisher =中国科学院微生物研究所| author =中国科学院动物研究所| accessdate =2009-04-11| deadurl =yes| archiveurl =https://web.archive.org/web/20160305031258/http://www.bioinfo.cn/db05/BjdwSpecies.php?action=view&id=4782| archivedate =2016-03-05}}</ref>；而從外觀看來，擬龜殼花的頭部較龜殼花圓些、體背花紋相對於龜殼花為較規則的橢圓形、頭頂顏色一致，沒有大塊雲班<ref name="snakeeeeee" />
。綜合以上幾點，其實不難與龜殼花分別<ref name="ROCbigdic">http://nrch.culture.tw/twpedia.aspx?id=7026&Keyword=%E6%93%AC%E9%BE%9C%E6%AE%BC%E8%8A%B1</ref>。
