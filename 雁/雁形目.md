'''雁形目'''（[[学名|学名]]：{{lang|la|Anseriformes}}）是[[脊索动物门|脊索动物门]][[鸟纲|鸟纲]]的一个[[目_(生物)|目]]。本目的鸟在中文中通常被称为“鸭”或“雁”，包括了人们通常所说的[[鸭科|鸭]]、[[潜鸭属|潜鸭]]、[[天鹅|天鹅]]、各种[[雁属|雁类]]等鸭雁类（或雁鸭类）的鸟。本目的鸟都是游禽，在世界分布广泛。

== 形态特征 ==
本目鸟类均为水栖性鸟类，体型大小不一，大者如[[天鹅|天鹅]]体长可达1.5米，小者如[[棉凫|棉凫]]体长仅30厘米。

本目鸟类头较大，有的种类具有明显的冠羽，喙多为扁平形，尖端具有[[嘴甲|嘴甲]]，大多长颈。

本目鸟类翅长而尖，适于长途跋涉，[[羽毛|初级飞羽]]10－11枚，次级飞羽缺第5枚，大多数种类的次级飞羽色彩艳丽，具有抢眼的金属光泽，被称作[[羽毛|翼镜]]，大多数种类的尾巴很短，但也有个别种类具有异乎寻常长的中央尾羽如[[针尾鸭|针尾鸭]]，本目鸟类绒羽发达。

本目鸟类脚短，多着生于身体的中后部，跗跖前侧覆盖网状磷。三趾向前，有蹼或半蹼相连；一趾向后，较其它三趾为短，行走中不接触地面。

本目鸟类多雄雌异色，部分种类如[[天鹅|天鹅]]雄雌同形同色。雄性具有交接器。

== 习性 ==
本目鸟类都属于[[游禽|游禽]]，栖息在各种水域环境，从[[咸水|咸水]]到[[淡水|淡水]]，从内陆到远洋都有本目鸟类栖息，本目鸟类善于游泳，部分种类精于潜水，观察其在水中游弋的姿态，多数种类躯体轴线与水面平行，头、颈、背、尾部露出水面，精于潜水的雁鸭类，如各种[[潜鸭属|潜鸭]]的尾部也沉入水中；另外还有一些种类上半身其完全潜入水中，尾部翘出水面，即躯体轴线与水面有一定交角，如[[鸿雁|鸿雁]]。本目鸟类中部分种类潜水技能惊人，如[[长尾鸭|长尾鸭]]常能潜入水下60米之深处。本目鸟类[[尾脂腺|尾脂腺]]发达，分泌的油脂通过喙涂布到[[羽毛|羽毛]]上，以达到疏水来保持体态。

本目鸟类食性多样，大部分种类常在[[繁殖季节|繁殖季节]]以[[鱼|鱼]][[虾|虾]]，[[甲壳类|甲壳类]]和[[软体动物|软体动物]]为食，而在迁徙和越冬时以[[水草|水草]]等植物型食物果腹。需要指出的是[[雁亚科|雁类]]多以水草为食，而[[秋沙鸭属|秋沙鸭属]]的各种鸭类，嘴形扁平前端具钩，边缘具锐齿，且精于潜水，是捕食鱼类的好手。

== 繁殖 ==
本目绝大部分鸟类的繁殖是一雄一雌相配，其中[[天鹅|天鹅]]、[[树鸭属|树鸭]]、[[雁属|雁]]、[[麻鸭属|麻鸭]]等种类雄雌鸟终身配对，实行严格的一夫一妻制，双亲都参与幼鸟的养育，[[树鸭属|树鸭]]的雄鸟甚至参与孵卵。而本目的其它种类夫妻关系大多并不确定，常只能维持一个[[繁殖季节|繁殖季节]]，甚至在孵卵开始之后雄鸟就弃雌鸟而去，长期以来被人们当作忠贞爱情象征的[[鸳鸯|鸳鸯]]其实就是这样。

本目各种类的交配行为仅限于[[繁殖期|繁殖期]]内，但常常在越冬集群时期便开始配对行为，在冬末初春季节常能看到发情行为。本目鸟类常有复杂而悦目的求偶炫耀行为。

本目鸟类所做巢多见三种类型：
* 露天巢：搭建在离水不远的地面上或浅水处，由手边易得的植物型材料搭建，大多数的[[鸭属|河鸭]]都是营露天巢的。
* 隐蔽巢：这类巢搭建在在岩石下面，石隙间，垂直斜坡的土洞中，[[麻鸭属|麻鸭]]等的巢就是隐蔽巢。
* 树洞巢：搭建在树洞中，[[鸳鸯|鸳鸯]]的巢就是树洞巢。

此外，本目中的黑头鸭是这一目中唯一的[[巢寄生|巢寄生]]者，寄主有其它[[鸭科|鸭科]]、秧鸡、鹮类、鹭类的鸟。 

本目鸟类的雏鸟均为[[早成雏|早成雏]]，并且本目鸟类的雏鸟都有明显的[[銘印|印记行为]]，即把出生后看到的第一个物体当作自己的母亲，本能性地跟在“母亲”身后形影不离直到长大。雏鸟出生时体表被绒羽，出生后数月才发育成能飞的幼鸟，幼鸟羽色哑暗，颇似雌性成鸟，幼鸟出生后七到八个月进行第一次[[换羽|换羽]]。

本目鸟类[[性成熟|性成熟]]的时间长短不一，快的如[[鸭亚科|鸭亚科]]的大多数种类，孵出后九到十个月即能繁殖，而发育较慢的如[[天鹅|天鹅]]则需要四到五年。

== 迁徙 ==
本目鸟类绝大多数种类都是[[鸟的迁徙|迁徙]]性的，秋季南迁，春季北迁，据研究在[[中国|中国]][[内蒙古自治区|内蒙古]]的[[乌梁素海|乌梁素海]]，每年3月初就可以看到[[疣鼻天鹅|疣鼻天鹅]]、[[灰雁|灰雁]]、[[绿翅鸭|绿翅鸭]]等本目鸟类，在随后的日子里，[[斑嘴鸭|斑嘴鸭]]、[[绿头鸭|绿头鸭]]、[[赤麻鸭|赤麻鸭]]等相继到达，9月中旬各种鸭雁类开始南迁，最早开始迁徙的是[[灰雁|灰雁]]，同时在更北方繁殖的一些鸭雁类也开始过境[[乌梁素海|乌梁素海]]，10月间乌梁素海的鸭雁类数量达到顶峰，12月，最迟南迁的[[秋沙鸭属|秋沙鸭]]开始离开乌梁素海。

通常，本目鸟类在秋季南迁比较缓慢，春季北徙则比较迅速。

== 演化 ==
最早的雁形目鸟类化石是发掘自[[法国|法国]]下白垩纪地层的[[疑禽|疑禽]]，这也是迄今为止发现的唯一一块[[中生代|中生代]]雁形目鸟类化石标本，疑禽标本仅为上臂骨的两片碎片。

在[[新生代|新生代]]地层，则有分别发掘自[[法国|法国]]和[[美國|美国]]的[[罗迈雁|罗迈雁]]和[[始新鸭|始新鸭]]。

化石资料显示，在[[新生代|新生代]][[渐新世|渐新世]]雁形目[[鸭科|鸭科]]已经有[[雁亚科|雁亚科]]和[[鸭亚科|鸭亚科]]的分化；在[[中新世|中新世]]已经出现[[雁属|雁]]、[[鸭属|河鸭]]、[[潜鸭属|潜鸭]]和[[天鹅|天鹅]]等现存属出现；在[[上新世|上新世]]本目绝大部分现存种都已经出现；至[[更新世|更新世]]则计有90[[种(生物)|种]]化石被发现。
白垩纪有一种[[黄昏鸟|黄昏鸟]]在形态上与雁形目类似，不过属于[[反鸟|反鸟]]类，与现存雁形目没有关系。
综合全世界迄今为止发现的所有雁形目化石，共计约170余种，其中已经灭绝的占一半左右，绝大多数化石发现于[[欧洲|欧洲]]和[[北美|北美]]提示本目可能起源于[[古北界|古北界]]。

== 与人类的關係 ==
本目鸟类与人类关系比较密切，部分种类在历史上被人类[[驯化|驯化]]成为重要的家禽。
[[File:DSCN0821.JPG|thumb]]

家[[鹅|鹅]]被认为是人类驯化的第一个家禽，它来自于野生的[[鸿雁|鸿雁]]或[[灰雁|灰雁]]。[[查尔斯·达尔文|达尔文]]在他的《[[动物和植物在家养下的变异|动物和植物在家养下的变异]]》一书中提到“在[[荷马史诗|荷马史诗]]中就提到过早在[[古希腊|古希腊]]时代鹅就是被驯养的家禽了，而[[古罗马|古罗马]]时代的神庙里人们饲养家鹅，作为奉献给神的祭品”；还有人根据[[古埃及|古埃及]]笔画遗留下来的资料，认为在公元前2000年左右，古埃及人就成功地驯化了家鹅。

中国古籍《[[庄子|庄子]]》中有记载：“命竖子杀雁而烹之，竖子请曰，其一能鸣其一不能鸣，请奚杀。主人曰：杀不能鸣者”说明在公元前400年以前中国就已经有成熟的家鹅驯养技术了。

目前普遍认为中国家鹅和[[欧洲|欧洲]]家鹅的起源不同，中国家鹅起源于[[鸿雁|鸿雁]]而欧洲家鹅起源于[[灰雁|灰雁]]。

鸭也是重要家禽之一，目前认为驯化的家鸭起源于野生的[[斑嘴鸭|斑嘴鸭]]和[[绿头鸭|绿头鸭]]，[[中国|中国]][[戰國|战国]]时期的古籍有记载“野鸭为凫，家鸭为鹜”，欧洲家鸭的驯养可能起源于[[古埃及|古埃及]]后经[[希腊|希腊]]传播于整个欧洲大陆。
