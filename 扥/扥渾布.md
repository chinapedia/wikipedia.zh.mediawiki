'''白扥渾布'''（{{lang-mnc|ᡨᠣᡥᠣᠮᠪᡠ|v=tohombu}}；{{bd|1799年||1843年|catIdx=T扥}}），[[博爾濟吉特氏|博爾濟吉特氏]]，[[表字|字]]'''安敦'''，又字'''子元'''、'''愛山'''<ref>《清史館傳稿》6102號</ref>，室名'''瑞留堂'''、'''瑞榴堂'''<ref>杨廷福杨同甫 編《清人室名別稱字號索引》下冊905</ref>。[[蒙古正藍旗|蒙古正藍旗]]人。[[中國|中國]][[清朝|清朝]]政治人物、詩人。

== 經歷 ==
[[嘉慶|嘉慶]]二十三年（1818年）戊寅舉人。嘉慶二十四年（1819年）己卯恩科第三甲第八十三名[[同進士出身|同進士出身]]。分發[[湖南|湖南]]以[[知縣|知縣]]任用，署[[安化縣|安化縣]]知縣。後歷署[[東安縣|東安縣]]、[[湘潭縣|湘潭縣]]知縣，補授龍山縣知縣。[[道光|道光]]三年（1823年）起任東安縣知縣。

道光八年（1828年），調任[[福建|福建]]，歷署[[福州府|福州府]][[知府|知府]]、[[漳州府|漳州府]]知府、[[興化府|興化府]]知府。十三年（1833年）兼署[[台灣府知府|台灣府知府]]，擢[[廣西|廣西]][[右江道|右江道]]。十四年（1834年）改福建[[督糧道|督糧道]]。

道光十六年（1836年），擢[[直隸|直隸]][[按察使|按察使]]。十七年（1837年），改[[布政使|布政使]]。十九年（1839年）八月七日擢[[山東巡撫|山東巡撫]]。

道光二十二年（1842年）五月二十日病假，十二月二十一日<ref>1843年1月21日</ref>以病免。二十三年（1843年）病卒。

== 著作 ==

== 家庭及關連 ==

孫：寶杰、寶樑、[[寶棻|寶棻]]。
