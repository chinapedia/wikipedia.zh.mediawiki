==剧情==
[[法国艺术|法国女画家]]“波琳”身患绝症，突然闯入世外桃源般的[[贵州省|贵州省]][[丹寨县|丹寨县]]，她晕倒在水田里，被那里热心而且淳朴的[[苗族|苗族]]孩子救回了家，为了给她治病，当地苗族把自己奉为神灵的“锦鸡”抓来给她治病，当她重病时，年轻苗族男子们抬着她送到自治州州立医院去治病，最终，她决定把自己的余生奉献给这里的青山绿水。<ref>{{cite web |url=http://ent.sina.com.cn/m/c/2011-12-05/23513499961.shtml |title=《云上太阳》北大献映 简单故事感动众人 |accessdate=2012 |publisher=新浪网 |work= |language=zh-hans}}</ref><ref>{{cite web |url= http://ent.163.com/12/0423/22/7VQEFPL300032DGD.html |title=《云上太阳》4月27日全国公映 故事令人鼓舞 |accessdate=2012 |publisher=网易 |work= |language=zh-hans}}</ref>

== 製作 ==
丑丑是贵州省[[黔东南苗族侗族自治州|黔东南苗族侗族自治州]]人，[[侗族|侗族]]，毕业于[[北京电影学院|北京电影学院]]，是中国大陆侗族历史上第一位“电影导演”。<ref>{{cite web |url=http://news.hexun.com/2012-04-16/140454997.html |title=80后女导演丑丑携《云上太阳》深圳会影迷 |accessdate=2012 |publisher=和讯网 |work= |language=zh-hans}}</ref><ref>{{cite web|url=http://www.gz.xinhuanet.com/2008htm/xhws/2012-05/07/content_25188209.htm|title=专访“80后”侗族导演：有一种美丽叫丑丑|accessdate=2012|publisher=新华网|work=|language=zh-hans|deadurl=yes|archiveurl=https://web.archive.org/web/20160304202843/http://www.gz.xinhuanet.com/2008htm/xhws/2012-05/07/content_25188209.htm|archivedate=2016-03-04}}</ref>
