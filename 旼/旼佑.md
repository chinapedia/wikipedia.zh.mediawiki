'''河旼佑'''（{{lang-ko|'''하민우'''}}，{{bd|1990年|9月6日}}），韓國歌手，韓國男子音樂團體[[ZE:A|ZE:A]]成員，是組合主領舞、副唱、副Rapper。曾為SMJ Junior舞團團長，以絢麗的舞蹈UCC直捲各網站，曾為JYP練習生。於2010年1月15日《[[Music_Bank|Music Bank]]》憑主打歌《MAZELTOV》正式以團體名義出道， 亦為ZE:A子團體[[ZE:A-FIVE|ZE:A-FIVE]]與[[ZE:A_J|ZE:A J]]成員之一<ref>{{cite web |language=zh-tw |url=http://kpopn.com/2013/01/10/160370/ | title=ZE:A 組5人子團前進日本 |publisher=kpopn |date= 2013-01-10}}</ref>。

2012年6月26日，旼佑與佐佐木喜英及二階堂隼人組成 3 Peace☆Lovers於日本正式出道，並發行首張日文單曲為《Virtual Love》，《Virtual Love》被列入Oricon Charts TOP10,並已經被選為日本主流音樂綜藝節目“Break Out”的開幕主題曲，根據著名的專輯經銷商Tower Records指出3 Peace☆Lovers在日本單曲的發行量創歷史新高。 同年11月6日發行單曲《Love Evolution》，被列入Oricon Charts NO.2。

2013年4月30日，3 Peace☆Lovers發行單曲《Illusion / My True Love》被列入Oricon Charts NO.3。同年6月24日，3 Peace☆Lovers首張發行專輯《3 Peace☆Lovers》。
