'''貝瑞塔Cx4 Storm'''（{{Lang-en|Beretta Cx4 Storm}}）是一款由[[意大利|意大利]]槍械製造商[[貝瑞塔|貝瑞塔]]為運動用途、個人防衛和[[警察|執法機關]]使用而研製和生產的[[半自動手槍|半自動手槍]][[口徑|口徑]][[卡賓槍|卡賓槍]]，能夠使用來自[[貝瑞塔92手槍|貝瑞塔92／96]]／[[貝瑞塔M9手槍|M9]]系列、[[貝瑞塔8000手槍|貝瑞塔8000“美洲獅”]]以及[[貝瑞塔Px4_Storm手槍|貝瑞塔Px4 Storm]]這些總共七款或以上不同型號的全尺寸型手槍的[[彈匣|彈匣]]，發射{{Link-en|9×21mm IMI|9×21mm|9×21毫米IMI}}（其原來[[口徑|口徑]]）、[[9×19mm魯格彈|9×19毫米]]、[[.40_S&W|.40 S&W]]和[[.45_ACP|.45 ACP]]這四種[[手枪|手枪]][[子彈|子彈]]。

不過，在意大利民用市場的Storm系列，不論是目錄和銷售都有所變化，以限制了其模塊化所造成的互換性和防止使用並非指定的彈匣。

'''貝瑞塔Cx4 Storm'''是[[貝瑞塔Mx4_Storm衝鋒槍|貝瑞塔Mx4 Storm]]的長槍管半自動民用型版本。<ref>{{cite web|url=http://www.strategypage.com/htmw/htweap/articles/20110308.aspx|title=MP5 Gets Replaced|date=2011-03-07|accessdate=2011-03-07}}</ref>

== 設計細節 ==
[[File:Storm_Complete.jpg|thumb]]、[[寬型前握把|寬型前握把]]和{{Link-en|槍背帶|Sling (firearms)}}的貝瑞塔Cx4 Storm。]]
貝瑞塔Cx4 Storm是一枝由[[合金|合金]]和[[聚合物|聚合物]]製造的{{Link-en|閉鎖式槍機|Closed bolt}}和[[反沖作用_(槍械)|直接反沖作用]]式操作的[[半自動手槍|半自動手槍]][[口徑|口徑]][[卡賓槍|卡賓槍]]。它的開發是為了配合其他的[[貝瑞塔|貝瑞塔]]半自動手槍一起使用。例如，[[貝瑞塔92手槍|貝瑞塔92FS]] [[9×19mm魯格彈|9×19毫米]][[口徑|口徑]]的[[彈匣|彈匣]]亦可以用在了Cx4（[[膛室|膛室]]也要是9毫米口徑）。這有助於不再需要同時攜帶步槍和手槍所專屬使用的彈匣而減少了全副裝備的重量，以及因增加了空間而在武裝時不會增加重量以下增加[[子彈|子彈]]攜帶總數量。

貝瑞塔Cx4 Storm與不少[[手枪|手枪]]、[[包絡式槍機|包絡式槍機]][[冲锋枪|冲锋枪]]一樣，將彈匣裝入與[[槍托|槍托]]相連成一體的{{Link-en|手槍握把|Pistol grip}}內。貝瑞塔Cx4 Storm在需要轉換使用[[貝瑞塔Px4_Storm手槍|貝瑞塔Px4 Storm]]、[[貝瑞塔92手槍|貝瑞塔92／96]]和[[貝瑞塔8000手槍|貝瑞塔8000／8040“美洲獅”]]等[[9×19mm魯格彈|9×19毫米]]／[[.40_S&W|.40 S&W]][[口徑|口徑]]貝瑞塔手槍的彈匣時需要兩件改裝套件（這不包括[[貝瑞塔Rx4_Storm半自動步槍|貝瑞塔Rx4 Storm]]）。這些改裝套件的例子包括：使用[[貝瑞塔Px4_Storm手槍|貝瑞塔Px4 Storm]]的彈匣需要安裝C5C620[[直向插槽|彈匣插座]]和C89210[[彈匣|彈匣釋放按鈕]]；使用[[貝瑞塔92手槍|貝瑞塔92／96]]的彈匣需要安裝C5A511彈匣插座和C89109[[彈匣|彈匣釋放按鈕]]；而使用[[貝瑞塔8000手槍|貝瑞塔8000／8040“美洲獅”]]的彈匣則需要安裝C5A670彈匣插座和C89110[[彈匣|彈匣釋放按鈕]]。<ref>{{cite web | title = Converting magazine type in the CX4 Storm | url = http://berettausa.custhelp.com/cgi-bin/berettausa.cfg/php/enduser/std_adp.php?p_faqid=206 | publisher = Beretta USA Customer Help | accessdate = 20 January 2011}}</ref>而目前[[.45_ACP|.45 ACP]][[口徑|口徑]]型號只能夠使用[[貝瑞塔8000手槍|貝瑞塔8045“美洲獅”]]的8發單排可拆卸式[[彈匣|彈匣]]。

貝瑞塔Cx4 Storm裝有內置伸縮式[[皮卡汀尼導軌|MIL-STD-1913戰術導軌]]，可以藉著推動前端{{Link-en|槍背帶|Sling (firearms)}}按鈕和利用拇指護耳將戰術導軌從[[前護木|前]][[護木|護木]]內沿著[[槍管|槍管]]的底部拉出。每種Storm系列槍械還可以選擇利用兩顆安裝[[螺絲|螺絲]]以裝上前護木兩側的戰術導軌，而另一種選擇是從生產商購買連底部的三條戰術導軌架連接座。前護木底部的戰術導軌同樣需要利用兩顆安裝螺絲從戰術導軌的兩端螺絲孔才能裝上前護木兩側之下。使用者也可以從生產商購買[[機匣|機匣]]頂部的戰術導軌和利用工具連安裝至貝瑞塔Cx4 Storm。其機匣頂部的戰術導軌需要使用三顆螺絲和三顆[[螺母|螺母]]與利用機匣頂部的預鑽洞安裝，使得使用者能夠利用戰術導軌作為光學瞄準鏡基座，並且將另外兩個預鑽洞隱藏在塑料薄膜底下。安裝頂部附加導軌所需要的正確尺寸的[[鑽頭|鑽頭]]是並不包括在卡賓槍或是頂部導軌套件上，而且必須自行購買。貝瑞塔Cx4 Storm可以利用貝瑞塔配件包裝上最多五條戰術導軌，然後和各種目前流行的[[机枪|机枪]]、[[步枪|步枪]]及[[冲锋枪|冲锋枪]]一樣，使用者可以利用這些戰術導軌以便安裝市面上各種對應導軌的各種[[瞄準镜|光學瞄準鏡]]、[[紅點鏡|紅點鏡／反射式瞄準鏡]]、[[全像武器照準器|全息瞄準鏡]]、[[夜視鏡|夜視鏡]]、[[熱成像儀|熱成像儀]]、[[輔助握把|前握把]]、[[戰術燈|戰術燈]]、[[雷射瞄準器|雷射瞄準器]]和／或[[兩腳架|兩腳架]]這些戰術配件；另外，貝瑞塔Cx4 Storm亦可以利用槍口以裝上{{Link-en|槍口消焰器|Flash Suppressor|消焰器}}或[[抑制器|消聲器]]。這枝槍械的設計上有著非常高的自行定製適應能力。

貝瑞塔Cx4 Storm裝上了來自原廠的可調節式[[機械瞄具|機械瞄具]]系統。前方的柱狀[[照準器|準星]]可以利用隨槍提供的調整工具調節風偏和海拔。後方「L」字型可翻轉調節的覘孔式[[照門|照門]]是兩孔式設計，分別是有迅速射擊和遠距離高精度射擊這兩種選擇。這兩個機械瞄具可以在利用[[機匣|機匣]]頂部加裝的[[皮卡汀尼導軌|戰術導軌]]以安裝日間／夜間[[瞄準镜|光學瞄準鏡]]、[[紅點鏡|紅點鏡／反射式瞄準鏡]]、[[全像武器照準器|全息瞄準鏡]]、[[夜視鏡|夜視鏡]]和／或[[熱成像儀|熱成像儀]]時折疊下來。

貝瑞塔Cx4 Storm的[[槍托底板|槍托底板]]可以被使用者拔出以調節全槍的拉距。調整槍托時需要裝上槍托墊板加厚（加長）。貝瑞塔Cx4 Storm的槍托內亦已裝有一塊槍托墊板，它亦可以為了減薄（縮短）而被拆除。在沒有裝上槍托墊板以下，貝瑞塔Cx4 Storm的最小拉距拉是337毫米（13.27英吋）。為了增加全槍的拉距，使用者可以自行購買額外的槍托墊板。在總共裝上三塊槍托墊板以下，貝瑞塔Cx4 Storm的最大拉距是382毫米（15.04英吋）。

除了能夠利用戰術配件自行定制貝瑞塔Cx4 Storm以外，它亦是較為少數能夠將手動保險、彈匣卡筍、{{Link-en|拉機柄|Cocking handle}}和拋殼口轉換到武器的左側，以適應[[左利手|左撇子]]的使用者的槍械。

== 道森學院槍擊案 ==
由於其外觀具有攻擊性或威脅性，<ref name="notexpert">http://fullcomment.nationalpost.com/2011/09/13/matt-gurney-getting-shot-doesnt-make-you-a-firearms-expert/{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>一些人會把貝瑞塔Cx4 Storm誤認為是一枝[[突击步枪|突击步枪]]。{{Link-en|道森學院槍擊案|Dawson College shooting}}之中的[[槍手|槍手]][[金維爾·吉爾|金維爾·吉爾]]（{{Lang-en|Kimveer Gill}}）使用的主要射擊武器就是一枝貝瑞塔Cx4 Storm，而其中一名受害者希馬·卡迪姆（{{Lang-en|Hayder Khadimm}}），正當它在[[加拿大|加拿大]]被提倡禁制的時候，把貝瑞塔Cx4 Storm標籤是一枝“戰爭武器”和“軍用突擊步槍”。<ref name=notexpert/>雖然貝瑞塔Cx4 Storm在加拿大刑事法典當中是被歸類為“限制類”，真正的突擊步槍在刑事法典當中才是“禁制類”。<ref>{{cite web|url=http://laws-lois.justice.gc.ca/eng/acts/C-46/page-36.html |title=Criminal Code |publisher=Laws.justice.gc.ca |date= |accessdate=2011-03-13}}</ref>
