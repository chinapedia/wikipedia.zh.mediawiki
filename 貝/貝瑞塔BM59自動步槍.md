'''貝瑞塔BM59'''是[[義大利|義大利]]以[[美國|美國]][[M1加蘭德步槍|M1加蘭德步槍]]作基礎研製的[[自動步槍|自動步槍]]，發射[[7.62×51mm_NATO|7.62×51毫米北約標準彈]]，並以可拆式[[彈匣|彈匣]]供彈<ref>[http://www.a-human-right.com/BM59.html Beretta BM 59 rifle.] Retrieved on October 5, 2008.</ref>。後期的型號更加入了許多現代步槍的元素。

== 發展歷史 ==
[[二戰|二戰]]過後，義大利決定以M1加蘭德步槍作制式步槍並在當地合法生產。這支步槍在二戰期間已被證明是可靠的，然而步入1950年代後它已被認為是過時的武器。當時義大利軍方亦正打算以一種發射7.62×51毫米北約標準彈的新型步槍以取代M1加蘭德。

為了滿足[[義大利軍隊|義大利軍隊]]的需求，[[貝瑞塔|貝瑞塔]]設計出一種稱作BM59的步槍。而BM59實際上是以M1加蘭德縮小口徑而成，並採用20發可拆式彈匣供彈，亦新增了[[兩腳架|兩腳架]]和能對應[[槍榴彈|槍榴彈]]的{{Link-en|槍口消焰器|Flash Suppressor|消焰器}}。此槍並加入了[[自動火器|全自動射擊]]功能。

BM59在1959年被正式採用，並在義大利、[[阿根廷|阿根廷]]、[[印尼|印尼]]和[[摩洛哥|摩洛哥]]的軍隊中服役。而在1980年代初期，其半自動版本亦被出口到[[美國|美國]]並賣給一些私人收藏家。最早期的BM59是以美國製的M1加蘭德零件生產的，當中包括了縮小了口徑的[[槍管|槍管]]。

在1990年，義大利軍方以更先進的[[貝瑞塔AR70突擊步槍|貝瑞塔AR70/90突擊步槍]]取代了BM59，儘管有部份仍然在[[義大利海軍|義大利海軍]]中服役。

儘管BM59在外型上與[[M14自動步槍|M14自動步槍]]相似，但兩種步槍並沒有直接關聯，相同的只是兩者皆以M1加蘭德步槍作基礎而已。

== 衍生型 ==
BM59具有以下的軍用與民用衍生型：<ref name="MF">[http://world.guns.ru/assault/as54-e.htm Modern Firearms' Beretta BM 59 page.] {{webarchive|url=https://web.archive.org/web/20100902002500/http://world.guns.ru/assault/as54-e.htm |date=2010-09-02 }} Retrieved on October 5, 2008.</ref>

=== 軍用型 ===
* BM-59 Mark I：具有木製[[槍托|槍托]]和一體化的{{Link-en|手槍握把|Pistol grip}}。
* BM-59 Mark II：具有木製槍托和可分離式手槍握把以令武器在[[自動火器|全自動射擊]]能更容易地被控制。
* BM-59 Mark III（別名為BM-59 Ital TA及“高山部隊”）：具金屬製可摺疊式槍托及手槍握把，此型號是針對山地部隊而設計的。
* BM-59 Para：與Mark III型類似，但是針對[[傘兵|傘兵]]而設計的。它具有一根更短的槍管及{{Link-en|槍口消焰器|Flash Suppressor|消焰器}}。
* BM-59 Mark IV：使用重型槍管及塑料製槍托，為一種[[班用自動武器|班用自動武器]]。

=== 民用型 ===
BM-62和BM-69為數量稀少的民用運動步槍，它並不具槍榴彈發射器和瞄具<ref name="BM 59">[http://www.gunsmagazine.com/bm59/GCA0283.pdf Beretta's BM 59.] {{webarchive|url=https://web.archive.org/web/20060509040711/http://www.gunsmagazine.com/bm59/GCA0283.pdf |date=2006-05-09 }} Retrieved on October 5, 2008.</ref>。詳細資料如下：
* BM-62：使用外觀與20發彈匣相同長度的10發彈匣<ref>[http://www.securityarms.com/20010315/galleryfiles/1700/1798.htm Beretta BM62.] Retrieved on October 5, 2008.</ref>，只有半自動射擊功能。此型號並不具消焰器和兩腳架<ref name="BM 59"/>。
* BM-69：具消焰器和兩腳架的半自動型。<ref name="BM 59"/>
