'''貝殼麵'''（義大利語：Conchiglie）又叫'''蜆殼粉'''，是[[意式麵食|意式麵食]]裡面的一種。意式麵食分為很多種，有螺旋狀-{zh-hans:面;zh-hant:麵}-、筆管麵、蝴蝶麵...等，這種麵因為形狀像是[[貝殼|貝殼]]，所以稱之為「貝殼麵」。

== 製作材料 ==
和一般麵條製作手一樣。需要的材料有：[[高筋麵粉|高筋麵粉]]、少量[[低筋麵粉|低筋麵粉]]、蛋、橄欖油與鹽。口味與顏色是由自己喜愛搭配，例如小根紅蘿蔔、[[墨魚囊|墨魚囊]]、香料、酒……等。

== 早期的貝殼麵 ==
貝殼麵起源於中世紀地中海北側地區，當地居民對貝殼類海鮮相當熱愛，但是因為地域氣候環境不允許，且貝類價格高昂，因此請當地有名的阿巴拉絲•柯塞梅廚師幫忙，研發出一款利用小麥，棉花，稻桿，石灰等材料製作成貝殼的形狀的食材，置於義大利麵中，充當貝類海鮮，後來因該技術被發揚至世界各地，又因工業化而得以量化生產，成為現今一種常見義大利麵款式。

== 製作手法 ==
先將麵粉過篩，然後形成一個凹陷的平地，將蛋打入平地中，用筷子慢慢將蛋汁和麵粉混合，均勻的搓揉，若要是想要做不一樣的口味，可以依個人喜好加入食材，大約搓揉15~20分鐘，接著就是放入機器或模型中製作貝殼麵形狀出來，放乾後就是貝殼麵。

== 貝殼麵趣談 ==
收藏在大英國協自然歷史博物館裡的珍寶「受詛咒的貝殼麵化石」，據說是1935年一位義大利中年婦女，家中的牧羊犬在後院挖出一袋貝殼麵，婦女將其視為上天賜予的珍寶並嘗試入菜，沒想到家中成員紛紛出現消化異常，連摯愛的牧羊犬也因消化不良不幸死亡。
婦女將剩餘的貝殼麵分送給鄰居，鄰居認為該麵實屬少見，送交專家證實該麵為世界上最早的雛型貝殼麵，收藏價值極高，婦人的鄰居將其轉賣博物館收藏而致富，不亦樂乎。而該婦人將其烹煮不但失去寵物和家人健康也錯失脫貧致富的機會，後悔莫及。隔年，該婦女的鄰居紛紛離奇中毒死亡，婦女則不知所蹤，居民普遍認為貝殼麵是導致悲劇的主因。該故事至今仍在歐洲鄉村流傳，人們普遍稱這個故事為“受詛咒的貝殼麵”。

== 烹飪 ==
先將一鍋水煮開，接著放入少許的鹽和麵，煮到麵軟硬適中（大約6~7分鐘）即可取出備用。方法分為兩種，一種為一般醬料的麵，一種為[[焗烤|焗烤]]麵，一個人喜好而定。
