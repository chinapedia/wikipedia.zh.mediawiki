'''公益財團法人癌症研究會'''（{{jpn|j=がん研究会|hg=がんけんきゅうかい}}、Japanese Foundation for Cancer Research）是1908年（[[明治|明治]]41年）創立的[[日本|日本]]首間[[癌症|癌症]]專門研究機構。

== 概要 ==
「癌研」是日本最權威的癌症研究及治療機構之一。癌症治療方面與{{link-ja|國立癌症研究中心|国立がん研究センター}}醫院齊名。

癌症研究會的標誌「螃蟹」，源自於癌症英語「cancer」。

名譽總裁為[[常陸宮正仁親王|常陸宮正仁親王]]。

== 沿革 ==
以「透過消滅癌症為人類福祉做出貢獻」為基本理念，由當時日本政財官學界重鎮{{link-ja|青山胤通|青山胤通}}、[[山極勝三郎|山極勝三郎]]、[[澀澤榮一|澀澤榮一]]、[[桂太郎|桂太郎]]為中心所創立。

在{{link-ja|宣仁親王妃喜久子|宣仁親王妃喜久子|高松宮妃}}與之後就任[[東京大學|東大]]總長的{{link-ja|長與又郎|長與又郎}}等人的努力下，1934年（[[昭和|昭和]]9年）開設癌症專門研究所與醫院。

2005年（[[平成|平成]]17年），老舊的癌症研究會附屬醫院從[[上池袋|上池袋]]（最近車站是[[大塚站_(東京都)|大塚]]）移往臨海地區新設的[[癌症研究會有明醫院|癌症研究會有明醫院]]（[[東京都|東京都]][[江東區_(東京都)|江東區]][[有明_(江東區)|有明]]）。研究所也一樣移至新址。

此外，新醫院還設有[[緩和醫療|緩和醫療]]病床，提升末期醫療水準。

2011年（平成23年）改制為[[公益財團法人|公益財團法人]]。

== 組成 ==
* 癌症研究所
* 癌症化學療法中心
* 基因體中心
* [[癌症研究會有明醫院|癌症研究會有明醫院]]
