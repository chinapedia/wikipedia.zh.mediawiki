'''明野陸軍飛行學校'''（[[日語|日語]]：あけのりくぐんひこうがっこう）為[[大日本帝國陸軍|大日本帝國陸軍]]飛行教育訓練機關之一，位於現在的[[三重縣|三重縣]][[伊勢市|伊勢市]][[小俣町|小俣町]]北部的明野。

== 概要 ==
1921年、進行空中射撃訓練、[[所澤陸軍飛行學校|陸軍航空學校]]明野分校開學典禮。依據{{和暦|1924}}3月16日軍令第6號「陸軍飛行學校令」，升格為明野陸軍飛行學校。該校目的為：戰鬥飛行隊戰技訓練、調査研究、教育普及、關係兵器調査・研究・試驗等之科目。

隨著入校生的增加、設置[[原町市|原町]]（福島縣）、北伊勢、[[天龍市|天龍]]（静岡縣）、[[泉佐野市|佐野]]（大阪府）、[[高松市|高松]]、[[都城市|都城]]等分教場。

1944年6月、改編為「[[明野教導飛行師團|明野教導飛行師團]]」。1945年7月、更進一步改編為「第1教導飛行隊」。同年8月29日解散。

== 沿革 ==
* {{和暦|1920}}4月22日 - '''陸軍航空學校射撃班'''開班典禮。
* {{和暦|1921}}4月1日 - 昇格'''陸軍航空學校明野分校'''、設置於[[度會郡|度會郡]]北濱村大字村松。
* {{和暦|1924}}5月17日 - '''明野陸軍飛行學校'''獨立。
* {{和暦|1939}}2月10日 - 設立[[陸軍氣象部|陸軍氣象部]]明野觀測所。
* {{和暦|1944}}6月20日 - 改編為'''明野教導飛行師團'''。
* {{和暦|1945}}2月22日與4月22日遭到美軍空襲。
** 7月18日 - 改編為'''第一教導飛行隊'''。
** 8月29日 - 部隊解散

== 教育課程 ==
; 甲種学生
*對象 [[航空兵|航空兵科]][[大尉|大尉]]
*教育期間 6個月
*修學内容 戰術、空中戰鬥學術。
; 乙種學生
*對象 獲得新飛機操縱術之航空相關[[兵科|兵科]][[尉官|尉官]]・[[見習士官|見習士官]]。
*教育期間 約6個月
*修學内容 戰鬥飛行隊操縱必要之技能・知識。
; 下士官候補者養成
