'''明和'''是[[日本|日本]]的[[年號|年號]]之一。在[[宝历_(桃园天皇)|寶曆]]之後、[[安永_(年号)|安永]]之前。指[[1764年|1764年]]到1772年的期間。這個時代的[[天皇|天皇]]是[[後櫻町天皇|後櫻町天皇]]、[[後桃園天皇|後桃園天皇]]。[[江戶幕府|江戶幕府]]的將軍是[[德川家治|德川家治]]。

== 改元 ==
*保曆14年[[6月2日_(陰曆)|6月2日]]（公元1764年[[6月30日|6月30日]]）因後櫻町天皇即位而改元
*明和9年[[11月16日_(陰曆)|11月16日]]（公元[[1772年|1772年]][[12月10日|12月10日]]）改元為安永

== 出典 ==
出自《[[書經|尚書]]》的「九族既睦、平章百姓、百姓昭'''明'''、協'''和'''萬邦」。

之後在[[1926年|1926年]]，以同一個典故制定了[[昭和|昭和]]的年號。

== 明和年間要事 ==
*明和2年9月、發行[[五匁銀|五匁銀]]
*明和5年、五匁銀停止使用。
*明和8年夏、[[山城國|山城國]]以宇治為中心，[[神助參拜|神助參拜]]（お蔭参り）大流行。
*明和9年2月[[明和大火|明和大火]]（目黑行人坂大火災）。為[[江戶三大火|江戶三大火]]之一。該年災害相繼發生，有「明和九（Meiwaku）年為迷惑（Meiwaku）年」的說法。

== 逝世 ==
*4年[[竹内式部|竹内式部]]（享年55）、[[山縣大弐|山縣大弐]]（享年42）
*6年[[青木昆陽|青木昆陽]]
