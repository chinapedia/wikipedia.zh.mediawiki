'''明星花露水'''是[[清朝|清朝]][[上海|上海]]藥商[[周邦俊|周邦俊]]所研發調製的[[花露水|花露水]]品牌，現由中華民國[[新北市|新北市]]的[[明星化工|明星化工]]生產。當時市面上的花露水都是難以洗淨的黃色，而周研製出的花露水即使沾染在白色衣服上亦能輕鬆洗淨，因此推出後受到極大的歡迎。

== 歷史 ==
清[[光緒|光緒]]三十三年（1907年），[[上海|上海]]「中西大藥房」[[董事長|董事長]]周邦俊醫師參考《[[藥典|中華藥典]]》一書，以[[酒精|酒精]]加上[[玫瑰|玫瑰]]、[[茉莉|茉莉]]等多種花香精調配而成花露水，由[[化妝品|化妝品]]部門銷售。民國18年（1929年），由周邦俊長女[[周文璣|周文璣]]接手經營並創辦「明星香水肥皂廠」，並將化妝品部門獨立成公司，以大量生產與擴大經營。[[對日抗戰|對日抗戰]]勝利後，公司與工廠員工曾達數千人，產量超過一千萬瓶，並在[[上海證券交易所|上海證券交易市場]][[首次公開募股|掛牌上市]]。之後因為[[國共內戰|國共內戰]]，周家財產充公。1949年，周文璣以結束香港的倉庫為理由，帶著配方輾轉逃到香港，而後再到臺灣。

1951年，周文璣與早先到臺北設立分公司的錢一清、張明湧、張廷玉、謝兆裕等人決定重新開始，於是成立「[[明星化工|明星化工]]股份有限公司」，生產花露水與[[爽身粉|爽身粉]]，一推出後即風靡全臺，供不應求。在1960至1970年代，成為當時女性的必備品。但隨後因為消費習慣的改變、平均收入的提高及其它外來[[香水|香水]]產品的競爭，明星花露水的銷售開始每況愈下。

1978年，明星化工將公司及廠房遷移至[[新北市|臺北縣]][[土城區|土城鄉]]；1998年，周文璣過世，張明湧接任明星化工董事長。2003年，由於[[嚴重急性呼吸道症候群|嚴重急性呼吸道症候群]]（SARS）在臺灣爆發，明星花露水因為[[酒精|酒精]]濃度約73%，與稀釋後的消毒用酒精相近，所以在當時又意外成為熱門產品。<ref>百年明星花露水遇SARS再發，[[臺灣壹週刊|臺灣壹週刊]]第105期。2003年，第105期。</ref>

2018年底[[明星化工|明星化工]]因設備老舊停產明星花露水，現仍在評估是否重新生產。<ref>{{Cite web|title=阿嬤百年時尚「明星花露水」停產了！清朝賣至今還創千萬業績|url=https://tw.appledaily.com/new/realtime/20190315/1533834/|work=蘋果日報|date=2019-03-15|accessdate=2019-03-15|language=zh-TW}}</ref>

2019年3月[[明星化工|明星化工]]總經理郭超受訪表示，目前的狀況確實是「停產」，但不代表永久停產，停產是創新的必要過程，外界無須過多臆測，更沒有所謂的歇業規劃。<ref>[https://www.businesstoday.com.tw/article/category/80392/post/201903150021/明星花露水停產？業者回應了?utm_source=今周刊&utm_medium=autoPage  明星花露水停產？業者回應了]，今週刊 2019年03月15日</ref>

== 功用 ==
明星花露水起初是當香水使用，但是因為酒精濃度高，亦常被使用者當作其它用途：
*醒腦—當有昏睡感時擦在[[太陽穴|太陽穴]]或[[人中|人中]]部位，或在洗澡時滴在浴缸。
*祛臭—噴灑在廁所或其它有異味的場所，或是抹、灑在身上以去[[體味|體味]]。
*止癢—塗抹在被蚊蟲叮咬的部位。
*消毒—噴灑或塗抹在需要消毒的地方，如電話話筒或是居家四週。<ref>梁翠娟。[http://www.tncg.gov.tw/news.asp?id=%7B6C56C7FD-4937-4C5E-A0ED-74DEBF782A58%7D 明星花露水防治果蠅效果佳] {{webarchive|url=https://web.archive.org/web/20070927230035/http://www.tncg.gov.tw/news.asp?id=%7B6C56C7FD-4937-4C5E-A0ED-74DEBF782A58%7D |date=2007-09-27 }}。臺南：[[臺南市政府|臺南市政府]]新聞稿，2004年9月30日。</ref>
*飲用—由於明星花露水所含的酒精是[[食用酒精|食用酒精]]，因而早期派駐在[[禁酒|禁酒]]的[[中東|中東]][[伊斯蘭教國家|伊斯蘭教國家]]的臺灣[[榮工處|榮工處]]員工以花露水加水來解酒癮。<ref name="zhang">張依依。絕世經典的香水。臺北：經濟日報，2004年8月2日。</ref>
*醒酒—在熱毛巾上灑數滴花露水，再敷在醉酒者的臉上及後腦，有醒酒止吐的效果。<ref name="zhang"/>

== 備考 ==
在台灣生產的明星花露水，除了正面有代表明星化工產品標誌的明星跳舞圖繪之外，外裝上另標有「越陳越香」的字樣，此也成為明星花露水的一大特色。
