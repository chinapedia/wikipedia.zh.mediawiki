'''明湖水庫'''，是[[臺灣|臺灣]]的一座[[水庫|水庫]]，位於[[南投縣|南投縣]][[水里鄉|水里鄉]]的[[水里溪|水里溪]]河谷，水-{里}-壩（[[明潭水庫|明潭水庫]]）上游四公里處，[[縣道131號|縣道131號]]公路旁，是[[台灣電力公司|台灣電力公司]][[大觀發電廠|大觀發電廠]][[抽蓄式發電|抽蓄式發電]]機組的下池，為[[十二項建設|十二項建設]]中配合明湖抽蓄水力發電工程所興建的人工[[湖泊|湖泊]]<ref name="國內工程"/>。

== 沿革 ==
1980年，台灣電力公司奉[[行政院|行政院]]核准興建明湖抽蓄水力發電工程。明湖抽蓄發電工程位於[[南投縣|南投縣]][[魚池鄉_(臺灣)|魚池鄉]]水社村與[[水里鄉|水里鄉]]車埕村之間<ref name="抽蓄發電廠建設時代的見聞記"/>。1981年4月，明湖抽蓄水力發電廠開工。1984年10月7日，明湖抽蓄水力發電廠第一部機組併聯，開始發電。1985年8月，明湖抽蓄水力發電廠完工，耗資約[[新台幣|新台幣]]300億元。明湖抽蓄水力發電廠開始商業運轉後，交由[[大觀發電廠|大觀發電廠]]營運，稱為「大觀發電廠二廠」，簡稱為「大觀二廠」，原本慣常式發電的大觀發電廠改稱為「大觀一廠」<ref name="明湖下池水庫"/><ref name="中區水庫簡介"/>，完工後明湖抽蓄發電廠成為臺灣第一座[[抽水蓄能电站|抽蓄式水力發電廠]]<ref name="抽蓄發電廠建設時代的見聞記"/>。

== 設備 ==
明湖下池壩係一座壩高57.5公尺、壩頂長169.5公尺、壩頂標高450公尺、總容量為745萬立方公尺之[[混凝土|混凝土]][[重力壩|重力壩]]，該壩所蓄成之人工湖泊稱為「明湖水庫」<ref name="南投縣水庫壩堰"/><ref name="中區水庫簡介"/>，又稱為「復旦池」。明湖抽蓄水力發電廠以[[日月潭|日月潭]]為上池，於水里溪[[大觀發電廠|大觀發電廠]]（[[日月潭第一發電所|日月潭第一發電所]]）上游興建下池壩<ref name="電力工程技術之回顧與展望"/>，形成明湖水庫作為下池，攔截日月潭沖下的湖水。明湖抽蓄水力發電廠的進水口位於日月潭西側水社附近，即頭社壩北側，經由兩條直徑7公尺、長度分別為2,380公尺及2,350公尺的引水隧道，穿過[[台21線|台21線]]中潭公路下方，跨越頭社溪，再以壓力鋼管將水輸送至水里溪上游山腹內的地下發電廠進行抽蓄發電。電廠共裝置四部豎軸可逆型法蘭西斯水輪發電機組，水輪機兼具抽水機功能，而發電機則兼具電動機功能。每部發電機的機組容量為25萬千瓩（250[[MW|MW]]），總計明湖發電廠的裝置容量達100萬瓩（1,000[[MW|MW]]），每年發電量約16億度<ref name="大觀發電廠簡介"/>。明湖抽蓄電廠的發電原理係利用上、下池間達310公尺的位能差帶動發電機。當正轉時，可帶動發電機發電；反轉時，則可將下池的庫水抽回至[[日月潭|日月潭]]上池蓄存，可配合電力負載需求，將水力能源配合火力及核能基載發電運作，有效增加尖峰時間之供電量。發電後的尾水，可蓄存於水里溪下游的[[明潭水庫|明潭水庫]]，作為[[明潭發電廠|明潭發電廠]]抽蓄機組及川流式水-{里}-機組發電之用<ref name="大觀發電廠簡介"/>。抽蓄發電運轉時，作為上下池的日月潭和明潭水庫之水位經常變化，水位於凌晨開始下降，夜間抽水時日月潭上池下降0.8公尺，下池則上升20.5公尺，直到白天時日月潭水沖下才逐漸滿漲，每日往復升降，水位變化量大，形成獨特的「[[潮汐|潮汐]]」景觀。
