《'''開朗少女成功记'''》（{{lang-ko|'''명랑소녀 성공기'''}}）是[[韩国|韩国]][[SBS_(韓國)|SBS]]於2002年3月13日至5月2日于播出的一部电视剧。同年在中国大陆播出。《明朗少女成功记》是韩国2002年度收视率最高的青春偶像剧，在韓國取得42.6%的收視率。

== 剧情简介 ==
车杨顺的父母是诈欺犯，受害者找到车杨顺要她代父母还钱，为了使她们不去警局告自己的父母，杨顺答应受害者前去[[首尔|汉城]]有钱人家做佣人来偿还债务。

== 角色列表 ==

== 翻拍 ==
2011年，[[吳尊|吳尊]]、[[楊丞琳|楊丞琳]]主演《[[阳光天使|阳光天使]]》根据该剧翻拍而成。

== 剧集评价 ==
该部典型的“韩式电视剧”是所有少女都曾梦寐以求的灰姑娘的故事，剧中张娜拉饰演的车阳顺用自己的倔强与坚强赢得了韩启泰本部长的爱，也最终赢得了观众的青睐。该剧以40%以上的收视率成了2002年度收视率最高的青春偶像剧。该剧中，张赫走的是搞笑路线，当他那标志性的“呵哈，呵哈”的夸张笑声一响起，就令观众忍俊不禁；张娜拉则清纯可爱。

== 其它 ==
*该剧的热播让张娜拉在韩国人气飙升，使得曾拒绝饰演该剧女主角车杨顺的[[金喜善|金喜善]]后悔不已。<ref>{{Citeweb|url=http://news.xinhuanet.com/ent/2004-08/05/content_3432554.htm |title=剧集简介 |publisher= 新华网 |date= 2004-08-05 |language= zh-cn |accessdate= 2011-11-04 }}</ref>
*2002年，该剧在中国大陸译制播出，也让张娜拉在中国一举成名，为她成为第一个来中国大陸发展的韩流明星奠定基础。<ref>{{Citeweb |url= http://entertainment.dbw.cn/system/2011/10/30/053480180.shtml |title= 张娜拉自曝工作重要不怕当“剩女” |publisher= 东北网 |date= 2011-10-30 |language= zh-cn |accessdate= 2011-11-04 |archive-url= https://web.archive.org/web/20111106070839/http://entertainment.dbw.cn/system/2011/10/30/053480180.shtml |archive-date= 2011-11-06 |dead-url= yes }}</ref>
*受该剧热播影响，2006年由冯兴华执导的一部台剧命名为《[[宝岛少女成功记|宝岛少女成功记]]》。
