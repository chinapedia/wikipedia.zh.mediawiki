《'''明朝那些事兒'''》是一套在網路上連載的關於[[中國|中國]][[明朝|明朝]][[歷史小說|歷史小說]]，作者為[[當年明月|當年明月]]。{{注|当年明月，本名石悅，是[[中國大陸|中國大陸]]一名公務員。}}{{参|{{cite web|url=http://www.chinawriter.com.cn/zxhy/member/10080.shtml|title=石悦 (1979～)|publisher=中国作家协会主办|accessdate=2013-05-12}}}}{{参|{{cite web|url=http://big5.china.com.cn/culture/txt/2010-01/29/content_19331813.htm|title=石悅：在歷史寫作 我只能算一“票友”|date= 2010-01-29|accessdate=2013-05-12}}}}於2006年3月10日在[[天涯虚拟社区|天涯社区]]首次發表，其內容始於[[明太祖|明太祖朱元璋]]的誕生，終於[[崇禎帝|崇禎帝朱由檢]]。《明朝那些事兒》在網絡連載期間，每月點擊率逾百萬人次。{{参|{{cite web|url=http://book.douban.com/subject/2052448/|title=明朝那些事儿（叁）|accessdate=2013-05-12}}}}后來，《明朝那些事兒》集結成書籍刊行，在[[台灣|台灣]]發行了[[正體中文|正體中文]]版本。之後被翻譯為日文、韓文及英文等多國語言{{参|{{cite web|url=http://culture.gog.com.cn/system/2011/11/02/011238532.shtml|title=图文精印版：明朝那些事儿|publisher=culture.gog.com|date=2011-11-02|accessdate=2013-05-12|deadurl=yes|archiveurl=https://web.archive.org/web/20130423170140/http://culture.gog.com.cn/system/2011/11/02/011238532.shtml|archivedate=2013-04-23}}}}。

== 作品介紹 ==
一共七冊，分別為：{{参|name=《明朝那些事儿》}}{{参|{{cite web|url=http://shopping.pchome.com.tw/?mod=store&func=style_show&SR_NO=DJAP21|title=《明朝那些事儿》 全集|accessdate=2013-05-12}}}}

*《明朝那些事兒：第壹部洪武大帝》（舊版名《明朝那些事兒：朱元璋卷》）
::此冊主要描寫由1344年[[明太祖|太祖朱元璋]]入皇覺寺當行童至1402年[[靖難之役|靖難之役]]結束期間的歷史。

*《明朝那些事兒：第貳部萬國來朝》（舊版名《明朝那些事兒：貳》）
::此冊主要描寫由1402年[[明成祖|成祖朱棣]]登基至1457年[[奪門之變|奪門之變]]期間的歷史。

*《明朝那些事兒：第叁部妖孽宮廷》（舊版名《明朝那些事兒：叁》）
::此冊主要描寫由1457年[[明英宗|英宗朱祁鎮]]復位至1521年[[明武宗|明武宗朱厚照]]駕崩期間的歷史。

*《明朝那些事兒：第肆部粉飾太平》（舊版名《明朝那些事兒：肆》）
::此冊主要描寫由1521年[[明世宗|世宗朱厚熜]]登基至1562年嚴嵩被撤去[[明朝内阁首辅|内阁首輔]]一職期間的歷史。

*《明朝那些事兒：第伍部帝國飄搖》（舊版名《明朝那些事兒：伍》）
::此冊主要描寫由1562年[[徐階|徐階]]擔任内阁首輔一職至1599年[[萬曆朝鮮之役|朝鮮之役]]結束後明軍班師期間的歷史。

*《明朝那些事兒：第陸部日暮西山》（舊版名《明朝那些事兒：陸》）
::此冊主要描寫由1582年[[張居正|張居正]]逝世至1626年[[清太祖|清太祖努爾哈赤]]駕崩期間的歷史。

*《明朝那些事兒：第柒部大結局》（舊版名《明朝那些事兒：大結局》）
::此冊主要描寫由1626年[[清太宗|清太宗皇太極]]登基至1644年[[崇禎帝|崇禎帝朱由檢]]自縊期間的歷史。

== 写作过程 ==

《明朝那些事兒》於2006年3月10日開始於中國天涯社區發表。截至2008年3月，當年明月發表至[[萬曆|萬曆]]朝的[[國本之爭|國本之爭]]，4月發表至[[李成梁|李成梁]]經略[[遼|遼]]東與[[努爾哈赤|努爾哈赤]]的崛起，7月寫[[魏忠賢|魏忠賢]]事。2008年底發表至[[崇禎|崇禎]]初年，[[袁崇煥|袁崇煥]]伏誅事。2009年1月初發表[[盧象昇|盧象昇]]、[[孫傳庭|孫傳庭]]等人剿寇。{{参|name=出版市場回顧}}

當年明月每天以發表文章一小篇，字數不足一千字，週末時大多休息不發表，有時會連休數天，並公佈於網路上。2009年3月21日，《明朝那些事兒》連載完畢。{{参|{{cite web|url=http://www.mingchaonaxieshier.com.cn/|title=明朝那些事兒|accessdate=2013-05-12|deadurl=yes|archiveurl=https://web.archive.org/web/20130606032332/http://www.mingchaonaxieshier.com.cn/|archivedate=2013-06-06}}}}

== 行文风格 ==
《明朝那些事兒》以一种既幽默又调侃的笔调看待历史，但有時又不免流於輕浮，如《明朝那些事兒：第陸部日暮西山》 [[道统|道统]](2)一章寫道：
{{cquote|[[魏忠贤|魏忠贤]]终于满意了，行动进行极其顺利，[[汪文言|汪文言]]已成为阶下囚，一切都已准备妥当，下面……

下面没有了。}} 

此句是採用[[紀曉嵐|紀曉嵐]]消遣[[宦官|宦官]]的典故，言下之意是在调侃魏忠贤被[[阉割|阉割]]。{{参|{{cite web|url=http://blog.sina.com.cn/s/blog_49861fd50100a2i2.html|title=（长篇）明朝那些事儿－历史应该可以写得好看[1485]|author=当年明月|accessdate=2013-05-12}}}}

《明朝那些事兒》的另一特點是段落極短，尤其在第三冊之後，每一段落幾乎很少超過三行。{{参|name=《明朝那些事儿》}}

同类型的《[[如果這是宋史|如果這是宋史]]》是高天流雲發表的仿《明朝那些事兒》的关于宋史的网络[[小说|小说]]。{{参|{{cite web|url=http://book.douban.com/subject/3071235/|title=如果这是宋史|accessdate=2013-05-12}}}}此外还有雾满拦江發表的《别笑，这是大清正史》。{{参|{{cite web|url=http://vip.book.sina.com.cn/book/index_144195.html|title=麻辣解读清朝历史：别笑，这是大清正史|accessdate=2013-05-12}}}}

== 榮譽 ==
《明朝那些事兒》銷量超過五百萬冊，為中国大陆[[改革開放|改革開放]]以來最暢銷史學讀本（也有一种说法为“1949年[[解放|解放]]以来最畅销的通俗历史读本”），全國十大暢銷書之一，2007-2008年度暢銷書第一名。{{参|name=出版市場回顧|{{cite web|url=http://isbn.ncl.edu.tw/NCL_ISBNNet/C00_index.php?PHPSESSID=6qiq6nr3r1v9s7grnq9cgut8v7&Pfile=1868&KeepThis=true&TB_iframe=true&width=900&height=650|title=2009年中國圖書出版市場回顧|author=萬麗慧|publisher=全國新書資訊網|date=2009-01|accessdate=2013-05-12}}}}多次獲得“新浪圖書風雲榜”最佳圖書、[[當當網|當當網]]“終身五星級最佳圖書”、“卓越亞馬遜暢銷書大獎”等榮譽。{{参|{{cite web|url=http://www.amazon.cn/%E6%98%8E%E6%9C%9D%E9%82%A3%E4%BA%9B%E4%BA%8B%E5%84%BF%E5%9B%BE%E6%96%87%E7%89%88%E5%85%A8%E9%9B%86-%E5%85%A89%E5%86%8C%E5%A5%97%E8%A3%85-%E6%98%8E%E6%9C%9D%E9%82%A3%E4%BA%9B%E4%BA%8B%E5%84%BF%E5%9B%BE%E6%96%87%E7%89%88%E5%85%A8%E9%9B%86-%E5%85%A89%E5%86%8C%E5%A5%97%E8%A3%85-%E5%BD%93%E5%B9%B4%E6%98%8E%E6%9C%88-%E6%9D%8E%E5%AE%9D%E5%89%91-%E7%8E%8B%E6%99%93%E6%98%8E/dp/B007KXMAJ4|title=明朝那些事儿图文版全集（1-9）全9册套装 (明朝那些事儿图文版全集（1-9）全9册套装) [平装]|publisher=Amazon.com|accessdate=2013-05-12}}}}
