《'''明镜'''》周刊<ref>《世界报刊、通讯社、电台译名手册》修订本，辛华（即新华通讯社译名室），北京：商务印书馆，1978</ref>（{{lang-de|Der Spiegel}}），常译作《'''明镜周刊'''》，在[[德国|德国]]发行的[[左派|左派]][[杂志|周刊]]，每周的平均发行量近110万册。该杂志自称是“德国最重要的且在欧洲发行量最大的新闻周刊”。

== 历史 ==
这份杂志於1946年由[[盟軍占領下的德國|英国占领军]]在[[汉诺威|汉诺威]]创办，当时叫《周刊》。1947年，德国人接手这份杂志。当年1月4日，《明镜》借鉴英美的新聞雜誌，取代德国的早期杂志《每周关注》（{{lang|de|Diese Woche}}）在德国中部城市[[汉诺威|汉诺威]]正式发行了第一版《明镜》杂志。杂志的第一任主编是[[鲁道夫·奥格斯坦|鲁道夫·奥格斯坦]]。杂志创刊初期年青的编辑们就确立了《明镜》杂志的主旨：政治批评与严肃的政治评论。当时，《明镜》杂志的主管驻扎在伦敦再加上周刊的政治倾向，当时德国的其它三个占领国对周刊的发行给予了极大的压力，所以杂志不得已还是用以前的《每周关注》（{{lang|de|Diese Woche}}）作为杂志名出版发行。

从1947年1月4日第一版《明镜》出版以来鲁道夫·奥格斯坦一直是周刊的[[總編輯|總編輯]]和发行人。即便過世之後，他依然掛名發行人。《明鏡》现任總編輯为Georg Mascolo。

《明鏡》周刊於1952年遷至[[汉堡|汉堡]]，並發行《经理杂志》（''{{lang|de|Manager-Magazin}}''）。

== 事件 ==
在1950年到1960年的十年当中，[[联邦德国|联邦德国]]政府十分黑暗。由奥格斯坦率领的《明镜》周刊首先向政府发起进攻，写了大量十分尖锐的政治评论，奥格斯坦与当时黑暗的联邦政府抗争的十年在德国历史上被称做“民主突击炮”（{{lang|de|Sturmgeschütz der Demokratie}}）。前德国外长根舍（Hans-Dietrich Genscher）曾这样评价奥格斯坦：“如果没有他，我们国家会完全不同，绝对不会如此自由开放。”

== 发展 ==
[[File:Hamburg.Spiegelgruppe.wmt.jpg|thumb]]）]]
从《明镜》创刊开始就充满了压力与争论，其与英国的主管一直在发行权的问题上纠缠不清。当《明镜》还没有成为德国的大众传媒领导者之前，该杂志就开始影响者人们的观点。经历过十年“民主突击炮”事件之后，《明镜》在德国传媒界站稳脚跟，成为真正的主流杂志。

之后《明镜》最大的竞争对手《焦点》杂志开始出版，《焦点》以其中性的观点赢得了不少的读者群。但是即便这样《明镜》杂志自始以来都没有改变自己的宗旨，还是以其批判性的特点拥有着大量的读者群。
== 假新闻虚假事件 ==
[[File:Sagen_was_ist.jpg|thumb]]
2018年12月19日《明镜》周刊爆出丑闻，称该杂志社一名德国获奖记者雷洛蒂斯在过去数年大规模制造[[假新闻|假新闻]]。该刊发现，最近数年由该名记者所撰写的多篇新闻，内容都是假的。《明镜》向读者道歉，自2011年由该记者所写的大约60篇报道中，有一些是准确的，但其他都是杜撰、错误引述的。

雷洛蒂斯自2011年起为《明镜》工作，承认当中最少有14篇含有部分虚假内容、虚构人物、对话和引述内容，部分报道曾经获奖，包括2014年CNN年度最佳记者、德国记者协会举办的年度最佳记者。《明镜》在网站发表声明，宣布革除记者雷洛蒂斯的职务。该事件也沉重打击《明镜》的声誉，震动新闻界。<ref>{{Cite web|url=https://tech.sina.cn/i/gj/2018-12-22/detail-ihqhqcir9282576.d.html?from=wap|title=外媒：《明镜》周刊记者造假丑闻震惊德国新闻界|accessdate=2018-12-22|author=|date=2018-12-22|publisher=新浪新闻}}</ref>

== 《明镜》在线 ==
1994年，《明镜》创办了自己的网络版，读者可以看到部分文章。2002年起，如果读者繳交一定的费用，就可以看到从第一版以来所有的《明镜》。
