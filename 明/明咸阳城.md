'''明咸阳城'''是[[明朝|明]][[清朝|清]]时[[咸阳县|咸阳县]]城，今[[咸阳市|咸阳市]]老城区所在。

明朝初年，咸阳县属[[西安府|西安府]]，[[唐咸阳城|县治]]在今[[兴平县|兴平县]]。[[洪武|洪武]]二年（1369年）<ref>《[[明史|明史]]·志第十八·地理三》西安府......咸阳府西北。旧治在渭河北，洪武二年徙於渭南。东北有泾水，东入渭。东南有沣水，北入渭。</ref>或四年（1371年）<ref name="a1"/>，迁治于渭水驿。[[景泰_(年号)|景泰]]三年（1452年），修建县城城墙。[[嘉靖|嘉靖]]二十六年（1547年），拓宽东西北城，共设九座城门<ref name="a1">{{Cite web |url=http://epaper.cnwest.com/xyrb/html/2012-07/17/content_4740788.htm |title=漫谈咸阳变迁史之三 |accessdate=2015-09-07 |author=刘为柏 |date=2012-07-17 |work= |publisher=西部网数字报刊，来源：咸阳日报数字报纸 |language=Zh-hans |deadurl=yes |archiveurl=https://web.archive.org/web/20160304122923/http://epaper.cnwest.com/xyrb/html/2012-07/17/content_4740788.htm |archivedate=2016-03-04 }}</ref>。此后至民国二十年（1931年），咸阳明城墙先后整修十次<ref>{{Cite web|url=http://epaper.cnwest.com/xyrb/html/2013-11/04/content_4807634.htm|title=明城墙遗址亟待保护|accessdate=2015-09-07|author=记者：李石|date=2013-11-04|work=|publisher=西部网数字报刊，来源：咸阳日报数字报纸|language=Zh-hans|deadurl=yes|archiveurl=https://web.archive.org/web/20160304201242/http://epaper.cnwest.com/xyrb/html/2013-11/04/content_4807634.htm|archivedate=2016-03-04}}</ref>，但至今仅[[渭城区|渭城区]]抗战南路留存两段共计二十余米的遗址。2010年，渭城区人民政府以“咸阳县东城墙遗址”的名称列为“区级重点文物保护单位”<ref>{{Cite web |url=http://xianyang.hsw.cn/system/2013/12/17/051818762.shtml |title=咸阳明城墙遗址边盖了间厕所 文旅局:已要求尽快拆除 |accessdate=2015-09-07 |author=记者：张林，编辑：彭乐 |date=2013-12-17 |work= |publisher=华商网，来源：今日咸阳|language=Zh-hans }}</ref>。另有北门[[城楼|城楼]]保存，名为[[凤凰台_(咸阳)|凤凰台]]，为[[陕西省文物保护单位|陕西省文物保护单位]]。
