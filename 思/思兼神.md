'''思兼神'''（{{lang-ja|オモイカネ}}）為《[[日本書紀|日本書紀]]》裡的記述，《[[古事記|古事記]]》則記作'''思金神'''或'''常世思金神'''，《[[先代舊事本紀|先代舊事本紀]]》另作'''八意思兼神'''、'''八意思金神'''，他是[[日本神話|日本神話]]裡掌管[[智慧|智慧]]的文神。

== 概要 ==
關於'''思兼神'''最有名的故事便是[[天照大神|天照大神]]躲入[[天磐戶|天磐戶]]，頓時世界陷入一片漆黑，於是他向八百萬[[神祇|神祇]]獻策，使天照大神聽到外頭戲謔嘻笑及舞蹈樂音，推開磐戶窺看，立於一旁的[[天手力男神|天手力男神]]一把拉住她，大地才重現光明。此外在「[[天孫降臨|天孫降臨]]」的故事中，天照大神之孫[[天津彥彥火瓊瓊杵尊|天津彥彥火瓊瓊杵尊]]由[[高天原|天]]降臨[[葦原中國|葦原中國]]之際，天照大神命[[天兒屋命|天兒屋命]]、[[布刀玉命|布刀玉命]]、[[天鈿女命|天鈿女命]]、[[伊斯許理度賣命|伊斯許理度賣命]]、[[玉祖命|玉祖命]]等五位[[神祇|神祇]]陪伴在側。另外一位隨侍在側的，便是思兼神。

《[[先代舊事本紀|先代舊事本紀]]》卷第三天神本紀則提到，思兼神隨天津彥彥火瓊瓊杵尊降臨葦原中國，落於[[信濃國|信濃國]]，與其子[[天表春命與天下春命|天表春命]]同為信乃阿智祝部的祖先，另一子[[天表春命與天下春命|天下春命]]則為武藏秩父國造之先祖。此外，思兼神的胞妹為[[栲幡千千姫命|栲幡千千姫命]]。

通常[[日本人|日本人]]祭拜思兼神的目的是為了祈求學業、[[測驗|考試]]順利，還有在建築新房屋的[[動土|動土]][[儀式|儀式]]（手斧初）上也會祭拜思兼神，也有人在這種儀式上祭拜手置帆負命、彥狹知命或者當地的[[產土神|產土神]]（類似[[土地神|福德正神]]的日本神祇）等神明。另外，[[江戶時代|江戶時代]]後期的[[日本國學|國學]]專家[[平田篤胤|平田篤胤]]認為，思兼神跟[[天儿屋命|天兒屋命]]實為同一人。

== 解說 ==
思兼神乃[[高皇産靈神|高皇産靈神]]（{{ja}}タカミムスビ）之子，[[日文|日文]]名「思（{{ja}}オモイ）」做「思慮」解釋，「兼（{{ja}}カネ）」為「兼備」，而「八意（{{ja}}やごころ）」則是聰明過人、相當有[[智慧|智慧]]之意，由此可看出思兼神乃[[高天原|高天原]]眾神的智囊。

== 信仰 ==
在日本，供奉思兼神為主祭神的神社計有下述：
* 秩父神社（[[埼玉縣|埼玉縣]][[秩父市|秩父市]]）
* 阿智神社（[[長野縣|長野縣]][[下伊那郡|下伊那郡]]阿智村）
* 安布知神社（長野縣下伊那郡阿智村）
* 戶隱神社中社（長野縣[[長野市|長野市]]）
* 思金神社（[[神奈川縣|神奈川縣]][[橫濱市|橫濱市]]）
* 地主神社（[[京都市|京都市]][[東山區_(日本)|東山區]]）※並非主祭神，而是相殿神
* 氣象神社（[[東京都|東京都]][[杉並區|杉並區]]的高圓寺冰川神社境内社）
* 高濱神社（[[島根縣|島根縣]][[出雲市|出雲市]]）※合祀
* 天安河原宮（[[宮崎縣|宮崎縣]][[西臼杵郡|西臼杵郡]][[高千穗町|高千穗町]]）
