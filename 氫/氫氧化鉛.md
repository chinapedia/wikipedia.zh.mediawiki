'''氫氧化鉛'''，是一種[[無機化合物|無機化合物]]，其[[化學式|化學式]]為Pb(OH)<sub>2</sub>，是[[鉛|鉛]]的[[氫氧化物|氫氧化物]]，由一個2價的[[鉛|鉛]]和[[2|2]]個[[氫氧根離子|氫氧根離子]]([[氧|O]][[氫|H]]<sup>[[陰離子|-]]</sup>)組成。'''氫氧化鉛'''是一種[[鉛鏽|鉛鏽]](其他還有:[[一氧化鉛|一氧化鉛]]、[[二氧化鉛|二氧化鉛]]、[[四氧化三鉛|四氧化三鉛]])，氫氧化鉛常溫下為固體，外觀通常是白色的，當鉛放在水裡久了[[生鏽|生鏽]]生成的[[鉛鏽|鉛鏽]]含有'''氫氧化鉛'''。

== 制備 ==
氫氧化鉛可以由[[氫氧化鈉|氫氧化鈉]]或[[氫氧化鉀|氫氧化鉀]]與[[硝酸铅|硝酸铅]]反应制得<ref name="scribd" />
:Pb(NO<sub>3</sub>)<sub>2</sub> + 2NaOH → Pb(OH)<sub>2</sub> + 2NaNO<sub>3</sub>

[[鉛|鉛]]在[[水|水]]中(要含[[氧|氧]])也會緩慢的生成氫氧化鉛，若至於[[碳酸水|碳酸水]]則會加快這個反應<ref>Lenntech: [http://www.lenntech.de/pse/wasser/blei/blei-und-wasser.htm Blei (Pb) und Wasser]</ref>
:2Pb + O<sub>2</sub> + 2H<sub>2</sub>O → 2Pb(OH)<sub>2</sub>

{{citation needed|用其他方式制出的氫氧化鉛其純度都不高，會是一種[[混合物|混合物]]，含氫氧化鉛、[[氫氧化氧鉛|氫氧化氧鉛]]、[[硝酸氫氧化鉛|硝酸氫氧化鉛]]}}

== 性質 ==
氫氧化鉛是一種白色無定形粉末，兩性，它溶解在水中呈[[弱鹼性|弱鹼性]]。它在脫水溫度[[130|130]] [[攝氏溫標|°C]]和[[145|145]][[攝氏度|°C]]時會分解為[[一氧化鉛|一氧化鉛]]和[[水|水]]<ref name="scribd" />。

:Pb(OH)<sub>2</sub><sup><math>\xrightarrow{\ 145 ^o C \ }</math></sup>PbO + H<sub>2</sub>O

氫氧化鉛與[[二氧化碳|二氧化碳]]反應會形成[[碳酸鉛|碳酸鉛]]
:Pb(OH)<sub>2</sub> + CO<sub>2</sub> → PbCO<sub>3</sub> + H<sub>2</sub>O

氫氧化鉛接觸酸會形成對應的鉛鹽(如接觸[[硫酸|硫酸]]形成[[硫酸鉛|硫酸鉛]]、若接觸[[檸檬酸|檸檬酸]]也會形成[[檸檬酸鉛|檸檬酸鉛]]、接觸[[硝酸|硝酸]]則形成[[硝酸鉛|硝酸鉛]])
:Pb(OH)<sub>2</sub> + H<sub>2</sub>SO<sub>4</sub> → PbSO<sub>4</sub> + 2H<sub>2</sub>O
:3Pb(OH)<sub>2</sub> + 2C<sub>6</sub>H<sub>8</sub>O<sub>7</sub> → Pb<sub>3</sub>(C<sub>6</sub>H<sub>5</sub>O<sub>7</sub>)<sub>2</sub> + 3H<sub>2</sub>O
:Pb(OH)<sub>2</sub> + 2HNO<sub>3</sub> → Pb(NO<sub>3</sub>)<sub>2</sub> + 2H<sub>2</sub>O
當它溶解於[[強鹼|強鹼]][[溶液|溶液]]的羥基形成一個複雜的離子[Pb(OH)<sub>4</sub>]<sup>2−</sup>、Pb<sub>4</sub>(OH)<sub>4</sub><sup>4+</sup>、Pb<sub>3</sub>(OH)<sub>4</sub><sup>2+</sup>、Pb<sub>6</sub>O(OH)<sub>6</sub><sup>4+</sup>.<ref name="H">Von Egon Wiberg, Nils Wiberg, Arnold Frederick Holleman, "Inorganic Chemistry", Academic Press, 2001 [http://books.google.de/books?id=Mtth5g59dEIC&pg=PA916&dq=%22lead%28II%29+hydroxide%22&hl=de&ei=ttE_Ta_tN8fj4wbjrq31Ag&sa=X&oi=book_result&ct=result&resnum=2&ved=0CCwQ6AEwATgK#v=onepage&q=%22lead%28II%29%20hydroxide%22&f=false (Google books)].</ref> 和形成[[铅酸盐|铅酸盐]]<ref>Therald Moeller; Chemistry with inorganic qualitative analysis; ISBN 0125033508</ref><ref name="HoWi">Arnold F.
