'''氫氧化鉈'''化學式為TlOH，是[[鉈|鉈(I)]]的氫氧化物。與其它鉈(I)化合物同樣具有強毒性，并具有[[腐蝕性|腐蝕性]]。

== 製備 ==

在[[硫酸亚铊|硫酸亚铊]]溶液中加入等當量的[[氫氧化鋇|氫氧化鋇]]后濾去[[硫酸鋇|硫酸鋇]]，即得鹼性強度與[[氫氧化鈉|氫氧化鈉]]相近的TlOH溶液：

Tl<sub>2</sub>SO<sub>4</sub> + Ba(OH)<sub>2</sub>  → 2 TlOH + BaSO<sub>4</sub>↓

蒸發後之可得黃色的TlOH晶體<ref name=Wuji>無機化學（第三版）下冊 武漢大學 吉林大學等校編，曹錫章 宋天佑 王杏橋 修訂</ref>。

將氧化亞鉈溶於水也可以制得:

Tl<sub>2</sub>O + H<sub>2</sub>O → 2 TlOH

== 性質 ==

純淨的氫氧化亞鉈是無色柱狀結晶，在空氣中一部份被氧化而呈黃色。
可溶於水和乙醇，水溶液顯強[[鹼性|鹼性]]。弱于鹼金屬氫氧化物的強鹼，固體或溶液均易從空氣中吸收水蒸氣和[[二氧化碳|二氧化碳]]並能腐蝕玻璃<ref name=Wuji></ref>。

2 TlOH + CO<sub>2</sub> → Tl<sub>2</sub>CO<sub>3</sub> + H<sub>2</sub>O

其鹼的電離常數為：

TlOH {{unicode|⇌}}  Tl<sup>+</sup> + OH<sup>−</sup>,　　p''K''<sub>b</sub> = 0.5

加熱到100℃脫水生成黑色Tl<sub>2</sub>O（熔點597℃）<ref name=Wuji></ref>。

2 TlOH → Tl<sub>2</sub>O + H<sub>2</sub>O

在空氣中部份被氧化，被氯水、溴水等氧化生成氫氧化鉈，Tl(OH)<sub>3</sub>，其[[標準電極電勢|標準電極電勢]]為：

Tl(OH)<sub>3</sub> + 2e<sup>−</sup> {{unicode|⇌}}  TlOH + 2 OH<sup>−</sup>,　　''E''°= −0.05 V

氢氧化亚铊是离子化合物，[[电解|电解]]时从溶液中沉积出[[铊|铊]]。

== 用途 ==

氢氧化亚铊的水溶液可以作为制取其他亚铊化合物的原料，但不适用制备游离的氢氧化亚铊固体。氢氧化亚铊固体可以用在氧气的存在下铊和乙醇反应制得。
