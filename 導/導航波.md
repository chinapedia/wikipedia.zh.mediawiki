[[理論物理學|理論物理學]]中，'''導航波理論'''（英文：pilot wave theory）是與[[量子力學|量子力學]]相關的[[隱變量理論|隱變量理論]]中的第一個例子，由[[德布罗意|德布罗意]]於1927年提出。

其更現代的版本為[[玻姆詮釋|玻姆詮釋]]（Bohm interpretation），於1952年由[[玻姆|玻姆]]提出。有別於傳統[[哥本哈根學派|哥本哈根學派]]所採用機率波的詮釋，此一隱變量理論具有一定程度的爭議性，試圖將量子力學的實驗結果詮釋為一項[[決定論|決定性]]理論，以避免一些麻煩，如：瞬間的[[波函數塌縮|波函數塌縮]]以及[[薛丁格貓|薛丁格貓]]這樣的[[悖論|悖論]]。與其他[[量子力學詮釋|量子力學詮釋]]使用相同數學式，因此相關量子力學實驗證據也可支持本理論。

2009年後，法國科學家{{le|易芙斯·庫代|Yves Couder}}等人利用矽油滴等流體進行實驗，製造出類比單粒子量子系統的效果，將導航波理論「實體化」。此實驗結果讓部份科學家對傳統上機率波詮釋產生一定的懷疑。

== 導航波理論 ==
導航波理論是一[[隱變量理論|隱變量理論]]，因此具有以下特性：
* 此理論中存在有[[實在性|實在性]]（意指具有獨立於觀測者之外的實在物理性質）。
* 此理論是[[決定論|決定性]]的。

一粒子的[[位置|位置]]與[[動量|動量]]可被視為隱變量，因[[海森堡不確定原理|海森堡不確定原理]]指出觀測者無法同時精準測出此二值。在一堆粒子組合中，存在有相應的[[物質波|物質波]]，根據薛丁格方程式演化。每顆粒子具有決定性（非隨機）的軌跡，由波函數引導。整體來說，諸多粒子的粒子密度與波函數[[幅|幅]]相符合。此波函數不受粒子影響，並可為一「空波函數」（Empty wave function）。

此理論顯示出[[非定域性|非定域性]]（[[相對論|相對論]]限制波動等訊息傳遞速限為光速，而產生定域性限制），並且滿足[[貝爾定理|貝爾定理]]。有意思的地方是：此非定域性符合No-communication定理而無法用作[[超光速|超光速]]通訊。

== 數學基礎 ==
欲導出電子的德布羅意-玻姆波，量子[[拉格朗日量|拉格朗日量]]為

:<math>L(t)={\frac{1}{2}}mv^2-(V+Q),</math>

期中<math>Q</math>是與量子力相對應的[[勢|勢]]，此量子力對波函數對粒子的推動；積分是沿著電子實際行進的唯一路徑（有別於[[路徑積分|路徑積分]]）。此結果可導得玻姆[[傳遞子|傳遞子]]（Bohm propagator）的關係式：

:<math>K^Q(X_1, t_1; X_0, t_0) = \frac{1}{J(t)^ {\frac{1}{2}} } \exp\left[\frac{i}{\hbar}\int_{t_0}^{t_1}L(t)\,dt\right]</math>。

此傳遞子讓我們得以精準追蹤在量子勢<math>Q</math>影響下隨時間變動的電子。
