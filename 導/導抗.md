'''導抗'''（{{lang-en|Immittance}}）是[[導納|導納]]與[[阻抗|阻抗]]的統稱。

[[負載|負載]]共分為[[電阻|電阻]]（[[電阻器|電阻器]]）、[[電容|電容]]（[[電容器|電容器]]）和[[電感|電感]]（[[電感器|電感器]]）三種。[[電阻|電阻]]只會影響 <math>\frac {V}{I}</math>（導抗矢量）的[[實數|實數]]部份，而[[電容|電容]]和[[電感|電感]]則只會影響其[[虛數|虛數]]部份，而三者的配合能夠影響[[正弦曲線|正弦曲線]]式[[電源|電源]]的剎那[[電壓|電壓]]、[[電流|電流]]或[[功率|功率]]等參數。

== 純直流電：電導與電阻 ==

I 為[[電流|電流]]，V 為[[電壓|電壓]]。

* '''[[電導|電導]]'''
:: <math>G = \frac {I}{V}</math>
: G 為[[電導|電導]]。

* '''[[電阻|電阻]]'''
:: <math>R = \frac {V}{I}</math>
: R 為[[電阻|電阻]]。

電導與電阻互為[[倒數|倒數]]關係。

== 純交流電：電納與電抗 ==

* '''[[電納|電納]]'''
:* [[容納|容納]] 
:: <math>B_C = \omega C = 2 \pi f C</math>
: X<sub>L</sub> 為[[容納|容納]]， ω 為導抗矢量角速度， f 為頻率， C 為[[電容|電容]]。

:* [[感納|感納]] 
:: <math>B_L = \frac {1}{\omega L} = \frac {1}{2 \pi f L}</math>
: X<sub>C</sub> 為[[感納|感納]]， ω 為導抗矢量角速度， f 為頻率， L 為[[電感|電感]]。

* '''[[電抗|電抗]]'''
:* [[感抗|感抗]] 
:: <math>X_L = \omega L = 2 \pi f L</math>
: X<sub>L</sub> 為[[感抗|感抗]]， ω 為導抗矢量角速度， f 為頻率， L 為[[電感|電感]]。

:* [[容抗|容抗]] 
:: <math>X_C = \frac {1}{\omega C} = \frac {1}{2 \pi f C}</math>
: X<sub>C</sub> 為[[容抗|容抗]]， ω 為導抗矢量角速度， f 為頻率， C 為[[電容|電容]]。

電納與電抗互為[[倒數|倒數]]關係。

== 兩者組合：導納與阻抗 ==

j 為[[虛數單位|虛數單位]]。

* '''[[導納|導納]]'''
:: <math>Y = G + B j</math>
: Y 為[[導納|導納]]，G 為[[電導|電導]]，B 為[[電納|電納]]。

* '''[[阻抗|阻抗]]'''
:: <math>Z = R + X j</math>
: Z 為[[阻抗|阻抗]]，R 為[[電阻|電阻]]，X 為[[電抗|電抗]]。

導納與阻抗互為[[倒數|倒數]]關係。
