'''凌云'''（{{bd|1917年|6月29日|2018年|3月15日|L凌}}），[[浙江|浙江]][[嘉兴|嘉兴]]人，[[中国共产党|中国共产党]]及[[中华人民共和国|中华人民共和国]]官员，[[中华人民共和国国家安全部|中华人民共和国国家安全部]]首任部长<ref name=shishi>{{cite news|title=国家安全部原部长凌云逝世|url=http://news.ifeng.com/a/20180321/56922369_0.shtml|date=2018-03-21|agency=新华社|publisher=凤凰网}}</ref>。

==生平==
1938年4月，参加[[中国共产党|中国共产党]]领导的革命工作并加入中国共产党，先后在[[山西新军|山西新军]]教导第二师、山西青年军官教导团、晋绥军政民干部训练委员会工作。1939年12月到[[陕北公学|陕北公学]]学习，任校党总支委员。1941年8月起，历任[[中央直属机关工委|中央直属机关党委]]组织干事、保卫委员会秘书，[[中共中央社会部|中央社会部]]科员、科长，中央土改工作团团员、秘书。1948年8月，起历任中共[[潍坊|潍坊]]特别市委委员、市公安局副局长，中共[[济南市|济南市]]委委员、市人民政府委员、市政府党组成员、市公安局副局长、局长。1952年2月起，历任公安部一局副局长、局长，公安部党组成员、副部长<ref name=shishi/>。1961年，兼任改造右派分子工作领导小组成员，1962年该领导小组因给右派分子摘帽面过大而受到中共中央批评<ref>{{Cite web |url = http://history.sina.com.cn/bk/ggkfs/2013-11-25/104274881.shtml |title =往事：解决55万人“帽子”问题的历史波折  |author =  |date =2013-11-25  |publisher = 新浪 |language =  |accessdate =  |quote =  }}</ref>。

[[文革|文革]]爆发后，公安部被指责有一个“以[[罗瑞卿|罗瑞卿]]为首的地下黑公安部”。1966年6月5日，公安部党组副书记、副部长[[徐子荣|徐子荣]]，党组成员、副部长[[刘复之|刘复之]]、凌云，政治部主任[[尹肇之|尹肇之]]等人被先后宣布边工作边审查。8月后，调[[沈阳军区|沈阳军区]]副政委[[李震_(官员)|李震]]、二十一军政委[[施义之|施义之]]等人前来领导公安部的“文化大革命”，接着又调来41名军队干部对公安部实施没宣布军管的“军管”。12月12日，中共中央批准将徐子荣撤职审查，刘复之、凌云、尹肇之停职反省。公安部领导几乎全部被停止工作。1968年2月7日，中共中央批准[[谢富治|谢富治]]《关于公安部改组情况的第一次报告》，成立公安部领导小组，原公安部干部除留85人（其中40人为借用）外，其余1000余人全部集中到中央政法干校办“学习班”接受审查。“学习班”期间，公安部干部受到[[法西斯|法西斯]]式审查，225名干部、职工被诬为“叛徒”、“特务”、“反革命”、“走资派”，45人被逮捕入狱（其中包括副部长徐子荣、[[杨奇清|杨奇清]]、[[汪金祥|汪金祥]]、凌云、[[严佑民|严佑民]]及17位局长、副局长)。1968年5月28日，谢富治、李震等人向中共中央提交报告称，挖出“地下黑公安部”，“公安部属于敌我矛盾的重点分子中有副部长7名，正副局长43名，正副处长63名”，并再次提出要“彻底砸烂地下黑公安部”。1975年1月，[[第四届全国人民代表大会第一次会议|第四届全国人民代表大会第一次会议]]任命国务院副总理[[华国锋|华国锋]]兼公安部部长，恢复[[于桑|于桑]]的副部长的职务。同年10月，任命杨奇清、严佑民、凌云、刘复之、[[黄庆熙|黄庆熙]]为副部长<ref name=wengejg>{{Cite web |url =http://www.hprc.org.cn/gsw/detail_zzslk.jsp?channelid=75034&record=2892  |title = 组织史资料 附卷一 中部 第一编　中华人民共和国中央政权组织机构 第二章　中央政权组织工作机构 第二节　国务院工作机构 |author =  |date =  |publisher =中华人民共和国国史网  |language =  |accessdate =2018-03-21  |quote =  }}</ref>。 

文革结束后，凌云任公安部党组副书记、副部长，1978年起兼[[中国人民警官大学|国际政治学院]]党委书记、院长。1983年7月至1985年9月任国家安全部党组书记、部长<ref name=shishi/><ref>{{Cite web |url =http://www.npc.gov.cn/wxzl/gongbao/2000-12/12/content_5001697.htm  |title =全国人民代表大会常务委员会决定任免的名单（1985年9月）  |author =  |date = 2000-12-12 |publisher = 中国人大网  |language =  |accessdate =  |quote =  }}</ref><ref>[[俞强声|俞强声]]出走美国事件发生在1986年，和凌云卸任无关</ref>。2004年9月30日离职休养。凌云是[[中共十二大|中共十二大]]代表，第三届、第五届[[全国人大|全国人大]]代表<ref name=shishi/>。

2018年3月15日，凌雲在北京病逝，終年100歲。新華社發稿形容他為「中國共產黨的優秀黨員，久經考驗的忠誠的共產主義戰士」。<ref name=shishi/>。
