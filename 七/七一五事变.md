'''七一五-{}-事变'''（[[中国共产党|中国共产党]]称之为'''七一五反革命政变'''；[[中国国民党|中国国民党]]称之为'''武汉-{}-分共'''，或視之為「[[清党|清党]]」的一個環節）爆发于1927年7月15日。在[[中國國民黨|中國國民黨]]進行[[国民革命军北伐|国民革命军北伐]]（1926年至1928年）時，原先採取「[[聯俄容共|聯俄容共]]」的方針，與[[中國共產黨|中國共產黨]]合作。但在北伐期間，國民党內部就容納共產黨與否的問題，於1927年分成[[國民政府#南京（1927年4月18日—1937年11月20日）|南京]]（[[蔣中正|蔣中正]]等領導）及[[國民政府#武漢（1927年2月21日—1927年8月）|武漢]]（[[汪兆銘|汪兆銘]]等領導）兩个[[国民政府|国民政府]]，史稱[[寧漢分裂|寧漢分裂]]。武漢方面最初仍主張容共，其後與共產黨發生磨擦，再加上[[馮玉祥|馮玉祥]]等軍事實力派人士的表態要求分共，汪兆銘乃於7月15日當日，在[[武汉|武汉]]召集会议，宣布停止[[国共合作|与中国共产党的合作]]。至此第一次国共合作正式结束。

== 背景 ==
[[File:七一五之前的汪兆銘.jpg|thumb]]

=== 四一二事件之前的国共关系与汪精卫 ===
虽然在[[孙中山|孙中山]]的联共政策下，国共两黨进行合作，但由于两党对于中国革命的终极目标不一致等原因，从[[西山会议|西山会议]]到[[中山舰事件|中山舰事件]]，两党间一直存在摩擦。在[[国民革命军北伐|国民革命军北伐]]期间由共产党领导的[[中国工人运动|中国工人运动]]使一部分原本支持国民政府的资本家的利益受损；而[[土地革命|土地革命]]更是造成了[[国民革命军|国民革命军]]中一部分[[地主|地主]]家庭出身的将士的不满{{NoteTag|如[[何键|何键]]出身地主家庭，土地革命期间其父被共产党游街。}}。

在此情形下国民党之中出现与共产党分袂的声音。[[蒋中正|蒋中正]]等[[国民党右派|国民党右派]]主张尽早采用武力方式“清共”。而对此有所觉察的[[国民党左派|国民党左派]]、中共和[[共产国际|共产国际]]代表等决定拥举此前受排挤出国、而名义上在任的[[国民政府主席|国民政府主席]][[汪精卫|汪精卫]]出来对抗。<ref>《蔣介石三次下野秘录》，59页</ref>

1927年4月1日汪精卫回国，抵达被[[北伐军|北伐军]]占领的[[上海|上海]]。4月3日蒋中正、[[李宗仁|李宗仁]]等人先后与其见面，劝说其以国府主席身份领导国民党分共；但汪精卫表示他“站在工农一边”<ref name="《真实的汪精卫》">《真实的汪精卫》，第二部分之三：从拥共到反共</ref>，主张暂且容共。4月4日举行的国民党在沪高官会议上，甚至出现了[[吴稚晖|吴稚晖]]哀求汪精卫而下跪的场面，令人啼笑皆非<ref>《[[李宗仁回忆录|李宗仁回忆录]]》，323页</ref>。在蒋中正等人的劝解下，会议继续进行。会议最后就4月15日在[[南京|南京]]召开二届四中全会解决国共纠纷一事达成几点共识：
{{quote|
会议依汪同志主张，望再暂时隐忍，出于和平解决之途，以四中全会解决纠纷。然在此会议召集前，依蒋同志之主张，采诸项应急之法。
* 一曰通告中共领袖[[陈独秀|陈独秀]]，立即制止国民政府辖区之共产党员，暂时停止一切活动，听候开会解决；
* 二曰对中央党部之国民政府迁[[鄂|鄂]]后因被共党操纵，所发命令不能健全，若以为妨碍党国前途者，在四中全会召集前，可不接受之；
* 三曰军队应饬令各该管长官，取缔有共党捣乱之党部、团体。
* 工人纠察队及其他武装团体，应服从总司令（蒋介石）指挥，否则应视为反革命，严行取缔。<ref>1927年4月8日《[[申报|申报]]》</ref>
}}

然而，汪精卫为了争取政治上的主动，于4月5日与[[陈独秀|陈独秀]]发表了《[[国共两党领袖汪兆铭、陈独秀联合宣言|国共两党领袖汪兆铭、陈独秀联合宣言]]》，声称国共将合作到底。这份宣言激怒了反共的国民党人，吴稚晖更讥讽汪精卫要做“国民党的[[家长|家长]]”。当晚汪精卫离开上海，前往共产党和国民党左派占多数的[[武汉国民政府|武汉国民政府]]。

=== 四一二事件和宁汉分裂 ===
{{Main|四一二事件|宁汉分裂}}

汪精卫以召集在汉的中央委员到南京开二届四中全会名义，於4月10日抵达[[武汉|武汉]]，与[[唐生智|唐生智]]、[[谭延闿|谭延闿]]等人会合。他原本以为会议会正常举行，不料蒋中正联合了[[胡汉民|胡汉民]]、[[李济深|李济深]]等势力，於4月12日开始进行“清共”，并另行组建了[[南京国民政府|南京国民政府]]。震怒之下汪精卫以国民党中央执行委员会名义，宣布对蒋中正“开除党籍，免去本人所兼各职”<ref>1927年4月17日[[中国国民党中央执行委员会|中国国民党中央执行委员会]][[通电|通电]]</ref>。4月18日，南京国民政府成立当天，武汉召开群众大会，汪精卫发表演说，对蒋中正大加鞭挞。宁汉正式分裂。

为避免武汉“东征”，李宗仁及部下[[应榆汉|应榆汉]]分别到[[江西|江西]]和武汉，对汪精卫和汉方的实力派[[朱培德|朱培德]]进行劝说。宁汉双方得以在互不干涉的情况下继续北伐{{NoteTag|1927年4月19日，武汉国民政府以[[唐生智|唐生智]]为北伐总指挥，[[冯玉祥|冯玉祥]]为第二集团军司令，[[阎锡山|阎锡山]]为第三集团军司令，继续北伐。5月1日，蒋中正、李宗仁等也继续北伐。}}。

== 国共合作走向全面破裂 ==

=== 武汉内部的反共 ===
{{main|马日事变}}

武汉在四一二事变之后，国共关系仍甚为紧密，经常举行“国共联席会议”<ref>《李宗仁回忆录》，333页</ref>。汉方辖区内工农运动也继续展开，但这些激起了一些国军将士，特别是地主或工商业出身的子弟不满。5月21日国军第三十五军三十三团团长[[许克祥|许克祥]]就已发动[[马日事变|马日事变]]。不久独立第十四师师长[[夏斗寅|夏斗寅]]也发动反共兵变。虽然武汉国民政府对此明令讨伐，但在[[何键|何键]]等人的偏袒下，没有实际效果。许、夏两人转投南京，导致武汉国民政府失去了对湖北一部和湖南大部的控制<ref name="《中苏关系史纲》">[http://news.xinhuanet.com/theory/2008-01/04/content_7359363.htm 《中苏关系史纲》 第二章 苏联援助下的国民革命  “五月指示”与国共分家]</ref>。6月29日武汉警备司令[[李品仙|李品仙]]和何键也公开反共，解除了汉口工人纠察队武装<ref name="李宗仁回忆录">《李宗仁回忆录》，338页</ref>。

=== [[共产国际|共产国际]]的五月指示 ===
6月1日，共产国际代表[[罗易|罗易]]携一份《关于中国问题之决议》（五月指示）到达武汉。出于信任，他把这份指示先拿给汪精卫看。内容大意如下：
{{quote|
#坚决主张从下面实际占领土地，对此必须千方百计地加以促进。但必须同过火行为作斗争，不要触及军官和士兵的土地，要对手工业者、商人和小地主做出让步；
#动员两万共产党员和两湖地区五万革命工农，组建自己的军队；
#从下层吸收新的工农领导人加入国民党中央，改变国民党目前的构成。有旧思想的一律驱逐；
#成立以著名国民党人和非共产党人为首的革命军事法庭，惩办反动军官。<ref>《[[联共|联共]]（布）中央政治局会议第107号（特字第85号）记录》，1927年6月2日，《联共（布）、共产国际与中国国民革命运动（1926－1927）》（下），第298－299页。</ref>
}}
汪精卫认为这几个指示会让国民党灭亡。但还是进行了讨价还价，提出要[[苏俄|苏俄]]提供1500万[[卢布|卢布]]贷款，而[[莫斯科|莫斯科]]同意提供200万援助。<ref>《[[斯大林|斯大林]]给[[莫洛托夫|莫洛托夫]]的信》，1927年6月24日；《联共（布）中央政治局会议第113号（特字第91号）记录》，1927年6月27日，《联共（布）、共产国际与中国国民革命运动》（下），第352，364页。</ref>其后汪精卫还对[[苏联|苏联]]顾问[[鲍罗廷|鲍罗廷]]宣泄不满，并要驱逐其回国。<ref name="《真实的汪精卫》" />
[[File:1927年冯玉祥与蒋介石会面.jpg|thumb]]

=== [[冯玉祥|冯玉祥]]反共 ===
《五月指示》让汪精卫开始决定分共，但他希望能争取到宁汉之外的第三方[[冯玉祥|冯玉祥]]的支持，从而达到分共制蒋的一箭双雕效果。为此在中共不知情的情况下<ref>《蔣介石三次下野秘录》，74页</ref>，他派遣[[邓演达|邓演达]]联络冯玉祥，於6月10日-6月12日与冯在[[郑州|郑州]]会晤。会上，尽管汪精卫同意给予冯玉祥尽可能的实惠，但冯玉祥坚持汪精卫先行分共。当时唐生智与[[张作霖|张作霖]]在[[河南|河南]]大战后元气大伤，而冯玉祥渔翁得利占领了河南，且在地理上接近湖北。汪精卫只能迁就於冯玉祥。<ref>《蔣介石三次下野秘录》，75页</ref>

不料冯玉祥另遣[[李鸣钟|李鸣钟]]与蒋中正联络，并於6月19日与蒋在[[徐州|徐州]]另行会议。比起武汉答应的一些虚职，蒋中正在财政上给予冯更多的实惠，且不强求一致对付汉方，令冯玉祥大为满意，乃至与蒋中正“相见恨晚”<ref>冯玉祥自传《我的生活》：徐州与蒋中正一见“见其风采及言谈态度，无不使我敬慕，大有相见恨晚之情”</ref>。会后冯玉祥以两人名义发[[电报|通电]]，要求武汉早日驱逐鲍罗廷，实行[[清党|清党]]，完成北伐等等。这使得武汉方面更加被动。

== 事件经过 ==

=== 中共的对策 ===
关于《五月指示》，中共内部也有不同意见。陈独秀刚看到它时，也认为它不符合中国实际，并回电报表示暂难执行。这引来了[[联共|联共]]和共产国际的不满<ref>{{cite web |title=1927年大革命失败的主要原因与陈独秀的责任 |url=http://www.66wen.com/07lsx/lishixue/lishixue/20080408/53551_3.html |publisher=66论文 |date=2008-04-08 |archiveurl=https://archive.is/20120719192136/http://www.66wen.com/07lsx/lishixue/lishixue/20080408/53551_3.html |archivedate=2012-07-19 |deadurl=yes }}</ref>。6月30日中央扩大会通过了《国共合作十一条》，开始建议共产党员以“请假”方式脱离武汉国民政府。<ref>{{cite web |author=[[统战部|统战部]] |title=第一次国共合作破裂的原因及其教训 |url=http://cpc.people.com.cn/GB/64107/65708/65722/4445288.html |publisher=中国共产党新闻网 |date=2007 |accessdate=2016-12-25}}</ref>

7月4日中共在[[汉口|汉口]][[三教街|三教街]]紧急召开[[政治局|政治局]]扩大会议，总书记陈独秀和鲍罗廷均主张暂不应执行这份指示<ref name="《真实的汪精卫》" />。但遭到了激进派的反对。此外，陳獨秀與鮑羅廷又決定在[[兩湖|兩湖]]地區停止工農運動，以重建[[國民黨|國民黨]]左派與共產黨的聯合戰線。共產國際認為陳獨秀、鮑羅廷以停止工農運動來換取在國民黨內苟存，是犯了「機會主義」的錯誤，急令鮑羅廷返[[俄國|俄國]]。7月8日共产国际要求中共党员退出武汉国民政府。7月12日陈独秀被解除[[中共总书记|中共总书记]]职务，另组成[[周恩来|周恩来]]、[[李立三|李立三]]、[[李维汉|李维汉]]、[[张太雷|张太雷]]、[[张国焘|张国焘]]为首的五人小组领导全局（后加入[[瞿秋白|瞿秋白]]）。據學者[[張玉法|張玉法]]所說，共產國際的目的是要中共「準備實行武裝暴動，以與國民黨爭奪革命領導權」。7月13日，中共发表宣言，正式退出国民政府，又責[[汪精衞|汪精衞]]是「公開贊助反革命」。<ref name="《中苏关系史纲》" /><ref>張玉法《中國近代現代史》第十二章第五節，民國八十八年（1999年）版，ISBN 957-636-217-2，348-349頁。</ref>

=== 7月15日 ===
当天汪精卫主持召开国民党会议。汪精卫在会上公开了《五月指示》，指责共产党的政策，但主张以和平方式遣散共产党人。这一主张得到了大多数人的赞同，只有[[宋庆龄|宋庆龄]]的代表[[陈友仁|陈友仁]]抗议并退场。会议最后针对中共的宣言通过了《统一本党政策案》，要国民党和军队中的共产党人立即声明脱离共产党，否则予以停职。但汪精卫亦表示无意采取暴力分共。 <ref name="《真实的汪精卫》" />武漢政府更派員到[[莫斯科|莫斯科]]，討論切實聯合辦法，保護中共黨員的個人身體自由。在當天，汪精衞得知了7月13日中共發表的宣言，次日，汪精卫又以政治委员会主席团名义，抨击共产党“破坏革命”，并将武汉国民政府内的共产党员一律停职。<ref name="李宗仁回忆录" /><ref name="《近代中國史綱》">郭廷以《近代中國史綱》（下冊）第十五章第三節，香港中文大學出版社1996年版，ISBN 962-201-352-X，562頁。</ref>

在武漢政府分共的同時，又没有放弃讨蒋。仍视蒋中正为“党国唯一的敌人”<ref>《蔣介石三次下野秘录》，76页</ref>。

== 后果和影响 ==
=== [[南昌起事|南昌起事]]及镇压 ===
武漢國民政府於1927年7月15日的進行分共事件，令其與中共的矛盾升溫；7月19日，中共及[[共青團|共青團]]對國民黨及汪精衞續行攻擊；26日，國民黨採取嚴厲措施，免去各機關中的中共黨員職務<ref name="《近代中國史綱》" />。决定组建自己武装的中共於8月1日在南昌发动起事，而中共方面则一直称大革命的失败才导致了军队建立<ref>[http://cpc.people.com.cn/GB/69112/69114/69126/4741684.html 《建军报告》 陈毅 一九四五年三月一日]</ref>。武汉方面决定予以镇压，被舆论评价“较南京政府之清党似犹过之”<ref>《蔣介石三次下野秘录》，81页</ref>。[[中国大陆|中国大陆]]的[[历史教科书|历史教科书]]等用“宁可枉杀千人，不可使一人漏网”形容当时的血腥景象，但没有明确提及这句话是在什么时候说的，且认为是汪精卫本人的发言。据[[谢觉哉|谢觉哉]]回忆，湖北曾有“宁肯错杀三千，不肯放松一个”的布告<ref>《谢觉哉日记》906页</ref>。而当时国民革命军第19军副军长[[陶钧|陶钧]]最以此发言闻名<ref>郑奇，《腥风血雨的一九二八年》，《武汉文史资料》1983年第1辑</ref>。汪精卫本人至少在南昌起义前，一直否认主张对共产党开杀戒{{NoteTag|1927年7月25日，汪在政治委员会第四十次会议上发言:“我们几时杀过人？他们为什么要这样凭空造谣……”}}。

=== [[宁汉合流|宁汉合流]] ===
汉方在分共的同时，并未放弃讨蒋。而蒋中正主持北伐不利，徐州失守。李宗仁为首的[[桂系|桂系]]又软硬兼施<ref>[[中国第二历史档案馆|中国第二历史档案馆]]藏《北京政府陆军部档案》：“闻此次南京内变，李宗仁实为主动之一。”</ref>，终于逼迫蒋中正与[[胡汉民|胡汉民]]、[[张人杰|张人杰]]等人决定下野。李宗仁领衔发电庆祝汉方分共，当时蒋中正仅列名第四位<ref>[[董显光|董显光]]《蒋总统传》，1958年[[台北|台北]]出版</ref>。8月13日，蒋中正正式下野。8月底9月初，武汉国民政府机关陆续迁往南京。
