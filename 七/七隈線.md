'''七隈線'''（{{jpn|j=七隈線|hg=ななくません|rm=Nanakuma sen|f=y}}）是日本九州[[福岡縣|福岡縣]][[福岡市|福岡市]]內一條連接[[西區_(福岡市)|西區]]的[[橋本車站_(福岡縣)|橋本車站]]與[[中央區_(福岡市)|中央區]]的[[天神南車站|天神南車站]]兩車站的[[地下鐵|地下鐵]]路線，由[[福岡市交通局|福岡市交通局]]營運。雖然平常營運上都是使用七隈線的稱呼，但在「福岡市交通事業設置相關條例」（{{lang|ja|福岡市交通事業の設置等に関する条例}}）中，本路線的名稱為'''3號線'''。

除了既有已經通車的路段外，該路線正在進行天神南站與[[博多車站|博多車站]]之間的延伸線工程，該路段預計在2022年時通車。

== 路線資料 ==
* 路線距離（{{link-ja|營業距離|営業キロ}}）：橋本－天神南間 12.0公里
* [[軌距|軌距]]：1435毫米
* 站數：16個
* 複線路段：全線
* 電氣化路段：全線（[[直流電|直流電]]1500V）
* [[閉塞_(鐵路)|閉塞方式]]：車內信號式

== 簡介 ==
七隈線是一條使用鋼輪式[[線型馬達|線型馬達]]作為列車驅動方式的地下鐵路線，屬於日本所謂的「迷你地下鐵」（{{lang|ja|ミニ地下鉄}}，一種類似其他國家中運量系統、使用較小型車輛運輸旅客的設計）。但有別於福岡市地下鐵另外兩條路線——[[福岡市地下鐵機場線|機場線]]（{{lang|ja|空港線}}）與[[箱崎線|箱崎線]]所使用的1067mm窄軌軌距，七隈線使用了1435mm的標準軌軌距，並且與另兩條線互不相通。

== 历史 ==
* 2005年2月3日：橋本 - 天神南間通車。
* 2009年3月7日：沿線所有車站正式導入福岡市交通局所發行的智慧型IC票卡[[快捷卡|快捷卡]]（{{lang|ja|はやかけん}}）。
* 2016年11月8日：七隈線線延伸段在博多站附近的工地發生地陷。
