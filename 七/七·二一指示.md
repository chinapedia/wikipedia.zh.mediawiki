'''七·二一指示'''，是1968年7月21日[[毛泽东|毛泽东]]在《[[人民日报|人民日报]]》关于《从上海机床厂看培养工程技术人员的道路（调查报告）》的编者按清样中加写的一段话，对[[中华人民共和国教育|中华人民共和国高等教育]]体制产生了重大影响。

==背景==
1958年，毛泽东提出教育“必须同生产劳动相结合”。1960年起，毛泽东对教育的批评增加<ref name=yjlb/>。1966年5月7日，毛泽东审阅[[中国人民解放军总后勤部|中国人民解放军总后勤部]]《关于进一步搞好部队副业生产的报告》后致信[[林彪|林彪]]，此信后被称为“[[五七指示|五七指示]]”，毛泽东在信中要求全国各行各业都要办成一个大学校<ref>{{Cite web|url = http://agzy.youth.cn/qsnag/zxbd/201505/t20150506_6618166.htm|title = 纪念毛主席“五七指示”49周年|author = |date = 2015-05-06|publisher = 中国青年网|language = |accessdate = |quote = }}{{Dead link|date=2019年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref>{{Cite web |url = http://news.163.com/14/0814/15/A3KA5GC500014SEH.html |title =我观《邓小平》你知道吗，文革时期初高中各只读两年  |author =  |date =2014-08-14  |publisher =  网易|language =  |accessdate =  |quote =  }}</ref>。

1966年“[[文化大革命|文化大革命]]”爆发后，高等院校及中等学校学生纷纷[[大串联|大串联]]及参加造反，高等院校陆续停办裁撤，人才培养陷入停滞，各行各业人才严重短缺。尽管1967年《关于大、中、小学校复课闹革命的通知》提出“[[复课闹革命|复课闹革命]]”，但高等院校多已人去楼空<ref name=yjlb/>。

[[文化大革命|文化大革命]]初期，1968年[[春节|春节]]后，记者组遵照上海市的意图来到[[上海机床厂|上海机床厂]]调查“文化大革命”“[[斗、批、改|斗、批、改]]”情况，上海的《[[文汇报|文汇报]]》与中央的[[新华社|新华社]]记者共同写出了《从上海机床厂看培养工程技术人员的道路（调查报告）》。当时，上海机床厂的青年技术员中，大专院校毕业生约350人（其中10%是研究生及留学生），从工人中提拔的技术人员约250人（其中少数曾被派往中等专科学校进修）。调查报告中称：“过去[[走资派|走资派]]竭力把[[反动学术权威|反动‘权威’]]捧为青年技术人员的学习‘偶像’，要他们‘向专家看齐’，‘为当工程师而奋斗’。现在，许多青年技术人员思想面貌发生了显著变化，认识到名利思想是‘[[修正主义|修]]’字的根，[[资产阶级|资产阶级]]的头衔不能争。”调查报告的结论是：“从工人中选拔技术人员，这是一条培养无产阶级工程技术人员的道路。”<ref name=yjlb/>

调查报告还提供例证：一位14岁当学徒的工人，“18岁被保送到上海机器制造学校学习4年，1957年开始在磨床研究所担任技术员。今年4月，由他担任主任设计师，试制成功了一台具有国际先进水平、为我国工业技术的发展所急需的大型平面磨床。”而另一方面，“一个是上海某大学的毕业生，毕业后又专门学了一年外语，再去外国留学4年，得了一个‘[[副博士|副博士]]’的学位，1962年开始到磨床研究所实验室担任技术员。像这样一个在学校里读了20多年书的人，过去由于理论脱离实际，又没有很好地同工人结合，所以长时期在科学研究方面没有作出突出的成就。”调查报告进而提出“教育革命的方向”是“学校教育要与生产劳动相结合，学校要由有经验的工人去当教师，让工人登上讲台。有些课程就可以在车间里由工人讲授”，“由基层选拔政治思想好的，具有两三年或四五年劳动实践经验的初、高中毕业生进入大专院校学习”<ref name=yjlb/>。

但上述调查报告多有不实之处。实际上自1950年起，工人便已进入上海机床厂的技术队伍。1997年出版的《上海成人高等教育志（1863-1990)》载，截至1965年，上海机床厂636名技术人员中，有272名是从工人中提拔的。1966年6月文革爆发后，上海机床厂业余大学停办。调查报告中的副博士是留学[[苏联|苏联]]归国的曹婉倩，而“具有国际先进水平”的平面磨床正是采用了曹婉倩研发出来的轴承<ref name=yjlb/>。

== 指示内容及发表经过 ==
1968年7月21日，[[中国共产党中央委员会主席|中国共产党中央委员会主席]][[毛泽东|毛泽东]]在《[[人民日报|人民日报]]》关于《从[[上海机床厂|上海机床厂]]看培养工程技术人员的道路（调查报告）》的[[编者按|编者按]][[清样|清样]]中加写道：“大学还是要办的，我这里主要说的是理工科大学还要办，但学制要缩短，教育要革命，要[[无产阶级|无产阶级]]政治挂帅，走上海机床厂从工人中培养技术人员的道路。要从有实践经验的工人农民中间选拔学生，到学校学几年以后，又回到生产实践中去。”这段话即“七·二一指示”。1968年7月22日，《人民日报》发表了该调查报告及编者按。编者按称，该调查报告“提出了学校教育革命的方向”<ref>{{cite web |url = http://cpc.people.com.cn/GB/4162/64165/67447/68000/4613640.html  |title =  1968年7月21日 毛泽东对《从上海机床厂看培养工程技术人员的道路》作批示 |author =  |date =  |website = 中国共产党新闻网 |language =  |accessdate = 2017-07-23  |quote =  }}</ref><ref name="yjlb">{{cite web |url = http://news.sina.com.cn/c/2009-07-27/181118307641.shtml  |title = 1968：工厂里的特殊大学  |author =  |date = 2009-07-27 |publisher = 新浪 |language =  |accessdate =  |quote =  }}</ref><ref> {{cite book |editor = 刘英杰 主编 | title = 中国教育大事典（上册）  | location =  | publisher = 浙江教育出版社 |year = 2004年 | pages =  | ISBN =  | accessdate =  | url =  |language =  | quote = 创办“七·二一”大学 }} </ref>。

==影响==
“七·二一指示”对文革时期中华人民共和国高等教育体制产生了重大影响，主要有如下方面：
*[[职工大学|七·二一大学]]：各地厂矿企业开办的职工大学，从职工中选拔学员，在“七·二一大学”学习几年后，又回到生产岗位上继续当职工。1968年9月，上海机床厂为贯彻“七·二一指示”，创办了“七·二一大学”，这是中华人民共和国全国第一所“七·二一大学”<ref name=sht>{{Cite web |url = http://www.shtong.gov.cn/newsite/node2/node2245/node4471/node56291/node56302/node56304/userobject1ai42881.html |title =上海工运志 第二十篇职工教育 第四章高等教育 第二节　职工大学  |author =  |date =  |publisher = 上海通 |language =  |accessdate =2017-07-23  |quote =  }}</ref><ref name=yjlb/>。1975年6月，[[中华人民共和国第一机械工业部|中华人民共和国第一机械工业部]]和[[中华人民共和国教育部|中华人民共和国教育部]]在上海召开“全国七·二一工人大学教育革命经验交流会”，称其“体现了社会主义教育事业的发展方向”<ref name=sht/>。此后“七·二一大学”在全国获得迅猛发展<ref name=gsm>{{Cite web |url = http://www.wyzxwk.com/Article/lishi/2015/04/341164.html |title = 郭松民：“七二一工人大学”的原则是永存的 |author =  |date = 2015-04-01 |publisher = 乌有之乡 |language =  |accessdate =  |quote =  }}</ref><ref name=yjlb/>。
*[[工农兵上大学|工农兵上大学]]：高等院校废除考试制度，通过“群众推荐、领导批准、学校复审相结合”的办法招收工农兵学员。

==否定==
1977年8月8日，[[邓小平|邓小平]]在[[科学和教育工作座谈会|科学和教育工作座谈会]]上讲话称：“教育还是要两条腿走路。就高等教育来说，大专院校是一条腿，各种半工半读的和业余的大学是一条腿，两条腿走路。”1977年9月19日，邓小平在同[[方毅|方毅]]、[[刘西尧|刘西尧]]、[[雍文涛|雍文涛]]、[[李琦_(磁县)|李琦]]等人的谈话中指出：“毛泽东同志的七二一指示要正确地去理解。七二一大学、[[江西农业大学|共产主义劳动大学]]，各省自己去搞，办法由他们自己定，毕业生不属国家统一分配范围。但是清华大学、北京大学恐怕不能这样办，并不是所有大学都要走上海机床厂的道路。”<ref>{{Cite web |url = http://www.ce.cn/xwzx/gnsz/szyw/200708/23/t20070823_12646522_1.shtml |title =珍闻:1977年邓小平关于恢复高考的讲话和批示选载  |author =  |date = 2007-08-23 |publisher =中国经济网  |language =  |accessdate =  |quote =  }}</ref>

此后，各地“[[职工大学|七·二一大学]]”进行了大规模调整整顿，另外中国还恢复了[[高考|高考]]制度。

1984年，中共上海机床厂党委做出决定，彻底否定了1968年的《从上海机床厂看培养工程技术人员的道路（调查报告）》<ref name=gsm/><ref name=yjlb/>。
