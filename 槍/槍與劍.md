《'''槍與劍'''》（{{lang-ja|'''ガン×ソード'''}}、'''Gun x Sword'''）是一部[[日本|日本]][[動畫|動畫]]，於2005年7月至12月在[[日本|日本]][[東京電視台|東京電視台]]播放，總共有26話。

== 概要 ==
宣傳詞為「'''痛快娛樂復仇劇'''」。因為有優秀的作畫、白熱化的故事發展以及聲優們的熱烈演出而受到好評。

[[秋田書店|秋田書店]]所發行[[漫畫|漫畫]]雜誌「[[週刊少年Champion|週刊少年Champion]]」，有本作的漫画版（劇本：兵頭一步・作画：{{lang|ja|ひのき一志}}）並且短期集中連載（全8話、全1卷）。動畫版和漫畫版的設定有所差異（特別是梵安的性格、裝甲設定方面以及故事的展開）、可說是兩款不同的作品（不過動畫版最終集，記者有碰觸到關於漫畫版的內容、似乎看起來是同一個世界的故事）。

另外[[幻冬舍Comics|幻冬舍Comics]]也將動畫版改編成漫畫版，並且發行「槍與劍-another-」之名（作画：守屋直樹・協力：谷口悟朗・倉田英之・木村貴宏・山根理宏）的單行本。標題和設定繼承原作故事、內容以梵安、溫蒂、卡爾曼、凱吉、布里希洛等人的外傳故事為主。

== 劇情簡介 ==
梵安因為未婚妻在婚禮當天被神秘的勾爪男殺死，所以不斷尋找他的下落，溫蒂·格雷特與卡爾曼99、喬西亞等人為了各自的理由，他們也因故了加入這趟追逐勾爪男的行列之中。

== 主要角色 ==
=== 梵安一行人 ===
[[File:Gun_X_Sword_Characters.jpg|thumb]]
* '''梵安'''({{lang|ja|'''ヴァン'''}}、'''Van''')[[配音員|配音員]]：[[星野貴紀|星野貴紀]]／[[康殿宏|康殿宏]]（台灣）
:男主角，一位有著數個稱號並且到處流浪的男人，在以勾爪男為主的組織中則是一名空缺的成員。與最愛的女人艾蓮娜結婚的當天，在教堂中遭到勾爪男偷襲並且當著他的面前殺害艾蓮娜，受重傷的艾蓮娜與卡德貝多將瀕死的邦動手術改造成與機械鎧同步的駕駛員(第十二集有敘述)，從此為了向勾爪男復仇而展開旅行。不過平常並不會展現自己復仇的決心，體型瘦長而且駝背，看起來總是一副慵懶的模樣，雖然容貌上不太年輕但行動卻相當迅速，戰鬥時能徹底發揮自己身體的力量。
:戴著一頂絕不脫下的寬邊高呢帽和穿著一件黑色燕尾服（似乎是提醒自己不要忘記復仇，被認為他其實穿的是一件喪服，但詳情不明）讓自己的造型彷彿是西部電影的神槍手。腰上所攜帶的武器看起來像是槍，實際上是一個彎刀。此刀就像是絲帶一樣能伸縮自如，使用時刀身會變得比鋼鐵還要堅銳。然後在機器人戰時，梵安會先以手指將帽子轉半圈，然後在天空中用刀揮出Ｖ字型，叫喚宛如彎刀形狀的機器人「Dann of Thursday」出來作戰。在最終集數年後梵安偶然拜訪溫蒂的房子時，身上帶著自己和一行人的合照，證明他沒有忘記那段時光仍然懷念。
:天生就無父無母，一身孤獨，童年時就像是饑餓的野獸一樣地存活下來，因為這個成長背景，所以有著許多奇特的行為。像是以下幾個例子。
:* 有著喜歡極度的甜和東西不辣就無法滿足的味覺。吃飯時不管和料理本身合不合，會要求調味料並且大量的加入菜色裡。這或許是身體被改造後所留下的後遺症。另外喜歡喝牛奶。
:* 對於艾蓮娜以外的女性沒有興趣，第四集因為生病被溫蒂照顧後，會叫溫蒂，怎麼樣也記不起其他女孩的名字。
:* 言語相當粗暴，經常在一些奇怪的場合攙入敬語使用。
:* 有著各式各樣的稱號，本人通常會自報自己新的稱號(會在名字前面加稱號)。第八集時，因為使用前集稱號而未產生新的，第六話時把本來叫作「好人邦」的稱號，自己改成「「邦·The nice guy」」。
: 根據本人所談，自己還是處男，但好像對於艾蓮娜以外的女孩並沒有意願奉獻其「純潔」的感情。
: 從第十六集最後的對話中可知道，其實他很害怕烏龜
:這個角色可能是根據電影《[[追殺比爾|追殺比爾]]》中的主角莉坦而來的。
:名字及造型可能是致敬自電影《[[黃昏雙鏢客|黃昏雙鏢客]]》主角之一[[李·范·克里夫|李·范·克里夫]]。

: 與梵安一起旅行的13歲的少女，為了找尋被勾爪男帶走的哥哥而與邦一起行動。雙親死後，搬去與叔叔一起住，在叔叔死後則和哥哥兩人一起生活，兩人的感情相當深。但在與被勾爪男洗腦的哥哥再見面後而對他失望，於是為了用自己的眼睛確認「什麼是惡，什麼是正義」而繼續與邦一起旅行。背後背著一把「哥哥過去持有的最重要事物」的槍，和哥哥再次分別後，把這把槍正式化為「自己的東西」。以後這把槍就成為代表「不再顧慮哥哥的想法，而是以自己的思想獨立前進」的溫蒂意志的符號。
: 在最終集數年後梵安偶然拜訪溫蒂的房子時，她的髮型和服裝並沒有太多改變，這是為了再見到梵安時，讓邦知道她是誰的作法。
:髮型的特徵是留著兩個像是麻花的雙馬尾，所以觀眾都叫她『螺旋麵包髮』。

* '''卡爾曼99'''({{lang|ja|'''カルメン 99'''}}、'''Carmen 99''')配音員：[[井上喜久子|井上喜久子]]／[[鄭仁君|鄭仁君]]（台灣）
: 本名為'''卡爾曼・曼多莎'''。23歳。在邦和溫蒂面前出現的謎之情報商人，行動多數在收集情報。如果被捲入危險的場合中，會使用有四枚刀子的溜溜球應戰。對於金錢相當執著，在大家把認為值錢的東西帶回來，並且拜托她鑑定和分析時會被她強行要求費用。至於在名字後面加上"99"的原因，好像是因為有著99個秘密跟99cm的胸圍，所以才用這個數字。
: 在立志當情報商人前是做女侍的工作，所以泡咖啡的技巧相當好。對於自己的本名有些許的情結存在，被叫作「卡爾爾」時，每次她都會叮嚀要叫她「卡爾曼」。有著對梵安有好感的描寫，但是大部份的情況卻沒有表現出來，直到最終集才說出。

* '''雷·郎格蘭'''({{lang|ja|'''レイ・ラングレン}}'''、'''Ray Lundgren''')配音員：[[櫻井孝宏|櫻井孝宏]]／[[夏治世|夏治世]]（台灣）
:因為妻子被勾爪男所殺，所以到處尋找勾爪男報仇。
: 打扮成和[[武士|武士]]一樣的裝扮，腰上掛著像是刀形狀的槍並且有著一張精悍面貌的青年。腳跟處隱藏便攜式手槍以及專屬裝甲鎧Volkein的搖控裝置。他也和梵安一樣，妻子志乃遭到勾爪男殺害，於是開始追蹤他要進行復仇。有著對於無論什麼東西，只要對他的復仇有所障礙就會面無表情地加以排除的冷酷一面，不過也會展露出擔心自己弟弟喬西亞的一面。根據喬西亞所說，他的哥哥以前是個安靜溫柔的人，但是為了復仇，於是揮別過去的自己而成為現在冷酷無情的性格。駕駛著由妻子所設計的機器人「Volkein」並且在平時利用它作為移動之用。雖然並非原裝機甲鎧, 但是Volkein有一套巨大鑽地機造型的移動機庫, 因此也能做到隨時召喚的效果。每次召喚時雷會用配槍360度向地面射擊, Volkein隨即破土而出。和梵安相反無論是個人還是機甲鎧都是注重射擊的風格，但是他的視力卻隨時間逐漸惡化。
: 多次比梵安他們搶先一步找到和勾爪男有關的情報並且持續追蹤著他。當初和同樣遭遇的梵安是敵對關係，兩人在找尋勾爪男的途中有多次的交手。決戰時因視線變得模糊，雖然正面對上鈎爪男射擊卻無法命中，其中一顆子彈卻碰巧卡在鈎爪男機體系統中，使得他的計劃被迫中止作修復。 看見鈎爪男少有露出的驚慌表情後，對自己成功奪去仇人的夢想感到滿足，隨後遭到敵軍人員開槍反擊而亡。 縱使未能手刃仇人，卻讓梵安得以趕上完成兩人的復仇。
:因某間中華風格的餐廳的侍應貌似亡妻而成為常客。

* '''喬西亞·郎格蘭'''({{lang|ja|'''ジョシュア・ラングレン}}'''、'''Joshua Lundgren''')配音員：[[野田順子|野田順子]]）／[[石采薇|石采薇]]（台灣）
:雷的弟弟，因為想要尋找雷的去向所以與邦他們一起行動。
: 雷的弟弟、唯一的血親。為了找尋哥哥而展開旅行，和梵安等人一起行動。單方面仰慕著梵安，常讓梵安感到苦惱(第14集喬西亞住院時，梵安就打算把他丟在醫院裡)，不過他本人並沒有注意到。有著感覺非常遲鈍、不懂人情世故和超級我行我素的性格，像是進入女廁找溫蒂，在布里希洛洗澡時直接掀開簾子要找她談話。由於這種性格被梵安和布里希洛指責說「你才不是我的朋友」。

: 決戰時向雷坦白曾經仰慕雷的妻子志乃。

* '''布里希洛'''({{lang|ja|'''プリシラ'''}}、'''Priscilla''')配音員：[[千葉紗子|千葉紗子]]／[[石采薇|石采薇]]（台灣）

: 駕駛著像是貓一樣型狀的粉紅機器人「布朗尼」的少女。從故事中段登場。也許是因為駕駛著強力的機器人，所以讓梵安能記住她的名字。在機器人格鬥大會上與梵安初次見面時，因為和他有著「痛快的戰鬥」所以對他產生興趣。但在梵安離去後而有一點失落，於是和布里希洛住在一起的孩子們(布里希洛本來也是孤兒，被孤兒院長的修女媽媽養大，後來媽媽去世後，繼承她的機器人參加格鬥比賽賺取獎金養育這些孤兒們)看不下去這種情況於是支持她離去，因此她追隨梵安並且加入他們。

* '''赫賽'''({{lang|ja|'''ホセ'''}})配音員：[[清川元夢|清川元夢]]

=== 勾爪男集團 ===

==== 歐利吉那魯7 (Original 7) ====
在這個E.I星球有著曾被當作是流放惡人的流放地的歷史(因此這個星球已經經過數個世代，所以已經與原來的星球失去聯絡)，所以作為監視兇惡犯人並且施以必要懲罰的監視者們是這個星球不可或缺的存在，"歐利吉那魯7"原本是擔任監視者和施以懲罰的組織，但是經過數個世代後，為了滿足自己的私慾而使用自己強大的力量加以動用私刑的情況不斷發生，使得組織慢慢腐敗。

至於在故事中登場的歐利吉那魯7，是在原本的組織遭到勾爪男所清除後，募集這些仰幕勾爪男的同志們所創立的(卡德貝多除外)。成員就像名稱一樣是以七個人所組成，但因為其中空缺的成員梵安與勾爪男為敵對關係，所以目前只有六人。全部的人都有和星期有關所命名的機器人。大部份的成員因為本身有著特殊體質，所以能不經改造手術就能駕駛機器人。

:港口巴帕‧帕雷多的黑社會老大，「蒙大拿家族」的首領，巴妮的父親。是這個港口第一名的裝甲收集狂，愛機則是有著三千馬力和配備三千武器的「多尼3000」。不過由於是由敞篷車所改造，所以最後遭到梵安直接用他的番刀輕鬆的打敗。

== 機器人一覽 ==
=== 動畫版 ===
在E.I星球、這種有高度技術的機器人都被稱作「裝甲」。由於裝甲並未有統一的形狀和規格，所以甚至在不同地區的人也會把武裝汽車叫作裝甲，因此會有各式各樣的裝甲出現。不過雖然種類不一，但大家還是習慣性地以裝甲來稱呼、而劇中所出現的裝甲大部分比較接近巨大的人形機器人。

==== 原体（Original）用的裝甲 ====
在E.I星球最初製作了七台裝甲。其種類可參考「[[#原体用的裝甲一覧|原体用的裝甲一覧]]」参照。梵安的丹也是原体用的七台裝甲之一。為了完成原体本來的目的、所以每人所駕駛的裝甲都會有專屬的系統、而且和一般裝甲的不同處就在於這些裝甲的外表裝備了許多的外殼。

以下幾點是它們的特徵。

; 裝甲界面系統
: 能夠藉由駕駛員腦部所流動的電氣訊息來加以感應其思考狀態、使得裝甲本身有更高的機動力（詳細 "[[#歐利吉那魯|歐利吉那魯]]" 請參考此項目）、另外還會在駕駛艙的營幕上呈現駕駛員的狀態和週遭情況並且能夠接受或傳送衛星控制系統的訊息的功能。當駕駛員的精神高亢時（當然也要很專注）駕駛艙會發出光芒並且可能出現強大的力量和電磁防護。

; G-ER流体控制系統
: 一種只要有電流通過、就能從固體變成液體的青黑色流体。主要是拿來驅動機體，而歐利吉那魯的專用裝甲沒有G-ER流体就無法啟動，所以透過這種流體所形成的強韌肌肉群才能使這些裝甲以直立兩足步行來行動，這也是它們與複製裝甲的不同。之後也有一些複製裝甲也因為有這種流體而能以直立兩足步行來行動（像是艾爾多拉五）。另外G-ER流体也被推測是讓裝甲更有行動力和擁有強大力量的根源。

; 衛星控制系統
: 這是為了每個裝甲而在E.I星球的衛星軌道上以一定間隔所設置的人工衛星〈衛星控制系統〉、平常時則是用來收納裝甲。衛星需要駕駛員作出特定動作(例如梵安用刀畫一個V字)才會發射裝甲到地面上。由於這個系統、使得歐利吉那魯能在各地叫喚它們出來。另外衛星也擁有修復裝甲的機能、能在裡面自動修復裝甲破損的地方。因此歐利吉那魯不太需要有維修人員。各裝甲被設定有專用的衛星、但也能透過從裝甲那邊輸入聲音來替換專用的衛星。然後最終集Saudade of Sunday、Diablo of Monday及Dann of Thursday的衛星皆遭破壞。另外原本丹的衛星遭到薩吾達第的破壞，後來是透過喬西亞的幫助而能切換使用卡德貝多的衛星來加以修復丹。

===== 原体（Originals） =====
如上所述，原体裝甲的性能皆優於其他裝甲、但代價是為了能夠讓駕駛員有更多的神經電力，所以必須施以改造手術（裝甲界面系統移植・各種調整）。這是因為人的腦神經電流很微弱、不改造就無法讀取到駕駛員的思考、也就無法讓裝甲動起來。本作有接受手術的人都被叫作 "原体（Original）" 。至於新原体的成員不用手術，是因為原成員的卡德貝多的加以改造而成。

成為原体後，身体會留有刺青（梵安在胸口上）、外表則和一般人沒什麼不同。但他們也被課以必須定期使用裝甲，否則無法維持生命（這是由於會取出G-ER流体的關係）。因此只要裝甲一定期間沒使用（最長似乎1週）、裝甲或衛星遭到破壞、也會喪失生命維持機能、使駕駛員只能等死。不過只要遵守這個規則就能得到「近乎不死的身体」。

基本上、原体的裝甲們都有和其外型相當的格鬥專用武器和電磁屏障、而勾爪男的成員們的裝甲則是追加有槍炮的裝備（除了卡德貝多）。

===== 新原体（Neo Originals） =====
現在原体用的裝甲都是勾爪男在各地透過裝甲研究者和技術人員加以組織起來改裝和强化性能的作品。勾爪男為了填補原体中失去的成員們、於是急於研發出能夠讓未改造者也能駕駛的技術，結果成功研發出透過感應一般人微量的神經電流就能夠動作，而不需要被施以手術的技術出來的装甲。

因此、像是女性法薩麗娜、還有小孩卡洛沙和梅洛莎、都能夠有適合他們的機體。未改造的成員們、為了和過去的成員有所區別，於是被叫作 "新原体（Neo Originals）" 。技術的進步讓過去「人來適合裝甲」到變成用反倫理的方法而成為「裝甲來適合人」。

如果一个新原体被施以了改造手术，他可以发挥出超越双方的力量。详情请参见最后一集。

===== 複製裝甲 =====
只要是除了歐利吉那魯專用裝甲以外的都被叫作「複製裝甲」。但其性能劣於專用裝甲、也未裝備G-ER流体、所以無法以兩足來步行〈大部分以輪子來行動。但是由於艾爾多拉V被改良成艾爾多拉魂、因此有裝備G-ER流体、所以得以用心兩足步行〉。但是專用裝甲會對駕駛員造成負擔、而複製裝甲則不會，所以在實用性和使用度也更高。

不過例外的，也有不輸於專用裝甲的複製裝甲、代表就是雷駕駛的伯魯凱因。原因是因為新歐利吉那魯所追加的光學射擊武器、全都是從伯魯凱因的基本技術中得到、因此能說伯魯凱因就是「新歐利吉那魯追加装備的原型」。

其他還有布里希洛的布朗尼和艾魯多拉魂。布朗尼外表看似能夠用兩足步行、實際上是在腳部的地方裝置輪子來移動。不過卻有著高機動力和高運動性的優點。但由於攻擊力低、所以有無法參加一對多戰鬥的缺點。

至於艾爾多拉魂原本是艾爾多拉V、但經過布奇的改造、讓它成為正式使用G-ER流体的複製裝甲（不過第一個使用G-ER流体的複製裝甲是布奇的BadRose、而正式使用的就是艾爾多拉魂）並且因其威力大大超越其它裝甲。由於使用G-ER流体、而能夠完全以二足步行、也讓它在戰用的使用度上大大增加。
