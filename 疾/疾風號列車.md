'''「疾風」'''（{{jpn|j=はやて|rm=Hayate|f=y}}）是一種在[[東日本旅客鐵道|東日本旅客鐵道]]（JR東日本）[[東北新幹線|東北新幹線]]及[[北海道旅客鐵道|北海道旅客鐵道]]（JR北海道）運行的[[特別急行列車|特別急行列車]]（特急列車）班次的名稱。「疾風」是東北新幹線[[盛岡站|盛岡]]至[[八戶站|八戶]]的路段在2002年12月1日開業時所增設的班次，並於2016年3月26日延伸至[[北海道新幹線|北海道新幹線]][[新函館北斗站|新函館北斗]]。<ref>
[http://www.jreast.co.jp/press/2002_1/20020911/index.html JR東日本－2002年12月1日訂正時間表的新聞稿]</ref>，曾經是東北新幹線中唯一行駛盛岡至八戶路段的班次。2010年末隨[[東北新幹線|東北新幹線]]伸延至[[新青森站|新青森]]，部份班次轉為[[隼號列車|隼號列車]]，班次亦逐步減少，至2014年，定期班次只剩下4.5來回，定期東京～新青森班次已全數取消。

但隨著[[北海道新幹線|北海道新幹線]]開業，服務盛岡～[[新函館北斗站|新函館北斗]]的列車亦定名為「疾風」<ref>[http://www.jrhokkaido.co.jp/press/2014/141120-1.pdf JR東日本及JR北海道的新聞稿，]2014年11月20日。{{ja}}</ref>，相信有助把班次再次增加。

== 名稱由來 ==
和其他大部分列車名稱一樣，JR東日本在2002年4月為此班次和連接八戶和函館的特急班次向公眾公開招募列車名稱<ref>[http://www.jreast.co.jp/press/2002_1/20020405/index.html JR東日本－列車命名公開招募的新聞稿]</ref>。最終，獲最多人提名的兩個名稱分別為「[[陸奧號列車|陸奧]]」和「[[津輕號列車|津輕]]」，而「疾風」只排名19位<ref name="result">[http://www.jreast.co.jp/press/2002_1/20020505/index.html JR東日本－列車命名公開招募結果的新聞稿]</ref>。但和東海道新幹線的列車命名一樣，經討論後最終JR東日本採用「疾風」的命名。除了顯示列車的速度，更希望能顯示一種嶄新的風格<ref name="result"/>。至於同時公開招募名稱的特急班次，來往八戶和函館的特急班次被命名為「[[白鳥號列車|白鳥]]」，而來往八戶和青森、弘前的特急班次則被命名為「津輕」。

== 歷史 ==
* 2002年12月1日：東北新幹線盛岡至八戶路段開業時開設。
* 2003年：增加盛岡至八戶的「疾風271號」班次。
* 2005年12月10日：訂正時間表，正式服務東北新幹線全線。自此「疾風」取代「[[山彥_(列車)|山彥]]」，成為東北新幹線行車速度最高的列車班次。
* 2007年3月18日：按法例規定，所有車廂禁煙。
* 2007年8月15日：增開由仙台開出的「疾風332號」臨時班次，是「疾風」開業以來首班由仙台開出的上行列車。
* 2007年12月1日：增開東京至仙台的「疾風305號」臨時班次，是2005年宮城縣南部地震的特例以來首班以仙台為終點站的下行列車。
* 2008年3月15日：增加仙台至東京的「疾風100號」班次，但暫停一班自盛岡開出的班次。
* 2010年12月4日：延長營運區間至[[新青森站|新青森]]。每日定期來往東京及新青森有15往復。
* 2011年11月19日：E5系列車開始營運疾風號。2往復東京至新青森及1往復東京至盛岡改以E5系列車營運（部分連結E3系[[小町號列車|小町號列車]]運行）。
* 2012年3月17日：訂正時間表，來往東京及新青森有15往復中，其中7往復以E5系列車營運。
* 2013年3月16日：訂正時間表，所有由新青森站始發的列車，全部改為E5系。部分最快速的班次昇格為[[隼號列車|「隼」號列車]]。而設定為「疾風」號列車的E5系列車，特等車廂將不設車廂服務員服務，並且改沿用[[綠色車廂|綠色車廂]]的附加費用。
* 2013年9月28日：訂正時間表，更多列車改為「隼」號，並由開始引入E6系列車。E2系及E3系0番台定期併結列車終止。
* 2014年3月15日：訂正時間表，將所有服務東京～新青森之間的列車統一稱為「隼」號列車，「疾風」號列車班次大幅減少，定期班次只剩下每日1來回盛岡～新青森區間（但不停[[岩手沼宮內站|岩手沼宮內]]），及3.5來回東京～盛岡間（[[仙台站|仙台]]以北為各站停車），並不再與小町號列車併結。
* 2016年3月26日：北海道新幹線開業，本列車服務延伸至[[新函館北斗站|新函館北斗]]。

== 所需時間 ==
; 以2011年3月5日起使用的時間表為準
* 東京－仙台：最快1小時36分鐘（12號，不停上野），平均1小時40～45分鐘
* 東京－盛岡：最快2小時21分鐘（12號，不停上野），平均2小時25～30分鐘
* 東京－新青森：最快3小時33分鐘（36號，不停八戶），平均3小時35分鐘～3小時55分鐘
