'''范泰'''（{{Bd|355年||428年|9月21日|}}）<ref>《宋書·文帝紀》：秋八月壬戌，特進、左光祿大夫范泰卒。</ref>，字'''伯伦'''，[[东晋|东晋]][[顺阳郡|顺阳]]山阴人（今[[河南省|河南省]][[淅川县|淅川县]][[李官桥镇|李官桥镇]]<ref>陈文英 《范缜籍贯考辨》《商丘师范学院学报》 2001年05期</ref><ref>龚剑锋 《范晔籍贯考》《史学史研究》 1987年01期</ref>）。[[東晉|東晉]]末及[[南朝宋|南朝宋]]官員，[[范甯|范甯]]之子。

== 生平 ==
范泰最初任[[太学|太学]]博士，後又先後任衞將軍[[謝安|謝安]]及[[驃騎將軍|驃騎將軍]][[司馬道子|司馬道子]]的[[參軍|參軍]]。表弟[[王忱|王忱]]於389年上任[[荊州_(古代)|荊州]][[刺史|刺史]]後，又請其任天門太守。392年王忱死後，司馬道子再召范泰為諮議參軍，後遷中書侍郎。當時道子子[[司馬元顯|司馬元顯]]專權，內外官員都請假回避，不再上表皇帝，只管順從元顯而已。范泰認為這種行為並不恰當，向元顯進言但不被接受。其後范泰父親范甯去世，范泰襲父爵'''陽遂鄉侯'''。元興元年（402年），荊州刺史[[桓玄|桓玄]]率兵擊敗了司馬元顯，攻下[[建康|建康]]並執掌朝政，就命[[御史中丞|御史中丞]][[祖台之|祖台之]]彈劾范泰與[[王準之|王準之]]、[[司馬珣之|司馬珣之]]居喪無禮，范泰因而被罷黜並外放[[丹徒縣|丹徒縣]]。

元興三年（404年），[[劉裕|劉裕]]在桓玄稱帝後兩個月舉兵討伐，桓玄兵敗出逃，最終被殺，東晉復祚。范泰任國子博士，後历任[[黃門郎|黃門郎]]、[[御史中丞|御史中丞]]及東陽太守。義熙六年（410年），盧循起兵反叛，范泰出兵一千人協助抵禦，並開糧倉支援晉軍兵糧，劉裕於是加授范泰振武將軍。次年（411年），范泰就還朝，先後任[[侍中|侍中]]、[[度支尚書|度支尚書]]及[[太常|太常]]，又曾轉大司馬左長史，右衞將軍，加散騎常侍。范泰後遷尚書，加散騎常侍，並於義熙十二年（416年）以兼司空身份與[[袁湛|袁湛]]授予劉裕宋公爵位及[[九錫|九錫]]禮遇<ref>《資治通鑑·卷一百一十七》義熙十二年：「十二月，壬申，詔以裕為相國、緦百揆、揚州牧，封十郡為宋公，備九錫之禮，位在諸侯王上，領征西將軍、司豫北徐雍四州刺史如故。裕辭不受。」</ref><ref>《宋書·袁湛傳》：「時高祖北伐，湛兼太尉，與兼司空、散騎常侍、尚書范泰奉九命禮物，拜授高祖。」</ref>。時劉裕正在北伐後秦，二人就隨軍至[[洛陽|洛陽]]。

范泰後轉護軍將軍。[[永初_(劉裕)|永初]]元年（420年），以范泰為[[金紫光禄大夫|金紫光禄大夫]]，又加[[散骑常侍|散骑常侍]]。次年因興辦國學而任[[國子祭酒|國子祭酒]]。[[景平|景平]]元年（423年），加特進，次年退休，解任國子祭酒。當時宋少帝[[劉義符|劉義符]]行為有不少缺失，范泰於是上書極力勸諫，但都不為劉義符所納。景平二年（424年），輔政的[[徐羨之|徐羨之]]、[[傅亮|傅亮]]、[[謝晦|謝晦]]先殺廬陵王[[劉義真|劉義真]]，後廢殺劉義符。范泰向來都與徐羨之不睦，見其殺害皇帝及宗王，於是向親信說：「吾觀古今多矣，未有受遺顧託，而嗣君見殺，賢王嬰戮者也。」斥責徐羨之等人的行為。[[元嘉|元嘉]]二年（425年）[[春節|春節]]是宋文帝[[劉義隆|劉義隆]]即位後的首個春節，范泰卻在那時上表祝賀並上陳當時旱災後到[[東陽郡|東陽郡]]出遊，不理朝政。當時有官員彈劾范泰，但劉義隆並不問罪。

元嘉三年（426年），劉義隆誅殺徐羨之及傅亮，進范泰為特進、侍中、左光禄大夫、國子祭酒，領[[劉義恭|江夏王]]師。當時更以范泰既為晉朝舊臣，亦有腳患，行動不便，故此特地准許他坐轎上朝。

范泰於元嘉五年（428年）逝世，享年七十四，獲追贈車騎將軍，賜[[謚號|謚號]]'''宣'''。

== 性格特徵 ==
*范泰一生博览群书，笔耕不辍，也會鼓勵後進，撰有《古今善言》二十四篇及文集二十卷。
*范泰喜歡飲酒，為人不拘小節，直率隨心，在外亦像在家那樣，故甚得劉裕親信，劉裕北伐班師回彭城時，更與范泰同登城上，還命人抬轎去載有腳患的范泰登城。不過由於政治能力不高，都未能擔當朝中要職。
*范泰晚年信了佛，還相當虔誠，在家中西邊建了一座[[祗洹精舍|祗洹精舍]]。

== 逸事 ==
*范泰死後，朝廷本來打算追贈[[開府|開府]]，不過[[殷景仁|殷景仁]]就認為范泰向來聲望不高，不能享有開府這種能比擬三公的贈官。最終朝廷只追贈車騎將軍。下葬時，[[王弘_(劉宋)|王弘]]就撫著范泰的棺材說：「君平生重殷鐵，今以此為報。」<ref>《建康實錄·卷十二》</ref>

== 子女 ==
*[[范昂|范昂]]，長子，早卒
*[[范暠|范暠]]，次子，宜都太守
*[[范晏|范晏]]，三子，侍中、光祿大夫
*[[范曄|范曄]]，四子，過繼給[[范弘之|范弘之]]，歷史學家，撰《[[後漢書|後漢書]]》。因謀反而被誅殺。
*[[范廣淵|范廣淵]]，幼子，撫軍諮議參軍，領記室，受范曄株連被殺。
