'''范明'''（{{bd|1964年|10月12日||}}），[[山东省|山东省]][[临沂市|临沂市]]人，[[中國|中國]][[演員|演員]]、[[導演|導演]]，為中國[[國家二級演員|國家二級演員]]，毕业于[[解放军艺术学院|解放军艺术学院]]表演系。代表作有《[[我和连长|我和连长]]》、《[[炊事班的故事|炊事班的故事]]》、《[[武林外传|武林外传]]》等喜剧电视剧。

== 艺术简历 ==
*毕业于解放军艺术学表演系
*江苏省戏剧家协会会员

== 家庭 ==
妻子厉玲，他有一个女儿范美辰和儿子韩冷（与妹妹是同母异父）。

== 主演的小品 ==
*《山海相连》荣获96年全军文艺新作品奖优秀表演奖；
*主演并导演小品《听球》荣获第七届全军文艺会演小品表演一等奖，并获导演二等奖；
*执导的小品《水》荣获2003年中国“曹禺戏剧奖”，小品小戏评选导演奖；
*参演的小品《采访》获优秀表演奖

== 获奖 ==
*[[第13届华鼎奖|第13届华鼎奖]] 全国观众最喜爱的影视明星
*电视剧《炊事班的故事》饰演老高，获第十六届[[金星奖|金星奖]]最佳男演员奖
*电视电影《我和连长》饰连长，本人与另一位主演双双获得第九届[[北京大学生电影节|北京大学生电影节]]首届电视电影最佳表演搭档奖
*2016年 [[第19屆華鼎獎|第19屆華鼎獎]]中國百強電視劇滿意度調查發佈盛典－中國百強電視劇最佳男配角《[[爱情碟中谍|爱情碟中谍]]》
