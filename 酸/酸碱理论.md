'''酸碱理论'''指阐述[[酸|酸]]、[[碱|碱]]及酸碱反应本质的各种理论。在历史上曾有多种酸碱理论，其中重要的包括：
* [[阿伦尼乌斯|阿伦尼乌斯]]酸碱理论——[[酸碱电离理论|酸碱电离理论]]
* {{link-da|布朗斯特|Johannes Nicolaus Brønsted}}-{{link-en|劳里|Thomas Martin Lowry}}酸碱理论——[[酸碱质子理论|酸碱质子理论]]
* [[吉爾伯特·路易士|路易士]]酸碱理论——[[酸碱电子理论|酸碱电子理论]]
* 酸碱溶剂理论
* [[软硬酸碱理论|软硬酸碱理论]]

== 常用的酸碱理论 ==
=== 拉瓦锡的定义 ===
[[拉瓦锡|拉瓦锡]]是最早提出酸碱概念的人。他在1776年左右提出一套酸碱理论。在那时，[[强酸|强酸]]主要是[[硝酸|HNO<sub>3</sub>]]和[[硫酸|H<sub>2</sub>SO<sub>4</sub>]]一类的[[含氧酸|含氧酸]]，基本上都含有氧元素和高[[氧化态|氧化态]]的中心原子。因此拉瓦锡认为氧是酸中不可或缺的组分，将氧定义为酸生成者（οξυς γεινομαι），并且认为当时还未研究清楚成分的[[氢卤酸|氢卤酸]]中也含有氧元素。这个定义一直推行了30年，直到1810年[[汉弗里·戴维|戴维]]证明了[[硫化氢|H<sub>2</sub>S]]、[[碲化氢|H<sub>2</sub>Te]]和[[卤化氢|卤化氢]]虽也属于酸，但不含氧原子。

=== 李比希的定义 ===
[[尤斯图斯·冯·李比希|尤斯图斯·冯·李比希]]在研究了很多[[有机酸|有机酸]]的组成后，于1838年左右提出一套酸碱理论，认为酸是含氢元素的物质，并且其中的氢可以被
金属原子替换。<ref name="liebig_1">Miessler, L. M., Tar, D. A., (1991) p166 - table of discoveries attributes Justus von Liebig's publication as 1838</ref><ref name=meyers_156>Meyers, R. (2003) p156</ref> 这个理论在推行了50年后，被更加全面的阿伦尼乌斯酸碱电离理论所替代。<ref name="liebig_2">H. L. Finston and A. C. Rychtman, A New View of Current Acid-Base Theories, John Wiley & Sons, New York, 1982, pp. 140-146.</ref>

=== 阿伦尼乌斯的定义 ===
{{main|酸碱电离理论}}
[[File:Arrhenius2.jpg|thumb]]

该理论以[[阿伦尼乌斯|阿伦尼乌斯]]与[[威廉·奥斯特瓦尔德|威廉·奥斯特瓦尔德]]在1884年左右的研究为基础，相比其他酸碱理论更加简明易懂。阿伦尼乌斯本人也因此获得1903年的[[诺贝尔化学奖|诺贝尔化学奖]]。

至于该理论中的酸碱定义，可用以下一句话来描述：
{{cquote|阿伦尼乌斯酸碱理论中，酸在水溶液中解离出氢离子，而碱则解离出氢氧根离子。}}

酸碱反应的本质是氢离子与氢氧根离子反应生成水。
:2 H<sub>2</sub>O → OH<sup>−</sup> + H<sub>3</sub>O<sup>+</sup><ref>水合氢离子H<sub>3</sub>O<sup>+</sup>简写作H<sup>+</sup>：2 H<sub>2</sub>O → H<sub>3</sub>O<sup>+</sup> + OH<sup>−</sup> = H<sub>2</sub>O → H<sup>+</sup> + OH<sup>−</sup></ref>
因此在该理论下，酸碱反应生成盐和水的过程也被称作[[中和反应|中和反应]]。<ref name="miessler_165" />
:酸<sup>+</sup> + 碱<sup>−</sup> → 盐 + 水

碱的阳离子可与酸中的阴离子成盐。比如，两[[摩尔|摩尔]]的[[氢氧化钠|氢氧化钠]]（NaOH）与一摩尔的硫酸（H<sub>2</sub>SO<sub>4</sub>）反应，产物是两摩尔水和一摩尔[[硫酸钠|硫酸钠]]：
:2NaOH + H<sub>2</sub>SO<sub>4</sub> → 2 H<sub>2</sub>O + Na<sub>2</sub>SO<sub>4</sub>

=== 布忍斯特 (Brønsted)及羅瑞 (Lowry)酸鹼學說 ===
{{main|酸碱质子理论}}
丹麦化学家{{tsl|en|Johannes Nicolaus Brønsted|约翰内斯·尼古拉斯·布朗斯特}}和英国化学家{{tsl|en|Martin Lowry|托马斯·马丁·劳瑞}}于1923年分别提出酸碱质子理论，也称为布朗斯特-劳瑞酸碱理论。该理论认为，凡是能给出[[质子|质子]]（H<sup>+</sup>）的物质都是酸，凡能接受质子的物质都是碱，而既能给出质子，也能接受质子的物质称为[[两性_(化学)|两性物质]]。<ref name="miessler_165" /><ref name="Clayden_1">Clayden, J., Warren, S., et al. (2000), p182-184</ref><ref name="miessler_167">Miessler, L. M., Tar, D. A., (1991), p167-169 -- According to this page, the original definition was that "acids have a tendency to lose a proton"</ref> 酸和碱不是孤立的，它们通过质子互相联系，用通式可以表示为：
:酸 → 碱 + 质子
这样的一对酸碱称为[[共轭酸碱对|共轭酸碱对]]，其中的酸和碱分别称为相应物质的共轭酸及共轭碱。<ref name="miessler_165" /><ref name="miessler_167" />

与阿伦尼乌斯酸碱理论不同的是，布忍斯特酸碱不仅限于电中性的分子，也包括带电的阴阳离子。而该理论之下的酸碱反应则是两对共轭酸碱对之间传递质子的反应，不一定生成盐和水：
:'''酸<sub>1</sub> + 碱<sub>2</sub> → 碱<sub>1</sub> + 酸<sub>2</sub>'''
:AH + B → A<sup>−</sup> + BH<sup>+</sup>
:HCl (aq) + H<sub>2</sub>O → H<sub>3</sub>O<sup>+</sup> (aq) + Cl<sup>−</sup> (aq)
:CH<sub>3</sub>COOH + NH<sub>3</sub> → NH<sub>4</sub><sup>+</sup> + CH<sub>3</sub>COO<sup>−</sup>

=== 路易斯的定义 ===
{{see|路易斯酸|路易斯碱}}
路易斯酸碱理论由[[吉尔伯特·牛顿·路易斯|吉尔伯特·牛顿·路易斯]]在1923年提出，<ref name="lewis_1">Miessler, L. M., Tar, D. A., (1991) p166 - Table of discoveries attributes the date of publication/release for the Lewis theory as 1923.</ref> 结合了布朗斯特-劳里和酸碱溶剂理论的特点，在水溶液和非水溶剂中都有很广的应用。<ref name="lewis_2">Miessler, L. M., Tar, D. A., (1991) p170-172</ref> 该理论着重探讨电子的给予与获得，<ref name="lewis_2" />[[路易斯酸|路易斯酸]]被定义为电子接受体，而[[路易斯碱|路易斯碱]]则是电子给予体。

: Ag<sup>+</sup> (酸) + 2 :NH<sub>3</sub> (碱) → [H<sub>3</sub>N:Ag:NH<sub>3</sub>]<sup>+</sup> (酸碱加合物)

路易斯酸与路易斯碱反应时，路易斯碱含有孤对电子的最高占有轨道（[[HOMO|HOMO]]）向路易斯酸[[缺电子|缺电子]]的最低未占轨道（[[LUMO|LUMO]]）贡献电子生成配位键，产物称作酸碱[[加合物|加合物]]。<ref name="lewis_2" /> 在强极性分子如[[三氟化硼|三氟化硼]]中，<ref name="lewis_2" />[[电负性|电负性]]强的元素吸引电子，带有部分负电荷，电负性弱的元素则带有部分正电荷，孤对-成键电子（Lp-Bp）之间作用力超过成键-成键电子（Bp-Bp）之间的作用力。<ref name="lewis_2" /> 金属离子的加合物被称为[[配位化合物|配位化合物]]。<ref name="lewis_2" />

=== 溶剂理论 ===
该理论與阿伦尼乌斯对所有[[自偶解离|自偶解离]]溶剂的定义有关。在这些溶剂中，存在中性溶剂分子与解离出的阳离子和阴离子之间的平衡：
: 2H<sub>2</sub>O {{unicode|⇌}} H<sub>3</sub>O<sup>+</sup> ([[水合氢离子|水合氢离子]]) + OH<sup>−</sup> ([[氢氧根|氢氧根]])
: 2NH<sub>3</sub> {{unicode|⇌}} NH<sub>4</sub><sup>+</sup> ([[铵|铵]]) + NH<sub>2</sub><sup>−</sup> ([[胺|胺]])
非质子溶剂：
: N<sub>2</sub>O<sub>4</sub> {{unicode|⇌}} NO<sup>+</sup> ([[亚硝基正离子|亚硝基正离子]]) + NO<sub>3</sub><sup>−</sup> ([[硝酸根|硝酸根]])
: 2SbCl<sub>3</sub> {{unicode|⇌}} SbCl<sub>2</sub><sup>+</sup>  + SbCl<sub>4</sub><sup>−</sup> 

酸导致溶剂阳离子浓度上升，阴离子浓度下降；而碱则导致阳离子浓度下降，阴离子浓度上升。例如在液[[氨|氨]]中，[[氨基钾|KNH<sub>2</sub>]]提供NH<sub>2</sub><sup>−</sup>离子，是[[强碱|强碱]]，而[[硝酸铵|NH<sub>4</sub>NO<sub>3</sub>]]提供NH<sub>4</sub><sup>+</sup>离子，是[[强酸|强酸]]。在液态[[二氧化硫|二氧化硫]]（SO<sub>2</sub>）中，[[亚硫酰基|亚硫酰基]]化合物是酸，提供SO<sup>2+</sup>离子；而[[亚硫酸盐|亚硫酸盐]]提供SO<sub>3</sub><sup>2−</sup>离子，可看作碱。

该理论下，液氨中的酸碱反应包括：
: 2NaNH<sub>2</sub> (碱) + Zn(NH<sub>2</sub>)<sub>2</sub> ([[两性_(化学)|两性]]) → Na<sub>2</sub>[Zn(NH<sub>2</sub>)<sub>4</sub>]
: 2NH<sub>4</sub>I (酸) + Zn(NH<sub>2</sub>)<sub>2</sub> ([[两性_(化学)|两性]]) → [Zn(NH<sub>3</sub>)<sub>4</sub>)]I<sub>2</sub>
硝酸在纯硫酸中是碱：
: HNO<sub>3</sub> (碱) + 2H<sub>2</sub>SO<sub>4</sub> → NO<sub>2</sub><sup>+</sup> + H<sub>3</sub>O<sup>+</sup> + 2HSO<sub>4</sub><sup>−</sup>
液态[[四氧化二氮|四氧化二氮]]中：
: AgNO<sub>3</sub> (碱) + NOCl (酸) → N<sub>2</sub>O<sub>4</sub> + AgCl 

酸碱溶剂理论中，同一化合物在不同溶剂中可以改变其酸碱性质，比如[[高氯酸|HClO<sub>4</sub>]]在水中是强酸，在[[乙酸|乙酸]]中是弱酸，而在[[氟磺酸|氟磺酸]]中则是弱碱。

== 其他酸碱理论 ==
=== Usanovich的定义 ===
关于酸碱最基本的定义来自于[[俄罗斯|俄罗斯]][[化学家|化学家]]Mikhail Usanovich。根据该定义，只要是可以接受[[负电荷|负电荷]]或放出[[正电荷|正电荷]]，就是酸；反之则是碱。因为这个定义与氧化还原的定义有些重合，所以化学家并不是很倾向于使用这个定义。这是因为氧化还原主要集中讨论物理上的电子转移过程，而并非是键的形成与断裂过程，尽管要将两者完全区分是不可能的。

=== Lux-Flood的定义 ===

這個定義由德國化學家[[:en:Hermann_Lux|Hermann Lux]]<ref>Franz, H. (1966), p4</ref><ref name=lux>
{{cite journal|title="Säuren" und "Basen" im Schmelzfluss: die Bestimmung. der Sauerstoffionen-Konzentration|first=Hermann|last=Lux|authorlink=Hermann Lux|journal=Ztschr. Elektrochem|date=1939|volume=45|issue=4|pages=303–309}}</ref> 在1939年時所提出，其後[[:en:Håkon_Flood|Håkon Flood]]約在1947年作進一步的修正，<ref name=flood>{{cite journal|title=The Acidic and Basic Properties of Oxides|last=Flood|first=H.|authorlink=Håkon Flood|coauthors=Forland, T.|journal=Acta Chem. Scand.|date=1947|volume=1|pages=592}}</ref> 
现在主要用于现代熔盐的[[地球化学|地球化学]]和[[电化学|电化学]]研究中。在该定义中，酸被定义为一个氧离子受体，而碱则是一个氧离子供体。例如：
: MgO (碱) + CO<sub>2</sub> (酸) → MgCO<sub>3</sub>
: CaO (碱) + SiO<sub>2</sub> (酸) → CaSiO<sub>3</sub>
: NO<sub>3</sub><sup>−</sup> (碱) + S<sub>2</sub>O<sub>7</sub><sup>2−</sup> (酸) → NO<sub>2</sub><sup>+</sup> + 2SO<sub>4</sub><sup>2−</sup><ref name=drago>{{cite journal|title=The Synthesis of Oxyhalides Utilizing Fused-Salt Media|first=Russel S.|last=Drago|coauthors=Whitten, Kenneth W.|journal=[[Inorg._Chem.|Inorg. Chem.]]|date=1966|volume=5|issue=4|pages=677 - 682|doi=10.1021/ic50038a038}}</ref>

=== 皮尔逊的定义 ===
{{main|软硬酸碱理论}}
1963年，<ref name=pearson>{{cite journal|title=Hard and Soft Acids and Bases|last=Pearson|first=Ralph G.|journal= [[J._Am._Chem._Soc.|J. Am. Chem. Soc.]] |date=1963| volume= 85 |issue=22|pages=3533 - 3539|doi=10.1021/ja00905a001}}</ref> 拉斐尔·皮尔逊提出了一个高级的定性概念——[[软硬酸碱理论|软硬酸碱理论]]。1984年，在[[Robert_Parr|Robert Parr]]的协助下，该理论发展成为一个定量的理论。“硬”对应的是小的、高[[氧化态|氧化态]]的粒子，这些粒子都很难被[[极化|极化]]。相反，“软”是指大的，低氧化态的粒子，很容易被极化。软-软和硬-硬之间的酸碱反应最为稳定。这个理论在有机化学和无机化学均有应用。
