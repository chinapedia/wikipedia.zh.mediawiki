'''新標點和合本'''（{{lang-en|'''Chinese Union Version with New Punctuation'''}}，[[縮寫|縮寫]]：'''CUNP''' 或 '''CUVNP'''）於1988年出版，由聯合聖經公會組織聖經學者、翻譯顧問和編輯人員，以《[[和合本|和合本]]》為基礎展開修訂。《新標點和合本》在用語和文體上基本沒有改動<ref>[http://www.hkbs.org.hk/themes/hkbs2/files/HKBS_CU90th_booklet.pdf 百載聖道的旅程]</ref>。目前香港聖經公會銷售的中文聖經，主要也是以《新標點和合本》和《[[和合本修訂版|和合本修訂版]]》為主<ref>[http://www.cdn.org.tw/News.aspx?key=9916 香港聖經公會銷售的中文聖經]</ref>。1988年《新標點和合本》與1919年《[[和合本|和合本]]》的關係，就像是1769年《欽定版聖經》與1611年《[[欽定版聖經|欽定版聖經]]》，前者更新了拼寫與標點，使之和現代一致。

== 特點 ==
* 主要是以現代通用的標點符號代替《[[和合本|和合本]]》「凡例」所使用的舊標點符號；
* 原經文是詩體裁就用詩體排法；
* 新舊約用詞盡量統一，並使用現代通用字，如用「夠」，不用「彀」；用「才」，不用「纔」等；<ref>[http://mygets.org/articles.php?id=140 聖經版本介紹]</ref>
* 使用更適合的人名，例如以「呂便」代替「流便」、「該撒」改為「凱撒」等；
* 使用當今通用的地名，如:羅馬書 15:23「士班雅」改為「西班牙」、創世記 15:18「伯拉大河」改為「幼發拉底河」等；
* 代名詞的第三人稱，按情況分為男性用「他」，女性用「她」，動物用「牠」，事物用「它」，使讀者更易辨別；
* 在經文中有從屬地名或地名和人名連在一起時，在少數地方加了「的」，以免混淆（例如: <ref>[https://zh.wikisource.org/wiki/%E8%81%96%E7%B6%93_(%E5%92%8C%E5%90%88%E6%9C%AC)/%E5%A3%AB%E5%B8%AB%E8%A8%98#.E7.AC.AC.E5.8D.81.E4.B8.83.E7.AB.A0 士師記17:7]</ref>「猶大伯利恆」改為「猶大的伯利恆」）；
* 小標題有平行經文章節，以便讀者參考。

== 爭議 ==
* 譯名修改
1.「流便」: 《新標點和合本》出版說明提到將不雅的譯名，如以「呂便」代替[[流便|流便]](Reuben)。而未考慮「流便」意思也是指「文筆流暢不滯澀」。且若是「便」字有不便之處，應該改為「本」字，如﹕便雅憫，便哈達..等<ref>[http://www.ebaomonthly.com/ebao/readebao.php?a=20040904 聖經與文字]</ref>。

2.「推羅」: 1988年《新標點和合本》將推羅(Tyre)改為泰爾，但是中國基督教協會1989年出版的《簡化字現代標點和合本》卻未改爲「泰爾」，因爲「推羅」更接近原文的發音<ref>[http://dhost.info/pingke/C_T-BibleT-H-WanZhaoyuan-XiudingHehebenShenjingDeQiquLicheng.htm 译事未竟路——修订《和合本圣经》的崎岖历程] {{webarchive|url=https://web.archive.org/web/20150105113306/http://dhost.info/pingke/C_T-BibleT-H-WanZhaoyuan-XiudingHehebenShenjingDeQiquLicheng.htm |date=2015-01-05 }}</ref>。而2010年《和合本修訂版》又改回「推羅」。

* 版本問題
由於1988年出版的《新標點和合本》改動了《和合本》的標點、分段、地名，而1919年的《和合本》的版權已經期滿，所以就有不少出版商就競相效尤自行出版其新標點的《和合本》，網路上更有許多電子版新標點的《和合本》在網路上任人下載<ref>[http://www.bstwn.org/bmag/bmag01/whichcuv.htm 你手上拿的是哪一本《和合本》呢？ ]</ref>。這些未經[[聯合聖經公會|聯合聖經公會]]授權的"新標點"版本，可能有手民之誤的問題<ref>[http://www.translatebible.com/blog/c20130705 由《和合本》的版權問題談起]</ref>。
