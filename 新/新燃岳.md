'''新燃岳火山'''是[[日本|日本]][[九州岛|九州岛]]南部的一座[[活火山|活火山]]，位于[[霧島山脈|霧島山脈]]的中部，处于[[鹿儿岛县|鹿儿岛县]][[雾岛市|雾岛市]]和[[宫崎县|宫崎县]][[小林市|小林市]]的交界。海拔1420.77米。
[[File:Shin-moe_Eruption_2011_0127.jpg|210px]]
[[File:Shinmoe-dake_Volcano_Erupts_on_Kyushu_Feb_2011.jpg|thumb]]

==火山噴發==
新燃岳火山曾於1716年、1717年、1771年、1822年、1959年、1991年、2008年、2009年、2011年、2017年和2018年噴發。<ref>{{cite news|title=Kyushu’s Shinmoedake volcano erupts for first time in six years|url=https://www.japantimes.co.jp/news/2017/10/11/national/kyushus-shinmoedake-volcano-blows-stack-first-time-six-years/#.Wd4thVuPJaQ}}</ref>

上次於[[2011年1月|2011年1月]]開始噴發。<ref>{{cite web | language = Zh-hans | publisher = 新华网 | title =日本雾岛山新燃岳火山喷发| url = http://news.xinhuanet.com/world/2011-01/26/c_121028245.htm| date = 2011-01-26 | accessdate = 2011-01-26 }}</ref>於2011年3月13日下午3點45分左右，新燃岳又開始向上噴發大約有 4,000米-{}-高度的煙塵，並夾帶著大量的[[火山灰|火山灰]]與[[石頭|石頭]]。<ref>中廣新聞／葉柏毅,"日「新燃岳火山」噴火 煙塵高約四千公尺"{{cite web |url=http://news.chinatimes.com/world/50405768/132011031301073.html |title=存档副本 |accessdate=2011-03-16 |deadurl=yes |archiveurl=https://web.archive.org/web/20110316173807/http://news.chinatimes.com/world/50405768/132011031301073.html |archivedate=2011-03-16 }},[[中國時報|中國時報]],2011-03-13.</ref>從1月26日首次爆發以來，至13日的噴發已經是第14次。<ref>記者徐尚禮／綜合報導,"新燃岳火山噴火 煙塵高4000公尺"{{cite web |url=http://news.chinatimes.com/focus/50108307/112011031400079.html |title=存档副本 |accessdate=2011-03-14 |deadurl=yes |archiveurl=https://web.archive.org/web/20110317084029/http://news.chinatimes.com/focus/50108307/112011031400079.html |archivedate=2011-03-17 }},[[中國時報|中國時報]]/[[旺報|旺報]],2011-03-14</ref>

2017年10月，火山再次爆發。<ref>睽違6年　日本九州新燃岳火山噴發[https://m.nownews.com/news/2622612],[[NOWnews_今日新聞|NOWnews 今日新聞]],2017-10-11</ref>

2018年3月，再次爆發噴火達4500公尺。<ref>日新燃岳火山爆發　噴火4500公尺[https://tw.appledaily.com/new/realtime/20180310/1311848/],[[蘋果即時|蘋果即時]],2018-03-10</ref>

日本[[立命館大學|立命館大學]][[環太平洋|環太平洋]]文明研究中心教授高橋學表示，以往地震情況作為案例得出，所有大型地震都有預兆，而直下型地震、板塊型地震與火山噴發之間存在密切聯繫。新燃岳火山活動已經進入活躍期，這預示著2年內發生「巨大南海地震」的可能性極高，可能在[[名古屋|名古屋]]、[[大阪|大阪]]等地引發海嘯，將致超過47萬人死亡或失踪。

2018年4月5日再度噴火達5,000公尺，同年5月14日和6月22日再次噴發。[http://m.ltn.com.tw/news/world/breakingnews/2386954]
[http://m.ltn.com.tw/news/world/breakingnews/2425709]

== 地理 ==
=== 周邊的火山 ===
位於霧島火山群中央位置、北西有獅子戶岳（1,429米）、南東有中岳（1,332米）等山岳相連。北北西約3.7公里處有火口湖[[大浪池|大浪池]]、北東約2.8公里處有[[大幡池|大幡池]]。
