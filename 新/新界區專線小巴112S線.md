'''新界區專線小巴112S線'''是由[[保升國際投資有限公司|保升國際投資有限公司]]營辦的一條通宵綠色專線小巴路線，由[[將軍澳站公共運輸交匯處|將軍澳站公共運輸交匯處]]往來[[將軍澳工業邨|將軍澳工業邨]]（駿昌街）<ref>{{Cite web |url=http://www.681busterminal.com/version2/index.php?fi=a201&id=482 |title=681巴士總站 - 新界專線小巴112S線 |access-date=2011-02-04 |archive-url=https://web.archive.org/web/20160305095446/http://www.681busterminal.com/version2/index.php?fi=a201&id=482 |archive-date=2016-03-05 |dead-url=no }}</ref><ref>[http://www.td.gov.hk/tc/transport_in_hong_kong/public_transport/minibuses/green/gmb_online_guide/new_territories_gmb_routes/nt_112s/index.html 運輸署 - 新界區專線小巴112S號線] {{webarchive|url=https://web.archive.org/web/20111024073202/http://www.td.gov.hk/tc/transport_in_hong_kong/public_transport/minibuses/green/gmb_online_guide/new_territories_gmb_routes/nt_112s/index.html |date=2011-10-24 }}</ref>，是[[新界區專線小巴112M線|112M]]的深宵班次，於2011年2月15日凌晨起投入服務。<ref>{{Cite web |url=http://www.lohas-park.com/thread-2340-1-1.html |title=日出康城(首都,領都,領峯)討論區：112M, 112S小巴線2月14日投入服務  |access-date=2011-02-04 |archive-url=https://web.archive.org/web/20110303084843/http://www.lohas-park.com/thread-2340-1-1.html |archive-date=2011-03-03 |dead-url=no }}</ref><ref>《小巴net第十八期》，2012年</ref>

==歷史==
*2009年12月18日：政府刊憲公開招標4組專綫小巴路線<ref>[http://www.gld.gov.hk/cgi-bin/gld/egazette/gazettefiles.cgi?lang=c&year=2009&month=12&day=18&vol=13&no=51&gn=7928&header=1&part=0&df=1&nt=gn&newfile=1&acurrentpage=12&agree=1&gaz_type=mg 第7928號公告：《道路交通 (交通管制) 規例 (第374章)--香港島、九龍及新界的公共小型巴士（專線）路線組別》] {{Wayback|url=http://www.gld.gov.hk/cgi-bin/gld/egazette/gazettefiles.cgi?lang=c&year=2009&month=12&day=18&vol=13&no=51&gn=7928&header=1&part=0&df=1&nt=gn&newfile=1&acurrentpage=12&agree=1&gaz_type=mg |date=20101001044909 }}，香港特別行政區政府憲報第13卷第51期，2009年12月18日。</ref>，其中包括本線，然而最後並未成功開辦。
*2010年7月16日：政府刊憲公開招標2組專綫小巴路線，其中第二組的深宵路線即為112S線<ref>[http://www.gld.gov.hk/cgi-bin/gld/egazette/gazettefiles.cgi?lang=c&year=2010&month=7&day=16&vol=14&no=28&gn=4477&header=1&part=0&df=1&nt=gn&newfile=1&acurrentpage=12&agree=1&gaz_type=mg 第4477號公告：《道路交通 (公共服務車輛) 規例 (第374D章)--九龍及新界的公共小巴 (專線) 路線組別]》，香港特別行政區政府憲報第14卷第28期，2010年7月16日。</ref>。
*2011年2月15日：本路線由當日凌晨起投入服務<ref>[http://www.td.gov.hk/tc/traffic_notices/index_id_24766.html 開辦新界專線小巴第112M及112S號] {{Wayback|url=http://www.td.gov.hk/tc/traffic_notices/index_id_24766.html |date=20110618053256 }}，運輸署交通通告，2011-02-14</ref>，為將軍澳南、日出康城及將軍澳工業村第一條通宵專線小巴路線。
*2012年6月17日：往[[將軍澳站公共運輸交匯處|將軍澳站]]方向繞經日出康城（[[日出大道|日出大道]]），並增設日出大道往將軍澳站之$6.00分段收費。<ref>{{Cite web |url=http://www.lohas-park.com/thread-56924-1-1.html |title=日出康城(首都,領都,領峯)討論區：112M 於6月17日正式駛入日出大道 |access-date=2012-06-21 |archive-url=https://web.archive.org/web/20130413012237/http://lohas-park.com/thread-56924-1-1.html |archive-date=2013-04-13 |dead-url=no }}</ref>

==服務時間及班次==
*[[將軍澳站公共運輸交匯處|將軍澳站公共運輸交匯處]]開：0045-0245（30分鐘一班）
*[[將軍澳工業邨|將軍澳工業邨]]（駿昌街）開：0045-0245（30分鐘一班）

然而由於客量偏低，實際上01:30以後已經沒有車輛行走。

==收費==
*全程收費：$7
**[[日出大道|日出大道]]往將軍澳站：$6
（此線所有車輛均接受[[八達通|八達通]]卡付款；不設找續。）

==行車路線==
*[[將軍澳站公共運輸交匯處|將軍澳站公共運輸交匯處]]開：經[[唐德街|唐德街]]，[[唐俊街|唐俊街]]，[[寶邑路|寶邑路]]，[[環保大道|環保大道]]，[[駿宏街|駿宏街]]，[[駿才街|駿才街]]，[[駿日街|駿日街]]，將軍澳工業邨巴士總站，駿宏街，[[駿光街|駿光街]]及[[駿昌街|駿昌街]]。
*[[將軍澳工業邨|將軍澳工業邨]]（駿昌街）開：經駿昌街，駿光街，駿宏街，駿日街，駿才街，駿宏街，環保大道，寶邑路及唐俊街。
