'''新鎌谷車站'''（{{jpn|j=新鎌ヶ谷駅|hg=しんかまがやえき|rm=Shin-kamagaya eki|f=y}}）是位於[[日本|日本]][[千葉縣|千葉縣]][[鎌谷市|鎌谷市]]{{link-ja|新鎌谷|新鎌ケ谷}}，是[[北總鐵道|北總鐵道]]、[[京成電鐵|京成電鐵]]、[[新京成電鐵|新京成電鐵]]、[[東武鐵道|東武鐵道]]的[[鐵路車站|鐵路車站]]。是鎌谷市內唯一的[[轉乘站|轉乘站]]，所有通過市內的鐵路線都停靠此站。

過去[[都營新宿線|都營新宿線]]有延伸本站的計畫，但已在2010年（平成22年）廢止<ref name="chibanippo201394">{{Cite news
  | title = 新鎌ケ谷への延伸計画廃止 都営新宿線
  | newspaper = 千葉日報
  | pages = 朝刊 4
  | publisher = 千葉日報社
  | date = 2013-09-04
 }}</ref>。

== 停靠路線 ==
此站有[[北總鐵道北總線|北總鐵道北總線]]、[[京成成田機場線|京成成田機場線]]（愛稱「成田Sky Access線」）、[[新京成電鐵新京成線|新京成電鐵新京成線]]、[[東武野田線|東武鐵道野田線]]（愛稱「東武都市公園線」）停靠。當中，京成電鐵借用北總鐵道的軌道運行（北總是此站前後路段的第一種鐵道事業者、京成是第二種鐵道事業者，共用設施），視作同一條路線。因此，此站實質上是3條路線的轉乘站。

3路線各自擁有[[車站編號|車站編號]]，北總（京成）是'''HS08'''、新京成是'''SL11'''、東武是'''TD 30'''。

== 簡介 ==
過去東武[[野田線|野田線]]與[[新京成線|新京成線]]沒有在交會點附近設立車站，兩路線上最接近的兩個車站——東武的[[鎌谷車站|鎌谷]]與新京成的{{link-ja|初富車站|初富駅|初富}}之間有1[[公里|公里]]的距離，導致轉車困難。直到1991年時[[北總線|北總線]]（當時稱為北總、公團線）第二期[[京成高砂站|京成高砂]]～新鎌谷之間路段通車後，北總、公團線新鎌谷車站啟用。1年後1992年新京成線與北總線廢除[[直通運轉|直通運轉]]，新京成車站啟用。

1989年，東武在野田線與新京成線交會處附近設置「新鎌谷信號所」。在地方公共團體出資下，東武於1999年增設野田線新鎌谷站。

北總線月台在1979年通車時就規劃為2面4線，1991年本站啟用時僅使用北側的1面2線，南側月台長期放置。之後考慮到成田Sky Access線所帶來的輸送力，2007年啟用南側月台。

2010年7月17日，[[京成電鐵|京成電鐵]]的[[成田機場線|成田機場線]]（{{lang-ja|成田空港線}}，開發初期原名「成田新高速鐵道」）通車，與北總鐵道共用月台設施，使得新鎌谷從原本的三家業者增加成四家業者共用的車站。新鎌谷是成田機場線次快列車「[[Access特快號列車|Access特快]]」（{{lang|ja|アクセス特急}}）的停車站之一。配合停靠班次的增加，北總線月台南側增建一座京成電鐵專用的[[島式月台|島式月台]]，成為2座月台4條路線的配置。
== 歷史 ==
===設站前 ===
* 1923年（[[大正|大正]]12年）12月27日 - 北總鐵道船橋線（今日的東武鐵道野田線）[[船橋車站|船橋]]～[[柏車站|柏]]之間路段通車。
* 1955年（[[昭和|昭和]]30年）4月21日 - 新京成線鎌谷初富車站（{{lang|ja|鎌ヶ谷初富駅}}，也就是今日的{{link-ja|初富車站|初富駅}}）～[[松戶車站|松戶]]間路段通車，野田線與新京成線互相交叉。

== 車站結構 ==
北總線、京成成田機場線是[[高架車站|高架車站]]，新京成線是[[地面車站|地面車站]]（正在進行高架化工程），兩站體並列因此轉乘容易。東武野田線是在土塹中，位於北總線、新京成線車站東側，轉乘距離稍遠。各線檢票口在同一平面上，高架下空間是商店與轉乘通道。

所有站區都可使用{{link-ja|docomo Wi-Fi|docomo Wi-Fi}}與{{link-ja|Wi2|ワイヤ・アンド・ワイヤレス}}公共無線網路服務。

本站是[[島式月台|島式月台]]2面4線高架車站，站房在高架下，月台設有電扶梯、電梯與候車室。

2007年11月改建為2面4線（待避站）後，本站可進行待避與[[緩急接續|緩急接續]]。
2010年7月17日起，京成電鐵運行的Access特急停靠本站，從此本站共有四間公司提供服務。

由於是北總鐵道管轄站。因此車站編號採用北總的'''HS08'''<ref>京成時刻表Vol.25 2010年7月9日</ref>。車站外觀雖然僅顯示北總鐵道，但在剪票口等指標會以「北總線、成田Sky Access線」表示。
站內北總鐵道的售票機可購入新京成線的車票以及經京成高砂站的京成線連絡車票。

本站是新京成線唯一在平成時代開設的車站。

月台由地面1面1線與高架1面1線組成。檢票口等站房設備與北總鐵道共用，剪票業務也委託北總鐵道管理。

新京成電鐵設有可發售、儲值PASMO的定期券售票機（北總鐵道的定期券售票處無法購入新京成單獨定期券）。新京成線各站乘車券、回數券可在北總鐵道售票機購入（無法購入新京成線其它車站發售的經[[京成津田沼站|京成津田沼站]]往京成線各站（本站僅可購入經北總線京成高砂站往京成線車票）、經[[北習志野站|北習志野站]]往[[東葉高速線|東葉高速線]]與JR線的連絡乘車券）。

開業時為[[相對式月台|地面相對式月台]]2面2線構造。因應高架化工程，2014年2月22日在臨時線啟用臨時島式月台，設有地下通道連通站房與月台。2017年10月，下行線高架月台啟用，2018年4月完成站房與上行線地面月台的連絡階梯與斜坡，關閉地下通道<ref>{{Cite press release |title=新鎌ヶ谷駅上り線ホーム(松戸方面)への通路が変更(4/1) |url=http://www.shinkeisei.co.jp/topics/2018/12760/ |publisher=新京成電鉄 |date=2018-03-09 |accessdate=2018-07-16}}</ref>。高架化工程完成後，本站將成為高架島式月台構造。

從本站步行前往前後的{{link-ja|北初富站|北初富駅}}與{{link-ja|初富站|初富駅}}約需10分鐘左右。

== 車站周邊 ==
[[File:Kamagaya_Police_Station_Shin-Kamagaya_Ekimae_Koban.jpg|thumb]]
[[File:Keiyo_Bank_193-Shin-Kamagaya_Branch.jpg|thumb]]
本站過去並不在市區內，周邊多為農地與[[梨|梨]]園，除了稍遠處的市役所廳舍，沒有其他重要建築與設施。2000年代起周邊開始開發，加上2004年以後大型購物中心於站前開幕，逐漸展現與過去不同的風貌。

車站周邊在以前稱作「{{link-ja|初富|初富}}」{{Refnest|group="注釋"|新京成線北鄰車站北初富站的地名是「{{link-ja|中澤 (鎌谷市)|中沢 (鎌ケ谷市)|北中澤}}」、南鄰車站初富站的地名是「{{link-ja|中央 (鎌谷市)|中央 (鎌ケ谷市)|中央}}」。}}，2008年2月25日實施{{link-ja|住居表示|住居表示}}<ref>{{Cite web |date=2007-10-15 |title=鎌ケ谷市告示第74号 住居表示の実施に伴う字区域の変更及び町区域の設定の案 |url=http://www.city.kamagaya.chiba.jp/news/shiminn/jyuukyohyouzi.pdf |format=PDF |publisher=鎌ケ谷市 |accessdate=2015-05-21 |archive-url=https://web.archive.org/web/20150617040445/http://www.city.kamagaya.chiba.jp/news/shiminn/jyuukyohyouzi.pdf |archive-date=2015-06-17 |dead-url=yes }}</ref>，新式「新鎌谷一丁目」、「新鎌谷二丁目」、「新鎌谷三丁目」、「新鎌谷四丁目」，站名成為正式地名{{Refnest|group="注釋"|一樣以站名作為地名的案例還有北總鐵道{{link-ja|牧之原 (印西市)|牧の原 (印西市)|（印西）牧之原}}、東武鐵道[[中板橋|中板橋]]，以及其他公司的[[國立市|國立]]、{{link-ja|西船 (船橋市)|西船 (船橋市)|西船}}、{{link-ja|新橫濱|新横浜}}、{{link-ja|南浦和|南浦和}}等。}}。
