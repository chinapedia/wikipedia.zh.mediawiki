'''新世橋停留場'''（{{jpn|j=新世橋停留場|hg=しんせいばしていりゅうじょう|rm=Shinseibashi teiryūjō|f=y}}）是位於[[北海道|北海道]][[函館市|函館市]]宮前町的[[函館市交通局|函館市交通局]]（函館市電車）宮前線已被廢除的[[鐵路車站|鐵路車站]]。

== 歷史 ==
* 車站的名稱曾經出現「新世橋停留場」及「新世橋前停留場」。
* 參考[[函館市交通局|函館市交通局]]的歷史。

== 車站構造 ==
* 新世橋停留場設有2個月台（[[側式月台|側式月台]]）、2條電車路線（相反方向）。

== 車站周邊 ==
* [[北海道道517號五稜郭公園線|北海道道517號五稜郭公園線]]
* 天主教宮前町教會
* 函館藤幼稚園
* [https://web.archive.org/web/20100630083738/http://yamagataya.hakodate.jp/ 山形屋]
* 現在車站附近亦有石川內科醫院及宮林外科醫院。
