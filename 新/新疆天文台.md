'''中国科学院新疆天文台'''位于[[中华人民共和国|中华人民共和国]][[新疆维吾尔自治区|新疆维吾尔自治区]][[乌鲁木齐市|乌鲁木齐市]]，成立于1957年，原名为[[中国科学院|中国科学院]]乌鲁木齐[[人造卫星|人造卫星]]观测站，1987年更名为中国科学院乌鲁木齐天文站，2001年4月更名为中国科学院国家天文台乌鲁木齐天文站，2011年1月更为现名并于当年8月正式揭牌<ref name="xao_named">{{ cite web |title = 新疆天文台更名揭牌 | url = http://www.xao.cas.cn/xwzx/tpxw/201108/t20110829_3329035.html |publisher = 中国科学院新疆天文台 |accessdate =2011-08-29 }}</ref>。新疆天文台研究领域包括[[射电天文|射电天文]]、[[光学天文|光学天文]]与技术应用、应用天文，主要从事[[脉冲星|脉冲星]]、[[恒星形成|恒星形成]]与演化、[[星系|星系]][[宇宙学|宇宙学]]、[[高能|高能]][[天体物理|天体物理]]、[[微波|微波]][[接收机|接收机]]、数字技术应用、[[空间|空间]]目标与[[太空垃圾|碎片]]、[[卫星导航系统|卫星导航]]等方面的研究与观测。<ref name="xao_intro">{{cite web |title = 中国科学院新疆天文台简介| url = http://www.xao.ac.cn/jggk/wzjj/ |publisher=中国科学院新疆天文台 |accessdate = 2013-09-29}}</ref>

==观测台站==
===南山基地===
[[File:XAO_entrance.jpg|thumb]]
[[File:XAO_Nanshan_25-m_radio_telescope_2007-08-22.jpg|thumb]]

新疆天文台'''南山基地'''始建于1991年，位于乌鲁木齐市乌鲁木齐县甘沟乡，海拔2080米，距乌鲁木齐市区75公里，坐标为： [https://maps.google.com/maps?q=43.471476+N,87.178096+E&ll=43.471476,87.178096&spn=0.001131,0.002411&sll=39.938884,116.397459&sspn=0.611751,1.234589&t=h&z=19&iwloc=near 43.471476 N, 87.178096 E]。南山基地拥有25米射电望远镜系统、GPS数据接收系统和太阳色球望远镜等仪器设备。其中25米[[射电望远镜|射电望远镜]]目前承担的科学任务有脉冲星、厘米波分子谱线和活动星系核等观测研究，同时它也是国际[[VLBI|VLBI]]网的重要站点。此外，南山基地也是中国探月工程中VLBI测轨系统的四个站点之一。<ref name="nanshan">{{cite web| title=南山天文观测基地| url = http://www.xao.ac.cn/jgsz/ywtz/nsjd/ | publisher = 中国科学院新疆天文台 | accessdate = 2013-09-29}}</ref>

南山25米射电望远镜目前的工作波段有1.3厘米、3.6厘米、6厘米、13厘米和18厘米，天线指向精度达到 18-{”}-，该望远镜几乎全年无间断运行，2010年累计工作天数349天，天线运行时间6998小时，有效观测时间6453小时，系统维护时间813小时<ref name="nanshan25m-report">{{cite web | title = 新疆天文台 25 米射电望远镜技术报告 | url = http://www.xao.ac.cn/kycg/jsbg/201307/P020130725649484285236.pdf | publisher = 中国科学院新疆天文台 | author = 刘祥，宋华刚，陈晨雨 | accessdate = 2007-09-10 }}</ref> <ref name="nanshan25m">{{cite web | title = 南山25米射电望远镜2010年工作情况 | url = http://www.xao.ac.cn/xwzx/kydt/201101/t20110110_3057425.html | publisher = 中国科学院新疆天文台 | author = 张华 | accessdate = 2011-01-10 }}</ref> <ref name="nanshan25m-mol" />。南山25米射电望远镜接受来自各地科研工作者的射电天文观测申请，[http://www.xao.ac.cn/gcsq/gcsq1/201206/P020120618345683919564.doc 可点击此处下载申请表]。

====脉冲星观测====
1996年2月，南山25米射电望远镜实现了中国首次[[脉冲星|脉冲星]]课题观测，随后建立了国内第一个脉冲星到达时间的观测系统。2002年，在对接收机进行制冷升级改造后，南山25米射电望远镜可观测脉冲星约270颗，2008年引进世界先进的脉冲星数字消色散系统（Digital Filter Bank, DFB）后，已实现对300颗脉冲星（包括10余颗[[毫秒脉冲星|毫秒脉冲星]]）的到达时间监测，该望远镜同时还开展了旋转射电暂现源（也称旋转射电瞬变源，来自英文 Rotating radio transient）、[[磁星|磁星]]、脉冲星巡天等观测工作。<ref name="nanshan-pulsar">{{cite web | title = 脉冲星观测研究 | url = http://www.xao.ac.cn/xwzx/kyzt/201010/t20101015_2987591.html | publisher = 中国科学院新疆天文台 | accessdate = 2010-06-12  }}</ref>

====分子谱线观测====
2010年，南山25米射电望远镜厘米波分子谱线观测系统的频谱终端更新为1GHz带宽、8192通道的双路DFB系统。2011年底1.3厘米波段高灵敏度双极化制冷接收机投入观测。目前该系统的主要工作波段为18厘米、6厘米和1.3厘米，可以观测羟基脉泽、甲醛吸收线、氢复合线、水脉泽、甲醇脉泽和氨分子等谱线。<ref name="nanshan25m-mol">{{cite web | title = 恒星形成与演化 | url = http://www.xao.ac.cn/xwzx/kyzt/201010/t20101015_2987589.html | publisher = 中国科学院新疆天文台 | accessdate = 2010-06-12  }}</ref>

====活动星系核观测====
南山25米射电望远镜可实现“有吉赫兹峰频谱的河外源（ Giga Hertz Peaked Spectrum，GPS 源）”、“日变源（Intra-day variability，IDV 源）”等[[活动星系核|活动星系核]]（Active Galactic Nucleus，AGN）观测研究。<ref name="nanshan25m-agn">{{cite web | title = 活动星系核 | url = http://www.xao.ac.cn/xwzx/kyzt/201301/t20130125_3757146.html |  publisher = 中国科学院新疆天文台 | accessdate = 2013-01-25 }}</ref>

=== 喀什卫星地面站 ===
'''喀什卫星地面站'''位于新疆[[喀什市|喀什市]]东湖小区墓士塔格东路，是中国区域定位系统在新疆自治区内的唯一测轨站，承担“中国区域定位系统”和“中国北斗卫星导航系统”等诸多任务，可提供包括电离层、气象、参考框架、精密时间传递、高分辨的推算地球自转速率及其变化、地壳运动等数据。<ref name="kashi">{{cite web| title=喀什卫星地面站简介 | url = http://www.xao.ac.cn/jgsz/ywtz/ksz/ | publisher = 中国科学院新疆天文台 | accessdate = 2013-09-29 }}</ref>

=== 巴音布鲁克GNSS基准站 ===
'''巴音布鲁GNSS基准站'''位于新疆[[巴音布鲁克区|巴音布鲁克区]]二乡，于2010年建成，是中国陆态网络260个基准站之一<ref>{{cite web |title = 巴音布鲁克GNSS基准站简介| url = http://www.xao.ac.cn/jgsz/ywtz/byblk/ | publisher = 中国科学院新疆天文台 |accessdate = 2013-09-29 }}</ref>。

===乌拉斯台GNSS基准站===
'''乌拉斯台GNSS基准站'''位于[[巴音郭楞蒙古自治州|巴音郭楞蒙古自治州]][[和静县|和静县]]阿拉沟乡查汗村。<ref>{{cite web |title = 乌拉斯台GNSS基准站简介 |url = http://www.xao.ac.cn/jgsz/ywtz/wlstz/ | publisher = 中国科学院新疆天文台 |accessdate = 2013-09-29}}</ref>

===110米射电望远镜（规划中）===
'''新疆奇台110米射电望远镜（QTT）'''是规划中的口径为110米全方位可转动射电望远镜，选址为新疆[[奇台县|奇台县]]天文谷，建成投入使用后，将与美国100×110米 GBT（Green Bank Telescope）、德国 Effelsberg 100米射电望远镜并列为世界最大的地面可动[[射电望远镜|射电望远镜]]，该望远镜也能满足中国在深空探测领域的需要<ref name="qitai">{{cite web |title = 新疆110米射电望远镜项目网站 |url = http://qtt.xao.cas.cn/ |publisher = 中国科学院新疆天文台 |accessdate = 2013-09-29 }}</ref>。
