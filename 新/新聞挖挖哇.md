《'''新聞挖挖哇！'''》是[[臺灣|臺灣]][[JET綜合台|JET綜合台]]的[[談話性節目|談話性節目]]，2002年1月7日於[[超級電視台|超視]]開播，2010年1月4日改在JET綜合台播映。前身為[[衛視中文台|衛視中文台]]夜間談話性節目《[[新聞e點靈|新聞e點靈]]》。

== 節目概要 ==
*討論主題主要為[[新聞|新聞]]事件，包羅萬象，無所不談。[[臺灣歷史|臺灣歷史]]、[[宗教|宗教]]、[[族群|族群]]、[[演藝|演藝]]、[[社會|社會]]、[[法律|法律]]、[[心理|心理]]、[[性別|性別]]、[[健康|健康]]、等議題皆概括其內；常也討論基層的民生[[經濟|經濟]]與[[政治|政治]]問題，有時甚至談論[[野史|野史]]、[[刑案|刑案]]、[[臺灣民間信仰|臺灣民間信仰]]、[[靈異|靈異]]現象。另外，本節目雖也如同一般談話節目寫有流程腳本，但是從不事先排演腳本（俗稱「Re稿」），因此時常會隨著話題走向而不按牌理出牌。

== 節目開場白 ==
*第一版：[[鄭弘儀|鄭弘儀]] ：「挖掘新聞內幕」 [[于美人|于美人]]：「挖苦新聞人物」 [[鄭弘儀|鄭弘儀]] ：「我是鄭弘儀。」 [[于美人|于美人]]：「我是于美人。」 一起：「歡迎收看今天的新聞挖挖哇！」（播出日期：2002年1月7日-2016年8月12日）
*第二版：[[鄭弘儀|鄭弘儀]] ：「挖掘新聞內幕，挖苦新聞人物，我是鄭弘儀，歡迎收看今天的新聞挖挖哇！」（播出日期：2016年8月15日-至今）
