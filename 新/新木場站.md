'''新木場站'''（{{jpn|j=新木場駅|hg=しんきばえき|rm=Shinkiba eki|f=y}}）是位於[[日本|日本]][[東京都|東京都]][[江東區_(東京都)|江東區]][[新木場|新木場]]一丁目，屬於[[東日本旅客鐵道|東日本旅客鐵道]]（JR東日本）、[[東京地下鐵|東京地下鐵]]、[[東京臨海高速鐵道|東京臨海高速鐵道]]（TWR）的[[鐵路車站|鐵路車站]]。

== 停靠路線 ==
此站有以下3間公司3條路線停靠，是轉乘站。

* JR東日本 [[File:JR_JE_line_symbol.svg|15px]] [[京葉線|京葉線]]：除了京葉線列車，還有[[西船橋站|西船橋站]]接入的[[武藏野線|武藏野線]]列車。依據[[特定都區市內|特定都區市內]]制度，本站屬於「[[特定都區市內#設定區域一覧|東京都區內]]」。車站編號為「'''JE 05'''」。
* 東京地下鐵 [[File:Subway_TokyoYurakucho.png|15px]] [[有樂町線|有樂町線]]：本站為終點站，車站編號為「'''Y 24'''」。
* 東京臨海高速鐵道 [[File:Rinkai_Line_symbol.svg|15px]] [[臨海線_(東京臨海高速鐵道)|臨海線]]：本站為起點站。並與[[東日本旅客鐵道|東日本旅客鐵道]]（JR東日本）[[埼京線|埼京線]]、[[川越線|川越線]][[直通運行|互相直通運行]]。車站編號為「'''R 01'''」。

== 历史 ==
* 1988年（[[昭和|昭和]]63年）
** 6月8日：[[帝都高速度交通營團|帝都高速度交通營團]]（營團地下鐵）有樂町線新木場站開業。
** 12月1日：JR東日本京葉線通車，JR東日本站區同日開業，為京葉線終點站。
* 1990年（[[平成|平成]]2年）3月10日：京葉線新木場－[[東京站|東京]]段延線通車（京葉線全線通車）。
* 1996年（平成8年）3月30日：東京臨海高速鐵道臨海副都心線（現稱臨海線）新木場－[[東京電訊港站|東京電訊港]]段通車，東京臨海高速鐵道站區同日開業。
* 2001年（平成13年）11月18日：JR東日本的智能卡系統[[Suica|Suica]]正式投入服務。
* 2002年（平成14年）12月1日：臨海線[[天王洲島站|天王洲島]]－[[大崎站|大崎]]延線開業（臨海線全線開通），臨海線並開始與[[埼京線|埼京線]]相互直通運行。
* 2004年（平成16年）
** 4月1日：營團地下鐵民營化，車站由東京地下鐵接手管理。
** 10月16日：JR東日本訂正列車時刻表，通勤快速列車增停新木場站。
* 2007年（平成19年）
** 3月18日：東京地下鐵啟用IC卡「[[PASMO|PASMO]]」。
** 3月24日：京葉線月台引入發車音樂。1號線為《海岸通り》、2號線為《シンコペーション》，皆由櫻井音樂工房製作。
* 2008年（平成20年）5月3日：東京地下鐵增設來往新木場站及[[小田急電鐵|小田急電鐵]][[小田原線|小田原線]][[本厚木站|本厚木站]]的不定期臨時列車班次「Bay Resort」號。
* 2011年（平成23年）9月24日：「Bay Resort」號。結束營運。
* 2013年（平成25年）6月29日：有樂町線月台啟用月台門。
* 2016年（平成28年）9月25日：JR月台導入[[東京圈輸送管理系統|東京圈輸送管理系統（ATOS）]]，同時啟用進站音樂<ref>[http://response.jp/article/2016/09/28/282595.html JR京葉線のホームに新たな列車接近音　9月25日から使用開始] - レスポンス、2016年9月28日、同年9月29日閲覧。</ref>。

== 車站結構 ==
本站是一[[高架車站|高架車站]]，所有站區均高於地面，包括東京地下鐵的站區。車站地面層及附近高架路段下的空間是[[食店|食店]]等店舖，2樓是有樂町線與臨海線月台，3樓是三家公司共用的車站大堂（由東京地下鐵負責管理），4樓則是京葉線月台。由於三家公司的剪票口非常接近，因此換乘非常方便。可是也不少乘客因此而誤進其他公司的站區。而臨海線站區則在剪票口側設有臨時剪票口，供車站非常繁忙時使用。

本站是島式月台1面2線高架車站。車站月台與大堂間只以兩條樓梯連接，繁忙時間會稍見擠擁，需要較多時間才能通過剪票口。

過去通勤快速過站不停。

本站是{{link-ja|東日本旅客鉄道千葉支社|東日本旅客鉄道千葉支社|千葉支社}}管轄的直營站，設有[[綠色窗口|綠色窗口]]、指定席售票機、自動售票機、自動驗票閘門、自動精算機。本站為管理站，管理[[潮見站|潮見站]]、[[葛西臨海公園站|葛西臨海公園站]]。

京葉線在部分通車時，列車使用蘇我方向的剪式橫渡線折返，在本站與東京站區間通車後停止使用，無法再進行折返運行。但2006年9月28日東京站機器室發生火災時，本站在整備連動裝置後恢復折返功能。從此持續保留此機能。

蘇我側有連通臨海線的[[聯絡線|連絡線]]。全線通車前，[[東京臨海高速鐵道70-000形電車|70-000形電車]]可利用此線前往京葉電車區（現{{link-ja|京葉車輛中心|京葉車両センター}}）或經蘇我前往大井工場（現{{link-ja|東京綜合車輛中心|東京総合車両センター}}），也可提供團體臨時列車運行使用。

由於轉乘有樂町線的轉乘客眾多，因此原先過站不停的通勤快速在2004年（平成16年）10月16日改點起增停本站。2009年（平成21年）3月6日至6月26日（3月20日除外）的週五與3月19日的「細波21號」增停本站。

====月台配置====
月台採島式月台1面2線設計。近蘇我一邊設有交叉道岔，在京葉線延伸至東京站前供列車折返。京葉線延伸至東京站後曾經停用，但東京站於2006年9月28日發生火災後，此道岔立即被重新啟用。現時當列車班次出現嚴重延誤時，會使用此道岔調整班次。

== 使用情況 ==

== 其他 ==
* 建設站體時，原先打算以木材作為主要建材以代表此地，但因東京都法規禁止建造3樓以上的木材建築，因此改作為裝飾材料。
* 開業前，營團地下鐵有樂町線計畫（當時終點為[[新富町站|新富町站]]）稱為「灣岸站」，京葉線計畫（當時終點為西船橋站）稱為「新砂町站」。
* 依據最初計畫，京葉線作為{{link-ja|東京外環狀線|東京外環状線}}的一部分，將從新木場經台場延伸至{{link-ja|東京貨物總站|東京貨物ターミナル駅}}，沒有新木場至東京的計畫。之後雖然完成新木場至東京貨物總站之間的隧道，但京葉線改為延伸至東京站，而由新設的臨海線繼承（東京電訊港 - 東臨運輸區間作為回送線）。蘇我側的連絡線便是最初計畫所留下的設備。
