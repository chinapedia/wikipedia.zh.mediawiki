'''新時代廣場'''（英語：'''New Trend Plaza'''）是[[香港|香港]][[炮台山|炮台山]][[英皇道|英皇道]]278-288號北角中心大廈的1及2樓商場，地下則名為'''英皇柏麗大道購物中心'''（英語：'''King's Park Lane'''）。

新時代廣場在1974年啟用，曾出租予[[裕華國貨|裕華國貨]]。1993年，當時的業主均偉投資有限公司替新時代廣場進行翻新，並將商場名稱改為新時代廣場和分拆成180個鋪位出售。<ref>{{Cite web |url=https://www.hk01.com/社區/80275/-%E7%94%9F%E5%8C%96%E5%A3%BD%E5%B1%8D-%E4%B8%89-%E5%95%86%E5%A0%B4%E6%AC%A0%E7%AE%A1%E7%90%86-%E8%88%96%E9%A0%AD%E6%9B%B4%E8%B2%BC%E5%9C%B0-%E7%82%BA%E4%BD%95%E4%BB%8D%E7%84%B6%E5%8D%81%E5%AE%A4%E4%B9%9D%E7%A9%BA-|title=【生化壽屍．三】商場欠管理　舖頭更貼地　為何仍然十室九空？|date=2017-03-26|publisher=香港01}}</ref>

1990年代，新時代廣場的租戶包括遊戲卡店、漫畫店、玩具店、精品店和影印店等。現時，新時代廣場有不少空置的鋪位，餘下的租戶主要是補習社、僱傭代理和中醫診所。<ref name=hk01part1>{{Cite web |url=https://www.hk01.com/社區/80196/-%E7%94%9F%E5%8C%96%E5%A3%BD%E5%B1%8D-%E4%B8%80-%E5%8C%97%E8%A7%92%E6%96%B0%E6%99%82%E4%BB%A3%E5%BB%A3%E5%A0%B4%E5%BB%BF%E5%B9%B4%E5%86%87%E8%A3%9D%E4%BF%AE-%E6%8B%8D%E5%AE%8C%E9%9B%BB%E5%BD%B1%E8%AE%8A%E6%AD%BB%E5%A0%B4-|title=【生化壽屍．一】北角新時代廣場廿年冇裝修　拍完電影變死場？|date=2017-03-26|publisher=香港01}}</ref><ref>{{Cite web |url=https://www.hk01.com/社區/80197/-%E7%94%9F%E5%8C%96%E5%A3%BD%E5%B1%8D-%E4%BA%8C-%E5%8C%97%E8%A7%92%E6%96%B0%E6%99%82%E4%BB%A3%E5%BE%9E%E4%BE%86%E5%86%87%E5%A3%BD%E5%8F%B8-%E5%8F%8D%E6%98%AF%E5%AF%B6%E9%A6%AC%E5%B1%B1%E5%AD%B8%E7%94%9F%E7%86%B1%E9%96%80%E8%92%B2%E9%BB%9E|title=【生化壽屍．二】北角新時代從來冇壽司？反是寶馬山學生熱門蒲點|date=2017-03-26|publisher=香港01}}</ref>
