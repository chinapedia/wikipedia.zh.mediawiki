'''新碶街道'''，是[[中国|中国]][[浙江省|浙江省]][[宁波市|宁波市]][[北仑区|北仑区]]的一个[[街道_(行政区划)|街道]]和北仑区政府所在地。东临[[东海|东海]]与[[舟山市|舟山市]][[金塘岛|金塘岛]]相望，南面与[[霞浦街道|霞浦街道]]相连，西面与[[小港街道_(宁波市)|小港街道]]、[[大碶街道|大碶街道]]相连，北面与[[戚家山街道|戚家山街道]]相接。总面积46.3平方公里，人口13万，[[街道办事处|街道办事处]]位于新建路2号。

== 沿革 ==
新碶得名于境内的永丰碶。由于建设时间晚于东西的太和碶和备碶因而称新碶。新碶街道原为镇海县新碶镇。1985年设立北仑区后，新碶镇成为北仑区政府所在地。

== 下辖 ==
目前，新碶街道辖有紫荆、海棠、红梅、芝兰、杜鹃、牡丹、芙蓉、百合、迎春、玉兰、银杏、大港、高塘共13个社区，大路、星阳、高潮、横浦、贝碶、隆顺、小山、下史、千丈、向家、备碶、沿海、凤洋、永丰、算山、许胡、永久、岭南、妙林、高塘、大同、镇安、大树、五星共24个行政村。

== 经济 ==
[[北仑港|北仑港]]位于新碶街道，是中国大陆四大深水港之一。[[北仑发电厂|北仑发电厂]]位于新碶街道，是中国大陆第一座装机容量达三百万千瓦的巨型火电厂。宁波经济技术开发区和宁波保税区也位于新碶街道，下辖许多大型民营企业。
*[[申洲集团|申洲集团]]，中国著名针织服装出口企业
*[[宝新不锈钢|宝新不锈钢]]，[[宝钢|宝钢]]集团投资钢铁企业
