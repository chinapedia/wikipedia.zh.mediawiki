'''新世界街'''（{{lang-pl|Ulica Nowy Świat}}）是[[华沙|华沙]]主要的干道之一。 

新世界街是[[皇家之路_(华沙)|皇家之路]]（{{lang|pl|Trakt królewski}}）的一段，皇家之路从华沙的[[皇家城堡_(华沙)|皇家城堡]]和[[华沙老城|华沙老城]]，向南延伸到[[约翰三世_(波兰)|约翰三世]]的17世纪王宫维拉诺（{{lang|pl|Wilanów}}）。

==地理==
新世界街南起[[三十字广场|三十字广场]]（''Plac Trzech Krzyży''），从此向北，与[[耶路撒冷大道|耶路撒冷大道]]（Aleje Jerozolimskie）和[[圣十字街|圣十字街]]（ulica Świętokrzyska）交汇。新世界街北到[[华沙大学|华沙大学]]校园和[[哥白尼|哥白尼]]纪念碑附近，再向北就是[[克拉科夫郊区街|克拉科夫郊区街]]（Krakowskie Przedmieście），最后到达[[皇家城堡_(华沙)|皇家城堡]]。

从新世界街南端的[[三十字广场|三十字广场]]继续向南，就变成了 elegant 乌亚兹多夫大道（Aleje Ujazdowskie），然后是贝尔韦街（ulica Belwederska）和[[约翰三世_(波兰)|梭毕也斯基]]街（ulica Sobieskiego），最后到达维拉诺（Wilanów）。

==历史==
[[File:Warsaw6df.jpg|thumb]]
[[File:Władysław_Podkowiński,_Ulica_Nowy_Świat_w_Warszawie_w_dzień_letni.jpg|thumb]]
直到16世纪，新世界街是通往[[华沙|华沙]]以南无数[[贵族|贵族]]宫殿和村庄的一条主要道路。这条街现在的名称始于17世纪这座城市大发展之后（约1640年）<ref name="warsawtour">{{pl icon}} {{cite web  |author = |url = http://www.warsawtour.pl/index.php?id=56&pid=1593121032 |title = Trakt Królewski  |work = www.warsawtour.pl |publisher =  |pages =  |page =  |date = |accessdate = 2008-09-19}}</ref><ref name="spacery">{{pl icon}} {{cite web|author =|url =http://spacery-po-warszawie.w.interia.pl/ns.html|title =Nowy Świat|work =|publisher =|pages =|page =|date =|accessdate =2008-09-19}}{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>。到18世纪，沿着新世界街的农田已经变为密集的街市，大部分是木质的庄园府邸<ref name="warsawtour" />。

在[[拿破仑|拿破仑]]时期，[[华沙|华沙]]迅速增长，新世界街几乎进行彻底重建。木质庄园让位给砖石建筑，大部分是三层高的[[新古典主义|新古典主义]]建筑。到19世纪末，新世界街已经成为[[华沙|华沙]]的主要商业街之一<ref name="spacery" />，也是交通最繁忙的街道之一，拥有许多商店和餐馆。较大的新建筑取代了原来的建筑，这条街的特征也发生了变化。在20世纪初，几乎所有新古典主义建筑的痕迹全部消失，因为新的建筑大多是[[新艺术运动|新艺术运动]]风格。

在1944年8月到10月[[华沙起义|华沙起义]]期间，新世界街几乎被彻底摧毁<ref name="warsawtour" />。战争结束后，决定重建新世界街。由于恢复到战前新艺术运动状态，会过于高昂，于是又回到了19世纪初的外观。
