'''新华门'''，位于[[北京市|北京市]][[西城区|西城区]][[西长安街|西长安街]]中段北侧，是[[中南海|中南海]]的正门。

==简介==
[[File:GATEWAY_TO_FORMER_IMPERIAL_PALACE.jpg|thumb]]拍摄。当时“新华门”匾额悬挂在二层。]]
新华门原名'''宝月楼'''，始建于[[清朝|清朝]][[乾隆|乾隆]]二十三年（1758年）。该年春，[[乾隆帝|乾隆帝]]登[[瀛台|瀛台]]南眺，嫌[[太液池|太液池]]南岸空旷无屏蔽，乃下令建造此楼。起初拟建三层，后觉得太侈费，便改建为二层，东西面阔七间，南北纵二丈。同年秋竣工。因为月映池中，楼立池边，池和月共呈楼前，故定名“宝月楼”。楼上悬有乾隆帝御书“仰观俯察”匾额，以及“佳兴四时同，图呈苑里；清光千里共，鉴彻池心”等楹联<ref name=dong>冬泉，新华门，档案工作1989年06期</ref>。

宝月楼并不与外界相通，其正南是西苑的南墙（也是[[北京皇城|皇城墙]]），宝月楼位于墙内。墙外正南是回子营（又称回回营），从西域迁来的[[回部|回部]]移民聚居于此，并新建有清真礼拜寺（即[[清真普宁寺|清真普宁寺]]，俗称回回营清真寺）。因而民间传说，宝月楼是乾隆帝为[[香妃|香妃]]而建，香妃登上宝月楼可见到墙外的清真寺，略解乡愁，民间也根据该传说将宝月楼附会为望乡楼<ref name=dong/>。
[[File:Bundesarchiv_Bild_137-009052,_Peking_-_Sowjet._Botschafter_Karachan_fährt_zur_Audienz.jpg|thumb]]时期的新华门，图为[[苏联|苏联]]驻华大使[[加拉罕|加拉罕]]到中南海。]]
[[中华民国|中华民国]]成立后，[[袁世凯|袁世凯]]任[[中华民国临时大总统|中华民国临时大总统]]（后改为[[中华民国大总统|中华民国大总统]]），[[中华民国临时政府_(袁世凯)|袁世凯政府]]接管了清室的西苑三海，1913年将中海和南海辟为[[中华民国大总统府|大总统府]]，从此中海和南海合称“[[中南海|中南海]]”<ref name=zuobie>{{Cite web |url = http://news.163.com/09/0420/15/57BQI3T800011SM9.html |title = 作别老长安街 |author =  |date =2009-04-20  |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref>。1913年，在内务总长兼京都市政公所督办[[朱启钤|朱启钤]]主持下，将宝月楼改建为中南海正门，称“'''新华门'''”。将新华门前的一段[[西长安街|西长安街]]命名为“府前街”，将中南海西侧新辟道路命名为“[[府右街|府右街]]”（“府”均指大总统府）。此后，又将[[六部口|六部口]]以南的排水沟上改建的道路命名为“新华街”（今[[北新华街|北新华街]]、[[南新华街|南新华街]]），还在新华门对面的“府前街”南侧砌筑一道[[花墙_(西长安街)|西洋式花墙]]，以遮挡背后杂乱破旧的平房，改善大总统府门前的观瞻<ref name=weixiu>{{Cite web |url =http://www.takefoto.cn/viewnews-868753.html  |title =北京中南海新华门开始维修 崭新面貌值得期待  |author =  |date =2016-08-03  |publisher = 北晚新视觉 |language =  |accessdate =  |quote =  }}</ref>。同时，此次工程中还拆除了回回营清真寺位于北面的门楼式的正门<ref name=xiangfei>{{Cite web |url =  http://finance.ifeng.com/city/msls/20090609/767252.shtml|title = 东安福胡同：一座与香妃有关的清真寺 |author =  |date = 2009-06-09 |publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref><ref name=zuobie/>。

在此次改建工程中，将宝月楼南侧的一段皇城墙拆除，在东西两侧新砌两堵八字墙，将皇城墙和宝月楼连接起来。又在楼北面濒太液池处兴建了一座大影壁，以遮挡外人的视线。“新华门”匾额由[[袁励准|袁励准]]书写，匾高0.64米，宽1.6米，字体为[[馆阁体|馆阁体]]，此匾至今仍悬于门上<ref name=weixiu/>。门前放置一对石狮，据说是原[[端王府|端王府]]的遗物<ref>{{Cite web |url = http://book.people.com.cn/n/2012/1107/c69360-19520142-2.html |title =让乾隆念念不忘的神秘女人 揭秘香妃的真实身份  |author =  |date = 2012-11-07 |publisher = 人民网 |language =  |accessdate =  |quote =  }}</ref>。据称过去这是北京旧城内最大的一对石狮<ref name=weixiu/>。
[[File:新华门_-_panoramio.jpg|thumb]]”五个字]]
[[中华人民共和国|中华人民共和国]]成立后，中南海成为[[中共中央|中共中央]]及[[中华人民共和国中央人民政府_(1949–1954)|中央人民政府]]（1954年撤销，改设[[中华人民共和国国务院|国务院]]）的所在地。“[[文化大革命|文化大革命]]”爆发后，1966年8月[[红卫兵|红卫兵]]在北京城“[[破四旧|破四旧]]”，新华门前的石狮也被红卫兵视为“破四旧”的目标。1966年8月26日，国务院总理[[周恩来|周恩来]]从保护文物古迹出发，耐心说服了红卫兵，下令将新华门前的石狮迁走，从而将其保护起来。后来这对石狮又回到了原处<ref name=weixiu/>。

约在1967年5、6月份，在新华门，用木头刻了“[[为人民服务|为人民服务]]”五个红字粘在门内的影壁上；用木头刻了两条标语“伟大的[[中国共产党|中国共产党]]万岁！”“战无不胜的[[毛泽东思想|毛泽东思想]]万岁！”分别粘在门前两侧的八字墙上<ref name=weixiu/>。1970年左右，中南海修缮处对新华门影壁上的“为人民服务”五个字和两条标语进行加工，将“为人民服务”五个字贴上金箔变为红底金字，将两条标语做成[[有机玻璃|有机玻璃]]灯箱。后来，两条标语又改为金箔镶边的白字。[[毛泽东|毛泽东]]在中华人民共和国成立后曾多次书写“为人民服务”五个字，新华门影壁上的这五个字是从毛泽东中华人民共和国成立后书写的许多条幅中挑选出来的写得最好的一幅<ref name=weixiu/>。
[[File:Mao_slogan,_Xinhuamen.jpg|thumb]]

这两条标语和北京一些重点建筑的“红色革命化”布置是同期形成。1966年至1969年初，遵照周恩来及[[国务院机关事务管理局|国务院机关事务管理局]]领导的指示，该局的[[杨亚人|杨亚人]]负责[[人民大会堂|人民大会堂]]、[[北京饭店|北京饭店]]、中南海几个大门、紫光阁、西花厅和[[北京宋庆龄故居|宋庆龄住地]]等处的“红色革命化”布置工作。中南海几个大门里，唯独新华门不是由杨亚人“红化”布置，而是由[[中央警卫团|中央警卫团]]政治处文化干事姚发昌负责。姚发昌曾拿着两幅美术字体的标语征求杨亚人的意见，一条是“伟大的中国共产党万岁”，另一条是“战无不胜的毛泽东思想万岁”。杨亚人修改部分美术字体的线条后，向姚发昌介绍了人民大会堂正门外两侧墙上灯光标语的制作工艺及效果，姚发昌作了记录，并称回去会建议领导照此制作。后来这两条标语就被制成了灯光式标语<ref name=yangsi>{{Cite web |url =http://news.163.com/11/0514/14/7418NIO300014AED.html|title =“破四旧”运动中人民大会堂文物何以保存|author =杨亚人  |date =  |publisher =百年潮2011年04期  |language =  |accessdate =  |quote =  }}</ref>。
[[File:CPC_slogan,_Xinhuamen.JPG|thumb]]

据[[米開朗基羅·安東尼奧尼|安东尼奥尼]]拍摄的[[纪录片|纪录片]]《[[中国_(纪录片)|中国]]》显示，在1972年的某个时段，新华门东侧八字墙标语曾为“伟大的领袖毛主席万岁！”。

1981年6月，[[中共十一届六中全会|中共十一届六中全会]]审议通过了《[[关于建国以来党的若干历史问题的决议|中共中央关于建国以来党的若干历史问题的决议]]》，彻底否定“文化大革命”。全中国各地开始清理标语、语录、口号等“文化大革命”痕迹，同时对中南海里留下的标语和语录牌也开展了清理。[[中央警卫局|中央警卫局]]曾专门研究中南海里哪些标语和语录要取消，哪些要保留，其中包括新华门前的两条标语和影壁上的字，此后上报中共中央。最终新华门前的两条标语和影壁上的字保留至今<ref name=weixiu/>。

1990年颁布实施的《[[中华人民共和国国旗法|中华人民共和国国旗法]]》第五条规定，新华门“应当每日升挂国旗”<ref>{{Cite web |url =http://www.npc.gov.cn/wxzl/wxzl/2008-12/15/content_1462082.htm  |title =中华人民共和国国旗法  |author =  |date = 2008-12-15 |publisher = 中国人大网 |language =  |accessdate =  |quote =  }}</ref>。1991年颁布实施的《[[中华人民共和国国徽法|中华人民共和国国徽法]]》第四条规定，县级以上各级人民政府应当悬挂国徽<ref>{{Cite web |url =http://www.npc.gov.cn/wxzl/wxzl/2008-12/15/content_1462081.htm  |title = 中华人民共和国国徽法 |author =  |date = 2008-12-15 |publisher =中国人大网  |language =  |accessdate =  |quote =  }}</ref>。

1999年，为迎接[[中华人民共和国国庆|中华人民共和国国庆]]五十周年，对新华门地区实施了照明改造，这是[[长安街|长安街]]整体改造的项目之一<ref>关利，追溯历史 展现辉煌——北京新华门地区照明置景设计思路，混凝土世界1999(1):42-43</ref>。

2016年8月，新华门前搭起彩色围挡，新华门修缮工作全面展开<ref name=weixiu/>。8月14日修缮完成，以崭新面貌亮相<ref>{{Cite web |url = http://m.takefoto.cn/viewnews-882429.html |title = 新华门修缮一新 蓝天下熠熠生辉 |author =  |date =2016-08-14  |publisher = 北晚新视觉 |language =  |accessdate =  |quote =  }}</ref>。

==建筑==
[[File:Xinhuamen_Gate_of_Zhongnanhai_across_Changan_Street.JPG|thumb]]
新华门为两层砖木结构楼阁。面阔七间，下层中央三间为门洞。卷棚[[歇山顶|歇山顶]]，黄[[琉璃瓦|琉璃瓦]]绿剪边。在二层明楼正上方悬挂着[[中华人民共和国国徽|中华人民共和国国徽]]。楼前一层檐下悬挂着石青地金字楷书“新华门”三字横匾。新华门内的影壁上有毛泽东题写的“为人民服务”红底金字。新华门两侧的八字墙上，有红底白字的“伟大的中国共产党万岁！”“战无不胜的毛泽东思想万岁！”两条标语。门前两侧有一对石狮，门正前方立有一根旗杆，升挂[[中华人民共和国国旗|中华人民共和国国旗]]<ref name=chaifch>柴法臣，中南海新华门的故事，办公室业务2011(1):62-63</ref>。
