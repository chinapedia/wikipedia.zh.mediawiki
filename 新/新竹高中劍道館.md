'''新竹高中劍道館'''位於臺灣[[新竹市|新竹市]][[東區_(新竹市)|東區]]的[[新竹高中|新竹高中]]校內，於2001年5月31日公告為市定古蹟{{r|文}}。該建築是在日治時期的新竹中學校首任校長[[大木俊九郎|大木俊九郎]]任內建成，他本身有相當的[[劍道|劍道]]造詣，親自教導學生劍道{{r|走}}{{r|竹}}。

該建築除被稱為劍道館之外，又有被稱做「武德館」、「武德殿」等，但據竹中校友鄭華生（1943年畢業生）表示正確名稱應為「武道場」{{NoteTag|1=鄭華生表示類似功能的建築物依等級有不同稱呼，在東京的[[武道館|武道館]]之下有縣市級的[[武德殿|武德殿]]，學校級的是武道場{{r|潘}}。}}{{r|潘}}。

== 沿革 ==
新竹高中劍道館原又稱武道場，於昭和元年（1926年）6月20日竣工，因為當時新竹中學校的劍道水平很高，故使得該棟建築被稱為劍道館{{r|竹}}{{r|潘}}{{r|張}}。二次大戰後，該建築曾改為圖書館，後來一度因為校方有意要將之拆除興建音樂館，使得新竹高中（新竹中學校）校友多方奔走力求保存，最後此棟建築被指定為古蹟{{r|走}}{{r|竹}}。劍道館保存下來之後，現在為學生社團所使用{{r|走}}{{r|竹}}。

== 建築 ==
新竹高中劍道館為磚造木構架建築，屋頂原鋪屋瓦，現改成鋼浪板{{r|走}}{{r|竹}}。外牆採用[[扶壁|扶壁]]的設計，將屋頂重量透過突出的磚柱分散到地面，也因此牆上可以開較大的窗戶，以利通風和採光{{r|走}}{{r|竹}}。而劍道館四面牆上的拱窗也是其特色，外觀大致呈現東西、南北對稱{{r|竹}}。

== 其他 ==
對過去在新竹中學校就讀的臺籍學生來說，這裡是個可以「合法」打日本人的地方，可以發洩對日籍學生頤指氣使的態度與不公平待遇的不滿{{r|潘}}{{r|張}}。當時的校友鄭肇芳曾表示他常與[[林為恭|林為恭]]（前[[苗栗縣|苗栗縣]]縣長）向日本同學挑戰，鄭翼宗也常在這裡用竹劍回報平日被欺負的羞辱{{r|張}}。
