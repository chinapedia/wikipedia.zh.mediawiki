'''新万金海堤'''是位于[[韩国|韩国]][[全羅北道|全羅北道]]西海岸的[[人工|人工]][[海堤|海堤]]，平均高度34米，最高处54米，全长33.9公里，是世界最长的海堤<ref name=xh20100427>{{cite web |url = http://news.xinhuanet.com/photo/2010-04/27/c_1259847.htm |title=世界最长海堤在韩竣工 |publisher =新华网 |date=2010-04-27}}</ref>，2010年6月被收录在《[[吉尼斯世界纪录大全|吉尼斯世界纪录大全]]》<ref>{{cite web |url = http://www.chinanews.com/gj/gj-yt/news/2010/06-23/2358738.shtml |title=韩国海堤34公里全球最长 获吉尼斯世界纪录 |publisher = 中国新闻网|date=2010-06-23}}</ref>。

新万金海堤-{zh-tw:透過;zh-cn:通过}-围海将造地283平方公里（相当于首尔2/3面积），是目前世界上最大的围海造地工程，为每个韩国人增加1平方米的土地面积<ref name=sohu>{{cite web |url =http://business.sohu.com/20120628/n346756212.shtml  |title= 韩国围海造283平方公里土地 相当于首尔2/3面积 |publisher = 搜狐网 |date=2012-06-28}}</ref>，被称为“海上万里长城”<ref name=xh20100529>{{cite web |url=http://news.xinhuanet.com/world/2010-05/29/c_12155605.htm |title=韩国打造“海上的万里长城” |publisher = 新华网 |date=2010-05-29}}</ref>。2012年，韩国制定《新万金特别法》将新万金指定为自由经济区<ref name=gov>{{cite web |url = http://www.mofcom.gov.cn/article/difang/shandong/201407/20140700674641.shtml |title=韩中经合迈首步 韩将为新万金“给力” |publisher =中华人民共和国商务部 |date=2014-07-25}}</ref>。

“新万金”一词是全罗北道[[群山市|群山市]]一侧的“万顷平原”和[[扶安郡|扶安郡]]一侧的“金堤平原”的开头二字，加上了“新”字<ref name=xhrb>{{cite web|url=http://xh.xhby.net/mp2/html/2011-08/10/content_408615.htm|title=沿海开发，新万金对我有何借鉴|publisher=新华日报|date=2011-08-10|deadurl=yes|archiveurl=https://web.archive.org/web/20141204090902/http://xh.xhby.net/mp2/html/2011-08/10/content_408615.htm|archivedate=2014年12月4日|df=}}</ref>。该地区过去是有名的[[黄金|黄金]]产地，周边地区带“金”字提名的很多如金堤、金沟等。该地区现在也出黄金。新万金后面是韩国最好的粮仓是富庶之地。因此“新万金”有“新的万金土地”的含义<ref>{{cite web |url =http://news.dichan.sina.com.cn/2012/03/16/456573.html  |title=项目融资：韩国新万金成中国资本热土 |publisher = 新浪地产 |date=2012-03-16}}</ref>。

==历史沿革==
新万金工程开始于[[卢泰愚|卢泰愚]]政府时期的1989年，最初目的是打造农业生产基地{{r|gov}}。1991年，“第一工程”的防波堤动工。由于工程巨大和重重的环保阻力，工程历时16年于2006年合围，期间由于[[环境保护|环境保护]]的问题暂停了3年<ref>{{cite web|url=http://xh.xhby.net/mp2/html/2009-10/29/content_70409.htm|title=“新万金”：一个饱含期待的壮举|publisher=新华日报|date=2009-10-29|deadurl=yes|archiveurl=https://web.archive.org/web/20141204091117/http://xh.xhby.net/mp2/html/2009-10/29/content_70409.htm|archivedate=2014年12月4日|df=}}</ref>。随着韩国粮食问题的缓解，经过多次争论后，韩国政府在2009年将新万金的开发方向改为打造以经济为中心的综合城市{{r|gov}}。2010年，新万金海堤工程竣工。海堤比[[荷兰|荷兰]]须得海围海造地工程的海堤长1.4公里，成为世界最长的海堤<ref name=xh20100529/>。2012年，韩国制定了《新万金特别法》，将把新万金建设成为[[自由经济区|自由经济区]]，引起国内外投资者的高度注意{{r|gov}}。2014年3月，新万金韩中经济合作园区开发项目先于其它国家的合作园区正式启动<ref>{{cite web|url=http://www.ckbiz.net/news/show-4994.html|title=新万金韩中经济合作园区开发项目正式启动|publisher=韩商网|date=2014-03-06}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

==环保问题==
新万金工程在施工期间遇到重重[[环保|环保]]阻力。[[环保团体|环保团体]]曾和韩国政府三次对簿公堂。法院2003年裁定新万金工程停工。韩国政府为此放弃原来以[[农业|农业]]开发为主的计划，而改为建设注重生态环保的综合城市。修改后的方案最终在2005年12月被首尔高等法院改判为工程继续施工{{r|xhrb}}。

修改后的新万金计划可以明显感受到对环保的服从。原本回填的锯齿状海岸在吸纳了环保组织的意见后不再回填，完全保留原生态{{r|xhrb}}。填海取料被炸平的山坡重新被覆盖植被。工业园区也设置了许多生态门槛，不许危及环境的项目进入。[[三星集团|三星集团]]还与韩国政府签订了“绿色能源综合工业园区”的备忘录，将在新万金建设风力、太阳能发电和燃料电池等项目{{r|xhrb}}。

据韩国环境部表示新万金将采用环保开发模式，以最大限度地避免对环境的破坏。生态环境用地是规划面积最大的，其中的84.4%用于建立[[湿地|湿地]]、生态林等生态空间和高端环境研究院。在环境用地北部，还将建设一个20平方公里的东方第一大[[湿地公园|湿地公园]]；在环境用地南部，则将建设占地10平方公里的野生动植物自然生态公园和环境生态体验场<ref name=xh20100529/>。

=== 生態破壞 ===
在工程完結前，此地區一直是[[候鳥|候鳥]]遷徙的重要棲地，但自從工程進行而至完成階段，已令為數極多的侯鳥數目大幅減少。據統計每年有400,000只[[涉禽|涉禽]]經此地橫跨[[亞洲|亞洲]]及[[俄羅斯|俄羅斯]]地區，<ref>http://www.birdskorea.org/Habitats/Wetlands/Saemangeum/BK-HA-Saemangeum-Mainpage.shtml accessed 2008-01-28</ref>當中包括於近年數目大幅減少的[[小青脚鹬|小青脚鹬]]（[[瀕危|瀕危]]）及[[勺嘴鹬|勺嘴鹬]]（[[極危|極危]]）。而其中[[勺嘴鹬|勺嘴鹬]]的數目近由1970年代的約2400對大幅下降至2005年的少於100對。<ref>{{Cite news| url=http://news.bbc.co.uk/mpapps/pagetools/print/news.bbc.co.uk/2/hi/asia-pacific/6649233.stm | work=BBC News | title=Birds 'starve' at S Korea wetland | date=2007-05-12 | accessdate=2010-05-23}}</ref>有保育團體指南韓政府在進行如此大型的[[填海|填海]]活動前沒有做好環境評估的工作，直接或間接地做成生態災難，並自發進行有關的監察研究。<ref>Moores, N.; Battley, P.; Rogers, D.; Park M-N; Sung H-C; Van de Kam, J.; & Gosbell, K. (2006). Birds Korea – AWSG Saemangeum Shorebird Monitoring Program Report, 2006. Birds Korea publication: Busan.</ref>
