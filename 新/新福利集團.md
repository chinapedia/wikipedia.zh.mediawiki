'''新福利集團有限公司'''（簡稱'''新福利集團'''，'''{{lang|en|Transportes Urbanos de Macau}}'''）於2004年1月在[[澳門特別行政區|澳門特別行政區]]註冊成立，總部亦設在[[澳門|澳門]]。現任集團董事局主席為廖僖芸，集團董事總經理陳曉陽。

== 簡介 ==
新福利集團的前身是創辦於1952年的「澳門福利公共汽車股份有限公司」，經過50多年的發展，在[[澳門|澳門]]、[[廣州|廣州]]、[[新會|新會]]、[[佛山|佛山]]、[[蕪湖|蕪湖]]等地經營[[公共汽車|公共汽車]]服務。目前集團的業務走向多元化，集[[公共交通|公共交通]]、[[運輸|運輸]]、[[廣告|廣告]]、[[科技|科技]]、[[房地產|房地產]]、[[貿易|貿易]]等投資業務於一身。其中，經營公共汽車營運公司有五間，包括[[澳門新福利公共汽車|澳門新福利公共汽車有限公司]]、[[廣州市新福利巴士|廣州市新福利巴士服務有限公司]]等企業。集團的經營理念為「以人為本、以客為先」，並以「新福利服務、創新求進步」為企業目標。集團各公司為不斷適應現代公交發展的要求，統一實施[[ISO_9001|ISO 9001]]：2000國際優質管理體系，推動各地的公交服務進入國際化標準，使公共交通事業的服務質素更加專業化。

== 屬下公司及創立背景 ==
新福利集團屬下公司包括：
# [[澳門新福利公共汽車有限公司|澳門新福利公共汽車有限公司]]：1988年7月18日：承接有40年歷史的[[澳門福利公共汽車股份有限公司|澳門福利公共汽車股份有限公司]]重組而成。
# [[蕪湖市新福利公交|蕪湖市新福利公交有限公司]]：成立於2004年11月，起初與香港珍寶公司合作。2005年2月因珍寶公司退出，改由新福利集團正式全面接管。2007年再與運泰集團合資成立蕪湖市新福利公交有限公司。
# 澳門億能旅遊會展項目策劃及服務有限公司：為集旅遊、會展、項目策劃、旅遊包車的公司。

=== 已脫離集團的公司 ===
# [[廣州市新福利巴士|廣州市新福利巴士服務有限公司]]：成立於1993年12月，與[[廣州市電車公司|廣州市電車公司]]合資，成為首家進入[[廣州市|廣州市]]的中外合資巴士公司。由於[[廣州市交通委員會|廣州市交通委員會]]出台公交資源整合措施，由2009年10月1日起易手，由[[廣州市第二公共汽車|廣州市第二公共汽車公司]]收購。
# [[江門市新會新福利巴士|江門市新會新福利巴士服務有限公司]]：成立於1999年7月，是[[廣東省|廣東省]]首家獨資外資公交企業。配合政府當地公交政策，於2010年6月11日起，被广东省江门市汽车运输集团收购。
# [[佛山市新福利巴士|佛山市新福利巴士服務有限公司]]：成立於2004年7月，由於多年虧損，由2009年7月1日起易主，由佛山汽運集團收購。

== 重大事件 ==
*1995年10月23日，集團其下的澳門新福利試行巴士自動收費系統，於[[新福利3路線|3]]路線試行聰明卡。成了[[亞洲|亞洲]]地區首個使用這種新科技的城市。
*1997年，集團獲澳門政府批給一幅[[筷子基|筷子基]]林茂塘的土地用作巴士總站及修理廠，及後集團以1.6億澳門元的溢價金把該地段轉到同為廖氏兄弟[[廖澤雲|廖澤雲]]、廖僖芸旗下的新天康投資股份有限公司名下，該公司其後重點投資於房地產業務，用以興建豪宅海擎天。此舉被部份人士認為是不合理交易，原因是集團持有人本為時任[[澳門行政長官|澳門行政長官]][[何厚鏵|何厚鏵]]先父[[何賢|何賢]]，及後售予廖氏兄弟繼續經營，而廖澤雲亦為澳門行政委員，與何厚鏵關係密切。因而令此次事件引起一些社會議論。<ref>[http://newmacau.org/newmacauweb/Issue_34_index.htm 《新澳門》第34期──「五一餘波」-《發財不立品》- 巴士公司虧本嗎？] {{webarchive|url=https://web.archive.org/web/20071204172320/http://newmacau.org/newmacauweb/Issue_34_index.htm |date=2007-12-04 }}</ref><ref>{{cite news|title=澳門賣地高底價9倍兩地值14億業界：不排除有人托市|publisher=蘋果日報 |date=2008-01-10}}</ref><ref>{{cite news|author=汪長南 |title=為何此時賣地？|publisher=訊報 |date=2008-01-18}}</ref> 而[[澳門特別行政區|澳門特別行政區]]政府亦發出聲明，解釋當中因由，以正視聽。惟部份立法會議員仍然抗爭，要求政府規定該公司將盈力轉向巴士業務。<ref>[http://www.safp.gov.mo/legismac-orgtxt/2005/S2/2005_13/DESSTOP32XX05C.htm 澳門運輸工務司司長批示第32/2005號] {{webarchive|url=https://web.archive.org/web/20080619130940/http://www.safp.gov.mo/legismac-orgtxt/2005/S2/2005_13/DESSTOP32XX05C.htm |date=2008-06-19 }}</ref>
* 1999年，澳門新福利公共汽車有限公司獲[[澳門消費者委員會|澳門消費者委員會]]評為最受市民歡迎的公用事業單位。
* 1999年3月20日，澳門新福利公共汽車有限公司正式推出命名為「[[澳門通|澳門通]]」的自動收費系統，採用由集團屬下公司「康澤科技」經多年不斷研究而自行生產的「無接觸式IC卡自動收費系統」。
* 1999年12月20日起，因原董事會主席何厚鏵出任行政長官一職而不能擔任董事會主席，改由廖澤雲擔任至今。
* 2012年4月，被傳媒揭露透過其妻子向當時競逐香港特首的梁振英捐款<ref>[http://hk.apple.nextmedia.com/template/apple/art_main.php?iss_id=20120429&sec_id=4104&subsec_id=12731&art_id=16291643 英治八卦：澳門發展商妻子100萬撐 CY]</ref>。
