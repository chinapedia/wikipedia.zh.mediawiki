'''新麗華酒店''' （'''Hotel Sintra'''）位於[[澳門|澳門]][[南灣_(澳門)|南灣區]][[約翰四世大馬路|約翰四世大馬路]]58-62號，於1975年落成，由澳門旅遊娛樂有限公司管理；及至1990年，[[信德集團|信德集團]]屬下的[[富利達國際酒店|富利達國際酒店]]集團管理的三星級酒店。酒店間數目約240間。

== 摩卡(Mocha) ==
酒店內設有由[[新濠博亞博彩(澳門)股份有限公司|新濠博亞博彩(澳門)股份有限公司]]營運的[[摩卡角子機娛樂場|摩卡角子機娛樂場]]，面積達7,500平方呎、設有約135台[[角子機|角子機]]。

== 交通配套 ==
新麗華酒店設有來往酒店與[[港澳碼頭_(澳門)|港澳碼頭]]的接駁專巴服務。

== 景點 ==
酒店位於澳門中區地帶，毗鄰[[永利澳門|永利澳門]]、[[新葡京|新葡京]]、[[葡京|葡京]]、[[新馬路|新馬路]]等旅遊景點。
