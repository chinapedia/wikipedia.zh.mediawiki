'''新逗子站'''（{{jpn|j=新逗子駅|hg=しんずしえき|rm=Shinzushi eki|f=y}}）是一個位於[[神奈川縣|神奈川縣]][[逗子市|逗子市]]逗子五丁目，屬於[[京濱急行電鐵|京濱急行電鐵]][[逗子線|逗子線]]的[[鐵路車站|鐵路車站]]。是逗子線的[[終點站|終點站]]。[[車站編號|車站編號]]是'''KK53'''。

==年表==
* 1930年（[[昭和|昭和]]5年）4月1日 - 湘南電氣鐵道「湘南逗子站」啟用。當時位於現址往金澤八景方向200公尺處（{{coord|35|17|45|N|139|34|52|E|region:JP_type:railwaystation|name=新逗子站北口（舊京濱逗子站）}}）。
* 1931年（昭和6年）4月1日 - 路線延長400公尺並開設「湘南逗子站葉山口」（{{coord|35|17|34|N|139|34|48.5|E|region:JP_type:railwaystation|name=舊逗子海岸站}}）。原來的車站改稱「湘南逗子站沼間口」。
* 1942年（昭和17年）9月1日 - 車站改稱為'''湘南逗子站'''。
** 5月1日 - 公司合併，成為[[東京急行電鐵|東京急行電鐵]]（大東急）車站。
** 9月1日 - 葉山口關閉，路線縮短至原來的沼間口，更名為「湘南逗子站」。
* 1948年（昭和23年）
** 6月1日 - 成為京急車站。
** 7月3日 - 路線延長，葉山口以「逗子海岸站」復活。
* 1963年（昭和38年）11月1日 - 車站改稱為'''京濱逗子站'''。
* 1985年（昭和60年）3月2日 - 為停靠加強輸送力的8輛編組列車，兩站合併在中間地點新設'''新逗子站'''。北口使用舊京濱逗子站房，南口為新設。
* 1991年（[[平成|平成]]3年）3月15日 - 北口車站大樓「Neat新逗子」開幕，舊京濱逗子站站房拆除。
* 1999年（平成11年）7月31日 - 京急線改點，廢止京急蒲田站 - 本站間的急行。
* 2008年（平成20年）11月21日 - 導入[[進站音樂|進站音樂]]。
* 2010年（平成22年）5月16日 - 改點，成為新設的機場急行停靠站。
* 2020年3月 - 更改站名為'''逗子、葉山車站'''（{{lang|ja|逗子・葉山駅}}），並加註[[副站名|副站名]]「舊站名 新逗子」（{{lang|ja|旧駅名　新逗子}}）<ref>{{cite web|url=https://www.keikyu.co.jp/company/news/2018/20190125HP_18229TS.html|title=沿線地域の活性化を目的に2020年3月 4駅の駅名を変更します|publisher=京浜急行電鉄株式会社|date=2019-01-25|language=ja}}</ref>。

==車站結構==
本站為對應8輛編組的[[單式月台|單式月台]]1面1線[[地面車站|地面車站]]。逗子線全線為[[複線|複線]]，但站內為[[單線|單線]]。

本站是由兩站整合而成，剪票口設在南北二個舊站附近。北口車站大樓「Neat新逗子」2樓剪票口與月台之間有約100公尺長的天橋。南口可直通月台。列車資訊顯示器上顯示為「1號線」月台。另外，月台在4輛、6輛、8輛編組最尾端有車掌用的發車開關。開關與平交道、信號機、電鈴裝置連動。

=== 巴士路線 ===
由[[京濱急行巴士|京濱急行巴士]]與其子公司湘南京急巴士運行，可至逗子市內各地與橫須賀線[[鎌倉站|鎌倉站]]、[[田浦站_(日本)|田浦站]]方向、以及[[葉山町|葉山町]]和[[橫須賀市|橫須賀市]]西部。

北口與南口皆有巴士站。南口的南向巴士站在舊逗子海岸站用地上，北向巴士站在縣道24號上。所有經由站的巴士皆從JR逗子站起訖，本站無終點站機能。

==== 北口（逗子橋側平交道對向） ====
: 田浦線 - 葉櫻、Itohpia（イトーピア）、東逗子、Green Hill（グリーンヒル）、田浦站方向／往逗子站

==== 北口（逗子市役所正對面） ====
: Highland（ハイランド）線 - 久木五丁目、Highland循環
: 龜岡團地循環線 - 久木西小路、綠丘入口、龜岡團地循環
: 名越線 - 久木西小路、綠丘入口、名越、鎌倉站方向
: 小坪線 - 久木西小路、小坪、{{link-ja|材木座海岸|材木座海岸|材木座}}、鎌倉站方向

==== 南口 ====
: 長井線 - 葉山大道、大楠蘆名口、湘南渚之丘（湘南なぎさの丘）、佐島Marina（佐島マリーナ）入口、長井方向
: 海岸線 - 元町、森戶海岸、葉山、{{link-ja|葉山町立南鄉中學校|葉山町立南郷中学校|南鄉中學校}}方向
: 湘南國際村線 - 長柄橋、葉山大道、{{link-ja|湘南國際村|湘南国際村}}方向（部分經由三浦半島中央道路）
: 逗子線 - 長柄橋、葉山大道、池上十字路、[[衣笠站|衣笠站]]方向
: 往逗子站（上述各線北向）
