'''新丁粄'''是[[客家人|客家人]]多元米食文化之一，在家中[[添丁|添丁]]時製作，用來跟[[祖先|祖先]]及上蒼報告並乞求小孩平安<ref name="長平">{{cite web |url=http://www.peoplenews.tw/news/79dbd668-135f-4ba5-9ea4-63a729f5df28 |date=2017-03-13 |title=傳統粄食新丁粄 有如客家人一生的縮影 |author=長平 |publisher=民報 |language=zh-tw |deadurl=no |accessdate=2018-08-03 |archiveurl= |archivedate=}}</ref>。

== 新丁粄的由來 ==
早期台灣[[農業|農業]]社會家中[[添丁|添丁]]是[[家族|家族]]大事，要跟[[祖先|祖先]]及上蒼報告並乞求小孩平安，因此會在[[元宵節_(華人)|元宵節]]（農曆一月十五）或農曆十月中旬製作像紅龜粿的新丁粄一個個堆疊起來敬拜天地，感謝老天並乞求小孩能健康長大<ref name="2018臺中東勢新丁粄節">{{cite web |url=https://festival.hakka.gov.tw/Festival-Content.aspx?a=20&l=1 |date= |title=2018臺中東勢新丁粄節 |author= |publisher=客家委員會 |language=zh-tw |deadurl=no |accessdate=2018-08-03 |archiveurl= |archivedate=}}</ref><ref name="林建鋒2017-02-11">{{cite web |url=https://tw.appledaily.com/new/realtime/20170211/1053742/ |date=2017-02-11 |title=「新丁粄」顏色不對　惹怒東勢客家 |author=林建鋒 |publisher=蘋果日報 |language=zh-tw |deadurl=no |accessdate=2018-08-03 |archiveurl= |archivedate=}}</ref>。

在塔頂的新丁粄染成紅色且不壓花紋，敬神後新丁粄會分送給親朋好友分享添丁的喜悅<ref name="新北市客家觀光美食館" />。

新丁粄通常和[[紅龜粿|紅龜粿]]一樣，花紋是以象徵[[壽命|長壽]]幸福的龜甲，不過新丁粄不包餡<ref name="新北市客家觀光美食館">{{cite web |url=https://www.hakka-cuisine.ntpc.gov.tw/files/15-1006-3459,c394-1.php?Lang=zh-tw |date= |title=新丁粄（又稱紅粄、紅龜粿） |author= |publisher=新北市客家觀光美食館 |language=zh-tw |deadurl=no |accessdate=2018-08-03 |archiveurl= |archivedate=}}</ref>。

== 新丁粄的作法 == 
1. 將[[糯米|糯米]]、蓬萊米磨成[[米漿|米漿]]，脫水七分乾，成米糰。

2. 水滾後，將生米糰撕成小塊，放入蒸籠，蒸約十五至二十分鐘煮熟。

3. 取一份約一斤二兩的熟米糰，抹上沙拉油，放入模型後，擦上食用大紅。

4. 食用時可將新丁粄切成長條狀，放入少許油煎成金黃色，灑上鹽水再煎乾即可食用。或是不加鹽水，在煎成金黃色後，沾蒜泥、[[辣椒|辣椒]]、 [[醬油|醬油]]一起食用。

== 新丁粄賽 ==
[[東勢區|東勢]][[客家|客家]]新丁粄賽是全台罕見的[[台灣元宵節|元宵節]]民俗，固定於每年的元宵節夜晚時舉辦，而擠往鎮上各里看新丁粄者人山人海，比[[春節|春節]]還要熱鬧。

粄為客家話，闽南話則稱為粿，東勢新丁粄起源於早期的農業社會，因重男輕女觀念濃厚，於是產生重視「出丁」風俗，此外主要是為叩謝伯公（[[土地公|土地公]]）添丁賜福因而得名。

每年元宵節時的夜晚，以各廟宇為中心成立新丁粄會，若是本年家中有添新丁、新孫、新曾孫或家有新婚等喜事者，必須於次年的元宵節製作紅龜粄到廟中祭祀神祇，會員人數有多少，紅龜粄就要作幾塊，並讓所有的會員分享，而在眾多的紅龜粄中需選一個做得最大最重者，由公會公開表揚並發給獎金，參加者為奪頭彩，自然越作越大，使得競爭十分激烈，往往一塊紅龜粄重二、三十斤，數十塊擺在一起，相當地壯觀，成為東勢地區客家人過元宵節的一大特色<ref name="連宜方2018-03-03">{{cite web |url=https://bobee.nownews.com/20180303-19283 |date=2018-03-03 |title=台中東勢新丁粄節　客家限定版求子紅龜粿 |author=連宜方 |publisher=NOWnews |language=zh-tw |deadurl=no |accessdate=2018-08-03 |archiveurl= |archivedate=}}</ref>。
