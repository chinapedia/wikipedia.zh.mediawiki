《'''新宋'''》是一部由[[中華人民共和國|中國]]作家阿越撰写的[[架空歷史小說|架空歷史小說]]，讲述了现代大学生石越穿越到[[北宋|北宋]]，利用现代的知识积累，对北宋王朝的各个方面进行改革的故事。<ref>{{Cite web |url= http://news.xinhuanet.com/book/2009-06/26/content_11604160_1.htm |title= "网络文学十年盘点"十佳优秀作品简介 |author=  |date= 2009年6月26日 |publisher= 新华网 |language= zh-Hans |accessdate= 2013年8月17日 |quote=  }}</ref>整部小说分为“十字”、“权柄”与“燕云”三大部分。

== 劇情 ==
===十字===
小說第一卷“十字”的情節發生于[[宋神宗|宋神宗]]時期，[[熙寧|熙寧]]二年西元1069年。<ref>{{Cite web |url=http://www.gxxwcbj.gov.cn/index.php?m=content&c=index&a=show&catid=167&id=6152 |title=两岸同时推出历史幻想小说《新宋》 |date=2005年12月8日 }}</ref>

21世紀的歷史系大學生石越，<ref>{{Cite web |url=http://game.people.com.cn/GB/48642/48649/3935279.html |title= 幻想小说《新宋》“颠覆历史” |date=2005年12月12日  }}</ref>無故穿越到[[北宋|北宋]]年間的[[開封|開封]]，偶遇商人家庭出身的唐棣，通過千年之後的知識與技術，使唐氏的生意突飛猛進，也通過重新解釋《[[論語|論語]]》與堯舜禹時期的政治贏得盛名。而後逐步進入仕途，與[[王安石|王安石]]、[[司馬光|司馬光]]、[[呂惠卿|呂惠卿]]等北宋政治家周旋捭闔，與[[沈括|沈括]]、[[蘇軾|蘇軾]]、[[程顥|程顥]]、[[程頤|程頤]]等北宋名士交遊論道，通過引進了報紙、大學、流水線生產等新理念加入了北宋[[熙寧變法|神宗改革]]的大幕。

===权柄===
“新宋”第二卷“权柄”主要讲述了石越成为[[陕西路|陕西路]][[安抚使|安抚使]]后，讨伐[[西夏|西夏]]的故事。

==评论==
有评论认为作者用他的对宋朝历史的思考，观照着自己的“新宋史”，以细腻的笔触推动着这段历史的发展；对宋代人物、风貌、习俗的细节，均考据有加。<ref>{{Cite web |url= http://www.gmw.cn/content/2008-03/31/content_755421.htm |title= 造访宋朝　重塑历史 |author= 横刀 |date= 2008-03-31 |publisher= 光明网 |language= zh-Hans |accessdate= 2013年8月17日 |quote=  }}</ref>也有评论认为作者既没有让《新宋》成为影射史学，也没有让它成为炫才式的僵死的历史学知识，反而既对历史保持了必要的忠诚，同时他又通过石越这个人物，顽强而又鲜活地呈现了写作者个人的历史观。<ref>{{Cite web |url= http://book.sina.com.cn/cul/techcul/techcul/tech/2012-07-02/1535300569.shtml |title= 陈福民评《新宋》：好看又具有诸种启示性 |author= 陈福民 |date= 2012年7月2日 |publisher= 新浪读书 |language= zh-Hans |accessdate= 2013年8月17日 |quote=  }}</ref>
