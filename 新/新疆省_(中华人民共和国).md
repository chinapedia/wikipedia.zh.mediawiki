'''新疆省'''為[[中華人民共和國|中華人民共和國]]建国之初的省份之一，前後存在6年。

[[第二次國共內戰|第二次國共內戰]]末期，[[中國人民解放軍|中國人民解放軍]]盡佔優勢，國軍節節敗退。在[[彭德懷|彭德懷]]指挥的[[第一野戰軍|第一野戰軍]]1949年8月26日佔領[[蘭州|蘭州]]繼而佔領[[甘肃省_(中华民国)|甘肅]]全境之後，新疆的軍政長官[[陶峙岳|陶峙岳]]、[[包爾漢|包爾漢]]等宣佈脫離[[廣州|廣州]]的臨時[[中华民国政府|中华民国政府]][[起義|起義]]。期間已經投向共產黨一邊的[[張治中|張治中]]在中斡旋，而[[中共|中共]]方面的談判代表是[[鄧力群|鄧力群]]。同年前蘇聯讓起義軍停止進入[[烏魯木齊|烏魯木齊]]，並讓起義軍投降於中華人民共和國政府。9月25日第一野戰軍第一兵團在[[王震|王震]]將軍率領下攻佔[[烏魯木齊|烏魯木齊]]，成為[[中華人民共和國|中華人民共和國]]的一省。12月17日成立新疆省人民政府，包爾漢為首任主席。

1955年9月13日，[[第一届全國人民代表大會常務委員會|第一届全國人民代表大會常務委員會]]第21次會議決定，撤銷新疆省，並以原新疆省的行政區域為新疆維吾爾自治區的行政區域。10月1日，[[新疆維吾爾自治區|新疆維吾爾自治區]]成立，隨之成立新疆維吾爾自治區[[人民委員會|人民委員會]]，新疆為中國的五大省級少數民族自治區之一，「新疆省」名稱走入歷史。

== 人口 ==
中華人民共和國國家統計局於1954年11月1日公報的「第一次全國人口調查登記」，統計至1953年6月30日時止，新疆省人口是4,873,608人﹐其中維吾爾族3,640,125人﹐佔新疆全省人口的四分之三，漢族和其他民族佔四分之一。<ref>中華人民共和國國家統計局關於全國人口調查登記結果的公報（1954年11月1日）；「1955年人民手冊」（大公報出版）P.251</ref>

== 行政區劃 ==
1954年，新疆省轄1行政區、10專區、5自治州、1省轄市，2市、73縣、6自治縣。

== 政府 ==
1949年，[[新疆省|新疆省]]向[[中国共产党|中国共产党]]投降後，原[[新疆省政府|新疆省政府]]於同年12月17日被新成立的[[新疆省人民政府|新疆省人民政府]]取代<ref>[http://cpc.people.com.cn/GB/64162/64165/70486/70487/4792074.html 1955年10月1日] 新疆維吾爾自治區成立</ref>，[[包爾漢|包爾漢]]任[[新疆省人民政府|新疆省人民政府]]主席（1949－1954）。

1955年10月1日，[[新疆維吾爾自治區|新疆維吾爾自治區]]成立，隨之成立[[新疆維吾爾自治區人民委員會|新疆維吾爾自治區人民委員會]]。
