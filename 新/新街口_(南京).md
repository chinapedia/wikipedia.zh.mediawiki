'''新街口'''位于[[中国|中国]][[南京市|南京市]][[新街口街道_(玄武区)|新街口街道]]的中心区域，是一处著名的商业中心。“新街口”一词可能用来指[[新街口广场|新街口广场]]，也可能指以新街口广场为中心的地区，有新街口商圈、新街口中央商务区等称谓；广义的新街口则指[[新街口街道_(玄武区)|新街口街道]]，辖区总面积为2平方公里。新街口每逢节假日日均客流量峰值七十万人次，七十九家世界五百强分支机构进驻，其商贸集中度超过北京[[王府井|王府井]]、上海[[徐家汇|徐家汇]]并翘首中国，成为中国商贸密集度最高的地区<ref>{{cite web|url=http://finance.people.com.cn/GB/6364749.html|title=南京新街口成为中国商贸密集度最高的地区|publisher=''中国新闻网''|date=2007-10-11}}</ref>，因此有“中华第一商圈”的说法。

== 历史 ==
南京新街口的历史可追溯至1500年前<ref>[http://news.xinmin.cn/rollnews/2009/06/05/2051524.html 1500年前古遗迹惊现南京新街口]</ref>。
[[File:Deji_Plaza.jpg|thumb]]]]
新街口是南京一个古老的地名，但在1929年以前只是一片冷清的普通旧式街区，街道狭窄，房屋低矮，沿街房屋后面还有不少空地及池塘。1929年12月完成的[[首都计划|《首都计划》]]将新街口规划为商业区。而1927年至1937年的“黄金十年”间的[[首都|首都]]建设彻底改变了新街口的面貌，[[中山东路_(南京)|中山东路]]、[[中山南路_(南京)|中正路]]、[[汉中路|汉中路]]和[[中山路_(南京)|中山路]]此4条宽40余米的林荫大道在此交汇，中间形成环形广场。因成为新的全市交通枢纽，新街口迅速形成新兴的商业中心。1930年代，这一地带陆续建成[[国货银行|国货银行]]、[[浙江兴业银行|浙江兴业银行]]、[[交通银行|交通银行]]、[[中央商场|中央商场]]、[[大华大戏院|大华大戏院]]、[[新都大戏院|新都大戏院]]、[[福昌饭店|福昌饭店]]等众多商企。新街口中山南路与中山东路交汇处，于1988年，南京近代建筑普查时，近代建筑尚存13座。至2009年，尚存四座[[南京民国建筑|民国建筑]]，分别为：现处于整修状态的中山东路1号，[[民国时期|民国时期]]南京交通银行总部、原[[汪伪政府|汪伪政府]]的中央储备银行，现在的工商银行老楼；中山东路3号；[[金陵饭店|金陵饭店]]北侧的邮局老大楼，市总工会使用的[[原盐务署大楼|原盐务署大楼]]<ref>{{cite web|url=http://www.cityup.org/news/rebuild/20090317/46423.shtml|title=南京新街口“最老”民国建筑整修后再现经典地标|publisher=都市世界－城市规划与交通网|date=2009-03-17}}</ref>。
[[File:ZhaoshangBuilding_Nanjing.jpg|thumb]]
1980年代，在广场西北角建成当时中国最高建筑[[金陵饭店|金陵饭店]]。

== 新街口广场 ==
[[新街口广场|新街口广场]]处于四条马路的相交处，东为[[中山东路|中山东路]]，通向[[中山门_(南京)|中山门]]，南为[[中山南路|中山南路]]，通向[[中华门|中华门]]，西为[[汉中路|汉中路]]，通向[[汉中门|汉中门]]，北为[[中山路|中山路]]，通向[[鼓楼广场|鼓楼广场]]。1942年11月，[[汪伪政府|汪伪政府]]将1928年由[[筱原金作|筱原金作]]和[[牧田祥哉|牧田祥哉]]所铸孙中山铜像从[[中央陆军军官学校|中央陆军军官学校]]移至新街口广场中心<ref>{{cite web|url=http://www.jllib.cn:8080/njmgjz.cn/doc/81030649|title=南京孙中山塑像|publisher=南京民国建筑网|accessdate=2009-06-22}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

2001年之前新街口广场一直是环岛式建筑，2001年，由于[[南京地铁|地铁一号线]]施工，广场拆除，[[孙中山|孙中山]]铜像（1996年版）迁移<ref group="注">此铜像为中国雕塑家戴广文于1995年重铸，1996年完成。2001年，孙中山铜像迁至南京城城北新门口的仓库中。2002年，再次动迁迁至浦口汤泉的部队营区仓库储藏。</ref>。2009年9月，[[南京地铁|地铁二号线]]新街口地面施工将全部结束，2009年5月，孙中山铜像的回归问题也引起了各方讨论，[[民国时期|民国时期]]铸造、体积较小的孙中山铜像<ref group="注">该铜像为1928年孙中山的日本友人梅屋庄吉变卖家产捐赠，由日本的“筱原金作工场”计划，雕塑家牧田祥哉制作。1985年至今，安放于南京中山陵藏经楼前。</ref> 也被相关部门纳入铜像重回新街口计划的考虑范围之内<ref>{{cite web|url=http://news.njdushi.com/nj/0531/20091851.html|title=孙中山铜像回归新街口还是鼓楼 引市民热议|publisher=南京都市网（来源：江南时报）|date=2009-05-31}}</ref>。2009年8月，《新街口广场复建方案》编制完成，考虑到在建的地铁二号线工程，预计2010年[[春节|春节]]后，以1996年版的孙中山铜像回归新街口。方案拟以孙中山铜像为中心，四周扩展成直径8米的环岛，在环岛外围增加宽幅为一米的铺装道路，整个中心最终为直径10米的环岛带。同时周边景观以绿化为主，并且使景观三季有花，四季常青<ref>{{cite web|url=http://epaper.yangtse.com/images/2009-08/05/1249408338093YZA03805C.pdf|title=孙中山铜像 明年春节后重回新街口|publisher=扬子晚报（电子版）|date=2009-08-05|deadurl=yes|archiveurl=https://web.archive.org/web/20150512215035/http://epaper.yangtse.com/images/2009-08/05/1249408338093YZA03805C.pdf|archivedate=2015-05-12}}</ref>。

== 新街口商圈 ==
据相关规划，大新街口商圈的区域东至[[太平南路|太平南路]]、[[太平北路|太平北路]]，南至[[建邺路|建邺路]]、[[白下路|白下路]]，西至[[莫愁路|莫愁路]]、[[上海路|上海路]]，北至[[长江路|长江路]]、[[华侨路|华侨路]]。商圈核心区域东至[[洪武路|洪武南路]]、[[洪武路|洪武北路]]，南至[[石鼓路|石鼓路]]、[[淮海路|淮海路]]，西至[[王府大街|王府大街]]、[[管家桥|管家桥]]，北至[[长江路|长江路]]、[[华侨路|华侨路]]，面积约1平方千米。日常人流量达60多万人次，双休日约100万人次。新街口商圈内同业商家聚集的功能街区有“洪武路金融街”、“正洪路商业步行街”、太平南路黄金珠宝一条街”、“王府美食街”、“石鼓路酒吧街”、“冶山民俗文化街”、 “朝西古玩字画一条街”、“莫愁路古玩一条街”、“长江路文化一条街”、“仓巷明清一条街”、“淮海路民国文化一条街”等。

== 新街口中央商务区 ==
新街口地区是南京的[[中央商务区|中央商务区]]（CBD），这一地区分布的金融机构、咨询公司、法律事务所、会计事务所和大公司总部、办公室数量居南京之首，其中世界500强企业机构约80家。
