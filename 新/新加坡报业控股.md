'''新加坡报业控股'''（{{lang-en|Singapore Press Holdings}}；[[首字母缩略字|缩写]]：{{lang|en|SPH}}，{{sgx|T39}}）简称'''新加坡报业'''，是[[新加坡|新加坡]]最大報業集團，目前擁有11家報紙、16家雜誌，几乎[[垄断|垄断]][[新加坡|新加坡]]国内所有的[[報紙|报章]]的[[出版|出版]]与[[发行|发行]]，其旗舰报章为新加坡的英文大报[[海峡时报|海峡时报]]。[[邮政编码|邮政编码]]為318994。

== 广播业务 ==
;電台
报业控股透过'''报业控股电台私人有限公司''' (SPH Radio Pte Ltd)（2014年前称报业控股联盟传讯（SPH UnionWorks）並和职总媒体合作社合营）经营五家电台，报业控股也经营ST RazorTV。
 
2017年3月，报业控股獲得{{link-en|信息通信媒體發展管理局|Info-communications Media Development Authority}}兩個額外的FM頻率：89.3㎒和96.3㎒（原[[XFM_96.3|XFM 96.3]]的頻率），96.3㎒的新電台命名為「96.3 好FM」（Hao FM），已於2018年1月8日開播；英語電台「MONEY FM」（89.3㎒），已於2018年1月29日 08:00開播<ref>[https://www.facebook.com/shinmindailynewsxinmingribao/photos/a.358346434223942.79082.357787027613216/1545246232200617/ 新明日報]</ref><ref>[http://www.zaobao.com.sg/znews/singapore/story20171111-810043 報業控股兩中英新電台明年1月開台]</ref>。

;電視
報業控股也從2001年至2004年間經營'''报业控股传讯'''（SPH MediaWorks）電視廣播業務。旗下的優頻道（華語廣播）和i頻道（前名「電視通」（TVWorks），以英語廣播）與[[新傳媒電視|新傳媒電視]]的[[新传媒5频道|5頻道]]（英語廣播）和[[新传媒8频道|8頻道]]（華語廣播）進行了激烈的競爭。

報業傳訊在2004年開始與新傳媒電視相討合併。最後，兩台決定於2005年1月1日（星期六）[[新加坡標準時間|新加坡標準時間]][[凌晨|凌晨]]01時30分合併。合併後，報業控股得到新傳媒電視的20％股權。而報業傳信旗下的優頻道被改名為[[新传媒U频道|新传媒U频道]]，並繼續經營。i頻道則因為新加坡英語電視市場過小的關係，在合併後停止播放。

報業傳訊和新傳媒電視在合併前一共裁減了204名普通員工。合併後一共有20名報業傳訊藝人轉移到新傳媒電視，繼續他們的電視生涯。

== 戶外媒体业务 ==
报业控股透过持有'''媒嘉多媒体'''（SPH MediaBoxOffice）80%股权，成為新加坡最大的戶外广告媒体供应商：包括位於城市心臟地帶的五部大型LED显示屏，與及六百部等離子/液晶体显示器和其他广告热点。

== 通讯业务 ==
公司也持有本地第二大流动电信服务公司[[第一通|第一通]]（M1）的股份。

== 房地产业务 ==
在房地产方面，报业控股收购了百利宫（Paragon）和宝龙坊（Promenade），并将它们联合发展为'''百利宫'''——一个处于[[乌节路|乌节路]]心脏地带的高级购物中心和商业大厦。

而报业控股全资附屬的Times Development Pte Ltd，以经在位於[[湯申路|湯申路]]的时报工业大廈旧址上兴建四座樓高43層的高檔次共管公寓—'''天一閣'''（Sky@eleven）[http://www.sky11.com.sg]；而全部273個單位已於2007年1月29日的發售日沽清。报业控股也在金文泰兴建一座新的购物中心，并于2011年正式营业。

2017年6月22日新加坡報業集團及日本鹿島建設各一半股權的合資公司以11.32億坡元的最高競標價格，相當於每平方呎約1300坡元，標得比達達利的綜合發展項目地段，佔地面積為2萬5440.8平方公尺（約27萬3842平方英尺），預計建造600多個私宅單位和提供31萬平方英尺商業與零售面積。
