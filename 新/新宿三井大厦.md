'''新宿三井大厦'''（{{lang-ja|新宿三井ビルディング}}）是[[日本|日本]][[东京都|东京都]][[新宿区|新宿区]][[西新宿|西新宿]]超高层建筑群中的一栋[[摩天大楼|摩天大楼]]。<ref name="SMB SCP">{{cite web |url=http://skyscraperpage.com/cities/?buildingID=6951|title=Shinjuku Mitsui Building |accessdate= |publisher=Skyscraperpage.com}}</ref><ref name="SMB EMP">{{cite web |url=http://www.emporis.com/en/wm/bu/?id=shinjukumitsuibuilding-tokyo-japan |title=Shinjuku Mitsui Building |accessdate= |publisher=Emporis.com}}</ref>

== 概要 ==
新宿三井大厦是东京都新宿区西新宿第四高的摩天大楼，完成时曾是日本第一高楼，其后被1978年竣工，位于东京都[[丰岛区|丰岛区]][[池袋|池袋]]的高239.7米的[[太阳城60|太阳城60大厦]]的打破。是一栋黑色调玻璃外观的建筑，主要的商店和餐厅设在地下1楼～3楼，4楼与5楼设有诊所，5楼到55楼中除了54楼外都是作为办公室之用，54楼的[[新宿三井俱乐部|新宿三井俱乐部]]需要会员才能进入。没有设置展望台。

== 主要租户·设施 ==
