《'''新生'''》（{{lang-en|'''''Regeneration'''''}}）是一部1915年美國[[無聲電影|無聲]][[傳記片|傳記]][[犯罪片|犯罪]]電影，由{{link-en|魯爾·窩路殊|Raoul Walsh}}執導及共同編劇。這是窩路殊的第一部執導的長篇電影<ref name=mcdannell>{{cite book|title=''Catholics in the Movies''|date=2008|publisher=Oxford University Press|isbn=0-195-30656-2|page=38|editor=McDannell, Colleen |language = en}}</ref>，主演是{{link-en|Rockliffe Fellowes|Rockliffe Fellowes}}和{{link-en|安娜·Q·佩雷利|Anna Q. Nilsson}}，本片是由{{link-en|卡爾·哈博|Carl Harbaugh}}和窩路殊根據{{link-en|歐文·基爾達爾|Owen Kildare|歐文·弗勞利·基爾達爾}}的回憶錄《''My Mamie Rose''》，以及基爾達爾和{{link-en|沃爾特·C·哈克特|Walter C. Hackett}}的改編舞台劇而成的改編電影。<ref>{{cite book|last=Solomon|first=Aubrey|title=''The Fox Film Corporation'', 1915-1935: A History and Filmography|date=2011|publisher=McFarland|isbn=0-786-48610-4|page=229 |language = en}}</ref>

本片被引述為第一部長篇[[黑幫電影|黑幫電影]]的其中之一<ref>{{cite book|last=Hahn Rafter|first=Nicole|title=''Shots in the Mirror: Crime Films and Society''|date=2006|publisher=Oxford University Press|isbn=0-195-17506-9|page=23 |language = en}}</ref>。《新生》講述一名貧窮的孤兒長大後掌控犯罪團夥，直至他遇到一名令他想改變的女人。

== 概要 ==
故事講述Owen（Rockliffe Fellowes 飾演）的生活，Owen是一個年輕的[[愛爾蘭裔美國人|愛爾蘭裔美國]]男孩，他的母親死後被迫陷入貧困。結果，Owen被迫住在街上，最終變成了犯罪生活。Owen最終被和藹的Marie Deering（安娜·Q·佩雷利 飾演）改邪歸正。還有加入了一場遊覽渡輪上的大火，與1904年的{{link-en|斯洛昆將軍號|PS General Slocum|斯洛昆將軍號災難}}很相似。

== 製作 ==
以[[紐約市|紐約市]]為背景，《新生》在紐約市的[[下東城|下東城]]拍攝，並用真正的[[性交易|妓女]]、[[黑幫|黑幫]]和[[露宿者|無家可歸的人]]作[[臨時演員|臨時演員]]。<ref name=finler />它是第一部由[[二十世紀福斯|二十世紀福斯]]的先驅[[二十世紀福斯|福斯電影公司]]所製作的。<ref name=silentera>{{Cite web |url = http://www.silentera.com/PSFL/data/R/Regeneration1915.html |title = Regeneration |publisher = silentera.com |language = en}}</ref>

== 發行 ==
這部電影最初於1915年9月13日上映，備受好評及賣座。<ref name=finler>{{cite book |last=Finler |first=Joel Waldo |title=''Movie Directors Story'' |date=1986 |publisher=Random House Value Publishing |isbn=0-517-48079-4 |page=80 |language = en}}</ref><ref>{{cite book |last=Moss |first=Marilyn Ann |title=''Raoul Walsh: The True Adventures of Hollywood's Legendary Director'' |date=2013 |publisher=The University Press of Kentucky |isbn=0-813-14444-2 |page=50 |language = en}}</ref>它在1919年1月12日被重映<ref name=silentera />。

== 家庭媒體 ==
2001年，{{link-en|Image Entertainment|Image Entertainment}}發售《新生》區域碼1的DVD，連同1915年電影《''{{link-en|Young Romance (film)|Young Romance (film)|Young Romance}}''》。本片目前是處於[[公有領域|公有領域]]。<ref name=silentera />
