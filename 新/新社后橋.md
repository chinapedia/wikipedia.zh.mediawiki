'''新社后橋'''，是[[台灣|台灣]][[新北市|新北市]][[汐止區|汐止區]]的一座重要[[橋樑|橋樑]]，完工後將成為全[[台灣|台灣]]首座「跨河捷運共構橋」。

== 簡介 ==
該橋的地理位置於原社后橋上游約200公尺處，興建經費約新台幣6.89億元(中央負擔50,085,000元，新北市自籌476,915,000元)，由[[台灣世曦工程|台灣世曦工程]]設計，[[春原營造|春原營造]]負責土建施工，鋼橋部份則由[[春源鋼鐵|春源鋼鐵]]負責製裝施工，其主要的特殊設計為上層設置[[台北捷運民生汐止線|汐止民生線捷運]]軌道，預計2015年五月完工。
並將成為台灣跨度最大的雙層曲弦華倫式([[:en:Truss_bridge#Warren_truss|Warren truss]])鋼[[桁架橋|桁架橋]]梁，橋體本身因需同時承載車輛及日後捷運列車的荷重，已設計二個橋墩搭配四個高強度盤式支承，在未來承載全部鋼構重量約5,300餘公噸，這相當於同跨距橋梁的兩倍以上。<ref>[http://61.222.185.194/?FID=14&CID=251068 朱立倫視察新社后橋共構工程]</ref>

== 橋梁諸元 ==
*橋梁下層：
**雙向雙車道供汽機車行駛。
**兩側設置自行車聯絡道連接基隆河南北兩側的自行車道。
**兩側設置行人專用通道供行人通行。
*橋梁上層：預鑄捷運汐止民生線之鋼架及底座。<ref>[https://drive.google.com/file/d/0B8Rupc-zEPr1R1RBMVhUazJsVW8/edit?pli=1 新北市政府捷運工程局標註之捷運行駛新社后橋的示意圖 ]</ref>

== 工程效益 ==
*完工後將連接基隆河北岸的福德一路及南岸的工建路，減輕社后橋及中興路的交通負荷，並改善福德一路聯外的交通品質。<ref>[http://www.mygonews.com/news/detail/news_id/111247/  全國首座捷運共構橋梁，市長、議員視察新社后橋工程]</ref>
*提供捷運汐止民生線路廊的共構，減少捷運日後二次施工的費用與界面。
*配合[[基隆河|基隆河]]兩百年洪水頻率的整治計畫，提升該橋的防洪標準。<ref>[http://www.co.ntpc.gov.tw/PublicAffainDetail.aspx?entry=44 新北市政府新建工程處-新社后橋工程說明] {{webarchive|url=https://web.archive.org/web/20141222043626/http://www.co.ntpc.gov.tw/PublicAffainDetail.aspx?entry=44 |date=2014-12-22 }}</ref>
