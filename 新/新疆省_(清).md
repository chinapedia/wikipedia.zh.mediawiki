'''新疆省'''（{{lang-mnc|ᡳᠴᡝ <br>ᠵᡝᠴᡝ <br>ᡤᠣᠯᠣ|v=ice jecen golo}}），全名為'''甘肃新疆省'''，是[[清朝|清朝]]末期22個省份之一。[[同治|同治]]四年（1865年），[[浩罕汗國|浩罕汗國]][[阿古柏|阿古柏]]入侵[[喀什噶爾|喀什噶爾]]、[[阿克蘇|阿克蘇]]、[[烏魯木齊|烏魯木齊]]等地。[[光緒|光緒]]二年（1876年），[[陝甘總督|陝甘總督]][[左宗棠|左宗棠]]率軍出甘肅，於次年收復了新疆大部分地區，並從[[俄罗斯帝国|俄国]]手中收復[[伊犁|伊犁]]。光緒十年（1884年）十一月正式建省，[[行政中心|行政中心]]由[[伊犁|伊犁]]遷至[[省会|省会]][[迪化|迪化]]（今烏魯木齊）。

== 歷史 ==
{{中國歷史上的新疆}}

乾隆二十年（1755年），清軍[[準噶爾之役|滅準噶爾]]，三年後又消滅[[阿睦爾撒納|阿睦爾撒納]]勢力，西域底定。[[乾隆帝|乾隆帝]]把這片土地命名為「新疆」，取名為新附旧疆。

[[File:The_Lifting_of_the_Siege_of_the_Black_River_(Khara-Usu).jpg|200px]]

乾隆二十四年（1759年），清朝平定[[回部|回部]]的[[大小和卓之亂|大小和卓之亂]]，採用[[伯克制|伯克制]]和軍府制，開始統治南疆。其後大和卓之孫[[和卓玉素普|和卓玉素普]]、[[張格爾|張格爾]]等數次回國叛亂。回部之前在準噶爾暴力統治之下。而準噶爾部的滅亡也導致新疆及中亞部分（吉爾吉斯斯坦全部，塔吉克斯坦、烏茲別克斯坦部分，哈薩克斯坦東部）完全伊斯蘭化。之前此區域統治民族準噶爾部蒙古全民信奉藏傳佛教。乾隆二十七年（1762年），清朝在[[伊犁|伊犁]]設立[[伊犁將軍|伊犁將軍]]，統一行使對[[天山|天山]]南北各地的軍政管轄。

道光六年（1826年）大和卓之孫[[張格爾|張格爾]]依附英國叛亂，後被擒至北京處死。

[[同治回亂|同治回亂]]，從同治元年（1862年）至十二年（1873年）在中國陝西、甘肅爆發的一次境內的穆斯林對漢族之間的仇殺。它維持了十年多，波及陝西、甘肅、寧夏、青海和新疆等地區，最後被以漢人湘軍為主的清朝軍隊鎮壓。而同治回亂的很多穆斯林首領也退到新疆，或借道新疆逃往[[俄罗斯帝国|俄国]]。

19世紀中葉，俄國威脅清朝的整個北部邊境。同治三年（1864年）的[[塔城條約|塔城條約]]將新疆西北部[[巴爾喀什湖|巴爾喀什湖]]以南大片土地（即[[外西北|外西北]]）割讓給俄國，今[[哈薩克斯坦|哈薩克斯坦]]、[[吉爾吉斯斯坦|吉爾吉斯斯坦]]和[[塔吉克斯坦|塔吉克斯坦]]。

同治十年（1871年），俄國侵佔包括[[伊犁|伊犁]]城（[[伊寧市|伊寧市]]）在內的[[伊犁河|伊犁河]]谷，當時清朝在新疆只剩下[[塔城|塔城]]、[[哈密|哈密]]等少数据點。

光緒元年（1875年），清朝[[陝甘總督|陝甘總督]][[左宗棠|左宗棠]]就任欽差大臣，督辦新疆事務。到光緒三年（1877年）底，清軍陸續收復了中亞[[浩罕汗國|浩罕汗國]][[阿古柏|阿古柏]]侵佔了多年的天山南北，史稱「[[清軍收復新疆之戰|清軍收復新疆之戰]]」。光緒七年（1881年），清政府收復被俄国佔領長達11年之久的[[伊犁|伊犁]]地區。光緒十年（1884年）十一月十九日，[[清政府|清政府]]頒發[[光绪帝|光绪帝]][[上諭|上諭]]，任命[[劉錦棠|劉錦棠]]為[[甘肃新疆巡撫|甘肃新疆巡撫]]，[[魏光燾|魏光燾]]為新疆[[布政使|布政使]]，標誌著新疆正式建省。'''甘肃新疆省'''實行與[[中國本部|中國本部]][[内地十八省|内地十八省]]一樣的行政制度，由巡撫統管全疆各項軍政事務，新疆政治中心由[[伊犁|伊犁]]移至[[迪化|迪化]]（今[[烏魯木齊|烏魯木齊]]）。<ref>{{cite web |url = http://wen.org.cn/modules/article/view.article.php/1219/c18 |title = 清末新疆建省研究綜述 |author = |language = zh-hans }}</ref>

== 行政區劃 ==
清[[同治|同治]]四年（1865年），[[浩罕國|浩罕國]]阿古柏入侵[[喀什噶爾|喀什噶爾]]、阿克蘇、烏魯木齊等地。光緒二年（1876年），[[陝甘總督|陝甘總督]]左宗棠帥官軍出甘肅，7月入烏魯木齊，次年1月攻佔[[和闐|和闐]]，收復了新疆大部分地區。光緒七年（1881年）從[[俄羅斯|俄羅斯]]手中收復[[伊犁|伊犁]]。光緒十年，左宗棠奏請設立「新疆省」，由陝甘總督管轄，行政中心由伊犁移至迪化。新疆省初置[[鎮迪道|鎮迪道]]、[[阿克蘇道|阿克蘇道]]、[[喀什噶爾道|喀什噶爾道]]、[[伊塔道|伊塔道]]4道。至清末已廢除[[伯克制|伯克制]]，有[[迪化|迪化]]、[[伊犁|伊犁]]、[[焉耆|焉耆]]、[[溫宿|溫宿]]、[[疏勒|疏勒]]、[[莎車|莎車]]6府，吐魯番、精河、英吉沙爾、塔爾巴哈臺、庫爾哈喇烏蘇、哈密、鎮西7[[直隸廳|直隸廳]]、庫車、烏什、和闐3[[直隸州|直隸州]]、1散廳、1散州、21縣。

== 地方官員 ==
{{See also|陝甘總督|甘肅新疆巡撫|伊犁將軍|甘肅新疆布政使|甘肅新疆鎮迪道}}

新疆建省後廢除了維吾爾[[伯克制|伯克制]]，形成了[[陝甘總督|陝甘總督]]、[[甘肅新疆巡撫|甘肅新疆巡撫]]與[[伊犁將軍|伊犁將軍]]、[[參贊大臣|參贊大臣]]共治的局面。其中，由[[甘肅新疆巡撫|甘肅新疆巡撫]]統轄[[甘肅新疆布政使|甘肅新疆布政使]]（負責行政）和[[甘肅新疆鎮迪道|甘肅新疆鎮迪道]]（1885年兼按察使銜，1910年更名為提法使銜。兼管全疆刑名，即司法）等。
