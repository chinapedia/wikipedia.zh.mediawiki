'''新界區專線小巴807K線'''是由富明有限公司營辦的一條綠色專線小巴路線，由[[大學站|大學站]]往來[[井頭|井頭]]<ref>{{Cite web |url=http://hketransport.td.gov.hk/ris_page/get_gmb_detail.php?route_id=2392&lang=TC |title=運輸署- 新界區專線小巴路線第807K號 |access-date=2016-12-18 |archive-url=https://web.archive.org/web/20161221013144/http://hketransport.td.gov.hk/ris_page/get_gmb_detail.php?route_id=2392&lang=TC |archive-date=2016-12-21 |dead-url=no }}</ref>。另設特快輔助線'''807S'''，取道[[馬鞍山繞道|馬鞍山繞道]]往返，袛於星期一至五早上繁忙時間提供服務。<ref>{{Cite web |url=http://681busterminal.com/version2/index.php?fi=a811&id=582&vid=7 |title=681巴士總站：新界區專線小巴807S線 |access-date=2018-04-01 |archive-url=https://web.archive.org/web/20180401212707/http://681busterminal.com/version2/index.php?fi=a811&id=582&vid=7 |archive-date=2018-04-01 |dead-url=no }}</ref>

'''新界區專線小巴807A線'''<ref>{{Cite web |url=http://hketransport.td.gov.hk/ris_page/get_gmb_detail.php?route_id=2636&lang=TC |title=運輸署 - 新界區專線小巴路線第807A號 |access-date=2018-04-30 |archive-url=https://web.archive.org/web/20180401212634/http://hketransport.td.gov.hk/ris_page/get_gmb_detail.php?route_id=2636&lang=TC |archive-date=2018-04-01 |dead-url=no }}</ref>
<ref>{{Cite web |url=http://www.681busterminal.com/version2/index.php?fi=a201&id=450 |title=681巴士總站 - 新界區專線小巴807A線 |access-date=2018-04-30 |archive-url=https://web.archive.org/web/20180401144432/http://www.681busterminal.com/version2/index.php?fi=a201&id=450 |archive-date=2018-04-01 |dead-url=no }}</ref>及'''新界區專線小巴807C線'''<ref>{{Cite web |url=http://681busterminal.com/version2/index.php?fi=a811&id=580&vid=7 |title=681巴士總站 - 新界區專線小巴807C線 |access-date=2018-04-30 |archive-url=https://web.archive.org/web/20180401144355/http://681busterminal.com/version2/index.php?fi=a811&id=580&vid=7 |archive-date=2018-04-01 |dead-url=no }}</ref>。是由富明有限公司營辦的綠色專線小巴路線，兩者皆來往[[海柏花園|海柏花園]]至[[大學站|大學站]]。前者去程途經[[欣安邨|欣安邨]]與[[頌安邨|頌安邨]]，回程經[[馬鞍山路|馬鞍山路]]直達[[大學站|大學站]]；而後者去程經[[馬鞍山路|馬鞍山路]]特快直達馬鞍山市中心，回程途經[[頌安邨|頌安邨]]與[[欣安邨|欣安邨]]才到大學站。

'''新界區專線小巴807B線'''是由富明有限公司營辦的一條綠色專線小巴路線，由[[海栢花園|海栢花園]]往來[[黃竹灣|黃竹灣]]（麥邊迴旋處）<ref>{{Cite web |url=http://hketransport.td.gov.hk/ris_page/get_gmb_detail.php?route_id=2637&lang=TC |title=運輸署- 新界區專線小巴路線第807B號 |access-date=2018-05-04 |archive-url=https://web.archive.org/web/20180401144707/http://hketransport.td.gov.hk/ris_page/get_gmb_detail.php?route_id=2637&lang=TC |archive-date=2018-04-01 |dead-url=no }}</ref>，以循環線運作。請見2018年4月1日之站頭通告。

'''新界區專線小巴807X線'''是由富明有限公司營辦的一條綠色專線小巴路線，來往[[馬料水公共運輸交匯處|大學站]]及[[烏溪沙站|烏溪沙站]]，取道[[馬鞍山繞道|馬鞍山繞道]]往返兩地，並繞經[[白石|白石]]。

==歷史==
*1995年：807K投入服務，為小巴大王[[馬亞木|馬亞木]]首條投得往來[[沙田區|沙田區]]及[[大埔區|大埔區]]（只限[[西貢北|西貢北]]）的日間專線小巴路線，初時往來大學站至泥涌。
*1998年：路線延長至[[井頭|井頭]]。
*1999年：807P投入服務。
*2002年1月27日：接受八達通卡繳付車資。 
*2002年：由泥涌延長至井頭。

===八達通轉乘優惠===
;807系列轉乘優惠
*807B互轉807S：由807S往大洞方向於上車後1小時內轉乘807B往黃竹灣方向，或由807B往馬鞍山方向轉乘807S往大學站方向，兩程車費合共$8。
;港鐵轉乘優惠
由807K線於上車後1小時內於[[馬鞍山站_(香港)|馬鞍山站]]或[[烏溪沙站|烏溪沙站]]轉乘港鐵，或乘搭港鐵馬鞍山線於馬鞍山站或烏溪沙站出閘後1小時內轉乘本路線，次程可獲$0.5折扣優惠。

乘客請注意以下各項細則：
*乘客須以同一張八達通卡繳付兩程車費。
*必須於指定時限內完成轉乘，期間乘客不可以用同一張八達通卡乘搭其他交通工具，否則轉乘優惠將被取消；惟以該八達通卡進行與乘車無關的交易，如增值、購買汽水、使用公共電話、即影即有照片不多於9次則不受影響。
*如乘客剛以同一張八達通卡享用機場快線「免費港鐵接駁」，則不可同時享用轉乘優惠。
*如港鐵同時提供多項優惠，乘客只可享用最高優惠的一項。
*在轉乘次程車程前，必須確保八達通卡之餘額為正數，設有自動增值系統之八達通卡則不受影響。

==小資料==
*807K線初時的井頭村總站，位處[[西貢北|西貢北]]，是屬於[[大埔區|大埔區]]而並非西貢區，在井頭村總站是有大埔區議會的標誌，西貢北列入大埔區是因為昔日交通上有緊密聯繫，西貢北居民經常前往大埔作買賣交易，因而跟大埔建立密切關係，至今仍維持不變，但一般市民不會把西貢北當作大埔區，反之當作西貢區，因此本路線是首條往來[[沙田區|沙田區]]及大埔區的日間專線小巴路線，但首條真正往來大埔新市鎮至沙田的專線小巴路線，卻是2004年11月開辦的[[新界區專線小巴28K線|新界區專線小巴28K線]]。
*807B線初時的水浪窩總站，位處西貢北，是屬於大埔區而並非西貢區，在水浪窩總站是有大埔區議會的標誌，西貢北列入大埔區是因為昔日交通上有緊密聯繫，西貢北居民經常前往大埔作買賣交易，因而跟大埔建立密切關係，至今仍維持不變，但一般民眾不會把西貢北當作大埔區，反之當作西貢區。延長路線後，澳頭、黃竹灣及麥邊迴旋處已屬於西貢區，因此成為首條往來[[沙田區|沙田區]]經大埔區至西貢區的跨區專線小巴路線，不過兩邊總站分別距離沙田市中心及西貢市中心甚遠。
