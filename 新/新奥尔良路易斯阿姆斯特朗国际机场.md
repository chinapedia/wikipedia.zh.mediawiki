'''新奥尔良路易斯阿姆斯特朗國際機場'''（{{lang-en|Louis Armstrong New Orleans International Airport}}，IATA：'''MSY'''，IACO：'''KMSY'''），是一座位於[[美国|美国]][[路易斯安那州|路易斯安那州]][[杰佛逊堂区_(路易斯安那州)|傑佛遜郡]]的民用機場。归属于新奥尔良市并位于市中心商业区西10海里（19公里）处。机场地址为路易斯安那州[[肯纳_(路易斯安那州)|肯纳市]]航线路900号。一小部分跑道位于[[圣查尔斯堂区_(路易斯安那州)|聖查爾斯郡]]的飞地。阿姆斯特朗机场为大新奥尔良区和东南路易斯安那的主要商业机场。过去该机场被称为莫伊森机场，也叫做路易斯阿姆斯特朗国际机场和新奥尔良国际机场。该机场坐落在平均海拔4.5英尺（1.4米）的平面上，为世界上第二海拔低的国际机场，仅次于位于阿姆斯特丹平均海平面下11英尺（4.5米）荷兰[[阿姆斯特丹史基浦機場|阿姆斯特丹史基浦機場]]。卡特里娜飓风前，每年970万人访问该机场，并且几乎全部以此为最终目的地。2008年，共7,944,397旅客访问该机场，较前一年提高了5.5%。该机场具有全美机场最优良的安全纪录。

2008年2月，该机场被《[[美国新闻与世界报道|美国新闻与世界报道]]》杂志在47家机场中选为排名第4良好旅行体验，基于该机场的航班较少推迟以及经常性的较低的行李托运。

该机场曾是美国前往拉美国家旅行的主要关口。现在，大多数旅行通过其他设有国际航空公司枢纽机场的城市中转。

该机场二战后开始运营，以取代较老的{{tsl|en|New Orleans Lakefront Airport|紐奧良湖畔機場}}（IATA代码为NEW，ICAO代码为KNEW，现为专用航空用途），成为新奥尔良主要机场。2001年后，该机场以一名著名的爵士乐艺人[[路易斯·阿姆斯特朗|路易斯·阿姆斯特朗]]命名。该地区的[[美国国家气象局|美国国家气象局]][[天气预报|預報]]辦公室曾位于该机场，但是已经搬到[[斯莱德尔_(路易斯安那州)|斯莱德尔]]郊区，并以LIX为IATA代码和KLIX为ICAO代码。

== 历史 ==
[[Image:ArmstrongAirportJune2007.jpg|thumb]]

该机场因勇敢的飞行家{{tsl|en|John Moisant|约翰·莫圣特}}而得名，1910年他死于一場发生在这片土地上（當時還是农田）的一场飞机坠毁事故中。由于雷克弗隆特机场保留了NEW作为其IATA代码，新奥尔良路易斯阿姆斯特朗国际机场因莫圣特饲料场而得其IATA代码MSY。

莫圣特机场计划始于1940年，因为新奥尔良原有的树山机场（新奥尔良雷克弗隆特机场）需要扩建或替代。随着二次世界大战出现这片土地成为政府的一个空军基地。战后，重归民用并于1946年五月开始在莫圣特机场进行商业服务。

== 設施和飞机 ==
新奥尔良路易斯阿姆斯特朗国际机场占地1500英亩（607公顷）,海拔4英尺（1米）。机场有三条跑道：10/28为10,104 x 150英尺（3,080 x 46 米）混凝土表面；1/19为7,001 x 150英尺（2,134 x 46 米）混凝土表面；6/24为1,088 x 150英尺（1,088 x 46 米）沥青表面。

截至2007年12月31日,在12个月内机场有129,228架飞机运行，每天约354架：58%为商业航班、21%为专用飞行、19%为短程小客机及1%为军用飞机。其中有19架以该机场为主场：21%为单引擎、21%为多引擎、42%为喷气式與16%的直升机。

== 终端站、航空公司及终点 ==
该机场有两个终端：东和西。由一个中央售票通道相连。连接着4个大厅：A、B、C與D。

[[Image:msyconcourseC.jpg|thumb]]
