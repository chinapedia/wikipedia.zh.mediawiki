'''新北產業園區'''，前稱'''五股工業區'''，為[[中華民國|中華民國]][[經濟部工業局|經濟部工業局]]所創辦的國家級[[工業區|工業區]]，位於今[[台灣|台灣]][[新北市|新北市]][[五股區|五股區]]與[[新莊區|新莊區]]。

== 歷史 ==
為積極發展工業、促進地方繁榮與發展，當時台北縣政府（今[[新北市政府|新北市政府]]）得到[[行政院|行政院]]核准，在1984年1月依據《[[獎勵投資條例|獎勵投資條例]]》的規定，開發五股工業區。一方面容納[[二重疏洪道|二重疏洪道]]的合法[[工廠|工廠]]拆遷戶使用，一方面提供興辦工業人作為建廠用地，以紓解工業用地殷切之需求。此外，當時的台北縣政府在此工業區內設立了[[新北市工商展覽中心|工商展覽中心]]，提供了一個國際級展覽場地給當地的廠商。2011年4月由[[新北市市長|新北市市長]][[朱立倫|朱立倫]]出面協調，於100年6月1日起正式更名為新北產業園區<ref>{{cite news|publisher=中時電子報|title=捷運五工站 將更名新北站|date=2011-04-29}}</ref>。

== 地理位置 ==
新北產業園區位於[[新北市|新北市]][[五股區|五股區]]及[[新莊區|新莊區]]交界，東臨[[二重疏洪道|二重疏洪道]]，北臨[[中山高速公路|中山高速公路]]，南鄰[[新莊區|新莊區]]市區，西界為[[中港大排|中港大排]]，在中華民國經濟部轄工業區中最為接近[[台北市|台北市]]，區位條件非常優越。
== 公共設施 ==
*道路、路燈：全區闢設14條道路總長13,850公尺，路燈354盞。
*排水系統：主要排水溝全長3,800公尺，雨水管線 23,800公尺。
*自來水系統：每日最大供水量9,000噸，供水量充裕。
*電話系統：區內供應電話40,000門，目前使用24,000門。
*電力：區內有變電所，電力供應穩定，最高使用達180,000千瓦。
*綠帶：區內綠帶3.309公頃，於各主要、次要道路栽種

== 概況 ==
新北產業園區的業種廣泛，舉凡[[食品|食品]]、[[紡織|紡織]]、[[家具|家具]]、[[印刷|印刷]]、[[塑膠化學|塑化]]、[[電子|電子]]、[[機械|機械]]等等業種一應俱全，用地幾乎完全開發完畢，新進廠商一地難求。然而，最近十年，因為產業轉型與外移緣故，閒置土地高達45.92公頃，相當於2座大安森林公園，5座大巨蛋，浪費超過60億元。<ref>產業園區土地閒置 立院：十年浪費超過60億 聯合報 2017-03-23 </ref>

== 教育 ==
