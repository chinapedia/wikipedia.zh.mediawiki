《'''新參者'''》是[[日本|日本]]作家[[東野圭吾|東野圭吾]]的[[推理小說|推理小說]]，也是「'''[[加賀恭一郎系列|加賀恭一郎系列]]'''」的第六本小說。本作最初是在[[講談社|講談社]]的《{{link-ja|小說現代|小説現代}}》自2004年8月號開始連載，之後5年內斷斷續續連載了9個短篇。小說每個短篇中的主角都不同，但隨著各個短篇事件的解決，與事件並無直接關係的周邊人物與謎團逐漸慢慢浮現。單行本於2009年由[[講談社|講談社]]出版。

2010年以東野圭吾《新參者》的改編成同名電視劇。

== 故事簡介＆登場人物 ==
在[[日本橋|日本橋]]附近獨居的45歲女性遭到勒斃，周圍一直環繞著一個聲音：「為什麼，那樣的好人……」。為了得知真相，剛到任日本橋警署的警官加賀恭一郎解開九個謎題，在未知的土地上奔波來回。

關於'''[[加賀恭一郎系列#加賀恭一郎|加賀恭一郎]]'''、'''[[加賀恭一郎系列#松宮脩平|松宮脩平]]'''等人物請參看「[[加賀恭一郎系列#登場人物|加賀恭一郎系列－登場人物]]」。

=== 第一章·煎餅店的女兒 ===
* 原題：{{lang|ja|煎餅屋の娘}}
* 初出：『小說現代』2004年8月號
: 在[[日本橋|日本橋]][[人形町站|人形町]]裡，菜穗和祖母聰子及父親文孝經營著一家煎餅店，有天一個名叫加賀的刑警慕名到來。加賀正在調查一個45歲女人被勒斃的案件，而案發當日來過煎餅店的保險推銷員田倉是案件的知情人。菜穗和聰子都認為以田倉的性格是不可能殺人的，但周圍的人的供詞卻與田倉供述的案發當日的不在場證明相矛盾。究竟田倉在隱瞞些什麼？

:*; {{nihongo|上川 菜穗|{{ruby|上川|かみかわ}} {{ruby|菜穂|なほ}}|Kamikawa Naho}}
:: 煎餅屋「甘辛」的女兒，在美容學校上課。母親早逝，由父親和祖母養大。
:*; {{nihongo|上川 聰子|{{ruby|上川|かみかわ}} {{ruby|聡子|さとこ}}|Kamikawa Satoko}}
:: 經營「甘辛」的上代店主的妻子。性格頑固，愛嘮叨。前幾天因患病而在鬼門關徘徊過。
:*; {{nihongo|上川 文孝|{{ruby|上川|かみかわ}} {{ruby|文孝|ふみたか}}|Kamikawa Fumitaka}}
:: 「甘辛」的現店主。為了聰子而演了一場戲，卻和這次的問題聯繫上了。
:*; {{nihongo|田倉 慎一|{{ruby|田倉|たくら}} {{ruby|慎一|しんいち}}|Takura Shinichi}}
:: 新都生命保險推銷員。

=== 第二章·料理店的學徒 ===
* 原題：{{lang|ja|料亭の小僧}}
* 初出：『小說現代』2005年6月號
: 料理店「松矢」的學徒修平因案件關係而被加賀問話。理由是當日修平買了7個有餡，3個沒餡的重盛人形燒，與被害者家中放著的重盛人形燒是同一個比例。但修平聲稱買了的人形燒被自己全部吃光了。除此之外料理店「松矢」還有著幾個秘密。修平究竟隱瞞著些什麼？「松矢」的秘密又是？

:*; {{nihongo|佐佐木 修平|{{ruby|佐々木|ささき}} {{ruby|修平|しゅうへい}}|Sasaki Shuuhei}}
:: 料理店「松矢」的學徒。案發當日買了十個人形燒。似乎在為老闆保守著某個秘密。
:*; {{nihongo|枝川 泰治|{{ruby|枝川|えだかわ}} {{ruby|泰治|たいじ}}|Edakawa Taiji}}
:: 料理店「松矢」的老闆，負責為料理店選購「真妻」的[[芥末|芥末]]。但背地裡似乎在搞外遇。
:*; {{nihongo|枝川 賴子|{{ruby|枝川|えだかわ}} {{ruby|頼子|よりこ}}|Edakawa Yoriko}}
:: 料理店「松矢」的老闆娘，負責打點料理店里里外外的事。表面不動聲色但暗裡對丈夫在外的事瞭如指掌。

=== 第三章·瓷器店的媳婦 ===
* 原題：{{lang|ja|瀬戸物屋の嫁}}
* 初出：『小說現代』2005年10月號
: 瓷器店的店主鈴江和她的媳婦麻紀相處不融洽，經常吵架。面對兩人的爭執，店主兒子尚哉左右為難。這時加賀來到店裡詢問，查明了案發前被害的女人曾給麻紀發送過郵件，因麻紀案發前曾拜託她幫自己買一把剪刀。但是麻紀家裡已經有了同樣的剪刀，這點令加賀覺得奇怪。另一方面，鈴江正準備要外出旅行，麻紀為什麼要暗中買一把剪刀呢？

[[日劇|電視劇]]於2010年4月18日至6月20日於[[TBS電視|TBS]]的[[周日劇場|周日劇場]]播出，由[[阿部寬|阿部寬]]主演（自「[[龍櫻|龍櫻]]」之後首次在TBS演出連續劇），另外同劇的[[黑木美沙|黑木美沙]]、[[向井理|向井理]]與[[溝端淳平|溝端淳平]]則是首次於TBS演出電視劇。片頭和片尾的旁白由黑木美沙擔任。

電視劇劇情大抵忠於原作，但在原作中擔任重要記者角色的青山亞美的戲份被削減，另外加賀的表弟松宮修平的角色設定裡有些地方和原作不同。

作為本作的續篇，改編自《[[紅手指|紅手指]]》的電視劇「[[紅手指#電視劇|紅手指～『新參者』加賀恭一郎再次登場！]]」亦在2011年1月3日播出。

宣傳語是「案件的嫌疑人是在日本橋人形町來往的人們」<ref>原文：「{{lang|ja|事件の容疑者は､日本橋･人形町を行き交う人たち。}}」</ref>、「兇手就在我的心中…」<ref>原文：「{{lang|ja|犯人はこの中にいる…。}}」</ref>。
