'''新竹縣私立內思高級工業職業學校'''是一所位於[[中華民國|中華民國]][[台灣省|台灣省]][[新竹縣|新竹縣]]私立[[高職|高級工業職業學校]]，由[[耶穌會|耶穌會]]創辦的學校。簡稱為'''內思高工'''，因期望學生效法天主教聖人「St. Aloysius」(中文譯作「聖內思」)之精神，亦希望聖內思作學校青年的主保，故而取名「內思高工」。創校於1954年。校地面積為2.9398公頃。

== 校史 ==
* 民國24年創校於[[安徽|安徽]][[蕪湖|蕪湖]]，以教育機械工業人才為主。
* 民國43年在台復校，秉持耶穌會-----「在落後地區興學以嘉惠貧寒子弟」的傳統，校址選在新埔鎮天主堂。
* 45年遷入現址，47年立案為中級補校，設機工、電工二科。
* 54年改制為五年制中級業職業學校，55年成立「電工科」。
* 57年，改制為三年制「高級工業職業學校」。
* 60年，成立進修補校，設有「機工」、「電工」兩科。
* 62年，設立「電子自動控制科」。
* 75年新課程修訂，電工科改「電機科」電子自動控制科改「控制科」，次年機工科改「機械科」。
* 79年成立「資訊科」。
* 88年「控制科」改名為「電子科」。
* 101年增設「多媒體設計科」。
* 103年增設「電機電子群不分科實驗班」。

== 學校願景 ==
「以明辨求更，積極創新的依納爵精神，激勵學生追求卓越，進而獻身服務社會」。
 
== 使命 ==
以耶穌會中華省教育委員會訂定之「耶穌會中華省學校畢業生的特質」為藍本，本校使命為：「提供發揮個人卓越智能，培養終身學習能力、愛人愛己、重視道德信仰與社會正義之友善學習校園」。
