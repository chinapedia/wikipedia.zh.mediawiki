'''新竹縣立信勢國民小學'''（即「'''新竹縣湖口鄉信勢國民小學'''」，簡稱「'''信勢國小'''」），位於[[新竹縣|新竹縣]][[湖口鄉|湖口鄉]]信義村成功路360號。

== 校史沿革 ==
*（1956年）[[民國45年|民國45年]]8月1日 創設「'''新湖國民學校信勢分校'''」。
*（1958年）[[民國47年|民國47年]]3月20日 由[[新湖國校|新湖國校]]遷移現校址。
*（1958年）[[民國47年|民國47年]]8月1日 奉准[[獨立|獨立]]改稱為「'''新竹縣湖口鄉信勢國民學校'''」。
*（1962年）[[民國51年|民國51年]]11月14日 新購[[操場|操場]]約0.5[[甲_(單位)|甲]]地。
*（1968年）[[民國57年|民國57年]]8月1日 奉令改為「'''新竹縣湖口鄉信勢國民小學'''」。
*（1974年）[[民國63年|民國63年]]8月10日 建綜合球場一座。
*（1976年）[[民國65年|民國65年]]8月26日 建[[圍牆|圍牆]]、[[圓環|圓環]]、鐵門、[[涼亭|涼亭]]等工程。
*（1980年）[[民國69年|民國69年]]1月 增購校地1.0433[[公頃|公頃]]（3155[[坪|坪]]）。
*（1986年）[[民國75年|民國75年]]8月1日 成立自立[[幼稚園|幼稚園]]。
*（1987年）[[民國76年|民國76年]]5月9日 校園填土、建排水溝、[[孔子|孔子]]塑像、花園工程驗收使用。
*（1988年）[[民國77年|民國77年]]8月 校地擴增0.6[[公頃|公頃]]。
*（1989年）[[民國78年|民國78年]]8月1日 該校自立[[幼稚園|幼稚園]]奉准改為[[附設幼稚園|附設幼稚園]]。
*（1993年）[[民國82年|民國82年]] [[操場|操場]]完成鋪設[[PU跑道|PU跑道]]。
*（1994年）[[民國83年|民國83年]]12月1日[[活動中心|活動中心]]工程開工。
*（1996年）[[民國85年|民國85年]]3月6日 [[活動中心|活動中心]]落成。
*（1999年）[[民國88年|民國88年]]2月1日 開辦學生[[營養午餐|營養午餐]]。
*（2002年）[[民國91年|民國91年]]10月31日 至善樓新建工程開工[[動土|動土]]典禮，[[鄭永金|鄭永金]]縣長、[[黃德共|黃德共]]鄉長蒞臨[[剪綵|剪綵]]。
*（2003年）[[民國92年|民國92年]]8月1日 [[資源班|資源班]]成立。
*（2004年）[[民國93年|民國93年]]4月17日 至善樓落成啟用。
*（2005年）[[民國94年|民國94年]]8月1日 附設[[幼稚園|幼稚園]]由半天班改為全天班。
*（2007年）[[民國96年|民國96年]]新建廁所2間。
*（2008年）[[民國97年|民國97年]]4月19日 舉辦創校五十周年[[校慶|校慶]]。
*（2010年）[[民國99年|民國99年]]校舍屋頂防漏隔熱工程。
*（2011年）[[民國100年|民國100年]]11月圍牆整修並加舖學生專用道工程完工。

== 學校環境 ==
該校位於'''[[湖口鄉|湖口鄉]]成功路北勢溪畔'''，原屬信勢村，後因行政區調整改為信義村。緊鄰[[湖口鄉|湖口]]市區。

== 學校特色 ==
*鼓舞隊
:該校'''[[鼓舞隊|鼓舞隊]]'''起源於[[民國七十年代|民國七十年代]]初期，當時由鄭靜英老師、徐榮蔘老師和孫純惠老師共同指導學生而成立。從成立迄今三十餘年間，均為該校[[校慶|校慶]]大會及歷屆[[湖口鄉|湖口鄉]]運動會[[開幕式|開幕式]]之重要表演節目，亦曾多次參與[[中華民國國慶日|國慶日]]遊街活動。2013年3月應邀參加[[臺灣燈會|臺灣燈會]]踩街活動，2014年2月參與[[新竹縣|新竹縣]]中小學運動大會開幕表演，2015年10月參加[[2015台灣國際客家文化嘉年華|2015台灣國際客家文化嘉年華]]及2016年10月參加[[2016新竹縣國際花鼓藝術節|2016新竹縣國際花鼓藝術節]]踩街活動。[[鼓舞隊|鼓舞隊]]從創立到目前已歷經多位老師的接棒，主要者如楊淑貞老師、黃世欽老師、呂惠珠老師、林靜玲老師、蘇珮瑛老師、周貴蘭老師、陳寅暉老師、徐鳳君老師、黎士嘉老師……等，現在已發展成信勢國小重要的傳統節目與精神象徵。
*金龍隊
*合唱團
:2009年12月，參加新竹縣九十八學年度學生音樂比賽，榮獲合唱國小B組優等第一名，代表新竹縣參加全國賽。
*童軍團
*管樂團
:'''[[管樂團|管樂團]]'''成立於2006年11月。樂團以社團發展為取向，多利用課餘時間練習。
*桌球隊
*跳繩隊
