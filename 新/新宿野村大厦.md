'''新宿野村大厦'''（{{lang-ja|新宿野村ビルディング}}）是[[日本|日本]][[东京都|东京都]][[新宿区|新宿区]][[西新宿|西新宿]]超高层建筑群中的一栋[[摩天大楼|摩天大楼]]。<ref name="SNB SCP">{{cite web |url=http://skyscraperpage.com/cities/?buildingID=6868 |title=Shinjuku Nomura Building |accessdate= |publisher=Skyscraperpage.com}}</ref><ref name="SNB EMP">{{cite web |url=http://www.emporis.com/en/wm/bu/?id=shinjukunomurabuilding-tokyo-japan |title=Shinjuku Nomura Building |accessdate= |publisher=Emporis.com}}</ref>

== 概要 ==
新宿野村大厦由[[野村不动产|野村不动产]]投资兴建，是一栋白色调与蓝绿色调玻璃外观的建筑，与[[日本损保总部大厦|日本损保总部大厦]]相邻。地下1楼与地下2楼设有餐馆和商店，2楼设有[[野村证券|野村证券]]新宿野村大厦支店，4楼设有自助餐厅，5楼设有诊所，49楼～50楼设有餐厅街，50楼设有免费展望台。

== 主要租户 ==
* 野村不动产株式会社总部
* [[ASKA制药|ASKA制药]]株式会社
* 美国[[美亚保险|美亚保险]] 
* 株式会社[[损害保险日本|损害保险日本]]
* [[日本兴亚损害保险|日本兴亚损害保险]]
等等。（2010年11月）
