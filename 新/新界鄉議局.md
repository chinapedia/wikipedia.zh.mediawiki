'''新界鄉議局'''（官方稱為'''鄉議局'''，英文：'''Heung Yee Kuk'''）是[[香港|香港]][[新界|新界]]700條[[新界原居民|原居民]]鄉村的諮詢及協商的[[香港法定機構|法定組織]]，成立於1926年。現時分為三約及二十七鄉鄉事委員會。

該局的職能，受到《[[香港法例|香港法例]]》第1097章《[[鄉議局條例|鄉議局條例]]》<ref>[http://www.heungyeekuk.org/003a.pdf 鄉議局條例（PDF格式）] {{webarchive|url=https://web.archive.org/web/20070930190935/http://www.heungyeekuk.org/003a.pdf |date=2007-09-30 }}</ref>及[[香港基本法|香港基本法]]第40條<ref>[http://www.heungyeekuk.org/004.htm 香港基本法第40條] {{webarchive|url=https://web.archive.org/web/20070620132753/http://www.heungyeekuk.org/004.htm |date=2007-06-20 }}</ref>所確立。

== 主要職責 ==
* 促進新界居民之間的合作及了解；
* 作為香港政府與新界居民的溝通橋樑；
* 就新界的社會及經濟發展向香港政府提供意見；
* 鼓勵遵守新界居民的傳統風俗和習慣；
* 執行[[香港行政長官|香港行政長官]]邀請的職能。

== 歷史 ==
鄉議局的前身為「新界農工商業研究總會」，由[[荃灣|荃灣]][[鄉紳|鄉紳]][[楊國瑞|楊國瑞]]、[[粉嶺|粉嶺]]鄉紳[[李仲莊|李仲莊]]及[[元朗|元朗]]鄉紳[[鄧偉棠|鄧偉棠]]於1923年發起組織，對政府擬推行之建屋補價政策，提出反對，最終迫使政府終止有關政策。

1926年，當時[[香港總督|香港總督]][[金文泰|金文泰]]為改善[[香港政府|政府]]與原居民的關係，飭令「新界農工商業研究總會」改組為鄉議局，並賦予更多權力，成立初期連一般[[民事|民事]]案件也會交由鄉議局辦理。1959年，[[香港法例|香港法例]]第1097章《鄉議局條例》實施，鄉議局也成為[[香港法定機構|香港法定機構]]之一，並與當時的[[新界民政署|新界民政署]]保持密切聯繫。

雖然隨後[[香港區議會|區議會]]及[[香港區域市政局|區域市政局]]先後成立，作為地區性事務的諮詢及管理議會機構，但鄉議局至今仍然有一定的地位，並分別於香港區議會及[[香港立法會|香港立法會]]擁有當然議席及[[功能界別|功能界別]]議席，參見[[香港選舉|香港選舉]]及[[鄉議局功能界別|鄉議局功能界別]]條目。

== 局徽 ==
[[File:HK_HeungYeeKukBuilding_FrontFacade.JPG|thumb]]
新界鄉議局是在第十六屆第十八次常務會議從多個草圖經過商討後，決定採用現時的局徽設計，但因配色意見不一，延至第十七屆第二次常務會議才通過設計，一直沿用至今。

局徽最外的圓圈表示「新界居民之團結象徵，新界鄉議局是新界民意的高機構」，圓圈內上方有弧形中文[[隸書|隸書]]局名「新界鄉議局」，下方亦有弧形英文局名「HEUNG YEE KUK N.T.」，上方有三個互相重疊成倒「品」字形的小圓圈，右上方小圓圈代表[[鄉議局大埔區|大埔區]]，圈內是該區地形，貫穿其中的是[[九廣鐵路|九廣鐵路]]；左上方的小圓圈代表[[鄉議局元朗區|元朗區]]，圈內是該區地形，[[青山公路|青山公路]]穿越全區<ref>按《新界鄉議局史》第106頁稱為[[屯門公路|屯門公路]]，但於設計局徽時的1960年代尚未建成，相信為手民之誤。</ref>；下方小圓圈代表[[鄉議局南約區|南約區]]，圈內是該區地形，除[[香港島嶼|島嶼]]外，是一片[[海洋|海洋]]，三圈相連「代表新界之三大區之緊密團結，互相依存之象徵」；三個小圓圈之下是繫以紅色綢帶的金色對稱[[稻|稻穗]]，代表新界社會是以[[香港農業|農業]]經濟為主體。

== 局址 ==
[[File:HeungYeeKuk.jpg|thumb]]
[[File:HK_HeungYeeKukNewTerritoriesBuilding.JPG|thumb]]
{{香港政治}}
新界鄉議局前身農工商會於1926年5月3日以9,000港元向商人俞宏波購入[[大埔墟|大埔墟]]祟德街地皮，面積1,980平方呎，並於1926年建成第一代局址，樓高兩層，部份用作祟德高初小學。

直到1970年代，新界鄉議局不斷擴大後地方不敷應用，遂於[[九龍|九龍]][[油麻地|油麻地]]北海街增設辦公室以處理日常行政事務，但不少特別委員會的會議需要到酒樓開會，於第廿一屆成立「籌建大會堂委員會」（其後更名「籌建大廈委員會」），並於1974年7月10日的會員大會通過在[[界限街|界限街]]以北、[[獅子山隧道|沙田隧道]]以南興建新局址的決議。1975年5月委員會選定[[九龍塘|九龍塘]][[金巴倫道|金巴倫道]]面積12,975[[平方呎|平方呎]]地皮作為新局址，為大埔舊局址的六倍，地價連建築費合共200萬港元，由新界鄉議局全資購入，並獲[[新界民政署|新界民政署]]資助5萬元。建築工程分兩期進行，首期用了6個月將地皮上的原有建築物改裝為辦公及會議大樓，並加建門樓；第二期工程於次年完成，在辦公及會議大樓後方新建一座可容300人的大禮堂。

經運作達30年後金巴倫道局址亦逐漸不敷應用，2004年新界鄉議局提議於沙田石門市地段547段（沙田[[城門河|城門河]]畔[[小瀝源|小瀝源]]安睦街地段）興建新局址，[[行政長官會同行政會議|行政長官會同行政會議]]於2005年11月1日通過私人協約方式（Private Treaty Grant），以象徵式1,000港元地價撥出估值約3億元面積約38,167[[平方呎|平方呎]]的原商貿用地皮，用地為期50年，每年收取應課[[差餉|差餉]]租值的3%作為[[地租|地租]]。鄉議局成立籌建新局址專責小組，新局址採用南中國鄉僑大宅作為設計模式，以「維護傳統，走向未來」為主題，樓高三層，樓面面積超過10萬[[平方呎|平方呎]]，內部將有粵劇館、多功能禮堂、展覽館、圖書館、活動室和新界文物館等設施，弘揚華夏文化藝術，維繫新界氏族文化精神。於2006年11月16日舉行動土典禮，2009年1月16日舉行新廈興工及奠基儀式，2010年8月5日舉行上樑典禮，2011年4月2日[[新界鄉議局大樓|新界鄉議局大樓]]舉行入-{伙}-儀式。

新界鄉議局大樓整項建築工程費原本預計約8,300萬港元，但因材料價格上升及提升設計質素後，有關工程超支至1億多元，加上一場金融海嘯，一度令新界鄉議局資金緊絀，曾考慮出售金巴倫道局址，籌措資金興建新大樓。新界鄉議局於2005年2月開始進行進行本地籌款工作，動員1,500名村代表連續四年，每季每人捐出1,000元村代表金資助新大樓的建造費，又四-{出}-舉辦籌款活動，城中富豪、鄉親、團體等大力支持，另得[[香港賽馬會|香港賽馬會]]透過其慈善信託基金撥款近4,200萬港元資助，才使財政壓力大減。新界鄉議局並成立「新局廈籌建委員會海外籌款委員會」，於2005年6月組織訪歐團，向散居[[英國|英國]]、[[荷蘭|荷蘭]]、[[比利時|比利時]]、[[德國|德國]]各地的[[新界原居民|新界原居民]]籌集資金，翌年又邀請「[[嗚芝聲劇團|嗚芝聲劇團]]」在[[倫敦|倫敦]]、[[曼徹斯特|曼徹斯特]]和[[格拉斯哥|格拉斯哥]]三地巡迴演出，並再派出代表團訪歐。經過多方努力，才籌募足夠資金完成新大樓的建築工程。

== 組成 ==
鄉議局由屬下的27個地區[[鄉事委員會|鄉事委員會]]所組成。每個鄉事委員會屬下有多條鄉村，而鄉委會主席由村代表選舉產生。27個鄉委會由分別屬於三約：[[大埔區_(鄉議局)|大埔約]]、[[元朗區_(鄉議局)|元朗約]]及[[南約|南約]]。

現時鄉議局主席是劉業強（屯門區），兩位副主席分別是[[張學明|張學明]]（大埔區）[[林偉強|林偉強]]（南約區）及，三人於2015年改選均在沒有競爭對手下自動當選。

== 教育 ==
1961年鄉議局之執行委員會會議通過設立鄉議局籌建三區中學委員會。經與政府商討後，以鄉議局選舉區為單位，於[[元朗區_(鄉議局)|元朗]]、[[大埔區_(鄉議局)|大埔]]、[[南約區|南約]]三區各建一所中學。當時每所中學之建築費，預算約為港幣200萬元。建校委員會承擔每所百分之三十建築費。1968年[[新界鄉議局元朗區中學|新界鄉議局元朗區中學]]校舍開幕；1982年9月[[新界鄉議局南約區中學|新界鄉議局南約區中學]]落成啟用；1984年5月[[港督|港督]][[尤德|尤德]]爵士為[[新界鄉議局大埔區中學|新界鄉議局大埔區中學]]主持開幕禮。
