'''新字體與簡化字的差異'''指的是日文的新字体和中文的简化字之间的差异。

==中日两种简化间的差异==
下面列出了旧字体、新字体与其对应的汉字对照。<ref>[https://drive.google.com/open?id=0B_v5DhUsRRWgeXFINWRRM1BUTjA&authuser=0  PDF file]. It also served as the source for the lists above. Please note the file is from 2011, so it does not include the 196 new Secondary-School Kanji, but the 5 Kanji removed from the Jōyō-Kanji List.</ref>

在以下列表中，字符按日本汉字的部首排序。旧字体-汉字列表中没有中文对照字的日本国字的{{lang|ja|働}}和{{lang|ja|畑}}未列入其中。

请注意，{{lang|ja|弁}}字在日本被简化前的三种传统字体分别是辨、瓣和-{辯}-。
