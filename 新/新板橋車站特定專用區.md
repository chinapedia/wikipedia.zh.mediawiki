'''新板橋車站特定專用區'''是位於[[台灣|台灣]][[新北市|新北市]][[板橋區_(台灣)|板橋區]]的一個國家重點建設計畫區域，普遍簡稱為「'''新板特區'''」或「'''新站特區'''」，區域內有全臺第一個成型的[[共構與共站|四鐵共構]]（高鐵、臺鐵、捷運、客運）車站、萬坪都會公園及大型廣場，[[新北市政府|新北市政府]]各局處及中央政府轄下的[[中華民國金融監督管理委員會|行政院金融監督管理委員會]]、[[智慧財產法院|智慧財產法院]]、[[交通部鐵道局|交通部鐵道局]]等部會亦進駐於此，區內亦有多家複合式[[購物中心|購物中心]]及商場[[影城|影城]]，未來則有[[大台北新劇院|大台北新劇院]]及會展中心、觀光飯店的規劃。

板橋區被定位為新北市或[[台北都會區|台北都會區]]西側（[[淡水河|淡水河]]西岸）之主要都心城市及交通樞紐，而新板特區則為其商業核心（[[中心商务区|CBD]]），目前仍持續發展中並有範圍擴大之趨勢。

== 發展背景 ==
[[File:新北市政府大樓.jpg|thumb]]
[[File:BanciaoStation.jpg|thumb]]

位於[[府中站|捷運府中站]]附近的[[台鐵|台鐵]]舊板橋車站建於1952年，在尚未有[[台北捷運|台北捷運]]分擔[[台北都會區|台北都會區]]通勤運輸的1990年代之前，其每日進出客量即已高達六萬餘人次，是當時台鐵全線營運量第二多的車站，僅次於[[台北車站|台北車站]]。由於舊板橋車站無法負荷成長中的乘客量，且周邊腹地有限，便於附近原[[臺灣省菸酒公賣局|臺灣省菸酒公賣局]][[板橋酒廠|板橋酒廠]]（靠近[[文化路_(板橋區)|文化路]]）與台鐵的板橋客車場、[[火車調度場|火車調度場]]（今[[縣民大道|縣民大道]]周邊）的範圍，由台北縣政府收購土地，興建新[[板橋車站_(臺灣)|板橋車站]]（同時實施[[鐵路地下化|鐵路地下化]]）。上述範圍面積約19公頃，為新板特區的最早期雛形。在時任台北縣長[[尤清|尤清]]力主及排除許多抗爭的過程下，參酌[[國立臺灣大學|國立臺灣大學]]建築與城鄉研究所對於台北縣城鄉發展的研究結論，將新板特區範圍及規畫升級為現今占地54公頃的淡水河西岸[[中心商業區|都會政經中心]]，並推動原於[[板橋府中商圈|府中]]的原台北縣政府大樓遷移至此。

== 區域範圍 ==
由板橋[[文化路_(板橋區)|文化路]]（[[台三線|台三線]]）一段、[[縣民大道_(新北市)|縣民大道]]二段、中山路（[[縣道114號|縣道114號]]）一段、區運路等四條橫向道路，與漢生東路、新站路、新府路、民權路（[[縣道106號|縣道106號]]）四條縱向道路所圍成的「囲」字形區域，是目前狹義的新板特區範圍，面積約為54公頃。此範圍內有[[新北市政府|新北市政府]]及市民廣場、四鐵共構的[[板橋車站_(臺灣)|板橋車站]]及大樓、[[萬坪都會公園|萬坪都會公園]]、[[大台北新劇院|大臺北新劇院]]（預定地）等政府機關及公共設施，以及[[遠東百貨|遠東百貨FE21']]（板橋店）、[[大遠百|大遠百]]（板橋店）等多家購物中心及影城 ，周邊則有[[板橋高中|板橋高中]]、[[海山高中|海山高中]]、[[板橋國小|板橋國小]]、[[中山國中|中山國中]]等學校及板橋第一和第二體育場區等。

== 近況發展 ==

擁有台灣最先[[共構|四鐵共構]]成型的板橋車站（高鐵、台鐵、[[板南線|捷運板橋線]]、[[環狀線_(台北捷運)|捷運環狀線]]），加上附近有長途客運站及短程公車站，使得新板特區成為新北市最完善的交通樞紐。

車站對面的新北市政府（原臺北縣政府）辦公大樓於2003年1月完工，其前方的「縣民廣場」（今市民廣場）除了提供都會核心區域的大型開闊空間外，也提供大型活動舉辦之場地，至今已舉辦了包括[[金馬獎|金馬奬]]、[[亞太影展|亞太影展]]、新北市冬季系列活動（[[新北市歡樂耶誕城|新北市歡樂耶誕城]]、連結[[台北最HIGH新年城|台北最HIGH新年城]]<ref>市民廣場的液晶螢幕播完整版</ref>、寒假活動、年貨大街、春節活動<ref>小年夜至大年初八</ref>以及元宵燈節活動）、民歌之夜等多元重量級活動。

中山路上的[[遠東百貨|遠東百貨]]、新站路上的[[遠東百貨|大遠百MegaCity購物中心]]及[[威秀影城|威秀影城]]、第一個於車站規劃的複合式百貨[[環球購物中心|環球購物中心]]板橋店、位於縣民大道上[[板信雙子星花園廣場|板信雙子星花園廣場]]大樓內的[[誠品集團|誠品生活]]新板店和板橋[[秀泰影城|秀泰影城]]，是目前新板特區中已營運的主要商場，也因此而使得[[新板特區商圈|新板特區商圈]]成型，並成為新北市最國際化的都會型商圈。

2014年底[[板橋大遠百二期辦公大樓|遠東50層複合商辦大樓]]開始招商營運成為新北市最高的建築地標、未完成招商的[[麗寶百貨廣場|麗寶百貨廣場]]、台電第二營運大樓、曼哈頓金融大樓也將加入營運，形成更為完整的商務辦公圈。

新板特區後方的[[板橋體育場|板橋體育場]]舉辦過各式運動比賽及演場會、大型集會活動（因此也有把此一部分納入新板特區的說法）{{Citation needed}}；文化路旁的追風廣場，原是供民眾溜冰的廣大場地，後來改建為[[萬坪都會公園|萬坪都會公園]]綠化環境<ref>[http://www.epochtimes.com/b5/8/8/4/n2216190.htm 萬坪公園發包 追風廣場暑假後走入歷史].大紀元.2008-08-04</ref>，形成「都會綠色軸線」的概念。

2014年[[宏國關係事業|宏國關係事業]]旗下[[凱撒大飯店|凱撒大飯店]]投資60億元興建天燈國際觀光旅館，預計2017年第3季啟用，屆時將提供地上31樓，地下5樓800間國際觀光旅館及地上20樓，地下5樓337間一般觀光旅館及地上7層樓185桌宴會廳及1棟停車塔，目前已進入完工階段，完工取名新板凱撒大飯店，另外與新板凱撒大飯店共構的「新板[[希爾頓酒店|希爾頓酒店]]」及「[[趣淘漫旅|趣淘漫旅]]」等2間飯店也將同步開幕。<ref>{{Cite news|last=陳志仁|url=https://www.nownews.com/news/20170711/2587667|title=重返台灣市場　希爾頓酒店選擇進駐新板特區|newspaper=NOWnews 今日新聞|date=2017-07-11|accessdate=2017-07-12|language=zh-TW}}</ref><ref>{{cite web |url=http://taipei.caesarpark.com.tw/what-s-new/news-of-caesar-park-taipei.htm |title=台北凱薩大飯店,飯店新聞剪輯 |accessdate=2014-12-01 |deadurl=yes |archiveurl=https://web.archive.org/web/20150104171143/http://taipei.caesarpark.com.tw/what-s-new/news-of-caesar-park-taipei.htm |archivedate=2015-01-04 |df= }}</ref>

== 周邊建設 ==
新板特區被定位成大台北淡水河西岸約250萬人口腹地的都會中心。其建設具體展現在市民廣場、萬坪都會公園等大型都會開放空間、多元的大眾運輸功能如[[三鐵共構|多鐵共構]]車站、富實驗色彩的複式空中及地下通廊、以及新北市政府在內的行政商辦大樓。未來的[[大台北新劇院|大臺北新劇院]]BOT及仍在討論的巨蛋運動園區BOT及數座複合購物中心，將更進一步強化其豐富性。

另外，為了強化其都會中心的功能，除了台鐵、高鐵、[[捷運板南線|捷運板南線]]、[[捷運環狀線|捷運環狀線]]在此交會外，亦於其周邊開闢特一號及特二號兩條快速道路通過，可快速串聯[[國道一號|國道一號]]及[[國道三號|國道三號]]。

== 未來發展 ==
板橋新站特定區仍處開發狀態，目前以陸續完工的高總價住宅進度較快，新北市政府及部分中央機構已進駐，並擁有全臺第一個四鐵共構的車站。正在開發的則有萬坪的都會型公園綠地、國際級[[歌劇院|歌劇院]]（[[大台北新劇院|大台北新劇院]]，[[新十大建設|新十大建設]]計劃）、五星級觀光飯店、捷運[[環狀線|環狀線]]，以及數個財團商業開發案。<br />
