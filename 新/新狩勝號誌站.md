'''新狩勝號誌站'''（{{jpn|j=新狩勝信号場|hg=しんかりかちしんごうじょう|rm=Shin-karikachi shingōjō|f=y}}）位於[[日本|日本]][[北海道|北海道]][[上川郡|上川郡]][[新得町|新得町]]，是[[北海道旅客鐵道|北海道旅客鐵道]]（JR北海道）[[根室本線|根室本線]]與[[石勝線|石勝線]]上的號誌站。

== 車站構造 ==
位於[[新狩勝隧道|新狩勝隧道]]新得側出口處，擁有2線的號誌站。靠近建築物的1號線供下行列車（新得、帶廣方向）使用，2號線供上行列車（瀧川、札幌方面）使用。

== 車站周邊 ==
*[[國道38號|國道38號]]
附近常有例如[[熊|熊]]、[[野豬|野豬]]等的野生動物出沒。

== 历史 ==
*1966年（[[昭和|昭和]]41年）9月30日 - 根室本線[[落合站|落合]]～[[新得站|新得]]間切換為新線時設立。
*1981年（昭和56年）10月1日 - 石勝線開始營業。
*1987年（昭和62年）4月1日 - [[國鐵分割民營化|國鐵分割民營化]]後成為JR北海道轄下的車站。
