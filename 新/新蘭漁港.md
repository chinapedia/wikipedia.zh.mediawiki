'''新蘭漁港'''為一座位於[[臺灣|臺灣]][[臺東縣|臺東縣]][[東河鄉_(臺灣)|東河鄉]]新蘭村的漁[[港|港]]。新蘭漁港距離南方的[[富岡漁港|富岡漁港]]約15[[公里|公里]]，在[[行政院|行政院]][[農業委員會|農業委員會]][[漁業署|漁業署]]新制的漁港法中，新港漁港被歸類在'''第二類漁港'''之中，目前新蘭漁港由[[臺東縣政府|臺東縣政府]]所有。

== 沿革 ==
=== 興建 ===
新蘭漁港周圍海岸淺灘巨石遍佈，無天然隱蔽地形，也無天然[[港灣|港灣]]之地勢，故最初不利於建港。

後[[臺灣省政府|臺灣省政府]]鑒於當地漁民的少數[[漁船|漁船]]及漁筏之作業需要，於1984年到1985年（民國73年到民國74年）共編列預算，新台幣1,200萬元進行新蘭漁港建設工程。漁港建設工程項目包過興建防波堤112公尺、開挖泊地0.2公頃及興建曳船道一座。

=== 擴建 ===
新蘭漁港於第二期漁港擴建方案中為增加港區穩靜度，因此共增建防波堤兼[[碼頭|碼頭]]41公尺、突堤50公尺、護岸120公尺。

== 設施 ==
新蘭漁港港區面積450平方公尺，水深3公尺，碼頭長41公尺，曳船道一座，並有[[行政院|行政院]][[海岸巡防署|海岸巡防署]][[東部地區巡防局|東部地區巡防局]]所設立之安檢所一座，名為'''新蘭安檢所'''，隸屬在'''岸巡八一大隊'''之旗下。<ref>{{cite web |url=http://www.ttcpb.gov.tw/ttckp/home.jsp?mserno=201105180009&serno=201108150001&contlink=ap/police_view.jsp&dataserno=201107250003 |title=臺東縣警察局 成功分局 觀光警政 | accessdate=2015-02-20}}</ref><ref>{{cite web |url=http://www.cga.gov.tw/GipOpen/wSite/ct?xItem=48041&ctNode=6197&mp=9995 |title=行政院海岸巡防署東部地區巡防局 新蘭安檢所 | accessdate=2015-02-20}}</ref>
