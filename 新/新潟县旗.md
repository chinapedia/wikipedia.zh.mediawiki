'''[[新潟县|新潟县]]旗'''（{{jpn|j=新潟県旗|hg=にいがたけんき|rm=Nīgata ken ki|f=}}）是日本的47面[[都道府县旗|日本都道府县旗]]之一。该条目是对'''新潟县旗'''以及'''新潟县章'''（{{jpn|j=新潟県章|hg=にいがたけんしょう|rm=|f=}}）的解说。

== 概要 ==
县章、县旗在1963年8月23日县公告上通过制定<ref>{{Cite web |url=http://www1.g-reiki.net/niigataken/reiki_honbun/ae40100031.html |title=県花、県章及び県旗 |access-date=2013-04-05 |archive-url=https://web.archive.org/web/20150102110440/http://www1.g-reiki.net/niigataken/reiki_honbun/ae40100031.html |archive-date=2015-01-02 |dead-url=yes }}</ref>。[[间隔字符|间隔字符]]写成的[[汉字|汉字]]“[[新|新]]”在圆的上部，[[片假名|片假名]]“[[ガ|ガ]]”“[[タ|タ]]”都呈半圆状。这样的设计代表了融合与期望，县发展前途的平坦。
县章部分在县旗上是[[金|金]]色的，有时候使用[[白色|白色]]的。
== 徽记 ==
[[文件:Symbol_mark_of_Niigata_Prefecture.svg|thumb]]
1992年3月27日县公告通过制定<ref>{{Cite web |url=http://www1.g-reiki.net/niigataken/reiki_honbun/e4010004001.html |title=新潟県のシンボルマークの制定 |access-date=2013-04-05 |archive-url=https://web.archive.org/web/20150102110656/http://www1.g-reiki.net/niigataken/reiki_honbun/e4010004001.html |archive-date=2015-01-02 |dead-url=yes }}</ref>。县土环[[日本海|日本海]]的中心位置变形为[[扇形|扇形]]的设计，青色的圆象征自然、滋润、无限发展可能性，[[扇形|扇形]]代表县的文化与信息途径的增多，以及国际性、积极性。近几年，县章被徽记代替的机会越来越多。[[东京都|东京都]]、[[鹿儿岛|鹿儿岛]]尚未制定徽记旗。
