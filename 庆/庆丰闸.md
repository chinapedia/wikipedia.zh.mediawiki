'''庆丰闸'''，位于[[北京市|北京市]][[朝阳区|朝阳区]][[双井街道|双井街道]]，是[[通惠河|通惠河]]上的一座水闸。<ref name=zgfj>[http://www.chla.com.cn/htm/2011/1227/109894.html 北京庆丰公园欣赏，中国风景园林网，2011-12-27]</ref><ref name=bqw>[http://bjyouth.ynet.com/3.1/1202/15/6788123.html 桥闸层层入帝京锦绣画卷满惠通，北青网，2012-02-15] {{webarchive|url=https://archive.is/20130917074105/http://bjyouth.ynet.com/3.1/1202/15/6788123.html |date=2013-09-17 }}</ref>

==简介==
庆丰闸位于[[东便门|东便门]]以东2.7公里的通惠河上。1292年，[[元朝|元朝]][[都水监|都水监]][[郭守敬|郭守敬]]主持开凿通惠河时所建。初名籍东闸，俗称二闸（因在上游[[广源闸|广源闸]]之下），起初为上下两座木闸。[[元贞|元贞]]元年（1295年）更名为庆丰闸。[[至顺|至顺]]元年（1330年）因岁久木朽，改建为石闸，闸基长12丈（36.39米），金门（闸室）宽2丈2尺（6.6米）。明朝嘉靖吴仲重修通惠河时，下闸废弃。所以现在的庆丰闸是元朝的庆丰上闸。<ref name=gong>{{Cite web |url = http://news.xinhuanet.com/travel/2012-11/02/c_123904924.htm |title =  “京畿秦淮”庆丰公园|author =  |date = 2012-11-02 |publisher = 新华网 |language =  |accessdate =  |quote =  }}</ref><ref name=wangshk/>

[[明朝|明朝]][[嘉靖|嘉靖]]六年（1527年），巡仓御史[[吴仲|吴仲]]力排众议，奏请修浚通惠河，获[[嘉靖帝|嘉靖帝]]批准。嘉靖七年（1528年），由吴仲亲自设计指挥，当年春季开工并于同年六月完成了通惠河修浚工程。<ref name=hujx>胡吉勋，通惠河的修浚及争议——兼论明代朝廷决策中的利益权衡，古代文明2015(2):74-80</ref>经过此次修浚，在东便门至北运河段的通惠河上改建成五座船闸，即头闸[[大通闸|大通闸]]、二闸庆丰闸、三闸[[平津闸|平津上闸]]（高碑店闸）、四闸[[平津闸|平津下闸]]（花园闸）、五闸[[普济闸|普济闸]]。通常人们仅称前三闸为一闸、二闸、三闸，不称四闸和五闸。通惠河通过这五座船闸，提升航道水位，解决漕船逆行入北京城的困难。<ref name=bqw/>

从[[平津闸|平津闸]]（高碑店闸）向西十五里便是庆丰闸。庆丰闸是从东便门到通州的通惠河上的二闸，素来以风景优美而著称。[[清朝|清朝]][[曹雪芹|曹雪芹]]喜爱庆丰闸的水色风景，曾多次来庆丰闸游玩，并在庆丰闸畔的酒楼饮酒作诗。曹雪芹逝世后，其好友[[敦敏|敦敏]]来到庆丰闸，写下《河干集饮题壁兼吊雪芹》诗：“花明两岸柳霏微，到眼风光春欲归。逝水不留诗客杳，登楼空忆酒徒非。河干万木飘残雪，村落千家带远晖。凭吊无端频怅望，寒林萧寺暮鸦飞。”[[中华民国|中华民国]]时期，文人雅士们来到庆丰闸游玩，将此处比作北方的[[秦淮河|秦淮河]]。在中华民国时期庆丰闸的老照片上，庆丰闸的清雅胜于[[秦淮河|秦淮河]]。<ref name=bqw/>

庆丰闸原来有十三块闸板，后来仅存一块闸板，年久却不朽，当地人称“万年青”。闸口石砌基址呈燕尾八字形，闸口两岸是码头，其中北岸是杂货码头，南岸是皇粮码头。闸口以西的通惠河沿岸是往来船只停泊之所。北岸原来有一座龙王庙，庙前[[舞獅|舞狮子]]、[[放河灯|放河灯]]时非常热闹。<ref name=bqw/>

清朝[[光绪|光绪]]二十六年（1900年），通惠河因水源缺乏而停止[[漕运|漕运]]，沿线各闸废弃。<ref name=wangshk/>[[中华人民共和国|中华人民共和国]]成立后，政府多次整修通惠河。<ref name=gong/>1992年2月，[[北京市人民政府|北京市人民政府]]批准成立通惠河整治工程指挥部，再次整治通惠河，要求“要以水为中心进行综合整治”，以便于泄洪、改善水质。此次通惠河整治工程中，开展了“庆丰闸遗址保护工程”，由北京市水利规划设计研究院进行工程设计，对庆丰闸的文物进行了保护。庆丰闸遗址保护工程于1995年3月落成，该工程的主要内容有：<ref name=wangshk>王世宽，通惠河庆丰闸的发掘及保护，北京水利学会水利史研究会20周年纪念会暨水文化学术研讨会，2007年，第32-34页</ref>

*（一）保护庆丰古闸旧貌：原庆丰闸口门（金门）宽6.6米，但在通惠河整治工程中，通惠河河道拓宽至40米。为保留原闸旧貌，在河底原闸位处，将原闸形状及材料固定在低于河底0.2米处。水清时，从岸边和新建的庆丰桥上可以看见水中的庆丰古闸。<ref name=wangshk/>
*（二）闸身保护：在通惠河整治工程中，设计边墙为混凝土结构。为保护水上部分闸边墙旧貌，沿着闸基边墙，采用原闸墙花岗条石，尽量按原样恢复，长度仍为原长度36.39米。<ref name=wangshk/>
*（三）恢复边墙上的龙马石壁雕：原闸左右岸边墙上镶嵌有一块长3.5米，宽0.6米，厚0.4米用青石雕成的巨龙（青龙）和四匹飞奔的骏马（泗马），相传是“龙马负图”的神话故事，即“[[河图洛书|河出图洛出书]]”。施工中将原龙马石壁雕移走保存，将仿制的壁雕放在原位置。<ref name=wangshk/>
*（四）恢复上下游石雕镇水兽：原闸边墙上下游，左右两岸安放4只石雕镇水兽。每只兽梗颈扭头，披鳞带角，长尾回蜷，怒视河水。施工中将原镇水兽移走保存，将仿制的镇水兽放在原位置。<ref name=wangshk/>
*（五）复制《二闸修契》壁画：在庆丰闸左侧位置，河北岸的巡河路挡墙上，建成一座墨玉石刻艺术壁画，长27米，高3米。壁画顶上是[[汉白玉|汉白玉]]石雕屋脊，屋脊两端有石雕鸱吻。壁画刻在墨玉石板上，用[[四川|四川]]红天然大理石镶边。壁画分为三部分，第一部分是“庆丰闸遗址”题字（由原水利部部长[[张含英|张含英]]题写）；第二部分是《鸿雪因缘图》一书中清朝官员[[穆见亭|穆见亭]]画的《二闸[[修禊节|修禊]]》图，描绘了[[清朝|清朝]]河道总督[[麟庆|麟庆]]在1820年春（农历三月初三日）邀请16位友人来二闸（庆丰闸）祭礼、乘船、流杯饮酒、吟诗作画春禊的场面；第三部分罗列了从1292年郭守敬主持建通惠河起，明清时期修建庆丰闸的历史，包括明朝嘉靖七年（1528年），吴仲修浚通惠河，[[嘉靖帝|嘉靖帝]]乘船视察通惠河；同年秋，曾陪嘉靖帝视察通惠河的大臣[[张璁|张璁]]、[[杨一清|杨一清]]、[[翟銮|翟銮]]再度游览通惠河并作《侍上泛通惠河同官联句》：<ref name=wangshk/><ref>《[[钦定日下旧闻考|钦定日下旧闻考]]》卷八十九《郊垧东二》记载：“原：嘉靖丁亥，御史吴仲请修通惠河，三月告成功，上登舟观之，大学士张璁等聨句以闻，上喜，给光禄馔，又分御膳赐焉。（《帝京景物略》）”</ref>

*（六）庆丰闸遗址碑：在壁画西侧，立有一通庆丰闸遗址碑。高4米，宽1.6米，厚0.4米，大青石制，[[螭首|螭首]]（四龙戏水）方[[须弥座|须弥座]]，碑额阳面[[篆书|篆书]]“庆丰闸遗址碑”，碑身阳面刻有《庆丰闸遗址碑记》，记载通惠河整治工程、庆丰闸遗址保护工程建成的历史。碑文为原[[北京市人大常委会|北京市人大常委会]]副秘书长[[段天顺|段天顺]]撰文：<ref name=bqw/>

{{引用|庆丰闸遗址碑记：

庆丰闸，初名籍东闸，俗称二闸。公元1392年（元至元29年）都水监郭守敬主持开凿通惠河时所建。初为上下两座木闸，后于公元1330年（元至顺元年）改建石闸。闸基长120尺，金宽22尺，并易名庆丰。明嘉靖中吴仲重修通惠河时，下闸废弃。仅存庆丰闸遗迹当时元代庆丰上闸。通惠河系京杭运河首段，庆丰闸仍是通惠河上的重要通航船闸，以时畜泄，因水转漕，舳舻相衔，功莫大焉。又以地处近郊，碧波清流，风光秀丽，明清以来皆为市民游憩胜地，素有燕京秦淮河之美誉。岁月流逝，原闸废弃，遗址尚存。

中华人民共和国成立后，北京人民政府多次整修通惠河。自1992年底再次兴工，拓宽河道，以利泄洪，改善水质，复现清波。为保存文物，光耀先祖贤，遂将原闸石雕易地保存，闸底略有降低，并仿元制重石兽，修建虹桥，爰于原址立碑为记。

1992年至1995年，在通惠河整治工程中开展“庆丰闸遗址保护工程”的同时，在庆丰闸原址下游（即庆丰闸原址东侧）不远处新建[[汉白玉|汉白玉]]拱桥一座，名“庆丰桥”（张含英题写桥名），供行人过河使用。高7.8米，长38米，宽4.4米。该桥的拱形结构、栏杆、板柱均采元代建筑式样。上述“庆丰闸遗址保护工程”各项内容均位于庆丰桥西侧。<ref name=bqw/>

2009年，庆丰闸旧址以南的通惠河南岸建成[[庆丰公园|庆丰公园]]并对外开放。从[[国贸桥|国贸桥]]向南，便可到达通惠河畔的庆丰公园。通惠河北岸是[[北京CBD|北京CBD]]，高楼林立。通惠河在庆丰闸处，两岸原来都是街巷与村落，为配合[[北京CBD|北京CBD]]的建设，将街巷和村落拆迁，南岸辟为庆丰公园。<ref name=bqw/>从2009年8月拆迁庆丰小区起，仅用1个月，便于2009年9月28日建成了庆丰公园。该公园因庆丰闸而得名。庆丰公园位于通惠河南岸，横跨[[北京三环路|北京东三环]]，南邻北京铁路机务段，北隔通惠河与[[建外SOHO|建外SOHO]]、[[中国航空工业第一集团公司|中国一航]]、[[北京电视台|北京电视台]]新址遥对。庆丰公园自东至西长1700米，占地面积26.7公顷。<ref name=zgfj/>

2013年，庆丰闸作为[[大运河|大运河]]北京段的一部分，被并入[[全国重点文物保护单位|全国重点文物保护单位]][[大运河|大运河]]内。<ref name=xinzeng>[http://www.bjxch.gov.cn/XCHxxgk/XCHzhengwuxiangqing.ycs?GUID=540638 我区新增十处全国重点文物保护单位概况，北京西城报，2013-07-05] {{Wayback|url=http://www.bjxch.gov.cn/XCHxxgk/XCHzhengwuxiangqing.ycs?GUID=540638 |date=20160304070959 }}</ref>
