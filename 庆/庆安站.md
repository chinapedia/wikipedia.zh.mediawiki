'''庆安站'''位于[[中华人民共和国|中华人民共和国]][[黑龙江省|黑龙江省]][[绥化市|绥化市]][[庆安县|庆安县]]庆安镇，建于1937年，现为[[四等站_(中鐵)|四等站]]。客运办理旅客乘降、行李及包裹托运；货运办理整车及零担货物发到<ref>{{cite book |title=中国铁路站名词典 |author=孙本祥 编 |publisher=中国铁道出版社 |year=2003 |isbn=9787113039417 |pages=393 }}</ref>。

==事件==
{{main|庆安火车站枪击事件}}
2015年5月2日，一名中年男子徐纯合其在[[上访|上访]]途中，因庆安县火车站不允许其上火车，因而和车站执勤民警发生冲突，并抢夺警械，后警察开枪将其击毙。<ref>{{Cite web |url = http://news.hexun.com/2015-05-06/175562635.html |title = 黑龙江庆安枪击案 |author =  |date = 2015-05-06 |publisher = [[和讯网|和讯网]] |language = zh-hans |accessdate = 2015-05-14|quote = }}</ref><ref>{{Cite web|url =http://www.dqdaily.com/shishi/2015-05/09/content_2661924.htm|title =媒体三问黑龙江庆安枪案：是否因违法截访引发|date =2015-05-09|publisher =大庆网|language =zh-hans|accessdate =2015-05-14|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20150518095223/http://www.dqdaily.com/shishi/2015-05/09/content_2661924.htm|archivedate =2015-05-18}}</ref>
