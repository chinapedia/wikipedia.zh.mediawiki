'''蔣光鼐'''（{{bd|1888年|12月17日|1967年|6月8日|catIdx=Jiang蔣}}）名憬然，[[廣東|廣東]][[東莞|東莞]]太平（今廣東省東莞市[[虎門鎮|虎門鎮]]南栅三蔣村） 。1913年畢業於[[保定軍官學校|保定軍官學校]]第一期騎兵科。曾任[[國民革命軍|國民革命軍]]師長，第十一軍副軍長，十九路军总指挥。他所統領的[[十九路軍|十九路軍]]，亦為1934年[[閩變|閩變]]主力。1946年，蔣光鼐參與發起組織[[中國國民黨民主促進會|中國國民黨民主促進會]]。[[中华人民共和国|中华人民共和国]]成立后，在[[中央人民政府|中央人民政府]]等任职。1966年[[文革|文革]]期間遭抄家等迫害，翌年去世。

== 粵軍 ==
'''蔣光鼐'''出身於[[粵軍第一師|粵軍第一師]]。1926年粵軍第一師改編為[[國民革命軍第四軍|國民革命軍第四軍]]，[[李濟深|李濟深]]為軍長，[[陳銘樞|陳銘樞]]為當中第十師的師長，蔣光鼐則擔任副師長。

== 北伐 ==
[[北伐|北伐]]中第四軍屢有戰果，擴編為第十一軍。後陳銘樞離開，第十一軍交由[[張發奎|張發奎]]指揮。中共在南昌發動[[南昌起義|兵變]]，第十一軍改投向[[蔣介石|蔣介石]]陣營，參與[[中原大戰|中原大戰]]，擊敗[[馮玉祥|馮玉祥]]及[[閻錫山|閻錫山]]後擴編為[[十九路軍|十九路軍]]，蔣光鼐為總指揮，[[蔡廷鍇|蔡廷鍇]]為軍長。之後[[十九路軍|十九路軍]]亦參加了第三次圍剿紅軍。

== 抗日 ==
1931年[[九一八事變|九一八事變]]後，[[十九路軍|十九路軍]]調防[[上海|上海]]，同時蔣介石受壓下野，南京政府由廣東派系主持，陳銘樞任京滬戍衛司令。1932年[[一二八事變|一二八事變]]，[[日本|日本]]派陸戰隊登陸上海。[[十九路軍|十九路軍]]奮起迎戰抵抗，因而成為聲名大噪的抗日軍隊。
任總指揮的蔣光鼐也獲頒[[青天白日勳章|青天白日勳章]]。後來日本多番增援；蔣介石復出後，國軍亦派出中央軍共同作戰，最後事件以外交談判解決，第十九路軍從上海撤下，並被調到福建剿共，蔣光鼐同時兼任[[福建省|福建省]]主席。過程中，主和派[[蔡廷鍇|蔡廷鍇]]等將領在前線與[[中共|中共]]展開和談，並取得停火。

== 閩變 ==
1933年11月20日，[[閩變|閩變]]正式爆發，在[[福州|福州]]的「[[生產人民黨|生產人民黨]]」及「[[第三黨|第三黨]]」舉行「中國全國人民臨時代表大會」，決議成立[[中華共和國人民革命政府|中華共和國人民革命政府]]，更改國旗為上紅下藍，中嵌黃色五角星，同時廢止[[國民黨|國民黨]]。並推[[李濟深|李濟深]]、[[陳銘樞|陳銘樞]]、[[陳友仁|陳友仁]]、蔣光鼐、[[蔡廷鍇|蔡廷鍇]]、[[黃琪翔|黃琪翔]]等為政府委員。之後即與[[江西|江西]]中共[[中央蘇區|中央蘇區]]之「[[中華蘇維埃共和國|中華蘇維埃共和國]]臨時中央政府」簽署協定，停止軍事對抗和開始經濟合作。22日「中華共和國人民革命政府」宣佈成立，以李濟深為政府主席，陳友仁為外交部長，改年號為「[[中華共和國|中華共和國]]」元年。

之後，[[蔣介石|蔣介石]]調八個師的陸軍入閩，並以空軍、海軍配合攻擊。[[十九路軍|十九路軍]]迅速潰散。蔣光鼐連同[[蔡廷鍇|蔡廷鍇]]、[[陳銘樞|陳銘樞]]、[[李濟深|李濟深]]等人則逃至[[香港|香港]]。[[十九路軍|十九路軍]]的番號取消，部隊亦被分散收編。

== 民主党派人士 ==
1945年，[[中國抗日戰爭|抗戰]]勝利後，他從香港返回中國大陆，任國民黨第七戰區副司令長官，1946年參與發起組織[[中國民主促進會|中國民主促進會]]，支持[[中國共產黨|中國共產黨]]。1949年，出任[[中華人民共和國|中華人民共和國]]的政協委員，後又歷任中國[[紡織工業部|紡織工業部]]部長，[[全國政協|全國政協]]常委，[[中国国民党革命委员会|中国国民党革命委员会]]中央常务委员等職務。

== 文革迫害 ==
1966年，蒋光鼐被确诊为癌症，从上海请来专家会诊并动了手术。「[[文化大革命|文化大革命]]」發動後，居住於北京的蔣光鼐連同多位黨外民主人士被[[紅衛兵|紅衛兵]]點名遭到整肅。期間，雖[[周恩來|周恩來]]力保，仍遭抄家，自我認罪等迫害。

1967年癌症复发，6月8日逝於[[北京|北京]]。

== 评价 ==
* 周恩来曾致函给蒋光鼐，尊敬地称道：“先生以抗日前导而为华南和平民主之支柱，力挽狂澜，举国瞩望。”
* 1988年12月17日，在纪念蒋光鼐诞辰100周年的大会上，[[中共中央|中共中央]]对蒋光鼐的一生作了高度评价：“蒋光鼐先生是[[中国国民党革命委员会|中国国民党革命委员会]]的一位卓越的创始人和领导人，是同中国共产党长期合作的亲密朋友，他把自己毕生精力献给了中国民主革命和社会主义事业。他的爱国精神和历史功绩，坚定不移的政治节操，严于律己、宽以待人的品德，永远值得我们学习和纪念。”
