'''蔣祖曼'''（{{lang|en|'''Joman Chiang'''}}，{{bd|1985年|5月26日||}}），在[[香港|香港]]出生，[[香港|香港]]電影女演員，被稱為「獨立電影女皇」<ref>[http://hk.face.nextmedia.com/article/318/16971873 告別獨立　蔣祖曼]FACE周刊 第318期</ref><ref>[http://hd.stheadline.com/arts/arts_content.asp?contid=112055&srctype=g 蔣祖曼　尋突破] [[星島日報|星島日報]]</ref><ref>[http://the-sun.on.cc/cnt/lifestyle/20111117/00485_001.html 蔣祖曼　一門10貓]星島日報</ref>。

== 簡介 ==
蔣祖曼畢業於[[東華三院港九電器商聯會小學|東華三院港九電器商聯會小學]]以及[[田家炳中學_(香港)|田家炳中學]]。她曾參與大量香港獨立電影拍攝，演出屢獲好評，代表作品有《[[蝴蝶_(2004年電影)|蝴蝶]]》、《[[三條窄路|三條窄路]]》、《[[霧_(電影)|霧]]》、《[[圍城_(電影)|圍城]]》、《[[戀人路上|戀人路上]]》、《[[酒徒_(電影)|酒徒]]》、《[[大藍湖|大藍湖]]》等，作品曾赴[[威尼斯國際電影節|威尼斯國際電影節]]、[[釜山國際電影節|釜山國際電影節]]、[[東京電影節|東京電影節]]、[[瑞典斯德哥摩電影節|瑞典斯德哥摩電影節]]、[[愛丁堡國際電影節|愛丁堡國際電影節]]、阿姆斯特丹CinemAsia、挪威國際電影節等參展；此外蔣祖曼在商業電影的表現同樣出色，曾參與電影有《[[古惑仔：江湖新秩序|古惑仔：江湖新秩序]]》、《[[竊聽風雲2|竊聽風雲2]]》、《[[龍門飛甲|龍門飛甲]]》等，2013年更有首部英語電影《[[:en:Lost_for_Words_(2013_film)|愛·真·承]]》上映，向國際影壇邁出漂亮的一步。

蔣祖曼近年足迹更遍及舞台劇界，分別在2011年憑《Reason to be pretty》獲劇協最佳女主角提名、及2013年憑《[[紅海人。藍海戰|紅海人。藍海戰]]》再度獲劇協最佳女配角提名。

== 家庭 ==
蔣祖曼和任職電影幕後的陳卓麒於2014年1月3日註冊結婚，取其日子諧音愛你一世一生。<ref>[http://hk.apple.nextmedia.com/entertainment/art/20140105/18579255 周俊偉賀蔣祖曼結婚] [[蘋果日報_(香港)|蘋果日報 (香港)]] 2014年1月5日</ref>

2014年12月，蔣祖曼宣佈懷孕三個多月。<ref>[http://hk.apple.nextmedia.com/entertainment/art/20141204/18956691 《警界線》炸差館 親情大催淚][[蘋果日報_(香港)|蘋果日報 (香港)]] 2014-12-04</ref>2015年5月誕下女兒。<ref>[http://hk.on.cc/hk/bkn/cnt/entertainment/20150524/bkn-20150524110948695-0524_00862_001.html 恭喜！蔣祖曼順利誕下B女] 2015年5月24日 [[東方日報_(香港)|東方日報 (香港)]]</ref>
