'''蔣公遺德顯彰會'''，[[臺灣|臺灣]]稱為「先總統蔣公遺德顯彰會」<ref>[http://www.mofa.gov.tw/News_Content.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=F1FB5D609E86C2AC 總統府蔣彥士秘書長偕同亞東關係協會馬紀壯會長及中國國民黨中央委員會]，中華民國外交部</ref>，是[[日本|日本]]為紀念[[蔣介石|蔣介石]]在[[日本投降|日本投降]]後對日本寬厚對待而成立的協會。

==協會背景==
1985年10月25日，蔣經國函電宋美齡：
{{quote|
「……日本發起之父親遺德彰顯會為明年九至十月在[[東京|東京]][[北海道|北海道]]等地集會並辦史蹟展覽此事共匪雖正對日橫加壓力迫使發起人名單略有變動但據稱仍將積極進行大人所示洞見日人肺腑謹當密切注意……兒經國跪稟十月二十五日」<ref name="與美電">[[周美華|周美華]]、[[蕭李居|蕭李居]] 編：《蔣經國書信集——與宋美齡往來函電》下，[[國史館|國史館]]出版，2009年。</ref>{{rp|605}}
}}

1986年9月，為紀念蔣介石百歲誕辰，日本一批政治[[元老|元老]]和工商巨頭，由前[[日本內閣總理大臣|日本內閣總理大臣]][[岸信介|岸信介]]牽頭，在日本[[東京|東京]]發起成立。初創成員高達6000餘人，包括岸信介、[[灘尾弘吉|灘尾弘吉]]等人。9月6日，蔣經國函電宋美齡：
{{quote|
「……此次日本朝野舉行之遺德彰顯會日本各界——包括其[[國會|國會]]議員一百三十餘人在內自覺到者高達六千五百餘人據云為歷來中日關係最盛大之一次集會岸信介灘尾泓吉鹿內佐藤等名流共同主持致詞祝酒並奏英雄樂章同時同地展出父親衣冠文獻所有與會者皆獲得『近百年來中日關係圖錄』及『三民主義建設成果』各一大冊母親電文亦及時傳真到達立即精印中日文本在會場宣讀分發反應至為熱烈灘尾致詞且謂此舉不止在紀念蔣公恩德目的尤在教育下一代懷永不忘而孔德成院長致詞亦極為得體匪對日人此舉曾不斷力圖阻止不謂甶而激使日人憤慨終於造成此盛況空前之局面令匪沮喪失色此實母親仁心遠見所發生之歷久不磨之影響謹電稟報並懇母親順時珍護肅叩福安兒」<ref name="與美電" />{{rp|711-712}}
}}
9月8日，宋美齡函電蔣經國：
{{quote|
「……老一輩日人對遺德顯彰會熱忱擁戴可慮者日少青年代代輩出對我中國人民寬恕教養及父親仁慈為懷之胸襟已漶然淡忘此類提醒工作我駐外單位宜積極隨時贊襄藉開外交困境……母九月八日」<ref name="與美電" />{{rp|716}}
}}
9月11日，蔣經國函電宋美齡：
{{quote|
「……日本自東京遺德彰顯會後據云尚有[[奈良|奈良]][[名古屋|名古屋]]等處繼起將予共匪以更大之難堪[[韓國|韓]]亦將有紀念會活動有關方面已飭全力協助進行……兒經國跪稟九月十一日」<ref name="與美電" />{{rp|716-717}}}}

10月下旬，遺德彰顯會在日本名古屋舉行首屆紀念大會，名人政要雲集，場面頗為壯觀<ref name="C19861021">{{cite web |url = http://dava.ncl.edu.tw//MetadataInfo.aspx?funtype=0&id=193565 |title=日本名古屋紀念蔣公百年誕辰舉行蔣公遺德顯彰紀念大會 |publisher=[[臺視新聞|臺視新聞]]  |date=1986年10月21日 |accessdate=2013年10月8日 |language=zh-hant}}</ref>。而岸信介更在「蔣公遺德顯彰會」蔣介石畫像前，高舉酒杯祝「中日兩國友誼永固」<ref name="C0001534820">{{cite web|url=http://nrch.cca.gov.tw/ccahome/search/search_meta.jsp?xml_id=0001534820&dofile=cca220001-hp-cdn0030193-0001-i.jpg|title=故總統蔣公與日本來華人士|publisher=國家文化資-{}-料庫|year=1986-09-05|accessdate=2013年10月8日|language=zh-hant|deadurl=yes|archiveurl=https://web.archive.org/web/20130928114943/http://nrch.cca.gov.tw/ccahome/search/search_meta.jsp?xml_id=0001534820&dofile=cca220001-hp-cdn0030193-0001-i.jpg|archivedate=2013年9月28日}}</ref>。

協會發起人岸信介在[[日本政治|日本政壇]]背景深厚，除本人是日本第56、57任首相外，其外孫[[安倍晉三|安倍晉三]]為日本第90、96任首相；其弟[[佐藤榮作|佐藤榮作]]為日本第61、62、63任首相、[[諾貝爾和平獎|諾貝爾和平獎]]得主。1975年蔣中正在[[台北|台北]]逝世之時，岸信介親赴台北奠蔣，佐藤榮作同行。

協會感念蔣介石對日恩澤。蔣介石在日本戰敗後，以「不念舊惡、以德報怨」為對日政策宗旨，對日本施行寬大處理，包括：
* 建言[[美國|美國]]保留日本傳統[[天皇|天皇]]制
* 反對[[蘇聯|蘇聯]]戰後佔領[[日本列島|日本四島]]，而支持美國單獨佔領
* 日本滯留中國的[[軍隊|軍隊]]、[[戰俘|戰俘]]，悉數遣返日本
* 日本居住中國的僑民，悉數遣返日本
* 減除日本戰爭賠款
