'''蔣卓嘉'''（{{lang-en|'''GJ'''}}，{{bd|1984年|10月1日|}}），詞曲創作歌手、音樂製作人、吉他手、HitFm聯播網[[台北之音|台北之音]]電台DJ。

== 经历 ==

=== 出道前 ===
中國[[上海市|上海市]]出生，九歲移民至[[澳洲|澳洲]]雪梨生活，因為讀男校有很多同學都叫George（當時的英文名字），每當有人叫George都有10人一起回頭，為了分辨，朋友都叫他GJ（英文名字縮寫）。初中時候因為叛逆逃學而受吉他導師父親關在房間啟蒙天賦學習結他，從14歲起加入了學校的爵士樂隊、吉他社、西班牙樂隊等等，並開始在校外表演。16歲創建第一個樂隊( Kyanite )擔任吉他手以及主唱，編寫的第一首個人原創作品獲得校外樂隊比賽的第一名。

2001年，獲得雪梨教會中學校園樂隊比賽冠軍。也就太沉迷於音樂和練習吉他，廢寢忘餐一天練習八小時，父親後來擔心他的學業追不上進度而不能升讀大學沒收他的吉他，GJ本人認為很諷刺，當初給他好好學的樂器，喜歡上了又要拿掉，所以在高中時候會偷偷走進音樂大樓的課室練習，當時老師知道他的苦心也同意簽署讓他可以使用音樂室，GJ認為真心真意去想做好一件事的時候，全世界的人都會盡力幫助，直至大學考上了澳洲音樂學院的他拿著入學通知書回家，父親才把吉他還給他。 後來GJ也有提到感謝父親當時沒收吉他的決定，也讓他嘗到失去的滋味，學會了珍惜。

GJ 17歲就讀[[澳洲音樂學院|澳洲音樂學院]]，現代音樂系，主修吉他、副修鼓及其他樂器，另外還輔修了音控課程（Audio Engineering），並組創第2個樂隊（2051）擔任主音吉他手，同樣也得到了樂隊比賽（kick up the dust）的第一名及最佳吉他手獎。

澳洲雪梨組過日本樂團Level 28擔任吉他手，團員們都是他的學長和學姐，也在許多live house演出。因為當時在讀音樂學院時經好友推薦下把demo寄給製作人Jim Lee，後來也成為了動畫電影《[[風雲決|風雲決]]》主題曲，由[[任賢齊|任賢齊]]主唱，大學畢業後一心想從事華語音樂創作的GJ自覺在澳洲雪梨跟臺灣距離實在太遠，不顧身邊的人反對下為了追求自己的夢想和努力實踐。2009年4月26日，下定決心買了單程機票獨自到臺灣發展，在首張創作專輯《十面》裡收錄的《我的夢》也就是他的寫實故事。

=== 2013年：首張專輯《十面》 ===
在臺灣短短幾年內，GJ幫許多位中港臺歌手寫歌和製作，獲得了許多經驗，也擁有了很豐富的履歷。後來更得到香港金牌經紀人黃柏高賞識簽約[[太陽娛樂文化|太陽娛樂文化]]成為歌手，於2013年10月30日發行首張創作專輯《[[十面|十面]]》<ref>{{Cite web |url=http://weifang.dzwww.com/yl/201310/t20131024_9059853.htm |title=蒋卓嘉新专辑将发行 获王力宏制作人护航 |author=蒋卓嘉 |date=2013-10-24 |publisher=大众网·潍坊 |language= |accessdate=2014-01-07 |quote= }} </ref>。

=== 2016年 ===
1月11日起，正式成為Hit Fm台北之音（FM 107.7）電台DJ，主持時段為每週一至週五、中午12點到下午2點的「翹班DJ」<ref>{{Cite web |url=http://www.hitoradio.com/newweb/76926hitonews |title=GJ蔣卓嘉暖男歌手搖身變Hit Fm電台DJ　擔心開天窗拒當夜貓族 |author=Hit Fm聯播網 |date=2016-01-10 |publisher=Hit Fm聯播網 |language= |accessdate=2016-01-14 |quote= }} </ref>。
