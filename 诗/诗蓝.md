'''诗蓝'''（{{lang-fr|'''Li Chevalier'''}}，{{bd|1961年|3月30日|}}）当代旅法艺术家。在巴黎和北京两工作室创作。欧亚艺术平台上最活跃的华裔女艺术家之一。知名为东西方艺术元素混合的布面实验水墨绘画，和多媒介装置，展览空间构造和舞美设计。 毕业于英国中央圣马丁美术与设计学院美术系油画专业。

留法期间曾师从于法国艺术联盟主席艾融Remy Aron和法国国立艺术家协会画家主任Pierre Henry。 1990年获法国索帮大学哲学硕士学位 。国际造型艺术联盟法国执委委员，法国国立美协与泰勒艺术基金会会员，法国视觉艺术家与作家协会会员，法国艺术联盟成员。

主要个展: 意大利罗马当代美术馆 2017  中国美术馆2010，今日美术馆2011，上海美术馆2012，法国波尔多潜艇当代艺术馆2014，伦敦中央圣马丁美院阪烁艺术馆2007，美国弗吉尼亚英联邦艺术与设计 大学 艺术馆2004。2014年联手法国哲学家吕克费里，中央美术学院院长潘公凯，艺术理论家彭锋在京举行美学论坛。2013年，诗蓝携手巴黎歌剧院指挥卓丹及中国国家交响乐团在国家大剧院推出大型音乐画展《视觉交响曲》。诗蓝作品于2011年5月与旅法前辈赵无极，朱德群作品并列悬挂法国新建驻华大使官邸接待大厅。2007年参加[[皇家艺术学院|英国皇家艺术馆]]国际艺术家竞赛，

== 艺术历程 ==

诗蓝生于北京。80年代移居法国。1986年考入[[巴黎政治科学院|巴黎政治科学院]]本科。1990年考入[[索邦大学|巴黎索邦大学]]政治哲学研究生班。1991年获硕士后学位哲学论文《政治异化》。文革时代对诗蓝和她同代人带来的精神创伤使政治和哲学的研究成为她不可避免的求知历程。哲学随后成为她的艺术主题。

上世纪90年代中期，诗蓝转向艺术训练。首先就读于佛罗伦撒艺术与绘画修复学院。之后曾经在威尼斯国际绘画学院兼修版画。20世纪初在法国参加Moulin de Perrot艺术学院绘画大师班。从而结识法国国立美协画家主任Pierre Henry和法国艺术家联盟主席Remy Aron。2003年法国国立美协[SNBA]和法国泰勒基金会Fondation Taylao接纳诗蓝为会员。同年她诗蓝被任命联合国国际造型艺术家协会法国执委成员。2004年，诗蓝个展《沙帘下的寂静》在美国佛吉尼亚联邦大学艺术学院卡它尔学院美术馆举行，首次推出她独创的实验水墨印象作品<ref>香港大公記者贾垒文章《[http://www.photofans.cn/blog/showarticle.php?threadyear=2012&threadid=59296 詩藍：在當代藝術語境中「凝視東方」] {{Wayback|url=http://www.photofans.cn/blog/showarticle.php?threadyear=2012&threadid=59296 |date=20160304092955 }}》</ref>。在次期间，诗蓝赴伦敦参加中央圣马丁美术与设计学院综合材料大师班。随后考入学院油画研究生系，在导师Stephen Williams指导下于2007年获取美术学硕士学位。2006年，[[伦敦艺术大学|伦敦艺术大学]]校长Michaël Bichard发现诗蓝作品并成为诗蓝作品收藏家。2007年伦敦中央圣马丁美院阪烁艺术学院为诗蓝再次举办个展。同年，诗蓝作品获选参加英国皇家艺术馆夏日联展。英国收藏家Simon Oliver在开幕式上收藏诗蓝作品《寂静之声》。自2003年，诗蓝作品多次参展法国美协[[卢浮宫|卢浮宫]]艺术展和法国[[大皇宫|大皇宫]]大都示艺术展“L'Art en Capital”。自2013年诗蓝画室入法国Louis Vuitton中国艺术家工作室中法双语指南。2013年应Hermes 为其在周末画报广告。 2014年与法国著名室内设计师Bruno Moinard 合作，2014 年 获法国作家艺术家家协会装置头等奖。 2011年与法国文化部室内设计师合作法国驻华使馆项目。两幅巨作入法国使馆第一接待大厅。与赵无极，朱德群两位旅法前辈并列悬挂。

数百张作品被欧亚美各艺术机构与个人收藏: 中国美术馆，中国国家大剧院，法国马格莱艺术基金会[Institut Bernard Magrez]，法国住中国大使馆，意大利复萌艺术基金会, 环铁美术馆，丹麦妇女艺术基金，卡塔尔工商银行艺术基金，中国星艺术杂志，中国艺周刊杂志，中国艺术新闻网，流动艺术，视觉艺术，颐和会馆画廊

私人收藏：中国，法国，英国，瑞士，美国，意大利，丹麦，比利时，南非。日本，韩国，伊朗，黎巴嫩，摩洛哥，伊拉克，西班牙，希腊，德国，马达加丝加，巴西等其中包括法国前总理阿兰于贝，法国前教育部长吕克费里，法国驻华大使白林，伦敦艺术大学校长Sir Michael Bichard,  美国弗吉尼亚大学艺术学院副院长 Christina Lindholm.

== 艺术媒介 ==

诗蓝知名于个人创意的布面实验水墨风格。她在东西方艺术媒介中各取所需，塑型膏、颜料粉、丙烯、矿物芯片、砂与中国墨，宣纸和书法等元素混合被纳入当代水墨印象画作中。超越古典水墨纸笔书画模型。她营造带有浓厚东方审美倾向的画作并把之嵌入当代艺术媒介中。展览方式突出光的利用和展厅空间设计，把观者纳入绘画和装置作品之中，以达到戏剧性感官效果。

== 风格与主题 ==

诗蓝作品主题脱离中国当代艺术政治波普的总体风格，秉承东方审美传统和文化心性。借艺术把观者提升到精神反思的层面。营造禅境审美语义下的谦卑，静虚、内敛等诗意境界。色彩处理显示对清秀雅典，潇洒超脱等意境的追求。90年代初诗蓝曾旅居日本近两年之久。多次身临日本和歌山縣东密佛教圣地[[高野山|高野山]]（こうやさん）。空旷和朴素典雅的环境无疑影响到了艺术家审美品位。
[[File:Les_Stèles_de_Lumiere.jpg|thumb]]

诗蓝作品介于抽象与具象之间的山水。从传统中国绘画的角度来看，诗蓝的作品保持了中国绘画对于意境的追求，是一种境界和形象的结合。她的作品虽然有对现实和自然的抽象过程，但是没有完全转换为抽象艺术。摇摆在齐白石所言的“似与不似之间”，用古人名言，可谓“大象无形”。她作品中所保持的形象的简约和对抽象边界的控制，对极端和强烈刺激的避免，对平衡的追求是作品中东方的优雅魅力的所在。”《殷双喜-中国美术馆诗蓝艺术研讨会讲话整理》同西方高度理性化的园林艺术相比，诗蓝营造的禅境山水是直觉和感性。它是抽象与具象，真实与虚幻的交融。

== 视觉艺术和音乐 ==

诗蓝15岁被征文艺兵入伍。音乐是她视觉艺术的同伴。在巴黎就学期间，考入欧洲爱乐交响乐团业余合唱团。1988年结识法国指挥家并在他的指挥棒下多次在欧洲个城市演出。1991年入巴黎交响乐合唱团并在[[谢苗·毕契科夫|毕契科夫]]指挥下参演法国作曲家[[柏辽兹|柏辽兹]]歌剧排练和演出。演出中结识[[巴黎歌剧院|巴黎歌剧院]]明星音乐家Frederic Laroque。1991-1993年，诗蓝两次联手法国指挥家Hugues Reiner、Mark Trauteman和小提琴家Frederic Laroque与中国广播交响乐团和中国电影乐团在北京音乐厅合作演出。2005法国使馆和卡塔尔基金推出诗蓝与巴黎歌剧[[室内乐|室内乐]]团合作的音乐画展《绘画与音乐》。2004诗蓝设计巴黎歌剧院明星独奏家出版的《德彪西》乐曲唱片封面。2007年与巴黎城市交响乐团合作在巴黎演出[[福雷|福雷]]的《葬礼进行曲》。2010年9月北京法国文化中心主办《视觉交响曲》。这是诗蓝与中央音乐学院艺术家首次合作的音乐画展，作品创意主题来自拉脱维亚作曲家[[彼得里斯·瓦斯克斯|瓦斯克斯]]的《生命之声》。2013 年诗蓝联手中国国家交响乐团，巴黎歌剧院指挥和首席小提琴在国家大剧院推出大型音乐视觉艺术展<视觉交响曲>

== 主要作品介绍 ==

=== 复调 ===
复调是诗蓝的近期大型混合媒体装置作品 。创意来源于作曲家彼得斯·瓦斯（[[彼得里斯·瓦斯克斯|Peteris Vasks]] <Cantabile per archi>）的诗歌交响乐。2013年诗蓝携手法国巴黎歌剧院指挥[[Philippe_Jordan|菲利普 卓丹]]和该乐团首席拉洛克，联合中国国家交响乐团在[[国家大剧院|中国国家大剧院]]首创此作品。名为<视觉交响曲>:展览作为2013年中法文化之春闭幕式

2017年作品经过重新设计附加印象光源及音乐组合，再现于[https://web.archive.org/web/20170121000119/http://www.museomacro.org/mostre_ed_eventi/mostre/li_chevalier_trajectory_of_desire 意大利罗马当代美术馆]。改名为<复调> 该装置由上百把弦乐乐器，小提琴，中提琴，大提琴，和摄影，书法和艺术墨水艺术绘画等元素组合成森林。以由艺术家设计的铁结构作为支柱，以适合各展览空间的场地特性。视频投影和灯光和声同步， 并结合音乐表演。 复调作为一个文明相遇的“共同”空间， 表述 一个不可能放慢文化交融步伐，天人合一的世界.

《'''诗意栖居大地'''》是诗蓝与2010年12月在上海美术馆呈现给观众的大型装置作品。中国古代诗歌碑林拓片，中国传统灯艺等传统艺术被应用在当代艺术手法之中。作品的黑白两阵呈现东方阴阳宇宙观。诗象征文化，灯象征‘辉煌’, 碑作为古人的足迹与未来交接。在明与暗交界之处，艺术家别有用心地保留出中间甬道。观者走入这《中庸之道》以体会东方的天人合一之道。这副作品的副题《黑，白，与黑白之间》由比利时作曲家迪尔-波赛（Dirk Brosse）小提琴协奏曲题目而来。
