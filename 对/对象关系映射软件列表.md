== PHP ==
* [[Axon|Axon]]，[http://fatfree.sourceforge.net Fat-Free Framework]的ORM插件。基于PDO并且无需设置(GPL3)。
* [[CakePHP|CakePHP]]，PHP5的ORM和框架，开源 (scalars，arrays，objects)。Based on database introspection。无需类扩展或代码生成。
* [[Doctrine_(PHP)|Doctrine]]，PHP 5.2.3的开源ORM，免费软件(GNU LGPL)
* [[PdoMap_(PHP)|PdoMap]]，PHP 5.x的开源 ORM，灵感来源于Hibernate，免费软件(GNU LGPL)
* [[Propel_(PHP)|Propel]]，PHP 5的ORM和查询工具箱，灵感来源于Apache Torque，免费软件(MIT)
* [[Rocks_(PHP)|Rocks]]，PHP 5.1+的开源ORM，非商业用途免费 (GPL)
* [[Qcodo|Qcodo]]，PHP5的ORM和框架，开源
* [[Redbean_(software)|Redbean]]，PHP 5的ORM层(BSD)。
* [[Sphorm_PHP|Sphorm]]，PHP 5的ORM，免费软件，使用访问者模式，二叉树和Fluent Pattern。[https://web.archive.org/web/20111017212111/http://sphorm.net/]
* [[Syrius|Syrius]]，PHP 5的ORM，免费软件，使用访问者模式，二叉树和Active Data Pattern。[http://www.domenicobiancardi.com]
* [[Torpor_(PHP)|Torpor]]，PHP 5.1+的开源ORM，免费软件(MIT)，数据库和操作系统无关
