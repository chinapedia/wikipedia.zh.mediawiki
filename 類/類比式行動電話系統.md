'''高级移动电话系统'''（{{lang-en|Advanced Mobile Phone System}}，缩写AMPS）又稱'''類比式行動電話系統'''、'''北美行動電話系統'''，是由[[贝尔实验室|贝尔实验室]]开发的基于[[模擬信號|模擬信號]]的第一代[[行動電話系統|行動電話系統]]。

AMPS於1980年在[[美國|美國]]發展，是一種蜂巢式系統，將所涵蓋的區域劃分為細胞群，使用800[[MHz|MHz]]的[[UHF|UHF]]頻段。因易受電波干擾及竊聽，2000年代後逐漸被[[GSM|全球行動通訊系統]]（GSM）所取代。

AMPS于1983年10月13日引入[[美洲|美洲]]<ref name="Testing the First Cell Phone Network">{{cite web|author=AT&T Tech Channel |url=http://techchannel.att.com/play-video.cfm/2011/6/13/AT&T-Archives-AMPS:-coming-of-age |title=AT&T Archives : Testing the First Public Cell Phone Network |publisher=Techchannel.att.com |date=2011-06-13 |accessdate=2013-09-28 |deadurl=yes |archiveurl=https://web.archive.org/web/20131029194138/http://techchannel.att.com/play-video.cfm/2011/6/13/AT%26T-Archives-AMPS%3A-coming-of-age |archivedate=2013-10-29 }}</ref><ref name="PrivateLine">[http://www.privateline.com/dailynotes/index13.html Private Line] {{webarchive|url=https://web.archive.org/web/20120610001231/http://www.privateline.com/dailynotes/index13.html |date=2012-06-10 }}.</ref><ref name = "MilestonesPast">[http://www.milestonespast.com/exbringing.htm MilestonesPast].</ref>，1986年引入[[以色列|以色列]]、1987年引入[[澳大利亚|澳大利亚]]、1988年引入[[新加坡|新加坡]]、1990年引入[[巴基斯坦|巴基斯坦]]。<ref name="Australian Mobile Telecommunications Association">{{cite web|url=http://www.amta.org.au/default.asp?Page=142 |title=Ten years of GSM in Australia |accessdate=2008-08-16 |work=AMTA |publisher=Australian Mobile Telecommunications Association |year=2003 |archiveurl=https://web.archive.org/web/20080720110058/http://www.amta.org.au/default.asp?Page=142 |archivedate=20 July 2008 |deadurl=yes }}</ref>它在二十世纪八十年代到二十一世纪初是[[北美洲|北美洲]]及其他地区的主要移动电话系统。截至2008年2月18日，美国各运营商已不再支持AMPS，[[AT&T|AT&T]]和[[威訊通訊|威訊通訊]]已永久终止此服务。AMPS于2000年9月在澳大利亚停运，2004年10月在巴基斯坦停运<ref name="Australian Mobile Telecommunications Association"/><ref>http://www.pta.gov.pk/index.php?cur_t=vtext&option=com_content&id=237&scid=146&Itemid=1</ref>，2010年在以色列和巴西停运<ref>http://g1.globo.com/tecnologia/noticia/2010/07/celulares-analogicos-estao-extintos-no-brasil-diz-anatel.html</ref>。
