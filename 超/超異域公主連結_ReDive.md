{{vgname|超異域公主連結 Re:Dive|ja=プリンセスコネクト！Re:Dive }}是一款由[[Cygames|Cygames]]所製作的奇幻風格角色扮演手機遊戲，前身為2015年至2016年間營運的《公主連結》。本遊戲除了在日本上市外，也與海外業者推出包含繁體中文、簡體中文、韓文等版本<ref>{{Cite web |url = https://gnn.gamer.com.tw/6/159596.html |title = 《Princess Connect！Re:Dive》確定將推繁體中文版 預計 2018 年上市 |author = 犬拓 |date = 2018-03-01 |publisher = [[巴哈姆特電玩資訊站|巴哈姆特電玩資訊站]] |language = zh-tw |accessdate = 2018-08-05 }}</ref>。

2019年2月15日，在本游戏一周年生放送上宣布《公主连结 Re:Dive》TV动画化。<ref>{{cite web|url=https://www.animenewsnetwork.com/news/2019-02-15/cygames-princess-connect-re-dive-smartphone-game-gets-tv-anime/.143484|title=Cygames' Princess Connect! Re:Dive Smartphone Game Gets TV Anime|publisher=animenewsnetwork|date=2019-2-15|language=en}}</ref>

==簡介與特色==
主要特色為動畫方式表現的劇情，由[[WIT_STUDIO|Wit Studio]]所製作，並結合腳本家[[日日日|日日日]]、作曲家[[田中公平|田中公平]]<ref>{{Cite web |url = https://www.4gamers.com.tw/news/detail/35805/redive-published-by-so-net |title = 《超異域公主連結☆Re:Dive》繁中雙平台上架，藝人李毓芬現身站台 |author = zizi |date = 2018-08-02 |publisher = [[4Gamers|4Gamers]] |language = zh-tw |accessdate = 2018-08-05 }}</ref>。玩家可將蒐集到的角色，組成五人編隊，依前鋒、中鋒、後衛三排與[[NPC|NPC]]交戰；而玩家也可組織[[公會|公會]]，除了與NPC對戰外，也能與其他玩家進行對戰。

=== 主線任務與劇情 ===
遊戲以無數關卡作為區分，每個關卡會有三波怪物來襲，每波有一分半的時間，在時間內將所有怪物打倒便算過關並獲得獎勵。
通過特定關卡之後便可閱讀更多主線劇情的章節。
除了 Normal (一般)之外，還有 Hard (困難)的關卡，
困難的關卡和一般的關卡是分開的，沒有通過困難的關卡並不會影響一般關卡的推進，並且，困難的關卡不會影響閱讀主線劇情，
但是困難關卡通關會贈送特定腳色的記憶碎片，這些記憶碎片可以用來升級角色的星等或是獲得新的腳色。

=== 競技場 ===
分為戰鬥競技場與公主競技場，戰鬥競技場每個玩家需要準備一組 5 名角色的隊伍，並挑戰其他比自己名次靠前的玩家，與其他遊戲內的戰鬥不同，
競技場除了選擇角色以外，玩家無法控制戰鬥的過程，包含必殺技的施放。
每當玩家得到歷史新高的排名時，系統會予以獎勵。
另外，每天下午三點系統會根據玩家排名再額外給予獎勵。
公主競技場和戰鬥競技場類似，只是玩家必須準備三組隊伍。
值得一提的是，並非所有伺服器的玩家都待在同一個競技場內，系統會依照玩家通過主線任務關卡的時間進行分區，以確保新手不會與資深玩家分在同一個競技場內。

=== 活動 ===
公主連結R每個月初會舉辦為期20日左右的活動，活動會伴隨著特殊的劇情與任務，通常會伴隨活動在卡池內新增角色，有時活動任務也會贈送玩家角色。

=== 戰隊競賽 ===
2~30名的玩家可以組成戰隊，戰隊內的成員可以通訊並交換道具、租借任務用的角色。
在月底的最後 10 日會舉辦以戰隊為單位的比賽。所有戰對成員在消耗了 300 體力之後，可以選擇時間攻擊一隻血條非常長的 Boss，
戰對內的所有成員的傷害將會累積，
當累積傷害將 Boss 打死時可以再挑戰下一隻 Boss，10日之後官方會對所有伺服器的戰隊總傷害排名並給予獎勵。

==故事大綱==
本遊戲設定在一個稱作「阿斯特萊亞」的虛構大陸上，其都城「蘭德索爾」居住著[[獸人|獸人]]、[[精靈|精靈]]、[[人類|人類]]、[[魔族|魔族]]等生物；玩家在無意之中，伴隨著空白的記憶，進入了這個異世界。<br/>
映入玩家眼簾的包含飢餓難耐的劍士、和善的少女魔法師、徬徨無助的貓耳獸人；隨著與她們之間的邂逅，玩家與她們一同過著在異世界的幸福快樂日子。<br/>
然而，有個不為人知的威脅與陰謀，伺機籠罩在阿斯特萊亞大陸，並威脅著玩家和伙伴們……

=== '''主人公'''（{{lang|ja|主人公}}）===
:配音：[[阿部敦|阿部敦]]
:故事的男主角，代表玩家的角色。
:遊戲內設名稱為「佑樹」，前作「公主連結」的男主角且代表玩家的角色，前作結局中與優衣等主角群與「七冠」之一的霸瞳皇帝決戰，並將之擊敗；而在续作的開頭接續前作與霸瞳皇帝決戰後，與其他同伴被霸瞳皇帝送入異空間，傳送到「蘭德索爾」後失忆。
:固有能力兼身分為「公主騎士」，有此能力的能力者本身無法進行戰鬥，但與公主們的「羈絆」連結會使她們的戰鬥能力獲得飛躍性強化。

:本名为枣可萝（{{lang|ja|棗こころ}}），在现实世界是个小学五年级女生。

:[[貓耳|貓耳]]獸人，一開始倒臥在路邊，後來偶遇玩家才成為其夥伴，实质上是霸瞳皇帝派来监视主人公的卧底，霸瞳皇帝將部分公主騎士的能力分與她，在本作中常用公主騎士的能力控制魔獸，但似乎掌握得不是很好，常常發生魔獸失控。
:前作中就有登場的人物，本名为百地希留耶（{{lang|ja|百地 希留耶}}），現實世界是霸瞳皇帝的粉絲兼遠親，与其他角色不同，凱留在现实世界中并不认识主角。

大招台詞：{{lang|ja|覚悟はいいかしら！グリムバースト！}}

== 世界觀與用語 ==
=== 公主與公主騎士 ===
公主為前作「公主連結」中可以實現任何願望的力量。
本作開頭反派勝利之後便為霸瞳皇帝所有。

=== 七冠 ===
: 與前作相同創造這個虛擬世界的七人，又稱「超越者」，其定位為虛擬世界的「神」，掌管虛擬世界的不同權能，七人也各有不同的名號。
: 在現實世界與虛擬世界都有超常的能力，而此七人都是在現實世界中對於虛擬世界舞台的開發者、研究者、贊助商之一。
: 在本作中除了霸瞳皇帝、迷宮女王以外的人都失去記憶了。

=== 與前作的關係 ===
劇情與世界觀基本上與前作「公主連結」共通，本作的開頭正好是前作的結局。
不同的是這個世界是霸瞳皇帝擊敗前作主角群的世界。
霸瞳皇帝在獲勝之後反轉現實世界與虛擬世界，並且奪取了公主之力與公主騎士之力。
