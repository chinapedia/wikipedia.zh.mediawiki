'''超大質量黑洞'''是[[黑洞|黑洞]]的一種，其[[質量|質量]]是<math>10^5</math>至<math>10^9</math>倍的[[太陽質量|太陽質量]]。現時一般相信，在所有的[[星系|星系]]的中心，包括[[銀河系|銀河系]]在內，都會有超大質量黑洞。
== 比較 ==
比較超大質量黑洞與其他[[質量|質量]]相對較低的[[黑洞|黑洞]]，可見一些特別的性質：

* 超大質量黑洞平均[[密度|密度]]可以很低，甚至比[[空氣|空氣]]密度還要低。這是因為[[史瓦西半徑|史瓦西半徑]]與其質量成正比，而密度則與[[體積|體積]]成反比。由於球體（如非旋轉黑洞的[[事件視界|事件視界]]）體積是與[[半徑|半徑]]立方成正比，而質量差不多以直線增長，體積增長率則會更大。故此，密度會隨黑洞半徑增長而減少。

* 在事件視界附近的[[潮汐力|潮汐力]]會明顯較弱。倘有一太空人向黑洞中央移動，在他到達黑洞深處之前不會感受到明顯潮汐力，此係因事件視界距離位處中央之[[引力奇點|引力奇點]]很遠所致。

== 形成 ==
[[File:Supermassiveblackhole_nasajpl.jpg|right]]
超大質量黑洞的形成有幾個方法。最明顯的是以緩慢的[[吸積|吸積]]（由[[恆星|恆星]]的大小開始）來形成。另一個方法涉及[[星雲|星雲]]萎縮成數十萬[[太陽質量|太陽質量]]以上的[[相對論星體|相對論星體]]。該星體會因其核心產生正負[[電子|電子]]對所造成的徑向擾動而開始出現不穩定狀態，並會直接在沒有形成[[超新星|超新星]]的情況下萎縮成[[黑洞|黑洞]]。第三個方法涉及了正在核塌縮的高[[密度|密度]]星團，它那負[[熱容|熱容]]會促使核心的分散速度成為[[相對論|相對論]]速度。最後是在[[大爆炸|大爆炸]]的瞬間從外壓製造[[原生黑洞|原生黑洞]]。

形成超大質量黑洞的問題在於如何將足夠的[[物質|物質]]加入在足夠細小的[[體積|體積]]內。要做到這個情況，差不多要將物質內所有的[[角動量|角動量]]移走。向外移走角動量的過程就是限制黑洞膨脹的因素，並會導致形成[[吸積盤|吸積盤]]。

根據觀測，黑洞的類別有著一些差距。一些從恆星塌縮的黑洞，最多約有10倍太陽質量。最小的超大質量黑洞約有數十萬太陽質量。但卻沒有在它們之間的[[中介質量黑洞|中介質量黑洞]]。不過，有模型指[[異常明亮的X射線源|異常明亮的X射線源]]有可能是在這個遺失範圍的黑洞。

== 都卜勒效應量度 ==
直接量度圍繞鄰近[[星系|星系]]核心[[激微波|邁射]]的[[都卜勒效應|都卜勒效應]]，只有在中央高[[物質|物質]][[密度|密度]]的情況下，才可以發現很快速的[[克卜勒定律|克卜勒運動]]。現時唯一已知可以在細小空間中包含足夠物質的是[[黑洞|黑洞]]，或是在[[天體物理學|天體物理學]]上很短的時間內將變成黑洞的物體。對於較遠的[[活動星系核|活躍星系]]，寬[[譜線|譜線]]的闊度可以用來探測圍繞近[[視界|視界]]的[[氣體|氣體]]。[[反射繪圖|反射繪圖]]的技術就是利用這些譜線的變化來量度其[[質量|質量]]，而黑洞的旋轉有可能加速了活躍星系的「引擎」能量。

在很多星系中心的超大質量黑洞被認為是活躍星系（如[[賽弗特星系|賽弗特星系]]及[[類星體|類星體]]）的「引擎」。[[馬普地外物理研究所|馬普地外物理研究所]]及[[洛杉磯加利福尼亞大學|洛杉磯加利福尼亞大學]]<ref>{{cite web|url=http://www.astro.ucla.edu/~ghezgroup/gc/?|title=UCLA Galactic Center Group|work=ucla.edu}}</ref>基於[[歐洲南天文台|歐洲南天文台]]<ref>{{cite web |url=http://www.eso.org/outreach/press-rel/pr-2002/pr-17-02.html |title=存档副本 |accessdate=2005-10-25 |0=2005-10-25 |deadurl=yes |archiveurl=https://web.archive.org/web/20051025013223/http://www.eso.org/outreach/press-rel/pr-2002/pr-17-02.html |archivedate=2005-10-25 }}</ref>及[[凱克天文台|凱克天文台]]<ref>{{cite web |url=http://www.keckobservatory.org/news/old_pages/andreaghez.html |title=存档副本 |accessdate=2013-07-14 |0=2012-03-01 |deadurl=yes |archiveurl=https://web.archive.org/web/20120301030647/http://www.keckobservatory.org/news/old_pages/andreaghez.html |archivedate=2012-03-01 }}</ref>的數據，提供了證據指[[人馬座A*|人馬座A*]]就是在[[銀河系|銀河系]]中心的超大質量黑洞。根據計算，它可能有431萬倍的[[太陽質量|太陽質量]]。

== 銀河系以外的超大質量黑洞 ==
於2004年5月，Paolo Padovani及其他[[天文學家|天文學家]]發表他們發現了在[[銀河系|銀河系]]以外30個超大質量黑洞。他們的發現令我們知道超大質量黑洞的數量最少是以往所知的兩倍。現時相信每一個[[星系|星系]]的中央包含一個超大質量黑洞，而它們大部份都處於「不活躍」的狀態且[[吸積|吸積]]不多。相反在[[球狀星團|球狀星團]]的中央卻沒有[[黑洞|黑洞]]，不過相信一些如在[[飛馬座|飛馬座]]的[[M15|M15]]及在[[仙女座星系|仙女座星系]]的[[马亚尔II|马亚尔II]]的中央仍有黑洞，估計質量約有<math>10^4</math>的太陽質量。

一些星系，如[[0402+379星系|0402+379星系]]有兩個超大質量黑洞，形成一個二元系統。若它們相撞，將會產生強勁的[[重力波_(相對論)|重力波]]。

== 超大質量黑洞質量與星系形成 ==
超大質量黑洞的質量與其身處的[[星系|星系]]形態有關。這顯示了星系球體的質量與超大質量黑洞的質量有著相互的關連。而黑洞的質量亦與星系的分散速度有著更緊密的關連。但是這個關連卻未被解開。
==特大质量黑洞==
特大质量黑洞（英语：ultramassive black hole）則是拥有着超过{{val||e=10}}[[太阳质量|太阳质量]]，目前位于[[巨大质量黑洞列表|巨大质量黑洞列表]]的前两名[[TON_618|TON 618]]以及[[S5_0014+81|S5 0014+81]]很可能属于特大质量黑洞。
