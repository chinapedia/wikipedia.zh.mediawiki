本列表為[[帕布雷斯特|帕布雷斯特]]（現NAMCO BANDAI Games Inc.,）的電子遊戲作品《'''[[超級機械人大戰系列|超級機械人大戰系列]]'''》（以下機戰），及其原生、派生作品登場的原創機動兵器一覽。

=== 人型機兵 ===
'''Personal Trooper'''，簡稱'''PT'''，毛氏工業（{{lang|ja|マオ・インダストリー社}}）開發的人型[[機動兵器|機動兵器]]。第一號機是PTX-001 亡靈（{{lang|ja|ゲシュペンスト}}，Gespenst<ref>[[德語|德語]][[亡靈|亡-{}-靈]]、[[幽靈|幽靈]]或[[幻影|幻影]]之意</ref>），也是人類史上最初的人型機動兵器。其他譯名有：蓋修班斯特，肯修貝斯特，極殊兵，擊修柏特。

==== 亡靈系列 ====
;{{nihongo|PTX-001 亡靈·R型|ゲシュペンスト・タイプR|Gespenst Type-Rapidity}}
:是毛氏工業開發的第一架人型機兵，亡靈的一號機（Type R）。有很多機體從高成本的決戰用機到低成本的量產機都是以它為原型製造的。該機是注重機動性能機型（R = Rapidity）。機體完成後配給特殊戰技教導隊進行測試。後來成為奇里亞姆·葉格的專用機。

;{{nihongo|PTX-001RV 亡靈·RV型|ゲシュペンスト・タイプRV|Gespenst Type-RV}}
:是奇里亞姆改造的機體。在注重機動性能的機能下，機動性能得到大幅度提升。

;{{nihongo|PTX-001XNG 亡靈·XN|XN ゲスト|XN Geist/Natch Geist X}}

;{{nihongo|PTX-002 亡靈·S型|ゲシュペンスト・タイプS|Gespenst Type-Strength}}
:Type S是試作二號機。之所以稱為Type S，是因其為注重攻擊力的機體（S = Strength）。此機後來由特殊戰技教導隊隊長劉家衛（{{lang|ja|カーウァイ・ラウ}}）帶往宇宙進行測試，其後失蹤。
:其實真正故事是家衛與本機被虛視者擄獲後，將亡靈S型的機體強化改裝與家衛的人格精神被強行清洗記憶，最後他演變成為巴爾瑪帝國星球人的殺手身份，並改名為加因·馬尼（{{lang|ja|ガルイン・メハベル}}）。
:在OG1中毛氏工業被攻擊一戰（SRX劇情線的第37集）或白星進攻戰（ATX劇情線的第40集）中以敵方身份出現，最後被ATX和SRX等等聯合擊毀告終。

;{{nihongo|PTX-003 亡靈·T型|ゲシュペンスト・タイプT|Gespenst Type-Test}}
:Type T是試作三號機。之所以稱為Type T，是因為此機是作為將來後繼機開發的測試機（T = Test），'''古鐵'''便是由此機改裝而來。

;{{nihongo|PTX-007-01 亡靈Mk-II·R型|ゲシュペンストMk-II・タイプR|Gespenst Mk-II Type-Rapidity}}
:人型機兵第一號機PTX-001「亡靈·R型」的後繼機，三架量產試作機中的一機。沒有使用EOT技術<ref>Extra Over Technology的總稱，從隕石流星3號發現的超技術總稱</ref>而使用既有技術應用及發展出可靠的高性能的機體。之所以稱為Type R，是因為它是注重機動性的機體（R = Rapidity）。過去，Mk-II曾是特殊戰技教導隊成員取得資料所使用的機體，DC戰爭爆發後，由毛氏工業重新調整後再投入戰線。

;{{nihongo|PTX-007-02 亡靈Mk-II·S型|ゲシュペンストMk-II・タイプS|Gespenst Mk-II Type-Strength}}
:Type S是Mk-II的試作二號機。之所以稱為Type S，是因其為注重攻擊力的機體（S = Strength）。胸部安裝了重型破壞加農砲（改良自'''亡靈·S型'''的破壞加農砲）並能使出名為「究極！亡靈踢」（{{lang|ja|究極！ ゲシュペンストキック}}）的強力必殺技。它的數據被運用在古倫加斯特的開發。

;{{nihongo|PTX-007-03 亡靈Mk-II·T型|ゲシュペンストMk-II・タイプT|Gespenst Mk-II Type-Test}}
:Type T是Mk-II的試作三號機。之所以稱為Type T，是因為此機是作為將來後繼機開發的測試機（T = Test），'''白騎士'''便是由此機改裝而來。

;{{nihongo|RPT-007 量產型亡靈Mk-II|量産型ゲシュペンストMk-II|Gespenst Mk-II Mass Product Model}}
:毛氏工業首次開發的量產型人型機兵，亡靈Mk-II·R型的量產轉移型。性能基本上和量產試作型一樣，強化了感應器。左腕的武裝從原先的電漿劍換成了裝有電漿棍的噴射麥格農。由於量產延遲的緣故，DC戰爭爆發時僅配置了30架左右（當時聯邦軍的主力兵器為戰鬥機與戰車）。卡琪娜中尉的專機機身為紅色，北村開少佐的專機機身為綠色。

;{{nihongo|RPT-007TT 量產型亡靈Mk-II·TT型|量産型ゲシュペンストMk-II・タイプTT|Gespenst Mk-II Mass Product Model T-Link Test Type}}
:搭載了T-LINK系統的量產型亡靈Mk-II，機體為全白色，裝備了名為T-Link Ripper的精神操作武器。

;{{nihongo|RPT-007K-P1 量產型亡靈Mk-II改|量産型ゲシュペンストMk-II改|Gespenst Mk-II Mass Product Model Custom}}
:量產型亡靈Mk-II的強化型，總共開發了3台，有兩台配備了泣狼隊（OG外傳），配備了類似白騎士的新型射擊武器；另一架則由北村開架駛，其機體與前者的分別是原先的一個噴射麥格農增加至兩個，令近戰能力大為提高。

:型式編號：RTX-008L / R
:全長：19.8m　重量：60.4t

:受聯邦軍極東支部委託，由毛氏工業開發的最早搭載EOT的人型機兵。此機是以豪豬（{{lang|ja|ビルトシュバイン}}）為基礎，為了作為亡靈Mk-II的後繼量產機而開發的機體，但是由於開發途中EOT發現了隕石三號而使用黑洞引擎作為其動力部，故其攻擊力遠高於預先估定的大小。雖然開發了三架，但008R在啟動實驗中因為失控事件的發生，消滅了一個基地，從此以後，凶鳥就有了「消逝的騎兵（Vanishing Trooper）」的稱號，連帶同樣搭載黑洞引擎的008L也被封印了。
:至於009（第三架）也因此除去黑洞引擎，改用核融合引擎，作為後繼機開發用資料的取得機體，送到極東支部手中。

==== ATX計劃 ====
地球圈防衛計劃中，北美支部蘭利（{{lang|ja|ラングレー}}）基地進行的強襲用人型機動兵器的開發計劃。開發責任者是{{nihongo|瑪莉詠·拉多姆|マリオン・ラドム}}博士。由於當時形勢所逼，在瑪莉詠博士不願意下使用了部分EOT技術（搭載特斯拉驅動器）。開發了「古鐵」和「白騎士」。

;{{nihongo|PTX-003-C 古鐵|アルトアイゼン|Alteisen}}
:其他譯名：亞爾特安瑟，艾迪臣
:聯邦軍北美支部蘭利基地的ATX計畫中，以亡靈試作3號機（PTX-003）作为母体进行大幅的改造和加强，武器以适合中近距离战斗的实弹兵器为主，是以「擁有絕對強大的火力並能正面突破」的觀念設計而成的PT，開發者為瑪莉詠博士，驾驶员是{{nihongo|'''南部響介'''|キョウスケ・ナンブ}}。雖然在接近戰、格鬥戰能有很大的發揮空間，但這種與潮流逆向而行的設計概念很快便遭到凍結，所以并没有成为下一代的主力机体。由於古鐵是由亡靈試作3號機改造而成，所以隱藏著「亡靈Mk-III」這個並未正式使用的名字。在另一世界，直接稱為Mk-III。
:<span style="font-size:smaller;">注：{{lang|ja|アルト}}和{{lang|ja|アイゼン}}分别是德语的Alt和Eisen，相当于英语Old Iron（古舊的鐵）。</span>

;{{nihongo|PTX-003-SP1 古鐵巨人|アルトアイゼン・リーゼ|Alteisen Riese}}
:其他譯名：亞爾特安瑟·里傑，艾迪臣·里傑
:古鐵的改造机体。在原机师南部響介的意见基础上，由瑪莉詠·拉多姆博士完成。使用了白騎士的后备零件，装备了以前被认为严重影响机体平衡而没有采用的{{nihongo|'''轉輪衝角'''|リボルビング・バンカー|Revolving Bunker}}，机体其它的武器也全部换成了更加大型化的装备。新安装的特斯拉驅動器提高了机体的平衡性和突进力，可是依然不能飞行。它的火力和突进力都已经超越了PT的范畴，但是机体的操纵要比古鐵更加困难。
:<span style="font-size:smaller;">注：{{lang|ja|リーゼ}}是德语的Riese，相当于英语Giant（巨人）。</span> 

;{{nihongo|PTX-007-03C 白騎士|ヴァイスリッター|Weiβritter}}
:其他譯名：拜斯里塔
:ATX計劃中，由亡靈Mk-II試作3號機（{{lang|ja|ゲシュペンストMk-II・タイプT}}）改造的PT。引入了從DC流出來，里翁（{{lang|ja|リオン}}、Lion）系列的技術，搭載了小型的特斯拉驅動器，可以單體飛行、而且機動性還超越了里翁系列。裝備了可以發射實彈或光束武器的{{nihongo|'''騎槍重砲'''|オクスタン<ref>騎槍（奧古斯坦，{{lang|ja|オクスタン}}，正式写法为{{lang|ja|オックスタン}}）指的是[[Ox_tongue_Spear|Ox tongue Spear]]，一种15-16世纪欧洲的双刃短枪。</ref>・ランチャー|Oxtongue Launcher}}，擁有優秀的砲擊戰能力。但是由于过分追求高机动性而犧牲了裝甲，加上生产成本昂贵，使得它和古鐵同样无法量产化。駕駛員為{{nihongo|'''艾克瑟蓮·白朗寧'''|エクセレン・ブロウニング|Excellen Browning}}。
:<span style="font-size:smaller;">注：{{lang|ja|ヴァイ}}和{{lang|ja|スリッター}}分别是德语WEIß和RITTER，相当于英语的White Knight（白骑士）。</span>

;{{nihongo|PTX-007-UN 純白騎士|ライン・ヴァイスリッター|Rein Weiβritter}}
:其他譯名：萊茵·拜斯里塔
:白騎士被謎一般的技术改造之后提高了机体性能的形态。整个机体有60%以上产生了无法解析的变化，就连原来的主武器騎槍重砲都变成了{{nihongo|'''咆哮重砲'''|ハウリング・ランチャー|Howling Launcher}}。可以展开强力的能量场，能用思念波动进行攻击。
:<span style="font-size:smaller;">注：{{lang|ja|ライン}}是德语的Rein，相当于英语Pure（纯的）。</span> 

;{{nihongo|PTX-015L/R 野伯勞(L/R)|ビルトビルガー(L/R)|Wildwüerger}}

;{{nihongo|PTX-016L/R 野隼(L/R)|ビルトファルケン(L/R)|Wildfalken}}
:L5戰役後，且ATX計畫解除凍結後，由月面企業毛氏工業開發的試作型人型機兵。此機是使用了白騎士的概念而開發的砲擊戰用機體，由於搭載了高性能特斯拉驅動器，在PT中其機動性可說是一流水平。主要武裝是使用能源彈的{{nihongo|'''騎槍步槍'''|オクスタン・ライフル|Oxtongue Rifle}}，與野伯勞不同的是，野隼是使用H骨架。開發者為卡克博士。Type R機身為藍色；另有同型機Type L存在，機身為紅色。

==== SRX計劃 ====
===== R系列 =====
地球圈防衛計劃中，極東支部伊豆基地進行的對外星人戰鬥用人型兵器開發計劃。開發責任者是英格蘭姆少佐。使用了EOT的技術，開發了R系列、古倫加斯特貳式（{{lang|ja|グルンガスト貳式}}）、凶鳥Mk-II（{{lang|ja|ヒュッケバインMk-Ⅱ}}）。

;R-1（Real Personal Trooper Type 1）
:R系列的1號機，主要用作格鬥戰。使用了最新的PT技術和EOT技術。是3機中平衡性最好的。可以變型成飛行型態'''R-Wing'''（{{lang|ja|R-ウィング}}）。駕駛員是'''伊達隆聖'''（{{lang|ja|リュウセイ・ダテ}}）。

;R-2（Real Personal Trooper Type 2）
:R系列的2號機，主要是中距離砲擊戰用。搭載了使用EOT技術的特洛尼姆引擎（{{lang|ja|トロニウム・エンジン}}）。火力高，裝甲厚。但是調整出力困難，機動性低。裝備了武器高能重金屬粒子砲（Hi-Sol Launcher）及追加裝甲以後成為'''R-2強化型'''（{{lang|ja|R-2パワード}}）， 可以和'''R-1'''，'''R-3強化型'''合體成'''SRX'''。駕駛者是'''萊迪斯·F·布蘭修坦'''（{{lang|ja|ライディース・F・ブランシュタイン}}）。

;R-3（Real Personal Trooper Type 3）
:R系列的隊長機，特徵是搭載了T-LINK系統（念動力感知增幅裝置），用以遠距離操作念動武器和提高捕捉敵人、情報收集的能力。裝備了T-LINK飛行系統（T-link Bright System）以後成為'''R-3強化型'''（{{lang|ja|R-3パワード}}），可以和'''R-1'''，'''R-2強化型'''合體成'''SRX'''。駕駛者是'''古林彩'''（{{lang|ja|アヤ・コバヤシ}}）。

:因原開發構想為「特殊人型機動兵器（特機）與人型機兵的特徵，由一機或分離狀態的三機來改變戰局的人型機動兵器」的關係，它們具有著出類拔萃的攻擊能力。為此，它們大多使用在敵據點的破壞及敵方部隊的短期消滅作戰上。以異星人帶來的資源——「特洛尼姆」作媒介而擁有高出力的特洛尼姆引擎及機體的耐久度配合下，在沒有進行整備的情形下只能進行十次的合體（初期時連三分鐘以上的合體都無法維持）。而且，在出力開到最大的情形下特洛尼姆引擎爆炸的話會將半徑50公里以上所有可見物質化成灰燼，因此換個角度來說SRX可謂是一把擁有獨一無二的攻擊力的雙刃劍。
