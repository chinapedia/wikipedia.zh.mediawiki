'''超度量空间'''是一种特殊的[[度量空间|度量空间]]，其中[[三角不等式|三角不等式]]用''d''(''x'', ''z'') ≤ max{''d''(''x'', ''y''), ''d''(''y'', ''z'')}来代替。有时相关的度量也称为'''非阿基米德度量'''或'''超度量'''。虽然超度量空间中的一些定理看来奇怪，它们在许多应用中都自然出现。

==正式定义==

正式地，超度量空间是点''M''的[[集合|集合]]与一个相关的距离函数（又称为[[度量|度量]]）：

:''d'' : ''M'' × ''M'' → '''R''' 

（其中'''R'''是[[实数|实数]]集合），使得对于所有''M''内的''x''，''y''和''z''，都有：
# ''d''(''x'', ''y'') ≥ 0
# ''d''(''x'', ''y'') = 0  [[当且仅当|当且仅当]]  ''x''=''y''
# ''d''(''x'', ''y'') = ''d''(''y'', ''x'')  （'''对称性'''）
# ''d''(''x'', ''z'') ≤ max{''d''(''x'', ''y''), ''d''(''y'', ''z'')}  （'''强三角不等式'''或'''超度量不等式'''）。

==性质==
[[File:Strong_triangle_ineq.svg|thumb]]
从以上的定义中，我们可以推出超度量空间的一些典型的性质。例如，在超度量空间内，对于所有''M''内的''x''，''y''和''z''以及'''R'''内的所有''r''和''s''，都有：
* 每一个三角形都是[[等腰三角形|等腰]]的，也就是说，''d''(''x'',''y'') = ''d''(''y'',''z'')，或''d''(''x'',''z'') = ''d''(''y'',''z'')，或''d''(''x'',''y'') = ''d''(''z'',''x'')。
* [[球体|球体]]内的每一点都是它的中心，也就是说，如果''d''(''x'',''y'') < ''r''，则B(''x''; ''r'') = B(''y''; ''r'')。
* 相交的球体互相包含，也就是说，如果B(''x''; ''r'') ∩ B(''y''; ''s'')是非空的，则要么B(''x''; ''r'') ⊆ B(''y''; ''s'')，要么B(''y''; ''s'') ⊆ B(''x''; ''r'')。
在这里，（开）球体的概念和记法与[[度量空间|度量空间]]中的球体一样，也就是说：
: B(''x; r'') = { ''y ∈ M'' | ''d''(''x, y'') < ''r'' }。
* 所有的球体在诱导拓扑中都既是开集又是闭集。也就是说，开球体也是闭球体，闭球体（把<换成≤）也是开球体。
