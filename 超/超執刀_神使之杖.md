《'''超執刀 神使之杖'''》（日版名：{{lang-ja|超執刀 カドゥケウス}}，英文版名：''Trauma Center: Under the Knife''）是[[Atlus|Atlus]]製作於發售的2005年6月16日[[任天堂DS|任天堂DS]]遊戲。因為意想不到的大受歡迎，Atlus於2007年4月26日發售《DS 超執刀 神使之杖（Atlus廉价版）》。

《'''神使之杖Z 雙超執刀'''》（日版名：{{lang|ja|カドゥケウスZ 2つの超執刀}}，英文版名：''Trauma Center: Second Opinion''）是[[Atlus|Atlus]]製作於發售的2006年12月2日[[Wii|Wii]]遊戲，《超執刀 神使之杖》移植到[[Wii|Wii]]版本上的作品。

== 概要 ==
活用了[[任天堂DS|任天堂DS]]的[[觸控式螢幕|觸控式螢幕]]這個特點，並創立了一種名為「科幻外科動作」的新種型遊戲。玩家是一位天才的外科醫生月森孝介，與助手利根川安祺一起拯救被迷之奇病「罪蝕病」（Guilt）患者。因為本遊戲會隨劇情發展，就像冒險遊戲一般，但到了手術場景是的確是像種型「科幻外科動作」所顯示一樣是一個動作遊戲。遊戲難度相當高，需要玩家選擇正確的工具，以應付不同狀況。

== 故事大綱 ==

故事發生於近未來的2018年，那時候醫療技術已經相當發達，但人類仍然每天與新的疾病和創傷作戰。故事舞台圍繞一個為消滅所有疾病的世界組織Caduceus(聯合國世界衛生組織「神使局」)，組織表面研究難治之症，例如癌症，愛滋病以至可傳人禽流感，但實際上是要與恐怖活動和傳染病作戰的特殊機關。主角月森孝介是一位Caduceus的外科醫生，與致命的怪病「罪蝕（GUILT）」作戰。

== 登場人物 ==
<span style="font-size:smaller;">注：日版名稱 - 美版名稱</span>

冷知識：英文版本的名字大部份有借美國著名的醫療劇集：如「[[仁心仁術|仁心仁術]]」、「[[豪斯医生|豪斯医生]]」中的演員或者角色名字作參考。

*月森孝介（{{lang|ja|つきもり こうすけ}}）- Derek Stiles
:本遊戲的主角，因為年少喪父而立志當醫生。實習期間吊兒郎當，常被上司與護士同事指責。後來一次突發意外喚醒了他的「超執刀」神技。成為新一代的神醫。26歳。
*利根川安祺（{{lang|ja|とねがわ あんじゅ}}）- Angie Thompson
:月森的女助手。21歳。擁有四分之一德國血統的混血兒。對醫療擁有豐富的知識，並擁有國際護理執照的精英份子。遊戲開始時對月森沒有信心，隨著故事的發展，慢慢與月森建立信賴。
*古村百恵（{{lang|ja|ふるむら ももえ}}）- Mary Fulton
:遊戲開始時協助月森執刀策略的助手，亦是北崎醫院的資深護士。39歲。非常照料月森。離職後由利根川接任月森助手一職。
*沓掛真一（{{lang|ja|くつかけ しんいち}}）- Greg Kasal
:月森工作的北崎醫院外科局長。35歲。北崎醫院實力最高的醫生。為人風趣幽默，是月森的導師亦是他發現了月森有「超執刀」的才能。是神使局局長沓掛宗二的孿生兄長。
*北崎威一郎（{{lang|ja|きたざき いいちろう}}）- Robert Hoffman
:北崎醫院的院長。62歳。以前是代表日本的名醫，上一代擁有「超執刀」的神醫，但因某些事而放下手術刀。並曾經勸告月森以他為鑑放棄超執刀的能力。
*沓掛宗二（{{lang|ja|くつかけ そうじ}}）- Sidney Kasal
:與沓掛真一是孿生弟弟。35歲。聯合國世界衛生組織轄下的先進醫療機構「神使局」日本分局局長。性格與哥哥相反為人認真沉實，自妻子因病去世後便專心從事醫療研究工作。
*明神沙耶香（{{lang|ja|みょうじん さやか}}）- Cybil Myers
:34歲，是沓掛真一的後輩。「神使局」日本分局麻醉師。曾經任職刑警，為人冷靜具膽識，同僚間稱她「鐵娘子」。
*佐倉東吾（{{lang|ja|さくら とうご}}）- Tyler Chase
:月森讀醫時的同學，現任職於神使局兒科。26歲。因為親歷其導師痛苦抗病，立志成為不讓病人受苦的醫生而走上支持安樂死之路。加上親妹妹佐倉凛患上罪蝕症長期受治療之苦，令他對安樂死的態度更為執著。
*新垣修也（{{lang|ja|あらがき しゅうや}}）- Victor Niguel
:神使局開發研究部長，自命天才並且為人孤癖。除了沓掛局長外沒有人能收服他。
*寶生卓而({{lang|ja|たからお たくじ}}）- Stephen Clarks
:擁有圓滿家庭育有二名兒子的神使局資深外科醫生。42歲。
*伊東優乃（{{lang|ja|いとう ゆうの}}）- Leslie Sears
:神使局資深護士，主要照顧兒童與患罪蝕症的病人。佐倉東吾的泡妞對象。
*日比谷克彥（{{lang|ja|ひびや こくひこ}}）- Richard Anderson
:日本厚生相(衛生大臣)兼日本神使局總監。與北崎威一郎份屬同窗，行事以市民健康作大前提。
*君島美娜（{{lang|ja|ミラ キミシマ}}）- Nozomi Weaver
:於Wii版本登場的另一個主角，與月森一樣擁有「超執刀」能力，但過去是個謎的女醫生。她的英文版本名字有著「編織希望」的意思。27歲。

== 故事用語 ==
*超執刀（{{lang|ja|ちょうしっとう}}）- Healing Touch
:是對部份具有超凡手術能力技能的醫生的稱謂。擁有「超執刀」的醫生在進行手術時會有非常高的集中力，能以極高速度進行手術術式，並令病人的維生指標迅速回復。因為手術時間長短與病人性命有著直接關係，所以這種技能十分重要。

*神使局（{{lang|ja|カドゥケウス}}）- Caduceus
:隸屬聯合國世界衛生組織的醫療機構，總部設於美國。致力研究對抗各種致命疾病的方法，而背後則為一個對抗醫療恐怖活動的組織。Caduceus一字為希臘神話中神的使者[[赫耳墨斯|赫耳墨斯]]所持的[[雙蛇杖|雙蛇杖]]。神使局亦以此杖作為徽號。(與一般的醫療組織所用的[[單蛇杖|單蛇杖]]不同）

*醫療恐怖活動（{{lang|ja|いりょうテロ}}）- Medical Terror
:有別於一般自殺式恐怖活動，醫療恐怖活動在於以藥物或者人造病源體引發致命疾病以達至社會恐慌。

*罪人（{{lang|ja|つみびと}}）- Sinner
:由利根川安祺命名，所指的是月森等人在恐怖組織的要塞內發現的一群小孩。這群小孩都帶有特殊的基因，因而被用作測試罪蝕體的實驗品。

*特爾非（{{lang|ja|デルフォイ}}）- Delphi
:故事中的恐怖組織。認為醫學進步是違反天道的行為因而利用罪蝕體策劃一連串的醫療恐怖活動。

《'''神使之杖Z 雙超執刀'''》（日版名：{{lang|ja|カドゥケウスZ 2つの超執刀}}，英文版名：''Trauma Center: Second Opinion''）是[[Atlus|Atlus]]製作於發售的2006年12月2日[[Wii|Wii]]遊戲，《超執刀 神使之杖》移植到[[Wii|Wii]]版本上的作品。

除了重製《超執刀 神使之杖》，還加上新的故事和系統。和前作一樣，故事會是隨劇情發展的冒險遊戲，而手術場景就是新類型「科幻外科動作」遊戲。本遊戲只對應4:3螢幕，沒有對應寬螢幕。新加入電擊器代替前作的按摩心臟，以Wii遙控和nunchuk按節拍按Z按鈕和B按鈕來恢復心律，手電筒在黑暗中照亮局部範圍，以及閃光燈在黑暗中照亮一瞬間。
