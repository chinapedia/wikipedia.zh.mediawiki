'''超级联赛方程式'''（'''Superleague Formula'''，简称'''SF'''）是一项新创办的赛车比赛，创建于2008年7月<ref name="超级联盟方程式赛车大赛(Super League Formula) 简介
">{{cite web|url=http://www.98957.net/show.aspx?id=7995&cid=908|title=超级联盟方程式赛车大赛(Super League Formula) 简介|accessdate=2008-11-28|work=中国体育招商网}}{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，是第一个由世界顶级[[足球俱乐部|足球俱乐部]]参加的赛车比赛<ref name="What we want/do">{{cite web|url=http://www.superleagueformula.com/superleague/What-is-SF/What-we-want-do|title={{english}}What we want/do|accessdate=2008-11-28|work=superleagueformula.com|deadurl=yes|archiveurl=https://web.archive.org/web/20090129153515/http://superleagueformula.com/superleague/What-is-SF/What-we-want-do|archivedate=2009-01-29}}</ref>。第一站比赛于2008年8月31日在[[英格兰|英格兰]][[多宁顿公园赛道|多宁顿公园赛道]]举行<ref name="足球豪门染指赛车 超级联赛方程式即将拉开战幕
">{{cite web |url=http://sports.sina.com.cn/f1/2008-08-01/09203824362.shtml|title=足球豪门染指赛车 超级联赛方程式即将拉开战幕|accessdate=2008-11-28 |work= 足球报}}</ref>。该赛事的参赛车队全部由[[足球俱乐部|足球俱乐部]]赞助，首届比赛参赛俱乐部一共18支，包括[[意大利|意大利]][[AC米兰|AC米兰]]、[[英格兰|英格兰]][[利物浦足球俱乐部|利物浦]]等多支著名俱乐部，而来自中国的[[北京国安足球俱乐部|北京国安足球俱乐部]]也参加了第一年的比赛，并夺得总冠军<ref name="SF General Standings
">{{cite web|url=http://www.superleagueformula.com/superleague/Results/General-Standings|title={{english}}SF General Standings|accessdate=2008-11-28|work=superleagueformula.com|deadurl=yes|archiveurl=https://web.archive.org/web/20081101210934/http://superleagueformula.com/superleague/Results/General-Standings|archivedate=2008-11-01}}</ref>。

== 赛事筹办 ==
该项赛事筹划于2005年<ref name="足球豪门染指赛车 超级联赛方程式即将拉开战幕"/>，历经两年的市场调研，赛事组织者认为“无论是车迷还是球迷，都会支持这项赛事”<ref name="“超级联赛方程式赛”成立 马尔蒂尼为米兰试车">{{cite web |url=http://sports.sohu.com/20070405/n249223388.shtml|title=“超级联赛方程式赛”成立 马尔蒂尼为米兰试车|accessdate=2008-11-28 |work= 重庆晚报}}</ref>，于是在2007年4月，意大利AC米兰，[[荷兰|荷兰]][[PSV埃因霍温|埃因霍温]]、[[希腊|希腊]][[奥林匹亚克斯|奥林匹亚克斯]]三家俱乐部率先加入超级联赛方程式赛车，三个月后[[德国|德国]][[多特蒙德足球俱乐部|多特蒙德]]、[[比利时|比利时]][[安德莱赫特足球俱乐部|安德莱赫特]]和[[巴西|巴西]][[弗拉门戈|弗拉门戈]]也决定参与其中，另外一支在2007年加入比赛的俱乐部是[[瑞士|瑞士]][[巴塞尔足球俱乐部|巴塞尔]]<ref name="Participating Clubs">{{cite web|url=http://www.superleagueformula.com/superleague/Clubs|title={{english}}Participating Clubs|accessdate=2008-11-28|work=superleagueformula.com|deadurl=yes|archiveurl=https://web.archive.org/web/20081205053039/http://superleagueformula.com/superleague/Clubs|archivedate=2008-12-05}}</ref>。

2008年初，[[葡萄牙|葡萄牙]][[波尔图足球俱乐部|波尔图]]、[[西班牙|西班牙]][[塞维利亚足球俱乐部|塞维利亚]]、巴西[[科林西安|科林西安]]、[[土耳其|土耳其]][[加拉塔萨雷|加拉塔萨雷]]和[[苏格兰|苏格兰]][[格拉斯哥流浪者|格拉斯哥流浪者]]也决定加入，超级联赛方程式的参赛球队达到了12支。随后来自亚洲的俱乐部[[阿联酋|阿联酋]][[艾恩足球俱乐部|阿尔-艾因]]和[[中国|中国]][[北京国安|北京国安]]的加盟为比赛增添了新的看点，最后一批入围的俱乐部包括意大利[[罗马足球俱乐部|罗马]]、[[英格兰|英格兰]][[托特纳姆热刺|托特纳姆热刺]]和利物浦以及来自西班牙的[[马德里竞技|马德里竞技]]<ref name="Participating Clubs"/>。

超级联赛方程式赛车并不强行要求参与的俱乐部提供经济上的支持<ref name="Participating Clubs"/>，但是要求各俱乐部为自己的赛车及车手提供车身彩绘、标志使用、着装设计等一系列后勤支持，同时还要求各俱乐部用创新的方式开拓[[球迷|球迷]]和[[车迷|车迷]]市场<ref name="Participating Clubs"/>。赛事的最终目标是通过会员俱乐部“'''投入并争胜（In it and win it）'''”<ref name="Participating Clubs"/> 的不懈努力创造“'''完美的赛车比赛（Beautiful Race）'''”<ref name="What we want/do"/>。

== 赛车技术 ==
超级联赛方程式参赛车辆由主办方统一提供，目前发动机采用的是[[梅纳尔兹竞技技术工程公司|梅纳尔兹竞技技术工程公司]]（Menard Competition Technologies Limited (MCT)）所提供的产品，而车身结构则由[[雅伦赛车科技公司|雅伦赛车科技公司]]（Elan otorsports Technologies）负责提供支持<ref name="赛车与足球交汇 超级联盟方程式欲登金字塔顶端">{{cite web|url=http://f1.sports.sohu.com/20070404/n249204175.shtml|title=赛车与足球交汇 超级联盟方程式欲登金字塔顶端|accessdate=2008-11-28 |work= 搜狐体育}}</ref>。

目前赛车规格统一，发动机为4.2升，12V，最大输出为750[[马力|马力]]<ref name="Car & Engine overview">{{cite web|url=http://www.superleagueformula.com/superleague/Car-Engine/Engine-Overview|title={{english}} Car & Engine overview|accessdate=2008-11-28|work=superleagueformula.com|deadurl=yes|archiveurl=https://web.archive.org/web/20081021162803/http://www.superleagueformula.com/superleague/Car-Engine/Engine-Overview|archivedate=2008-10-21}}</ref>，整车的最高时速可达200[[公里|公里]]<ref name="赛车与足球交汇 超级联盟方程式欲登金字塔顶端"/>。

== 俱乐部的筹备 ==
[[File:SF_Donington_9.jpg|thumb]]
为了参加首届比赛，各个俱乐部都做了充分的准备。其中AC米兰是最为积极的一家，他们在2007年4月第一个发布了自己的赛车<ref name="超级联赛方程式将开赛 多恩伯斯加盟AC米兰车队">{{cite web |url=http://sports.eastday.com/s/20080522/u1a3606553.html|title=超级联赛方程式将开赛 多恩伯斯加盟AC米兰车队|accessdate=2008-11-28 |work= 东方网}}</ref>，而且还邀请了前[[F1|F1]][[红牛车队|红牛车队]]车手[[多恩伯斯|多恩伯斯]]为其签约车手，同时多位米兰[[球员|球员]]也参加了发布仪式，包括[[马尔蒂尼|马尔蒂尼]]、[[罗纳尔多|罗纳尔多]]、[[西多夫|西多夫]]和[[卡福|卡福]]等<ref name="超级联赛方程式将开赛 多恩伯斯加盟AC米兰车队"/>，而[[巴雷西|巴雷西]]则亲自出席车队的首场比赛<ref name="方程式超级联赛将刮起红黑旋风">{{cite web|url=http://www.racing-china.com/pub/acmilan/2008/0830/85947.htm|title=方程式超级联赛将刮起红黑旋风|accessdate=2008-11-28|work=AC米兰 中文官方网站}}{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

意大利罗马俱乐部也对赛事投入了很高的热情，俱乐部与F1车手[[詹卡洛·费斯切拉|詹卡洛·费斯切拉]]合作对比赛进行了筹备<ref name="足球豪门染指赛车 超级联赛方程式即将拉开战幕"/>。费斯切拉作为[[罗马|罗马]]人对罗马俱乐部提供了有力的支持，他将自己名下的[[GP2|GP2]]“费斯切拉车队”提供给俱乐部参加比赛<ref name="足球豪门染指赛车 超级联赛方程式即将拉开战幕"/>。此外，土耳其加拉塔萨雷俱乐部则得到了该国汽联的全力支持，该国汽联主席甚至令自己的儿子代表球队出赛<ref name="足球豪门染指赛车 超级联赛方程式即将拉开战幕"/>。

== 中国俱乐部的参与 ==
[[中国足球超级联赛|中国足球超级联赛]]俱乐部[[北京国安|北京国安]]受邀成为首支加入比赛的亚洲俱乐部<ref name="国安组建方程式赛车队 加盟国际超级联赛">{{cite web |url=http://www.espnstar.com.cn/pub/csl/2008/0827/85494.htm|title=国安组建方程式赛车队 加盟国际超级联赛|accessdate=2008-11-28 |work= espnstar.com.cn}}</ref>。国安俱乐部同多特蒙德俱乐部一起与德国Zakspeed赛车机构合作参加比赛。车手方面，俱乐部选择了赛会推荐的意大利年轻车手，只有22岁的[[达维德·里贡|达维德·里贡]]<ref name="国安“1号”神似耶利奇 里贡愿让中国球迷成为车迷">{{cite web|url=http://csl.cnsoccer.titan24.com/08-09-03/111604.html|title=国安“1号”神似耶利奇 里贡愿让中国球迷成为车迷|accessdate=2008-11-28|work=体坛周报}}{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，而由于赛会规定每个车队必须拥有一名本国车手，所以国安俱乐部选择了一名中国人作为自己的试车手<ref name="超级联赛方程式首站 国安力压欧洲诸豪门勇夺冠军">{{cite web|url=http://csl.cnsoccer.titan24.com/08-09-02/111516.html|title=超级联赛方程式首站 国安力压欧洲诸豪门勇夺冠军|accessdate=2008-11-28|work=体坛网}}{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，同时将自己的赛车命名为“绿色狂飙”<ref name="国安集团祝贺北京国安“绿色狂飙赛车”勇得冠军">{{cite web|url=http://www.fcguoan.com/news/index.aspx?id=081030104526741e3ecd|title=国安集团祝贺北京国安“绿色狂飙赛车”勇得冠军|accessdate=2008-11-28|work=体坛网}}{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。俱乐部车队在超级联赛方程式的首场分站赛中夺得冠军<ref name="超级联赛方程式首站 国安力压欧洲诸豪门勇夺冠军"/>，并且在随后的比赛中积分一直处于领先，最终在赛季结束时以413分的总成绩成为超级联赛方程式历史上第一个总冠军<ref name="SF General Standings"/>。赛后，[[中信国安集团公司|中信国安集团公司]]对俱乐部能够取得这样的成绩给予了较高的评价<ref name="国安集团祝贺北京国安“绿色狂飙赛车”勇得冠军"/>。

由于第一年度超级联赛方程式的比赛竞争激烈，具备了一定的影响力，2009年的比赛预计会有更多的俱乐部参与。而中超联赛[[上海申花足球俱乐部|上海申花足球俱乐部]]很有可能成为另外一支参赛的中国足球俱乐部<ref name="国安第二战场压AC米兰夺冠 联赛方程式申花欲09出战">{{cite web |url=http://sports.sina.com.cn/j/2008-11-24/10274088838.shtml|title=国安第二战场压AC米兰夺冠 联赛方程式申花欲09出战|accessdate=2008-11-28 |work= 足球报}}</ref>。
