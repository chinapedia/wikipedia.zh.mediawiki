《'''超危險特工'''》（{{lang-en|'''''RED'''''}}），是一部[[美國|美國]][[好萊塢|好萊塢]][[電影|電影]][[動作片|動作片]]，於2010年10月[[15|15]]日首映，改編自[[DC漫畫|DC漫畫]]的一套同名[[漫畫|漫畫]]。電影由曾執導《高凶三萬尺》、《時光旅的戀人》[[導演|導演]]{{link-en|勞勃·史溫凱|Robert Schwentke}}（Robert Schwentke）負責。演員包括[[布魯斯·威利|布魯斯·威利]]、[[海倫·米蘭|海倫·米蘭]]、[[摩根·費里曼|摩根·費里曼]]、[[約翰·馬可維奇|約翰·馬可維奇]]、[[瑪麗-露易斯·帕克|瑪莉·露易絲·派克]]和[[卡爾·厄本|卡爾·厄本]]。

== 故事 ==
法蘭克莫西斯（Frank Moses；[[布魯斯·威利|布魯斯·威利]]飾演）是一名自[[CIA|CIA]]退休的前[[特工|特工]]，被CIA的分類為'''RED'''，即為'''已退休而極度危險的'''（'''R'''etired, '''E'''xtremely '''D'''angerous），他感到退休生活十分無趣沒有變化，唯一的樂趣是每月撕毀養老金[[支票|支票]]藉故與養老金管理人莎拉羅斯（Sarah Ross；[[Mary-Louise_Parker|瑪莉·露易絲·派克]]飾演）在電話裡的談天說地時間，且法蘭克對莎拉在心中漸生情愫；一日法蘭克家中突然闖入了數名傭兵，想要將法蘭克滅口，法蘭克憑著過去的矯健身手將傭兵團擊退後，發現攻擊的背後有著巨大的陰謀存在；因此為了保護所愛的人，法蘭克決定召集過去的老戰友們，來對抗陰謀的幕後黑手並揭發真相……

== 續集 ==
{{main|超危險特工2：狠戰}}
由於《'''猛火爆'''》全球收入超過2億美元及成為2011年度美國出租電影冠軍, 故製作電影公司[[獅門娛樂|獅門娛樂]]已決定開拍續集，預計美國上映時間為2013年8月2日。除[[摩根·費里曼|摩根·費里曼]]因所飾演的角色陣亡而無戲份之外, 其他主角將會悉數回歸。更獲得奧斯卡金像獎女星[[凱薩琳·麗塔瓊斯|凱薩琳·麗塔瓊斯]]（Catherine Zeta-Jones）及韓國影星[[李炳憲|李炳憲]]加盟。另外奧斯卡金像獎男星[[安東尼·鶴健士|安東尼·鶴健士]]正洽談中。<ref name=Hollywoodreporter>{{cite web |url=http://www.hollywoodreporter.com/heat-vision/catherine-zeta-jones-red-2-bruce-willis-helen-mirren-dean-parisot-323126 |title=Catherine Zeta-Jones Joins Bruce Willis, Helen Mirren in 'RED 2' |work=[[The_Hollywood_Reporter|The Hollywood Reporter]]|accessdate=22 June 2012}}</ref>
