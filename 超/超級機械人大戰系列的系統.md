
'''超級機械人大戰系列的系統'''是「[[超級機械人大戰系列|超級機械人大戰系列]]」遊戲的系統與遊戲進行方面的解說。

* 遊戲類型不同的『Scramble Commander』系列、『Link Battler』、『無限邊疆』、『機戰學園』的系統則不會在此提及。
* 因為『[[超級機械人大戰NEO|超級機械人大戰NEO]]』的戰鬥系統和其他作品的系統性質有很大差別，下述的解説可能有一部分不適用於該作。該作品的戰鬥系統詳細請參閱該作頁面。

== 遊戲流程 ==
遊戲本編以「章」來分割。各章開始時，會像TV動畫一樣表示「話數 / 副題 」。一些根據原作的章會直接採用原作作品中使用的副題。每一章的構成如下所述。

# シナリオデモ（Scenario Demo／劇情）
#* 透過角色間的對話交代故事劇情發展。
# 戦闘マップ（Battle Field／戰鬥版面）
#* 選擇機體出撃。（每一戰鬥地圖有限定可出擊的機體數目）
#* 戰鬥：操作我方機體以達成勝利条件。
# シナリオエンドデモ（Scenario End Demo／戰後劇情）
#* 戰鬥完結，再次透過角色間的對話交代故事劇情發展。
# インターミッション（Intermission／整備畫面）
#* 進行機體改造、駕駛員育成、儲存等。
# 進入下一章

== シナリオデモ（Scenario Demo／劇情） ==
在劇情中各個角色為了作戰而召開作戰會議或進行一些雜談。基本上遇到敵襲就會中止對話並移往戰鬥版面。另外不只我方的對話，連敵人的對話也可以見到。

=== 路線分岐 ===
隨著劇情發展，會出現不同的選擇肢讓玩家選擇不同路線，戰鬥版面亦會隨選擇出現變化。另外，我方部隊也會被分割，只能操縱其中一條路線的部隊。作品中若有主人公的話，主人公的選擇將交由玩家決定。

== 戦闘マップ（Battle Field／戰鬥版面） ==
戰鬥版面的形狀由正方形組成的長方形版面構成，大小會隨不同的版面出現變化。戰鬥版面上的機體分為3種類。
; 戰艦
: 指揮官的機體。能夠收容我方機體，亦能像普通機體般進行移動、攻擊等行動。但是在『[[超級機器人大戰F完結編|超級機器人大戰F完結編]]』開始若戰艦被破壞就等於敗北（F完結篇前的作品中，若戰艦被破壞只需支付鉅額修理費但不會立即敗北）。
; 我方機體
: 玩家主要操作的機體。出撃機體及配置在一定程度上可以任意決定。
; 敵方機體
: 攻撃目標。在既定位置上展開，與我方機體一樣會進行攻撃。敵方部隊的戰艦型機體跟一般機體沒有分別，被擊墜也不會立即勝利。

基本上全滅敵方就勝利，但有時勝利條件會有不同。

除了部分作品外，戰鬥是以回合制進行的。兩軍階段完了等於一回合完了。為了達成勝利條件，玩家需要移動機體並進行攻擊（概括稱為「行動」），再全部機體完成行動後便完結我方階段。不過即使並非全部機體都行動完畢也能完結我方的階段。敵方階段採取的行動跟我方一樣。如此不斷重複，首先達成勝利條件的一方便是勝方。

基本上是自軍與敵軍2股勢力間進行戰鬥，但也有與第3以上的勢力戰鬥的時候。這個時候則會變成自軍→敵軍→第3勢力→自軍…的順序。

有些時候勝利條件不是全滅敵方，而是在特定回合數以內到達指定地點或擊破特定敵方機體等等。

== 駕駛員能力 ==

=== 性格 ===
不能在遊戲中直接確認的隱藏機體。每一駕駛員都會有此設定，並在我方機體被擊墜時對氣力增減產生影響。近年的作品中，在戰鬥時攻擊命中對方、受到攻擊・回避攻擊、撃墜敵人時也會對氣力的增減產生影響。

* '''弱氣''' - 我方機體被擊墜時氣力會下降。思想較負面的角色或普通的女性通常都是弱氣。
** 近年的作品中已不存在。
* '''普通''' - 我方機體被擊墜時氣力不會有變化。
* '''強氣''' - 我方機體被擊墜時氣力會上升。主人公角色通常都是強氣。
** 近年的作品中若攻擊命中・成功回避時氣力也會上升。
* '''超強氣''' - 我方機體被擊墜時氣力會大幅上升。熱血角色或高傲的角色多是超強氣。
** 近年的作品中若攻擊命中・成功回避時氣力也會大幅上升，但若攻撃被回避時氣力會下降。
基本的性格分為上述的4種。近年亦有除這4種以外的性格。
* '''慎重''' - 戰鬥時的氣力上升模式與強氣相似，我方機體被擊墜時氣力會下降。近年的作品才追加的性格。
* '''冷静''' - 若攻擊命中・成功回避時氣力會大幅上升，但若攻撃被回避或受到攻擊時氣力會下降（『OG』系列取消了這個缺點，但撃墜敵人時的氣力上昇值較低）。我方機體被擊墜時氣力也不會有變化。近年的作品中才被追加的性格。部隊的指揮官通常都是冷靜。
* '''敵（大物/超大物）'''、'''ボス''' - 無論任何情況都會提升氣力。近年的作品中才追加的敵方駕駛員專用的性格。

=== Ace Bonus（王牌獎勵） ===
駕駛員的撃墜數達一定程度時追加的獎勵要素。撃墜數達到一定數量時駕駛員能力欄裡的「A」圖案會發亮，部分作品中在达到更高的击坠数时可以点亮第二个「A」图案。『K』則只有在Intermission的頭3名才能成為王牌。

基本的王牌獎勵一律為初期気力由100變為105，双A时提高到110，以及撃墜敵人時的獲得資金會増加。在近年的作品中則會追加各式各樣的獎勵要素予王牌。『R』更會影響周回特典，並根據撃墜數的排位對駕駛員初期能力進行補正。
