'''湖广'''，一般是指[[湖北|湖北]]、[[湖南|湖南]]兩省，又稱'''兩湖'''。如[[湖廣總督|湖廣總督]]、湖廣[[安撫使|安撫使]]。

'''湖广'''有行政上和文化、语言上的两重含义。

== 行政 ==
行政上的湖广指位于[[中国南方|中国南方]]的“[[湖广行省|湖广行省]]”、“[[湖广省|湖广省]]”，为[[元朝|元朝]]和[[明朝|明朝]]时期直属[[中国|中国]]中央政府管辖的国家[[一级行政区|一级行政区]]。其包含[[湖北|湖北]]南部、[[湖南|湖南]]、[[重庆|重庆]]东南部、[[广东|广东]]北部[[韶关|韶关]]以北一带和[[广西|广西]]少部分地区，在一段时期中，湖广曾经与[[广东|广东]]、[[广西|广西]]、[[福建|福建]]南部行政区，即[[两广|两广]]合并，亦称大湖广。

# [[元朝|元朝]]一级行政区[[湖广等处行中书省|湖广等处行中书省]]简称“湖广（行省）”
# [[明朝|明朝]]一级行政区[[湖广等处承宣布政使司|湖广等处承宣布政使司]]简称“湖广（布政使司、省）”
# [[清朝|清朝]]“總督湖北湖南等處地方提督軍務、糧饟兼巡撫事”通称“[[湖广总督|湖广总督]]”

== 语言 ==
[[四川话|四川话]]、[[桂柳话|桂柳话]]等[[西南官话|西南官话]]受到了湖广方言的很大影响，其中由于[[湖广填四川|湖广填四川]]的缘故四川（含[[重庆|重庆]]）的人口很多来自湖广和[[两广|两广]]。明季河南人言，湖廣填四川實在是[[孝感|孝感]]填四川，孝感填四川實则是[[麻城|麻城]]填四川。
