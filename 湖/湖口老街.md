'''湖口老街'''是位在[[臺灣|臺灣]][[新竹縣|新竹縣]][[湖口鄉|湖口鄉]]的一條街道{{NoteTag|1=「湖口老街」不是別稱而是這條街道的正式名稱。}}，其中包括街頭、橫街、新街三條街{{r|知}}。其興衰與鐵路密切相關{{r|沈}}，這一帶因為[[劉銘傳|劉銘傳]]所鋪設的鐵路延伸至此，設有「大湖口火車碼頭」而繁榮，並在[[臺灣日治時期|日治時期]]整建成今日風貌，但後來因為鐵路路線調整，「[[湖口車站|湖口驛]]」移到下北勢而使街道商業衰退{{r|湖}}。街道上有新竹縣縣定古蹟[[湖口三元宮|湖口三元宮]]與歷史建築[[老湖口天主堂|老湖口天主堂]]，而湖口天主堂即是在舊車站原址上建造而成{{r|黃}}。近年來經由居民與政府社區總體營造之下，翻修老街牌樓、店亭、石板路與綠化工程，湖口老街有了新風貌{{r|知}}。

== 沿革 ==
[[File:Catholic_church_at_Hukou.jpg|thumb]]，因昭和四年（1929年）鐵路改線後所廢棄。]]
[[File:湖口老街豐瑞（左一）與興泉（左二）.jpg|thumb]]
[[File:拱圈內的湖口老街豐瑞（左一）與興泉（左二）（1）.jpg|thumb]]
湖口地區舊稱「大湖口」或「大窩口」，原是[[平埔族|平埔族]]竹塹社屯墾地，清乾隆時期開始有漢人進入開墾{{r|湖}}。此地因為是台地與山區的交界，而發展成貨物集散地{{r|黃}}，後來光緒十八年（1892年）鐵路鋪設到老湖口{{NoteTag|1=老湖口指舊車站所在地，也就是湖口老街一帶，至於新湖口則是指今[[湖口車站|湖口車站]]（位在仁勢村）一帶{{r|沈}}}}，更是促進了當地的發展。光緒十九年（1893年）正式通車後，新竹與基隆間每天約有六班車，平均每天載客量為400人{{r|湖}}。

後來進入日治時期後，日本大幅變動臺北到桃園間的鐵路路線，但桃園經大湖口到新竹間的路段則未有太大變化。後來明治四十一年（1908年）縱貫鐵路全線完工後，大湖口便成為了新竹北部的商業重鎮{{r|沈}}。由於「湖口驛」一帶日益繁榮，當地以羅氏家族為主的鄉紳與富商遂在大正五年（1916年）{{NoteTag|1=由於老街上有「建寅」、「寅春」等店號，因此有學者及地方人士認為老街是建於歲次甲寅的1914年。此處依《湖口老街：歷史空間與其遊憩資源系統之調查研究：湖口老街修復工程計畫》的說法：「老街建於1916年至1920年間。」}}開始在車站東南興建起今天的湖口老街{{r|沈}}{{r|湖}}，當時稱為「新街」，以與湖口地區已有的「老街」、「橫街」區分{{r|黃}}。大正七年（1917年）時，村民還配合新街走向重建了三元宮{{NoteTag|1=湖口三元宮是在明治三十七年（1904年）左右才從中崙三元宮分香出來，原本是坐東朝西，但後來發現與新街之間的整體動線有問題，遂在原地重建成今天坐南朝北的模樣{{r|湖}}。}}{{r|湖}}。

然而在昭和四年（1929年），日人認為[[楊梅區|楊梅]]到湖口間路段坡度過陡且路基不穩，便將鐵路路線改從楊梅以西丘地，途經[[伯公岡臺地|伯公岡]]（富岡）而到下北勢的新站。自此人潮便流向數公里遠的新湖口，老湖口因而沒落，但老街的風貌也因此完整保留{{r|沈}}。

近年來透過[[營建署|營建署]]、[[文建會|文建會]]等政府單位的援助，湖口老街居民統合地方力量並積極於[[社區總體營造|社區總體營造]]，目前已完成管線地下化以及整體綠化工程，並且把地面翻修為成適合步行的石板路，而且老街牌樓以及店亭也整修完畢{{r|知}}。老街居民因取得天主教會信任，長期借用教堂的部分空間，以傳統宗教公益組織架構，設立了[[大窩口促進會|大窩口促進會]]{{r|知}}。

== 街景建築 ==
湖口老街以[[湖口三元宮|三元宮]]為轉折點，主要部分呈現「L」型。短邊位於廟北，長約120公尺；長邊位於廟西，長約640公尺{{r|湖}}。建築多為磚造牆面，正面一樓為單拱，二樓則開有三扇拱型或方形的窗戶{{r|沈}}。在臺灣老街之中，此處的騎樓與拱圈較為寬廣，約有4到5公尺，而其外觀裝飾不多，可能與當地居民多為[[客家人|客家人]]有關{{r|湖}}。

== 人文風情 ==
當地的宗教信仰主要為在老街尾端的三元宮，主祀[[三官大帝|三官大帝]]，還供奉有[[觀音菩薩|觀音菩薩]]、[[媽祖|媽祖]]、[[註生娘娘|註生娘娘]]與[[福德正神|伯公]]（即[[土地公|土地公]]）等神{{r|沈}}。另外還有有幾座不同姓氏的家廟與天主教堂{{r|旅}}，[[天主教|天主教]]因教友人數減少，天主堂便在1993年停止活動，並閒置下來{{r|文}}。直到後來大窩口促進會獲得[[文建會|文建會]]補助下，將閒置的天主堂改建成文物館{{r|窩}}。
