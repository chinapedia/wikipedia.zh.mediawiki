{{湖西線RDT}}
'''湖西線'''（{{jpn|j=湖西線|hg=こせいせん|rm=Kosei sen|f=y}}）是一條連結[[京都府|京都府]][[京都市|京都市]][[山科區|山科區]]的[[山科站|山科站]]，途經[[琵琶湖|琵琶湖]]西岸至[[滋賀縣|滋賀縣]][[長濱市|長濱市]]的{{link-ja|近江鹽津站|近江塩津駅}}，屬於[[西日本旅客鐵道|西日本旅客鐵道]]（JR西日本）的鐵路線（幹線）<ref name="sone 15">[[#sone42|『歴史でめぐる鉄道全路線 国鉄・JR』42号 15頁]]</ref>。

== 概要 ==
因為經過[[琵琶湖|琵琶湖]]西岸地區而命名為湖西線<ref>與之相對的東海道本線大津站－米原站間（[[琵琶湖線|琵琶湖線]]一部分）也稱為湖東線，自從昭和30年代已經有類似說法。</ref>。起點雖然是山科站，但列車都[[直通運行|直通運行]]至[[京都站|京都站]]。[[永原站|永原站]]以南為[[都市網路|都市網路]]。與[[北陸本線|北陸本線]]和[[東日本旅客鐵道|東日本旅客鐵道]]（JR東日本）[[信越本線|信越本線]]、[[羽越本線|羽越本線]]、[[奥羽本線|奥羽本線]]構成[[日本海縱貫線|日本海縱貫線]]。

連結[[大阪|大阪]]和[[北陸地方|北陸地方]]的[[特別急行列車|特急列車]]，連結[[西日本|西日本]]日本海側各地和[[北海道|北海道]]的[[日本貨物鐵道|日本貨物鐵道]]（JR貨物）[[貨物列車|貨物列車]]皆經過本路線，較經[[琵琶湖線|琵琶湖線]]到[[米原車站|米原車站]]再轉進北陸本線省下約20公里路程。沿線亦是往[[京阪神|京阪神]]方向的通勤、通學路線。

全區間由JR西日本{{link-ja|西日本旅客鐵道近畿統括本部|西日本旅客鉄道近畿統括本部|近畿統括本部}}管轄，根據{{link-ja|旅客營業規則|旅客営業規則}}列為[[大都市近郊區間_(JR)|大都市近郊區間]]的「[[大都市近郊區間_(JR)#大阪近郊區間|大阪近郊區間]]」，可以使用[[IC卡|IC]][[儲值車票|儲值車票]]<ref>[http://www.jr-odekake.net/icoca/area/map/kinki.html ご利用可能エリア 近畿圏エリア｜ICOCA：JRおでかけネット] - 西日本旅客鐵道</ref>。

2016年12月20日，執政的[[自由民主黨_(日本)|自由民主黨]]敲定[[北陸新幹線|北陸新幹線]][[敦賀車站|敦賀車站]]以西至[[新大阪車站|新大阪車站]]採用「小濱、京都路線」，因此JR西日本有意在此段新幹線完工通車時，將並行的湖西線移交給滋賀縣接管，轉型為[[日本第三部門鐵道|第三部門鐵道]]，但滋賀縣各自治體反對此計劃<ref>[http://www.sankei.com/west/news/161220/wst1612200088-n1.html 【北陸新幹線】小浜京都ルート正式決定　与党ＰＴ「早期開通へ財源確保」] - 産経新聞、2016年12月20日</ref>。

== 沿線概況 ==
為設計成行走關西與北陸之間的短途高速鐵道，湖西線上完全沒有[[平交道|平交道]]，全部區間幾乎都是[[隧道|隧道]]及[[高架橋|高架橋]]，最小[[曲率|曲線半徑]]原則上為約1400公尺（有例外），[[坡度|坡度]]亦在千分之19以下，是高規格的在來線。除了因通過[[園城寺|三井寺]]及[[白鬚神社|白鬚神社]]範圍（雖然是以隧道形式）的補償爭議外，區外的反對及[[江若鐵道|江若鐵道]]（後述）的關係亦令路線決定有困難之處；

計劃時因預計[[東海道本線|東海道本線]]的使用量將到達上限，所以打算由山科 - [[片町線|片町線]][[長尾車站_(大阪府)|長尾車站]]之間建造一條新線，貨運列車在新設的長尾調車場，特急列車則由尾町線經由[[JR東西線|片福連絡線]]進入大阪。為此，山科車站西側須要預留興建支線位置（西側支線因當年國鐵財政惡化而擱置）。

現在[[雷鳥號列車|雷鳥號]]等列車，在湖西線上幾乎全線以130km/h行走。還有，在考慮剎車性能後，只能行走120km/h的[[日本國鐵485系電力動車組|485系電力動車組]]「雷鳥」，在沒有平交道的湖西線上，可以將行走速度提高至130km/h。雷鳥號行駛於京都～敦賀之間，不停站直達下行列車需時約51分鐘，上行則是53分鐘。

為使成為高速及舒適的鐵道線，由[[日本國有鐵道|日本國有鐵道]]（國鐵）時代開始，著手於行駛湖西線的列車[[日本國鐵381系電力動車組|381系]]、[[JR西日本221系電力動車組|221系]]、[[四國旅客鐵道|四國旅客鐵道]]（JR四國）[[JR四國8000系電力動車組|8000系]]進行車速提昇試驗。

==== 新快速 ====
{{See also|新快速#JR西日本|京阪神快速}}
[[FILE:JR_West_223_series_EMU_051.JPG|thumb]]

開業時午間1小時1班的新快速由大阪方向至堅田站直通、1986年11月1日延長至近江舞子站（湖西線區間各站停車）、1996年3月16日延長至近江今津站。

新快速只於午間1小時1班運行。近江舞子站 - 敦賀站間為各站停車、近江舞子站 - 近江今津站各站與後述的普通組合成約1小時有2班列車停車。

==== 快速 ====
{{See also|京阪神快速}}
為增加便利性，1996年3月16日起，朝早繁忙時間加開近江今津發車行駛至大阪的快速列車。1997年3月8日起，黃昏繁忙時間加開京都發車行駛至永原的快速列車。2002年3月23日起，上下行快速列車增停雄琴溫泉站。2004年10月16日起，京都站 - 大阪站間變更為新快速。2006年10月21日延長行駛區間變更為敦賀站到開。平日朝早繁忙時間敦賀發姫路行、黃昏繁忙時間大阪發敦賀行（京都站 - 大阪站、姫路站間為新快速）。

湖西線内新快速與快速的差別，只有在雄琴溫泉站有無停車這一點。

==== 普通 ====
普通車為運行區間内全部車站皆停車。

午間時間帶為京都站 - 近江舞子站間1小時3班、近江舞子站 - 近江今津站間1小時1班運行。

在沒有新快速運行的黃昏以後，京都站 - 近江舞子站間每小時4班、近江舞子站 - 近江今津站間每小時2班運行。

== 歷史 ==
湖西線是作為大阪、京都與北陸方面連結的捷徑而建設，比繞經[[東海道本線|東海道本線]]至[[米原車站|米原車站]]進入[[北陸本線|北陸本線]]可節省時間。原來的[[濱大津站|濱大津站]] - 近江今津站間有地方資本的[[江若鐵道|江若鐵道]]開業，計劃路線時並行的江若鐵道成為問題點而決定廢止。最終江若鐵道於1969年10月鐵道事業廢止後，改名[[江若交通|江若交通]]。

=== 年表 ===
==== 湖西線開業前 ====
* 1922年（[[大正|大正]]11年）4月11日：濱大津 - 三宅（現在的小濱線[[上中站|上中站]]）間編入國鐵予定線。
* 1962年（[[昭和|昭和]]37年）5月31日：濱大津 - 塩津間成為國鐵調査線。
* 1964年（昭和39年）6月25日：山科 - 沓掛間昇格為國鐵工事線。
* 1966年（昭和41年）12月28日：山科 - （暫稱）北大津（現在的大津京站）間、近江今津 - 近江塩津間第一次認可。
* 1967年（昭和42年）1月12日：湖西線開工儀式於[[皇子山總合運動公園|皇子山總合運動公園]]舉行。
* 1969年（昭和44年）11月1日：江若鐵道線濱大津站 - 近江今津站間廢止。
* 1974年（昭和49年）6月13日：[[中央控制行車|列車集中制御裝置]]（CTC）導入<ref>『近畿地方の日本國有鐵道 - 大阪・天王寺・福知山鐵道局史』大阪・天王寺・福知山鐵道局史編集委員会、2004年、p.140, p.375。</ref>。

==== 湖西線開業後 ====
* 1974年（昭和49年）
** 7月20日：山科站 - 近江塩津站間（74.1km）開業。新快速至堅田站9往復、普通23往復、臨時列車12往復、近江今津站 - 敦賀站間3往復<ref>『近畿地方の日本國有鐵道 - 大阪・天王寺・福知山鐵道局史』大阪・天王寺・福知山鐵道局史編集委員会、2004年、p.141。</ref>。
** 10月1日：湖西線的貨物列車開始運行<ref>『近畿地方の日本國有鐵道 - 大阪・天王寺・福知山鐵道局史』大阪・天王寺・福知山鐵道局史編集委員会、2004年、p.375。</ref>。
* 1975年（昭和50年）3月10日：大阪站 - 北陸、東北間的特急與急行「立山」改經過湖西線。
