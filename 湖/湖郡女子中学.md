'''湖郡女子中学'''（Virginia School）曾是[[监理会差会|美南监理会差会]]在中国[[湖州|湖州]]设立的一所[[女校|女子学校]]。现位于今人民路东侧湖州市第一人民医院内北侧，为湖州市文物保护单位。

== 历史 ==
湖郡女中历史可追溯到清[[光绪|光绪]]四年（1878）在江苏省[[嘉定县|嘉定县]][[南翔镇|南翔镇]]创立的悦来书塾，后分男塾华英学堂和女塾文洁女塾（取自英文Virginia谐音）。1901年两校部分师生迁至湖州城内马军巷，1905年在[[海岛堂|海岛堂]]附近（今湖州市第一医院和湖州市第十二中学所在地域）建立新校舍，女塾位于西侧，中文校名改称湖郡女塾，英文校名未变。学校原只有小学部，1907年增设蒙养园（后改为幼稚园）和中学部，1927年更名为湖郡女子中学<ref>{{cite web|url=http://hzrb.hz66.com/news/hubaokuaixun/2013-10-16/34611.html|title=《湖郡女学与红楼》|accessdate=2014-04-16|work=陈连根|publisher=湖州日报|deadurl=yes|archiveurl=https://web.archive.org/web/20140416192152/http://hzrb.hz66.com/news/hubaokuaixun/2013-10-16/34611.html|archivedate=2014-04-16}}</ref><ref>{{cite web|url=http://www.hzwenxue.com/News_View.asp?NewsID=7931|title=《湖州海岛》|accessdate=2014-04-16|work=|publisher=湖州晚报|deadurl=yes|archiveurl=https://web.archive.org/web/20140416184915/http://www.hzwenxue.com/News_View.asp?NewsID=7931|archivedate=2014-04-16}}</ref>。湖郡女中和姐妹学校上海[[中西女中|中西女中]]、苏州[[景海女校|景海女校]]是监理会在中国开办的三大女校。

1952年12月，湖郡女中与监理会开设的另一所学校东吴大学吴兴附属中学合并，成立湖州初级中学，后发展为今[[湖州市第二中学|湖州市第二中学]]（高中部，1998年迁址学士路）和[[湖州市第十二中学|湖州市第十二中学]]（初中部，高中部迁出后更为现名，仍在原址办学），原湖郡女中校址于1956年划入浙江省湖州师范学校（今[[湖州师范学院|湖州师范学院]]），1997年学校迁址，又转让与湖州市第一人民医院。

== 现状 ==
原湖郡女中教学楼位于今人民路东侧湖州市第一人民医院内北侧，现存两幢东西相邻的两层西式建筑，外墙红褐相间，屋顶采用美国坡顶乡村别墅风格，西楼南侧门楣上镌刻有1905字样，现为医院医技楼（3号楼），为湖州市区仅存的大型西式建筑<ref>{{cite web |url=http://www.hz1y.com/index.php?m=Article&a=show&id=718|title=《湖郡女校“红房子”的故事》|accessdate=2014-04-16|work=王增清|publisher=湖州市第一人民医院-湖州师范学院附属第一医院}}</ref>。
