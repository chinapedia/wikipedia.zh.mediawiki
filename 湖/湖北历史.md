据文献记载，商代在湖北境内，有楚（今南漳荆山）、卢（今襄阳）、彭（今房县）、庸（今竹山）等封国。

== 春秋战国 ==
[[File:Wuhan_Chu_town.jpg|thumb]]]]
[[File:YellowCraneTower.jpg|thumb]]]]

春秋战国时期，'''湖北地区属于[[楚国|楚国]]'''，西部为楚文化的发源，东部大部分被[[沼泽|沼泽]]覆盖，是历史上著名的“[[云梦泽|云梦泽]]”。楚国是当时南方的大国，'''[[楚庄王|楚庄王]]'''曾经问鼎中原，同齐桓公、晋文公、秦穆公、宋襄公并称'''[[春秋五霸|春秋五霸]]'''。到战国时，楚国力更强，同齐、燕、韩、赵、魏、秦并称为'''[[战国七雄|战国七雄]]'''。楚国的京城长期位于今荆州——纪南故城。

== 秦汉 ==
[[秦朝|秦代]]统一中国后，要经湖北南下，因此成为交通要道，人口开始增加，沼泽经过排水逐渐成为良田。秦在今湖北地区设立郡县。为避秦始皇父亲的名讳，将楚改称荆山（今南漳）之"荆"。因此后来湖北就多称荆州，有时合称荆楚。

[[汉朝|汉代]]在湖北设置江夏郡和南郡，均属于天下13州中的荆州。

[[三国时代|三国时代]]，湖北地区是[[蜀汉|蜀汉]]和[[吴国|吴国]]必争之地。今天仍有众多的古三国遗迹，如赤壁古战场、襄阳隆中。

== 隋唐 ==
湖北在[[唐朝|唐代]]分属4个道：山南东道、淮南道、江南西道和黔中道，山南东道辖襄州、荆州、随州、郢州、复州、峡州、归州、房州、均州，治所在襄州；淮南道辖沔州（汉阳）、安州（安陆）、黄州、蕲州，治所在今江苏扬州；江南西道辖鄂州（武昌），治所在洪州（今江西南昌）；黔中道辖施州（恩施），治所在黔州（今重庆彭水）。

== 宋元 ==
[[北宋|北宋]]时期，湖北主要归属[[荆湖北路|荆湖北路]]和[[京西南路|京西南路]]。荆湖北路是以洞庭湖以北至荆山，西包沅澧二水之地，简称湖北路，湖北之名即由此始。荆湖北路辖江陵府（荆州）、鄂州（武昌）、复州、峡州、归州、安州（安陆）、汉阳军、荆门军，治所在江陵府。京西南路辖襄州、随州、郢州、房州、均州、光化军，治所在襄州。东南角设置了兴国军（阳新），属于江南西路（治所在洪州，今江西南昌）；西南角设置了施州，属于夔州路（治所在夔州，今重庆奉节）。

今天湖北东南部的[[黄州|黄州]]、[[蕲州|蕲州]]在北宋初期处于[[淮南路|淮南路]]，治所在今江苏[[扬州|扬州]]，后淮南路分为东西两路，[[黄州|黄州]]、[[蕲州|蕲州]]属于[[淮南西路|淮南西路]]；[[江南西路|江南西路]]辖治所在洪州（今江西南昌）管辖今天[[黄梅|黄梅]]部分土地；西南角[[恩施|恩施]]，[[利川|利川]]等地设置了施州，属于[[夔州路|夔州路]]。

北宋末年，由于北方金兵入侵，战乱不止，大批人口南迁，湖北逐渐繁荣，带来北方先进的[[文化|文化]]和[[农业|农业]]技术，湖北成为重要的粮食产地。
[[File:YUAN(South_China).jpg|thumb]]
[[元朝|元代]]的湖北大体以长江为界，分属[[湖广行中书省|湖广行中书省]]和[[河南江北行中书省|河南江北行中书省]]。湖广行省包括今日湖北的东南部、湖南、广西、海南和广东的西南角，在湖北东南部设有武昌路、兴国路（阳新）和汉阳府。[[武昌|武昌]]是湖广行省的省会，也是两湖流域农产品的最大集散地。湖北的大部分地区属于河南江北行省，辖襄阳路、黄州路、蕲州路、峡州路（宜昌）、中兴路（荆州）和德安、沔阳、安陆（钟祥）等府。

== 明朝 ==
[[明朝|明代]]设置湖广布政使司，驻扎武昌。湖广布政使司在今湖北辖武昌府、汉阳府、荆州府、襄阳府、黄州府、承天府（钟祥）、德安府（安陆）、郧阳府。

明末，汉水改道以后，新河道的北侧凹岸形成新兴的商业城镇——'''[[汉口|汉口]]'''，吸引了大批徽州、山西、江西各帮商人前来，经营盐、茶、木、药等业，街区沿汉水北岸绵延二十里，列为当时'''[[四大名镇|四大名镇]]'''之一。

明代，在湖北西北部的'''[[武当山|武当山]]'''，兴建了规模庞大的道教建筑群。

== 清朝 ==
[[清朝|清代]]康熙3年（1664年）分湖广布政使司为左右2个布政使司，康熙6年 （1667年），湖广左右布政使司分别改名为湖北布政使司和湖南布政使司，湖北、湖南两省从此定名，沿袭至今。湖北省会仍在武昌。清代的湖北省下辖武昌府、汉阳府、荆州府、襄阳府、黄州府、安陆府（钟祥）、德安府（安陆）、郧阳府、宜昌府、施南府（恩施）和荆门直隶州。

1860年[[第二次鸦片战争|第二次鸦片战争]]后，外国资本沿[[长江|长江]]上溯，汉口辟为商埠，并在旧市镇的下游设立了[[汉口英租界|汉口英租界]]。1895年到1898年，又陆续设立了[[汉口德租界|汉口德租界]]、[[汉口俄租界|汉口俄租界]]、[[汉口法租界|汉口法租界]]和[[汉口日租界|汉口日租界]]。另外，在1877年和1896年，[[宜昌|宜昌]]和[[沙市|沙市]]也相继开辟为商埠。

同一时期，[[洋务运动|洋务运动]]兴起，湖广总督[[张之洞|张之洞]]在湖北兴办大量洋务企业，包括汉阳铁厂、汉阳兵工厂等，使得湖北成为中国主要的工业省份之一。

== 中华民国 ==
{{main|湖北省 (中华民国)}}
[[File:Wuhan_13.jpg|thumb]]
1911年10月10日，[[辛亥革命|辛亥革命]]在湖北打响了第一枪（[[武昌起义|武昌起义]]），在湖北谘议局成立了武汉起义军政府。[[中华民国|中华民国]]成立。

[[中华民国|中华民国]]成立后，以[[汉口|汉口]]为中心，湖北的[[经济|经济]]一度迅速发展。但在[[抗日战争|抗日战争]]中，湖北经济又受到严重破坏。

1920年代，[[武汉三镇|武汉三镇]]是历次革命风潮的中心之一。包括京汉铁路二七大罢工、等。

== 中华人民共和国 ==
[[中华人民共和国|中华人民共和国]]成立后，原中南局设置在武汉，管辖湖北、[[湖南省|湖南]]、[[河南省|河南]]、[[江西省|江西]]、[[广东省|广东]]、[[广西壮族自治区|广西]]等六个省区，湖北又有较大的发展，武汉在20世纪60年代以前是中南地区的中心。
