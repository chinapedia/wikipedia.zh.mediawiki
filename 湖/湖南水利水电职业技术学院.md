'''湖南水利水电职业技术学院'''（{{lang-en|''' Hunan Hydroelectric Occupational College '''}}）位于[[湖南省|湖南省]][[长沙市|长沙市]]，为湖南省的一所高级[[专科大学|专科大学]]。<ref name="nsa ">{{cite web|url=http://www.hnslsdxy.com/faimages/xygk.asp|title=湖南水利水电职业技术学院概况|accessdate=2011年|work=|publisher=湖南水利水电职业技术学院|language=zh-cn|deadurl=yes|archiveurl=https://web.archive.org/web/20120320132307/http://www.hnslsdxy.com/faimages/xygk.asp|archivedate=2012-03-20}}</ref>

== 历史 ==
1980年，[[湖南省水利厅|湖南省水利厅]]创建了湖南省水利技工学校。

2005年，湖南水利水电工程学校更名为“湖南水利水电职业技术学院”。

== 学校院系 ==
湖南水利水电职业技术学院包含4个系部。
*水利工程系、水利建筑工程系、电力工程系、经济管理系

== 学校文化 ==
* 校训：上善若水{{noteTag|1=上善若水出自《[[道德经|道德经]]》 }}，求真致远{{noteTag|1=致远出自《[[淮南子|淮南子]]》 }}
* 办学理念：依托水利行业办学，立足为区域经济服务，走特色办学之路
* 办学定位：立足湖南，面向全国，依托行业，面向社会，培养适应生产、建设、管理、服务一线需要的高素质技能型人才
* 办学特色：校企合作，订单式培养<ref name="nsa " />

== 荣誉 ==
教学工作优秀学校、毕业生就业先进单位、省级文明卫生单位、市级文明卫生单位、花园式单位、省直系统双文明单位、省直思想政治工作先进单位<ref name="nsa " />
