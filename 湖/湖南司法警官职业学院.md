'''湖南司法警官职业学院'''（{{lang-en|Hunan Judicial Officer  Vacational  College}}）位于[[湖南省|湖南省]][[长沙市|长沙市]][[远大二路|远大二路]]1069号，为湖南省一所高等[[专科大学|专科大学]]。<ref name="kls ">{{cite web|url=http://www.hnsfjy.cn/ws2004/SchoolIntroduce/ZhaoShengXinXi/|title=湖南司法警官职业学院学院概况|accessdate=2011年|work=|publisher=湖南司法警官职业学院|language=zh-cn|deadurl=yes|archiveurl=https://web.archive.org/web/20120325032047/http://www.hnsfjy.cn/ws2004/SchoolIntroduce/ZhaoShengXinXi/|archivedate=2012-03-25}}</ref>

== 历史 ==
1980年，[[湖南省|湖南省]]建立了“湖南司法警官学校”，是湖南司法警官职业学院的前身。<ref name="kls " />

2003年，“湖南司法警官学校”升级为高职院校，并且更名为“湖南司法警官职业学院”。<ref name="kls " />

== 学校院系 ==
湖南司法警官职业学院开设了7个教学系部，14个专业。<ref name="kls " />

*系：法律系、司法系、执行系、管理系

*部：政治课教研部、警体部、基础课部

== 校训 ==
厚德重法、崇文尚武<ref>{{cite web|url=http://www.hnsfjy.cn/ws2004/SchoolIntroduce/rule/|title=校训|accessdate=2011年|work=|publisher=湖南司法警官职业学院|language=zh-cn|deadurl=yes|archiveurl=https://web.archive.org/web/20120318044316/http://www.hnsfjy.cn/ws2004/SchoolIntroduce/rule/|archivedate=2012-03-18}}</ref>

== 对外合作 ==
湖南司法警官职业学院和[[湖南师范大学|湖南师范大学]]、[[西南政法大学|西南政法大学]]、[[湖南农业大学|湖南农业大学]]建立了合作伙伴关系。<ref name="kls " />

== 荣誉 ==
湖南司法警官职业学院先后被[[湖南省人民政府|湖南省人民政府]]及相关机构评选为：“全省司法行政系统教育工作先进单位”，并多次荣立集体三等功。先后被有关部门评为“教学管理优秀学校”、“学生管理先进单位”、“后勤管理先进集体、“食品卫生A级单位”，被授予“省级卫生文明单位”、“长沙市花园式单位”、“芙蓉区平安单位”。<ref name="kls " />
