'''湖南科学技术出版社'''是一家[[中国|中国]][[湖南省|湖南省]]的[[出版社|出版社]]，1979年成立于[[长沙|长沙]]，侧重出版科学类和技术类图书。

除主打的科学技术类图书外，该社出版的图书还涵盖医学、经管、生活及教材等领域，年出版新书近500种，并出版《医药界》杂志。其主推的《[[第一推动丛书|第一推动丛书]]》20多年来在科普图书市场长盛不衰。<ref name="官网概况页"/>该社也免费发行过一些由政府部门编辑的生活应急类读物。<ref>{{cite news |title=湖南科学技术出版社出版《防震减灾基础知识长沙读本》 |author= |url= http://www.changsha.gov.cn/xxgk/szfgbmxxgkml/czgldwxxgkml/sdzj/gzdt_8515/201312/t20131218_528596.html |publisher=长沙市政府门户网站 |language=zh-cn |date=2013年12月18日 |accessdate=2017年1月13日 |quote="湖南省地震局的领导和专家给予编写组大力支持和帮助，北京市地震局宣教中心为该书提供大量插图，使该书权威性强，图文并茂。该书首印一万册，将免费赠送给全市防震减灾示范学校、示范社区及广大市民。"}}</ref>

== 历史 ==
湖南科学技术出版社成立于1979年，2008年底变更为湖南科学技术出版社有限责任公司，是[[中南出版传媒集团|中南出版传媒集团]]下属[[子公司|子公司]]。现在的總部地址為湖南省长沙市湘雅路276号。<ref name="官网概况页">{{cite web |title=官方网站 - 企业概况介绍 |author= |url= http://www.hnstp.com/about_home.aspx |publisher=湖南科学技术出版社 |language=zh-cn |date=2015年 |accessdate=2017年1月3日}}</ref>

2008年，该社与湖南当地[[新华书店|新华书店]]合作开展电子音像出版业务，该社由此也成为湖南首家通过整合出版、发行资源设立的具备电子出版物、音像制品出版发行资格的企业。<ref>{{cite web |title=科技出版社联手新华书店 出版湘军添生力军 |author=刘胜军 |url= http://www.xwcbj.gd.gov.cn/news/html/gdxx/article/1217252152068.html |publisher=湖南[[红网|红网]], 广东省新闻出版广电局 (转载) |language=zh-cn |date=2008年7月29日 |accessdate=2017年1月3日 |quote="28日上午，由湖南科技出版社与湖南新华书店集团共同出资的湖南科学技术电子音像出版社有限公司正式挂牌成立，成为湖南出版发行产业新的生力军。该公司亦是湖南首家通过整合出版、发行资源设立的具备电子出版物、音像制品出版发行资格的企业。","湖南科技出版社是湖南首家具有图书、期刊、电子、音像制品综合出版资质和能力的现代出版机构，曾出版《第一推动丛书》、《时间简史》、《世界是平的》等畅销全国的书籍，在科技类图书出版社中属佼佼者。","公司成立后将充分发挥湖南科技出版社在科技类图书出版方面长期积累的资源和品牌优势，同时借助湖南新华书店强大的发行网络平台，为广大人民群众提供更多更好的精神文化产品。"}}</ref>

== 部分知名图书 ==
===时间简史===
[[斯蒂芬·霍金|斯蒂芬·霍金]]的《[[时间简史：从大爆炸到黑洞|时间简史：从大爆炸到黑洞]]》是具备影响力的一本科学理论的宣传读物。与许多普及读物不同，这本书本身并不够通俗易懂，但其市场营销却非常成功。20世纪90年代初，霍金的学生[[吴忠超|吴忠超]]亲自翻译了该书，并将该书推荐给中国大陆的数家出版社，但最终只有湖南科学技术出版社愿意出版该书。1992年，湖南科学技术出版社出版该书。因为中国读者对科技文化的兴趣不大，而且这本书也比较晦涩，出版社决定以“阅读霍金，懂与不懂都是收获”作为宣传词，开始了长达3年的宣传。到1997年，《时间简史》和霍金终于在中国普通读者中火了起来。霍金事后非常感谢湖南科学技术出版社，并指定该社作为其著作在中国大陆的唯一授权发行商。<ref name="霍金钦点">{{cite web |title=东方早报：畅销中的霍金 |author=陈佳 (东方早报记者) |url= http://tech.sina.com.cn/d/2006-06-22/08421002499.shtml |publisher=东方早报, 新浪网 (转载) |language=zh-cn |date=2006年6月22日 |accessdate=2017年1月3日 |quote="当霍金得知他的新书《时间简史》(普及版)一出版就受到了中国读者的欢迎，并登上了畅销书的排行榜时，当即表示，他的最大读者群一定是在中国，他愿意在出版社的安排下与中国读者对话，并委托湖南科学技术出版社出版他未来一切著作的中文版。","1988年，霍金的《时间简史》首版以来，已成为全球科学著作的里程碑，他本人也被称为“活着的爱因斯坦”，然而在我国却几乎无人知晓。上世纪90年代初霍金的学生吴忠超把该书推荐给我国几家出版社，却未被接受。1992年，湖南科学技术出版社出版该书后，以“阅读霍金，懂与不懂都是收获”的宣传词开始了长达三年的宣传，到1997年，《时间简史》和霍金终于在中国普通读者中火了起来，至2002年霍金来华时达到高潮。","“霍金作品有个奇怪的现象，似乎越难懂的卖得越好。”林澧波笑着说。"}}</ref>

霍金热也伴随了一些负面效应，大众对霍金本人的好奇超过了对内容求真的热情。少数与霍金作品实际关系不大的书籍也以霍金作为著作者的名字，为自己造势，而且读者还趋之若鹜。例如由王宇琨和董志道编著、北京联合出版公司出版的《图解时间简史》<ref>{{cite book|title=图解时间简史 |author=[[斯蒂芬·霍金|斯蒂芬·霍金]] (未授权), 王宇琨, 董志道 |isbn=9787550216495 |publisher=北京联合出版公司 |language=zh-cn |year=2013年}}</ref>——这本书为了迎合读者，把封面的“(英) 斯蒂芬·霍金 [原著]”字样放得很大，书中把霍金吹捧为“先知”，还说他“解决了第一推动力的来源问题”，还有一些科学家名字张冠李戴的低级错误。

===走近费曼丛书===
美国物理学家[[理查德·费曼|理查德·费曼]]是美国的科学理念推广者。[[比尔·盖茨|比尔·盖茨]]称赞说“很少有书在写了40年之后还值得人去读，只有少数的经典，像费曼的物理学或是[[理查德·道金斯|理查德·道金斯]]的那些书。”<ref>{{cite web |title=《自私的基因》:争议了40年的“科幻小说” |author=沈沣 |url= http://sf.kedo.gov.cn/stuff/culture/book/868600.shtml |publisher=北京晚报, 蝌蚪五线谱 (转载网站) |language=zh-cn |date=2017年1月10日 |accessdate=2017年1月3日}}</ref>台湾[[天下文化|天下文化]]出版社早在90年代初就翻译和出版了费曼所写的一系列大众读物，且其代表作《[[别闹了，费曼先生！|别闹了，费曼先生！]]》的中文版还引起了不错的反响<ref>{{cite web |title=天下25周年选出最具影响30本书 |author=江北 |url= http://www.taiwan.cn/sy/meititushu/200708/t20070802_438020.htm |publisher=出版之门, 中国台湾网 (转载网站) |language=zh-cn |date=2007年8月2日 |accessdate=2017年1月13日 |quote="另外，作家江才健所著的《杨振宁传》出版时登上《中国时报》头版。而《别闹了，费曼先生》是理科学生必读之书..."}}</ref>。相比之下，中国大陆引进费曼的通俗读物的时间则比较晚。湖南科学技术出版社从21世纪初开始陆续出版的《走近费曼丛书》是大陆首次集中出版的传播费曼文化的丛书。
