'''湖南女子学院'''始建于1985年的'''湖南女子职业大学'''，2010年升格为[[本科|本科]]院校，由[[全国妇联|全国妇联]]与湖南省人民政府共建。现有6个系，3个教学部，30多个专业。<ref>{{cite web |url=http://hunan.voc.com.cn/article/201310/201310261041058582.html |title=湖南女子学院增家政专业 本科生也去做保姆? |accessdate=2013 |publisher=湖南在线 |work= |language=zh-hans}}</ref>

== 沿革 ==
[[File:Hunan_Women's_University01.jpg|thumb]]

1985年，“湖南女子职业大学”建立，[[中共中央总书记|中共中央总书记]][[胡耀邦|胡耀邦]]亲自题写校名。

2010年3月18日，[[中华人民共和国教育部|中华人民共和国教育部]]批准它升格为全日制普通本科院校，并且加入了“[[世界女子教育联盟|世界女子教育联盟]]”。
