'''市来 光弘'''{{BD|1982年|1月10日|catIdx=Ichiki Mitsuhiro}}是一名[[日本|日本]]男性[[聲優|聲優]]。隸屬於[[Mausu_Promotion|Mausu Promotion]]。出生於[[鹿兒島縣|鹿兒島縣]]。血型為A型，身高為170公分。

==人物==
*2008年12月為止隸屬於[[元氣_Project|元氣 Project]]，於2009年2月1日起加入[[Mausu_Promotion|Mausu Promotion]]。
*為[[櫻井孝宏|櫻井孝宏]]擔任團長的劇團松井之門生，以Little 松井第二次成員的身分在各媒體活躍中（在2008年4月26日的播放中發表畢業宣言）。
*最近在網路廣播，以及[[埼玉電視台|埼玉電視台]]的節目『關口先生II』中演出萬田仁（万田ヒトシ），之後在接續其後的節目『玉 New Town（玉ニュータウン）』裡扮演Icchi（イッチィ）。
*主要演出少年、青年角色。
*聲優朋友中與[[杉田智和|杉田智和]]、[[羽多野涉|羽多野涉]]、[[鈴木達央|鈴木達央]]、[[增川洋一|增川洋一]]等人交情好，為杉田組的一員。
*非常偏食，最喜歡肉類和甜的食物，討厭一部分的蔬菜以及海鮮。
*喜歡玩遊戲『[[魔物獵人系列|魔物獵人系列]]』。常常在部落格寫有關遊戲的事。
*雖然是偶然，和[[福圓美里|福圓美里]]的出生年月日、血型、職業一樣，連過去的住所、母親的姓名也相同。也曾拿過同機型的手機<ref name="福圓美里的部落格_497"> 參照福圓美里的部落格 <br/> {{Cite web |date= 2009年1月9日(金) |url= http://misato.idt.jp/cgi-bin/sfs6_diary/sfs6_diary/200901.html#2009010949 |title= No.497「さらば26歳」 |work= ++Misato・Diary++ |author= 福圓美里 |accessdate= 2009-08-11 |deadurl= yes |archiveurl= https://web.archive.org/web/20090828113639/http://misato.idt.jp/cgi-bin/sfs6_diary/sfs6_diary/200901.html#2009010949 |archivedate= 2009年8月28日 }} </ref>。2009年1月9日在乙女企画クロジ☆劇團策劃的驚喜下兩人第一次一起過了生日<ref name="福圓美里的部落格_497"/>。
*2015年3月9日於Blog宣布與同為聲優的[[井上奈奈|井上奈奈]]結婚的消息<ref>{{Cite web |url= http://ameblo.jp/garoumow/entry-11999348230.html |title= ご報告 |date= 2015年3月9日 |accessdate= 2015年3月9日 }} </ref>。
*2017年6月15日初為人父。
