{{jp|c='''將棋'''|j=将棋|hg=しょうぎ|rm= shōgi |lead=yes |help=no}}，又称'''日本象棋'''、亦稱'''本將棋'''（本，基本之意），是一种盛行于[[日本|日本]]的[[棋類|棋類]][[遊戲|遊戲]]。

== 歷史 ==
將棋的始祖可能是中国的[[宝应象棋|宝应象棋]]或是從東南亞傳入的[[印度|印度]][[恰图兰卡|恰图兰卡]]，特別是與[[泰國象棋|泰國象棋]]的相似性。在日本最早的紀錄是[[平安將棋|平安將棋]]，之後分支成[[小將棋|小將棋]]、[[中將棋|中將棋]]、[[大將棋|大將棋]]、[[大大將棋|大大將棋]]、[[天竺大將棋|天竺大將棋]]、[[摩訶大大將棋|摩訶大大將棋]]、[[泰將棋|泰將棋]]、[[大局將棋|大局將棋]]等諸多[[日本將棋變體|日本將棋變體]]，傳說再由[[後奈良天皇|後奈良天皇]]改良小將棋，去除棋子'''[[醉象|醉象]]'''，成為現在流行的日本將棋。

== 規則 ==
=== 棋盤 ===
將棋的棋盤是一個由10條橫線及10條直線相交的方格陣，而棋子則置於方格之內，也就是9列9行的棋盤。
靠近自己的3列是本陣，遠離自己而靠近對手的3列是敵陣。

=== 棋子 ===
將棋的棋子呈鐘形，前端較尖。和[[中國象棋|中國象棋]]及[[國際象棋|國際象棋]]不同，將棋是以棋子前端的指的方向來區別所屬。將棋共有八種棋子，依據棋子重要性和強度棋子形狀有不同大小。每種棋子均有獨特的簡稱及走法，分述如下：

=== 開局配置 ===
當一局將棋開始時，雙方各有二十隻棋子，分佈如下：

开局摆子的方式，可分为'''大桥流'''和'''伊藤流'''两类。大桥流是先从底线正中的玉将（王将）开始，由内向外依次摆入金将、银将、桂马、香车（先左后右交替），然后依次放角行、飞车于次底线，最后在其前方从正中起，由内而外，先左后右交替放置步兵；伊藤流则是玉将（王将）到桂马同大桥流，而接下来从左往右依次放置步兵，回到底线先左后右放两枚香车，最后依次放角行、飞车。

== 棋譜記法 ==
日式的記法通常只記錄棋子移動後的座標，先寫'''筋'''（相等於平面直角坐標系x軸的「橫坐標」，或是中國象棋的「路」／「直線」），再寫'''段'''（相當於平面直角坐標系y軸的「縱坐標」，或是中國象棋的「橫線」），最後寫棋子名稱。

平面坐標以先手方看，「橫坐標」是從最右行到最左行的「直排」，用阿拉伯數目字表示，為1到9；「縱坐標」是從最上列到最下列的「橫排」，用漢字表示，為一到九。第一至第三段屬於後手方的'''自陣'''／先手方的'''敵陣'''，第七至第九段屬於先手方的自陣／後手方的敵陣。

有別於中國象棋，將棋一方走一著棋為之一「手」，雙方各走一著共計兩手，象棋五十回合即等於將棋一百手。若手數為單數時，即由先手走子，屬先手番，手數為雙數時，由後手走子，屬後手番。 

棋譜記錄先手著法以▲表示，後手著法以▽表示，舉例如下： 

１　▲２六步　　　２　▽３四步 
３　▲７六步　　　４　▽８八角成 
５　▲同　銀　　　６　▽２二銀 
７　▲５八金右　　 

每一手棋先記錄棋子移動後的位置，先寫筋，再寫段，最後記錄該棋子的名稱。譬如先手第一手推進'''飛車'''前的'''步兵'''，著法為「▲２六步」，後手推進角筋（該方'''角行'''位在的直行）上的'''步兵'''，著法為「▽３四步」，如此類推。 

如其中一手棋走到的位置與對方上一手棋所到的位置一樣並取去對方的棋子，則用「同」表示位置。 

如有一手棋走進敵陣並昇變為另一棋種， 則在該棋子加上「成」字，如「▲２一飛成」；若選擇不成，則加上「不成」二字，如「▽４七銀不成」。升級的棋子則用其新的名字記錄，如'''竜'''、'''馬'''、'''成銀'''、'''成香'''等。 

若將棋子'''打入'''到棋盤時，並無其他一樣的棋子可走到該格，則無需特別注釋，否則要加上一個「打」字，如「▲４五桂打」。參見'''打入及吃子'''一節。
 
另外，若遇到棋盤上多於一個相同兵種的棋子可走到同格，如先手分別位於４九及６九的'''金將'''均可則走到'''玉將'''前的５八位，則分別以各個棋子的位置、方向加以注釋，方法如下： 

* 棋子從左邊向右走，以「''左''」表示； 
* 棋子從右邊向左走，以「''右''」表示； 
* 棋子從後面向前直進，以「''直''」表示，但是对于步兵，以「''突''」表示； 
* 棋子從側面向前走，不論左右，均以「''上''」表示； 
* 棋子向橫移，以「''寄''」表示； 
* 棋子往後退，以「''引''」表示。

== 其他規則 ==
=== 升变 ===
將棋的棋子設有升变制度。除了'''王將'''（'''玉將'''）、'''金將'''及已經升变的棋子外，棋盘上所有棋子均可升变，若棋子剛打入敵陣，則要'''多走一步'''才可升級。

當一枚可升級的棋子移進、移出、或者在敵陣（位於距離棋手最遠的三行）中移動時，棋手可以選擇把該棋子翻轉升級或保持原狀，但是如果該棋子不升級的話會沒有辦法再走（例如走到敵陣的底線的步兵）則該棋子會被迫升級。

=== 吃子及打入 ===
日本將棋最特別的是棋手可以花一手將己方吃掉的棋子放回棋盤成為己方棋子，這在日文稱為'''打入'''，大幅提高了將棋的複雜度。當一枚已升級的棋子被吃時，它的升級會被取消，打入時用原先的棋種表示。

打入有四項限制：
# 剛打入的棋子即使落在敵陣亦不能馬上升級，一定要以後多下一手才行。
# 不能把被吃的棋子打入在一些不能再走的位置。例如'''步兵'''、'''香車'''等都不能落在敵陣的底線；'''桂馬'''不能落在敵陣的底線及次底線。
# 步兵不能打入於已有己方（未升級的）步兵的一路，這一種犯規稱為'''同筋二步'''。
# 步兵打入時不能立刻''將死''對方，這一種犯規稱為'''打步詰'''；反之若不會立刻將死則不犯規，此稱為'''打步將'''。

=== 犯規 ===
在將棋當中，在不能升級的情況下為棋子升級、重覆四次局面的連將、將兩個己方步兵置於同一縱路（日本略稱「二步」）、破壞打入的限制、連下多於一手、對方王手時沒有「應將」、或移開自己的棋子令到自己的王將自動被王手、違反棋子的允許行法（如角行不依斜線走）、超時、打步诘等皆屬犯規，其中以「二步」在日本的將棋職業賽中最為常見，自一九七七年迄今，在日本的將棋職業賽中已有44次「二步」的記錄。

=== 將、將死 ===
如能下了能擒拿對方王將的一手，亦即象棋中的'''將軍'''，則稱為'''王手'''（{{lang|ja|おうて}}）。被反將則是稱為'''逆王手'''（{{lang|ja|ぎゃくおうて}}）。王將不能逃脫，則此手構成'''詰み'''（{{lang|ja|つみ}}），亦即象棋的''將死''。將死對方的玩家勝出此局。

若下了一手沒有'''王手'''，但對方無論如何防禦都會在下一手被將死，則稱為'''必死'''（{{lang|ja|ひっし}}），相當於象棋的'''絕殺'''。

將棋容許''連將''但是不能重複同樣的手法長照（長將），若雙方重複循環同樣的方式照將、應照達四次時，則照將方違規，判負。
舉例說明，若甲方以飛車照將，乙方把玉將往左移一格，甲方又將飛車右移一格照將，乙方又把玉將往右移一格，甲方又將飛車左移一格照將......如此重複循環到第四次時，則甲方違規，判負。此規則在日本稱為「{{lang|ja|連続王手の千日手}}」。

=== 將棋對局的結束 ===
除了''將死''外，將棋尚可於以下情況結束：
* 其中一方行違規步時，犯規的玩家在正式比賽中會被直接判為敗北。
* 雙方重複循環同樣的著法，使得局面沒有進一步變化達四次時，則視為和局。此規則在日本稱為「千日手」。
* 如雙方的'''王將'''均於升級區內（相入玉），且無法將死對方時，對局結束。雙方需為自己擁有的棋子計分，計分方法為：'''王將'''不計分、'''角行'''及'''飛車'''各計五分，其他棋子各計一分。升級的棋子不會額外加分。如只有一方得分少於 24 則少分的一方判為負，否則不分勝負。

在正式比賽中如有不分勝負的情況，雙方需重賽並對調行子先後次序 (以至減少時限)。業餘比賽則一般判和了事。

=== 讓子 ===
為了讓相對較弱的棋手也有獲勝的機會，較强的棋手有時會讓子（日文稱為'''駒落'''）。在讓子的情況下，讓子方（即'''上手'''）必須在開局時永久除去自己一部分的棋子（即不能以任何方式將之放上棋盤）。不過，在國際賽上，讓子並不會因為兵力上的差異而為相對較弱的棋手造成明顯的優勢。較常見的讓子局會按照嚴重程度遞增，現臚列如下：
* 讓香局與平局（日文稱為'''平手'''）交替連下（日文稱為'''半香'''）
* 除去'''左旁香車'''（日文稱為'''香落'''或'''左香落'''）
* 除去'''角行'''（日文稱為'''角落'''或'''角行落'''）
* 除去'''飛車'''（日文稱為'''飛落'''或'''飛車落'''）
* 除去'''飛車'''與'''左旁香車'''（日文稱為'''飛香落'''）
* 除去'''飛車'''與'''角行'''（日文稱為'''二枚落'''）
* 除去'''飛車'''、'''角行'''與'''左旁香車'''（日文稱為'''三枚落'''）
* 除去'''飛車'''、'''角行'''與'''香車兩乘'''（日文稱為'''四枚落'''）
* 除去'''飛車'''、'''角行'''、'''香車兩乘'''與'''桂馬一匹'''（日文稱為'''五枚落'''）
* 除去'''飛車'''、'''角行'''、'''香車兩乘'''與'''桂馬兩匹'''（日文稱為'''六枚落'''）
* 除去'''飛車'''、'''角行'''、'''香車兩乘'''、'''桂馬兩匹'''與'''一名銀將'''（日文稱為'''七枚落'''）
* 除去'''飛車'''、'''角行'''、'''香車兩乘'''、'''桂馬兩匹'''與'''兩名銀將'''（日文稱為'''八枚落'''）
* 除去'''飛車'''、'''角行'''、'''香車兩乘'''、'''桂馬兩匹'''、'''兩名銀將'''與'''一名金將'''（日文稱為'''九枚落'''）
* 除去'''飛車'''、'''角行'''、'''香車兩乘'''、'''桂馬兩匹'''、'''兩名銀將'''與'''兩名金將'''（日文稱為'''十枚落'''）
* 除去'''飛車'''、'''角行'''、'''香車兩乘'''、'''桂馬兩匹'''、'''兩名銀將、兩名金將'''與'''全部步兵'''（日文稱為'''裸玉'''）

其他的讓子局偶然會出現。最嚴重的情况，是讓子方在盤面只有一枚'''王將'''在正常位置，而駒台則有三枚'''步兵'''（日文稱為'''步三兵'''）。

在將棋不同的機制之下，讓子嚴重程度與棋手等級差異之間的關係，並無公認的標準。但一般來說（尤其是對業餘者而言），讓子的嚴重程度應該約相等於雙方等級之間的差距。例如差一個等級則讓香，差兩個則讓角，如此類推。

== 詰將棋 ==
將棋的排局在日文稱為{{lang|ja|'''詰め將棋'''}}，過程中必須不斷'''王手'''，不可'''千日手'''（相當於中國象棋的長將、長捉、長攔等不斷重複局面的僵持狀態），最末將死對方（若有特別聲明可包括'''必至'''），相當於中國象棋的連將殺局。由于存在“王将”以及“玉将”两种，詰將棋不一定要將雙方的'''玉將'''都擺上，通常只擺防守的那一方的。因為「王」字有時難以分辨上下，故在只有一個王將的排局中，例必把'''玉將'''擺上。有玉將的一方叫做'''玉方'''（或'''受方'''），摆王将的一方叫做'''攻方'''（或'''詰方'''）。一定是攻方先行。{{cn}} 排局一般只會標明攻方的持子，除非有特別聲明，否則假設玉方持有餘下沒標明的棋子（當然王將除外）。雙方都有玉將的排局則叫做'''雙玉'''。詰將棋一般只顯示棋盤右上角（即１一位附近）有關的部份。

詰將棋的取勝手數，是以攻守雙方的最佳着法計算。攻方的最佳着法，是指攻方能保證在若干最短的手數內將死玉方的一系列對策；而玉方的最佳着法，是指玉方能保證在若干最長的手數內保持不敗的一系列對策。由攻守雙方的最佳着手所構成的進程，就是棋局的'''正解'''。譬如說，「五手詰」的意思是指，當攻方在第一手下出最佳着法之後，無論以後玉方如何應，都必會在第三或第五手落敗；同理，當玉方在第二手下出最佳着法之後，玉方都有辦法在第四手保持不敗。攻方的最佳着法不包括重複局面的廢着；玉方的最佳着法不包括不能改變結局的打入。若攻方沒有下出最佳着法，但因玉方同樣沒有下出最佳應手而讓攻方其後獲勝，則稱為'''紛れ'''，意為僥倖。當攻方一直下出最佳着法，但玉方沒有下出最佳應手、或下出正解以外的最佳應手，則稱為'''變化'''。變化的手數一定不會長過正解。當玉方一直下出最佳應手，但攻方在某一手可以有正解以外的着法保證獲勝，則稱為'''餘詰'''。若無論攻方如何出手，玉方都有辦法逃避敗局，則稱為'''不詰'''。攻方的'''持駒'''（不論是一開始標明抑或中途吃進）在正解的最後一手尚未用完，或正解以外有餘詰或不詰，則稱為'''不完全作'''。

{{Fact|詰將棋具增進將棋功力的實用價值，亦具藝術價值，即使是三手詰也可以很有趣味。因只擺上一方的王將，故此玉方只能守或逃，不能反攻，比起中國象棋排局崇尚和局的剛柔並濟之美，詰將棋純粹剛猛、緊湊。象棋和將棋的妙趣，實乃各有千秋。詰將棋在江戶時代就大量出現了相關書籍，其中的代表作包括'''將棋無雙'''及'''將棋圖巧'''。}}

日本民間亦有像中國民間藝人依擺棋攤為生的江湖排局，稱作'''大道詰將棋'''，其中尤以'''香步問題'''最為廣泛研究。

== 发展 ==
[[File:Shogi_Koma_Ryoko.jpg|thumb]]
{{Fact|在[[中国大陆|中国大陆]]，[[青少年|青少年]]主要从接触到的[[日本漫画|日本漫画]]对日本将棋有些许了解。}}
1994年，中国[[北京|北京]]开始有青少年活动中心教授日本将棋。1995年10月19日～20日，在中國北京市[[京倫饭店|京倫饭店]]举行了第八届“龙王杯”日本将棋决赛的第一局比赛（决赛为七局四胜制），由当时的[[羽生善治|羽生善治]]竜王・六冠对战挑战者佐藤康光七段（佐藤康光是第六屆龍王杯冠軍，第七屆羽生善治挑戰成功，佐藤康光衛冕失敗），結果這一局佐藤康光七段勝出。2000年10月19日～20日，在中國上海市[[花園飯店|花園飯店]]舉行了第十三屆“龍王戰”決賽（七局四勝制）的第一局比賽，由1999年衛冕成功的[[藤井猛|藤井猛]]竜王對戰挑戰者[[羽生善治|羽生善治]]五冠。在比赛的同时还举行了多场中日青少年间的将棋比赛。
