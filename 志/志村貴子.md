'''志村貴子'''（{{bd|1973年|10月23日|catIdx= Shimura Takako}}）是[[日本|日本]]女性[[漫畫家|漫畫家]]，出身於[[神奈川縣|神奈川縣]]，現居住在[[東京都|東京都]]。

==簡介==
作品主要以[[LGBT|LGBT]]（特别是[[百合_(類型)|百合]]和[[跨性別|跨性別]]）題材為主。由她本人創作的《[[青花_(漫畫)|青花]]》和《[[放浪男孩|放浪男孩]]》分别於2009年及2011年被改編成[[電視動畫|電視動畫]]。早年以加藤雅一（{{lang|ja|加藤マサイチ}}）、東京堂Eru Eru（{{lang|ja|東京堂えるえる}}）<ref>{{cite web |url=http://www.mandarake.co.jp/information/buy/ladies/girl_a5.html |script-title=ja:少女コミック買取情報|accessdate=2011-01-13|publisher= mandarake.co.jp|language=ja}}</ref>等名義活動，但當初的作品均未被[[單行本|單行本]]收錄。1997年後以志村貴子名義相繼推出《我是女生》（[[尖端出版社|尖端出版社]]譯名，日文原標題為：{{lang|ja|ぼくは、おんなのこ}}）<ref>{{cite web |url=http://www.books.com.tw/products/0010311544 |title= 我是女生(全)|accessdate=2015-12-26|publisher=博客來|language=zh-tw}}</ref>、《[[叛逆思春期|叛逆思春期]]》（[[長鴻出版社|長鴻]]譯名，日文原標題為：{{lang|ja|敷居の住人}}）<ref>{{cite web |url= http://www.enterbrain.co.jp/product/comic/beam_comic/00172301.html|script-title=ja:敷居の住人1|accessdate=2011-01-13 |publisher= enterbrain.co.jp|language=ja}}</ref>等一系列漫畫作品。 

2009年和其他四位漫畫家[[青木俊直|青木俊直]]、[[石出電|石出電]]、[[岩岡壽枝|岩岡壽枝]]、[[谷川史子|谷川史子]]結成[[同人|同人]]團體「{{lang|ja|腹ペコ戦隊はしレンジャー}}」。

2015年作品《淡島百景》獲得第19屆日本[[文化廳媒體藝術祭|文化廳媒體藝術祭]]漫畫部門優秀賞。<ref>[http://archive.j-mediaarts.jp/festival/2015/manga/works/19m_awajima_hyakkei/ 優秀賞 - 淡島百景]</ref>
