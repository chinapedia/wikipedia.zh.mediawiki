'''顺庆府'''，[[南宋|南宋]]时设置的[[府_(行政区划)|府]]，属[[潼川府路|潼川府路]]，[[元朝|元朝]]属[[四川等处行中书省|四川行省]]，[[明朝|明]]、[[清朝|清]]因之。民國二年（1913年）废。

[[宝庆|宝庆]]三年（1227年）升[[梁州_(曹魏)|梁州]]置，[[治所|治所]]在[[南充县|南充县]]（今[[南充市|南充市]]，[[淳祐|淳祐]]九年迁青居山，元还旧治）。元朝[[至元_(忽必烈)|至元]]四年（1267年）改置[[东川路|东川路]]，至元十五年复为府，二十年升为[[路_(行政区划)|路]]。[[明朝|明朝]][[洪武|洪武]]中，复为府。辖境相当今四川省南充、[[西充|西充]]、[[仪陇|仪陇]]、[[蓬安|蓬安]]、[[营山|营山]]、[[渠县|渠县]]、[[岳池|岳池]]、[[邻水|邻水]]、[[大竹|大竹]]等州县地<ref>[[清朝|清]]·[[张廷玉|张廷玉]]等，《[[明史|明史]]》（卷43）：“顺庆府元顺庆路。洪武中，为府。领州二，县八。西南距布政司六百里。”</ref>。

== 宋代的順慶府 ==

順慶府，本[[果州|果州]]南充郡，寶慶三年，以[[宋理宗|宋理宗]]<ref>《宋本方輿勝覽》作[[宋寧宗|宋寧宗]]潛邸</ref> 初潛之地，升府，隸劍南東路。端平三年，兵亂。淳祐九年，徙治青居山。領縣三：[[南充縣|南充縣]]、[[西充縣|西充縣]]、[[流溪縣|流溪縣]]<ref>《宋史》卷八十九　志第四十二／地理五／潼川府路／順慶府</ref>

== 元代的順慶府 ==

元至元四年置[[東川路|東川路]]，後改[[東川府|東川府]]，十五年復曰順慶府，又升為[[順慶路|順慶路]]，屬[[四川等处行中书省|四川行省]]。<ref>嘉慶[[大清一統志|大清一統志]]</ref>

元中統元年，立征南都元帥府。至元四年，置東川路統軍司，後改[[東川府|東川府]]。十五年，復為[[順慶府|順慶府]]。二十年，升為[[順慶路|順慶路]]，設錄事司。領司一、縣二、 府一、州二。府領二縣，州領五縣。
* 錄事司縣二:
# [[南充縣|南充縣]]。至元二十年，併[[漢初縣|漢初縣]]入。
# [[西充縣|西充縣]]。至元二十年，併[[流溪縣|流溪縣]]入。
* 府一：[[廣安府|廣安府]]，至元二十年，升為廣安府。舊領[[渠江縣|渠江縣]]、[[岳池縣|岳池縣]]、[[和溪縣|和溪縣]]、[[新明县_(唐朝)|新明縣]]四縣，後併和溪縣、新明縣入[[岳池縣|岳池縣]]。領縣：渠江縣、岳池縣。
* 州二：
# [[蓬州|蓬州]]。元初立宣撫都元帥府，後罷。至元二十年，立[[蓬州路|蓬州路]]總管府，後復為蓬州。領三縣：
## [[相如縣|相如縣]]，至元二十年，以金城寨入。
## [[營山縣|營山縣]]，至元二十年，併[[良山縣|良山縣]]入。
## [[儀隴縣|儀隴縣]]。至元二十年，併[[蓬池縣|蓬池縣]]、[[伏虞縣|伏虞縣]]入。
# [[渠州|渠州]]，元至元十一年，立渠州安撫司。二十年，罷安撫司，以渠州為散郡。領二縣：
## [[流江縣|流江縣]]　
## [[大竹縣|大竹縣]]。至元二十年，併[[鄰山縣|鄰山縣]]、[[鄰水縣|鄰水縣]]入。

== 明代的順慶府 ==

元代的[[順慶路|順慶路]]在洪武四年（1371年）分為順慶府和[[廣安府|廣安府]]，西北[[南充縣|南充縣]]、[[西充縣|西充縣]]及[[蓬州|蓬州]](轄[[營山縣|營山縣]]、[[儀隴縣|儀隴縣]])屬順慶府，東部[[渠江縣|渠江縣]]、[[岳池縣|岳池縣]]和[[渠州|渠州]](轄[[大竹縣|大竹縣]])屬廣安府。洪武九年（1376年）四月，廣安府降為順慶府屬州，渠州則降為縣，與大竹、岳池、渠江三縣一起為廣安州下屬縣。此時順慶府的管轄範圍與元末順慶路基本一致。洪武十年（1377年）五月戊寅，因戶糧不及數，西充縣、營山縣、儀隴縣、渠江縣4縣被廢，前3縣洪武十三年（1340年）十一月庚戌復置。此後至明末，除成化元年分[[大竹縣|大竹縣]]增復設廣安州鄰水縣外，順慶府的州縣再未發生過變化。成化元年以後，順慶府有2直轄縣、2州、6州轄縣。

* [[蓬州|蓬州]]：元末為順慶路屬州，領[[相如縣|相如縣]](附郭)、營山縣、儀隴縣3縣。洪武四年為順慶府屬州，省[[相如縣|相如縣]]入州。洪武十年五月營山、儀隴俱廢，十三年十一月復置。
* [[广安州_(明朝)|廣安州]]：元順慶路廣安府，洪武四年為廣安府，直隸四川行省，領[[渠江縣|渠江縣]](附郭)、[[岳池縣|岳池縣]]，[[渠州|渠州]]和[[渠州|渠州]]屬下的大竹縣。九年四月甲午改廣安府為州，渠州為縣，隸順慶府<ref>《太祖實錄》卷105。</ref>，州為順慶府屬州，渠縣、大竹縣歸本州，十年五月戊寅省[[渠江縣|渠江縣]]入州。成化元年七月新增鄰水縣，此後州共領[[岳池縣|岳池縣]]、[[渠縣|渠縣]]、[[大竹縣|大竹縣]]、[[鄰水縣|鄰水縣]]4縣。

== 清代的順慶府 ==
* 清初因[[明代|明代]]，領州二縣八。
* 康熙七年（1668年）裁[[岳池縣|岳池縣]]歸併[[广安州_(明朝)|廣安州]]，領州二縣七。
* 康熙六十年（1721年）復置[[岳池縣|岳池縣]]，仍領州二縣八。
* 嘉慶十九年（1814年）以[[渠縣|渠縣]]、[[大竹縣|大竹縣]]往屬[[綏定府|綏定府]]，後轄縣六州二，稱為'''順慶八屬'''（[[南充縣|南充縣]]、[[西充縣|西充縣]]、[[蓬州|蓬州]]、[[營山縣|營山縣]]、[[儀隴縣|儀隴縣]]、[[廣安州_(明朝)|廣安州]]、[[鄰水縣|鄰水縣]]、[[岳池縣|岳池縣]]）
