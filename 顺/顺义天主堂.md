==简介==
2013年6月23日（常年期第十二主日），北京教区[[李山|李山]]主教为新落成的顺义祈祷所举行了祝福仪式。[[北京天主教神哲学院|北京天主教神哲学院]]的修生、若瑟修女会的修女和本地及北京教区其他部分堂区的教友参加。北京天主教神哲学院院长[[韩文生|韩文生]]神父、教区秘书长[[甄雪斌|甄雪斌]]神父、[[西直门天主堂|西直门天主堂]]张洪波神父参加了共祭弥撒。顺义祈祷所此次落成的建筑，是本地教友经过十多年努力，最终在本堂鲍静坡神父的带领下建成。<ref>{{Cite web |url =http://www.chinacatholic.org/News/index/id/25204.html  |title = 北京：李山主教祝福新落成的顺义祈祷所 |author =  |date = 2013-06-29 |publisher = 信德网 |language =  |accessdate =  |quote =  }}</ref>
