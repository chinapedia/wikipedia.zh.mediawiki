'''顺义区'''是[[北京市|北京市]]下辖的[[市辖区|市辖区]]，位于北京市东北部，毗邻[[城六区|城六区]]内的[[朝阳区_(北京市)|朝阳区]]，面积1021[[平方公里|平方公里]]<ref name = "自然概况">{{cite web | author = 北京市顺义区人民政府 | title = 自然概况 | publisher = 顺义网城 | accessdate = 2014-08-06 | language = zh-cn | url = http://www.bjshy.gov.cn/Item/42454.aspx}}</ref>，常住人口100.4万人，户籍人口61.5万人<ref name = "人口民族">{{cite web | author = 北京市顺义区人民政府 | title = 人口民族 | publisher = 顺义网城 | accessdate = 2014-08-06 | language = zh-cn | url = http://www.bjshy.gov.cn/Item/42455.aspx}}</ref>。顺义区是北京市东北部发展带的重要节点、重点发展新城之一，也是首都国际航空中心核心区<ref name = "顺义概况">{{cite web | author = 北京市顺义区人民政府 | title = 顺义概况 | publisher = 顺义网城 | accessdate = 2014-08-06 | language = zh-cn | url = http://www.bjshy.gov.cn/Item/14815.aspx}}</ref>。

[[北京首都国际机场|北京首都国际机场]]一号、二号航站楼虽位于顺义区内，但行政上为隶属于[[朝阳区_(北京市)|朝阳区]]的[[飞地|飞地]]。

== 历史 ==
=== 唐朝及以前 ===
今顺义区辖境在[[夏|夏]]、[[商朝|商]]、[[周朝|周]]三朝时属[[冀州|冀州]]、[[幽州|幽州]]、[[燕国|燕国]]，[[秦朝|秦朝]]时属[[渔阳郡|渔阳郡]]<ref>{{cite book |author=《建筑创作》杂志社 |title= 《北京顺义 绿色国际港》 |publisher= 中国城市出版社 |date=2005年10月}}</ref>。[[西汉|西汉]][[汉高祖|高祖]]五年（前202年）至十二年（前195年）置[[狐奴县|狐奴]]、[[安乐县|安乐]]两县，属[[渔阳郡|渔阳郡]]<ref name = "沿革历史">{{cite web | author = 北京市顺义区人民政府 | title = 沿革历史 | publisher = 顺义网城 | accessdate = 2014-08-06 | language = zh-cn | url = http://www.bjshy.gov.cn/Item/42456.aspx}}</ref>。

[[唐朝|唐朝]][[貞觀_(唐朝)|贞观]]二十二年（648年），以内附[[契丹|契丹]]别帅析纥便部置[[归顺州_(贞观)|归顺州]]，后改称[[归化郡|归化郡]]、[[顺州_(唐朝)|顺州]]，属契丹[[松漠都督府|松漠都督府]]弹汗州。[[开元|开元]]四年（716年）置归顺州，[[天宝_(唐朝)|天宝]]元年（742年）改称归化郡，[[乾元_(唐朝)|乾元]]元年（758年）复称归顺州，领怀柔县{{efn|非今日北京市[[怀柔区|怀柔区]]}}，县治驻今顺义城区地区，州、郡、县治所同一<ref name = "沿革历史"/>。

=== 辽朝至元朝 ===
{{see also|燕云十六州}}
937年，[[石敬瑭|石敬瑭]]在[[契丹|契丹]]的帮助下灭后唐后，建立[[后晋|后晋]]。次年，按约定将包括顺州在内的燕云十六州献给契丹。[[后周|后周]]、[[北宋|北宋]]时期，燕云十六州地区开战了长期的战争，但一直未能收复顺州。

[[宣和_(北宋)|宣和]]四年（1122年），宋[[金|金]]订立“[[海上之盟|海上之盟]]”，次年，金朝如约将[[燕京|燕京]]、[[顺州_(唐朝)|顺州]]等归还给宋朝。[[完颜阿骨打|完颜阿骨打]]死后，金以[[张觉|张觉]]事变为由伐宋。宣和七年（1125年），金又重新占领整个燕云十六州地区。[[靖康之变|靖康之变]]之后，北宋灭亡，金定都并扩建[[燕京|燕京]]，即[[金中都|中都]]。1213年，燕云十六州被[[蒙古帝国|蒙古帝国]]占领。
[[File:Sixteen_Prefectures.png|thumb]]

=== 明朝至今 ===
[[明朝|明朝]][[洪武|洪武]]元年（1368年），[[朱元璋|朱元璋]]遣[[徐达|徐达]]、[[常遇春|常遇春]]攻克[[大都|大都]]。是年，降顺州为[[顺义县|顺义县]]，属[[北平府|北平府]]。[[明成祖|成祖]][[朱棣|朱棣]]迁都北平后，改[[北平府|北平府]]为[[顺天府|顺天府]]。直至[[民国十七年|民国十七年]]（1928年），改属[[河北省_(中華民國)|河北省]]。

[[中华人民共和国|中华人民共和国]]成立后，顺义县属[[河北省|河北省]][[通州专区|通州专区]]。1958年3月划归北京市。1998年3月撤顺义县，设顺义区。

==自然地理==
顺义位于[[北京市|北京市]]东北郊，城区距市中心30公里，地处[[北纬|北纬]]40°00＇－40°18＇，[[东经|东经]]116°28＇－116°58＇。东邻[[平谷|平谷]]，北连[[怀柔|怀柔]]、[[密云|密云]]，西接[[昌平|昌平]]、[[朝阳区_(北京市)|朝阳]]区，南界[[通州区_(北京市)|通州]]区、[[河北|河北]]三河市。区境东西长45公里，南北宽30公里，总[[面积|面积]]1020平方公里。地处[[燕山|燕山]]南麓，华北平原北端，属[[潮白河|潮白河]]冲积扇下段。平原面积占95.7%。地势北高南低，坡度为6/10000，北部山地最高点海拔637米，平原[[海拔|海拔]]25-45米，平均[[海拔|海拔]]35米。

顺义区气候属暖温带半湿润大陆性季风性气候,年平均[[气温|气温]]为11.5°C。1月平均气温4.9°C，最低气温零下19.1°C；7月平均气温 25.7°C，最高气温达40.5°C。年日照2750小时，无霜期195天左右。年均相对[[湿度|湿度]]50%，年均降雨量约625毫米,为[[华北|华北]]地区降水量较均衡的地区之一全年降水的75%集中在[[夏季|夏季]]。

==人口==
顺义区[[人口|人口]]稳定，2004年末户籍人口为554104人，其中，[[农业人口|农业人口]]364459人，[[非农业人口|非农业人口]]189645人。男性人口273692人，女性人口 280412人。全区常住人口72.4万人，其中流动人口16.6万人，常住人口密度为每平方公里710人。全年出生人口3050人，出生率为5.53‰ ，比上年降低了0.01个千分点；死亡人口3624人，死亡率为6.57‰，比上年降低了0.2个千分点；人口自然增长率为-1.04‰，比上年上升了0.1个千分点。

2008年最新资料：户籍人口56.4万，常住人口70.3万，区境内有[[回|回]]、[[满|满]]、[[蒙古族|蒙]]等25个[[少数民族|少数民族]]。城乡劳动力在二三产就业人数达到21.8万人，占本区劳动力总数的88％。2007年，累计安置[[城乡劳动力|城乡劳动力]]1.5万人，城镇登记失业率为1.62％。
