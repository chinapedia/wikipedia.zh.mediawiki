'''顺外站'''位于[[江西省|江西省]][[南昌市|南昌市]][[青山湖区|青山湖区]]顺外路与洪都中大道交叉路口处，是[[南昌地铁|南昌地铁]]{{南昌地铁线路名|2}}的一个地下车站；车站计划于2017年12月31日启用。

== 车站结构 ==
站厅在地下一层，站台在地下二层为岛式站台<ref name=":0" />。

== 历史 ==
* 2010年4月8日南昌市轨道交通二号线一期工程环境影响评价第二次公示，公布了《南昌市轨道交通二号线一期工程环境影响报告书（简本）》文件中本站名为'''洪都大道站'''<ref>{{Cite web|url=http://xxgk.nc.gov.cn/bmgkxx/hbj/gzdt/gggs/201004/t20100409_222696.htm|title=《南昌市轨道交通二号线一期工程环境影响报告书（简本）》|accessdate=2016-07-07|date=2010年4月9日|last=中铁第四勘察设计院集团有限公司|first=|work=中铁第四勘察设计院集团有限公司|archive-url=https://web.archive.org/web/20170829203132/http://xxgk.nc.gov.cn/bmgkxx/hbj/gzdt/gggs/201004/t20100409_222696.htm|archive-date=2017年8月29日|dead-url=yes}}</ref><ref>[http://www.lsnc.cn/news/gonggao/2010-4-8/201048131811.htm 南昌市轨道交通二号线一期工程环境影响评价第二次公示] {{webarchive|url=https://archive.is/20150902063409/http://www.lsnc.cn/news/gonggao/2010-4-8/201048131811.htm |date=2015-09-02 }}</ref>。

* 2012年2月9日2号线站名确认，本站名为没有做任何变动<ref>{{Cite web|url=http://jiangxi.jxnews.com.cn/system/2012/02/09/011893506.shtml|title=南昌地铁2号线21个站点名确定|accessdate=2016-07-10|author=|date=2012-02-09|work=大江网|publisher=}}</ref>。
* 2014年1月23日3、4号线优化调整规划批前公示，[[南昌轨道交通4号线|4号线]]不再经过本站<ref>{{Cite web |url = http://jiangxi.jxnews.com.cn/system/2014/01/23/012919119.shtml |title = 南昌地铁3、4号线规划调整 4号线调整较大 线路增加8公里 |author =  |date = 2014-01-23 |publisher = 中国江西网-信息日报 |language = |accessdate = |archiveurl= |archivedate=}}</ref><ref>{{Cite web |url = http://ghj.nc.gov.cn/qtpqgg/6540.jhtml |title = 南昌市轨道交通3、4号线优化调整规划批前公示 |author =  |date = 2014-01-21 |publisher = 南昌市城乡规划局 |language = |accessdate = |archiveurl= |archivedate= }}</ref>。
