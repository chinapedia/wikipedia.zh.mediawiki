'''高崇文'''（{{bd|746年||809年|catIdx=G高}}），[[唐代|唐代]]名將，[[幽州|幽州]]人。先祖是[[渤海郡|渤海郡]]（今河北省[[沧州市|沧州市]]）人。崇文[[不識字|不識字]]，樸實莊重、沈默寡言。曾大破[[吐蕃|吐蕃]]、收復[[西川|西川]]，官至[[同中書門下平章事|同中書門下平章事]]，封南平[[郡王|郡王]]，贈'''[[司徒|司徒]]'''。

== 大破吐蕃 ==
[[貞元_(唐朝)|貞元]]中，隨[[韓全義|韓全義]]鎮守[[長武城|長武城]]（今陕西长武县）。累官[[金吾将军|金吾将军]]。[[貞元|貞元]]五年（789年）夏，[[吐蕃|吐蕃]]軍三萬人犯寧州（今甘肃[[宁县|宁县]]），崇文奉命率甲士三千前往解救，激戰[[佛堂原|佛堂原]]，崇文以少胜多，大破之。迁长武城都知兵马使。

== 收復西川 ==
[[元和|元和]]元年（806年），[[西川|西川]]節度副使-{zh-hans:[[劉闢_(唐朝)|劉辟]]; zh-hant:[[劉闢_(唐朝)|劉闢]]}-，割據[[西蜀|西蜀]]，發兵圍攻[[東川|東川]]節度使[[李康|李康]]於[[梓州|梓州]]（今四川[[三台|三台]]）。宰相[[杜黃裳|杜黃裳]]力主討伐-{zh-hans:劉辟; zh-hant:劉闢}-，推薦時為[[神策軍|神策軍]]將領的[[高崇文|高崇文]]為帥，[[唐憲宗|憲宗]]批准，拜检校[[工部尚书|工部尚书]]、兼[[御史大夫|御史大夫]]，以士兵五千人，自长武城出兵。

是年九月，高崇文連破劉辟軍，从[[阆州|阆州]]（今四川[[阆中县|阆中县]]）直撲[[梓州|梓州]]（今四川三台），-{zh-hans:劉辟; zh-hant:劉闢}-命人在[[鹿头山|鹿头山]]（今四川[[德阳县|德阳县]]东北）筑城，以八道栅力阻，崇文派[[高霞寓|高霞寓]]攻下[[鹿头城|鹿头城]]以东的[[万胜堆|万胜堆]]（今四川德阳县东北），“士扳缘而上，矢石如雨；又命敢死士连登，夺其堆，烧其栅，栅中之贼歼焉。”，八戰皆勝，隔年（[[元和|元和]]二年，807年）二月二十六日[[段文昌|段文昌]]開門投降。崇文進入[[成都|成都]]，秋毫无犯，市肆不驚。），-{zh-hans:劉辟; zh-hant:劉闢}-等二十餘口被押送回京師，蜀境遂平。-{zh-hans:劉辟; zh-hant:劉闢}-有二妾，有姿色。眾將士勸高崇文納之，高崇文不為所動。監軍建議獻給皇上，崇文說：“天子命我討平凶豎，當以安撫百姓為先。奉獻美人以求媚，豈天子之意麼！崇文義不為此。”

== 朝廷酬庸 ==
[[元和|元和]]三年冬，加封[[同中书门下平章事|同中书门下平章事]]。封南平郡王。元和四年（809年）卒，追赠[[司徒|司徒]]，諡'''威武'''。

==後代==
夫人董氏，汝州长史董同珍之女，春秋三十有一，以大历十有四年五月一日终。元和三年追封郇国夫人。
*嗣子[[高承简|高承简]]（高士政），义成军节度使，赠司空，太和元年八月卒。
*次子高士荣，检校秘书监兼御史中丞
*季子高承明（高士明），左卫率府胄曹参军、神策军虞候。子[[高駢|高駢]]為唐末名將，大破[[南詔|南詔]]，但在[[黃巢之亂|黃巢之亂]]中養兵自重，晚年昏庸，重用方士[[呂用之|呂用之]]、[[張守一|張守一]]等人，苛待[[幕府|幕府]]將領，被[[兵變|兵變]]殺害。
