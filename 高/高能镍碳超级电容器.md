'''高能镍碳超级电容器'''，是基于[[镍|镍]][[碳|碳]]等新材料的高效率[[电容|电容]]，将[[活性碳|活性碳]]材料引入镍氢电池负极，使普通超级电容器与电池结合为一体，它的研制成功实现了中国大陆在纯[[电动车|电动车]]动力[[电源|电源]]研究的突破。<ref>{{cite web | url=http://news.ifeng.com/mil/2/detail_2011_09/02/8881371_0.shtml | title=中国科研新突破：高能镍碳超级电容器研制成功 | publisher=解放军报 | work=凤凰网 | accessdate=September 3, 2011}}</ref>2011年9月1日，[[天津市人民政府|天津市人民政府]]在[[天津大礼堂|天津大礼堂]]召开高能镍碳超级电容器产品新闻发布会宣布研制成功，产品由[[中国工程院|中国工程院]][[周国泰|周国泰]]<ref>{{cite web | url=http://www.xys.org/xys/ebooks/others/science/dajia14/zhouguotai.txt | title=“神人”院士周国泰 | publisher=新语丝 | work=zgthu | accessdate=July 8, 2013}}</ref>院士团队研发，定名“高能镍碳超级电容器”。

== 与传统电容的区别 ==
高能镍碳超级电容器，与传统[[电容器|电容器]]相比，首先，在研发出新材料的基础上尽可能地扩大比表面积，使存储的[[电量|电量]]大幅增加；其次，高能镍碳超级电容器在[[电极|正负极]]的材料结构上获突破，其比功率比传统电容高得多；另外，高能镍碳超级电容在结构上实现了电池和传统电容的内并，实现了电池和电容的优点兼备。<ref>{{cite web| url=http://cpc.people.com.cn/GB/64093/64094/15573253.html| title=高能镍碳超级电容器产品新闻发布会举行 张高丽等出席| publisher=天津日报| work=中国共产党新闻网| accessdate=September 3, 2011| deadurl=yes| archiveurl=https://web.archive.org/web/20110927120114/http://cpc.people.com.cn/GB/64093/64094/15573253.html| archivedate=2011年9月27日}}</ref>

== 技术优势 ==
与传统电容和传统动力电池相比，高能镍碳超级电容器和以其为基础生产的动力电源产品具有能量密度大、功率密度高、充放电效率高、温度适应性好、循环寿命长、安全环保、性价比高等技术优势，实现产业化之后将可以有效解决中国大陆目前电动汽车动力电源技术瓶颈问题。<ref>{{cite web | url=http://www.caigou.com.cn/News/Detail/77045.shtml | title=天津宣布研制成功高能镍碳超级电容器产品 | accessdate=September 3, 2011}}</ref><ref>{{cite web | url=http://www.itdcw.com/archives/25563 | title=电动车新电源 我国成功研发镍碳超级电容器 | accessdate=September 3, 2011}}</ref>

== 产业化 ==
[[天津市|天津市]]将新能源列为八大支柱产业之一，并将配合科研成果建立高能镍碳超级电容器产业化基地，推动高能镍碳超级电容器在天津实现规模产业化，一期达到年产1000万只30亿安时的产能，二期工程将建成1亿只300亿安时生产能力。产业化后的高能镍碳超级电容器并将率先在公交系统逐步推广，装配在超级电容公交车辆上，并探索基于高能镍碳超级电容器的新型公交运营模式，真正实现纯电动汽车在公交系统的大规模应用，以实现节能减排。<ref>{{cite web| url=http://www.tj.xinhuanet.com/news/2011-09/01/content_23602390.htm| title=天津将在公交系统率先推广高能镍碳超级电容车| accessdate=September 3, 2011| deadurl=yes| archiveurl=https://web.archive.org/web/20120618014753/http://tj.xinhuanet.com/news/2011-09/01/content_23602390.htm| archivedate=六月 18, 2012}}</ref>
