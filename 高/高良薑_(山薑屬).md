'''高良薑'''（[[學名|學名]]：''{{lang|la|Alpinia officinarum}}''，[[英文|英文]]名稱：{{lang|en|Galangal}}）<ref name="浸大">{{cite encyclopedia | encyclopedia=香港浸會大學中醫藥學院藥用植物圖像數據庫| title= 高良薑 Gaoliangjiang  | volume= | pages=|url=http://libproject.hkbu.edu.hk/was40/detail?lang=ch&record=1&channelid=1288&searchword=Alpinia+officinarum&extension=all&sortfield=%2Bname_formal_latin_sort}}</ref>，別稱'''良姜'''、'''小良姜'''、'''蠻姜'''、'''風姜'''、'''膏良姜'''、'''膏涼姜'''、'''雷州高涼姜'''、'''佛手根'''、'''伙哈'''、'''賀哈'''及'''比目連理花'''等<ref name="自然標本館"/>，為[[薑科|薑科]][[山薑屬|山薑屬]]植物<ref name="浸大"/>。

==分佈==
本種現分佈於[[中國|中國]]、[[日本|日本]]、[[越南|越南]]、[[泰國|泰國]]及[[台灣|台灣]]等地，[[中國|中國]]國內野外分佈於[[廣東|廣東]]、[[海南|海南]]、[[廣西|廣西]]南部等省份<ref name="ars"/>，[[雲南|雲南]]亦人工引種栽培<ref name="浸大"/>。野外常生於灌叢或疏林之中<ref name="浸大"/>。本種[[模式標本|模式標本]]採自於[[廣東|廣東]][[海南|海南]]<ref name="中國">{{cite encyclopedia| title=高良薑| encyclopedia=中國植物誌| volume=16(2)| pages=100-102頁| url=http://v2.cvh.org.cn/zhiwuzhi/page/16(2)/100a.pdf}}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

==形態特徵==
高良薑是一種多年生[[草本|草本]][[植物|植物]]<ref name="安徽">{{cite encyclopedia| title=高良薑| encyclopedia=安徽植物誌| volume=5| pages=521-522頁| url=http://v2.cvh.org.cn/difangzhi/anhui/page/5/521a.pdf}}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，植株高約40－150[[厘米|厘米]] <ref name="海南">{{cite encyclopedia| title=高良薑| encyclopedia=海南植物誌| volume=4| pages=98頁| url=http://v2.cvh.org.cn/difangzhi/hainan/page/4/098a.pdf}}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。根莖圓柱形，棕紅色，橫生，具節，節上長有膜質環狀鱗片，直徑約1－1.5[[厘米|厘米]]<ref name="浸大"/>。<ref name="中國"/>

===葉===
[[葉|葉]]片線形，革質<ref name="海南"/>，先端尾狀漸尖，基部漸狹，兩面無毛，無柄或近無柄<ref name="海南"/>，葉長約20－30[[厘米|厘米]]，寬約1.2－2.5[[厘米|厘米]]<ref name="安徽"/>；[[葉舌|葉舌]]披針狀線形，薄膜質，先端漸尖，不分裂<ref name="海南"/>，全緣<ref name="浸大"/>，長約1.5－3[[厘米|厘米]]<ref name="安徽"/>，長可達5厘米<ref name="中國"/>。

===花===
[[花|花]]排列成總狀[[花序|花序]]，頂生，直立或略彎，披短柔毛<ref name="海南"/>，長約6－10[[厘米|厘米]]<ref name="安徽"/>；[[花序軸|花序軸]]披絨毛；小[[苞片|苞片]]卵形，細小，遲落<ref name="海南"/>，長不及1[[毫米|毫米]]<ref name="中國"/>；小[[花梗|花梗]]長約1－2[[毫米|毫米]]<ref name="中國"/>；[[花萼|花萼]]管狀，披短柔毛，先端具不明顯的3齒裂，一側開裂至中部，長約8－14[[毫米|毫米]]<ref name="海南"/>；[[花冠|花冠]]白色，冠管比萼管為短，長約1[[厘米|厘米]]<ref name="海南"/>；[[花冠|花冠]]裂片長圓形，披絹毛，長約1－1.5[[厘米|厘米]]<ref name="海南"/>；後方一枚裂片較大，呈兜狀<ref name="海南"/><ref name="安徽"/>；[[唇瓣|唇瓣]]長圓狀卵形，先端微卷及微缺<ref name="海南"/>，白色帶紅色線條，長約2[[厘米|厘米]]<ref name="中國"/>；[[雄蕊|雄蕊]]1枚<ref name="浸大"/>，長於花冠管喉部上方<ref name="浸大"/>，長約1.6<ref name="安徽"/>－1.7<ref name="海南"/>[[厘米|厘米]]；退化[[雄蕊|雄蕊]]側生，呈錐狀<ref name="浸大"/>，長約3[[毫米|毫米]]<ref name="安徽"/>；[[花絲|花絲]]線形<ref name="海南"/>，長約1[[厘米|厘米]]<ref name="中國"/>；[[花藥|花藥]]藥隔無附屬物<ref name="海南"/>，長約6[[毫米|毫米]]<ref name="中國"/>；[[子房|子房]]3室<ref name="浸大"/>，密披絨毛<ref name="中國"/>；[[花柱|花柱]]細長，柱頭2唇裂，基部長有2枚合生的圓柱形蜜腺<ref name="浸大"/>。

===果===
[[果|果]]為蒴果<ref name="浸大"/>，圓球形，表面不具縱紋<ref name="海南"/>，成熟時紅色，直徑約1－1.2[[厘米|厘米]]<ref name="海南"/>；種子多數<ref name="海南"/>，純棱角，棕色，具假種皮<ref name="浸大"/>。

==醫藥用途==
{{medical}}
本種以乾燥根莖入藥，[[中藥|中藥]]名為高良薑<ref name="浸大"/>，藥用之名始載於《[[名醫別錄|名醫別錄]]》，列為中品<ref name = "藥典">{{cite book|title=《當代藥用植物典 4》|author= |series=蕭培根|publisher=香港賽馬會中藥研究院|page=第21-24頁|isbn=978-988-99226-3-4|date=2007年8月}}</ref>，味辛，性熱<ref name="浸大"/>，歸脾、胃經<ref>{{cite book|title=《南方藥用植物 近600種中國南方藥用植物的彩色圖鑒》|author= |series=王玉生、蔡岳文|publisher=南方日報出版社|page=第313頁|isbn= 978-7-5491-0180-1|date=2011年4月}}</ref>，藥材主產於[[廣東|廣東]]、[[廣西|廣西]] <ref name="香港中藥材">{{cite book | title=香港中藥材圖鑑 | publisher=香港浸會大學中醫藥學院 | author=趙中振 | page=279 | isbn=988-97448-1-3|date=2003年10月}}</ref>，具溫胃散寒、消食<ref name = "藥典"/>、理氣止痛等功效<ref name="浸大"/>，主治胃寒嘔瀉、嘔吐、噫氣、消化不良、消積食滯、及脘腹冷痛等<ref name="浸大"/><ref name = "藥典"/>。現代藥理研究表明高良薑具抗氧化、抗[[潰瘍|潰瘍]]、抗[[腹瀉|腹瀉]]、抗[[腫瘤|腫瘤]]、抗菌、抗凝血、抗[[血栓|血栓]]、降[[血壓|血壓]]及鎮痛等作用<ref name = "藥典"/>，臨床應用於胃脘疼痛及脘腹脹滿之症<ref name = "藥典"/>。

==藥材鑑定==
[[中藥|中藥]]高良薑以乾燥[[根莖|根莖]]入藥，於[[夏|夏]]末[[秋|秋]]初採收，清除鬚根及鱗片等雜質後，洗淨切段曬乾<ref name = "香港中藥材"/>；本品呈圓柱形，紅棕至暗紅色，表面具細密的縱皺紋及灰棕色的波狀環節，多彎曲，具節及分枝，長約5－9[[厘米|厘米]]，直徑約1－1.5[[厘米|厘米]]<ref name = "香港中藥材"/>；節間長0.2－1[[厘米|厘米]]，一面具圓形的根痕<ref name = "香港中藥材"/>；質地堅韌，不易折斷，斷面灰棕色或紅棕色，呈纖維狀或裂片狀，中柱佔約三分之一<ref name = "香港中藥材"/>；氣香，味辛辣<ref name="常用中藥材">{{cite book | title=常用中藥材鑑別圖典 | publisher=萬里機構 | author=趙中振, 陳虎彪 | page=166頁 | isbn=978-962-14-4150-8|date=2010年1月}}</ref> ；傳統經驗則認為以色紅棕、氣香、味辣、分枝少者為佳<ref name="常用中藥材"/>；本品載錄於《[[中國藥典|中國藥典]]》2005年版中，定為[[中藥|中藥]]高良薑的的法定原植物來源種，主要透過[[氣相色譜法|氣相色譜法]]等指標測定，規定[[藥材|藥材]]中的[[桉葉素|桉葉素]]含量不得少於0.15%，以控制[[藥材|藥材]]的質量<ref name = "藥典"/>。
