'''高速分組接入'''（{{lang-en|High Speed Packet Access}}，缩写：'''HSPA'''），是[[W-CDMA|W-CDMA]]第一個進化技術，HSPA可增加資料傳輸速率（1－3Mbps），可減少延遲。HSPA 由[[HSDPA|HSDPA]]、[[HSUPA|HSUPA]]組成，隨後還有[[HSPA+|HSPA+]]的推出。HSPA+在[[3GPP|3GPP]] release 7中定義。

[[HSDPA|HSDPA]]在W-CDMA下行鏈路中提供封[[分組數據|分組數據]]業務，在一個5MHz[[載波|載波]]上的傳輸速率可達14.4Mbit/s（如採用[[MIMO|MIMO]]技術，則可達28.8 Mbit/s）。

高速上行分組接入（High Speed Uplink Packet Access的縮寫[[HSUPA|HSUPA]]）是一種因 W-CDMA上傳速度不足（只有384Kbit/s）而開發的，亦稱為3.75G，可在一個5MHz載波上的下行傳輸速率可達14.4Mbit/s（如採用MIMO技術，則可達28.8Mbit/s），上行傳輸速率可達5.76Mbit/s。

==香港營運商情況==
在[[香港|香港]]，目前五間提供[[3G|3G]]服務的網絡供應商和二間提供[[4G|4G]] [[LTE|LTE]]服務的網絡供應商，均已提供[[HSPA+|HSPA+]] / [[4G|4G]] [[LTE|LTE]]服務

*[[數碼通電訊集團有限公司|數碼通電訊集團有限公司]]
是全世界第一個把3.6Mbps [[HSDPA|HSDPA]] 作商用的網絡供應商 ，2007年將傳送速度提升至14.4Mbps ，2009年11月再將傳送速度提升至21Mbps ，2010年4月 ( 利用 Dual Carrier 技術 ) 提升至28.8Mbps ，在 [[The_new_iPad|The new iPad]] 推出時 ，把現有iPad數據計畫的最高下載傳送速度堤升至 DC-[[HSPA+|HSPA+]]42Mbps ，將於2012年第四季推出[[4G|4G]] [[LTE|LTE]]服務
*[[電訊盈科流動通訊|電訊盈科流動通訊]]
於2007年8月提供3.6Mbps [[HSDPA|HSDPA]]服務 ，2009年8月將傳送速度提升至14.4Mbps ，2009年12月提升至21Mbps ，2011年4月再提升至 DC-[[HSPA+|HSPA+]]42Mbps ，2012年2月宣佈用[[MIMO|MIMO]]技術提升現有DC-[[HSPA+|HSPA+]]的速度至84Mbps ，並宣佈與[[和記電訊國際有限公司|和記電訊國際有限公司]]合營[[4G|4G]] [[LTE|LTE]]服務 ，將於2012年第二季推出[[4G|4G]] [[LTE|LTE]]服務
*[[3_(電訊)|3 (電訊)]]
於2006年9月提供3.6Mbps [[HSDPA|HSDPA]]服務 ，2008年將傳送速度提升至7.2Mbps ，2010年提升至21Mbps ，2011年4月再將傳送速度提升至 DC-[[HSPA+|HSPA+]]42Mbps ，並宣佈與[[電訊盈科有限公司|電訊盈科有限公司]]合營[[4G|4G]] [[LTE|LTE]]服務 ，將於2012年第二季推出[[4G|4G]] [[LTE|LTE]]服務
*[[香港流動通訊有限公司|香港流動通訊有限公司]]
於2009年3月提供當時全球最快的[[HSPA+|HSPA+]]服務 ，傳送速度高達21Mbps ，2011年5月客戶可以升級[[4G|4G]] [[LTE|LTE]]試用計畫 ，2011年8月正式宣佈推出 [[4G|4G]] [[LTE|LTE]] / DC-[[HSPA+|HSPA+]] ( 下載傳送速度100Mbps ，上載傳送速度50Mbps ，與此同時如沒有 [[4G|4G]] [[LTE|LTE]] 信號會自動轉用DC-[[HSPA+|HSPA+]] )
*[[中國移動香港有限公司|中國移動香港有限公司]]
於2011年12月租用[[香港流動通訊有限公司|香港流動通訊有限公司]]的[[3G|3G]]服務 ，開始提供7.2Mbps的無線接達服務 ，於2012年4月25日推出[[4G|4G]] [[LTE|LTE]]服務 ，下載傳送速度100Mbps ，上載傳送速度35Mbps ，中國移動香港[[4G|4G]] [[LTE|LTE]]服務計劃內的數據用量更可中港互通 ，在中國境內無須繳付任何[[4G|4G]] [[LTE|LTE]]漫游造成的額外費用
