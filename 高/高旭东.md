'''高旭东'''（{{bd|1962年|12月}}），旅居[[加拿大|加拿大]]的中國[[琵琶|琵琶]]、[[古筝|古筝]]女演奏家。中国琵琶研究会会员，中国民族管弦乐学会会员。

==职业生涯==
===中国国内时期===
1978年2月考入中国人民解放军[[海政歌舞团|海政歌舞团]]，担任独奏演员，弹拨乐首席。1982年考入[[中央音乐学院|中央音乐学院]]。在[[海政文工团|海政文工团]]的20年里，曾作为独奏演员随[[海政文工团|海政文工团]]出访[[美国|美国]][[夏威夷|夏威夷]]、[[俄罗斯|俄罗斯]]、[[泰国|泰国]]等地。为音乐故事片《山叮咚》配乐演奏筑琴。在大型[[歌剧|歌剧]]《刘胡兰》、《壮丽的婚礼》、《琴箫月》、《冬雪春花》、《红珊瑚》中担任领奏。受邀团中央，全国青联为欢迎各国友谊大军代表团演出独奏。曾出版独唱专辑磁带作品：《夏天的舞蹈》、《成吉思汗》、《卖馄饨》、《什锦全味》、《狂人舞伴》，并为电视连续剧《西游记》录制独唱《白骨精之歌》。1986年参加首体《群星灿烂歌星演唱会》。电视台，电台播送专题演唱节目。
 
===移民加拿大===
2000年移民[[加拿大|加拿大]]。在[[温哥华|温哥华]]创建《东方琴韵乐团》任艺术总监。<ref>{{cite web|url=http://www.fairchildtv.com/programarchive_detail.php?n=19|title=新時代電視 Fairchild TV|publisher=}}</ref><ref>{{cite web | url=http://v.qq.com/page/h/t/9/h0160f6ult9.html | title=加拿大东方琴韵乐团 艺术总监高旭东 采访}}</ref>。

高旭东创作改编了琵琶古筝合奏曲《快乐的边寨》、《梁祝》、《彝族舞曲》、《天路》、《红河谷》等作品。<ref>http://www.lahoo.ca/article-35035-1.html 加拿大乐活网. [2014-12-1].  </ref><ref>http://oldgcp.gcpnews.com/articles/2014-01-31/C1000_111264.html </ref><ref>{{cite web | url=http://www.lahoo.ca/article-16151-1.html | title=2月2日汪峰相约温哥华 压轴福满温诚}}</ref> 2008年在温哥华组织"你我同心"大型慈善晚会，2010年同[[美国|美国]]西雅图交响乐团合作演出琵琶[[协奏曲|协奏曲]]《草原小姐妹》，2013年在交响乐[[大闹天宫|大闹天宫]]担任琵琶领奏，2013年被温哥华华人妇女协会评为「加拿大杰出女性」<ref>{{cite web | url=http://www.lahoo.ca/article-5607-1.html | title=温哥华2013杰出女性颁奖盛典成功举办 | work=温哥华乐活网 | accessdate=2015-12-28}}</ref>，<ref>{{cite web | url=http://www.lahoo.ca/thread-643-1-1.html | title=2013杰出女性颁奖盛典之温哥华东方琴韵乐团表演}}</ref>。 2015年担任「国际珠宝女王大赛」[[加拿大|加拿大]]赛区评委，<ref>{{cite web | url=http://www.lahoo.ca/thread-43588-1-1.html | title=第二届奥秘纳国际珠宝女王大赛温哥华赛区新闻发布会}}</ref>。2015年获得小金钟大赛加拿大赛区音乐教育成就奖。及社区各大机构的表彰感谢奖状。被[[人民日报|人民日报]]海外版誉为“传承东方古韵的使者”。<ref>{{cite web|url=http://paper.people.com.cn/rmrbhwb/html/2006-08/28/content_10216372.htm|title=高旭东：传承东方古韵的使者|publisher=}}</ref><ref>{{cite web|url=http://www.dushi.ca/tor/news/bencandy.php/fid12/lgngbk/aid116333|title=民乐教师高旭东：东方琴韵中的文化传承|publisher=}}</ref>
