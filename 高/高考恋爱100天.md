《'''高考恋爱100天'''》是一款独立制作的[[戀愛模擬遊戲|戀愛模擬遊戲]]，由[[中国大陆|中国大陆]]团体“落叶岛项目组”和“橘子班”联合制作；讲述主角在中国大陆[[普通高等学校招生全国统一考试|高考]]前夕发生的一系列恋爱故事。该游戏于2015年5月29日始在包括[[Steam|Steam]]平台在内的多个平台发售<ref name=":0">{{Cite web|url = http://www.yxdown.com/news/201506/194510.html|title = 《高考恋爱100天》制作人：国产AVG走进Steam是挑战|accessdate = 2015-06-20|author = 游迅网，大众软-{}-件|date = 2015-06-06|publisher = |archive-url = https://web.archive.org/web/20150620181539/http://www.yxdown.com/news/201506/194510.html|archive-date = 2015-06-20|dead-url = yes}}</ref>。

== 剧情大纲 ==

就读于“华懿高级中学”高三六班的男子高中生——高练，在离高考只有一百天时向同班女生木馨告白并被接受。然而高考迫在眉睫，他必须在班级中隐瞒二人的恋情，并在高考和恋爱中做出平衡或抉择；而这些抉择将会决定他以及其他人的命运。<ref name=":1">落叶岛项目组&橘子班.《高考恋爱100天》.</ref>

全部结局皆为主人公从高中毕业和毕业后情况。游戏结局共有54种（Steam版为57种）。

== 游戏系统 ==

在该游戏中设有“学习”；“体力”；“压力”等不同能力值，玩家每隔一段时间即可发出指令，对能力值造成影响。能力值亦受固定剧情与随机事件的影响。而触发的事件，剧情选项与能力值将共同影响到剧情走向与故事的最终结局<ref name=":1" />。

另外在游戏过程中可以获得一些道具来提升主人公的能力值，部分道具可以直接影响到最终结局走向。

== 主要登场人物 ==
; 高练 {{efn|该名字为主角的默认名，可由玩家更改。}}
: 本作男主角，玩家主视角角色。成绩偏中，性格迟钝，似乎在高考问题上和父母有隔阂。在高考前一百天向同班同学木馨告白，成为她的恋人。

; 木馨（{{配音2|CN|霜叶}}）
: 本作女主角之一。短发，可爱的“仓鼠系”美少女。喜爱美食和小动物；常受母亲严管。在高考前一百天成为高练的恋人。

; 魏通（{{配音2|CN|小伊}}）
: 高练的朋友。因以胃痛为由缺课以及名字谐音而得到“胃痛”的绰号。富有激情，称呼高练为“伙伴”。

; 曲琪（{{配音2|CN|Ophelly}}）
: 高三六班班长。对待同学颇为严格负责。害怕突发事件。常误称乔伊为副班长。

; 罗小涵（{{配音2|CN|月夜桥}}）
: 本作女主角之一。黑色长发的美少女，性格平和坚定，对一些事物有见解。因父母离异而独立生活，但在特定情况下有脆弱的一面。

; 乔伊（{{配音2|CN|Magic 小狼}}）
: 高三六班学习委员。知识丰富，擅长处理突发事件。在游戏系统中会负责帮助与提示。<ref>{{Cite web|url = http://www.gamersky.com/handbook/201506/597636_4.shtml|title = 《高考恋爱100天》人物资料介绍 有哪些人物|accessdate = 2015-06-20|author = 游民星空|date = 2015-06-04|publisher = }}</ref>

== 发售情况 ==

2014年起该游戏已先后发布试玩版与通常版<ref name="a">{{Cite web|url = http://www.sonkwo.com/products/1680|title = 高考恋爱100天|accessdate = 2015-06-20|author = 杉果游戏|date = |publisher = }}</ref><ref>{{Cite web|url = http://blog.sina.com.cn/s/blog_e1b7541f0101k2zp.html|title = 《高考恋爱一百天》DEMO 下载|accessdate = 2015-06-20|author = 高考恋爱一百天|date = 2014-05-14|publisher = }}</ref>；Steam版则于2015年5月29日发行<ref name=":0" /><ref>{{Cite web|url = http://store.steampowered.com/app/347620|title = Gaokao.Love.100Days|accessdate = 2015-06-20|author = Steam|date = |publisher = }}</ref><ref>{{Cite web|url = http://news.52pk.com/shtml/20150527/6427716.shtml?1432697532|title = 最接地气的AVG 国人新作《高考恋爱100天》登steam|accessdate = 2015-06-20|author = 52PK|date = 2015-05-27|publisher = }}</ref>，提供英文，和通常版不同的结局以及追加人物，结局数亦增加到57个<ref>{{Cite web|url = http://www.yxdown.com/news/201505/192104.html|title = 《高考恋爱100天》Steam版VS原版 结局增加到57个！|accessdate = 2015-06-20|author = 游迅网|date = 2015-05-29|publisher = }}</ref>。游戏制作组成员“炒饭”于2015年6月接受采访时指出，Steam版发行后，该游戏的销量已逾2500份，并称“英文玩家的数量相当多”<ref name=":0" /><ref name=":2">{{Cite web|url = http://www.yxdown.com/news/201506/193557.html|title = 老外评《高考恋爱100天》：有日漫感觉虽然中国校服很丑|accessdate = 2015-06-20|author = 游迅网|date = 2015-06-03|publisher = }}</ref>。

== 评价 ==

该游戏发售后获得了包括非中文玩家在内的一定数量的好评<ref name=":2" />；搜狐称一些评论指出游戏内容能够反映中国大陆高中生的现状，“具有中国大陆地域特色”，而游戏画风则偏向于日本漫画<ref name=":3">{{Cite web|url = http://mt.sohu.com/20150528/n414010947.shtml|title = 《高考恋爱100天》将登Steam平台：高考+恋爱=？|accessdate = 2015-06-20|author = 搜狐|date = 2015-05-28|publisher = }}</ref>。

== 其它 ==
