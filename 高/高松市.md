'''高松市'''（{{jpn|j=高松市|hg=たかまつし|rm=Takamatsu shi|f=y}}）是位於[[日本|日本]][[四國地方|四國地方]]東北部[[香川縣|香川縣]]中部的城市，也是香川縣[[縣廳所在地|縣廳所在地]]，目前是[[中核市|中核市]]。高松市位於[[讚岐平原|讚岐平原]]，臨[[瀨戶內海|瀨戶內海]]，是一座港口城市。

由於高松市是四國距離本州最近的城市之一，加上過去[[日本國有鐵道|国鐵]]運行的{{link-ja|宇高聯絡船|宇高連絡船}}停靠在高松，使得高松是四國的門戶，眾多中央政府在四國的分支機關和全國性企業的四國分公司都設在高松。加上[[四國電力|四國電力]]和[[四國旅客鐵道|四國旅客鐵道]]等以四國全境為營業範圍的公共服務企業也將總部設在高松，使得高松是四國的政治經濟中心之一。

[[江戶時代|江戶時代]]的高松是[[高松藩|高松藩]]藩主{{link-ja|高松松平家|高松松平家}}的[[城下町|城下町]]，[[高松城_(讚岐國)|高松城]]天守曾是高松的象徵，但在[[明治|明治時代]]被拆除。現在四國最高建築物，於2004年竣工的[[高松地標塔|高松地標塔]]是高松的新地標。

== 歷史 ==
[[File:Yashima01s3200.jpg|thumb]]

高松市內發現了[[舊石器時代|舊石器時代]]的中間西井坪遺跡、國分台遺跡，自20,000年前就已經有人類活動<ref name="History1">{{cite web |url=https://www.city.takamatsu.kagawa.jp/kyouiku/bunkabu/rekisi/naiyou/nenpyou/nenpyou1/nenpyou/n1%20data/nenpyou1.htm　 |script-title=ja:高松の原始・古代 |accessdate=2017-05-09 |publisher={{lang|ja|高松市}} |language=ja }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。[[繩文時代|繩文時代]]時，瀨戶內海水面上升，形成現在的地形。高松市亦有發現這一時期的弓箭等文物<ref name="History1"></ref>。[[彌生時代|彌生時代]]之後，隨著水稻種植和金屬器技術的傳來，高松市區內已開始出現稻作村莊<ref name="History1"></ref>。高松市內的[[古墳時代|古墳時代]]前期遺跡有茶臼山古墳等{{link-ja|前方後圓墳|前方後円墳}}遺跡<ref>{{cite web |url=https://www.city.takamatsu.kagawa.jp/7428.html　 |script-title=ja:高松市茶臼山古墳 |accessdate=2017-05-09 |publisher={{lang|ja|高松市}} |language=ja }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，古墳時代後期的遺跡則有久本古墳等{{tsl|ja|横穴式石室|横穴式石室}}遺跡<ref>{{cite web |url=https://www.city.takamatsu.kagawa.jp/7438.html　 |script-title=ja:久本古墳 |accessdate=2017-05-09 |publisher={{lang|ja|高松市}} |language=ja }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

據《[[日本書紀|日本書紀]]》記載，667年時，為防備[[唐朝|唐朝]]和[[新羅|新羅]]的進攻，當局在[[屋島|屋島]]修築屋島城<ref>{{cite web |url=https://www.city.takamatsu.kagawa.jp/4727.html　 |script-title=ja:屋嶋城 |accessdate=2017-05-09 |publisher={{lang|ja|高松市}} |language=ja }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。[[律令制|律令制]]實施之後，[[讚岐國|讚岐國]]的佛教中心[[讚岐國分寺|讚岐國分寺]]及國分尼寺位於現在的高松市區內<ref>{{cite web |url=http://www.city.takamatsu.kagawa.jp/kyouiku/bunkabu/rekisi/kokubunzi/kokubunzi/kokubunzi.htm　 |script-title=ja:讃岐国分寺跡資料館　讃岐国分寺 |accessdate=2017-05-09 |publisher={{lang|ja|高松市}} |language=ja }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

高松這一地名首次出現在歷史是在[[平安時代|平安時代]]的文獻《[[和名類聚抄|和名類聚抄]]》。在和名類聚抄中，高松鄉是[[山田郡_(香川縣)|山田郡]]的十一鄉之一。相傳古代這裡有很高的松樹，因而得名<ref name="Name">{{cite book |author={{lang|ja|竹内理三}}|script-title=ja:《角川日本地名大辞典 37 香川県》 |year=1978 |publisher={{lang|ja|角川書店}} |location=東京都 |isbn=4-04-001370-0|language=ja}}</ref>{{rp|480}}。

1185年，[[源氏|源氏]]和[[伊勢平氏|平氏]]在屋島爆發[[屋島之戰|屋島之戰]]，是[[源平合戰|源平合戰]]末期的重要戰役<ref>{{cite web |url=http://www.yashima-navi.jp/history/genpei.php　 |script-title=ja:源平合戦古戦場 |accessdate=2017-05-10 |publisher={{lang|ja|屋島ナビ}} |language=ja }}{{dead link|date=十一月 2017 |bot=InternetArchiveBot }}</ref>。自[[南北朝_(日本)|南北朝]]時代開始至[[戰國_(日本)|戰國時代]]，高松由[[細川氏|細川氏]]統治。但在1558年（[[永祿|永祿]]元年），[[阿波國|阿波國]]的[[三好氏|三好氏]]侵略讚岐國並成為高松新的統治者<ref>{{cite web |url=https://www.city.takamatsu.kagawa.jp/kyouiku/bunkabu/rekisi/naiyou/nenpyou/nenpyou1/nenpyou/n2%20data/nenpyou2.htm　 |script-title=ja:高松の中世 |accessdate=2017-05-10 |publisher={{lang|ja|高松市}} |language=ja }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。此後[[長宗我部元親|長宗我部元親]]和[[仙石秀久|仙石秀久]]亦曾是讚岐的統治者<ref name="History2">{{cite web |url=https://www.city.takamatsu.kagawa.jp/kyouiku/bunkabu/rekisi/naiyou/nenpyou/nenpyou1/nenpyou/n3%20data/nenpyou3.htm　 |script-title=ja:高松の近世 |accessdate=2017-05-10 |publisher={{lang|ja|高松市}} |language=ja }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

[[File:Takamatsu_castle14.jpg|thumb]]

1588年，[[豐臣秀吉|豐臣秀吉]]封[[生駒親正|生駒親正]]為讚岐國主。生駒親正在香東郡篦原莊修築[[高松城_(讚岐國)|高松城]]及城下町，此為高松城下町歷史之始<ref name="History2"></ref>。1640年（[[寬永|寬永]]17年），生駒家由於爆發{{tsl|ja|生駒騒動|生駒騒動}}被移封至[[出羽國|出羽國]]{{tsl|ja|矢島藩|矢島藩}}<ref name="History2"></ref>，[[水戶德川家|水戶德川家]]出身的[[親藩大名|親藩大名]][[松平賴重|松平賴重]]成為新的[[高松藩|高松藩]]主。此後直到幕末時期，高松都由{{link-ja|高松松平家|高松松平家}}治理。

高松是一座重視商工業的城下町，市內[[町人|町人]]町的面積大於{{link-ja|武家屋敷|武家屋敷}}，這在10萬石規模的城下町中頗為罕見<ref name="Chishi">{{cite book |author={{lang|ja|森川洋·篠原重則·奥野隆史}}|script-title=ja:《日本の地誌9　中国・四国》 |year=2005 |publisher={{lang|ja|朝倉書店}} |location=東京都 |isbn=4-254-16769-5|language=ja}}</ref>{{rp|423}}。江戶中期的高松人口約有25,000人，其中[[武士|武士]]有1,022人、[[足輕|足輕]]有2,268人，町人佔據壓倒性多數。從人口比例也可看出高松是一座以工商業為主的城下町<ref name="Chishi"></ref>{{rp|424}}。

1871年[[廢藩置縣|廢藩置縣]]之後，高松藩改設為高松縣；但同年又與同屬讚岐國的丸龜縣合併為香川縣<ref>{{cite web |url=https://www.city.takamatsu.kagawa.jp/kyouiku/bunkabu/rekisi/naiyou/nenpyou/nenpyou1/nenpyou/n4%20data/nenpyou4.htm　 |script-title=ja:高松の近・現代 |accessdate=2017-05-10 |publisher={{lang|ja|高松市}} |language=ja }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。1873年，香川縣被併入名東縣，兩年曾短暫恢復設縣；但在1876年，香川縣又被併入[[愛媛縣|愛媛縣]]，直到1888年，香川縣才得以再次恢復設縣<ref>{{cite web |url=http://www.pref.kagawa.lg.jp/content/etc/subsite/kids/rekishi/index.shtml　 |script-title=ja:香川の由来 |accessdate=2017-05-10 |publisher={{lang|ja|香川県}} |language=ja }}{{dead link|date=十一月 2017 |bot=InternetArchiveBot }}</ref>。高松也成為香川縣廳的所在地。

1890年，高松正式設市。當時的高松市面積只有2.85平方公里<ref name="Name"></ref>{{rp|483}}；自明治維新開始至高松設市為止，由於歸屬行政區劃頻繁變動，導致高松的發展緩慢。然而在1910年{{link-ja|宇高連絡船|宇高連絡船}}通航之後，高松市一躍成為四國的大門，開始飛速發展<ref>{{cite web |url=https://www.city.takamatsu.kagawa.jp/kyouiku/bunkabu/rekisi/naiyou/nenpyou/nenpyou2/hyou/meizi/meizi.htm　 |script-title=ja:高松市の明治時代 |accessdate=2017-05-10 |publisher={{lang|ja|高松市}} |language=ja }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

[[大正|大正]]時代之後，高松市開始與附近的町村進行合併，市轄區面積逐漸擴大<ref>{{cite web |url=https://www.city.takamatsu.kagawa.jp/kyouiku/bunkabu/rekisi/naiyou/nenpyou/nenpyou2/hyou/taisyou/taisyou.htm　 |script-title=ja:高松市の大正時代 |accessdate=2017-05-10 |publisher={{lang|ja|高松市}} |language=ja }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。高松還在大正至[[昭和|昭和]]初期發展為四國的鐵路交通中心，便利的交通使得眾多企業將其在四國的分公司設在高松，強化了高松的四國大門地位<ref name="Hyakunen">{{cite book |author={{lang|ja|平岡昭利}}|script-title=ja:《中国・四国 (地図で読む百年)》 |year=1999 |publisher={{lang|ja|古今書院}} |location=東京都 |isbn=4-7722-5030-1|language=ja}}</ref>{{rp|120}}。在[[第二次世界大戰|第二次世界大戰]]期間，高松曾於1945年7月遭到{{link-ja|高松空襲|高松空襲|大規模空襲}}，1,359人遇難，市區約80%的面積被燒毀<ref>{{cite web |url=http://www.soumu.go.jp/main_sosiki/daijinkanbou/sensai/situation/state/shikoku_02.html　 |script-title=ja:高松市における戦災の状況（香川県） |accessdate=2017-05-10 |publisher={{lang|ja|総務省}} |language=ja }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

高松在戰後制定了{{link-ja|戰災復興都市計劃|戦災復興都市計画}}，著重拓寬道路寬度並新設公園。自[[高松車站_(香川縣)|高松車站]]出發的{{link-ja|中央通 (高松市)|中央通り (高松市)|中央通}}沿線集中了眾多企業，成為高松的主幹道<ref name="Hyakunen"></ref>{{rp|120}}。高松市也繼續和附近的町村進行了大規模合併，為都市規模的擴大提供了土地基礎<ref>{{cite web |url=https://www.city.takamatsu.kagawa.jp/kyouiku/bunkabu/rekisi/naiyou/nenpyou/nenpyou2/hyou/syouwa/syouwa.htm　 |script-title=ja:高松市の昭和時代 |accessdate=2017-05-10 |publisher={{lang|ja|高松市}} |language=ja }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

1988年[[瀨戶大橋|瀨戶大橋]]的通車打通了本州和四國之間的陸路交通，更加強了高松的四國交通及經濟中心的地位。加上這一時期正值[[日本泡沫經濟|日本泡沫經濟]]的巔峰期，使得高松市一度出現地價暴漲的局面<ref name="Chishi"></ref>{{rp|412}}。進入[[平成|平成]]時代之後，隨著新[[高松機場|高松機場]]的啟用和[[高松自動車道|高松自動車道]]的開通，高松市的交通更加便捷<ref>{{cite web |url=https://www.city.takamatsu.kagawa.jp/kyouiku/bunkabu/rekisi/naiyou/nenpyou/nenpyou2/hyou/heisei/heisei.htm　 |script-title=ja:高松市の平成時代 |accessdate=2017-05-10 |publisher={{lang|ja|高松市}} |language=ja }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。然而在[[明石海峽大橋|明石海峽大橋]]開通之後，四國的交通動線發生很大變化，高松市的四國大門地位相對下降。為提高高松市的據點地位，當地政府對高松車站進行了名為{{link-ja|Sunport高松|サンポート高松}}的大規模再開發工程。1999年，高松市成為[[中核市|中核市]]，擁有比普通城市更多的自治權<ref>{{cite web |url=https://www.city.takamatsu.kagawa.jp/3458.html　 |script-title=ja:中核市について |accessdate=2017-05-10 |publisher={{lang|ja|高松市}} |language=ja }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。高松市還在「[[平成大合併|平成大合併]]」中合併了部分附近町村，試圖通過擴大市轄區面積來提高都市競爭力<ref>{{cite web |url=http://www.city.takamatsu.kagawa.jp/1703.html　 |script-title=ja:市町合併コーナー |accessdate=2017-05-10 |publisher={{lang|ja|高松市}} |language=ja }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

== 地理 ==
[[File:Takamatsu200910cut.JPG|thumb]]
高松市位於四國東北部，市轄區東西寬約24公里，南北長約36公里，海岸線長115公里，市轄區面積約佔香川縣面積的兩成。香川市地處[[讚岐平原|讚岐平原]]內的高松平原，北臨[[瀨戶內海|瀨戶內海]]，形成了以[[高松港|高松港]]和高松新港為中心的半圓形市區。高松市區西部是由{{tsl|ja|香東川}}的土砂堆積出的[[沖積扇|沖積扇]]、東部是春日川和新川沖積出的泛濫平原、河流的河口部分是[[三角洲|三角洲]]。高松市東北部是突入[[瀨戶內海|瀨戶內海]]的桌形台地，[[屋島|屋島]]及四國本島的最北端{{link-ja|竹居觀音岬|竹居観音岬}}位於這一地區。市區西北部則是丘陵。市轄區海上有[[女木島|女木島]]、{{link-ja|男木島|男木島}}等離島<ref name="Geography">{{cite web |url=https://www.city.takamatsu.kagawa.jp/file/17969_L12_P1_16.pdf　 |script-title=ja:高松市の概要 |accessdate=2017-05-10 |publisher={{lang|ja|高松市}} |language=ja }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

=== 氣候 ===
高松市屬於[[瀨戶內海式氣候|瀨戶內海式氣候]]，降水量較少且日照時間長，因此全年氣候較為溫暖，年平均氣溫約有16度。高松在冬季至春季時有[[黃沙|黃沙]]發生，春季至初夏多[[霧|霧]]。夏季時由於受瀨戶內海沿岸地區特有天氣現象{{link-ja|零級風|凪}}和[[焚風|焚風]]的影響而經常出現高溫天氣<ref name="Geography"></ref>。高松市降水量較少且超過一半集中在梅雨和颱風時期，歷史上多次飽受乾旱困擾，1973年曾發生連續限制用水天數達63天的大乾旱<ref>{{cite web |url=https://www.city.takamatsu.kagawa.jp/kyouiku/bunkabu/rekisi/naiyou/nenpyou/nenpyou2/mini/mi07/mi68.htm　 |script-title=ja:『高松砂漠』 |accessdate=2017-05-10 |publisher={{lang|ja|高松市}} |language=ja }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。1974年{{link-ja|香川用水|香川用水}}竣工之後，高松市的缺水狀況狀況有很大改善<ref>{{cite web |url=http://www.pref.kagawa.jp/tochikai/water/about.html　|script-title=ja:香川用水について |accessdate=2017-05-10 |publisher={{lang|ja|香川県}} |language=ja}}</ref>。但進入平成之後，高松亦在1994年、2005年等年份發生過乾旱<ref>{{cite web |url=http://www.jma-net.go.jp/takamatsu/4/4-6/kassui_takamatsu.html　|script-title=ja:渇水年における高松の降水量 |accessdate=2017-05-10 |publisher={{lang|ja|高松地方気象台}} |language=ja}}</ref>。降水量少的氣候特徵也使得高松市內有眾多蓄水池。由於高松市區直接臨海，因此在颱風經過四國時容易被高潮侵襲。2004年[[颱風暹芭_(2004年)|颱風暹芭]]經過時，高松市內有大片地區被淹<ref>{{cite web |url=http://www.skr.mlit.go.jp/bosai/bosai/kiroku/higai/taihuu16gou.pdf　 |script-title=ja:台風１６号による四国地方の被害状況 |accessdate=2017-05-10 |publisher={{lang|ja|国土交通省}} |language=ja }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

== 人口 ==
1890年設市時，高松市有人口33,863人<ref name="Name"></ref>{{rp|483}}。明治及大正時期高松人口增長較緩慢。1920年日本首次人口普查時，高松有人口46,550人<ref name="Name"></ref>{{rp|483}}。但在進入昭和時代後，隨著都市發展和市轄區範圍擴大，高松的人口增長明顯加速，1935年時有人口86,840人<ref name="Name"></ref>{{rp|483}}。二戰雖然導致高松人口一度大幅減少，但在戰後很快就得到恢復。1950年時，高松市已有人口124,545人，超過戰前最高值<ref name="Name"></ref>{{rp|483}}。1966年時，在經過大規模合併之後，高松市人口更達到259,387人。1976年，高松市人口突破30萬大關<ref>{{cite web|url=https://www.city.takamatsu.kagawa.jp/file/17567_L37_S511001.pdf　|script-title=ja:昭和５１年１０月|accessdate=2017-05-10|publisher={{lang|ja|高松市}}|language=ja}}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。1980年代中期之後，由於[[老齡化|老齡化]]和[[少子化|少子化]]的進展，高松市人口增長速度趨緩。2006年，因實施大規模合併，高松市人口一舉由約34.3萬增加到約42.6萬<ref>{{cite web|url=https://www.city.takamatsu.kagawa.jp/file/17565_L21_H181001.pdf　|script-title=ja:平成１８年１０月|accessdate=2017-05-10|publisher={{lang|ja|高松市}}|language=ja}}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。2017年5月1日時，高松市有人口428,604人<ref>{{cite web |url=https://www.city.takamatsu.kagawa.jp/file/28299_L12_001.xls　 |script-title=ja:年齢別人口 |accessdate=2017-05-10 |publisher={{lang|ja|高松市}} |language=ja }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。近年高松市人口自然增長率為負數，但社會增長率是正數。高松市的[[總和生育率|總和生育率]]有1.62，高於日本平均值，也高於香川縣和四國各主要都市平均水準<ref>{{cite web |url=https://www.city.takamatsu.kagawa.jp/file/24963_L12_jinkou.pdf　 |script-title=ja:たかまつ人口ビジョン |accessdate=2017-05-10 |publisher={{lang|ja|高松市}} |language=ja }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

== 政治 ==
在日本[[眾議院_(日本)|眾議院]]選舉中，平成大合併之前的原高松市轄區與[[土庄町|土庄町]]、[[小豆島町|小豆島町]]、[[直島町|直島町]]同屬於[[香川縣第1區|香川縣第1區]]，在「平成大合併」中併入高松市的地區則與[[丸龜市|丸龜市]]的一部分、[[坂出市|坂出市]]、[[讚岐市|讚岐市]]、[[東香川市|東香川市]]、[[三木町|三木町]]、[[綾川町|綾川町]]、[[宇多津町|宇多津町]]同屬於[[香川縣第2區|香川縣第2區]]<ref>{{cite web |url=http://www.soumu.go.jp/main_content/000240081.pdf　 |script-title=ja:香川県 |accessdate=2017-05-08 |publisher={{lang|ja|総務省}} |language=ja }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。四國是[[自由民主黨_(日本)|自民黨]]有壓倒性優勢的{{link-ja|保守王國|保守王国}}，香川1區亦不例外。在2000年之後的選舉中，除了[[第45屆日本眾議院議員總選舉|2009年總選舉]]由[[民主黨_(1998年)|民主黨]]候選人{{link-ja|小川淳也|小川淳也}}當選之外<ref>{{cite web |url=http://www.yomiuri.co.jp/election/shugiin2009/kaihyou/ya37.htm　 |script-title=ja:小選挙区 香川 |accessdate=2017-05-09 |publisher={{lang|ja|読売新聞}} |language=ja }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，歷次選舉都是自民黨獲勝。該選區現任眾議院議員是{{link-ja|平井卓也|平井卓也}}，他在[[第42屆日本眾議院議員總選舉|2000年眾議院選舉]]時首次當選<ref>{{cite web |url=http://www.jiji.com/jc/giin?d=5de4c4f6e35abe1d059649f987036805&c=syu&rel=ja　|script-title=ja:平井 卓也 |accessdate=2017-05-09 |publisher={{lang|ja|時事ドットコム}} |language=ja}}</ref>，最近的[[第47屆日本眾議院議員總選舉|2014年總選舉]]中以50.7%的得票率連任<ref>{{cite web |url=http://www.yomiuri.co.jp/election/shugiin/2017/kaihyou/ya37.html　 |script-title=ja:【小選挙区】香川 |accessdate=2017-10-24 |publisher={{lang|ja|読売新聞}} |language=ja }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

高松市現任市長是{{link-ja|大西秀人|大西秀人}}，他在2007年首次競選市長時因無競爭對手而無投票當選<ref>{{cite web |url=http://www.yomiuri.co.jp/election/local2007/l_kaihyou/yj37.htm　 |script-title=ja:開票結果 |accessdate=2017-05-09 |publisher={{lang|ja|読売新聞}} |language=ja }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，此後在2011年和2015年也順利連任<ref>{{cite web |url=http://www.yomiuri.co.jp/election/local/2015/kaihyou/yj37.html　 |script-title=ja:開票結果 |accessdate=2017-05-09 |publisher={{lang|ja|読売新聞}} |language=ja }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。{{link-ja|高松市議會|高松市議会}}共有40名議員，其中自民黨議員最多，有18人<ref>{{cite web |url=https://www.city.takamatsu.kagawa.jp/24829.html　 |script-title=ja:会派別名簿 |accessdate=2017-05-09 |publisher={{lang|ja|高松市議会}} |language=ja }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。{{link-ja|香川縣議會|香川県議会}}中高松市則有15名議員<ref>{{cite web |url=https://www.pref.kagawa.lg.jp/gikai/　|script-title=ja:議員の選挙区 |accessdate=2017-05-09 |publisher={{lang|ja|香川県議会}} |language=ja}}</ref>。2015年度，高松市財政收入約1634億日元，支出約1568億日元，略有實現財政盈餘<ref>{{cite web |url=http://www.city.takamatsu.kagawa.jp/file/3646_L25_H28.9.15kouhou.pdf　 |script-title=ja:平成２８年９月１５日号 広報たかまつ |accessdate=2017-05-09 |publisher={{lang|ja|高松市}} |language=ja }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。高松市的財政力指數為0.813，略高於中核市平均水準<ref>{{cite web |url=http://www.city.takamatsu.kagawa.jp/10713.html　 |script-title=ja:3-(1)財政力指数の推移 |accessdate=2017-05-09 |publisher={{lang|ja|高松市}} |language=ja }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

== 文化教育 ==
[[File:Kama-age_sanuki_udon_by_sekido_in_Takamatsu,_Kagawa.jpg|250px]]]]
高松市的方言屬於{{link-ja|四國方言|四国方言}}中的{{link-ja|讚岐方言|讃岐弁}}。由於高松是四國島內和本州往來最密切的地區之一，導致方言也受到[[近畿方言|近畿方言]]和{{link-ja|中國方言 (日本)|中国方言|中國方言}}較強的影響。但讚岐方言使用{{link-ja|讚岐式音調|讃岐式アクセント}}，和四國大多數地區使用的{{link-ja|京阪式音調|京阪式アクセント}}不同<ref>{{cite book |author={{lang|ja|真田信治}}|script-title=ja:《方言学》 |year=2011 |publisher={{lang|ja|朝倉書店}} |location=東京都 |isbn=978-4-254-51524-4|language=ja}}</ref>{{rp|91}}。[[文藝春秋|文藝春秋]]及日本兩大文學獎[[直木獎|直木獎]]和[[芥川龍之介獎|芥川龍之介獎]]的創辦人[[菊池寬|菊池寬]]是最能代表高松的作家，代表作有《[[真珠夫人|真珠夫人]]》。現在高松市內有菊池寬紀念館，收藏展示有關菊池寬的資料和藏品<ref>{{cite web |url=http://www.city.takamatsu.kagawa.jp/kyouiku/bunkabu/kikuti/shisetsu.html |script-title=ja:菊池寛記念館 |accessdate=2017-05-09 |publisher={{lang|ja|高松市}} |language=ja |title=存档副本 |archive-url=https://web.archive.org/web/20171107010030/http://www.city.takamatsu.kagawa.jp/kyouiku/bunkabu/kikuti/shisetsu.html |archive-date=2017-11-07 |dead-url=yes }}</ref>。

香川縣的飲食文化以[[讚岐烏冬|讚岐烏冬]]而著稱，高松市亦不例外。讚岐烏冬的特徵包括發麵時加水量較多、麵質較為筋道、使用佐料（生薑、蔥）較多等特徵<ref>{{cite web |url=http://www.hokuto-kona.net/udon/zatsugaku/sanuki-udon.html |script-title=ja:さぬきうどんの特徴 |accessdate=2017-05-09 |publisher={{lang|ja|北東製粉}} |language=ja}}</ref>。高松最大的節慶活動是每年8月12日至14日舉行的{{link-ja|讚岐高松祭|さぬき高松まつり}}，以盛大的煙花大會而著稱<ref>{{cite web |url=http://www.shikoku.gr.jp/event/82 |script-title=ja:さぬき高松まつり |accessdate=2017-05-09 |publisher={{lang|ja|巡るゆく四国}} |language=ja}}</ref>。高松市還在每年2月舉辦{{link-ja|讚岐電影節|さぬき映画祭}}，以通過電影振興當地文化並培育電影人才<ref>{{cite web |url=http://sanukieigasai.com/about/index.html |script-title=ja:さぬき映画祭とは |accessdate=2017-05-09 |publisher={{lang|ja|さぬき映画祭}} |language=ja}}</ref>。{{link-ja|香川縣立博物館|香川県立ミュージアム}}和{{link-ja|高松市美術館|高松市美術館}}是高松的兩大文化設施。香川縣立博物館的前身是香川縣歷史博物館，主要收藏展示有關香川縣歷史與文化的藏品，收藏有[[日本國寶|日本國寶]]{{link-ja|藤原佐理|藤原佐理}}筆詩{{link-ja|懐紙|懐紙}}<ref>{{cite web |url=http://www.pref.kagawa.lg.jp/kmuseum/shoukai/ |script-title=ja:館のご紹介 |accessdate=2017-05-09 |publisher={{lang|ja|香川県立ミュージアム}} |language=ja}}</ref>。高松市美術館以收藏展示現代美術品和香川縣的美術品為主<ref>{{cite web |url=https://www.city.takamatsu.kagawa.jp/museum/takamatsu/about/index.html |script-title=ja:高松市美術館概要 |accessdate=2017-05-09 |publisher={{lang|ja|高松市美術館}} |language=ja |title=存档副本 |archive-url=https://web.archive.org/web/20170218181740/http://www.city.takamatsu.kagawa.jp/museum/takamatsu/about/index.html |archive-date=2017-02-18 |dead-url=yes }}</ref>。

{{link-ja|四國新聞|四国新聞}}總部位於高松市，是香川縣內唯一的地方報紙，日發行數約21萬份，佔香川縣內約六成的市場份額<ref>{{cite web |url=https://www.shikoku-np.co.jp/info/company/recruit/information.htm |script-title=ja:会社概要 |accessdate=2017-05-09 |publisher={{lang|ja|四国新聞}} |language=ja}}</ref>。設在高松市的電視台有[[NHK高松放送局|NHK高松放送局]]、[[西日本放送|西日本放送]]<ref>{{cite web |url=http://www.rnc.co.jp/coinfo/ |script-title=ja:会社概要 |accessdate=2017-05-09 |publisher={{lang|ja|西日本放送}} |language=ja}}</ref>、[[瀨戶內海放送|瀨戶內海放送]]<ref>{{cite web |url=http://www.ksb.co.jp/company/index |script-title=ja:会社概要 |accessdate=2017-05-09 |publisher={{lang|ja|瀬戸内海放送}} |language=ja}}</ref>。由於香川縣和[[岡山縣|岡山縣]]共享一個電視收視區域，因此高松市除了這三家電視台之外，亦能收看[[岡山放送|岡山放送]]、[[山陽放送|山陽放送]]、[[瀨戶內電視台|瀨戶內電視台]]的節目。以高松為主場的職業體育俱樂部包括[[日本職業足球乙級聯賽|日本職業足球乙級聯賽]]球隊[[讚岐釜玉海|讚岐釜玉海]]<ref>{{cite web |url=http://www.kamatamare.jp/player/club.html |script-title=ja:カマタマーレについて |accessdate=2017-05-09 |publisher={{lang|ja|カマタマーレ讃岐}} |language=ja |title=存档副本 |archive-url=https://web.archive.org/web/20170529232100/http://www.kamatamare.jp/player/club.html |archive-date=2017-05-29 |dead-url=yes }}</ref>、棒球[[四國島聯盟|四國島聯盟]]球隊{{link-ja|香川橄欖勇士|香川オリーブガイナーズ}}<ref>{{cite web |url=http://www.oliveguyners.jp/beginner/ |script-title=ja:四国アイランドリーグplusとは |accessdate=2017-05-09 |publisher={{lang|ja|香川オリーブガイナーズ}} |language=ja}}</ref>、籃球[[B聯賽|B聯賽]]球隊{{link-ja|香川五箭|香川ファイブアローズ}}<ref>{{cite web |url=https://www.fivearrows.jp/roster/ |script-title=ja:選手一覧 |accessdate=2017-05-09 |publisher={{lang|ja|香川ファイブアローズ}} |language=ja}}</ref>。高松市每年還舉辦馬拉松、鐵人三項大賽等普通市民亦可參加的體育活動。

[[香川大學|香川大學]]是高松市和香川縣唯一的國立大學。香川大學成立於1949年，是一個擁有六個學院的綜合性大學。除了醫學院和農學院之外，其他四個學院均位於高松市<ref>{{cite web |url=http://www.kagawa-u.ac.jp/information/outline/development/ |script-title=ja:大学の沿革 |accessdate=2017-05-09 |publisher={{lang|ja|香川大学}} |language=ja}}</ref>。高松市還有{{link-ja|香川縣立保健醫療大學|香川県立保健医療大学}}<ref>{{cite web |url=http://www.kagawa-puhs.ac.jp/about/ |script-title=ja:大学紹介 |accessdate=2017-05-09 |publisher={{lang|ja|香川県立保健医療大学}} |language=ja}}</ref>一所公立大學，{{link-ja|高松大學|高松大学}}一所私立大學<ref>{{cite web |url=http://www.takamatsu-u.ac.jp/intro/history.html |script-title=ja:沿革 |accessdate=2017-05-09 |publisher={{lang|ja|高松大学}} |language=ja}}</ref>。

==交通==

和日本的眾多地方都市類似，高松市的公共交通較不發達，私家車承擔了過半數交通流量。由於高松市地形平坦，非常適合腳踏車出行，腳踏車亦在高松市的交通總量中佔比約兩成<ref name="Traffic">{{cite web|url=https://www.mlit.go.jp/common/000210984.pdf　|script-title=ja:高松市交通戦略計画|accessdate=2017-05-10|publisher={{lang|ja|国土交通省}}|language=ja}}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。高松是四國地方鐵路交通最發達的地區。JR四國在高松有連接愛媛、香川兩縣的[[予讚線|予讚線]]和連接高松、[[德島市|德島]]兩市的[[高德線|高德線]]兩條路線<ref>{{cite web |url=http://www.jr-shikoku.co.jp/01_trainbus/st_number/map.pdf |script-title=ja:路線図 |accessdate=2017-05-10 |publisher={{lang|ja|JR四国}} |language=ja}}</ref>，但[[土讚線|土讚線]]和[[本四備讚線|本四備讚線]]的部分列車經由予讚線行經高松市。[[高松車站_(香川縣)|高松車站]]是高松的鐵路交通中心，日均客流量達12,579人<ref name="JR"></ref>。高松車站的主要快速和[[特別急行列車|特急列車]]有開往[[岡山市|岡山]]的{{link-ja|Marine Liner|マリンライナー}}、開往德島的[[渦潮號列車|渦潮號列車]]、開往[[松山市|松山]]的[[石鎚號列車|石鎚號列車]]、開往[[高知市|高知]]的[[四萬十號列車|四萬十號列車]]<ref>{{cite web |url=http://www.jr-shikoku.co.jp/01_trainbus/jikoku/pdf/takamatsu.pdf |script-title=ja:発車時刻 |accessdate=2017-05-10 |publisher={{lang|ja|JR四国}} |language=ja}}</ref>。[[高松琴平電氣鐵道|高松琴平電氣鐵道]]則是高松唯一的私鐵，運營有[[琴平線|琴平線]]、{{link-ja|長尾線|高松琴平電気鉄道長尾線}}、{{link-ja|志度線|高松琴平電気鉄道志度線}}三條路線<ref>{{cite web |url=http://www.kotoden.co.jp/publichtm/kotoden/kensaku/routemap.htm |script-title=ja:ことでん路線図 |accessdate=2017-05-10 |publisher={{lang|ja|高松琴平電気鉄道}} |language=ja}}</ref>。高松市內還有{{link-ja|八栗纜車|八栗ケーブル}}一條纜車路線<ref>{{cite web |url=http://www.shikoku-cable.co.jp/yakuri/annai.htm |script-title=ja:八栗ケーブルの概要 |accessdate=2017-05-10 |publisher={{lang|ja|八栗ケーブル}} |language=ja |title=存档副本 |archive-url=https://web.archive.org/web/20160518201037/http://www.shikoku-cable.co.jp/yakuri/annai.htm |archive-date=2016-05-18 |dead-url=yes }}</ref>。

==觀光資源==
[[File:Ritsurin.JPG|250px]]
高松市人文觀光資源豐富，是瀨戶內海沿岸地區觀光熱點都市之一。[[栗林公園|栗林公園]]位於高松市區南部，面積75公頃，被日本政府認定為特別名勝。栗林公園起源於16世紀後期時當地的豪族佐藤氏的庭園，此後經過了生駒氏和松平氏兩代高松統治者的擴建，成為高松藩的別邸。廢藩置縣之後，栗林公園交由政府管理並對公眾開放<ref>{{cite web |url=http://www.my-kagawa.jp/files/user/site/32551436c07aacb57666148a0c7a78319bf31512.pdf |script-title=ja:栗林公園の歴史 |accessdate=2017-05-10 |publisher={{lang|ja|香川県}} |language=ja}}</ref>。現在栗林公園被公認為是[[日本庭園|日本庭園]]的最高傑作之一，獲得米其林指南的三星級肯定<ref>{{cite web|url=http://www.my-kagawa.jp/ritsuringarden/feature/ritsuringarden/garden　|script-title=ja:特別名勝　栗林公園（りつりんこうえん）|accessdate=2017-05-10|publisher={{lang|ja|うどん県旅ネット}}|language=ja}}{{dead link|date=十一月 2017 |bot=InternetArchiveBot }}</ref>。公園內的赤壁、紫雲山等景點巧妙了利用了自然地形作為借景，並且冬可賞梅、春可賞櫻、夏可賞荷、秋可觀紅葉，四季皆有不同景觀。在栗林公園的諸多景觀中，又以自外觀仿效[[富士山|富士山]]的飛來峰最為知名<ref>{{cite web|url=http://www.my-kagawa.jp/ritsuringarden/feature/ritsuringarden/viewspot　|script-title=ja:鑑賞ポイント|accessdate=2017-05-10|publisher={{lang|ja|うどん県旅ネット}}|language=ja}}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

在[[四國八十八箇所|四國八十八箇所]]中，第80番至第85番（[[讚岐國分寺|國分寺]]、{{link-ja|白峰寺|白峯寺}}、{{link-ja|根香寺|根香寺}}、{{link-ja|一宮寺|一宮寺}}、{{link-ja|屋島寺|屋島寺}}、{{link-ja|八栗寺|八栗寺}}）位於高松市境內<ref>{{cite web|url=http://www.88shikokuhenro.jp/kagawa.html　|script-title=ja:香川県(讃岐)の霊場|accessdate=2017-05-10|publisher={{lang|ja|四国八十八ヶ所霊場}}|language=ja}}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。[[四國村|四國村]]是一個以自江戶時代至明治時代的民宅為主題的露天博物館，保存復原了日本各地的民居<ref>{{cite web|url=http://www.shikokumura.or.jp/intro/index.html　|script-title=ja:四国村とは？|accessdate=2017-05-10|publisher={{lang|ja|四国村}}|language=ja}}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。[[高松城_(讚岐國)|高松城]]是日本三大水城之一，修築時充分利用了臨海這一地形特徵，[[護城河|護城河]]引海水進入，使軍艦可直接進入城內。現在雖然[[天守|天守]]已不存在，但仍保存有天守台、月見櫓、艮櫓、水手御門、渡櫓等建築<ref>{{cite web|url=http://www.takamatsujyo.com/rekisi.htm　|script-title=ja:高松城の歴史|accessdate=2017-05-10|publisher={{lang|ja|玉藻公園}}|language=ja}}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，是[[日本100名城|日本100名城]]之一。
