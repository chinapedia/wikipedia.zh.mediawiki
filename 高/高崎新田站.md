'''高崎新田站'''（{{jpn|j=高崎新田駅|kyu=高崎新田驛|hg=たかさきしんでんえき|hgt=|rm=Takasaki-Shinden eki|f=y}}）位於[[宮崎縣|宮崎縣]][[都城市|都城市]][[高崎町|高崎町]]，是[[九州旅客鐵道|九州旅客鐵道]]（JR九州）
[[吉都線|吉都線]]上的車站。車站片假名為「{{lang|ja|たか'''さ'''き}}」 ，與町名「{{lang|ja|たか'''ざ'''き}}」有些微差異。

== 車站構造 ==
島式月台1面2線的地上站。無人車站，有一座混凝土建築的站房。

== 車站周邊 ==
* 都城市公所 高崎總合分所（原為高崎町公所）
* 高崎郵局
* 都城市立高崎小學

== 历史 ==
* 1913年（大正2年）5月11日 - 設立。
* 1968年12月 - 完成現有站房。
* 1980年10月1日 - 廢止貨物業務。
* 1987年（昭和62年）4月1日 - [[國鐵分割民營化|國鐵分割民營化]]後成為九州旅客鐵道（JR九州）轄下的車站。
