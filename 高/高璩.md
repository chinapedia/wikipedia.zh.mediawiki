'''高璩'''（{{bd|？||865年|catIdx=G高}}），字'''瑩之'''，[[唐朝|唐朝]]官员，[[唐懿宗|唐懿宗]]年间曾短暂任[[宰相|宰相]]，为期两个月。

== 家世 ==
高璩是[[北齐|北齐]]皇族后裔，先祖[[高士廉|高士廉]]在[[唐太宗|唐太宗]]朝拜相，尽管高家并未因此而显贵，<ref>《[[新唐书|新唐书]]》卷七十一</ref>但高璩的伯父[[高少逸|高少逸]]和父亲[[高元裕|高元裕]]都成为朝廷名臣。<ref name=NBT177>[[:s:新唐书/卷177|《新唐书》卷一百七十七]]</ref>

== 早期仕途 ==
高璩在[[唐宣宗|唐宣宗]][[大中|大中]]三年（849年）中[[进士|进士]]，<ref>[[:s:旧唐书/卷178|《旧唐书》卷一百七十八]]</ref>后在地方上任幕僚。宣宗年间又任[[左拾遗|左拾遗]]、[[翰林學士|翰林學士]]。尽管当时翰林学士被提拔是罕事，但高璩却被提拔为[[諫議大夫|諫議大夫]]。<ref name=NBT177/>历任丞郎、判度支。<ref name=BT171>[[:s:旧唐书/卷171|《旧唐书》卷一百七十一]]</ref>

== 拜相 ==
唐宣宗之子唐懿宗继位后，[[咸通|咸通]]三年（862年）八月任时为翰林学士承旨、朝议大夫、守尚书金部侍郎、知制诰、[[上柱国|上柱国]]、赐紫金鱼袋的高璩为检校[[礼部尚书|礼部尚书]]、兼[[梓州|梓州]]刺史、[[御史大夫|御史大夫]]、充[[东川节度使|剑南东川节度副使、知节度事、管内观察处置等使]]，<ref>《重修承旨学士壁记》</ref>散官勋如故。<ref>[[:s:授高璩剑南东川节度使制|:s:授高璩剑南东川节度使制]]</ref>不久又召回[[长安|长安]]，于六年（865年）四月任为[[兵部侍郎|兵部侍郎]]，授[[同中書門下平章事|同中書門下平章事]]，于是成为宰相。<ref>[[:s:旧唐书/卷19上|《旧唐书·懿宗纪》]]作四年（863年）十一月拜相，五年（864年）五月加[[中书侍郎|中书侍郎]]，六年二月罢相。</ref>[[薛逢|薛逢]]作诗《越王楼送高梓州入朝》。高璩又守[[中书侍郎|中书侍郎]]。<ref name=BT171/>与[[刘瞻|刘瞻]]都引荐故人故弘文馆学士判馆事[[刘三复|刘三复]]子[[刘邺|刘邺]]为[[左拾遗|左拾遗]]。<ref>[[:s:旧唐书/卷177|《旧唐书》卷一百七十七]]</ref>六月高璩就去世了，<ref>[[:s:新唐书/卷9|《新唐书》卷九]]</ref><ref>[[:s:资治通鉴/卷250|《资治通鉴》卷二百五十]]</ref>并得到追谥。但因为高璩生前和不良人士来往频繁，在[[太常博士|太常博士]][[曹邺|曹邺]]坚持下，他得了一个“'''剌'''”的恶谥。<ref name=NBT177/>

《旧唐书·[[崔彦昭|崔彦昭]]传》称懿宗驾崩后[[唐僖宗|唐僖宗]]初登基时，高璩与同年进士的同僚宰相[[赵隐|赵隐]]推荐同为同年进士的崔彦昭长于治财赋。疑误，因高璩早已去世。
