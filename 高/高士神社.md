'''高士神社'''位於[[臺灣|臺灣]][[高雄州|高雄州]][[恆春郡|恆春郡]]蕃地（今[[屏東縣|屏東縣]][[牡丹鄉|牡丹鄉]]高士村），建於[[日本統治時期|日本統治時期]][[昭和|昭和]]十四年（1939年），1945年二戰後一度毀壞廢棄，2015年重建<ref>{{Cite web |url=http://www.appledaily.com.tw/realtimenews/article/new/20150810/667010/|title=台版「靖國神社」　明牡丹鄉揭幕|author=|date=2015年8月10日|publisher=蘋果日報|language=zh-tw|accessdate=2016年3月11日|quote=台版「靖國神社」，明天將在屏東縣牡丹鄉高士村揭幕。日本《產經新聞》今天報導，日本神職人士佐藤健一有感於311東日本大震災時台灣對日本的援助，以及村民希望重建當地神社的心願，決定自費1千萬日圓（約254萬元台幣）協助重新打造，新神社將於明天揭幕。}}</ref>，是[[臺灣|臺灣]][[第二次世界大戰|戰後]]<ref>{{Cite web |url=http://www.sankei.com/west/news/150810/wst1508100053-n1.html|title=台湾の小さな「靖国」再建　日本人神職、住民の熱望応え　「ここで会おう」誓い出征　１１、１２日に神事|author=|date=2015年8月10日|publisher=産経新聞|language=zh-tw|accessdate=2016年3月11日|quote=台湾南部の小さな村に、ある小さな神社があった。「日本軍」として出征した住民もいた村にとって、戦前の日本統治下につくられた神社は、日本の靖国神社のような存在だった。戦後荒廃が進んだが、住民からの強い要望を受け、横浜市中区の神職、佐藤健一さん（４４）が新しい社を制作した。戦後７０年の終戦日を前に宮司となり、遷座祭などの神事を行い、現地の戦没者も英霊として祭る。}}</ref>第一座供奉神祇的神社。

== 沿革 ==
1939年配合[[皇民化運動|皇民化運動]]而建立，名為「クスクス祠」（Kuskus祠╱高士祠），主祀[[天照大神|天照大神]]。

戰後原址曾荒廢多時，只剩石垣基座。[[耆老|耆老]]陳清福表示，二次大戰時的[[台籍日本兵|台籍日本兵]]族人，出征前役男們在神社前與家人互別並約定「如果我回不來了，以後就來神社相見」，因此村民希望重建神社實現親人的遺願。[[日本李登輝之友會|日本李登輝之友會]]會員、日本[[神職人員|神職人員]]佐藤健一<ref>{{Cite web |url=https://www.youtube.com/watch?v=IfVMEJNQ4wk|title=台湾 高士神社再建とクスクス村の今|author=[[樱花频道|チャンネル桜]]|date=2015年10月8日|publisher=YouTube|language=ja|accessdate=2016年3月11日|quote=}}</ref>為回饋[[日本|日本]][[311大地震|311大地震]]台灣踴躍的捐助，耗資1000萬日圓贈送木造神社<ref>{{Cite web |url=http://news.ltn.com.tw/news/world/breakingnews/1322068|title=日人耗資256萬幫屏東復原日治時代神社|author=張茂森|date=2015年5月19日|publisher=自由時報|language=zh-tw|accessdate=2016年3月11日|quote=日本神職人士佐藤健一耗資1千萬日圓（約256萬新台幣）和一年多的時間，為屏東縣牡丹鄉高士村復元已經荒廢70年的日治時代舊神社，整座全新神社已在18日運抵橫濱碼頭等待運至台灣，預定七月上旬抵達高雄。}}</ref>，於2015年8月11日舉行招魂遷座祭，同月12日舉行鎮座祭，奉祀高士佛部落戰亡的先靈，採用日本[[神道|神道]]與[[排灣族|排灣族]]傳統儀式，並在[[台灣基督長老教會|台灣基督長老教會]][[牧師|牧師]]祈禱下成立。此外，成立高士佛神社管理委員會，由佐藤健一任高士神社宮司，祭儀將採用傳統排灣族部落的儀式。2016年5月7日豎立鳥居。<ref>{{Cite web |url=http://news.ltn.com.tw/news/world/breakingnews/1661947|title=日人捐贈屏東高士神社大鳥居 5/7豎立|author=張茂森|date=2016年4月12日|publisher=自由時報|language=zh-tw|accessdate=2016年4月15日|quote=}}</ref>

== 建築樣式 ==
重建後的高士神社，高約2.30公尺，寬約2公尺，扁柏<ref>{{Cite web |url=http://news.pts.org.tw/article/303734?NEENO=303734|title=日人出資重建 屏東高士神社重生|author=郭采彥、孟昭權|date=2015年8月11日|publisher=公共電視台|language=zh-tw|accessdate=2016年3月11日|quote=佐藤建一:「因為與村民達成了約定 所以今年能夠完成 很開心 高230公分、寬166公分，重400公斤的神社，全部以扁柏打造，村民也將整修周邊成紀念公園。}}</ref>為原材，屋頂則全面銅板，石垣基座則保留戰前原貌。建築造型採「全面千鳥破風」與「軒唐破風」<ref>{{Cite web |url=http://blogs.yahoo.co.jp/munehico/folder/507200.html|title=社殿概要|author=晃胤|date=2015年8月25日|publisher=台湾 復興第一之宮 髙士神社公式ブログ|language=ja|accessdate=2016年3月11日|quote=当神社御本殿は躯体のみで高さ２３０センチで約３尺３寸（和尺≒約１メートル）の基礎の上に鎮座しており、地面から屋根テンバまでは、およそ３．３メートルです。}}</ref>。

== 年表記事 ==
* 1939年：初代高士神社建立
* 1945年：初代神社廢棄
* 2015年8月10日：神社重建落成
* 2016年5月5日：鳥居安裝完成
* 2017年5月6日：官方授權線上社務所成立
* 2018年5月5日：首度由台灣人神官舉行神社例祭
