'''高桥镇'''是[[中华人民共和国|中华人民共和国]][[浙江省|浙江省]][[宁波市|宁波市]][[海曙区|海曙区]]一个[[建制镇|建制镇]]，位于海曙区西部，西接[[余姚市|余姚市]]，南邻[[集士港镇|集士港镇]]、[[古林镇|古林镇]]，东临[[海曙区|海曙区]][[望春街道|望春街道]]，辖区面积53平方千米，镇政府驻文化路11号。<ref name="xzqh">[http://www.xzqh.org/html/show.php?contentid=6336 浙江省行政区划 >> 宁波市 >> 鄞州区 >> 高桥镇]</ref>

== 沿革 ==
高桥镇得名于境内的一座单孔石拱[[高桥_(宁波桥梁)|高桥]]。1949年前为后扩乡，1949年后为高桥乡，1992年望春、岐阳二乡并入后改高桥镇。<ref name="xzqh" />

== 下辖 ==
高桥镇下辖3个居民区，20个行政村。<ref name="xzqh" />
* 居民区：高桥、长乐新村、紫薇。

== 旅游资源 ==
高桥镇为中国民间传说[[梁祝传说|梁祝传说]]发源地之一，境内建有梁山伯庙。近年建有[[梁祝文化公园|梁祝文化公园]]，依照梁祝传说为主线，形成大型爱情主题公园。<ref>[http://www.gaoqiao.gov.cn/12/93.htm 宁波梁祝文化公园]{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>其中的梁祝化蝶雕塑是宁波著名的城市雕塑。<ref>[http://www.gaoqiao.gov.cn/21/662.htm 《梁祝》雕塑，阿拉宁波人顶喜欢] {{webarchive|url=https://web.archive.org/web/20110325040050/http://www.gaoqiao.gov.cn/21/662.htm |date=2011-03-25 }}</ref>

== 交通 ==
高桥镇交通便捷。[[宁波绕城高速公路|宁波绕城高速公路]]在高桥镇境内设高桥枢纽。望春路向西连接G92高速公路大隐出口。建设中的[[通途路|通途路]]快速路通过高桥镇。[[宁波轨道交通1号线|宁波轨道交通1号线]]起于高桥镇，设[[高桥西站_(宁波市)|高桥西]]、[[高桥站_(宁波市)|高桥]]、[[梁祝站|梁祝]]、[[芦港站|芦港]]四站和[[江南停车场|江南停车场]]。
