'''高俅'''（{{bd|？||1126年|catIdx=G}}），[[北宋|北宋]]末[[禁軍|禁軍]]統帥之一。因為是[[四大名著|四大名著]]之一《[[水滸傳|水滸傳]]》的主要反面人物而廣為人知。

== 生平 ==

[[北宋|北宋]][[政和|政和]]七年（1117年）正月，[[宋徽宗|宋徽宗]]以[[殿前都指揮使|殿前都指揮使]]高俅為[[太尉|太尉]]。<ref>《宋史·徽宗紀》：「（政和七年正月）庚子，以殿前都指揮使高俅為太尉。」</ref>[[宣和|宣和]]四年（1122年）五月，高俅加[[開府儀同三司|開府儀同三司]]。<ref>《宋史·徽宗紀》：「（宣和四年）五月壬戌，以高俅為開府儀同三司。」</ref>

[[靖康|靖康]]元年（1126年）五月，高俅死。<ref>《宋史·欽宗紀》：「（靖康元年五月己卯）開府儀同三司高俅卒。」</ref>按照慣例，宋欽宗應當為其舉哀，[[太學博士|太學博士]][[李若水|李若水]]上書說：「高俅因受寵信而登據顯赫之位，敗壞國家軍政，導致金人長驅直入，其罪應當與[[童貫|童貫]]相等。他已經未受懲處而死，但還應當追奪其[[官秩|官秩]]，向天下昭示受到朝廷唾棄，而禮部尚且因循慣例，欲加以繁文縟節之禮，豈能平息公論。」<ref>《宋史·李若水傳》：「開府儀同三司高俅死，故事，天子當掛服舉哀，若水言：『俅以幸臣躐躋顯位，敗壞軍政，金人長驅，其罪當與童貫等。得全首領以沒，尚當追削官秩，示與眾棄；而有司循常習故，欲加縟禮，非所以靖公議也。』」</ref>兩日後，高俅被追奪官秩。<ref>《宋史·欽宗紀》：「（靖康元年五月辛巳）追削高俅官。」</ref>

== 宋人筆記 ==

[[南宋|南宋]][[王明清|王明清]]《[[揮麈錄|揮麈錄]]》（「麈」讀音「主」，由鹿、主二字合成）有以下記載：

:高俅者，本[[蘇軾|東坡先生]]小史，筆札頗工。東坡自翰苑出帥中山，留以予[[曾布|曾文肅]]，文肅以史令已多辭之，東坡以屬[[王詵|王晉卿]]。元符末，晉卿為樞密都承旨時，祐陵為端王，在潛邸日，已自好文，故與晉卿善。在殿廬待班，邂逅。王云：「今日偶忘記帶篦子刀來。欲假以掠鬢，可乎？」晉卿從腰間取之。王云：「此樣甚新可愛。」晉卿曰：「近創造二副，一猶未用，少刻當以馳內。」至晚，遣俅賫往。值王在園中[[蹴鞠|蹴鞠]]，俅候報之際，睥睨不已。王呼來前詢曰：「汝亦解此技也？」俅曰：「能之。」漫令對蹴，遂愜王之意，大喜，呼隸輩云：「可往傳語都尉，既謝篦刀之況，並所送人皆輟留矣。」由是日見親信。逾月，王登寶位。上優寵之，眷渥甚厚，不次遷拜。其儕類援以祈恩，上云：「汝曹爭如彼好腳跡邪！」數年間建節，循至使相，遍曆三衙者二十年，領殿前司職事，自俅始也。父敦復，復為節度使。兄伸，自言業進士，直赴殿試，後登八坐。子侄皆為郎。潛延閣恩幸無比，極其富貴。然不忘蘇氏，每其子弟入都，則給養問恤甚勤。靖康初，祐陵南下，俅從駕至臨淮，以疾為解，辭歸京師。當時侍行如[[童貫|童貫]]、[[梁師成|梁師成]]輩皆坐誅，而俅獨死於牖下。胡元功云。

== 小說事蹟 ==

高俅是古典小說《[[水滸傳|水滸傳]]》的首席歹角，與蔡京、童貫、楊戩同為《水滸傳》四大奸臣。第二回的高俅發蹟史，就是根據《揮麈錄》改編而成。高俅以一市井小流氓的身份出場，因為具有很高[[蹴鞠|蹴鞠]]技術，被喜愛蹴毬的端王賞識。端王[[登基|登基]]成為[[宋徽宗|道君皇帝]]後，高俅便飛黃騰達，很快官至太尉。

高俅後來因為養子高衙內看上了[[林冲|林冲]]的妻子，便設計陷害林冲，使他被發配[[滄州|滄州]]。又遣[[陸謙|陸謙]]和富安去追殺林冲，兩人反被林冲所殺。林冲由此上了[[梁山|梁山]]。

[[梁山|梁山]]壯大之後，高俅想要剷除這口眼中釘，領兵討伐，反而被活捉上山。

[[宋江|宋江]]被招安，為朝廷平定各方叛逆後，高俅仍舊猜忌梁山的人，暗中用毒酒鴆殺[[盧俊義|盧俊義]]和宋江。而[[阮小七|阮小七]]亦被其黨羽奪去官位，貶為平民。

《[[蕩寇志|蕩寇志]]》中，高俅與[[蔡京|蔡京]]等人勾結梁山，後被[[雷橫|雷橫]]及[[朱仝|朱仝]]所殺。朱仝把其首級送給林冲，藉此醫治他，但林冲反被其首級氣死。
