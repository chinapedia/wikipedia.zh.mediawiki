'''高木紗友希'''（{{bd|1997年|4月21日}}），[[Hello!_Pro研修生|Hello! Pro研修生]]第八期成員。現為[[Hello!_Project|Hello! Project]]旗下女子偶像團體組合[[Juice=Juice|Juice=Juice]]的成員之一。

隸屬於[[UFG#UP-FRONT_AGENCY|UP-FRONT AGENCY]]旗下公司。2013年夏天正式出道。血型為A型，身高156公分。

2009年便加入[[Hello!_Project_Egg|Hello! Project Egg]]<ref>[2011年底才更名為Hello! Pro研修生。]</ref>，在[[Juice=Juice|Juice=Juice]]中的演藝資歷僅次於[[宮本佳林|宮本佳林]]。

== 簡歷 ==
=== 2009 ===
*11月23日，於「2009ハロー！プロジェクト新人公演11月～横浜FIRE！～」活動中，發表成為[[Hello!_Project_Egg|Hello! Project Egg]]第八期成員。

=== 2010 ===
*10月16日，上映與[[新垣里沙|新垣里沙]]、[[中島早貴|中島早貴]]及同為egg的佐藤綾乃主演的『毛骨悚然撞鬼經驗』3D劇場版<ref>[http://www.fjmovie.com/main/news/2010/news08_kowai3d.html ３Ｄで体感するホラー『ほんとうにあった怖い話　３Ｄ劇場版』10月公開 ]</ref>。

=== 2012 ===
*1月11日，開始播出與早安家族成員共演的電視劇『数学♥女子学園』。

=== 2013 ===
*2月3日，「Hello! Project 誕生15周年記念Live 2013冬」中發表組成團體，第一彈成員為[[宮本佳林|宮本佳林]]、[[高木紗友希|高木紗友希]]、[[大塚愛菜|大塚愛菜]]、[[植村あかり|植村あかり]]、[[金澤朋子|金澤朋子]]，以及SATOYAMA MOVEMENT組合[[GREEN_FIELDS|GREEN FIELDS]]成員[[宮崎由加|宮崎由加]]。組合有增加或刪減成員的可能。<ref>[http://www.helloproject.com/news/1302031700_hello.html ハロプロ研修生内、新ユニット結成のお知らせ。 ]</ref>
*2月25日，製作人[[淳君|淳君]]在部落格發表了新團體的名稱及團名含意。<ref>[http://ameblo.jp/tsunku-blog/entry-11478354838.html ハロプロ研修生新ユニットに関して ]</ref>
*4月3日，發行第一張獨立製作單曲「[[在我說之前_請抱緊我|在我說之前 請抱緊我]]」，於4月15日的日本公信榜單曲週榜取得第25名。是早安家族中首個以第一張獨立製作單曲就登上公信榜週榜的組合。
*4月24日，開設官方Ameba部落格。
*6月13日，於第三張獨立製作單曲「[[攀登到天空去!|攀登到天空去!]]」發行活動中，[[淳君|淳君]]宣佈[[Juice=Juice|Juice=Juice]]於夏天正式出道<ref>[http://www.youtube.com/watch?v=RyUBTp-7JuU Juice=Juice 密着映像 #10－「­天まで登れ！」発売記念イベント～Juice=Juice 聖地サンシャイン噴水広場1stイベント～』の様子をご覧ください。 ]</ref>。

== 特徵 ==
*一直以來憧憬的前輩是[[早安少女組|早安少女組]]第六期的[[田中麗奈|田中麗奈]]。
*曾說過對自己的長相沒自信，但也努力的展現，想讓大家看到自己閃閃發亮的地方<ref>[2013年6月13日，關於出道的感想文中提到。]</ref>。
*最喜歡的興趣是唱歌，也是團中的副主唱之一。
*喜歡的食物是酪梨；討厭的食物是魚板<ref>[2013年6月26日，Hello! Project Station第21回中的個人訪問。]</ref>。
*拿手的科目是國語；不拿手的科目則是數學<ref>[同上註。]</ref>。
