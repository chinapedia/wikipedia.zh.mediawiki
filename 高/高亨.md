'''高亨'''{{BD|1900年|7月29日|1986年|2月2日|catIdx=G高}}<ref>1900年7月29日[[阴历|阴历]]为七月初四，故有资料显示其生日为7月4日.</ref>，又名'''晋生'''，初名'''仙翘'''，训诂学、先秦学术及中国古典文学专家，[[吉林省|吉林省]][[双阳县|双阳县]]（今[[长春市|长春市]][[双阳区|双阳区]]）山嘴子人。

== 生平 ==
1910年入私塾读书。1918年春考入[[吉林省立第一师范学校|吉林省立第一师范学校]]，1922年冬毕业。1923年春入[[北京|北京]][[弘达学院|弘达学院]]。同年秋考入[[北京师范大学|北京师范大学]]。1924年秋考入[[北京大学|北京大学]]。1925年秋改名高亨，考入[[清华学校研究院|清华学校研究院]]，1926年夏毕业。初任[[吉林省_(中華民國)|吉林省]]立[[法政专门学校|法政专门学校]]教授兼第一师范学校教员。1929年任[[沈阳|沈阳]][[東北大學_(中國)|东北大学]]教育学院国文专修科教授。[[九一八事变|九一八事变]]后随东北大学来到[[北平市|北平市]]。之后历任[[河南大学|河南大学]]、东北大学、[[武汉大学|武汉大学]]、[[齐鲁大学|齐鲁大学]]、[[西北大学|西北大学]]和[[湘辉学院|湘辉学院]]教授等。1953年起任[[山东大学|山东大学]]教授，1957年起兼任[[中国科学院|中国科学院]]哲学研究所研究员。1967年调至[[北京市|北京市]]专门从事古代学术、古典文学研究工作。<ref name="国学网站">{{cite web |title=高亨 |publisher=国学网站 |url=http://www.guoxue.com/article/gaoheng/gh003.htm}}</ref><ref name="中国文学家辞典">{{Cite book |title=中国文学家辞典·现代第一分册 |publisher=四川人民出版社 |year=1979 |unified=17118·16 |pages=397-399}}</ref>

== 学术贡献 ==
高亨早年对[[训诂学|训诂学]]颇有研究，曾写就《金石甲骨文字通笺》十四册，不幸在抗战时期亡失。中期致力于[[先秦诸子|先秦诸子]]的研究，特别是《[[老子|老子]]》一书。后期主要研究《[[周易|周易]]》、《[[诗经|诗经]]》、《[[尚书|尚书]]》三部古经，其中对《周易》的研究成绩尤著。<ref name="国学网站"/><ref name="中国文学家辞典"/>
