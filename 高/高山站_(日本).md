'''高山站'''（{{jpn|j=高山駅|hg=たかやまえき|rm=Takayama eki|f=y}}）是位於[[日本|日本]][[岐阜縣|岐阜縣]][[高山市|高山市]]昭和町一丁目的[[東海旅客鐵道|東海旅客鐵道]]（JR東海）[[高山本線|高山本線]]鐵道車站。

== 概要 ==
日本有數的觀光地[[飛驒國|飛驒]]中有[[小京都|小京都]]稱呼的舊高山市地域代表車站。

現在有[[特別急行列車|特急]]「[[飛驒號列車)|Wideview飛驒]]」（{{lang|ja|ひだ}}）停車。2001年（平成13年）9月30日之前有[[名古屋鐵道|名古屋鐵道]]的柴油特急「北阿爾卑斯」（{{lang|ja|北アルプス}}）1日1往復班次。

== 車站構造 ==
此站為可供[[列車交會|列車交會]]的[[地上車站|地上車站]]，設有[[島式月台|島式月台]]1面2線與[[單式月台|單式月台]]1面1線。[[月台|月台]]的東側於站舍正面為1號線、原本以地下道通往鄰之月台2、3號線，2016年因新站房啟用，改以天橋連通各月台。1號線為上行本線、2號線為下行本線、3號線為副本線，此月台下行、上行方向的出發到着亦可能使用。

原本站房為1934年（昭和9年）開業時建設的混凝土2層階建築，2014年起拆除舊站站房，改建為2層樓[[跨站式站房|跨站式站房]]。新站房外牆為黑色，站內[[天花板|天花板]]與[[牆壁|牆壁]]使用的木材為飛驒當地的[[日本扁柏|日本扁柏]]。

此站為[[站長|站長]]、[[站員|站員]]配置站（[[直營站|直營站]]），同時負責[[飛驒小坂站|飛驒小坂]] - [[杉原站|杉原]]間各站的管理，設置有[[綠色窗口|綠色窗口]]及[[自動售票機|自動售票機]]，並設有賣店（[[Kiosk_(便利商店)|Kiosk]]）、有販賣車站便當。

== 車站周邊 ==
本站位於高山市街地西端位置。為觀光地的玄關站、徒歩1 - 2分圈内酒店、旅館有5間。市街地中心於東方向、往高山的象徴上三町古町徒歩約12分鐘。

== 歷史 ==
* 1934年（[[昭和|昭和]]9年）10月25日 - 高山本線的飛驒小坂 - 坂上間開通時開業。旅客、貨物營業開始。
* 1968年（昭和43年）9月 - 月台延長（上行月台向南60米延長）。
* 1968年（昭和43年）10月1日 - 綠色窗口開設。
* 1969年（昭和44年）10月 - 無煙化、站内的給炭設備解體。
<!--10月1日なのか？** 高山站開業以来の[[号鐘|号鐘]]を[[列車集中制御装置|CTC]]化のため取り外し、その後は[[站長室|站長室]]に保管する。-->
* 1980年（昭和55年）5月27日 - 車站站場内高山本線[[電氣化|電氣化]]開工。但是電氣化工程於其後中止。
* 1986年（昭和61年）11月1日 - 貨運業務廢止。
* 1987年（昭和62年）3月31日 - 貨運業務再開。
* 1987年（昭和62年）4月1日 - [[國鐵分割民營化|國鐵分割民營化]]，由JR東海和[[日本貨物鐵道|日本貨物鐵道]]（JR貨物）繼承。
* 1993年（[[平成|平成]]5年）3月 - 站場内[[扇形車庫|扇形車庫]]和[[轉車台|轉車台]]拆除解體。
* 1993年（平成5年）12月17日 - 站場内原本供蒸氣機車加水用的[[高架水槽|高架水槽]]拆除解體。
* 2007年（平成19年）4月1日 - JR貨物車站廢止、貨物營業終了。
* 2014年12月1日 - 新站房動工，舊站房開始拆除
* 2016年10月2日 - 新站房完工啟用
