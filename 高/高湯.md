'''高汤'''，[[粵語|粵語]]稱'''上湯'''，是用作烹調的[[清湯|清湯]]。一般用[[肉类|肉类]]，加入适量[[调味料|调味料]]（[[中国菜|中国菜]]加[[葱|葱]]、[[薑|薑]]、[[料酒|料酒]]；[[西餐|西餐]]加[[香叶|香叶]]、[[胡椒|胡椒]]、[[香芹|香芹]]、[[百里香|百里香]]等）、[[蔬菜|蔬菜]]等，在[[水|水]]中小[[火|火]]慢[[炖|炖]]几个小时，撇去[[泡沫|泡沫]]和表面[[油脂|油脂]]而成。[[粵菜|粵菜]]中會把上湯的剩肉加水再煮，得出來的叫'''二湯'''，也可作高湯之用。[[欧洲烹饪|欧洲烹饪]]有加入[[蛋白|鸡蛋蛋白]]澄化清汤的。

粵菜中的高湯，是被世界認為是三大名湯之一，另外兩個是[[泰國|泰國]]的[[冬陰湯|冬陰湯]]和[[法國|法國]]的[[馬賽魚湯|馬賽魚湯]]。<ref name="jiten2007_p650">{{harvnb|日仏料理協会|2007|p=650}}</ref>

== 粵菜 ==
* 頂湯
*: 以[[瘦肉|瘦肉]]和[[老雞|老雞]]熬製，適用於製作[[鮑魚|鮑魚]]、[[海參|海參]]、[[魚翅|魚翅]]、[[花膠|花膠]]等名貴[[海味|海味]]<ref name="粵菜烹飪">{{cite book |author=烹飪技術研究所 |title=《粵菜烹飪》 |pages=第90-91頁 |date=1996年6月 |publisher=鴻光書店 |location=香港 |isbn=9628123017 }}</ref>
* 上湯
*: 材料與頂湯同，但材料減半以相同份量清水熬製，用於較名貴的菜<ref name="粵菜烹飪"/>
* 二湯
*: 將熬過頂湯和上湯的材料加水番煮後加[[鹽|鹽]]，用於一般菜<ref name="粵菜烹飪"/>
* 芡湯
*: 上湯加鹽、[[糖|糖]]，用於[[勾芡|勾芡]]<ref name="粵菜烹飪"/>
* 紅湯
*: 上湯加入鴨汁、鹽、油、糖、[[老抽|老抽]]，用於[[鴨|鴨]]、[[豬手|豬手]]、[[陳皮鴨湯|陳皮鴨湯]]等。<ref name="粵菜烹飪"/>

== 其他中國菜中 ==
* 奶湯
*: 含悬浮[[脂肪|脂肪]]多而呈乳白色的高汤
* 吊湯
*: 是指对高汤按下列步骤进一步精製：将高湯加热到接近[[沸腾|沸腾]]，加入剁细的[[鸡肉|鸡肉]]，在不沸腾的情况下让其进一步吸附汤内的悬浮物和增加溶解的[[蛋白质|蛋白质]]等[[鲜味|鲜味]]成分，然後捞出鸡肉过滤（例如[[川菜|川菜]]名菜[[開水白菜|開水白菜]]）。吊汤可进行一次或重複多次。

== 日本料理中 ==
* [[出汁|出汁]]
*: [[日本料理|日本料理]]中常用的一种清汤，用特製的乾[[柴鱼|柴鱼]]（鲣鱼）、[[昆布|昆布]]（海带）、小魚乾为材料製成，成品为口感清淡细緻的[[海鲜|海鲜]]味。
