'''高畑勳'''（{{jpn|j=高畑勲|hg=たかはた いさお|rm=Takahata Isao}}，{{bd|1935年|10月29日|2018年|4月5日|catIdx=Takahata, Isao}}）<ref name="Ghibli">{{cite web|url=http://www.ghibli.jp/info/012850/ |script-title=ja:高畑勲監督　訃報 |work=[[吉卜力工作室|吉卜力工作室]] |language=ja |date=2018年4月6日 |accessdate=2018年4月6日}}</ref>，是出生於[[日本|日本]][[三重縣|三重縣]][[伊勢市|伊勢市]]的[[動畫電影|動畫電影]][[導演|導演]]、[[製作人|製作人]]。他是在日本動畫業黎明時期便投入動畫製作的人物，曾推出改編自名著小說的電視動畫影集《[[小天使|小天使]]》、《[[清秀佳人_(動畫)|清秀佳人]]》等作品。之後有與[[宮崎駿|宮崎駿]]、[[鈴木敏夫|鈴木敏夫]]等人創辦[[吉卜力工作室|吉卜力工作室]]，執導出《[[螢火蟲之墓#動畫版|螢火蟲之墓]]》、《[[平成狸合戰|平成狸合戰]]》、《[[輝耀姬物語|輝耀姬物語]]》等電影動畫。<ref>{{cite web |url=https://eiga.com/person/45172/ |script-title=ja:高畑勲 |work=Eiga |language=ja |accessdate=2018年5月26日}}</ref>

他在作品裡講求細膩的寫實風格，實際的日常生活描寫，對於日本動畫界有著影響。<ref>{{cite web |url=http://www3.nhk.or.jp/shutoken-news/20180406/0010192.html |archiveurl=https://web.archive.org/web/20180408010231/http://www3.nhk.or.jp/shutoken-news/20180406/0010192.html |deadurl=yes |script-title=ja:高畑さん「日常生活の哲学貫く」 |work=[[NHK|NHK]] |language=ja |date=2018年4月6日 |archivedate=2018年4月8日 |accessdate=2018年5月26日}}</ref>畫面表現與執導手法上也讓宮崎駿、[[富野由悠季|富野由悠季]]、[[片淵須直|片淵須直]]等人產生啟發。<ref>{{cite web |url=https://www.oricon.co.jp/special/51017/ |script-title=ja:富野由悠季が語る『ガンダム』のリアルを生んだ"高畑勲イズム" 「高畑さんは僕にとっても師匠」 |work=[[Oricon|Oricon]] |language=ja |date=2018年4月21日 |accessdate=2018年5月26日}}</ref><ref>{{cite web |url=https://animeanime.jp/article/2018/05/07/37663.html |script-title=ja:「この世界の片隅に」片渕須直監督から見た“高畑勲監督”とは... TOKYO FM「未来授業」で語る |work=Anime Anime! |language=ja |date=2018年5月7日 |accessdate=2018年5月26日}}</ref>

== 生平 ==
=== 早年 ===
於1935年10月29日出生於日本[[三重縣|三重縣]][[伊勢市|伊勢市]]，為7兄弟當中的老么。父親高畑淺次郎在[[二次大戰|二次大戰]]期間曾在[[岡山縣|岡山縣]]的{{tsl|ja|岡山県立岡山朝日高等学校|岡山一中}}擔任校長，戰後則就任過該縣的教育長。<ref name="Kyoto">{{cite web |url=http://www.kyoto-up.org/archives/1931 |script-title=ja:〈特集〉高畑勲とその時代 ～『かぐや姫』を迎え撃つために～(2013.11.16) |work=京都大學新聞 |language=ja |date=2013年11月16日 |accessdate=2018年5月26日}}</ref>

小時因父親要擔任校長一事緣故前往[[岡山縣|岡山縣]]居住，便有在1945年初夏遭遇到[[美軍|美軍]]轟炸該處一帶的{{tsl|ja|岡山大空襲|岡山大空襲 (日本)|岡山大空襲}}經歷。<ref name="Kyoto"/>

青年期間高畑勳前往[[東京|東京]]就讀[[東京大學|東京大學]]文學部[[法國文學|法文科]]，之後因在戲院觀賞到由法國動畫導演{{tsl|en|Paul Grimault|保羅·葛林莫}}創作的動畫《斜眼暴君》（{{lang|fr|La Bergère et le ramoneur}}）<ref group="註">該影片當時在日本上映的名稱為《{{lang|ja|やぶにらみの暴君}}》，且為往後《[[國王與鳥|國王與鳥]]》的雛形作品。</ref>時大受感動，便決定從事動畫界的工作。<ref>{{cite book |title=《樂在工作：進入宮崎駿．高畑勳的動畫世界》 |publisher=[[台灣東販|台灣東販]] |author=[[鈴木敏夫|鈴木敏夫]] |location=台灣 |isbn=978-986-176-961-5 |date=2009年7月15日 |chapter=〈高畑勳的理論與實踐〉|page=109頁}}</ref>

=== 東映動畫時期 ===
1959年從東京大學畢業後，高畑勳選擇加入[[東映動畫|東映動畫]]公司。起初有在《[[安壽與廚子王丸|安壽與廚子王丸]]》、《{{tsl|ja|わんぱく王子の大蛇退治|小王子與八頭龍|}}》等作品擔任導演助手，<ref name="EigaPage487">{{cite book |script-title=ja:映画を作りながら考えたこと 1955～1991 |publisher=[[德間書店|德間書店]] |author=高畑勳 |location=日本 |isbn=978-419-554-639-0 |date=1991年8月31日 |chapter=〈{{lang|ja|高畑勲監督作品リスト}}〉 |page=487頁}}</ref>之後在《{{tsl|ja|狼少年ケン|狼少年 (動畫)|狼少年}}》裡開始有接任執導方面事務的經歷。<ref name="EigaPage487"/>

此期間高畑勳結識了往後與他有長期合作關係的公司後輩[[宮崎駿|宮崎駿]]，當時高畑勳與宮崎駿分別各擔任了東映動畫公會的副委員長與書記長。<ref>{{cite book |title=《出發點 1979～1996》 |publisher=台灣東販 |author=[[宮崎駿|宮崎駿]] |location=台灣 |isbn=978-957-473-843-4 |date=2006年2月1日 |chapter=〈年譜〉 |page=540頁}}</ref>

=== 太陽王子霍爾斯 ===
高畑勳首部執導的長篇電影動畫，是與宮崎駿、[[大塚康生|大塚康生]]、[[原徹|原徹]]等人合作於1965年上映的《[[太陽王子_霍爾斯的大冒險|太陽王子 霍爾斯的大冒險]]》，為一部以當時國際社會因[[越戰|越戰]]局勢而興起的各種動蕩不安社會氣氛為題材之作品。<ref>{{cite book |title=《樂在工作：進入宮崎駿．高畑勳的動畫世界》 |publisher=台灣東販 |author=鈴木敏夫 |location=台灣 |isbn=978-986-176-961-5 |date=2009年7月15日 |chapter=〈與高畑勳、宮崎駿的相遇〉 |page=19頁}}</ref>

該影片雖然讓夥伴宮崎駿認為是部足以影響大衆感官的作品，<ref name="EigaPage493">{{cite book |script-title=ja:映画を作りながら考えたこと 1955～1991 |publisher=德間書店 |author=高畑勳 |location=日本 |isbn=978-419-554-639-0 |date=1991年8月31日 |chapter=〈{{lang|ja|大ナマケモノの子孫}}〉 |page=493頁}}</ref>但因票房並未特別亮眼，且高畑勳在執導此片時有嚴重進度落後的情況，而被東映動畫降格成執導助手，直到東映動畫製作《[[鬼太郎#ゲゲゲの鬼太郎#60年代（第一作）|怪怪怪的鬼太郎]]》第二期電視動畫時才將他升格在一些集數裡執導。<ref>{{cite book |script-title=ja:映画を作りながら考えたこと 1955～1991 |publisher=德間書店 |author=高畑勳 |location=日本 |isbn=978-419-554-639-0 |date=1991年8月31日 |chapter=〈{{lang|ja|高畑勲監督作品リスト}}〉 |page=488頁}}</ref>

=== 離任東映動畫 ===
《太陽王子 霍爾斯的大冒險》上映3年後，當時人在A Production動畫公司<ref group="註">現今的[[SHIN-EI動畫|SHIN-EI動畫]]公司</ref>的大塚康生，向高畑勳提出將瑞典小說家[[阿斯特麗德·林格倫|阿斯特麗德·林格倫]]之著作《[[長襪子皮皮|長襪子皮皮]]》改編為動畫的邀約，<ref name="Page441to442">{{cite book |title=《出發點 1979～1996》 |publisher=台灣東販 |author=宮崎駿 |location=台灣 |isbn=978-957-473-843-4 |date=2006年2月1日 |chapter=〈宮崎駿作品自述〉 |page=441至442頁}}</ref>之後高畑勳便帶著宮崎駿與[[小田部羊一|小田部羊一]]一同離開東映動畫。<ref>{{cite book |title=《樂在工作：進入宮崎駿．高畑勳的動畫世界》 |publisher=台灣東販 |author=鈴木敏夫 |location=台灣 |isbn=978-986-176-961-5 |date=2009年7月15日 |chapter=〈高畑勳的理論與實踐〉|page=110頁}}</ref><ref name="Page441to442"/>

之後因阿斯特麗德·林格倫不允許授權改編造成企劃取消，高畑勳便擔任A Production下《[[魯邦三世_(TV第1期)|魯邦三世]]》第一期電視動畫的導演。當時魯邦三世電視動畫前幾集的導演為{{tsl|ja|おおすみ正秋|大隅正秋}}，內容上偏向充斥著裸露及暴力動作，隨後高畑勳接任時便將作品修改為普級的風格，而當時高畑勳是以{{nihongo|「A Pro導演團隊」|Aプロ演出グループ}}的名義擔任導演。

在1972年3月期間結束《魯邦三世》的電視動畫後，年底高畑勳與宮崎駿合作推出了電影動畫《[[熊貓家族|熊貓家族]]》，並在隔年發表了該作品續作《{{tsl|ja|パンダコパンダ 雨ふりサーカスの巻|熊貓家族 雨中的馬戲團|}}》。在《熊貓家族》中，高畑勳與宮崎駿也改編了一些當時沒機會用在《長襪子皮皮》上的靈感。<ref name="Page441to442"/>

=== 日本動畫公司‧Telecom時期 ===
1973年高畑勳與宮崎駿、小田部羊一三人轉職到[[日本動畫公司|日本動畫公司]]旗下，一同參與以改編世界名著為電視動畫的《[[世界名作劇場|世界名作劇場]]》系列。當時高畑勳接手了將[[約翰娜·施皮里|約翰娜·施皮里]]小說《[[海蒂_(小說)|海蒂]]》所改編成的《[[小天使|小天使]]》。該作品公開後獲得了成功，全部52集數裡有平均近27%的收視率。<ref>{{cite web |url=https://www.excite.co.jp/News/entertainment_g/20101117/Real_Live_2536.html |script-title=ja:ヤマトを“撃沈”させた名作アニメ |work=Excite Japan Co., Ltd. |language=ja |date=2010年10月17日 |accessdate=2018年5月26日}}</ref>之後高畑勳相繼執導了《[[尋母三千里|尋母三千里]]》、《[[清秀佳人_(動畫)|清秀佳人]]》等同由小說所改編電視動畫作品。<ref>{{cite web |url=https://av.watch.impress.co.jp/docs/news/1115746.html |script-title=ja:高畑勲監督が死去。5月15日に「お別れの会」 |work=AV Watch |author=山崎健太郎 |language=ja |date=2018年4月6日 |accessdate=2018年5月26日}}</ref>

製作完《清秀佳人》後，高畑勳前往Telecom動畫公司將漫畫家{{tsl|ja|はるき悦|春木悅巳}}所創作帶有[[大阪|大阪]]人情物風格的作品《[[小麻煩千惠|小麻煩千惠]]》改編成電影動畫。《小麻煩千惠》的電影動畫在1981年上映後，高畑勳在當年繼續接手替《小麻煩千惠》製作電視版動畫影集。此期間高畑勳認識了當時要替動漫雜誌《[[Animage|Animage]]》編寫內容而找他訪談，往後成為他長期合作夥伴的[[鈴木敏夫|鈴木敏夫]]。高畑勳之後有在與鈴木敏夫多次會談過程中取得啟發，修改了《小麻煩千惠》電影動畫版的劇情走向。<ref>{{cite book |title=《樂在工作：進入宮崎駿．高畑勳的動畫世界》 |publisher=台灣東販 |author=鈴木敏夫 |location=台灣 |isbn=978-986-176-961-5 |date=2009年7月15日 |chapter=〈與高畑勳、宮崎駿的相遇〉 |page=26頁}}</ref>

1982年高畑勳在Oh! Production動畫公司推出了改編自[[宮澤賢治|宮澤賢治]]小說《[[大提琴手高修|大提琴手高修]]》的[[大提琴手高修_(1982年動畫)|同名電影動畫]]。同一年裡宮崎駿因創作理念上的問題，退出了Telecom原先與美國一方動畫師合作，將[[溫瑟·麥凱|溫瑟·麥凱]]的漫畫《{{tsl|en|Little Nemo|小尼莫 (漫畫)|小尼莫}}》改編為電影動畫的方案，到了隔年高畑勳也因同樣緣故選擇了退出。<ref>{{cite web |url=http://www.pelleas.net/hm/28.shtml |title=Nemo |work=Pelleas |language=en |accessdate=2018年5月26日}}</ref>

=== 風之谷 ===
1983年宮崎駿開始進行製作電影動畫《[[風之谷|風之谷]]》時，高畑勳在鈴木敏夫的遊說下，替宮崎駿擔任該作品的製作人。<ref name="ShigotoPage72">{{cite book |script-title=ja:《映画道楽》 |publisher= {{lang|ja|びあ}} |author=鈴木敏夫 |location=日本 |isbn=4-8356-1540-9 |date=2005年4月11日 |chapter=〈{{lang|ja|プロデューサー高畑勲から学んだこと。}}〉|page=72頁}}</ref>之後和宮崎駿向先前曾在《[[太陽王子_霍爾斯的大冒險|太陽王子 霍爾斯的大冒險]]》一同合作過的[[原徹|原徹]]請求協助，將對方所創立動畫室[[Topcraft|Topcraft]]借用為製作《風之谷》的場地。<ref>{{cite book |title=《順風而起》 |publisher=台灣東販 |author=鈴木敏夫 |location=台灣 |isbn=978-986-331-438-7 |date=2014年8月15日 |chapter=〈鈴木敏夫成為鈴木敏夫的瞬間〉 |page=105至106頁}}</ref>

《風之谷》上映後，高畑勳與宮崎駿等人討論打算將該影片賺取的資金用來拍攝以[[福岡縣|福岡縣]][[柳川市|柳川市]]一帶運河為背景的紀錄片《[[柳川堀割物語|柳川堀割物語]]》。不過因在拍攝時高畑勳便已將資金耗盡，鈴木敏夫便請宮崎駿拍攝動畫《[[天空之城|天空之城]]》來賺取費用。<ref>{{cite book |title=《順風而起》 |publisher=台灣東販 |author=鈴木敏夫 |location=台灣 |isbn=978-986-331-438-7 |date=2014年8月15日 |chapter=〈鈴木敏夫成為鈴木敏夫的瞬間〉 |page=115頁}}</ref>

=== 吉卜力工作室時期 ===
為了能有方便製作新動畫《天空之城》的地點，隨後鈴木敏夫與[[德間書店|德間書店]]討論，成立了專門讓高畑勳、宮崎駿使用的製作動畫場所「[[吉卜力工作室|吉卜力工作室]]」。<ref>{{cite book |title=《樂在工作：進入宮崎駿．高畑勳的動畫世界》 |publisher=台灣東販 |author=鈴木敏夫 |location=台灣 |isbn=978-986-176-961-5 |date=2009年7月15日 |chapter=〈『風之谷的娜烏西卡』及創立STUDIO GHIBLI〉|page=52頁}}</ref>其中高畑勳在吉卜力成立時雖然給予許多建議和協助，但他個人始終不把自己列為此工作室的正式成員，而是以類似[[顧問|顧問]]的身份進出吉卜力。<ref>{{cite book |title=《樂在工作：進入宮崎駿．高畑勳的動畫世界》 |publisher=台灣東販 |author=鈴木敏夫 |location=台灣 |isbn=978-986-176-961-5 |date=2009年7月15日 |chapter=〈高畑勳的理論與實踐〉|page=95頁}}</ref>

在吉卜力期間裡頭，高畑勳首部參與作品是擔任《天空之城》的製作人，而他首部執導的吉卜力動畫，是於1988年所推出改編自[[野坂昭如|野坂昭如]]同名小說的《[[螢火蟲之墓#動畫版|螢火蟲之墓]]》。<ref>{{cite book |title=《順風而起》 |publisher=台灣東販 |author=鈴木敏夫 |location=台灣 |isbn=978-986-331-438-7 |date=2014年8月15日 |chapter=〈鈴木敏夫成為鈴木敏夫的瞬間〉 |page=125頁}}</ref>

高畑勳於1991年推出的《[[兒時的點點滴滴|兒時的點點滴滴]]》在票房上獲得了31億日圓<ref group="註">扣除利潤分配後成績約18億日圓。</ref>左右成績，是當年度日本國產電影的首位。<ref>{{cite web|url=http://www.eiren.org/toukei/1991.html |script-title=日本国内 1991年 年間総合興行収入ランキング |work={{lang|ja|映画ランキングドットコム}} |language=ja |accessdate=2018年5月26日}}</ref>之後1994年的《[[平成狸合戰|平成狸合戰]]》則達到44億日圓<ref group="註">扣除利潤分配後成績約26億日圓。</ref>，除同為該年度日本國產電影的第一名，也擊敗當年假想敵動畫作品《[[獅子王|獅子王]]》的在日票房34億。<ref>{{cite web|url=http://www.eiga-ranking.com/boxoffice/japan/yearly/total/1994.html |script-title=日本国内 1994年 年間総合興行収入ランキング |work={{lang|ja|映画ランキングドットコム}} |language=ja |accessdate=2018年5月26日}}</ref><ref>{{cite book |title=《順風而起》 |publisher=台灣東販 |author=鈴木敏夫 |location=台灣 |isbn=978-986-331-438-7 |date=2014年8月15日 |chapter=〈電影製作、票房成功的法則〉 |page=165頁}}</ref>但到了1999年的《[[隔壁的山田君|隔壁的山田君]]》僅有15多億日圓<ref group="註">扣除利潤分配後成績約7億日圓。</ref>，是吉卜力自1989年《[[魔女宅急便|魔女宅急便]]》後票房最差的作品。<ref>{{cite book |title=《樂在工作：進入宮崎駿．高畑勳的動畫世界》 |publisher=台灣東販 |author=鈴木敏夫 |location=台灣 |isbn=978-986-176-961-5 |date=2009年7月15日 |chapter=〈德間康快的生存之道〉 |page=124頁}}</ref>

自《隔壁的山田君》後一段時期高畑勳未再有執導其它作品，此期間高畑勳另翻譯了一些國外動畫名作如《[[嘰哩咕與女巫|嘰哩咕與女巫]]》、《[[佳麗村三姊妹|佳麗村三姊妹]]》等等製成日文版交由吉卜力發行。<ref>{{cite web|url=https://natalie.mu/eiga/news/283329 |script-title=ja:「キリクと魔女」「王と鳥」など3本収録、フランスアニメのBlu-rayBOX発売 |work=[[Natalie_(網站)|Natalie]] |language=ja |date=2018年5月24日 |accessdate=2018年11月7日}}</ref>

2005年高畑勳開始有想將日本古老傳說《[[竹取物語|竹取物語]]》拍攝爲動畫的念頭，之後2008年高畑勳與[[西村義明|西村義明]]合作開始計劃推出此故事的改編動畫《[[輝耀姬物語|輝耀姬物語]]》。<ref>{{cite book |script-title=ja:《かぐや姫の物語 ビジュアルガイド》 |publisher=[[角川書店|角川書店]] |location=日本 |isbn=978-404-110-568-9 |date=2013年11月22日 |chapter=〈{{lang|ja|『かぐや姫の物語』ができるまで}}〉|page=70頁}}</ref>該作品以超出50億日圓的製作費規格投入，在歷經約8年時間後於2013年年底上映。<ref>{{cite web |url=https://cinema.ne.jp/recommend/kaguyahime2018051817/ |script-title=ja:『かぐや姫の物語』、もっと深く洞察できる3つのポイント |work=Cinema |language=ja |date=2018年5月18日 |accessdate=2018年5月26日}}</ref>

結束了《輝耀姬物語》後，當時吉卜力另與荷蘭動畫導演[[麥可·度德威特|麥可·度德威特]]以及其它數間法國動畫室合力創作《[[紅烏龜：小島物語|紅烏龜：小島物語]]》動畫，高畑勳有出面為麥可·度德威特擔任片中美術方面的監製。<ref>{{cite web|url=http://natalie.mu/eiga/news/168850 |script-title=ja:スタジオジブリが贈る長編「レッドタートル」、高畑勲が海外の作家とタッグ |work=Natalie |language=ja |date=2015年12月10日 |accessdate=2018年5月26日}}</ref>

== 個人 ==
=== 性格 ===
高畑勳的個人暱稱為{{nihongo|「阿朴」|パクさん}}，<ref>{{cite book |title=《出發點 1979～1996》 |publisher=台灣東販 |author=宮崎駿 |location=台灣 |isbn=978-957-473-843-4 |date=2006年2月1日 |chapter=〈宮崎駿作品自述〉 |page=437頁}}</ref>源自於他年輕時在動畫室上班快遲到進來後，忙著大口咀嚼（{{lang|ja|パクパク}}）麵包的模樣。<ref>{{cite book |script-title=ja:風の谷のナウシカ Guide Book |publisher=德間書店 |location=日本 |isbn=978-419-720-309-3 |date=2010年7月15日 |chapter=〈{{lang|ja|宮崎駿自作を語る}}〉 |page=121頁}}</ref>

個人的興趣是讀書和音樂鑑賞。<ref name="EigaPage493"/>喜好的作家有[[宮澤賢治|宮澤賢治]]、<ref>{{cite book |script-title=ja:映画を作りながら考えたことII 1991～1999 |publisher=德間書店 |author=高畑勳 |location=日本 |isbn=978-419-861-047-0 |date=1999年7月31日 |chapter=〈{{lang|ja|いつか映像化したい賢治}}〉 |page=212頁}}</ref>[[法國|法國]]詩人[[賈克·普維|賈克·普維]]，並在2004～2006年期間親自翻譯賈克·普維的詩集《{{lang|fr|Paroles}}》等其它作品的日文譯本。<ref>{{cite web |url=https://netatopi.jp/article/1121776.html |script-title=ja:今日15日(火)、ジブリ美術館で「高畑勲 お別れの会」　一般およびファンによる献花の時間も～委員長は宮崎駿氏。コミュニティバスも臨時運行 |work=Natatopi |language=ja |date=2018年5月14日 |accessdate=2018年5月26日}}</ref>在對於[[樂理|樂理]]擁有一定程度的理解，於[[久石讓|久石讓]]在為宮崎駿製作《[[風之谷|風之谷]]》、《[[魔女宅急便|魔女宅急便]]》等作品配樂時，便與他數次討論關於處理配樂的事情。<ref>{{cite book |script-title=ja:魔女の宅急便 ロマンアルバム |publisher=德間書店 |location=日本 |isbn=978-419-720-158-7 |date=2008年9月25日 |chapter=〈{{lang|ja|音楽 久石譲}}〉 |page=129頁}}</ref>之後在2013年執導的《[[輝耀姬物語|輝耀姬物語]]》中也親自創作數首插入歌。<ref>{{cite book |script-title=ja:《かぐや姫の物語 ビジュアルガイド》 |publisher=角川書店 |location=日本 |isbn=978-404-110-568-9 |date=2013年11月22日 |chapter=〈{{lang|ja|[音楽]久石 譲}}〉|page=84頁}}</ref>欣賞的畫家則有[[岩崎知弘|岩崎知弘]]，他對於岩崎知弘所創作兒童繪本{{nihongo|《下雨天一個人在家》|あめのひのおるすばん}}書中氣氛的拿捏有著深刻印象。<ref>{{cite book |script-title=ja:アニメーション、折にふれて |publisher=岩波書店 |location=日本 |isbn=978-400-022-080-4 |date=2013年12月6日 |page=82頁}}</ref>

在面對作品完工進度的態度上則較為消極，常有製作進度落後的情況出現，<ref>{{cite book |title=《樂在工作：進入宮崎駿．高畑勳的動畫世界》 |publisher=台灣東販 |author=鈴木敏夫 |location=台灣 |isbn=978-986-176-961-5 |date=2009年7月15日 |chapter=〈『風之谷的娜烏西卡』及創立STUDIO GHIBLI〉|page=44頁}}</ref>因而曾被宮崎駿在《兒時的點點滴滴》的記者會資料上戲稱是{{nihongo|「[[樹懶|樹懶]]的子孫」|大ナマケモノの子孫}}。<ref name="EigaPage493"/>嚴重的一次是在執導[[新潮社|新潮社]]的《[[螢火蟲之墓#動畫版|螢火蟲之墓]]》時，讓對方首次推出的動畫是在部份畫面未完工的情況下上映。<ref>{{cite book |title=《樂在工作：進入宮崎駿．高畑勳的動畫世界》 |publisher=台灣東販 |author=鈴木敏夫 |location=台灣 |isbn=978-986-176-961-5 |date=2009年7月15日 |chapter=〈高畑勳的理論與實踐〉|page=102頁}}</ref>當自己是否處在導演的立場時會有不同思維，與他長期共事的鈴木敏夫表示過「若高畑勳在擔任影片的製作人時可以把預算控制得很好，但如果是擔任影片的導演就會把預算給花到精光」，<ref>{{cite book |script-title=ja:文藝春秋88巻11号 |date=2010年9月1日 |publisher=[[文藝春秋|文藝春秋]] |page=272頁 |accessdate=2014年6月8日}}</ref>以及「姑且不管高畑勳在影片製作人方面作得多出色，一當他成為導演時就常把與他合作的影片製作人給弄到受不了」。<ref>{{cite book |title=《樂在工作：進入宮崎駿．高畑勳的動畫世界》 |publisher=台灣東販 |author=鈴木敏夫 |location=台灣 |isbn=978-986-176-961-5 |date=2009年7月15日 |chapter=〈高畑勳的理論與實踐〉|page=96頁}}</ref>

統整事物上有著擅長整理資料、製作筆記的能力。過去在答應接下鈴木敏夫請求擔任《[[風之谷|風之谷]]》影片製作人一事之前，曾秀出一整本記錄有關製作人工作要點的筆記向鈴木敏夫說明「我不適合當製作人」的原因。<ref name="ShigotoPage72"/>之後在執導《[[兒時的點點滴滴|兒時的點點滴滴]]》時，因片中有出現採集紅花場景的緣故，便整理出一套能詳細說明該如何採集紅花、製作紅花花餅的手冊。<ref name="ShigotoPage100">{{cite book |title=《樂在工作：進入宮崎駿．高畑勳的動畫世界》 |publisher=台灣東販 |author=鈴木敏夫 |location=台灣 |isbn=978-986-176-961-5 |date=2009年7月15日 |chapter=〈高畑勳的理論與實踐〉|page=100頁}}</ref>

平時善於構思各種想法來說理爭論。<ref>{{cite book |title=《樂在工作：進入宮崎駿．高畑勳的動畫世界》 |publisher=台灣東販 |author=鈴木敏夫 |location=台灣 |isbn=978-986-176-961-5 |date=2009年7月15日 |chapter=〈高畑勳的理論與實踐〉|page=105頁}}</ref>曾有花費了三個小時的時間，來向鈴木敏夫說明為何《[[小天使|小天使]]》與《[[小麻煩千惠|小麻煩千惠]]》兩者對他而言內涵上是相同動畫的緣故。<ref>{{cite book |title=《順風而起》 |publisher=台灣東販 |author=鈴木敏夫 |location=台灣 |isbn=978-986-331-438-7 |date=2014年8月15日 |chapter=〈鈴木敏夫成為鈴木敏夫的瞬間〉 |page=94頁}}</ref>也在拍攝《[[螢火蟲之墓|螢火蟲之墓]]》期間因進度嚴重落後時，引用了法國動畫導演保羅·葛林莫在製作《斜眼暴君》中所遭遇到的狀況，來解釋為何應該讓《螢火蟲之墓》延期上映。<ref>{{cite book |title=《順風而起》 |publisher=台灣東販 |author=鈴木敏夫 |location=台灣 |isbn=978-986-331-438-7 |date=2014年8月15日 |chapter=〈鈴木敏夫成為鈴木敏夫的瞬間〉 |page=134頁}}</ref>

政治傾向方面是支持[[左派|左派]]的[[日本共產黨|日本共產黨]]，有與作曲家{{tsl|ja|池辺晋一郎|池邊晉一郎}}及漫畫家{{tsl|ja|高口里純|高口里純}}等人公開表態過。<ref>{{cite web |url=http://www.jcp.or.jp/akahata/aik12/2012-12-04/2012120401_04_1.html |script-title=ja:日本共産党に期待します 各界著名４９氏が表明 |publisher=[[日本共產黨|日本共產黨]] |language=ja |date=2012年12月4日 |accessdate=2014年6月8日}}</ref>在2013年日本首相[[安倍晉三|安倍晉三]]計劃針對[[日本國憲法第九條|憲法第九條]]的內容做出修改時，有和宮崎駿等人在當年7月吉卜力的刊物《熱風》上撰文批評。<ref>{{cite web|url=http://www.itmedia.co.jp/news/articles/1307/19/news116.html |script-title=ja:宮崎駿や高畑勲が9条改正を語った「熱風」7月号 |work=IT Media |language=ja |date=2013年7月19日 |accessdate=2018年5月26日}}</ref>

=== 行事 ===
作品的風格上講究[[寫實主義|符合現實的描寫]]，如《[[螢火蟲之墓|螢火蟲之墓]]》裡[[B-29超級堡壘轟炸機|B-29轟炸機]]在天空中飛行的方向，或是《[[小天使|小天使]]》裡出現的麵包等等，都有下過考察功夫。<ref name="ShigotoPage101">{{cite book |title=《樂在工作：進入宮崎駿．高畑勳的動畫世界》 |publisher=台灣東販 |author=鈴木敏夫 |location=台灣 |isbn=978-986-176-961-5 |date=2009年7月15日 |chapter=〈高畑勳的理論與實踐〉|page=101頁}}</ref><ref name="ShigotoPage100"/>其中高畑勳當時為了讓《小天使》的場面表現自然，而實際走訪了動畫原作小說的舞台[[瑞士|瑞士]]，這種前往實地考察的方式是當時日本動畫業界裡少見的。<ref>{{cite web|url=https://www.sankei.com/entertainments/news/180410/ent1804100005-n1.html |script-title=ja:高畑勲さん　日本アニメの方向性大きく変えた功労者 |work=產經新聞 |author=冰川龍介 |language=ja |date=2018年4月10日 |accessdate=2018年5月26日}}</ref>

不同於夥伴宮崎駿會把內心情感投入到自己所創作角色的情況，<ref>{{cite book |title=《出發點 1979～1996》 |publisher=台灣東販 |author=宮崎駿 |location=台灣 |isbn=978-957-473-843-4 |date=2006年2月1日 |chapter=〈愛的火花〉 |page=555頁}}</ref>表現上不會讓觀眾在欣賞影片時把自己想像成是劇中角色，而是讓觀眾像第三者般去試著了解角色。<ref>{{cite book |title=《我在吉卜力思考的事 內容力─想像力─感受力─創造力》 |publisher=城邦讀書花園 |author=[[川上量生|川上量生]] |location=台灣 |isbn=978-986-459-121-3 |date=2018年3月1日 |chapter=〈如何與觀眾產生共鳴？〉|page=156頁}}</ref>劇情手法上較偏向客觀，不會過於理想化的方式，並避免讓觀眾產生[[悲觀主義|悲觀]]。<ref>{{cite web|url=http://www.style.fm/as/05_column/365/365_030.shtml |script-title=ja:第30回　『赤毛のアン』 |work=Web Animation Magazine |author=小黑祐一郎 |language=ja |accessdate=2018年5月26日}}</ref>

製作動畫上不像宮崎駿擁有能夠繪製出精美[[分鏡圖|分鏡圖]]的技術，個人僅會繪製簡單的構圖，因此是透過下指示讓其他動畫師完成畫面為主。<ref name="RealSound">{{cite web|url=https://realsound.jp/tech/2018/04/post-179739.html |script-title=ja:高畑勲監督は“アニメの作り方”を抜本的に変革したーー遺作『かぐや姫の物語』の功績 |work=Real Sound |author=松田廣宣 |language=ja |date=2018年4月6日 |accessdate=2018年11月7日}}</ref><ref name="YK">{{cite web|url=http://www.yk.rim.or.jp/~rst/rabo/takahata/gijyutsukou.html |script-title=ja:高畑勲演出の技術考 |work=Institute Of Isao Takahata & Hayao Miyazaki |author=葉精二 |language=ja |date=2018年4月6日 |accessdate=2018年11月7日}}</ref>向動畫師給予指示時，會設法交代出具體讓人易理解的條件，而非解讀上會有模擬兩可情況的要求內容。<ref name="ShigotoPage101"/>

畫面上會嘗試採用各種不同手法來挑戰，如《[[兒時的點點滴滴|兒時的點點滴滴]]》裡角色在喜怒哀樂時的面部輪廓表現。<ref name="YK"/>在《[[隔壁的山田君|隔壁的山田君]]》跟《[[輝耀姬物語|輝耀姬物語]]》中，則採用類似[[寫生|寫生]]、淡描的風格來繪製，企圖單純透過線條在畫面上的表現，來呈現出角色栩栩如生的模樣。<ref>{{cite web|url=http://animestyle.jp/2014/02/05/6834/ |script-title=ja:西村義明プロデューサー　インタビュー 第１回　線の向こうにある本物を |work=Web Animation Magazine |language=ja |date=2014年2月5日 |accessdate=2018年5月26日}}</ref>其中《輝耀姬物語》有捨棄一般動畫常採用的先把角色和背景分開繪製，然後再合併畫面的方法；而是改讓角色和背景在同一張圖上作畫，以達到精細的表現效果。<ref name="RealSound"/>

== 關係人物 ==
在[[東映動畫|東映動畫]]公司工作期間相遇的[[宮崎駿|宮崎駿]]，是高畑勳提拔、一路合作的後輩。他們在年輕時期到成立吉卜力工作室階段裡便多次互相搭檔，創作出多部的電視、電影動畫作品。宮崎駿雖曾與高畑勳發生數次爭執，<ref>{{cite book |title=《樂在工作：進入宮崎駿．高畑勳的動畫世界》 |publisher=台灣東販 |author=鈴木敏夫 |location=台灣 |isbn=978-986-176-961-5 |date=2009年7月15日 |chapter=〈『風之谷的娜烏西卡』及創立STUDIO GHIBLI〉|page=51頁}}</ref>但長期都是向對方抱持著亦兄亦友的敬重態度。<ref name="ShigotoPage111">{{cite book |title=《樂在工作：進入宮崎駿．高畑勳的動畫世界》 |publisher=台灣東販 |author=鈴木敏夫 |location=台灣 |isbn=978-986-176-961-5 |date=2009年7月15日 |chapter=〈高畑勳的理論與實踐〉|page=111頁}}</ref>高畑勳肯定著宮崎駿本身的才能，並認為就是因為宮崎駿的存在，才能夠鞭策他不使自己中途放棄。<ref>{{cite book |title=《出發點 1979～1996》 |publisher=台灣東販 |author=宮崎駿 |location=台灣 |isbn=978-957-473-843-4 |date=2006年2月1日 |chapter=〈愛的火花〉 |page=550頁}}</ref>而宮崎駿則提及過他不時會做出跟高畑勳相關事情的夢，<ref name="ShigotoPage111"/>也是他最希望能去觀賞他動畫作品的人選。<ref name="ShigotoPage111"/>

其他在日本動畫界結識的友人中，[[大塚康生|大塚康生]]是高畑勳投入動畫界的前輩，在《[[太陽王子_霍爾斯的大冒險|太陽王子 霍爾斯的大冒險]]》與《[[魯邦三世_(TV第1期)|魯邦三世]]》等高畑勳代表作中擔任作畫指導，並私下曾一同與他和宮崎駿等人聚會，來互相鼓舞未來要在動畫界裡達成的目標。<ref>{{cite book |title=《出發點 1979～1996》 |publisher=台灣東販 |author=宮崎駿 |location=台灣 |isbn=978-957-473-843-4 |date=2006年2月1日 |chapter=〈「中傷」繪畫〉 |page=200頁}}</ref>[[小田部羊一|小田部羊一]]是與高畑勳長年共事的人物之一，多次一同在各間不同動畫公司裡同進退，他也是高畑勳在執導[[世界名作劇場|世界名作劇場]]系列作品期間擔任作畫指導及角色設計的核心成員。<ref>{{cite book |title=《出發點 1979～1996》 |publisher=台灣東販 |author=宮崎駿 |location=台灣 |isbn=978-957-473-843-4 |date=2006年2月1日 |chapter=〈年譜〉 |page=543頁}}</ref><ref>{{cite book |script-title=ja:《アニメーション、折にふれて》 |publisher=岩波書店 |author=高畑勳 |location=日本 |isbn=978-400-022-080-4 |date=2013年12月6日 |page=108頁}}</ref>[[近藤喜文|近藤喜文]]是從《[[魯邦三世_(TV第1期)|魯邦三世]]》到進入[[吉卜力工作室|吉卜力工作室]]期間參與多部高畑勳作品的動畫師，其才華也是讓高畑勳和宮崎駿常爭取的人物。<ref>{{cite book |title=《順風而起》 |publisher=台灣東販 |author=鈴木敏夫 |location=台灣 |isbn=978-986-331-438-7 |date=2014年8月15日 |chapter=〈電影製作、票房成功的法則〉 |page=166頁}}</ref>

原是動畫雜誌《[[Animage|Animage]]》記者的[[鈴木敏夫|鈴木敏夫]]，因有專題採訪需求而結識了高畑勳，之後也成為吉卜力工作室的核心人物及製作人。<ref>{{cite book |title=《樂在工作：進入宮崎駿．高畑勳的動畫世界》 |publisher=台灣東販 |author=鈴木敏夫 |location=台灣 |isbn=978-986-176-961-5 |date=2009年7月15日 |chapter=〈與高畑勳、宮崎駿的相遇〉 |page=17頁}}</ref>鈴木敏夫在與高畑勳的共事過程中，學到了許多如何成為一名製作人的要領。<ref name="ShigotoPage72"/>而高畑勳認為他與宮崎駿之間的友情最後不會走向撕破臉局面，就是因為有鈴木敏夫在他們兩者之間協調的緣故。<ref>{{cite book |title=《出發點 1979～1996》 |publisher=台灣東販 |author=宮崎駿 |location=台灣 |isbn=978-957-473-843-4 |date=2006年2月1日 |chapter=〈愛的火花〉 |page=558至559頁}}</ref>

海外結識的動畫人物方面，高畑勳是[[俄羅斯|俄羅斯]]動畫導演[[尤里·諾里斯金|尤里·諾里斯金]]的影迷，其中諾里斯金的《{{tsl|en|Hedgehog in the Fog|霧中的刺蝟}}》是高畑勳特別推崇的，除了有著交流往來關係外，也特別有為對方的動畫作品《故事中的故事》（{{lang|ru|Сказка сказок}}）編寫一本講解書籍{{nihongo|《故事中的故事─映像詩的世界》|話の話―映像詩の世界}}。<ref>{{cite web |url=https://natalie.mu/eiga/news/208414 |script-title=ja:高畑勲、ユーリー・ノルシュテインとの対談で作品の魅力熱弁 |work=Natalie |language=ja |date=2016年11月7日 |accessdate=2018年5月26日}}</ref>執導《[[種樹的牧羊人|種樹的牧羊人]]》的加拿大動畫師{{tsl|en|Frédéric Back|費德烈·貝克}}是另位影響高畑勳的人物，高畑勳在2011年有讓吉卜力工作室為對方舉行在日本的美術展覽外，<ref>{{cite web |url=https://eiga.com/news/20110701/20/ |script-title=ja:“アニメ界の至宝”フレデリック・バック緊急来日、高畑勲の決断に感謝 |work=Eiga |language=ja |date=2011年7月1日 |accessdate=2018年5月26日}}</ref>也在2013年完成《輝耀姬物語》後，有幸於費德烈·貝克離世前數天前往到加拿大親自將此作品讓對方觀賞。<ref>{{cite web |url=http://www.jaa.gr.jp/modules/news/photo.php?lid=259 |script-title=ja:フレデリック・バックさんご逝去 |work=Japan Animation Association |language=ja |date=2013年12月25日 |accessdate=2018年5月26日}}</ref>
