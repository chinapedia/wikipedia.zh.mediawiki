《'''高捷少女角色形象歌曲 Station.1'''》是[[臺灣|臺灣]][[高雄捷運|高雄捷運]]虛擬[[吉祥物|代言人]][[高捷少女|高捷少女]]的第一張[[單曲|單曲]]專輯。於2016年1月30日在[[開拓動漫祭|第27屆開拓動漫祭]]首次發售，亦於[[買動漫|買動漫]]上架<ref>{{Cite web|url=https://www.myacg.com.tw/goods_detail.php?gid=320445|script-title=高捷少女角色形象歌曲Station.1 販售頁面|date=2016-01-27|website=買動漫|accessdate=2017-10-21|language=zh-tw}}</ref>，已於2017年2月下架。

== 概要 ==
高捷少女系列首張專輯《高捷少女角色形象歌曲 Station.1》中，收錄2015年8月6日發布的小穹形象歌曲「'''下一站．與你''' 」
<ref>{{Cite web |url =  http://gnn.gamer.com.tw/6/118896.html|title =高雄捷運推出《高捷少女》痛捷運列車 並釋出第 4 位官方角色與官方形象歌曲試聽|author = [[巴哈姆特|巴哈姆特]] |date = 2015年8月5日 |language = zh-tw|quote = }}</ref><ref>[https://www.facebook.com/K.R.T.GIRLS/videos/1681316928765612/ 官方粉絲團發布-【完整版】小穹形象歌曲「下一站．與你」(已失效)]</ref>，以及艾米莉亞的形象歌曲「未完成START」、也是《[[前進吧！高捷少女_Initiating_Station|前進吧！高捷少女 Initiating Station]]》遊戲主題曲<ref>[https://www.facebook.com/K.R.T.GIRLS/videos/1740554719508499/ 官方粉絲團發布-【完整版】艾米莉亞形象歌曲「未完成START」](已失效)</ref>，兩首歌曲一同收錄在其中。

==後續發布==
為刺激專輯銷量，負責錄製此專輯的艾耳音樂於3月釋出消息，表示將在專輯訂購至一定數量之後將有可能籌備進行其餘兩位角色的歌曲<ref>{{Cite web|url=https://www.facebook.com/AyreMusicOfficial/photos/1115703411794668/|script-title=關於婕兒與NANA形象歌曲...|date=2016-03-12|website=艾耳音樂-官方粉絲團|accessdate=2017-10-21|language=zh-tw}}</ref>，且最快將於秋季開始錄製<ref>{{Cite web|url=https://www.facebook.com/AyreMusicOfficial/posts/1148548048510204|script-title=關於大家所期待的婕兒與NANA的歌曲製作...|date=2016-04-14|website=艾耳音樂-官方粉絲團|accessdate=2017-10-21|language=zh-tw}}</ref>。並於隔年2017年2月公布婕兒的角色歌<ref>{{Cite web |url =  http://gotv.ctitv.com.tw/2017/02/373714.htm|title =高捷少女將推新「動漫」　台、日火熱上映|author = [[中天新聞|中天新聞]] |date = 2017年2月14日 |language = zh-tw|quote = }}</ref>。

截至2017年7月，由於雙方合作契約已經到期等相關因素，仍暫時並沒有對外公布後續歌曲的消息(官方指出目前雙方權利金談不攏)。

合約到期後，雖然歌曲在[[KKBOX|KKBOX]]與[[Spotify|Spotify]]可以找得到，但是專輯封面已移除高捷少女的部分，也改為Station.1。

2018年1月，粉絲於官方粉絲俱樂部詢問結果，艾耳音樂將與希萌創意繼續談權利金部份，也已經準備好最終合約，將與希萌創意繼續討論<ref>{{Cite web|url=https://www.facebook.com/photo.php?fbid=1549849171762545&set=gm.903431563165580&type=3&theater|script-title=艾耳音樂─專輯求償自救會|date=2018-01-9|website=Facebook|accessdate=2018-01-09|language=zh-tw}}</ref>。

2018年1月22日，艾耳音樂表示一切順利，將在一兩個星期內公佈電子專輯與PV，並開放免費下載3週。<ref>{{Cite web|url=https://www.facebook.com/events/869899956518741/permalink/910543522454384/|script-title=艾耳音樂─專輯求償自救會|date=2016-03-12|website=Facebook|accessdate=2018-01-22|language=zh-tw}}</ref>。

2018年1月27日晚上7點，艾耳音樂正式公佈Station.2，收錄婕兒形象歌曲《Recharge!》和耐耐形象歌曲《約定，別說再見》，並且開放專輯免費下載至同年4月30日<ref>{{Cite web|url=https://www.facebook.com/AyreMusicOfficial/videos/1771592456205757/?hc_ref=ARTIBNOHh7zPL8M8EDBzSAs2VIVCwEtR5K5ydYpCmAZpwH_n8QqencbToPp3fwrU1g8&fref=gs&dti=1212038035533315&hc_location=group|script-title=《Station. 2》電子專輯期間限定免費公益下載|date=2018-01-27|website=Facebook 艾耳音樂-官方粉絲團|accessdate=2018-02-01|language=zh-tw}}</ref><ref>{{Cite web|url=https://gnn.gamer.com.tw/9/158719.html|script-title=艾耳音樂推出新 EP《Station. 2》 歌曲 PV 結合《高捷少女》角色|date=2018-02-02|website=GNN新聞網-巴哈姆特|accessdate=2018-02-03|language=zh-tw}}</ref>。

2018年4月30日，關閉下載。
