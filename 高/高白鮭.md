'''高白鮭'''（[[学名|学名]]：{{lang|la|''Coregonus peled''}}），為[[輻鰭魚綱|輻鰭魚綱]][[鮭形目|鮭形目]][[鮭科|鮭科]]的一[[種|種]]，是一种高耗氧冷水性鱼类，生活在高纬度的冷水河流和湖泊中，原产於北纬50°以北的[[俄罗斯|俄罗斯]]境内[[梅津河|梅津河]]至[[科雷马河|科雷马河]]一带的湖泊<ref name="growth"/>，尤其多见於俄罗斯[[西伯利亚|西伯利亚]][[鄂毕河|鄂毕河]]流域<ref name="GBGDRGFH"/>，分布於[[保加利亚|保加利亚]]、[[丹麦|丹麦]]、[[芬兰|芬兰]]、[[德国|德国]]、[[立陶宛|立陶宛]]、[[波兰|波兰]]、[[俄罗斯|俄罗斯]]、[[瑞典|瑞典]]、[[中国|中国]]、[[北美|北美]]等地。

高白鮭出水即死，出水後身体脂肪的固有香味会散发出来，肉质肥厚，白嫩鲜美，营养丰富，生长迅速<ref name="growth"/>，已在中国新疆、[[青海|青海]]、[[中国东北地区|东北]]等地的水库中成功养殖。

高白鲑和[[秋白鲑|秋白鲑]]（''Coregonus autumnalis''）於1998年4月被移植到[[新疆|新疆]][[赛里木湖|赛里木湖]]，在当地生长良好<ref name="biofeat"/>。

== 形态 ==
高白鮭体长且高，体侧稍扁平，鱼体无斑；两颌等长或下颌稍突出，头小，吻尖，口端位、斜裂，口裂後缘达眼前缘下方，眼侧上位，鳃盖膜有鳃弧7－9个；背部银灰色，头后背部弧形黑色隆起，头部有小黑斑，头顶和头侧面有较大黑斑，背鳍和鳃盖有小的黑斑；背鳍3－4个，鳍条9－10个，具脂鳍，与臀鳍後部相对，臀鳍2－3个，鳍条12－16个，基较长，尾鳍深叉形，两叶等长或下叶略长於上叶，胸鳍1个，鳍条13－16个，腹鳍一个，鳍条11－12个；腹部圆，银白色，体被小圆鳞，侧线鳞数76－98个，鳃耙数49－68个。<ref name="growth"/><ref name="biofeat"/><ref>{{cite journal |author=张志华 |coauthors=邹民、孔令杰、王昕阳 |year=2008 |month= |title=高白鲑移植增殖技术研究 |journal=黑龙江水产 |volume=127|issue=5 |pages=4－7 |url=http://www.cqvip.com/qk/83450X/200805/28491282.html |accessdate= 2009-07-31 |language=zh}}</ref>

== 生态 ==
体长40－55厘米，平均体重达到2.5－3千克，一般不超过4－5千克，寿命12－13年，小个体寿命5－6年，不同种群的生长率大大不同。3－6岁时首次产卵，雄鱼比雌鱼早成熟一年。雌鱼在产卵前很快成熟，大多数雌鱼每年都要产卵。产卵期受当地气候影响为秋季至初冬，产於湖泊浅滩的硬砂岩、细沙、粗砂或河流中的石底。在北方的湖泊中，产卵在冰下进行。产卵场受到冰的厚度和水位变化的影响，深度通常为1－3米。产卵期持续12－16日，在2－3℃的条件下进行。它们以[[浮游动物|浮游动物]]（主要为[[甲壳亚门|甲壳类]]）、[[底栖生物|底栖生物]]（尤其是昆虫幼虫、贝类以及海藻）以及水面上的昆虫为食。<ref name="IUCN"/>

高白鲑是由[[二倍体|二倍体]]经双二倍体演化而成的[[多倍体|四倍体]]鱼类<ref>{{cite journal |author=杨成 |coauthors=王国杰、王振吉 |year=2008 |month= |title=高白鲑乳酸脱氢酶同工酶研究 |journal=青海农牧业 |volume=95|issue=3 |pages=34－35 |url=http://www.cqvip.com/qk/90978A/200803/1000398019.html |accessdate= 2009-07-31 |language=zh}}</ref>，[[遗传多样性|遗传多样性]]不高<ref>{{cite journal |author=张俊丽 |coauthors=高天翔、韩志强、郭焱、董崇智 |title=3种白鲑线粒体细胞色素b和16S rRNA基因片段序列分析 |journal=中国水产科学 |volume=14|issue=1 |pages=8－14 |url=http://www.cqvip.com/qk/98270X/200701/23528859.html |accessdate=2009-08-01 |language=zh|date=2007年1月}}</ref>。

== 养殖 ==
高白鲑的适宜生存温度范围1－28℃，最适生长温度为12－16℃，适宜[[pH|pH]]为6－9，适宜盐度为0.1－1.3%，最适盐度为0.6%以下，对低[[溶氧|溶氧]]耐受力高，酸碱度、盐度适应范围广<ref name="ap">{{cite journal |author=牟振波 |coauthors=李怀明、李永发、李小龙 |title=高白鲑的人工繁殖 |journal=淡水渔业 |volume=|issue=6 |pages=17，20－21 |url=http://www.cqvip.com/qk/93836X/199106/459271.html |accessdate= 2009-08-01 |language=zh|date=1991年6月}}</ref>，可分为三种类型：河流型、湖泊型、河湖型。在适宜生长温度范围内，随温度升高，高白鲑的生长速率明显提高<ref>{{cite journal |author=傅平 |year=1996 |month= |title=池塘培育高白鲑仔鱼鱼种试验 |journal=河北渔业 |volume=87|issue=2 |pages=26－27 |url=http://www.cqvip.com/qk/97830X/199602/2090650.html |accessdate= 2009-08-01 |language=zh}}</ref>，随个体增大雄鱼的生长速率会越来越低<ref name="growth">{{cite journal |author=张北平 |coauthors=杨文荣、张人铭、蔡林钢、吐尔逊、刘昆仑、刘萍、李红、付雅丽、郭焱 |title=赛里木湖移植的高白鲑（''Coregonus peled''）生长发育 |journal=水产学杂志 |volume=13|issue=2 |pages=36－47 |url=http://www.cqvip.com/qk/85010A/200002/5599952.html |accessdate=2009-08-02 |language=zh|date=2000年11月}}</ref>。

高白鲑性腺成熟对水温要求非常严格，要求水温必须在3℃以下才能性成熟，开始产卵繁殖<ref name="GBGDRGFH">{{cite journal |author=张北平 |coauthors=蔡林钢、吐尔逊、马燕武、张人铭、李精明、郭焱 |title=高白鲑受精卵人工孵化及胚胎发育观测 |journal=水产学杂志 |volume=14|issue=2 |pages=24－27 |url=http://www.cqvip.com/qk/85010A/200102/6001246.html |accessdate=2009-08-02 |language=zh|date=2001年11月}}</ref>，雄性个体较雌性个体性成熟早<ref name="growth"/>，因此需采卵後[[人工授精|人工授精]]。高白鲑的繁殖期為初冬，轻压成熟雌鱼腹部有[[卵|卵]]流出，若为黏稠橘红色则为优质卵，稀薄暗灰色则为过熟或差卵，轻压成熟雄鱼腹部有[[精液|精液]]流出，若为黏稠淡黄色则为优质精液，若为稀薄淡蓝色则为死亡精液，第二年春4－5月仔鱼孵化，仔鱼平均长8－10毫米，体重约3.5[[毫克|毫克]]，体色浅灰近透明<ref name="ap"/>，这时并不能摄食，只能吸收卵黄囊的营养，待5日後卵黄囊被完全吸收後就开始摄食。仔鱼的适宜摄食温度为10－15℃，稚鱼的适宜摄食温度为15－20℃，而仔鱼在10℃、稚鱼在15℃时摄食强度最高，摄食过程大概需要20多分钟<ref>{{cite journal |author=董崇智 |coauthors=牟振波、赵彩霞、李怀明 |title=高白鲑仔鱼、稚鱼在不同温度条件下的摄食强度试验 |journal=水利渔业 |volume=|issue=5 |pages=13－16 |url=http://www.cqvip.com/qk/91056X/199105/608481.html|issn=1003-1278 |accessdate=2009-08-02 |language=zh|date=1991年9月}}</ref>。水质要保证清洁，无过多悬浮物，否则会因其[[水霉病|水霉病]]、[[气泡病|气泡病]]等疾病，而不良的养殖环境会导致仔鱼畸形<ref>{{cite journal |author=拉玉兰 |title=高白鲑鱼苗培育中的病害防治 |journal=科学养鱼 |volume=|issue=8 |pages=55 |url=http://www.cqvip.com/qk/92435X/200708/25129410.html |accessdate=2009-08-02 |language=zh|date=2007年8月}}</ref>。仔鱼喜爱的食物只有[[卤虫|卤虫]]无节幼虫，这种食物能降低疾病的发生率和幼鱼的死亡率<ref name="biofeat">{{cite journal |author=田永胜 |coauthors=阿斯亚 |title=高白鲑的生物学特性及增养殖技术 |journal=淡水渔业 |volume=30|issue=9 |pages=11－13 |url=http://www.cqvip.com/qk/93836X/200009/4562213.html |accessdate=2009-08-01 |language=zh|date=2000年9月}}</ref>。

== 营养价值 ==
高白鲑肌肉中营养全面，[[蛋白质|蛋白质]]、[[不饱和脂肪酸|不饱和脂肪酸]]、[[必需氨基酸|必需氨基酸]]含量远高於大多数常见养殖鱼类，而肌肉蛋白质含量平均为18.83%，仅比[[乌鳢|乌鳢]]低，脂肪含量平均为13.75%，[[脂肪酸|脂肪酸]]中[[不饱和脂肪酸|不饱和脂肪酸]]比[[饱和脂肪酸|饱和脂肪酸]]含量高，而且其中的高度不饱和脂肪酸含量较高，氨基酸含量平均为17.48%，而且[[氨基酸|氨基酸]]种类齐全，[[鲜味|鲜味]]氨基酸（[[谷氨酸|谷氨酸]]、[[天冬氨酸|天冬氨酸]]、[[丙氨酸|丙氨酸]]、[[甘氨酸|甘氨酸]]）含量较高，因此肉质鲜美。而高白鲑的卵所含蛋白质更为丰富，其中[[维生素E|维生素E]]含量很高，可防止其中所含的不饱和脂肪酸受到氧化破坏。<ref>{{cite journal |author=郭焱  |coauthors=马燕武、蔡林钢、张人铭、吐尔逊、刘昆仑 |title=赛里木湖高白鲑和凹目白鲑肌肉、卵的营养分析评价 |journal=水产学杂志 |volume=17|issue=1 |pages=62－67 |url=http://www.cqvip.com/qk/85010A/200401/9827120.html |accessdate=2009-08-02 |language=zh|date=2004年5月}}</ref>
