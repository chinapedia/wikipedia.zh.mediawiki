'''高雄市立旗津國民小學'''，簡稱'''旗津國小'''，位在[[高雄市|高雄市]][[旗津區|旗津區]]，前身是[[台灣日治時期|日治時期]]的'''打狗公學校'''，是高雄市最早的新式初等教育設施。其舊校舍是高雄市的市定古蹟<ref>{{cite web|url=https://cijin.kcg.gov.tw/Cijin/History.php|title=歷史沿革 - 打狗發源地旗津的聚落群|accessdate=|publisher=高雄市旗津區公所}}</ref>。

== 歷史 ==
[[File:Takao_Kougakkou.jpg|thumb]]

1897年，日本人的打狗俱樂部捐款，將[[旗後|旗後]]山麓的[[旗津臨水宮|臨水宮]]設為「打狗國語傳習所」，於當年6月14日開學，並且聘請[[橋本勝伸|橋本勝伸]]任教，首期的學生有14名<ref name="chij">{{cite web|url=https://cijin.kcg.gov.tw/Cijin/Historic.php?ID=4|title=古蹟導覽 - 旗津國小校舍|accessdate=|publisher=高雄市旗津區公所}}</ref>。1898年，在當地士紳[[林璣璋|林璣璋]]等人的捐款下，學生增加至36人<ref name = "chij"/>。

《[[台灣公學校令|台灣公學校令]]》頒布後，[[陳中和|陳中和]]等人透過申請，將傳習所改為「打狗公學校」，於1898年9月30日創立，是高雄最早的新式初等教育設施<ref name = "tai">{{citation|author=洪郁嫺|title=日治時期「高雄第一公學校」之研究|publisher=國立台南師範學院台灣文化研究所|year=2003}}</ref>。首任校長由橋本勝伸代理擔任。

1903年，首屆學生畢業<ref name="chij" />。

1907年4月，打狗公學校招收了32位女性學生，但未有人實際就學；到1911年，才有17位女生正式入學<ref name="chij" />。

1913年時，打狗公學校開始在今日的旗津國小所在地開設新校舍，並且在1914年正式遷至當地<ref name="chij" />。

1921年，打狗公學校改名為「高雄第一公學校」<ref name = "geo">{{cite web|url=http://www.geocities.jp/idu1952/itiran/itiran1/takao.htm|title=戦前高雄州公学校名称変遷一覧|accessdate=2010-08-29|publisher=泉君の日本語教育史講座}}</ref>，簡稱「一公」<ref name = "tai"/>。在這段期間，高雄第一公學校還曾在棒球拿下全台第一。當時他們先在高雄州決賽擊敗高雄第一小學校(今日的鼓山國小)，取得州冠軍後和另外4個州冠軍(這4個都是小學校)展開決戰。結果，高雄第一公學校除了和台北州冠軍握手言和外，其他3戰全勝，以3勝1和的佳績拿下冠軍。

1937年，改為「高雄市平和公學校」<ref name = "geo"/>。

1941年，再改為「高雄市平和國民學校」。<ref name = "geo"/>

1945年10月25日，國民政府接收台灣後，改為「高雄市第一國民學校」，隔年再改成旗津國民學校。

1968年，台灣實施九年[[國民教育|國民教育]]，學校改為現在的名稱。

2008年，新校舍'''旗峯樓'''落成，大樓內附設大禮堂、視聽中心、

2017年，'''高雄市立旗津國民小學'''120週年校慶。
