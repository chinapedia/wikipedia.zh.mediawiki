'''高尔山风景区'''，原名'''高尔山公园'''，位于[[辽宁省|辽宁省]][[抚顺市|抚顺市]][[顺城区|顺城区]][[高山路|高山路]]，地处抚顺市的北高尔山上，占地240公顷，是集自然山林，名胜古迹为一体的综合性市级公园，也是辽宁省唯一建立在古城遗址上的公园。2014年10月31日晋升为国家“AAA”级景区<ref>{{cite web | language = zh-hans | publisher = 抚顺传媒网 | title = 高尔山风景区晋升国家“AAA”级景区 | url = http://daily.0245.cc/html/2014-11/14/content_499687.htm | author = 来源：抚顺传媒网；记者：田旭 | date = 2014年11月14日 | accessdate = 2014年11月14日 | archive-url = https://web.archive.org/web/20150213191923/http://daily.0245.cc/html/2014-11/14/content_499687.htm | archive-date = 2015年2月13日 | dead-url = yes }}</ref>。

== 简介 ==
高尔山公园原属[[抚顺市城市管理局|抚顺市城市管理局]]园林处，2011年8月1日正式由市管单位划归顺城区，更名为抚顺市顺城区高尔山风景区。园内地势起伏多变，海拔高度80－230米之间，因地势雄险、高屋建瓴，历史上为各个朝代所重视。[[汉代|汉代]]的[[高句丽山城|高句丽山城]]和[[唐朝|唐朝]]时的[[安东都护府|安东都护府]]均建于此。园内建有皇家古典园林建筑锁阳槐荫、锁阳楼、配殿、八角亭、五角亭及长廊等，有距今900余年的辽代八角九级密檐式古塔；有清代的古刹观音阁；有土门叠翠、松林杏雨、万山朝塔、登道流丹等景观。高尔，[[满语|满语]]为[[山槐|山槐]]，意即此地山槐树颇多。而事实上高尔山却也多树，森林覆盖率高达90%<ref>{{cite web| language =zh-hans| publisher =抚顺旅游体验网>智慧游>景区景点>城市公园>顺城区| title =高尔山——国家3A级旅游景区| url =http://www.ncnfs.com/index.php?show--cid-4-id-371.html| author =来源：抚顺旅游体验网| date =| accessdate =| deadurl =yes| archiveurl =https://web.archive.org/web/20150213170320/http://www.ncnfs.com/index.php?show--cid-4-id-371.html| archivedate =2015-02-13}}</ref>。

== 高尔山辽塔 ==
高尔山上有一座[[辽代|辽代]]的[[砖塔|砖塔]]，因居于高尔山上而称高尔山辽塔。辽塔用青砖砌筑，造型为[[八角九级密檐|八角九级密檐]]，顶部[[塔刹|塔刹]]已无存。塔身八角有砖磨椅柱，八面均有进深15厘米的长方形券顶[[佛龛|佛龛]]，龛上有砖雕宝盖，左右两侧有飞天拱托，龛内设须弥座，座上原置[[浮雕佛像|浮雕佛像]]。九级密檐间距从下至上递减收分。《抚顺县志胜迹略》中载：“疑为唐制住持复赓藏骨之塔”。1993年6月，在古塔附近出土一个[[石经幢|石经幢]]，经幢第一面末行刻有“[[大安_(辽道宗)|大安]]四年八月日建”字样。由此认定高尔山塔建于辽代大安四年（1088年）<ref name="中国城市综合实力五十强丛书·抚顺市》编委会／编">{{cite book zh| language =zh-hans| author =中国城市综合实力五十强丛书·抚顺市》编委会／编| title =《中国城市综合实力五十强丛书·抚顺市》>第三章 综合实力>第十八节 旅游风光>高尔山辽塔与观音阁| format =| url =http://book.xuexi365.com/ebook/detail_10397137.html| date =1994年03月1日第1版| publisher =中国城市出版社| location =北京| id =ISBN 750740641| accessdate =1994年03月1日第1次印刷| pages =第126页}}{{dead link|date=十一月 2017 |bot=InternetArchiveBot }}</ref>。

== 观音阁 ==
观音阁为[[明代|明代]]建筑，以供奉[[观世音菩萨|观世音菩萨]]而得名。位于高尔山辽塔南侧的平坦山崖上。观音阁分上下两院，上院青砖建筑有观音阁、东殿、[[钟楼|钟楼]]。东侧靠山有一[[清虚洞|清虚洞]]。上院之内共有大小[[石碑|石碑]]七座，多为修缮庙宇捐款之人名录，分为[[康熙|康熙]]五十五年（1716年）、[[道光|道光]]八年（1828年）、[[光绪|光绪]]十三年（1887年）、[[民国年间|民国年间]]所立。阁内多刻有[[文人雅士|文人雅士]]诗文对联。观音阁两侧门柱置有一副楹联描写观音阁的形势云：“山逼诸天，到此已穷千里目；门依半岭，何须再上一层楼”。下院兴建较晚，为长方形院落，院内原有大殿和两配殿。下院因坍塌拆除，改造成仿古建筑“锁阳楼”<ref name="中国城市综合实力五十强丛书·抚顺市》编委会／编">{{cite book zh| language =zh-hans| author =中国城市综合实力五十强丛书·抚顺市》编委会／编| title =《中国城市综合实力五十强丛书·抚顺市》>第三章 综合实力>第十八节 旅游风光>高尔山辽塔与观音阁| format =| url =http://book.xuexi365.com/ebook/detail_10397137.html| date =1994年03月1日第1版| publisher =中国城市出版社| location =北京| id =ISBN 750740641| accessdate =1994年03月1日第1次印刷| pages =第126页}}{{dead link|date=十一月 2017 |bot=InternetArchiveBot }}</ref>。
