'''高等教育出版社'''，簡稱'''高教社'''，是一家直属于[[中华人民共和国教育部|中华人民共和国教育部]]的专业教育出版机构，成立于1954年5月，主要出版[[高等教育|高等教育]]、[[职业教育|职业教育]]、成人及社会教育等教育类、专业类、科技类出版物。<ref name=intro>{{cite web |url=http://www.hep.com.cn/aboutus/intro |title=高教社简介 |publisher=高等教育出版社 |accessdate=2014-12-03}}</ref>

==歷史==
1954年4月28日，[[政务院文化教育委员会|政务院文化教育委员会]]批復[[中央人民政府高等教育部|中央人民政府高等教育部]]與[[中央人民政府出版總署|中央人民政府出版總署]]，同意成立高等教育出版社。1954年5月1日，在[[商務印書館|商務印書館]]進行[[公私合營|公私合營]]的基礎上，高等教育出版社成立於[[北京市|北京市]][[宣武区|宣武区]][[琉璃廠|琉璃廠]]170號，[[邓小平|邓小平]]题写社名。1954年5月17日，中央人民政府高等教育部發文，任命[[武劍西|武劍西]]擔任高等教育出版社首任社長兼[[總編輯|總編輯]]。1958年4月，高等教育出版社搬遷至北京市[[西城区|西城区]][[承恩胡同|承恩胡同]]7號。1960年1月7日，[[中华人民共和国国务院|中华人民共和国国务院]]總理[[周恩來|周恩來]]簽署任命書，任命武劍西擔任人民教育出版社副社長。1960年4月，高等教育出版社搬遷至北京市[[東城區|東城區]]景山東街45號（今[[沙灘後街|沙灘後街]]55號）。1960年4月1日，隨著[[中央人民政府高等教育部|中央人民政府高等教育部]]與[[中央人民政府教育部|中央人民政府教育部]]的合併，高等教育出版社與[[人民教育出版社|人民教育出版社]]合併為人民教育出版社。1965年1月1日，隨著[[中华人民共和国高等教育部|中华人民共和国高等教育部]]與[[中华人民共和国教育部|中华人民共和国教育部]]分立，高等教育出版社與人民教育出版社分立。1966年5月，[[文化大革命|文化大革命]]全面爆发，高等教育出版社被迫停止编辑、出版業務。1972年1月，中华人民共和国教育部與人民教育出版社一起被撤銷。1972年8月7日，在周恩來的關心下，高等教育出版社與人民教育出版社合併重建。1983年3月9日，[[中华人民共和国文化部|中华人民共和国文化部]]函復中华人民共和国教育部，同意高等教育出版社恢復獨立建制。1985年4月，高等教育出版社在[[上海市|上海市]]成立上海出版分部。1987年9月，高等教育出版社[[照相排版|照相排版]]中心正式投產。1988年7月，高等教育出版社印刷廠在[[天津市|天津市]][[河東區|河東區]]萬新村奠基。1988年10月，[[中央教育電影洗印廠|中央教育電影洗印廠]]與高等教育出版社合併，高等教育出版社於中央教育電影洗印廠原址成立分部。2003年4月，高等教育出版社搬遷至北京市西城区德外大街4号。2006年4月，高等教育出版社啟用北京市[[朝阳区|朝阳区]]惠新东街4号富盛大厦辦公區。2010年1月，高等教育出版社啟用位於北京市[[大興區|大興區]]的物流中心。2010年12月18日，[[中国教育出版传媒集团有限公司|中国教育出版传媒集团有限公司]]成立，高等教育出版社成为其核心成员单位之一。2011年11月，高等教育出版社重組改制為'''高等教育出版社有限公司'''。<ref>{{cite web |url=http://www.hep.com.cn/xpe/portal/a237a64a-1468-1000-9cc7-3fafc67de19c?viewer=1&uuid=a273f829-1468-1000-b695-3fafc67de19c |title=輝煌歷程 發展新貌 |publisher=高等教育出版社 |accessdate=2015-07-04}}</ref>

2004年，高等教育出版社成立法务部，打擊[[盜版|盜版]]。

2015年6月11日，高等教育出版社与[[华中师范大学|华中师范大学]]签署战略合作协议<ref>{{cite web |url=http://www.hep.com.cn/news/details?uuid=829a4d91-14e0-1000-801b-cf7b669182c9 |title=高教社与华中师范大学签署战略合作协议 |publisher=高等教育出版社 |date=2015-06-19 |accessdate=2015-07-04}}</ref>。2015年6月18日，高等教育出版社61周年社慶，社长苏雨恒、[[党委书记|党委书记]]宋永刚與总编辑杨祥共同启动两大[[电子商务|电子商务]]平台：“高教社网上书城”（www.landraco.com）與“高教社[[天猫|天猫]]旗舰店”（gdjycbs.tmall.com）<ref>{{cite web |url=http://news.xinhuanet.com/edu/2015-06/19/c_127933210.htm |title=高等教育出版社61岁生日 电商平台正式上线 |publisher=新华网 |date=2015-06-19 |accessdate=2015-07-04}}</ref>。
