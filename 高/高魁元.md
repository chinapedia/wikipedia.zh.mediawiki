'''高魁元'''（{{bd|1907年|3月26日|2012年|5月7日|catIdx=Gao高}}），[[表字|字]]'''煜辰'''，[[中華民國陸軍|中華民國陸軍]][[一級上將|一級上將]]，[[山東省|山東省]][[嶧縣|嶧縣]]人，畢業於[[陸軍官校|陸軍官校]]4期、[[美國陸軍|美國陸軍]]參謀大學，曾任[[中華民國參謀總長|中華民國參謀總長]]與[[中華民國國防部|國防部]]部長、總統府戰略顧問。

== 早年經歷 ==
1926年，高魁元考入[[黃埔軍校|黃埔軍校]]第4期。畢業後，初任[[國民革命軍|國民革命軍]]總司令部獨立[[中華民國憲兵|憲兵]]團連[[排長|排長]]與[[連長|連長]]。1928年3月，升任[[中華民國陸軍|陸軍]]獨立第二師第二團第一營營長。1929年5月，調任[[陸軍軍官學校|陸軍軍官學校]]武漢分校第六隊區隊長。

1931年至1936年間，高魁元參與[[第一次國共內戰|第一次國共內戰]]。1936年，因戰功升任[[國民革命軍第十八軍|國民革命軍第十八軍]]第十四師第四十二旅第八十三團團長。

1937年，[[中國抗日戰爭|中國抗日戰爭]]爆發，高魁元參與[[淞滬會戰|淞滬會戰]]、[[崑崙關戰役|崑崙關戰役]]及[[湘西會戰|湘西會戰]]。戰爭中，升任國民革命軍第九十九師[[少將|少將]][[師長|師長]]。1941年，以師長之職參加第三次[[長沙會戰|長沙會戰]]，奉命守備[[湘陰|湘陰]]之[[新牆河|新牆河]]、[[歸義|歸義]]、[[營田|營田]]一線。

1942年3月，高魁元任[[國民政府軍事委員會|國民政府軍事委員會]]高級參謀。[[第二次國共內戰|第二次國共內戰]]中，再任第一一八師師長，因治軍頗力偶有戰績，於當時敗多勝少的國軍中為少見。1949年4月，調任第十八軍[[軍長|軍長]]。

1949年10月，高氏戍守[[金門|金門]]，是[[中華民國國軍|國軍]]於「[[古寧頭戰役|古寧頭戰役]]」戰勝關鍵人物。後任台北防衛區副司令、軍長。1955年，任[[陸軍總司令部|陸軍總司令部]]政戰主任兼[[行政院|行政院]][[反共義士|反共義士]]輔導處副處長，輔導總隊總隊長。1957年，任陸軍總司令部副總司令。1961年，任[[國防部政治作戰局|國防部總政戰部]]主任。

1965年，蔣介石明令以高魁元為[[陸軍總司令|陸軍總司令]]。<ref name="年表，第110頁">[[陳布雷|陳-{布}-雷]]等編著：《蔣介石先生年表》，台北：[[傳記文學|傳記文學]]出版社，1978年6月1日，第110頁</ref>

== 晚年經歷 ==
1967年，任國防部[[參謀總長|參謀總長]]，晉升陸軍[[一級上將|一級上將]]。1970年，任[[總統府|總統府]][[參軍長|參軍長]]。1973年，[[蔣經國|蔣經國]]出任第9任行政院長後，再將臨退的高魁元找回，擔任[[國防部長|國防部長]]。1980年11月，發生[[黃植誠|黃植誠]]駕機叛逃事件。

1981年，高魁元請辭國防部長，當時高齡73歲。11月22日，蔣經國函電[[宋美齡|宋美齡]]：
{{quote|
「國防部長高魁元因病請辭其情懇切不得已業予照准擬即以參謀總長[[宋長志|宋長志]]調任國防部長陸軍總司令[[郝柏村|郝柏村]]升任參謀總長以資熟手又行政院部會及省市主管為配合需要亦將予以局部改組母親[[休士頓|休士頓]]之行想甚安好專肅敬叩福安兒」<ref>{{cite book|editors=周美華、蕭李居|title=《蔣經國書信集——與宋美齡往來函電》(下)|location=台北|publisher=「[[國史館|國史館]]」|year=2009}}</ref>{{rp|211}}
}}

高魁元獲頒[[青天白日勳章|青天白日勳章]]，轉任[[中華民國總統府|總統府]]戰略顧問。

高魁元是[[中華民國歷史|中華民國歷史]]上年紀最大、擔任時間最久的國防部長，亦曾是[[中華民國國軍|中華民國國軍]]年齡最大的現役將領，「最長壽的上將」實當之無愧。

== 逝世 ==
2012年5月7日晚間9點41分，高魁元因多重器官衰竭，於[[台北市|台北市]][[三軍總醫院|三軍總醫院]]病逝，享壽105歲。<ref>{{cite press release |publisher=[[中央通訊社|中央通訊社]] |date=2012-05-07|title=戰略顧問高魁元病逝 |url=http://www.cna.com.tw/News/aIPL/201205070346.aspx}}</ref><ref>{{cite press release |publisher=《[[自由時報|自由時報]]》 |date=2012年5月8日 |title=總統府戰略顧問 高魁元病逝 |url=http://www.libertytimes.com.tw/2012/new/may/8/today-p8.htm}}</ref>6月2日，中華民國總統[[馬英九|馬英九]]出席高魁元安息彌撒，並頒發[[褒揚令|褒揚令]]。<ref>{{cite press release |publisher=《[[大紀元時報|大紀元時報]]》 |date=2012年6月2日 |title=高魁元病逝 馬英九頒褒揚令 |url=http://www.epochtimes.com/b5/12/6/2/n3603345.htm}}</ref>
褒揚令原文如下：
{{Wikisource|總統華總二榮字第10100113500號令}}
<blockquote style="border: 1px solid blue; padding: 0.5em 0.8em;">
　　總統府戰略顧問、陸軍一級上將高魁元，勁節軒闢，剛毅槃才。少歲國故頻仍，四郊多壘，矢志效力戎車，迺投身中央軍校。七七抗戰軍興，歷預淞滬、崑崙關、湘西、第三次長沙會戰諸役，縱橫馳騁，轉進征伐，襲日寇強弩之末，開民族必勝之勢。臺海煙硝禍起，承命戍守金門，痛擊來犯共軍，俘獲敵虜五千餘人，史稱「[[古寧頭大捷|古寧頭大捷]]」，魯陽揮戈，綏靖赤氛，撐持前哨聖戰，允為臺澎雄藩，籌維遙制，鼙鼓振旅；貫頤奮戟，逆轉變局。歷任金門防衛司令部副司令、陸軍總司令、國防部參謀總長暨部長等職，碩擘政策組織修訂，鋪謀建軍整備事宜，秉鉞鷹揚，弼成生聚。曾獲頒一等雲麾、寶鼎暨青天白日等國內外勳章殊譽，復獲聘總統府戰略顧問，拱衛樞宸，聲徽廣衍。綜其生平，鋒鏑浴血，作保民護土之干城；繕甲治兵，奠國家社稷於磐石，榮光延風，訏謨勳業；忠藎垂型，丹青千古。遽聞上壽凋零，衣冠盛事，應予明令褒揚，用示政府崇報耆勳之至意。<br>

行政院院長　[[陳冲|陳　冲]]　　　　</p>
</blockquote>

== 評價 ==
高魁元一帆風順，主持國防大政，他經歷重要軍政之完整而多，在並世將領中，沒有誰能夠和他相比。<ref name="評傳，第190頁">[[王禹廷|王禹廷]]著：《胡璉評傳》，台北：傳記文學出版社，1987年6月15日，第190頁</ref>胡璉生前，常以有此好朋友好同志為榮為樂。<ref name="評傳，第190頁"/>
