'''高彦俦'''（{{bd|？||965年|1月11日|catIdx=G}}<ref name=SS1>[[:s:宋史/卷001|《宋史》卷一]]</ref>），[[中国|中国]][[五代十国|五代十国]]时期[[后蜀|后蜀]]军事人物，[[并州|并州]][[太原|太原]]人。<ref name=SS479>[[:s:宋史/卷479|《宋史》卷四百七十九]]</ref><ref name=SGCQ54>《[[十国春秋|十国春秋]]·高彦俦传》</ref>《[[五代通录|五代通录]]》及《[[新五代史|新五代史]]·[[王环|王环]]传》误作'''高处俦'''。

== 早期经历 ==
高彦俦父高晖是[[晋_(五代)|晋国]]宣威军使。<ref name=SS479/><ref name=SGCQ54/>高彦俦年轻时善骑射，慷慨有大节。晋王[[李存勖|李存勖]]建立[[后唐|后唐]]后，以姐夫[[孟知祥|孟知祥]]为北都[[太原|太原]]留守，孟知祥召高彦俦为军校。<ref name=h9k>《[[九国志|九国志]]·高彦俦传》</ref>

== 效力后蜀高祖 ==
后来孟知祥为[[西川节度使|西川节度使]]，高彦俦随他入蜀，<ref name=SS479/>被授亲卫指挥使。<ref name=h9k/>孟知祥自立，高彦俦被任为昭武军监押。<ref name=SGCQ54/>[[长兴_(后唐)|长兴]]三年（932年）二月，西川节度副使[[赵季良|赵季良]]与诸将议遣时任昭武都监高彦俦率兵攻取[[壁州|壁州]]，以绝转入[[秦岭|秦岭]]后诸州的[[山南西道|山南西道]]军队，但孟知祥与僚佐商议时，被掌书记[[李昊|李昊]]劝止。<ref>[[:s:资治通鉴/卷277|《资治通鉴》卷二百七十七]]</ref>

孟知祥称帝建立后蜀，即高祖皇帝。

== 效力后蜀后主 ==
[[孟昶|孟昶]]嗣蜀帝位后，高彦俦迁[[茂州|茂州]]、[[邛州|邛州]][[刺史|刺史]]。[[广政|广政]]十年（947年）正月，因[[契丹|契丹]]灭[[后晋|后晋]]，后晋[[雄武节度使|雄武节度使]][[何重建|何重建]]降蜀，孟昶署高彦俦领雄武节度使，<ref name=SS274>[[:s:宋史/卷274|《宋史》卷二百七十四]]</ref><ref>《宋史·赵玭传》作孟知祥所署，误。</ref>领众守雄武军军部[[秦州|秦州]]，不久召还领右卫圣马步军都指挥使。<ref name=h9k/>十一年（948年）六月，[[后汉|后汉]][[凤翔节度使|凤翔节度使]][[王景崇_(五代)|王景崇]]降蜀，为后汉军所攻，十二月，告急于蜀，孟昶派[[山南西道节度使|山南西道节度使]][[安思谦|安思谦]]相救。时汉军已入[[大散关|大散关]]，克安都砦。安思谦进屯散关，遣高彦俦、[[眉州|眉州]]刺史[[申贵|申贵]]攻箭筈安都寨，高彦俦率所部先进军，汉军烧砦毁阁遁去，高彦俦尽出精锐追击，复砦而还。<ref>[[:s:资治通鉴/卷288|《资治通鉴》卷二百八十八]]</ref>不久，领[[赵州|赵州]]刺史。不久为奉銮肃卫都指挥副使，改右骁锐马军都指挥使，加光圣马军都指挥使，真拜[[源州|源州]][[武定军节度使|武定军节度使]]。<ref name=SS479/><ref name=SGCQ54/>

{{main|后周攻后蜀之战}}
十八年（955年）五月，[[后周|后周]]派[[凤翔节度使|凤翔节度使]][[王景_(宋)|王景]]、宣徽南院使、镇安节度使[[向训|向训]]攻秦州、[[凤州|凤州]]，孟昶令捧圣控鹤都指挥使、[[保宁节度使|保宁节度使]][[李廷珪_(后蜀)|李廷珪]]为北路行营都统，时任左卫圣步军都指挥使高彦俦为招讨使，<ref>《五代通录》《新五代史·王环传》误作高彦俦此时为雄武军节度使。时任节度使应为[[韩继勋|韩继勋]]。</ref>[[武宁节度使|武宁节度使]][[吕彦珂|吕彦珂]]（或作[[吕彦琦|吕彦琦]]）副之，客省使[[赵崇韬|赵崇韬]]为都监。八月，高彦俦还没到，已传来蜀军接连溃败于唐仓等地的消息，李廷珪、高彦俦等溃走，想退保青泥岭。雄武军节度判官[[赵玭|赵玭]]闭关不纳蜀军，以秦州归周。<ref name=SS274/><ref>[[:s:资治通鉴/卷292|《资治通鉴》卷二百九十二]]</ref><ref>《九国志·李廷珪传》</ref><ref name=NHFD64>[[:s:新五代史/卷64|《新五代史》卷六十四]]</ref>高彦俦逃回都城[[成都|成都]]，<ref>[[:s:新五代史/卷50|《新五代史》卷五十]]</ref>孟昶不怪罪他，<ref name=h9k/>以为右奉銮肃卫都指挥使，改功德使。<ref name=SS479/><ref name=SGCQ54/>

二十年（957年）六月，孟昶母[[李太后_(后蜀高祖)|李太后]]对孟昶说“在我看来，只有高彦俦是你父亲在太原的故人，秉心忠实，多所经练，不会背叛你，可以委任”，<ref name=SGCQ54/>但孟昶不能用其言。<ref name=SS479/><ref>[[:s:资治通鉴/卷293|《资治通鉴》卷二百九十三]]</ref>

二十一年（958年）十二月，高彦俦在峡路巡检制置使任上被任为招讨使。<ref>[[:s:资治通鉴/卷294|《资治通鉴》卷二百九十四]]</ref>二十二年（959年），出授[[夔州|夔州]]宁江军都巡检制置招讨使，<ref name=h9k/>加宣徽北院事、[[利州|利州]][[昭武军节度使|昭武军节度使]]。<ref name=SS479/><ref name=SGCQ54/>

=== 以身殉国 ===
{{main|夔州之战}}
二十七年（964年）十二月，[[宋灭后蜀之战|北宋攻蜀]]，分兵由峡路攻夔州。蜀军已在夔州的长江上用船只等物封锁江面，并在其上放置了多重敌栅，并沿江放置数门火炮，但在[[宋太祖|宋太祖]]预先指点下，宋大将[[归州|归州]]路副都部署[[刘廷让|刘光义]]顺流而上，距锁江处三十里舍船步行前进夺取浮桥，破除江面障碍，再乘战船兵临夔州城下。高彦俦对副使赵崇济、监军武守谦说：“北军涉远而来，速战对他们有利，不如坚壁以待为上。”武守谦不从，独自领麾下千余人出战。时刘光义、都监枢密承旨[[曹彬|曹彬]]屯兵白帝庙西，遣骑将马军都指挥使张廷翰等引兵与武守谦战于猪头铺，武守谦败走，张廷翰等乘胜登城，刘光义率大军随后而至。高彦俦整肃众将正要出战拒敌，宋军已乘城而入。高彦俦惶骇不知所措，无计可施，力战于城下，身受十余枪，左右都散去，奔回家中，闭牙城拒守。都巡检判官[[罗济|罗济]]劝他单骑归成都，高彦俦说：“我昔已失[[天水|天水]]（即秦州），今又不能守夔州，纵皇帝不忍杀我，我又有何面目见蜀人！”罗济又劝其降宋，高彦俦说：“我老幼百口在成都，若我一身偷生，全族怎么办？我今日只有死！”于是解下符节印绶给罗济。宋军破牙城门而入，高彦俦挺剑拒敌，杀十余人，整衣冠望西北两次下拜，登楼纵火自焚。<ref name=SS1/><ref name=NHFD64/><ref>[[:s:宋史/卷259|《宋史》卷二百五十九]]</ref>刘光义怜其忠，数日后在灰烬中得其骨殖，以礼收葬。<ref name=SS479/><ref name=SGCQ54/><ref name=h9k/><ref>[[:s:续资治通鉴/卷004|《续资治通鉴》卷四]]</ref>

== 评价 ==
