'''高翔'''（{{bd|1932年|12月20日||}}）[[辽宁省|辽宁省]][[盖县|盖县]]人，[[中国人民解放军海军航空兵|中国人民解放军海军航空兵]][[海空雄鹰团|海空雄鹰团]]战斗英雄<ref name=zgw>{{Cite web |url = http://military.china.com.cn/2012-08/30/content_26382921.htm |title =  战斗英雄高翔：47年前椰林上空击落美军战机|author =  |date =2012-08-30  |publisher =中国网  |language =  |accessdate =  |quote =  }}</ref>。

==生平==
高翔出生在一个贫寒家庭，是独生子，由母亲养育长大，并供他读书4年零8个月。15岁时，高翔回家当民兵。1947年冬，县里在每个村招收1名参政员，高翔依靠当村长的叔叔的关系被录取。1947年底，参加[[东北人民自卫军|东北人民自卫军]]。高翔因有高小学历，便被分配到辽南军政干校卫生队担任卫生员。1948年底，辽南军政干校改编，成立整训5师，1949年入[[山海关|山海关]]后成为[[中国人民解放军第四野战军|中国人民解放军第四野战军]]整训5师。高翔随军入关。1949年冬，高翔被抽调出部队，协助医生护送华南工作团赴[[广州|广州]]。此后高翔被留在[[广东省人民政府|广东省人民政府]]工作两年，担任广东省人民政府办公厅行政处司药<ref name=zgw/>。

1951年，高翔报名重回军队当飞行员。1951年8月至9月间进入航校。1952年初，在[[河北|河北]][[保定|保定]]开始驾驶初级教练机，不到半年又到河北[[涿州市|涿州]]试飞中级教练机。后到河北[[定兴县|定兴]]试飞喷气式战斗机，飞到1年零10个月毕业<ref name=zgw/>。

1964年[[北部湾事件|北部湾事件]]后，[[美军|美军]]全面扩大[[越南战争|越南战争]]，时常派飞机侵入中国[[海南岛|海南岛]]领空侦察，中国人民解放军决定反击。1964年底，高翔所在团派第一批小分队3人2架飞机到[[海南岛|海南岛]]打美军无人驾驶飞机。1965年6月13日，高翔随第二批小分队派往海南岛<ref name=zgw/>。

1965年9月20日10时27分，一架美制F-104C型战斗轰炸机从[[越南|越南]][[岘港国际机场|岘港机场]]起飞后，沿中国海南岛西部的领海线北上，在中国领海上空进行“S”线飞行，正在值班的高翔（海航10团二大队大队长）、黄凤生（海航10团二大队副大队长）双机编队起飞。高翔从距敌机291米一直射击至距敌机39米，敌机爆炸坠海，飞行员菲利普·史密斯跳伞后被[[中国民兵|中国民兵]]符气合俘虏。1965年9月，海军为高翔记一等功。高翔受到[[毛泽东|毛泽东]]、[[周恩来|周恩来]]等中国[[党和国家领导人|党和国家领导人]]接见。战俘史密斯在中国受到优待。1973年3月，美国自越南撤军后，中国政府释放了战俘史密斯。1989年10月18日，史密斯在[[上海|上海]][[锦江饭店|锦江饭店]]见到了高翔，并进行了轻松愉快的交流<ref>{{Cite web |url =http://www.81.cn/2016hjcllqzn/2016-04/21/content_7016968.htm  |title =  战斗英雄高翔|author =  |date =2016-04-21  |publisher = 中国军网 |language =  |accessdate =  |quote =  }}</ref><ref name=zgw/><ref name=qilu>齐鲁会举办高翔同志八十寿辰恳谈会，集邮报2012年12月26日第52期，第2版</ref><ref name=youq>老英雄高翔八十寿诞集邮收藏展在青岛举行，中国集邮报2012年12月25日第95期，第1版</ref>。

晚年高翔热爱集邮。2012年80岁时，高翔在[[青岛|青岛]]举办集邮收藏展<ref name=qilu/><ref name=youq/>。
