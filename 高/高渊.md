'''高渊'''（{{lang-ms|Nibong Tebal}}），是[[马来西亚|马来西亚]][[槟城州|槟城州]][[威南縣|威南縣]]最南端也是[[威省|威省]]境内第三大的市镇。南有[[巴里文打|巴里文打]]，东有[[万拉峇鲁|万拉峇鲁]]。

==历史==
高渊的名字而来是因为高渊有许多名叫Nibong的大树，Tebal则是因为早期人民把Nibong的叶放在屋顶上，而中文名则是取至早上当地一位官员的名字，而该官员的名字叫「高远」，久而久之就变成了「高渊」。高渊的历史悠久，在1900年，高渊就有人居住，并定居在[[吉辇河|吉辇河]]（Sungai Kerian）两岸。

==人口==
高渊人口以讲[[潮州语|潮州语]]的华裔为主，还有[[海南语|海南语]]等。

== 地形 ==
高渊地形平缓，较少高地。东边地势较高，西边地势较平缓，适合发展。区内有一座州立公园（武吉班卓州公园）。

== 经济 ==
高渊的经济以[[渔业|渔业]]为主。高渊的渔业可说是[[马来西亚|马来西亚]]最大的。高渊是拥有[[马来西亚|马来西亚]]最多的[[养鱼场|养鱼场]]，高渊也是一个建于[[港口|港口]]的地方，故称高渊。[[高渊港口|高渊港口]]（Sungai Udang）建于河岸，使许多渔民出售鱼的地方。近年来，由于捕鱼业受到[[苏丹阿都哈林大桥|槟城二桥]]的工程所影响，使得捕鱼数量逐年减少。因此，许多商人渐渐转向投资海上[[养殖业|养殖业]]，造成养殖地区爆满，所以需要要有一个全面的方案，解决海上养殖业的问题。此外，高渊的[[农业|农业]]也较发达。有大片的油棕园。高渊还有些[[服务业|服务业]]、[[制造业|制造业]]、[[汽车业|汽车业]]等等。武吉班卓工业花园则是高渊的一个工业区，制造了许多就业机会，许多工厂都设立在这里。

== 教育 ==
此区有三间小学、四间中学，而大部分[[马来西亚华人|华裔]]学生都就读于培德华小。

== 基建 ==
高渊拥有良好的基建，在高渊卫星市有一所[[诊所|诊所]]，方便附近的居民来看诊。此外，成功花园里设有高渊最大的高渊巴刹，并在近年来进行翻新，售卖商品多样化。此外，横跨高渊市中心的火车站也是高渊繁荣的强心剂，但已被电气化的[[高渊站|高渊站]]取代为当地居民提供服务。<ref>[http://www.sinchew.com.my/node/776442]</ref>

==交通==
高渊有良好的交通网，汽车可以自由往来。高渊没有飞机场，但可通过高速工具，或[[苏丹阿都哈林大桥|槟城二桥]]，可以轻易抵达高渊。高渊地处一个十分良好的地段，北上有[[大山脚|大山脚]]、[[乔治市|乔治市]]等城市，南下则能够抵达[[太平_(霹雳)|太平]]、[[怡保|怡保]]、[[吉隆坡|吉隆坡]]等大型城市。如今，「[[檳島|檳島]]南部」已建設一座[[苏丹阿都哈林大桥|大橋]]，與對岸的[[威省|威省]]連接（此座[[東南亞|東南亞]]最長的橋是由[[馬來西亞|馬來西亞]]與[[中華人民共和國|中華人民共和國]]共同出資興建）。现在，人们可以更快的从槟岛抵达高渊，也可通过[[马来亚铁道|马来亚铁道]][[KTM通勤铁路北马区|通勤铁路(KTM Komuter)]]的[[高渊站|高渊火车站]]抵达这里。

== 社会与文化 ==
每年，高渊会举办一年一度的[[盂兰胜会|盂兰胜会]]。每逢这个时候，高渊有几个花园都会举办这一个盛事，热闹非凡。

== 景点 ==
=== 大伯公庙 ===
高渊有一个大伯公庙，历史悠久，是游客必有的其中一个地方。大伯公庙由一个小庙宇变成如此庞大的庙，实在堪称奇迹。

===观赏萤火虫 ===
高渊还有观赏[[萤火虫|萤火虫]]的地方。只要乘船去[[高渊港口|高渊港口]]大约一公里外河边，就能观赏萤火虫了。但是，游客在观赏萤火虫时必须安静，以免吓走萤火虫。

===高渊大街===
高渊大街是高渊重要的道路。它可衔接至多个地方。高渊大街大约1公里。中间一排的路灯，造型独特。晚上时灯光闪闪，十分漂亮。

===高渊卫星市===
高渊卫星市是高渊其中一个花园，素有好吃街以及 “高渊夜市场" 之名称。高渊卫星市設有数家饮食中心，夜晚时十分热闹，提供多种食物选择。每逢星期日晚上，卫星市都会有华人夜市（Pasar Malam），主要由当地华人经营。

===百货公司===
高渊一座百货公司——华达（YAWATA），是高渊最大的百货公司，前方有许多档口。
