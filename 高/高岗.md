'''高岗'''（{{bd|1905年|10月25日|1954年|8月17日|catIdx=G高}}）原名'''高崇德'''，[[表字|字]]'''硕卿'''，[[中国|中国]][[陕西|陕西]][[米脂县|米脂县]][[武镇|武镇]]（今属[[榆林市|榆林市]][[横山区|横山区]]）人，[[中国共产党|中国共产党]]和[[中华人民共和国|中华人民共和国]]已故主要領導人之一。

高岗早年在西北地区进行革命活动，后任陕甘边红军临时总指挥部政治委员、[[中国工农红军第二十六军|红二十六军]]政治委员、[[中国工农红军第十五军团|红十五军团]]政治部主任等职务，是[[陕北红军|陕北红军]]的代表人物之一。[[中国抗日战争|抗日战争]]时期，任[[陕甘宁边区|陕甘宁边区]]保安司令、[[中共中央西北局|中共中央西北局]]书记、[[陕甘宁留守兵团|陕甘宁晋绥联防军]]代政委等职务。第二次国共内战时期，任[[中共中央东北局|中共中央东北局]]副书记、书记、东北军区司令员兼政治委员等职。[[中华人民共和国|中华人民共和国]]成立后，任[[中华人民共和国中央人民政府_(1949–1954)|中央人民政府]]副主席、[[东北人民政府|东北人民政府]]主席，集东北党政军大权于一身，号称“东北王”。1953年，奉调进京出任[[中央人民政府国家计划委员会|国家计划委员会]]主席，后与[[饶漱石|饶漱石]]密谋取代[[刘少奇|刘少奇]]和[[周恩来|周恩来]]而遭批判，自杀身亡。1955年，被开除党籍。

== 生平 ==
=== 早期革命生涯 ===
{{see also|刘志丹|谢子长|红二十六军|红十五军团|长征}}
高岗出身农家，小时候在[[米脂县|米脂县]][[龙镇_(米脂县)|龙镇]]小学读书，后考入[[横山县|横山县]]立第一高等小学堂学习。1925年参与组织横山一高的学潮，此时结识[[刘志丹|刘志丹]]{{r|biography|page=14}}。同年，进入[[榆林中学|榆林中学]]学习。高岗由[[史巍然|史巍然]]介绍于1926年加入[[中国共产党|中国共产党]]，并于次年进入[[冯玉祥|冯玉祥]][[国民军|国民军]]的[[西北中山军事学校|西北中山军事学校]]学习{{r|biography|page=20}}。

1927年至1931年，在国军西北地方部队中秘密开展兵运工作，发动武装起义。1932年任陕甘工农红军游击队队委书记，后因游击队战斗失利，高岗被派到陕西省委工作。1933年7月28日，因中共叛徒告密，陕西省委在西安福盛楼饭馆开会时遭破坏，省委书记[[袁岳栋|袁岳栋]]等被捕，只有高岗和[[贾拓夫|贾拓夫]]侥幸逃脱。

1933年8月，高岗任陕甘边红军临时总指挥部政治委员；11月后，任[[红二十六军|红二十六军]]第四十二师政治委员。1934年1月初，高岗在[[正宁|正宁]]南邑堡战斗后因调戏妇女违纪，被撤销师政委职务，下派到第二路陕甘边工农游击队，任总指挥部政委。1934年4月，二路游击队配合第四十二师取得西华池战斗的胜利，次月高岗复任第四十二师政治委员。1935年7月的阎家洼子会议上，高岗被[[郭洪涛|郭洪涛]]批为“梢山主义”，被免去第四十二师政治委员的职务。

1935年2月，高岗任[[西北革命军事委员会|西北革命军事委员会]]前敌指挥部政治委员。9月[[红二十五军|红二十五军]]与红二十六军、[[红二十七军|红二十七军]]会师成立[[红十五军团|红十五军团]]后，任军团政治部主任，参与[[劳山战役|劳山战役]]，在战斗进入到最后阶段时，高岗赤膊上阵{{r|biography|page=63}}。后与[[刘志丹|刘志丹]]、[[习仲勋|习仲勋]]等在肃反中被逮捕，[[中央红军|中央红军]]到达陕北后，高岗被释放<ref>{{cite book |author = 吴葆朴，李志英著 |title = 《秦邦宪（博古）传》 | year = 2007 | publisher = 北京：中共党史出版社  |isbn = 978-7-80199-6855|pages=196}}</ref>。1936年1月，高岗被派到内蒙古[[三边|三边]]地区，6月至10月任中共蒙古工作委员会负责人。10月任中共陕甘省委少数民族工作委员会副书记，12月任书记。

=== 抗日战争时期 ===
{{see also|陕甘宁边区|陕甘宁留守兵团|土地改革运动}}
全面抗日战争爆发后，高岗留守陕北，出任[[陕甘宁边区|陕甘宁边区]]保安司令员、中共陕甘宁边区党委少数民族工作委员会书记。1937年11月，任[[八路军|八路军]]骑兵司令{{r|biography|page=82}}。1937年12月中共中央政治局确定高岗为中共“七大”准备委员会前25位准备委员之一（排名列最后）。1938年5月，任中共陕甘宁边区委书记。1939年1月起，任陕甘宁边区参议会参议长、议长。1939年6月,高岗担任了八路军留守兵团的政委。1940年9月，任中共陕甘宁边区中央局书记。

1941年5月，高岗任[[中共西北中央局|中共西北中央局]]书记{{r|biography|page=91}}，兼任西北局统战部部长、[[民族学院|民族学院]]院长，负责西北地区党务，并具体负责陕甘宁边区政府和民众[[大生产运动|大生产运动]]。1943年，高岗兼任[[陕甘宁晋绥联防军|陕甘宁晋绥联防军]]副政治委员、代政治委员。1945年6月，高岗当选[[中央政治局|中央政治局]]委员。

=== 第二次国共内战时期 ===
{{See also|中共中央东北局|土地改革运动}}

1945年10月下旬，高岗和[[张闻天|张闻天]]等一起乘坐美国调停小组的飞机从延安飞往东北。11月任[[北满|北满]]军区司令，在北满四省开辟中共根据地。次年4月，兼任哈尔滨前线野战司令，组织攻取[[哈尔滨|哈尔滨]]{{r|biography|page=160}}。1946年6月。高岗任[[中共中央东北局|中共中央东北局]]副书记、[[东北民主联军|东北民主联军]]副政委，主持东北局的后方工作{{r|biography|page=191}}<ref>Maurice Meisner, Mao's China: A History of the People's Republic (New York, 1977), p. 131.</ref>，领导东北土地改革事务。1947年，高岗曾去[[赤峰|赤峰]]，指导冀察热辽分局的土地改革工作。[[程子华|程子华]]在回忆录中指高岗在热河土改中推行极左路线<ref>{{cite book |author=程子华 |title=《程子华回忆录》|year=1987 |publisher=解放军出版社 |ISBN = |page =294 }}</ref>。在同年的[[全国土地工作会议|全国土地工作会议]]后，高岗认为[[刘少奇|刘少奇]]片面听取[[彭真|彭真]]的叙述，有偏袒彭真的倾向，开始对刘少奇有所不满{{NoteTag|高岗《我的反省》：“对少奇产生隔阂，大概始于1947年。当时[[赵德尊|赵德尊]]去平山开土地会议，我们让他向少奇汇报东北的情况，其中包括对东北工作方针的争论、彭真等人的错误，以及根据中央决定对东北局进行调整等情况，希望少奇支持我们的工作。但是少奇听了汇报后，不说东北争论的是与非，只说‘要注意团结，不要落井下石。过去满洲党和朝解党就是因为不团结，结果被敌人搞垮了。’赵德尊回来传达少奇的谈话后，我们很不理解：明明是纠正错误，怎么说是‘落井下石’、‘不团结’？因此认为少奇对彭真等人的错误有些袒护，不支持我们的工作。从此对少奇产生了隔阂，但这只是工作上的问题。”{{r|biography|page=178}}}}。1948年1月，东北民主联军改称东北军区，高岗任东北军区第一副司令员兼副政治委员<ref>{{cite book |author=张明金、刘立勤 |title=《中国人民解放军历史上的200个军区》|year=2010年 |publisher=解放军文艺出版社 |location=北京市西城区地安门西大街40号 |page = 344 |ISBN = 9787503322525 |language = zh-cn }}</ref>。

1948年底[[辽沈战役|辽沈战役]]结束后，[[林彪|林彪]]、[[罗荣桓|罗荣桓]]率领大军入关，高岗留在东北，先后担任中共中央东北局第一书记、[[东北人民政府|东北人民政府]]主席、[[东北军区|东北军区]]司令兼政委等职务，是当时六个大区唯一一个身兼党政军四大要职的领导人，是名符其实的“东北王”。1949年7-8月，高岗与[[刘少奇|刘少奇]]、[[王稼祥|王稼祥]]密访[[莫斯科|莫斯科]]<ref>{{cite web |url = http://www.people.com.cn/GB/paper2086/10205/934151.html |title = 高岗：在毛泽东和斯大林之间（知情者说） |author = 潘琪译 |publisher =  |date = 2003 |accessdate = 2013-11-16 |archive-url = https://web.archive.org/web/20141206002407/http://www.people.com.cn/GB/paper2086/10205/934151.html |archive-date = 2014-12-06 |dead-url = no }}</ref>。

1949年9月，高岗出席[[中国人民政治协商会议第一届全体会议|中国人民政治协商会议第一届全体会议]]，当选[[中华人民共和国中央人民政府|中华人民共和国中央人民政府]]副主席，并出席[[开国大典|开国大典]]。

=== 中华人民共和国成立后 ===
[[File:Gaogang1949.gif|thumb]]上，高岗代表全中国[[解放区|解放区]]讲话]]
{{See also|东北人民政府|朝鲜战争|高岗饶漱石事件|饶漱石|中央人民政府国家计划委员会}}

中华人民共和国成立后，高岗主政东北，组织恢复经济等工作。在高岗治下，东北1949年粮食产量比上年平均增长20%左右，全年工业生产总值超过原计划4.2%，并成立了[[鞍钢|鞍钢]]{{r|biography|page=222-223}}。[[朝鲜战争|朝鲜战争]]期间，负责志愿军后勤保障工作，曾三次入朝与彭德怀协商，受彭德怀高度评价{{NoteTag|[[张明远|张明远]]的回忆录《我的回忆》中写到：彭德怀对东北在战争期间的后勤工作非常满意（他对中央军委总后勤部和中央财经委员会的工作颇为不满），他不止一次在志愿军干部中和会议上说过，志愿军胜利，主要是得到高岗和东北的大力支持，说若要论军功，“百分之六十至百分之七十应归功于后勤。” 彭总还说后勤工作主要靠两个麻子（指高岗和洪学智）！}}<ref>{{cite web |language = zh-hans |publisher = 凤凰网 |title = 彭德怀为何在志愿军会议上数次公开感谢高岗和东北？ |url = http://news.ifeng.com/a/20140910/41921490_3.shtml |author = 姬文波 |date = 2014-09-10 |accessdate =  |archive-url = https://web.archive.org/web/20180929181205/http://news.ifeng.com/a/20140910/41921490_3.shtml |archive-date = 2018-09-29 |dead-url = no }}</ref>。高岗还在东北率先开展[[三反运动|三反运动]]<ref>Spence, Jonathan D. The Search for Modern China. WW Norton & Company publishing.1991.ISBN 978-0-393-30780-1</ref>，其经验被毛泽东推向全国。1951年10月，高岗又兼任中央人民政府人民革命军事委员会副主席。

1952年高岗调任[[中央人民政府国家计划委员会|中央人民政府国家计划委员会]]任主席，1953年赴京。在高岗前后调京的还有[[邓小平|邓小平]]、[[饶漱石|饶漱石]]、[[邓子恢|邓子恢]]、[[习仲勋|习仲勋]]<ref>{{cite web |language = zh-hans |publisher = 凤凰网 |title = 回忆父亲邓子恢 |url = http://news.ifeng.com/history/1/200709/0929_335_243629.shtml |author = 邓毅生、邓瑞生、邓淮生、邓汉生、邓建生、邓小涟、邓小燕、邓小兰 |date = 2007-09-29 |accessdate =  |archive-url = https://web.archive.org/web/20141226203213/http://news.ifeng.com/history/1/200709/0929_335_243629.shtml |archive-date = 2014-12-26 |dead-url = no }}</ref>，随之有“[[五马进京|五马进京]]，一马当先”之说，而所谓“一马”即指高岗，可想其受倚重之势。作为中共根据地出身的干部，高岗对于长期在[[国民党统治区|国民党统治区]]（“白区”）工作的[[刘少奇|刘少奇]]、[[周恩来|周恩来]]等人的势力不满。

「[[高岗饶漱石事件|高饶事件]]的实质是他们（高岗、[[饶漱石|饶漱石]]等人）试图把[[刘少奇|刘少奇]]和[[周恩来|周恩来]]从中共的第二和第三的位置上拉下来。主要的目标是刘少奇。」{{r|Cambridge|page=88}}高岗夺权的关键因素是他对毛泽东态度的估计，「毛泽东在1953年初期与高岗的几次私下谈话中表示了这种（主要是经济建设和发展农业合作社方面）不满。不管毛泽东的意图是什么，高岗把这看成是一种信任他的信号和反对刘和周的机会。」<ref>Frederick C. Teiwes, 'Extract from Mao and his Lieutenants', in G. Benton (eds), Mao Zedong and the Chinese Revolution (Volume II) (New York, 2008), p. 93-94.</ref><ref>Frederick C. Teiwes, Politics At Mao's Court: Gao Gang and Party Factionalism in the Early 1950s (New York, 1990), p. 37.</ref>{{r|Cambridge|page=89}}

1953年，在[[全国财经会议|全国财经会议]]上，高岗等人以批[[薄一波|薄一波]]来对抗刘少奇、周恩来；而饶漱石则在[[第二次全国组织工作会议|第二次全国组织工作会议]]上以批[[安子文|安子文]]来对抗刘少奇、周恩来<ref>{{cite web |language = zh-hans |website = 中国共产党新闻网 |title = 第二次全国组织工作会议(1953年9月16日-10月27日) |url = http://dangshi.people.com.cn/GB/151935/176588/176595/10555546.html |author =  |date =  |accessdate = 2014-11-29 |archive-url = https://web.archive.org/web/20141206003327/http://dangshi.people.com.cn/GB/151935/176588/176595/10555546.html |archive-date = 2014-12-06 |dead-url = no }}</ref>。也就是在全国财经会议上，毛泽东开始注意高岗等人的举动{{r|biography|page=311}}。同年10月，高岗借休假之机到杭州拉拢[[林彪|林彪]]加入其阵营，后[[陈云|陈云]]奉毛泽东指示劝告林彪，林遂表示不再支持高岗<ref>{{cite book |author = [[中共中央文献研究室|中共中央文献研究室]] |title = 《陈云年谱（中卷）》 |year = 2000 |location = 北京 |publisher = [[中央文献出版社|中央文献出版社]] |ISBN = 9787507307870 |page = 192 }}</ref>。1953年12月24日政治局扩大会议上，毛泽东不点名批评高岗：“北京有两个司令部，一个是以我为首的司令部，就是刮阳风，烧阳火；一个是以别人为司令的司令部，就是刮阴风，烧阴火，一股地下水。究竟是政出一门，还是政出多门？”{{r|biography|page=333}}{{r|YCG|page=169}}

1954年2月6日，[[中共七届四中全会|中共七届四中全会]]上，[[陈云|陈云]]、周恩来发言暗批高岗搞“[[独立王国|独立王国]]”{{r|YCG|page=167}}。2月17日高岗用[[手枪|手枪]][[自杀|自杀]]未遂。1954年4月29日，高岗向中央交了《我的反省》。前一天，时任[[中共中央东北局|中共中央东北局]]第二副书记[[张秀山|张秀山]]、东北局第三副书记[[张明远|张明远]]、东北局组织部部长[[郭峰|郭峰]]、[[中央人民政府国家计划委员会|中央人民政府国家计划委员会]]秘书长[[马洪|马洪]]、[[中共中央东北局|中共中央东北局]]秘书长兼农村工作部部长[[赵德尊|赵德尊]]五人被撤销一切党内职务{{NoteTag|此五人被指为高岗的“五虎上将”。}}{{r|biography|page=389}}。1954年8月初，高岗试图触电自杀但未成功{{r|biography|page=362}}。8月17日，高岗服大量[[安眠药|安眠药]]自杀身亡。

1955年3月，高岗在[[中共七届五中全会|中共七届五中全会]]上被开除出中国共产党。官方对[[高饶事件|高饶事件]]做结论时，称之为“阴谋活动”和为加强个人权力而进行的“无原则”活动。高岗死后，葬于北京[[万安公墓|万安公墓]]。“[[文化大革命|文化大革命]]”開始時，高崗的墓碑被砸爛半截。

中國境內第一部公開出版的《高崗傳》在2011年5月問世，此書由戴茂林和趙曉光撰寫，陕西人民出版社出版<ref>{{cite web |url = http://dangshi.people.com.cn/GB/138903/14880178.html |title = 用最新史料解開歷史疑雲 實事求是寫《高崗傳》） |author = 劉景巍 |date = 2011年6月13日 |website = 中國共產黨新聞網 |language = zh-cn |accessdate = 2015年7月2日 |archive-url = https://web.archive.org/web/20150703181829/http://dangshi.people.com.cn/GB/138903/14880178.html |archive-date = 2015-07-03 |dead-url = no }}</ref>。

==家庭==
*[[杨芝芳|杨芝芳]] 1905年12月出生于[[陕西省|陕西省]][[米脂县|米脂县]]。1925年与高岗结婚，1938年与高岗解除婚约。当时陕甘宁边区政府民政厅厅长[[王子宜_(1907年)|王子宜]]要求杨芝芳跟高岗离婚。他告诉杨芝芳：高岗活动很多，经常出头露面，你是小脚，与高岗一起出面不太合适，最好与高岗分开。杨芝芳就同意了<ref>{{Cite book|title=高岗传|last=戴茂林，赵晓光著|first=|publisher=西安：陕西人民出版社|year=2011.04|isbn=978-7-224-09634-7|location=|pages=93}}</ref>。
*[[李力群|李力群]] 1919年12月26日出生于[[江苏省|江苏省]][[宿迁县|宿迁县]]。1940年元旦与高岗结婚。

== 评价 ==
=== 正面评价 ===
《[[高岗传|高岗传]]》评价高岗一生有五大功绩：参与创建了[[中国西北地区|西北]]的[[陕甘边区革命根据地|陕甘边区革命根据地]]；[[中國抗日戰爭|抗日战争]]时期连任三届[[陕甘宁边区|陕甘宁边区]]参议会议长；[[国共内战|国共内战]][[中共|中共]]成功夺取[[中国东北地区|东北]]立下功劳；[[新中国|新中国]]建国初主持东北的建设；[[抗美援朝|抗美援朝]]战争提供了[[后勤|后勤]]保障。<ref name="webHuanqiu ">{{cite web|url=http://history.huanqiu.com/people/2012-01/2372711.html|title=高岗的是非功过解读 一生五项功绩十大罪行|accessdate=2012-01-19|work=|publisher=环球时报|language=zh-hans|deadurl=yes|archiveurl=https://web.archive.org/web/20120129013149/http://history.huanqiu.com/people/2012-01/2372711.html|archivedate=2012年1月29日|df=}}</ref>

=== 负面评价 ===
1954年2月，[[中共中央|中共中央]]批判高、饶的座谈会上，[[周恩来|周恩来]]总结-{}-发言列举了高岗的十大罪行：“一是散布枪杆子上出党、红区白区论；二是进行宗派活动，反对中央领导同志；三是造谣挑拨，制造党内不和；四是实行派别性的干部政策，尤其是对干部私自许愿封官；五是把自己领导的地区看作个人资本和独立王国；六是假借中央名义，破坏中央威信；七是剽窃别人文稿，抬高自己，蒙蔽中央；八是在[[中苏关系|中苏关系]]上，拨弄是非，不利中苏团结；九是进行夺取党和国家权力的阴谋活动；十是私生活腐化”。<ref name="webHuanqiu " />周恩来于2月25日在座谈会上作了总结發言。这个总结-{}-发言的提纲，后来送毛泽东审阅过。“在长期的革命斗争中，高岗有其正确的有功于革命的一面，因而博得了党的信任，但他的[[个人主义|个人主义]]思想长期没有得到纠正和制止，并且在全国胜利后更大的发展了，这就是他的黑暗的一面。其实就是高岗用另一种更有利于国家人民的政治手段夺权的政治运动。以其失败告终。”<ref>[http://book.sina.com.cn/nzt/his/ggrsssjsm/9.shtml 高岗饶漱石事件始末] {{Wayback|url=http://book.sina.com.cn/nzt/his/ggrsssjsm/9.shtml |date=20080507043147 }}马畏安，当代中国出版社，新浪读书</ref>

据[[李锐_(1917年)|李锐]]女儿李南央回忆，李锐在[[解放战争|解放战争]]中短暂担任高岗秘书期间，发现高岗“乱搞女人”，甚至担心高岗将魔爪伸向李锐当时的妻子范元甄，因而对高岗人品产生怀疑而离职。{{fact}}
