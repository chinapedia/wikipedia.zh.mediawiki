'''高建''' （{{lang-ko|'''고건'''}}，{{bd|1938年|1月2日|catIdx=Goh, Kun}}），[[韩国|韩国]]政治家，曾两次出任[[大韩民国国务总理|大韩民国国务总理]]和[[首尔市|首尔市]][[首尔市长列表|市长]]。他擅长行政管理，为政清廉，是韩国口碑甚佳的行政高官{{r|dbw}}。1975年，高建当选韩国[[全罗南道|全罗南道]]知事，成为韩国历史上最年轻的道[[知事|知事]]{{r|新华}}。之后，他历任韩国[[大韩民国行政安全部|内政部]]部长、[[大韩民国國土交通部|交通部]]部长、[[大韩民国农林水产食品部|农林水产部]]部长、[[首尔特别市|首尔特别市]]市长、[[明知大学|明知大学]]校长{{r|腾讯网|网易}}。1997年3月，时任韩国总统[[金泳三|金泳三]]任命高建为[[大韩民国国务总理|国务总理]]。{{r|范永红|page=66}}2003年1月26日，高建接受了[[卢武铉|卢武铉]]的任命，再次成为韩国总理{{r|范永红|page=67}}。2004年，在时任[[韩国总统|韩国总统]][[卢武铉|卢武铉]]遭到[[大韩民国国会|国会]][[弹劾|弹劾]]期间，高建代理行使总统权力2个月。在卢武铉被恢复权力后，高建辞职退休。{{r|范永红|page=139}}<ref>{{cite web |url=http://news.sina.com.cn/w/2004-03-12/11263015996.shtml |title=韩国会表决通过弹劾案 总理高建将暂行总统权 |publisher=新浪网 |date=2004-03-12}}</ref>2006年8月28日，高建曾經跟在野黨議員合組政黨“希望連帶”，但在2007年1月16日在表明不會出選總統的同時，宣布在政界引退{{r|dbw}}。

高建是韩国政坛难得的“清廉之士”，曾被推举为韩国“反腐败国民联合会”会长。2001年，[[透明国际|透明国际]]马来西亚分会授予了他“世界廉正奖章”（Global Integrity Medal）。{{r|范永红|新华|网易|page=67}}<ref name=KT>{{cite news|url=http://www.highbeam.com/doc/1G1-72492309.html|title=Mayor Goh Receives Global Integrity Medal|publisher=Korea Times|date=2001-03-31}}</ref>

==生平==
高建1938年1月2日出生于[[韩国|韩国]][[首尔|首尔]]。[[首尔大学|首尔大学]]文理学院政治系毕业后，高建又继续在环境学院研修城市规划，后赴[[美国|美国]][[哈佛大学|哈佛大学]]和[[麻省理工学院|麻省理工学院]]留学。留学归国后，高建一心从政，考取了国家特等公务员行政科。{{r|范永红|page=139}}1975年至1979年，高建出任韩国[[全罗南道|全罗南道]]知事，成为韩国历史上最年轻的道知事{{r|新华}}。由于他管理有方，为政清廉，自[[朴正熙|朴正熙]]之后的6位总统都很看重他，其中5位将他纳入内阁委以重任。他历任内政部长、交通部长、农林水产部长、[[首尔市|首尔市]]市长、[[明知大学|明知大学]]校长{{r|腾讯网|网易}}。1997年3月，时任韩国总统[[金泳三|金泳三]]任命高建为[[大韩民国国务总理|国务总理]]。[[金大中|金大中]]执政期间，虽然没有将高建拉入内阁，但迫于“朝小野大”的局势，也让高建出面提名内阁成员，使他成为“不挂名的总理”。1998年7月至2002年6月，高建再次出任首尔市长一职。{{r|范永红|page=66}}2003年1月26日，高建接受了[[卢武铉|卢武铉]]的任命，再次成为韩国总理{{r|范永红|page=67}}。在卢武铉遭到国会弹劾期间，高建代理行使总统权力2个月，在卢武铉被恢复权力后辞职退休。{{r|范永红|page=139}}

高建两次出任[[首尔市|首尔市]]长时，首尔分别成功举办了[[1988年奥运会|1988年奥运会]]和[[2002年世界杯足球赛|2002年世界杯足球赛]]。在他担任首尔市长期间，他把昔日的垃圾山打造成了“环保绿色公园”，并建起了“首尔上岩洞世界杯主体育场”，此举备受首尔市民称赞。{{r|范永红||新华|page=66}} 1998年，高建推出“网上民事申请程序强化系统”，用于提高市政府的办事效率和透明度，杜绝索贿受贿，使首尔市政公务员的腐败现象大为减少。首尔民众对此赞不绝口。由于他的廉政作风深得人心，他被推举为韩国“反腐败国民联合会”会长。2001年，[[透明国际|透明国际]]马来西亚分会授予了他“世界廉正奖章”（Global Integrity Medal)。{{r|范永红|新华|网易|page=67|KT}}

2004年3月13日，时任[[韩国总统|韩国总统]][[卢武铉|卢武铉]]遭到国会的[[弹劾|弹劾]]后，高建代理卢武铉行使总统职能两个月。在韩国政坛历史上的首次总统弹劾风暴中，高建稳健地保持了韩国政府各个部门和各项政策的平稳延续，将弹劾风暴的负面效应降到最低。{{r|范永红|page=67}}[[韩国宪法法院|韩国宪法法院]]驳回国会对卢武铉的弹劾案后，事先就已提出辞职的高建辞职退休。他原本打算去美国继续深造，但又因其99岁高龄的父亲健康问题而改变了初衷。{{r|范永红|page=139}}<ref>{{cite web |url=http://news.bbc.co.uk/chinese/simp/hi/newsid_3740000/newsid_3745100/3745157.stm |title=韩国总统接受总理高建辞呈 |publisher=BBC |date=2004-05-25}}</ref>

2015年10月19日，高建与[[崇实大学|崇实大学]]校长韩献洙等7人出访了[[朝鲜|朝鲜]]，并出席在[[平壤科学技术大学|平壤科学技术大学]]举行的国际学术大会<ref>{{cite web |url=http://news.sina.com.cn/o/2015-10-19/doc-ifxivsee8740230.shtml |title=韩前国务总理等7人出访朝鲜 出席国际学术大会 |publisher=新浪网 |date=2015-10-19}}</ref><ref>{{cite web |url=http://www.chinanews.com/gj/2015/10-19/7576856.shtml |title=韩前国务总理等7人出访朝鲜 出席国际学术大会 |publisher=中国新闻网 |date=2015-10-19}}</ref>。
