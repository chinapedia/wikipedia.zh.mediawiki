'''高雄无障碍计程车'''，是[[高雄市|高雄市]]為提供身心障礙者更多元搭乘選擇及疏解[[復康巴士|復康巴士]]的運能需求，於2013年2月向[[中華民國交通部|交通部]]爭取補助購置的20輛無障礙[[計程車|計程車]]。高雄無障礙計程車採用[[德國|德國]]名為CADDY IPC的福祉車，是以CADDY Maxi 1.6TDI為基礎，並於第三排導入德國AMF-BRUNS公司所生產的內部[[輪椅|輪椅]]斜坡套件，讓後方空間變大成為無障礙空間，輪椅可直接上車，不須折疊收起。首批20輛無障礙計程車，在交通局評選後由[[凱旋大都會車隊|凱旋大都會車隊]]獲得經營權，並於2013年8月正式啟用，並設立無障礙叫車專線07-7241111

== 概況 ==
目前高雄市身心障礙者人數約13.2萬，為提供更多元搭乘選擇，及疏解高雄市復康巴士的運能需求，高雄市政府於今年2月向交通部爭取補助購置20輛無障礙計程車，並於2013年8月率先推出10輛無障礙計程車上路，預估11月即可達20輛規模，並定下2014年6月達成40輛的目標。

== 特色 ==
高雄無障礙計程車有三大特色：
# 與各大醫療院所合作設置臨停專區或免費路外停車服務，交通局也於路邊首度規劃「無障礙計程車專用格位」，未來將以[[大同醫院|大同]]及[[小港醫院|小港醫院]]作為示範區，達成無縫就醫的目標。
# 將無障礙計程車隊納入高雄市觀光計程車隊中，家裡有不便於行的老人或身障團體，都可闔家或攜伴同遊，發揮就醫及觀光的全方位服務。
# 預訂不到復康巴士的市民，可透過復康巴士叫車中心轉介，提供無障礙計程車的多元選擇。

== 收費及叫車方式 ==
高雄無障礙計程車24小時預約及一般叫車專線電話為07-7241111，收費方式與一般計程車按錶計費方式相同，但若持高雄社會局博愛卡者將享有段次補貼，100元以下抵扣1段次（18元），101元-200元抵扣2段次（36元），201元以上抵扣3段次（54元）。<ref> [http://www.mtaxi.com.tw/Event.asp?id=2 大都會計程車與微信支付合作，全台9500臺計程車可使用]</ref>
