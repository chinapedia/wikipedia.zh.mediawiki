'''高锰酸钾'''（化学式：[[钾|K]][[锰|Mn]][[氧|O]]<sub>4</sub>），强[[氧化剂|氧化剂]]，紫红色晶体，可溶于[[水|水]]，遇[[乙醇|乙醇]]即被还原。常用作[[消毒剂|消毒剂]]、[[水净化剂|水净化剂]]、[[氧化剂|氧化剂]]、[[漂白剂|漂白剂]]、[[毒气吸收剂|毒气吸收剂]]、[[二氧化碳|二氧化碳]]精制剂等。1659年被[[約翰·格勞勃|约翰·格劳勃]]發現。醫療上有用作清潔消毒，和用來消滅[[真菌|真菌]]之用。

==来源==
由於實驗室的大量需求，高錳酸鉀被大規模地製造。一般常見的製備方法有以下兩種：

礦石中取得的[[二氧化錳|二氧化錳]]和[[氢氧化钾|氢氧化钾]]在空氣中或混合[[硝酸鉀|硝酸鉀]]（提供氧氣）加熱，產生[[錳酸鉀|錳酸鉀]]，再於鹼性溶液中與氧化劑進行[[電解|電解]]氧化得到高錳酸鉀。
:<chem>2 MnO2 + 4OH- +O2 -> 2MnO4^2- +2H2O</chem>
:<chem>2 MnO4^2- + Cl2 -> 2MnO4- + 2Cl-</chem>

或者也可以通过Mn<sup>2+</sup>和[[二氧化鉛|二氧化鉛]]（PbO<sub>2</sub>）或[[鉍酸鈉|鉍酸鈉]]（NaBiO<sub>3</sub>）等強氧化劑的反應產生。此反應也用於檢驗Mn<sup>2+</sup>的存在，因為高錳酸鉀的顏色明顯。

==化學性质==
===对热不稳定===
高锰酸钾加热可以分解，生成锰酸钾、二氧化锰和氧气
:<math>\rm 2KMnO_4\xrightarrow{\triangle} K_2MnO_4+MnO_2+O_2\uparrow</math>

这是实验室制取氧气的方法之一。加热时如果温度更高，生成的锰酸钾则可以继续分解。

也有文献指出，高锰酸钾的分解过程不会产生二氧化锰，而只有K<sub>2</sub>MnO<sub>4</sub>、K<sub>2</sub>Mn<sub>4</sub>O<sub>8</sub>和KMnO<sub>2</sub>：<ref>黄晗达 李友银. 高锰酸钾加热分解过程研究[J]. 《化学教育》. 2009年第2期. P<sub>65-66</sub></ref>
:<math>\rm 6KMnO_4 \xrightarrow{\triangle} K_2MnO_4+K_2Mn_4O_8+4O_2\uparrow</math>
:<math>\rm KMnO_4 \xrightarrow{\triangle} KMnO_2+O_2\uparrow</math>

另有实验得出不同的分解历程：<ref>谭荣高 陈国强. 高锰酸钾热分解历程的研究[J]. 《北光通讯》. 1991年. P<sub>34~38</sub></ref>
:<math>\rm 4KMnO_4\xrightarrow{290^{\circ}C}K_2MnO_4+K_2O\cdot 3MnO_2+\frac{5}{2}O_2\uparrow</math>
:<math>\rm 3K_2MnO_4\xrightarrow{600\sim 650^{\circ}C}3K_3MnO_4+MnO_2+O_2\uparrow</math>

===强氧化性===
過锰酸钾有强氧化性，可以在常温下氧化盐酸，这也是实验室制取[[氯气|氯气]]的方法之一：

2KMnO<sub>4</sub>+16HCl → 2KCl+2MnCl<sub>2</sub>+5Cl<sub>2</sub>↑+8H<sub>2</sub>O

過锰酸钾也可以被亚硫酸钠还原，得到不稳定的亮蓝色的五价锰盐（MnO<sub>4</sub><sup>3-</sup>）<ref name=wj>《无机化学丛书》.第九卷 锰分族 铁系 铂系. 谢高阳 等. 科学出版社. 2.8.2 過锰酸盐. P<sub>39-40</sub></ref>，極容易再被亞硫酸鈉還原產生更低價的錳化合物。

過锰酸钾与[[过氧化氢|过氧化氢]]会剧烈反应，生成[[氧气|氧气]]；在酸性环境下则会有锰盐生成：

<Math>\rm 2KMnO_4 + 3H_2O_2 {=\!=\!=\!=} 2KOH + 2MnO_2 + 2H_2O + 3O_2 \uparrow </Math>

<Math>\rm 2KMnO_4 + 5H_2O_2 + 6H^+ {=\!=\!=\!=} 2K^+ + 2Mn^{2+} + 8H_2O + 5O_2 \uparrow </Math>

過锰酸钾氧化Mn<sup>2+</sup>生成MnO<sub>2</sub>沉淀。但有焦磷酸存在时，则按下式反应：
: 4 Mn<sup>2+</sup> + MnO<sub>4</sub><sup>−</sup> + 8 H<sup>+</sup> + 15 H<sub>2</sub>P<sub>2</sub>O<sub>7</sub><sup>2-</sup> → 5 [Mn(H<sub>2</sub>P<sub>2</sub>O<sub>7</sub>)<sub>3</sub>]<sup>3-</sup> + 4 H<sub>2</sub>O

利用其氧化性的特点，可用具有[[还原性|还原性]]的[[草酸|草酸]]、[[维生素C|维生素C]]或[[盐酸羟胺|盐酸羟胺]]等的溶液来去除過锰酸钾造成的污渍。

另外，過锰酸钾可以和[[红磷|红磷]]、[[镁|镁]]粉、[[铝|铝]]粉、[[硼氢化钠|硼氢化钠]]等还原剂形成爆炸性的混合物，摩擦或撞击即可爆炸。因此不可将過锰酸钾和它们混合在一块。

===其他===
高锰酸钾可以和硫酸铝反应，产生[[過锰酸铝|過锰酸铝]]和[[明矾|明矾]]：<ref name=wj />
:6 KMnO<sub>4</sub> + 4 Al<sub>2</sub>(SO<sub>4</sub>)<sub>3</sub>·18H<sub>2</sub>O + 6H<sub>2</sub>O → 2 Al(MnO<sub>4</sub>)<sub>3</sub> + 6 KAl(SO<sub>4</sub>)<sub>2</sub>·12H<sub>2</sub>O

在不同的pH下，過錳酸根被還原的產物也有所不同：強鹼性下生成墨綠色的錳酸根MnO<sub>4</sub><sup>2-</sup>，在中性和弱鹼性下生成棕黑色的MnO<sub>2</sub>水合物，而在強酸性下則生成無色的Mn<sup>2+</sup>。

== 安全性 ==
高锰酸钾是一种强氧化剂，所以应使其与可被氧化的物质隔开存放。高锰酸钾与浓[[硫酸|硫酸]]反应生成易爆物[[七氧化二锰|七氧化二锰]]。高锰酸钾固体与纯[[甘油|甘油]]或一些醇反应会剧烈燃烧。
: <Math>3{\rm C}_{3}{\rm H}_{5}{\rm (OH)}_{3} + 14{\rm KMnO}_{4} =\!=\!=\!= 14{\rm MnO}_{2} + 7{\rm K}_{2}{\rm CO}_{3} + 2{\rm CO}_{2} \uparrow + 12{\rm H}_{2}{\rm O}</Math>
高锰酸钾留下的褐色污渍与[[亚硫酸氢钠|亚硫酸氢钠]]或[[草酸|草酸]]反应后会褪色。

常温下即可和[[甘油|甘油]]等有机物反应甚至燃烧（但有时与甘油混合后反应极为缓慢，甚至感受不到温度的升高，其原因尚不明确）；在酸性环境下氧化性更强，能氧化负价态的氯、溴、碘、硫等离子及二氧化硫等。与皮肤接触可腐蚀皮肤产生棕色染色，数日不褪（可以使用维生素C或草酸溶液洗去）；粉末散布于空气中有强烈刺激性，可使人连打喷嚏。[[尿液|尿液]]、[[二氧化硫|二氧化硫]]等可使其褪色。与较活泼金属粉末混合后有强烈燃烧性，危险。该物质在加热时分解成[[氧气|氧气]]、[[二氧化锰|二氧化锰]]、[[锰酸钾|锰酸钾]]（K<sub>2</sub>MnO<sub>4</sub>）；在中性溶液中还原产物一般为[[二氧化锰|二氧化锰]]。另外，该物质的溶液在碱性环境下不稳定，容易生成墨绿色的[[锰酸钾|锰酸钾]]（K<sub>2</sub>MnO<sub>4</sub>），加入酸又能生成高锰酸钾。

== 應用 ==
=== 顯色劑 ===
在實驗室中，常作為顯色劑，用於薄層層析的顯色。
=== 淨化空氣 ===
一些空氣清淨機會有CPZ濾網，用來吸收有害氣體。

CPZ濾網的成分為過錳酸鉀、[[沸石|沸石]]、[[活性炭|活性炭]]。

=== 滴定 ===
[[維生素C|維生素C]]的水溶液能使過錳酸鉀溶液褪色，并且維生素C溶液越濃，用量就越少。根據這一特性，就能夠用高锰酸钾測定蔬菜或水果中的維生素含量。

此外，在[[分析化学|分析化学]]中，高锰酸钾还能用来滴定一些还原性物质，来测定其含量。
=== 工业生产 ===
在化学工业中用于生产[[维生素C|维生素C]]、[[糖精|糖精]]等。在轻化工业中用作[[纤维|纤维]]、[[油脂|油脂]]的[[漂白|漂白]]和脱色。
=== 消毒杀菌 ===
高锰酸钾具有强氧化性，其溶液用于消毒杀菌。如稀释一定比例后，可以作用于人体，治疗某些炎症。
==储存==
高锰酸钾应避光、干燥保存。

其溶液不稳定，缓慢分解并放出氧气。在酸性或碱性溶液中稳定性更差。
