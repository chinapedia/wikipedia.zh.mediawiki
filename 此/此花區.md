'''此花區'''（{{jpn|j=此花区|hg=このはなく|rm=Konohana ku|f=y}}）是[[日本|日本]][[大阪府|大阪府]][[大阪市|大阪市]]的24區之一，位於大阪市的西部，[[淀川|淀川]]出海口和{{Tsl|ja|旧淀川|舊淀川||安治川}}之間，西側臨[[大阪灣|大阪灣]]，轄區包括位於大阪灣上的[[人工岛|人工岛]]{{tsl|ja|舞洲}}及{{tsl|ja|夢洲}}。

區名「此花」取自[[和歌|和歌]]{{Tsl|ja|難波津 (和歌)|難波津 (和歌)|難波津}}。<ref>{{Cite web| url=http://www.city.osaka.lg.jp/konohana/page/0000001529.html| title=此花区の町名の由来| date=2014-10-31| publisher=大阪市此花區公所| accessdate=2017-10-20|language=ja}}</ref>

== 歷史 ==
過去為[[淀川|淀川]]河口區域的許多[[河岛|河岛]]，曾有許多漁村。[[江戶時代|江戶時代]]開始以[[围垦|围垦]]方式取得更多的土地，因此有些地名便是以當時開發者的姓名來命名。<ref name="Brief">{{Cite web| url=http://www.city.osaka.lg.jp/konohana/page/0000001552.html| title=此花区の概要| date=2017-03-22| publisher=大阪市此花區公所| accessdate=2017-10-20|language=ja}}</ref>1620年，幕府曾在此設立最早的船奉行所，負責管理大阪以西地區經由船法川進入大阪市區的船隻。

現在的此花區範圍在1889年實施町村制屬於[[西成郡|西成郡]]野田村、稗島村、{{tsl|ja|川北村 (大阪府)|川北村 (大阪府)|川北村}}和傳法村4個行政區劃。1897年野田村、川北村和傳法村的部分地區被併入大阪市，最初劃入[[北區_(大阪市)|北區]]和[[西區_(大阪市)|西區]]，此後在1898年至1910年期間由於挖掘[[淀川|新淀川]]讓淀川從此花區的北側直接通過進入大阪灣，此花區許多土地因此成為淀川的河道。同時以[[住友財閥|住友財閥]]為主在此興建了大量的工廠。

1925年先前併入大阪市的區域加上原屬{{tsl|ja|野田村 (大阪府西成郡)|野田村 (大阪府西成郡)|野田村}}、下福島村、上福島村的區域被劃出設立為'''此花區'''<ref name="Brief"/>；同年川北村和傳法町其他未在1897年併入大阪市的部分也被併入大阪市，但此次併入的區域在當時是被劃入[[西淀川區|西淀川區]]。

1927年位於新淀川河口與舊淀川河口之間的海埔新生地完成，[[住友電氣工業|住友電氣工業]]、[[住友金屬工業|住友金屬工業]]、[[住友化學|住友化學]]、{{Tsl|ja|日立造船}}、{{tsl|ja|大阪ガス|大阪瓦斯}}、[[汽車製造|汽車製造]]的大型工廠也隨之完成。

1943年過去原屬野田村、下福島村、上福島村的區域被劃出新設為[[福島區|福島區]]，同時位於淀川以南，屬於西淀川區的舊船法町和川北村部分區域被劃入此花區，也形成現在的此花區轄區範圍。

在第二次世界大戰期間，由於為重工業區域，曾遭到[[美軍|美軍]]密集的轟炸；在戰爭結束後，也由於公害因素讓工廠開始外移，工廠遷移後閒置的空地有部分成為在2001年開幕的[[日本環球影城|日本環球影城]]。

== 交通 ==
[[西日本旅客鐵道|西日本旅客鐵道]][[大阪環狀線|大阪環狀線]]通過轄內東部，設有[[西九條車站|西九條車站]]，西九條車站同時也是通往[[日本環球影城|日本環球影城]]的[[櫻島線|櫻島線]]的轉乘車站，櫻島線在此往西穿過整個此花區，沿線設有[[安治川口車站|安治川口車站]]、[[環球城車站|環球城車站]]、[[櫻島車站|櫻島車站]]。

[[阪神電氣鐵道|阪神電氣鐵道]][[阪神難波線|阪神難波線]]同樣也通過西九條車站，除了西九條車站，在此花區轄內還設有{{link-ja|千鳥橋車站|千鳥橋駅}}和{{link-ja|傳法車站|伝法駅}}。
