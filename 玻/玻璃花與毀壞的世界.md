'''玻璃花與毀壞的世界'''（{{lang-ja|ガラスの花と壊す世界}}，{{lang-en|GARAKOWA -Restore the World-}}<ref>{{Cite web |url=http://garakowa.us/ |title=美國版官方網站 |accessdate=2015-02-04}}</ref>）是一部[[日本|日本]][[動畫電影|動畫電影]]，於2016年1月9日上映。

== 概要 ==
本作是以2013年由波麗佳音主辦的「動畫化大獎 Powered by 波麗佳音」的大獎作品《D.backup》為原案進行改編<ref>{{Cite web |url=http://www.animen.com.tw/NewsArea/NewsItemDetail?NewsId=8351&categoryId=600&tagName=%E7%8E%BB%E7%92%83%E8%8A%B1%E8%88%87%E6%AF%80%E5%A3%9E%E7%9A%84%E4%B8%96%E7%95%8C&realCategoryId=1&subCategoryId=3 |title=劇場版動畫《玻璃花與毀壞的世界》發表最新劇情概要、主視覺圖及角色介紹！ |work=Animen |accessdate=2016-02-04}}</ref>。

本作在公開上映前，曾於2015年10月22日至31日舉辦的第28屆[[東京國際電影節|東京國際電影節]]上先行上映<ref>{{Cite web |url=http://moca-news.net/article/20150919/2015091912000a_/01/?afid=zeroes |title=1月9日（土）公開の劇場版アニメ『ガラスの花と壊す世界』、追加キャストとして茅野愛衣を発表。「第28回 東京国際映画祭」での上映も決定  |work=moca |accessdate=2015-02-04 |language=ja}}</ref>。

為莉茉配音的花守由美里，是第一次出演動畫電影。

== 故事簡介 ==
在名為「知識之箱」的空間內，存在著不同時間的世界。名為杜兒及桃樂絲的兩名少女，肩負著對抗敵人「病毒」及刪除被「病毒」感染的世界的重任。

某日，兩人因感受到有新的病毒誕生而前往消滅，但在過程中卻發現一名被病毒襲擊的少女。在救出少女且等待其清醒後，她們詢問其的身份及來自何處，可惜少女喪失了記憶，只記起自己名為「莉茉」及「我必須回到花田」這一句話……
