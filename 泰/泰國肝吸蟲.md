'''泰國肝吸蟲'''（[[双名法|學名]]：''Opisthorchis viverrini''），又稱“'''香貓肝吸蟲'''”<ref>不是[[貓肝吸蟲|貓肝吸蟲]]。</ref>，是一種可[[寄生|寄生]]在[[肉食性|肉食性]]動物[[膽管|膽管]]的[[吸蟲|吸蟲]]，其生活史與[[中華肝吸蟲|中華肝吸蟲]]極為相似，以[[膽汁|膽汁]]為食，可造成[[肝|肝]]、[[膽|膽]]病變，主要分布在[[泰國|泰北]]、[[寮國|寮國]]一帶，當地約有八、九成鄉村人口以及一半的城市人口受到感染，估計約有九百萬人受到影響。

== 形態 ==
[[File:Opisthorchis_viverrini_egg.png|200px]]
成蟲尺寸介於長5至10毫米、寬1至2毫米的範圍，體型狹長而扁平，蟲體一端具有口吸盤，其腸管自此延蟲體兩側分叉延伸至尾端，在前端約五分之一界線處，另有一個與口吸盤約略等大的腹吸盤。此外泰國肝吸蟲雌雄同體，一對[[睪丸|睪丸]]呈現圓形花瓣狀是此寄生蟲的特色，分布在口吸盤的另一端，並佔蟲體後端三分之一的區域，[[卵巢|卵巢]]則位於中後段三分之一線上，[[子宮|子宮]]則在中段盤繞，佔全長約三分之一，兩旁可見網點狀的卵黃腺。

== 生活史 ==
泰國肝吸蟲的[[生活史|生活史]]與[[中華肝吸蟲|中華肝吸蟲]]相似，蟲卵必須在[[淡水螺|淡水螺]]中孵化，經過四個發育階段後離開螺體，並尋找[[鯉科|鯉科]]的[[淡水魚|淡水魚]]作為第二中間宿主，在魚鱗、皮膚、肌肉組織發育後，透過其他動物生吃、未完全熟食這些魚，幼蟲可在食用者體內脫去外囊，經過四週發育成蟲，可在宿主體內活上40年之久，每日約產下1000至2500顆卵，可隨[[膽汁|膽汁]]一同進入[[消化道|消化道]]，最後隨糞便排出，是診斷的重要依據。

== 臨床表現 ==
多數感染者不產生症狀，或是僅有如[[消化不良|消化不良]]（dyspepsia）、[[腹痛|腹痛]]、[[腹瀉|腹瀉]]、或[[便秘|便秘]]的症狀輕微；但長期下來，會發展成較為劇烈的[[感染症|感染症]]，像是[[肝腫大|肝腫大]]（hepatomegaly）或出現[[膽管炎|膽管炎]]（cholangitis）、[[肉芽腫性膽囊炎|肉芽腫性膽囊炎]]（cholecystitis）、甚至引發[[膽囊上皮細胞癌|膽囊上皮細胞癌]]（chlolangiocarcinoma），嚴重時甚可致命。

== 診斷和治療 ==
診斷時主要依據[[光學顯微鏡|光學顯微鏡]]對排泄物中的蟲卵鑑定，但是泰國肝吸蟲蟲卵亦與[[中華肝吸蟲|中華肝吸蟲]]相似，因此顯微鏡下不易區分，因此可透過問診比對盛行區域，或進一步使用[[PCR|PCR]]、[[ELISA|ELISA]]等技術做進一步分析，由於[[病理|病理]]機制相似，使用的藥物也是[[Praziquantel|Praziquantel]]，因此無法確認物種時，仍可以相同的方式治療。
