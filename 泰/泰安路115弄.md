'''泰安路115弄'''是位于[[中國|中國]][[上海|上海]][[长宁区|长宁区]]的一片[[里弄#花园里弄|花园里弄]]，共由八幢花园别墅组成。里弄由美商[[德士古洋行|德士古洋行]]于1948年委托黄迈士设计，大陆工程有限公司施工建成<ref name="优秀">上海优秀历史建筑.</ref>。住宅分英式和西班牙式两种，最初曾作为洋行所属的高级职员住宅使用<ref name="新民晚报">{{Cite web |title = 泰安路115弄花园“缩水” 洋房“变脸” |url = http://www.news365.com.cn/wxpd/sh/cn/200905/t20090522_2330778.htm |publisher = [[新民晚报|新民晚报]] |date = 2009-05-22 |accessdate = 2014-12-28 }}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

1989年9月，泰安路115弄花园里弄住宅成为第一批[[上海市优秀历史建筑|上海市优秀历史建筑]]以及[[上海市文物保护单位|上海市文物保护单位]]。历史学家[[周谷城|周谷城]]曾长期居住于此弄6号。

== 建筑特色 ==
弄内共有别墅三排八幢，占地面积1.39万平方米，建筑面积5062平方米。其中英式别墅立面部分木构架外露漆绛红色，山墙有凸窗而东面山墙则凸出壁炉和烟囱，有小阳台并呈挑出式<ref name="优秀"/>。西班牙式别墅屋面呈缓坡式，盖以圆筒瓦，窗间有绞绳纹小柱<ref name="优秀"/>。每幢住宅的南面均有大花园。
