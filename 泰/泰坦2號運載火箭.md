'''泰坦二號運載火箭'''原為洲際飛彈，後由[[格伦·L·马丁公司|格伦·L·马丁公司]]改良，[[泰坦一號運載火箭|泰坦一號運載火箭]]也是如此。

== 任務 ==

大力神II型洲际导弹為獨創性的[[長程洲際彈道飛彈|長程洲際彈道飛彈]]（[[ICBM|ICBM]]），以前為[[美國空軍|美國空軍]]（[[美國太空總署|美國太空總署]]及[[美國國家海洋和大氣管理局|美國國家海洋和大氣管理局]]所屬）的中型[[火箭|火箭]]，所發射[[衛星|衛星]]包含USAF防禦飛[[氣象衛星|氣象衛星]]（DMSP）RU6[[美國國家海洋和大氣管理局|美國國家海洋和大氣管理局]]所屬的[[大氣衛星|大氣衛星]]，改良後的泰坦二號火箭由[[加利福尼亞洲|加利福尼亞洲]]的[[范登堡|范登堡]][[空軍|空軍]]基地發射至2003年止。

== 特徵 ==
泰坦二號火箭為兩節式[[液態火箭|液態火箭]]，設計用來發射中小型[[衛星|衛星]]，他發射4200磅（1900公斤）的[[衛星|衛星]]至[[低地球軌道|低地球軌道]]，第一節由兩種液態燃料組成[[發動機|發動機]]為飛行噴射（Aerojet）LR87型；第二節則是飛行噴射（Aerojet）LR91型。

== 演變 ==
[[File:Gemini-Titan_11_Launch_-_GPN-2000-001020.jpg|thumb]]，在1966年9月12日]]
泰坦系列在1955年10月就成立了，[[公司|公司]]名稱為[[馬丁公司|馬丁公司]]，並贏得[[美國太空總署|美國太空總署]]的合約，所以[[馬丁公司|馬丁公司]]開始建造[[長程洲際彈道飛彈|長程洲際彈道飛彈]]（[[ICBM|ICBM]]）那便是[[泰坦一號火箭|大力神I型洲际导弹]]，它是第一台有兩節式的[[長程洲際彈道飛彈|長程洲際彈道飛彈]]（[[ICBM|ICBM]]）及地下隱藏的[[發射台|發射台]]。[[泰坦一號火箭|大力神I型洲际导弹]]存在一些不足：因使用液氧，液氧必须低温保存，并会在存储期间蒸发，所以[[泰坦一號火箭|大力神I型洲际导弹]]以3座发射井共用同一地下液氧制造厂和指挥所部署；[[泰坦一號火箭|大力神I型洲际导弹]]不能直接从发射井发射，须先以升降平台整个推出发射井口，于露天才可加注燃料及点火发射。

[[美國太空總署|美國太空總署]]便多給了一個改良的機會，可以酬載更大的[[彈頭|彈頭]]也可以提高命中率並減少發射所需的時間。[[馬丁公司|馬丁公司]]收到一份契約書，因此設計了SM-68B引擎，為泰坦二號運載火箭之引擎，在1960年6月，泰坦二號運載火箭比[[泰坦一號運載火箭|泰坦一號運載火箭]]重了百分之五十，而第一節比[[泰坦一號運載火箭|泰坦一號運載火箭]]長且第二節比[[泰坦一號運載火箭|泰坦一號運載火箭]]的[[直徑|直徑]]寬，泰坦二號運載火箭也用穩定性較高的[[四氧化二氮|四氧化二氮]]/[[聯氨|聯氨]]之推進劑，可以在常温下长期保存，免去了[[泰坦一號運載火箭|泰坦一號運載火箭]]所需的液氧制造厂，但其缺點為具强腐蚀性和自燃現象，所以只要有任何一地破洞漏氣，即可能發生爆炸。而[[泰坦二號運載火箭|泰坦二號運載火箭]]的发射井也做了改变，增设了耐火壁和溢流槽，可在发射井中加注燃料并发射，也免去了用升降机推出井口的时间。但与同样使用[[四氧化二氮|四氧化二氮]]/[[聯氨|聯氨]]推進劑的苏联[[R-36|R-36]]撒旦不同，[[泰坦二號運載火箭|泰坦二號運載火箭]]的燃料罐防腐蚀工艺一般，不能长期存贮燃料，所以实用中燃料仍需在發射前加注，与[[泰坦一號運載火箭|泰坦一號運載火箭]]相同。

第一次發射泰坦二號運載火箭的時候是1960年12月,而與泰坦二號運載火箭較相似的是[[LGM-25|LGM-25]],且較具發展潛力，泰坦二號運載火箭部署核子武器則要等到1963年10月,其所搭載的為W-53核子彈頭，可飛行達15000[[公里|公里]](9325[[哩|哩]])，爆炸威力約等於900萬[[噸|噸]][[黃色炸藥|黃色炸藥]]，而且具有導行之功能，精確瞄準目標。泰坦二號運載火箭之後由[[美國太空總署|美國太空總署]]接手，載1960年代發射十次[[双子星座计划|雙子星座]]號宇宙飞船，同一時期也有運載更強的核子武器，約等同3500萬噸黃色炸藥。

== 退役 ==
泰坦二號運載火箭發射系統從1982年7月逐漸減量，至1987年6月完成，全面停止使用，儲存於亞利桑那洲的戴維斯----馬森(Davis Monthan)AFB倉庫，在此之前，泰坦二號運載火箭是最強大的後備部署核子火箭。

== 歷史 ==
'''泰坦二號運載火箭'''從1963年發射至1987年，原先63枚部署在[[范登堡空軍基地|范登堡空軍基地]],1978年8月24日，[[氧化劑|氧化劑]]因漏出導致兩名工人喪身，另一次發生在1980年9月19日,燃料滲出導致一名太空人死亡，而且發射臺也損毀。
