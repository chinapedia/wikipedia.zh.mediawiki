'''泰豐輪胎股份有限公司'''為[[臺灣|臺灣]]的[[輪胎|輪胎]]製造商，成立於1955年11月23日，主要產品為[[載貨汽車|卡車]]、[[客車|客車]]、[[轎車|轎車]]、輕型卡車等車型用的輻射層輪胎。該公司與[[日本|日本]][[普利司通|普利司通]]、[[住友橡膠|住友橡膠]]等公司擁有密切的技術合作關係。其品牌計有「FEDERAL飛德勒輪胎」、「HERO英雄輪胎」、「MAXON美克森」等；若以年度銷售額計算，2014年該公司在全世界排行第62名<ref>參看[http://big5.askci.com/chanye/2014/09/11/14132freh.shtml 中商情報網：2014年度全球輪胎75強排行榜出爐]。</ref>。

== 公司沿革 ==
[[File:Baili_Tire_Shop_20181116.jpg|thumb]]
* 1954年：馬歧山、馬積善兄弟創立泰豐橡膠廠於臺灣桃園縣中壢市。
* 1955年：11月23日正式登記成立'''泰豐橡膠工業股份有限公司'''，以產銷橡膠皮帶、傳動三角皮帶等橡膠製品為主。
* 1958年：開發出[[載貨汽車|卡車]]、輕卡車和[[轎車|轎車]]斜交層輪胎，及[[摩托車|摩托車]]、農業用車胎。
* 1960年：與[[日本|日本]][[普利司通|普利司通]]技術合作。
* 1963年：取得[[中華民國國家標準|CNS]][[正字標記|正字標記]]認證合格。
* 1964年：更名成'''泰豐輪胎股份有限公司'''。
* 1965年：開始外銷，並獲得[[美國|美國]]DOT標記認證。
* 1978年：開發出轎車輻射層鋼絲胎。
* 1979年：7月16日於[[臺灣證券交易所|臺灣證券交易所]][[股票上市|股票上市]]。
* 1981年：與[[日本|日本]][[住友橡膠|住友橡膠]]技術合作。
* 1982年：通過[[美國|美國]]U.T.Q.G.認證。
* 1986年：獲選為[[福特汽車|福特汽車]]A級廠家。
* 1987年：通過[[歐盟|歐盟]]ECE認證與[[日本|日本]][[日本工業規格|JIS]]認證。
* 1990年：獲得[[福特汽車|福特汽車]]Q1品質獎，以及[[通用汽車|通用]] / [[歐寶|歐寶]]品質獎。
* 1993年：獲得[[三陽工業|三陽汽車]]繼俊獎。
* 1995年：通過[[ISO_9000|ISO 9001]]品質管制認證。
* 1997年：通過QS-9000認證，[[收購|收購]]江西橡膠廠，成立泰豐輪胎（江西）有限公司。
* 1998年：江西廠開始量產[[載貨汽車|卡車]][[客車|客車]]、輕型卡車與農用車輻射層輪胎。
* 1999年：通過ISO-14001認證，江西廠開始量產[[轎車|轎車]]用輻射層輪胎。
* 2000年：通過[[巴西|巴西]]INMETRO認證。
* 2001年：江西廠取得[[歐盟|歐盟]]ECE認證。
* 2002年：江西廠取得中國輪胎品質協會CCCT認證，獲得SASO（Saudi Arabia Standard Organization）的GCC認證。
* 2005年：自製研發[[臺灣|臺灣]]第一條熱融胎FZ101。
* 2006年：SS595獲得[[中華民國經濟部|中華民國經濟部]][[臺灣精品獎|臺灣精品獎]]。
* 2007年：通過TS16949認證和[[荷蘭|荷蘭]]E-Sound認證。
* 2015年：飛達輪胎改名為「飛德勒輪胎」。
* 2017年：1月17日台灣桃園市中壢區廠區火警<ref>參閱[[三立新聞台|三立新聞台]][http://www.setn.com/News.aspx?NewsID=217182 驚悚！中壢輪胎廠火警　大量黑煙猛竄直衝天際]</ref>，場內存放超過二十萬條輪胎可能燒毀<ref name="LTN1">參閱[[自由時報|自由時報]][http://news.ltn.com.tw/news/society/breakingnews/1950317 ]</ref>。濃烈黑煙不僅在[[大園區|大園區]]、[[平鎮區|平鎮區]]、[[龍潭區_(台灣)|龍潭區]]可見到<ref>參閱鉅亨網[https://tw.stock.yahoo.com/news_content/url/d/a/20170117/%E6%B3%B0%E8%B1%90%E8%BC%AA%E8%83%8E%E4%B8%AD%E5%A3%A2%E5%BB%A0%E9%A9%9A%E5%82%B3%E5%A4%B1%E7%81%AB-%E7%81%AB%E5%8B%A2%E5%A4%A7%E5%88%B0%E6%A1%83%E6%A9%9F%E9%83%BD%E7%9C%8B%E5%BE%97%E5%88%B0-085735048.html 泰豐輪胎中壢廠驚傳失火 火勢大到桃機都看得到]</ref>，甚至[[台北市|台北市]][[內湖區|內湖區]]也可看到濃煙<ref name="LTN1"></ref>。
