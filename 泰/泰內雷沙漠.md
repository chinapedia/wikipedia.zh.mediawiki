'''泰內雷'''（Ténéré、或譯「特內雷」）是位於[[非洲|非洲]][[撒哈拉沙漠|撒哈拉]]南端；介於[[尼日共和國|尼日共和國]]東北方與[[查德共和國|查德共和國]]西方之間的[[沙漠|沙漠]]，所覆蓋的面積在40萬平方公里以上。<ref name="Tenere">{{cite web |url=http://naturalwondersofafrica.com/tenere/ |title=Tenere |work=Natural Wonders Network |accessdate=2015年10月15日}}</ref>

所界定的邊境範圍是北起[[阿哈加爾高原|阿哈加爾高原]]、西接[[阿依爾高原|阿依爾高原]]、南端為[[查德湖|查德湖]]，東部則是[[提貝斯提高原|提貝斯提高原]]一帶，另外[[比爾馬|比爾馬]]是鄰近泰內雷沙漠中央的城鎮。

名稱「泰內雷」在[[圖阿雷格語|圖阿雷格語]]裡即為「沙漠」的意思。<ref name="Tenere"/>

== 氣候 ==
與鄰近的[[撒哈拉沙漠|撒哈拉]]相同，泰內雷為炎熱乾燥的[[沙漠氣候|沙漠氣候]]。在夏天期間白天氣溫平均可達[[攝氏|攝氏]]40度（[[華氏溫標|華式]]111度）、並曾有測量到攝氏52度（華式125度）的紀錄，冬季白日則降至約攝氏29度（華式84度），年均溫方面則是攝氏35度（華式86度）左右。<ref>{{cite web |url=http://www.eoearth.org/view/article/150351/ |title=Aïr and Ténéré Natural Reserves, Niger |work=Encyclopedia Of Earth |author=Mark McGinley |date=2009年3月26日 |accessdate=2015年10月15日}}</ref>

泰內雷是世界上降雨量最少的地方之一，平均一年約10毫米雨量，且常有數年未降雨的情況。而因泰內雷上空雲量少，一年下來累積有4000小時左右的日照曝曬時間，根據[[美國國家航空暨太空總署|美國國家航空暨太空總署]]的觀測泰內雷在接近尼日東南邊的一帶被觀測為地球表面上日照最強的地方。<ref>{{cite web |url=http://articles.latimes.com/2007/dec/17/business/ft-climate17 |title=Mapping a path to solar power |publisher=[[洛杉磯時報|洛杉磯時報]] |author=Alister Doyle |date=2007年12月7日 |accessdate=2015年10月15日}}</ref>

== 歷史 ==
約3億年前[[石炭紀|石炭紀]]裡泰內雷沙漠一帶是非陸地的[[海床|海床]]，之後地面隆起成為熱帶雨林並有[[恐龍|恐龍]]等生物出沒，而泰內雷東南端曾挖掘出眾多恐龍[[化石|化石]]。

在人類出現時泰內雷仍是有綠意的土地，並曾發現過約六萬年前[[舊石器時代|舊石器時代]]裡人們捕捉獵物所留下的石器；或是約一萬年前[[新石器時代|新石器時代]]人們在洞穴裡所作的壁畫。之後受到北方[[撒哈拉|撒哈拉]]沙漠的影響，泰內雷在約2500年前變化成今日荒漠模樣。

== 人口 ==
泰內雷沙漠的人口稀少、人群是以聚集在沙漠邊界活動居多，而[[比爾馬|比爾馬]]是少數位於泰內雷中央的[[綠洲|綠洲]]城鎮之一。

在沙漠西邊[[阿依爾高原|阿依爾高原]]到[[阿加德茲|阿加德茲]]一帶的住民以[[圖瓦雷克人|圖瓦雷克人]]佔多數，經濟活動是利用[[駱駝|駱駝]][[商隊|商隊]]運[[鹽|鹽]]和[[豪薩人|豪薩人]]之間買賣交易為主。其它於泰内雷沙漠生活的民族另有[[桑海人|桑海人]]或[[摩爾人|摩爾人]]等。

== 規劃 ==
1960年尼日獨立後，尼日國土被區分成七個行政區域，其中泰内雷沙漠是位在[[阿加德茲大區|阿加德茲大區]]裡，往後泰内雷沙漠中央一帶則規劃成[[阿德爾和泰內雷自然保護區|阿德爾和泰內雷自然保護區]]並列為[[世界遺產|世界遺產]]。<ref>{{Cite web |url=http://www.birdlife.org/datazone/sitefactsheet.php?id=6733 |title=NNR Aïr - Ténéré |work=[[國際鳥盟|國際鳥盟]] |accessdate=2015年10月15日}}</ref>

== 相關地點 ==
在泰內雷沙漠裡有被認定為是世界上處於最被孤立環境的樹木「[[泰内雷之樹|泰内雷之樹]]」，在1973年遭卡車撞倒之前；都是遊走當地沙漠駱駝商隊的地標。<ref>{{cite web |url=http://www.border-crossings.net/site/journal/present/niger/1.html |title=Eastern Mali to Niger to Timbuktu - 2-19 September 06 |work=Border-Crossings.Net |accessdate=2015年10月15日}}</ref>

而發生在1989年的[[法國聯合航空772號班機空難|法國聯合航空772號班機空難]]；當時曾有飛機殘留物墜落於泰內雷沙漠上。多年之後在2007年5月到6月期間，有在距離發現殘留物地點的北緯16度51分53秒東經11度57分13秒一帶，以石頭堆積成直徑達200英呎、中央為飛機模樣的圓形圖示來紀念事件的受難者。<ref>{{cite web |url=http://www.viralnova.com/i-noticed-this-tiny-thing-on-google-maps-and-when-i-zoomed-in-well-nothing-could-prepare-me/ |title=I Noticed This Tiny Thing On Google Maps.
