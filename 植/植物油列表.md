{{植物油}}

'''植物油'''是自植物中萃取的[[脂肪|脂肪]]與[[油|油]]類，依照提煉方式的不同，分為壓榨油、提煉油和香精油等三種油品。將欲榨取部位加[[壓力|壓]]，進而提取所得的植物油稱為「[[壓榨油|壓榨油]]」，其又分為「冷壓油」和「熱壓油」。在水或其他[[溶劑|溶劑]]中，通過[[溶解|溶解]]植物的相關產油部位而得的油品，稱為「[[提煉油|提煉油]]([[萃取油|萃取油]])」，其油液可以與植物材料分離、再加以濃縮。植物油也可以採[[蒸餾|蒸餾]]方法自植物中萃取出芳香分子，稱為「[[精油|精油]]」。相較油壓榨油和提煉油，精油普遍具有更多元的特性和用途。「[[浸泡油|浸泡油]]」則是經由[[萃取|浸軟]]程序，將所需植物部分放入植物油中浸泡、使底油吸收植物的各種養分製成。

'''植物油'''的制取方法分为三类。首先是压榨法，[[植物|植物]]含油部位在压力的作用下产生[[压榨油|压榨油]]。通过[[水|水]]或者其它[[溶剂|溶剂]][[萃取|萃取]]植物中的[[油脂|油脂]]以产生[[萃取油|萃取油]]。压榨和萃取两者结合产生的粗油，经过[[蒸馏|蒸馏]]浓缩精制可产生[[精油|精油]]。

尽管几乎所有的[[植物|植物]]均含有不同程度的[[油脂|油脂]],被广泛接受和使用的只有少数[[油料作物|油料作物]]<ref>{{cite book |url=http://usda.mannlib.cornell.edu/MannUsda/viewDocumentInfo.do?documentID=1288
 |title=Oil Crops Outlook
 |author=Economic Research Service
 |publisher=United States Department of Agriculture
 |date=1995–2006
}}This publication is available via email subscription.</ref>。

== 食用油 ==
{{Main|食用油}} 

=== 主要非熱帶植物食用油 ===


以下是在全球佔有極高食用比例的植物油品，亦皆可用作[[燃料油|燃料油]]。

* [[玉米油|玉米油]]：又稱粟米油，為普遍的烹飪用油，香氣和口感較淡。<ref>{{cite web |url = http://www.bulkoil.com/scripts/oiltype_item.asp?p=24 |title=Bulk Oil: Corn oil |accessdate=2006-07-24}}</ref>
* [[棉籽油|棉籽油]]：用於生產[[洋芋片|洋芋片]]等零食食品<ref>{{cite web |url = http://www.bulkoil.com/scripts/oiltype_item.asp?p=27 |title=Bulk oil: Cottonseed oil |accessdate=2006-07-24}}</ref>，但是需要去除會破壞生育能力的[[棉酚|棉酚]]。
* [[橄欖油|橄欖油]]：用於烹飪、香妝品、[[香皂|香皂]]製造，也可作為傳統[[油燈|油燈]]的燃料。<ref>{{cite web |url = http://www.globalgourmet.com/food/egg/egg0397/oohistory.html |title = Olive oil history |accessdate = 2006-07-24 |deadurl = yes |archiveurl = https://web.archive.org/web/20060815130653/http://www.globalgourmet.com/food/egg/egg0397/oohistory.html |archivedate = 2006-08-15 }}</ref>
* [[花生油|花生油]]：油質清澈，可用於[[沙拉|沙拉]]調味，且由於[[煙點|煙點]]較高，常作油炸、煎炒用途。<ref>{{cite web |url = http://allrecipes.com/advice/ref/ency/terms/7832.asp |title = Cook's encyclopedia: Peanut oil |accessdate = 2006-07-24 |archiveurl = https://web.archive.org/web/20060704154242/http://allrecipes.com/advice/ref/ency/terms/7832.asp |archivedate = 2006-07-04 |deadurl = yes }}</ref>
* [[油菜|菜籽油]]：世界上最普遍的烹飪用油之一。<ref>{{cite web |url=http://www.canolainfo.org/|title=Canola Oil|accessdate=2006-07-24}}</ref>
* [[紅花|紅花籽油]]：最早用作[[塗料|油漆塗料]]，現今主要用於烹飪。<ref>{{cite web |url=http://www.bulkoil.com/scripts/oiltype_item.asp?p=25 |title=Bulk oil: safflower |accessdate=2006-07-24}}</ref>
* [[芝麻油|芝麻油]]：又稱麻油、香油，冷壓製成淡色食用油，熱壓則為深色、口感濃郁。<ref>{{cite web |url = http://www.bulkoil.com/scripts/oiltype_item.asp?p=34| title=Bulk oil: sesame oil|accessdate=2006-07-24}}</ref>
* [[大豆油|大豆油]]：大豆粉加工製造的附產品。<ref>{{cite web|url=http://southeastfarmpress.com/mag/farming_world_soybean_consumption/index.html|title=Southeast Farm Press: World soybean consumption quickens|accessdate=2006-07-31|deadurl=yes|archiveurl=https://web.archive.org/web/20060605232122/http://southeastfarmpress.com/mag/farming_world_soybean_consumption/index.html|archivedate=2006-06-05}}</ref>
* [[葵花油|葵花油]]：最普遍的烹飪用油之一，亦用於[[生質柴油|生質柴油]]。<ref>{{cite web|url = http://www.bulkoil.com/scripts/oiltype_item.asp?p=11|title=Bulk oil: Sunflower oil |accessdate=2006-07-24}}</ref>
* [[大麻籽油|大麻籽油]]

== 對健康的影響 ==
一些廠商會將植物油都氫化加工，以維持穩定性，氫化加工會增加飽和脂肪酸，而較為落後的加工方式則會產生比飽和脂肪更有害的[[反式脂肪|反式脂肪]]；因此氫化植物油對健康不利，尤其是含反式脂肪的油脂，宜禁止人畜食用。<ref>{{cite web |url = http://www.cfs.gov.hk/tc_chi/programme/programme_rafs/programme_rafs_n_01_05.html |title = 食物安全中心 - 風險評估研究報告書 - 揭露含反式脂肪的熱門食品 |accessdate=2007-10-15}}</ref>
