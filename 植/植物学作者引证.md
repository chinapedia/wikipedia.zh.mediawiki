'''植物学作者引证'''是指第一次根据[[国际植物命名法规|国际植物命名法规]](''ICBN'').<ref name="icbn">{{cite web|url=http://www.bgbm.fu-berlin.de/iapt/nomenclature/code/SaintLouis/0000St.Luistitle.htm|title=International Code of Botanical Nomenclature (St. Louis Code)|last=Greuter|first=W.|date=2003-04-20|isbn=3-904144-22-7|publisher=International Association for Plant Taxonomy|accessdate=2009-01-21}}</ref>确定[[植物|植物]]的[[拉丁种名|拉丁种名]]的作者的名称缩写引证。

缩写则依据ICBN推荐的46A, Note 1执行.<ref name="icbn" /><ref>{{cite book|last = Brummitt|first = RK|coauthors = CE Powell|title =  Authors of Plant Names  
|publisher = Royal Botanic Gardens, Kew |date = 1992 |isbn = 0947643443}}</ref>作者缩写放到植物的拉丁名后面，不用斜体。

例如：
* ''Lens'' Mill.  缩写“Mill.”代表[[菲利普·米勒|菲利普·米勒]]（Philip Miller）
* ''Lens culinaris'' Medik. 缩写“Medik.”代表[[弗里德里希·卡西米尔·麦狄库斯|弗里德里希·卡西米尔·麦狄库斯]]

如果有两个缩写名称，中间加入“ex”，则代表第一个人发现并描述了这种[[植物|植物]]，但没有给出正确的拉丁名称，正确的拉丁名称是由第二个人给出的。

例如：
*''Andropogon aromaticus'' [[Sieber|Sieber]] ex [[Schult.|Schult.]]
:参见 ''[[species:Andropogon_aromaticus|Andropogon aromaticus]]''.

如果两位作者，前面加括号，则表示前一位作者已将这种植物给出名称和分类，但后一位作者改正了前一位的分类，将其列入另一属，重新确定其拉丁种名。

例如：
*''Helianthemum  aegyptiacum'' (L.) Mill.
其原来名称为''Cistus aegyptiacus'' L. <ref name="icbn" />

ICBN的Art. 46.1规定“在出版物中，尤其是讲分类学或拉丁命名法的出版物中，应该加上作者引证缩写”，但在不是分类学著作中，引用植物名称可以不加作者缩写

和[[国际动物命名法规|国际动物命名法规]]不同，植物命名每一个[[门|门]]、[[科|科]]、[[属|属]]、[[种|种]]都有单独的作者命名缩写，例如：

: [[门|门]] Magnoliophyta [[Cronquist|Cronquist]] & al.

: [[亚门|亚门]] Magnoliophytina [[Frohne|Frohne]] & [[U.Jensen|U.Jensen]] ex [[Reveal|Reveal]]

: [[纲|纲]] Magnoliopsida [[Brongn.|Brongn.]]

: [[亚纲|亚纲]] Rosidae [[Takht.|Takht.]]

: [[超目|超目]] Rosanae [[Takht.|Takht.]]

: [[目|目]] Rosales [[Perleb|Perleb]]

: [[亚目|亚目]] Rosineae [[Rchb.|Rchb.]]

: [[科|科]] Rosaceae [[Adans.|Adans.]]

: [[亚科|亚科]] Rosoideae [[Arn.|Arn.]]

: [[族|族]] Roseae [[Lam.|Lam.]] & [[DC.|DC.]]

: [[亚族|亚族]] Rosinae [[J.Presl|J.Presl]]

: [[属|属]] ''Rosa'' [[L.|L.]]

: [[种|种]] ''Rosa damascena'' [[Mill.|Mill.]]

但植物的全称还应该包括作者出版的时间和地点等详细信息，例如：
  
* ''Sequoia sempervirens'' (D.Don) Endl., Syn. Conif. 198 (1847) 
“D.Don”代表[[大卫·顿|大卫·顿]]（David Don）首先给出名称（''Taxodium sempervirens'' D.Don），然后由“Endl.”代表[[史蒂芬·恩德利希|史蒂芬·恩德利希]]在1847年出版的《Synopsis Coniferarum》第198页中给出了目前的名称。
