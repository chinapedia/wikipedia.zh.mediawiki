'''植村吉明'''（{{jpn|j=植村 吉明|hg=うえむら よしあき|rm=Uemura Yoshiaki|f=h}}，{{bd|1912年|1月28日|1973年|5月20日||catIdx=Uemura植村}}），[[日本|日本]][[兵庫縣|兵庫縣]]人，為[[原子物理學|原子物理學]]家。他在[[日治臺灣|日治臺灣]]接受中學教育，並在戰後成為京都大學理學部物理學博士。植村吉明曾任職於[[臺北帝國大學|臺北帝國大學]]以及[[京都大學|京都大學]]，並曾在臺灣參與[[國立臺灣大學物理學系|臺北帝國大學物理學講座]][[荒勝文策|荒勝文策]]教授團隊於1934年完成的[[東亞|東亞]]首次[[原子核|原子核]]擊破實驗。植村吉明也曾參與重建戰後被[[聯合國軍最高司令官總司令部|聯合國軍最高司令官總司令部]]下令拆除的京都大學加速器，並對[[真空|真空]]技術方面抱持著高度的興趣。1973年5月，植村吉明晉升正教授，但也在同月20日因病與世長辭，享壽61歲。逝世後，京都大學化學研究所特別出版《京都大學化學研究所報告》植村吉明特別紀念刊號，以紀念他的貢獻。

=== 京都大學時期 ===
[[File:拆除中的荒勝研究室加速器.png|200px]]
[[File:植村吉明.png|200px]]
1936年，植村吉明回到了京都帝國大學，並且在1939年10月成為研究助理，加入京都帝大化學研究所的荒勝實驗室<ref name="略歴"/><ref name = "張幸真"/>{{rp|107}}。他們也在京大建造了考克饒夫－瓦耳頓型加速器，該加速器可將粒子加速至約50萬[[電子伏特|電子伏特]]的能量<ref name = "鄭伯昆"/>{{rp|575}}。除了利用加速器進行實驗，植村吉明也曾在荒勝文策的領導之下，協同木村毅一利用宇宙射線進行[[高能物理|高能物理]]研究，並將其實驗結果發表至1937年8月的《[[自然_(期刊)|自然]]》雜誌上<ref name = "Nature">{{cite journal lua|author=B. ARAKATSU|coauthors=K. KIMURA & Y. UEMURA|title=Expulsion of Neutrons from Lead by Cosmic Rays|journal=[[自然_(期刊)|Nature]]|volume=140|issue=3537|publisher=Macmillan Publishers|pages=277-278|url=http://www.nature.com/nature/journal/v140/n3537/pdf/140277a0.pdf|date=1937-08-14|accessdate=2015-03-05|language=en}}</ref>。

1945年[[第二次世界大戰|第二次世界大戰]]結束，日本戰敗，京都帝國大學改名為京都大學。之後，[[聯合國軍最高司令官總司令部|聯合國軍最高司令官總司令部]]於該年9月28日下令禁止日本進行有關[[原子物理|原子物理]]與[[航空|航空學]]的研究<ref name = "張幸真"/>{{rp|111}}，拆除荒勝研究室的迴旋加速器並將之傾倒入[[琵琶湖|琵琶湖]]<ref name = "遺伝子">{{cite book|publisher=日経ネット関西|title=湯川秀樹の遺伝子(1)(2)|date=2008年1月|language=ja}}</ref>。由於這造成許多研究資料的喪失，且該設施純為學術性用途，該次拆除行動被多位學者質疑有無其必要性，並引來了國際間的一陣撻伐<ref name = "張幸真"/>{{rp|116}}。1948年4月，植村吉明成為京大的講師，隨後並於1952年8月升任為助理教授<ref name="略歴"/>。1949年，植村吉明在荒勝文策的指導之下，進行關於17[[MeV|MeV]]與6.1MeV[[光子|光子]]在不同物質下吸收係數的研究並發表論文，並於1951年7月以該篇論文得到了京都大學理學博士學位<ref name="略歴"/><ref name = "張幸真"/>{{rp|96}}<ref name="畢業論文"/>。

在[[原子物理學|原子物理學]]相關研究之外，植村吉明在這段期間也曾進行[[應用物理學|應用物理學]]方面的研究。1949年7月，植村吉明與柳父琢治為了改進鉛筆，曾經對[[鉛筆|鉛筆]]芯與紙張之間的[[摩擦係數|摩擦係數]]做研究，比較各種結構的鉛筆芯在這方面之差異<ref name="鉛筆"/>。同年12月，他們二位更與吉田俊夫一同對[[黏土|黏土]]的性質做研究，比較其吸水性與可塑性<ref name="粘土"/>。到了1951年，他們三人更對黏土和[[矽石|矽石]]的[[介電質|介電特性]]差異做分析比較<ref name="誘電"/>。另外，植村吉明也曾和木村毅一與山下佐明一起在1949年發表論文，介紹荒勝研究室的[[齒輪|齒輪]]零件自動選擇裝置<ref name="ローラーチェイン"/>。

1951年，美國解禁日本核物理實驗，木村毅一隨後重建被拆除的核子物理研究室<ref name = "鄭伯昆"/>{{rp|575}}<ref name="木村">{{cite web|url=http://www.rri.kyoto-u.ac.jp/JRS/dayori/D-18.pdf |title=故 木村毅一先生追悼記念特集号 |publisher=原子炉實驗所だより |accessdate=2014-11-28 |language=ja |archiveurl=https://web.archive.org/web/20141204232109/http://www.rri.kyoto-u.ac.jp/JRS/dayori/D-18.pdf |archivedate=2014-12-04 |deadurl=no }}</ref>，植村吉明也從隔年開始參與重建工程<ref name="略歴"/>。1956年京大加速器重建完成，植村吉明也在這段期間關注核物理技術的發展，尤其對真空技術方面抱持著高度的興趣<ref name="略歴"/>。直到1970年，京大加速器被拆除重組，植村吉明再度負責京大加速器的改良，並於1973年5月晉升正教授。然而同月20日植村吉明也因病逝世，享壽61歲<ref name="略歴"/>。

=== 身後 ===
1974年1月，京都大學化學研究所特別以其刊物《京都大學化學研究所報告》出版植村吉明特別紀念刊號，以紀念植村吉明的貢獻<ref name="略歴"/>。在文中，水渡英二說「植村吉明教授是個具有意志力且具備溫暖人格的人。他被所有認識他的人所喜愛，他富有責任感而且被他的老少朋友所信任」，並認為他的一生是「自我犧牲的人生」<ref name="略歴"/>。
