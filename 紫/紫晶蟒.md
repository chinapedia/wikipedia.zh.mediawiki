'''紫晶蟒'''（學名：''{{lang|la|Morelia amethistina}}''），亦稱'''冠叢莫瑞蟒'''、'''水晶蟒'''、'''澳洲灌木蟒'''<ref>{{cite web |url=http://digimuse.nmns.edu.tw/Default.aspx?ObjectId=0b00000181c736a3&Domin=z&tabid=319&Field=r1&ContentType=Exhibit&FieldName=&Language=CHI|title=澳洲灌木蟒 |work=爬蟲學習資源}}</ref>或簡單只叫'''灌木蟒'''<ref name="singtao">{{cite news |url=http://www.singtao.ca/toronto/3997507/2019-02-15/post-%e6%be%b3%e6%b4%b24%e7%b1%b3%e9%95%b7%e5%b7%a8%e8%9f%92%e5%87%ba%e6%b2%92-%e9%af%a8%e5%90%9e%e6%95%b4%e9%9a%bb%e8%a2%8b%e9%bc%a0/?variant=zh-hk |title=澳洲4米長巨蟒出沒 鯨吞整隻袋鼠
|newspaper=[[星島日報|星島日報]]加東版 |date=2019-02-15 |access-date=2019-02-16 |language=zh-hant }}</ref>，是[[蛇亞目|蛇亞目]][[蟒科|蟒科]][[樹蟒屬|樹蟒屬]]下的一個無毒蛇種，主要分布於[[大洋洲|大洋洲]]的[[印尼|印尼]]、[[巴布亞新畿內亞|巴布亞新畿內亞]]及[[澳洲|澳洲]]東北部。紫晶蟒是相當著名的[[爬蟲類|爬蟲類]]動物，牠是澳洲當地體型最巨大的蛇類，也是世界上最巨型的數種蛇類之一。目前尚未有任何亞種被確認。<ref name="ITIS">{{ITIS|ID=634775|taxon=''Morelia amethistina''|year=2007|date=18 September}}</ref>

== 特徵 ==
目前所知最巨大的紫晶蟒可以擁有超過8.5[[米_(單位)|米]]的長度，但事實上這種長度的紫晶蟒並不常見，在當地一條長5.5米的紫晶蟒，已可視為巨型蟒蛇。體色多呈[[棕色|棕色]]，或以[[橄欖色|橄欖色]]及暗[[黃色|黃色]]為基調。[[眼睛|眼睛]]較大，[[頭部|頭部]]有大片鱗片，吻部有熱能感應頰窩。與其它蟒蛇相比，紫晶蟒的軀體顯得較纖幼。其學名中的「[[紫水晶|紫水晶]]」（amethistina），是比喻其於光線下閃耀生輝的乳白色鱗片。

== 地理分布 ==
紫晶蟒主要分布於[[印尼|印尼]]（包括[[摩鹿加群島|摩鹿加群島]]等眾多群島及[[新畿內亞|新畿內亞]]西部的大部分地區）、[[巴布亞新畿內亞|巴布亞新畿內亞]]（包括[[俾斯麥群島|俾斯麥群島]]等地區）以及[[澳洲|澳洲]]（包括[[托雷斯海峽|托雷斯海峽]]、[[約克角半島|約克角半島]]北部及[[大分水嶺|大分水嶺]]等地區）。標準產地未明。<ref name="McD99"/>

== 習性 ==
紫晶蟒主要棲息於雨林、林地及稀樹大草原中，包括[[印尼|印尼]]及[[澳洲|澳洲]][[昆士蘭|昆士蘭]]的[[熱帶雨林|熱帶雨林]]。紫晶蟒是夜行型蛇類，擅長攀爬，幼蛇多棲息於大樹之上，成長後則多在陸地上活動，亦能於水源附近游弋。在交配高峰期，紫晶蟒的求偶表現相當積極進取，有些時候還會與同伴展開激烈的鬥爭。另外，紫晶蟒是[[卵生|卵生]]的蛇類，母蛇每次能產下5至21枚蛇卵。紫晶蟒是凶猛的蛇類，多以[[哺乳類|哺乳類]]動物（如[[齧齒目|齧齒目]]動物、小[[麋鹿|麋鹿]]）、[[鳥類|鳥類]]、[[果蝠|果蝠]]等為主要食物，有時甚至連一隻小袋鼠也可以吞下{{r|singtao}}。
