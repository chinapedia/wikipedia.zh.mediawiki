'''調景嶺站'''（{{lang-en|'''Tiu Keng Leng Station'''}}）是[[港鐵|港鐵]][[將軍澳綫|將軍澳綫]]及[[觀塘綫|觀塘綫]]的一個地底[[跨月台轉車站|跨月台轉車站]]、觀塘綫的東面終點站，位於[[新界東|新界東]][[西貢區|西貢區]][[將軍澳|將軍澳]][[調景嶺|調景嶺]][[翠嶺路|翠嶺路]]與[[景嶺路|景嶺路]]之間，於2002年8月18日啟用。該站是觀塘綫唯一一座位於[[新界|新界]]的[[鐵路站|鐵路站]]。

==車站設計==
雖然很多位於新界的車站都以地面或高架形式建造，但調景嶺站的月台則因為路段及城市規劃問題而須建於地底。再者，它與其他將軍澳區內的車站不同，部分地方是位處山坡上。<ref>[http://www.oocities.org/hk/tiukenglengstation/tik.htm 調景嶺站]</ref>
[[File:半建於山內的調景嶺港鐵站.jpg|thumb]]

===樓層===
調景嶺站的車站結構設計與油塘站相似，車站設有3層，1層大堂及2層月台。大堂位於地面，當中一半樓面面積藏在山底，而月台則位於地底。

===月台===
{{ multiple image | align = right | direction = vertical | width = 220
| image1 = TKL monitor.JPG | caption1 = 列車班次顯示屏，只限往寶琳／康城月台，現已被「鐵路願景」乘客資訊顯示屏取代
}}
調景嶺站設有4個月台，所有月台均有安裝[[月台幕門|月台幕門]]。1號及2號月台為觀塘綫月台，該站為觀塘綫的終點站，因此列車在1號月台清客後，便會前往2號月台接載乘客往[[黃埔|黃埔]]/[[何文田|何文田]]（或尾班車時開往[[觀塘站|觀塘]]或[[九龍灣車廠|九龍灣車廠]]）。每當上下班繁忙時間之後，部份觀塘綫的港鐵列車在此站1號月台清客後，便會暫停服務，先前往2號月台，然後返回[[港鐵九龍灣車廠|九龍灣車廠]]。

而3號及4號月台為將軍澳綫月台，3號月台往[[寶琳站|寶琳]]或[[康城站_(香港)|康城]]，而4號月台在繁忙時間部分列車前往[[北角站|北角]]，非繁忙時間則有由寶琳開出的列車前往北角，或由康城開出的列車於此月台暫停服務、清客並返回3號月台接載乘客。

調景嶺站和[[油塘站|油塘站]]將軍澳綫部分在五桂山隧道的一段設有[[袋狀軌|袋狀軌]]（五桂山隧道），當[[將軍澳綫|將軍澳綫]]的列車故障或臨時需要在此站作為終點站時，月台便會出現很多的人流。現時非繁忙時段的「[[康城站_(香港)|康城站]]穿梭列車」在此站4號月台落客後，便會駛進該袋狀軌，掉頭進入3號月台上客。此外，此站南面設有[[渡綫|渡綫]]，可供列車調度又或讓列車來回觀塘綫和將軍澳綫之間。當列車有車務調動時，列車可以於入站前由油塘站1號月台經渡綫往此站3號月台，或於出站後由此站4號月台經渡綫往油塘站2號月台，每天均有港鐵列車使用本渡綫進行同方向的車務調度。由於調景嶺站是部分班次的終點站之一，所以在上下層月台的端牆(近油塘站的一方)設有各一個員工專用洗手間及司機休息室。

==鄰接車站==
調景嶺站是將軍澳綫的中途站、觀塘綫的上行終點站及將軍澳綫的康城站穿梭列車的終點站。同時，是一個[[跨月台轉車站|跨月台轉車站]]，因此本站是一個重要的兩綫轉車站，乘客可以在觀塘綫及將軍澳綫之間進行轉乘，往返[[香港島|香港島]]、[[九龍|九龍]]及[[新界|新界]]各地。

==利用狀況==
調景嶺站啟用初期，由於四周只有[[彩明苑|彩明苑]]及[[維景灣畔|維景灣畔]]兩個屋苑，因此車站出入閘人流一直比較少。但由於車站是觀塘綫及將軍澳綫的轉車站，因此，大部份利用車站的人流均為轉車的乘客，當時成為前[[香港地鐵|香港地鐵]]系統中出入閘人流較少車站之一。直至鄰近屋苑，例如[[健明邨|健明邨]]、[[城中駅|城中駅]]、[[都會駅|都會駅]]及學校落成後，車站人流開始上升，並帶動車站商店的生意，加上位於車站旁的[[香港專業教育學院李惠利分校|香港專業教育學院李惠利分校]]、[[香港知專設計學院|香港知專設計學院]]及[[明愛專上學院|明愛專上學院]]的新校舍啟用，[[善明邨|善明邨]]亦已於2011年4月入伙後，車站人流絡繹不絕。

由於調景嶺站佔盡觀塘綫和將軍澳綫之便，故能吸引附近居民利用港鐵出行。再加上本站與[[將軍澳站|將軍澳站]]相差僅約800米，僅次於[[太子站|太子站]]與[[旺角站|旺角站]]之間，對部分居於兩站之間（將軍澳中心及唐明苑）的乘客來説，他們也樂於徒歩到本站直接乘搭觀塘綫來往九龍中部，或乘搭將軍澳綫來往港島，以便節省車費及轉少一次車。

通往[[將軍澳華人永遠墳場|將軍澳華人永遠墳場]]的新行人路，已於2012年2月26日啟用，分流過往由[[油塘站|油塘站]]前往的乘客，每逢[[清明節|清明節]]和[[重陽節|重陽節]]，將會進一步增加本站進出人次。

==加建洗手間==
{{ multiple image | align = right | direction = vertical | width = 220
| image1 = Tiu Keng Leng Station 2018 01 part1.jpg | caption1 = 興建中的洗手間位置
}}
本站與油塘站洗手間興建工程已於2017年第四季正式動工，根據港鐵公開介紹，車站日後將設有男廁、女廁、傷殘廁及育嬰室，預計最快於2019年初竣工。

==歷史==
===增建B出入口===
調景嶺站大堂的客務中心對出早已預留B出入口，可以連接[[都會駅|都會駅]]、[[城中駅|城中駅]]及[[調景嶺站公共運輸交匯處|調景嶺站公共運輸交匯處]]，由於調景嶺站啟用時上蓋物業項目尚未完成，所以出入口延至2006年11月27日啟用。

===康城站穿梭列車===
2009年7月20日，港鐵為了讓乘客熟悉在[[康城站_(香港)|康城站]]通車後的列車班次安排，開始在非繁忙時段派出往返康城站及調景嶺站的「康城站穿梭列車」接載乘客，往來康城站的列車在非繁忙時間會以此站為尾站，但這些列車在到達[[將軍澳站|將軍澳站]]後會改以不載客方式前往未正式開放的康城站。

2009年7月26日，前往將軍澳南康城站的支綫正式通車，班次的分配情況有所改變。其中逢星期一至六非繁忙時段、星期日及公眾假期，前往康城站之列車，均不會從[[北角站|北角站]]開出，改為每12分鐘由此站開出，名為「康城站穿梭列車」，從北角站出發往康城站的乘客須先乘至此站或將軍澳站，再轉乘穿梭列車。而該段時間從北角站開往[[寶琳站|寶琳站]]的班次則維持現狀。
