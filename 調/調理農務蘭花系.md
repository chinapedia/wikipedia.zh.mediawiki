'''調理農務蘭花系'''（{{lang-en|The Faculty Of Orchid Gardening}}），簡稱'''蘭花系'''，為一個[[香港本土派|香港本土派]][[極右翼|極右翼]]政治組織，其政治立場偏向[[港獨|港獨]]派。此組織強烈反對[[中國共產黨|中國共產黨]]、[[建制派|建制派]]及[[泛民主派|泛民主派]]（尤其[[民主黨_(香港)|民主黨]]）<ref name="蘭花系三首腦起底 揭「失落中年」變「港獨中堅」" />，並與[[黃毓民|黃毓民]]等修憲派人士關係密切，部分成員視黃為教主<ref name="蘭花系三首腦起底 揭「失落中年」變「港獨中堅」" />，以及多次提倡香港市民[[移民|移民]]至外地居住。

其中，蘭花系成員綽號牛精文的陳文，曾在2013年代表蘭花系參與[[油尖旺區議會|油尖旺區議會]]京士柏選區的補選，然而陳終以大比數於該場選舉中落敗。<ref name="涉「殺警」言論 三蘭花系成員被捕">{{cite news |language = 中文 |author =  |coauthors =  |url = http://std.stheadline.com/daily/news-content.php?id=91296&target=2 |title = 涉「殺警」言論 三蘭花系成員被捕 |publisher = 星島日報(香港) |pages =  |date = 2015-05-28 |accessdate = 2017-04-02}}</ref>

==沿革==
蘭花系的初期成員由[[激進民主派_(香港)|激進民主派]]組織[[人民力量|人民力量]]的退黨成員組成，主要成員為一群在[[大埔區|大埔區]]長大、不滿香港政局的市民。<ref name="蘭花系三首腦起底 揭「失落中年」變「港獨中堅」" />香港政府在2012年底推出[[新界東北發展計劃|新界東北發展計劃]]引發爭議後，後來成為該組織創會幹事的[[業餘|業餘]][[蘭花|蘭花]][[種植|種植]]者[[莊明堅|莊明堅]]（人稱堅叔），在2012年12月8日初次以「調理農務蘭花系」組織名義出席[[香港立法會|香港立法會]]新界東北發展的公眾諮詢會議，當日莊明堅的發言廣為香港市民關注。<ref name="隔牆有耳︰蘭花系莊明堅　調理農務　開心抗爭">{{cite news |language = 中文 |author = 李八方 |coauthors =  |url = http://hk.apple.nextmedia.com/news/art/20121224/18112326 |title = 
隔牆有耳︰蘭花系莊明堅　調理農務　開心抗爭 |publisher = 蘋果日報(香港) |pages =  |date = 2012-12-24 |accessdate = 2017-04-01}}</ref>

調理農務蘭花系作為政治組織則於2013年1月正式成立，初時以「團結農民，維護本土農業發展；推動本土化，港人優先；勇武抗爭，不平則鳴」為宗旨，爭取推翻該計劃。<ref name="「調理農務蘭花系」大起底">{{cite news |language = 中文 |author =  |coauthors =  |url = http://paper.wenweipo.com/2013/08/15/HK1308150001.htm |title = 「調理農務蘭花系」大起底 |publisher = 文匯報 |pages =  |date = 2013-08-15 |accessdate = 2017-04-01}}</ref>

蘭花系主席[[馬健賢|馬健賢]]指自己因受不住[[梁振英|梁振英]]的施政方式，因而決定走出街頭，反對[[梁振英政府|梁振英政府]]，開展了其參與政治的生涯。<ref name="離地中產變怒火中年 經紀無悔失生意：忍唔到689">{{cite news |language = 中文 |author =  |coauthors =  |url = http://nextplus.nextmedia.com/news/spot/20160717/411348 |title = 離地中產變怒火中年 經紀無悔失生意：忍唔到689 |publisher = 壹週Plus |pages =  |date = 2016-07-17 |accessdate = 2017-04-01}}</ref>

== 東北發展諮詢會議 ==
2012年12月8日，莊明堅在立法會新界東北發展的公眾諮詢會議以「調理農務蘭花系幹事」身份發言，該次發言技驚四座，引起市民關注，發言的影片在[[分享網站|分享網站]][[YouTube|YouTube]]錄得約40萬人次的瀏覽。莊當日的粵語發言如下：<ref>{{cite web |url=https://www.youtube.com/watch?v=ANdDXFcf_BI |title=莊明堅, 調理農務蘭花系幹事打倒梁振英，反對新東北計劃 @ 2012.12.8 |date=2012-12-08 }}</ref>

:「主席，我係調理農務蘭花系嘅幹事。蘭花，係花中君子，王者之香。係咪梁振英係特首、小人、大話精、偽君子，就可以僭建花槽種蘭花？就唔俾地我哋調理農務？大陸、台灣出口嘅蘭花，係百幾二百種呀！賺本港數以億計嘅外匯。點解我哋調理農務係咁難呀？ 主席，我哋調理農務蘭花系，農務蘭花三四十年；日出而作，日入而息，清茶淡飯，與世無爭。儲咗十零萬，諗住可以安享晚年；但係點解你個『福佳』嘅梁振英，都係左查右查我哋？仲要割地賣港，毁我家園！我哋調理農務都唔得。主席，我哋調理農務蘭花系反對東北嘅發展計劃！所謂『斷人衣食，尤如殺人父母』，所謂『搬遷』，就好似將海龜搬去沙漠，死梗㗎！道理講完，你哋會聽咩？主席，我哋調理農務蘭花系，對於毫無誠信嘅政府，講乜都冇用。你哋語言嘅『偽術』，已經去到『不攻自破、不打自招、不知所謂』嘅地步。而家唯一可以令市民滿意嘅，就係取消方案，同埋梁振英落台！我哋調理農務蘭花系對『東北』說不！一月一號，『CY元旦大遊行』見！打倒梁振英！打倒港共政權！﻿」

:（語譯：「主席，我是調理農務蘭花系的幹事。蘭花，是花中君子，王者之香。是否因梁振英是特首、小人、慣性說謊者、偽君子，便可以僭建花槽種植蘭花？便不給土地我們調理農務？大陸、台灣出口的蘭花，有百多二百種啊！賺取本港數以億計的外匯。為什麼我們調理農務是這樣困難呢？ 主席，我們調理農務蘭花系，農務蘭花三四十年；日出而作，日入而息，清茶淡飯，與世無爭。儲蓄了十多萬元，以為可以安享晚年；但是為什麼你這位『福佳』的梁振英，總是不斷的查核我們？還要割地賣港，毁我家園！我們調理農務都不可以。主席，我們調理農務蘭花系反對東北的發展計劃！所謂『斷人衣食，尤如殺人父母』，所謂『搬遷』，就好像將海龜搬到沙漠，是必死的！道理說完，你們會聽嗎？主席，我們調理農務蘭花系，對於毫無誠信的政府，說什麼也沒有用。你們語言的『偽術』，已經到了『不攻自破、不打自招、不知所謂』的地步。現在唯一可以令市民滿意的，就是取消方案，連同梁振英落台！我們調理農務蘭花系對『東北』說不！一月一日，『CY元旦大遊行』見！打倒梁振英！打倒港共政權！﻿」）

==政治行動==
===反對香港警察===
{{more|林慧思事件}}

===反對激進建制派集會===
2013年8月4日，在[[旺角|旺角]][[西洋菜街|西洋菜街]]行人專用區內發生群眾集結事件，支持或抗議由[[極左翼|極左翼]][[激進建制派|激進建制派]]政治組織香港行動及[[李偲嫣|李偲嫣]]領導的香港家長聯會所舉辦的以「支持警隊嚴正執法 粗鄙文化遠離學園」為題的集會，蘭花系成員亦到場反對集會，且與在場香港警察互相有肢體衝撞。<ref name="「調理農務蘭花系」大起底" />

2014年6月4日，在[[北角|北角]]金殿大廈外，蘭花系成員與正在集會的激進建制派組織[[愛港之聲|愛港之聲]]成員互相對罵，其中蘭花系成員盧建民多次以粗口指罵愛港之聲的成員，最終被警察票控以[[公眾地方|公眾地方]]行為不檢。該案件審訊期間，控方表示警方在當日於1小時內對盧作出4次警告不果才作出[[拘捕|拘捕]]行動。案件最後因控方庭上同意不提證供起訴，容許盧以[[自簽擔保守行為|自簽擔保守行為]]1年半而不再追究了事，但不準盧再次「干擾或企圖干擾社會秩序、對任何人或威脅對任何人行使武力、作出喧嘩或有意圖激使他人破壞社會安寧」。<ref name="隔牆有耳：蘭花系成員爆粗守行為">{{cite news |language = 中文 |author = 李八方 |coauthors =  |url = http://hk.apple.nextmedia.com/news/art/20150325/19089345 |title = 隔牆有耳：蘭花系成員爆粗守行為 |publisher = 蘋果日報(香港) |pages =  |date = 2015-03-25 |accessdate = 2017-04-02}}</ref>

===發起「唱紅」行動呼籲大陸旅客回國內消費===
2014年3月17日，蘭花系連同其它組織發起「唱紅」行動，[[反諷|反諷]][[中國大陸|中國大陸]][[旅客|旅客]]在香港購買外國貨是「不愛國」的行為。集會人士在[[銅鑼灣|銅鑼灣]][[香港崇光百貨|崇光百貨]][[門市|門市]]附近，打扮成類似[[文化大革命|文化大革命]]期間[[紅衛兵|紅衛兵]]的裝束、播紅歌、揮動[[五星紅旗|五星紅旗]]及[[中國共產黨黨旗黨徽|中共黨旗]]等，及後更發起[[遊行|遊行]]。蘭花系成員在集會中提出「解放銅鑼灣」的訴求，呼籲大陸旅客回國內消費，提醒他們不要受[[資本主義|資本主義]]「荼毒」，又高呼「愛祖國用國貨，買洋貨是賣國賊」。集會期間，有2名操[[普通話|普通話]]的旅客指罵集會人士，集會人士回罵對方是「漢奸」。<ref name="「紅衞兵」大戰普通話男女">{{cite news |language = 中文 |author =  |coauthors =  |url = http://hk.apple.nextmedia.com/news/art/20140317/18659147 |title = 「紅衞兵」大戰普通話男女 |publisher = 蘋果日報(香港) |pages =  |date = 2014-03-17 |accessdate = 2017-04-03}}</ref>

===網台上發表強烈反對警察及建制派言論===
2015年5月27日，香港警察在[[大埔|大埔]]及[[天水圍|天水圍]]等地區拘捕蘭花系三名核心成員（包括主席馬健賢、網台主持陳文及綽號大咪的何啟明），警方指他們「涉嫌不誠實取用電腦及其他罪行」。<ref name="狂言「殺警」「滅門」 蘭花系三成員被拘">{{cite news |language = 中文 |author = 李自明 |coauthors = 齊正之 |url = http://hk.apple.nextmedia.com/news/art/20150325/19089345 |title = 狂言「殺警」「滅門」 蘭花系三成員被拘 |publisher = 文匯報 |pages =  |date = 2015-05-28 |accessdate = 2017-04-02}}</ref>其中，有2名成員被指涉嫌在2014年12月11日於網台低俗頻道的節目第79集《現代豪俠傳》中發表「殺警」言論，指「殺警就要夠薑」（意即殺死警察就先要有種）、「有自殺傾向的人，一定要殺幾個警察再死」等強烈反對警察的言論。<ref name="涉「殺警」言論 三蘭花系成員被捕" />另外，馬健賢更涉嫌以「滅門」字眼在網上對表示支持警察的建制派作家[[屈穎妍|屈穎妍]]及其家人作出恐嚇。<ref name="狂言「殺警」「滅門」 蘭花系三成員被拘" />事後，蘭花系在[[社交網站|社交網站]][[Facebook|Facebook]]的專頁上載寫有「維民所止」字樣的圖片，並斥以「雍正無頭、言論自由，退回清朝！」，諷刺香港的言論自由已降至有如昔日[[大清|大清]]初年[[文字獄|文字獄]]興盛的年代，指控[[香港政府|香港政府]]以言入罪。<ref name="蘭花系兩成員被警拘捕　專頁貼「維民所止」諷文字獄">{{cite news |language = 中文 |author =  |coauthors =  |url = http://www.passiontimes.hk/article/05-28-2015/23265 |title = 蘭花系兩成員被警拘捕　專頁貼「維民所止」諷文字獄 |publisher = 熱血時報 |pages =  |date = 2015-05-28 |accessdate = 2017-04-02}}</ref>
