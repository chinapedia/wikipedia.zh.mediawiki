《'''开皇律'''》是[[隋文帝|隋文帝]]在位时期制定颁行的[[法律|法律]]，共12篇500条。其重新确定了[[五刑|五刑]]、[[八议|八议]]、[[十恶_(法律名词)|十恶]]制度，为后朝[[法典|法典]]相继沿用，在[[中国法制史|中国法制史]]上占有重要地位。

== 制定 ==
[[开皇|开皇]]元年（581年），[[隋文帝|隋文帝]]为改革旧律，命[[高颎|高颎]]及其它重臣[[郑译|郑译]]、[[杨素|杨素]]、[[苏威|苏威]]、[[裴政|裴政]]等十四人，广泛采用[[曹魏|魏]]、[[晋朝|晋]]、[[北齐|齐]]、[[南梁|梁]]各律的优点，“沿革轻重，取其折衷”，制定《新律》。十月完成，颁布实行。三年（583年），文帝“因览[[刑部|刑部]]奏，断狱数犹至万条。以为律尚严密，故人多陷罪”。因此又命[[苏威|苏威]]、[[牛弘|牛弘]]等人，本着删繁就简，以轻代重为原则，对新律重新更定，制成《开皇律》。

== 律文 ==
律文共十二篇，五百条。与旧律相比，删去[[死刑|死罪]]八十一条，[[流放|流罪]]一百五十四条，[[徒刑|徒]]、[[杖刑|杖罪]]一千余条。主要参照[[北齐|北齐]]，部分参照[[北周|北周]]的法律而制成。其篇目分别是：名例、卫禁、职制、户婚、厩库、擅兴、贼盗（亦作盗贼）、斗讼、诈伪、杂律、捕亡、断狱十二篇。还制定了一些有利于[[平民|平民]]的[[诉讼|诉讼]]程序，如百姓有冤情，可逐级上告，甚至直到[[朝廷|朝廷]]。

== 刑罚 ==
[[杨坚|杨坚]]在[[北周|北周]]辅政时，鉴于其“刑政苛酷，群心崩骇”，主张“法令清简”，并且“躬履节俭”，结果“天下悦之”<ref>《[[隋书|隋书]]·高祖纪上》</ref>。即帝位后，在制定新法时，废除前朝的[[鞭刑|鞭刑]]、[[枭首|枭首]]、[[车裂|轘裂]]等[[酷刑|酷刑]]，以及“孥戮相坐之法”<ref>《隋书·刑法志》</ref>。确立[[死刑|死刑]]、[[流刑|流刑]]、[[有期徒刑|徒刑]]、[[杖刑|杖刑]]、[[笞刑|笞刑]]五种刑罚共分二十等，即死刑二等：[[絞刑|绞]]、[[斬首|斩]]；流刑三等：一千里、一千五百里、二千里，或服劳役二年、二年半、三年；徒刑五等：一年、一年半、二年、二年半、三年；杖刑五等：自六十到一百；笞刑五等：自十至五十。

新的[[五刑|五刑]]制度自《开皇律》确定为法定刑法后，取代了[[墨刑|墨]]、[[劓刑|劓]]、[[刖刑|刖]]、[[宫刑|宫]]等割裂肌肤、残害肢体的[[肉刑|肉刑]]，为历史形制的进步，被以后历代律典所沿用。

=== 十恶 ===
{{main|十恶 (法律名词)}}

《开皇律》在《[[北齐律|北齐律]]》“重罪十条”基础上，改为“十恶之条”，即：一、[[谋反|谋反]]，二、谋大逆，三、谋叛，四、恶逆，五、不道，六、[[大不敬|大不敬]]，七、不孝，八、不睦，九、不义，十、内乱。按规定，“犯十恶及故（意）杀人狱成（经审讯已定案）者，虽会赦（[[大赦|大赦]]）尤除名。”十恶的罪名自《开皇律》确立后，历代法典均相继沿用、不作修改。

=== 特权 ===
《开皇律》对[[贵族|贵族]]、[[官员|官员]]的[[犯罪|犯罪]]在[[法律|法律]]上享有特权。凡是在[[八议|八议]]范围内的人或[[七品|七品]]以上官吏犯罪（[[十恶_(法律名词)|十恶]]除外），都可以减罪一等。[[九品|九品]]以上[[官吏|官吏]]犯罪，可以用[[钱|钱]]来[[赎罪|赎罪]]。

== 影响 ==
[[隋朝|隋朝]]虽仅存三十七年，但其[[法制|法制]]建设又很大成就，而《开皇律》被[[史学家|史学家]]誉为好法，其上承[[汉律|汉律]]的源流，各项基本制度均被[[唐律|唐律]]直接继承，成为唐律的直接蓝本。后来又为[[宋朝|宋]]、[[明朝|明]]、[[清朝|清]]各朝所沿用，在中国法律制度发展史上占着重要的地位。并对古代[[日本|日本]]、[[朝鲜|朝鲜]]、[[越南|越南]]等国建立和完善法制产生过影响。
