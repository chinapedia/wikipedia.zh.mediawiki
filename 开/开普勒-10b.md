'''克卜勒-10b'''是使用NASA的[[克卜勒任務|克卜勒望遠鏡]]發現，被確認的第一顆岩質[[系外行星|系外行星]]<ref name="BBCNews"/>。{{As of|2011|1|}}克卜勒-10b也是使用[[凌日法|凌日法]]發現的最小的系外行星。這顆系外行星的發現是以2009年5月至2011年1月，長達8個月的觀測資料為基礎<ref name="NASANews"/>，再輔以[[凱克天文台|凱克天文台]]使用[[凱克望遠鏡#儀器|HIRSE]]的[[都卜勒光譜儀|徑向速度測量]]確認了克卜勒-10b的存在。

== 恆星 ==
{{main|克卜勒-10}}
系外行星，克卜勒-10，是第一顆被確認窩藏著小的凌日行星的一顆恆星，它一直被列在位於夏威夷的地基10米凱克望遠鏡優先觀測目標的頂端。克卜勒-10位於[[天龍座|天龍座]]，距離我們的太陽系560光年，大小與太陽相似，估計這顆恆星的年齡已經有119億歲。

== 行星 ==
[[File:Lightcurve20090111Kepler-10b.jpg|200px]]
克卜勒-10b的大小是[[地球|地球]]的1.4倍，它繞行恆星的週期短於一天，距離連有[[水星|水星]]到太陽距離的20分之一都不到。它的表面温度大約是1,600[[熱力學溫標|K]]，足以將[[黃金|黃金]]都熔解。

== 意義 ==
[[File:CompositionOfKepler-10b.jpg|thumb]]
克卜勒-10b的發現具有重大的意義，因為這增加並確認了發現比地球更大或更小的[[類地系外行星|類似我們地球的岩質行星]]的可能性。
