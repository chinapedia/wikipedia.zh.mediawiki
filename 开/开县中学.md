'''开县中学'''位于重庆市[[开州区|开州区]][[云枫街道|云枫街道]]，是原[[四川省|四川省]]首批重点中学，也是重庆市首批[[重点中学|重点中学]]。<ref>参阅四川省、重庆市档案</ref>。学校前身为1839年成立的培俊堂，现占地面积400[[亩|亩]]，有学生7000人，教师366人。<ref>{{cite web |url=http://www.cqskxzx.com/xzkk/ShowArticle.asp?ArticleID=109 |title=奋进中的开县中学 |date=2012-03-20 |accessdate=2016-08-11 |publisher=开县中学  }}</ref>

== 历史沿革 ==
=== [[中华人民共和国成立|中华人民共和国成立]]前 ===
*1839年，培俊堂成立
*1905年，培俊堂更名为公立中西学堂
*1907年，公立中西学堂更名为[[开县|开县]]高等小学堂
*1923年，开县高等小学堂更名为开县县立初级中学校
*1944年，开县县立初级中学校更名为开县县立中学校
=== [[中华人民共和国成立|中华人民共和国成立]]后 ===
*1950年，开县县立中学校更名为[[川东|川东]]区开县中学校
*1952年，川东区开县中学校更名为四川省开县中学校
*1956年，四川省开县中学校更名为四川省开县第一中学校
*1963年，四川省开县第一中学校更名回四川省开县中学校
*1994年，学校更名四川省开县中学，被列为四川省首批重点中学
*1997年，学校更名为重庆市开县中学<ref>{{cite web |url=http://www.cqskxzx.com/xzkk/ShowArticle.asp?ArticleID=118 |title=校史沿革 |accessdate=2016-08-11 |publisher=开县中学 }}</ref>
