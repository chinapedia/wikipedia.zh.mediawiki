'''开普勒5'''（{{lang en|'''Kepler-5'''}}），是一颗位于[[天鹅座|天鹅座]]的[[類太陽恆星|類太陽恆星]]。這顆恆星是由[[开普勒太空望远镜|开普勒太空望远镜]]發現的。开普勒太空望远镜於2010年发现开普勒5擁有一颗[[太阳系外行星|行星]][[开普勒5b|开普勒5b]]<ref name="datatable"/>。這個發現是於2010年1月4日被公佈的。

==命名和歷史==
[[File:Kepler_first_five_exoplanet_size.jpg|thumb]]

开普勒4是开普勒太空望远镜發現的第五顆恆星，因此得名「开普勒5」。<ref>{{cite web |url=http://kepler.nasa.gov/Mission/QuickGuide/ |title=Kepler: About the Mission |author= |date=2011 |work=Kepler Mission |publisher=NASA |accessdate=26 February 2011}}</ref>值得注意的是，首三個被开普勒太空望远镜發現的恆星（[[GSC_03549-02811|开普勒1]]、[[HAT-P-7|开普勒2]]、[[开普勒3|开普勒3]]）皆已被其他天文學家搶先發現，所以开普勒5是第二個被开普勒太空望远镜首次發現的恆星。<ref name="datatable"/>[[美國航空航天局|美國航空航天局]]於2010年1月4日，在[[華盛頓哥倫比亞特區|華盛頓哥倫比亞特區]][[美國天文學會|美國天文學會]]第215次會議中，宣佈他們發現了[[开普勒4|开普勒4]]、开普勒5、[[开普勒6|开普勒6]]、[[开普勒7|开普勒7]]和[[开普勒8|开普勒8]]。在被公佈的行星中，开普勒5b是體積第二大的行星，以及質量第一大的行星。<ref>{{cite web |url=http://cs.astronomy.com/asycs/blogs/astronomy/archive/2010/01/05/215th-aas-meeting-update-kepler-discoveries-the-talk-of-the-town.aspx |title=215th AAS meeting update: Kepler discoveries the talk of the town |author=Rich Talcott |date=5 January 2010 |work=Astronomy.com |publisher=''[[Astronomy_(magazine)|Astronomy]]'' magazine |accessdate=26 February 2011}}</ref>

在开普勒太空望远镜發現了开普勒5b後，天文學家們又在[[夏威夷|夏威夷]][[冒纳凯阿火山|冒纳凯阿火山]][[凱克天文台|凱克天文台]]、[[得克薩斯州|得克薩斯州]][[麦克唐纳天文台|麦克唐纳天文台]]、[[亞利桑那州|亞利桑那州]][[多镜面望远镜|多镜面望远镜]]、{{link-en|WIYN天文台|WIYN Observatory}}、[[弗雷德·勞倫斯·惠普爾天文台|弗雷德·勞倫斯·惠普爾天文台]]、[[加利福尼亞州|加利福尼亞州]][[帕洛马山天文台|帕洛马山天文台]]、[[利克天文台|利克天文台]]和[[加那利群島|加那利群島]][[穆查丘斯罗克天文台|穆查丘斯罗克天文台]]驗證這個發現。<ref>{{cite web |url=http://www.nasa.gov/mission_pages/kepler/news/kepler-5-exoplanets.html |title=NASA's Kepler Space Telescope Discovers its FIrst Five Exoplanets |author= |date=4 January 2010 |work= |publisher=[[NASA|NASA]] |accessdate=26 February 2011}}</ref>

==恆星狀況==
开普勒5是一顆[[類太陽恆星|類太陽恆星]]。其質量為[[太陽質量|太陽質量]]的1.374 ± 0.056倍，而半徑則為[[太陽|太陽半徑]]的1.793 ± 0.053倍。這代表开普勒7比太陽重37%，並且比太陽闊79%。<ref name="EPE"/>這顆星的估計年齡為30 ± 6億年。<ref>[http://kepler.nasa.gov/Mission/discoveries/kepler5b/ Kepler-5b - NASA]</ref>這顆星的估計[[金屬量|金屬量]]為[Fe/H] = 0.04 ± 0.06，比太陽的金屬豐度多<ref name=EPE>{{cite web |url=http://exoplanet.eu/star.php?st=Kepler-5 |title=Notes for star Kepler-5 |author= |date=2010 |work= |publisher=[[Extrasolar_Planets_Encyclopaedia|Extrasolar Planets Encyclopaedia]] |accessdate=26 February 2011 |deadurl=yes |archiveurl=https://web.archive.org/web/20110121155432/http://exoplanet.eu/star.php?st=Kepler-5 |archivedate=2011年1月21日 }}</ref>，因此很有可能擁有行星。<ref>{{cite web |url=http://www.spacedaily.com/news/extrasolar-04zl.html |title=Extrasolar Planets: A Matter of Metallicity |author=Henry Bortman |date=12 October 2004 |work= |publisher=Space Daily |accessdate=26 February 2011}}</ref>而[[有效溫度|有效溫度]]則為6297 ± 60 [[開氏溫標|K]]。<ref>{{cite web |url=http://nssdc.gsfc.nasa.gov/planetary/factsheet/sunfact.html |title=Sun Fact Sheet |author=David Williams |date=1 September 2004 |work=[[Goddard_Space_Flight_Center|Goddard Space Flight Center]] |publisher=[[NASA|NASA]] |accessdate=26 February 2011}}</ref>相比而言，[[太陽|太陽]]的年齡為45.7億年<ref>{{cite web |url=http://www.universetoday.com/18237/how-old-is-the-sun/ |title=How Old is the Sun? |author=Fraser Cain |date=16 September 2008 |work= |publisher=Universe Today |accessdate=25 February 2011}}</ref>，有效溫度僅為5778 K，金屬量為[Fe/H] = 0.0122。<ref>

开普勒5的[[視星等|視星等]]為13.4，代表人類肉眼是看不見這顆星的，必須以望遠鏡觀測。<ref name="datatable"/>

== 行星系統 ==
[[File:Exoplanet_Comparison_Kepler-5_b.png|thumb]]（左）的大小比較]]

於2010年，[[开普勒太空望远镜|开普勒太空望远镜]]发现它的一颗[[太阳系外行星|行星]][[开普勒5b|开普勒5b]]<ref name="datatable"/>。

开普勒5b的質量為[[木星質量|木星質量]]的2.114倍，半徑為[[木星|木星半徑]]的1.431倍。其[[半長軸|半長軸]]長0.05064 [[天文單位|AU]]，[[軌道周期|軌道周期]]為3.5485天，因此是一個[[熱木星|熱木星]]。<ref name="datatable"/>這個距離為水星半長軸的13%（水星的半長軸長0.3871 AU）。<ref>{{cite web |url=http://nssdc.gsfc.nasa.gov/planetary/factsheet/mercuryfact.html |title=Mercury Fact Sheet |author=David Williams |date=17 November 2010 |work=[[Goddard_Space_Flight_Center|Goddard Space Flight Center]] |publisher=[[NASA|NASA]] |accessdate=26 February 2011}}</ref>這顯示开普勒4b是一個[[熱海王星|熱海王星]]。开普勒4b的[[軌道離心率|軌道離心率]]被假設為0。其表面溫度達1930 ± 100 K，比木星的表面溫度（175 K）要高得多。<ref name="datatable"/>
