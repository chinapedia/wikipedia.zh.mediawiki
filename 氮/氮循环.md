'''氮循环'''({{lang-en|Nitrogen cycle}})是描述自然界中[[氮|氮]][[单质|单质]]和含氮化合物之间相互转换过程的[[生态系统的物质循环|生态系统的物质循环]]。

== 基本概念 ==
[[地球大气|空气]]中含有大约78%的氮气，占有绝大部分的氮元素。氮是许多生物过程的基本元素；它存在于所有组成[[蛋白质|蛋白质]]的[[氨基酸|氨基酸]]中，是构成诸如[[DNA|DNA]]等的核酸的四种[[基本元素|基本元素]]之一。在植物中，大量的氮素被用于制造可进行[[光合作用|光合作用]]供植物生长的叶绿素分子。

加工，或者固定，是将气态的[[游离态|游离态]]氮转变为可被有机体吸收的[[化合态|化合态]]氮的必经过程。一部分氮素由[[闪电|闪电]]所固定，同时绝大部分的氮素被非共生或共生的[[固氮细菌|固氮细菌]]所固定。这些细菌拥有可促进氮气氢化成为[[氨|氨]]的[[固氮酶|固氮酶]]，生成的氨再被这种细菌通过一系列的转化以形成自身组织的一部分。某一些固氮细菌，例如[[根瘤菌|根瘤菌]]，寄生在[[豆科植物|豆科植物]]（例如[[豌豆|豌豆]]或[[蚕豆|蚕豆]]）的[[根瘤|根瘤]]中。这些细菌和植物建立了一种[[互利共生|互利共生]]的关系，为植物生产氨以换取[[糖类|糖类]]。因此可通过栽种[[豆科植物|豆科植物]]使氮素贫瘠的土地变得肥沃。还有一些其它的植物可供建立这种共生关系。

其它植物利用根系从[[土壤|土壤]]中吸收[[硝酸根|硝酸根]]离子或[[铵|铵]]离子以获取氮素。动物体内的所有氮素则均由在[[食物链|食物链]]中进食植物所获得。

== 氨 ==
{{seealso|魚菜共生}}
氨来源于[[腐生生物|腐生生物]]对死亡动植物器官的分解，被用作制造[[銨根|銨根]]离子（NH<sub>4</sub><sup>+</sup>）。在富含氧气的土壤中，这些离子将会首先被亚硝化细菌转化为[[亚硝酸根|亚硝酸根]]离子（NO<sub>2</sub><sup>−</sup>），然后被硝化细菌转化为硝酸根离子（NO<sub>3</sub><sup>−</sup>）。铵的两步转化过程被叫做[[硝化作用|硝化作用]]。

氨对于[[鱼类|鱼类]]来说有剧毒，因此必须对[[废水处理厂|废水处理厂]]排放到水中的氨的浓度进行严密的监控。为避免鱼类死亡的损失，应在排放前对水中的铵进行硝化处理，在陆地上为硝化细菌通风提供氧气进行硝化作用成为一个充满吸引力的解决办法。

== 硝酸盐危害 ==
氨离子很容易被固定在土壤中（尤其是[[腐殖质|腐殖质]]和[[黏土|粘土]]）。而[[硝酸根|硝酸根]]离子和[[亚硝酸根|亚硝酸根]]离子则因它们自身的负电性而更不容易被固定在正离子的交换点（主要是腐殖质）多于负离子的土壤中。在强降雨后或过量灌溉后，硝酸根和亚硝酸根移动到[[地下水|地下水]]的情况经常会发生。地下水中，硝酸盐含量的提高关系到饮用水的安全，因为水中过量的硝酸根离子会影响婴幼儿血液中的氧浓度并导致[[正鐵血紅蛋白血症|-{zh-hant:正鐵血紅蛋白血症;zh-hans:高铁血红蛋白症;}-]]或[[蓝婴综合征|蓝婴综合征]]。如果，过量硝酸盐通过径流或地下水进入[[地表水|地表水]]，会导致水体的[[富营养化|-{zh-hans:富营养化;;zh-tw:優養化;zh-hk:水華;}-]]，使得[[藍菌門|蓝藻菌]]和其它[[藻|藻]]类大量繁殖，导致[[水生生物|水生生物]]因缺氧而大量死亡。虽然不像铵一样对鱼类有毒，硝酸盐可通过富营养作用间接影响鱼类的生存。氮素已经导致了一些水体的富营养化问题。从2006年起，在英国和美国使用氮肥将受到更严厉的限制，磷肥的使用也将受到了同样的限制。这些措施被普遍认为是为了治理恢复被富营养化的水体而采取的。
 
在无氧或低氧条件下，[[厌氧细菌|厌氧细菌]]的“[[反硝化反应|反硝化作用]]”将会发生。最终将硝酸中氮的成分还原成氮气归还到大气中去。

== 氮气(N<sub>2</sub>)的转化 ==
有三种将游离态的N<sub>2</sub>（大气中的氮气）转化为化合态氮的方法：
#'''生物固定''' – 一些共生细菌（主要与豆科植物共生）和一些非共生细菌能进行固氮作用并以有机氮的形式吸收。
#'''工业固氮''' – 在[[哈伯-博施法|哈伯-博施法]]中，N<sub>2</sub>与[[氢|氢]]气被化合生成[[氨|氨]]（NH<sub>3</sub>）肥。
#'''化石燃料燃烧''' – 主要由交通工具的引擎和热电站以NOx的形式产生。
另外，闪电亦可使N<sub>2</sub>和O<sub>2</sub>化合形成NO<sub>2</sub>，是大气化学的一个重要过程，但对陆地和水域的氮含量影响不大。

由于[[豆科|豆科]]植物（特别是[[大豆|大豆]]、[[紫苜蓿|紫苜蓿]]和[[苜蓿|苜蓿]]）的广泛栽种以及使用哈伯-博施法生产化学肥料和交通工具和热电站释放的含氮污染成分，人类使得每年进入生物利用形态的氮素提高了不止一倍。这所导致的[[富营养化|-{zh-hans:富营养;zh-tw:優養化;zh-hk:水華;}-作用]]已经对[[湿地|湿地]][[生态系统|生态系统]]产生了破坏。
