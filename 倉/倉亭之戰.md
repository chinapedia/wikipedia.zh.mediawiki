'''倉亭之戰'''，亦稱'''平丘之戰'''，為[[袁紹|袁紹]]和[[曹操|曹操]]繼[[官渡之戰|官渡之戰]]後所對峙之最後一場[[戰役|戰役]]，袁紹在這場[[戰爭|戰爭]]中湊足七萬餘兵力攻曹，曹操因[[漢獻帝|漢獻帝]]之故命北方各路[[諸侯|諸侯]]聚兵五萬攻袁。[[東漢|東漢]][[汉獻帝|汉獻帝]][[建安_(东汉)|建安]]六年（201年）四月，於[[倉亭|倉亭]]大破袁紹，隔年五月袁紹吐血而亡。此战最终决定了[[黄河|黄河]]以北地区的最终归属。

== 背景 ==
{{main|官渡之戰}}
[[汉献帝|汉献帝]][[建安|建安]]五年（200年）八月，袁紹不聽[[沮授|沮授]]的持久作戰之建議，自[[陽武|陽武]]逐漸進逼至[[官渡|官渡]]，依[[沙塠|沙塠]]為屯，東西相隔數十里。九月，曹操與袁紹交戰不利，轉而堅壁拒敵。袁紹建高櫓、壘土山，居高而射，曹兵蒙盾而行。曹操屬下[[劉曄_(三國)|劉曄]]製作[[霹靂車|霹靂車]]，發石擊破櫓樓。袁軍又掘地道，曹軍挖長溝阻絕。曹軍糧草將盡，士卒疲乏，曹操寫信給[[荀彧|荀彧]]，商議要退守[[許都|許都]]，荀彧引[[楚漢相爭|楚漢相爭]]之史勸之<ref>《[[三國志|三國志]]·荀彧傳》：「承尊命使決進退之疑，愚以袁紹悉眾聚於官渡，欲與明公決勝負，公以至弱當至強，若不能制，必為所乘；是天下之大機也。紹軍雖眾，而不能用；以公之神武明哲，何向而不濟？今軍實雖少，未若楚、漢在滎陽、成皋也。公今畫地而守，扼其喉而使不能進，情見勢竭，必將有變。此用奇之時，斷不可失。惟明公裁察焉。」</ref>，曹操遂決心繼續堅守待機，同時令[[徐晃|徐晃]]等將騷擾袁軍的[[補給線|補給線]]。

十月，袁紹派遣[[淳于瓊|淳于瓊]]率兵萬餘護送軍糧，宿於距袁紹大營40里的[[烏巢|烏巢]]（今[[河南|河南]][[封丘|封丘]]西）。沮授建議增派[[蔣奇_(東漢)|蔣奇]]率部於側翼掩護，紹不從。[[許攸|許攸]]獻計輕軍夜襲許都，亦不被採納。許攸之家屬犯法下獄，遂與袁決裂，轉投曹操，並告知袁軍輜重萬餘乘在[[故市|故市]]和[[烏巢|烏巢]]，守備不嚴，勸他輕兵破襲，則袁將自敗。

曹操遂令[[曹洪|曹洪]]、[[荀攸|荀攸]]留守官渡大營，自領步騎五千人，偽裝成袁軍蔣奇部隊，人銜枚馬縛口，攜帶柴草，從小路夜行，迅速趕到烏巢，縱火圍攻，淳于瓊部驚慌混亂。拂曉，淳于瓊才發現曹操兵少，出陣反攻，曹操率軍突擊，淳于瓊又退守營中。袁紹得知軍情，只派輕騎救援淳于瓊，而命令[[張郃|張郃]]、[[高覽|高覽]]重兵攻打曹操官渡大營。張郃認為，烏巢戰場才是戰局關鍵，應全力救援，袁紹不聽。援軍迫近烏巢時，曹操鼓舞士氣，集中兵力，先斬淳于瓊，再破援騎，將袁軍物資全部焚燬。

消息傳至官渡前線，謀士[[郭圖|郭圖]]進讒言陷害張郃、高覽，張、高二人獲知後方有變，率部降曹。於是袁軍完全崩潰，袁紹與長子[[袁譚|袁譚]]僅率兵八百渡河退回北方。

== 經過 ==
[[汉献帝|汉献帝]][[建安|建安]]六年（201年）[[四月|四月]]，袁紹於[[平丘|平丘]]渡河，意圖襲取[[陳留郡|陳留郡]]，再由陳留攻取[[許昌|許昌]]，欲擊敗曹操。曹操一度欲趁袁紹新敗南攻[[劉表|劉表]]，謀士[[荀彧|荀彧]]準確判斷了袁紹軍的意圖，為免袁紹死灰復燃，於是建議曹操乘官渡之勢主動北上迎擊，遂破袁绍仓亭军。<ref>《[[資治通鑑|資治通鑑]]·卷六十四》：「曹操就谷於安民。以袁紹新破，欲以其間擊劉表。荀彧曰：『紹既新敗，其眾離心，宜乘其困，遂定之。而欲遠師江、漢，若紹收其餘燼，承虛以出人後，則公事去矣。』操乃止。夏，四月，操揚兵河上，擊袁紹倉亭軍，破之。」</ref>

== 結果 ==
兩次戰勝後，曹操威震天下，而袁紹元氣大傷，已無力與之爭衡。建安七年（202年），袁紹憂病而死，[[袁譚|袁譚]]、[[袁尚|袁尚]]爭位，[[河北|河北]]大亂。曹操乘亂進取，於[[建安|建安]]十二年（207年）統一北方。

== 演義情節 ==
於[[羅貫中|羅貫中]]所著之歷史[[章回小說|章回小說]]《[[三國演義|三國演義]]》中，情節大致與正史雷同，唯增加了[[程昱|程昱]]獻[[十面埋伏|十面埋伏]]之計等橋段，以增加小說之可看性。曹操主動撤退到[[黃河|黃河]]邊，並用[[許褚|許褚]]引誘袁軍發動攻擊，因曹方[[背水一戰|背水一戰]]，手下將士歷經百戰精銳，再次將袁軍擊敗。袁紹敗退時，十支伏兵依次殺出，袁軍開始潰散。曹操再次擊敗袁紹，將袁紹徹底擊散。<ref>《[[三國演義|三國演義]]·三十一回》：曹操倉亭破本初 玄德荊州依劉表</ref>
