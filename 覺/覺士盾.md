'''覺士盾'''（Cox Plate）是澳洲[[滿利谷馬場|滿利谷馬場]]舉行的賽馬比賽，是2040米草地國際一級賽，供三歲及以上馬匹參與的分齡讓磅賽事。1922年首次舉行，此項紀念滿利谷馬場創辦人威廉·森美·覺士。總獎金達300萬[[澳元|澳元]]，是澳洲最高獎金的分齡讓磅賽<ref name=history>{{cite web |url= http://www.theage.com.au/opinion/society-and-culture/moonee-valley-signing-its-death-warrant-20110406-1d4eg.html |title=Developer Proposes Housing Plan For Moonee Valley Racecourse |author=Andrew Lemon |date=April 7, 2011 |work=The Age |accessdate=8 April 2011}}</ref>。1999年至2005年曾經是世界巡迴錦標賽其中一場賽事之一。近年在[[墨爾本盃|墨爾本盃]]前十天進行，是滿利谷馬場馬場春季最重要的賽事。

== 讓磅準則 ==
南半球五歲或以上雄馬負59公斤，四歲雄馬負57.5公斤，三歲雄馬負49.5公斤。<ref>{{cite web |title=2018 LADBROKES COX PLATE |url=https://cdn.racing.com/-/media/rv/2018-src/international/pdfs/2018-ladbrokes-cox-plate--terms--conditions-140618.pdf?la=en |publisher=cdn.racing.com}}</ref>北半球四歲或以上雄馬負59公斤，三歲雄馬負56公斤，兩歲雄馬負46.5公斤。雌馬再受讓2公斤。不設見習騎師減磅。<ref>{{cite web |title=2017 LADBROKES COX PLATE |url=https://cdn.racing.com/-/media/rv/files/international-src/2-ladbrokes-cox-plate-conditions-2017-final.pdf?la=en |website=cdn.racing.com}}</ref>
