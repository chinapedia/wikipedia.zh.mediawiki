'''柯西數'''（'''Ca'''）是[[流體力學|流體力學]]中有關{{link-en|可壓縮流|Compressible flow}}的[[無量綱|無量綱]]，得名自法國數學家[[奧古斯丁·路易·柯西|奧古斯丁·路易·柯西]]。當可壓縮性有顯著影響時，在考慮動態相似性的慣性力時，也需要考慮[[彈力|彈力]]，柯西數是流體慣性力和可壓縮力（彈力）比例，可以表示如下：

:<math> \mathrm{Ca} = \frac{\rho v^2}{K}</math>,

其中
: <math>\rho</math>為流體密度（[[國際標準制|國際標準制]]單位：[[公斤|kg]]/[[公尺|m]]<sup>3</sup>）
: ''v''為局部的流體速度（國際標準制單位：m/[[秒|s]]）
: ''K'' 為[[體積模量|體積模量]]（國際標準制單位：[[帕斯卡|Pa]]）

==柯西數和馬赫數的關係==

在[[等熵流|等熵流]]中柯西數可以用[[馬赫數|馬赫數]]表示，等熵體積模量<math> K_s = \gamma p</math>，其中<math>\gamma</math>為[[熱容比|熱容比]]，而''p''為流體壓力。
若流體為[[理想氣體|理想氣體]]，則依照[[理想氣體定律|理想氣體定律]]

:<math> K_s = \gamma p = \gamma \rho R T = \,\rho a^2</math>,

其中

: <math>a = \sqrt{\gamma RT}</math>為聲速（國際標準制單位：m/s）
: ''R'' = [[氣體常數|氣體常數]]（國際標準制單位：[[焦耳|J]]/(kg [[克爾文|K]])）
: ''T'' = 溫度（國際標準制單位：K）

取代柯西數公式中的''K''（''K<sub>s</sub>''），可得

:<math> \mathrm{Ca} = \frac{v^2}{a^2} = \mathrm{M}^2</math>.

因此在[[理想氣體|理想氣體]]的等熵流中，柯西數是馬赫數的平方。
