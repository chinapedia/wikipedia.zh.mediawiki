'''柯蒂斯·迈克尔·“麦克”·斯卡帕洛蒂'''（{{lang-en|'''Curtis Michael "Mike" Scaparrotti'''}}，{{bd|1956年|3月5日|||CatIdx=Curtis Michael Scaparrotti}}），现为[[北大西洋公約組織|北大西洋公約組織]][[歐洲盟軍最高司令部|歐洲盟軍最高司令部]]司令、[[美国欧洲司令部|美国欧洲司令部]]司令，之前担任[[联合国军|联合国军]]、[[驻韩美军|驻韩美军]]、韓美聯合司令部司令。

==军事生涯==


# 斯卡帕洛蒂是俄亥俄州洛根人，于1978年从[[美国军事学院|西点]]毕业获少尉军衔，被分配到北卡罗来纳[[布拉格堡|布拉格堡]]，在[[第82空降师|第82空降师]]325步兵团3营担任步兵排和反坦克排排长，之后升任作战参谋、连长。
# 1984年，斯卡帕洛蒂在佐治亚[[本宁堡|本宁堡]]完成了步兵军官高级课程，随后，在[[南卡罗来纳大学|南卡罗来纳大学]]获得行政教育硕士学位。
# 1985年，回到西点，任技术参谋和校长副官，直至1988年。之后前往堪萨斯[[利文沃思堡|利文沃思堡]]在[[美国陆军指挥参谋学院|美国陆军指挥参谋学院]]继续从事军事研究。
# 1989年7月，分配到[[纽约德拉姆堡|纽约德拉姆堡]]，在[[第10山地师|第10山地师]]87步兵团1营任作战参谋，随后在师部任作战科长。
# 1992年到1994年，在[[陆军总人事司令部|陆军总人事司令部]]和[[陆军参谋长室|陆军参谋长室]]任职。
# 1994年5月，前往意大利[[维琴察|维琴察]]，在[[南欧特遣队|南欧特遣队]]325空降战斗队第3营担任营长，此期间，曾参与在扎伊尔、卢旺达、波斯尼亚和黑塞哥维那和利比里亚等地的国际军事行动。
# 1996年，在第10山地师，担任作战参谋，并在位于宾夕法尼亚州[[卡莱尔兵营|卡莱尔兵营]]的[[美国陆军战争学院|美国陆军战争学院]]继续他的研究工作。
# 1998年，在陆军G3/5副参谋长办公室，任办公组长。
# 1999年，回到初任军官的布拉格堡，任第82空降师2旅旅长。
# 2001年到2003年，在[[联合参谋部|联合参谋部]]作战部（J3）任副主任。
# 2003年7月至2004年7月，参加[[伊拉克自由行动|伊拉克自由行动]]，担任[[第1装甲师|第1装甲师]]分管机动的副师长。
# 2004年8月至2006年7月，在母校西点担任学员队司令。
# 此后，再次回到中东，[[中央司令部|中央司令部]]任作战作战总监（J3）。
# 2008年10月1日，担任第82空降师师长，并率部部署至阿富汗东部，部队重组为敌82联合特遣队，任司令。
# 2010年10月15日，回到华盛顿[[刘易斯堡|刘易斯堡]]，任[[第1军|第1军]]司令，兼[[刘易斯麦科德联合基地司令|刘易斯麦科德联合基地司令]]。2011年7月11日开始，兼任[[驻阿富汗国际维和部队|驻阿富汗国际维和部队]]司令部司令，和[[驻阿美军|驻阿美军]]副司令。
# 2012年7月， 任联合参谋部主任。
# 2013年8月， 担任[[联合国军|联合国军]]司令、[[美韩联军|美韩联军]]司令、[[驻韩美军|驻韩美军]]司令。
# 2016年5月， 担任[[美国欧洲司令部|美国欧洲司令部]]司令、[[北約|北約]][[歐洲盟軍最高司令部|歐洲盟軍最高司令部]]司令。
