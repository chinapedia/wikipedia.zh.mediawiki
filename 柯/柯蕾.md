'''柯蕾'''（{{lang-en|Colette}}，{{bd|1975年|5月27日}}），全名'''柯蕾·瑪利諾'''，是[[美國|美國]][[伊利諾州|伊利諾州]][[芝加哥|芝加哥]]的[[浩室音樂|浩室音樂]][[DJ|DJ]]與歌手。她演出的足跡遍及世界各地，包括[[劍橋|劍橋]]、[[新加坡|新加坡]]、[[卡加利|卡加利]]、[[好萊塢|好萊塢]]與[[蒙特婁|蒙特婁]]。柯蕾曾被與[[凱莉·米洛|凱莉·米洛]]<ref name="imeem">{{cite web |url=http://djcolette.imeem.com/ |title=IMEEM 簡介}}</ref>、[[黛比·哈瑞|黛比·哈瑞]]<ref name="chicagomag">{{cite web |url=http://www.chicagomag.com/Chicago-Magazine/December-2007/You-Should-Know-DJ-Colette/ |publisher=芝加哥雜誌 |title=You Should Know...DJ Colette}}</ref>等人相比，並被譽為[[OM唱片|OM唱片]]旗下的浩室第一夫人<ref name="imeem" />。
