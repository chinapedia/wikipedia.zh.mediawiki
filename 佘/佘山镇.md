'''佘山镇'''（[[漢語拼音|漢語拼音]]：shé shān zhèn），是{{PRC admin/fullname|31/01/17/103/000|3}}下辖的一个[[乡级行政区|乡级行政区]]<ref>{{PRC admin/ref sgc|{{PRC admin/parent|31/01/17/103/000}}}}</ref>，因境内有[[佘山|佘山]]而得名<ref name="第二章街道、镇、乡">
{{Cite web |url = http://www.shtong.gov.cn/newsite/node2/node2245/node70862/node70868/node70908/index.html |title = 第二章街道、镇、乡|publisher =上海市地方志办公室 |language = zh-cn |accessdate = 2014-08-25 |format=html |quote = 以山名得名，如松江县佘山镇、天马山镇；}}</ref>。佘山於1955年5月前屬青浦縣，後劃入松江縣建佘山區，1957年撤區建鄉，1958年建人民公社，1984年改設為鄉，1986年6月改為鎮<ref name="松江县志 佘山乡">{{Cite web |url = http://www.shtong.gov.cn/newsite/node2/node4/node2250/songjiang/node46165/node46186/node46188/userobject1ai32358.html |title = 松江县志 - 第一卷建置 - 第三章乡镇 - 第十二节　佘山乡 |author = 上海市松江县地方史志编纂委员会 |date = 1991-08 |publisher = 上海市地方志办公室 |language = zh-cn |accessdate = 2014-12-01 |format= html |quote = 乡人民政府设在[[陈坊桥|陈坊桥镇]]。该地区原属[[青浦县|青浦县]]，1955年5月划入本县，成立[[佘山区|佘山区]][[人民政府|人民政府]]。1957年撤区并乡后为佘山乡。1958年建立佘山[[人民公社|人民公社]]。1984年改设[[佘山乡|佘山乡]]。1986年6月改为镇建制。 |archiveurl = https://web.archive.org/web/20141201171548/http://www.shtong.gov.cn/newsite/node2/node4/node2250/songjiang/node46165/node46186/node46188/userobject1ai32358.html |archivedate= 2014-12-01
}}</ref>，至2001年1月[[天馬山鎮|天馬山鎮]]併入佘山鎮<ref name="松江区历史沿革">{{Cite web|url =http://www.xzqh.org/html/2008/0428/4972.html|title =松江区历史沿革|author =|date =2009-08-06|publisher =行政区划网|language =zh-cn|accessdate =2014-05-04|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20131203132804/http://www.xzqh.org/html/2008/0428/4972.html|archivedate =2013-12-03}}</ref>。面积55.70平方公里。户籍人口3.30万人（2008年）。

== 旅遊景觀 ==
'''上海佘山國家旅遊度假區'''是于1995年8月由[[国务院|国务院]]批准建立的[[国家旅游度假区|国家旅游度假区]]，其中[[中华人民共和国林业部|国家林业部]]批准建立的[[佘山国家森林公园|佘山国家森林公园]]占全区总面积的8.7%，是区内重要的旅游项目之一。<ref name="第三节　佘山国家森林公园">{{Cite web|url =http://shtong.gov.cn/newsite/node2/node2245/node69854/node69860/node69918/node69933/userobject1ai69599.html |title =第三节　佘山国家森林公园|publisher =上海市地方志办公室|language =zh-cn|accessdate = 2014-08-25 |format=html |quote =1993年5月，国家林业部批准建立包括12座山头在内的佘山国家森林公园。公园采取与上海市纺织品进出口公司联合开发的办法，筹措建园资金1000多万元，在东佘山修复和辟建景点，东佘山景区规划由程绪珂、周在春等负责。经过整修后的东佘山园于1994年11月首先开放，与松江县几个单位及佘山镇共同投资建设的佘山客运索道也同时投入运行。1995年5月，与佘山天主教堂、上海天文台等景点组成的西佘山园开放。<br>1995年8月，国务院批准建立面积为46.26平方公里的佘山国家旅游度假区，森林公园占全区总面积的8.7%，是区内重要的旅游项目之一。}}</ref>度假区内包含有[[佘山国家森林公园|佘山国家森林公园]]（包括[[佘山天文台|天文台]]、[[佘山进教之佑圣母大殿|进教堂]]、[[上海地震科普馆|上海地震科普馆]]）、[[辰山_(松江)#上海辰山植物园|上海辰山植物园]]、[[上海欢乐谷|欢乐谷]]、[[上海玛雅海滩水公园|瑪雅水上樂園]]、[[上海佘山国际高尔夫俱乐部|佘山高尔夫]]、[[月湖公园|月湖公园]]等旅游景点。
