'''淡水清水巖'''，又稱'''淡水祖師廟'''，位於[[臺灣|臺灣]][[新北市|新北市]][[淡水區|淡水區]]清水街八十七號。建於[[日治|日治]][[昭和|昭和]]十二年（1937年）。主祀[[閩南|閩南]][[安溪|安溪]]高僧[[清水祖師|清水祖師]]，為一座受[[道教|道教]]與[[臺灣民間信仰|臺灣民間信仰]]影響極多的[[佛教|佛教]][[祖師廟|祖師廟]]，是淡水地方上的信仰中心。每年農曆五月初五、初六舉辦「清水祖師得道紀念日繞境慶典」，俗稱「淡水大拜拜」，是淡水的盛事<ref>[https://www.ntpc.gov.tw/ch/home.jsp?id=28&parentpath=0,6,27&mcustomize=news_view.jsp&dataserno=201705310010&mserno=201309160001 淡水清水祖師得道紀念日 朱立倫前往祈求 台灣風調雨順國泰民安].新北市政府新聞局.2017-05-31</ref>。

淡水清水巖與[[艋舺祖師廟|艋舺祖師廟]]、[[三峽祖師廟|三峽祖師廟]]、[[瑞芳祖師廟|瑞芳祖師廟]]，稱為「大臺北四大祖師廟」<ref>[https://www.peopo.org/news/368243  淡水清水巖祖師廟].PeoPo公民新聞.2018-05-29</ref>。清水巖、[[鄞山寺|鄞山寺]]、[[淡水福佑宮|福祐宮]]與[[淡水龍山寺|淡水龍山寺]]合稱為「淡水四大廟」<ref>李雅雯.[http://news.ltn.com.tw/news/local/paper/796733 〈台北都會〉列文資？ 淡水清水巖祖師廟意願低].自由時報.2014-07-17</ref>。

== 祀神 ==
[[清水祖師|清水祖師]]、[[地藏菩薩|地藏菩薩]]、[[保生大帝|保生大帝]]、[[西秦尊王|西秦尊王]]、[[蕭望之|蕭府千歲]]、[[文昌帝君|文昌帝君]]、[[斗姥元君|斗姥元君]]、[[太歲星君|太歲星君]]。

== 源由 ==
相傳[[安溪清水巖|安溪清水巖]]在[[宋代|宋代]]得到一塊巨大的沈香木，雕製六尊清水祖師神像，後代稱為「六古佛」。[[清文宗|清文宗]][[咸豐|咸豐]]年間（1851年－1861年），一[[僧|僧]]自[[泉州府|泉州府]][[安溪清水巖|安溪清水巖]]，奉一尊「古佛」來臺，在[[臺北|臺北]][[淡水區|淡水]]一帶向人[[化緣|化緣]]重修安溪清水巖，尊為「蓬萊老祖」。孰料緣金失竊，淡水[[仕紳|仕紳]]翁種玉、翁瑞玉兄弟慷慨解囊，給予該僧四百元[[鷹洋|鷹洋]]資助之，此僧返回唐山前，此尊留於淡水奉祀。參拜者眾，翁氏兄弟請神於東興街（今[[淡水區|淡水區]]中正路）約五十[[坪|坪]]的平房「濟生號」奉祀。

另說則是一名[[安溪清水巖|安溪清水巖]]攜清水祖師神像「蓬萊老祖」，來淡水興福寮向安溪籍民眾化緣重修清水巖，[[興福寮|興福寮]]有數名挑運[[菅芒|菅芒]]草的[[挑夫|挑夫]]因為連續遇到該僧三日，認為是一種緣分，於是發起募捐，眾人當日將賣三十餘擔菅芒草的所得[[寄付|寄付]]予該僧，並要求該僧將清水祖師留在興福寮供奉三日，祈禱平安。三日期滿，該僧於是要將神像請回，向祖師[[擲筊|擲筊]]起駕，卻連擲十幾個「笑杯」，無法獲得[[神示|神示]]同意離開，眾人稱祖師要留在臺灣大顯神威，於是當地仕紳翁種玉、翁瑞玉兄弟，以「[[翁姓|翁]]、[[龚姓|龚]]、[[方姓|方]]、[[洪姓|洪]]、[[江姓|江]]、[[汪姓|汪]]」[[六桂堂|六姓檀越]]名義，請神於東興街（今[[淡水區|淡水區]]中正路）濟生號奉祀。

== 神蹟 ==
清[[同治|同治]]六年（[[1867年|1867年]]），此尊「蓬萊老祖」神像在一次於今[[石門區|石門區]]出巡時，顯靈落鼻示警，人稱'''落鼻祖師'''，村民驚相走報，人人離家圍觀。突然發生[[地震|地震]]，全村屋毀，由於村民盡出，村中無人，竟無人傷亡，而祖師更受民眾崇敬。在[[清德宗|清德宗]][[光緒|光緒]]十年（1884年）10月，[[清法戰爭|清法戰爭]][[西仔反|西仔反]]時，[[法國|法國]][[軍隊|軍隊]]進攻淡水。淡水人迎出清水祖師助戰，竟然大勝[[法軍|法軍]]。<ref>[https://nchdb.boch.gov.tw/assets/overview/folklore/20130130000002 淡水清水巖清水祖師遶境].文化部文化資產局 </ref>據說祖師顯靈，本地屋頂有和尚護衛，使法軍懼而撤退，鄉民普信是祖師顯神威<ref name="神明">{{cite journal|author1=黃其豪|title=神明落鼻 傳說預告災厄|url=http://news.ltn.com.tw/news/society/paper/307951|work=自由時報|date=2009-06-02}}</ref>。

到了[[二次大戰|二次大戰]]，受到[[盟軍|盟軍]]轟炸，有二枚掉到祖師廟的未爆彈，信眾歸因於祖師護祐，清除[[火藥|火藥]]後，改成燭臺，置於正殿兩側紀念<ref name="神明"/>。

== 紛爭 ==
清[[光緒|光緒]]十年（1884年），[[西仔反|西仔反]]中，淡水人迎清水祖師佛像到戰場祈福，後清兵順利擊退法軍，卻引發了淡水與艋舺兩派的爭執，淡水人認為這尊顯靈的清水祖師神像（落鼻祖師），本為淡水所有，只是因為淡水並無建造祖師廟，卻又獲[[清代|清代]][[光緒帝|光緒帝]][[詔|詔]]贈「功資拯濟」的[[敕額|敕額]]，只好把神像及敕額輾轉安置於艋舺；而[[艋舺|艋舺]]方面則認為這尊顯靈的清水祖師是由[[艋舺祖師廟|艋舺祖師廟]]借去的，造成了淡水與艋舺兩派信徒的大糾紛，[[日治時期|日治時期]]經[[官府|官府]]協調，由艋舺與淡水二廟輪流奉祀。

== 建廟 ==
清水祖師神威顯赫，然久而無廟，僅僅奉祀於民宅，淡水人久慕之。[[臺灣日治時期|臺灣日治時期]]昭和六年（1931年）成立「淡水清水巖祖師廟興建委員會」，同年10月1日《[[臺灣日日新報|臺灣日日新報]]》〈淡水祖師廟興工建廟〉，報導了淡水清水巖建廟鳩資：

{{cquote|淡水街崇祀清水祖師，神靈顯赫，信仰者甚多。惟廟宇係以民家充用，且甚狹窄，故每年祭典之日，參拜者輒擁擠不開，交通為之杜絕。於是該地士商每次提議別建廟宇，以壯觀瞻，然久議未果。者番，幸[[李文珪|李文珪]]氏等熱心鼓舞，擇定廟地，請官批認，遂於中秋前二日，由發起人等，邀集該地有志數十名，開建築磋商會於信用組合樓上，先由管理人陳有生氏起述開會辭，然後選任建築委員，[[許丙|許丙]]氏為委員長，[[汪明燦|汪明燦]]氏為副。其他[[盧阿山|盧阿山]]、[[李文珪|李文珪]]、[[吳傳宗|吳傳宗]]、[[雷俊臣|雷俊臣]]……諸氏，皆為常務委員，即時協議建築諸事宜，滿場贊成通過。建築事務所設於淡水信用組合內，四方善信，有志[[香火錢|寄附]]者，可向該事務所報明-{云}-。}}

[[昭和|昭和]]七年（1932年）淡水士紳李文珪、[[許丙|許丙]]等，開始將附近音樂團體「淡水軒」的[[西秦王爺|西秦王爺]]廟、碼頭[[工人|工人]]祭祀的[[蕭望之|富美蕭王爺]]廟拆解，將神合併於清水巖奉祀<ref>淡水清水巖管理委員會.[http://www.newtaipeiheritage.tw/Item/Detail/56d6c052-bd87-4bb1-b177-308587e53599 淡水清水巖清水祖師遶境].新北市文化資產數位學習網.2017-11-07 </ref>，昭和十二年（1937年）完工，廟貌莊嚴宏大<ref name="好廟">{{cite journal|1=|title= 廟宇通 淡水清水巖|url=https://miao.temple01.com/about.php?tpl=d10b377|work=臺灣好廟網}}</ref>。

== 風水 ==
傳說淡水清水巖廟地極佳：廟前[[大屯山|大屯山]]、[[觀音山|觀音山]]互相扶持，而[[淡水河|淡水河]]位於兩山之間，似一[[金龍|金龍]]入廟，為[[堪輿學|堪輿學]]中的大吉之穴<ref name="好廟"/>，廟方說這塊廟地是清水祖師[[神諭|神諭]]欽選的，祖師神案的座址為龍首，到廟中大門右側大鐘下方有一塊圍著紅布的石頭，廟方認為這顆石頭就是龍珠<ref>[https://bobee.nownews.com/20171204-16129 羅漢降龍大吉利　祖師指點淡水清水巖].保庇NOW.2017-12-04</ref>。

本穴昔年被[[北臺灣|北臺灣]]風水大師[[林琅|林琅]]評論為「[[羅漢|羅漢]]降龍」之穴，林琅預言有一位大禪師前來本地弘法傳教，弟子甚眾，[[禪師|禪師]]施展[[神通|神通]]，驅逐番兵後，又雲遊幾十年，有[[檀越|檀越]]為他設立[[伽藍|伽藍]]，得以在[[滬尾|滬尾]]廣施佛法。一直到[[日治時期|日治時期]]，鄉民才知道林琅說的高僧，不是甚麼[[武僧|武僧]]或[[僧兵|僧兵]]，是清水祖師「蓬萊老祖」的神像，番兵是指[[西仔反|西仔反]]的[[法軍|法軍]]，「伽藍」是指淡水祖師廟<ref>[https://www.ca.ntpc.gov.tw/ImageAndArticle/Details/46?wnd_id=131 淡水清水巖]，新北市政府民政局 </ref>。

== 藝文 ==
淡水清水巖請來了當時名家[[廖石成|廖石成]]、[[郭塔|郭塔]]分建前後殿。而龍柱更具有古風，一柱一龍，龍首自底部躍起，龍身向下，更顯威猛。石雕的部分，由[[張火廣|張火廣]]造，火廣卒於任內，其子[[張木成|張木成]]續雕，多為《[[封神榜|封神榜]]》之人物，技巧高妙。木雕則由[[黃龜理|黃龜理]]、剪黏由[[陳天乞|陳天乞]]展現其不凡技藝。

祖師廟的[[對聯|對聯]]、[[書法|書法]]亦皆出於一時名士、名書法家之手。這使[[清水巖|清水巖]]不但成為淡水的信仰中心，更成為匯集淡水宗教[[藝術|藝術]]的殿堂。

2004年[[檀越|檀越]]黃文雄[[還願|還願]]，贈獻一頂以20兩黃金打造，鑲有[[紅寶石|紅寶石]]與[[藍寶石|藍寶石]]，價值新臺幣40餘萬元的五佛帽給「鎮殿祖」，而後黃文雄夫人又打造了一頂給「蓬萊老祖」。不久檀越洪家生，為感謝祖師庇佑，也打造了一頂[[黃金|黃金]]帽奉獻祖師，洪家生夫人也跟進，而淡水祖師廟之金帽也已有四、五頂之譜，戴上金帽之祖師神像，常於祭典時巡行淡水，頗見神威<ref name="金冠">{{cite journal|author1=黃其豪|title=〈台北都會〉淡水文藝節 神明戴金冠|url=http://news.ltn.com.tw/news/local/paper/73482|work=自由時報|date=2006-05-26}}</ref>。

== 祭典 ==
[[舊曆|舊曆]]正月初六，是清水祖師誕辰。五月初六是祖師成道，原先只是普通祭典，由於日治時期曾經傳出[[瘟疫|瘟疫]]，因淡水地區的鄉民最崇信清水祖師，故迎請清水祖師在每年[[舊曆|舊曆]]五月初五傍晚開始舉行「[[暗訪|暗訪]]」，五月初六中午開始展開日間遶境，巡遊全區，希望能夠驅逐瘴癘、[[瘟神|瘟神]]，庇佑平安。[[官方|官方]]名為「淡水清水巖清水祖師遶境」，俗稱「淡水大拜拜」。2013年新北市政府已將「淡水大拜拜」，列為「無形文化資產」。相較之下，淡水另一個公廟[[淡水福佑宮|淡水福佑宮]]的「媽祖遶境」範圍，與清水祖師遶境相近，但[[媽祖|媽祖]]的繞境並不是境內所有[[角頭|角頭]]廟都會參與，更突顯出清水祖師作為淡水全境[[境主|境主]]神祇的代表性，以及清水祖師遶境活動凝聚地方的重要意義<ref>[https://nchdb.boch.gov.tw/assets/overview/folklore/20130130000002 淡水清水巖清水祖師遶境].文化部文化資產局</ref>。

清水祖師的各尊神像中，以「蓬萊老祖」最受檀越崇敬，淡水清水巖與[[艋舺清水巖|艋舺清水巖]]兩廟在日治時期為了神尊歸屬，還打官司，後來才決定兩方共有，輪流供奉，農曆單月在艋舺，雙月在淡水，因應淡水大拜拜，五月、六月份兩廟互換供奉月份，2016年起，「蓬萊老祖」不再接受其他廟宇迎請<ref>李雅雯.[http://news.ltn.com.tw/news/local/paper/1006015 〈台北都會〉蓬萊老祖神尊 不再掛單請出廟].自由時報.2016-06-30</ref>。
