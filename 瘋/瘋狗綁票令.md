《'''瘋狗綁票令'''》（{{lang-en|'''''Seven Psychopaths'''''}}）是一部2012年的[[黑色幽默|黑色幽默]]犯罪電影，由[[馬丁·麥多納|馬丁·麥多納]]執導和編劇，領銜主演為[[科林·法瑞尔|科林·法瑞尔]]、[[山姆·洛克威尔|山姆·洛克威尔]]、[[伍迪·哈里森|伍迪·哈里森]]、[[克里斯托弗·沃肯|克里斯托弗·沃肯]]，配角有[[汤姆·威茨|汤姆·威茨]]、[[艾比·寇尼許|艾比·寇尼許]]、[[欧嘉·柯瑞兰寇|欧嘉·柯瑞兰寇]]和{{link-en|柴爾柯·伊凡奈克|Željko Ivanek}}。

《瘋狗綁票令》的首映日期為2012年9月7日（[[2012年多倫多國際影展|多倫多國際電影節]]）<ref name="SP-TIFF">{{cite web |author=Ferguson, Liz |url=http://blogs.montrealgazette.com/2012/09/13/tiff-2012-seven-psychopaths-is-that-a-great-name-for-a-film-or-what/seven-psychopaths-premiere-2012-toronto-international-film-festival-2 |title='Seven Psychopaths' Premiere – 2012 Toronto International Film Festival |work=MontrealGazette.com |publisher=Postmedia Network Inc |date=13 September 2012 |accessdate=27 September 2012 |deadurl=yes |archiveurl=https://web.archive.org/web/20131214174601/http://blogs.montrealgazette.com/2012/09/13/tiff-2012-seven-psychopaths-is-that-a-great-name-for-a-film-or-what/seven-psychopaths-premiere-2012-toronto-international-film-festival-2/ |archivedate=2013年12月14日 }}</ref>，美國和加拿大為2012年10月12日，英國為2012年12月7日<ref name="ReleaseDates">{{cite web |url=http://www.imdb.com/title/tt1931533/releaseinfo |title=Release dates for 'Seven Psychopaths' |work=IMDb |publisher=Amazon.com |accessdate=27 September 2012}}</ref>，香港為2013年4月11日。

美國[[洛杉磯|洛杉磯]]為主要拍攝場地。這個故事主要綁架狗，為[[西施|西施]]。
