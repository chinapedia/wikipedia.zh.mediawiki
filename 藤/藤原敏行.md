'''藤原敏行'''（{{jpn|j=藤原 敏行|hg= ふじわら の としゆき|rm=Fujiwara no Toshiyuki}} ，生年不詳，约卒于907年（一说901年））是日本[[平安時代|平安時代]]初期的[[歌人|歌人]]、書法家。[[藤原南家|藤原南家]][[藤原巨勢麻呂|藤原巨勢麻呂]]的后裔，陸奥出羽按察使[[藤原富士麻呂|藤原富士麻呂]]之子。官位为从四位上右兵衛督。他是[[三十六歌仙|三十六歌仙]]中一员。
== 生平 ==
[[贞观_(清和天皇)|貞观]]8年（866年）任[[内記|少内記]]。之后历任图書頭、[[因幡国|因幡]]守、右兵衛権佐等职。[[元慶|元慶]]6年（882年）升从五位上，[[仁和|仁和]]2年（886年）任右近衛少将。后任右近衛中将，[[宽平|宽平]]7年（895年）任藏人頭，宽平9年（897年）官至从四位上右兵衛督。

== 书法 ==
[[小野道風|小野道風]]将藤原敏行与[[空海|空海]]并列为古今最高的书法家。

据《渤海国志长编》卷二载：“宽平四年（892年）六月丙申，遣渤海敕书，令左近少将藤原敏行朝臣书之。辛丑，太政官赐渤海国牒二通，一者令左近卫少将藤原敏行朝臣书之”，
“（敏行）历仕贞观以来五朝，能倭歌，且有能书之名，多写佛经，且其墨痕传于渤海。”

== 代表和歌 ==
[[File:Hyakuninisshu 018.jpg|thumb|200px|藤原敏行
----[[百人一首|百人一首]]第18番・藤原敏行朝臣]]
作为勅撰歌人，《[[古今和歌集|古今和歌集]]》收录其和歌18首，后来的[[勅撰和歌集|勅撰和歌集]]另收28首。个人歌集为《敏行集》。

== 逸話 ==
据《[[宇治拾遗物语|宇治拾遗物语]]》记载，藤原敏行受众人邀抄写二百余部《[[法华经|法华经]]》，因在抄写时曾吃过鱼，以不净之身抄写佛经，死后堕入地狱受苦。
