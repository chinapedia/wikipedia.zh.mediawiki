'''藤田麻衣子'''（{{bd|1984年|1月16日|catIdx=Fujita, Maiko}}）為[[日本|日本]][[愛知縣|愛知縣]][[名古屋市|名古屋市]]綠區<ref name="fujita"/>出身的[[創作歌手|創作歌手]]。經紀公司為GOOD-DAY。

== 簡歷 ==
2006年9月以單曲「恋に落ちて」出道，現在共發行了10張單曲及5張專輯。所有的歌曲均由本人親自作詞作曲，歌曲多為抒情曲或打氣曲。除了為自己創作外，也有為其他藝人提供樂曲。

在2012年2月，「私らしく」被選定為[[日本公文教育研究会|KUMON]]的廣告歌。在4月，「ねぇ」被起用為電視動畫『[[緋色的碎片|緋色的碎片]]』的主題曲，這亦是藤田首次擔任電視動畫的主題曲。隨即在10月，「高鳴る」亦被起用為『緋色的碎片 第二章』的主題曲。在2012年的巡迴演出，所有公演場次的門票均售罄，隨後於2013年7月在NHK Hall再次舉辦演唱會。在2013年10月11日，藤田於[[日本武道館|日本武道館]]舉辦獨奏獨唱的一人音樂會<ref>{{cite web |url=http://fujitamaiko.com/budokan/ |title=存档副本 |accessdate=2014-07-15 |deadurl=yes |archiveurl=https://web.archive.org/web/20131101081718/http://fujitamaiko.com/budokan/ |archivedate=2013-11-01 }} 藤田麻衣子公式ホームページ 10月11日 武道館弾き語りワンマンライブ</ref>，這在獨立音樂人之中非常罕見。

於2014年1月16日發表了移籍到Victor Entertainment的消息，並決定商業出道。在3月26日發行了商業出道單曲「涙が止まらないのは」<ref>http://www.musicman-net.com/artist/32469.html 藤田麻衣子、武道館LIVEを経て遂にメジャーデビュー発表&Ust生中継 今夜OA （2014年1月18日閲覧）</ref>。

有很多女性歌迷，演唱會有七成觀眾是女性。

擁有[[潔牙師|潔牙師]]執照<ref>{{Cite web|date=|url=http://profile.ameba.jp/maiko-fujita/|title=藤田麻衣子のプロフィール|work=|author=|publisher=サイバーエージェント|accessdate=2013-05-20}}</ref>。

因為主唱遊戲『[[緋色的碎片|緋色的碎片]]』系列的主題曲而獲得遊戲玩家支持。
