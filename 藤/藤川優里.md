'''藤川優里'''{{BD|1980年|3月8日|catIdx=Fujikawa Yuri}}是[[日本|日本]]一位[[女性|女性]][[政治家|政治家]]，現任[[青森縣|青森縣]][[八戶市|八戶市]]市議會議員。藤川是一位無派別的[[自由民主黨_(日本)|自由民主黨]]黨員，但加入了屬於保守派系的「[[自由民主會|自由民主會]]」。

藤川是日本政壇內少有的[[美女|美女]]，有「美女議員」、「美過頭的議員」的稱號。2009年，更擊敗全球美女政客，在[[西班牙|西班牙]]報章「20 Minutos」的網路投票中被選為「全球最美麗的女性政治人物」<ref>[http://n.yam.com/tlt/international/200904/20090420311665.html yam天空新聞－議員藤川優里 全球政壇第一美女]</ref>。

== 簡介 ==
藤川優里出生於一個政治家的家庭，父親[[藤川友信|藤川友信]]是[[自由民主黨_(日本)|自由民主黨]][[津島派|津島派]]政治人物，曾任八戶市議會副議長和津島派領袖[[津島雄二|津島雄二]]的秘書。但在大學畢業前，藤川都沒有參與過政治活動。2003年，藤川自東京返回八戶，負責在宣傳車為父親的選戰宣傳，首次參與政治工作。其後加入八戶市一家老人護理院工作。

2007年4月8日，父親第二次競逐縣議員失敗，藤川當下宣佈出選市議員，受到雙親強烈反對。兩天後，藤川在苦苦懇求下，獲父親支持出選，並開始籌備選舉工作。4月22日，藤川在47人競逐36個席位的選舉中以6962票高票當選，遠遠拋離第二位當選者的3665票<ref>[http://www.senkyo.janjan.jp/election/2007/2/00005967.html 2007年4月八戶市議會選舉結果] {{webarchive|url=https://web.archive.org/web/20090522061624/http://www.senkyo.janjan.jp/election/2007/2/00005967.html |date=2009-05-22 }}</ref>，並成為八戶市議員選舉中得票最高的紀錄<ref name="sina081126">[http://news.sina.com/int/chinanews/105-103-102-106/2008-11-26/19593453534.html 日本美女議員穿泳裝推介旅遊引發爭議]</ref>。此外，藤川以27歲之齡當選，也成為八戶市最年輕的市議員<ref name="sina081126"/>。

藤川上任後成為民生常任委員會和環境及先進都市推進特別委員會委員<ref>藤川優里官方網站</ref>，一直關心當地的發展和護理服務。其後由於[[青森朝日放送|青森朝日放送]]播出她出鏡的影片，因而被日本網民發現而備受追捧，更是許多男生的性幻想對象。而影片播出當天更因為太多人瀏覽（23萬人次）藤川的網誌而導致網誌伺服器當機了。2008年11月19日，藤川與出版商合作推出推廣八戶旅遊的[[DVD|DVD]]「love navi八戶」和[[寫真集|寫真集]]「moe navi八戶」，是首位市議員推出同類產品。產品推出後大受歡迎，其中DVD更登上[[Oricon|Oricon]]每周排行榜的第一位<ref name="sina081126"/><ref>[http://www.oricon.co.jp/news/rankmusic/60440/full/ {{lang|ja|“美人すぎる議員”藤川ゆりのDVDがアイドル・イメージ部門で1位に}}]</ref>。

== 簡歴 ==
* 1995年4月：就讀青森縣立八戶東高等學校人文科（現稱「表現科」）。
* 1998年4月：就讀[[帝京大學|帝京大學]][[文學院|文學院]][[心理學|心理學系]]。
* 2002年4月：大學畢業，在東京以兼職為生（俗稱[[飛特族|飛特族]]）。
* 2003年6月：返回八戶市。
* 2004年2月：加入老人護理院「雲雀之里」。
* 2006年：透過遙距課程合在東京都實習，取得病歷資料管理資格。
* 2007年4月：從護老院辭職出選八戶市議會議員，以最高得票當選。
* 2007年7月：在[[第21屆日本參議院議員通常選舉|第21屆日本參議院議員通常選舉]]支持自民黨，曾與眾議員[[片山五月|片山五月]]一起為競逐連任的青森縣參議員[[山崎力|山崎力]]作聲援演說，惜山崎最終落敗。
* 2008年11月19日：推出DVD「love navi八戶」和寫真集「moe navi八戶」以推廣八戶市旅遊業，是首個市議會議員推出個人DVD和寫真集。曾登上Oricon每周流行榜的第一位。
* 2009年5月：在西班牙報章「20 Minutos」的網路投票中被選為「全球最美麗的女性政治人物」。
* 2010年1月24日：出席自由民主黨黨代表大會，並作為其中一位女性代表朗讀呼籲詞。
