== 简历 ==
藤原绘理香出身于[[日本|日本]][[千叶县|千叶县]]。<ref name=A>{{cite web |title=藤原絵理香 |url=http://www.minnano-av.com/actress137284.html |website=みんなのAV |accessdate=2019-04-17}}</ref>

2009年出道，成为一名熟女系[[AV女优|AV女优]]；同年5月10日，她的第一部AV作品正式发行。<ref name=A/>
