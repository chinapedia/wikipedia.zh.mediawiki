'''藤泽秀行'''（原名'''藤泽保'''，{{bd|1925年|6月14日|2009年|5月8日|catIdx=Fujisawa, Hideyuki}}），[[日本|日本]]著名[[围棋|围棋]]棋手，日本棋界传奇人物，其棋风豪爽，布局尤为精巧被称为“前五十手天下第一”，1950年代至1960年代与“剃刀”[[坂田荣男|坂田荣男]]齐名。获得冠军称号23个，包括[[棋聖戰_(日本圍棋)|棋聖]]六連冠，受封[[名譽棋圣|名譽棋圣]]。

藤泽个性豪爽，拥有众多的棋迷。以自己起名“秀行”、自己颁发“[[段位证书|段位证书]]”等故事为传奇。他以私生活放荡成名，酗酒、赌博和纵欲，三次罹患[[癌症|癌症]]。1992年，藤泽以67岁高龄卫冕[[王座戰_(日本圍棋)|王座戰 (日本圍棋)]]成功，成为史上年龄最大的围棋锦标获得者。

藤泽终身为围棋国际化而奋斗，对[[中国大陆|中国大陆]]围棋发展做出了巨大的贡献。从1981年起，14次带领“秀行军团”访华，中国著名的棋手[[聂卫平|聂卫平]]、[[马晓春|马晓春]]到[[常昊|常昊]]等皆受其影响。在中国的棋坛，藤泽秀行之名可谓家喻户晓。

== 生平 ==
*藤泽秀行於1925年6月14日生于日本[[横滨|横滨]]，原名藤澤保。父藤泽重五郎，母藤泽绢子。兄弟19人，父为买卖人。藤泽从小“顽劣”，五岁时玩耍掉入污水沟。险些丧命。后父令其学棋，藤泽天分极高。一年后拜[[濑越宪作|濑越宪作]]为启蒙教师。

*1934年，入日本棋院為院生，后拜{{link-ja|福田正义|福田正義_(囲碁棋士)}}五段为师（名义上）。研究[[明治|明治]]初期[[本因坊秀甫|本因坊秀甫]]棋谱。1937年，到中国慰问侵华日军。

*1939年入段。1940年，随[[坂田荣男|坂田荣男]]到[[中国|中国]][[大连|大连]]，在中国学习棋术，并研究中国古诗。1942年升三段，改名“秀行”。

*1945年4月，全家迁至[[千叶县|千叶县]]；5月，[[日本棋院|日本棋院]]毁于战火；8月，日本[[二战|二战]]投降；9月，其父去世。1946年，参加段位赛。1948年9月，创《围棋研究》。仅办三期后停刊。1952年7月，被《{{link-ja|棋道}}》评为“战后三杰”。

*1959年，日本棋院第1位决定战（现[[碁圣战|碁圣战]]）优胜。1960年，[[日本围棋升段赛#最高位战|最高位战]]优胜。1962年，获得首届[[日本围棋名人战|名人]]。1965年，{{link-ja|专业十杰战|プロ十傑戦}}优胜。

*1967年，获得王座。1968年，专业十杰战优胜。1968年，王座优胜。1969年，[[NHK杯电视围棋淘汰赛|NHK杯]]优胜。1969年，[[早碁選手權戰|快棋选手权战]]优胜。1969年，获得王座。1970年，获得名人位。1976年，获得[[日本围棋天元战|天元]]位。

*1977年，获得[[棋聖|棋聖]]位。以后六连霸，成為[[名譽棋聖|名譽棋聖]]。1981年，NHK杯优胜。

*1981年起，藤泽开始组织“秀行军团”，前後共14次。秀行军团成员几乎都是后来日本名将，像[[武宫正树|武宫正树]]、[[梅泽由香里|梅泽由香里]]等。更难能可贵的是，藤泽秀行先生背负巨额债务仍然自费组团到中国帮助刚摆脱[[文革|文革]]的中国棋手进步<sup id="fn_4_back">[[#fn_4|4]]</sup>。据记载，秀行军团初到北京，盛况空前。许多棋迷不顾政府的禁令<sup id="fn_5_back">[[#fn_5|5]]</sup>，到[[北京饭店|北京饭店]]听课。“秀行军团”的到来，帮助中国围棋获得质的飞跃。因此，藤泽受到中国棋迷喜爱。

*1991年，获得王座位。1992年，卫冕王座位（67岁、史上年龄最大的围棋锦标赛获得者）。2002年引退。2004年，获得“中日围棋交流功劳奖”。

*2009年5月8日上午7時16分（日本時間）去世。

* 其家人多有棋手，甥[[藤泽朋斋|藤泽朋斋]]九段、子（五男）{{link-ja|藤泽一就|藤沢一就}}八段、孫女[[藤澤里菜|藤澤里菜]]四段。

== 行事風格 ==
藤泽秀行以刚正不阿，仗义洒脱受到棋迷的喜爱。但私生活也荒诞无比，酗酒常常出洋相；赌博导致倾家荡产；风流事不断。但不管如何，藤泽秀行对围棋的热爱和其对围棋推广做出的贡献是不可抹灭的。
* 起名“秀行”：事实上，传统的围棋界的约定：“秀”字是本因坊才允许使用的名字。当时的藤泽保名气不大，却擅自改名“秀行”，引起了许多围棋前辈的不满。
* 自己制作段位证书：藤泽因日本棋院发给业余棋手的段位证书太贵，自己制作并发放业余段位证书。此举使藤泽名声斐然。
* 酗酒：1981年第五届NHK杯决胜战，藤泽醉醺醺的与对手[[高木祥一|高木祥一]]对战，虽然藤泽连棋子都无法摆正。但却赢得了胜利，因此得名怪杰。大體上每天都飲酒，但據說喝得不多，尤其1983胃癌手術後喝一點就醉了。酒醉時有亂講女性器官俗名的壞毛病，甚至與[[鄧小平|鄧小平]]會面時也喝得酩酊大醉，「中文的陰道怎麼講啊？」等胡言亂語，執傲的糾纏，在場人員不得不中止會面。
* 泥醉名人：1962年，獨賣新聞創辦名人戰，並由十三位棋手以循環賽方式決出首屆名人，最後一場循環賽的陣容是[[藤澤秀行|藤澤秀行]]對[[桥本昌二|桥本昌二]]，[[吳清源|吳清源]]對[[坂田榮男|坂田榮男]]，其中藤澤秀行九勝二負獨居排頭，吳清源與坂田榮男則以八勝三負緊追在後，假設藤澤勝出，那麼無疑將成為名人，可惜藤澤最終輸給桥本，要和另一場的勝方加賽決定名人歸屬，藤澤覺得自己無望，失望之餘就和好友上酒館喝得爛醉，結果吳清源和坂田榮男竟下成和棋，依規則雖要算是持白的吳清源獲勝，但只是「半勝」，換句話說不及九勝的藤澤，結果藤澤秀行就在爛醉之中意外成為了名人，故稱「泥醉名人」。
* 女性關係：藤澤生性風流，曾在情人家裡待了3年沒回過家。據說後來有事必須回去時，連自己家都不會走，只好打電話找妻子帶路。將棋棋士[[米長邦雄|米長邦雄]]太太曾向藤澤太太抱怨「我先生一周5天不回家」，後者回答「我家的3年沒回家」。
* 赌博與債務：据藤泽的回忆录自己介绍，光赌博就输了快十亿[[日元|日元]]。<sup id="fn_2_back">[[#fn_2|2]]</sup>兩千五百萬的棋圣冠军奖金，不到一年就输光了。藤泽从来不掩饰自己对金钱的渴望，对于最高奖金的棋圣战冲劲十足，共获得6次，独享名誉棋圣的称号。因此被称为围棋界的“借钱王”。因為賭債，秀行一生家徒四壁，就連他所住的房子，也是以他的兒子名義置下的，如果以他的名字購買，那麼早已被法院拍賣。<sup id="fn_3_back">[[#fn_3|3]]</sup>他連霸六屆棋聖的巨額獎金仍不足以填補巨額債務的零頭。在一次飛往中國的飛機上，藤澤秀行曾說：“多麼希望現在飛機掉進大海啊。”
*三次癌症折磨： 秀行先是因為酗酒患胃癌，好不容易撿條命，又發現有淋巴癌，好不容易捱過這道鬼門關，年近八旬時卻又檢查出了前列腺癌，一生遭受三次癌症折磨。然而，無論遭受怎樣的病魔折磨，藤澤秀行從未放棄過圍棋，他在圍棋上的毅力也並未因病魔的摧殘而受到影響。
* 67岁的王座：1991年，他在日本第39期王座戰五番棋決戰中，3比1挑戰羽根泰正王座成功，創66歲奪冠紀錄。次年他接受小林光一挑戰，在先失一局的情況下，仍然豪氣滿懷地宣稱：“輸了第一局後，我對戰勝他更有信心了！”結果，藤澤秀行3比2技壓小林光一，衛冕王座成功，同時再創67歲奪冠紀錄。至今，該紀錄還由他保持著。此时的藤泽秀行不仅年邁，而且患有胃癌和[[淋巴癌|淋巴癌]]，胃被切去3/4。这样的情况还能获胜，不得不成为围棋界的传奇。 

此外，藤泽在围棋界的趣谈不计其数。常常闹得其它棋手尴尬，但在日本，几乎所有的棋手都对藤泽秀行表示尊敬。

== 其它荣誉 ==
1987年[[紫绶褒章|紫绶褒章]]、1988年度[[棋道奖|棋道奖]]《特别奖》、1991年、1992年度《优秀棋士奖》受奖。第16回秀哉奖、1989年[[大仓喜七郎奖|大仓喜七郎奖]]。J·C奖、电视广播围棋制作者会奖。

== 名言 ==
* 我为如今的胜负深深偏离棋道的本质而感到痛心，如果把下棋比作两个人争100元，几乎所有的人都认为只要拿到51元就可以了，但我认为应该拿到全部，本来能杀的棋不杀，即便是胜利了也称不上真正的胜利者。<sup id="fn_6_back">[[#fn_6|6]]</sup>——于第2期棋圣赛胜加藤正夫。

* 这种封闭自守的愚蠢行为只会让日本围棋越来越萎靡，只有我们的对手强大了，日本围棋的强大才有意义，围棋是没有国界的，只要是棋道中人都是一家子。<sup id="fn_7_back">[[#fn_7|7]]</sup>——第3期中日围棋对抗赛答记者问。

== 评价 ==
* 从胜负的角度看，[[赵治勋|赵治勋]]无疑是最强的，但是从棋力上看，最强者首推秀行先生，吾辈自叹不如。——[[武宫正树|武宫正树]]
* 对于秀行的布局中的某些招法，即使让我想上一百万年，我也想不出来。或许只有神才能与他讨论前50手棋的得失吧。——[[木谷实|木谷实]]
* 或者说，中国的围棋能有今天，是和藤泽先生教导分不开的。——[[陈祖德|陈祖德]]
* 这一时期的秀行老师简直跟鬼一样厉害。我原想，把秀行老师所有的技艺偷到手，传授给后来人是我的义务，所以尽可能多下几局，但是根本就抵挡不住。厉害的秘密在于：他对围棋的全局观。——[[大竹英雄|大竹英雄]]

== 注释 ==
* <cite id="fn_1">[[#fn_1_back|注 1:]] </cite>罗马音与英文正好相反：Fujisawa Hideyuki。这是因为英语习惯是名在前，姓在后。而日本语和汉语一样姓前名后。
* <cite id="fn_2">[[#fn_2_back|注 2:]] </cite>20世纪70年代乃至更早时的日元的购买力很强一些。以《藤泽秀行回忆录》的说法，藤泽赌博输掉的钱近十亿日元。
* <cite id="fn_3">[[#fn_3_back|注 3:]] </cite>事实上，藤泽秀行现在的财产总数应为零，并有很多的债务。藤泽如今生活所用的一切物品，理论上都是他儿子的财产。
* <cite id="fn_4">[[#fn_4_back|注 4:]] </cite>后期，“秀行军团”还是得到了一些赞助。
* <cite id="fn_5">[[#fn_5_back|注 5:]] </cite>1981年，北京涉外旅馆还不允许普通百姓无证件进出。因此这么说。
* <cite id="fn_6">[[#fn_6_back|注 6:]] </cite> 第2届棋圣战，第五局。对加藤正夫下出名局：“百目屠龙”，事实上当时藤泽秀行有一定的优势。而且屠龙一手，变化极多，不容易控制。藤泽长考近2.5小时。决定屠龙，而不是平稳获得胜势。
* <cite id="fn_7">[[#fn_7_back|注 7:]] </cite>三届中日围棋对抗赛，中国队全军获胜。因此有些日本媒体指责藤泽的“秀行军团”教中国棋手下棋从而导致的。
