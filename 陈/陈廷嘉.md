'''陈廷嘉'''（{{bd|1980年|11月15日|||catIdx= C陈}}，原名'''陈丽丽'''<ref name=" cll" >{{cite web |url=http://data.ent.sina.com.cn/star/18219.html |title=陈廷嘉_明星资料|accessdate= 2012年|work= |publisher=新浪娱乐 |language =zh-cn}}</ref>）是[[中国大陆|中国大陆]]知名[[演员|演员]]，出生于[[江苏省|江苏省]][[江阴市|江阴市]]，2005年毕业于[[重庆大学美视电影学院|重庆大学美视电影学院]]播音主持專業，被誉为“小[[上官云珠|上官云珠]]”。<ref name=" lks" >{{cite web |url=http://www.tianjinwe.com/rollnews/wy/201010/t20101019_2126756.html |title=金鸡电影节 陈廷嘉艳压群芳“小上官云珠” |accessdate= 2012年|work= |publisher=天津网 |language =zh-cn}}</ref>

== 简介 ==
陈廷嘉的家庭都是[[艺术|艺术]]出身，她从小就获得了良好的家庭教育，因出演电视剧《[[大汉天子|大汉天子3]]》和电影《[[财神到|财神到]]》而一举走红。<ref name=" lks" />
