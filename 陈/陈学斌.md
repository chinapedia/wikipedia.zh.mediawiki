'''陈学斌'''（{{bd|1959年|10月||}}）籍贯不详，[[中国人民解放军中将|中国人民解放军空军中将]]。

==生平==
陈学斌曾任[[中国人民解放军海军北海舰队|中国人民解放军海军北海舰队]]航空兵某师政治部主任、政委。2009年，出任海军北海舰队政治部副主任。2010年7月，晋升为海军少将军衔。2012年7月，升任海军北海舰队副政委兼海军北海舰队航空兵政委（前任[[白文奇|白文奇]]海军少将升任[[济南军区|济南军区]]副政委兼北海舰队政委）。2015年12月，陈学斌升任海军纪委书记、海军党委常委<ref>{{Cite web |url =http://news.qq.com/a/20151230/030500.htm  |title = 北海舰队副政委陈学斌升任海军党委常委 |author =  |date = 2015-12-30 |publisher = 腾讯 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://mil.sohu.com/20151228/n432741620.shtml |title = 少将陈学斌调任海军纪委书记 晋升副大军区职 |author =  |date = 2015-12-28  |publisher = 搜狐 |language =  |accessdate =  |quote =  }}</ref>。2017年7月31日，晋升为海军中将军衔<ref>{{cite news|title=海军举行将官军衔晋升仪式：陈学斌、魏钢晋升为中将|url=http://www.thepaper.cn/newsDetail_forward_1748153|accessdate=2017-09-16|newspaper=澎湃新闻}}</ref>。2018年1月，调任海军副政委兼政治工作部主任<ref>{{cite web|title=邱延鹏任海军副司令员，陈学斌任副政委兼政工部主任|url=http://news.sina.com.cn/o/2018-01-30/doc-ifyqyqni5363832.shtml|website=新浪|accessdate=2018-02-01|date=2018-01-30}}</ref>。2018年5月，改任[[中国人民解放军空军|空军]]副政委<ref>{{cite web|title=海军副政委兼政工部主任陈学斌跨军种调任空军副政委|url=https://www.thepaper.cn/newsDetail_forward_2149963|website=澎湃新闻|accessdate=2018-05-25}}</ref>。
