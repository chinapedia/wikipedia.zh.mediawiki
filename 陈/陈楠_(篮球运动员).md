'''陈楠'''（{{bd|1983年|1月8日||}}），出生于[[山东省|山东省]][[青岛市|青岛市]][[胶南县|胶南县]]<ref name="a1">{{Cite web |url=http://sports.sina.com.cn/k/2010-11-26/14475337879.shtml |title=连续三届亚运会上摘金 陈楠成就青岛第一人伟业 |accessdate=2018-02-12 |author=记者：刘长周、孙飞 |date=2010-11-26 |work= |publisher=新浪网，来源：青岛新闻网-青岛晚报|language=Zh-hans }}</ref>。[[中国|中国]]女子[[篮球|篮球]]運動員，亦為[[中国国家女子篮球队|中国国家女子篮球队]]成員、[[助教|助教]]<ref name="a2"/>。

==簡歷==
曾就读于胶南实验小学<ref name="a2"/>，五年级时进入青岛市体校，练习[[三项全能|三项全能]]。一年多后，陈楠改练篮球<ref name="a1"/>。1995年<ref name="a2"/>入选解放军[[八一体工大队|八一队]]。2000年，陈楠第一次入选[[中国国家女子篮球队|中国国家女子篮球队]]<ref name="a2">{{Cite web |url=http://news.bandao.cn/news_html/201607/20160722/news_20160722_2650390.shtml |title=陈楠成青岛参加里约奥运三大球唯一选手(图) |accessdate=2018-02-12 |author=记者：邢成博 |date=2016-07-22 |work= |publisher=半岛网，来源：半岛都市报|language=Zh-hans }}</ref>。2010年11月，陈楠代表[[2010年亞洲運動會中國代表團|中國]]出戰[[廣州|廣州]][[2010年亞洲運動會|亞運會]]，參加[[2010年亞洲運動會籃球比賽|籃球比賽]]，奪得金牌<ref name="廣州亞運"/>。
