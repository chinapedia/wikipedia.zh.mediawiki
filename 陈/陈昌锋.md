'''陈昌锋'''（{{bd|1962年|12月||}}）[[江苏|江苏]][[阜宁|阜宁]]人，[[中国人民解放军海军|中国人民解放军海军]]大校<ref name=yideng>{{Cite web |url = http://www.81.cn/2016hjcllqzn/2016-04/21/content_7017020.htm |title =一等功臣：陈昌锋  |author =  |date = 2016-04-21 |publisher = 中国军网 |language =  |accessdate =  |quote =  }}</ref>。

==生平==
1981年10月入伍。1984年6月加入[[中国共产党|中国共产党]]。1985年，陈昌锋从院校毕业，到[[南海舰队|南海舰队]]海军陆战一旅报到，适逢海军陆战队抽调干部到[[西沙群岛|西沙群岛]]换防，乃奉派到西沙守备部队担任坦克排排长。1988年底，回到海军陆战队。1989年5月，陈昌锋出任坦克一连连长。任内用三年时间将该连带成海军正规化管理先进连，陈昌锋也被海军评为优秀主官。后来任某坦克营营长。1991年初，其所在的海军陆战一旅军务科想调陈昌锋去当参谋。时任旅长[[陈川熙|陈川熙]]却让陈昌锋多在基层历练，要求他在基层呆着。陈川熙旅长调离后，旅政治部保卫科又想调陈昌锋，新任旅长[[周乃文|周乃文]]也认为陈昌锋更适合在基层工作，所以也没有同意。由此，陈昌锋在该旅下属的两栖装甲团连续工作十多年，从副连长一直干到团参谋长。1997年2月，旅党委派陈昌锋到[[南京|南京]][[中国人民解放军海军指挥学院|中国人民解放军海军指挥学院]]深造一年，学习合同战术指挥<ref name=rmw>{{Cite web |url =http://www.people.com.cn/GB/shizheng/1026/2669416.html  |title = 海军某陆战旅党委和领导培育陈昌锋纪事 |author =  |date = 2004-07-27 |publisher = 人民网 |language =  |accessdate =  |quote =  }}</ref><ref name=yideng/><ref name=ygjs>{{Cite web |url = http://military.cnr.cn/jstp/jstph/20140828/t20140828_516328425_1.html |title = 国防大学举办纪念“五四”青年节座谈会9名学习成才典型精彩讲述成才体会 |author =  |date = 2012-04-28 |publisher =  央广军事|language =  |accessdate =  |quote =  }}</ref>。

1999年8月，旅党委派陈昌锋赴[[德国联邦国防军|德国联邦国防军]]指挥学院留学，成为[[中国人民解放军海军陆战队|中国人民解放军海军陆战队]]首位出国留学的陆战队员。2001年毕业归国，出任海军陆战一旅（旅长[[王革军|王革军]]）下属的两栖装甲团团长。陈昌锋任团长后，团队连续3年被舰队评为“军事训练一级单位”；2002年底，团队因全面建设过硬而立集体三等功；陈昌锋个人被舰队评为“军事训练先进个人”、“正规化管理先进个人”、“优等指挥员”。2004年，陈昌锋被[[中共中央宣传部|中共中央宣传部]]、[[中国人民解放军总政治部|中国人民解放军总政治部]]、中国人民解放军海军确立为“勇立军事变革潮头的两栖先锋”。2005年，中国人民解放军海军为陈昌锋记一等功<ref name=rmw/><ref name=yideng/><ref name=ygjs/><ref>{{Cite web|url =http://www.jb.mil.cn/cp/cpzjyfz/wwzjhg/201212/t20121207_7440.html|title =赴南海舰队征集文物|author =|date =2012-12-07|publisher =中国人民革命军事博物馆|language =|accessdate =|quote =}}{{dead link|date=十一月 2017 |bot=InternetArchiveBot }}</ref>。

2006年，陈昌锋升任海军陆战一旅旅长<ref name=xhw>{{Cite web|url =http://news.xinhuanet.com/mil/2010-07/27/content_13690863.htm|title =“军中之军”陆战队--专访海军陆战队某旅旅长陈昌锋|author =|date =2010-07-27|publisher =新华网|language =|accessdate =|quote =}}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref name=ygjs/>。后来，陈昌锋出任[[中国人民解放军海军西沙水警区|中国人民解放军海军西沙水警区]]司令员，2012年卸任<ref>{{Cite web |url = http://news.ifeng.com/gundong/detail_2012_12/02/19745465_0.shtml |title = 以昂扬的精神状态开创国防和军队建设新局面 |author =  |date = 2012-12-02 |publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://www.hinews.cn/news/system/2012/04/29/014346828.shtml |title = 肩负起发展先进军事文化的重任 |author =  |date =  |publisher =  南海网|language =  |accessdate =  |quote =  }}</ref>。

2013年3月，中国人民解放军海军兵种指挥学院正式更名为[[中国人民解放军海军陆战学院|中国人民解放军海军陆战学院]]，陈昌锋任院长<ref>{{Cite web |url = http://news.163.com/15/0409/03/AMNTCK6P00014AED.html |title = 海军陆战精英从这里出发 |author =  |date = 2015-04-09 |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref><ref name=yideng/>。
