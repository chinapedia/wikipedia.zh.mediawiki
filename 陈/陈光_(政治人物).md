'''陈光'''（{{bd|1914年|11月|1986年|12月20日}}）原名'''胡德辉'''，曾用名'''胡德瀚'''，[[江西|江西]][[瑞金|瑞金]]人，[[中华人民共和国|中华人民共和国]]政治人物。

==生平==
1914年11月，陈光生于江西省瑞金县壬田区竹山下村一户贫苦农民家庭。1926年参加农民协会和农民赤卫军。1930年参加[[中国工农红军|中国工农红军]]，并加入[[中国共产主义青年团|中国共产主义青年团]]。1931年加入[[中国共产党|中国共产党]]。参加了[[中国共产党第一次反围剿战争|第一]]、[[中国共产党第二次反围剿战争|第二、[[中国共产党第五次反围剿战争]]<ref name=shishi/><ref name=xinlang/>。[[长征|长征]]途中，陈光任[[红五军团|红五军团]]政治部科长、中共[[西康省|西康省]]丹坝县委副书记，1936年12月任陕甘宁省政府主席团巡视员<ref name=xinlang>{{Cite web |url =http://news.sina.com.cn/c/2015-07-17/063932117326.shtml  |title =鞠躬尽瘁为革命——陈光  |author =  |date = 2015-07-17 |publisher =新浪  |language =  |accessdate =  |quote =  }}</ref>。

[[抗日战争|抗日战争]]全面爆发后，陈光起初在[[延安|延安]]任中央马列学院科长，不久到抗日前线任中共[[皖南|皖南]]特委委员兼[[泾县|泾县]]县委书记，协助[[新四军|新四军]]军部成立工业生产合作社。1940年，陈光任苏皖区党委丹北中心县委书记、苏北区党委委员兼京沪路北特委书记。1941年3月，丹北的地方武装组建为新四军第六师第十八旅第五十一团，陈光任该团政治委员。陈光初到丹北时，该地区还没有公开成立抗日民主政权。1941年4月，在陈光领导下，江南第四行政专署成立，在根据地遭敌分割的情况下重建县制。此后陈光又开展区乡政权改造，建起了中国共产党领导的[[抗日民族统一战线|抗日民族统一战线]]政权<ref name=xinlang/>。

1941年11月，日伪军3000多人对丹北地区大扫荡。当时新四军主力转移至铁路以南，路北特委仅有主力第四团的第三营及少量地方武装。陈光率丹北抗日军民坚持了丹北抗日阵地，粉碎了日伪“清乡”，使丹北地区成为苏南、苏中、淮南三大抗日根据地的纽带<ref name=xinlang/>。

1943年，陈光调任淮南区党委组织部长、淮南区津浦路东地委书记，并且兼任新四军二师四旅政治委员、淮南区津浦路东军分区政治委员。任内他领导各级中共党组织抓好党员干部培训，建立健全党内民主生活，使[[延安整风|整风运动]]深入开展。他还和地委其他领导号召在路东开展[[大生产运动|生产运动]]、互助工作，取得了成功经验，对当时华中其他地区及对[[中华人民共和国|中华人民共和国]]成立后的互助合作运动都起了借鉴作用<ref name=xinlang/>。

1945年，陈光返回苏南任新四军苏浙军区第一分区政委、苏南区党委副书记，任内为南下的新四军提供后勤保障，同军分区首长部署地方武装作战，调配干部接管新解放的地域及城镇<ref name=xinlang/>。

[[第二次国共内战|第二次国共内战]]时期，历任[[中共中央华中分局|中共中央华中分局]]二地委书记、[[扬州|扬州]]军管会主任等职。中华人民共和国成立后，历任[[中共江苏省委|中共江苏省委]]书记处书记、代理省委第一书记、省委常务书记等职<ref name=xinlang/>。他是[[第一届全国人民代表大会代表名单|第一届]]<ref>{{cite book|author=全国人民代表大会常务委员会办公厅|title=中华人民共和国第一届全国人民代表大会代表名单|year=1955年|publisher=全国人民代表大会常务委员会办公厅|url=http://book.ucdrs.superlib.net/views/specific/2929/bookDetail.jsp?dxNumber=000005772167&d=546843D0AE2464CD1C95896344A85614&fenlei=04070402}}</ref>、[[第二届全国人民代表大会代表名单|第二届全国人民代表大会代表]]<ref name=shishi/>。

[[文革|文革]]期间，陈光蒙冤和受迫害八年。[[中共十一届三中全会|中共十一届三中全会]]后，陈光获得平反。1978年调任[[中华人民共和国民政部|中华人民共和国民政部]]常务副部长、党组副书记。1983年6月，在[[全国政协|全国政协]]六届一次会议上当选为政协常务委员<ref name=shishi/>。

1986年12月20日，陈光在[[南京|南京]]病逝，享年72岁<ref name=xinlang/><ref name=shishi>我党优秀党员、全国政协常委陈光同志在南京逝世，人民日报1986年12月30日</ref>。1986年12月29日，陈光同志追悼会在南京举行，追悼会由民政部副部长[[张德江|张德江]]主持，中共江苏省委书记[[韩培信|韩培信]]致悼词<ref name=shishi/>。
