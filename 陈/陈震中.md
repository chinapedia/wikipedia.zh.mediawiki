'''陈震中'''（{{bd|1926年|||}}）[[浙江|浙江]][[海宁|海宁]]人，[[中华人民共和国|中华人民共和国]]官员<ref name=sht/>。

==生平==
陈震中是[[陈巳生|陈巳生]]的次子。1937年，进入上海中学念初中。时值[[抗日战争|抗日战争]]爆发，日军发动[[八一三事变|八一三事变]]进攻上海，其父陈巳生积极参加抗战。后来陈震中也在学校参加反对汉奸[[汪精卫|汪精卫]]的活动。1943年，陈震中考入[[上海圣约翰大学|上海圣约翰大学]]医科。1945年2月，陈震中在圣约翰大学加入[[中国共产党|中国共产党]]。在上海从事地下学运工作时，陈震中和三弟陈震海以及[[钱其琛|钱其琛]]在同一个中共地下党支部<ref name=mj>{{Cite web |url =http://www.mj.org.cn/zsjs/content/2015-02/26/content_172559_4.htm  |title =  陈巳生、陈震中父子的上海往事|author =  |date =  |publisher = 中国报道2009年第四期 |language =  |accessdate =  |quote =  }}</ref>。

1946年6月23日，为反对[[蒋介石|蒋介石]]发动[[第二次国共内战|第二次国共内战]]，上海市民派出代表赴首都[[南京|南京]]请愿。陈震中等人被推选为学生代表，与人民代表[[马叙伦|马叙伦]]、[[胡厥文|胡厥文]]、[[蒉延芳|蒉延芳]]、[[雷洁琼|雷洁琼]]等11人组成上海人民和平代表团，乘坐火车赴南京。在南京[[下关站|下关火车站]]下车后，代表们当即遭到预先聚在站台上的国民党特务和打手的殴打，马叙伦、雷洁琼、[[阎宝航|阎宝航]]、陈震中身受重伤，此即“[[下关惨案|下关惨案]]”。惨案发生后，在南京的中共代表[[周恩来|周恩来]]、[[董必武|董必武]]、[[滕代远|滕代远]]、[[邓颖超|邓颖超]]、[[齐燕铭|齐燕铭]]等人迅速赴医院看望伤者。7月5日，[[毛泽东|毛泽东]]、[[朱德|朱德]]委托周恩来转发慰问电，对陈震中等伤者表示慰问。陈震中在上海的家人此时十分焦急，因陈巳生身份特殊无法出面，乃由陈震中的继母徐剑英独自赴南京，将陈震中接回上海，并且日夜在陈震中身边守护<ref name=mj/>。

因为国民党政府下令要逮捕陈震中，经中共党组织安排，陈震中在胞弟陈震海（也是中共地下党员）的掩护下，化装离开上海赴[[香港|香港]]。1948年，陈震中经中共党组织安排，离开香港赴[[中共中央|中共中央]]驻地[[西柏坡|西柏坡]]，参加中华全国学生第14届代表大会的筹备工作，任大会筹委会主任。1949年3月召开的此次大会上，陈震中作为上海学联代表，当选为[[中华全国学生联合会|中华全国学生联合会]]副主席<ref name=mj/>。

1949年9月底，[[中国人民政治协商会议第一届全体会议|中国人民政治协商会议第一届全体会议]]选举了[[中央人民政府委员会|中央人民政府委员会]]。执行主席[[刘少奇|刘少奇]]、[[李立三|李立三]]等任选举总监督，另外由主席团指定60人分为20组监票。陈震中等人是第14组监票人，[[胡耀邦|胡耀邦]]等人是第19组监票人。1949年10月1日，陈巳生、陈震中父子应邀参加了[[中华人民共和国开国大典|中华人民共和国开国大典]]<ref name=mj/>。

1951年，陈震中任[[中国新民主主义青年团|中国新民主主义青年团]]上海市委军体部长。后来任华东工委军体部副部长，1953年任华东区体委办公室副主任。1954年底，华东体委撤销，陈震中调[[国家体委|国家体委]]工作。1958年，陈震中调回上海，任上海市体委竞技指导科副主任。1963年至1966年2月，任上海市体委党委副书记。后曾任上海市体委革命委员会副主任、科技情报研究所副所长等职<ref name=sht>{{Cite web |url = http://www.shtong.gov.cn/newsite/node2/node2245/node4455/node15142/node15146/node60899/userobject1ai15492.html |title =上海体育志 第七编人物 第二章名录 一、市体育机构高级管理人员  |author =  |date =  |publisher = 上海通 |language =  |accessdate = 2017-03-15 |quote =  }}</ref>。
