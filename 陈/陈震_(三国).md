'''陳震'''（{{bd|?||235年|catIdx=C陳}}），[[表字|字]]'''孝起'''，[[荆州_(古代)|荊州]][[南陽郡|南陽]]人，[[三國|三國]]时期[[蜀漢|蜀漢]]官吏。

== 生平 ==
209年，[[劉備|劉備]]為荊州[[牧|牧]]時，被辟為[[從事|從事]]。後隨劉備入蜀，於214年平定益州後，升為[[蜀郡|蜀郡]]北部都尉，後改郡名為[[汶山郡|汶山]]，立為[[太守|太守]]，再轉守於[[犍為郡|犍為]]。225年，拜為[[尚書|尚書]]，遷尚書令，奉命出使[[東吳|吳國]]，頻繁的穿梭蜀吳之间。

229年，[[孫權|孫權]]自立為帝，封陳震為[[衛尉|衛尉]]，陳震到吳的邊界，說：「東之與西，驛使往來，冠蓋相望，申盟初好，日新其事。東尊應保聖祚，告燎受符，剖判土宇，天下響應，各有所歸。於此時也，以同心討賊，則何寇不滅哉！西朝君臣，引領欣賴。震以不才，得充下使，奉聘敘好，踐界踊躍，入則如歸。獻子適魯，犯其山諱，春秋譏之。望必啟告，使行人睦焉。即日張旍誥眾，各自約誓。順流漂疾，國典異制，懼或有違，幸必斟誨，示其所宜。」陳震到達[[武昌|武昌]]，孫權親自與陳震歃盟，定下蜀漢與東吳平分天下：以[[徐州|徐]]、[[豫州|豫]]、[[幽州|幽]]、[[青州|青]]屬吳，[[并州|并]]、[[涼州|涼]]、[[冀州|冀]]、[[兗州|兗]]屬蜀，[[司州|司州]]則以[[函谷關|函谷關]]為界平分。陳震歸還，封為城陽亭侯。

陳震一直認為[[李严_(蜀汉)|李嚴]]不可信任。231年，李嚴果然運糧錯失、假傳聖旨而阻礙了[[诸葛亮北伐|北伐]]，[[諸葛亮|諸葛亮]]也向[[蔣琬|蔣琬]]、[[董允|董允]]感歎自己沒有認真考慮陳震的話。於235年逝世。

== 特徵 ==
陳震為人正直、純樸，老成持重，說話較輕鬆和順。

== 家庭 ==

* 陳濟，陳震之子。
== 評價 ==
《[[三國志|三國志]]》評曰：「[[董和|董和]]蹈[[羔羊|羔羊]]之素，[[劉巴|劉巴]]履清尚之節，[[馬良|馬良]]貞實，稱為令士，陳震忠恪，老而益篤，[[董允|董允]]匡主，義形於色，皆蜀臣之良矣。」

[[諸葛亮|諸葛亮]]於其兄[[諸葛瑾|諸葛瑾]]書信中：「孝起忠純之性，老而益篤，及其贊述東西，歡樂和合，有可貴者。」

== 演義描述 ==
初登場身分為[[袁紹|袁紹]]部下，主要擔任外交事務。曾以密使身分潛入曹營，將[[劉備|劉備]]寄居袁紹之下的消息告知[[關羽|關羽]]，導致後來的掛印封金、過五關斬六將等事件；後又出使[[孫策|孫策]]處，洽談雙方同盟事宜，不料遭逢[[于吉|于吉]]事件，孫策因此逝世後，繼任的[[孫權|孫權]]乃回絕了袁紹的同盟要求。

袁紹敗亡後，輾轉至[[荊州|荊州]]投靠劉備。[[夷陵之戰|夷陵之戰]]前，曾言[[青城山|青城山]]上有一隱士[[李意|李意]]，能知人生死吉凶，於是奉命邀請其面見劉備。李意以圖畫方式暗喻劉備將會戰敗，惹得劉備不悅，後夷陵一戰果然失利。

諸葛亮北伐時，被任命為[[侍中|侍中]]。
