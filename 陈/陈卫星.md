'''陈卫星'''（{{lang-en|'''Chen Weixing'''}}，{{bd|1972年|4月27日}}），[[奥地利|奥地利]][[乒乓球|乒乓球]]运动员，前[[中国国家乒乓球队|中国国家乒乓球队]]队员，生于[[内蒙古自治区|内蒙古自治区]][[包头市|包头市]]<ref>{{Cite web |url=http://szb.northnews.cn/nmgrb/html/2008-08/17/content_127702.htm |title=钢城出了个陈卫星 |accessdate=2015-09-19 |author=记者：刘红星、孙一帆、阿荣 |date=2008-08-17 |work= |publisher=内蒙古日报社数字报刊 |language=Zh-hans |deadurl=yes |archiveurl=https://web.archive.org/web/20160306011852/http://szb.northnews.cn/nmgrb/html/2008-08/17/content_127702.htm |archivedate=2016-03-06 }}</ref>，右手横拍[[削球|削球]]打法。

== 生涯 ==
陈卫星生于[[内蒙古自治区|内蒙古自治区]]的一个军人家庭，17岁时从内蒙古体工大队被选拔进中国国家队，其后长期在队中担任陪练。1996年，陈卫星从国家队退役，前往德国为俱乐部打球，后移居奥地利，并入籍奥地利代表其国家队参加国际大赛。<ref>[http://sports.sina.com.cn/o/2013-03-19/09366474543.shtml 41岁陈卫星曾是刘国梁陪练 旅欧18年已成中产阶级].半岛都市报.2013-03-19</ref>[[2003年世界乒乓球锦标赛|2003年世界乒乓球锦标赛]]，陈卫星于男单八分之一决赛中以4:3爆冷淘汰中国新星[[王皓|王皓]]<ref>[http://sports.sohu.com/39/72/news209477239.shtml 小将王皓失手 陈卫星“放卫星”--海外兵团发飚].东方体育日报.2003.5.24</ref>。2003年[[欧洲乒乓球锦标赛|欧洲乒乓球锦标赛]]，陈卫星夺得男双冠军。2007年及2010年[[世界杯乒乓球赛|世界杯乒乓球赛]]，陈卫星皆以[[奥地利|奥地利]]国家队主力队员身份，夺得男子团体铜牌。<ref>[http://www.ittf.com/ittf_stats/All_events3.asp?ID=1187&NAMES=CHEN+Weixing+%28AUT%29&Assoc1=AUT&Assoc=%20&s_Gender=%20&s_Names=Chen+Weixing& CHEN Weixing (AUT) ] {{webarchive|url=https://web.archive.org/web/20131110074534/http://www.ittf.com/ittf_stats/All_events3.asp?ID=1187&NAMES=CHEN+Weixing+(AUT)&Assoc1=AUT&Assoc=&s_Gender=&s_Names=Chen+Weixing& |date=2013-11-10 }}</ref>
