'''陳寶蓮'''（英文名：Pauline Chan，{{bd|1973年|5月23日|2002年|7月31日|catIdx=C陳}}），已故[[上海|上海]]、[[香港|香港]][[女演員|女演員]]。原名'''趙靜'''<ref name="bl">{{cite web | url=http://www.epochtimes.com/b5/2/8/7/n206897.htm | title=陳寶蓮家屬首次公開确認自殺原因 | publisher=大紀元時報 | date=8/7/2002 10:55:04 PM | accessdate=2011-10-31}}</ref>。

== 生平 ==
陳寶蓮是家中獨女；4岁時父母离婚，之後被外婆带大。12歲時跟隨母親移居[[香港|香港]]，15歲起兼職[[模特兒|模特兒]]。17歲時參選1990年度[[亞洲小姐競選|亞洲小姐競選]]落選後被經紀公司簽為[[藝人|藝人]]，第一個主持的節目為《[[冧莊|冧莊]]》。<ref name="multiple11 " >{{cite web |url=http://www.southcn.com/ent/hotspot/chenbl/200208011586.htm |title=香港三级艳星陈宝莲情史大起底 |accessdate=2002-8 |work=|publisher=南方网 |language=zh-hans}}</ref>

入行之初正值香港[[三級片|三級電影]]盛行，她亦隨波逐流接拍多部三級影片《灯草和尚》《我来自北京》《五月樱唇》等，希望成名之後再洗脫性感艷星形象，可是卻未成功，且是非不斷。後來疑似因爲受藥物影響而經常神智不清，被香港媒體把她和[[藍潔瑛|藍潔瑛]]、[[蔡楓華|蔡楓華]]、[[洪朝豐|洪朝豐]]封為「四大癲王」。<ref>{{cite web |url=http://hd.stheadline.com/ent/ent_content.asp?contid=163878&srctype=n |title=四大癲王 |accessdate=2014-08-09 |work=頭條日報 |language=zh-TW}}</ref>

2002年7月31日傍晚5時半，陳寶蓮於上海靜安區南陽路24樓的寓所跳樓[[自殺|自殺]]身亡，遺下剛滿月的兒子。《[[再見螢火蟲_(台灣電視劇)|再見螢火蟲]]》成了她生前最後一部作品<ref name="posthumous">{{cite news|url=http://www.libertytimes.com.tw/2002/new/aug/1/today-show3.htm |title=流星墜落上海 陳寶蓮自殺身亡 |publisher=自由電子新聞網 |date=2002-08-01|accessdate=2011-05-10 |language=zh-tw }}</ref>。

== 感情生活 ==
陳寶蓮不僅星路不如預期，感情生活也是波折重重。1993年，20岁的陈宝莲和[[莫少聪|莫少聪]]在《[[剑奴之血契约|剑奴之血契约]]》认识相恋，不过很快就分手了。1995年，和[[黄子扬|黄子扬]]的戀情也无果而终。<ref name="multiple11 " />

與乾爹黄任中之間的曖昧關係也最常為媒體的話題。黄任中曾向媒体公开和干女儿兩人同床有五十至六十次，從來沒發生過關係；他們之間是仰慕、父女、好友的那种感情。<ref>[http://www.epochtimes.com/b5/2/8/4/n206188.htm 黃任中痛悼陳寶蓮﹕不是最寵但感情深厚]</ref>

1997年，陈宝莲為進入[[台湾|台湾]]市场，主动追求台湾男主持[[张菲|张菲]]，后因[[黄任中|黄任中]]插手介入而失败。后来与歌手[[蔡枫华|蔡枫华]]及曾参加[[亚视|亚视]]《[[港男选举|港男选举]]》的吴学明也陸續传出过绯闻。<ref name="multiple11 " />

2002年6月，陈宝莲生下一个男婴。其向好友透露，嬰兒的父親是在台北[[迪斯科|迪斯科]]邂逅的[[DJ|DJ]]，一位25岁的[[美籍华人|美籍华人]]。<ref name="multiple11 " />

據2016年5月8日出版的[[明報週刊|明報週刊]]報道，此子現正由[[王菲|王菲]]好拍檔[[邱瓈寬|邱瓈寬]]<ref>[http://news.mingpao.com/pns/dailynews/web_tc/article/20160508/s00016/1462643938204 陳寶蓮兒子恨入行 養母邱瓈寬不准]</ref>收養。
