'''陳强'''（{{bd|1918年||2012年|6月26日|catIdx=Chen陳强}}），原名'''陈庆三'''，中國大陸著名[[电影|电影]]表演艺术家，获得第29届[[百花奖|百花奖]]终身成就奖。曾主演经典影片《[[红色娘子军|红色娘子军]]》、《[[白毛女|白毛女]]》等，以反派角色见长，此外也曾参加拍摄多部喜剧片。如《三年早知道》、《父与子》、《瞧这一家子》等。<ref>{{cite news|title=著名电影表演艺术家陈强病逝 享年94岁|url=http://ent.sina.com.cn/m/c/2012-06-26/23163668906.shtml|accessdate=2012-06-26|newspaper=新浪娱乐|date=2012-06-26}}</ref>

其次子为著名[[小品|小品]]、[[影视|影视]]、[[戏剧|戏剧]]演员[[陈佩斯|陈佩斯]]。

== 生平 ==
1918年，陈强出生于[[直隶省|直隶省]][[宁晋县|宁晋县]]（今[[河北省|河北省]][[宁晋县|宁晋县]]）[[徐家河乡|徐家河乡]]，幼年时期随父母逃难到[[山西|山西]]，1936年，陈强在[[太原|太原]]上学，业余时间参加太原青年剧社和新生剧院的演出活动。<ref name="bull " />

1938年，陈强进入[[延安|延安]][[鲁迅艺术学院|鲁迅艺术学院]]戏剧系学习表演理论和表演技巧，1939年，陈强参加[[晋察冀边区|晋察冀边区]]联大文工团和北战地服务团等演出活动。

1942年，陈强加入[[中国共产党|中国共产党]]，1944年，陈强担任北战地服务团演员。1946年，陈强开始拍摄[[电影|电影]]，从事演艺事业。1947年，陈强出任[[东北电影制片厂|东北电影制片厂]]演员。1953年，陈强出任[[北京电影制片厂|北京电影制片厂]]演员。1955年，陈强在北京电影学校表演专修班学习。<ref name="bull " />

2000年，陈强因为[[脑中风|脑中风]]导致部分记忆丧失。<ref name="bull " >{{cite web |url= http://news.ifeng.com/mainland/detail_2012_06/27/15576534_0.shtml|title= 著名表演艺术家陈强去世|accessdate= 2012年6月27日|publisher=凤凰网 |work= |language =zh-cn}}</ref>

2012年6月26日，陈强在[[北京市|北京市]][[安贞医院|安贞医院]]因为[[肺炎|肺炎]]和[[肾衰竭|肾衰竭]]病逝。<ref name=" fish" >{{cite web |url=http://news.ifeng.com/mainland/detail_2012_06/27/15577398_0.shtml |title=陈佩斯之父陈强在北京安贞医院因病逝世 享年94岁 |accessdate=2012年6月27日 |publisher= 凤凰网|work= |language =zh-cn}}</ref>

== 艺术风格 ==
陈强擅长出演反面人物，不拘于“脸谱化”的俗套，在《红色娘子军》中出演的“南霸天”，被誉为是“中国反派人物难以跨越的高峰”，他的风格是“逗而不厌、闹而不乱、笑而不俗、趣味由衷”，讲究自然和真实，强调要生活情趣、人物性格和戏剧矛盾三者结合，充满幽默感和节奏感。<ref name="bull " /><ref name=" fox" >{{cite web |url= http://ent.163.com/12/0627/08/8508K8GH00031H2L.html|title=艺术家陈强昨逝世 造就新中国电影事业多项第一 |accessdate=2012-06-27  |publisher=网易 |work= |language =zh-cn}}</ref>

== 获奖情况 ==
[[File:_Bai_Mao_Nv_1950.jpg|thumb]]
1962年，获第一届电影[[百花奖|百花奖]]最佳男配角奖。

1964年，获[[印度尼西亚|印度尼西亚]]第三届[[亚非电影节|亚非电影节]]最佳男演员奖。

1951年，在[[捷克|捷克]]第6届[[卡罗维·发利国际电影节|卡罗维·发利国际电影节]]上获特别荣誉奖。

1995年，荣获第13届大众电视[[金鹰奖|金鹰奖]]最佳男配角。

1995年，在纪念世界电影诞生100周年，中国电影诞生90周年中，获[[中国电影世纪奖|中国电影世纪奖]]：男演员奖。

2008年9月13日晚，第17届[[金鸡百花电影节|金鸡百花电影节]]在[[大连|大连]]举行闭幕晚会暨颁奖典礼，陈强获第29届[[大众电影百花奖|大众电影百花奖]]终身成就奖。<ref name=" fish" />
