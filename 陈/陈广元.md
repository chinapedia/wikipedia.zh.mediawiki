'''陈广元'''（{{bd|1932年|8月|||CatIdx=Chen}}），经名'''希拉伦丁'''，[[回族|回族]]，[[河北|河北]][[文安县|文安]]人，[[中华人民共和国|中华人民共和国]][[伊斯兰教|伊斯兰教]][[阿訇|阿訇]]，第九届[[中国伊斯兰教协会|中国伊斯兰教协会]]会长<ref name=jianli>{{Cite web |url = http://news.sina.com.cn/china/2000-09-28/131108.html  |title =  中国伊斯兰教协会会长陈广元简历|author =  |date =2000-09-28  |publisher =新浪  |language =  |accessdate =  |quote =  }}</ref>。

==生平==
陈广元生于[[河北省|河北省]][[文安县|文安县]]。1944年，进入[[沈阳清真寺|沈阳清真寺]]任海里凡（宗教学员）。1952年至1954年，进入[[北京市回民学校|北京市回民学院]][[阿拉伯语|阿拉伯语]]专修班学习。1955年至1958年，在[[中国伊斯兰教经学院|中国伊斯兰教经学院]]学习。1958年至1966年，先后在北京[[昌平清真寺|昌平县清真寺]]、[[牛街礼拜寺|牛街礼拜寺]]任阿訇。1966年起，在北京[[东四清真寺|东四清真寺]]任阿訇<ref name=jianli/>。

2000年1月至2016年11月，任[[中国伊斯兰教协会|中国伊斯兰教协会]]会长。其间兼任[[中国伊斯兰教经学院|中国伊斯兰教经学院]]院长，[[北京市伊斯兰教协会|北京市伊斯兰教协会]]会长，[[北京伊斯兰教经学院|北京伊斯兰教经学院]]院长，东四清真寺阿訇，[[中国宗教与和平委员会|中国宗教与和平委员会]]副主席、[[中国国际友好联络会|中国国际友好联络会]]顾问、北京市海外联谊会理事等职务。主持东四清真寺教务，曾发表多篇伊斯兰教方面的文章<ref name=jianli/>。

2008年，当选[[第十一届全国政协|第十一届全国政协]]常务委员<ref>{{cite news|title=第十一届全国政协委员名单|url=http://www.indaa.com.cn/zt/2009qglh/zxzs/200903/t20090303_127573.html|accessdate=2013-04-09|newspaper=新华社北京1月25日电 中国人民政治协商会议第十一届全国委员会委员名单(共2237人，2008年1月25日政协第十届全国委员会常务委员会第二十次会议通过)|date=2008年1月25日|deadurl=yes|archiveurl=https://web.archive.org/web/20130928014403/http://www.indaa.com.cn/zt/2009qglh/zxzs/200903/t20090303_127573.html|archivedate=2013年9月28日}}</ref>，代表宗教界，分入第五十一组。并担任民族和宗教委员会副主任<ref>{{cite news|title=全国政协信息-陈广元|url=http://www.cppcc.gov.cn/CMS/wylibary/showJcwyxtInfoWylibary.action?tabJcwyxt.guid=11W0018250&wytype=wjwy|accessdate=2013-04-09|newspaper=全国政协委员会}}</ref>。2013年3月，当选[[第十二届全国政协|第十二届全国政协]]常务委员。
