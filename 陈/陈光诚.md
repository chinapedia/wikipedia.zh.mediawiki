'''陈光诚'''（{{bd|1971年|11月12日|catIdx=C陳}}），[[山东|山东]][[沂南|沂南]]人，著名中華人民共和國盲人[[維權|維權]]人士<ref name=c11chen>{{cite news|title=纽约大学称陈光诚手机平板电脑未装间谍软件|url=http://cn.nytimes.com/usa/20131211/c11chen/|accessdate=2014-03-12|newspaper=纽约时报中文网|date=2013-12-11|deadurl=no|archiveurl=https://web.archive.org/web/20140312213404/http://cn.nytimes.com/usa/20131211/c11chen/|archivedate=2014-03-12}}</ref>，入选[[美国|美国]]《[[時代雜誌|時代雜誌]]》2006年[[时代百大人物|时代百大人物]]<ref name="time2006">[http://news.bbc.co.uk/chinese/simp/hi/newsid_4960000/newsid_4965500/4965572.stm 五位中国人上美国《时代》周刊风云人物榜 ] {{webarchive|url=https://web.archive.org/web/20060629031715/http://news.bbc.co.uk/chinese/simp/hi/newsid_4960000/newsid_4965500/4965572.stm |date=2006-06-29 }}，BBC，2006年5月2日</ref>。無律師資格，未通過中國[[國家司法考試|國家司法考試]]，未经法律訓練，擁有的法律知识都是自学，藉此维护村民與残疾人士的权益，被媒体譽為「赤脚律师」。

他曾於2006年至2010年入獄，自2005年起妻子、女兒就遭軟禁於家中（陳向[[CNN|CNN]]強調应称“非法监禁”）。2012年4月下旬，他進入了[[北京|北京]]的[[美國駐華大使館|美國駐華大使館]]。5月19日，他與家人離開中國，抵達[[美國|美國]]。

==生平==
===早年生涯===
陈光诚係1971年出生于山東省临沂[[沂南县|沂南县]][[双堠镇|双堠镇]][[东师古村|东师古村]]的一戶家庭。1歲時得眼疾，卻因家貧未就醫，而導致双眼[[失明|失明]]，18歲才就讀小学。

1994年，就读于[[青岛市|青岛市]]盲校。1996年寒假，因为临沂当地政府违反1991年开始实施的《[[中华人民共和国残疾人保障法|中华人民共和国残疾人保障法]]》中残疾人士可减免税收的规定，直到1996年依然在向残疾人士征税，陈光诚与当地政府多次交涉未果的情况下到[[北京|北京]][[上访|上访]]制止临沂地方政府違法徵稅，后临沂地方政府接到上级批示后，终于表态要停止向残疾人士征税。

1997年，家鄉东师古村的[[村委會|村委會]]开始实行[[两田制|两田制]]可能加重了村民的负担。陈光诚通过《[[半月谈|半月谈]]》了解到上级政府不允许搞“两田制”。於1998年夏，到北京上访，最后终于制止了村里的違法兩田制。

1998年，從青岛市盲校畢業，就读于[[南京中医药大学|南京中医药大学]]。2000年到2001年，陈光诚在[[中国法学会|中国法学会]]发起并负责「残疾人维权项目」。2001年，從[[南京中医药大学|南京中医药大学]]畢業。

東師古村因為造紙廠排污，水源惡臭，有村民突然得怪病死亡，村民孩子入學體檢不合格，陳光誠判斷「與河水有關」投入取證，並聯合村幹部募集河流兩岸約四萬名村民的簽名，把造紙廠告上法庭、停止生產並治污。2000年，陳光誠主動申請英國聯邦基金20多萬元人民幣扶貧資金，為村民修建163米的深水井，村民也自發參與水路建設，僅用了政府工程開價的三分之一費用就建成。村民感念表示「沒想到是瞎老五、村裡看上去最沒有用的人，讓俺們吃上了甜井水。」 <ref name=story2012>{{cite news|url=http://epochweekly.com/b5/276/10773p4.htm|title=陳光誠傳 出逃和被困七年的根本原因|work=新紀元雜誌 274期|author=齊先予|date=2012-05-10|deadurl=no|archiveurl=https://web.archive.org/web/20150726094313/http://www.epochweekly.com/b5/276/10773p4.htm|archivedate=2015-07-26}}</ref>

2002年，他试图在北京成立残疾人士的民间维权组织，未果。2003年，陈光诚与外语教师袁伟静结婚。同年7月至8月，陈光诚夫妇作为访问学者，前往美国访问，参观并了解了当地的残疾人士机构。

===控告北京地铁違法對外地残疾人士收费（2003年）===
2003年9月，陈光诚在北京乘坐地铁，检票员以其没有当地的盲人免费乘车证为由，要求其购买车票。但根据《[[中华人民共和国残疾人保障法|残疾人保障法]]》第44条的规定，残疾人可以免费乘坐地铁。陈光诚而后将北京市地铁运营公司告上法庭，并获得胜诉。通过此案，终于使外地盲人也可免费乘坐地铁。

===揭露临沂市政府非法強制墮胎（2005年）===
{{main|临沂计划生育事件}}
临沂市地方政府在计划生育工作上，存在野蛮行为，比如强行对育龄妇女进行绝育手术、对生二胎的孕妇强行堕胎、引产，甚至随意抓捕亲属、逼迫家人交纳巨额罚金。2005年，陈光诚律师和其它人士向媒体和有关部门揭露，陈光诚基本上被[[软禁|软禁]]在家，与外界的通讯也被限制，并且受到多次殴打，一些同情支持他（并将这种同情付诸行为）的村民也曾因他而遭殴打。<ref>[http://news.bbc.co.uk/chinese/simp/hi/newsid_4310000/newsid_4311100/4311180.stm 中国山东盲人维权人士遭到殴打] {{webarchive|url=https://web.archive.org/web/20110728090918/http://news.bbc.co.uk/chinese/simp/hi/newsid_4310000/newsid_4311100/4311180.stm |date=2011-07-28 }}，BBC，2005年10月5日</ref> {{Fact|数名律师先后介入此案，并前往临沂进行调查取证，但受到很大阻力，甚至遭暴力袭击。|time=2018-10-12T05:16:25+00:00}}而后由海外媒体曝光后，受到广泛关注。迫于舆论的压力，[[中华人民共和国国家人口和计划生育委员会|中国国家人口计生委]]官员就临沂计生事件发表了講话，{{Fact|并称要着手调查此事；|time=2018-10-12T05:16:25+00:00}}但此案随后并未出现转折，{{Fact|陈光诚继续被软禁。|time=2018-10-12T05:16:25+00:00}}

===入狱（2006年－2010年）===
2006年3月11日，陈光诚被临沂警方从家中带走后与外界失去联系。6月11日，陈光诚家人收到其被刑事拘留的通知书，說明陈光诚被指控于2月5日与其妻冲击政府大楼及打砸公共财物。8月18日，山东临沂沂南法院开庭审理，陈光诚被指控犯有故意破坏财物罪和聚众扰乱交通秩序罪，当天法庭没有做出裁决。这次的庭审引发极大的争议：陈光诚的辩护律师张立辉以及陈光诚的妻子不被允许进入法庭；辩护律师[[许志永|许志永]]博士前一天晚上被沂南公安以偷窃为借口羁押，在陈光诚庭审结束后许志永才获释；法庭临时指派的两名辩护律师，没有为陈光诚做任何辩护；陈光诚明确拒绝法庭指派律师的情况下，法庭并不依照法律规定进行休庭。<ref>[http://www.rfa.org/mandarin/yataibaodao/chenguangcheng1-20060817.html 陈光诚案开审前夕律师被困警署] {{webarchive|url=https://web.archive.org/web/20110813191635/http://www.rfa.org/mandarin/yataibaodao/chenguangcheng1-20060817.html |date=2011-08-13 }}，RFA，2006年8月17日</ref>8月24日，[[沂南县|沂南县]]人民法院以故意破坏财产和聚众扰乱交通罪，判处陈光诚有期徒刑四年零三个月。<ref>[http://www.rfa.org/mandarin/yataibaodao/chen_guangcheng2-20060824.html 山东盲人维权人士陈光诚被判4年3个月] {{webarchive|url=https://web.archive.org/web/20110813191756/http://www.rfa.org/mandarin/yataibaodao/chen_guangcheng2-20060824.html |date=2011-08-13 }}，RFA，2006年08月24日</ref>

=== 遭受軟禁（2010年－2012年）===
2010年9月9日，陈光诚出狱，被警车接送回家，此后即被软禁在家。当地政府称陈光诚的问题属于“敌我矛盾”。陈光诚家被安装了监控摄影機、手机屏蔽器、强光灯，他和妻子被禁止外出，他家周围由数十人轮流把守，禁止外人来往，其生活用品只能由陈光诚的母亲带入，邻居若提供帮助即被威胁警告。<ref>[http://www.bbc.co.uk/zhongwen/simp/china/2010/09/100927_chenguangchen.shtml 维权律师陈光诚出狱后与家人一起被软禁] {{webarchive|url=https://web.archive.org/web/20101002095023/http://www.bbc.co.uk/zhongwen/simp/china/2010/09/100927_chenguangchen.shtml |date=2010-10-02 }}，BBC，2010年9月28日</ref><ref>Clifford Coonan .[http://www.independent.co.uk/news/world/asia/blind-activist-freed-from-jail-ndash-but-china-is-still-watching-2075433.html Blind activist freed from jail – but China is still watching] {{webarchive|url=https://web.archive.org/web/20110906012321/http://www.independent.co.uk/news/world/asia/blind-activist-freed-from-jail-ndash-but-china-is-still-watching-2075433.html |date=2011-09-06 }} guardian.10 september 2010</ref>在此期间，从各地前去看望的人士（其中包括女性和外籍记者）多次受到拦截、殴打、凌辱、抢劫和强制遣返。<ref name="voa20110924">{{cite web|url=http://www.voanews.com/chinese/news/20110924-Blind-activist-130497003.html|title=陈光诚事件引起一些中国学者的批评|work=美国之音|date=2011-09-24|accessdate=2011-10-23|deadurl=no|archiveurl=https://web.archive.org/web/20111028055740/http://www.voanews.com/chinese/news/20110924-Blind-activist-130497003.html|archivedate=2011-10-28}}</ref>

===離開中國大陸（2012年）===

== 紐約大學施壓爭議 ==

紐約大學則發出嚴正之聲明，表示當初原本就設定一年訪問期，在此一年中，校方提供獎學金、公寓、辦公室、翻譯人員和一名助理﹐並安排討論中國人權的公共活動。紐約大學發言人約翰·貝克曼（John Beckman）更發言說明，紐約大學上學期初向陳光誠表示不可能無限期地提供幫助。表示早在2012年10月，校方就發出書面聲明，表示其住所將於2013年6月到期。
貝克曼說，最初的計劃就是提供陳光誠和家人一年的幫助，讓他們得以在美國「適應和過渡」。<ref>{{cite news|url=http://www.cna.com.tw/News/aCN/201306190293-1.aspx|title=孔傑榮稱陳光誠不該反咬恩人|date=2013-06-19|publisher=中央社|location=台北|language=zh-tw|deadurl=yes|archiveurl=https://web.archive.org/web/20130726072909/http://www.cna.com.tw/News/aCN/201306190293-1.aspx#|archivedate=2013-07-26|access-date=2013-07-22}}</ref>

關於此事，協助陳赴美的紐大教授孔傑榮受訪強調，報導與批評都不是事實，紐大對陳光誠一家已經夠好，人都不該反咬恩人一口。美國華盛頓郵報旗下雜誌「外交政策」（Foreign Policy）電話訪問孔傑榮，詢問他對紐約郵報報導及陳光誠聲明的看法，孔傑榮表明「胡說八道（It's all nonsense!）」，沒有人拿得出證據說紐大限制陳光誠。孔傑榮(Jerome Cohen)也說：「沒有一個政治避難者，哪怕是愛因斯坦，能從美國的學術機構中得到像陳光誠從紐約大學獲得的這樣的幫助。」

至於陳光誠為何發表聲明，孔傑榮說是他要求陳發聲明，表明自己對風波的立場。但是陳光誠顯然已經被某些人帶著走，而這些人與他看法不同，「不過他們也許知道我所不知道的消息」。
另外，孔傑榮表明，因為他並未參與紐大上海分校事務，因此可以客觀看待中國是否藉此要脅紐大。他說，「就事論事，如果我發現紐大限制陳光誠的行動，我一定第一個拉警報」。
孔傑榮說，打從陳光誠一到美國，很多人就開始想誘騙利用陳；他不知道在此風波之後，事態會如何轉變，但陳必須學會言論自由仍然有其代價，「在美國當然可以自由發言，但也需切記可能的負作用與傷害。」

避居法國的異議學者張倫表示(六四後逃離至法國，現任法國塞爾奇。蓬多瓦茲大學副教授)針對此事表示，大陸經常性施壓並不是新聞，所有來自非西方國家的流亡人士，特別是來自中國大陸的流亡人士，可能都會有一些對西方理想化的過高期望。他們對西方社會具體運作的本身似乎也並不是十分清楚。在這裏沒有什麼人能包你一輩子，它自身就是這樣一個自由體制，需要你自己來生存和發展。他說，這可能是許多流亡人士需要面臨的一個挑戰。從這個意義上講，陳光誠可能也面臨這樣的挑戰。他需要從媒體的包圍和光環中解脫出來，調整好心態，安頓好生活，更好地扮演維權活動家的角色。

== 家庭生活 ==
2003年，陈光诚与外语教师袁伟静结婚。2005年7月，女儿陈克斯出生，由于受到父亲被监禁的影响，一直被监禁在家。此外陈还有一个早于其女儿出生的儿子，在其软禁期间被寄养在临沂市[[临沭县|临沭县]]岳父母家。陈光诚的四哥陈光新在沂南县党校工作，是沂南县党校的干部。陈光诚的女儿陈克斯的户口在陈光新名下。
