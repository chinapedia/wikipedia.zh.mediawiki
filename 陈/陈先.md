'''陈先'''（{{bd|1919年|12月|2018年|9月1日}}），[[浙江|浙江]][[宁波|宁波]]人，[[中华人民共和国国家计划委员会|中华人民共和国国家计划委员会]]统计局、[[中华人民共和国国家统计局|国家统计局]]局长<ref name="Chen Xian">鲍杰，情系故里 造福桑梓——记原国家计委常务副主任陈先，宁波通讯2011年05期</ref><ref name=liren>{{Cite web |url =http://www.stats.gov.cn/ztjc/zthd/50znjn/200206/t20020617_35560.html  |title =国家统计局历任局长  |author =  |date =2004-10-18  |publisher = 中华人民共和国国家统计局 |language =  |accessdate =  |quote =  }}</ref><ref name=hxjw>{{Cite web |url = http://www.huaxia.com/lasd/hxrwk/ddrw/dl/2003/08/487260.html |title =陈先  |author =  |date =2003-08-08  |publisher = 华夏经纬网  |language =  |accessdate =  |quote =  }}</ref>。

==生平==
陈先祖居浙江[[鄞县|鄞县]][[姜山镇|姜山镇]]。1919年12月，陈先出生于浙江宁波城区<ref name="Chen Xian"/><ref name=liren/>。曾就读于星荫小学。小学毕业后曾入钱庄做学徒，业余补习英语、数学、经济学<ref name="Chen Xian"/>。1939年，同7名宁波青年一起赴皖南加入[[新四军|新四军]]教导队。1940年，陈先加入[[中国共产党|中国共产党]]。后来历任淮南第一货物检查处处长，[[中共中央华中局|华中]]第三行政区货物管理局局长，华东建设大学（[[山东大学|山东大学]]和华中建设大学于1948年2月合并成立的大学）财经系主任。后任[[山东省|山东省]][[烟台市|烟台市]]工商局局长<ref name=hxjw/>。

1949年5月，陈先参与[[上海市|上海市]]接管工作，曾任上海市军事管制委员会金融处“小五行”特派员，负责联系[[中国通商银行|中国通商银行]]、[[新华银行|新华银行]]、[[国贸银行|国贸银行]]、[[招商银行_(民国)|招商银行]]、[[四明银行|四明银行]]<ref name="Chen Xian"/>。后调到[[福建省|福建省]]，历任[[中国人民银行|中国人民银行]]福建省分行副行长，福建省财经委员会副秘书长，华东财经委员会计划局副局长<ref name=hxjw/>。

1954年陈先调入[[北京|北京]]，参与中国经济统计工作，历任[[中华人民共和国国家计划委员会|中华人民共和国国家计划委员会]]（简称“国家计委”）综合局副局长、局长<ref name="Chen Xian"/><ref name=hxjw/>。

1974年9月，陈先出任国家计委统计局局长。1978年12月，[[中华人民共和国国家统计局|中华人民共和国国家统计局]]挂牌成立，陈先任局长，任至<ref name="Chen statistical">{{cite web|url=http://www.gov.cn/test/2005-06/08/content_4789.htm |title=国家统计局历任局长 |publisher=中华人民共和国中央人民政府|date=2005-06-08}}</ref><ref name=liren/><ref name=hxjw/>。

1981年8月起，陈先先后任国家计委副主任、党组副书记、常务副主任。1986年7月，退出领导岗位，出任国家计委顾问。此后曾任国务院物价小组组长，还曾任国家计委计划研究中心总干事、经济研究所研究员<ref name="Chen Xian"/><ref name=hxjw/>。

1985年10月，出任国务院宁波经济开发区协调小组副组长。是宁波经促会名誉会长。1990年至2005年，担任中国信息协会会长。他还曾任[[中国人民大学|中国人民大学]]兼职教授<ref name="Chen Xian"/><ref name=hxjw/>。

陈先是[[第七届全国人大常委会|第七届全国人大常委会]]委员，[[中共十二大|中共十二大]]代表<ref name="Chen statistical"/>。

[[2018年|2018年]][[9月1日|9月1日]]，於北京逝世<ref>[http://www.xinhuanet.com/politics/2018-09/17/c_1123440388.htm 陈先同志逝世]新华社 2018-09-17</ref>
。
