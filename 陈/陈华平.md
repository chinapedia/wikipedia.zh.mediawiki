'''陈华平'''（{{bd|1957年|6月||}}）[[河南|河南]][[开封|开封]]人，[[中华人民共和国|中华人民共和国]]政府官员<ref name=hns>{{Cite web|url =http://www.henan.gov.cn/zwgk/system/2011/07/22/010259284.shtml|title =省住房城乡建设厅副厅长：陈华平|author =|date =2011-07-22|publisher =河南省人民政府|language =|accessdate =|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20170202005954/http://www.henan.gov.cn/zwgk/system/2011/07/22/010259284.shtml|archivedate =2017-02-02}}</ref>。

==生平==
1976年2月至1983年3月，在建筑工程兵309团服役。1983年3月至1987年6月，历任[[武汉军区|武汉军区]]工程维护大队副连长、连指导员。1987年6月至1997年2月，历任[[中国人民解放军第20集团军|中国人民解放军第20集团军]]高炮旅技术员、指导员、教导员、政治部副主任。1997年2月至2002年3月，任[[中国人民解放军第一五五医院|中国人民解放军第一五五医院]]政委<ref name=hns/>。2002年3月至2007年11月，任中国人民解放军第20集团军装甲旅政委<ref name=hns/><ref name=qf>{{Cite web |url = http://www.chinanews.com/gn/news/2008/10-27/1425740.shtml |title =  前锋-2008演习进入临战状态 将向外军留学生展示|author =  |date = 2008-10-27 |publisher = 中国新闻网 |language =  |accessdate =  |quote =  }}</ref>。2007年11月至2011年6月，任[[济南军区|济南军区]][[中国人民解放军陆军确山合同战术训练基地|确山训练基地]]政委<ref name=hns/><ref name=qf/>。

2011年6月起，转业到地方，出任[[河南省住房和城乡建设厅|河南省住房和城乡建设厅]]副厅长、党组成员<ref name=hns/>。
