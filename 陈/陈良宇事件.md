== 背景 ==
=== 陈良宇相关 ===
{{Main|陈良宇}}
2001年之后，陈良宇先后当选为[[上海市市长|上海市市长]]、[[市委书记|市委书记]]，上任后，陈良宇制定了大量政策以加速上海经济发展并力促上海的民生保障工程，力图在20年内将上海建设成为国际经济、金融、贸易、航运中心，其中有不少政策具有相当程度改革创新性。同时，在陈良宇的推动下，许多重大的市政建设和生产项目上马，上海经济成长迅速，而上海市民的实际生活水平亦得到相当程度的改善。另一方面，由于陈良宇制定政策活跃市场经济，使得上海的房地产业快速繁荣，房价逐步上涨。而中共第四代领导人上台后，意图进行经济[[宏观调控|宏观调控]]防止经济过热，而时[[陈良宇|陈良宇]]却认为经济发展应尊重客观规律，反对过度的行政干预手段，并认为宏观经济调控压制上海发展，于是对中央经济方针进行抵制从而导致中央经济政策无法通行于全国，甚至反驳顶撞时任总理[[温家宝|温家宝]]，而时任[[胡温|胡温]]政府[[团派|本身]]就需要打击削弱[[上海帮|上海帮]]的势力，陈良宇则被视为上海帮的重要成员<ref>{{cite news|url = http://renminbao.com/rmb/articles/2005/2/4/34352b.html |title = 继黄丽满之后 陈良宇即将被踢出上海 |publisher=人民报 |date=2005年2月4日}}</ref>。

===上海社保基金情况===
{{See Also|社会保障基金 (中国)}}
1990年，[[张荣坤|张荣坤]]在江苏创立沸点投资发展有限公司，注册资金3000万元人民币。2000年，又在上海设立上海沸点投资发展有限公司。2002年初，福禧投资控股有限公司在上海成立，注册资本10亿元。
2002年，神秘富豪张荣坤突然崛起于上海商界。上海市政府以32.07亿元[[人民币|人民币]]的价格，将国有的[[上海路桥发展有限公司|上海路桥发展有限公司]]的99.35%股权出售给了当时尚不知名的[[福禧投资有限公司|福禧投资有限公司]]，这意味着福禧的实际控制人张荣坤一举获得了[[沪杭高速公路|沪杭高速公路]]的30年经营权。此后，张荣坤又先后以高价竞拍得[[嘉金高速公路|嘉金高速公路]]25年的经营权。到2005年，张已控制了200公里的收费公路使用权，並入股[[上海电气集团|上海电气集团]]，成为其副董事长。张还在[[华安基金|华安基金]]中占有40%的股份。2005年，他以49亿元资产名列《[[福布斯|福布斯]]》中国富豪榜第16位。但在2002年以前，张荣坤毫无知名度，没有任何出彩的从商经历。从2002年到2006年6月30日，福禧投资控股有限公司资产猛增，达到136.2亿元。但不为人所知的是，张的启动资金全部来自上海社保基金。根据[[中华人民共和国中央人民政府|中国中央政府]]的要求，任何人不得挪用社保基金。{{来源请求}}

== 发展 ==
2005年底，[[香港|香港]]著名[[经济学家|经济学家]][[郎咸平|郎咸平]]接到神秘爆料，称张荣坤的资金来源有问题，郎随即进行调查，发现属实，准备在[[上海电视台|上海电视台]]财经频道的《[[财经郎闲评|财经郎闲评]]》栏目中加以揭发。郎旋即遭到上海市[[有关部门|有关部门]]的封杀，该档节目也于2006年2月被取消<ref>[http://old.news.hexun.com/1713_1625853A.shtml 郎咸平离职长江商学院 “财经郎闲评”已经停播]</ref>。

2006年初，在原[[中共上海市委|中共上海市委]]书记、时任[[国务院副总理|国务院副总理]][[黄菊|黄菊]]被查出患有[[癌症|癌症]]，不得不离职休养后，[[中纪委|中纪委]]开始调查上海社保基金案件，于6月派出[[调查组|调查组]]进驻上海，并于7月初将[[上海市财政局|上海市财政局]]局长[[刘红薇|刘红薇]]调往[[中华人民共和国财政部|财政部]]，以原[[上海市审计局|上海市审计局]]局长[[葛爱玲|葛爱玲]]接任<ref>{{cite news|author1=陈千|title=2006年7月中央地方人事任免|url=http://news.xinhuanet.com/politics/2006-07/27/content_4883874.htm|work=新华网|date=2006-07-27|archiveurl=https://web.archive.org/web/20160815095947/http://news.xinhuanet.com/politics/2006-07/27/content_4883874.htm|archivedate=2016-08-15}}</ref>，对社保基金帐目进行审计。

7月17日，张荣坤和上海市社保局长[[祝均一|祝均一]]被勒令接受调查。在调查中，调查组发现张荣坤涉嫌非法取得上海电气和华安基金的股权，8月中旬，[[上海电气|上海电气]]董事长[[王成明|王成明]]、副总裁[[韩国璋|韩国璋]]相继涉嫌违规，被中纪委公开调查。

随着调查的不断深入，矛头逐渐指向上海市委书记[[陈良宇|陈良宇]]。8月24日，陈良宇原秘书、时任[[上海市宝山区|上海市宝山区]]区长[[秦裕|秦裕]]被[[双规|双规]]。不久，陈在[[黄浦区|黄浦区]]任职时的亲信之一，[[新黄浦集团|新黄浦集团]]总裁[[吴明烈|吴明烈]]落马。

9月24日，胡锦涛召开[[中共中央政治局常委|中共中央政治局常委]]会议，审议了[[中共中央纪律检查委员会|中共中央纪律检查委员会]]《关于陈良宇同志有关问题初核情况的报告》，决定对陈良宇立案调查，并免去其在上海市一切职务，停止其在中央担任的职务。当天，[[中共中央办公厅|中共中央办公厅]]通知陈良宇前来北京参加[[中共十六届六中全会|中共十六届六中全会]]筹备会议，当陈良宇抵达[[北京|北京]]后，即被中纪委宣布“[[双规|双规]]”。次日，中国各媒体全面公开报道陈良宇被停职且双规的消息。

陈良宇被调查之后，相继又有[[孙路一|孙路一]]、[[刘红薇|刘红薇]]、[[韩方河|韩方河]]、[[郁知非|郁知非]]、[[薛全荣|薛全荣]]等人因涉嫌腐败案件而被调查。其中孙路一、韩方河、郁知非等人相继落马被捕。随着案件的不断发展，国家统计局局长邱晓华也因涉案被捕。

2007年1月28日，上海市市长[[韩正|韩正]]说，已全部收回了社保违规资金，连本带息共计37亿元<ref>{{cite news|author1=季华|title=上海已全部收回社保违规资金 连本带息共37亿元|url=http://news.xinhuanet.com/legal/2007-01/29/content_5667038.htm|work=新华网|date=2007-01-29|archiveurl=https://web.archive.org/web/20131001193534/http://news.xinhuanet.com/legal/2007-01/29/content_5667038.htm|archivedate=2013-10-01|deadurl=yes}}</ref>。

2007年6月2日，中共中央政治局常委、国务院副总理[[黄菊|黄菊]]去世，不久，其办公室主任[[王维工|王维工]]因涉嫌向陈良宇通风报信而遭到中纪委“[[双规|双规]]”。据称其已交代出另外30多官员涉嫌犯罪的问题，导致中纪委又单独设立[[专案组|专案组]]进行深入调查<ref>[http://law.cctv.com/20070912/100844.shtml 上海社保案内幕曝光:王维工曾向陈良宇通风报信]</ref>。

== 影响及后果 ==
上海社保基金案爆发之后，[[上海帮|上海帮]]的许多政府官员受到压力，一度甚至有传闻称黄菊也要在此事件中倒台，迫使其于2006年9月抱病复出。最终，黄菊本人并未受波及，于2007年6月2日去世，但是在其去世后，其办公室主任王维工旋即被双规和逮捕。

中共中央则高调宣传对此案的查处，作为自身反对腐败的证明，并提出“特殊利益集团”这一新概念，指出要防止政府官员成为“特殊利益”代理人。<ref>[http://news.sina.com.cn/c/2006-10-05/015810164946s.shtml 我国将防范特殊利益集团衍生]</ref>

随着此案调查的深入，上海的各大房地产商大多被发现有非法获取社保基金的行为，[[罗康瑞|罗康瑞]]等人被迫发表声明，表示将立即归还挪用的社保基金。另外，[[周正毅|周正毅]]的案件又被重新翻起，据传言，已被释放的周正毅又正重新接受调查。目前，此案的调查还在进行中，估计全案的涉案金额将达到100亿元人民币以上。

但亦有不少人认为，类似于上海这样挪用社保基金的行为，每个省市或多或少均有。此案不足以在根本上改变中国的政治经济环境。比如稍晚在广东省发生的社保基金挪用的案件，并未在广东省引发类似反应。<ref>[http://finance.sina.com.cn/money/fund/20070405/11383476079.shtml 广州社保基金被挪用相关责任人已被查处]</ref>

此外由于该案使大批熟悉上海情况的官员遭免职和查处，造成上海政坛的大地震，对[[上海经济|上海经济]]社会发展造成了很大的负面效应（参见：[[上海市地区生产总值|上海市地区生产总值]]），而从投资本质来说，上海社保基金投资获得了盈利，社保基金也得到了扩充<ref>{{cite news|url=http://finance.people.com.cn/GB/5334693.html|title=上海成立处理小组清盘福禧公司偿还社保案债务|publisher=人民网|date=2007年1月27日}}</ref>。在该案爆发不到一年之后，中国[[社保基金|社保基金]]即被用于投资[[京沪高铁|京沪高铁]]的建设并参股营运，而随着中国[[通货膨胀|通货膨胀]]的加剧，社保基金也极大的面临贬值风险<ref>{{cite news|url=http://www.chinahighway.com/news/2007/224148.php|title=京沪高速铁路公司在京成立 社保基金入股|publisher=中国公路网|date=2007年12月28日}}</ref><ref>{{cite news|url=http://www.tianjinwe.com/hotnews/wlrd/201112/t20111219_4821408.html|title=社保基金面临贬值|publisher=天津网|date=2011年12月19日}}</ref>
。2015年，[[李克强|李克强政府]]则进一步扩大了社保基金的投资范围<ref>{{cite news|url=http://news.sina.com.cn/o/2015-04-02/015931671953.shtml|title=李克强：适当扩大全国社保基金投资范围|publisher=重庆商报|date=2015年4月2日}}</ref>。在陈良宇下台几年之后，部分上海市民趋向于怀念其在上海主政时期的政绩，认为陈仅仅是权力博弈的牺牲品，且陈良宇的下台导致了上海经济发展受阻、[[上海人|本地市民]]权益受损等诸多问题<ref>{{cite news |url = http://news.creaders.net/china/subject_details.php?nid=522882&id=495654 |title = “小开”书记陈良宇 罪行越审越缩水 |publisher=明镜网 |date=2012-07-10}}</ref>{{R|上海札记}}。
2007年1月13日出版的《[[2006年中国法治蓝皮书|2006年中国法治蓝皮书]]》将此案及诸多重大法治事件均被收入其中。<ref>{{Cite news|url=http://news.sina.com.cn/c/l/2007-01-14/025612030300.shtml|title=陈良宇案被收入2006年中国法治蓝皮书|website=新浪|date=2007-01-14|author=}}</ref>

==冤案的质疑==
有评论指出，陈良宇的社保投资是一种改革创新性的政策，为中国社保基金的保值增值做了探索性尝试，而陈良宇的三宗罪名都是冤枉。而自从陈良宇被判刑之后，对于[[陈良宇案|陈良宇案]]实际上属“[[冤案|冤案]]”，应当予以“[[平反|平反]]”的呼声，则一直未曾停止过<ref>{{cite news|url=https://boxun1.azurewebsites.net/news/gb/pubvp/2008/04/200804111246.shtml|title=紧急呼吁中央：停止内斗，释放陈良宇，共同对外|publisher=博讯新闻|date=2008年4月11日|deadurl=yes|archiveurl=https://web.archive.org/web/20150713111856/https://boxun1.azurewebsites.net/news/gb/pubvp/2008/04/200804111246.shtml|archivedate=2015年7月13日}}</ref><ref>{{cite news|url=https://boxun1.azurewebsites.net/news/gb/pubvp/2008/04/200804070735.shtml|title=陈良宇案是冤假错案|publisher=博讯新闻|date=2008年4月7日|deadurl=yes|archiveurl=https://web.archive.org/web/20150713105852/https://boxun1.azurewebsites.net/news/gb/pubvp/2008/04/200804070735.shtml|archivedate=2015年7月13日}}</ref>。
