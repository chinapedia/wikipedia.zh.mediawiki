'''陈群'''（{{bd|1890年||1945年|8月17日|catIdx=C}}）乳名'''荣福'''，[[表字|字]]'''人鹤'''，[[号|号]]'''中之'''，[[福建|福建]][[长汀|长汀]][[童坊镇|童坊镇]]人，[[中华民国|中华民国]]政治人物，[[中華民国維新政府|中華民国維新政府]]要人。<ref name=xiamen>[http://www.xmnn.cn/dzbk/xmwb/20061106/200611/t20061106_52039.htm 深山里，大汉奸老宅无人知，厦门晚报，2006-11-06]</ref><ref name=xu>{{Cite book|author = 徐友春主編|title = 民国人物大辞典 増訂版|year = 2007|publisher = 河北人民出版社|isbn = 978-7-202-03014-1}}</ref><ref name=zuixin>{{Cite book|author = |title = 最新支那要人伝|year = 1941|publisher = [[朝日新聞社|朝日新聞社]]|isbn = }}</ref><ref name=liu>{{Cite book|author = 劉寿林等編|title = 民国職官年表|year = 1995|publisher = [[中華書局|中華書局]]|isbn = 7-101-01320-1}}</ref>

==生平==
[[File:Chen_Qun2.jpg|thumb]]
陈群之父在[[福州|福州]]霞浦街经营纸业，因此陈群早年自家乡[[长汀|长汀]]来到[[福州|福州]]念[[私塾|私塾]]。15岁时，陈群在福州城头施放书写有反清标语的风筝，随即遭到拘押。稍长，陈群考入高等警官学校。毕业后，陈群又入水井福建私立法政学校。[[辛亥革命|辛亥]]福州光复後，陈群参加福建学生北伐军。<ref name=xiamen/>

1913年，陈群赴日本留学。<ref name=xiamen/><ref>[[徐友春|徐友春]]主編《[[民国人物大辞典|民国人物大辞典]] 増訂版》及《[[最新支那要人伝|最新支那要人伝]]》作获得[[明治大学|明治大学]][[法学士|法学士]]、[[東洋大学|東洋大学]][[文学士|文学士]]学位，未明时间。《深山里，大汉奸老宅无人知》作1913年赴[[早稻田大学|早稻田大学]]留学，学习法律、经济。</ref>在校期间，参加了[[孙中山|孙中山]]领导的[[中华革命党|中华革命党]]，还曾变卖福州、永安等处的房产资助革命。1916年，陈群回到[[上海|上海]]，在[[环龙路|环龙路]]44号中华革命党总部担任干事。<ref name=xiamen/>1917年（[[民國紀元|民国]]6年）9月，陳群任[[广州|广州]]的大元帥府秘書。<ref name=xu/><ref name=zuixin/><ref name=liu/>[[护法战争|护法战争]]期间，奉孙中山之命准备自上海前往[[漳州|漳州]]从事秘密活动，不料行前被朋友[[李道开|李道开]]出卖，遭到[[松沪护军使|松沪护军使]][[卢永祥|卢永祥]]逮捕。陈群拒绝了卢永祥的招降，险些被卢永祥处决，因[[南北和谈|南北议和]]而脱险。<ref name=xiamen/>

1921年，陈群出任[[广州|广州]]孙中山总统府秘书。1922年6月[[陈炯明|陈炯明]]发动[[六一六事变|六一六事变]]，炮轰总统府，孙中山避难[[永丰舰|永丰舰]]，陈群随舰帮同。1922年10月，上海总部派[[林森|林森]]担任福建省省长，陈群亦奉派随同回福建。此后，直到[[萨镇冰|萨镇冰]]主政福建初期，陈群一直在福建政要之间周旋。后来，陈群离开福建，经上海回到广州。<ref name=xiamen/>

再次回到广州后，1923年（民国12年）5月，陈群任大元帥府宣传委員。此后，陈群任广东[[陆海军大元帅大本营|陆海军大元帅大本营]]的[[中国国民党|中国国民党]]党務準備委員。[[黄埔军校|黄埔军校]]创办后，陈群兼任该校[[三民主义|三民主义]]教员。不久，历任广东民政厅民政科长、警官学校校长、课吏馆馆长。<ref name=xiamen/>

从1926年起，陈群历任中国国民党[[上海市|上海]]政治分会委員、[[国民革命軍|国民革命軍]]東路总指揮部政治部主任、上海警備総司令部軍法处处長、中国国民党中央宣传部駐滬办事处主任、第26軍政治部主任、上海法政学院总務長等职务。<ref name=xu/><ref name=zuixin/><ref name=liu/>
[[File:Chen_Qun1.jpg|thumb]]
[[北伐战争|北伐战争]]开始後，陈群担任北伐军东路军前敌总指挥部政治部主任。1927年初，时任北伐军东路军前敌总指挥部政治部主任的陈群驻[[江西|江西]]。“[[蒋介石|蒋介石]]派遣[[邓祖禹|邓祖禹]]寻陈于江西某旅社。陈随邓晤蒋，蒋以巨款交陈自江西至上海，嘱其就‘[[清党|清党]]’办法商诸[[青帮|青]][[红帮|红帮]]头子[[杜月笙|杜月笙]]、[[张啸林|张啸林]]、[[黄金荣|黄金荣]]。”1927年，陈群奉命前往上海，同杜月笙等人秘密商讨清除[[中国共产党|中国共产党]]势力。4月12日，在陈群等人的策划下，[[四一二事变|四一二事变]]爆发。<ref name=xiamen/>

[[四一二事变|四一二事变]]後，陈群、[[杨虎|杨虎]]、[[潘宜之|潘宜之]]于5月1日联名致电[[宋庆龄|宋庆龄]]称：“[[武汉|武汉]]为贼窃据，夫人以未亡人之身，有何顾虑？应请拼死脱离贼巢，速赴[[南京|南京]]，拥护本党。继续[[孙中山|总理]]遗志，完成国民革命。不然将何面目见先生于地下？痛哭陈词，敬希垂纳。”但此议遭宋庆龄拒绝。<ref name=xiamen/>

蒋介石抵达上海後，于1927年5月11日命陈群主持上海政治工作，任命杨虎为上海警备司令。陈群、杨虎指使手下大刀队，数日便杀害[[中国共产党|中国共产党]]党员和群众400多人。二人手段残酷，被称为“狼虎成群”（“杨虎陈群”[[上海话|上海话]]的谐音）。后两人又被派往[[宁波|宁波]]进行清党。<ref name=xiamen/>

1932年（民国21年）1月，陈群出任[[中华民国内政部|中华民国内政部]]政务次长。不久，改任首都警察厅厅长。由于清党经费问题，陈群逐渐失宠于蒋介石，转而同[[胡汉民|胡汉民]]、[[孙科|孙科]]等人接近，替他们与杜月笙牵线搭桥。1932年，由于孙科受蒋介石逼迫下台，陈群遂随之去职，移居上海，充当[[律师|律师]]，托庇于杜月笙门下。<ref name=xu/><ref name=zuixin/><ref name=liu/>
<ref name=xiamen/>

[[抗日战争|抗日战争]]爆发后，陈群维持上海治安，受日本人的引诱，参与[[梁鴻志|梁鴻志]]组织日本傀儡政府的活动。1938年3月，[[中华民国维新政府|中华民国维新政府]]成立，陈群出任内政部长。同年9月，維新政府与[[王克敏|王克敏]]的[[中華民国臨時政府_(北京)|中華民国臨時政府]]组织[[中華民国政府联合委員会|中華民国政府联合委員会]]，陳群任委員。<ref name=xu/><ref name=zuixin/><ref name=liu/>

1940年3月[[汪精卫政权|汪精卫政权]]成立后，陈群任[[行政院_(汪精卫政权)|行政院]][[中华民国内政部_(汪精卫政权)|内政部长]]、文治委員会委員、[[中央政治委员会_(汪精卫政权)|中央政治委員会]]委員。其后，历任[[憲政实施委員会|憲政实施委員会]]常務委員、[[中日文化協会|中日文化協会]]常務理事、[[東亚联盟|東亚联盟]]中国总会常務理事、[[国民運動促進委員会|国民運動促進委員会]]常務委員、[[中央警察学校|中央警察学校]]校長、[[物資調査委員会|物資調査委員会]]委員長、[[新国民運動促進委員会|新国民運動促進委員会]]主任委員等职务。1941年内政部由日本知照设立青年团训练所，陈群出任所长。陈群在内政部任上，曾访问日本三次。1943年（民国32年）9月，[[江蘇省_(汪精卫政权)|江蘇省]]省長[[李士群|李士群]]被暗杀后，汪精卫得日本方面江苏顾问金子同意，调陈群任江苏省省长。1944年[[汪精卫|汪精卫]]死后，1944年11月，陈群就任[[考试院|考试院]]院长。<ref name=xu/><ref name=zuixin/><ref name=liu/><ref name=xiamen/>

1945年8月[[日本投降|日本投降]]後，1945年8月17日，陈群在南京寓所内服毒自杀身亡，自杀前手书《自剖书》。<ref name=xiamen/>

1942年，陈群在南京[[颐和路|颐和路]]2号建成一座三层楼的泽存书库，以收藏他搜罗的40万册新旧图书。后来此处曾是[[南京图书馆|南京图书馆]]古籍部。

==家庭==
*子：[[陈志|陈志]]，生于1938年，曾任北京[[中国广播吉他乐团|中国广播吉他乐团]]团长、艺术指挥，中国吉他学会理事长，中美音乐学会主席等职。<ref name=xiamen/>
