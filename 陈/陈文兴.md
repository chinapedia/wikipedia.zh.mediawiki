'''陈文兴'''（1964年12月-），[[纺织|纺织]]学教授，博导，获得[[浙江大学|浙江大学]][[理学博士|理学博士]]和[[日本|日本]][[信州大学|信州大学]][[工学博士|工学博士]]学位。曾任[[浙江理工大学|浙江理工大学]]材料与纺织学院常务副院长<ref >{{cite web|url=http://daxue.netbig.com/951/teacher/115110/ | language= | title=高校师资陈文兴 |publisher=网大 |date= |accessdate=2016年6月 }}</ref>，浙江理工大学副校长<ref>{{cite web |url=http://www.sust.edu.cn/info/1070/2746.htm |language= |title=浙江理工大学副校长陈文兴来校考察调研 |publisher=陕西科技大学 |date=2015-11-23 |accessdate= |archive-url=https://web.archive.org/web/20160809185201/http://www.sust.edu.cn/info/1070/2746.htm |archive-date=2016-08-09 |dead-url=yes }}</ref>。2016年6月被任命为浙江理工大学校长，以接替[[裘松良|裘松良]]。<ref >{{cite web|url=http://news.163.com/16/0624/14/BQB5SEQ600014JB5.html | language= | title=浙江任免季晓斌、徐敏俊、曹水萍、劳晓峰、王俭、陈文兴等职务 |publisher= 中国经济网 |date=2016-06-24 |accessdate= }}</ref>

== 荣誉头衔 ==
*浙江省特级专家
*[[国务院学位委员会|国务院学位委员会]]第六届学科评议组成员
*“先进纺织材料与制备技术”教育部重点实验室主任<ref >{{cite web|url=http://www.gk114.com/yjsyx/3078/3614/20120910144658.htm | language= | title=浙江理工大学博士生导师：陈文兴简介 |publisher= 中国招生考试网|date=2012-09-10 |accessdate= }}</ref>
*浙江省重中之重一级学科“纺织科学与工程”学科带头人

== 研究方向 ==
*功能性纤维
*功能高分子材料
*蚕丝蛋白质
