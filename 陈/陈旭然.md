'''陈旭然'''（{{bd|1968年|8月21日|1998年|12月30日}}），原[[广东电视台|广东电视台]]女节目主持人、制片人，[[太平洋影音公司|太平洋影音公司]]签约歌手。<ref>[http://www.gmw.cn/01shsb/1999-01/16/GB/861%5ESH12-111.htm 解读陈旭然]</ref><ref>[http://week-hzrb.hangzhou.com.cn/system/2009/09/01/010163011.shtml 陈旭然照片《摄影演变30年》] {{webarchive|url=https://web.archive.org/web/20140910200229/http://week-hzrb.hangzhou.com.cn/system/2009/09/01/010163011.shtml |date=2014-09-10 }}</ref><ref>[http://www.gmw.cn/01shsb/1999-01/16/GB/861%5eSH12-104.htm 陈旭然被杀之谜]</ref><ref>{{Cite web |url=http://www.show160.com/news/new-671.html |title=广东主持人陈旭然被害案揭秘 |access-date=2014-05-16 |archive-url=https://web.archive.org/web/20140517130621/http://www.show160.com/news/new-671.html |archive-date=2014-05-17 |dead-url=yes }}</ref><ref>[http://news.ifeng.com/society/1/200909/0901_343_1328722.shtml 广东电视台主持人陈旭然被害 警方引蛇出洞]</ref><ref>{{Cite web |url=http://news.21cn.com/zhuanti/social/tanjing/dongtai/2008/04/21/4624554.shtml |title=陈旭然之死疑点重重 |access-date=2014-05-16 |archive-url=https://web.archive.org/web/20160304201332/http://news.21cn.com/zhuanti/social/tanjing/dongtai/2008/04/21/4624554.shtml |archive-date=2016-03-04 |dead-url=yes }}</ref><ref>[http://ent.sina.com.cn/s/m/28616.html 死后身价反而“暴涨” 主持人陈旭然成摇钱树]</ref>

== 生平 ==
*1968年8月21日，在[[广州|广州]]出生于一个[[祖籍|祖籍]][[广东省|广东省]][[肇庆市|肇庆市]][[高要县|高要县]]的干部家庭；
*1984年，在[[广州市旅游职业高级中学|广州市旅游职业高级中学]]毕业后分配到[[远洋宾馆|远洋宾馆]]任服务员；
*1985年，加入[[广东电视台|广东电视台]]开始电视主持生涯，先后主持过《早晨》、《家庭百事通》等栏目，后调入文艺部，主持《[[万紫千红|万紫千红]]》、《共度好时光》、《一周荧屏》等[[广东电视台|广东电视台]]名牌栏目和各种大型文艺晚会，多次获得国家级和省级奖励；期间通过进修取得[[大专|大专]][[学历|学历]]。
*1995年，成为当时[[广东电视台|广东电视台]]最年轻的女制片人。
*1996年，发行个人歌曲[[CD|CD]]《情与梦的絮语》（[[发行商|发行商]]为[[太平洋影音公司|太平洋影音公司]]）。
*1998年12月30日，在其位于[[广州市|广州市]][[天河区|天河区]][[龙口西路|龙口西路]]92号天成大厦1座31楼的家中被入室[[抢劫|抢劫]]的原天成大厦[[保安员|保安员]][[丁国礼|丁国礼]]杀害，[[骨灰|骨灰]]葬于[[珠海|珠海]]。后丁国礼于1999年1月14日被警方抓获并于同年4月被广东省高级人民法院判处[[死刑|死刑]]。

== 家庭及情感 ==

=== 主要亲属 ===
陈旭然家中有父母及两个姐姐，其中父母现已[[移民|移民]][[澳大利亚|澳大利亚]]，一个姐姐嫁到[[澳大利亚|澳大利亚]]，另一个姐姐陈妙然曾任[[广百股份|广州百货大厦]]副总经理。

=== 婚姻及绯闻 ===
陈旭然早年曾与同为[[广东电视台|广东电视台]]节目主持人[[区志航|区志航]]结婚，后于1995年5月[[离婚|离婚]]。之后外界一直盛传其与时任[[广东电视台|广东电视台]]台长谢望新的[[绯闻|绯闻]]，陈旭然遇害不久谢望新即被调离广东电视台更使不少人相信两人之间存在亲密关系。也有曾与陈旭然共事的匿名人士指陈遇害前已经和一[[澳大利亚|澳大利亚]]籍男子秘密登记结婚并准备移民出国。

== 死因争议 ==
尽管警方公布陈旭然死于抢劫案，但由于其复杂的感情经历。故坊间猜测其可能死于情杀或仇杀。

== 相关作品 ==
张欣[[小说|小说]]《沉星档案》曾于2001年被改编为同名[[电视剧|电视剧]]<ref>[http://ent.sina.com.cn/v/2002-01-25/71029.html 湖北电视台今日播出电视剧《沉星档案》]</ref>，被外界认为是影射陈旭然之死。但原作者否认其作品影射该事件。<ref>[http://ent.sina.com.cn/s/m/28656.html 作家张欣否认《沉星档案》影射被害主持人陈旭然]</ref>
