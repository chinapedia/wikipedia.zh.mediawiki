'''陈燃'''{{BD|1985年|11月24日|||catIdx= C陈燃}}是中国知名[[模特儿|模特儿]]和[[演员|演员]]，被誉为“中国时尚教主”。<ref name=" fhws" >{{cite web |url=http://fashion.ifeng.com/photo/mm/detail_2012_03/12/13127470_0.shtml |title=安徽姑娘陈燃成中国时尚圈新宠 大胆穿着成就地位 |accessdate= 2012年|work= |publisher=凤凰网 |language =zh-cn}}</ref>

== 生平 ==
1985年出生在[[安徽省|安徽省]][[宿州|宿州]]。本名陈瑞，因热爱[[阿根廷国家足球队|阿根廷队]]，2010年世界杯期间与朋友打赌“如果阿根廷输了，我就改名！”，赌输后改名为“陈燃”。<ref name="txws " >{{cite web |url=http://ent.qq.com/a/20100709/000428.htm |title=陈瑞为阿根廷改名陈燃 写真曝光秀甜美性感 |accessdate= 2010年|work= |publisher=腾讯网 |language =zh-cn}}</ref>
她自己解释改名的原因说：“这个字有[[凤凰|凤凰]]涅盘，浴火重生的意思，我相信阿根廷不会就此一蹶不振，期待[[阿根廷国家足球队|蓝白军团]]能浴火前行，四年后再将[[大力神杯|大力神杯]]高高捧起。同时也希望我自己能够以全新的面貌再出发。”<ref name="txws " />在[[北京电影学院|北京电影学院]]2002級表演系毕业，现居于[[北京|北京]][[朝阳区|朝阳区]]，
