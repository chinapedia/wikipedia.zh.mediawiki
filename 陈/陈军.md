'''陈军'''（{{bd|1959年|8月|}}），女，[[中華人民共和國|中華人民共和國]]人，祖籍[[臺灣|臺灣]][[臺東縣|臺東]][[阿美族|阿美族]]，同時亦為[[中國|中國]]政治人物。

==生平==
陈军出生於1959年；他的[[父親|父親]]陳連生在十九歲時被[[國民政府|國民政府]]徵召參與[[國共內戰|國共內戰]]<ref>{{Cite web|url=|title=原民子孫從北京來尋根|accessdate=2015-04-23|author=謝恩得、羅緗綸、段鴻|date=2007-02-08|publisher=《聯合報》|language=zh-tw}}</ref>。

陈军毕业于[[北京师范大学|北京师范大学]]体育教育论专业，並加入[[台湾民主自治同盟|台湾民主自治同盟]]（台盟），擔任台盟北京市委常务副主委及北京市[[中华全国台湾同胞联谊会|台联]]副会长<ref>{{cite news|url=http://www.huaxia.com/jtsd/jtwl/2015/03/4323565.html|title=台盟北京市委和北京市台联共同举办"2015年在京台胞学习‘两会’精神报告会"|work=华夏经纬网|accessdate=2015-04-23}}</ref>。

2008年，開始担任[[全国人大代表|全国人大代表]]<ref>{{cite news|url=http://www.npc.gov.cn/delegate/viewDelegate.action?dbid=112713|title=全国人大代表信息-陈军|newspaper=全国人大网|accessdate=2013-04-05}}</ref><ref>{{cite news|url=http://www.npc.gov.cn/delegate/viewDelegate.action?dbid=122715|title=全国人大代表信息-陈军|newspaper=全国人大网|accessdate=2013-04-05}}</ref>。2018年，当选[[第十三届全国人大常委会|第十三届全国人大常委会]]委员<ref name=cwh>{{Cite web |url =http://www.npc.gov.cn/npc/xinwen/2018-03/19/content_2051300.htm  |title = 中华人民共和国全国人民代表大会公告（第五号） |author =  |date =2018-03-19  |publisher =  中国人大网|language =  |accessdate =  |quote =  }}</ref>。
