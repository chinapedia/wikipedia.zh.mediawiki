'''四川苏铁'''（[[学名|学名]]：{{lang|la|'''''Cycas szechuanensis'''''}}<ref name = "C132"/>），又稱'''南盤江蘇鐵'''、'''貴州蘇鐵'''<ref name = "iucn" /><ref name=eflora />，为[[苏铁科|苏铁科]][[苏铁属|苏铁属]]的植物<ref name = "COL">{{cite web |url= http://www.catalogueoflife.org/annual-checklist/2014/details/species/id/4345946 |title=Species 2000 & ITIS Catalogue of Life: 2014 Annual Checklist.|accessdate=2014-05-26 |editors=Roskov Y., Kunze T., Orrell T., Abucay L., Paglinawan L., Culham A., Bailly N., Kirk P., Bourgoin T., Baillargeon G., Decock W., De Wever A., Didžiulis V. |year=2014 |publisher=Species 2000 |location=Reading, UK.}}</ref><ref name="source">{{cite web |url=http://www.iopi.org |title=IOPI Global Plant Checklist |deadurl=yes |archiveurl=http://webarchive.loc.gov/all/20090612184547/http://www.iopi.org/ |archivedate=2009-06-12 }}</ref>，是中国的特有植物。分布在[[中国大陆|中国大陆]]的[[四川|四川]][[峨眉山|峨眉山]]、[[雅安|雅安]]、[[乐山|乐山]]及[[福建|福建]][[南平|南平]]等地，常生长在山谷溪边，目前尚未由人工引种栽培<ref>{{cite web| url =http://www.bioinfo.cn/db05/KmzwSpecies.php?action=view&id=6236| title =四川苏铁| work =《中国高等植物数据库全库》| publisher =中国科学院微生物研究所| author =[[昆明植物研究所|昆明植物研究所]]| accessdate =2009-02-24| language =zh-hans| deadurl =yes| archiveurl =https://web.archive.org/web/20160305045323/http://www.bioinfo.cn/db05/KmzwSpecies.php?action=view&id=6236| archivedate =2016-03-05}}</ref>。
