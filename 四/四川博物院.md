'''四川博物院'''是在[[中国|中国]][[四川省|四川省]][[成都市|成都市]]城西[[青羊区|青羊区]]，[[浣花溪公园|浣花溪公园]]旁，始建于1941年，是[[中国西南地区|中国西南地区]]规模最大的综合性博物馆，藏品数量在[[中国大陆|中国大陆]]各类博物馆中位居第六，现有院藏文物26万余件，其中珍贵文物5万余件，[[国家一级文物|国家一级文物]]1399件。

== 简介 ==
2012年，四川博物院被[[国家文物局|国家文物局]]核定为第二批[[国家一级博物馆|国家一级博物馆]]<ref>{{Cite web |url = http://www.gov.cn/zwgk/2012-12/07/content_2285163.htm |title =关于发布第二批国家一级博物馆名单的通知（文物博函〔2012〕2237号）  |author =  |date =2012-12-07  |publisher = 中国政府网 |language =  |accessdate =  |quote =  }}</ref>。

四川博物院藏品分为[[陶瓷|陶瓷]]类、[[砖石|砖石]]类、[[金属|金属]]类、[[钱币|钱币]]类、[[书画|书画]]类、民族民俗类、[[碑帖|碑帖]]类、近现代史类八大类三十余小类文物，具有浓郁的巴蜀地方特色。这些藏品非常多樣，上至25000年前旧石器时代的“[[资阳人|资阳人]]”头骨化石，下至近几年中國著名艺术家的作品，为研究四川各时期的[[历史|历史]]、[[文化|文化]]、[[经济|经济]]、[[政治|政治]]、[[军事|军事]]提供资料。

四川博物院展览面积达12900平方米，15个展厅，可一次性展出文物3300余件，可举办大中小各类文物和艺术品展，并拥有[[多媒体|多媒体]]陈列展示。此外，博物院设有现代化的对外文化交流中心和多功能学术报告厅，可为博物馆学术研究与文化交流提供平台。<ref>[http://www.scmuseum.cn/zh-gb/html/Show1_1.aspx 四川博物院简介] {{webarchive|url=https://web.archive.org/web/20100601051434/http://www.scmuseum.cn/zh-gb/html/Show1_1.aspx |date=2010-06-01 }}</ref>
