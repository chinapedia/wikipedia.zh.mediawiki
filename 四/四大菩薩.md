{{佛教}}
'''四大菩薩'''，是[[佛教|佛教]]信仰中，四尊[[菩薩|菩薩]]的組合，意義因宗派或依據經典而異。

關於四大菩薩有多種說法，在[[中國|中國]]最流行的說法是[[觀世音菩薩|觀音]]、[[文殊菩薩|文殊]]、[[普賢菩薩|普賢]]、[[地藏王菩薩|地藏]]，中國有[[四大佛教名山|四大菩薩道場]]之說，即是四大菩薩的聖地，<ref>[https://religion.moi.gov.tw/Knowledge/Content?ci=2&cid=622 四大菩薩, 全國宗教資訊網, 宗教知識 2018-09-09]</ref>
或由[[八大菩薩|八大菩薩]]選出四位大菩薩。

==通說==
* [[文殊菩薩|文殊菩薩]] － [[釋迦牟尼|釋迦牟尼]]左[[脅侍|脅侍]]，以「[[智慧|智慧]]」聞名而稱大智文殊菩薩。
* [[普賢菩薩|普賢菩薩]] － [[釋迦牟尼|釋迦牟尼]]右脅侍，以「行德」聞名而稱大行普賢菩薩。
* [[觀音菩薩|觀音菩薩]] － [[阿彌陀佛|阿彌陀佛]]左脅侍，以「慈悲」聞名而稱大悲觀音菩薩。
* [[地藏菩薩|地藏菩薩]] － [[娑婆三聖|娑婆三聖]]右脅侍，以「願力」聞名而稱大願地藏菩薩。
 
在[[彌勒佛|彌勒佛]]時代尚未來臨以前，釋迦牟尼佛與脅侍大智文殊菩薩、大行普賢菩薩會以佛法濟度眾生，為[[華嚴三聖|華嚴三聖]]。[[觀音菩薩|觀音菩薩]]以慈悲聞名，常以各種化身救濟世人苦難，廣受世人尊敬。[[地藏王菩薩|地藏王菩薩]]則發誓「地獄未空，誓不成佛」，願意犧牲自己、不立即成佛的救度[[地獄|地獄]][[眾生|眾生]]。一般常把文殊、普賢、觀音、地藏，合稱「四大菩薩」。

=== 道场 ===
[[中國|中國]]有四大菩薩道場（[[四大佛教名山|四大佛教名山]]）之說：觀世音菩薩的道場是[[浙江|浙江]][[普陀山|普陀山]]；文殊菩薩的道場是[[山西|山西]][[五台山|五台山]]；普賢菩薩的道場是[[四川|四川]][[峨眉山|峨眉山]]；地藏菩薩的道場是[[安徽|安徽]][[九華山|九華山]]。

阿彌陀佛另外一位脅侍：專修念佛法門的[[大勢至菩萨|大勢至菩萨]]，在[[藏传佛教|藏传佛教]]中示現為代表[[神通|神通]]力、現憤怒相的[[金剛手菩薩|金剛手菩薩]]，占有相当重要的地位。其顯化道場在[[江蘇|江蘇]][[狼山_(江蘇)|狼山]]。

== 密教的四菩薩 ==
根據[[密宗|密教]]的「胎藏曼荼羅」，在曼荼羅中央的「中台八葉院」的四菩薩是'''[[普賢菩薩|普賢菩薩]]'''、'''[[文殊菩薩|文殊菩薩]]'''、'''[[觀音菩薩|觀音菩薩]]'''、'''[[彌勒菩薩|彌勒菩薩]]'''。

== 華嚴經的四菩薩 ==
《[[華嚴經|華嚴經]]》的四菩薩是'''法慧'''、'''功德林'''、'''金剛幢'''、'''金剛藏'''四菩薩。

== 天台宗的四菩薩 ==
[[天台宗|天台宗]]系的四菩薩是[[阿彌陀如來|阿彌陀如來]]作為脇侍的'''金剛法'''、'''金剛利'''、'''金剛因'''、'''金剛語'''四菩薩。

== 日蓮宗・法華宗的四菩薩 ==
[[日蓮宗|日蓮宗]]・[[法華宗|法華宗]]根據《[[法華經|法華經]]》從地涌出品第十五的記述，以『法華經』中的'''上行'''、'''無邊行'''、'''淨行'''、'''安立行'''為四菩薩（或稱四士）。
