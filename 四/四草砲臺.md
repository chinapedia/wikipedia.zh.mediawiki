'''四草砲臺'''位於[[臺南市|臺南市]][[安南區|安南區]]，於民國七十四年（1985年）8月19日二級古蹟{{r|文}}。該建築為清道光年間的[[臺灣兵備道|臺灣兵備道]][[姚瑩|姚瑩]]所建的十七座砲臺之一{{r|石}}，其現存部分現在變成了'''臺南市鎮海國小'''的圍牆{{r|傅}}。

== 沿革 ==
因[[第一次鴉片戰爭|第一次鴉片戰爭]]的爆發，為強化臺灣的海防，當時的臺灣兵備道姚瑩於1840年修築了俗稱四草砲臺的鎮海城{{r|石}}，同時也修築了位於安平的[[安平小砲臺|安平小砲臺]]，分別拱衛安平大港之外港「四草湖」的北岸與南岸{{r|遠}}。

因為創建當時的防務相當急迫，所以最初是用裝沙的竹簍及麻袋等做成的臨時性防禦工事{{r|遠}}，設有30丈（約96公尺）長的護牆、十座砲墩{{r|傅}}和砲座1500斤二門、1000斤三門、800斤二門，此外在砲墩外挖有暗藏二萬支尖竹籤的壕溝，壕溝外又埋了800個釘桶、800塊釘板與兩萬枚鐵蒺藜{{r|王|page=261}}。後來翌年（1841年）改築在牆腰開有八字形（兩端口大，中央較小）砲洞的石牆，為較為永久的形式{{r|傅}}。

然而之後隨著駐軍調離、四草湖外港被廢，該砲臺遂因年久失修而坍毀{{r|石}}。二次大戰結束進入[[中華民國|中華民國]]時期後，1951年<ref>[http://www.jhps.tn.edu.tw/php_about/about.php 臺南市鎮海國小] {{webarchive|url=https://web.archive.org/web/20120414035957/http://www.jhps.tn.edu.tw/php_about/about.php |date=2012-04-14 }}</ref>鎮海國小在殘存的牆垣西邊建立校舍，四草砲臺所剩部分便成了學校外牆。1969年，因颱風來襲導致南段牆垣倒塌{{r|傅}}，後來在1977年修復{{r|石}}。近年曾再整修，而呈現今天的樣子{{r|傅}}。

== 建築特色 ==
該砲臺城垣是以鵝卵石、三合土與長條石板所建{{r|石}}，牆上的八字形砲洞外寬中窄，以便更改射角{{r|傅}}。而現在城垣上長了不少老樹，與砲臺城垣形成特殊的景緻{{r|傅}}。

== 其他 ==
在砲臺的對面是[[四草大眾廟|四草大眾廟]]，而在大眾廟後有存放[[台灣荷蘭統治時期|台灣荷蘭統治時期]]於此與[[陳澤|陳澤]]所率領之鄭軍作戰身亡的荷蘭軍隊遺骨的「海靈佳城」塚，乃史學家[[黃典權|黃典權]]集中其遺骨所立{{r|王|page=21}}。
