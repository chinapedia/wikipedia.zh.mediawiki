'''四人幫'''为[[中國共產黨|中國共產黨]]“[[文化大革命|無產階級文化大革命]]”時期（1966年—1976年）形成的一個[[政治|政治]]集團的名稱，形成於1973年[[中國共產黨第十次全國代表大會|中國共產黨第十次全國代表大會]]之後，其成員按“[[粉碎四人帮|粉碎四人幫]]”時[[中共中央|中共中央]]公布的顺序依次为[[王洪文|王洪文]]、[[张春桥|张春桥]]、[[江青|江青]]和[[姚文元|姚文元]]。江青为[[毛泽东|毛泽东]]妻子，张春桥、姚文元和王洪文均由毛泽东从[[上海市|上海]]提拔到中共中央并委以重任，四人在文革后期皆为[[中国共产党中央政治局|中共中央政治局]]委员，王洪文和张春桥二人更是成为排名第四和第九的[[中共中央政治局常委|中共中央政治局常委]]。1976年10月6日，[[毛泽东逝世|毛泽东逝世]]一个月后，时任中国[[国务院总理|国务院总理]][[华国锋|华国锋]]、时任[[中共中央军委副主席|中共中央军委副主席]][[叶剑英|叶剑英]]、时任[[中共中央办公厅|中共中央办公厅]]主任[[汪东兴|汪东兴]]与时任中共中央军委常务委员[[陈锡联|陈锡联]]以及时任[[北京市革命委员会|北京市革命委员会]]主任[[吴德|吴德]]等人联手将王张江姚四人拘禁，其主要追随者亦在随后的政治斗争中遭到清洗。

1980年四人帮被公审的时候，四人帮被正式命名为“[[林彪、江青反革命集团案|江青反革命集团]]”。

== 词源 ==
1974年7月17日，在[[中国共产党中央政治局|中共中央政治局]]会议上，[[中国共产党中央委员会主席|中共中央主席]][[毛泽东|毛泽东]]批评[[王洪文|王洪文]]、[[张春桥|张春桥]]、[[江青|江青]]、[[姚文元|姚文元]]搞[[幫派|帮派]]活动，说：“她（江青）算上海帮呢！你们要注意呢，不要搞成四人小宗派呢。”这是历史上第一次将江青等四人当成一个宗派提出。

1975年5月3日深夜，毛泽东召集在北京的中共中央政治局委員，在他的住所召開中央政治局會議，親自講話來劝告四人幫。這次會議是他生前最後一次主持政治局會議，也是他生前最後一次長篇講話。談話時，他再次提到「三要三不要」（要搞[[马克思列宁主义|馬列主義]]，不要搞[[修正主义_(马克思主义)|修正主義]]；要團結，不要分裂；要光明正大，不要搞陰謀詭計），並進一步對江青等四人說：「（你們）不要搞『四人幫』，你們不要搞了，為什麼要照樣搞呀？為什麼不和二百多個中央委員搞團結？搞少數人不好，歷來不好。」此後，「四人幫」就成了江青等四人的代稱。但毛与四人的关系有增无减。

== 历史 ==
四人帮是[[文化大革命|文化大革命]]期间一派重要政治势力，被视为[[毛泽东|毛泽东]]贯彻并到后期扭曲其文革政治思想的主要推手。

1965年，在毛泽东的支持下，由江青、张春桥指使，姚文元编写的《[[评新编历史剧《海瑞罢官》|评新编历史剧〈海瑞罢官〉]]》成为发动文化大革命的导火线。文革前期，江青、张春桥、姚文元一直是[[中央文化革命小组|中央文革小组]]的主要成员，对文革的发动和打倒所谓“[[刘少奇|刘少奇]]、[[邓小平|邓小平]][[资产阶级|资产阶级]]司令部”起到了至关重要的作用。1969年[[中国共产党第九次全国代表大会|中共九大]]上，三人全部进入[[中国共产党中央政治局|中共中央政治局]]；也是在这次大会上，林彪成为了唯一的[[中國共產黨中央委員會副主席|中共中央副主席]]、毛泽东的法定接班人。

1970年的[[中国共产党第九届中央委员会第二次全体会议|中共九届二中全会]]上，在[[林彪|林彪]]的支持下，[[陈伯达|陈伯达]]等人对张春桥发起了含沙射影的猛烈攻击；但在毛泽东的支持下，大会转而批判陈伯达。1971年，在“副统帅”林彪[[九一三事件|出逃并丧命]]之后，造反派起家的王洪文从上海上调中央，不久就和江青等三人结成联盟。1973年[[中国共产党第十次全国代表大会|中国共产党第十次全国代表大会]]上，王洪文被选为排名第二（仅在[[周恩来|周恩来]]之后）的中共中央委员会副主席，四人帮的势力空前强大。

1974年1月，江青等人发动了“[[批林批孔运动|批林批孔运动]]”，将矛头指向[[周恩来|周恩来]]。1974年10月周恩来入院后，[[邓小平|邓小平]]重新崛起，出任中共中央副主席、国务院第一副总理、中共中央军委副主席兼解放军总参谋长（合稱「三副一长」），实际负责中央日常工作。特别是在1975年的[[第四届全国人民代表大会|第四届全国人民代表大会]]上，四人帮的组阁计划以失败告终，四人帮的势力受到沉重打击。

晚年的毛泽东虽然并未将最高权力交给四人帮，但在第四届人大之后也没有再削弱四人帮的势力。相反，1976年1月[[周恩来逝世|周恩来逝世]]后，毛泽东却以“[[批邓、反击右倾翻案风|批邓、反击右倾翻案风]]”为名再次打倒邓小平。在逝世前，毛泽东将[[华国锋|华国锋]]确立为中共中央第一副主席兼[[国务院总理|国务院总理]]。

四人帮在毛泽东逝世时的具体职务是（按中共党内地位高低顺序）：王洪文任[[中国共产党中央政治局常务委员会|中共中央政治局常委]]、[[中國共產黨中央委員會副主席|中共中央副主席]]；张春桥任中共中央政治局常委、[[中华人民共和国国务院副总理|国务院副总理]]、[[中国人民解放军总政治部|解放军总政治部]]主任；姚文元任中共中央政治局委员，主管全国的[[舆论|舆论]]；江青任中共中央政治局委员，虽然没有具体权力，却以“毛泽东妻子”的特殊身份成为四人帮的核心人物。

== 粉碎四人帮 ==
{{see|中国人民解放军61889部队|粉碎四人帮}}
1976年4月，有很多群众在悼念周恩来的[[四五运动|四五运动]]中表示了对四人帮和文革的不满。同年9月9日，[[毛泽东之死|毛泽东逝世]]。

1976年10月6日，国务院总理[[华国锋|华国锋]]、中央军委副主席[[叶剑英|叶剑英]]、国务院副总理[[李先念|李先念]]等人在获得中央警卫部队领导，[[中共中央办公厅|中共中央办公厅]]主任[[汪东兴|汪东兴]]的支持后進行[[粉碎四人帮|怀仁堂事变]]，分别拘留了四人帮及其支持者（如[[毛远新|毛远新]]、[[马天水|马天水]]等人），继而隔离审查，一般以此事件作为“文化大革命”结束的标志。由于这个过程没有经过法律程序而是靠军队（警卫部队）完成的，所以也被认为是一种[[政变|政变]]。

事后华国锋出任[[中国共产党中央委员会主席|中国共产党中央委员会主席]]和[[中国共产党中央军事委员会主席|中国共产党中央军事委员会主席]]，成为[[党和国家最高领导人|党和国家最高领导人]]。王张江姚四人随后被中共中央永远开除党籍，当时给这四个人的定性是“形左实右”。中共中央对王、张、江、姚四人发起大批判，罪名包括“破坏无产阶级大革命”、“破坏[[批林批孔|批林批孔]]”等等。<ref>在全军政治工作会议上的讲话（一九七八年六月二日）：“林彪、四人帮是一伙，早就勾结在一起，破坏无产阶级文化大革命。揭批四人帮，联系揭批林彪，正是清算他们破坏文化大革命的罪行，保卫毛主席亲自发动和领导的无产阶级文化大革命的伟大成果。”</ref><ref>华国锋. 在中国共产党第十一次全国代表大会上的政治报告（一九七七年八月十二日报告，八月十八日通过）：“早在文化大革命初期，“四人帮”就同林彪反党集团相勾结，破坏无产阶级文化大革命。”</ref>

== 审判 ==
{{Main|林彪、江青反革命集团案}}
1978年[[中共十一届三中全会|中共十一届三中全会]]之后，[[邓小平|邓小平]]逐渐成为中共实际的[[中华人民共和国最高领导人|最高领导人]]。在他的倡导下，中华人民共和国政府开始对四人帮和林彪等两个集团在[[法律|法律]]上进行审判和定罪。值得注意的是，中共有意把张春桥的政敌陈伯达划归四人帮集团，以避免林彪问题的复杂化。

1980年9月29日，[[第五届全国人民代表大会常务委员会|第五届全国人民代表大会常务委员会]]第十六次会议通过《[[s:关于成立最高人民检察院特别检察厅和最高人民法院特别法庭检察、审判林彪、江青反革命集团案主犯的决定|关于成立最高人民检察院特别检察厅和最高人民法院特别法庭检察、审判林彪、江青反革命集团案主犯的决定]]》，任命[[中华人民共和国最高人民法院|最高人民法院]]院长[[江华_(政治人物)|江华]]兼特别法庭庭长。

1980年11月20日下午3时第一次开庭，对江青等人进行公开审判。1981年1月25日上午宣判，认定他们均为[[林彪、江青反革命集团案|林彪、江青反革命集团]]成员，并判处了相应的[[刑罚|刑罚]]：

* [[江青|江青]]和[[張春橋|張春橋]]被判处[[死刑缓期执行|死刑，緩刑兩年執行]]，[[剥夺政治权利|剥夺政治权利]]終身。
* [[王洪文|王洪文]]被判处[[无期徒刑|无期徒刑]]，剥夺政治权利终身。
* [[姚文元|姚文元]]被判处[[有期徒刑|有期徒刑]]二十年，剥夺政治权利五年。
* 其他：[[陳伯達|陳伯達]]、[[毛远新|毛远新]]、[[黃永勝|黃永勝]]、[[吳法憲|吳法憲]]、[[李作鵬|李作鵬]]、[[邱會作|邱會作]]和[[江腾蛟|江騰蛟]]，分别被判处有期徒刑十六至十八年、剥夺政治权利五年。

在法庭上，江青是唯一为自己辩护的四人帮成员，她的辩护理由是她不过是在执行[[毛泽东|毛泽东]]的命令，而且她所做的一切都是为了保卫毛泽东。

1981年6月27日，[[中国共产党第十一届中央委员会#六中全会|中共十一届六中全会]]通过《[[关于建国以来党的若干历史问题的决议|关于建国以来党的若干历史问题的决议]]》，将文革定性为“由领导者（毛泽东）错误发动，被[[江青|江青]]（四人帮）和[[林彪|林彪]]两个[[反革命|反革命集团]]利用，给党、国家和各族人民带来严重灾难的[[内乱|内乱]]。”

== 评价 ==
“四人帮”被打倒之后，中国政府揭露四人帮在文革期间互相勾结，逆行倒施，并对老同志进行无情迫害，对人民、对国民经济犯下了滔滔罪行。其中用[[漫画|漫画]]形式来形容四人帮，如“政治流氓”王洪文、“狗头军师”张春桥、“白骨精”江青、“吹鼓手”“文痞”姚文元等。

[[郭沫若|郭沫若]]所写的一首《[[水调歌头|水调歌头]]·粉碎四人幫》上半阕称“大快人心事，揪出四人帮。政治流氓文痞，狗头军师张。还有精生白骨，自比[[武则天|则天武后]]，铁帚扫而光。篡党夺权者，一枕梦黄粱。”

== 成员结局 ==
* [[江青|江青]]于1991年在[[秦城監獄|秦城監獄]]服大量[[安眠药|安眠藥]][[自殺|自殺]]身亡，享年77岁。
* [[王洪文|王洪文]]于1992年因[[肝癌|肝癌]]死于[[北京市|北京]][[首都医科大学附属复兴医院|复兴医院]]，享年56岁。
* [[张春桥|张春桥]]于2005年因[[胰腺癌|胰腺癌]]死于北京复兴医院，享年88岁。
* [[姚文元|姚文元]]于2005年因[[糖尿病|糖尿病]]死于[[上海市|上海]]，享年74岁。
