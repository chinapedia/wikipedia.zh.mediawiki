'''四面神'''或'''四面佛'''，華語則音'''帕剖樓麼'''（{{lang-th|พระพรหม}}，{{lang-en|Phra Phrom}}，源自{{lang-sa|परब्रह्म}}，Para Brahma），因為屬天神級，所以「四面佛」是其誤稱，源自[[印度教|印度教]]、[[婆羅門教|婆羅門教]]神話，為三主神之一的'''[[梵天|梵天]]'''（ब्रह्मा，Brahmā），所以亦稱「[[梵天_(佛教)|大梵天王]]」，乃創造宇宙之神，梵文字母（婆罗米系字母）的创制者。在[[佛教|佛教]]经典中，则是请[[释迦牟尼|释迦牟尼]]住世说法的[[护法神|护法天神]]<ref>{{cite book|title=《增一阿含經》|chapter=〈勸請品〉|volume=卷10|quote=「一時，佛在摩竭國道場樹下。爾時，世尊得道未久，便生是念：『我今甚深之法難曉難了，難可覺知，不可思惟，休息微妙，智者所覺知，能分別義理，習之不厭，即得歡喜。設吾與人說妙法者，人不信受，亦不奉行者，唐有其勞，則有所損。我今宜可默然，何須說法！』　爾時，梵天在梵天上，遙知如來所念，猶如士夫屈伸臂頃，從梵天上沒不現，來至世尊所，頭面禮足，在一面住。爾時，梵天白世尊曰：『此閻浮提必當壞敗，三界喪目。如來、至真、等正覺出現於世，應演法寶，然今復不暢演法味，唯願如來普為眾生廣說深法！』」|url=http://tripitaka.cbeta.org/T02n0125_010|translator=東晉罽賓三藏瞿曇僧伽提婆|publisher=中華電子佛典協會}}</ref>。在[[泰國|泰國]]信仰中，祂是[[財富|財富]]之神，也會保護信徒的平安。受泰國影響，在[[香港|香港]]、[[澳門|澳門]]及[[台灣|台灣]]，也有奉祠四面神的風氣。

== 概要 ==
梵天源自[[古印度|古印度]]，佛教称为[[梵天_(佛教)|大梵天王]]，居[[色界|色界]]初禅天，由於外形近似[[佛像|佛像]]，因此在中文多誤稱為四面佛。在[[東南亞|東南亞]]，尤其是[[泰國|泰國]]，梵天被視為[[佛教|佛教]]的[[護法神|護法神]]，法力無邊，掌管人間榮華富貴。其四面分別朝向東南西北，供信眾祈福。

泰人崇拜和向往梵天的观念由来已久。[[泰语|泰语]]phrom likhid指天数，借自梵语，直译是梵天刻痕。[[暹羅人|暹羅人]]自古认为，人出生后第六天梵天刻命数在其额上。<ref>[https://fo.ifeng.com/a/20150914/41474225_0.shtml 凤凰佛教综合 走近泰国四面佛(作者：吴圣扬)]</ref>在泰民族传统文化——佛教文化中也有梵天崇拜的根基。印度梵天信仰曾在[[印度支那半岛|印度支那半岛]]流传并对当地文化产生深刻的影响，但后来消退了，700多年来，泰人更热衷于崇拜印度教中的另一尊神祇[[因陀罗|因陀罗]]。

泰國四面神崇拜的主要案例為曼谷市中心君悅酒店门前的[[伊拉旺神壇|四面神壇]]，這是起源於近代的信仰據點。

== 造型 ==
泰国各地四面神的造型不固定。从脸和手的数量来看，多数为四面八手。3—6世纪时，印度梵天的造型已经出现四面四手了，但[[印度支那|印度支那]]的古文物显示，直至6—11世纪的[[陀罗钵地|陀罗钵地]]时期，当地人信仰的梵天仍是一面二手，12—13世纪时才多为四面四手。<ref>[https://fo.ifeng.com/a/20150914/41474225_0.shtml 凤凰佛教综合 走近泰国四面佛(作者：吴圣扬)]</ref>

所以现在泰国的梵天神像除了四面八手的外，还能见到四面四手和一面二手的造型。偶尔也能见到五面八手的，这是采用印度传说中五面梵天的造型，其中的四面朝向四个方向，第五面仰天朝上还有一头。

梵天創造了一個女神，名喚'''[[辯才天女|莎塔如帕]]'''（Shatarupa）后，卻開始迷戀這位擁有無雙美貌的女神，因莎塔如帕觉得不妥，于是经常转到梵天的左面、右面、后面甚至飞到上面，以避開梵天的目光。为了方便凝視莎塔如帕，原本一张脸的梵天遂长出五张脸来。<ref>唐卡中的梵天頭及白描圖[http://www.52tangka.net/html/201203311025.html]</ref>

后来梵天冒犯了印度教的另一大主神——毁灭神[[湿婆|湿婆]]，事起因於[[毗濕奴|毗濕奴]]問梵天誰是創造宇宙的至高無上創造者。梵天答：「他即創造宇宙的至高無上創造者，因為他是創造宇宙的至高無上創造者，所以毗濕奴也該崇拜他」。

梵天的話令濕婆大怒，因為濕婆認為祂才是創造宇宙的至高無上的創造者，憤怒的濕婆化身成為Bhairava（[[陪臚|陪臚]]-意為恐怖的殺戮者）。在盛怒之下，湿婆用额头上的第三只眼放射出来的火焰，燒向了梵天朝上方看的那一面，並用左姆指的指甲割下了那顆[[首級|首級]]，于是梵天只剩下了現在所常見的四张脸。雖然發生了梵天跟濕婆不和的事，但最終梵天跟濕婆還是和好如初。<ref>中、印、泰三國對四面神的理解{{cite web |url=http://www.zhfgwh.com/plus/view.php?aid=40652 |title=存档副本 |accessdate=2013-06-26 |deadurl=yes |archiveurl=https://web.archive.org/web/20131016000019/http://www.zhfgwh.com/plus/view.php?aid=40652 |archivedate=2013-10-16 }}</ref>

至於四面神手中的圣物，有这几种物件。各手分执不同的圣物，如令旗、经书、海螺、法轮、权杖、水壶、念珠等等。坐姿则基本半跏趺坐姿即单腿盘坐，通常盘右腿，偶尔也会看到盘左腿的。

四面佛手执法器与手印都有其意义：
#'''令旗'''：代表万能法力
#'''经书'''：代表智慧
#'''海螺'''：代表赐福
#'''法轮'''：代表消灾、降魔、灭烦恼
#'''权杖'''：代表至上成就
#'''水壶'''：代表解渴；有求必应
#'''念珠'''：代表轮回
#'''手印'''：代表保佑、保护

== 敬拜方法 ==
四面神掌管人間的所有事情，故此其四面所求都不相同。四面分別代表事業、感情、財富及健康。正面求事業、生意，左面求姻緣，右面求健康，後面求財。另一方面，四面神四面亦代表慈、悲、喜、捨。凡是信眾祈求升天必須勤修這四種功德。

据说四面神喜欢的花有[[萬壽菊|萬壽菊]]、[[蘭花|蘭花]]、[[玫瑰|玫瑰]]，故泰國通常用萬壽菊花、蘭花或玫瑰（愛情）去敬拜，分别以顺时针次序祭拜该神四面。還願方式可承諾以七色花、還願舞、還願舞人偶、木雕大象（依願望大小）等答謝四面神，而信众许下的[[还神|还神]]的诺言，一定要清楚仔细，言出必行，意志就一定要坚定。
* 正面祈求（功名：學業、事業、仕途等）
* 第二面祈求（感情：愛情、人緣、親情等）
* 第三面祈求（財富：正財（升職）、橫財（投資、賭博）等）
* 第四面祈求（健康：疾病、安全、親友健康等）

拜四面神要點︰
* 因四面神是慈祥的，所以許下的願望必須是以不損害他人利益為要點，靈驗與否除要信任及誠心外，更要根據本人的因果福報，而不能無中生有。超出個人福報範圍的願望是很難如願的
* 所以除誠心拜祭之外，也必須對任何人抱持愛心，持續多做善事，救急扶危，孝順父母，才得以最終善有福報
* 許下願望後，必須作出具體行動才會成功，坐著等是不會如願的
* 一般拜祭時必須以順時針方向，反時針方向只適合個別偏門行業的人士，一般人應盡量避免反時針方向

== 艾拉彎神壇 ==
[[File:Erawan_statue_restored.jpg|thumb]]
{{main|伊拉旺神壇}}
伊拉旺神壇是位于[[泰国|泰国]][[曼谷|曼谷]][[君悦酒店|君悦酒店]]旁的一座四面神坛，是泰国香火最鼎盛的膜拜据点之一，也是曼谷著名的旅游观光胜地，每日都有很多来自世界各地的信众前往参拜或祈祷。

這座四面神坛兴建于1956年。当年据说是在伊拉旺酒店（君悦酒店前身）兴建之时，发生了一连串的不幸工安意外事故。业主请来星象家察看並依其建议，后来便在建筑工地附近建造这座四面神坛供奉四面神，保佑大众诸事顺逐。酒店後來拆掉重建，四面神壇一直保留至今。

== 聖誕 ==
四面神的生日，在泰國、東南亞、中國大陸和港澳等地，定為國曆11月9日。而台灣廟宇或佛壇的四面佛聖誕日，許多處則以農曆11月9日來推算國曆聖誕日，因此，每年的四面神聖誕日期在各地都有所不同。

例如：台灣長春四面佛，自1983年對外開放參拜以來，一直以農曆11月9日為四面佛聖誕日，2013年長春四面佛聖誕日為國曆12月11日（三），而非國曆11月9日。建議信眾前往祝壽祭拜前，可向各地四面佛廟宇或佛壇先行詢問，其四面佛聖誕日是依照國曆或農曆而定。但依據正統四面佛誕辰為國曆11月9日。
