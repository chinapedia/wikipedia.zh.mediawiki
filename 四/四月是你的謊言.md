== 故事簡介 ==
[[File:The_scene_in_the_OP1_of_"Your_Lie_in_April".jpg|thumb]][[新座市|新座市]]]]
男主角有馬公生自小接受母親嚴格的鋼琴訓練，以按照樂譜精準彈奏的能力在各項音樂比賽中獲勝，被人稱作「神童」（動畫作品內，亦被人嘲諷為「媽媽的傀儡」、「人體節拍器」、「演奏機」、「鋼琴比賽機器」）。然而11歲時母親病逝，加上自身對彈奏的迷惘，陷入聽不見鋼琴聲音的心境障礙，選擇放棄鋼琴彈奏。

國中三年級時，某一天在青梅竹馬椿的介紹下，認識同年級的小提琴手——宮園薰，並於一場合奏的音樂比賽中被薰自由奔放的演奏風格所吸引，自此心境與日常生活開始轉變。

== 登場人物 ==
=== 主要人物 ===
; {{nihongo|有馬公生|有馬 公生（ありま こうせい）|}}
: 聲：[[花江夏樹|花江夏樹]]<ref>{{cite news|script-title=ja:「四月は君の嘘」オールナイト一挙上映会に花江夏樹がゲスト出演|url=http://www.ota-suke.jp/news/148170|publisher=おた☆スケ |date=2015-08-11|accessdate=2015-09-02}}</ref>（日本）：[[黃積權|黃積權]]，[[謝潔貞|謝潔貞]]（童年）（香港）／鋼琴演奏：阪田知樹
:生於3月28日，14歲，市立墨谷中學三年級生。喜歡雞蛋三明治和momo牌牛奶。自幼與鄰家的椿認識，與亮太三人是青梅竹馬的摯友，一起上學玩耍。
:母親早希畢業於音大鋼琴系，公生自出生起便常聽著母親彈琴，《愛的悲傷》更是他喜歡的安眠曲。4歲時偶然被紘子發現有彈琴的天賦，便慫恿早希教授公生鋼琴，起初只是隨性玩樂的性質。5歲首次參加區內鋼琴教室的表演會，天真純樸而歡樂的琴音同時感動兩位與他同齡的觀眾:宮園薰和鄰座的井川繪見。
:隨著公生的鋼琴天賦逐漸顯露，早希卻患上重病，需要不時進出醫院、坐著輪椅和戴著呼吸器。似乎感到時間緊迫，早希改以極為嚴厲的方式教授公生鋼琴，要他絲毫不差的跟隨樂譜指示，經常打罵外，也限制公生與朋友玩耍，練琴以外，便是讀琴譜。公生雖與繪見及武士常於比賽碰面及坐在一起，卻完全不認識他們，故他的童年殊不快樂。公生的鋼琴技巧雖然高超精準，在以技巧為先的比賽屢獲殊榮，但缺乏生氣和感情，被稱為｢人形節拍器｣。而公生認為如果努力練習及經常獲勝，也許可以令母親心情好些和早點康復，故一直忍耐著母親所做的一切，同時逐漸把自己的感情掩藏。
:11歲時參加一場比賽雖然獲勝，卻因些微差錯被母親當場訓斥，公生忍受不了，衝動下說了一句晦氣話，母親卻不幸當晚去世，使公生心裏一直充滿歉疚，以為母親之死是因他的一句無心失言，而懲罰的代價是奪走他聽到琴音的聽覺，當他彈琴途中時便會瞬間迷失，因為彈不了鋼琴，他從比賽中隱退。公生認為沒有音樂的日常生活猶像鋼琴黑白鍵般單調，他羨慕及奇怪周遭的人何以會閃耀著不同的色彩，而他也打算鬱悶地渡過餘生。
:四月某星期六下午被椿拉去湊合給亮太介紹｢喜歡亮太｣的女同學的約會，偶然拍到被強風吹起裙子、正與小朋友一起吹奏口風琴的宮園薰，更被誤會是偷拍狂而被狠揍了一頓。椿介紹公生給薰時戲稱他是友人A ，後卻成了公生不知是薰給他的親密稱呼。
:於薰的首次小提琴比賽中驚豔薰傾情而非按琴譜的刻板演奏，看到了豐富多姿的色彩而羨慕不已。從薰的音樂中感覺薰似乎想給他傳達一些埋藏已久且微妙的感情，但作為｢喜歡亮太的女孩子｣的亮太摯友及友人A，他以為這是他的錯覺，且認為不應該超越這個角色設定:｢我只是配角，僅此而已。｣然而對薰有點傾慕的印象已經深刻在他心裡。
:與薰的開朗樂觀不同，公生一直予人沈默寡言、冷淡刻板的印象。他不擅長應付鬼靈精般的薰，常給薰的奇思異想、變化多端的行為弄得團團轉。薰猶如小貓般捉摸不定，在他沒有心理準備下，時而親近，時而遠離，令他乍驚乍喜，忐忑不安，又常為了小事吵嘴，但公生並不以為忤，反感到樂在其中，若可以和薰說上話會感到安心，說不出口便感到失落。他以為早已沒有的喜怒哀樂的感情也慢慢地起著變化，在與薰一起時變得活躍起來，較以往更為坦率，並逐漸發覺他周遭的一切， 無論所見所聽、所思所感，均開始閃耀著不同的色彩。當時，他並不知道這是戀愛的感覺。
:因為一直以為薰喜歡著亮太，而自己只是薰找不著亮太時的代替品，或僅僅作為她伴奏的配角，似在近而實在遠的感覺，常使他感到不是味兒。但因可常與薰一起，看到她天使般的笑臉、聽到她輕快的笑聲、一起練習的時光、在回家的路上一同仰首觀看繁星滿空、輕唱著小星星，感到他們已經相連。他看過她的發怒、她的任性、她的悲傷落淚、她為鼓勵他所說的每一句說話，這些都是他永不能忘懷的景象。他以為互相喜歡的人之間只有歡樂，卻不知道真心坦率的感情只會向喜歡的人呈露。當他越思念這些景象時，他的內心便越糾結和哀傷。他對自己說不能超越友人A的這個角色，不能背叛摯友，但他早已在每報的音樂比賽中藉著琴音向薰訴說了他的真心: ｢即使一個人也好，只要有妳便已經足夠｣、｢我的心只有妳｣，也曾向相座凪承認喜歡上｢喜歡著摯友的女孩子｣。他每看到薰與亮太談笑自若，他便越不想插上話，進而不想與椿、亮太同去醫院探望。若不是薰要求他定期探望，他更沒有勇氣和獨自到醫院探望的動機。亮太對公生說薰只會向公生任性撒嬌，需要依賴時總會第一個想到公生，薰對公生其實有著不是普通朋友的感情。惟正當他準備向薰坦承喜歡她時，才發覺薰的病情已在垂危的邊緣。
:和薰相識後，公生因緣再次踏上參與音樂比賽的道路。在薰的第二次小提琴選拔賽時，他看到薰那堅定的背影和對音樂的熱誠是當時的他所沒有的，因此薰既成了他追趕的目標，也成了他在痛苦和躊躇時強撐下去的支持，他希望那天可以與薰並肩一起在音樂的道路上前行。薰認為公生的鋼琴並非缺乏技巧，而是欠缺了對樂曲的理解和想像力，重要是欠缺為何和為誰而彈的動機，這恰好是過往母親給他訓練時偏重於精確技巧導致公生音樂不完全的因素。每報音樂比賽時由起首的精準，中途如哭鬧的孩童叩擊，中斷後想起能重新擁抱鋼琴，全因為薰一直以來的鼓勵，變化後的琴音因此充滿對薰的回憶、感謝和對她的真意。GALA CONCERT中途因回想起母親的溫柔和慈愛，一切的感情盡已化作埋藏在心內的音符，母親其實一直與他一起，不再是他以為是歉疚的存在，使他可以與昨天告別。
:薰用她生命的餘暉仿若黑夜裡天上閃爍著的星星照耀著公生，｢有我在喔!｣使公生雖身心在黑暗的深淵裡仍可看到頭頂上的光明而不再感到孤獨，而當薰需要公生的支持時，也是他的這句說話給她奮力掙扎的勇氣。薰雖然幫助了公生，但因公生伴隨著她在音樂的道路上奔跑，同時看到公生拚盡全力練習和演奏的背影，也可說是公生給病重中的薰奮力生存下去的希望。當薰告知公生她的病情不樂觀，希望公生可以像按 RESET鍵一樣把她忘掉，而公生則請求相座凪在胡桃丘學園祭和他合奏，請求亮太將演奏現場轉播給薰，要薰不要忘記兩人曾一起在舞台上看到的風景，他會等她康復，再次與她一起合奏。公生也逐漸意識到薰只是一個普通的女孩，很需要他的支持和守護。
:盼望與公生認識，將自幼埋藏已久的心意傳達給公生，是薰自染病後堅持生存下去的動力。為了公生可以重返演奏家的舞台，她竭盡心力，和他練習，給他鼓勵提點，表面上她是表演的主角，從另一面看，她卻甘願充當他的配角，要在演奏中把公生的琴音凸顯出來。她要觀眾不是單獨的記起她，而是要永遠的記著他們，他們是一起的。 
:｢謝謝你!｣
:是曾經在痛苦深淵裡掙扎的兩人互相向施予援手而又無法忘懷的對方言說的一句已概括千言萬語的心裡話。

; {{nihongo|宮園薰|宮園 かをり（みやぞの かをり）|}}
: 聲：[[種田梨沙|種田梨沙]]<ref>{{cite news|script-title=ja:「四月は君の嘘」種田梨沙『台本を見ないくらい集中している』|url=http://news.walkerplus.com/article/51524/|publisher=[[ニュースウォーカー|ニュースウォーカー]]|date=2014-10-22|accessdate=2015-09-02}}</ref>（日本）：[[凌晞|凌晞]]（香港）／小提琴演奏：篠原悠那

:生於7月4日，14歲，市立墨谷中學的三年級生，與椿同班，與公生及亮太同級。小提琴手，家裡經營名為｢我的女兒｣的法式糕餅店，喜歡吃[[卡納蕾|卡納蕾]](可麗露)及閱讀花生漫畫，常引用漫畫中的句語表達她的想法。
