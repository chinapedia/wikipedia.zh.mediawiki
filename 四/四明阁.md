'''四明阁'''位于[[中国|中国]][[浙江省|浙江省]][[宁波|宁波]][[余姚市|余姚市]][[阳明街道|阳明街道]][[龙泉山_(余姚)|龙泉山]]西山脚处，为清代道光九年（1829）探花[[朱蘭_(清朝)|朱兰]]的藏书楼<ref>清·周炳麟修，邵友濂、孙德祖等纂，《光绪余姚县志·卷十四·古迹》：四明阁在驿下冬官里，朱阁学兰督学安徽，使竣归里。临江筑此，每集耆旧谈宴其中，一时称为香山洛下之比。山色江光，风帆沙鸟，遥瞻俯瞩，灵襟洒然，洵登临胜境也。</ref>，原名环翠楼，约建于同治六年（1867）前后，原在酱园街节制闸西首，朱兰故居“探花墙门”东约20米，坐南朝北、北临姚江，2001年迁至现址，现用作非遗展演馆<ref>{{cite web |url=http://daily.cnnb.com.cn/nbrb/html/2012-04/29/content_462537.htm|title=《余姚四明阁天天做戏文》|accessdate=2015-08-14|work=陈青|publisher=宁波日报}}</ref>。建筑现坐北朝南，面阔三间两层，进深八檩，歇山顶，楼梯设于西次间靠北首朝西<ref>{{cite web |url=http://www.nbwb.net/pd_wwbh/info.aspx?Id=1035&type=2|title=《四明阁》|accessdate=2015-08-14|work=|publisher=宁波文化遗产保护网}}</ref>。阁内藏书1949年后尚余数万册，大多由朱氏后人捐与[[浙江图书馆|浙江图书馆]]<ref>{{cite web |url=http://daily.cnnb.com.cn/nbwb/html/2013-06/02/content_607672.htm?div=-1|title=《话说四明阁》|accessdate=2015-08-14|work=朱炯|publisher=宁波晚报}}</ref>。
