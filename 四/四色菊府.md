'''四色菊府'''（{{thai|t=จังหวัดศรีสะเกษ|r=Changwat Si Sa Ket}}，{{IPA-th|t͡ɕāŋ.wàt sǐː sàʔ kèːt}}）是[[泰國|泰國]]東北部之一個府。

== 歷史 ==
該府原名為「區康城」，其後在其北畔建一新城池，稱為「四色菊城」。兩城曾發生地界紛爭，遂於1938年將兩城合二為一，昇格為「四色菊府」。

== 地理 ==
* 四色菊府的面積約為 8,840平方公里。距[[首都|首都]][[曼谷|曼谷]]約571公里。其疆域北接[[黎逸府|黎逸府]]、[[益梭通府|益梭通府]]；南瀕[[柬埔寨|柬埔寨]]；東連[[烏汶府|烏汶府]]；西毗[[素林府|素林府]]。
* 四色菊府的南部多為[[高原|高原]]地帶，山脈成簇狀；西北部的[[沼澤|沼澤]]縱橫交錯，[[地質|地質]]是[[泥|泥]]混[[砂|砂]]。

== 行政 ==

四色菊府行政區共分為22縣；再分成206區；最後再分為2,411條村。

== 名勝古蹟 ==
* 詩那卡隆公園(สวนสมเด็จศรีนครินทร์ Sri Nagarin Park)：位於四色菊府直轄縣，是當年為慶祝暹羅皇太后的壽辰而建。環境優美，面積廣達3.2平方公里，有森林、湖泊，遊客可租用帳蓬及船隻。每年3月至4月份，均在此舉行「喬木花節」， 有各種地方歌舞表演助慶，另有出售多種土特產、紀念品。
* 訕甘烹大石宮(ปราสาทสระกำแพงใหญ่ Sra Kampaeng Yai Khmer Ruins)：位於烏浦奔披柿縣，建於1042年，為高棉族建築藝術風格， 壁畫雕刻精巧。
* 訕甘烹小石宮(ปราสาทสระกำแพงน้อย Sra Kampaeng Noi Khmer Ruins)：位於烏浦奔披柿縣，類似「訕甘烹大石宮」，但規模較小，建於同一時代。
* 甫快石宮 ()：位於平他拉叻縣，採用紅土及山石建成。
* 班固佛塔：位於班固縣，採用成排大磚塊砌成，為高棉族建築藝術風格，已有一千多年歷史。
* 匯珍瀑布：位於坤漢縣，瀑布週團長滿各種樹木，有排列整齊的亭閣，環境優美，適合休閒和野餐。
* 明奴背喃拍明湖泊：位於沛晚縣，風景非常優美，值得一遊。
* 考莫意亮山：位於干他拉叻縣，可欣賞週圍：「考柏威漢石宮」的美景，附近尚可遇到凹凸不平的石雕畫。
* 哇三廊艾寺：位於烏浦奔披柿縣，寺內雕刻美麗。
* 哇芹目拉拍南寺：位於區康縣，寺內供奉一尊大佛像，深受東北部人崇拜。
* 哇鑾素孟卡拉喃寺：位於四色菊府直轄縣，寺內有該府最美麗的藏經閣，為大城時代的建築藝術風格，雕刻精美。
