'''四氯化硅'''是化学式为SiCl<sub>4</sub>的[[无机化合物|无机化合物]]，1823年由[[永斯·贝采利乌斯|永斯·贝采利乌斯]]首次发现。

== 化学 ==

四氯化硅室温下为无色粘稠液体，可通过[[硅|硅]]与[[氯气|氯气]]反应制备：

: Si  +  2 Cl<sub>2</sub> → SiCl<sub>4</sub>

与[[四氯化碳|四氯化碳]]不同，四氯化硅与水迅速反应生成[[二氧化硅|二氧化硅]]和[[氯化氢|氯化氢]]。该水解速率差异归咎于硅原子更大的[[原子半径|原子半径]]、硅原子存在的3d轨道以及Si-Cl键较低的键能。

: SiCl<sub>4</sub> + 2 H<sub>2</sub>O → SiO<sub>2</sub> + 4 HCl

与[[甲醇|甲醇]]或[[乙醇|乙醇]]反应则得到{{link-en|硅酸四甲酯|Tetramethyl orthosilicate}}和[[硅酸四乙酯|硅酸四乙酯]]：

: SiCl<sub>4</sub> + 4 ROH → Si(OR)<sub>4</sub> + 4 HCl

高温下与硅发生[[归中反应|归中反应]]：

: Si + SiCl<sub>4</sub> → Si<sub>2</sub>Cl<sub>6</sub> + [[同系物|同系物]]

== 用途 ==
四氯化硅沸点较低，可用于多次[[分馏|分馏]]生产高纯[[硅|硅]]；也可用于生产高纯的合成[[石英玻璃|石英玻璃]]（熔融石英）。它可被氢气[[还原|还原]]为硅，也可[[水解|水解]]为[[二氧化硅|二氧化硅]]。大量的高纯硅应用在[[半导体|半导体]]工业和[[太阳能电池|光電池]]中。
