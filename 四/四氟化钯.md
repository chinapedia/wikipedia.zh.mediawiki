'''四氟化钯'''是一种[[钯|钯]]和[[氟|氟]]形成的[[无机化合物|无机化合物]]，[[化学式|化学式]]为PdF<sub>4</sub>。其中钯的氧化态为+4，这也是钯已发现的最高[[氧化态|氧化态]]。<ref name="G&E-1152-1153" /><ref name="Housecroft-788">{{Housecroft3rd|pages=788–789}}</ref>

== 结构 ==
该化合物为[[反磁性|反磁性]]物质，其中钯的配位数为8，钯位于两个四面体的中心。

== 制备 ==
[[三氟化钯|三氟化钯]]与氟气在大约7个大气压和300 °C下反应数天可得四氟化钯：<ref name="Rao" />
:<math>\rm \ 2PdF_3 + F_2 \xrightarrow{7atm,300^oC} 2PdF_4</math>

== 性质 ==
四氟化钯是一种强[[氧化剂|氧化剂]]，在潮湿的空气中迅速[[水解|水解]]。<ref name="Rao" />
