'''四氟肼'''（N<sub>2</sub>F<sub>4</sub>）在常温常压下是一种无色的活泼气体。它是[[肼|肼]]的一种含氟衍生物，有剧毒，与有机化合物接触易爆炸。

==制法==

四氟肼可以由[[三氟化氮|三氟化氮]]在铁等金属还原剂存在下制得：

:<math>\rm \ 2NF_3 + Fe \rightarrow N_2F_4 + FeF_2</math>

三氟化氮与[[一氧化氮|一氧化氮]]反应也可制得四氟肼，并产生[[亚硝酰氟|亚硝酰氟]]<ref name="无机化学丛书">{{cite book|title=《无机化学丛书》第四卷：氮、磷、砷分族|pages=P397-398|author=张青莲|isbn=7-03-002238-6|location=北京|publisher=科学出版社}}</ref>：

:<math>\rm \ 2NF_3 + 2NO \xrightarrow{600^oC} N_2F_4 + 2NOF</math>

它在一些物质的化学合成中用作前体或[[催化剂|催化剂]]。

==结构==
四氟肼存在两种构象，能垒约为12.5kJ/mol。

==性质==

四氟肼与二氟化氮自由基处于分子-[[自由基|自由基]]平衡中：

:<math>\rm \ N_2F_4 \rightleftharpoons 2NF_2</math>·

二氟化氮自由基使得四氟肼在高温下略显深蓝色。正因为自由基的存在，使得四氟肼能发生许多相关的反应<ref name="无机化学丛书" />：

*与一氧化氮偶联：
:<math>\rm \ N_2F_4 + 2NO \rightleftharpoons 2NONF_2 \rightleftharpoons 2NO + 2NF_2</math>
*自由基取代反应：
:<math>\rm \ RCHO + N_2F_4 \rightarrow RCONF_2 + HNF_2</math>
:由这类反应可制备含二氟氨基的高能化合物。
*自由基加成反应，二氟氨基加在碳碳双键或三键的两端碳上。

四氟肼与水缓慢反应并产生肼和[[氟化氢|氟化氢]]：

:<math>\rm \ N_2F_4 + 4H_2O \rightarrow N_2H_4 + 4HF + 2O_2</math>

四氟肼可以用作[[火箭燃料|火箭燃料]]中的高能液体[[氧化剂|氧化剂]]。这项应用于1959年时提出。<ref> [http://stinet.dtic.mil/oai/oai?&verb=getRecord&metadataPrefix=html&identifier=AD0313998 Tetrafluorohydrazine] {{webarchive|url=https://web.archive.org/web/20070312051306/http://stinet.dtic.mil/oai/oai?&verb=getRecord&metadataPrefix=html&identifier=AD0313998 |date=2007-03-12 }} at DTIC.mil</ref>

四氟肼也能与路易斯酸反应形成加合物，例如它与[[五氟化锑|五氟化锑]]能形成N<sub>2</sub>F<sub>4</sub>·2SbF<sub>5</sub>和N<sub>2</sub>F<sub>4</sub>·3SbF<sub>5</sub>两种加合物。在低温下后者与[[二氧化硫|二氧化硫]]反应能转换为前者。[[核磁共振|核磁共振]]表明，N<sub>2</sub>F<sub>4</sub>·2SbF<sub>5</sub>实际上是离子化合物N<sub>2</sub>F<sub>3</sub><sup>+</sup>Sb<sub>2</sub>F<sub>11</sub><sup>-</sup><ref name="无机化学丛书" />。
