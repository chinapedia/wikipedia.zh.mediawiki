'''四平路'''是[[中国|中国]][[上海市|上海市]]跨[[虹口区|虹口区]]、[[杨浦区|杨浦区]]的一条交通干道，西南-东北走向，西南接[[吴淞路_(上海)|吴淞路]]，东北至[[江湾五角场|五角场]]。全长4970米，宽34米到37米。

四平路修筑于1930至1932年，因其连接[[闸北区|闸北]]市区与引翔乡，最初名翔闸路。尚未完成时，在1931年以[[辛亥革命|辛亥革命]]时期上海都督[[陈其美|陈其美]]定名为'''其美路'''，成为通往江湾[[新市区|新市区]]的主要干道之一。日本占领时期改名为'''松进通'''、'''维新路'''，1945年复称其美路，1950年以[[吉林省|吉林]][[四平市|四平]]改今名。1983年，为配合[[全运会|全运会]]召开，四平路全线拓宽至35米。<ref>[http://shtong.gov.cn/newsite/node2/node4/node2249/node4418/node20208/node21556/node62883/userobject1ai9088.html 区县志 >> 区志 >> 虹口区志 >> 第十三编房产 >> 第二章旧住房改造 >> 第六节 四平路沿线地区]上海地方志办公室</ref><ref>[http://shtong.gov.cn/newsite/node2/node2245/node68289/node68294/node68312/node68325/userobject1ai65726.html 专业志 >> 上海市政工程志 >> 第一篇市区道路 >> 第二章主要干道 >> 第二节　南北向主要干道] {{webarchive|url=https://web.archive.org/web/20140527225953/http://shtong.gov.cn/newsite/node2/node2245/node68289/node68294/node68312/node68325/userobject1ai65726.html |date=2014-05-27 }}上海市地方志办公室</ref>

日本占领时期，在四平路北端设立恒产株式会社（今[[南京政治学院|南京政治学院]]院办）。抗战结束以后，[[同济大学|同济大学]]从[[四川省|四川]]迁回上海，但没有回到[[吴淞|吴淞]]校址，而是迁入该路一所日本学校，在此基础上发展。

与四平路相交的道路主要有：[[海伦路_(上海)|海伦路]]、[[溧阳路|溧阳路]]、[[临平北路|临平北路]]、[[曲阳路|曲阳路]]、[[大连路_(上海)|大连路]]、[[赤峰路|赤峰路]]、[[彰武路|彰武路]]、[[中山北二路|中山北二路]]、[[国权路|国权路]]、[[国顺路|国顺路]]、[[国定路|国定路]]等。

== 轨道交通 ==

*[[上海轨道交通八号线|上海轨道交通八号线]]在四平路、[[大连路_(上海)|大连路]]口设有[[四平路站|四平路站]]。
*[[上海轨道交通十号线|上海轨道交通十号线]]沿四平路由北向南设有[[五角场站|五角场]]、[[国权路站|国权路]]、[[同济大学站|同济大学]]、[[四平路站|四平路]]、[[邮电新村站|邮电新村]]及[[海伦路站|海伦路]]等站点。
