'''四乙基鉛'''是有機金屬化合物的一種，其結構是一個[[鉛|鉛]]原子與四個乙基分子連在一起。

四乙基鉛是一种石油燃料添加剂; 首先与1920年代开始的汽油（汽油）混合作为专利提高[[辛烷值|辛烷值]]的燃料添加剂，使[[壓縮比|发动机压缩]]得到大幅提高。 这反过来又导致增加车辆性能（燃油经济性）<ref name=BMJ1928>{{cite journal|title=TETRA-ETHYL LEAD AS AN ADDITION TO PETROL|journal=British Medical Journal|volume=1|issue=3504|pages=366–7|date=3 March 1928|pmid=20773729|pmc=2455205|doi=10.1136/bmj.1.3504.366}}</ref><ref>{{citation|title=After Lead?|work=[[Popular_Science|Popular Science]]|page=94|edition=October 1987|url=https://books.google.com/books?id=oAAAAAAAMBAJ&&pg=PA94}}</ref>。四乙基鉛在19世纪中期被化学鉴定，但其抗爆效应是在1921年由通用汽车研究实验室发现的，该实验室花费了数年的时间试图找到既高效又廉价的添加剂。

1970年[[美国环境保护局|美国环境保护署]]根据{{le|美国清洁空气法案|Clean Air Act (United States)}}开始对机动车尾气排放做出规定。1986年美国停止销售含铅汽油。[[中华人民共和国|中华人民共和国]]《国务院办公厅关于限期停止生产销售使用车用含铅汽油的通知》（国办发[1998]129号），于2000年1月1日停产含铅汽油，2000年7月1日全国范围内停止销售使用含铅汽油。[[台灣中油股份有限公司|台灣中油股份有限公司]]自1999年6月1日起開始供應九八無鉛汽油，2018年加油站販售92、95、98三種無鉛汽油。<ref>{{Cite web|title=台灣中油全球資訊網|url=https://www.cpc.com.tw/cp.aspx?n=1343&Create=1|work=2.16.886.101.20003.20007.20123|accessdate=2019-01-27|language=zh-tw|last=台灣中油股份有限公司}}</ref><ref>{{Cite web|title=1999年5月31日，中油公司宣布，98無鉛汽油將自6月1日零時起正式在全省302座加油站上市|url=https://www.facebook.com/ZhangZheSheng/photos/a.10152144402114531/10152372499634531/?type=3|work=www.facebook.com|accessdate=2019-01-27|language=en|author=|date=|publisher=}}</ref>

== 製備 ==
四乙基鉛可由[[氯乙烷|氯乙烷]]與鈉鉛合金反應製取，方程式如下：
Na<sub>4</sub>Pb+4C<sub>2</sub>H<sub>5</sub>Cl→(C<sub>2</sub>H<sub>5</sub>)<sub>4</sub>Pb+4NaCl

== 特性 ==
四乙基鉛為無色、具黏性的液體，因為該化合物不帶電荷，且外圍由四個烷基包圍，使其具有親脂性，可溶於汽油。

值得注意的是，四乙基鉛結構中的四個C-Pb鍵的引力相當弱，內燃機燃燒的溫度可使其分解。分解時，會先轉變為[[三乙基鉛|三乙基鉛]](CH<sub>3</sub>CH<sub>2</sub>)<sub>3</sub>Pb和乙基自由基，這些[[自由基|自由基]]會清除其他自由基。透過自由基反應，從而不致過早開始燃燒，使點燃適當地延遲，預防[[震爆|震爆]]（[[爆震|爆震]]）。

== 用途 ==
四乙基鉛一度廣泛使用作為添加劑在汽油，以提高燃料的辛烷值，以防止發動機內發生震爆，從而能夠使用更高的壓縮比率，藉以提高汽車發動機效率和功率。最初使用四乙基鉛添加劑的美國，與最初使用酒精作添加劑的歐洲比較。含鉛汽油的優點從它的高能量含量和貯藏品質較高表現出來，最終成為了普遍使用的燃料添加劑。其中一個最大的優點，四乙基鉛比其他抗爆震劑或使用高辛烷值的汽油混合劑比例比較，僅需要非常低的濃度，就達到提高燃料的辛烷值。典型的製備方法，是以一分體積的乙基液（內含四乙基鉛）加到1260分體積未經處理的汽油。其他抗爆震劑必須在用較大量的份量和/或比天然汽油的能源值更低。高能源值的含鉛汽油會有更大的燃油效率。 

當酒精用來作為抗爆劑，會造成燃料吸收水分和空氣，高濕度燃油可導致燃料喉管生鏽和腐蝕。而四乙基鉛是較易溶於汽油，而乙醇則難溶於汽油，且溶解度隨燃料濕度增加。隨著時間的推移，水滴和積水的水分可以形成在燃油系統的燃料喉管結冰。此外燃料的高濕度也可以出現生物污染問題，由於某些細菌能夠在水面和汽油的表面繁殖，從而在燃料系統內造成細菌滋生。四乙基鉛的毒性，使其具殺菌特性，有助防止燃油污染和細菌生長而造成燃油降解。

此化合物常用於汽車汽油的添加劑，提高[[辛烷值|辛烷值]]，作為抗震爆之用，從而延長各零件的壽命。其燃燒會產生固體[[一氧化鉛|一氧化鉛]]和鉛。

固體鉛金屬與氧化鉛會在發動機內迅速積聚，損害發動機內各個零件。

:(CH<sub>3</sub>CH<sub>2</sub>)<sub>4</sub>Pb + 13O<sub>2</sub> → 8CO<sub>2</sub> + 10H<sub>2</sub>O + Pb
:2Pb + O<sub>2</sub> → 2PbO

因此會加入[[1,2-二溴乙烷|1,2-二溴乙烷]]或[[1,2-二氯乙烷|1,2-二氯乙烷]]，令鉛反應為可蒸發的[[溴化鉛|溴化鉛]]和[[氯化鉛|氯化鉛]]，但這些物質會造成空氣污染，對兒童腦部構成損害，因此石油公司開始推出[[無鉛汽油|無鉛汽油]]。此外，這種添加劑也會造成催化轉換器內的催化劑受污染，催化劑失效會使汽車的催化轉換器失去其功能。

== 對身體的毒性及中毒途径 ==
{{See also|鉛中毒}}
四乙基鉛為劇毒性物質，其毒性為金屬鉛的100倍，且常用於解救鉛中毒的金屬[[螯合劑|螯合劑]]，如[[依地酸鈣鈉|依地酸鈣鈉]]（乙二胺四乙酸二鈉鈣,[[EDTA|EDTA]] 2Na-Ca）對該化合物中毒無效<ref>《常用藥物手冊》</ref>。皮膚接觸，吸入揮發物皆可中毒。進入人體後，部分會轉變為三乙基鉛，三乙基鉛可穿透[[血腦屏障|血腦屏障]]，傷害[[中樞神經系統|中樞神經系統]]。

四乙基鉛的輕度中毒症狀為易興奮、急躁、易怒、焦慮不安和病症型類神經症，重度中毒症狀為躁動不安、精神錯亂、幻覺、妄想、譫妄、人格改變、甚至暴力行為等精神運動性興奮表現<ref>www.gdpcc.com/standard/standard36.htm</ref>。

== 淘汰和禁止 ==
在大多數工業化國家，由於擔心空氣和土壤鉛含量以及[[鉛|鉛]][[生物累積|累積]]的[[鉛中毒|神經毒性]]，在2000年初完成了逐步淘汰公路車輛燃料。 1975年美國強制使用[[催化转换器|催化转换器]]和更新的車型以滿足更嚴格的排放法規，開始逐步淘汰美國含鉛汽油。 汽車工程和石油化學的若干進展減少了四乙基鉛的需求。 用於製造高辛烷值共混原料如[[催化重整|重整油]]和[[2,2,4-三甲基戊烷|同分異構辛烷]]的更安全方法減少了對四乙基鉛的依賴，與其他各種毒性的抗爆添加劑一樣，包括金屬化合物如甲基環戊二烯基三羰基錳（MMT）以及含氧化合物包括甲基叔，[[甲基叔丁基醚|甲基叔丁基醚]]（MTBE），叔戊基甲基醚（TAME），和{{le|乙基叔丁基醚|Ethyl tert-butyl ether}}（ETBE）。

替代鉛添加劑已經過科學測試，其中一些已於1999年在{{le|英國汽車工業研究協會|MIRA Ltd.}}獲得英國歷史車輛俱樂部聯合會的批准。

{{as of|2016|6}} ，[[联合国环境署|联合国环境署]]贊助的淘汰工作已基本完成：只有[[阿爾及利亞|阿爾及利亞]]，[[也門|也門]]和[[伊拉克|伊拉克]]繼續普遍使用含鉛汽油。 沒有國家專門使用它<ref name=":0">{{Cite web|url=https://wedocs.unep.org/bitstream/handle/20.500.11822/17542/MapWorldLead_March2017.pdf?sequence=1&isAllowed=y|title=UNEP - Transport - Partnership for Clean Fuels and Vehicles|last=|first=|date=|website=Leaded Petrol Phase-out: Global Status as at March 2017|archive-url=https://web.archive.org/web/20171115235249/http://wedocs.unep.org/bitstream/handle/20.500.11822/17542/MapWorldLead_March2017.pdf?sequence=1&isAllowed=y|archive-date=2017-11-15|dead-url=yes|access-date=2018-04-28}}</ref>。
