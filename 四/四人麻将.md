{{vgname|四人麻将|ja=4人打ち麻雀}}是[[Hudson_Soft|Hudson Soft]]开发，[[任天堂|任天堂]]于1984年在日本[[红白机|红白机]]发行的电子[[麻将|麻将]]游戏。游戏是任天堂的第二款麻将游戏。游戏采用[[日本麻将|日本麻将]]规则<ref>{{cite web| url = http://www.retrogamezone.co.uk/4NinUchiMahjong.html| title = Complete set of video game tiles (from 4 Nin Uchi Mahjong)| publisher = Retro Game Zone| date = | accessdate = 2010-05-25| deadurl = yes| archiveurl = https://web.archive.org/web/20100831180840/http://www.retrogamezone.co.uk/4NinUchiMahjong.html| archivedate = 2010-08-31}}</ref>，玩家和三名电脑玩家对战。游戏在日本售出145万<ref>[http://geimin.net/da/db/m_domestic/index.php GEIMIN.NET／国内歴代ミリオン出荷タイトル一覧]</ref>。
