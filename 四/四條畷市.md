'''四條畷市'''（{{jpn|j=四條畷市|hg=しじょうなわてし|rm=Shijōnawate shi|f=y}}）是位於日本[[大阪府|大阪府]]東北部的城市。主要市區位於西側的平原上，但轄區東側三分之二位於{{Tsl|ja|生駒山地}}，東部的田原地區甚至已經位於生駒山地的東麓，也是大阪府內唯一位生駒山地東側的區域，也因此該區域常被誤認為屬[[奈良縣|奈良縣]]。而田原地區的東北側為奈良縣[[生駒市|生駒市]]的北田原町地區，東南側為同屬生駒市的南田原町地區，因此有時以「西田原」稱呼轄內的田原地區，「東田原」稱呼生駒市區。

本地自1932年開始使用「{{lang|ja|四條畷}}」的名稱以來，官方一直市使用「{{lang|ja|四條畷}}」之名稱，但在1946年日本政府發布[[當用漢字|當用漢字]]後，「{{lang|ja|条}}」成為日本主要使用的漢字，因此民間常將本地地名寫作「{{lang|ja|四条畷}}」，但本地政府仍堅持使用「{{lang|ja|條}}」作為正式名稱，因此造成現在會同時出現「{{lang|ja|四條畷}}」和「{{lang|ja|四条畷}}」的兩種用法。<ref>{{Cite web| url=http://www.city.shijonawate.lg.jp/faq/gyoseijoho/shijonawateshinituite/1409643599308.html| title=四條畷市の、「じょう」の漢字について| publisher=四條畷市政府| accessdate=2018-07-20| language=ja}}</ref>

== 歷史 ==

在令治國時代屬於[[河內國|河內國]][[讃良郡|讃良郡]]，6世紀時曾是[[渡來人|渡來人]]居住的地方；[[南北朝时代_(日本)|南北朝时代]]南北朝雙方在此發生[[四條畷之戰|四條畷之戰]]<ref>{{Cite web| url=http://www.city.shijonawate.lg.jp/gyosei/shijonawateshinitsuite/shinogaiyou/1408697371037.html| title=市の概要| publisher=四條畷市政府| accessdate=2018-07-20| language=ja}}</ref>，北朝在此役大勝，也造成南朝的勢力一度衰退。[[戰國時代_(日本)|戰國時代]]{{Tsl|ja|飯盛山城}}也成為多方搶奪的主要據點，在該時期由於[[天主教|天主教]]開始傳入日本，本地曾有大量接受信仰天主教並[[受洗|受洗]]的信徒，在{{Tsl|ja|田原城 (河内国)|田原城 (河內國)|田原城}}城主的[[菩提寺_(日本)|菩提寺]]－千光寺遺跡中，就發掘出1581年的天主教墓碑，這也是日本目前發掘到最古老的天主教墓碑。<ref>{{Cite web| url=http://www.city.shijonawate.lg.jp/kanko_sangyo_bunka/rekisibunnkazai/bunkazai/shiseki_kininbutsu_yukeibunkazai/1409379419741.html| title=8. 有形文化財　千光寺跡出土　田原礼幡キリシタン墓碑| publisher=四條畷市政府| accessdate=2018-07-20| language=ja}}</ref>

[[江戶時代|江戶時代]]為幕府直屬的天領，19世紀[[明治維新|明治維新]]後，曾先後隸屬大坂裁判所（後來的[[大阪府|大阪府]]）、[[河內縣_(日本)|河內縣]]、[[堺縣|堺縣]]，1881年隨著堺縣被併入大阪府後隸屬大阪府至今。

1889年實施[[町村制|町村制]]，現在的四條畷市西半部屬於甲可村，東半部屬於{{Tsl|ja|田原村 (大阪府)|田原村 (大阪府)|田原村}}，甲可村在1932年改名為'''四條畷村'''，1947年升格改制為'''四條畷町'''，1961年田原村併入四條畷町。

1950年代起，此地也由於大阪市的發展，四條畷町的主要市區開始成為大阪的[[通勤城市|通勤城市]]，人口快速成長；1969年，由於日本政府修改{{Tsl|ja|地方自治法}}，人口超過三萬的行政區劃即有資格升格為市，於是當年人口僅約3萬5千人的四條畷町即在1970年升格為'''四條畷市'''。<ref>{{Cite web| url=http://www.city.shijonawate.lg.jp/gyosei/shijonawateshinitsuite/shinogaiyou/rekishi_minwa/rekishi/1411954801866.html| title=市制施行(昭和45年から昭和63年）| publisher=四條畷市政府| accessdate=2018-07-20| language=ja}}</ref>

1980年代，東部的{{Tsl|ja|田原台}}也成為{{Tsl|ja|新興住宅地}}，隨即也被劃入{{Tsl|ja|関西文化学術研究都市|關西文化學術研究都市}}的範圍。

== 交通 ==

[[西日本旅客鐵道|西日本旅客鐵道]][[片町線|片町線]]以南北向穿過四條畷市西部的主要市區，轄內僅設有{{Tsl|ja|忍ケ丘駅|忍丘車站}}一個車站，而以「四條畷」為名的{{Tsl|ja|四条畷駅|四條畷車站}}則位於四條畷市南側市界一百公尺外的[[大東市|大東市]]境內，但一般仍是把位於大東市內的四條畷車站視為四條畷市的主要車站。

而以{{Tsl|ja|生駒山地}}將四條畷市區分為西部的主要市區與東部的田原地區，兩地之間有[[國道163號|國道163號]]{{Tsl|ja|学研都市連絡道路|學研都市連絡道路}}相接，兩地之間的客運主要由市政府委託{{Tsl|ja|京阪バス|京阪巴士}}經營的{{Tsl|ja|四條畷市コミュニティバス|四條畷市社區巴士}}所營運；但由於田原地區並無鐵路經過，當地居民如要搭乘鐵路大多是搭乘{{Tsl|ja|奈良交通}}的客運前往位於[[生駒市|生駒市]]的[[近畿日本鐵道|近畿日本鐵道]][[奈良線_(近畿日本鐵道)|奈良線]][[生駒車站|生駒車站]]。

== 觀光資源 ==

在轄區中央的生駒山地上，有由[[大阪府|大阪府]]政府設立的大阪府民之森室池園地（{{lang|ja|大阪府民の森むろいけ園地}}），為大阪府所設立的眾多{{Tsl|ja|県民の森|縣民之森|府民之森}}之一，占地49[[公頃|公頃]]，為一結合自然山林環境所設立的休閒設施。

== 教育 ==

轄內唯一的高等教育學校是{{Tsl|ja|大阪電気通信大学|大阪電氣通信大學}}的四條畷校區，該校區位於四條畷市市區東側的生駒山地山區，設有醫療福祉工學院、綜合情報學院。
