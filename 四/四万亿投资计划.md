'''四万亿投资计划'''<ref>[http://news.sina.com.cn/c/2011-01-05/015621758939.shtml 我国四万亿投资计划六成以上项目完工]</ref><ref>[http://finance.qq.com/a/20090113/000292.htm “四万亿”投资计划：各地投资基本到位]</ref>，简称'''四万亿'''，是[[温家宝|温家宝]]力主推动的一项经济政策。2008年11月[[中华人民共和国国务院|中华人民共和国国务院]][[国务院常务会议|常务会议]]提出用于应对[[2007-2008年环球金融危机|2008国际经济危机]]、稳定经济的一系列财政，货币等政策（大体分为十个大类），总规模约4万亿元[[人民币|人民币]]，约合5860亿[[美元|美元]]。<ref>[http://finance.sina.com.cn/g/20081109/19215484791.shtml 国务院出台扩大内需十措施确定4万亿元投资计划]</ref>

== 背景  ==
2008年下半年受金融危机影响，中国出口急转直下，从年初的超过两位数增长迅速回落至负增长。而2009年一季度，出口转为两位数负增长，且工业生产大幅下滑，发电量零增长，大量中小出口企业关闭，沿海地区失业潮出现。  <ref>[http://www.ftchinese.com/story/001052046 反思“四万亿”恐惧症]沈建光</ref>

== 制定与颁布 ==
2008年11月5日，时任中国[[国务院总理|国务院总理]]的[[温家宝|温家宝]]主持召开的[[国务院|国务院]][[国务院常务会议|常务会议]]正式作出此决策，并在11月9日星期日晚间对外公布<ref>{{cite web|accessdate=2014-05-30|url=http://www.ce.cn/xwzx/gnsz/gdxw/200811/10/t20081110_17329848.shtml|title=各界专家热议中国政府扩大内需十项措施|publisher=[[中国经济网|中国经济网]]}}</ref>。

当时外界普遍将之视为“中国政府迄今为应对全球[[2007年－2012年环球金融危机|经济危机]]所作出的最大动作”。[[国际货币基金组织|国际货币基金组织]]总裁[[多米尼克·斯特劳斯-卡恩|多米尼克·斯特劳斯-卡恩]]称中国的这个措施是一个“好消息”。他说：“这是一个规模非常大的措施。它的影响不仅仅是为[[世界经济|世界经济]]提供需求，而且对中国经济本身会有巨大影响。我认为这是一个纠正不平衡的好消息。”<ref>[http://www.reuters.com/article/gc04/idUSTRE4A94P220081110 路透社：国际货币基金组织认为中国帮助刺激全球经济]</ref>  

=== 官方政策清单 ===
十项措施包括：<ref>{{cite web | language =zh-cn | publisher=新华网 | title=温家宝主持国务院常务会 确定扩大内需十项措施 | url=http://news.xinhuanet.com/newscenter/2008-11/09/content_10331258_1.htm | author= | date=2008年11月9日 | accessdate=2008-11-14 }}</ref>
# 加快建设保障性安居工程。加大对[[廉租房|廉租房]]建设支持力度，加快棚户区改造，实施游牧民定居工程，扩大农村危房改造试点。
# 加快农村基础设施建设。加大农村[[沼气|沼气]]、[[饮水安全|饮水安全]]工程和农村公路建设力度，完善农村电网，加快[[南水北调|南水北调]]等重大水利工程建设和病险水库除险加固，加强大型灌区节水改造。加大扶贫开发力度。
# 加快铁路、公路和机场等重大基础设施建设。重点建设一批客运专线、煤运通道项目和西部干线铁路，完善[[高速公路|高速公路]]网，安排中西部干线机场和支线机场建设，加快城市电网改造。
# 加快[[医疗|医疗]]卫生、文化教育事业发展。加强基层医疗卫生服务体系建设，加快中西部农村初中校舍改造，推进中西部地区特殊教育学校和乡镇综合文化站建设。
# 加强生态环境建设。加快城镇污水、垃圾处理设施建设和重点流域水污染防治，加强重点防护林和天然林资源保护工程建设，支持重点节能减排工程建设。
# 加快自主创新和结构调整。支持高技术产业化建设和产业技术进步，支持服务业发展。
# 加快地震灾区<ref>此处所指地震灾区主要为[[汶川大地震|汶川大地震]]受灾地区。</ref>灾后重建各项工作。
# 提高城乡居民收入。提高明年粮食最低收购价格，提高农资综合直补、良种补贴、农机具补贴等标准，增加农民收入。提高低收入群体等社保对象待遇水平，增加城市和农村低保补助，继续提高企业退休人员基本养老金水平和优抚对象生活补助标准。
# 在全国所有地区、所有行业全面实施增值税转型改革，鼓励企业技术改造，减轻企业负担1200亿元。
# 加大金融对经济增长的支持力度。取消对商业银行的[[信贷|信贷]]规模限制，合理扩大信贷规模，加大对重点工程、“三农”、中小企业和技术改造、兼并重组的信贷支持，有针对性地培育和巩固消费信贷增长点。

== 效果 ==
2009年扩大内需十项措施出台后，很快便引发了[[吴敬琏|吴敬琏]]、[[厉以宁|厉以宁]]、[[许小年|许小年]]、[[张维迎|张维迎]]等经济学家的激烈批评<ref>[http://m.ftchinese.com/story/001052046 反思“四万亿”恐惧症]。</ref>，[[林毅夫|林毅夫]]则是少数对该计划公开表示支持的经济学家。<ref>[http://www.ftchinese.com/story/001051451 刘胜军，“林毅夫命题”关乎中国经济转型，FT中文网，2013-07-17]</ref>

[[华尔街日报|华尔街日报]]刊登Diana Choyleva文章， 认为此方案带来的过量货币供给作为原因之一导致了2010-2011年消费品价格、房价上涨等[[通胀|通胀]]现象<ref>[http://cn.wsj.com/gb/20110422/opn072504.asp  观点：中国高通胀的真正原因-华尔街日报] {{Wayback|url=http://cn.wsj.com/gb/20110422/opn072504.asp |date=20110425230330 }}</ref>。也有人认为此刺激计划带来了巨大的地方债务<ref>[http://money.163.com/11/0628/03/77JRV9VH00253B0H.html  地方债还贷高峰到来 贷款逾期将大量出现]</ref><ref>[http://cn.wsj.com/gb/20110628/bch100803.asp   中国地方债究竟有多少？] {{Wayback|url=http://cn.wsj.com/gb/20110628/bch100803.asp |date=20110701091754 }}</ref>

扩大内需十项措施的资金实际上进入了房地产行业、地方融资平台及[[产能过剩|产能过剩]]行业，还有部分资金被用于还本付息，这些资金在中国的金融体系内部空转，没有进入到实体经济之中。部分与政府有关的机构如地方政府及国有企业获得资金相对容易，资金流入到这些机构的低效高风险项目之中。<ref>{{cite web | url=http://finance.qq.com/zt2013/cjgc/qh.htm?pgv_ref=aio2012&ptlang=2052 | title=不缺钱的钱荒：钱去哪里了？ | publisher=腾讯 | accessdate=2013-06-24}}</ref>

以炼钢行业为例，2011年以后，扩大内需十项措施的负面作用开始对中国的炼钢行业构成严重冲击，产能过剩导致炼钢行业出现大规模亏损。<ref>[http://finance.qq.com/a/20120904/001377.htm?pgv_ref=aio2012&ptlang=2052 中钢协：8月钢材价格已崩盘 祸起四万亿计划]</ref>

也有评论认为， 4万亿不但导致产能过剩， 其问题也包括规模过大， 过于集中于固定资产和建筑业。 <ref>[http://www.businessinsider.com/chinas-excess-capacity-problem-2013-6    China's Excess Capacity Problem]</ref>2013年8月14日，《[[人民日报|人民日报海外版]]》刊登报道指出新的投资“不搞新版四万亿”，[[中国国际经济交流中心|中国国际经济交流中心]]信息部部长徐洪才接受该报记者采访时也指出，“当时四万亿投资是应急，有些项目没有充分论证其可行性和风险性，而且政府行政干预过多，造成了投资效益不太高，有些还加剧了经济结构的扭曲，加剧了产能过剩。”<ref>[http://paper.people.com.cn/rmrbhwb/html/2013-08/14/content_1283120.htm 不搞新版四万亿 明确重点补短板 新增投资不会“撒胡椒面”，人民日报海外版2013年8月14日，第02版]</ref>
