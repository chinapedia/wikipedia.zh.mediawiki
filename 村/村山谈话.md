'''村山談話'''是在1995年8月15日（即[[第二次世界大戰|第二次世界大戰]][[日本投降|日本宣佈無條件投降]]50周年紀念日），[[日本國|日本國]][[内阁总理大臣_(日本)|内閣總理大臣]][[村山富市|村山富市]]發表的談話。該談話承認，日本藉由[[殖民統治|殖民統治]]和[[侵略|侵略]]，對許多國家的人民造成重大傷害與痛苦，特別是[[亞洲國家|亞洲國家]]。該談話受到[[中華人民共和國|中]]、[[大韓民國|韩]]等曾遭受日本殖民禍害的國家及日本國内部分人士的肯定。[[日本右翼團體|日本右翼團體]]則稱[[外交|外交]]手腕過於軟弱，痛批其徹底否定日本的文明建設。

== 評價與繼承 ==
2013年7月，[[動畫導演|動畫導演]][[宮崎駿|宮崎駿]]在[[吉卜力工作室|吉卜力工作室]]免費[[月刊|月刊]]《熱風》2013年7月號藉村山談話批評，現在日本的政府和[[政黨|政黨]]首腦缺乏歷史觀、沒唸書，他們只會聽[[佞臣|佞臣]]之言決定方針，結果他們在[[國際社會|國際社會]]踢到鐵板就立刻慌張地改口「基本上尊重村山談話」<ref>{{cite news|url=http://www.cna.com.tw/News/FirstNews/201307190076-1.aspx|title=宮崎駿批安倍：要修憲 太扯了|author=楊明珠|date=2013-07-19|accessdate=2013-08-11|publisher=[[中央通訊社|中央通訊社]]}}</ref>。

2013年7月7日，村山富市在日本[[新潟市|新潟市]]演講，嚴厲批判日本內閣總理大臣[[安倍晉三|安倍晉三]]企圖修改村山談話的意向，認為此舉不僅將使日本在[[國際政治|國際政治]]失去信譽、也是背叛歷代[[自由民主黨_(日本)|自由民主黨]]政權的行為，“如果對村山談話進行修改，不承認‘侵略’事實，那麼這個談話不就變成了一個謊言？歷代自民黨政權一直強調要繼承這一談話精神，也是否意味著在撒謊？”<ref>{{cite web |url = http://world.huanqiu.com/exclusive/2013-07/4101572.html |title=村山富市：安倍若修改村山談話將使日本失信 |author=关超 |work= 环球网 |date=2013-07-08|accessdate=2015-09-20}}</ref>

2014年1月24日，[[日本外務大臣|日本外務大臣]][[岸田文雄|岸田文雄]]在[[日本國會|日本國會]]就[[外交政策|外交政策]]發表演說，表示將繼承村山談話等過去[[日本內閣|日本內閣]]的歷史認識，就安倍晉三參拜[[靖國神社|靖國神社]]一事尋求中韓兩國的理解，「今後也將堅持走和平國家的道路」<ref>{{cite web |url = http://tchina.kyodonews.jp/news/2014/01/67896.html |title=日本外相表示將繼承村山談話等歷史認識 |author=[[共同社|共同社]]1月24日電 |work=共同社 |date=2014-01-24|accessdate=2014-02-11}}</ref>。

2014年5月，針對[[第2次安倍內閣|第2次安倍內閣]]宣布調查[[河野談話|河野談話]]一事，村山富市說，「且不說前韓國[[慰安婦|慰安婦]]的證言可信與否，（日軍強徵韓國婦女充當慰安婦）當時的強制性可想像，並非是事實就一定有記載，那樣的事不可能有紀錄」<ref>{{cite web|url=http://www.bbc.com/zhongwen/trad/world/2014/06/140620_japan_heye_talk|title=日本政府發表「河野談話」調查報告|author=童倩|work=BBC中文網|date=2014-06-20|accessdate=2015-06-13}}</ref>。

2015年4月1日，村山富市接受[[鳳凰衛視|鳳凰衛視]]訪問時說，安倍晉三曾經質疑，並非只有日本進行殖民統治和侵略，很多大國都有過同樣行為，「為何只有日本必須道歉，是日本哪裡做得不對」；在他看來，安倍晉三宣稱將整體繼承村山談話，只是表面文章<ref>{{cite news |author=邱國強 |publisher=中央通訊社 |url = http://www.cna.com.tw/news/acn/201504020466-1.aspx |title=日前首相村山：中日應共同開發釣魚台|date=2015-04-02 |accessdate=2015-04-03}}</ref>。

2015年5月18日，村山富市呼籲安倍晉三發表第二次世界大戰結束70周年紀念日談話時效法村山談話，他說，如果村山談話只是他的個人發言，就不會有太大價值；但村山談話是獲[[日本內閣會議|內閣會議]]通過的決議，所以重要性不同；村山談話中提到「深切懊悔」與「誠心道歉」，日本歷任內閣「也自此向世界承諾，他們會遵守聲明」；安倍晉三似乎準備在第二次世界大戰結束70周年紀念日談話中「淡化」道歉的部份，他認為這是錯誤的舉動<ref>{{cite news |author=[[法新社|法新社]]東京18日電 |publisher=法新社 |url = https://tw.news.yahoo.com/%E5%AE%89%E5%80%8D%E4%B8%8D%E7%82%BA%E4%BA%8C%E6%88%B0%E9%81%93%E6%AD%89-%E5%89%8D%E6%97%A5%E7%9B%B8%E6%9D%91%E5%B1%B1%E5%96%8A%E8%A9%B1-120503642.html |title=安倍不為二戰道歉 前日相村山喊話|date=2015-05-18|accessdate=2015-06-07}}</ref>。

2015年6月25日，[[中國社會科學院|中國社會科學院]]日本研究所副院長[[高洪|高洪]]說，日本首相以個人或首相身分發表談話並無本質區別，“就算是個人，也能代表國家意志。比如村山談話就是在爭議聲中坐實的——談話以村山富市個人名義發表，卻被視作日本對戰爭受害國的政治承諾。”<ref>{{cite web |url = http://news.takungpao.com.hk/world/exclusive/2015-06/3033208.html |title=專家：中國沒有反日教育 中日戰略磨合至少十幾年 |author=常曉宇 |work=[[大公網|大公網]] |date=2015-06-26|accessdate=2015-09-03}}</ref>

2015年7月17日，日本的[[國際法|國際法]]、歷史及國際政治各領域專家學者共74人聯名發表聲明，要求安倍晉三在第二次世界大戰結束70周年紀念日談話中，繼承承認過去大戰的“殖民統治和侵略”並表明“深刻反省和由衷道歉”的以往談話；聲明強調，倘若不採用村山談話與[[小泉談話|小泉談話]]中包括的“由衷道歉”一詞，會引起國際關注，談話本身會受到負面評價，甚至會使日本政府一直以來強調的對過去的反省遭到誤解與不信任。此74人中，包括[[明治大學|明治大學]]特聘教授[[大沼保昭|大沼保昭]]、[[東京大學|東京大學]]名譽教授[[三谷太一郎|三谷太一郎]]、前[[聯合國難民事務高級專員|聯合國難民事務高級專員]][[緒方貞子|緒方貞子]]、作家[[半藤一利|半藤一利]]及東京大學名譽教授[[和田春樹|和田春樹]]<ref>{{cite web |url = https://tchina.kyodonews.jp/news/2015/07/101676.html |title=日本學者等74人聯名要求安倍新談話繼承“道歉”表述|author=共同社7月17日電|work=共同社 |date=2015-07-17|accessdate=2015-07-19}}</ref>。

2015年8月14日，韓國[[新政治民主联合|新政治民主联合]]發言人[[金聖洙|金聖洙]]說，安倍晉三的第二次世界大戰結束70周年紀念日談話，虽然包含村山谈话的四大核心措辞——“谢罪”、“反省”、“侵略”、“殖民统治”，但缺乏诚意，巧妙地逃避责任，让人失望<ref>{{cite web |url = http://chinese.yonhapnews.co.kr/domestic/2015/08/14/0401000000ACK20150814001600881.HTML |title=韩国朝野对安倍谈话内容立场略有差异 |author=[[韩联社|韩联社]]首尔8月14日电 |work=韩联社 |date=2015-08-14|accessdate=2015-09-04}}</ref>。

2015年8月31日，前[[德国总理|德国总理]][[施罗德|施罗德]]接受[[新华社|新华社]]记者书面采访时说，“直面历史，以坦诚和批判性的方式对历史追根问底”对每个国家都很重要，“愿意寻求和解和原谅”同样重要，“在日本，也有一些重要的动议。以日本前首相村山富市为例，1995年，他在谈话中反省[[侵略战争|侵略战争]]历史，为殖民统治和侵略历史谢罪。亚洲人民应该延续这条和解之路。”<ref>{{cite web |url = http://news.xinhuanet.com/world/2015-09/01/c_1116437432.htm |title=直面记忆 正视责任——访德国前总理施罗德 |author=商婧、何梦舒 |work= 新华网 |date=2015-09-01|accessdate=2015-09-05}}</ref>

2018年6月25日，和田春树在《[[三联生活周刊|三联生活周刊]]》的专访中说，在村山谈话与河野谈话画定了基线之后，“所有后继的日本首相都不得不就历史问题做出某种程度的反省，哪怕内心不完全赞同”；安倍晋三也不例外，所以“尽管他的确有否定村山谈话的念头，但在谋士的进谏之下，还是不得不采取一种隐蔽的模糊化处理”<ref>{{cite web |url = |title=超越“日本中心”史观——专访东京大学荣休教授和田春树|author=刘怡、刘周岩 |work=《三联生活周刊》2018年第25期 |date=2018-06-25|accessdate=2018-07-09}}</ref>。
