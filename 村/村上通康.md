'''村上通康'''（1519年—1567年11月23日）是[[日本戰國時代|日本戰國時代]]於[[伊予國|伊予國]]的武將。{{tsl|ja|河野氏}}家臣。[[村上水軍|村上水軍]]（來島水軍）的大將。{{tsl|ja|来島城|來島城}}城主。

以'''來島通康'''一名而為人所知，不過實際上，改姓[[來島氏|來島氏]]的是四男[[來島通總|通總]]仕於[[豐臣秀吉|豐臣秀吉]]以後的事，通康自身並無以來島氏為姓。

== 生平 ==
在[[永正|永正]]16年（1519年）出生。在年幼時期，[[正岡氏|正岡氏]]、[[重見氏|重見氏]]等有力[[國人|國人]]在府中（現今[[愛媛縣|愛媛縣]][[今治市|今治市]]）持續反亂，而在鎮壓當地勢力後，來島[[村上氏|村上氏]]成為府中方面的最有力者。

[[天文_(後奈良天皇)|天文]]10年（1541年），在[[伊予國|伊予國]]一之宮的[[大山祇神社|大山祇神社]]{{tsl|ja|大三島}}被{{tsl|ja|大内水軍|大內水軍}}襲撃。受河野氏[[當主|當主]][[河野通直_(彈正少弼)|河野通直]]之命，與[[得居通年|得居]]、[[平冈房实|平岡]]、今岡等[[警固眾|警固眾]]一同出陣，並得到能島村上氏的援軍，於是與[[神官|神官]][[大祝氏|大祝氏]]等人一同撃退大內水軍。

受通直深厚信任，並成為其女婿。在通直與兒子[[河野晴通|晴通]]對立時，投向通直，被選為河野氏的[[家督|家督]]繼承者，不過因為河野氏家臣團多數支持晴通，於是與通直一同逃到[[湯築城|湯築城]]。居城{{tsl|ja|来島城|來島城}}被攻撃，不過成功堅守，並與河野氏家臣達成和睦。雖然沒能成為後繼者，不過作為河野一族，被允許使用河野氏的[[家紋|家紋]]和改姓{{tsl|ja|越智氏}}。

在[[河野通宣_(左京大夫)|通宣]]時期，在河野氏的內政、外交、軍事各方面都是重臣，亦負責與[[室町幕府|室町幕府]]和周邊勢力聯絡。

[[弘治_(後奈良天皇)|弘治]]元年（1555年），[[毛利元就|毛利元就]]與[[陶晴賢|陶晴賢]]在[[嚴島|嚴島]]戰鬥時（[[嚴島之戰|嚴島之戰]]），率領旗下水軍支援毛利軍（一說指並無參戰）。此後，積極協力元就，率領水軍與[[大內氏|大內氏]]、[[大友氏|大友氏]]戰鬥。

[[永祿|永祿]]10年（1567年），與{{tsl|ja|伊予宇都宮氏}}戰鬥時，突然發病，於是返回湯築城，不過沒能痊癒，並在10月23日死去。此前，向元就請求支援河野氏，元就因為嚴島之戰的恩義而向伊予派遣援軍，毛利軍以[[小早川隆景|小早川隆景]]為中心，登陸伊予，擊破[[宇都宮豐綱|宇都宮豐綱]]和{{tsl|ja|土佐一条氏|土佐一條氏}}。

== 家臣團 ==
*[[村上吉繼|村上吉繼]]—[[河內守|河內守]]，[[伊予國|伊予國]]{{tsl|ja|甘崎城}}城主。作為河野氏的主力，與一條氏的戰鬥，亦有參與[[第一次木津川口之戰|第一次木津川口之戰]]。
*[[村上吉鄉|村上吉鄉]]—[[越後守|越後守]]。
*[[原興生|原興生]]—[[豐前守|豐前守]]。

== 子孫 ==
死後，其中1個女兒嫁予[[穗井田元清|穗井田元清]]，誕下兒子[[毛利秀元|毛利秀元]]，秀元在後來成為[[大名|大名]]。而直系的[[來島長親|來島長親]]亦在[[江戶時代|江戶時代]]成為大名。

== 逸話 ==
*不只擅於水戰，亦在陸戰中有許多武勇表現。在[[重見氏|重見氏]]的戰鬥中，曾被敵將以[[槍|槍]]刺穿[[鎧甲|鎧甲]]而受傷，卻反而殺死敵將；在得知友方[[古河內氏|古河內氏]]的首級被敵將取去後，衝入敵陣並奪回首級；在宇和島的合戰中，率先突入敵陣，勇名遠播。

*根據『[[予陽河野家譜|予陽河野家譜]]』記載，在[[河野通直_(彈正少弼)|河野通直]]與兒子[[河野晴通|晴通]]爭鬥時，居地被家臣團攻擊，此時阻止打算自殺的通直，並親自背其離開，並在居城{{tsl|ja|来島城|來島城}}堅守。
