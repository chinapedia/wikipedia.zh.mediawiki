'''村井良大'''（むらい りょうた）（{{bd|1988年|6月29日|}}）是[[日本|日本]]男[[演員|演員]]。隸屬於[[ヴァーサタイルエンタテインメント|ヴァーサタイルエンタテインメント]]旗下的藝人，出生於[[日本|日本]][[東京都|東京都]]，[[血型|血型]]O型。身高171cm。主要活躍于舞臺劇。

== 人物介紹 ==
*興趣是舞蹈，音樂和電影鑑賞。
*特長是歌唱和舞蹈。
*家中排行第三，上有哥哥和姐姐。
*家裏經營印刷公司
*擁有普通小型車和普通輕型機車的駕照
*關於“村井万能説”這個説法是因為在[[戦国鍋TV_〜なんとなく歴史が学べる映像〜|戦国鍋TV 〜なんとなく歴史が学べる映像〜]]中一人飾演多個不同類型角色而受到讚賞。
（SHICHIHON槍 : 加藤嘉明 / 天正遣欧少年使節 : 千々石ミゲル / 堺衆 : 津田宗及 / 信長と蘭丸 : 織田信長 / 徳川15代将軍 : 徳川家康 / AKR四十七 : 大石内蔵助 / 天草四郎と島原DE乱れ隊 : 大矢野松右衛門 / 戦国サポートセンター3 : 小早川隆景 役 他）。
*2006年以舞臺劇『[[赤毛のアン|赤毛のアン]]』出道。
*2007年首次出演電視劇『[[風魔小次郎|風魔小次郎]]』並擔當主角。
*2008年承接前作電視劇，首次擔當舞臺劇『[[風魔小次郎|風魔小次郎]]』主演。
*2012年2月 因所屬事務所ベルジーエンタテインメント・ディビジョン的解散，而移籍到ウェーブマスター事務所的旗下。
*2014年為拍攝寫真集而首次到關島
