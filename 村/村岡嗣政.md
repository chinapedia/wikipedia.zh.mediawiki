'''村岡 嗣政'''（むらおか つぐまさ；{{bd|1972年|12月7日|||Muraoka Tsugumasa}}），是[[日本|日本]]的[[政治人物|政治家]]和總務官僚，出任第47代山口縣知事（1期）。

== 經歷 ==
[[山口县|山口]]縣[[宇部市|宇部市]]出身。山口縣立宇部高等學校、東京大學經濟學部畢業。1996年進入[[自治省|自治省]]（現・總務[[總務省|省]]）工作。廣島市財政局財政課長、[[高知县|高知]]縣總務部財政課長、2012年4月後擔任總務省自治財政局財政課財政企劃官<ref>{{Template:Cite web|url = http://t-muraoka.jp/profile/|title = 村岡つぐまさ プロフィール|accessdate = 2014-02-23}}</ref>。

=== 山口縣知事 ===
2014年1月，原先擔任知事的山本繁太郎因病辭職，隨後被自民黨和公明黨推薦稱為山口縣知事的候補人選，同年2月23日超越高邑勉、藤井直子票數當選，於是從總務省退職出任山口縣知事（山本前知事3月15日逝去）<ref>{{Template:Cite news|url = http://www3.nhk.or.jp/news/html/20140223/k10015466293000.html|title = 山口県知事選 自公推薦の村岡氏が当確|date = 2014-02-23|accessdate = 2014-02-23|newspaper = NHK}}</ref><ref>{{Template:Cite web|url = http://www.pref.yamaguchi.lg.jp/cmsdata/9/f/6/9f60e86a4d83d87d1adc6c72b9e5372b.pdf|title = 平成26年山口県選挙管理委員会告示第36号「山口県知事選挙において当選人となった者の住所及び氏名」（山口県報平成26年号外10号1頁）|date = 2014-02-25|accessdate = 2014-02-25|publisher = 山口県|format = PDF}}</ref>。
