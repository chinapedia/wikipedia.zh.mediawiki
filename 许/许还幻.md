'''许还幻'''（{{bd|1977年|10月2日}}），[[中国|中国]][[女演员|女演员]]<ref>{{cite web |title=许还幻 |url=http://ent.sina.com.cn/star/mainland/file/xuhh.html |website=新浪娱乐 |accessdate=2019-02-20}}</ref>，毕业于[[北京电影学院|北京电影学院]]96级表演系<ref>{{cite web |title=我要成名：章子怡和中戏同学的这十年 |url=http://news.sina.com.cn/s/sd/2010-04-26/141520155327.shtml |website=新浪新闻 |accessdate=2019-02-20}}</ref>，曾经参加过多部影视剧的演出。
