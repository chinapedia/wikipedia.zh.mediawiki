'''许前飞'''（{{bd|1955年|10月|}}），[[河南|河南]][[孟津|孟津]]人，[[研究生|研究生]]学历，[[法学博士|法学博士]]学位，1976年7月加入中国共产党，1974年10月参加工作。毕业于[[武汉大学|武汉大学]][[哲学|哲学]]本科、[[国际法|国际法]]硕士、[[国际经济法|国际经济法]][[博士|博士]]专业，[[第十二届全国人民代表大会|第十二届全国人民代表大会]]江苏地区代表。

曾任武汉大学国际法学系副主任。1996年11月至2003年2月任[[海南省海口市中级人民法院|海南省海口市中级人民法院]]院长。2007年任[[海南省人民政府|海南省人民政府]]秘书长；2008年1月担任[[云南省高级人民法院|云南省高级人民法院]]院长。2008年起担任[[全国人大代表|全国人大代表]]<ref>{{cite news |title=全国人大代表信息-许前飞 |url = http://www.npc.gov.cn/delegate/viewDelegate.action?dbid=112362 |accessdate=2013-04-05 |website=全国人大网 }}</ref>。2013年，被选为[[全国人大代表|全国人大代表]]<ref>{{cite news |title=全国人大代表信息-许前飞 |url = http://www.npc.gov.cn/delegate/viewDelegate.action?dbid=120706 |accessdate=2013-03-28 |website=全国人大网 |deadurl=yes |archiveurl=https://web.archive.org/web/20140523225700/http://www.npc.gov.cn/delegate/viewDelegate.action?dbid=120706 |archivedate=2014-05-23 }}</ref>。2013年1月至2017年7月，任[[江苏省高级人民法院|江苏省高级人民法院]]院长<ref>{{cite web |url = http://district.ce.cn/newarea/sddy/201707/21/t20170721_24368595.shtml |title=许前飞辞去江苏高院院长及十二届全国人大代表职务(简历) |website=中国经济网 |date=2017年7月21日 |accessdate= }}</ref>。

==落马==

2017年7月21日，许前飞辞去第十二届全国人大代表职务。2017年7月24日，中共中央纪委网站宣布，对江苏省高级人民法院原党组书记、院长许前飞严重违纪问题进行了立案审查。经查，许前飞违反政治纪律和政治规矩，应与其关系密切的律师和私营企业主请托，干预和插手具体案件审判工作，以案谋私，严重损害司法公信力和人民法院形象；严重违反中央八项规定精神，违规出入私人会所和打高尔夫球，接受公款宴请；违反组织纪律，违规选拔任用干部，不按规定报告个人有关事项；违反廉洁纪律，收受礼品、礼金。依据《中国共产党纪律处分条例》等有关规定，经中央纪委常委会会议研究并报中共中央批准，决定给予许前飞同志撤销党内职务处分，由最高人民法院给予其撤职处分，降为正局级非领导职务；收缴其违纪所得<ref >{{cite web |url = http://district.ce.cn/newarea/sddy/201707/24/t20170724_24390640.shtml |title=江苏高院原院长许前飞被降为正局级非领导职务(简历) |website =中国经济网 |date=2017年7月24日 |accessdate= }}</ref>。此前网上已有关于许前飞的举报。江苏牧羊集团有限公司曾在今年2月22日发出公开信，举报许前飞滥用司法公权，以“内部电传”的形式违法指定管辖，再审五年前已判决的案件。信中还称，许前飞与案件一方当事人存在巨大的利益输送嫌疑。许是中共十八大以来，法院系统首位在任上被调查的省级高院院长<ref>{{cite web |url = http://news.163.com/17/0724/15/CQ4CDIE30001899N.html |title = 许前飞被降级 系十八大以来首个在任遭查高院院长 |website = 网易 |accessdate=2017-07-24 }}</ref>。2017年9月1日，[[第十二届全国人民代表大会常务委员会|第十二届全国人民代表大会常务委员会]]第二十九次会议通过其辞呈<ref>{{cite web |url = http://www.npc.gov.cn/npc/xinwen/2017-09/01/content_2027933.htm |title=第十二届全国人民代表大会常务委员会代表资格审查委员会关于个别代表的代表资格的报告(2017年9月1日第十二届全国人民代表大会常务委员会第二十九次会议通过) |accessdate=2017-09-06 |website = 中国人大网 }}</ref>。

2019年1月，前[[中央电视台|中央电视台]]主持人[[崔永元|崔永元]]在其个人社交媒体[[新浪微博|新浪微博]]发布了一段原[[最高人民法院|最高人民法院]]法官[[王林清_(法官)|王林清]]拍摄的一段9分多钟的自述视屏，其中提到2014年6月17日，王林清被最高院派到江蘇省沭陽縣向三級法院的法官講課，當晚时任[[最高人民法院監察局|中纪委驻最高人民法院纪检组]]組長[[張建南|張建南]]、副組長[[何莉|何莉]]致電时任江蘇省高院院长、黨組書記許前飛，要求以犯下嚴重罪行為由將王林清抓捕。许随即命令时任宿迁市中级人民法院代院长、党组书记[[汤小夫|汤小夫]]安排法警抓捕，并将王扣押在宿迁市中院，隔日由纪检组带回北京。王林清在视频中还透露，此前最高院纪检组办公室主任闫长林曾要求王在“山西省王永安和王见刚侵犯财产权一案”中改判原告败诉，遭其拒绝<ref>{{cite web|url=http://china.caixin.com/2019-01-06/101367011.html|title=法官王林清称另一起案件遭干预 拒贿曾受威胁|publisher=财新网|date=2019-01-06}}</ref>。
