==生平简要==
1973年，许志永生于[[河南省|河南省]][[民权县|民权县]]，1990年就读于[[兰州大学|兰州大学]]法律系，获法学学士学位。1995年，再入兰州大学法律系攻读法学硕士学位，1998年毕业论文《论经济法本土化的经济误区》中批判了以反对西化、强调本土价值、主张法治服务大局而著称的法学家[[朱苏力|朱苏力]]，没能拿到硕士学位。<ref>{{cite web|url=http://www.eeo.com.cn/observer/shijiao/2008/11/12/120096.shtml|title=行侠仗义许志永 |publisher=经济观察网|date=2008-11-13}}</ref> 1998年7月-1999年9月，在《[[南方周末|南方周末]]》等单位实习。1999年，就读于[[北京大学法学院|北京大学法学院]]，师从[[朱苏力|朱苏力]]，2002年获[[法学博士|法学博士]]学位。“教育平权”的主要发起人，积极倡导不分户籍出身平等的接受升学教育。在北京“非京籍与京籍义务教育阶段享有同等待遇”的工作中有突出贡献。

2002年之后为[[北京邮电大学|北京邮电大学]]文法学院讲师<ref name="VOA0806" />。2003年11月当选为北京[[海淀区|海淀区]][[人大代表|人大代表]]，2006年获得连任。2005年下半年，为[[耶鲁大学中国法研究中心|耶鲁大学中国法研究中心]]访问学者。2009年起许志永被剥夺上课资格。2011年7月30日，许志永与[[崔筝|崔筝]]举行婚礼，证婚人[[李昌平|李昌平]]。2013年4月，许志永遭到[[软禁|软禁]]，同年7月16日被警方拘留，8月2日被正式逮捕，12月4日许志永被移送检方。2014年1月22日受审，1月26日被宣布判刑4年。<ref>{{cite web|url=http://www.hrw.org/zh-hans/news/2014/01/26|title=人权观察要求撤销对许志永的不公判决|publisher=人权观察|date=2014年1月26日}}</ref>

2017年7月15日许志永刑满出狱，离开了北京郊区的监狱，他身体状况良好，对[[刘晓波|刘晓波]]去世的消息感到非常难过。<ref >{{cite web|url=https://www.voachinese.com/a/voanews-20170715-xu-zhiyong/3945446.html | language= | title=著名维权人士许志永获释 |publisher=美国之音 |date=2017年7月15日 |accessdate= }}</ref>

== 社会活动 ==

=== 公盟活动阶段 ===
{{main|公盟}}

2003年，[[孙志刚|孙志刚]]在广州收容站被殴打致死，许志永在“三博士上书事件”中首次走入公众视野。许志永与另外两名法学博士俞江、[[滕彪|滕彪]]联名上书[[全国人大常委会|全国人大常委会]]，要求对《[[城市流浪乞讨人员收容遣送办法|城市流浪乞讨人员收容遣送办法]]》进行[[违宪审查|违宪审查]]，废除[[收容遣送制度|收容遣送制度]]，对废止该办法起到推动作用<ref name="VOA0806" />。同年10月，许志永、[[滕彪|滕彪]]、俞江和[[张星水|张星水]]共同创办公益性民间组织“[[阳光宪道社会科学研究中心|阳光宪道社会科学研究中心]]”（简称“[[阳光宪政|阳光宪政]]”）<ref name="VOA0806" />。

2004年，与[[范亚峰|范亚峰]]、滕彪、秋风、[[王怡|王怡]]等法律学者共同起草完成《完善我国宪法人权保护条款的建议》，提出了全面修改中国宪法人权保护条款使之形成了一个完整的人权保护体系的建议。2005年，阳光宪道社会科学研究中心无法继续在北京民政局注册，改名为“北京公盟咨询有限责任公司”（简称公盟），在北京工商局注册。同年，为了撰写《中国信访报告》，许志永曾在[[北京南站|北京南站]]的“上访村”住过两个月，还曾多次探访京城内关押上访民众的“[[黑监狱|黑监狱]]”，并受到不法分子的殴打<ref name="VOA0806" />。2006年初，主持撰写2005年度《中国人权发展报告》，同年12月，公盟着手开始对以北京为主的打工子弟学校的生存状况和法律保障问题进行调查研究，开始关注城市新移民子女的受教育权利。2007年，主持《关于国内法与公民权利与政治权利国际公约衔接》的研究报告。

2008年6月，许志永与公盟大力协助北京维权律师推动北京律协的民主选举。8月下旬，律师们起草了《顺应历史潮流，实现律协直选——致全体北京律师、市司法局、市律协的呼吁》和《北京律师协会选举程序（草案）》等文章，这些文章在网络上迅速传播，被各界关注。9月中旬，向北京市近16000名律师分别发出信函。9月13日，直选律师开办的《律师沙龙》网站开通运行。11月，向北京市司法局和北京市律协发出《关于敦促律协换届审计并公开财务的公开信》。12月，北京市律师协会七届五次代表大会召开，会议审议通过了《北京市律师协会章程》。推动律协[[直选|直选]]的活动以失败告终。2009年7月，公盟被政府取缔，并以“涉嫌偷税”之名义指控许志永，<ref>[http://www.rfa.org/mandarin/Xinwen/gong-07312009120704.html 公盟负责人许志永被刑事拘留]</ref>8月13日，以“偷税罪”的名义被正式逮捕，之后又被取保候审，获得暂时自由。<ref name="BBC">[http://news.bbc.co.uk/chinese/simp/hi/newsid_8210000/newsid_8216700/8216755.stm 中国维权律师许志永获取保候审]</ref>

=== 公民活动阶段 ===
{{main|新公民运动}}

2010年3月，“公盟”改名为“公民”，继续从事公益维权活动。“公民”由许志永、王功权、黎雄兵、滕彪、[[李方平|李方平]]发起成立，是以建立约束权力的民主法治制度为目标，理性、建设性地推动民主、法治和社会正义的公民自发的公益组织。组织发起倡导以“自由，公义，爱”为主旨的[[新公民精神|新公民精神]]的[[新公民运动|新公民运动]]。同年8月21日，北京市公安局解除了对许志永的取保候审。

2011年11月，许志永参加北京[[海淀区|海淀区]]北邮选区基层[[人大代表|人大代表]]竞选。官方推出的候选人包括北邮校长[[方滨兴|方滨兴]]。许志永在竞选过程中受到官方层层阻挠，未能列入正式候选人名单。总有效票数为22000多（一票可投多人）。其中有3500多人用“另填他人”方式投给许志永，但不敌方滨兴的16000多票，最终落败。<ref>{{cite web|url=http://www.dw.de/dw/article/0,,15526153,00.html |title=人大代表选举：许志永PK方滨兴 |date=2011-11-11 |accessdate=2011-11-24 |publisher=德國之聲 }}</ref><ref>{{cite web|url=http://www.chinese.rfi.fr/中国/20111110-中国网络防火墙之父方滨兴高票当选地方人大代表 |title=中国网络防火墙之父方滨兴高票当选地方人大代表 |date=2011-11-11 |accessdate=2011-11-24 |publisher=法广中文网 }}</ref>

2012年11月15日，[[习近平|习近平]]在[[中共十八大|中共十八大]]当选[[中国共产党中央委员会总书记|总书记]]后，许志永发出《致习近平先生的公开信：一个公民对国家命运的思考》。<ref>{{cite web |url=http://www.chinese.rfi.fr/首页/20121119-许志永：-致习近平先生的公开信——-个公民对国家命运的思考 |title= 许志永：致习近平先生的公开信——一个公民对国家命运的思考 |accessdate=2012年11月19日  |publisher=法广中文网 |archiveurl= http://www.chinese.rfi.fr/首页/20121119-许志永：-致习近平先生的公开信——-个公民对国家命运的思考|archivedate= 2012年11月26日|work= |language =zh-cn}}</ref>11月24日，许志永被北京国保人员带走。<ref>{{cite web |url=http://www.voachinese.com/content/china-human-right-20121124/1552486.html |title=公盟许志永被警方问话至今无音信 |accessdate=2012年11月26日 |publisher=美国之音 |archiveurl=http://www.voachinese.com/content/china-human-right-20121124/1552486.html |archivedate=2012年11月26日 |work= |language =zh-cn}}</ref>12月22日在《[[紐約時報|紐約時報]]》中文網發表「一名漢人對一名自焚藏人的哀悼」，對[[藏區連環自焚事件|藏區連環自焚事件]]中「為自由而死」的藏人的關注。<ref>[http://cn.nytimes.com/article/opinion/2012/12/22/c22xu/zh-hk/ 一名漢人對一名自焚藏人的哀悼], [[紐約時報|紐約時報]]中文網, 2012年12月22日</ref>

2013年2月26日在[[北京地铁|北京地铁]]站口发放[[传单|传单]]，呼吁在京无北京户口的市民在2月28日到北京市教委大楼前聚众，表达要求教育权平等的呼声。27日遭北京国保传讯<ref>{{cite web|url=https://twitter.com/zhiyongxu/status/306770280838221825|title=许志永推特}}</ref>。同年4月12日，许志永应[[香港中文大学|香港中文大学]]和滕彪的邀请，计划去香港参加“孙志刚案十周年研讨会”。在机场过了边检候机之后被限制人身自由，从此被软禁在家。7月16日正式以涉嫌“聚众扰乱公共场所秩序”被刑事拘留，引发广泛的抗议<ref>{{cite news|title=维权人士许志永涉嫌扰乱公共秩序被拘|url=http://cn.nytimes.com/china/20130718/c18xuzhiyong/|accessdate=2013-07-18|newspaper=纽约时报中文网|date=2013年7月18日}}</ref>。8月22日，许志永被正式批捕<ref>{{cite news|title=Chinese citizens movement leader arrested|url=http://bigstory.ap.org/article/chinese-citizens-movement-leader-arrested|newspaper=美联社|author=Gillian Wong|date=2013年8月23日}}</ref>。

== 思想观点 ==
在2008年《经济观察报》的专访中，许志永称自己有英雄主义情结，自己所做的事情具有[[甘地|甘地]]色彩，他说：

{{cquote|
我觉得中国面临的改革不仅仅是政治制度的改革，也是政治文化的改革。现代文明政治不仅要有一套宪政制度，还要有一批人作为宪政的第一推动者，让宪政制度良好的运转，我觉得我们这代人要扮演这个角色。过去很多从事政治的，就是靠我有多少人马有多少枪，但是我认为那是一种暴力的力量，我们要走到另一个极端去，走到极致，用自己的承担、自己的受苦唤醒人性中善良的一面，直到这种善良成为整个社会的主流，让那些暴虐的气息荡然无存，让那些野蛮暴力的传统彻底被压制，它们将成为末流，只有在这个基础上，这个社会才能真正建立一个现代文明的国家。<ref>{{cite web|url=http://www.eeo.com.cn/observer/dajia/2008/11/12/119954.shtml |title=许志永：我有英雄主义情节 |publisher=南方周末官方网站 |date=2008-11-12|accessdate=|language=}}</ref>| }}

在《[[南方人物周刊|南方人物周刊]]》对许志永的一次采访中，许志永表示，在目前中国的现实社会中，在[[立法|立法]]方面发展一般、但由于[[民主制度|民主制度]]没有跟上时代的发展，[[公权力|公权力]]监督和约束的力度不够，权大于法的滥行，导致了在[[执法|执法]]和[[司法|司法]]层面所表现出来的法治不健全。并向记者透露了自己目前所做的一切的动因：

{{cquote|
有了健全的民主制度，比如真实的公开的选举，权力会受到制约，再加上民主制度健全过程中强大的道义力量对政治文化的改造，民主制度才能良好运转。这也是我们一直在推动制度建设的动因。<ref name="infzm ">{{cite web|url=http://www.infzm.com/content/30427 |title=最迫切的问题是独立司法 |publisher=南方周末官方网站 |date=2009-06-22|accessdate=2009-08-18 |language=}}</ref>| }}

2009年接受《[[时尚先生|时尚先生]]》杂志关于中国梦专题的采访以及2013年被逮捕后，许志永戴着手铐在狱中，他分别谈及他的中国梦：
{{cquote|
我希望我们是个自由幸福的国家。每个人不需要违背良心，只要靠自己的才能和品德就可以找到合适的位置；一个简单而幸福的社会，人性的善得到最大的张扬，恶得到最大的抑制；诚实、信用、友爱、互助将成为我们生活的常态，没有那么多烦恼和愤怒，每一个人脸上是纯真的笑容。不管这个社会变成什么样子，不管这个国家多么扭曲多么荒谬，它都需要有一批勇敢的公民站出来，遵照他们的信念，把他们的权利、责任与梦想变为现实。<ref name="Esquire ">{{cite web|url=http://cn.nytimes.com/opinion/20130821/c21hua/ 
|title=许志永的中国梦 |publisher=纽约时报 |date=2013年8月21日|accessdate=2014-01-29}}</ref>| }}

2014年1月22日，许志永在北京市第一中级人民法院法庭上的陈述被法官打断，之后媒体在网络上公开发表传达其观点：

{{cquote|
……全世界超过137个国家和地区建立了官员财产公示制度，为什么中国就不能？“人民公仆”到底害怕什么呢？不要太贪婪，不义的财富带给自己的不只是奢华的生活，同时还是深深的恐惧不安，以及来自民众的愤怒与仇视。......现代民主的价值和尺度根植于普遍的人性，不是东方或者西方的，不是[[社会主义|社会主义]]或者[[资本主义|资本主义]]的，而是普适全人类。......一系列重大社会问题的根源就在于，一个特权利益集团垄断了全部国家政治权力和经济命脉。......我们是新时代的公民，理念上，彻底告别了敌人、江山、推翻、打倒的专制意识，坚守自由、公义、爱的信仰，行为上彻底告别阴谋、暴力等野蛮模式，以和平改良方式推动社会进步，在阳光下健康成长。......这个民族有太多阴暗苦毒的灵魂需要救赎，人与人之间有太多的戒备、恐惧和敌意，这些埋藏于灵魂深处的魔鬼必须被驱除，但这驱除的过程不是仇恨，而是救赎。<ref name="ftcs"/>| }}

== 评价 ==
网络专栏作家[[许知远|许知远]]在其2007年写著的文章《我们的进步年代》里，称许志永是一个激情洋溢、信念单纯的人，并推测许志永是因为拥有着社会底层的生活经验以及家庭教养，使得他对社会上的[[弱势群体|弱势群体]]有着本能的同情心，因此寻求公正，也就演变成为许志永血液里最重要的组成部分。到了2009年，在许志永被捕之后，则写了一篇名为《我们这一代》的文章，文章里回顾了许志永之前所作的努力，并如此评价许志永和其他合作人员<ref name="ftchinese0818">{{cite web|url=http://www.ftchinese.com/story.php?storyid=001012512|title=我们的进步年代|publisher=金融时报|date=2007-07-05|accessdate=2009-08-18|language=zh-cn|deadurl=yes|archiveurl=https://web.archive.org/web/20090810075224/http://www.ftchinese.com/story.php?storyid=001012512|archivedate=2009-08-10}}</ref>：“对社会满怀责任与深情，想通过自己的法律知识去帮助困境中的陌生人，以减少这个社会的不公，给那些悲观无力者希望。”

维权人士汪海洋说：“许志永有法律的精神、法治的理念和他的美好愿望。他的美好愿望就是想把这个国家，通过法治的手段，使人人得到一种（与自己才能）相适应的社会地位，铲除人压迫人、人欺辱人的现象，......黑监狱看守打他，他就让他打，因为他正气凛然，使原来打过他的人见了他就闻风而逃。这种精神，在中国人来说，太少见了，也太可贵了。”<ref name="VOA0806"/>
