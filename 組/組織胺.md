'''組織胺'''（{{lang-en|Histamine}}），是一種[[有機物|有機]]含氮[[化合物|化合物]]。它參與局部[[免疫反應|免疫反應]]和[[炎症|炎症]]反應，並具有作為[[瘙癢|瘙癢]]介體中心和調節[[腸道|腸道]]生理功能的作用。它还被用作[[神經遞質|神經遞質]]。組織胺由[[嗜鹼性球|嗜鹼性球]]和附近[[結締組織|結締組織]][[肥大細胞|肥大細胞]]產生。另外，它也增加[[微血管|微血管]]對[[白血球|白血球]]和某些[[蛋白質|蛋白質]]的通透性，以允許白血球从微血管进入感染組織并吞噬其中的病原體。组织胺廣泛存在於[[动植物|动植物]]组织中，可人工合成。

== 属性 ==
组织胺的磷酸盐为无色结晶或白色结晶性粉末，易溶于水。它是一種活性胺化合物，化學式是[[碳|'''C''']]<sub>'''5'''</sub>[[氫|'''H''']]<sub>'''9'''</sub>[[氮|'''N''']]<sub>'''3'''</sub>，分子量是111。

它与H<sub>1</sub>和H<sub>2</sub>组胺受体发生作用，能提高[[胃酸|胃酸]]分泌，使各种[[平滑肌|平滑肌]]发生[[痉挛|痉挛]]，并擴張[[毛细血管|毛细血管]]，增加通透性。作為身體內的一種化學傳導物質，可以影響許多細胞的反應，包括[[過敏|過敏]]、[[發炎|發炎]]反應等，也可以影響[[腦部神經|腦部神經]]傳導，會造成嗜睡等效果。

== 合成和代谢 ==
组织胺是通过L-[[组氨酸脱羧酶|组氨酸脱羧酶]][[催化|催化]]的反应的脱羧从[[组氨酸|组氨酸]]得到的，它是一个[[亲水性|亲水性]]的血管活性（Vasoactive）[[胺|胺]]。
[[Image:Histidine_decarboxylase.svg|thumb]]能转变[[組氨酸|組氨酸]]到組織胺。]]

== 儲存與释放 ==
[[Image:SMCpolyhydroxysmall.jpg|thumb]]
組織胺存在於[[肥大細胞|肥大細胞]]內，亦存在於[[肺|肺]]、[[肝|肝]]及[[胃|胃]]的粘膜組織內。它在過敏與發炎的調節上扮演着很重要的角色。組織胺屬於一種化學訊息，亦是[[胺能神經遞質|胺能神經遞質]]，參與中樞與週邊的多重生理功能。在[[中樞系統|中樞系統]]中，組織胺由特定的神經所合成，例如位在[[下視丘後核|下視丘後核]]的 结节乳头体核(tuberomammillary nucleus)，神經細胞多向延伸至大腦其他區域與[[脊椎|脊椎]]，因此暗示組織胺可能參與[[睡眠|睡眠]]、[[荷爾蒙|荷爾蒙]]的[[分泌|分泌]]、[[體溫|體溫]]調節，[[食慾|食慾]]與[[記憶|記憶]]形成等功能，另外還位於 伦霍塞克氏纤维(formatio reticularis) 與 前脑胞(telencephalon)中；在週邊部分，組織胺主要儲存在肥大細胞，[[嗜鹼性粒細胞|嗜鹼性粒細胞]]和[[腸嗜鉻細胞|腸嗜鉻細胞]]，可引起癢、打噴嚏、流鼻水等現象，此外組織胺結合到[[血管平滑肌|血管平滑肌]]上的接受器（H1R）導致血管擴張因而產生局部[[水腫|水腫]]，組織胺會使肺的[[氣管平滑肌|氣管平滑肌]]收縮引起呼吸道狹窄進而呼吸困難，腸道平滑肌收縮降低血壓以及[[心跳过速|增快心率]]等多項生理反應。
