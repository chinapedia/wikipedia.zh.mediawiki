{{vgname|粘粘世界|en=World of Goo|cn=粘粘世界|tw=黏黏世界}}是一款由独立游戏开发者2D Boy制作并发行，涵盖[[WiiWare|WiiWare]]、[[Mac_OS_X|Mac OS X]]、[[Windows|Windows]]、[[Linux|Linux]]多个平台的物理学题材益智游戏。在2008年的[[独立游戏节|独立游戏节]]中，《粘粘世界》荣获创新设计奖和优秀技术奖，并得到了塞尤玛斯·麦克纳利奖的提名。<ref name="igf">{{cite web |url=http://www.igf.com/02finalists.html |title=2008年独立游戏界获奖名单|work=Independent Games Festival |publisher=Think Services}}</ref>2008年10月13日游戏PC、Wii版在北美-{}-发布。随后其他平台、其他地区的游戏陆续上市。

==游戏内容==
《粘粘世界》以凯勒·加布勒的概念游戏《Tower of Goo》为原型，主要玩法是利用粘性小球Goo来构建建筑框架。<ref name="EGP">{{cite web |url=http://www.joystiq.com/2007/03/05/tower-of-goo-evolves-into-world-of-goo-2d-boy-is-born/ |title=Tower of Goo evolves into World of Goo, 2D Boy is born |work=Joystiq |publisher=Weblogs, Inc |date=2007-03-05 |author=Bardinelli, John}}</ref>游戏被分为五个章节，并细分为多个关卡。游戏吸取了[[蒂姆·伯顿|蒂姆·伯顿]]电影作品的特点，<ref>{{cite web|url=http://www.eurogamer.net/articles/world-of-goo-review|title=World of Goo|last=Blyth|first=Jon|date=2008-10-02 |publisher=Eurogamer|accessdate=2009-04-03}}</ref>让每个章节都有不同的主题画面和音乐风格，渲染出不一样的气氛，<ref name="EuroGamer">{{cite web |url=http://www.eurogamer.net/article.php?article_id=90750 |title= World of Goo First Impressions |work=EuroGamer |publisher=Eurogamer Network |author=Gillen, Kieron |date=2008-01-16}}</ref>此外，游戏提供了联机竞技模式，玩家需要用有限的Goo建造尽可能高的结构。玩家的记录会被上传到2D Boys的服务器，让全球其它玩家来挑战记录。

===游戏规则===
每个关卡的基本目标是收集一定数量的Goo到引向关卡外的水管。玩家要想方设法运用Goo构造桥梁、高塔等结构，同时克服重力和裂口、山峰、峭壁、尖刺等不利地形，最终将建筑搭建到水管口，以让没有用于搭建结构的Goo进入。如果玩家成功地搜集到了足够多的Goo就过关了。如果收集的Goo多于标准数额，多出部分可以在联机模式使用。<ref name="IGN">{{cite web |url=http://pc.ign.com/articles/846/846912p1.html |title=《粘粘世界》前瞻|publisher=[[IGN|IGN]]|date=2008-01-22 }}</ref>

根据各关卡的设定，当玩家使用较少的步骤、收集一定量Goo、或在一定时间内完成关卡时，即可获得OCD奖。在关卡地图中，已通过OCD挑战的关卡会用写有“OCD”字样的旗帜标记。<ref name="2D Boy Blog">{{cite web |url=http://2dboy.com/2008/08/14/leipzig-and-some-mini-goo-updates |first=Kyle |last=Gabler |title=Leipzig! And some mini-goo updates |work=2D Boy Blog |publisher=2D Boy |date=2008-08-14 |accessdate=2009-04-15 |archiveurl=https://web.archive.org/web/20090225164840/http://2dboy.com/2008/08/14/leipzig-and-some-mini-goo-updates/ |archivedate=2009-02-25 |deadurl=yes }}</ref>

Wii版的游戏支持四名玩家同机对战，Linux版也可以通过适当修改达到类似的效果。<ref name="multimouse">{{cite web|url=http://goofans.com/faq/world-goo/technical-questions/how-do-i-use-more-one-mouse|title=How do I use more than one mouse?|accessdate=2009-05-02}}</ref>

===关卡===
游戏被分为五个章节，每章包括数个关卡，总共47个。章节按照时间的顺序排序，初始章节设定为夏天，而最终章则设定在来年春天。

联机模式（World Of Goo Corporation）则脱离于五个章节之外。玩家在单机关卡中多收集的Goo会在这里出现，而玩家要用拥有的Goo从一个三角形开始建造尽可能高的结构。联机模式使用的是黑色的Goo，它们和绿色Goo一样可被拆除，但是每次只能像普通Goo一样连接周围的两个Goo。联机模式中，各玩家的记录会以一定高度的云朵标记，云朵上面写有玩家游戏名、国籍、记录高度、收集的Goo、使用Goo的数量等细节。排行榜上会给出联机模式前50名的记录，以及每个单机关卡前10名记录。

开发者曾一度宣布欧洲版游戏将附加第六个章节，设定在月球，<ref name="WiiWare World">{{cite web |url=http://www.wiiware-world.com/news/2008/08/2d_boy_interview_world_of_goo |title=2D Boy：关于《粘粘世界》|work=WiiWare World |first=Darrent |last=Calvert}}</ref>支持[[沙盒|沙盒]]模式。然而当官方宣布游戏将登陆[[WiiWare|WiiWare]]商店时，同时表示附加内容被删除。<ref name="2D Boy">{{cite web |url=http://2dboy.com/2008/11/11/world-of-goo-coming-to-wiiware-in-europe |title=《粘粘世界》将登陆欧洲WiiWare |work=2D Boy Blog |publisher=2D Boy |date=2008-11-11 |accessdate=2009-10-20 |archiveurl=https://web.archive.org/web/20100102183610/http://2dboy.com/2008/11/11/world-of-goo-coming-to-wiiware-in-europe/ |archivedate=2010-01-02 |deadurl=yes }}</ref>

==游戏开发==
《粘粘世界》的灵感来自《Tower of Goo》，一款在实验性游戏计划网（Experimental Gameplay Project）中非常有名的免费概念小游戏。

作为一款”独立游戏“（Independent Game），《粘粘世界》的主要开发者只有凯勒·加布勒（Kyle Gabler）和罗恩·迦密（Ron Carmel）两人，都是[[美国艺电|美国艺电]]的前雇员。两人只用了个人积蓄的10000美元就完成了《粘粘世界》的开发工作。尽管起初游戏项目无人问津，但两人努力通过博客、网页宣传游戏，并在独立游戏界中获奖，引发了游戏界的关注。<ref name="venturebeat">{{cite web |title=How the World of Goo became one of the indie video game hits of 2008 |url=http://venturebeat.com/2009/01/02/the-world-of-goo-became-one-of-the-indie-hits-of-2008/ |publisher=[[Venture_Beat|Venture Beat]] |accessdate=2009-01-10 |date=2009-01-02}}</ref>

《粘粘世界》中使用了大量的开放源代码组件，包括Simple DirectMedia Layer、Open Dynamics Engine物理效果模拟系统，irrKlang声音效果系统，TinyXML动画效果系统等。<ref name="wii.ign.com">{{cite web |url=http://wii.ign.com/articles/918/918954p1.html |title=World of Goo Review |first=Matt |last=Casamassina |work=IGN Wii |publisher=IGN Entertainment |date=2008-10-10 |accessdate=2008-10-13}}</ref>

开发者将游戏的翻译权交给了游戏玩家群体，在后者的努力下，游戏在欧洲上市时集成了荷兰语、法语、德语、意大利语、西班牙语补丁。<ref name="2dboy.com">{{cite web |url=http://2dboy.com/forum/index.php/topic,924.0.html |title=World of Goo translation for EU release |author=Gabler, Kyle |work=2D Boy |date=2008-12-01 |accessdate=2008-12-07 |deadurl=yes |archiveurl=https://web.archive.org/web/20090704040553/http://2dboy.com/forum/index.php/topic,924.0.html |archivedate=2009-07-04 }}</ref>此后，游戏在波兰又推出波兰语版。

2D Boy表示《粘粘世界》将不会推出续作。<ref name="WiiFanboy">{{cite web |url=http://nintendo.joystiq.com/2008/11/12/wii-fanboy-interviews-2d-boy-world-of-goo/|title=Wii Fanboy采访2D Boy|work=Joystiq |publisher=Weblogs, Inc |date=2008-11-12 |author=Hinkle, David}}</ref>而玩家群则运用编辑器制造出一些第三方关卡和[[游戏模组|模组]]。<ref name="goofans">{{cite web |title=《粘粘世界》爱好者：用编辑器制作第三方内容|url=http://goofans.com/|accessdate=2009-05-02}}</ref>
