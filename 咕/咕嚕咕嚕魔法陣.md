{{日本作品|咕嚕咕嚕魔法陣|魔法陣グルグル}}，是[[衛藤浩幸|衛藤浩幸]]於1992年至2003年在[[艾尼克斯|艾尼克斯]]（今已合併改名為[[史克威爾艾尼克斯|史克威爾艾尼克斯]]）旗下漫畫雜誌《[[月刊少年GANGAN|月刊少年GANGAN]]》創作的[[日本|日本]]搞笑[[奇幻漫畫|奇幻]][[漫畫|漫畫]]，及原作改編動畫，其内容情節有很多戲仿其它[[電子遊戲|電子遊戲]]。外傳有《[[杰他他勇舞傳|舞勇傳吉他吉他]]》。2012年11月1日起在《GANGAN ONLINE》連載續篇《咕嚕咕嚕魔法陣2》。

== 概述 ==
故事內容為，300年前被封印的魔王紀力再次復活，主角'''仁傑'''和'''歌莉'''須合力再次把紀力封印。沿途上二人路經不同的村落，接觸不同的人，並透過修行提升等級。

== 登場角色 ==

=== 主要角色 ===
; {{nihongo|尼克|ニケ}}
: {{配音2|JP|[[瀧本富士子|瀧本富士子]]（第1作）／[[南央美|南央美]]（第2作）／[[石上静香|石上静香]]（第3作）|TW|[[詹雅菁|詹雅菁]]|HK|[[雷碧娜|雷碧娜]]（TVB）、[[周恩恩|周恩恩]]（有線）}}
: 背景：男主角。13歳。出身於傑姆賈姆大陸，吉米納村。被斯萊取的代號為「大頭蟋蟀」。
: 出生：咕嚕咕嚕暦1298年10月21日
: 星座：[[天秤座|天秤座]]
: 血型：O型
: 職業：自稱[[勇者_(虛構職業)|勇者]](實際"勇者"只是稱號，並非職業)，正確職業是[[盜賊|盜賊]]。
: 屬性：光，能使用光明魔法。得到基拉基拉，所以能夠使用火、地、水、風屬性之劍。
: 性格：好色、愛吹牛
: 設定：被稱為光明族裔的人，從小接受父親的勇者訓練（實際是[[雜耍|雜耍]]訓練），不擅用劍但身手靈活，而且一世好運。

; {{nihongo|柯柯麗|ククリ}}
: {{配音2|JP|[[吉田小南美|吉田古奈美]]（第1作、第2作）／[[小原好美|小原好美]]（第3作）|TW|[[林美秀|林美秀]]|HK|[[鄭麗麗|鄭麗麗]]（TVB）、[[姜麗儀|姜麗儀]]（有線）}}
: 背景：女主角。13歳。出身於葛爾尼大陸，貓舌谷，但是在傑姆賈姆大陸的吉米納村中被扶養長大。被斯萊取的代號為「栗子叉燒包」。
: 出生：咕嚕咕嚕暦1299年3月16日
: 星座：[[雙魚座|雙魚座]]
: 血型：B型
: 職業：[[魔法|魔法]]使
: 屬性：暗，能使出最強的黑暗魔法－咕嚕咕嚕。
: 性格：記性差，常常畫出錯誤的魔法陣，使勇者身陷險境。個性溫柔天真。後期大幅進步，除非受騷擾(例如吉他吉他老伯)，否則很少錯誤。
: 設定：黑暗族裔米古米古族的最後遺裔。同時也是本作中掌管古柏大陸一帶魔力的魔神，不過尚需學習掌握自己的力量。喜歡勇者，太過在意好色的勇者，所以有時會感到不安。

; {{nihongo|風之精靈吉布兒|風の精霊ギップル}}
: {{配音2|JP|[[高乃麗|高乃麗]]（第1作、第2作）／[[櫻井孝宏|櫻井孝宏]]（第3作）|TW|[[謝佼娟|謝佼娟]]|HK|[[林元春|林元春]]（TVB）、[[黃鳳兒|黃鳳兒]]（有線）}}
: 背景：出身於比古納皮納大陸的風之山，受風之王統治的風之精靈。
: 工作：勇者與柯柯麗的[[嚮導|嚮導]]。
: 性格：膽小怕事，一和魔物對峙就會躲回老家。對「[[肉麻|肉麻]]」非常敏感。當有「肉麻」的場面和說話出現時，會自己跳出來[[吐槽|吐槽]]。又偶爾被稱為「核突精靈」、「飯筒精靈」
: 設定：身穿的[[長袍|長袍]]能變成[[帳篷|帳篷]]。每次遇到危險就第一時間消失，長袍底下總慣穿[[兜襠布|兜襠布]]（風之精靈的習性）。

; {{nihongo|阿德堡·艾魯多|アドバーグ·エルドル}}
: {{配音2|JP|[[緒方賢一|緒方賢一]]（第1作、第2作）／[[小西克幸|小西克幸]]（第3作）|TW|[[陳宗岳|陳宗岳]]|HK|[[陳曙光|陳曙光]]（第1作、第2作）→[[黃子敬|黃子敬]]（第3作）（TVB）、[[葉偉麟|葉偉麟]]（有線）}}
: 背景：跳吉它舞的老伯，所以被稱為「吉他吉他老伯（{{lang|ja|キタキタおやじ}}）」。52歳。出身於傑姆賈姆大陸，吉他鎮。
: 出生：咕嚕咕嚕暦1260年1月1日
: 星座：[[山羊座|山羊座]]
: 血型：B型
: 職業：曾是吉他鎮（{{lang|ja|キタの町}}）的鎮長，惟一的絕技是跳吉他舞。
: 屬性：無
: 性格：厚臉皮，喜歡把功勞往自己身上攬，那怕自己只有在一旁跳舞。
: 設定：上身赤裸、下身著[[草裙|草裙]]的老伯，特點是腳毛長。四處尋找吉他舞的傳人而旅行著（據他說法，吉他吉他舞從前是由年輕的少女來負責跳）可惜因鎮上生的女孩子越來越少，漸漸沒有人可以學。老伯惟一的用處是作為主角的擋箭牌。老伯生命力非常頑強，多次被人獸攻擊或妖精朱莉嘉從高山踢下，依舊能絲毫無損。

; {{nihongo|多瑪·帕洛特|トマ·パロット}}
: {{配音2|JP|[[神代知衣|神代知衣]]（第1作、第2作）／[[藤井幸代|藤井幸代]]（第3作）|TW|[[謝佼娟|謝佼娟]]|HK|[[謝潔貞|謝潔貞]]（第1作、第3作）／[[黃鳳英|黃鳳英]]（第2作）}}

: 設定：因為沒有魔法的背景，而希望成為魔技師。在尼克和柯柯麗之中，職業等級最高的一個。對著特別的法寶就會馬上著迷起來，誰也無法阻止。

; {{nihongo|-{zh-hk:潮潮·歌·絲拉姆;zh-hans:朱朱·庫·修娜姆爾;}-|ジュジュ·クー·シュナムル}}
: {{配音2|JP|[[天野由梨|天野由梨]]（第1作）／[[荒木香惠|荒木香惠]]（第2作）／[[大地葉|大地葉]]（第3作）|TW|[[謝佼娟|謝佼娟]]|HK|[[梁少霞|梁少霞]]、[[黃麗芳|黃麗芳]]（第1作第44話代配）（TVB）、[[王夢華|王夢華]]（有線）}}

: 星座：[[天蝎座|天蝎座]]

=== 友方 ===
==== {{nihongo|傑姆賈姆大陸|ジェムジャム大陸}} ====
故事中的第一座大陸，面積是故事中最小的一座，外形像隻圓圓的小怪物。往西為葛爾尼大陸，往南為皮古納皮納大陸，往西南為藏恩大陸，往東為吉蘭大陸。

:; {{nihongo|吉米納村|ジミナ村}}
:: 故事中的初心者之村，尼克與柯柯麗旅行的起點。大致上沒什麼特別的平凡村莊，但是盛產梅克梅克，其毛皮做成的斗篷是相當高級的魔法防具。

::; {{nihongo|巴多|バド}}
::: {{配音2|JP|[[增岡弘|增岡弘]]（第1作）／{{link-ja|西松和彥|西松和彦}}（第2作）／[[相馬康一|相馬康一]]（第3作）|HK|[[張錦江|張錦江]]（第1作、第3作）、[[梁志達|梁志達]]（第2作）}}
::: 尼克的父親，40歲，勇者狂熱者，小時為成為勇者刻苦修行，但當時魔王尚未復活。如今體力大不如前，於是將這份慾望轉嫁到尼克身上，強迫他從小接受修行，並讓他出任勇者一職。國王兩度徵招勇者的牌子都被他直接拔走。

::; {{nihongo|雷娜|レナ}}
::: {{配音2|JP|[[井上喜久子|井上喜久子]]（第1作）／{{link-ja|栗田日津留|栗田ひづる}}（第2作）／[[恒松步|恒松步]]（第3作）|HK|[[沈小蘭|沈小蘭]]}}
::: 尼克的母親，35歲，基於經濟與發展性等實際理由，支持巴多的勇者狂熱，強迫尼克進行修行。吹箭相當在行。

=== 咕嚕咕嚕 ===
米古米古族代代相傳神秘魔法，雖然表面上被歸類為黑暗魔法，但是其實既不屬於黑暗亦不屬於光明，而是由心產生，能夠體現心靈的心之魔法。隨著使用者的心態威力可大可小，能救世亦能滅世，還可以在不知不覺中影響現實，是相當高深可怕的魔法。

正因為害怕這份力量，以國泰國為首，世界各國陸續發出了驅逐米古米古族的命令，迫使米古米古族遠離塵世，最後遷居天界阿那斯達希亞。

咕嚕咕嚕的的精隨在於旋轉、情緒以及想像力。音樂和舞蹈有助魔法師旋轉並畫出魔法陣，情緒可以決定使出的魔法會是救贖或是災難，而想像力則使咕嚕咕嚕千變萬化，雖然有教典和基本魔法陣可依循，但其實也可以由魔法師親手創造只屬於自己的全新魔法。也因此情緒波動最強烈，想像力最豐富，最為活潑的兒少時期是咕嚕咕嚕魔法師最強的時期。

魔法陣是咕嚕咕嚕的表現手法之一，是將心裡所想的明確表達並加以實現的媒介。然而由心靈產生的咕嚕咕嚕就算沒有魔法陣，其魔力也會不自覺外洩並影響周遭，有時更會造成一發不可收拾的事態。

==== 基本要素 ====

:; {{nihongo|圓陣|円陣}}
:: 一個正圓，魔法陣的基本，無法徒手畫圓的人是無法使用魔法陣的。當然也有魔法陣不是圓形而是球型。

:: 和其他基礎符號不同，可以獨立當成魔法使用。是柯柯麗第一個成功畫出的魔法陣。

:; {{nihongo|潛入地下|地にもぐる}}
:: 基礎魔法符號之一。在圓陣中央畫一條縱向直線。是讓魔法潛入地面下的指令。

:; {{nihongo|召喚|召喚}}
:: 基礎魔法符號之一。在圓陣中央畫一顆眼睛的符號。所有召喚陣的基礎。

:; {{nihongo|貓|ネコ}}
:: 基礎魔法符號之一。在圓陣內側上緣畫兩個貓耳形狀的記號。和貓有關的魔法陣都會使用到。

:; {{nihongo|攻擊|攻撃}}
:: 基礎魔法符號之一。在圓陣上緣畫兩撇向外突出。咕嚕咕嚕少有的主動攻擊指令。

:; {{nihongo|回復|回復}}
:: 基礎魔法符號之一。在圓陣中畫一個逆時針向外的方形螺旋。可使站位於上方的生物回復體力、治療病痛。
:: 國泰國王城下藏著一個，國王私藏來治療痔瘡用的。

:; {{nihongo|星|星}}
:: 基礎魔法符號之一。在圓陣中畫一個星形。

:; {{nihongo|防禦|防禦}}
:: 基礎魔法符號之一。在圓陣外圍上緣左右兩側各畫一個T字型。防禦型魔法陣的關鍵指令。

:; {{nihongo|幻惑|幻惑}}
:: 基礎魔法符號之一。在圓陣中畫一個順時針向外的螺旋。加入這個指令可使全體看到幻覺。

:; {{nihongo|植物|植物}}
:: 基礎魔法符號之一。在圓陣中畫一個豆芽符號。與植物有關的魔法陣的基礎。

:; {{nihongo|假象|フェイク、こけおどし}}
:: 基礎魔法符號之一。在圓陣左右兩側各畫一個小正三角形。

:; {{nihongo|封印|封印}}
:: 基礎魔法符號之一。在圓陣外圍上緣左右兩側各畫一個F字型。封印型魔法陣的關鍵指令。

:; {{nihongo|禁錮陷阱|罠、固定、引きとめる}}

=== 2000年《咕嚕咕嚕魔法陣 心跳♡傳說》 ===
2000年4月4日至12月26日在[[東京電視網|東京電視網]]首播，全38話。動畫繼續由日本動畫公司製作，也是日本動畫公司的第一部電腦上色動畫。此外，副標題「心跳♡傳說」則是經由原作連載雜誌《[[月刊少年GANGAN|月刊少年GANGAN]]》開放給讀者命名和經由票選結果而決定。還有，獲選者曾經是已經解散的「衛藤浩幸公認粉絲團」會員。
