《'''MUNTO'''》系列是[[京都动画|京都动画]]制作的原創動畫系列作品。以2003年推出的OVA《'''MUNTO'''》為起點，2005年推出另一部OVA《'''MUNTO～穿越時空之壁'''》後。2009年春推出電視動畫《'''仰望天空的少女瞳中的世界'''》<ref>{{cite web|url=http://www.kyotoanimation.co.jp/works/|title=作品情報|work=京都動畫|accessdate=2015年2月26日|language=ja}}</ref>，在电视播放時進行調整，第一話至第六話前半沿用重新編輯的OVA舊作畫面，再加上部分新繪製畫面組合而成；而後半開始則是新的故事。[[声优|声优]]和前两作不盡相同，音乐和主題曲亦重新制作。2009年3月11日宣佈推出劇場版《天上人和アクト人》（天上人與亞克-{托}-人最後的戰鬥）<ref>{{cite web|url=http://news.scotsman.com/topics.cfm?tid=663&id=1863962006|title=《MUNTO》TV并未完结?完结篇剧场版4月上映|work=天极动漫|date=2009年3月11日|language=zh}}</ref>，4月11日上映<ref>[https://www.japanese-cinema-db.jp/Details?id=36908 天上人とアクト人最後の戦い ，テンジョウビトトアクトジンサイゴノタタカイ]</ref>。

== 故事簡介 ==
古時天界和下界有過一場圍繞萬能之力－亞克托的大規模戰爭，從此以後兩個世界被時空分隔開來。在現代的地上世界，名為日高夢見的少女能夠看見天上的浮島。某天，自稱為了解決天上的亞克托危機，以及挽回兩個世界的未來的少年——魔導王穆恩特在夢見面前出現了。他聲稱夢見有著拯救世界的關鍵力量。為了世界的未來，必須要夢見伸出援手……<ref name=a1>{{cite web|url=http://www.kyotoanimation.co.jp/works/muntoTV/|title=空を見上げる少女の瞳に映る世界|trans_title=仰望天空的少女瞳中的世界|publisher=京都動畫|accessdate=2015年2月26日|language=ja}}</ref>

== 登場人物 ==
=== 主角 ===
; {{nihongo|日高夢見|日高ユメミ|Hidaka Yumemi|聲：[[相泽舞|相泽舞]]}}
: 主角，「現代」時空的女孩子。溫柔文靜，性格有點愛幻想。喜歡雨天，因為「可以跟大家看見一樣的天空」。
:「命運之女」，擁有看見天上界魔導國的能力，更經常夢到該國國內發生的事。由於害怕自己的異能，所以平時出門會用粉紅色洋傘阻擋向天的視線。故事前期，視前來求助的穆恩特為幻覺，直到接受他的存在並相信自己的能力。
: 一度協助穆恩特解決天上危機。能夠單憑自己的身體越過時空之壁。與穆恩特接觸之際兩人的記憶產生了交流現象。與穆恩特第一次接觸後一直惦掛著天上世界的事，經常看著天空發呆。於第六話與穆恩特再次相會，跟隨他到魔導國，從穆恩特得知世界的真實和過去。雖然對回復亞克托循環一事感到迷惘，但在穆恩特的請求下下定決心。
: 擁有從產生大量亞克托的能力，被稱為亞克托的「震源」。為了拯救世界，在琉繪理的指導下接受心靈試煉，成功接納自己的內心，明白世界的本質，同時拯救以知子和涼芽。能夠創造出連琉繪理也看不見的未來。最終話成功連結世界後回到地上，但根據畫面可推測其仍然和穆恩特有往來。
: 女主角直稱穆恩特為「穆恩特」或「你」。夢見的瞳孔其實是魔導國的形狀。

; {{nihongo|穆恩特|ムント|Munto|聲：[[小野大輔|小野大輔]]}}
: 橘紅色短髮的少年，「未來」時空的天上界魔導國之皇。性格有點固執及驕傲，擁有極強的自尊心，同時很勇敢及強壯。對於被稱為「你」或「穆恩特」很反感，要求其他人稱自己為「穆恩特大人」。
: 「命運之子」，為了解決天上界危機而尋找命運之女。雖然身為王，卻經常站在戰場前線當主力，有時會失去冷靜而令自己陷入險況。攻擊力量以火為主。在部下和國民之間有不錯的聲望。外貌酷似父親，但力量似乎有所不及。大概是因為古時魔導國是天上的領導者，穆恩特認為自己身為魔導王有責任拯救天界。被聯合所排斥，卻與伽斯應有不淺的交情。瞳孔是魔導國的形狀。
: 與夢見接觸之際兩人的記憶產生交流現象。由於看見了夢見的記憶知曉她的脆弱，一直抗拒在亞克托危機後與她再會。第四話因亞克托循環一事被伽斯強行送往地上，同時得到能使用「局外者」部份能力，與夢見再次相會。一直鼓勵夢見要相信自己和自己的力量，並強調絕不能放棄。成功連結世界後失去蹤影。
:母親預見自己和先王的孩子會成為回復亞克托循環的關鍵，決定誕下穆恩特。因為先王和王妃把拯救世界的重責交給穆恩特，穆恩特背負著使命出生，從小就受盡苦頭，更被各國稱為被詛咒的孩子，但多年來一直堅持下去。即使知曉天界是罪惡之地，由搶奪來的異界心靈創造，亦無損他的決心。

== 專有名詞 ==
; 天上界
:處於「未來」時空的天上，穆恩特所在的世界，居住的天上人都是進化後的人類。以亞克托為主要能源。有著包括魔導國在內的七片大陸。
:太古時代中天上界並不存在，而是地上界第一群能使用亞克托的人創造並居住下來，又把身體變成亞克托的一部份，自稱「天上人」。其後天上人厭倦一切，天上界開始崩壞，同一時空的下界更試圖攻佔。最終，天上人切斷與地上的時空連結，當天被稱為「災難之日」。假如失去亞克托，天界會掉落至下界，同時會影響其他時空，造成兩者毀滅現代世界。
:最終話與「現代」時空重新連結起來。不過受古里多利的力量影響，天界再次出現扭曲。

;「現代」時空
:夢見所在的世界，距離亞克托被開發還有一段長時間。天上的戰爭似乎會影響到「現代」時空，不時會發生地震。沉睡著大量的亞克托。
:天上界物體的墜毀可以經下界對現代地上界造成傷害。第三話因魔導國墜毀而受影響，令所有人均能看見浮島。第五話以後因昆塔努捕獲夢見的計劃而令夢見所住的城鎮受到破壞，被拉扯至半空中。最終話與天上連結起來，並準備與天上人的各國領導人會面。

;下界
:「未來」的地上，不存在亞克托。若天上人或天上物質降落至此的話會不斷消耗自身的亞克托而無法維持形體，最終會消失，因此被天上人稱為「死之大地」。「未來」時空的人能在下界穿越時空，接觸其他時空的人。惟「災難之日」後，中間有了時空之壁的障礙。
:「未來」太古時代，天上人破壞了下界一切文明，存活下來的下界人一直等待反擊的機會，全盛時期甚至超越了天上人的力量。但「災難之日」，天上王切斷了兩個世界的連繫，連結世界的柱全掉落至下界，把下界一切毀滅。

;亞克托
:一種萬能、不可思議的能源。第七話說明亞克托是由「人的思念、願望、心靈」這種心靈波動轉化而成的能量。但天上人反被力量所迷惑，殺害並破壞人類文明以換取更多亞克托。隨後他們發現亞克托需要經另一個時空異界循環，於是變本加厲透過侵略其他時空，獲得無盡的亞克托。其後「災難之日」，下界的時空通道被切斷，令亞克托無法再生，直至第七話循環逐漸回復。
:「現代」雖沒有使用，但天上人發現其實沉睡著大量的亞克托。

;亞克托循環
:「未來」的太古時代，兩個世界會形成一個亞克托的循環：天上人的出生會消耗地上的亞克托，而天上生物的死亡與物質的崩壞全都會回歸成地上的亞克托。但其後被時空之壁隔開，循環被迫中斷。
:天上的人一直以為亞克托是無窮無盡，加上過度依賴，造成亞克托不足的情況。穆恩特的父母曾為了增加亞克托而自願除去，穆恩特也藉著破壞自己的領土以增加那些微的亞克托。

;亞克托危機
:天上界亞克托枯竭的危機。失去亞克托的天上界會逐漸崩落、消失。在「災難之日」後誕生，天上人用各種保護自己的手段，霸佔天上界僅存的亞克托能量。不過「局外者」的出現成為拯救天上界的阻礙，因為不干涉時空便無法回復亞克托的循環。第三話穆恩特和夢見的接觸令大量的亞克托流入天上，解決了危機。

;亞克托之柱
:由高純度的亞克托結晶化後形成的柱，支撐著天上的浮島。用來平衡天上人的生活，有著重要的地位。戸部神社的巨石應該就是當時的支柱的一部份。

;時空之壁
:隔開「未來」和其他時空的「壁」。天上人有着穿越時空之壁的技術，但必須動用不少裝置。
:以往曾因天上人干涉過去的地上而發生災難(「災難之日」)，所以穿越時空之壁被視為禁忌。遇上干涉其他時空的人，特定的時空監察者和其後裔會阻止。以河流比喻的話，天上是處於上流，而下界則是下流，而時空之壁是堤壩。
:穿越時空之壁所承受的壓力相當於一個行星的重力。因為那極大的壓力，所以能不依靠任何裝備而且還能「逆流而上」的夢見的力量顯得異常珍貴。在第六話時空之壁被穆恩特和夢見打破，外觀變成粉色的固態物質，並有螺旋狀、稱之為「絆」的光柱直穿且連結至現代的地上。穆恩特的目的是令光柱實體化並令它穏定下來，以恢復亞克托的循環。粉色的固態物質從第七話開始逐漸崩壞，「絆」也非常不穩定，最終話裏更一度消失。不過後來由夢見建立了更鞏固的連結。

;魔導國
:穆恩特的國家。因為是衆國的眼中釘，曾因被聯合軍攻擊支撐著魔導國的柱(「亞克托之柱」)而差點掉落至下界。過去曾處於支配天上各國的地位。
:因為亞克托發展純熟，而出現了魔力戰士，穆恩特和路伊也是一份子。幾乎完全依靠亞克托發動魔力攻擊，所以在亞克托枯竭時難以防禦他國，反之天上充滿亞克托後實力大增。由於發動魔力需要大量亞克托，使魔導國特別被其它他國針對。

;聯合
:天上各國的聯合組織。過去是跟隨魔導國的組織，但穆恩特成為魔導王後便不再跟隨魔導國。為了管理有限的亞克托，由六個國家的長老作為代表，決定組織的行動。主要目標是嚴重浪費亞克托的魔導國。曾攻擊魔導國的亞克托之柱，企圖令它落至下界。
:第四話起因不再缺乏亞克托，使四個國家脫離了聯合。

;局外者
:不干涉天上戰爭的人，部分名為時空監察者。被賦予高於天上任何人的強大力量以維持秩序(防止任何人汗涉時空和過去)。倘若違反規定作出通融等行動，身體上會出現代表詛咒的深棕色圖案，多次違規會喪失能力而死。
