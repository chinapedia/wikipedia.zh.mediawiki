'''《仰望星空》'''（{{lang-en|'''"Looking up to the Starry Sky"'''}}）是时任[[中共中央政治局常委|中共中央政治局常委]]、[[中华人民共和国|中华人民共和国]][[中华人民共和国国务院_(第十届)|国务院]][[中华人民共和国国务院总理|总理]][[温家宝|温家宝]]所作的一首[[诗|诗]]，后被改编成[[歌曲|歌曲]]。全诗共四节，表达了对国家、民族乃至人类命运的关怀。<ref>[http://cpc.people.com.cn/GB/64093/64094/6372042.html 温家宝诗作《仰望星空》被谱成励志歌曲传唱]].人民网，2007年10月12日</ref>

2007年5月14日，温家宝在[[同济大学建筑与城市规划学院|同济大学建筑与城市规划学院]]钟厅作演讲时首次在公众场合发表<ref>[https://news-caup.tongji.edu.cn/topic.php?tid=16&id=2587 温家宝总理“仰望星空”] 同济大学建筑与城市规划学院新闻中心 2007年5月14日</ref>。2007年9月4日，该诗作在《[[人民日报|人民日报]]》文艺副刊上发表。
== 歌词 ==
2008年于[[江西省|江西省]][[武宁县|武宁县]]首映的电影《踏界》（[[陆剑民|陆剑民]]、[[石维坚|石维坚]]主演）采取《仰望星空》作为电影主题曲的歌词，[[中国煤矿文工团|中国煤矿文工团]]歌唱演员[[胡月|胡月]]谱曲。<ref>[http://www.forestry.gov.cn/portal/main/s/72/content-203377.html 我国首部林改题材电影《踏界》即将公映] {{webarchive|url=https://archive.is/20120803013124/http://www.forestry.gov.cn/portal/main/s/72/content-203377.html |date=2012-08-03 }}中国绿色时报 2008年6月26日</ref>

== 演唱 ==
2009年10月2日，香港[[红磡体育馆|红磡体育馆]]庆祝[[中华人民共和国|中华人民共和国]]建立60周年晚会上，著名歌手[[张学友|张学友]]、儿童歌手[[杨沛宜|杨沛宜]]与500名儿童共同演绎了根据《仰望星空》配乐的同名歌曲。<ref>[http://news.sohu.com/20090925/n266988283.shtml 香港国庆文艺晚会节目曝光三天王同台献艺(图)]搜狐新闻 2009年9月25日</ref>

== 校歌 ==
2010年5月13日，[[北京航空航天大学|北京航空航天大学]]宣布《仰望星空》成为该校的校歌，歌词为温家宝的诗作《仰望星空》，[[刘晖|刘晖]]作曲。

== 高考作文题 ==
2010年[[普通高等学校招生全国统一考试|普通高等学校招生考试]]北京卷作文题：《仰望星空与脚踏实地》。<ref>[http://edu.qq.com/a/20100607/000205.htm 2010年北京高考作文题：仰望星空与脚踏实地]腾讯教育 2010年06月07日</ref>
