'''川上眉山'''（{{bd| 1869年|3月5日|1908年|6月15日}}），出生于[[大日本帝国|大日本帝国]][[大阪|大阪]]，日本近代知名[[作家|作家]]。<ref name="xms " >{{cite web |url=http://www.jnocnews.jp/news/show.aspx?id=28291 |title= 因贫困而轻生的文人——川上眉山|accessdate= 2009年|work=|publisher=日本新华侨报网 |language=zh-hans}}</ref>

== 生平 ==
川上眉山出生时，家庭已经被父亲欠了巨额债务。<ref name="xms " /> 他成长于日本[[明治维新|明治维新]]晚期，社会矛盾日益突然，文学上处于[[浪漫主义|浪漫主义]]向[[自然主义|自然主义]]的过渡期，当时煊赫不已的[[自然主义|自然主义]]席卷全国，[[浪漫主义|浪漫主义]]从此一蹶不振，导致了他的作品卖不出去，造成了严重的经济问题。<ref name="xms " /> 1908年6月16日，他在自己的书房用剃刀切断[[颈动脉|颈动脉]]而死。<ref name="xms " />

== 评价 ==
[[日本|日本]]《[[国民新闻|国民新闻]]》发的讣告评价他：“以艳丽的笔触、清新的思想驰名文坛的小说家川上眉山氏于15日晨4时30分左右，在[[东京|东京]][[牛込区|牛込区]][[天神町|天神町]]67号寓所，自杀身亡。”<ref name="xms " />

[[宫岛乔|宫岛乔]]：“《[[自杀论|自杀论]]》的作者在列举自杀‘社会性’因素时，对贫穷、经济危机（失业、破产等）、病苦等因素，几乎没有给予注意。” <ref name="xms " />

[[大原健士郎|大原健士郎]]：“从社会学角度看，他经济拮据而不得不搬家，这些原因错综复杂，已经形成了他的自杀准备状态，使他下自杀决心的可以看做是因为他社会作用意识的丧失，……自杀的导火索就是搬家。”<ref name="xms " />
