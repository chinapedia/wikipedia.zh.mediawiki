'''川蔡通話'''是指[[2016年|2016年]][[12月2日|12月2日]]，[[中華民國總統|中華民國總統]][[蔡英文|蔡英文]]自[[臺灣|臺灣]]致電[[美國總統|美國總統]]當選人[[唐納·川普|唐納·川普]]，祝賀其當選新任總統。通話歷時10多分鐘，雙方針對[[政治|政治]]、[[經濟|經濟]]、[[安全|安全]]及[[亞洲|亞洲]]印太區域情勢交換意見，川普並多次於網站及[[推特|推特]]發表看法。這是史上首次美國候任總統與中華民國總統直接進行通話，在國際上引發熱議，[[華文|華文]]傳媒亦見「川[[菜|菜]]夠[[辣|辣]]」等討論<ref name = "接台灣賀電很應該">{{Cite news |url = http://www.appledaily.com.tw/realtimenews/article/new/20161203/1003503/ |title = 川普再推文：美國對台軍售　接台灣賀電很應該 |author = 施旖婕 |date = 2016年12月3日 |location = 台灣 |publisher = 《[[蘋果日報_(台灣)|蘋果日報]]》 |language = zh-hant |accessdate = 2016年12月4日 |quote = |archiveurl = https://web.archive.org/web/20161205171618/http://www.appledaily.com.tw/realtimenews/article/new/20161203/1003503/ |archivedate = 2016年12月5日 }}</ref><ref name="Thediplomat1">{{Cite news |url = http://thediplomat.com/2016/12/the-tsai-trump-call-the-dynamics-in-taiwan/ |title = The Tsai-Trump Call: The Dynamics in Taiwan |date = 2016年12月16日 |publisher = The Diplomat |language = EN |accessdate = 2016年12月16日 |quote = With much of Western media taking the lead in presuming to interpret Beijing's ire at news of the unprecedented congratulatory call from Tsai, ......as reckless and dangerous — hence media references to Trump’s “weekend antics,” describing him as a “diplomatic rookie.” |deadurl = no |archiveurl = https://web.archive.org/web/20161213021503/http://thediplomat.com/2016/12/the-tsai-trump-call-the-dynamics-in-taiwan/ |archivedate = 2016年12月13日 }}</ref>

[[美國在台協會|美國在台協會]]台北辦事處前處長[[司徒文|司徒文]]表示，這應該不是台灣單方面的行動，他認為這是很正面的事情，而美國早該這麼做了。<ref name="司徒文">[http://news.ltn.com.tw/news/politics/breakingnews/1907492 司徒文：美國過去「一個中國」政策是錯的] {{webarchive|url=https://web.archive.org/web/20161207141855/http://news.ltn.com.tw/news/politics/breakingnews/1907492 |date=2016-12-07 }}, [[自由時報|自由時報]], 2016-12-05</ref>替臺灣遊說的前[[荷蘭|荷蘭]]外交官[[韋傑理|韋傑理]]（Gerrit van der Wees）則認為，該通電話顯示川普會較不受外交政策慣例和限制束縛，並且發出美國對臺灣政策廣泛改變的訊號<ref name = "台灣的總統"/>。

美國[[哥倫比亞大學|哥倫比亞大學]]中國項目聯席主任[[孫哲_(學者)|孫哲]]認為，此次事件時間點敏感，認為臺灣可能經過指導，而蔡英文幕僚善於美國工作，能掌握機會加強公關遊說力道<ref name = "批蔡格局小"/>。

[[亞太和平研究基金會|亞太和平研究基金會]]顧問[[趙春山|趙春山]]說，此次事件引發美國國內不少爭議，臺灣不應介入中美強權鬥爭、也不應介入美國內政，否則這是最「stupid」（愚蠢）的行為<ref>{{Cite web|url=http://www.chinatimes.com/newspapers/20161220000374-260118|author=呂雪彗、周毓翔|title=川-{}-普勝選後捨TPP 包道格：推雙邊協定 對台更有利|publisher=中國時報|date=2016-12-20|accessdate=2017-02-11|language=zh-tw|deadurl=no|archiveurl=https://web.archive.org/web/20170211160512/http://www.chinatimes.com/newspapers/20161220000374-260118|archivedate=2017-02-11}}</ref>。

=== 中美關係 ===
{{see also|中美關係}}
[[File:Newt_Gingrich_by_Gage_Skidmore_3.jpg|thumb]]議長[[紐特·金瑞契|金瑞契]]表示川蔡通話向世界傳達「中國將不能再恐嚇我們」]]

川普重要顧問、前[[美國眾議院|美國眾議院]]議長[[紐特·金瑞契|紐特·金瑞契]]表示美國不得不被中華人民共和國指揮這條潛規則是[[美國國務院|國務院]]怯懦的表現，「不知何故，國務院覺得跟這個星球上的任何獨裁者談話都可以，卻唯獨不能跟一個民主社會的民選領袖談話...這是非常危險的。」川蔡通話向全世界傳達了一個良好信號，即川普將做自己的主人，「中國將不能再恐嚇我們」，這是重整美中關係的第一步。<ref name="大紀元"/><ref name="POLITICO"/><ref name="Fox News"/>

前[[美國駐聯合國代表|美國駐聯合國代表]][[約翰·波頓|約翰·波頓]]強調只要符合美國利益，美國總統有權決定和誰接觸，北京當局無權置喙，他並表示美、中關係「確實該動搖一下」，因為中方近年意圖掌控日、韓、台等國經濟，都對美國造成衝擊。<ref name="波頓評論"/>川普的經濟政策顧問摩爾抨擊中華人民共和國在東方是劍拔弩張、張牙舞爪的惡霸，現在是站起來反抗的時候了。「如果中國對川蔡通話不爽，管他的！」他批評美國政界太多軟弱者不敢得罪中華人民共和國，但他一點也不介意得罪該國人。<ref name="顧問摩爾"/>

許多政治專家則分析川普對於中華人民共和國的手段將轉向強硬，也不再依照慣例、而可能有多種方向<ref name = "彭斯料成美外交要角"/>。保守派報紙《[[華盛頓時報|華盛頓時報]]》編輯{{Link-en|莫妮卡·克勞里|Monica Crowley}}指出，川普是想要改變既有現狀的美國總統，其中美國存在「一個中國」政策、而臺灣則是民主國家，他選擇堅持民主價值<ref name = "彭斯料成美外交要角"/>。前[[美國在台協會|美國在台協會]]台北辦事處處長[[包道格|包道格]]認為通話舉動，是為了平息川普政府的各種猜測，包括可能在臺灣地位議題向中華人民共和國進一步讓步等傳聞<ref name = "張詠晴"/>。另一位前處長[[司徒文|司徒文]]則表示，美國過去「一個中國」的政策是錯的，他強調美國從來都沒說過中國對台灣擁有主權。<ref name="司徒文"/>[[哈德遜研究所|哈德遜研究所]]研究員與前[[美國海軍部|海軍部]]副次長{{link-en|Seth Cropsey|Seth Cropsey|Seth Cropsey}}則直指所謂「民主臺灣是極權中華人民共和國之部分」一說不過就像「[[國王的新衣|國王的新衣]]」。<ref>{{link-en|Seth Cropsey|Seth Cropsey|Seth Cropsey}}, [http://www.realcleardefense.com/articles/2016/12/07/doing_the_right_thing_the_taiwan_phone_call_110451.html Doing the Right Thing: The Taiwan Phone Call] {{webarchive|url=https://web.archive.org/web/20161209173739/http://www.realcleardefense.com/articles/2016/12/07/doing_the_right_thing_the_taiwan_phone_call_110451.html |date=2016-12-09 }}, 2016-12-7</ref>

[[中國人民大學|中國人民大學]]國際關係學者[[時殷弘|時殷弘]]則表示，川普越來越關注對於中華人民共和國的政策<ref name = "美聯社"/>。而前美國國家安全會議亞洲事務資深主任[[麥艾文|麥艾文]]（Evan Medeiros）認為，無論通話舉動是否經川普團隊謹慎考慮，中華人民共和國將認為川普具有負面戰略意圖，將深遠影響中美關係互信和競爭策略<ref name = "張詠晴"/>。

[[淡江大學|淡江大學]]國際研究學院院長[[王高成|王高成]]認為中華人民共和國的回應仍展現自信，同時警告川普上任後不會影響美國長期政策，並認定中華民國與美國仍是非官方關係<ref name = "陳培煌"/>。他並認為川普若將與臺灣的關係朝向官方化發展，甚至大幅提升對臺軍售，中華人民共和國一定會有反制與恫嚇作為<ref name = "陳培煌"/>。
