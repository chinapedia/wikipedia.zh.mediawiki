'''川上稔'''（{{bd|1975年|1月3日|catIdx=Kawakami Minoru}}）是[[日本|日本]][[輕小說|輕小說]][[作家|作家]]和[[遊戲開發者|遊戲開發者]]，出身於[[東京都|東京都]]。

== 人物 ==
是業界少數幾個會畫圖的小說家。

== 來歷 ==
[[城西大學|城西大學]]經濟學系畢業。目前隸屬於遊戲製作公司[[Tenky|Tenky]]。遊戲的代表作為「{{lang|ja|[[メリーメント・キャリング・キャラバン|メリーメント・キャリング・キャラバン]]}}」、「{{lang|ja|[[ソウ楽都市OSAKA|奏（騒）楽都市OSAKA]]}}」、「{{lang|ja|[[Twelve〜戦国封神伝〜|Twelve〜戦国封神伝〜]]}}」。1996年時以《{{lang|ja|[[パンツァーポリス1935|パンツァーポリス1935]]}}》拿下[[電擊遊戲小說大獎|電擊遊戲小說大獎]]（現名為[[電擊小說大獎|電擊小說大獎]]）出道後，也兼職參與輕小說寫作活動。

== 作風 ==
其作品'''終焉的年代'''、'''境界線上的地平線'''均以單本厚度與驚人頁數著稱。
