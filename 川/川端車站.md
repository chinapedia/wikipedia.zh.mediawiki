'''川端車站'''（{{jpn|j=川端駅|hg=かわばたえき|rm=Kawabata eki|f=y}}）位於[[日本|日本]][[北海道|北海道]][[夕張郡|夕張郡]][[由仁町|由仁町]]川端，是[[北海道旅客鐵道|北海道旅客鐵道]]（JR北海道）[[石勝線|石勝線]]沿線的車站，只停靠普通列車，目前每天只提供4班往{{ST|夕張|1}}的下行班次與6班往[[追分站_(北海道)|{{ST]]及{{ST|千歲|1}}的上行班次。

== 車站構造 ==
[[相對式月台|相對式月台]]和[[島式月台|島式月台]]，擁有2面3線的車站。無人車站，由[[追分車站_(北海道)|追分車站]]管理。

== 車站周邊 ==
*[[國道274號|國道274號]]、[[北海道道462號川端追分線|北海道道462號川端追分線]]
*{{Link-ja|栗山警察署|栗山警察署}}川端派出所
*川端郵局
*Unitopia川端公園高爾夫球場（{{lang|ja|ユニトピア川端パークゴルフ場}}）
*川端元氣村（{{lang|ja|かわばた元気村}}）

== 巴士路線 ==
現在車站周邊都沒有巴士路線停靠。

而在1990年代，{{Link-ja|北海道中央巴士|北海道中央バス}}曾營運由川端車站前往{{Link-ja|三川站 (北海道)|三川駅 (北海道)|三川}}、{{Link-ja|栗山站|栗山駅|栗山}}及岩見澤的巴士路線。

== 历史 ==
*1894年（[[明治|明治]]27年）8月1日 - {{link-ja|北海道煤礦鐵道|北海道炭礦鉄道}}追分～紅葉山（今[[新夕張車站|新夕張]]）通車時設立。
*1900年（明治33年）1月16日 - 車站往追分方向移動400米。
*1906年（明治39年）10月1日 - 北海道炭礦鐵道國有化後成為官設鐵道。
*1981年（[[昭和|昭和]]56年）5月25日 - 廢除貨運業務，降為無人車站。
*1987年（昭和62年）4月1日 - [[國鐵分割民營化|國鐵分割民營化]]後成為JR北海道轄下的車站。
