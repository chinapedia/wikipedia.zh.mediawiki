'''川原礫'''（{{bd|1974年|8月17日||}}，{{lang|ja|かわはら れき}}），[[日本|日本]]的[[輕小說|輕小說]]作家。[[青山學院大學|青山學院大學]]畢業{{efn|在SAOAⅡ第22回「本星期的川原先生」中，將同一大學畢業的武富健治介紹為學長。}}。出身於[[群馬縣|群馬縣]][[高崎市|高崎市]]，目前居住在[[東京都|東京都]][[練馬區|練馬區]]。

== 略歷 ==
2002年開始以{{nihongo|「九里史生」|くのり ふみお}}的筆名發表網路小說。特別是2002年11月到2008年7月在個人網站連載的《{{nihongo|[[刀劍神域|刀劍神域]]|ソードアート・オンライン}}》系列最受歡迎。2007年在小說投稿網站Arcadia<ref>http://www.mai-net.net/bbs/sst/sst.php?act=list&cate=all&page=1 Arcadia</ref>上以{{nihongo|「攻打引」|せめだいん}}為筆名發表了《{{nihongo|超絕加速Burst Linker|超絶加速バーストリンカー}}》。
 
2008年，由《超絕加速Burst Linker》改名的《{{nihongo|[[加速世界|加速世界]]|アクセル・ワールド}}》獲得第15屆[[電擊小說大獎|電擊小說大獎]]，2009年2月以該作品正式出道。同年4月開始發售文庫版的《刀劍神域》。

2011年10月，《加速世界》與《刀劍神域》做為[[ASCII_Media_Works|ASCII Media Works]]創立20周年的紀念作品進行動畫化和遊戲化，成為兩部作品同時動畫化的特例。川原礫並且在電視版《加速世界》動畫中擔任對戰虛擬角色{{nihongo|馬口鐵作家|ブリキ・ライター}}的配音，在對戰中為觀眾解說戰況。

2012年12月，著作的累積發行量突破1000萬部。從2009年2月出道開始，僅3年11個月即達成1000萬部的發行量，是電擊文庫的作家中最快的紀錄。

2013年，以《刀劍神域》獲得第12回[[東京動畫獎|東京動畫獎]]原作賞。

2014年6月10日開始發行文庫版小說《{{nihongo|[[絕對的孤獨者|絕對的孤獨者]]|絶対ナル孤独者}}》，原為2004年起在個人網站上發表的網路小說《{{nihongo|絕對的孤獨|絶対ナル孤独}}》。

目前出書進度維持在每年5~6本左右{{efn|2015年4月因為要進行刀劍神域2動畫初回特典小說的作業而暫停出書一次<ref>{{Cite web |url=https://twitter.com/kunori/status/565107098023763968 |title=また、お知らせというかお詫びですが、特典小説の作業がある関係で、4月の新刊はありません。 |author=川原礫 |accessdate=2015-07-17 |date=2015-02-10 |work=[[Twitter|Twitter]] |language=ja}}</ref>。}}。
