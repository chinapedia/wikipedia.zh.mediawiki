'''蔡玉坤'''{{MH}}（{{lang|en|'''Choi Yuk-kwan, Tony'''}}，{{bd|1968年|4月9日|catIdx=C蔡}}），[[香港|香港]][[壁球|壁球]]代表隊總[[教練|教練]]<ref>{{Cite web |url = http://sportsroad.hk/archives/22434 |title = 香港壁球隊總教練蔡玉坤回應呂碩言論 |date = 2014-02-14 |publisher = 《體路》（Sportroad） |language = zh-hant |accessdate = 2016-07-16 |quote = ……的意見，香港壁球總會總教練蔡玉坤於今日（12日）傍晚…… }}</ref>，[[香港體育學院|香港體育學院]]壁球總教練<ref>{{Cite web |url = http://www.ltfc.edu.hk/website/index.php/campus-news/952-hksi-and-lam-tai-fai-college-collaborate-on-a-tailor-made-secondary-curriculum-for-full-time-athletes |title = 體院與林大輝中學合作 設全職運動員高中課程 |date = 2015 |publisher = [[林大輝中學|林大輝中學]]網頁 |language = zh-hant |accessdate = 2016-07-16 |quote = ……鍾伯光教授、體院壁球總教練 蔡玉坤先生MH及壁球運動員林溢庭共同見證。 }}</ref>。

他由1999年起擔任此職<ref>{{cite news|title=蔡玉坤掏腰包請飲茶|url=http://www.takungpao.com/sport/top/2011-11-19/1025149.html|newspaper=《大公報》|date=2011-11-19|deadurl=yes|archiveurl=https://web.archive.org/web/20111122063147/http://www.takungpao.com/sport/top/2011-11-19/1025149.html|archivedate=2011-11-22}}</ref>，任內培育出[[歐詠芝|歐詠芝]]及[[陳浩鈴|陳浩鈴]]等優秀球手，又帶領港隊在[[2009年東亞運動會壁球比賽|2009年東亞運動會壁球比賽]]奪得7金5銀佳績<ref>{{cite news|title=教練蔡玉坤：中國香港壁球包攬７金「創造傳奇」|url=http://sports.sina.com.hk/cgi-bin/nw/show.cgi/1431/5/1/89154/1.html|newspaper=新華網|date=2009-12-13|deadurl=yes|archiveurl=https://web.archive.org/web/20100402203344/http://sports.sina.com.hk/cgi-bin/nw/show.cgi/1431/5/1/89154/1.html|archivedate=2010-04-02}}</ref>。2010年，獲[[香港特區政府|香港特區政府]]授予[[榮譽勳章_(香港)|榮譽勳章]]<ref>{{cite news|title=港協暨奧委會向優秀教練運動員頒獎|url=http://www.takungpao.com.hk/news/10/12/03/WF-1328752.htm|3=|newspaper=《大公報》|date=2010-12-03}}{{dead link|date=十月 2017 |bot=InternetArchiveBot }}</ref>。其妻為藝人[[劉彩玉|劉彩玉]]。

2016年12月1日，蔡玉坤榮升[[香港體育學院|香港體育學院]]副院長。
