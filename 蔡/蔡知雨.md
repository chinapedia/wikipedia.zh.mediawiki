蔡知雨，2004年7月11日出生于中国北京，中国内地少年偶像组合[[H2K|H2K]]成员之一。曾受邀参加央视综艺节目《嗨！2014》<ref name=":0">{{Cite web|url = http://www.hunantv.com/c/20140528/1002361718.html|title = 哈林拜师9岁天才吉他手 《嗨!2014》开启六一节专场|accessdate = 2016年1月6日|author = |date = |publisher = }}</ref>、湖南金鹰卡通频道《中国新声代第二季》等。

== 个人经历 ==
2014年5月18日参加吉他中国举办的全国吉他大赛并获得北京地区冠军<ref>{{Cite web|title = 第9届贝塔斯瑞乐队大赛北京赛区 成功上演创记录}}</ref>

2014年5月29日应邀参加央视综艺节目《嗨！2014》<ref name=":0" />

2014年7月19日《中国新声代第二季》第八期“集体男生：太青春”南瓜班（尚雯婕组），演唱原创歌曲《我的天》

2015年1月15日[[天娱传媒|天娱传媒]]旗下小鲜肉组合H2K（成员：蔡知雨、张汉盛、黄星羱、封迪、罗维）于北京正式宣布成立并发布首支出道单曲《骑士少年》<ref>{{Cite web|url = http://ent.163.com/15/0115/08/AG053KFK00031H0O.html|title = 天娱推出00后小鲜肉组合H2K 出道单曲首播|accessdate = 2016年1月6日|author = |date = |publisher = }}</ref>

2015年3月28日H2K新单曲《ROCO Time》首发，该单曲成为腾讯游戏《洛克王国》主题曲，并力邀曾和赵薇、周迅、萧亚轩、张韶涵都有过合作的音乐人闻震担纲制作人。于QQ音乐首发。<ref name=":1">{{Cite web|url = http://ent.qq.com/a/20150330/037635.htm|title = 天娱鲜肉团H2K新单《ROCO Time》传递正能量|accessdate = 2016年1月6日|author = |date = |publisher = }}</ref>
