'''蔡奇'''（{{bd|1955年|12月5日|||}}），[[福建|福建]]省[[三明|三明]]市[[尤溪|尤溪]]县人。1973年3月参加工作，1975年8月加入[[中国共产党|中国共产党]]。[[福建师范大学|福建师范大学]]政教系毕业，经济法律学院硕士，政治经济学博士。曾任[[中共浙江省委|中共浙江省委]]常委、省委组织部长、浙江省常务副省长，[[杭州市|杭州市]][[市长|市长]]<ref name="任命">{{cite web|url=http://news.sohu.com/20131122/n390610746.shtml | language= | title=浙江省人大常委会任命蔡奇为省人民政府副省长 |publisher=浙江在线 |date=2013年11月22日 |accessdate= }}</ref>，[[中央国家安全委员会|中央国家安全委员会]]辦公室副主任、[[北京市人民政府|北京市人民政府]][[北京市市长列表|市长]]。现任第十九届[[中共中央政治局|中共中央政治局]]委员、[[中国共产党北京市委员会|中共北京市委]]书记。蔡奇被视为[[习近平派系|习近平派系]]的重要一员，也是[[改革开放|改革开放]]以后极少数从中央委员会之外直接进入政治局的干部<ref>{{cite web|url=https://hk.news.appledaily.com/international/daily/article/20171120/20219637|title=習心腹蔡奇管北京　十九大三級跳|publisher=[[蘋果日報_(香港)|蘋果日報 (香港)]]|accessdate=2017-11-20}}</ref>。

== 早年经历 ==
1955年12月，蔡奇在[[福建|福建]][[永安|永安]]出生<ref>[http://www.china.com.cn/top/2016-11/01/content_39610831.htm 北京代理市长蔡奇被称大V网红 曾拥千万级粉丝]</ref>。1973年3月，17岁的蔡奇[[插队|插队]]劳动，在[[福建省|福建省]][[永安市|永安县]]西洋公社当[[知识青年|知青]]。1975年获得推荐，进入[[福建师范大学|福建师范大学]]政教系学习。1978年毕业后留校，在校党委办公室担任党工干部。1983年，调入[[中国共产党福建省委员会|中共福建省委]]办公厅工作，开启从政之路。

== 仕途 ==
===福建经历===
1983年，蔡奇任[[中国共产党福建省委员会|中共福建省委]]办公厅综合处副处长，1987年任正处级秘书，1991年任福建省委政改办公室副主任（副厅级）、党建办公室副主任。1993年到1996年任福建省委办公厅副主任，长期担任时任中共福建省委书记[[陈光毅|陈光毅]]的秘书。

1994年3月，蔡奇获得外放机会，挂职担任中共[[三明市|三明市]]委副书记。1994年到1997年，在[[福建师范大学|福建师范大学]]经济法律学院硕士研究生班学习。1996年9月，正式出任中共三明市委副书记。1997年11月，任三明市人民政府市长。

===浙江经历===
1999年5月，蔡奇调到邻省[[浙江省|浙江]]工作，任中共[[衢州市|衢州市]]委副书记、市长。2002年3月，任衢州市委书记、市人大常委会主任。2004年4月，任中共[[台州市|台州市]]委书记，在时任中共浙江省委书记[[习近平|习近平]]领导下工作。

2007年4月，蔡奇任中共[[杭州市|杭州市]]委副书记、市长，晋升副部级。2010年1月，升任[[中国共产党浙江省委员会|中共浙江省委]]常委、省委组织部长、省委党校校长。2013年11月，任中共浙江省常委、[[浙江省人民政府|浙江省人民政府]]常务副省长、党组副书记<ref name="任命"/><ref name="免职"/>。

===从国安委到北京===
2014年3月27日，在任职四个月后，蔡奇被免去浙江省人民政府副省长职务<ref name="免职">{{cite web|url=http://news.sina.com.cn/c/2014-03-27/104429804444.shtml | language= | title=浙江省人大常委会免去蔡奇副省长职务 |publisher=新华网浙江频道 |date=2014年3月27日 |accessdate= }}</ref>，调任新成立的[[中央国家安全委员会|中央国家安全委员会]]办公室副主任<ref>{{cite web|url=http://news.takungpao.com/mainland/focus/2014-03/2385800.html|title=港媒：大V官员蔡奇料入国安办 出任专职副主任|publisher=大公报|accessdate=2014-03-28}}</ref>，2015年任中央国家安全委员会办公室常务副主任（正部长级）。

2016年10月31日，蔡奇任[[中国共产党北京市委员会|中共北京市委]]副书记书记<ref>{{Cite web|url=http://news.ifeng.com/a/20161101/50184353_0.shtml|title=中央批准蔡奇、王安顺同志职务调整|accessdate=2016-11-16|work=凤凰网}}</ref>，第十四届北京市人民代表大会常务委员会决定，任命蔡奇为[[北京市人民政府|北京市人民政府]]副市长、代市长<ref>{{Cite news|url=http://www.voachinese.com/a/beijing-major-20161030/3571780.html|title=习近平闽浙亲信将任北京市长仕途看涨|last=海彦|newspaper=美国之音|accessdate=2016-10-31}}</ref>。2016年11月，兼任[[2022年冬季奥林匹克运动会|北京2022年冬奥会]]和[[2022年冬季残疾人奥林匹克运动会|冬残奥会]]组织委员会执行主席<ref>{{cite web|url=http://www.thepaper.cn/newsDetail_forward_1565017|title=国务院任免国家工作人员：苟仲文任国家体育总局局长|publisher=澎湃新闻|accessdate=2016-11-20}}</ref>。2017年1月20日，當選北京市人民政府市長<ref>{{cite news|title=蔡奇当选北京市长，张硕辅当选北京市监察委主任|url=http://www.thepaper.cn/newsDetail_forward_1604995|accessdate=2017-02-05|work=澎湃新闻}}</ref>。

2017年5月27日，蔡奇接替[[郭金龙|郭金龙]]，任[[中国共产党北京市委员会|中共北京市委]]書記、北京2022年冬奥会和冬残奥会组织委员会主席、党组书记，同时辞去北京市人民政府市长职务<ref>{{cite web |url= http://news.youth.cn/jsxw/201705/t20170527_9897156.htm|title=蔡奇任北京市委书记 郭金龙不再兼任 |first= |last= |work=新华网|accessdate=2017-05-27}}</ref>。2017年6月，蔡奇在公开场合开始使用“[[习近平新时代中国特色社会主义思想|习近平总书记重要思想]]”的提法<ref>{{cite news|author1=于泽远|title=北京市委书记蔡奇提用习“重要思想”举旗引路|url=http://www.zaobao.com/znews/greater-china/story20170809-785674|accessdate=2017-10-01|agency=联合早报网|date=2017-08-09}}</ref>。

值得一提的是，他既不是中共十八届中央委员，也不是中央候补委员，而作为北京市委书记在2017年10月举行的[[中国共产党第十九次全国代表大会|中共十九大]]上直接成为[[中国共产党中央政治局|中共中央政治局]]委员，这样的晋升速度在中共政坛非常罕见。他是中共首位非中央委员的[[北京|北京]]市委书记<ref >{{cite web|url=http://china.dwnews.com/news/2017-05-27/59817358.html | language= | title=京畿重地双易主 政坛新星集体崛起 |publisher=多维新闻 |date=2017-05-27 |accessdate= }}</ref><ref >{{cite web|url=http://china.dwnews.com/news/2017-05-27/59817259.html | language= | title=蔡奇主政北京 罕见的入局“三级跳”？ |publisher=多维新闻 |date=2017-05-27 |accessdate= }}</ref>〔时任[[中国共产党天津市委员会|中共天津市委]]书记的[[谭绍文|谭绍文]]（1989年至1992年）及时任[[中国共产党重庆市委员会|中共重庆市委]]书记的[[张德邻|张德邻]]（1997年6月至9月）也不是中央委员或中央候补委员）〕。

2017年10月，蔡奇在[[中国共产党第十九次全国代表大会|中共十九大]]当选[[中国共产党第十九届中央委员会委员列表|第十九届中央委员会委员]]，并于十九届一中全会上当选[[中国共产党中央政治局|中共中央政治局]]委员，成为[[党和国家领导人|党和国家领导人]]。

== 主政北京 ==
=== 清理北京外来人口 ===
{{seealso|北京低端人口}}
蔡奇在就任北京市委书记后召开的首次北京城市总体规划实施动员和部署大会上提出，由于北京是个严重缺水城市，这决定了必须控制人口规模，这是自己主政北京期间首当其冲的硬任务。总的要求是中心城区人口规模减少，中心城区以外平原地区人口规模有增有减，山区保持人口规模基本稳定。到2020年，常住人口规模要控制在2300万人以内，2020年以后长期稳定在这一水平<ref>{{cite web |url= http://news.sina.com.cn/o/2017-09-30/doc-ifymksyw4999436.shtml|title=北京控制人口规模是首当其冲的硬任务|first= |last= |work= 新浪新闻|accessdate=2017-09-30}}</ref>。

在蔡奇的这一思想的指导下，北京全市开展了“疏解整治促提升”行动，清理、拆除外来人口聚居的出租大院是该项工作的核心，位于城乡结合部的近千个出租大院被拆除<ref>{{cite web |url= http://news.163.com/17/1117/09/D3EDH79700018AOR.html|title=城乡结合部近千出租大院清拆 外来人口将落脚何处|first= |last= |work= 网易新闻|accessdate=2017-11-16}}</ref>。大量廉价公寓和出租大院被查封、关停导致许多“[[北漂|北漂]]”一族一夜之间被迫搬离住处，无家可归。无钱租住正规租赁企业建造的高价品牌公寓的“低层次租客”们只能是要么搬到更远的郊外居住，要么离开北京<ref>{{cite web |url= http://www.szhgh.com/Article/news/society/2017-11-23/153752.html|title=北京地毯式清理出租公寓，事实比预期更严重|first= |last= |work= 红歌会网|accessdate=2017-11-24}}</ref>。

=== 整治安全隐患排查整改行动 ===
{{main|2017年北京安全隐患排查整改行动}}
2017年11月18日晚上，北京市[[大兴区|大兴区]]西红门镇新建二村发生[[2017年北京大兴火灾|重大火灾事故]]，造成19人死亡，8人受伤。火灾发生后第二天，以蔡奇为首的北京市领导立即召开全市安全隐患大排查大清理大整治专项行动部署电视电话会，下令全面开展以消防安全为重点的安全隐患大排查大清理大整治行动，一家一家、一村一村地毯式摸排，大力清理“三合一”“多合一”场所、工业大院、散乱污企业及违法建设，彻底消除安全隐患<ref>{{cite web |url= http://cpc.people.com.cn/n1/2017/1120/c64102-29656390.html|title=蔡奇：开展安全隐患大排查大清理大整治|first= |last= |work= 中国共产党新闻网|accessdate=2017-11-20}}</ref>。

蔡奇在一次內部會議談話中強硬表示：「按照政策、按照要求、按照部署，早就該清掉的」、「到了基層、到了點上，那就是真刀真槍，就是要[[刺刀|刺刀]]見紅，就是要敢硬碰硬，就是要解決問題，如果不是這種態度去開展工作的話，那遲早有一天你這個地盤上還要出這樣的事」、「各區黨委政府要嚴格落實屬地責任，一把手要親自抓」。蔡奇還聲稱再發生類似（失火）事件「大家就要把手剁下來」<ref>[http://news.ltn.com.tw/news/world/breakingnews/2274177 「刺刀見紅」影片曝光！中共北京書記撂狠話清除低端], [[自由時報|自由時報]], 2017-12-5</ref><ref>[https://www.youtube.com/watch?v=LD3tCSuJxB8 中共北京市委書記蔡奇內部講話錄], [[明鏡新聞集團|明鏡新聞集團]], 2017-12-3</ref><ref>[http://news.ltn.com.tw/news/world/breakingnews/2282183 蔡奇曾說「再失火就剁手」...北京民宅又傳火災釀5死], [[自由時報|自由時報]], 2017-12-13</ref><ref>{{cite web|url=http://orientaldaily.on.cc/cnt/china_world/20171206/00178_004.html|title=北京書記嚴斥眾下屬：「再有火災 大家要剁手」|publisher=東方日報|date=2017-12-05|accessdate=2017-12-14}}</ref>。

之后，北京21个相关部门为期40天的[[2017年北京安全隐患排查整改行动|安全隐患大排查大清理大整治专项行动]]正式开始，德邦、安能、百世、[[顺丰|顺丰]]、[[京东|京东]]、[[中通|中通]]、[[韵达|韵达]]、[[圆通|圆通]]、速尔、快捷等十余家知名[[物流|物流]]快递企业在北京的仓库、分拨、网点等等都受到波及。北京的很多仓库被查封，处于暂停状态，如联想物流通州仓库被查封、安能北京分拨爆仓严重<ref>{{cite web |url= http://www.sohu.com/a/206316276_768245|title=北京快递行业开展安全隐患大排查、大清理、大整治！|first= |last= |work= 搜狐|accessdate=2017-11-23}}</ref>。其中，在整治力度最大的[[顺义区|顺义区]]沙峪镇，100%关停清退大宇物流、中外运敦豪等88家仓储物流企业<ref>{{cite web |url= http://www.szhgh.com/Article/news/society/2017-11-23/153752.html|title=北京地毯式清理出租公寓，事实比预期更严重|first= |last= |work= 红歌会网|accessdate=2017-11-24}}</ref>。

=== 拆除北京建筑物楼顶牌匾 ===
{{main|北京市牌匾标识设置管理规范}}
2017年9月24日，蔡奇与北京市委副书记、市长[[陈吉宁|陈吉宁]]一起率领北京16个区和相关部门主要负责人先后来到[[西城区|西城区]]与[[东城区|东城区]]，聚焦背街小巷环境整治提升，组织第二次现场推进会。蔡奇表示，各区要加强城市精细化管理，抓好环境整治提升，努力营造整洁、有序、文明、祥和的城市环境。明年要把屋顶拆违列入整治清单，对凡超出建筑体或不符合要求的广告牌匾分步予以清理，胡同内要逐步实现不停车等<ref>{{cite web |url= http://bj.people.com.cn/n2/2017/0925/c82844-30770662.html|title=蔡奇：胡同内要逐步实现不停车 明年屋顶拆违列入整治清单|first= |last= |work= 人民网|accessdate=2017-09-25}}</ref>。

在蔡奇的这一思想指导下出台的《[[北京市牌匾标识设置管理规范|北京市牌匾标识设置管理规范]]（修订版）》明确规定，楼顶禁止设置牌匾、标识、广告等，已有的也要进行清理拆除，核心城区、[[长安街|长安街]]沿线、二三四环沿线是重点<ref>{{cite web |url= http://bj.news.163.com/17/1202/08/D4KUF7K004388CS9.html|title=北京亮出“天际线”行动持续 八千多块牌匾被拆|first= |last= |work= 网易新闻|accessdate=2017-12-02}}</ref>。北京官方称，全市需要拆除的违规广告牌匾超过2.7万块，七成以上集中在城六区（[[东城区|东城区]]、[[西城区|西城区]]、[[朝阳区|朝阳区]]、[[海淀区|海淀区]]、[[丰台区|丰台区]]和[[石景山区|石景山区]]等六个核心城区）<ref>{{cite web |url= http://www.sohu.com/a/207752581_260616|title=北京已拆除各类违规广告牌匾8956块，共需拆2.7万多块|first= |last= |work= 搜狐|accessdate=2017-12-01}}</ref>。拆除后建筑物只能使用在规划部门备案的大楼名称，悬挂位置则在楼体3层以上，距离屋顶线不小于0.5米。同时，字体大小也有严格规定，建筑物名称字体本身高度不超过2米，单位名称字体本身高度不超过0.8米。新的牌匾标识禁止使用不锈钢等高反光材料，也不能用外射光源，只能用内投光源。如果同栋楼有多个产权单位或商业信息需求时，建筑物产权单位可在出入口处设置符合规定的标识进行区分，或在地面设置立式信息牌方便查询<ref>{{cite web |url= http://bj.news.163.com/17/1202/08/D4KUF7K004388CS9.html|title=北京亮出“天际线”行动持续 八千多块牌匾被拆|first= |last= |work= 网易新闻|accessdate=2017-12-02}}</ref>，引起广泛争议，很多人{{who}}都认为这是劳民伤财的行为，纯属乱折腾。

=== 清華、人大校友聯署促下台 ===
2017年12月13日，在网络上廣傳一封[[公開信|公開信]]，指北京市委書記蔡奇近日的驅趕行動，影響範圍廣大、手法殘忍、忤逆背量、違法亂規，甚至是「北京建城3062年以來最惡劣的行政作惡事件」，包括[[清華大學|清華大學]]、[[中國人民大學|中國人民大學]]等多名校友聯署要求蔡辭職下台。與此同時，[[國家行政學院|國家行政學院]]的[[汪玉凱|汪玉凱]]教授，建議中央追責北京市的錯誤決策。<ref>{{Cite web|url=https://news.mingpao.com/pns/dailynews/web_tc/article/20171214/s00013/1513188343155|title=國家行政學院教授﹕整治行動「給中央添亂」 - 20171214 - 中國 - 中國新聞|accessdate=2017-12-17|work=明報新聞網 - 每日明報 daily news|language=zh-tw}}</ref>

這封公開信，指责蔡奇以「殺刀見紅」的高壓之勢，調動大批荷槍實彈的警員、持棍保安城管，在凜冽寒風中連夜搗毀居民的棲身之地，停水停電停暖氣等，其行為嚴重違反中共的黨章、背離中共宗旨，同時違反包括侵佔公民合法私有財產、伺犯公民人身自由等《憲法》、《行政強制法》等條文。在「事實俱在，鐵證如山，不容抵賴」下，呼籲已經嚴重違法亂紀的蔡奇「立即辭職，以謝天下」<ref>{{cite web |url= https://hk.news.appledaily.com/international/daily/article/20171214/20243993|title=清華、人民大學校友聯署促蔡奇下台|first= |last= |work= 苹果新闻|accessdate=2017-12-16}}</ref>。

==轶事==
蔡奇曾经是使用[[微博|微博]]级别最高的现任官员之一，他于2011年的浙江省委组织部长会议上，首次公布其已开通个人微博，至2014年3月，他在腾讯微博上的粉丝突破一千万。2013年11月，蔡奇在微博上率先公布其离任浙江省委组织部部长的信息，引起众多网友关注，这在此前的重大人事变动新闻中没有这一先例。

而在2011年9月，一位基层[[公务员|公务员]]的母亲发微博称在国税局工作的儿子不能喝酒，却常陪领导喝酒到喝醉，求助问“我该怎么办”。第二天，蔡奇就转发了该微博，并且回复称：“告诉我你儿子在国税哪个单位？今后可以不用喝酒了。”这一条微博一时间也成为了热门微博。在2014年3月蔡奇赴中央任职之后，他的微博也被关闭<ref>{{cite web |url= http://news.china.com.cn/2016-11/01/content_39609740.htm|title=北京代理市长蔡奇被称大V网红 曾拥千万级粉丝|first= |last= |work= 封面新闻|accessdate=2017-05-27}}</ref>。
==之江新军==
从1983年到1999年，蔡奇曾经在福建省委与三明市工作长达16年，官至[[三明市|三明市]][[市长|市长]]。在这段时间内，[[习近平|习近平]]也曾经在[[福建省|福建省]]工作。而后，蔡奇与[[龙岩市|龙岩市]]市长[[黄坤明|黄坤明]]一起被调往[[浙江省|浙江省]]任职，习近平也在2002年11月任[[浙江|浙江]][[省委书记|省委书记]]。期间，蔡奇先后主政[[衢州|衢州]]、[[台州|台州]]两座城市，是习近平的下属。

在2007年3月习近平调任[[上海|上海]][[市委書記|市委書記]]后不久，蔡奇任[[杭州市|杭州市]]人民政府市长。2014年3月，习近平执政刚满一年，就把蔡奇上调[[中央国家安全委员会|中央国家安全委员会]]办公室任职， 并且在2017年5月任[[北京|北京]][[市委書記|市委書記]]，主政中国[[首都|首都]]。因为蔡奇长期在福建、浙江与习近平有共事经历，所以被认为是其嫡系[[之江新军|之江新军]]的重要成员。
