'''伍孟昌'''{{BD|1911年||2006年|10月6日|catIdx=W}}笔名'''孟昌'''，曾用名'''伍梦窗'''，[[广东人|广东人]]，毕业于[[上海|上海]][[复旦大学|复旦大学]]外国文学系，[[中国共产党|中国共产党]]党员，中国大陆翻译家。<ref name="fish" >{{cite web |url=http://www.gmw.cn/01ds/2006-10/18/content_494371.htm |title=著名翻译家、作家伍孟昌逝世 |accessdate=2006 |publisher=光明网 |work= |language=zh-hans}}</ref>

==生平==
1911年，伍孟昌出生在[[广东省|广东省]][[新宁县|新宁县]]。

1932年，伍孟昌加入[[左翼作家联盟|左翼作家联盟]]。

1935年，伍孟昌毕业于[[上海|上海]][[复旦大学|复旦大学]]外国文学系。

1936年，伍孟昌开始发表作品。

1952年，伍孟昌加入[[中国作家协会|中国作家协会]]。

2006年，伍孟昌逝世于[[北京|北京]]。

==家庭==
伍孟昌儿子伍永光。

==作品==
伍孟昌主要是翻译[[俄罗斯|俄罗斯]]作家[[高尔基|高尔基]]的作品。

*《高尔基全集》

*《巴黎的秘密》（[[欧仁·苏|欧仁·苏]]）

*《美国人在日本》（[[库尔冈诺夫|库尔冈诺夫]]）

*《捷克斯洛伐克》（[[梅德韦杰夫|梅德韦杰夫]]）

==奖项==
*1995年，[[鲁迅文学奖|鲁迅文学奖]]之全国优秀文学翻译彩虹奖荣誉奖。

*2004年，中国翻译协会“资深翻译家”荣誉称号。<ref name="man" >{{cite web |url=http://news.sina.com.cn/o/2004-11-07/18414168028s.shtml |title= 118位资深翻译家获表彰 中国译协使用新名称|accessdate= 2004|publisher=新浪网 |work= |language=zh-hans}}</ref>
