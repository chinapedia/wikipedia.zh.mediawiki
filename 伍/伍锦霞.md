'''伍锦霞'''（'''Esther Eng'''或'''Ng Kam-ha'''，{{bd|1914年||1970年||catIdx=伍}}），第二代[[美國|美國]][[華僑|華僑]]，祖籍[[廣東|廣東]][[台山|台山]]<small><ref>{{cite web|url=http://hk.cnmdb.com/name/2105/bio|title=人物簡介：伍錦霞|publisher=中國影視資料館|accessdate=2009-08-03|deadurl=yes|archiveurl=https://archive.is/20120710184823/http://hk.cnmdb.com/name/2105/bio|archivedate=2012-07-10}}</ref></small>，是早期[[華語電影|华语片]]女[[导演|导演]]，曾在[[好莱坞|好莱坞]]电影业界发展，当时被称为“好莱坞唯一华裔女导演”。其一生曾导演或制作过11部电影作品，在[[香港|香港]]拍了5部粤语片，在[[美国|美国]]执导了6部华语片，获得相当的赞誉。

伍锦霞平时多留短发，穿一身男西装或猎装，畢生沒有與異性相戀及結婚，香港影壇中人更以「霞哥」相稱<small><ref>{{cite web |url=http://www.hkcampus.net/tec_zone/article.php?containerid=4819&folderid=66 |title=香港紀錄:第一位電影女導演 |publisher=hkcampus.net |accessdate=2009-08-03}}</ref></small>。

==生平==
伍锦霞于1914年出生于美国[[旧金山|旧金山]]，其父[[伍于泽|伍于泽]]是当地华商，有10个儿女，伍锦霞是他的第一个女儿。伍锦霞自小就是[[粤剧|粤剧]]和[[电影|电影]]迷，曾为能免费看戏在「文华剧院」做义务带位员。

其父亲为满足女儿的梦想在家创建了一家电影制作公司。1935年，年僅21歲的伍锦霞在[[好莱坞|好莱坞]][[日落大道|日落大道]]租了一个[[影棚|影棚]]，集资拍摄[[粵語|粵語]]电影《心恨》<small><ref>{{cite web|url=http://www.mingpaomonthly.com/cfm/Archive2.cfm?File=200504/feature/04a.txt&Page=2|title=香港與內地電影的互動（羅卡）|publisher=[[明報月刊|明報月刊]]|accessdate=2009-08-04|deadurl=yes|archiveurl=https://web.archive.org/web/20060529173342/http://www.mingpaomonthly.com/cfm/Archive2.cfm?File=200504%2Ffeature%2F04a.txt&Page=2|archivedate=2006-05-29}}</ref></small>，並携該片回[[中國|中國]]公映<small><ref>{{cite web|url=http://ipac.hkfa.lcsd.gov.hk/ipac20/ipac.jsp?session=1RQ9C70742612.5531&profile=hkfa&uri=full=3100024@!25083@!6&ri=1&aspect=basic_search&menu=search&source=192.168.110.61@!horizon&ipp=20&staffonly=&&aspect=basic_search&menu=search&ri=1#focus|title=伍錦霞|publisher=[[香港電影資料館|香港電影資料館]]|accessdate=2009-08-04}}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref></small>。

1936年，伍锦霞到[[香港|香港]]，导演粤语片《民族女英雄》，自此在华语电影界树立声誉，并接连执导《十万情人》（爱情伦理喜剧）、《妒花风雨》（又名《偷龍轉鳳》，爱情悲剧片）、《一夜夫妻》（奇情悲喜剧）。1939年执导电影《女人世界》，是世界电影史上第一部全部采用女性角色的影片。

[[太平洋战争|太平洋战争]]爆发后，伍锦霞返回旧金山。于1941年与赵树桑、关文清合作制作美国华语片《金门女》。这段时间她和父亲合办「金门银光公司」，主营粤语片在[[北美洲|北美]]、[[中美洲|中]][[南美洲|南美]]地区的发行业务。据1946年9月香港《伶星》报道，该公司在太平洋战争时期在美洲共发行40多部粤语片。

1950年代起，伍锦霞移师[[纽约|纽约]]经营中餐馆和华语影片发行。1951年，她在[[曼哈頓|曼哈顿区]]开设「宝宝酒家」，后改名为「伍锦霞餐馆」，几次被纽约的美食家评为二星食府（最高为三星食府），多年后仍在纽约的饮食媒体上中被提及。

1961年，伍锦霞和[[胡鹏|胡鹏]]（电影《[[黄飞鸿|黄飞鸿]]》的导演）合作制作《纽约唐人街碎尸案》。1970年，伍锦霞病逝于纽约。

==评价==
伍锦霞其人观念受到美国思想影响，性格活泼，是早期将民族意识与[[女权主义|女权主义]]带入电影作品的华裔导演，其中不乏跨文化跨题材的作品，受到影评人们的高度评价。

==主要作品介绍==
;心恨
被称为“在好莱坞拍制的第一部华语片”，在[[香港|香港]]上映时叫《铁血芳魂》，讲述一对女[[粤剧|粤剧]]演员与飞机师结合的美国华裔情侣，在[[淞沪会战|淞沪会战]]爆发的背景下，毅然为国家民族命运放弃爱情，最后男主角参加空军并为国牺牲。媒体《每日电影》评价“以航空为背景的时装戏，有着美国片的规模”。

;民族女英雄
讲述立志救国的女青年自愿从军的故事，主题严肃之余又为适应市场需要加入歌舞及喜剧元素。1937年3月13日在香港首映，后在[[广州|广州]]等地公映，该影片获得「广东女权协进联会」赞扬并授予奖状。

;女人世界
原名《三十六女天罡》，剧中36个角色是来自香港各社会阶层的女性，描述她们生活中的各种遭遇，由著名演員紫羅蘭、梁洪緣、[[梅綺|梅綺]]、大口何等主演<small><ref>{{cite web |url=http://www.dianying.com/ft/title/nrs1939 |title=女人世界 (1939) |publisher=dianying.com |accessdate=2009-08-03}}</ref></small>。

;金门女
以[[旧金山|旧金山]]为背景的电影。当时只有三个月大的[[李小龙|李小龙]]扮演女主角的幼年<small><ref>{{cite web |url=http://www.iatopia.com/CEDB/magsearch.jsp?artid=10957 |title=1959某日某之前的李小龍電影 |publisher=電影雙周 |accessdate=2009-08-03 }}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref></small>。
