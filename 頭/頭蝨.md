'''頭蝨'''（[[双名法|學名]]： ''Pediculus humanus capitis''）是[[蝨|蝨]]的一種，[[寄生|寄生]]於[[人類|人類]]的[[頭髮|頭髮]]或[[頸|頸部]]的[[毛髮|毛髮]]之中。在人類身體其他毛髮上寄生的蝨都不是頭蝨。

== 外觀特徵 ==
頭蝨呈[[褐色|褐色]]或[[灰色|灰色]]，有頭、胸、腹三部份，以及六隻腳。

頭蝨的前腳比較大，以便抓緊[[毛囊|毛囊]]。如以強力[[放大鏡|放大鏡]]從側面觀察，一隻成年的頭蝨外貌與一隻小型的[[螞蟻|螞蟻]]相似。

== 生命週期 ==
[[File:Pediculus-egg.jpg|left]]
[[雌性|雌性]]的頭蝨在[[頭皮|頭皮]]附近的毛髮上[[蛋|產卵]]。蟲卵的大小相當於[[印刷品|印刷品]]的一[[點|點]]，呈淺[[黃色|黃色]]、褐色或灰色，需7天—9天孵化成[[幼虫|幼虫]]。如蟲卵無法孵化便會[[死亡|死亡]]。死亡的卵呈暗色，類似[[葡萄乾|葡萄乾]]。

[[幼虫|幼虫]]和[[成蟲|成蟲]]相似，只是體積較小。幼蟲以[[血液|血液]]為[[食物|食物]]，在孵化後約8天—10天便會變為成蟲，期間進行三次[[變態|變態]] ([[脫皮|脫皮]])。成蟲仍以血液為食物，雌蟲會產卵。一隻脫離了人體的成蟲會在兩天內[[死亡|死亡]]。

頭蝨的[[性別|性別]]在完全成為成蟲前不易判定。雌蟲一旦成形便可受精產卵，每天大約產下3顆—7顆蟲卵。

成蟲的壽命大約為27天—30天。

== 感染及症狀 ==
{{main|頭蝨感染}}
頭蝨不帶源病菌及病毒疾病

頭蝨是經[[接觸|接觸]][[傳染|傳染]]的寄生蟲。除與患者親密接觸外，與患者共用[[毛巾|毛巾]]、[[衣物|衣物]]、[[梳|梳]]等亦可能受到感染。

== 預防與治療 ==
{{medical}}

預防頭蝨的方法包括：
* 定時清洗衣物及[[床上用品|床上用品]]，並保持個人衛生。
* 避免與頭蝨患者作親密接觸或共用物品 (如[[梳|梳]]、[[帽|帽]]、衣服、[[枕頭|枕頭]]等)
* 適量洗頭──過於頻繁地洗頭會減少頭髮間的油脂，反而會提高感染頭蝨的風險。
* 使用有藥性的[[洗髮精|洗髮精]]可以殺死頭蝨。用密齒的梳子梳頭可以除去蟲卵。患者使用的梳子必須徹底清洗。
