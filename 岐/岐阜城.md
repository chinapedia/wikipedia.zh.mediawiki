'''岐阜城'''是位於[[美濃國|美濃國]]井之口（今[[岐阜縣|岐阜縣]][[岐阜市|岐阜市]]）的一座日式城堡。在[[織田信長|織田信長]]統治該城之前，稱為'''稻葉山城'''。

== 歷史 ==
在1201年，[[二階堂行政|二階堂行政]]在金華山山頂築城，不過在[[二階堂行藤|二階堂行藤]]死後，第一次被廢城。1460年由[[齋藤利永|齋藤利永]]將城堡復築，後來成為了[[齋藤氏|齋藤氏]]的領地。1567年當織田信長攻陷稻葉山城後，從「[[周文王|周之文王]]定點[[岐山|岐山]]」（阜是山的意思），改名為岐阜城。織田信長以此為居城期間，這裡是[[織田政權|織田政權]]前期的中心。1576年，信長遷居[[安土城|安土城]]。1582年，信長於[[本能寺之變|本能寺之變]]被殺害後，岐阜城曾經被[[明智光秀|明智光秀]]的部屬和[[織田信孝|織田信孝]]佔據。該城在[[賤岳之戰|賤岳之戰]]後，被[[豐臣秀吉|豐臣秀吉]]的勢力完全壓制，之後由秀吉的部下[[池田輝政|池田輝政]]所領。

== 拆卸 ==
1600年[[關原之戰|關原之戰]]爆發前，東軍攻陷西軍的[[織田秀信|織田秀信]]的岐阜城。戰後，[[德川家康|德川家康]]決定廢城，新的大名[[奧平信昌|奧平信昌]]以岐阜城的櫓和天守築起[[加納城|加納城]]，在1910年再次築城，但在1943年被燒毀，第二次築起的時間為1956年。

== 天守 ==
在信長為城主的時代，建造了一座天主，後來改名為天守。岐阜城被廢後，天守遷築至加納城，1728年因閃電被燒毀。

== 構造 ==
 
山城。对京都来说，是位于东面的重要据点，雖然以難攻不落的名城而聞名，实际上卻曾有七次被攻陷的历史記錄。山頂上平坦面積小，井中的储蓄水来源為雨水，因此在面對戰國末期用人海围城的长期戰事本质上是極為不利。

在织田信长时代，山顶是信长家族和人质生活的地方，这在1569年（永禄12年）来到岐阜的传教士路易斯·弗洛伊斯的书信中有所记录。城的构造可以追溯到斋藤道三的时代。岐阜城跟小牧城、安土城一样，从城下町看去是非常优美的景观。也就是说并不是为了战斗而建造，基本上是作为城主的居住空间，让人观赏的城。
