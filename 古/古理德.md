'''古理德'''（{{bd|2001年|4月9日}}），[[台灣|台灣]][[兒童演員|兒童演員]]。參與過歌唱比賽及戲劇演出，其中主演『第六屆台灣國際兒童影展』開幕片《剪刀．石頭．布》

==人物==
===簡歷===
* 古理德畢業於台中市西屯區國安國小, 就讀福科國中, 參加過許多歌唱比賽，進而被導演[[王小棣|王小棣]]挖掘<ref>{{cite web
|url=http://web.pts.org.tw/php/newsletter/view.php?MODE=&NAENO=1&NEENO=2881 
|title=李佳穎被穿上身 當起學生領袖 
|accessdate=2014-07-31
|publisher=[[公視新聞|公視新聞]]
|language= zh-tw 
|date= 2013-09-18
}}</ref>, 在電視劇《[[刺蝟男孩|刺蝟男孩]]》中首次展露頭角<ref>{{cite web
|url=http://www.appledaily.com.tw/realtimenews/article/new/20130726/231283/ 
|title=王小棣新戲　翻版大雄技安加持 
|accessdate=2014-08-09
|publisher=[[蘋果日報|蘋果日報]]
|language= zh-tw 
|date= 2013-07-26
}}</ref>。其中演出[[王小棣|王小棣]]導演作品《[[刺蝟男孩|刺蝟男孩]]》及《剪刀‧石頭‧布》<ref>{{cite web
|url=http://news.ltn.com.tw/news/entertainment/paper/766057 
|title=王小棣拍創意電影 邀兒童獻計寫結局 
|accessdate=2014-08-09
|publisher=[[自由時報|自由時報]]
|language= zh-tw 
|date= 2014-03-29
}}</ref>。
* 2013年首次演出電視劇-《[[刺蝟男孩|刺蝟男孩]]》，即獲得[[第49屆金鐘獎|第49屆金鐘獎]]最佳戲劇節目男配角獎提名。
* 2014年首次演出院線電影-《想飛 Dream Flight》。
* 2014年主演『第六屆台灣國際兒童影展』開幕片《剪刀．石頭．布》男主角。

===性格及特徵===
* 因為《[[刺蝟男孩|刺蝟男孩]]》的演出受過訓練，導演[[王小棣|王小棣]]在電影《剪刀．石頭．布》中找來《[[刺蝟男孩|刺蝟男孩]]》兒童組原班人馬主演。<ref>[http://pts50.pixnet.net/blog/post/40932292 刺蝟男孩的小朋友變身了]，公視大不同::痞客邦PIXNET。</ref>
* 參加《[[金牌麥克風|金牌麥克風]]》歌唱表演時，自介為智慧男孩，為刺蝟男孩的諧音。
* 本身為左撇子，在一些作品中可看到用左手寫字吃飯。
