'''古蜀文明'''是指从远古时期到[[春秋時期|春秋時期]]早期，於今日[[四川|四川]]地区（包括四川省和重庆市等地）不同于中原文明的古文明。

== 簡述 ==
据考古发现，200多万年前[[旧石器时代|旧石器时代]]早期，四川便开始有了人类活动。四川境内有已被命名的旧石器时代文化遗址四处。距今7~8千年的时候，四川地区逐渐进入[[新石器时代|新石器时代]]。新石器时代遗址分布很广，目前已发现200多处，其中最具代表性的有[[广汉|广汉]][[三星堆遗址|三星堆遗址]]、[[广元|广元]][[营盘梁遗址|营盘梁遗址]]、[[西昌|西昌]][[礼州遗址|礼州遗址]]等。

约4~5千年前，四川地区进入上古传说时期，这一时期大约同[[中原|中原]]的[[夏|夏]]、[[商|商]]、[[周|周]]时期相当。此时的古史传说内容主要关于，上古四川先王世系和活动，较著名的有大禹导江、杜宇化鹃、长苰化碧等。尽管没有史料记载，但[[三星堆|三星堆]]、[[金沙遗址|金沙]]、[[十二桥|十二桥]]等遗址的考古发掘和口头传说证明，最迟到商代，[[成都平原|成都平原]]已经进入奴隶社会。

商周时期，在成都平原建立的奴隶制政权[[蜀国|蜀国]]可能与古史传说中的“三代蜀王”——[[蚕丛|蚕丛]]、[[柏灌|柏灌]]、[[鱼凫|鱼凫]]有关。

唐代诗人[[李白|李白]]在《[[蜀道难|蜀道难]]》中有也有提及古蜀王[[鱼凫|鱼凫]]：“蚕丛及鱼凫，开国何茫然，尔来四万八千岁，不与秦塞通人烟”。大约在夏商之际，蜀人部落从今[[茂县|茂县]]一带迁徙至成都平原。“三代蜀王”之后，大约相当于中原[[西周|西周]]时期，[[杜宇|杜宇]]王朝建立，其间蜀国的都城迁至郫邑（今[[郫县|郫县]]），杜宇王朝采用君主世袭制，势力强大，其势力基本覆盖了整个四川盆地。大约相当于[[春秋時期|春秋時期]]早期，杜宇氏禅位于治水有功的蜀相[[鳖灵|鳖灵]]，鳖灵建立了[[开明_(蜀国)|开明]]王朝。

虽然从古至今都有关于古蜀文明的传说与记录，但在还没有足够的考古发现以前，一直没有受到重视。长期以来，由于古代“内诸夏而外夷狄”的思想观念，人们普遍把[[中原文化|中原文化]]视为中国唯一的文化中心，并认为其他文化都只是中原文化的延伸。<ref name="G">{{cite journal en||author = 黄剑华 |date = 2001年 |title = 《三星堆文明与中原文明的关系》 |journal = 《中原文物》 |publisher = 河南博物院 |location = 郑州 |accessdate = 2014-07-06}}</ref>即使是20世纪40年代初（那时已经在广汉三星堆遗址发现了一部分文物），学者首次提出“巴蜀文化”时，也只是把它作为与中原文化稍有区别的[[青铜器文化|青铜器文化]]对待。直到[[卫聚贤|卫聚贤]]于1942年在《说文月刊》上刊登的《巴蜀文化》一文提出“蜀国在春秋以前拥有自己的文字，并可以铸造青铜器”的观点在学术界引起轩然大波，人们才开始重视起古蜀文明。<ref name="H"> {{Cite book | author = 段渝 | title = 《三星堆文明》 | location = 成都 | publisher = 四川人民出版社 | date = 2006年1月 | ISBN = 9787220070327 | accessdate = 2014-07-06}} </ref>20世纪80年代，[[陈德安|陈德安]]、[[陈显丹|陈显丹]]带领考古队在[[三星堆遗址|三星堆遗址]]进行了大规模的发掘，出土了内容丰富的文物。这证明了猜想中的古蜀文明确实存在。<ref name="I">{{Cite web |url = http://www.cctv.com/program/tsfx/20040420/100972.shtml |title = 考古中国 第一部 文明前传之三星堆：文明孤旅（1） | author = 韦大军| date = 2004-04-20 |accessdate = 2014-07-06}}</ref>自那以后，大多数专家都认为中华文明起源不止一处<ref name="G" />，现任[[国家文物局_(中国)|中国国家文物局]]副局长的[[童明康|童明康]]也曾指出：“中国文明的起源，恰似满天星斗。虽然，各地、各民族跨入文明门槛的步伐有先有后，同步或不同步，但都以自己特有的文明组成，丰富了中华文明，都是中华文明的缔造者。”<ref name="J">{{cite journal en||author = 童明康 |date = 1987年 |title = 《进一步探讨中国文明的起源——苏秉琦关于辽西考古新发现的谈话》 |journal = 《史学情报》 |publisher = 中国史学会 |location = 北京 |accessdate = 2014-07-06}}</ref>

==古蜀文明与有巢氏==
近代考古发现认为[[成都平原|成都平原]]的远古时期的建筑结构接近于传说中的[[有巢氏|有巢氏]]。 <ref>[http://news.artxun.com/jiaguwen-375-1870028.shtml 成都平原 有巢氏乐土]</ref>

==越南[[安陽王|安陽王]]==
An Dương Vương，本名[[蜀泮|蜀泮]]（Thục Phán）或開明泮，原是古蜀的王子，乃為[[鱉靈|鱉靈]]的後裔。在秦國滅蜀國之後，輾轉到達現在越南北部，建立[[甌雒國|甌雒國]]，並自稱為[[安陽王|安陽王]]，建都[[古螺|古螺]]，乃今河内近郊的[[東英縣|東英縣]]。
