'''古川本舖'''（{{lang-ja|古川本舗（ふるかわほんぽ）}}，本名不明，{{bd|1980年|3月3日}}），日本的[[音樂家|音樂家]]。另一名義為'''古川P'''。名字來自居住地附近的停車場「古川Parking」<ref name="Ascii-P1">{{cite web|url=http://ascii.jp/elem/000/000/491/491929/ |title=音楽に新しい憧れを　初音ミク・古川Ｐが語る |author=四本淑三 |publisher=ASCII.jp |date=2010-01-23 |accessdate=2015-03-28 |work=四本淑三の「ミュージック・ギークス！」 |language=ja |deadurl=yes |archiveurl=https://web.archive.org/web/20140328030814/http://ascii.jp/elem/000/000/491/491929/ |archivedate=2014-03-28 }}</ref>。

過去因為在「[[NICONICO動畫|NICONICO動畫]]」發表作品因而成名，後來轉變為獨立的音樂家。於2015年7月20日發表年內引退的消息，並於12月舉辦最後一場演唱會後引退。<ref>{{Ja icon}}[http://spaceshowermusic.com/news/41023/ 古川本舖引退及年內演唱會的消息]</ref>

== 簡歷 ==
於[[大阪府|大阪府]]出生<ref name="theinterviews">{{cite web|url=http://theinterviews.jp/fullkawahonpo/profile |title=fullkawahonpo Profile |publisher=The Interviews |accessdate=2015-03-28 |language=ja |deadurl=yes |archiveurl=https://web.archive.org/web/20140326052101/http://theinterviews.jp/fullkawahonpo/profile |archivedate=2014-03-26 }}</ref>，與大學輕音樂班的同學組成樂隊，前往東京<ref name="Ascii-P1"/>。樂隊於出道前解散。

2009年開始把原創[[VOCALOID|VOCALOID]]歌曲上傳至[[NICONICO動畫|NICONICO動畫]]，以古川本舖的名義活動。

== 作曲風格 ==
古川P創作的曲風範圍有搖滾（Rock）、民間（Folk）、電子樂（Electronica）等，更有將其混合而成的鄉村電子風（Folktronica）以及後搖滾（Postrock）風格的作品。

亦是Vocaloid P當中少數發表後搖滾風格的音樂製作人，曲風十分多元化。

作品大多散發著鄉愁般、纖細而慢步調的味道。除此之外，古川P所填寫的歌詞也有獲得很高的評價。

在部分歌曲中會使用有特殊的「古川語」來唱，即是用英語的發音方式來講日語。（可以參照期作品「mugs」）
