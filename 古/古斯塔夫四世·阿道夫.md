'''古斯塔夫四世·阿道夫'''（{{lang|sv|Gustav IV Adolf}}，{{bd|1778年|11月1日|1837年|2月7日}}），是1792年至1809年間的[[瑞典國王|瑞典國王]]，也是最後一個統治芬蘭的瑞典君主。

== 早年 ==
古斯塔夫·阿道夫於[[斯德哥爾摩|斯德哥爾摩]]出生，當時有傳言指他其實是一名瑞典貴族的兒子，而非[[古斯塔夫三世|古斯塔夫三世]]國王之子，這說法沒有得到證實。他接受父親和自由主義學者[[尼爾斯·馮·羅森斯坦|尼爾斯·馮·羅森斯坦]]（{{lang|sv|Nils von Rosenstein}}）教導。1792年3月，古斯塔夫三世被刺殺，只有14歲的古斯塔夫·阿道夫繼位為王，由叔父[[卡爾十三世|南曼蘭公爵卡爾]]攝政。

1796年，卡爾安排年幼的國王前往俄羅斯[[聖彼得堡|聖彼得堡]]，向[[葉卡捷琳娜二世|葉卡捷琳娜二世]]女皇的孫女[[阿歷珊德拉·帕夫洛夫娜女大公|阿歷珊德拉·帕夫洛夫娜女大公]]（{{lang|ru|Александра Павловна}}）求親。然而，古斯塔夫四世拒絕娶一個奉行[[俄羅斯東正教會|俄羅斯東正教會]]禮儀的女子為妻，他的悔婚使俄羅斯女皇十分憤怒，兩個月後就中風去世。當時，沒有人懷疑古斯塔夫的宗教狂熱可能有問題。相反，當古斯塔夫四世成年，卡爾公爵結束攝政時，很多人為了瑞典已經沒有使他們懊惱的智者，只剩下一個既節儉又虔誠的平庸君主。

== 政治 ==
他驅逐了不受民眾歡迎的攝政公爵參謀[[古斯塔夫·阿道夫·羅伊特霍爾姆|古斯塔夫·阿道夫·羅伊特霍爾姆]]（{{lang|sv|Gustaf Adolf Reuterholm}}），為他本人帶來一點聲譽。1797年他娶了[[巴登的弗雷德里卡|巴登的弗雷德里卡]]為妻，這場婚姻看似會觸發與俄羅斯的戰爭，但由於俄羅斯皇帝[[保羅一世_(俄國)|保羅一世]]和古斯塔夫·阿道夫都痛恨[[法蘭西共和國|法蘭西共和國]]，兩國的關係沒有決裂。古斯塔夫四世極度恐懼[[雅各賓派|雅各賓派]]，使他過分擔憂，遲遲不肯加冕，也不願召開議會。然而，瑞典自古斯塔夫三世對俄羅斯征戰失利後，一直受於混亂，1798和1799年的失敗政變更使形狀雪上加霜，使他不得不於1800年3月和4月在[[北雪平|北雪平]]召開國會會議。他遭受國會的猛烈反對後，自此不再召開會議。

== 失去芬蘭 ==
1805年，他參與[[第三次反法同盟|第三次反法同盟]]，但出師不利，瑞屬[[波美拉尼亞|波美拉尼亞]]被法軍佔領。俄羅斯本為瑞典的盟友，但1807年在[[蘇維埃茨克_(加里寧格勒州)|蒂爾西特]]與法國媾和，只剩下瑞典和葡萄牙與英國並肩作戰。1808年2月21日，俄羅斯以威迫瑞典服從[[拿破崙|拿破崙]]的[[大陸封鎖|大陸封鎖政策]]為名[[芬蘭戰爭|入侵芬蘭]]，只一個月就攻下了大部分芬蘭。1809年9月17日，瑞典宣佈戰敗，簽訂《[[弗雷德里克港條約|弗雷德里克港條約]]》，把瑞典東部三分之一國土割讓給俄羅斯，成立[[俄羅斯帝國|俄羅斯帝國]]治下的[[芬蘭大公國|芬蘭大公國]]。

== 政變和退位 ==
古斯塔夫四世·阿道夫與外交和作戰都十分無能，加速了軍官將領把他廢黜的陰謀。1809年3月13日，以卡爾·約翰·埃德勒克羅茨（{{lang|sv|Carl Johan Adlercreutz}}）為首的7個將領闖進王宮，在國王的寓所中把他擄獲，一家人關進[[格里普斯霍爾姆宮|格里普斯霍爾姆宮]]。臨時政府隨即成立，由卡爾公爵主政，而召開的國會鄭重認可這場革命。

3月29日，古斯塔夫四世·阿道夫為了保障兒子的王位，自願退位。但[[瑞典軍方|瑞典軍方]]控制的國會宣稱，古斯塔夫一家人都喪失了王位繼承權，可能是因為有關古斯塔夫身世的傳言。6月5日，卡爾公爵接受新的自由憲法後登基為王，是為[[卡爾十三世|卡爾十三世]]。被廢黜的古斯塔夫和家人被送往德意志，1812年，他和妻子離婚。

他流亡時使用了一些頭銜，如「戈托普伯爵」和「荷爾斯泰因-歐丁公爵」，最後在[[瑞士|瑞士]][[聖加侖|聖加侖]]一間小旅館安頓下來。他過着孤獨和貧困的生活，並在那裏死於中風。後來，瑞典國王[[奧斯卡二世|奧斯卡二世]]提議把他的遺體送回瑞典，安葬在[[騎士島教堂|騎士島教堂]]。

== 家庭 ==
[[File:Gustav_IV_Adolf_of_Sweden_and_Frederica_of_Baden.png|thumb]]]]
他在1797年和[[巴登的弗雷德里卡|弗雷德里卡]]結婚，有5名子女：

# 古斯塔夫王儲，後來是瓦薩親王古斯塔夫·古斯塔夫松（1799-1877）
# [[索菲公主_(瑞典)|索菲公主]]（1801-1865），嫁給巴登大公；她的孫女[[巴登的維多利亞|維多利亞]]後來嫁給瑞典國王[[古斯塔夫五世|古斯塔夫五世]]。（因此，現任國王[[卡爾十六世·古斯塔夫|卡爾十六世·古斯塔夫]]是古斯塔夫四世的後裔。）
# 卡爾·古斯塔夫王子，[[芬蘭大公|芬蘭大公]]，斯莫蘭公爵（1802- 1805）
# 阿美利亞·瑪利亞·夏洛塔公主（1805-1853），未婚、無子
# 塞西莉亞公主（1807-1844)，嫁[[奥尔登堡统治者列表|奧爾登堡大公]][[奧古斯特_(奥尔登堡)|奥古斯特]]

1812年，他與元配離婚。
