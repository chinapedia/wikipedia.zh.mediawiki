一般來說，《'''風土記'''》乃是[[日本|日本]]記錄地方風俗民情、[[地理|地理]]、[[歷史|歷史]]、農業、神話與民俗等的[[地方志|地方志]]；但狹義上來說則是[[奈良時代|奈良時代]][[元明天皇|元明天皇]]命令[[律令制|律令制]]下之各國編撰的文獻報告。由於沒有正式名稱，為了和後代的風土記區別，又稱為「'''古風土記'''」。

古風土記原有四十八國，如今流傳的只有《[[出雲國風土記|出雲國風土記]]》、《[[播磨國風土記|播磨國風土記]]》、《[[肥前國風土記|肥前國風土記]]》、《[[常陸國風土記|常陸國風土記]]》、《[[豐後國風土記|豐後國風土記]]》等較為完整；另有散落在其他書籍的零碎段落（被其他書籍引用而流傳下來的文獻，即[[辑佚|佚文]]），收錄約三十國的風土和傳說，但有人認為這些佚文可能成文於[[奈良時代|奈良時代]]。

== 概要 ==
[[奈良時代|奈良時代]]初期[[元明天皇|元明天皇]]下詔各[[令制國|令制國]]的國衙開始編纂，主要以[[漢文|漢文]]書寫，至少耗費二十年時間完成。為了整備律令制度，故仿效[[中國|中國]]的地方志編纂，並做為地方統治的方針<ref>參看《図説 地図とあらすじでわかる! 風土記》，坂本勝監修，青春出版社，2011年1月，ISBN 978-4413043014，頁16-17。</ref>。關於記載事例，《[[續日本紀|續日本紀]]》六卷曾提到<ref>請看{{ja}}[http://kindai.ndl.go.jp/info:ndljp/pid/772184/53 近代デジタルライブラリー - 続日本紀. 巻第１−９]。</ref>包含下列五項，但此時「風土記」之名稱仍未使用，體裁只有下級單位對上司提出[[公文|公文]]「解」而已。
# 郡鄉之名（以吉字示之）
# 產物
# 土地肥沃之狀況
# 地名起源
# 流傳之舊聞佚事

現存的[[手抄本|手抄本]]僅有5冊，其中以《[[出雲國風土記|出雲國風土記]]》最為完整；其餘《[[播磨國風土記|播磨國風土記]]》、《[[肥前國風土記|肥前國風土記]]》、《[[常陸國風土記|常陸國風土記]]》、《[[豐後國風土記|豐後國風土記]]》等4冊則一部分闕失<ref>參看《日本の歴史4 天平の時代》，榮原永遠男著，[[集英社|集英社]]，1991年9月，ISBN  978-4081950041，頁40。</ref>。雖然其餘風土記多以不可考，若干逸文散見於其他後世書籍，但是也有人對於這些逸文是否完成於奈良時代存著疑問。
