'''古隆族'''（Gurung）是分布在[[尼泊爾|尼泊爾]]中部的山地民族。根據2006年的統計，人口約69萬，占尼泊爾全國人口的0.5%。

== 分布 ==
古隆族主要分布在尼泊爾中部，[[喜马拉雅山|喜马拉雅山]]支脉[[安纳普尔纳山脉|安纳普尔纳山脉]]南坡的[[甘达基专区|甘达基专区]]，即[[拉姆琼县|拉姆琼]]、[[卡斯基县|卡斯基]]、[[塔纳胡县|塔纳胡]]、[[廓尔喀县|廓尔喀]]、[[桑加县|桑加]]和[[锡安贾县|锡安贾]]等縣，以及[[道拉吉里專區|道拉吉里專區]]的[[帕尔巴特縣|帕尔巴特縣]]。也有少數分布于[[錫金|錫金]]、[[不丹|不丹]]以及[[印度|印度]]的[[西孟加拉邦|西孟加拉邦]]。

他們也是[[廓爾喀|廓爾喀]]士兵主要招募對象。

== 語言 ==
语言属[[漢藏語系|漢藏語系]]的[[藏缅语族|藏缅语族]]。

== 宗教 ==
古隆人主要信仰[[佛教|佛教]]、10%人口信仰[[印度教|印度教]]。
