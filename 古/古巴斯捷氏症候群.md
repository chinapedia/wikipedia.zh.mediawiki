'''古巴斯捷氏症候群'''（{{lang|en|Goodpasture syndrome}}，{{lang|en|GPS}}），又稱'''古德巴斯捷氏病'''（{{lang|en|Goodpasture's disease}}）、'''肺出血腎炎症候群'''、'''抗腎小球基底膜抗體病'''（{{lang|en|anti-glomerular basement antibody disease/anti-GBM antibody disease}}），或'''抗GBM病'''（{{lang|en|anti-GBM disease}}）是一種[[罕见病|罕見]]的[[自體免疫性疾病|自體免疫性疾病]]，其中抗體攻擊肺和腎臟，導致從肺及腎功能衰竭出血。它可能會迅速導致永久性肺、腎損害，經常導致死亡。它配合免疫抑制藥物、如[[皮質類固醇|皮質類固醇]]和[[環磷酰胺|環磷酰胺]]一起治療，並用[[血漿分離置換法|血漿分離置換法]](Plasmapheresis)將其中的抗體從血液中除去。

本病最早是由美國[[范德堡大学|范德堡大学]]的病理學家[[歐內斯特·古巴斯捷|歐內斯特·古巴斯捷]]（Ernest William Goodpasture）於1919年所發表，後來以其名榮譽命名。<ref>{{cite journal | author=Goodpasture EW | title=The significance of certain pulmonary lesions in relation to the etiology of influenza | journal=Am J Med Sci | year=1919 | volume=158 | pages=863–870 | doi=10.1097/00000441-191911000-00012 | issue=6}}</ref><ref>{{cite journal |author=Salama AD, Levy JB, Lightstone L, Pusey CD |title=Goodpasture's disease |journal=Lancet |volume=358 |issue=9285 |pages=917–920 |date=September 2001 |pmid=11567730 |doi=10.1016/S0140-6736(01)06077-9}}</ref>

==症狀和病徵==
{{seealso|腎功能|腎生理學}}
抗腎小球基底膜(anti-GBM)抗體主要攻擊腎臟和肺、雖然造成關節疼痛，而一般病徵像是全身乏力、消瘦、疲勞、畏寒、發燒等症狀也很常見。<ref name = PMSR>{{cite web|title=Goodpasture Syndrome Clinical Presentation|work=Medscape Reference|publisher=WebMD|date=21 May 2013|accessdate=14 March 2014|url=http://emedicine.medscape.com/article/240556-clinical#showall|editor=Batuman, C|author=Kathuria, P; Sanghera, P; Stevenson, FT; Sharma, S; Lederer, E; Lohr, JW; Talavera, F; Verrelli, M}}</ref>患有GPS的患者，僅有肺部及腎臟的患者佔60-80%；僅是腎臟疾病的患者佔20-40％，而僅為肺部疾病的患者佔不到10％。<ref name = PMSR/>肺部症狀一般早於腎臟症狀的發生、通常包括：[[咳血|咳血]]、胸痛(總病例少於50％)、呼吸、咳嗽，及[[呼吸困难|呼吸困难]]。<ref name = MM>{{cite web|title=Goodpasture Syndrome: Diffuse Alveolar Hemorrhage and Pulmonary-Renal Syndrome|work=Merck Manual Professional|date=November 2013|accessdate=14 March 2014|url=http://www.merckmanuals.com/professional/pulmonary_disorders/diffuse_alveolar_hemorrhage_and_pulmonary-renal_syndrome/goodpasture_syndrome.html|author=Schwarz, MI}}</ref>腎臟的症狀通常包括[[血尿|血尿]]、[[蛋白尿|蛋白尿]]、四肢或面部的[[水肿|水肿]]、[[尿毒症|尿毒症]]，以及[[高血壓|高血壓]]。<ref name = PMSR/>

==病因==
其確切原因尚不清楚，但被認為是從而到肺中"侵害血管取血"、而此機制造成了"抗GBM抗體"(anti-GBM antibody)與[[肺泡|肺泡]]的接觸。這樣的致病起因範例包括：<ref name = MSR/>
{{colbegin|2}}
*暴露於有機溶劑(如[[氯仿|氯仿]])或烴。
*暴露於[[煙草|煙草]]的煙霧下。
*某些基因突變([[HLA-DR15|HLA-DR15]])。
*感染、例如[[流感A|流感A]]。
*吸入可卡因。
*吸入金屬粉塵。
*[[菌血症|菌血症]](Bacteraemia)。
*[[敗血症|敗血症]]。
*高氧環境。
*用"抗淋巴細胞治療法"(尤其是[[單克隆抗體|單克隆抗體]])治療。
{{colend}}

==病理生理學==
通過血液的[[浆细胞|浆细胞]]，GPS會引起"抗GBM抗體"的異常產生。<ref name = MSR/>"抗GBM抗體"會攻擊肺泡及[[腎小球基底膜|腎小球基底膜]]。<ref name = MSR/>這些抗體結合其基膜的反應[[抗原表位|抗原表位]]，並激活補體級聯反應，導致標記細胞死亡。<ref name = MSR/>[[T細胞|T細胞]]也牽連在內。<ref name = MSR/>它通常被認為是一個"II型超敏反應"(type II hypersensitivity reaction)。<ref name = MSR/>

==診斷==
此疾病診斷通常是較困難的，因為許多其它的疾病可能引GPS病徵的各種顯現形式、且GPS病徵本身是不尋常的罕見。<ref name = WMSR>{{cite web|title=Goodpasture Syndrome Workup|work=Medscape Reference|publisher=WebMD|date=21 May 2013|accessdate=14 March 2014|url=http://emedicine.medscape.com/article/240556-workup#showall|editor=Batuman, C|author=Kathuria, P; Sanghera, P; Stevenson, FT; Sharma, S; Lederer, E; Lohr, JW; Talavera, F; Verrelli, M}}</ref>完成診斷的最準確的方法是經由[[活體組織切片|活體組織切片]]的方法檢測受影響的組織、特別是腎臟因其"抗GBM抗體"的存在，故而它是獲得樣本最好的"研究器官"(studied-organ)。<ref name = WMSR/>在GPS約三分之一受影響的主要"抗GBM抗體"他們的血液也有[[抗中性粒細胞胞漿抗體|抗中性粒細胞胞漿抗體]](cytoplasmic antineutrophilic antibodies)，通常預先產生於"抗GBM抗體"之前大約幾個月甚至幾年。<ref name = WMSR/>之後對於受影響的患者GPS被診斷出的病況有愈來愈差傾向。<ref name = MSR/>

==治療==
用於GPS治療的主要方法是[[血漿分離置換法|血漿分離置換法]](Plasmapheresis)，此法是GPS患者的血液通過[[離心機|離心機]]，且根據重量來分離不同的血液要素。<ref name = TMSR>{{cite web|title=Goodpasture Syndrome Treatment & Management|work=Medscape Reference|publisher=WebMD|date=21 May 2013|accessdate=14 March 2014|url=http://emedicine.medscape.com/article/240556-treatment#showall|editor=Batuman, C|author=Kathuria, P; Sanghera, P; Stevenson, FT; Sharma, S; Lederer, E; Lohr, JW; Talavera, F; Verrelli, M}}</ref>[[血浆|血浆]]、血液清澈的一部分、包括抗GBM抗體而攻擊受影響病患的肺與腎臟，並將其濾出。<ref name = TMSR/>血液的其他部分，即[[紅血球|紅血球]]、[[白血球|白血球]]及[[血小板|血小板]]被循環回並作為靜脈內之補充液。<ref name = TMSR/>大多數受該疾病影響的病患需要用[[免疫抑制劑|免疫抑制劑]](藥物抑制[[免疫系統|免疫系統]]的功能)治療，尤其是[[環磷酰胺|環磷酰胺]]、[[潑尼松|潑尼松]]及[[利妥昔單抗|利妥昔單抗]]等藥物處理，以防止新"抗GBM抗體"的形成、從而防止進一步損害到腎臟和肺部。<ref name = TMSR/>其他毒性較低的免疫抑製劑、像[[硫唑嘌呤|硫唑嘌呤]]則可被用於維持緩解。<ref name = TMSR/>

==預後==
如果不進行治療、幾乎所有患此病的人最終會因後期的[[腎衰竭|腎衰竭]]或[[肺出血|肺出血]]而死亡。<ref name = MSR/>隨著治療的開始、5年生存率為>80％，且只有不到30％受影響的病患需要長期透析治療。<ref name = MSR/>同樣地，在[[澳大利亞|澳大利亞]]及[[新西蘭|新西蘭]]的病患其"中位生存時間"(median survival time)約<5.93>年。<ref name = MSR/>

==流行病學==
在歐洲和亞洲、GPS罕見的影響約每年每百萬人<0.5-1.8>。<ref name = MSR>{{cite web|title=Goodpasture Syndrome|work=Medscape Reference|publisher=WebMD|date=21 May 2013|accessdate=14 March 2014|url=http://emedicine.medscape.com/article/240556-overview#showall|editor=Batuman, C|author=Kathuria, P; Sanghera, P; Stevenson, FT; Sharma, S; Lederer, E; Lohr, JW; Talavera, F; Verrelli, M}}</ref>這也是[[自體免疫性疾病|自體免疫性疾病]]不尋常之處，而在於它比女性更常見的男性，也黑人比白人不太常見，但在[[新西蘭|新西蘭]]的[[毛利人|毛利人]]更常見。<ref name = MSR/>開始病發的峰值年齡範圍為<20-30>歲及<60-70>歲。<ref name = MSR/>
