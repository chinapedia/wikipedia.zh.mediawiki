'''古川愛李'''（{{bd|1989年|12月13日||}}），來自[[愛知縣|愛知縣]]，曾是[[日本|日本]]女性偶像團體[[SKE48|SKE48]]Team KII隊長。

== 經歷 ==
; 2004年
* 9月，和[[AKB48|AKB48]]中的[[梅田彩佳|梅田彩佳]]一樣，曾參加[[早安少女組|早安少女組]]幸運7期徵選會，只晉級到第二次甄選。

; 2005年
* 2月，再次參加[[早安少女組|早安少女組]]於2005年的徵選會。雖然成功到達第三次審查，但是落選了。(參與[[中學生日記|中學生日記]]的演出)

; 2006年
* 6月，參加了愛貝克思的選拔活動『avex audition 2006』並成為了最終的58名合格者中的其中一人。

; 2008年
* 9月，曾在『響き渡る歌声!今、新たな歌の伝道師が誕生する!!「J:COM×アニマックス 第2回全日本アニソングランプリ」』出場，為最後出場者。

; 2009年
* 4月，在『[[「神公演予定」|「神公演予定」]]～因諸多因素、也有可能無法成為神公演、望體諒』中、以SKE48的成員之身分首次登場。
* 5月25日，Team KII结成。

; 2010年
* 11月17日，成為了SKE48的第四張單曲 - [[1!2!3!4!_请多关照!|1!2!3!4! 请多关照!]]的選拔成員。這亦是她個人第一次成為選拔成員。

; 2011年
* 3月9日，成為SKE48的第五張單曲 - [[万岁Venus|万岁Venus]]的選拔成員。
* 7月27日，SKE38的第六張單曲 - [[翡翠色的沙滩裙|翡翠色的沙滩裙]]的選拔成員。
* 11月9日，SKE48的第七張單曲 - [[Okey_Dokey|Okey Dokey]]第一次選拔落選。

; 2012年
* 1月25日，SKE48的第八張單曲 - [[单恋Finally|单恋Finally]]重回選拔。
* 5月16日，SKE48的第九張單曲 - [[I_love_you_我爱你!|I love you 我爱你!]]選拔成員。
* 5月至6月，在AKB48第27張單曲選拔總選舉中，取得第30名，作為Under-girls，排名首次進入總選舉圈內。
* 9月19日，SKE48的第十張單曲 - [[即使接吻也是左撇子|即使接吻也是左撇子]]選拔成員。

; 2013年
* 1月30日，SKE48的第十一張單曲 - [[巧克力的奴隶|巧克力的奴隶]]選拔成員。
* 6月8日，在AKB第32張單曲選拔總選舉中取得第27名，為Under-girls一員。
* 7月17日，SKE48的第十二張單曲 - [[美麗的閃電|美麗的閃電]]選拔成員。
* 11月20日，SKE48的第十三張單曲 - [[贊成卡哇伊！|贊成卡哇伊！]]選拔成員。

; 2014年
* 1月24日，[[Samsung_Galaxy|GALAXY]]與SKE合作，被選為新Unit 《GALAXY of DREAMS》成員之一<ref>[http://www.ske48.co.jp/news/?id=1390536101 GALAXY of DREAMS結成のお知らせ] - SKE48オフィシャルサイト（2014年1月24日）</ref>。
* 2月24日，「AKB48グループ大組閣祭り〜時代は変わる。だけど、僕らは前しか向かねえ!〜」中隊伍再編，就任SKE48 TeamKII新隊長<ref>[http://girlsnews.tv/akb/137902 AKB48グループ大組閣祭りで大幅シャッフル! 完全移籍に号泣するメンバーも!] -  GirlsNews（2014年2月26日）</ref>。
* 3月19日，SKE48的第十四張單曲 - [[未來是什麼？|未來是什麼？]]選拔成員。
* 6月7日，在AKB第37張單曲選拔總選舉中取得第37名，為Next-girls一員。
* 7月30日，SKE48的第十五張單曲 - [[笨拙的太陽|笨拙的太陽]]選拔成員
* 12月10日，SKE48的第十六張單曲 - [[十二月的袋鼠|十二月的袋鼠]]選拔成員

; 2015年
* 2月1日，於公演會場上宣布將於2015年3月31日自SKE48畢業<ref>{{cite web|url=http://www.ske48.co.jp/news/?id=1422793574 |script-title=ja:SKE48 チームKll 古川愛李 活動辞退による卒業のご報告 |publisher=SKE48官方網站 |language=ja |date=2015-02-01 |accessdate=2015-02-01}}</ref>。
* 3月31日，在[[橫濱國際平和會議場|橫濱國際和平會議場]]舉行的[[風情萬種塞車中|風情萬種塞車中]]發行活動中畢業。

*為聲優[[堀江由衣|堀江由衣]]的超級粉絲，偶然在20歲生日那天（2009年12月13日）參加了廣播節目「神田朱未のわたしのすきなこと。」、並從[[神田朱未|神田朱未]]手中收到了堀江由衣所贈送的含有本人簽名的CD當做禮物，而後看見了堀江本人的生日祝福訊息之後感動的留下了眼淚。<ref>神田朱未のわたしのすきなこと。</ref>

* 除了以偶像歌手的身分外也有以漫畫家身份活動，並且Google+上不時發表以自己為原形的自創角色ちびあいりん為主題的插畫或漫畫。

* 於2013年6月時就任[[豐田汽車|豐田汽車]]的「夏亞專用AURIS」「[https://web.archive.org/web/20140714215550/https://netz.jp/char-auris/about/ ジオニックトヨタ]」的特別廣報部員<ref>[http://natalie.mu/music/news/91936 SKE48古川愛李「ジオニックトヨタ」特別広報部員に就任] - ナタリー（2013年6月3日）2014年3月25日閲覧</ref>。

*公演口號為「比起三餐更喜歡動畫！真想變成你的女主角呀♥我是綽號あいりん的古川愛李<ref>{{Citation|date=2011-05-13|year=2011|periodical=AKB48総選挙公式ガイドブック2011|title=AKB48総選挙候補者名鑑|publisher=講談社|pages=113|isbn=978-4-06-389560-5}}</ref>。」在自介時，「真想變成妳的女主角呀」的台詞部分會轉換成動漫聲。

*在其他成員生日的時候，會畫成員的肖像畫並送給生日的成員<ref>『ENTAME』2010年12月号 </ref>。

*為第5號的SKE48二次元同好會會員。
