'''古巴共产党'''（{{lang-es|Partido Comunista de Cuba}}），简称'''古共'''（PCC），是[[古巴共和国|古巴共和国]]的[[执政党|执政党]]，也是古巴国内唯一有实际影响力的[[政党|政党]]。古巴宪法规定古巴共产党是[[马克思列宁主义|马克思列宁主义]]和[[何塞·马蒂|马蒂思想]]的先锋组织，是古巴社会和国家的领导力量。

现任[[古共中央第一书记|古共中央第一书记]]是[[劳尔·卡斯特罗|劳尔·卡斯特罗]]，[[古巴共产党中央委员会第一书记|第二书记]]是{{link-en|何塞·马查多|José Ramón Machado Ventura}}。

== 历史 ==
[[File:Comité_Central_PCC.jpg|革命宮為古巴共产党中央委員會現址]]
1902年[[古巴|古巴]]独立后不久就出现了一些[[共產主义|共產主义]]和[[无政府主义|无政府主义]]组织。[[布拉斯·罗加|布拉斯·罗加]]、[[埃尼巴尔·埃斯卡兰特|埃尼巴尔·埃斯卡兰特]]、[[法比奥·格罗巴特|法比奥·格罗巴特]]、[[阿方索·伯纳尔·德尔·列斯戈|阿方索·伯纳尔·德尔·列斯戈]]和[[胡利奥·安东尼奥·梅利亚|胡利奥·安东尼奥·梅利亚]]等人于1925年创立了最早的[[人民社会党_(古巴)|古巴共产党]]。该党是[[共产国际|共产国际]]的成员，接受[[莫斯科|莫斯科]]方面领导，后来改名为“人民社会党”。

1956年底，[[菲德尔·卡斯特罗|菲德尔·卡斯特罗]]等82人乘[[格拉玛号|格拉玛号]]返回古巴开展游击战争，人民社会党最初对菲德尔·卡斯特罗持怀疑态度，但还是与他领导的[[七二六运动|七二六运动]]进行了政治上和军事上的合作。

1959年[[古巴革命|古巴革命]]胜利后，在[[切·格瓦拉|切·格瓦拉]]和[[劳尔·卡斯特罗|劳尔·卡斯特罗]]的影响下，菲德尔·卡斯特罗使人民社会党合法化。

[[猪湾事件|猪湾事件]]后的1961年5月1日，菲德尔·卡斯特罗在[[哈瓦那|哈瓦那]][[革命广场|革命广场]]上宣布古巴走[[社会主义|社会主义]]道路。同年7月，卡斯特罗领导的七二六运动与人民社会党、[[三一三革命指导委员会|三一三革命指导委员会]]正式合并为[[统一革命组织|统一革命组织]]。1962年3月26日，统一革命组织改组为[[古巴社会主义革命统一党|古巴社会主义革命统一党]]。1965年10月3日，古巴社会主义革命统一党又改组为古巴共产党。1992年，修改后的古巴宪法将在国内组建其它政党的行为除罪化后，出现了一些反对党，但这些政党在古巴国内几乎没有任何影响力。<ref>[http://sshl.ucsd.edu/collections/las/cuba/1990.html Cuba: Elections and Events 1991-2001] {{webarchive|url=https://web.archive.org/web/20070301123039/http://sshl.ucsd.edu/collections/las/cuba/1990.html |date=2007-03-01 }} Latin American Election Statistics Home</ref>

最初十年左右规模小而且在政治局外不活跃。1969年的党员只有5万5千名，占总人口的0.7%，是世界上最小的执政共产党。70年代开始机构扩展。1975年第一次全党大会的时候有党员20万。1980年43万，1985年52万。

由于苏联解体，1990年8月古巴政府宣布进入“和平时期的特殊阶段”。1991年10月四次全党大会决定：继续坚持社会主义道路，坚持共产党的领导，坚持[[计划经济|计划经济]]。不过后来开始经济变革。1997年的第五次代表大会决定在坚持[[一党制|一党制]]、社会主义公有制、计划经济以外，强调把经济工作放在优先地位。

2011年4月16至19日，古巴共产党第六次代表大会召开。大会审议通过了劳尔·卡斯特罗所作中心报告、《经济社会政策纲领》、《关于完善人民政权机关、选举制度和行政区划的决议》、《关于召开党的全国代表会议的决定》等4份文件，选举产生了第六届中央委员会、中央政治局和书记处。[[菲德尔·卡斯特罗|菲德尔·卡斯特罗]]正式卸任古共中央第一书记，由劳尔·卡斯特罗接任；马查多出任第二书记。《经济社会政策纲领》规定，古巴经济模式将继续以生产资料社会主义公有制和“按能力和劳动”为依据的社会主义分配原则为基础；古巴经济模式的目标是保证社会主义制度的持续和巩固，国家经济的发展和人民生活水平的提高。这一文件还列举了古巴即将采取的[[经济改革|经济改革]]措施，其中包括开放[[私营企业|私营部门]]、建立税收制度、实现企业自主、取消不必要的补贴等。

== 组织结构 ==
1975年召开[[古巴共产党第一次全国代表大会|第一次全国代表大会]]，后来又在1980年、1986年、1991年和1997年相继召开。1991年前的领导为政治局和书记处，1991年合并到扩大的[[古巴共产党中央政治局|中央政治局]]（20多个成员）并继续设立[[古巴共产党中央书记处|中央书记处]]。两届大会之间有中央委员会主持事务。第四次大会决定，允许信仰宗教的先进革命分子入党。第五届大会把中央委员会从225人缩减到150人。菲德尔·卡斯特罗一直任[[古巴共产党中央委员会第一书记|第一书记]]，其弟[[劳尔·卡斯特罗|劳尔·卡斯特罗]]任第二书记。2011年4月，古共召开第六次代表大会，选举劳尔代替其兄出任中央委员会第一书记的职务。

1997年[[古巴共产党第五次全国代表大会|第五次全國代表大會]]时有超过78万党员，32.1%为工人，13.8%为技术职业人员，8.2%为教师，7.5%为服务业劳动者。

下属有各级党校。另外有[[青年共产主义联盟_(古巴)|青年共产主义联盟]]和[[何塞·马蒂先锋组织|何塞·马蒂先锋组织]]，分别对青年和少年进行政治教育。党报为《[[格拉玛报|格拉玛报]]》。

== 意识形态 ==
比起其它执政的共产党如[[越南共产党|越南共产党]]和[[老挝人民革命党|老挝人民革命党]]，古巴共产党更执著于[[马克思列宁主义|马克思列宁主义]]传统和[[计划经济|计划经济]]模式，坚持[[社会主义|社会主义]]，尽量避免[[市场经济|市场经济]]改革。但在苏联解体以后，迫于经济压力采取了一些[[资本主义|资本主义]]的措施。

支持其它国家的社会主义革命。[[冷战|冷战]]期间与苏联联合出兵[[安哥拉|安哥拉]]，与[[南非|南非]]军队展开战斗，最终导致了[[纳米比亚|纳米比亚]]从南非独立<ref>[http://jczs.sina.com.cn/2003-12-24/1127173207.html 天空战记：古巴空军与南非在安哥拉的战斗(组图)]</ref>。

在苏联解体后，由于缺少外援古巴减少了对国际事务的介入。但坚持向外派出医生、农业技术人员来支援发展中国家。21世纪后开始向拉丁美洲国家的倾向社会主义的领导人，如[[巴西|巴西]]的[[卢拉·达·席尔瓦|卢拉·达·席尔瓦]]、[[委内瑞拉|委内瑞拉]]的[[乌戈·查韦斯|乌戈·查韦斯]]和[[玻利维亚|玻利维亚]]的[[埃沃·莫拉莱斯|埃沃·莫拉莱斯]]寻求支持。
