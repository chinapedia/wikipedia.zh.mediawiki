'''六派哲学'''（''{{IAST|şad-darśana}}''），亦作'''正統派'''（Astika），是[[印度哲学|印度哲学]]的六個主要派別，興起於[[笈多王朝|笈多王朝]]時期，即5世紀前後。共同尊奉《[[吠陀經|吠陀經]]》。

==六派==
這六派分別為<ref>{{harvnb|Flood|1996|pp=231-2}}</ref>：
# [[弥曼差|弥曼差]]（{{IAST|Mimāṃsā}}）、
# [[吠檀多|吠檀多]]（{{IAST|Védānta}} 或 Uttara Mimamsa）、
# [[数论_(印度哲学)|数论]]（{{IAST|Sāṃkhya}}）：於公元前4世紀由[[迦毗羅|迦毗羅]]<ref>又譯[[劫比羅|劫比羅]] / [[卡皮拉|卡皮拉]] / [[伽毗羅|伽毗羅]]</ref>所創立。
# [[胜论|胜论]]（{{IAST|Vaiśeṣika}}）、
# [[正理论|正理论]]（{{IAST|Nyāya}}）、
# [[瑜伽|瑜伽]]（{{IAST|Yóga}}）。

六大派別有各自的經典，分別如下：
# 弥曼差：《[[弥曼差经|弥曼差经]]》、
# 吠檀多：《[[梵經|梵經]]》、
# 數論：《[[數論頌|數論頌]]》、
# 胜论：《[[胜论经|胜论经]]》、
# 正理论：《[[正理经|正理经]]》、
# 瑜伽：《[[瑜伽經|瑜伽經]]》。

而歷史上及概念上，這六大派普遍分為以下三大類：
* [[正理论-胜论|正理论-胜论]]（{{IAST|Nyāya-Vaiśeṣika}}）、
* [[数论-瑜伽|数论-瑜伽]]（{{IAST|Sāṃkhya-Yóga}}）及
* [[弥曼差-吠檀多|弥曼差-吠檀多]]（{{IAST|Mimāṃsā-Védānta}} 或 Purva and Uttara Mimamsa）。

== 語源 ==
''{{IAST|Āstika}}''（[[IAST|IAST]]:{{IAST|āstika}}）指六大派別中每一個派別，是[[梵語|梵語]]形容詞及名詞，源於''{{IAST|āsti}}''<ref name="MW">{{harvnb|Monier-Williams|2006}}</ref>，即“存在”的意思，可解作“信念”<ref>{{harvnb|Apte|1965|pp=240}}</ref>或“信仰”<ref name="MW"/>。

== 其他教派 ==
六派以外，還有三個看重《吠陀經》的非婆羅門系宗教，它們的很多傳說和世界觀來源於印度教神話。分別是：
* [[佛教|佛教]]
* [[耆那教|耆那教]]
* [[順世論|順世論]]

== 汉译 ==
正统哲学六派的经典数千年来只有少数汉译，大多数未译成汉语。2003年由姚卫群翻译的《古印度六派哲学经典》由北京商务印书馆出版，添补了一大空白，为中文世界的人研究古印度正统哲学提供了很大方便。
