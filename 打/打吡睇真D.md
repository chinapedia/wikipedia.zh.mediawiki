《'''打吡睇真D'''》（{{lang-en|Derby Mini}}），是[[香港|香港]][[亞洲電視|亞洲電視]]製作的賽馬資訊節目，於2015年3月9日至3月13日星期一至五晚上21:30-22:00於[[本港台|本港台]]現場直播<ref>[http://news.sina.com.hk/news/20150310/-85-3642770/1.html 全新打吡節目本周推出]，新浪香港，2015-3-10</ref>，全節目共5集。<ref>{{cite web |url=http://www.hkatv.com/events/hk_derby/index.php?mod=promo |title=存档副本 |accessdate=2015-03-09 |deadurl=yes |archiveurl=https://web.archive.org/web/20150326062953/http://www.hkatv.com/events/hk_derby/index.php?mod=promo |archivedate=2015-03-26 }}</ref>
[[File:打吡睇真D.png|缩略图]]

==內容==
2015年《寶馬[[香港打吡大賽|香港打吡大賽]]》於3月15日舉行，是本地馬壇其中一項最高榮譽賽事。本節目將會介紹關於[[香港打吡大賽|香港打吡大賽]]的背景資料，回顧歷屆賽事的歷史，重點介紹今年其中8隻參賽馬匹，透過有獎問答遊戲，吸引觀眾注意這項盛事，進入沙田馬場參與。本節目邀請體育節目評述[[徐嘉樂|徐嘉樂]]及藝員[[袁潔儀|袁潔儀]]擔任主持，透過[[徐嘉樂|徐嘉樂]]的專業意見，解構[[香港打吡大賽|香港打吡大賽]]的資料及比賽形式，向觀眾介紹今屆[[香港打吡大賽|香港打吡大賽]]的情況，而[[袁潔儀|袁潔儀]]則以輕鬆活潑的手法為大家帶來賽事冷知識。<ref>{{cite web |url=http://www.hkatv.com/events/hk_derby/index.php?mod=home |title=存档副本 |accessdate=2015-03-09 |deadurl=yes |archiveurl=https://web.archive.org/web/20150924030814/http://www.hkatv.com/events/hk_derby/index.php?mod=home |archivedate=2015-09-24 }}</ref>
