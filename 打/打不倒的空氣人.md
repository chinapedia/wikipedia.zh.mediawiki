《'''打不倒的空氣人'''》（{{lang-ja|エアーマンが倒せない}}）是日本[[同人音樂|同人音樂團體]]「{{lang|ja|てつくずおきば}}」的成員{{lang|ja|せらみかるちたん}}於2007年創作，並於[[互聯網|互聯網]]公開的同人歌曲。

== 歌詞 ==
歌詞靈感來自於在《[[洛克人2_威利博士之謎|洛克人2 威利博士之謎]]》出現的[[頭目_(角色類型)|頭目]]——空氣人（{{lang|ja|エアーマン}}），由於空氣人實在太難打倒，所以作者便把玩家需要面對無數次空氣人也不屈不撓的姿態描寫出來並寫成歌曲。歌曲的內容如下<ref>{{cite web| date= 2008-02-23 | title= 求!エアーマンが倒せない~~的中文+日文+羅馬拼音
|url= https://tw.knowledge.yahoo.com/question/question?qid=1608022308042| publisher=[[Yahoo!|Yahoo!]]}}</ref>：
*在第一段中描寫玩家來到「消失的踏足處」{{註|熱源人BOSS戰前}}時出現瓶頸，但怎樣也無法攻略它。如果需要輕鬆地過關便需要使用打倒空氣人後取得的「2號載具」，不過因為它的龍捲風而沒法擊倒它。
*而在第二段中則描述玩家為了戰勝空氣人而去挑戰樹木人，希望拿到空氣人弱點——「飛葉盾」，但同樣地也無法擊敗它。

== 歷史 ==
《打不倒的空氣人》首先於2007年5月26日在[[NICONICO動畫|NICONICO動畫]]上載<ref name=''video''>{{cite web|url=http://www.nicovideo.jp/watch/sm350170|title=エアーマンが倒せない　（ｶﾗｵｹﾊﾞｰｼﾞｮﾝ）‐ニコニコ動画(9)|accessdate=2010-01-20|date=2007-05-26}}</ref>，而人聲版本則由同人音樂團體{{link-ja|Team.ねこかん【猫】|Team.ねこかん【猫】}}於2007年6月1日與同年7月1日分別在[[Youtube|Youtube]]與NICONICO動畫上傳<ref>{{cite web|url=http://www.nicovideo.jp/watch/sm542936|title=エアーマンが倒せない（ＴＥＡＭねこかんｖｅｒｓｉｏｎ）‐ニコニコ動画(9)|accessdate=2010-01-20|date=2007-07-01|archiveurl=https://web.archive.org/web/20100107084938/http://www.nicovideo.jp/watch/sm542936|archivedate=2010年1月7日|deadurl=no}}</ref>。

== 商業版本 ==
2007年9月1日，《打不倒的空氣人》的原聲版與{{lang|ja|Team.ねこかん【猫】}}的人聲版本在[[Comic_Market|Comic Market]]發行，同時{{lang|ja|Team.ねこかん【猫】}}的人聲版和器樂也在該天於[[iTunes_Store|iTunes Store]]同步登錄<ref>{{cite web|url=http://itunes.apple.com/us/album/air-man-ga-taosenai-ep/id280693606|title=Air Man Ga Taosenai - EP|accessdate=2010-01-20|date=2007-09-01|archiveurl=https://web.archive.org/web/20100119162915/http://itunes.apple.com/us/album/air-man-ga-taosenai-ep/id280693606|archivedate=2010年1月19日|deadurl=no}}</ref>。另外，這首曲子也包含在2008年7月9日發行的NICONICO動畫官方專輯《{{link-ja|試試聽CD。～NICONICO動畫精選～|CDで聞いてみて。～ニコニコ動画せれくちょん～}}》中。

而另一個由{{link-ja|Ryu*|中原龍太郎}}改編，zaki主唱的版本則包含在2009年4月1日發行的專輯《{{lang|ja|EXIT TRANCE PRESENTS ウマウマできるトランスを作ってみた 4 ごらんの有様だよ♥}}》中<ref>{{cite web|url=http://exittunes.com/cd/exit_tunes/qwce00086.html|title=EXIT TUNES / QWCE-00086 EXIT TRANCE PRESENTS ウマウマできるトランスを作ってみた ４ ごらんの有様だよ|accessdate=2010-02-02}}</ref>。

除此之外，在《洛克人改》中此曲被用作為空氣人的關卡的[[BGM|BGM]]<ref>[http://suleputer.capcom.co.jp/product/cd/cpca10221.html ロックマン改 アレンジしてみた!!]</ref><ref>{{cite web|url=http://www.nicovideo.jp/watch/sm11850720|title=『ロックマン改』アレンジしてみた！！|accessdate=2014-09-27|date=2010-08-23}}</ref>。

== 誤報 ==
最初這首歌發表的時候，一些新聞網站錯誤地把它當成了《洛克人2》的BGM<ref>[http://gilcrows.blog17.fc2.com/blog-entry-1372.html 【ロックマン】ニコニコ動画で人気沸騰！「エアーマンが倒せない」] </ref>。
