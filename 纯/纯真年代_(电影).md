《'''纯情年代'''》（{{lang-en|'''''The Age of Innocence'''''}}）是一部1993年的美國古典爱情電影，劇本改編自[[伊迪丝·华顿|伊迪丝·华顿]]的1920年[[純真年代_(小說)|同名小说]]。故事發生在[[鍍金時代|鍍金時代]]期間，描繪了紐約的上流社會。本片由[[馬田·史高西斯|馬田·史高西斯]]執導，主演是[[丹尼爾·戴-劉易斯|丹尼爾·戴-劉易斯]]、[[米雪·菲花|米雪·菲花]]和[[薇诺娜·瑞德|薇诺娜·瑞德]]。

本片贏得[[奧斯卡最佳服裝設計獎|奧斯卡最佳服裝設計獎]]，以及提名[[奧斯卡最佳女配角獎|奧斯卡最佳女配角獎]]（薇诺娜·瑞德）、[[奧斯卡最佳改編劇本獎|奧斯卡最佳改編劇本獎]]、[[奧斯卡最佳原創音樂獎|奧斯卡最佳原創音樂獎]]和[[奧斯卡最佳藝術指導獎|奧斯卡最佳藝術指導獎]]。<ref>{{Cite web |url = http://www.oscars.org/oscars/ceremonies/1994 |title = The 66th Academy Awards (1994) Nominees and Winners |publisher = oscars.org |language = en}}</ref>

本片獻給馬田·史高西斯的父親[[查爾斯·史高西斯|查爾斯·史高西斯]]，他在完成本片前去世。

==剧情简介==
在1870年代的[[纽约|纽约]]，出身於上流社會的[[律师|律师]]纽兰·亚彻（Newland Archer，[[丹尼爾·戴-劉易斯|丹尼爾·戴-劉易斯]] 饰演）与美丽但传统的大家閨秀梅·威兰（May Welland，[[薇诺娜·瑞德|薇诺娜·瑞德]] 饰演）订下了婚约。但是，当她那成熟迷人的表姊艾倫·奥兰丝卡伯爵夫人（Countess Ellen Olenska，[[米雪·菲花|米雪·菲花]] 饰演）自歐洲返回紐約后，纽兰开始质疑他之前为自己所定下的生活，而艾倫不理社会輿論壓力，正在设法与暴力的丈夫寻求离婚。纽兰被威兰家族派去遊說艾倫打消离婚念头，但反而渐渐地被艾倫吸引。纽兰到底要追求自己心中的决定，仰或跟随大众意愿，与未婚妻结婚？

== 反響 ==
=== 票房 ===
本片在商業上並不成功，從3400萬美元的預算中，美國國內總共達到3230萬美元。<ref name = mojo />

=== 評價 ===
《純真年代》目前在[[爛番茄|爛番茄]]上獲得80％的得分<ref>{{Cite web |url = https://www.rottentomatoes.com/m/age_of_innocence/ |title = THE AGE OF INNOCENCE (1993) |publisher = rottentomatoes.com |language = en}}</ref>，而[[Metacritic|Metacritic]]評分為83分<ref>{{Cite web |url = http://www.metacritic.com/movie/the-age-of-innocence |title = The Age of Innocence 1993 |publisher = metacritic.com |language = en}}</ref>，表明大部分都是正面評價。
