'''邵玉轩'''（1866年—1920年），名'''镛'''，字'''行银'''，号'''玉轩'''，清末民国上海大颜料商、[[实业家|实业家]]。影视巨头[[邵逸夫|邵逸夫]]之父<ref name='hexun'>{{cite web|title=邵逸夫的商业帝国 |url=http://bschool.hexun.com/2014-01-07/161224191.html |accessdate=2014-01-08 |date=2014年1月7日 |publisher=[[和讯|和讯]]商学院}}</ref>。

==生平==

出生望族，系[[北宋|北宋]]梅花易數大師[[邵康节|邵康节]]第三十二世嫡孙<ref name='zhuangshi'>{{cite web |title=宁波帮庄市籍人士（部分） |url=http://zs.zhxww.net/rwzs/rwzs/rwzs_sbgl/20080414145054-2.htm |accessdate=2014-01-08 |date=日期:2008-4-13 14:50:30 |publisher=来源:镇海网讯 |archive-url=https://web.archive.org/web/20140108235706/http://zs.zhxww.net/rwzs/rwzs/rwzs_sbgl/20080414145054-2.htm |archive-date=2014-01-08 |dead-url=yes }}</ref>。出生[[大清|大清]][[宁波府|宁波府]][[镇海县|镇海县]]（现在的[[镇海区|镇海区]]）[[庄市|庄市]][[朱家桥|朱家桥]][[老邵村|老邵村]]。早年因父亲得病不治早逝，家道随中落，只能弃学从商养家。曾赴浙江[[湖州|湖州]]药房学习中医，1880年于[[乌镇|乌镇]]药肆习业<ref name='zhuangshi'/>。[[第二次鸦片战争|第二次鸦片战争]]之后，宁波五口通商，带来西方商机，邵玉轩早年也曾在宁波经营百货和进出口业务。

19世纪末，只身一人赴[[上海|上海]]经商，曾从业于上海万隆顺颜料号。几年后，成为万丰盛福记经理。1901年，在上海创设“锦泰昌”颜料号，不久后成为颜料业的翘楚之一，与[[周宗良|周宗良]]齐名。<ref name='hexun'/><ref name='zhuangshi'/>

邵玉轩在晚清民国初上海滩商界、政界均颇为活跃，人脉宽广，与[[虞洽卿|虞洽卿]]、[[朱葆三|朱葆三]]、[[严信厚|严信厚]]等为至交，与当时晚清政界中的[[维新派|维新派]]人士也经常往来。邵玉轩支持[[孙中山|孙中山]]、[[陈其美|陈其美]]的反清运动，与[[光复会|光复会]]、[[同盟会|同盟会]]秘密交往，曾斥巨资助[[辛亥革命|辛亥革命]]和[[上海光复|上海光复]]。<ref name='hexun'/><ref name='zhuangshi'/>

1920年，逝世于上海。当时“海上闻人”[[虞洽卿|虞洽卿]]，晚晴维新派领袖[[康有为|康有为]]，江苏督军[[卢永祥|卢永祥]]，国民党元老[[谭延闿|谭延闿]]，北洋政要[[王正黼|王正黼]]，民国公子[[袁克文|袁克文]]，晚清进士[[徐谦|徐谦]]等政商名流均为其赠挽致哀<ref name='hexun'/><ref name='zhuangshi'/><ref name='IT Times'>{{cite web|title=“娱乐之王”邵逸夫：百年人生远比电影更精彩 |url=http://www.ittime.com.cn/index.php?m=content&c=index&a=show&catid=83&id=7661 |accessdate=2014-01-08 |date=2014-01-08 |author=文章来源 / [[中国网|中国网]] |publisher=IT时代周刊}}</ref>。

==身后==
邵玉轩去世后，归葬镇海，家族颜料和进出口生意没能继续，因为几个儿子均不感兴趣而投身新兴的电影业<ref name='IT Times'/>。这一家族事业“后继无人”的窘况也恰巧出现在邵逸夫身上，邵逸夫晚年及去世后，两位儿子均表示对父亲生前的影视帝国“不感兴趣”，无意继承<ref name='Shaw inherit'>{{cite web|title=邵逸夫电视王国市值200亿港币 四子女拒接掌 |url=http://news.163.com/14/0108/13/9I2PDQQ400014JB6.html |accessdate=2014-01-08 |date=2014-01-08 |author=来源: [[中国新闻网|中国新闻网]](北京) |publisher=[[网易|网易]]新闻}}</ref>。

1958年，邵玉轩墓迁至[[宁波专区|宁波专区]]鄞县[[宝幢乡|宝幢乡]][[西岙村|西岙村]]（今属[[宁波市|宁波市]][[鄞州区|鄞州区]][[五乡镇|五乡镇]]）。

==身前慈善==

邵玉轩生前热衷公益慈善事业，曾斥资资助上海[[四明公所|四明公所]]。曾出资在家乡镇海修建桥梁，浚西大河，修葺[[海塘|海塘]]堤坝，并施舍救济老幼贫困。大清[[宣统|宣统]]二年（1910年），与[[宋炜臣|宋炜臣]]、[[阮雯衷|阮雯衷]]一道出资修建家乡道路，并斥资修缮后练浦庙。<ref name='zhuangshi'/><ref name='Shao Yusuan'>{{cite web |title=邵玉轩修祖庙求卜应验 |url=http://zs.zhxww.net/rwzs/wxyyw/w_yycs/20080415135013.htm |accessdate=2014-01-08 |date=日期:2008-4-15 13:49:19 |publisher=来源:镇海网讯 |archive-url=https://web.archive.org/web/20140108235709/http://zs.zhxww.net/rwzs/wxyyw/w_yycs/20080415135013.htm |archive-date=2014-01-08 |dead-url=yes }}</ref>

==家庭==
邵玉轩遗有6男1女。其中四个儿子投身当时亚洲新兴的电影行业。子女中著名的有<ref name='IT Times'/>：
* 长子[[邵醉翁|邵醉翁]]，即邵仁傑（英文：[[:en:Runje_Shaw|Runje Shaw]]），上海滩著名电影实业家
* 二子[[邵邨人|邵邨人]]，即邵仁隸（英文：[[:en:Runde_Shaw|Runde Shaw]]）
* 三子[[邵山客|邵山客]]，即邵仁枚（英文：[[:en:Runme_Shaw|Runme Shaw]]），新加坡著名企业家
* 六子[[邵逸夫|邵逸夫]]，即邵仁楞（英文：[[:en:Run_Run_Shaw|Run Run Shaw]]），日后成为亚洲影视大王，功夫片鼻祖<ref name='Kong-fu'>{{cite web|title=外媒评价：邵是功夫片先驱 |url=http://news.takungpao.com/paper/q/2014/0109/2163935.html |accessdate=2014-01-08 |date=2014-01-09 |publisher=[[大公网|大公网]]}}</ref>。1987年4月，邵逸夫曾回乡扫墓祭祖<ref name='zhuangshi'/>。
