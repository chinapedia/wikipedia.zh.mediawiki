'''邵鐵鴻'''（{{bd|1914年||1982年|4月11日}}），[[廣東省|廣東省]][[南海縣|南海縣]]三山鄉人，字雲生，號水雲鄉主；字習瘦金體；性格豪邁爽快的[[音樂家|音樂家]]，其中[[廣東音樂|廣東音樂]]名曲《[[紅豆曲|紅豆曲]]》(1941年)、《[[流水行雲|流水行雲]]》及《[[錦城春|錦城春]]》更是他的代表作，[[香港電影|香港電影]]演員。1940年代，邵鐵鴻在[[香港|香港]][[顯勤學校|顯勤學校]]唸書，後轉入[[大同中學|大同中學]]，求學時期已經常參加畢業典禮演奏會及慈善籌募會等活動。<ref>1979年7月8日(星期日)，香港《華僑日報》，第四張；第四頁。</ref><ref>1979年7月14日(星期六)，香港《華僑日報》，第五張；第三頁。</ref>。

== 個人音樂會 ==
1979年7月14日晚上，邵鐵鴻在[[中環|中環]][[香港大會堂|大會堂]]音樂廳舉行個人音樂作品欣賞會，獨奏[[揚琴|揚琴]]、[[二胡|二胡]]、[[古箏|古箏]]、[[洞簫|洞簫]]、[[琵琶|琵琶]]。[[胡章釗|胡章釗]]擔任司儀：
# 音樂合奏《平步青雲》及《錦城春》；
# [[梁潔芳|梁潔芳]]獨唱《紅豆曲》；
# 邵鐵鴻[[揚琴|揚琴]]獨奏《倒垂簾》及《到春雷》；
# [[李楓|李楓]]獨唱《憶王孫》及《愁怨》；
# [[黃月娥|黃月娥]]獨唱《花嬌人更嬌》；
# 邵鐵鴻[[二胡|二胡]]獨奏《彩鳳引金龍》及《鳥投林》；
# 邵鐵鴻[[古箏|古箏]]獨奏《相思扣》及《漢宮秋月》；
# 邵鐵鴻[[洞簫|洞簫]]獨奏《流水行雲》及《龍戲珠》；
# 邵鐵鴻[[琵琶|琵琶]]獨奏《風雪月》及《滿江紅》；
# 邵鐵鴻及[[劉艷華|劉艷華]]合唱《郎歸晚》。<ref>1979年7月15日(星期日)，香港《華僑日報》，第四張；第三頁。</ref>

== 逝世 ==
1982年4月11日，邵鐵鴻在[[灣仔|灣仔]][[軒尼詩道|軒尼詩道]]廣東酒樓出席[[披荊文會|披荊文會]]雅集，在席間突然[[腦充血|腦充血]]暈倒，急送[[鄧肇堅醫院|鄧肇堅醫院]]，延至凌晨12時卒告不治<ref>1982年4月14日(星期三)，香港《華僑日報》，第七張；第三頁。</ref>。同年4月17日下午1時，披荊文會在[[北角|北角]][[英皇道|英皇道]]679號[[香港殯儀館|香港殯儀館]]主持公祭，[[許世昌|許世昌]]恭讀祭文：
{{quote|
鐵板聲沉，銅琶絕響。鴻爪難尋，空餘惆悵。紅豆曲終，永懷形相。流水行雲，長為時尚。家室無憂，親朋有讓。淡蕩襟期，風高節亮。義結披荊，詩多酬唱。風雨晦明，向安無恙。詎意三春，勿歸泉壤。文酒當場，魂銷絳帳。一瞬千秋，人間天上。得住仙鄉，雲天萬丈。從此心魂，定多安暢。寄語殷殷，伏維尚饗。
}}
遺體在[[歌連臣角火葬場|歌連臣角火葬場]]舉行火葬，回程在灣仔軒尼詩道廣東酒樓飲[[解穢酒|解穢酒]]<ref name="ReferenceA">1982年4月18日(星期日)，香港《華僑日報》，第四張；第四頁。</ref>。

== 香港文人對邵鐵鴻的詩詞<ref name="ReferenceA"/> ==
奉題邵鐵鴻流水行雲曲：
* 趙少昂：
{{quote|
簫聲引鳳世無儔，流水行雲韻更幽。多難萬方傷未已，憑君一曲解煩憂。
}}
* 潘小聲：
{{quote|
少年同學故翩翩，中歲相逢法曲傳。我亦行雲逐流水，迴腸最是管弦邊！
}}
* 逸駿楊眉：
{{quote|
行雲流水意悠悠，化入風人十指頭。朔漠倘教傳逸響，厭聽笛羌譜涼州。
}}
