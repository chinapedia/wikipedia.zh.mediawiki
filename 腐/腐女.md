'''腐女'''（{{jpn|j=腐女子|hg=ふじょし|rm=Fujoshi}}），主要是指喜歡[[BL_(和製英語)|BL]]，也就是喜歡男男愛情的[[女性|女性]]。

== 語源 ==
「腐女子」一詞源自於[[日语|日语]]，是由同音的「{{lang|ja|婦女子}}（{{lang|ja|ふじょし}}）」轉化而來，為喜愛[[BL_(和製英語)|BL]]（boys love）的女性自嘲的用語。而腐女子是專門指稱對於男男愛情（BL系）作品情有獨鍾的女性，通常是喜歡此類作品的女性之間彼此自嘲的講法；非此族群以「腐女子」稱呼這些女性時，則是帶有貶低、蔑視意味的詞彙。在日本，直接稱呼對方為「腐女」通常被認為不禮貌。如同在台灣可以自稱宅用以自嘲，但說他人宅很可能冒犯別人。

== 與[[同人女|同人女]]的交叉關係 ==
同人女本来單純用来指愛好創作[[同人|同人]]作品的女性。由於腐女喜歡由BL的觀點詮釋原創作品，所以會有不少腐女親自繪畫和書寫相關同人作品。但同人女不一定是腐女，腐女亦不一定會進行同人創作。所以同人女與腐女的定義仍然應該清楚的區別開來。

== 社會關注 ==
經由《[[電車男|電車男]]》的轟動引起一般人對御宅族的關心，媒體間接注意到腐女族群。2006年日本出版了多本關於腐女的漫畫<ref>[[紺条夏生|紺条夏生]]《[[妄想少女御宅系|妄想少女御宅系]]》，双葉社。2006年。</ref><ref>[[小島アジコ|小島アジコ]]《[[我的801女友|我的801女友]]》，宙出版。2006年。</ref>和書籍<ref>[[ぺんたぶ|ぺんたぶ]]《[[我的腐女友|我的腐女友]]》，エンターブレイン。2006年。</ref>，電視上亦出現介紹腐女的節目<ref>[[東京電視台|東京電視台]]「出没！アド街ック天国」。2006年六月。</ref>，有些媒體將腐女當作女性[[御宅族|御宅族]]介紹，台灣也有娛樂節目製作過腐女專輯，縱然腐女本身就不是臺面下的群體（不同於御宅族）。

== 誤解 ==
隨著「腐女」這個詞更加廣為人知的同時，被誤用、曲解不時發生。最常被誤解的有：以為腐女是[[宅女|宅]]在家不出門的女子、腐女等同魚乾女（在外亮麗，在家邋遢）、腐女是[[女同性恋|女同性恋]]、長相不佳、交際障礙、喜歡「男同志交往」情節、皆為單身等等負面或錯誤的印象。{{Or|另外，認為孩子是腐女就等於[[女同性恋|女同性恋]]的家長尤其是非常多。|time=2018-04-14T04:47:07+00:00}}

===中國大陸===
*因為大眾有時並不清楚「BL」與現實中的男同性戀的區別，誤以為腐女就是「對男同友善的女性」。由此也衍生出「偽腐」一詞用以形容「喜歡BL，但對男同性戀不友善」的腐女。

*2012年6月，人力飛行劇團的[[舞台劇|舞台劇]]《[[你傷不起的腐女|你傷不起的腐女]]》，將劇名使用的「腐女」二字釋義成「等著腐爛的女生」，造成腐女極度不滿，引發腐女至臉書粉絲專頁抗議。而該劇工作人員僅以臉書消極回應，並解釋「此為創作，與腐女無關」、「創作自由，使用『腐女』二字並沒有錯」等言論。<ref>[https://www.facebook.com/mr.wing.theatre/posts/10151223861444008 人力飛行劇團的公開聲明]</ref>該另類劇以「腐女」名義企圖攏腐女與非腐女觀眾群，結果是雙方觀眾都無法接受；反而造成對於「腐女」二字無知的觀眾看了該劇，對「腐女」二字產生極度負面印象。

==衍生==
*[[腐男|腐男]]（腐男子（{{lang|ja|ふだんし}}，fudanshi）：即指喜歡「BL」的男性人群。常被誤解成喜歡百合，或者被誤解成男同性戀者。
*油腐、豆腐：喜歡「BL」的人群，不分性別。該詞主要在台灣地區使用。
