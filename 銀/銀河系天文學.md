'''銀河系天文學'''是研究我們的[[銀河系|銀河系]]和其所有内容。相對來說，[[星系天文學|星系天文學]]是研究在我們銀河系之外的一切，包括所有其他的星系。

不要將銀河系天文學和[[星系的形成和演化|星系的形成和演化]]混淆，後者一般是研究[[星系|星系]]的誕生、結構、成分、動力學、交互作用和它們的形式和範圍。

我們自己的[[銀河系|銀河系]]，就是我們的[[太陽系|太陽系]]所属于的星系，在很多方面是被研究得最多的星系，即使重要的部分在可見[[波長|波長]]區域被[[宇宙塵|宇宙塵]]遮蔽了，在20世紀發展的[[無線電天文學|無線電天文學]]、[[紅外線天文學|紅外線天文學]]、和{{le|次微米波天文學|Submillimetre astronomy}}仍將被氣體和塵埃遮蔽的區域首度呈現出銀河系的圖形。

== 子類型 ==
在天文期刊中將銀河系天文學分割成下列一組標準的子類型：
# 豐度－研究比氦更重元素的位置
# 核球－研究包圍著銀河系中心的隆起
# 中心－研究銀河系中心的區域
# 圓盤－研究銀河系的盤面 (銀河系大部分天體聚集的平面)
# 演化－銀河系的演化
# 形成－銀河系的形成
# 基本參量－銀河系的基本參量(質量、大小等)
# 球狀星團－在銀河系內的[[球狀星團|球狀星團]]
# 銀暈－包圍著銀河系的巨大暈體
# 運動學和動力學－恆星和星團的運動
# 核心－環繞著銀河系中心[[黑洞|黑洞]]的區域([[人馬座A*|人馬座A*]])
# 疏散[[星團|星團]]和星協－[[疏散星團|疏散星團]]和星協中的恆星
# 太陽周圍－鄰近太陽的恆星
# 恆星內容－銀河系內恆星的數量和類型
# 結構－構造(螺旋臂等)
