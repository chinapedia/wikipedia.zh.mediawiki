{{vgname|銀蓮公園|en=Thimbleweed Park}}是由[[羅恩·吉伯特|羅恩·吉伯特]]與[[蓋瑞·溫尼克|蓋瑞·溫尼克]]所合力創作的[[冒險遊戲|電子冒險遊戲]]，此遊戲專案在2014年期間開始於集資網站[[Kickstarter|Kickstarter]]上公開消息以進行[[群眾募資|群眾募資]]獲得開發資金，並於2017年3月30推出。<ref>{{cite web |url=https://www.kickstarter.com/projects/thimbleweedpark/thimbleweed-park-a-new-classic-point-and-click-adv/posts/1817770 |title=Release Date Locked In!! |work=[[Kickstarter|Kickstarter]] |date=2017年2月28日 |accessdate=2017年3月17日}}</ref>作品內容為模仿過去他們參與開發的《[[瘋狂大樓|瘋狂大樓]]》、《[[猴島的秘密|猴島的秘密]]》等遊戲之[[精神續作|精神續作]]。<ref name="Eurogamer">{{cite web |url=http://www.eurogamer.net/articles/2014-11-18-maniac-mansion-creators-launch-kickstarter-for-spiritual-successor-thimbleweed-park |title=Maniac Mansion creators launch Kickstarter for spiritual successor Thimbleweed Park  |work=[[Eurogamer|Eurogamer]] |author=Jeffrey Matulef |date=2014年11月18日 |accessdate=2016年4月20日}}</ref>

== 玩法 ==
遊戲操控介面上類似《瘋狂大樓》模式，即畫面下半部有著「開啟」（Open）、「交談」（Talk to）、「使用」（Use）等動作指令[[選單|選單]]以及道具列。玩家可以在遊戲畫面上觀察可能有著重要內容的事物後，來點選選單中的動作指令讓角色去執行。<ref name="Eurogamer"/>

與《瘋狂大樓》相同；《銀蓮公園》同樣可讓玩家切換操作數名不同角色闖關。<ref name="Eurogamer"/>

== 劇情 ==
內容為描述在一個名叫銀蓮公園（Thimbleweed Park）小鎮上發生一起神秘的謀殺案件，之後一對男女偵探芮（Ray）與雷耶斯（Reyes）來到小鎮後；開始試圖找出案件背後不為人知的謎團。<ref>{{cite web |url=http://www.shacknews.com/article/93856/thimbleweed-park-preview-welcome-home |title=Thimbleweed Park preview: Welcome Home |work=Shack News |author=Ozzie Mejia |date=2016年3月28日 |accessdate=2016年4月20日}}</ref>

== 開發 ==
=== 契機 ===
2014年11月18日，羅恩·吉伯特於自己的官方部落格「GrumpyGamer」上發佈一道消息，內容提到他數個月前與過去在遊戲商[[LucasArts|LucasArts]]共事的蓋瑞·溫尼克談天，當時他們提到過去1987年在LucasArts開發冒險遊戲《瘋狂大樓》的美好回憶；並喜愛那一段市面上還擁有著簡單、純真有趣的冒險遊戲作品年代，之後溫尼克進一步提議不如再製作一份像當時年代風格的冒險遊戲；並將它當成是「一款大家過去沒有玩到的LucasArts冒險遊戲作品」。而吉伯特認同溫尼克的想法，便在集資網站Kickstarter上發表出他們將進行開發遊戲《銀蓮公園》的募款計畫。<ref>{{cite web |url=http://grumpygamer.com/kickstarter |title=Please Join Us On Kickstarter |work=GrumpGamer |author=[[羅恩·吉伯特|羅恩·吉伯特]] |date=2014年11月18日 |accessdate=2016年4月20日 |deadurl=yes |archiveurl=https://web.archive.org/web/20160410072904/http://grumpygamer.com/kickstarter |archivedate=2016年4月10日 }}</ref>

之後《銀蓮公園》在2015年1月2日正式進行開發，吉伯特與溫尼克也將此遊戲作為向《[[雙峰_(電視劇)|雙峰]]》、《[[X檔案|X檔案]]》、《[[無間警探|無間警探]]》等懸疑電視影集致意的惡搞版。<ref>{{cite web |url=http://www.gamespot.com/articles/thimbleweed-park-is-the-classic-adventure-game-you/1100-6438864/ |title=Thimbleweed Park Is the Classic Adventure Game You Think You Remember |work=[[Gamespot|Gamespot]] |author=Scott Butterworth |date=2016年4月16日 |accessdate=2016年4月20日}}</ref>

=== 資金 ===
最初《銀蓮公園》在網站Kickstarter的募款金額預計需要37萬5千美元，之後到2014年12月18日已從15623名募款者上湊得62萬多美元的資金，<ref>{{cite web |url=https://www.kickstarter.com/projects/thimbleweedpark/thimbleweed-park-a-new-classic-point-and-click-adv |title=Thimbleweed Park: A New Classic Point & Click Adventure! |work=[[Kickstarter|Kickstarter]] |accessdate=2016年4月20日}}</ref>這些超出預期的資金讓《銀蓮公園》專案出現一些額外的開發計畫，例如替遊戲角色增加英語配音、補充德語和法語及義大利語等其它語言字幕，以及新增設計適用於智慧型手機平台的[[Android|Android]]與[[iOS|iOS]]遊戲版本。<ref>{{cite web |url=http://www.eurogamer.net/articles/2014-11-24-thimbleweed-park-succeeds-its-usd375k-goal-in-six-days |title=Thimbleweed Park succeeds its $375K Kickstarter goal in six days |work=[[Eurogamer|Eurogamer]] |author=Jeffrey Matulef |date=2014年11月14日 |accessdate=2016年4月20日}}</ref>

於募款期間結束後，扣掉支付給Kickstarter與負責販售此遊戲的[[亞馬遜公司|亞馬遜公司]]佣金費用；吉伯特與溫尼克獲得約56萬多美元可作為運用的資金。

=== 製作群 ===
在開發項目分配上，吉伯特負責了遊戲的程式以及[[遊戲引擎|引擎]]；溫尼克擔任了遊戲美術部分，他們兩人也合力構思遊戲的世界觀、劇情、與謎題。而過去同樣與他們在遊戲商LucasArts共事的[[大衛·福克斯|大衛·福克斯]]與馬克·費拉利（Mark Ferrari）也加入了《銀蓮公園》開發團隊，分別協助了程式與美術項目，<ref>{{cite web |url=http://mixnmojo.com/news/David-Fox-and-Ken-Macklin-Join-the-Thimbleweed-Park-Team |title=David Fox and Ken Macklin Join the Thimbleweed Park Team! |work=MixMojo |author=Jennifer |date=2014年12月10日 |accessdate=2016年4月20日}}</ref><ref>{{cite web |url=http://www.geekscape.net/we-talk-retro-gaming-art-with-thimbleweed-park-artist-mark-ferrari |title=We Talk Retro Gaming Art With ‘Thimbleweed Park’ Artist Mark Ferrari! |work=Geek Scape |author=Jonathan |date=2015年6月3日 |accessdate=2016年4月20日}}</ref>另來自[[西班牙|西班牙]][[巴塞隆納|巴塞隆納]]創作[[像素|像素]]風格繪圖的藝術家奧泰維·納瓦羅（Octavi Navarro）也為遊戲裡的美術內容分擔事項。<ref>{{cite web |url=http://pixelshuh.tumblr.com/about |title=About Me |work=Pixels Huh |author=Octavi Navarro |accessdate=2016年4月20日}}</ref>

=== 技術 ===
吉伯特在2014年8月期間便開始找尋適合開發此作品的遊戲引擎，考慮到避免花費時間將其它引擎改寫成符合自身要求的情況，吉伯特回頭採用過去自身以[[C語言|C]]/[[C++|C++]]研發；用在《[[大大城堡|大大城堡]]》與《[[狡猾海盜|狡猾海盜]]》上的[[2D|2D]]遊戲圖形引擎，並添加[[SDL|SDL]]函式庫讓此引擎能額外使用輸出入功能。而吉伯特原先計畫選用[[Lua|Lua]]程式語言來處理遊戲的[[手稿語言|腳本]]方面項目，之後因對於Lua的語法不上手改用了[[Squirrel|Squirrel]]語言。<ref>{{cite web |url=https://blog.thimbleweedpark.com/engine |title=Engine |work=Terrible Toybox, Inc. |author=[[羅恩·吉伯特|羅恩·吉伯特]] |date=2015年1月5日 |accessdate=2016年4月20日}}</ref>
