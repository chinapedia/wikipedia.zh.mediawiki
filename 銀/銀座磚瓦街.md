== 背景 ==
1872年2月，一場從和田倉門附近地帶燃起的大規模火災席捲了[[銀座|銀座]]、[[築地|築地]]一帶約95[[公頃|公頃]]大的地區，作為[[新橋車站_(日本)|鐵路終點站]]所在地的東京玄關——[[新橋_(東京都港區)|新橋]]也有災情。有鑑於此次災難，也為了替1869年以降、使東京取代[[京都|京都]]作為首都的整備計畫舖路，而在1872年實施了最初的[[都市計劃|都市計劃]]，師法歐美以磚、石等防火建材建造城市樓房的策略，著手進行改造計畫。該計畫的重點建設正是「銀座磚瓦街」<ref>{{cite book|
|title=『東京都史紀要』第3号「銀座煉瓦街の建設」
|year=1955
|page=p158、p486-488
|publisher=[[東京都|東京都]]
}}</ref>，而計畫核心人物則為時任東京府知事{{Link-ja|由利公正|由利公正}}及代理[[大藏省|大藏卿]][[井上馨|井上馨]]<ref name="中原"></ref>{{rp|p108}}。

同年3月，[[東京府|東京府]]府廳發布了〈磚瓦建築之御趣意告諭〉（{{lang|ja|れんが家屋建築、御趣旨告諭}}）<ref name="開化の街">{{Cite journal
|title=開化の街づくり―銀座煉瓦街計画
|journal=明治の東京計畫
|year=1990
|author=藤森照信
|pages=p8-p16
|accessdate=2014-02-02
}}</ref><ref>{{cite web
 |url         = http://www.linkclub.or.jp/~erisa-25/toke-nenpyo01.htm
 |title       = 建築物の耐震（耐火）基準の変遷
 |publisher   = リンククラブ（Linkclub）
 |accessdate  = 2014-02-02
 |deadurl     = yes
 |archiveurl  = https://web.archive.org/web/20081224045824/http://www.linkclub.or.jp/~erisa-25/toke-nenpyo01.htm
 |archivedate = 2008-12-24
}}</ref>，指出「...為免火災…此程類燒之各町、道路幅員需取擴、家屋全然以磚瓦石儘速著手建築...」<ref name="中原"></ref>{{rp|p108}}，3月22日時又發出另一份公告，表示將以地券收購遭到焚毀的地段、進行區劃重整之後再以舊價售與原地主。但由於土地評價的問題，使得收購計畫無法順利進行。原本的營建工程是預計在大藏省的監督下由東京府執行，但在由利公正的決定下，[[岩倉使節團|岩倉使節團]]也參加了計劃，並由大藏省建設局主導{{ref label|井上|註1|}}。當時銀座磚瓦街的營建模式是以官營{{ref label|官營|註2|}}為主、並委託當時替明治政府擔任[[御雇外國人|外籍顧問]]的[[英國|英國]][[建築師|建築師]]——[[湯馬士·華達士|湯馬士·華達士]]設計及指導。然而政府同時也認可私營模式（民間自費興建）的加入<ref name="藤森">{{cite book
|author=藤森照信
|title=『明治の東京計画』
|publisher=岩波書店
|year=1982
}}</ref>{{rp|p22-23,25}}。

==興建==
明治政府敲定了銀座磚街的計劃後，難以在市面上取得適合做為建材、並且符合工程需求量的磚塊<ref name="東京造街史"></ref>{{rp|p15}}。因此華達士決定將[[德國|德國]]工程師{{Link-de|佛德烈曲·埃德華·霍夫曼|Friedrich Eduard Hoffmann|佛德烈曲·霍夫曼}}發明的[[霍夫曼窯|霍夫曼窯]]引入日本<ref>{{Cite book
|title=恆春古城週邊文化資產整體規劃報告書
|chapter=第五章 古城週邊文化資產規劃
|publisher=[[屏東縣|屏東縣政府]]
|author=李永欣建築師事務所
|year=2008
|accessdate=2014-08-08
}}</ref>{{rp|p13}}，並在東京北部的{{Link-ja|小菅|小菅}}設立燒磚廠，由於廠內的窯室相連成環狀，故能使裝窯、燒成、冷卻等步驟得以不中斷地進行，進而得以大量產出銀座造街工程所需的紅磚<ref name="東京造街史"></ref>{{rp|p15}}。

磚瓦街的工事在1872年8月正式展開，其擁有兩項主軸環節：第一是以街道的路幅拓寬為重點實施重建，第二則是以防火效果較佳的磚瓦替沿街兩側的樓房舖設屋頂。政府耗資巨額經費——總預算的足足1/27，投入銀座磚瓦街的建設<ref name="Ginzaofficial">{{cite web
|url=http://www.ginza.jp/history?lang=en
|title=GINZA history - Ginza in the Meiji and Taisho eras
|publisher=Ginza Official – 銀座公式ウェブサイト
|accessdate=2014-02-02
}}</ref>。1873年時在拓寬的街道兩邊已出現了兩層樓高的聯排[[洋房|洋房]]，一般認為[[倫敦|倫敦]]的[[攝政街|攝政街]]是這條新街道的參考典範<ref name="藤森"></ref>{{rp|p37}}。

第一期工程結束後，東京府開始將此期工程內完工的洋房出售，但因為屋價過高，而使得申請的市民很少。雖然因此放寬並後延了房屋費的繳納期限，但也因此使成本回收變得更加困難。一期工程之外的計劃中建設也縮減了工程規模，政府也因為面臨市民反彈，而被迫放棄木挽町以東的市街改造計畫，最後整個銀座磚瓦街於1877年全面峻工，僅有銀座通（今[[中央通_(東京都)|中央通]]）一帶為中心的市街得到改造<ref name="中原"></ref>{{rp|p108}}。

==設計==
[[File:Ginza_in_1880s.JPG|thumb]]
完工時的銀座磚瓦街幹道從原本的12.6公尺拓寬至27公尺，約為原本路寬的兩倍，並且在車道的兩旁舖設了[[人行道|人行道]]、藉此實現了人車分流的設計。當時的人行道同樣由磚砌成，防止過路者的衣裝下襬在雨天時沾濕，車道的路面材質則為砂石。沿線設有煤氣[[路燈|路燈]]、增強夜間照明，以及遮陽用的[[櫻花|櫻花樹]]、[[松樹|松樹]]和[[楓樹|楓樹]]等[[行道樹|行道樹]]<ref name="東京造街史"></ref>{{rp|p16}}。街道交錯排列成棋盤狀、劃分出方型的街區<ref name="Ginzaofficial"></ref>。

磚瓦大街兩邊的房屋採用[[喬治亞式建築|喬治亞式建築]]的外觀<ref name="Ginzaofficial"></ref>，二樓的[[陽台|陽台]]由[[柱式|羅馬式圓柱]]架高，覆蓋了一樓的走廊和店面，形成了類似[[騎樓|騎樓]]的設計，提供市民避雨行走的空間。初期的樓房建材除了磚瓦以外還包括石塊，但後者卻因為磚塊的量產而逐漸減少使用。建築支撐結構則是採用歐美式的木頭桁架<ref name="東京造街史"></ref>{{rp|p17}}。

== 使用情形 ==
[[File:Choya_Newspaper_Ginza.JPG|thumb]]]]
當時除了價格不菲之外，一些諸如「住進了磚造建築，身體就會發青腫脹而斃命」的[[迷信|迷信]]說法也使得市民卻步，讓這些洋式磚屋在落成初期不得青睞，空置者相當多。但之後便有[[報紙|報社]]和進口品經銷商等類型的店家入駐，西服店、鐘錶行、家具店、西洋食堂和理髮廳也朝此聚集，逐漸升級成一個[[商圈|商圈]]、扮演了[[西洋文明|西洋文明]]之窗的角色，並發展成為當時的新興鬧區，超越了[[江戶時代|江戶時代]]的商業中心[[日本橋_(東京都中央區)|日本橋]]<ref name="ライブラリー版">{{Cite book
|author=藤森照信
|title=『明治の東京計畫』同時代ライブラリー版
}}</ref>{{rp|p41-43}}<ref name="東京造街史"></ref>{{rp|p18-19}}。

第一位搬入銀座磚瓦街的業主是[[大垣市|大垣]]出身的平野豐次郎，他於1883年開設了平野茶店（銀座4丁目11番地，現址為[[三越百貨|三越百貨]]銀座店）。該店也是首家裝設了當代文明利器——[[電話|電話]]的民間用戶，號碼為6號（1號至5號均為政府官廳機構）。木村屋麵包鋪與平野園共用同一地址<ref>{{Cite web
 |url        = http://www.hiranoen.jp/user_data/history.php
 |title      = 銀座平野園の歴史
 |publisher  = 銀座平野園
 |accessdate = 2014-02-04
}}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。其對面曾是{{Link-ja|朝野新聞|朝野新聞|朝野新聞社}}的銀座4丁目辦公處舊址，之後{{Link-ja|服部鐘錶店|服部時計店}}買下了新聞社的用地、準備建造新分店，1895年時正式開店營業。服部鐘錶店的位置在後來設置了成為銀座象徴的[[和光百貨|和光百貨]][[鐘樓|鐘樓]]<ref>{{Cite web
|url=http://www.wako.co.jp/clock_tower/index.html
|title=時計塔の歴史
|publisher=銀座・和光
|accessdate=2014-02-03
}}</ref>。斜對面則是17世紀就已創立的老牌文具店——{{Link-ja|鳩居堂|鳩居堂}}，販賣筆墨及香等各種傳統文具。

=== 20世紀以後 ===
[[File:Ginza_after_Great_Kanto_earthquake.JPG|thumb]]
進入了明治末期及[[大正|大正時代]]以後，磚瓦街上的洋房逐漸被住戶與店家改為和式風格，房外掛上門簾，內部也添加了[[榻榻米|榻榻米]]等家具，覆蓋了原本的洋味。街邊的行道樹被換成了[[柳樹|柳樹]]，成了銀座的新標誌，但在[[東京市|東京市]]市長[[後藤新平|後藤新平]]任內，為了拓寬車道的計畫，而決定改種[[銀杏|銀杏]]當成新的行道樹樹種，原本的柳樹則在1921年被除去<ref name="Ginzaofficial"></ref>。

1923年9月發生關東大地震時，磚瓦砌成的建築物在強大的破壞力之下形成嚴重龜裂或崩塌，因震災引發的後續大火也把房內的木造裝潢物及[[地板|地板]]通通焚毀。地震結束後，原本生氣蓬勃的銀座磚街上滿是瓦礫堆和殘破的聳立牆壁，以往風華蕩然無存，種植在平野園店前的、銀座最老的行道樹和其他的銀杏樹也都被燒掉{{ref label|震災|註3|}}。

關東大地震過去後，銀座地區的店家聯合起來進行復興工作，搭建了許多臨時板屋，並於11月時同時開張。災後重建工作進展快速，1924年及1925年之後已有[[松坂屋|松坂屋]]、{{Link-ja|松屋百貨|松屋 (百貨店)|松屋}}等大型[[百貨公司|百貨公司]]進駐此地，1930年代到來之前，即重新榮登為日本最繁榮的市街之一。1932年時，柳樹再次成為銀座街頭的主力行道樹種，重新排列在大路沿線<ref name="Ginzaofficial"></ref>。
