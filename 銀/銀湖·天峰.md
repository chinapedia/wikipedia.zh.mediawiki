'''銀湖·天峰'''（英文：'''Lake Silver'''），前稱'''銀鋒·湖泊'''（英文：'''Lake W'''）<ref>[http://property.mpfinance.com/cfm/pa3.cfm?File=20080408/paa01/lab1.txt 信和 Lake W 中文命名銀鋒‧湖泊]，[[明報|明報]]，2008年4月8日</ref>是由[[港鐵公司|港鐵公司]]和[[信和置業|信和置業]]合作發展的一項地產項目。位於[[香港|香港]][[沙田區|沙田區]][[馬鞍山_(香港市鎮)|馬鞍山]][[烏溪沙站|烏溪沙站]]上蓋，後方為[[翠擁華庭|翠擁華庭]]，前方遠處為[[迎海|迎海]]及[[吐露港|吐露港]]，最遠可眺望深圳[[大鵬半島|大鵬半島]]及[[廣東大鵬液化天然氣接收站|秤頭角天然氣接收站]]一帶。整個項目佔地370,000[[平方呎|平方呎]]，已於2009年11月落成，11月24日入-{伙}-。銀湖·天峰分為商舖、幼稚園和[[住宅|住宅]]，住宅建於商舖及幼稚園之上。

而在2007年10月9日，九鐵物業部主管曾帶領[[深圳市地鐵集團|深圳市地鐵集團]]物業發展分公司人員，到本項目地盤視察建築進度，並向深圳地鐵人員介紹九鐵公司物業發展情況。

其廣告歌曲《Con Te Partiro》由[[意大利|意大利]][[盲人|盲人]][[歌手|歌手]][[安德烈·波伽利|安德烈·波伽利]]主唱。

== 住宅 ==
住宅項目由7座樓高39至47層物業組成，以“L”形排列，提供2,218個單位，平均單位面積76[[平方米|平方米]]，商場面積4,000平方米，停車位340個<ref>[http://www.mtr.com.hk/chi/properties/maonshan_wks.html 港鐵烏溪沙發展內容]</ref>。項目以大單位為主，2房單位起步點由700平方呎起，並設相連、複式、連天台戶等特色單位，全屋苑有逾20款不同的間隔類型。單位樓底高度達10呎，每戶均建有環保露台，客廳則配用落地玻璃。屋苑部份會所設施位於[[港鐵|港鐵]][[烏溪沙站|烏溪沙站]]上蓋，由兩小駁橋連接。

目前，此項目尚有約1674平方米的住宅單位未售，至今仍未有重推消息。

== 樓宇設計 ==
銀湖天峰每座高度不一，採用了楷梯式高度設計，1座最矮，各座順序地層遞上去，7座最高，遠觀與[[馬鞍山|馬鞍山]]山脊線配合。另外單位大小亦有考量，向北（1—3、5座）單位，因為會被[[迎海|迎海]]遮擋景觀，故31樓以下全數為細單位，31樓以上因為比迎海樓宇高度為高，故採用大單位設計。向東（7、8座）單位，因可享永久無遮擋景觀，故全數以巨單位入則。第6座位處屋苑L型之角位，擁有向北及向東外牆，故此座同時包含了上述之大小單位。

== 住客會所 ==
銀湖·天峰住客會所命名為「PALACE BY THE SEA」，總面積達360,000平方呎，為[[信和置業|信和置業]]旗下的樓盤[[御龍山|御龍山]]「Palazzo Derby Clubhouse」的 1.5倍。會所設有全港首創的私人島嶼（位於[[烏溪沙站|烏溪沙站]]天台），島上設3幢獨立董事大屋及多個泳池。會所還設有三間宴會廳，包括水伊甸宴會廳、藍寶皇室宴會廳和紅寶皇室宴會廳。其他設施包括50米長歐洲湖泊式戶外泳池、室內恆溫泳池、樹屋親子樂園、健身室、室內孩子樂園、園林燒烤、室內多用途運動場等。

住客會所原由Methanoia Studio設計，最初方案採用現代泰式酒店風格。但後期則改用與旗下的樓盤[[御龍山|御龍山]]一樣風格的歐洲皇殿式設計。

== 失實宣傳 ==
2009年11月3日，銀湖天峰電視廣告<ref>[http://www.youtube.com/watch?v=NYbE3oOqMAc 銀湖天峰電視廣告(一分鐘版本)]</ref><ref>[http://www.youtube.com/watch?v=A0XBGCKYnnU 銀湖天峰電視廣告(兩分鐘版本)]</ref>，被[[香港廣播事務管理局|香港廣播事務管理局]]裁定失實及誤導<ref>[http://www.hkba.hk/cn/complaints/complaint_bacc/bacc200910c.html 廣播事務管理局廣告 銀湖天峰電視廣告 裁決] {{webarchive|url=https://web.archive.org/web/20110716144708/http://www.hkba.hk/cn/complaints/complaint_bacc/bacc200910c.html |date=2011-07-16 }}</ref>。局方考慮過相關持牌機構呈交資料及申述後認為： '''i''' 物業的景觀對其價值和受歡迎程度非常重要。廣告可以富想像力的方式表達，但物業的重要的資料必須真確，不應具有誤導成分。廣告在結尾的畫面展示了有關物業的全景。這個結尾的畫面顯示該物業位於綠樹環抱的環境，而鄰近並無任何其他建築物，觀眾會以為，這是有關物業的真實描繪，以致廣告失實和誤導；以及 '''ii''' 有關持牌機構並沒有作出合理努力，以確定該廣告中的資料屬實。他們應可從發展商的《物業發展資料》中得悉該物業鄰近有其他屋苑，但廣告的結尾畫面卻沒有展示這些建築物。

== 問題 ==
2016年10月，有住戶指單位所附送[[西門子|西門子]]煮食爐的強化玻璃面板突然發生爆裂。西門子回覆指住戶平常欠缺清潔，因過熱而導致強化玻璃碎裂。唯爐頭說明書並無提供清潔及保養方法。港鐵證實有關事件，指「管理處已派出職員協助及記錄事件，住戶已聯絡爐具商及煤氣公司跟進，近期亦未有接獲同類事件報告」。專家認為是煮食爐老化所致。<ref>{{cite news|title=爐具強化玻璃接連碎裂 西門子：因清潔不足 事主：說明書沒指示|url=http://news.mingpao.com/pns/dailynews/web_tc/article/20161030/s00002/1477763126230|accessdate=2016-10-30|publisher=明報|date=2016-10-30}}</ref>
