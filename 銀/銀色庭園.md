《'''你的银色庭院'''》（{{jpn|j='''君の銀の庭'''|kn=きみのぎんのにわ}}），直譯為「你的銀-{}-色庭院」，台灣壓制版翻譯為《銀色-{}-庭園》，為[[Kalafina|Kalafina]]樂曲，由[[梶浦由記|梶浦由記]]作詞、作曲，是Kalafina的第14張[[單曲|單曲]]，在日本2013年11月6日由[[SME_Records|SME Records]]發售。

== 概要 ==
本曲為[[動畫電影|動畫電影]]《[[劇場版_魔法少女小圓|劇場版 魔法少女小圓 [新編] 叛逆的物語]]》的[[主題曲|片尾曲]]。為了令曲中表現的[[晓美焰|晓美焰]]的選擇變的曖昧不清，製作方需要用1個月以上的時間{{sfn|リスアニ!|p=29}}。

第2曲目《misterioso》也用為《劇場版 魔法少女小圓 [新編] 叛逆的物語》的插入曲。

獲選為NHK[[日本動畫100|日本動畫100]]的「最佳動畫歌曲100」第78名。

== 音樂性 ==
在此曲明確表現出童話形象的同時若隱若現的表現出當中痛苦的情況下，標題的「銀色庭園」表現出[[鹿目圓|鹿目圓]]心中的理想世界<ref>{{Cite journal|journal=[[声優アニメディア|声優アニメディア]]|issue=2013年12月号|pages=|author=|date=2013-11-09|publisher=[[学研パブリッシング|学研パブリッシング]]}}</ref>。在此曲中有其獨唱部分的[[窪田啓子|窪田啓子]]對於歌詞表示「帶出了破壞力、而且令聽到的人很大的衝擊」。{{sfn|リスアニ!|p=21}}在錄音過程中，[[政井光|政井光]]被指示在唱到「門被緊緊的閉上」的時候要以可怕的感覺唱出來{{sfn|リスアニ!|p=22}}。在收錄過程中曾經以可愛，強勢等不同腔調進行幾次錄音，在最後採用可愛腔調錄音的版本{{sfn|リスアニ!|p=22}}。

== 單曲釋出 ==
2013年11月6日在日本由SME Records發售。約在Kalafina的前作單曲「[[哈利路亞_(Kalafina單曲)|哈利路亞]]」一個月後釋出。

販賣形式為初回生産限定盤A（SECL-1417/8）・初回生産限定盤B（SECL-1419/20）・通常盤（SECL-1421）・期間限定生産盤（SECL-1422）4種釋出類型、初回盤同捆DVD和Blu-ray Disk收錄了本曲的[[音樂影片|PV]]。

專輯內第二首曲目《misterioso》，是[[窪田啓子|Keiko]]以在前作《[[劇場版 魔法少女小圓|劇場版 魔法少女小圓
［後篇］永遠的物語]]》的未來的位置上、以輕快，耀眼，充滿活力的感覺完成的。此曲標題的「misterioso」為[[義大利語|義大利語]]「神秘的」的意思{{sfn|リスアニ!|p=23}}。
