[[美国|美国]]'''[[佛蒙特州|佛蒙特州]]'''總共擁有14個[[縣_(美國)|縣]]。這14個縣轄下有255個政治單位或地區，其中有237個為[[鎮|城鎮]]、9個為[[城市|城市]]、5個為未組成社團區域、以及4個[[戈爾_(行政單位)|戈爾]]。佛蒙特州的每個縣均擁有自己的[[縣治|縣治]]，稱為「郡鎮」（shire town）。

1779年，佛蒙特州僅擁有兩個縣。當時，位於西部的縣為[[本寧頓縣|本寧頓縣]]，而位於東部的縣則為[[坎伯蘭縣_(佛蒙特州)|坎伯蘭縣]]<ref name="traits">{{cite web |url = http://genealogytrails.com/ver/county_info.html |title=Vermont County Information  |accessdate=2007-07-22 |work=Genealogy Trails}}</ref>。1781年，坎伯蘭縣分裂成四個縣，而其中的華盛頓縣則變成屬於[[新罕布什爾州|新罕布什爾州]]的縣。位於州最東北部的三個縣：[[艾塞克斯縣_(佛蒙特州)|艾塞克斯縣]]、[[奧爾良縣_(佛蒙特州)|奧爾良縣]]和[[喀里多尼亞縣|喀里多尼亞縣]]被統稱為「[[東北王國|東北王國]]」<ref>{{cite web |url = http://www.vtliving.com/northeastkingdom/#.Ucwl2flmAmU |title=Northeast Kingdom - Vermont Living Magazine|publisher=Vermont Living Magazine |author= |accessdate=2013-06-27}}</ref>。

佛蒙特州每個縣都有自己的[[聯邦資料處理標準|聯邦資料處理標準代碼]]（簡稱FIPS代碼）。FIPS代碼是由5個數字組成，首兩個數字是州份的代碼，而後三個數字則是縣份的代碼。例如，[[阿迪森縣|阿迪森縣]]的縣代碼為001，而佛蒙特州的州代碼為50，那麼阿迪森縣的FIPS代碼就是50001。儘管州中每個縣份的代碼都不同，但是因為美國不同縣的代碼可能會重複，因此所有縣份的FIPS代碼皆要在縣代碼前加上州代碼，以作區別。例如，佛蒙特州阿迪森縣的縣代碼為001，但[[新罕布什爾州|新罕布什爾州]][[貝爾納普縣_(新罕布什爾州)|貝爾納普縣]]和[[佛羅里達州|佛羅里達州]][[阿拉楚阿縣_(佛羅里達州)|阿拉楚阿縣]]的縣代碼皆為001。為了區別這些縣份，所以要在其FIPS代碼前加上州代碼。<ref name="epa">{{cite web|url=http://www.epa.gov/enviro/html/codes/vt.html |title=EPA County FIPS Code Listing |publisher=EPA |accessdate=2007-07-22 |deadurl=yes |archiveurl=https://web.archive.org/web/20090817052642/http://www.epa.gov/enviro/html/codes/vt.html |archivedate=2009-08-17 }}</ref>
