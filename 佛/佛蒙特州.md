'''佛蒙特州'''（{{lang-en|State of Vermont}}）是[[美国|美国]]第14个州，以其美丽的景色、[[奶制品|奶制品]]、[[枫糖浆|枫糖浆]]和激进的政治而著称。

佛蒙特總共擁有14個[[郡|郡]]。這14個郡，轄下有255個政治單位或地區，其中有237個為[[城鎮|城鎮]]、9個為[[城市|城市]]、5個為未組成社團區域。

== 历史 ==
最初[[易洛魁人|易洛魁]]和[[阿尔珙人|阿尔珙]]等[[印第安人|印第安部落]]居住在今天的佛蒙特州。1609年[[法国|法国]]探险家[[萨缪尔·德·尚普兰|萨缪尔·德·尚普兰]]占今天[[尚普兰湖|尚普兰湖]]地区为己有，并将其周围的山脉称为「绿山」（Les Verts Monts），今天的州名就是由此而来的。

1763年的[[1763年巴黎條約|巴黎条约]]结束了[[法国印第安人战争|法国印第安人战争]]，这块土地由此归[[英国|英国]]，其中一些地方有时也归[[纽约州|纽约州]]和[[新罕布什尔州|新罕布什尔州]]管理。[[伊桑·艾伦|伊桑·艾伦]]和他的[[綠山男孩|綠山男孩]]（Green Mountain Boys）首先反抗英国统治，后来与纽约州和新罕布什尔州作战，终于于1777年建立了一个独立的[[佛蒙特共和国|佛蒙特共和国]]（在1777年的前半年中称为新康涅狄格，7月改为佛蒙特）。在此期间[[佛蒙特宪法|佛蒙特宪法]]被起草和签定，这是[[北美洲|北美洲]]的第一部宪法。1791年佛蒙特加入美国，成为其第14个州。

== 法律与政府 ==
佛蒙特每两年选举州长和副州长，是美國唯二，州長任期只有兩年的州，另一個為[[新罕布什爾州|新罕布什爾州]]（其他州的州長任期都是四年）。现任州长[[菲爾·斯科特|菲利普·斯戈特]]从2016年开始上任。

在[[美国参议院|美国参议院]][[帕特里克·莱希|帕特里克·莱希]]（民主党）和[[伯纳德·桑德斯|伯纳德·桑德斯]]（无党派，參與民主黨黨團。）代表佛蒙特州。在[[美国众议院|美国众议院]]，民主党人Peter Welch代表佛蒙特州。

佛蒙特州以其独立和激进的政治根據地而著称。由1850年代至1970年代，共和黨一直掌控佛蒙特州，1854年－1963年共和黨一直是佛蒙特州的執政黨。直至1960年代後，民主黨逐漸在佛蒙特州取得優勢，佛蒙特州成為美國[[新英格蘭|新英格蘭]]地區最自由派的州份之一。1988年以前，共和黨一直取得該州的選舉人票，除了[[1964年民權法案|1964年民權法案]]通過那年。1992年以後，佛蒙特州一直支持民主黨候選人。

佛蒙特州政府在环境保护、社会福利和防止城市化等问题上始终坚持激进[[自由主義|自由主義]]的态度。佛蒙特州是首個美國州份采用了[[民事结合|民事结合]]，使同性伴侣可以享受[[婚姻|婚姻]]的同等褔利。在1999年的一次起诉中佛蒙特州最高法院决定按照佛蒙特州宪法，州政府要么允许同性伴侣结婚，要么向他们提供一个享有同等待遇的地位。州議會采用了第二个选择建立了民事结合的地位。该法律得到约半数选民的支持，通过了立法过程，由当时的民主党州长[[霍华德·迪安|霍华德·迪安]]签署。但也有一些选民对此大为不满，并将其不满在州参议院的选举中表达出来。

2009年佛蒙特州议会以超过三分之二的票数推翻了州长的否决，正式通过同性婚姻法案，成为全美第一个由议会投票通过而不是由最高法院裁定将[[同性婚姻|同性婚姻]]合法化的州份。2009年9月1日，佛蒙特州正式成为全美第四个将同性婚姻合法化的州份。

虽然佛蒙特州本身比较小，但州議會奉行[[两院制|两院制]]，但外来的政治家往往无法在佛蒙特州获得一个议席。1998年，一位79岁的当地人击败了一个来自[[麻萨诸塞州|麻萨诸塞州]]的百万富翁和共和党人。这位当地人只花了201[[美元|美元]]的选举费用，赢得了55%的选票，然后他将这个位置退给了参议员帕特里克·里伊。

前任[[美国总统|美国总统]][[卡尔文·柯立芝|卡尔文·柯立芝]]和[[切斯特·A·阿瑟|切斯特·A·阿瑟]]来自佛蒙特州，兩個都是共和黨人。

== 政治 ==
眾所周知佛蒙特人的獨立傾向。佛蒙特是曾經個別獨立的四個州之一（其他的州分別是德州，加州與夏威夷州）。在國家級的選舉結果中有時會與其他地區完全不同。尤其特別的，佛蒙特是唯一一個州投票給反共濟會黨的州，而佛蒙特由於早年一直支持共和黨，因此也是在總統選舉中4次皆無把選擇人票投票給[[小羅斯福|小羅斯福]]總統的仅有的兩個州中的一個（另一個是緬因州）。

== 地理 ==
佛蒙特州是[[新英格兰|新英格兰]]地区的一部分。它东邻新罕布什尔州，西邻纽约州，南邻麻萨诸塞州，向北是[[加拿大|加拿大]]的[[魁北克|魁北克]]省。其面积约为25000平方公里。[[康涅狄格河|康涅狄格河]]是它的東部边界。[[绿山山脉|绿山山脉]]从南向北纵贯全州。全州约77%的面积是森林，其他的是草地、高原、湖泊、池塘和藻泽湿地。

佛蒙特州的春季短促多雨，夏季凉爽，秋季多色，冬季非常寒冷。尤其北部异常寒冷，往往比同州南部地区冷十度。佛蒙特州是美洲东海岸地区的[[滑雪|滑雪]]圣地。

秋季，佛蒙特州的山丘被[[糖枫|糖枫]]树葉染成红色、橙色和金色。这些不同的颜色不是由不同的树种形成的，而是由不同的土壤和气候导致的。

== 经济 ==
在过去两个世纪中，佛蒙特州的人口大增又大减。一开始在这里定居的是农民、伐木人和猎人。许多农民后来迁移向西方的[[大平原|大平原]]地区。同时由于过度采伐林木业也下降。这些人口损失给佛蒙特州的经济带来了一定的打击，但同时也使这里的土地和森林得到恢复。由于佛蒙特州没有工业，它避免了一些20世纪工业带来的问题。今天佛蒙特州的大多数森林是再植森林。

牛奶工业是佛蒙特州最大的农业收入来源。佛蒙特州的奶制品被出口到全世界。佛蒙特州美丽的自然和宽容的气氛吸引了许多自由思想主义者、独一无二的公司和小工业。[[旅游业|旅游业]]、许多夏季宿营地、家具制造业和[[滑雪|滑雪]]是佛蒙特州财政来源的重要方面。许多人到这里来钓鱼和长途跋涉。一些著名的驯马表演很受欢迎。[[高尔夫球|高尔夫球]]近几年来发展很大。

[[若伦德|若伦德]]是美国传统的[[大理石|大理石]]雕塑中心。

近几年中佛蒙特州在考虑在廉价的、自然的土地上建造公寓。此外州政府试图防止丧失其奶制品工业。

== 人口 ==
2000年佛蒙特州有608,827居民。州府是蒙彼利埃，最大的城市是伯林顿。佛蒙特州是美国人口最少的州之一。大约96.8%的人是白人，0.5%是黑人，0.4%是印第安人，0.9%是亚洲人。按1997年的统计大约9.7%的人生活在贫困线以下。
