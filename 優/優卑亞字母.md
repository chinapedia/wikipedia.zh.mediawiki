'''優卑亞字母'''或'''庫邁字母'''是公元前8世紀至公元前5世紀使用的早期[[希臘字母|希臘字母]]的西支變體，尤其在[[優卑亞島|優卑亞島]]、[[雅典|雅典]]以西地區和[[意大利|意大利]]南部的[[希臘|希臘]]殖民地如[[庫邁|庫邁]]和[[伊斯基亞|伊斯基亞]]盛行，後來演化成包括[[拉丁字母|拉丁字母]]在內的[[古意大利字母|古意大利字母]]。在希臘，優卑亞字母在公元前4紀被標準希臘字母所取代，它是由[[愛奧尼亞希臘語|愛奧尼亞希臘語]]所使用的東支變體發展而成的。

== 字母表 ==
優卑亞字母屬於所謂的“西部”類型的地域（epichoric）字母，依據Kirchhoff（1887）的分類，在此範圍內它沒有標準（東部）希臘字母<math>\Xi\!</math> = /ks/、<math>\Chi\!</math> = /kʰ/ 和<math>\Psi\!</math> = /ps/，轉而有<math>\Chi\!</math> = /ks/ 和<math>\Psi\!</math> = /kʰ/。像多數早期變體一樣它缺乏<math>\Omega\!</math>，並把<math>\vdash\!</math>（heta）= /h/ 用於輔音、而非元音<math>\Eta\!</math> = /ɛː/。它還保留了古老字母<math>\Digamma\!</math>（digamma）= /w/ 和<math>\Coppa\!</math>（koppa）= /k/。第三個古老字母<math>{\land\!\!\land}\!</math>（san）= /s/ 和<math>\Sampi\!</math>（sampi）= /s/一般不用於書寫中，但仍作為字母表的一部分而傳抄，下面是詳表：<ref>[http://www.unicode.org/Public/UCA/latest/allkeys.txt allkeys.txt GREEK*LETTER] {{en}}</ref>

==比較==
優卑亞字母和[[腓尼基字母|腓尼基字母]]相比，增加<math>\Upsilon\!\Chi\!\Phi\!\Psi\!</math>並減去<math>\Xi\!</math>。<math>\Upsilon\!</math>和<math>\Chi\!</math>是[[腓尼基字母|腓尼基字母]][[Waw_(letter)|waw]]和samek的變異體，只有<math>\Phi\!</math>和<math>\Psi\!</math>是由希臘人發明。
:[[File:Nestor_Cup_Cumae.jpg|none]]
[[伊斯基亞|伊斯基亞]]發現的{{tsl|en|Nestor's Cup|Nestor's Cup}}，上面刻著公元前8世紀的庫邁字母。
:[[File:masiliana_tablet.svg|none]]
[[埃特魯斯坎字母|埃特魯斯坎字母]]的古老變異體跟優卑亞字母幾乎相同，不同的是出現腓尼基字母Ξ和字母<math>\Zeta\!</math>的形狀仍接近腓尼基字母 zayin。
