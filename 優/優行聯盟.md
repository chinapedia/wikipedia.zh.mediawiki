'''優行聯盟'''（{{lang-en|'''U-FLY Alliance'''}}）是一個於2016年1月18日正式成立的[[航空聯盟|航空聯盟]]，航點主要遍佈[[東亞|東亞]]及[[東南亞|東南亞]]。聯盟成員互相合作建立飛行網絡及航班接駁，以提升個別成員的競爭力。這除了是全球首個由[[低成本航空|低成本航空]]公司組成的聯盟，亦是2010年代繼[[價值聯盟|價值聯盟]]第二家成立的航空聯盟。<ref>{{cite news |language = 中文 |author =  |coauthors =  |url = http://hk.apple.nextmedia.com/realtime/finance/20160118/54660448 |title = 香港廉航聯盟成立 推$118機票飛中國 |publisher = 蘋果日報 |pages =  |date = 2016年1月18日 |accessdate = 2016年2月11日}}</ref><ref>{{cite news |language = 中文 |author =  |coauthors =  |url = http://finance.takungpao.com.hk/gscy/q/2016/0119/3270039.html |title = 全球首個低成本航空聯盟—優行聯盟正式成立 |publisher = 大公報 |pages =  |date = 2016年1月19日 |accessdate = 2016年2月11日}}</ref><ref>{{cite news |language = 中文 |author =  |coauthors =  |url = http://udn.com/news/story/7332/1451200-%E5%84%AA%E8%A1%8C%E8%81%AF%E7%9B%9F-%E5%9C%A8%E6%B8%AF%E6%88%90%E7%AB%8B |title = 優行聯盟 在港成立 |publisher = 聯合報 |pages =  |date = 2016年1月19日 |accessdate = 2016年2月11日}}</ref>

==簡介==
優行聯盟於2016年在香港成立，與其他航空聯盟相比，優行聯盟以低成本航空公司組成，通過與其他低成本航空公司連接飛行網絡，提供低票價前往亞洲多個旅遊的目的地。

==運作模式==
聯盟強調由獨立的低成本航空公司組成，採取獨立管理和操作模式，並專注於各自的本土市場發展。聯盟是通過各低成本航空公司的合作，擴大航點版圖，為現有客戶及將來計劃暢遊亞洲各國的旅客提供更多低成本的旅遊機會。
