'''社口號誌站'''，位於[[台灣|台灣]][[臺中市|臺中市]][[神岡區|神岡區]]，曾為[[臺灣鐵路管理局|臺灣鐵路管理局]][[神岡線|神岡線]]（現已廢止）通車後所增設的[[鐵路|鐵路]][[號誌站|號誌站]]，位於已停用的民族路[[平交道|平交道]]旁，已廢站。站址位置大約位於今神岡區民族路上<ref>{{cite web|language=zh-tw|url=http://kcchu.pixnet.net/blog/post/37428108-%E7%A5%9E%E5%B2%A1%E7%B7%9A-%E7%A4%BE%E5%8F%A3%E8%99%9F%E8%AA%8C%E7%AB%99|title=神岡線 社口號誌站|author=離家不出走|publisher=|date=2011-01-25|accessdate=2013-11-07}}</ref>。

== 歷史 ==
*1957年8月21日：設站<ref name="tra">{{cite web|language=zh-tw|url=http://www.railway.gov.tw/Tanzi/CP.aspx?SN=11290|title=潭子車站簡介|author=|publisher=臺灣鐵路管理局|date=|accessdate=2013-11-07}}</ref>。
*1961年5月15日：撤銷潭子-神岡間牌劵式及社口、神岡兩站之號誌設備<ref name="tra"/>。
*1999年7月1日：隨著神岡線停駛而廢止。

== 車站構造 ==
[[File:Shekou_Signal_Station_Yard_in_2014.JPG|thumb]]
*水泥站房一棟，站場設有兩股線供列車交會。

== 利用狀況 ==
*為有人號誌站，派駐站員處理行車業務。
*社口號誌站同時是維護神岡線的台鐵社口道班駐紮處。
*站房後方是台鐵道班的員工宿舍。
