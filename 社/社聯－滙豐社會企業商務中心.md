'''社聯－滙豐社會企業商務中心'''（'''{{lang-en|HKCSS-HSBC Social Enterprise Business Centre}}'''）是香港一間集[[香港政府|香港政府]]、商界和社會福利界的力量，扶助[[社會企業|社會企業]]成長。

該中心是透過[[滙豐銀行慈善基金|滙豐銀行慈善基金]]和[[社會福利署|社會福利署]]《[[攜手扶弱基金|攜手扶弱基金]]》合資，由[[香港社會服務聯會|香港社會服務聯會]]負責營運。為社會企業提供顧問服務、培訓計劃、產品推廣平台及協助投資者挑選合適社企等服務<ref>{{cite web |language=zh-hk |url=http://www.takungpao.com.hk/news/08/11/15/GW-990725.htm |title=政商界全力扶助社企 |publisher=[[大公網|大公網]] |date=2008-11-15 |accessdate=2011-07-25 }}{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

==主要項目==
'''專責為社會企業申請人提供協助'''

香港[[慈善團體|慈善團體]]可以向香港政府申請基金成立社企，該中心會專責為申請人提供協助。成立社企的目的，不但要協助弱勢社群<ref>{{cite web |language=zh-hk |url=http://paper.wenweipo.com/2011/05/16/ED1105160028.htm | title=賺錢幫助弱勢 嶄露頭角  |publisher=[[香港文匯報|香港文匯報]] |date=2011-05-16 |accessdate=2011-07-25}}</ref>，亦要賺取基本的利潤作營運。如果社會企業有利潤，只能留作營運基金，或作慈善用途<ref>{{cite web |language=zh-hk |url=http://hk.news.yahoo.com/%E5%AE%B6%E5%B8%B8%E8%82%89%E9%A4%85%E5%8A%A0%E9%AE%91%E9%AD%9A-013649146.html | title=助慈善團體創業  |publisher=[[明報|明報]] |date=2011-07-05 |accessdate=2011-07-25}}</ref>。

==例子==
社聯－滙豐社會企業商務中心致力為弱勢社群提供就業機會。在[[2010年10月|2010年10月]]，[[電影演員|影視紅星]][[曾志偉|曾志偉]]和[[戚美珍|戚美珍]]為了協助聾人就業<ref>{{cite web |language=zh-hk |url=http://www.hkcd.com.hk/content/2010-12/06/content_2647626.htm |title=社企齊鋪樂農傷健共融 |publisher=[[香港商報|香港商報]] |date=2010-12-06 |accessdate=2011-07-31 |deadurl=yes |archiveurl=https://web.archive.org/web/20141108123046/http://www.hkcd.com.hk/content/2010-12/06/content_2647626.htm |archivedate=2014-11-08 }}</ref><ref>{{cite web |language=zh-hk |url=http://paper.wenweipo.com/2010/10/05/EN1010050038.htm | title=戚美珍助失聰人士就業 |publisher=[[文匯報_(香港)|文匯報 (香港)]] |date=2010-10-05 |accessdate=2011-07-31}}</ref>，他們得到該中心的協助<ref>{{cite web |language=zh-hk |url=http://hk.news.yahoo.com/%E6%BB%99%E8%B1%90%E5%B9%AB%E7%A4%BE%E4%BC%81%E5%81%9A%E7%94%9F%E6%84%8F-223000719.html | title=滙豐幫社企做生意 |publisher=[[星島日報|星島日報]] |date= 2011-09-08 | accessdate=2011-09-08}}</ref><ref>{{cite web |language=zh-hk |url=http://www.facebook.com/video/video.php?v=10150264193529145&oid=143545595698458&comments | title=社會企業--樂農 |publisher=[[滙豐銀行慈善基金|滙豐銀行慈善基金]] |date=  |accessdate=2011-07-31}}</ref>，便成功開設社會企業素食餐廳《[[樂農|樂農]]》。該餐廳並透過慈善機構「[[龍耳|龍耳]]」聘請聾人<ref>{{cite web |language=zh-hk |url=http://www1.hk.apple.nextmedia.com/template/apple/art_main.php?iss_id=20101219&sec_id=4104&art_id=14780972 | title=生活存關愛：首間素食餐廳聘用聽障人士 |publisher=[[蘋果日報_(香港)|蘋果日報 (香港)]] |date=2010-12-19 |accessdate=2011-07-31}}</ref>，希望此舉能夠為聾人創造就業機會。
