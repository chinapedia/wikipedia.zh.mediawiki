'''戴資穎未穿羽協贊助球鞋遭懲處事件'''<ref group="註">2016年8月18日，是由[[中華民國教育部體育署|中華民國教育部體育署]]對於該事件的稱呼。</ref>，又稱'''戴資穎球鞋爭議事件'''，台灣媒體及民間戲稱整起事件為「球鞋門」<ref>{{Cite news|url=http://news.ltn.com.tw/news/life/breakingnews/1800005|title=戴資穎「球鞋門」事件 法界人士批：羽協凌駕國際規範|date=2016-08-19|accessdate=2017-05-03|publisher=自由時報}}</ref>，是發生於[[2016年夏季奧林匹克運動會|2016年夏季奧林匹克運動會]]期間的事件，以及其引發的後續爭議。整起事件以及自里約奧運前後發生的[[網球選手謝淑薇奧運教練問題事件|網球選手謝淑薇奧運教練問題事件]]和其他台灣運動員相關爭議，使台灣民眾愈發注重運動員的培養及發展環境，並加速《國民體育法》推動<ref>{{Cite news|url=http://www.peoplenews.tw/news/d10201bd-de2d-41dc-a018-7a38ca588599|title=戴資穎球衣風波 林德福承諾未來不再發生|date=2017-05-03|accessdate=2017-05-03|publisher=民報}}</ref><ref>{{Cite news|url=http://sports.ltn.com.tw/news/breakingnews/2056176|title=戴資穎球衣事件「助攻」《國民體育法》今初審通過|date=2017-05-03|accessdate=2017-05-04|publisher=自由體育}}</ref>。

== 背景 ==
台灣[[優乃克|優乃克]]於2010年起正式簽約贊助[[中華民國羽球協會|中華民國羽球協會]]，合約中要求選手若以國家代表隊身分出賽時，必須穿著優乃克提供的服裝、球鞋、球拍及球袋。至2014年，已有至少三名選手及一名教練遭羽協懲處<ref>{{Cite web|url=http://sports.ltn.com.tw/news/breakingnews/1801203|title=與YONEX簽約以來 羽協對3球員1教練開罰|date=2016-08-20|accessdate=2016-08-23|publisher=自由體育}}</ref>，因選手必須透過中華羽協才能報名參加[[國際羽聯|國際羽聯]]所舉辦之賽事，迫使選手因此就範<ref>{{Cite web|url=http://sports.ltn.com.tw/news/breakingnews/1803090|title=羽協如何逼迫選手就範？ 著名球評舉3懲處案例|date=2016-08-22|accessdate=2016-08-23|publisher=自由體育}}</ref>。

[[臺灣|臺灣]][[羽球|羽球]]選手[[戴資穎|戴資穎]]為[[勝利體育|勝利體育]]所贊助之選手，但[[中華民國羽球協會|中華民國羽球協會]]則與優乃克簽約，羽協因此要求[[2016年夏季奧林匹克運動會中華台北代表團|中華台北代表團]]參加[[2016年夏季奧林匹克運動會|2016年夏季奧林匹克運動會]]的羽球選手身上不能有其它廠商的商標。<ref>{{Cite web|title=比賽鞋行不行? 戴資穎恐遭羽協處分? 臉書訴心情吞下去！|publisher=民報|url=http://www.peoplenews.tw/news/f3dacaf8-4492-46f4-9f4c-6cdbe05acd7e|date=2016-08-15|author=|accessdate=2016-08-22|language=zh-hant}}</ref>

== 經過 ==
2016年8月16日，在[[巴西|巴西]][[里約熱內盧|里約熱內盧]]傳出因戴資穎未穿著中華羽協贊助商所提供之服裝參加[[2016年夏季奧林匹克運動會羽毛球女子單打比賽|2016年夏季奧林匹克運動會羽毛球女子單打比賽]]，中華羽協可能會對其做出懲處；而戴資穎表示由於自己兩腳尺寸不同，因此[[勝利體育|勝利體育]]為她特製一雙沒有商標的球鞋<ref>{{Cite web|url=http://sports.ltn.com.tw/m/news/breakingnews/1796799|title=奧運》未穿官方贊助商球鞋 羽協考慮懲處戴資穎|date=2016-08-16|accessdate=2016-08-17|publisher=自由體育}}</ref>。

2016年8月17日，中華羽協秘書長[[蔡鴻鵬|蔡鴻鵬]]第一次出面說明，並指出9月將會召開選訓會議和紀律委員會，決議是否要對戴資穎進行懲處<ref>{{Cite web|url=http://www.peoplenews.tw/news/3bb82552-34a9-4604-af25-81ee095e80ec|title=是否開罰 羽協不鬆口：要是其他選手比照 怎辦？|date=2016-08-17|accessdate=2016-08-23|publisher=民報}}</ref>。

2016年8月19日，由於不堪輿論壓力，中華羽協發出新聞稿表示為及早平息風波，將會在23日召開理監事會；而優乃克總公司在臉書致歉並表明會調查此次事件細節<ref>{{Cite web|url=http://www.peoplenews.tw/news/d39fa332-46b8-44e2-b940-f90359493e7c|title=盼止血！羽協23日開理監事會 YONEX總公司道歉、調查始末|date=2016-08-19|accessdate=2016-08-23|publisher=民報}}</ref>。

2016年8月22日，[[臺灣|臺灣]][[羽球|羽球]]選手[[廖冠皓|廖冠皓]]向[[民主進步黨|民主進步黨]][[立法委員|立法委員]][[黃國書|黃國書]]陳情，將在[[2014年亞洲運動會羽毛球男子團體比賽|2014年亞洲運動會羽毛球男子團體比賽]]奪得銅牌的[[國光獎金|國光獎金]]用以繳清罰款[[新臺幣|新臺幣]]30萬元<ref>{{Cite web|title=不只戴資穎 廖冠皓向立委陳情同遭遇|publisher=中央廣播電臺|url=http://news.rti.org.tw/news/detail/?recordId=290285|date=2016-08-22|author=蕭照平|accessdate=2016-08-23|language=zh-hant|deadurl=yes|archiveurl=https://web.archive.org/web/20160826050957/http://news.rti.org.tw/news/detail/?recordId=290285|archivedate=2016年8月26日|df=}}</ref>。另外，[[2012年夏季奧林匹克運動會羽毛球女子雙打比賽|2012年夏季奧林匹克運動會羽毛球女子雙打比賽]]球員[[程文欣|程文欣]]在比賽期間用膠帶貼住優乃克商標，協會請求公司原諒，對於選手以口頭警告；[[2013年世界青年羽毛球錦標賽|2013年世界青年羽毛球錦標賽]]選手[[王子維|王子維]]跟[[廖哲宏|廖哲宏]]比賽時穿贊助商勝利牌服裝，球拍的拍面塗上勝利品牌商標，最後羽協對王子維罰款新臺幣30萬元、廖哲宏罰款新臺幣10萬元、帶隊教練[[鄭永成|鄭永成]]罰款新臺幣5萬元。<ref>{{cite news|title=沒用贊助球拍挨罰30萬及禁賽　廖冠皓：用銅牌獎金繳清|publisher=東森新聞|url=http://www.ettoday.net/news/20160822/760195.htm|date=2016-08-22|author=翁嫆琄|accessdate=2016-09-06|language=zh-hant}}</ref>

2016年8月23日，中華羽協理事長[[程俊琅|程俊琅]]在理監事會議後向社會大眾鞠躬道歉，表示不會懲處戴資穎，並宣布將取消官方贊助合約中球拍和球鞋的要求<ref>{{Cite web|title=羽協理事長鞠躬道歉 取消官方贊助球鞋、球拍要求|publisher=自由體育|url=http://sports.ltn.com.tw/news/breakingnews/1804123|date=2016-08-23|author=卓佳萍|accessdate=2016-08-23|language=zh-tw}}</ref>。

== 各界反應 ==
;官方
*[[體育署|體育署]]發布聲明表示，在6月中旬要求羽協一切以奧運參賽備戰為重，不能因此事件懲處選手；有關協會與選手雙方贊助商合約規範有所衝突部分，應參考國際案例及維護選手權益原則，建立具有共識性的規範機制，並且未來將邀集法律專家，提供專業法務諮詢，並配合國民體育法第八條訂定相關輔導辦法，健全體育團體運作機制<ref>{{Cite web|title=戴資穎球鞋爭議 體育署：不應懲處選手|publisher=中央社|url=http://www.cna.com.tw/news/aspt/201608180428-1.aspx|date=2016-08-18|author=林宏翰|accessdate=2016-08-23|language=zh-tw}}</ref>。

;體育界
*台灣網球國手[[盧彥勳|盧彥勳]]在社群網站上以自身經驗舉例說明，表示球員在球場上是穿著自己簽約品牌的服裝及球鞋，在一些抽籤儀式、開幕儀式、頒將儀式會套上國家隊的隊服。並表示對於中華羽協在與廠商簽約時，沒有替選手維護權益而只看到自身利益表示非常不解，認為如果選手在賽場上穿著不熟悉的裝備，還沒打已經輸了一半<ref>{{Cite web|title=戴資穎球鞋爭議 盧彥勳批羽協只顧利益|publisher=中央社|url=http://www.cna.com.tw/news/aspt/201608195003-1.aspx|date=2016-08-19|accessdate=2016-08-23|language=zh-tw}}</ref>。

;政界
*[[親民黨|親民黨]]黨團總召[[李鴻鈞_(立委)|李鴻鈞]]表示，他認為中華民國體育的發展，相較於[[新加坡|新加坡]]、[[韓國|韓國]]，顯然有待加強，政府應成立「財團法人運動產業發展研究院」，並且在體育方面任何的補助、推廣、軟硬體設備的建構，都必須透明化、納入監督機制；此外，還要籌設國家級體育選手的生活照顧基金<ref>{{Cite web|title=羽協揚言懲處案 立委：修法納公益理事|publisher=台灣醒報|url=https://anntw.com/articles/20160822-9syx|date=2016-08-22|author=林亭妤|accessdate=2016-08-23|language=zh-hant}}</ref>。

== 影響 ==
2017年4月27日，立法院教委員針對《國民體育法》草案第二十條進行諸多激烈討論，立委不諱言指出第二十條第四項就是「戴資穎條款」，因為戴資穎在奧運比賽的鞋子問題，要讓國內選手更有保障，不能讓協會以不正當理由去處分選手。「戴資穎條款」內文為：「特定體育團體團代表國家參加國際運動賽會，其與合作廠商訂定之贊助契約，應參考國際慣例與考量選手比賽之需要及權益為之；選手有個別之贊助廠商者，特定體育團體、選手及雙方贊助商，應於比賽之前協商，並尊重運動員之特殊需求，不得對運動員有顯失公平之約定。」<ref>{{cite news|title=「戴資穎條款」過關 選手享有個人贊助權利|publisher=中時電子報|url=http://www.chinatimes.com/realtimenews/20170427005261-260403|date=2017-04-27|author=陳志祥|accessdate=2017-04-27|language=zh-hant}}</ref><ref>{{cite news|title=「戴資穎們」的煩惱 修體育相關法律有解？|publisher=信傳媒|url=https://www.cmmedia.com.tw/home/articles/3570|date=2017-04-27|author=陳彥驊|accessdate=2017-04-27|language=zh-hant}}</ref>

時任中華羽協秘書長同時也是[[中華台北羽球大師賽|中華台北羽球大師賽]]主要贊助商冠鈞公司負責人的[[蔡鴻鵬|蔡鴻鵬]]，因受到此事件影響而去職，造成該項賽事資金出現缺口，中華羽協遂決定2017年起停辦該賽事<ref>{{cite news|title=贊助抽腿 4百萬蒸發 台北大師賽明年停辦|url=http://www.appledaily.com.tw/appledaily/article/sports/20161013/37414602/|agency=蘋果日報|date=2016-10-13}}</ref>，[[2017年世界羽聯大獎賽|2017年世界羽聯大獎賽]]也確定無此項賽事。
