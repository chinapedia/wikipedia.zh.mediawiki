'''戴相龙'''（{{bd|1944年|10月1日|catIdx=D戴相龙}}），[[江苏|江苏]][[仪征|仪征]]人，原[[中央财政金融学院|中央财政金融学院]][[会计|会计]]系毕业，高级[[经济师|经济师]]、[[研究员|研究员]]、[[博士生导师|博士生导师]]。1973年5月加入[[中国共产党|中国共产党]]；是中共第十四届中央候补委员，第十五、十六、十七届中央委员。曾历任[[中国人民银行|中国人民银行]]行长、[[天津市市长|天津市市长]]，[[全国社会保障基金理事会|全国社会保障基金理事会]]理事长等职。现任[[中国国际经济交流中心|中国国际经济交流中心]]副理事长<ref>{{cite web|url =http://www.cciee.org.cn/NewsInfo.aspx?NId=202|title =中国国际经济交流中心领导成员名单|publisher =中国国际经济交流中心网站|accessdate =2012-08-20|deadurl =yes|archiveurl =https://web.archive.org/web/20120926025447/http://www.cciee.org.cn/NewsInfo.aspx?NId=202|archivedate =2012-09-26}}</ref>，[[南开大学|南开大学]]教授<ref>{{cite web |url = http://www.022net.com/2012/5-22/463140322669441.html |title = 戴相龙到南开大学参加博士生毕业论文答辩会 |publisher = 人民网 |accessdate = 2012-08-20 |language = zh-cn |deadurl = yes |archiveurl = https://web.archive.org/web/20120728040228/http://www.022net.com/2012/5-22/463140322669441.html |archivedate = 2012-07-28 }}</ref>。著有《戴相龙金融文集》<ref name="戴相龙金融文集">{{cite web |url = http://news.xinhuanet.com/newscenter/2008-12/13/content_10498782.htm |title =《戴相龙金融文集》问世 |publisher = 新华网 |accessdate = 2012-08-20 |language = zh-cn}}</ref>。

== 基层历练 ==
1968年，自[[中央財政金融學院|中央財政金融學院]]（今[[中央财经大学|中央财经大学]]）会计系财务会计专业毕业后，戴相龙被分配到[[云南省|云南]]圭山煤矿工作；历任财务科会计、宣传科干事、工区党支部副书记，省煤炭工业局政治部干事。1979年，调回原籍江苏省工作。

在江苏工作期间，戴相龙由[[中国人民银行|中国人民银行]]江苏省分行的一名普通干部做起；历任[[中国农业银行|中国农业银行]]江苏省分行拨款处副处长，[[江苏省|江苏省]][[丰县|丰县]]副县长，[[中国农业银行|中国农业银行]][[江苏省|江苏省]]分行副行长等职。1985年至1995年，他先后出任[[中国农业银行|中国农业银行]]副行长，[[交通银行|交通银行]]总经理、副董事长、党组书记，[[中国人民银行|中国人民银行]]副行长、党组副书记等职。<ref>{{cite web | url=http://news.xinhuanet.com/ziliao/2002-03/01/content_295750.htm | title=戴相龙官方简历 | publisher=新华网 | accessdate=2012-08-19 | language=zh-cn}}</ref>

== 执掌央行 ==
1995年6月起，戴相龙出任[[中国人民银行|中国人民银行]]行长、党组书记。在担任[[中国人民银行|央行]]第十任[[中国人民银行|行长]]期间，经历了[[亚洲金融危机|亚洲金融危机]]，戴相龙推动银行管理体制和经营机制改革；<ref name="尽快转换角色迎接挑战">{{cite web | url=http://news.sina.com.cn/c/2003-01-24/211338996s.shtml | title=天津市新市长戴相龙：尽快转换角色迎接挑战 | publisher=新浪网 | accessdate=2012-08-20}}</ref>并曾先后主导连续九次降低存贷款利息，被戏称为“降息行长”；2001年6月起，还兼任[[中共中央金融工作委员会|中共中央金融工作委员会]]副书记。2002年12月，在[[朱镕基|朱镕基]]“[[内阁|内阁]]”任期仅剩三个月时间时，戴相龙离开[[中华人民共和国国务院|国务院]]，调[[天津|天津]]工作。<ref name="星岛资讯" />

== 天津市長 ==
2002年12月至2007年12月28日，戴相龙先后出任[[中共天津市委|中共天津市委]]副书记，天津市代市长、市长等职，被媒体称为“金融市长”。<ref name="银行并不是看我的面子">{{cite web| url=http://news.hexun.com/2007-10-26/100967739.html| title=天津市市长戴相龙：银行并不是看我的面子| publisher=和讯网| date=转自《华夏时报》| accessdate=2012-08-19| language=zh-cn| deadurl=yes| archiveurl=https://web.archive.org/web/20130424193713/http://news.hexun.com/2007-10-26/100967739.html| archivedate=2013-04-24}}</ref>2003年1月24日，戴相龙以全票当选为天津市市长<ref>{{cite web | url=http://hsb.hsw.cn/gb/newsdzb/2003-01/25/2003-01-25-06gnxw9.htm | title=戴相龙昨全票当选天津市长 | publisher=新华网 | accessdate=2012-08-20}}</ref>；负责[[天津市人民政府|天津市人民政府]]全面工作，并分管财政、审计工作。戴相龙任天津市长的五年，在与诸多城市竞争的情况下，空客A320系列飞机总装线、[[中新生态城|中新生态城]]、百万吨乙烯等国际合作项目，都选择了落户[[天津滨海新区|天津滨海新区]]。

在任[[天津市|天津市]]市长期间，戴相龙展开改革及整顿，使当地的腐败问题得到一定程度上的治理；并积极推进“[[天津滨海新区|天津滨海新区]]”的开发和建设。<ref name="戴相龙当选市长">{{cite web | url=http://news.sohu.com/31/34/news205963431.shtml | title=房凤友当选天津市人大主任 戴相龙当选市长 | publisher=搜狐网 | accessdate=2012-08-19 | language=zh-cn}}</ref> 2006年5月，[[中华人民共和国国务院|国务院]]正式批准滨海新区为[[国家综合配套改革试验区|国家综合配套改革试验区]]；戴相龙明确为天津设计了”不争金融中心，而是建立与中国北方经济中心相适应的金融服务体系和金融改革创新基地“的建设目标。<ref>{{cite web | url=http://finance.people.com.cn/GB/1037/4814249.html | title=天津市长戴相龙否认天津"金融中心"地位 | publisher=网易 | accessdate=2012-08-20 | language=zh-cn}}</ref>
[[File:CBD_of_TBNA_2011.jpg|戴相龙任天津市长期间规划并开始建设的[[滨海新区]][[于家堡金融区|于家堡金融区]]|thumb|400px|left]]
此外，戴相龙还一直在推动[[渤海银行|渤海银行]]的筹建工作。2005年12月31日，[[渤海银行|渤海银行]]正式挂牌成立；这是自1996年后[[国务院|国务院]]批准的第一家全国性股份制银行。2006年，由戴相龙主导的”渤海产业投资基金“挂牌成立，是[[国家发改委|国家发改委]]批准的中国首支人民币产业投资基金。<ref name="社保掌门人戴相龙的“PE投资经”">{{cite web | url=http://stock.hexun.com/2010-06-01/123851453.html | title=言行合一 社保掌门人戴相龙的“PE投资经” | publisher=和讯网 | work=转自《上海证券报》 | accessdate=2012-08-19 | language=zh-cn}}</ref>而天津港的货物吞吐量，已由2003年不到1亿吨跃升至3亿吨，成为中国北方最大的港口；而内地最大的保税港区天津港[[东疆保税港区|东疆保税港区]]也已经封关运作。<ref>{{cite web| url=http://news.hexun.com/2008-01-03/102596130.html| title=“前央行行长的天津市长”戴相龙这五年| publisher=和讯网| accessdate=2012-08-19| language=zh-cn| deadurl=yes| archiveurl=https://web.archive.org/web/20120114093406/http://news.hexun.com/2008-01-03/102596130.html| archivedate=2012-01-14}}</ref>

2007年10月，在[[中国共产党第十七次全国代表大会|中国共产党第十七次全国代表大会]]上，已经年届正部级官员退休年龄的戴相龙第三次当选[[中央委员会|中央委员会]]委员，但在级别上并未获得进一步晋升。两个月后的12月28日，在距任职整五年还有3天的时候，戴相龙辞去了天津市市长职务。<ref>{{cite web |url=http://news.enorth.com.cn/system/2007/12/29/002571507.shtml | title=天津市十四届人大常委会举行第四十二次会议 | publisher=北方网 | accessdate=2012-08-20}}</ref>时任[[中共天津市委|中共天津市委]]书记[[张高丽|张高丽]]在评价戴相龙时，说：
{{quote|戴相龙同志为加快天津的发展倾注了大量心血，对天津广大干部群众充满了深厚感情。对戴相龙同志所作出的贡献，天津人民是不会忘记的……向戴相龙同志表示崇高的敬意和衷心的感谢。<ref>{{cite web | url=http://finance.sina.com.cn/g/20080103/08004360222.shtml | title=戴相龙这五年 | publisher=新浪网 | accessdate=2012-08-19 | language=zh-cn}}</ref>}}

== 社保基金 ==
离开天津后，2008年1月30日，新华网公布[[中共中央|中共中央]]和[[国务院|国务院]]的决定，由戴相龙接替前财政部长[[项怀诚|项怀诚]]出任[[全国社会保障基金理事会|全国社会保障基金理事会]]理事长、党组书记。<ref name="星岛资讯">{{cite web | url=http://www.singtaonet.com/china/200801/t20080130_723370.html | title=戴相龙任社保基金理事长 掌五千亿元社保金 | accessdate=2012-08-19 | language=zh-cn}}</ref>在戴的运作下，全国社保基金成为中国国内最大的专业[[私募股权|私募股权]]投资机构；到2011年底，全国社保基金基金会管理的总资产达到8688.20亿元，承诺投资13只私募股权机构，承诺金额达195亿元。<ref>{{cite web | url=http://www.ssf.gov.cn/zyjh/ldjh/201204/t20120424_3680.html | title=在第四届全球PE北京论坛上的演讲 | publisher=全国社保基金理事会网站 | accessdate=2012-08-20 | language=zh-cn | deadurl=yes | archiveurl=https://web.archive.org/web/20120915085237/http://www.ssf.gov.cn/zyjh/ldjh/201204/t20120424_3680.html | archivedate=2012-09-15 }}</ref><ref>{{cite web | url=http://www.ssf.gov.cn/cwsj/ndbg/201206/t20120618_5601.html | title=2011年全国社会保障基金年度报告 | publisher=全国社保基金理事会网站 | accessdate=2012-08-20 | language=zh-cn}}</ref>

2013年3月，刚刚卸任财政部部长的[[谢旭人|谢旭人]]出任全国社保基金理事会理事长，戴相龙任党组书记；2014年1月，69岁的戴相龙，正式退休。<ref>{{cite web | url=http://news.ifeng.com/mainland/detail_2014_01/28/33452787_0.shtml| title=社保基金党组书记戴相龙退休 曾力主延长退休年龄 | publisher=凤凰网| accessdate=2014-01-28| language=zh-cn}}</ref>

== 温家宝亲信 ==
香港《[[东方日报_(香港)|东方日报]]》引述消息称戴相龙曾向时任中共中央总书记[[江泽民|江泽民]]状告总理[[朱镕基|朱镕基]]，因此激怒朱镕基，朱镕基内阁在任期仅剩3个月时，于2002年12月将戴相龙调离中央往[[天津|天津]]任市长。因为[[温家宝|温家宝]]是天津人的关系，其后出任国务院总理的温家宝等人成了戴相龙的后台，因此戴相龙在离任天津市长之职后，他又担任[[全国社会保障基金|全国社会保障基金]]理事会理事长等职。戴相龙家族在戴相龙担任[[中国人民银行|中国人民银行]]行长期间，通过买卖[[平安保险|平安保险]]股票暴富，与时任总理温家宝的家族一起赚大钱。与温家宝的家族和中国平安保险公司并列中国亿万富翁权贵俱乐部。<ref>{{cite web | url=http://cn.on.cc/cn/bkn/cnt/news/20150327/bkncn-20150327210849016-0327_05011_001_cn.htm| title=戴相龙传涉贪受查　被揭与温家宝家族赚大钱| publisher=东网| accessdate=2015-03-27}}</ref>
