'''戴旭'''{{BD|1964年|9月30日||catIdx=Dai,戴}}，[[河南省|河南省]][[民权县|民权县]]人，[[中国人民解放军空军|中国人民解放军空军]][[大校|大校]]，自主择业退役，现任[[民营企业|民营企业]][[华信能源|华信能源]]旗下的中华能源基金委员会战略分析师（化名：龙韬）。《[[环球时报|环球时报]]》[[军事评论员|军事评论员]]，以在媒体和网络上发表强硬和极端言论而著称。<ref>[https://amp.rfi.fr/cn/中国/20181209-鹰派学者为习近平下套-怒吼撞沉航南海美舰 鹰派学者疑为习近平下套 怒吼撞沉航南海美舰]</ref><ref name="戴旭：我掌握的资料没必要告诉你" />

== 生平 ==
毕业於空军电讯工程学院（现[[中国人民解放军空军工程大学|中国人民解放军空军工程大学]]电讯工程学院）和空军政治学院（现[[中国人民解放军南京政治学院|中国人民解放军南京政治学院]]上海分院），是一名军事评论员，并在《[[环球时报|环球时报]]》以及[[人民网|人民网]]、[[乌有之乡|乌有之乡]]等媒体开设了专栏。<ref>{{cite web|url=http://mil.huanqiu.com/Observation/2009-11/627685.html |title=著名军事专家谈中国空军发展60年 |website=环球网 |date=2009-11-10 |accessdate=2010-02-24 |language=zh-cn |deadurl=yes |archiveurl=https://web.archive.org/web/20091227123702/http://mil.huanqiu.com/Observation/2009-11/627685.html |archivedate=2009-12-27 }}</ref>

担任[[中国人民解放军国防大学|中国人民解放军国防大学]]教授。2010年12月，与[[郎咸平|郎咸平]]、[[郭一平|郭一平]]、[[张宏良|张宏良]]等九人一道，被选为“[[2010中国互联网九大风云人物|2010中国互联网九大风云人物]]”<ref>{{cite web|url =http://china.huanqiu.com/info/2011-12/2242062.html|title =环球时报年会嘉宾：国防大学教授戴旭|website =环球网|date =2011-12-07|language =zh-cn|deadurl =yes|archiveurl =https://web.archive.org/web/20120823023958/http://china.huanqiu.com/info/2011-12/2242062.html|archivedate =2012-08-23}}</ref>（“中国互联网九大风云人物”）。

== 作品 ==
戴旭出版过多部军事题材图书：
*《以血祭天：一个中国空军上尉关于现代战争的预言》
*《盛世狼烟：一个空军上校的国防沉思录》
*《20世纪世界空战》
*《海图腾》
*《读史随想》
*《当代军事经典理论点评》
*《C形包围》
*《2030肢解中国——美国全球战略与中国危机》

==争议言论==
戴旭时常通过[[新浪微博|新浪微博]]上其实名认证的账号发表对时事的看法，同时也不时发出极端言论被斥“没有人性”<ref>{{cite web |url = https://m.sohu.com/n/371874584/ |title = 热辣评:戴旭 请捡起你的人性 |publisher = 搜狐 |accessdate = 2013-07-08}}</ref>，因其一贯强硬的态度也被称为中国的“[[鹰派|鹰派]]”。<ref name="戴旭：我掌握的资料没必要告诉你" />

2012年9月28日，戴旭在新浪微博表示“[[比尔盖茨|比尔盖茨]]是[[共济会|共济会]]成员。他用[[疫苗|疫苗]]搞[[慈善|慈善]]，首先是在[[非洲|非洲]]推广，目的是以种族暗杀的方式，实行[[美国|美国]]和共济会对全世界的“[[计划生育|计划生育]]”。他到中国来，也是这个意思。如此而已，岂有它哉！”

2013年4月6日，戴旭在新浪微博表示，[[禽流感|禽流感]]和[[SARS|SARS]]是美国的生物心理武器，同时称中国“死不了几个，连中国车祸千分之一都不到。”此言一出即引起争议，其对不尊重生命的态度遭到网民批判，随后其删除了该言论。<ref>{{cite news|title=空軍大校戴旭：對禽流感不要高調 死不了幾個人|url=http://finance.people.com.cn/BIG5/n/2013/0408/c1004-21048399.html|publisher=人民網|date=2013-04-08|accessdate=2018-12-09}}</ref><ref name="戴旭：我掌握的资料没必要告诉你">{{cite web |url = http://www.infzm.com/content/89736 |title=“我本来就怀疑美国搞生物武器” 戴旭：我掌握的资料没必要告诉你 |publisher=南方周末 |accessdate=2013-07-08}}</ref><ref>{{cite web |url = http://news.qq.com/a/20130407/001261.htm |title=空军大校戴旭发微博：对禽流感不要高调 死不了几个 |publisher=腾讯网 |accessdate=2013-07-08}}</ref>
