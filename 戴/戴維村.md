戴維村（英語：{{lang|en|Davie Village}}）是[[加拿大|加拿大]][[英屬哥倫比亞省|英屬哥倫比亞省]][[溫哥華市|溫哥華市]]的[[同志村|同志村]]，位於西端區的戴维街，在[[布勒街|布勒街]]({{lang|en|Burrard Street}})與杰維斯街(Jervis Street)的中間。市內大多數同性戀夜店和酒吧都集中在戴维村。溫哥華戴維村每年一度的[[驕傲遊行|驕傲遊行]]是全國最大規模之一的同性戀活動，今年會開展在2016年7月31日（星期天）。<ref>{{cite web |year=2009 |url=http://bc.ctvnews.ca/milk-prot-g-praises-vancouver-pride-celebration-1.422070 |title=Milk Protege Praises Vancouver Pride Celebration |accessdate =2016-03-30}}</ref>

==歷史==
1999年由戴維街經濟協會命名，他們以第八屆（1887年到1889年）英屬哥倫比亞省省長{{tsl|en|Alexander Edmund Batson Davie||亞歷山大·戴維}}（{{lang|en|A.E.B. Davie}}）之名給整個地區取名為“戴維村”。<ref>{{cite web |year =2016 |url=http://www.biographi.ca/en/bio/davie_alexander_edmund_batson_11E.html |title=A.E.B Davie Biography |accessdate =2016-03-30}}</ref>

溫哥華戴維村每年舉行省內最大的同性戀驕傲遊行。<ref>{{cite web |year=2009 |url=http://bc.ctvnews.ca/milk-prot-g-praises-vancouver-pride-celebration-1.422070 |title=Milk Protege Praises Vancouver Pride Celebration |accessdate =2016-03-30}}</ref>驕傲遊行開始於1978年，從始至終都一直由溫哥華驕傲協會（[http://vancouverpride.ca/ Vancouver Pride Society]）舉辦。<ref>{{cite web |year=2016 |title=2016 Vancouver Pride Festival |url=http://vancouverpride.ca/content/event/38th-annual-vancouver-pride-parade-0 |accessdate=2016-03-30}}</ref>

==今日的戴維村==
沿街有各種各樣的商店、夜店、酒吧、及餐廳，他們不僅向同性戀人開放，而且歡迎所有其他的非同性戀的客人。 明星（[http://celebritiesnightclub.com/ Celebrities]）、數字（[http://www.numbers.ca/ Numbers]）、及源泉酒吧（[http://thefountainheadpub.com/ Fountainhead Pub]）等都稱得上是戴維村出名的夜店。<ref>{{cite web |title= Vancouver Gay Bars & Night Spots |url=http://www.mydavievillage.com/Bars.html |accessdate=2016-03-30}}</ref>溫哥華同性戀雙週報的報社[http://www.dailyxtra.com/ Daily Xtra]，建立於1971年，位於戴維村，給溫哥華同志提供新聞與娛樂。<ref>{{cite web |title=Daily Extra |url=http://www.thestar.com/entertainment/2015/01/14/gay-newspaper-xtra-to-stop-printing-go-digital-only.html |accessdate=2016-03-30}}</ref>社區（[http://qmunity.ca/ Qmunity]），溫哥華市的同性戀社區中心也在戴維村，這個社區給大多數的同性戀人提供了社交和娛樂的平台。

同性戀的標誌是彩虹，這條彩虹有七個顏色，表達同性戀及其他類人的多樣性。<ref>{{cite web |title=The Rainbow Flag |url=http://www.cs.cmu.edu/afs/cs.cmu.edu/user/scotts/bulgarians/rainbow-flag.html |accessdate =2016-03-30}}</ref>戴維村到處都有彩虹的圖像；窗戶、斑馬線、及路燈都有彩虹的紋飾。甚至連桑德曼溫哥華市中心酒店（[http://www.sandmanhotels.ca/hotels/vancouver-davie-street/ Sandman Hotel Vancouver])的樓上也畫了彩虹的標誌。<ref>{{cite web |title=Vancouver's Sandman Hotel paints massive rainbow on roof ahead of pride week |url=http://www.vancouversun.com/life/Vancouver+Sandman+Hotel+paints+massive+rainbow+roof+ahead+pride+week/10014973/story.html |accessdate=2016-03-30}}</ref>

==問題和爭議==
溫哥華同志村的社會特色正在面臨慢慢消失的危險。<ref>{{cite web |year=2014 |title=There goes gaybourhood: study finds enclaves "straightening"|url=http://www.vancouversun.com/technology/there+goes+gaybourhood+study+finds+enclaves+straightening/10069109/story.html |accessdate=2016-03-30}}</ref>據說，戴維村像北美洲別的同志村一樣，同性戀村的特質越來越少。主要原因來自於戴維村位置地理優越，那裡的房價近年一直飛漲，許多的同性戀人開始負擔不起戴維村昂貴的房價，並且慢慢搬出戴維村，然而越來越多的非同性戀人開始購買戴維村的房子。除此之外，越來越多的人開始接受同性戀人，<ref>{{cite web |url=http://bc.ctvnews.ca/there-goes-the-gayborhood-ubc-prof-says-davie-street-changing-not-dying-1.1945844 |title=There goes the gayborhood? UBC prof says Davie Street changing, not dying |accessdate=2016-03-30 ''British Columbia''}}</ref>同性戀文化也受到大多數人的歡迎。由於以上種種原因，戴維村的同性戀特質漸漸消失，原有的同性戀文化也慢慢被城市融合。<ref>{{cite web |title=Gay neighbourhoods "straightening" as LGBT community finds increased acceptance |year=2014|url=http://news.nationalpost.com/news/canada/gay-neighbourhoods-straightening-as-lgbt-community-finds-increased-acceptance |accessdate=2016-03-30}}</ref>

==未來的計劃==
溫哥華市政府預計用兩百三十萬加幣來建立吉姆·德華（Jim Deva)公共廣場。<ref>{{cite web |title=Jim Deva Plaza West End Vancouver |year=2016 |url=http://www.vancitybuzz.com/2015/12/jim-deva-plaza-west-end-vancouver/ |accessdate=2016-03-30}}</ref>
