'''戴文·青木'''（'''Devon Edwenna Aoki'''，{{bd|1982年|8月10日|catIdx=A}}）出生於[[美國|美國]][[紐約|紐約]]，是有[[英格兰人|英格兰]]、[[德意志裔人|德意志]]、[[日本人|日本]]血统的[[欧亚混血|欧亚混血]][[超級名模|超級名模]]和[[演員|演員]]。

== 早年 ==
戴文的父親是[[日裔美國人|日裔美國人]]企業家[[青木廣彰|青木廣彰]]，母親潘蜜拉·希爾伯格是[[德國人|德國人]]，哥哥是知名電子音樂家[[史蒂夫·青木|史蒂夫·青木]]。戴文從13歲開始模特兒生涯，她的乾媽將她介紹給英國[[超模|超模]][[凱特·摩絲|凱特·摩絲]]認識<ref>{{Cite web|url=https://www.harpersbazaar.com/tw/fashion/FashionModels/news/g560/devon-aoki-the-shortes-super-model/?slide=2 |title=非比尋常的袖珍超模傳奇－戴文青木Devon Aoki |first1=Celine |last1=Hsu |first2=Quenti |last2=Lu |date=2016-10-29 |accessdate=2018-11-09 |work=[[时尚芭莎|时尚芭莎]] |pages=2 |language=zh |deadurl=no |archiveurl=https://web.archive.org/web/20181109215136/https://www.harpersbazaar.com/tw/fashion/FashionModels/news/g560/devon-aoki-the-shortes-super-model/?slide=2 |archivedate=2018-11-09}}</ref>。

戴文14歲與英國知名模特兒經紀公司Storm Model Management簽約<ref>{{Cite web|url=https://www.harpersbazaar.com/tw/fashion/FashionModels/news/g560/devon-aoki-the-shortes-super-model/?slide=5 |title=非比尋常的袖珍超模傳奇－戴文青木Devon Aoki |first1=Celine |last1=Hsu |first2=Quenti |last2=Lu |date=2016-10-29 |accessdate=2018-11-09 |work=[[时尚芭莎|时尚芭莎]] |pages=5 |language=zh |deadurl=no |archiveurl=https://web.archive.org/web/20181109215419/https://www.harpersbazaar.com/tw/fashion/FashionModels/news/g560/devon-aoki-the-shortes-super-model/?slide=5 |archivedate=2018-11-09}}</ref>。在1998年，當時戴文16歲，她取代[[英國|英國]]超級名模[[娜歐蜜·坎貝兒|娜歐蜜·坎貝兒]]，成為[[凡賽斯|凡賽斯]]代言人。

== 模特兒生涯 ==
戴文為[[蘭蔻|蘭蔻]]、[[凡賽斯|凡賽斯]]及[[香奈兒|香奈兒]]等高級品牌做模特兒。
