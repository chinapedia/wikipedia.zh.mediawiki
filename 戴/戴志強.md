'''戴志强'''（{{bd|1944年|||}}），[[浙江|浙江]][[绍兴|绍兴]]人，出生于[[上海|上海]]，中国[[钱币学|钱币学家]]，曾任河南[[安阳博物馆|安阳博物馆]]副馆长、[[中国人民银行|中国人民银行]]货币发行司副司长、[[中国钱币博物馆|中国钱币博物馆]]馆长等职。

== 生平 ==
[[File:Dai_Baoting_and_Dai_Zhiqiang.jpg|thumb]]
戴志强父亲[[戴葆庭|戴葆庭]]是[[浙江|浙江]][[绍兴|绍兴]]人，中国著名钱币收藏家、钱币学家，[[上海泉币学社|上海泉币学社]]发起人之一。1944年，戴志强出生于[[上海|上海]]。1953年夏，戴家搬到上海[[江西中路|江西中路]]，因父母在广东路古玩市场经营一家钱币专营店<ref>{{cite journal |author1=戴莹 |title=平生只言古泉币 不顾现世富贵钞——访钱币鉴定专家戴志强 |journal=收藏界 |date=2007 |issue=6 |pages=99-101}}</ref>，戴志强放学经过时常去逗留<ref>{{cite journal |author1=戴志强 |title=记忆中的彭信威先生 |journal=金融博览 |date=2013 |issue=8 |pages=64-65}}</ref><ref>{{cite journal |author1=刘明杉 |title=平生只言 古钱币 不顾现世富贵钞——著名的古钱币学家戴志强和他的文集(上) |journal=艺术市场 |date=2006 |issue=4 |pages=54-55}}</ref>。高中时，数理化成绩优异的戴志强决定大学学习文史经济来做好钱币研究。1961年，戴志强考入[[复旦大学|复旦大学]]，就读于历史系历史地理专业。1964年，因[[中宣部|中宣部]]时任副部长[[周扬_(政治人物)|周扬]]敦促国内加强[[世界史|世界史]]研究，复旦历史系设立世界史专业，戴志强按学校安排转学世界史，1966年以亚非拉美[[民族解放運動|民族解放运动]]史专业（简称世界史专业）毕业。因时值[[文化大革命|文化大革命]]，戴志强的分配方案由留校变为到上海《[[解放日报|解放日报]]》，并最终变为到[[河南|河南]][[安阳|安阳]]工作<ref name="刘娜2013">{{cite journal |author1=刘娜 |title=戴志强:人生有拐弯,没弯路 |journal=金融博览 |date=2013 |issue=6 |pages=64-65}}</ref>。

1978年，时任河南安阳化工局局长秘书的戴志强申请调到[[安阳博物馆|安阳博物馆]]工作，当时[[中国社会科学院|中国社会科学院]]在安阳设有考古工作队，主要负责挖掘[[殷墟|殷墟]]。戴志强常到安阳文物工作队查阅考古刊物，并向考古队成员请教，提升自己考古学方面的知识素养。1981年8月，戴志强被任命为河南安阳博物馆副馆长，同年先后在国家级刊物《[[文物_(杂志)|文物]]》上发表了《安阳殷墟出土[[贝币|贝化]]初探》和《漳州军饷银饼年代考》两篇文章。1982年6月底，[[中国钱币学会|中国钱币学会]]成立，戴志强参加了学会的成立大会，并当选为第一届理事会理事<ref name="文集前言">{{cite journal |author1=戴志强 |title=《戴志强钱币学文集》前言 |journal=中国钱币 |date=2006 |issue=2 |pages=16,81-82}}</ref>。1983年10月，戴志强调河南安阳市文化局任副局长<ref>{{cite web |title=历届馆长 |url=http://www.aybwg.org/about/11.html |publisher=安阳博物馆 |access-date=2018-10-30 |archive-url=https://web.archive.org/web/20180321050706/http://www.aybwg.org/about/11.html |archive-date=2018-03-21 |dead-url=no }}</ref>。因为在《文物》上发表的两篇文章，戴志强引起了[[国家文物局|国家文物局]]的关注，经过专门了解后，国家文物局决定将其调到[[北京|北京]]工作<ref name="刘娜2013" />。

1984年10月，戴志强调入[[中国人民银行|中国人民银行]]印制管理局（既今[[中国印钞造币总公司|中国印钞造币总公司]]），11月任中国钱币学会副秘书长、秘书处副处长{{efn|当时中国钱币学会秘书处由中国人民银行印制局代管<ref name="文集前言" />。}}。1986年11月，中国钱币学会第二次会员代表大会召开，戴志强接任秘书长<ref name="文集前言" />、常务理事。1990年5月18日，[[中国钱币博物馆|中国钱币博物馆]]筹备委员会成立，戴志强兼任委员会副主任<ref name=dsj>{{Cite web |url = http://www.cnm.com.cn/zgqbbwg/lshgc/140148/index.html |title = 钱币工作大事记 |date = 2015-10-10 |publisher = 中国钱币博物馆 |access-date = 2018-10-30 |archive-url = https://web.archive.org/web/20170118054713/http://www.cnm.com.cn/zgqbbwg/lshgc/140148/index.html |archive-date = 2017-01-18 |dead-url = no }}</ref>。1991年任[[中国人民银行|中国人民银行]]货币发行司副司长、中国钱币学会学术委员会副主任。1993年，获研究员职称。1994年至2004年任[[中国钱币博物馆|中国钱币博物馆]]馆长，2004年受聘为中国人民银行参事<ref name="刘娜2013" /><ref name="故事上">{{cite journal |author1=梅辰 |title=戴志强先生鉴赏故事(上) |journal=收藏 |date=2007 |issue=3 |pages=158-160}}</ref>。

1986年，受聘兼任[[国家文物鉴定委员会|国家文物鉴定委员会]]委员<ref>{{cite journal |author1=戴莹 |title=本月人物 戴志强 |journal=收藏界 |date=2007 |issue=4 |page=2}}</ref>，1987年兼任{{le|国际钱币学委员会|International Numismatic Council}}委员，1993年兼任国家流通文物专家组成员，1995年兼任[[国际博物馆协会|国际博物馆协会]]委员。1998年至2004年间，兼任国际钱币与银行博物馆委员会（ICOMON）理事、执委会委员、亚洲地区主席。2002年，任[[中国博物馆学会|中国博物馆学会]]常务理事。2005年，任中国钱币学会副理事长、学术委员会副主任。还曾兼任《[[中国钱币_(杂志)|中国钱币]]》杂志主编，[[中国金融学会|中国金融学会]]常务理事<ref name="刘娜2013" />。

戴志强曾在[[中国人民银行研究生部|中国人民银行研究生部]]等高校讲授中国钱币学、货币史<ref name="故事上"/>，曾于2001年受聘为[[北京科技大学|北京科技大学]]兼职教授<ref>{{cite web |title=科学技术与文明研究中心时期 |url=http://ihmm.ustb.edu.cn/lishiyange/ |website=北京科技大学科技史与文化遗产研究院}}</ref>、博士生导师，于2003年受聘为[[中国科技大学|中国科技大学]]兼职教授、博士生导师，于2005年受聘为[[中央民族大学|中央民族大学]]客座教授<ref>{{cite journal |author1=李飞 |title=戴志强:万变不离“钱”宗 访著名钱币学家戴志强 |journal=艺术与投资 |date=2006 |issue=5 |pages=86-89}}</ref>。

== 著作 ==
据戴志强2006年在《戴志强钱币学文集》前言中称，其从1979年到2004年间先后发表短文160余篇，除早期几篇文章是有关[[殷商|殷商]]历史和安阳博物馆馆藏[[甲骨|甲骨]]的，其他几乎都和钱币相关；另外，其曾为中国钱币学专著、文集书写过六十余篇序文<ref name="文集前言" />。
