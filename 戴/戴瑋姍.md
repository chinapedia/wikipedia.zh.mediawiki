'''戴瑋姍''' （{{bd|1990年|1月23日||}}），[[臺灣|臺灣]][[高雄市|高雄市]]人，第三屆新北市議員，是民進黨的新生代。<ref>[http://www.dpp30.com.tw/?interview=03]，我們的民主時代：比民進黨還年輕的新生代。</ref>

[[高雄女中|高雄女中]]畢業、[[國立臺灣大學|國立臺灣大學]]政治系畢業，曾任台大研究生協會會長。2018年獲民進黨提名，當選第三屆新北市板橋區市議員。

== 公共參與 ==
研究所時期投入學生自治，為會長[[林飛帆|林飛帆]]幹部。後戴瑋姍當選第45屆台大研究生協會會長，擔任會長期間，為紀念「陳文成事件」，落實校園轉型正義，於校務會議提案命名「陳文成紀念廣場」，推動校務會議多數支持而通過<ref>[http://www.new7.com.tw/talk/talkView.aspx?i=TXT201406181617151GR]，新新聞，陳文成不該白死！</ref><ref>[http://www.peoplenews.tw/news/166627bc-50df-4772-b4d7-d41c9a56877b]，民報，林飛帆窗台燭光追思陳文成 柯文哲：轉型正義非報復</ref>；爭取勞動權益，與台大工會合作，要求校方承認與研究生助理之間的僱傭關係<ref>[http://www.coolloud.org.tw/node/76048]，拒認僱傭關係 台大告勞委會 工會要求立即撤告。</ref><ref>[http://pnn.pts.org.tw/main/2014/02/24/%E5%AD%B8%E7%94%9F%E5%8A%A9%E7%90%86%E7%B4%8D%E5%8B%9E%E5%81%A5%E4%BF%9D-%E5%B7%A5%E6%9C%83%E6%86%82%E5%8F%B0%E5%A4%A7%E6%B8%9B%E9%A1%8D%E6%B8%9B%E8%96%AA%E5%9B%A0%E6%87%89/] {{Wayback|url=http://pnn.pts.org.tw/main/2014/02/24/%E5%AD%B8%E7%94%9F%E5%8A%A9%E7%90%86%E7%B4%8D%E5%8B%9E%E5%81%A5%E4%BF%9D-%E5%B7%A5%E6%9C%83%E6%86%82%E5%8F%B0%E5%A4%A7%E6%B8%9B%E9%A1%8D%E6%B8%9B%E8%96%AA%E5%9B%A0%E6%87%89/ |date=20140408205601 }}，學生助理納勞健保 工會憂台大減額減薪因應</ref>；重啟研究生學生代表選舉，健全學生自治民主機制。<ref>[http://cpaper-blog.blogspot.tw/2014/11/blog-post_11.html]，台大意識報，不再路迢迢的研究生自治─專訪研協第四十五屆會長戴瑋姍。</ref> 

2014年，[[太陽花學運|太陽花學運]]學生衝入行政院抗議遭警方以警棍與水槍暴力驅離，戴瑋姍與多所學校學生會共同聲援太陽花學運，發表《譴責暴力、呼籲罷課》聲明 。<ref>[http://www.peoplenews.tw/news/59f5bbad-410c-4a33-a66d-d26b779ee253]，民報，譴責暴力鎮壓​　台大學生會呼籲自主罷課。</ref> 

== 政治生涯 ==
民主進步黨政治工作者，為民進黨創黨三十週年青年代表。
加入台北市議員吳思瑤團隊，從基層服務開始；2016年，戴瑋姍身兼文宣組長、新聞發言人，助吳思瑤當選立法委員，後續任其國會助理。

據媒體報導，戴瑋姍有意角逐台北市士林北投區議員。<ref>[http://www.nexttv.com.tw/news/realtime/latest/11883719/%E6%88%B02018%EF%BC%9F%E3%80%80%E5%82%B3%E5%B0%8F%E5%90%B3%E6%80%9D%E7%91%A4PK%E6%AD%A3%E5%A6%B9%E5%89%AF%E7%99%BC%E8%A8%80%E4%BA%BA]，戰2018？傳小吳思瑤PK正妹副發言人。</ref> 後轉戰新北市板橋區市議員，並以選區第四高票順利當選。

== 爭議 ==
參選台大研究生學會會長時，因延誤繳交報名表，而遭對手陳思宇質疑當時現任會長林飛帆黑箱護航接班人，引發爭議。

2018年9合1大選後，在中天政論節目中，稱韓國瑜為「韓導」，經媒體報導後，引發其支持者不滿，並在粉絲專頁上受到網友批判。

被網紅蔡阿嘎評選為十大正妹議員候選人第四名，並被蘋果日報選為「選戰之最」當中的「最美麗候選人」。她謙虛表示，已經嫁給新北市、期許未來專業問政，感謝市民給予機會。<ref>[https://tw.appledaily.com/new/realtime/20181125/1473190/]，新北最美新科議員戴瑋姍　「嫁給新北市、專業問政」。</ref>
