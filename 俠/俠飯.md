《'''俠飯'''》（日語：{{lang|ja|侠飯}}）為日本作家福澤徹三創作的小説。2014年由[[文藝春秋|文藝春秋]]出版。2016年改編為同名電視劇。台灣由OTT平台[[KKTV|KKTV]]獨家播出。

== 劇情概要 ==
在求職活動中不斷碰壁的大四生若水良太，在自家門前遭遇到黑道火拼。槍戰中黑道「柳刃組」老大柳刃龍一為拯救若水而身中槍傷，此時接獲報案的警方已經抵達，情急之下，柳刃只能與左右手火野丈躲進若水的公寓中。柳刃在若水家避風頭期間，用既有的材料做出一道道的料理。大學生與黑道老大的奇妙同居生活就此展開。

== 登場人物 ==
; 若水良太
: 大學文學部4年級生。因求職接連碰壁，心中因而感到苦惱。
; 柳刃龍一
: 黑道組織「柳刃組」<ref group="注">「柳刃」在日文漢字中有「菜刀」的意思。「柳刃組」即「菜刀幫」之意。</ref>老大。左臉頰有一到大傷痕。相當喜愛做料理。
; 火野丈治
: 「柳刃組」舍弟。與柳刃一同暫居在良太的公寓避風頭，並隨時外出打探消息回報給柳刃。
; 結城春菜
: 良太的同學。喜愛男男戀小説的腐女，面容姿態姣好，良太心中的理想女性。
; 米倉信也
: 良太的同學。個性開朗的動漫宅男。老家在埼玉縣，父親經營解體業。
; 鹽之谷洋介
: 良太的同學。知名大企業社長之子。傑尼斯系美男子容姿，擁有高級名車，已獲得3家公司錄取，但他卻只是享受這種獲得錄取的滿足感。
