==遊戲模式==
===玩法===
遊戲採用第一人稱視角，使用滑鼠和鍵盤控制角色的行動。<br>
戰鬥方式採回合制，擊倒所有敵人便可以勝利，我方人員全滅或指定人物死亡便會導致戰鬥失敗。
=== 特色 ===
* '''時序系統'''：遊戲中，並無遊玩時間限制，且玩家可以在大地圖中隨意選擇想要前往的地方，但根據前往地點地距離，會消耗相應的時間<ref name="時序系統">{{cite web |url=http://xkqz.fhyx.com/time.html |title=《侠客风云传前传》时序系统|publisher=侠客前传官网 |date=2016-09-28|accessdate=2016-10-09}}</ref>。
* '''組隊系統'''：採用俠客小隊的形式，來推進劇情，讓玩家體驗這個全新動態的武林。玩家將在武林中交到不同朋友，進而邀請其成為隊友<ref name="組隊系統">{{cite web |url=http://xkqz.fhyx.com/team.html |title=《侠客风云传前传》組隊系統|publisher=侠客前传官网 |date=2016-09-28|accessdate=2016-10-09}}</ref>。
* '''天賦系統'''：每個角色都有其獨特的天賦，使每個角色的特點更加鮮明<ref name="天賦系統">{{cite web |url=http://xkqz.fhyx.com/talent.html |title=《侠客风云传前传》天賦系統|publisher=侠客前传官网 |date=2016-09-28|accessdate=2016-10-09}}</ref>。
==劇情概述==
===背景===
故事發生在《[[俠客風雲傳|俠客風雲傳]]》的兩年前<ref>{{Cite_web|title=《侠客风云传前传》人气女角青涩登场|url=http://xkqz.fhyx.com/news/160803/167.html|accessdate=2017-08-17|date=2016-08-03|language=zh-cn|via=鳳凰遊戲|archiveurl=https://web.archive.org/web/20160810192308/http://xkqz.fhyx.com/news/160803/167.html|archivedate=2016-08-10|dead-url=no|quote=《侠客风云传前传》主要讲述了本传中侠客们年轻时的故事，具体时间大约发生在两年前。}}</ref>，武林时至乱世，中原既有少林，峨眉，武当等名门正派，亦有酆都这样的邪教异端，近年來外族势力天龙教也对中原虎视眈眈，大英雄小虾米神秘失踪后已经过去100多年，江湖上偶有势力相争，意图武林霸主之位。<br>
此时，朝廷突发《法外三旬》法令，官府将不再过问江湖仇杀，一时间，各門派人人自危，屠杀灭门惨案更是举不胜举。<br>
逍遙派弟子谷月轩和荆棘奉师傅之命外出查探此事，意外结识神秘少女卫紫绫三人结伴为解开这一切的幕后黑手而闯荡江湖，渐渐发现，这并不是简单的江湖仇杀，这一切的背后似乎有一只大手，正在操纵著这一切。<ref>{{Cite_news|title=俠客風雲傳前傳 故事背景介紹|url=https://kknews.cc/game/xm6b3g.html|publisher=每日頭條|date=2016-07-08|accessdate=2017-08-17|language=zh-tw|quote=武林時至亂世，中原既有少林，峨眉，武當等名門正派，亦有酆都這樣的邪教異端。近年外族勢力天龍教也對中原虎視眈眈。在大英雄小蝦米神秘失蹤後已經過去了100多年，江湖上又偶有勢力相爭，意圖武林霸主之位。此時，朝廷突發《法外三旬》法令，官府將不再過問江湖仇殺。一時間，各門各派人人自危，屠殺滅門慘案更是舉不勝舉！谷月軒及荊棘奉師傅之命外出查探此事，意外結識神秘少女衛紫綾三人結伴為解開這一切的幕後黑手而闖蕩江湖，漸漸的他們發現，這並不是簡單的江湖仇殺，而在這一切的背後，似乎有一隻大手，正在操縱這一切！}}</ref>
===故事===
故事開始於天龍教和酆都兩派爭奪武林霸主之位相互爭鬥起來，戰鬥結束後兩敗俱傷勝敗未分，暫時休兵以圖重來。<br>
逍遙派弟子[[谷月軒|谷月軒]]和[[荊棘|荊棘]]奉師尊無暇子之令前往援救弦劍山莊，並調查『法外三旬』之事，兩人到達弦劍山莊後，發現山莊已被自稱酆都的傀師所滅。<br>
擊退傀師後，兩人回到逍遙谷回報山莊覆滅之事，並結識衛豹之女[[衛紫綾|衛紫綾]]。
