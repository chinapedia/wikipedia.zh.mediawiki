'''锦绣苋'''（学名：{{lang|la|''Alternanthera bettzickiana''}}），俗稱'''旱生西洋菜'''或'''細葉枸杞菜'''<ref name="seablue.hk">{{cite web |url=https://seablue.hk/?p=6475 |title=〈香港陽台盆栽種植〉台灣細葉枸杞菜 / 泰國枸杞葉 / 錦繡莧 / 旱生西洋菜 |date=2018-09-30 |access-date=2018-11-01 |language=zh-hant }}</ref>，是[[苋科|苋科]][[莲子草属|莲子草属]]的植物。分布于[[巴西|巴西]]以及[[中国|中国]]各大城市等地，目前已由人工引种栽培。

== 别名 ==
五色草（北京） 红草（广东） 红节节草、红莲子草（福建）
