'''低氧化硫'''是硫的一类氧化物，包括以下几种：<ref name = "Greenwood">{{Greenwood&Earnshaw}}</ref>
*SO，[[一氧化硫|一氧化硫]]和它的双聚体S<sub>2</sub>O<sub>2</sub>
*S<sub>2</sub>O
*一氧化多硫，S<sub>n</sub>O，在S<sub>n</sub>（n为5-10）环上连接一个硫氧双键
*S<sub>7</sub>O<sub>2</sub>
*聚合氧化物，聚合硫氧化物有时被称作PSO's

<!-- In these the average oxidation state of sulfur is one or two. -->人们对低氧化硫的兴趣因为研究需要而上升，例如大气中硫污染的原因。对含有大量硫氧化物的外星大气进行研究，例如木星的卫星[[木卫一|木卫一]]和[[金星|金星]]。早期由一些工人报道的这类化合物，例如将硫溶于液态三氧化硫中得到的蓝色“三氧化二硫”（S<sub>2</sub>O<sub>3</sub>），实际上似乎是多硫阳离子S<sub>4</sub><sup>2+</sup>和S<sub>8</sub><sup>2+</sup>形成的盐的混合物。<ref name = "Greenwood"/> 

;[[一氧化硫|一氧化硫]]，单体(SO)和双聚体(S<sub>2</sub>O<sub>2</sub>)
:这两种分子在低温下能稳定存在于冷凝阱中(参见[[一氧化硫|一氧化硫]])。
;[[一氧化二硫|一氧化二硫]]，S<sub>2</sub>O

;S<sub>n</sub>O
:多种通式为S<sub>n</sub>O（n= 5-10）的一氧化多硫是已知的，这些化合物中氧在环外以硫氧双键与S<sub>n</sub>环相连。<ref name = "Greenwood"/>它们可以用[[三氟过乙酸|三氟过乙酸]]（CF<sub>3</sub>C(O)OOH）氧化对应的环状S<sub>n</sub>[[硫的同素异形体|硫单质]]制得。<ref name = "Greenwood"/>这些物质都是深色的，分解时产生硫和[[二氧化硫|二氧化硫]]。 <ref name = "Greenwood"/>

;S<sub>6</sub>O<sub>2</sub>, S<sub>7</sub>O<sub>2</sub>
:S<sub>6</sub>O<sub>2</sub>和S<sub>7</sub>O<sub>2</sub>可以分别用三氟过乙酸氧化环状S<sub>6</sub>和S<sub>7</sub>制得。不过S<sub>6</sub>O<sub>2</sub>晶体还没有被分离出来。<ref name = "Greenwood"/>
