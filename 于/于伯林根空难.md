'''俄罗斯巴什克利安航空第2937次班机'''，是一架[[图-154|图-154]]型客机，原计划由[[俄罗斯|俄罗斯]]首都[[莫斯科|莫斯科]]飞往[[西班牙|西班牙]]的[[巴塞罗纳|巴塞罗纳]]。'''DHL快遞公司第611次航班'''，是一架[[波音757|波音757-23APF]]型货机，原航线是从[[巴林國際機場|巴林國際機場]]經[[意大利|意大利]]的[[贝尔加莫国际机场|贝尔加莫国际机场]]飞往[[比利时|比利时]]的[[布鲁塞尔|布鲁塞尔]]。两架飞机于当地时间晚上9:35在[[德国|德国]]南部[[博登湖|博登湖]]畔毗邻[[瑞士|瑞士]]的小镇[[于伯林根|于伯林根]]附近的半空中相撞。共有包括双方机组在内的71人遇难。德方事故调查团认定[[空中交通管制|空中交通管制]]系统的处置失当是导致此次灾难发生的主要原因。此次空難通称'''于伯林根空难'''。

事发当日值班的[[丹麥|丹麥籍]][[航空管制員|空管员]]皮特·尼尔森事后遇刺身亡，行凶者是俄羅斯[[奧塞梯人|北奧塞梯人]]建筑师[[维塔利·卡洛耶夫|维塔利·卡洛耶夫]]，他在此次空难中失去了自己的妻子和两个孩子。

== 事件双方 ==
* 2937次航班为[[包机|包机]]，载有60名乘客和9名机组人员，其中有前往西班牙旅行的45名俄罗斯[[烏法|烏法市]]的儿童。
* 611次航班有2名机组人员，[[英国|英国]]籍的机长保罗·菲利普（Paul Phillips）和加拿大籍副驾驶布兰特·坎皮奥尼（Brant Campioni），后者原计划负责飞机后半段航程的操控。

== 事故经过 ==
[[image:Gedenkstätte_Flugzeugabsturz_Überlingen_Perlenkette.jpg|thumb]]
[[image:Gedenkstatte_Flugzeugabsturz_Uberlingen_Opfer.jpg|thumb]]
[[image:Bodensee_-_Salem,_Meersburg_&_Co_(45).JPG|thumb]]
事发当日两架班机同在约11000米的高度以互相衝突的航道飛行，尽管两机已经进入德国领空，但此地区空域由位于[[瑞士|瑞士]][[苏黎世|苏黎世]]的空管公司「天導」负责。当晚天導空管中心只有空管员皮特·尼尔森1人值班，他同时在两个控制台上进行调度操作，直到空难发生前1分钟他才发现两架班机的航線衝突，随后，他首先同2937次班机取得了联系，通知其飞行员降低高度300米以避免同611次班机相撞。

俄方机组依照指挥开始下降高度，但几秒后，飞机的[[空中防撞系统|空中防撞系统]]提示他们将飞机拉高，但在系統提示與空管指揮兩相矛盾下，機組選擇下降高度。几乎在同一时刻，另一方611次班机上的空中防撞系统提示机组下降飞机高度。如果兩架航班上的飞行员都按各自的防撞系统提示操作，即可以避免这场灾难。611次航班遵照防撞系统的提示下降了高度，由于他们将注意力都集中在了[[雷达|雷达]]屏幕上的2937航班，而没有及时将自身状况通知空管员。而此时另一方的俄国飞行员则是按照空管员的指示下降高度忽略空中防撞系統提示爬升，并第2次将他们的[[磁方位|磁方位]]向同一方向又更改了10度。

随后，尼尔森再次提示2937次班机下降高度。就这样，俄航班遵照空管员的指示，继续下降高度。不過，隨著機上防撞系統指示有飛機越來越接近及空中防撞系统不斷提示要爬升，機組人員已開始質疑空管員的指示正確性。兩機在相撞前3.8秒終於可以互相目視對方，儘管俄方機組員已立即爬升飛行高度，但為時已晚。

最終，2架班机在10600米左右高空相撞，611次航班的[[垂直尾翼|垂直尾翼]]从2937次航班机身左下方劃过，图-154客机随即爆炸并解体为两段，全部人員罹難。611次班机则失去控制並繼續飞行了7公里，其右翼引擎在坠机前爆炸并脱离机翼，2分鐘后坠毁在一个山腰附近，2架班機上共有71名乘客及机组人员全部遇难。

== 事故调查 ==
事发当晚苏黎士空中管制中心只有皮特·尼尔森一名空管员指挥着这一空域的往来航班。而其它的空管员在另外一个屋子里面休息，这是不符合规定的，但空管员们这一习惯性行为数年来得到了相关管理机构的默认和忽视。由于空管站处在检修期，当晚有另外几位空管员及部门主管随时准备着换班，而尼尔森并没有意识到这一点，也可能是他不想让同事冒疲劳工作的风险。

另外，一个位于地面用来提示空管员避免撞机的光学碰撞预警系统在事发前因例行检修而关闭，而尼尔森并没有留意此一状况。另一部安装在空管中心内的音频防撞预警系统事发当晚9:35，在碰撞发生前32秒发出了声响警报，但并没有任何人听到，事后对此设备的技术检查也证实其工作状态良好。空管中心的Skyguide专线电话也因检修工作而暂时切断，这部电话可接受到德国境内[[卡尔斯鲁厄|卡尔斯鲁厄]]空管站的空管员对事发空域异常情况的报警。

在空难发生前10分钟，尼尔森指挥着1架已经延遲的[[劳埃德航空公司|劳埃德航空公司]]的班机飞往[[腓特烈港机场|腓特烈港机场]]。他使用着有故障的电话系统，同时忙于两个控制台之间，以至于直到撞机发生前43秒他才通知俄国飞机员降低高度。如果他能够早一点发出警报及时將两机航线分离，就不会引发两机防碰撞系统的开启。当他预感到自己已经无法同时应付兩台控制台的工作而召唤其它空管员帮助时已经太晚了。

在德方调查报告基础上，瑞士声称事件中Tu-154客机低于瑞士空管员要求的航线高度33米并以每分钟580米的速度下降，如果俄飞行员不顾瑞士空管员给出的错误方位和警告信息而遵照防撞系统的提示执行就可避免灾难。

基于同样的调查报告，俄方声称事发时俄机组曾在飞机下降至10800米后遵照了防撞系统的警告提示执行，但当时空管员没有成功的使另一架飞机在11000米的高度。而且事件中的DHL货机也完全有机会避免碰撞，因为他们能够听到俄国机组同地面空管员之间的无线电对话。

2937次航班飞行员在碰撞发生之前的操控过程中改变飞机磁方位20度（从254调至274）一事并没有出现在官方报告中。

== 事故之後 ==
由於2937航班遵照航管人員的指示而非空中防撞系統，使得2937航班與611航班在空中相撞。在這之後，各國航空公司均要求當空中防撞系統建議與航管指示有衝突時，駕駛應遵從空中防撞系統指示，以免再度發生同樣事故。[[國際民航組織|國際民航組織]]亦跟進。

=== 航管員遇刺 ===
皮特·尼尔森于2004年2月24日在苏黎世自己家房前遇刺身亡。几天后，警方逮捕了1位名叫[[维塔利·卡洛耶夫|维塔利·卡洛耶夫]]的俄国人。卡洛耶夫的妻子和两个孩子是空难中2937航班上的乘客，死于这次空难中。有报道称他在失去了所有家庭成员后精神几近崩溃，另外他也是第1名亲临空难地点的死难者家属。卡洛耶夫妻子和孩子的葬礼推迟了1年，在1年之后的空难纪念日，他曾向SKYGUIDE的负责人提出希望能安排他与当年事件中的空管员（皮特·尼尔森）见面，但遭到拒绝。刺杀事件发生之后，警方在一家酒店内拘捕了还在发抖的卡洛耶夫，由于卡洛耶夫声称并不记得自己所做之事，警察随后将他送入精神病医院做进一步检查。不久，卡洛耶夫出庭，並當庭出示了于2003年11月11日由[[汉堡|汉堡]]一家律师事务所发给他的一份和解协议，协议中Skyguide承诺付给卡罗耶夫60,000[[瑞士法郎|瑞士法郎]]作为他失去妻子的补偿，另有100,000瑞士法郎是对他失去的两个孩子的补偿。作为回报，Skyguide要求维塔利·卡洛耶夫在接受此协议后不可再对其提出任何要求。这份协议激怒了卡洛耶夫，他随后要求亲自面见Skyguide主管[[Alan_Rossier|阿兰·罗歇儿]]和空管员皮特·尼尔森。

2005年10月26日卡洛耶夫被判8年监禁，2007年因表現良好[[假釋|假釋]]回國。

尼尔森的死使得管理层的过失以及导航系统的不合格成为了这次空难的主要责任方。
