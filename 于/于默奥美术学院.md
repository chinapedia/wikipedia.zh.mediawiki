'''于默奥大学美术学院'''成立于1987年，位于[[于默河|于默河]][[Öst_på_stan|东部城镇]][[于默奧|于默奧]]。美术学院与[[Arkitekthögskolan_vid_Umeå_universitet|建筑学院]]、[[Designhögskolan_i_Umeå|设计学院]]、[[Bildmuseet|视觉文化博物馆]]、[[Sliperiet|Sliperiet]]和HumlabX共同构成了[[Konstnärligt_campus_vid_Umeå_universitet|于默奥大学艺术校区]]。

== 培训 ==
美术学院可提供两个[[美術|美術]]领域的培训项目：三年制的[[學士|学士学位项目]](BFA)和两年制的[[Masterexamen|硕士]]研究生项目(MFA)。此外美术学院还提供I,II，III级[[美学|美学]]课程和暑期培训课程。美术训练包括实践和理论方面的指导。学生可学习到基于木材、金属、石膏、铜、版画的艺术创作以及照片、视频、音频和计算机技术。

自1993年以来学生的作品都被展示于学院的画廊中<ref>{{Cite web|title=Om galleriet|url=http://www.art.umu.se/sv/galleriet/om-galleriet/|publisher=Konsthögskolan vid Umeå universitet|accessdate=17 april 2014}}</ref>，每年学院还会举办公众艺术展，这通常是在校园内或附近的[[Bildmuseet|视觉文化博物馆]]举行。<ref>{{Cite web|title=Examensutställningar|url=http://www.art.umu.se/sv/utbildning/examensutstaellningar/|publisher=Konsthögskolan vid Umeå universitet|accessdate=17 april 2014}}</ref>

== 研究 ==
美术学院还与国家艺术研究院<ref>{{Cite web|title=Utbildning på forskarnivå|url=http://www.art.umu.se/sv/utbildning/utbildning-paa-forskarnivaa/|publisher=Konsthögskolan vid Umeå universitet|accessdate=17 april 2014}}</ref><ref>[http://www.konstnarligaforskarskolan.se Konstnärliga forskarskolan] Läst 17 april 2014</ref>联合设立了一个艺术基金会并开展了一系列研究项目。<ref>{{Cite web|title=Forskningsprojekt|url=http://www.art.umu.se/sv/forskning/forskningsprojekt/|publisher=Konsthögskolan vid Umeå universitet|accessdate=17 april 2014}}</ref>

== 校舍 ==
美术学院最初的校址位于[[Umeå_träsliperi|乌默奥木浆厂]]内的一栋建筑中，其外墙由[[Sigge_Cronstedt|西格·克隆斯塔德]]于1909年设计。 该工厂始建于1910年，直到1954年一直是于默奥市最大的制造业工厂。<ref name="Ahnlund">{{Cite book| editor=Ahnlund Mats|title=Äldre industrier och industriminnen vid Umeälvens nedre del: [Older industries and industrial monuments in the lower part of the Ume river valley]|series=Norrländska städer och kulturmiljöer, 0348-2618 ; 6|year=1980|publisher=Inst. för konstvetenskap, Umeå univ.|language=sw}}</ref> 美术学院于1986年成立<ref>{{Cite web |url=http://www.umea.se/download/18.1382718810a4aa4f727800021780/arkitekturguiden.pdf |title=Arkitekturguide |accessdate=30 juni 2012 |efternamn= |förnamn= |efternamn2= |förnamn2= |date=2001 |verk= |publisher=Umeå kommun |archive-url=https://web.archive.org/web/20130126065553/http://www.umea.se/download/18.1382718810a4aa4f727800021780/arkitekturguiden.pdf |archivedate=2013-01-26 |deadurl=yes}}</ref>并于一年后搬迁至这栋建筑中<ref>{{Cite web|title=En arena för kreativitet och gränsöverskridande kunskap skapas i Umeå|url=https://www.wallenberg.com/maw/projekt/reportage/en-arena-kreativitet-och-gransoverskridande-kunskap-skapas-i-umea|accessdate=22 mars 2014|archive-url=https://web.archive.org/web/20140323020143/https://www.wallenberg.com/maw/projekt/reportage/en-arena-kreativitet-och-gransoverskridande-kunskap-skapas-i-umea|archive-date=2014-03-23|dead-url=yes}}</ref>。

2012年春美术学院搬迁至目前的新校舍，即[[Konstnärligt_campus_vid_Umeå_universitet|艺术校区]]。以前的校舍于2013-2014年间改建为会议中心和艺术产业项目[[企业孵化器|孵化器]]<ref>{{Cite web |title=Konstnärligt Campus |url=http://www.umea.se/umeakommun/kommunochpolitik/planerochstyrdokument/utvecklingochplanering/stadsplaneringochbyggande/projekt/konstnarligtcampus.4.32feb01187db6237c800020301.html |publisher=[[Umeå_kommun|Umeå kommun]] |accessdate=22 mars 2014 |date=28 juni 2013 |deadurl=yes |archive-url=https://web.archive.org/web/20140323003632/http://www.umea.se/umeakommun/kommunochpolitik/planerochstyrdokument/utvecklingochplanering/stadsplaneringochbyggande/projekt/konstnarligtcampus.4.32feb01187db6237c800020301.html |archivedate=2014-03-23 }}</ref>''，''但老校舍将依然用作举办学院的画展。<ref>{{Cite web|url=http://www.sliperiet.umu.se/sv/|title=Sliperiet|accessdate=17 april 2014}}</ref>
