'''于丹《论语》心得'''，2006年11月1日由[[中华书局|中华书局]]出版的书籍，由[[北京师范大学|北京师范大学]][[教授|教授]][[于丹|于丹]]写作。<ref name="bkx " >{{cite book |language =zh-hans|author=于丹 |title=《于丹〈论语〉心得》|year=2006年 |publisher=中华书局 |location=北京 |pages=导读 |isbn=9787101053982 }}</ref>

== 内容 ==
该书由[[于丹|于丹]]教授在[[百家讲坛|百家讲坛]]的讲稿提简精练而成，对二千多年前的《[[论语|论语]]》进行现代汉语的诠释和解读。<ref name="bkx " /> 于丹笔下的[[孔子|孔子]]和《[[论语|论语]]》，是多彩多姿的孔子，是圣贤也是贴近心灵的医生。<ref name="bkx " /> 

== 作者简介 ==
[[于丹|于丹]]，是[[中国|中国]][[古代|古代]][[文学|文学]][[硕士|硕士]]，影视学[[博士|博士]]，[[北京师范大学|北京师范大学]]艺术与传媒学院院长助理、影视传媒系系主任，硕士生导师。

== 章节 ==
全书共分为七章：
*天地人之道
*心灵之道
*处世之道
*君子之道
*交友之道
*理想之道
*人生之道
书尾附带《[[论语|论语]]》原文。

== 评价 ==
[[厦门大学|厦门大学]][[教授|教授]][[易中天|易中天]]：“于丹为我们讲述的是这样的孔子，一位链接了多彩世界的灰色孔子，链接是广泛的，东西南北，古今中外。主题却是单纯的，单纯到没有色彩，没有性别，没有时间和空间，只有温度。”<ref name="bkx " />
