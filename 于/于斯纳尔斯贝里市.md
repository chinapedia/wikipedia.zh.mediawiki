'''于斯纳尔斯贝里市'''（[[瑞典文|瑞典文]]：{{lang|sv|''Ljusnarsbergs kommun''}}）是[[瑞典|瑞典]]中部[[厄勒布鲁省|厄勒布鲁省]]的一个[[瑞典自治市|自治市]]。其治所位于{{link-en|科帕尔贝里|Kopparberg}}。

== 地理 ==
于斯纳尔斯贝里市境内主要是含有[[矿脉|矿脉]]的[[火山岩|火山岩]]，在东部也有[[花岗岩|花岗岩]]。在这些往往布满裂缝的[[岩层|岩层]]上面则覆盖着一层[[冰碛|冰碛]]。在冰碛上生长有[[针叶|针叶]]林。在[[谷地|谷地]]里往往有[[沼泽|沼泽]]。在许多地方[[采矿|采矿]]把过去的[[森林|森林]]砍掉了，在市区的东部还有大片的[[森林|森林]]和许多[[湖泊|湖泊]]。

== 历史 ==
考古已经发现一些[[石器时代|石器时代]]的居民点。今天的市区是17世纪里由于[[矿业|矿业]]的发展形成的，在18世纪里达到顶峰，当时市内有许多[[矿井|矿井]]和[[冶炼厂|冶炼厂]]。1635年当地造了一座[[教堂|教堂]]，[[教堂|教堂]]内的[[牧师|牧师]]说[[芬兰语|芬兰语]]。一直到20世纪内矿山依然是当地的主要[[经济|经济]]部门，最后一座大[[矿井|矿井]]是1987年关闭的。

从1960年开始市区内的[[矿井|矿井]]逐渐被关闭。到1989年为止还有一座开采[[白钨矿|白钨矿]]的[[矿井|矿井]]，但是在当年也被关闭了。

== 经济 ==
所有矿井关闭后市内最大的企业是一座生产纤维产品的工厂，此外市内经济由一座啤酒厂、小企业和公共设施组成。由于矿山及其关闭市内的病例和失业率均很高。
