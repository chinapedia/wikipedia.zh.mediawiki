== 性质 ==
由于积聚[[双键|双键]]的存在，性质很活泼，易加成及聚合。

[[乙烯酮|乙烯酮]]是无色有强烈气味的气体，刺激性和毒性很大，熔点-150.5°C(-239°F)，沸点-56.1°C(-69°F)。

== 制备 ==
*烯酮可由[[酰氯|酰氯]]通过[[消除反应|消除反应]]消去HCl制得：
:[[File:Mecanisme-de-la-formation-des-cetenes.png|由酰氯制备烯酮]]
:[[三乙胺|三乙胺]]之类的碱可以脱去羰基alpha位的酸性氢，生成碳碳双键，氯离子离去。
*alpha-[[重氮|重氮]][[酮|酮]]在[[Wolff重排反应|Wolff重排反应]]中转化为烯酮。
*苯乙酸alpha质子酸性较强，在碱存在下失水生成苯基烯酮。
*实验室中可由[[丙酮|丙酮]]蒸汽[[热裂|热裂]]制取乙烯酮（CH<sub>2</sub>=C=O）：<ref>C. D. Hurd (1941). "[http://www.orgsyn.org/orgsyn/orgsyn/prepContent.asp?prep=cv1p0330 Ketene]". ''[[Org._Synth.|Org. Synth.]]''; ''Coll. Vol.'' 1: 330.  </ref>

== 反应 ==
烯酮含有两个正交的π键，化学性质很活泼，是很好的[[酰基化|酰基化]]试剂。

:[[File:Ketene_reactions.png|500px]]

它可以[[二聚|二聚]]生成[[二乙烯酮|二乙烯酮]]的衍生物，也可与富电子的[[炔烃|炔烃]]发生[2+2][[环加成反应|环加成反应]]生成环丁烯酮衍生物。

:[[File:Diketene_nucleophilic_reaction.png|550px]]

[[二醇|二醇]]（HO-R-OH）与二烯酮（O=C=CH-R'-CH=C=O）反应，生成[[聚酯|聚酯]]含有(-O-R-O-CO-R'-CO-)重复单元。

乙烯酮与[[乙酸|乙酸]]反应生成[[乙酸酐|乙酸酐]]，工业上合成乙酸酐即是利用此法。
