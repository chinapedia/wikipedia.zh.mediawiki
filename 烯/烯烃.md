S.| last3=Tavernier| first3=D.| issue=8–9 }}</ref>。

最简单的烯烃是[[乙烯|乙烯]]（C<sub>2</sub>H<sub>4</sub>），[[國際純化學和應用化學聯合會|國際純化學和應用化學聯合會]][[IUPAC命名法|命名]]为ethene。乙烯是工业上生产规模最大的[[有机化合物|有机化合物]]<ref name="cenews">{{cite journal |title=Production: Growth is the Norm |journal=Chemical and Engineering News |volume=84 |issue=28 |pages=59 |date=July 10, 2006 |doi=10.1021/cen-v084n034.p059}}</ref>。

“烯”字是新造字，左边的火取自“[[碳|碳]]”字，代表可以燃烧；右边的希取自“稀”字，表示氢原子和化合价稀少之意，意味着烯是[[烷烃|烷]]（完整）烃的一类[[不饱和化合物|不饱和]][[衍生物|衍生物]]。

== 乙烯的结构 ==
=== 乙烯的形状 ===
根据[[价层电子对互斥理论|价层电子对互斥理论]]，乙烯中每一个双键碳原子的键角都約是120°。这个角度会受连接在双键碳原子上的[[官能团|官能团]]的交互作用产生的[[范德瓦耳斯力|范德瓦耳斯力]]（凡得瓦力）影响。比如，C-C-C 键角在[[丙烯|丙烯]]中为123.9°。烯烃的碳-碳双键[[键能|键能]]比[[单键|单键]]高，133[[皮米|pm]]的[[键长|键长]]也比单个共价键短。

=== 分子的几何形态 ===
如同单个[[共价键|共价键]]，双键也能被描述为原子轨道的重叠。与单键不同的是，单键只包含一个[[σ键|σ键]]，而碳-碳双键包含了一个[[σ键|σ键]]和一个[[π键|π键]]。

每一个双键碳原子利用它本身的三个''sp<sup>2</sup>''[[杂化轨道|杂化轨道]]与三个原子形成[[σ键|σ键]]，那一个未杂化的"2p"轨道，垂直于由三个''sp<sup>2</sup>''[[杂化轨道|杂化轨道]]形成的平面，重叠形成[[π键|π键]]。

[[File:Cis-2-butene.PNG|thumb]][[File:Trans-2-butene.PNG|thumb]]因为需要很多的能量才能破坏一个[[π键|π键]]（对于乙烯是264 千焦/摩），所以碳-碳双键间的转动很困难并严重受限。产生的一个结果是取代烯烃存在一个或两个[[异构体|异构体]]（[[顺式异构|顺式异构]]和[[反式异构|反式异构]]）。例如，[[顺-2-丁烯|顺-2-丁烯]]中的两个[[甲基|甲基]]取代在双键的同一侧，而[[反-2-丁烯|反-2-丁烯]]的在不同的侧面。

双键不可能被扭曲。事实上，一个90°的扭曲需要的能量约等于<math>\pi</math>键键能的一半。[[p轨道|p轨道]]之间的偏差与预期的要小，因为形成了[[三角形平面|三角形平面]]。[[反-环辛烯|反-环辛烯]]是不稳定的，轨道误差仅仅是19°，[[二面角|二面角]]137°（正常应该是120°），并且还有18°锥形化。这解释了为什么这个化合物有0.8 D的[[偶极矩|偶极矩]]（顺式为0.4 D，预期均应该为零）。反式[[环庚烷|环庚烷]]只能在低温下稳定存在。

== 物理性质 ==
烯烃的物理性质可以与[[烷烃|烷烃]]对比。物理状态决定于[[分子质量|分子质量]]。简单的烯烃中，[[乙烯|乙烯]]、[[丙烯|丙烯]]和[[丁烯|丁烯]]是气体，含有五至十六个[[碳|碳]]原子的直链烯烃是液体，更高级的烯烃则是蜡状固体。烯烃為非[[極性|極性]]化合物，其唯一的分子間作用力為[[分散力|分散力]]，因此烯烃不溶於[[水|水]]，但可溶於[[四氯化碳|四氯化碳]]等有機溶劑。液體烯類之密度小於[[水|水]]，會浮在水面上。
反式烯烃比顺式烯烃的沸点低，熔点高。

== 化学性质 ==
烯烃的化学性质比较稳定，但比[[烷烃|烷烃]]活泼。考虑到烯烃中的碳-碳双键比烷烃中的碳-碳单键强，所以大部分烯烃的反应都有双键的断开并形成两个新的[[单键|单键]]。

== 合成 ==
* 最常用的工业合成途径是[[石油|石油]]的[[裂解作用|裂解作用]]。
* 烯烃可以通过酒精的[[脱水|脱水]]合成。例如，乙醇[[脱水|脱水]]生成乙烯：
:CH<sub>3</sub>CH<sub>2</sub>OH + H<sub>2</sub>SO<sub>4</sub> → CH<sub>3</sub>CH<sub>2</sub>OSO<sub>3</sub>H + H<sub>2</sub>O → H<sub>2</sub>C=CH<sub>2</sub> + H<sub>2</sub>SO<sub>4</sub>
:其他醇的消去反应都是[[Chugaev消去反应|Chugaev消去反应]]和[[Grieco消除反应|Grieco消去反应]]，产生烯烃。
* 高级α-烯烃的[[催化|催化]]合成可以由乙烯和[[有机金属化合物|有机金属化合物]][[三乙烯基铝|三乙烯基铝]]在[[镍|镍]]，[[钴|钴]]和[[铂|铂]]催化的情况下实现。
* 烯烃可以由[[羰基|羰基]]化合物通过一系列反应合成，比如乙醛和酮。
** 和一个[[烷基卤化物|烷基卤化物]]发生[[Wittig反应|Wittig反应]]
** 和一个[[苯基|苯基]][[砜|砜]]发生[[Julia成烯反应|Julia成烯反应]]
** 和两个不同的[[酮|酮]]发生[[Barton-Kellogg反应|Barton-Kellogg反应]]
** 结合一个酮，[[Bamford-Stevens反应|Bamford-Stevens反应]]或者[[Shapiro反应|Shapiro反应]]
* 烯烃可以由[[乙烯基|乙烯基]][[卤化物|卤化物]]结合生成。
* 烯烃可以由[[炔烃|炔烃]]的选择性[[还原|还原]]合成。
* 烯烃可以由[[Diels-Alder反应|Diels-Alder反应]]或[[Ene反应|Ene反应]]重排制得。
* 烯烃可以由α-氯代砜通过[[Ramberg-Bäcklund反应|Ramberg-Bäcklund反应]]合成。

== 反应 ==
烯烃因为可以广泛参与[[石化|石化]]工业的反应，被誉为石化工业的原材料。

=== 加成反应 ===
烯烃能进行很多种[[加成反应|加成反应]]：

==== 亲电加成 ====
大部分的烯烃加成反应是依照[[亲电加成|亲电加成]]机理完成的。一个例子就是[[Prins反应|Prins反应]]，其中[[羰基|羰基]]是亲电试剂。
* 与[[卤化氢|卤化氢]]HX加成，不对称烯烃加成时遵循[[马氏规则|马氏规则]]：
:RCH＝CH<sub>2</sub> + HX → RCHX-CH<sub>3</sub>
* 与[[水分子|水]]加成，遵循[[马氏规则|马氏规则]]：
:RCH＝CH<sub>2</sub> + H<sub>2</sub>O → RCHOH－CH<sub>3</sub>
* 与[[卤素|卤素]]X<sub>2</sub>反式加成：
:RCH＝CH<sub>2</sub> + X<sub>2</sub> → RCHX－CH<sub>2</sub>X
* 与[[次卤酸|次卤酸]]HOX加成：
:RCH＝CH<sub>2</sub> + X-OH → RCHOH－CH<sub>2</sub>X

==== 与氢加成 ====
烯烃经[[催化|催化]][[氢化|氢化]]可得到相应的[[烷烃|烷烃]]。这种反应需要在高压环境下由金属[[催化剂|催化剂]]参与进行。一般工业催化剂包含了[[镍|镍]]、[[鈀|鈀]]和[[铂|铂]]。实验室合成中，[[Raney镍|Raney镍]]是常用催化剂。一个例子就是[[乙烯|乙烯]]催化加氢得到[[乙烷|乙烷]]：
:CH<sub>2</sub>＝CH<sub>2</sub> + H<sub>2</sub> → CH<sub>3</sub>－CH<sub>3</sub>

==== 与水加成 ====
烯烃在[[酸|酸]]（常用[[硫酸|硫酸]]或[[磷酸|磷酸]]）的催化作用下，与水直接加成生成[[醇|醇]]。酸催化下，烯烃的直接[[水合|水合]]是亲电加成反应。反应的第一步是酸中的质子加到双键碳原子上，生成碳正离子中间体，这一步是控制反应速率的慢步骤。然后水分子与碳正离子反应，生成质子化的醇。最后质子化的醇失去质子生成醇。不对称烯烃与水的加成也遵循马氏规则。例如：
:CH<sub>2</sub>＝CH<sub>2</sub> + [[水|HOH]] → [[乙醇|CH<sub>3</sub>－CH<sub>2</sub>－OH]]     （H<sub>3</sub>PO<sub>4</sub>，280-300℃，7-8MPa）
:CH<sub>3</sub>－CH＝CH<sub>2</sub> + HOH → [[异丙醇|CH<sub>3</sub>CHOHCH<sub>3]]</sub>     （H<sub>3</sub>PO<sub>4</sub>，195℃，2MPa）

上述反应工业上生产乙醇和[[异丙醇|异丙醇]]等[[低级醇|低级醇]]的一种方法，称为“[[直接水合法|直接水合法]]”。除乙烯外，其他烯烃均不生成[[伯醇|伯醇]]。

==== 与卤素加成 ====
烯烃与[[溴|溴]]或者[[氯|氯]]的加成得到邻二溴或二氯烷烃。溴水的褪色可以作为检验乙烯的测试：
:CH<sub>2</sub>＝CH<sub>2</sub> + Br<sub>2</sub> → BrCH<sub>2</sub>－CH<sub>2</sub>Br
:以下是此反应机理：
:[[File:AlkeneAndBr2Reaction.png|600px]]
:此反应之所以能够发生，是因为双键的高电子云密度导致了溴-溴间电子的临时移动产生一个临时的偶极，最终导致靠近双键的溴略显正电性变为亲电试剂。
* 与[[卤化氢|卤化氢]]加成：例如HCl或HBr，最终生成相应的[[卤代烷|卤代烷]]。
:CH<sub>3</sub>－CH＝CH<sub>2</sub> + HBr → CH<sub>3</sub>-CH'''Br'''－CH<sub>3</sub>
:如果两个双键碳原子连接的氢原子数量不等，卤素将优先加成在拥有氢较少的碳上（[[马氏规则|Markovnikov's规则]]）
:以下是反应机理：
:[[File:AlkeneAndHBrReaction.png|600px]]
* [[碳烯|碳烯]]（[[卡宾|卡宾]]）的加成得到相应的[[环丙烷|环丙烷]]衍生物。

==== 与次卤酸加成 ====
烯烃与[[次卤酸|次卤酸]]（常用[[次氯酸|次氯酸]]或[[次溴酸|次溴酸]]）加成生成β-卤代醇。例如：
:CH<sub>2</sub>＝CH<sub>2</sub> + HOCl → CH<sub>2</sub>ClCH<sub>2</sub>OH

在实际生产中，由于次氯酸不稳定，常用氯和水直接反应。例如，将乙烯和氯气直接通入水中以生产[[β-氯乙醇|β-氯乙醇]]。这时反应的第一步是烯烃与氯气进行加成，生成环状[[氯鎓离子|氯鎓离子]][[中间体|中间体]]。在第二步反应中，由于大量水的存在，水进攻氯鎓离子生成β-氯乙醇。但溶液中还有[[氯离子|氯离子]]存在，它进攻氯鎓离子，故有[[副产物|副产物]][[1,2-二氯乙烷|1,2-二氯乙烷]]生成。

不对称烯烃与次卤酸的加成，也遵循马氏规则。亲电试剂的氯加到含氢较多的双键碳原子上，水加到含氢较少的双键碳原子上。例如：
:CH<sub>3</sub>CH＝CH<sub>2</sub> + HOCl → CH<sub>3</sub>CHOHCH<sub>2</sub>Cl

上述两个反应的产物——[[氯乙醇|氯乙醇]]和[[1-氯-2-丙醇|1-氯-2-丙醇]]是制备[[环氧乙烷|环氧乙烷]]和[[甘油|甘油]]等化合物的重要化工原料。

==== 与硫酸加成 ====
和烯烃与卤化氢的加成相似，烯烃与[[硫酸|硫酸]]的加成也是离子型的亲核加成。例如，将乙烯通入冷的[[浓硫酸|浓硫酸]]中，首先质子加到一个双键碳原子上，生成碳正离子中间体，然后硫酸氢负离子与之结合，生成[[硫酸氢乙酯|硫酸氢乙酯]]：
:CH<sub>2</sub>＝CH<sub>2</sub> + HOSO<sub>2</sub>OH → CH<sub>3</sub>－CH<sub>2</sub>-OSO<sub>2</sub>OH

因为硫酸拥有两个活泼氢原子，所以在一定条件下可与两分子乙烯进行加成，生成[[硫酸二乙酯|硫酸二乙酯]]（中性[[硫酸酯|硫酸酯]]）：
:CH<sub>2</sub>＝CH<sub>2</sub> + HOSO<sub>2</sub>OH + CH<sub>2</sub>＝CH<sub>2</sub> → CH<sub>3</sub>CH<sub>2</sub>OSO<sub>2</sub>OCH<sub>2</sub>CH<sub>3</sub>

不对称烯烃与硫酸的加成反应，也符合[[马氏规则|马氏规则]]。例如（括号内为反应温度及硫酸浓度）：
:CH<sub>3</sub>CH＝CH<sub>2</sub> + HOSO<sub>2</sub>OH → CH<sub>3</sub>CH(OSO<sub>2</sub>OH)CH<sub>3</sub>            （50℃，75-85%）
:(CH<sub>3</sub>)<sub>2</sub>C＝CH<sub>2</sub> + HOSO<sub>2</sub>OH → (CH<sub>3</sub>)<sub>3</sub>COSO<sub>2</sub>OH            （10-30℃，50-65%）

上述反应说明，烯烃对反应条件和硫酸浓度的要求，随着烯烃双键碳原子上连接的甲基增多而降低，即烯烃的活性和烯烃与卤素、卤化氢的加成活性相同。

若将C<sub>12</sub>至C<sub>18</sub>的直链[[α-烯烃|α-烯烃]]与硫酸反应，再将生成的酸性硫酸酯用[[碱|碱]]中和，则会得到一种硫酸酯盐型阴离子[[表面活性剂|表面活性剂]]，可作为液体[[洗涤剂|洗涤剂]]的原料，也可用作[[纺织助剂|纺织助剂]]：
:R-CH＝CH<sub>2</sub> + HOSO<sub>2</sub>OH → R－CH(OSO<sub>2</sub>OH)CH<sub>3</sub>
:R－CH(OSO<sub>2</sub>OH)CH<sub>3</sub> + NaOH → R－CH(OSO<sub>2</sub>ONa)CH<sub>3</sub> + H<sub>2</sub>O

=== 氧化反应 ===
烯烃可以被许多氧化剂所[[氧化|氧化]]。

* 在有[[氧气|氧气]]的情况下，烯烃可以燃烧，有明亮火焰，伴随[[二氧化碳|二氧化碳]]和[[水|水]]的生成。
* 与[[氧气|氧气]]或[[羧酸|羧酸]]发生催化氧化生成环氧化物。
* 与[[臭氧|臭氧]]发生[[臭氧化反应|臭氧化反应]]导致双键的断裂，得到两个[[醛|醛]]或[[酮|酮]]
:R<sub>1</sub>－CH＝CH－R<sub>2</sub> + O<sub>3</sub> → R<sub>1</sub>－CHO + R<sub>2</sub>－CHO + H<sub>2</sub>O
:这个反应可以用来测定未知烯烃的双键位置。
*[[乙烯|乙烯]]與[[高锰酸钾|高錳酸鉀]]水溶液反應產生[[二氧化錳|二氧化錳]],[[氫氧化鉀|氫氧化鉀]]及[[乙二醇|乙二醇]]
3CH<sub>2</sub>=CH<sub>2</sub>+2KMnO<sub>4</sub>+4H<sub>2</sub>O→2MnO<sub>2</sub>+2KOH+3HO-CH<sub>2</sub>-CH<sub>2</sub>-OH

=== 聚合反应 ===
烯烃的[[聚合反应|聚合反应]]是一个经济意义极高的反应，得到的高聚物有很高的工业价值，例如塑料中的[[聚乙烯|聚乙烯]]和[[聚丙烯|聚丙烯]]，[[合成润滑油|合成润滑油]]中的聚a-烯烃。聚合反应既可以通过[[自由基|自由基]]机理来实现，又可以通过[[离子|离子]]机理来实现。

== 烯烃的命名 ==
=== IUPAC名称 ===
根据[[IUPAC命名法|IUPAC]]命名规则，为了给烯烃主链命名。英文命名将中缀-an-换为-en-。例如'''CH<sub>3</sub>-CH<sub>3</sub>''' 是''eth'''an'''e''。因此 '''CH<sub>2</sub>=CH<sub>2</sub>'''的名字是''eth'''en'''e''。中文命名是直接将“'''烷'''”变为“'''烯'''”，例如'''CH<sub>3</sub>-CH<sub>3</sub>'''是[[乙烷|乙'''烷''']]，因此'''CH<sub>2</sub>=CH<sub>2</sub>'''的名字是[[乙烯|乙'''烯''']]。

在高级烯烃中，因为双键位置不同而导致[[异构体|异构体]]的出现，我们运用下面的数字系统：
# 命名含有双键的最长碳链为主链，使得双键碳原子的数字尽可能最小。

=== 一般名称 ===
尽管IUPAC命名系统有很高的通用性和精确性，但是一些烯烃的一般名称已经被广泛接受。
例如：
{| cellspacing="5" cellpadding="2"
|-
|
 || (CH<sub>3</sub>)<sub>2</sub>C=CH<sub>2</sub>
|-
| align="right" | '''IUPAC 名稱:''' || 2-甲基丙烯
|-
| align="right" | '''一般名稱:''' || 異丁烯
|}
