《'''世纪末超自然學院'''》（{{lang-ja|世紀末オカルト学院}}）是[[A-1_Pictures|A-1 Pictures]]製作[[日本電視動畫|日本電視動畫]]，「[[動畫的力量|動畫的力量]]」的第3部作品，從[[2010年7月|2010年7月]]開始在[[東京電視台|東京電視台]]播放。

== 故事簡介 ==
在[[長野縣|長野縣]]的{{tsl|ja|皆神山|皆神山}}，有一間名為華德斯坦學院的私立學校。由於學院的主科包括許多[[超自然|超自然]]研究，所以又被人稱為「超自然學院」。1999年7月，校長神代純一郎逝世，其女兒神代瑪雅來到學院代理工作；同日黃昏，瑪雅突然見到一位裸體男生從天而降……

== 角色介紹 ==
=== 主要角色 ===
; {{nihongo|神代瑪雅|神代 マヤ（くましろ マヤ）||CV：[[日笠陽子|日笠陽子]]}}
: 超自然學院的學院長也是其中的學生，前學院長的唯一女兒。父親死後，出現在學院并接任了學院長一職。初時希望廢除學院，現在則爲了調查父親的死的真相而與文明聯手。
: 幼年時因為父親的影響，而擁有十分豐富的超自然學說知識。由於父親專注于超自然的研究，后來更是把精力全部投入到學院的建設上，終導致家庭離異。瑪雅跟隨母親生活，從此十分討厭超自然學說。
: 平日顯得相當冷酷，但內心跟典型少女沒大分別。喜歡以極權方式操控他人，特別是文明。
: 從未來到來的文明逆轉時空後，成為其妻子。
; {{nihongo|内田文明|内田 文明（うちだ ふみあき）||CV：[[水島大宙|水島大宙]]、[[矢作紗友里|矢作紗友里]]（1999年）}}
: 從2012年來到1999年的青年，自稱是時空特務№.6。
: [[念力|念力]]使用者，在1999年時因湯匙彎曲，而廣受矚目的超能力少年。後來不知為何失去超能力，性格亦變得十分膽小。文明很不喜歡別人稱呼他為“{{lang|ja|ぶんめい}}”（日文中“文明”的[[音讀|音讀]]）。
: 在學院時偽裝成一名[[日本史|日本史]]老師，化名'''安倍實'''，是瑪雅的班主任。對美風小姐抱有好感，但也因此受盡折磨。
: 最終為拯救世界犧牲自己。
; {{nihongo|中川美風|中川 美風（なかがわ みかぜ）||CV：[[茅原實里|茅原實里]]}}
: 在中川食堂工作，製作的食物包羅萬有但質素不穩。對市內的超自然事物有一定認識。
: 開起車來很瘋狂，不惜貸款購入名牌跑車。文明親人之外唯一一個會稱呼文明為“{{lang|ja|ふみあき}}”（[[訓讀|訓讀]]）而非“{{lang|ja|ぶんめい}}”（音讀）的人。
: 真實身分為黑魔女。

=== 學院學生 ===
; {{nihongo|黑木亞美|黒木 亜美（くろき あみ）||CV：[[高垣彩陽|高垣彩陽]]}}
: 瑪雅的兒時玩伴，受瑪雅影響小時候一同研究超自然現象。確信著瑪雅還喜歡著超自然學說。
; {{nihongo|成瀨梢|成瀬 こずえ（なるせ こずえ）||CV：[[花澤香菜|花澤香菜]]}}
: 亞美的同班同學。異常熱衷于超自然現象，崇拜著前學院長，夢想是成為超自然學說的專家。自言從未在清晰知覺下體驗超自然現象，所以持續著修行，時不時會獨自跑到深山里閉關修行等等。

=== 學院教職員·相關人員 ===
; {{nihongo|JK|JK（ジェイケイ）||CV：[[子安武人|子安武人]]}}
: 真名不詳的學院發明家。一頭紅髮的胖子，在第一話時被瑪雅稱為“紅豬”還頗自豪的認同。是個耳朵靈敏的順風耳，專長是探測靈物。常与斯馬爾一起行动。
; {{nihongo|斯馬爾|スマイル||CV：[[高橋廣樹|高橋廣樹]]}}
: 學院專屬的機械工，喜歡爭吵和擅長打架。工作服的胸前佩戴著微笑表情的徽章。常与JK一起行动。
; {{nihongo|川島千尋|川島 千尋（かわしま ちひろ）||CV：[[小林優|小林優]]}}
: 學院的訓導主任，知道關於學院長逝世的秘密。因其髮型像當時流行的“糰子二兄弟”而被學生們嘲笑。因為誇獎了其髮型而喜歡上了文明。
: 真實身分為白魔女。
; {{nihongo|神代純一郎|神代 純一郎（くましろ じゅんいちろう）||CV：[[矢島正明|矢島正明]]}}
: 超自然學院前院長。一生鑽研超自然學說，成立的學院亦有很多異樣，對外宣傳死因為[[心肌梗死|心肌梗死]]，但當中有很大的秘密，並未死亡。其生前為了解開世紀末預言而私底下去尋找'''諾斯特拉達繆斯之鑰'''。之後留下一本筆記給瑪雅，並說明該筆記本會成為保護瑪雅的力量。

=== 其他 ===
; {{nihongo|№.5|No5|CV：[[住吉屋力|住吉屋力]]}}
: 文明（№.6）之前的時空特務。能力是發火術（Pyrokinesis）。因被皆神山洞窟內疑似[[天蛾人|天蛾人]]的巨大飛蛾襲擊未能撤回而身亡，特殊手機被發現在巨大糞便之中。
; {{nihongo|神代香奈枝|神代 香奈枝（くましろ かなえ）}}
: 瑪雅的母親，於1997年死亡。
; {{nihongo|黑木茂|黒木 茂||CV：[[島香裕|島香裕]]}}
: 亞美的父親，黑木建築公司的老闆，身強體壯，性格豪爽，喜歡講鬼怪故事。製作的飯糰十分巨大。
; {{nihongo|岡本明|岡本 あかり（おかもと あかり）||CV：[[水瀨祈|水瀨祈]]}}
; {{nihongo|岡本健吾|岡本 健吾（おかもと けんご）||CV：[[上田祐司|上田祐司]]}}

== 用語 ==
; [[諾斯特拉達姆士|諾斯特拉達繆斯]]之匙
: 諾斯特拉達繆斯在《諸世紀》（《百詩集》）中[[預言|預言]]於1999年世界末日的發生，而這項預言亦間接導致2012年的[[外星人|外星人]]入侵，[[地球|地球]]瞬間被毀滅一半。鑰匙是引起一切發生的源頭，如果能在之前及時破壞，所有災難都不會發生。

; [[時空|時空]]特務
: 部分2012年的人類在某次因緣下得到了[[外星人|外星人]]的科技，發明出了[[時光機器|時光機器]]。他們就此送出時空特務到1999年破壞諾斯特拉達繆斯之匙。每一位特務均有其№代號。

; 特殊手機
: [[外星人|外星人]]科技之下的發明，擁有多種協助特務的功能。最基本功能是可以跟不同時間點的人通話，以及與其他特務手機作內部連絡。
: 另一功能為拍攝[[未來|未來]]照片。結果會因應人的行動思想而有所不同；若然有意識向拍攝對象進行干擾，照片就會顯示出干擾以後的未來，否則就會顯示出沒被干擾的未來。

; 次元扭曲
: 本作的次元扭曲源於一個未來的人類，即情報生命體，與過去的同位體有了接觸，令二人所理解的記憶和歷史情報出現前後矛盾，造成了次元扭曲，引來了怪物。
