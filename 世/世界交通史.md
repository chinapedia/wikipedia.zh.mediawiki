在人類發展歷史的[[上古時代|上古時代]]中，世界各地民族之間的互動以及對外交通就遠遠超越今日的人們對他們所理解之程度並且還更加頻繁，然而古代人們也並沒有因為交通工具的簡陋就因此讓他們活動範圍縮小或侷限在一個區域，相反的他們的活動範圍若與今日現代化國家與民族之間的互動往來相比較的話，事實上古代各個民族間往來交流也不會差到哪裡去；即使是與[[西亞|西亞]]、[[地中海|地中海]]文明區相隔甚遠的[[中國|中國]]文明，於上古時代與西方世界交通也是不受天然屏障的局限。

實地的研究古代各個文明的發展路程會發現古代各地的民族並不是各自完全獨立發展他們各自文化的，從很多考古遺物顯示出他們彼此之間是有相互影響的訊息，縱然與今日世界各國交通發達相比較下，或許古代各地沒有那麼活絡，但是他們還是有往來交通的痕跡，從這裡可以得知古代民族的交通狀況即是今天[[歷史學|歷史學]]界應該亟待暸解的一個課題，由交通史的研究能夠找到許多現代歷史著作中它們所缺漏與忽略的部分。而這裏主要以[[中國歷史|中國歷史]]為主要闡述重點，並且連結[[中國文明|中國文明]]與[[西方文明|西方文明]]、[[印度文明|印度文明]]交通為主要的敘述方向，並試圖找出中國與[[美洲|美洲]]來往、[[哥倫布|哥倫布]]到達[[美洲|美洲]]之前[[歐洲|歐洲]]與美洲的來往紀錄，闡述其各自對外交通。

這裏不論述戰爭引起的交通，只純粹探討和平時期交通狀況。主要採用[[考古|考古]]出土[[史料|史料]][[文物|文物]]、文獻紀錄來剖析民族交通的概況，並輔以其他各民族之間的往來狀況所遺留的疑點來參考，以此管窺古代民族交通與文化交流的情形。以下章節分段以發源地、主要根據地對外界交通為主要劃分標準，因此即使有跨越[[欧洲|歐]][[亞洲|亞]][[非洲|非]]三洲的帝國，即以其發源地當做基準。

== 概念 ==
世界是一個整體，雖然地球上各個文明都有各自的特點，但是文明之間彼此卻互有往來並且相互影響。

== 亞洲地區 ==
亞洲可以說是人類主要文明的最大發祥地，[[東亞|東亞]]的中國、[[南亞|南亞]]的[[印度|印度]]、西亞的[[美索不達米亞|美索不達米亞]]等等古老文化都主要發生在亞洲，因此探討世界交通史亞洲最應該詳加介紹。

=== 中國與各地交通史 ===
{{main|中外交通史}}
至少在15世紀之前的中國，其科技地位、國家財富、文化成就一直都是舉世獨步的，幾無能有其他文明可以撼動其地位，相對的世界其他民族對中國的交通便是趨之若鶩。歷史上著名的[[絲路|絲路]]就是中外交通史上最具意義的交通路線；然而中國對世界貢獻並非單單只有絲綢或絲綢的製作技術而已，事實上中國人對世界文明貢獻很多；此外中國人與高加索人種的往來更可以追溯到[[殷商|殷商]]時代出土的白種人遺骸可以得知，因此早在上古時代的中國就有不同民族往來的遺跡紀錄。

==== 殷商的遺跡 ====

古代的中國地區事實上就有不少[[高加索人種|高加索人種]]在中國境內出現的紀錄與證據，尤其在早期沒有所謂的國土疆域的界線下，民族之間的往來可以說是相當的容易，而從這些[[史料|史料]]可以發現到中國古代對外交通的發達狀況，所以說中國文明並非是孤立在[[東亞|東亞]]地區自行發展。

從相關的考古挖掘紀錄與成果發現到最早在殷商時代中，殷商帝王墓就發現到[[高加索人|高加索人]]（[[白種人|白種人]]）、[[美拉尼西亞|美拉尼西亞]]（Melanesia）人的骨骸；從這個人種特徵的發現讓人們知道當時[[蒙古人種|蒙古人種]]與高加索人種的民族之間互動狀況，但是這些白種人怎麼來的、目的是什麼就不得而知。

有學者認為上古時代的中國事實上是黃種人與白種人民族彼此交錯在相同的地理環境的，不過目前此說法證據不足。

此外關於其他中國有高加索人種出沒的證據還有：考古工作者也在[[秦始皇陵|秦始皇陵]]與位於今天[[青海省|青海省]]的[[漢代|漢代]]陵墓發現到高加索人的遺骸，由此又得知早期中國有與外地互動的紀錄。

其次，古代的[[中亞|中亞]]地區事實上一直都是文明傳播的通道以及聯繫紐帶，中亞的[[遊牧民族|遊牧民族]]充分扮演文化交流的角色。

==== 周穆王西行 ====
關於西周穆王西行的紀錄可以見於《穆天子傳》、《[[竹書紀年|竹書紀年]]》。

==== 中國絲綢遠至希臘 ====
[[古希臘|古希臘]]人的[[賽里斯|賽里斯]]（Seres）國事實上是一個完全模糊不清的概念。在古希臘時代，中國製的絲綢就已經通行到歐洲[[巴爾幹半島|巴爾幹半島]]，由於當時的希臘人向[[斯基泰人|斯基泰人]]（Scythians）貿易所獲得這項商品，使得希臘人對於絲綢的來源產生諸多猜測，他們也將生產絲綢國度稱之為賽里斯國。然而在上古時代也唯有中國人才會生產絲綢的；相傳最早的絲是由[[嫘祖|嫘祖]]發明養蠶技術而將蠶絲抽取出來製成絹帛的一項發明，因此古代中西貨物的交通已經可以從中國到希臘。

==== 漢朝與西域的交通 ====
漢朝與[[安息_(国家)|安息]]（[[波斯|波斯]]）的交通往來直接的結果就是文化交流與貿易，其中中國的一些產物如：絲綢、桑樹、黃連、桃、樟腦、肉桂、梨、生薑、茶葉、陶瓷也經過波斯轉運[[印度|印度]]西方各地。而中國在對西域的交流中，也因文化交流的關係產生一些外來語，如：[[獅子|獅子]]、[[葡萄|葡萄]]。

然而有學者認為[[張騫通西域|張騫通西域]]所帶回來的葡萄的發音是希臘文Bστρυξ的譯音；而中國流傳的[[海馬葡萄鏡|海馬葡萄鏡]]是希臘文化的遺制。

==== 唐朝對外的交通 ====
中國在漢朝即有和西域諸國交通往來，到了唐朝與外界交通更是頻繁；當時的日本曾經招攬大批唐人移居歸化[[日本|日本]]。
==== 宋朝對外貿易 ====
[[南宋|南宋]]時期，由於商業的發展以及當時朝廷為了解決財政問題，因而對外貿易佔了很重要的地位，也因為如此促進了文化交流與傳播；此外南宋時代的[[造船業|造船業]]也有蓬勃的發展，而且船上陪配有指南針以利航行。

==== 明朝瓷器輸往東非 ====
明朝最著名的[[鄭和下西洋|鄭和下西洋]]讓中國人足跡遠達東非一帶，甚至打通了中國與[[非洲|非洲]]貿易的路線，東非海岸沿線發現不少明代瓷器，這些瓷器不少產自於[[景德鎮|景德鎮]]，中國瓷器也由東非轉運自歐洲。

==== 古代中國人到達美洲 ====
{{Fact|根據1965年[[祕魯|祕魯]]出土的銅人像，手上的石牌刻有“[[武當山|武當山]]”字樣，除了祕魯，在[[波利維亞|波利維亞]]、[[墨西哥|墨西哥]]也有中國文物出土；美洲目前在考古學與歷史學上還有很多謎團，而且難以解釋也相當古怪，像是美洲出土中國文物中還曾經有：“大齊田人之墓”、“木武本”、“太歲”、“明月松間照”等等[[漢字|漢字]]字樣。在[[厄瓜多|厄瓜多]]還有挖到[[王莽|王莽]]時代的古錢幣。此外[[美國|美國]]東岸也發掘出一面明朝“宣德金牌”，其上鑄有“大明宣德委錫”六字，以此證明中國似乎與美洲有些微往來，但是這些仍然無法說明中國與美洲直接往來交通。還有即使出土一面宣德金牌也不能夠證明鄭和曾經到達美洲大陸，只能採取保留態度以中國文物曾經流傳到美洲來看待。|time=2011-04-22T15:37:19+00:00}}

=== 波斯對外交通 ===
波斯帝國在古代文明史上是佔有一定的影響地位，由於其領土東可通往中國、印度；西可通往歐洲，很明顯的是歐亞之間的一個橋樑。
==== 波斯與中國的交通 ====
波斯在中國漢代就有與其往來紀錄，漢朝時候稱他們為安息。在彼此的往來當中波斯對[[中國文化|中國文化]]也具有一定的影響力，像是波斯樂器[[琵琶|琵琶]]、[[嗩吶|嗩吶]]傳入中國，成為中國傳統樂器之一；而波斯曾經向中國進貢獅子，中國語言中的“獅”其原始發音來自波斯。

東漢明帝時期，曾經派前甘英出使西域，甘英曾經抵達安息（波斯）西邊邊境，並且了解海西還有大秦（羅馬）的文明大國。
魏晉南北朝時代中國地區和波斯的交通更加頻繁。

=== 阿拉伯對外的交通 ===
古代阿拉伯人擅長經商，因此他們經常通行於歐亞之間，擔任著文化交流與傳播者的角色。

==== 阿拉伯與中國的交通 ====
漢朝時代就已經知道有阿拉伯地區，當時被稱為“條支”。阿拉伯人在中國南宋時期就經由海路與中國貿易通商。

== 歐洲地區 ==
歐洲在上古時代是文明的接納者，[[克里特島|克里特島]]據目前歷史資料的認定是最早有文化活動的地點，而此地的文化也是來自美索不達米亞與[[古埃及|古埃及]]地區。歐洲由於海岸線曲折，因此海上貿易就成為一個重要的行業。

=== 羅馬帝國對外交通 ===
==== 古羅馬遣使中國 ====
{{main|漢朝和羅馬的關係}}
中國史書上有曾經記載的大秦王安敦派使出使中國的紀錄是來自《[[後漢書|後漢書]]·西域傳》所記載的歷史事件，當時被中國史書稱之為“大秦王安敦”的古羅馬皇帝[[安東尼·庇烏斯|安東尼·庇烏斯]]（Anthonius Pius.）曾派遣使者取道當時稱之為[[日南|日南]]（今[[越南|越南]]）的地方來朝貢，然而這段歷史並不見於羅馬史書記載。

==== 拜占庭帝國對外交通 ====
1945年在[[甘肅|甘肅]][[武威|武威]]出土的拜占庭金幣即可表示出歷史上大唐帝國與拜占庭帝國之間雖然沒有政治上直接的接觸，但是確有間接接性質的往來，還有[[新疆|新疆]]、[[寧夏|寧夏]]亦有拜占廷金幣的出土，因此從這些證據可以看出當時拜占庭經濟也有稍微影響中亞附近地區。印度與[[錫蘭|錫蘭]]島也有出土大量拜占廷金幣，因此從這些出土文物來看，古代歐洲與[[東方|東方]]世界並沒有完全獨自發展，而是仍有交通往來，拜占廷的貨幣也影響到南亞。

=== 歐洲與其他地區交通 ===
==== 維京人的對外探訪 ====
西方的學者曾經在[[新疆|新疆]]地區進行過考古挖掘，並且找出了古代[[維京人|維京人]]遺骸。
==== 比哥倫布早先到達美洲的民族 ====
哥倫布並非第一個到達美洲的歐洲人，有證據顯示[[北歐|北歐]]的維京人以及居住於[[西班牙|西班牙]]的少數民族[[巴斯克人|巴斯克人]]（Basque）已經比哥倫布先到達美洲。

== 非洲地區 ==
非洲地區對外交通狀況唯一能夠闡述也只有[[古埃及|古埃及]]這一個文明，因為當上古時代的非洲，唯一有[[文明|文明]]的誕生只有在[[北非|北非]][[尼羅河|尼羅河]]流域，其他地區都多處於是原始蠻荒狀態。

=== 古代[[腓尼基人|腓尼基人]]的環繞非洲一圈 ===
腓尼基人在歷史上最著名的就是他們的航海技術相當發達，根據[[古希臘|古希臘]]著名[[歷史學家|歷史學家]][[希羅多德|希羅多德]]（Herodotus）記載指出，埃及[[法老|法老]]尼科曾經派遣他們繞行非洲；而腓尼基人一共花了三年時間突破重重困難完成任務，比後代[[葡萄牙|葡萄牙]]人要早了2000多年。

=== 古埃及與南美交通 ===
事實上這個關於〝古埃及蘆葦船跨越[[大西洋|大西洋]]到達[[南美洲|南美洲]]〞的論點，它是由上個世紀[[挪威|挪威]]的[[人類學家|人類學家]][[托爾·海爾達|托爾·海爾達]]（Thor Heyerdahl）所提出來的假說。他認為古埃及與南美地區[[的的喀喀湖|的的喀喀湖]]原住民使用的蘆葦船型制一模一樣。於是海爾達為了證明這個假說，他便以重新模擬古代埃及蘆葦船方式橫渡大西洋，並且也已經證明古埃及蘆葦船確實能夠到達南美洲。但是古埃及人曾經到美洲這項說法目前仍然顯的薄弱。

== 美洲地區 ==
事實上以今天歷史學比較主要的看法是認為美洲地區的文明事實上是一直處在孤立的地位自行發展，由於美洲東有大西洋、西有太平洋來隔離歐亞非大陸，因此這個地區的文明與外界交通要到哥倫布到達這裡之後，此地才有與歐洲文明接觸的歷史，不過根據近人考古挖掘發現美洲大陸有羅馬、維京的古文物，這點對於世界史的研究上產生很重要的一個課題。

===南美對外交通===
這項說法也是海達爾的假設，他認為[[玻里尼西亞|玻里尼西亞]]（Polynesia）人士來自於南美洲的[[祕魯|祕魯]]，不過這項說法仍然證據薄弱。
