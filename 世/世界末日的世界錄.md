《'''世界末日的世界錄'''》是[[細音啟|細音啟]]所作的[[日本|日本]][[輕小說|輕小說]]作品，[[插圖|插圖]]由{{link-ja|冬野春秋|ふゆの春秋}}繪畫。並有漫畫、廣播劇CD等衍生作品<ref>{{cite web|script-title=ja:既刊全巻重版中！ 王道ファンタジー『世界の終わりの世界録』新刊＆ドラマCD同時リリース|url=http://ddnavi.com/news/265035/a/|website=ダ・ヴィンチニュース|accessdate=2016年5月17日|date=2015年10月23日}}</ref>。

== 故事簡介 ==
英勇艾爾萊英和傳說中的三大姬為了拯救世界而戰鬥，並留下一個至寶「世界錄」，記載着世界的終焉與再臨。世界各地的國家與旅團無不尋求其所在，造就一個世界錄大爭奪時代。

終焉戰爭結束三百年後——少年雷英立志當個騎士，雖然外貌與昔日英勇一模一樣，卻缺乏劍術才華，因而被稱為「偽英勇」，過着每天受人輕視的日子。

某天，傳說中的龍姬克黎榭從封印中覺醒並出現在他面前。她誤以為雷英是艾爾萊英本人，對他虛有其表的事實感到失望，另一方面又看出雷英身上隱藏的潛力，進而邀請這名少年加入她和大天使妃雅及先代魔王艾利潔的旅程，前往探索世界錄。

「要不要跟我一起走？」──這是一首邁向騷亂的序曲，由眾多英雄演奏。現在，少年偽英勇就要和傳說邂逅。

== 登場角色 ==
; {{nihongo|雷英·E·麥斯威爾|レン＝E・マクスウェル|聲：[[内田雄馬|内田雄馬]]}}
: 17歲，聖飛歐拉旅學園1年級學生。
: 因為術士資質低下，亦缺乏劍術才華，加上外貌與昔日「艾爾萊英」一模一樣，所以被稱為「偽英勇」。

; {{nihongo|艾爾萊英·E·麥斯威爾|エルライン＝E・マクスウェル}}
: 傳說中的劍士，因為平定了引起世界滅亡危機「終焉戰爭」，而擁有「英勇」稱號，是歷史上唯一有該稱號的人。

; {{nihongo|克黎榭|キリシェ|聲：[[洲崎綾|洲崎綾]]}}
: 龍姬，統帥所有龍種，是英勇艾爾萊英昔日率領的傳說三大姬。

; {{nihongo|妃雅|フィア|聲：[[巽悠衣子|巽悠衣子]]}}
: 大天使，天界君主女神直屬的最高階天使，是英勇艾爾萊英昔日率領的傳說三大姬。

; {{nihongo|艾利潔|エリーゼ|聲：[[西明日香|西明日香]]}}
: 前任魔王，是英勇艾爾萊英昔日率領的傳說三大姬。
: 因三百年前的戰鬥使身體消滅，因此辭退王座，在十年前以現在年幼的姿態成功轉生。復活後擁有昔日一部份的記憶和力量。
