'''吐尔逊娜依·依不拉音江'''（{{bd|1964年|2月||}}）女，[[维吾尔族|维吾尔族]]，[[新疆|新疆]][[乌鲁木齐|乌鲁木齐]]人，[[中华人民共和国|中华人民共和国]]舞蹈家。

==生平==
中专学历。1978年，进入[[中央民族学院|中央民族学院]]民族舞蹈班学习，1982年毕业。1982年7月参加工作，在[[中央民族歌舞团|中央民族歌舞团]]任舞蹈演员。1985年，新疆维吾尔自治区成立三十周年，随中央代表团到新疆演出。1987年，回到新疆，进入[[新疆维吾尔自治区歌舞团|新疆维吾尔自治区歌舞团]]任独舞演员。她是新疆第四代舞蹈家的代表，是舞蹈大师[[康巴尔汗·艾买提|康巴尔汗]]的第一代学生，艺术家依不拉音江的最后一位弟子。1989年获人民喜爱的青年舞蹈家称号。1990年3月，参加了中国丝绸之路艺术家代表团访问[[日本|日本]]，表演独舞《花之恋》。1990年秋，作为中国新疆艺术家代表团成员，参加了[[苏联|苏联]][[阿拉木图|阿拉木图]]举办的“世界金苹果联欢节”，表演的舞蹈《婚礼舞》获表演奖。1990年6月，获新疆丝绸之路舞蹈大奖赛一等奖，并获十佳之首。随后在全国少数民族单双三舞蹈比赛上获得表演二等奖。1991年，应[[哈萨克斯坦|哈萨克斯坦]]邀请，第四次赴阿拉木图参加民间艺术节，表演维吾尔族民间舞蹈《刀郎姑娘》。1992年参加了大型音乐舞蹈史诗《[[中国革命之歌|中国革命之歌]]》的演出。1994年曾为[[中国中央电视台|中国中央电视台]]《[[正大综艺|正大综艺]]》栏目的特邀嘉宾。1995年新疆维吾尔自治区成立四十周年大型演出中，担任领舞，参加了中央代表团演出的大型综合文艺晚会。1995年9月，代表新疆艺术界参加了在北京举办的{{link-en|第四次世界妇女大会|World Conference on Women, 1995}}。1995年到[[上海|上海]]参加了全国少数民族舞蹈音乐展演，表演的《打鼓舞》被评为最受欢迎节目。1996年获全国有突出贡献专家称号。1996年参加了文化部春节文艺晚会。1997年参加了[[古巴|古巴]][[哈瓦那|哈瓦那]]举行的第十四届[[世界青年联欢节|世界青年联欢节]]，获评杰出艺术家、最佳表演奖。2000年7月加入[[中国共产党|中国共产党]]。后当选为[[第九届全国人大代表|第九届全国人大代表]]。2004年被评为新疆十大杰出青年，2005年被评为新疆维吾尔自治区先进工作者。享受国务院[[政府特殊津贴|政府特殊津贴]]。她担任新疆艺术剧院歌舞团党委委员、舞蹈队队长，为国家一级演员。2011年11月30日晚，在乌鲁木齐的新疆艺术剧院举办个人《圆月》舞蹈晚会。2012年成为[[中共十八大|中共十八大]]代表。<ref>{{cite news |language =  |author = |coauthors =  |url = http://www.cflac.org.cn/wywzt/2012/bangyang/daibiao/201208/t20120814_144640.html |title =  全国宣传文化系统十八大代表吐尔逊娜依·依不拉音江简历|publisher = 中国文艺网 |pages =  |date =  2012-08-14|accessdate = }}</ref><ref>{{cite news |language =  |author = |coauthors =  |url =http://cpc.people.com.cn/GB/64093/64387/17755293.html  |title =新疆出席党的十八大代表候选人初步人选公示（名单）  |publisher = 中国共产党新闻网 |pages =  |date = 2012-04-26 |accessdate = }}</ref>

==作品==
她的主要舞蹈作品有：《花之恋》、《婚礼舞》、《打鼓舞》、《刀郎姑娘》、《东方姑娘》、《摘星星》、《乌夏克木卡姆朱拉》、《圆月》、《花冠舞》等。<ref name=xjw/>

她主演的电影有《买买提外传》、《男子汉舞厅的女明星》、《西部舞狂》、《滚烫的青春》，并且在《[[阿曼尼莎汗|阿曼尼沙汗]]》影视剧中任舞蹈演员。《买买提外传》曾被[[中华人民共和国广播电影电视部|中华人民共和国广播电影电视部]]评为最佳影片奖和[[金鸡奖|金鸡奖]]特别奖。她主演的电影还曾在[[土耳其|土耳其]]、[[乌孜别克斯坦|乌孜别克斯坦]]的电影节上获最佳影片奖。<ref name=xjw/>
