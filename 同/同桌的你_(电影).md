== 剧情概述 ==
1993年的夏天，文静秀气、戴着一副小孔眼镜的转校生周小栀（周冬雨 饰），第一次见到了自己的同桌——坐在最后一排的捣蛋男生林一（林更新 饰），他们一起经历了初中、高中和大学的青葱岁月。林一对周小栀说：“总有一天，我会带你去一个只有我们俩的地方。”一晃十年，当初的承诺不知是否已实现……

== 制作 ==
1994年，一首校园民谣《同桌的你》红透中国大江南北，歌手[[老狼|老狼]]和词曲作者[[高晓松|高晓松]]亦由此成名。本片正是根据这首歌曲改编而成，讲述了一对男女同桌从初中、高中到大学的青葱记忆和甜蜜初恋。[[80后|80后]]导演郭帆表示：“《同桌的你》这首歌曲陪伴我度过了我的整个学生时代，它动人的旋律至今萦绕在我心间，这部同名电影更是勾起了我的无穷创作欲，因为它讲述的就是我们这一代人的集体青春回忆。”<ref>[http://pyrb.dayoo.com/html/2013-07/10/content_2314505.htm 周冬雨林更新上演青春恋曲] {{webarchive|url=https://web.archive.org/web/20140407075934/http://pyrb.dayoo.com/html/2013-07/10/content_2314505.htm |date=2014-04-07 }} 2013-07-10 番禺日报 </ref>

== 评价 ==
[[网易|网易]]影评认为：“[[80后|80后]]所能经历或想象的典型事件被密集又潦草地搬上银幕，正如其“同哭、同笑、同青春”的宣传词一样，成功唤起了80后的青春回忆。《同桌的你》是那种只追求热闹表象的拼盘式电影，它把青春片流行的典型元素统统拿来，讨巧地切入历史事件串联剧情，什么都往里装，“大杂烩”一样忙于罗列各种元素，搞笑段子尚能博人一笑，但人物情感描写心理刻画欠奉，情节如散沙一般，勉强跟随着主人公的成长一直往前赶，使得影片的剧情完成度极低，也许林一和周小栀真正动人的感情故事都在电影画面的背后发生了吧。<ref>[http://ent.163.com/14/0425/08/9QLP4T8200034R77.html 《同桌的妳》：成功唤起80后青春回忆]</ref>

电影影评网影评认为：“如果说《[[致青春|致青春]]》是70后的回忆，那《同桌的你》就是80后的回忆。尽管还是那些事，那些拍摄手法，但是只要把青春的代人感展现在我们眼前，电影总是会戳中自己的点，这就是青春的记忆，一个人永恒的记忆。”<ref>[http://www.yp136.com/archives/3837 《同桌的你》影评:青春是永恒的记忆] {{Wayback|url=http://www.yp136.com/archives/3837 |date=20140502005152 }}2014-04-25</ref>

[[南方周末|南方周末]]影评指出本片不止是部浅显的青春纯爱电影，抽离出那些我们80后一代人的青春记忆元素，高考、军训、四级考试、非典、留学，这个故事的根底上其实是一个女孩用最宝贵的青春、真情陪伴一个男孩漫长、不给力的成长。这个电影最精彩的地方，就在于它展示了梦想的虚妄和生活的真相，那些看起来光鲜靓丽的生活表面有着截然相反的里子<ref>[http://www.infzm.com/content/100287 《同桌的你》：等不来的成长]2014-04-30</ref>。

== 票房 ==
本片4月25日上映后大受欢迎，首周三日即票房破亿且观众口碑爆棚<ref>[http://ent.qq.com/a/20140429/000004.htm 《同桌的妳》票房三天过亿 高晓松老狼青春不散]腾讯娱乐 2014-04-29</ref>，并以1.11亿获当周票房冠军<ref>[http://weibo.com/2304129841/B1Y4b7xsl 2014第17周（4.21-4.27）电影票房]中国电影报 2014年4月29日</ref>，次周以2.17亿蝉联周冠军<ref>[http://ent.ifeng.com/a/20140506/40048038_0.shtml "同桌的妳"两亿蝉联内地冠军]2014年05月06日</ref>。截止到2014年5月21日，该片总票房已经达到4.5亿元人民币，最终票房4.561亿<ref>{{cite web|url=http://www.sarft.gov.cn/articles/2014/07/18/20140718163716930032.html|title=2014年1--6月全国电影票房统计|date=2014-07-18|publisher=国家广电总局|accessdate=2014-08-02|deadurl=yes|archiveurl=https://web.archive.org/web/20141020181932/http://www.sarft.gov.cn/articles/2014/07/18/20140718163716930032.html|archivedate=2014-10-20}}</ref>。有观点认为这部制作成本只有2000万<ref>{{cite news|url=http://yule.sohu.com/20140512/n399427068.shtml|title=闷声赚大钱 《同桌的你》的黑马之路|date=2014-05-12|publisher=搜狐|accessdate=2014-08-02}}</ref>的电影之所以票房大逆袭，一是题材优势，情怀动人，再掀全民“忆青春”；二是新媒体话题营销，人人“找同桌”；三是光线影业和影联传媒两强的联合发行<ref>{{cite news|url=http://news.xinhuanet.com/ent/2014-05/12/c_126490001.htm|title=小成本电影大逆袭 《同桌的妳》营销策略揭秘|date=2014-05-12|publisher=新华网|accessdate=2014-08-02}}</ref>。

== 更换校名事件 ==
[[华侨大学|华侨大学]]为《同桌的你》剧组提供拍摄场地，但上映的电影里华大厦门校区的校门口却被挂上了“[[厦门大学|厦门大学]]”的牌子，这引发了网上华大师生纷纷跟帖声讨。华大校长[[贾益民|贾益民]]也表示“将视事件发展情况采取进一步行动，包括法律行动，坚决维护学校名誉和权益”<ref>[http://news.163.com/14/0430/11/9R2TT4SK00014AEE.html 《同桌的你》乌龙：华大校门却被挂上厦大牌子]2014-04-30</ref>。5月9日，《同桌的你》导演郭帆、总制片人杜扬及执行制片人刘然、编剧傲立等几名主创人员来到华侨大学登门致歉，承诺在网络及DVD发行版本中删除校门的画面，并正式公开发表《致歉函》，内容与5月23日在[[泉州晚报|泉州晚报]]所登的内容大致一致，华大副校长张禹东代表学校接受了制片方的道歉。<ref>{{cite news|title=华大"校名保卫战"赢了 《同桌的你》摄制组今日在晚报头版致歉|url=http://www.fj.xinhuanet.com/dszx/qz/2014-05/23/c_1110828940.htm|accessdate=2014-05-24|newspaper=泉州晚报|date=2014-05-23|deadurl=yes|archiveurl=https://web.archive.org/web/20140902225735/http://www.fj.xinhuanet.com/dszx/qz/2014-05/23/c_1110828940.htm|archivedate=2014-09-02}}</ref>
