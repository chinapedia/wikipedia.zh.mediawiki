'''同孚大楼'''（{{Lang-en|'''Yates Apartments'''}}），原称'''中国银行西区分行'''，曾更名为'''吴江大楼'''，位于[[上海|上海]][[南京西路_(上海)|南京西路]]、[[石门一路|石门一路]]口，[[德义大楼|德义大楼]]的斜对面。

==历史==
在1935年由[[中国银行|中国银行]]出资建设<ref name="dd">[http://www.shtong.gov.cn/newsite/node2/node71994/node71995/node71999/node72013/userobject1ai77402.html 特色志 >> 上海名街志 >> 三、专业特色街道 >> 美食休闲街——吴江路] 上海地方志办公室</ref>。大楼因地处同孚路(今石门一路)上，故而得此名<ref name="dd"/>。建筑由[[陆谦受|陆谦受]]、[[吴景奇|吴景奇]]设计，大昌建筑公司承建，并于1936年竣工。原为中国银行西区分行，现底层为[[中国工商银行|中国工商银行]]静安石门一路支行所用。1994年，同孚大楼入选第二批[[上海市优秀历史建筑|上海市优秀历史建筑]]。

== 建筑特色 ==
同孚大楼为一幢半月形的公寓，共9层，高34.45米，建筑面积2887平方米，并设有电梯<ref name="xd">[http://www.shtong.gov.cn/newsite/node2/node4/node2249/node4412/node17436/node18417/node62251/userobject1ai6802.html 区县志>>区志>>静安区志>>第八编房屋建筑>>第三章旧有住宅>>第二节　公寓] {{webarchive|url=https://web.archive.org/web/20131213164126/http://www.shtong.gov.cn/newsite/node2/node4/node2249/node4412/node17436/node18417/node62251/userobject1ai6802.html |date=2013-12-13 }} 上海地方志办公室</ref>。大楼为[[现代主义建筑|现代主义建筑]]，[[钢筋混凝土|钢筋混凝土]]结构，外墙贴深褐式面砖，立面强调横线条处理。该楼因三面环路，即[[南京西路_(上海)|南京西路]]、[[石门一路|石门一路]]与[[吴江路|吴江路]]，处于边角地形，故采用点状型布局，使得在该地形十分局迫的情况下，提高容积率，以获得较多的建筑面积<ref name="xd"/>。建成时，底层和地下室为[[中国银行|中国银行]]所用；二至九层为出租公寓，每层设置二室户、三室户和四室户各一户。因地块面积所限，房间面积狭小，在当时备受批评。<ref>{{cite journal |title=公寓 （同孚大楼）|publisher=中国建筑|year=1936|issue=24|page=4-7|author=陆谦受，吴景奇}}</ref>
