'''同人'''，原指有着相同志向的人們、同好。後來轉變成指「自創、不受商業影響的自我創作」，或「自主」的創作。它比商業創作有較大的創作自由度，以及「想創作甚麼，便創作甚麼」的味道。[[同人誌|同人誌]]則是這種創作的自製出版物。這個文化圈則被稱為「同人界」。

== 語源 ==
'''同人'''一詞來自[[日語|日語]]的「{{lang|ja|どうじん}}」<ref>{{Cite book|title=网络时代的文学引渡|author=邵燕君|publisher=[[广西师范大学出版社|广西师范大学出版社]]|year=2015|isbn=978-7549577613|location=|pages=}}</ref>，原指有着相同習慣、興趣、志向等的人們，即「同好」。又指同在一處做事的人<ref>唐·[[張南史|張南史]]詩〈陸勝宅秋暮雨中探韻同作〉：「同人永日自相將，深竹閒園偶辟疆。」</ref>，即「同事」、「同仁」<ref>清·[[鄭用錫|鄭用錫]]詩〈再贈斯盛社同人〉</ref>。现代汉语中的“同人”一词最早似出现在[[新文化运动|新文化运动]]时期，[[魯迅|魯迅]]、[[周作人|周作人]]等曾使用「同人刊物」一詞来称呼他们所创办的《[[语丝|语丝]]》周刊<ref>{{Cite journal|title=我和《语丝》的始终|author=鲁迅|url=|journal=萌芽月刊|issue=2|doi=|others=|year=1930|volume=1|page=|pmid=}}</ref>，其意为非商業性“自编自写”的刊物，即编辑同时也是撰稿人。然而，現在這個同人文化的興起，以[[ACG|ACG]]文化為主流。

== 個人參與與同人社團 ==
要參與同人創作，可以以個人身份，也可以跟志同道合的朋友走在一起。如果你選擇後者，與一群有相同嗜好的人，組成「同人社團」（同人サークル，又稱「同人組織」或「同人團體」），就可以更方便地與朋友交流創作靈感，分享資訊，或製作同人誌、同人遊戲等商品販售。一些著名的商業誌作家團體如[[CLAMP|CLAMP]]，之前就是由同人社團開始發跡的。

== 同人創作 ==
同人作品包括了改編的──對正式出版的作品甚至於同人作品進行[[二次創作|二次創作]]（[[再創作|再創作]]，re-creation），也包括自創的。「同人」並不一定是再創作，也可以是自創；而「二次創作」也不一定是同人，可以是官方授權請人再創作的作品。

由於許多漫畫同人作品是以[[商業漫畫|商業漫畫]]中的人物為基礎進行的[[二次創作|二次創作]]（又稱[[再創作|再創作]]），在日常語彙的方便上，「同人」被廣泛用於指代愛好者用特定文學、動漫、電影、遊戲作品中人物再創作、情節與原作無關的文學或美術作品，即[[同人小說|同人小說]]與同人畫作的合稱。

同人界中的人，時常會創作出不同形式的同人作品，包括[[同人小說|同人小說]]、[[同人誌#漫畫上的同人誌|同人漫畫]]等。出版[[同人誌|同人誌]]是把這些同人創作跟同道中人分享的一個途徑。許多同人作品都會由同人們製成同人誌，自費出版。

但也有些同人作品，不刊在同人誌上，而是透過[[網站|網站]]發表。一般來說，繪畫[[CG|CG]]在網站發表會最為方便。而其他媒體的作品，通常都能透過網站發佈。

除此以外，還有[[同人遊戲|同人遊戲]]、[[同人音樂|同人音樂]]、同人MV（[[MAD片|MAD片]]）、同人廣播劇等创作。而在[[同人誌即賣會|同人誌即賣會]]的會場內，亦常會見到正在[[Cosplay|Cosplay]]或攜帶着[[娃娃|娃娃]]的參與者。在同人之內，並沒形式的限制。

部分同人作家由於對角色有[[性幻想|性幻想]]，也有[[H_(日語)|H]]的創作。主打[[美少女|美少女]]角色的[[ACG|ACG]]作品衍生H同人作品，這點幾乎沒有例外；甚至並非主打美少女角色的《[[七龍珠|七龍珠]]》、《[[航海王|航海王]]》、《[[火影忍者|火影忍者]]》、《[[勇者鬥惡龍系列|勇者鬥惡龍系列]]》等作品，也各有以其美少女角色為主角的H同人作品。亦有許多[[BL_(和製英語)|BL]]的同人作品。
