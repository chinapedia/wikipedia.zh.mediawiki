'''諸葛爽'''（{{bd|9世纪||886年||catIdx=Z}}），[[山东|山东]][[博昌|博昌]]人，生年不详，[[唐朝|唐朝]]将领，一度控制[[河陽|河陽]]，在朝廷和[[黄巢|黄巢]]农民起义军建立的[[齐_(黄巢)|大齐]]政权之间摇摆不定。

== 背景和早期经历 ==
诸葛爽曾在家乡当过伍长，因故被[[县尉|县尉]]杖责后开小差，沦为乞丐。[[咸通|咸通]]九年（868年）[[庞勋|庞勋]]起事攻占[[徐州|徐州]]，<ref>[[:s:资治通鉴/卷251|《资治通鉴》卷二百五十一]]</ref>诸葛爽加入了叛军，积累军功成为庞勋的小校，聚徒于青、棣之间，劫掠郡县，[[临沂|临沂]]人[[王虔裕|王虔裕]]跟随之。<ref name=HFD21>[[:s:旧五代史/卷21|《旧五代史》卷二十一]]</ref>庞勋战败后，诸葛爽和[[泗州|泗州]]守将[[陽群|陽群]]率百余士兵向朝廷投降。[[乾符|乾符]]元年（874年）四月，朝廷下诏讨民变首领[[王仙芝|王仙芝]]，褒奖诸葛爽等反正之人。<ref name=BT19-2>[[:s:旧唐书/卷19下|《旧唐书》卷一十九下]]</ref><ref>[[:s:招讨王仙芝等诏|:s:招讨王仙芝等诏]]</ref>才子[[罗隐|罗隐]]也上书招讨使[[宋威|宋威]]，举诸葛爽反正事。<ref>[[:s:上招讨宋将军书|:s:上招讨宋将军书]]</ref>诸葛爽累迁[[汝州|汝州]][[防御使|防御使]]。<ref name=BT182>[[:s:旧唐书/卷182|《旧唐书》卷一百八十二]]</ref><ref name=NBT187>[[:s:新唐书/卷187|《新唐书》卷一百八十七]]</ref><ref name=NHFD23>[[:s:新五代史/卷23|《新五代史》卷二十三]]</ref>

[[广明_(唐朝)|广明]]元年（880年），朝廷又面临[[沙陀|沙陀]]首领[[李国昌|李国昌]]及其子[[李克用|李克用]]的叛乱。二月，沙陀叛军威胁重镇[[河东道_(唐朝)|河東]]的[[太原府|太原府]]，诸葛爽奉命率领东都[[洛阳|洛阳]]守军北上赴[[代州|代州]]救援，<ref name=NHFD23/>四月被北面行营招讨使[[李琢|李琢]]表为副招讨，<ref name=BT182/><ref>《旧唐书·僖宗纪》作三月被任为副招讨。</ref>与其他东北面行营兵马归李琢统属，<ref name=BT19-2/>五月，被任为蔚、朔招讨副使，<ref name=NBT9>[[:s:新唐书/卷9|《新唐书》卷九]]</ref>又取代被召回[[长安|长安]]任[[左金吾大将军|左金吾大将军]]的[[吳師泰|吳師泰]]任[[振武节度使|振武节度使]]，但吴师泰抗命，指使军民上表挽留自己。十月，[[唐僖宗|唐僖宗]]无法迫使吴师泰从命，只得允许他留任振武，改任诸葛爽为[[夏綏节度使|夏綏节度使]]，<ref>[[:s:资治通鉴/卷253|《资治通鉴》卷二百五十三]]</ref>加检校尚书右仆射。<ref name=NBT187/><ref>《新五代史·孟方立传》载广明年间[[昭义节度使|昭义节度使]][[高浔|高浔]]攻诸葛爽于河阳，误。</ref>

十一月，僖宗命[[河东节度使|河东节度使]][[鄭從讜|鄭從讜]]将军队交给诸葛爽和代州刺史北面行营马步都虞候后院军使[[朱玫_(唐朝)|朱玫]]等军，由后者率代北行营兵马赴长安阻击黄巢，<ref>[[:s:旧唐书/卷158|《旧唐书》卷一百五十八]]</ref><ref>[[:s:赐郑从谠诏|:s:赐郑从谠诏]]</ref>郑从谠遣牙将论安率军五千跟随诸葛爽。<ref>[[:s:新唐书/卷165|《新唐书》卷一百六十五]]</ref>但在长安陷落之前，诸葛爽和朱玫并没能及时救援，十二月，僖宗被迫逃往[[成都|成都]]。黄巢称帝，建立大齐政权。时任代北行营兵马使的诸葛爽与朱玫、左武卫将军权知夏绥银节度事[[拓跋思恭|拓跋思恭]]等屯军渭桥。诸葛爽到[[同州|同州]]，率工北行营兵在长安附近的[[櫟陽|櫟陽]]驻军对抗屯东渭桥的黄巢部将[[朱温|朱温]]。黄巢命朱温诱诸葛爽投降，诸葛爽因兵败而投降了。<ref name=BT19-2/><ref name=BT182/><ref name=NBT187/><ref name=NHFD23/><ref name=NBT9/><ref>[[:s:旧五代史/卷1|《旧五代史》卷一]]</ref><ref name=ZZTJ254>[[:s:资治通鉴/卷254|《资治通鉴》卷二百五十四]]</ref><ref>此处皆从《资治通鉴》。《旧唐书·僖宗纪》作郑从谠等奉命勤王在中和元年正月，诸葛爽、朱玫、拓拔思恭等军屯渭桥及诸葛爽投降黄巢在八月。《旧唐书·郑从谠传》作奉命勤王在十二月僖宗出奔后。《新唐书·僖宗纪》作诸葛爽投降时已为河阳节度使，误。</ref>

== 接管河阳 ==
诸葛爽降齐后，被黄巢任为[[河陽节度使|河陽节度使]]，<ref name=NHFD23/>派往河阳。<ref name=BT19-2/>朝廷的河阳节度使[[羅元杲|羅元杲]]因搜刮钱财为人所怨，发兵抵抗，但他的士兵都弃甲迎接诸葛爽，羅元杲被迫逃奔僖宗[[行在|行在]]。诸葛爽就此接管河阳，同乡[[牛存节|牛礼]]跟从他。<ref name=HFD22>[[:s:旧五代史/卷22|《旧五代史》卷二十二]]</ref><ref name=NHFD22>[[:s:新五代史/卷22|《新五代史》卷二十二]]</ref><ref>《旧五代史·王虔裕传》<牛存节传>皆误作诸葛爽任河阳节度使为乾符末年事。</ref>[[中和_(唐朝)|中和]]元年（881年）三月，他却又上表僖宗，要求重归朝廷。僖宗同意了，任他为河阳节度使，<ref name=ZZTJ254/>进位检校[[司徒|司徒]]，授京师东南面招讨诸行营副都统。二年（882年）正月，任为诸道行营左先锋使。<ref>[[:s:新唐书/卷185|《新唐书》卷一百八十五]]</ref>五月，当时拥兵自重无心勤王的[[淮南节度使|淮南节度使]][[高骈|高骈]]表称“自来所用将帅，上至帅臣，下及裨将，以臣所料，悉可坐擒，用此为谋，安能集事”，僖宗下诏反驳称“卿尚不能缚黄巢于天长，安能坐擒诸将？只如拓拔思恭、诸葛爽辈，安能坐擒耶？勿务大言，不堪垂训。”<ref name=BT182/><ref>[[:s:切责高骈诏|:s:切责高骈诏]]</ref>当月，又加诸葛爽[[同中書門下平章事|同中書門下平章事]]的[[宰相|宰相]]荣衔。<ref name=ZZTJ255>[[:s:资治通鉴/卷255|《资治通鉴》卷二百五十五]]</ref><ref>[[:s:诸葛爽相公|崔致远《诸葛爽相公》]]</ref>六月，朱温据同州，诸葛爽率轻兵入城，朱温偃旗设伏以待，诸葛爽以为朱温已遁逃，军士解甲住宿，伏兵发作，诸葛爽丢弃全部铠马奔还。<ref name=NBT187/>黄巢部将[[尚让|尚让]]派朱温攻西关，败诸葛爽，又破行营右司马[[王重荣|王重荣]]，诸葛爽闭关不出。<ref>[[:s:新唐书/卷225下|《新唐书》卷二百二十五下]]</ref>

== 对抗韩简 ==
八月，邻镇的[[魏博节度使|魏博节度使]][[韓簡_(唐朝)|韩简]]野心勃勃，想吞并邻镇，亲率军三万攻打河阳，在[[修武|修武]]击败诸葛爽，诸葛爽放弃河阳逃走。<ref name=BT19-2/><ref>《新唐书·僖宗纪》《新唐书·魏博传》称此时诸葛爽仍效力黄巢，误。</ref>韩简命部将[[趙文弁|趙文弁]]驻守河阳，自己东征[[天平军|天平军]]。高骈派[[崔致远|崔致远]]写信褒扬韩简而骂诸葛爽。<ref>[[:s:魏博韩简侍中|崔致远《魏博韩简侍中》]]</ref>三年（883年）二月，河阳百姓示好诸葛爽，请求他回来。当时，诸葛爽已经到了[[金商节度使|金商]]，但得知消息后率兵一千返回河阳，进入军部[[孟州|孟州]]，<ref name=BT181>[[:s:旧唐书/卷181|《旧唐书》卷一百八十一]]</ref>魏博守军大惊。诸葛爽却礼貌地对待了他们，给予赏赐后令赵文弁率他们回魏博。<ref name=BT182/><ref name=NBT210>[[:s:新唐书/卷210|《新唐书》卷二百一十]]</ref><ref name=dis>《旧唐书·诸葛爽传》作中和元年四月魏军大败诸葛爽夺取河阳，十月诸葛爽复入河阳，十一月攻新乡，次年正月韩简被杀。《新唐书·僖宗纪》作中和二年十月诸葛爽复夺孟州。</ref>之前，黄巢在唐军压力下已经退出长安，向东返回之际，知[[光州|光州]]事[[李罕之|李罕之]]被投降黄巢的[[奉国军节度使|奉国军节度使]][[秦宗权|秦宗权]]攻打，率余部投奔诸葛爽，被诸葛爽署为[[懷州|懷州]]刺史。<ref name=NBT187/><ref name=ZZTJ255/><ref name=HFD15>[[:s:旧五代史/卷15|《旧五代史》卷一十五]]</ref><ref name=HFD56>[[:s:旧五代史/卷56|《旧五代史》卷五十六]]</ref><ref name=NHFD42>[[:s:新五代史/卷42|《新五代史》卷四十二]]</ref>黄巢所任吏部尚书的充水运使[[张居言|张居言]]也依附诸葛爽，累迁至裨校，屡有战功，诸葛爽表为[[泽州|泽州]]刺史。<ref name=HFD63>[[:s:旧五代史/卷63|《旧五代史》卷六十三]]</ref><ref name=NHFD45>[[:s:新五代史/卷45|《新五代史》卷四十五]]</ref>

这时，韩简攻天平军军部[[鄆州|鄆州]]不克，刚和天平军[[留后|留后]][[朱瑄|朱瑄]]议和，遂回攻河阳，诸葛爽派李罕之抵御魏博军，在新乡遭遇，李罕之在[[武陟|武陟]]大败韩简八万军，<ref name=BT19-2/><ref name=BT181/><ref>《旧唐书·韩简传》作中和元年十一月事。</ref>清水为之不流。韩简后也在偏将[[樂行達|樂行達]]领导的兵变中被杀。<ref name=NBT187/><ref name=ZZTJ255/><ref name=NBT210/><ref name=dis/>诸葛爽军从此大振。<ref name=BT182/>

而黄巢则仍然对河南的许多[[藩镇|藩镇]]构成威胁。[[忠武节度使|忠武节度使]][[周岌|周岌]]、[[武宁军节度使|感化节度使]][[時溥|時溥]]、[[宣武节度使|宣武节度使]]朱温（已降唐，改名[[朱全忠|朱全忠]]）都向李克用求助。李克用已经重新降唐，且在收复长安时起到重要作用。<ref name=ZZTJ255/>四年（884年）二月，时任河东节度使的李克用率蕃汉之师五万到天井关，向诸葛爽借道救援被黄巢围攻的[[陈州|陈州]]刺史[[赵犨|赵犨]]和周岌，但诸葛爽有所顾忌，借口洛阳附近跨[[黄河|黄河]]的桥尚未修好，拒绝了。他还驻军[[泽州|泽州]][[萬善|萬善]]，堙井以防李克用。<ref name=BT19-2/>最后李克用绕过河阳，改道[[河中节度使|河中]]。<ref name=BT182/><ref name=NBT187/><ref name=ZZTJ255/><ref>[[:s:新唐书/卷218|《新唐书》卷二百一十八]]</ref><ref>[[:s:旧五代史/卷25|《旧五代史》卷二十五]]</ref><ref>[[:s:新五代史/卷4|《新五代史》卷四]]</ref>

[[光启|光启]]元年（885年）三月时，据有河阳、洛阳的诸葛爽与其他众多军阀一样自擅兵赋，不纳贡，朝廷不能制，只能姑息。<ref>[[:s:旧唐书/卷164|《旧唐书》卷一百六十四]]</ref>朝廷累加诸葛爽检校[[司空|司空]]，诏为东南面招讨使，伐秦宗权，诸葛爽表李罕之为副，令屯[[宋州|宋州]]，又表为河南尹、东都留守。<ref>《旧五代史·李罕之传》误作中和四年事。</ref>秦宗权遣都将[[孙儒|孙儒]]攻东都洛阳，李罕之出奔，诸葛爽与孙儒战于洛水，败。<ref>[[:s:新唐书/卷188|《新唐书》卷一百八十八]]</ref>诸葛爽派将接收洛阳，被李罕之驱逐，诸葛爽不能制止。<ref name=NBT187/>

==最终结局==
唐僖宗[[886年|光啟二年]]十月，河陽節度使[[諸葛爽|諸葛爽]]卒，帐中大將[[劉經|劉經]]与张居言立其子[[諸葛仲方|諸葛仲方]]為[[留後|留後]]<ref name=BT182/><ref name=NBT187/><ref>[[:s:資治通鑑/卷256|《資治通鑑》卷二百五十六]]</ref><ref>《新五代史·[[王虔裕|王虔裕]]传》载[[孙儒|孙儒]]陷河阳后王虔裕随诸葛爽奔梁。此处诸葛爽疑为诸葛仲方之误。</ref>，而牛礼则率十余人改投朱全忠<ref name=HFD22/><ref name=NHFD22/>。

== 評價 ==
尽管诸葛爽以叛军起家，但善于理政，法令严明，人无怨叹，所以得到了人士的称赞和百姓的支持<ref name=BT182/>。
