《'''諸葛四郎'''》是台灣漫畫家[[葉宏甲|葉宏甲]]開始於1958年《[[漫畫大王|漫畫大王]]》（之後改名為《[[漫畫週刊|漫畫週刊]]》）上連載的漫畫系列作品。其畫風簡單、劇情緊湊，受到當時台灣小孩子的歡迎。在當紅時曾二度拍攝成電影，以及在80年代拍攝成[[電視劇|電視劇]]。

本作品可說是台灣本土漫畫萌芽時期的先驅，也是老一輩台灣人的童年回憶。也曾被一些如[[羅大佑|羅大佑]]創作的歌曲《[[童年|童年]]》中，引用寫入了「諸葛四郎和魔鬼黨，到底誰搶走了那隻寶劍」作為歌詞。<ref group="註" name="note1"/>

== 作品歷史 ==
[[File:Rd1011-L.jpg|thumb]]創作、[[張艾嘉|張艾嘉]]演唱的歌曲《童年》也引用了諸葛四郎]]
起初葉宏甲在《漫畫大王》刊物上發表《諸葛四郎》時，原先只計畫以短篇的形式發表，之後因受大量讀者迴響，而開始展開長篇作品的創作。<ref>{{cite book |title=傑出漫畫家. 亞洲篇 |year=2000年 |publisher=雄獅圖書 |author=[[洪德麟|洪德麟]] |isbn=957-474-005-6 |pages=19頁 |accessdate=2014年3月23日}}</ref>

於《漫畫大王》上連載時，葉宏甲推出了《[[大戰魔鬼黨|大戰魔鬼黨]]》、《[[大鬥雙假面|大鬥雙假面]]》等系列故事。之後1963年因《漫畫週刊》停止發售，葉宏甲另自行開設出版公司「宏甲出版社」，於宏甲出版社發表單行本《[[大戰金銀島|大戰金銀島]]》、《[[蛇谷風雲|蛇谷風雲]]》等其它作品後才結束創作。<ref>{{cite book |title=臺灣漫畫閱覽 |date=2000年3月 |publisher=玉山出版 |author=[[洪德麟|洪德麟]] |isbn=986-7819-19-5 |pages=21頁 |accessdate=2014年3月23日}}</ref><ref>{{cite book |title=非凡台灣人 |year=2002年 |publisher=遠流出版社 |author=陳怡方 |isbn=957-32-4584-1 |pages=156頁 |accessdate=2014年3月23日}}</ref>

後續二十多年後，葉宏甲先前創作的《諸葛四郎》系列作品，則改由故鄉出版社重新印製於1990年再版。2003年2月24日，葉宏甲之子葉佳龍將本作手稿與漫畫捐贈給[[國立交通大學|國立交通大學]]典藏。<ref>{{Cite web|url=http://140.113.2.143/Report/report_more.php?id=2192|title=諸葛四郎 網路世界復出|work=國立交通大學|publisher=[[聯合報|聯合報]]|author=彭芸芳|date=2003年2月25日|accessdate=2014年3月23日}}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

==作品人物==
《諸葛四郎》系列故事主要圍繞在一個虛構的國度「衛國」裡，由名為「[[諸葛四郎_(漫畫角色)|諸葛四郎]]」的劍客和他的義弟夥伴「[[真平|真平]]」一同打倒入侵、威脅衛國安危的賊黨惡人。而隨著章節發展，葉宏甲另增加樵夫之子「林少青」暱稱（林小弟）、俠女「沈少玉」，以及四郎之妹「諸葛玉真」等人成為諸葛四郎的夥伴。<ref>{{Cite web|url=http://ir.lib.ntnu.edu.tw/retrieve/47529/metadata_02_07_s_05_0014.pdf%E2%80%8E|title=製造英雄─以葉宏甲漫畫「諸葛四郎」系列為中心|author=林佳叡|format=pdf|work=國立臺灣師範大學臺灣史研究所|pages=65～67頁|accessdate=2014年3月23日|date=2000年5月}}{{dead link|date=十月 2017 |bot=InternetArchiveBot }}</ref>

在反派角色上，葉宏甲多以蒙面的方式作為此類型人物的造型風格。另因葉宏甲曾前往日本教育，而被一些漫畫研究書籍認為受到當時日本漫畫的影響。<ref>{{cite book |title=台灣漫畫風城50年 |year=1999年 |publisher=新竹市立文化中心 |isbn=957-024-745-2 |pages=79頁 |accessdate=2014年3月23日}}</ref>

== 作品世界觀 ==
故事的舞台上以諸葛四郎的母國「衛國」為中心，而衛國兩旁有著兄弟國「桂國」和友邦「梁國」。而反叛角色所屬的國家則以「章國」、「韋國」為主。<ref>{{cite book |title=台灣漫畫風城50年 |year=1999年 |publisher=新竹市立文化中心 |isbn=957-024-745-2 |pages=104頁 |accessdate=2014年3月23日}}</ref>

後續在《[[魔境歷險記|魔境歷險記]]》中，另有設定出登場各種野獸、怪物等荒蠻的國家「加達國」、「百獅國」等。<ref>{{Cite web|url=http://ir.lib.ntnu.edu.tw/retrieve/47529/metadata_02_07_s_05_0014.pdf%E2%80%8E|title=製造英雄─以葉宏甲漫畫「諸葛四郎」系列為中心|author=林佳叡|format=pdf|work=國立臺灣師範大學臺灣史研究所|pages=101頁|accessdate=2014年3月23日|date=2000年5月}}{{dead link|date=十月 2017 |bot=InternetArchiveBot }}</ref>

電視劇版由[[華視|華視]]改編推出，以《四郎與真平》名稱發表<ref group="註" name="note2" />，於1985年6月1日至9月21日的每週六下午12:45至14:15播出，每集90分鐘，共17集。其主題曲則由[[陶大偉|陶大偉]]創作。<ref>{{Cite web |url=http://www.rti.org.tw/ajax/recommend/Literator_content.aspx?id=125 |title=台灣文學作家系列 葉宏甲 |publisher=中央廣播電台 |accessdate=2014年3月23日 }}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>
 
而《四郎與真平》的劇情與漫畫原作有一些不同。漫畫裡的四郎與真平是好朋友，並無血緣關係；電視劇則將四郎與真平之間的關係更改為失散多年的同父異母兄弟，並增加了一位將真平撫養長大的阿姨「雲仙仙」。另外，原本漫畫裡幼時父親遭黑騎士殺害、與年邁母親及妹妹玉真一同生活的諸葛四郎，電視劇裡改為母親被殺（凶手正是雲仙仙）、父親諸葛長風（同時也是真平之父）則活得好好的。
