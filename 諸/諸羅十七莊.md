'''諸羅十七莊'''，係指[[清朝|清朝]]治臺初期首先設置於[[諸羅縣|諸羅縣]]境內，已為移民開墾的北路村莊，其中的外九莊是[[漢族|漢族]]移民最早拓墾的聚落<ref> 《諸羅縣志》卷二規制志中「坊里」，臺灣省文獻會，1993年6月20日版 </ref>。

==  漢人入墾 ==
[[明朝|明朝]][[萬曆|萬曆]]二年(1574)，海盜林鳳被迫逃往東番[[魍港|魍港]](今[[嘉義縣|嘉義縣]][[布袋鎮|布袋鎮]]好美里)，三年(1575)又被明軍所敗，再次遁入魍港，更前進到劈破甕(今嘉義縣水上鄉)。[[天啟|天啟]]元年(1621)，[[顏思齊|顏思齊]]率眾侵占東番，以魍港爲根據地，與部眾分設十寨，便利於出獵海上。所謂十寨，應是多數之義，是指清領初期所謂的「外九莊」一帶。諸羅縣首任知縣季麒光說：「台灣天末荒島．．。三代以來不通貢賦、不登記載，自海盜[[顏思齊|顏思齊]]竊踞其地，乃有台灣之名，而中國民實始居之。思齊死，紅彝因其餘眾，用以耕作，民番雜處，漸成殷阜。」<ref>《臺灣府志》序 </ref> 。
天啟五年(1625)，[[鄭芝龍|鄭芝龍]]歸附顏思齊，同年七月顏思齊至[[諸羅山|諸羅山]]打獵返回，病卒。鄭芝龍繼為部眾首領，所謂「外九莊」成為鄭的私有領地。崇禎元年(1628)，鄭率所部降於明朝政府，離開臺灣。三年(1630)年荷蘭人勢力進入魍港，九年(1636)築菲立辛根碉堡(清時稱青峰闕炮台，即今布袋鎮好美里)。明永曆十五年(1661)四月，鄭成功登陸鹿耳門（今臺南市安南區顯宮里），次年二月收復熱蘭遮城(今安平古堡)，結束荷蘭在臺灣的統治。鄭氏軍隊派至各地駐紮，一面開墾，一面守衛。康熙二十三年(1684)，臺灣納入清政府版圖，設一府三縣，[[臺灣府|臺灣府]]，[[臺灣縣|臺灣縣]]、[[諸羅縣|諸羅縣]]、[[鳳山縣|鳳山縣]]。其中[[諸羅縣|諸羅縣]]共轄四里、三十四社。而所謂「外九莊」，成為施琅家族的私有領地，不在編制內。
== 外九莊 == 
[[鄭芝龍|鄭芝龍]]入墾臺灣事蹟，記載於黃宗羲所著《賜姓始末》一文<ref>《行朝錄》卷十一賜姓始末，《黃宗羲全集》第二冊，里仁書局，七十六年(1987年)出版。</ref>，說崇禎元年至五年期間(1628-1632)，福建發生饑荒，巡撫請鄭想辦法，於是想出「三金一牛」理論，巡撫同意這個論調，以船舶搭載饑民數萬人東渡臺灣，分地耕種，並結有九個村落。一代大儒黃宗羲的論著說法較可疑的是「饑民數萬」及鄭入墾臺灣的時間點，與實際較不符合，但鄭於1625年至1628年曾為[[臺灣|臺灣]]開墾的首領。在明鄭統治期間(1661年至1683年)，此所謂「外九莊」仍為鄭氏家族私有領地。清領之後，外九莊為施琅家族繼續佔有，其所收租稅、穀糧不必上繳朝廷。
清領初期的外九莊是大小槺榔莊、井水港莊、茅港尾莊、鹿仔草莊、龜佛山莊、南勢竹莊、大丘田莊、龜仔港莊、大奎璧莊。後官府接龜仔港莊、大奎璧莊，補入北新莊、土獅仔莊，即1717年《諸羅縣志》所記載的外九莊。

== 十七莊 ==
清[[康熙|康熙]]二十三年(1684)，依《[[臺灣府志|臺灣府志]]》記載，諸羅縣下轄里四、社三十四，並無莊的設置。
三十三年(1694)，依《[[重修臺灣府志|重修臺灣府志]]》記載，[[諸羅縣|諸羅縣]]下轄里四、莊十四、社四十，十四莊是指，新舊咯莊、大奎壁莊、下茄苳莊、井水港莊、鹿仔草莊、龜佛山莊、南勢竹莊、大坵田莊、龜仔港莊、槺榔莊、諸羅山莊、打貓莊、他裡霧莊、半線莊。
五十五年(1717)，依《[[諸羅縣志|諸羅縣志]]》記載，縣設四里九保十七莊，「十七莊」出現在史冊裡有其歷史意義，一是開墾人口及面積數量的增加，二是縣治重心移到諸羅山莊(今嘉義市正式登上政治舞臺 ，並於1704年創建[[臺灣|臺灣]]首座城池)，另一是仍保有「外九莊」的雙重規制。除上述十四莊，新增加諸羅山莊、北新莊、土獅仔莊，後二者是替補外九莊。
[[雍正|雍正]]元年(1723)，以虎尾溪的北方，增設彰化縣、淡水廳。原半線莊移屬彰化縣。
乾隆五年(1740)，依《[[重修福建憂灣府志|重修福建憂灣府志]]》及《[[重修臺灣府志|重修臺灣府志]]》記載[[諸羅縣|諸羅縣]]下轄里四、保七、「莊十七」。新增茅港尾莊。這是史冊最後關於十七莊記載。到了二十七年(1762)，依《[[續修臺灣府志|續修臺灣府志]]》所記，舊四里、七保、十七莊，新增三十九保、一莊，即有四十五保、十八莊，增加臺仔挖莊。五十二年(1787)，[[諸羅縣|諸羅縣]]改名[[嘉義縣|嘉義縣]]，縣治諸羅山亦改名[[嘉義|嘉義]]。
[[道光|道光]]年間(1821-1850)，保下置莊，[[嘉義縣|嘉義縣]]下轄四十一保，及數百個莊。
同治元年(1862)，依《[[臺灣府輿纂要|臺灣府輿纂要]]》記錄，[[嘉義縣|嘉義縣]]下轄保三十六，莊九三六<ref>《十七莊》，賴景寔著，載於南靖草堂，2011年6月2日。 </ref>。

== 先民足跡 == 
關於明清先民開墾[[臺灣|臺灣]]事跡或見於史冊，或私人筆記，或傳聞，有頗多爭議，如[[顏思齊|顏思齊]]登台地、十寨現今所在地。(一)魍港，連橫於所著《臺灣通史》說「北港一名魍港，即今之笨港，地在雲林縣西」，認為在笨港(今[[雲林|雲林]][[北港|北港]])，現北港鎮西勢里亦立有「顏思齊開拓台灣登陸念碑」，但有學者以為荷蘭古地圖將Poonkan(笨港)及Wanckan(魍港)分別標示，此笨港非魍港(應在布袋鎮好美里)。(二)十寨，有以為現址在今雲林縣水林鄉、北港鎮，但在1621年時這十個現址多還在海上；與「外九莊」的開墾過程及位置更為一致。
