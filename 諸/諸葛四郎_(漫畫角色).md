'''諸葛四郎'''為臺灣漫畫家[[葉宏甲|葉宏甲]]所發表的《[[諸葛四郎|諸葛四郎]]》系列漫畫作品主角。

== 角色來源 ==
諸葛四郎名字的來源為作者葉宏甲引用歷史人物[[諸葛亮|諸葛亮]]和[[楊延輝_(楊家將)|楊四郎]]的組合。<ref>{{cite book |title=台灣漫畫風城50年 |year=1999年 |publisher=新竹市立文化中心 |isbn=957-024-745-2 |pages=79頁 |accessdate=2010年9月18日}}</ref><ref>{{cite book |title=漫畫百寶箱 : 自編．自導．畫漫畫 |date=2000年5月 |publisher=[[雄獅圖書|雄獅圖書]] |isbn=957-474-007-2 |pages=74頁 |accessdate=2010年9月18日}}</ref>，而當時的立意為要讓此角色能展現諸葛亮的「忠」與楊四郎的「孝」之雙全表現。<ref>{{cite book |title=臺灣漫畫文化史 : 從文化史的角度看臺灣漫畫的興衰 |year=2006年 |publisher=杜葳廣告 |isbn=957-29-6548-4  |pages=45頁 |accessdate=2014年3月18日}}</ref><ref>{{Cite web|url=http://ir.lib.ntnu.edu.tw/retrieve/47529/metadata_02_07_s_05_0014.pdf%E2%80%8E|title=製造英雄─以葉宏甲漫畫「諸葛四郎」系列為中心|author=林佳叡|format=pdf|work=[[國立臺灣師範大學|國立臺灣師範大學]]臺灣史研究所|pages=49頁|accessdate=2014年3月18日|date=2000年5月}}{{dead link|date=十月 2017 |bot=InternetArchiveBot }}</ref>

葉宏甲曾提及，諸葛四郎的外型為自身在[[鏡子|鏡子]]裡的模樣給修改、再理想化後的模樣。<ref>{{cite book |title=台北人物誌 第參冊 |date=2000年11月 |publisher=[[台北市政府新聞處|台北市政府新聞處]] |isbn=957-02-7166-3  |pages=206頁 |accessdate=2014年3月18日}}</ref><ref>{{Cite web|url=http://ir.lib.ntnu.edu.tw/retrieve/47529/metadata_02_07_s_05_0014.pdf%E2%80%8E|title=製造英雄─以葉宏甲漫畫「諸葛四郎」系列為中心|author=林佳叡|format=pdf|work=國立臺灣師範大學臺灣史研究所|pages=50頁|accessdate=2014年3月18日|date=2000年5月}}{{dead link|date=十月 2017 |bot=InternetArchiveBot }}</ref>

== 作品的描述 ==
諸葛四郎為虛構國家「衛國」裡的一名劍士，任職於該朝廷的將軍，在作品中定位為掃除各種惡人賊黨的角色。外觀上頭頂綁著兩個圓形的髮髻為其最大特徵。初期身著長袖的袍衣，於《[[大戰雙騎士|大戰雙騎士]]》中改成較輕便的短袖上衣、手臂前臂纏繞白布帶的外型。

個性上擁有智勇兼具、忠孝雙全的性格，在劇情中多次表現各種機智、沉穩深思的場面。如在《[[大戰魔鬼黨|大戰魔鬼黨]]》中，藉著對手行為上的破綻、進而拆穿出對方賊酋假面具下的真正身份。而於《[[決戰黑蛇團|決戰黑蛇團]]》裡的攻城戰中展露出擅於各種攻防戰略的一面，另在《[[大鬥雙假面|大鬥雙假面]]》也出現著不少陷入敵方陣營、急中生智的情節。

葉宏甲有隨著不同的故事發展讓四郎奪得利器「青雲劍（遇火而削鐵如泥）」、寶馬「飛雲」等各種提昇武力的物品，<ref>{{cite book |title=台灣漫畫風城50年 |year=1999年 |publisher=新竹市立文化中心 |isbn=957-024-745-2 |pages=80頁 |accessdate=2010年9月18日}}</ref>之後《[[大戰龍虎十劍士|大戰龍虎十劍士]]》中則讓他習得武林絕學「龍風掌」；但此設定打亂了往後作品人物的強弱平衡表現，<ref>{{Cite web|url=http://ir.lib.ntnu.edu.tw/retrieve/47529/metadata_02_07_s_05_0014.pdf%E2%80%8E|title=製造英雄─以葉宏甲漫畫「諸葛四郎」系列為中心|author=林佳叡|format=pdf|work=國立臺灣師範大學臺灣史研究所|pages=37頁|accessdate=2014年3月18日|date=2000年5月}}{{dead link|date=十月 2017 |bot=InternetArchiveBot }}</ref>而讓葉宏甲在《[[蛇谷風雲|蛇谷風雲]]》中作了取消了四郎身上絕技的更動。

四郎在作品最好朋友為他的義弟[[真平|真平]]；而後續相識的樵夫之子「林小弟」林少青、異性友人沈少玉，以及四郎的妹妹諸葛玉真，都是在往後的作品中與四郎一同奮戰的夥伴。另四郎所處的衛國旁的「桂國」與「梁國」皆有四郎的盟友。

== 真人演出 ==
最早《諸葛四郎》系列改編的真人演出的作品1962年電影《雙雄大鬥雙假面》裡，諸葛四郎是由[[唐菁|唐菁]]飾演<ref>{{Cite book |title=聯合報印縮本25(中華民國五十一年一至四月) |location=台灣 |publisher=[[聯經出版|聯經出版]] |date=1980年9月 |pages=283頁 |id=行政院新聞局出版事業登紀證局版台業字第0130號 |accessdate=2014年3月18日}} </ref><ref>{{Cite web |url=http://subtpg.tpg.gov.tw/web-life/taiwan/9910/9910-10.html |title=【懷舊系列】之10 諸葛四郎與真平 |publisher=[[臺灣省政府|臺灣省政府]]《臺灣月刊雙月電子報》|author=李慶恭|date=2010年10月|accessdate=2014年3月18日 }}</ref>。之後1978年電影版《諸葛四郎大鬥雙假面》中則改由女星[[上官靈鳳|上官靈鳳]]反串演出，<ref>{{Cite web |url=http://folkartist2.e-lib.nctu.edu.tw/collection/yeh/03_report/03_1977report04.htm |title=閃亮的星群 他們都是主角 |work=浩然藝文數位博物館 |publisher=[[國立交通大學|國立交通大學]] |accessdate=2014年3月18日}}</ref>而1985年電視版《四郎與真平》則是[[衛子雲|衛子雲]]擔當，而先前衛子雲曾在電影《諸葛四郎大鬥雙假面》中扮演真平。<ref>{{Cite web |url=http://folkartist2.e-lib.nctu.edu.tw/collection/yeh/03_report/03_1977report08.htm |title=諸葛四郎大鬥雙假面 暢銷漫畫名著上銀幕 |work=浩然藝文數位博物館 |publisher=[[國立交通大學|國立交通大學]] |accessdate=2014年3月18日 }}</ref>

== 影響評價 ==
在葉宏甲[[中風|中風]]時期，當時主治葉宏甲的醫生曾向葉宏甲表示，他小時候便是諸葛四郎的漫畫迷、並曾一度因學業一事感到灰心，之後則是想到諸葛四郎在作品裡的勇氣表現而重新激起「不放棄」念頭。<ref>{{Cite book |title=聯合報印縮本83(中華民國六十七年一至二月) |location=台灣 |publisher=聯經出版 |date=1981年5月 |pages=201頁 |id=行政院新聞局出版事業登紀證局版台業字第0130號 |accessdate=2014年3月18日}} </ref>

諸葛四郎曾被歌手[[羅大佑|羅大佑]]引用在1981年歌曲《童年》<ref>{{cite book |title=傑出漫畫家. 亞洲篇 |year=2000年 |publisher=雄獅圖書 |author=[[洪德麟|洪德麟]] |isbn=957-474-005-6 |pages=21頁 |accessdate=2014年3月18日}}</ref>，也被歌手[[陶大偉|陶大偉]]引用在1985年歌曲《諸葛四郎》。[[洪德麟|洪德麟]]等漫畫研究者則將諸葛四郎定位為台灣漫畫史上第一個漫畫英雄偶像。<ref>{{cite book |title=非凡台灣人 |year=2002年 |publisher=遠流出版社 |author=陳怡方 |isbn=957-32-4584-1 |pages=156頁 |accessdate=2014年3月18日}}</ref><ref>{{cite book |title=臺灣歷史辭典 |date=2004年5月18日 |publisher=遠流出版社 |isbn=957-01-7429-3 |pages=990頁 |accessdate=2014年3月18日}}</ref>
