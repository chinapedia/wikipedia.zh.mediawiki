'''卡西欧 fx-82MS'''是[[卡西欧|卡西欧]]发行的一款[[科学计算器|科学计算器]]。该款计算器用于取代原先的型号[[卡西欧_fx-82TL|fx-82TL]]，并在原型号上进行改进，增加了功能。fx-82MS为卡西欧MS系列计算器的入门级机型。

== 历史 ==
fx-82MS上市于2000年代初期，2003年进入[[中华人民共和国|中华人民共和国]]市场。

2004年，卡西欧发行了支持全点阵自然显示的[[卡西欧_fx-82ES|fx-82ES]]作为fx-82MS的后继者，之后fx-82MS在[[中华人民共和国|中华人民共和国]]大陆地区停止销售。然而fx-82MS直到现在仍在生产，并在其他国家或地区销售。

== 功能 ==
fx-82MS使用点阵字体和七段数码管双行显示，使用线性输出格式，具有以下功能：
* 基本四则运算
* 存储器运算
* 变量运算（8个变量、1个独立存储器、一个答案存储器）
* 隐含乘法
* 二次、三次、多次乘方、开方以及求倒数
* 以10或[[E_(数学常数)|e]]为底的[[指数|指数]]、[[对数|对数]]函数
* [[圆周率|圆周率]]和[[E_(数学常数)|e]]常数输入
* [[三角函数|三角函数]]、[[反三角函数|反三角函数]]
* [[双曲函数|双曲函数]]、[[反双曲函数|反双曲函数]]
* 阶乘函数
* 分数计算
* 百分比计算
* 度分秒计算
* [[科学记数法|科学记数法]]转换
* 计算历史记录
* 算式重放
* 排列组合运算
* [[伪随机性|伪随机数]]、随机整数生成
* [[数值修约|四舍五入]]
* [[笛卡儿坐标系|直角坐标]]、[[极坐标系|极坐标]]转换
* 单变量、双变量统计计算，包括求和、[[平均数|平均数]]、[[標準差|標準差]]计算、双变量的[[线性回归|线性回归]]、对数回归、e指数回归、乘方回归、逆回归、二次回归

== 版本变迁 ==
随着PLUS系列的推出，新批次的fx-82MS改用PLUS系列的硬件，外观没有任何变化。软件方面，新批次的fx-82MS使用与ES PLUS系列小字体相同的字体，内部计算精度从12位提高到15位，新增了液晶显示器[[对比度|对比度]]调节，以及关机时在七段数码管上显示“CASIO”字样的画面，并修复了旧批次软件中存在的大量[[程序错误|程序错误]]。
