'''柯利弗德·卡伍德“李普”李普頓'''少尉（{{lang|en|Clifford Carwood "Lip" Lipton}}，{{bd|1920年|1月30日|2001年|12月16日|catIdx=Lipton}}）是[[美國陸軍|美國陸軍]][[美國陸軍第101空降師|101空降師]]{{link-en|美国陆军第506步兵团|506th Infantry Regiment (United States)|第506步兵團}}第2營[[E連|E連]]的[[軍官|軍官]]。李普頓在1942年入伍的軍階是[[二等兵|二等兵]]。在歐洲戰場時晉升為連[[軍士長|軍士長]]，其後李普頓獲得[[戰場委任|戰場委任]]而成為[[少尉|少尉]]。李普頓說這是對他最大的榮譽。

== 早年生活 ==
李普頓在[[西維吉尼亞|西維吉尼亞]][[亨廷顿_(西弗吉尼亚州)|-{zh-hans:亨廷顿; zh-hant:亨廷顿;}-]]出生和長大。在十歲時父親因交通意外逝世，成為了家中年紀最大的男人，後獲得母親的信賴。由於家中的經濟困難和希望從事軍事生涯，因此就讀了亨廷顿[[馬歇爾大學|馬歇爾大學]]一年就離開。其後在[[生活雜誌|生活雜誌]]中閱讀了一編有關傘兵訓練的文章，該文章講述傘兵訓練及其多麼困難，並在1942年，李普頓應徵入伍及加入[[傘兵|傘兵部隊]]。

== 服役期間 ==
在1944年6月6日的[[D日|D日]]行動中，李普頓是由[[C-47|C-47]]空降[[諾曼第|諾曼第]]的傘兵之一，著陸後與[[E連|E連]]指揮官[[理查德·溫特斯|理查德·溫特斯]][[中尉|中尉]]及兩個[[美國陸軍第82空降師|82空降師]]傘兵會合，其後會合更多E連隊友在[[卡朗唐|卡朗唐]]南部作戰。

李普頓與[[二等兵|二等兵]][[邁倫·蘭尼|邁倫·蘭尼]]參與[[布里考特突擊|布里考特突擊]]，E連被指派破壞[[猶他海灘|猶他海灘]]的4門105毫米榴彈砲（原為88毫米，但當他們到達時已改為105毫米）。完成任務後李普頓獲得[[銅星勳章|銅星勳章]]。

李普頓參與[[卡倫坦|卡倫坦]]突擊時被[[德軍|德軍]]88毫米砲彈碎片擊中受傷，其後獲得[[紫心勳章|紫心勳章]]。李普頓離開了連數天及其後很高興能返回E連。

李普頓參與E連攻佔[[埃因霍溫|埃因霍溫]]的行動，並和一部分前沿的的偵察隊偵察前方橋樑靜止的連隊。

[[飛馬行動|飛馬行動]]是一個低風險的小規模行動，E連在[[加拿大|加拿大]][[工兵|工兵]]幫助下，利用小艇帶回140名在[[阿纳姆|阿纳姆]]陷入困境的[[英軍|英軍]][[紅魔鬼部隊|紅魔鬼部隊]]。這行動在李普頓和[[佛瑞德·海力格|佛瑞德·海力格]]中尉監督下順利完成。

在比[[賀伯·索柏|賀伯·索柏]]更差的新連長[[諾曼·戴克|諾曼·戴克]]中尉指揮下，李普頓帶給在[[阿登森林|阿登森林]][[佛依|佛依]]的士兵們希望，亦成為E連的非正式指揮官。當E連成功突擊進入佛依的時候，李普頓被告知將會獲得戰場委任成為[[少尉|少尉]]。隨後李普頓在[[海格納|海格納]]獲得戰場委任成為少尉。

E連在[[蘭茨貝格|蘭茨貝格]]發現令人震驚的[[猶太人大屠殺|猶太人大屠殺]]集中營，之後並解放它。李普頓和E連佔領象徵納粹之家的[[貝希特斯加登|貝希特斯加登]]。在這裡李普頓認識了能說一口流利英語的[[斐迪南·保時捷|斐迪南·保時捷]]（[[豹式坦克|豹式]]和[[虎I|虎式]]坦克的設計者）。他們在L.A.G.A.R戰俘營共進了一餐。

李普頓在二戰結束前一直留在E連。在[[德国|德国]]及[[日本|日本]]正式投降後，E連解散。於[[韓戰|韓戰]]期間，他一直留在陸軍服務，但並沒有再晉級。

== 退役後 ==
李普頓回到[[美國|美國]]後，申請進入[[馬歇爾大學|馬歇爾大學]]，完成最後三年的學業，並獲得[[工程學|工程學]]學士學位。

== 李普頓在媒體 ==
卡伍德·李普頓曾經在兩個電視節目出現過。在电视剧《[[諾曼第大空降|諾曼第大空降]]》中，由演员[[唐尼·華伯格|唐尼·華伯格]]飾演他。

== 逝世 ==
卡伍德·李普頓在2001年12月16日因肺囊腫性纖維化，病逝於[[北卡羅萊納州|北卡羅萊納州]]的南松市。他的過世對尚存的E連老兵而言，是個重大的損失。
