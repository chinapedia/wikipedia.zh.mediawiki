'''卡拉細盲蛇'''（[[學名|學名]]：''Leptotyphlops carlae''）是[[蛇亞目|蛇亞目]][[細盲蛇科|細盲蛇科]][[細盲蛇屬|細盲蛇屬]]下的一種小型盲蛇，也是目前世界上體型最細小的蛇類。<ref name="zootaxa"/>主要分布於[[加勒比海|加勒比海]]群島中的[[巴巴多斯|巴巴多斯]]，因此其英語俗稱是「巴巴多斯盲蛇（Barbados Threadsnake）」。首位將卡拉細盲蛇列為獨立物種的人，是任教於[[賓夕法尼亞州立大學|賓夕法尼亞州立大學]]的生物學家赫傑斯（S. Blair Hedges）。<ref>[http://evo.bio.psu.edu/hedgeslab/People/sbh.htm S. == 特徵 ==
[[File:Relative-offspring-size-hi-res.jpg|thumb]]
成年卡拉細盲蛇的體長約為10[[公分|公分]]，而目前紀錄中最長者亦僅達10.4公分。<ref name="zootaxa"/>此蛇曾被形容為「幼如[[意大利粉|意大利粉]]」。條目上方顯示了一條卡拉細盲蛇與一枚[[直徑|直徑]]長24.26毫米的[[25美分硬幣|25美分硬幣]]並列的情況，可以作為此蛇體型的比例參照。

卡拉細盲蛇以[[白蟻|白蟻]]及蟻卵為主要食糧。繁殖方面，盲蛇均屬於[[卵生|卵生]]蛇類，此類卡拉細盲蛇的雌蛇，每次產卵均只能產出一枚較大顆的蛇卵；初生幼蛇的體型，已足有母蛇的一半體積。<ref name="eureka">Barbara K. Kennedy. "[http://www.eurekalert.org/pub_releases/2008-08/ps-wss072808.php. World's smallest snake found in Barbados]{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}". Penn State University. Retrieved on 2008-08-03.</ref>

== 地理分布 ==
目前學界相信卡拉細盲蛇是[[加勒比海|加勒比海]][[巴巴多斯|巴巴多斯]]島的[[特有種|特有種]]，至今有兩條研究用的活蛇便是來自巴巴多斯島中部至東部的一個[[再生森林|再生森林]]遺址附近，該區亦是巴巴多斯島最古老的地帶。<ref name="zootaxa"/>再生森林人跡罕至，似乎為卡拉細盲蛇提供了理想的棲息環境。但現在的巴巴多斯島遭受[[森林開伐|森林開伐]]的影響，目前可以讓卡拉細盲蛇棲息的適當地帶，大概僅餘不足數[[平方公里|平方公里]]之地。

== 保育狀態 ==
目前關於卡拉細盲蛇的生態、數量及分布情況資料均甚不足。<ref name="zootaxa"/>卡拉細盲蛇似乎慣性以森林地帶為棲息對象，但現在巴巴多斯島上的森林已所餘無幾，這對卡拉細盲蛇可能是一個威脅。<ref name="zootaxa"/>基於暫時所知的少量卡拉細盲蛇只集中分布於巴巴多斯島的東部，因此其未來的生存空間是一個相當值得顧慮的生態問題。
