'''卡博特公司'''（Cabot Corporation）是一家特殊化工與應用材料公司，於1882年成立，總部位於[[美國|美國]][[麻薩諸塞州|麻薩諸塞州]]的[[波士頓|波士頓]]，其產品業務範圍涵蓋四大類：[[碳煙|碳煙]]、特種[[流體|流體]]、超級金屬、金屬氧化物等。

== 早年歷史 ==
1880年代時，碳煙早就被廣泛應用於[[報紙|報紙]]及[[雜誌|雜誌]]的[[印刷|印刷]]油墨，以及新興的[[汽車|汽車]]工業使用的[[輪胎|輪胎]]與其他[[橡膠|橡膠]]製品。{{link-en|高弗瑞·卡博特|Godfrey Lowell Cabot}}（Godfrey Lowell Cabot）於1882年申請「碳煙製造設備」的[[專利|專利]]，並創建'''高弗瑞卡博特公司'''（Godfrey L. Cabot, Inc.，為卡博特公司之前身）。到了1890年，卡博特公司已經成為全美第四大碳煙製造廠。

== 產品範圍 ==
卡博特公司的主要產品包含：
# [[碳煙|碳煙]]：橡膠用碳煙、特殊碳煙
# [[氣凝膠|氣凝膠]]
# 母料（masterbatch）：[[顏料|顏料]]、添加劑濃縮液
# 超級[[金屬|金屬]]：[[鉭|鉭]]、[[鈮|鈮]]
# [[二氧化硅|二氧化矽]]、[[氧化鋁|氧化鋁]]
# 特種流體：石油業使用的甲酸銫鑽井流體、完井液
# 噴墨色漿
# 超微粒粉末

== 批評與非議 ==
=== 環境汙染 ===
山姆·博德曼（Sam Bodman）曾於1987年至2000年間出任卡博特公司的執行長，隨後於2004年被當時的[[美國總統|美國總統]][[乔治·沃克·布什|布希]]任命為[[美國能源部長|能源部部長]]。不過根據《[[赫芬頓郵報|赫芬頓郵報]]》（The Huffington Post）[[記者|記者]]{{link-en|傑生·利奧波德|Jason Leopold}}（Jason Leopold）2005年9月29日的撰文指出，在博德曼任內的卡博特公司是全美最大的污染者，每年排放6萬[[英噸|英噸]]的毒霧<ref>參看{{en}}[http://www.huffingtonpost.com/jason-leopold/the-environment-is-doomed_b_8083.html Jason Leopold: The Environment Is Doomed]一文，赫芬頓郵報，2005年9月29日。</ref>。

=== 在非洲採礦 ===
為了響應[[聯合國安全理事會|聯合國安全理事會]]之文件（編號S/2001/357），卡博特公司於2008年8月公開聲明將於[[非洲|非洲]]採礦<ref>參看{{en}}[http://www.cabot-corp.com/Tantalum/GN200809161037AM6983/?WT.mc_id=%7C Cabot Position on Tantalum and Coltan and the Democratic Republic of Congo]一文。</ref>這份聲明稿概括了該公司的[[鉭|鉭]]礦來源原為[[加拿大|加拿大]]、[[莫桑比克|莫桑比克]]和[[澳大利亞|澳大利亞]]，雖然他們宣布在非洲開採，但將防止自[[剛果共和國|剛果共和國]]、[[剛果民主共和國|剛果民主共和國]]、[[贊比亞|贊比亞]]、[[蒲隆地|蒲隆地]]及[[盧旺達|盧旺達]]等國家非法開採鉭礦。

=== 無衝突精煉廠審查委員會 ===
由全球30多個大型電子、電氣產品製造商所組成的電子行業公民聯盟（Electronic Industry Citizenship Coalition，縮寫成EICC）及全球永續議題e化倡議組織（Global e-Sustainability Initiative，縮寫成GeSI）合組的「無衝突精煉廠審查委員會」（Conflict-Free Smelter Review Committee，專門審查各製造商的原料是否來自無衝突國家地區的獨立第三方機構）審查卡博特的鉭原料，結果於2010年12月認定合乎其核准協議；該組織評估了卡博特位於美國[[賓夕法尼亞州|賓夕法尼亞州]]波依爾鎮（Boyertown）與[[日本|日本]]會津的製造地點。

== 卡博特在中國 ==
卡博特公司於2005年進軍[[中國|中國]]市場，於[[上海市|上海市]]設立亞太地區總部，轄下共有四家製造廠：
* 亞太區總部：上海市[[閔行區|閔行區]]雙柏路558號
* 卡博特高性能材料（天津）有限公司：[[天津市|天津市]]天津經濟技術開發區化學工業園瑤山路1號
* 卡博特化工（天津）有限公司：天津市天津經濟技術開發區化學工業園瑤山路1號
* 上海卡博特化工有限公司：上海市閔行區雙柏路558號
* 卡博特藍星化工（江西）有限公司：[[江西省|江西省]][[九江市|九江市]][[永修縣|永修縣]]星火工業園
