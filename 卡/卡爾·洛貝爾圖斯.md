'''約翰·卡爾·洛貝爾圖斯'''（'''Johann Karl Rodbertus'''，1805年8月12日—1875年12月6日），也被稱呼為'''卡爾·洛貝爾圖斯—亞格措夫'''，他是[[德國|德國]][[經濟學家|經濟學家]]和[[社會主義|社會主義者]]，出生於德國[[格賴夫斯瓦爾德|格賴夫斯瓦爾德]]。他推崇[[勞動價值理論|勞動價值理論]]<ref>G.D.H. Cole 指出洛貝爾圖斯提倡此理論 “不是以馬克思主義的形式，而是以一些早期經濟學家，例如[[威廉·威廉·湯普遜|威廉·威廉·湯普遜]]和 [[約翰·法蘭西斯·布雷|約翰·法蘭西斯·布雷]]所預見的情況，他得到普魯東的贊同”參見 G.D.H. Cole's ''History of Socialist Thought: Volume II Part II''. London: Macmillan, 1960. p.21</ref>，並從中得出[[利息|利息]]和[[利潤|利潤]]的分配形同對[[工人|勞動階級]]的[[偷竊|偷竊]]等結論。

洛貝爾圖斯被後人稱爲“洛貝爾圖斯—亞格措夫”，得名於他在1834年在[[波美拉尼亞|波美拉尼亞]]地區[[亞格措夫|亞格措夫]]購買的一個[[莊園|莊園]]，從那以后他成爲[[地主|地主]]階級，[[政治|政治]]思想也漸漸[[右翼|右傾]]。洛貝爾圖斯家族來自接近[[丹麥|丹麥]]的[[石勒蘇益格|石勒蘇益格]][[公國|公國]]，遠祖[[彼得魯斯·洛貝爾圖斯|彼得魯斯·洛貝爾圖斯]]是一位在[[德國|德國]]北部[[博倫|博倫]]公國供職的[[牧師|牧師]]。他父親[[約翰·克里斯多弗·洛貝爾圖斯|約翰·克里斯多弗·洛貝爾圖斯]]是一位[[羅馬法|羅馬法]][[法學|法學]]教授及司法官，外祖父是[[重農主義|重農主義]]經濟學家[[約翰·奧古斯特·施勒特魏因|約翰·奧古斯特·施勒特魏因]]。洛貝爾圖斯3歲的時候他父親便退職，並遷居至[[梅克倫堡|梅克倫堡]][[別捷利茨|別捷利茨]][[莊園|莊園]]。洛貝爾圖斯在梅克倫堡完成[[中學|中學]]課程，以後分別在1823年至1825年和1825年至1826年，先後在[[哥廷根|哥廷根]]和[[柏林|柏林]]學習[[法學|法學]]。1830年之後，他到了[[海德堡|海德堡]]攻讀[[哲學|哲學]]，他更在這段時間廣泛遊歷了[[荷蘭|荷蘭]]、[[法國|法國]]和[[瑞士|瑞士]]。直至1836年，他定居於兩年前購置的亞格措夫莊園<ref>G.D.H. Cole's ''History of Socialist Thought: Volume II Part II''. London: Macmillan, 1960. p.20</ref>，成爲[[容克|容克]]階級的一員。

== 仕途 ==
洛貝爾圖斯定居亞格措夫以後，在政界非常活躍，初期甚至可說是步步高陞，不過他的政治生涯在[[1848年革命|1848年革命]]之後變得黯淡。

*1841年當選為州議員。
*1847年起擔任省自治會[[騎士|騎士]]階層代表。稍后又任[[普魯士|普魯士]]政府的第二屆地方自治聯合會委員。
*1848年參加[[魯道夫·馮·奧爾斯瓦爾德|魯道夫·馮·奧爾斯瓦爾德]]内阁，擔任文教大臣僅14天。[[大衛·漢澤曼|大衛·漢澤曼]]任財政大臣，實際上是内閣的掌權人。

[[1848年革命|1848年革命]]后，洛貝爾圖斯入選[[普魯士國民議會|普魯士國民議會]]，屬於中間偏左派系。曾經擔任“波羅的海地區工人階級福利同盟”主席。<ref>{{cite book |author=卡·洛貝爾圖斯|title=《關於德國國家經濟狀況的認識——五大原理》|pages=第5頁}}</ref>

1848年之後，洛貝爾圖斯淡出政壇，專注於[[社會主義|社會主義]]理論研究。他繼續從事[[經濟學|經濟學]]論著的寫作，他的思想深深地影響了[[阿道夫·瓦格納|阿道夫·瓦格納]]和[[俄國|俄國]]的[[米哈伊爾·杜岡—巴拉諾夫斯基|米哈伊爾·杜岡—巴拉諾夫斯基]]<ref>{{cite book |author=卡·洛貝爾圖斯|title=《關於德國國家經濟狀況的認識——五大原理》|pages=第50頁}}</ref>。前者是[[俾斯麥|俾斯麥]]就任[[普魯士|普魯士]]和[[德意志帝國|德意志帝國]]首相時期最有影響力的[[經濟學家|經濟學家]]之一，後者短暫擔任[[烏克蘭人民共和國|烏克蘭人民共和國]]的財政部長，可見洛貝爾圖斯的經濟政策對[[政府|政府]]政策也產生影響。

== 洛氏的經濟學研究 ==
洛貝爾圖斯為[[勞動價值理論|勞動價值理論]]立下三個命題：

*一、只有[[勞動|勞動]]創造的物品才是[[經濟物品|經濟物品]]。其他物品（如[[陽光|陽光]]）只要不是由勞動創造，它們就該是自然物品，[[經濟學|經濟學]]的目標不應該研究這些自然物品。
*二、經濟物品主要是勞動的產品。例如[[穀物|穀物]]的價值在洛貝爾圖斯看來，沒有一部分是得力於[[土壤|土壤]]和陽光的貢獻。
*三、經濟物品的成分由勞動產品所構成，以及勞動能創造工具輔助[[生產|生產]]。例如[[穀物|穀物]]的[[價值|價值]]不獨是犁工的功勞，還應該計算[[犁|犁具]]制造者的[[勞動|勞動]]在内。

洛貝爾圖斯打破[[德國|德國]][[德國歷史學派（經濟學）|歷史學派]]對[[經濟學|經濟學]]領域的壟斷，他首先在[[德國|德國]]公開倡議[[李嘉圖|李嘉圖]]的[[勞動價值理論|勞動價值理論]]<ref>{{cite book |author=卡·洛貝爾圖斯|title=《關於德國國家經濟狀況的認識——五大原理》|pages=第35頁}}</ref>。他強調價值的意義只有使用價值<ref>{{cite book |author=卡·洛貝爾圖斯|title=《關於德國國家經濟狀況的認識——五大原理》|pages=第33頁}}</ref>。

[[奧地利|奧地利]]經濟學家[[歐根·博姆-巴維克|歐根·博姆-巴維克]]評價洛貝爾圖斯陳述的[[剝削理論|剝削理論]]，他認爲不論從連貫性或是深度來説要優於[[馬克思|馬克思]]。不過[[歐根·博姆-巴維克|歐根·博姆-巴維克]]本人並不相信任何[[社會主義|社會主義者]]的經濟理論，他主張資本家是有權得到作爲使用[[資本|資本]]的報酬——[[利息|利息]]，所以稱之爲“剝削”是荒唐的。

洛貝爾圖斯雖然經歷過[[1848年革命|1848年革命]]，但他並不認爲[[工人|工人]]展開[[階級鬥爭|階級鬥爭]]是正確的。他支持提高工人的[[工資|工資]]，不過他反對犧牲[[地租|地租]]和[[利息|利息]]的收入而轉移到工資去，他相信工人提高勞動生產率便能增加工資<ref>{{cite book |author=卡·洛貝爾圖斯|title=《關於德國國家經濟狀況的認識——五大原理》|pages=第24頁}}</ref>。他直言：“我們這門科學（[[政治經濟學|政治經濟學]]）最逼切的任務，只是謀求[[勞動|勞動]]同[[土地|土地]]和[[資本|資本]]所有權之間的妥協。<ref>{{cite book |author=卡·洛貝爾圖斯|title=《關於德國國家經濟狀況的認識——五大原理》|pages=第25頁}}</ref>”洛貝爾圖斯處理利息和地租的態度是充滿矛盾的，他承認勞動創造了“租金”（包括[[利息|利息]]和[[地租|地租]]），可是他又堅稱[[資本家|資本家]]和[[地主|地主]]都履行[[社會|社會]]功能，有權分享勞動成果。他強調階級保持和諧：“應當以一切可能的手段，保持勞動、資本和土地所有權這樣一種社會經濟階級的劃分情況，一切努力都應當只勇於改變勞動產品的分配”<ref>{{cite book |author=卡·洛貝爾圖斯|title=《關於德國國家經濟狀況的認識——五大原理》|pages=第37頁}}</ref>

[[恩格斯|恩格斯]]認爲洛貝爾圖斯“接近於發現[[剩餘價值|剩餘價值]]”，但基於他的[[容克|容克]][[地主|地主]]階級身份，造成他不能否定地租和利息等收入的合理性<ref>{{cite book |author=卡·洛貝爾圖斯|title=《關於德國國家經濟狀況的認識——五大原理》|pages=第3頁}}</ref>。

== 洛氏的政治立場 ==

洛貝爾圖斯的[[容克|容克]][[地主|地主]]階級立場決定了他的政治取態，他的[[左派|左派]]身份相當曖昧，主要是因爲得自他重視[[勞動價值理論|勞動價值理論]]的[[經濟學|經濟學]]觀點，但他有別於同時代的[[社會主義|社會主義者]]：他反對採用[[罷工|罷工]]甚至[[暴動|暴動]]等手段，讓工人階級爭取權益。他認同[[普魯士|普魯士]][[君主制|君主制]]，他希望工人階級、資產階級和普魯士君主及其[[貴族|貴族]]通力合作，保存現有的階級制度，為[[泛日耳曼主義|日耳曼民族主義]]服務，建立統一的[[德意志帝國|德意志帝國]]。這可從他在《正常工作日》中的論點看出：

{{Quote|不！社會問題不會在街頭上，靠[[罷工|罷工]]、築[[街壘|街壘]]或[[石油|石油]]……來解決。社會問題有其獨有的性質；它猶如[[含羞草|含羞草]]，一遇到粗野的強有力的手就愕然退縮。確立持久的社會和平，建立統一的國家政權，[[工人|工人]]階級與這個政權保持牢固可靠的聯盟，事先進行大量研究和各項工作，在和平時期，在一系列深入結合的基礎上，雷厲風行、秩序井然地建立各種制度——這些，就是解決社會問題所需的條件……。如果說[[保守主義|保守主義]]就是保持陳腐的舊事物，無論這舊事物叫作[[自由主義|自由主義]]的或非自由主義的，那末，沒有什麽比社會問題再不保守的了。如果說保守主義就是加強[[君主制|君主制]]國家政權，進行和平的改良工作，使社會各階級在光輝的各得其所的原則保護和指導下相安無事，那末，沒有什麽比社會問題再保守的了。<ref>{{cite book |author=卡·洛貝爾圖斯|title=《關於德國國家經濟狀況的認識——五大原理》|pages=第22-23頁}}</ref>}}

洛貝爾圖斯相信可以依靠[[立法|立法]]解決他所謂的“社會問題”，即改變階級的經濟分配制度，而無需要徹底推翻現存制度。他強調在非衝突的條件下幫助工人階級，讓其和[[普魯士|普魯士]]政府保持團結，是[[國家社會主義|國家社會主義]]的起源。

像[[馬克思|馬克思]]這些極左的社會主義者看來，洛貝爾圖斯把[[工人|工人]]階級的鬥爭看成是一般的社會問題，簡直是不可理喻。1885年[[恩格斯|恩格斯]]跟[[考茨基|考茨基]]的通信稱：“一切資產階級分子現在都聚集在洛貝爾圖斯的周圍”<ref>{{cite book |author=卡·洛貝爾圖斯|title=《關於德國國家經濟狀況的認識——五大原理》|pages=第27頁}}</ref>，其時洛貝爾圖斯已逝，恩格斯的説話意即洛貝爾圖斯的經濟理論和政治主張給了[[德國|德國]]的資產階級指導。

== 洛氏和其他社會主義者的爭執 ==
洛貝爾圖斯沒有提出任何有關[[工資|工資]]的理論，但他卻常常談論[[工資鐵律|工資鐵律]]，甚至跟[[德國|德國]]工人運動活動家[[斐迪南·拉薩爾|斐迪南·拉薩爾]]爭論誰先發現工資鐵律<ref>{{cite book |author=卡·洛貝爾圖斯|title=《關於德國國家經濟狀況的認識——五大原理》|pages=第39頁}}</ref>。事實上，工資鐵律與他們兩人毫不相干，最先提出的是[[英國|英國]][[經濟學家|經濟學家]][[李嘉圖|李嘉圖]]和[[馬爾薩斯|馬爾薩斯]]。

他私底下跟友人通信時稱他的理論“被[[馬克思|馬克思]]在其《[[資本論|資本論]]》中滿不壞地予以利用……，當然，並未説明是摘引我的話。<ref name="s11">{{cite book |author=卡·洛貝爾圖斯|title=《關於德國國家經濟狀況的認識——五大原理》|pages=第11頁}}</ref>”他認爲馬克思把他“洗劫一空”<ref name="s11"/>。他自認早於馬克思發現“[[剩餘價值|剩餘價值]]”。
