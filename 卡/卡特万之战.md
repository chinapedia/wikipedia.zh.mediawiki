'''卡特万之战'''是[[西辽|西辽]]和[[葛逻禄|葛逻禄]]联军与[[塞尔柱帝国|塞尔柱帝国]]军队于1141年9月9日在[[撒马尔罕|撒马尔罕]]以北的[[卡特万草原|卡特万草原]]发生的战争。塞尔柱帝国联军惨败，势力退出[[河中地区|河中地区]]，西辽成为[[中亚|中亚]]霸主。卡特万之战也是继751年[[怛罗斯战役|怛罗斯战役]]后，历史上第二次[[东亚|东亚]]文明与[[伊斯兰|伊斯兰]]文明的对抗。<ref>{{Cite book|author=肖爱民|title=《从怛逻斯到卡特万—试述两次会战及其对中西文化交流的影响》|location=赤峰市|publisher=赤峰学院学报(汉文哲学社会科学版)1997年第一期|date=1997年|pages=|ISBN=|accessdate=|url=|language=|quote=}}</ref>

== 背景 ==
西辽自建都于[[虎思斡耳朵|虎思斡耳朵]]（今[[吉尔吉斯斯坦|吉尔吉斯斯坦]][[托克玛克|托克玛克]]东南[[布拉纳|布拉纳]]）后，国力蒸蒸日上。1134年，[[耶律大石|耶律大石]]征讨[[金朝|金朝]]受挫，转而继续在中亚地区扩张。<ref name="研究"/>1137年，耶律大石率军向[[塔什干|察赤]]（今[[乌兹别克斯坦|乌兹别克斯坦]][[塔什干|塔什干]]）、[[费尔干纳盆地|费尔干纳盆地]]及[[泽拉夫尚河|泽拉夫尚河]]流域进兵，同年5至6月，在[[忽毡|忽毡]]（今[[塔吉克斯坦|塔吉克斯坦]][[苦盏|苦盏]]）遭到了西喀喇汗国可汗[[马黑木二世|马赫穆德·伊本·穆海默德]]的抵抗。西喀喇汗国战败，马赫穆德败逃回[[撒马尔罕|撒马尔罕]]。这次战败使马黑木二世的臣民感到震惊、惊恐和沮丧，但耶律大石并没有继续进兵。<ref name="全史"/>而马黑木二世将忽毡战役失败的责任推卸给[[葛逻禄|葛逻禄]]人，<ref>{{Cite book|author=哈萨克苏维埃社会主义共和国科学院历史所|title=《哈萨克苏维埃社会主义共和国史》|location=阿拉木图|publisher=哈萨克苏维埃社会主义共和国科学院|date=1979年|pages=第二卷·第37页|ISBN=|accessdate=|url=|language=ru|quote=}}</ref>他强迫葛逻禄人离开撒马尔罕边境，并向舅父塞尔柱帝国[[苏丹_(称谓)|苏丹]][[阿哈马德·桑贾尔|阿哈马德·桑贾尔]]求援，但桑贾尔正忙于和[[花剌子模|花剌子模]]的战争，并未出兵帮助。<ref>{{Cite book|author=加富罗夫|title=《塔吉克远古、古代和中世纪史》|location=莫斯科|publisher=苏联科学院|date=1972年|pages=第404页|ISBN=|accessdate=|url=|language=ru|quote=}}</ref>

由于西辽在[[伊斯兰世界|伊斯兰世界]]的崛起以及[[东喀喇汗国|东喀喇汗国]]臣服于西辽，这引起了塞尔柱帝国苏丹桑贾尔的不满。1141年，西喀喇汗国与葛逻禄人爆发冲突，马赫穆德再次向宗主国塞尔柱帝国求援。桑贾尔动员伊斯兰诸国参战，集中了[[呼罗珊|呼罗珊]]、[[錫斯坦－俾路支斯坦省|锡斯坦]]、[[伽色尼王朝|伽色尼]]、[[马赞德兰|马赞德兰]]、[[古尔王朝|古尔]]等国的军队近10万人，单单阅兵就耗费了半年时间。同年7月，桑贾尔率军渡过[[阿姆河|阿姆河]]，进入河中地区。面对强大的敌军，葛逻禄人急忙派使者向耶律大石求救。<ref name="全史">{{Cite book|author=苏联科学院|title=《吉尔吉斯人和吉尔吉斯地区历史资料》|location=莫斯科|publisher=苏联科学院|date=1973年|pages=第一辑·第64页至第67页|ISBN=|accessdate=|url=|language=ru|quote=}}</ref>

耶律大石写信给桑贾尔替葛逻禄人说情，但桑贾尔十分傲慢的回信命令耶律大石加入[[伊斯兰教|伊斯兰教]]，并称自己的军队能用箭截断敌人的须发。当耶律大石听完桑贾尔的使者读完书信后，下令拔下他的一撮胡须，然后给他一根针让他当场示范，使者不能做到。耶律大石说既然针不能截断胡须，那那个人又怎么能用箭折断须发呢？于是下令进兵，双方在撒马尔罕以北的卡特万草原对峙，西辽的军队中有[[契丹人|契丹人]]、[[突厥人|突厥人]]、[[汉人|汉人]]和[[蒙古人|蒙古人]]。<ref name="全史"/>耶律大石观察了战场的地形后，让军队背靠[[达尔加姆峡谷|达尔加姆峡谷]]安营。两军于1141年9月9日展开会战，战前耶律大石指出桑贾尔的联军人多少谋，如果全力进攻他们就会首尾不顾。耶律大石派[[南院大王|六院司大王]][[萧斡里剌|萧斡里剌]]、[[招讨副使|招讨副使]][[耶律松山|耶律松山]]等率兵2500攻打联军右翼，[[枢密副使|枢密副使]][[萧剌阿不|萧剌阿不]]、[[招讨使|招讨使]][[耶律术薛|耶律术薛]]等率兵2500攻打其左翼，耶律大石亲率部队攻打中军；<ref name="卡特万">《辽史·卷三十·天祚皇帝四》：（耶律大石）至寻思干，西域诸国举兵十万，号忽儿珊，来拒战。两军相望二里许。谕将士曰：“彼军虽多而无谋，攻之，则首尾不救，我师必胜。”遣六院司大王萧斡里剌、招讨副使耶律松山等将兵二千五百攻其右；枢密副使萧剌阿不、招讨使耶律术薛等将兵二千五百攻其左；自以众攻其中。三军俱进，忽儿珊大败，僵尸数十里。驻军寻思干凡九十日…又西至起儿漫。</ref>桑贾尔的联军右翼是[[埃米尔|埃米尔]][[库马吉|库马吉]]，左翼是锡斯坦国王，他自己亲率中军，有战斗经验的老兵负责殿后。<ref name="研究">{{Cite book|author=魏良弢|title=《西辽史研究》|location=银川市解放西街105号|publisher=宁夏人民出版社|date=1987年11月|pages=第76页至82页|ISBN=7227000443|accessdate=|url=|language=|quote=}}</ref>

== 过程 ==
在战场上，锡斯坦贵族作战英勇，但西辽军队中的葛逻禄人发挥了重要的作用，迫使桑贾尔的联军败逃。桑贾尔和马赫穆德逃奔至[[泰尔梅兹|泰尔梅兹]]，桑贾尔的妻子、左、右翼统帅和伊斯兰[[法学家|法学家]][[胡萨姆·奥玛尔·伊本·阿布杜·阿齐兹·伊本·马扎·布哈里|胡萨姆·奥玛尔·伊本·阿布杜·阿齐兹·伊本·马扎·布哈里]]均被俘虏。一种观点认为桑贾尔的联军中出现了起义，直接影响到了战争的结果。<ref>{{Cite book|author=乌兹别克苏维埃社会主义共和国科学院历史所|title=《乌兹别克苏维埃社会主义共和国史》|location=塔什干|publisher=乌兹别克苏维埃社会主义共和国|date=1955年|pages=第一卷·第273页|ISBN=|accessdate=|url=|language=ru|quote=}}</ref>桑贾尔的联军损失惨重，仅达尔加姆峡谷就装下1万名死者。<ref name="巴托尔德"/>《[[辽史|辽史]]》记载塞尔柱帝国联军的阵亡者横尸数十里。<ref name="卡特万"/>卡特万之战后，塞尔柱帝国的势力退出河中地区，西辽成为中亚霸主。耶律大石随后率军进入撒马尔罕，立马赫穆德之弟[[伊卜拉欣三世|伊卜拉欣·伊本·穆海默德]]为桃花石汗，继续让其统治西喀喇汗国。<ref name="研究"/>耶律大石还派大将[[额儿布思|额儿布思]]出兵花剌子模，在该地烧杀抢掠，迫使花剌子模[[沙阿|沙阿]][[阿拉丁·阿比兹|阿拉丁·阿比兹]]向西辽臣服并且每年缴纳价值3万金[[第纳尔|第纳尔]]的货物和牲畜。<ref>{{Cite book|author=志费尼|coauthors=何高济（译）|title=《世界征服者史》|location=呼和浩特市新城西街82号|publisher=内蒙古人民出版社|date=1980年5月|pages=上册·第418页|ISBN=|accessdate=|url=|language=|quote=}}</ref>耶律大石在撒马尔罕驻扎90天后，至[[起儿漫|起儿漫]]（今[[乌兹别克斯坦|乌兹别克斯坦]][[卡尼梅赫镇|卡尼梅赫镇]]）巡行后班师返回虎思斡耳朵。卡特萬之戰西遼大獲全勝。<ref name="卡特万"/>

== 数字 ==
关于卡特万之战双方参战的人数，伊斯兰史料方面所提供的双方人数比例有1:3（10万对30万）或1:10（7万对70万），强调西辽军队数量处于优势，[[伊本·艾西尔|伊本·艾西尔]]记载西辽联军中包括3万至5万名葛逻禄骑兵，部分伊斯兰史料记载双方兵力对等。《辽史·卷三十·天祚皇帝四》记载西辽左右翼各2500人，推测中军数量为1万至2万人。<ref name="Biran">{{Cite book|author=Michal Biran|title=《The Empire of the Qara Khitai in Eurasian History: Between China and the Islamic World》|location=New York|publisher=Cambridge University Press|date=2005年|pages=第43页至第44页|ISBN=9780521842266|accessdate=|url=|language=|quote=}}</ref>按《辽史》中耶律大石的派兵数量和战前训谕，西辽军队的数量应远少于桑贾尔的10万联军。<ref name="研究"/>

关于塞尔柱帝国联军的阵亡人数，伊本·艾西尔的《[[全史|全史]]》中记载为10万人被杀，其中有12000名戴头巾者（伊斯兰教法学家）和4000名妇女，<ref name="全史"/>[[巴托尔德|巴托尔德]]认为塞尔柱帝国联军阵亡3万人。<ref name="巴托尔德">{{Cite book|author=巴托尔德|title=《巴托尔德文集》|location=莫斯科|publisher=苏联科学院|date=1963年|pages=第一卷·第389页|ISBN=|accessdate=|url=|language=ru|quote=}}</ref>一般认为阵亡人数在11000至10万之间。<ref name="Biran"/>

== 影响 ==
耶律大石的西征事迹被传到[[欧洲|欧洲]]，正逢[[第二次十字军东征|第二次十字军东征]]，于是在欧洲流传着东方世界有一位神秘的[[祭司王约翰|祭司王约翰]]，是[[基督教|基督教]]的捍卫者。<ref>{{Cite book|author=巴托尔德|coauthors=耿世民（译）|title=《中亚简史》|location=乌鲁木齐市解放南路348号|publisher=新疆人民出版社|date=1980年|pages=第33页|ISBN=|accessdate=|url=|language=|quote=}}</ref>[[俄语|俄语]]、[[阿拉伯语|阿拉伯语]]、[[拉丁语|拉丁语]]和[[古英语|古英语]]中[[中国|中国]]的发音类似于“契丹”，都是受耶律大石西征的影响。<ref name="史论">{{Cite book|author=吉宗安|title=《西辽史论·耶律大石研究》|location=乌鲁木齐市解放南路348号|publisher=新疆人民出版社|date=1996年8月|pages=第141页|ISBN=7228039904|accessdate=|url=|language=|quote=}}</ref>而耶律大石的名字也成了西辽帝国的代称，在耶律大石死后，金、[[西夏|西夏]]、[[南宋|南宋]]等国家对西辽的后代君主皆称为“大石”。<ref>《金史·卷一百二十一·粘割韩奴列传》：此时大石林牙已死，子孙相继，西方诸部仍以大石呼之。</ref>

== 评价 ==
卡特万之战是对中亚历史有重要影响的一场战争，它使西辽帝国和塞尔柱帝国对中亚地区的政治影响和相互关系发生了根本性的改变。<ref name="史论"/>伊本·艾西尔评价卡特万之战：在伊斯兰的历史上再没有比这更大的战争，在呼罗珊也没有比这更多的伤亡。<ref name="全史"/>[[拉施特|拉施特]]评价卡特万之战：（耶律大石）在撒马尔罕西北的卡特万草原击溃最后一个“伟大的塞尔柱君主”桑贾尔的军队，征服了花剌子模，并首次迫使中亚伊斯兰教徒接受“[[异教徒|异教徒]]”的统治。<ref>{{Cite book|author=拉施特|coauthors=余大钧（译）|title=《史集》汉译本|location=北京市|publisher=商务印书馆|date=1986年|pages=第一卷·第二册·第273页|ISBN=|accessdate=|url=|language=|quote=}}</ref>[[魏特夫|魏特夫]]和[[冯家昇|冯家昇]]指出：1137年和1141年是[[菊儿汗|菊儿汗]]把胜利引向[[突厥斯坦|突厥斯坦]]西部的两个里程碑，这两次战役积累的胜利果实，使他们建立起“伟大的喀喇契丹帝国”，自此“塞尔柱帝国不可挽救的失去了阿姆河以远的地区”。<ref>{{Cite book|author=魏特夫|coauthors=冯家昇|title=《中国社会史—辽》|location=费城|publisher=费城|date=1944年|pages=第621页|ISBN=|accessdate=|url=|language=en|quote=}}</ref>
