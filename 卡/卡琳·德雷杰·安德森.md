'''卡琳·德雷耶·安德松'''（{{lang-sv|Karin Dreijer Andersson}}，1975年4月7日─）是[[瑞典|瑞典]]電子團體[[刀子_(瑞典團體)|刀子]]的主唱。原是另類搖滾樂團[[Honey_Is_Cool|Honey Is Cool]]的主音和吉他手。2009年，她化名為'''Fever Ray'''發行了個人專輯。她以尖銳而深邃的唱腔，結合有特色的口音的扭曲音調以及對變調的應用著稱。她通常在照片中帶著[[面具|面具]]和其他戲劇元素。

她同時也是位[[女性主義|女性主義]]者，她與弟弟組成的[[刀子_(瑞典團體)|刀子]]曾捐錢給瑞典女性主義政黨[[女性主義行動先鋒|女性主義行動先鋒]]。

== 個人作品 ==


卡琳·德雷耶·安德松化名為'''Fever Ray'''發行過一張專輯與數支單曲，在瑞典、丹麥、比利時反應不俗；並持續以'''Fever Ray'''為名與其他樂手合作。

2009年她為[[女性主義|女性主義]]色情短片選集〈[[骯髒日記|骯髒日記]](Dirty Diaries)〉的原聲帶製作，風格強烈。

== 合作 ==
Andersson为Röyksopp2005年的专辑The Understanding的歌曲“What Else Is There?”献唱。Andersson为dEUS2008年的专辑Vantage Point的歌曲Slow献唱。

Andersson为Röyksopp2009年的专辑Junior中的歌曲“This Must Be It”和“Tricky Tricky”献声。扮演女人的演员Rickard Engfors在[[The_Knife|The Knife]]的“Pass This On”音乐录影带中表演。
