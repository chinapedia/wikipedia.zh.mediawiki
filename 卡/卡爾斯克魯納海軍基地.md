'''卡爾斯克魯納海軍基地'''（{{lang-sv|Karlskrona Örlogsbas}}）位於[[瑞典|瑞典]]南部的[[卡爾斯克魯納市|卡爾斯克魯納市]]，是[[瑞典海軍|瑞典海軍]]的作戰中樞<ref name="海軍基地隊">{{en}}{{lang|en|{{Cite web |title=The Naval Base |url=http://www.forsvarsmakten.se/en/our-organisation/training-units-schools-and-centres/the-naval-base/ |work=[[瑞典國防軍|Swedish Armed Forces]] |accessdate=2014-12-09 |deadurl=yes |archiveurl=https://web.archive.org/web/20141024200603/http://www.forsvarsmakten.se/en/our-organisation/training-units-schools-and-centres/the-naval-base/ |archivedate=2014-10-24 }}}}</ref>兼首要基地<ref name="Guide">{{en}}{{lang|en|{{Cite book |title=Travel Guide: Traces of the Cold War Period ; the Countries Around The Baltic Sea |publisher=Nordic Council of Ministers |date=2010 |author=Johannes Bach Rasmussen, Nordic Council of Ministers |isbn=9289321210}}}}</ref>{{rp|p214}}，同時是[[聯合國教科文組織|聯合國教科文組織]]認定的[[世界遺產|世界文化遺產]]，登錄名稱為'''卡爾斯克魯納軍港'''（{{lang-en|Naval Port of Karlskrona}}）。基地內部共駐有1,750名瑞典海軍官兵、文職僱員，並設有[[潛艇|潛艇]]、水面船艦等各式部隊<ref name="海軍技術網">{{en}}{{lang|en|{{Cite web |work=Naval-technology.com |title=Karlskrona Naval Base, Sweden |url=http://www.naval-technology.com/projects/karlskrona-base/ |accessdate=2014-12-09 |deadurl=no |archiveurl=https://web.archive.org/web/20131421075200/http://www.naval-technology.com/projects/karlskrona-base/ |archivedate=2018-06-11 }}}}</ref>，以及軍港以外的[[造船廠|造船廠]]、防禦工事等周邊設施。{{Link-en|瑞典海岸防衛隊|Coast Guard (Sweden)}}的總部也設在此處<ref name="Coast Guard">{{en}}{{lang|en|{{Cite web |work={{Link-en|瑞典海岸防衛隊|Coast Guard (Sweden)|Swedish Coast Guard}} |title=Organisation |url=http://www.kustbevakningen.se/sv/the-swedish-coast-guard/about-us/organisation/ |accessdate=2014-12-13 |deadurl=yes |archiveurl=https://web.archive.org/web/20141213171501/http://www.kustbevakningen.se/sv/the-swedish-coast-guard/about-us/organisation/ |archivedate=2014-12-13 }}}}</ref>。

卡爾斯克魯納海軍基地於[[瑞典帝國|瑞典帝國]]全盛期的17世紀即已存在，1680年竣工後，作為該國在[[波羅的海|波羅的海]]的重要據點，港口周邊也連帶建起了海軍城鎮。18世紀後，該基地甚至於軍事及建築科技上居於[[歐洲|歐洲]]之首<ref name="Initiative">{{en}}{{lang|en|{{Cite web |work=The Baltic Initiative and Network |title=The Naval Museum, Karlskrona |url=http://coldwarsites.net/country/sweden/the-naval-museum-karlskrona/ |accessdate=2014-12-09 |deadurl=no |archiveurl=https://web.archive.org/web/20141213180226/http://coldwarsites.net/country/sweden/the-naval-museum-karlskrona/ |archivedate=2014-12-13 }}}}</ref>，且當時興建的軍港設施、布局和設計在數百年後仍得到完善保存。1998年，教科文組織下的[[世界遺產委員會|世界遺產委員會]]正式將該港列為文化類世界遺產<ref name="UNESCO">{{en}}{{lang|en|{{Cite web |work=[[聯合國教科文組織|United Nations Educational, Scientific and Cultural Organization]] |title=Naval Port of Karlskrona |url=http://whc.unesco.org/en/list/871 |accessdate=2014-12-09 |deadurl=no |archiveurl=https://web.archive.org/web/20141119031423/http://whc.unesco.org/en/list/871 |archivedate=2014-11-19 }}}}</ref>。

卡爾斯克魯納海軍基地所處地點為瑞典[[布萊金厄省|布萊金厄省]]境內的東部地帶，西邊為省內的另一大城[[龍訥比市|龍訥比市]]，該港位於首都[[斯德哥爾摩|斯德哥爾摩]]南方約382[[公里|公里]]，但離[[丹麥|丹麥]][[哥本哈根|哥本哈根]]反而較近，兩者之間的距離為198[[公里|公里]]。海軍營區緊鄰著卡爾斯克魯納市中心所在的{{Link-en|特羅索島|Trossö}}南側，港口被該島及林侯門島（{{lang|sv|Lindholmen}}）、瑟德雅納島（{{lang|sv|Söderstjärna}}）三方包圍，開口朝向東方。基地內的軍用補勤設施和海軍人員曾大量部署於東側的{{Link-en|斯圖火門島|Stumholmen}}上，但在[[冷戰|冷戰]]後期已全數撤出<ref name="Trossö">{{en}}{{lang|en|{{Cite web |title=Trossö |url=http://www.orlogsstadenkarlskrona.se/page/86/the-naval-dockyard-and-harbour.aspx |work=[[卡爾斯克魯納市|Karlskrona Kommun]] |accessdate=2014-12-11 |deadurl=no |archiveurl=https://web.archive.org/web/20141213054945/http://www.orlogsstadenkarlskrona.se/page/86/the-naval-dockyard-and-harbour.aspx |archivedate=2014-12-13 }}}}</ref><ref name="Stumholmen">{{en}}{{lang|en|{{Cite web |title=Stumholmen island |url=http://www.orlogsstadenkarlskrona.se/page/88/the-environs.aspx |work=[[卡爾斯克魯納市|Karlskrona Kommun]] |accessdate=2014-12-11 |deadurl=no |archiveurl=https://web.archive.org/web/20141213051738/http://www.orlogsstadenkarlskrona.se/page/88/the-environs.aspx |archivedate=2014-12-13 }}}}</ref>。

基地周邊的水域由{{Link-en|布萊金厄群島|Blekinge archipelago}}中的其他數個島嶼所圍繞，自東到西分別為{{Link-sv|森諾倫島|Senoren}}、{{Link-sv|圖尤克島|Tjurkö}}、[[阿斯普島|阿斯普島]]及[[哈斯爾島|哈斯爾島]]。這些島在基地外部排列成了一道天然屏障，還設有{{Link-sv|德羅特寧謝爾要塞|Drottningskärs kastell|德羅特寧謝爾}}、{{Link-sv|孔斯霍姆碉堡|Kungsholms fort|孔斯霍姆}}兩座守港要塞<ref name="Initiative"></ref>。由內港通往波羅的海的航道狹長而深，易於防禦<ref name="pdf">{{en}}{{lang|en|{{Citation |title=The naval town of Karlskrona |url=https://karlskrona.se/Global/Karlskrona%20kommun/Dokument/Om%20kommunen/Broschyrer/Kulturf%C3%B6rvaltningen/Folder_eng%20%282%29.pdf |format=pdf |work=[[卡爾斯克魯納市|Karlskrona Kommun]] |accessdate=2014-12-10 |deadurl=yes |archiveurl=https://web.archive.org/web/20160304223328/https://karlskrona.se/Global/Karlskrona%20kommun/Dokument/Om%20kommunen/Broschyrer/Kulturf%C3%B6rvaltningen/Folder_eng%20(2).pdf |archivedate=2016年3月4日 |df= }}}}</ref>、又足以讓深吃水的軍用艦艇進出，港內的海水也不易結冰，這些條件使卡爾斯克魯納更加適合作為海軍基地<ref name="海軍技術網"></ref>。

卡爾斯克魯納基地是瑞典海軍參謀部所在點<ref name="海軍技術網"></ref>，而瑞典海軍除了這座主要設施外，尚於[[哥特堡|哥特堡]]、首都圈外的{{Link-en|穆斯克島|Muskö}}等地設有其他的分遣基地<ref name="海軍基地隊"></ref>。卡爾斯克魯納不僅扮演著水面艦艇、作戰潛艦部署地和造船、補給基地的角色，基地內也有新訓設施，替為數1,200名的服役士兵和軍方潛水隊員提供基礎訓練場所。除了海軍部隊外，布萊金厄省、[[卡爾馬市|卡爾馬市]]和[[克魯努貝里省|克魯努貝里省]]三地所駐的[[瑞典鄉土防衛隊|鄉土防衛隊]]（後備軍）也由本基地訓練<ref name="海軍技術網"></ref><ref name="海軍基地隊"></ref>。

==歷史==
===帝國時代至18世紀末===

17世紀末的瑞典帝國為歐洲強權，透過海軍實力使其疆土擴至波羅的海周邊區域，涵蓋了今日的[[芬蘭|芬蘭]]、[[拉脫維亞|拉脫維亞]]、[[愛沙尼亞|愛沙尼亞]]及[[德國|德國]]北部<ref name="pdf"></ref>。1679年[[斯科訥戰爭|斯科訥戰爭]]結束後，儘管在戰中交鋒的瑞典與丹麥簽訂了和平條約，雙方之間仍存有對立，使[[瑞典國王|瑞典國王]][[卡爾十一世|卡爾十一世]]打算建造一座接近丹麥的海軍基地<ref name="海軍技術網"></ref>，經過一番審視和評議之後，決定在瑞典南部的布萊金厄尋找合適的建港地點，因為當地與原本做為海軍基地的斯德哥爾摩相比，冬季時被海冰凍結的時日較少<ref name="pdf"></ref>。

同年秋季下旬，卡爾十一世與他的隨從們踏上了特羅索島，將當地敲定為軍港新址，島上唯一的農家奉命遷出，而瑞典國內頂尖的造船師、建築師、要塞工程師則隨即遷入該島，於1680年展開工事<ref name="pdf"></ref>。新的軍港被命名為卡爾斯克魯納，意為「卡爾的皇冠」，工程則在軍需總監{{Link-en|埃里克·達赫伯格|Erik Dahlbergh}}的指導和監督下進行。最早開始施作的基地建築為造船廠，廠中建有兩座造船台、兩座鍛鐵爐和五座倉庫<ref name="UNESCO"></ref>，以及用來打造、維修戰艦的傾船修理碼頭（{{lang|sv|Kölhalningsbroarna}}）<ref name="Careening">{{en}}{{lang|en|{{Cite web |title=20. Kölhalningsbroarna (The Careening Wharf) |url=http://www.orlogsstadenkarlskrona.se/page/117/kolhalningsbroarna.aspx |work=[[卡爾斯克魯納市|Karlskrona Kommun]] |accessdate=2014-12-10 |deadurl=no |archiveurl=https://web.archive.org/web/20141213055736/http://www.orlogsstadenkarlskrona.se/page/117/kolhalningsbroarna.aspx |archivedate=2014-12-13 }}}}</ref>。第一座[[龍骨|龍骨]]於該年12月正式在廠內安放，首艘戰艦則於翌年下水<ref name="UNESCO"></ref>。除了海軍基地之外，瑞典還特地在特羅索島上規劃出與軍港並存的[[新市鎮|新市鎮]]，提供住房、商業活動及行政機能<ref name="pdf"></ref>。當該城鎮在1683年升格為[[省會|省會]]所在地時，達赫伯格總監又為其制定了市街和要塞計劃案<ref name="UNESCO"></ref>。

[[File:Karlskrona_marinbas_-_Mönster_och_modellsalsbyggnaden.JPG|thumb]]
啟用後的卡爾斯克魯納基地逐漸取代了斯德哥爾摩、成為新的海軍主力基地<ref name="海軍技術網"></ref>。進入18世紀時，該基地已被開發為遠近馳名的歐洲頂尖軍事科技、造船及高級建築技術中心，而基地內的造船廠更是瑞典全國最大的就業集散地<ref name="Initiative"></ref>。在此世紀內，卡爾斯克魯納軍港增設了更多設施，包括五座位於基地西側、排列成扇形的「五指船塢」（{{lang|se|Femfingerdockan}}）<ref name="五指">{{en}}{{lang|en|{{Cite web |title=23. Femfingerdockan (The “Five finger” docks) |url=http://www.orlogsstadenkarlskrona.se/page/121/femfingerdockan.aspx |work=[[卡爾斯克魯納市|Karlskrona Kommun]] |accessdate=2014-12-12 |deadurl=no |archiveurl=https://web.archive.org/web/20141213061126/http://www.orlogsstadenkarlskrona.se/page/121/femfingerdockan.aspx |archivedate=2014-12-13 }}}}</ref>，以及位於港口西南岸、林侯門島上的波勒姆乾塢（{{lang|sv|Polhemsdockan}}），這些製船設施均持續使用到21世紀<ref name="波勒姆">{{en}}{{lang|en|{{Cite web |title=26. Polhemsdockan (The Polhem Dry Dock) |url=http://www.orlogsstadenkarlskrona.se/page/125/polhemsdockan.aspx |work=[[卡爾斯克魯納市|Karlskrona Kommun]] |accessdate=2014-12-12 |deadurl=no |archiveurl=https://web.archive.org/web/20141213050847/http://www.orlogsstadenkarlskrona.se/page/125/polhemsdockan.aspx |archivedate=2014-12-13 }}}}</ref>。

1780年代中，身為瑞典知名造船師的海軍少將{{Link-en|腓特烈·查普曼|Fredrik Henrik af Chapman}}在卡爾斯克魯納基地擔任船廠廠長，他任內負責主導基地的造船業務，還著手對整個廠區實施了全面改造。卡爾斯克魯納基地在這樣的措施下，達成了三年內產出20艘船艦（戰艦10艘、巡航艦10艘）的成效<ref>{{en}}{{lang|en|{{Cite web |title=Fredrik Henrik af Chapman |url=http://www.sjohistoriska.se/en/Collections/Archives/ChapmanNet/Fredrik-Henrik-af-Chapman/ |work={{Link-sv|瑞典航海歷史博物館|Sjöhistoriska museet|Sjöhistoriska}} |accessdate=2014-12-12 |deadurl=no |archiveurl=https://web.archive.org/web/20141214065713/http://www.sjohistoriska.se/en/Collections/Archives/ChapmanNet/Fredrik-Henrik-af-Chapman/ |archivedate=2014-12-14 }}}}</ref>。此期間新建的基地地標包括了用於制式化造船流程的裝備棚（{{lang|sv|Materialförråd}}）<ref>{{en}}{{lang|en|{{Cite web |title=16. Materialförråd I och III (Furnishing Sheds Nos. I and III) |url=http://www.orlogsstadenkarlskrona.se/page/115/chapmanbostallet.aspx |work=[[卡爾斯克魯納市|Karlskrona Kommun]] |accessdate=2014-12-12 |deadurl=no |archiveurl=https://web.archive.org/web/20141213044952/http://www.orlogsstadenkarlskrona.se/page/115/chapmanbostallet.aspx |archivedate=2014-12-13 }}}}</ref>、廠長官邸（後為基地司令官邸）<ref>{{en}}{{lang|en|{{Cite web |title=18. Chapmanbostället (Admiral Chapman´s Residence) |url=http://www.orlogsstadenkarlskrona.se/page/115/chapmanbostallet.aspx |work=[[卡爾斯克魯納市|Karlskrona Kommun]] |accessdate=2014-12-12 |deadurl=no |archiveurl=https://web.archive.org/web/20141213044952/http://www.orlogsstadenkarlskrona.se/page/115/chapmanbostallet.aspx |archivedate=2014-12-13 }}}}</ref>、以查普曼命名的大門<ref>{{en}}{{lang|en|{{Cite web |title=17. Chapmanporten (The Chapman Gate) |url=http://www.orlogsstadenkarlskrona.se/page/114/chapmanporten.aspx |work=[[卡爾斯克魯納市|Karlskrona Kommun]] |accessdate=2014-12-12 |deadurl=no |archiveurl=https://web.archive.org/web/20141213041941/http://www.orlogsstadenkarlskrona.se/page/114/chapmanporten.aspx |archivedate=2014-12-13 }}}}</ref>。除此之外，基地的中央還蓋了一座模型設計樓（{{lang|sv|Mönster- och modellsalsbyggnaden}}），樓內收藏軍艦、基地設施的模型，亦作為官兵乘船出航前的集合地點<ref>{{en}}{{lang|en|{{Cite web |title=15. Mönster- och modellsalsbyggnaden (The Ships´ Models Room and Mustering Hall) |url=http://www.orlogsstadenkarlskrona.se/page/112/monster-och-modellsalsbyggnaden.aspx |work=[[卡爾斯克魯納市|Karlskrona Kommun]] |accessdate=2014-12-12 |deadurl=no |archiveurl=https://web.archive.org/web/20141213040316/http://www.orlogsstadenkarlskrona.se/page/112/monster-och-modellsalsbyggnaden.aspx |archivedate=2014-12-13 }}}}</ref>。基地外則於1790年時建造了一組名為「史培爾營」（{{lang|sv|Bastion Sparre}}）的樓群，供海軍[[軍官|軍官]]和[[士官|士官]]作為宿舍，一直到19世紀末才移交給政府機構當公署<ref>{{en}}{{lang|en|{{Cite web |title=12. Bastion Sparre (The Battalion Sparre Buildings) |url=http://www.orlogsstadenkarlskrona.se/page/109/bastionsparre.aspx |work=[[卡爾斯克魯納市|Karlskrona Kommun]] |accessdate=2014-12-12 |deadurl=no |archiveurl=https://web.archive.org/web/20141213041848/http://www.orlogsstadenkarlskrona.se/page/109/bastionsparre.aspx |archivedate=2014-12-13 }}}}</ref>。

進入19世紀後，雖然瑞典的強權地位已經隨著[[古斯塔夫三世|古斯塔夫三世]]身亡和[[芬蘭大公國|芬蘭脫離瑞典統治]]而向下滑落，但卡爾斯克魯納依然是瑞典海軍的首要基地<ref name="UNESCO"></ref>。在該世紀中，基地西北側建起了一道外部屏障，並稱之為「船塢圍牆」（{{lang|sv|Varvsmuren}}），以保護營區內的扇形船塢設施<ref>{{en}}{{lang|en|{{Cite web |title=22. Varvsmuren (The dockyard wall) |url=http://www.orlogsstadenkarlskrona.se/page/120/varvsmuren.aspx |work=[[卡爾斯克魯納市|Karlskrona Kommun]] |accessdate=2014-12-12 |deadurl=no |archiveurl=https://web.archive.org/web/20141213164212/http://www.orlogsstadenkarlskrona.se/page/120/varvsmuren.aspx |archivedate=2014-12-13 }}}}</ref>。這道牆一開始採用木製結構支撐，後來全面改為石造壁體<ref>{{sv}}{{lang|sv|{{Cite web |title=22. Varvsmuren |url=http://www.orlogsstadenkarlskrona.se/page/44/22-varvsmuren.aspx |work=[[卡爾斯克魯納市|Karlskrona Kommun]] |accessdate=2014-12-12 |deadurl=no |archiveurl=https://web.archive.org/web/20141213164208/http://www.orlogsstadenkarlskrona.se/page/44/22-varvsmuren.aspx |archivedate=2014-12-13 }}}}</ref>。1821年至1826年間，海軍基地北面的入口處設置了一座[[新古典主義建築|新古典主義風格]]的衛兵樓（{{lang|sv|Högvakten}}），正對著鎮上的街區<ref>{{sv}}{{lang|sv|{{Cite web |title=13. Högvakten |url=http://www.orlogsstadenkarlskrona.se/page/34/13-hogvakten.aspx |work=[[卡爾斯克魯納市|Karlskrona Kommun]] |accessdate=2014-12-12 |deadurl=no |archiveurl=https://web.archive.org/web/20141213164216/http://www.orlogsstadenkarlskrona.se/page/34/13-hogvakten.aspx |archivedate=2014-12-13 }}}}</ref>，並充當基地玄關<ref name="pdf"></ref>。1847年時，瑞典海軍在斯圖火門島上增設海軍軍營（{{lang|sv|Båtsmanskasernen}}），內部可供500名水手長居住。1870年代內的基地大門西側也替{{Link-sv|瑞典少年水兵隊|Skeppsgossekåren|少年水兵隊}}建造了一棟少年水兵營房（{{lang|sv|Skeppsgossekasernen}}），作為他們服役時的訓練所<ref>{{en}}{{lang|en|{{Cite web |title=7. Skeppsgossekasernen (The Boy Seamen Barracks) |url=http://www.orlogsstadenkarlskrona.se/page/104/skeppsgossekasernen.aspx |work=[[卡爾斯克魯納市|Karlskrona Kommun]] |accessdate=2014-12-13 |deadurl=no |archiveurl=https://web.archive.org/web/20141213044849/http://www.orlogsstadenkarlskrona.se/page/104/skeppsgossekasernen.aspx |archivedate=2014-12-13 }}}}</ref>。1895年時，又在大門東側蓋了卓利營（{{lang|sv|Bataljon af Trolle}}）設施，歸{{Link-sv|卡爾斯克魯納海岸砲兵團|Karlskrona kustartilleriregemente}}使用<ref>{{en}}{{lang|en|{{Cite web |title=9. Bataljon af Trolle (The af Trolle Battalion) |url=http://www.orlogsstadenkarlskrona.se/page/106/bataljonaftrolle.aspx |work=[[卡爾斯克魯納市|Karlskrona Kommun]] |accessdate=2014-12-13 |deadurl=no |archiveurl=https://web.archive.org/web/20141213050546/http://www.orlogsstadenkarlskrona.se/page/106/bataljonaftrolle.aspx |archivedate=2014-12-13 }}}}</ref>。

20世紀時的卡爾斯克魯納基地仍持續進行硬體更新。由於海軍的[[水上飛機|水上飛機]]曾於1914年至1949年間駐在斯圖火門島，因此軍方於1926年和1929年時在島上蓋了3號及4號機庫（{{lang|sv|Flyghangarerna}}）<ref>{{en}}{{lang|en|{{Cite web |title=39. Flyghangarerna (The Hangars Nos. 3 and 4) |url=http://www.orlogsstadenkarlskrona.se/page/139/flyghangarerna.aspx |work=[[卡爾斯克魯納市|Karlskrona Kommun]] |accessdate=2014-12-13 |deadurl=no |archiveurl=https://web.archive.org/web/20141213055344/http://www.orlogsstadenkarlskrona.se/page/139/flyghangarerna.aspx |archivedate=2014-12-13 }}}}</ref>。同於20世紀初期增添的還有1910年的軍事監獄（{{lang|sv|Kronohäktet}}）<ref>{{en}}{{lang|en|{{Cite web |title=35. Kronohäktet (The Military Gaol) |url=http://www.orlogsstadenkarlskrona.se/page/135/kronohaktet.aspx |work=[[卡爾斯克魯納市|Karlskrona Kommun]] |accessdate=2014-12-13 |deadurl=no |archiveurl=https://web.archive.org/web/20141213040312/http://www.orlogsstadenkarlskrona.se/page/135/kronohaktet.aspx |archivedate=2014-12-13 }}}}</ref>及1921年的海軍製衣廠（{{lang|sv|Beklädnadsverkstaden}}）<ref name="Beklädnadsverkstaden">{{sv}}{{lang|sv|{{Cite web |title=36. Beklädnadsverkstaden |url=http://www.orlogsstadenkarlskrona.se/page/58/36-bekladnadsverkstaden.aspx |work=[[卡爾斯克魯納市|Karlskrona Kommun]] |accessdate=2014-12-13 |deadurl=no |archiveurl=https://web.archive.org/web/20141213160812/http://www.orlogsstadenkarlskrona.se/page/58/36-bekladnadsverkstaden.aspx |archivedate=2014-12-13 }}}}</ref>。[[第二次世界大戰|第二次世界大戰]]期間，一些年代較久遠的要塞和防禦體系也進行了現代化，還強化了抵抗敵機襲擊的基地防空能力<ref name="UNESCO"></ref>。1960年代及1970年代時，軍用設施逐漸遷出特羅索島上的鬧區和斯圖火門島<ref name="Stumholmen"></ref>，此前的卡爾斯克魯納鎮中心有將近一半的土地被海軍基地佔據<ref name="海軍技術網"></ref>。

1981年10月時的卡爾斯克魯納基地是{{Link-en|瑞典海域潛艇入侵事件|Swedish submarine incidents|蘇聯潛艇入侵案}}的事發之處。當時瑞典海軍於該基地周邊的海上舉辦[[軍事演習|軍事演習]]、測試[[電腦|電腦]]導引的新型[[魚雷|魚雷]]<ref name="情報與反情報">{{zh-hant}}{{Cite book|title=情報與反情報 |author=張殿清 |publisher=時英出版社 |location=台北 |date=2001 |accessdate=2014-11-28 |isbn=957-8890-71-0 |page=頁298-299}}</ref>，而[[蘇聯海軍|蘇聯海軍]]為了詳加刺探，派出{{Link-en|蘇聯S-363號潛艇|Soviet submarine S-363|S-363號潛艇}}前往當地執行情報蒐集任務。該潛艇抵達瑞典近海後，又闖入了卡爾斯克魯納基地周邊的管制水域，出沒位置距離鎮中心僅有30公里<ref name="Guide"></ref>{{rp|p215}}。10月27日晚間，S-363號潛艇在管制水域內擱淺<ref name="Aspects">{{en}}{{lang|en|{{Cite book |title=The Future a Memory: The Cold War and Intelligence Services – Aspects |publisher=LIT Verlag Münster |date=2013 |author=Heiner Timmermann |isbn=3643904428 |page=p. 145}}}}</ref>，因而暴露行蹤。瑞典軍方進入戒備狀態，出動了[[直升機|直升機]]、[[巡洋艦|巡洋艦]]、[[驅逐艦|驅逐艦]]和[[獵潛艦|獵潛艦]]進行圍捕，避免該潛艇逃離<ref name="情報與反情報"></ref>。此起國際事件引發了{{Link-sv|蘇聯－瑞典關係|Relationer mellan Sovjetunionen och Sverige}}的緊繃，因為瑞典軍方事後還發現蘇軍在艇內裝配了[[核子武器|核子武器]]<ref name="Guide"></ref>{{rp|p215}}。兩國之間的僵局直到11月6日才落幕<ref name="Aspects"></ref>，以蘇聯向瑞典道歉賠款<ref name="情報與反情報"></ref>、瑞典交還潛艇和被俘人員告終。該艘潛艇的艇長{{Link-ru|阿納托利·古申|Гущин, Анатолий}}在被押期間，曾於卡爾斯克魯納基地內的{{Link-sv|韋斯特維克號飛彈快艇|HMS Västervik (T136/R136)}}上接受審問<ref name="Initiative"></ref>。

至1990年代時，瑞典海軍的設施和活動已悉數從斯圖火門島撤離<ref name="Initiative"></ref>，全島也都完全化為居民活動地區<ref name="Stumholmen"></ref>，還有部分土地轉作民宅用途<ref name="Beklädnadsverkstaden"></ref>。1997年，{{Link-en|瑞典海軍博物館|Marinmuseum}}在該島正式開館，透過館藏品和展覽活動來公開瑞典海軍、基地和船塢自17世紀以降的發展史<ref name="Stumholmen"></ref>。1998年，[[聯合國教科文組織|聯合國教科文組織]]下的世界遺產委員會正式將卡爾斯克魯納軍港列為文化遺產，稱其為「一座獨一無二的、保存完好的歐洲海軍城鎮規劃案例，融入了他國早期軍港的元素，又替後世具有類似機能的城鎮建立了楷模」<ref>{{en}}{{lang|en|{{Cite web |title=Naval City of Karlskrona |url=http://worldheritagesweden.se/en/world-heritages-in-sweden/naval-city-of-karlskrona/ |work=[[瑞典世界遺產列表|World heritages in Sweden]] |accessdate=2014-12-13 |deadurl=no |archiveurl=https://web.archive.org/web/20141213181008/http://worldheritagesweden.se/en/world-heritages-in-sweden/naval-city-of-karlskrona/ |archivedate=2014-12-13 }}}}</ref>。

==海軍派駐部隊==
[[File:Stockholmclass.jpg|thumb]]]]

卡爾斯克魯納基地的戰鬥部隊包括瑞典海軍{{Link-en|瑞典海軍第1潛艇艦隊|1st Submarine Flotilla (Sweden)|第1潛艇艦隊}}（{{lang|sv|1. ubflj}}）和{{Link-en|瑞典海軍第3作戰艦隊|3rd Naval Warfare Flotilla (Sweden)|第3作戰艦隊}}（{{lang|sv|3. sjöstridsflj}}）。第1艦隊是瑞典海軍唯一的潛水艇部隊，負責船艦情報蒐集<ref name="1st">{{en}}{{lang|en|{{Cite web |title=The 1st Submarine Flotilla |url=http://www.forsvarsmakten.se/en/our-organisation/training-units-schools-and-centres/the-1st-submarine-flotilla/ |work=[[瑞典國防軍|Swedish Armed Forces]] |accessdate=2014-12-13 |deadurl=yes |archiveurl=https://web.archive.org/web/20141024200540/http://www.forsvarsmakten.se/en/our-organisation/training-units-schools-and-centres/the-1st-submarine-flotilla/ |archivedate=2014-10-24 }}}}</ref>、全年監控並保衛瑞典領海，也會參與海上特戰行動。旗下艇種包括[[哥得蘭級潛艇|哥得蘭級潛艇]]3艘（{{Link-sv|哥得蘭號潛艇|HMS Gotland (Gtd)|哥得蘭號}}、{{Link-sv|烏普蘭號潛艇|HMS Uppland (Upd)|烏普蘭號}}、{{Link-sv|哈蘭號潛艇|HMS Halland (Hnd)|哈蘭號}}）及{{Link-en|南曼蘭級潛艇|Södermanland-class submarine}}2艘（{{Link-sv|南曼蘭號潛艇|HMS Södermanland (Söd)|南曼蘭號}}、{{Link-sv|東哥得蘭號潛艇|HMS Östergötland (Ögd)|東哥得蘭號}}）<ref name="海軍技術網"></ref>。

[[File:HMS_Carlskrona_(M04).jpg|thumb]]
第3作戰艦隊是瑞典海軍的快速反應部隊<ref name="3rd">{{en}}{{lang|en|{{Cite web |title=The Third Naval Warfare Flotilla |url=http://www.forsvarsmakten.se/en/our-organisation/training-units-schools-and-centres/the-third-naval-warfare-flotilla/ |work=[[瑞典國防軍|Swedish Armed Forces]] |accessdate=2014-12-13 |deadurl=yes |archiveurl=https://web.archive.org/web/20141024194825/http://www.forsvarsmakten.se/en/our-organisation/training-units-schools-and-centres/the-third-naval-warfare-flotilla/ |archivedate=2014-10-24 }}}}</ref>，由第31護衛艦支隊（{{lang|sv|31. Korvettdivisionen}}）、第33除雷支隊（{{lang|sv|33. Minröjningsdivisionen}}）和第34補勤支隊（{{lang|sv|34. Underhållsdivisionen}}）組成。31支隊有2艘{{Link-en|斯德哥爾摩級護衛艦|Stockholm-class corvette}}（{{Link-sv|斯德哥爾摩號護衛艦|HMS Stockholm (K11)|斯德哥爾摩號}}、{{Link-sv|馬爾默號護衛艦|HMS Malmö (K12)|馬爾默號}}）、2艘[[偉士比級護衛艦|偉士比級護衛艦]]（{{Link-sv|赫爾辛堡號護衛艦|HMS Helsingborg (K32)|赫爾辛堡號}}、{{Link-sv|海訥桑德號護衛艦|HMS Härnösand (K33)|海訥桑德號}}）及3艘{{Link-en|塔帕級巡邏艇|Tapper-class patrol boat}}。第33支隊配有{{Link-en|寇斯特級水雷反制艇|Koster-class mine countermeasures vessel|寇斯特級}}和{{Link-en|史塔索級水雷反制艇|Styrsö-class mine countermeasures vessel}}<ref name="海軍技術網"></ref>，處理掃雷及獵雷勤務。第34支隊則負責支援其他船艦的食糧、燃料等物資補給，該隊的{{Link-sv|卡爾斯克魯納號遠洋巡邏艦|HMS Carlskrona (P04)|卡爾斯克魯納號}}曾扮演過佈雷艦、補給艦及海外行動指揮艦等多種角色<ref name="3rdsv">{{sv}}{{lang|sv|{{Cite web |title=Tredje sjöstridsflottiljen |url=http://www.forsvarsmakten.se/sv/organisation/tredje-sjostridsflottiljen/ |work=[[瑞典國防軍|Försvarsmakten]] |accessdate=2014-12-13 |deadurl=no |archiveurl=https://web.archive.org/web/20141213202911/http://www.forsvarsmakten.se/sv/organisation/tredje-sjostridsflottiljen/ |archivedate=2014-12-13 }}}}</ref>。

卡爾斯克魯納基地還派駐有{{Link-sv|瑞典海軍作戰訓練所|Sjöstridsskolan}}（{{lang|sv|SSS}}），其職責是提供全體海軍官兵的海上作戰、兩棲作戰培訓項目，也包括單兵裝備、海戰武器、相關科技及戰術的研發<ref>{{sv}}{{lang|sv|{{Cite web |title=Sjöstridsskolan |url=http://www.forsvarsmakten.se/sss/ |work=[[瑞典國防軍|Försvarsmakten]] |accessdate=2014-12-13 |deadurl=no |archiveurl=https://web.archive.org/web/20141213204420/http://www.forsvarsmakten.se/sss/ |archivedate=2014-12-13 }}}}</ref>。{{Link-sv|瑞典海軍基地隊|Marinbasen|海軍基地隊}}（{{lang|sv|MarinB}}）的隊部也設在卡爾斯克魯納基地，主要任務是支援瑞典海軍各個單位，以及為海軍在國內外的行動給予技術、運輸或後勤協助。基地隊下轄的一個連負責卡爾斯克魯納等海軍基地的戍衛及防禦，另外兩個海上情報連（{{lang|sv|Sjöinformationskompani}}）分別駐在斯德哥爾摩與哥特堡，透過[[雷達|雷達]]、船隻或飛機監控邊境<ref name="海軍基地隊"></ref>。
