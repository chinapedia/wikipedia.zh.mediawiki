'''卡伦河三号大坝'''（{{lang-fa|'''سد کارون ۳'''}}），是一座[[水力发电|水力发电]][[拱坝|拱坝]]，位于[[伊朗|伊朗]][[胡齐斯坦省|胡齐斯坦省]]的[[卡伦河_(伊朗)|卡伦河]]上游，距[[伊泽市|伊泽市]]28公里，距[[沙希德阿巴斯普尔大坝|沙希德阿巴斯普尔大坝]]约120公里。

== 概况 ==
卡伦河三号大坝的建设兼具发电和防洪的作用<ref>{{cite web|url=http://www.karun3.com/basic%5Finfo/?item=1|title=Basic information|accessdate=2007-02-04|deadurl=yes|archiveurl=https://web.archive.org/web/20061031100444/http://www.karun3.com/basic_info/?item=1|archivedate=2006-10-31}}</ref>，工程于1999年开工，土建工程于2004年完工。2005年3月第一台机组并网发电，2007年8台机组全部投入运行<ref name="chinanet">{{cite web|title=卡伦三级水电站|url=http://news.china.com.cn/txt/2015-05/15/content_35580193.htm|accessdate=2016-11-12}}</ref>。

== 坝体结构 ==
大坝属于混凝土双曲拱坝，大坝顶端距离坝基高度为205米，距离河床高度为185米，坝顶长462m、顶厚5.5m，坝基厚度29.5米，坝体混凝土方量130万立方米。<ref name="chinanet"/><ref>{{cite web|url=http://www.karun3.com/technical/?item=1 |title=Karum3 dam body - technical information |accessdate=2007-02-04 |deadurl=yes |archiveurl=https://web.archive.org/web/20061031101012/http://www.karun3.com/technical/?item=1 |archivedate=2006-10-31 }}</ref>。大坝被设计在一个狭窄的河谷中，能够将河水阻挡和蓄存在岩石峡谷之间，形成的水库库容为29.7亿立方米。由于大坝的拱形设计，能够将水库的水压，通过坝体传导，形成向下的压力，从而进一步增强坝基的承受能力<ref>{{cite web|url=http://www.karun3.com/history/?item=1|title=Dam history|accessdate=2007-02-04|deadurl=yes|archiveurl=https://web.archive.org/web/20080706005450/http://www.karun3.com/history/?item=1|archivedate=2008-07-06}}</ref>，。

== 发电 ==
大坝引水发电系统位于右岸，引水洞长551.5米，内径12.6米，开挖直径14米，混凝土衬砌。发电厂房为地下厂房，位于大坝下游500米的右岸山体内，主厂房长250米，高48米，宽25米。电站有8台单机容量为{{Convert|285|MW}}的[[法兰西斯式水轮机|法兰西斯式水轮机组]]，由[[中华人民共和国|中国]][[哈尔滨电气集团公司|哈尔滨电机厂]]生产，总装机容量{{Convert|2,280|MW}}，年均发电能力为{{Convert|4,137|GWh}}<ref name="chinanet"/><ref>{{cite web |url=http://www.industcards.com/hydro-iran.htm |title=存档副本 |accessdate=2013-02-21 |deadurl=yes |archiveurl=https://archive.is/20120908213617/http://www.industcards.com/hydro-iran.htm |archivedate=2012-09-08 }}</ref> 。在用电高峰时，大坝的发电机组所产生的电能，能够供应到伊朗国家电网系统，从而缓解缺电情况。
