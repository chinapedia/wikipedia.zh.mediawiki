'''卡尔顿山'''（Calton Hill）是[[苏格兰|苏格兰]][[爱丁堡|爱丁堡]][[爱丁堡新城|新城]]东侧的一座小山。这座山的风景，以及从这座山看到的风景，常用于这座城市的照片和绘画。

对于卡尔顿这个名字的来源，较为更合乎逻辑的解释，是其源于[[盖尔语|盖尔语]]"choille-dun"（森林山）或 "cauldh -dun" （黑山）。由于这座山的黑色玄武岩结构，后者是最适合的解释。

卡尔顿山是[[苏格兰政府|苏格兰政府]]总部的所在地<ref name = Youngson>Youngson, A.J. (2001): “The Companion Guide to Edinburgh and the borders”, Chapter 9 (Calton Hill), Polygon Books, Edinburgh, UK, ISBN 0-7486-6307-X</ref>，[[圣安德鲁大厦|圣安德鲁大厦]]就位于这座山陡峭的南坡；而[[苏格兰议会大楼|苏格兰议会大厦]]，以及[[荷里路德宫|荷里路德宫]]<ref name = RLS/>等重要建筑，就位于山脚下附近。这座山还有一些标志性的古迹和建筑：[[国家纪念堂_(爱丁堡)|国家纪念堂]]<ref name = RLS>[http://www.undiscoveredscotland.co.uk/usebooks/steveson-edinburgh/chapter08.html The Calton Hill], Robert Louis Stevenson(1879), "Edinburgh: Picturesque Notes", Chapter 8, ASIN: B00158QM62. Accessed on 2009-08-06.</ref>、[[纳尔逊纪念塔|纳尔逊纪念塔]]<ref name = RLS/><ref name = OldEdin/>、[[杜格尔德·斯图尔特纪念亭|杜格尔德·斯图尔特纪念亭]]<ref name = RLS/><ref name = OldEdin/>、[[皇家中学_(爱丁堡)|老皇家中学]]<ref name = RLS/><ref name = OldEdin/>、[[罗伯特·伯恩斯|罗伯特·伯恩斯]]纪念碑<ref name = RLS/>、[[政治烈士纪念碑|政治烈士纪念碑]]以及市天文台<ref name = OldEdin/><ref name = Anne>Mitchell , Anne (1993), “The People of Calton Hill”, Mercat Press, James Thin, Edinburgh,  ISBN 1873644 183.</ref>。

==政治符号==
卡尔顿山是苏格兰分權自治（devolution）的一个重要地点。[[皇家中学_(爱丁堡)|老皇家中学]]曾被指定为未来的苏格兰大会（Scottish Assembly）所在地，1998年苏格兰法案通过后，又作为[[苏格兰议会|苏格兰议会]]有潜力的竞争地点，然而，当时的苏格兰事务大臣唐納德·杜瓦（Donald Dewar）担心这个地点成为“[[民族主义|民族主义]]者的[[示播列|示播列]]”，最终[[苏格兰议会大楼|苏格兰议会大厦]]选址在山下的荷里路德。2004年10月，《卡尔顿山声明》也是在卡尔顿山发表，提出未来成立苏格兰[[共和国|共和国]]的要求。

==早期历史==
卡尔顿山自从古代就被用作刑场。有名的术士Major Weir在这里被处决。1456年，苏格兰国王[[詹姆斯二世|詹姆斯二世]]允许爱丁堡居民使用山的西北坡进行[[马上长矛比武|马上长矛比武]]<ref name = Youngson/>。这也是个天然的圆形剧场，用作露天剧场，苏格兰人在此第一次演出"The Three Estaites"。1518年，[[加尔默罗会|加尔默罗会]]修士在山的西侧建立了一座修道院<ref name = OldEdin>[http://www.oldandnewedinburgh.co.uk/volume3/page112.html Calton Hill] Grant, James "Old and New Edinburgh"  Volume 3, Chapter XVI, printed as a periodical in the 1880s, Accessed 2009-08-12</ref>。

1560年[[苏格兰宗教改革|苏格兰宗教改革]]后，修道院的土地归属巴尔梅里诺勋爵。后来在16世纪，建立了一个麻疯病院<ref name = Anne/>。1631年，巴尔梅里诺勋爵授权给卡尔顿商人（主要是一批鞋匠）在这片土地定居，建立了一个小村庄，中间有一个广场。1718年，这个广场用围墙封闭，成为一片墓地，今天称为旧卡尔顿公墓。

1669年，该地区获得自治市镇的地位。1724年，爱丁堡皇家自治市向巴尔梅里诺勋爵购买了这座山<ref name = Anne/>，因此这一地区随即升格，拥有皇家自治市的权利地位（1725年）。直到1859年，卡尔顿山才正式与爱丁堡市合并。

==建筑==
[[File:Regent_Terrace_Edinburgh3.JPG|thumb]]
卡尔顿山上原本设有臭名昭著的卡尔顿监狱<ref name = RLS/><ref name = OldEdin/>（后来被索顿监狱所取代），以及关押合伙债务人的布里奇韦尔监狱。这两个监狱大部分被拆除，兴建圣安德鲁大厦，驻有苏格兰的高级公务员。布里奇韦尔监狱的围墙在圣安德鲁大厦南侧的卡尔顿路上仍可见到。

旧卡尔顿公墓位于卡尔顿山的东南侧<ref name = OldEdin/><ref name = Anne/>，哲学家[[大卫·休谟|大卫·休谟]]安葬于此<ref name = Youngson/><ref name = OldEdin/>。休谟这样写他自己的[[墓志铭|墓志铭]]：
“生于1711年，死于 [----]。其余的留给后人添加。”在他去世的1776年，雕刻在一个“简单的罗马墓地”，如同他所希望的，这块墓地俯瞰着他在新城圣大卫街1号的家。[[政治烈士纪念碑|政治烈士纪念碑]]也位于公墓内<ref name = Anne/>，是为了纪念五位法国革命的崇拜者，他们被处以叛乱罪名，于1793年流放到澳大利亚<ref name = Youngson/><ref name = OldEdin/>。

著名的苏格兰建筑师威廉·亨利·普莱费尔负责修建了三面环山的优雅道路，包括皇家台（Royal Terrace）、卡尔顿台（Carlton Terrace）和[[摄政台|摄政台]]（Regent Terrace）<ref name = Youngson/>，最大的联排住宅位于皇家台。第一座住宅兴建于普莱费尔规划发布的1819年，现在皇家台有40座<ref name = Anne/>。占山顶一半以上面积的花园由当地的居民协会管理<ref name = Anne/>。这里有许多旅馆<ref>[http://www.primahotels.co.uk/royal/default-en.html The Royal Terrace Hotel] 普莱费尔在山顶上修建了许多纪念性建筑，其中最著名的是[[国家纪念堂_(爱丁堡)|国家纪念堂]]<ref name = Anne/>。这座纪念性建筑以[[雅典|雅典]][[帕特农神庙|帕特农神庙]]为蓝本，纪念在[[拿破仑战争|拿破仑战争]]中阵亡的苏格兰士兵。国家纪念堂于1826年开工，1829年由于缺乏资金被迫停建，未能完成<ref name = Youngson/><ref name = OldEdin/>，被称为“苏格兰的耻辱”。 

华丽的摄政桥东端建在山的一侧，穿越深邃的峡谷（谷底是电影《[[迷幻列车|迷幻列车]]》的拍摄外景），连接卡尔顿山和[[王子街|王子街]]，现在爱丁堡的主要购物街。负责在1815年修建摄政桥的工程师罗伯特·史蒂文森，是作家[[罗伯特·路易斯·史蒂文森|罗伯特·路易斯·史蒂文森]]的祖父<ref name = Anne/>。

在卡尔顿山的西侧有一条街的名称也称为卡尔顿山。艾格尼丝·麦理浩，以[[罗伯特·伯恩斯|罗伯特·伯恩斯]]的克拉林达而广为人知，居住在14号<ref>[http://books.google.be/books?id=Fe4VAAAAIAAJ&pg=PA80-IA1&dq=clarinda+calton&hl=en#v=onepage&q=clarinda%20calton&f=false The Edinburgh Literary Journal: or, Weekly register of criticism ..., Volume 3] Contains a letter from Agnes Maclehose dated 1829-10-18 and her address of 14 Calton Hill. Accessed 2009-09-08</ref>，1841年在这里去世。苏格兰诗人伯恩斯花费数年，送给她许多首诗，但是未能成功地诱惑这位美丽的已婚女子<ref name = Anne/>。

==露天性爱==
近年来，卡尔顿山作为露天性爱的场所变得臭名昭著。随着吸毒者和未成年饮酒者的到来，男性之间的[[卖淫|卖淫]]正在增加。在爱丁堡，这一地区是传统的同性性行为场所<ref>McLaughlin, Mark. [http://edinburghnews.scotsman.com/edinburghssexindustry/Calton-Hill-39rent-boys39-move.3677172.jp "Calton Hill 'rent boys' move trade into residential streets"]. ''The Scotsman'' 16 Jan 2008</ref>。在一个不名譽的事件中，一名男子因在山脚下用[[交通錐|交通錐]]刺激自己而被捕<ref>[http://edinburghnews.scotsman.com/weirdoddandquirkystories/Police-charge-man-over-sex.2358479.jp "Police charge man over sex with traffic cone"]. ''The Scotsman'' 5 Sept 2002</ref>。

==活动==
[[File:Beltane_on_Calton_Hill_by_Bruce_McAdam.jpg|thumb]]
一年中，在卡尔顿山会举办许多活动，其中规模最大的是每年4月30日举行的五朔节（Beltane Fire Festival），有12,000人参加。这是古代凯尔特五朔节在卡尔顿山的复兴<ref>[http://heritage.scotsman.com/beltanefirefestival/Turning-up-the-heat-for.3278000.jp Beltane festival]</ref>。每年十月初，印度教达瑟拉节（Dussehra）也在卡尔顿山庆祝<ref>McEwan, Allen, [http://edinburghnews.scotsman.com/evening-news-video-archive/Festival-is-flaming-marvellous.4560948.jp Festival is flaming marvellous],Edinburgh Evening News, 2008-10-06, Accessed 2009-09-02</ref>。
