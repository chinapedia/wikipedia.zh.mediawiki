'''卡尼西'''（{{lang-fr|Canisy}}）是[[法国|法国]][[芒什省|芒什省]]的一个[[市镇_(法国)|市镇]]，属于[[圣洛区|圣洛区]]（Saint-Lô）[[卡尼西县|卡尼西县]]（Canisy）。该市镇总面积6.25平方公里，2009年时的人口为1028人。<ref>[http://www.insee.fr/fr/ppp/bases-de-donnees/recensement/populations-legales/france-departements.asp?annee=2009 法国INSEE人口数据-2009年]</ref>

卡尼西于2017年和相邻的[[圣埃布雷蒙德邦福塞_(芒什省)|圣埃布雷蒙德邦福塞]]合并形成了新的卡尼西。

历史
 
==拉芒什七大古迹之一 ==

卡尼西城堡被列为《拉芒什七大古迹之一》。

城堡主体分为两部分，这两部分所建的时代分别为十三世纪跟十六世纪。城堡内部共有70个区域，其中有18间是豪华客房。城堡内部巨大的花岗岩楼梯是建筑师弗朗索瓦·加布里埃尔的作品，（他是建造凡尔赛宫和小特里亚农宫的建筑师Ange Gabriel的祖先）。城堡内还有一座阁楼外加数个中世纪的古塔，同时为了丰富贵族的娱乐生活，还设有多个大小不一的沙龙：一个音乐客厅，两个中世纪，一个红色剧院，两间会议室，一个复古酒吧，一个舞池disco，甚至还有游戏厅，包括了法式和美式台球，和数个酒窖与顶楼。

==城堡内套间精美的装潢==

说到城堡内的十八间套房，其实每一间都独具特色，被全部重新装修成具有十八世纪古典的高雅风格。每间房子都有不同的名字，不同的风格，不同的寓意，且每间浴室都各有特色：配有大理石浴缸或铜制浴缸的奢华浴室。屋内的丝织品来自印度的手工艺制品，墙上的画作复制了一些十八世纪风格的名作。在经过了精心的构思及细致搭配后，室内装潢便得以完美的与城堡整体风格和历史特性相融合。这也是令许多客人惊叹不已的地方。

== 一个活着的历史见证地 ==

事实上，现存的卡西你城堡是在十三世纪旧古堡的废墟上建立的，不少地方仍保留着原有遗迹的面貌。由于历史悠久景色宜人的缘故，曾有不少的著名历史人物光顾于此：即位之前的路易四世，马蒂尼翁元帅，政治哲学家托克维尔，还有众多王公贵族的成员也都曾被邀请居住于此。近代著名的故事则是在第二次世界大战时，处在战争中心的卡尼西城堡被德军占领，之后被二战英豪及美国著名军事家、统帅James Rudder将军，与陆军五星上将Omar Bradley收回作为军事医院。


 
== 令人惊叹的卡尼西城堡 ==

这是一个拥有着悠远的历史，优美的环境，丰富的人文的古堡。卡尼西城堡坐落在一个占地4500亩的庄园内，四周被湖泊跟丛林所环绕，在巨大的庄园内，有着大小不一的数个湖泊，植物园，农场等区域。城堡被de Kergorlay家族所拥有，城堡已在家族内传承了千年，这在所有现存完好的法国城堡中实属罕见。城堡位于法国诺曼底地区，地处法国马匹养殖的第一大区中心。它周边可浏览的美景胜地也非常多，距奥马哈海滩不到一个小时，距圣米歇尔山只需45分钟车程，而到贝叶市（挂毯的故乡，著名大教堂）和拉芒什海峡仅需大约30分钟。

 
== 家族传承依旧热情好客 ==

在二战结束后，城堡成为了法国与美国之间友好合作的见证，以及相互沟通交流的桥梁。这种情谊通过城堡de Kergorlay家族的热情好客传统一直延续至今。近些年间在这座城堡中，曾接待了许多赫赫有名的人物，从政界到教育界，从皇族到历史文化爱好者：如麦克纳马拉，亨利·基辛格，中国追随毛泽东的老战士，日本小林洋太郎，许多美国与亚洲的著名政治领袖，及ASPEN峰会和其他许多国际会议上的政要。西班牙的海伦娜公主，日本的雅子皇太妃和她的家人，还有比利时的菲利普国王都曾在此下榻。

城堡凭借它得天独厚的地理位置和独特的历史，经常被作为诺曼底登录战争纪念日等相关活动的举办地，承办大大小小各种活动事宜。每年都会接待州长，军人和一些美国老兵及他们的家人。同时城堡的庄园还是诺曼底跑马会的举办地，由于这是法国著名的跑马胜地，历年来都会在此举办跑马赛，城堡主人丹尼斯伯爵也是此项运动的爱好者。

== 肩负着文化传播的使命 ==

城堡是文化传播与交融的圣地，城堡所有者de Kergorlay家族长期以来，都肩负着传播文化促进沟通的使命，曾经有各种艺术家聚集在此：如法国著名演员劳伦特·特兹弗，剧院曾以他的名字来命名，现在我们还可以在城堡一楼的大厅找到许多关于他的珍贵影像资料。还有法国的戏剧大师尚·侯谢弗，他在三十多岁时发现了对马术的爱好与激情，从此热衷骑马和马术表演。城堡定期举办各种音乐会，常来演出的有钢琴家安妮·克菲莱克，法国的著名古典钢琴表演家。竖琴师玛赫耶拉诺德曼，曾在巴黎演奏过备受关注的竖琴表演。还有歌手琼·贝兹，美国乡村民谣女歌手兼作曲家，她的很多作品都与时事和社会问题有关，并于1960年代活跃于反战运动。这些活跃于社会及艺术界的大艺术家们，都是城堡主人的朋友，时常会来城堡小住，给凡尘俗世忙碌的生活画一个休止符，惬意潇洒的在这里得到片刻休闲。

城堡的现任主人是de kergorlay伯爵夫妇，他们有两个孩子：玛丽·维克多利亚跟和亨利路易。他们都是英国国王以及诺曼底公爵亲属的后代。这个历经千年的家族中有很多著名人物，伯爵夫妇用其中一些先祖的名字给城堡的套房命名，使得房间富有故事和历史，也赋予自己祖先的荣光于此。例如：拿破仑的第一任皇后“约瑟芬皇后”，一楼的一个精致大气的房间便以此命名，其中的画作和家具装潢也与此相关。
