'''卡拉延市'''（{{lang-tl|Kalayaan}}）是[[菲律宾|菲律宾]]在[[南沙群岛|南沙群岛]]（菲方称“卡拉延群岛”）上所设置的行政区域。

卡拉延市在行政区划上属于[[民马罗巴区|民马罗巴区]][[巴拉望省|巴拉望省]]，其行政中心位于中业岛，下辖一村。1978年6月11日，菲律宾总统[[费迪南德·马科斯|费迪南德·马科斯]]签署法律文件，在南沙群岛菲占岛屿上设立卡拉延市。其所实际控制的岛屿有：[[中业岛|中业岛]]（Pag-asa）、[[西月岛|西月岛]]（Likas）、[[北子岛|北子岛]]（Parola）、[[马欢岛|马欢岛]]（Lawak）、[[南钥岛|南钥岛]]（Patag）、[[杨信沙洲|杨信沙洲]]（Panata）、[[火艾礁|火艾礁]]（Balagtas shoal）、[[仁爱礁|仁爱礁]]（Ayungin shoal）和[[司令礁|司令礁]]（Rizal）。

卡拉延市自设立以来，一直不被中国及越南所承认。

== 历史 ==
1971年，菲律宾政府对探险家[[托马斯·克洛马|托马斯·克洛马]]在1956年占领并纳入[[自由地|自由地]]领土的60个岛屿、岛礁和沙洲提出主权要求，并将它们命名为“卡拉延群岛”（{{lang|en|''Kalayaan Island Group''}}，[[他加禄语|他加禄语]]意思是“自由地”）。1974年11月，克罗马被菲律宾总统[[费迪南德·马科斯|费迪南德·马科斯]]（{{lang|en|''Ferdinand Marcos''}}）逮捕，并被迫以一[[菲律宾比索|菲律宾比索]]的价格将自由地出售给菲律宾，以换取自己的自由<ref>{{cite web |language = zh-hans |publisher = 南沙群岛在线>南沙群岛历史点滴 |title = 南海诸岛大事年表（1950年－1974年） |url = http://www.nansha.org.cn/history/3/nanhai_zhudao_chronology_1950_1974.html |author = 张涛 (zt) |date = 2006年4月24日 |accessdate = 2006年4月24日}}</ref>。

1978年，菲律宾以其地设置卡拉延市。1980年1月30日，Aloner M. Heraldo当选首任市长。

2013年5月13日，[[欧亨尼奥·B·比托-奥农|欧亨尼奥·B·比托-奥农]]（Eugenio B. Bito-Onon Jr.）当选卡拉延市市长。
