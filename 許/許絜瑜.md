'''許絜瑜'''（'''"Connie" Hsu Chieh-Yu'''，{{bd|1992年|1月14日|catIdx=C許}}）是[[臺灣|臺灣]]職業[[網球|網球]]選手。

== 生平 ==

許絜瑜5歲接觸網球，曾贏得四維膠帶盃學童網球錦標賽二年級組（第12屆）及四年級組（第14屆）冠軍<ref>{{Cite web |url = http://www.fptennis.com.tw/page_b_3-1.htm |title = 第一屆至三十一屆四維膠帶盃學童網球錦標賽成績 |language = zh-tw |accessdate = 2017年3月5日 }}</ref>；9歲時全家移民至美國[[辛辛那堤|辛辛那堤]]接受培訓、並在12歲及14歲時取得全美青少女第一。大學申請就讀[[賓夕法尼亞大學|賓夕法尼亞大學]]，一年後休學並專注投入職業網壇。<ref>{{Cite web |url = http://2017.taipei/files/15-1000-1579,c258-1.php |title = 轉籍台灣 瞄準世大運奪金台灣網球甜心 許絜瑜 |language = zh-tw |accessdate = 2017年3月5日 }}</ref>

2014年4月，許絜瑜申請轉籍回國取得[[國際網球總會|國際網球總會]]認可，同年並代表參加[[2014年亞洲運動會|仁川亞運]][[2014年亞洲運動會中華台北代表團#.E7.B6.B2.E7.90.83|網球女子單打]]項目；2015年就讀[[臺北市立大學|臺北市立大學]]，並參與[[2015年夏季世界大學運動會|光州世大運]]贏得女雙銀牌及女團金牌。
