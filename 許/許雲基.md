'''許雲基'''（{{bd|1924年|7月16日||||catIdx=許}}），[[臺灣|臺灣]][[臺中|臺中]]人，為[[原子核物理學|原子核物理學]]家<ref name="对话春运"/><ref name="再談早期台大核子物理實驗室">{{cite journal |author=鄭伯昆 |title=再談早期台大核子物理實驗室 |url=http://psroc.phys.ntu.edu.tw/bimonth/download.php?d=1&cpid=33&did=3 |journal=物理雙月刊 |publisher=[[中華民國物理學會|中華民國物理學會]] |date=1999年2月 |volume=29 |issue=1 |language=zh-tw |pages=23-35 |accessdate=2015-08-14 |archiveurl=https://www.webcitation.org/6amDpJgjm?url=http://psroc.phys.ntu.edu.tw/bimonth/download.php?d=1 |archivedate=2015-08-14 |deadurl=no }}</ref>{{rp|34}}。在從[[臺中一中|臺中一中]]畢業後，許雲基曾赴日本內地的[[京都帝國大學|京都帝國大學]]工學部就讀，並於戰後轉回[[國立臺灣大學|國立臺灣大學]]電機系並自該系畢業<ref name="張幸真">{{cite journal zh|author=張幸真 |date=2003-07-31 |title=台灣知識社群的轉變－以台北帝國大學物理講座到台灣大學物理系為例 |pages=20 |url=http://ndltd.ncl.edu.tw/cgi-bin/gs32/gsweb.cgi?o=dnclcdr&s=id=%22091NTU00493013%22.&searchmode=basic |accessdate=2012-06-11 |language=zh-tw |archiveurl=https://web.archive.org/web/20150628071851/http://ndltd.ncl.edu.tw/cgi-bin/gs32/gsweb.cgi?o=dnclcdr&s=id%3D%22091NTU00493013%22.&searchmode=basic |archivedate=2015-06-28 |deadurl=no }}</ref>{{rp|161}}。

1947年，許雲基成為[[國立臺灣大學物理學系|國立臺灣大學物理學系]]的助教，與[[戴運軌|戴運軌]]和[[太田賴常|太田賴常]]等人一同於臺大重建[[荒勝文策|荒勝文策]]團隊當年在臺大製作、曾完成世界第二次人工撞擊[[原子核|原子核]]實驗的{{le|考克饶夫-瓦尔顿加速器|Cockcroft–Walton generator}}，並成功於1948年完成了[[中華民國|中華民國]]暨戰後全亞洲首次人工撞擊原子核實驗。爾後，許雲基接手了太田賴常主持的原子核物理實驗室<ref name="林夏玉"/>，並領導臺籍技師們屢次改良該實驗室之設備<ref name = "張幸真"/>{{rp|166}}，使其成為1960年代亞洲最先進的原子核實驗室<ref name="臺灣物理史數位典藏計畫"/>{{rp|4}}<ref name="新活水"/>{{rp|99-100}}。爾後，許雲基由助教漸次升為講師、副教授、教授，進而在1962年成為臺大物理系的系主任<ref name="系主任"/>。1985年，許雲基從臺大物理系退休，於1989年成為該系的名譽教授<ref name="臺灣物理史數位典藏計畫"/>{{rp|15,21}}<ref name = "張幸真"/>{{rp|180}}，並在2005年起率領重建原子核實驗室成為[[臺大物理文物廳|臺大物理文物廳]]<ref name="原子核物理"/><ref name ="原子核物理實驗室"/><ref name ="Hi! NTU"/>{{rp|151}}。

==生平==
===早年===
許雲基於1924年生於[[日治臺灣|日治臺灣]]的[[臺中州|臺中州]][[大甲郡|大甲郡]][[外埔庄|外埔-{庄}-]]{{efn|今[[臺中市|臺中市]][[外埔區|外埔區]]。}}<ref name="对话春运"/>，中學時期就讀於[[臺中一中|臺中一中]]<ref name="林夏玉"/>。在從臺中一中畢業後，許雲基赴日本內地的[[京都帝國大學|京都帝國大學]]工學部就讀，並在就讀該校時期聽過[[荒勝文策|荒勝文策]]開設的課程，同時得知該校[[原子核|原子核]]實驗室的[[木村毅一|木村毅一]]等人正從事的原子核實驗<ref name = "張幸真"/>{{rp|161}}。[[臺灣戰後時期|二戰結束後]]，許雲基從京都帝國大學電氣系轉至臺大電機系就讀<ref name="林夏玉"/>，並於1947年夏天從該系畢業<ref name="对话春运"/>。在畢業前夕，許雲基受到化學系助教張苕旭{{efn|許雲基在臺中一中的同學。}}推薦前往[[國立臺灣大學物理學系|國立臺灣大學物理學系]]的原子核擊破實驗小組，該系副教授[[太田賴常|太田賴常]]並希望許雲基能在畢業後第二天就來報到<ref name="科學月刊"/>{{rp|914}}。但是許雲基在畢業後卻先回大甲家中休息了一段期間，並於該年9月底才返回[[臺北|臺北]]，10月正式開始於臺大物理系任職<ref name="林夏玉"/><ref name="对话春运"/><ref name="中国科技史杂志"/>{{rp|157}}。為此，許雲基受到了狠狠的一番責備<ref name="科學月刊"/>{{rp|914}}。根據許仲平的說法，當時的許雲基為人溫和且謙虛，講話時常無意地用一隻手遮住口，直到晚年仍是如此<ref name="科學月刊"/>{{rp|914}}。

name="時空1967"/>{{rp|3-4}}。許雲基團隊的這些研究讓臺大物理系的原子核實驗室成為1960年代亞洲最先進的原子核實驗室，並發表了近三十篇論文<ref name="臺灣物理史數位典藏計畫"/>{{rp|4}}<ref |archivedate=2015-08-21 |deadurl=no }}</ref>{{rp|8}}。在臺大物理系任職期間，許雲基經由一篇篇論文的累積，由助教漸次升為講師、副教授、教授，進而在1962年10月起擔任臺大物理系的系主任<ref name="系主任">{{cite web|title=臺大物理系 歷屆系主任 ===晚年===
[[File:NTU_PhysicsOldBuilding.JPG|200px]]理化學教室校舍。內有臺大物理文物廳，即為當年的原子核實驗室。]]
1985年，許雲基從臺大物理系退休，並在1989年與[[沃爾夫岡·克洛爾|沃爾夫岡·克洛爾]]一同成為臺大物理系的名譽教授<ref name="臺灣物理史數位典藏計畫"/>{{rp|15,21}}<ref name = "張幸真"/>{{rp|180}}。在他退休後，由於原子核實驗室空間不足，導致其加速器被迫拆解，但林松雲和許玉釧二位技師因為不捨而將其零件四處分藏各處近二十年<ref name ="原子核物理實驗室"/><ref name ="Hi! NTU"/>{{rp|151}}。他們二人甚至將那些零件分藏在倉庫、地下室、浴室天花板等處<ref name="原子核物理">{{cite web|title=臺大物理文物廳 原子核物理 人物歷史 |url=http://web.phys.ntu.edu.tw/gphyslab/nuphys/DeHistoryPlace_1.html |language=zh-tw |publisher=[[臺大物理文物廳|臺大物理文物廳]] |accessdate=2015-08-24 |archiveurl=https://web.archive.org/web/20150604034325/http://web.phys.ntu.edu.tw/gphyslab/nuphys/DeHistoryPlace_1.html |archivedate=2015-06-04 |deadurl=no }}</ref><ref name="新活水"/>{{rp|100-101}}。到了2004年，臺大物理系決定重組該加速器，並在2005年起由許雲基偕同林松雲和許玉釧二位技師，一同率領重建原子核實驗室成為[[臺大物理文物廳|臺大物理文物廳]]，並於該年11月21日正式開幕<ref name="原子核物理"/><ref name ="原子核物理實驗室"/><ref name ="Hi! NTU"/>{{rp|151}}。該次重建過程並被製成了紀錄片《[[衝破原子核|衝破原子核]]》<ref name="臺灣物理史數位典藏計畫"/>{{rp|9,21}}。許雲基總結三十八年的教職生涯：「很困難的時代，但大家很認真」<ref name="林夏玉"/>。
