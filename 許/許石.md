'''許石'''（{{bd|1919年|9月24日<ref name="許石維基恐被刪">{{cite news |language = zh-tw |author = 陽昕翰 |coauthors = 許朝欽 |url = http://ent.ltn.com.tw/news/breakingnews/1426473 |title = 許石維基恐被刪 徵高手護台灣歌謠 |publisher = 自由時報 |pages =  |date = 2015-08-28 |accessdate = 2015-09-02}}</ref>|1980年|9月10日<ref name="許石維基恐被刪" />|catIdx=Hsu許石}}），是[[二戰|二戰]]後的台灣作曲家，創作出許多台灣歌謠，《[[安平追想曲|安平追想曲]]》<ref group="影片">{{youtube|wnzzCOJg6SE|錄下當時許石親自開唱《安平追想曲》}}，發布時間：2013年4月14日，引用時間：2015年8月29日。</ref><ref name="a" group="參">{{cite news| url=http://www.peoplenews.tw/news/7cb373c0-e4b4-4e14-9a4c-161d15089004 |title=五線譜上的許石| publisher=台灣民報新聞網| date=2015年5月27日 | id={{ISBN|978-9-869-18790-9}} | accessdate=2015年8月29日}}</ref>、《南都之夜》<ref group="影片">{{youtube|gVIyVjqqv_0|南都之夜之台語歌曲}}，發布時間：2011年11月13日，引用時間：2015年8月29日。</ref>為其代表作。1946年，許石自[[日本|日本]]回台，逐步收集整理散落在各地方的[[台灣民謠|台灣民謠]]與推廣鄉土音樂<ref group="參">{{cite news| url=https://sites.google.com/site/hsushih1980/|title=臺灣民謠的記事--許石先生}}</ref>，其運用[[交響樂|交響樂]]形式提升台灣民謠地位<ref  group="參">{{cite news| url=http://journal.nmtl.gov.tw/opencms/nmtl_search/download.html?atype=B&ctype=F&volume=47&id=2015060047063065|title=許石與郭一男 - 台灣文學研究學報}}</ref>，並組織[[中國|中國]]風的合唱團到處演唱宣傳，《解析臺語流行歌曲：鄉愁、翻轉與逆襲》一书评价許君開台灣流行音樂風氣之先河<ref>徐玫玲﹐2014﹐《解析臺語流行歌曲：鄉愁、翻轉與逆襲》﹐東和音樂出版社﹐第180-181頁。</ref><ref>石計生﹐2015﹐《漂亮你一人：戰後台灣歌謠先驅者許石研究》﹐「台灣流行歌謠─與日本、中國的時代交織」國際學術研討會，日本名古屋大學，2015年7月18日。</ref>。

==生平==
[[File:臺南許石故居.jpg|thumb]]
1936年時，許石坐船赴[[東京|東京]]的[[日本歌謠學院|日本歌謠學院]]唸書，接受日本知名作曲家秋月、{{tsl|ja|大村能章|大村能章}}、吉田恭章教導，研習理論作曲、聲樂與演歌{{sfn|葉添進|2009|p=2}}，而[[吳晉淮|吳晉淮]]是同校出身的[[前輩|前輩]]<ref>{{Cite web |url = http://event.moc.gov.tw/sp.asp?xdurl=ccEvent2013/ccEvent_cp.asp&cuItem=2120388&ctNode=676&mp=1 |title = 臺灣歌謠專題演講：臺灣歌謠之路-從許石、郭一男與「日本歌謠學院」談起 |author = 石計生 |date = 2015-03-06 |publisher = [[中華民國文化部|中華民國文化部]] |language = zh-tw |accessdate = 2015-09-02 |quote = 位於東京的「日本歌謠學院」，是1930-60年間影響臺灣歌謠甚鉅的民間歌手與音樂養成學校，學制分為「就學組」與「函授教育組」。臺灣曾經負笈前往留學的「就學組」包括許石與吳晉淮、而接受過「函授教育」的有郭一男等，均是臺南人。本演講將以臺灣歌謠戰後先驅作曲家許石和民間歌仔本印製家、亞洲唱片吉他手郭一男為起點，及吳晉淮、紀露霞、洪一峰和文夏等歌壇巨星的歌曲，探究近百年來臺灣歌謠之路。…… }}</ref>。完成學業後，在當時日本的東京紅風車劇座和東賓歌舞團擔任專屬[[歌手|歌手]]，直到1946年，因[[母親|母親]]病危，才自日本返台{{sfn|葉添進|2009|p=2}}。許石於1936年－1946年求學的日子非常辛苦，平日送[[牛奶|牛奶]]和[[報紙|報紙]]，寒暑假時就到[[北海道|北海道]]打工，一邊接受日本歌謠學院的音樂訓練。為了和日本內地[[同學|同學]]並駕齊驅，許石常於[[凌晨|凌晨]]進行[[馬拉松|馬拉松]]式[[慢跑|慢跑]]，鍛鍊體力，練習發聲；為了要練[[鋼琴|鋼琴]]，常練到[[手指|手指]]僵硬，平日因為沒[[錢|錢]]都在[[紙|紙]]上練彈，必須存上約一個月的[[工資|工資]]才能彈上一個[[小時|小時]]。<ref> {{Cite book | author = 石計生 | title = 許石與郭一男 淺談「日本歌謠學院」對台灣的影響 | location = 台灣 | publisher = 台灣文學館通訊 第47期 | date = 2015年6月 | pages = 63 | ISBN =  | accessdate = 2015-09-03 | url = http://journal.nmtl.gov.tw/opencms/nmtl_search/download.html?atype=B&ctype=F&volume=47&id=2015060047063065 | language = zh-tw | quote = 那時歌謠學院位於東京市麹町區九段二丁目二番地九段ビル內，在今天的千代田區。 }} </ref>

1946年，許石首次環台巡演即發表《南都之夜》{{sfn|葉添進|2009|pp=4–5}}，隨後風靡全台{{sfn|葉添進|2009|pp=2–3}}。1947年便帶著還是高中生的[[文夏|文夏]]到[[恆春|恆春]]採集鄉土歌曲，並紀錄[[陳達|陳達]]演唱《思想起》等的傳統民謠，為台灣民謠開始做紀錄及保存。此後每隔幾年，許石將所採集到的鄉土民謠重新編曲，並且央請當時社會上的文人雅士、知名作詞家（如[[許丙丁|許丙丁]]、[[陳達儒|陳達儒]]），來補全台灣鄉土民謠。同時，許石為了積極推廣和分享台灣鄉土民謡，舉辦大小十餘場的台灣鄉土音樂發表會。

1952年，許石成立了中國唱片公司，後更名為女王、大王、太王唱片公司，皆為推廣台灣鄉土民謠和歌謠而成立{{noteTag|從中國唱片公司改名為女王唱片公司，是因為唱片技術上面的紛爭，和原合夥人不和而拆夥，而由女王跟大王改為太王唱片公司，是因為財務狀況和經營的理念不同。}}{{sfn|葉添進|2009|p=3}}。然而當時版權意識不高，盜版猖獗使得正規經營的唱片公司難以生存，許石的唱片生意難以持續，造成日後生活經濟上的負擔。<ref group="參">{{cite news| url=https://sites.google.com/site/hsushih1980/|title=臺灣民謠的記事--許石先生}}</ref>許石在[[三重市|三重市]]（今新北市[[三重區|三重區]]）河邊北街90號設製片廠，出版作品包括：《[[安平追想曲|安平追想曲]]》、《港都夜雨》、《[[孤戀花|孤戀花]]》、《夜半路燈》。這些歌曲藉由許石的巡迴發表會，和歌舞會而風行，之後便灌錄[[唱片|唱片]]。是年由大王唱片舉辦「台灣鄉土民謠演唱會」，並發表《六月茉莉》、《水犁歌》、《鬧五更》、《卜卦調》、《丟丟銅》、《潮洲調》、《思想起》等……歌曲，同時透過大王唱片發行。<ref name="戀戀三重埔">{{Cite web |url = http://www.sanchong.ntpc.gov.tw/web66/_file/1258/upload/%E5%AE%9A%E6%9C%9F%E5%87%BA%E7%89%88%E5%88%8A%E7%89%A9/%E6%88%80%E6%88%80%E4%B8%89%E9%87%8D%E5%9F%94%E7%B3%BB%E5%88%97/%E6%88%80%E6%88%80%E4%B8%89%E9%87%8D%E5%9F%94%E7%B3%BB%E5%88%97%E4%BA%8C/20090629_223313.27934.pdf |title = 三重影歌星史 |author =  |date = 2009-06-29 |publisher = 三重區公所 |language = zh-tw |accessdate = 2015-09-02 |quote = 許石是五十年代起台語歌曲唱片業的萌牙者，初成立唱片廠時，資金自己籌措，許石並是戰後初期崛起的流行歌明星，曾培養很多基本歌手【註腹另兩位唱片廠老板是陳秋霖與汪思明】。 |deadurl = yes |archiveurl = https://web.archive.org/web/20150924093940/http://www.sanchong.ntpc.gov.tw/web66/_file/1258/upload/%E5%AE%9A%E6%9C%9F%E5%87%BA%E7%89%88%E5%88%8A%E7%89%A9/%E6%88%80%E6%88%80%E4%B8%89%E9%87%8D%E5%9F%94%E7%B3%BB%E5%88%97/%E6%88%80%E6%88%80%E4%B8%89%E9%87%8D%E5%9F%94%E7%B3%BB%E5%88%97%E4%BA%8C/20090629_223313.27934.pdf |archivedate = 2015-09-24 }}</ref>

1980年8月2日，因心臟病病逝台北林口[[長庚醫院|長庚醫院]]，時年60歲{{sfn|葉添進|2009|p=3}}。

=== 代表作 ===
《南都之夜》暗藏許石對新台灣期許，描寫男女情愛的《南都之夜》，當初原名為《新臺灣建設歌》，詞由薛光華所填，但彼時習慣，歌詞常由作曲家與填詞者討論後所著，其中「見如今米珠薪桂／生活竟難求」兩句，隱含音樂家對[[國民黨政府|國民黨政府]]來台後物價騰漲的溫和批判。<ref name="獨家／你一定不知道！">{{cite news |language = zh-tw |author = 楊媛婷 |coauthors =  |url = http://ent.ltn.com.tw/news/breakingnews/1683326 |title = 獨家／你一定不知道！《南都之夜》暗藏音樂家許石對新台灣期許 |publisher = 自由時報 |pages =  |date = 2016-05-02 |accessdate = 2016-06-11}}</ref>《安平追想曲》是1950年（民國39年）由許石作曲與[[陳達儒|陳達儒]]完成，從這首歌奠立了在台灣歌謠界的創作地位。另一首名曲《[[鑼聲若響|鑼聲若響]]》，據聞是在1955年（民國44年）作詞家林天來到[[正聲廣播電台|正聲廣播電台]]找[[朋友|朋友]]林禮涵（當時有名的作曲與編曲家），恰巧林不在，而遇到許石。林天來便把[[歌詞|歌詞]]給了許石，再請他譜曲。<ref name="戀戀三重埔" />

《[[安平追想曲|安平追想曲]]》發表後被改編為同名廣播劇，由[[楊麗花|楊麗花]]主演；《[[鑼聲若響|鑼聲若響]]》描繪台灣討海人與家人離別的辛酸，其中《安平追想曲》、《鑼聲若響》，被認為是許石用音樂加深[[台灣歷史|台灣歷史]]厚度的代表作。<ref name="獨家／你一定不知道！" />

許石曾到日本歌謠學院研習[[作曲|作曲]]、[[演唱|演唱]]，後因日籍[[老師|老師]]表示「你該將[[台灣|台灣]]風土寫進歌謠」，他回台後積極採集、推廣台灣本土[[民謠|民謠]]，再從事創作，曾譜寫〈[[安平追想曲|安平追想曲]]〉、〈鑼聲若響〉等……歌曲<ref name="懷念台灣歌謠交響詩 歌后紀露霞助陣">{{cite news |language = zh-tw |author = 楊媛婷 |coauthors =  |url = http://news.ltn.com.tw/news/supplement/paper/884023 |title = 懷念台灣歌謠交響詩 歌后紀露霞助陣 |publisher = 自由時報 |pages =  |date = 2015-05-27 |accessdate = 2015-09-01}}</ref>。許石將大部分採集到的民謠重新編排，並邀請適合各個台灣鄉土民謠味道的歌手，安排演唱並製作成唱片，總計有39首，列表如下。採集的台灣民謠範圍涵蓋[[臺灣日治時代|臺灣日治時代]]的台灣[[民謡|民謡]]、[[宜蘭|宜蘭]]民謠、[[農家|農家]]民謠、北部的民謠，[[都馬調|都馬調]]、[[車鼓|車鼓]]民謠、[[恆春|恆春]]民謠、[[客家|客家]]民謠、[[山地|山地]]民謠、[[枋寮|枋寮]]民謠、[[月旦調|月旦調]]等。<ref name="a" group="參"/>。

[[台灣大學|台灣大學]]音樂學研究所兼任助理教授陳峙維表示：「學成返台後許石就展開全島探訪，記錄這塊土地上各個族群的歌曲，[[福佬|福佬]]、[[客家|客家]]、[[台灣原住民|原住民]]都是他的採集對象，這比[[許常惠|許常惠]]與[[史惟亮|史惟亮]]發起的『[[民歌|民歌]]採集運動』早了近二十年。1964年，許、史兩位學者才要開始對全台的[[民俗音樂|民俗音樂]]進行調查時，同年10月10日許石在台北國際學舍已舉行音樂會，發表作品《台灣鄉土交響曲》，這是他是從事鄉土音樂工作多年心血的結晶。全曲漢洋樂器合璧，他還特別使用了台灣特有的大廣弦和長頸月琴。」<ref name="陳峙維">{{cite news |language = zh-tw |author = 陳峙維 |coauthors =  |url = http://www.appledaily.com.tw/realtimenews/article/new/20150829/679908/ |title = 被中國管理員綁架的「維基百科」 |publisher = 蘋果日報 |pages =  |date = 2015-08-29 |accessdate = 2015-09-03}}</ref>

[[成功大學|成功大學]]台灣文學系教授呂興昌表示，許石對[[台灣音樂|台灣音樂]]貢獻卓鉅，那時流行直接翻唱日本或外國歌曲，不然就是國語歌曲，但許石和他的姻親、音樂家楊三郎都始終堅持台灣本土創作。呂興昌指出，許石是最典型、最早的尋根作曲家，「他不僅錄音、採譜，更邀請戲劇學家呂訴上、文學家許丙丁等填詞，並灌錄唱片，復活台灣民謠！」現今《[[草螟弄鷄公|草螟弄鷄公]]》、《思想起》、《牛犁歌》、《卜卦調》等歌謠演唱，即都採許石當初於大王唱片時代所發行的版本，不僅如此，許石還將採集到的本土民謠編為《台灣鄉土交響曲》，這首漢洋合璧樂章的珍貴原譜，也在許石的遺物中發現，但因損害嚴重，仍待修復。<ref name="獨家／你一定不知道！" />

== 已發表作品 ==
許石在1959年、1969年、1979年的10、20、30週年的作品發表會與作曲集第一冊中，列出61首創作，未發表的散譜手稿有近40首<ref group="參">{{cite web| url=http://www.mcat.org.tw/member1_detail.php?name=%E9%84%AD%E6%B7%91%E8%8F%AF| title=許石作品| publisher=社團法人台灣音樂著作權人聯合總會MCAT| accessdate=2015年8月29日}}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

曾被[[楊三郎|楊三郎]]欽定為多首作品演唱人的[[紀露霞|紀露霞]]憶述，許石跟楊三郎的音樂，每段[[樂章|樂章]]及[[歌詞|歌詞]]都如[[電影|電影]]般具有畫面。紀露霞舉許石〈安平追想曲〉為例：「以前的作曲／詞家不似現今，3段旋律都會搭配不同歌詞，音樂與詞意相互搭配下，宛若從耳膜仔細『欣賞』一齣影片」，她說許石發表〈安〉一曲後不久，電影界即將該曲改編為電影，由[[楊麗花|楊麗花]]主演。<ref name="懷念台灣歌謠交響詩 歌后紀露霞助陣" />紀露霞還記得她幫《鑼聲若響》的大王唱片錄[[唱片|唱片]]時，就是在[[戲院|戲院]]裡[[錄音|錄音]]。<ref>{{zh-tw}} [http://www.tri.org.tw/per/49/49-4.pdf 戰後台語紅歌星紀露霞]</ref>

[[正修科技大學|正修科技大學]]通識中心講師蔡添進曾探究《南都之夜》傳唱及[[旋律|旋律]]來源，他提到：「2008年台灣最熱門的電影《[[海角七號|海角七號]]》裡，茂伯第一次參加[[樂團|樂團]]時，開口唱出了『我愛我的妹妹啊』，這首由許石作曲、鄭志峰作詞的歌曲《南都之夜》，被稱為是『[[台灣光復|台灣光復]]以後第一首最流行的歌曲』，此歌有時又被冠上〈台灣小調〉的歌名。在不同[[時代|時代]]、不同[[地區|地區]]被填上各種[[歌詞|歌詞]]來傳唱，而其旋律來源又被認為來自日本歌曲。」<ref> {{Cite book | author = 蔡添進 | title = 許石〈南都之夜〉傳唱及旋律來源探究 | location = [[高雄市|高雄市]] | publisher = 高雄文化研究 2009 年年刊 | date = 2009年8月 | pages = 1-52 | ISBN =  | accessdate = 2015-09-03 | url = http://cgeweb.csu.edu.tw/full_teacher/refereed/data/0361/201076114534.pdf | language = zh-tw | quote = 因研究蕭泰然的〈1947 序曲〉時，在所參看的兩本碩士論文中，陳宏略提到「〈我愛台灣〉是許石的創作歌謠」，而莊婉琪在之後的論文中提到同一首歌時，又說成此曲是「〈南都之夜〉為許石的作品，後來此曲重新填詞，易名為〈台灣小調〉」。同一首歌，卻有〈我愛台灣〉〈南都之夜〉〈台灣小調〉三種歌名，引發了研究動機，想對此歌曲有完整的認識。而在資料找尋中，又發現在網路上有一些說法，「這首閩南歌曲的來源有兩個說法，有一說是許石留學日本時聽到一首日本歌〈蘋果之歌〉而改編，也有說是他看過歌仔戲後，學起此首曲調再加以改編」。以上這些對此歌曲的說法，不禁讓人對歌名及此歌曲之來源產生好奇，而有此次之研究。 }}{{dead link|date=十月 2017 |bot=InternetArchiveBot }} </ref>

== 懷舊音樂會 ==
1991年，因參與楊三郎紀念[[音樂會|音樂會]]演唱，[[上揚唱片|上揚唱片]]透過紀利男<ref>{{cite news |language = zh-tw |author = 吳冠瑾 |coauthors =  |url = http://www.chinatimes.com/newspapers/20141222000860-260308 |title = 紀利男爆 豬哥亮跑路投靠求作曲 |publisher = 中時電子報 |pages =  |date = 2014-12-22 |accessdate = 2015-09-02}}</ref>與紀露霞接洽錄製台語老歌的唱片[[計劃|計劃]]。且因紀利男對[[音樂|音樂]]非常有興趣，紀露霞便介紹紀利男隨楊三郎學[[小喇叭|小喇叭]]，紀露霞透過紀利男牽線，就答應為上揚唱片錄製[[唱片|唱片]]。<ref>{{cite news |language = zh-tw |author = 郭麗娟 |coauthors =  |url = http://www.newtaiwan.com.tw/bulletinview.jsp?period=346&bulletinid=10393 |title = 台語歌后—紀露霞　為你再唱黃昏嶺 |publisher = 新台灣新聞週刊 |pages =  |date = 2002-11-08 |accessdate = 2015-09-02 }}{{Dead link|date=2018年7月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

2015年5月29日，為了回溫當時許石與楊三郎一同合唱的登場畫面<ref>{{Cite web |url = http://event.moc.gov.tw/sp.asp?xdurl=ccEvent2013/ccEvent_cp.asp&cuItem=2132158&ctNode=676&mp=1 |title = 許石與楊三郎-懷念臺灣歌謠交響詩 |author = 臺北市立交響樂團 |date = 2015-05-06 |publisher = 中華民國[[文化部|文化部]] |language = zh-tw |accessdate = 2015-09-02 |quote = 許石與楊三郎之經典台灣老歌再現。 }}</ref>，台北市立交響樂團首度舉辦純[[台語|台語]]音樂會<ref>{{Cite web |url = http://www.artsticket.com.tw/CKSCC2005/Product/Product00/ProductsDetailsPage.aspx?ProductId=oK4bYlG1GfzGUfkhHcyeHw |title = 【TSO】許石與楊三郎懷念臺灣歌謠交響詩 Taiwanese Symphonic Poem Memorial Concert for Shih |author =  |date =  |publisher = 兩廳院售票 |language = zh-tw |accessdate = 2015-09-03 |quote = Taiwanese Folk Songs by Shih Hsu & San-Lang Yang }}</ref>，並邀請[[紀露霞|紀露霞]]<ref group="參">{{cite news| url=https://www.youtube.com/watch?v=NPpB4wJzpso|title=寶島歌后紀露霞 「封麥」演唱會登場－民視新聞}}</ref>、[[劉福助|劉福助]]、林秀珠及[[李靜美|李靜美]]演出，追溯50年代經典台語作品。劉福助10多歲就跟著許石學習唱歌，他想到恩師逝世35年，劉感慨表示：「應該很早之前就要舉辦了，時間隔了那麼久，實在感觸良多。」<ref group="參">{{cite news| url=http://ent.ltn.com.tw/news/breakingnews/1332858 |title=「許石與楊三郎音樂會彩排」紀露霞抱病中氣足 劉福助開口就哽咽| publisher=[[自由時報|自由時報]]新聞網| date=2015年5月29日| accessdate=2015年8月29日}}</ref>。
