'''許靖'''（{{bd||147|222年|catIdx=X許}}），[[表字|字]]'''文休'''，[[豫州|豫州]][[汝南郡|汝南]][[平輿縣|平輿]]人，[[蜀漢|蜀漢]]政治家。三國時代人物評論家[[許劭|許劭]]的堂兄、許瑒的堂弟、[[許欽|許欽]]之父、[[許游|許游]]祖父。

== 生平 ==
許靖自年輕時就與堂弟許劭同為評論人物的名士，但與許劭關係惡劣<ref>《[[太平御覽|太平御覽]]》卷四九六引曹丕《典論》曰：「汝南許劭與族兄靖俱避地江東，保吳郡。爭論於太守許貢座，至於手足相及。」</ref>，所以當許劭擔任[[汝南郡|汝南]]'''[[功曹|功曹]]'''時（'''功曹'''，郡的人事長官），故意不用許靖<ref>《三國志·卷三八·許靖傳》載，靖「少與從弟劭俱知名，並有人倫臧否之稱，而私情不脅。劭為郡功曹，排擯靖不得齒敘，以馬磨自給」。</ref>。之後[[劉翊|劉翊]]來了做[[太守|太守]]，才推舉許靖為[[孝廉|孝廉]]，任[[尚書_(官職)|尚書郎]]。

在[[董卓|董卓]]掌權時，[[周毖|周毖]]和許靖負責吏部，任命了[[孔伷|孔伷]]、韓馥、張邈等。許靖被任命為[[巴郡|巴郡]]太守，但沒有上任。後來韓馥等攻打[[董卓|董卓]]，董卓於是殺了周毖，許靖擔心被殺，遂轉徙各地，先後投靠[[豫州|豫州]][[刺史|刺史]]孔伷、[[揚州_(古代)|揚州]]（與今日之揚州市無關）刺史[[陳禕|陳禕]]、[[吳郡|吳郡]][[都尉|都尉]][[許貢|許貢]]、[[會稽郡|會稽]][[太守|太守]][[王朗|王朗]]；王朗被[[孫策|孫策]]滅亡後，又逃往[[交州|交州]]避難。後來受到[[益州|益州]][[州牧|牧]][[劉璋|劉璋]]邀請，受任為[[巴郡|巴郡]]、[[廣漢郡|廣漢]]太守，在原[[蜀郡|蜀郡]]太守[[王商_(文表)|王商]]過世之後，又轉任蜀郡太守。

當[[劉備|劉備]]包圍[[成都|成都]]，許靖想要棄城逃跑但失風被捕。劉備輕視許靖無責任感而棄用他；但參謀[[法正|法正]]認為許靖雖然無才且沒有節操，但卻是名滿天下的名士，如果不予任用會對劉備聲望造成影響，以後人才召募以及政治號召上將多有困難。劉備遂採納法正意見任用許靖為[[左將軍|左將軍]][[長史|長史]]，給許靖沒有實權的虛職。

劉備自稱[[漢中王|漢中王]]時，被任命為[[太傅|太傅]]，章武元年（221年）劉備稱帝時，更升任為[[司徒|司徒]]（[[三公|三公]]之一），僅次於[[丞相|丞相]][[諸葛亮|諸葛亮]]但並無實權。六月，為劉備兩個庶子[[劉永_(三國)|劉永]]和[[劉理|劉理]]做封王詔命<ref>見《三國志·蜀書·二主妃子傳》</ref>。許靖自知自己的地位只是幌子所以也不強求功名，過著樸素節儉的生活。次年（222年）逝世，年過七十。

== 亲属 ==
*子：许钦，许靖长子，在许靖死前即去世。
*孙：许游，许钦之子，景耀（258年—263年）年间任尚书。
*堂兄：許瑒，官至陈国相，与[[豫州|豫州]][[刺史|刺史]][[孔伷|孔伷]]合兵反[[董卓|董卓]]。
*堂弟：[[許劭|許劭]]，字子将。
另有一哥哥，名不詳，有外孫[[陳祗|陳祗]]，因少孤而被許靖收養，後為[[劉禪|劉禪]]的寵臣。<ref>《三國志·[[董允|董允]]傳》：祗字奉宗，汝南人，許靖兄之外孫也。少孤，長於靖家。</ref>

== 评价 ==
*[[袁徽|袁徽]]：“许文休英才伟士，智略足以计事。自流宕已来，与群士相随，每有患急，常先人后己，与九族中外同其饥寒。其纪纲同类，仁恕恻隐，皆有效事，不能复一二陈之耳。”（《三国志·蜀书·许麋孙简伊秦传第八》）
*[[宋忠_(三國)|宋忠]]：“文休倜傥瑰玮，有当世之具，足下当以为指南。”（《三国志·蜀书·许麋孙简伊秦传第八》）
*[[诸葛亮|诸葛亮]]：“靖人望，不可失也，借其名以竦动宇内。”
*[[法正|法正]]：“天下有获虚誉而无其实者，许靖是也。”（《三国志·蜀书·庞统法正传第七》）
*[[蒋济|蒋济]]：“许文休者，大较廊庙器也，而子将贬之。若实不贵之，是不明也；诚令知之，盖善人也。”（《三国志·许靖传》注引《万机论》）
*[[杨戏|杨戏]]：“司徒清风，是咨是臧，识爱人伦，孔音锵锵。”（《三国志·蜀书·邓张宗杨传第十五》）
*[[陈寿|陈寿]]：“许靖夙有名誉，既以笃厚为称，又以人物为意，虽行事举动，未悉允当，蒋济以为‘大较廊庙器’也。”（《三国志·蜀书·许麋孙简伊秦传第八》）
*[[孙盛|孙盛]]：“夫礼贤崇德，为邦之要道，封墓式闾，先王之令轨，故必以体行英邈，高义盖世，然后可以延视四海，振服群黎。苟非其人，道不虚行。靖处室则友于不穆，出身则受位非所，语信则夷险易心，论识则殆为衅首，安在其可宠先而有以感致者乎？若乃浮虚是崇，偷薄斯荣，则秉直仗义之士，将何以礼之？正务眩惑之术，违贵尚之风，譬之[[郭隗|郭隗]]，非其伦矣。”
*[[裴松之|裴松之]]：“文休名声夙著，天下谓之英伟，虽末年有瑕，而事不彰彻，若不加礼，何以释远近之惑乎！”“许靖羁客会稽，闾阎之士，孙策之来，於靖何为？而乃泛万里之海，入疫疠之乡，致使尊弱涂炭，百罹备经，可谓自贻矣。谋臣若斯，难以言智。孰若安时处顺，端拱吴、越，与张昭、张纮之俦同保元吉者哉？”（《三国志·许靖传》）
*[[叶适|叶适]]：“许靖以汝南胜士著名，既尝预汉谋议，及其流播，无所茍容其身者，则臣役之晚岁，遂有翻成见薄之事。然则如荀彧之出处，殆未可以轻责也。法正薄夫，乃知靖为当敬。发悟刘备用贤之机，诸葛亮之亲厚，正有以哉。”
*[[杨维桢|杨维桢]]：“汝南许文休，丧乱一驽士。敢当诸葛拜，合受玄德鄙。 士论推指南，无乃失臧否。乃知群公曹，排摈有公是。”（《览古四十二首·其二十一》）
