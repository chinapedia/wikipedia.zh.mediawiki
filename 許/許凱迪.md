'''許凱迪'''（{{lang-en|Kai Ti Hsu}}，{{bd|1984年||}}）是臺灣[[插畫家|插畫家]]。

== 生平 ==
曾於2015年參加[[苗栗縣政府國際文化觀光局|苗栗縣政府國際文化觀光局]]舉辦的「苗栗意象•創意海報設計競賽」獲得金獎<ref>{{Cite web |url = http://n.yam.com/greatnews/arts/20150518/20150518040500.html |title = 苗栗意象•創意海報設計競賽成果揭曉! |author = 劉鳳瑩 |date = 2015-05-18 |publisher = 大成報 |language = zh-hant |accessdate = 2015-05-31}}</ref>。

== 爭議 ==
2015年許凱迪在網路上發表作品，被義大利的艾米朗若·龐茲（Emiliano Ponzi）藝術工作者於5月13日在臉書上指出與他[[紐約時報|紐約時報]]創作的作品雷同，而他表示的確是抄襲<ref>{{Cite web |url = http://udn.com/news/story/1/906160 |title = 諷刺台灣畫作者涉抄襲 臉書道歉挨轟菜英文! |date = 2015-05-16 |publisher = 聯合報 |language = zh-hant |accessdate = 2015-05-31}}</ref><ref>{{Cite web |url = http://udn.com/news/story/1/904365 |title = 17張諷刺台灣畫作者 驚傳抄襲被噓爆 |quote= I never write posts that goes behind uploading my personal illustrations but a friend of mine bring to my attention a really bad example of what a pro shouldn't do, never. I'm not angry but disappointed. Becoming a great illustrators means being able to develop your own point of view over the world and design your own version of it. But clearly mr Kai Ti Hsu thinks different. On the left side my image did for NY Times and on the right side mr Hsu one...I understand inspiration but this is another story. |date = 2015-05-15 |publisher = 聯合報 |language = zh-hant |accessdate = 2015-05-31}}</ref>。也有與英國插畫家約翰·霍爾克羅夫特（John Holcroft）、日本設計師[[福田繁雄|福田繁雄]]的作品相同<ref>{{Cite web |url = http://www.appledaily.com.tw/realtimenews/article/new/20150515/610767/ |title = 【更新】我創作變你遺照？　被抄襲畫家：「毛毛的」 |author = 蔡明倫 |date = 2015-05-15 |publisher = 蘋果日報 |language = zh-hant |accessdate = 2015-05-31}}</ref>。並被媒體發現意圖諷刺[[阿帕契打卡案|阿帕契打卡案]]的作品，像是另一名義大利插畫家達維德·博納齊（Davide Bonazzi）的專欄插圖[[華爾街日報|華爾街日報]]「Business Schools Are Expanding in Africa」及[[科学美国人|科学美国人]]「Diagnosing Alzheimer's」<ref>{{Cite web |url = http://ent.appledaily.com.tw/enews/article/entertainment/20150523/36567351 |title = 旁白哥抓漏 瑪丹娜泰勒絲 捲入盜版風波 |date = 2015-05-23 |publisher = 蘋果日報 |language = zh-hant |accessdate = 2015-06-01}}</ref>。
