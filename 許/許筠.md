'''許筠'''{{BD|1569年||1618年|}}，為[[李氏朝鮮|李氏朝鮮]]中期的文臣和政治人、詩人、作家、小說家。小說《[[洪吉童傳|洪吉童傳]]》的著者。

== 簡介 ==
許筠，出生[[江原道|江原道]][[江陵市|江陵市]]，[[本貫|本貫]][[陽川許氏|陽川許氏]]。字'''端甫'''，號'''蛟山'''、惺所、'''惺叟'''、又號'''白月居士'''。

== 生平 ==
許筠於1569年出生，父親[[許瞱|許瞱]]是著名文人[[徐敬德|徐敬德]]的弟子，在文壇上頗負盛名，官至副提學，父親許曄於許筠12歲時逝世。姊姊[[許楚姬|許楚姬]]（號[[許蘭雪軒|蘭雪軒]]）亦有文名。

許筠的老師為當時名傾一時的「三唐詩人」之一的[[李達|李達]]，因李達出身庶子，在當時的社會環境下頗受壓抑不得志，對現實社會深懷不滿。而許筠在人格及學問等各方面上，受到李達很大的影響，也造成他日後在政治上走向反抗的遠因。

許筠於1589年 (宣祖22年) 考上[[生員|生員]]，1594年庭試文科及第。之後擔任過檢閱、[[世子侍講院|世子侍講院]]說書。1597年文科重試狀元，任[[黃海道|黃海道]]道使，但後因遭妓女迷惑被彈劾罷職。而後歷任[[春秋館|春秋館]]記注官、[[刑曹|刑曹]]正郎等職。1602年又任司藝、司僕寺正、典籍、遂安郡守等官職。1606年許筠擔任遠接使從事官，迎接[[明朝|明朝]][[朱之蕃|朱之蕃]]，以寫作文章聲名大噪。1610年以陳奏副使身分出使明朝，在[[中國|中國]]受洗成為韓國最早的[[天主教|天主教]]徒。同年擔任試官，但因親族參榜遭彈劾罷職, 而後許筠來到[[泰仁|泰仁]]從事寫作。1613年[[癸丑獄事|癸丑獄事]]，許筠的好友[[朴應犀|朴應犀]]等被處死，為了安全許筠依附權臣[[李爾瞻|李爾瞻]]。之後擔任禮曹參、戶曹參、[[承文院|承文院]]副提調。1617年主張廢母論，得到[[朝鮮國王|朝鮮國王]]的信任，同年晉升左參贊。然而許筠之後在[[漢城|漢城]]和[[金闓|金闓]]、[[辛光業|辛光業]]等人，製造邊防警訊及散播謠言進行鼓動，以便亂中舉事，推翻[[光海君|光海君]]，但後因事機敗露而遭處死。

== 作品 ==
許筠著有小說《[[洪吉童傳|洪吉童傳]]》，詩歌評論集《蛟山詩話》，地方風俗飲食誌《屠門大嚼》，筆記《識小錄》、《閑情錄》，詩文集《惺所覆瓿稿》。此外，亦編有歷代詩選《國朝詩刪》。

== 文學地位 ==
許筠的小說《[[洪吉童傳|洪吉童傳]]》是他的代表作品。《洪吉童傳》描述一個武藝高超，具有高強法術的[[庶出子女|庶子]][[洪吉童|洪吉童]]，組織活貧黨，劫富濟貧和官府作對。後來離開朝鮮，在海島上建立自己的國家。

朝鮮王朝中期，暴政、外敵入侵逐趨劇烈，農民遭受的負擔的壓迫亦加深重。政治上接連的士禍、黨爭，各種社會矛盾不斷激化。許筠對於這樣的社會現實深感不滿，《洪吉童傳》可以說是描寫在[[壬辰倭亂|壬辰倭亂]]後，社會的階層問題、政治的腐敗的社會寫實小說。《洪吉童傳》的出現，對於當時平民階級的抬頭，有很大的刺激與啟示作用，從而開創了韓國文學史上寫實小說的起端。雖然當時的文人多對許筠的作品持否定的態度，然而許筠的作品中的進步傾向及社會意義，卻奠定了許筠在韓國文學史上重要的地位。
