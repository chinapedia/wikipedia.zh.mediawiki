'''昆明地铁'''，是服务于[[中国|中国]][[云南省|云南省]][[昆明市|昆明市]]及其周边地区的[[城市轨道交通系统|城市轨道交通系统]]，首条路线[[昆明轨道交通6号线|6号线]]于2012年6月28日开通营运。1993年昆明首次提出轨道交通概念，2009年，昆明地铁项目获国务院批准，2010年5月1日，昆明地铁1、2号线首期工程正式开工建设。6号线的开通营运，使得昆明成为继[[重庆|重庆]]、[[成都|成都]]、[[西安|西安]]之后，[[中国西部|中国西部]]地区第四座开通轨道交通路线的城市。2013年5月20日，1号线南段开通试营运。2014年4月30日，昆明地铁1、2号线首期工程全线贯通试营运。至此，昆明打通城市南北向的地铁主干线，全城地铁营运总里程达60公里，真正进入“地铁时代”。昆明地铁的营运商为昆明地铁营运有限公司。2012年至2016年间，地铁6号线与其他路線完全不連接，并于2016年3月5日起暂停运营。2017年8月29日上午，昆明轨道交通3号线、6号线一期正式开通载客试运营。

== 历史 ==
{{NoteUTC+8}}
*1993年，在昆明与[[苏黎世|苏黎世]]共同编制完成的《城市发展与公共交通总体规划》中，首次提到了轨道交通。
*2004年，昆明市政府决定启动轨道交通项目。
*2005年，昆明市完成了初步的地铁规划，规划有五条路线。
*2006年，轨道交通规划经过省建设厅审批。
*2007年，轨道交通规划修改，从五条路线增加为六条，昆明市政府“轨道交通建设小组”正式成立。
*2008年3月12日，昆明轨道交通规划正式报批[[国家发改委|国家发改委]]和[[建设部|建设部]]。
*2009年6月8日，[[国务院|国务院]]批准昆明轨道交通建设规划。<ref name=r1>[http://news.qq.com/a/20090629/000245.htm 昆明地铁项目获批准 计划3年后建成(组图)]</ref>

*2010年5月1日，昆明地铁1、2号线首期工程正式开工建设。<ref name=r2>[http://news.sina.com.cn/c/2010-05-02/165120194281.shtml 昆明地铁1号2号线首期工程开工]</ref>
*2010年8月1日，昆明地铁6号线一期正式开工建设。
*2010年8月9日，昆明地铁3号线试验段开工建设。

=== 1号线 ===
[[File:Kunming_Metro_in_Kunluo_road.jpg|thumb]]
{{main|昆明地铁1号线}}
标识色：{{Fontcolor|{{昆明地铁颜色|1}}|█}} 红色<ref>[http://society.yunnan.cn/html/2011-01/16/content_1470487.htm “六彩”地铁线 绘出昆明特色] {{webarchive|url=https://web.archive.org/web/20130923061758/http://society.yunnan.cn/html/2011-01/16/content_1470487.htm |date=2013-09-23 }}</ref>

昆明地铁1号线为连接昆明主城至[[呈贡|呈贡新区]]的南北向骨干线，目前和[[昆明地鐵2號線|2號線]][[直通運行|直通運行]]，待其中一條線路二期建成後，兩線將各自獨立運行。路线起于二环北路，终点位于呈贡南部大学城南站，全长约41.4千米，设26站。1号线南段最早于2009年中在昆洛路段上进行了试验段建设，一期工程（环城南路站-大学城南站）同2号线一期共同组成了昆明地铁首期工程，在2010年5月1日全面开工建设。1号线南段（晓东村站-大学城南站）于2012年11月5日开始空载试运行<ref>{{Cite web |title=2013年2月6日地铁首期南段载客开行 直达大学城 |url=http://www.km.gov.cn/structure/sylm/kmxwxx_218124_1.htm |date=2012年12月29日 |accessdate=2012年12月29日 |publisher=昆明市人民政府 |deadurl=yes |archiveurl=https://web.archive.org/web/20140517115256/http://www.km.gov.cn/structure/sylm/kmxwxx_218124_1.htm |archivedate=2014年5月17日 }}</ref>，2013年5月20日开通载客试营运。<ref name="r4" />1号线支线于2016年12月26日开通运营，1号线支线起点为春融街站，终点为昆明南火车站，全长5.3公里，设市级行政中心站、宜和路站、白龙潭站、昆明南火车站共4站4区间，全线均为地下线。<ref name=r5>{{Cite web |title=地铁1号线支线开通试运营 |url=http://www.kmgdgs.com/content/2016-12/27/content_4468511.htm |date=2016年12月27日 |accessdate=2016年1月11日 |publisher=昆明轨道交通有限公司 |deadurl=yes |archiveurl=https://web.archive.org/web/20161231074849/http://www.kmgdgs.com/content/2016-12/27/content_4468511.htm |archivedate=2016年12月31日 |df= }}</ref>

=== 2号线 ===
{{main|昆明地铁2号线}}
标识色：{{Fontcolor|{{昆明地铁颜色|2}}|█}} 蓝色

2号线为昆明主城内南北方向的骨干线，於2014年4月30日正式營運，目前和[[昆明地鐵1號線|1號線]][[直通運行|直通運行]]，待其中一條線路二期建成後，兩線將各自獨立運行。路线起点位于主城北部汽车站，终点位于主城南部会展中心站，全长约22.4公里，拟设22站。2号线一期（北部汽车站-环城南路站）与1号线一期（环城南路站-大学城南站）一起于2010年5月1日开工建设，建成初期将贯通营运，待二期工程建成后再拆分营运。二期工程起于环城南路站，穿过[[昆明火车站|昆明火车站]]后，到达位于[[滇池|滇池]]边上的昆明新会展中心，在环湖东路向东拐入宝丰村停车场。2014年4月30日，昆明地铁2号线首期北段通车试营运。
{{昆明地鐵2號線|nocat}}

=== 3号线 ===
{{main|昆明地铁3号线}}
标识色：{{Fontcolor|{{昆明地铁颜色|3}}|█}} 粉红色

3号线为昆明主城内东西方向的骨干线，路线经过昆明大部分最古老的商业街区。路线起点位于主城西部石咀站，终点位于主城东部汽车站，沿春雨路向北至人民西路，沿人民西路向东，至小西门转向东南，沿东风西路、南屏街、东风东路向东于大树营穿过东二环进入东连接线，止于规划的东部客运站附近东绕城连接线南侧，路线经过春雨路、人民西路、东风东路、南屏街、东风东路、人民东路延长线、金马寺村、太平路、人民东路延长线。一期路线全长19.16公里，其中高架线1.17公里，地下线和过渡段长17.99公里。共设车站17座，全地下车站，平均站间距1.15公里。3号线试验段在2010年8月开工建设，2011年7月起，3号线各站点分批陆续开始围挡施工。二期工程将延伸到[[西山_(昆明)|西山森林公园]]。2017年1月21日，昆明地铁3号线铺轨工程大树营至金马寺区间最后200m整体道床浇筑完毕，标志着昆明地铁3号线全线实现轨通。2017年8月29日通车运营。
{{昆明地鐵3號線|nocat}}

=== 6号线（机场线）===
[[File:Kunming_Metro_L6.jpg|thumb]]
{{main|昆明地铁6号线}}
标识色：{{Fontcolor|{{昆明地铁颜色|6}}|█}} 青色

6号线（机场线）是连接昆明主城和航空港的辅助线，定位于一条机场专线，该线连接昆明市中心和[[昆明长水国际机场|昆明长水国际机场]]。6号线一期工程共4站3区间（[[东部汽车站站|东部汽车站站]]至[[机场中心站|机场中心站]]），路线全长约18.018km，于2010年8月1日全面开工建设，2012年6月28日开通观光试运行。首末班时间为每天9:00至19:10，发车间隔时间约25分钟，单程运行时间 23分钟，单程票价5元。曾于2016年3月5日至2017年8月28日間暂停运营并开展提质调试工作，現已重新投入运营。重新开通运营后，列车运营时间为早上6点20分至晚上22点20分，单程运行时间16分30秒，发车间隔时间缩短至15分钟。6号线可于[[东部汽车站站|东部汽车站]]同3号线进行换乘。

二期工程为[[塘子巷站|塘子巷站]]至东部汽车客运站站，路线全长约6.99km。路线规划全长约25.67km，设车站6座。按照规划，乘客可在[[塘子巷站|塘子巷站]]换乘2号线，在[[东部汽车站站|东部汽车站站]]换乘3号线。

==== 4号线 ====
4号线为昆明主城内部及主城与呈贡新区之间的补充路线。该路线走向与昆明市内[[滇越铁路|滇越铁路]]的城市米轨路线大致相同，全长约42.9千米，拟设22个车站。

==== 5号线 ====
5号线是主城东北-西南方向的辅助线。起于东北部的世博园，经[[翠湖_(昆明)|翠湖]]一直到达西南部的滇池旅游度假区，连接了昆明市内各主要旅游景点，全长约21公里，拟设14个车站。

==== 9号线 ====
9号线串联城市东部环滇新区：[[晋宁|晋宁]]、[[呈贡|呈贡]]、经开区、空港区，连接了[[昆明南站|昆明南站]]及[[昆明长水国际机场|长水机场]]。路线起点位于李其站，终点位于三槐村站。全长约53公里，拟设站22座。

====嵩明线-安宁线====
嵩明线-安宁线是市域轨道轨道交通东西线的的组成部分，是连接安宁——太平——主城——空港经济区——嵩明的线路。

嵩明线全长约59.07km，共设车站15座。 平均站间距4.2km，最大站间距 9.15km，最小站间距2.25km。全 线以地下和高架线为主，其中地 下线长约29.37km，高架线长约 28.85km，地面线约0.85km。嵩明线起于巫家坝，止于嵩明，于巫家坝中心设巫家坝中心站，出巫家坝中心站后沿巫家坝规划绿轴向北敷设，穿成昆铁路，上跨东三环至 东部汽车客运站，后平行于6号线 和机场高速前行至金马村，上跨贵昆铁路后沿新320国道走廊前行至机场北，引入长水综合交通枢纽。出长水枢纽后线路穿小哨组 团进入嵩昆大道走廊，沿嵩昆大道中央15m绿化带前行至凤溪寺南侧后出嵩昆大道，沿嵩明县黄龙大街引入嵩明县城。<ref>{{cite web|title=昆明轨道交通嵩明线试验段 顺利完成钻孔灌注桩首件验收|url=http://www.kmgdgs.com/image/site402/20171129/507b9d1b02ce1b885df204.pdf|access-date=2018-02-22|archive-url=https://web.archive.org/web/20180222225506/http://www.kmgdgs.com/image/site402/20171129/507b9d1b02ce1b885df204.pdf|archive-date=2018-02-22|dead-url=yes}}</ref> <ref>{{cite web|title=昆明市城市轨道交通建设规划（2017~2023）环境影响报告书|url=http://www.ynepb.gov.cn/zmhd/qygzcygscl/201703/W020170303534280662017.pdf}}</ref>

安宁线线路全长23.6km，共设车站7座。其中地下线8.6km，高架线13.5km，地面线1.5km。

=== 规划路线 ===
{{YN Future infrastructure}}

近期共规划有6条路线（即1~6号线）。远景规划，昆明地铁共有15条路线，目标总长达到602公里。其中：地铁线网将在近期规划的6条线162.6公里基础上新增3条路线（141.4公里），总长将达到304公里；同时还将有连接[[嵩明|嵩明]]、[[宜良|宜良]]、[[晋城|晋城]]、[[安宁|安宁]]、[[富民|富民]]等城市总长达298公里的6条都市快线。<ref>{{Cite web |title=昆明优化地铁规划 6条线总长拟延伸25公里 |url=http://news.yninfo.com/yn/kmxw/201106/t20110629_1684982.htm |date=2011年6月29日 }}</ref>

==== 7号线 ====
7号线为主城西南部片区的加密线，路线起点位于上沙河，终点位于黄土坡村。全长约28公里，拟设站24座。

==== 8号线 ====
8号线为服务主城东部片区南北向交通出行，带动[[巫家坝国际机场|巫家坝副中心]]发展。路线起点位于广南卫站，终点位于小河嘴站。全长约24公里，拟设站16座。

==== 都市快线 ====
[[File:Kunming_Metro_Line_6_Ticket_Vending_Machine.JPG|thumb]]

昆明地铁远景线网规划有6条都市快线，都市快线为连接昆明主城到周边地区的[[市域铁路|市域铁路]]，以东西快线（安宁—嵩明）和南北快线（主城—昆阳）形成的“十字线”为主骨架，预留连接中心城至富民、宜良、海口和澄江4条路线。全长约300公里。<ref>{{Cite web |url=http://www.kmghj.gov.cn/articledetails.aspx?id=2037&cd=cid-110 |title=城市道路及轨道交通建设 |publisher=昆明市规划局 |accessdate=2013-04-25 |deadurl=yes |archiveurl=https://web.archive.org/web/20140517120222/http://www.kmghj.gov.cn/articledetails.aspx?id=2037&cd=cid-110 |archivedate=2014年5月17日 |df= }}</ref><ref>{{Cite web |url=http://yn.yunnan.cn/html/2011-10/22/content_1875522.htm |title=昆明东西向都市快线年内开工 设计时速最高不超160公里 |publisher=云南网 |accessdate=2013-04-25 |deadurl=yes |archiveurl=https://web.archive.org/web/20140517132459/http://yn.yunnan.cn/html/2011-10/22/content_1875522.htm |archivedate=2014年5月17日 |df= }}</ref>

=== 售票方式 ===
昆明地铁车站站厅层均设有无人自动售票机，多功能无人交通一卡通服务系统以及人工服务问讯窗口，为乘客提供票务服务。

==== 自动售票机 ====
仅支持纸币购票（5元，10元，20元，50元）。

==== 多功能无人交通一卡通服务系统 ====
支持站点信息查询与交通一卡通智慧卡充值，充值服务仅支持纸币（50元，100元）。

==== 人工问询服务窗口 ====
提供站点信息，列车时刻，换乘信息等问讯服务，提供交通一卡通智慧通服务，如开卡（押金20元），充值（任意金额，仅支持纸币）。

==== 手机移动支付APP ====
自2019年3月始，昆明地铁开始支持手机APP进站乘车，乘客可使用智慧通行App绑定银联卡后可生成二维码刷码进站乘车。4月昆明地铁开始支持银联闪付进站乘车，乘客可使用Apple Pay，HUAWEI Pay，SAMSUNG Pay等支持NFC闪付功能的手机通过感应方式进站乘车。
