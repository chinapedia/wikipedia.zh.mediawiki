'''浙江省镇海中学'''，简称'''镇中'''，是[[中国|中国]][[浙江省|浙江省]]一所[[中学|中学]]。学校位于[[宁波市|宁波市]][[镇海区_(宁波市)|镇海区]][[招宝山街道|招宝山街道]]，[[甬江|甬江]]口北岸，临近[[东海|东海]]。

== 校史 ==
镇海中学原名镇海县中学堂，由清翰林院编修盛炳纬创办于1911年（宣统三年）。创办之时，正值满清政府兴学堂、废科举。当时，盛炳纬一方面从上海、汉口等地镇海同乡募集建校资金，另一方面，通过镇海县知事选拔曹位康等6位青年赴日本公费留学，以备学校师资。1911年，在梓荫山南侧建成镇海中学堂，1912年学校改名为镇海县中学校，并正式招生。

中华民国成立后一段时间曾经改为商校。[[抗日战争|抗日战争]]爆发之后，学校一度四处迁移，其间曾经改为私立学校，1944年改为镇海县立中学，次年搬迁至位于镇海孔庙的校址。1956年与私立辛成中学合并为完全中学。1995年成为浙江省第一所一级重点中学。<ref>{{cite web|url =http://www.zhzx.net.cn/campus/01introduce.asp|title =镇海中学-学校概况|publisher =镇海中学|accessdate =2011-04-09|deadurl =yes|archiveurl =https://web.archive.org/web/20101130171057/http://zhzx.net.cn/campus/01introduce.asp|archivedate =2010-11-30}}</ref>镇海中学在2016年中国内地高中美国留学排行榜上位居第27<ref>"http://gz.people.com.cn/n2/2016/0224/c358162-27802234.html" 人民网 2016-02-24</ref><ref> {{cite web |url=http://www.collegenode.com/list.php |title=存档副本 |accessdate=2016-04-05 |0=2016-10-14 |deadurl=yes |archiveurl=https://web.archive.org/web/20161014013045/http://www.collegenode.com/list.php |archivedate=2016-10-14 }} 联校网排名</ref>。

==设施==
镇海中学学校面积约80000平方米，学校建筑面积约42000平方米。校门口悬挂有“浙江省镇海中学”匾额，为[[郭沫若|郭沫若]]亲笔手书。

镇海中学教学设施极为现代化，拥有高速大容量的校园网络。普通教室、[[物理|物理]]、[[化学|化学]]、[[生物|生物]]实验室设施齐全，均配备有多媒体教学平台。教师每人配备[[电脑|电脑]]。图书馆藏书丰富。拥有室内冷暖水游泳池，体艺馆楼顶设有网球场。室内篮球场亦设施先进齐全，曾承办多次国家级[[篮球|篮球]]比赛。拥有400米标准塑胶跑道。学校并建有占地约50亩的农业科技实验园地。在百年校庆来临之际，学校对大部分建筑都进行了翻新。

==奖学金==
[[籍贯|籍贯]]镇海的日本实业家[[傅在源|傅在源]]先生在镇海中学设有“富的奖学金”。学校也会给成绩优异的同学颁发奖学金。

==竞赛成绩==
2006年，镇海中学高二学子[[沈才立|沈才立]]代表中国队参加在[[斯洛文尼亚|斯洛文尼亚]]首都[[卢布尔雅那|卢布尔雅那]]举行的第47届[[国际数学奥林匹克竞赛|国际数学奥林匹克竞赛]]，获得金牌，实现了[[浙江省|浙江省]]数奥金牌零的突破。2007年，沈才立以全国第一的成绩再次入选中国国家队，参加了在[[越南|越南]][[河内|河内]]举行的第48届国际数学奥林匹克竞赛，再次夺得金牌。

中国东南地区数学奥林匹克竞赛（[[闽|闽]][[浙|浙]][[赣|赣]]三省+[[香港|香港]]、[[上海|上海]]、[[广东|广东]]等地的代表队），至2007年，镇海中学已连续四届获得团体总分[[冠军|冠军]]和个人冠军。
