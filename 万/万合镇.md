'''万合镇'''是[[江西省|江西省]][[泰和县|泰和县]]下辖的一个镇，位于县境东北部、[[赣江|赣江]]东岸，东邻[[吉安市|吉安市]][[青原区|青原区]]。现万合镇为2001年由原万合镇和樟塘乡合并而成，总面积176平方公里，人口约5.3万人<ref name="gk">[http://218.64.81.28/jxth/bmgkxx/whz/gkxx/bjzfbmjs/200808/t20080815_10311.htm 泰和县万合镇概况]{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}，泰和县政府信息公开平台，2008-08-27 ，于2009年5月18日查阅</ref>。[[万合竹篙薯|万合竹篙薯]]泰和县最著名的特产之一，境内有[[梅冈古村|梅冈古村]]、[[华盖山|华盖山]]、[[文昌宫|文昌宫]]等名胜景观。

== 交通 ==
万合镇位于泰和县东北部、赣江东岸，距县城20公里，距[[泰和火车站|泰和火车站]]10公里。主要公路交通有泰（和）万（合）公路、永（昌）万（合）公路、古（坪）竹（山）公路、万（合）陂（头）公路以及在建的[[泉南高速公路|泉南高速公路]]。水运则有赣江航运码头。目前万合至泰和县城主要有两条线路：一条是在印霞江乘轮渡，沿泰万公路直达县城；另一条是沿永万公路，经永昌市绕行[[泰和大桥|泰和大桥]]，再折至县城。目前在建的大型水利工程[[赣江石虎塘航电枢纽工程|赣江石虎塘航电枢纽工程]]主坝址位于境内石虎塘村，建成后坝顶可通行，直达赣江西岸，将可大大缩短往来县城的时间。

== 地理 ==
全镇主要位于赣江[[河谷|河谷]]地带，万合圩周围为[[平原|平原]]区，东北、东南则以[[丘陵|低丘]]为主。[[仙槎河|仙槎河]]、[[仁善河|仁善河]]自南而来，在樟塘圩附近汇合后，注入赣江。境内的[[华盖山|华盖山]]为[[文天祥|文天祥]]抗元战场。

== 沿革 ==
1950年为万合、高樟、虎溪、樟塘、严家等乡，1956年并乡设万合乡、樟塘乡。1958年两乡分别改为万合人民公社、樟塘人民公社，1968年两公社合并为万合人民公社，1972年重新分为两公社，1984年撤消人民公社，分别恢复万合乡、樟塘乡<ref>李宝库主编：《中国政区大典》第2册，杭州：浙江人民出版社，1999年，ISBN 7-213-01789-6，第1187页</ref>。1999年9月7日，万合乡撤乡设镇。2001年12月3日，撤消樟塘乡，并入万合镇<ref>〈[http://www.jxqhdm.com/NEWS/newshow.aspx?ID=2200 吉安市乡镇行政区划变更]{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}〉，江西区划地名网，2007-10-16，于2009年4月26日查阅</ref>。

== 行政区划 ==

万合镇现辖1个[[居委会|居委会]]和28个[[村委会|村委会]]：万合镇居委会、湖尾村、高樟村、南垄村、坪上村、沙垄村、湖头村、罗家村、江南村、集义村、前进村、桑园村、沙湖村、华盖山村、哨丰村、陂溪村、店边村、菰塘村、塘尾村、南坑村、霞山村、黄坑村、宁溪村、大鹏村、赤溪村、铜陂村、樟塘村、竹山村、钟阜村，镇政府驻坪上村<ref>〈[http://www.jxqhdm.com/index-ja.aspx 2009年泰和县行政区划] {{webarchive|url=https://web.archive.org/web/20091025153207/http://www.jxqhdm.com/index-ja.aspx |date=2009-10-25 }}〉，吉安区划地名网（江西区划地名网子网站），于2009年4月1日查阅。</ref>。
