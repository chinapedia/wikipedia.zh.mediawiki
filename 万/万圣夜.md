'''万圣夜'''（{{lang-en|Halloween}}或{{lang|en|Hallowe'en}}，為“{{lang|en|Hallows' Even}}”或“{{lang|en|Hallows' Evening}}”的缩写，意為“[[諸聖節|諸聖節]]的前夜”），也常被称作'''万圣节'''，在每年的10月31日晚慶祝。是[[西方世界|西方世界]]的傳統節日，主要流行於[[盎格魯撒克遜人|盎格魯撒克遜人]]後裔雲集的[[美國|美國]]、[[不列顛群島|不列顛群島]]、[[澳大利亚|澳大利亚]]、[[加拿大|加拿大]]和[[新西蘭|新西蘭]]等[[西方國家|西方國家]]。當晚小孩會穿上化妝服，戴上面具，挨家挨戶收集[[糖果|糖果]]。

萬聖節通常與靈異的事物聯繫起來。西洋傳統上認為萬聖夜是[[鬼怪|鬼怪]]世界最接近人間的時間，這與[[中国|中国]]的[[中元節|中元節]]{{notetag|但與[[佛教|佛教]]的[[盂蘭盆節|盂蘭盆節]]稍有不同：佛教僧人在農曆4月15日-7月15日要結夏安居，用功修行，在7月15日當天則是解夏，民眾用上妙百味飲食（盂蘭盆）供僧，能得無量功德，能以此功德超渡身處惡道的七世父母。}}以及[[日本|日本]]的[[百鬼夜行|百鬼夜行]]類似。美國[[明尼蘇達州|明尼蘇達州]]的[[阿诺卡_(明尼苏达州)|阿诺卡]]號稱是「世界萬聖夜之都」，每年都舉行大型的巡遊慶祝<ref>{{cite web |title=Anoka, Minnesota: The Halloween Capital of the World |url=http://www.americaslibrary.gov/es/mn/es_mn_hallown_1.html |website=America's Story from America's Library |publisher=[[美國|美國]][[國會圖書館_(美國)|國會圖書館]] |accessdate=2018-10-31}}</ref>。

== 起源 ==
萬聖節（又稱[[諸聖節|諸聖節]]）與[[圣诞节|圣诞节]]一樣都是来自于[[基督宗教|基督宗教]]对其它文化的节日的吸收和重新诠释的节日，真正的日期是在十一月一日。

關於萬聖節的起源說法不一，多數認為源于[[不列颠|不列颠]][[凯尔特人|凯尔特人]]慶祝豐收的传统节日（可能為{{link-en|夏末节|Samhain}}），凯尔特人相信10月的最后一天是[[夏季|夏天]]的终结，冬天的开始，是一年的重要标志、最重要的节日之一，称其为「死人之日」或“鬼节”。相傳这一天各种恶鬼出没，死去人们的[[灵魂|灵魂]]也会离开阴间，在世间游走，这天晚上也就格外危险。為了嚇走邪惡的鬼魂，凱爾特人會戴上[[面具|面具]]。

另外在基督宗教方面，早期的大公教会（[[天主教|天主教]]）因為聖徒數目很多，無法一一立日紀念，就立在同一個日子來追念他們。按历史文献，最早如此纪念的是安提约基雅（[[安提阿|安提阿]]）教会，紀念日定為每年圣神降临后的第一个主日。后来教宗[[格里高利三世|格里高利三世]]（690—741年）在[[罗马|罗马]][[圣伯多禄大殿|圣伯多禄大殿]]中，把一座小堂奉献给诸圣，并把纪念诸圣的日期改为每年的11月1日，目的是為了對抗當時已經開始流行的萬鬼節（{{lang|en|Halloween}}）。教会在这个诸圣日（{{lang|en|All Saints Day}}）中纪念所有圣徒，表彰他们的圣德，并使教徒视他们为效法的楷模。诸圣日的第二天为诸灵日（{{lang|en|All Souls Day}}），是[[法国|法国]][[克呂尼修道院|克呂尼修道院]]的院长圣奥迪罗（{{lang|fr|St. Odilo}}）於西元998年推广的，在當日纪念殉教的信徒，也是基督徒扫墓的日子。天主教會也稱之為「追思已亡日」，以[[彌撒|彌撒]]聖祭及各種[[祈禱|祈禱]]來幫助尚未到達[[天堂|天堂]]的祖先、親友的日子。

古時[[愛爾蘭人|愛爾蘭人]]認為十月三十一日及十一月一日間亡靈會回人間，故有習俗生者穿上奇裝異服，遊行驅鬼。當時[[歐洲|歐洲]]也有為亡者布施糖果的習慣，接受者則答應以「為亡者祈禱」回報。於是發展出孩子們到各家索取糖果的習慣。

來自[[愛爾蘭|愛爾蘭]]和[[蘇格蘭|蘇格蘭]]的移民於19世紀將相關的習俗傳播到各個[[英屬殖民地|英屬殖民地]]，例如[[美國|美國]]、[[加拿大|加拿大]]、[[澳大利亞|澳大利亞]]、[[新西蘭|新西蘭]]等地，其他國家則於20世紀末受[[美國文化|美國文化]]影響而開始慶祝萬聖夜。

=== 象徵 ===
萬聖夜的主題是鬼怪、嚇人，以及與[[死亡|死亡]]、[[陰陽|陰陽]]、[[魔法|魔法]]、[[怪物|怪物]]有關的事物。通常與萬聖夜扯上關係的事物有[[南瓜|南瓜]]、[[鬼魂|鬼魂]]、[[食屍鬼|食屍鬼]]、[[女巫|女巫]]、[[蝙蝠|蝙蝠]]、[[猫|黑貓]]、[[貓頭鷹|貓頭鷹]]、[[精靈|精靈]]、[[殭屍|殭屍]]、[[骷髏|骷髏]]和[[惡魔|惡魔]]等，還有虛構人物如[[吸血鬼|吸血鬼]]和[[科學怪人|科學怪人]]。而[[黑色|黑色]]和[[橙色|橙色]]是萬聖夜的傳統顏色。現代萬聖夜的產品也大量使用[[紫色|紫色]]、[[綠色|綠色]]和[[紅色|紅色]]。[[秋天|秋天]]的元素如[[南瓜|南瓜]]和[[稻草人|稻草人]]等，也成為萬聖節的象徵。

[[杰克南瓜灯|傑克燈]]（{{lang-en|Jack-o'-lantern}}）是萬聖夜最廣為人知的象徵物。在英國和愛爾蘭，當地人在挖空的[[蕪菁|蕪菁]]或[[甜菜|甜菜]]根中燃點蠟燭製成傑克燈。後來移民到美國的人，將本來用於慶祝豐收的雕刻南瓜，改為傑克南瓜燈，因為南瓜比較大而且容易在上面雕刻圖案。不少家庭在南瓜上鑿刻嚇人的面容，並放在大門口的階梯上，意在驅走惡魔與妖怪。

=== 食物 ===
由於萬聖夜鄰近蘋果的豐收期，[[苹果糖|太妃糖蘋果]]（{{lang|en|toffee apples}}）成為應節食品。製法是把蘋果插上竹籤，然後手持竹籤把蘋果放在[[太妃糖|太妃糖漿]]中轉動，有時會再粘上[[果仁|果仁]]。從前，各家各戶會準備太妃糖蘋果送給小孩，但當傳聞有人把[[大頭針|大頭針]]和刀片放入蘋果中，送太妃糖蘋果的習慣便逐漸消失。雖然大部分個案只是惡作劇，即使是在真實個案中，小孩也僅受輕傷，但是不少家長仍然以為在蘋果中放刀片是十分普遍的。

其他特色食品還有：

*{{link-en|粟米糖|candy corn}}
* 熱[[蘋果酒|蘋果酒]]
* 烘[[南瓜|南瓜子]]

== 萬聖節活動 ==

=== 不給糖就搗蛋 ===
{{main|不给糖就搗蛋}}
「不给糖就搗蛋」是萬聖夜的主要活動之一，主要流行於[[英語世界|英語世界]]，如[[英國|英國]]、[[美國|美國]]、[[加拿大|加拿大]]等地区。小孩裝扮成各種鬼怪，逐門逐户按響鄰居的門鈴，大叫“{{lang|en|Trick or Treat!}}”（意即「不請客的話就搗蛋！」），主人家（可能同樣穿著恐怖服裝）便會拿出一些[[糖果|糖果]]、[[巧克力|巧克力]]或是小禮物，有些人更會給些許零錢。部分家庭甚至使用聲音特效和製煙機器營造恐怖氣氛。小孩一晚取得的糖果往往數以袋計，整袋整袋的搬回家。

在蘇格蘭，小孩要糖果時會說：“{{lang|en|The sky is blue, the grass is green, may we have our Halloween.}}”（“天是藍色，草是綠色，齊來慶祝萬聖節前夜。”），然後以唱歌跳舞等表演來博得糖果。

傳統的萬聖夜装飾包括[[殭屍|殭屍]]、[[鬼魂|鬼魂]]、[[女巫|女巫]]、[[惡魔|惡魔]]等。典型的服飾是蓋着一塊剪了兩個眼孔的白床單來扮鬼。

在19世紀的蘇格蘭和愛爾蘭，當時的人認為萬聖夜鬼魂會來到人間，故會打扮得與鬼魂一樣，讓鬼魂把自己當成同類。近年，萬聖節前夜的裝扮已不限於恐怖，而可以作更加多元的變裝打扮，如扮成影視明星或公眾人物，例如[[2004年美國總統選舉|2004年美國總統選舉]]期間，[[乔治·布什|乔治·布什]]和[[约翰·克里|约翰·克里]]成為美國流行的萬聖夜打扮。而2001年[[九一一襲擊事件|九一一襲擊事件]]後，[[消防員|消防員]]、[[警察|警察]]等成為小孩的扮演對象。據估計，2004年有大約215萬個美國小孩裝扮成[[蜘蛛人|蜘蛛人]]，是當年最受歡迎的打扮。<ref>[https://web.archive.org/web/20080926113938/http://www.nrf.com/content/default.asp?folder=press%2Frelease2004&file=costumes1004.htm&bhcp=1]</ref>

由於此活動在美國、加拿大十分普遍，於1950年曾推出一項募捐計劃，將籌款錢箱通過學校分發予小孩，讓小孩討糖果時順便募集捐款。據估計，該計畫推出以來已籌得超過1.19億美元。

研究機構BIGresearch曾為美國全國零售聯盟進行調查，發現53.3%的顧客為2005年萬聖夜購物，平均消費額38.11美元，以此推算，美國消費者2005年為萬聖夜支出的金額高達33億美元。

=== 其他活動 ===
此外，還有一些和萬聖節派對有關的遊戲。其中最常見的就是「[[咬蘋果|咬蘋果]]」（{{lang|en|apple bobbing}}），在遊戲中，[[蘋果|蘋果]]漂浮在一個裝滿水的大盆子或是木桶的水面上，參加者設法用牙齒咬起漂浮在水上的蘋果，這個習俗被認為是由古羅馬讚頌女神[[波摩纳|波莫娜]]（[[古羅馬|古羅馬]]象徵豐收的[[羅馬神話|女神]]）的慶典演變而來。

愛爾蘭有一種傳統[[占卜|占卜]]遊戲，參加者矇着眼，從放着幾個小碟的桌上選出其中一隻，如摸到的碟盛有泥土，代表來年會有與參加者有關的人過世，如盛有水代表將會遠行，盛有錢幣代表會發財，盛有豆代表會窮困，等等。在19世紀的愛爾蘭，少女會在灑有麵粉的碟上放[[蛞蝓|蛞蝓]]，而蛞蝓爬行的痕跡會是少女將來丈夫的模樣。在北美，傳說如果未婚女子在萬聖夜坐在黑暗的房間中，便可以在鏡中看見未來丈夫的樣貌。不過，如果她們將於結婚前死去，鏡中便會出現一個骷髏頭。這個習俗於19世紀後期相當流行，也有相關的[[賀卡|賀卡]]售賣。

說[[鬼故事|鬼故事]]及看[[恐怖片|恐怖片]]也是萬聖夜派對中常見的活動。以萬聖夜為主題的[[電視特輯|電視特輯]]通常在萬聖夜假期當天或之前播放，對象多數是兒童。

== 文化历史 ==

=== 凱爾特人的薩溫節 ===
在古[[凱爾特人|凱爾特人]]的信仰裡，新的一年於11月1日開始，或稱[[薩溫節|薩溫節]]（{{lang|en|Samhain}}）。正如比較短的白天象徵新一年的開始，日落亦象徵新一天的開始；所以每年收割的節慶於10月31日晚上開始。[[不列顛群島|不列顛群島]]的[[德魯伊|德魯伊]]教徒會燃點農作物作為祭品，而當他們圍著火堆跳舞時，太陽季節便會完結而薩溫節隨即開始。凱爾特人相信[[死神|死神]]在10月31日的晚上會和鬼魂一起重返人間，尋找[[替身_(鬼魂)|替身]]。因此他們燃點火炬，焚燒動物以作為死神的獻禮。還會用動物的頭或皮毛做成的服飾打扮自己，發出古怪的聲音，使死神認不出自己，避過災難。這就是今天萬聖節化妝舞會的由來。過了當晚，鬼魂回到陰間，一切回復平靜。

11月1日早上，德魯伊把火堆中仍在燃烧的炭烬分给每一家，用来点燃新的厨房火焰。这些火焰用来让家里保持温暖，并赶走恶灵。因为此时被认为是一年中连接阴阳世界的无形之门开启，兩個世界互相通聯的时刻。

[[篝火|篝火]]是节日裡的重要节目。村民们把宰牲的牛骨扔到火堆上；据传“篝火”（{{lang|en|bonfire}}）一词源自“骨头火堆”（{{lang|en|bone}}－{{lang|en|fire}}）。篝火燃起之后，村民们把其余的灯火全部熄灭。然后每个家庭用同一个火种庄严地点燃自家壁炉，以此作为全村人紧密团结的标志。至今在爱尔兰，仍然有数百处篝火在万圣夜当晚点燃。

[[新異教主義|新异教徒]]仍然在万圣夜举行薩溫節的[[太陽慶典_(新异教)|太陽慶典]]，以及其它传统节日活动。

=== 挪威圣灵节（{{lang|no|Elven Blót}}） ===
在古老传说（[[挪威神话|挪威宗教]]）以及如今的恢复传统研究（[[日耳曼動|Ásatrú]]）中提到，万圣夜当天曾是一个祭献的节日{{tsl|no|blót||blót}}，宰杀牲口祭献[[精灵|精灵]]，供奉食物祈神赐福。

[[挪威|挪威]]國王[[奥拉夫二世|奥拉夫二世]]的宮廷詩人席格瓦特·索爾達松（{{lang|no|Sigvatr Þorðarson}}）在1020年左右創作的一首诗作《东方旅行诗》（''{{lang|no|Austrfaravísur}}''）有此记录。索爾達松作为一个基督徒，曾拒绝在[[瑞典|瑞典]]的一家异教徒那里因为那儿正在举行“[[精灵|精灵]]祭献”（{{lang|no|álfablót}}）。但是，有关精灵祭献的内容我们无法获得更多可靠的资料。不过，像其他献祭仪式（{{lang|no|blót}}）一样，有供奉食物这个环节，后来[[斯堪的纳维亚|斯堪的纳维亚]]人也保留了以牲口祭献精灵传统。从献祭的日子靠近[[分點|秋分]]来看，又因为精灵和生育以及祖先有联系，我们可以猜测这些祭礼实质是在祭奠祖先，同时祈求人丁兴旺。

=== 万圣夜传统 ===
自17世纪起，庆祝万圣夜的传统在英格兰南部逐渐消失，代之以在11月5日举行的[[火药阴谋|火药阴谋]]纪念会。不过在苏格兰、爱尔兰和英格兰北部万圣夜依然盛行。直到最近十年，南英格兰人才重新开始庆祝万圣夜，只是方式完全美国化了。

传统的庆祝仪式在爱尔兰几乎被完全保留。在那里10月份的最后一个星期一是公众假日，所有学校停课，准备在接下来的一个星期举行期中考试。这个假日通常叫做万圣节假期。爱尔兰是在万圣夜给学生们放假的唯一一个国家，所以可以自由地庆祝，保留了古老相传的传统。

== 宗教觀點 ==
大部分人不認為或不知道萬聖夜有重要的宗教意義，只把這節日當作是普通的「節慶」看待，如同[[聖誕節|聖誕節]]一樣。

基督宗教对此议题并不完全一致。有些基督宗教團體認為，萬聖夜是一個歡慶死亡並榮耀魔鬼與邪靈的日子，因此[[天主教|天主教]]、[[基督新教|基督新教]]等对于教徒慶祝萬鬼節活動持不鼓励或者反对态度<ref>{{cite web |url =http://actoflove.hkskh.org/site/portal/Site.aspx?id=A17-1126&lang=zh-TW |title = 基督徒與「萬聖節」|publisher = |date = |accessdate =}}</ref><ref>{{cite web |url =http://catholic-dlc.org.hk/halloween.doc |title = 有關萬聖節注意事項 - 天主教香港教區禮儀委員會|publisher = |date = |accessdate =}}</ref><ref>{{cite web |url =http://www.sjfsw.cn/ouzhoufengsu/1518.html|title =梵蒂冈吁万圣节改神圣节|publisher = |date = |accessdate =}}</ref>。近年也有一些福音派機構（如[[影音使團|影音使團]]<ref>[http://www.media.org.hk/main/asp/corporate_web_20071013.asp Halloween廣告內容驚嚇全港  調查發現過半數人認為萬聖節的宣傳廣告內容對兒童心理帶來負面影響]</ref>）也認為萬聖夜根本沒關心節日原本的慶祝目的，還間接歡迎了魔鬼，故不建議人參與任何萬聖節活動。有些團體在萬聖夜當晚另外舉行「[[期待耶穌節|期待耶穌節]]」（{{lang|en|Jesusween}}，取名於「[[耶稣|耶稣]]」Jesus及「期待」的古英語字Ween），使信徒得以在萬聖夜當晚可以參與另類活動<ref>{{cite web |url=http://jesusween.com/ |title=Jesus Ween Christian Festival |trans_title=期待耶穌節：一個基督徒的節慶 |accessdate=2011-10-30 |language=en}}</ref>；又鼓勵信徒在小朋友前來索取糖果時，給他們送贈[[聖經|聖經]]<ref>{{cite news|url=http://www.anntw.com/awakening/news_center/show.php?itemid=26238|title=要糖送聖經！ 加牧師創意過萬聖節|author=莊瑞萌|publisher=[[醒報|醒報]]|accessdate=2011-10-31|language=zh-hant|deadurl=yes|archiveurl=https://web.archive.org/web/20111029233918/http://www.anntw.com/awakening/news_center/show.php?itemid=26238|archivedate=2011-10-29}}</ref>或[[十字架|十字架]]等其他基督宗教禮品<ref>{{cite news |url=http://christiantimes.org.hk/Common/Reader/News/ShowNews.jsp?Nid=69088&Pid=6&Version=0&Cid=150&Charset=big5_hkscs|title=「佈道訓練新世代」系列：從萬聖節說起 | author=[[柴樹良|柴樹良]] |publisher=《[[時代論壇|時代論壇]]》 |date=2011=10-26 |accessdate=2011-10-31 |language=zh-hant }}</ref>。
