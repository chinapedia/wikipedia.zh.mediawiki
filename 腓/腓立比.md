'''腓立比'''（{{lang-grc|Φἱλιπποι}}，转写：Philippoi）是在[[東馬其頓|東馬其頓]]的一個[[城市|城市]]，在主前356年由[[腓力二世_(馬其頓)|腓力二世]]建立。在[[十四世紀|十四世紀]]的[[鄂圖曼帝國|鄂圖曼帝國]]統治之下被摒棄。

== 起源 ==
腓立比由[[马其顿国王|马其顿国王]]腓力二世建立，建造该市的目标是为了控制附近[[潘盖翁山|潘盖翁山]]的[[金矿|金矿]]，并控制[[暗妃波里|暗妃波里]]和[[尼亚波利|尼亚波利]]之间的道路，从东方穿过马其顿到西方伟大的皇家道路的一部分。

== 羅馬時代 ==
该市于[[凯撒|凯撒]]被暗杀后的罗马内战期间再次出场。他的继承人[[马克·安东尼|马克·安东尼]]和[[屋大维|屋大维]]面对凯撒的行刺者，[[马可斯·布鲁图斯|马可斯·布鲁图斯]]和[[盖乌斯·卡西乌斯|盖乌斯·卡西乌斯]],在前42年10月在城西平原进行[[腓立比战役|腓立比战役]]。安东尼和屋大维在这反对共和派最终的战役中获胜。他们解散了一些老兵，可能是第28[[罗马军团|罗马军团]]，安置他们在该市，新建了''Colonia Victrix Philippensium''。前30年，屋大维成为[[罗马皇帝|罗马皇帝]]，改组该殖民地，在此定居更多殖民者，老兵们可能是之前的 [[羅馬禁衛軍|羅馬禁衛軍]]和其他意大利人。该市改名为''Colonia Iulia Philippensis''，前27年1月，屋大维从[[罗马元老院|罗马元老院]]接受[[元首|元首]]称号。 于是再改为''Colonia Augusta Iulia Philippensis''

第二次改名后，也许是第一次改名后，腓立比土地分为正方形，分配给殖民者。该市保留了马其顿城墙、其总体规划通过在希腊agora遗址略东的位置建造一个广场缓解偏于一边的情况。这是一个"微型的罗马" ，通行罗马国内法，并由2位直接由罗马委任的军官''duumviri''统治。

49年或50年，使徒[[保罗|保罗]]受[[异象|异象]]引导来到这里(使徒行传16:9-10)，同来的有[[西拉|西拉]]、[[提摩太|提摩太]]，可能还有[[路加|路加]]，[[使徒行传|使徒行传]]的作者，保罗第一次在欧洲土地上的腓立比讲道(使徒行传16:12-40)，并在城西的河里为一个卖[[紫色|紫色]]布匹的商人吕底亚施浸。在腓立比时，他赶逐一个行巫术的使女身上的邪灵，引起全城大骚乱，使得保罗和西拉被众人棍击并被捕(使徒行传16:16-24)。一场地震打开了监牢的门。禁卒醒来，以为犯人已经逃走，准备自杀。保罗阻止了他，说所有犯人还在那里。于是禁卒成为欧洲最早的基督徒之一(使徒行传16:25-40)。这时，那里还有一个[[犹太|犹太]]团体和一个[[犹太会堂|犹太会堂]]（有碑刻证明）。

== 早期的基督宗教時代 ==
[[File:Philippi_basilica_B.jpg|thumb]]

该市第一个[[教堂|教堂]]证实是一个小型建筑，可能起源于一个小型祈禱室。據Porphyrio主教所述，这个''保罗长方形教堂''建於343年，及地板上鑲嵌著[[馬賽克|馬賽克]]。 

尽管这里没有保罗创立基督徒团体的考古或文字证据，该市在5世纪和6世纪的繁荣要因他和他的殉道團體而起。和其他城市一样，这时建造了许多新的教会建筑。从4世纪中叶到6世纪末，腓立比建造了7个教堂，有些在规模和装饰上可和[[帖撒罗尼迦|帖撒罗尼迦]]、甚至[[君士坦丁堡|君士坦丁堡]]最美的建筑竞争。與位於君士坦丁堡的教堂比較，复杂的主教座堂圍繞着八角座堂，以取代5世纪末的保罗长方形教堂。 

在同一个年代，该城重建了防御工事以抵御自[[巴尔干|巴尔干]]而來的威脅。473年，该市被[[Ostrogoths|Ostrogoths]]围困，但未能攻取，只摧毁了附近的村庄。

== [[拜占庭|拜占庭]]和[[鄂圖曼帝國|鄂圖曼帝國]]時代 ==
6世纪末[[斯拉夫人|斯拉夫人]]和547年[[查士丁尼大瘟疫|查士丁尼大瘟疫]]的入侵削弱该市，毁灭了马其顿农业经济，619年，该市几乎被一次[[地震|地震]]完全摧毁，此后再未恢复。7世纪这里还有少量的活动，但现在该市比一个村庄大不了多少。不知何时，该市被抛弃。当16世纪[[法国|法国]]旅行家Pierre Belon来到这里时，这里除了废墟一无所有，被土耳其人当作采石场。该市的名称被附近平原的一个土耳其村庄 Philibedjik所继承，该村已经消失，现在有一个希腊村庄在山上。

== 遗址的考古學挖掘 ==
[[File:Philippi_Daumet_Direkler.jpg|thumb]]

16世纪的旅行者注意到该市并作了简单描述，对该市第一次考古描述是1856年的Perrot，然后是1861年的L. Heuzey 和H. Daumet的著名的''马其顿考古传教会''。不过第一次挖掘直到1914年夏天才开始，不久被[[第一次世界大战|第一次世界大战]]中断。1920年恢复，由[[雅典法国学校|雅典法国学校]]挖掘、运输，持续到1937年。这期间希腊剧院、广场、长方形教堂A和B、浴室和城墙都被挖掘出来。[[第二次世界大战|第二次世界大战]]后，希腊考古学家回到遗址。从1958年到1978年考古协会和帖撒罗尼迦大学挖出了主教地区和八角形教堂、大型私人住宅、靠近博物馆一个新的长方形教堂和城东[[necropolis|necropolis]]的另外2座。
