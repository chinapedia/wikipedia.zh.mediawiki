'''腓特烈·威廉三世'''（{{lang-de|Friedrich Wilhelm III}}，{{bd|1770年|8月3日|1840年|6月7日|Friedrich Wilhelm III}}），或译'''弗里德里希·威廉三世'''，[[霍亨索伦王朝|霍亨索伦王朝]]的[[普鲁士|普鲁士]]国王（1797年11月16日—1840年6月7日在位）和布蘭登堡的[[選帝侯|選帝侯]]。

他还创作了军乐《普鲁士检阅进行曲》（{{lang|de|Preußischer Präsentiermarsch}}）。

== 早年生活 ==
腓特烈·威廉三世是[[普鲁士国王列表|普鲁士国王]][[腓特烈·威廉二世|腓特烈·威廉二世]]与黑森-达姆施塔特公主弗里德里克·路易丝的长子，1770年8月3日生於波茨坦。在他幼年时，他父亲正与情妇威廉敏妮·冯·利希特瑙（{{lang|de|Wilhelmine Enke, Gräfin von Lichtenau}}）打的火热，将小腓特烈·威廉交给[[家庭教师|家庭教师]]抚养。他常在[[腓特烈二世_(普鲁士)|腓特烈二世]]时代的老兵汉斯·冯·布伦蒙塔尔伯爵（{{lang|de|Hans Graf von Blumenthal}}）的领地帕雷茨生活，与汉斯的儿子一起长大。腓特烈·威廉在帕雷茨的生活非常开心，以至他长大后买下了这块领地。腓特烈·威廉是个忧郁的人，个性优柔寡断，虔诚忠实。

腓特烈·威廉青年时代加入了普鲁士军队，1784年成为[[中尉|中尉]]，1790年晋升[[上校|上校]]。他参加了1792年至94年的反法战争。

1797年11月16日，腓特烈·威廉的父亲去世，他继位成为普鲁士国王。

== 反法战争和拿破仑战争 ==
[[File:Friedrich_Wilhelm_III_von_Gebauer.jpg|thumb]]
成为国王后的腓特烈·威廉三世在[[第二次反法同盟|第二次反法同盟]]中保持中立，普鲁士在欧洲的威望下降。1803年，普鲁士的中立立场使他得到了[[希尔德斯海姆|希尔德斯海姆]]和[[帕德博恩|帕德博恩]]教区、[[明斯特|明斯特]]教区的一部分、库尔美因茨的艾希斯菲尔德、[[爱尔福特|爱尔福特]]等地。1805年8月24日，腓特烈·威廉三世与法国皇帝[[拿破仑|拿破仑]]签定条约，法国将[[汉诺威_(地区)|汉诺威]]让给普鲁士，普鲁士保持在未来战争中的中立态度。俄罗斯曾诱劝普鲁士加入反法同盟，但腓特烈·威廉三世由于对拿破仑的恐惧和汉诺威的诱惑没有出兵。[[奥斯特里茨战役|奥斯特-{里}-茨战役]]后，[[神圣罗马帝国|神圣罗马帝国]]瓦解，拿破仑建立了依附于法国的[[莱茵联邦|莱茵联邦]]，这威胁到了普鲁士的利益，引起了普鲁士的不安。
[[File:Charles_Meynier_-_Napoleon_in_Berlin.png|thumb]]
由于拿破仑在与[[英国|英国]]和谈中表示出将汉诺威归还英国的想法，普鲁士宫廷感觉受到了欺骗，反法情绪高涨。1806年7月25日，普鲁士与俄罗斯签订条约，[[第四次反法同盟|第四次反法同盟]]形成。在1806年10月的[[耶拿-奥厄施塔特战役|耶拿-奥厄施塔特战役]]中，普鲁士军队由于武器落后、战术保守、动作迟缓，几乎全军覆没，连普军总司令[[不伦瑞克统治者列表|不伦瑞克-沃尔芬比特尔公爵]][[卡尔·威廉·斐迪南_(不伦瑞克-沃尔芬比特尔)|卡尔·威廉·斐迪南]]也光荣牺牲。10月27日，拿破仑进驻[[柏林|柏林]]，普鲁士王室逃亡[[东普鲁士|东普鲁士]]，受到[[俄国君主列表|俄罗斯沙皇]][[亚历山大一世_(俄国)|亚历山大一世]]的保护。

1807年6月[[弗里德兰战役|弗里德兰战役]]后，法军占领整个普鲁士领土。7月9日普法签订[[提尔西特和约|提尔西特和约]]，普鲁士割让16万平方公里土地，包括普属[[波兰|波兰]]的绝大部分领土（第二次、第三次[[瓜分波兰|瓜分波兰]]所得，以及第一次瓜分波兰所得领土的南半部），以及[[易北河|易北河]]以西的全部领土，并赔款1.3亿法郎。普鲁士王国只剩下了“旧普鲁士”、[[勃兰登堡|勃兰登堡]]、[[波美拉尼亚|波美拉尼亚]]和[[西里西亚|西里西亚]]四个省。易北河以西的领土被拿破仑整合为[[威斯特法伦王国|威斯特法伦王国]]，他的弟弟[[热罗姆·波拿巴|热罗姆]]成为国王；普属波兰成为[[华沙公国|华沙公国]]。

1812年，普鲁士出兵2万助法国入侵俄罗斯。同年底，法国在俄罗斯惨败，普俄签署停战协议，次年3月普鲁士对法开战。俄罗斯、普鲁士、英国、瑞典、西班牙、葡萄牙结成[[第六次反法同盟|第六次反法同盟]]。普鲁士将军[[格布哈德·冯·布吕歇尔|格布哈德·冯·布吕歇尔]]在战争中起到了重要作用。[[莱比锡战役|莱比锡战役]]联军胜利后，普军直指[[巴黎|巴黎]]，在受到几次挫败后1814年3月31日联军进驻巴黎。在次年拿破仑百日复辟后，各国结成[[第七次反法同盟|第七次反法同盟]]。布吕歇尔与英军统帅威灵顿公爵[[阿瑟·韦尔斯利|阿瑟·韦尔斯利]]在[[滑铁卢战役|滑铁卢战役]]中将拿破仑彻底打败。

[[维也纳会议|维也纳会议]]使普鲁士重新得到[[拿破仑战争|拿破仑战争]]失去的领土，但在瓜分[[萨克森|萨克森]]时，与其他列强发生分歧，几乎与英、法、奥发生战争，后来达成了妥协。普鲁士得到了三分之二的萨克森领土和莱茵河沿岸的土地，而将波兰让给俄罗斯。普鲁士成为德意志邦联内德语居民占优势的唯一强国，以及欧洲列强之一。

== 普鲁士的改革 ==
[[File:Denkmal_Friedrich_WilhelmIII.JPG|thumb]]
1806年普鲁士在[[耶拿|耶拿]]和奥厄施塔特惨败后，[[普鲁士首相列表|普鲁士首相]][[卡尔·施泰因|卡尔·施泰因]]男爵（{{lang|de|Heinrich Friedrich Karl Reichsfreiherr vom und zum Stein}}）开始推行改革，其措施包括让公民参与政治以唤醒其民族主义情感；释放农奴；实行地方自治；改组中央政府机构等。

施泰因在拿破仑的压力下被解职后，[[卡尔·奥古斯特·冯·哈登堡|卡尔·奥古斯特·冯·哈登堡]]侯爵（{{lang|de|Karl August Fürst von Hardenberg}}）于1810年继承了他的改革事业。他宣布工商业自由、取消行会特权等。另外[[格哈德·冯·沙恩霍斯特|格哈德·冯·沙恩霍斯特]]将军开始对普鲁士军队进行改革，设立了国防部和总参谋部，并发布了《普遍兵役制》。

[[威廉·馮·洪堡|威廉·馮·洪堡]]对普鲁士的教育进行了改革，他重新改革了普魯士引以為傲的[[義務教育|義務教育]]制度，讓所有階層的子女都有相同的機會接受教育；创建了[[柏林洪堡大學|柏林洪堡大學]]。此后，[[布累斯劳大学|布累斯劳大学]]、[[波恩大学|波恩大学]]陆续建立。

腓特烈·威廉三世曾对从拿破仑占领下的'''[[哈勒大学|哈勒大学]]'''逃出来的教授们讲到：“这个国家必须用它精神上的力量来弥补它物质上的损失。正是因为贫穷，所以要办教育。我还从未听说过一个国家是因为办教育而办穷了的，办亡国了的。教育不仅不会使国家贫穷，恰恰相反，教育是摆脱贫困的最好手段！”国王甚至喊出了这样的口号：“大学是科学工作者无所不包的广阔天地，科学无禁区，科学无权威，科学自由！”

教育改革为普鲁士迟到的工业化飞速发展奠定了雄厚的科学技术人才基础，成为普鲁士重新崛起的动力。

== 晚期统治 ==
[[拿破仑战争|拿破仑战争]]后，腓特烈·威廉三世的统治变得[[保守主义|保守]]，奥地利外交部长[[克萊門斯·梅特涅|克萊門斯·梅特涅]]对他有一定影响。他不仅没有实现给普鲁士一部[[宪法|宪法]]的诺言，还强烈制止立宪运动。作为[[神圣同盟|神圣同盟]]的一员，普鲁士出兵镇压了[[西班牙|西班牙]]、[[意大利|意大利]]、[[波兰|波兰]]等地的革命。他在位的最后25年里，普鲁士受限於外部俄奧聯盟的規範（1818年[[亞琛會議|亞琛會議]]上挫敗），以及內部離心力量的影響（西部大量的[[天主教|天主教]]城市抗拒新教國策），国势渐弱。

腓特烈·威廉三世于1840年6月7日在柏林去世，终年69岁。

== 婚姻与子女 ==
[[File:Friedrich_Wilhelm_III_mit_Königin_Luise_im_Park_von_Schloss_Charlottenburg.jpg|thumb]]
腓特烈·威廉三世在1793年12月24日与[[梅克伦堡-施特雷利茨|梅克伦堡-施特雷利茨]]公爵[[卡尔二世_(梅克伦堡-施特雷利茨)|卡尔二世]]的第三女路易丝·奥古斯苔·威廉明妮·阿玛莉埃（Luise Auguste Wilhelmine Amalie，1776年3月10日—1810年7月19日）结婚，有五子四女：
* 长子'''[[腓特烈·威廉四世|腓特烈·威廉]]'''（'''Friedrich Wilhelm'''，1795年10月15日—1861年1月2日），继位成为普鲁士国王。 1823年与巴伐利亚王国的伊丽莎白公主结婚，无嗣。 享壽65岁。
* 次子'''[[威廉一世_(德国)|威廉·路德维希]]'''（'''Wilhelm Ludwig'''，1797年3月22日—1888年3月9日），1829年与[[萨克森-魏玛-艾森纳赫|萨克森-魏玛-艾森纳赫]]公国的奥古斯塔郡主结婚，1861年成为普鲁士国王，1871年成为[[德国君主列表|德意志帝國皇帝]]。 有一子一女，享耆壽91岁。
* 长女腓特烈卡·路易丝·'''夏洛特'''·威廉敏娜（Friederike Luise ''Charlotte'' Wilhelmine，1798年7月13日—1860年11月1日），1817年与[[俄罗斯帝国|俄罗斯帝国]]的尼古拉·巴甫洛维奇[[大公|大公]]（未来的俄皇[[尼古拉一世_(俄国)|尼古拉一世]]）结婚，改信[[东正教|东正教]]并改名'''[[亚历山德拉·费奥多罗芙娜皇后_(尼古拉一世)|亚历山德拉·费奥多罗芙娜]]'''（'''{{lang|ru|Александра Фёдоровна}}'''），后成为俄罗斯帝国的[[皇后|皇后]]，享壽62岁。
* 次女'''弗里德里克'''·奥古斯特·卡洛琳·阿玛莉（''Friederike'' Auguste Karoline Amalie，1799年—1800年）
* 三子腓特烈·'''卡尔'''·亚历山大（Friedrich ''Karl'' Alexander，1801年6月29日—1883年1月21日），1827年与[[萨克森-魏玛-艾森纳赫|萨克森-魏玛-艾森纳赫]]大公[[卡尔·弗里德里希_(萨克森-魏玛-艾森纳赫)|卡尔·弗里德里希]]的女儿玛莉·路易丝结婚，有一子二女， 81岁时去世。 她的妹妹[[奥古斯塔_(德国皇后)|奥古斯塔]]后来嫁给了卡尔的次兄威廉·路德维希。
* 三女弗里德里克·威廉敏娜·'''亚历珊德琳'''·玛莉·海伦（Friederike Wilhelmine ''Alexandrine'' Marie Helene，1803年2月23日—1892年4月21日），1822年嫁给[[梅克伦堡统治者列表|梅克伦堡-什未林大公]][[保罗·弗里德里希_(梅克伦堡-什未林)|保罗·弗里德里希]]，梅克伦堡-什未林大公[[弗里德里希·弗朗茨二世_(梅克伦堡-什未林)|弗里德里希·弗朗茨二世]]之母。
* 四子'''斐迪南'''·弗里德里希·尤利乌斯·利奥波德（''Ferdinand'' Friedrich Julius Leopold，1804年—1806年）
* 四女'''路易丝'''·奥古斯特·威廉敏娜·阿玛莉（''Luise'' Auguste Wilhelmine Amalie，1808年2月1日—1870年12月6日），1825年嫁给[[荷兰君主列表|荷兰国王]][[威廉一世_(荷兰国王)|威廉一世]]的小儿子腓特烈。
* 五子腓特烈·亨利·'''阿尔布雷希特'''（Friedrich Heinrich ''Albrecht''，1809年10月4日—1872年10月14日），1830年与荷兰国王威廉一世的女儿玛丽安妮结婚，1849年离婚，有一子三女。

腓特烈·威廉三世在1824年续娶了奥古斯塔·冯·哈拉赫女伯爵（{{lang|de|Auguste Gräfin von Harrach}}）为第二任妻子，并封她为利格尼茨女亲王（{{lang|de|Fürstin von Liegnitz}}），二人没有子嗣。
