'''腓特烈二世'''（Friedrich II，{{bd|1194年|12月26日|1250年|12月13日|catIdx=F}}），[[霍亨斯陶芬王朝|霍亨斯陶芬王朝]]的[[德国君主列表|罗马人民的国王]]（1211年—1250年在位）和[[神圣罗马帝国|神圣罗马帝国]]皇帝（1220年加冕）。他也是[[西西里王國|西西里]]国王（称腓特烈一世，1198年起），[[耶路撒冷|耶路撒冷]]国王（1225年—1228年），[[意大利|意大利]]国王和[[勃艮第|勃艮第]]领主。

腓特烈二世的父亲是神圣罗马帝国皇帝[[亨利六世_(神圣罗马帝国)|亨利六世]]，母为[[科斯坦察_(西西里女王)|西西里的康斯坦萨]]（由[[诺曼人|诺曼人]]统治的西西里王国唯一的女继承人）。出生于[[意大利|意大利]][[安科纳|安科纳]]边防区（[[马克|马克]]）的耶西（Jesi Mark Ancona），独生子。
[[File:Friedrich_ii_hrr.png|thumb]]

腓特烈二世极其注重君威，留下了大量雕塑，使今人得以猜测他的容貌。青年时，其容貌相当地俊美清秀，显得很沉静。但随着岁月增长，而越来越清瘦，表情越来越严厉，在他老年的雕塑中，人们看到的是一位皱着双眉、似乎相当严厉的君主。

雖是被[[绝罚|開除教籍]]的皇帝，但利用外交手段指導第六次十字軍，毫無死傷地進入聖地耶路撒冷，被稱為「王座上第一個近代人」(The First modern ruler)的知識份子。

== 身世和童年 ==
他的父亲是德意志国王和神圣罗马帝国皇帝亨利六世，母亲是西西里女王康斯坦丝。[[乔万尼·薄伽丘|薄伽丘]]在《[[名女传|名女传]]》里记载关于他的母親康斯坦丝皇后的传说。康斯坦丝是西西里国王[[古列尔莫二世_(西西里)|古列爾莫二世]]的姑母，有预言称她的婚姻将毁灭西西里，所以她尚在童年就被囚禁在[[修道院|修道院]]为[[修女|修女]]，直至30岁时与年轻的亨利六世订婚并在2年后成婚。

腓特烈二世3岁时，父亲去世，随母到西西里岛逃避战火，一年后母亲去世，教皇[[依諾增爵三世|英诺森三世]]成为他的监护人。他从小在西西里长大，后来也一直热爱西西里。
[[File:Frederick_II_birth.jpg|thumb]]
康斯坦丝皇后在婚后一直没有孩子，直到40岁才怀孩子，有些不怀好意的人宣称一个屠夫的儿子将被调换成皇子，这对靠血缘世袭的西西里王室而言是十分危险的流言。1194年12月26日，皇后在赶往西西里（皇帝已经于[[圣诞节|圣诞节]]加冕成为西西里国王）途中感到分娩的阵痛，她在耶西的市集上搭起帐篷，为维护孩子正统的继承权力，她允许镇上的女公民观看皇后的分娩，并把赤裸的正滴着乳汁的胸脯给众人看。

在与父亲[[亨利六世_(神圣罗马帝国)|亨利六世]]唯一一次见面——施洗中，被赋予FEDERICO RUGGERO的名称：意寓将继承祖父红胡子（FEDERICO I）和外公[[鲁杰罗二世_(西西里)|罗杰二世]]（RUGGERO）的名字以及他们的丰功伟绩，成为西西里与日尔曼两地之王。
 
1196年12月，父親亨利六世让诸侯选举年幼的腓特烈为[[德国君主列表|罗马人民的国王]]，但同时皇帝企图实现的王位世袭制度却因教廷的强烈反对而失败。但当亨利六世患[[疟疾|疟疾]]突然去世时，腓特烈二世还是个孩子，皇帝的遗嘱（有可能是伪造的）为他规划“理想的未来”。以帝国对教廷的让步保证腓特烈二世得到巨大的帝国。然而皇帝一死，托孤大臣安魏勒的马克瓦尔特 (Markward of Annweiler)就被康斯坦丝皇太后赶出西西里，年幼的腓特烈二世放弃德国的王位。

在康斯坦丝皇太后授意下，腓特烈二世于1198年5月17日加冕为西西里国王。在丈夫去世一年后，44岁的康斯坦丝太后也走到生命的尽头，她于1198年11月28日去世，她在去世前请求强大的教皇[[依諾增爵三世|英诺森三世]]保护这个年幼的国王。因而这位年轻的国王是在教皇的监护下成长起来的，被称做“神父哺育的国王”，德国人则喜欢称他为“阿普利亚的孩子”。

这个儿童虽然戴着王冠，其实却是个时时被监视的囚徒。幼年的腓特烈二世是在不受重视甚至是歧视和虐待中度过的，他在[[西西里王國|西西里]]的首府[[巴勒莫|巴勒莫]]度过了整个童年，有传说6岁的他穿着国王的华服用尖利的手指甲不停地划破绸衣和自己幼嫩的皮肤，有时候连吃饭都成为问题，在[[巴勒莫|巴勒莫]]的街头流浪，依靠富有爱心支持皇帝的家庭的接济。但正是这样悲惨坎坷的童年经历影响腓特烈二世一生，他待人平等，可以同马夫平等交谈问题，他只相信自己，极端地自我，相当地世故。他见多识广，对知识孜孜不倦也是从他童年在市井间的生活开始的。

在他15岁的时候，他的监护人[[依諾增爵三世|英诺森三世]]终于做一件重大的事情，让他结婚。为他选一位新娘，比年轻的国王至少大10岁还是个寡妇，[[阿拉贡王国|阿拉贡]]的公主。

1211年，因为奥托四世的不受控制，英诺森三世提议腓特烈为日尔曼国王，而他的婴儿长子亨利加冕为西西里国王，妻子为王国摄政。于是乎腓特烈踏上前往日尔曼的道路。3月，自巴勒莫出发，于复活节夕到达罗马，与[[依諾增爵三世|英诺森三世]]会面——这也是两者唯一的一次会面。腓特烈宣誓效忠教宗。随后經过热那亚，于7月到达康斯坦茨，并在之后3个月与法国不久后的国王路易八世签订共同对抗奥托四世。1213年7月12日，在教宗的黄金训令中，承托神圣罗马帝国与西西里王国的分离，放弃日尔曼在意大利半岛的一切权力，并承诺进行[[第五次十字军东征|第五次十字军东征]]。教皇满意认为，他已经完全控制腓特烈二世。1214年，奥托四世在[[鲍文思战役|鲍文思战役]]中惨败，这使得腓特烈二世确保神圣罗马帝国的皇位。

== 西西里和十字军东征 ==

等到腓特烈二世成年，他就显示出一种截然非英诺森三世所想的态度。英诺森三世的继任[[何諾三世|霍诺留斯三世]]是一个优柔寡断之人，他使得腓特烈二世能够放手把西西里作为他的政治基地。1220年帝国会议选举腓特烈二世为罗马人民的国王。于是皇帝的地盘把教皇国包围了起来，而这正是历代教皇所极力反对的。总的来说，腓特烈二世对德国的事务反倒不甚关心，他统治的中心是西西里。[[意大利|意大利]]的城市刚刚依靠[[伦巴第同盟|伦巴第同盟]]击退了腓特烈二世的祖父[[腓特烈一世_(神圣罗马帝国)|红胡子腓特烈一世]]，现在他们发现自己正面对一位更恐怖的腓特烈。

1226年，腓特烈二世发给[[条顿骑士团|条顿骑士团]]大团长[[赫尔曼·冯·萨尔扎|赫尔曼·冯·萨尔扎]]一个特许状，准许其征服[[普鲁士|普鲁士]]。萨尔扎是腓特烈二世一生中最信赖的顾问之一。

1227年性格倔强的[[額我略九世|格列高利九世]]当选为教皇，他的后半生都在与腓特烈二世争斗。他命令腓特烈二世立即率领十字军东征。但是腓特烈二世的十字军遇到了瘟疫，以至很快折返。格列高利九世不相信这是事实，他对腓特烈二世处以[[绝罚|绝罚]]。腓特烈二世只好再次出征[[耶路撒冷|耶路撒冷]]。

腓特烈二世领导的十字军是十字军东征史上绝无仅有的一次。他的行动令东方和西方都大吃一惊，然而却得到了最好的效果。实际上，腓特烈二世避免任何战场上的交锋，他通过谈判手段换回了耶路撒冷。[[埃及|埃及]]苏丹阿卡邁爾(Al-Kamil)称赞他为最好的欧洲君主。根据和约，腓特烈二世重建了耶路撒冷王国并自任国王，同时与原国王的女儿[[伊莎贝拉二世_(耶路撒冷)|布里昂的约兰德]]结婚。他的这些史无前例的活动成了格列高利九世反对他的藉口。格列高利九世谴责腓特烈公然与异教徒签约，并派军队进攻西西里([[教皇国|教宗國]]在[[羅馬市|羅馬]]一帶擁有自己的領地和軍隊)。然而，教皇大大低估了腓特烈二世的力量。腓特烈二世迅即班师回意大利击溃教皇军队。双方于1230年休战，格列高利九世解除了对腓特烈的绝罚。

== 与教皇的争斗 ==

腓特烈二世与教皇国的矛盾已经根深柢固。1234年，教皇支持腓特烈二世的儿子罗马人民的国王[[亨利七世(德意志)|亨利七世]]反对自己的父亲。腓特烈二世很快扑灭了叛乱，他废除了儿子的王位，并把他终身监禁在意大利的监狱里（1235年）。他在意大利有一个有力的支持者：[[维罗纳|维罗纳]]长官[[埃泽利诺三世·达·罗马诺|埃泽利诺三世·达·罗马诺]]。在后者支持下，他对意大利南方的反皇帝城市联盟进行征讨（1236年）。1237年，腓特烈二世在科尔泰努瓦大败圣泽诺内联盟。1241年他袭击了[[热那亚|热那亚]]的舰队，俘虏了两名正准备参加教皇召开的，谴责腓特烈二世的会议的[[樞機|红衣主教]]。

新任教皇[[依諾增爵四世|英诺森四世]]表现出一种比格列高利九世还要严厉的态度。1245年，他在[[法国|法国]]的[[里昂|里昂]]对腓特烈二世处以[[绝罚|破門律]]（他选择里昂是为了避开皇帝的直接军事威胁）。腓特烈二世面临着严峻的形势：教皇的制裁损害了他的声望，在那个人民笃信[[天主教會|羅馬公教]]的时代，君主被开除教籍是极其危险的，以前就有不少这方面的例子。更严重的是，[[德意志|德意志]][[诸侯|诸侯]]开始与[[意大利|意大利]][[城市|城市]]联合反对他，这些诸侯一直是困扰每一代[[皇帝|皇帝]]的威胁力量。1246年，[[图林根公国|图林根]][[圖林根統治者列表|公爵]][[亨利|亨利]]在教皇与诸侯支持下当选为敌对国王。在种种困难之下，腓特烈二世在弗萨尔塔战役中被意大利同盟打败，他的儿子被俘。

== 去世和影响 ==

腓特烈二世一直未能找到反对[[依諾增爵四世|英诺森四世]]的可靠同盟。1250年12月，他在一次行猎中突然病倒了，可能是[[中风|中风]]{{来源请求}}。当然也有流言说是被自己的儿子[[曼弗雷迪_(西西里国王)|曼弗雷德]]杀死{{来源请求}}。1250年12月13日，在[[痢疾|痢疾]]的折磨下，腓特烈二世死在了弗奥伦蒂诺城堡（现今[[意大利|意大利]][[福贾省|福贾省]][[聖塞韋羅|圣塞韦罗]]地区），享年56岁。死去时，他穿着[[熙笃会|熙笃会]]修士的白色法衣。

在他死后，遵照[[遗嘱|遗嘱]]他的头衔被如下划分：合法继承人[[康拉德四世|康拉德四世]]获得[[德国君主列表|罗马人民的国王]]和[[西西里君主列表|西西里国王]]称号；[[曼弗雷迪_(西西里国王)|曼弗雷德]]受封[[塔兰托|塔兰托]]，并摄政王国事务；[[亨利七世_(德意志)|亨利七世]]虽然因为叛乱而被剥夺了继承权，但他的子嗣继承了[[奥地利公国|奥地利]][[公爵|公爵]]和[[施泰爾馬克州|施蒂利亚]][[公爵|公爵]]。腓特烈二世的遗愿是，在不影响国家威望的前提下，将所有从教皇手里夺取的土地归还教皇，同时[[大赦|大赦]]天下，减少赋税。然而，康拉德四世在继位后四年就去世，王朝地位岌岌可危。腓特烈二世的这一遗愿最终也未能实现。

[[File:Tomb_of_Frederick_II,_Holy_Roman_Emperor_-_Cathedral_of_Palermo_-_Italy_2015.jpg|left]]
在腓特烈二世去世之后，军队中的[[伊斯兰教|伊斯兰教]][[穆斯林|穆斯林]]们将他收殓，并带到了[[巴勒莫|巴勒莫]]。最终安放他遗体的石棺被放置在[[巴勒莫主教座堂|巴勒莫主教座堂]]的礼拜堂内，与他的父亲[[亨利六世_(神圣罗马帝国)|亨利六世]]和[[西西里国王|西西里国王]][[鲁杰罗二世_(西西里)|羅傑二世]]放在一起。

19世纪时，当时的历史学家们对腓特烈二世的遗体进行了学术调查。石棺中他的尸体穿着[[伊斯兰|伊斯兰]]风格的服装，外衣的衣袖上用[[阿拉伯语|阿拉伯语]]绣着“朋友，宽宏大量的人，诚实的人，富有知识的人，胜利者”。

== 才艺 ==
腓特烈二世给人留下深刻的印象，仿佛他不是个君主而是位学者。他本人掌握了七种[[语言|语言]]：[[德语|德语]]，[[意大利语|意大利语]]，[[法语|法语]]，[[拉丁语|拉丁语]]，[[希腊语|希腊语]]，[[希伯来语|希伯来语]]和[[阿拉伯语|阿拉伯语]]。这在教育比较落后的[[中世纪|中世纪]]是非常令人惊奇的。

由于具有一些新颖的观念，他在发展工商业方面很有成就。腓特烈二世重视教育，他创办了[[那不勒斯大学|那不勒斯大学]]（1224年），在西西里办了一个诗歌学校。皇帝本人还写有一部诗集“猎鸟的艺术”。腓特烈二世把西西里的法律编成法令集。在自然科学方面，腓特烈二世非常重视伊斯兰学者掌握的知识，当时伊斯兰世界的文化水平远远超过了[[欧洲|欧洲]]。他有时会做一些实验，不过这些实验据说变成了暴行（将囚徒在酒桶里慢慢淹死以观察灵魂是否存在，将婴儿由聋哑人抚养以观察语言如何起源），当然这也可能是教皇对他的诽谤。

== 个人生活 ==

腓特烈二世是个有强烈感官刺激欲望的男人，换而言之他有很强烈的性欲。他对女色的追求可能来自于过早地丧失母爱，童年生活在一个完全没有关心和爱的环境中有关。

他爱女人，爱过许多许多女人并为她们写出许多情真意切，感情热烈奔放，甚至相当肉麻的情诗。传说他在年少流浪时便和妓女们打成一片，15岁时便和一位比他至少大十岁的美丽寡妇，前匈牙利的王后[[阿拉貢的康斯坦絲|阿拉貢的康斯坦絲]]结婚了。17岁就有了嫡出的长子，未来的共同统治者[[亨利七世_(德意志)|德意志的亨利]]。在他喜爱的发妻死后，他又结了两次婚，这两位夫人为他生了许多孩子后也都去世了。在他最爱的情人临死的时候他在她的灵床边和她举行了婚礼的仪式。他还有一个奇特的爱好，就和以前的西西里君王一样，喜爱[[阿拉伯|阿拉伯]]的美丽宫妃，他经常带着一些[[阿拉伯|阿拉伯]]女奴和珍禽异兽出游。
