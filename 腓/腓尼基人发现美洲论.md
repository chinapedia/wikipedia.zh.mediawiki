'''腓尼基人发现美洲论'''是[[哥伦布时代前的跨大洋接触|哥伦布时代前的跨大洋接触]]的{{Link-en|边缘理论|Fringe theory}}之一。它声称新世界的人和[[腓尼基|腓尼基人]]或其他[[闪米特人|闪米特人]]在[[前1千年|前1千年]]就有了接触。<ref>{{cite book |last=吕克-诺尔芒·泰利耶尔 |title=''城市世界历史：经济与地理观点'' |year=2009 |publisher=魁北克大学出版社 |page=301 |url=http://books.google.com.au/books?id=cXuCjDbxC1YC&pg=PA301 }} {{en icon}}</ref>这一理论还引向了其他的更为特殊的理论。

==20世纪前的理论==
腓尼基人发现美洲这一理论曾流行于18和19世纪。在18世纪晚期，有一些人曾推测过{{Link-en|戴顿岩|Dighton Rock}}上的[[岩刻|岩画]]的来源。当时的[[耶鲁学院|耶鲁学院]]校长{{Link-en|埃兹拉·斯泰尔斯|Ezra Stiles}}相信它们是希伯来文。<ref>{{cite book |last=弗兰克·爱德华·马努埃尔 |last2=弗里齐·普里戈济·马努埃尔 |title=''詹姆斯·博多因与爱国哲人'' |year=2003 |publisher=[[美国哲学会|美国哲学会]] |pages=197-198 |url=http://books.google.com.au/books?id=vZaY5Vxx0qIC&pg=PA197 }} {{en icon}}</ref>[[塔罗牌|塔罗牌]]的现代使用的发起者{{Link-en|安托万·库尔·德·热伯兰|Antoine Court de Gébelin}}在《原始世界》中主张，它们是古代的一群来自[[迦太基|迦太基]]的水手在造访了马萨诸塞海岸之后留下的纪念。<ref>{{cite news |last=阿尔伯特·B·索斯威克 |title=''戴顿岩之谜'' |url=http://www.telegram.com/article/20130110/NEWS/101109827/1020 |accessdate=2013年1月29日 |newspaper={{Link-en|电报与公报|Telegram & Gazette}} |date=2013年1月10日 }} {{en icon}}</ref>

在19世纪，以色列人造访美洲的信仰在[[摩尔门教|摩尔门教]]中被牢固确立。他们相信[[约瑟_(旧约圣经)|约瑟]]的后人{{Link-en|缪莱克|Mulek}}在逃离[[犹大王国|犹大]]后，去了今天的美洲，并在那里建立了一个新的文明。{{Link-en|罗斯·泰勒·克里斯滕森|Ross T. Christensen}}提出理论说，《[[摩尔门经|摩尔门经]]》中的缪莱克一族人“大多数人的种族来源都是腓尼基人”。<ref>{{cite web |last=罗斯·泰勒·克里斯滕森 |title=''腓尼基人与美洲古文明'' |url=http://www.ancientamerica.org/library/media/HTML/1ofnf5kk/15.%20THE%20PHOENICIANS%20AND%20THE%20ANCIENT%20CIVILIZATIONS%20OF%20AMERICA.htm?n=0 |accessdate=2013年1月29日 }} {{en icon}}</ref> 

{{Link-en|约翰·丹尼森·鲍德温|John Denison Baldwin}}在他1871年的书《古代美洲》中说：
<blockquote>已知的腓尼基族的进取心，和对美洲的这一古老的了解，被表达得如此地多方面，这强烈地鼓励了人们假设称，腓尼基人来到了这片大陆，找到了变为了废墟的城市，并在该地区建立了殖民地，给那里填充了开化的生活。被坚决主张的是，他们航行于“大外洋”，并且如此的航海家们一定穿越了大西洋；此外，类似于腓尼基人的象征性图案的东西在美洲的废墟中被找到了，并且当地的墨西哥人和中美洲人的老传统描述说，第一批开化者是“有胡子的白人”，他们“乘船从东方来”。<ref>{{cite web |last=约翰·丹尼森·鲍德温 |title=''腓尼基人论'' |url=http://www.djmcadam.com/phoenician-theory.html |accessdate=2013年1月29日}}  {{en icon}}</ref>
</blockquote>

在19世纪70年代，在[[巴西|巴西]][[帕拉伊巴|帕拉伊巴]]有一处石头铭文被发现了；从那时起，对它的各种各样的抄写和解释总是会定期地广泛地引发人们的想象。<ref>《帕拉伊巴铭文》上了1968年6月10日的《[[生活_(杂志)|生活]]》，评注者是居鲁士·赫茨尔·戈登。</ref>有一份抄写本被{{Link-en|巴西国家博物馆|National Museum of Brazil}}的负责人{{Link-en|拉迪斯劳·德·索扎·梅洛·内托|Ladislau de Souza Mello Netto}}看见了。内托相信了该铭文是真的并接受了它，但是后来它又被宣称是一场骗局，于是内托退还了它并指责了那些制造它的外国人。不过，在20世纪60年代，{{Link-en|居鲁士·赫茨尔·戈登|Cyrus H. Gordon}}宣称该铭文是真的，并提供了一份译本，它的开头是：“我们是来自贸易国王城市的西顿迦南人……”<ref>居鲁士·赫茨尔·戈登, ''来自巴西的迦南文本'', 出自: ''东方史料'', 第37期, 1968, 第425-436页.</ref>

==20世纪的理论==
[[File:Bat_Creek_Inscription_1890_Lithograph_Figure_7_Inverted.jpg|right]]
在20世纪，该理论的支持者包括居鲁士·赫茨尔·戈登、{{Link-en|约翰·菲利普·克黑恩|John Philip Cohane}}、罗斯·泰勒·克里斯滕森、{{Link-en|巴里·费尔|Barry Fell}}及{{Link-en|马克·麦克梅纳明|Mark McMenamin}}。戈登相信，古希伯来铭文在美国东南部的两个地方被找到了，这暗示了犹太人在[[哥伦布|哥伦布]]之前就抵达了那里。这一推测相关的发现品之一便是{{Link-en|巴特溪铭文|Bat Creek inscription}}。戈登相信那些铭文是[[古希伯来字母|古希伯来文]]，但是那些铭文被普遍认为是伪造的。<ref>[http://ramtops.co.uk/bat1.html 小罗伯特·C·梅恩福特; 玛丽·L·夸斯, ''巴特溪石：田纳西犹太人？'' 出自: ''田纳西人类学者'' 第16卷, 第1期, 1991年春.] {{webarchive|url=https://web.archive.org/web/20070816103039/http://ramtops.co.uk/bat1.html |date=2007-08-16 }} {{en icon}}</ref>另一个被宣称支持闪米特人发现美洲论的发现品是{{Link-en|洛斯卢纳斯诫文石|Los Lunas Decalogue Stone}}，它也被视为了一件伪造品。

在1996年，马克·麦克梅纳明提出了一个理论说，腓尼基水手大约在公元前350年发现了新世界。<ref>{{cite web|title=''麦克梅纳明为有争议的理论提供了新的证据''|url=http://www.mtholyoke.edu/offices/comm/csj/990108/mcman.html|publisher=学院街杂志|accessdate=2013年1月29日|deadurl=yes|archiveurl=https://web.archive.org/web/20130301003714/https://www.mtholyoke.edu/offices/comm/csj/990108/mcman.html|archivedate=2013年3月1日}} {{en icon}}</ref>腓尼基城邦迦太基在公元前350年铸造了金{{Link-en|斯塔忒耳|stater}}。对于这些钱币背面的[[硬幣#刻记|刻记处]]上的图案，麦克梅纳明将之解释为 一张地中海的地图，并且在其中，美洲被显示为向西跨过了大西洋之处。<ref>马克·A·麦克梅纳明 1996. ''迦太基人的地图制作：一幅风格化的刻记处地图'' 明马出版社, 南哈德利, 马萨诸塞.</ref><ref>马克·A·麦克梅纳明 1997. ''腓尼基世界地图：墨卡托的世界'' 2 (3): 46-51.</ref><ref>J·M·司各特 2005. ''早期犹太教与基督教中的地理学'' 剑桥大学出版社, 第182-183页.</ref><ref>{{cite web |last=凯文·麦卡弗里 |title=''谁发现了美洲？'' |url=http://www.mtholyoke.edu/offices/comm/vista/9606/4.html |accessdate=2013年1月29日 }} {{en icon}}</ref>麦克梅纳明之后证实了在美国找到的所谓的迦太基钱币其实是现代的伪造品。<ref>{{cite book |author=麦克梅纳明 |title=腓尼基人、骗子及巴里·费尔：解答美国发现迦太基钱币之谜 |date=2000 |publisher=明马出版社 |place=南哈德利, 马萨诸塞 |ISBN=1893882012 }}</ref>

关于腓尼基人或迦南人或迦太基人出现在新世界的各种各样的理论都在被争论着，它们的证据在1979年被{{Link-en|马歇尔·麦卡西克|Marshall McKusick}}在《圣经考古学家》上评论并驳回；<ref>麦卡西克, ''美洲迦南人：石头上的新圣典？'' 出自: ''圣经考古学家'' '''42'''.3 (1979年夏), 第137页起.</ref>他评述说，“在这年头，每个人都想成为自己的权威，而个人在文化选择上的搜寻，似乎使得每一个想法或理论在价值上都是平等的。”

==最近的发展==
在2012年7月，英国人菲利普·比尔宣告计划驾驶他复制的腓尼基小船穿越大西洋以驗證可能性，并且比尔事先已经用他的小船环航了非洲。<ref>{{cite news |last=保罗·肯多尔 |title=''英国冒险者计划驾驶腓尼基航船复制品穿越大西洋'' |url=http://www.telegraph.co.uk/news/worldnews/northamerica/usa/9434701/British-adventurers-plan-to-sail-replica-of-Phoenician-sailing-boat-across-Atlantic.html |accessdate=2013年1月30日 |newspaper=[[每日电讯报|每日电讯报]] |date=2012年7月29日 }} {{en icon}}</ref>

在2013年1月，[[历史频道|历史频道]]播放了一集名为《美国巨石阵》的《{{Link-en|美国出土|America Unearthed}}》，这一集讲了{{Link-en|美国巨石阵遗址|America's Stonehenge}}，并论述了腓尼基人殖民北美洲的理论。<ref>{{cite web |last=诺曼·伯德 |title=''美国出土：这儿有巨石阵，那儿有巨石阵，哪儿都有腓尼基人'' |url=http://www.huliq.com/12092/america-unearthed-stonehenge-here-stonehenge-there-phoenicians-everywhere |publisher=www.huliq.com网站 |accessdate=2013年1月29日 }} {{en icon}}</ref>

==学术评价==
格伦·马科说，到底腓尼基人是否曾抵达了美洲“也许永远不会被知晓”。他评论道：
<blockquote>铭文形式的证据，比如在巴西北部的帕拉伊巴找到的著名的所谓的腓尼基文本，依然是不太像是真的。而另一块详述了一群来自西顿的人因暴风的驱使而登陆的铭文石，很早前就被认为是一件聪明的伪造品了。如果这样的一场决定性的远征真地发生过，那么要找证据的话，则更有可能找到的是一些腓尼基人的陶器碎片。<ref>{{cite book |last=格伦·马科 |title=''腓尼基人'' |year=2000 |publisher={{Link-en|加利福尼亚大学出版社|University of California Press}} |page=13 |url=http://books.google.com.au/books?id=smPZ-ou74EwC&pg=PA13 }} {{en icon}}</ref>
</blockquote>
