'''当卢'''，又名'''钖'''，是[[中国|中国]]古代[[马具|马具]]的一种，缀于[[笼头|笼头]]之上，置于[[马|马]][[额头|额头]]中央。质地以[[铜|铜]]为主，亦有[[金|金]]质、[[银|银]]质、[[骨|骨]]质等。[[商代|商代]]就已经出现，[[秦朝|秦]][[汉朝|汉]]最为流行，[[唐朝|唐]][[宋朝|宋]]以后逐渐演变为“面帘”。其起源可能受到[[面具|面具]]的影响，由于对马脸兼备装饰和防护两项功能，因而长期流行不衰。最初本只是简单的[[几何图形|几何图形]]，如[[河南|河南]][[安阳|安阳]]大司空村出土的商代当卢，仅为一铜质圆泡，不加装饰；到秦汉时期，正面多装饰花纹，还有[[鎏金|鎏金银]]等工艺，形状也逐渐拉长，甚至衍变出马面形象。<ref>{{cite book |year=1996 |author=顾朴光 |title=中国面具史 |publisher=贵州民族出版社 |location=贵阳 |page=151、152 |ISBN=7-5412-0531-1}}</ref><ref>{{cite book |year=1998 |author=秦勇，周汉 编著 |title=个人收藏 古玩类 |publisher=河北人民出版社 |location=石家庄 |page=56 |ISBN=7-202-02254-0}}</ref><ref>{{cite book |year=2014 |author=王巍 总主编 |title=中国考古学大辞典 |publisher=上海辞书出版社 |location=上海 |page=30 |ISBN=978-7-5326-4027-0}}</ref>

'''钖'''是[[先秦|先秦]]时代的称呼，又作'''鐊'''。《[[说文解字|说文解字]]》：“鐊，馬頭飾也。”<ref>{{cite web |url = http://ctext.org/shuo-wen-jie-zi/jin-bu3/zh | title=许慎 说文解字·卷十五·金部 }}</ref>《[[诗经|詩]]·大雅·韓奕》：“鉤膺鏤錫”。<ref>{{cite web |url = http://ctext.org/book-of-poetry/han-yi/zh | title= 詩經·大雅·蕩之什·韓奕 }}</ref>[[东汉|东汉]][[郑玄|郑玄]]笺注此句说：“眉上曰鍚，刻金飾之，今當盧也。”<ref>{{cite web |url = http://ctext.org/library.pl?if=gb&file=77331&page=116| title= 毛詩·大雅·蕩之什·韓奕 影印本}}</ref>可知汉代已称之为当卢。唐《[[慧琳音义|慧琳音义]]》解释“当卢”的[[语源学|词源]]说：“當盧，字宜作[[头颅|顱]]……言馬面當顱。”<ref>释惠琳 一切經音義·卷五十三·樓炭經·第二卷</ref>指出“卢”字是“颅”的[[通假字|通假字]]。

[[印度|印度]][[喀拉拉邦|喀拉拉邦]]有一种置于[[大象|大象]]前额的[[金属|金属]]饰品，称为"Nettipattam”（നെറ്റിപ്പട്ടം）与当卢有相通之处。
