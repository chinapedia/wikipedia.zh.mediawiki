'''氨甲酰磷酸<ref>[http://www.term.gov.cn/pages/homepage/result2.jsp?id=334719&subid=10001798&subject=%E6%96%B0%E9%99%88%E4%BB%A3%E8%B0%A2&subsys=%E7%94%9F%E7%89%A9%E5%8C%96%E5%AD%A6%E4%B8%8E%E5%88%86%E5%AD%90%E7%94%9F%E7%89%A9%E5%AD%A6]{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>'''（{{lang-en|Carbamoyl phosphate}}）是一种在[[生物化学|生物化学]]方面有重要意义的[[阴离子|阴离子]]。在陆生生物体内，氨甲酰磷酸作为一个中间代谢物参与通过[[尿素循环|尿素循环]]进行的[[氮|氮]]排泄以及[[嘧啶|嘧啶]]的[[生物合成|生物合成]]。

此物质由[[二氧化碳|二氧化碳]]、[[氨|氨]]（衍生自[[谷氨酸|谷氨酸]]）以及[[磷酸|磷酸]]（来自[[腺苷三磷酸|腺苷三磷酸]]）经[[氨甲酰磷酸合成酶|氨甲酰磷酸合成酶]]催化而合成，如下：
* HCO<sub>3</sub><sup>−</sup> + ATP → ADP + HO–C(O)–OPO<sub>3</sub><sup>2−</sup>（[[羰基磷酸酯|羰基磷酸酯]]）
* HO–C(O)–OPO<sub>3</sub><sup>2−</sup>  +  NH<sub>3</sub>  + OH<sup>−</sup>   →  HPO<sub>4</sub><sup>2−</sup>  +  [[Carbamate|<sup>−</sup>O–C(O)NH<sub>2</sub>]]  +  H<sub>2</sub>O
*<sup>−</sup>O–C(O)NH<sub>2</sub>  +  ATP  →  ADP +  H<sub>2</sub>NC(O)OPO<sub>3</sub><sup>2−</sup>
