'''徽州区'''是[[中國|中國]][[安徽省|安徽省]][[黄山市|黄山市]]下辖的一个区。1987年由原[[歙县|歙县]]西部地区析置。

== 历史 ==
1987年11月成立黄山市，同时设立[[市辖区|市辖区]]徽州区。<ref>{{Cite web |url=http://www.huizhouqu.gov.cn/news_detail.asp?w_id=265&lbid=100101 |title=徽州简介 |accessdate=2011-01-08 |publisher=徽州区人民政府网站 |deadurl=yes |archiveurl=https://web.archive.org/web/20100630193638/http://www.huizhouqu.gov.cn/news_detail.asp?w_id=265&lbid=100101 |archivedate=2010-06-30 }}</ref>

==行政区划==
徽州区下辖1个街道、4个镇、3个乡：
* 街道：[[徽州街道|徽州街道]]
* 镇：[[岩寺镇|岩寺镇]]、[[西溪南镇|西溪南镇]]、[[潜口镇|潜口镇]]、[[呈坎镇|呈坎镇]]
* 乡：[[富溪乡|富溪乡]]、[[杨村乡|杨村乡]]、[[洽舍乡|洽舍乡]]。

== 地理 ==
地势北高南低。北部为黄山余脉，以[[山地|山地]]为主，建有[[丰乐水库|丰乐水库]]。南部以[[丘陵|丘陵]][[平原|平原]]为主。

== 教育 ==
2008年徽州区有高级中学、职业中学各1所，初级中学6所，小学57所。小学、初中学龄儿童入学率分别为99.72%、98.04%，15周岁人口初等教育完成率99.94%。<ref>{{Cite web |url=http://www.hzedu.com.cn/hzedu/hzjy/ShowInfo.asp?InfoID=218 |title=徽州区教育概况 |accessdate=2011-01-08 |publisher=徽州教育网 |date=2008-12-05 }}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>
