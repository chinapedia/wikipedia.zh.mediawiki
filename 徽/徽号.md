'''徽號'''，是指[[太皇太后|太皇太后]]、[[皇太后|皇太后]]、[[太妃|太妃]]、[[先朝嬪御|先朝嬪御]]、[[嬪妃|嬪妃]]等後宮女性所使用的[[尊號|尊號]]，[[皇后|皇后]]因為與[[皇帝|皇帝]]一樣地位極尊，僅一人，所以除了極少數例子以外，一般不上徽號。

== 中國 ==
后妃的徽號使用，隨著中國歷代後宮典制的演變，都有不同的變化。

=== 明朝以前 ===
在[[明|明]][[清|清]]兩朝以前，嬪妃的稱號是將她的姓氏加上她的[[位號|位號]]而成的，如唐朝[[唐玄宗|唐玄宗]]寵妃—[[楊貴妃|楊貴妃]]，她本姓楊，[[貴妃|貴妃]]則是她在宮中的位號，她的姓氏後面加上她在宮中的位號即成為她的稱號。

[[明|明]][[清|清]]兩朝以前，徽號的使用沒有很嚴密的規制，常常出現許多模稜兩可的稱號和徽號，如；

*[[南朝宋|南朝宋]][[劉義隆|文帝劉義隆]]之[[妃嬪|妃嬪]][[沈容姬|沈容姬]]，為明帝[[劉彧|劉彧]]之母；死後先後被追諡號為'''湘東國太妃'''及'''宣太后'''
*[[后趙|后趙]]武帝[[石虎|石虎]]的[[皇后|皇后]][[鄭櫻桃|鄭櫻桃]]；被石虎加封為'''天王皇后'''，之後因其子[[石邃|石邃]]荒淫無道，鄭櫻桃受到牽連而被廢-{后}-位，並降之為'''東海太妃'''
*[[北周宣帝|北周宣帝]]的[[皇后|皇后]][[朱滿月|朱滿月]]；曾被兒子[[北周靜帝|北周靜帝]]尊為'''帝太后'''

=== 明清两朝 ===
==== 皇后 ====
就理論上而言，後宮皇后只能有一位，因為已經地位極尊，所以皇后生前一般是不用上徽號區分的(皇后崩逝後，才需加上[[謚號|謚號]])。

基本上，當皇后會被恭上[[徽號|徽號]]，有兩種狀況：
* 皇后和繼任皇帝之間為[[平輩關係|平輩關係]]（大多是[[叔|叔]][[嫂|嫂]]關係），就因為是平輩，因而繼任皇帝的皇后嫂嫂，無法被尊為皇太后，而是由繼任皇帝上徽號，稱○○[[皇后|皇后]]，如；
**[[汉朝|汉朝]]：[[孝惠張皇后|孝惠皇后]]
**[[晋朝|晋朝]]：[[羊獻容|惠皇后]]
**[[南北朝|南北朝]]：[[李祖娥|昭信皇后]]
**[[宋朝|宋朝]]：[[孝章皇后|开宝皇后]]、[[昭慈圣献皇后|元祐皇后]]、[[昭懷皇后|元符皇后]]
**[[明朝|明朝]]；[[孝靜毅皇后|莊肅皇后]]、[[孝哀悊皇后|懿安皇后]]
**[[清朝|清朝]]；[[孝哲毅皇后|嘉順皇后]]。
上述是明清兩朝的情況，而在明清兩朝以前，為了使先朝皇后和當朝皇后之間有所區別和，多會以大行皇帝的最後一個年號來稱呼該皇后。
* 還有一種情形皇后會擁有徽號，那就是亡國皇后、皇太后，或是被廢位的皇帝之皇后、皇太后，她們因為原本的國家亡覆（或是未改朝，但皇帝被廢位），在新執政者登基為帝時，如果她們未在戰亂中失蹤或被殺害，幾乎都會被新的君王加上徽號，有些甚至會被降級，皇后多被降為夫人、王太妃、或更低階的品級。而皇太后則多降為皇后，如；
**[[西漢|西漢]][[漢平帝|漢平帝]]的[[皇后|皇后]][[孝平王后|王氏]]；王氏是[[王莽|王莽]]之長女，在王莽竄漢建[[新朝|新朝]]后，她從原本的漢朝皇后改封為新朝的'''黃皇室主'''（王莽建制，改[[公主|公主]]號為'''室主'''），之後在[[新朝|新朝]]覆亡時，她在[[未央宫|未央宮]]中投火自盡而亡。
**[[東漢|東漢]][[汉献帝|漢獻帝]]的[[皇后|皇后]][[獻穆皇后|曹節]]；漢獻帝被迫退位後封為山陽公，其妻曹節也被降為'''山陽公夫人'''
**[[晉惠帝|晉惠帝]]的[[皇后|皇后]][[羊獻容|羊獻容]]；[[西晋|西晉]]滅亡-{zh-hans:后;zh-hant:後}-，她被[[刘曜|前趙末帝]]立為皇后，並稱之為'''惠皇后'''
**[[蜀漢|蜀漢]]懷帝[[劉禪|劉禪]]的皇后[[張皇后_(劉禪)|張氏]]；在[[劉禪|劉禪]]遜位後，[[劉禪|劉禪]]被降為安樂縣公，而張氏也被降為'''安陽縣夫人'''
**[[齐文宣帝|北齊文宣帝]]的[[皇后|皇后]][[李祖娥|李祖娥]]；曾被加封為'''可賀敦皇后'''，之後因[[齐孝昭帝|高演]]篡位為孝昭帝，又將之降為'''昭信皇后'''
**[[刘宋|劉宋]][[刘义符|少帝]]的[[皇后|皇后]][[司馬茂英|司馬茂英]]；其夫[[劉義符|劉義符]]被廢位後，她被降為'''營陽王妃'''，之後又改稱為'''南豐王太妃'''
**[[刘宋|南朝宋]][[宋明帝|明帝劉彧]]的[[皇后|皇后]][[王貞風|王貞風]]；在[[蕭道成|蕭道成]]竄朝建立[[南齐|南朝齊]]之後，將之降為'''汝陰王太妃'''。

此外，在元朝時期，[[皇后|皇后]]可以有十數位之多，眾皇后之間的等級和身分貴賤是以[[斡兒朵|斡兒朵]]（宮帳）來區別，一座斡兒朵裡面會同時居住好幾位皇后和[[妃子|妃子]]，在眾多斡兒朵之中，是以第一斡兒朵中的第一皇后為所有皇后之尊。

==== 嬪妃 ====
皇后以下的[[妃嬪|妃嬪]]眾多，皆以徽號區分辨別。(除了[[清朝|清朝]]的[[皇貴妃|皇貴妃]]以外，因為只有一名以示尊貴，則不用加[[徽號|徽號]]區分，皇貴妃崩逝以後才加[[謚號|謚號]])

*[[明朝|明朝]]妃的号不再有严密的限制，妃嫔也会被赐予专属的徽號，是將寓意美好，吉祥等的徽號加在姓氏之後位號之前。如[[明世宗|明世宗]]的[[杜康妃_(明世宗)|妃子杜氏]]，杜氏的位號是[[妃|妃]]，杜是她的娘家姓氏，而她的徽號則是'''康'''字，因此她在宮中的全稱為'''杜康妃'''。

*明朝不同皇帝的妃嫔可以在同一时期使用完全相同的徽號，如[[明穆宗|明穆宗]]和其子[[明神宗|明神宗]]都有姬妾王[[荣妃|荣妃]]。明朝妃嫔若在皇帝之前去世，同一皇帝新封的妃嫔也可重复使用她用过的徽號，如明神宗的[[常順妃_(明神宗)|常顺妃]]去世后，又有李氏成为[[李順妃_(明神宗)|李顺妃]]。同一时期不同等级的妃嫔也可使用相同徽號，如嘉靖帝的[[文贵妃_(明世宗)|文敬妃]]和[[李敬嫔_(嘉靖帝)|李敬嫔]]。

*[[清朝|清朝]]時，因後宮嬪妃等多為滿族或蒙古族人，這兩個民族所使用的姓氏即使音譯成漢字，仍然字數較多，因此無法像明朝一樣在姓氏後面在加上徽號和位號，所以清朝的嬪妃稱號，省去了嬪妃的姓氏，只在位號前加上徽號，如：光緒帝嬪妃[[珍妃|珍妃]]和[[瑾妃|瑾妃]]，兩人姓他他拉氏，為同父異母的姊妹，入宮後皆受冊封為[[妃|妃]]，而個別以'''珍'''字和'''瑾'''字為徽號。

*一般來說，清朝時期的後宮妃嬪徽號皆為一字，而在她升級或降級後，雖然位號改變，但徽號多不變。但也有不少人在低階嬪妃晉升為中高階嬪妃時更改徽號，或是在晉升後才有徽號，尤其答應與常在有許多在史書上只以姓氏稱呼。

*但也有部分姓氏字數較少的嬪妃，雖然有正式的徽號，但一般人們還是以她的姓氏加上位號做為她的稱呼，如[[清世祖|清世祖]]寵妃—[[董鄂妃|董鄂妃]]，在清史稿—后妃列傳中，她的徽號為'''賢'''字，位號為[[妃|妃]]，所以如果依慣例，她應該被稱為'''賢妃'''而不是'''董鄂妃'''，但當她在宮中的同時，宮中還有一位她的親族董鄂氏，同樣的這另一位董鄂妃的位號也是妃，但她的徽號是'''貞'''字，因此後世為了將兩位董鄂妃有所區別，將這兩位董鄂氏個別依徽號稱之為'''董鄂賢妃'''和'''董鄂貞妃'''。

==== 太妃 ====
當繼任皇帝登基之後，有部分前任皇帝的遺孀，能有幸的被尊封為[[太妃|太妃]]。

[[清朝|清朝]]一般在尊封[[前朝妃嬪|前朝妃嬪]]為太妃的時候，多會一同加上徽號，而太妃的徽號一般多為兩字，如[[清朝|清朝]][[光緒帝|光緒帝]]的[[妃子|妃子]][[瑾妃|瑾妃]]，她在[[宣統|宣統]]朝被尊封為太妃，同時她的徽號是'''端康'''二字，因此她的全稱為'''端康皇贵太妃'''（可簡稱為'''端康太妃'''）。

而未被尊封為太妃的[[先朝嬪御|先朝嬪御]]，在新皇帝登基后，依然保有和使用她們原本的舊稱號，如[[清朝|清朝]][[康熙帝|康熙帝]]的[[妃子|妃子]][[定妃_(康熙帝)|定妃]]，她在康熙帝生前所冊封到最高階的位號只有妃，而她長壽地一直活到[[乾隆|乾隆]]年間，這期間她除了被[[雍正帝|雍正帝]]尊封為'''皇考定妃'''之外，沒有更高階的尊封。

==== 皇太后 ====
當女子成為[[皇太后|皇太后]]時，大多是因她們的皇帝丈夫駕崩，因而被繼任的皇帝依之間的輩分關係及當時的局勢而被尊為皇太后，當被尊為皇太后時，繼任皇帝都會恭上徽號。

* 亡國（或被降級）皇太后
** [[西漢|西漢]][[漢元帝|漢元帝]]的[[皇后|皇后]][[王政君|王政君]]，[[王莽|王莽]]為其侄子，王莽篡漢建[[新朝|新朝]]，將當時已身為[[太皇太后|太皇太后]]的王政君改稱號為—'''新室文母太皇太后'''。
** [[西漢|西漢]][[漢成帝|漢成帝]]的[[皇后|皇后]][[趙飛燕|趙飛燕]]；漢成帝駕崩-{zh-hans:后;zh-hant:後}-，她支持以[[漢哀帝|漢哀帝]]繼位，因而在漢哀帝即位後被尊為皇太后，但漢哀帝駕崩-{zh-hans:后;zh-hant:後}-，她失去政治依靠，因而被降為'''孝成皇后'''，之後又降為[[庶人|庶人]]，最終自盡而亡。
** [[南宋|南宋]][[宋理宗|宋理宗]]的[[皇后|皇后]][[謝道清|謝道清]]；南宋滅亡-{zh-hans:后;zh-hant:後}-，她曾抱著年僅五歲的[[宋恭帝|宋恭帝]]向[[元朝|元朝]]大軍投降，之後被元朝降為'''壽春郡夫人'''，是'''中國歷史上亡國皇太后降級降得最低的一位'''，僅僅是一位'''郡夫人'''而已。

==== 太皇太后 ====
一般而言，能夠成為[[太皇太后|太皇太后]]並非易事，因此中國歷朝所出現的太皇太后之人數並不多，而能夠成為太皇太后，在這之必先經過[[皇太后|皇太后]]這一階級，因此待成為太皇太后時，並未再恭上新的徽號，而是在原本的皇太后徽號加上新字，如清朝開國皇帝清太宗之妃（同時也是清朝首位皇太后及太皇太后{{efn|清朝第二位也是最後一位太皇太后，是清末期的[[慈禧太后|慈禧太后]]，當她得知[[光緒帝|光緒帝]]駕崩後，便詔命[[溥儀|溥儀]]繼位，她也因此勉強成為清朝第二位太皇太后，但她甚至來不及等到溥儀正式登基及為她加上太皇太后徽號，就在[[光緒帝|光緒帝]]駕崩的第二天早上薨亡。}}）—[[孝莊文皇后|孝莊文皇后]]，她在[[順治|順治]]朝被尊為[[皇太后|皇太后]]，而一直到[[順治帝|順治帝]]駕崩前，她所受尊封的徽號為—'''昭聖慈壽恭簡安懿章慶皇太后'''，之後在[[康熙|康熙]]朝，她被尊為太皇太后，並加新徽號為—'''昭聖慈壽恭簡安懿章慶敦惠太皇太后'''，此後一直到她死前，她所受尊封的徽號為—'''昭聖慈壽恭簡安懿章慶敦惠溫庄康和仁宣弘靖太皇太后'''。

==== 嫡庶之分 ====
皇位的繼承，有時會因為皇帝非前任皇帝之皇后所生、乃為前朝嬪御所生，因而出現'''一朝兩太后並尊'''（或稱'''兩宮並尊'''）的情形，及一位是'''先朝皇后'''（繼任皇帝需認這位先朝皇后為'''嫡母'''），另一位為自己的'''生母'''（即大行皇帝的遺孀之一），這時，先朝皇后為'''嫡'''、皇帝生母為'''庶'''，因此在徽號和[[諡號|諡號]]的使用也有區別。

在清朝，兩宮並尊時的兩宮太后，會將；

'''先朝皇后'''為'''母后皇太后'''；庶出的繼位皇帝，需認先朝皇后為'''嫡母'''，因此稱'''母后'''

'''皇帝生母'''稱'''聖母皇太后'''；皇帝為天之子、是聖人，因此皇帝生母當然也是'''聖母'''
*徽號

[[明朝|明朝]]宮制，繼位皇帝需尊先朝皇后為皇太后，但若是以庶出身份繼位的話（及先朝皇后非繼任皇帝生母，繼位皇帝生母乃先朝的普通嬪御），則將皇帝生母及先朝皇后並尊為皇太后，再為先朝皇后恭上徽號（皇帝生母無徽號），以區嫡庶。

不過在[[明神宗|明神宗]]朝，因明神宗生母—[[李貴妃_(明穆宗)|明穆宗朝李貴妃]]並非皇后，而先朝皇后—[[陳皇后_(明穆宗)|陳皇后]]仍在，所以出現兩宮並尊的情形，原本神宗生母是無徽號的，但當時的太監[[馮保|馮保]]欲討神宗生母之歡心，因此和[[張居正|張居正]]連同，由[[張居正|張居正]]上奏章，請神宗為生母恭上徽號，之後神宗生母為'''慈聖皇太后'''、陳皇后為'''仁聖皇太后'''，從此皇太后徽號再也無嫡庶之分。

[[清朝|清朝]]時，兩宮並尊的皇太后，並沒有以徽號來區別兩宮太后之間的嫡庶。

== 朝鲜半岛 ==
=== 李氏朝鮮 ===
古代朝鮮為中國附屬國，君主正妻只稱王妃而不稱-{后}-，過世後被追諡為王后，高宗宣佈獨立之後，追封先代王后為皇后。

==== 王妃 ====
[[朝鮮王朝|朝鮮王朝]]的王妃過世後被追諡為○○王后之外，也會上徽號、[[陵號|陵號]]，而且，每個字都有它的含意。例如[[定順王后|定順王后]]，諡號是'''定順'''（純行不爽曰定, 和比于理曰順。），徽號是'''端良齊敬'''（守禮執義曰端, 中心敬事曰良, 執心克莊曰齊, 夙夜儆戒曰敬。）。

==== 嬪御 ====
[[朝鮮王朝|朝鮮王朝]]時期[[嬪御|嬪御]]的正式稱號是位號再加上姓氏某氏，例如中宗的[[熙嬪洪氏|熙嬪洪氏]]、[[淑媛李氏|淑媛李氏]]，但有時在非正式文書中會先稱姓氏再加位號，如[[張禧嬪|張禧嬪]]。

此外，除去[[嬪|嬪]]以外的其餘位號之女子沒有徽號，如[[淑媛李氏|李淑媛]]，李是她的姓氏，淑媛是位號。而嬪的徽號是由王妃所擬定的。

==== 王大妃 ====

== 日本 ==
日本後宮並沒有像朝鮮王朝或中國後宮一般，有完整的宮制在使用徽號，相反的，日本後宮眾后妃的稱號有著相當大的彈性。

# 依照該后妃的出生地或籍貫以稱之，如高倉[[女御|女御]]、堀河[[中宮|中宮]]。
# 冠之以該后妃所居之殿舍名（但居住在內裏五舍的后妃則冠之以該舍的別稱），如[[弘徽殿|弘徽殿]]女御、藤壺更衣。
# [[齋王|齋王]]后妃；這是一個比較特殊的情況，及該后妃在入宮前曾擔任過[[伊勢齋宮|伊勢齋宮]]或[[齋院|賀茂齋院]]，因此在入宮之後，該后妃會被冠上齋王的名號，如[[篤子內親王|齋院女御]]，齋王更衣。
# [[女院|女院]]；在[[平安時代|平安朝]][[一條天皇|一條天皇]]時期創立了女院制之後，雖然這些女院已不再身在宮中，但當被人們提起時，都會以院號來稱呼。

在比較不正式的情況之下，為了將眾多的后妃有所區別，後人還是會直接稱呼該后妃的本名，或是先稱本名、再於名字後面加上位號（也有反過來的情形），如[[白河天皇|白河天皇]][[中宮|中宮]][[藤原賢子|藤原賢子]]，就可被稱為賢子中宮或中宮賢子。

== 越南 ==
[[越南|越南]][[阮朝|阮朝]]國王的妻子在國王生前不能封后，只能封為妃嬪，由最高級的一階妃（[[越南文|越南文]]：{{lang|vi|nhất giai phi}}）至第九級的[[才人|才人]]（越南文：{{lang|vi|cửu giai tài nhân}}，漢字：九階才人）<ref>[http://www.4dw.net/royalark/Vietnam/titles.htm 越南贵族尊号]</ref>，有些亦有徽號。[[保大帝|保大帝]]的正妻[[南芳皇后|南芳皇后]]是第一位君主在位就封后的君主配偶。「南芳」就是她的徽號。
