'''孖寶'''（{{lang|en|'''Double'''}}）為賽馬博彩的一個[[彩池|彩池]]，澳門賽馬會或香港賽馬會接受投注投注者需於指定的連續兩場賽事均選中第一名馬匹便可獲得「正獎」。如果在指定的連續兩場賽事中，第一關選中第一名馬匹及第二關選中第二名馬匹亦可得到「安慰獎」。在澳洲、紐西蘭及美國賽馬有類似的彩池稱為「Daily Double」<ref> What is daily double?https://extra.betamerica.com/betting-info/how-to-bet/daily-double-bet</ref> <ref> daily double wikipedia [https://en.wikipedia.org/wiki/Daily_double Daily double]</ref> ，開放首兩場及尾兩場賽馬事投注。
[[File:HKJC_Double.jpg|right]]

== 投-{注}-金額 ==
以香港及澳門為例，最少每注港幣或澳門幣10元。

== 中獎方法 ==
*正獎﹕在指定的連續兩場賽事均選中第一名馬匹。
*安慰獎﹕在指定的連續兩場賽事中，第一關選中第一名馬匹及第二關選中第二名馬匹。

== 有關香港賽馬孖寶的派彩紀錄 == 
紀錄於2012年5月12日產生，頭關金威勝及尾關亮亮的。每港幣$10派$77,825.5。<ref>東網派彩紀錄 http://racing.on.cc/racing/sta/current/rjstai0001x0.html</ref>
