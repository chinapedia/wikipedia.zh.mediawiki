'''熱河省'''，為[[中國共產黨|中國共產黨]]在1945年9月成立的省級行政區。[[中華人民共和國|中華人民共和國]]建國後，於1955年7月30日撤銷該省，前後存在10年。

== 政区沿革 ==
熱河省南部在抗日戰爭時為冀熱遼根據地的一部分。1945年8月8日蘇對日宣戰後，冀熱遼軍中路部隊速配合蘇軍攻下承德，並於9月間成立熱河省政府，省會為承德。1946年8月，國軍佔領承德，省政府遷至林西，此時轄有4專區、1盟、28縣、12旗。1947年6月，省會遷至赤峰。1949年4月，省會駐承德市。轄2市、23縣旗。
1949年10月，[[中華人民共和國|中華人民共和國]]成立，保留熱河省，重新劃分為2市、16縣、4旗。

{{Wikisource|中華人民共和國第一屆全國人民代表大會第二次會議關於撤銷熱河省、西康省並修改中華人民共和國地方各級人民代表大會和地方各級人民委員會組織法第二十五條第二款第一項規定的決議}}
1955年7月30日第一屆[[全國人民代表大會|全國人民代表大會]]第二次會議決議，撤銷熱河省。原熱河省所屬承德市及承德、平泉、青龍、興隆、灤平、豐寧、隆化、圍場等8縣劃歸[[河北省|河北省]]；建昌、淩源、建平、朝陽、北票等5縣及喀喇沁左旗劃歸[[遼寧省|遼寧省]][[錦州專區|錦州專區]]；赤峰、烏丹、寧城3縣及[[敖漢旗|敖漢旗]]、[[喀喇沁旗|喀喇沁旗]]、[[翁牛特旗|翁牛特旗]]（原翁牛特蒙古族自治旗改設）3旗劃歸[[內蒙古自治區|內蒙古自治區]][[昭烏達盟|昭烏達盟]]。

== 行政區劃 ==
1945年9月成立熱河省政府，是年轄4專區、1盟、1市、24縣、12旗：
*熱東專署，轄凌南、凌源、義縣、興城、綏中、錦縣、錦西7縣及喀喇沁左旗。
*熱中專署，轄隆東、承北、平寧、建赤、建西5縣及喀喇沁中旗、喀喇沁右旗。
*熱遼專區，轄建平、朝陽、北票、北阜義、新惠、新東6縣及敖漢旗、土默特旗。
*第二十二區，轄赤峰市，圍北、赤西、赤峰3縣及翁牛特後旗、翁敖旗。
*昭烏達盟，轄經棚、林西、林東3縣及克什克騰旗、巴林左旗、巴林右旗、阿魯科爾沁旗、扎魯特旗。

1945年10月，凌南縣改名建昌縣，增設北票縣（以朝陽縣北票礦區為主）。是年增設烏丹縣。

1946年11月，第二十二專區撤銷，併入熱中專區。是年，增設寧城縣（以平泉縣北部為主）。

1947年4月，增設葉柏壽縣；6月，撤銷赤峰市；7月，增設青平縣（以青龍、平泉2縣交界地為主）。

1948年初，復置赤峰市；11月，增設承德市（以承德縣城區為主）。是年增設羊山縣（以朝陽縣北部為主）。

1949年4月，轄2市、19縣、4旗：
* 省轄市：[[承德市|承德市]]、[[赤峰市|赤峰市]]
* 縣：承德縣、赤峰縣、凌源縣、平泉縣、[[建昌縣|建昌縣]]、建平縣、寧城縣、圍場縣、[[烏丹縣|烏丹縣]]、[[青龍縣|青龍縣]]、葉柏壽縣、隆化縣、豐寧縣、灤平縣、[[興隆縣|興隆縣]]、青平縣、[[北票縣|北票縣]]、朝陽縣、羊山縣
* 旗：[[喀喇沁右旗|喀喇沁右旗]]、[[敖漢旗|敖漢旗]]、[[翁牛特旗|翁牛特旗]]、[[喀喇沁左旗|喀喇沁左旗]]

1949年，由東北人民政府領導，轄2市16縣4旗：
* 省轄市：[[承德市|承德市]]、[[赤峰市|赤峰市]]
* 縣：承德縣、赤峰縣、淩源縣、平泉縣、[[建昌縣|建昌縣]]、建平縣、寧城縣、圍場縣、[[烏丹縣|烏丹縣]]、[[青龍縣|青龍縣]]、隆化縣、豐寧縣、灤平縣、[[興隆縣|興隆縣]]、[[北票縣|北票縣]]、朝陽縣
* 旗：[[喀喇沁右旗|喀喇沁右旗]]、[[敖漢旗|敖漢旗]]、[[翁牛特旗|翁牛特旗]]、[[喀喇沁左旗|喀喇沁左旗]]

1949年7月，撤銷葉柏壽、青平2縣。

1950年，增設[[葉柏壽縣|葉柏壽縣]]、[[羊山縣|羊山縣]]（以朝陽南部羊山鎮為中心）；喀喇沁右旗改稱喀喇沁旗。 

1951年，撤銷葉柏壽、羊山2縣。 

1952年，由東北行政委員會領導。撤銷赤峰市，併入赤峰縣。 

1953年，赤峰縣由初頭朗遷駐赤峰縣城關區。原翁牛特旗改設翁牛特旗蒙族自治區。

== 人口 ==
1953年人口普查總人口516萬0822，其中漢族480萬1385（93.0%），蒙古族18萬2256（3.5%），滿族14萬7223（2.8%）。<ref>《中華人民共和國人口統計資料匯編1949—1985》，中國財政經濟出版社，1988。第974頁。</ref>

== 政府 ==
{{Main|熱河省人民政府|熱河省人民委員會}}

[[中國抗日戰爭|抗日戰爭]]結束後，[[中國共產黨|中國共產黨]]在熱河省舉行[[熱河省人民代表大會|熱河省人民代表大會]]，選舉產生熱河省政府主席。

1948年11月12日，[[中國人民解放軍|中國人民解放軍]]攻占[[承德市|承德]]，控制了[[热河省_(中华民国)|熱河全省]]。设立熱河省新政权，歸[[中共中央東北局|中共中央東北局]]領導。1949年1月10日，[[熱河省人民政府|熱河省人民政府]]成立。1949年10月1日，[[中華人民共和國|中華人民共和國]]成立。1955年2月，熱河省[[人民委員會|人民委員會]]成立，取代省[[人民政府|人民政府]]。
