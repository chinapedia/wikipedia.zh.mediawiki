《'''史记菁华录'''》，[[清朝|清朝]][[文学批评家|文学批评家]][[姚祖恩|姚祖恩]]的作品。<ref name="boksc " >{{cite book |author=姚苎田 |title=《史记菁华录》|year=2010年 |publisher=中华书局 |location=北京 |pages= |isbn=978-7-101-07603-5 |language = zh-hans}}</ref>

== 简介 ==
该书一共六卷，五十一篇，均出自[[司马迁|司马迁]]的《[[史记|史记]]》。<ref name="boksc " />

== 作者简介 ==
姚祖恩，生卒年不详，号苎田，出生于[[浙江|浙江]][[钱塘|钱塘]]，大致活跃于[[康乾|康乾]]时期，[[清朝|清朝]]著名文学批评家。<ref name="boksc " />

== 文风 ==
该书分为原文，姚祖恩的[[眉批|眉批]]、[[夹批|夹批]]、和[[尾批|尾批]]三种。眉批和夹批每篇都有，这两种批语侧重于品读原文中的章、句、段。<ref name="boksc " /> 尾批很少有，主要用来总结文章的立意。<ref name="boksc " /> 总得来说，三种批语一般是：分析章法、分析句法、分析立意、指出微言大义，或赞，或叹，或咏，或泣，文笔甚为生动活泼，清新洒脱。<ref name="boksc " /> 

== 评价 ==
[[姚祖恩|姚祖恩]]自谓：“特采录而详阅之，务使开卷释然，皆可成诵，间加论断，必出心裁。”<ref name="boksc " />
