'''史明'''（{{bd|1918年|11月9日|catIdx=Shih史}}）本名'''施朝暉'''，是[[中華民國|中華民國]]（[[臺灣|臺灣]]）的革命家、政治人物、作家、歷史學者。他是[[臺獨左派|左派]][[臺灣獨立運動|臺灣獨立運動]]重要領導人之一，提倡「[[臺灣民族主義|臺灣民族主義]]」與「[[臺灣共和國|獨立建國路線]]」，並且主張[[二階段革命理論|二階段革命理論]]。其著有《[[臺灣人四百年史|臺灣人四百年史]]》、《台灣民族革命與社會主義》、《民族形成與台灣民族》、《民主主義》、《台灣民族主義》、《西洋哲學序說》、《史明回憶錄：追求理想不回頭》等。

史明出身[[臺北市|臺北市]][[士林區|士林區]]的施家望族，早年求學便產生反抗意識。後來留學[[日本|日本]][[早稻田大學|早稻田大學]]的{{Link-ja|早稻田大學政治經濟學部|早稲田大学政治経済学部|政治經濟學部}}，因而認同[[馬克思主義|馬克思主義]]。1943年，在[[中國抗日戰爭|中國抗日戰爭]]期間，他前往[[中國大陸|中國大陸]]並擔任地下情報員。但在1949年，其因不滿[[中國共產黨|中國共產黨]]的作為，返回臺灣。後來他組織「[[台灣獨立革命武裝隊|台灣獨立革命武裝隊]]」，準備暗殺[[蔣中正|蔣中正]]。因為行動失敗，遭[[中國國民黨|中國國民黨]]的[[臺灣警備總司令部|臺灣警備總司令部]]列為「叛亂第一司令」通緝，並在1952年流亡日本。

在日本期間，他開設「[[新珍味中華料理店|新珍味中華料理店]]」，販賣麵食和餃子，並以此作為革命基地。1962年，其撰寫的臺灣通史《臺灣人四百年史》出版。1967年，他創辦「[[獨立臺灣會|獨立臺灣會]]」。1981年，他開始前往各國宣揚「臺灣民族主義」，影響許多海外留學生。在解除《[[臺灣省戒嚴令|臺灣省戒嚴令]]》後，史明的著作仍被列為禁書，並在1991年發生「[[獨立台灣會案|獨立台灣會案]]」。1993年，史明潛返回到臺灣，繼續在街頭推動臺灣獨立運動。2016年，獲聘成為[[中華民國總統府資政|中華民國總統府資政]]。

== 早年生活 ==
=== 家庭背景 ===
[[File:1925臺灣文化協會講演團攝於協會新竹州支部《臺灣民報》批發處_Taiwanese_Cultural_Association_Speakers_at_Min_Bao_office_in_Shinchiku_Prefecture.jpg|thumb]]長期參與[[臺灣文化協會|臺灣文化協會]]活動，並與[[林獻堂|林獻堂]]、[[蔣渭水|蔣渭水]]有所往來。]]

1918年11月9日，史明在[[日治臺灣|日治臺灣]][[臺北廳|臺北廳]][[士林庄|士林庄]]（現今士林區）施家大戶出生，原先出生名字為「林朝輝」<ref name = "家世背景">{{Cite web |url = https://web.archive.org/web/20160827023251/http://tw400.org.tw/1sitebrief/1abstract/1family/1family.html |title = 家世背景、童年 |author =  |date =  |publisher = [[獨立臺灣會|獨立臺灣會]] |language = Zh-hant |accessdate = 2019年3月24日 |quote =  }}</ref>{{sfnm|黃英哲|1991年|1loc=第26頁至第27頁|Claude Geoffroy|1997年|2loc=第63頁|蘇振明|2011年|3loc=第8頁、第11頁至第12頁、第16頁、第170頁}}。在5個兄弟中，只有史明被[[過繼母系|過繼母系]]，後來依從母親姓氏改名為「施朝輝」{{sfnm|陳銘城|1992年|1loc=第75頁至第76頁|Claude Geoffroy|1997年|2loc=第63頁|蘇振明|2011年|3loc=第16頁、第20頁}}。其母系家族是擁有鄰近160甲土地的大地主，外祖母師邱桂是臺北廳士林施家望族的總管{{sfnm|Claude Geoffroy|1997年|1loc=第63頁|蘇振明|2011年|2loc=第12頁、第168頁、第170頁}}。母親施阿秀是出身士林施家大戶的獨女，從小接受[[儒家|儒家]]教育{{sfnm|蘇振明|2011年|1loc=第13頁、第168頁、第170頁|江宗翰|2012年|2loc=第104頁至第105頁}}。而父親[[林濟川|林濟川]]是[[豐原郡_(台灣)|豐原郡]]人，早年[[留學|留學]]日本[[明治大學|明治大學]]而成為農學家，是[[臺灣日治時期|殖民地時期]]少數的高級知識分子{{sfnm|陳銘城|1992年|1loc=第75頁至第76頁|Claude Geoffroy|1997年|2loc=第63頁|蘇振明|2011年|3loc=第14頁|江宗翰|2012年|4loc=第101頁至第102頁}}。

當時的[[大日本帝國|大日本帝國]]對臺灣地區實行殖民統治{{sfnm|江宗翰|2012年|1loc=第101頁至第102頁}}。林濟川和[[林獻堂|林獻堂]]都是[[臺中市|臺中人]]，兩人私交甚篤，在日治時期都是曾經參加[[臺灣文化協會|臺灣文化協會]]的成員{{sfnm|Claude Geoffroy|1997年|1loc=第63頁|蘇振明|2011年|2loc=第14頁、第24頁、第168頁、第170頁|江宗翰|2012年|3loc=第101頁至第102頁、第115頁}}。當時兩人一樣屬於[[右派|右派]]陣營，曾在《[[臺灣青年|臺灣青年]]》發表論文，林濟川還擔任過《臺灣青年》的編輯{{sfnm|陳銘城|1992年|1loc=第75頁至第76頁|Claude Geoffroy|1997年|2loc=第63頁}}。因為父親的交往關係，史明經常跟著進出[[蔣渭水|蔣渭水]]在[[太平町|太平町]]開設的的大安醫院，並與[[林茂生|林茂生]]之子[[林宗義|林宗義]]是幼年玩伴<ref name = "家世背景"/>{{sfnm|陳銘城|1992年|1loc=第75頁至第76頁|蘇振明|2011年|2loc=第11頁、第14頁、第24頁|江宗翰|2012年|3loc=第101頁至第102頁}}。在1922年，義父施振興前往日本留學，後來啟發史明喜歡上西洋[[古典音樂|古典音樂]]{{sfnm|蘇振明|2011年|1loc=第24頁|江宗翰|2012年|2loc=第105頁、第117頁}}。

在小時候，史明由外祖母貼身照顧，過程中聽到許多口述歷史，但也被母親嚴格管教{{sfnm|蘇振明|2011年|1loc=第16頁至第20頁、第22頁至第23頁、第25頁、第31頁、第40頁至第41頁}}。1926年，史明原本要與其他臺灣孩子一起就讀[[臺北州|臺北州]]的[[士林公學校|士林公學校]]，但是因為母親的特別請託，後來他轉進以招收日本學童為主的臺北州「[[建成小學校|建成小學校]]」，並於1932年畢業{{sfnm|黃英哲|1991年|1loc=第26頁至第27頁|蘇振明|2011年|2loc=第29頁、第32頁至第34頁、第168頁、第170頁}}。而在1933年，他進入[[臺灣教育史|舊制五年制]]的[[臺北州立臺北第一中學校|臺北州立臺北第一中學校]]就讀，當時積極參與校園中的「硬派」學生隊伍，並表現出濃厚的[[反日|反抗日本]]情緒{{sfnm|黃英哲|1991年|1loc=第26頁至第27頁|陳銘城|1992年|2loc=第75頁至第76頁|蘇振明|2011年|3loc=第35頁至第36頁、第168頁、第170頁|江宗翰|2012年|4loc=第115頁至第116頁}}。

1936年3月，儘管母親希望其就讀醫學院、從而成為醫生，但不願接受此安排的史明，在念完臺北州立臺北第一中學校四年級後，選擇肄業並離家出走，孤身前往日本求學{{sfnm|蘇振明|2011年|1loc=第8頁、第25頁、第29頁、第36頁至第38頁、第170頁|江宗翰|2012年|2loc=第104頁至第105頁、第119頁至第120頁}}。到了1937年，為了讓臺灣能夠脫離惡劣的殖民地環境，他前往日本[[早稻田大學|早稻田大學]]，進入{{Link-ja|早稻田大學政治經濟學部|早稲田大学政治経済学部|政治經濟學部}}的政治科就讀<ref name = "求學時期">{{Cite web |url = https://web.archive.org/web/20161018112543/http://tw400.org.tw/1sitebrief/1study/1study.html |title = 各時期的史明—在臺、日求學時期 |author =  |date =  |publisher = [[獨立臺灣會|獨立臺灣會]] |language = Zh-hant |accessdate = 2019年3月24日 |quote =  }}</ref>，主要攻讀[[政治學|政治學]]和[[政治經濟學|政治經濟學]]{{sfnm|黃英哲|1991年|1loc=第26頁至第27頁|陳銘城|1992年|2loc=第75頁至第76頁|蘇振明|2011年|3loc=第8頁、第25頁、第36頁至第37頁、第170頁|江宗翰|2012年|4loc=第120頁}}。有別於台灣學生喜愛的商科與醫學，當時史明選擇攻讀的政治經濟學科，仍然是殖民地時期極少人選擇的冷門學科<ref name = "許建榮 01">{{Cite web |url = http://www.thinkingtaiwan.com/content/258 |title = 革命先行者史明（上）：穿越中國紅色浪潮 |author = 許建榮 |date = 2012年11月8日 |publisher = 想想 |language = Zh-hant |accessdate = 2019年3月24日 |quote =  |archive-url = https://web.archive.org/web/20160406065830/http://www.thinkingtaiwan.com/content/258 |archive-date = 2016年4月6日 |dead-url = no }}</ref>{{sfnm|Claude Geoffroy|1997年|1loc=第63頁|蘇振明|2011年|2loc=第25頁}}。

當時正值[[第一次世界大戰|第一次世界大戰]]戰後，早稻田大學流行著[[社會主義|社會主義]]、[[共產主義|共產主義]]與[[納粹主義|納粹主義]]的思想，在此受到社會主義啟蒙的史明，也深深受到[[馬克思列寧主義|馬克思列寧主義]]的影響，並轉而認同社會主義{{sfnm|陳銘城|1992年|1loc=第75頁至第76頁|蘇振明|2011年|2loc=第8頁、第45頁至第49頁、第60頁、第170頁|江宗翰|2012年|3loc=第120頁}}。對於蘇聯在1917年的[[十月革命|社會主義革命]]獲得成功，這也讓他受到極大的鼓舞，「廣泛閱讀」所有社會主義與[[無政府主義|無政府主義]]作家的作品，特別是被[[卡爾·馬克思|卡爾·馬克思]]的[[馬克思主義|理論]]所吸引{{sfnm|黃英哲|1991年|1loc=第26頁至第27頁|Claude Geoffroy|1997年|2loc=第63頁|蘇振明|2011年|3loc=第49頁|江宗翰|2012年|4loc=第120頁、第122頁至第124頁}}。與此同時，史明還接受{{Link-ja|高田早苗|高田早苗}}、{{Link-ja|大山郁夫|大山郁夫}}的課程教育，以及受到[[大隈重信|大隈重信]]等人影響{{sfnm|蘇振明|2011年|1loc=第46頁至第48頁、第58頁至第59頁}}。

與此同時，各個[[殖民地|殖民地]]也開始出現[[獨立運動|獨立運動]]，由[[第三國際|第三國際]]提出的「反殖民主義」、「民族解放」的主張成為風潮，並且衝擊到日本<ref name = "求學時期"/>{{sfnm|蘇振明|2011年|1loc=第49頁}}。為了決心從事反日、[[反帝國主義|反帝國主義]]的運動，史明也特別攻讀[[殖民主義|殖民主義]]政策{{sfnm|黃英哲|1991年|1loc=第26頁至第27頁|陳銘城|1992年|2loc=第75頁至第76頁|蘇振明|2011年|3loc=第49頁}}。但在1941年，[[太平洋戰爭|太平洋戰爭]]爆發{{sfnm|陳銘城|1992年|1loc=第75頁至第76頁}}。隨著太平洋戰爭的局勢日漸激烈，儘管原本應該在1943年6月從早稻田大學政治經濟學部政治科畢業，因為日本政府命令全國各大學{{Link-ja|學生出陣|学徒出陣|提前}}半年結束應屆畢業生的課業，史明被迫提前在1942年9月結束學業，甚至省略原先的畢業典禮和畢業紀念冊{{sfnm|黃英哲|1991年|1loc=第26頁至第27頁|蘇振明|2011年|2loc=第49頁至第50頁、第170頁}}。

== 早期活動 ==
=== 中國活動 ===
{{Quote box| width = 300px | align = left | quoted = true | bgcolor = #FFFFF0 | salign = right | quote = ……這種分化政策正是清朝、日本殖民統治一貫採取的伎倆。從中共對臺灣兵的種種行徑，促使我重新思考，更加自覺到我是臺灣人，不是中國人。 | source= 史明{{sfnm|黃英哲|1991年|1loc=第25頁至第26頁}} }}

1942年，在離開早稻田大學後，憧憬馬克思主義的史明透過早稻田大學時期的同學引薦<ref name = "民視新聞台 01"/>，基於濃厚的抗日意識、及對社會主義理念的信仰，決心前往[[中國大陸|中國大陸]]投入[[中國抗日戰爭|抗日作戰]]，選擇加入[[中國共產黨|中國共產黨]]陣營的行動{{sfnm|黃英哲|1991年|1loc=第26頁至第27頁|陳銘城|1992年|2loc=第75頁至第76頁|Claude Geoffroy|1997年|3loc=第63頁|蘇振明|2011年|4loc=第8頁、第50頁至第51頁、第57頁、第60頁、第170頁|江宗翰|2012年|5loc=第128頁、第132頁至第134頁}}。1943年，在前往中國大陸參與抗日運動前夕，史明返回臺灣探望外祖母，當面承諾在戰爭結束後，將早日返鄉、陪伴家人終老{{sfnm|蘇振明|2011年|1loc=第53頁、第127頁至第128頁}}。他隨即橫渡至[[中華民國_(大陸時期)|中華民國]][[上海市|上海市]]，再輾轉到[[延安市|延安市]]、[[太行山|太行山]]受訓，後來因其懂[[日語|日語]]，奉命派往上海市、[[北京市|北京市]]等地，從事中國共產黨抗日的地下情報員工作<ref name = "民視新聞台 01"/><ref>{{Cite web |url = https://web.archive.org/web/20161018115331/http://tw400.org.tw/1sitebrief/1abstract/1abstract.html |title = 史明身世素描 |author =  |date =  |publisher = [[獨立臺灣會|獨立臺灣會]] |language = Zh-hant |accessdate = 2019年3月24日 |quote =  }}</ref>，化名「林鐸」<ref>{{Cite web |url = https://tw.appledaily.com/new/realtime/20170722/1156140/ |title = 史明為革命結紮不婚　最愛還是日籍前女友平賀 |author = 林修卉 |date = 2017年7月22日 |publisher = 《[[蘋果日報_(臺灣)|蘋果日報]]》 |language = Zh-hant |accessdate = 2019年3月24日 |quote =  |archive-url = https://web.archive.org/web/20171208062425/https://tw.appledaily.com/new/realtime/20170722/1156140/ |archive-date = 2017年12月8日 |dead-url = no }}</ref>{{sfnm|黃英哲|1991年|1loc=第26頁至第27頁|陳銘城|1992年|2loc=第75頁至第76頁|蘇振明|2011年|3loc=第54頁、第170頁|江宗翰|2012年|4loc=第128頁}}。儘管他極力反對[[鴉片|鴉片]]買賣，但為了籌措游擊隊的經費，仍然受命[[特貨|運輸鴉片]]<ref name = "許建榮 01"/>。1945年11月，隨著日本在[[第二次世界大戰|第二次世界大戰]][[日本投降|戰敗]]，他返回北京市，並被派到[[張家口市|張家口市]]的[[華北聯合大學|華北聯合大學]]接受訓練<ref name = "許建榮 01"/>{{sfnm|江宗翰|2012年|1loc=第129頁}}。

1946年夏天，[[第二次國共內戰|第二次國共內戰]]爆發，[[中國人民解放軍|中國人民解放軍]]向擁有美式裝備的[[中國國民黨|中國國民黨]]部隊展開[[游擊戰|游擊戰]]{{sfnm|蘇振明|2011年|1loc=第54頁}}。史明開始投入中國共產黨的[[無產階級國際主義|無產階級]][[新民主主義|革命行動]]{{sfnm|江宗翰|2012年|1loc=第129頁}}。在武力懸殊的情況下，他隨著[[中國人民解放軍|中國人民解放軍]]退入根據地[[河北省|河北省]]，並被派到[[狼牙山|狼牙山]]帶領游擊隊<ref name = "許建榮 01"/>{{sfnm|蘇振明|2011年|1loc=第54頁|江宗翰|2012年|2loc=第129頁至第131頁}}。他還曾護送[[蒙疆銀行|蒙疆銀行]]撤退，與中國國民黨進行3個月的游擊戰<ref name = "許建榮 01"/>。同年，中國共產黨實施[[土地改革運動|土地改革運動]]，他親眼見到中國人民解放軍的鬥爭屠殺，這是其第一次對中國感到失望，並認為中國共產黨本質上[[對中國共產黨的評論|相當虛偽]]<ref name = "民視新聞台 01"/>{{sfnm|黃英哲|1991年|1loc=第27頁至第28頁|Claude Geoffroy|1997年|2loc=第63頁|蘇振明|2011年|3loc=第45頁、第56頁至第57頁、第60頁至第61頁、第86頁|江宗翰|2012年|4loc=第129頁至第131頁、第133頁至第135頁}}。在1947年初春，他在河北省透過媒體，聽聞中國國民黨接收的臺灣發生[[二二八事件|二二八事件]]，[[彭孟緝|彭孟緝]]更在[[高雄市|高雄市]]發動流血鎮壓{{sfnm|蘇振明|2011年|1loc=第68頁}}。同年，他在[[晉冀魯豫邊區|晉冀魯豫邊區]]的軍事政治大學，開始參加「[[臺灣隊_(國共內戰)|臺灣隊]]」的設立{{sfnm|黃英哲|1991年|1loc=第27頁至第28頁|蘇振明|2011年|2loc=第56頁|江宗翰|2012年|3loc=第131頁至第132頁}}。當時中國人民解放軍俘虜收編300多位臺灣士兵，並由其出任代理隊長和政治教員負責訓練{{sfnm|黃英哲|1991年|1loc=第27頁至第28頁|陳銘城|1992年|2loc=第76頁|Claude Geoffroy|1997年|3loc=第63頁|蘇振明|2011年|4loc=第55頁至第56頁|江宗翰|2012年|5loc=第131頁至第132頁}}。

由於先前許多臺灣士兵被派往前線，史明親眼見識到[[中國人|中國人]]的「漢人種族主義」{{sfnm|Claude Geoffroy|1997年|1loc=第63頁至第64頁|蘇振明|2011年|2loc=第55頁至第56頁}}；同時因不忍見到中國共產黨利用[[閩南民系|閩南民系]]、[[客家民系|客家民系]]的矛盾分化[[臺灣人|臺灣人]]，他開始對中國共產黨不滿，從此認為「台灣人不能和中國人一起」<ref>{{Cite web |url = https://web.archive.org/web/20160827001858/http://tw400.org.tw/1sitebrief/1study/1ch/1ch.html |title = 中國參共抗日時期 |author =  |date =  |publisher = [[獨立臺灣會|獨立臺灣會]] |language = Zh-hant |accessdate = 2019年3月24日 |quote =  }}</ref>{{sfnm|陳銘城|1992年|1loc=第76頁|Claude Geoffroy|1997年|2loc=第63頁至第64頁|蘇振明|2011年|3loc=第56頁至第57頁、第68頁}}。1949年5月，中國共產黨在國共內戰即將勝利前夕，由於對中國共產黨失望，此時萌生返回臺灣念頭的他，評估脫隊潛逃至臺灣的情勢{{sfnm|陳銘城|1992年|1loc=第76頁|Claude Geoffroy|1997年|2loc=第64頁|蘇振明|2011年|3loc=第8頁、第56頁至第57頁、第75頁、第170頁|江宗翰|2012年|4loc=第129頁至第132頁}}。趁著[[毛澤東|毛澤東]]巡視中國人民解放軍完畢轉赴北方，隨行人員得以優先通行的契機，他便假造路條（通行證），謊稱自己將被派往臺灣從事地下工作<ref name = "許建榮 01"/>，請沿途同志予以保護放行{{sfnm|蘇振明|2011年|1loc=第57頁}}。最終其從中國共產黨[[解放區|解放區]]逃亡，突破封鎖前往[[青島市|青島市]]，重新返回臺灣{{sfnm|黃英哲|1991年|1loc=第28頁|陳銘城|1992年|2loc=第76頁|蘇振明|2011年|3loc=第8頁、第57頁、第68頁至第69頁、第127頁至第128頁、第170頁|江宗翰|2012年|4loc=第131頁至第132頁}}。同年，[[蔣中正|蔣中正]]也因國共戰爭失敗，轉進退守臺灣<ref name = "回台革命">{{Cite web |url = https://web.archive.org/web/20150407044928/http://www.tw400.org.tw/1sitebrief/1study/1tw/1tw.html |title = 回台革命期 |author =  |date =  |publisher = [[獨立臺灣會|獨立臺灣會]] |language = Zh-hant |accessdate = 2019年3月24日 |quote =  }}</ref>{{sfnm|蘇振明|2011年|1loc=第68頁至第69頁}}。

=== 臺灣抵抗 ===
[[File:228_Incident_h.jpg|thumb]]的不滿，史明等人組織「[[台灣獨立革命武裝隊|台灣獨立革命武裝隊]]」，計畫暗殺蔣中正。]]

1949年，史明返回臺灣後，經由[[陳寶川|陳寶川]]介紹，認識曾為[[蔣經國|蔣經國]]同學的[[高理文|高理文]]{{sfnm|江宗翰|2012年|1loc=第128頁}}。雖然未曾親身經歷二二八事件，但他在中國大陸時期已有聽聞，由於二二八事件的影響、以及蔣中正以軍事[[臺灣戰後時期|統治]]臺灣，深感不滿的史明認為中國國民黨與中國共產黨一樣<ref name = "許建榮 02">{{Cite web |url = http://www.thinkingtaiwan.com/content/259 |title = 革命先行者史明（下）：回歸祖國台灣 跨越時空四百年 |author = 許建榮 |date = 2012年11月8日 |publisher = 想想 |language = Zh-hant |accessdate = 2019年3月24日 |quote =  |archive-url = https://web.archive.org/web/20190323165833/http://www.thinkingtaiwan.com/content/259 |archive-date = 2019年3月23日 |dead-url = no }}</ref>{{sfnm|黃英哲|1991年|1loc=第28頁|陳銘城|1992年|2loc=第76頁至第77頁|Claude Geoffroy|1997年|3loc=第53頁至第55頁、第62頁|蘇振明|2011年|4loc=第66頁、第68頁至第69頁、第75頁、第86頁|江宗翰|2012年|5loc=第135頁至第136頁}}。由於在中國大陸有著豐富的游擊實戰經驗，他認為台灣人必須展開武力行動對抗蔣中正<ref name = "許建榮 02"/><ref name = "周俊男 01">{{Cite web |url = https://gushi.tw/life-of-shi-ming-1/ |title = 追求理想不回頭：史明歐吉桑的革命人生（上） |author = 周俊男 |date = 2016年1月28日 |publisher = 故事 |language = Zh-hant |accessdate = 2019年3月24日 |quote =  |archive-url = https://web.archive.org/web/20190323171821/https://gushi.tw/life-of-shi-ming-1/ |archive-date = 2019年3月23日 |dead-url = no }}</ref>，決心成立一支臺灣人的武裝部隊，這吸引同樣不滿二二八事件的青年<ref name = "回台革命"/>{{sfnm|陳銘城|1992年|1loc=第76頁至第77頁|蘇振明|2011年|2loc=第8頁、第66頁、第69頁至第70頁|江宗翰|2012年|3loc=第135頁至第136頁}}。從1950年2月起，他與[[周潔|周潔]]、[[黃元秘|黃元秘]]等人集合當時在二二八事件倖存的30多名有志青年，在臺北市郊外的山上秘密組織「[[台灣獨立革命武裝隊|台灣獨立革命武裝隊]]」{{sfnm|黃英哲|1991年|1loc=第28頁|陳銘城|1992年|2loc=第4頁、第76頁至第77頁、第236頁|Claude Geoffroy|1997年|3loc=第109頁|蘇振明|2011年|4loc=第66頁、第69頁至第70頁、第75頁、第170頁|江宗翰|2012年|5loc=第135頁至第136頁}}。這批武裝部隊在臺北市士林區[[外雙溪|外雙溪]]和[[草山_(臺北市)|草山]]、[[苗栗縣|苗栗縣]][[大湖鄉_(台灣)|大湖]]和[[南庄鄉_(台灣)|南庄]]等地，蒐集[[日本軍|日本軍]]留下來的20多把步槍武器<ref name = "許建榮 02"/>，並預藏在[[陽明山國家公園|陽明山]]後的施家柑園，準備發動武裝革命，準備伺機開槍暗殺將臺灣當作殖民地統治的蔣中正<ref name = "周俊男 01"/>{{sfnm|黃英哲|1991年|1loc=第28頁|陳銘城|1992年|2loc=第4頁、第76頁至第77頁、第236頁|蘇振明|2011年|3loc=第8頁、第69頁至第70頁、第75頁|江宗翰|2012年|4loc=第135頁至第136頁}}。

1951年底，因為槍枝被中國國民黨政府發覺，刺殺蔣中正的反抗行動失敗<ref name = "許建榮 02"/>，主導計劃的史明遭到監視追捕、並被通緝{{sfnm|黃英哲|1991年|1loc=第28頁|陳銘城|1992年|2loc=第4頁、第76頁至第77頁、第236頁|Claude Geoffroy|1997年|3loc=第109頁|蘇振明|2011年|4loc=第66頁、第70頁、第127頁至第128頁、第170頁}}。最初史明從山上回到士林區老家，見到自家門口突然有陌生的中國女子賣花生，基於多年地下情報工作經驗的敏感度，讓其立即產生警覺<ref name = "回台革命"/>{{sfnm|陳銘城|1992年|2loc=第76頁至第77頁|蘇振明|2011年|2loc=第70頁}}。接著又看到住家對面有中國青年搬入，更讓他覺得情勢有異，快速攜帶著些許資金和組織名冊準備逃亡<ref name = "許建榮 02"/><ref name = "回台革命"/>{{sfnm|陳銘城|1992年|2loc=第76頁至第77頁|蘇振明|2011年|2loc=第70頁至第72頁、第76頁}}。1952年3月，他決定[[非法移民|非法移民]]到[[日本|日本]]，先逃至[[基隆港|基隆港]]，向工頭買通上船工作的紅帽，混入港口碼頭，擔任搬運輸往日本香蕉的工人{{sfnm|陳銘城|1992年|1loc=第77頁|陳銘城|1992年|2loc=第4頁|Claude Geoffroy|1997年|3loc=第109頁|蘇振明|2011年|4loc=第66頁、第73頁、第76頁、第170頁}}。5月19日，遭通緝的他得到逃亡機會，託人將紅帽攜出後，自己藉由藏匿在隸屬[[陽明海運|輪船招商局]]、載運香蕉至日本的5,000噸貨輪「天山丸號」的貨艙裡，從[[基隆市|基隆市]]秘密偷渡到日本[[神戶市|神戶市]]{{sfnm|黃英哲|1991年|1loc=第28頁|陳銘城|1992年|2loc=第4頁、第73頁、第77頁、第236頁|Claude Geoffroy|1997年|3loc=第109頁|蘇振明|2011年|4loc=第73頁、第170頁|江宗翰|2012年|5loc=第135頁至第136頁}}。

儘管順利逃離臺灣，但當史明從日本神戶市登陸上岸，就立刻遭[[日本警察|日本警察]]逮捕，且欲將其遣返回臺灣，這導致他預藏剃刀準備自殺{{sfnm|黃英哲|1991年|1loc=第28頁|陳銘城|1992年|2loc=第4頁、第77頁|Claude Geoffroy|1997年|3loc=第109頁|蘇振明|2011年|4loc=第73頁至第74頁、第170頁}}。不過從1952年起，中國國民黨的[[臺灣警備總司令部|臺灣警備總司令部]]以「叛亂第一司令」的罪名，向海內外全面通緝，要求日本引渡其返回臺灣，這也是他第一次被中國國民黨政府通緝{{sfnm|黃英哲|1991年|1loc=第28頁|陳銘城|1992年|2loc=第4頁、第77頁|蘇振明|2011年|3loc=第8頁、第74頁至第75頁、第77頁、第86頁、第124頁}}。這反而讓[[日本國政府|日本國政府]]認定其是[[政治犯|政治犯]]，並基於人道理由提供[[庇護權|政治庇護]]，而能夠在日本長期居留，展開長達41年的通緝流亡生活{{sfnm|黃英哲|1991年|1loc=第28頁|陳銘城|1992年|2loc=第4頁、第77頁|Claude Geoffroy|1997年|3loc=第109頁|蘇振明|2011年|4loc=第75頁、第77頁、第103頁、第124頁、第170頁|江宗翰|2012年|5loc=第135頁至第136頁}}。不過這個罪名也讓他無法返回家鄉，甚至連最疼愛他的外祖母、阿姑與雙親逝世時，都無法返回臺灣參加往生葬禮{{sfnm|蘇振明|2011年|1loc=第124頁、第127頁至第128頁}}。

1952年起，為了在僑居日本期間的生活，史明與在中國大陸結婚的日籍妻子[[平賀協子|平賀協子]]，在[[東京|東京]][[池袋|池袋]]的[[池袋車站|池袋車站]]前廣場推車擺攤，販賣麵食、餃子{{sfnm|陳銘城|1992年|1loc=第73頁|Claude Geoffroy|1997年|2loc=第64頁|蘇振明|2011年|3loc=第8頁、第81頁至第85頁、第90頁、第92頁至第93頁、第170頁}}。從1955年起，史明的經濟計劃執行的相當成功，為自己在東京的料理事業打下基礎{{sfnm|蘇振明|2011年|1loc=第84頁、第87頁}}。1960年，為了實踐其社會改造理念，在還清擺攤的創業貸款後，買下現在位於池袋車站西口的店面樓房，興建、經營五層樓高的「[[新珍味中華料理店|新珍味中華料理店]]」，為當年池袋車站站前最醒目的大樓{{sfnm|蘇振明|2011年|1loc=第84頁、第170頁|陳銘城|1992年|2loc=第73頁}}。當時他販賣[[餃子|餃子]]、[[燒賣|燒賣]]、[[陽春麵|陽春麵]]等，賺進約5億日圓的資產{{sfnm|黃英哲|1991年|1loc=第28頁至第29頁|蘇振明|2011年|2loc=第81頁、第90頁}}。但他仍然持續關心革命事務，除了開始隨時抽空鑽研[[臺灣歷史|臺灣歷史]]，沒有放棄社會主義思想的他還前往圖書館深入研讀，重新確認社會主義與馬克思主義理論<ref name = "周俊男 01"/><ref name = "亡命日本">{{Cite web |url = https://web.archive.org/web/20160826235012/http://tw400.org.tw/1sitebrief/1study/1jpstay/1jpstay.html |title = 亡命日本期 |author =  |date =  |publisher = [[獨立臺灣會|獨立臺灣會]] |language = Zh-hant |accessdate = 2019年3月24日 |quote =  }}</ref>{{sfnm|黃英哲|1991年|1loc=第28頁至第29頁|陳銘城|1992年|2loc=第73頁|Claude Geoffroy|1997年|3loc=第109頁至第110頁|蘇振明|2011年|4loc=第86頁至第87頁、第170頁|江宗翰|2012年|5loc=第140頁}}。

02"/>。史明還曾經加入「[[台灣青年獨立聯盟|台灣青年獨立聯盟]]」，直至1965年[[辜寬敏|辜寬敏]]出任中央委員會主席才離開，也曾協助[[黃昭堂|黃昭堂]]、[[許世楷|許世楷]]等人{{sfnm|Claude Geoffroy|1997年|1loc=第109頁至第110頁|江宗翰|2012年|2loc=第137頁}}。

1955年，在東京的史明與臺灣島內刺殺蔣中正計劃的舊同志取得聯繫，開始建立對抗中國國民黨的「地下行動管道」{{sfnm|蘇振明|2011年|1loc=第87頁、第171頁}}。基於理念，他持續主張在臺灣島內發動武裝革命行動，計劃推翻在臺灣執政的政府<ref name = "民視新聞台 01"/>{{sfnm|Claude Geoffroy|1997年|1loc=第110頁|蘇振明|2011年|2loc=第87頁、第110頁、第128頁至第130頁|江宗翰|2012年|3loc=第138頁}}。雖然最初沒有具體成效，其在東京開店販賣餃子賺取的資金，幾乎全用來培訓和資助前往日本的臺灣民主運動人士，而將料理店視為革命基地<ref name = "民視新聞台 01"/>{{sfnm|陳銘城|1992年|1loc=第74頁|蘇振明|2011年|2loc=第112頁至第113頁、第152頁}}。他還特別贊助、支持臺灣地下革命的同志，在東京料理店樓上秘密訓練<ref name = "民視新聞台 01"/>，以執行破壞敵人交通工具的任務{{sfnm|Claude Geoffroy|1997年|1loc=第110頁|蘇振明|2011年|2loc=第87頁、第102頁、第112頁至第113頁、第171頁}}。1959年，在臺灣的地下同志相繼燒毀[[臺北府城北門|臺北府城北門]]附近的[[國立臺灣博物館鐵道部|鐵路局]]工廠，及炸燬軍用火車<ref name = "民視新聞台 01"/>{{sfnm|Claude Geoffroy|1997年|1loc=第110頁|蘇振明|2011年|2loc=第171頁}}。1967年，史明從報紙上得知臺灣有火車翻覆時，高興地認為這代表其培訓的地下行動獲得初步成果，亦反映台灣人民對於蔣中正集權統治的覺醒{{sfnm|蘇振明|2011年|1loc=第152頁}}。

=== 組織工作 ===
[[File:New_Gourmet_MiNe-M5_100-0311UG.jpg|thumb]]」視為培訓和資助[[台灣獨立運動|台灣獨立運動]]的基地。]]

1967年4月，在史明、[[何文燦|何文燦]]為首的代表主導下，於東京成立「[[臺灣獨立聯合會|臺灣獨立聯合會]]」，嘗試串聯、結合在日本的各個[[臺灣獨立運動|臺灣獨立運動]]派系<ref name = "亡命日本"/>，參加的團體包括[[台灣民主獨立黨|台灣民主獨立黨]]（[[郭泰成|郭泰成]]）、[[台灣自由獨立黨|台灣自由獨立黨]]（[[廖明耀|廖明耀]]）、[[台灣共和黨|台灣共和黨]]（[[林台元|林-{台}-元]]）、[[台灣獨立戰線|台灣獨立戰線]]（[[李伯仁_(臺灣)|李伯仁]]、何文燦）、[[台灣公會|台灣公會]]（史明、[[黃介一|黃介一]]）等<ref name = "時代背景">{{Cite web |url = https://web.archive.org/web/20161221072042/http://tw400.org.tw/3about-tati/3-bkg/3-bkg.html |title = 獨台會成立的時代背景 |author =  |date =  |publisher = [[獨立臺灣會|獨立臺灣會]] |language = Zh-hant |accessdate = 2019年3月24日 |quote =  }}</ref>{{sfnm|蘇振明|2011年|1loc=第100頁至第103頁}}。然而，因為辜寬敏的「台灣青年獨立聯盟」、及[[張春興|張春興]]和[[林水|林水]]的「[[台灣獨立總同盟|台灣獨立總同盟]]」不願加入，「台灣獨立連合會」無法獲得在日本的全體團體參加，史明遂以「不能達成在日台獨運動大團結初志」為由決意解散<ref name = "亡命日本"/><ref name = "時代背景"/>{{sfnm|蘇振明|2011年|1loc=第101頁至第103頁}}。

同年6月30日，史明接續在日本東京自行創立臺灣獨立建國的團體「[[獨立臺灣會|獨立臺灣會]]」（簡稱「獨臺會」），成員約70人左右；他將該海外組織定義為「一個社會主義的黨」，同時提出「主戰場在島內」、「臺灣民族主義」、「臺灣社會主義」等行動準則與主張<ref>{{Cite web |url = https://web.archive.org/web/20160827071439/http://tw400.org.tw/3about-tati/3aim/3aim.html |title = 成立宗旨 |author =  |date =  |publisher = [[獨立臺灣會|獨立臺灣會]] |language = Zh-hant |accessdate = 2019年3月24日 |quote =  }}</ref>，積極從事對臺灣島內的地下工作和群眾運動<ref name = "許建榮 02"/><ref>{{Cite web |url = https://web.archive.org/web/20160827071525/http://tw400.org.tw/3about-tati/3strategy/3strategy.html |title = 策略∕進程 |author =  |date =  |publisher = [[獨立臺灣會|獨立臺灣會]] |language = Zh-hant |accessdate = 2019年3月24日 |quote =  }}</ref>{{sfnm|黃英哲|1991年|1loc=第29頁|陳銘城|1992年|2loc=第74頁、第239頁|Claude Geoffroy|1997年|3loc=第109頁|蘇振明|2011年|4loc=第101頁至第102頁、第113頁、第128頁至第130頁、第171頁|江宗翰|2012年|5loc=第138頁}}。他持續單線訓練「獨立臺灣會」的成員<ref name = "民視新聞台 01"/>{{sfnm|蘇振明|2011年|1loc=第128頁至第130頁}}。同年，他創辦《[[獨立台灣|獨立台灣]]》月刊，將會刊定位為「臺灣人民解放革命陣線機關誌」，並長期出版發行{{sfnm|黃英哲|1991年|1loc=第29頁|陳銘城|1992年|2loc=第74頁|Claude Geoffroy|1997年|3loc=第110頁|蘇振明|2011年|4loc=第101頁至第102頁、第171頁|江宗翰|2012年|5loc=第138頁}}。不過當時留學日本的學生[[顏尹謨|顏尹謨]]，因為涉及「[[林水泉|林水泉事件]]」而遭到逮捕，並被指控曾接受史明的資金，返回臺灣從事地下組織{{sfnm|陳銘城|1992年|1loc=第74頁、第239頁|蘇振明|2011年|2loc=第103頁至第104頁}}。

1968年，史明利用地下管道，潛回至臺灣島內，佈置地下組織{{sfnm|蘇振明|2011年|1loc=第171頁}}。1971年，原本在台灣經營一家麵包店的[[台灣基督教|基督徒]][[鄭評|鄭評]]，經人介紹前往日本結識史明，並加入獨立臺灣會{{sfnm|陳銘城|1992年|1loc=第75頁|江宗翰|2012年|2loc=第138頁}}。他與史明商談刺殺[[行政院院長|行政院院長]]蔣經國的計劃<ref name = "民視新聞台 01"/>，並參與游擊戰與政治的訓練{{sfnm|陳銘城|1992年|1loc=第75頁|Claude Geoffroy|1997年|2loc=第110頁|江宗翰|2012年|3loc=第138頁}}。這時候的蔣經國已因「[[四二四刺殺蔣經國案|四二四刺殺蔣經國案]]」而加強安全警衛，史明安排三位互不認識的人馬：一組負責調查、一組負責槍枝、一組負責執行刺殺蔣經國，並由他從日本派人轉送槍枝，而要鄭評在1年半內不要有動作{{sfnm|陳銘城|1992年|1loc=第75頁}}。但在1974年，沉不住氣的鄭評自己先行購買槍枝，行動曝光而被逮捕，最後[[台灣獨立革命軍|主要的成員]]遭到槍決<ref name="民視新聞台 01" />{{sfnm|陳銘城|1992年|1loc=第75頁、第241頁|蘇振明|2011年||2loc=第113頁}}。

史明長期不願公開「獨立臺灣會」成員資料，只有因刺殺蔣經國未成而被槍斃的鄭評是例外，並被其形容是個「感性的反國民黨者」{{sfnm|陳銘城|1992年|1loc=第75頁}}。1972年，他開始接觸工農勞苦大眾{{sfnm|陳銘城|1992年|1loc=第74頁}}。透過臺灣島內的地下管道，他取得臺灣經濟方面的秘密資料，並利用販賣餃子的空檔時間繼續寫作{{sfnm|陳銘城|1992年|1loc=第74頁}}。1975年，為了整理臺灣島內的地下組織，他再次潛回臺灣{{sfnm|蘇振明|2011年|1loc=第171頁}}。一直到1975年之後，獨立臺灣才不再從事武裝革命行動，而轉向群眾運動，從事群眾的思想啟蒙與組織培訓{{sfnm|蘇振明|2011年|1loc=第137頁}}。不過在1978年，獨立臺灣會的受訓同志返回臺灣後，仍從事張貼標語「臺灣民族獨立，勞苦大眾出頭天」任務，並破壞敵人設施{{sfnm|蘇振明|2011年|1loc=第171頁}}。

=== 海外訪問 ===
[[File:Taiwan_Independence_Association_09720042_(6587910735).jpg|thumb]]」的理念，但也遭到挑戰。]]

「獨立臺灣會」後來也和歐洲留學生[[張維嘉|張維嘉]]、[[盧修一|盧修一]]等人有所接觸{{sfnm|陳銘城|1992年|1loc=第74頁|Claude Geoffroy|1997年|2loc=第110頁|蘇振明|2011年|3loc=第103頁}}。1971年，歐美同志張維嘉由法國[[巴黎|巴黎]]前來日本，與史明談及建立「獨立臺灣會歐洲支部」{{sfnm|蘇振明|2011年|1loc=第171頁}}。同年，「獨立臺灣會」在會刊《獨立臺灣》發表《致釣魚臺行動委員會的一封公開信》，聲明[[釣魚臺列嶼|釣魚臺列嶼]]屬於臺灣的神聖領土，及其海域為臺灣人漁民既得的生活圈{{sfnm|蘇振明|2011年|1loc=第171頁}}。在這時期，由於史明在日本被歸類為沒有國籍的「亡命者」，因此一旦出境就無法入境{{sfnm|陳銘城|1992年|1loc=第77頁|蘇振明|2011年|2loc=第103頁}}。一直到1978年，日本有關政治庇護的居留政策獲得鬆綁，該法令更改成一定時數內自由出入日本國境，仍可獲准居留{{sfnm|陳銘城|1992年|1loc=第77頁|蘇振明|2011年|2loc=第103頁、第113頁}}。

1980年9月，史明根據《臺灣人四百年史》日文版的內容與骨架，重新以漢文撰寫翻譯、並進行相當份量的資料補充增訂，完成的《臺灣人四百年史》漢文版由[[蓬島文化公司|蓬島文化公司]]出版<ref name = "許建榮 02"/><ref name = "書籍"/>{{sfnm|黃英哲|1991年|1loc=第29頁|陳銘城|1992年|2loc=第74頁|蘇振明|2011年|3loc=第99頁、第171頁}}。在1981年夏天，為了推廣《臺灣人四百年史》，及宣傳「臺灣獨立」和「臺灣民族主義」的理念，他首度以亡命者身分，在沒有護照的情況下被允許短暫離開日本{{sfnm|蘇振明|2011年|1loc=第103頁、第105頁至第106頁、第113頁、第171頁}}。其陸續前往[[美國|美國]]、[[加拿大|加拿大]]、歐洲（[[西德|西德]]、[[法國|法國]]、[[奧地利|奧地利]]等國）及南美洲[[巴西|巴西]]等地巡迴訪問，廣泛向各國臺灣僑胞和各地留學生進行文化宣講<ref name = "民視新聞台 01"/>{{sfnm|蘇振明|2011年|1loc=第103頁、第113頁、第171頁}}。但因為當時[[陳文成|陳文成]]從美國返回臺灣卻[[陳文成事件|遇害]]的[[臺灣白色恐怖時期|白色恐怖事件]]才發生不久，因此只能私密邀請他進行演講座談{{sfnm|蘇振明|2011年|1loc=第105頁至第106頁}}。

而從1981年起，史明的公開活動大多就以演講、授課的思想傳播，作為主要的活動方式<ref name = "民視新聞台 01"/>{{sfnm|蘇振明|2011年|1loc=第99頁、第103頁至第104頁}}。他經常背著臺灣民族主義的掛圖，前往歐美各地臺灣同鄉會巡迴宣講，並為演講準備手繪海報和手繪漫畫{{sfnm|蘇振明|2011年|1loc=第103頁至第110頁、第116頁至第119頁}}。直至1992年前後，他便往訪美國共15次{{sfnm|蘇振明|2011年|1loc=第171頁}}。同年，他協助《[[美麗島週報|美麗島週報]]》渡過財務危機<ref name = "許建榮 02"/>。1982年，「獨立臺灣會」在美國出版《[[臺灣大眾|臺灣大眾]]》，積極倡導「臺灣民族主義」為「臺灣獨立運動」的理念{{sfnm|蘇振明|2011年|1loc=第99頁、第171頁}}。1983年，其捐款[[郭倍宏|郭倍宏]]等人在美國成立的「[[台灣學生社|台灣學生社]]」，出版學生運動刊物《[[台灣學生|台灣學生]]》<ref name = "許建榮 02"/>。但在同年，當時的大學系主任盧修一因為與史明的得力助手[[前田光枝|前田光枝]]接觸，被以涉嫌叛亂為由遭到逮捕，並交付感化3年<ref name = "民視新聞台 01"/>，這次事件也導致史明第二度被中國國民黨政府通緝{{sfnm|黃英哲|1991年|1loc=第28頁|陳銘城|1992年|2loc=第74頁}}。1986年，《臺灣人四百年史》英文版於美國[[華盛頓哥倫比亞特區|華盛頓哥倫比亞特區]]出版{{sfnm|蘇振明|2011年|1loc=第171頁}}。

== 臺灣活動 ==
=== 返回臺灣 ===
{{Quote box| width = 300px | align = left | quoted = true | bgcolor = #FFFFF0 | salign = right | quote = 我回臺灣並不是並不是回歸故鄉，而是為了推翻國民黨獨裁集團！ | source= 史明{{sfnm|蘇振明|2011年|1loc=第126頁、第135頁}} }}

1987年7月15日，在[[中華民國|中華民國]]解除《[[臺灣省戒嚴令|臺灣省戒嚴令]]》後，由於《[[懲治叛亂條例|懲治叛亂條例]]》尚未廢除，史明仍然被視為叛亂分子，《臺灣人四百年史》等著作仍然屬於禁書<ref name = "民視新聞台 01"/>{{sfnm|陳銘城|1992年|1loc=第4頁|蘇振明|2011年|2loc=第90頁、第122頁至第123頁}}。1991年，獨立臺灣會會員[[廖偉程|廖偉程]]等4人，因為研讀史明的著作，被以「分裂國家、顛覆政府」的涉嫌叛亂理由[[獨立台灣會案|遭到逮捕]]<ref name = "民視新聞台 01"/>，臺灣各地的大專院校師生舉行靜坐遊行抗議{{sfnm|陳銘城|1992年|1loc=第232頁至第233頁、第247頁|蘇振明|2011年|2loc=第123頁至第124頁、第172頁}}。到了1993年10月26日，有鑑於島內的政治形勢轉變，史明以「臺灣最後一位[[黑名單_(臺灣)|黑名單]]」的身分，先行潛返回到家鄉臺灣<ref name = "獨台會">{{Cite web |url = https://web.archive.org/web/20161219162306/http://tw400.org.tw/3about-tati/3about-tati.html |title = 關於獨台會 |author =  |date =  |publisher = [[獨立臺灣會|獨立臺灣會]] |language = Zh-hant |accessdate = 2019年3月24日 |quote =  }}</ref>，結束流亡海外41年的生活{{sfnm|Claude Geoffroy|1997年|2loc=第64頁|蘇振明|2011年|2loc=第8頁、第121頁、第124頁至第126頁、第135頁至第136頁、第172頁}}。

當時雖然已經是《臺灣省戒嚴令》解除後的第六年，由於史明被中國國民黨通緝的罪名尚未結案，因此他在[[中山高速公路|中山高速公路]][[新營交流道|新營交流道]]收費站前，被中國國民黨的臺灣警備總司令部跟蹤逮捕{{sfnm|蘇振明|2011年|1loc=第121頁、第124頁至第126頁、第135頁至第136頁、第172頁}}。隨後潛返被捕的他以10萬元交保，並在隔日召開記者會宣示立場{{sfnm|蘇振明|2011年|1loc=第121頁、第124頁至第126頁、第135頁至第136頁、第172頁}}。儘管在1994年10月遭到起訴，最後仍以保釋結案，過往的暴力行為則不追究{{sfnm|Claude Geoffroy|1997年|1=第303頁|蘇振明|2011年|2loc=第121頁、第172頁}}。他也驚訝地發現自己在臺北市士林區公所的[[戶籍謄本|戶籍謄本]]，已經被宣告在1964年3月31日死亡{{sfnm|蘇振明|2011年|1loc=第127頁}}。後來他以出生本名「施朝輝」進行法院訴訟，在1995年6月回復合理的戶籍身分{{sfnm|蘇振明|2011年|1loc=第127頁}}。同時，他還為家族建造「施林墓園」{{sfnm|蘇振明|2011年|1loc=第127頁至第128頁}}。

基於個人理念和獨立臺灣會的宗旨，史明後來仍持續推動臺灣民族革命，以及實踐臺灣島內的獨立建國運動{{sfnm|蘇振明|2011年|1loc=第126頁、第130頁、第145頁至第146頁}}。不過順應臺灣社會的變遷，他改採取「啟蒙、覺悟、自決、行動」的軟性革命策略，將臺灣民眾的思想啟蒙列為最重要的目標<ref name = "周俊男 03"/>，並啟動街頭獨立運動{{sfnm|蘇振明|2011年|1loc=第121頁、第130頁}}。透過遊行、宣講、研習與出版等活動，他積極地向臺灣民眾宣揚臺灣民族主義、及臺灣獨立建國運動的理念主張{{sfnm|蘇振明|2011年|1loc=第130頁、第140頁、第145頁至第146頁、第156頁}}。1991年12月30日，史明出版《台灣不是中國的一部份》<ref name = "書籍"/>。1992年6月20日，史明出版《台灣形成與台灣民族》<ref name = "書籍"/><ref name = "活動內容">{{Cite web |url = https://web.archive.org/web/20160827062926/http://tw400.org.tw/2act-new/2-actdetail/2-actdetail.html |title = 活動內容詳介 |author =  |date =  |publisher = [[獨立臺灣會|獨立臺灣會]] |language = Zh-hant |accessdate = 2019年3月24日 |quote =  }}</ref>。1993年10月20日，他則在東京出版《台灣民族革命與社會主義》<ref name = "許建榮 02"/><ref name = "書籍"/><ref name = "活動內容"/>。這時期，史明還拒絕與中華民國殖民體制妥協，並未競逐任何官職<ref name = "周俊男 01"/>。

=== 轉移基地 ===
[[File:Taiwan_Independence_Association_P5173889_(3539598306).jpg|thumb]]」。]]

早在1993年，史明便計劃將獨立臺灣會轉往到臺灣島內運作，先是在[[高雄縣|高雄縣]][[鳳山市|鳳山市]]成立辦事處{{sfnm|蘇振明|2011年|1loc=第130頁}}。1994年，他把獨立台灣會海外總部從東京遷回臺北市，正式成立「獨立臺灣會」臺北市總部<ref name = "獨台會"/>{{sfnm|蘇振明|2011年|1loc=第172頁}}。其後相繼在[[高雄市|高雄市]]、[[嘉義市|嘉義市]]、[[臺中市|台中市]]、[[台東縣|台東縣]]、[[新竹市|新竹市]]等地成立獨立臺灣會的聯絡處<ref name = "獨台會"/>。到了8月，史明率領獨立臺灣會的車隊，前往[[桃園市|桃園縣]][[臺灣桃園國際機場|中正國際機場]]和[[國立故宮博物院|國立故宮博物院]]，抗議中國共產黨的[[海峽兩岸關係協會|海峽兩岸關係協會]]副會長前來臺灣，並與中國國民黨的[[海峽交流基金會|海峽交流基金會]]會談{{sfnm|蘇振明|2011年|1loc=第172頁}}。

史明還善用自日本帶回的約新臺幣6,000萬元基金，陸續實踐「舉辦獨臺會社運同志研習教室」、「組織獨立建國宣傳車隊」、「設立臺灣大眾廣播電臺」三項獨立臺灣會的業務{{sfnm|蘇振明|2011年|1loc=第121頁、第130頁}}。1996年底起，為了進行社會民眾的思想啟蒙工作，由其提供資金，在臺北市和平東路成立「[[臺灣大眾廣播電臺|臺灣大眾廣播電臺]]」（FM99.3），透過媒體教育促成臺灣民眾獨立意識的政治覺醒，並持續至1999年{{sfnm|蘇振明|2011年|1loc=第132頁至第133頁、第136頁、第172頁}}。在這段期間，他還主持晚上7時到晚上9時的「我愛臺灣」節目，講述臺灣歷史時事評論{{sfnm|蘇振明|2011年|1loc=第132頁至第133頁}}。而為了提高臺灣民眾對於中國國民黨霸權政治的警覺性，史明在臺灣各都會啟動獨立臺灣會的「[[獨立建國宣傳車隊|獨立建國宣傳車隊]]」計劃{{sfnm|蘇振明|2011年|1loc=第121頁、第136頁、第172頁}}。每逢周六、日下午，這些宣傳車隊透過敲鑼打鼓方式，長期在街頭宣揚「臺灣民族主義」{{sfnm|蘇振明|2011年|1loc=第121頁、第136頁、第172頁}}。

1998年4月，《臺灣人四百年史》漢文增訂版出版<ref name = "書籍"/>{{sfnm|蘇振明|2011年|1loc=第172頁}}。1999年，基於悼念祖先過去為族群公共利益的犧牲奉獻、以及發揚臺灣「[[台灣中元法會|7月普渡]]」的民俗文化，史明曾經以獨立台灣會的名義，首次在[[新莊區|新莊區]]舉行主導祭拜大墓公活動，之後持續舉辦數次普渡活動<ref name = "周俊男 02">{{Cite web |url = https://gushi.tw/life-of-shi-ming-2/ |title = 台灣．民族．主義：史明歐吉桑的革命人生（中） |author = 周俊男 |date = 2016年1月28日 |publisher = 故事 |language = Zh-hant |accessdate = 2019年3月24日 |quote =  |archive-url = https://web.archive.org/web/20180609100945/https://gushi.tw/life-of-shi-ming-2/ |archive-date = 2018年6月9日 |dead-url = no }}</ref>{{sfnm|蘇振明|2011年|1loc=第40頁、第172頁}}。2000年，[[中華民國總統|中華民國總統]][[陳水扁|陳水扁]]在[[民主進步黨|民主進步黨]][[2000年中華民國總統選舉|贏得政權]]後拜會史明，後者在當時表示不需要特別有所安排，但是建議陳水扁必須做到謹言慎行、用人嚴謹、及研究中國共產黨三件事情<ref name = "許建榮 02"/>。2001年10月，其著作《臺灣民族主義與臺灣獨立革命》、《我的故鄉，八芝蘭—士林》出版<ref name = "書籍"/><ref>{{Cite web |url = https://web.archive.org/web/20161219145150/http://tw400.org.tw/1sitebrief/1oldtime/1oldtime.html |title = 昔日的故鄉《書—我的故鄉 八芝蘭—士林》 |author =  |date =  |publisher = [[獨立臺灣會|獨立臺灣會]] |language = Zh-hant |accessdate = 2019年3月24日 |quote =  }}</ref>{{sfnm|蘇振明|2011年|1loc=第172頁}}。同年11月9日，史明教育基金會於臺北市[[國賓大飯店|國賓大飯店]]成立<ref>{{Cite web |url = https://web.archive.org/web/20161219202531/http://tw400.org.tw/7foundation/7foundation.html |title = 財團法人 史明教育基金會—宗旨、組織 |author =  |date =  |publisher = [[獨立臺灣會|獨立臺灣會]] |language = Zh-hant |accessdate = 2019年3月24日 |quote =  }}</ref>{{sfnm|蘇振明|2011年|1loc=第172頁}}。

=== 反對行動 ===

== 思想主張 ==
=== 民族主義 ===
[[File:【哲學星期五@台北】如何喚醒一個沉睡的民族─台灣獨立運動的歷史與展望_20160311.webm|thumb]]

史明被視為「[[臺獨左派|臺獨左派]]」<ref>{{Cite web |url = http://www.thinkingtaiwan.com/content/262 |title = 孤獨但巨大的形象 － 左翼、台灣民族主義旗手、革命家、流亡者史明 |author = [[李敏勇|李敏勇]] |date = 2012年11月8日 |publisher = 想想 |language = Zh-hant |accessdate = 2019年3月24日 |quote =  |archive-url = https://web.archive.org/web/20160405191631/http://www.thinkingtaiwan.com/content/262 |archive-date = 2016年4月5日 |dead-url = no }}</ref>{{sfnm|黃英哲|1991年|1loc=第26頁|陳銘城|1992年|2loc=第4頁}}。其思想主張結合馬克思的社會主義、臺灣社會的[[人文主義|人文主義]]與臺灣民族主義，並受到高田早苗史學觀思想的影響{{sfnm|蘇振明|2011年|1loc=第168頁}}。儘管排斥中華人民共和國的共產主義，但他從未放棄對於馬克思主義的信仰{{sfnm|Claude Geoffroy|1997年|1loc=第64頁}}。其中，為了保存臺灣歷史、及替革命整理出一套理論基礎，流亡時期的他轉而潛心理論研究與寫作，獨自整理臺灣歷史，最終在1962年出版《臺灣人四百年史》日文版<ref>{{Cite web |url = https://web.archive.org/web/20160827073306/http://tw400.org.tw/4thoughts/4wrt-his/4wrt-his.html |title = 為何寫台灣人四百年史 |author =  |date =  |publisher = [[獨立臺灣會|獨立臺灣會]] |language = Zh-hant |accessdate = 2019年3月24日 |quote =  }}</ref>。而在海外流亡期間，史明還主張「臺灣民族主義」與「獨立建國運動」這兩路線，並在返回臺灣後繼續堅持台灣獨立革命運動的路線<ref name = "活動內容"/>{{sfnm|蘇振明|2011年|1loc=第156頁、第168頁}}。

史明在其出版的《臺灣人四百年史》中，建構臺灣人史觀的理念，亦強調該書是基於臺灣人的立場和觀點撰寫的臺灣史書{{sfnm|黃英哲|1991年|1loc=第29頁|蘇振明|2011年|2loc=第88頁至第89頁、第94頁至第95頁、第110頁、第114頁}}。他認為必須以「[[唯物辯證法|唯物辯證史觀]]」等方法，剖析臺灣社會的發展{{sfnm|黃英哲|1991年|1loc=第30頁}}。而當中最重要的核心思想理念便是臺灣民族主義<ref name = "民族主義">{{Cite web |url = https://web.archive.org/web/20161203191345/http://www.tw400.org.tw/4thoughts/4principle/4principle.html |title = 台灣民族主義 |author =  |date =  |publisher = [[獨立臺灣會|獨立臺灣會]] |language = Zh-hant |accessdate = 2019年3月24日 |quote =  }}</ref>，即「台灣人關心台灣前途而形成想要建立台灣成為獨立國家，想要建設國民經濟，同時想要發展固有台灣文化的思想和行動」<ref name = "革命理論">{{Cite web |url = https://web.archive.org/web/20160827071400/http://tw400.org.tw/4thoughts/4rev-2ph/4rev-2ph.html |title = 台灣獨立２階段革命理論 |author =  |date =  |publisher = [[獨立臺灣會|獨立臺灣會]] |language = Zh-hant |accessdate = 2019年3月24日 |quote =  }}</ref>{{sfnm|蘇振明|2011年|1loc=第95頁}}。史明認為臺灣民族主義的形成，是基於臺灣在歷史、社會上都不是中國的一部分，而在文化上亦發展出臺灣固有的[[臺灣文化|文化]]<ref name = "周俊男 02"/><ref name = "民族主義"/>{{sfnm|黃英哲|1991年|1loc=第30頁|蘇振明|2011年|2loc=第4頁|江宗翰|2012年|3loc=第21頁至第25頁、第78頁至第81頁}}。

史明因而提出臺灣人意識論述，並基於此建立有關中國認識與殖民背景的思考{{sfnm|江宗翰|2012年|1loc=第1頁至第8頁、第17頁至第19頁}}。其提出的史觀展開與發展，則排除關於[[臺灣原住民族|臺灣原住民族]]和中國認同的因素，而基於[[漢族|漢族]]中心史觀與[[去中國化|去中國]]史觀，形成由民族主義與社會主義結合的[[無產階級|無產階級]]史觀<ref name = "周俊男 02"/>{{sfnm|江宗翰|2012年|1loc=第25頁至第28頁、第51頁至第75頁、第150頁至第152頁}}。他認為臺灣的歷史便是「臺灣無產階級的被殖民歷史」，這包括[[臺灣荷蘭統治時期|臺灣荷蘭統治時期]]、[[明鄭時期|明鄭時期]]、[[臺灣清治時期|臺灣清治時期]]、[[臺灣日治時期|臺灣日治時期]]，以及中華民國的蔣中正[[民國軍閥|軍閥]]政權的殖民統治{{sfnm|黃英哲|1991年|1loc=第30頁|蘇振明|2011年|2loc=第4頁|江宗翰|2012年|3loc=第25頁至第50頁、第57頁至第66頁、第92頁至第96頁}}。同時，基於去中國史觀和無產階級史觀，他對於中國的認識呈現出排除與不滿的立場{{sfnm|江宗翰|2012年|1loc=第77頁至第98頁、第152頁至第153頁}}。

爾後史明還對臺灣民族的構成有更明確的表述，主張以「[[社會階級|社會階級]]/臺灣民族」的方式區分，由為數最多的無產階級作為主體，而[[中產階級|中產階級]]、[[資產階級|資產階級]]為輔，共同構成台灣民族<ref name = "周俊男 02"/>；並且以推翻殖民制度，而非推翻殖民者為主要的目的{{sfnm|江宗翰|2012年|1loc=第46頁至第48頁、第66頁至第73頁}}。這意味著其認為只要能夠開明理解臺灣民族主義及臺灣獨立運動的必然性，任何人都可以成為臺灣人<ref name = "周俊男 02"/>{{sfnm|江宗翰|2012年|1loc=第47頁至第48頁}}。因此，他將在1949年以後從中國大陸轉往臺灣的部分[[外省人|外省人]]，同樣視為臺灣人的一部分，因為這些人共同受到中國國民黨政府的壓迫，並且逐漸融入臺灣人當中{{sfnm|江宗翰|2012年|1loc=第47頁至第48頁、第148頁}}。

=== 政治革命 ===
{{Quote box| width = 300px | align = left | quoted = true | bgcolor = #FFFFF0 | salign = right | quote = 通過組織，來壯大大眾的力量，才能打倒殖民體制，才能保障我們在台灣共和國所建設的民主，不再被壟斷。 | source= 史明<ref name = "革命理論"/> }}

透過在1942年至1949年的中國共產黨經驗，史明摸索出地下工作的種種細節，成為往後其在日本從事獨立運動的工作方式<ref name = "周俊男 01"/>。他主張臺灣的資本主義仍然是一種「跛腳的資本主義」，一方面要在政治上推翻中國國民黨的殖民體制、並且建立獨立的國家，另一方面則要在經濟上建立國民經濟<ref name = "民族主義"/><ref>{{Cite web |url = https://web.archive.org/web/20160827073203/http://tw400.org.tw/4thoughts/4faq-ti/4faq3/4faq3.html |title = 台灣獨立革命—5問5答 |author =  |date =  |publisher = [[獨立臺灣會|獨立臺灣會]] |language = Zh-hant |accessdate = 2019年3月24日 |quote =  }}</ref>，進而提出臺灣獨立運動的[[二階段革命理論|二階段革命理論]]<ref name = "周俊男 02"/><ref name = "革命理論"/>。其中第一個革命階段是「民族、民主革命」，認為首先要根除外來的殖民勢力，並且建立民主制度<ref name = "周俊男 02"/><ref name = "革命理論"/>{{sfnm|江宗翰|2012年|1loc=第70頁至第72頁}}。史明強調民族革命並非要消滅中國人，而是要「打倒中國國民黨集團及其幫凶的台灣[[買辦|買辦]]所形成的殖民體制」，並使他們無法繼續操縱臺灣的[[台灣政治|政治]]、[[臺灣經濟|經濟]]、[[台灣軍事|軍事]]、文化<ref name = "革命理論"/>。

不過史明亦認為，如果僅有中產階級取代中國國民黨，並且形成新的[[社會特權|特權階級]]，對於絕大多數的台灣人民而言，並未真正達到民主革命的目的<ref name = "革命理論"/>。因此主張應當要有第二階段的「社會主義革命」，最終要求建立一個人人「[[各盡所能、各取所需|各盡所能、各取所需]]」的國家<ref name = "周俊男 02"/><ref name = "革命理論"/>{{sfnm|江宗翰|2012年|1loc=第70頁至第72頁}}。史明還曾經具體地表示，當前的臺灣仍處在中華民國的殖民體制統治之下，而為了要達成臺灣民族的獨立建國目標，必須要採取「體制外革命」和「體制內改革」二條路線<ref>{{Cite web |url = https://web.archive.org/web/20160827001638/http://tw400.org.tw/4thoughts/4faq-ti/4faq-ti.html |title = 為什麼要獨立—理念、理論 |author =  |date =  |publisher = [[獨立臺灣會|獨立臺灣會]] |language = Zh-hant |accessdate = 2019年3月24日 |quote =  }}</ref><ref>{{Cite web |url = https://web.archive.org/web/20160827012601/http://tw400.org.tw/4thoughts/4faq-ti/4faq2/4faq2.html |title = 台灣獨立革命—為何、革命 |author =  |date =  |publisher = [[獨立臺灣會|獨立臺灣會]] |language = Zh-hant |accessdate = 2019年3月24日 |quote =  }}</ref><ref>{{Cite web |url = https://web.archive.org/web/20160827071653/http://tw400.org.tw/4thoughts/4faq-ti/4faq4/4faq4.html |title = 台灣獨立革命—匯編 |author =  |date =  |publisher = [[獨立臺灣會|獨立臺灣會]] |language = Zh-hant |accessdate = 2019年3月24日 |quote =  }}</ref>，以及「在臺灣島內改革與革命」和「在國際關係上與美國、日本結盟」的二個工作方向{{sfnm|蘇振明|2011年|1loc=第134頁至第135頁}}。

相對地，史明認為臺灣最嚴重的危機是可能遭到中國國民黨[[海峽兩岸關係|出賣]]，以致於自身被中華人民共和國的[[中國統一|併吞]]，因此主張民族民主革命是現階段最為緊要的課題，而等到獨立之後，台灣才能夠進行更進一步的改造運動<ref name = "革命理論"/>{{sfnm|蘇振明|2011年|1loc=第147頁至第149頁}}。他還認為中國國民黨只是中國共產黨在臺灣的「看門狗」，而中國共產黨則會成為中國國民黨永遠殖民臺灣的後臺{{sfnm|蘇振明|2011年|1loc=第138頁至第139頁、第147頁至第149頁}}。而從臺灣和臺灣人的觀點來看，他認為中國國民黨的中華民國和中國共產黨的中華人民共和國，同樣都是阻礙臺灣實踐獨立建國的兩個革命對象與敵人{{sfnm|蘇振明|2011年|1loc=第138頁至第139頁、第147頁至第149頁}}。

== 個人生活 ==
[[File:11.08_總統出席史明百歲生日會_(38226043862).jpg|thumb]][[蔡英文|蔡英文]]出席史明百歲生日會。]]

史明經歷日本殖民統治及中國國民黨在臺灣初期的專制統治，之後流亡日本著書寫作，直到1993年才返回臺灣{{sfnm|黃英哲|1991年|1loc=第31頁|江宗翰|2012年|2loc=第5頁}}。由於強調清楚認識歷史的重要性，因此其在《臺灣人四百年史》日文版出版後，開始將其筆名取做「史明」，寓意即「把歷史弄明暸」<ref name = "周俊男 02"/><ref>{{Cite web |url = http://focustaiwan.tw/news/aftr/201711050016.aspx |title = Legendary revolutionist Su Beng places hope on younger generation |author = Shih Hsiu-chuan |date = 2017年11月5日 |publisher = [[中央通訊社|中央通訊社]] |language = en |accessdate = 2019年3月24日 |quote =  |archive-url = https://web.archive.org/web/20181011013641/http://focustaiwan.tw/news/aftr/201711050016.aspx |archive-date = 2018年10月11日 |dead-url = no }}</ref>，希望能夠讓臺灣歷史更加明瞭、且更廣為人們知悉<ref name = "許建榮 02"/>{{sfnm|蘇振明|2011年|1loc=第20頁、第89頁}}。他還經常被其他人稱作「歐吉桑」（{{lang|ja|おじさん}}）<ref name = "許建榮 01"/>。而不同於許多台灣獨立運動者，他在過去就已經被視為政治犯，並且擁有日本所提供的政治流亡居留權{{sfnm|江宗翰|2012年|1loc=第136頁}}。而在當前，他則是居住在[[新北市|新北市]]新莊區的住處<ref name = "周俊男 03">{{Cite web |url = https://gushi.tw/life-of-shi-ming-3/ |title = 直到台灣獨立那天：史明歐吉桑的革命人生（下） |author = 周俊男 |date = 2016年1月28日 |publisher = 故事 |language = Zh-hant |accessdate = 2019年3月24日 |quote =  |archive-url = https://web.archive.org/web/20180609101017/https://gushi.tw/life-of-shi-ming-3/ |archive-date = 2018年6月9日 |dead-url = no }}</ref>。

史明的生活規律簡樸，很少將金錢花在自身的物質享受上，經常穿著簡單上衣、牛仔褲和牛仔大衣<ref name = "許建榮 01"/>{{sfnm|蘇振明|2011年|1loc=第102頁、第151頁}}。其身高約有180公分，自稱食量比年輕人還大，並且即便在冬天也保持每周游泳的習慣，並且有著大量的行程<ref name = "周俊男 03"/>{{sfnm|黃英哲|1991年|1loc=第26頁}}。不過其左眼因[[白內障|白內障]]開刀失敗而失明<ref name = "民視新聞台 01"/>，而且腰骨不好<ref name = "周俊男 03"/>，出門必須乘坐輪椅，對民眾演講也經常需要撐著枴杖{{sfnm|蘇振明|2011年|1loc=第115頁}}。在早年前往中國大陸參加抗日戰爭的地下工作時，儘管其身為施家的長孫，因為自己認為從事革命工作者不應該有子嗣家庭，因此在動身前就決定到上海市[[北四川路|北四川路]]一家日本人醫院，接受[[結紮|結紮]][[輸精管切除術|手術]]{{sfnm|陳銘城|1992年|1loc=第73頁至第74頁|蘇振明|2011年|2loc=第54頁}}。

史明所有的革命與宣傳活動資金，均靠經營「新珍味中華料理店」的餐飲事業收入，從未公開募款{{sfnm|黃英哲|1991年|1loc=第31頁|陳銘城|1992年|2loc=第74頁|蘇振明|2011年|3loc=第96頁、第102頁、第160頁}}。對於外界曾經批評其接受中國共產黨的資金，他則是鄭重否認該說法{{sfnm|陳銘城|1992年|1loc=第74頁}}。在中國大陸時期，史明曾經向前[[日本駐華大使館|日本駐華大使館]]的職員平賀協子表白，兩人後來共同生活一陣子<ref name = "民視新聞台 01"/>{{sfnm|蘇振明|2011年|1loc=第90頁、第102頁}}。但是在生活困難的情況下，由於史明仍然不時寄送資金回臺灣，以提供地下組織的同志從事革命行動，這引起妻子的不安{{sfnm|江宗翰|2012年|1loc=第136頁|陳銘城|1992年|2loc=第73頁|蘇振明|2011年|3loc=第90頁、第102頁}}。最後堅持臺灣獨立運動立場的史明為了不拖累妻子，在1963年決定與平賀協子離婚，後來史明便一直獨居{{sfnm|黃英哲|1991年|1loc=第31頁|陳銘城|1992年|2loc=第73頁|蘇振明|2011年|3loc=第90頁|江宗翰|2012年|4loc=第136頁、第102頁至第103頁}}。

== 造成影響 ==
[[File:《藝想世界》姚文智、陳麗貴談史明紀錄片《革命進行式》.webm|thumb]]和[[陳麗貴|陳麗貴]]為史明製作紀錄片《[[史明·革命進行式|史明·革命進行式]]》。]]

自從1950年組織武裝隊反抗蔣中正後，史明長期致力於臺灣的獨立建國運動<ref>{{Cite web |url = https://web.archive.org/web/20161018112922/http://tw400.org.tw/1sitebrief/1sitebrief.html |title = 史明一生的傳奇故事 與 獨台會的因緣 |author =  |date =  |publisher = [[獨立臺灣會|獨立臺灣會]] |language = Zh-hant |accessdate = 2019年3月24日 |quote =  }}</ref>，且為了傳達革命理念而有大量著作論述<ref name="活動內容" />。他被視為[[左派|左派]]臺灣獨立運動的領導人<ref name = "民視新聞台 01"/>，是臺灣獨立運動的主要先驅者，也是早期臺灣獨立運動人士中少數主張馬克思主義的人{{sfnm|黃英哲|1991年|1loc=第31頁|Claude Geoffroy|1997年|2loc=第62頁|蘇振明|2011年|3loc=第4頁、第8頁}}。其在日本時期就是有名的社會主義者，除了以《臺灣人四百年史》成名外，也是獨立臺灣會的創辦人{{sfnm|黃英哲|1991年|1loc=第26頁|蘇振明|2011年|2loc=第105頁至第106頁}}。其對於臺灣社會的主要影響，包括建構出臺灣史觀、啟蒙許多臺灣人的意識和價值觀，甚至因而被視為「臺灣獨立運動教父」、「臺灣獨立建國的精神導師」{{sfnm|蘇振明|2011年|1loc=第111頁、第168頁}}。包括[[蔡丁貴|蔡丁貴]]、[[簡余晏|簡余晏]]、[[吳叡人|吳叡人]]、[[江昺崙|江昺崙]]、[[林飛帆|林飛帆]]、[[張之豪|張之豪]]、[[藍士博|藍士博]]等人，都曾經受到史明的影響{{sfnm|蘇振明|2011年|1loc=第106頁至第107頁、第152頁至第153頁、第154頁至第155頁|江俊宜等|2012年}}。

對於海內外的左翼臺灣獨立運動者及留學生來說，史明更被視為臺獨左派理論的開創者{{sfnm|陳銘城|1992年|1loc=第74頁}}。作為介紹臺灣歷史的書籍，他在1962年完成的《臺灣人四百年史》宣揚[[臺灣主體性|臺灣主體意識]]，是第一部以臺灣人立場撰寫、且橫跨年代最久遠的臺灣通史<ref name = "民視新聞台 01">{{Cite web |url = https://www.youtube.com/watch?v=N64O6iBbkzA |title = 2017.11.05【台灣演義】永遠的革命家 史明 |author = [[民視新聞台|民視新聞台]] |date =  |publisher = [[YouTube|YouTube]] |language = Zh-hant |accessdate = 2019年3月24日 |quote =  }}</ref>{{sfnm|蘇振明|2011年|1loc=第87頁至第89頁|江宗翰|2012年|2loc=第147頁至第148頁}}。在中國國民黨的戒嚴時期，他便在《臺灣人四百年史》中率先剖析二二八事件{{sfnm|蘇振明|2011年|1loc=第77頁}}。不過《臺灣人四百年史》出版之後，也被列為禁書長達30年<ref name = "民視新聞台 01"/>{{sfnm|蘇振明|2011年|1loc=第90頁}}。直到1987年解除《臺灣省戒嚴令》後，臺灣的歷史學家才開始參與二二八事件的調查，逐漸釐清事件的經過{{sfnm|蘇振明|2011年|1loc=第77頁}}。相對地，針對其所提出的「臺灣民族主義」理念，也曾遭遇[[台灣獨立建國聯盟|台灣獨立建國聯盟]]等聽眾、學者的質疑或反對{{sfnm|蘇振明|2011年|1loc=第111頁、第114頁、第157頁}}。而針對此類質問，他亦有更進一步的論述回答{{sfnm|蘇振明|2011年|1loc=第114頁|江宗翰|2012年|2loc=第1頁至第2頁}}。

過去在歐美各地曾經參與史明宣講的海外青年學者，在1987年《臺灣省戒嚴令》解除後陸續回國，在海內外的學術界、藝文界和臺灣社會運動界，扮演著極重要的關鍵角色{{sfnm|蘇振明|2011年|1loc=第110頁}}。2013年，臺灣重金屬樂團[[閃靈樂團|閃靈樂團]]的專輯《[[武德_(專輯)|武德]]》，收錄的歌曲《共和》中便有史明歌唱《臺灣民族主義》的部分內容<ref>{{Cite web
