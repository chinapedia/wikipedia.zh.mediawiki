'''史蒂文·普赖斯''' （{{lang-en|'''Steven Price'''}}，1977年4月22日-）是一名[[英国|英国]][[电影配乐|电影配乐]]家，主要作品为2013年的《[[世界尽头_(电影)|世界尽头]]》和《[[地心引力_(电影)|地心引力]]》<ref name="IMDB">{{cite web |url=http://www.imdb.com/name/nm1888527/ |title=Steven Price (II) |publisher=Internet Movie Database |accessdate=2013-10-27}}</ref><ref name="AllMusic">{{cite web |url=http://www.allmusic.com/artist/steven-price-mn0002759735/credits |title=Steven Price |publisher=AllMusic |accessdate=2013-10-27}}</ref>，并以后者获得[[第67届英国电影学院奖|第67届英国电影学院奖]]和[[第86届奥斯卡金像奖|第86届奥斯卡金像奖]]最佳原创音乐。
