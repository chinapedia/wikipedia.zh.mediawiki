'''史密斯威森M39'''是[[美國軍隊|美國軍隊]]於1954年測試的一種[[半自動手槍|半自動手槍]]<ref name="EPR2002">{{cite book| last = Hartink| first = A.E.| title = The Complete Encyclopedia of Pistols and Revolvers| publisher = Chartwell Books, Inc| year = 2002| pages = 87–88| location = Edison, New Jersey| isbn = 978-0-7858-1519-8}}</ref>。其後因美軍放棄尋找新的手槍，此槍在1955年被改在民用市場上銷售，亦是[[史密斯威森|史密斯威森]]公司的第一代半自動手槍。由於曾被[[美國海軍|美國海軍]][[海豹部隊|海豹部隊]]以及[[伊利諾伊州|伊利諾伊州]][[警察|警察]]採用過，所以此槍被普遍認為是一種相當具有突破性的手槍<ref name="EPR2002"/>。

== 概述 ==
史密斯威森M39是第一種[[美國|美國]]製、並在當地市場銷售的雙動式半自動手槍。[[二次大戰|二次大戰]]期間，[[德國|德國]][[卡爾·瓦爾特運動槍有限公司|瓦爾特公司]]推出了一種名為[[瓦爾特P38手槍|P38]]的雙動式半自動手槍，該槍的出現令美國的軍械人員留下了深刻的印象。而陸軍軍械總隊更提出了一個方案，就是開發出一種類似於P38的雙動式手槍。於1949年，史密斯威森開始了對M39手槍的研發，於1955年才正式商業銷售此槍。最早推出的M39為第一代手槍，其後史密斯威森不斷對其作出各種改進，最終推出了現在於市場上銷售的第三代版本。不同世代的手槍在型號命名上皆有著差異，第一代的使用兩位數字、第二代使用三位數字，如此類推<ref name="EPR2002"/>。

M39最初是使用[[陽極氧化|陽極氧化]]的鋁製槍身，彎曲的握把背護片，以及一個連手動保險的鋼製[[手槍套筒|滑套]]。其握把護片均為木製品；彈匣釋放鈕位於[[扳機護圈|扳機護圈]]後面，這些方面與當時美軍使用的[[M1911手槍|M1911手槍]]十分類似<ref name="EPR2002"/>。

另外此槍也有一種使用鋼製槍身的版本，然而它的生產量有限。儘管如此，其鋼製槍身卻為後來的M52手槍埋下了基礎。M52有著較長的槍管（5吋）和滑套，亦是少數地發射[[.38_Special|.38特種彈]]的半自動手槍。M52的載彈量為5發；該槍已於1992年停產<ref name="EPR2002"/>。M52的衍生型，M952是一種發射9毫米魯格彈的半自動手槍，這個版本至今仍然有限的被史密斯威森性能中心所生產。

M39的設計也為後來的[[史密斯威森M59手槍|史密斯威森M59手槍]]埋下了基礎。M59依舊是發射9毫米魯格彈，但它使用了更寬闊的槍身和直形的握把背護片以對應14發容量的雙排彈匣<ref name="EPR2002"/>。

M39採用了許多來自[[瓦爾特P38手槍|瓦爾特P38手槍]]的設計，當中包括其待擊解脫桿，以及其8發容量的彈匣。但在槍管和滑套方面M39卻採自傳統的半自動手槍設計，而非像P38的外露式槍管設計。另外的一些機制則取自[[白朗寧大威力手槍|白朗寧大威力手槍]]。

該槍全長7.6吋，槍管長4吋，槍重1.72磅。會使它重量低的主要原因是其使用了鋁製槍身<ref name="EPR2002"/>。

[[伊利諾伊州|伊利諾伊州]][[警察|警察]]於1967年採用M39，這一舉動令許多執法部門也相繼地採用半自動手槍，同時也為更適合警方於特別行動時使用的M59手槍提供了商機和宣傳。

而在[[越戰|越戰]]期間，[[美國海軍|美國海軍]][[特種部隊|特種部隊]]也採用了M39<ref name ="SEALS">{{cite book |last=Dockery |first=Kevin|title=Weapons of the Navy SEALs|year= 2004|publisher=Berkeley Hardcover|location=California |isbn=0-425-19834-0}}</ref>。其中[[海豹部隊|海豹部隊]]採用了一種稱作'''Mk 22 Mod 0'''的改型，這種改型被暱稱為「靜寂的小狗」（Hush Puppy）<ref name ="SEALS"/>。這些手槍均裝上了[[抑制器|抑制器]]和滑套鎖（射擊時會維持滑套處於閉鎖狀態，從而降低噪音），因此它實際上是一支手動上膛和拋殼的手動式手槍。另外它們均有著較凸出的[[機械瞄具|機械瞄具]]，以令射手能更方便地瞄準，不用受那笨重的抑制器妨礙。Mk 22 Mod 0是讓特戰人員可在不需驚動主要目標的情況下殲滅敵方哨兵吠和守衛時使用<ref name="EPR2002"/>。

== 衍生型 ==
* Mk 22 Mod 0－美國[[美國特種作戰司令部|特種部隊]]於[[越南戰爭|越南戰爭]]期間裝備的消音版本，裝有[[抑制器|抑制器]]和滑套鎖，以及較高的[[機械瞄具|機械瞄具]]。
* M459 / M559 / M659－M59的第二代版本。M459使用[[鋁合金|鋁合金]]槍身、M559使用[[鋼|鋼]]製槍身，M659則使用不銹鋼製槍身。其中M559曾參加[[美國陸軍|美國陸軍]]的制式手槍競選，然而最終敗了給[[貝瑞塔92手槍|貝瑞塔92]]而沒有被採用。該系列於1980年代開始在美國的民用市場上銷售。
* M645－為史密斯威森的第一款[[.45_ACP|.45 ACP]]口徑雙動式[[半自動手槍|半自動手槍]]，此版本並有一種稱為「M745」的競賽級版本，並由原來的雙動式板機改為單動式。而M645的第三代版本則被稱為「M4506」。
* M3913－由M39稍作改良的縮短版本，主要提供給婦女用作防身用途而設。該版本已被[[日本警察|日本警察]]列為制式武器，並在2005年發現被[[北海道|北海道]]的[[特殊急襲部隊|特殊急襲部隊]]用作射擊訓練。其他多個部門亦有採用，並似乎已開始列裝到制服警察。
* M4506－M645的第三代版本，並有一種稱為「M4505」的黑色槍身型。
* M5904 - M59的衍生型，被[[台灣|台灣]][[警政署|警政署]]與[[刑事警察局|刑事警察局]]所使用，現正被[[瓦爾特PPQ手槍|Walther PPQ M2]]所取代。
* M5906－M59的第三代版本，目前已被[[日本海上保安廳|日本海上保安廳]]所採用。

* 3566－1993年推出的競賽級版本。
* M4006TSW－美國[[加州|加州]]公路巡警使用的特別版本，[[.40_S&W|.40 S&W]]口徑。
