'''史瑞克三世'''（{{lang-en|''Shrek the Third''}}）是一部[[電腦動畫|動畫]][[喜劇|喜劇]]電影，是《[[史瑞克系列電影|史瑞克系列]]》的第三部作品。本片制片由[[夢工廠|夢工廠]][[傑弗瑞·卡森伯格|傑弗瑞·卡森伯格]]完成，[[派拉蒙影業|派拉蒙影業]]發行，在[[美國|美國]]於2007年5月18日上映（墨西哥6月8日，西班牙6月22日，巴西6月15日，法国6月13日，英国爱尔兰6月29日），[[台灣|台灣]]和[[香港|香港]]則分別在2007年6月16日與6月28日上映。

在製作時的暫定片名為《'''史瑞克3'''》（'''''Shrek 3'''''）。和前兩集的電影一樣，《史瑞克三世》中改編了許多[[童話故事|童話故事]]（特別是[[格林童話|格林童話]]和[[迪士尼|迪士尼]]經典動畫）。

== 剧情 ==
哈洛德國王陷入重病，而他的怪物女婿[[史瑞克|史瑞克]]與女兒費安娜公主安全成為了國王與王后的繼承人。但是史瑞克斷然拒絕，他認為一定有其他人比他更適合這個工作。在吞下最後一口氣之前，國王告訴史瑞克有另外一個繼承人可以成為「很遠很遠的王國」的（Far Far Away）國王－也就是他的姪子[[亞瑟王|亞瑟]]，史瑞克帶著驢子和長靴貓外出去尋找這位繼承人。當他們出航之時，費歐娜跑到甲板上向史瑞克宣布自己懷孕的消息。在驚喜與震撼之下，史瑞克在前往[[烏斯特郡|烏斯特郡]]學院的旅程中，因為即將出世的孩子而開始產生許多夢魘。最後他們在這個學院發現了亞瑟－他是這個學院的學生，也是個常被其他學生找碴的輸家。亞瑟在一開始對於他新的身份感覺頗佳，直到驢子和長靴貓告訴他該負的責任而嚇到了他。亞瑟試著控制船隻，並且避免船隻撞上一座小島（在這小島上，他們遇到了亞瑟退休的巫師老師梅林）。

於此同時，白馬王子來到了[[毒蘋果酒吧|毒蘋果酒吧]]說服所有[[童話故事|童話故事]]中的反派人物（包括[[鐵鉤船長_(角色)|-{zh-hk:鐵鉤船長; zh-tw:虎克船長; zh-cn:铁钩船长;}-]]、白雪公主中的壞皇后、一位獨眼巨人、[[龍佩爾施迪爾欽|龍佩爾施迪爾欽]]（Rumpelstiltskin）、灰姑娘的醜繼姐Mable、[[無頭騎士|無頭騎士]]與Stromboli)，加入一場為他們「永遠幸福生活」而戰的鬥爭中。這些壞蛋們覺得屬於他們的故事從未被人講述過，而現在就是完成此事的最好時機。白馬王子與其他壞蛋入侵了王國，他們的犯下的罪行包含了囚禁費歐娜公主、莉莉安王后與費歐娜出名的公主朋友們（包括了[[灰姑娘|灰姑娘]]、[[睡美人|睡美人]]與[[白雪公主|白雪公主]]。）

當史瑞克回到城市之後，發現白馬王子為了報復史瑞克「偷走他永遠幸福快樂」的生活而進行的復仇行為，並計畫在一個夜晚殺死史瑞克。白馬王子的同黨很快速的逮到了史瑞克與亞瑟。公主們與王后計畫逃出監獄並且救出史瑞克。為了援救亞瑟，史瑞克告訴白馬王子－亞瑟只不過是個將取代他成為國王的膽小鬼。白馬王子相信了史瑞克的話，並且讓當時生氣且失望的亞瑟逃離當地。白馬王子策劃了一場戲劇演出，為自己「登基」籌備，本劇的高潮是出現在史瑞克於舞台上，因其歌唱和演出贏得觀眾支持而讓白馬王子懊悔不已。正當白馬王子準備殺害史瑞克之時，費歐娜和她的朋友們跳到舞台上並且正面迎戰那些惡棍。白馬王子威脅要殺死亞瑟，以達到自己能獲得「永遠幸福快樂」的要求。但史瑞克保護了亞瑟，而白馬王子則遭受到致命的一擊。史瑞克把皇冠讓給了亞瑟，他也在眾人的同意下接受了皇冠。到了最後，退休的史瑞克與費歐娜一起回到他們在沼澤的家，並帶著三個孩子真正的享受著成為父親的新生活，故事結束。

== DVD、HD DVD与Blu-ray的发行 ==
本片的[[DVD|DVD]]與[[HD_DVD|HD DVD]]都會在2007年11月13日同步發行，DVD將會以全螢幕以及1.85:1縱橫比寬螢幕方式上市。<ref>{{cite web |url=http://www.dvdtown.com/search/search-result-releases.php?movie-id=21386 |title=存档副本 |accessdate=2007-10-30 |deadurl=yes |archiveurl=https://web.archive.org/web/20071029032814/http://www.dvdtown.com/search/search-result-releases.php?movie-id=21386 |archivedate=2007-10-29 }}</ref>兩種版本都有特別收藏<ref>{{cite web |url=http://www.dvdtown.com/moviedatabase/release-details/shrek-3-the-third/22160 |title=存档副本 |accessdate=2007-10-30 |deadurl=yes |archiveurl=https://web.archive.org/web/20071016203158/http://dvdtown.com/moviedatabase/release-details/shrek-3-the-third/22160 |archivedate=2007-10-16 }}</ref>正片、刪除片段、預告片、記事、MV，HD DVD則另有HDi功能，內容包括特別軼事、電影指南，以及對話的彩色故事書。<ref>http://www.highdefdigest.com/news/show/Exclusive_HD_Content/DreamWorks/New_Shrek_Specs_Promise_Several_HD_DVD_Firsts/1016</ref> <ref>http://www.engadgethd.com/2007/09/28/shrek-the-thirds-hd-dvd-special-features-revealed/</ref>

HD DVD特別收藏將會以1.85:1寬螢幕比例以及1080p像素呈現，提供Dolby Digital Plus5.1的音效選項，但隨後HD DVD的地位被[[藍光光碟|藍光光碟]]所取代。

== 週邊商品 ==
許多系列的玩具、遊戲、故事書、服裝和其他產品都有在商店販售，例如費歐娜公主娃娃初販售時，就有包括變身成怪物的臉部，以及「腿部功夫」片段，其他如睡美人、灰姑娘、白雪公主、長髮姑娘的娃娃也會上市。

《[[史瑞克三世_(電玩遊戲)|史瑞克三世]]》遊戲版本同樣是改編自影片而成，屆時將會在[[Wii|Wii]]、[[PS2|PS2]]、[[Xbox_360|Xbox 360]]、[[PSP|PSP]]、[[PC|PC]]、[[任天堂DS|任天堂DS]]、[[Game_Boy_Advance|Game Boy Advance]]平台發行。
