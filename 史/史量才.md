'''史量才'''（{{bd|1880年|1月2日|1934年|11月13日|catIdx=Shi史}}），原名'''家修'''，祖籍[[江蘇|江蘇]][[江宁县|江宁]]。[[中华民国|中华民国]][[中华民国大陆时期|大陆时期]]《[[申报|申报]]》掌门，中國著名報業家，辦報之餘，曾編著《[[稱謂雜記|稱謂雜記]]》。於1934年返沪途中遭到[[暗杀|暗杀]]。

== 生平 ==
出生於[[上海|上海]][[青浦縣|青浦縣]]的商人家庭，1898年参加[[松江府|松江府]][[府试|府试]]，1899年入松江府[[婁縣|婁縣]][[縣學|縣學]]，因被查出异地投考，以[[冒籍|冒籍]]降为[[附生|附生]]，而后抛弃[[科舉|科舉]]，开始讲求[[实学|实学]]。1901年秋考入[[杭州蠶學館|杭州蠶學館]]（今[[浙江理工大学|浙江理工大学]]），毕业后赴上海。

1904年創辦女子蠶桑學校，同时在[[育才学堂|育才学堂]]、[[南洋中学|南洋中学]]等任教。1905年發起成立江蘇教育總會。后来[[狄楚青|狄楚青]]在上海创办《[[時報_(上海)|时报]]》，应邀担任编辑，1908年后任主笔。1912年9月与[[张謇|张謇]]等人出资十二萬元购得《申报》[[产权|产权]]，10月20日正式移交，自任总经理。三年后合伙人悉数退出，《申报》归其一人所有。1919年10月，[[世界报业大会|世界报业大会]]在[[檀香山|檀香山]]举行，被推选为大会副会长。1929年，陆续购进《[[新聞報_(上海)|新闻报]]》、《[[时事新报|时事新报]]》大部分股权，一跃成为上海报业巨擘。

[[九一八事變|九一八事變]]爆發後，反對[[國民黨|國民黨]]與[[共產黨|共產黨]]的[[第一次国共内战|内战]]，主张抗日。[[一·二八事变|一·二八事变]]時發起成立[[上海市民地方維持會|上海市民地方維持會]]，被推选为会长，积极筹集款项，支持[[十九路军|十九路军]]抗战。维持会解散后，被推选为[[上海地方协会|上海地方协会]]会长、[[上海市参议会|上海市参议会]]议长，对上海地方事务多有主持。同时借《申报》出版60周年之际，邀请民主人士[[黄炎培|黄炎培]]、[[陶行知|陶行知]]、[[戈公振|戈公振]]等进入报馆，对《申报》进行改革。

1932年創辦「申報流通圖書館」，出版《申報月刊》，一面編著《稱謂雜記》，紀錄民間逸事與個人讀書的心得。1934年10月前往[[杭州|杭州]]調養胃病，1934年11月13日，与二夫人[[沈秋水|沈秋水]]自[[秋水山莊|秋水山莊]]回家途中在滬杭公路翁家埠（今浙江省[[海宁市|海宁市]][[许村镇|许村镇]]翁埠村）遭暗殺身亡，史量才和他儿子的同学、司机3人遇害，其子逃脱<ref>{{cite web |url = http://www.gmw.cn/content/2004-11/13/content_130515.htm |accessdate =2009-06-12 |title = 报业巨子史量才被暗杀|author =  |date = 2004年11月13日 |publisher = [[光明日报|光明日报]] |language = zh-hans}}</ref>。14日，《申報》刊出《本報總理史量才先生噩耗》，全國輿論嘩然，全國各地舉行追悼會。1935年3月15日，追查案件的[[魯滌平|魯滌平]]身亡，案件成为谜团。1936年，葬於杭州[[天馬山|天馬山]]、[[吉慶山|吉慶山]]麓。[[章太炎|章太炎]]撰《史君墓志铭》：“史氏之直，肇自[[子鱼|子鱼]]。子承其流，奋笔不纡。”“唯夫白刃交胸，而神气自如。”

==死因==
* [[中共建政|中共建政]]后，国民党起义人员、原[[军统|军统]]总务处处长、保密局云南站站长[[沈醉|沈醉]]在其回忆录中称此事件是军统特務所为<ref>沈醉《我這三十年》：“如暗殺愛國民主人士[[楊杏佛|楊杏佛]]、史量才等等，都是我間接或親自進行的。”</ref>：“参加这一次罪恶行动的特务除[[赵理君|赵理君]]外，还有行动组副组长王克全、组员李阿大、施芸之、许建业等六人。”

* 根据新闻、申两报内部传说，辛亥革命时，史有一位好友担任沪军都督府的军需官，因隐瞒华侨捐款，被都督[[陈英士|陈英士]]枪杀，其次妻（即上海名妓[[沈秋水|沈秋水]]）携带赃款避居史家，史即人财两得。后来那位朋友的儿子声称要报此仇，史非常担心自己的安全，经常练习拳术，并雇用保镖，备有钢甲避弹汽车。……据说凶手就是杭州[[笕桥|笕桥]]飞机场的航空人员，为首一人就是声称要报家仇的陶某。<ref>政协文史研究委员会编，《文史资料选辑》第4辑（中国文史出版社合订本，1989年），页139注3。</ref>

* 据原《申报》编辑龚德柏回忆，由于以前史量才打输过一场官司，被判罚36万[[银元|银元]]，即[[白银|白银]]25万两，史量才便找日商洋行借了25万两。后来，龚德柏将史量才亲口所说“借了[[日本人|日本人]]的钱，不能骂日本人”的真相整理成一份黑材料，交给了申报的竞争对手，并由他们再上报国民党中央宣传部。多年以后，这位龚编辑还在回忆录中愤愤不平地写道:“我这种行动，对于史量才虽不利，但对于国家是绝对应该的。因为他们以中国第一大报，与日本——而且是绝对敌人日本，订有卖身契约，这是何等重大的卖国行为”。<ref>{{龚德柏回忆录，转引自[http://news.ifeng.com/history/minjianshuoshi/wangyong/detail_2009_03/25/331665_0.shtml 王勇，史量才遇刺的另一种说法]}}
</ref>

==轶事==
* 史量才生前友人[[冯亚雄|冯亚雄]]回忆，有一次，史量才在南京受国民党要人盛宴款待，席间有人故意宣传当道的威力，说有雄兵千万，足以安内攘外。史不以为然，嘲弄说：“我只能在报言报，约略估计，有数千万读报者拥护。”合座为之默然。<ref>冯亚雄，《〈申报〉与史量才》，收于《文史资料选辑》合订本第五册第17辑（中国文史出版社，1989年），页161-162</ref>

* 据[[黄炎培|黄炎培]]自述《八十年来》中记载：“……实际上如史量才这一群朋友，对爱国抗日思想热诚的高度怎样呢？说一件故事：1931年9月18日日军占领沈阳之夜，我从申报馆到史家。史正和一群朋友打牌。我说：电报到了，日本兵在沈阳开火了，沈阳完全被占了，牌不好打了。一人说：中国又不是黄任之独有的，要你一个人起劲！我大怒，一拳打牌桌中心，哭叫：你们甘心做亡国奴吗！别人说：收场罢！……”<ref>黄炎培，《八十年来》（文汇出版社，2000年再版），页136。</ref>

* 据《[[胡适之|胡适之]]批评史量才》一文说：“若论《申报》的政治主张，九一八以前向来是再稳重和平不过的，唯近年以来也不大安于现实；所以有人骂他，只顾推扩销路，迎合一般年轻人好乱的心理；尤其是以‘自由谈’供左翼作家作用武之地，最为一部分人所不满。”<ref>{{cite web|title=傅国涌：报业巨子史量才为何遭暗杀？|url=http://www.21ccom.net/articles/lsjd/article_201001204547.html|date=2010-01-20|publisher=[[共识网|共识网]]|archiveurl=https://web.archive.org/web/20140222212551/http://www.21ccom.net/articles/lsjd/article_201001204547.html|archivedate=2014-02-22|deadurl=yes}}</ref>
