'''茉莉香'''（{{lang-ja|まりか}}、{{bd|1984年|10月29日}}），是[[日本|日本]][[東京都|東京都]]出身的[[寫真女星|寫真女星]]、[[AV女優|AV女優]]及[[模特兒|模特兒]]。

== 人物 ==
*在出道之初參加活動、輪到出場前，為免首次見面讓人易誤記為「真理子」或「真理」，才決定暱稱{{nihongo|'''長谷真理'''|はせまり}}。<ref>{{cite web|url=http://blog.livedoor.jp/mfuruta1/archives/50386780.html |script-title=ja:はせまりぃ |date=2006-07-31 |publisher={{lang|ja|長谷 真理香 公式ブログ}} |accessdate=2011-08-21}}</ref>
*在舊[[網誌|網誌]]以{{nihongo|'''長谷川真理子'''|長谷川 真理子}}名義公告活動近況。<ref>{{cite web|url=http://blog.livedoor.jp/hasemari111/archives/50953432.html|script-title=ja:案内人 長谷川真理子さん。 |date=2007-09-07 |publisher={{lang|ja|はせまり修行中！}} |accessdate=2011-08-21}}</ref>
*擅於寫文章，不論是在{{nihongo|「長谷真理学園」|はせまり学園}}官網連載小說，或在[[EROTICA|EROTICA]]兼任宣傳，都能展現其長才。

== 簡歷 == 
*2006年，當時[[藝名|藝名]]{{nihongo|'''長谷真理香'''|長谷 真理香}}從事寫真女星活動，對外常自稱{{nihongo|'''真理香'''|MARIKA、まりか}}。
*2007年4月，以'''長谷真理香'''為首，成立團體「長谷真理学園」，擔任{{nihongo|「班長」|学級委員長}}一職。
*2007年6月，[[Maxing|Maxing]]任命{{nihongo|「現役偶像宣傳」|現役アイドル広報}}。<ref>{{cite web |url=http://www.maxing.jp/blog/maxing_public/detail/1211.html |script-title=ja:マキシング新広報！ |date=2007-06-01 |publisher={{lang|ja|マキシング広報部ブログ}} |accessdate=2011-08-21 |deadurl=yes |archiveurl=https://web.archive.org/web/20100329125359/http://www.maxing.jp/blog/maxing_public/detail/1211.html |archivedate=2010-03-29 }}</ref>
*2009年4月，直接改用{{nihongo|'''真理香'''|まりか}}為藝名發表作品。
*2009年8月，第1回{{nihongo|「SODstar Cinderella Audition」|SODstarシンデレラオーディション}}投票獲得第2名；12月，在[[Soft_On_Demand|Soft On Demand]]的[[AV|AV]]作品出道。
*2010年6月，移籍至EROTICA，並兼任{{nihongo|「特別宣傳」|特別広報}}。
*2010年11月以降，成為單體女優。
*2011年7月，在[[中國大陸|中國大陸]]時尚雜誌《[[ELLE|ELLE MAN 睿士]]》8月號出道擔當[[模特兒|模特兒]]，中文譯名{{nihongo|'''茉莉香'''|まりか}}。<ref>{{cite web|url=http://blog.livedoor.jp/cinderella_02/archives/3872894.html |script-title=ja:ショッピング茉莉香☆ |date=2011-07-25 |publisher={{lang|ja|まりかのぶっちぎり清純派}} |accessdate=2011-08-21}}</ref>
*2012年3月，宣佈赴美國進行AV工作。<ref>{{cite web|url=http://blog.livedoor.jp/cinderella_02/archives/4156094.html |script-title=ja:皆さんへ大切なお知らせ☆ |date=2012-03-31 |publisher={{lang|ja|まりかのぶっちぎり清純派}} |accessdate=2012-05-27}}</ref>英文藝名''Marica Hase''，大量與黑人拍無碼片，內容多為肛交或多P等較激烈的重口味影片。
