'''鄱阳湖之战'''（1363年8月30日─1363年10月4日）是中國[[元朝|元朝]]末年的一場重要戰役，[[朱元璋|朱元璋]]在此役擊敗並殺死強敵[[陳友諒|陳友諒]]，也被視為[[中世紀|中世紀]]世界規模最大水戰。此战的胜利，奠定了朱元璋平定江南的基础，并为以后的[[明太祖北伐|北伐]]和攻灭元朝，统一全国创造了极为有利的条件<ref name="鄱阳湖水战">{{cite web|title=鄱阳湖之战|url=http://military.china.com/zh_cn/history2/06/11027560/20050407/12224277_1.html|publisher=中华网军事|accessdate=2014-01-05|language=中文}}</ref> 。

== 背景 ==
{{main|洪都之戰}}
元朝末年，朝政废弛，社会动乱，农民起义如火如荼。1351年[[刘福通|刘福通]]领导的[[红巾军|红巾军]]高举义旗，各地群起响应<ref>{{cite web|title=鄱阳湖之战|url=http://military.china.com/zh_cn/history2/06/11027560/20050407/12224277.html|publisher=中华网军事|accessdate=2014-01-05|language=中文}}</ref> 。江南地区各[[元末民變|反元武装]]不断取得胜利。在长江中、下游地区形成以湖北武昌为统治中心、自立为汉帝的[[陈友谅|陈友谅]]，以应天（今南京）为统治中心的吴国公[[朱元璋|朱元璋]]和以平江（今江苏苏州）为统治中心、自称吴王的[[张士诚|张士诚]]三大势力。由于辖区相邻，彼此展开武力兼并<ref name="鄱阳湖之战"/>。

至正十七年（1357年）九月，陳友諒襲殺[[倪文俊|倪文俊]]，自稱宣慰使，以[[鄒普勝|鄒普勝]]為[[太師|太師]]，[[張必先|張必先]]為[[丞相|丞相]]，起兵攻下江西諸路。迎[[徐寿辉|徐寿辉]][[迁都|迁都]][[江州|江州]]（治今江西九江）。陈友谅挟徐寿辉，自称汉王，建都江州，国号汉，年号大义。

至正二十年（1360年），陳友諒[[东征|东征]]朱元璋，并在途中[[刺殺|刺殺]][[徐寿辉|徐寿辉]]，自立为大汉皇帝。陳衝破朱元璋长江太平防线，兵临[[集庆|集庆]]（今[[南京|南京]]）城下。陳朱雙方在集庆城西北的龍灣展開惡戰，不巧江水[[退潮|退潮]]，百艘巨艦擱淺，陳友諒大敗，逃走江州（九江）。

至正二十三年（1363年）四月，陳友諒再出兵圍攻江西洪都（今[[南昌|南昌]]），水陸兩軍號稱60萬人，陳友諒還動用水陸兩棲作戰，水船使用高大但是遲緩的[[樓船|樓船]]，卻圍攻洪都不下（守將是朱元璋之姪[[朱文正_(明朝宗室)|朱文正]]），朱元璋在七月親率20萬人救洪都，<!--出身原[[元軍|元軍]]水師叛軍的-->陳友諒於是撤圍，在中國最大[[淡水湖|淡水湖]][[鄱陽湖|鄱陽湖]]選擇所擅長水戰迎戰朱元璋。

== 過程 ==
[[File:Hayohko.png|thumb]]
才九天朱元璋軍艦隊大軍自[[南京|南京]]出發便抵達[[南昌|南昌]]救援，朱元璋軍艦隊分成11隊，大船隊在中央，餘小船隊載[[步兵|步兵]]至南昌登陸支援南昌守軍，七月二十日（8月29日）兩軍主力在鄱陽湖對峙，次日（8月30日）黎明開始大戰，前三天朱元璋軍船小兵少處劣勢，漢軍「樓船」居高放炮石攻低處明軍船艦，朱元璋座艦也中彈攻毀幾乎被漢軍俘虜，幸賴[[錦衣衛|錦衣衛]]數船奔赴救出朱元璋。

後來朱元璋採納郭興的建議，用火攻、[[投石機|投石機]]，第四天（9月2日）起燒毀陳軍二十幾艘樓船艦船，漢軍傷亡慘重，至少兩萬五千名漢軍戰歿，陳友諒弟陳友仁、陳友貴及其平章陳普略等皆陣亡。陳友諒更悲痛不已，陳軍憤而於陣前殺朱元璋軍俘虜，導致朱元璋軍自覺退無可退、非背水一戰不可；之後陳軍形勢開始不利：鄱陽湖因為秋陽久曬導致水位降低，漢軍樓船大重卻吃水深慢甚至擱淺、不敢靠岸，反觀朱元璋軍船小速快可利攻擊，也可以靠岸紮水寨作持久戰；八月下旬，朱元璋軍陸軍克復[[南昌|南昌]]，部份朱元璋軍艦隊撤出至[[贛江|贛江]]、[[長江|長江]]，僅留部份艦隊與陳續戰，等待步軍來援完成水陸包圍。到了八月二十六日（10月4日），陳軍企圖突圍退回[[武昌|武昌]]，在[[湖口縣|湖口]]受到朱元璋軍岸上步兵萬箭遮天連發阻截，陳友諒從船艙中探頭出來，竟中流矢死，陳軍於此再喪兩萬士卒，餘五十萬殘部潰散逃敗<ref name="鄱阳湖之战"/>。

== 后续 ==
张定边与陈友谅次子[[陈理|陈理]]逃回武昌。次年二月，朱元璋率师围攻武昌，迫陈理投降，尽占陈友谅所辖长江中游广大地区<ref name="鄱阳湖之战">{{cite web|title=鄱阳湖之战|url=http://gfjy.jxnews.com.cn/system/2010/04/20/011363291.shtml|publisher=江西国防教育网|accessdate=2014-01-05|location=江西|language=中文}}</ref> 。

== 评价 ==
[[朱元璋|朱元璋]]乘[[陈友谅|陈友谅]]大军久攻坚城受挫，分兵据守[[鄱阳湖|鄱阳湖]]口，先断其退路；继集中兵力，巧用火攻，歼其主力；后水陆截击，全歼陈友谅大军于突围之际。鄱陽湖之戰是中国水战史上以少胜多的著名战例，其为朱元璋统一[[江南|江南]]奠定了坚实基础<ref name="鄱阳湖之战"/><ref name="鄱阳湖水战"/>。
