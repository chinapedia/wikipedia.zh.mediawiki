'''{{ruby-py|鄱|-{zh-cn:pó;zh-tw:ㄆㄛˊ}-}}阳湖'''僅次於[[興凱湖|興凱湖]]為中国第二大[[淡水湖|淡水湖]]（因[[洞庭湖|洞庭湖]]已縮小成中國第三大淡水湖）、第三大[[湖|湖]]（僅次於[[青海湖|青海湖]]）和[[中国五大淡水湖|中国五大淡水湖]]之一。位于[[江西省|江西省]]北部、[[长江|长江]]南岸，鄱阳湖上承赣、抚、信、饶、修五河之水，下接中国第一大河——长江。鄱阳湖介于北纬28度22分～29度45分，东经115度47分～116度45分。跨[[南昌市|南昌]]、[[南昌县|南昌县]]、[[进贤县|进贤]]、[[余干县|餘乾]]、[[鄱阳县|鄱阳]]、[[都昌县|都昌]]、[[湖口县|湖口]]、[[九江市|九江]]、[[庐山市|庐山]]、[[德安县|德安]]和[[永修县|永修]]等市县。

== 水文特征 ==
鄱阳湖南北长173公里，东西最宽处74公里，平均宽16.9公里，湖岸线长1200公里，湖口水位21.71米时湖面面积3283平方公里，是中国最大的淡水湖。

湖面以松门山为界，分南北两部分，南部主湖区宽广，长133公里，最宽74公里，湖水较浅；北部狭长，为入江水道，长40公里，最窄处约2.8公里，湖水较深。

湖水补给包括[[赣江|赣江]]、[[抚河|抚河]]、[[信江|信江]]、[[饶河|饶河]]、修河和[[博阳河|博阳河]]、[[西河|西河]]等河道，经调蓄后，向北由[[湖口縣|湖口]]注入长江。入长江水量年均1460亿立方米，超过[[黄河|黄河]]、[[淮河|淮河]]、[[海河|海河]]水量的总和。鄱阳湖水系流域面积16.22万km<sup>2</sup>，约占江西省流域面积的97%，占长江流域面积的9%：年均径流量为1525亿立方米，约占长江流域年均径流量的16.3%。

鄱阳湖是一个季节性变化巨大的吞吐型湖泊，洪水期与枯水期面积、蓄水量差异悬殊。年内水位变幅在9.79～15.36米，绝对水位变幅达16.69米。每年春夏之交，湖水猛涨，水面迅速扩大，烟波浩渺；但到了冬季，湖水剧降，洲滩裸露，湖面仅剩几条蜿蜒的水道。形成“洪水一片、枯水一线”的景观。

多年平均水位12.86米；最高水位22.59米（1998年7月31日），面积达4070平方公里，蓄水量300亿立方米；最低水位5.90米（1963年2月6日），面积仅146平方公里，蓄水量4.5亿立方米。

== 地质地貌 ==
鄱阳湖区[[第三纪|第三纪]]时是一巨大盆地。[[喜马拉雅运动|喜马拉雅运动]]时，西侧断裂上升为[[庐山|庐山]]，东侧陷落为鄱阳湖入江水道。[[第四纪|第四纪]]时鄱阳湖区再度下沉。

“高水是湖，低水似河”是鄱阳湖的特征。湖底水道高程自南向北渐降，由海拔12米降至湖口约1米，鞋山附近为-1米，褚溪口低达-2米。

湖区由水道、洲滩、岛屿、内湖和汊港组成：
*赣江自南昌以下分4支，主支在吴城与修水汇合，进入湖北部，为湖区西水道；南、北、中3支与抚江、信江均先后汇入湖南部，为湖区东水道。东、西水道在褚溪汇合为入江水道。
*洲滩分沙滩、泥滩、草滩3种。沙滩多在水位14米以下，面积约1895平方公里，草滩多在14～18米，面积约1235平方公里。
*全湖现有岛屿25处，共41座（中、低水位时多为滩丘），面积约100平方公里。
*内湖出现在枯水期，中、高水位时与大水面联成一片，主要分布在东、南、西部；
*汊港多分布于入江水道东岸，主湖区北岸和东北、东南湖隅，主要汊港共约20处。

== 历史变迁 ==
鄱阳湖古称彭蠡泽、彭泽和官亭湖。

古代彭蠡泽范围大致包括今长江北岸的[[湖北|湖北]]东部的源湖、[[安徽|安徽]]西部的[[龙感湖|龙感湖]]、[[大官湖|大官湖]]和[[泊湖|泊湖]]等滨江湖泊区，与长江水面相连接。当时长江南岸由古赣江汇注而成的水域也与长江汊道在湖口相通，但湖面面积很小，直至[[秦漢|秦漢]]之时仍仅限于罂子口（今星子县境，[[東吳|東吳]][[晋朝|晉]]时古赣江流注彭蠡泽的交汇口）以北至湖口的狭长地带，因罂子口旁有宫亭庙，故称为宫亭湖。[[西汉|西汉]]后期，彭蠡泽与长江完全分离，江北水域逐渐演变成今江北诸湖，而江南水域则逐渐扩大，演化成今日的鄱阳湖<ref name="gailun">王育民：中国历史地理概论（上册），北京：人民教育出版社，1987年，ISBN 7-107-10151-X</ref>。由于[[班固|班固]]在《[[汉书|汉书]]·地理志》有“豫章郡……县十八：……彭泽，《禹贡》彭蠡泽在西<ref name="haihu">[[:wikisource:zh:汉书/卷028上|班固（东汉）：汉书 卷二十八上 地理志第八上]]，维基资源</ref>”之语，此后“彭蠡泽”遂指今鄱阳湖。

对于“彭蠡泽”名称的变迁，学者看法不同，有的根据《[[尚书_(经)|尚书]]·[[禹贡|禹贡]]》所载“导漾……东汇泽为彭蠡”<ref name="yugong">[[:wikisource:zh:尚书/禹贡|尚书·禹贡]]，维基资源</ref>，认为古彭蠡泽仅指江北水域，班固之说为附会之言；有的认为古彭蠡泽地跨长江两岸，包括江南水域，今鄱阳湖是古彭蠡泽解体演化而成<ref name="hupozhi">王苏民、窦鸿身主编，《中国湖泊志》，北京：科学出版社，1998年，ISBN 7-03-006706-1，218-224页</ref><ref name="dituji">谭其骧主编：《中国历史地图集》光盘版，ISBN 7-980013-01-8,中国地图出版社，1982年</ref>。

在[[汉代|汉代]]，现鄱阳湖水域乃是湖汉水（今赣江）下游平原，汉代在此设置了阳县，汉昌邑王封地亦在此。那时，长江主泓道在今龙感湖一带，在庐山以北的仅是长江一汊道，当时称为小江。约在400年，即東晉後期，长江主泓道移至今九江-湖口一带。江水阻挡了赣江的排水，还倒灌入赣江下游平原，逐步形成鄱阳湖。到了[[盛唐|盛唐]]時，鄱阳湖面积最大达6000平方公里。

在鄱阳湖大举南侵之前，低洼的鄱阳盆地上原本是人烟稠密的城镇，随着湖水的不断南侵，鄱阳湖盆地内的[[鄱阳县|鄱阳县]]城和[[海昏县|海昏县]]治先后被淹入湖中，而位于[[海昏县|海昏县]]邻近较高处的[[吴城|吴城]]却日趋繁荣成[[江西|江西]]大古镇之一，因此，历史上曾有“淹了海昏县，出了吴城镇”之说。

== 事件 ==

鄱阳湖流域自古以来是中国经济较为发达的富裕地区，中国历史上很多杰出人物如[[東漢|東漢]]後期的[[徐稚|徐稚]]、[[東晉|東晉]]的[[陶渊明|陶渊明]]、[[隋代|隋代]]的[[林士弘|林士弘]]、[[北宋|北宋]]的[[刘恕|刘恕]]、洪适、[[南宋|南宋]]中後期的[[江万里|江万里]]、[[晚明|晚明]]的[[朱耷|朱耷]]等都在湖区出生成长。这里又发生许多威武雄壮的英雄事迹，如周瑜操练水师、朱元璋与陈友谅之间的鄱阳湖之战、王守仁生擒朱宸濠、太平军“湖口大捷”、李烈钧在湖口发起“二次革命”等。鄱阳湖是古代从北方进入江西的唯一水道。唐代诗人王勃在《滕王阁序》中的名句“渔舟唱晚，响窮彭蠡之滨”，描述的正是鄱阳湖上的渔民捕鱼归来的欢乐情景。

“鄱湖接近长江处，二水相交奇景生；澈液浑流互排斥，浊清界线见分明”。长江、鄱阳湖的汇合处，界线分明，一边混浊，一边清澈，互不相融，可见一条明显的清浊分界线，乃一大奇观，令人惊叹不已。

== 生态保护 ==
"鄱阳湖畔鸟天堂，鹬鹳低飞鹤鹭翔；野鸭寻鱼鸥击水，丛丛芦苇雁鹄藏"。鄱阳湖是[[白鹤|白鹤]]等珍稀水鸟的重要栖息地。每年秋末冬初，从俄罗斯西伯利亚及中国东北等地，飞来成千上万只候鸟到鄱阳湖越冬，如今，鄱阳湖保护区内鸟类已达300多种，近百万只，其中珍禽50多种，已是世界上最大的鸟类保护区。世界上现有白鹤大约为4000多只。其中90%在鄱阳湖越冬。鄱阳湖是白鹤的世界，水鸟的天堂。

目前在鄱阳湖湖区设立的各类保护区有<ref name="tj2008">《江西统计年鉴2008电子版》，ISBN 978-7-900225-28-3，中国统计出版社</ref>：
* 国家级：江西省鄱阳湖自然保护区、鄱阳湖生态功能保护区
* 省级：江西省鄱阳湖南矶湿地自然保护区、江西省鄱阳湖河蚌保护区、江西省鄱阳湖湖鲤鲫鱼产卵场自然保护区、江西省鄱阳湖银鱼自然保护区、江西省鄱阳湖长江江豚自然保护区。
