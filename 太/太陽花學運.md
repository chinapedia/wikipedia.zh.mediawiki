'''太陽花學運'''又被稱作'''318學運'''、'''佔領國會事件'''，是指在2014年3月18日至4月10日期間，由[[中華民國|中華民國]]大學學生與公民團體共同發起的[[社會運動|社會運動]]。這次運動由抗議學生主導，佔領位在[[臺北市|臺北市]]的[[立法院|立法院]]，還曾一度嘗試佔領[[行政院|行政院]]。抗議運動的主要原因在於《[[海峽兩岸服務貿易協議|海峽兩岸服務貿易協議]]》遭強行通過審查，而該協議被反對者視為將損害[[臺灣經濟|臺灣經濟]]，並且強化[[中華人民共和國|中華人民共和國]]的政治影響力。其他參與運動理由還包括要求民主程序、反對自由貿易等。

在3月17日的立法院聯席會議上，[[中國國民黨|中國國民黨]]立法委員[[張慶忠|張慶忠]]宣布《海峽兩岸服務貿易協議》的審查超過90天，依法視為已經審查，強行送交立法院院會存查。此舉引發部分公眾的不滿。3月18日晚上9時，反對《海峽兩岸服務貿易協議》的學生、群眾、公民團體，突然[[318青年佔領立法院行動|佔領]]立法院議場，並用座椅封鎖門口。隨後在立法院議場內的成員建立行動決策核心，展開大量的組織分工合作，在立法院外則有大量支持者聲援。

3月23日上午，[[中華民國總統|中華民國總統]][[馬英九|馬英九]]針對學生運動發表講話。同日晚間，抗議群眾試圖轉而[[323佔領行政院事件|佔領]]行政院，隔日便遭警方強制驅離。在與[[中華民國總統府|總統府]]的會面破局後，學生在3月30日於[[凱達格蘭大道|凱達格蘭大道]]發起[[330反服貿遊行|反服貿遊行]]，並有數萬名民眾參與。但與此同時，支持《海峽兩岸服務貿易協議》的[[張安樂|張安樂]]等人士，則是發起[[返還國會運動|返還國會運動]]。到了4月6日，[[立法院院長|立法院院長]][[王金平|王金平]]提出制定《[[兩岸協議監督條例|兩岸協議監督條例]]》的承諾。最終在4月10日，抗議學生與民眾退出議場。

太陽花學運是臺灣從1980年代以來最大規模的「[[公民不服從|公民不服從]]」行動，也是立法院首次遭到民眾佔領。在運動結束後，該運動仍然持續影響著臺灣社會，包括促進臺灣公民社會的復興、中國國民黨在[[2014年中華民國地方公職人員選舉|2014年選舉]]和[[2016年中華民國總統副總統及立法委員選舉|2016年選舉]]的落敗、以及「第三勢力」政黨的出現等。這次運動還影響中華人民共和國的相關政策，並啟發[[雨傘革命|香港]]、[[自由民主主義學生緊急行動|日本]]等地的抗爭。2018年3月，[[台灣高等法院|台灣高等法院]]宣判行動的主要人物[[林飛帆|林飛帆]]、[[陳為廷|陳為廷]]、[[黃國昌|黃國昌]]等22人無罪。

早在太陽花學運之前，運動者的「亞文化」發展有助於運動者家發展其公共領域<ref>{{Cite web |url = https://daybreak.newbloommag.net/2017/08/10/activist-subculture/ |title = Activist Subculture |author =  |date = 2017年8月10日 |publisher = Daybreak Project |language = en |accessdate = 2019年3月1日 |quote =  }}</ref>。隨著臺灣青年活動的興起，臺灣出現許多服裝品牌，這些品牌商品在運動者間很受歡迎，並表達有關臺灣民間社會認可的政治訊息<ref>{{Cite web |url = https://daybreak.newbloommag.net/2017/07/20/activist-merchandise/ |title = Activist Merchandise |author =  |date = 2017年7月20日 |publisher = Daybreak Project |language = en |accessdate = 2019年3月1日 |quote =  }}</ref>。部分書籍和電影在運動者亞文化中很受歡迎<ref>{{Cite web |url = https://daybreak.newbloommag.net/2017/07/20/activist-books-films/ |title = Books And Film Popular Among Activists |author =  |date = 2017年7月20日 |publisher = Daybreak Project |language = en |accessdate = 2019年3月1日 |quote =  }}</ref>，部分咖啡館、住宅和書店也是青年運動者的聚會場所<ref>{{Cite web |url = https://daybreak.newbloommag.net/2017/07/20/coffee-bookstores-live-houses/ |title = Activist Coffee Shops, Live Houses, and Bookstores |author =  |date = 2017年7月20日 |publisher = Daybreak Project |language = en |accessdate = 2019年3月1日 |quote =  }}</ref>，某些臺北市頻繁發生抗議空間亦是成員聚會的地點<ref>{{Cite web |url = https://daybreak.newbloommag.net/2017/07/25/frequent-protest-spaces-in-taipei/ |title = Frequent Protest Spaces In Taipei |author =  |date = 2017年7月25日 |publisher = Daybreak Project |language = en |accessdate = 2019年3月1日 |quote =  }}</ref>。而自從學運以後，「黑箱」已經成為臺灣社會常用的詞彙<ref>{{Cite web |url = https://daybreak.newbloommag.net/2017/07/20/what-was-the-black-box/ |title = What Was The “Black Box” And Why Were Occupiers Opposed To It? |author =  |date = 2017年7月20日 |publisher = Daybreak Project |language = en |accessdate = 2019年3月1日 |quote =  }}</ref>{{sfnm|黃恐龍|2014年|1loc=第34頁|港千尋|2015年|2loc=第95頁至第98頁、第106頁至第115頁}}。同時，鹿茸、香蕉、太陽餅也成為群眾嘲諷的對象，更有網友創意團購太陽餅、「伏冒」（感冒藥）送去行政院<ref>{{Cite web |url = https://tw.news.appledaily.com/politics/realtime/20140325/366667 |title = 太陽餅過時囉！　現在買伏冒送政院正夯 |author = 胡治言 |date = 2014年3月25日 |publisher = 《[[蘋果日報_(台灣)|蘋果日報]]》 |language = Zh-hant |accessdate = 2019年3月1日 |quote =  }}</ref>{{sfnm|公民聲音團隊|2014年|1loc=第9頁|黃恐龍|2014年|2loc=第190頁至第193頁|港千尋|2015年|3loc=第240頁、第242頁至第243頁}}。

而一些[[獨立音樂|獨立樂團]]和藝術家創作有關社會問題的歌曲，同樣在臺灣青年運動者中贏得許多觀眾，包括[[拷秋勤|拷秋勤]]樂團創作的《官逼民反》等{{Sfnm|公民聲音團隊|2014年|1loc=第5頁}}<ref>{{Cite web |url = https://daybreak.newbloommag.net/2017/07/20/activist-artists-and-bands/ |title = Activist Artists and Bands |author =  |date = 2017年7月20日 |publisher = Daybreak Project |language = en |accessdate = 2019年3月1日 |quote =  }}</ref>。而太陽花學運的運動代表曲，則是由[[滅火器樂團|滅火器樂團]]主唱[[楊大正|楊大正]]為運動創作的歌曲《[[島嶼天光|島嶼天光]]》<ref>{{Cite web |url = http://focustaiwan.tw/news/aipl/201404100034.aspx |title = Festive rally closes Legislature occupation as protesters head home |author = Chen Chih-chung和Elaine Hou |date = 2014年4月10日 |publisher = [[中央通訊社|中央通訊社]] |language = en |accessdate = 2019年3月1日 |quote =  }}</ref><ref>{{Cite web |url = https://thediplomat.com/2014/04/sunflowers-end-occupation-of-taiwans-legislature/ |title = Sunflowers End Occupation of Taiwan’s Legislature |author = [[寇謐將|寇謐將]] |date = 2014年4月11日 |publisher = 《[[外交家雜誌|外交家雜誌]]》 |language = en |accessdate = 2019年3月1日 |quote =  }}</ref>{{Sfnm|公民聲音團隊|2014年|1loc=第10頁|港千尋|2015年|2loc=第243頁至第244頁|黃恐龍|2014年|3loc=第195頁至第198頁|晏山農等|2015年|4loc=第81頁至第82頁|港千尋|2015年|5loc=第83頁至第85頁、第87頁、第192頁至第193頁}}。3月27日，他在立法院現場教唱、錄音錄影，之後由臺灣的藝術大學學生製作成兩部音樂影片，並在網際網路上傳播{{Sfnm|公民聲音團隊|2014年|1loc=第10頁晏山農等|2015年|2loc=第81頁至第82頁|港千尋|2015年|3loc=第83頁、第243頁至第244頁}}。到了3月31日，滅火器樂團便在遊行中演唱《島嶼天光》，並播放在立法院拍攝製作的音樂影片{{Sfnm|公民聲音團隊|2014年|1loc=第11頁|黃恐龍|2014年|2loc=第195頁至第196頁|晏山農等|2015年|3loc=第91頁}}。另外在3月21日，[[施明德|施明德]]女兒[[施蜜娜|施蜜娜]]在議場內部學生同意下，在議場外牆上噴寫「[[當獨裁成為事實，革命就是義務|當獨裁成為事實，革命就是義務]]」{{Sfnm|公民聲音團隊|2014年|1loc=第7頁|晏山農等|2015年|2loc=第44頁}}。

在太陽花學運中，許多藝術家也反對《海峽兩岸服務貿易協議》，同時存在著許多表達訴求、詆毀攻擊、挪用戲仿、呈現共同記憶的藝術創作<ref>{{Cite web |url = https://daybreak.newbloommag.net/2017/07/25/art-sunflower-movement/ |title = Art and the Sunflower Movement |author =  |date = 2017年7月24日 |publisher = Daybreak Project |language = en |accessdate = 2019年3月1日 |quote =  }}</ref>，這包括創意海報、各式照片、版畫等作品{{sfnm|黃恐龍|2014年|1loc=第18頁至第22頁、第114頁至第176頁、第228頁至第232頁|晏山農等|2015年|2loc=第183頁至第184頁|港千尋|2015年|3loc=第13頁、第15頁、第92頁至第94頁、第208頁至第219頁|吳叡人等|2016年|4loc=第138頁至第139頁}}。4月1日，[[郝明義|郝明義]]在Facebook上發起「100人的太陽花」聲援行動，號召藝術家以創作發聲，並引起熱烈迴響{{Sfnm|公民聲音團隊|2014年|1loc=第12頁}}。到了4月6日，郝明義與藝文界人士召開記者會，公布6天來蒐集到的150件太陽花學運作品{{Sfnm|公民聲音團隊|2014年|1loc=第15頁}}。「太陽花學運」還被視為美學與政治的實踐案例，並在同年和「大腸花論壇」入圍[[台新藝術獎|台新藝術獎]]提名{{sfnm|晏山農等|2015年|1loc=第290頁至第291頁|吳叡人等|2016年|2loc=第373頁至第383頁}}。

== 運動評價 ==
[[File:Hong_Kong_Umbrella_Revolution_-umbrellarevolution_-UmbrellaMovement_(15292823874).jpg|thumb]]爆發[[雨傘革命|雨傘革命]]。]]

太陽花學運是一個結構複雜的運動，參與主體與訴求相當複雜多元<ref name = "吳介民 01"/>{{sfnm|晏山農等|2015年|1loc=第320頁至第324頁|吳叡人等|2016年|2loc=第126頁至第127頁}}。而作為一場抗議活動，太陽花學運是深深影響著臺灣社會的歷史事件之一<ref name = "Francisco Luis Pérez 01"/><ref>{{Cite web |url = https://daybreak.newbloommag.net/2017/07/15/sunflower-movement-significance/ |title = What Was The Significance Of The Sunflower Movement? |author =  |date = 2017年7月15日 |publisher = Daybreak Project |language = en |accessdate = 2019年3月1日 |quote =  }}</ref>，也是[[中華民國歷史|中華民國歷史]]上首次發生立法院議場遭到民眾佔領<ref name = "比嘉清太 03"/><ref>{{Cite web |url = https://archive.is/20140319044548/http://news.rti.org.tw/index_newsContent.aspx?nid=492136#selection-829.1-832.0 |title = 群眾占領議場 國會史上首次 |author = 陳沂庭 |date = 2014年3月19日 |publisher = [[中央廣播電臺|中央廣播電臺]] |language = Zh-hant |accessdate = 2019年3月1日 |quote =  }}</ref><ref>{{Cite web |url = https://www.bloomberg.com/news/articles/2014-03-19/taiwan-students-occupy-legislature-over-china-trade-pact-vote |title = Taiwan Students Occupy Legislature Over China Pact |author = Adela Lin和Tim Culpan |date = 2014年3月19日 |publisher = [[彭博社|彭博社]] |language = en |accessdate = 2019年3月1日 |quote =  }}</ref><ref>{{Cite web |url = https://www.scmp.com/news/china/article/1740013/one-year-impact-sunflower-movement-protests-taiwan-continue-blossom |title = One year on: impact of ‘sunflower’ movement protests in Taiwan continues to blossom |author = [[法新社|法新社]]|date = 2015年3月1日 |publisher = 《[[南華早報|南華早報]]》 |language = en |accessdate = 2019年3月1日 |quote =  }}</ref>{{sfnm|劉定綱|2014年|1loc=第16頁|黃恐龍|2014年|2loc=第5頁|晏山農等|2015年|3loc=第324頁至第330頁}}。而從動員的角度觀察，這是從1980年代啟動民主化以來，最大強度的「[[公民不服從|公民不服從]]」集體行動。<ref name = "吳介民 01"/>。雖然其影響難以量化，但許多人的報告表示在太陽花學運之後，臺灣的社會氛圍發生變化<ref>{{Cite web |url = https://daybreak.newbloommag.net/2017/11/13/social-atmosphere-changes/ |title = What Changed About The Social Atmosphere In Taiwan After The Sunflower Movement? |author =  |date = 2017年11月13日 |publisher = Daybreak Project |language = en |accessdate = 2019年3月1日 |quote =  }}</ref>。黃國昌和陳為廷都把這次運動視為臺灣政治的「第三勢力」，林飛帆亦強調這場運動完全獨立於中國國民黨或民主進步黨<ref name = "William Lowther 01"/>{{sfnm|吳叡人等|2016年|1loc=第78頁}}。

許多參與太陽花學運的年輕人，過去在日常生活中曾感到在政治上已經失去權力，但在運動之後突然擁有控制自己命運的感覺<ref>{{Cite web |url = https://daybreak.newbloommag.net/2017/07/24/everyday-rupture/ |title = Why Did The Sunflower Movement Rupture The Everyday For Many Participants? |author =  |date = 2017年7月24日 |publisher = Daybreak Project |language = en |accessdate = 2019年3月1日 |quote =  }}</ref>{{sfnm|吳叡人等|2016年|1loc=第107頁至第108頁、第157頁至第160頁、第369頁至第370頁}}。而作為標誌著一代人成熟的事件，太陽花學運也造成不小的世代衝突<ref>{{Cite web |url = https://daybreak.newbloommag.net/2017/06/10/generational-conflict/ |title = Generational Conflict And The Sunflower Movement |author =  |date = 2017年6月10日 |publisher = Daybreak Project |language = en |accessdate = 2019年3月1日 |quote =  }}</ref>{{sfnm|吳叡人等|2016年|1loc=第107頁至第108頁}}。此外，許多太陽花學運的運動者堅持進步的左翼政治立場，這一事實對年輕人的政治觀點發展具有重要意義<ref>{{Cite web |url = https://daybreak.newbloommag.net/2017/07/26/progressive-politics/ |title = The Left Political Vision Of The Sunflower Movement? |author =  |date = 2017年7月26日 |publisher = Daybreak Project |language = en |accessdate = 2019年3月1日 |quote =  }}</ref>。2014年9月20日，[[唐獎|唐獎]]獲獎人[[余英時|余英時]]在演講中表達對該運動的支持，其讚賞學生的意圖，並認為民主社會的公民都應該用其選票抵制政府<ref>{{Cite web |url = http://www.taipeitimes.com/News/front/archives/2014/09/21/2003600203 |title = Tang Prize’s Yu praises Sunflower protests |author = [[中央通訊社|中央通訊社]] |date = 2014年9月21日 |publisher = 《[[台北時報|台北時報]]》 |language = en |accessdate = 2019年3月1日 |quote =  }}</ref>。而包括[[埃里克·奧林·賴特|埃里克·奧林·賴特]]、[[子安宣邦|子安宣邦]]等重要學者都曾觀察太陽花學運，而[[吳叡人|吳叡人]]則提出《黑潮論》說法{{sfnm|晏山農等|2015年|1loc=第97頁至第98頁、第102頁|吳叡人等|2016年|2loc=第19頁、第25頁至第50頁}}。

太陽花學運的最後階段與結束方式，對後來的運動方式產生很大影響<ref>{{Cite web |url = https://daybreak.newbloommag.net/2017/11/13/movement-endgame/ |title = The Endgame Of The Movement |author =  |date = 2017年11月13日 |publisher = Daybreak Project |language = en |accessdate = 2019年3月1日 |quote =  }}</ref>，甚至被認為是啟發2014年的香港[[雨傘革命|雨傘革命]]、及2015年的日本[[自由民主主義學生緊急行動|自由民主主義學生緊急行動]]<ref>{{Cite web |url = https://www.bbc.com/zhongwen/trad/china/2015/10/151001_hk_occupy_anniversary_taiwan_view |title = 台灣學運領袖陳為廷談「佔中」一週年：「香港太難了」 |author = 劉子維 |date = 2015年10月1日 |publisher = [[英國廣播公司新聞網|英國廣播公司新聞網]] |language = en |accessdate = 2019年3月1日 |quote =  }}</ref>{{sfnm|公民聲音團隊|2014年|1loc=第16頁|晏山農等|2015年|2loc=第193頁至第194頁、第324頁至第326頁|吳叡人等|2016年|3loc=第156頁、第264頁至第265頁}}。不過許多傳播媒體對於太陽花學運的報導，都集中在被視為運動主要領導者的林飛帆和陳為廷兩人身上，這也被認為是促成個人主義<ref>{{Cite web |url = http://focustaiwan.tw/news/aipl/201403310016.aspx |title = Activist profiles: the leaders of the Sunflower Student Movement |author = John Scot Feng |date = 2014年3月31日 |publisher = [[中央通訊社|中央通訊社]] |language = en |accessdate = 2019年3月1日 |quote =  }}</ref><ref>{{Cite web |url = https://www.wsj.com/articles/young-protesters-shaking-up-taiwans-china-policy-1396351842 |title = Young Protesters Shaking Up Taiwan's China Policy |author = Jenny W. Hsu |date = 2014年4月1日 |publisher = 《[[華爾街日報|華爾街日報]]》 |language = en |accessdate = 2019年3月1日 |quote =  }}</ref><ref>{{Cite web |url = https://daybreak.newbloommag.net/2017/07/24/personalism-leadership/ |title = Personalism and Leadership in the Sunflower Movement |author =  |date = 2017年7月24日 |publisher = Daybreak Project |language = en |accessdate = 2019年3月1日 |quote =  }}</ref>，甚至後來還有對學生領袖的「告白牆」{{sfnm|黃恐龍|2014年|1loc=第237頁至第245頁|晏山農等|2015年|2loc=第105頁、第325頁}}。在整個運動過程中，立法院領中央導層必須解決一系列問題，其長期的主要挑戰是在公眾眼中維護運動的合法性，但也因而遭到各式批評<ref>{{Cite web |url = https://daybreak.newbloommag.net/2017/11/14/sunflower-central-leadership/ |title = Criticisms Of The Central Leadership In The Legislative Yuan |author =  |date = 2017年11月14日 |publisher = Daybreak Project |language = en |accessdate = 2019年3月1日 |quote =  }}</ref><ref>{{Cite web |url = https://daybreak.newbloommag.net/2017/07/24/movement-legitimacy/ |title = Maintaining The Legitimacy Of The Movement |author =  |date = 2017年7月24日 |publisher = Daybreak Project |language = en |accessdate = 2019年3月1日 |quote =  }}</ref>{{sfnm|晏山農等|2015年|1loc=第320頁至第322頁}}。與此同時，運動中的部份內部爭論異議，被批評為與具體的政治分歧無關，而是在某種程度上成為社會資本的鬥爭<ref>{{Cite web |url = https://daybreak.newbloommag.net/2017/06/10/internal-dissent/ |title = Internal Dissent Within The Movement |author =  |date = 2017年6月10日 |publisher = Daybreak Project |language = en |accessdate = 2019年3月1日 |quote =  }}</ref>{{sfnm|晏山農等|2015年|1loc=第325頁}}。
