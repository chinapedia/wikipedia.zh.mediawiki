'''太极图'''是指代表[[阴阳|阴阳]]（[[太极|太极]]）思想的中国符号，有时也泛指历史上其他文化中所使用的与中国太极图相似的[[几何|几何]]图案。这类符号标志大部分是由两个不同颜色的半圆形泪珠状曲线组成，稱為黑白陰陽双魚。以太極圓直徑的一半作兩個圓，太極圓被分成四部分，每兩部分組合起來，便成為一對陰陽双魚。或者是由一个反S型曲线将一个[[圆|圆]]分成两部分，每一部分都标记有一个不同颜色的圆点，一般由[[黑|黑]]和[[白|白]]两种颜色组成。在西元前五世紀[[凯尔特艺术|凯尔特艺术]]和[[古典时代晚期|古典时代晚期]][[西罗马帝国|西罗马帝国]]军服的[[纹章|纹章]]中都有这种类型的符号。

== 太極圖 ==
[[File:Taiji.svg|thumb]]
[[File:Yin_yang.svg|thumb]]
[[File:Taijitu_06075171.jpg|thumb]]

很多人都將「太極陰陽圖」（又稱「太極兩儀圖」）錯認為「太極圖」。「太極」是陰陽兩分前之[[混沌|混沌]]狀態，自然沒有裏面之「陰陽人」，只有外面之圓圈，代表宇宙周而復始之本質<ref>劉力紅，《思考中醫》，台灣：2004，ISBN 986-7863-61-5</ref>。另有說法是只有圓圈是「[[無極|無極]]」、圓圈內有點點是「太極」，代表事情開始成形、然後是「兩儀（一般所說之『太極圖』）」、陰陽中有陰陽是「[[皇極|皇極]]」<ref>{{cite web|title=一、太極源起|url=http://www.kztjq.com/big5/kzdx21.htm|accessdate=2014-01-07|language=zh-tw|deadurl=yes|archiveurl=https://web.archive.org/web/20140519023738/http://www.kztjq.com/big5/kzdx21.htm|archivedate=2014-05-19}}</ref>。

=== 無極圖 ===
太極圖據說是[[宋朝|宋朝]]处士[[陳摶|陳摶]]所傳出，原叫《無極圖》。陳摶是五代至宋初的一位隐士，相傳對內丹術和易學都有很深造詣。據史書記載，陳摶曾將《[[先天圖|先天圖]]》傳給其學生[[种放|种放]]，种放以之分別傳[[穆修|穆修]]等人；种放又有傳《河圖》、《洛書》給李溉等人。後來穆修則傳〈太極圖〉傳給[[周敦頤|周敦頤]]<ref>《[[s:宋史|宋史·卷435·列傳第194·儒林五·范沖朱震胡安國子:寅 宏 寧]]》记载，“震經學深醇，有《漢上易解》云：「陳摶以《先天圖》傳种放，放傳穆修，穆修傳李之才，之才傳邵雍。放以《河圖》、《洛書》傳李溉，溉傳許堅，許堅傳范諤昌，諤昌傳劉牧。穆修以《太極圖》傳周惇頤，惇頤傳程顥、程頤。」”</ref>。周敦頤寫了《[[太極圖說|太極圖說]]》加以解釋<ref>[[北宋|北宋]][[朱震|朱震]]《[[漢上易傳卦圖上·太極圖|漢上易傳卦圖上·太極圖]]》記載,“右太極圖，周敦實茂叔傳二程先生。茂叔先生曰: 無極而太極，太極動而生陽，動極而靜，靜而生陰，靜極復動，一動一靜，戶為其根.....斯其至矣！”</ref>。

《太極圖》，和原圖《無極圖》的圖形一樣，但讀法不同。

涵括[[內丹|內丹]]修煉之理的“無極圖”。讀它的順序是從下而上。
最下一圈稱為玄牝之門，是內丹修煉的起點，指人身的下丹田命門兩腎空隙之處。這是人身祖氣所在之地，一切內丹根基，都從這裡開始。內丹術語，又稱為得竅。

第二圈稱煉氣化神，指提起第一圈的祖氣，進行化煉，將後天之精化為先天之氣，再將先天之氣化為先天之神。這一層功夫 ，叫做煉己，主要在煉去後天的陰質，透出先天之神。

第三層，是五行之氣混合之象，指肝（木）、心（火）、脾（土）、肺（金）、腎（水）五臟之氣攢簇為一，稱為五氣朝元，五氣調和，故又稱為和合。更上一層，係由坎離 二卦變形而成，叫做取坎填離。到這一境界，已經獲得體內的精華，煉丹之藥成熟，叫做得藥。坎卦中爻為陽，為實。離卦中爻為陰爻，為虛。它們代表內丹中的腎水與心火，水火相交，在卦象言，就是將坎卦中的陽爻抽出填入，成乾卦，意味人變成純陽之體。最上一層圓圈，稱煉神還虛，復歸無極，指在得藥的基礎上，進一步回返到無極 ，那便是虛空大道了。整個煉內丹的過程便告完成，脫出煉成的聖胎，成為[[仙人|仙人]]，所以叫做脫胎求仙。

至於太極圖，是從上往下讀的。道教認為，從大道化萬物的順序看，是從太極分陰陽開始的，煉內丹則是逆著來路返回到大道，與道同體，自然長生不死成神仙。所以順化逆化方向相反，但所沿著的路徑則是同一條 。

周敦頤寫下《太極圖說》，解釋太極的奧秘。前半段基本上符合[[道教|道教]]的思想。後半段比較多地講了[[儒家|儒家]]的仁義道德思想。 他的解釋，第一句是「自無極而太極」，意思是從無極生出太極，但南宋著名的理學家朱熹整理注解《太極圖說》時，刪去了「自」字，改成「無極而太極」，認為無極只是形容太極 ，說明它之上沒有更高的本原。這並不符合陳摶所傳《太極圖》的原意，而且抹去了它出自道教的事實。同時，朱熹對圖也有個別改動。

[[道教|道教]]界人士，從宋以後也認同周敦頤的觀點。比如南宋時的[[蕭應叟|蕭應叟]]就將之採入自己的重要著作《元始無量上品妙經內義》，以後四十三代天師[[張宇初|張宇初]]也都採納其圖與說。

太極圖有另一種更加簡捷的畫法：就是用一個以曲線分隔的圓圈表示。

在這一圖中，一條曲線將它分為兩半，形成一半白一半黑，白者像陽，黑者像陰，白中又有一個黑點，黑中又有一個白點，表示陽中有陰，陰中有陽。分開的兩半，酷似兩條魚，所以俗稱陰陽魚。這一圖，與周敦頤太極圖有密切關係，可以說是前者的更加簡明的表述。它說明了世界由[[陰陽|陰陽]]二氣、二元對立相互聯結的統一體。這一圖，過其[[圓心|圓心]]作任何一條直線將之分成兩半，任何一半中都包含陰陽兩個因素，絕不存在孤立的沒有內在矛盾的成份。有時人們又在其外面圍以先天六十四卦，表示太極是一切運動的發動者。

== 欧洲 ==
太极图是一种简单的几何形状，这一类符号出现在许多的文化中。早在公元前五世纪，在[[凯尔特艺术|凯尔特艺术]]中就有与后来宋代[[道教|道教]]的太极图相似的图案。后来在[[罗马帝国|罗马帝国]]的军服徽章中出现了与[[远东|远东]]地区几乎完全一样的太极图图案，只是颜色不一样。

== 道教 ==
道教的太极图据传是[[宋朝|宋朝]]的[[陈抟|陈抟]]所创。原叫“无极图”。据史书记载，陈抟曾将“太极图”传给其学生[[种放|种放]]，种放以之分别传[[穆修|穆修]]、[[李溉|李溉]]等人，后来穆修将“太极图”传给[[周敦颐|周敦颐]]。周敦颐写了《太极图说》加以解释。现在我们看到的太极图，就是周敦颐所传的。

如今的太极图还是一些[[武术|武术]]，特别是[[太极拳|太极拳]]的标志之一。[[韩国国旗|韩国国旗]]的图案也是源自太极图。[[法轮功|法轮功]]的法轮图形上就有4个太极图（分别在左上、左下、右上、右下4个角落处）。
