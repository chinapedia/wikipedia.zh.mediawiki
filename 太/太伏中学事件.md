'''太伏中學事件'''，是指2017年4月1日[[四川省|四川省]][[泸州市|泸州市]][[泸县|泸县]]太伏中学学生赵鑫非正常死亡後，所引起的一系列民众对政府的不信任、警民对持事件<ref name="a1">{{Cite web |url=http://news.youth.cn/gn/201704/t20170408_9437353.htm |title=太伏中学事件官方调查结果 |accessdate=2017-04-10 |author= |date=2017-04-08 |work= |publisher=中国青年网，来源：新华社|language=Zh-hans }}</ref>。

== 死者身份 ==

死者赵鑫为泸县[[太伏镇|太伏镇]]照南山村人，生于2002年12月，生前就读于太伏镇太伏中学八年级（初二），为学校住宿生。赵鑫父母已离异，父亲拥有赵鑫[[监护权|监护权]]，但在[[贵州省|贵州省]]务工。赵鑫由祖父母照料，祖父母均为当地[[农民|农民]]<ref name="a1"/>。

== 官方通报 ==
2017年4月7日，四川泸州市委市政府召开媒体见面会,通报泸县太伏中学学生死亡事件的相关情况。出席本次媒体见面会的有中共泸州市委书记蒋辅义，泸州市政府副市长、市公安局局长何绍明，事件调查技术负责人、省公安厅刑侦局刑事技术处处长王庆红，泸州市委宣传部副部长、政府新闻办主任魏仁泉，泸州市教育局局长黄辉，泸州市公安局网监支队支队长 李红兵。会议由泸州市委政法委书记封安主持。泸州市副市长、公安局局长何绍明在会上介绍了公安机关相关调查情况，并且表示，通过省市县三级技术人员的现场勘察、尸体检验、物证鉴定，结合对赵鑫的老师、同学、室友、保安、小卖部等人员的大范围深入细致的调查，可以确认：赵鑫的损伤为高坠伤，无其他暴力加害形成的损伤，可以排除他杀。<ref>{{Cite web|url=http://legal.people.com.cn/n1/2017/0407/c42510-29195963.html|title=泸州公安局长：泸县太伏中学学生死亡事件排除他杀--法治--人民网|accessdate=2018-02-14|author=|date=|last=3089|work=legal.people.com.cn|publisher=}}</ref><ref>{{Cite news|url=http://legal.people.com.cn/n1/2017/0407/c42510-29195963.html|title=泸州公安局长：泸县太伏中学学生死亡事件排除他杀--法治--人民网|date=2018-02-14|newspaper=archive.is|accessdate=2018-02-14|deadurl=yes|archiveurl=https://archive.today/20180214125148/http://legal.people.com.cn/n1/2017/0407/c42510-29195963.html|archivedate=2018-02-14}}</ref>

== 事件经过 ==

=== 4月1日 ===

1日6点20分，泸县公安局太伏镇派出所接报：太伏镇太伏街太伏中学男生宿舍楼外发现一无动静的卧地男性。派出所出警后，通知120救治，并通知相关人员进行现场勘验<ref name="a1"/>。

=== 4月3日 ===

4月5日报道，可见在太伏中学外有大量人群聚集，与官方人员对峙。当地政府派出警察、[[特警|特警]]进驻[[维稳|维稳]]<ref name="a2">{{Cite web |url=http://www.rfa.org/mandarin/yataibaodao/renquanfazhi/yf2-04052017103656.html |title=泸州太伏中学事件逐渐平息 家属拒尸检民众仍存疑 |accessdate=2017-04-10 |author= |date=2017-04-05 |work=特约记者：扬帆，责编：石山、陈平 |publisher=自由亚洲电台普通话|language=Zh-hans }}</ref>。

=== 4月4日 ===

赵鑫家属同意[[尸检|尸检]]，启动尸检程序<ref name="a2"/>。

=== 4月5日 ===

4月5日报道，当地事态已逐步平息。同时，太伏镇工作人员对记者表示，赵鑫家属反悔，已拒绝尸检<ref name="a2"/>。

=== 4月6日 ===

赵鑫家属同意尸检，当天完成尸检<ref name="a1"/>。

=== 4月7日 ===

中共泸州市委、泸州市人民政府召开媒体见面会，通报事件相关情况<ref name="a1"/>。
