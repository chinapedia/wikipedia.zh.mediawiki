'''太史'''，中国古代官名。

== 沿革 ==
太史在[[西周|西周]]、[[春秋时期|春秋]]时掌管起草文书，策命[[诸侯|诸侯]][[卿_(官制)|卿]][[大夫_(官名)|大夫]]，记载史事，编写[[史书|史书]]，兼管典籍、[[天文学|天文]][[历法|历法]]、[[祭祀|祭祀]]等，为朝廷大臣。《[[後汉书|後汉书]]·光武纪》注:“史官之长也。”后来职位渐低，[[秦朝|秦朝]]称[[太史令|太史令]]，[[汉朝|汉朝]]属[[太常|太常]]，掌天文历法，[[品秩|秩]]六百石<ref>班固所著《汉书 百官公卿表上》：“自太常至执金吾，秩皆中二千石，丞皆千石。”由此可知太史的俸禄应为一千石。博士俸禄只有600石。</ref>；[[魏晋|魏晋]]以后修史的职务划归[[著作郎|著作郎]]，太史仅掌管推算历法；[[隋朝|隋朝]]改[[太史监|太史监]]，[[唐朝|唐朝]]改[[太史局|太史局]]，[[唐肃宗|唐肃宗]]时又改为[[司天台|司天台]]，至明清两朝，修史之事由[[翰林院|翰林院]]负责，因此又称[[翰林|翰林]]为太史。

清朝時，康熙二十九年（1690年）、雍正元年（1723年）、乾隆元年（1736年）曾開立非常設的[[國史館|國史館]]。乾隆三十年（1765年），國史館成為常設機構。<ref>{{cite web |url=http://repository.lib.ied.edu.hk/jspui/handle/2260.2/19066 |title=賴太史與香港的中文教學 |author=施仲謀 |date=2015-12}}</ref>

[[司马迁|司马迁]]官职即为太史，在《[[史记|史记]]》中的作者评述多以“太史公曰”的句子起始，所以太史一词有时也指[[司马迁|司马迁]]所著的《[[史记|史记]]》。如：[[唐朝|唐朝]]的[[柳宗元|柳宗元]]《[[柳河东集|柳河东集]]》：“参之太史。”后代作者则模仿司马迁自称“太史氏”。
