'''太田康資'''（1531年—1581年11月8日／1588年）是[[日本戰國時代|日本戰國時代]]至[[安土桃山時代|安土桃山時代]]的武將。父親是[[太田資高|太田資高]]。母親是[[北條氏綱|北條氏綱]]的女兒淨心院。

== 生平 ==
在[[享祿|享祿]]4年（1531年）出生，家中次男（嫡男）。[[天文_(後奈良天皇)|天文]]17年（1547年），在父親資高死去後，繼承[[家督|家督]]，並成為[[江戶城|江戶城]][[城代|城代]]。

武勇優秀。天文23年（1554年），在北條氏在[[駿河國|駿河國]]與[[甲斐武田氏|甲斐武田氏]]戰鬥時，撃退[[原虎胤|原虎胤]]隊等。不過因為得到的恩賞不多而不滿（一説是因為不能成為曾祖父[[太田資長|資長]]（道灌）所築的江戶城城主而不滿，當時在江戶城本丸的是{{tsl|ja|富永氏}}（[[富永直勝|富永直勝]]），二之丸的是{{tsl|ja|遠山氏}}（[[遠山綱景|遠山綱景]]），而自身則在三之丸，被迫成為3個城代之一），[[永祿|永祿]]5年（1562年），與同族的[[太田資正|太田資正]]連結，計畫投向上杉家。根據北條方的史料，是受到資正和[[里見氏|里見氏]]的建議。不過計畫失敗，同年10月，逃亡到資正之下。翌年（1563年），氏康攻略[[武藏國|武藏國]][[松山城|松山城]]，並計畫乘勢攻撃資正、康資。[[上杉謙信|上杉謙信]]為了救援屬少數投向自己的武藏國人，於是邀請[[安房國|安房國]]、[[上總國|上總國]]、[[下總國|下總國]]的領主[[里見氏|里見氏]]出兵，里見家的[[當主|當主]][[里見義堯|里見義堯]]派遣嫡男[[里見義弘|義弘]]為總大將，率軍進入{{tsl|ja|国府台城|國府台城}}。

永祿7年（1564年）初，北條軍與里見太田聯軍激戰，結果被北條軍大敗（[[第二次國府台之戰|第二次國府台合戰]]）。此後，逃亡到安房投靠里見氏，之後亦繼續與北條戰鬥。[[天正_(日本)|天正]]9年（1581年），在里見家中出現爭奪[[家督|家督]]的混亂時，被捲入[[正木憲時|正木憲時]]的反亂中，因為支援憲時，最終與憲時一同在[[小田喜城|小田喜城]]自殺。另一説法指，與資正一同逃亡並投靠[[佐竹氏|佐竹氏]]。

== 異說 ==
一說指[[德川家康|德川家康]]寵愛的側室[[英勝院|英勝院]]是康資的女兒。另外，亦有說法指在天正19年（1591年）被[[佐竹氏|佐竹氏]]流放的[[江戶重通|江戶重通]]的女兒成為康資的養女（如此則與此前在天正9年（1581年）死去的說法不符）。在[[江戶時代|江戶時代]]領有5萬石的[[太田氏|太田氏]]是英勝院的親屬，而太田氏的血脈在此後不明，太田氏的家系圖亦多數被認為可信性甚低，很可能是在[[太田資宗|太田資宗]]一代被竄改。

== 人物 ==
*所領約2千[[貫|貫]]（後來直轄領減至一半），相當於[[後北條氏|後北條氏]]家臣的第6位貫高，可見得到氏康重視。不過在所領役帳中，高級[[國眾|國眾]]的名字後會加上「殿」字，不過雖然本領有931貫384文，卻沒有加上「殿」字，因此被認為在江戶地域內，影響力低下。另外，名字中的「康」字是[[北條氏康|北條氏康]]下賜的[[偏諱|偏諱]]。

*在『[[關東古戰錄|關東古戰錄]]』卷六中記述，「身丈6尺以上，筋骨很大，見者稱之為「大人」。聲音很大，像雷震一樣，擁有1人舉起要30個普通男人才能搬起的岩石的強力」（{{lang|ja|身丈6尺以上、筋骨は太く、見るものは「大人」と呼ぶ。声は太く、雷が震えるようで、普通の男が30人程で持ち上げる岩を1人で軽々と持ち上げる強力でもある}}）。
