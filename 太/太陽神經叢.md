'''太陽神經叢'''（[[梵語|梵語]]：मणिपूर、{{IAST|Maṇipūra}}，意謂「寶石之城」）是[[印度教|印度教]]傳統的第三個主要[[脈輪|脈輪]]。也是道教下丹田的别称[见《[[唱道真言|唱道真言]]》]

== 描述 ==
太陽神經叢位於[[肚臍|肚臍]]的區域，有十片[[花瓣_(脈輪)|花瓣]]，對應精神上的無知、渴望、[[妒忌|妒忌]]、[[背叛|背叛]]、[[羞恥|羞恥]]、[[恐懼|恐懼]]、[[厭惡|厭惡]]、[[妄想|妄想]]、愚昧與[[悲傷|悲傷]]等心念（[[:en:vritti|vritti]]）。

太陽神經叢與動力、能量、以及意志力（[[:en:itcha_shakti|itcha shakti]]）有關。它也和火能量與消化有關。據說太陽神經叢會將[[普拉納|普拉納]]（[[氣_(印度醫學)|氣]]）送往身體其他部分。從這一點來看，它和[[氣功|氣功]]裡的[[丹田|丹田]]大致相似。

== 象徵 ==
太陽神經叢與以下有關：
* 神祇：[[樓陀羅|樓陀羅]]、萊基尼（Lakini）
* 元素：[[火|火]]
* 動物：[[羊|公羊]]
* 身體部位：[[消化系統|消化系統]]、[[眼|眼]]、[[腳|足]]

== 修練 ==
在亢達里尼瑜伽中，有一些不同的修練方式可以激發並平衡太陽神經叢能量，包括許多對該部位起作用的體位法、呼吸法、收腹收束法（吐氣，並將腹部往後收、橫膈膜往上提）、以及腹疊背法（進行收頷收束法，同時反覆擴張、收縮腹部）。

== 其他關聯 ==
在[[內分泌系統|內分泌系統]]中，太陽神經叢據說與[[胰臟|胰臟]]和[[腎上腺皮質|腎上腺皮質]]（腎上腺的外部）有關。這些腺體會製造參與消化的重要激素，將食物轉化為供給身體的能量，就像太陽神經叢把[[普拉納|普拉納]]散發到全身一樣。中國的[[氣功|氣功]]裡有三個[[丹田|丹田]]，會散布並調節[[氣|氣]]的能量（與印度的[[普拉納|普拉納]]相似的概念）。位置最低的丹田（下丹田）在胃部區域。

西方神秘學家對於太陽神經叢與[[卡巴拉|卡巴拉]]的關聯各有不同的說法。有些人認為它和[[Hod|Hod]]與[[Netzach|Netzach]]兩個原質（sephira）相關，其中Netzach代表克服各種障礙的能量品質，而Hod則是控制能量並將其分化為不同形式的傾向，這兩股能量相互抗衡，就像人體的[[同化作用|合成代謝]]與[[異化作用|分解代謝]]一樣。
