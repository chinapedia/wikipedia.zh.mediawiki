'''太麻里圳'''為一座位於[[臺灣|臺灣]][[臺東縣|臺東縣]][[太麻里鄉|太麻里鄉]]的[[灌溉|灌溉]]水圳系統，太麻里圳最早興建於1884年（民國前28年），由當地[[原住民族|原住民族]]自行開鑿建造，而在2010年（民國99年）時，[[莫拉克颱風|莫拉克颱風]]侵襲導致太麻里圳進水口、沉沙池以及多條支線與分線遭到暴漲的太麻里溪水淹沒，現已修復，目前由[[台東農田水利會|台東農田水利會]][[知本|知本]][[工作站|工作站]]進行管理與維護。'<ref>{{cite web |url=http://www.ttia.gov.tw/page6.php |title=台東農田水利會 主要圳系建造年代表 |accessdate=2015-02-09 |deadurl=yes |archiveurl=https://web.archive.org/web/20141208020113/http://www.ttia.gov.tw/page6.php |archivedate=2014-12-08 }}</ref>

== 設施 ==

=== 進水口 ===

太麻里圳進水口設置於太麻里溪北岸[[堤防|堤防]]上，並有攔汙閘的配置，屬於[[閘門|閘門]]控制式。'<ref name="灌排系統 台東農田水利會灌排系統 太麻里圳">{{cite web |url=http://student.kmes.tc.edu.tw/taiculture/water/14/C/04/08/C-14-04-08-menu.htm |title=灌排系統 台東農田水利會灌排系統 太麻里圳 |accessdate=2015-02-09 |deadurl=yes |archiveurl=https://web.archive.org/web/20150209164615/http://student.kmes.tc.edu.tw/taiculture/water/14/C/04/08/C-14-04-08-menu.htm |archivedate=2015-02-09 }}</ref>

=== 沉沙池 ===

太麻里圳沉沙池設置在進水口後方，沈砂池全長70公尺、寬7公尺。<ref name="灌排系統 台東農田水利會灌排系統 太麻里圳"/>

=== 導水設施 ===

太麻里圳幹線全長2.40公里，最大輸水容量為1.0C.M.S.，總灌溉面積60.2公頃（莫拉克災區重建後統計）。<ref name="灌排系統 台東農田水利會灌排系統 太麻里圳"/>

== 改善工程 ==

* '''太麻里圳支分線改善工程'''<ref name="台東農田水利會 太麻里圳支分線改善工程">{{cite web |url=http://www.ttia.gov.tw/page6-1-1.php?PK=48 |title=台東農田水利會 太麻里圳支分線改善工程 | accessdate=2004}}</ref>

該工程目的為改善太麻里圳圳路老舊之問題，工程於2004年（民國93年）施工。

* '''臺東農田水利會太麻里圳進水口修復工程'''<ref name="台東農田水利會 專題報導 太麻里圳進水口修復工程">{{cite web |url=http://www.ttia.gov.tw/page13-8.php |title=台東農田水利會 專題報導 太麻里圳進水口修復工程 | accessdate=2010-08-08}}</ref>

太麻里圳進水口因2010年（民國99年）8月8日莫拉克颱風帶來豪雨造成太麻里溪兩岸堤防潰堤，將原有進水口及沉砂池沖毀，嚴重影響太麻里地區當地[[農田|農田]]灌溉，為維持正常供灌，台東農田水利會撥款進行太麻里圳進水口災害修復工程，總工程費用[[新台幣|新台幣]]1375萬3千元。<ref name="台東農田水利會 專題報導 太麻里圳進水口修復工程"/>

本工程包含進水口1座、暗渠工141公尺、明渠工35.5公尺、電動吊門機1組、閘門體及門框1組、沉砂池1座、暗渠工64公尺及其他1全，受益面積為214公頃，年產值約新台幣2000萬。<ref name="台東農田水利會 專題報導 太麻里圳進水口修復工程"/>
