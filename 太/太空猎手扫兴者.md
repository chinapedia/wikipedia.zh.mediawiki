《'''掃興者'''》（{{lang-en|''Killjoys''}}）是一部[[加拿大|加拿大]]太空冒險电视剧，于2015年6月19日在[[SPACE_(电视频道)|Space]]首播第一季，共10集。<ref name=CraveKilljoysPremiere>{{cite web|last1=Cox|first1=Jennifer|title=Space Original Series Killjoys Premieres June 19th|url=http://www.craveonline.com/tv/articles/856395-space-original-series-killjoys-premieres-june-19th|website=CraveOnline.com|publisher=Evolve Media, LLC|date=May 14, 2015|accessdate=June 30, 2015}}</ref>本劇於2015年9月訂續第二季，在2016年7月1日首播。<ref>{{Cite web |url = http://deadline.com/2015/09/killjoys-dark-matter-renewed-season-2-syfy-1201511362/ |title = 'Killjoys' & 'Dark Matter' Renewed For Season 2 By Syfy |author = Denise Petski |date = 2015-09-01 |publisher = Deadline Hollywood |language = en}}</ref><ref>{{Cite web |url = http://tvline.com/2016/04/29/killjoys-season-2-premiere-date-syfy-dark-matter/ |title = Killjoys, Dark Matter Get Season 2 Premiere Dates at Syfy |author = Ryan Schwartz |date = 2016-04-29 |publisher = TVLine |language = en}}</ref>2016年9月續訂第三季，於2017年6月30日播出。<ref>{{Cite web |url = http://deadline.com/2016/09/killjoys-dark-matter-renewed-third-season-syfy-1201812120/ |title = 'Killjoys' & 'Dark Matter' Renewed For Third Season By Syfy |author = Denise Petski |date = 2016-09-01 |publisher = Deadline Hollywood |language = en}}</ref>2017年9月1日，Syfy續訂第四季和第五季，都是由10集組成，第四季於2018年播放。<ref>{{cite web |url = http://deadline.com/2017/09/killjoys-to-end-run-two-season-final-renewal-syfy-1202159831/ |title = 'Killjoys' To End Run With Two-Season Final Renewal By Syfy |publisher = Deadline Hollywood |author = Denise Petski |date = 2017-09-01 |language = en}}</ref>第五季也是最後一季在2019年播放。

== 劇情 ==
主人公是三個星際賞金獵人，分別是Dutch、John和D'avin。他們一方面要穿越政治漩渦，一方面要面對道德觀的挑戰，還要隨時處理複雜的「內部關係」。在遙遠的銀河系深處，Qresh星球上的人們在三顆衛星上進行殖民，建立起四個分層的世界（主行星和三顆衛星各為一層）。每一個世界都有獨特的社會經濟結構，但每一個世界都動盪不安。在被稱作Quad的這個行星體中，賞金獵人獲得政府許可在整個銀河系範圍內完成各種各樣的委託命令，他們被稱作「掃興者」（Killjoys）。「掃興者」必須嚴格服從既定的規矩，並且他們一旦接到命令就不能反悔，如果反悔必然帶來嚴重後果。「掃興者」不允許參與政治，不允許在任務中帶有私人感情——女「掃興者」Dutch就面臨這樣的考驗。Dutch美艷、性感、身手不凡，但是背景極為複雜，她在這個行業中是數一數二的人物。Dutch的搭檔John綽號「混蛋」，他的性格和Dutch正好相反——他不喜歡和人發生衝突，更願意當個和事老。John的兄弟、性感、自信的D'avin很快也加入到他們當中，這三人的關係從此糾纏在一起，一段詭異的三角戀開始了。
