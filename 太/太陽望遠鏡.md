[[File:Swedish_Solar_Telescope.jpg|thumb]]的[[瑞典太陽望遠鏡|瑞典太陽望遠鏡]]，位於[[加那利群島|加那利群島]][[拉帕爾瑪島|拉帕爾瑪島]]的[[穆查丘斯罗克天文台|穆查丘斯罗克天文台]]。]]

'''太陽望遠鏡'''是專門用於[[太陽|太陽]]觀測，是用途特殊的[[光學望遠鏡|光學望遠鏡]]。

== 專業的太陽望遠鏡 ==
太陽望遠鏡需要有足夠大的口徑以獲取最佳的[[繞射#繞射極限|繞射極限]]，不同於其他天文望遠鏡的是，太阳望远镜無需考量望遠鏡的集光力。因為太陽望遠鏡是在白天工作，並且目標是非常明亮的[[太陽|太陽]]，並且因為[[視寧度|視寧度]]受到大氣層的影響遠比夜間的嚴重，所以這些望遠鏡的[[物鏡|物鏡]]口徑都在1公尺以下，甚至更小。被聚焦的陽光所凝聚的熱也是設計上的一個問題，專業的太陽望遠鏡會使用非常長的[[焦距|焦距]]，並通過[[真空|真空]]的光路以減少大气擾動與望遠鏡內部[[對流|對流]]運動的影響，因此太陽望遠鏡相對的都很巨大（有些是世界上最巨大的光學望遠鏡）。它們目標的影像（太陽）以狹窄且固定的路徑橫越過天空，太陽望遠鏡通常是固定不動的（有些還被掩蓋在地下），唯一的運動機制是追蹤太陽的[[定日鏡|定日鏡]]。這些望遠鏡使用濾鏡和投射技術直接觀測，另外還有配置不同濾鏡的各種照相機。專業的工具，例如[[光譜儀|光譜儀]]和[[太陽單色光觀測鏡|太陽單色光觀測鏡]]在各種不同的[[波長|波長]]上觀測太陽。

== 其它型式的天文台 ==
大部分觀測太陽的天文台都在可見光、近紅外和紫外線的波段上觀測，在其他的波段上也可以觀測太陽－－但受限於[[地球大氣層#吸收|大氣吸收]]，不能在地球表面進行觀測：
*太陽X射線天文學：在太空，以X射線的波段觀測太陽。
*多光譜太陽望遠鏡陣列（[[MSSTA|MSSTA]]）：在1990年代，以火箭發射紫外線望遠鏡進入太空觀測。
* [[里翁西多綜合天文台|里翁西多綜合天文台]]：以次微米波太陽望遠鏡觀測太陽。
* [[電波太陽望遠鏡網路|電波太陽望遠鏡網路]]（RSTN）：是由[[美國|美國]][[美國空軍氣象局|空軍氣象局]]維護的一個太陽觀測站網路。
* [[CERN軸子太陽望遠鏡|CERN軸子太陽望遠鏡]]（CAST）：在2000年代早期尋找假想的太陽[[軸子|軸子]]。

== 業餘的太陽望遠鏡 ==
[[File:Solarborg.jpg|thumb]]系統。]]

在[[業餘天文學|業餘天文學]]的領域內有許多方法被用來觀測太陽。業餘使用的方法從最簡單的將太陽影像投影在白紙上、使用{{link-en|赫歇爾楔型太陽稜鏡|Herschel wedge}}[[赫歇尔棱镜|赫歇尔棱镜]]從目鏡移除95%的陽光<ref>[http://books.google.com/books?id=1F_zSwe9iU4C&pg=PA37&dq=herschel+wedge&hl=en&ei=e8YZTsONFYrY0QHZ8dCWBQ&sa=X&oi=book_result&ct=result&resnum=1&ved=0CC0Q6AEwAA#v=onepage&q=herschel%20wedge&f=false, Pierre Guillermier, Serge Koutchmy, Total eclipses: science, observations, myths, and legends, page 37]</ref>，到使用[[H-α#濾鏡|Hα濾鏡]]的系統，甚至還有在家中安裝[[太陽單色光觀測鏡|太陽單色光觀測鏡]]的。相較於專業的望遠鏡，業餘的太陽望遠鏡通常都小了許多。
