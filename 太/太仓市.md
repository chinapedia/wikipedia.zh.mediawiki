'''太仓市'''，位于[[中华人民共和国|中国]][[江苏省|江苏省]]最南部，是[[苏州市|苏州市]]代管的一个[[县级市|县级市]]。太仓是典型的[[江南|江南]]水乡城市，文化底蕴丰厚，工业发展迅速，是距离[[上海|上海]]最近的一座城市，太仓市区与上海市[[嘉定区|嘉定区]]相邻。太仓市区到[[上海地铁11号线|上海地铁11号线]]（[[嘉定北站|嘉定北站]]）有公交相连，用时20分钟。太仓市经济增速指标近年常位居苏州市域首位，太仓也是中国经济最为发达的[[县级行政区|县市]]之一，[[中国百强县#县域经济论坛评价|2012年列全国百强县（市）第四位]]。根据公安部门的数据统计，2012年全市常住人口为94.7万，户籍人口为47.26万，其中市区常住人口超过30万。预计2013年GDP将超过1000亿元，全口径财政总收入有望超过300亿元。

== 历史 ==
太仓之名，传因春秋吴王置仓于此而得名。首见于宋郏亶《吴门水利书》：“昆山之东，地名太仓，俗号冈身”。

春秋时属吴地，秦汉为娄县惠安乡。梁天监六年（507年）析娄县置信义县，为信义县地。梁大同初（535年）为昆山县。元延祐元年（1314年）昆山州治迁至太仓。吴元年（1367年）设太仓卫。明弘治十年（1497年）割嘉定县乐智、循义二乡，昆山县新安、惠安、湖川三乡，常熟县双凤乡并建[[太仓州|太仓州]]，辖崇明县，隶苏州府。清雍正二年（1724年），升为江苏直隶州，辖镇洋、崇明、嘉定、宝山4个县。民国元年（1912年），太仓州和镇洋县合并，定名太仓县，至民国16年属沪海道。民国24年9月属无锡督察区。
明代为[[太仓州|太仓州]]，属[[南直隶|南直隶]][[苏州府|苏州府]]，清代为[[江蘇省_(清)|江苏省]][[太仓直隶州|太仓直隶州]]，下辖四县：[[镇洋县|镇洋县]]、[[嘉定县|嘉定县]]、[[宝山县|宝山县]]、[[崇明縣|崇明縣]]，辖境除今太仓市以及今上海市的[[嘉定|嘉定]]、[[寶山區_(上海市)|宝山]]、[[崇明|崇明]]三区以外，还包括今上海市[[杨浦|杨浦]]、[[虹口|虹口]]的部分、[[闸北区|闸北]]、[[普陀区_(上海市)|普陀]]两区的大部分和今[[启东市|启东市]]的部分地区。[[辛亥革命|辛亥革命]]后，全国[[废府州厅改县|废州]]，为[[太仓县|太仓县]]。

太仓素有“鱼米之乡”的美称，河道纵横，土地肥沃，农业曾占重要地位，近年来的工业大发展使得很多农田已被用来筑路和修建厂房，经济发展迅速。

== 地理 ==
太仓地处土地肥沃的太湖平原，全境无山。太仓位于[[长江|长江]]口南岸，东南方毗邻[[上海市|上海市]][[嘉定区|嘉定区]]和宝山区，西南方毗邻[[昆山市|昆山市]]，西北方毗邻[[常熟市|常熟市]]，隔长江与[[海门市|海门市]]及上海市[[崇明区|崇明区]]相望。

太仓市区和上海嘉定区相邻，上海轨交11号线嘉定北站到太仓市区有公交快线相连接，用时约20分钟。

城厢镇（市区）境内有太仓最大的湖泊-金仓湖，水面面积超过1000亩，总投资8亿元建成了金仓湖公园，面积达5.8平方公里，是太仓的绿肺以及市民休闲娱乐的好去处。其西将建成德国特色的小镇，以北形成万丰生态园。太沙公路穿湖区而过，在公路两侧形成万亩高产水稻和油菜地。

== 经济 ==
[[File:中国江苏省太仓市现代农业园.JPG|缩略图]]
太仓市区是全市的经济龙头，人口也较集中于市区。市区环境最好的在金仓湖，是国家4A级风景区。市区包括城厢镇、科教新城、娄东街道所辖区域。2012年，太仓市完成地区生产总值955亿元，地方公共财政预算收入90.2亿元，工业总产值2436.09亿元，全社会固定资产投资472.04亿元，注册外资16亿美元，实际利用外资8.1亿美元，新增内资注册资本102亿元，外贸进出口总额126.21亿美元。城镇人均可支配收入和农民人均纯收入分别达到39422元和19411元。太仓港成为全国第一个享受海港待遇的内河港口，完成集装箱吞吐量400万标箱、货物吞吐量逾1.2亿吨。主要以化学工业，塑胶，铝合金和包装、港口物流产业为主。大型企业有[[SHIMANO|SHIMANO]]、[[舍弗勒|舍弗勒]]等。目前，太仓市全境共有德资企业超过180家，被誉为“中国德企之乡”，成为中国与欧盟双方在经济贸易领域合作交流的重要示范区域。此外，太仓渔业亦非常发达，[[浏河|浏河]]是[[长江|长江]]沿线最大的远洋渔港。

　2016年全年实现地区生产总值1155.13亿元，按可比价格计算，比2015年增长7.3%。其中，第一产业增加值36.76亿元，下降5.5%；第二产业增加值583.87亿元，增长6.0%；第三产业增加值534.50亿元，增长9.7%。按常住人口计算，人均地区生产总值162523元，增长7.0%。第一产业增加值占地区生产总值的比重为3.2%，第二产业增加值比重为50.5%，第三产业增加值比重为46.3%。
　　2016年，太仓市共实现公共财政预算收入127.71亿元，比上年增长11.5%；其中税收收入110.52亿元，增长13.0%；税收占比为86.5%。全年公共财政预算支出115.84亿元，比上年增长6.1%。
    2016年全年新批外资项目49个，完成注册外资6.23亿美元，比2015年下降6.2%。完成实际利用外资5.61亿美元，增长11.9%，其中服务业到帐外资2.58亿美元，增长145.6%。实现服务外包接包合同额7.82亿美元，增长10.1%，离岸执行额2.75亿美元，增长1.2%；中方境外投资企业额3.23亿美元，增长46.2%。2016年全年实现进出口总额109.64亿美元，比上年下降13.9%，其中出口总额54.30亿美元，下降13.9%；进口总额55.34亿美元，下降13.8%。

== 名人 ==
*[[毕沅|毕沅]]（1730年—1797年）：史学家、经学家、政治家，乾隆二十五年（1760）状元。
*[[陆增祥|陆增祥]]（1815年－1882年）：道光三十年（1850年）状元 
*[[吴健雄|吴健雄]]（1912年—1997年）：（女）实验物理学家
*[[钱宝琛|钱宝琛]]（1785－1859）：嘉庆二十四年（1819）进士，改庶吉士，散馆授编修。
*[[钱鼎铭|钱鼎铭]]（1823－1875）：字新之，号调甫、定舫。宝琛子。少而劬学。
*[[钱抵千|钱抵千]]（1919-1989）：中国人民解放军著名理论家、中将，原任昆明军区政治部主任，解放军报社长，国防大学副校长。
*[[王锡爵|王锡爵]]（1534－1610） 字元驭，号荆石。明太仓人。與[[王世貞|王世貞]]同族、属于[[太倉王氏|太倉王氏]]。
*[[唐飛|唐飛]] (1932年3月15日－)：[[中華民國空軍|中華民國空軍]]一級上將，曾任[[中華民國參謀總長|參謀總長]]、[[中華民國國防部|國防部]]部長、[[行政院院長|行政院院長]]。
*[[朱棣文|朱棣文]]（英文名Steven Chu，1948年2月28日—）物理学家，曾任[[美國能源部|美國能源部]]部長，祖籍江苏太仓，[[诺贝尔物理学奖|诺贝尔物理学奖]]得主。
*[[王世贞|王世贞]]（1526年12月8日－1590年12月23日），字元美，号凤洲，又号弇（yǎn）州山人，南直隶苏州府太仓州（今江苏太仓）人，明代文学家、史学家。
*[[王掞|王掞]]（1645～1728）清朝官员。字藻儒，一作藻如，号颛庵、西田主人，江南太仓（今属江苏）人，明代首辅王锡爵曾孙。康熙九年进士，授编修，官至文渊阁大学士。康熙六十年，请重立胤礽为太子忤旨，应谪戍，以年老由子代行，寻致仕。有《西田集》
*[[张溥|张溥]]（注音：ㄓㄤ　ㄆㄨˇ，拼音：Zhāng pǔ）（1602年－1641年），字天如，太仓（属今江苏）人。明代文学家，复社领导人。
*[[王时敏|王时敏]]（1592年－1680年），初名赞虞，字逊之，号烟客，晚号西庐老人，南直隶太仓县（今江苏太仓市）人，祖籍山东莘县，明末清初重要画家。万历首辅王锡爵孙，翰林王衡独子。崇祯初年荫官为太常寺少卿，入清后不仕。
*[[仇英|仇英]]（约1494年－1552年）字实父，号十洲，中国明代画家，原籍江苏太仓，后移居苏州。生年不详，考订约生于弘治甲寅（一四九四年），卒于嘉靖壬子（一五五二年）秋冬之际。吴门四家之一。
*[[唐文治|唐文治]]（1865年12月3日－1954年4月），字颖侯，号蔚芝，晚号茹经，江苏太仓人，教育家，原交通大学校长。
*[[邹世昌|邹世昌]]（1931年7月27日－），中国材料科学家。原籍江苏太仓，生于上海。1952年毕业于唐山交通大学冶金工程系。1958年获苏联莫斯科有色金属学院副博士学位。中国科学院上海冶金研究所研究员。1991年当选为中国科学院院士（学部委员）。
*[[吴晓邦|吴晓邦]]（1906年－1995年）江苏太仓人。中华民国及中华人民共和国舞蹈家、舞蹈教育家。
*[[朱屺瞻|朱屺瞻]]（1892年5月27日－1996年4月20日），名增钧，号起哉，又号二瞻老民，斋名梅花草堂、癖斯居、养菖蒲室、修竹吾庐等。现代中国画家，兼修国画和西洋画，以水墨山水见长，善于融合西洋画的长处。曾任上海美术家协会常务理事。祖籍江苏省太仓县浏河镇。
