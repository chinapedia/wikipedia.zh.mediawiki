'''进步党'''是[[中华民国|中华民国]]初期的一个政党，成立于1913年5月29日。

==成立過程及宗旨==
1912年（民国元年）末，[[民元国会|国会]][[參議院|參議院]]與[[眾議院|眾議院]]举行选举，[[宋教仁|宋教仁]]领导[[国民党_(宋教仁)|国民党]]取得眾議院269个議席、參議院123個議席，國民黨在兩院議席佔百分之四十五多數議席。<ref name="成立">周佳榮、林啟彦：《共和國的追求與挫折——辛亥革命與中華民國的成立》，香港：[[商務印書館|商務印書館]]，1992年</ref>{{rp|112}}第2大党[[共和党_(中国)|共和党]]获得120个議席，第3大党[[統一党_(中国)|統一党]]获得18个議席，第4大党[[民主党_(1912年中华民国)|民主党]]获得16个議席。[[袁世凱|袁世凱]]拉攏共和、統一、民主三党牽制国民党。<ref name="成立"/>{{rp|112}}

1913年4月，國會正式開幕，[[共和党_(中国)|共和党]]（[[黎元洪|黎元洪]]领导）、[[民主党_(1912年中华民国)|民主党]]（[[梁启超|梁启超]]领导）、[[统一党|统一党]]（[[张謇|张謇]]领导）三党為进一步对抗国民党，於5月29日合并為为进步党，黎元洪為[[黨魁|黨魁]]，梁啟超亦為重要領袖之一。<ref name="成立"/>{{rp|112}}

進步党的主要宗旨为如下两点：
# 欲将全国政治导入轨道
# 欲造成一种可为模范之政党

具体政纲有如下三点：
# 采取国家主义，建设强善政府
# 尊重人民公意，拥护法赋自由
# 顺应世界大势，增进平和实利

==领导==
進步党的主要干部如下：
*理事長：[[黎元洪|黎元洪]]
*理事：[[梁啟超|梁啟超]]、[[張謇|張謇]]、[[伍廷芳|伍廷芳]]、[[孫武_(民国)|孫武]]、[[那彦图|那彦图]]、[[湯化龙|湯化龙]]、[[王揖唐|王揖唐]]、[[蒲殿俊|蒲殿俊]]、[[王印川|王印川]]
*名誉理事：[[冯国璋|冯国璋]]、[[蔡锷|蔡锷]]、[[汪大燮|汪大燮]]、[[熊希龄|熊希龄]]等23人
*秘书长：[[林長民|林長民]]
*政務部長：[[林長民|林長民]]
*党務部長：[[丁世峄|丁世峄]]

== 活動与终结 ==
三党合组进步党后，内部意见不能统一。比如民主党出身的[[孫洪伊|孫洪伊]]被任命为党務部副部長，但对梁啓超接近[[袁世凱|袁世凱]]失望，实际上并未加入進步党<ref>謝彬『民国政党史』1924年（中華書局版、54頁）。潘荣「孫洪伊」、161頁。</ref>。此外，一些旧共和党人不满原民主党骨干占据进步党的中枢，乃脱离进步党重组共和党（为同合并前的共和党相区別，一般称作「新共和党」）<ref>謝、55-56頁。</ref>。

進步党是支持袁世凱政府的政党，[[二次革命|二次革命]]後，袁世凯取得对全国的绝对控制，图谋解散国会，该党于是同留在国会的国民党温和派携手继续使国会得以运转。袁世凱见此情况，于1913年9月18日组建新的親袁世凯的御用政党[[公民黨_(1913年)|公民党]]（[[北京政府|北京政府]]总統府秘書長[[梁士詒|梁士詒]]任党魁。[[梁士詒|梁士詒]]是[[旧交通系|旧交通系]]的首领）。10月6日，袁世凯给梁士詒下指示，并派軍警包围国会議場逼迫国会选举袁世凯为大总统，袁世凯乃当选正式大总統<ref>謝、60頁。</ref>。

進步党对此形势抱有强烈的危机感。同年10月21日，国民党、進步党的有志之士结成了[[民憲党|民憲党]]<ref>謝、61頁。</ref>。11月4日，袁世凯以勾结叛逆为名剥夺了国民党籍国会議員438人的議員資格，国会因议员人数不足而无法继续开会，11月15日正式停会，而進步党党团近于瓦解<ref>謝、62頁。</ref>。

此後，梁啓超和進步党的主要幹部反对1915年（民国4年）12月[[袁世凱称帝|袁世凱称帝]]，并发动了[[護国战争|護国战争]]。

黎元洪任總統，國會恢復后，各政黨均標榜無黨主義，進步黨組成所謂“研究系”，繼續與國民黨系各政治組織在國會中對抗。
