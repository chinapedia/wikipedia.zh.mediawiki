'''额骨'''(Frontal bone)，组成[[颅骨|颅骨]]的29块骨头之一。位于前额处。它由可分为三部分：
额鳞：大而垂直，在前额。
眶部：环状或水平，对[[眼眶|眼眶]]顶部和[[鼻腔|鼻腔]]的形成至关重要。
[[鼻|鼻]]部：与[[鼻骨|鼻骨]]和[[颌骨|颌骨]]的[[额突|额突]]在[[鼻|鼻]]根部形成关节。
它前与[[筛骨|筛骨]]和[[鼻骨|鼻骨]]相连，后通过冠状缝与[[顶骨|顶骨]]相连。额骨内前下方有称为额窦的[[空腔|空腔]]。

== 胚胎学 ==
额骨被认定为衍生自[[神经嵴细胞|神经嵴细胞]]。

== 边缘 ==
额鳞的边缘厚，呈明显的锯齿状，其内表面上部为削面与[[顶骨|顶骨]]连接，以任一侧的外表面承受这些骨的侧压；边缘向下延伸至一个三角形粗糙面，与[[蝶骨|蝶骨]]之大翼成关节。[[眶板|眶板]]后缘窄且为锯齿状，与[[蝶骨|蝶骨]]之小翼成关节。

== 其他动物 ==
大多数[[脊椎动物|脊椎动物]]的额骨有一对而不是人类的一整块，一般地，位于头的上部，两眼之间，但对很多非哺乳动物它不构成[[鼻腔|鼻腔]]的一部分。在[[爬行动物|爬行动物]]、[[硬骨鱼|硬骨鱼]]和[[两栖动物|两栖动物]]的身体中，它和[[眼眶|眼眶]]被一两块额外的骨分开，但这一现象没有在[[哺乳动物|哺乳动物]]中发现。[[前额骨|前额骨]]和[[后额骨|后额骨]]共同构成眼眶上缘，可能倒向额骨的任一侧。
