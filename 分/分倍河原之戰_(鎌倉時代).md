'''分倍河原之戰'''（{{lang-ja|'''分倍河原の戦い'''}}）是[[鎌倉時代|鎌倉時代]]後期的[[元弘|元弘]]3年（1333年）5月15日至16日，[[北條泰家|北條泰家]]率領的[[鎌倉幕府|鎌倉幕府]]勢力與[[新田義貞|新田義貞]]率領的反幕府勢力在[[武藏國|武藏國]][[多摩川|多摩川]]河畔的[[分倍河原|分倍河原]]（現今[[東京都|東京都]][[府中市|府中市]]）進行的合戰。

== 背景 ==
[[元弘|元弘]]3年（1333年）5月8日，[[新田義貞|新田義貞]]為了打倒[[鎌倉幕府|鎌倉幕府]]而在[[上野國|上野國]][[生品神社|生品神社]]舉兵。這時的新田軍有義貞以下一族的[[大舘宗氏|大舘宗氏]]、[[堀口貞滿|堀口貞滿]]、[[岩松經家|岩松經家]]、[[里見義胤|里見義胤]]、[[脇屋義助|脇屋義助]]、[[江田行義|江田行義]]、[[桃井尚義|桃井尚義]]等總勢150騎。但是一直南行至[[利根川|利根川]]時，與[[越後國|越後國]]的新田黨（里美、鳥山、田中、大井田、羽川的各家）以及[[甲斐|甲斐]]、[[信濃|信濃]]的[[源氏|源氏]]一派合流，於是軍勢增加至7千騎。5月9日，越過利根川後與[[足利高氏|足利高氏]]（後來的足利尊氏）的嫡子千壽王（後來的[[足利義詮|足利義詮]]）由[[紀政綱|紀五左衛門]]陪伴而合流，因為有外様[[御家人|御家人]]最有力者足利高氏的嫡男加入，上野、[[下野|下野]]、[[上總|上總]]、[[常陸|常陸]]、[[武藏|武藏]]等國內對鎌倉幕府抱持不滿的[[武士|武士]]亦漸漸聚集，新田軍的兵力膨脹至20萬7千人。

之後新田軍沿著[[鎌倉街道|鎌倉街道]]南下並渡過[[入間川|入間川]]。迎撃[[櫻田貞國|櫻田貞國]]率領的鎌倉幕府軍，在5月11日[[小手指原之戰|小手指原之戰]]、5月12日[[久米川之戰|久米川之戰]]中逐漸撃破幕府軍。幕府軍為了防止武藏國最後的要害[[多摩川|多摩川]]被新田軍佔領，於是往[[分倍河原|分倍河原]]（現今[[東京都|東京都]][[府中市|府中市]]）方向撤退。

== 經過 ==
[[鎌倉幕府|鎌倉幕府]]在接到小手指原和久米川的戰報後，派遣[[北條高時|北條高時]]的弟弟[[北條泰家|北條泰家]]率領10萬軍勢迎撃新田軍，泰家領軍在[[分倍河原|分倍河原]]與櫻田貞國的軍勢合流。5月15日，經過2日休息的新田軍開始向分倍河原的幕府軍進行攻撃，但是得到援軍而士氣高昂的幕府軍把新田軍撃破。新田軍被迫撤退到堀金（[[狹山市|狹山市]]堀兼）附近。在敗走之際，[[武藏國|武藏國]]分寺（[[東京都|東京都]][[國分寺市|國分寺市]]）被燒燬。

翌日（5月16日）新田軍因為援軍[[三浦義勝|三浦義勝]]獻策，在天曉前急襲幕府軍。幕府軍敗走，在關戶（東京都[[多摩市|多摩市]]）遭到毀滅性的打撃。北條泰家因為家臣横溝八郎的奮戦而得以保留性命並向[[鎌倉|鎌倉]]方向逃走。關於形勢突然劇變的理由，有幕府軍因為初日的勝利而大意輕敵的説法；亦有[[足利尊氏|足利尊氏]]攻略[[六波羅探題|六波羅探題]]的情報到達關東後，幕府軍的增援部隊倒戈的説法。

== 影響 ==
在分倍河原之戰中，新田軍得到了決定性的勝利，幕府軍完全轉為守勢。之後新田軍漸漸得到其他援軍的加勢，變成60萬的大軍勢（『[[太平記|太平記]]』）。幕府軍在[[鎌倉|鎌倉]]堅守不出。新田軍持續攻向幕府的根據地鎌倉，之後強行突破[[稻村崎|稻村崎]]而突入幕府軍的背後並繼續攻向鎌倉。於是之後發生了倒幕運動最後的合戰（[[東勝寺合戰|東勝寺合戰]]）。

== 史跡 ==
在[[府中市|府中市]][[分梅町|分梅町]]的新田川分梅公園有分倍河原古戰場碑。而[[分倍河原站|分倍河原站]]前的[[迴旋處|迴旋處]]有[[新田義貞|新田義貞]]的像。新田義貞的面向南朝往鎌倉方向。
