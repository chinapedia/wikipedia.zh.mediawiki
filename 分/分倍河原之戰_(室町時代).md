'''分倍河原之戰'''（{{jp|j=分倍河原の戦い|hg=ぶばいがわらのたたかい|rm=Bubaigawaranotatakai|lead=yes}}）是在[[室町時代|室町時代]]後期的[[享德|享德]]4年（1455年）1月21日至22日於[[武藏國|武藏國]][[多摩川|多摩川]]河畔[[分倍河原|分倍河原]]（現今[[東京都|東京都]][[府中市|府中市]]）由[[足利成氏|足利成氏]]率領的[[鐮倉公方|鐮倉公方]]勢力和[[上杉顯房|上杉顯房]]率領的（總大將是在[[京都|京都]]的[[上杉房顯|上杉房顯]]）[[關東管領|關東管領]]勢力之間進行的合戰。

這次戰役成為與[[應仁之亂|應仁之亂]]並稱為[[室町時代|室町時代]]最大戰亂的[[享德之亂|享德之亂]]的序幕。

== 背景 ==
在[[永享之亂|永享之亂]]中滅亡的[[鐮倉公方|鐮倉公方]]在經過10年後，為了再興{{tsl|ja|鎌倉府|鐮倉府}}而展開行動。在[[京都|京都]]的[[室町幕府|室町幕府]]，[[管領|管領]][[細川勝元|細川勝元]]因為過去幕府和鐮倉公方之間的對立，對於這些行動有所懷疑，因此表示反對，而前管領[[畠山持國|畠山持國]]和[[越後國|越後國]][[守護|守護]][[上杉房定|上杉房定]]則表示贊成。最後，幕府同意再興。

被選為新鐮倉公方的是被幕府消滅的前公方[[足利持氏|足利持氏]]的遺兒永壽王丸，而被選為輔佐永壽王丸的新[[關東管領|關東管領]]則是前關東管領[[上杉憲實|上杉憲實]]的兒子[[上杉憲忠|憲忠]]（[[山內上杉家|山內上杉家]]）。已經出家的憲實考慮到此前的永享之亂中，是因自己與持氏的意見對立而起，因此永壽王丸會怨恨自己和自己的兒子，於是強迫憲忠辭退。不過在山內上杉家的[[家宰|家宰]][[長尾景仲|長尾景仲]]和[[扇谷上杉家|扇谷上杉家]]的家宰[[太田資清|太田資清]]強烈勸導之下，憲忠最終答應。

[[文安|文安]]4年（1447年），永壽王丸和憲忠進入[[鐮倉|鐮倉]]，憲忠被任命為關東管領。[[寶德|寶德]]元年（1449年），永壽王丸[[元服|元服]]，被任命為[[從五位下|從五位下]]{{tsl|ja|左馬頭}}，並拜領將軍[[足利義成|足利義成]]（後來的義政）的[[偏諱|偏諱]]，改名為[[足利成氏|足利成氏]]，正式就任第5代鐮倉公方。此後，成氏登用在永享之亂和[[結城合戰|結城合戰]]中為鐮倉公方家殉死的武將遺兒[[結城成朝|結城成朝]]、[[簗田持助_(室町時代)|簗田持助]]、[[里見義實|里見義實]]等人為側近，引起[[上杉氏|上杉氏]]和上杉家臣團的強烈反對。

寶德2年（1450年），持助奉成氏之命，{{tsl|ja|押領}}[[相模國|相模國]]{{tsl|ja|鎌倉郡|鐮倉郡}}[[長尾鄉|長尾鄉]]（現今[[橫濱市|橫濱市]][[榮區|榮區]][[長尾台|長尾台]]）（長尾鄉是上杉氏的第一重臣[[長尾氏|長尾氏]]的發祥地，此地的[[御靈宮|御靈宮]]是長尾氏{{tsl|ja|一門}}祭祀祖先的中心）。此事激起[[長尾景仲|長尾景仲]]等長尾氏一族的憤慨，於是與同情長尾氏的太田資清（景仲的女婿）一同強烈抗議，但是成氏側並不打算歸還領地。

4月20日，景仲和資清率領5百騎進入鐮倉並發動政變。不過成氏在事前就得到情報，於夜裡在[[小山持政|小山持政]]等人保護下逃離鐮倉，並在[[江之島|江之島]]堅守。21日，突入鐮倉的景仲等人為了追擊成氏而追往江之島，而[[小田持家|小田持家]]、[[宇都宮等綱|宇都宮等綱]]、[[千葉胤將|千葉胤將]]為了救援成氏，率軍趕到江之島，兩軍在[[由比濱|由比濱]]交戰。長尾太田軍惨敗，加上主君憲忠在不明情況下，為了救援成氏而命令{{tsl|ja|小幡氏}}等出陣，於是景仲和資清就逃進資清的主君[[上杉持朝|上杉持朝]]（前扇谷上杉家[[當主|當主]]）的[[糟谷館|糟谷館]]（現今[[神奈川縣|神奈川縣]][[伊勢原市|伊勢原市]]）。雖然憲忠完全沒有參與此事，不過襲撃者是長尾和太田的士兵，因此受到處罰（[[江之島合戰|江之島合戰]]）。

得知這次事態的幕府讓當時復歸管領職的畠山持國作出仲裁，持國下達對成氏有利的裁定。8月4日，成氏返回鐮倉。10月，憲忠恢復職務。在憲忠的求情下，景仲等人亦被赦免。此後，成氏側和憲忠側的[[武士_(日本)|武士]]經常因為押領對方的所領而對立。

對此，取替持國而再次就任管領的[[細川勝元|細川勝元]]提出削減[[鐮倉公方|鐮倉公方]]的權力。另一方面，憲忠的義父上杉持朝考慮到繼續對此放任不理的話，會對女婿憲忠的性命有危險，於是秘密與長尾景仲一同進入[[上杉氏|上杉氏]]的本國[[上野國|上野國]]，並準備討伐成氏。而掌握以上行動的成氏和周邊勢力，亦開始秘密準備對抗的策略。

== 經過 ==
[[享德|享德]]3年12月27日（1455年1月15日）晚上，身在[[鐮倉|鐮倉]]管領屋敷的[[上杉憲忠|憲忠]]收到[[足利成氏|成氏]]發出的緊急出仕命令（此時，[[長尾景仲|景仲]]因為年終將近，於是拜託同樣是[[家宰|家宰]]的義兄[[長尾實景|長尾實景]]留守，自身則前往[[長尾鄉|長尾鄉]]的[[御靈宮|御靈宮]]準備参詣），於是憲忠前往成氏的御所。

憲忠進入御所後，被[[結城成朝|結城成朝]]、[[里見義實|里見義實]]、[[武田信長|武田信長]]等軍勢包圍，憲忠在完全無法抵抗的情況下，被結城家臣[[多賀谷高經|多賀谷高經]]（後來的朝經）、[[多賀谷氏家|氏家]]兄弟殺死。同時，[[岩松持國|岩松持國]]率領的別働隊襲撃管領屋敷，殺死實景等[[上杉氏|上杉氏]]家臣。

聞知憲忠被暗殺的景仲返回鐮倉，直接前往管領屋敷放火，並把憲忠的正室（[[上杉持朝|上杉持朝]]的女兒）等生還者送往持朝的[[糟谷館|糟谷館]]避難。到達糟谷館的景仲與持朝和持朝的嫡男[[上杉顯房|顯房]]（扇谷上杉家當主）、[[上杉氏#犬懸上杉家|犬懸上杉家]]的[[上杉憲秋|憲秋]]、[[上杉氏#千秋上杉家（小山田上杉家）|小山田上杉家]]的[[上杉藤朝|藤朝]]等上杉一族要人協議，決定要迎身在[[京都|京都]]的[[上杉房顯|上杉房顯]]（憲忠的弟弟）為次任[[關東管領|關東管領]]和討伐成氏。之後，景仲直接進入領國上野聚集士兵，並派遣使者向[[越後國|越後國]][[守護|守護]][[上杉房定|上杉房定]]請求援軍，還直接派遣嫡男[[長尾景信|景信]]前往京都，把這次事件報告幕府，以及要迎接房顯一事。

翌年（1456年）1月5日，成氏為了攻略[[上杉氏|上杉氏]]的本國上野，從鐮倉出發，並進入[[武藏國|武藏國]]府中的{{tsl|ja|高安寺}}。另一方面，得知此事的持朝為了奪取鐮倉而出陣。翌日，持朝在[[相模國|相模國]][[島河原|島河原]]（現今[[神奈川縣|神奈川縣]][[平塚市|平塚市]]）被留守在鐮倉的武田信長迎撃而敗退。得知此事的長尾景仲則直接率領上野和武藏的士兵向府中出撃，上杉一族亦為了與其合流而出陣。

1月21日，結集在府中近郊的上杉軍2千騎攻向高安寺，不過成氏軍率領5百騎向[[分倍河原|分倍河原]]出擊，在成氏軍的突撃下，上杉軍陷入混亂，先鋒上杉憲秋在[[立河原|立河原]]（現今[[東京都|東京都]][[立川市|立川市]]）被敵人直接攻擊，受到致命傷的憲秋在千鈞一髮間被家臣救走，在{{tsl|ja|高幡不動}}（一説指是[[荏原郡|荏原郡]]池上）自殺。得知憲秋自殺的上杉顯房等人被激怒，翌日，顯房率領新力軍5百騎向分倍河原進軍。

在戰鬥開始時，上杉軍的先鋒[[大石房重|大石房重]]等人戰死，不過因為成氏軍亦受到很大傷害，雙方不分勝負，結城成朝等軍勢襲擊上杉軍，因此上杉軍開始後退，更因為前往相模的退路被切斷，上杉軍開始往東面潰走。成朝率領成氏軍繼續追擊，顯房和藤朝在武藏夜瀨（[[三鷹市|三鷹市]]）被包圍。24日，顯房和藤朝自殺。成功逃走的長尾景仲則率領殘軍，逃到[[常陸國|常陸國]][[小栗城|小栗城]]（現今[[茨城縣|茨城縣]][[筑西市|筑西市]]）。

== 戰後 ==
此後，[[足利成氏|足利成氏]]乘勢攻略上杉側在[[武藏國|武藏國]]的據點，得知[[長尾景仲|景仲]]在小栗城後，在3月3日進入[[下總國|下總國]][[古河城|古河城]]，得到[[那須資持|那須資持]]、[[筑波潤朝|筑波潤朝]]、[[小田朝久|小田朝久]]等人支援後，開始攻撃小栗城。途中，朝久突然死去。閏4月，成氏軍攻陷小栗城，令景仲敗走。另一方面，[[上杉顯房|顯房]]的兒子[[上杉政真|政真]]被立為在[[扇谷上杉家|扇谷上杉家]][[當主|當主]]，而[[太田資清|太田資清]]則負起顯房戰死的責任而[[出家|出家]]，資清的兒子資長（後來的[[太田道灌|太田道灌]]）就任[[家宰|家宰]]。

而在[[京都|京都]]方面，[[室町幕府|室町幕府]]在年初就收到此戰的消息，因為亦有意見同情成氏，於是議論變得相當複雜。3月，因為[[管領|管領]][[細川勝元|細川勝元]]的意見，幕府向以[[上杉氏|上杉氏]]一族為首和周邊的[[守護|守護]][[今川範忠|今川範忠]]（[[駿河國|駿河國]]）、[[小笠原光康|小笠原光康]]（[[信濃國|信濃國]]）、[[宇都宮等綱|宇都宮等綱]]（[[下野國|下野國]]）、[[千葉胤直|千葉胤直]]（[[下總國|下總國]]前守護）等人在下達成氏討伐令。{{tsl|ja|宇都宮氏}}和[[千葉氏|千葉氏]]在[[江之島合戰|江之島合戰]]中屬於成氏側，宇都宮氏的父親[[宇都宮持綱|持綱]]（前代當主）被成氏的父親[[足利持氏|持氏]]所殺，因此一直等待復仇的機會，而千葉氏則因為江之島合戰時期的當主兼守護[[千葉胤將|胤將]]突然死去，與持氏對立的前代當主[[千葉胤直|胤直]]復歸，於是兩家與[[今川氏|今川氏]]、{{tsl|ja|小笠原氏}}一同加入成氏討伐軍。

身在京都的今川範忠受[[後花園天皇|後花園天皇]]賜予{{tsl|ja|錦御旗}}後，直接返回本國並帶兵出陣。6月16日，擊破[[武田信長|武田信長]]等人，並佔領[[鐮倉|鐮倉]]。而[[越後國|越後國]]守護[[上杉房定|上杉房定]]亦秘密從京都進入越後，並擁立新的[[關東管領|關東管領]][[上杉房顯|上杉房顯]]，於[[上野國|上野國]][[三宮原|三宮原]]（現今[[群馬縣|群馬縣]][[吉岡町|吉岡町]]）與[[岩松持國|岩松持國]]等在上野的成氏派交戰。另一方面，宇都宮等綱受成氏直接攻撃，因為[[宇都宮城|宇都宮城]]被攻陷而被重臣們追放；千葉胤直亦被擁立反對討伐軍的親成氏派的[[馬加康胤|馬加康胤]]的重臣[[原胤房|原胤房]]攻滅。

成氏見到這個狀況後，判斷自己難以返回鐮倉，於是進入{{tsl|ja|古河城}}，並宣佈古河城成為新的御所，於是被稱為[[古河公方|古河公方]]。此後，成氏軍和上杉軍在[[關東地方|關東地方]]各地繼續交戰，雙方以[[利根川|利根川]]（當時的利根川與現在的不同，是流向南方的[[江戶灣|江戶灣]]）為界，東側被成氏軍佔領，而西側則被上杉側佔領。7月，朝廷以關東的戰亂為理由，把[[元號|元號]]改為[[康正|康正]]，於是利根川西側使用「康正」的元號，但是東側的成氏陣營不跟隨追討自己不當的朝廷和幕府的改元，依然繼續使用「享德」的元號（之後朝廷5次改元，但是成氏陣營在之後23年間仍然使用「享德」的元號，期間，[[京都|京都]]爆發[[應仁之亂|應仁之亂]]時，依然保持敵對的中立）。

此後，關東地方在之後的28年都陷入被稱為「[[享德之亂|享德之亂]]」的狀態。
