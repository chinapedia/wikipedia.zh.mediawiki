'''分水工'''是從一條[[水管|水管]]或是[[灌溉|灌溉]]用的供水[[渠道|渠道]]將水分配至兩條以上的水管或是渠道，並且設有[[閘門|閘門]]或是[[閘板|閘板]]，讓其具有針對性供水以及調節渠道水位的功能的一種[[組織結構|組織結構]]。<ref name="農田水利會入口網 詞彙介紹 分水工">{{cite web |url=http://doie.coa.gov.tw/vocabulary/vocabulary-detail.asp?id=180044 |title=農田水利會入口網 詞彙介紹 分水工 | accessdate=2015-01-17}}</ref><ref>{{cite web |url=http://www.ilia.gov.tw/express/%E6%B0%B4%E5%B7%A5%E6%A7%8B%E9%80%A0%E7%89%A9%E4%BB%A3%E8%99%9F.htm |title=水工構造物代號 | accessdate=2015-01-19}}</ref>

==功能==
當分水工分水處不需經過測量水量後再進行供水時，一般採用比率分水，可隨當時渠道之水流量變化來進行針對性的分水。當分水工須先測量水量再進行供水時，可以利用[[閘門|閘門]]或閘板在分水工進行各渠道的水量分配。<ref name="農田水利會入口網 詞彙介紹 分水工"/>

==實例==
[[林內鄉|林內]]分水工是一座直徑100公尺、面積8660[[平方公尺|平方公尺]]、深度6.8公尺的小型[[水庫|水庫]]。採用八角形分水工。最高蓄水量45000[[立方公尺|立方公尺]]，主要是提供[[麥寮鄉|麥寮]]地區的工業用水。

== 分類 ==

依水理性質可將分水工分類如下：<ref name="農田水利會入口網 詞彙介紹 分水工"/>
*潛流型-孔口式
*[[水門|水門]]式
*[[涵管|涵管]]式
*短嘴式
*[[溢流|溢流]]型-明槽式
*[[堰|堰]]式
*[[比率|比率]]式
*控制閘式
*混合型-分[[水井|水井]]式
*[[虹吸|虹吸]]分水式
*八角形分水式
