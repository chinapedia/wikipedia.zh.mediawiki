'''分錄'''（{{lang-en|'''Entry'''}}）是指[[會計|會計]]中的一項程序，是將原始憑證（[[收據|收據]]或[[發票|發票]]）編製記帳憑證（指傳票），再登錄到會計日誌（日記簿，也稱為又稱序時帳簿、分錄簿和原始紀錄簿）中。分錄內容會分為借方或貸方，也會依記帳憑證決定其[[會計科目|會計科目]]，借方的總金額需等於貸方的總金額，且金額的數目應每3位畫一撇，如2,000,000元等。

== 分錄簡介 ==
分錄在會計的處理流程裡通常為第一個流程，若有傳票制度則為第二個流程。分錄中常見的資料有：分錄編號、批号、種類（經常性或非經常性）、金額、名稱、回轉分錄（auto-reversing）、会计期间及說明。貨方的會計科目和金額會內縮。一般而言會計軟體對於說明文字的長度有嚴格的限制，最長30字的限制是很常見的。因此特殊交易的分錄資料可以一行顯示完成。在紀錄分錄時，一個會計科目只能出現一次，且在交易分錄裡，同屬性的會計科目應合併計算。例如：水費、電費、瓦斯費應合併為水電瓦斯費計算。
<ref>[http://www.zwbk.org/zh-tw/Lemma_Show/109391.aspx 會計分錄─中文百科在線]</ref>

== 記錄分錄的方法 ==

記錄分錄的過程：
假設100年6月20日購買一個辦公用桌子花費了950元。

1.經濟業務發生後，根據審核後的原始憑證編制記賬憑證，按業務發生的程序在日記簿中進行序時登記。 先知道交易內容大概的模式：花費了950元買了一個"辦公用"桌子，顯然就是要拿來辦公上使用，並非無目的就交易。

2.寫分錄至日記簿：如此交易分錄為

   辦公設備  950
       現金     950

這個分錄意思是指現金少了950元的原因，是因為拿去買了辦公用的設備桌子，將日記簿中所記載的會計事項，過入各種分類賬的有關科目。

其它程序如有必要再做，假使只是為了讓自己記帳而不是要給予企業上來看，那可以只簡單的寫分錄就好。而如果紀錄至日記簿的形式話如表格：

3.分錄過帳至分類帳。

4.期末時如分錄有錯誤，做調整之動作。根據總分類賬各科目編制調整前試算平衡表。

5.結帳、編表。為了正確計算損益，真實反映報告或財務狀況，企業按權責發生制要求，對一些應計、預計等賬項進行調整，編制調整分錄據以過入總分類賬各有關科目。

6.編制調整後試算平衡表，以檢查調整分錄及其過賬是否正確。

7.根據調整後的試算平衡表編制財務報表，財務報表包括[[利潤表|利潤表]]、[[資產負債表|資產負債表]]、[[綜合損益表|綜合損益表]]、[[權益變動表|權益變動表]]和[[現金流量表|現金流量表]]。
<ref>[http://www.memdb.com/MemAcc/personal1.htm 資產負債表製作方式]</ref>

8.編制財務報表後，在日記簿中作結賬分錄，結算所有收入及費用科目，結算本期淨收益。 

9.編制結賬後試算平衡表，檢查總分類賬各科目餘額是否正確。
<ref>[http://www.zwbk.org/zh-tw/Lemma_Show/69954.aspx 會計程序─中文百科在線]</ref>

10.檢查各流程中會計科目使用、借貸方金額有無計算錯誤。如果有的話，應回頭從日記簿開始重新檢查。

== 分錄種類 ==
1.簡單分錄(單項式分錄)：指借方和貸方使用的會計科目皆只有1個的分錄。

2.複合分錄(多項式分錄)：指借方和貸方有多個會計科目。

3.現金分錄：交易完全以現金作為收付。

4.轉帳分錄：交易完全不涉及現金作為收付。

5.混合分錄：部份以現金作收付，部分非以現金作收付。

6.開業分錄：業主剛開始營業的第一筆交易。

7.一般分錄：企業一般日常交易之分錄，交易之分錄過到日記簿。

8.更正分錄：在過帳後發現錯誤而須作更正之分錄。

9.調整分錄：期末對平時會計事項加以補充或修正之分錄。

10.結帳分錄：年底結帳時將調整後之本期收益及費損之帳戶結轉至本期損益帳戶的分錄。

11.回轉分錄；期初開帳時，將上期期末部份調整分錄予以迴轉之分錄。

== 交易相關分錄 ==
=== 交易日分錄 ===
#[[用品盤存|用品盤存]]在交易日的分錄：<BR/>權責發生基礎(記實轉虛法)：借方：用品盤存、貸方：現金<BR/>聯合基礎(記虛轉實法)：借方：文具用品、貸方：現金
#[[預收收益|預收收益]]在交易日的分錄：<BR/>權責發生基礎(記實轉虛法)：借方：現金、貸方：預收收益(會計科目)<BR/>聯合基礎(記虛轉實法)：借方：現金、貸方：收益(會計科目)
#[[進貨|進貨]]在交易日的分錄：<BR/>現金支付進貨費用分錄：借方：進貨、貸方：現金<BR/>先賒帳，之後再以現金付款分錄：借方：進貨、貸方：應付帳款<BR/>先賒帳，之後以票據支付分錄：借方：進貨、貸方：應付票據
*其它會計科目如運輸設備、電腦設備等，皆只要將進貨改成對應的會計科目即可。

=== 期末調整分錄 ===
以下為期末可能會出現的調整相關分錄：
#[[損益表|損益表]]法(又稱加提法)：根據[[收益|收益]]、[[費損|費損]]之配合原則，不考慮調整前[[備抵呆帳|備抵呆帳]]的餘額。期末調整分錄：<BR/>借方：[[呆帳|呆帳]](損失)、貸方：備抵呆帳─應收帳款。
#[[折舊|折舊]]在期末的調整分錄：<BR/>借方：折舊、貸方：[[累計折舊|累計折舊]]─不動產、廠房及設備(會計科目)。
#[[攤銷|攤銷]]在期末的調整分錄：<BR/>借方：各項攤銷、貸方：累計攤銷─[[無形資產|無形資產]]
#[[用品盤存|用品盤存]]在期末的調整分錄：<BR/>權責發生基礎(記實轉虛法)：借方：文具用品 (已耗用金額)、貸方：用品盤存 (已耗用金額)<BR/>聯合基礎(記虛轉實法)：借方：用品盤存 (未耗用金額)、貸方：文具用品 (未耗用金額)
#[[預收收益|預收收益]]在期末的調整分錄：<BR/>權責發生基礎(記實轉虛法)：借方：預收收益 (已實現部份)、貸方：收益 (已實現部份)<BR/>聯合基礎(記虛轉實法)：借方：收益 (未實現部份)、貸方：預收收益 (未實現部份)
