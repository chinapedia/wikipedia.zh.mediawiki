'''湘湖站'''位于[[杭州|杭州]][[萧山区|萧山区]]风情大道与湘西路相交路口，为[[杭州地铁1号线|杭州地铁1号线]]的地下车站及起始站<ref>[http://zzhz.zjol.com.cn/05zzhz/system/2007/04/09/008318161.shtml 杭州地铁湘湖站预计今年7月动工]</ref>，于2012年11月24日启用。

== 车站结构 ==
湘湖站为地下岛式站台和起讫站，主体结构建筑面积约36082.5平方米，车站结构总长约934. 5m, 标准段宽20.5m,站台宽度12m，最大埋深约 17.7m。<ref>{{cite web |title=杭州地铁事故：[转]杭州地铁湘湖站坍塌事故案一审宣判 - 奇趣后备箱 |url=https://qqhbx.com/xiuxianbagua/126980.html}}</ref>站前设单渡线、站后设有一股折返线和出入库线。回库列车将会从湘湖站沿着风情大道地下继续往南，再陆续转向西北和东北穿越王家湖山和黄泥山的隧道，向东进入[[湘湖停车场|湘湖停车场]]，全长约1250米。<ref>{{cite web |title=探营1号线起点湘湖站：杭州地铁的“大本营”，讲究-浙江新闻-浙江在线 |url=http://zjnews.zjol.com.cn/05zjnews/system/2012/07/18/018664141.shtml |website=浙江在线}}</ref>

== 历史 ==
*2008年4月，车站开工。
*2008年11月15日15点左右，施工中的本站“北2基坑”工地发生大规模[[杭州地铁工地坍塌事故|坍塌事故]]，共造成21人死亡。
*2010年11月25日12时许，施工中的本站基坑内土体滑落，砸伤两名作业中的工人，其中一人抢救无效死亡。<ref>{{Cite news|url = http://zjnews.zjol.com.cn/05zjnews/system/2010/11/25/017116444.shtml|title = 杭州地铁1号线湘湖站基坑发生事故 一死一伤|last = |first = |date = 2010-11-25|work = |accessdate = }}</ref>
*2012年11月24日，车站与1号线一起开放。

== 车站周边 ==

该站周边1km内缺少商业设施，以学校、居民区和村落为主，但有完善的公交接驳系统。站点周边有停车场、公交总站等，方便换乘。
