'''雅昵斯比'''（或阿尼亞斯貝，{{lang-fr|'''agnès b.'''}}{{Audio|Fr-Agnès_b..ogg|法语发音}}）是[[法國|法國]][[時裝|時裝]]品牌，1975年在[[巴黎|巴黎]]成立，分店分佈全球，創辦人為法國[[時裝設計師|時裝設計師]]{{lang|fr|Agnès Troublé}}（1941年在[[凡爾賽|凡爾賽]]出生），她喜用自己名字作為品牌名稱，業務包括[[時裝|時裝]]及[[藝術|藝術]]推廣。

== 業務擴展 ==
{{lang|fr|agnès}}繼早年設計女裝服飾後，於1981年開始設計男裝服飾，到了1983年，她在[[紐約市|紐約市]][[蘇豪區|蘇豪區]]開設首間海外{{lang|fr|agnès b.}}分店。她對城市的熱愛使得她對美式的罪案電影感到興趣，曾為[[法國|法國]]的電影進行配音工作，間接影響到分店的裝修風格，店內會張貼不同年代的大型電影海報，此舉動引起了電影導演Harmony Korine的注意，並且於1999年與{{lang|fr|agnès}}進行藝術性合作。 

{{lang|fr|agnès b.}}於1987年推出「Le B Perfume」香水品牌系列，包含護膚及[[化妝|化妝]]用品。後來，她的設計範圍擴展到孕婦服裝、鞋子及手袋產品。同時，{{lang|fr|agnès b.}}為[[精工錶|精工錶]]設計了一系列手錶及眼鏡產品，並與化妝品牌[[歐萊雅|歐萊雅]]合作。

{{lang|fr|agnès b.}}的分店據點有[[倫敦|倫敦]]、[[巴黎|巴黎]]、[[紐約|紐約]]、[[柏林|柏林]]、[[芝加哥|芝加哥]]、[[比華利山|比華利山]]、[[阿姆斯特丹|阿姆斯特丹]]、[[東京|東京]]、[[大阪|大阪]]、[[香港|香港]]、[[澳門|澳門]]、[[上海|上海]]、[[新加坡|新加坡]]、[[台灣|台灣]]、[[杜拜|杜拜]]等地，而據品牌設計師兼創辦人Agnes Trouble表示，今後將在[[中國內地|中國內地]]例如[[北京|北京]]、[[上海|上海]]等內地城市開設更多分店。

== 藝術推廣 ==
{{lang|fr|agnès b.}} 熱心於[[藝術|藝術]]推廣，曾[[贊助|贊助]]多個藝術[[展覽|展覽]]及演出。[[香港藝術中心|香港藝術中心]]的{{lang|fr|agnès b.}}[[電影院|電影院]]，亦為其[[贊助|贊助]]項目之一。全球共有12間分公司的國際公司，另於[[巴黎|巴黎]]設有1間[[工作室|工作室]]。

== 爭議 ==
2012年3月31日，有網民發現將軍澳新開幕的商場[[PopCorn|PopCorn]]內的agnès b. Cafe L.P.G.及DÉLICES，其餐牌上只有[[英文|英文]]及[[簡體字|簡體字]]，而[[食物|食物]]的中文名稱均為中國大陸用語，例如[[牛油|牛油]]稱為[[黃油|黃油]]、[[朱古力|朱古力]]稱為[[巧克力|巧克力]]，有網民認為有關安排是歧視[[香港人|香港人]]，在網上發起[[抵制|抵制]]，有團體更發起到餐廳抗議。其後，店方在[[Facebook|Facebook]]發出道歉啟事，表示無意侮辱或歧視香港人，承諾盡快更換餐牌。<ref>[http://news.sina.com.hk/news/20120403/-1-2625393/1.html 明報：餐牌只供英文簡體 agnes b.速道歉 (2012年4月3日)] {{webarchive|url=https://web.archive.org/web/20130117054311/http://news.sina.com.hk/news/20120403/-1-2625393/1.html |date=2013年1月17日 }}</ref>同時，在《[[香港經濟日報|香港經濟日報]]》表示全綫12間Cafe的餐牌由2008年至今只有英文及簡體字，直至2012年3月時才有人投訴。發言人表示，不清楚當初為何公司有如此政策，但是他們將會更換所有餐牌，掛牆的將會使用[[繁體字|繁體字]]及英文，餐牌則更改為繁、簡體及英文三文並重。
