'''雅典奥林匹克体育场'''（{{lang-el|Ολυμπιακό Στάδιο Αθήνας}}），是[[希臘|希臘]][[雅典奥林匹克体育中心|雅典奥林匹克体育中心]]的组成部分，为[[2004年夏季奥林匹克运动会|2004年夏季奥林匹克运动会]]主体育场。这座体育场也被称为'''斯皮里宗·路易斯体育场'''，得名于[[1896年夏季奥林匹克运动会|首届现代奥林匹克运动会]][[马拉松|马拉松]]比赛的[[金牌|金牌]]得主、[[希腊人|希腊人]][[斯皮里宗·路易斯|斯皮里宗·路易斯]]。<ref name="xinhua">{{cite web|url=http://news.xinhuanet.com/sports/2004-08/12/content_1766770.htm |title=聚焦雅典奥运场馆 |publisher=[[新华网|新华网]] |date=2004年8月12日 |accessdate=2008年11月24日 |language=zh }}</ref>

== 概况 ==
雅典奥林匹克体育场位于[[希腊|希腊]][[雅典|雅典]]北郊的马鲁西<ref>{{cite web |url=http://beijing.titan24.com/08-09-13/114662.html |title=雅典奥林匹克体育场--奥运后闲置 如今为AEK主场 |publisher=体坛网 |date=2008年9月13日 |accessdate=2008年11月24日 |language=zh }}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，是一座多功能综合体育场。体育场内设有105米×68米的标准[[足球场|足球场]]，9道400米环形[[跑道_(体育)|跑道]]，以及各种[[田赛|田赛]]设施。<ref name="main">{{cite web|url=http://www.oaka.com.gr/article_detail.asp?e_cat_serial=001003001001&e_cat_id=132&e_article_id=142 |title=Main Olympic Stadium |publisher=雅典奥林匹克体育中心 |date= |accessdate=2008年11月24日 |language=en |deadurl=yes |archiveurl=https://web.archive.org/web/20110929021349/http://www.oaka.com.gr/article_detail.asp?e_cat_serial=001003001001&e_cat_id=132&e_article_id=142 |archivedate=2011年9月29日 }}</ref>目前体育场有69618个观众席位、480个[[媒体|媒体]]席位，以及10多个贵宾包厢。<ref name="main"/>

雅典奥林匹克体育场设计于1979年，并于1980年动工建设。1982年9月8日，时任希腊总统[[康斯坦丁·卡拉曼利斯|康斯坦丁·卡拉曼利斯]]宣布体育场落成。这座体育场在1982年第13届欧洲田径锦标赛时首次投入使用。<ref name="HistoricApproach">{{cite web |url=http://www.oaka.com.gr/article_detail.asp?e_cat_serial=001002001&e_cat_id=128&e_article_id=194 |title=Historic Approach |publisher=雅典奥林匹克体育中心 |date= |accessdate=2008年11月24日 |language=en |deadurl=yes |archiveurl=https://web.archive.org/web/20140427083003/http://www.oaka.com.gr/article_detail.asp?e_cat_serial=001002001&e_cat_id=128&e_article_id=194 |archivedate=2014年4月27日 }}</ref>雅典成功申办2004年夏季奥运会之后，决定将奥林匹克体育场翻新后作为那届奥运会的主体育场。雅典奥组委整体工作进展缓慢，奥林匹克体育场翻新工程直到2000年才开始。体育场在2004年7月30日正式开放<ref name="aig">{{cite web|url=http://www.athensinfoguide.com/olympicvenues.htm |title=2004 Olympic Summer Games venues in Athens |publisher=Athens Info Guide |date= |accessdate=2008年11月24日 |language=en }}</ref>，但所有的工程实际在2004年8月5日才正式完工<ref name="southcn">{{cite web|url=http://www.southcn.com/sports/2004/pics/200408061343.htm |title=雅典奥运会主体育场完工 宏伟外观亮相世界 |publisher=[[南方网|南方网]] |date=2004年8月6日 |accessdate=2008年11月24日 |language=zh }}</ref>，离雅典奥运会开幕只剩8天时间。翻新工程采用了[[西班牙|西班牙]][[建筑师|建筑师]][[圣地亚哥·卡拉特拉瓦|圣地亚哥·卡拉特拉瓦]]的设计方案，其亮点是精巧的顶棚，面积达2.5万平方米，重达1.87万吨，覆盖体育场的绝大部分座位。<ref name="xinhua"/>

== 举办赛事 ==
雅典奥林匹克体育场作为2004年夏季奥运会的主体育场，承办了该届赛事中的大部分[[田径|田径]]比赛和[[足球|足球]]决赛。<ref name="aig"/>此外，2004年8月13日的[[开幕式|开幕式]]和8月29日的[[闭幕式|闭幕式]]也都在这座体育场内举行。<ref name="aig"/>

奥运会之外，雅典奥林匹克体育场在平时经常成为足球比赛的举办场地。[[希腊国家足球队|希腊国家足球队]]以及[[奥林匹亚科斯足球俱乐部|奥林匹亚科斯]]、[[帕纳辛奈科斯足球俱乐部|帕纳辛奈科斯]]和[[雅典AEK足球俱乐部|雅典AEK]]这三家当地主要[[足球俱乐部|足球俱乐部]]都曾在不同的赛事当中使用雅典奥林匹克体育场作为主场。其中，帕纳辛奈科斯在2005-2006和2006-2007两个赛季将奥林匹克体育场作为其所有比赛的主场，2007-2008赛季为庆祝球队建成百年又迁回原来的[[尼科莱季斯球场|尼科莱季斯球场]]，而自2008-2009赛季起再度使用奥林匹克体育场，并预计将在2010年迁往[[沃塔尼科斯体育场|沃塔尼科斯体育场]]（建设中）。而奥林匹克体育场观众数最多的纪录则由1982年11月3日奥林匹亚科斯对[[汉堡体育俱乐部|汉堡]]的主场比赛所创造，这场[[欧洲冠军杯|欧洲冠军杯]]比赛共吸引了75263名球迷到现场观看。<ref>{{cite web|url=http://www.stadia.gr/oaka/oaka-f.html |title=Athens Olympic Stadium "Spyros Louis" |publisher=stadia.gr |date= |accessdate=2008年11月25日 |language=en }}</ref><ref name="rsssf198283">{{cite web|url=http://www.rsssf.com/ec/ec198283det.html#cc |title=European Champions' Cup 1982-83 - Details |publisher=rsssf.com |date= |accessdate=2008年11月25日 |language=en }}</ref>

雅典奥林匹克体育场被[[欧洲足联|欧洲足联]]评定为[[欧洲足联五星级足球场|五星级足球场]]，曾多次举办欧洲足联各项赛事的决赛。[[1983年欧洲冠军杯决赛|1983年欧洲冠军杯决赛]]，[[汉堡体育俱乐部|汉堡]]在这里1比0战胜[[尤文图斯足球俱乐部|尤文图斯]]，首次夺冠。<ref name="rsssf198283"/>[[1987年欧洲优胜者杯决赛|1987年欧洲优胜者杯决赛]]，[[阿贾克斯足球俱乐部|阿贾克斯]]在这里1比0击败[[莱比锡火车头足球俱乐部|莱比锡火车头]]。<ref name="rsssf198687">{{cite web|url=http://www.rsssf.com/ec/ec198687.html#cwc |title=Cup Winners' Cup 1986-87 |publisher=rsssf.com |date= |accessdate=2008年11月25日 |language=en }}</ref>欧洲冠军杯改制为欧洲冠军联赛之后，有两次决赛在雅典奥林匹克体育场举行，分别是[[1994年欧洲冠军联赛决赛|1994年]]和[[2007年欧洲冠军联赛决赛|2007年]]，两次的胜者凑巧都是[[米兰足球俱乐部|AC米兰]]，对手分别是[[巴塞罗纳足球俱乐部|巴塞罗纳]]<ref name="rsssf199394">{{cite web|url=http://www.rsssf.com/ec/ec199394.html |title=European Competitions 1993-94 |publisher=rsssf.com |date= |accessdate=2008年11月25日 |language=en }}</ref>和[[利物浦足球俱乐部|利物浦]]<ref>{{cite web|url=http://www.uefa.com/newsfiles/ucl/2006/300099_fr.pdf |title=Full Time Report |publisher=[[欧洲足联|欧洲足联]] |date=2007年5月23日 |accessdate=2008年11月25日 |format=PDF |language=en }}</ref>。

此外，雅典奥林匹克体育场还曾举行过1982年欧洲田径锦标赛、1997年[[世界田径锦标赛|世界田径锦标赛]]等重大体育赛事。
