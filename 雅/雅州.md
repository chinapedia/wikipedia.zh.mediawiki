'''雅州'''，[[中国|中国]][[隋朝|隋朝]]时设置的[[州|州]]。

== 隋代 ==
[[仁寿_(隋朝)|仁寿]]四年（604年）置，[[治所|治所]]在[[蒙山县|蒙山县]]（后改[[严道县|严道县]]，今[[四川省|四川省]][[雅安市|雅安市]]）。[[大业_(年号)|大业]]三年（607年）改为[[臨邛郡|临邛郡]]。
== 唐代<ref>《舊唐書》卷四十一　志第二十一／地理四／十道郡國　四／劍南道／雅州下都督府</ref> ==
* [[唐朝|唐朝]][[武德|武德]]元年（618年）复为雅州，領[[嚴道縣|嚴道縣]]、[[名山縣|名山縣]]、[[盧山縣|盧山縣]]、[[依政縣|依政縣]]、[[臨邛縣|臨邛縣]]、[[蒲江縣|蒲江縣]]、[[臨溪縣|臨溪縣]]、[[蒙陽縣|蒙陽縣]]、[[漢源縣|漢源縣]]、[[火井縣|火井縣]]、[[長松縣|長松縣]]、[[靈關縣|靈關縣]]、[[楊啟縣|楊啟縣]]、[[嘉良縣|嘉良縣]]、[[大利縣|大利縣]]、[[陽山縣|陽山縣]]十六縣。同年割[[依政縣|依政縣]]、[[臨邛縣|臨邛縣]]、[[蒲江縣|蒲江縣]]、[[臨溪縣|臨溪縣]]、[[火井縣|火井縣]]五縣置[[邛州|邛州]]；[[漢源縣|漢源縣]]、[[陽山縣|陽山縣]]二縣置[[登州|登州]]，領縣九。
* [[武德|武德]]二年（619年），置[[榮經縣|榮經縣]]，領縣十。
* [[武德|武德]]六年（623年），省[[嘉良縣|嘉良縣]]、[[楊啟縣|楊啟縣]]、[[大利縣|大利縣]]、[[靈關縣|靈關縣]]、[[蒙陽縣|蒙陽縣]]、[[長松縣|長松縣]]六縣，領縣四。
* [[武德|武德]]九年（626年），廢[[登州|登州]]，還以[[漢源縣|漢源縣]]、[[陽山縣|陽山縣]]來屬，領縣六。
* [[貞觀|貞觀]]二年（628年），又以[[漢源縣|漢源縣]]、[[陽山縣|陽山縣]]屬[[巂州|巂州]]，領縣四。
* [[貞觀|貞觀]]八年（634年），又置[[百丈縣|百丈縣]]，領縣五。
* [[永徽|永徽]]五年（654年），又以[[巂州|巂州]][[漢源縣|漢源縣]]來屬，領縣六。
* [[儀鳳|儀鳳]]四年（679年），置[[飛越縣|飛越縣]]、[[大渡縣|大渡縣]]二縣，領縣八。
* [[大足|大足]]元年（701年），又割[[漢源縣|漢源縣]]、[[飛越縣|飛越縣]]二縣置[[黎州|黎州]]，領縣六。
* [[神龍|神龍]]三年（707年），廢[[黎州|黎州]]，[[漢源縣|漢源縣]]、[[飛越縣|飛越縣]]還屬雅州，領縣八。
* [[開元|開元]]三年（715年），又割[[漢源縣|漢源縣]]、[[飛越縣|飛越縣]]二縣置黎州，又置都督府，領縣六。
* [[天寶|天寶]]元年（742年），改為[[盧山郡|盧山郡]]。
* [[乾元|乾元]]元年，復為雅州。

都督羈縻一十九州並生羌、生獠羈縻州
* [[生羌羈縻州|生羌羈縻州]]、[[生獠羈縻州|生獠羈縻州]]、[[嘉梁州|嘉梁州]]、[[東石孔州|東石孔州]]、[[西石孔州|西石孔州]]、[[林波州|林波州]]、[[涉邛州|涉邛州]]、[[汶東州|汶東州]]、[[金林州|金林州]]、[[費林州|費林州]]、[[徐渠州|徐渠州]]、[[會野州|會野州]]、[[雉州|雉州]]、[[中川州|中川州]]、[[鉗矢州|鉗矢州]]、[[強雞州|強雞州]]、[[長臂州|長臂州]]、[[楊常州|楊常州]]、[[林燒州|林燒州]]、[[當仁州|當仁州]]、[[當馬州|當馬州]]

== 宋代<ref>《宋史》卷八十九　志第四十二／地理五／成都府路／雅州</ref> ==
[[宋朝|宋朝]]时属[[成都府路|成都府路]]，[[盧山郡|盧山郡]]。

* 領縣五：[[嚴道縣|嚴道縣]]、[[盧山縣|盧山縣]]、[[名山縣|名山縣]]、[[榮經縣|榮經縣]]、[[百丈縣|百丈縣]]
* 領羈縻州四十四:[[當馬州|當馬州]]、[[三井州|三井州]]、[[來鋒州|來鋒州]]、[[名配州|名配州]]、[[鉗恭州|鉗恭州]](鉗泰州)、[[斜恭州|斜恭州]](隸恭州)、[[畫重州|畫重州]]、[[羅林州|羅林州]]、[[籠羊州|籠羊州]]、[[林波州|林波州]]、[[林燒州|林燒州]]、[[龍蓬州|龍蓬州]]、[[敢川州|敢川州]]、[[驚川州|驚川州]]、[[禍眉州|禍眉州]]、[[木燭州|木燭州]]、[[百坡州|百坡州]]、[[當品州|當品州]]、[[嚴城州|嚴城州]]、[[中川州|中川州]]、[[鉗矣州|鉗矣州]]、[[昌磊州|昌磊州]]、[[鉗并州|鉗并州]]、[[百頗州|百頗州]]、[[會野州|會野州]]、[[當仁州|當仁州]]、[[推梅州|推梅州]]、[[作重州|作重州]]、[[禍林州|禍林州]]、[[金林州|金林州]]、[[諾祚州|諾祚州]]、[[三恭州|三恭州]]、[[布嵐州|布嵐州]]、[[欠馬州|欠馬州]]、[[羅蓬州|羅蓬州]]、[[論川州|論川州]]、[[讓川州|讓川州]]、[[遠南州|遠南州]]、[[卑盧州|卑盧州]]、[[夔龍州|夔龍州]]、[[耀川州|耀川州]](輝川州)、[[金川州|金川州]]、[[東嘉梁州|東嘉梁州]]、[[西嘉梁州|西嘉梁州]]

== 元代 ==
[[元朝|元朝]]时属[[吐蕃宣慰司|吐蕃宣慰司]]。辖境相当今四川省[[雅安市|雅安]]、[[名山区|名山]]、[[荥经县|荥经]]、[[芦山县|芦山]]等市县。

== 明代 ==
[[明朝|明朝]][[洪武|洪武]]四年（1371年），省州治严道县入州，[[直隶|直隶]][[四川布政司|四川布政司]]，是为[[雅州直隶州|雅州直隶州]]。
== 清代 ==
[[清朝|清朝]][[雍正|雍正]]七年（1729年），升雅州直隶州为[[雅州府|雅州府]]。
