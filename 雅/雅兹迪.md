'''雅兹迪教'''（[[庫爾德語|庫爾德語]]：{{lang|ku|Êzidîtî}}／{{lang|ku|Êzidî}}、[[阿拉伯文|阿拉伯文]]：{{rtl-lang|ar|يزيدي}}／{{lang|ar|ايزيدي|}}），是中東一種古老而獨特的[[宗教|宗教]]。雅茲迪教徒可以說是一種族教群體（Ethnoreligious group），但雅茲迪人也屬於庫爾德人的一部分。雅茲迪教徒只佔[[库尔德人|库尔德人]]的少數，而大多數库尔德人則为[[伊斯兰教|伊斯兰教]]的[[逊尼派|逊尼派]][[穆斯林|穆斯林]]。目前雅茲迪人正遭到恐怖主義極端分子的武裝團體[[伊斯蘭國|伊斯蘭國]]以宗教名義進行大批量的屠殺與迫害，處在滅族的危殆中。<ref>{{cite news|last=Nebehay|first=Stephanie|date=2016-06-16|title=Islamic State committing genocide against Yazidis: U.N.|url=http://www.reuters.com/article/us-mideast-crisis-syria-yazidi-idUSKCN0Z20WR|newspaper=Reuters|location=Geneva|access-date=2016-06-16}}</ref>

== 人口 ==
雅兹迪教派在[[伊拉克|伊拉克]]的库尔德人口中仍占一定比例，为[[伊拉克|伊拉克]]的宗教少数派之一。雅兹迪教徒大多使用[[库尔德语|库尔德语]]。

雅兹迪教徒总人口150万人，絕大多数分布于[[伊拉克|伊拉克]]北部，其中位於[[摩蘇爾|摩蘇爾]]以西約100公里的[[辛賈爾|辛賈爾]]（{{lang|en|Sinjar}}），少数分布于[[亚美尼亚|亚美尼亚]]、[[德国|德国]]、[[俄罗斯|俄罗斯]]等地。他们与世隔绝，生活贫困。

==宗教信仰==
雅兹迪教派是一种混合了[[多神教|多神信仰]]的[[一神教|一神教]]，它与[[琐罗亚斯德教|琐罗亚斯德教]](祆教)并无关系<ref>{{cite web |url=http://ezidipress.com/blog/weltorganisation-der-zoroaster-eziden-sind-keine-zoroaster-interview-mit-ezidipress/ |title=Weltorganisation der Zoroaster: "Êzîden sind keine Zoroastrier" – Interview mit êzîdîPress  – ÊzîdîPress |work=ezidipress.com |accessdate=17 December 2015}}</ref><ref>{{cite web |url=http://ezidipress.com/blog/zarathustra-statue-als-geschenk-fuer-eziden-in-efrin-wieso/ |title=Zarathustra-Statue als Geschenk für Êzîden in Efrîn  – ÊzîdîPress |work=ezidipress.com |accessdate=17 December 2015}}</ref><ref>{{cite web |url=http://ezidipress.com/blog/qewl-falschungen/ |title=Qewl-Fälschungen |work=ezidipress.com |accessdate=17 December 2015}}</ref>，而是与古代美索不达米亚宗教相关、亦包含了原始的自然崇拜、[[基督教|基督教]]、[[伊斯蘭教|伊斯蘭教]]、[[猶太教|猶太教]]、[[婆羅門教|婆羅門教]]，是一种古老而独特的宗教。

雅兹迪教派相信有一位至高无上的神，即[[上帝|上帝]]，但非善恶[[二元论|二元论]]，否定[[魔鬼|魔鬼]]的存在。雅茲迪教徒並不強調升[[天堂|天堂]]或下[[地獄|地獄]]，他們相信[[輪迴|輪迴]][[轉世|轉世]]，並稱之為『靈魂換衣裳』。

雅茲迪教相信至高无上的神[[創世|創世]]的時候，首先以自身的光明（Ronahî）創造了[[孔雀天使|孔雀天使]]（{{lang|en|Tawsi Melek}}，亦稱{{lang|en|Shaytan}}），繼而創造六個[[熾天使|熾天使]]，並對其領導者[[孔雀天使|孔雀天使]]說：「孔雀天使有權力不向其他一切事物跪拜，並且有責任引導其他事物。」接著對六個熾天使說：「你們必須向[[孔雀天使|孔雀天使]]跪拜。」而後，上帝吩咐六名熾天使，叫他們取來塵土，並用塵土製造出[[亞當|亞當]]，神以自身的氣息賦予[[亞當|亞當]]生命，命令熾天使們向[[亞當|亞當]]跪拜。除了[[孔雀天使|孔雀天使]]抗命，所有天使都跪拜了。[[孔雀天使|孔雀天使]]說：「我無法遵從您的命令，因為我是由您的榮耀所創造，而亞當僅僅出自於塵土。」於是上帝就稱讚孔雀天使，並叫其他天使向孔雀天使跪拜。因此雅茲迪人相信[[孔雀天使|孔雀天使]]代替神引導人類，並於每年的四月的第一個禮拜三降臨人間，這也就成為雅茲迪人的新年。比較特別的是，雅茲迪人相信自己是[[亞當|亞當]]的兒子舍西德-宾-杰爾(Shehid bin Jer)的後裔，但與[[夏娃|夏娃]]無關。

因為拒絕向[[亞當|亞當]]跪拜的部份與《[[古蘭經|古蘭經]]》中的[[惡魔|惡魔]][[伊布力斯|伊布力斯]]（[[撒但|撒但]]）相似，所以在[[穆斯林|穆斯林]]眼中，[[孔雀天使|孔雀天使]]就成為了[[伊布力斯|伊布力斯]]。故雅茲迪人普遍被[[穆斯林|穆斯林]]視為「拜魔鬼的人」<ref name="追杀">[http://news.mingpao.com/20140808/tba2_image1.htm?Mode=1 話你知﹕雅茲迪人被指拜魔鬼遭追殺]，明報 2014-08-08。{{zh-hant}}</ref>。

儘管周遭民族視雅茲迪為惡魔的崇拜者，但雅茲迪人從未曾將兩者相連結，事實上，在雅茲迪人面前提到[[撒但|撒但]]是很不妥當的。雅茲迪人认为[[撒但|撒但]]已在[[基督|基督]]下降至[[陰間|陰間]]的三日期間，向基督[[懺悔|懺悔]]了，所以基督靠著[[上帝|上帝]]給予的權柄，[[赦免|赦免]]了他的罪，並恢復了他的[[天使|天使]]地位。

雅茲迪人相信善惡共存於人心，為善或為惡取決於人類的自由意志。因此對孔雀天使的奉獻格外重要，因為孔雀天使在神面前選擇了善。

雅茲迪的聖經為啟示書（Kitêba Cilwe）與黑書（Mishefa Reş）。這兩本書分別於1911與1913由一名西方探險家出版，根據雅茲迪手稿編寫而成。雅茲迪的殘存核心文本大多是讚美詩與諺語，多數歷史均由口述傳承，因此雅茲迪社群亦同意將其歷史文化傳說整理成書。由於雅茲迪教的諺語寓意晦澀不明，而攥寫人十分喜歡雅茲迪文化，一般相信這兩本書的內容十分忠實地描述了雅茲迪信仰。而這兩本書亦成為雅茲迪人的聖經。

雅茲迪的諺語充滿了神秘的典故，述說時往往必須由敘事人加以比喻解釋。這些諺語大略包含了兩個層面 1) 保持雅茲迪教信仰的純粹性  2) 關於輪迴的信仰。
第一部份包含了雅茲迪人的宗族制度，飲食，雅茲迪社群的傳統偏好，生活中的禁忌。第二部份是最關鍵的，雅茲迪人相信七天使會定期轉世為人，稱為“Koasasa”。

==婚姻==
雅茲迪人的婚配僅限於其族群內部，是一種制度化的“自行結合”。雅茲迪男子與女子相戀，取得她同意後，隨即便可帶離她自組家庭。{{cn}}雅茲迪教禁止信徒改宗或與外人通婚。2007年一位雅茲迪女性與穆斯林男子私奔並改信伊斯蘭教之後，遭其族人投石虐殺而死。而後，眾多雅茲迪人也因此事件遭到伊斯蘭教極端分子的瘋狂報復。

==歷史==
許多人相信雅茲迪教是[[基督教|基督教]]或[[伊斯蘭教|伊斯蘭教]]的分支，但其實它是種截然不同的宗教。歷史上，雅茲迪人據稱是古代[[米底亞|米底亞]]人的後代，在中東伊斯蘭化浪潮興起後，一直受到[[穆斯林|穆斯林]]的迫害和歧視。在近現代的[[亞美尼亞種族大屠殺|亞美尼亞種族大屠殺]]期間，雅茲迪人也遭到了[[奧斯曼|奧斯曼]][[土耳其|土耳其]]的大規模屠殺和迫害，大批原居住在[[安納托利亞|安納托利亞]]（現[[土耳其|土耳其]]東部）的雅茲迪教徒被迫改信伊斯蘭教，或者逃亡至[[伊拉克|伊拉克]]北部或[[高加索地區|高加索地區]]。

由於雅茲迪人自信為神另外所造，與其他人不同，故雅兹迪教派严禁其成員与[[異教徒|異教徒]]通婚，也不歡迎外教人皈依，違反此教規者會被驅逐。近年最轟動的事件為[[2007年4月|2007年4月]]發生的「千人投石虐殺少女事件」，原因只在于该少女不顾族人反对，与一名[[伊斯蘭教|伊斯蘭教]][[遜尼派|遜尼派]]男子相戀[[私奔|私奔]]並[[皈依|皈依]][[伊斯蘭教|伊斯蘭教]]。而此千人懲罰少女的殘暴事件另一方面也激怒了[[伊斯蘭教|伊斯蘭教]]極端主義份子，隨後給雅茲迪人群體帶來了滅頂之災。

正是由于这起「千人投石虐杀少女事件」，宗教和族群矛盾骤然升级，自2007年8月开始，忠于[[基地组织|基地组织]]的伊斯蘭教[[逊尼派|逊尼派]][[激進|激進]]組織「[[伊斯蘭國|伊斯蘭國]]」[[聖戰|聖戰]]分子对北伊拉克的雅兹迪教徒发动了一连串疯狂的报复行为。其中最血腥的一起暴力报复事件为2007年8月14日，伊拉克北部小镇[[加哈坦尼亚|加哈坦尼亚]]郊外的雅兹迪教派聚居地突遭4辆載有[[炸彈|炸彈]]的货车连环[[自杀式袭击|自杀式袭击]]，数百平房瞬间夷平，超过400名雅茲迪人平民死亡，这成为自[[2003年美伊戰爭|2003年美伊戰爭]]以来最血腥的一次恐怖暴力袭击，而西方国家则抨击此暴力袭击的施袭者是在发动一场針對雅茲迪人的「[[种族清洗|种族清洗]]」。

[[2014年8月|2014年8月]]，「伊斯蘭國」再度入侵，[[處決|處決]]了500名雅茲迪教徒，大多數亂槍射死，少數人被[[軍刀|軍刀]][[斬首|斬首]]，或被斬斷手腳釘在[[十字架|十字架]]上殺死，或以繩索[[縊死|縊死]]，另有許多人被[[活埋|活埋]]，或者負傷再活埋。許多婦女被[[姦殺|姦殺]]，超過300名婦女與女童被虜，可能被[[人口販賣|轉賣]]，或成為[[軍妓|軍妓]]、[[性奴隸|性奴隸]]。另外還有數萬雅兹迪教徒被圍困在[[辛賈爾|辛賈爾]]山，至少56名兒童[[脫水|脫水]]而死。[[美國總統|美國總統]][[歐巴馬|歐巴馬]]授權[[美軍|美軍]]展開救援行動，已[[空襲|空襲]]炸毀伊斯蘭國許多[[裝甲車|裝甲車]]、武裝貨車，並且[[空投|空投]]大量的[[口糧|口糧]]與[[礦泉水|礦泉水]]，掩護了三萬名雅茲迪信徒逃到[[敍利亞|敍利亞]]。[[美國共和黨|美國共和黨]][[參議員|參議員]][[馬侃|馬侃]]，批評其前總統大選對手歐巴馬對於救援與戰鬥不夠積極。歐巴馬表示將會「持續作戰，努力不懈」。<ref>[http://www.appledaily.com.tw/appledaily/article/international/20140811/36015030/ 不改信教就滅村 IS屠殺500人]</ref><ref>[http://www.ettoday.net/news/20140811/388394.htm 數百雅茲迪女子被俘！ ISIS或賣或留「都是當性奴」]</ref><ref>[http://www.ettoday.net/news/20140809/387803.htm 伊斯蘭國行「種族淨化」　屠殺世界最古老「雅茲迪族」]</ref><ref>{{Cite news|url=http://hk.apple.nextmedia.com/international/art/20140811/18828193 |title=3萬雅茲迪人逃出　揭伊斯蘭國暴行　聖戰分子殺500人活埋婦孺 |publisher=hk.apple.nextmedia.com |date=2014-08-11}}</ref>
