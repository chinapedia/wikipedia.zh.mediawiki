'''抚顺城'''，是指[[明清时期|明清时期]]在今[[辽宁省|辽宁省]][[抚顺市|抚顺市]]境内修建的[[城池|城池]]。

== 明代抚顺城 ==
明代抚顺城位于[[浑河|浑河]]北岸[[高尔山|高尔山]]下，现已无存，大体位置在今[[抚顺北火车站|抚顺北火车站]]西边不远之处。明王朝建城之后，命名为“抚顺”。抚顺自[[汉代|汉代]]设置行政机构后，数易其名，直至明[[洪武|洪武]]十七年（1384年），才正式定名为抚顺，并沿用至今。

洪武二十一年（1388年），明朝在抚顺置[[千户所|千户所]]，属[[沈阳中卫|沈阳中卫]]。又在抚顺以东的[[官岭|官岭]]增设[[抚顺关|抚顺关]]，并筑辽东边墙与边堡。[[成化|成化]]元年（1465年），又于抚顺城南马驿设边墩，以防患。成化四年（1468年），由[[辽东|辽东]]副总兵[[韩斌|韩斌]]主持重修抚顺城。重修后的抚顺城稍大于旧城，其周三里，[[护城河|护城河]]宽二丈，深一丈，开东、南、北三门，为当时抚顺地区最大的军事城堡及指挥中心，也是辽东地区重要的军事前哨。

明[[正统_(年号)|正统]]十四年（1449年），[[女真|女真]]首领[[李满柱|李满柱]]，乘发生“[[土木之变|土木之变]]”朝野混乱之际，率领1万多人包围了抚顺城。明将[[鲁全|鲁全]]率守城将士誓死守城，最终击退了李满柱的进攻，这是明代抚顺城最大的一次战斗。明[[万历|万历]]四十六年（1618年），[[努尔哈赤|努尔哈赤]]誓师伐明，亲率大军进攻抚顺城。守将[[李永芳|李永芳]]畏敌投降。努尔哈赤轻取抚顺城。为防止明军重至，努尔哈赤撤走时一举焚毁城池。以抚顺命名的这座古城兴于兵乱，毁于战火，共历时235年<ref name="book_10397137_p127-128">{{cite book zh| language =zh-hans| author =《中国城市综合实力五十强丛书·抚顺市》编委会／编| title =《中国城市综合实力五十强丛书·抚顺市》>第三章 综合实力>第十八节 旅游风光>明代抚顺城| format =| url =http://book.xuexi365.com/ebook/detail_10397137.html| date =1994年03月1日第1版| publisher =中国城市出版社| location =北京| id =ISBN 750740641| accessdate =1994年03月1日第1次印刷| pages =第127－128页}}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

== 清代抚顺城 ==
明万历四十六年（1618年），努尔哈赤攻下抚顺城，撤走时将城焚毁，一直到清[[康熙|康熙]]二十一年，[[康熙皇帝|康熙皇帝]]第二次东赴[[清永陵|永陵]]谒祖时，抚顺仍是一片荒凉。由于抚顺是清王朝“开国立基”的“龙兴圣地”，又是[[清朝皇帝|清朝皇帝]]谒永陵的必经之路，故从[[乾隆|乾隆]]四十三年（1778年）起，到乾隆四十八年（1783年）止，用了六年时间，重新修筑的抚顺城。

清代抚顺城的位置在明代抚顺城南面，按现在的区划是：西至[[宁远街|宁远街]]，东至[[新华街|新华街]]，北至[[永吉路|永吉路]]，南至[[新城路|新城路]]。据《[[盛京通志|盛京通志]]》记载：城周围共三里，有东、南、北三个城门。南门叫“嘉会门”，北门称“广润门”，东门为“便门”。重新建城之后，城内商业随之繁荣。商店、铺店、手工业作坊遍布街市。较大型的店铺有抚西丝布店、永和当铺、福粮庄和一品位饭店。还有肉市、牲畜市、柴市等自由交易市场。成为辽东东部地区重要的农副产品交流市场和集市中心。

清[[道光|道光]]九年（1829年），清政府曾拨款对抚顺城进行了一次大的维修。[[光绪|光绪]]三十一年（1905年）[[日俄战争|日俄战争]]时期，抚顺城遭大炮轰击。“广润门”匾额被[[俄军|俄军]]炮弹击毁。光绪三十四年（1908年），[[兴仁县|兴仁县]]改称[[抚顺县|抚顺县]]，移治于抚顺城。抚顺城内工商业和居住中心逐渐移至浑河南岸的[[千金寨|千金寨]]。此后，抚顺城日趋萧条，而城墙“嘉会门”则一直较完整存在。1950年，为了扩建道路，将城墙全部拆除<ref name="book_10397137_p129">{{cite book zh| language =zh-hans| author =《中国城市综合实力五十强丛书·抚顺市》编委会／编| title =《中国城市综合实力五十强丛书·抚顺市》>第三章 综合实力>第十八节 旅游风光>明代抚顺城| format =| url =http://book.xuexi365.com/ebook/detail_10397137.html| date =1994年03月1日第1版| publisher =中国城市出版社| location =北京| id =ISBN 750740641| accessdate =1994年03月1日第1次印刷| pages =第129页}}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。
