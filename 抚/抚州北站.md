'''抚州北站'''在[[江西省|江西省]][[抚州市|抚州市]][[臨川區|臨川區]][[抚北鎮|抚北鎮]]{{efn|改造后新建客运站房站场位于抚北工业园区}}，是[[抚江线|抚江线]]和[[昌福铁路|昌福线]]上的一座客、货运[[三等站_(中国大陆)|三等站]]，由[[南昌铁路局|南昌铁路局]][[南昌车务段|南昌车务段]]管辖<ref>{{Cite web|url=http://www.chnrailway.com/news/20130128/0128433756.html|title=南昌车务段举行行车应急知识考试|accessdate=2016-09-29|author=|date=2013-01-28|last=中华铁道网|work=www.chnrailway.com|publisher=}}</ref>；距離的城區[[臨川區|臨川區]]約6公里。

== 历史 ==
1961年8月[[向乐线|向乐线]]开始建设，1963年9月[[江家站|江家]]到抚州北开始施工，年底铺轨至抚州北<ref name="江西省铁路志7-5035-3">{{cite book |title=《江西省铁路志》 |author=傅金土 |year=1994年12月 |edition=1994年12月第1次印刷 |publisher=中共中央党校出版社 |isbn=7-5035-3 |ignore-isbn-error=true |oclc=36212671}}</ref>{{rp|53}}。

1964年3月开始工程临时运营，11月抚州北以南开始铺轨；1965年10月16日到26日[[南昌铁路局|南昌铁路局]]进行验收，11月全线办理临时运营，12月8日到24日办理交接，1966年1月1日正式交付运营<ref name="江西省铁路志7-5035-3"/>{{rp|53}}。

1995年为向乐线上的三等站，由[[上海铁路局|上海铁路局]][[南昌铁路分局|南昌铁路分局]]管辖<ref>{{cite book |title=《中国铁路交通地图集》 |author= |year=1995年1月 |edition=1995年1月河北第1次印刷 |publisher=中国铁道出版社 中国地图出版社 |isbn=7-113-01810-6 |oclc=48531409}}</ref>{{rp|100}}。

2003年为向乐线上客、货运三等站，由南昌铁路局管辖，离[[向塘西站|向塘西]]62公里，距[[江边村站|江边村]]60公里<ref>{{cite book |title=《中国铁路站名词典》|author=孙本祥 编 |year=2003年4月 |edition=2003年4月第1次印刷 |publisher=中国铁道出版社 |isbn=7-113-03941-3 |oclc=53211800}}</ref>{{rp|382}}。

2004年为向乐线上三等站，由南昌铁路局管辖<ref>{{cite book |title=《中华人民共和国铁路地图集》 |author= |year=2004年6月 |edition=2004年6月第2次印刷 |publisher=中国铁道出版社 |oclc=62999371 |id=统一书号17113·1219}}</ref>{{rp|77}}；2006年同<ref>{{cite book |title=《中华人民共和国铁路地图集》 |author= |year=2006年6月 |edition=2006年6月第3次印刷 |publisher=中国铁道出版社 |oclc=879963547 |id=统一书号17113·1214}}</ref>{{rp|40}}。

2007年11月24日[[向莆铁路|向莆铁路]]江西段开工，确认沿线将设抚州北站<ref>{{Cite web|url=http://jiangxi.jxnews.com.cn/system/2007/11/23/002617848.shtml|title=向莆铁路（江西段）今日开工（组图）|accessdate=2016-09-24|author=|date=2007-11-24|work=jiangxi.jxnews.com.cn|publisher=大江网|archiveurl=https://web.archive.org/web/20150509120906/http://jiangxi.jxnews.com.cn/system/2007/11/23/002617848.shtml|archivedate=2015-05-09}}</ref>。

2008年为向乐线上客、货运三等站，由南昌铁路局管辖<ref>{{cite book |title=《中华人民共和国铁路地图集》 |author= |year=2008年12月 |edition=2008年12月第4次印刷 |publisher=中国铁道出版社 |oclc=502099863 |id=统一书号17113·1308}}</ref>{{rp|40}}；2012年同<ref>{{cite book |title=《中国铁路地图集》 |author= |year=2012年2月1日 |edition=2012年7月第3次印刷 |publisher=中国铁道出版社 |isbn=978-7-113-13840-0 |oclc=818363258}}</ref>{{rp|40}}。

由于新建的向莆铁路经过抚州北站，因此本站需改造，2011年6月21日改造工程开始<ref>{{Cite web|url=http://txy.chnrailway.com/news/20110624150732.shtml|title=中铁十七局向莆铁路抚州北站开工|accessdate=2016-09-30|author=|date=2011-06-24|work=txy.chnrailway.com|publisher=}}{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref><ref>{{Cite web|url=http://jxfz.jxnews.com.cn/system/2011/06/26/011698478.shtml|title=向莆铁路线上又一站 新抚州北站开工|accessdate=2016-09-30|author=|date=2011-06-26|work=jxfz.jxnews.com.cn|publisher=}}</ref>；8月11日车站主体封顶<ref>{{Cite web|url=http://www.zgfznews.com/fznews/fuzhouxinwen/jingji/2011/8/14/735481.shtml|title=抚州新闻网-经济-向莆铁路抚州北站主体结构封顶|accessdate=2016-10-01|author=|date=2011-08-14|work=www.zgfznews.com|publisher=|archive-url=https://web.archive.org/web/20161012074219/http://www.zgfznews.com/fznews/fuzhouxinwen/jingji/2011/8/14/735481.shtml|archive-date=2016-10-12|dead-url=yes}}</ref>。

2012年7月31日改造后的抚州北站正式启用，[[向乐线|向乐线]]也恢复运行<ref>{{Cite web|url=http://www.jxfz.gov.cn/xxgk/gzdt/zwyw/201207/t20120730_1467192.htm|title=新抚州北站明日正式启用|accessdate=|author=|date=2012-07-30|publisher=抚州市人民政府|deadurl=yes|archiveurl=https://web.archive.org/web/20160304140105/http://www.jxfz.gov.cn/xxgk/gzdt/zwyw/201207/t20120730_1467192.htm|archivedate=2016-03-04}}</ref><ref>{{Cite web|url=http://txy.chnrailway.com/news/20120801105856.shtml|title=向莆铁路向乐线提前开通 新抚州北站同时启用|accessdate=2016-09-29|author=|date=2012-08-01|work=txy.chnrailway.com|publisher=}}{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>。

2013年5月6日向乐线为保证向莆铁路联调联试停运<ref>{{Cite web|url=http://jxfz.jxnews.com.cn/system/2013/04/27/012393307.shtml|title=5月5日南昌至江边村7205次列车停驶 5月6日向乐线停运|accessdate=2016-09-30|author=|date=2013-04-27|work=jxfz.jxnews.com.cn|publisher=}}</ref>，之后确认向乐线在向莆铁路通车后不会恢复客运服务因此抚州北站也将停止客运服务<ref>{{Cite web|url=http://jiangxi.jxnews.com.cn/system/2013/08/30/012609406_02.shtml|title=鹿心社与网民在线交流 37个问题落实情况公布-鹿心社-江西新闻网|accessdate=2016-09-30|author=|date=2013-08-30|work=jiangxi.jxnews.com.cn|publisher=}}</ref>。

2015年为[[抚江线|抚江线]]（原向乐线）和[[昌福线|昌福线]]（建设时称为[[向莆铁路|向莆铁路]]的大部分路段）上的客、货运站{{efn|参考资料中昌福线沿线车站均标志四、五等车站}}，且为抚江线起讫站，由南昌铁路局管辖<ref>{{cite book |title=《中国铁路地图集》 |author= |year=2015年2月28日 |edition=2015年3月第4次印刷 |publisher=中国铁道出版社 |isbn=978-7-113-13840-0}}</ref>{{rp|40}}。

2016年南昌铁路局重新公布了管内的车站等级，本站为三等站<ref>{{cite journal |author=南昌铁路局办公室 |journal = 南昌铁路局文件 |date=2016-06-20 |issue= |title = 《南昌铁路局关于重新公布车站等级等事项的通知》 |id = 南铁劳卫〔2016〕226号 |url=http://www.aiweibang.com/yuedu/126493427.html |page = }}</ref>。

== 车站结构 ==
新站房主体一层，高12.43米，总建筑面积999平方米，只有一个侧式站台靠着站房，站台有悬挑钢架雨棚；车站总体覆盖2193平方米。

向莆铁路的建设时本站进行了改造，改造前车站规模为2站台5线，以站房起为：[[侧式站台|侧式站台]]、股道、股道、[[岛式站台|岛式站台]]、股道、股道、股道。

改造后原客运区域只有多条股道，而新客运区域相比原客运区域北移了近700米<ref>{{Cite web|url=http://jxfz.jxnews.com.cn/system/2012/07/31/012059761.shtml|title=新抚州北站惊艳亮相 今日正式启用|accessdate=2016-09-29|author=|date=2012-07-31|work=jxfz.jxnews.com.cn|publisher=}}</ref>；由于向莆铁路通车后[[抚州站|抚州站]]将承当主要客运服务，因此本站改造后的规模为1站台6线，站台为侧式站台，为低站台。
