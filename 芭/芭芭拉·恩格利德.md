'''芭芭拉·恩格利德'''（德語：Barbara Engleder，{{bd|1982年|9月16日||}}）是一名[[德国|德国]]女子射击运动员。<ref>{{cite web |title=Barbara Engleder |url=http://www.london2012.com/athlete/engleder-barbara-1092605/ |publisher=London 2012 |accessdate=2012-09-17 |deadurl=yes |archiveurl=https://archive.is/20130128061602/www.london2012.com/athlete/engleder-barbara-1092605/ |archivedate=2013-01-28 }}</ref>她在[[2012年伦敦奥运会|2012年伦敦奥运会]]上，获得女子50米步枪三姿第六名。<ref>{{cite web |title=Women's 50m Rifle 3 Positions Finals |url=http://www.london2012.com/shooting/event/women-50m-rifle-3-positions/index.html?v=20120812-012220930 |publisher=London 2012 |accessdate=2012-09-17 |deadurl=yes |archiveurl=https://archive.is/20130128080329/http://www.london2012.com/shooting/event/women-50m-rifle-3-positions/index.html?v=20120812-012220930 |archivedate=2013年1月28日 |df= }}</ref>2016年[[里约奥运会|里约奥运会]]，她赢得女子50米步枪三姿[[金牌|金牌]]。
