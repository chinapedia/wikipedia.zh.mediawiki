'''災防告警細胞廣播訊息系統'''（{{lang-en|Public Warning System}}，縮寫：{{lang|en|PWS}}）又稱'''災防告警系統'''，是[[中華民國政府|中華民國政府]]利用[[細胞廣播|細胞廣播]]技術，在短時間內經由[[台灣電信業者列表#行動寬頻業務業者（4G）|4G業者]]的行動寬頻系統，大量傳送[[強震即時警報|地震速報]]、[[土石流|土石流]]警戒、[[公路|公路]]封閉等防災警示訊息到[[手機|手機]]，即時通知民眾的系統。<ref name="gov0224">{{Cite web |url = http://www.cdprc.ey.gov.tw/News_Content.aspx?n=DE11525F5E6F3240&sms=348A76B6B4BCF95E&s=8BE27A79C8EFA804 |title = 1050224災防週報 |date = 2016-02-24 |work = 中央災害防救會報 |publisher = 中華民國行政院 |language = zh-tw |accessdate = 2016-03-30 |archiveurl = https://web.archive.org/web/20160403034211/http://www.cdprc.ey.gov.tw/News_Content.aspx?n=DE11525F5E6F3240&sms=348A76B6B4BCF95E&s=8BE27A79C8EFA804 |archivedate = 2016-04-03 |dead-url = yes}}</ref>此系統可以在數秒內傳送訊息到特定區域內的數十萬到百萬臺手機，手機在接收到訊息時，會發出特殊的警告聲響與振動。<ref name="ncc1231">{{cite press release |title = 4G災防告警細胞廣播訊息系統建置完成，105年1月5日將於屏東縣滿州鄉進行告警訊息測試 |url = http://www.ncc.gov.tw/chinese/news_detail.aspx?site_content_sn=8&is_history=0&pages=0&sn_f=34785 |publisher = 中華民國國家通訊傳播委員會 |date = 2015-12-31 |accessdate = 2016-03-30 |language = zh-tw |archiveurl = https://web.archive.org/web/20160305035924/http://www.ncc.gov.tw/chinese/news_detail.aspx?site_content_sn=8&is_history=0&pages=0&sn_f=34785 |archivedate = 2016-03-05 |dead-url = yes}}</ref>

==架構==
災防告警細胞廣播訊息系統的架構為政府災防警訊端、訊息匯整與派送端、訊息傳送管道端、應用終端，政府災防警訊端為訊息來源，由中央部會或地方政府發布告警訊息，訊息匯整與派送端為災防訊息廣播平台（CBE），負責接收告警訊息及統一訊息交換格式，訊息傳送管道端為細胞廣播控制中心（CBC），負責傳送訊息至[[基地台|基地台]]，應用終端為手機，負責接收訊息。<ref>{{cite web |title = PWS系統架構 |url = https://www.ncc.gov.tw/chinese/gradation.aspx?site_content_sn=3746&is_history=0 |publisher = 中華民國國家通-{}-訊傳播委員會 |date = 2016-05-03 |accessdate = 2018-09-14 |language = zh-tw |archiveurl =  |archivedate =  |dead-url = }}</ref>[[國家災害防救科技中心|國家災害防救科技中心]]規劃的細胞廣播訊息碼則分別為國家級警報（部份手機顯示為總統級警報<ref>{{cite news |author = 生活中心 |title = 警報沒統一？總統還是國家級　原來和手機翻譯有關 |url = http://www.setn.com/News.aspx?NewsID=146233 |work = 三立新聞網 |date = 2016-05-12 |accessdate = 2016-05-30 |language = zh-tw |archiveurl = https://web.archive.org/web/20160617161518/http://www.setn.com/News.aspx?NewsID=146233 |archivedate = 2016-06-17 |dead-url = no}}</ref>）、緊急警報、警訊通知和每月測試用訊息。<ref name="cbs_ide">{{cite web |title = CBS訊息碼 |url = https://www.ncc.gov.tw/chinese/news_detail.aspx?site_content_sn=3727&sn_f=35445 |publisher = 中華民國國家通-{}-訊傳播委員會 |date = 2016-04-27 |accessdate = 2018-09-14 |language = zh-tw |archiveurl =  |archivedate =  |dead-url = }}</ref>

==訊息等級==
