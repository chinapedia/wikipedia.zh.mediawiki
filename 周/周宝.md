'''周宝'''（{{bd|814年||888年|2月12日|catIdx=Z}}<ref name=ZZTJ257>[[:s:資治通鑑/卷257|《资治通鉴》卷二百五十七]]</ref>），字'''上-{珪}-'''，'''汝南郡王'''，[[唐朝|唐朝]]将领，以[[节度使|节度使]]身份控制[[镇海|镇海]]八年。

== 家世和早期仕途 ==
周宝生于[[唐宪宗|唐宪宗]]年间的814年，是[[平州|平州]]人氏。曾祖[[周待选|周待选]]在[[安史之乱|安史之乱]]时为[[鲁城|鲁城]]令，试图阻止[[安禄山|安禄山]]进军时被败杀。祖父[[周光济|周光济]]为唐朝对安禄山的[[燕_(安史之乱)|大燕]]政权作战的平卢节度使[[侯希逸|侯希逸]]牙将，因其父之死而寻求复仇，只要抓到参与燕军曾围鲁城者，都手刃之。随后他又效力侯希逸的继任者实际独立统治平卢的[[李正己|李正己]]。李正己死后，周光济随其堂兄[[李洧|李洧]]以[[徐州|徐州]]归朝廷。周宝父[[周怀义|周怀义]]，通书记，官至检校工部尚书、[[天德|天德]]西城防御使，但因宰相[[李吉甫|李吉甫]]拒绝支持他迁徙天德城的建议而忧死。<ref name=NBT186>[[:s:新唐書/卷186|《新唐书》卷一百八十六]]</ref><ref>李吉甫在807—808年间和811—814年间两度拜相，为周怀义去世的日子提供了潜在的范围。见[[:s:舊唐書/卷148|《旧唐书》卷一百四十八]]。</ref>

因家族为朝廷效力，周宝得以荫为千牛备身。周怀义参军[[殷侑|殷侑]]成为天平军节度使后，邀周宝为部将。[[唐武宗|唐武宗]][[会昌_(唐朝)|会昌]]年间（841年—847年），周宝回[[长安|长安]]入宿卫，隶右[[神策军|神策军]]，因而与同僚[[高骈|高骈]]为密友，高骈比周宝年轻，事周宝如兄长。周宝历任良原镇使，善[[马球|马球]]，将其作为军将训练的一部分。后因未得升官，他见了同样爱好马球的武宗，请求做武宗宫中的马球手。武宗同意了，擢金吾将军。但周宝在打马球时失去了一只眼睛。进检校工部尚书、[[泾原节度使|泾原节度使]]。他支持泾原的城防，聚粮二十万斛，被号为良将。<ref name=NBT186/>

== 控制镇海 ==
[[乾符|乾符]]六年（879年）十月，<ref>《旧唐书·僖宗纪》作十一月。</ref>周宝以神策大将军被任为检校尚书左仆射，兼润州刺史、镇海军节度、浙江西道观察等使，代替被调往邻镇[[淮南|淮南]]的高骈。<ref name=BT19-2>[[:s:旧唐书/卷19下|《旧唐书》卷一十九下]]</ref><ref>[[:s:資治通鑑/卷253|《资治通鉴》卷二百五十三]]</ref>最初周宝和高骈关系友好；当时高骈领盐铁事，辟周宝子[[周佶|周佶]]为支使，周宝也请高骈的从子来他的幕府。<ref name=NBT186/>但尽管他们曾在右神策军彼此以兄弟相待，高骈作为将军名声远盖周宝，开始轻视他，他们的治所又相邻，开始有争执。[[中和_(唐朝)|中和]]元年（881年），长安已落入[[黄巢|黄巢]]农民大军之手，迫使[[唐僖宗|唐僖宗]]逃往[[成都|成都]]。高骈称在准备一场攻打黄巢收复长安的大规模行动，且作为诸道行营都统发檄写信要求周宝调军为备。当时周宝已加检校[[司空|司空]]。<ref>[[:s:浙西周宝司空|崔致远《浙西周宝司空》]]</ref>周宝大喜，照办，点阅军队准备勤王。有人告知周宝高骈有意吞并江东学汉末[[孙策|孙策]]三分天下，周宝尚不信。后周宝侦察发现高骈并无真正的用兵计划，<ref name=BT182>[[:s:旧唐书/卷182|《旧唐书》卷一百八十二]]</ref>便以为高骈其实想攻打他。当高骈邀请他到军中议事时，他称病不出，二人互相侮辱，友谊完全破裂。于是高骈以周宝和[[浙东节度使|浙东节度使]][[刘汉宏|刘汉宏]]为后患为由，不发兵助朝廷，<ref name=BT19-2/><ref name=NBT224-3>[[:s:新唐书/卷224下|《新唐书》卷二百二十四下]]</ref><ref name=ZZTJ254>[[:s:資治通鑑/卷254|《资治通鉴》卷二百五十四]]</ref><ref>[[:s:谢诏上升地墨敕状|崔致远《谢诏上升地墨敕状》]]</ref>甚至故意请求[[致仕|致仕]]。<ref>[[:s:让官请致仕表|崔致远《让官请致仕表》]]</ref>

同时，镇海军自身也被农民军攻打，城镇沦陷，因而组织了“八都”抵御之——[[董昌|董昌]]的石镜都，[[陈晟|陈晟]]的清平都，[[吴文举|吴文举]]的于潜都，[[徐及|徐及]]的盐官都，[[杜棱|杜棱]]的新登都，[[饶京|饶京]]的唐山都，[[文禹|文禹]]的富春都，[[凌文举|凌文举]]的龙泉都。<ref name=NBT186/><ref>“八都”也有其他版本，如《十国春秋·吴越武肃王世家上》作陈晟主余杭都，杜棱主新城都，饶景主唐山都，文禹（一作闻人宇）主富阳都；《杭州府志·杜棱传》又称杜棱为东安都将，董昌为临安都将，刘孟安、阮结为钱塘都将，凌文举为余杭都将，曹信为临平都将，与闻人宇、徐及号八都。</ref>但九月，八都之一的董昌趁新任[[杭州|杭州]]刺史[[路审中|路审中]]将赴任之机，夺取杭州，自称杭州都押牙、知州事，并遣将吏请命于周宝。周宝自认为力不能制董昌，表其为杭州刺史。<ref>[[:s:新唐书/卷225下|《新唐书》卷二百二十五下]]</ref>十一月，唐僖宗加周宝荣誉宰相衔[[同中书门下平章事|同中书门下平章事]]，<ref name=ZZTJ254/>兼天下租庸副使，封汝南郡王。二年（882年）七月，周宝奏高骈承制以变军首领[[孙端|孙端]]为[[宣歙观察使|宣歙观察使]]。僖宗诏命周宝与宣歙观察使[[裴虔余|裴虔余]]发兵拒之。<ref>[[:s:資治通鑑/卷255|《资治通鉴》卷二百五十五]]</ref>

周宝温和宽宏，喜欢接纳士人，见京师沦陷，有心勤王，组织了一支叫“后楼都”的新军，军饷比镇海军多一倍，镇海军因而怨恨。周宝让儿子[[周玙|周玙]]统领后楼都，但周玙驭军无方，后楼都骄横无军纪。周宝也开始饮宴娱乐度日，不注意军队的需求。他任女婿[[杨茂实|杨茂实]]为[[苏州|苏州]]刺史，杨茂实课以重税，百姓负担过重。控制僖宗朝廷的当权宦官[[田令孜|田令孜]]派[[赵载|赵载]]代杨茂实。周宝累次上表请求让杨留任，但均未获准，于是让杨在离任前毁坏官署。<ref name=NBT186/>

但是，周宝对镇海诸州的实际控制还在继续崩溃。早在中和元年二月，馀杭镇使陈晟夺取[[睦州_(仁寿)|睦州]]。<ref name=NBT186/><ref name=ZZTJ256>[[:s:資治通鑑/卷256|《资治通鉴》卷二百五十六]]</ref><ref>陈晟夺取睦州是否是兵变的结果尚待考。《资治通鉴》称他从睦州刺史[[柳超|柳超]]手中夺取睦州，《新唐书·周宝传》却称陈晟从睦州刺史[[韦诸|韦诸]]手中夺取睦州，而柳超则是早先韦诸败杀的农民军头领。[[郁贤皓|郁贤皓]]《唐刺史考全编》取韦诸于880年杀柳超，881年为陈晟所执。参考《严州图经》《新唐书·僖宗纪》。</ref>光启二年（886年）正月，因同样打马球而为他所宠的部下牙将[[张郁|张郁]]兵变攻陷[[常州|常州]]，六月周宝派另一牙将[[丁从实|丁从实]]败之，收复常州。周宝还发牒文给高骈商议处置临淮叛卒事，高骈回信责周宝纵容养奸。<ref>[[:s:浙西周宝司空书|崔致远《浙西周宝司空书》]]</ref>十二月，董昌部下都知兵马使[[钱鏐|钱鏐]]败刘汉宏，董昌占有浙东，<ref name=ZZTJ256/>将杭州交给钱鏐，周宝同意，承制以钱镠权知杭州军州事兼杭州管内都指挥使。<ref name=NBT186/><ref name=HFD133>[[:s:旧五代史/卷133|《旧五代史》卷一百三十三]]</ref>当时，田令孜下属右散骑常侍[[沈诰|沈诰]]使江南，仗着和田令孜的关系，做出很多腐败行为。僖宗的远亲襄王[[李煴|李煴]]短期登基（同年他和主要支持者静难节度使[[朱玫_(唐朝)|朱玫]]被杀，使僖宗再次成为无可争议的皇帝），下令搜捕田令孜党羽，周宝趁机捕杀沈诰和赵载。<ref name=NBT186/>

== 败亡 ==
三年（887年），周宝耽于享乐，不问政事。他征发民夫为军部[[润州|润州]]建罗城二十余里并建造东面官邸，百姓苦之。三月，<ref>《旧唐书·僖宗纪》《旧唐书·高骈传》作二月。《实录》作四月，疑为奏报到日。《资治通鉴》取《吴越备史》《新唐书·僖宗纪》。详见《[[资治通鉴考异|资治通鉴考异]]》。《梁启圣匡运同德功臣、淮南镇海镇东等军节度使、淮南浙江东西等道观察处置营田招讨安抚兼盐铁制置发运等使、开府仪同三司尚父守尚书令、扬杭越等州大都督府长史、上柱国吴越王钱公生祠堂碑》亦作光启三年春。</ref>周宝和僚属饮宴，有人说镇海军对后楼都怀怨，周宝答：“他们敢作乱就杀了他们！”度支催勤使[[薛朗|薛朗]]将此言告诉朋友镇海军牙将[[刘浩|刘浩]]，以为警告，刘浩说：“我们只有兵变才能免死！”当夜率其党同僚刁頵及薛朗等兵变，攻周宝官邸。周宝正睡觉，醒来，见外兵格斗，火照城中，惊出，谕道：“为吾所用则是吾兵，否则是寇。六州皆我镇，去哪不行？”试图召后楼都对抗变军，但很快发现后楼都也加入了变军。他别无他法迎战变军，于是率家人从青阳门出奔，步行逃到常州投丁从实。<ref name=BT182/>乱兵大掠，刘浩杀周宝僚佐崔绾、陆锷、田倍等，推薛朗为留后，薛朗自称知府事。<ref name=BT19-2/><ref name=HFD133/><ref name=NBT9>[[:s:新唐书/卷9|《新唐书》卷九]]</ref><ref name=NHFD67>[[:s:新五代史/卷67|《新五代史》卷六十七]]</ref><ref name=TQ>李琪《梁启圣匡运同德功臣、淮南镇海镇东等军节度使、淮南浙江东西等道观察处置营田招讨安抚兼盐铁制置发运等使、开府仪同三司尚父守尚书令、扬杭越等州大都督府长史、上柱国吴越王钱公生祠堂碑》</ref><ref name=TWY>[[:s:吴越国武肃王庙碑铭|皮光业《吴越国武肃王庙碑铭》]]</ref><ref>《旧唐书·僖宗纪》作刘浩自称留后。</ref>周宝先前兼租庸副使，储存财货堆积如山，都落入变军之手。当高骈得知周宝倒台，为之庆贺，还送齑一瓶、葛粉十斤给他，<ref>[[:s:致周宝书|高骈《致周宝书》]]</ref>以为讥讽。周宝怒而将其掷在地上，说：“你有[[吕用之|吕用之]]，难说以后会发生什么！”<ref name=ZZTJ256/>吕用之是高骈信任的方士，当时已对高骈治下的淮南有了很大的控制权。最终因吕用之腐败，左厢都知兵马使[[毕师铎|毕师铎]]兵变，高骈对吕用之说“你好自为之，不要让我成为周侍中！”结果毕师铎驱逐吕用之，但随即囚禁了高骈，九月，高骈被杀。<ref name=ZZTJ257/><ref name=NBT224-3/>

周宝知淮南六合镇遏使[[徐约|徐约]]有精兵，以[[苏州|苏州]]诱其攻占据苏州的[[张雄_(唐朝)|张雄]]。<ref name=ZZTJ256/>张雄出逃海上，使别将[[赵晖|赵晖]]据[[上元|上元]]，周宝所部溃卒多归赵晖。<ref name=ZZTJ257/><ref>[[:s:旧唐书/卷190|《新唐书》卷一百九十]]</ref>

十月，钱鏐派东安都将杜稜攻常州，丁从实逃到[[海陵|海陵]]。钱鏐以部将对节度使的仪式以军礼从郊外迎周宝归杭州，居于樟亭驿。先前周宝去[[丹阳|丹阳]]，州人总说“待钱来”，至此应验。<ref>《十国春秋·吴越武肃王世家上》</ref>十二月，周宝卒于樟亭驿。<ref>《僖宗实录》</ref>赠'''太保'''。<ref name=ZZTJ257/>《新唐书·僖宗纪》称十月丁卯日钱鏐杀之，<ref name=NBT9/>《新唐书·周宝传》载十月钱鏐杀之，<ref name=NBT186/>《十国纪年》亦载周宝或系钱鏐所杀，但《资治通鉴》取《吴越备史》。<ref>[http://www.guoxue123.com/shibu/0101/01zztjhz/257.htm [[胡三省|胡三省]]注释版《资治通鉴》卷二百五十七] {{webarchive|url=https://web.archive.org/web/20110711135040/http://www.guoxue123.com/shibu/0101/01zztjhz/257.htm |date=2011-07-11 }}</ref>钱鏐攻破润州擒住薛朗后，<ref name=HFD133/><ref name=TWY/>剖其心以祭周宝。<ref name=ZZTJ257/><ref name=NHFD67/><ref name=TQ/><ref>[[:s:旧唐书/卷20上|《旧唐书·昭宗纪》]]则载被擒杀剖心者为刘浩。</ref>
