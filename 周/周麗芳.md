'''周麗芳'''（1963年-)，現任[[國立政治大學|國立政治大學]]財政學系特聘專任教授，曾任[[臺北市政府|臺北市政府]]副市長、[[行政院衛生署|行政院衛生署]][[中央健康保險局|中央健康保險局]]副總經理等職務，研究專長為政府基金管理、教育財政、人力資源政策、綠色能源財經、社會福利與社會政策、醫務管理與醫藥經濟。2015年2月11日，[[台北市長|台北市長]][[柯文哲|柯文哲]]宣布台北市第三位副市長由周麗芳出任。在2015年2月16日正式就職上任。<ref>{{Cite web |title=台北市第三位副市長　政大財政系周麗芳|url=http://www.appledaily.com.tw/realtimenews/article/new/20150211/558048/|publisher=蘋果日報 |language=zh-tw |accessdate=2015-02-11}}</ref><ref>{{Cite web |title=北市第三位副市長、消防局長　今正式上任|url=https://tw.news.yahoo.com/%E5%8C%97%E5%B8%82%E7%AC%AC%E4%B8%89%E4%BD%8D%E5%89%AF%E5%B8%82%E9%95%B7-%E6%B6%88%E9%98%B2%E5%B1%80%E9%95%B7-%E4%BB%8A%E6%AD%A3%E5%BC%8F%E4%B8%8A%E4%BB%BB-054451327.html|publisher=Yahoo奇摩新聞、NOWnews|language=zh-tw |accessdate=2015-02-16}}</ref>2016年2月卸任。

==近況==
*2015年2月18日，台北市長[[柯文哲|柯文哲]]宣布，2017年[[台北世大運|台北世大運]]執行長由台北市副市長周麗芳兼任，副執行長則由[[臺北市政府研究發展考核委員會|臺北市政府研究發展考核委員會]]主委[[陳銘薰|陳銘薰]]兼任。<ref>{{Cite web |title=【更新】柯文哲宣布　世大運執行長周麗芳|url=http://www.appledaily.com.tw/realtimenews/article/new/20150218/561675/|publisher=蘋果日報 |language=zh-tw |accessdate=2015-02-18}}</ref>
*2016年1月13日，周麗芳表示為了學術生涯規畫，加上北市府工作結束階段性任務，因而決定請辭，台北市長柯文哲也已批准；[[臺北市政府人事處|臺北市政府人事處]]下午發布人事令，預計在2016年2月1日生效。<ref>{{Cite web |title=北市府異動！副市長周麗芳請辭　柯文哲准了|url=http://www.ettoday.net/news/20160114/630411.htm|publisher=Ettoday 東森新聞雲|language=zh-tw |accessdate=2016-01-14}}</ref>
