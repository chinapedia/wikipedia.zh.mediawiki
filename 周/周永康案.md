'''周永康案'''，指前[[中国共产党中央政治局常务委员会|中共中央政治局常委]]、[[中共中央政法委员会|中央政法委]][[中共中央政法委员会书记|书记]][[周永康|周永康]]2012年退休后，于2013年起被中共内部调查，2014年被[[中共中央纪委|中共中央纪委]]宣布立案审查后开除中共党籍的一起党纪案件：'''周永康严重违纪案'''，及随后其被[[最高人民检察院|最高人民检察院]]立案侦查、逮捕后于2015年移送[[天津市人民检察院|天津市人民检察院]]第一分院向[[天津市第一中级人民法院|天津市第一中级人民法院]]提起公诉（判处无期徒刑）的一起刑事案件'''周永康受贿、滥用职权、故意泄露国家秘密案'''<ref name="公诉" />。广义上的“周永康案”也包括其亲属、相关商人以及众多其派系官员等一系列犯罪案件，其中包括周永康儿子[[周滨|周滨]]，四川商人[[刘汉_(四川商人)|刘汉]]，周永康的秘书[[冀文林|冀文林]]等人的犯罪案件。<ref>[http://www.hx168.com.cn/hxzq/public/info_detail.jsp?infoId=21197462&classID=10681 涉黑四川富豪刘汉攀附高官 曾高价从周永康儿子手中买项目]，华西证券</ref><ref>[http://www.cb.com.cn/hots/2014_0220/1039467.html 冀文林落马“秘书帮”沦陷被曝染指政法操控石油 任周永康秘书10年]</ref><ref>[http://money.163.com/14/0222/01/9LLBUP5R00253B0H.html 中油国际党委书记失联 某卸任高层四秘书均沦陷]，中国经营报(北京)</ref><ref>[http://www.cb.com.cn/hots/2014_0219/1038592.html 海南副省长冀文林被调查 曾任周永康秘书]</ref><ref>[http://news.cjn.cn/gnxw/201402/t2430359.htm 海南省副省长冀文林被调查 曾任周永康秘书] {{webarchive|url=https://web.archive.org/web/20140228062031/http://news.cjn.cn/gnxw/201402/t2430359.htm |date=2014-02-28 }}，新京报</ref>

周永康案及其一系列案件的范围覆盖政治、经济、文化等诸多领域，时间跨度达数十年之久，地域横跨[[中国东北地区|中国东北]]、[[中国西南地区|西南]]、[[华东地区|华东]]乃至[[京畿|京畿]]、[[中国共产党中央委员会|中央]]，涉案官员的级别几乎囊括[[中国共产党|中国共产党]]整个[[中华人民共和国公务员|干部体系]]，从县处级、厅局级、省部级到国家级<ref name="双规">{{cite web |url = http://www.ccdi.gov.cn/yw/201407/t20140730_37357.html |title = 中共中央决定对周永康严重违纪问题立案审查 |accessdate = 2014年7月29日 |publisher = 中央纪委监察部网站}}</ref><ref>{{cite web |url = http://dailynews.sina.com/bg/chn/chnnews/ausdaily/20131228/17285313753.html |title = 周永康案或定性为中共建党后最大贪腐集团 |publisher = 新浪网 |date = 2013年12月28日}}。</ref>

周永康是1949年[[中华人民共和国开国大典|中华人民共和国成立]]以来首位因腐败问题而接受调查并被开除中共党籍的[[中国共产党中央政治局常务委员会|中共中央政治局常委]]，也是被查处的最高级别官员。周永康的落马，被外界认为是打破了坊间传闻中中共党内自邓小平时代以来“刑不上常委”的政治[[潜规则|潜规则]]<ref>{{cite web |author = [[吴思|吴思]] |title = 吴思：周永康被立案 “刑不上常委”潜规则已破？|url = http://news.qq.com/a/20140729/095862.htm?pgv_ref=aio2012&ptlang=2052 |accessdate=2014-08-13}}</ref>。

周永康长子[[周滨|周滨]]在1989年至1992年就读[[西南石油大学|西南石油大学]]科技英语专业，与[[米晓东|米晓东]]和[[朱莉萍|朱莉萍]]为校友关系。后至美国德克萨斯大学达拉斯分校留学，认识[[黄渝生|黄渝生]]与[[詹敏利|詹敏利]]的女儿[[黄婉|黄婉]]并结婚。2001年二人归国。周滨在美国留学期间就通过关系将国外的石油设备卖给中国的石油企业。<ref>{{cite web|title=周永康之子周滨涉非法经营罪被批捕 |url = http://news.qq.com/a/20140729/092389.htm?pgv_ref=aio2012&ptlang=2052 |accessdate=2014-08-26}}</ref><ref name="c21">{{cite web|title=周滨攫财录：倒卖油田 设备采购中间人 |url = http://news.qq.com/a/20140729/096124.htm |accessdate=2014-08-26}}</ref><ref name="c21"/>其主要的业务为倒卖石油设备，倒卖油田股权，作为中石油的供应商获得项目，以及未经证实的代持股企业。<ref name="wvvd">{{cite web|title=周滨：“以父之名”的敛财之道|url=http://finance.qq.com/a/20140730/006655.htm?pgv_ref=aio2012&ptlang=2052|accessdate=2014-08-27}}</ref>

2001年，在周滨的控制下，四川商人吴兵成立中旭投资。2006年3月注册的中旭实业，旗下两家水电开发公司，一家房地产开发公司和多家水电、房产业务延伸公司。其主要开发两个水电项目：总装机70万千瓦、投资53亿元的大渡河龙头石水电站，总装机45万千瓦、投资32亿元的革什扎水电站。大渡河龙头石水电站有詹敏利的部分控股。<ref name="c21"/>

2004年，回国的周滨建立北京中旭阳光石油天然气科技有限公司，由吴兵管理。中旭阳光的大股东最初是周滨和詹敏利，后詹敏利将股权卖给周滨。中旭阳光能源成立不久即获得中石油十多家省级分公司涉及8000座加油站的零售管理系统信息化开发工程，又参与中石油的成品油物流配送系统和工程项目管理系统及信息系统的建设，及与中石油旗下的塔里木油田、吉林油田、长庆油田、辽河油田等分公司有设备销售关系。<ref name="c21"/>

2007年1月，米晓东在[[西安|西安]]参股设立陕西秋海汲清石油科技有限公司，同年底在西安的同一办公地址成立陕西德淦石油科技有限公司（两公司实际控制人是周滨），这两个公司以1000万至2000万左右的低价获得长庆油田石油合作开发项目长印、长海区块。米晓东将德淦及其长印项目以5.5亿元卖给吉林民营公司华海能源集团董事长王乐天。后王乐天发现长印项目能够开采的面积只有13平方公里，不足50平方公里，要求与米晓东和周滨重新谈判，其结果是秋海汲清的长海项目和王盘山的一个小油田被补偿给王乐天。<ref name="c21"/>

2009年，詹敏利和米晓东参股成立北京秋海旭荣房地产开发公司，获得北京市南口农场NC-01街区公租房项目的承建资格。2010年8月，詹敏利将控股秋海旭荣的“北京昀滢旭荣投资管理有限公司”的大部分1550万股权转让给北京天恒联信置业有限公司。<ref name="c21"/>

2010年和2011年期间，周滨作为中间人协助米晓东，将数批油气开采设备卖给与[[伊拉克|伊拉克]]国有公司[[米桑石油|米桑石油]]油田合作的中方作业公司。<ref name="wvvd"/>

2011年汇盛阳光等5家由詹敏利担任大股东的公司，向北京市朝阳区来广营乡租赁奶白路3号约200平米的办公用房和周边300亩原本用于高尔夫球场的土地，租期20年，每亩年租金6000元，用途未知。汇盛阳光又投资于周永康秘书[[郭永祥|郭永祥]]之子郭连星的北京汇润阳光能源科技有限公司（同样在奶白路3号）。詹敏利与米晓东还有同样注册于奶白路3号的北京浩盛益佳投资管理有限公司，在河北、四川和海南投资三家公司：保定中茂能源有限公司，浩盛益佳与中石油旗下昆仑能源有限公司华气清洁能源投资有限公司、保定光阳天然气利用有限公司。<ref name="c21"/>

除了中旭系之外，一家石油设备供应商惠生工程技术服务有限公司的董事长华邦嵩被传为周滨代持公司股权。于2013年8月被有关部门调查。<ref name="wvvd"/>

凭借他的获利，周滨在北京观唐别墅拥有一套2000万元人民币的别墅。在望京新城方恒国际大厦，有至少8套房产在周滨的岳母詹敏利名下，建筑面积约1000平方米。北四环奥林匹克中心旁边的华亭嘉园有一套詹敏利名下的公寓。2012年他将北京银湖别墅的一套房转手，售价在3500万-4000万元。<ref name="c21"/>

===贾晓烨及贾晓霞案===
周永康第二任妻子[[贾晓烨|贾晓烨]]，曾在[[中国中央电视台|央视]]二套财经频道工作，妹妹贾晓霞曾就读复旦大学外语学院，在周永康支持下进入石油系统，先后在中石油[[厄瓜多尔|厄瓜多尔]]和[[加拿大|加拿大]]分公司工作，后担任中油国际加拿大公司总经理。<ref name="c21"/>

在贾晓霞运作下，[[中石油|中石油]]在储量未探明情况下匆忙出资40亿加元收购加拿大两个油砂项目，中石油在未来十年中必须连续投资至少300亿加元，才有可能看到石油。这次收购使大量国有资产进入周永康家族囊中。周永康被曝在加拿大设有秘密帐户，在加资产惊人，贾晓霞是周永康在加拿大代理人。多年来，贾晓霞至少敛财数十亿美元，实际控制着中石油百亿美金的资产。在2014年的北京亚太经合组织(APEC)领导人非正式会议上，[[习近平|习近平]]与加拿大总理[[史蒂芬·哈珀|哈珀]]对话，哈珀表示，“加拿大无意收留逃犯，愿意在遣返方面同中方开展合作。” <ref>{{cite web |url = http://www.gcpnews.com/viewer.php?aid=123096&cid=1063&pid=1 |title = 周永康小姨子或将被遣返回国受审 |publisher =  |date =  |language = zh-hans }}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

===周元兴、周元青案===
周永康的胞兄周元兴在周永康发迹后，与儿子周晓华以[[五粮液|五粮液]]代理商发迹，另又通过自己与周永康的关系替人在打官司时说情和捞人，向江苏一家警校输送学生，在没有工厂的状况下做油田的钢管生意。<ref name="c21" />

另一位胞弟周元青，曾任无锡市厚桥镇副镇长、锡山市经济技术协作办公室主任、惠山区国土局副局长等职务。其妻子周玲英是锡山市昌隆物贸有限公司的股东，该公司涉足加油站业务。她又创办无锡骏峰农资发展有限公司，销售化肥、农药、金属材料、消防设施设备及器材。二人儿子周锋投资创办位于北京、东莞等地的多家公司，与中石油相关公司关系复杂<ref name="c21" />。[[有关部门|有关部门]]在调查周元青家的时候，搜出大量的黄金珠宝烟酒，还有一大串[[奥迪|奥迪]]车钥匙<ref>{{cite web |title=周家12月被抄：搜出大量黄金珠宝 拥B99999车牌 |url=http://money.163.com/14/0730/08/A2D0OKAL00252G50.html?frp091 |accessdate=2014-08-27 |deadurl=yes |archiveurl=https://web.archive.org/web/20140814081643/http://money.163.com/14/0730/08/A2D0OKAL00252G50.html?frp091 |archivedate=2014-08-14 }}</ref>。

===蒋洁敏案===
受到周永康提携的中石油集团总经理[[蒋洁敏|蒋洁敏]]，将长庆油田的一个准备勘探的项目换成一个已经产油的项目，再批给与其有关的人士对外合作开发。<ref name="c21" />

===与刘汉案的关系===
周永康于1999年调任[[中共四川省委|中共四川省委]]书记，带来其秘书[[郭永祥|郭永祥]]和[[冀文林|冀文林]]并留任，由此扩展自己在四川的势力<ref>{{cite web|title=周永康的凶在圈子里出名 没人敢给他提意见 |url = http://news.qq.com/a/20140731/011009.htm?pgv_ref=aio2012&ptlang=2052|accessdate=2014-08-26}}</ref>。虽然他并不同意周滨在四川扩展势力，但周滨依然在这里开设公司。<ref name="c21" />

2002年，包括周滨妻子黄婉在内的5人在成都成立四川超越有限公司，在其正式成立半个多月前，该公司与茂县人民政府签订九鼎山旅游景区开发协议，有效期为50年。2004年10月，超越公司把景区度假开发权和无形资产，以2000万元转让给了刘汉的汉龙集团。据评估，该景点价值大约只有500-600万元。<ref name="c21" />

后来，刘汉与周滨四川阿坝州成立兴鼎电力公司，开发毛尔盖河电力资源。周滨持20%的股份。该项目得到县、州和省三级发改委批准并申请到6亿元的银行贷款。2009年，汉龙集团从周滨手中购回兴鼎电力公司20%的股权，但实际上没有付款。<ref name="c21"/>

2015年4月3日上午九点整，[[中纪委|中纪委]]网站发布消息，周永康案由[[最高人民检察院|最高人民检察院]]侦查终结，并经指定管辖，移送[[天津市|天津市]][[人民检察院|人民检察院]]第一分院审查起诉。同日，天津市人民检察院第一分院以周永康涉嫌[[受贿|受贿]]、[[滥用职权|滥用职权]]、[[故意泄露国家秘密罪|故意泄露国家秘密罪]]，向[[天津市第一中级人民法院|天津市第一中级人民法院]]提起公诉。起诉书中指控：“被告人周永康在担任中国石油天然气总公司副总经理，中共四川省委书记，中共中央政治局委员、公安部部长、国务委员和中共中央政治局常委、中央政法委书记等职务期间，利用职务上的便利，为他人谋取利益，非法收受他人巨额财物；滥用职权，致使公共财产、国家和人民利益遭受重大损失，社会影响恶劣，情节特别严重；违反保守国家秘密法的规定，故意泄露国家秘密，情节特别严重，依法应当以受贿罪、滥用职权罪、故意泄露国家秘密罪追究其刑事责任。”<ref name="公诉">{{cite web |url =http://www.ccdi.gov.cn/yw/201504/t20150403_54235.html|title = 天津检察机关依法对周永康案提起公诉 |accessdate = 2015年4月3日|publisher = 中央纪委监察部网站}}</ref>
2015年6月11日，天津市第一中级人民法院依法对周永康受贿、滥用职权、故意泄露国家秘密案进行了一审宣判。由于涉及国家机密，没有公开进行审理。认定周永康犯受贿罪，判处[[无期徒刑|无期徒刑]]，[[剥夺政治权利|剥夺政治权利]]终身，并处没收个人财产；犯滥用职权罪，判处[[有期徒刑|有期徒刑]]七年；犯故意泄露国家秘密罪，判处有期徒刑四年，三罪并罚，决定执行无期徒刑，剥夺政治权利终身，并处没收个人财产。周永康对所指控的上述犯罪事实证据均当庭表示属实、没有异议，服从法庭判决，不上诉。<ref name="判决">{{cite web |url = http://www.ccdi.gov.cn/xwtt/201506/t20150611_57735.html |title = 周永康一审被判处无期徒刑 |accessdate = 2015年6月11日 |publisher = 中央纪委监察部网站}}</ref>

[[英国广播公司|英国广播公司]]引述法国一大学教授张伦的观点称，周案公布后网上一片欢呼，但拥有新闻自由的国家不会出现这种现象，由此提醒人们，不管中国怎样得意于其现代化成就，但政治落后于时代。这种依靠剥夺人民权益、压制人们表达自由的[[中国模式|中国模式]]，使得权贵肆无忌惮地掠夺，执政者用利益输送维持官僚集团的内部凝聚，精英层的责任和批判意识被麻木，使社会正义缺失。周永康作为维稳体制的代言人，也是最大受益者。周永康的覆灭虽有偶然性，但深层原因是中国模式无法继续按旧的方式维系，中国到了一个需要调整发展路线和文明方向的阶段。中国反腐远没有到最后定论的时候，比如[[江泽民|江泽民]]、[[曾庆红|曾庆红]]和[[温家宝|温家宝]]等。习近平在反腐后若不能引领中国走向自由和公正，其获得的权威终将丧失。<ref>{{cite web|url=http://www.bbc.co.uk/zhongwen/simp/china/2014/08/140804_zhongyongkang_chinese_political_direction.shtml|title=周永康案与中国的政治走向|publisher=BBC|date=2014年8月4日 | language =zh-hans }}</ref>

中国学者[[章立凡|章立凡]]认为，中国现在还是靠中纪委、[[中央巡视组|中央巡视组]]，靠党内的自查自纠来反腐，国家司法体系只是配角。中国的根本的问题是[[政治体制|政治体制]]是一个腐败的温床，如果没有政治体制上的变革，现在的人治反腐只能暂时有效。<ref>{{cite web|url=http://www.dw.de/%E5%8D%81%E6%9C%88%E5%9B%9B%E4%B8%AD%E5%85%A8%E4%BC%9A%E8%BF%BD%E8%AE%A4%E5%91%A8%E6%B0%B8%E5%BA%B7%E6%97%A7%E9%97%BB/a-17824186|title=十月四中全会追认“周永康旧闻”？|publisher=德国之声|date=2014-07-31  | language =zh-hans }}</ref>

周永康案会否延烧至江泽民和曾庆红，中国独立政治评论人士李伟东认为反腐的大高潮是扳倒江泽民和曾庆红，但前提是江泽民不识趣，步步紧逼<ref>{{cite web|url=http://www.voachinese.com/content/xijinping-liweidong-20140804/1971429.html|title=习近平反腐最终目标—打垮太上皇架构 |publisher=美国之音|date=04.08.2014 | language =zh-hans }}</ref>。历史学者[[高文谦|高文谦]]认为会对江泽民动手，因为习近平的反腐已经没有回头箭，要想继续巩固自身的政权，只能继续的进行反腐，甚至对老一辈已经退休的常委也要进行反腐<ref>{{cite web|url=http://www.voachinese.com/content/pro-and-con-0801-1/1969561.html|title=周老虎就擒，江核心若隐若现？ |publisher=美国之音|date=01.08.2014| language =zh-hans }}</ref>。也有消息人士透露，周永康被正式起诉得到了习近平和前中共总书记江泽民的批准。<ref >{{cite web|url=http://www.voachinese.com/content/zhou-yongkang-spied-on-xi-20150420/2726576.html | language= | title=彭博：周永康曾监视习近平等中国领导人 |publisher=美国之音 |date= 2015-04-20 |accessdate= }}</ref>

[[新加坡国立大学|新加坡国立大学]]教授[[郑永年|郑永年]]称周永康案后，要求反腐败运动告一段落的声音主要来自中国国内的寡头及海外在中国的既得利益集团。习近平如果抱着“段落”的观念来反腐，腐败就会卷土重来，同时还会给外界一种不好的印象，即中共的反腐败仅仅是为了政治需要的政治运动，而并非要建立廉洁政府。<ref>{{cite web |url = http://news.uschinapress.com/2014/0804/987624.shtml |title=郑永年谈周永康案：寡头腐败威胁执政党 |publisher=侨报网|date=2014-08-04 |language =zh-hans }}</ref>
