《'''周日八點黨'''》，簡稱《'''周八'''》，為[[中國電視公司|中國電視公司]]於2000年4月16日至2008年7月27日每個星期日20:00 - 22:00播出的[[台灣|台灣]][[綜藝節目|綜藝節目]]，由[[全能製作公司|全能製作公司]]製作，[[吳宗憲|吳宗憲]]等人主持。

== 節目單元 ==
以下是該節目各期主要的單元。

=== 第一代 ===
此時期的主持人各有一個「[[頭銜|頭銜]]」：吳宗憲的頭銜是「黨主席」，康康、郁方、柯受良的頭銜都是「黨代表」。首次參加錄影的藝人，稱為「首次入黨」。

==== 全民薦寶 ====
〈全民薦寶〉，諧音「[[全民健保|全民健保]]」，是開放[[兒童|兒童]]報名的才藝表演單元。

==== 當我們銬在一起 ====
〈當我們銬在一起〉是由康康與另外一名觀眾朋友以[[手銬|手銬]]「銬」在一起，且必須持續四十八小時。康康與被銬之觀眾在此期間一起生活。可以達到標準者，就可以由其他主持人在攝影棚內解開（實際上，大多是剪開）手銬，並獲得高額獎金。

但有時候康康會對參賽來賓胡鬧，讓參賽者不知所措或是感到不好意思，這也是該單元的笑點之一。

該單元的名稱來源，是來自[[兒歌|兒歌]]《當我們同在一起》，並將「同」字改成「銬」字，以符合該單元的宗旨。

==== 對不起　我害你 ====
吳宗憲與康康互整的外景單元。

==== 大牌''class'' ====
以[[教室|教室]]佈景為主的短劇單元。吳宗憲扮演「班主任」，其他人（包括康康、來賓與固定班底）扮演[[學生|學生]]。來賓進入教室前，必須先敲門。

=== 第二代、第三代 ===
==== 食字路口 ====
〈食字路口〉，諧音「[[平面道路交叉|十字路口]]」，是此時期的「招牌單元」之一。其構想來源，是來自《[[火焰挑戰者|火焰挑戰者]]》內「擔保挑戰」的一個[[食物|食物]]名稱[[接龍|接龍]][[遊戲|遊戲]]，亦以接龍為主體，但所使用的[[語言|語言]]為[[漢語|中文]]。

參賽隊伍分為由吳宗憲所領軍的「憲憲隊」、康康所領軍的「康康隊」、許傑輝所領軍的「輝輝隊」，以及NONO所領軍的「NONO隊」，各自進行「食物接龍」遊戲，以字首、字尾為主進行接龍。

單元開始，由「小天使」為參賽隊伍及[[電視機|電視機]]前[[閱聽人|觀眾]]公開謎頭，並由三隊開始一較高下。每個參賽隊伍的謎頭都是相同的，但在第二道食物開始就不盡相同。

參賽者吃到第五道食物的時候，小天使就會再給謎尾。參賽者必須想辦法接到謎尾，並回到起點敲[[鑼|鑼]]。參賽隊伍最先敲鑼者，即算獲勝。每隊收到的謎尾都是相同的。

食物名稱可用諧音，也可以使用任何種類的語言的一小段[[音節|音節]]，或者以食物的其他[[名稱|名稱]]，來代替其原來的名稱。而遊戲也有規定，參賽者於每次遊戲中均只能到[[便利商店|便利商店]]一次，不允許二次以上。

;舉例說明一：純[[漢字|國字]]接法
*[[檸檬愛玉|檸檬愛玉]]→[[玉米|玉蜀黍]]→[[薯條|薯條]]→[[牛乳|調味牛乳]]→[[滷味|滷味]]→[[味噌湯|味噌湯]]→[[水餃|湯餃]]…
<small>（''其中的「[[牛乳|調味牛乳]]」與「[[滷味|滷味]]」，雖然[[中文|中文]]內「滷」與「乳」的發音不同，但口語發音非常接近，可取其諧音''）</small>

;舉例說明二：混他國語言接法
*[[拿鐵|拿鐵]]→[[提拉米蘇|Tiramisu]]（提拉米蘇）→[[蘇打水|蘇打水]]→[[水煎包|水煎包]]→[[包穀|包穀]]→[[彈珠汽水|古早味彈珠汽水]]→[[水餃|水餃]]→絞肉…
<small>（''其中的「[[拿鐵|拿鐵]]」與「[[提拉米蘇|提拉米蘇]]」，雖然[[中文|中文]]內「[[鐵|鐵]]」與「提」發音不同，但[[口語|口語]]發音類似，可取其諧音。''）</small>

另外，品嚐[[料理|料理]]的過關規定如下：
#如果該料理是整道菜，則該隊伍必須將該道菜吃光。例如，如果接的是「[[螞蟻上樹|螞蟻上樹]]」，該隊伍就必須共同將整盤螞蟻上樹吃光。

#如果該料理是試吃品，則該隊只要吃一部份即可。
#如果該料理是他人正在食用者，該隊可以向正在吃東西的他人取得一點食物，吃完這部份即可。
#如果接的是如[[蔬菜|蔬菜]]、[[水果|水果]]的名稱，而料理中剛好有相對應的食材（例如，隊伍要接[[櫻桃|櫻桃]]，而店家提供有櫻桃裝飾的[[蛋糕|蛋糕]]）者，也可以如法泡製；但相對的，該隊參賽者就必須將整塊蛋糕吃完，方可過關。

〈食字路口〉另有「超激版」，開放觀眾報名，規則同上；但最輸者必須被現場其他參賽者砸刮鬍泡，作為處罰。

食字路口在2005年改成文武超激版，一樣採用食物文字接龍進行，但過程中加入「文」與「武」兩個關卡，「文」是小天使出題目，參賽者必須回答問題，例如：與NONO傳過緋聞女友有幾個。「武」是必須通過小天使出的任務，各隊的文與武關卡各不相同，通過文武關卡才能繼續進行食物文字接龍的遊戲，最後輸的隊伍則是必須進行處罰，處罰是由贏家採用當天處罰的食物放入果汁機榨成果汁讓輸家喝完。

==== 周日色教館 ====
〈周日色教館〉是由主持人出題，參賽者即刻回答[[顏色|顏色]]的一個單元。「色教館」一名的來源構想，是來自「[[臺北市立社會教育館|臺北市立社會教育館]]」的簡稱「'''社教館'''」，並取其諧音。

主持人按照節拍，隨意說出三種[[顏色|顏色]]；而這三種[[顏色|顏色]]之中，可以重複，也可以不重複。參賽者必須按照主持人所指示的[[顏色|顏色]]，並遵照樂隊所演奏的節拍，說出相關的物品為何。失誤達一定次數者，或者重複物品太多次者，會被判出局。

舉例說明：
#如果[[主持人|主持人]]說：「[[白色|白色]]、[[黑色|黑色]]、[[綠色|綠色]]」，參賽者可說：「[[雲|雲]]、[[鋼琴|鋼琴]]、[[樹葉|樹葉]]」。
#但如果[[主持人|主持人]]說：「[[橘色|橘色]]、[[紫色|紫色]]、[[黃色|黃色]]」，參賽者卻說：「[[橘子|橘子]]、[[茄子|茄子]]、[[襪子|襪子]]」，則[[主持人|主持人]]會有可能判定[[襪子|襪子]]是不正確的答案（因為[[襪子|襪子]]可以有好幾種[[顏色|顏色]]，[[黃色|黃色]]不是[[襪子|襪子]]唯一的[[顏色|顏色]]）。
#如果[[主持人|主持人]]的提示是：「[[黑色|黑色]]、[[黑色|黑色]]、[[黑色|黑色]]」，而參賽者卻說：「[[瓜子|瓜子]]、[[瓜子|瓜子]]、[[瓜子|瓜子]]」的話，就很明顯的出局（因為[[瓜子|瓜子]]的重複次數太多）。

==== 看我七十二騙 ====
2006年6月18日開播的整人單元，測驗藝人遇到奇怪狀況時會有什麼反應。名稱來源是取[[蔡依林|蔡依林]]的歌曲《[[看我72變_(歌曲)|看我72變]]》，並將「變」取其諧音，改為「騙」。

==== 案發現場 ====
〈[[案發現場|案發現場]]〉是開放[[家庭|家庭]]與[[公司|公司]]行號報名的遊戲單元，副標題為〈[[捉迷藏|躲貓貓]]〉。報名者與製作單位聯手，於報名者所在的住所範圍內「設計」好五個「找碴題」，並在一般人不會去特別注意的地方隱匿三位成員，總計八個疑點；另外，還會額外藏匿一些整人玩具來-{干}-擾參賽者。每次邀請兩位藝人或一個藝人團體參賽，參賽者稱為「探員」。只要參賽者能夠於三十分鐘內沒有完全找到這八個疑點，該家庭或公司行號即可獲得製作單位準備的獎金[[新台幣|新台幣]]六萬元。

參賽者宣稱發現疑點時，無論是否真的找到疑點，助理主持人都要暫停計時，直到主持人宣布繼續計時；此時參賽者必須明確說明可疑之處何在，答錯者會被主持人以[[警棍|警棍]]打[[屁股|屁股]]。每找到一個疑點，都可稱為「破案」。報名者（被隱匿的成員除外）要坐在住所內的特定位置，透過製作單位預先建置的[[閉路電視|閉路電視]]全程監看參賽者的活動情形；這個位置稱為「[[基地台|基地台]]」，基地台內絕對不會存在任何疑點。屋主也要待在基地台內，而且會與一台「心跳測量機」連線，該機器會測量出屋主每一分鐘的心跳數。參賽者接近任何一個疑點，屋主心跳數過高時，參賽者就能知道有疑點存在。每集結束前，無論參賽者有無找齊全部疑點，吳宗憲都會帶領大家一起呼口號：「案發現場躲貓貓，成功！」

「找碴題」是極為不合常理的事物，例如：
#[[電話機|電話機]]的[[數字|數字]][[按鍵|按鍵]]全部都是[[自然數|自然數]]-{“1”}-；
#[[小提琴|小提琴]]的弦數不是五根；
#[[馬鈴薯|馬鈴薯]]不是長在樹上；
#[[獎盃|獎盃]]內不該盛著[[湯|湯]]汁……等。

====我的秘密情人====
2006年4月9日開播。開放一位年輕觀眾報名作為主角，男女不拘。主角的父母與特別來賓群猜，主角的男朋友或女朋友究竟是誰。

====一個都不能少====
〈一個都不能少〉是一個開放觀眾報名的單元，以及格分定淘汰制為主的遊戲。第一名參賽者登場後，由下一名參賽者同台登場。如果這個「下一名參賽者」超過觀眾給分的標準分（六十分）者，則上一名參賽者就被判定淘汰，直至最後一名參賽者為止。到該單元尾聲後，持續在舞台上的參賽者，即可獲得高額獎金。

===第四代、第五代===
自2006年7月2日起算。茲以2006年8月6日為例作說明，當日的特別來賓是六位[[黑澀會美眉|黑澀會美眉]]（[[周怡君|大牙]]、[[郭婕祈|MeiMei]]、[[詹雅文|丫頭]]、[[張筱婕|筱婕]]、[[蔡玓彤|彤彤]]、[[吳映潔|鬼鬼]]）。

====Show Time====
讓來賓們進場跳舞兼暖場，全部來賓都要跳。

====看誰最愛演====
每次-{出}-五題，每題從來賓中抽-{出}-兩位依據指定的題目來「演戲」，其他來賓分兩隊來猜正確答案。答案揭曉之後，答對者在自己的桌上加放一隻猩猩[[布偶|布偶]]，答錯者會立即被噴[[乾冰|乾冰]]作為懲罰。單元結束時，統計兩隊的布偶數目，布偶數目最少者也會被噴乾冰。

2006年8月6日的第一個題目是大牙與筱婕「誰真的在喝[[苦茶|苦茶]]」，第二個題目是大牙與筱婕「誰真的在使用[[腹部震動器|腹部震動器]]」，第三個題目是彤彤與丫頭「誰真的在摸『可愛箱』」（一個「可愛箱」裡-{面}-是空的，另一個有裝[[蚯蚓|蚯蚓]]），第四個題目是彤彤與丫頭「誰真的在搬[[磚頭|磚頭]]」（一堆磚頭是[[保麗龍|保麗龍]]做的磚頭，另一堆是真的磚頭），第五個題目是鬼鬼與MeiMei「誰真的被打腳底板」。

==== 無厘頭猜謎王 ====

===== 抄襲爭議 =====

該單元播出後，有些眼尖的網友表示，〈電視仙不仙〉有抄襲2007年由[[英國|英國]][[MTV|MTV]]製作的節目《'''[[忍天堂|忍天堂]]'''》（''Fist of Zen''，於2008年在臺灣有購得版權播出）。MTV版內的「大師」，雖然由彼得·洛（Peter Law）飾演，但與一些佈景與遊戲製作上較嚴謹些，也使彼得·洛在扮演禪大師有幾分神韻。

==== 田邊俱樂部 ====
〈田邊俱樂部〉是主持人親自「下鄉」的節目，以「親近」鄉親為主旨。單元名稱是源自台灣最長壽的[[綜藝節目|綜藝節目]]《[[五燈獎|五燈獎]]》的舊名《田邊俱樂部》。

〈田邊俱樂部〉包括下列單元：

;厝邊頭尾借過來

: 〈厝邊頭尾借過來〉是一個參賽者向當地居民借物品的遊戲。參賽者必須以比手畫腳的方式，或者以不直接說出答案的方式，使居民了解要借什麼，才有可能借到居民所擁有的物品。獲勝者，可以獲得[[獎學金|獎學金]]；反之，失敗者則什麼也沒有。
: 〈厝邊頭尾借過來〉的遊戲名稱來源，原是[[閩南語|閩南語]]「厝邊頭尾照過來」，意思是「左鄰右舍看過來」；製作單位將其中的「照」取諧音「借」，以達遊戲主旨。
