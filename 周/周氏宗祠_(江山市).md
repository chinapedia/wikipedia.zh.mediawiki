'''周氏宗祠'''位于[[浙江省|浙江省]][[江山市|江山市]][[石门镇_(江山市)|石门镇]]江郎山村下属的泉井自然村<ref>{{Cite web |url=http://jsnews.zjol.com.cn/jsxww/system/2013/05/22/016461683.shtml |title=石门镇江郎山村：江郎与须女|accessdate=2017-03-09 |author= |date=2013-05-22 |work= |publisher=江山新闻网|language=Zh-hans }}</ref>。

周氏宗祠系学坦周氏家族祠堂，据建筑风格和家谱推断为明代后期建筑<ref name="a1">{{Cite web |url=http://www.zjww.gov.cn/news/2009-04-07/173686086.shtml|title=江山市一处普查新发现周氏宗祠得到妥善保护 |accessdate=2017-03-09 |author=周晓东 |date=2009-04-07 |work= |publisher=浙江文物网|language=Zh-hans }}</ref>，至清朝[[道光|道光]]七年（1827年）完成整座宗祠建筑<ref name="a2">{{Cite web |url=http://edu.qz828.com/system/2010/12/06/010292083.shtml |title=泉井流长 |accessdate=2017-03-09 |author= |date=2010-12-06 |work=责任编辑：占奕璐 |publisher=衢州新闻网——教育频道|language=Zh-hans }}</ref>。宗祠八字型砖雕大门刻有“学坦世家”四字<ref name="a1"/><ref name="a2"/>。2008年前，宗祠沦为家俱厂加工场地，在[[第三次全国文物普查|第三次全国文物普查]]中被再度发现<ref name="a1"/>。2011年与[[泉井|泉井]]列入第六批[[浙江省文物保护单位|浙江省文物保护单位]]。
