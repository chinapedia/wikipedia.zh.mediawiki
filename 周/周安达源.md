'''周安达源'''{{SBS}}{{BBS}}（{{bd|1947年||}}），[[汉族|汉族]]，[[中华人民共和国|中华人民共和国]]政治人物、[[第十一届全国政协委员|第十一届全国政协委员]]。香港润海贸易公司总经理、山西省外商投资企业协会副会长。

== 政治经历 ==
2008年，当选第十一届全国政协委员<ref>{{cite news |title = 中国人民政治协商会议第十一届全国委员会委员名单(共2237人，2008年1月25日政协第十届全国委员会常务委员会第二十次会议通过) |url = http://www.indaa.com.cn/zt/2009qglh/zxzs/200903/t20090303_127573.html |accessdate = 2013-04-09 |work = 新华社北京2008年1月25日电 |date = 2008年1月25日 |deadurl = yes |archiveurl = https://web.archive.org/web/20130928014403/http://www.indaa.com.cn/zt/2009qglh/zxzs/200903/t20090303_127573.html |archivedate = 2013年9月28日 }}</ref>，代表[[特邀香港人士|特邀香港人士]]，分入第五十二组<ref>{{cite news |title = 全国政协信息-周安达源 |url = http://www.cppcc.gov.cn/CMS/wylibary/showJcwyxtInfoWylibary.action?tabJcwyxt.guid=11W0019420&wytype=wjwy |accessdate = 2013-04-11 |website = 全国政协 }}</ref>。2018年1月，当选[[中国人民政治协商会议第十三届全国委员会委员名单|第十三届全国政协委员]]<ref>{{cite web |url = http://sports.163.com/18/0126/06/D92BRKK400051C89.html|title=中国人民政治协商会议第十三届全国委员会委员名单 |accessdate=2018-01-26 |website = 网易 }}</ref>。

2014年成為「[[保普選反佔中大聯盟|保普選反佔中大聯盟]]」執行委員會成員。
