'''周思源'''（{{bd|1938年|4月}}，原名'''盛公正'''），男，[[浙江省|浙江]][[杭州市|杭州]]人，[[北京语言大学|北京语言大学]]汉语学院教授，著名[[红学家|红学家]]。现从事小说创作、《[[红楼梦|红楼梦]]》及古代小说研究，现当代文艺批评、中国文化研究。

== 概述 ==
* 1957年[[无锡市第一中学|无锡市第一中学]]毕业。
* 1962年[[复旦大学|复旦大学]]中文系文学专业毕业。
* [[北京市第三十一中学|北京市第三十一中学]]任语文教师。
* 1970年代末至北京语言学院（即[[北京语言大学|北京语言大学]]）。
* 现为[[北京语言大学|北京语言大学]]汉语学院教授、[[中国作家协会|中国作家协会]]会员、[[中国电影家协会|中国电影家协会]]会员、[[中国鲁迅研究会|中国鲁迅研究会]]会员、[[中国红楼梦学会|中国红楼梦学会]]常务理事、《红楼梦学刊》编委、中国中外传记文学研究会理事。

== 主要作品 ==
* 历史小说：《文明太后》上下卷、《吴大帝孙权》等
*《红楼梦魅力探秘》、《红楼梦创作方法论》等
* 学术性文章：《质疑康雍乾“盛世”》、《评亨廷顿“文明的冲突”》、《二十一世纪中国文化的世界地位》、《大观园为什么没有原型》、《论鲁迅对<孩儿塔>的评价》等
* 学术报告：《说不尽的〈红楼梦〉》、《〈红楼梦〉的创作方法》、《大观园为什么没有原型》、《二十世纪九十年代红学热点》、《正确看待康雍乾之世》、《从〈水浒传〉谈现代企业管理》等
* 策划与点评

== 家庭 ==
盛传周思源是著名[[红学大师|红学大师]][[周汝昌|周汝昌]]的儿子，是[[周汝昌|周汝昌]]前妻所生，[[周汝昌|周汝昌]]前妻被休后改嫁[[绍兴|绍兴]]人氏盛某，故起名为盛公正。[[文化大革命|文化大革命]]后，盛公正认祖归宗，更名为周思源。<ref>[《周思源看红楼》][[中华书局|中华书局]]2005年版</ref>

而有报道称周思源本人出面辟谣，否认此说法。<ref>[http://news.sctv.com/rw/rwcq/201105/t20110518_690210.shtml 周思源：我与周汝昌毫无关系 父亲并不姓“周”] {{webarchive|url=https://web.archive.org/web/20160304193850/http://news.sctv.com/rw/rwcq/201105/t20110518_690210.shtml |date=2016-03-04 }}北京晨报 2011-05-18</ref>

== 个人言论 ==
2010年5月，对于愈演愈烈的[[名人故里之争|名人故里之争]]，周思源表示“把文化当作利益的奴隶，任意支使，不仅是不健康的文化观，也是不健康的经济观。绝对不要让文化屈从于金钱，把文化变成金钱的婢女，乃至妓女”。<ref>[http://news.163.com/10/0519/14/67289TQN000146BD.html 访文化学者周思源:别把文化当作金钱的奴隶] 2010年5月19日 中国经济网</ref>
