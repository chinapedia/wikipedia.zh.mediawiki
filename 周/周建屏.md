'''周建屏'''（{{bd|1892年|8月26日|1938年|6月13日|catIdx=Z周}}<ref name="a1"/>），原名'''周宗尧'''，别名周兴唐、周兴堂等，[[祖籍|祖籍]][[江西省|江西省]][[金溪县|金溪县]]，生于[[云南省|云南省]][[宣威州|宣威州]]（今云南省[[宣威市|宣威市]]）倘塘村<ref name="a1">{{Cite web |url=http://www.zgdx.gov.cn/Article.asp?id=104936 |title=时代先驱 民族精英——记江西籍高级革命将领周建屏|accessdate=2016-06-13 |author=编辑：吴丽刚 |date=2012-07-28 |work= |publisher=中国铜都德兴网|language=Zh-hans }}</ref>，[[中国工农红军|中国工农红军]]和[[八路军|八路军]]高级将领。

== 生平 ==
周毕业于云南[[讲武堂|讲武堂]]第八期，1910年投入云南军队当兵，随部参与[[辛亥革命|辛亥革命]]，1915年[[护国战争|护国战争]]时在[[朱德|朱德]]部下任连长，后在云南军中逐渐升任至旅参谋长。1924年弃职返江西祖籍务农，不久又复出，先在[[黄埔军校|黄埔军校]]第一期学习。1926年冬<ref name="a1"/>或1927年春<ref name="a2">{{Cite web |url=http://pingshan.hebei.com.cn/system/2011/08/26/006785177.shtml |title=周建屏|accessdate=2016-06-13 |author= |date=2012-10-15 |work= |publisher=平山信息网，来源：石家庄日报|language=Zh-hans }}</ref>，由朱德介绍参加[[中国共产党|中国共产党]]。[[北伐战争|北伐战争]]期间，周在[[国民革命军第四军|国民革命军第四军]]任营长。

1927年8月1日，周率部参加[[南昌起义|南昌起义]]，后在[[三河坝战役|三河坝战役]]负伤，与部队失散。其后，周前往[[上海|上海]]，与中共中央恢复联系。1929年，周被派往[[闽浙赣苏区|闽浙赣苏区]]担任军事指挥，与[[方志敏|方志敏]]、[[邵式平|邵式平]]等人合作，打下[[景德镇|景德镇]]等多处要地，屡次击败[[国民革命军|国民革命军]]。1930年9月，周所部被扩编为[[红十军|红十军]]，周担任军长。不久，红十军被调往[[中央苏区|中央苏区]]，整编成为[[红十一军|红十一军]]，周继续担任军长。1933年10月，红军整编后，周担任红十九师师长，后又调任独立红24师师长，[[长征|长征]]开始后率部留在南方进行[[游击战|游击战]]。

1935年，周在一次战斗中负重伤，被送往上海抢救，次年伤愈后赴[[延安|延安]]，任[[抗大|抗大]]第二科科长。[[抗日战争_(中国)|抗日战争]]爆发后，周被任命为八路军[[115师|115师]][[343旅|343旅]]副旅长，参加了[[平型关战役|平型关战役]]。1937年年底，周率部开创了[[晋察冀边区|晋察冀边区]]四分区。

== 逝世、纪念 ==

1938年因旧伤复发，周建屏在[[河北省_(中華民國)|河北省]][[平山县|平山县]][[小觉镇|小觉镇]]病逝，终年46岁。遗体安葬于小觉镇烈士公墓。侵华日军曾三次毁坏周建屏墓，中国民众三次修复。1953年，迁葬于[[华北军区烈士陵园|华北军区烈士陵园]]<ref name="a3"/>。小觉镇后改名为[[建屏镇|建屏镇]]<ref name="a2"/>。

1940年，[[晋察冀边区政府|晋察冀边区政府]]在[[获鹿县|获鹿县]]（今[[鹿泉区|鹿泉区]]）牛山设立[[建屏县_(1940年)|建屏县]]。1945年，县域调整，重组[[建屏县_(1945年)|建屏县]]，至1950年代并入平山县<ref name="a2"/>。[[中华人民共和国民政部|中华人民共和国民政部]][[追认|追认]]为[[革命烈士|革命烈士]]<ref name="a1"/>。

== 家庭 ==

周建屏的女儿周彩花在江西出生后，被留于江西，与父母离散多年，至周建屏逝世未能团聚<ref name="a3">{{Cite web |url=http://sjzrb.sjzdaily.com.cn/html/2015-07/07/content_1176974.htm|title=周建屏：“要抗日到底”|accessdate=2016-06-13 |author= |date=2015-07-07 |work= |publisher=石家庄日报社数字报，稿件来源：石家庄新闻网|language=Zh-hans }}</ref>。2015年，85岁的周彩花以抗日英烈子女身份参加[[抗战胜利70周年阅兵|抗战胜利70周年阅兵]]<ref>{{Cite web |url=http://epaper.jxnews.com.cn/jxrb/html/2015-09/04/content_321502.htm |title=我所亲历的大阅兵 |accessdate=2016-06-13 |author= |date=2015-09-04 |work= |publisher=中国江西网 |language=Zh-hans |deadurl=yes |archiveurl=https://web.archive.org/web/20160808043858/http://epaper.jxnews.com.cn/jxrb/html/2015-09/04/content_321502.htm |archivedate=2016-08-08 }}</ref>。
