'''能見度'''又稱'''可見度'''，指觀察者離物體多遠時仍然可以清楚看見該物體。[[氣象學|氣象學]]中，能見度被定義為[[大氣|大氣]]的[[透明度|透明度]]，因此在氣象學裏，同一空氣的能見度在白天和晚上是一樣的。能見度的[[單位|單位]]一般為[[米|米]]或[[公里|公里]]。能見度對於[[航空|航空]]、[[航海|航海]]和陸上運輸都非常重要。

== 能見度的高低 ==
能見度通常要好，看的公里數遠，需要有下列因素：
# 剛下過[[雨|雨]]（通常是夏季[[午後雷陣雨|午後雷陣雨]]：或[[颱風|颱風]]過後「也須視情況而定」）
# [[大氣擴散條件|大氣擴散條件]]良好，[[懸浮微粒|懸浮微粒]]與污染物較容易擴散
# [[風向|風向]]與[[地形|地形]]配合
# [[風速|風速]]
# 颱風來臨前（視情況而定：但機會相對颱風剛過的高）
而低能見度出現的因素與時間，可能有下列因素所致：
# 大氣擴散條件不好，懸浮微粒與污染物不容易擴散，天空上方容易形成污染層，並可能伴隨光化污染，二次污染
# 下雨中
# 有[[霾|霾]]的時間
# [[逆溫|逆溫]]

== 能見度的報告 ==
能見度的呈現是按照[[世界氣象組織|世界氣象組織]]的數值調整指引。
