'''須崎站'''（{{jpn|j=須崎駅|hg=すさきえき|rm=Susaki eki|f=y}}）是一位於[[日本|日本]][[高知縣|高知縣]][[須崎市|須崎市]]原町一丁目、隸屬於[[四國旅客鐵道|四國旅客鐵道]]（JR四國）的[[鐵路車站|鐵路車站]]。車站編號為'''K19'''，是須崎市的中心車站，所有等級的列車均停靠，而由[[高知站|高知站]]開出的下行普通區間列車，超過半數班次以此為終站。再往南行的普通列車班次則大幅減少，以[[高知站|高知站]]開出的共29班下行普通列車，到達須崎站後只剩下7個班次（另加由本站始發往窪川的1個班次）。上行的23個普通列車班次，約三分二以高知站為總站，另外三分一則遠至[[土佐山田站|土佐山田站]]。

由於[[土佐電氣鐵道|土佐電氣鐵道]][[伊野線|伊野線]]的路面電車只能到達[[伊野站前站|伊野站前站]]（即JR 的[[伊野站_(JR四國)|伊野站]]）。因此JR列車成為了須崎市往來高知市的重要交通工具。

==車站構造==
本站為單層[[水泥|水泥]]建築。設有[[郵局|郵局]]、旅遊中心<ref>{{cite web |url=http://www.jr-eki.com/kakueki/susaki/images/kounai.gif |title=存档副本 |accessdate=2011-09-16 |deadurl=yes |archiveurl=https://web.archive.org/web/20100102104340/http://jr-eki.com/kakueki/susaki/images/kounai.gif |archivedate=2010-01-02 }}</ref>及站務室。

本站設有[[側式月台|側式月台]]及[[島式月台|島式月台]]各1個，共提供2面3線的月台配置，有效長度為6輛。其中靠近站舍的1號月台，屬長而闊的側式月台，而2、3號月台闊度則非常窄。月台和站舍過往靠平交道連接，現時已改用[[行人天橋|行人天橋]]連接。3號月台外另設有停泊區間車用側線3條；1號月台頭端向[[大間站|大間站]]方向則保留2條舊有貨物列車用軌道。

== 歷史 == 
* 1924年3月30日：隨高知線（土讚線的前身）而開業。
* 1955年：第二代站舍完成，沿用至今。
* 1987年4月1日：[[日本國鐵分割民營化|日本國鐵分割民營化]]，車站的營運由JR四國繼承。
* 2009年11月30日：須崎站前郵政局轉移至站內，改稱為「須崎站內郵政局」。
