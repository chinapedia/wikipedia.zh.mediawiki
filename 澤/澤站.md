'''澤站'''（{{jpn|j=沢駅|hg=さわえき|rm=Sawa eki|f=y}}）是位於[[日本|日本]][[長野縣|長野縣]][[上伊那郡|上伊那郡]][[箕輪町|箕輪町]]大字中箕輪的[[東海旅客鐵道|東海旅客鐵道]]（JR東海）[[飯田線|飯田線]]車站。

== 車站構造 ==
*單式月台1面1線，不可以[[列車交會|列車交會]]
*[[地上車站|地上車站]]
*[[候車區|候車區]]在上行線旁。
*由[[伊那市站|伊那市站]]管理的[[無人站|無人站]]
*在2008年9月13日前，為2面2線的車站可以列車交會。

== 車站周邊 ==
車站附近有一些住宅區。

* 箕輪北小學
* 澤簡單郵局
* [[國道153號|國道153號]]
* 箕輪町國民健康保險北部診所

== 历史 ==
* 1909年（[[明治|明治]]42年）12月28日 - 伊那電車軌道（1919年改名為[[伊那電氣鐵道|伊那電氣鐵道]]）松島（現在的伊那松島）至辰野（即是西町站）之間開業，新設。
* 1923年（[[大正|大正]]12年）3月16日 - 伊那松島至辰野之間建設新線，新設'''澤站'''，'''澤停車站'''停用。
* 1943年（[[昭和|昭和]]18年）8月1日 - 伊那電氣鐵道線國有化，成為[[日本國有鐵道|國鐵]]飯田線車站。
* 1971年（昭和46年）4月1日 - 貨物起卸服務停止，只設旅客服務。
* 1983年（昭和58年）2月24日 - 成為無人站。
* 80年代 - 月台延長至可容納4卡列車。
* 1987年（昭和62年）4月1日 - 因[[國鐵分割民營化|國鐵分割民營化]]，本站成為東海旅客鐵道的車站。
* 1996年（[[平成|平成]]8年）1月尾 - 站舍停用，新建[[候車區|候車區]]。
* 2008年（平成20年）9月14日 - 由2面2線變為1面1線。
