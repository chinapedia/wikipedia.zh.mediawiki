'''瑪麗亞·克里斯蒂娜·卡瓦列羅'''（{{lang-es|'''María Cristina Caballero'''}}）是一位因為報道有組織犯罪、腐敗和[[準軍事部隊|準軍事部隊]]而聞名的[[哥倫比亞|哥倫比亞]]記者。於1999年，她獲頒[[國際新聞自由獎|國際新聞自由獎]]。<ref name=CPJ/>

==早期生涯==
卡瓦列羅在哈韋里安納大學獲得一個通訊和新聞系學士學位<ref name=NWWP/>，並於[[哈佛大學|哈佛大學]][[約翰·F·甘迺迪政府學院|約翰·F·甘迺迪政府學院]]獲得公共管理系碩士學位。她在那時候亦是一位梅森學者。<ref name=IWMF />

卡瓦列羅於16歲時已經開始其記者生涯。當時，她開始為[[波哥大|波哥大]]報紙《共和國報》（La República）工作。<ref name=IWMF />卡瓦列羅指出，她作為一個在「大男子主義」社會中工作的女記者，起初她經常獲分配較難的工作。但是，她指出自己漸漸懂得享受記者的工作，並開始變成一個工作狂。<ref name=IWMF />

==記者生涯==
之後，她轉往哥倫比亞最大的報紙《時間》（El Tiempo）擔任調查編輯一職，並於新聞雜誌《改變》（Cambio）兼職。從1998年到2001年，她轉往周刊《一周》（Semana）擔任調查主任一職。<ref name=NWWP>{{cite web |url=http://newsweek.washingtonpost.com/postglobal/maria_cristina_caballero/2007/03/anna_politkovskayas_journalist.html |title=María Cristina Caballero |author= |date=8 March 2007 |work= |publisher=newsweek.washingtonpost.com |accessdate=29 January 2012}}</ref> 她寫的文章亦曾出現於《[[紐約時報|紐約時報]]》、《[[新聞周刊|新聞周刊]]》、[[CNN|CNN]]、《[[波士頓環球報|波士頓環球報]]》、《[[邁阿密先驅報|邁阿密先驅報]]》、以及《[[国际先驱论坛报|国际先驱论坛报]]》。<ref name=NWWP />《[[哥倫比亞新聞評論|哥倫比亞新聞評論]]》和《[[尼曼新聞基金會|尼曼報告]]》亦曾刊登其文章。<ref name=IWMF>{{cite web |url=http://iwmf.org/archive/articletype/articleview/articleid/136/maria-cristina-caballero-colombia.aspx |title=Maria Cristina Caballero, Colombia |author= |date=March 2002 |work= |publisher=[[International_Women's_Media_Foundation|International Women's Media Foundation]] |accessdate=29 January 2012 |archiveurl=https://web.archive.org/web/20101217230639/http://www.iwmf.org/archive/articletype/articleview/articleid/136/maria-cristina-caballero-colombia.aspx |archivedate=2010-12-17 |deadurl=yes }}</ref>卡瓦列羅指出至少有一打哥倫比亞政客因其報道而入獄。<ref name=IWMF />

於1997年，她獲允許訪問哥倫比亞右派準軍事部隊領袖之一[[卡洛斯·卡斯塔諾·吉爾|卡洛斯·卡斯塔諾·吉爾]]。卡瓦列羅乘坐了8小時的馬方能到達卡斯塔諾的藏身之地。<ref name=IWMF />在採訪中，卡斯塔諾第一次表示自己有意進行和談。<ref name=PBS>{{cite web |url=http://www.pbs.org/newshour/media/j_awards/caballo_profile.html |title=María Cristina Caballero |author= |date=22 November 1999 |work= |publisher=[[Online_NewsHour|Online NewsHour]] |accessdate=29 January 2012}}</ref> 

於1999年，卡瓦列羅多次收到死亡威脅。而且，當時亦有一位持槍的保安人員在其居所外等候她。<ref name=IWMF />在[[哈佛大學|哈佛大學]][[約翰·F·甘迺迪政府學院|約翰·F·甘迺迪政府學院]]院長[[約瑟夫·奈爾|約瑟夫·奈爾]]的邀請下，她離開了哥倫比亞，並移民至美國[[麻薩諸塞州|麻薩諸塞州]][[剑桥_(马萨诸塞州)|剑桥]]。<ref name=PBS />

==獎勵和表彰==
於1993年，卡瓦列羅獲《[[時代_(雜誌)|時代]]》[[華盛頓特區|華盛頓特區]]分部承認為阿爾弗雷德友好出版社資深會員。於1997年，她獲哈佛大學承認為[[尼曼資深會員|尼曼資深會員]]。於1991年，她因報道政府腐敗而獲頒哥倫比亞[[西蒙玻利瓦爾國家新聞獎|西蒙玻利瓦爾國家新聞獎]]。於1998年，她因訪問卡斯塔諾而再次獲頒此獎項。<ref name=NWWP />

於1990年，[[美洲報業協會|美洲報業協會]]頒予卡瓦列羅人權獎。<ref name=NWWP />於1999年11月，她獲[[保護記者委員會|保護記者委員會]]頒發[[国际新闻自由奖|国际新闻自由奖]]。<ref name=CPJ>{{cite web |url=http://cpj.org/awards/1999/awards.php |title=1999 Awards - Announcement |author= |date=23 November 1999 |work= |publisher=[[Committee_to_Protect_Journalists|Committee to Protect Journalists]] |accessdate=29 January 2012}}</ref><ref>{{cite web |url=http://cpj.org/awards/ |title=CPJ International Press Freedom Awards 2011 |year=2011 |publisher=Committee to Protect Journalists |accessdate=29 January 2012}}</ref>
