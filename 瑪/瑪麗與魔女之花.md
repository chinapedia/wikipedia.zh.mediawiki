《'''瑪麗與魔女之花'''》（{{lang-ja|メアリと魔女の花}}）是由[[米林宏昌|米林宏昌]]所執導，改編自英國作家[[瑪麗·史都華|瑪麗·史都華]]在1971年發表的兒童小說《小掃帚》（The Little Broomstick），在2017年夏季上映的日本電影動畫。<ref>{{cite web |url=http://www.asahi.com/articles/ASJDH5V8RJDHUCVL01M.html |script-title=ja:ジブリ出身の米林監督、来夏に長編「メアリと魔女の花」 |work=[[朝日新聞|朝日新聞]] |author=小原篤 |language=ja |date=2016年12月15日 |accessdate=2016年12月16日}}</ref><ref>{{cite video |script-title=ja:「メアリと魔女の花」予告 |date=2017年2月23日 |url=https://www.youtube.com/watch?v=ZCCifYcFBWk |medium=Youtube | publisher=[[東寶|東宝MOVIEチャンネル]] | accessdate=2017年2月24日}}</ref>

此為米林宏昌離開[[吉卜力工作室|吉卜力工作室]]後首次發表的動畫作品，負責製作的動畫工作室[[Studio_Ponoc|Studio Ponoc]]由[[西村義明|西村義明]]所成立，也是過去米林宏昌拍攝吉卜力動畫《[[回憶中的瑪妮|回憶中的瑪妮]]》的影片製作人。<ref>{{cite web |url=http://www.animatetimes.com/news/details.php?id=1481764970 |script-title=ja:アリエッティ、マーニー手掛けた米林宏昌監督の最新作、アニメ映画『メアリと魔女の花』発表！気になる公開時期も明らかに |work=Animate Times |author=古瀨敏之 |language=ja |date=2016年12月15日 |accessdate=2016年12月16日}}</ref>多名先前曾與米林和西村合作過的人士，有在此作品裡再次一同攜手，如曾擔任《回憶中的瑪妮》配樂的[[村松崇繼|村松崇繼]]負責此片的音樂創作，<ref>{{cite web |url=http://animeanime.jp/article/2016/12/16/31789.html |script-title=ja:米林宏昌監督最新作「メアリと魔女の花」ダイナミックな動きが魅力の“動”の劇場アニメ |work=Anime Anime |author=川俁綾加 |language=ja |date=2016年12月16日 |accessdate=2016年12月16日}}</ref>和西村義明在另部吉卜力動畫《[[輝耀姬物語|輝耀姬物語]]》合作過的[[坂口理子|坂口理子]]也協助這部動畫劇本方面撰寫。<ref>{{cite web |url=http://www.cinra.net/news/20161215-marytomajonohana |script-title=ja:元ジブリ・米林宏昌監督『メアリと魔女の花』夏公開、少女が空飛ぶ特報も |work=Cinra |language=ja |date=2016年12月15日 |accessdate=2016年12月16日}}</ref>

== 劇情 ==
一名紅髮少女偷走魔法王國「夜間飛行」的種子而被追殺著，最終她力量耗盡而落到地面，夜間飛行的種子因此而散落。

時光流逝，一名11歲的女孩{{nihongo|瑪麗|メアリ}}因搬家緣故，來到她奶奶{{nihongo|夏洛特|シャーロット}}的房館開始新生活，並且在新環境認識了一名叫{{nihongo|彼得|ピーター}}的少年。<ref>{{cite web|url=http://eiga-watch.com/mary-and-the-witchs-flower/ |script-title=ja:「メアリと魔女の花」のネタバレあらすじ結末 |work=Eiga Watch |language=ja |date=2017年7月21日 |accessdate=2017年8月31日}}</ref>

某天夜晚瑪麗在追逐彼得所飼養的貓咪過程當中，來到了一處森林裡。在森林中瑪麗看到了每隔7年時間才盛開一次的夜間飛行花朵，好奇之下瑪麗便摘了一朵帶回自己的房間裡。<ref name="Hochi">{{cite web |url=http://www.hochi.co.jp/entertainment/20170223-OHT1T50259.html |script-title=ja:杉咲花、映画「メアリと魔女の花」で３年ぶり声優…主役の声は初 |work=[[體育報知|體育報知]] |language=ja |date=2017年2月24日 |accessdate=2017年2月25日 |deadurl=yes |archiveurl=https://web.archive.org/web/20170224214351/http://www.hochi.co.jp/entertainment/20170223-OHT1T50259.html |archivedate=2017年2月24日 |df= }}</ref>

之後瑪麗把夜間飛行的不可思議力量用在掃帚上，她便藉著魔力掃帚飛行到藏在天空[[積雨雲|雲海]]內的魔法大學，就此展開了一段不可思議經歷。<ref name="Hochi"/><ref>{{cite web|url=http://www.oricon.co.jp/special/49834/ |script-title=ja:『メアリと魔女の花』米林宏昌監督スタジオポノック第1作アニメ予告映像 |work=[[Oricon|Oricon]] |language=ja |date=2017年2月24日 |accessdate=2017年2月25日}}</ref>

== 製作 ==
=== 過程 ===
過去米林宏昌與西村義明在製作《回憶中的瑪妮》時期，管理吉卜力工作室的[[鈴木敏夫|鈴木敏夫]]私下曾向他們透露出，吉卜力已有面臨到需要解散動畫製作部門的情況。在得知工作室的問題狀況後，西村義明曾詢問過米林宏昌在部門即將解散的局勢下，是否仍有意願繼續拍攝長篇電影動畫，對此米林宏昌向對方答覆出內心仍有想再拍一部作品的念頭。<ref>{{cite web |url=http://www.cinematoday.jp/page/N0088351 |script-title=ja:ジブリの血を受け継ぐ新作アニメに宮崎駿、高畑勲、鈴木敏夫の反応は？ |work=Cinema Today |language=ja |date=2016年12月15日 |accessdate=2016年12月31日}}</ref>《回憶中的瑪妮》上映結束後，米林宏昌與西村義明在2014年12月選擇退出吉卜力。離開吉卜力後，西村義明開始著手進行將英國作家[[瑪麗·史都華|瑪麗·史都華]]的兒童小說《小掃帚》改編成動畫之企劃。<ref name="VisualGuidePage98">{{cite book |script-title=ja:《メアリと魔女の花　ビジュアルガイド》 |publisher=角川書店 |location=日本 |isbn=978-404-105-562-5 |date=2017年7月20日 |chapter=〈{{lang|ja|東宝ラインナップ発表会}}〉|page=98頁}}</ref>

2015年因西村義明過去與吉卜力導演[[高畑勳|高畑勳]]合作的《[[輝耀姬物語|輝耀姬物語]]》被提名為[[第87屆奧斯卡金像獎|第87屆奧斯卡金像獎]][[奧斯卡最佳動畫片獎|最佳動畫作品]]因素，仍代表了吉卜力前去美國參加會場典禮。<ref name="VisualGuidePage98"/>雖然《輝耀姬物語》最後未獲獎，但當時同樣有入圍且也沒獲獎的動畫《[[馴龍高手2|馴龍高手2]]》的導演[[迪恩·戴布洛伊|迪恩·戴布洛伊]]，在獎後聚會時表示著：前來製作《馴龍高手2》的創作群都是世界上最優秀人才，背後也有願意出大筆經費的上司老闆，對他來說《馴龍高手2》才是最棒作品的以上一番發言，讓西村義明聯想到吉卜力反而要解散動畫製作部門的情況，便興起另自行成立工作室的念頭。<ref name="Ponoc">{{cite web |url=http://www.ponoc.jp/news/20161215/ |script-title=ja:全文掲載「メアリと魔女の花」製作発表会見 |work=[[Studio_Ponoc|Studio Ponoc]] |language=ja |date=2016年12月23日 |accessdate=2016年12月31日}}</ref>回到日本後西村義明開始進行有關新工作室的計劃，之後在當年4月期間成立了個人工作室[[Studio_Ponoc|Studio Ponoc]]，並邀請米林宏昌加入。<ref name="Ponoc"/>

米林宏昌成為Studio Ponoc的一份子後，西村義明將他之前所挑選的《小掃帚》介紹給米林宏昌來改編成動畫，並將此定位為雖然和過去吉卜力動畫《[[魔女宅急便|魔女宅急便]]》同樣是內容有著魔女出現，但是是個全新不同風格的魔女動畫。<ref name="Ponoc"/>在改編題材敲案後，米林宏昌想在新作品上呈現出偏向動態方面的表現，不同於過去的靜態風格作品《回憶中的瑪妮》。<ref name="mainchi">{{cite web |url=http://mainichi.jp/articles/20161216/dyo/00m/200/024000c |script-title=ja:スタジオジブリから託された思い　宮崎駿監督「覚悟を持ってやれ」 |work=[[每日新聞|每日新聞]] |language=ja |date=2016年12月16日 |accessdate=2016年12月31日}}</ref>

在過去曾與米林宏昌與西村義明共事的吉卜力主要人物方面，[[宮崎駿|宮崎駿]]得知兩人要繼續拍攝長篇動畫後便表示著請他們要抱持覺悟，<ref>{{cite web |url=http://eiga.com/news/20161215/21/ |script-title=ja:米林宏昌監督、最新作「メアリと魔女の花」で21世紀版「魔女の宅急便」目指す！ |work=Eiga |language=ja |date=2016年12月15日 |accessdate=2016年12月31日}}</ref><ref name="mainchi"/>和宮崎駿相同鈴木敏夫也回應出請他們要好好堅持下去，高畑勳則表示著請他們就當作是替他完成過去與宮崎駿想將《[[小尼莫|小尼莫]]》改編成電影動畫的念頭心願。<ref name="mainchi"/>

《瑪麗與魔女之花》完成時，米林宏昌與西村義明有回到吉卜力報告相關情況，當時宮崎駿回應著雖然沒有想看這部片的意願，但認為米林他們做了相當大的努力。鈴木敏夫個人則恭喜他們擺脫了在吉卜力招牌之下的約束，完成了一部很好的影片。而一向不喜愛奇幻題材電影的高畑勳，也對此作品抱持著不錯的印象。<ref>{{cite web |url=http://natalie.mu/eiga/news/237825 |script-title=ja:「メアリと魔女の花」米林宏昌、宮崎駿や高畑勲からのねぎらいの言葉振り返る |work=Natalie |language=ja |date=2017年6月22日 |accessdate=2017年6月24日}}</ref>

=== 主題 ===
{{quote box|quoted=true|width=25%|salign=center|quote={{nihongo|在採用動作、娛樂等要素的同時，想要弄成一部讓人「想要替瑪麗加油」，走出戲院後心裡能有著「有和瑪麗一同向前邁進」這樣感覺的作品。|アクション、エンターテインメントを取り入れつつ、「メアリのことを支え、応援していきたい」と思える、映画館を出たら「メアリと共に一歩前に進もう」と思えるような作品になればと思っています。}}|source=──米林宏昌在《瑪麗與魔女之花》發表會上對於自身所執導影片的看法。<ref name="Ponoc"/>}}
就米林宏昌而言，此影片繼承著他過去待在吉卜力的目的，就是做出適合孩子們觀賞，以及成人觀眾看了也會開心的作品。<ref>{{cite web |url=http://www.animatetimes.com/news/details.php?id=1499423491 |script-title=ja:ジブリの志を受け継ぐ米林宏昌監督が『メアリと魔女の花』を通して伝えたかった想いとは？ |work=Animate Times |language=ja |date=2017年7月7日 |accessdate=2017年8月31日}}</ref>與過去自己執導過的前兩部作品《借物少女艾莉緹》、《回憶中的瑪妮》相比之下，米林宏昌認為這三部影片的共同點都有採用到能讓當觀眾試著替劇中主角打氣時，觀眾心裡也能有感受到一同邁進的表現手法。<ref>{{cite book |script-title=ja:《メアリと魔女の花　ビジュアルガイド》 |publisher=角川書店 |location=日本 |isbn=978-404-105-562-5 |date=2017年7月20日 |chapter=〈{{lang|ja|米林宏昌 [脚本・監督]}}〉|page=84頁}}</ref>

劇本創作者之一[[坂口理子|坂口理子]]方面，當時她與其他成員所討論出，對於故事宗旨一致的方向是不想把劇情弄成是「魔法無所不能」這樣的故事，<ref>{{cite book |script-title=ja:《メアリと魔女の花　ビジュアルガイド》 |publisher=角川書店 |location=日本 |isbn=978-404-105-562-5 |date=2017年7月20日 |chapter=〈{{lang|ja|坂口理子 [腳本]}}〉|page=88頁}}</ref>相對地想傳遞出「魔法雖然什麼都辦得到，但是不是有哪裡錯了」的如此訊息。<ref>{{cite book |script-title=ja:《メアリと魔女の花　ビジュアルガイド》 |publisher=角川書店 |location=日本 |isbn=978-404-105-562-5 |date=2017年7月20日 |chapter=〈{{lang|ja|坂口理子 [腳本]}}〉|page=89頁}}</ref>

另外對於西村義明來說，作品裡包含了三個主軸。一個是瑪麗在失去藉由花所獲得的魔力後，要如何在受創的心情下成長；另一個是失去吉卜力後的米林宏昌，就如同劇中瑪麗的情境，想看著他個人會用何種方法來做出表現；最後一個是作品內容彷彿像活在當下人們的寫造，劇情裡瑪麗所經歷的冒險就像諷刺人類總是不斷重演歷史，其中紅髮魔女就像過去人們的代表，要將事物託付給像是代表著未來的瑪麗身上。<ref>{{cite web |url=http://www.yomiuri.co.jp/fukayomi/ichiran/20170807-OYT8T50004.html?page_no=2 |script-title=ja:映画『メアリと魔女の花』はニセモノなのか（後編）Page 2 |work=[[讀賣新聞|讀賣新聞]] |language=ja |date=2017年8月8日 |accessdate=2017年8月31日}}</ref>

=== 畫面 ===

動畫的[[作畫監督|作畫指導]]由稻村武志擔任。當時西村義明為了不要讓觀眾覺得作品畫面風格有過時的感覺，有向他提出在繪製角色瞳孔時要比之前吉卜力作品略為加大的請求。在瞳孔的處理步驟方面，稻村武志也多補上一種色彩來強調出瞳孔的光源反射表現。<ref name="WebRonza"/><ref name="VisualGuidePage57">{{cite book |script-title=ja:《メアリと魔女の花　ビジュアルガイド》 |publisher=角川書店 |location=日本 |isbn=978-404-105-562-5 |date=2017年7月20日 |chapter=〈{{lang|ja|稲村武志 [作画監督]}}〉|page=57頁}}</ref>在角色的前額部份儘量不把前髮劉海畫在眉毛上方，以節省耗於處理畫面處理的時間，另外透過減少繪製衣服上的皺摺，來呈現出角色的活力。<ref name="WebRonza"/><ref name="VisualGuidePage57"/>

=== 音樂和片尾曲 ===
基於原著是以英國為舞台的作品，配樂創作者[[村松崇繼|村松崇繼]]有使用[[愛爾蘭|愛爾蘭]]樂器[[錫口笛|錫口笛]]在伴奏裡頭來表現。<ref name="VisualGuidePage90">{{cite book |script-title=ja:《メアリと魔女の花　ビジュアルガイド》 |publisher=角川書店 |location=日本 |isbn=978-404-105-562-5 |date=2017年7月20日 |chapter=〈{{lang|ja|村松崇継 [音楽]}}〉|page=90頁}}</ref>其中為了表現出現實與魔法世界兩者之間不同的風貌，便藉著[[非洲|非洲]]的[[說話鼓|說話鼓]]、[[伊朗|伊朗]]的[[桑圖爾|桑圖爾]]，等其它多款異國民族樂器來呈現出。<ref name="VisualGuidePage90"/>

在配樂風格的要求上，西村義行有向村松崇繼提出過想採用可以表現出「把現實與魔法世界兩者之間作牽連」如此氣氛的樂器，對於此需求村松崇繼認為可使用[[德西馬琴|德西馬琴]]來解決。<ref name="VisualGuidePage91">{{cite book |script-title=ja:《メアリと魔女の花　ビジュアルガイド》 |publisher=角川書店 |location=日本 |isbn=978-404-105-562-5 |date=2017年7月20日 |chapter=〈{{lang|ja|村松崇継 [音楽]}}〉|page=91頁}}</ref>因在日本擅長德西馬琴的人選不多，之後另請了來自於[[美國|美國]][[北卡羅萊納州|北卡羅萊納州]]的樂手約書亞·梅西克（{{lang|en|Joshua Messick}}）協助此樂器敲奏。<ref name="VisualGuidePage91"/><ref>{{cite web |url=http://www.joshuamessick.com/blog/announcements/soundtrack-mary-witchs-flower |title=Soundtrack – Mary and The Witch’s Flower |work=Joshua Messick Hammered Dulcimer Music |language=en |accessdate=2017年6月24日}}</ref>其中約書亞剛好是米林宏昌所執導動畫《借物少女艾莉緹》的愛好者，便在巧合的因素下一同合作。<ref name="VisualGuidePage91"/>

影片片尾曲《[[RAIN_(SEKAI_NO_OWARI單曲)|RAIN]]》是由四人組搖滾樂團[[SEKAI_NO_OWARI|SEKAI NO OWARI]]所提供。<ref>{{cite web |url=http://www.oricon.co.jp/news/2092108/full/ |script-title=ja:SEKAI NO OWARI、新曲はアニメ映画『メアリと魔女の花』主題歌 |work=[[Oricon|Oricon]] |language=ja |date=2017年6月9日 |accessdate=2017年6月9日}}</ref>SEKAI NO OWARI表示著此曲目內容主要是將主角瑪麗內心情境比擬為天空的模樣來創作，而米林宏昌也肯定這是一首能在影片結尾時為觀眾內心帶來溫暖的曲子。<ref>{{cite web |url=http://www.musicvoice.jp/news/20170609065686/ |script-title=ja:セカオワ、映画「メアリと魔女の花」主題歌担当　米林監督も絶賛 |work=MusicVoice |language=ja |date=2017年6月9日 |accessdate=2017年6月9日}}</ref>

作品使用的標語為{{nihongo|「魔女，再臨。」|魔女、ふたたび。}}。<ref name="YomuriPage1">{{cite web |url=http://www.yomiuri.co.jp/fukayomi/ichiran/20170807-OYT8T50004.html?page_no=1 |script-title=ja:映画『メアリと魔女の花』はニセモノなのか（後編）Page 1 |work=[[讀賣新聞|讀賣新聞]] |language=ja |date=2017年8月8日 |accessdate=2017年8月31日}}</ref>採用依據是基於過去吉卜力已有以魔女為題材製作的動畫《[[魔女宅急便|魔女宅急便]]》，讓日本當地多數人對於一般的魔女印象都是來自於該作品，便讓標語暗蔵著含有向《魔女宅急便》挑戰的意圖。<ref name="YomuriPage1"/>

影片上映前的電視宣傳廣告部份，則有和贊助廠商[[森永乳業|森永乳業]]一同合作公開出內容為瑪麗品嚐森永乳業點心產品的數部廣告動畫。<ref>{{cite web |url=https://www.jiji.com/jc/article?k=000000087.000021580&g=prt |script-title=ja:映画『メアリと魔女の花』　森永乳業オリジナルシーンが多数登場！森永乳業オリジナル新CM　6月24日（土）より放映開始 |work=[[時事通信社|時事通信社]] |language=ja |date=2017年6月26日 |accessdate=2017年8月31日}}</ref>

== 上映 ==
此影片在正式上映前，有先在6月期間於[[東京|東京]]、[[大阪|大阪]]一帶舉行試映會。<ref>{{cite web |url=http://natalie.mu/eiga/news/235477 |script-title=ja:米林宏昌「メアリと魔女の花」のトークイベントと試写会が開催、応募受付中 |work=Natalie |language=ja |date=2017年6月6日 |accessdate=2017年6月9日}}</ref><ref>{{cite web |url=http://movie.walkerplus.com/shisyakai/9385/ |script-title=ja:『メアリと魔女の花』大阪試写会 |work=Movie Walker |language=ja |date=2017年5月18日 |accessdate=2017年6月9日}}</ref>和米林宏昌執導的前兩部電影動畫相同，《瑪麗與魔女之花》也有在電影上映期間舉辦與作品相關的美術展覽。<ref>{{cite web |url=http://www.cinemacafe.net/article/2017/05/18/49506.html |script-title=ja:米林宏昌監督最新作『メアリと魔女の花』の展覧会開催！ |work=Cinema Cafe |language=ja |date=2017年5月18日 |accessdate=2017年6月9日}}</ref>

娛樂週刊《[[綜藝_(雜誌)|綜藝]]》表示米林宏昌在這部片裡將想像力給自由奔放發揮，能使人感受到Studio Ponoc工作室具備著潛在的實力。<ref>{{cite web |url=http://variety.com/2017/film/reviews/mary-and-the-witchs-flower-review-1202588100/ |title=Film Review: 'Mary and the Witch’s Flower' |work=[[綜藝_(雜誌)|綜藝]] |author=Peter Debruge |language=en |date=2017年10月23日 |accessdate=2017年12月10日}}</ref>新聞網站[[The_Verge|The Verge]]則反應影片裡有許多和過去吉卜力作品似曾相識的元素與細節，對於較包容的吉卜力影迷來說這可以是一種致意手法；相對地較不寬容的影迷會認為這是缺乏新意的表現。<ref>{{cite web |url=https://www.theverge.com/2017/9/25/16358234/mary-and-the-witchs-flower-movie-review-anime-studio-ghibli-ponoc |title=Mary and the Witch’s Flower is everything fans want from Studio Ghibli |work=[[The_Verge|The Verge]] |author=Tasha Robinson |language=en |date=2017年9月25日 |accessdate=2017年12月10日}}</ref>動漫雙月刊《Otaku USA》認為米林宏昌在作品中雖然並不吝於刻畫角色特性，流暢的畫面搭著配村松崇繼所創作配樂也是影片裡表現很好的地方，但即使如此仍稱不上是一部傑作。其中影片工作室Studio Ponoc名稱含義裡雖然有代表著全新一天的開始，但《瑪麗與魔女之花》感覺是生根在過去吉卜力的輝煌，就像是再好的翻唱作品仍是翻唱。<ref>{{cite web |url=http://www.otakuusamagazine.com/the-ghosts-of-ghibli-haunt-mary-and-the-witchs-flower-review/ |title=The Ghosts of Ghibli Haunt Mary and the Witch’s Flower |work=Otaku USA |author=Matt Schley |language=en |date=2017年7月10日 |accessdate=2017年12月10日}}</ref>
