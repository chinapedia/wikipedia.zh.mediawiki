'''奧地利的瑪麗·路易莎'''（{{lang-de|Maria Luise von Österreich}}，{{bd|1791年|12月12日|1847年|12月17日|catIdx=L}}），原為'''奧地利女大公瑪麗·路易莎'''({{lang-de|Erzherzogin Maria Luise von Österreich}})，婚後成為'''法國皇后瑪麗·路易莎''' ({{lang-fr|impératrice Marie Louise des Français}})。1817年，受封為'''帕爾馬、皮亞琴察和瓜斯塔拉女公爵'''。

她是[[法蘭西帝國|法蘭西帝國]]皇帝[[拿破崙一世|拿破崙一世]]的第二位妻子，並因此是法蘭西帝國皇后。她是[[瑪麗·安托瓦內特|瑪麗·安托瓦內特]]之姪孫女、羅馬王[[拿破崙二世|拿破崙二世]]之母。

== 早年 ==
瑪麗·路易莎（洗禮名 '''Maria Ludovica Leopoldina Francisca Theresa Josepha Lucia'''）生於[[維也納|維也納]]，奧地利帝國皇帝[[弗朗茨二世_(神聖羅馬帝國)|弗朗茨一世]]（即神聖羅馬帝國皇帝弗朗茨二世）和第二任妻子[[兩西西里王國|兩西西里王國]]的瑪麗亞·特麗莎之女。她是[[玛丽亚·特蕾西亚|瑪麗亞·特蕾西亞]]曾孫女、外曾孫女，[[瑪麗·安托瓦內特|瑪麗·安托瓦內特]]的姪孫女、外姪孫女，[[利奧波德二世_(神聖羅馬帝國)|利奧波德二世]]的孫女，[[瑪麗亞·卡羅萊娜|瑪麗亞·卡羅萊娜]]的外孫女。

為了路易莎的婚事，路易莎的父母從小教導她多種語言，除了她的母語德語外，她還精通英語、法語、意大利語、拉丁語、西班牙語。<ref>Schom, Alan. ''Napoleon Bonaparte''. p. 548</ref>

== 法蘭西帝國皇后 ==
[[File:Marie_Louise_Empress.jpg|thumb]]
[[File:Napoleon_Marie_Louise_Marriage1.jpeg|thumb]]
1810年3月11日，18歲的奧地利女大公瑪麗·路易莎嫁給法國皇帝拿破崙一世，這是一宗政治婚姻，婚禮於1810年4月1日[[羅浮宮|羅浮宮]]舉行。新娘的父親弗朗茨希望藉此加強奧地利帝國和法蘭西帝國的連繫。拿破崙則希望通過與歐洲最爲顯赫的[[哈布斯堡王朝|哈布斯堡王朝]]聯婚確立他的合法性，並盼望有一位合法的皇位繼承人。拿破崙本來打算與俄國[[羅曼諾夫王朝|羅曼諾夫王朝]]的[[安娜·帕夫洛芙娜_(荷蘭王后)|安娜·帕芙洛娜公主]]結婚，不過被她的長兄沙皇[[亞歷山大一世_(俄國)|亞歷山大一世]]拒絕。

1811年3月20日，瑪麗·路易莎為拿破崙誕下皇子——[[拿破崙二世|弗朗索瓦·夏爾·約瑟夫·波拿巴]]（羅馬王、萊希斯塔德公爵）。當拿破崙離開法國作戰時，瑪麗·路易莎曾二度擔任法國[[攝政王|攝政王]]，分別是1812年4月－12月的俄國戰役和1813年4月－1814年1月的德國戰役。1814年，拿破崙被迫退位，流放到[[厄爾巴島|厄爾巴島]]。瑪麗·路易莎回到奧地利，兩人從此沒再見面。

拿破崙曾指出一點，比起第一任妻子[[约瑟芬·德博阿尔内|約瑟芬]]，拿破崙更喜歡瑪麗·路易莎。拿破崙聲稱雖然他曾愛過約瑟芬，但他不尊重約瑟芬，與瑪麗·路易莎不同——大概是指約瑟芬的婚外情傳聞和名聲敗壞。<ref>Markham, Felix, ''Napoleon'', p.245</ref>況且瑪麗·路易莎在政治上的能力也完全不是約瑟芬能夠相比的。
[[File:Marie_Louise_von_Österreich_Napoleon_Zweite.jpg|thumb]]]]
在1814年4月，拿破崙皇位被廢，瑪麗·路易莎帶同兒子從巴黎布洛瓦逃到維也納。1814的楓丹白露條約允許她保留其皇族身份和稱號（瑪麗·路易莎皇后陛下）、[[帕爾馬公爵|帕爾馬，皮亞琴察和瓜斯塔女公爵]]頭銜和封地，並保證她兒子的繼承權。然而，在1815年，[[維也納會議|維也納會議]]修訂了這一安排，瑪麗·路易莎只留下帕爾馬女公爵的頭銜，也不指定帕爾馬公爵的繼承者。1817年，條約規定將從女大公從波旁王室中除名。1844年確定瓜斯塔拉大公國將由[[摩德納公爵|摩德納公爵]]所繼承。

1821年，拿破崙死後4個月，瑪麗·路易莎下嫁她的情人{{Tsl|en|Adam Albert von Neipperg|奈佩格伯爵}}（Adam Albert von Neipperg，1775－1829）。這對夫婦有三個孩子，頭兩個孩子出生之後，瑪麗·路易莎才結婚：

* 阿爾貝汀（Albertine, Countess of Montenuovo，1817－1867年）
* 威廉·阿布雷契（Wilhelm Albrecht, Count of Montenuovo，1819－1895年），蒙特諾佛王子 
* 瑪蒂爾德（Mathilde, Countess of Montenuovo，1822年出生）

1834年2月17日，瑪麗·路易莎再次結婚，下嫁邦貝爾伯爵（Charles-René, Count of Bombelles，1785－1856年）。<ref>[http://www.thepeerage.com/p11161.htm The Peerage.com]</ref>  

不少人認為瑪麗·路易莎是一個有能力和智慧的[[帕爾馬公國|帕爾馬]]統治者，她曾作出多項改革。1847年，瑪麗·路易莎死於帕爾馬。

== 頭銜 ==
* 奧地利女大公、匈牙利和波希米亞公主殿下（1791-1810）
* 法蘭西帝國皇后陛下（1810－1815）
* 帕爾馬、皮亞琴察和瓜斯塔拉女公爵、奧地利女大公、匈牙利和波希米亞公主殿下（1815－1847）

路易莎其後的兩段婚姻又使她獲得'''奈佩格伯爵夫人'''（1821－1834）和'''邦貝爾伯爵夫人'''（1834－1847）的頭銜。此外，因為[[1814年巴黎條約|1814年巴黎條約]]，路易莎得以保留皇族身份。

== 佚聞 ==
路易莎從奧地利帶來了很多食譜到法國，她做的洛克福釀碎牛肉後來成為很受歡迎的料理。
