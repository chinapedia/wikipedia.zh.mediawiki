'''亚硝酸异戊酯'''（化学式：C<sub>5</sub>H<sub>11</sub>ONO）是[[异戊醇|异戊醇]]生成的[[亚硝酸酯|亚硝酸酯]]。

== 性质 ==
无色至淡黄色、青黄色具有挥发性澄清液体。有水果香味。不溶于水，能溶于[[醇|醇]]、[[醚|醚]]。遇光和空气分解。

== 制取 ==
由[[异戊醇|异戊醇]]和[[亚硝酸钠|亚硝酸钠]][[酯化|酯化]]而得。

== 用途 ==
用作[[血管扩张剂|血管扩张剂]]、[[氰化物|氰化物]]的[[解毒剂|解毒剂]]，也用于香料、药物和[[重氮化合物|重氮化合物]]的合成。在西方国家也被滥用，用于改变意识，产生快感。它可使血管扩张，引起血压降低，心动过速。机理为释放[[一氧化氮|一氧化氮]]血管舒张因子。大剂量使用可能引起[[高铁血红蛋白血症|高铁血红蛋白血症]]。
