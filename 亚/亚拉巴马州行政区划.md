[[美国|美国]][[亚拉巴马州|亚拉巴马州]]一共有67个[[县_(美国)|县]]<ref>{{cite web|url=http://www.uscounties.org/cffiles_web/counties/state.cfm?statecode=al |accessdate=2013-11-18 |title=Find A County |work=National Association of Counties |deadurl=yes |archiveurl=https://web.archive.org/web/20120902083719/http://www.uscounties.org/cffiles_web/counties/state.cfm?statecode=AL |archivedate=2012-09-02 }}</ref>。每个县都在其境内拥有一个地方级别政府，目前州界范围内的土地是逐步加入美利坚合众国的。[[美国独立战争|美国独立战争]]结束后，西[[佛罗里达半岛|佛罗里达]]通过条约割让给[[西班牙|西班牙]]，其余部分先是划归密西西比领地，之后成为亚拉巴马领地<ref name="hist1702">{{cite web |title = Alabama History Timeline, 1701-1800 |publisher = Alabama Department of Archives and History |url = http://www.archives.state.al.us/timeline/al1702.html |accessdate = 2013-11-18 |deadurl=no |archiveurl=https://web.archive.org/web/20120523200305/http://www.archives.state.al.us/timeline/al1702.html |archivedate=2012-05-23}}</ref>。该领地最早组建的一些县也存留至今，其中包括州内历史最悠久，成立于1800年6月4日的[[华盛顿县_(亚拉巴马州)|华盛顿县]]<ref name="NACO">{{cite web|url=http://www.naco.org/Template.cfm?Section=Find_a_County&Template=/cffiles/counties/state.cfm&state.cfm&statecode=AL |title=NACo - Find a county |author=National Association of Counties |accessdate=2009-07-25 |deadurl=yes |archiveurl=https://web.archive.org/web/20090725232546/http://www.naco.org/Template.cfm?Section=Find_a_County&Template=%2Fcffiles%2Fcounties%2Fstate.cfm&state.cfm&statecode=AL |archivedate=2009-07-25 }}</ref>。1814年的杰克逊堡条约打开了美国人定居亚拉巴马领地的大门，这也让新县创建的速度大幅增长。1819年，亚拉巴马州加入联邦，成为第22个州<ref>{{cite web |title = Alabama History Timeline, 1801-1860 |publisher = Alabama Department of Archives and History |url = http://www.archives.state.al.us/timeline/al1801.html |accessdate = 2013-11-18 |archiveurl=https://web.archive.org/web/20130807155441/http://www.archives.state.al.us/timeline/al1801.html |archivedate=2013-08-07 |deadurl=no}}</ref>。随着州内不同地区人口的增长以及《[[印第安人迁移法案|印第安人迁移法案]]》的生效，亚拉巴马州议会也逐步组建了多个新县<ref name="Cherokee County">{{cite web |title = Alabama Counties: Cherokee |publisher=Alabama Department of Archives and History |archivedate=2012-10-23 |deadurl=no |url = http://www.archives.state.al.us/counties/cherokee.html |accessdate = 2013-11-18 |archiveurl= https://web.archive.org/web/20121023084755/http://www.archives.state.al.us/counties/cherokee.html}}</ref>。1820年时全州县的总数为29个，到了1830年已经增加至36个，不过州内僅剩东北部和西部偏远地区仍由[[美洲原住民|美洲原住民]]所據有。1840年时州内的县已增长至49个，之后10年又增加了3个，1870年增至65个，到1903年达到67个县并已维持超过一个世纪<ref name="Foscue">Foscue, Virginia O. (1989) ''Place Names in Alabama''. Tuscaloosa, Alabama: University of Alabama Press. ISBN 0-8173-0410-X</ref>。其中最后一个成立的县是1903年2月9日建立的[[休斯敦县_(亚拉巴马州)|休斯敦县]]<ref name="NACO" />。

根据[[2010年美国人口普查|2010年美国人口普查]]的数据，亚拉巴马州平均每个县拥有7万1399人，其中[[杰斐逊县_(亚拉巴马州)|杰斐逊县]]以65万8466人高居榜首，[[格林县_(亚拉巴马州)|格林县]]则仅有9045人，是全州人口最少的县<ref name="Census FactFinder">{{cite web |title = American FactFinder |url = http://factfinder2.census.gov/faces/nav/jsf/pages/index.xhtml |publisher = U.S. Census Bureau |accessdate = 2013-11-18}}</ref>。平均每个县的面积为1958[[平方公里|平方公里]]。最大的是4118平方公里的[[鲍德温县_(亚拉巴马州)|鲍德温县]]，最小的是1386平方公里的[[埃托瓦县_(亚拉巴马州)|埃托瓦县]]<ref name="Census Gazetteer">{{cite web |title = 2010 Census U.S. Gazetteer Files |url = http://www.census.gov/geo/www/gazetteer/gazetteer2010.html |publisher = U.S. Census Bureau |accessdate = 2012-10-21 |deadurl=yes |archiveurl = https://web.archive.org/web/20121021075704/http://www.census.gov/geo/www/gazetteer/gazetteer2010.html |archivedate = 2012-10-21}}</ref>。亚拉巴马州[[宪法|宪法]]规定州内新的县至少要有1600平方公里面积，这也有效地限制了新县城创建<ref name="County size">{{cite web |title = Alabama Constitution of 1901 |publisher = Wikisource, The Free Library |archivedate=2012-10-05 |deadurl=no |archiveurl= https://web.archive.org/web/20121005052524/http://en.wikisource.org/wiki/Alabama_State_Constitution_of_1901/Initial_Constitution#Section_39 |url = http://en.wikisource.org/wiki/Alabama_State_Constitution_of_1901/Initial_Constitution#Section_39 |accessdate = 2013-11-18}}</ref>。

亚拉巴马州收益部的机动车辆分局发布了汽车车牌标准，其中以一或两位数字表明该车辆注册的县，这个数字将在下表从左至右第四列列出。前三个前缀由州历史上人口最多的县保留，其后按字母排列继续。每个牌照号码由各发放机构按顺序分配，其格式为“{{lang|en|XA1111A}}”或“{{lang|en|XXA111A}}”，具体取决于该县是以一个还是两个数字表示。如果出现数字不够用的情况则会以字母来代替，如“XXZ999Z”的后面是“XXA0A0A”<ref name="15q.net">{{cite web |url = http://www.15q.net/al.html |title = Alabama License Plates, 1969-present |accessdate = 2013-11-18 |author = Nicholson, David |publisher = License Plates of North America, 1969–present |archivedate=2012-10-15 |deadurl=no |archiveurl=https://web.archive.org/web/20121015092903/http://www.15q.net/al.html}}</ref>。

以下表格会在每一个县的名称后面列出其[[联邦资料处理标准|联邦资料处理标准]]代码（简称{{lang|en|FIPS}}代码），该代码是联邦政府用来区分各州和各县的唯一识别码<ref>{{cite web |url = http://www.itl.nist.gov/fipspubs/fip6-4.htm |title = FIPS Publish 6-4 |publisher = National Institute of Standards and Technology |accessdate = 2013-11-18 |archiveurl = https://web.archive.org/web/20130929074056/http://www.itl.nist.gov/fipspubs/fip6-4.htm |deadurl = yes |archivedate = 2013-09-29 }}</ref>。亚拉巴马州的代码是01，结合任何一个县的代码则为“{{lang|en|01XXX}}”。所有的{{lang|en|FIPS}}代码将链接至该县最近的一次人口普查数据<ref name="epa">{{cite web|url=http://www.epa.gov/enviro/html/codes/al.html |title=EPA County FIPS Code Listing |accessdate=2013-11-18 |work=US Environmental Protection Agency |archivedate=2012-10-08 |deadurl=yes |archiveurl=https://web.archive.org/web/20121008031339/http://www.epa.gov/enviro/html/codes/al.html }}</ref>。
