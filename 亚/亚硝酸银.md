'''亚硝酸银'''（{{lang-en|Silver nitrite}}）是[[亚硝酸|亚硝酸]]的[[银|银]]盐，化学式为AgNO<sub>2</sub><ref>[http://www.americanelements.com/silver-nitrite.html American elements]</ref>。它由银离子和亚硝酸根离子组成。

== 制备 ==
* 亚硝酸银可通过[[硝酸银|硝酸银]]与亚硝酸的碱金属盐反应制备<ref name=Silver />，如硝酸银与[[亚硝酸钾|亚硝酸钾]]在溶液中反应制备，生成亚硝酸银絮状沉淀<ref name="Creighton" />：
:<math>\mathrm{AgNO_3 (aq) + KNO_2 (aq) \rightarrow\ AgNO_2 \downarrow + KNO_3 (aq)}</math>

* 此外还可以用[[硫酸银|硫酸银]]与[[亚硝酸钡|亚硝酸钡]]反应制备：
:<math>\mathsf{Ag_2SO_4 + Ba(NO_2)_2 \ \xrightarrow{}\ 2AgNO_2 + BaSO_4\downarrow }</math>

== 物理性质 ==
无色或黄色晶体。[[正交晶系|正交晶系]]，[[空间群|空间群]]'''''P mmn'''''，晶胞参数''a'' = 0.3505 nm，''b'' = 0.614 nm，''c'' = 0.516 nm，Z = 2。

微溶于冷水，随温度升高溶解度增大，易重结晶。不溶于[[乙醇|乙醇]]。

== 化学性质 ==
* 在亚硝酸银溶液中通入[[氨|氨]]，蒸发得亚硝酸一氨合银：
:<math>\mathsf{AgNO_2 + NH_3 \ \xrightarrow{}\ [AgNH_3]NO_2 }</math>

* 溶于过量亚硝酸的碱金属盐溶液：
:<math>\mathsf{AgNO_2 + KNO_2 \ \xrightarrow{}\ K[Ag(NO_2)_2] }</math>

* 见光逐渐分解为银和二氧化氮<ref name="Eagleson"/>：
:<math>\mathsf{AgNO_2 \xrightarrow{hv}\ Ag + NO_2\uparrow }</math>

== 应用 ==
亚硝酸银有许多应用。著名的例子包括：
* [[苯胺|苯胺]]化合物的生产。
* 通用[[氧化剂|氧化剂]]。
* 与[[有机溴化合物|有机溴化合物]]或[[有机碘化合物|有机碘化合物]]发生[[维克托·梅耶#科学贡献|维克托·梅耶]][[亲核取代反应|亲核取代反应]]，生成[[硝基化合物|硝基化合物]]<ref name=Silver>{{cite journal|last=Kornblum, N. and Ungnade, H. E.|title=1-NITROÖCTANE|journal=Organic Synthesis|year=1958|volume=38|page=75|url=http://www.orgsyn.org/Content/pdfs/procedures/cv4p0724.pdf|accessdate=6 January 2014}}</ref>。
* [[硝基烯烃|硝基烯烃]]的合成，亚硝酸盐与[[碘|碘]]元素原位生成硝基碘化物<ref>{{cite journal|last=Waldman, Steve; Monte, Aaron, Monte; Bracey, Ann; and Nichols, David|title=One-pot Claisen rearrangement/O-methylation/alkene isomerization in the synthesis of ortho-methoxylated phenylisopropylamines|journal=Tetrahedron Letters|year=1996|volume=37|issue=44|pages=7889–7892|doi=10.1016/0040-4039(96)01807-2|url=http://www.sciencedirect.com/science/article/pii/0040403996018072|accessdate=4 January 2014}}</ref>。
