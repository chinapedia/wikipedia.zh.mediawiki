'''加拿大亚太基金会'''是在1984年依照[[加拿大国会|加拿大国会]]法案（Act of Parliament）成立，旨在研究[[加拿大|加拿大]]与[[亚洲|亚洲]]关系的独立[[非政府机构|非政府机构]]，[[总部|总部]]设在[[温哥华|温哥华]]。亚太基金会是融合知识、技能和人才为一体的综合性研究机构，並为促进加拿大与亚洲地区的对话、交流、[[安全|安全]]、[[政治|政治]]和[[社会问题|社会问题]]的提供重要政策支持。

亚太基金会也是[http://www.pecc.org/ 太平洋经济合作理事会][[委员|委员]]，以及[[亚太经济合作组织|亚太经济合作组织]]工商界[[理事会|理事会]][[理事|理事]]。

== 历史 ==
亚太基金会最初是由[[加拿大联邦政府|加拿大联邦政府]]出资5000万[[加元|加元]]作为初始资本，并先后集合其他联邦政府机构、地方政府机构和工商机构的出资。亚太基金会定于每年3月31日给[[加拿大外交部|加拿大外交部]]呈报年度报告，外交部将报告传递至国会的各个部门，报告将以英法两种官方语言呈现给民众。

== 组织治理 ==
基金会组织是遵循加拿大[[联邦政府|联邦政府]]《[http://laws.justice.gc.ca/en/A-13 亚太基金会法案]》（1985年修订版 R.S., 1985, c.A-13）所組成。基金会现任[[董事会|董事会]][[主席|主席]]是曾任[[哈佛商学院|哈佛商学院]]院长的約翰·海克特·麥克阿瑟（[[John_H._McArthur|John H. McArthur]]）[[教授|教授]]。现任[[总裁|总裁]]兼首席[[执行长|执行长]]是[[胡元豹|胡元豹]]，他同時也是[http://www.pecc.org/ 太平洋经济合作理事会]加拿大首席代表，及[[不列颠哥伦比亚大学|不列颠哥伦比亚大学]]亚洲研究院[[客座教授|客座教授]]。

== 研究学者 ==
基金会热衷研究同时，注重对下一代亚太地区研究人才的培养。包括同[[不列颠哥伦比亚大学|不列颠哥伦比亚大学]]合作的[https://web.archive.org/web/20081205200632/http://www.iar.ubc.ca/mapps 亚太政策学硕士学位]，以及每年世界范围选招三名优秀[[加拿大|加拿大]][[居民|居民]]身份的[[硕士生|硕士生]]/[[博士生|博士生]]，培养未来亚太事务研究人才。

== Jack Austin 亚太商务研究中心 ==
基金会与西蒙菲沙大学商学院合作成立了该中心，专注亚太商务和管理方面的学术研究。
