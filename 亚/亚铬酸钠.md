'''亚铬酸钠'''是一种无机化合物，化学式为NaCrO<sub>2</sub>。它在水中以[[四羟基合铬(III)酸钠|四羟基合铬(III)酸钠]]（Na[Cr(OH)<sub>4</sub>]）的形式存在。
==制备==
亚铬酸钠可由[[三氧化二铬|三氧化二铬]]或[[氢氧化铬|氢氧化铬]]与[[氢氧化钠|氢氧化钠]]反应得到<ref name=wjx>《无机化学》（第二版）下册.庞锡涛 主编.高等教育出版社. 一、铬(III)的化合物. P<sub>395~396</sub>. 2.铬(III)盐和亚铬酸盐.[[Special:网络书源/9787040053876|ISBN 978-7-04-005387-6]]</ref>。
==化学性质==
亚铬酸钠在沸水中完全水解<ref name=wjx />：
: NaCrO<sub>2</sub> + 2 H<sub>2</sub>O(沸水) → Cr(OH)<sub>3</sub> + NaOH
亚铬酸钠在碱性条件下有显著的还原性<ref name=wjx /><ref>《无机化学反应方程式手册》.曹忠良 王珍云 编.湖南科学技术出版社.第十一章 铬分族.P<sub>280</sub>.【NaCrO<sub>2</sub>】</ref>：
: 2 NaCrO<sub>2</sub> + 3 Br<sub>2</sub> + 8 NaOH → 2 Na<sub>2</sub>CrO<sub>4</sub> + 6 NaBr + 4 H<sub>2</sub>O
: 2 NaCrO<sub>2</sub> + 3 H<sub>2</sub>O<sub>2</sub> + 2 NaOH → 2 Na<sub>2</sub>CrO<sub>4</sub> + 4 H<sub>2</sub>O
: 2 NaCrO<sub>2</sub> + 3 Na<sub>2</sub>O<sub>2</sub> + 2 H<sub>2</sub>O → 2 Na<sub>2</sub>CrO<sub>4</sub> + 4 NaOH
: 2 NaCrO<sub>2</sub> + 3 NaClO + 2 NaOH → 2 Na<sub>2</sub>CrO<sub>4</sub> + 3 NaCl + H<sub>2</sub>O
