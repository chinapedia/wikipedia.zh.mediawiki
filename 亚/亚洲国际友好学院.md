'''亚洲国际友好学院'''（{{lang-id|Sekolah Tinggi Bahasa Asing Persahabatan Internasional Asia}}），简称亚院（STBA-PIA），是印尼教育部批准的唯一一所可颁发华文专业本科文凭的民办高等院校。<ref>{{Cite web |url = http://www.fmprc.gov.cn/chn//pds/gjhdq/gj/yz/1206_43/1206x2/t1001086.htm |title = 驻印尼大使刘建超走访亚洲国际友好学院、分析报和讯报 |author =  |date = 2012年12月27日 |publisher = [[中华人民共和国外交部|中华人民共和国外交部]] |language = zh |accessdate = 2016年2月5日 |quote =  }}</ref>该校由印尼苏北华社慈善与教育联谊会于2008年8月20日建立，建校资金由苏门答腊北部数个华人团体以及马来西亚[[星洲日报|星洲媒体集团]]捐助。该校获得了[[华南师范大学|华南师范大学]]教学课程及提供师资方面的协助。此外，[[中国海外交流协会|中国海外交流协会]]和广东省海外交流协会也协助该校寻找师资，所有被派往该校执教的中国讲师薪水均由中国政府支付。<ref name=zhongwuling>{{Cite web |url = http://www.thenewslens.com/post/233147/ |title = 印尼華人裡的獨特群體：在大部分地區都已「印尼化」後，為何他們仍能用流利的閩南語交談？ |author = 鍾武凌 |date = 2015年10月21日 |publisher = 當今大馬 |language = zh |accessdate = 2016年2月5日 |quote =  }}</ref>
