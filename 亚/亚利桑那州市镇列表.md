截至2010年，美国[[亚利桑那州|亚利桑那州]]共有91个[[市法团|注册成立]]的[[城市|市]]和[[镇|镇]]，这些市镇拥有[[地方自治权|地方自治权]]，可以通过[[市议会|市或镇议会]]形式组建地方政府。根据[[2010年美国人口普查|2010年美国人口普查]]的统计数字，全州居民总数为639万2017，其中住在这些市镇的有502万1810，占78.56%。全州大部分人口都生活在[[凤凰城都会区|凤凰城都会区]]，2010年的人口普查显示此都会区共有419万2887位居民，占全州人口总数的65.6%。

[[图森_(亚利桑那州)|图森]]于1877年注册成立，是亚利桑那州历史最悠久的城市，2010年3月成立的[[萨扬_(亚利桑那州)|萨扬]]则是最年轻的镇<ref name="tusayan">{{cite news |publisher=The Arizona Republic |deadurl=no |archiveurl=https://archive.is/20130102004518/http://www.azcentral.com/news/articles/2010/04/12/20100412new-arizona-town-tusayan.html?nclick_check=1 |url=http://www.azcentral.com/news/articles/2010/04/12/20100412new-arizona-town-tusayan.html |title=Competing interests divide new Arizona town |date=2010-04-12 |accessdate=2015-10-07 |first=Dennis |last=Wagner |archivedate=2013-01-02 }}</ref>。根据2010年的人口普查，州首府兼人口最多的城市[[凤凰城_(亚利桑那州)|凤凰城]]还是[[按人口排列的美国城市列表|全美人口第6多的城市]]，州内尚有其它多个城市人口数在全国居前100名，分别是图森、[[梅萨_(亚利桑那州)|梅萨]]、[[格兰岱尔|格兰岱尔]]、[[钱德勒_(亚利桑那州)|钱德勒]]、[[吉尔伯特_(亚利桑那州)|吉尔伯特]]和[[斯科茨代尔_(亚利桑那州)|斯科茨代尔]]。

== 市镇注册 ==
[[File:Tucson_Stone_Ave_year_1880.jpg|thumb]]于1877年注册成立，是亚利桑那州历史最悠久的城市，上方照片摄于1880年，是图森市中心附近的一条街道。]]
根据1912年批准的《[[亚利桑那州宪法|亚利桑那州宪法]]》，州内任何人口不少于3500的社区都可以自行组建成市或镇<ref>{{cite web |url=http://www.azleg.gov/FormatDocument.asp?inDoc=/const/13/2.htm |title=Article 13, Section 2 |publisher=Arizona State Legislature |work=Arizona Constitution |accessdate=2015-10-07 |archiveurl=https://archive.is/20120719105320/http://www.azleg.gov/FormatDocument.asp?inDoc=/const/13/2.htm |deadurl=no |archivedate=2012-07-19 }}</ref>。根据宪法，市政宪章只能由各社区自行制订并作为当地一般法，不能由[[亚利桑那州议会|州议会]]或特别法制订<ref>{{cite web |url=http://www.azleg.gov/FormatDocument.asp?inDoc=/const/13/1.htm |title=Article 13, Section 1 |publisher=Arizona State Legislature |work=Arizona Constitution |accessdate=2015-10-07 |archiveurl=https://archive.is/20120721082920/http://www.azleg.gov/FormatDocument.asp?inDoc=/const/13/1.htm |archivedate=2012-07-21 |deadurl=no }}</ref>。不过，州议会之后通过法律降低了宪法中规定的人口数下限，绝大部分社区只需有1500 人就能组建成镇，如果地处[[国家公园|国家公园]]或[[国家纪念区|国家纪念区]]方圆16公里以内，这个数字还可以下降到500<ref>{{cite web |deadurl=no |url=http://www.azleg.gov/FormatDocument.asp?inDoc=/ars/9/00101.htm&Title=9&DocType=ARS |title=Arizona Revised Statutes §9-101. Incorporation; definition |publisher=Arizona State Legislature |accessdate=2015-10-07 |archivedate=2012-07-24 |archiveurl=https://archive.is/20120724135939/http://www.azleg.gov/FormatDocument.asp?inDoc=/ars/9/00101.htm&Title=9&DocType=ARS }}</ref>。州法中还进一步规定，已有市镇周围的指定缓冲区（称为城市化区）不能再成立新的市镇<ref>{{cite web |archiveurl=https://archive.is/20120729181322/http://www.azleg.gov/FormatDocument.asp?inDoc=/ars/9/00101-01.htm&Title=9&DocType=ARS |archivedate=2012-07-29 |deadurl=no |url=http://www.azleg.gov/FormatDocument.asp?inDoc=/ars/9/00101-01.htm&Title=9&DocType=ARS |title=Arizona Revised Statutes §9-101.01. Incorporation, urbanized area |publisher=Arizona State Legislature |accessdate=2015-10-07 }}</ref>。

州法中允许社区自行组建成市或镇，其中组建成市要求至少有3000人口<ref>{{cite web |deadurl=no |archiveurl=https://archive.is/20120719004545/http://www.azleg.gov/FormatDocument.asp?inDoc=/ars/9/00281.htm&Title=9&DocType=ARS |url=http://www.azleg.gov/FormatDocument.asp?inDoc=/ars/9/00281.htm&Title=9&DocType=ARS |title=Arizona Revised Statutes §9-281. Adoption procedure; board of freeholders; election |publisher=Arizona State Legislature |accessdate=2015-10-07 |archivedate=2012-07-19 }}</ref>。亚利桑纳州的市和镇总体上拥有同等地位，但市拥有部分镇宪章中不具备的权力，主要是有关该市范围内公用事业的监管和建设<ref>{{cite web |title=Arizona Revised Statutes §9-276. Additional powers of cities |accessdate=2015-10-07 |url=http://www.azleg.gov/FormatDocument.asp?inDoc=/ars/9/00276.htm&Title=9&DocType=ARS |deadurl=no |publisher=Arizona State Legislature |archiveurl=https://archive.is/20120718100002/http://www.azleg.gov/FormatDocument.asp?inDoc=/ars/9/00276.htm&Title=9&DocType=ARS |archivedate=2012-07-18 }}</ref>。州法还允许多个相邻的镇合并，或是市将镇兼并，但不允许多个城市合并<ref>{{cite web |archiveurl=https://archive.is/20120729081546/http://www.azleg.gov/FormatDocument.asp?inDoc=/ars/9/00121.htm&Title=9&DocType=ARS |accessdate=2015-10-07 |url=http://www.azleg.gov/FormatDocument.asp?inDoc=/ars/9/00121.htm&Title=9&DocType=ARS |title=Arizona Revised Statutes §9-121. Consolidation of towns |publisher=Arizona State Legislature |archivedate=2012-07-29 |deadurl=no }}</ref><ref>{{cite web |archiveurl=https://archive.is/20120802004537/http://www.azleg.gov/FormatDocument.asp?inDoc=/ars/9/00122.htm&Title=9&DocType=ARS |archivedate=2012-08-02 |deadurl=no |url=http://www.azleg.gov/FormatDocument.asp?inDoc=/ars/9/00122.htm&Title=9&DocType=ARS |title=Arizona Revised Statutes §9-122. Unification of a city and a town |publisher=Arizona State Legislature |accessdate=2015-10-07 }}</ref>。此外，人口达到3000的镇还可以把政府形式更改成市<ref>{{cite web |archiveurl=https://archive.is/20120802210211/http://www.azleg.gov/FormatDocument.asp?inDoc=/ars/9/00271.htm&Title=9&DocType=ARS |deadurl=no |url=http://www.azleg.gov/FormatDocument.asp?inDoc=/ars/9/00271.htm&Title=9&DocType=ARS |title=Arizona Revised Statutes §9-271. Procedure for change; city officers |publisher=Arizona State Legislature |accessdate=2015-10-07 |archivedate=2012-08-02 }}</ref>。不过，仍然有部分镇在人口达到这一标准后选择不变更成市。如拥有超过20万人口的吉尔伯特就仍然是镇。

亚利桑那州在1912年加入美国联邦前就已经成立了20个市或镇。这些市镇的成立方式与之后的市镇不同，例如凤凰城就是通过亚利桑那领地议会的法案于1881年成立。<ref>{{cite web | deadurl = yes | archiveurl = https://web.archive.org/web/20140214011434/http://phoenix.gov/citygovernment/facts/history/index.html | archivedate = 2014-10-19 | url = http://phoenix.gov/CITYGOV/history.html#INC | title = Out of the Ashes: The History of the City of Phoenix | publisher = City of Phoenix | accessdate = 2014-02-14}}</ref>
