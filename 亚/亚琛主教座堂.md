'''亚琛主教座堂'''（{{lang-de|Aachener Dom}}）是位于[[德国|德国]][[亚琛|亚琛]]並由許多不同時期[[基督教|基督教]]建築藝術組合而成的[[主教座堂|主教座堂]]。其最早及最中心的部分，建成于805年，為[[查理大帝|查理大帝]]的國王行宮，建築物31米高，直徑約為32米。此教堂在[[東羅馬帝國|東羅馬帝國]]時期，為[[拜占廷|拜占廷]]及[[地中海|地中海]]地區之[[:en:Imperial_Cathedrals|國王教堂]]寫照。特別是[[義大利|義大利]][[拉韦纳|拉韦纳]]的[[聖維塔教堂|聖維塔教堂]](Basilica of San Vitale)影響[[查理大帝|查理大帝]]甚多。然而建築並非直接地複製，而是各自獨立的建築。

==世界文化遺產 ==
[[查理大帝|查理大帝]]在西元800年時在[[亚琛|亚琛]]建立了歐洲中心--[[:en:Carolingian_Empire|第二羅馬帝國]]，並在這裡建立他的皇宮，其中包括行宮教堂（瑪麗亞教堂），也就是現在的阿亨主教座堂的主要建築物中心。當[[查理大帝|查理大帝]]在西元814年過世時，這個教堂變成了埋葬查理曼的地方。從西元936年[[奥托一世_(神圣罗马帝国)|奧托一世]]在此教堂加冕起，之後的六百年，阿亨主教座堂具備完整的加冕史。這裡所保存的文物，如四大聖壇，就是中世紀以來基督教朝聖的所在。主教座堂的型態已經維持了1200年。皇帝、國王、朝聖者在這裡留下了豐富的裝潢及寶物。他所代表的[[卡洛林|卡洛林]]式建築，到今日歐洲仍是非常具代表性的文化紀念物。1978年登錄為[[德國|德國]]第一個[[世界文化遺產|世界文化遺產]]。

== 歷史 ==
很少有行宮教堂如阿亨主教座堂這樣有名。西元786年—787年間，教宗阿德理安(Hadrian)授權於查理大帝，將義大利的大理石運到阿亨。西元798教堂裡昂貴的柱列立起。西元805年在教宗[[良三世|雷奧三世（Leo III.）]]的宣布下落成。西面建築物有著最原來的主要入口大廳，在上層樓的皇帝包廂，一對旋轉樓梯塔，及兩層樓十六角形規模的巨大穹隆建築等，這些至今仍存留著。接近這棟無價的建築物，他的大廳柱列，青銅格櫺與青銅大門，顯示[[卡洛林|卡洛林]]時期工坊創意與高水準藝術。

唱诗班席于1410年建成，为[[哥特式建筑|哥特式建筑]]风格。

==查理大帝的墓 ==
[[查理大帝|查理大帝]]於西元814年1月28日駕崩，埋葬在他的瑪麗亞教堂（阿亨主教座堂），因此行宮教堂成為皇帝的埋葬地。皇帝[[腓特烈一世|腓特烈一世]]以神聖之名，將查理大帝的骨頭搬至皇帝神龕。因此埋葬的石棺放置於今日的教堂寶物室，神龕置於今日平面多邊形[[哥特式|哥特式]]唱詩廳中間。

== 羅馬-德意志君主加冕教堂 ==
奧托一世於西元936年選擇阿亨為他的加冕地，此後的六百年都是阿亨的加冕史，約有三十個國王在此加冕。教堂中的加冕寶座為卡洛林形式：由昂貴的大理石片所組成。現[[亚琛市政厅|阿亨市政廳]]建築物上的石像列示當時加冕的國王石像。

== 具意義的朝聖教堂 ==
在[[哥特式建筑|哥特式建筑]]時期，阿亨的瑪麗亞教堂為具代表性的[[朝聖|朝聖]]地。自1439年起，每隔7年都有来自欧洲特别是[[匈牙利|匈牙利]]的[[天主教|天主教]]徒前来朝圣。圆顶与西正面尖塔之间的天桥，每7年对[[东欧|东欧]]来的[[朝圣|朝圣]]团开放一次。

== 傳說 ==
阿亨主教座堂的構築，也帶來了一些有趣的傳說。

據說在[[查理大帝|查理大帝]]過世後，教堂仍未建完，但阿亨的居民希望能夠將教堂完成，但苦無經費，因此決定跟魔鬼作交易。魔鬼答應借錢給阿亨人，但也開出條件，需要第一個進教堂的人的心臟。

阿亨人拿到錢後，不想害居民，因此想到一個方法，抓一只狼放進教堂，讓魔鬼上當。後來魔鬼進教堂把狼的心臟抓出來後，才發現自己被騙了，非常生氣地匆匆離開教堂，結果手指被夾在門上。

因此進入阿亨主教座堂的大廳會看到狼的與狼的心臟的銅像，並在阿亨主教座堂的大門上可以摸到惡魔的手指。

後來魔鬼想報仇，扛了一堆大包沙，打算把阿亨給淹了，但來到阿亨的路上卻迷路，因此只好問當地一位老農婦。
老農婦一眼就看出這是魔鬼的喬裝，便騙魔鬼說阿亨還很遠很遠，無法到達。
魔鬼這次又被耍了，生氣地把手上那包沙丟下便離開。後來那包沙成了現在的[[盧斯山|盧斯山]](Lousberg)。
在盧斯山可以看到魔鬼與老農婦的銅像。
阿亨人說的lous，也可以是小聰明之意。
