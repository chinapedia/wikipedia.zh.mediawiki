'''亚铁氰化钠'''，一般为十[[水合物|水合物]]形式，化学式Na<sub>4</sub>Fe(CN)<sub>6</sub>·10H<sub>2</sub>O。

== 性质 ==

亚铁氰化钠是一种柠檬黄色[[单斜晶系|单斜晶系]]的棱形或针状结晶，溶于水，不溶于[[醇|醇]]。在空气中易[[风化|风化]]，在50～60 °C的条件下晶体会很快失去[[结晶水|结晶水]]，在更高温度下干燥生成坚硬的块状无水盐。在常温稀酸中不分解，但在沸浓酸中分解为[[氢氰酸|氢氰酸]]。可被氧化为[[铁氰化钠|铁氰化钠]]。强烈灼烧时分解为[[氰化钠|氰化钠]]、[[渗碳体|碳化铁]]和氮气。

== 制备 ==

1、焦炉气副产回收法 [[焦炉气|焦炉气]]中的[[氰化氢|氰化氢]]在填充有铁刨花的塔中用[[纯碱|纯碱]]液循环吸收，经沉降除杂得亚铁氰化钠。<ref name="zghg">中国化工产品大全 上卷，Bc151 亚铁氰化钠，页155</ref>

: <math>\rm \ 6 HCN+Fe+3Na_2CO_3 \rightarrow Na_4Fe(CN)_6+H_2\uparrow + </math><math>\rm \ 2CO_2\uparrow +2H_2O</math>

2、含氰化钠废液回收法 [[氰化钠|氰化钠]]溶液与[[氯化钙|氯化钙]]、[[硫酸亚铁|硫酸亚铁]]在加热条件下反应30～40分钟生成亚铁氰化钠。<ref name="zghg" />

: <math>\rm \ 6NaCN+FeSO_4+CaCl_2\rightarrow Na_4Fe(CN)_6+CaSO_4</math><math>\rm \ +2NaCl</math>

3、氰熔体法 [[氰熔体|氰熔体]]与[[硫酸亚铁|硫酸亚铁]]反应生成亚铁氰化钠与[[亚铁氰化钙|亚铁氰化钙]]混合溶液及其[[硫酸钙|硫酸钙]]沉淀，过滤后加入纯碱使亚铁氰化钙转化为亚铁氰化钠，过滤除去[[碳酸钙|碳酸钙]]，滤液经蒸发、结晶、分离、干燥得亚铁氰化钠成品。<ref name="zghg" />

: <math>\rm \ 4Ca(CN)_2+4NaCN+2FeSO_4 \rightarrow Ca_2Fe(CN)_6+</math><math>\rm \ Na_4Fe(CN)_6+2CaSO_4\downarrow</math>
: <math>\rm \ Ca_2Fe(CN)_6+2Na_2CO_3\rightarrow Na_4Fe(CN)_6+2CaCO_3\downarrow</math>

== 用途 ==

用作制造蓝色颜料的原料，用于油漆、涂料和油墨。医药工业中用作去铁剂。
