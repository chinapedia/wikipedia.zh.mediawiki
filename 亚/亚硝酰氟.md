'''亚硝酰氟'''（[[氮|N]][[氧|O]][[氟|F]]）是一种[[共价键|共价型]]的[[亚硝酰|亚硝酰]]化合物。

== 制法 ==
*氟与一氧化氮直接反应：

:2 NO + F<sub>2</sub> → 2 NOF

*[[氮氧化物|氮氧化物]]用[[二氟化银|二氟化银]]、[[氟化铯|氟化铯]]等试剂氟化：

:NO + AgF<sub>2</sub> → NOF + AgF
:N<sub>2</sub>O<sub>4</sub> + CsF → NOF + CsNO<sub>3</sub><ref name="无机化学丛书">{{cite book|title=《无机化学丛书》第四卷：氮、磷、砷分族|pages=P162-163|author=张青莲|isbn=7-03-002238-6|location=北京|publisher=科学出版社}}</ref>

*用纯净的氟硼酸亚硝酰与干燥的[[氟化钠|氟化钠]]反应在300°C下反应：

:NOBF<sub>4</sub> + NaF → NaBF<sub>4</sub> + NOF

== 反应 ==
NOF是一种十分活泼的[[氟化剂|氟化剂]]，能将多种[[金属|金属]]转化为它们的[[氟化物|氟化物]]，并释放出[[一氧化氮|一氧化氮]]：

:''n'' NOF + M → MF<sub>''n''</sub> + ''n'' NO

NOF也能与路易斯酸反应产生类似于盐的加合物，例如[[氟硼酸亚硝酰|NOBF<sub>4</sub>]]和[[八氟合氙(VI)酸亚硝酰|(NO)<sub>2</sub>XeF<sub>8</sub>]]。

-78°C的真空条件下，亚硝酰氟能与[[一氟化氯|一氟化氯]]形成加合物：
:NOF + ClF → NO<sup>+</sup>ClF<sub>2</sub><sup>-</sup>

产物是白色固体，在-110°C以下稳定，一旦受热又分解成原来的反应物。类似加合物还有NO<sup>+</sup>ClF<sub>4</sub><sup>-</sup>，它比NO<sup>+</sup>ClF<sub>2</sub><sup>-</sup>略稳定。<ref name="无机化学丛书"/>

亚硝酰氟的水溶液能溶解金属，其作用机理与[[王水|王水]]类似。亚硝酰氟先与水反应产生[[亚硝酸|亚硝酸]]和[[氢氟酸|氢氟酸]]，然后亚硝酸歧化分解产生[[硝酸|硝酸]]和一氧化氮：

:NOF + H<sub>2</sub>O → HNO<sub>2</sub> + HF

:3 HNO<sub>2</sub> → HNO<sub>3</sub> + 2 NO + H<sub>2</sub>O

亚硝酰氟也能将[[醇|醇]]转化为[[亚硝酸酯|亚硝酸酯]]<ref>{{Greenwood&Earnshaw}}</ref>：

:ROH + NOF → RONO + HF

== 用途 ==
亚硝酰氟在[[有机合成|有机合成]]中用作[[溶剂|溶剂]]、[[氟化剂|氟化剂]]和[[硝化试剂|硝化试剂]]。例如可由酸甾胆酯制备5a-氟代甾酮，从环烯制ω-氰基醛。它也在[[火箭推进剂|火箭推进剂]]中用作氧化剂。
