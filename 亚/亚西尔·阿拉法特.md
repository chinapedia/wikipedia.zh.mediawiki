'''拉赫曼·阿卜杜勒·拉乌夫·阿拉法特·古德瓦·侯赛尼'''（{{lang-ar|'''محمد ياسر عبد الرحمن عبد الرؤوف عرفات القدوة الحسيني'''}}，{{bd|1929年|8月24日|2004年|11月11日|catIdx=A}}），通称'''亚西尔·阿拉法特'''（{{lang-ar|'''ياسر عرفات'''}}）或他的[[阿拉伯人名#阿拉伯人名的结构#昵称|昵称]]（كنية；Kunya） '''阿布·阿马尔'''（{{lang-ar|أبو عمار}}，{{transl|ar|'Abū `Ammār}}），[[巴勒斯坦解放运动|巴勒斯坦解放运动]]领袖，[[巴勒斯坦|巴勒斯坦]]领导人，[[巴勒斯坦解放组织|巴勒斯坦解放组织]]主席及巴解组织最大派别[[法塔赫|法塔赫]]的领导人，1994年[[巴勒斯坦民族权力机构|巴勒斯坦民族权力机构]]成立后任主席。他是1994年[[诺贝尔和平奖|诺贝尔和平奖]]的获得者之一。

== 生平 ==
阿拉法特的父亲是个商人，共有七个孩子。在[[开罗大学|开罗大学]]发现的出生证上说阿拉法特于1929年8月24日生在[[埃及|埃及]][[开罗|开罗]]。

阿拉法特小时候大多住在开罗，除丧母初期（5岁—9岁）时住在耶路撒冷。他毕业于开罗大学，专业为土木工程。1950年代，阿拉法特加入埃及军队，1956年[[苏伊士运河危机|苏伊士运河危机]]时为埃及作战。建立了活跃的[[法塔赫|法塔赫组织]]之后，1969年2月3日，他被选为[[巴解组织|巴解组织]]领导人，他主張把猶太人從阿拉伯土地上連根拔起，並於1970年代至1980年代策劃多次恐怖襲擊，巴解組織備受國際譴責，譴責鄙視阿拉法特放棄恐怖襲擊手段，改為以和平手段和以色列談判，1990年，阿拉法特与他以前的秘书27岁的[[蘇哈·阿拉法特|苏哈·塔维尔]]（Suha Tawil）结婚。苏哈原来为[[基督徒|基督徒]]，结婚前[[改宗|改宗]][[伊斯兰教|伊斯兰教]]，成為[[穆斯林|穆斯林]]。阿拉法特夫妇1995年生养一个女儿扎赫瓦（Zahwa，意为希望和美丽，从阿拉法特母亲之名）。[[以色列|以色列]]开始攻击[[拉马拉|拉马拉]]以后，妻女移居[[法国|法国]][[巴黎|巴黎]]。苏哈后来加入法国国籍（也有说法是苏哈本来就有法国籍，因为她的父母都是法国的巴勒斯坦裔银行家和富商），搬到[[突尼斯|突尼斯]]。虽然一般認为阿拉法特夫妇因為意見不合而分居，阿拉法特仍然供养苏哈非常奢侈的生活。

=== 建立法塔赫组织 ===
[[苏伊士运河危机|苏伊士运河危机]]之后，阿拉法特搬到[[科威特|科威特]]做工程师。在科威特时，他帮助建立法塔赫组织，以独立的巴勒斯坦为目标。1963年，[[叙利亚|叙利亚]]雇佣法塔赫组织做它的第一次军事行动，计划在1964年12月炸毁一个以色列的水泵。这个行动虽然失败，但是引起以色列官方的注意。1967年[[六日战争|六日战争]]结束之后，以色列开始留心防备各种巴解组织，包括法塔赫。 

法塔赫在失去西岸总部之后，搬到了[[约旦|约旦]]。法塔赫势力增长飞快，在约旦领土建立了自己的国中国，控制了多处战略要地，包括扎尔卡（Az Zarq）附近的炼油厂。1970年6月开始与政府公开作战。并策划刺杀约旦国王[[侯赛因·宾·塔拉勒|侯赛因·宾·塔拉勒]]，侯赛因国王因被保镖推入路边水沟，并用身体舍命保护，幸免於难。事后，虽然其他[[伊斯蘭國家|伊斯蘭國家]]尽力调和，同年9月劫机并摧毁三架国际班机之后，约旦国王侯赛因于9月16日宣布[[军令状|军令状]]，下令驱逐法塔赫。当日阿拉法特成为巴勒斯坦解放军总司令。约旦内战开始。

在[[苏联|苏联]]支持下，[[叙利亚|叙利亚]]出兵支援巴解组织，派出200多辆[[坦克|坦克]]入侵约旦。为防止约旦受到前苏联支持的叙利亚入侵，约旦向时任美国总统[[尼克森|尼克森]]求助，尼克森遂命令[[美国第六舰队|美国第六舰队]]由训练状态转换战斗状态，并抵近至[[地中海|地中海]]东部，距约旦仅50[[海里|海里]]处，并向前苏联发出[[核威胁|核威胁]]警告。与约旦还处于战争状态的以色列，在美国的要求和约旦的默许下，出动4架[[F-4|幽靈II戰鬥機]]低空飞越叙利亚坦克编队示威（但没有开火）。发现受到以色列空军威胁，刚在六日战争遭到毁灭性打击的叙利亚军队立刻调头撤退，后表示叙利亚只想调解约旦的冲突，并不打算入侵约旦这样一个阿拉伯兄弟国家。自此，约旦军队得以集中坦克、[[飞机|飞机]]以及所有[[重型武器|重型武器]]打击巴解组织，巴解组织力战不敌。到9月24日，约旦军队获得勝利，逼迫阿拉法特签订[[停火协议|停火协议]]。整个冲突中，巴解组织一半以上成员被打死，被[[黎巴嫩|黎巴嫩]]好心接收的残部，没有足够的人手抬伤员，需要黎巴嫩[[空姐|空姐]]协助抬伤员下飞机。这个月份在约旦历史上称为“[[1970年约旦内战|黑色九月]]”。

=== 黎巴嫩 ===
离开约旦之后，阿拉法特把巴解组织搬到黎巴嫩。他的职务继续提升，1973年成为巴解组织组织政委会首脑。黎巴嫩中央政府懦弱無力情況下，巴解完全自由活动。以色列称它的领域为“法塔赫区”。

黎巴嫩基督教馬龍派势力对巴解组织的恐怖主義行為日益不满，终于被[[慕尼黑慘案|慕尼黑事件]]引爆，1975年[[黎巴嫩内战|黎巴嫩内战]]爆发，促使敘利亞和以色列全面介入黎巴嫩。直到1981年，受黎巴嫩基督教势力要求，以色列國防部長[[阿里埃勒·沙龙|阿里埃勒·沙龙]]率以色列军队武装干涉前，内战共导致黎巴嫩穆斯林、黎巴嫩基督教民兵、巴解组织共将近40,000名武装人员伤亡；非武装人员和平民伤亡则不计其数。有“东方小巴黎”之称的黎巴嫩首都[[贝鲁特|贝鲁特]]变得满目疮痍，黎巴嫩成为最贫穷的阿拉伯国家之一。

当夏隆的部队包围贝鲁特后，黎巴嫩穆斯林武装开始转变态度，配合以军作战，提供阿拉法特指挥所的位置和其他重要情报，最终导致巴解组织伤亡惨重，被迫同意撤出黎巴嫩。

=== 巴勒斯坦民族权力机构 ===
[[File:Bill_Clinton,_Yitzhak_Rabin,_Yasser_Arafat_at_the_White_House_1993-09-13.jpg|thumb]]、[[比尔·克林顿|比尔·克林顿]]和阿拉法特於1993年9月13日达成[[奥斯陆协议|奥斯陆协议]]的一刻]]

1993年9月13日，《[[奥斯陆协议|奥斯陆协议]]》签订，准许巴勒斯坦在五年之内获得西岸和加萨加沙自治。以色列总理[[拉宾|拉宾]]，阿拉法特和以色列国外交部长[[希蒙·佩雷斯|希蒙·佩雷斯]]因此获得1994年的[[诺贝尔和平奖|诺贝尔和平奖]]。这个协定建立[[巴勒斯坦当局|巴勒斯坦当局]]，阿拉法特为第一个首脑。1995年，拉宾被暗杀之后，和平进程停滞。

1996年1月20日，巴勒斯坦大选，阿拉法特以87%投票获胜。国际观察员宣布这个选举是自由并公平，但是也有人认为大多数反对阿拉法特的组织决定不参加竞选，这个竞选并非真正的民主选举。巴勒斯坦民族权力机构首脑的名词是阿拉伯语ra'is（或称“头”，在中文中一般译作“主席”，在英语中有两种翻译法。以色列资料一般翻译为“Chairman”（主席），美国一般同以色列。而巴勒斯坦文件译为“President”（总统/主席），联合国使用这个词汇）。

== 去世 ==
2004年10月底，阿拉法特健康出现问题，据说10月27日开会时出现呕吐，并晕倒了10分钟。起初报告是流感，但此说法存在激烈争论。之后，巴勒斯坦和突尼斯医生进入他的住处进行病因诊断。10月29日，又有约旦和埃及的医生加入诊断医疗组。有一名医生认为他患的是血液病[[特发性血小板减少性紫癜|特发性血小板减少性紫癜]]（ITP），但其他医生觉得这个诊断并不能解释他所有的病状。

这些醫療報告发布之后，以色列政府宣告阿拉法特可以离开住处寻医。阿拉法特开始不愿走，因为以色列政府曾经宣布他走了之后无法再回巴勒斯坦。以色列政府再次宣布如果因健康问题离开之后，准许回国。10月29日，阿拉法特飞到法国巴黎治疗。11月11日去世，并被安葬在[[拉姆安拉|拉姆安拉]]。

==死因爭議==
2012年7月3日据媒体报道，瑞士一家实验室的检验结果显示，阿拉法特可能死于高剂量放射性[[钋|钋]]中毒（见[http://www.skynews.com.au/topstories/article.aspx?id=768071&vId=]及[http://www.aljazeera.com/programmes/whatkilledarafat/2012/07/20127383653774794.html])。

瑞士Radiophysique研究所在阿拉法特穿过内衣上发现的钋-210含量足以杀死20人。该研究所主任[[弗朗索瓦·博查德|弗朗索瓦·博查德]]（Francois Bochod）说：“我可以证实，我们在沾有阿拉法特体液的衣物上发现了无法解释的大量钋-210。”

这些衣物是阿拉法特的遗孀交给[[半岛电视台|半岛电视台]]的，半岛电视台随后又将它们送到Radiophysique研究所进行检测。

这一检测结果重新引发巴勒斯坦人对阿拉法特死因的怀疑，认为他可能死于以色列情报机构[[摩萨德|摩萨德]]的暗杀。但是由于阿拉法特去世已近8年，经过了21个半衰期，以至于检材本身都会对结果产生很大的影响，用于空白对照的两把新牙刷，就测到了21mBq/g和1.0mBq/g相去甚远的结果。因此测试结果的可信度也遭到了质疑。<ref name="阿拉法特真的死于钋中毒吗？">http://www.guokr.com/article/269207/ ，果壳网，阿拉法特真的死于钋中毒吗？。</ref>

阿拉法特死因调查委员会於2012年7月12日在约旦河西岸城市拉姆安拉举行的新闻发布会上宣布，根据现有-{zh-hans:材料; zh-hant:資料; zh-cn:材料; zh-tw:資料; zh-hk:資料; zh-sg:材料}-和阿拉法特生前临床症状可以推断他是中毒身亡。阿拉法特死因调查委员会医学组负责人、巴民族权力机构主席阿巴斯的约旦私人医生阿卜杜拉·巴希尔当天在新闻发布会上说，根据法国[[贝尔西军医院|贝尔西军医院]]提供的医学报告和阿拉法特生前临床症状推断，其并非死于某种疾病或自然死亡，而是中毒身亡。<ref name="阿拉法特死因调查委员会称阿拉法特死于中毒">http://news.ifeng.com/world/detail_2012_07/12/15982989_0.shtml ，凤凰网，阿拉法特死因调查委员会称阿拉法特死于中毒。</ref>

2012年11月27日，据媒体报导，[[巴勒斯坦|巴勒斯坦]]消息人士表示，阿拉法特于27日重新入土为安，调查人员已从他的遗体取出样本，将检验是否有毒杀迹象。<ref>{{cite web |url=http://news.chinatimes.com/world/130504/132012112701211.html |title=存档副本 |accessdate=2012-11-27 |deadurl=yes |archiveurl=https://web.archive.org/web/20121130001007/http://news.chinatimes.com/world/130504/132012112701211.html |archivedate=2012-11-30 }} 验尸完成 阿拉法特重新下葬</ref>

巴勒斯坦消息人士2013年10月24日说瑞士和俄罗斯研究人员关于巴解前领导人阿拉法特的尸检报告排除了阿拉法特死于放射性物质钋中毒的可能性。<ref>http://news.qq.com/a/20131024/017345.htm 阿拉法特尸检报告排除其死于钋中毒可能性</ref>11月16日，[[半島電台|半島電台]]報導，瑞士驗屍報告，發現其遺體中，驗出高劑量放射性[[釙|釙]]。其家屬認為，阿拉法特可能是遭毒死。<ref>{{cite news|url=http://www.bbc.co.uk/zhongwen/trad/world/2013/11/131106_middleeast_arafat_polonium.shtml|title=驗屍報告指阿拉法特「可能死於釙中毒」|publisher=[[BBC|BBC]]中文網|date=2013-11-16|accessdate=2013-11-22}}</ref>

== 个人财产 ==
阿拉法特是个富豪，但是到底有多少财产有争论。

2002年8月，[[以色列军事情报局|以色列军事情报局]]局长Aharon Ze'evi估计阿拉法特个人财产超过13亿美金（见[https://web.archive.org/web/20051203174917/http://www.haaretzdaily.com/hasen/pages/ShArt.jhtml?itemNo=197188&contrassID=1&subContrassID=0&sbSubContrassID=0])。

2003年美国商业杂志《[[福布斯|福布斯]]》估计他有3亿美金，排名为国家领导人中第六位（见[http://www.forbes.com/forbes/2003/0317/134.html] [http://www.forbes.com/2003/02/24/cz_royalslide_6.html]）。杂志社没有说明数据来源。

== FBI调查谋杀美国外交官 ==
[[美国联邦调查局|美国联邦调查局]]正在调查阿拉法特是否在1973年3月1日下令谋杀两位美国外交官。那天八个[[黑色九月|黑色九月]]恐怖组织分子闯入[[沙特阿拉伯|沙特阿拉伯]]在[[苏丹共和国|苏丹]]首都[[喀土穆|喀土穆]]的大使馆，绑架多人，包括美国大使Cleo Noel，临时代办George Curtis Moore，和[[比利时|比利时]]外交官Guy Eid。第二天这三个外交官被杀。美国国家安全部声称拥有录音，证明阿拉法特下令杀害三人，但是至今录音仍未公布。阿拉法特的代理人Saeb Erekat否认此指控<REF>http://www.maarivintl.com/index.cfm?fuseaction=article&articleID=11082</REF>。
