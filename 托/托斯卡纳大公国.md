'''托斯卡納大公國'''（[[意大利語|意大利語]]：{{lang|it|Granducato di Toscana}}；[[拉丁語|拉丁語]]：{{lang|la|Magnus Ducatus Tusciae}}）為一個在1569年至1859年間，存在於[[意大利|意大利]]中部的國家。為第一個[[廢除死刑|廢除死刑]]的主權國家。

此大公國取代之前的[[佛羅倫斯公國|佛羅倫斯公國]]，並一直維持在今意大利[[托斯卡納|托斯卡納]]地區一帶的統治。而在其北方，同屬今意大利托斯卡納地區的一部分土地，則另外建立了[[馬薩與卡拉拉公國|馬薩與卡拉拉公國]]，以及[[盧卡共和國|盧卡共和國]]（{{lang|it|Repubblica di Lucca}}）與後來建立的[[盧卡公國|盧卡公國]]。1860年，托斯卡納大公國被由[[薩丁尼亞王國|薩丁尼亞]]主導的[[意大利王國|意大利王國]]所合併。

== 建立及發展 ==
[[File:Bandiera_del_granducato_di_Toscana_(1562-1737_).gif|220px]]

托斯卡納大公國最初由[[意大利|意大利]]著名的[[美第奇家族|美第奇家族]]所統治。當時[[科西莫一世_(托斯卡納)|美第奇的科西莫一世]]經過多番戰鬥，把托斯卡納周邊的敵人消除後，於1569年建立此一大公國，並自封為托斯卡納大公。直至1737年，當時的統治者[[吉安·加斯托內·德·美第奇|吉安·加斯托內·德·美第奇]]去世，而此君亦無任何子嗣，令美第奇家族在托斯卡納的統治告終。適逢當時[[波蘭王位繼承戰爭|波蘭王位繼承戰爭]]結束，[[法蘭西王國|法國]]、[[西班牙|西班牙]]及[[薩伏依王朝|薩伏依]]聯軍在戰爭中取得勝利，參戰國協定[[洛林公爵|洛林公爵]][[弗朗茨一世_(神聖羅馬帝國)|弗朗茨·史蒂芬]]把[[洛林公國|洛林公國]]讓予當初被廢黜的波蘭國王[[斯坦尼斯瓦夫一世|萊什琴斯基]]。為了彌補他的損失，當時出現統治空位的托斯卡納大公國便成為其補償。托斯卡納大公國遂在1745年始由[[哈布斯堡王朝君主列表#哈布斯堡-洛林皇朝|哈布斯堡-洛林皇室]]的弗朗茨統治，稱弗朗切斯科一世（{{lang|it|Francesco I}}）。自此哈布斯堡掌握了此地的統治權。當弗朗切斯科一世在1765年去世後，其子[[利奧波德二世_(神聖羅馬帝國)|利奧波多一世]]繼位，直至1790年。在利奧波多一世統治期間，托斯卡納大公國成為第一個[[廢除死刑的國家或地區|廢除死刑的主權國家]]。

== 廢除 ==
1790年，利奧波多一世之子[[斐迪南三世_(托斯卡納)|費迪南多三世]]繼任為托斯卡納大公。直至1800年，[[拿破崙一世|拿破崙]]崛起，征服[[帕爾馬公國|帕爾馬公國]]。為了補償帕爾馬公爵，拿破崙於是在翌年與費迪南多三世簽署[[阿蘭胡埃斯條約|阿蘭胡埃斯條約]]（{{lang|it|Treaty of Aranjuez}}）。此條約要求費爾南多三世放棄托斯卡納大公國，轉任[[薩爾茨堡選帝侯|薩爾茨堡選帝侯]]。而在原有的土地上，建立新的[[伊特魯里亞王國|伊特魯里亞王國]]，由來自[[波旁王朝君主列表#帕爾瑪公爵(西班牙波旁王朝)|波旁家族]]分支的帕爾馬公爵[[路易一世_(伊特魯里亞)|盧多維科一世]]出任統治者。托斯卡納在1801年3月21日遭到廢除。

== 再次建立 ==
1807年，伊特魯里亞王國被拿破崙解散，原有領土被併入法國的行政區劃內，隨後的七年一直為法國直接控制。1814年，拿破崙於[[萊比錫戰役|萊比錫戰役]]被擊敗，費迪南多三世再次復位。其後在1824年，費迪南多三世的子嗣[[利奧波德二世_(托斯卡納)|利奧波多二世]]取代其位置，並維持統治至1859年。在此期間，發生了牽涉整個歐洲的[[1848年革命|1848年革命]]，托斯卡納亦受影響。1849年2月9日，托斯卡納的共和派人物建立短暫的共和統治並頒布憲法，但不久即為[[奧地利帝國|奧地利]]軍隊所鎮壓。利奧波多再次繼續其統治。

[[1848年革命|1848年革命]]失敗以後，年輕的奧國新皇帝[[弗朗茨·约瑟夫|弗朗茨·约瑟夫]]在1849年登基，鎮壓新聞自由和恢復私刑審判，將帝國帶向[[開明專制|開明專制]]的強勢統治：他任命的首相[[亚历山大·冯·巴赫男爵|巴赫]]，開始了被稱為「巴赫專制」的鐵腕治國；托斯卡納也深受影響，1852年憲法被廢除，[[利奧波德二世_(托斯卡納)|利奧波多二世]]大公失去了原有的人氣而被嫌棄。譬如書刊檢查恢復執行，並實行更嚴格的警察監督，以致所有（[[資產階級|資產階級]]）政治活動的自由都被鎮壓了。行政、司法和財政制度也改組地很有效率，但加重了財務壓力與人民的稅負。

== 消亡 ==
1859年6月24日，[[法蘭西第二帝國|法]]、[[薩丁尼亞王國|薩]]兩國聯合向奧地利帝國宣戰，即[[第二次意大利獨立戰爭|第二次意大利獨立戰爭]]。其後法國單方面向奧地利媾和，簽署[[維拉弗蘭卡停戰協定|停戰協定]]，隨後更在[[蘇黎世|蘇黎世]]的三國會議中，提出反對使用武力的條款。此舉令奧地利不能再輕易動用軍隊。亦因如此，具哈布斯堡家族血統的利奧波多二世，失去了奧地利的保護，令群眾趁機反對其統治。同年7月，利奧波多逃亡至[[維也納|維也納]]，讓位予其子[[斐迪南四世_(托斯卡納)|費迪南多四世]]。不過當此君繼承爵位時，托斯卡納已經落入薩丁尼亞的管理下，而他亦在同年8月16日被托斯卡納實際上的政府所廢黜。1859年12月，托斯卡納大公國與[[摩德納公國|摩德納公國]]及[[帕爾馬公國|帕爾馬公國]]組成[[中意大利聯合省|中意大利聯合省]]，隨後更在1860年3月被薩丁尼亞征服。自此托斯卡納大公國即告消亡。
