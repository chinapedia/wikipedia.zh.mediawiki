'''托哥巨嘴鳥'''（[[學名|學名]]：{{lang|la|''Ramphastos toco''}}），又名'''鞭苔巨嘴鳥'''，是最大的[[巨嘴鳥|巨嘴鳥]]。牠們分佈在[[南美洲|南美洲]]中部及東部半開放的環境。

== 特徵 ==
[[File:Toucan-2-4-2005.jpg|thumb]]
托哥巨嘴鳥的[[羽毛|羽毛]]主要是黑色，[[喉嚨|喉嚨]]、[[胸部|胸部]]及上尾底白色，下尾底呈紅色。牠們的[[眼睛|眼睛]]周圍呈淺藍色，外圍呈橙色。牠們的喙巨大，呈黃橙色，下部及嘴峰呈紅橙色，尖端底部黑色。喙是空心的，故其實是很輕的。[[舌頭|舌頭]]差不多與喙一樣長，形狀扁平。牠們總長55-65厘米，當中喙長20厘米，重500-860克，是最大的[[巨嘴鳥|巨嘴鳥]]。<ref name="nfss">[http://www.nfss.org/Birds/Species/Softbill/Toucans/Toco.html Toco Toucan (Ramphastos toco)<!-- Bot generated title -->]{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>雄鳥較雌鳥大，但兩者相似。雛鳥較沉色，喙也較短。

== 分佈 ==
托哥巨嘴鳥分佈在[[玻利維亞|玻利維亞]]北部及東部、[[秘魯|秘魯]]東南部、[[阿根廷|阿根廷]]北部、[[巴拉圭|巴拉圭]]東部及中部、及[[巴西|巴西]]東部及南部。另外亦有一些群落分佈在[[亞馬遜河|亞馬遜河]]下游、巴西極北的[[羅賴馬州|羅賴馬州]]及[[圭亞那|圭亞那]]的沿岸地區。牠們是[[留鳥|留鳥]]，但也有地區性的遷移。

== 棲息地及保育 ==
托哥巨嘴鳥並非如其他[[鵎鵼屬|鵎鵼屬]]般棲於[[森林|森林]]內。牠們棲息在半開放的環境，如林地、[[大草原|大草原]]等。牠們的棲息地主要是在低地，最高至[[海拔|海拔]]1750米，位於[[玻利維亞|玻利維亞]]近[[安地斯山脈|安地斯山脈]]的地區。牠們稍為普遍，故被列為[[無危|無危]]。<ref name=IUCN/>

== 行為 ==
托哥巨嘴鳥主要吃[[果實|果實]]、[[昆蟲|昆蟲]]、雛鳥及鳥蛋。牠們也會吃細小的[[鳥類|鳥類]]。牠們的長喙可以幫助捕捉獵物及採集果實，並可以剝皮，也可以嚇走[[掠食者|掠食者]]。<ref name="kidsbiology">[http://www.kidsbiology.com/animals-for-children.php?animal=Toco%20Toucan KidsBiology.com - Ramphastos toco<!-- Bot generated title -->]</ref>牠們很多時都是一對或小群一同生活的。牠們會拍打雙翼或滑翔來飛行。牠們[[繁殖|繁殖]]是有季節性的，但不同地區則會於不同月份。牠們的巢一般都是築在高樹上，並會在樹上挖穴。牠們也會將巢築在地穴或[[白蟻|白蟻]]丘中。牠們每年只會繁殖一次。雌鳥[[交配|交配]]後幾日內就會生蛋，每次生約2-4隻蛋。雙親會一同孵蛋，孵化期為17-18日。牠們特別具有保護性。

== 飼養 ==
托哥巨嘴鳥需要大量的[[果實|果實]]食物，且易於患上[[血色病|血色病]]，飼養上要特別小心。現時有保育計劃來增加牠們的數量。<ref>{{Cite web|url=http://www.nashvillezoo.org/piciformes/pdf/BreedingPMPTocoToucan.pdf|title=Toco Toucan Population Management plan|author=Riverbanks Zoo|accessdate=2008-06-28|deadurl=yes|archiveurl=https://web.archive.org/web/20070928024725/http://www.nashvillezoo.org/piciformes/pdf/BreedingPMPTocoToucan.pdf|archivedate=2007-09-28}}</ref>
