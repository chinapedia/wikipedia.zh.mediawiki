'''托克馬克'''（[[吉爾吉斯語|吉爾吉斯語]]：Токмок；[[俄語|俄語]]：Токмак）是一個位於[[吉尔吉斯斯坦|吉尔吉斯斯坦]][[楚河州|楚河州]]的北部的城市，2005年時有人口60,654人。托克馬克的海拔約816米，地理位置是北緯42度50分，東經75度17分。在城中可望见远处的皑皑的雪山。现今托克马克的居民多有[[东干人|東干人]]。<ref>{{cite web |url=http://www.mzb.com.cn/html/report/19504-1.htm |title=存档副本 |accessdate=2010-02-24 |deadurl=yes |archiveurl=https://web.archive.org/web/20150418064905/http://www.mzb.com.cn/html/report/19504-1.htm |archivedate=2015-04-18 }}</ref>在托克马克附近出土的[[斯基泰|斯基泰]]人的物品被送到了[[聖彼得堡|聖彼得堡]]與[[比什凱克|比什凱克]]的博物館裡。

托克马克位于吉尔吉斯斯坦与哈萨克斯坦的边界边上，北面是两者的界河[[楚河_(中亞)|楚河]]，托克马克距离吉尔吉斯斯坦首都[[比什凯克|比什凯克]]60公里。靠近楚河州现在的首府'''Chuy'''。是楚河州的一个区级行政单位，托克马克从2004年到2006年4月19日曾经是吉尔吉斯楚河州的首府。有[[土西鐵路|土西鐵路]]通往吉尔吉斯斯坦首都[[比什凯克|比什凯克]]。城南为新建的工业区。

== 名字 ==

由于中亚在历史上的战乱不止，各个民族之间的迁徙异常频繁，也给这块土地留下了各种各样的名字，如[[碎葉城|碎葉城]]、[[八剌沙衮|八剌沙衮]]（Balasagun）、虎思[[斡耳朵|斡耳朵]]等。

*'''[[碎叶城|碎叶城]]'''，设于唐朝[[贞观|贞观]]年间，又作素叶城、素叶水城，因其依傍[[素叶水|素叶水]]，故得此名。一度是[[安西都护府|安西都护府]]所在地。碎叶城在现在的托克马克西南，
*'''八剌沙衮'''，作为[[喀喇汗|喀喇汗]]国都城。在今托克马克以南，略微偏西处。
*'''虎思斡耳朵'''，或译虎思斡耳朵，[[西辽|西辽]]开国君主[[耶律大石|耶律大石]]建都于八剌沙衮，并将其改名为别称虎思斡耳朵，在今托克马克东南。

== 历史 ==
[[File:Burana_tower_2009.jpg|150px]]古迹]]
托克马克地处[[丝绸之路|丝绸之路]]两条干线的交汇处，古时中西商人汇集于此，东西使者的必经之路。由于地处楚河河谷的中东部，有来自附近雪山上的雪水滋养，土地肥沃，物产丰富，气候宜人，对于中世纪那些强有力的征服者来说，这块土地是颇有吸引力，曾作为[[喀喇汗国|喀喇汗国]]、[[西辽|西辽]]的都城。成吉思汗征服西辽的时候，伊斯兰教徒杀死城中守军，哲别不战而入该城<ref>{{cite web |url=http://www.unitedcn.com/01zgzz/16songjinxiayuan/new_page_1119.htm |title=存档副本 |accessdate=2010-02-24 |deadurl=yes |archiveurl=https://web.archive.org/web/20070807035955/http://www.unitedcn.com/01ZGZZ/16songjinxiayuan/new_page_1119.htm |archivedate=2007-08-07 }}</ref>。

1830年代，这里曾作为[[浩罕汗国|浩罕汗国]]的一个军事前哨，30年后，它落入[[俄罗斯帝国|俄罗斯帝国]]的手中，俄国人拆毁了堡垒，现代的城市是Mikhail Chernyayev上校在1864年5月13日兴建的。

俄罗斯帝国灭亡后，托克马克成为了[[苏联|苏联]]的一部分，1927年建市。1938年通往[[比什凯克|比什凯克]]的铁路建成。

托克馬克從2004年到2006年4月19日曾經是[[吉爾吉斯|吉爾吉斯]][[楚河州|楚河州]]的首府。

=== 附近遗址 ===
唐代[[碎叶城|碎叶城]]遗址在今托克马克东南8公里处。

[[郭沫若|郭沫若]]考證認為今日吉爾吉斯北部、[[巴爾喀什湖|巴爾喀什湖]]南面的[[楚河|楚河]]流域的[[托克馬克|托克馬克]]附近，是[[中國|中國]][[唐朝|唐朝]]著名偉大詩人[[李白|李白]]的出生地。李白的祖籍是[[隴西|隴西]][[成紀|成紀]]（現在的[[甘肅|甘肅]][[秦安|秦安]]東），在[[武則天|武則天]]長安元年可能出生在[[西域|西域]]的[[碎葉|碎葉]]。

在托克馬克南方15公里處、11世紀的[[布拉納塔|布拉納塔]]（Burana Tower）就是建在這個堡壘的附近，現在只殘存東半部的土丘。據信這堡壘可能是[[粟特|粟特]]人建立的，後來成了[[黑汗|黑汗王朝]]的首都[[八剌沙衮|八剌沙衮]]（Balasagun）与[[西遼|西遼]]國首都'''[[虎思斡耳朵|虎思斡耳朵]]'''。在托克馬克南方15公里處、11世紀的[[布拉納塔|布拉納塔]]（Burana Tower）就是建在這個堡壘的附近。
