'''香港中文大學學生會欖球會'''（[[英文|英文]]：'''The Chinese University of Hong Kong Rugby Football Club'''）為[[香港中文大學學生會|香港中文大學學生會]]轄下之學生組織。該會以於大專學界（尤以[[香港中文大學|香港中文大學]]為主）推廣英式[[欖球|欖球]]（Rugby Union）為目標，為香港中文大學之在學學生提供系統欖球訓練，並成立男女子欖球隊，參與[[香港欖球總會|香港欖球總會]]（Hong Kong Rugby Union）以及[[香港大專體育協會|香港大專體育協會]]（University Sports Federation，簡稱USF）所舉辦之大專[[七人制橄欖球|七人制]]及十五人制英式欖球比賽。

球會的男子欖球隊於2010－11、2011－12年度連續兩個球季盡攬包括渣打大專男子七人欖球邀請賽第一及第二組別盃賽冠軍、香港大專體育協會男子七人欖球賽冠軍、渣打大專男子十五人欖球聯賽盃賽冠軍在內所有香港大專界英式欖球冠軍，是史上第一間及至今唯一成為「男子四冠王」的大學欖球會；女子欖球隊於2013－2014、2014－2015、2015－2016及2016－2017年度亦連續四季奪得香港大專體育協會女子七人欖球賽冠軍，成為唯一「女子四連冠」的大學欖球會。2016－2017年度男、女子隊更成為唯一一間包攬香港大專體育協會男子及女子七人欖球賽冠軍的大學球會。在2018－2019年度，球會重奪香港大專體育協會男子七人欖球賽冠軍及女子七人欖球賽季軍。

== 球會歷史 ==
第一屆的香港中文大學學生會欖球會委員會成立可追溯至1999－2000年度，惟由於當時[[香港欖球總會|香港欖球總會]]僅舉辦[[帶式欖球|帶式欖球]]比賽，故會員大多不具備傳統英式衝撞式欖球的經驗。<ref>http://www.oocities.org/hk/cuhkrfc/com/com9900.htm</ref>

及至2006－07年度，香港欖球總會及大專體育協會開始舉行英式欖球比賽，當時心理學系一年級生黎智沖（Kelvin）剛剛從英國回港升讀中大。在校園遇上一位中大碩士生嚴嘉兒（Edith），對方構思組織中大英式欖球校隊，故邀請有打球經驗的Kelvin加入，後來Kelvin更成為新一屆中大欖球會會長。<ref>http://www.sportsoho.com/pg/magcontent/655574/</ref>

== 會徽意義 ==
[[File:9a3feb9687c6b8356cccb75fa635358128434445.jpg|thumb]]

新一代的香港中文大學學生會欖球會會徽由2007年沿用至今，當中以香港中文大學之校徽上之[[鳳凰|鳳凰]]為設計概念，配合香港中文大學之主色紫色，以及欖球形狀融合於會徽之中。

==球隊近況==
# 2011年1月9日 香港中文大學男子七人欖球隊於「渣打大專七人欖球邀請賽2010/11」宣佈衛冕成功，更連奪男子第一及第二組別盃賽冠軍，其中在第一組別盃賽決賽中，中大以24-0擊敗城市大學，隊員謝竣恆更獲選為[[最有價值球員|最有價值球員]]。<ref>{{cite web |url=http://www.hkrugby.com/node/1430 |title=存档副本 |accessdate=2011-06-15 |deadurl=yes |archiveurl=https://web.archive.org/web/20110820232647/http://www.hkrugby.com/node/1430 |archivedate=2011-08-20 }}</ref> 
# 2011年3月20日 香港中文大學男子七人欖球隊於香港大學何鴻燊體育中心舉行的「香港大專體育協會男子七人欖球賽」決賽中以 21:0擊敗「黑馬」科技大學奪得大專七人欖球界最高殊榮之「大專盃」。<ref>http://www1.hk.apple.nextmedia.com/template/apple/art_main.php?iss_id=20110321&sec_id=25391&art_id=15093658</ref>
# 2011年6月11日 香港中文大學男子十五人欖球隊於「2010/11渣打大專十五人欖球聯賽」銀盃賽－大學盃決賽中以16-0勇挫衛冕的港大，捧走「大學盃」，一嘗十五人聯賽冠軍滋味。<ref>{{cite web |url=http://www.hkrugby.com/node/1685 |title=存档副本 |accessdate=2011-06-14 |deadurl=yes |archiveurl=https://web.archive.org/web/20110821003406/http://www.hkrugby.com/node/1685 |archivedate=2011-08-21 }}</ref>
# 2012年1月8日 香港欖球總會1月8日於京士柏運動場，舉辦了渣打大專七人欖球邀請賽2012，今年共33支男、女子隊伍角逐各組別冠軍，當中包括兩支來自中國華南農業大學的邀請隊。在男子第一組別賽事，中文大學衛冕成功，在加時突然死亡時段首先成功達陣，擊敗香港大學，捧走銀盃冠軍；兩隊於法定時間內5-5平手。而中文大學II隊亦成功衛冕男子第二組別銀盃冠軍。中文大學男子隊連續三年成為第一組別銀盃冠軍，以及連續兩年成為第二組別銀盃冠軍。 <ref>{{cite web |url=http://www.hkrugby.com/node/1945 |title=存档副本 |accessdate=2012-03-10 |deadurl=yes |archiveurl=https://web.archive.org/web/20120308104926/http://www.hkrugby.com/node/1945 |archivedate=2012-03-08 }}</ref>
# 2012年1月8日 香港欖球總會1月8日於京士柏運動場，舉辦了渣打大專七人欖球邀請賽2012, 中文大學女子隊以15-0擊敗樹仁大學, 勇奪女子組銀盾, 為女隊史上首個錦標. <ref>{{cite web |url=http://www.hkrugby.com/node/1945 |title=存档副本 |accessdate=2012-03-10 |deadurl=yes |archiveurl=https://web.archive.org/web/20120308104926/http://www.hkrugby.com/node/1945 |archivedate=2012-03-08 }}</ref>
# 2012年3月18日 中大男子欖球隊衞冕成功，再次贏得香港大專體育協會欖球比賽冠軍。女子隊亦成績驕人，贏得亞軍。比賽於3月18日在沙灣徑球場舉行，中大隊員均訓練有素，英姿颯颯，比拼體育技巧與合作精神。<ref>http://www.iso.cuhk.edu.hk/chinese/publications/newsletter/article.aspx?articleid=41100</ref>
# 2012年6月17日 香港中文大學男子十五人欖球隊於「2011/12渣打大專十五人欖球聯賽」銀盃賽－大學盃決賽中以17-14再次擊敗[[香港大學|香港大學]]，成功衛冕十五人聯賽冠軍。<ref>http://www.noodls.com/viewNoodl/15014361/hong-kong-rugby-football-union/chinese-university-of-hong-kong-take-the-spoils-at-the-terti</ref>
# 2013年1月6日 香港欖球總會1月6日於京士柏運動場，舉辦了渣打大專七人欖球邀請賽2013，中文大學男子隊連負兩支分別來自新加坡國立大學及台灣長榮大學的外隊，僅取得殿軍，未能衞冕此前連奪三年的第一組別盃賽冠軍。<ref>http://www.hkrugby.com/eng/news/2799.php</ref>
# 2013年4月7日 於京士柏運動場舉辦的香港大專體育協會欖球比賽2013，因多名主力球員畢業而處於重建階段的中文大學男子隊僅取得第八名，為球隊史上在該項賽事的最低排名。<ref>http://www.hkrugby.com/downloads/USF.pdf{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>
# 2013年4月7日 於京士柏運動場舉辦的香港大專體育協會欖球比賽2013，中文大學女子隊取得殿軍，為隊史上第二個大專七人賽獎盃。<ref>http://www.hkrugby.com/downloads/USF.pdf{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>
# 2014年1月5日 「渣打大專七人欖球邀請賽2014」於2014年1月4及5日(星期六、日)假京士柏運動場順利舉行。香港中文大學女子一隊於決賽加時賽以15-10擊敗了香港理工大學成為女子組盃賽冠軍得主。男子隊方面則奪得第一組別盃賽殿軍的平上屆紀錄。<ref>{{cite web |url=http://www.hkrugby.com/downloads/Standard_Chartered_-_Tertiary_Rugby_Invitational_Rugby_2014_Updated_on_7_1_2014.xlsx |title=存档副本 |accessdate=2014-01-08 |deadurl=yes |archiveurl=https://web.archive.org/web/20140108153724/http://www.hkrugby.com/downloads/Standard_Chartered_-_Tertiary_Rugby_Invitational_Rugby_2014_Updated_on_7_1_2014.xlsx |archivedate=2014-01-08 }}</ref>
