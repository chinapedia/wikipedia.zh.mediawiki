'''香魚'''（[[學名|學名]]：''{{lang|la|Plecoglossus altivelis}}''），為知名的[[食用魚|食用魚]]，是[[胡瓜魚目|胡瓜魚目]]'''香魚科香魚屬'''的[[單種屬|唯一一種]]。又稱為國姓魚或𫙮魚。<ref>《劍花室詩集》：春水初添三四尺，登盤已薦𫙮魚香。《雅堂文集》：而台北之𫙮魚亦曰國姓魚。《苗慄縣志》：𫙮：有鱗，色白，長不滿尺。《劍花室詩集》：春水初添三四尺，登盤已薦𫙮魚香。《噶瑪蘭廳志·卷六》：𫙮魚：出淡、蘭一帶，近內山者尤多。《台灣詩鈔》：𫙮俗稱「國姓魚」，亦曰香魚。《台灣通志》：𫙮魚，方言名桀魚：巨口細鱗、無刺，形如鯔，味甚美。《淡水廳志》：鳶錘、交臘味清而佳、𫙮近內山溪澗甚多，俗呼國姓魚……</ref>

== 分布 ==

香魚分布於[[中國|中國]]、[[臺灣|臺灣]]、[[韓國|韓國]]、[[日本|日本]]，为温水性中小型[[淡水鱼|淡水鱼类]]。<ref>{{cite journal|author=[[李思忠_(鱼类学家)|李思忠]]|title=香鱼的名称、习性、分布及渔业前景|journal=动物学杂志|year=1988|issue=6|url=http://dwxzz.ioz.ac.cn/ch/reader/view_abstract.aspx?file_no=88060003}}</ref>

== 特徵 ==

香魚體延長而側扁，頭小，吻端下彎，形成鉤吻。下颌前端各有一突起，兩突起之間成明顯的凹陷。上下頷各有一行寬扁細牙，能活動，著生於皮上。犁骨無牙，顎骨和舌上具牙。全身除頭部外均備有細小圓鱗。具脂鰭，體背淡黃綠色，腹部銀白色；胸鰭後方有一鮮黃斑點。體長可達30公分左右。

== 生態 ==

香魚在其分佈區內有三個不同的生態觀，即終生生活於湖泊的「陸封型」；在湖中生活，但在繁殖期上溯到河川2至3公里處產卵的「溯河型」；以及棲息於河川上游，每年秋冬之際，順流到河川下游產卵，產卵後親魚死亡，孵化後仔魚會順游至河口，以浮游生物為食，之後就生活在河川的中上游的「降海型」。成長後的香魚會以附著在岩石上的藻類為食。具強烈的領域性，強壯的成魚會佔領一塊長滿藻類的大礫石，不准其他香魚入侵。因此釣者利用此特性，發明了「友釣法」，利用活的香魚去靠近原佔有領域的香魚，引誘原領域內的香魚去撞擊餌魚而觸發魚鉤。在台灣原產的降河型香魚相傳由[[鄭成功|鄭成功]]於[[九龍江|九龍江]]帶香魚到新店溪繁殖，故稱為國姓魚，<ref>《雅言》：「台北之𫙮魚，亦名「國姓魚」；「淡水廳志」謂鄭氏至台始有。」</ref>
因河川汙染及水庫興建而絕種，現由日本引進陸封型，香魚也是[[日本|日本]][[岐阜縣|岐阜縣]]的縣魚。

== 經濟、學術利用 ==
[[Image:Charcoal_broiled_Ayu.JPG|right]]
香魚在其分布區內形成了三種不同的生態類型，這是在[[魚類|魚類]]中是比較少的。特別是「陸封型」這一生態類型的形成，和[[地殼|地殼]]運動密切相關，通過對「陸封型」香魚的研究，可能闡明地殼運動在局部地區的歷史提供有價值的佐證。

香魚早在[[明朝|明朝]][[萬曆|萬曆]]年間就已有史料記載，曾有「雁山出香魚，清甜味有餘」的詩句，被譽為「[[雁盪山|雁山]]五珍」之一，具高價值的經濟魚類，香魚肉不多但風味絕佳，以燒烤烹調可完全激發其美味。
