'''香港新界鄧氏'''是[[香港新界原居民|香港新界原居民]]中的一個[[氏族|氏族]]，族人原居於[[中國|中國]][[江西|江西]]及[[福建|福建]]等地。[[北宋|北宋]]初期，居於[[江西|江西]][[吉安府|吉安府]][[吉水縣|吉水縣]][[白沙鎮_(吉水縣)|白沙鄉]]的[[鄧漢黻|鄧漢黻]]徙遷往[[廣東|廣東]]；[[南宋|南宋]]末年，另一支原居於[[福建|福建]][[汀州府|汀州府]][[寧化縣|寧化縣]][[石壁鄉|石壁鄉]]的族人亦由[[鄧志齋|鄧志齋]]率領在[[廣東|廣東]][[嘉應州|嘉應州]][[吉梅鄉|吉梅鄉]]開基。

==鄧漢黻==
{{link-yue|鄧漢黻}}的四世孫[[鄧符協|鄧符協]]於[[熙寧|熙寧]]二年（1069年）中[[進士|進士]]，獲授[[廣東|廣東]][[陽春縣|陽春縣]][[縣令|縣令]]。鄧符協在任期間，曾遊歷[[屯門|屯門]]、[[元朗|元朗]]一帶，因感這裡「風俗之淳，山水之勝」，於是於[[崇寧|崇寧]]三年（1103年）遷居於[[岑田|岑田]]（今[[錦田|錦田]]）定居，並遷葬三代祖先於今元朗、屯門及[[荃灣|荃灣]]。鄧符協於[[雞公嶺|圭角山]]下創辦[[力瀛書院|力瀛書院]]，讀書講學。

但根據鄧氏族譜，指早於鄧漢黻一代已親自遷居於岑田<ref>{{cite web|url=http://pingshan.com/ts-qc-Whole.html|title=鄧族遷岑的真實年代|archiveurl=https://web.archive.org/web/20090129203809/http://pingshan.com/ts-qc-Whole.html|archivedate=2009年1月29日|website=pingshan.com}}</ref>。

===鄧氏五大房===
[[鄧符協|鄧符協]]生兩子，長子名[[鄧陽|鄧陽]]，而次子名[[鄧布|鄧布]]。鄧陽生[[鄧珪|鄧珪]]，而鄧珪生子[[鄧元英|鄧元英]]和[[鄧元禧|鄧元禧]]；鄧布生[[鄧瑞_(莞城)|鄧瑞]]，而鄧瑞生子[[鄧元禎|鄧元禎]]、[[鄧元亮|鄧元亮]]和[[鄧元和|鄧元和]]。元英、元禧、元禎、元亮和元和的後人各自繁衍，並散居[[東莞|東莞]]各地（[[宋朝|宋]]、[[元朝|元]]時今[[香港|香港]]地區屬東莞），合稱「五大房」，於[[清朝|清]][[康熙|康熙]]四十六年（1707年）在[[莞城区|東莞城]]大南門建成稱為「都慶堂」的「鄧氏宗祠」。

# '''鄧元禧'''後人居東莞的福隆（福容里）；
# '''鄧元禎'''後人居[[屏山_(香港)|屏山]]、白蠔、白崗、白石及塘尾等地；
# '''鄧元亮'''後人居[[岑田|岑田]]（今[[錦田|錦田]]）；
# '''鄧元和'''後人居東莞懷德。

鄧元禎一族是香港新界鄧氏的[[屏山鄧氏|屏山系]]，而鄧元亮一族則由[[錦田鄧氏|錦田系]]分支到[[廈村鄧氏|廈村]]、[[龍躍頭鄧氏|龍躍頭]]和[[大埔頭鄧氏|大埔頭]]。

===元禎房===
==== 屏山 ====
{{main|屏山鄧氏}}
[[鄧元禎|鄧元禎]]生子[[鄧從光|鄧從光]]（號萬里）<ref name=屏山鄧族>{{cite book|author=蘇萬興|title=《坐言集之屏山鄧族》|url=http://www.somanhing.com/recommend/book/book/ebook/pingshan.pdf|edition=初版|location=香港|publisher=超媒體|date=2008年9月|accessdate=2019-04-08|isbn=978-988-17778-1-2|language=zh-hk}}</ref>{{rp|8}}，於[[南宋|南宋]]年間遷居[[屏山_(香港)|屏山]]，是為屏山鄧氏之開基祖。其後子孫繁衍，漸漸發展成「三圍六村」，即[[上璋圍|上璋圍]]、[[橋頭圍|橋頭圍]]、[[灰沙圍_(屏山)|灰沙圍]]、[[洪屋村|洪屋村]]、[[坑頭村|坑頭村]]、[[坑尾村|坑尾村]]、[[塘坊村|塘坊村]]、[[屏山新村|新村]]和[[新起村|新起村]]。屏山鄧氏於區內建立「屏山市」作為村民買賣之所。

===元亮房===
==== 錦田 ====
錦田鄧氏六世祖[[鄧壽祖|鄧壽祖]]仕官於[[元朝|元]]，後與子孫復居岑田<ref name=錦田鄧族>{{cite book|author=蘇萬興|title=《坐言集之錦田鄧族》|url=http://www.somanhing.com/recommend/book/book/ebook/kamtin.pdf|edition=初版|location=香港|publisher=超媒體|date=2008年3月|accessdate=2019-04-08|isbn=978-988-17279-4-7|language=zh-hk}}</ref>{{rp|42}}<ref name=贄惠碑>{{cite book|author=蘇萬興|chapter=廈村友恭堂 鼎建贄惠二公祠配享碑|title=《坐言集之廈村》|url=http://www.somanhing.com/gotowalk/dist/yuenlong/hacun/bei.pdf |location=香港|publisher=超媒體|pages=第2-1頁|accessdate=20190408|language=zh-hk}}</ref>。鄧壽祖生[[鄧處安|鄧處安]]、[[鄧敬安|鄧敬安]]。鄧處安生[[鄧康仁|鄧康仁]]、[[鄧康義|鄧康義]]，鄧敬安生[[鄧德荷|鄧德荷]]、[[鄧德蘭|鄧德蘭]]。鄧康仁生[[鄧洪生|鄧洪生]]、[[鄧洪惠|鄧洪惠]]，鄧德荷生[[鄧洪儀|鄧洪儀]]、[[鄧洪贄|鄧洪贄]]。錦田鄧族族人均為洪儀祖之子孫。

==== 廈村 ====
錦田鄧氏九世祖鄧洪贄（號白馬）與鄧洪惠（號西園）[[堂親#遠堂（同房）|再從兄弟]]與子孫於[[明朝|明朝]]中葉遷居[[廈村|廈村]]東頭里（今[[東頭村_(廈村)|東頭村]]）及西頭里（今[[祥降圍|祥降圍]]）<ref name=贄惠碑/>，其後子孫繁衍。洪贄祖屬上房，子孫分居東頭村、[[羅屋村_(廈村)|羅屋村]]、[[巷尾村|巷尾村]]（東頭三村）、[[新生村|新生村]]及[[輞井村|輞井村]]。洪惠祖屬下房，子孫分居祥降圍、[[新圍_(廈村)|新圍]]、[[錫降圍|錫降圍]]、[[錫降村|錫降村]]、[[新屋村|新屋村]]及[[紫田村|紫田村]]<ref name=友恭堂>{{cite book|author=蘇萬興|chapter=友恭堂|title=《坐言集之廈村》|url=http://www.somanhing.com/gotowalk/dist/yuenlong/hacun/yaukung.pdf |location=香港|publisher=超媒體|pages=第2-1頁|accessdate=20190408|language=zh-hk}}</ref>。廈村鄧氏於區內建立「廈村市」作為村民買賣之所<ref name=廈村市>{{cite book|author=蘇萬興|chapter=廈村市|title=《坐言集之廈村》|url=http://www.somanhing.com/gotowalk/dist/yuenlong/hacun/hacunmarket.pdf |location=香港|publisher=超媒體|pages=第1-1頁|accessdate=20190408|language=zh-hk}}</ref>。

==== 龍躍頭 ====
[[鄧元亮|鄧元亮]]於[[宋徽宗|宋徽宗]]時官至承務郎，任[[江西|江西]][[贛州|贛州]][[縣令|縣令]]。[[建炎|建炎]]三年[[金朝|金人]]南侵，元亮起兵勤王，於道上救得[[宋高宗|宋高宗]]之女趙氏皇姬，其後許配予其子[[鄧惟汲|鄧惟汲]]（鄧自明），二人育有四子[[鄧林_(莞城)|鄧林]]、[[鄧杞|鄧杞]]、[[鄧槐_(莞城)|鄧槐]]和[[鄧梓|鄧梓]]。[[鄧林|鄧林]]後人[[鄧季琇|鄧季琇]]（號松嶺）於[[元朝|元朝]]末年遷居[[龍躍頭|龍躍頭]]凹下，逐漸發展為「五圍六村」，即[[老圍_(龍躍頭)|老圍]]、[[麻笏圍|麻笏圍]]、[[永寧圍|永寧圍]]、[[東閣圍|東閣圍]]（或稱嶺角圍）、[[新圍_(龍躍頭)|新圍]]（或稱覲龍圍）、[[麻笏圍#麻笏村|麻笏村]]、[[永寧村|永寧村]]（或稱大廳）、[[祠堂村_(龍躍頭)|祠堂村]]、[[新屋村_(龍躍頭)|新屋村]]、[[小坑村_(龍躍頭)|小坑村]]和[[新圍_(龍躍頭)#覲龍村|覲龍村]]。

==== 萊洞 ====
[[鄧槐_(莞城)|鄧槐]]後人六世祖[[鄧仲昌|鄧仲昌]]約元朝年間遷居黎姓居民為主的[[黎洞|黎洞]]，後改名[[萊洞|萊洞]]，即[[萊洞老圍|萊洞老圍]]。鄧仲昌生[[鄧觀孟|鄧觀孟]]。鄧觀孟生四子[[鄧金|鄧金]]、[[鄧玉|鄧玉]]、[[鄧滿|鄧滿]]和[[鄧堂|鄧堂]]。後老圍環境改變而子孫遷出<ref name=萊洞老圍>{{cite web|author=周樹佳|url=http://www.hkcd.com/content/2018-09/25/content_1103655.html|title=風水大迫遷 萊洞鄧族四散創新天|publisher=[[香港商報|香港商報]]|date=2018-09-25|accessdate=2019-04-07|language=zh-hant}}</ref>，鄧金北遷東莞，鄧滿南遷大埔頭，鄧玉則遷往萊洞南，鄧堂則遷往萊洞北，後再分支到萊洞新屋（即今[[大塘湖|大塘湖]]）。現今九成居民皆源自鄧堂一支。

==== 大埔頭 ====
明代|龍躍頭鄧氏分遷[[大埔_(香港)|大埔]]開族<ref name=風物志/>{{rp|54}}，由[[敬羅公|敬羅公]]（敬章公？）首建[[大埔頭|大埔頭村]]，其後子孫繁衍而分居鄰近地區。[[清朝|清]][[康熙|康熙]]十一年（1672年）在濱海地區建立[[大埔舊墟|大埔舊墟]]<ref name=風物志/>{{rp|15}}，供村民進行買賣。因寇患嚴重，於居住村莊周圍加建圍牆，並挖護城河及建更樓守望。1900年代初，[[香港殖民地時期#香港政府|香港政府]]開建[[九廣鐵路|九廣鐵路]]，村莊遭路軌穿越一分為二，鐵路以東仍稱大埔頭村，而南面則改稱[[水圍|水圍]]或大埔頭老圍<ref name=風物志>{{cite book|url=https://www.districtcouncils.gov.hk/tp/english/links/files/TaiPo_Book1.pdf|title=大埔風物志|edition=增訂版|date=2007年|publisher=大埔區議會|author=蕭國健}}</ref>{{rp|54}}。大埔頭村內建有[[敬羅家塾|敬羅家塾]]。

==鄧志齋==
鄧志齋的後人鄧氏十四世鄧廷桂於[[康熙|康熙]]二十七年 (1688) 從惠州到元朗[[橫台山|橫台山]]落擔，成為了橫台山鄧氏的開基祖。鄧廷桂兒子鄧洪達，以及鄧廷桂兄弟廷彬，兩房鄧氏的後人皆有成員繁衍至[[水流田|水流田]]地居住。

除了鄧廷桂之外，鄧志齋的另一脈後人鄧國茂，最初定居於荃灣城門，他的兒子鄧球光於[[清朝|清]][[光緒|光緒]]十六年（1890）由城門遷移至水流田，村中的國茂鄧公祠是鄧球光及其兩兄弟所建。
