'''香港警察輪艇'''為隸屬於[[香港警務處|香港警務處]]的官方海上[[交通工具|交通工具]]，至2014年12月，[[水警總區|水警總區]]共有119艘警輪（Police Launch）及警艇（Police Vessel），包括：
*警察[[躉船|躉船]]：4艘
*物流躉船：2艘
*訓練警輪：1艘
*分區巡邏警輪：12艘
*中型巡邏警輪：17艘
*近岸巡邏警輪：23艘
*分區快速巡邏艇：12艘
*快速追截小艇：8艘
*高速攔截艇：5艘
*小艇及附屬艇：35艘

此外，亦有數十艘不包括在編制內，經由[[充公|充公]]途徑得來、僅用作訓練用途的[[馬達快艇|馬達快艇]]（[[大飛|大飛]]）。本條目僅列出水警總區編制（包括[[小艇分區|小艇分區]]）內之警察輪艇，未包括[[反恐特勤隊|反恐特勤隊]]及[[特別任務連|特別任務連]]兩個水警總區外擁有船艇的[[香港警務處架構|警察單位]]的編制內之[[快艇|快艇]]。

視乎船隻的建造物料及[[海事處處長|海事處處長]]的狀況評估，[[香港政府|香港政府]]船隻的使用年期一般介乎8年至20年。鋁身船隻的使用年期預計為15年，鋼身船隻的使用年期預計為20年。警察輪艇更換程序包括，水警總區向海事處處長徵詢建議，獲同意後：擬備顧問文件、甄選[[顧問|顧問]]、擬備招標文件、[[招標|招標]]、評審及批核標書、批出合約、設計及建造船隻、驗收及交付船隻，最後訓練人員使用船隻及安排船隻投入服務。因此，水警總區通常提早數年為有需要更換的船隻向[[立法會保安事務委員會|立法會保安事務委員會]]提交建議<ref>[http://www.legco.gov.hk/yr13-14/chinese/fc/fc/papers/f14-43c.pdf 財務委員會討論文件]</ref>。

== 中文命名 ==
香港警察輪艇最初並無中文名稱，直至1980年代中期，先後共15艘新達汶型警輪下水，警務處遂邀請[[香港大學|香港大學]]中文系[[陳耀南|陳耀南]]提名，陳耀南以15種[[中國傳統|中國傳統]][[美德|美德]]為新警輪命名，分別為禮、義、廉、孝、悌、忠、信、德、智、體、群、美、仁、勇及勤，從此，警輪有了警禮、警義及警廉等中文名稱<ref>[http://www.police.gov.hk/offbeat/1025/chi/2069.html 昔日警隊小典故] 《警聲》第1025期</ref>。

== 歷史 ==
成立初年，[[水警總區|水警]]主要採用[[划艇|划艇]]方式推動旗下幾艘木製[[帆船|帆船]]。1846年，[[香港殖民地時期#香港政府|香港政府]]購置一艘[[三桅帆船|三桅帆船]]予水警用作巡邏[[香港港口|港口]]；兩年後，該艘帆船在一場[[颱風|颱風]]中[[擱淺|擱淺]]及[[沉沒|沉沒]]，船上17名人員全數[[殉職|殉職]]。

歷時半個世紀之久，香港海港一直由水警在[[划艇|划艇]]上執行水上巡邏。1893年，[[香港警務處|差役]]訂購了首批[[蒸汽機|蒸汽機]][[舢舨|舢舨]]，為效率上一大改進。

1922年[[海員大罷工|海員大罷工]]發生後，水警改革，先以蒸氣發動船隻取代划艇，後來再以[[汽油|汽油]]取代[[煤|煤]]料，逐漸，水警船隊發展成為一個強而有力的海事保安組織。

1952年，水警開始使用[[雷達|雷達]]，為當時最先進，以及世界上最先採用雷達系統的船隊之一<ref>[http://www.police.gov.hk/offbeat/688/024_c.htm 軍人加入香港警隊概況（五）] 《警聲》 第688期</ref>，當時船隊包括比較大型的木製、長70呎警輪，相繼投入服務。

1972年，水警總區引進[[現代化|現代化]]、長78呎的達汶型警輪（50至56），[[鋼鐵|鋼鐵]]船身，各安裝有兩副[[油渣|油渣]][[機車|機車]]，時速達20海浬，每艘載員12名，由[[新加坡|新加坡]]製造。

2005年年中，水警總區逐步推行[[水警總區#海上警視系統|海上警視系統]]，運用尖端的航海科技，包括佈下俗稱為[[天眼|天眼]]的天羅地網（[[海|海]][[陸|陸]]雷達裝置、全天候[[熱能探測器|熱能探測器]]及數碼視像儀器等），同時配合一支船身比較小型、航海速度比較高及操作性能比較靈活的警輪隊伍，務求在更為符合[[經濟效益|經濟效益]]的原則下，加強海上保安能力。在此計劃之下，所有新型警輪皆配備了日夜攝像系統、差位全球定位系統、測深器、[[雷達|雷達]]、[[電子|電子]][[海圖|海圖]]及繯星羅盤等先進儀器，並且採用嶄新的水力噴射推進系統等，<ref>[http://www.info.gov.hk/gia/general/200011/15/1115128.htm 新水警輪加強近岸巡邏工作] 《新聞公報》 2000年11月15日</ref>以提升水警總區堵截水上非法活動的能力及處理緊急事故的效率<ref>[http://www3.news.gov.hk/ISD/ebulletin/tc/category/lawandorder/050613/html/050613tc08001.htm 警推「海上警視」系統加強保安]{{dead link|date=十一月 2017 |bot=InternetArchiveBot }}</ref><ref>[http://www3.news.gov.hk/isd/ebulletin/tc/category/lawandorder/040505/html/040505tc08001.htm 兩「健警」水警輪投入服務 ] {{webarchive|url=https://web.archive.org/web/20051027084454/http://www3.news.gov.hk/ISD/ebulletin/tc/category/lawandorder/040505/html/040505tc08001.htm |date=2005-10-27 }}</ref><ref>[http://www.news.gov.hk/tc/category/lawandorder/090807/html/090807tc08001.htm 七中型巡邏警輪下水] {{webarchive|url=https://web.archive.org/web/20090809202012/http://www.news.gov.hk/tc/category/lawandorder/090807/html/090807tc08001.htm |date=2009-08-09 }}</ref>。引進此系統共耗資4億4,500萬港元<ref>[http://www.police.gov.hk/offbeat/803/chi/ 海上警視系統 水警大計 全速推行] 《警聲》第803期</ref><ref name="水警4億添裝備 巡邏輪快1倍">[http://hk.news.yahoo.com/article/080415/4/6dxv.html 水警4億添裝備 巡邏輪快1倍]{{dead link|date=十一月 2017 |bot=InternetArchiveBot }}</ref><ref name="水警4億添裝備 巡邏輪快1倍"/>，包括中央指揮系統、兩艘躉船行動平台、17艘中型巡邏警輪及23艘分區快速巡邏艇。中央指揮系統於2009年12月落成，整套海上警視系統於2010年年底全面地實行。

2012年，水警總區為水師配置連接啓動雷達系統的新光電感應器，以提升輪艇的夜視水平<ref>[http://www.police.gov.hk/info/review/2012/text/tc/HKPF_TChi05.html 水警總區] 《香港警察年報》2012</ref>。

2014年5月13日，水警總區向立法會保安事務委員會提交文件指出需要更換3種合共18艘警輪，包括水警總區訓練船、6艘分區巡邏警輪及11艘海濤型警輪，作價6億5,841萬港元<ref>[http://news.takungpao.com.hk/paper/q/2014/0510/2469281.html 警申撥6.5億換18新船] 《大公報》 2014年5月10日</ref>，為2010年代最大規模的一次船隊更新。同年7月11日，水警總區向[[立法會財務委員會|立法會財務委員會]]申請撥款。

== 行動平台 ==
[[File:Marine_Police_Ship_4.jpg|thumb]]]]
'''警察躉船'''（[[英文|英文]]：'''Police Barge'''）'''行動平台'''於2002年起啟用，由[[廣州|廣州]]製造，主要作用為行動基地及偵測平台。躉船重達227噸，船身長30米，闊13米，高15米，吃水量為0.8米。躉船設施包括臨時拘留室、接見室、急救室和直升機吊運平台等等。內裡配備有先進的[[雷達|雷達]]、夜間偵察儀器、全球定位儀器及電子羅盤等等<ref>[http://www.police.gov.hk/offbeat/733/chi/ 先進躉船投入服務 打擊海上非法活動] 《警聲》 第733期</ref>。於2008年至2011年期間，再另外引進了兩艘警察行動平台。兩艘合共附設14艘小型支援艇的警察躉船行動平台停泊在[[吐露港|吐露港]]，一艘停泊在[[赤門海峽|赤門海峽]]響螺角作為海上檢查站及反走私浮動屏障，餘下一艘配備快速追截小艇隊的警察躉船行動平台則停泊在[[西貢區|西貢區]]以南的[[糧船灣|糧船灣]]範圍。

在[[沙頭角海|沙頭角海]]的偷渡及走私勾當情況持續，水警總區於2010年年底在有關海域設立反走私浮動屏障後，情況有所改善，然而基於駐守的近岸巡邏艇無足夠海域空間及設施用以管制、檢查及處理不合作之船艇，人員以赤門及[[馬料水|馬料水]]為行動基地，於事故發生後才調配船艇前往亦造成延誤，未能夠根治性堵截。加上反走私浮動屏障缺乏洗手間、休息地方及抵擋惡劣天氣的遮蔽處，工作環境並不理想，嚴重限制行動效率。有鑑於此，並且減少對後備近岸巡邏艇的依賴，水警總區曾經考慮一艘警輪或者警察躉船行動平台至該處，惟12艘現役具備足夠空間、設備及儀器的警輪均已經被調遣至深水海域執行任務，與現役4艘警察躉船均基於需要而無法被抽調，後者更為經過特定設計，僅繫泊於水深海域，不適合於淺水及封閉的沙頭角海運作。為了提升執法效率，[[水警總區|水警總區]]於2014年12月提出添置一艘針對在淺水活動的警察躉船行動平台，涉及預算開支3,576萬2千港元，預計於2018年4月驗收及交付，於同年5月完成培訓人員及投入服務。此艘警察躉船身長30米，闊13米，離水面2米高，具備基本推進系統，能夠自行移動；配備具備目標跟蹤功能的雷達、夜間偵察儀器、態勢知感系統、符合舉證標準的熱能探測儀器、全球定位系統及電子羅盤等等，能夠實時向位於岸上的[[指揮及控制中心|指揮及控制中心]]傳送資訊、現場錄像及[[數據|數據]]；並且配備一支小型支援艇隊<ref>[http://news.takungpao.com.hk/paper/q/2015/0103/2876377.html 水警將添躉船助海上執法] 《大公報》 2015年1月3日</ref>，預計率先於2016年1月驗收及交付<ref>[http://paper.wenweipo.com/2015/01/04/HK1501040020.htm 水警擬購躉船遏沙頭角走私] 《大公報》 2015年1月4日</ref>。

=== 小型支援艇 ===
'''小型支援艇'''為警察躉船行動平台的附屬艇，一支小型支援艇隊包括兩艘小型鋁質柴油噴射艇、兩艘充氣式小艇（海騎式小艇：船身長5.4米，載員兩名，合共可以運載10名<ref>[http://www.fso-createhk.gov.hk/tc/locations_search_details.php?File_No=01365&search_type=1 香港警務處海騎式小艇]</ref>。）及淺水救援艇<ref>[http://www.legco.gov.hk/yr14-15/chinese/panels/se/papers/se20150106cb2-534-7-c.pdf 立法會保安事務委員會 香港警務處水警總區購置一艘躉船行動平台] 立法會CB(2)534/14-15(07)號文件 2015年1月6日</ref>。

== 訓練船及指揮平台 ==

=== 現役 ===
[[File:HKPF_Police_Boat_3.JPG|thumb]]

'''水警總區訓練船'''──警輪3號，船長40米，最高時速達14海里，載員34名，合共可以運載100名，為現役排水量最大的警輪，鋼製，於1980年代末期投入服務，被設計為一艘具備分區指揮功能的離岸海上搜救船隻，惟由於需要執行離岸[[搜索及拯救|搜索及拯救]]任務需要利用船倉大部份空間以容納搜索及拯救人員及裝備，因此警輪3充當重大事故指揮平台的能力有所局限，其分區指揮職能於2002年由ASI分區巡邏警輪執行。自此以後，警輪3號主要用作訓練用途<ref>[http://www.fso-createhk.gov.hk/tc/locations_search_details.php?File_No=01364&search_type=1 香港警務處水警總區訓練船]</ref>。

=== 未來 ===
'''流動應變及指揮平台'''[[雙體船|雙體船]]，船長40米，以航海級[[鋁合金|鋁合金]]製，最高時速達25海里，為於大型海上事故，包括嚴重罪行、[[恐怖襲擊|恐怖襲擊]]威脅及災難支援等情況之下作為指揮平台，此外，能夠提升水警總區在淺水水域長時間採取行動及在偏遠水域運用小艇執行任務的水平。當無需要執行指揮任務或者支援前線行動時，流動應變平台則為訓練船的角色。流動應變及指揮平台配備熱能顯像、雷達、導航設備及通訊裝置等，現場的視像片段和數據記錄均能夠即時傳送至岸上的警察處所，供予[[水警總區總部|水警總區總部]]指揮官所使用；而駕駛台、特設的指揮及控制室及容易受到襲擊的地方均設置防彈裝置。此外，流動應變及指揮平台配備多種小型支援船，能夠在淺水水域及海灘／前灘範圍處理事故的能力。流動應變及指揮平台於投入服務後，將會取締警輪3號。

== 中型巡邏警輪 ==
'''中型巡邏警輪'''（英文：'''Medium Patrol Launch'''）船長19米、闊5米、吃水1米；由三台摩天宇(MTU)柴油引擎(每部1900匹馬力)搭載噴射系統推動，最高時速高達45海里，能夠於離岸比較遠的水域、淺水以至狹窄的河道中航行，並且具有高速迴轉能力。由[[臺灣|臺灣]][[龍德造船工業股份有限公司|龍德造船工業股份有限公司]]製造<ref>[http://www.lts.com.tw/web/ProductOverview_c.jsp?productId=8a3fb0641302f67754a991fc40bb71db 19M Ultra Fast Patrol Boat]</ref>，於2008年起陸續引進共17艘，用以取代自1980年代初服役至今的達文三型警輪。

== 多功能巡邏船 ==
'''多功能巡邏船'''船長30至35米，以[[鋼鐵|鋼鐵]]製，負責在指定巡邏區域巡邏，設置熱能顯像、夜視、雷達、通訊技術、消防及船隻管理系統等，並且配備兩艘小型支援船。多功能巡邏船可以幫助各[[分區|分區]]調配船隻及監察，亦能夠夠初步處理船隻失火事故。由於其特別設計的船體形狀以及推進系統，即使遇上惡劣環境，亦能夠保持船隻穩定及運作，致使人員在無即時支援下仍然可以持續地執行一段比較長時間的行動。多功能巡邏船設置主動式船隻穩定裝置，因而能夠減少因為船體擺動而引致人員疲勞的情況，從而增加人員的行動時間。多功能巡邏船的駕駛台及行動指揮範圍均設置防彈裝置。多功能巡邏船投入服務後，將會取締警輪51號至56號<ref>[http://www.legco.gov.hk/yr13-14/chinese/panels/se/papers/se0513cb2-1453-10-c.pdf 立法會保安事務委員會 更換18艘警輪]</ref>。

== 近岸巡邏警輪 ==
'''近岸巡邏警輪'''（英文：'''Inshore Patrol Launch'''）用作巡邏近岸，尤其是經過刊憲的[[泳灘|泳灘]]、[[避風塘|避風塘]]、[[海洋生物|海洋生物]]養殖區、船隻停泊處、屏蔽的錨泊點及其他擠塞的水面、離島及支援偏遠的水警總區轄屬地區。船長7.5米至8.1米，載員3名。近岸巡邏警輪可以獨立執行任務，亦可以在主要的警輪指揮下執行任務<ref>[http://www.fso-createhk.gov.hk/tc/locations_search_details.php?File_No=01221&search_type=1 香港警務處近岸巡邏警輪]</ref>。
