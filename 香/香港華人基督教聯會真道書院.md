'''香港華人基督教聯會真道書院'''（英語：The Hong Kong Chinese Christian Churches Union Logos Academy, HKCCCU Logos Academy）是[[香港華人基督教聯會|香港華人基督教聯會]]的第一所學校，位於[[香港|香港]][[新界|新界]][[西貢區|西貢區]][[調景嶺|調景嶺]]設有兩所校舍，並是一間直資一條龍[[中學|中]][[小學|小學]]，校訓為「求真行道，達善臻美」。真道書院是一所直資的[[國際文憑組織|國際文憑組織]]學校。

== 學校課程架構 ==
學校的架構與其它一般的中小學不同，乃因學校採用十一年制的學制，即兩年基礎階段、五年拓展階段及四年的通達階段。另外，該校校內是「一校兩制」，即[[三三四學制|三三四學制]]及[[國際文憑大學預科課程|國際文憑（IBDP）]]，而該校的中五學生是最後一屆參加中學會考（最後一年會考為2010年）的其中一批學生。該校最後一屆不是一條龍直升的中學生為2012年畢業的中六及中七。

== 校本規劃 ==
真道書院是[[香港|香港]]一所同時授教[[香港|香港]][[新高中|新高中]]課程及[[國際文憑大學預科課程|國際文憑（IBDP）]]課程的國際文憑組織學校。

若用[[香港|香港]]傳統學校的學制作參照，該校的小學課程只有五年，全個中小學課程只需11年，比傳統學校少一年。

== 特色 ==
FS1至MS4每級共有6至7班，班名十分特別，分別為V、E、R、I、T、Y，或V、E、R、I、T、A、S，拼合即為[[拉丁文|拉丁文]]Verity或Veritas，即真理的意思，與校名「真道」互相呼應。
與其他學校最不同之處是學生不用穿[[校服|校服]]。雖然如此，學校亦提供印有校徽的校服可供購買。除印有大圖示以外，可以穿著淨色有領上身衣服<ref>服飾儀容守則已於2007年9月起修改，學生只可穿著有領衣服上學。</ref>；另外，學生可以穿著藍色、黑色、卡其色，非[[牛仔褲|牛仔褲]]的褲子，惟褲子長度必須要長於大腿的一半。學生可穿著黑色皮鞋，以及黑、白、藍或灰色的運動鞋。

== 賬目混亂指控 ==
真道書院於2006年至2009年間，未經[[校董會|校董會]]審批，於2006年8月購置物業，作為外籍英語教師的員工宿舍。不過，學校並非該三項物業的註冊業主，而是由校長和校監以個人名義註冊，以受託人身份持有物業，但該信託聲明並非以文書設立，亦未曾在土地註冊處辦理註冊。另外，真道書院疑斥資 15萬元，培訓內地的校長，提供免費交通及食宿，同樣未得校董會批准，學校事前也沒有就計劃擬備財政預算；該校在未有公開招聘下，聘請了 36人。

== 新任行政團隊風波 ==
2015年7月11日，[[Facebook|Facebook]]用戶Eileen Mok於[[教育界功能界別|教育界]][[香港立法會|立法會]]議員[[葉建源|葉建源]]的「葉建源議員辦事處」Facebook專頁發佈一篇名為「真道書院風光背後風聲鶴淚」的文章，並稱是由「一群不平的真道教職員」所撰寫，內容主要評論真道書院新任行政團隊管理學校的手法，包括提前與個別教職員解約、縱容個別教職員，以及在未有諮詢下刪減科目等。校方回應指文章立論偏頗、信息錯誤、陳述虛假，意圖惡意破壞該校多年來由校董及老師所建立的基業與校譽、分化教師團隊。
