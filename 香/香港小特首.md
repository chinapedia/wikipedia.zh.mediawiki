'''香港小特首'''（英文縮稱：'''JCE'''，全名為'''Junior Chief Executives'''）由[[香港小童群益會|香港小童群益會]]主辦。首屆香港小特首於2001年就職。

== 理念 ==

「香港小特首」的中心理念是「[[兒童權利|兒童權利]]」以及「[[兒童參與|兒童參與]]」。

要建設[[兒童友好|兒童友好]]（[[:en:Child-friendly|child-friendly]]）的居住環境，必須考慮兒童的意見和意願。兒童也是社會大眾之中的一群，他們的想法應該受到尊重及考慮。透過各種活動為他們提供發表意見和參與社區事務的平台，提升高小及初中學生對社會事務的參與，亦能鼓勵兒童及青少年發聲，並藉此提醒社會大眾對「[[兒童權利|兒童權利]]」及「[[兒童參與|兒童參與]]」之關注。

== 歷屆成果 ==

=== 第一屆（2001年至2003年） ===
2001年就職，他們在2001年10月到[[廣州|廣州]]與[[廣州小市長|廣州小市長]]進行會談，獲當時任職[[廣州|廣州]][[廣州市市長#中華人民共和國時期|市長]][[張廣寧|張廣寧]]先生接見。

=== 第二屆（2003年至2005年） ===
於2003年就職，他們曾進行「與[[廣州小市長|廣州小市長]]在[[香港立法會|立法會]]進行辯論會」、「與前衞生署署長[[陳馮富珍|陳馮富珍]]女士進行真情對話」、舉辦「2003年及2004年[[兒童十大新聞選舉|兒童十大新聞選舉]]」、發表「2003年及2004年[[小小施政報告|小小施政報告]]」等活動。卸任前成立「[[香港小特首基金|香港小特首基金]]」，資助國內貧童的教育需要。

=== 第三屆（2005年至2007年） ===
於2005年就職，他們曾經舉辦多項活動包括發表「十大施政綱領」、「到廣州與[[廣州市|廣州市]][[少年宮|少年宮]]交流，聯合進行兩地兒童公民素質調查」、「小特首評大特首選舉論壇」及「立法激辯為兒童——論應否立法禁止體罰」等。
 
=== 第四屆（2007年至2009年） ===
於2007年11月正式就職，任期兩年。他們曾籌備並參與的大型活動包括「童心新聞眼——十大新聞選舉2007及2008」、發表《2008[[小小施政報告|小小施政報告]]》、「[[廣州|廣州]]文化保育交流」、「與[[香港教育學院|香港教育學院]]準教師交流」、「與[[大法官|大法官]]對話」、「[[香港律師會|香港律師會]]法律週2008」、「兒童權利論壇」、「[[電子教學|電子教學]]探討」及「接受多個電台及電視台訪問」等等。

=== 第五屆（2009年至2012年） ===
於2009年12月正式就職，任期兩年。他們曾籌備並參與的大型活動包括「2010最觸動兒童事件選舉」、發表2010及2011之《[[小小施政報告|小小施政報告]]》、參與「[[台灣|台灣]][[環保|環保]]交流活動」、「[[青年高峰會議|青年高峰會議]]」、「[[兒童權利論壇|兒童權利論壇]]」、及接受[[商業電台|商業電台]]、[[香港電台|香港電台]]及[[無線電視|無線電視]]、[[亞洲電視|亞洲電視]]、[[有線電視|有線電視]]等多個電視台、報章訪問等。小特首在各項活動中經常與社會知名人士交流意見及心得。
==== 重大事件 ====
* 2010年9月19日：發佈首份《小小[[施政報告|施政報告]]》，以「推行環保」為題。
* 2010年12月31日：發佈《2010最觸動兒童事件選舉報告書》，由香港兒童及青少年在10個類別中，選出同年10件最觸動的全球新聞。<ref>{{cite book |author=第五屆香港小特首 |title=《2010最觸動兒童事件選舉報告書》 |url=http://juniorce.bgca.org.hk/9228/files/file/5jce/5thJCEs_2010.pdf |date=2010年12月31日 |pages=p.1 |chapter=前言 }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }} </ref>
* 2011年10月9日：發佈任內最後一份《小小[[施政報告|施政報告]]》，以「『童』創綠色和諧新香港」為題。
* 2011年12月30日：[[香港小童群益會|香港小童群益會]]、[[少年財政司|少年財政司]]及第五屆香港小特首主辦，[[未雨行動|未雨行動]]及[[香港保險業聯會|香港保險業聯會]]協辦：「童想新香港」[[論壇|論壇]]。<ref>{{Cite web |url= http://www.bgca.org.hk/news.aspx?id=b1ba30a4-36cc-4e18-8fb2-5ff6df19421b |title= 新聞發佈：梁振英當選為兒童心目中的「特首」 }}</ref><ref>{{Cite web |url= http://hk.news.yahoo.com/馮檢基稱貧富差距擴闊因高薪工作不足-081600271.html |title=馮檢基稱貧富差距擴闊因高薪工作不足 |newspaper= 商業電台 |date=2011年12月30日 }}</ref><ref>{{Cite web |url= http://news.on.cc/cnt/news/20111230/bkn-20111230183943645-1230_00822_001.html |title=2011年12月30日新聞報道 |newspaper= 東方日報 |date=2011年12月30日 }}</ref><ref>{{Cite web |url= http://the-sun.on.cc/cnt/news/20111231/00407_038.html |title=兒童直選梁振英當特首 |newspaper= 太陽報 |date=2011年12月31日 }}</ref>

=== 第六屆（2012年至2013年） ===
於2012年1月15日宣誓，共32位第六屆香港小特首正式就職。<ref>{{Cite web |url= http://www.bgca.org.hk/news.aspx?id=3643925b-b098-469c-a40d-99d24b4648b0 |title= 新聞發佈：第六屆「香港小特首」宣誓就職禮 }}</ref> 他們曾籌備及參與的活動包括：「童想新香港─給特首一點建議攝影比賽」、「發表2012及2013小小施政報告」、「2012童看快樂校園」、「參加台灣交流活動」、「參加城市論壇」、「參加兒童權利論壇」等，並曾接受多個電台及電視台訪問；並於2014年元旦日與特首[[梁振英|梁振英]]先生會面。
==== 重大事件 ====
* 2012年10月6日：發佈任內首份《小小施政報告》，以「快樂、安全、共融校園」為題。<ref>{{Cite web |url= http://www.bgca.org.hk/news.aspx?id=0ad2da98-4f4d-400e-90c4-cafc6fbfbd02 |title= 新聞發佈：「童」創綠色和諧新香港──第六屆「香港小特首」發表《小小施政報告》／「童想新香港 – 給新任特首的一點建議」攝影比賽頒獎禮 }}</ref>
* 2013年7月16日至19日：前往台灣進行交流，並體驗「童玩節」，感受台灣兒童一年一度的盛事。
* 2013年12月：發佈「跨世代Happy Child － 口述童年小故事」，訪問了34位不同年代出生的朋友，分享如何渡過童年生活，從而讓小特首至讀者一同反思何謂「快樂童年」。
* 2013年12月28日：進行「小小施政報告發佈會」，同日發佈「2013小小施政報告」，主題為「童創快樂童年：從兒童『精神健康』、『家庭健康』、『醫療體系』及『遊戲權』看快樂童年」。

=== 第七屆（2014年至2015年） ===
於2014年1月正式就職，任期兩年。他們曾發表2014「小小施政報告」，並舉辦「台灣交流活動」、「韓國交流活動」、出席「城市論壇」、「立法會教育事務委員會公聽會及支持爭取十五年免費教育」、「兒童高峰會2014」及「教育改革分享論壇」等等，並接受多間本地媒體訪問。

==== 重大事件 ====
* 2014年4月22日至25日：前往台灣進行交流。
* 2014年12月28日：發佈任內首份《小小施政報告》，以「我有我的起跑線」為主題，透過「學習模式」、「課後學習」、「特殊學習需要」及「興趣啟迪」四個副題，探討香港兒童的學習情況。<ref>{{Cite web |url= http://std.stheadline.com/yesterday/edu/1229go01.html |title= 小特首《施政報告》促加強特教支援 }}</ref> <ref>{{Cite web |url= https://hk.news.yahoo.com/%E8%AA%BF%E6%9F%A5-%E5%8F%97%E8%A8%AA%E8%80%85%E8%AA%8D%E7%82%BA%E5%AD%B8%E7%AB%A5%E4%B8%BB%E8%A6%81%E5%A3%93%E5%8A%9B%E4%BE%86%E8%87%AA%E7%88%B6%E6%AF%8D%E6%9C%9F%E6%9C%9B-074900576.html |title= 調查﹕受訪者認為學童主要壓力來自父母期望 }}</ref> <ref>{{Cite web |url= http://www.hkatv.com/zh-hk/atvnews/37257/2014-12-28/%E5%88%9D%E4%B8%AD%E5%AD%B8%E7%94%9F%E9%A0%90%E8%A3%BD%E5%B0%8F%E6%96%BD%E6%94%BF%E5%A0%B1%E5%91%8A%E5%B8%8C%E6%9C%9B%E6%94%BF%E5%BA%9C%E8%81%BD%E5%8F%96%E6%84%8F%E8%A6%8B |title= 亞視新聞：初中學生預製小施政報告希望政府聽取意見 }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

=== 第八屆（2016年至2017年） ===
於3月29日至4月2日前往台灣交流

於2016年7月6日與特首梁振英先生會面。

於2017年12月3日發佈任內首份《小小施政報告》，以「重塑童年路」為主題，透過「採討小學全日制」、「考試與快樂學習」、「香港青少年睡眠質素」及「青少年自殺問題」等四個範疇，希望社會能協助重塑快樂童年。
