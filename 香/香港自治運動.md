'''香港自治運動'''（{{lang-en|'''Hong Kong Autonomy Movement'''}}）是[[香港|香港]][[本土派_(香港)|本土派]]推動的政治運動，理念建基於香港[[學者|學者]][[陳雲_(學者)|陳雲]]所編著的《[[香港城邦論|香港城邦論]]》為基礎，認為香港是一個[[城邦|城邦]]格局。

香港自治運動主張香港在《[[香港基本法|基本法]]》和[[一國兩制|一國兩制]]的大前提下，應享有真正的[[自治|自治]]權利，並非主張[[香港獨立運動|香港獨立]]<ref>[http://hkam2011.blogspot.com/2011/06/blog-post_25.html 堅持香港自治 保衛自由城邦—香港城邦自治運動總綱 (文：陳雲)]</ref>。[[陳雲_(學者)|陳雲]]本為運動的提倡者及顧問，但於2013年6月13日，「香港自治運動」的FB組織宣布與陳雲終止合作關係，原因未有解釋。<ref name="與HKAM終止合作關係">[http://www.facebook.com/permalink.php?story_fbid=478113522272116&id=118166011600204 香港自治運動Facebook專頁公告與陳雲終止合作關係]</ref>。現時多數的[[泛民主派|泛民主派]]及[[自決派|自決派]]皆爭取香港有真正的自治及[[真普選|真普選]]，而非支持香港獨立。

==目標==
* [[普選|普選]][[香港特別行政區行政長官|行政長官]]與[[香港立法會議員|立法會議員]]。
* [[香港政府|香港政府]]制定政策時，以[[香港人|香港人]]需要與長遠[[利益|利益]]為依歸。
* 規劃長遠[[房屋|房屋]]與[[土地|土地]]政策，讓香港人安居樂業。
* 重振[[香港工業|香港工業]]，復興[[香港農業|香港農業]]。
* 重新制定[[移民|移民]]政策，收回[[中國大陸|中國大陸]]移民審批權。
* 檢討[[中華人民共和國香港特別行政區基本法|基本法]]行憲以來之得失，完善[[憲政|憲政]]。
* 捍衞香港城邦。拒绝[[宜居灣|宜居灣]]等大陸化政策，以及[[德育及國民教育科|德育及國民教育科]]。
* 制定[[香港語言|香港語言]]政策，捍衛[[香港文化|香港文化]]。
* 檢討[[貨幣|貨幣]]本位與[[金融|金融]]政策，鞏固[[財政|財政]]主權。

== 徽號 ==
[[File:Hong_Kong_Autonomy_Movement_Flag.svg|230px]]

[[龍獅香港旗|龍獅香港旗]]是香港自治運動的代表旗幟，旗幟構圖帶有經修改加上[[漢字|漢字]]「香港」二字的英殖時期[[香港盾徽|香港盾徽]]。據香港自治運動的介紹，於原有香港盾徽中加入「香港」兩字是象徵同時繼承[[中華文化|中華]]及[[英國文化|英國文化]]，而以象徵自由的貴族藍作為底色是代表香港以本土為家，自立自治。而龍獅旗提倡的是[[普選|普選]][[香港特別行政區行政長官|特首]]、港人自治、法治及廉潔<ref>{{cite news |url = http://hk.apple.nextmedia.com/news/art/20120304/16124219|title=高舉龍獅旗 緬懷昔日香港 |newspaper = 蘋果日報|date=2012-03-04}}</ref>。

== 背景 ==
2011年5月下旬，香港學者陳雲撰文指出[[六四事件|六四事件]]啟動了香港人對中國的驚恐（China Fear），而[[香港泛民主派|民主派]]則提出「民主抗共論」為香港人定驚，但是其認為「[[五區公投|五區公投]]」令[[民主|民主]]進程受到挫折，[[民主黨_(香港)|民主黨]]因為「大中華情意結」，除了口頭上支持「建設民主中國」和「平反六四」，行動上卻未能夠以香港本土利益為先，等同投向[[中國共產黨|中國共產黨]]。<ref name="fb110530">[http://www.facebook.com/permalink.php?story_fbid=118832698200202&id=118166011600204 香港自治運動 － 龍獅香港旗Facebook專頁 ]</ref>陳雲說：「中共不會害怕一群向它哀求民主的港人，它現在變得更高傲了，連敷衍也懶得做。」他認為沒有香港自治的目標，香港人不會看到希望，香港的民主運動不會有持久力，並且表示香港要準備好自治的決心，鞏固香港的政治底盤。<ref name="fb110530" />5月30日，網民在社交網站[[Facebook|Facebook]]成立「香港自治運動」的專頁。

[[File:中環抗議標語「香港大陸化，慘過被油炸」_Protest_Poster_"Mainlandization_of_Hong_Kong_Is_Worsed_Than_Being_Fried_in_Oil".jpg|250px]]行人路上的抗議標語，當中「俾」應為「畀」或「被」]]
陳雲表示，[[中港矛盾|中港矛盾]]不斷激化，主要是由於北京不斷開放[[港澳自由行|陸客自由行]]，在教育方面企圖推行國民教育，抹殺[[港英時期|港英殖民時期]]的歷史及制度，令很多香港人感到一國兩制變質。北京會將以前殖民政府留下的文明，或者殖民政府保存到一些華夏文化，例如廣東話的漢音及[[正體字|正體字]]，以及好的、文雅的公共用語、公共的中文或者英文，這些慢慢會當是前朝歷史，或者當作殖民地餘孽來掃除。甚至[[三權分立|三權分立]]、[[司法獨立|司法獨立]]、[[人權|人權]]法這些由殖民政府引入香港的現代東西，都會被當作外來的殖民霸權，慢慢淡忘，承接中國大陆的蘇維埃或者共產黨的體制。他表示，中共承諾卻不容許港人治港、香港擁有高度自治，因此預期未來中港矛盾將會激化。他說：如果香港人不反抗，國際社會不監察的話，是會走向一國一制，即是走向非法的中國大陆殖民統治，這個要看香港人能否覺醒及反抗，爭取普選及抗拒一切違反《[[香港基本法|香港基本法]]》的、中共的干預，因為香港的《基本法》寫得很明確，也算是相當好的一部小[[憲法|憲法]]、鬥爭或者抗爭的基礎其實是存在的。<ref>[http://www.voacantonese.com/content/yearender-hong-kong-china-contradictions/1574444.html 香港年終回顧: 中港矛盾激化], [[美國之音|美國之音]], 2013年02月10日</ref>

== 評價 ==
前[[全國政協委員|全國政協委員]][[劉夢熊|劉夢熊]]曾將香港自治運動與[[台灣獨立運動|台獨]]相提並論，認為香港自治運動是由[[美国驻香港及澳门总领事列表|美國駐港澳總領事]][[楊甦棣|楊甦棣]]暗示及策動，企圖把香港從[[中華人民共和國|中國]]分裂出去。<ref>[http://trans.wenweipo.com/b5/paper.wenweipo.com/2011/08/29/HK1108290022.htm 指點江山：楊甦棣策動「香港自治運動」包藏禍心] {{webarchive|url=https://web.archive.org/web/20150920013345/http://trans.wenweipo.com/b5/paper.wenweipo.com/2011/08/29/HK1108290022.htm |date=2015-09-20 }} 劉夢熊 - 文匯報 2011-08-29</ref>

[[香港特別行政區基本法委員會|香港基本法委員會]]主任[[李飞_(官员)|李飛]]在2016年11月[[全國人大常委會解釋香港特別行政區基本法|對特區第五次釋法]]的記者會上表示，香港自古以來是中國一部分，所以主張港人民族自決本質上也是港獨。<ref>{{cite news|title=【人大釋法】不真誠莊重宣誓即失資格 李飛：民族自決是港獨|url=http://hk.apple.nextmedia.com/realtime/news/20161107/55882633|accessdate=2016-11-07|agency=蘋果日報 (香港)|date=2016-11-07}}</ref>
