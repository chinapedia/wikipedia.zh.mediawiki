[[填海|填海]][[工程|工程]]是增加[[香港土地供應|土地供應]]的其中一個方式，香港百多年來的填海工程，深遠地影響着[[香港社會|香港社會]]的發展。

截至2013年3月，香港從填海工程獲得的土地面積逾67平方公里，佔香港土地總面積約7%<ref>{{cite news| url=http://hk.news.yahoo.com/陳茂波-本港填海比例不及新加坡-010800239.html| title=陳茂波：本港填海比例不及新加坡}}</ref><ref>{{cite news| url=https://news.now.com/home/local/player?newsId=63205| title=陳茂波：填海比發展新界便宜| publisher=[[now新聞|now新聞]]| date=2013-03-23}}</ref>，容納了27%的[[香港人口|香港人口]]及70%的商業活動<ref>{{cite news| url=http://hk.news.yahoo.com/%E9%99%B3%E8%8C%82%E6%B3%A2%E7%9B%BC%E7%A4%BE%E6%9C%83%E9%96%8B%E6%94%BE%E8%A8%8E%E8%AB%96%E5%A1%AB%E6%B5%B7-023215220.html| title=陳茂波盼社會開放討論填海| newspaper=[[明報|明報]]| date=2013-03-22| deadurl=yes| archiveurl=https://archive.is/20130428155413/hk.news.yahoo.com/%E9%99%B3%E8%8C%82%E6%B3%A2%E7%9B%BC%E7%A4%BE%E6%9C%83%E9%96%8B%E6%94%BE%E8%A8%8E%E8%AB%96%E5%A1%AB%E6%B5%B7-023215220.html| archivedate=2013-04-28}}</ref><ref>{{cite news| url=http://hk.news.yahoo.com/陳茂波指填海更易掌供應時間-042500609.html| title=陳茂波指填海更易掌供應時間| newspaper=[[星島日報|星島日報]]| date=2013-03-23}}</ref><ref>{{cite news| url=http://hk.news.yahoo.com/陳茂波-填海造地免臨渴掘井-220310416.html| title=陳茂波：填海造地免臨渴掘井| newspaper=[[星島日報|星島日報]]| date=2013-03-24}}</ref>。

== 方式 ==
最初期的填海方式，只是把開發時所產生的沙泥碎石，直接傾倒進海裡去。此種方式，造出來的陸地最接近自然海岸。不過，若然填海地所處的水流比較急，水流對海岸的衝擊，將會漸漸侵蝕填海區。過去[[香港政府|香港政府]]對[[淺水灣|淺水灣]]的擴展工程，就遭遇過此種問題。

若果堆填範圍比較大，比較好的填海辦法為先用比較穩固的填料在填海範圍修築堤圍，然後再把填料傾倒進堆填區內。這種堆填方式最為常用，亦為最普遍。現時[[西九龍填海區|西九龍填海區]]就是使用這種方法，把[[油麻地避風塘|油麻地避風塘]]的堤圍連結，再把沙石傾倒進堤圍內，再等候填料沉澱，然後再在新填海區開展建造工程。[[沙田新市鎮|沙田新市鎮]]亦是採用這種方式進行填海工程。若果填料只是沙泥，則可以直接傾倒進海裡。不過，若果填料是汙泥或者垃圾，在填海前，必須先行平整海床。平整海床的方式是先把海底的汙泥挖起，在海底及堤岸舖設防止污水滲漏的膠膜，然後才將填料倒進堆填範圍。否則，汙泥在表層建築物的壓力作用下會滑動，使到地面造成[[不平均沉降|不平均沉降]]。

== 歷史 ==
[[File:Commemorative_Plaque_for_the_Original_Waterfront_in_1841.jpg|thumb]]]]

=== 香港開埠 ===

==== 第一次非正式填海 ====
香港於1842年，即[[香港開埠|香港開埠]]後的第二年，就進行第一次非正式的填海工程。當時，[[香港島|香港島]]、[[中環|中環]]、香港島第一條道路——[[荷李活道|荷李活道]]、[[皇后大道|皇后大道]]及[[雲咸街|雲咸街]]的興建造成大量沙石，為避免搬運至其他地區存放，於是直接把沙石傾倒進[[維多利亞港|維多利亞港]]，擴大[[維多利亞城|維多利亞城]]的發展面積，這日後便成為香港第一幅填海土地。[[查理·义律|查理·义律]]將新填海地域劃分為40段，每段約100英呎，讓英商進行投標，以在中環開闢一名為「女王城」（{{lang|en|Queen's Town}}）的商業區。結果共售出33幅土地，價格由20英鎊至265英鎊不等<ref name="香港歷史之謎">{{cite book|author=爾東|editor=周海燕|title=《香港歷史之謎》|date=2007年4月|publisher=明報出版社有限公司|location=香港|isbn=978-962-973-634-7|page=86}}</ref>。

==== 首次正式填海 ====
至於香港首次的正式填海工程，則是於1852年展開的《[[文咸填海計劃|文咸填海計劃]]》，位置在今日[[上環|上環]][[文咸東街|文咸東街]]一帶，目的是將上環的發展面積進一步增加，以興建[[香港政府部門|香港政府部門]]及[[港口|港口]]設施。其後，雖然於1856年倡議的《[[寶靈填海計劃|寶靈填海計劃]]》備受反對而未推行，香港政府於1860年代至1900年代進行了一系列的填海工程，使到[[香港島|香港島]]北岸西至[[堅尼地城|堅尼地城]]，東至[[銅鑼灣|銅鑼灣]]，製造了不少土地。1920年代，[[灣仔|灣仔]]再進行一輪填海工程後，之後直至[[第二次世界大戰|戰]]前香港島再無任何填海工程。

==== 九龍半島填海工程 ====
[[File:HK_Statue_Square_1910.jpg|thumb]]及[[香港立法會大樓|最高法院]]，都是興建於中環填海土地上]]

[[九龍半島|九龍半島]]方面，自從英國於1860年取得該土地後，於1867年於[[九龍角|九龍角]]出現首次非正式填海工程。1876年，在[[油麻地|油麻地]]擁有地段的業主自行進行填海工程，成為今日[[新填地街|新填地街]]一帶。[[東鐵綫|九廣鐵路英段]]工程中，在1914年於[[尖沙咀|尖沙咀]][[梳士巴利道|梳士巴利道]]以南填海，以興建[[尖沙咀火車站|尖沙咀火車站]]的路軌。雖然九龍半島此後進行了不少填海工程，惟在[[第二次世界大戰|戰]]前，大多為私人進行。這是因為當時的香港政府集中發展香港島北部，對九龍半島的土地需求甚少。唯一例外為於1930年興建[[啟德機場|啟德機場]]時，由香港政府所進行的填海工程。這是1920年代[[何啟|何啟]]與[[區德|區德]]於[[土瓜灣|土瓜灣]]至[[九龍灣|九龍灣]]之間填海發展啟德機場住宅區失敗後，遺留下來的剩餘工程。而於[[香港日治時期|香港日治時期]]，[[日本軍隊|日本軍隊]]亦為進一步擴建啟德機場而進行了大規模填海，填海所使用的沙石來源包括來自拆毀[[九龍寨城|九龍寨城]]的城牆及炸毀[[宋王臺|宋王臺]]等破壞行為。總括而言，至1945年，香港有約80公頃土地是從填海得來的。

=== 第二次世界大戰後 ===

==== 香港重光 ====
[[香港重光|香港重光]]後，首個填海工程位於[[新界|新界]][[沙田區|沙田區]]，亦是新界的第一個大規模填海工程，於1950年在[[城門河|城門河]]畔由一[[劉姓|劉姓]]商人填海，以興建[[沙田墟|沙田墟]]住宅區。香港政府同樣在1950年填平[[銅鑼灣避風塘|銅鑼灣避風塘]]，於原址興建[[維多利亞公園|維多利亞公園]]。1950年代至1960年代，香港政府於[[柴灣|柴灣]]與[[小西灣|小西灣]]、啟德機場、[[觀塘工業區|觀塘工業區]]及[[紅磡灣|紅磡灣]]等地進行填海。而隨著[[香港新市鎮|香港新市鎮]]的規劃，香港政府亦開始在新界進行大規模填海工程。

==== 新市鎮及產業用地 ====
1966年，為配合[[屯門新市鎮|屯門新市鎮]]發展計劃，香港政府開始在[[青山灣|青山灣]]填海。1970年代，[[荃灣|荃灣]]、[[醉酒灣|醉酒灣]]（今[[葵涌貨櫃碼頭|葵涌貨櫃碼頭]]）及[[沙田新市鎮|沙田新市鎮]]等都有大規模填海工程。1970年代後期至1980年代，香港島及九龍半島也進行了不少填海工程，範圍不僅限於維多利亞港內，例如[[南區_(香港)|香港島南部]]的[[香港仔|香港仔]]、[[鴨脷洲|鴨脷洲]]及[[鋼綫灣|鋼綫灣]]（今[[數碼港|數-{}-碼港]]）等，均進行以興建住宅為目的的填海。除住宅外，也有填海計劃是為了發展工業的，包括於1985年落成的[[大埔工業邨|大埔工業邨]]。第二次世界大戰後至1980年，香港的填海面積約有4,000公頃。

==== 基礎建設 ====
1980年代初，[[香港地鐵|地鐵]][[港島綫|港島綫]]興建期間，在[[東區_(香港)|港島東部]]進行了多項移山填海的工程，包括夷平[[康山|康山]]，及在[[太古城|太古城]]、[[西灣河|西灣河]]、[[筲箕灣|筲箕灣]]至[[杏花邨|杏花邨]]海面填海。所得的土地興建了[[港島東區走廊|港島東區走廊]]、[[東區海底隧道|東區海底隧道]]及住宅物業。

=== 1990年代 ===

==== 玫瑰園計劃 ====
1989年，香港政府公佈《[[香港機場核心計劃|香港機場核心計劃]]》，當中不少計劃需要進行填海，包括在[[赤鱲角|赤鱲角]]興建[[新香港國際機場|新香港國際機場]]、[[北大嶼山新市鎮|北大嶼山新市鎮]]及[[西九龍|西九龍]]等。1990年代初期，香港政府也在香港港口及機場發展策略中提出多個填海建議，在[[中環|中環]]、[[灣仔|灣仔]]、[[九龍灣|九龍灣]]、[[紅磡灣|紅磡灣]]、[[昂船洲|昂船洲]]、[[青洲_(中西區)|青洲]]、[[交椅洲|交椅洲]]及[[竹篙灣|竹篙灣]]等進行填海，其中中環及灣仔填海計劃其後被落實。因此，1990年代可以說是香港填海的黃金年代。

==== 維港填海爭議 ====
然而，香港政府的鴻圖大計惹起[[環境保護主義|環境保護主義]]人士的不滿意（特別是[[保護海港協會|保護海港協會]]），認為填海工程沒有需要，破壞海港及生態環境。2003年，有人士於[[香港終審法院|香港終審法院]]推翻《[[灣仔填海計劃|灣仔填海計劃]]》，而且法官提及需要「有迫切及凌駕性的當前需要」才能夠填海，從此直接影響到香港的未來規劃，多座[[基礎建設|基礎建設]]項目及多份長遠計劃被逼暫時停止、中止或者需要重新檢討，更令到部份計劃的完成日期遙遙無期。

=== 2000年至今 ===

==== 不浚挖式填海 ====
2011年，[[港珠澳大橋|港珠澳大橋]]香港段動工，其香港口岸及[[屯門至赤鱲角連接路|屯門至赤鱲角連接路]]均需填海，以提供出入境及連接道路，合共約196.5公頃。<ref>{{Cite web|url=http://hzmb.hk/tchi/about_overview_03.html|title=香港口岸|accessdate=2017-11-21|author=路政署|date=|publisher=}}</ref><ref>{{Cite web|url=http://hzmb.hk/tchi/about_overview_04.html|title=屯門至赤鱲角連接路|accessdate=2017-11-21|author=路政署|date=|publisher=}}</ref>特點為首次採用不浚挖式填海，以鋼圓筒圍起填海，較一般填海更環保。惟2016年3月發現鋼圓筒出現較大移位問題，需要加固<ref>{{cite web|publisher=[[明報|明報]]|title=港珠澳橋人工島 填海鋼筒全移位 專家：倘沒加劇應不影響工程|url=https://news.mingpao.com/pns/dailynews/web_tc/article/20160430/s00002/1461952624998|accessdate=2017-11-21|date=2016-04-30}}</ref>。

==== 維港以外填海 ====

==影響==
香港百多年來的填海工程深遠地影響著香港社會的發展，期間使到大量[[香港已消失島嶼|島嶼]]（例如[[昂船洲|昂船洲]]及[[佛堂洲|佛堂洲]]等等）連陸，亦使到一些[[香港已消失海灣|海灣]]（例如[[紅磡灣|紅磡灣]]、[[大赤沙|大赤沙]]及[[小赤沙|小赤沙]]等等）縮少甚至消失。

===正面===
香港山多平地少，可以供予發展的土地不多，因此自[[香港開埠|香港開埠]]以來，香港政府多次進行填海工程。時至今日，香港很多繁華的地區都是依靠填海而取得土地，當中包括[[德輔道|德輔道]]以北的[[中環|中環]]、[[灣仔|灣仔]]、[[銅鑼灣|銅鑼灣]]、[[尖沙咀東|尖沙咀東]]、[[啟德機場|啟德機場]]、[[港澳碼頭_(香港)|港澳碼頭]]、[[九龍灣|九龍灣]]及[[觀塘商貿區|觀塘商貿區]]等等的重要發展地方。當中有不少重要[[建築物|建築物]]或者[[地標|地標]]都座落填海區上，包括[[國際金融中心_(香港)|國際金融中心]]及[[香港會議展覽中心|香港會議展覽中心]]等等。由此可見，香港填海工程對香港的[[城市規劃|城市規劃]]發展以至整個[[香港社會|香港社會]]的發展都有著深遠的影響。此外，填海工程為香港帶來了就業機會，尤其是香港建築業。不論是填海造地，還是在填海地上興建建築物，都需要大量[[勞動人口|勞動人口]]，這些就業機會亦養活了不少[[香港市民|香港市民]]。此外，填海工程可以美化[[海岸線|海岸線]]，改善沿海景觀。例如正在興建中的[[中環海濱長廊|中環海濱長廊]]，落成後就可以使到公眾更加親近海港，享受休憩設施。

===負面===
今日的[[維多利亞港|維多利亞港]]經過多次填海造地後，面積大幅減少。海上交通日益頻繁，造成海浪比較過急的情況，使到體形比較小的船隻在航行時搖曳不定。這亦與拉直了的海岸線，使到海水流動速度變急有關係。此外，大興土木、填海和運輸建築材料時，亦對水質造成影響。亦有人士擔心填海將會變本加厲地破壞了維多利亞港的景緻及原有的[[地理環境|地理環境]]，亦將會減少了一處香港旅遊景點，影響[[香港旅遊業|香港旅遊業]]。

==保護海港==
{{see also|保護海港協會|保護海港條例}}1980年代，由於環境保護的議題尚未成熟，加上填海為香港提供很多可見的好處，所以當時幾乎完全無人反對填海計劃。踏入1990年代中期開始，由於部份環境保護人士的提倡，不少[[香港人|香港人]]開始關注維多利亞港的填海計劃，開展了不少「保護維港」的運動，當中以1995年成立[[保護海港協會|保護海港協會]]最為人所認識，該會在1996年1月發動保護海港簽名行動，獲得17萬名公眾簽名支持，逼使香港政府於同年5月擱置[[青洲_(中西區)|青洲]]填海計劃。此後，該會於同年6月呈交《[[保護海港條例|保護海港條例]]》，於12月收集了148,041個簽名遞交給時任[[香港總督|香港總督]][[彭定康|彭定康]]，結果後於1997年6月於[[香港立法會|立法局]]最後一次會議時成為法例。該條例列明「海港須作為香港人的特別公有資產和天然財產而受到保護和保存」<ref name="彩色香港1970">{{cite book|author=高添強|editor=任秀雯|title=《彩色香港1970s-1980s》|date=2014年7月|publisher=三聯書店（香港）有限公司|location=香港|isbn=978-962-04-3379-5|page=頁23}}</ref>。

[[香港主權移交|香港主權移交]]後，反對填海的聲音漸趨強烈。1998年，因為保護海港協會為首的多個環境保護團體的反對，逼使香港政府先後需要暫緩及重新檢討[[啟德發展計劃|啟德發展計劃]]及添馬艦發展工作中的填海部份。於1999年10月發表的《[[1999年度香港行政長官施政報告|施政報告]]》中，時任[[香港行政長官|香港行政長官]][[董建華|董建華]]承諾將會減少填海，並且遵守[[可持續發展|可持續發展]]之原則。2001年，青洲填海計劃亦因此擱置。

香港反對填海最重要的事件是於2004年1月9日保護海港協會成功挑戰香港政府，在[[香港終審法院|香港終審法院]]推翻灣仔北填海計劃。香港法院作出宣判時指出，進行填海需要符合以下三個原則：
# 有迫切及凌駕性的當前需要（包括社群的經濟、環境和社會需要）
# 沒有其他可行方法
# 對海港的損害減至最少

雖然及後[[香港政府|香港政府]]於2005年在《[[中環填海計劃|中環填海計劃]]》案件上勝訴，惟其他填海工程備受[[香港社會|香港社會]]密切注視。而因為需要符合「有迫切及凌駕性的當前需要」的原則，令到絕大部份計劃於規劃時構成了一定的局限性。例如[[香港會議展覽中心|香港會議展覽中心]]對出原來擬建的直升機坪，因此不能夠填海，而需要削減計劃規模，甚至擱置至今。此外，為了避免再次引起訴訟，香港政府亦成立諮詢性的[[共建維港委員會|共建維港委員會]]，以聽取民間對於填海工程的有關意見。
