'''香芹'''（[[学名|学名]]：{{lang|la|''Petroselinum crispum''}}），是一种[[蔬菜|蔬菜]]，又有'''巴西利'''、'''巴西里'''、'''洋香菜'''、'''歐芹'''、'''洋芫荽'''或'''番芫荽'''、'''荷蘭芹'''等名稱。[[伞形科|伞形科]][[欧芹属|欧芹属]]的[[二年生|二年生]][[草本|草本]]。原产[[地中海|地中海]]沿岸。高15～20cm、具有清爽的香味和鲜綠色。[[古罗马|古罗马]]时代起用于[[烹调|烹调]]，也是世界上用的最广的一种[[草药|草药]]。对地质和气候的适应性高，栽培也容易，故世界各地都有栽培，但对乾燥的适应性差。

== 种类 ==
皱叶（Curly parsley）和非皱叶的两种。前者是一般的'''香芹'''或一些烹飪節目所叫的'''荷蘭香芹'''<ref>{{cite web |url=http://www.weibo.com/mimimba#_rnd1443322388245 |title=#回到厨房运动#DAY212: 第212个厨房健康自制美味 |work=MissMi甜蜜厨房-加拿大 |accessdate=2015-09-27 |date=2015-09-01 |language=zh-hans }}</ref>，後者是'''意大利香芹'''。

== 营养 ==
营养价值较高，含有[[维生素A|维生素A]] （[[β-胡萝卜素|β-胡萝卜素]]）、[[维生素B1|维生素B<sub>1</sub>]]、[[维生素B2|维生素B<sub>2</sub>]]、[[维生素C|维生素C]] 等多种[[维生素|维生素]]、[[钙|钙]]质、[[镁|镁]]质、[[铁|铁]]等[[灰分|灰分]]和[[食物纤维|食物纤维]]、[[叶绿素|叶绿素]]等，这些[[营养素|营养素]]含量在蔬菜中属前列。
