《'''香港時報'''》（'''Hong Kong Times'''）前稱《國民日報》，在1939年7月創刊，至1949年改名為《香港時報》，於1993年2月17日停刊<small><ref>{{cite web |language=zh-hk |url=http://www.nshk.org.hk/html/d_50cel/c50_0402.htm |title=本港主要報業一覽：《國民日報》(1949年改名為《香港時報》) |publisher=香港報業公會 |date= |accessdate=2010-02-23 |deadurl=yes |archiveurl=https://web.archive.org/web/20090809113500/http://www.nshk.org.hk/html/d_50cel/c50_0402.htm |archivedate=2009-08-09 }}</ref></small>。

==歷史==
1938年[[日本侵華|日本侵華]]，[[廣州|廣州]][[淪陷|淪陷]]，這引致大批難民湧入[[香港|香港]]，同時多份[[中國|中國]]報刊遷移香港出版。《國民日報》是當時[[國民黨|國民黨]][[香港|港]][[澳門|澳]]灣（即[[廣州灣|廣州灣]]，今廣東湛江一部分）總支部在港主辦報刊，集[[天津|津]][[上海|滬]]港三家黨報的資材而重起爐灶，當中以宣傳國民黨的[[中国抗日战争|抗戰日本]]等策略為主，起初由[[陶百川|陶百川]]主持，及後改由[[陳訓悆|陳訓悆]]等人接任，直至1949年8月4日（星期四）正式改名為《香港時報》<ref>1949年8月4日（星期四），香港《星島晚報》，第一版，封面廣告</ref>。由於虧損嚴重及臨近[[香港回歸|香港回歸]]，於1993年停刊。雖然報章停刊，但仍保留註冊刊號「44」<small><ref>{{cite web |language=zh-hk |url=http://chinareviewagency.org/doc/1001/9/2/8/100192820.html?coluid=67&kindid=1734&docid=100192820 |title=香港媒體在兩岸關係中的橋樑作用 |publisher=中國評論通訊社 |date=2006-08-20 |accessdate=2010-02-23}}</ref></small>。其後[[沈誠|沈誠]]創辦《新香港時報》，身兼社長及總編輯，於1995年亦告停刊。

《香港時報》結業後，其位於[[灣仔|灣仔]][[告士打道|告士打道]]與[[盧押道|盧押道]]交界的「香港時報大廈」（正名為「華夏大廈」）價值高達5.8億港元，但當時的國民黨黨營事業大總管[[劉泰英|劉泰英]]僅以1.9億港元出售，被稱賤賣國民黨產業<small><ref>{{cite web |language=zh-hk |url=http://the-sun.on.cc/channels/news/20021130/20021130010417_0001_2.html |title=賤賣國民黨產業 |publisher=[[太陽報_(香港)|''the-sun.on.cc'']] |date=2002-11-30 |accessdate=2010-02-23}}</ref></small>。其後香港時報大廈在70日內轉售三次，獲利高達3.9億港元，傳說[[李登輝|李登輝]]的妻子[[曾文慧|曾文慧]]分得2,000萬港元，其餘各級人員都有分潤<small><ref>{{cite web|language=zh-hk|url=http://boxun.com/hero/201001/huzhiweiwenji/8_1.shtml|title=一代詞宗陳蝶衣軼事|publisher=博讯博客独立中文笔会网|date=|accessdate=2010-02-23}}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref></small>。香港時報大廈現時已改建為「筆克大廈」。

==風格==
《香港時報》作為[[國民黨|國民黨]]喉舌報，除報導親台言論外，其體育版及副刊亦有一定影響力。體育版主力報導[[香港足球|香港足球]]消息，由[[香港電台|香港電台]]足球評述員[[盧震喧|盧震喧]]以筆名「原子塵」擔任主筆；而[[劉以鬯|劉以鬯]]及[[陸離|陸離]]亦曾擔任副刊主編多年。前[[立法會議員|立法會議員]][[黃毓民|黃毓民]]於1980年代任[[珠海書院|珠海書院]]新聞及傳播學系講師期間，亦在《香港時報》擔任採訪主任<small><ref>{{cite web |language=zh-hk |url=http://www.rthk.org.hk/broadcast75/talk_guest07_wong.htm |title=香港廣播七十五年：演講廳-嘉賓簡介 黃毓民 |publisher=[[香港電台|''rthk.org.hk'']] |date= |accessdate=2010-02-23}}</ref></small>。
