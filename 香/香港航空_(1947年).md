{{香港航空}}
'''香港航空'''（'''Hong Kong Airways'''，简稱为「'''港航'''」）原為一家在1940年代至50年代以[[香港|香港]]為基地的航空公司，是香港第二家提供民航服務的航空公司。

香港航空的主要股東是[[怡和洋行|怡和洋行]]和[[英國海外航空|英國海外航空]]（BOAC，[[英國航空|英國航空]]的前身），經營期間兩家公司的股權比例曾大幅度變動，因此兩家公司均曾主導港航的營運。

港航旗下機隊主要由[[道格拉斯DC-3|道格拉斯DC-3]]和[[維克斯子爵|維克斯子爵]]式（Vickers Viscount）組成。

== 誕生 ==
港航於1947年成立，英國海外航空是最大出資者。當時中國實施[[開放天空|天空開放政策]]，容許兩地的其中兩間航空公司飛行中港航線。因此，英國海外航空遂透過港航經營往返大陸的航線。

== 發展 ==
後來，香港政府為了舒緩港航與另一間本地航空公司——[[國泰航空|國泰航空]]的競爭，實施了「南北分界」：港航獲分配往香港以北目的地（包括[[中國大陸|中國大陸]]、[[日本|日本]]、[[韓國|韓國]]及[[台灣|台灣]]等地）的航線牌照，而國泰航空則擁有前往香港以南目的地（包括[[東南亞|東南亞]]及[[澳洲|澳洲]]等地）的航線牌照。至於往[[菲律賓|菲律賓]]航線，則由兩間航空公司合作經營。

[[四十年代|四十年代]]末是港航的全盛時期。當時大批內地富裕難民為逃避戰亂（[[國共內戰|國共內戰]]）而來港，令港航經營的多條往返中國大陸的航線皆十分繁忙——當時單是來往[[廣州|廣州]]的航線，一天便有4班以上。但[[中華人民共和國|中華人民共和國]]成立後跟香港斷航，令港航失去多條主要航線。繼後，由於[[韓戰|韓戰]]的理由，[[朝鮮半島|朝鮮半島]]航線也因安全理由而斷航（至韓戰結束後恢復至[[南韓|南韓]]的航線），經營變得更見困難。後來，港航的航線更只剩下[[日本|日本]]和[[台灣|台灣]]航線。

隨著經營惡化，英國海外航空將大部份股權讓給怡和洋行。怡和入主港航後為減低營運成本，放棄了原本的DC-3機隊，改為向其他航空公司租用飛機。

== 結業 ==
1954年，英國海外航空再次入主港航，並購置兩部全新的[[維克斯子爵|維克斯子爵]]式客機（Vickers Viscount），銳意革新。港航增開[[漢城|漢城]]、[[東京|東京]]航線及重開[[馬尼拉|馬尼拉]]航線，但仍然未能擺脫虧損。1958年，[[太古洋行|太古洋行]]旗下的國泰航空全面收購香港航空併入，1959年原牌照繳交。

== 後續 ==
怡和洋行出售香港航空股權後，仍有參與航空業務。旗下的[[怡和航空|怡和航空]]一直獨家代理世界多家航空公司（包括英國航空）在香港的機票業務，至2000年止。在1988年，怡和航空進行重組，並成立了[[怡中航空服務|怡中航空服務]]有限公司，以作為怡和集團與中航的合資公司，專門在機場就旅客、飛行調度、配載平衡、貨運等範疇提供完善的地勤服務。另外，怡和亦有經營在[[香港國際機場|香港國際機場]]的地勤服務。

2005年底，[[海航集團|海航集團]]收購了[[中富航空|中富航空]]的45%股權，並於2006年9月30日改名為[[香港航空|香港航空]]（Hong Kong Airlines）。但該公司與此原有的「香港航空」（Hong Kong Airways）並沒有任何關係。
