'''盧布林—布列斯特攻勢'''（{{lang-ru|Люблин‐Брестская наступательная операция}}，1944年7月18日—8月2日）是由蘇軍實施的[[巴格拉基昂行動|巴格拉基昂行動]]戰略性攻勢的一部分，以掃蕩在波蘭東部和中部的德軍。該攻勢由[[白俄羅斯第1方面軍|白俄羅斯第1方面軍]]的左（南）翼實施及發生在1944年7月，與之對抗的是[[德國南烏克蘭集團軍|德國南烏克蘭]]及[[德國中央集團軍|中央集團軍]]。 

這次行動是伴隨著其他幾個攻勢而展開，特別是在南方[[烏克蘭第1方面軍|烏克蘭第1方面軍]]的[[利沃夫-桑多梅日攻勢|利沃夫-桑多梅日攻勢]]；這幾個攻勢是在北面以掃除在大部分[[白俄羅斯|白俄羅斯]]的[[巴格拉基昂行動|巴格拉基昂行動]]成功後數個星期後實施。

在達到其預期的目標後，該攻勢版繼續直至蘇聯軍隊在8月間進攻[[華沙|華沙]]（1944年8月2日至9月30日）；但是蘇軍并沒有援助波蘭人的[[華沙起義|華沙起義]]，此事引起了一些爭議。

==參戰部隊==
6月15日[[德國南烏克蘭集團軍|德國南烏克蘭集團軍]]在[[費迪南德·舍納爾|費迪南德·舍納爾]]指揮下是由德國和羅馬尼亞軍隊組成，每方均有兩個軍團的兵力：[[羅馬尼亞第3軍團|羅馬尼亞第3軍團]]及[[羅馬尼亞第4軍團|羅馬尼亞第4軍團]]、[[德國第6軍團|德國第6軍團]]（在之前的第6軍團於[[史達林格勒戰役|史達林格勒戰役]]中被消滅後重新建立）及[[德國第8軍團|德國第8軍團]][http://www.axishistory.com/index.php?id=6474]。而[[德國中央集團軍|德國中央集團軍]]擁有[[德國第2軍團|德國第2軍團]]、[[德國第4軍團|德國第4軍團]]、[[德國第9軍團|德國第9軍團]]和[[德國第3裝甲軍團|德國第3裝甲軍團]]。

蘇聯[[白俄羅斯第1方面軍|白俄羅斯第1方面軍]]在[[康斯坦丁·羅科索夫斯基|康斯坦丁·羅科索夫斯基]]指揮下包括第8親衛軍團、第28、第47、第61、第65、第69和第70（混合武器）軍團、[[蘇聯第2親衛坦克軍團|第2親衛坦克軍團]]、[[第6航空軍團|第6]]和[[第16航空軍團|第16航空軍團]]、第11坦克軍、[[波蘭第1軍團|波蘭第1軍團]]、第2和第7親衛騎兵軍。

==攻勢：攻佔盧布林及布列斯特==
經過7月9日至10日蘇軍在[[利沃夫-桑多梅日攻勢|利沃夫-桑多梅日攻勢]]假裝的準備中吸引了德軍的注意後，及該攻勢在第1天（7月13日）取得明顯的成功後，[[康斯坦丁·羅科索夫斯基|康斯坦丁·羅科索夫斯基]]開始自己向西的推進<ref name = "Titans">When Titans…</ref>。

7月18日部署在[[平斯克沼澤|平斯克沼澤]]以南戰線左翼的白俄羅斯第1方面軍之5個軍團（包括1個波蘭軍團，[[波蘭第1軍團|波蘭第1軍團]]），攻擊中並摧毀了[[德國南烏克蘭集團軍|德國南烏克蘭集團軍]]在[[德國第4裝甲軍團|德國第4裝甲軍團]]以西[[科威爾|科威爾]]的防線。在幾個小時內，方面軍的[[蘇聯第2親衛坦克軍|第2親衛坦克軍]]和若干機動軍開始向西突破成功及步兵從後跟隨<ref name = "Glantz">Glantz.</ref>。

[[尼古拉·古謝夫|尼古拉·古謝夫]]中將 的[[蘇聯第47軍團|蘇聯第47軍團]]和[[瓦西里·崔可夫|瓦西里·崔可夫]]上將的[[蘇聯第8親衛軍團|第8親衛軍團]]撕破德軍的防禦和7月21日，他們到達了[[布格河|布格河]]。第2天[[謝苗·波丹諾夫|謝苗·波丹諾夫]]中將的第2坦克軍團開始向盧布林和維斯瓦河前進，而[[蘇聯第11坦克軍|第11坦克軍]]和[[蘇聯第2親衛騎兵軍|第2親衛騎兵軍]]領頭轉各西北面的[[謝德爾採|謝德爾採]]以切斷了防守布列斯特和比亞韋斯托克的[[德國中央集團軍|德國中央集團軍]]之退路<ref name = "Titans" />，在盧布林附近納的粹集中營[[麥達內克|麥達內克]]於7月22日獲得解放。

雖然波格丹諾夫在7月23日於盧布林的戰鬥中受傷和由[[拉德齊夫斯基|拉德齊夫斯基]]少將接任，快速推進持續，令領頭的第8親衛軍團和第2親衛騎兵軍在7月25日到達維斯瓦河沿岸<ref name = "Titans" />。7月24日[[康斯坦丁·羅科索夫斯基|康斯坦丁·羅科索夫斯基]]的部隊攻佔了[[盧布林|盧布林]]，並向西面、華沙以南的[[維斯瓦河|維斯瓦河]]前進<ref name = "Glantz" />。蘇聯最高統帥部命令羅科索夫斯基其部隊轉向北面的華沙前進以切斷中央集團軍部隊之退路<ref name = "Titans" />。

7月28日布列斯特被佔領<ref name="Brest">[http://www.brest-belarus.com/History_1900_2005.shtml#section10  Brest, Belarus]{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}.</ref> 。8月2日白俄羅斯第1方面軍的左翼攻佔維斯瓦河上在[[馬格努谢夫|馬格努谢夫]]（崔可夫的第47軍團）和[[普瓦維|普瓦維]]（弗拉基米爾·科爾帕克奇中將的第69軍團）的[[橋頭堡|橋頭堡]]，並與反擊的“[[德意志國防軍|德意志國防軍]]”部隊展開了近兩個月的爭奪戰，以保留這些重要的橋頭堡作為未來的、[[維斯瓦河-奧得河攻勢|甚至更大規模的攻勢以突入波蘭中部]]朝向[[柏林|柏林]]之起點。 [[德國中央集團軍|中央集團軍]]的[[德國第46裝甲軍|第46裝甲軍]]從8月8日起進行反擊，以圖縮小橋頭堡<ref name=hinzep400>Hinze, p.400</ref> 。[[德軍第19裝甲師|第19]]和“[[第1空降赫爾曼·戈林裝甲師|赫爾曼戈林]]”裝甲師在8月初期間數度展開攻擊，但蘇軍的防線依然堅固<ref name=glantz>See also Glantz, ''Failures of Historiography''</ref><ref name=zalogap78>Zaloga, pp.78-79</ref>。

這一時期進一步戰事包括Studzianki戰役。

==爭論：橋頭堡，而不是華沙==
{{See Also|華沙起義缺乏外間支援}}

在進攻期間，白俄羅斯第1方面軍的左翼接近維斯瓦河，[[波蘭家鄉軍|波蘭家鄉軍]](“Armia Krajowa”)[[華沙起義|在華沙進行起義]]，蘇聯的進攻是加速了起義的其中一個因素，因為波蘭人都指望得到蘇軍的支持，並希望獨自攻佔首都(如部分的[[暴風雨行動|暴風雨行動]])。 

在華沙起義開始(8月1日)前幾天，[[蘇聯最高統帥部|蘇聯最高統帥部]](蘇軍總指揮部)命令羅科索夫斯基將他的第2坦克軍團派往華沙以東郊區([[布拉格(波蘭)|布拉格]])。<ref name = "Glantz" />  7月28日，羅科索夫斯基的軍隊中，有三個軍並進，與[[德軍第73步兵師|德軍第73步兵師]]和[[戈林裝甲師|戈林裝甲師]]在華沙東南40公里處進行戰鬥。設法從東面攻佔進入華沙道路的羅科索夫斯基，和企圖繼續維持這些路線開放及保留華沙控制權的德軍，展開一場競賽。<ref name = "Titans" />

第2坦克軍團在右邊被一個騎兵軍(第2親衛)和[[蘇聯第47軍團|第47軍團]]掩護，但是該軍團的坦克於緩慢的第47軍團能提供支援前在7月29日到達了華沙以東地區；而第47和第2親衛軍團在東面50公里的[[謝德爾採|謝德爾採]]進行戰鬥。而德軍則以2個裝甲軍([[德國第39裝甲軍|第39]]和[[武裝親衛隊第4裝甲軍|武裝親衛隊第4裝甲軍]])實施反擊，即後來所熟知的[[拉濟明戰役_(1944年)|拉濟明戰役]]。7月29日，羅科索夫斯基派出他的[[蘇聯第8親衛坦克軍|第8親衛]]和[[蘇聯第3坦克軍|第3坦克軍]]北上，試圖轉向東北面的華沙和擊倒德國守軍的左翼，而他的[[蘇聯第16坦克軍|第16坦克軍]]繼續在該市東南面郊區的戰鬥。雖然[[波波夫|波波夫]]中將的第8親衛坦克團成功地推進至該市以東20公里範圍內，而[[韋傑涅耶夫|韋傑涅耶夫]]少將的第3坦克軍遇到莫德爾的裝甲部隊一系列的反擊。7月30日開始，赫爾曼·戈林和[[德軍第19裝甲師|第19裝甲師]]，在華沙東北15公里，[[沃沃明|沃沃明]]以北的[[拉濟明|拉濟明]]附近，攻擊被過度削弱了的坦克軍。雖然該些軍經受住了3天的反擊，8月2日和8月3日，[[德軍第4裝甲師|第4]]和武裝親衛隊[[第5維京裝甲擲彈兵師|第5維京裝甲擲彈兵師]]加入了戰鬥。在3天的激烈戰鬥中，第3坦克軍團被嚴重打擊，和第8親衛坦克軍也被非常壓迫。從7月30日王8月5日，德軍成功地把蘇軍迫退，對蘇軍單位造成重大損失。8月5日，第47軍團的部隊已到達該地區，和第2坦克軍團被撤回休整和重組。第47軍團的3個步兵軍分別在前線，從華沙以南到謝德爾採，伸出超過80公里，並無法立即重新有足夠的兵力和快速的裝甲部隊向華沙或對納雷夫河流域前進。德軍向東面的中央集團軍的連繫已遭到破壞，但並未切斷。<ref name = "Titans" /><ref name = "Glantz" />

這次失敗之後，在整個期間直至1944年8月20日，白俄羅斯第1方面軍的第47軍團是唯一主要部署在橫跨華沙對面維斯瓦河的紅軍部隊。蘇軍沒有試圖援助起義，而集中於鞏固西岸。相反，第47軍團集中守衛在河東岸的蘇軍陣地，甚至不向叛亂分子提供炮火支援。在當時，大部分的白俄羅斯第1方面軍中央心和右翼正在努力克服德軍在[[謝德爾採|謝德爾採]]以北朝向[[納雷夫河|納雷夫河]]方向的防線，據蘇聯記錄說，無法實施任何行動直接援助華沙。西方和當代波蘭記錄聲稱[[史達林|史達林]][[沙起義缺乏外間支援|故意不支持波蘭本土軍]]，因為他希望人民軍 - 支持[[波蘭流亡政府|波蘭流亡政府]]與親蘇的[[波蘭民族解放委員會|波蘭民族解放委員會]]之競爭對手 - 被摧毀。<ref name = "Titans" />

8月20日[[齊格蒙特·伯林|齊格蒙特·伯林]]將軍的[[波蘭第1軍團|波蘭第1軍團]]加入了蘇聯第47軍團。在華沙以北的紅軍部隊終於在9月3日渡過[[布格河|布格河]]，翌日迫近[[納雷夫河|納雷夫河]]，並在9月6日進行進入橫跨納雷夫河橋頭堡的戰鬥。第1軍團2個波蘭師的領頭單位在9月13日發動渡過維斯瓦河進入華沙的戰鬥，但沒有取得什麼進展，10天以後被撤回到河的對岸。起義部隊在10月2日投降；蘇軍在1945年初沒有經過一場重要戰役便攻佔了華沙。美國軍事歷史學家[[大衛·格蘭茨|大衛·格蘭茨]]注意到，雖然蘇聯能攻佔華沙和幫助叛亂分子，但從純軍事角度來看這將需要從華沙南北負責保衛橋頭堡的部隊抽出，所涉及的蘇軍將參與代價昂貴的市街戰，並得不到他們的最佳位置以作進一步的攻勢，在這一點上，再加上政治因素意味著，蘇聯決定不援助華沙起義不僅是基於政治，但也跟隨軍事方面的考慮。<ref name = "Glantz" />

==餘波 - 納雷夫河橋頭堡戰役==

在[[西布格河|布格河]]和[[納雷夫河|納雷夫河]]匯流處，[[塞羅茨克|塞羅茨克]]的橋頭堡 ，是由[[蘇聯第65軍團|蘇聯第65軍團]]在盧布林-布列斯特攻勢結束時建立的。[[第2軍團_(德國國防軍)|第2軍團]]中的[[第20軍_(德國國防軍)|第20軍]]被部署在這裡進行防禦。

10月3日，德軍[[第3裝甲師_(德國國防軍)|第3]]和[[第25裝甲師_(德國國防軍)|第25裝甲師]]的單位，在[[第252步兵師_(德國國防軍)|第252步兵師]]的支援下，轉入進攻，以消除第65軍團橋頭堡的陣地。<ref name=scheiderbauer>See the account of Armin Scheiderbauer in Williams and Rodgers, p.109</ref> 在南部，德軍的單位在10月5日到達了納雷夫河河岸。第65軍團司令[[帕維爾·巴托夫|帕維爾·巴托夫]]將軍在回憶錄中，形容[[蘇軍第5步兵師|第44親衛步兵師]]試圖阻止德軍前進。<ref name=batov>See Batov, В походах и боях, Moscow 1962</ref> 

一個攻擊橋頭堡北部的計劃於10月8日實施，由[[第19裝甲師_(德國國防軍)|第19裝甲師]]和武裝親衛隊維金師實施，但取得的成果被蘇軍在10月14日的反擊收復。<ref name=wrp110>Williams and Rodgers, p.110; Batov gives the date as October 19</ref>
