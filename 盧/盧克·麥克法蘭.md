'''盧克·麥克法蘭'''（'''Thomas Luke Macfarlane'''，{{bd|1980年|1月19日}}）是[[加拿大|加拿大]]的一位演員和音樂人<ref>[http://www.ctv.ca/Satisfaction/Cast/Luke-Macfarlane.aspx]</ref>。他出生在安大略省倫敦，曾出演ABC電視劇《[[兄弟姐妹_(美国电视剧)|兄弟姐妹]]》（2006年–2011年）中的{{link-en|Scotty Wandell|Scotty Wandell}}角色，以及在[[SPACE_(電視頻道)|SPACE]]科幻電視劇《[[掃興者|掃興者]]》（2015年–現在）中飾演RAC特工D'avin Jaqobis。

== 個人生活 ==
麥克法蘭於2008年4月15日在接受《[[環球郵報|環球郵報]]》訪問期間公開自己是[[同性戀|同性戀]]身份。<ref>{{Cite web |url = http://www.theglobeandmail.com/arts/a-commitment-to-himself/article1351965/ |title = A commitment to himself |author = Pamela Chelin |date = 2009-03-13 |publisher = The Globe and Mail |language = en}}</ref>
