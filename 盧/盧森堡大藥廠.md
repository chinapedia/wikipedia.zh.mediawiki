'''盧森堡大藥廠'''在1954年於香港成立，創辦人將佩璐医学博士及其夫人珮鲁思化学师所研發的獨特止咳配方引進香港，以[[珮夫人|珮夫人]]為品牌名稱並大量生產。

==旗下品牌==
主要分為[[珮夫人|珮夫人]]及[[珮氏|珮氏]]。除了[[珮夫人|珮夫人]]經典系列的三款止咳露，近年發展不同類型的藥品及健康產品，先後於2007年及2009年推出全新的「珮夫人專業系列」及新品牌[[珮氏|珮氏]]，前者以呼吸道有關之產品為主，後者主攻個人護理與家庭用品，並以創新及功能性作為產品定位。

== 珮夫人 ==
'''珮夫人'''為'''盧森堡大藥廠'''旗下品牌之一，憑其止咳露之獨特配方，成為家傳戶曉的品牌。逾半個世紀以來，珮夫人的銷售網絡已擴展至香港、中國以至美國等地，並且獲頒多個本地及國際殊榮，品質聲譽俱有保證。
產品通過SGS檢定，並符合香港、澳洲、加拿大GMP及美國FDA的標準。2002年更成為大中華地區首間榮獲法國Trade Leaders' Club頒發第30屆國際優質金獎，並於2004年榮獲Superbrands超級品牌殊榮。2006年連續兩年榮獲「中國止咳、化痰用藥市場消費者滿意最具影響力品牌」。
'''盧森堡大藥廠'''一直致力研發更多新產品，令產品提供的類型更全面。發展至今，不論「經典系列」、「草本系列」及「專業系列」均圍繞著呼吸道發展，透過不同的產品類型，滿足不同顧客的需要，務求鞏固珮夫人為「呼吸道專家」的地位。

== 珮氏 ==
'''珮氏'''為'''盧森堡大藥廠'''旗下品牌之一，主攻個人護理與家庭用品，並以創新及功能性作為產品定位，為家庭提供安全有效的優質生活新方案。2009年以全球首創的天然野紅茄提取物IBI-246 （Methyl-Nonyl Ketone）所製成的驅蚊產品「珮氏驅蚊爽」打入市場。「珮氏驅蚊爽」為全港唯一一隻利用專利技術以微形粒子，將有效成份鎖緊於防蚊貼內的產品，不含化學成份避蚊胺（DEET）。

== 公益活動 ==
藥廠積極參與社會公益活動，包括贊助中小學的賣物會及嘉年華會、大學迎新營、長者的一線通平安鐘服務、愛心太極操及文化村院社老人活動等。同事們更獲管理層的支持，組織義工隊，參與義工服務。
* 2006年與[[苗圃行動|苗圃行動]]合作舉辦「珮夫人奧海城苗圃同心齊建校」計劃，為雲南省永德縣小學籌募經費。
* 2008年舉辦「珮夫人我最敬愛的醫生慈善活動」，透過心意卡創作比賽、義賣鼻爽貼等活動，為願望成真基金籌款，幫助患重病的病童實現願望。
* 2008年贊助環境保護署與Take me Home合辦的家居廢物分類比賽。
* 2009年舉辦「做善事 創健力士紀錄」，製作全世界最大防蚊貼，為願望成真基金籌款，有關紀錄已獲健力士世界紀錄大全確認。
* 2010年義賣珮夫人鼻爽潔，為願望成真基金籌款。

== 獎項 ==
2007年首次獲得香港社會服務聯會頒贈「[[商界展關懷|商界展關懷]]」，截至2011年，已經連續4年獲得「商界展關懷」商業機構之加許。
