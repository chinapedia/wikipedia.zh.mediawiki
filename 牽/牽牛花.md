'''牽牛'''（[[学名|学名]]：''{{lang|la|Ipomoea nil}}''）屬[[旋花科|旋花科]][[牵牛属|牵牛属]]，有些地区称它'''喇叭花'''，而日本人称其为'''朝顏'''（{{lang|ja|朝顔、あさがお}}），原產地[[亞熱帶|亞熱帶]][[亞洲|亞洲]]、[[北美洲|北美洲]]。

== 形态 ==
一年或多年生缠绕[[草本植物|草本植物]]，或称[[攀缘植物|藤本植物]]，全株具有短毛。因此種植牽牛花一般在春天播種，夏秋開花，其品種很多，花的顔色有藍、緋紅、桃紅、紫等，亦有複色品种，花瓣邊缘的變化較多，是常見的觀賞植物。果實卵球形，可以入藥。

牽牛葉子通常三裂，基部心形。花呈白色、紫紅色或紫藍色，漏斗狀，雄蕊5枚，长度不一，花丝基部被柔毛，萼片5枚，雌蕊1枚，子房3室，柱头头状。花期以夏季最盛。種子具有藥用價值。

牵牛的近缘种包括[[三色牵牛|三色牵牛]]（''{{lang|la|Ipomoea tricolor}}''）、[[圆叶牵牛|圆叶牵牛]]（''{{lang|la|Ipomoea purpurea}}''）和[[变色牵牛|变色牵牛]]（''{{lang|la|Ipomoea indica}}''）。

== 名稱由來==
[[劉宋|南朝]]醫學家[[陶弘景|陶弘景]]所輯的《名醫別錄》載：「此藥始出山野牽牛謝藥」。<ref>《本草綱目》〈田野人牽牛謝藥〉</ref>

牽牛花的名稱由來有兩個說法：
一是根據[[唐慎微|唐慎微]]《證類本草》記載，有一農夫因為服用牽牛花種子而治好痼疾，感激之餘牽著自家的[[水牛|水牛]]，到田間蔓生牽牛花的地方謝恩<ref>[[唐愼微|唐愼微]]《證類本草》牽牛子下引[[陶弘景|陶弘景]]云：“作藤生花，狀如藊豆，黃色子作小房，實黑色形如梂子核。比來服之，以療腳氣急，得小便利，無不差。此藥始出田野人牽牛易（或作「謝」）藥，故以名之。”</ref>；另一種說法是因為牽牛花的花朵內有星形花紋，花期又與[[牛郎織女|牛郎織女星]]相會的日期相同，故而稱之。

== 药用 ==
{{medical}}
牽牛花是有毒植物，不可胡亂食用。作為中藥用的是牽牛的种子，中藥名為“牽牛子”，别名二丑、黑丑（表面灰黑色）、白丑（表面淡黄白色），主要来源是[[裂葉牽牛|裂葉牽牛]]和[[圓葉牽牛|圓葉牽牛]]兩個品種。﻿<ref>[https://weidetang.weebly.com/blog/pharbitis-seed 牽牛子﻿-惟德堂藥房]</ref>
; 性味：
性寒，味苦
; 功用：
歸[[肺|肺]]、[[腎|腎]]、[[大腸|大腸]]經；有小毒；瀉水，驅蟲。
; 主治：
適用於腹水、腹脹[[便秘|便秘]]、[[蛔蟲|蛔蟲]]病等。

牽牛子含[[棕櫚酸|棕櫚酸]]、[[赤霉素|赤霉素]]甲3、赤霉素甲20、赤霉素甲26、甲27，[[牽牛子酸|牽牛子酸]]丙、丁等。雖有小毒，但可治[[水腫|水腫]]、[[痰滿喘咳|痰滿喘咳]]、[[便秘|便秘]]、[[蛔蟲|蛔蟲]]、[[腳氣|腳氣]]等症。孕婦及氣虛脾虛者忌服。

有毒部位： 全草有小毒，種子毒性強。 
中毒症狀： 幼兒或家畜誤食，有四肢無力、頭昏、食慾不振等症狀。 
急救方法： 催吐 
分佈： 全台低海拔山野及荒地皆有分布，嘉義以南縣市尤多，各離島上也十分常見。 
用途： 可栽培供觀賞。

== 文化 ==
2008年3月14日，[[香港郵政|香港郵政]]推出主題為「香港花卉」的[[郵票|特別郵票]]，其中5元的郵票圖案為牽牛花<ref>[http://www.hongkongpoststamps.com/chi/stamps/latest_stamp_issues/2008/20080314a/introduction.htm 郵票策劃及拓展處 - 香港郵品]{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

2012年5月20日，新成立的[[中華民國文化部|中華民國文化部]]（文建會改制）以牽牛花為部徽，象徵文化草根性<ref>{{Cite news|url=http://news.pts.org.tw/article/210370?NEENO=210370|title=以牽牛花為部徽 象徵文化草根性 - 公視晚間新聞|author=|date=|work=|newspaper=公視新聞網|accessdate=2012-05-16|language=zh-TW}}</ref>。
