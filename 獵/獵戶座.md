'''獵戶座'''（Orion）是一個非常顯著的[[星座|星座]]，也許是夜空中最出名的一個。全世界的人都能看到它那些分佈在[[天赤道|天赤道]]上耀眼的星，也是各地人都認得的星座，也因此獵戶座一直有著「星座之王」的美譽，形如獵人[[俄里翁|俄里翁]]站在[[波江座|波江座]]的河岸，身旁有他的兩頭獵犬[[大犬座|大犬座]]和[[小犬座|小犬座]]，與他一起追逐著[[金牛座|金牛座]]。一些其他的獵物如[[天兔座|天兔座]]都在他的附近。

== 特點 ==

這星座有很多光亮的[[恆星|恆星]]及[[深空天體|深空天體]]。以下是它其中的幾顆星：
* [[觜宿一|觜宿一]]（Meissa，λ）是獵戶的頭。
* [[參宿四|參宿四]]（Betelgeuse，α），獵戶的右肩，是一顆[[紅超巨星|紅超巨星]]──它的直徑比[[木星|木星]][[公轉|公轉]][[轨道_(力学)|軌道]]的直徑還要大。它其實有其他五個伴星，但它們都太小而難以看到。參宿四與[[南河三|南河三]]及[[天狼星|天狼星]]組成[[冬季大三角|冬季大三角]]。
* [[參宿五|參宿五]]（Bellatrix，γ），是獵戶的左肩。
* [[參宿一|參宿一]]（Alnitak，ζ）、[[參宿二|參宿二]]（Alnilam，ε）及[[參宿三|參宿三]]（Mintaka，δ）就是[[獵戶腰帶|獵戶腰帶]]：三顆亮星由東向西連成一線，就是這一個[[星群|星群]]可以令人輕易認出獵戶座。
* [[參宿六|參宿六]]（Saiph，κ）位於獵戶的右膝。
* [[參宿七|參宿七]]（Rigel，β），位於獵戶的左腳踝，是一顆大型的白色的星。它是全天第七亮的星。它有三顆難以看見的伴星。

除參宿四外，獵戶座中主要的星都有相似的年齡及特徵，表示它們可能有一共同的起源。

透過獵戶座我們能夠很容易找到其他的星。把獵戶的腰帶往东南方伸延就能找到[[天狼星|天狼星]]（[[大犬座|大犬座]]α）；向西北方則會碰到[[畢宿五|畢宿五]]（[[金牛座|金牛座]]α）。沿著獵戶的肩膀往東就是南河三（前犬星，[[小犬座|小犬座]]α）。從參宿七往參宿四的方向一直伸延就可見到[[北河二|北河二]]（[[雙子座|雙子座]]α）及[[北河三|北河三]]（雙子座β）。

== 深空天體 ==
[[File:The_stars_in_constellation_Orion.jpg|thumb]]
從獵戶腰帶挂下來的是他的劍，它是由四合星（獵戶座θ1及獵戶座θ2）及[[獵戶座大星雲|獵戶座大星雲]]（[[梅西耶天體|M42]]）所組成，在中國稱為伐星。即使用肉眼也能夠清楚辨認出這個壯麗的天體不是一顆星；如果利用到[[雙筒望遠鏡|雙筒望遠鏡]]，我們更可看到它那些年輕的恒星、發光的氣體及塵埃。

另一著名的星雲就是位於獵戶座ζ的[[馬頭星雲|馬頭星雲]]（[[星雲星團索引|IC 434]]），它的名字來自當中的一團形似馬頭的黑色塵埃。

如果以一支小型[[望遠鏡|望遠鏡]]去觀察獵戶座就能看到更多有趣的深空天體。

== 歷史 ==
[[File:The_hidden_fires_of_the_Flame_Nebula.jpg|thumb]]]]
這一個耀眼的星座在不同的古代文明中都有它的位置，只是形象不同。

古代[[蘇美人|蘇美人]]把這些星視爲一隻綿羊。參宿四的拉丁名Betelgeuse意爲“腋窩”，其實指的就是“綿羊的腋窩”。

在古[[中國|中國]]，獵戶座是[[二十八宿|廿八宿]]之一，即“[[參宿|參宿]]”。-{zh-tw:「參」就是「三」的國字[[中文数字|大寫]]; zh-cn:“参”同“三”的[[中文数字|大写]]“叁”;}-，指獵戶腰帶上的三顆星。中国传统中，有“三星高照”的说法，认为三颗亮星高照，象征吉祥幸福、健康长寿和富裕。“三星”即指[[参宿一|参宿一]]、[[参宿二|参宿二]]与[[参宿三|参宿三]]。还有“三星正南，就要过年”的谚语，表示这三颗星在天空中的位置和农历新年的关系。

古[[埃及|埃及]]人認爲這些星是奉獻給光之神──[[欧西里斯|欧西里斯]]（Osiris）的貢物，一些[[金字塔|金字塔]]與獵戶座有密切聯係。

無論在古代或現代的文學作品中，獵戶的腰帶與劍經常都有出現。它甚至成爲[[美國陸軍|美國陸軍]]第廿七師在[[世界大戰|兩次大戰]]中的佩章，可能是因爲當時這一師是由奧瑞恩將軍（General O'Ryan）所統領。

== 神話 ==

[[File:Orion_constellation_Hevelius.jpg|thumbnail]]筆下的獵戶座]]

在[[希臘神話|希臘神話]]中，有關一個如此顯眼的星座的故事有多個版本並不令人覺得驚奇。

在其中一個版本中，俄里翁揚言自己是世上最偉大的獵人。宙斯的妻子[[赫拉|赫拉]]聽到後就派出一隻蠍子去殺死俄里翁。結果俄里翁被刺死，宙斯對此感到可惜，就把這名獵人放上天上。那只蠍子也被放上天而成爲[[天蠍座|天蠍座]]。有趣的是當這兩個的其中一個從[[地平線|地平線]]升起的時候，另一個就已經落下，所以這對仇人就永遠不會再見。
