{{vgname|獵魂|en=Prey|cn=掠食|tw=獵魂}}是一部有恐怖元素的[[第一人稱射擊遊戲|第一人稱射擊]][[電子遊戲|電子遊戲]]，由[[Arkane_Studios|Arkane Studios]]開發，[[貝塞斯達軟件|貝塞斯達軟件]]公司發行。遊戲於2017年5月5日在[[Microsoft_Windows|Microsoft Windows]]、[[PlayStation_4|PlayStation 4]]和[[Xbox_One|Xbox One]]上發售。

最初{{link-en|人頭工作室|Human Head Studios}}計劃開發2006年遊戲《[[獵魂_(2006年遊戲)|獵魂]]》的續集《[[獵魂2|獵魂2]]》，後來其知識產權從[[3D_Realms|3D Realms]]轉移到貝塞斯達公司而陷入了{{link-en|開發地獄|Development hell}}中，並最終在2014年取消遊戲。貝塞斯達公司於[[電子娛樂展2016|電子娛樂展2016]]中宣佈原裝遊戲的重塑作品《獵魂》，Arkane Studios的[[奧斯丁|奧斯丁]]分部負責開發工作。Arkane的版本不使用任何已被計劃的續集資產，並且只收錄了名稱和在先前遊戲的主角被外星人獵殺的概括主題。Arkane打造《獵魂》作為《''{{link-en|System Shock|System Shock}}''》的[[精神續作|精神續作]]，為玩家提供了幾種在遊戲內的潛在發展。

== 玩法 ==
《獵魂》是一款[[第一人稱射擊遊戲|第一人稱射擊遊戲]]，加上[[電子角色扮演遊戲|角色扮演遊戲]]元素和牢固的敘述。玩家使用的是摩根·余（Morgan Yu）角色，這是一名在有許多敵意外星人的太空站上的人類。玩家可以選擇某些余的特質，包括性別，以及玩家所做的決定會影響遊戲的故事。玩家要控制余來收集及使用武器和資源以抵擋在太空站上的外星人。<ref name="eurogamer aug2016">{{Cite web |url = http://www.polygon.com/2016/7/27/12297122/prey-arkane-studios-bethesda-human-head-system-shock-2 |title = The new Prey will not be like the old Prey |author = Thomas Biery |date = 2016-07-27 |publisher = Polygon |language = en}}</ref>根據創意總監Raphael Colantonio所說，該太空站將會是完全連續不間斷的，而不是單獨的關卡或任務，有時要求玩家回到他們之前已探索的區域。玩家也能夠在零重力下的太空站外面移動，並尋找連接太空站各部分的捷徑。<ref>{{Cite web |url = http://www.eurogamer.net/articles/2016-08-04-prey-reboot-will-be-a-metroidvania-adventure |title = Prey reboot will be a metroidvania adventure |author = Jeffrey Matulef |date = 2016-08-04 |publisher = Eurogamer |language = en}}</ref>Colantonio還表示，外星人有大批不同的能力，玩家的角色可以隨著時間而獲得，這樣的外星人有能力模仿日常物品，例如一張椅子。<ref name="eurogamer aug2016" />

根據首席設計師Ricardo Bare所說，遊戲將會有多個結局，結局分為兩個主要敘述結構，要依靠玩家如何與太空站和倖存人類的大體上互動，但是Bare說會基於特定的事件而有「許多的小組合」。<ref>{{Cite web |url = http://www.gamespot.com/articles/prey-will-have-a-wide-variance-of-endings/1100-6447860/ |title = Prey Will Have "A Wide Variance" Of Endings |author =  Chris Pereira |date = 2017-02-15 |publisher = Gamespot |language = en}}</ref>

== 故事 ==
《獵魂》發生在1963年美國總統[[約翰·甘迺迪|約翰·甘迺迪]]於[[甘迺迪遇刺案|暗殺襲擊]]中倖存下來的[[架空歷史小說|架空歷史]]。作為總統，甘迺迪將更多的資金投入[[美國太空政策|太空計劃]]，使其蓬勃發展。被人類在太空的活動所吸引，許多不同的物種組成了一個外星人部隊，統稱為Typhon，出現並開始攻擊地球。<ref>{{Cite web |url = http://www.gameinformer.com/b/features/archive/2016/12/07/why-preys-aliens-are-different-than-anything-youve-encountered.aspx |title = Why Prey's Aliens Are Different Than Anything You've Encountered |author = Ben Reeves |date = 2016-12-07 |publisher = Game Informer |language = en}}</ref>美國和蘇聯（U.S.S.R.）聯合起來打擊和捕獲Typhon，他們在環繞月球運行的軌道上一起製作了太空站Talos I，以作監禁Typhon之用。隨著時間的推移，這個太空站被不同的團體，幾個不同的裝飾從[[復古未來主義|復古未來主義]]到[[裝飾風藝術|裝飾藝術]]而成。<ref>{{Cite web |url = http://www.vg247.com/2016/09/09/preys-alternate-timeline-depicts-a-future-where-jfk-lived-to-see-the-space-program-flourish/ |title = Prey's alternate timeline depicts a future where JFK lived to see the space program flourish |author = Stephany Nunneley |date = 2016-09-09 |publisher = VG247 |language = en}}</ref>美國最終完全取得所有權，並在監獄區的頂部建立了研究實驗室以研究Typhon。在太空站上的科學家和Typhon之間發生災難性的事故後，美國便關閉了這個計劃。

幾年後，TranStar公司收購了這個太空站。隨著神經科學的進步，他們能夠利用和控制Typhon，然後研究牠們的生理機能來創建Neuromods，這可以重組人類的大腦，給他們新的能力，包括超人類。TranStar從地球上Neuromods的銷售中獲得財政上的成功。遊戲設定的時間大約2032年，TranStar進一步擴大了太空站，為花費兩年時間在太空站定期往返地球之間的工作人員製造合適的生活區。<ref>{{Cite web |url = http://www.gameinformer.com/b/features/archive/2016/12/09/building-preys-interconnected-world.aspx |title = Building Prey's Interconnected World |author = Jeff Cork |date = 2016-12-09 |publisher = Game Informer |language = en}}</ref>

== 開發 ==
{{see also|獵魂2}}
原裝《獵魂》的成功導致在2006年8月宣佈續集《[[獵魂2|獵魂2]]》，由[[3D_Realms|3D Realms]]繼續開發。然而，該項目面臨了一些問題，包括在2011年某天將知識產權轉讓給[[貝塞斯達軟件公司|貝塞斯達軟件公司]]（[[ZeniMax_Media|ZeniMax Media]]旗下）。2011年3月，貝塞斯達宣佈《獵魂2》將改為由{{link-en|人頭工作室|Human Head Studios}}開發，使用修改過的[[id_Tech_4|id Tech 4]]引擎。<ref>{{Cite web |url = https://www.engadget.com/2011/04/18/prey-2-producer-on-taking-new-direction-with-capable-id-tech/ |title = Prey 2 producer on taking new direction, with 'capable' id Tech 4 |author = Jonathan D Deesing |date = 2011-04-18 |publisher = Engadget |language = en}}</ref><ref>{{Cite web |url = http://www.mtv.com/news/2462803/prey-2-announcement/ |title = 'Prey 2' Bringing An 'Open, Alien World' For A 2012 Release From Bethesda Softworks |author = Adam Roseberg |date = 2011-03-14 |publisher = MTV |language = en}}</ref>

2013年5月31日，《''[[Kotaku|Kotaku]]''》報導稱，其開發已經轉移到[[Arkane_Studios|Arkane Studios]]，而且開發已被重啟，廢除所有人頭工作室在《獵魂2》上的工作，並對準在2016年推出。<ref>{{Cite web |url = http://kotaku.com/we-hear-the-people-behind-dishonored-are-now-working-on-510646344 |title = We Hear The People Behind Dishonored Are Now Working On Prey 2 |author = Jason Schreier |date = 2013-05-31 |publisher = Kotaku |language = en}}</ref>經過大約一年的更多傳聞後，貝塞斯達正式取消《獵魂2》，儘管像如下所述，Arkane已經開始製作一個被認為是重啟作的《獵魂》遊戲，而不是續集，這會使用《獵魂》概念，但完全沒有人頭工作室以前的開發。<ref name="gamespot quakecon">{{Cite web |url = http://www.gamespot.com/articles/prey-dev-explains-why-its-named-after-an-unrelated/1100-6442447/ |title = Prey Dev Explains Why It's Named After an Unrelated Series |author = Chris Pereira |date = 2016-08-08 |publisher = GameSpot |language = en}}</ref>

2016年6月12日，貝塞斯達在E3新聞發布會上正式發表重啟作品《獵魂》。遊戲的開發目前由Arkane Studios的CEO和總監Raphaël Colantonio與他的團隊在[[德克薩斯州|德克薩斯州]][[奧斯丁|奧斯丁]]帶領。<ref>{{Cite web |url = http://www.vg247.com/2016/06/13/e3-2016-prey-re-revealed/ |title = Prey coming 2017 from Dishonored dev's Austin studio |author = James O'Connor |date = 2016-06-13 |publisher = VG247 |language = en}}</ref>{{link-en|Chris Avellone|Chris Avellone}}也確認正在這個項目上工作。<ref>{{Cite web |url = http://www.pcgamer.com/bethesda-confirms-that-prey-is-coming-back/? |title = Prey is back, and there's a trailer |author = Andy Chalk |date = 2016-06-13 |publisher = PC Gamer |language = en}}</ref>根據Colantonio所說，新的《獵魂》不是一個真正的續集，而是一個「知識產權的重新想像」。<ref>{{Cite web |url = http://www.gamespot.com/articles/prey-revealed-at-e3-2016-not-a-sequel-or-remake-up/1100-6440736/ |title = Prey Revealed at E3 2016, Not a Sequel or Remake [UPDATE] |author = Chris Pereira |date = 2016-06-16 |publisher = GameSpot |language = en}}</ref>根據[[CNET|CNet]]的Seamus Byrne的說法，2016年E3展上展示的前導預告片展示了遊戲的主角像「《[[偷天情緣|偷天情緣]]》的太空恐怖版」。貝塞斯達的營銷副總裁Pete Hines解釋說，新遊戲沒有任何元素來自已被取消的《獵魂2》。<ref name="gamespot announce"> {{Cite web |url = http://www.gamespot.com/articles/new-prey-has-nothing-to-do-with-cancelled-prey-2-b/1100-6440943/ |title = New Prey Has Nothing to Do With Cancelled Prey 2, Bethesda Head Says |author = Mat Paget |date = 2016-06-15 |publisher = GameSpot |language = en}}</ref>

== 擴充內容 ==
在{{link-en|2018年電子娛樂展|Electronic Entertainment Expo 2018}}期間，貝塞斯達宣布在2018年6月9日發布免費更新，為《獵魂》增加了兩種新的遊戲模式，包括[[二周目|二周目]]模式，以及生存模式。貝塞斯達也宣布本遊戲的第一個DLC「Mooncrash」，這個DLC是一個[[程序化生成|程序化生成]]的冒險活動，也在當天可以買到。他們還宣布計劃發布一個多人模式「Typhon Hunt」，其中一些玩家扮演Typhon怪物，可以在遊戲中偽裝自己。<ref>{{Cite web |url =https://kotaku.com/preys-dlc-is-called-mooncrash-infinitely-replayable-1826717511 |title = Prey's DLC Is Called Mooncrash, 'Infinitely Replayable' |author = Riley MacLeod |publisher = Kotaku |language = en}}</ref>
