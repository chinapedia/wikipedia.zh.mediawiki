{{Vgname|獵魂2|en=Prey 2}}是一款製作中止的[[第一人稱射擊遊戲|第一人稱射擊]][[電子遊戲|電子遊戲]]，由{{link-en|人頭工作室|Human Head Studios}}開發，[[貝塞斯達軟件公司|貝塞斯達軟件公司]]負責發行。該作為2006年遊戲《[[獵魂|獵魂]]》的續集。

== 情節 ==
《獵魂2》的故事集中在[[美國法警|美國法警]]Killian Samuels，他於遊戲開始時是在一班航機上，卻突然墜毀在球體上（the Sphere，其墜毀曾在原裝《獵魂》中展示出來）。在與一些外星人的短暫戰鬥結束時，他被打昏至不省人事，之後的情節便跳進了幾年。Samuels在外星世界Exodus中是一名[[賞金獵人|賞金獵人]]。雖然他知道他的職業，並保留了他的技能，但他對他被綁架以來所發生的事情沒有記憶。他最初相信自己是唯一一個在Exodus上的人類，直到他碰到Domasi "Tommy" Tawodi（原裝《獵魂》的主人公）。Killian恢復他的記憶，同時也繼續他的賞金獵人活動。

== 開發 ==
在《[[獵魂|獵魂]]》發行不久後，[[3D_Realms|3D Realms]]發布續作《獵魂2》的開發已經進行<ref>{{cite web |title=Next-Gen People: Scott Miller |url=http://www.next-gen.biz/features/next-gen-people-scott-miller |work=next-gen.biz |publisher=Edge |date=2006-08-09 |accessdate=2009-06-03 |deadurl=yes |archiveurl=https://web.archive.org/web/20110815201206/http://www.next-gen.biz/features/next-gen-people-scott-miller |archivedate=2011-08-15 }}</ref>，但{{link-en|人頭工作室|Human Head Studios}}並未開始開發工作。直至2009年，3D Realms將《獵魂》的版權轉讓給貝塞斯達軟件的母公司[[ZeniMax_Media|ZeniMax Media]]。貝塞斯達軟件公司於2011年早期正式發布標題，展示了玩家的主角和遊戲性變化到一個更[[開放世界|開放的世界遊戲]]。人頭工作室在2011年底低調地停止了遊戲的開發，儘管已經進展到接近[[軟件版本週期#Alpha|預覽版本]]狀態，但卻沒有說明原因。其後，有一些謠言散布《獵魂2》已被取消或更換開發商，包括[[Arkane工作室|Arkane工作室]]接管開發的證據<ref>{{cite web|url=http://www.rockpapershotgun.com/2013/08/02/bethesda-talks-prey-2-denies-arkane-involvement/|title=Bethesda Talks Prey 2, Denies Arkane Involvement|first=Nathan|last=Grayson|work=[[Rock,_Paper,_Shotgun|Rock, Paper, Shotgun]]|date=2013-08-02|accessdate=2015-04-07|language=en |deadurl=yes |archiveurl=https://web.archive.org/web/20180813184837/https://www.rockpapershotgun.com/2013/08/02/bethesda-talks-prey-2-denies-arkane-involvement/|archivedate=2018-08-14}}</ref>。2014年，貝塞斯達軟件公司的公關行銷副總裁Pete Hines在PAX Australia展覽宣布遊戲製作中止的消息<ref>{{cite web|url=https://gnn.gamer.com.tw/1/105941.html|title=Bethesda 證實《獵魂 2》取消開發 原研發團隊 Human Head Studios 發表看法
|publisher=巴哈姆特|date=2014-11-04|accessdate=2018-08-14|language=zh|deadurl=yes|archiveurl=https://web.archive.org/web/20180813185228/https://gnn.gamer.com.tw/1/105941.html|archivedate=2018-08-14}}</ref>，說明遊戲並沒有達到他們的期望<ref>{{Cite web |url=http://www.polygon.com/2014/10/31/7137487/prey-2-is-officially-dead |title=Prey 2 is officially dead |author=Justin McElroy |date=2014-10-31 |publisher=Polygon |language=en |deadurl=yes |archiveurl=https://web.archive.org/web/20180813184135/https://www.polygon.com/2014/10/31/7137487/prey-2-is-officially-dead |archivedate=2018-08-14}}</ref>。

後來在2016年，貝塞斯達軟件公司宣布其專利權的重啟版《[[獵魂_(2017年遊戲)|獵魂]]》，準備在2017年發售，並由Arkane工作室開發，會採取《獵魂》的概念和主題元素，但報廢了任何以前人頭工作室所做的工作<ref>{{Cite web |url=http://www.pcgamesn.com/prey/prey-release-date-story-setting |title=Prey: everything we know - release date, setting, story and more |date=2017-01-03 |publisher=PCGamesN |language=en|deadurl=yes|archiveurl=https://web.archive.org/web/20180813183818/https://www.pcgamesn.com/prey/prey-release-date-story-setting|archivedate=2018-08-14}}</ref>。
