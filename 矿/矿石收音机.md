'''矿石收音机'''，简称'''矿石机'''，是利用天然[[矿石|矿石]]或晶体[[二极管|二极管]]作为检波元件，加上[[天线|天线]]、[[地线|地线]]、[[调谐电路|调谐电路]]和耳机等制成的[[收音机|收音机]]。

== 历史 ==
第一次由Greenleaf Whittier Pickard在大约1900年制造，并且使用条形[[晶体|晶体]]矿石来探测[[无线电|无线电]]信号。
 
人們第一次製造和使用了不需要[[電池|電池]]或電能的、簡單低價的礦石收音機。即使在[[第一次世界大戰|第一次世界大戰]]之後[[電子管|電子管]]收音機被普遍使用, 礦石收音機依舊非常流行, 特別是在初級[[業餘無線電|業餘無線電]]愛好者和學生當中。他們製作礦石收音機以學習基本的[[電子|電子]]和[[通信|通信]]知識。

== 构成 ==
矿石收音机是一个简单的[[无线电|无线电]][[接收机|接收机]]，由长导线[[天线|天线]]，用于选择信号[[频率|频率]]的[[调谐器|调谐器]]和[[检波器|检波器]]组成。
检波器可以使用[[晶体|晶体]]矿石,也可使用检波[[二极管|二极管]]，还可以使用锗高频三极管检波器。常见的可用作检波器的矿石种类有[[硫化铅矿|硫化铅矿]]、[[方铅矿|方铅矿]]、[[红亚铅矿|红亚铅矿]]、[[黄铜矿|黄铜矿]]、[[黃鐵礦|黃鐵礦]]（愚人金）和自然铜等。<ref>{{cite web|author=林葆浏 |title=矿石收音机怎样接收广播 |url=http://www.pa555.com/display.php?id=832 |publisher=公共广播技术网 |accessdate=2013-04-15 |deadurl=yes |archiveurl=https://web.archive.org/web/20121219211148/http://www.pa555.com/display.php?id=832 |archivedate=2012-12-19 }}</ref>

== 谍报 ==
矿石收音机能依然被[[间谍|间谍]]使用。矿石收音机没有[[振盪器|振盪器]]，因此间谍组织不能侦测到监听與被監聽的频率。
