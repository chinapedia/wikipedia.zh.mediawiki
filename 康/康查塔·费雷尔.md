'''康查塔·费雷尔'''（[[英文|英文]]：'''Conchata Galen Ferrell'''，{{bd|1943年|3月28日|catIdx=C}}）是一名[[美国|美国]]演员，知名于在[[情景喜剧|情景喜剧]]《[[好汉两个半|好汉两个半]]》中扮演管家波塔。康查塔也因此在2005年和2007年两次被[[艾美奖|艾美奖]]提名。

== 早年生活 ==
康查塔出生于[[美国|美国]][[西弗吉尼亚州|西弗吉尼亚州]][[查尔斯顿_(西弗吉尼亚州)|查尔斯顿]]<ref>[http://www.filmreference.com/film/91/Conchata-Ferrell.html Conchata Ferrell Biography (1943-)<!-- Bot generated title -->]</ref>。高中毕业后，康查塔前往西弗吉尼亚大学读书，并在随后拿到马歇尔大学的社会学学位。康查塔的丈夫名为阿尼·安德森，二人育有一女<ref>[http://www.cbs.com/primetime/two_and_a_half_men/bio/conchata_ferrell/bio.php Cast Bios for the CBS Primetime Comedy TV Show "Two and a Half Men" - CBS.com<!-- Bot generated title -->] {{webarchive|url=https://web.archive.org/web/20080418040510/http://www.cbs.com/primetime/two_and_a_half_men/bio/conchata_ferrell/bio.php |date=2008-04-18 }}</ref>。

== 职业生涯 ==
康查塔早期曾在剧场工作，并出演了一些戏剧。在随后的几十年间，康查塔在《迪兹先生》、《[[K-PAX|K-PAX]]》、《都市郊区的罪与罚》、《[[永不妥协|永不妥协]]》、《[[剪刀手爱德华》|剪刀手爱德华》]]、《现代灰姑娘》、《电视台风云》等电影中担任配角。除电影外，康查塔也在一些电视剧中客串或出演配角。2003年，康查塔开始在情景喜剧《[[好汉两个半|好汉两个半]]》中扮演管家波塔，该部喜剧大获成功，也使得康查塔获得了两次[[艾美奖|艾美奖]]提名<ref>{{cite web|url=http://www.playbill.com/news/article/149900-Minka-Kelly-Susan-Sullivan-Set-for-Love-Loss-Off-Broadway |title=Minka Kelly, Susan Sullivan Set for Love, Loss... Off-Broadway |accessdate=2011-04-21 |date=2011-04-14 |work=Playbill |author=Gans, Andrew |deadurl=yes |archiveurl=https://web.archive.org/web/20110420223347/http://www.playbill.com/news/article/149900-Minka-Kelly-Susan-Sullivan-Set-for-Love-Loss-Off-Broadway |archivedate=2011-04-20 }}</ref>。
