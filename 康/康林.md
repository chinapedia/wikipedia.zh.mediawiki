'''康林'''（{{bd|1916年|5月|1995年|7月9日}}），出生于[[江西省|江西省]][[于都县|于都县]]。[[中国人民解放军少将|中国人民解放军少将]]。[[中国共产党第九届中央委员会候补委员|中国共产党第九]]、[[中国共产党第十届中央委员会候补委员|十]]、[[中国共产党第十一届中央委员会候补委员|十一届中央委员会候补委员]]<ref name="a1"/>。

== 生平 ==
出身于一个农民家庭<ref name="a1"/>，幼年放牛，受过两年教育，亦曾为木匠学徒两年<ref name="a2"/>。1932年3月，参加中国工农红军。同年9月，加入[[中国共产主义青年团|中国共产主义青年团]]。1933年9月，转为中国共产党党员。参加了五次[[中央苏区反围剿战爭|中央苏区反围剿战爭]]。历任警卫员、班长、特务员、副指导员、参谋<ref name="a1">{{Cite web |url=http://dangshi.people.com.cn/GB/165617/165618/166491/167909/12289532.html |title=康林|accessdate=2018-06-16 |author= |date= |work= |publisher=人民网|language=Zh-hans }}</ref>、[[新四军|新四军]]第一支队教导大队队长、[[新四军第一师|第一师]]三旅七团营长、如皋警卫团参谋长等职。1944年4月，任南通县警卫团团长<ref name="a2">{{Cite web |url=http://sz.tz.gov.cn/tzsz/main.html |title=第三十三篇人物》第一章人物传略|accessdate=2018-06-16 |author= |date= |work= |publisher=南通市通话区史志网，来源：《通州市志》（1993-2009）|language=Zh-hans }}</ref>。[[第二次国共内战|第二次国共内战]]期间，担任华东野战军第四纵队十一师参谋长，先遣纵队第四支队参谋长，[[第三野战军|第三野战军]]24军73师师长，参加[[孟良崮战役|孟良崮战役]]。

[[朝鲜战争|朝鲜战争]]中，率领72师参加[[上甘岭战役|上甘岭战役]]<ref>{{cite news|author=皮效农　皮卫华|title=父亲皮定均在上甘岭的日日夜夜|url=http://www.gmw.cn/content/2010-07/07/content_1170449.htm|accessdate=2014-01-01|newspaper=光明网}}</ref>；后升任[[中国人民解放军第24军|中国人民解放军第24军]]副军长，后改任[[中国人民解放军第28军|中国人民解放军第28军]]副军长、军长，[[北京军区|北京军区]]副司令员。1961年晋升为少将。获三级[[八一勋章_(1955年)|八一勋章]]、二级[[独立自由勋章|独立自由勋章]]和一级[[解放勋章|解放勋章]]<ref>{{cite news|title=中国人民解放军开国将军－－康林少将|url=http://www.10000xing.cn/x088/2008files/20090305142944953.html|accessdate=2014-01-01|newspaper=万家姓}}</ref>。

1995年7月，康林在北京逝世。安葬于通州市干部公墓（位于今南通市[[通州区_(南通市)|通州区]]）。[[彭冲|彭冲]]题词“戎马一生，丹心报国”<ref name="a2"/>。
