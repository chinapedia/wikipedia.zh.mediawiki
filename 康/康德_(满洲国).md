'''康德'''（1934年3月1日－1945年8月17日）是[[溥仪|溥仪]]在[[滿洲國|滿洲國]]執政所使用的第二個[[年號|年號]]。

== 大事紀 ==
[[File:_Manchukuo_0.5_FEN.jpg|thumb]]
*元年3月1日：[[溥儀|溥儀]]在首都[[新京|新京]]南郊杏花村舉行登基典禮，改“'''滿洲國'''”為“'''大滿洲帝國'''”，改元'''康德'''。
*二年3月21日：[[张景惠|张景惠]]取代[[鄭孝胥|鄭孝胥]]成為滿洲國第二任國務總理大臣。
*二年4月6日：滿洲國皇帝溥儀第一次訪問[[日本|日本]][[東京|東京]]。
*七年6月26日：滿洲國皇帝溥儀第二次訪問日本東京，[[昭和天皇|昭和天皇]]親自迎接。
*十年4月1日：[[日本首相|日本首相]][[東條英機|東條英機]]訪問滿洲國。
*十二年8月10日：[[蘇聯|蘇聯]][[蘇聯紅軍|紅軍]]開始進攻滿洲國。
*十二年8月17日：[[溥仪|溥仪]]宣讀退位詔書，滿洲國滅亡。
*十二年8月20日：蘇聯紅軍佔領滿洲國首都新京。

== 出生 ==
*二年4月25日——[[李敖|李敖]]，著名歷史學者，生於滿洲國[[哈尔滨市|哈爾濱]]。
*二年7月14日——[[根岸英一|根岸英一]]，2010年[[諾貝爾化學獎|諾貝爾化學獎]]得主，生於滿洲國[[新京|新京]]（今[[长春市|長春]]）。
*二年——[[王洪文|王洪文]]，[[四人幫|四人幫]]之一，生於[[新京|新京]]。
*五年2月26日——[[愛新覺羅·慧生|愛新覺羅·慧生]]，末代皇弟[[溥傑|愛新覺羅·溥傑]]與其日本籍妻子[[嵯峨浩|嵯峨浩]]的長女，生於[[新京|新京]]。
*九年1月15日——[[鍾鐵民|鍾鐵民]]，[[台灣|台灣]]作家，[[鍾理和|鍾理和]]之子，生於滿洲國[[沈阳市|奉天]]（今[[瀋陽市|瀋陽市]]）。
*十年4月13日——[[劉家昌|劉家昌]]，知名音樂人，生於[[哈尔滨市|哈爾濱]]。
*十年6月——[[徐才厚|徐才厚]]，[[中國|中國]]將領、[[中國共產黨|中國共產黨]][[中国共产党第十七届中央委员会|第十七屆]][[中国共产党中央政治局|中央政治局]]委員，生於[[奉天省_(滿洲國)|奉天省]]復縣（今[[瓦房店市|瓦房店市]]）。
*十一年10月——[[回良玉|回良玉]]，[[中國|中國]]領導人、[[中國共產黨|中國共產黨]]第十六、十七屆[[中央政治局|中央政治局]]委員，生於[[吉林省_(滿洲國)|吉林省]]榆樹縣（今[[榆樹市|榆樹市]]）。

== 逝世 ==
*五年3月28日——[[鄭孝胥|鄭孝胥]]，滿洲國首任國務總理大臣，卒於[[新京|新京]]。
*九年8月14日——[[譚玉齡|譚玉齡]]，[[滿族|滿族]]，[[溥儀|溥儀]]妃嬪，卒於[[新京|新京]]。
