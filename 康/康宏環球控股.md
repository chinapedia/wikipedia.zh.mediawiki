'''康宏環球控股有限公司'''（{{hkex|1019}}，'''「康宏環球」'''）， 是香港的金融集團，於1993年成立，由一間十人本地公司，迅速發展至今日擁有數千位專業理財顧問的亞洲企業，並於2010年在[[香港交易所|香港交易所]]上市。目前公司主要發展三大業務，包括理財顧問、金融產品及[[金融科技|金融科技]]。旗下子公司'''康宏理財服務有限公司'''（'''「康宏理財」'''） 從事[[獨立理財顧問|獨立理財顧問]]（Independent Financial Adviser，IFA）業務，致力從客戶角度出發，揀選及建議最合適的理財方案，規劃人生。

在[[大中華地區|大中華地區]]，集團於[[北京|北京]]及[[深圳|深圳]]設立附屬公司及合營企業，提供人壽及一般保險經紀業務、風險管理諮詢服務、再保險經紀服務、以及機構層面上的保險及投資策劃諮詢服務等<ref>[http://www.etnet.com.hk/www/tc/news/special_news_detail.php?category=%E5%B0%88%E8%A8%AA&newsid=ETN190929684 康宏理財業務「往北望」]經濟通，2009年9月</ref><ref>[http://www.convoyfinancial.com/zh_TW/about-convoy/corporate-news/20100419 康宏理財踴躍參與及贊助保良局慈善保齡球賽] {{webarchive|url=https://web.archive.org/web/20110727145115/http://www.convoyfinancial.com/zh_TW/about-convoy/corporate-news/20100419 |date=2011-07-27 }}康宏新聞稿，2010年4月19日</ref>。

== 公司歷史 ==
康宏環球控股有限公司（「康宏環球」）於1993年成立，旗下子公司康宏理財服務有限公司（「康宏理財」）於1998年轉型並引入獨立理財顧問（IFA）概念，開展理財策劃業務，2010年公司在[[香港交易所|香港交易所]]上市，股份編號1019；2012年推出網上投保平台iConvoy；2016年公司改名為「康宏環球控股有限公司」；2018年公司重組業務，聚焦三大核心零售板塊：理財顧問、金融產品及金融科技。

2015年7月9日康宏環球公布，以每股0.53元向[[富邦|富邦]][[蔡明興|蔡明興]]及其家族控股的Eagle Legacy Limited及Oceana Glory Limited配售32.94億股新股，集資淨額16.8億元，該批股份佔經配發及發行認購股份擴大後之已發行股份29.85%，蔡明興及其家族將成為康宏最大單一股東。

2017年11月，[[佳兆業|佳兆業]]（1638）大股東兼董事會主席[[郭英成|郭英成]]兒子[[郭曉群|郭曉群]]，現時持有44億4818萬股，相當於百分之29.91股權，成為集團的第二大股東。

2017年12月7日，[[廉政公署_(香港)|廉政公署]]及[[證券及期貨事務監察委員會|證監會]]採取聯合行動，出示搜查令到康宏集團的總部辦公室搜查，調查康宏集團在股份交易及借貸活動所涉嫌出現的貪污，行動中拘捕集團董事局副主席[[馮雪心|馮雪心]]及執行董事陳麗兒<ref>{{cite news|title=ICAC與證監會今搜8處所 拉3上市公司高層|url=https://hk.news.appledaily.com/breaking/realtime/article/20171207/57550810|accessdate=2017-12-08|publisher=蘋果日報|date=2017-12-07}}</ref>。康宏在港交所上市的股份（1019）在當日上午主動申請停牌<ref>{{cite news|title=3高層涉貪被查 康宏停牌|url=http://paper.wenweipo.com/2017/12/08/HK1712080016.htm|accessdate=2017-12-08|publisher=文匯報|date=2017-12-08}}</ref>。集團董事局主席[[王利民|王利民]]於12月8日被廉署拘捕<ref name="st-2017-12-09a">{{cite news|title=康宏王利民被捕 廉署或通緝曹貴子|url=http://std.stheadline.com/daily/news-content.php?id=1711469&target=2|publisher=星島日報|date=2017-12-09}}</ref>。

2017年12月10日，康宏環球宣佈委任陳志宏先生為董事會臨時主席和執行董事，另外委任葉怡福和葉宜君為執行董事，有關任命由12月9日生效。<ref>{{cite web |url=https://www.hk01.com/社會新聞/140295/康宏風暴-董事會換班-陳志宏任臨時主席-29日開股東特別大會 |title=康宏風暴-董事會換班-陳志宏任臨時主席-29日開股東特別大會 www.hk01.com 2017-12-10 |access-date=2018-07-28}}</ref>

== 品牌重塑 ==
康宏於2018年進行品牌重塑<ref>[https://www.convoy.com.hk/zh/about-convoy/media-and-news/corporate-news/20180427 康宏全新標誌 以誠信、熱誠和專業，打造IFA新世代] 康宏新聞發佈. 2018-04-27.</ref> ，並在4月27日公佈全新的公司標誌，象徵公司邁進新里程。新標誌由紅、白、藍三種顏色組成，分別代表熱誠、誠信和專業，充分展示康宏的核心價值，並以此釐定IFA業界新標準，展示「IFA就是CONVOY」的精神。全新的公司標誌以正方形為主，代表公正客觀、可靠穩重的公司形象；正中的白色三角形向上指，不單寓意公司持續發展，更象徵康宏會從客戶角度，提供最佳理財方案。全新標誌，進一步強調誠信、熱誠和專業的康宏核心價值，貫徹「在你角度，投資人生路」的獨立理財顧問精神，全力打造IFA新世代。

康宏亦為品牌重塑推出大型宣傳活動<ref>[https://www.convoy.com.hk/en/about-convoy/media-and-news/corporate-news/20180605 康宏全新電視廣告 展現IFA行業價值：熱誠、誠信、專業 陪伴客戶在人生路上作最佳理財選擇] 康宏新聞發佈. 2018-06-05.</ref>，包括於6月推出三個全新電視廣告，以溫情手法帶出三位IFA的生活故事，突顯熱誠、誠信、專業IFA三大價值，為行業定下標準。三個廣告分別以夫妻、父子和姊妹三種關係，以生活寫照道出IFA的故事，演繹康宏IFA具備的行業標準，並與客戶同行，陪伴他們在人生路上作出最合適他們的理財決策。
