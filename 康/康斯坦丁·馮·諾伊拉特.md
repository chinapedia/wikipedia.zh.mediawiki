'''康斯坦丁·馮·紐賴特男爵'''（'''Konstantin Freiherr von Neurath'''，{{bd|1873年|2月2日|1956年|8月14日|catIdx=Neurath}}） 是一位[[德國|德國]][[外交官|外交官]]、[[政治人物|政治人物]]，1932年至1938年擔任德國外交部長；1939年至1941年擔任[[波希米亞和摩拉維亞保護國|波希米亞和摩拉維亞保護國]]保護者（[[總督|總督]]），仍保留該保護者名號至1943年。

== 早年 ==
[[File:Bundesarchiv_N_1310_Bild-011,_Konstantin_von_Neurath.jpg|thumb]]
紐賴特1873年出生於[[符騰堡王國|符騰堡王國]]的[[恩茨河畔韋興根|恩茨河畔韋興根]]，為不顯注的[[士瓦本|士瓦本]]貴族之子。他於[[杜賓根大學|杜賓根大學]]及[[柏林洪堡大學|柏林洪堡大學]]攻讀法律，1892年畢業並回到家鄉當地的法律事務所工作。1901年進入政府部門並在[[柏林|柏林]]的外交部辦公室工作。1903年被派赴[[倫敦|倫敦]]的{{Link-de|德國駐英大使館|Deutsche Botschaft London}}，1909年起作為該館的法律顧問。1914年被派往[[伊斯坦堡|伊斯坦堡]]。
1901年5月30日與Marie Auguste Moser von Filseck於[[斯圖加特|斯圖加特]]結婚，1902年生一子Konstantin，1904生一女Winifred 。

一戰期間，他作為一步兵團軍官服役至1916年其重傷。1914年獲[[鐵十字勳章|鐵十字勳章]]。其後重回外交領域派駐在[[奧圖曼帝國|奧圖曼帝國]]。戰爭結束，其帶領符騰堡政府。

== 政治生涯 ==
[[File:Bundesarchiv_N_1310_Bild-014,_Konstantin_von_Neurath.jpg|thumb]]
1919被派駐[[丹麥|丹麥]][[哥本哈根|哥本哈根]]，1921年至1930年為駐[[義大利|義大利]]大使。1929年曾考慮加入德國內閣。1930年擔任駐[[英|英]]大使。

1932年，紐賴特被召回擔任[[帕彭|帕彭]]內閣的外交部長，持續在之後的[[庫爾特·馮·施萊謝爾|施萊謝爾]]及[[阿道夫·希特勒|希特勒]]內閣裡，擔任該職位至1940年。在希特勒擔任總理早期時，紐賴特增添了希特勒擴張政策的光環。

他曾參與1933年德國退出[[國際聯盟|國際聯盟]]、1935年英德海軍條約之制定及重佔萊茵蘭。1937年加入[[納粹黨|納粹黨]]及同年9月被授予[[黨衛隊|黨衛隊]]上將榮譽軍銜。

1938年2月4日，紐賴特被解除外交部長職務。他感受到他被希特勒邊緣化也不符希特勒的積極擴張的口味。外交部長被[[里賓特洛甫|里賓特洛甫]]接任，他仍保留[[不管部部長|不管部部長]]一職。

1939年3月，紐賴特被任命為[[波希米亞和摩拉維亞保護國|波希米亞和摩拉維亞保護國]]保護者（[[總督|總督]]），他作為希特勒於該地區的代理人，建立德國的法治以控制媒體及廢除政黨、工會。

1939年10、11月期間，強硬鎮壓[[學運|學生抗議活動]]。希特勒仍認為其統治手段過於溫和，在1941年9月逐漸減少其職務，並以[[海德里希|海德里希]]取代，1941年紐賴特已提出辭呈，但至1943年8月才被批准。

== [[紐倫堡審判|紐倫堡審判]] ==
紐賴特被指控四項罪名皆成立，被判15年徒刑。1954年因心臟疾病提早獲釋，1956年於恩茨韋英根逝世。
