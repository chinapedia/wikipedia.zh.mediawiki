'''楔蚌属'''（[[学名|学名]]：{{lang|la|'''''Cuneopsis'''''}}）为一個在淡水水域生活{{r|WoRMS_747072}}的[[雙殼綱|雙殼綱]][[蚌科|蚌科]]动物<ref name='id11267'>{{cite web| url =http://www.bioinfo.cn/db05/BjdwSpecies.php?action=view&id=11267| title =鱼尾楔蚌| work =《[[中国动物物种编目数据库|中国动物物种编目数据库]]》| publisher =中国科学院微生物研究所| author =中国科学院动物研究所| accessdate =2009-04-28| deadurl =yes| archiveurl =https://web.archive.org/web/20160305031859/http://www.bioinfo.cn/db05/BjdwSpecies.php?action=view&id=11267| archivedate =2016-03-05}}</ref>，只在[[東亞|東亞]]地區出現。

==物種==
綜合[[美國|美國]][[威斯康辛州大學斯提芬斯岬分校|威斯康辛州大學斯提芬斯岬分校]]的[[MusselP數據庫|MusselP數據庫]]及[[中国科学院|中国科学院]]的《[[中国动物物种编目数据库|中国动物物种编目数据库]]》記載，本屬現時有六個物種<ref>{{cite web |url=http://mussel-project.uwsp.edu/db/db.php?p=div&l=gen&n=251 |title=''Cuneopsis'' Simpson, 1900: species |publisher=[[MusselP數據庫|MusselP數據庫]] |language=en |accessdate=2015-04-13 }}</ref>，分別如下：
*  [[巨首楔蚌|巨首楔蚌]] ''Cuneopsis capitatus'' (Heude, 1874)：中國特有物種，見於[[長江|長江]]流域。
* ''[[Cuneopsis_celtiformis|Cuneopsis celtiformis]]'' (Heude, 1874)：中國特有物種，見於長江流域及[[江西|江西]][[湖南|湖南]]兩省。
* ''[[Cuneopsis_demangei|Cuneopsis demangei]]'' (Haas, 1929)：本屬唯一在中國以外的物種，見於[[越南|越南]]東北部[[富壽省|富壽省]][[越池市|越池市]]的[[沱江_(紅河)|沱江]]流域，因為沱江興建水壩而令其棲息地被破壞，所以被評為「極危物種」<ref>{{cite web |url=http://www.iucnredlist.org/details/171828/0 |title=Cuneopsis demangei |work=[[IUCN|IUCN]] Red List 2014.03 |date=2014-03 |accessdate=2015-04-13 |language=en }}</ref>，估計已在野外滅絕<ref>{{cite web |url=http://cmsdocs.s3.amazonaws.com/summarystats/2017-1_Summary_Stats_Page_Documents/2017_1_RL_Stats_Table_9.pdf |title=Table 9: Possibly Extinct and Possibly Extinct in the Wild Species |work=[[IUCN|IUCN]] Red List 2017.01 |date=2017-05-04 |accessdate=2017-07-21 |format=[[PDF|PDF]] |language=en }}</ref>。
* [[圓頭楔蚌|圓頭楔蚌]] ''Cuneopsis heudei'' (Heude, 1874)：中國特有物種，遍佈全國。
* [[魚尾楔蚌|魚尾楔蚌]] ''Cuneopsis pisciculus''  (Heude, 1874)：中國特有物種，見於[[黃河|黃河]]及長江流域。
* [[微紅楔蚌|微紅楔蚌]] ''Cuneopsis rufescens''  (Heude, 1874)：中國特有物種，見於長江流域。
