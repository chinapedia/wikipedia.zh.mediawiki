在'''前5千纪'''（或称'''前第5个千年'''）时代，[[亚洲|亚洲]]、[[北非|北非]]和[[欧洲|欧洲]]南部的人类已经广泛从事[[农业|农业]]生产，在[[小亚细亚半岛|小亚细亚半岛]]城市已经普遍建立，[[铜|铜]]器开始使用，车轮被发明出来，[[畜牧业|畜牧业]]在欧亚大陆逐渐普及，[[中国|中国]]进入农耕时代，全世界的人口可能已经达到5-7百万。

== 文化 ==

* [[温查文明|温查文明]] （[[前6千纪|前6千纪]]-[[前3千纪|前3千纪]]）
* [[欧洲|欧洲]]盘绳陶器文化（[[前6千纪|前6千纪]]-'''前5千纪'''）
* [[欧洲|欧洲]]梳纹陶器文化（[[前6千纪|前6千纪]]-[[前3千纪|前3千纪]]）
* [[中国|中国]]'''[[上宅文化|上宅文化]]'''开始（'''前5500年'''-'''前4500年'''）
* [[古埃及|古埃及]]的[[前王朝|前王朝]]时代（前4800年-前4000年） 
* 起源于中国南方沿海的[[南亚语系|南亚语系]]民族开始扩张，发展了航海技术、用渔钩和渔网捕鱼的技术和[[园艺|园艺]]。

== 发明与发现 ==

* '''前5000年''' — [[东南亚|东南亚]]普遍进行[[水稻|水稻]]种植，并逐渐传入[[恒河|恒河]]流域。
* '''前5000年''' — 农业文化从近东传播到[[欧洲|欧洲]]的[[大西洋|大西洋]]沿岸。
* '''前5000年''' — [[墨西哥|墨西哥]]开始种植[[玉米|玉米]]。<ref name="roberts">Roberts, J: "History of the World." Penguin, 1994.</ref>
* '''前5000年''' — [[象形文字|象形文字]]出现在世界几个文明起源地。
* '''前4500年''' — 耕作方法开始传入欧洲。
* '''前4000年''' — 在埃及发现人类最早制作的[[金属|金属]]器物。<ref name="roberts"/>
* [[中国|中国]]驯化[[水牛|水牛]]。
* 发明酿造[[啤酒|啤酒]]。
* [[印度|印度]]和[[美索不达米亚|美索不达米亚]]使用车轮。
* '''前4200年''' — [[北欧|北欧]]进入[[中石器時代|中石器時代]]。

== 环境变化 ==

* '''前5000年-前4900年''' — 是[[全新世|全新世]]中期佩伦海进时期，全球气候温暖。

== 历法 ==
* '''前4713年''' — [[1月1日|1月1日]]的中午12时为[[儒略日|儒略日]]的起点。
* '''前4121年''' — [[德国|德国]]历史学家爱德华·梅耶确定的[[埃及|埃及]]历法的起点。
* '''前4004年''' — [[北爱尔兰|北爱尔兰]]阿玛夫大主教詹姆士乌舍尔创立的日历中，确定当年[[10月23日|10月23日]]前夜为[[创世纪|创世纪]]起点。
* '''前4300年''' — [[天槍三|天槍三]]成为和[[地球|地球]]天顶[[北极|北极]]点最接近的[[星|星]]，直到[[前3942年|前3942年]]，才被[[右樞|右樞]]所取代。
