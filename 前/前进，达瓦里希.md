'''《前进，达瓦里希》'''是[[北京电影学院|北京电影学院]]动画学院2013届的学生[[王一琳|王一琳]]的毕业设计短片。该短片以[[苏联解体|苏联解体]]为主题，在互联网上发布后，引发了观众的强烈反响。

片名中的“达瓦里希”为[[俄语|俄语]]“[[革命同志|同志]]”一词的[[汉语|汉语]][[普通话|普通话]][[谐音|谐音]]。

== 内容简介 ==
影片背景为1980年代末、1990年代初，主角是一名小女孩，生活在一个落后的小村庄中，玩着老式积木，与家养动物们为伴，动物们的名字为[[前苏联|前苏联]]历史人物（如[[列宁|列宁]][[费利克斯·埃德蒙多维奇·捷尔任斯基|菲利克斯]][[贝利亚|贝利亚]]等），小女孩玩着社会主义建设的游戏，之后搬家到真正的高楼，旧玩具、宠物和课本被家长变卖，同时外部世界在发生巨大的变化，苏联解体、美国代替了苏联在他们日常生活中的位置，这一切对小女孩的心灵带来巨大的触动，她回到了已成瓦砾的“老房子”，想起“乌托邦”的梦想，守望着未来。整部影片充满了象征手法，例如以苏联人物命名的宠物相继死去，搬新家时小女孩儿说：“妈妈，我们的大楼会不会被美国炸掉了啊？”等。

==反响与评价==
该短片在网友之中得到了两极分化的评价。豆瓣网上该片有60%左右的观众给了4星及以上评分（满分5星），另有15%左右的观众给了1星（最低分）评分<ref>[http://movie.douban.com/subject/24756984/ 《前进，达瓦里希》在豆瓣网上的页面]</ref>。

赞扬者一般认为这部短片在对苏联的解体进行反思，并认为苏联的解体启发中国应该继续像现在一样走独立自主、稳定发展的道路，且大部分赞扬者对[[社会主义|社会主义]]的思想表示赞成，或是对目前[[中国共产党|中国共产党]]的执政成果持正面评价<ref name="达瓦里希从未走开">{{cite web |url = http://www.qstheory.cn/wh/sxdl/201306/t20130626_243366.htm |title = 丁栩翔：达瓦里希从未走开 |date = 2013-06-26 |access-date = 2013-07-01 |archive-url = https://web.archive.org/web/20130704222712/http://www.qstheory.cn/wh/sxdl/201306/t20130626_243366.htm |archive-date = 2013-07-04 |dead-url = yes }}</ref>。部分赞扬者针对批评者的批评，表示观赏《前进，达瓦里希》时要“去政治化审美”；批评者则多数为[[中华人民共和国民主运动|中国民主运动]]支持者，认为中国应尽快进行政治改革以结束一党专政、实现自由民主，进而抨击该短片矫情造作，美化[[红色恐怖|红色恐怖]]，对苏联执政时期的斑斑劣迹避而不谈<ref name="《前进，达瓦里希》被指矫情造作的政治卖萌秀">{{cite web |url = http://cul.sohu.com/20130701/n380310001.shtml |title=《前进，达瓦里希》被指矫情造作的政治卖萌秀 |date = 2013-07-01}}</ref><ref name="前进……但奔向何方，我的达瓦里希？">{{cite web | url = http://www.dfdaily.com/html/1170/2013/6/30/1023134.shtml | title = 前进……但奔向何方，我的达瓦里希？ | date = 2013-06-30 | deadurl = yes | archiveurl = https://archive.is/20130701131906/http://www.dfdaily.com/html/1170/2013/6/30/1023134.shtml | archivedate = 2013-07-01 }}</ref>，例如在短片里名为[[弗拉基米尔·伊里奇·乌里扬诺夫|弗拉基米尔]]的猫、以[[费利克斯·埃德蒙多维奇·捷尔任斯基|费利克斯]]为首的鸡、名叫[[拉夫连季·帕夫洛维奇·贝利亚|贝利亚]]的鸭子任人宰割，而现实中[[弗拉基米尔·伊里奇·乌里扬诺夫|弗拉基米尔]]（即[[列宁|列宁]]）是[[全俄肃清反革命及怠工非常委员会|全俄肃清反革命及怠工非常委员会]]（即[[契卡|契卡]]）的创始人，[[费利克斯·埃德蒙多维奇·捷尔任斯基|费利克斯·埃德蒙多维奇·捷尔任斯基]]是契卡的首任首脑、[[克格勃|克格勃]]的创始人，[[拉夫连季·帕夫洛维奇·贝利亚|贝利亚]]则是最有名的秘密警察头目（[[内务人民委员部|内务人民委员部]]首脑）<ref>{{cite news |url = http://money.163.com/13/0727/00/94OG1EEF00253B0H.html |title = 别强制前进，达瓦里希 |newspaper = 《[[中国经营报|中国经营报]]》 |date=2013-07-26 |accessdate=2014-03-16}}</ref>。这些都是导致[[苏联|苏联]]大量国民被迫害、杀害的重要人物。

该片被上传到[[YouTube|YouTube]]以及[[俄罗斯|俄罗斯]]本地的视频网站上后，亦引发了俄罗斯网友的热议，许多俄罗斯网友在评论中反思苏联解体的原因，为苏联的解体感到惋惜，并怀念现在已被本国政府所抛弃的社会主义思想；也有一些俄罗斯网友借此契机表达了希望本国能与中国增进友好关系的看法。该短片被上传到YouTube后不久，该影片俄语字幕版转为禁止评论<ref>{{cite web |url = http://www.guancha.cn/Literature/2013_06_25_153641.shtml |title = 俄网友评《前进，达瓦里希》：不知怎么，流泪了 |date=2013-06-25 |accessdate = 2013-10-13 |publisher = [[观察者网|观察者网]]}}</ref><ref>{{cite web |url = http://www.ltaaa.com/wtfy/9496.html |title = youtube网友讨论《达瓦里希》 |date = 2013-07-01 |accessdate = 2013-10-13 |publisher = [[龙腾网|龙腾网]] |archive-url = https://web.archive.org/web/20131024070223/http://www.ltaaa.com/wtfy/9496.html# |archive-date = 2013-10-24 |dead-url = yes }}</ref>。
