'''前马厂胡同'''，是[[北京市|北京市]][[西城区|西城区]]北部的胡同。

==简介==
前马厂胡同平面呈“厂”字形，东到[[旧鼓楼大街|旧鼓楼大街]]，西段向南折至[[鼓楼西大街|鼓楼西大街]]。全长370米，平均宽度为3米。[[明朝|明朝]]与[[后马厂胡同|后马厂胡同]]合称“马厂胡同”<ref name=dhly/>。据说此处原称“养马场胡同”，是明朝官马饲养场，后来才逐渐谐音称为“养马厂胡同”、“马厂胡同”<ref name=ma/>。[[清朝|清朝]]将马厂胡同析分为南北两条胡同，该胡同在南侧，故得名“前马家厂”，俗称“前马厂”。1965年北京市整顿地名，将果子观并入，定名为“前马厂胡同”<ref name=dhly>{{Cite web |url = http://www.bjdclib.com/subdb/laneculture/lane/200906/t20090630_3020.html|title = 前马厂胡同 |author =  |date = 2009-06-30 |publisher = 东华流韵 |language =  |accessdate =  |quote =  }}</ref>。“果子观”是原先从前马厂西部通到鼓楼西大街的岔路，一直到1980年代当地居民还将这一段称为“果子观”<ref name=ma>{{Cite web|url =http://bjrb.bjd.com.cn/html/2014-03/04/content_157347.htm|title =马与胡同不了情|author =朱小平|date =2014-03-04|publisher =北京日报|language =|accessdate =|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20170202020255/http://bjrb.bjd.com.cn/html/2014-03/04/content_157347.htm|archivedate =2017-02-02}}</ref>。前马厂胡同中部有另一条岔路，南北走向，向南通向[[西魏胡同|西魏胡同]]。

据说，清朝将马厂胡同一分为二始自[[雍正|雍正]]年间，此前这里应是八旗马圈。清朝中期，此地成了[[内务府|内务府]]官员“钟杨家”的宅院。钟杨家是内务府[[汉军镶黄旗|汉军镶黄旗]]人，汉姓杨。钟杨家到了[[鍾祥_(嘉慶進士)|钟祥]]（字云亭）考中[[进士|进士]]，累官至山东按察使、浙江布政使、[[山东巡抚|山东巡抚]]、[[闽浙总督|闽浙总督]]、[[库伦办事大臣|库伦办事大臣]]、[[河道总督|河道总督]]等职。史書記載，钟杨家“庐舍连云，几遍前后两街，四乡田地尤广，存终年取不尽之租”。这里的“前后两街”便指钟杨家的宅院占地前马厂、后马厂这两条胡同。钟杨家宅院今仍存原有格局，但建筑已面目全非，已成居民杂居的“大杂院”<ref name=ma/>。
