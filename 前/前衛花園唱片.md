'''前衛花園唱片'''，為[[臺灣|臺灣]]一獨立音樂廠牌，創立於2001年3月，以「辛勤的耳蘑菇養殖業者」自居
<ref>{{cite web | url = http://www.ag.com.tw/ | title = 前衛花園網路專門店 | publisher = | accessdate=2012-01-01}}</ref>
，專門代理[[另類搖滾|另類搖滾]]、[[Bossa_Nova|Bossa Nova]]等各類型多國唱片，代理如[[德國|德國]][[麥斯米蘭|麥斯米蘭]]、[[法國|法國]][[胡椒月亮|胡椒月亮]]樂團、[[西班牙|西班牙]]妮莎樂團、心虹熱樂團、[[日本|日本]]夢的雅朵、[[冰島|冰島]][[Múm|Múm]]等歌手在[[台灣|台]]發行。

2007年正式成為[[獨立音樂|獨立音樂]]廠牌，前[[台灣|台灣]][[樂團|樂團]][[自然捲|自然捲]]主唱[[魏如萱|魏如萱]]發行《[[La_Dolce_Vita_甜蜜生活|La Dolce Vita 甜蜜生活]]》專輯，而後又發行[[中華人民共和國|中國]]籍雙人組[[卡奇社|卡奇社]]的《日光傾城》專輯。

== 成立源起 ==
'''前衛花園唱片'''是一人唱片公司，從老闆、倉管、專輯進貨、包裝、影印側條的文字解說都是由一人（路先生）包辦。

路先生從小是位音樂迷。因公司倒閉而給他遣散費，他有感於音樂市場都走流行路線，於是拿出遣散費中的20萬元向國外廠商引進他在網路上找到的國外專輯，原先只是帶著大袋CD跟有購買意願的人約在咖啡店，由於太突兀，而後改在家裡賣，最後終於在[[誠品書店|誠品書店]]設立專櫃，打出口碑。[[林暐哲|林暐哲]]、[[陳珊妮|陳珊妮]]、[[范曉萱|范曉萱]]、[[萬芳|萬芳]]和[[黃韻玲|黃韻玲]]都是前衛花園的顧客。

2005年路先生到[[西班牙|西班牙]]，比手畫腳和使用[[英文|英文]]溝通，終於獲得[[西班牙|西班牙]]廠牌點頭同意授權代理，開始在[[台灣|台灣]]壓片生產，也使得其代理專輯終於在[[玫瑰唱片|玫瑰唱片]]、[[大眾唱片|大眾唱片]]等連鎖唱片行可以買到。

== 與國外獨立廠牌合作 ==
與[[西班牙|西班牙]]廠牌'''Slesta'''、'''Elefant'''以及[[南韓|南韓]]廠牌'''Pastel Music'''，不僅代理各廠牌旗下歌手[[專輯|專輯]]在[[台灣|台灣]]發行，亦曾與'''Pastel Music'''聯合主辦麥斯米蘭的亞洲巡迴演唱會。

== 與台灣公司合作 ==
'''前衛花園'''與[[誠品書店|誠品書店]]有密切的合作關係，在[[金石堂書店|金石堂書店]]亦有設置專櫃。
