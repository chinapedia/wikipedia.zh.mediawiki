== 大事记 ==
*亚洲
**[[趙國|趙]]、[[燕國|燕]]會於[[阿|阿]]。
**[[趙國|趙]]、[[齊國|齊]]、[[宋國|宋]]會於[[平陸|平陸]]，魏國面臨著被諸國圍攻的可能，於是魏國對韓、宋、魯、衛等小國施壓，迫使小國君主入朝仕魏。
**[[秦孝公|秦孝公]]六年，[[秦國|秦國]]左庶長[[公孫鞅|公孫鞅]]公佈新法令，史稱的'''[[商鞅變法|商鞅變法]]'''開始。
*欧洲
**[[7月21日|7月21日]]，[[世界七大奇迹|世界七大奇迹]]之一的[[阿耳忒弥斯神庙|阿耳忒弥斯神庙]]被一个名为[[黑若斯达特斯|黑若斯达特斯]]的年轻人纵火焚毁。

== 出生 ==
*[[9月20日|9月20日]]--[[亚历山大大帝|亚历山大大帝]]，[[马其顿|马其顿]]国王 ([[前323年|前323年]]逝世)。
*[[列昂納托|列昂納托]]（公元前356年－[[前322年|前322年]]），亞歷山大帝將領及繼業者之一。

== 逝世 ==
*[[黑若斯达特斯|黑若斯达特斯]]，希腊人，因纵火罪（纵火焚毁[[亚底米神庙|亚底米神庙]]）被处死。
