== 早年生活和教育 ==
科恩于1966年在[[斯坦福大学|斯坦福大学]]获得[[历史|历史]]学士学位，并于1972年在[[加州大學爾灣分校|加州大學爾灣分校]]获得[[社会科学|社会科学]]博士学位。<ref name="UMbio">{{Template:Cite web|url = http://www.fordschool.umich.edu/faculty/Michael_Cohen|title = Michael D. == 生涯 ==

=== 垃圾桶模型 ===
1972年，作为斯坦福大学的[[国家科学基金会|NSF]]-[[美国社会科学研究会议|SSRC]]博士后研究员，科恩在1972年与詹姆斯·马奇和[[卑尔根大学|卑尔根大学]]的客座教授约翰·奥尔森共事。他们一同发表了一篇名为《组织选择的垃圾桶模式》(''A Garbage Can Model of Organizational Choice'')的论文。<ref name="garbagecan">{{Template:Cite journal|last1 = Cohen|first1 = Michael D.|authorlink1 = Michael D. Cohen|first2 = James G.|last2 = March|authorlink2 = James G. March|first3 = Johan P.|last3 = Olsen|authorlink3 = Johan Olsen|title = A Garbage Can Model of Organizational Choice|journal = [[Administrative_Science_Quarterly|Administrative Science Quarterly]]|volume = 17|year = 1972|pages = 1–25|jstor = 2392088|doi = 10.2307/2392088|issue = 1}}</ref> 这篇论文阐述了一个使问题、解决方案和决策者相互独立的模型，并被频繁的引用。相比于传统的决策论而言，垃圾桶模型是一个全新的方法论。<ref>{{Template:Cite web|url = http://faculty.washington.edu/krumme/gloss/g.html|title = Glossary G|work = Economic Geography Glossary|publisher = [[Washington_University|Washington University]]|accessdate = 6 February 2011}}</ref>
