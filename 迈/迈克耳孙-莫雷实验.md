'''迈克耳孙－莫雷实验'''是为了验证“[[以太|以太]]”存在与否而做的一个实验，1887年由[[阿尔伯特·迈克耳孙|阿尔伯特·迈克耳孙]]与[[爱德华·莫雷|爱德华·莫雷]]合作在[[美国|美国]]的[[克利夫兰_(俄亥俄州)|克利夫兰]]进行。

== 实验原理 ==

[[File:Michelson-Morley_experiment_(zh).svg|400px]]
当时的物理理论认为，[[光|光]]的传播介质是“[[以太|以太]]”，由此产生一个问题：[[地球|地球]]以每秒30公里的[[速度|速度]]绕[[太阳|太阳]]运动，就必然迎面受到每秒30公里的“以太风”，从而必然对光的传播产生影响。这个问题出现以后，立即引起人们探讨“以太风”存在与否。迈克耳孙－莫雷实验就是在这个基础上进行的。

当“以太风”速度为0时，两束光应同时到达，因而相位相同；若“以太风”速度不为零，即装置相对以太运动，则两列光波相位不同。

假设装置在以太中向右以速度<math>v</math>运动，且从部分镀银的玻璃片到两面镜子的距离为<math>L</math>，那么向右的那一束光在向右的过程中相对装置速度为<math>c-v</math>，花费的时间<math>t_1=\frac{L}{c-v}</math>，返回时速度为<math>c+v</math>，时间<math>t_2=\frac{L}{c+v}</math>。所以总的时间是

<math>t_1+t_2=\frac{2Lc}{c^2-v^2}=\frac{2L}{c\cdot(1-\frac{v^2}{c^2})}</math>

而对于向上的那一束光，设它到达镜子所需的时间为<math>t_3</math>，在这段时间里镜子向右移动了<math>vt_3</math>，所以光走过的路程是一个直角三角形的斜边，于是有

<math>L^2=(ct_3)^2-(vt_3)^2=(c^2-v^2)t_3^2</math>

由此可得<math>t_3=\frac{L}{\sqrt{c^2-v^2}}</math>

而返回时间与此相同，所以总时间<math>2t_3=\frac{2L}{\sqrt{c^2-v^2}}=\frac{2L}{c\cdot\sqrt{1-\frac{v^2}{c^2}}}</math>

所以两束光的到达时间是不同的，根据这个实验应该能测量出地球通过以太的速度。

=== 空气介质的影响 ===

由于实验是在空气介质中进行，此时光的介质相对于实验装置的速度为0， 所以在两个路径上的来回的时间均为2l/c。 因此，在装置旋转90°后，没有干涉条纹的移动。<ref name="The model of wave propagation in classical physics">[http://www.piers.org/piersproceedings/download.php?file=cGllcnMyMDEyTW9zY293fDNQN2JfMTI4Ny5wZGZ8MTIwMzA3MTUyNzI5 The model of wave propagation in classical physics], Luo Siwei, Progress in Electromagnetics Research Symposium (Moscow) 2012 Aug.</ref>

== 实验结果 ==

莫雷不确信他自己的结论，继续与{{tsl|en|Dayton Miller|戴頓·米勒}}做更多的实验。米勒制作了更大的实验设备，最大的安装于[[威尔逊山天文台|威尔逊山天文台]]的臂长32米（有效长度）的仪器。为了避免实体墙可能造成的对以太风的阻挡，他使用了帆布为主体的流动墙。他每次旋转设备都会观测到不同的小偏移，不论是[[恒星日|恒星日]]还是年。他的测量值仅达到约10 km/s，而不是从地球轨道运动所期待的约30 km/s。他仍然不确信这是由于局部拖拽造成的，他没有尝试进行详细的解释。

肯尼迪后来在威尔逊山上作了实验，米勒发现1/10的漂移，并且不受季节影响。米勒的发现当时认为非常重要，并于1928年在一份会议报告上与迈克耳孙、[[亨德里克·洛伦兹|洛伦兹]]等人讨论。普遍认为需要更多的实验来检验米勒的结果。洛伦兹认可这个结论，造成漂移的原因不符合他的以太说或者爱因斯坦的[[狭义相对论|狭义相对论]]。爱因斯坦没有出席会议，但是感觉这个实验结果恐怕是实验误差。后来的实验没能重新获得米勒的结果，现代实验的精度推翻了此实验结论。

== 近代实验 ==

近代版的迈克耳孙－莫雷实验变得司空见惯。[[激光|激光]]和[[激微波|激微波]]通过让光线在充满高能原子的精心调整的空间内来回反射，以放大光线。这样的有效长度可达千米。还有一个好处，同一光源在不同光线角度产生同样的相位，给干涉计增加了额外精确度。

第一个这样的实验是由[[查尔斯·汤斯|查尔斯·汤斯]]做的，第一个[[激微波|激微波]]制作者之一。他们1958年的实验把漂移的上限，包括可能的实验误差，降低到仅仅30m/s。在1974年通过三角形内修剪工具精确的激光重复实验把这个值降低到0.025m/s，并且在一个光臂上放上玻璃来测试拖拽效果。在1979年布里耶-霍尔实验把人以方向的上限降低到30m/s，但是双向因素降低到0.000001 m/s（即静止或者夹带以太）。希尔斯和霍尔在经过一年的重复实验之后，于1990年公布，各向异性的极限降低到2×10<small><sup>-13</sup></small>。

实验结果证明，不论地球运动的方向同光的射向一致或相反，测出的[[光速|光速]]都相同，在地球同设想的“以太”之间没有相对运动。当时迈克耳孙因此认为这个结果表明以太是随着地球运动的。

== 对实验结果的解释 ==

[[喬治·斐茲杰惹|喬治·斐茲杰惹]]在1892年对迈克耳孙－莫雷实验提出了一种解释。他指出如果[[物质|物质]]是由带[[电荷|电荷]]的粒子组成，一根相对于以太静止的量杆的长度，将完全由量杆粒子间取得的[[静电平衡|静电平衡]]决定，而量杆相对于以太在运动时，量杆就会缩短，因为组成量杆的带电粒子将会产生[[磁场|磁场]]，从而改变这些粒子之间的间隔平衡。这一来，迈克耳孙－莫雷实验所使用的仪器，当它指向地球运动的方向时就会缩短，而缩短的程度正好抵消光速的减慢。

有些人曾经试行测量[[喬治·斐茲杰惹|喬治·斐茲杰惹]]的缩短值，但都没有成功。这类实验表明喬治·斐茲杰惹的缩短，在一个运动体系内是不能被处在这个运动体系内的观察者测量到的，所以他们无法判斷他们体系内的绝对速度，[[光学|光学]]的定律和各种[[电磁现象|电磁现象]]是不受绝对速度的影响的。再者，動系中的短縮，乃是所有物體皆短縮，而動系中的人，是無法測量到自己短縮值的。

1904年，荷兰物理学家[[亨德里克·洛伦兹|洛伦兹]]提出了著名的[[洛伦兹变换|洛伦兹变换]]，用于解释迈克耳孙-莫雷实验的结果。他提出运动物体的长度会收缩，并且收缩只发生运动方向上。如果物体静止时的长度为<math>L_0</math>，当它以速度<math>v</math>以平行于长度的方向运动时，长度收缩为

<math>L=L_0\cdot\sqrt{1-\frac{v^2}{c^2}}</math>

引入这条规律后<math>t_1+t_2=\frac{2L\cdot\sqrt{1-\frac{v^2}{c^2}}}{c\cdot(1-\frac{v^2}{c^2})}=\frac{2L}{c\cdot\sqrt{1-\frac{v^2}{c^2}}}=2t_3</math>，成功地解释了实验结果。

1905年，[[爱因斯坦|爱因斯坦]]在抛弃以太、以[[光速不变原理|光速不变原理]]和[[狭义相对性原理|狭义相对性原理]]为基本假设的基础上建立了[[狭义相对论|狭义相对论]]。狭义相对论认为空间和时间并不相互独立，而是一个统一的四维时空整体，并不存在绝对的空间和时间。在狭义相对论中，整个时空仍然是平直的、各向同性的和各点同性的。结合狭义相对性原理和上述时空的性质，也可以推导出洛伦兹变换。

[[里茨|里茨]]在1908年设想光速是依赖于[[光源|光源]]的速度的，企图以此解释迈克耳孙－莫雷实验。但是[[威廉·德西特|威廉·德西特]]于1931年在[[莱顿大学|莱顿大学]]指出，如果是这样的话，那麼一对相互环绕运动的星体将会出现表观上的异常运动，而这种现象并没有观察到。由此也证明了爱因斯坦提出的光速和不受光源速度和观察者的影响是正确的，而且既然没有一种静止的以太传播光波振动，[[艾萨克·牛顿|牛顿]]关于光速可以增加的看法就必须抛弃。

有人认为，爱因斯坦在提出[[狭义相对论|狭义相对论]]的过程中，曾经受到过迈克耳孙－莫雷实验结果的影响。{{tsl|en|John Stachel|约翰·施塔赫尔}}在《爱因斯坦和以太漂移实验》（''Einstein and aether drift experiments''）一文中指出，有间接的有力证据表明，爱因斯坦在1889年一定知道迈克耳孙－莫雷实验，并从1889～1901年间，持续感兴趣于设计光学实验，以检查地球穿行于以太的假定运动。

爱因斯坦在1922年，在《我是怎样创造了相对论》中说道：「那时我想用某种方法演示地球相对以太的运动……，在给自己提出这一问题时，我没有怀疑过以太的存在和地球的运动。于是，我预料如果把光源发出的光线用镜子反射，则当它的传播方向是平行或反平行于地球的运动方向时，应该具有不同的能量。所以我提出使用两个[[热电偶|热电偶]]，利用测量它们所生热量的差值，来证实这一点。」

== 对实验结果的再验证 ==

1893年[[歐里佛·洛茲|歐里佛·洛茲]]在[[伦敦|伦敦]]发现，光通过两块快速转动的巨大钢盘时，速度并不改变，表明钢盘并不把以太带着转。对[[恒星|恒星]][[光行差|光行差]]的观测也显示以太并不随着地球转动。

人们在不同地点、不同时间多次重复了迈克耳孙－莫雷实验，并且应用各种手段对实验结果进行验证，精度不断提高<ref>{{lang|en|Shankland, R.S., McCuskey, S.W., Laone, F.C., Kuerti,G., ''Review of Modern Physics'', '''27'''（1955）167.}}</ref>。除光学方法外，还有使用其他技术进行的类似实验<ref>郭硕鸿，''电动力学（第二版）''，高等教育出版社, 1997, 232页. ISBN 7-04-005550-3</ref>。如1958年利用[[微波|微波]]激射所做的实验得到地球相对以太的速度上限是3×10<sup>-2</sup> km/s<ref>{{lang|en|Cedarholm, J.P., Bland, G.F., Havens, B.L., Townes, C.H., ''Review of Modern Physics'', '''1'''（1958）342.}}</ref>，1970年利用[[穆斯堡尔效应|穆斯堡尔效应]]所做的实验得到此速度的上限只有5×10<sup>-5</sup> km/s<ref>{{lang|en|Isaak, G.R., ''Physics Bulletin'', '''255'''（1970）.}}</ref>。综合各种实验结果，人们基本可以判定地球不存在相对以太的运动。
