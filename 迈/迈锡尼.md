'''迈锡尼'''（古希腊语：'''{{polytonic|Μυκῆναι}}'''，現代{{lang-el|'''Μυκήνες'''}}）是位于[[希腊|希腊]][[伯罗奔尼撒半岛|伯罗奔尼撒半岛]]东北[[阿尔戈斯|阿尔戈斯]]平原上的一座[[爱琴文明|爱琴文明]]的城市遗址，位于[[科林斯|科林斯]]和[[阿尔戈斯|阿尔戈斯]]之间。它是[[荷马史诗|荷马史诗]]传说中[[亚该亚人|亚该亚]]人的都城，由[[珀耳修斯|珀耳修斯]]所建，在[[特洛伊战争|特洛伊战争]]时期由[[阿伽门农|阿伽门农]]所统治。

這座一度被認為只是傳說虛構的城市，是由[[德國|德國]]傳奇的[[考古學家|考古學家]][[海因里希·施里曼|海因里希·施里曼]]在十九世紀時重新挖掘出來才得以重見天日。实际上，在公元前的第二个千年中它确实是希腊文明中最重要的城市，统治着爱琴海南部广大的地区，所代表的文明称为[[迈锡尼文明|迈锡尼文明]]。

在[[古希腊|古典希腊时期]]之前，这个城市已经被（可能是）[[多利安人|多利安人]]入侵所破坏，其宫殿于公元前12世纪被废弃，前468年阿尔戈斯军队占领了此地，将居民尽数驱逐，迈锡尼城从此被消灭。

古典时期，旅行家[[保萨尼亚斯|保萨尼亚斯]]曾经访问过这个遗址，并记录了{{link-en|狮子门 (迈锡尼)|Lion Gate|狮子门}}等今天还屹立在原处的纪念建筑。其他著名的遗迹还有[[阿特柔斯|阿特柔斯]]宝库、[[阿伽门农|阿伽门农]]之墓等。今天它是希腊的著名旅游景点，也是联合国教科文组织评选的[[世界遗产|世界遗产]]。

== 詞源 ==
[[File:The_acropolis_at_Mycenae,_Dec._2001.jpg|left]]
雖然邁錫尼城是由希臘人所構築，但是其名稱卻非希臘人所賦予，而是{{link-en|Pre-Greek substrate|Pre-Greek substrate|前希臘基礎期}}留下來的許多名詞之一，並且由後來移居進來的希臘人所沿用。<ref>{{cite book |last = Chadwick |first = John |authorlink = :en:John Chadwick |title = ''The Mycenaean World'' |publisher = Cambridge University Press |year =1976 |page =1 |url = |isbn = 0-521-21077-1 hardback }} [http://books.google.com/books?id=RMj7M_tGaNMC&lpg=PP1&pg=PA88#v=onepage&q&f=false]</ref>

此名稱是來自於傳說，有關他的希臘單字為：“μύκης”（''mycēs''）、‘mushroom’。因此，[[保薩尼亞斯|保薩尼亞斯]]認為這個名稱應該是源自傳說中的英雄[[珀耳修斯|珀耳修斯]]所命名的。
