'''米高·安東尼·艾普頓'''（{{lang-en|'''Michael Antony Appleton'''}}，{{bd|1975年|12月4日|catidx=Appleton, Michael}}）出生於[[英格蘭|英格蘭]][[大曼徹斯特|大曼徹斯特]][[索尔福德|索尔福德]]（Salford），是一名前[[足球|足球]][[運動員|運動員]]，出身[[曼聯預備隊及青訓學院|曼聯青訓系統]]，司職[[中場|中場]]，未有為母會一隊在聯賽上陣，曾以借用身份短暫效力[[林肯城足球俱乐部|林肯城]]和[[格林斯比镇足球俱乐部|甘士比]]，其後轉投[[普雷斯顿足球俱乐部|普雷斯頓]]，2003年在[[西布罗姆维奇足球俱乐部|西布朗]]因嚴重[[膝蓋|膝]]傷提早退役後，留隊擔任[[教练|教练]]職務，於2011年11月獲聘任教[[英冠|英冠]]球會[[朴茨茅斯足球俱乐部|樸茨茅夫]]，是他首份全職領隊工作。其後曾先後短暫執教[[英冠|英冠]]球會[[黑池足球會|黑池]]和[[布莱克本流浪者足球俱乐部|布莱克本流浪者]]。

2014年夏季獲聘任教[[英乙|英乙]]球會[[牛津联足球俱乐部|牛津聯]]<ref name="Michael Appleton">{{citenews |language=en |url=http://www.bbc.com/sport/0/football/28163679 |title=Oxford United: Michael Appleton named head coach in shake-up |work=''[[BBC_Sport|BBC Sport]]'' |date=2014-07-04 |accessdate=2014-08-13 }}</ref>。[[2015年至2016年英格蘭足球乙級聯賽|2015/16球季]]，帶領牛津聯升上英甲。2017年6月，改任[[李斯特城足球會|李斯特城]]助教。10月，李斯特城解僱[[奇治·莎士比亞|莎士比亞]]，由艾普頓暫代領隊。

===教練及領隊===
;西布朗
艾普頓以27歲之齡因[[膝蓋|右膝]]嚴重傷患而提前退役後，留在[[西布罗姆维奇足球俱乐部|西布朗]]加入教練團隊，先後擔任足球學院發展教練、U18青年隊教練、預備隊教練。和一隊教練。2011年2月6日他於領隊[[罗伯托·迪马特奥|迪馬堤奧]]留薪停職期間暫代處理一隊職務<ref>{{cite web |language=en |url=http://www.wba.co.uk/page/News/0,,10366~2286720,00.html |title=Di Matteo relieved of duties |publisher=[[西布罗姆维奇足球俱乐部|''WBA.co.uk'']] |date=6 February 2011 |accessdate=6 February 2011 |deadurl=yes |archiveurl=https://web.archive.org/web/20110209072430/http://www.wba.co.uk/page/News/0,,10366~2286720,00.html |archivedate=2011年2月9日 }}</ref>。

;樸茨茅夫
2011年11月10日[[英冠|英冠]]球會[[朴茨茅斯足球俱乐部|樸茨茅夫]]宣佈聘任艾普頓為新領隊，雙方簽署三年半合約<ref name="Michael Appleton2">{{cite news |language=en |url=http://news.bbc.co.uk/sport2/hi/football/15616925.stm |title=Portsmouth name Michael Appleton as new manager |publisher=''[[BBC_Sport|BBC Sport]]'' |date=2011-11-10 |accessdate=2011-11-11}}</ref>。

;黑池
2012年11月7日，[[英冠|英冠]]球會[[黑池|黑池]]宣佈艾普頓為新領隊，合約為期一年<ref>{{cite web |language=en |url=http://www.bbc.co.uk/sport/0/football/20236445 |title=Michael Appleton: Blackpool name Portsmouth manager as new boss |publisher=''[[BBC_Sport|BBC Sport]]'' |date=7 November 2012|accessdate=12 January 2013}}</ref>。

;布莱克本流浪者
2013年1月11日，[[英冠|英冠]]球會[[布莱克本流浪者足球俱乐部|布莱克本流浪者]]宣佈艾普頓為新領隊，合約為期二年半<ref>{{cite web |language=en |url=http://www.bbc.co.uk/sport/0/football/20972061 |title=Michael Appleton: Blackburn Rovers announce new manager |publisher=''[[BBC_Sport|BBC Sport]]'' |date=11 January 2013|accessdate=12 January 2013}}</ref>。成為[[布莱克本流浪者足球俱乐部|布莱克本流浪者]]於賽季內第三位領隊。

2013年3月19日，上任僅67日執教15場的他，因成績未如理想而被撤職。
