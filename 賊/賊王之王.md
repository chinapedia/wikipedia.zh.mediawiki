《'''賊王之王'''》（{{lang-en|''Lepke''}}）是1975年的[[黑幫電影|黑幫電影]]，導演是{{link-en|梅納漢姆·哥蘭|Menahem Golan}}，主演是[[東尼·寇蒂斯|東尼·寇蒂斯]]，飾演猶太裔美國黑幫{{link-en|Louis Buchalter|Louis Buchalter|Louis "Lepke" Buchalter}}。<ref>{{Cite web |url = http://www.tcm.com/tcmdb/title/81206/Lepke/ |title = LEPKE: OVERVIEW |publisher = Turner Classic Movies |language = en}}</ref>本片經常被影迷視為東尼·寇蒂斯被低估的電影和他最出色的演出之一。
