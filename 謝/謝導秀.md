'''謝導秀'''（{{bd|1940年|2月25日|2019年|2月20日}}），[[廣東|廣東]][[梅縣|梅縣]]人，中國[[嶺南琴派|嶺南派]][[古琴|古琴]]演奏家，現任廣東古琴研究會會長，[[中華人民共和國文化部|中華人民共和國文化部]]認定的“国家级非物质文化遗产项目（古琴艺术）代表性传承人”之一。<ref>{{Cite web|title=古琴家谢导秀仙逝，弟子追忆“他是岭南古琴史上里程碑式的人物”|url=http://www.gzwl.org.cn/doc/2019/02/25/17850.shtml|work=www.gzwl.org.cn|accessdate=2019-03-04}}</ref>

== 生平 ==
謝導秀於民國二十九年（1940年）2月25日出生於廣東梅縣周溪村，小時候受到[[客家文化|客家文化]]的影響，對客家山歌、[[鬧八音|鬧八音]]很感興趣。1953年8月2日隨父母遷居[[廣州|廣州]]，在[[廣州第十七中學|廣州第十七中學]]上學。在中學時便開始學習二胡、揚琴、笛子等民族樂器，參加了學校的民族樂隊。<ref name="古曲">{{Cite web |url= http://info.guqu.net/guqindashi/20060729225752_558.html|title=岭南古琴艺术大师谢导秀|accessdate=2014-07-09 |work= |publisher=中國古曲網|date=2006-07-29}}</ref>

1960年進入廣州音樂專科學校（今[[星海音樂學院|星海音樂學院]]），最初想學二胡，但被分配到當時頗為冷門的古琴專業，便師從當年才在廣州音樂專科學校成為兼職教師的[[楊新倫|楊新倫]]學習古琴<ref name="中音在線">{{Cite web |url=http://www.musiceol.com/news/html/2008-7/200872816332111773964.html|title=謝導秀的古琴守望之旅|accessdate=2014-07-09 |work= |publisher=中音在線|date=2008-07-28}}</ref>。同年10月在[[廣州電視台|廣州電視台]]與黃金城琴簫合奏《[[關山月_(古琴曲)|關山月]]》，此後也曾多次參加[[中國音樂家協會|中國音樂家協會]]廣東分會舉辦的演奏會。<ref name="古曲"/>

1963年畢業，從事古琴相關工作。1975年與老師楊新倫一起重新編纂《[[古岡遺譜|古岡遺譜]]》。1980年10月與楊新倫、[[莫尚德|莫尚德]]一起成立廣東古琴研究會，出任廣東古琴研究會首任秘書長。1983年5月參加全國第三次打譜大會。1990年楊新倫逝世，謝導秀繼任[[廣東古琴研究會|廣東古琴研究會]]會長<ref name="古曲"/>。除去兼職的星海音樂學院外，也在[[中山大學|中山大學]]哲學系、[[廣州城市職業學院|廣州城市職業學院]]教授古琴。<ref name="中音在線"/>
