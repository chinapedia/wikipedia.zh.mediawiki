'''謝萬'''（{{bd|320年||361年|catIdx=X}}），[[表字|字]]'''萬石'''，[[陳郡|陳郡]][[陽夏|陽夏]]（今[[河南|河南]][[太康|太康]]）人。[[晉朝|晉朝]]名士，[[謝裒|謝裒]]之子，[[謝奕|謝奕]]及[[謝安|謝安]]弟。曾接替謝奕為[[豫州|豫州]][[刺史|刺史]]，並領導[[北伐|北伐]]，但終失敗而還，被廢為[[庶人|庶人]]。

== 生平 ==
謝萬才能和器量皆優異出眾，雖然器量仍不及謝安，但因擅長展現自我，故此早有聲譽。成年後，撫軍將軍[[司馬昱|司馬昱]]輔政，因聽聞謝萬的名氣，故此召他為其[[從事中郎|從事中郎]]。後遷[[吳興|吳興]][[太守|太守]]。

昇平二年（358年），任安西將軍、豫州刺史的哥哥謝奕去世，謝萬任西中郎將，[[持節|持節]]監[[司州|司]]、[[豫州|豫]]、[[冀州|冀]]、[[并州|并]]四州諸軍事，兼任豫州刺史。然而，當時[[王羲之|王羲之]]就認為以謝萬出鎮豫州，領導軍隊北伐的決定是違才易務，曾寫信勸說桓溫不要以謝萬外鎮，但桓溫沒有接納王羲之的建議<ref>《晉書·謝萬傳》：「[[王羲之|王羲之]]與[[桓溫|桓溫]]箋曰：『謝萬才流經通，處廊廟，參諷議，固是後來一器。而今屈其邁往之氣，以俯順荒餘，近是違才易務矣。』溫不從。」</ref>。王羲之當時亦寫信給謝萬，勸他要與士卒們同甘共苦<ref>《[[資治通鑑|資治通鑑]]》卷一百：「（王羲之）又遺萬書曰：『以君邁往不屑之韻，而俯同群碎，誠難為意也。然所謂通識，正當隨事行藏耳。願君每與士卒之下者同甘苦，則盡善矣。』萬不能用」。</ref>，但謝萬不能這樣作，雖然肩負北伐大任，然而仍表現得高傲豪縱，毫不安撫士眾。謝安見此十分憂心，不但親自慰問和勉勵謝萬的部下，更要謝萬與手下將帥們多見面、聚餐談話，不要再表現得那麼高傲。然而，當謝萬與眾將聚餐時，卻無話可說，只拿著[[如意|如意]]指著眾將說：「諸將皆勁卒。」因為兵、卒是最低下的[[軍階|軍階]]，既為將帥，被以兵卒稱呼是一種羞辱，故眾將聽後對謝萬更為忿恨。

升平三年（359年），謝萬與北中郎將[[郗曇|郗曇]]兵分兩路，北伐[[前燕|前燕]]。謝萬先遣征虜將軍[[劉建|劉建]]修治[[馬頭|馬頭]]新城，自己就親率部眾打算支援[[洛陽|洛陽]]。然而及後郗曇因病而退屯[[彭城|彭城]]，謝萬卻以為對方是因前燕兵強而退，於是倉卒退兵，士眾於是自行潰敗，謝萬亦隻身狼狽逃還，其間部下卻打算趁謝萬兵敗而殺了他，只因念及謝安，才沒有實行。然而謝萬這次潰敗卻令[[許昌|許昌]]、[[潁川|潁川]]、[[譙郡|譙郡]]、[[沛郡|沛郡]]等豫州各郡落入前燕之手，故謝萬回時就被废為庶人。

升平五年（361年），朝廷重召謝萬為[[散騎常侍|散騎常侍]]<ref>《[[北堂书钞|北堂书钞]]》卷五八引《[[晋起居注|晋起居注]]》有升平五年（361年）起用谢万为散骑常侍诏。</ref>，適逢謝萬逝世，享年四十二歲<ref>此據《晉書·謝萬傳》，《宋書·謝莊傳》載其玄孫[[謝莊|謝莊]]書，言其卒年為四十歲：「家世無年，亡高祖四十。」</ref>，朝廷亦以散騎常侍作為其贈官。

== 性格特徵　==
*謝萬貪睡，謝安手叩屏風呼他起床<ref>《[[太平御覽|太平御覽]]》卷七○一引《俗說》</ref>。
*謝萬長於議論亦能寫文章，曾經以[[漁父|漁父]]、[[屈原|屈原]]、[[司馬季主|司馬季主]]、[[賈誼|賈誼]]、[[楚老|楚老]]、[[龔勝|龔勝]]、[[孫登_(隱士)|孫登]]及[[嵇康|嵇康]]八個隱士和顯士寫了《八賢論》，又以此文給[[孫綽|孫綽]]看並與其辯論。
*謝萬曾與[[蔡系|蔡系]]等人一同到[[征虜亭|征虜亭]]送別[[支遁|支遁]]，期間謝萬乘蔡系離座而坐了蔡系的位置，更接近支遁。蔡系回來後竟連著坐墊推了謝萬下地，自己坐回那個位置。謝萬倒地弄得帽和頭巾都歪了，但就慢慢站起來，揚一揚衣服就找位置坐下，神情都很平和，並沒有憤怒或沮喪的樣子。坐好後，謝萬就說：「你是奇人呀，差點就弄傷我的臉。」蔡系答:「我從來就沒考慮過你的臉。」二人對此都沒有介意，於是受當時人亦稱許。<ref>《世說新語·雅量篇》</ref>

== 評論 ==
*殷浩：「文理轉遒，成殊不易」<ref>《世說新語·賞譽篇》</ref>
*桓溫：「萬石撓弱凡才，有何嚴顏難犯！」<ref>《世說新語·方正篇》</ref>
*王羲之：「在林澤中，為自遒上。」<ref>《世說新語·雅量篇》</ref>
*謝玄：「中郎衿抱未虛，復哪得獨有。」<ref>《世說新語·輕詆篇》</ref> 指斥謝萬心胸狹窄，不夠謙虚。
