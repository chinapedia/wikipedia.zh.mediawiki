'''謝龍介'''（{{bd|1961年|10月3日|catIdx=Xie謝}}），[[中華民國|中華民國]]政治人物，[[臺南市|臺南市]]人，現任[[中國國民黨籍|中國國民黨籍]][[臺南市議員|臺南市議員]]，曾任[[臺南市_(省轄市)|省轄市]]時期的[[臺南市議員|臺南市議員]]<ref>[http://www.tncc.gov.tw/tnccp2/ccp_01a.asp?CName=%E8%AC%9D%E9%BE%8D%E4%BB%8B 台南市議會資訊網–謝龍介]</ref> 。

== 臺南市議員 ==
* 謝龍介酷愛講台語，經常在議會質詢<ref>[https://www.youtube.com/watch?v=L2PBQav4uOA 互嗆11分鐘！賴清德、謝龍介為「幾趴」大翻臉｜三立新聞網SETN.com]</ref>及政論節目<ref>[https://www.youtube.com/watch?v=gW__fv2a-vI 《新聞深喉嚨》精彩片段　請辭都是打嘴砲？謝龍介：陳菊、賴清德是在演哪齣？]</ref>全程臺灣話，但卻被有些[[媒體|媒體]]謠傳是「苦練[[臺灣話|臺灣話]]有成的[[臺灣外省人|外省人]]」，他說這是誤傳，他是不折不扣的[[閩南裔臺灣人|本省閩南人]]，當初是因為一名外省籍的[[民意代表|民意代表]]，主持電台節目時稱讚他「外省[[眷村|眷村]]子弟卻能講得一口臺語」，結果[[原臺南市|原臺南市]]前市長[[許添財|許添財]]得知後，在政壇廣傳，連前[[政務委員|政務委員]][[楊秋興|楊秋興]]都以為他是外省人<ref>[http://www.appledaily.com.tw/appledaily/article/headline/20160610/37262600/ 誤會大了！謝龍介自揭身世之謎　「哇係ＸＸ郎」].蘋果日報.2016-06-10</ref>。
* 台南市議員[[唐碧娥|唐碧娥]]在媒體上表示，謝龍介曾私下說，[[2004年中華民國總統選舉|2004年中華民國總統選舉]]中，[[陳水扁|陳水扁]]總統如果當選，他會[[切腹|切腹]]自殺<ref>白喬茵、姜毅宏.[http://www.setn.com/News.aspx?NewsID=87334 煞氣a謝龍介！洪辦新發言人爭議多　曾揚言阿扁當選要切腹].三立新聞網.2015-07-29</ref>。
* 2006年高雄市長選舉中，謝龍介在中國國民黨候選人[[黃俊英_(台灣)|黃俊英]]競選總部舉辦成立大會上脫口說出「冬瓜菊」形容民進黨候選人[[陳菊|陳菊]]（消遣陳菊身材），遭外界視為失言，也使民進黨陣營以此反擊黃俊英。最後雖然黃俊英本人親赴陳菊總部送花「致意」<ref>{{Cite news|url = http://www.appledaily.com.tw/appledaily/article/headline/20061031/2995652/|title = 黃為冬瓜菊風波「致意」晚會主持人失言 招來綠營婦女踢館|last =涂建豐 |first = |date = 2006-10-31|work = 蘋果日報|accessdate = 2016-04-21}}</ref>，但已對黃俊英選情造成殺傷，被外界分析為黃俊英落選原因之一。
* 2013年12月19日中國國民黨台南市議員謝龍介的兒子，19日凌晨駕駛改裝自小客車，遇到警方攔檢，沒停車受檢，還加速逃逸，最後警方在車內搜出2公克[[K他命|K他命]]，當場坦承[[吸毒|吸毒]]<ref>{{Cite news|url = https://tw.news.yahoo.com/%E8%AC%9D%E9%BE%8D%E4%BB%8B%E4%B9%8B%E5%AD%90%E6%8C%81k-%E9%A3%9B%E8%BB%8A%E9%81%BF%E6%94%94%E6%AA%A2-030201917.html|title = 謝龍介之子持K 飛車避攔檢|last =顏孜育、翁英修 |first = |date = 2013-12-28|work = 民視|accessdate = 2016-01-20}}</ref>。
* 2014年[[太陽花學運|太陽花學運]]期間，民進黨議員參選人[[蔡麗青|蔡麗青]]曾經到中國國民黨台南黨部抗議[[服貿|服貿]]協議，謝龍介手持[[麥克風|麥克風]]大聲調侃蔡，說「賴清德全力支持陸美祈，放棄了蔡麗青」，蔡麗青非常生氣。蔡的支持者意圖去搶麥克風，謝龍介則怒喝道：「你敢碰到我試試看，你不曾被鬼嚇過麼？」最後謝龍介以諷刺地口吻調侃說：「我是蔡麗青父親的朋友，要為蔡麗青加油，[[民調|民調]]唯一支持蔡麗青。」蔡麗青事後宣稱打算反擊，但其宣傳車的廣播器故障，喊不出聲音，故被謝龍介奚落一番而離去。<ref>[http://www.ettoday.net/news/20140326/339332.htm 蔡麗青反服貿被拋棄？　藍營謝龍介喊加油「酸度破表」].ETtoday.2014-03-26</ref>
* 2014年5月，臺南市長賴清德曾力挺[[台糖售地案|台糖售地案]]中前[[民進黨秘書長|民進黨秘書長]][[吳乃仁|吳乃仁]]和前[[立法委員|立法委員]][[洪奇昌|洪奇昌]]是清白的，謝龍介當時向賴清德打賭，「若兩人都有罪，賴辭職；若兩人都無罪，我自己辭」。而後謝表示「如今判決已出爐，吳乃仁有罪，洪奇昌無罪，如此算兩人皆輸」為由辭職，希望賴清德也辭去市長一職，賴清德反嗆「你要辭就自己辭嘛！」<ref>李恩慈.[http://www.appledaily.com.tw/realtimenews/article/new/20140522/402541/ 嗆賭賴清德　謝龍介今辭議員立即生效].蘋果日報.2014-05-22</ref>
* 2015年6月[[2015年八仙樂園派對粉塵爆炸事故|八仙樂園爆炸案]]造成近500人遭燒燙傷，謝龍介於2015年6月30日捐助一個月薪資，響應救援行動，期盼能為後續的醫療與照護提供協助，希望各界共同陪伴意外受傷的年輕人及家屬度過這個難關<ref>{{cite news|url=http://news.cts.com.tw/nownews/society/201506/201506301630988.html|last=莊漢昌|title=謝龍介捐一個月所得　助八仙傷者後續復健|publisher=華視新聞網|date=2015-06-30|deadurl=yes|archiveurl=https://web.archive.org/web/20150715154604/http://news.cts.com.tw/nownews/society/201506/201506301630988.html#.VaYiwKSqqko|archivedate=2015年7月15日|df=}}</ref>。
* 2015年謝龍介質詢時痛批[[臺南市長|臺南市長]][[賴清德|賴清德]]支持[[臺灣獨立|臺灣獨立]]，在質詢時謝龍介以激烈口氣抨擊「你已經在做行政首長，結果言行還和立法委員一樣，這就是賴清德啦！」。遭賴清德回應：「你這款質詢，不會增加你立委的選票啦！」。這段質詢影片流出後，引發網友製作各種KUSO影片嘲諷消遣。<ref>[https://www.youtube.com/watch?v=nsECPxPFCn8 謝龍介質詢賴清德 神打臉變仁醫KUSO版].台灣蘋果日報頻道.2015-10-07</ref><ref>[http://www.peoplenews.tw/news/6f3ebc18-8174-4965-bf07-62a75e6792cf 賴清德vs.謝龍介，布袋戲Kuso版網友笑翻].民報.2015-10-06</ref>
* 2016年12月5日，謝龍介質詢[[臺南市長|臺南市長]][[賴清德|賴清德]]時，展現其[[布袋戲|布袋戲]]天分<ref>[http://www.setn.com/News.aspx?NewsID=204489 布袋戲偶畸形核食害的？謝龍介出奇招　賴清德憋笑打臉].三立新聞網.2016-12-05</ref>
* 2017年9月5日，謝龍介在[[賴清德|賴清德]]確定接任[[行政院長|行政院長]]後，對賴清德表示自己無論天涯海角都會追過去，並且「一生只監督你一人」<ref>李世泓、陳嘉筠.[http://www.setn.com/News.aspx?NewsID=291422 賴清德北上當閣揆　謝龍介堅定告白：天涯海角，絕對追到你].三立新聞網.2017-09-05</ref>。此番言論被部分網友視為謝龍介對賴清德的告白，並將兩人的關係製作成[[BL_(和製英語)|BL作品]]，[[維基百科|維基百科]]謝龍介條目的配偶欄也被短暫地改成賴清德<ref>{{cite news|title=是真愛無誤❤️ 謝龍介維基「配偶」被改成賴清德 |url=http://news.ltn.com.tw/news/life/breakingnews/2184408 |author= |publisher=自由時報 |date=2017-09-05 |accessdate=2017-09-05 }}</ref><ref>{{cite news|title=真愛無誤！謝龍介要「一生監督」賴清德 網笑瘋：台男腐城 |url=http://news.ebc.net.tw/news.php?nid=77343|author= |publisher=東森新聞 |date=2017-09-06 |accessdate=2017-09-06 }}</ref>。而实际上谢的配偶名叫李佳芬，碰巧和国民党籍前立委[[韩国瑜|韩国瑜]]妻子同名；韩在谢的造势大会上就曾拿这点开涮<ref>{{cite web |author1=[[东森新闻|东森新闻 CH51]] |title=韓流魅力！韓國瑜替謝龍介站台 造勢晚會如歌廳秀 |url=https://www.youtube.com/watch?v=54Ho8hVcfB0 |website=[[YouTube|YouTube]] |accessdate=2018-10-31}}</ref>。
* 2018年9月9日，八月份[[台南市|台南市]][[慰安婦人權平等促進會|慰安婦人權平等促進會]]在[[中國國民黨|中國國民黨]]台南市黨部旁樹立[[慰安婦少女銅像|慰安婦少女銅像]]。日本政府表達抗議，且「慰安婦之真相國民運動組織」，包括守護英靈名譽彰顯會、向世界的史實發布會、明辨捏造慰安婦問題的日本有誌會、為了解決捏造日本軍「慰安婦」問題的北海道會等16個團體，於9月6日派代表[[藤井實彥|藤井實彥]]偕同翻譯前往台南市中國國民黨部表達意見。事後民眾向謝龍介表達該代表腳踹銅像；謝龍介經調閱監視器，確有此事，在Facebook發文表示：「台灣人的肉給人家吃，但骨頭不給人家啃」「藤井你以為台灣還是日本的殖民地嗎」，謝龍介於9月10日將前往[[日本台灣交流協會|日本台灣交流協會]]台北事務所表達譴責並要求道歉<ref>{{cite news|title=太囂張了，侵門踏戶!! 日本人腳踢台南慰安婦銅像 謝龍介：不道歉不能出境|url=http://www.mission-tw.com/mission-tw/index.php/mission/article/21420|author=修瑞瑩|publisher =[[密訊|密訊]] |archivedate=2018年9月9日}}</ref><ref>{{cite news|title=日人腳踢台南慰安婦銅像 謝龍介：不道歉不能出境|url=https://udn.com/news/story/6656/3358195|author=修瑞瑩|publisher =[[聯合新聞網|聯合新聞網]] |archivedate=2018年9月9日}}</ref>。
