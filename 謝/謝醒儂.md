'''謝醒儂'''（Tse Sing Nong），活躍於20世紀20年代至40年代末期的一線[[粵劇|粵劇]]乾班男花旦；[[廣州|廣州]]、[[上海|上海]]、[[香港|香港]][[電影|電影]][[演員|演員]]，其女妝造形扮相比真的[[女人|女人]]更嬌魅上鏡；落妝後亦是一位青靚白淨的美男子。他曾在唐天寶班（[[白玉堂|白玉堂]]）、覺先聲班（[[薛覺先|薛覺先]]）、日月星班（[[桂名揚|桂名揚]]、[[曾三多|曾三多]]）等一線粵劇團擔任正印男花旦。首本戲有《[[冰山火線|冰山火線]]》、《[[璇宮艷史|璇宮艷史]]》、《[[天姬送子|天姬送子]]》、《皇姑嫁何人》、《荒唐女俠》等多齣。

1930年，謝醒儂擔任正印花旦，參加「唐天寶班」與[[新珠|新珠]]（[[文武生|文武生]]）、[[靚元亨|靚元亨]]（[[小生|小生]]）、[[薛覺先|薛覺先]]（[[小生|小生]]）、蝴蝶仔、譚猩猩、吳鏡斌、[[嫦娥英|嫦娥英]]、[[新丁香耀|新丁香耀]]演出《玉梅花》、《重建姑蘇台》（編劇：[[黃不廢|黃不廢]]）<ref>香港利源書報社，古岡《廣東戲劇史：紅伶篇》（第二卷），第159頁</ref>。

1931年[[夏季|夏季]]，謝醒儂居住在[[廣州市|廣州市]][[河南|河南]][[福仁里|福仁里]]卅六號。1931年7月18日（星期六），投宿在[[廣州|廣州]][[東亞酒店|東亞酒店]]349號房的16歲[[南洋|南洋]][[華僑|華僑]][[少女|少女]]戲迷余洛美為戀謝醒儂服毒[[自殺|自殺]]。她被送[[醫院|醫院]][[急救|急救]]醒後，就立刻寫信給謝醒儂要求見他<ref>1931年7月21日（星期二），香港《[[工商日報|工商日報]]》，第二張；第二版。</ref>。

謝醒儂在1932年與[[天津|天津]][[導演|導演]]姜白谷、[[廣州|廣州]][[越秀山|越秀山]][[永漢南路|永漢南路]]的[[南關戲院|南關戲院]]之[[經理|經理]]薛兆榮、[[粵劇|粵劇]]名宿[[白玉堂|白玉堂]]、撰曲家[[麥嘯霞|麥嘯霞]]等人，在[[廣州|廣州]][[西關|西關]]開設『華藝影片公司』，謝醒儂為 [[經理|經理]]，拍攝[[默片|默片]]《裂痕》、《前進》及《花落見花心》等[[默片|默片]]，後來週周轉不靈，無法清還債務而例倒閉。

[[香港|香港]]東莞商會在[[農曆|農曆]]壬申年六月初十日（1932年7月13日）起，一連三日三晚，在[[九龍|九龍]][[油麻地|油麻地]][[彌敦道|彌敦道]]380號[[普慶戲院|普慶戲院]]（Astor Theatre），為貧民義學[[籌款|籌款]]義演[[粵劇|粵劇]]，省港一線紅伶雲集。謝醒儂在1932年7月15日（星期五）日戲演《淺水蛟龍》，與[[白玉堂|白玉堂]]及[[林坤山|林坤山]]合演，票價[[港幣|港幣]]一毫半至三元<ref>1932年7月15日（星期五），香港《[[工商日報|工商日報]]》，第三張；第二頁。</ref>。

[[粵劇|粵劇]]伶人常到[[香港島|香港島]][[中環|中環]]的「黃龍跳舞學院」找[[舞女|舞女]]伴舞消遣，謝醒儂卻鮮有涉足歡場<ref>1934年10月6日（星期六），香港《[[工商日報|工商日報]]》，第四張第一版。</ref>。

[[太平洋戰爭|太平洋戰爭]]期間，謝醒儂與[[羅文煥|羅文煥]]、[[梁少南|梁少南]]、[[黃鶴聲|黃鶴聲]]、[[歐陽儉|歐陽儉]]等[[粵劇|粵劇]]紅伶在[[越南|越南]]演出，並兩度轉赴[[泰國|泰國]]演出，除演出傳統[[粵劇|粵劇]]劇碼外，還演情節離奇曲折的《荒唐女俠》等新編劇，謝醒儂由於長期與[[薛覺先|薛覺先]]拍演，技藝精湛，深受[[泰國|泰國]][[觀眾|觀眾]]歡迎<ref>2007年，廈門大學出版，周寧《東南亞華語戲劇史》（第一卷），ISBN：7561527543，第77頁。</ref>。 戰後鮮見他的[[消息|消息]]。

1933年底的[[香港|香港]]報章[[報導|報導]]謝醒儂的[[妻子|妻子]]是[[默片|默片]]女明星[[馮潔貞|馮潔貞]]，她亦曾經參演香港默片《鐵骨蘭心》（1931年11月29日香港公映）、《洞房雙屍案》（1934年1月21日香港公映）、《婚後的問題》（1934年8月5日香港公映）。亦提及謝醒儂與[[香港|香港]]當時的名女人、舞女、交際花[[孔雀_(演員)|孔雀]]合作到[[上海|上海]]表演，謝醒儂反串以女妝子喉唱曲；[[孔雀_(演員)|孔雀]]跳臀舞（Booty Dance）<ref>1933年11月1日（星期三），香港《[[工商日報|工商日報]]》，第四張第一版。</ref><ref>1933年12月2日（星期六），香港《[[天光報|天光報]]》，第四版。</ref>。

== 謝醒儂的電影 ==
# [[廣州|廣州]][[默片|默片]]《[[裂痕|裂痕]]》：謝醒儂、[[馮潔貞|馮潔貞]]、[[羅文煥|羅文煥]]、[[葉弗弱|葉弗弱]]主演<ref>2006年，中山大學出版社《中國非物質文化遺產》第 11 期，第106頁</ref>；
# [[廣州|廣州]][[默片|默片]]《[[前進|前進]]》；
# [[廣州|廣州]][[默片|默片]]《[[花落見花心|花落見花心]]》，[[導演|導演]]：姜白谷；謝醒儂、[[倩影儂|倩影儂]]、[[陳艷儂|陳艷儂]]、[[馮潔貞|馮潔貞]]、[[羅文煥|羅文煥]]、[[葉弗弱|葉弗弱]]合演；
# [[上海|上海]]粵語片《[[紅船外史|紅船外史]]》（又名：《[[美德夫人|美德夫人]]》或《[[麥夫人|麥夫人]]》）：謝醒儂飾演的[[英俊|英俊]]粵劇團班主，因與女主角麥周淑明（[[胡蝶|胡蝶]]飾演）參予義演[[活動|活動]]，被長舌[[舞女|舞女]]露斯（梁賽珠飾演）向[[女主角|女主角]]的[[丈夫|丈夫]]麥劍秋（[[胡藝星|胡藝星]]飾演）[[搬弄是非|搬弄是非]]，令麥家幾乎鬧至[[離婚|離婚]]收場，最後娛會冰釋。謝醒儂在戲裡反串演出他的三齣首本折子戲《[[天姬送子|天姬送子]]》（謝醒儂反串飾演[[七姐|七姐]]，與另一男花旦[[馮小非|馮小非]]合演）、《[[冰山火線|冰山火線]]》（謝醒儂反串飾演夏家璧、[[曾三多|曾三多]]飾演仗義[[知縣|知縣]]仇淵明、[[馮小非|馮小非]]反串飾演封大飄、[[七歲紅|七歲紅]]反串飾演封小飄）、《[[璇宮艷史|璇宮艷史]]》（謝醒儂反串飾演Louise[[女皇|女皇]]）、[[陳錦棠|陳錦棠]]飾演[[音樂家|音樂家]]Count Alfred Renard）。此片在1934年10月9日（星期二）在[[香港島|香港島]][[西環|西環]][[皇后大道西|皇后大道西]]（Queen's Road West）270－276號之[[中央戲院|中央戲院]]（Central Theatre）首映<ref>1934年10月8日（星期一），香港《[[工商日報|工商日報]]》，第三張；第三版。</ref>。
# [[香港|香港]][[粵語|粵語]]歌舞片《[[紅伶歌女|紅伶歌女]]》：[[觀眾|觀眾]]看謝醒儂反串扮女旦的[[機會|機會]]極多，此齣片裡的謝醒儂卻扮[[王子|王子]]，造型極之[[英俊|英俊]]，與[[女主角|女主角]]艷旦[[譚玉蘭|譚玉蘭]]及十多位[[舞女|舞女]]，大跳[[百老匯|百老匯]]式歌舞。此片在1935年3月8日（星期五）在[[香港島|香港島]][[西環|西環]][[皇后大道西|皇后大道西]]（Queen's Road West）270－276號之[[中央戲院|中央戲院]]（Central Theatre）首映<ref>1935年3月8日（星期五），香港《[[工商日報|工商日報]]》，第一張；第一版。</ref>。
