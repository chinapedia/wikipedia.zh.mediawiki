'''謝修平'''，是[[基隆|基隆]]地區[[政治世家|政治世家]]的成員之一，也是其[[父親|父親]]謝清雲的派系事業的[[繼承|繼承]]者兼[[領袖|掌控者]]。

== 政治生涯 ==
* 1973[[臺灣省議會|臺灣省議會]]第五屆議員，<ref>{{Cite web|url = http://www.tpa.gov.tw/opencms/digital/area/past/past01/member0008.html|title = 數位典藏>臺灣省議會專區>歷屆省議員查詢> 謝修平|accessdate = 2016-01-15}}</ref>
* 1984年當選基隆市第十屆議會市議員同時擔任基隆市議長之職。
* 1991年擔任第二屆[[國民大會|國民大會]]代表，任期結束後再度投身商界。

== 派系活動 ==
謝派區域性聯合獨佔經濟主要在[[基隆市第二信用合作社|基隆市第二信用合作社]]、[[基隆市|基隆市]]起卸運輸勞動合作社、台灣省合作社聯合社與合作金庫，參與投資經營的企業主要有[[味全食品工業|味全食品工業]]、[[台灣玻璃|台灣玻璃]]、[[三功投資|三功投資]]與[[國揚建設|國揚建設]]等公司。

其中，基隆市第二信用合作社是謝修平父子積聚經濟力量、進入政壇發展的重要基礎，謝家進而成為基隆市的政商家族。

1987年，謝修平以三功興業名義收購國揚建設，取得經營權，由其妻[[謝林曼麗|謝林曼麗]]任董事長兼總經理，也顯示出謝修平家族具有相當雄厚的經濟實力。<ref>{{Cite web|url = http://big5.china.com.cn/chinese/zhuanti/2004twldr/447084.htm|title = 基隆市：新舊派系更替，政黨競爭顯現|accessdate = 2016-01-15|author = |date = |publisher = }}</ref>
