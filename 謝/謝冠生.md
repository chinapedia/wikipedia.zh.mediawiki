'''谢冠生'''（{{bd|1897年|11月19日|1971年|12月22日|catIdx=Xie謝}}）本名'''寿昌'''，字'''冠生'''，[[以字行|以字行]]，[[浙江省|浙江省]][[绍兴府|绍兴府]][[嵊县|嵊县]]（今[[嵊州市|嵊州市]]）人。[[中华民国|中华民国]]法学家、政治人物。<ref name=xu>{{Cite book|author = 徐友春主編|title = 民国人物大辞典 増訂版|year = 2007|publisher = 河北人民出版社|isbn = 978-7-202-03014-1}}</ref><ref name=kmt>{{Cite book|author = 劉国銘主編|title = 中国国民党百年人物全書|year = 2005|publisher = 团結出版社|isbn = 7-80214-039-0}}</ref><ref name=zx>{{Cite book|author = 東亜問題調査会|title = 最新支那要人伝|year = 1941|publisher = [[朝日新聞社|朝日新聞社]]|isbn = }}</ref><ref name=liu>{{Cite book|author = 劉寿林等編|title = 民国職官年表|year = 1995|publisher = 中華書局|isbn = 7-101-01320-1}}</ref>

== 生平 ==
谢冠生的祖父谢廷钧是一名商人，父亲谢匡（字兰碗）是[[清朝|清朝]][[秀才|秀才]]。<ref>文史資料選輯 第78-80期，中国文史出版社，第80页</ref>1910年，谢冠生从小学毕业後，考入浙江省城[[杭州|杭州]]的[[浙江省立第一中学|浙江省立第一中学]]。1912年转入上海[[耶稣会|耶稣会]]所办的[[徐汇中学|徐汇中学]]，毕业後留任该校教员。一年後，进入[[商务印书馆|商务印书馆]]参与编辑《[[辞源|辞源]]》及《中国地名大辞典》。徐汇中学[[法国|法国]]籍校长[[姚缵唐|姚缵唐]]升任[[震旦大学|震旦大学]]校长後，邀谢冠生担任其秘书，并就读于震旦大学法科。1922年，谢冠生毕业，赴法国[[巴黎大学|巴黎大学]]法学研究所学习，1924年获该校法学博士学位。<ref name=xu/><ref name=kmt/><ref name=zx/>

归国后，谢冠生历任[[震旦大学|震旦大学]]、[[复旦大学|复旦大学]]、[[中国公学|中国公学]]、[[上海法政大学|法政大学]]等校教授。1926年冬，[[国民革命军|国民革命军]]攻占[[武汉|武汉]]，谢冠生任[[武汉国民政府|武汉国民政府]][[中華民國外交部|外交部]]秘书，参与收回[[汉口英租界|汉口英租界]]、[[九江英租界|九江英租界]]事宜。[[南京国民政府|南京国民政府]]成立后，谢冠生留在外交部，历任外交部秘書、条約委員会委員。<ref name=xu/><ref name=kmt/><ref name=zx/>同时，他还兼任[[国立中央大学|国立中央大学]]法律系主任、法学院代院长。<ref>薛化元，國史館現藏民國人物傳記史料彙編第一輯，1988年</ref><ref name=xu/><ref name=kmt/><ref name=zx/>1930年4月，调任国民政府[[司法院|司法院]]参事处参事，参与起草《[[训政时期约法|训政时期约法]]》。1934年10月，任国民政府[[司法行政部|司法行政部]]政务次长。1936年3月，任司法院秘書处秘書長。<ref name=xu/><ref name=kmt/><ref name=zx/><ref name=liu/>

[[抗日战争|抗日战争]]爆发后，1937年8月升任[[司法行政部|司法行政部]]部长，一直任至1948年12月。1945年5月，当选[[中国国民党|中国国民党]]第六届中央監察委員。1947年4月23日，[[国民政府|国民政府]]准免司法行政部部長謝冠生本職；特任謝冠生為[[行政院|行政院]]政务委员，兼司法行政部部长。<ref name="民國史大事記">{{cite book|editor=李新總主編，[[中國社會科學院|中國社會科學院]]近代史研究所[[中華民國史|中華民國史]]研究室編，韓信夫、姜克夫主編|title=《中華民國史大事記》|location=[[北京|北京]]|publisher=[[中華書局|中華書局]]|date=2011年7月}}</ref>{{rp|8340}}1948年12月離任司法行政部部長之後，转任公務员惩戒委員会委員長兼司法院秘書長。<ref name=xu/><ref name=kmt/><ref name=zx/><ref name=liu/>

[[File:三峽行修宮內謝冠生所題「行脩宮」匾額.jpg|thumb]]內謝冠生所題「行脩宮」[[匾額|匾額]]]]
[[第二次国共内战|第二次国共内战]]中国国民党敗北，謝冠生逃往[[台湾|台湾]]。1950年（民国39年）5月，被任命为司法院副院長。1958年（民国47年）6月，任[[司法院院長|司法院院長]]。<ref name=xu/><ref name=kmt/><ref name=liu/>

1971年（民国60年）12月22日，謝冠生在[[台北市|台北市]]病逝。享年75岁（满74歳）。<ref name=xu/><ref name=kmt/>

==逸事==
[[書法|書法]]知名，時常為各大廟宇題字，[[行天宮|行天宮]]之牌匾其中一面就出自於謝冠生之筆，行天三宮、[[關渡宮|關渡宮]]等名廟都有他的真跡。
