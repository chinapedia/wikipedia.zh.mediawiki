'''考試之神'''是[[日本電視台|日本電視台]]放送網於2007年7月開始播放的日本[[電視劇|電視劇]]。主演為[[山口達也|山口達也]]和[[成海璃子|成海璃子]]。此劇是成海自日劇[[琉璃之島|琉璃之島]]後時隔兩年零三個月再次在同系的日本電視台「[[日本電視台週六連續劇|週六連續劇]]」({{lang|ja|土曜ドラマ}})日劇時段中出任女主角的日劇。此劇為講述一個[[單親家庭|單親家庭]]的父子去接受中學入學考試的原創故事（即非改編漫畫或小說等等）。

== 故事簡介 ==
與妻子[[離婚|離婚]]的一個工作狂'''梅澤勇'''為了兒子'''梅澤廣'''的[[中學|中學]]入學[[考試|考試]]而煩惱不已。為了讓兒子考上自己心儀的學校，他找到了號稱「考試之神」的初中生菅原道子，請她做兒子的[[家庭教師|家庭教師]]，在眾家長、學生間流傳著，只要有「考試之神」擔任家庭教師，就一定可以考上理想的學校。在補習的過程中，考試之神還挑選了西園寺義嗣、手塚惠美，一同與梅澤廣進行培訓。

== 主要登場人物、演員 ==

; 菅原道子（14）（[[成海璃子|成海璃子]]）
:被稱為“考試之神”的初中三年級天才，秉承“中學考試的事情，在讀的中學生最了解”這一信念，能夠幫助自己指導的學生考上理想的初中，是個厲害的家庭教師。其本人自身也是橫掃連續17所初中入學試最高分寶座的得主。在名校就讀，為人冷淡高傲。
