'''稚内燈塔'''為[[日本|日本]][[北海道|北海道]][[稚內市|稚內市]][[野寒布岬|野寒布岬]]建立之[[燈塔|燈塔]]。與[[宗谷岬燈塔|宗谷岬燈塔]]及對岸國境[[庫頁島|樺太]]的燈塔是同様功能，是一座守護著為[[國際海峽|國際海峽]][[拉彼魯茲海峽|宗谷海峡]]航路上的重要燈塔。進一步，被選為「[[日本燈塔50選|日本燈塔50選]]」之一。由燈塔南延到西側的[[海岸線|海岸線]]，被指定劃入[[利尻禮文佐呂別國立公園|利尻禮文佐呂別國立公園]]範圍。從宗谷海峽的方向可以望見[[利尻島|利尻島]]・[[禮文島|禮文島]]等風景名勝之地。

== 歷史 ==
:1900年（[[明治|明治]]33年）12月10日初代稚内燈塔點燈。（現在的[[稚內分屯地|稚內分屯地]]内）
:1966年（昭和41年）1月10日第二代稚内燈塔移轉到現在的位置。

==燈塔結構與行政諸元==
===燈塔結構諸元===
* 航路標識番號：0510 [F6109]。
* 塔身塗色・構造：塔身在白色基質上塗6條赤横帯，塔形為水泥混凝土造。
* 燈器：LB-90型燈器。
* [[燈質|燈質]]：群閃白光（毎14秒內、隔6秒(暗)閃2次(明)）。
* 實効[[光度|光度]]：32萬[[坎德拉|cd]]。
* [[光程|光程]]：18.0[[浬|浬]]＜約33[[km|km]]＞。
* [[明弧|明弧]]：從6度至315度。
* 塔高＜地上～塔頂＞：42.7[[m|m]]。（全[[日本|日本]]排名第2位）
* 燈高＜平均海面～燈火＞：42.1m。
*燈塔座標：[[緯度|北緯]]45[[度|度]]26分58.51秒、[[經度|東經]]141度38分42.82秒。

===燈塔行政諸元===
* 管轄機關：[[日本海上保安廳|日本海上保安廳]][[第1管區海上保安本部|第1管區海上保安本部]]。
* 初點燈：1900年（[[明治|明治]]33年）12月10日。
* 所在地：[[北海道|北海道]][[稚內市|稚內市]]野寒布。

== 附屬施設 ==
* [[霧信號所|霧信號所]]（隔膜式霧笛(Diaphragm horn)：毎30秒吹鳴2回）

== 交通路線 ==
*從[[稚內港|稚內港]]旁[[稚內站|JR稚內站]]前的汽車招呼站（由[[稚內站|JR稚內站]]徒歩走5分鐘）出發，搭乘[[宗谷汽車|宗谷汽車]]1系統市内線往「野寒布」方向。在「野寒布」站下車（所需要時間約15分鐘），下車後從汽車招呼站徒歩走5分鐘即可抵達稚内燈塔（位於「[[稚內市立野寒布寒流水族館|稚內市立野寒布寒流水族館]]」與「[[稚內市青少年科學館|稚內市青少年科學館]]」旁）。
