'''豚足袋狸'''（''Chaeropus ecaudatus''）是一種細小及[[草食性|草食性]]的[[袋狸|袋狸]]，分佈在[[澳洲|澳洲]]內陸乾旱及半乾旱的平原。

== 特徵 ==
豚足袋狸約有小[[貓|貓]]的大小，外觀像[[兔耳袋狸|兔耳袋狸]]，有幼長的四肢，[[耳朵|耳朵]]大而且尖，尾巴很長。豚足袋狸在近看時則很奇特，前肢有兩指，指甲像蹄，有點像[[豬|豬]]或[[鹿|鹿]]。後腳的四趾較大，趾上的爪像小馬的蹄，其他的趾都已退化。

== 棲息地 ==
豚足袋狸棲息在多種的環境，包括林地及草原等。

== 行為 ==
很少人可以見到活的豚足袋狸。唯一的紀錄指牠們移動得很緩慢，似是拖著後肢來走路。但[[原住民|原住民]]則指牠們日間在巢中睡覺，被騷擾時則走得很快。有指牠們是棲息在樹穴中，或會挖掘短的隧道來築巢。

== 標本 ==
豚足袋狸的第一個標本是於1836年在[[維多利亞州|維多利亞州]]北部近[[穆理河|穆理河]]及[[馬蘭比吉河|馬蘭比吉河]]交界處採集的。其餘的標本是在[[南澳州|南澳州]]、[[西澳州|西澳州]]及[[北領地|北領地]]的乾旱郊區採集的。最後確實的標本是於1901年採集的。有指牠們在南澳州生存多了20年以上，而在西澳州則生存至1950年代。

== 保育狀況 ==
豚足袋狸在[[歐洲|歐洲]]殖民前已經不怎麼普遍，但於19世紀有科學紀錄時就已經大幅減少。到了20世紀初時，牠們就已經在[[西澳州|西澳州]][[滅絕|滅絕]]了。牠們滅絕的原因不明。消失當時[[狐狸|狐狸]]及[[野兔|野兔]]並未到達西澳州，而[[野貓|野貓]]則有可能掠食牠們。牠們最有可能的是失去棲息地而滅亡。

根據其[[食道|食道]]及[[齒列|齒列]]結構，並標本內的食物，豚足袋狸似乎主要是[[草食性|草食性]]的。另外牠們亦喜歡吃肉，包括[[螞蟻|螞蟻]]及[[白蟻|白蟻]]。

== 分類 ==
豚足袋狸以往是與[[兔耳袋狸|兔耳袋狸]]分類在[[袋狸科|袋狸科]]。但是由於牠們的外觀與普通的[[袋狸|袋狸]]及兔耳袋狸相距甚遠，加上[[分子|分子]]證據顯示牠們的不同，牠們現正分類在自己的[[豚足袋狸科|豚足袋狸科]]中。<ref name=msw3/>
