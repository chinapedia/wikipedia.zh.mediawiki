'''文化'''是[[日本|日本]]的[[年號|年號]]之一。在[[享和|享和]]之後、[[文政|文政]]之前。指[[1804年|1804年]]到1818年的期間。這個時代的[[天皇|天皇]]是[[光格天皇|光格天皇]]、[[仁孝天皇|仁孝天皇]]。[[江戶幕府|江戶幕府]]的將軍是[[德川家齊|德川家齊]]。

== 改元 ==
*享和4年[[二月十一|2月11日]]（公元1804年[[3月22日|3月22日]]） 因[[甲子革令|甲子革令]]而改元
*文化15年[[四月廿二|4月22日]]（公元[[1818年|1818年]][[5月26日|5月26日]]） 改元為文政

== 出典 ==
*《[[周易|周易]]·賁卦彖傳》的「觀於天文、以察時變、觀乎人'''文'''、以'''化'''成天下」
*《[[後漢書|後漢書]]·[[荀淑|荀淑]]傳》的「宣'''文'''教以章其'''化'''、立武備以秉其威」

== 文化年間要事 ==
*元年9月 俄羅斯使節[[尼古拉·雷薩諾夫|尼古拉·雷薩諾夫]]為求通商航行至長崎
*5年[[間宮林藏|間宮林藏]]至[[庫頁島|庫頁島]]探險
*5年8月[[費頓號事件|費頓號事件]]
*8年[[葛羅寧事件|葛羅寧事件]]
