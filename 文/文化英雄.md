'''文化英雄'''（'''culture hero'''），是以[[火|火]]、作物栽培法等有意義的發明或發現，促成人類世界文化秩序的[[傳說|傳說]]人物，或甚至是動物。

文化英雄也通常民族、部族的始祖或[[圖騰|圖騰崇拜]]等重要位置相關。但文化英雄與至高的[[神|神]]或[[創世神話|創世神]]不同、而是在既存世界上、創造特定文化要素或設定秩序的一個存在象徵。文化英雄之中、也有以喜愛惡作劇的頑童神靈的形式存在的例子。

例如：[[玻里尼西亞|玻里尼西亞]]神話的[[毛伊|毛伊]]，有過無數的冒險，除了將火傳予人類，也發明許多主要的技術，而他也有許多惡作劇及反社會的行動。他身為異常出生的半神半人，屢次在天、地上、地下界滯留，不斷在眾神與人間來回活動。也就是說，他使文化英雄成了模糊的角色，給了它一個頑童般的位置，即是只要是從異界為人類帶來所缺乏之物，便可視作一文化英雄。

[[印第安|印第安]]的神話中的[[郊狼|郊狼]]等特定動物也是同時擁有文化英雄與頑童兩面性質的一個代表，同時占有秩序與破壊、文化與自然媒介者的位置。

== 中国神话传说中的主要的文化英雄 ==

; [[仓颉|仓颉]]
: 是黄帝的臣子，发明了[[汉字|文字]]。

; [[嫘祖|嫘祖]]
: 黄帝之正妃，发明并教导人民养[[蚕|蚕]]。

; [[伶伦|伶伦]]
: 是黄帝的臣子，发明了[[十二律|音律]]。

; [[神农|神农]]
: 三皇之一，发明了[[耒耜|耒耜]]，并教导人民耕种。他还遍尝百草，首创用[[生药|天然草药]]来治疗疾病。

; [[禹|大禹]]
: 首创以疏导的方法治水，并以此功绩受[[禅让|禅]]帝位，建立了中国第一个朝代[[夏朝|夏朝]]。标志着华夏族以成功地进行[[黄河|黄河]]水利工程，联合了周围的部族形成联盟，并向国家过渡。

== 其他古代文明传说中的主要的文化英雄 ==

; [[普羅米修斯|普羅米修斯]]
: 以神話中登場的文化英雄而廠為人知的，是[[希臘神話|希臘神話]]普羅米修斯。他違背了[[宙斯|宙斯]]，將火與穀物帶給人類。被他所做所為激怒的宙斯，將他禁錮於高加索山的山頂，並使巨鷹食其[[肝臟|肝臟]]。

; [[魁札爾科亞特爾|魁札爾科亞特爾]]
: 在中美俱有強大勢力的羽蛇神[[魁札爾科亞特爾|魁札爾科亞特爾]]亦是著名的文化英雄。他不僅是[[阿茲特克神話|阿茲特克神話]]主要的神、人類的創造神、也是將火、農耕、文化、藝能等必需技能授予人類的神。

; [[撒但|蛇]]
: [[舊約聖經|舊約聖經]][[創世記|創世記]]中誘導[[夏娃|夏娃]]與[[亞當|亞當]]吃下[[知善惡樹|知善惡樹]]的果實，傳予人類知善惡能力的角色。

; [[Grigori|Grigori]]
: 舊約聖經[[偽經|偽經]]『[[以諾書|以諾書]]』中[[墮天使|墮天使]]的一員。也稱作Egrigori。傳予了人類禁忌的知識。

; [[維納莫寧|維納莫寧]]
: 在[[芬蘭|芬蘭]]的[[民俗學|民俗傳承]]及[[卡勒瓦拉|卡勒瓦拉]]佔有重要位置。
