'''文津街'''是位于中国[[北京市|北京市]][[西城区|西城区]]东北部的一条大街。<ref name=bjdmd>北京地名典，中国文联出版社，2008年，第155页</ref>

==简介==
东起[[北长街|北长街]]北口，与[[景山前街|景山前街]]连接，西至[[府右街|府右街]]北口，与[[西安门大街|西安门大街]]连接。[[明朝|明朝]]、[[清朝|清朝]]时，此街无名称，位于[[西安门|西安门]]内，是[[北京皇城|北京皇城]]西部的通道。[[中华民国|中华民国]]后，在[[明朝|明朝]][[玉熙宫|玉熙宫]]（[[清朝|清朝]]改为[[御马监|御马监]]）旧址，兴建了北平图书馆新馆（今为[[中国国家图书馆古籍馆|中国国家图书馆古籍馆]]）。民国二十年（1931年），[[承德|承德]][[避暑山庄|避暑山庄]]文津阁内的《[[四库全书|四库全书]]》移入该馆，于是此街以阁名，称为“文津街”<ref name=bjdmd/>。为了纪念文津阁《四库全书》在国立北平图书馆入藏，馆方于1931年4月10日致函北平市公安局，拟将馆舍门前大街定名“文津街”。1931年5月18日，北平市公安局函复国立北平图书馆，同意将其馆舍门前大街定名“文津街”。并立即着手安装路牌标志和门牌等事<ref name=youlai>{{Cite web|url =http://www.bjxch.gov.cn/BJXCHHuDongJiaoLiu/BJXCHXCBAXiangQing.ycs?GUID=561847|title =文津街的由来|author =|date =2013-11-22|publisher =北京西城报|language =|accessdate =|quote =}}{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>。

起初，文津街东起[[金鳌玉蝀桥|金鳌玉蝀桥]]（北海大桥），西到[[养蜂夹道|养蜂夹道]]南口，全长约70米，同国立北平图书馆南院墙长度相等。是北平最短的街。全街仅一个门牌即文津街1号，为国立北平图书馆。这条以阁命名的街在北平很特殊，在一条街上只有一个门牌也无先例。当时[[中南海|中南海]]是北平市政府所在地，其正门在[[西长安街|西长安街]][[新华门|新华门]]，开在文津街的门是后门，不另编门牌号。金鳌玉蝀桥西侧的[[北海公园|北海公园]]西南门是旁门，也未另编门牌号<ref name=youlai/>。
[[File:ISUZU_N-series_in_Beijing.jpg|thumb]]南侧的文津街]]
北海公园南侧的文津街上，东、西原先各有一组“三座门”，分别称“北海东三座门”、“北海西三座门”。北海东三座门原址位于金鳌玉蝀桥（北海大桥）以东，有东、西两道红墙，西面红墙断面位置在北海公园南门（正门）售票处以东紧挨着的南北向红墙，与其平行的东面红墙断面位置是自西面红墙往东约20米处。每道红墙并列有三座方形琉璃门楼，中间大两边小，[[汉白玉|汉白玉]]底座，黄琉璃瓦顶，左右嵌有黄绿琉璃团花和花饰。两道红墙门之间，南面是中南海的蕉园门（现仍可见中南海外墙上留下的“蕉园门”原位置遗迹），北面是北海的桑园门。当“北海东三座门”东、西两道门共六个门关闭时，宫廷人员可在这个封闭区域往来北海和中南海之间。另外，至少在[[中华民国|中华民国]]时期，在西面三座方形琉璃门楼之间的红墙上已又各开一个方形门洞，正对着东面三座方形琉璃门楼中的南北两座较小的门，以便通行车辆。三座方形琉璃门楼每个门的东西两侧各有半米高的小松柏树围成南北两圈，这两圈长圆形灌木之间是人行道，灌木及人行道周围还有一圈马路牙子。因为妨碍交通，北海东三座门于1955年1月2日至6日被拆除。如今其原址北部留有北海桑园门东西两侧的两小段南北向红墙，北海东三座门的原址现为地下通道上方的文津街马路和停车场<ref name=beihai>{{Cite web |url = http://news.163.com/12/0316/22/7SOKL35R00014AED.html |title =北海“三座门”在哪儿  |author =  |date = 2012-03-16 |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref>。

北海西三座门原址位于金鳌玉蝀桥（北海大桥）以西，北海西南门（阳泽门）和[[中国国家图书馆古籍馆|国家图书馆古籍馆]]之间的文津街上。西三座门有东西三道红墙，每道并列有三座方形琉璃门楼，三个门是紧挨着的，共计9座门楼，中间大两边小。西三座门以东靠南是中南海的福华门，与北海的阳泽门相对。中国第一条皇家专用铁路[[西苑铁路|西苑铁路]]南起中海瀛秀园，穿过中南海北门福华门，进入正对的北海西南门阳泽门，沿北海西岸北行，经极乐世界往东，到终点镜清斋，全长1510.4米。铁路建于[[光绪|光绪]]十四年（1888年），1900年[[八国联军|八国联军]]攻入北京，该铁路遭严重破坏。1925年北海公园开园前，西苑铁路被拆除。1955年北海西三座门也被拆除<ref name=beihai/>。

北海公园南侧原为三座门街，现已并入文津街。原文津街1号国立北平图书馆后改为文津街7号[[中国国家图书馆古籍馆|中国国家图书馆古籍馆]]。2010年代，文津街门牌只有1号和7号，没有2号、3号、4号、5号和6号<ref name=youlai/>。

文津街景色美丽。文津街东段是金鳌玉蝀桥（北海大桥），横跨[[北海公园_(北京)|北海]]与[[中南海|中南海]]之间，建于明朝，1950年代加宽，[[文化大革命|文化大革命]]期间将[[汉白玉|汉白玉]]桥栏改为栅栏，涂白漆，高两米多。大桥东、西两端原有牌楼，东题“玉栋”，西题“金鳌”，1950年代拆除。文津街北侧有[[北海公园_(北京)|北海及团城]]，南侧为[[中南海|中南海]]<ref name=bjdmd/>。
