'''文理分科'''是在部分[[東亞|東亞]]国家和地区实行的教育制度，即将教学课程分为[[文科|文科]]和[[理科|理科]]让学生选择后进行分别教育。

== 中國大陆 ==

文理分科在中國始于1977年的恢复[[高考|高考]]。但事实上这种制度源于对[[苏联|苏联]]模式的照搬。[[二战|二战]]使[[苏联|苏联]]损失了大批[[知识分子|知识分子]]，因此战后重建急需以分科教育来快速培养各类专业人才。

中國1949年后，同样急需专业人才，因此开始了文理分化的步伐，首先成立了[[西安|西安]]、[[南京|南京]]、[[重庆|重庆]]等最早的8所外语学校，并在50年代开始了对[[高校|高校]]的合并。当时实现[[工业化|工业化]]是经济建设的主要目标，为了适应对专业人才的需求，自1952年起，将同类专业院校合并为专业学院，建起适应[[计划经济|计划经济]]体制的高校体制。这样的合并加强了[[工科院校|工科院校]]的专业性，但也造成专业面窄，且理工、文理分家，文法、财经等学科也被削减。

但在[[文革|文革]]结束前，中國的[[高中|高中]]一般都没有实行文理分科。1977年，恢复[[高考|高考]]制後，为分类选拔人才，[[高中|高中]]文理分科才开始。

分科一般是在高一进入高二阶段进行的。虽然实行文理分科，但是不管文、理都包括所分的科目为[[政治|政治]]、[[历史|历史]]、[[地理|地理]]、[[物理|物理]]、[[化学|化学]]、[[生物|生物]]。

*必修科：[[语文|语文]]、[[数学|数学]]、[[外语|外语]]（[[英语|英语]]、[[俄语|俄语]]或[[日语|日语]]，近年亦加入[[西班牙语|西班牙语]]及[[韩语|韩语]]）
*文科綜合：[[政治|政治]]、[[历史|历史]]、[[地理|地理]]
*理科綜合：[[物理|物理]]、[[化学|化学]]、[[生物|生物]]

但现行的制度要求文、理科仍保留其他科目教学。通常在高二开始的文理分科后，学生会将重点放在文综或理综上，实际的教学上老师也不太重视，而大部分的学生也轻视这些科目，随着学业水平测试的结束，高三时学生会全力攻克高考科目。而如果在会考或其它的检查之后学校就直接将这些科目从[[课程表|课程表]]上删除。

文科、理科的数学和文理综教材上也有所区别。数学文科一般比理科要简单，在教学和考试都有很大差异。

现前各省的高考制度略有不同，某些地区增加学业水平测试作为参加考试的判定，或有综合能力测试。目前江苏省考生可在文理综中选考2门课，单科考试的难度会比文理综考试高。

对于学生选文还是选理，学生乃至家长、老师都有一番艰难的抉择。当然有人对分科持反对态度，但面对高考的指挥仍须作出决定，不是选文就是选理。总体而言选理的人数要多于选文，在一般的学校这个比例是3：1。影响学生作出选择的因素有和很多，主要包括：某个学生是擅长文还是理、未来就业形势（这主要是受家长及老师的影响）、个人的兴趣、爱好、及理想。但有时候其它因素也会影响到一个学生的选择；老师的实力对比，如果某科教师的实力强劲，则易吸引学生选择此科。

2017年开始，上海、浙江的考生不再进行文理分科，选课增加至三门。

==香港==
{{see|三三四高中教育改革}}

===新高中學制 （2009年-）===
* 取消會考及高考，由[[香港中學文憑考試|香港中學文憑考試]]取代。
* 中英數為必修（包括非華裔學生）；而且所有學生必須修讀[[通識教育|通識教育]]科，務求達到「文中有理、理中有文」。
* 不再強制文理分科，學生可以選擇文理兼備的科目；但有些學校提供給學生的選擇組別仍是類似以往的文理分科
* [[附加數學|附加數學]]、純數學、應用數學、數學與統計學課程改為[[數學科延伸部分|數學科延伸部分]]。


以往，香港大部份中學於中四（即高中）開始分科；過去[[文法中學|文法中學]]主要分為文（文商）、理兩科，部份學校則分為文、理、[[商科|商]]三組，[[工業學校|工業學校]]另以[[工科|工科]]代替部份理科。
各校修讀的科目會略有差異，因為並非所有學校都有開辦全部科目，部份學校亦容許學生選修不同組別的科目。

==中華民國文理分科==
從高中二年級開始分組。文、理都必修國文、英文、數學（文組與理組的數學在高二時分流，有A、B版，並於高三時分為自然組的「數學甲」，與社會組的「數學乙」），直至高三結束；歷史、地理、公民與社會等三科，文、理組至二年級皆為必修，高三文組會繼續修習；自然科之物理（有分流A、B版，理組上後者）、化學、生物、地球科學，文、理組高一皆為必修，高二視各校配課時數，於四科設定不同權重給文、理組，高三理組會繼續修習。

過去[[大學聯合招生考試|聯考]]時代，文、理分為甲組（理工）、乙組（文哲）、丙組（農醫）、丁組（法商）共四組，其中甲組、丙組為自然組；乙組、丁組為社會組。

目前則分為三類組：
*'''第一類組'''（簡稱一類）：社會組/'''文組'''，主修課程有國文、英文、數學（乙）、歷史、地理、公民與社會，物理、化學、生物、地球科學則僅在二年級必修或選修，通常在[[大學入學指定科目考試|指定科目考試]]中應考國文、英文、數學乙、歷史、地理、公民與社會。大學可選擇[[文學|文]]、[[歷史學|史]]、[[哲學|哲]]、[[政治學|政]]、[[法學|法]]、[[商學|商]]、[[傳播學|傳播]]、[[教育學|教育]]、[[藝術|藝術]]、[[外語|外語]]、[[社會科學|社會科學]]等學系。

*'''第二類組'''（簡稱二類）：自然組/'''理組'''，主修科目有國文、英文、數學（甲）、物理、化學，生物、地球科學、歷史、地理、公民與社會則僅在二年級必修，通常在指定科目考試應考國文、英文、數學甲、物理以及化學。二類與三類的主要差異為不著墨於生物，因此大學以[[理學|理學院]]、[[工學|工學院]]為主要學系。

*'''第三類組'''（簡稱三類）：自然組/'''理組'''，主修科目有國文、英文、數學（甲）、物理、化學、生物，地球科學、歷史、地理、公民與社會則僅在二年級必修，通常在指定科目考試應考國文、英文、數學甲、物理、化學以及生物。大學可選擇[[醫學|醫學院]]、[[生物學|生物學]]、[[農學|農學院]]、[[獸醫學|獸醫學]]等學系。
部分學校僅區分為社會組與自然組，不區分二類與三類。

目前每個類組都必須參加[[大學學科能力測驗|學科能力測驗]]，且考試範圍相同。指定科目考試並未限制應考身分，因此類組可互相跨考，但由於商學院近來趨向只採計文、理組都有的國文、英文、數學，而自然組學生修習的版本較為高深，使得自然組學生較能跨考社會組科目，也較易拿到高分。2009年起列公民與社會為指定科目考試的考科之一。

== 選組的選擇 ==

对于学生选組，学生乃至家长、老师都有一番艰难的抉择。当然有人对分組持反对态度，但面对將來升讀大學的選系，仍须作出决定。

影响学生作出选择的因素有和很多，主要包括：個人專長、未来就业形势、个人的[[兴趣|兴趣]]和[[理想|理想]]、家長的期望等。

但有时候其它因素也会影响到一个学生的选择，如成绩较差的学生更多会被建议选擇文組；老师的实力对比，如果某科教师的实力强劲，则易吸引学生选择此組。

== 争议 ==
目前文理分科与[[应试教育|应试教育]]一样存在着大争议。
===反对者观点===
有部分[[教育家|教育家]]、[[学者|学者]]及教师、家长反对文理分科。他们认为文理分科让理科学生不再学习历史、地理，更少阅读[[文学经典|文学经典]]等，使理科生知识面偏窄，致使缺乏[[人文精神|人文精神]]；而同时这也导致文科生对物理、化学、生物陌生，不知道基本的[[自然科学|自然科学]]知识，导致缺乏[[科学精神|科学精神]]和[[理性|理性]]，也不懂得用已證明行之有效的[[科学方法|科学方法]][[解決問題|解決問題]]，阻碍学生全面发展和提升[[素質教育|综合素质]]。

=== 支持者观点 ===
也有部分的学生、家长和老师及其他人士认为文理分科有利学生个性发展，让他们自由选择所要学的科目，使学生学有专长。同时文理分科可以减轻学习负担、压力，尤其面对大學入學試，学生更可花充分时间在要考的科目上。
高中文理分科实际上解除了相当多学生的厌学感，让他们重新认识了学习之必要，进而“术业有专攻”。正是对文科与理科的正确认识，有学生很早就放弃了自己不想学又学不好的科目。如果强制性地不分科，真不知道有多少人知难而退，中辍学业。

== 改革 ==

由于文理分科存在着弊端，因此部分人士提出改革方案或直接取消文理分科，支持改革者一般倾向认为应将文理分科推迟进行，不宜在高二，而是在高三总复习时期，由学生自由地作出选择。复习阶段选择文科的学生主攻文科，而理科生则主攻理科。他们这样可以较充分地保证学生综合素质的发展。但是，也有部分人士认为，推迟分科将进一步加大学生课业负担。

也有部分人士呼籲取消文理分科，在2005年，中國大陆[[政协|政协]]委员[[朱永新|朱永新]]提出应组织专家进行取消高中、高考文理分科的论证，争取从2006年高中入学和2008年高考取消文理分科。目前，中国大陆仍在實行文理分科制度。

=== 香港 ===
於2009年開始推行[[三三四高中教育改革|新高中學制]]，[[通識教育|通識教育]]為必修科之一，並鼓勵學校以彈性的選科組合和時間表取代傳統的文理分科制度。
