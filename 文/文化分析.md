'''文化分析'''（英語：'''Cultural analytics'''）是一種對大量[[文化|文化]]相關[[視覺|視覺]]材料的研究，視覺材料包括數位化的文物以及當代的視覺、互動媒體。面對如何有效率地探索大量文化材料的挑戰，文化分析學者透過高解析度的[[可视化_(计算机图形学)|可視化]]、數位[[影像處理|影像處理]]的方法，以及直觀的視覺技術。這些方法用以處理既存的[[人文學|人文學]]研究問題，也用以探索新問題，並發展出合乎[[21世紀|21世紀]]初數位文化規模的新理論。

== 歷史 ==
「文化分析」一詞由[[列夫·曼諾維奇|列夫·曼諾維奇]]於2007年提出，文化分析與可視化分析論、與數據分析有許多共同的觀念與方法：<blockquote><div>視覺數據分析將先進的計算方法與負責的圖形引擎融為一體，以挖掘人類在複雜畫面中發現模式與結構的非凡能力。視覺數據分析大量應用於大量、異質、動態的數據集上，如在[[天體物理學|天體物理學]]、[[流體力學|流體力學]]、[[生物學|生物學]]等研究複雜過程的研究中所產生的數據，這些技術已經能夠進行實時互動操作變量。超高解析度顯示器則使研究人員能放大特定檢視，或依循有意思的路徑進行瀏覽。新的研究開始將這些工具應用於社會科學與人文學上，這些技術對於協助我們理解負責社會過程（如學習、政治、組織變革、知識傳播）等，提供了相當大的希望。</div><ref>{{cite web|url=http://wp.nmc.org/horizon2010/chapters/visual-data-analysis/ |title=Four to Five Years: Visual Data Analysis |work=2010 Horizon Report |publisher=The New Media Consortium |deadurl=yes |archiveurl=https://web.archive.org/web/20110810143644/http://wp.nmc.org/horizon2010/chapters/visual-data-analysis/ |archivedate=2011-08-10 }}</ref></blockquote>計算能力與互動可視化的技術發展使得使用視覺探索大型數據集變得容易，然而以此方法探討文化與社會過程的嘗試早於電腦的發展。1869年[[查爾斯·約瑟夫·米納德|查爾斯·約瑟夫·米納德]]用以呈現拿破崙對俄戰爭中法軍一路軍力損失情形的圖即是一例<ref>[http://www.edwardtufte.com/tufte/minard Napoleon's March on Moscow]</ref> 。近期則有[[皮耶·布迪厄|皮耶·布迪厄]]在《區分》（La Distinction）一書中對20世紀中期[[巴黎|巴黎]]人文化消費的歷史調查，開啟了以大量數據集研究文化與[[美學|美學]]的方法。更近期的則有[[佛朗哥·莫雷蒂|佛朗哥·莫雷蒂]]（Franco Moretti）的《圖表、地圖、樹木：文學史的抽象模型》<ref name="Moretti, 2005">{{Cite book|title=Graphs, maps, trees: abstract models for a literary history|last=Moretti|first=Franco|publisher=Verso|year=2005|isbn=1-84467-026-0 <!--(alk. paper)-->|pages=119}}</ref> 一書，與其他[[數位人文學|數位人文學]]計畫一般，展現了大規模分析文化資料的益處。

== 目前的研究 ==
到目前為止，文化分析技術已經應用於[[電影|電影]]、[[動畫|動畫]]、[[視訊遊戲|電子遊戲]]、[[漫畫|漫畫、]][[雜誌|雜誌、]][[書籍|書籍]]和其他印刷出版物、[[藝術品|藝術作品]]、[[照片|照片]]和各種其他媒體內容上。所使用的技術從[[个人电脑|個人電腦]]上的[[开源软件|開源軟體]]到處理HIPerSpace（42000 x 8000 pixels）的[[超級電腦|超級電腦]]<ref>[http://vis.ucsd.edu/mediawiki/index.php/Research_Projects:_HIPerSpace HIPerSpace]</ref>。

== 方法論 ==
屬於文化分析的方法包括對大量文化相關數據的數據挖掘（如對圖書館目錄、圖集、社群網路資料庫的研究）、影像處理等等。文化分析方法也用以研究與詮釋遊戲及其他軟體，從現象學層次（[[人機介面裝置|人機介面]]、[[特徵提取|特徵擷取]]）到物件層次（[[源碼分析|源碼分析]]）都有。文化分析高度依賴以軟體為基礎的工具，並與新近的軟體研究相關。儘管文化分析的研究對象通常是一物的數位化
形式，研究對象本身不一定要是數位作品。
