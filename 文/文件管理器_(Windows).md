'''文件管理器'''（{{lang-en|'''File Manager'''}}）是[[Microsoft_Windows|Microsoft Windows]]在1990年至1999年的产品中附带的[[文件管理器|文件管理器]]程序。<ref name="WinHistoryDesktop">{{Cite web|url=http://www.microsoft.com/windows/WinHistoryDesktop.mspx|title=Windows Desktop Products History|accessdate=2006-08-19|author=Microsoft|date=2006-03-07|work=Windows History|publisher=Microsoft}}</ref>它是一个单实例的图形界面，取代了[[MS-DOS|MS-DOS]]的命令行界面来管理[[電腦檔案|文件]]（复制、移动、打开、搜索等）。尽管文件管理器在[[Windows_95|Windows 95]]和[[Windows_NT_4.0|Windows NT 4.0]]及之后的一些版本中也有提供，[[檔案總管|Windows Explorer]]后被引入并作为主要的文件管理器。

== 概述 ==
此程序的界面会在左侧面板显示[[目录_(文件系统)|目录]]列表，在右侧面板显示当前目录内容的列表。文件管理器允许用户创建、重命名、移动、[[打印机|打印]]、复制和搜索，以及删除文件与目录、设置[[文件系统权限|权限]]（[[文件属性|属性]]；包括存档、只读、隐藏和系统），还有关联各文件类型的打开程序。另外还有标签和[[格式化|格式化]]磁盘，管理文件夹的文件共享，以及连接和断开[[文件服务器|网络驱动器]]的工具。在[[Windows_NT|Windows NT]]系统上，它还可以设置[[NTFS|NTFS]]分区上的文件或文件夹的[[存取控制串列|ACL]] ，借助shell32安全配置对话框（也被Explorer和其他Windows文件管理器使用）。在NTFS驱动器上，用户可以压缩单个文件或整个文件夹。

文件管理器的Windows NT版本允许用户更改目录、文件、网络和用户权限。

从Windows 95和Windows NT 4.0开始，文件管理器被Windows Explorer取代。不过<code>WINFILE.</code><code>EXE</code>程序仍存在于Windows 95、[[Windows_98|Windows 98]]和[[Windows_Me|Windows Me]]（16位可执行文件）及Windows NT 4.0（32位可执行文件）中。32位版<code>WINFILE.</code><code>EXE</code>文件（4.0.1381.318）在Windows NT 4.0 Service Pack 6a（SP6a）中分发。最后一个16位版<code>WINFILE.</code><code>EXE</code>文件（4.90.3000）在Windows Me操作系统中分发。

{{tsl|en|Ian_Ellison-Taylor|Ian Ellison-Taylor}}是Windows 3.1团队中负责文件管理器和打印管理器[[殼層|壳层]]的开发人员。<ref>{{Cite web|url=http://channel9.msdn.com/ShowPost.aspx?PostID=284474|title=Dr Sneath and Ian Ellison-Taylor: Windows History|accessdate=2007-03-10|author=Microsoft|date=2007-02-20|publisher=Microsoft}}</ref>

== 版本 ==

=== 16位 ===
文件管理器的原始版本是一个16位程序，它支持那时使用的8.3文件名。

它不支持扩展的文件名，直至Windows 95——包含[[长文件名|长文件名]]和文件名包含空格的支持。<span class="cx-segment" data-segmentid="110"></span>

在Windows 3.1x和Windows for Workgroups 3.1x中分发的16位版本存在[[2000年问题|2000年问题]]。微软后为所有Windows 3.1x环境分发了纠正后的二进制文件。<sup class="noprint Inline-Template" style="white-space:nowrap;">]</sup><ref>{{Cite web|url=http://support.microsoft.com/kb/q85557/|title=File Manager Shows Garbled Date for Year 2000 or Later|accessdate=2006-08-18|author=Microsoft|date=2006-08-22|work=Microsoft Help and Support|publisher=Microsoft}}</ref>

=== Windows NT ===
文件管理器曾被重写为适用Windows NT的32位程序。新的版本纠正了长文件名的处理和支持[[NTFS|NTFS]]文件系统。它被包含于[[Windows_NT_3.1|Windows NT 3.1]]、[[Windows_NT_3.5|3.5]]、[[Windows_NT_3.51|3.51]]和[[Windows_NT_4.0|4.0]]。

===Windows 10===
2018年4月，微软将文件管理器源代码以[[MIT許可證|MIT許可證]]释出，并可在Windows 10上运行<ref name="GHWF">{{cite web |url=https://github.com/microsoft/winfile |title=Microsoft/winfile: Original Windows File Manager (winfile) with enhancements |author=Microsoft Corporation |publisher=GitHub |access-date=7 April 2018}}</ref>
。2019年1月21日，微软以UWP应用的形式将文件管理器发布至[[Windows商店|Windows商店]]。<ref>{{Cite web|title=购买 Windows File Manager - Microsoft Store zh-CN|url=https://www.microsoft.com/zh-cn/p/windows-file-manager/9p7vbbbc49rb|work=Microsoft Store|accessdate=2019-02-02|language=zh-cn}}</ref>
