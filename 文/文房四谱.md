'''文房四谱'''，[[北宋|北宋]]著名作家[[苏易简|苏易简]]的作品，该书主要用来介绍[[文房四宝|文房四宝]]的专著。<ref name="oks " >{{cite web |url=http://news.ifeng.com/history/zhongguogudaishi/detail_2009_11/13/324951_0.shtml |title=《文房四谱》:“文房四宝”的最早专著 |accessdate= 2009年|work= |publisher= 凤凰网|language =zh-cn}}</ref>

== 内容 ==
该书主要介绍文房四宝，研究和记述了[[笔|笔]]、[[砚|砚]]、[[纸|纸]]、[[墨|墨]]的诞生、制造的工艺和技术、流传的民间故事以及关于文房四宝的诗词曲赋，该书还收录了[[王羲之|王羲之]]的《[[笔经|笔经]]》、[[韦仲将|韦仲将]]的《[[笔墨方|笔墨方]]》、《[[冀公墨法|冀公墨法]]》等宋朝之前的关于文房四宝的书籍。<ref name="oks " />

== 章节 ==
全书分笔谱、砚谱、纸谱、墨谱五卷。每卷分为“之叙事”、“之造”、“之杂说”、“之辞赋”。<ref name="oks " />

== 作者简介 ==
[[苏易简|苏易简]]，字太简，[[梓州|梓州]][[铜山|铜山]]（今[[四川|四川]][[中江|中江]]）人，[[宋太宗|宋太宗]]时期[[状元|甲科状元]]，苏易简及其孙子[[苏舜钦|苏舜钦]]、[[苏舜元|苏舜元]]也被人们称为“[[三苏|三苏]]”。<ref name="oks " />
