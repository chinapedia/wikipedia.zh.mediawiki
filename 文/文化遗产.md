'''文化遗产'''，又可稱'''文化資產'''、'''文化財產'''或'''文化財'''，是指具有[[歷史|歷史]]、[[藝術|藝術]]、[[科學|科學]]等[[文化|文化]]保存價值，並經[[政府機構|政府機構]]或[[國際組織|國際組織]]指定或登錄之[[物品|物品]]。[[中文|中文]]在概念上分为“有形文化遗产”和“无形文化遗产”。

== 物質文化遗产 ==
“物質文化遗产”即传统意义上的“文化遗产”，根据《[[保护世界文化和自然遗产公约|保护世界文化和自然遗产公约]]》，包括历史[[文物|文物]]、历史[[建筑|建筑]]、人类[[文化|文化]][[遗址|遗址]]。
* 文物，從[[歷史|歷史]]、[[藝術|藝術]]或[[科學|科學]]角度看具有突出的普遍價值的建築物、[[碑刻|碑刻]]和[[雕塑|雕塑]]、[[书籍|书籍]]、[[书法|书法]]与[[绘画|绘画]]、具有考古性質成份或結構、[[銘文|銘文]]、[[洞窟|洞窟]]以及聯合體；
* 建築群，從歷史、藝術或科學角度看在建築式樣、分佈均勻或與環境景色結合方面具有突出的普遍價值的單立或連接的建築群；
* 文化遺址，從歷史、[[審美|審美]]、[[人種|人種]]學或[[人類學|人類學]]角度看具有突出的普遍價值的[[人類|人類]][[工程|工程]]或[[自然|自然]]與人聯合工程以及[[考古|考古]]等区域。

== 非物質文化遗产 ==
{{Main|非物质文化遗产}}

根據[[聯合國|聯合國]]教科文組織《保護非物質文化遺產公約》(Convention for the Safeguarding of the Intangible Cultural Heritage)的定義，是指：被各群體、團體、有時為個人視為其文化遺產的各種實踐、表演、表現形式、知識和技能及其有關的工具、實物、工藝品和文化場所。

== 東亞的文化遺產保護政策 ==
==={{PRC}}===
[[中華人民共和國|中華人民共和國]]制定有《[[中华人民共和国文物保护法|中华人民共和国文物保护法]]》（1982年通過，1991年、2007年兩次修正、2002年修訂<ref>[http://www.gov.cn/flfg/2007-12/29/content_847433.htm 全国人民代表大会常务委员会关于修改《中华人民共和国文物保护法》的决定（主席令第八十四号）]</ref>），於1985年正式加入《保護世界文化與自然遺產公約》，並於2002年由[[中华人民共和国文化部|文化部]]、[[國家文物局|國家文物局]]、[[國家計委|國家計委]]、[[財政部|財政部]]、[[教育部|教育部]]、[[建設部|建設部]]、[[國土資源部|國土資源部]]、[[環保總局|環保總局]]、[[國家林業局|國家林業局]]向各地方政府發佈了《关于加强和改善世界遗产保护管理工作的意见》<ref>[http://www.cnwh.org/ 中國世界遺產網]</ref>。
