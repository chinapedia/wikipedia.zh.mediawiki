《'''妇女参政论者'''》（{{lang-en|''Suffragette''}}）是2015年英国[[时代剧|历史剧情]]电影，由{{le|莎拉·加夫隆|Sarah Gavron}}导演，{{le|阿比·摩根|Abi Morgan}}编剧。演出阵容包括[[凯瑞·穆里根|凯瑞·穆里根]]、[[海伦娜·博纳姆·卡特|海伦娜·博纳姆·卡特]]、[[布兰顿·葛利森|布兰顿·葛利森]]、{{le|安-玛莉·杜芙|Anne-Marie Duff}}、[[本·威士肖|本·威士肖]]和[[梅丽尔·斯特里普|梅丽尔·斯特里普]]。

摄影于2014年2月24日开始。影片是第一部获[[国会议员|国会议员]]批准在[[威斯敏斯特宫|国会大厦]]拍摄的历史电影。影片于2015年10月12日由[[百代电影公司|百代电影公司]]在英国发行，2015年10月23日由[[焦点影业|焦点影业]]在美国[[有限上映|有限上映]]，11月12日登陆香港，2016年1月8日登陆台湾。

== 剧情 ==
1912年，莫德·瓦斯（[[凯瑞·穆里根|凯瑞·穆里根]] 饰）是一名24岁的洗衣女工。一天在运送包裹时，她碰上了[[妇女参政论者|婦女參政論者]]发起的一场砸碎玻璃窗的暴乱，其间认出她的同事之一维奥莱特·穆勒（{{le|安-马莉·杜芙|Anne-Marie Duff}}饰）。后来，国会议员的妻子爱丽丝·霍顿（{{le|萝玛拉·嘉瑞|Romola Garai}}）发动洗衣店女工去议会畅所欲言，为确保投票权而作证词。维奥莱特本来要去作证，却被她那残暴的丈夫殴打，之后莫德成为了证人。穆德的证词让她充满热情，打算跟维奥莱特和其他女性去看看女性是否能获得投票权。当她得知议员们并没有打算给她们投票时，警察出动并镇压了与会的女性。穆德因与大部队齐驱并进而被逮捕关押一个星期。在监狱里，她遇见[[艾米琳·潘克斯特|艾米琳·潘克斯特]]的得力助手[[埃米莉·戴维森|埃米莉·戴维森]]。

刑满释放回到家，穆德被邻居和同事冷眼相待。她向丈夫桑尼承诺会远离妇女参政运动。然而，莫德被邀请去一场秘密集会中听潘克斯特（[[梅丽尔·斯特里普|梅丽尔·斯特里普]] 饰）演讲。在那里，她与潘克斯特交流了一小会，之后因被警察推倒在她家门前再次遭到逮捕。之后一次回家，她丈夫直接把她仍在大街上。穆德坚持要看儿子一眼，但被丈夫阻止。之后穆德进行参与斗争，她的画像被印在妇女参政论-{}-者的报纸。穆德之后被开除，她一怒之下抓起一个烫斗烫伤她男老板的手（他在几年间性侵了她和洗衣房的其他女工）。警察被叫来，督察斯蒂德（[[布兰顿·葛利森|布兰顿·葛利森]] 饰）让她离开，但前提是把其他成员的情报告诉他，但被穆德拒绝。

桑尼继续阻止穆德看她们的儿子，指出法律规定他有权这样做。这迫使穆德对修改法律以符合妇女权更激进。最终她得知桑尼由于一直受到社会的排斥，他不再感觉有能力照顾他们的孩子。他把他们的儿子乔治给别人收养。由于没有家庭关系，穆德变得愈发激进，制造邮箱爆炸，并切断电报线。之后，她和战友们炸掉了属于国会的一座空房子。在监狱里，穆德参与[[绝食|绝食]]，导致被残酷地灌食，这甚至让斯蒂德感到触目惊心。

然而，警方开始向报社施加压力，让他们放弃报道。妇女参政论-{}-者认为必须要发动更加剧烈的活动，以引起人们注意她们的目的。妇女们决定参加[[叶森打吡大赛|埃普瑟姆德比]]，届时英王[[乔治五世|乔治五世]]将出席赛事。到时她们站在镜头前，打出妇女参政运动的标语。当她们被挡在国王所在区域的外面时，艾米莉决定无论如何都要执行计划。尽管比赛异常激烈，艾米莉还是奋不顾身地冲入赛道，最终被马蹄踩死，这一幕被穆德亲眼目睹。后来穆德加入她的送葬队伍。影片结尾的滚动字幕透露英国妇女权利于1928年得到承认，之后还展示了其他国家妇女获得权利的日子。

== 制作 ==
=== 开发 ===
2011年4月，Film4 Productions、[[焦点影业|焦点影业]]和Ruby Films宣布正筹拍一部关于19世纪末20世纪初英国妇女参政运动的历史剧情片<ref name=Development>{{cite web |url=http://variety.com/2011/biz/news/film4-focus-develop-suffragettes-1118035050/ |title=Film4, Focus develop ‘Suffragettes’ |last=Dawtrey |first=Adam |date=2011-04-06 |work=Variety |accessdate=2014-02-26}}</ref>。{{le|阿比·摩根|Abi Morgan}}担任编剧，{{le|莎拉·加夫隆|Sarah Garvon}}执导<ref name="Development" />。2013年10月24日，据透露[[百代电影公司|百代]]已取代[[焦点影业|焦点影业]]，[[英国电影协会|英国电影协会]]电影基金（{{lang|en|BFI Film Fund}}）为电影提供资助<ref name=Carey>{{cite web |url=http://www.hollywoodreporter.com/news/carey-mulligans-suffragette-pathe-replaces-650662 |title=Pathe Replaces Focus Features International On Carey Mulligan's 'Suffragette' |last=Kemp |first=Stuart |date=2013-10-24 |work=The Hollywood Reporter |accessdate=2014-02-26}}</ref> 。

2014年10月，[[相对论传媒|相对论传媒]]获得影片北美独家影院发行权，而国际发行权则由百代获得。然而，2015年3月17日，在《[[万物理论_(电影)|万物理论]]》大获成功后，焦点影业接管电影北美发行权，他们这样做的主要原因是相对论已申请破产，所以焦点接管了美国的发行权<ref>{{cite web |url=http://www.comingsoon.net/movies/news/420493-focus-features-acquires-suffragette-starring-carey-mulligan-and-meryl-streep |title=Focus Features Acquires Suffragette, Starring Carey Mulligan and Meryl Streep |last=Evry |first=Max |date=2015-03-17 |work=ComingSoon.net |accessdate=2015-03-22}}</ref>。

== 发行 ==
[[百代电影公司|百代]]于2015年10月12日在英国和爱尔兰发行本片<ref>{{cite web |url=http://www.theguardian.com/film/2015/oct/07/suffragette-review-votes-women-carey-mulligan-abi-morgan-bonham-carter |title=Suffragette review – a valuable, vital film about how human rights are won |last=Bradshaw |first=Peter |date=2015-10-07 |work=The Guardian |accessdate=2015-10-15}}</ref>。

2015年3月27日，[[相对论传媒|相对论传媒]]原本打算只在美国发行电影，由于相对论申请破产，制片方已经把英国放入市场，新的发行方还不得而知。之后[[焦点影业|焦点影业]]于2015年3月17日经相对论从百代手中收购了电影的北美发行权，宣布影片于2015年10月23日在美国有限发行<ref>{{cite web |url=http://deadline.com/2015/03/meryl-streeps-suffragette-gets-fall-release-date-1201400063/ |title=Meryl Streep’s ‘Suffragette’ Gets Fall Release Date |last=Pedersen |first=Erik |date=2015-03-27 |work=Deadline |accessdate=2015-04-06}}</ref>。

2015年6月，电影被宣布为[[伦敦影展|伦敦影展]]开幕影片，于2015年10月7日欧洲首映。伦敦影展主任克莱尔·斯图尔特（{{lang|en|Clare Stewart}}）表示莎拉·加夫隆的故事片是一部“紧迫且令人信服的、由英国妇女制作的、关于改变历史进程的英国妇女的电影<ref>{{cite web |url=http://whatsworthseeing.com/suffragette-to-open-london-film-festival-2015/ |title= What's Worth Seeing: Suffragette to open London Film Festival 2015 |last=Korsner |first=Jason |date=2015-06-03 |work=What’s Worth Seeing… |accessdate=2015-06-03}}</ref>。”2015年9月4日，影片在{{le|特柳赖德电影节|Telluride Film Festival}}首映。

{{le|姐妹不删剪|Sisters Uncut}}组织在伦敦首映礼上示威，抗议删掉家庭暴力机构的镜头<ref>{{cite web|author=Keiligh Baker|url=http://www.dailymail.co.uk/news/article-3263943/Suffragette-red-carpet-invaded-campaigners-against-domestic-abuse.html|title=Suffragette premiere is invaded by domestic violence campaigners|date=2015-10-07|accessdate=2015-10-08|work=Daily Mail}}</ref>。海伦娜·博纳姆-卡特称该镜头“完美。如果你有种强烈的感觉，那时你可以把不公正说出来，尝试改变事情。”凯瑞·穆里根表示抗议活动“非常棒”，但她因错过活动而伤心<ref>{{cite web|author=Gander, Kashmira & Townsend, Megan|url=http://www.independent.co.uk/arts-entertainment/films/suffragette-premiere-protesters-lie-on-red-carpet-and-let-off-purple-and-green-smoke-in-a6685236.html|title=Suffragette premiere: Protesters lie on red carpet in demonstration against cuts to domestic violence services|date=2015-10-07|accessdate=2015-10-08|work=The Independent}}</ref>。

== 评价 ==
=== 票房 ===
2016年3月7日，影片北美票房470万美元，其他地区票房2530万美元，全球总票房3000万美元，而预算为1400万美元<ref name="BOM"/>。

=== 专业评价 ===
影片获得普遍积极的评价。[[烂番茄|烂番茄]]根据131条评论给影片72%的“新鲜度”，平均得分6.6（满分10）。网站公示指出：“《妇女参政论者》戏剧化了一个重要且至今仍满是痛楚的真实改编故事，足本的工序和诚意足以盖过其缺点<ref name=rotSF>{{cite web  |url=http://www.rottentomatoes.com/m/suffragette/ |title= Suffragette (2015) |work=Rotten Tomatoes |accessdate=2015-11-12}}</ref>。”[[Metacritic|Metacritic]]根据37条评论给予影片67分（满分100分），表示“普遍好评”<ref>{{cite web |url=http://www.metacritic.com/movie/suffragette |title=Suffragette |work=Metacritic |accessdate=2015-11-12}}</ref>。演员受到突出的表扬，尤其是凯瑞·穆里根和海伦娜·博纳姆·卡特。尽管梅丽尔·斯特里普的短暂亮相受到赞扬，但也有人批评她在宣传中的显著地位误导群众。

=== 无种族反映 ===
上映前夕，影片被指对种族麻木不仁<ref>{{cite web |url=http://www.theguardian.com/film/2015/oct/05/suffragette-film-publicity-campaign-erasure-feminism |title=Suffragette |work=Guardian |accessdate=2015-11-21}}</ref>。另外，根据美国的情况，影片的其中一场宣传活动中，四位白人女主角所穿的写着出自埃米琳·潘克斯特演讲的“宁反叛，不奴隶”（{{lang|en|I'd rather be a rebel than a slave}}）字样的白色衬衫，放到现在就是种族上的不敏感<ref>{{cite web|url=http://www.theguardian.com/film/2015/oct/05/suffragette-film-publicity-campaign-erasure-feminism|title=Suffragette's publicity campaign and the politics of erasure|author=Rebecca Carroll|work=the Guardian}}</ref><ref>{{cite web|url=http://www.huffingtonpost.com/entry/ava-duvernay-on-why-we-cant-write-off-criticism-of-that-suffragette-shoot_5615264ee4b021e856d2edf1|title=Ava DuVernay On Why We Can't Write Off Criticism Of That 'Suffragette' Shoot|date=2015-10-07|work=The Huffington Post}}</ref>。

2015年11月17日在英国电影电视艺术学院放映电影时，该片的编剧阿比·摩根解释称，由于英国1911年到1913年非欧洲移民水平较低，有色妇女参政论-{}-者很少，那其中有几个，如印度公主{{le|索菲亚·杜雷普·辛格|Sophia Duleep Singh}}，是出自上层阶级的妇女，在电影背景所处的1912年伦敦时并未搬到工人阶级的圈子。影片聚焦于小型的妇女代表团体，目的是描绘当时特定的政治背景，展现失去孩子或生计的女性的情绪影响，这是权力剥夺法律所造成后果的例子。历史学家宝拉·巴特利（{{lang|en|Paula Bartley}}）博士专注于历史上的女性和妇女参政运动，曾撰写埃米琳·潘克斯特的传记。她向《新政治家》（{{lang|en|''New Statesman''}}）证实影片对种族的描述从历史角度来看是准确的：“（1911-1913年的）英国是一个白人至上的社会，妇女参政运动可以反映这一点。”巴特利博士强调，英国妇女参政运动“跟美国、澳大利亚和新西兰的情况不同，因为纵使那时英国有少数族裔，公民身份的规模和问题都跟其他国家不一样<ref>{{cite web|url=http://www.newstatesman.com/politics/feminism/2015/10/what-did-suffragette-movement-britain-really-look|title=What did the suffragette movement in Britain really look like?|publisher=}}</ref>。”
