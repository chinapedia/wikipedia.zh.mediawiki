'''現金流量表'''（[[GAAP|GAAP]]舊制：{{lang|en|[[:en:Cash_Flow_Statement|Cash Flow Statement]]}}；[[IFRS|IFRS]]制：{{lang|en|[[:en:Statement_of_Cash_Flows|Statement of Cash Flows]]}}）是[[財務報表|財務報表]]的[[財務三表|三個基本報告]]之一，所表達的是在一固定期間（通常是每月或每季）內，一家[[機構|機構]]的[[現金|現金]]（包含[[銀行存款|銀行存款]]）的增減變動情形。現金流量表的出現，主要是要反映出[[資產負債表|資產負債表]]中各個項目對現金流量的影響，並根據其用途劃分為[[經營|經營]]、[[投資|投資]]及[[融資|融資]]三個活動分類。現金流量表可用於分析一家機構在短期內有沒有足夠現金去應付開銷。[[國際財務報告準則|國際財務報告準則]]第7號公報規範現金流量表的編製。

'''現金流量表'''是一份顯示於指定時期（一般為一個月，一季。主要是一年的年報）的現金流入和流出的[[財務報表|財務報表]]。這份報告顯示[[資產負債表|資產負債表]]（Balance Sheet）及[[損益表|損益表]]（Income Statement / Profit and Loss Account）如何影響現金和-{zh-hans:现金等价物;zh-hant:等同現金;}-，以及根據公司的經營，投資和融資角度作出分析。作為一個财务分析工具，'''現金流量表'''的主要作用是決定公司短期生存能力，特別是繳付帳單的能力。

過去的企業經營都強調「資產負債表」與「損益表」兩大表，隨著企業經營的擴展與複雜化，對財務資訊的需求日見增長，更因許多[[企業|企業]]經營的中斷肇因於資金的週轉問題，漸漸地，報導企業資金動向的現金流量表也獲得許多企業經營者的重視，將之列為必備的財務報表。

對現金流量表有興趣的人士包括：
* 需要知道公司能否繳費薪金及即時支出的會計員工。
* [[出借人|出借人]]（[[債權人|債權人]]）想知道公司償還能力的人。
* 想判斷公司是否財政穩建的[[投資者|投資者]]。
* 需知道公司能否給予賠償的僱員或立契約者。

對初成立而且持有有限速動資產的公司來說，現金流量表是非常重要。即使應收帳餘額與長期財政穩健有關，但在驚人的資金短缺下，這些公司通常都是容易受傷的。

== 營業活動 ==
'''營業活動'''包括[[生產|生產]]、[[銷售|銷售]]、[[運送|運送]]公司的產品，以及從顧客中收取款項。例如；購入原料、建立存貨、廣告及運送貨物。

一、因營業活動發生者（Operating activities）：
流入：主要項目為現銷與帳款、票據收現
流出：主要項目為現購商品與帳款、票據付現，支付各項費用、稅捐等

以下項目需當作營運活動：
* [[純利|-{zh-hans:净收益;zh-hant:純利;}-]]（在[[損益表|損益表]]中找到）
* [[折舊|折舊]]
* [[非現金物品|非現金物品]]（例如；[[呆壞帳撥備|呆壞帳撥備]]）
* [[遞延稅項|遞延稅項]]
* [[利息攤銷|利息攤銷]]
* 應計物品，如應付薪金。

== 投資活動 ==
'''投資活動'''針對公司因生製及售賣貨物而購買的[[固定資產|固定資產]]，以及變賣任何公司不再需要的[[固定資產|固定資產]]。

二、因投資活動發生（Investing activities）
流入：出售資產、收回貸款、債券，出售權益證券等
流出：購買資產、放款、購買債券、權益證券等

投資活動包括：
* [[資本支出|資本支出]]，包括以賒帳方式購買機器。
* [[投資|投資]]

== 融资活動 ==
'''融资活動'''包括從投資者如[[銀行|銀行]]及[[股東|股東]]投入的現金，以及當企業把收入回饋投資者的現金流出。其他影響企業長期負債及股本的活動亦列為融資活動。

三、因融資活動發生者（Financing activities）
流入：現金增資，舉債
流出：支付股利，償還借款

融資活動包括以下物品：
* 已附[[股息|股息]]
* 賣出已購買的[[股票|股票]]
* 淨借入

== 流量表的準備 ==
=== 直接法 ===
此法按照主要類別（运营、投资和融资）列示整體現金收入及支出。顺序则为從銷售數字開始，並以損益表和流動資產及流動負債為借鑒。根据《国际财务报告准则第7号》，获得的股息可以被列入运营所得，或列入投资所得。如果所缴税收直接与运营相关，列入运营；同理如果与投资或融资相关，则列入投资或融资所得。美国一般公認會計原則中，股息分红则应被列入运营所得，而非投资所得。

=== 間接法 ===
間接法是先顯示純利／虧損，然後調整所有非現金項目。
