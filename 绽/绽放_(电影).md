《'''绽放'''》（{{lang-en|'''Blossom'''}}）是中国大陆拍摄青春励志片。导演吴祖云，主演姚星彤、顾琴烨。该片是“2008中国残奥会”的[[献礼影片|献礼影片]]。<ref>{{cite web|url=http://www.sn.xinhuanet.com/2008-09/16/content_14408045.htm|title=电影《绽放》为残奥会献礼|accessdate=2008|publisher=新华网|work=|language=zh-hans|deadurl=yes|archiveurl=https://web.archive.org/web/20150924103021/http://www.sn.xinhuanet.com/2008-09/16/content_14408045.htm|archivedate=2015-09-24}}</ref><ref>{{cite web |url= http://www.chinanews.com/olympic/news/2008/09-09/1376458.shtml|title=影片《绽放》演员姚星彤：扮演盲女让我更坚强 |accessdate=2008 |publisher=中新网 |work= |language=zh-hans}}</ref>该片的原型人物是浙江省残疾人艺术团演员陈敏。<ref>{{cite web |url=http://ent.sina.com.cn/m/c/2008-05-26/14352037604.shtml |title= 姚星彤谈《绽放》：盲人的心里有双明亮的眼睛|accessdate=2008 |publisher=新浪网 |work= |language=zh-hans}}</ref>

==剧情==
该片讲述了中国大陆正常人和残疾人互帮互助、互敬互爱的感人故事。<ref>{{cite web |url=http://gb.cri.cn/17844/2008/09/09/1425s2232987.htm |title= 影片《绽放》为残奥会献礼 讲述盲女钢琴手故事|accessdate=2008 |publisher=国际在线 |work= |language=zh-hans}}</ref>
