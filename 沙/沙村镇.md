'''沙村镇'''是{{PRC admin/fullname|36/08/26/109/000|3}}下辖的一个[[行政建制镇|镇]]<ref>{{PRC admin/ref sgc|{{PRC admin/parent|36/08/26/109/000}}}}</ref>，位于泰和县东南部，是河东的交通枢纽、物资集散地。全镇面积79[[平方公里|平方公里]]，常住人口约1.3万。

== 交通 ==

沙村镇位于泰和县河东部分中部，地处交通要冲。在沙村交汇的[[319国道|319国道]]、沙村-小龙公路、沙村-水槎公路是泰和东南边沿几个乡镇主要的对外通道；[[京九铁路|京九铁路]]自东南向西北穿境而过，在境内设有[[沙村站|沙村火车站]]。

== 地理 ==

沙村位于山地与低丘结合部，地势自东南的山地向西北逐次降低。[[云亭河|云亭河]]自东南经[[老营盘水库|老营盘水库]]入境，在沙村圩与自南而来的[[东沔河|东沔河]]汇合，穿越西部平原，向西北流入[[冠朝镇|冠朝]]。

== 沿革 ==

现沙村镇辖境在1950年时属沙村、沙市、高陇等乡，1956年合并为高陇乡。1958年改为高陇人民公社，1983年更名为沙村人民公社。1984年撤消人民公社，改设沙村乡<ref>李宝库主编：《中国政区大典》第2册，杭州：浙江人民出版社，1999年，ISBN 7-213-01789-6，第1185页</ref>。1997年6月5日改设沙村镇<ref name="jaxz">〈[http://www.jxqhdm.com/NEWS/newshow.aspx?ID=2200 吉安市乡镇行政区划变更]{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}〉，江西区划地名网，2007-10-16，于2009年4月26日查阅</ref>。

== 行政区划 ==

沙村镇现辖1个[[居委会|居委会]]和8个[[村委会|村委会]]：沙村镇居委会、坪洲村、高陇村、凰冈村、绵溪村、新圩村、沙村村、兴华村、良村，镇政府驻沙村圩<ref>〈[http://www.jxqhdm.com/index-ja.aspx 2009年泰和县行政区划] {{webarchive|url=https://web.archive.org/web/20091025153207/http://www.jxqhdm.com/index-ja.aspx |date=2009-10-25 }}〉，吉安区划地名网（江西区划地名网子网站），于2009年4月1日查阅。</ref>。
