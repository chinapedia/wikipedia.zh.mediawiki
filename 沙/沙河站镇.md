'''沙河站镇'''，是{{PRC admin/fullname|37/09/23/101/000|3}}下辖的一个[[乡镇|乡镇]]级行政单位<ref>{{cite web|url=http://www.stats.gov.cn/tjsj/tjbz/tjyqhdmhcxhfdm/2017/{{#switch:{{PRC admin/level|{{PRC admin/parent|37/09/23/101/000}}}}{{Dead link|date=2018年12月 |bot=InternetArchiveBot |fix-attempted=yes }}|0=index|1={{PRC admin/stripcode|{{PRC admin/parent|37/09/23/101/000}}}}|#default={{PRC admin/stripcode|{{PRC admin/parent|{{PRC admin/parent|37/09/23/101/000}}}}}}/{{PRC admin/stripslashes|{{PRC admin/stripcode|{{PRC admin/parent|37/09/23/101/000}}}}}}}}.html|title=2017年统计用区划代码和城乡划分代码：{{PRC admin/plainname|{{PRC admin/parent|37/09/23/101/000}}}}|publisher=[[中华人民共和国国家统计局|中华人民共和国国家统计局]]|year=2017}}</ref>。该乡镇位于东平县南端，人口约5.8万人。

== 历史 ==
自汉代开始。今沙河站镇大多数时间属于东平管辖。北宋咸平三年，官府在今沙河站村一带设置[[驛站|驿站]]，由于驿站附近有小沙河，此驿站就被称为沙河站，后来此地逐渐形成村落和集镇<ref name="官网沙河站">{{Cite web|url=http://www.dongping.gov.cn/xq/xzjd/201801/t20180129_776983.html|title=中国·东平--山东省东平县人民政府门户网站|accessdate=2018-12-13|author=沙河站镇人民政府|date=2018-01-29|work=东平县县情—乡镇街道|publisher=东平县人民政府网|archive-url=https://web.archive.org/web/20180201032354/http://www.dongping.gov.cn/xq/xzjd/201801/t20180129_776983.html|archive-date=2018-02-01|dead-url=no}}</ref>。清代，今沙河站镇属[[东平州|东平州]]，境域分属崇礼保、学礼保、向礼保、尚礼保和兴仁保<ref name="年鉴1993历史">{{Cite web|url=http://lib.sdsqw.cn/bin/mse.exe?seachword=&K=c94&A=1&rec=119&run=13|title=《东平年鉴》（1986-1993）,乡镇概况-沙河站镇-历史要述|accessdate=|author=东平县史志办公室|date=|publisher=}}</ref>。

1938年日军攻下东平，不久后在三官庙村、二十里铺村和沙河站村设立据点。1939年，中共东平六区区委和抗日区公所成立，以与日本控制下的东平六区政府分庭抗礼。1942年2月24日，日军包围[[东平县政府#抗日民主政府时期|东平抗日民主政府]]的临时驻地李圈村，八路军被迫突围，时任东平县抗日民主政府县长[[崔宜平|崔宜平]]重伤自杀。1943年1月，八路军昆张支队袭击了日军位于三官庙的据点，年末，东平六区抗日区队成立。至1944年，今沙河站镇西部仍由日本军队占据，东部则为中国共产党军队游击区。1945年5月[[东平战役|东平战役]]开始，沙河站境内的日军及亲附日军的军队撤往[[州城|州城]]，东平六区解放<ref name="年鉴1993历史"/>。

1953年8月，东平六区改成东平八区，1955年，改以行政驻地小杨庄命名，次年末小杨庄区驻地迁往沙河站村，小杨庄区因此改名为沙河站区。1958年2月撤区建乡，沙河站区撤销，成立沙河站乡和三官庙乡，同年10月两乡有合并为沙河站人民公社。1959年10月至1962年1月东平县撤销期间先后隶属[[汶上县|汶上县]]和[[梁山县|梁山县]]，东平县恢复后重归东平县。1964年7月，管辖今新湖镇的湖区人民公社建立，湖区公社机关暂时驻扎在沙河站公社的李庄。1984年4月撤销沙河站人民公社建立沙河站区，下辖沙河站乡、纸坊乡<ref group="注">为了与当时的州城镇纸坊乡（孙纸坊乡）区别，该乡多被称为李纸坊乡。</ref>、乔村乡、三官庙乡，同时今新湖镇的前身湖区人民公社撤销，建立[[新湖镇_(东平县)|湖区区]]，暂驻在沙河站区长达二十年的湖区领导机关离开沙河站下辖的李庄迁往唐营。1985年12月，东平县撤区，调整行政区划，沙河站镇建立<ref name="年鉴1993领导人更迭">{{Cite web|url=http://lib.sdsqw.cn/bin/mse.exe?seachword=&K=c94&A=1&rec=117&run=13|title=《东平年鉴》（1986-1993），乡镇概况-沙河站镇-领导人更迭表|accessdate=|author=东平县史志办公室|date=|publisher=}}</ref><ref name="年鉴1993历史"/>。

== 行政区划 ==

沙河站镇辖5个管理区，辖83个自然村，65个行政村，镇政府驻地在沙南村。沙河站镇所辖管理区和行政村如下（括号中为自然村）<ref>{{Cite web|url=http://www.stats.gov.cn/tjsj/tjbz/tjyqhdmhcxhfdm/2017/37/09/23/370923101.html|title=2017年统计用区划代码河城乡分类代码：沙河站镇|accessdate=2018-12-23|author=中国国家统计局|date=2017|publisher=}}</ref>：

:'''沙河站管理区'''： 沙南、沙北、冯洼、中李庄、黄路沟、于楼、丰庄、王堂、辛庄、老庄、北场、曾庙（含唐庄）、小王营、大王营、贾海、董堂。

:'''乔村管理区'''：乔村、马河、东杨楼、西杨楼（含马村）、双楼、熊村、吴桃园、徐楼、常庙、杨营、大巴柳、东巴柳（含西巴柳）、张庄、屈楼（含庙庄）、洪福寺（含赵阁）。

:'''三官庙管理区'''：李村、三官庙南、三官庙西、三官庙北、孟庄、八角堂、韩村（含蔡庄、解庄）、蔡庄店、樱桃园（含前、后樱桃园、月河子）、华堂、丁堂、林海、西张圈（含吕庄、十里铺）、西李庄、二郎庙（含杨庄）。

:'''杨庄管理区'''：前河涯、小杨庄、大杨庄、王庄、侯圈、韩圈（含王海子、何圈）、李圈。

:'''纸坊管理区'''：大刘庄、廖村（含桑村）、董寨、何村、孙楼（含崔村）、张村（含陆庙）、东李庄（含罗曼）、纸坊、孙村、芦村、柏村集、尹村。

== 地理 ==
沙河站镇位于东平县南部边沿，位于北纬35°47′~35°56′，东经116°19′~116°24′之间，南北长10千米，东西长13.31千米，面积69.2平方公里，其疆土大致呈现向南凹陷的新月形{{r|年鉴1993地形概貌}}。其行政驻地沙河站村位于沙河站镇南端，紧邻汶上县，距离东平县城18公里。沙河站镇与其他乡镇大致以河流和堤坝为界。小清河位于沙河站镇北部，为沙河站镇与彭集街道和州城街道的界河；小汶河位于沙河站镇东部，为沙河站镇与汶上县的界河；在西侧，沙河站镇与新湖镇大致以[[东平二级湖|东平二级湖]]湖堤为界；在南侧，沙河站镇与汶上县直接接壤{{r|年鉴1993地形概貌}}。

沙河站镇位于黄河、[[大汶河|汶河]]冲积平原之上，整体地势平坦，其中地面最高处海拔49.1米，最低处海拔39.6米，起伏不到十米。地形以平原为主，占85%，也分布有少量洼地，占总面积的15%{{r|年鉴1993地形概貌}}。土壤以中壤、轻壤为主，较为肥沃。主要的矿产资源有铁矿、煤炭和河砂<ref name="官网沙河站" />{{r|年鉴1993地形概貌}}。沙河站镇气候温和，四季分明，属[[温带季风气候|温带季风性大陆气候]]，平均年降水量659.2毫米，平均气温13.3摄氏度<ref name="年鉴1993地形概貌">{{Cite web|url=http://lib.sdsqw.cn/bin/mse.exe?seachword=&K=c94&A=1&rec=118&run=13|title=《东平年鉴》（1986-1993）,乡镇概况-沙河站镇-地形概貌|accessdate=|author=东平县史志办公室|date=|publisher=}}</ref>。

== 人口 ==

2010年[[中国第六次人口普查|中国第六次人口普查]]时，沙河站镇共有人口54614人。共有家庭14171户，平均每户3.81人。14岁以下的少年儿童共8316人，占总人口15.22%；15-64岁人口共40625人，占总人口74.38%；65岁及以上的老年人共5673人，占总人口的10.38%。男性共计27382人，占总人口50.13%；女性共计27232，占总人口49.86%。本地居住的人口中，拥有本地户籍的人口为54207人，占99.25%<ref name="人口普查2010">{{cite web| url =http://www.yearbookchina.com/navipage-n2013030152000017.html| title =《中国2010年人口普查分乡、镇、街道资料》| author =中国国务院人口普查办公室、中国国家统计局人口就业统计司等| year =2010| website =统计年鉴分享平台| page =表15 山东省乡、镇、街道人口| format =收费| access-date =2018-07-19| archive-url =https://web.archive.org/web/20180719084356/http://www.yearbookchina.com/navipage-n2013030152000017.html| archive-date =2018-07-19| dead-url =no}}</ref>。
