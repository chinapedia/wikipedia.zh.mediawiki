'''沙没巴干府'''<ref name="zhou">{{cite book|title=[[世界地名翻译大辞典|世界地名翻译大辞典]]|author=周定国|publisher=中国对外翻译出版公司|year=2008|isbn=978-7-5001-0753-8|page=1234}}</ref>（{{thai|t=จังหวัดสมุทรปราการ|r=Changwat Samut Prakan}}，{{IPA-th|t͡ɕāŋ.wàt sāmùt prāːkāːn}}）是位於[[泰國|泰國]]中部的一個府。[[蘇凡納布機場|國家機場]]位於此地。

該府泰語別名'''巴南'''（{{thai|t=ปากน้ำ|r=Paknam}}），意思是「河口」。當地華人稱其為'''北欖府'''，是該名字的潮州話音譯（[[潮州話|潮州話]]：{{IPA|[p˭ǎ[i] nám]}}）。越南人稱其為'''北喃'''。

== 歷史 ==
該府建於[[大城王國|阿瑜陀耶時代]]，因恰好位於[[泰國灣|泰國灣]]的[[昭披耶河|昭披耶河]]口，因此[[泰文|泰文]]為邁巴喃（[[中文|中文]]意義為“河口城”）。在1619年，[[拉瑪二世|拉瑪二世]]下令重建城池，並在昭披耶河兩旁建築數座[[碉堡|碉堡]]，以抵禦[[歐洲|歐洲]]殖民侵略者的襲擊，現在僅餘兩座[[碉堡|碉堡]]。

== 地理 ==
該府的面積約1,004.1平方公里，距[[首都|首都]][[曼谷|曼谷]]約25公里。其疆域北連[[曼谷|曼谷]]；南毗[[泰國灣|泰國灣]]；東瀕[[差春騷府|差春騷府]]；西臨[[曼谷|曼谷]]及[[沙沒沙空府|沙沒沙空府]]。該府大部分屬低窪地帶，當地民眾主要靠打漁維生。目前該府已有不少工業區，大工廠林立。

== 行政及管理 ==
沙没巴干府行政上被劃分為6個縣（Amphoe），又進一步被劃分為50個區（Tambon）及396個村（Muban）。

== 名勝古蹟 ==
[[File:_Erawanmuseum2003.jpg|300px]]
* [[泰國北欖鱷魚湖動物園|泰國北欖鱷魚湖動物園]]（ฟาร์มจระเข้และสวนสัตว์，Samut Prakan Crocodile Farm and Zoo）：1950年開設，距[[首都|首都]][[曼谷|曼谷]]約10公里。
* [[古蹟城|古蹟城]]（เมืองโบราณ，Ancient City）：它是[[世界|世界]]最大的戶外[[博物館|博物館]]，佔地高達800,000平方公尺。位於[[素坤逸路|素坤逸路]]第33公里處，仿製[[泰國|泰國]]地形把全國七十六府的知名建築物、[[紀念碑|紀念碑]]及古剎廟宇等複制塑造，成為[[泰國|泰國]]的縮影。
* 拍沙目佛塔（พระสมุทรเจดีย์，Phra Samut Chedi）：位於府署對面，又稱為水中大塔。建於[[拉瑪二世|拉瑪二世]]時代，每年[[佛曆|佛曆]]11月均舉行禮佛盛會，吸引各地的觀光客。
* 幫拍朱壯告堡（ป้อมพระจุลจอมเกล้าหรือป้อมพระจุล，Fort Chulachomklao）：位於直轄縣建於[[拉瑪五世|拉瑪五世]]時代的軍事[[碉堡|碉堡]]，迄今保留當年的[[望遠鏡|望遠鏡]]及槍炮，由[[泰國|泰國]][[海軍|海軍]]負責管理。
* 壽苑：位於直轄縣的挽蒲區之中式園林食堂，園裡繁花似錦，風景優美，且有餐廳及其它娛樂場所，是一休閒好去處。
* 哇阿鎖咖喃寺（วัดอโศการาม，Wat Asokaram）：位於直轄縣，寺裡有十三座佛塔，中央的最大；其餘12座小佛塔則環繞四周。
* 新城宋干節：新城縣每年皆舉行盛會，有放生隊伍遊行、花車比賽、潑水、宋干美女選舉等節目，吸引各地的觀光客。
* 海軍博物館（พิพิธภัณฑ์ทหารเรือ，Naval Museum）：位於直轄縣之水師學堂對面，除介紹[[泰國|泰國]][[海軍|海軍]]的[[歷史|歷史]]之圖文展覽外，尚有各種軍民用船艦的模型。
* 沙旺卡尼挖園：距直轄縣約五公里，位於[[素坤逸|素坤逸]]路第31公里處，它是[[泰國|泰國]][[紅十字會|紅十字會]]轄下的[[療養院|療養院]]，遍布各種鮮花樹木，設有[[水族館|水族館]]及[[餐廳|餐廳]]，院後有河邊亭閣，為一避暑勝地。
* 眼鏡蛇園：位於挽披縣，面積約16,000平方公尺，養殖各種[[毒蛇|毒蛇]]，示範擠毒液及武蛇表演，並銷售各種蛇皮制品。
* [[暹羅代天宮|暹羅代天宮]]：位於直轄縣，建於1992年，它是[[中華民國|中華民國]]的信眾為供奉代天巡狩的五府千歲：五皇爺，香火鼎盛。
