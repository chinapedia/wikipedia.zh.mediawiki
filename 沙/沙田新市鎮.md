'''沙田新市鎮'''（{{lang-en|Sha Tin New Town}}）是[[香港新市鎮|香港新市鎮]]之一，位於[[香港|香港]][[新界|新界]][[沙田區|沙田區]]。自1980年代起發展帶擴展至[[馬鞍山_(香港市鎮)|馬鞍山]]。

==地理==
<!-- 注释出：[[File:Sha_Tin_Hoi_沙田海,_Ma_On_Shan_馬鞍山,_1950s.jpg|左]]和[[馬鞍山_(香港山峰)|馬鞍山]]1950年代的原貌，現在的城門河口因填河而收窄]] -->
沙田新市鎮主要位於[[沙田海|沙田海]]的[[填海|填海]]地，包括[[沙田市中心|沙田市中心]]、[[瀝源邨|瀝源邨]]、[[禾輋邨|禾輋邨]]、[[沙角邨|沙角邨]]、[[乙明邨|乙明邨]]、[[圓洲角|圓洲角]]、[[石門_(香港)|石門]]、[[馬鞍山_(香港市鎮)|馬鞍山]]北面等地。[[紅梅谷|紅梅谷]]、[[九肚山|九肚山]]、[[小瀝源|小瀝源]]、馬鞍山南面等是[[山地|山地]]，[[大圍|大圍]]及[[火炭|火炭]]等地則是由[[農田|農田]]開發而來。[[荃灣新市鎮|荃灣新市鎮]]是在沙田的西面，[[大埔新市鎮|大埔新市鎮]]則是在沙田的北方，南面則為[[九龍群峰|九龍群峰]]。沙田新市鎮最繁榮的地方是[[沙田市中心|沙田市中心]]。

由於沙田海被填海，原來的[[城門河|城門河]]被延長，沿填海地上的人工河道出海。

== 歷史 ==
[[File:HKHistory_ShaTinNewTown1.jpg|thumb]]鳥瞰的沙田新市鎮歷史舊貌(1983年)]]1972年10月當時的[[港督會同行政局|港督會同行政局]]批准一個大型的建屋計劃，旨在於1980年代中期提供足夠房屋，供180萬人居住。按照計劃，這些新建房屋超過一半在[[新市鎮|新市鎮]]興建，而[[沙田|沙田]]、[[屯門|屯門]]和[[荃灣|荃灣]]正是由政府指定發展的第一代新市鎮。

沙田第一份規劃圖則於1961年由[[城市規劃委員會|城市規劃委員會]]制定，其後經當時的港督會同行政局批准。根據這份圖則，沙田的人口可達約360,000人，而人口密度最高為每公頃750人。雖然圖則建議把少量土地劃為工業用地，但當時沙田只視作[[九龍|九龍]]一個[[近郊|近郊]]住宅區。

為了紓緩市區的擠逼情況，當局需要在沙田興建大量[[徙置屋邨|徙置屋邨]]。因此當局檢討1961年的圖則，以容許在沙田進行較高密度的住宅和工業發展。經過進一步的規劃和工程研究後，當局制定了一份修正分區計劃大綱草圖。在1965年當時的[[工務局|工務局]]擬備一個綜合發展計劃，建議在沙田發展一個新市鎮，人口約為100萬。根據這個計劃而擬備的分區計劃大綱草圖最後於1967年獲得通過。於1970年代初，當局擬備一份經修訂的發展計劃草圖，把沙田發展完成時的人口定為50萬人。沙田於1970年代初期開始大規模[[填海|填海]]，人口不斷增加。[[File:Hong_Kong_reclamation.png|thumb]]1979年政府批准發展馬鞍山，作為'''沙田新市鎮的擴展部分'''。1983年馬鞍山運輸研究獲得通過。該項研究建議，沙田及馬鞍山的整體人口界限共704,000人，而馬鞍山的人口界限為15萬人。1986年馬鞍山發展檢討完成後，當局批准該次檢討的建議，同意沙田新市鎮的人口總數應以750,000人為限。1991年3月22日一份為馬鞍山而擬備的分區計劃大綱圖首次於[[香港政府憲報|憲報]]公布。沙田及馬鞍山的分區計劃大綱圖曾經數次修訂，以納入最新的建議，而住宅、鄉村式發展、商業、工業、休憩用地、政府、機構或社區、綠化地帶和其他指定用途等主要的土地用途，以及運輸系統已納入上述分區計劃大綱圖內。

== 規劃概念 ==
[[File:HK_ShingMunRiverview.jpg|thumb]]
沙田新市鎮屬帶型發展，以[[城門河|城門河]]的天然河谷一帶為重心，發展一個均衡和自給自足的社區，使區內居民可以在一個舒適愉快的環境中工作、休憩、成長和學習。新市鎮的規劃大部分受到地形和鐵路線的限制，在規劃不同的土地用途時，當局考慮把河谷及填海區劃作較高密度的住宅和工業用途，而地勢較高的地方則劃作較低密度的住宅用途。

== 人口及房屋組合 ==
[[File:Tung_Lo_Wan_2016.jpg|thumb]]
沙田區內房屋種類眾多，適合不同收入水平和要求的人士居住。在全面發展後，新市鎮的公共租住房屋、資助出售房屋、私人永久性房屋（包括鄉村房屋）及其他私人臨時房屋及非住宅用房屋的整體房屋組合比率將會是 27：34：36：3，而新市鎮的人口約為 730,000人。

== 社區設施 ==
根據《香港規劃標準與準則》，在新市鎮內預留足夠土地，發展教育、社區和康樂設施，為居民服務。

新市鎮的專上院校包括[[香港中文大學|香港中文大學]]、[[香港恒生大學|香港-{恒}-生大學]]和[[香港專業教育學院|香港專業教育學院]]（沙田分校）。[[香港體育學院|香港體育學院]]則提供另一類型的專上教育。

[[威爾斯親王醫院|威爾斯親王醫院]]於1982年正式啟用，是一家分區醫院。其他的輔助醫院服務則由位於[[亞公角|亞公角]]的[[沙田醫院|沙田醫院]]和[[沙田慈氏護養院|沙田慈氏護養院]]（補助醫院）提供。沙田另外有一間私人醫院（[[仁安醫院|仁安醫院]]）。新市鎮亦設有四間普通科診療所/健康中心，為區內居民服務。

== 康樂及休憩用地 ==
[[城門河|城門河]]是沙田休憩用地的中樞，兩岸設有公園、長廊、單車徑和其他康樂設施。佔地九公頃的[[沙田公園|沙田公園]]，則是休憩用地的中心。馬鞍山烏溪沙西面海旁的休憩用地已發展為[[馬鞍山公園_(香港)|馬鞍山公園]]。當局亦興建了一條附設單車徑的海濱長廊，由南面的[[大水坑|大水坑]]通往北面的馬鞍山公園。

當局亦已規劃及提供康樂設施，這些設施如[[沙田運動場|沙田運動場]]，多是位於城門河兩岸的。[[烏溪沙青年新村|烏溪沙青年新村]]是馬鞍山著名的康樂區之一。新市鎮多個地點均設有游泳池場館、網球場、壁球場和室內康樂中心，供居民使用。

== 市中心 ==
{{main|沙田市中心}}

[[港鐵|港鐵]][[東鐵綫|東鐵綫]][[沙田站|沙田站]]附近一帶已發展為沙田市中心，作為舊墟鎮的擴展區。這處亦為新市鎮提供一個購物中心和主要的公共交通交匯處。文娛館與商業樓宇及沙田車站相連，是沙田市中心的焦點。文娛館提供全面的設施，包括[[沙田公共圖書館|圖書館]]和[[沙田大會堂|大會堂]]。[[沙田法院|沙田法院]]位於文娛館旁。[[沙田公園|沙田公園]]位於文娛館與[[城門河|城門河]]之間，是市中心的一項主要發展。沙田公園是城門河兩岸休憩用地的中心。市中心亦設有商店、辦公室、電影院和其他商業及康樂設施。[[File:Ma_On_Shan_200905.jpg|thumb]]{{main|馬鞍山市中心}}

馬鞍山的市中心包括一個商業中心區，為住宅發展所圍繞，外層為政府及社區設施用地、馬鞍山公園和吐露港長廊。

== 工業 ==
[[File:Tai_Wai_Overview_201602.jpg|thumb]]
工業分布於四個主要地區，即[[大圍|大圍]]、[[火炭|火炭]]、[[石門_(香港)|石門]]及[[圓洲角|圓洲角]]。當局並為這些工業區提供方便的交通設施，多以快速公路連接主要幹線，減少對市民的影響。沙田新市鎮的工業以製造業、貨倉和貯物為主。

== 運輸 ==
=== 公路、隧道及道路 ===
[[File:Tai_Po_Road_Shatin_Section_201403.jpg|thumb]]
新市鎮發展初期，連接沙田與[[九龍|九龍]]的道路交通主要依賴[[獅子山隧道|獅子山隧道]]和[[大埔公路|大埔公路]]。1978年獅子山隧道擴闊成雙線雙程的隧道。[[吐露港公路|吐露港公路]]於1985年9月通車，聯同[[沙田路|沙田路]]和大埔公路組成了一個高速公路系統，把新市鎮與[[大埔_(香港)|大埔]]連接起來。[[大老山公路|大老山公路]]亦把馬鞍山與吐露港公路連接起來及[[大老山隧道|大老山隧道]]快捷前往東九龍。[[西沙公路|西沙公路]]於1988年10月通車，提供了一條連接沙田與[[西貢區|西貢]]的道路。[[城門隧道|城門隧道]]於1990年4月通車，將新界東與新界西道路連接起來。連接九龍西部的[[香港8號幹線|8號幹線]][[長沙灣|長沙灣]]至沙田段，以及在沙田新市鎮內接駁8號幹線的T3號公路([[青沙公路|青沙公路]])於2008年3月完成。

=== 公共交通 ===
[[File:Shui_Chuen_O_public_housing_under_construction_(Hong_Kong).jpg|thumb]]
沙田新市鎮設有多條[[巴士|巴士]]路線，由各個人口聚點直通都會區，亦有部分巴士路線由這些人口聚點接駁火車站。除巴士服務外，亦有綠色[[專線小巴|專線小巴]]行走人口較少的住宅區。除[[沙田馬場|沙田馬場]]、[[大圍|大圍]]、[[大埔公路|大埔公路]]、[[沙田路|沙田路]]、[[獅子山隧道公路|獅子山隧道公路]]及[[獅子山隧道|獅子山隧道]]外，沙田新市鎮均禁止紅色小巴行駛。另外，新界的士亦不能在[[香港中文大學|香港中文大學]]以外沙田新市鎮區域載客。

=== 九廣鐵路（港鐵） ===
[[File:Tai_Wai_Station_Property_Site_201810.jpg|thumb]]
[[九廣鐵路|九廣鐵路]]（營運已與[[香港地鐵|香港地鐵]]合併為[[港鐵|港鐵]]）對於新市鎮的發展至為重要。新市鎮共有五個[[東鐵綫|東鐵綫]]車站，即[[大圍站|大圍]]、[[沙田站|沙田]]、[[火炭站|火炭]]、[[馬場站|馬場]]和[[大學站_(新界)|大學]]站。[[馬鞍山綫|馬鞍山綫]]全綫均位於沙田區內，連接[[大圍|大圍]]及馬鞍山[[烏溪沙|烏溪沙]]（利安邨東北面），全長約11.4[[公里|公里]]，共有九個車站及一個維修中心，這條鐵路於[[大圍站|大圍站]]設有轉車設施，連接現有的[[東鐵綫|東鐵綫]]。

== 未來發展 ==
[[白石_(香港)|白石]]從前用作難民羈留中心，坐落於[[馬鞍山市中心|馬鞍山市中心]]以東的策略性位置，是[[吐露港|吐露港]]的陸岬。白石提供了發展康樂設施的機會，現時白石主要為高球場。

[[大圍站上蓋發展項目|大圍站上蓋發展項目]]正興建8幢樓高逾49層的住宅連66.7萬呎商場。加上三幅位於大圍的居屋預計在2022年落成後，大圍人口將進一步上升。
