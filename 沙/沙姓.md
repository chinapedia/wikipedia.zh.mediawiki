'''沙姓'''是[[中国姓氏|中国姓氏]]之一，关于沙姓的起源众说纷纭，一般认为沙姓的得姓[[始祖|始祖]]是[[神农氏|神农氏]]，目前沙姓在[[大陆|大陆]]和[[台湾|台湾]]的百家姓排名都没有进入前一百位。

==起源==
关于沙姓的起源众说纷纭，多奉[[神农氏|神农氏]]为沙姓的始祖，主要分為以下十二種，相當複雜：

# 出自[[神农氏|神农氏]]，为[[炎帝|炎帝]]之后。炎帝为部落首领时，其下有臣[[夙沙氏|夙沙氏]]，后省改为沙姓。 
# 出自[[子姓|子姓]]，是[[商湯|商湯]]之[[苗裔|苗裔]]，以地名为氏。商朝末年，殷纣王庶兄开（一名启）被封于微，世称微子。武王克商后，封微子于商丘，建立[[宋国|宋国]]。[[微子|微子]]的后裔有人被封于沙这个地方，即今天河北省大名县东面。他们以地名为姓，成为沙姓。 
# 出自以国为氏, [[西周|西周]]有[[沙|沙侯国]]，在[[河北省|河北省]][[涉县|涉县]]，涉县之称由[[西汉|西汉]]时沙县改，居住在那里的人也有以沙为氏。
# 出自[[沙|沙随氏]]所改。沙随是春秋时宋国的领地，位于[[河南省|河南省]]宁陵东北部。据《[[春秋|春秋]]》中记载，成公十六年，晋会诸侯于沙随，指的就是这个地方。古代诸侯公爵，凡失国或没爵后，即称为公沙氏或沙随氏，其后人省为沙氏。
# 出自[[鲜卑|鲜卑族]]的[[沙陀|沙陀]]氏, 改姓為沙氏。
# 出自[[突厥|突厥族]]的[[沙陀|沙金]]氏, 改姓為沙氏。
# 出自朝鮮半島上的[[百济国|百济国]][[沙|砂宅]]氏、[[沙|沙宅]]氏、[[沙|沙吒]]氏、[[沙門|沙門]]氏、[[沙|沙随]]氏, 改姓為沙氏。从《万姓统谱》和《百家姓考略》两书记载考证，沙氏系出沙随氏，源于[[百济国|百济国]]，百济国是建于东汉年间，位于现在朝鲜半岛的一个古国。
# 出自汉朝时有一名为沙公穆的人，他的后世子孙去公以沙为氏。
# 出自夏姓，像沙吒氏、沙陀氏、沙 氏等。
# 出自朝鮮半島上的{{fact|[[百洛国|百洛国]]}}有[[倭寇|倭寇]]之[[沙|沙古]]、 [[沙|沙]]氏。[[沙|沙古]]汝武、[[金忠善|沙也可]]、沙汝某。
# 出自回族沙姓：沙姓为[[回族|回族]]中的大姓。中亚[[穆斯林|穆斯林]]的姓Sadiq等人迁入中土後，发展成为沙姓[[回族|回族]]。故《[[千家姓|千家姓]]》称：“今回族多此姓，别为一族。”该姓大多取自经名首音和尾音。如元初的沙全，其“父沙的，世居[[沙漠|沙漠]]……（沙）全初名抄儿赤，甫五岁，为宋军所虏。年十八，留刘整幕下，宋人以其父名沙的，使以沙姓，而名曰全。”（《元史》）再如，“马合马沙或阿思兰沙或木八儿沙之后裔姓沙。”（《回教民族说》）在回族经名中，带有“[[沙|沙]]”的比较多。如《新元史》也黑迭儿传载：“也黑迭儿，西域人……子马谋沙……子密儿沙，次木八人喇沙……次忽都鲁沙……次阿鲁浑沙。”这都是沙姓之源。另，“沙在[[波斯语|波斯语]]义为王。正如[[突厥语|突厥语]]族称王为汗。如在波斯王国旧称‘沙因沙’，义为‘万王之王’……因此，我认为沙姓族源多半是来自波斯。”（《回回姓氏考》）沙姓中也有赐姓。如“舍班，古里国回回，入附授南京锦衣卫镇抚，宣宗宣德五年（1430年）出使西域，以功迁到千户，赐姓沙，名班。”（《回回历史与伊斯兰文化》）当时给舍班赐姓沙，还有着另一个原因，即“舍”和“沙”谐音。沙姓也为“回族十三姓”之一。沙姓回族主要分布在西北、[[江浙|江浙]]地区。
# 出自满族沙姓：傳說得姓始祖為[[神农氏|神农氏]]。历史上[[满族|满族]]人归化多有改为沙姓。

==名人==
===古人===

*[[沙元炳|沙元炳]]：清朝政治人物。
*[[沙春元|沙春元]]：清朝軍事將領。

*[[沙千里|沙千里]]中國政治人物。
*[[沙祖康|沙祖康]]：中國外交官。
*[[沙溢|沙溢]]：中國演員。
*[[沙正文|沙正文]]：中國運動員。
*[[沙國河|沙國河]]：中國物理化學家。
*[[沙葉新|沙葉新]]：中國作家。
*[[沙孟海|沙孟海]]：中國書法家。
*[[沙桐|沙桐]]：[[中国中央电视台|中国中央电视台]]体育节目主持人。
*[[沙寶亮|沙寶亮]]：中國歌手。
*[[沙志一|沙志一]]：行政院農委會副主委。
*[[沙道维|沙道维]]：中國现代著名書法家。
*[[沙季凡|沙季凡]]：台灣補教界社會科名師。
*[[沙永傑|沙永傑]]：台灣學者，[[TRIZ|萃思學]]之先驅權威。
*[[沙立|沙立]]：台灣醫師。
