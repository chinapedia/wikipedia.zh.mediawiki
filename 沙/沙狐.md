'''沙狐'''（[[學名|學名]]：''Vulpes corsac''；[[英文|英文]]：''Corsac Fox''），亦稱'''東沙狐'''，[[食肉目|食肉目]][[犬科|犬科]][[狐属|狐属]]。春末夏初產子，每年一胎，通常每胎3～6隻。2歲性成熟。壽命約6年。

== 分布 ==
[[File:_Korsak.jpg|thumb]]的沙狐]]
主要分布於[[中國|中國]]（[[新疆|新疆]]、[[青海|青海]]、[[甘肅|甘肅]]、[[寧夏|寧夏]]、[[內蒙古|內蒙古]]、[[西藏|西藏]]）、[[蒙古|蒙古]]、[[外貝加爾|外貝加爾]]、[[阿富汗|阿富汗]]、[[尼泊爾|尼泊爾]]等地區。

== 特徵 ==
體長約50～60厘米，尾長約25～35厘米，體重約2～3[[公斤|公斤]]。[[四肢|四肢]]短，耳大而尖，[[基部|基部]]（Base）寬。背部呈淺[[棕色|棕]][[灰色|灰色]]或淺[[紅色|紅]][[褐色|褐色]]，腹部呈淡[[白色|白色]]或淡[[黃色|黃色]]。

== 習性 ==
主要棲息於開闊的[[草原|草原]]和半[[沙漠|沙漠]]地帶，夜間活動，共同生活在一個巢穴內。以小型[[齧齒類|齧齒類]]動物為食，也捕食[[鳥類|鳥類]]、[[蜥蜴|蜥蜴]]和[[昆蟲|昆蟲]]。1～3月[[發情|發情]]。[[妊娠期|妊娠期]]50～60天。

== 亞種 ==
此種[[狐狸|狐狸]]含有三個[[亞種|亞種]]：
* ''Vulpes corsac corsac''
* ''Vulpes corsac kalmykorum''
* ''Vulpes corsac turkmenicus''
