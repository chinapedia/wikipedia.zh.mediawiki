'''磁碟鏡像'''，即映像檔，係指將檔案燒錄成虛擬光碟，並透過掛載虛擬光碟機的功能來模擬光碟運行。使用時機為光碟對烤(A to A)或是複製(A to B)又或是將其轉換成虛擬光碟(A to C)等。

== 词义辩析 ==

“磁盘镜像”一词一般有两种不同含义。一种是指复制到相同功能的存储装置中以起到增强数据整合度、增强容错功能、增加吞吐量等作用（如[[RAID|RAID]]）。这时对应英文一般为Disk Mirror，以下称为'''磁盘镜像'''。另一种含义是指复制到不同的装置或数据格式，主要用于数据[[备份|备份]]。这时对应英文一般为Disk Image，以下称为“磁盘映像”。通常在使用中这两者都称为“镜像”或“磁盘镜像”。Disk Mirror在台灣IT界被称为“磁碟鏡射”，它可以將Disk drive的內容與資料，猶如照鏡子時影像的反射一般，將資料一模一樣地複製一份，因此被称为“磁碟鏡射”。

== 磁盘镜像 ==

{{link-en|磁盘镜像|Disk Mirror}}，指[[实时|实时]]的将一个逻辑[[磁盘卷|磁盘卷]]上的数据复制到若干个逻辑磁盘卷上，以确保其[[连续可用性|连续可用性]]、一致性及准确性。'''镜像卷'''是指某个逻辑磁盘卷的单独且完整的[[拷贝|拷贝]]。远程磁盘镜像的技术可用于[[灾难恢复|灾难恢复]]。由所使用技术的不同，磁盘镜像可以由同步、异步、半同步、时间点等等方式建立。

磁盘镜像由[[储存子系统|储存子系统]]里的[[微代码|微代码]]来实现，有时也借助于主机上的其它软件。由于镜像的过程所依赖的微代码和硬件密切相关，不同的储存设备供应商对于磁盘镜像的实现都会有所不同。

真正的同步磁盘镜像可以实现无损失的{{link-en|恢复点目标|Recovery Point Objective}}。异步磁盘镜像可以实现几秒钟的RPO。其它的技术则可以实现几分钟到几小时的RPO。

磁盘镜像主要用于[[企业数据储存|企业数据储存]]和[[磁盘阵列|磁盘阵列]]技术（例如[[RAID|RAID-1]]）。

== 磁盘映像 ==

[[磁盘映像|磁盘映像]]，是指將有某種儲存裝置（例如[[CD|CD]]）的完整內容及結構保存為一個[[電腦檔案|電腦檔案]]，所以通常這些檔案都會是很大。一般只有一个檔案，裡面包含了许多的檔案備份。镜像的制作方法除了工具把實體磁碟的內容保存起来之外，也有專門的工具可不從實體磁碟製作出鏡像（例如不需讀取實體CD即可製作CD鏡像）

最常用到的磁碟鏡像是光盘镜像，是指從CD或[[DVD|DVD]]製作的鏡像。簡單地說，光盘镜像就是CD或DVD的拷貝，所有的資料都存在一個檔案中，藉此保存CD或DVD的結構及完整性。光盘镜像通常是以[[ISO_9660|ISO 9660]]格式存储的，
[[隨身碟|隨身碟]]或[[軟碟|軟碟]]的鏡像檔為[[.img|IMG格式]]；而[[諾頓魅影系統|諾頓魅影系統]]則可以為[[硬碟|硬碟]]產生GHO格式（版本9.0以後為V2I格式）的鏡像；[[Mac_OS_X|Mac OS X]]常用的磁盘映像文件为DMG格式，常常用于打包软件用于网络分发，或备份磁盘等。
