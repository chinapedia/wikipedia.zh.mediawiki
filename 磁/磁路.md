{{磁路相关}}

'''磁路'''是一个包含[[磁通量|磁通量]]的闭合回路。它一般含有[[磁|磁]]的成分，例如[[永久磁铁|永久磁铁]]、[[铁磁性|铁磁性]]材料，以及[[电磁铁|电磁铁]]，但也可能含有空气间隙和其它的物质。磁路被用于许多设备以有效地引导[[磁场|磁场]]，如[[电动机|电动机]]，[[发电机|发电机]]，[[变压器|变压器]]，[[继电器|继电器]]，起重[[电磁铁|电磁铁]]，[[超导量子干涉仪|超导量子干涉仪]]，[[检流计|检流计]]，和磁性{{le|记录头|Recording head}}。

== 磁动势 ==
{{Main|磁动势}}
磁动势<ref>''The Penguin Dictionary of Physics'', 1977, ISBN 0-14-051071-0</ref>的标准定义是电流流过导体所产生磁通量的势力(''force'')<ref name=force>'''势力'''(''force''):在此处是指一种[[工作潜能|工作潜能]](''work potential''),相似但区别于力学中用[[牛顿|牛顿]]来度量的'''[[力|力]]'''。</ref>，是用来度量[[磁场|磁场]]或[[电磁场|电磁场]]的一种量，类似于[[电场|电场]]中的[[电动势|电动势]]或[[电压|电压]]。它被描述为线圈所能产生磁通量的势力<ref name=force />，这样科学家就能够用它来衡量或预见通电线圈实际能够激发磁通量的势力<ref name=force />。此外，[[永久磁鐵|永久磁鐵]]也會有磁动势。

== 磁路定律 ==
[[File:Magnetischer_Kreis.svg|thumb]]

如果<math>\Phi</math>是磁路的磁通量，F 是[[磁动势|磁动势]]，R 是[[磁阻|磁阻]]，则：

:<math>\Phi = \frac F R</math>

这与电路中的[[欧姆定律|欧姆定律]]类似，欧姆定律是[[电流|电流]]等于[[电压|电压]]（有时称为[[电动势|电动势]]）除以[[电阻|电阻]]。

如果''A''是[[面积|面积]]，''μ''是物质的[[磁导率|磁导率]]，''l''是[[长度|长度]]，则

:<math>R_m = \frac{l}{\mu A}</math>

这与电阻的方程类似。形状越长、细，磁导率越低，磁阻就越大。

磁路还服从其它与电路类似的定律。例如，把磁阻<math>R_1,\ R_2,\ \dots</math>串联起来，则总的磁阻<math>R_T</math>为：

:<math>R_T = R_1 + R_2 + \dots</math>

这与[[基尔霍夫第一定律|基尔霍夫第一定律]]类似。在任何一个节点，磁通量<math>\Phi_1,\ \Phi_2,\ \dots</math>的和总是等于零：

:<math>\Phi_1 + \Phi_2 + \dots = 0</math>. 
这与[[基尔霍夫第二定律|基尔霍夫第二定律]]类似。

这三个定律形成了一个用来分析磁路的完整的系统。比较磁路和电路，可知：
* 电阻''R''与'''磁阻''' ''R''<sub>m</sub>相对应；
* 电流''I''与'''磁通量''' ''Φ''相对应：
* 电压''V''与'''磁动势''' ''F''相对应。
