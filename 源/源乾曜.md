'''源乾曜'''（{{bd|7世纪||731年|12月22日|catIdx=Y源}}），封'''安阳郡公'''，[[唐朝|唐朝]]官员，[[唐玄宗|唐玄宗]]年间两度担任[[宰相|宰相]]。

== 家庭背景 ==
源乾曜来自[[相州_(北周)|相州]]（今[[邯郸|邯郸]]）。他的家族可以追溯到鲜卑政权[[南凉|南凉]]后裔，后来在[[北魏|北魏]]受封为陇西宣王的[[源贺|源贺]]。他的先祖后来在[[北周|北周]]和[[隋朝|隋朝]]为臣。<ref>《[[新唐书|新唐书]]》卷七十五上·宰相世系表五上：「（前略）……【贺】位太尉、陇西宣王，生侍中冯翊惠公怀，怀二子：子邕、子恭，子恭字灵顺，中书监、临汝文献公，周、隋之际，居邺都安阳。」</ref>源乾曜的祖父[[源师民|源师民]]在隋朝任[[刑部侍郎|刑部侍郎]]，父亲[[源直心|源直心]]则担任[[唐高宗|唐高宗]]的司刑太常伯，但因事被流配[[岭南|岭南]]而死。<ref>源直心任司刑太常伯的经历，见两唐书源乾曜传。《新唐书》卷七十五上·宰相世系表五上则称他的官职为司刑[[太常丞|太常丞]]，这个官职只存在于662年 - 670年。根据《[[资治通鉴|资治通鉴]]》卷二百零一，源直心很可能是在此期间被流放。</ref>源乾曜举[[进士|进士]]，入仕时已四十多岁。曾任[[夔州|夔州]]参军，其马厩在城东北约三百步，后为[[孔子|孔子]]庙。<ref>[[:s:夔州都督府记|李贻孙《夔州都督府记》]]</ref>

== 唐中宗、唐睿宗第二次登基年间 ==
[[唐高宗|唐高宗]]之子[[唐中宗|唐中宗]]第二次登基时（705年 - 710年），源乾曜任[[殿中侍御史|殿中侍御史]]，[[神龙_(年号)|神龙]]二年（706年）二月黜陟江东道，<ref>[[:s:旧唐书/卷102|《旧唐书》卷一百零二]]</ref><ref>[[:s:资治通鉴/卷208|《资治通鉴》卷二百零八]]</ref>为朝散大夫行度支员外郎摄右台侍御史。他被认为是一位称职的官员。<ref>[[:s:县令岑君德政碑|张景毓《县令岑君德政碑》]]</ref>[[唐隆|唐隆]]元年（710年），中宗的弟弟相王、先前的[[唐睿宗|唐睿宗]][[李旦|李旦]]也再次登基。[[景云|景云]]年间，累迁[[谏议大夫|谏议大夫]]。此时，[[公卿百官三九射礼|公卿百官三九射礼]]不行已久。二年（711年）二月，源乾曜引用[[孔子|孔子]]的话上疏：「尔爱其羊，我爱其礼。」主张复兴。<ref>[[:s:请举行射礼疏|:s:请举行射礼疏]]</ref>历史上并未记载睿宗是否采纳了他的意见，但不久他就被任为[[梁州|梁州]]（大约在今[[汉中|汉中]]）[[都督|都督]]。[[太极|太极]]元年（712年），诏举文可以经邦国者，源乾曜时任宣劳使，推荐独孤通理作赋，独孤通理从数百人中夺魁。<ref>[[:s:唐故朝散大夫颍川郡长史赠秘书监河南独孤公灵表|独孤及《唐故朝散大夫颍川郡长史赠秘书监河南独孤公灵表》]]</ref>

== 唐玄宗年间 ==
[[延和_(唐朝)|延和]]元年（712年），唐睿宗禅位给太子[[李隆基|李隆基]]，后者即位为唐玄宗。玄宗在位早期的[[开元|开元]]（713年 - 741年）年间，玄宗的堂兄、故[[章怀太子|章怀太子]][[李贤_(唐朝)|李贤]]之子邠王[[李守礼|李守礼]]的属下官吏犯法，玄宗希望给邠王任命一位[[长史|长史]]，玄宗近臣太常卿[[姜皎|姜皎]]推荐源乾曜公正清明有吏干。玄宗召见源乾曜，深为他的神气清爽和对答有序所打动，于是拜行少府少监，兼邠王府长史，<ref>[[:s:旧唐书/卷86|《旧唐书》卷八十六]]</ref><ref>[[:s:新唐书/卷81|《新唐书》卷八十一]]</ref>不久以正议大夫、行少府监、[[上柱国|上柱国]]、'''安阳县开国男'''迁行尚书[[户部侍郎|户部侍郎]]，散官勋封如故，<ref>[[:s:授源乾曜户部侍郎制|苏颋《授源乾曜户部侍郎制》]]</ref>兼[[御史中丞|御史中丞]]，未几又转[[尚书左丞|尚书左丞]]。唐玄宗对宦官[[高力士|高力士]]说自己速度提拔源乾曜是因为源乾曜的相貌和言辞像先前因[[太平公主|太平公主]]谋反案伏诛、但为人得到玄宗认可的[[萧至忠|萧至忠]]。但[[欧阳修|欧阳修]]认为萧至忠不贤，评价“帝以乾曜似之，遽使当国，是帝举不知至忠之不可用，又不知乾曜之所可用也”。<ref>[[:s:新唐书/卷123|《新唐书》卷一百二十三]]</ref>

开元四年（716年）十一月，源乾曜擢拜[[黄门侍郎|黄门侍郎]]，授[[同中书门下平章事|同紫微黄门平章事]]，为实质宰相，勋封如故。但玄宗更倚重身患[[疟疾|疟疾]]的宰相[[姚崇|姚崇]]，军国大事都问他，其他宰相源乾曜、[[卢怀慎|卢怀慎]]等唯唯诺诺而已。姚崇在[[长安|长安]]没有住所，寓居[[罔极寺|罔极寺]]养病，玄宗经常让源乾曜去那里咨询姚崇。当源乾曜提出的建议让玄宗满意时，玄宗总说：“这一定是姚崇的意思。”而当源乾曜提出的建议与玄宗不合时，玄宗则说：“你应该先去咨询姚崇。”源乾曜也只得谢罪，承认确实没有咨询过。<ref name=NBT124>[[:s:新唐书/卷124|《新唐书》卷一百二十四]]</ref>源乾曜奏称姚崇气力虚弱，不能走路，希望令他移居四方馆，特许家人视疾，玄宗准奏。但姚崇很快因为儿子们光禄少卿姚彝、宗正少卿姚异及亲信中书主书[[赵诲|赵诲]]受贿事发而失宠，姚崇因此辞职，推荐[[宋璟|宋璟]]接替自己。<ref name=BT96>[[:s:旧唐书/卷96|《旧唐书》卷九十六]]</ref>源乾曜拜相才两个月，闰十二月就以黄门侍郎、安阳男的官爵和姚崇一同罢相，<ref name=NBT5>[[:s:新唐书/卷5|《新唐书》卷五]]</ref>由宋璟和[[苏颋|苏颋]]替代。<ref>[[:s:资治通鉴/卷211|《资治通鉴》卷二百一十一]]</ref>玄宗行幸东都[[洛阳|洛阳]]时，任源乾曜守[[京兆尹|京兆尹]]管理长安。<ref name=BT8>[[:s:旧唐书/卷8|《旧唐书》卷八]]</ref><ref name=BT105>[[:s:旧唐书/卷105|《旧唐书》卷一百零五]]</ref>源乾曜任京兆尹三年，把长安管理得很好。一次，玄宗打猎时不慎放走了一只白鹰，命手下抓回。当源乾曜的手下发现白鹰时，它已经挂在荆棘丛里死了。这些手下害怕获罪，但源乾曜认为皇帝不会怪罪，即使怪罪，他自己也会一力承担，于是上表请罪。玄宗果然没有处罚源乾曜和他的手下。

八年（720年）正月，源乾曜以尚书左丞、上柱国、安阳县开国男的身份再次任黄门侍郎、同中书门下三品，<ref>[[:s:授源乾曜黄门侍郎同平章事制|:s:授源乾曜黄门侍郎同平章事制]]</ref>一同拜相的还有[[张嘉贞|张嘉贞]]，取代了宋璟和苏颋。<ref>[[:s:资治通鉴/卷212|《资治通鉴》卷二百一十二]]</ref>不久，他又加[[银青光禄大夫|银青光禄大夫]]，五月迁[[侍中|侍中]]。<ref name=NBT5/><ref name=BT8/>姜皎外甥太子中允[[李林甫|李林甫]]通过源乾曜之子源洁请求为司门郎中，源乾曜说：“郎官须是有素行且才望高的人，哥奴（李林甫小字）岂是郎官？”数日后，除其为谕德。<ref name=BT106>[[:s:旧唐书/卷106|《旧唐书》卷一百零六]]</ref><ref>[[:s:新唐书/卷223上|《新唐书》卷二百二十三上]]</ref>源乾曜上表指出有权势的官员的儿子和兄弟都在京城任职，迫使没有这层关系的初级官吏只能到边远地区任职，提议把这些官家的儿子、兄弟都遣离京城，并且要求把自己在京城任职的三个儿子中的两个也遣出京城。<ref>[[:s:请出二子为外官疏|:s:请出二子为外官疏]]</ref>玄宗表扬了他，同意了他的建议，下令每个官员家不得同时有三人在京城任职。公卿子弟做京官的因此出外者百余人。源乾曜子河南府参军源弼出为绛州司功，太祝源洁出为郑县尉。

兵部员外郎兼侍御史[[宇文融|宇文融]]奏置劝农判官十人，并摄御史，为括客之法，阳翟尉皇甫憬、左拾遗杨相如、户部侍郎杨瑒上疏反对，而源乾曜和中书舍人[[陆坚_(唐朝)|陆坚]]都赞成，于是皇甫憬被贬为盈川尉，<ref name=BT105/>杨瑒也被贬。<ref>[[:s:新唐书/卷134|《新唐书》卷一百三十四]]</ref>

十年（722年）四月，与张嘉贞、兵部尚书[[张说|张说]]奉敕去赤帝坛祈雨。<ref>[[:s:奉敕赤帝坛祈雨文|张说《奉敕赤帝坛祈雨文》]]</ref>夏，伊、汝闹水灾，在中书舍人知制诰[[许景先|许景先]]建议下，源乾曜奏报玄宗，玄宗下诏遣户部尚书[[陆象先|陆象先]]赈灾。<ref>[[:s:旧唐书/卷190中|《旧唐书》卷一百九十中]]</ref>十一月，玄宗采纳建议，敕宰相食实封三百户，源乾曜和张嘉贞、张说成为首批受益者。<ref>[[:s:源乾曜等食实封敕|:s:源乾曜等食实封敕]]</ref>

十一年（723年）十一月，玄宗亲祠南郊，大赦天下，赐源乾曜、张说、兵部尚书[[同中书门下三品|同中书门下三品]][[王晙|王晙]]物五百匹。<ref>[[:s:南郊赦书|张九龄《南郊赦书》]]</ref>冬，[[许州|许州]][[刺史|刺史]]王乔家奴告王乔与王晙潜谋构逆，源乾曜和时任[[中书令|中书令]]的张说奉命审案，查出王晙无反状，但王晙仍以他罪被贬。<ref>[[:s:旧唐书/卷93|《旧唐书》卷九十三]]</ref><ref>[[:s:新唐书/卷111|《新唐书》卷一百一十一]]</ref>十二年（724年），新拜户部尚书兼益州长史判都督事的张嘉贞恨张说排挤自己，奉敕在中书省与宰相会宴时挽起袖子大骂，源乾曜、王晙一起为他们和解。<ref>[[:s:旧唐书/卷99|《旧唐书》卷九十九]]</ref><ref>[[:s:新唐书/卷128|《新唐书》卷一百二十八]]</ref>同年，玄宗为源乾曜、张说亲作褒奖之词，称源乾曜“謇謇匪躬，谦谦自牧，正身率下，直道事人，无闻伐己之功，每立致君之节，顾问则出纳斯允，左右则启沃居多。德行可称，自宜升擢。”

十三年（725年），玄宗采纳时任宰相的张说建议，准备封禅[[泰山|泰山]]，源乾曜不同意，<ref>[[:s:答源乾曜等请封禅手诏|:s:答源乾曜等请封禅手诏]]</ref><ref>[[:s:答源乾曜等重请封禅手诏|:s:答源乾曜等重请封禅手诏]]</ref><ref>[[:s:请封禅第三表|:s:请封禅第三表]]</ref>从此与张说不合。同年，[[贺知章|贺知章]]迁礼部侍郎，兼集贤院学士，入内谢恩。源乾曜对张说说：“贺公两命之荣，足为光宠，然学士、侍郎孰为美？”张说答二者相去甚远。<ref>[[:s:新唐书/卷196|《新唐书》卷一百九十六]]</ref>十一月，封禅结束，源乾曜随同封禅，撰《社首坛颂》，<ref>[[:s:旧唐书/卷23|《旧唐书》卷二十三]]</ref>作迎神音乐《灵具醉》，<ref>[[:s:旧唐书/卷30|《旧唐书》卷三十]]</ref>以正议大夫、行尚书户部侍郎、上柱国、安阳县开国男、兼御史中丞拜尚书左丞相，仍兼侍中，<ref name=BT8/><ref name=BT106/>勋封如故。<ref>[[:s:授源乾曜等尚书右丞等制|苏颋《授源乾曜等尚书右丞等制》]]</ref>封禅后，张说奏[[吐蕃|吐蕃]]悔过请和事，玄宗说要和[[河西陇右节度使|河西陇右节度使]]、右羽林军将军判[[凉州|凉州]]都督事[[王君㚟|王君㚟]]商议，张说出去后，玄宗又对源乾曜说：“王君㚟勇而无谋，常思侥幸，一旦两国和好，他怎么立功？如果他进来说他的谋划，我的计划就不成了。”不久王君㚟即入朝奏事请求率兵深入讨吐蕃。<ref>[[:s:旧唐书/卷196上|《旧唐书》卷一百九十六上]]</ref><ref>[[:s:新唐书/卷125|《新唐书》卷一百二十五]]</ref>十四年（726年）四月，张说被御史中丞宇文融、御史大夫[[崔隐甫|崔隐甫]]、中丞李林甫弹劾受贿，玄宗命源乾曜与刑部尚书[[韦抗|韦抗]]、大理少卿[[胡珪|胡珪]]、崔隐甫去尚书省审理，最后张说虽然获得了宽大处理，但被罢[[中书令|中书令]]。<ref name=BT97>[[:s:旧唐书/卷97|《旧唐书》卷九十七]]</ref><ref name=ZZTJ213>[[:s:资治通鉴/卷213|《资治通鉴》卷二百一十三]]</ref>

在和张嘉贞、张说共事期间，源乾曜往往处事谨慎，在重大问题上附和他们。玄宗曾考其为中上。<ref>[[:s:赐源乾曜张说考中上词|:s:赐源乾曜张说考中上词]]</ref>十年（722年），由于张嘉贞排挤，当年举荐源乾曜、又是源乾曜侄孙源光乘妻兄的姜皎被流放，源乾曜却不相救，因此广受批评。<ref>[[:s:旧唐书/卷59|《旧唐书》卷五十九]]</ref><ref>[[:s:新唐书/卷91|《新唐书》卷九十一]]</ref>源乾曜以子求婚于李令一，李令一坚拒，言辞有所冒犯，被贬为[[黔州|黔州]][[彭水|彭水]]尉。<ref>[[:s:故歙州长史陇西李府君墓志铭|李翱《故歙州长史陇西李府君墓志铭》]]</ref>

十五年（727年）五月，诏中书门下引文武举人就中策试，蓝田县尉萧谅、右卫胄曹梁涉、邠州柱国子张玘等对策稍优，录奏时，玄宗对源乾曜及其同僚宰相[[李元纮|李元纮]]、[[杜暹|杜暹]]等说：“朕宵衣旰食，侧席求贤，所以每念搜扬者，恐草泽遗才无繇自达。至如畿尉卫佐，未经推择，更与褐衣争进，非朕本意。”于是唯以张玘为下第放选，其余都罢去。<ref>《[[册府元龟|册府元龟]]》</ref>

源乾曜和李元纮、杜暹共事，唯唯诺诺署名而已，李、杜政见不合，他也不作调停。这让玄宗对他们三人都不满，十七年（729年）六月，将他们一并罢相，<ref name=NBT5/>改任[[裴光庭|裴光庭]]、[[宇文融|宇文融]]，另一位宰相[[萧嵩|萧嵩]]留任。此后短期内，源乾曜仍任左丞相。<ref name=ZZTJ213/><ref>[[:s:停源乾曜侍中制|:s:停源乾曜侍中制]]</ref>他与张说都请求以玄宗生日八月五日为千秋节，休假一日，群臣献甘露、玄宗酹酒，士庶村社宴乐，于是成为常例。<ref>[[:s:旧唐书/卷17下|《旧唐书》卷一十七下]]</ref><ref>[[:s:请禁以降诞日为节假奏|李元素《请禁以降诞日为节假奏》]]</ref><ref>[[:s:请以降诞日为千春节奏|李愚《请以降诞日为千春节奏》]]</ref>八月，他被任命为太子[[李瑛_(唐朝)|李鸿]]的属官[[太子少师|太子少师]]，但因为官名犯了祖父源师民的讳，他坚持不肯就职，于是玄宗改太子少师为[[太子少傅|太子少傅]]。<ref name=BT8/>与此同时，宋璟任尚书[[右丞相|右丞相]]，张说任[[左丞相|左丞相]]，玄宗为他们三人举行了盛大的就职典礼，提供[[御膳房|御膳房]]饮食及命太常奏乐，亲自作三杰诗赐给他们，<ref name=NBT124/><ref name=BT96/><ref name=BT97/>并封源乾曜为安阳郡公。

十九年（731年），玄宗再次幸洛阳，此时的源乾曜已年老患病，无法随行，留在长安。12月22日，源乾曜病逝<ref>《旧唐书》卷八·玄宗纪上：「开元十九年十一月甲子【十九日】（731年12月22日），太子少傅源乾曜薨。」</ref>，以礼安葬，赠幽州大都督。玄宗在洛阳南门为他举哀，辍朝两天以表哀悼。

后源乾曜得绘图[[凌烟阁|凌烟阁]]。<ref>[[:s:新唐书/卷191|《新唐书》卷一百九十一]]</ref>

== 轶闻 ==
[[郗昂|郗昂]]性捷直，源乾曜曾戏言他：“[[谢安|谢安]]说‘郗生（[[郗超|郗超]]）可谓入幕之宾矣’，莫非是你的远祖？”郗昂说：“犹胜以氏为秃发。若不遇[[北魏道武帝|后魏道武]]，称之为同源，赐之姓源氏，岂可列于《[[姓苑|姓苑]]》？”源乾曜于是折服。<ref>《[[唐语林|唐语林]]》</ref>
