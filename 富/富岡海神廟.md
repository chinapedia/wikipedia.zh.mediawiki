'''富岡海神廟'''位於[[臺灣|臺灣]][[臺東縣|臺東縣]][[臺東市|臺東市]][[富岡漁港|富岡漁港]]旁，是一間主祀如意娘娘的[[道教|道教]][[廟宇|廟宇]]。自[[第二次國共內戰|第二次國共內戰]]末期經多次遷徒後，來到該地區定居的[[大陳義胞|大陳義胞]]所供奉的廟宇，乃臺灣唯一主祀如意娘娘、亦是唯一由[[中華民國空軍|中華民國空軍]]所興建的廟宇。<ref name="臺灣海洋教育中心"/>

== 沿革 ==
海神廟主祀的「如意娘娘」，最初原為[[漁山列島|漁山列島]]上的娘娘廟所供奉之神祇。如意娘娘的事蹟酷似[[閩南|閩南]]篤信的[[媽祖|媽祖]]，相傳如意娘娘原是一位名喚「如意」的少女，因父親捕魚遇到[[船難|船難]]，如意跳水尋父而仙逝，而當地人敬佩萬分，就以如意跳水處的木材雕刻[[神位|神位]]供奉，視為[[海神|海神]]。據說能保佑漁民與水手出海平安。

1955年2月，[[中華民國國軍|中華民國國軍]]在[[美國海軍|美國海軍]][[第七艦隊|第七艦隊]]的掩護之下，執行了名為「[[金剛計劃|金剛計劃]]」的[[大陳島撤退|大陳島撤退]]，其中便包含了漁山列島，因此當時島上的居民，在倉促撤退之下，便將廟內如意娘娘的神像給裝進木箱中，由於如意娘娘的神像高約40到50公分，無法放入木箱中，過於倉促之下只得將神像頭上的鳳冠取下才能放入木箱搬運。<ref name="大紀元"/>隨後就由撤退的信徒輪流揹著木箱撤退來臺<ref name="教育大市集"/>。

漁山列島的居民來到臺灣後，暫時安置在[[基隆市|基隆市]]的臨時接待所，這時部分信徒便將神像自木箱取出，並簡易供奉起來燒香參拜。之後漁山列島居民再度被政府遷移到今日臺東縣[[東河鄉_(臺灣)|東河鄉]]的興昌一帶。1956年再遷到今日縣立富岡國小旁，後形成聚落，稱為富岡新村。<ref name="臺東大學"/>

這段期間，信徒們輪流協助搬運如意娘娘神像在各處遷徒，直到在富岡國小旁形成聚落後，神像才暫時安放在民宅中供信徒參拜。1965年，信徒們發起集資建廟計劃，這時如意娘娘神像才有了真正的安放位置，並於廟宇興建期間，信徒們請來時任[[中國國民黨|中國國民黨]]臺東縣委員會主任委員的蕭煥復為廟題名，當時蕭煥復考量到大陳義胞多以捕魚為業，加上居民以祭祀來祈求出海平安、滿載而歸，因此便將原本的娘娘廟改名為海神廟，之後再致贈匾額一副以資紀念。<ref name="臺東大學"/>

然而，1970年[[中華民國空軍|中華民國空軍]]將在[[海岸山脈|海岸山脈]]末端與猴子山之間興建大型空軍基地，稱為[[空軍志航基地|空軍志航基地]]，當時的海神廟座落位置首當其衝，因此富岡新村又再次搬遷到富岡漁港上方的台階地上，後來空軍也協助村民重新新建海神廟，使得海神廟成為全臺唯一由空軍興建的廟宇，並且由於是空軍興建，其廟宇建築的全盤功能性更為周到，因此海神廟反而因禍得福<ref name="教育大市集"/><ref name="臺灣海洋教育中心"/>。

1976年6月，海神廟信徒蔡欽玉、蔡家錚兩人捐款為村裡的[[土地公廟|土地公廟]]新建金爐一座，自此兩間廟成為友誼廟<ref name="中研院"/>。

2007年11月，海神廟發起整建工程，當時共有200位信徒捐款，最後共募得[[新臺幣|新臺幣]]258萬2千元用以整修廟宇建築，整建項目包括將中殿和左右殿的神龕重新妝點，並於整建完工後設立海神廟管理委員會，後再於廟裡設立[[寧波市|寧波市]][[象山縣|象山縣]]小石浦大陳義胞發展協會的聯合辦公室，作為召開村里會議與廟務會議的集會場所<ref name="臺東大學"/>。

2008年9月，中國浙江省寧波市象山縣將在該月14日將舉辦「開魚祭」，而海神廟的信徒也決定，將迎駕如意娘娘「回娘家」，回到漁山列島出巡<ref name="大紀元"/>。

2014年9月6日，擔任海神廟主委的柯受球、胞弟柯受雄，以及海神廟管理委員會團隊邀請來自[[中國|中國]][[福建省|福建省]][[南安市|泉州南安市]][[詩山鎮|詩山鎮]]的[[鳳山寺(南安)|鳳山寺]][[保安廣澤尊王|保安廣澤尊王]]本尊來到海神廟出巡，並於返回中國前，分靈出一尊廣澤尊王的神像供奉在海神廟內供信徒祭拜<ref name="更生日報"/>。

近幾年，來自[[苗栗縣|苗栗縣]][[後龍鎮_(台灣)|後龍鎮]]的[[後龍南文宮|後龍南文宮]]信徒特地前來海神廟認親，由於後龍南文宮主祀的[[金母|金母]]娘娘指示，富岡海神廟所祭祀之如意娘娘是其六妹，自此兩廟成為姐妹廟，時常往來<ref name="教育大市集"/>。

== 建築特色 ==
海神廟總占地面積約有300多坪，其中廟埕及佔去土地的一大半，建築物包含戲台、涼亭、拜亭，以及正殿，上述的建築設計上可連成一線，廟宇的左右側均可以出入。戲台與其屋頂皆為[[混凝土|混凝土]]結構，屋頂上雕塑有八仙泥塑，涼亭部分，屋頂為鐵皮結構<ref name="教育大市集"/>。

拜亭與正殿為磚造建築，並以[[鋼筋|鋼筋]]混凝土結構補強，其中屋頂設計有多個剪黏藝術，拜亭屋頂的剪黏藝術為歇山燕尾脊、火型山牆，正脊上裝飾著鳳來儀拜三星，雙排頭均以民間傳說人物的剪黏作品為主、背面排頭則裝飾著雙魚銜錢躍向正殿。拜亭設有混凝土結構的龍柱，龍柱前放有[[石獅子|石獅子]]一對，<ref name="教育大市集"/>是由有亞洲飛人之稱的[[柯受良|柯受良]]所捐贈<ref name="中研院"/>。

另外廟裡還展示著一艘漁船的模型，以彰顯大陳義胞以漁為業的海民特色，該漁船是由[[屏東縣|屏東縣]][[東港鎮_(臺灣)|東港鎮]][[東港東隆宮|東港東隆宮]]所捐贈<ref name="中研院"/>。

正殿屋頂為三川燕尾脊、脊上的剪黏為雙龍拜塔，尾脊又有雙龍護衛，在塔尖處設有[[避雷針|避雷針]]，是臺灣民間廟宇中較罕見的設計，排頭處均為民間傳說人物。三川步口為彩繪混凝土龍柱，龍柱上有泥塑青龍白虎壁配上[[千里眼|千里眼]]與[[順風耳|順風耳]]<ref name="教育大市集"/>。

== 重要祭典 ==
海神廟內主祀著如意娘娘，另外還同時祭祀著[[池府千歲|池府千歲]]、[[保生大帝|保生大帝]]、[[福德正神|福德正神]]、[[廣澤尊王|廣澤尊王]]以及[[中壇元帥|中壇元帥]]等神祇。而這些神祇皆來自[[福建省|福建省]]南部，其原因在於當初漁山列島的居民大多數屬於和臺灣主流漢族一樣的[[閩南民系|閩南民系]]，這些閩南族群都來自[[福建省|福建省]][[泉州府|泉州府]][[同安縣|同安縣]]，因此才會有如此特色。海神廟每年重要的祭典包括[[農曆|農曆]]7月6日的如意娘娘誕辰兼[[中元普渡|中元普渡]]，以及農曆6月18日的池府千歲千秋。<ref name="教育大市集"/>

另外在每年的7月海神廟也會結合、大陳義胞、社區志工等組織共同舉行神明出海祈福的活動，稱為海神祭<ref name="水保局"/>。
