'''富山輕軌TLR0600型電車'''（{{jpn|j=富山ライトレールTLR0600形電車}}）是[[富山輕軌|富山輕軌]][[富山港線|富山港線]]的[[路面電車|路面電車]]車輛，暱稱「'''PORTRAM'''」（{{lang|ja|ポートラム}}），2007年[[鐵道友之會|鐵道友之會]][[藍絲帶獎_(鐵路)|藍絲帶獎]]獲獎車輛<ref>特急形ではない車両がブルーリボン賞を受賞するのは第48回の[[日本貨物鐵道|JR貨物]][[JR貨物M250系電車|M250系]]以来のこと。</ref>，於2006年（平成18年）4月29日開始開始投入營運。

== 概要 ==
富山輕軌0600型是由[[新潟運輸系統|新潟運輸系統]]製造的[[關節式轉向架|雙節式]][[超低床電車|超低底盤]][[路面電車|路面電車]]，在富山港線開業時共導入了7列。該款車輛是屬於[[德國|德國]]鐵路車輛製造商[[Adtranz|ADTranz]]（後被[[龐巴迪運輸|龐巴迪運輸]]併購）技術授權生產之車輛，同類型的尚有[[熊本市交通局9700型電車|熊本市交通局9700型電車]]、[[熊本市交通局0800型電車|熊本市交通局0800型電車]]、[[岡山電氣軌道9200型電車|岡山電氣軌道9200型電車]]、[[萬葉線MLRV1000型電車|万葉線1000型電車]]（俗稱「{{lang|ja|アイトラム}}」）和[[富山地方鐵道9000型電車|富山地方鐵道9000型電車]]（俗稱「{{lang|ja|セントラム}}」）。

外裝是以[[立山_(日本)|立山]][[雪|新雪]]為意象的白色基調設計、各車車頭擋風玻璃下方及乗降口周圍分別以[[彩虹|彩虹]]七色其中一色點綴。

車輛型號定為0600型、車体上標記的車号「TLR」為「'''T'''oyama '''L'''ight '''R'''ail」之開首，編號末数字依彩虹色順1至7編號。末尾字母為列車[[集電弓|集電弓]]展開方向，印有字母A的車輛之集電弓指向[[岩瀨濱站|岩瀨濱站]]、B則是指向[[富山站|富山站北站]]。

列車[[愛稱|愛稱]]是於2005年公開募集所選的「'''ポートラム'''（PORTRAM）」，為「港」（＝port）與「路面電車」（＝tram）組成的組合詞。

官方[[吉祥物|吉祥物]]角色[[富山縣立富山北部高等學校|縣立富山北高校]]信息設計系學生設計，以貓與電車為題的「とれねこ」（Train Cat）。並與編組数相同有七隻不同特征與性格的角色，而每輛列車均有其對應之吉祥物。<ref>開業記念時には先頭車両ガラス部分に、それぞれのマスコットキャラクターのシールが貼られていた。また乗車人数100万人達成時にも同様のシールが貼られている。</ref>

== 運用 ==
在尖峰時段富山港線也不會出動全部七列編成運行，而是輪替運營使用。當初預定引入六組列車，後於運營企畫階段提出引入七列列車，原因是「建議採用數字7，因為既是彩虹顏色的數字、也是七福神的數字、會更具吸引力」，最終獲採納改導入七列編成。

== 車輛諸元 ==
車輛全長18.4m、寬2.4m、高3.407m。車重約25t。[[軌距|軌距]]因路線自JR承接同樣採用1067mm。
* 設計最高速度：70km/h
* 營業最高速度：60km/h
* [[起動加速度|起動加速度]]：3.0km/h/s
* 減速度：4.0km/h/s（最大），4.6km/h/s（緊急）
