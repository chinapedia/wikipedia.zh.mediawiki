'''InFocus M810'''智慧型手機是[[InFocus|InFocus]]的品牌手機，由[[富智康集團|富智康集團]]代工製造。正反面都有[[大猩猩玻璃|大猩猩玻璃三代鏡面抗刮玻璃]](藍寶石特仕版則在背面採用藍寶石玻璃，邊框為銀色)並且有防指紋塗層。邊框是[[鋁合金|鋁合金]]設計，電源鍵的外型採用腕錶錶冠的設計，機身厚度僅6.99公厘。InFocus經由[[亞馬遜公司|亞馬遜公司]]在[[互联网|互联网]]的售價是14999[[印度盧比|印度盧比]]<ref>[http://www.moneydj.com/funddj/ya/yp050000.djhtm?a={FF0F017F-96B7-493F-AA07-9CFF4240A6DD} 槓上小米、華碩！鴻海夥伴InFocus M810/M2接力攻印]</ref>。

== 規格 ==
*處理器：[[Qualcomm|Qualcomm]] [[Snapdragon|Snapdragon]] 801 2.5GHz 四核心處理器
*系統內存：16GB ROM
*記憶體：2GB RAM
*記憶卡：micro sd, 最高64GB
*作業系統：[[Android|Android]] 4.4.2 KitKat 
*手機介面：InLife UI
*顯示屏幕：5.5 吋螢幕
*電池容量：2600 mAh不可拆卸的電池
