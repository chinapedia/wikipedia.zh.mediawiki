'''富澤站'''（{{jpn|j=富沢駅|hg=とみざわえき|rm=Tomizawa eki|f=y}}）是一個位在[[宮城縣|宮城縣]][[仙台市|仙台市]][[太白區|太白區]]的[[車站|車站]]。唯一通過的路線是[[仙台市地下鐵南北線|仙台市地下鐵南北線]]，且為該路線之終點站。富澤站的南方有地下鐵的機廠，列車在此處轉向。
[[File:ZaruGawaTomizawaEki2005-4.jpg|thumb]]

== 車站構造 ==
為一擁有[[島式月台|島式月台]]1面2線的[[高架車站|高架車站]]。車站南邊面對著[[名取川|名取川]]的支流[[笊川|笊川]]。其月台位於車站三樓。

== 車站週邊 ==

*仙台大野田郵便局
*仙台市水道局

== 瑣事 ==
*開業前的名稱為泉崎站。
*因為位在[[仙台市體育館|仙台市體育館]]的前方，所以在站名下加註了'''*仙台市體育館前'''的字樣。
*[[秋保電氣鐵道|秋保電氣鐵道]]原本有一同名車站，但該車站後來改名為[[西多賀站|西多賀站]]。
