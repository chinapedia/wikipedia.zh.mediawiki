'''富樫氏'''（とがしし）是以{{link-ja|藤原利仁|藤原利仁}}為始祖的氏族。[[室町時代|室町時代]]是支配[[加賀國|加賀國]]（現[[石川縣|石川縣]]南部）的[[守護大名|守護大名]]。

== 概要 ==
代代在[[加賀國|加賀國]]擴張勢力，世襲[[加賀國|加賀介]]，一族是號稱武門榮耀的[[八介|八介]]之一的[[富樫介|富樫介]]。[[家紋|家紋]]是[[八曜紋|八曜紋]]。[[替紋|替紋]]是[[鹿角紋|鹿角紋]]。

居館在現在的[[野野市市|野野市市]]<ref group="注">{{Cite web |url=http://www.city.nonoichi.lg.jp/rekishitabi/rekishi_2_5.html |title=富樫館の発掘 |publisher=野々市市 |accessdate=2013-07-28}}</ref><ref group="注">{{Cite web |url=http://tiikijiten.jp/~digibook/tanbou/navi.php?file=b_couse/b_26.php |title=富樫館跡 |publisher=ののいち歴史探訪 |accessdate=2013-08-03}}</ref>。[[勸進帳|勸進帳]]有名的富樫左衛門（[[富樫泰家|富樫泰家]]）也是出自此富樫氏。

1335年（[[建武_(後醍醐天皇)|建武]]2年），[[富樫高家|富樫高家]]因追隨[[足利尊氏|足利尊氏]]討幕有功而就任加賀國守護職之後，代代擔任加賀守護職。但是，1387年（[[至德_(日本)|至德]]4年），[[富樫昌家|富樫昌家]]歿後，[[管領|管領]][[斯波氏|斯波氏]]奪取加賀守護職將近30年。1414年（[[應永|應永]]21年），以[[征夷大將軍|將軍]][[足利義持|足利義持]]為後盾，奪回加賀守護。

1488年（[[長享|長享]]2年），[[富樫政親|富樫政親]]在[[高尾城_(加賀國)|高尾城]]被[[加賀一向一揆|加賀一向一揆]]消滅（長享一揆），富樫氏的加賀國支配時期告終。一揆以[[富樫泰高|富樫泰高]]作為名目上的守護。

之後，泰高之孫・[[富樫稙泰|富樫稙泰]]捲入[[本願寺|本願寺]]的內紛[[大小一揆|大小一揆]]，兵敗被剝奪守護的地位，富樫家勢力更加的衰退。稙泰之孫[[富樫家俊|家俊]]仕於[[佐久間盛政|佐久間盛政]]，改名後藤彌右衛門，使富樫家存續<ref group="注">{{Cite web |url=http://www.city.nonoichi.lg.jp/rekishitabi/rekishi_2_7.html |title=押野村の後藤家 |publisher=野々市市 |accessdate=2013-07-28}}</ref>。子孫作為[[加賀藩|加賀藩]]的[[十村制|十村肝煎]]直到[[明治維新|明治維新]]。
