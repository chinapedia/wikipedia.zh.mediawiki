'''富士急高原乐园'''（{{jpn|j=富士急ハイランド|hg=ふじきゅうハイランド|rm=Fujikyuu Hairando}}，{{lang-en|Fujikyu Highland}}、又稱「'''{{lang|en|Fuji-Q}}'''」），是一個位於[[日本|日本]][[山梨县|山梨县]][[富士吉田市|富士吉田市]]的[[主題樂園|主題樂園]]，临近[[富士山|富士山]]山脚。

== 歷史 ==
富士急乐园前身是1961年开业的'''富士五湖国际滑冰中心'''（{{lang|ja|富士五湖国際スケートセンター}}），同年[[富士急行|富士急行]][[河口湖线|河口湖线]]的高原站（现为[[富士急高原乐园车站|富士急高原乐园车站]]）开业，所以改名为富士急乐园。主要的附带设施有专业保龄球场「Q-BOWL」（2012年更新开业），度假设施「富士急高原乐园度假温泉酒店」（1986年开业，旧称酒店·高原休养地），富士山题材的美术品的[[美术馆|美术馆]]「fujiyamamyujiamu」（2003年开馆），热水浴设施「紫藤山温泉」（2006年开业）等。

富士急乐园以其拥有很多高刺激性过山车有名。从2000年开始，也陆续加入以家庭为客层目标的游乐设施。事实上，在日本高刺激性游乐设施在主题乐园中不断地减少，只有富士急乐园和{{link-ja|长岛温泉乐园|ナガシマスパーランド}}比较知名，被称为“东有富士急，西有长岛”。据说富士急乐园中高刺激性游乐设施较多，也是因为富士急行的社长堀内光一郎为爱好者。<ref>{{Cite web|url = http://allabout.co.jp/contents/sp_oyako_c/datespot/CU20060710A/index/|title = AllAbout『超絶コースター"ええじゃないか”"試乗レポ』|accessdate = 2009-11-25|deadurl = yes|archiveurl = https://web.archive.org/web/20091213023825/http://allabout.co.jp/contents/sp_oyako_c/datespot/CU20060710A/index|archivedate = 2009-12-13}}</ref>1996年“FUJIYAMA”登场以后，为了不断刷新[[吉尼斯世界纪录|吉尼斯世界纪录]]，樂園开始每隔5年就新建大型高刺激性游乐设施。2011年7月16日登场的「高飞车」完成总计9项加演节目14项的吉尼斯记录。一些老旧的设施会被逐渐拆除。除了高刺激性游乐设施，乐园内也有许多其他主题游乐设施，比如托马斯天地、哈姆太郎乐园等，其他主题游乐设施也在不断地登场。

在山梨县外，乐园一般被称作“富士急高原乐园”或“富士急”，广告中红色的富士“Q”也具有很高的识别度。从[[东海道新干线|东海道新干线]][[新富士車站_(靜岡縣)|新富士站]]的发出的快速巴士中，前往乐园的巴士的罗马字标记为“FOR FUJIKYU”。在山梨县的富士北麓地区和静冈县东部等地区，富士急行线沿线及富士急行的巴士运行区域里，“富士急”一般指铁道与巴士，乐园则多被称呼为“高地”。

2011年2月24日，乐园开设了[[Facebook|Facebook]]官方主页，并不断地发布新资讯。现在乐园亦开设了[[mixi|mixi]]和[[LINE_(应用程式)|Line]]的主页。

2014年3月27日发表了新的方针，以后高刺激性游乐设施将安装在同一系列的主题乐园{{link-ja|相模湖森林游乐园|さがみ湖リゾート プレジャーフォレスト}}。<ref>{{Cite web|url = http://www.nikkei.com/article/DGXNASDZ2504R_W4A320C1TJ2000/?n_cid=TPRN0004|title = 富士急、絶叫マシンを相模湖へ　富士山世界遺産指定で|accessdate = 2014-03-27}}</ref>

== 交通 ==
{{Unreferenced section|time=2015-03-24T21:30:57+00:00}}
=== 铁路 ===
[[文件:Fujikyu_6000_Series.jpg|thumb]]]]

[[富士急高原乐园车站|富士急高原乐园车站]]是[[富士急行|富士急行]][[富士急行线|富士急行线]]上的铁路车站，与乐园第二入口直接相连接。搭乘电车是前往乐园的方式之一，该站以富士急高原乐园命名。该站舍除了「托马斯乐园号」（{{lang|ja|トーマスランド号}}）等其他普通列车使用以外，还有来自JR[[中央本线|中央本线]]的[[直通列车|直通列车]]、在大月站-河口湖站之间运行的全景特急列车“[[富士山特急|富士山特急]]”。

=== 巴士 ===
乐园内巴士车站分为富士急高原乐园与高原巴士终点站两个车站。

=== 高原巴士终点站 ===
富士急行巴士终点站（{{lang|ja|ハイランドバスターミナル停留所}}）位于乐园南侧的第一停车场，富士急高原乐园度假温泉酒店前。一般停靠公共巴士以及旅游巴士，有时也会停靠周游巴士。

高速巴士“富士急高原乐园”、“富士急高原乐园（高速公交车站）”以及周游巴士“富士急高原乐园度假温泉酒店”、“富士急高原乐园巴士终点站”一般都指这个车站。

=== 富士急高原乐园巴士车站 ===
富士急高原乐园巴士车站（{{lang|ja|富士急ハイランド停留所}}）位于乐园北侧的第二停车场（富士急高原乐园车站前）的[[公交站|巴士车站]]，一般停靠普通公共巴士。

== 广告巴士 ==
