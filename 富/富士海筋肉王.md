'''富士海筋肉王'''（[[日語|日語]]：{{lang|ja|海筋肉王 ～バイキング～}}，簡稱'''《{{lang|ja|海筋肉王}}》'''或'''《{{lang|ja|バイキング}}》'''或'''《VIKING》）是[[日本|日本]][[富士|富士]][[電視台|電視台]]所舉辦的運動娛樂特別節目。

此節目與[[極限體能王SASUKE|極限體能王SASUKE]]幾近相同，每集製作成本超過一億元日幣。第一集在2005年3月播出。由於節目頗受好評，在2005年8月還推出了親子版的富士海筋肉王。最後一次播出是在2006年10月播出的個人版第四回大會。

富士海筋肉王不只體力要好，還要有智力以及冷靜的判斷力，才能過關。現在此節目除了日本、[[台灣|台灣]]有播出外，在[[美國|美國]]、[[法國|法國]]播出時也有不錯的評價。許多外國人也可以參加比賽，為國爭光。

== 摘要 ==

此節目的規則是，每集會有100位參賽者出場（也有些特例，像個人版第一回只有80人參賽，親子版分別是70、90、80組親子），闖過一道道難關，不過只要沒在限制時間內闖關成功或是中途落水都當場出局。共分四個舞台及18道關卡。只要是過了第一[[舞台|舞台]]，就可以繼續挑戰第二舞台，以此類推，個人版全破就可拿到200萬元日幣。（相當於6.70萬元新台幣），親子版全破就可以拿到由[[Honda|Honda]]汽車贊助的車款。

個人版比賽的流程是一個人闖過關卡，像極限體能王一樣。值得一提的是，親子版的流程是一對親子各自闖自己的難關，對於孩子會有比較簡單的關卡（例如跳箱、二位數的[[加法|加法]]）等，只要最後兩人能在限制時間內一起按下按鈕，即算過關。親子版的第一舞台例外，即使沒能在限制時間內趕上，但是只要是前十五名，方可進入第二舞台。個人版沒有人能完全稱霸，不過親子版在第一、二回大會[[大路弘文|大路弘文]]和[[大路海斗|大路海斗]]親子曾經拿下兩次全破，除此之外第二回[[佐藤弘道|佐藤弘道]]和[[佐藤久哉|佐藤久哉]]親子也拿下了全破。

個人版第一回播出時，沒有人闖進第三舞台，創下本節目的最低紀錄。[[極限體能王SASUKE|極限體能王SASUKE]]之中的[[極限體能王全明星|極限體能王全明星]]也有挑戰的經驗，全都留下不遜色的成績。值得紀念的是個人版第二回大會，[[長野誠|長野誠]]只差一隻手的距離就可以達成全破，但是最後還是體力用盡，由此可見，此節目有一定的難度。

== 最新關卡名稱（親子版） ==
* 第一舞台限制時間145秒，第二舞台130秒。
=== 第一舞台 ===
<span style="color: Green;">'''⒈ランデブーラン（攜手跑步）'''</span>
: 親子各分左、右，通過一些奇形怪狀的浮島。
<span style="color: Green;">'''⒉スーパーダイビング（水中尋寶）'''</span>
: 前所未聞的水中關卡，一位選手拉起手把，再由另一位選手鑽過直徑60公分的洞，取得高速滑軌用的竿子。
<span style="color: Green;">'''⒊壁登り（繩索攀爬）'''</span>
: 抓住繩索爬上1.9公尺的牆壁。默契不好的選手都相繼在這裡失敗。
<span style="color: Green;">'''⒋高速スライダー（高速滑軌）'''</span>
: 利用取得的竿子架設在滑軌上，快速滑下至一不穩定的浮島。
<span style="color: Green;">'''⒌跳び箱（障礙跳箱）'''</span>
: 由一位參賽者按下按鈕開關，再由另一位選手跳過八層的跳箱，中間有水坑必須越過。
<span style="color: Green;">'''⒍パーレイ（海盜小店）'''</span>
: 兩位數的加法。在這裡先記住答案往下一關卡前進。
<span style="color: Green;">'''⒎ライフボート（激流勇士）'''</span>
: 坐上橘色的救生艇滑下水道。翻船就當場出局，記下的答案也全都泡湯。如果通過了就將剛剛記下來的答案打在答案欄上。
<span style="color: Green;">'''⒏宿題ロード（家庭作業）'''</span>
: 小孩必須從高蹺、跳跳樂及獨輪車選其中一項，藉由這些工具通過60公分的平衡橋。
<span style="color: Green;">'''⒐ラストロープ（最後之網）'''</span>
: 水面上有一個網子載浮載沉，必須越過這個網子朝下一關前進。
<span style="color: Green;">'''⒑サバイバルクライム（逃生爬行）'''</span>
: 爬上35度的斜坡。到了¾處時變成平地，最後的¼斜坡有架設繩子。
=== 第二舞台 ===
<span style="color: Blue;">'''⒒ペアバンク（雙人水潭）'''</span>
: 類似「女子版體能王」的斜面走。
<span style="color: Blue;">'''⒓親離れ子離れ（各奔東西）'''</span>
: 在一個中空的圓圈上面藉由兩公尺的橡皮繩個在兩端通過。
<span style="color: Blue;">'''⒔ハンマーロード（槌子道路）'''</span>
: 在一獨木橋上通過左右搖晃的槌子。
<span style="color: Blue;">'''⒕２人乗り自転車（雙人腳踏車）'''</span>
: 藉由一腳踏車騎過20公分的獨木橋。考驗默契及平衡感。
<span style="color: Blue;">'''⒖ジャングルロープ（叢林奇譚）'''</span>
: 抓住繩子然後盪到對岸。類似「極限體能王」的泰山跳躍。放手的時機很重要。
<span style="color: Blue;">'''⒗ウォーターウォール（瀑布之牆）'''</span>
: 類似「女子體能王」的暴風道。瀑布流下來的水會讓手沾濕，對於下一個關卡非常不利。留下來的時間進入第三關卡。
=== 第三舞台 ===
<span style="color: Red;">'''⒘脫出ポール（逃生棒子）'''</span>
: 用第二舞台所剩下的時間，爬上棒子向上到達終點。
=== 最終舞台 ===
<span style="color: Purple;">'''⒙ファイナルバイキング（最終戰書）'''</span>
: 一位選手計算數學關卡，再由另一位通過雲梯。如果計算順利的話就會過得很輕鬆。限制時間為90秒，此大會沒有任何人通過這一關。

<span style="color: Green;">'''オデッセイ（巨大羅盤）【1.2.3】'''</span>

: 越過一個類似羅盤的東西，羅盤會搖晃，所以平衡感也很重要。女性脫落者多。

<span style="color: Green;">'''ローディングドック（滾動棧橋）【1.2.3】'''</span> 

: 從三個不同的地方落下來七公斤的沙包，接住其中一個之後度過一個細細的橋。

<span style="color: Green;">'''キャスタウェイ（劫後餘生）【1.2.3】'''</span> 

: 從一個浮島上跳躍抓住金球，再滑行10公尺。第四回起移植至第三舞台。

<span style="color: Green;">'''キャノンボール（砲彈飛車）【1.2.3】'''</span> 

: 4個大球兩端用繩索固定，必須從上面快速通過。第四回起移植至第三舞台。山田勝己首次出場時在此出局，波比歐羅剛連續兩次在此出局，脫落者多。

<span style="color: Green;">'''スイングマスト（搖擺桅杆）【1.2.3】'''</span> 

: 必須自己用繩子纏住前面上方的木桿，然後盪過一水池。極限體能王全明星之中的山本及秋山有在此脫落的經驗。

<span style="color: Green;">'''キャプテンキャビン（風車船長）【1.2.3】'''</span> 

: 直徑5.5M的巨型舵在旋轉，舵上有突起物必須往上爬。

: 從1.8M高處往下跳到一不穩定的浮島，然後抓著繩子渡過8M到達對岸。
