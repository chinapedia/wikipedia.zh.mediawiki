'''庐剧'''，又名'''倒七戏'''，中国[[戏曲|戏曲]]剧种之一，流行于[[安徽省|安徽省]][[合肥市|合肥市]]及其周边的[[六安市|六安市]]、[[巢湖市|巢湖市]]、[[芜湖市|芜湖市]]等地。“庐剧”得名于合肥的古称“庐州”。<ref>[http://www.hefei.gov.cn/n1070/n304695/n433310/n1765531/1765663.html  旅游服务>>合肥风情 >>地方戏曲 > 庐剧 ]{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>
2006年5月，庐剧被列为中国第一批[[国家级非物质文化遗产|国家级非物质文化遗产]]。<ref>[http://www.luan.gov.cn/common/isearch_view.php?filename=08061909534998 关注非物质文化遗产之皖西庐剧 ]{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref>[http://news.sohu.com/20060608/n243632250.shtml 首批国家级非物质文化遗产名录公布 庐剧上榜]</ref>

== 艺术特色 ==

庐剧唱腔分为主调和花腔两大类。主调主要用于整本戏与折子戏，有二凉、三七、寒腔、老生调、小生调、老旦调等調門(曲牌)，长于叙事或抒情；花腔則用于小戏。其中三七、寒腔是廬劇的代表性調門，被應用的最廣最常用。

庐剧的唱腔有几个明显的特点。一是在唱腔中不断出现用假声演唱的旋律，称做小嗓子。二是演唱中的帮腔吆台，即当舞台上的演员唱到一定的时候，由场面和后台的演员齐声帮唱，高亢辽阔，烘托剧情，渲染舞台气氛。其乡土味很浓，风格明朗。<ref name="godpp"></ref>

庐剧的伴奏乐器有锣鼓、高胡等，以[[高胡|高胡]]为主胡。<ref>李大武.庐剧音乐伴奏及主胡的演奏特点.《安徽新戏》，2001年03期.</ref>

庐剧的表演程式多借鉴自其他剧种，如[[京剧|京剧]]、[[徽剧|徽剧]]等。庐剧行当有[[花旦|花旦]]、[[小生|小生]]、[[青衣|青衣]]、[[老旦|老旦]]、[[老生|老生]]、[[小丑|小丑]]等。<ref name="godpp"></ref>

== 流派 ==
庐剧因地域特色分为西路、中路、东路三路。西路（亦称上路）以[[六安|六安]]为中心，音乐上较为高亢；中路以[[合肥|合肥]]为中心，音乐上兼有东路与西路的特点，朴实明快；东路（亦称下路）以[[芜湖|芜湖]]为中心，音乐细腻婉转。<ref>[http://www.china.com.cn/culture/zhuanti/whycml/2007-06/06/content_8353137.htm 国家级非物质文化遗产名录-庐剧]</ref>

== 历史 ==
庐剧的形成年代尚无定论，但至少可追溯到清代中期。<ref name="sina">[http://news.sina.com.cn/o/2006-03-31/09568578457s.shtml 庐剧走过两百年衰荣时光]</ref>早期庐剧班社多为半职业性的，无固定的角色体制，表演较为粗糙，主要靠唱来刻画人物。这一时期的化装亦非常简单，服装则借自生活服装，如花旦上穿花布短褂，下穿[[裙子|裙子]]，小生则穿蓝色大褂，其他剧种常用的[[水袖|水袖]]也付阙如，生角旦角均由男性扮演。从1920年代开始，庐剧与其他剧种搭班演出，开始借鉴其他剧种来丰富自己的表演，剧目不断丰富，逐渐走向成熟。<ref name="godpp">[http://www.godpp.gov.cn/wmzh/2007-08/13/content_11076630.htm 第一批非遗目录——庐剧 ]</ref>

1949年后，在政府力量主导下，对庐剧进行了一系列改革，如建立导演制，革新舞台美术等。一系列国有剧团相继成立。1955年3月，成立了国营剧团安徽省庐剧团。此时庐剧得到了迅速的发展。[[文革|文革]]时，庐剧同其他戏曲剧种一样，受到严重的冲击。文革后，庐剧一度出现复兴，庐剧演出也得到了恢复。但是，由于种种原因，庐剧跟其他戏曲剧种一样，陷入了衰落之中。<ref>[http://travel.anhuinews.com/system/2007/07/30/001807597_01.shtml 庐剧：二百多年几度衰荣]</ref>

== 剧目 ==
庐剧剧目可分为可分为花腔小戏、折戏和本戏三类，代表性剧目有《[[彩楼配|彩楼配]]》、《[[秦雪梅|秦雪梅]]》、《[[药茶记|药茶记]]》、《[[柴斧记|柴斧记]]》、《[[借罗衣|借罗衣]]》、《[[打芦花|打芦花]]》、《[[雪梅观画|雪梅观画]]》等。<ref>[http://www.newta.com.cn/public.xp?doAction=view&id=10210&siteid=0001&menuid=0005 安徽名戏——庐剧] {{webarchive|url=https://web.archive.org/web/20150607062454/http://www.newta.com.cn/public.xp?doAction=view&id=10210&siteid=0001&menuid=0005 |date=2015-06-07 }}</ref>

== 著名人物 ==
著名庐剧演员有[[黄冰|黄冰]]、[[武克英|武克英]]、[[丁玉兰|丁玉兰]]、王本银、孙邦栋、董少轩等，其中黄冰、武克英被列为国家级非物质文化遗产传承人。<ref>[http://news.qq.com/a/20080306/002078.htm 安徽22人当选“非遗”传承人]</ref><ref>[http://www.ah.xinhuanet.com/zhuanti/2007-09/05/content_11065237.htm 庐剧玉兰花 ] {{webarchive|url=https://web.archive.org/web/20070910162149/http://ah.xinhuanet.com/zhuanti/2007-09/05/content_11065237.htm |date=2007-09-10 }}</ref><ref name="godpp"></ref>

當代著名的庐剧演员有[[盛小五|盛小五]]、[[周小五|周小五]]、[[朱德顺|朱德顺]]、[[刘长芬|刘长芬]]、[[孙晓云|孙晓云]]、[[魏小波|魏小波]]、[[汪莉|汪莉]]、[[夏巧云|夏巧云]]，尤其[[盛小五|盛小五]]以演出反串角色嶄頭露角，且身兼多樣行當，戲路寬廣，不管是男的女的，老的少的，耍寶的正經的，君子痞子他都能演，網評：「盛小五演喜劇讓人笑的要命、演悲劇又能讓人哭得要死」，因此倍受廣大戲迷歡迎。<ref>[http://www.anhuixiqu.com/2011/hmxyy_0708/662.html 戏曲名家 盛小五]</ref>

== 团体 ==
庐剧的主要演出团体有[[合肥市庐剧院|合肥市庐剧院]]、[[皖西庐剧团|皖西庐剧团]]等。<ref>[http://www.hefei.gov.cn/n1105/n32756/n175331/n175808/n176193/1359211.html 合肥市庐剧院]{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref>[http://www.ah.xinhuanet.com/xinwen/2003-06/20/content_626691.htm “西路庐剧”大旗不倒] {{webarchive|url=https://web.archive.org/web/20050310172210/http://www.ah.xinhuanet.com/xinwen/2003-06/20/content_626691.htm |date=2005-03-10 }}</ref>
