'''豬牙花'''（[[学名|学名]]：''{{lang|la|Erythronium japonicum}}''），是[[百合科|百合科]][[豬牙花屬|豬牙花屬]]的一種[[多年生植物|多年生植物]]，原產於[[日本|日本]]，[[韓國|韓國]]和[[中國|中國]]東北部。

== 特徵 ==
多年生[[草本植物|草本植物]]，株高約16～20[[公分|公分]]，群生於[[落葉植物|落葉性]]的闊葉樹下。地下部生有[[鱗莖|鱗莖]]。葉淡綠色，上面有紫色斑紋。葉長10～11公分，葉寬2.5～6.5公分，葉柄長3～4公分。早春時開花，[[花|花]]單生，花淡紫色至桃紅色。[[種子|種子]]上生有一種稱為[[油質體|油質體]]的肉質器官，[[螞蟻|螞蟻]]很喜歡食用這種物體，會檢拾種子帶回蟻巢，豬牙花的種子也藉由螞蟻的搬運散佈至其他地方。

== 用途 ==
豬牙花在日本稱為'''片栗'''，它的鱗莖含有[[澱粉|澱粉]]，可以加工作成片栗粉[http://ja.wikipedia.org/wiki/%E7%89%87%E6%A0%97%E7%B2%89] （日本太白粉）。片栗粉的用法和[[太白粉|太白粉]]一樣，一般都是作為料理勾芡之用或是用來增加食物的黏稠度，在炸[[天婦羅|天婦羅]]時也會使用片栗粉。日本現在的片栗粉多是用[[馬鈴薯|馬鈴薯]]加工製造而成，因為馬鈴薯的成本比較便宜。真正的片栗粉反而比較少見，而且價格也比較昂貴。

花可供觀賞，盆栽或種植於庭園花壇中。春天時，在野外也可以看到經過人工復育的豬牙花一大片群生開花的景象。

== 象徵 ==
豬牙花是日本[[富山縣|富山縣]][[高岡市|高岡市]]的市花。
