'''《修女也疯狂》'''（Sister Act）是一部由[[美国|美国]][[试金石影业|试金石影业]]于1992年发行的[[喜剧|喜剧]]电影。本片由埃米利·阿朵里诺（[[:en:Emile_Ardolino|Emile Ardolino]]）执导，并由马克·施艾曼（[[:en:Marc_Shaiman|Marc Shaiman]]）担任音乐制作。电影讲述了由[[乌比·戈德堡|乌比·戈德堡]]饰演的在[[内华达州|内华达州]][[雷諾_(內華達州)|雷諾市]]一酒吧担任驻唱歌手目睹了黑道老闆殺人情狀，而上了老板的必杀名单，不得不避入修道院假装修女的故事。本片同时还有[[玛吉·史密斯|玛吉·史密斯]]、[[哈維·凱托|哈維·凱托]]（Harvey Keitel）、玛丽·威克斯（[[:en:Mary_Wickes|Mary Wickes]]）、比尔·纳恩（[[:en:Bill_Nunn|Bill Nunn]]）、凯西·奈吉米（[[:en:Kathy_Najimy|Kathy Najimy]]）和温迪·麦凯奈（[[:en:Wendy_Makkena|Wendy Makkena]]）等人参与演出。《修女也疯狂》在由[[美国精彩电视台|美国精彩电视台]]评选的“史上最搞笑电影”中排名第83位。

《修女也疯狂》的票房表现，让其续集《[[修女也疯狂2|修女也疯狂2：旧习难改]]》迅速提上制作日程，并在1993年上映。同时《修女也疯狂音乐剧》于2006年在美国[[加州|加州]][[帕萨迪纳_(加利福尼亚州)|帕萨迪纳]]的帕萨迪纳剧场完成首演；2009年5月7日，音乐剧在伦敦西区的伦敦白金剧院预演；2011年3月24日，《修女也疯狂音乐剧》在[[百老汇|百老汇]]的[[百老匯劇院|百老匯劇院]]完成预演，并最终在当年8月正式上演。

== 剧情概要 ==
故事开始于1968年，在一所天主教学校内，一名名叫德劳瑞斯的女孩故意用愚蠢的答案回答修女老师的提问。很快场景便切换到现在，德劳瑞斯·卡地亚在[[内华达州|内华达州]][[雷諾_(內華達州)|雷诺市]]一间名叫“月光酒吧”的赌场担任驻唱歌手。这间赌场由她的男朋友、黑道首领文斯·拉罗卡开设。一次突发事件中，德劳瑞斯目睹了文斯杀人，因此文斯命令他的手下乔伊和威利将德劳瑞斯处理掉。艾迪·少泽警官建议她指证文斯，并保证在开庭前为她提供证人保护。

德劳瑞斯被带到[[旧金山市|旧金山市]]附近的一间名叫“圣凯瑟琳”的天主教教堂，并被建议伪装成修女躲避追杀，而知道这一切只有少泽警官、教堂主教奥哈拉、與教堂女院長。女院長因急需警方的捐獻以修建教堂，勉強答應毫無信仰的德劳瑞斯化名“玛丽·克劳伦斯”，成为了院中的一个修女。在简单枯燥的修女生活中，德劳瑞斯与乐观的玛丽·帕特里克修女、文静的玛丽·罗伯特修女，以及严肃的玛丽·拉匝禄修女成了好朋友。由于德劳瑞斯后来进入酒吧，被教堂女院長惩罚进入唱诗班，担任唱诗班教练一职。而此时的唱诗班由于成员间缺乏协调性而陷入困境。在一次主日弥撒的表演中，德劳瑞斯将[[摇滚乐|摇滚乐]]和[[福音音乐|福音音乐]]结合起来，演出了歌曲《万福圣母》（Hail Holy Queen）。

教堂女院長被德劳瑞斯的离经叛道所激怒，但是奥哈拉主教却对摇滚福音将街上的叛逆青年吸引进教堂而感到兴奋。唱诗班修复了教堂和社区居民的关系，为社区带来了改变。而在少泽隊長光临的那次布道会上，唱诗班演唱了《我的主》（My God）。随后奥哈拉主教宣布，[[教皇|教皇]][[若望·保禄二世|若望·保禄二世]]将莅临教堂，欣赏唱诗班的演出。与此同时，由于感到自己的权威被无意间破坏，教堂女院長宣布辞职。作为文斯的贿赂对象，泰德警官发现了德劳瑞斯的去向并将行踪报告给文斯。于是文斯派遣乔伊和威利将德劳瑞斯抓回来。少泽中尉发现了泰德的背叛，于是将其逮捕。他赶往旧金山警告德劳瑞斯，但此时德劳瑞斯已经落入乔伊和威利的手里。

修女们在教堂女院長的带领下，赶往雷诺市去拯救德劳瑞斯。而此时德劳瑞斯已经摆脱了乔伊和威利，在赌场内和他们玩起了捉迷藏。在修女们找到她之后，她决定带着她们偷偷地逃出赌场。文斯、乔伊和威利最后捉住了德劳瑞斯和修女们，但少泽隊長及时赶到拯救了她们并逮捕了文斯等人。在影片的最后，德劳瑞斯指挥修女唱诗班，为教皇献上了歌曲《我将追随他》(I Will Follow Him)。曲毕，教皇、教堂女院長及全场观众为唱詩班献上掌声。

通过的电影的片尾显示，德劳瑞斯将她的修女生涯出售给了媒体，引发了一系列轰动。

== 制作花絮 ==
编剧保罗·鲁德尼克在1987年向制片人斯科特·鲁丁推荐了《修女也疯狂》，并且认为应该由[[贝蒂·米勒|贝蒂·米勒]]担任主角。这个剧本随后出售给了[[华特迪士尼影片|华特迪士尼影片]]。<ref name="new yorker">{{cite web |url=http://www.newyorker.com/reporting/2009/07/20/090720fa_fact_rudnick |title=Fun With Nuns|accessdate=July 26, 2009 |author= |last=Rudnick |first=Paul |coauthors= |date=2009-07-20 |publisher=[[纽约客|纽约客]] |pages=37–41 }}</ref> 但是米勒最后推辞了演出，她担心她的影迷无法接受她的修女造型。<ref name="new yorker"/> 最后，主角由[[乌比·戈德堡|乌比·戈德堡]]担纲演出。随着电影的制作，编剧团大约更换掉了一打的编剧，这其中包括：卡莉·费希尔（Carrie Fisher）、罗伯特·哈林（Robert Harling）和[[南希·迈耶斯|南希·迈耶斯]]。<ref>[http://www.imdb.com/title/tt0105417/trivia Sister Act—Trivia]</ref> 由于最终的剧本与鲁德尼克的原始剧本已经大相径庭，因此鲁德尼克被要求更换剧本署名，最终他选择了“约瑟夫·霍华德”这个化名。<ref name="new yorker"/>

影片中的“圣凯瑟琳教堂”实际上是位于诺伊丘区内山丘路和教堂路交叉处的圣保禄天主教堂，诺伊丘区是旧金山一个较富裕的[[中产阶级|中产阶级]]社区。教堂对面的街道做了修改，使其看上去像是[[犹太人|犹太人]]社区。

== 反响评论 ==
=== 票房 ===
《修女也疯狂》在票房成绩上大获成功，美国本土收获票房近1.4亿美元，非美国本土票房则高达9200万美剧，最终在全球范围内斩获票房2.3亿美元，成为1992年全球票房纪录第八名。<ref name="boxofficemojo">[http://www.boxofficemojo.com/movies/?id=sisteract.htm Sister Act as BoxOfficeMojo.com]</ref>

=== 争议 ===
1993年6月10日，美国联合艺人娱乐公司签约女艺人唐娜·道格纳斯和她的伴侣柯特·威尔逊正式起诉[[迪士尼|迪士尼]]、[[乌比·戈德堡|乌比·戈德堡]]、[[贝蒂·米勒|贝蒂·米勒]]及其制作公司和创意艺人经纪社，指责其电影《修女也疯狂》抄袭威尔逊作品《橱柜里的修女》，并求偿2亿美元。道格拉斯和威尔逊声称他们在1985年就完成了该书的剧本改编，并在1987年到1988年期间，分别将剧本三次投递给迪士尼、乌比·戈德堡和贝蒂·米勒。他们宣称《修女也疯狂》和《橱柜里的修女》在剧情上至少有100处以上的相似之处。<ref>{{Citation| last = Haring| first = Bruce|publisher = [[名利场|名利场]]| title =$200 mil suit targets 'Sister Act'| date =1993-06-10| url=http://www.variety.com/article/VR107715| accessdate = 2012-01-02}}</ref>

唐娜·道格纳斯和柯特·威尔逊在1994年下降了他们的赔偿金请求，要求赔偿1亿美元。而法官相对倾向[[华特迪士尼影片|华特迪士尼影片]]及其他被告人，威尔逊当时宣称“他们不得不逐字逐句的复制我的剧本”。<ref>{{Citation| last = Friend| first = Tad| publisher = [[纽约客|纽约客]]| title = Copy Cats: Hollywood Stole My Story!| date = September 1998| url = http://byliner.com/tad-friend/stories/copy-cats-hollywood-stole-my-story| accessdate = January 2, 2012| deadurl = yes| archiveurl = https://web.archive.org/web/20120318192423/http://byliner.com/tad-friend/stories/copy-cats-hollywood-stole-my-story| archivedate = 2012年3月18日}}</ref>

在2011年11月，一名叫做德劳瑞斯·布莱克尼的修女起诉[[华特迪士尼影片|华特迪士尼影片]]和[[索尼影视娱乐|索尼影视娱乐]]，声称《修女也疯狂》是改编自其于1987年完成的自传《哈林街修女》。她宣称在电影监制表示改编的兴趣之后，她写了三页的概要给电影公司。她因此起诉上诉两家公司“违反合同、肖像挪用及不当得利”。<ref>{{Citation| last=Gardner| first=Eriq| publisher=[[好莱坞报道|好莱坞报道]]| title=Harlem Nun Sues Disney, Sony Claiming They Stole 'Sister Act': File under, "Divine Inspiration."| date=2011-11-110| url=http://www.hollywoodreporter.com/thr-esq/sister-act-nun-disney-sony-260274| accessdate=2012-01-03}} </ref>

=== 对华语地区的影响 ===
同對地，《修女也瘋狂》也影響到[[華語地區|華語地區]]的人士，開始挑選電影任一部分當作改編的參考題材。像是，2004年3月，[[S.H.E|S.H.E]]於第六張專輯《[[奇幻旅程|奇幻旅程]]》的宣傳期時，導演賴偉康在構想《安全感》的[[音乐录影带|MV]]內容當中，挪用[[琥碧·戈柏|琥碧·戈柏]]在電影中的修女形象，作為歌手在MV當中的造型之一
<ref>{{cite news|author=中安娛樂|title=S.H.E扮修女拍MTV 比拼誰的高跟鞋最美(圖)|url=http://entertainment.big5.anhuinews.com/system/2004/03/10/000587433.shtml|accessdate=2012-04-07|newspaper=中安娛樂|date=2004年3月10日}}</ref> <ref>{{cite news|author=NOWnews 今日新聞網|title=新歌MV搞笑演出　S.H.E扮修女也瘋狂|url=http://www.nownews.com/2004/03/09/37-1598680.htm|accessdate=2012-04-06|newspaper=NOWnews 今日新聞網|date=2004年3月9日}}</ref>。
另外，2005年9月24日、25日，[[蔡琴|蔡琴]]在[[北京工人體育館|北京工人體育館]]公開上演的小型[[歌舞剧|歌舞剧]]《跑路救天使》，也是參照《修女也瘋狂》而改編的<ref>{{cite news|author=[[北京娱乐信报|北京娱乐信报]]|title=带歌舞剧再进京 蔡琴：我从不考虑去拍影视剧|url=http://news.xinhuanet.com/ent/2005-07/30/content_3286640.htm|accessdate=2012-04-01|newspaper=[[新华网|新华网]]|date=2005年7月30日}}</ref>。

== 光碟发售 ==
《修女也疯狂》[[DVD區域碼|第一区]][[DVD|DVD]]于2001年11月6日发行，类似[[博伟|博伟]]早期发行的DVD光碟，该DVD并没有对电影的[[失真|失真]]进行修复。碟片中的[[彩蛋_(視覺)|彩蛋]]内容包含：
# 《修女也疯狂》的剧场版宣传片；
# 《I Will Follow Him》和《If My Sister's In Trouble》两首电影[[插曲|插曲]]的[[音乐录影带|音乐录影带]]以及包含这两首歌曲的电影剪辑视频；
# 制作花絮《Inside Sister Act: The Making Of》。
