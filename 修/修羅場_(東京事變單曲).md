《'''修羅場'''》（{{lang|ja|修羅場}}，''Shuraba''／''The Rat's-nest''）是日本樂團[[東京事變|東京事變]]的第三張[[單曲|單曲]]，由[[EMI音樂_(日本)|東芝EMI]]／[[維京唱片|Virgin Music]]於2005年11月2日發行<ref name="JPN" />，同時也是二期東京事變的首張作品。在跨業合作部分，歌曲「修羅場」更成為日本[[富士電視台|富士電視台]]連續劇‘[[大奧_(富士電視劇)#大奧-華之亂|大奧~華之亂]]’的主題曲<ref name="ShurabaDRAMASONG" />。初回限定包含《[[DOMESTIC!_Virgin_LINE|DOMESTIC! Virgin LINE]]》演唱會門票優先抽選劵。

在單曲銷售成績方面，本張單曲在日本Oricon銷售榜上最高位居單週第5名<ref name="ShurabaOriconWeek" />，名列2005年年度銷售榜第125位<ref name="2005salessingle" />。歌曲「修羅場」在日本告示牌所公布的告示牌百強單曲榜中，最高曾名列單週第3名<ref name="BillboardChart" />。在銷售認證上，單曲《修羅場》銷售突破10萬張，在2005年11月通過日本唱片協會認定為金唱片<ref name="RIAJ200511" />。而歌曲「修羅場」的手機和電腦下載也分別在2007年7月及2012年1月通過日本唱片協會認定為金唱片<ref name="RIAJdigital200707" /><ref name="RIAJdigital201201" />。

== 單曲概要 ==
《'''修羅場'''》是東京事變第三張單曲，同時也是繼2004年10月發表《[[遭難|遭難]]》後，睽違一年的新單曲。不過，在發行作品前，東京事變於2005年7月1日在官方網站上宣布，由於樂團負責彈奏鋼琴的{{link-ja|H是都M|H ZETT M}}及吉他手{{link-ja|畫海幹音|ヒラマミキオ}}宣布退團，樂團東京事變將進行重組，並且希望歌迷能夠期待第二期東京事變的來到<ref name="OUT1">{{cite web |url=http://www.vibe-net.com/news/?news=0023287 |title=東京事変　メンバー2人が脱退 |publisher=vibe-net |language=ja |deadurl=yes |archiveurl=https://web.archive.org/web/20141129073747/http://www.vibe-net.com/news/?news=0023287 |archivedate=2014-11-29 }}</ref><ref name="OUT2">{{cite web |url=http://www.emimusic.jp/tokyojihen/news/ |title=東京事変セカンドシーズン突入 ( 2005/07/01 UP) |publisher=EMI |language=ja}}</ref>。

在經過兩個月的空窗期後，東京事變於09月16日在官方網站上宣布[[浮雲_(吉他手)|浮雲]]及[[伊澤一葉|伊澤一葉]]將加入樂團，同時還表示將發行新單曲<ref>{{cite web|url=http://www.emimusic.jp/tokyojihen/news/ |title=11/2発売　ニューシングル｢修羅場」のタイアップが決定しました。 ( 2005/09/16 UP) |publisher=EMI |language=ja}}</ref>。而本張單曲《修羅場》正是東京事變在浮雲和伊澤一葉加入後的首張作品。

歌曲「修羅場」是椎名林檎應戲劇「大奧」製作人的邀約下所完成的。為了完成歌曲，椎名林檎甚至跑去看前二季的戲劇內容，並表示「女主角的美麗與魄力真是無人可比，讓我忍不住寫下『竟會有所懷疑簡直太過膚淺』這句歌詞<ref name="ShurabaINTERVIEW">{{cite web|url=http://inews.iwant-radio.com/archives/10688 |title=女人心修羅場 東京事變唱出女性犀利心聲 |publisher=銀河娛樂新聞網 |language=zh-tw}}</ref>。」而單曲的封面是「椎名林檎手拿著一瓶打著馬賽克的香水橫躺<ref name="ShurabaINTERVIEW" />」，而在日後發行的專輯《大人》中，它的專輯封面則是那罐香水的瓶子，香水外殼的瓶子則是寫上專輯的英文名子「ADULT」，讓歌迷不禁聯想單曲和專輯似乎有種特別的關係。而團員看到單曲封面的樣子，還開玩笑的說：「如果是拿著小酒瓶的話更好。」這樣的回應讓椎名林檎哭笑不得<ref name="ShurabaINTERVIEW" />。此外，椎名林檎以東京事變的名義，首次將歌詞本寫上英文，這與椎名林檎在歌唱時期用日文拼英文的記法有很大的不同。

在單曲發售時程上，本張單曲首先於2005年9月27日在官方網站上發布單曲封面<ref>{{cite web|url=http://www.emimusic.jp/tokyojihen/news/ |title=「修羅場」ジャケ写 ( 2005/09/27 UP) |publisher=EMI |language=ja}}</ref>。而在戲劇‘大奧~華之亂’首播日前一日的10月12日，不但開始接受歌曲「修羅場」音樂錄影帶的視聽外，同時也開放手機下載的服務<ref name="JPNRingtone" />。最後，11月2日在日本發行單曲與提供數位下載<ref name="JPN" />，台灣則於11月4日同步發行進口版與台壓盤<ref name="TWN" />。

最後，在單曲銷售成績方面，
本張單曲在發行當週以5.9萬張的銷售量在日本Oricon銷售榜上位居第5名<ref name="ShurabaOriconWeek" />，總計在Oriocn銷售榜上榜15週<ref name="ShurabaOriconConsecutiveWeek" />，累積銷售達11.0萬張，名列2005年年度銷售榜第125位<ref name="2005salessingle" />。同時，單曲銷售突破10萬張，在2005年11月通過日本唱片協會認定為金唱片<ref name="RIAJ200511" />。而歌曲「修羅場」的手機和電腦下載也分別在2007年7月及2012年1月通過日本唱片協會認定為金唱片<ref name="RIAJdigital200707" /><ref name="RIAJdigital201201" />。

=== 音樂錄影帶 ===
[[Image:Tokyojihen01-03-mv.jpg|thumb]]
導演番場秀一將「修羅場」的音樂錄影帶以白色為主軸，因此所有東京事變的團員全部都被漆成白色，搭配上紅色鮮血的效果與偶爾穿插主唱椎名林檎帶著假髮望著前方發呆的神情，讓人不禁驚嘆他們不僅對音樂的要求甚高，對於音樂錄影帶的視覺效果的呈現也完全不馬虎。另外，眼尖的歌迷還發現到音樂錄影帶在後段，即開始變色的時候，隱約出現「映日紅之花」音樂錄影帶的片段。

== 批評及評價 ==
音樂網站''CD Journal''的評論家認為這首龐克搖滾歌曲「修羅場」堅定的表現出椎名林檎是個出色的作曲家，評論家同時也表示因為在專輯版本中採用電吉他彈奏，所以收錄在專輯的版本比單曲版本更能表現出搖滾的模樣<ref name="CDJournal">{{cite web | url=http://artist.cdjournal.com/d/-/4105090624 |title=東京事変 / 修羅場 |publisher=CDJournal |language=ja}}</ref><ref>{{cite web | url=http://artist.cdjournal.com/d/adult-special-edition-pour-homme/4105101781 |title=東京事変 / 大人(アダルト) |publisher=CDJournal |language=ja}}</ref>。同時，評論家也提到「落日」這首歌曲僅用鋼琴彈奏，伴隨著椎名林檎優美的歌聲，這樣的風格是東京事變少數的抒情歌曲，同時強烈的展現出椎名林檎歌唱實力的作品<ref name="CDJournal" />。

== 銷售排行 ==
=== Oricon 銷售榜(日本) ===
《修羅場》於2004年10月20日發行。發行首週，在日本公信榜Oricon的單曲榜2005年10月第5週付的榜單中以58,659張銷售量居單週第五名。同週第一名是銷售量突破50萬大關的期間限定歌唱團體[[修二與彰|修二與彰]]的單曲「[[青春Amigo|青春Amigo]]」、第二名則是演唱二人組[[WaT|WaT]]的第二張單曲「{{link-ja|我的心意|僕のキモチ}}」<ref name="ShurabaOriconWeek" />。第二週則位居十位，同週第一、二名分別是修二與彰的單曲「青春Amigo」及兩人樂隊[[柚子_(組合)|柚子]]的單曲「[[超特急/旭日東昇|超特急/旭日東昇]]」<ref>{{Cite web |url=http://www.oricon.co.jp/rank/js/w/2005-11-21/ |title=2005年11月第1週の邦楽シングルランキング情報 |publisher=Oricon |language=ja}}</ref>。總計《修羅場》總共在日本公信榜Oricon的單曲榜上榜15週，累積銷售達11.0萬張<ref name="ShurabaOriconConsecutiveWeek" />。同時在2005年年銷售榜中，《修羅場》以8.5萬張銷售量居全年第125名<ref name="2005salessingle" />。
