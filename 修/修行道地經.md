'''修行道地經'''（梵語：Yoga-cārya-bhūmi），印度[[說一切有部|說一切有部]]著名[[譬喻師|譬喻師]]、大[[禪師|禪師]][[僧伽羅剎|僧伽羅剎]]（Saṃgharakṣa）所作的[[止觀|禪觀]]論著，收於《[[大正藏|大正藏]]》經集部。內容論禪觀修道，前二十七品為[[止觀|止觀]]方法，後人增附後三品，談[[大乘佛教|大乘]][[菩薩|菩薩]]禪。本經綜合了[[瑜伽|瑜伽]]觀行之要義而纂成<ref name="香光"/>。

== 題解 ==
本經一開始即-{云}-：「榆迦遮復彌經，晉-{云}-修行道地」。「榆迦遮復彌」，梵語Yogācārabhūmi。義譯為「瑜伽行地」、「瑜伽師地」。這一名稱，為禪觀集的通名<ref name="說一切有部"/>。

== 傳譯 ==
本經由[[竺法護|竺法護]]所譯出，原本僅二十七品，後三品為後人把同為竺法護所譯的《三品修行經》增附於原本而成三十品<ref name="說一切有部"/><ref>《[[出三藏記集|出三藏記集]]》：「三品修行經，（小字注）安公-{云}-近人合大修行經。……修行經七卷二十七品。（小字注）舊-{云}-修行道地經 ，太康五年二月二十三日出。」</ref>。異譯本有後漢[[安世高|安世高]]譯《道地經》一卷、後漢支曜譯《小道地經》一卷，僅為竺法護譯本的一部分<ref name="香光">{{cite web|author1=佛書解題（香光版）|title=修行道地經|url=http://www.gaya.org.tw/library/readers/guide-xxddj.htm}}</ref>。

== 內容 ==
本經說明禪觀的次第進展，可歸納為入門階段[[不淨觀|不淨觀]]、[[慈心觀|慈心觀]]、[[數息觀|數息觀]]、觀[[十二因緣|十二因緣]]、觀[[無常_(佛教)|無常]]的[[五停心觀|五停心觀]](主要為[[奢摩他|奢摩他]]範疇)<ref>
《修行道地經》卷2：「行者情欲熾盛，為說人身'''不淨'''。……瞋怒而熾多者，為說'''慈心'''。……設多愚癡，當'''觀十二因緣'''。……設多想念，則為解說'''出入數息'''。……修行道者，<u>設多憍慢</u>，為說此義：……當作此計：城外塜間，棄捐骨鎖，頭身異處，無有血脈，皮肉消爛，當往<u>'''觀此'''貧富貴賤、男女大小、端正醜陋，枯骨正等，'''有何殊別'''</u>？本末終時，肉衣、皮裹、血潤、筋束，衣服、香花、瓔珞其身，譬如幻化巧風所合，因心意識周旋而行，至於城郭、國邑、聚落，出入進止。作是觀已，無有憍慢。……見於塜間及一切人，等而無異。」<br/>
中阿含57經：「彼有此五習法已，復修四法，云何為四？修惡露令斷欲、修慈令斷恚、修息出息入令斷亂念、<u>修'''無常想'''令斷我慢</u>。 」<br/>
雜阿含815經：「有比丘修不淨觀斷貪欲，修慈心斷瞋恚，修無常想斷我慢，修安那般那念斷覺想。」<br/>
增支部9集1經：「比丘在這五法上住立後，應該更進一步修習四法：應該為捨斷貪而修習不淨、應該為捨斷惡意而修習慈、應該為斷絕尋而修習入出息念、應該為根除我是之慢而修習無常想。」
</ref>。接著說明[[止觀|止與觀]]二者的必要性。又說明觀[[四諦|四諦]][[十六行相|十六行相]](為[[毗婆舍那|毗婆舍那]]範疇)，斷除[[煩惱_(佛教)|煩惱]]，得[[解脫|解脫]]的次第<ref name="香光"/>。

計分三十品︰集散、五陰本、五陰相、分別五陰、五陰成敗、慈、除恐怖、分別相、勸意、離顛倒、曉了食、伏勝諸根、忍辱、棄加惡、天眼見終始、天耳、念往世、知人心念、地獄、勸悅、行空、神足、數息、觀、經學地、無學地、無學、弟子三品修行、緣覺、菩薩。

== 考證 ==
[[印順法師|印順法師]]認為僧伽羅剎的原作，僅有[[偈頌|偈頌]]，頌義古樸，近於[[譬喻師|譬喻師]]舊說，長行為後人解說，增入[[阿毘達磨|阿毘達磨]]新義。每品先讚頌後結偈的形式，也應是改編者所增入<ref name="說一切有部">{{cite book|author1=印順|title=說一切有部為主的論書與論師之研究－第二項 修行道地經|url=http://yinshun-edu.org.tw/en/book/export/html/3326}}</ref>。

後三品說明[[聲聞乘|聲聞]]弟子、[[緣覺|緣覺]]、[[菩薩|菩薩]]的修行，會三乘而歸一乘，無論在形式內容上，都與前二十七品不合。[[印順法師|印順法師]]主張此為融攝「[[法華經|法華經]]」、「[[般若經|般若經]]」的意旨而為後人附於本經之中，非僧伽羅剎所作，應當即是同為竺法護所譯出的《三品修行經》<ref name="說一切有部"/>。

[[賴鵬舉|賴鵬舉]]主張《三品修行經》來自於四世紀後葉以前的「中亞禪法」。他認為在整體禪法結構上，三品修行經以「[[般若|般若]][[實相|實相]]觀」為究竟，又有二品提及「現在定意見[[十方佛|十方佛]]」三昧，正是中亞大乘禪法的二大特色 <ref>{{cite journal|author1=賴鵬舉|title=中亞的禪法|journal=圓光佛學學報|date=1997|page=63-78|url=http://enlight.lib.ntu.edu.tw/FULLTEXT/JR-AN/an333512.pdf}}</ref>。
