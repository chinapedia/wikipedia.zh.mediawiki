《'''戰國外傳 桶狹間戰記'''》是[[宮下英樹|宮下英樹]]所著的[[日本歷史|日本歷史]]系列漫畫。此為宮下戰國系列外傳，全5冊。

==故事簡介==
以兩名[[桶狹間之戰|桶狹間之戰]]總指揮官為作品主角。描述今川義元由出生到成為大名及年輕時的織田信長情況。

==登場人物==
===今川家===
*[[今川義元|今川義元]]
: 通稱'''五郎'''，後稱'''治部大輔'''。主角之一，幼時跟隨[[太原雪齋|太原雪齋]]學習，別名'''梅岳承芳'''，被喻為「'''唐鏡賜子'''」。父親[[今川氏親|氏親]]五男，安排他出家。從家督鬥爭中取勝，成為今川氏當主。外型是參考[[傑米羅奎爾|傑米羅奎爾]]而來。
*[[太原雪齋|太原雪齋]]
: 別名'''九英承菊'''，仕官於義元時稱為'''崇孚'''，後來成為了今川義元的軍師，義元亦稱其為師父，被喻為「'''今川家的黑衣宰相'''」。在[[小豆坂之戰|小豆坂之戰]]曾擊退織田軍，但亦提醒義元織田家實力不能輕視。促成戰國史上的外交奇蹟「[[甲相駿三國同盟|甲相駿三國同盟]]」的大功臣。
*[[壽桂尼|壽桂尼]]
: 義元的母親，希望義元能夠當個僧侶安然度過一生。會輔佐今川家的政務，作品中稱其為「唯一的女性戰國大名」。作為本作品起始與結束的象徵人物。
*[[今川氏親|今川氏親]]
: 通稱'''修理大夫'''，義元的父親，[[今川假名目錄|今川假名目錄]]的制定者。
*[[今川氏輝|今川氏輝]]
: 氏親的長子，義元的同母兄，自尊心很強烈，但身體很不好。
*[[今川彥五郎|今川彥五郎]]
: 氏親的次子，義元的同母兄，以輔佐兄長氏輝為責。
*[[今川良真|今川良真]]
: 氏親的三子，義元的異母兄，別名'''玄廣惠深'''，有著豪膽的性格。跟弟弟義元競爭繼承家督，引起[[花倉之亂|花倉之亂]]。
*[[太年尼|太年尼]]
: 雪齋的姊姊。
*[[飯尾長門守|飯尾長門守]]
: 義元側近的文官。
*[[朝比奈泰能|朝比奈泰能]]
: 通稱'''備中守'''，義元的重臣之一，在[[小豆坂之戰|小豆坂之戰]]有著不錯的表現。
*[[朝比奈泰朝|朝比奈泰朝]]
: 通稱'''備中守'''，義元的重臣之一，泰能之子。雪齋隱退後，與三浦義就共同輔佐義元。
*[[三浦義就|三浦義就]]
: 通稱'''左衛門尉'''，義元的重臣之一。雪齋隱退後，與朝比奈泰朝共同輔佐義元。
*[[岡部元信|岡部元信]]
: 通稱'''五郎兵衛'''，今川家的猛將。在[[小豆坂之戰|小豆坂之戰]]時，於雪齋麾下打擊織田軍表現活躍而出名。後來連敵對的織田信長都表示「能夠看穿我方計策的，在今川家中除了義元本人跟雪齋外，就屬岡部元信。」。
*[[松井宗信|松井宗信]]
: 通稱'''兵部少輔'''，今川家的武將。義元侵攻[[尾張國|尾張國]]時擔任先鋒。
*[[孕石光尚|孕石光尚]]
: 通稱'''郷左衛門'''，今川家的武將。

===織田家===
*[[織田信長|織田信長]]
: 通稱'''三郎'''，後稱'''上總介'''。主角之一，幼名吉法師，在家臣眼中是個問題兒童。
*[[織田信秀|織田信秀]]
: 通稱'''彈正忠'''。忙於與今川家及同族之間的鬥爭。
*[[平手政秀|平手政秀]]
: 通稱'''中務丞'''。信秀的家老，以指導信長為使命。

===松平家===
*[[松平廣忠|松平廣忠]]
: 通稱'''次郎三郎'''。由於相繼遭到同族背叛，變得依賴今川義元。後來甚至稱義元為「大哥」，而義元也都喚他為「廣少爺」。
*[[松平元康|松平元康]]
: 幼名'''竹千代'''，廣忠的長子。幼時遭今川家與織田家互相搶奪作為人質，最後被今川家得手，也使松平家正式從屬於今川家。很受到義元的喜愛，名字中的'''元'''便是接受今川義元的偏諱而來，也受過太原雪齋的指導。個性較為血氣方剛。

===其他勢力===
====武田家====
*[[武田晴信|武田晴信]]
: 通稱'''大膳太夫'''，[[甲斐國|甲斐國]]的大名。
*[[山本晴幸|山本晴幸]]
: 通稱'''勘助'''，武田家的軍師，擅長外交與築城。

====北条家====
*[[北条氏康|北条氏康]]
: 通稱'''左京太夫'''，[[相模國|相模國]]的大名。
*[[北条幻庵|北条幻庵]]
: 連續輔佐北条家五代的政務，北条家的軍師。

====長尾家====
*[[長尾景虎|長尾景虎]]
: 通稱'''彈正少弼'''，[[越後國|越後國]]的大名。在與武田晴信的[[川中島之戰#第二次（犀川之戰）|第二次川中島之戰]]處於膠著的狀態下接受今川義元的調停，但他感受到義元的行動中有受到遠在[[駿河國|駿河國]]的太原雪齋之影響。

====齋藤家====
*[[齋藤道三|齋藤道三]]
: 通稱'''山城入道'''，[[美濃國|美濃國]]的大名。看過織田信長假裝瘋癲實為聰慧的行為舉止後，認為他自己以後的子孫會為信長做牛做馬。
*[[齋藤義龍|齋藤義龍]]
: 通稱'''新九郎'''。與父親道三不同，在外交立場上與信長保持敵對狀態。
