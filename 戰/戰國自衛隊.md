『'''戰國自衛隊'''』（せんごくじえいたい）是[[半村良|半村良]]的[[科幻小說|SF小說]]和1979年以此為原作，由[[千葉真一|千葉真一]]主演的[[#電影|電影]]，[[票房|票房]]達到23億圓<ref name = "興行収入">{{Cite web |url = http://www.generalworks.com/databank/movie/title4/sengo.html |title = 作品データ「戦国自衛隊」 |work = 作品データ |publisher = SF MOVIE DataBank |language =ja<!--|archiveurl = http://megalodon.jp/2012-0429-0809-38/www.generalworks.com/databank/movie/title4/sengo.html |archivedate = 2012-04-29--> |accessdate = 2012-04-29 |deadlinkdate =  }}</ref>。還有[[#劇畫|劇畫]]、2005年的電影『[[戰國自衛隊1549|戰國自衛隊1549]]』、2006年的[[電視劇|電視劇]]『[[#電視劇|戰國自衛隊・關原之戰]]』的[[重製|重製]]版。

== 原作 ==
「以近代兵器武裝的現代軍隊」和「以弓矢或刀武裝的戰國時代鎧武者軍團」作戰的話將會如何？」的[[架空戰記|架空戰記]]。

== 電影 ==
電影版本於1979年12月5日在日本首映，由[[千葉真一|千葉真一]]主演兼武術指導。並於1980年8月28日至9月4日期間在香港及澳門的金公主院線上映<ref>[https://www.facebook.com/permalink.php?story_fbid=117628638922510&id=117441668941207 香港電影票房 1980 [外語電影]]</ref><ref>本港新聞第三張第四頁.華僑日報.1980年8月28日</ref>，並以粵語配音播出。
