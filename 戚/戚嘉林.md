'''戚嘉林'''（{{bd|1951年||||}}），生於[[台灣|台灣]]，[[籍貫|籍貫]][[湖北|湖北]][[沔陽|沔陽]]，曾任[[中華民國|中華民國]][[外交官|外交官]]，[[退休|退休]]後從事教學，現為[[國際關係|國際關係]]與歷史學者，專長於國際關係與[[台灣研究|台灣研究]]，現為[[世新大學|世新大學]]國際關係兼任助理教授、[[中國統一聯盟|中國統一聯盟]]主席、[[台灣抗日志士親屬協進會|台灣抗日-{志}-士親屬協進會]]顧問、《[[祖國文摘|祖國文摘]]》發行人。

==生平==
戚嘉林畢業於[[輔仁大學|輔仁大學]]商學系，[[中國文化大學|中國文化大學]]經濟研究所碩士，南非[[比勒陀利亞大學|比勒陀利亞大學]]國際關係學博士。中華民國[[外交領事人員特考|外交領事人員特考]]及格，曾於[[阿根廷|阿根廷]]、[[新加坡|新加坡]]與[[南非|南非]]等地駐外館處任職外交官。退休後以業餘史學愛好者身分從事教職，在世新大學通識教育中心教授[[台灣史|台灣史]]。2007年8月，戚嘉林出版《台灣史》一書，書中帶有強烈的[[中國民族主義|中國民族主義]]，認為[[中華民國總統|中華民國前總統]][[李登輝|李登輝]]阻礙了中國統一，並懷疑他可能是日本警官的後代而非李金龍之子<ref>{{Cite web |url=http://www.nownews.com/n/2013/04/08/296188|language=zh-tw|author=林修卉 |title= 戚嘉林：李登輝對統一大業 造成深遠負面影響|publisher=[[今日新聞網|今日新聞網]]|date=2013-04-08 |accessdate=2013-04-08}}</ref>。

2010年9月18日，戚嘉林參與[[中華民國教育部|中華民國教育部]]在[[高雄中學|高雄中學]]舉行的「普通高級中學歷史科課程綱要(草案)公聽會」，在會中登記發言，反對[[台灣主體性|台灣主體性]]，主張將[[中國民族主義|中國民族主義]]納入教科書<ref>{{cite news |language = zh-hk |author=李仲維|url =http://hk.crntt.com/doc/1014/5/0/3/101450317.html | title =台民間社團：盼教科書具中華民族歷史觀點| publisher =中國評論新聞網| date =2010年9月18日 | accessdate =2013年5月30日}}</ref>。

2012年8月11日，戚嘉林參與[[兩岸和平發展論壇|兩岸和平發展論壇]]在[[台北市|台北市]]舉辦的「當前亞太形勢下的兩岸關係和平發展」研討會，在會中提出報告〈南海問題升高的原因及其影響〉，以歷史及[[法理|法理]]依據，支持中華人民共和國擁有[[南沙群島|南沙群島]]的主權，並認為中華人民共和國堅決捍衛[[黃岩島|黃岩島]]主權的強硬姿態可以有效遏止台灣[[分離主義|分離主義]]的發展<ref>{{Cite web |url=http://hk.crntt.com/doc/1021/9/8/3/102198313.html |language=zh-hk|author=倪鴻祥 |title= 戚嘉林：大陸維護主權強硬 警惕台分離主義|publisher= 中國評論新聞網|date=2012-08-12 |accessdate=2012-08-12}}</ref>；全文刊載於《[[中國評論|中國評論]]》月刊2012年9月號<ref>{{Cite web|url=http://hk.crntt.com/doc/1022/2/3/0/102223041.html|language=zh-hk|author=戚嘉林 |title=南海爭端升高的原因及影響|publisher=中國評論新聞網|date=2012-09-27|accessdate=2014-09-08}}</ref>。

2013年3月，戚嘉林參與[[海峽兩岸關係研究中心|海峽兩岸關係研究中心]]在[[福建|福建]][[平潭|平潭]]舉辦的第11屆兩岸關係研討會，主張推動國族認同應與兩岸政治對話互利融合同步，應在台灣推動中國民族主義，讓台灣人認同自己是[[中國人|中國人]]，以利中國統一<ref>{{Cite web |url=http://hk.crntt.com/doc/1024/7/9/2/102479241.html|language=zh-hk|author=王宗銘|title=戚嘉林：讓台灣人認同自己是中國人|publisher= 中國評論新聞網|date=2013-03-23|accessdate=2013-03-23}}</ref>。

2013年4月，由[[監察委員|監察委員]][[周陽山|周陽山]]與[[李炳南|李炳南]]提出的[[228事件|228事件]]史料調查報告中，引述戚嘉林發言，記錄李登輝為日本人[[私生子|私生子]]、且曾加入[[中國共產黨|中國共產黨]]。《[[中國時報|中國時報]]》據此報導<ref>{{Cite web |url=https://tw.news.yahoo.com/%E7%9B%A3%E9%99%A2%E5%A0%B1%E5%91%8A-%E6%9D%8E%E7%99%BB%E8%BC%9D%E7%94%9F%E7%88%B6%E6%98%AF%E6%97%A5%E6%9C%AC%E4%BA%BA-213000599.html |language=zh-tw|author=楊毅|title=監院報告 李登輝生父是日本人|publisher=中國時報|date=2013-04-08 |accessdate=2013-04-08}}</ref>，引起爭議；《中國時報》也同時報導了戚嘉林的回應，戚認為，他在訪談中是用[[疑問句|疑問句]]口氣，非[[肯定句|肯定句]]，是記錄人員疏失造成誤會。學者[[陳芳明|陳芳明]]引述李登輝本人的澄清，認為他只加入讀書會、未正式加入中國共產黨，對戚嘉林的研究可靠性表示質疑<ref>{{Cite web |url=http://hk.crntt.com/doc/1024/9/5/0/102495029.html |language=zh-hk|author=黃筱筠 |title= 陳芳明：李登輝早釐清過沒參加共產黨|publisher= 中國評論新聞網|date=2013-04-08 |accessdate=2013-04-08}}</ref>；李登輝本人接受訪問時，也否認戚嘉林的說法；李登輝辦公室主任[[王燕軍|王燕軍]]則以「[[滑稽|滑稽]]」形容這份報告，認為完全不值得回應<ref>{{Cite web|url=http://www.ettoday.net/news/20130408/189248.htm|author=政治中心|language=zh-tw|title=李登輝生父是日本人？ 監院報告挨批「滑稽」|publisher=[[東森新聞雲|東森新聞雲]]|date=2013-04-08|accessdate=2014-09-08}}</ref>。2013年8月6日，監察委員[[錢林慧君|錢林慧君]]向李炳南轉達李登輝的女兒[[李安妮|李安妮]]的[[陳情|陳情]]，周陽山與李炳南決定將學者發言從報告中略除；2013年8月25日，李炳南說，當初調查案本身與李登輝身世本來就沒有直接關係，但顯然已對李登輝及其家屬構成嚴重影響，因此他們決定修正，未來官方報告版本將不再包含這些文字<ref>{{Cite web|url=http://www.appledaily.com.tw/realtimenews/article/new/20130825/248407|author=吳家翔|language=zh-tw|title=李登輝女兒出手 監院報告刪「日本人私生子」|publisher=[[台灣蘋果日報|台灣蘋果日報]]即時新聞|date=2013-08-25|accessdate=2014-09-08}}</ref>。

2013年5月28日，戚嘉林說，台灣歷經[[漢族|漢族]]與[[台灣原住民|台灣原住民]]「四百年融合、四百年血脈」，已[[內化|內化]]為血脈同源的中國人，是現狀也是現在進行式；台獨人士大多是漢族的後代，他們的祖先之所以能來台灣墾殖，實受惠於[[明鄭|明鄭]]與[[清朝|清朝]]等漢族政權；台獨論述卻將這些漢族政權蔑稱為「外來政權」，罔顧台灣社會現實，最終只能自陷於「外來政權」的認同矛盾而無法自拔<ref>{{Cite web|url=http://www.chinatimes.com/newspapers/20130528000524-260109|author=戚嘉林|language=zh-tw|title=四百年融合 四百年血脈|publisher=中國時報|date=2013-05-28|accessdate=2014-02-03}}</ref>。

2013年7月22日，戚嘉林說，近一、二十年來，分離主義的[[皇民化教育|皇民化教育]]使台灣人民選擇性忘記台灣歷史：[[清朝|清朝]]修台灣入版圖後，台海兩岸「同屬一國」、「同屬一中」長達212年，台灣人民形塑建構並傳承「視中國為祖國」的國族認同；因此，1895年[[馬關條約|台灣割讓日本]]後，台灣人民先後以武裝和非武裝方式從事[[台灣抗日運動|台灣抗日運動]]；日本統治台灣50年間，台灣先賢都視中國為祖國，這在日本殖民當局留下的許多機密檔案中斑斑可考；1945年8月15日[[日本投降|日本投降]]，台灣全島「歡騰慶光復」表達「視中國為祖國」的國族認同；這些都是七、八十年前的台灣現代史，雖經二十年分離主義皇民化教育，仍無法抹煞<ref>{{Cite web|url=http://www.chinatimes.com/newspapers/20130722000415-260109|author=戚嘉林|language=zh-tw|title=恢復台灣民眾的祖國認同|publisher=中國時報|date=2013-07-22|accessdate=2014-09-08}}</ref>。

2014年2月24日，戚嘉林說，228事件發生在日本投降離開台灣不到2年時，當時日本留下的是破敗的台灣；而且日本殖民政府在交還台灣前，惡意製造台灣大糧荒和超級[[通貨膨脹|通貨膨脹]]，讓台灣陷入絕境，在政治、經濟、文化各種社會矛盾激化之下導致228事件，日本才是228事件的元凶<ref>{{Cite web|url=http://www.cna.com.tw/news/aipl/201402240401-1.aspx|author=陳偉婷|language=zh-tw|title=陳鵬仁：二二八事件沒有元凶|publisher=[[中央通訊社|中央通訊社]]|date=2014-02-24|accessdate=2014-02-28}}</ref>。他並認為是二二八事件是美國與日本聯手催化<ref>{{Cite web |url=http://www.chinatimes.com/newspapers/20150301000341-260109|language=zh-tw|author=戚嘉林|title=美日催化228 更應警惕|publisher= 中國時報|date=2016-03-03|accessdate=2015-03--1}}</ref>。

2014年9月8日，戚嘉林說，[[民主進步黨|民主進步黨]]一向標榜民主並以「台灣有說話的自由」為傲，但一遇到[[統獨問題|統獨問題]]就立即無限上綱至[[敵我矛盾|敵我矛盾]]，全力圍剿「台灣前途從來不是台灣人決定的」、「台灣前途就是中華民國前途，而中華民國前途就應由全體中國人決定」、「台獨不可行」等語，「除了以祭出『台灣人民的對立面』和『台灣優先』等高階政治領域話語嚴厲抨擊，甚至還扯上『[[退休金|退休金]]』這樣的低階政治領域話語，意涵以退休金威懾之意」；而回顧歷史，當代的[[世界地圖|世界地圖]]是基於歷史理由及[[地緣政治|地緣政治]]理由而決定的，台灣可說是經典案例之一；60年來台海兩岸關係與時推移地走到今天的和平發展，現今兩岸都有意願用和平來解決統一爭端，這是歷史上的第一次，台灣應把握歷史機遇，揚棄[[冷戰|冷戰]]敵我思維，致力於兩岸和平統一<ref>{{Cite web|url=http://www.chinatimes.com/newspapers/20140908000374-260109|author=戚嘉林|language=zh-tw|title=台灣前途誰說了算|publisher=中國時報|date=2014-09-08|accessdate=2014-09-08}}</ref>。

2014年10月26日，戚嘉林說，1945年10月25日，中國與日本在台灣[[台北市中山堂|台北市中山堂]]舉行“中國戰區台灣省受降典禮”，日本政府將台灣和澎湖歸還中國，「是時，全台民眾歡騰慶祝台灣回歸祖國，其原動力就是視中國為祖國的祖國意識；當時台灣人民強烈的祖國意識體現了台灣回歸祖國的正當性，所以參加『中國戰區台灣省受降典禮』的美國官員與日本前文武官員都理所當然地認為台灣屬於中國」；1946年春，台灣的日僑近30萬人及日軍16萬人被遣返日本，完成了台灣重返中國的作業程序；1946年8月，國民政府明訂以10月25日為[[台灣光復節|台灣光復節]]。他說，就因為台灣光復節有上述這樣重大的劃時代歷史意義，李登輝與陳水扁執政實施分離主義政策時就推動“去光復節”運動，不但將“台灣光復”改稱為“[[終戰|終戰]]”，並且取消原為國定假日的台灣光復節；前者是在抽象意識上細雨無聲地修改定義，否定台灣光復，也就是否認台灣歸還中國的史實；後者則是廢除具體化的紀念日及其相關紀念儀式，使台灣光復的歷史意義神形俱滅，在國族記憶中斬草除根，使人們忘掉這段歷史，欲從根本上否定台灣歸還中國的歷史；恢復台灣光復節為國定假日，也就是恢復[[公權力|公權力]]的紀念儀式，使台灣光復的歷史意義神形俱復<ref>{{Cite web |url=http://hk.crntt.com/doc/1034/4/5/9/103445928.html|language=zh-hk|author=戚嘉林|title=戚嘉林：不能忘 需恢復“台灣光復紀念日”|publisher= 中國評論新聞網|date=2014-10-26|accessdate=2014-12-14}}</ref>。

2016年11月6日，[[習洪會|習洪會]]與[[兩岸經貿文化論壇|兩岸經貿文化論壇]]順利在北京落幕後，戚嘉林表示：「對於兩岸分治應訂定落日條款，才能推動治理一體化的[[一國兩制|一國兩制]]」。<ref>{{cite news|title=台統派：兩岸攜手推和平統一才能救台灣|url=http://www.crntt.tw/doc/1044/5/7/8/104457874.html?coluid=93&kindid=15870&docid=104457874|agency=中評社|date=2016-11-06}}</ref>

2018年2月24日，[[中國統一聯盟|中國統一聯盟]]舉行新春團拜，戚嘉林表示：「我們呼籲蔡英文政府為天下蒼生計，順應歷史大勢，把握接受和平統一，一國兩制的成熟時機」。<ref>{{cite news|title=戚嘉林：在台灣講一國兩制的時機成熟了|url=http://hk.crntt.com/doc/1049/8/4/5/104984595.html?coluid=46&kindid=0&docid=104984595&mdate=0225002241|agency=中評社|date=2018-02-25}}</ref>
