'''猿島'''，是一個位於[[日本|日本]][[神奈川縣|神奈川縣]][[橫須賀市|橫須賀市]]的[[無人島|無人島]]。其四面為[[東京灣|東京灣]]所環繞，是東京灣內唯一自然形成的島嶼。

== 地勢 ==
* 面積 約5.55km<sup>2</sup>
* 標高 最高39.3m　
* 位在橫須賀市平成町東北約1.15km處，或是[[三笠公園|三笠公園]]東方約1.75km處

== 概要 ==
猿島的交通，搭乘[[京急本線|京急本線]]在[[橫須賀中央站|橫須賀中央站]]下車後，徒步及搭船約30分可達，尚稱便利。島上環境適合從事海水浴、露營、釣魚、散步等休閒活動。島內曾出土一些[[繩文時代|繩文時代]]及[[彌生時代|彌生時代]]的土器及人骨，此外也有與[[日蓮|日蓮上人]]有關的傳說流傳。

[[File:Sarushima_fort.JPG|thumb]]
從[[幕末|幕府末期]]到[[第二次世界大戰|第二次世界大戰]]前，猿島成為了東京灣中的首都防衛據點。在幕府末期，[[江戸幕府|江戸幕府]]在猿島上建立了{{link-ja|台場 (建築)||台場|台場}}（{{lang|ja|だいば}}，幕府末期為了防範外國船隻而興建的要塞），進入[[明治時代|明治時代]]以後，該要塞則為[[陸軍省|陸軍省]]及[[海軍省|海軍省]]所管轄，並興建了新的要塞及砲台。目前島內還殘存著挖空岩壁覆上[[紅磚|紅磚]]而成的要塞遺跡。

二戰後至1961年（[[昭和|昭和]]36年），猿島為[[美軍|美軍]]接收。在美軍接收期間，1947年（昭和22年）開始有渡船通航、1957年（昭和32年）開闢了海水浴場。之後在1993年（平成5年）海水浴場封閉，航錄也廢止，猿島成為禁止進入的場所。1995年（平成7年），[[橫須賀市|橫須賀市]]受到[[大藏省|大藏省]]委託，進行了步道的重新整備，也再度開始通航。2003年（平成15年），橫須賀市由中央政府接受了猿島的無償讓與。

== 交通 ==
* 在三笠碼頭（三笠公園內）搭船，約需10分鐘。渡船3月至11月為每日出航，12月至2月則只有週六日出航。
**要到三笠碼頭，可以從[[京急本線|京急本線]][[橫須賀中央站|橫須賀中央站]]徒步15分鐘，或是從JR[[橫須賀線|橫須賀線]][[橫須賀站|橫須賀站]]徒步30分鐘。
