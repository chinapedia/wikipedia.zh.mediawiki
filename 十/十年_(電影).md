《'''十年'''》（{{lang-en|'''Ten Years'''}}）是2015年12月上映的[[香港電影|香港電影]]，為5個故事構成的短篇電影合集<ref name=opinion>{{Cite web |url=http://opinion.huanqiu.com/editorial/2016-01/8425632.html |title=社评：《十年》吓唬香港社会，内地管不了 |publisher=環球網 |date=2016-01-22}}</ref>，當中包括[[郭臻|郭臻]]导演的《浮瓜》、[[黃飛鵬|黃飛鵬]]导演的《冬蟬》、[[歐文傑|歐文傑]]导演的《方言》、[[周冠威|周冠威]]导演的《自焚者》、[[伍嘉良|伍嘉良]]导演的《本地蛋》，由[[廖啟智|廖啟智]]、[[梁健平|梁健平]]、[[周家怡|周家怡]]、[[游學修|游學修]]、[[吳肇軒|吳肇軒]]等領銜主演。香港於[[2015年電影列表|2015年]]12月17日在[[百老匯電影中心|百老匯電影中心]]上映，因反映熱烈而陸續於不同戲院再上映，累計票房突破600萬港元，並獲得[[第35屆香港電影金像獎|第35屆香港電影金像獎]]最佳電影。2016年8月在[[臺灣|臺灣]]上映<ref>[http://udn.com/news/story/8/1607294 「十年」奪香港金像獎 8月台灣看得到]</ref>。該電影亦獲選為2015年{{le|香港亞洲電影節|Hong Kong Asian Film Festival|亞洲電影節}}推介電影。

該電影因內容涉及[[香港人權|香港人權]]、[[民主|民主]]、[[言論自由|言論自由]]受威脅的現況，加上涉及包含[[香港獨立|香港獨立]]、[[民族自決|自決]]等敏感政治話題，被指是[[香港社會|香港社會]]的「預言書」。[[中華人民共和國政府|北京政府]]全面封殺此片，下令[[中國大陸|中國大陸]]各大[[網路|網路]]平台禁止直播、轉播[[金馬獎|金馬獎]]及[[香港電影金像獎|香港電影金像獎]]實況，令至少千萬觀眾无法收看第35屆香港電影金像獎頒獎典禮<ref>{{cite web|url= http://hk.on.cc/hk/bkn/cnt/entertainment/20160222/bkn-20160222194520997-0222_00862_001.html?eventsection=hk_entertainment&eventid=402882b15286314801528bb952da3206| title= 金像獎未能在內地轉播　損失7位數收入|publisher=東方報業 |date=2016-02-22}}</ref><ref>{{cite web|url=http://www.voachinese.com/content/VOAWeishi-ProandCon-20160304-is-it-wise-for-china-to-ban-hong-kong-and-taiwan-film-awards-shows/3219559.html | language= | title=焦点对话：禁播香港电影奖，北京怕什么？ |publisher=美国之音 |date=2016-03-04 |accessdate= }}</ref>。

2017年，「十年電影工作室」正式籌備《十年》國際版，連結世界，讓更多世界互通的價值理念傳播更廣。消息指曾獲得[[康城金棕櫚獎|康城金棕櫚獎]]的[[泰國|泰國]]導演阿彼察邦將參與《十年泰國》。其餘參與的導演包括Wisit Sasanatieng、Aditya Assarat、Chookiat Sakveerakul 和 Chulayarnnon Siriphol。此片將由 Aditya Assarat 成立的 Pop Pictures 和 Films For Free 負責監製。而 《十年台灣》由劉嘉明成立的[[佳映娛樂國際股份有限公司|佳映娛樂國際股份有限公司]]負責監製，邀請了鄒隆娜、廖克發、勒嘎‧舒米、謝沛如和呂柏勳擔任導演。 《{{link-ja|十年日本|十年 Ten Years Japan}}》則由高松美由紀旗下的 Free Stone Productions 擔任監製，導演名單將會在釜山電影節公佈<ref>[http://bkb.mpweekly.com/%E8%97%9D%E6%96%87%E9%83%A8%E8%90%BD/%E5%8D%81%E5%B9%B4-%E5%9C%8B%E9%9A%9B%E7%89%88-%E5%BA%B7%E5%9F%8E%E5%B0%8E%E6%BC%94 【文化新鮮事】《十年》國際版 康城導演參與]</ref>。

== 創作背景 ==
策劃人導演伍嘉良指，《十年》創作意念源於2014年初完成紀錄片作品《趁還有墟》後，導演與街坊交流時談及令人欷歔的香港現況。導演希望會有更多可能性，能對改變現狀有更長遠的看法。其後接觸5名導演，了解各自關心的議題後，在幾乎沒有限制下開始製作。由於電影沒有申請[[電影發展基金|電影發展基金]]，製作成本有限，所以採用比較寫實的手法拍攝。<ref>{{cite news|url=https://thestandnews.com/culture/%E6%88%91%E5%9C%B0%E5%94%94%E4%BF%82%E6%BF%80%E9%80%B2-%E5%8F%AA%E4%BF%82%E5%A4%A0%E5%9D%A6%E7%99%BD-%E5%B0%88%E8%A8%AA%E9%9B%BB%E5%BD%B1-%E5%8D%81%E5%B9%B4-%E7%AD%96%E5%8A%83%E4%BA%BA%E4%BC%8D%E5%98%89%E8%89%AF/ |title=「我地唔係激進，只係夠坦白」專訪電影《十年》策劃人伍嘉良 |newspaper=立場新聞|date= 2015-11-20}}</ref>

伍嘉良亦指出，《十年》徹頭徹尾是一個[[實驗|實驗]]，同時是藝術作品。為引發觀眾思考整個社會的將來，團隊在製片過程中做了不少訪問，希望以不同角度去思考各持分者的想法，平衡各方的考慮。<ref>{{cite news|url= http://www.passiontimes.hk/article/01-15-2016/28160 |title= 《十年》導演及策劃人伍嘉良：獨立製作難獲支持是土地問題 |newspaper=熱血時報 |date= 2016-01-15}}</ref>

== 票房==
截至2016年1月31日，《十年》累積票房共收540萬港元。<ref>[https://web.archive.org/web/20160206061022/http://www.hkfilmart.com/weeklyboxoffice.asp?lang=cn 香港影視娛樂網：2016年1月25日－1月31日票房紀錄]</ref>

《十年》於2015年12月17日上畫時，最初只於[[百老匯電影中心|百老匯電影中心]]公映。由於票房較預期理想，至2015年12月底開始陸續獲[[娛藝|UA]]（太古城、CineMoko及MegaBox）、[[星影匯|星影匯]]及[[馬鞍山戲院|馬鞍山戲院]]上映，當中星影匯更使用四百多個座位的影院放映。雖然戲票一直供不應求，但院商表示隨著1月底農曆新年檔期到臨，需預留足夠場次予大製作，大部份戲院已相繼將其落畫<ref>{{Cite news|url = http://hk.apple.nextmedia.com/enews/realtime/20160127/54696503|title = 【《環時》加持後】《十年》再有兩院落畫 馬鞍山戲院逆流：一日開六場|last = |first = |date = 2016-01-27|work = 蘋果日報|accessdate = 2016-01-27}}</ref>。

在獲得金像獎提名後，《十年》在[[香港藝術中心|香港藝術中心]]agnes b.電影院(現稱古天樂電影院)上映，連同在[[香港國際電影節|香港國際電影節]]的唯一一場放映<ref>[https://40.hkiff.org.hk/zh/film/40126/ten-years 十年—第四十屆香港國際電影節]{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，六場門票均迅速全部售罄。<ref>{{Cite news|url =http://news.mingpao.com/ins/instantnews/web_tc/article/20160404/s00001/1459756100113|title =【《十年》膺最佳電影】藝術中心電影院下周加開5場　門票已售罄|date = 2016-04-04|work=明報|accessdate = 2016-04-04}}</ref>

在[[高先電影|高先電影]]取得發行權後，曾以減少分帳至三成以下為誘因，爭取第二輪商業放映，唯被院商以「沒有空間」為理由被拒，適逢藝術中心裝修，亦不能繼續放映。<ref>{{Cite news|url = http://www.hk01.com/港聞/14980/-十年-再公映遭拒-戲院稱-沒有空間-料暑假發行DVD|title = 《十年》再公映遭拒　戲院稱「沒有空間」　料暑假發行DVD|last = |first = |date = 2016-04-05|work = HK01|accessdate = 2016-10-02}}</ref>為了讓更多向隅的觀眾能欣賞本片，遂改以社區放映會形式繼續放映。

*《十年》獲頒金像獎後，引來一些電影業界人士公開批評，而這些批評大多來自[[建制派|建制派]]。[[建制派|建制派]]電影投資者[[林建岳|林建岳]]以電影投資人身分評論該片獲獎之消息，稱《十年》得金像獎「最佳電影」是香港電影界的不幸。他本人認為該片並沒有得到任何獎項提名，亦非最賣座電影，不具備最佳電影的質素，同時對電影人來說不公平。<ref>{{cite news|title=【金像獎】林建岳指《十年》得獎是香港電影界的不幸　稱政治綁架了專業|work=明報|url=http://news.mingpao.com/ins/instantnews/web_tc/article/20160404/s00001/1459705541641|date=2016-04-04}}</ref>香港電影商協會成員[[黃百鳴|黃百鳴]]指出得獎是「笑話」和「極大錯誤」，即使不論及政治，「一套50萬製作嘅電影，喺其他獎項一個獎都攞唔到，點可以叫最佳電影？」。認為電影人並非理性投票，對追求認真電影的投資者並不公平。但當被問及有否觀看《十年》時，其表示其從未觀看過。<ref>{{cite news| url=https://www.thestandnews.com/politics/%E9%BB%83%E7%99%BE%E9%B3%B4%E8%BD%9F-%E5%8D%81%E5%B9%B4-50%E8%90%AC%E8%A3%BD%E4%BD%9C%E9%BB%9E%E5%8F%AB%E6%9C%80%E4%BD%B3%E9%9B%BB%E5%BD%B1-%E5%86%87%E8%B5%B7%E8%BA%AB%E8%B5%B0%E4%BA%BA%E5%B7%B2%E7%B6%93%E4%BF%82%E5%BF%8D%E4%BD%8F/| title=黃百鳴轟《十年》：50萬製作點叫最佳電影　冇起身走人已經係忍住| work=立場新聞| date=2016-04-05}}</ref>
