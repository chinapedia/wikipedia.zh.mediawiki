'''十二月黨人起義''' （'''{{lang|ru|Восстание декабристов}}'''）是一場在1825年12月26日（[[儒略曆|儒略曆]]12月14日）發生，由[[俄國|俄國]]軍官率領3,000士兵針對[[帝俄|帝俄]]政府的起義。由於這場革命發生於12月，因此有關的起義者都被稱為「'''十二月黨人'''」（'''{{lang|ru|Декабристы}}'''），而這次革命發生在[[聖彼得堡|聖彼得堡]]的[[枢密院广场_(圣彼得堡)|元老院廣場]]。在1925年，[[蘇俄|蘇俄]]為了紀念這場革命發生一百週年，[[元老院|元老院]]廣場改名為「'''十二月黨人廣場'''」（'''{{lang|ru|Площадь Декабристов}}'''）。

== 背景 ==
在18世紀末，19世紀初，[[俄羅斯|俄羅斯]]受到[[歐洲|歐洲]]的[[啟蒙思想|啟蒙思想]]及[[自由主義|自由主義]]思潮的衝擊。尤其是[[法國大革命|法國大革命]]後的[[法蘭西第一帝國|法蘭西第一帝國]]对俄国的影響最為深遠。俄國在[[奥斯特里茨战役|奥斯特-{里}-茨战役]]慘敗於法國後，國內的自由主義思想更為高漲。

[[反法同盟|反法同盟]]最终打敗了拿破崙，在欧洲前線征戰的俄軍凱旋。战争期間，与法军作战的俄国官兵，看到了西歐的繁荣和先進，反觀自己的國家仍舊是落後不堪，開始對專制政府感到不滿。

== 組織 ==
1816年，有一部份俄國軍人成立了「'''救濟同盟'''」，後來在1818年改組為「'''福祉同盟'''」。但基於政見不合，這個組織在1821年分裂為以軍官彼斯捷爾（{{lang|ru|Павел Пестель}}）為首，主張[[共和制|共和制]]的「'''南方同盟'''」和希望[[君主立憲|君主立憲]]，由軍官穆拉維約夫（{{lang|ru|Никита Муравьев}}）領導的「'''北方同盟'''」。1823年春，雙方協議共同發動武裝起義。

== 經過 ==
1825年，[[沙皇|沙皇]][[亚历山大一世_(俄国)|亞歷山大一世]]駕崩。由於他膝下無嗣，本應由其二弟[[康斯坦丁·巴甫洛維奇|康斯坦丁·巴甫洛維奇]]大公繼承皇位，但在1823年，他即與一波蘭貴婦成婚而自動放棄皇位，儘管從法理上而言，康斯坦丁大公與之結婚並不影響繼承權，他之所以不願繼承皇位是因為哥哥參與了謀殺父親[[保罗一世_(俄国)|保羅]]的陰謀，不願再登上那滴血的寶座。在互相推辭了16天後，皇位繼承權落在了亞歷山大一世的三弟，[[尼古拉一世_(俄国)|尼古拉一世]]的身上。12月26日，3,000官兵屯駐在元老院廣場，開始了武裝起義。翌日，尼古拉一世調動約一萬的士兵鎮壓起義軍，並逮捕了彼斯捷爾等人。後來，在1月10日，穆拉維約夫再次發動起義，但最終亦以失敗收場。

== 影響 ==
雖然十二月黨人的起義以失敗告終，但亦敲響了俄國自由主義革命運動的鐘聲，同時亦引發大量有關自由主義的文學創作如[[列夫·托尔斯泰|列夫·托尔斯泰]]的《[[戰爭與和平|戰爭與和平]]》及[[普希金|普希金]]的詩作等。
