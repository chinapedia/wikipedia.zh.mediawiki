'''十二烷二酸'''是含有十二个[[碳|碳]]原子的[[二羧酸|二羧酸]]，结构式HOOC(CH<sub>2</sub>)<sub>10</sub>COOH。

== 性质 ==
白色粉末状或片状结晶。水溶解度小。热稳定性好。毒性低。

== 制备 ==
[[环己烷|环己烷]]与[[过氧化氢|过氧化氢]]在[[甲醇|甲醇]]中反应生成[[环己基过氧化物|环己基过氧化物]]，再经催化开环、二聚，得[[十二烷二酸二甲酯|十二烷二酸二甲酯]]，精制后，[[皂化|皂化]]，得十二烷二酸。

:[[File:Dodecanedioic_acid_synthesis.svg|315px]]

== 用途 ==
十二烷二酸与[[六亚甲基二胺|六亚甲基二胺]]缩合，可得到工程塑料[[尼龙6-12|尼龙6-12]]；十二烷二酸与醇类[[酯化|酯化]]生成的酯类衍生物，例如[[十二烷二酸二辛酯|十二烷二酸二辛酯]]、[[十二烷二酸二丁酯|十二烷二酸二丁酯]]，可作为聚氯乙烯、[[醋酸纤维素|醋酸纤维素]]、[[硝酸纤维素|硝酸纤维素]]的增塑剂，而且二辛酯还可作为喷气式发动机和燃气轮机使用的高级润滑油，或低温润滑脂的基础油料。

十二烷二酸也用作金属沉淀剂、饱和聚酯改性剂、粉末涂料以及[[大环麝香|大环麝香]]的合成原料。
