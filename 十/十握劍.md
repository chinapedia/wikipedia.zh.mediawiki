'''十握劍'''（{{jpn|j=十束剣|hg=とつかのつるぎ|rm=Totsuka no Tsurugi}}）乃[[日本神話|日本神話]]中的[[劍|劍]]，又名'''十拳劍'''、'''十束劍'''、'''十掬劍'''等，泛指長度約為十個[[拳頭|拳頭]]長度的劍，並不僅只一把，泛見於《[[古事記|古事記]]》、《[[日本書紀|日本書紀]]》等古藉。

== 天之尾羽張劍（伊都之尾羽張劍） ==
{{Main|天之尾羽張}}
神世七代以降的眾神之父[[伊奘諾尊|伊奘諾尊]]，因妻子[[伊奘冉尊|伊奘冉尊]]生下火神[[迦具土|迦具土]]時遭灼傷而亡，盛怒之下便以[[天之尾羽張|天之尾羽張]]斬下迦具土的頭顱。那時從劍鋒滴下的[[血|血]]誕生了八個神，據說這些神祇都和劍的鍛造有關。後來伊奘諾尊在[[黃泉|黃泉國]]被八個[[豫母都志許賣|泉津醜女]]追趕時，亦揮動此劍而逃。

== 天羽羽斬劍 ==
{{Main|天羽羽斬}}
《[[古事記|古事記]]》上卷記載，遭到諸神自[[高天原|高天原]]放逐的素戔嗚尊流落至[[出雲國|出雲]]後，趁著八岐大蛇酒醉昏睡之際以此劍砍殺之。但此劍竟出現磨損，甚感奇怪的素戔嗚尊遂剖開蛇尾，獲得[[天叢雲劍|天叢雲劍]]（{{lang|ja|あまのむらくも の つるぎ}}）<ref>亦名草薙劍（{{lang|ja|くさなぎ の つるぎ}}）。</ref>。因此，這把斬殺大蛇之'''十握劍'''稱為'''天羽羽斬'''，其中「羽羽」表大蛇之意。

== 大量劍（神度劍） ==
依據《古事記》上卷之內容，在[[葦原中國|葦原中國]]平定後，[[味耜高彥根神|阿遲鉏高日子根神]]前往弔唁[[天稚彥|天若日子]]之喪，但因二人容貌神似，天若日子之父、妻誤認阿遲鉏高日子根神為天若日子。阿遲鉏高日子根神覺得自己被誤認為死者而勃然大怒，拔出名為大量（亦名神度劍）的十拳劍切開靈堂，並將之踹往一旁。

== 大葉刈（神戶劍） ==
《日本書紀》卷第二神代下記載類似前段之故事，不同的是忿然大怒的[[味耜高彥根神|味耜高彥根神]]拔出大葉刈（又名神戶劍）的十拳劍劈開喪屋，落而爲山，成為[[美濃國|美濃國]]藍見川之上喪山。

== 布都御魂劍（韴靈劍） ==
{{Main|布都御魂}}
[[葦原中國|葦原中國]]紛亂之際，[[天照大神|天照大神]]派遣[[建御雷|建御雷神]]及[[鳥之石楠船神|天鳥船神]]下降至[[出雲國|出雲國]]伊那佐海邊，拔出十拳劍威嚇[[大國主神|大國主神]]交出葦原中國。後來[[神武天皇|神武天皇]][[神武東征|東征]]大和，無法打敗[[長髓彥|長髓彥]]，並困於熊野山中之際，受到建御雷神託夢指示的[[高倉下|高倉下]]向神武天皇獻上橫刀，隨即斬殺熊野山的凶神，受大熊之毒倒地的士卒亦隨著起身醒來。

== 其他十握劍 ==
依據《古事記》上卷之內容，[[天照大神|天照大神]]懼怕[[素戔嗚尊|須佐之男命]]侵占[[高天原|高天原]]，以生子為誓約證明。故取須佐之男命的十拳劍折成三段，以河水清洗置入口中嚼碎，並吹出霧氣變成[[多紀理毘賣命|多紀理毘賣命]]、[[市寸島比賣命|市寸島比賣命]]、[[多岐都比賣命|多岐都比賣命]]等三女神（即[[宗像三女神|宗像三女神]]）。

另一神話故事「[[山幸彥與海幸彥|山幸彥與海幸彥]]」裡，兄[[火闌降命|海幸彥]]（漁夫）與弟[[火折尊|山幸彥]]（獵人）交換各自的工具，但山幸彥不慎遺失哥哥的魚鉤，於是搗破自己佩帶的十拳劍製成5百個、1千個魚鉤，但海幸彥執意要找回原來那個魚鉤。

《日本書紀》卷第八則記載，[[仲哀天皇|仲哀天皇]]征伐[[熊襲|熊襲]]途中，岡縣主熊鰐拔五百枝良木，立九尋船之舳<ref>船尾謂之舳。</ref>，上枝掛白銅鏡，中枝掛十握劍，下枝掛八尺瓊。而伊覩縣主五十迹手亦拔取五百根良木，立於舳艫<ref>艫為船首。</ref>，上枝掛八尺瓊，中枝掛白銅鏡，下枝掛十握劍。
