'''十全路（Shihcyuan Rd.）'''是[[高雄市|高雄市]]的東西向重要道路，本道路共分為四個部分。東起於[[覺民路|覺民路]]522巷口（正興國中西側），西止於[[同盟路|同盟三路]]口，僅經[[三民區|三民]]一區。

== 行經行政區域 ==
* [[三民區|三民區]]

== 分段 ==
十全路共分為四個部分：
* 十全路：東起於[[覺民路|覺民路]]522巷口（正興國中西側），西於[[民族路_(高雄市)|民族一路]]口接十全一路。
* 十全一路：東於民族一路口接十全路，西於[[博愛一路|博愛一路]]口接十全二路。
* 十全二路：東於博愛一路口接十全一路，西於[[中華二路|中華二路]]口接十全三路。
* 十全三路：東於中華二路口接十全三路，西止於[[同盟路|同盟三路]]口。

== 歷史 ==
{{expand section}}
位於民族一路與正興國中西側巷道之間新開通的十全路，已於1970年代完成土地徵收，但因牴觸戶不願遷離，歷經四十四年的協調之後，直到2016年10月27日全數執行完畢，2017年7月31日當日傍晚道路完工開放通行<ref>{{cite web|url=https://news.ltn.com.tw/news/life/breakingnews/2148901|title=8任市長無法解決 高雄十全路今天傍晚終於通車了|work=[[自由時報|自由時報]]|author=自由電子報|date=2017-07-31|accessdate=2019-03-06|language=zh}}</ref><ref>{{cite web|url=https://www.youtube.com/watch?v=GEdMn5X4G0s|title=高市十全路通車 居民等了40年終於連通覺民路|work=[[YouTube|YouTube]]|author=[[東森新聞|東森新聞]]|date=2017-07-31|accessdate=2019-03-06|language=zh}}</ref>。
