'''十勝川'''（{{jpn|j=十勝川|hg=とかちがわ|rm=Tokachigawa|f=y}}）是位於日本[[北海道|北海道]]東部流入[[太平洋|太平洋]]的[[一級河川|一級河川]]，名稱源自十勝川在出海口前的分流{{tsl|ja|浦幌十勝川}}河口的「十勝太」聚落，地名則來自[[阿伊努語|阿伊努語]]的「tokap-us-i」（後訛變為「Tokapci」），意思是「乳房的地方」，指十勝川出海口像是乳房一樣。<ref>{{Cite web| url=http://www.tokachi.pref.hokkaido.lg.jp/gaiyo/| title=十勝的概要| work=十勝綜合振興局| language=ja| accessdate=2016-02-22}}</ref>

== 地理 ==
十勝川源頭位於[[上川郡_(十勝國)|上川郡]][[新得町|新得町]][[十勝岳|十勝岳]]東南側，源頭地區現被列為「原生自然環境保全地域」，經過多個支流匯流後，於[[中川郡_(十勝國)|中川郡]][[豐頃町|豐頃町]]流入[[太平洋|太平洋]]，流域面積9,010km²佔了整個[[十勝綜合振興局|十勝綜合振興局]]轄區內約九成的區域，流域為北海道第二大的河川，在全日本排第六<ref>{{Cite web| url=http://www.ob.hkd.mlit.go.jp/hp/tokachi/river/gaiyou.html| title=十勝川水系的概要| work=國土交通省北海道開發局帶廣開發建設部| language=ja| accessdate=2016-02-22| deadurl=yes| archiveurl=https://web.archive.org/web/20130106034546/http://www.ob.hkd.mlit.go.jp/hp/tokachi/river/gaiyou.html| archivedate=2013-01-06}}</ref> 。上游有{{tsl|ja|十勝ダム|十勝水壩}}、{{tsl|ja|岩松ダム|岩松水壩}}、屈足水壩。

== 生態 ==
過去曾經擁有豐富的，有[[大麻哈鱼|大麻哈鱼]]迴游，也曾有釣到長度超過兩公尺[[遠東哲羅魚|遠東哲羅魚]]的紀錄。但現在由於水質的汙染，以及在[[十勝川溫泉|十勝川溫泉]]下游設置了[[千代田堰堤|千代田堰堤]]，影響了這些魚類的棲息。

十勝川溫泉的[[十勝中央大橋|十勝中央大橋]]周邊在冬季有候鳥[[天鵝|天鵝]]在此過冬棲息。<ref>{{cite web |url=http://www.tokachigawa.net/sightseeing/swan.html |title=白鳥飛来地 |publisher=音更町十勝川溫泉觀光協會 |accessdate=2016-02-22| language=ja}}</ref>
