'''十八人之獄'''又称'''十八先生之獄'''，是指[[南明|南明]]时期，[[永历帝|永历帝]]调集南方明军并联合大西军农民义军抗清，[[孙可望|孙可望]]并欲自立称帝，朝臣[[吴贞毓|吴贞毓]]等奉永历命欲召[[李定国|李定国]]护驾{{参|《明末滇南纪略》、《称兵犯阙》。}}。事情败露之后，孙可望派大臣[[郑国|郑国]]举兵问罪，最后孙可望以「盗宝矫诏、欺群害良」拟罪，赐[[吴贞毓|吴贞毓]]自缢并斩吴以下十七朝臣，此即为南明史上的“十八先生之狱”{{参|name=安龙十八先生墓|{{cite web|title=安龙十八先生墓|url=http://www.itravelqq.com/2011/0704/157827.html|publisher=腾讯网|accessdate=2014-01-03|language=中文}}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}}}。

== 背景 ==
明朝中央政权灭亡后，原来在江南地区的明朝宗室诸王纷纷建立政权以图恢复明朝，「永历」就是其中之一{{参|{{cite journal|author=秦刚|title=《评说不一的十八先生之狱》|journal=《风景名胜》|date=1996年|issue=03期}}}} 。1646年（南明隆武二年）秋，唐王[[朱聿键|朱聿键]]政权被清军击溃，[[丁魁楚|丁魁楚]]、[[瞿式耜|瞿式耜]]等于广东肇庆拥立桂王[[朱由榔|朱由榔]]为帝，即永历帝。第二年一月改元[[永历|永历]]{{参|《明史·列传第一百六十八》：顺治三年九月，大兵破汀州。式耜与魁楚等议立永明王由榔，乃迎王梧州，以十月十日监国肇庆。}}。在清军的不断进攻下，永历政权辗转迁到[[贵州|贵州]][[安隆|安隆]]。因有[[大西|大西]]农民军做支柱，他们的抗清势力有所发展。但农民军内部的矛盾和永历政权的党争日渐加剧{{参|name=安龙十八先生墓}}。

== 过程 ==
[[大西军|大西军]]余部几经周折与永历政权谈判「联合恢剿」，即农民起义军欲借助永历政权的“正统”旗号，而永历君臣也要借助农民起义军的实力，巩固永历政权，抗击南下的[[满清|清军]]，企图收复失地。各地抗清武装力量纷纷向[[湖广|湖广]]和[[云贵|云贵]]集聚{{参|name=六百年的文化回望}}。

1652年（永历六年），正月二十日，永历帝一行50余人，在大西军余部孙可望部将的护卫下由广西辗转来到由安顺军民府管辖下的[[贵州|贵州]][[安隆|安隆所]]{{参|name=六百年的文化回望|{{cite news|title=黔西南：六百年的文化回望|url=http://epaper.gywb.cn/gyrb/html/2013-08/21/content_354278.htm|accessdate=2014-01-04|newspaper=贵州日报|date=2013-08-21|location=贵州|language=中文|archive-url=https://web.archive.org/web/20140104213244/http://epaper.gywb.cn/gyrb/html/2013-08/21/content_354278.htm|archive-date=2014-01-04|dead-url=yes}}}} 。也就是[[孙可望|孙可望]]把永历帝接到他自己的势力范围内，改名[[安龙|安龙府]]，作为南明的行在，永历政权在安龙定居下来后，陆续有一批南明王朝的官员、宫人从[[广西|广西]]赶来，永历政权又重派各执事官员，俨然尊依明朝旧制{{参|name=六百年的文化回望}}。

但是[[永历帝|永历帝]]被大西军余部“迎驾”到贵州安龙以后成了[[孙可望|孙可望]]的掌中物。踞守贵阳的[[孙可望|孙可望]]专横跋扈，没有真诚拥戴永历帝的诚意，不过是借助[[南明|残明]]势力的封建正统旗号，建立和巩固云贵抗清基地，以他自己为领袖{{参|name=六百年的文化回望}}。最终滋长了「挟天子以令诸侯」甚至割据称帝的野心{{参|佚名：《明末滇南纪略》“初以天子为名，实挟之以自私也”。}}。而大西军的另一领导人[[李定国|李定国]]则是永历政权真诚的护卫者。他与他的部将们对既想称王称帝的[[孙可望|孙可望]]心存不满，而对永历政权廷则抱有幻想和扶持“正统”反抗清军的真诚愿望{{参|name=六百年的文化回望}}。

另一方面，永历政权自建立伊始，内部争权夺利斗争就显示出来。暂居安龙以后，以执掌政权的文安侯[[马吉翔|马吉翔]]为首的一派，亲附权倾势重的[[孙可望|孙可望]]，企图借助孙可望的实力排除异己，进而拥戴孙可望称王称帝，取代永历帝。而以内阁首辅[[吴贞毓|吴贞毓]]为首的一派，忠贞不渝地拥戴永历皇帝，并团结了宫廷内外[[南明|残明]]的大多数朝臣，也希望借助大西军中拥戴永历帝的力量，使南明王朝重振[[明朝|大明王朝]]的雄风{{参|name=六百年的文化回望}}。

正值[[李定国|李定国]]、[[刘文秀|刘文秀]]率军北伐抗清之际，孙可望驻扎[[贵阳|贵阳]]，征发民工，大兴土木，营造宫殿，设立五府六部三衙门，分封文武百官，命令四川、贵州、云南三省归属南明朝的文臣武将，克期到秦王府朝见。同时还建立[[太庙|太庙]]和社稷，制订朝仪。名义上是建立[[南明|南明]]永历政权的秩序，实质上是为将来的篡位做准备。孙可望的举动，既引起李定国的愤慨，又加剧了永历君臣对他的怨恨{{参|name=六百年的文化回望}}{{参|name=评价|{{cite web|title=永历帝不愿做傀儡 十八大臣终被谋害|url=http://culture.gog.com.cn/system/2013/07/09/012457412.shtml|publisher=金黔文化|accessdate=2014-01-03|language=中文|deadurl=yes|archiveurl=https://web.archive.org/web/20140103223233/http://culture.gog.com.cn/system/2013/07/09/012457412.shtml|archivedate=2014-01-03}}}}。

永历帝慑于孙可望势力，为摆脱控制，永历帝和[[吴贞毓|吴贞毓]]、[[李元开|李元开]]、[[胡士端|胡士端]]、[[蒋乾昌|蒋乾昌]]等十八位朝臣密谋于1652年（永历六年）十一月定计企图借助在广西北伐抗清的安西王李定国的势力，翦除孙可望及卧榻之侧的[[马吉翔|马吉翔]]、[[庞天寿|庞天寿]]等人的势力。他们经过精心策划，拟定密诏，私铸“屏翰亲臣”金印，遣使赴广西诏[[李定国|李定国]]回师，保卫安龙{{参|name=六百年的文化回望}}。1653年（永历七年）十一月，[[李定国|李定国]]接密敕后，表示：「只要我李定国活一天，就不会让陛下受到屈辱，希望陛下暂时忍受，我这就去救驾，我宁可离开孙可望，也不会背叛陛下。{{参|邵廷采：《西南纪事·卷10》：“臣定国一日未死，宁令陛下久蒙幽辱，幸稍忍待之。臣兄事可望有年，宁负友必不负君。”」}}并告知俟恢复粤东，即来迎驾{{参|江之春：《安龙纪事》，《黔南丛书》第五集}}。不料文安侯[[马吉翔|马吉翔]]为讨好孙可望，将此事具告，事情败露，十八位大臣为保全[[永历帝|永历帝]]，承担了密谋责任{{参|name=悲情历史与风花雪月|{{cite web|title=安龙古城：悲情历史与风花雪月|url=http://epaper.xkb.com.cn/view.php?id=774820|publisher=新快网|accessdate=2014-01-03|language=中文|deadurl=yes|archiveurl=https://web.archive.org/web/20140103232402/http://epaper.xkb.com.cn/view.php?id=774820|archivedate=2014-01-03}}}}。1654年（永历八年）三月，孙可望派心腹[[郑国|郑国]]招大学士[[吴贞毓|吴贞毓]]等十八人，诬以「欺君误国，盗宝矫诏」罪，赐内阁首辅[[吴贞毓|吴贞毓]]自缢（实处以绞刑），内监[[张福禄|张福禄]]、[[全为国|全为国]]和刑科给事中[[张镌|张镌]]三人凌迟处死，其余全部处死。临刑前，大臣们大义凛然，向北跪拜永历帝行宫完毕，写下了绝命诗，从容就义{{参|name=安龙十八先生墓}}{{参|name=评价}}。

孙可望派人将遗骸丛葬于北关马场，史称十八人之狱{{参|name=安龙十八先生墓}}{{参|name=评价}}。

== 墓地 ==

吴贞毓等十八人殉难后，其家属及安龙百姓将他们合葬于城内西北隅的天榜山下。后人为他们建了墓陵{{参|name=悲情历史与风花雪月}}。1656年（永历十年）农民起义军将领[[李定国|李定国]]率军至安龙护驾，建十八先生祠堂，包括正殿和东、西配殿，永历帝亲题墓碑。清康熙年间在墓后树林；1879年（道光二十九年）兴义府(今安龙)知府张瑛建多节亭、虚舟；1916年南笼县（今安龙）知事洪寅于墓前建石牌坊；1883年（光绪九年）和1939年先后两次凿刻摩岩{{参|name=安龙十八先生墓}}。

墓后刻有十八先生的绝命诗碑碣和近代名人（如[[蒋介石|蒋介石]]、[[居正|居正]]、[[李烈均|李烈均]]、[[吴鼎昌|吴鼎昌]]等）题咏的众多摩崖石刻{{参|name=悲情历史与风花雪月}}。

== 评价 ==
十八人之獄事件历代史家有褒有贬，毁淮不一{{参|{{cite web|title=安龙 明十八先生墓|url=http://www.xyzc.cn/travel/zoujinqianxinan/jinzhoujingdian/2010-04-16/12251.html|publisher=兴义|accessdate=2014-01-03|deadurl=yes|archiveurl=https://web.archive.org/web/20140104040815/http://www.xyzc.cn/travel/zoujinqianxinan/jinzhoujingdian/2010-04-16/12251.html|archivedate=2014-01-04}}}}。把持南明政权的孙可望最终还是投降了满清，而十八先生的从容就义，慷慨赴死的气节最终千古流芳，为人们所铭记，一度被后人赋予忠君爱国的精神加以传颂{{参|name=悲情历史与风花雪月}}。

有人认为，十八人之獄是南明[[永历政权|永历政权]]内部激烈的秦、晋党争白热化的结果，其根本是明朝腐败官僚体制的产物。[[南明|南明]]沿袭了北京朝廷的腐朽气息和文官集团无休无止的党争，进一步加速了永历政权的覆灭{{参|name=评价}}。

== 轶事 ==
十八先生殉难之时，「安龙之三尺童子，也无不流连叹息」{{参|江之春，《安龙纪事》。}}。据传，当时天色突变，风雨交作，天榜山麓忽有泉水涌出，百姓说是天公流泪，因此，将此泉名为「忠泉」{{参|name=评价}}。
