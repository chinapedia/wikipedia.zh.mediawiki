'''什一奉獻'''（或'''什一税'''、'''什一捐'''），常用於指[[猶太教|猶太教]]和[[基督宗教|基督宗教]]的宗教奉獻，歐洲[[封建社會|封建社會]]時代用來指教會向成年教徒徵收的[[宗教稅|宗教稅]]。 

源於《[[聖經舊約|聖經舊約]]》時代，其[[希伯來文|希伯來文]]原意是「十分之一」。在古代近東的國家，其中有[[迦南|迦南]]、[[腓尼基|腓尼基]]、[[阿拉伯|阿拉伯]]等國都有什一奉獻的習俗以維持崇拜及國家支出。<ref>[http://www.christianstudy.com/data/ot/tithes.html 當納之十分之一]，武林英雄帖</ref>

== 沿革 ==
據《[[聖經|聖經]]·[[創世記|創世記]]》記載，[[亞伯拉罕|亞伯拉罕]]把所得的十分之一獻給撒冷城的[[麥基洗德|麥基洗德]]，這普遍被視為什一奉獻之起源。直至[[摩西律法|摩西律法]]將其具體制度化地執行，[[以色列人|以色列人]]將此獻給[[上帝|上帝]][[耶和華|耶和華]]及支持[[利未支派|利未支派]]的事奉工作。<ref>《聖經·民數記》18:21,24 ：「凡以色列中出產的十分之一、我已賜給利未的子孫為業、因他們所辦的是會幕的事、所以賜給他們為酬他們的勞。……因為以色列人中出產的十分之一、就是獻給耶和華為舉祭的、我已賜給利未人為業、所以我對他們說、在以色列人中不可有產業。」</ref>在4世紀以前的[[基督教會|基督教會]]並沒有奉行什一奉獻，信徒隨自己意願自由捐獻。自[[羅馬帝國|羅馬帝國]]將基督教納為[[國教|國教]]後，教會開始鼓吹徵收。<ref name="tithe">中國大百科智慧藏：什一稅</ref>在第六世紀以前，什一奉献又漸漸在天主教和[[西歐|西歐]]普及。及至公元567年和585年，什一奉獻被提到的[[大公會議|大公會議]]，並最終採納為對教會奉獻的必要標準。<ref name="tithe" />

公元779年[[法蘭克王國|法蘭克王國]][[查理大帝|查理大帝]]訂立[[法律|法律]]實施什一税，西歐各國在10世紀中葉相繼實行，視之為[[稅金|稅金]]。[[宗教改革運動|宗教改革運動]]之後，什一稅制仍在[[天主教|天主教]]和[[基督新教|基督新教]]國家裡繼續執行。[[羅馬教廷|羅馬教廷]]與[[德意志|德意志]]地區的皇帝在1563年召開[[天特會議|特倫多大公會議]]重申教徒必須繳付什一稅。<ref name="tithe" />隨後歐洲國家政權與教會間發生矛盾，歐洲國家逐漸廢除此[[稅制|稅制]]，以其它的稅收形式取替。時至現代，仍有一些[[基督教派|基督教派]]引申《[[舊約聖經|舊約聖經]]》，規定信徒實行什一奉獻；另一些基督教派則鼓勵信徒憑信心奉行什一奉獻，以應付教會運作的財務需要和開支。

== 摩西律法下的什一奉獻 ==
根據[[摩西律法|摩西律法]]所載，以色列人的一切出產之十分之一都當歸給[[上帝|上帝]][[耶和華|耶和華]]。十分之一與祭物要獻在指定地方，祭司會將其作獻祭及利未支派之酬勞，每三年用當年土產的十分之一來幫助社會內的有示弱細小眾。相關的重點條例如下：

{{Cquote|1=
地上所有的、無論是地上的種子、是樹上的果子、十分之一是耶和華的、是歸給耶和華為聖的。人若要贖這十分之一的甚麼物、就要加上五分之一。凡牛群羊群中、一切從杖下經過的、每第十隻要歸給耶和華為聖。不可問是好是壞、也不可更換、若定要更換、所更換的與本來的牲畜都要成為聖、不可贖回。這就是耶和華在[[西乃山|西乃山]]為以色列人所吩咐摩西的命令。（[[利未記|利未記]]27:30-34） 
}}

{{Cquote|1=
你要把你撒種所產的，就是你田地每年所出的，十分取一分；又要把你的五穀、新酒、和油的十分之一，並牛群羊群中頭生的，喫在耶和華你神面前，就是他所選擇要立為他名的居所。這樣，你可以學習時常敬畏耶和華你的上帝。……每逢三年的末一年，你要將本年的土產十分之一都取出來，積存在你的城中。在你城裡無分無業的利未人，和你城裡寄居的，並孤兒寡婦，都可以來，喫得飽足。這樣，耶和華你的上帝必在你手裡所辦的一切事上賜福與你。（[[申命記|申命記]]14:22-23,28-29）
}}

== 类似的中國的什一税 ==
在[[周朝|周朝]]，當時實行'''徹法'''，除了[[井田制|公田]]之外，另外農民還要將其所得的十分之一納予國家<ref>《[[尚書大傳|尚書大傳]]》卷二：「王者十一而稅，而頌聲作矣」</ref>，但是在[[春秋戰國|春秋戰國]]時代之後，這種制度便逐漸不獲採用。

==伊斯蘭中==
在古代伊斯蘭國家，耕作官地的農民需繳交收成1/10，名哈拉吉稅。
