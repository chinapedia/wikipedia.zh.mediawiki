'''十時連久'''（[[天文_(日本)|天文]]20年?（1551年？）－[[文祿|文祿]]2年（1593年）），日本[[戰國_(日本)|戰國時代]]、[[安土桃山時代|安土桃山時代]]的武將。通稱'''傳右衛門'''，一名'''惟道'''。[[十時惟長|十時惟長]]次子、十時惟元之弟。[[立花家|立花家]]侍大將、[[家老|家老]]兼[[武者奉行|武者奉行]]，擁有異名「生摩利支天」。

== 出身 ==
[[十時氏|十時氏]]為九州豐後國之中的國人眾，仕於其中勢力較大的[[戶次氏|戶次氏]]，並且依附統領豐後的[[大友家|大友家]]。主君戶次鑑連（[[立花道雪|立花道雪]]）奉命轉封筑前立花山城後，連久作為十時氏的一員也跟隨前往。這個時期立花家的軍團主力有約三分之一是為十時氏，並且連久之父十時惟長（十時和泉守）以上的十時一族幾乎都為家老職。

連久本名惟道，和另一名堂兄弟[[十時連貞|十時連貞]]雖然都是十時家庶流，但十時家嫡流的[[十時惟忠|十時惟忠]]傳到次子[[十時連秀|十時連秀]]為立花家老，皆因戰功彪炳而得以從主君道雪處受賜「連」字，於是改名連久。

而連久早期曾繼承道雪家臣板井兵部而改名板井新四郎，但在其兄長十時惟元於筑後善導寺遭到伏擊戰死後，繼承了兄長三十二町的領地並回歸十時氏，後來隨主君[[立花宗茂|立花宗茂]]轉封筑後柳川，領有一千二百石俸祿，擔任家老以及總奉行四人眾之一人。

== 生摩利支天 ==
[[天正|天正]]年間，因主家大友家於[[耳川合戰|耳川合戰]]以及[[九州|九州]]三強之一的[[龍造寺|龍造寺]]家在肥前沖田畷之戰皆敗給[[島津家|島津家]]，九州開始呈現許多國人眾脫離主家叛亂的現象。

連久所仕的[[立花道雪|立花道雪]]身為大友家於北九州筑前的重臣，屢次率兵出征平亂，連久作為立花家侍大將，每至戰場必定能立下一番槍、一番首、一番乘等赫赫戰功，被主君道雪稱讚「勇猛果敢，臂力超群」。生涯所獲感狀約有數十張，以其戰場上的豪勇被眾人稱讚形容為活人顯靈的戰神[[摩利支天|摩利支天]]而得到「生摩利支天」的響亮異名。

此外於九州征伐後，[[豐臣秀吉|豐臣秀吉]]於豐前小倉的入鹿原建設了規模甚大的[[相撲|相撲]]場地，並要求日本34處諸侯推舉自家一位勇士參加比賽，立花家於是推舉了連久參加，而連久不僅臂力超群，還因為極為熟知相撲法則，被眾人推舉為[[行司|行司]]（也就是大會司儀），使其名聲更加廣傳。

== 怪奇武勇傳 ==
天正13年（1585年）6月，正當連久隨[[立花道雪|立花道雪]]參加筑後遠征之時，連久被道雪任命為偵查物見隊探查敵情，連久於山中偵查期間因為炎熱而暫時脫離任務於附近的溪流中洗澡，洗澡途中卻發現敵兵靠近，連久一時心急，就這麼沒穿衣服裸身衝岀水中並一一將敵兵擊退！

隨後連久回軍向道雪報告，就連一向軍規嚴謹的道雪對於連久如此狼狽的軍功也只能認同苦笑，會發生像這樣奇怪的事實為連久的一大魅力。

== 無意居戰功 ==
[[天正|天正]]14年（1586年）8月24日，連久隨少主[[立花宗茂|立花宗茂]]率軍1千5百追擊撤退包圍立花山城的島津軍。翌日，攻打島津軍的筑前據點，由星野吉実、吉兼兄弟所鎮守的高鳥居城。

雖然城方僅3百人，但是仍以鐵砲攻勢和大木重石攻擊，立花軍將士一時有所損傷，然而靠著[[立花宗茂|立花宗茂]]和家中大將[[小野鎮幸|小野鎮幸]]的鼓舞，立花軍成功殺進城內。

這時立花家一門的勇將[[立花統春|立花統春]]很快的殺進敵大將[[星野吉実|星野吉実]]所在之處，統春面對吉実以討取總大將的禮儀押住吉実的額頭並且想割取頭顱，然而吉実在抵抗下只被割斷其頭盔的繩子並且逃脫，就在這一瞬間，連久經門而入快速又撞倒吉実並以槍刺殺。

後來立花家武者奉行[[池邊永晟|池邊永晟]]在記錄戰功之時，將討取[[星野吉実|星野吉実]]的戰功記給了十時連久。連久則說：「這是統春殿下的功勞才對！」而統春也謙虛的說：「我沒有討取成功，所以應該是給予致命的連久殿下的戰功才對！」

感到為難的[[池邊永晟|池邊永晟]]為此報告主公宗茂，宗茂聽聞兩人互讓戰功的舉動，就將兩人都給予戰功和感狀，並且感狀中引用孔子的「子曰、孟之反不伐、奔而殿、將入門、策其馬曰、非敢後也、馬不進也」之句來褒揚二人互讓戰功的表現。

之後連久因其不居功的行為更加受到許多其他家臣的欣賞和好感，並且和立花統春、[[內田統續|內田統續]]等皆為知心好友。此後池邊永晟轉任旗奉行，武者奉行則因此由連久擔任。

== 猛烈衝破敵 ==
高鳥居城戰後不久，島津軍以一門名將[[島津家久|島津家久]]率軍二萬進攻大友家主[[大友宗麟|大友宗麟]]的丹生島城，[[立花宗茂|立花宗茂]]受命和援軍[[毛利元康|毛利元康]]（末次元康）合兵四千為先鋒隊趕往救援，並且後部有[[吉川廣家|吉川廣家]]一萬人為後續部隊。

[[島津家久|島津家久]]很快便對此作出應對，留下二千人佯裝攻城，自率一萬八千兵打算伏擊這支援軍。結果宗茂和元康四千兵果真陷入苦戰，並且兵力上本來就呈現不利狀態，但是宗茂冷靜的聯繫元康並且命令連久組成[[敢死隊|敢死隊]]，連久一隊開始集中往丹生島城的方向進攻以誘使島津軍分散兵力加強對城方的防守，宗茂和元康則趁機衝向島津軍兵力虛弱的地方成功撤退，一早衝入敵陣的連久聽聞宗茂撤退後，也在亂軍中奮迅衝鋒，勇烈的突破了重圍，展現了不輸給薩摩隼人武士的風采。

往後，連久的各項武勇事績幾乎成了立花家飯後閒聊的必談項目，並且在立花家的軍記物中時常出現其豪勇的記錄。

== 老將的遠慮 ==
立花家經過九州征伐後領受了九州筑後柳川的領地，期間有位浪人天野源右衛門貞成（[[安田國繼|安田國繼]]）因仰慕[[立花宗茂|立花宗茂]]而希望入仕立花家，並且以曾為[[明智光秀|明智光秀]]重臣「明智三羽烏」之一的經歷，以及曾刺傷[[織田信長|織田信長]]為豪，下了每臨戰陣必立下一番槍的豪語並要求俸祿二千石。而此時連久不過一千石俸祿，且自身時常立下一番槍之故，宗茂在考慮此兩人俸祿之權衡是否得當時，連久對其要求必立下其所言之功，否則將不配二千石的俸祿，並對主君宗茂說：「此人為英傑，必會盡力，我本身不會憎恨自身的俸祿比之較少，請主公放心登用吧！」於是天野貞成往後出戰[[肥後一揆|肥後一揆]]、[[朝鮮戰役|朝鮮戰役]]盡皆如所言般表現的出色。

連久也參與鎮壓肥後一揆的動亂，於亂事最終階段的黑門之戰，連久和堂弟連貞皆作為放討作戰的討手一員而表現活躍，擊殺了隈部政利（隈部親永次男，又稱內古閑鎮房）。

後隨立花軍參加[[豐臣秀吉|豐臣秀吉]]發動的侵略朝鮮戰役，在其中[[文祿之役|文祿之役]]的最大一場戰事[[碧蹄館之戰|碧蹄館之戰]]中，立花軍被認命為先鋒。

[[立花宗茂|立花宗茂]]於朝鮮京城北方的礪石領將軍隊分為先陣[[小野鎮幸|小野鎮幸]]和[[米多比鎮久|米多比鎮久]]七百人、中備十時連久和[[內田統續|內田統續]]五百人以及本隊宗茂和其弟[[高橋統增|高橋統增]]二千人。

就在開戰前夕，連久帶著統續前往先陣的[[小野鎮幸|小野鎮幸]]處要求讓予自身為前鋒。連久當時說了：「小野殿下請後退至中備位置吧！請允許我連久為先鋒。」

鎮幸聽後怒叱：「無禮！貴殿竟然如此這般破壞了[[軍法|軍法]]，是想汙辱我嗎？」

連久急忙解釋：「絕無此事！因為小野殿下和米多比殿下實為家中的重臣，二位的武勇和智略是家中不可欠缺的，如果讓已經老年的我和統續為先陣的話，面對明國如此大軍而犧牲並不足惜，還能讓敵軍先行疲憊，屆時小野和米多比殿下為中備接續作戰，並配合主公宗茂的軍勢的話必定能夠取得勝利！所以請讓在下擔任先鋒吧！」

鎮幸聽畢向主公宗茂取得同意後順利切換軍隊順序，而鎮幸也了解到連久的心意和遠慮，感到佩服。

== 死（十）時真勇士 ==
終於，立花軍和明軍在[[內田統續|內田統續]]的鐵砲攻勢下開戰，連久則和天野貞成（[[安田國繼|安田國繼]]）相爭為一番槍，各自皆以「投槍」的方式擊落數騎明兵，投完後連久又拔刀驅馬殺入敵陣斬落了五、六明騎兵，連久的五百兵力就有如穿透石塊的細針一般突入明軍[[查大受|查大受]]所率的二、三千的軍陣之中，面對倍數於己的敵軍，連久毫無懼色，即使明軍[[李寧|李寧]]又再不久之時來援約七千兵力和火器助陣，連久仍然正確的判斷明軍的動向，打算先以正面破敵引誘左右夾攻的明軍呈一路追擊後在迴轉奮力突破和[[小野鎮幸|小野鎮幸]]的中備軍交替。

因此連久在正面突破期間又將手持大槍的敵兵討取了數十人，前進到了山谷望客硯下，這時的連久則已負傷多處，浴血奮戰之時連頭盔的裝飾物也被擊斷，這時[[小野鎮幸|小野鎮幸]]終於越過明軍火器的爆風和連久會合，就在成功撤退之際連久又擊落敵方一名騎兵之時，卻被明將[[李如梅|李如梅]]趁機以毒箭射中右胸，然而連久再此時又奮力擊退想進一步攻擊自己的敵兵並繼續撤退，並將先鋒隊的撤退指揮交由同僚[[池邊永晟|池邊永晟]]，苦戰之下連久終於回到本陣，最終向主公宗茂傳達「下一任家老兼武者奉行請主公交給內田統續……」的遺言後，於翌日箭傷毒發死亡。<ref>『日本戦史・朝鮮役』(補伝 第六十九十時傳右の戦死)[http://dl.ndl.go.jp/info:ndljp/pid/936356/194]</ref>(據《筑後國史》撐到2月5日才死去<ref>『筑後国史』(十時氏系図)によれば、死去の日は2月5日[http://dl.ndl.go.jp/info:ndljp/pid/1175327/28]</ref>)。

連久作為武士，此戰後的[[內田統續|內田統續]]對於自己沒有和連久一同戰死大感羞愧，並且包刮戰前對小野鎮幸所說的話、為立花家所建的軍功以及為自己所建立的名聲，連久真的是一位至死也令人敬配的「真勇士」。
