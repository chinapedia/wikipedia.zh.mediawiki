《'''十二金钱镖'''》，2014年上映于中国大陆的[[3D|3D]]民初剧，改編自[[白羽|白羽]]《[[十二金錢鏢_(小說)|十二金钱镖]]》，导演游达志、李宏宇，主演陈键锋、元彪、陶昕然、夏花、王佩宁，<ref>{{cite web |url=http://news.xinhuanet.com/ent/2012-09/13/c_123710163.htm |title=3D大剧《十二金钱镖》热拍 陶昕然拒婚惹众怒/图 |accessdate= 2012|publisher= 新华网|work= |language=zh-hans}}</ref><ref>{{cite web |url=http://ent.ifeng.com/tv/news/mainland/detail_2012_09/04/17329997_0.shtml |title= 《十二金钱镖》热拍 元彪陈键锋”反目成仇”|accessdate=2012 |publisher= 凤凰网|work= |language=zh-hans}}</ref>该剧是中国大陆第一部3D电视剧。<ref>{{cite web |url=http://news.xinhuanet.com/ent/2012-09/11/c_123699614.htm |title= 《十二金钱镖》造型曝光 元彪霸气陈键锋“锅盖头”|accessdate=2012 |publisher=新华网 |work= |language=zh-hans}}</ref>2013年3月杀青，2014年7月22日首播。而泰國Channel One 2016年3月31日-5月18日週一-週五14:00到15:00。 

==剧情==
该剧根据宫白羽同名小说改编，讲述[[中华民国|中华民国]]初年[[天津市|天津市]]“十二金钱镖”的历史故事。
