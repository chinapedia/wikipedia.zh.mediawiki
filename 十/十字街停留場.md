'''十字街停留場'''（{{jpn|j=十字街停留場|hg=じゅうじがいていりゅうじょう|rm=Jūjigai teiryūjō|f=y}}）是一位於[[日本|日本]][[北海道|北海道]][[函館市|函館市]]豐川町2番地先、7番地先，[[函館市交通局|函館市交通局]]函館市電本線沿線的[[鐵路車站|電車站]]。2007年7月1日，根據[[命名權|命名權]]制度，被命名為'''十字街（明治館、紅磚倉庫群前）'''，但是現在已用回原名稱。

== 通過路線 ==
;函館市交通局
* 本線（往末廣町至函館船塢前方向，{{color|blue|■}}5系統）
* 寶來、谷地頭線（往寶來至谷地頭方向，{{color|red|■}}2系統）

== 概述 ==
由於接近元町西面區域，因此在煙火大會及聖誕夢幻等大型活動舉行，都會配置裝備無線對講機的調度員及整備員等職員作出協助。此外，此處亦是2系統及5系統的指定轉車站，以及[[函館巴士|函館巴士]]的轉車指定車站。

== 歷史 ==
* 1913年（[[大正|大正]]2年）10月30日 - 開始營業。
* 2007年（[[平成|平成]]19年）7月1日 - 根據命名權制度，被命名為'''十字街（明治館、紅磚倉庫群前）'''。
* 日期不明 - 用回原名稱「'''十字街'''」。

== 車站構造 ==
* 十字街站擁有2個月台（[[側式月台|側式月台]]）、2條電車路線（相反方向）。
* 兩個方向的月台都設置了殘障人士[[無障礙設施|無障礙設施]]及電車接近顯示器。
* 由於此車站為前往函館船塢前或谷地頭的分歧車站，前往這個方向的電車都會因為[[交通燈|交通燈]]及[[轉轍器|轉轍器]]的運作，令停車時間較正常的長。
