'''函館地方氣象台'''（{{Jpn|j=函館地方気象台|hg=はこだてちほうきしょうだい|rm=Hakodate chihō kishōdai}}）是[[日本|日本]][[氣象廳|氣象廳]][[札幌管區氣象台|札幌管區氣象台]]下屬的一個[[地方氣象台|地方氣象台]]，位於[[北海道|北海道]][[函館市|函館市]]，管轄範圍為[[渡島綜合振興局|渡島綜合振興局]]和[[檜山振興局|檜山振興局]]。其前身為1872年設立的'''函館氣候測量所'''，是日本歷史上第一個氣象觀測所<ref name = "date">{{Cite web |url = http://www.jma-net.go.jp/hakodate-c/guide/history.html |title = 沿革「日本最初の気象観測所」 |author = 函館地方気象台|language = ja|accessdate = 2014-04-20}}</ref>。

== 職責 ==
函館地方氣象台負責[[渡島綜合振興局|渡島綜合振興局]]和[[檜山振興局|檜山振興局]]的氣象觀測、沿岸防災觀測和防災氣象信息的發佈<ref>{{Cite web |url = http://www.jma-net.go.jp/hakodate-c/guide/work.html |title = 業務概要 |author = 函館地方気象台|language = ja|accessdate = 2014-04-20}}</ref>。

=== 沿岸防災觀測 ===
為防止和減輕[[海嘯|海嘯]]、[[風暴潮|風暴潮]]和天文大潮等災害造成損失，函館市海岸町設有潮位觀測所，[[檜山郡|檜山郡]][[上之國町|上之國町]]設有沿岸雷達式波浪計。

* 上之國沿岸波浪計

== 歷史 ==
[[幕府末年|幕府末年]]，[[箱館|箱館]]（1869年起改稱函館）成為[[日本|日本]]對外開放開放的窗口之一，逐漸有外國人前往居留。為解決自身的不便，他們開始有單獨、分散的氣象觀測活動。1872年，[[開拓使|開拓使]]函館[[支廳|支廳]]的福士成豐在函館市船場町（今函館市末廣町）的住所安設觀測機器，於8月26日開始氣象觀測，這也是日本氣象觀測史的開始。3年之後，首都[[東京|東京]]也正式開始了氣象觀測活動<ref name = "date"></ref>。
