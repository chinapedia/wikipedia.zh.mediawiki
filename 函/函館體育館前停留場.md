'''函館體育館前停留場'''（{{jpn|j=函館アリーナ前停留場|hg=はこだてアリーナまえていりゅうじょう|rm=Hakodate-ariina-mae teiryūjō|f=y}}）是位於[[日本|日本]][[北海道|北海道]][[函館市|函館市]]湯川町1丁目34番地先的[[函館市交通局|函館市交通局]]（[[函館市電|函館市電]]）[[湯之川線|湯之川線]]的[[鐵路車站|鐵路車站]]。

== 歷史 ==
* 1913年（[[大正|大正]]2件） - '''芦堀停留所'''開始營業。
* 1920年（大正9年） - 改稱為'''新世界前停留所'''。
* 日期不明 - 改稱為'''湯之川遊樂園前停留所'''。
* 日期不明 - 被廢除。
* 1983年（[[昭和|昭和]]58年）9月10日 - 以'''市民會館前停留所'''名義再次開始營業。
* 2015年（[[平成|平成]]27年）7月 - 伴隨函館體育館開業，加上副名稱為'''市民會館前（函館體育館前）'''。
* 2016年（[[平成|平成]]28年）3月1日 - 改稱為'''函館體育館前（市民會館前）停留所'''。

== 車站構造 ==
* 函館體育館前站擁有2個月台（[[側式月台|側式月台]]）、2條電車路線（相反方向）。
* 往五稜郭方向的月台為有蓋月台。
* 往湯之川方向的月台在2007年1月向湯之川方向延長1米。
