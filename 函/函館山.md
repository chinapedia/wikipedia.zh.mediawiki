'''函館山'''（{{lang-ja|はこだてやま}}）是一處位于[[北海道|北海道]][[函館市|函館市]]市區西部的山地。因函館山也是一座[[陸連島|陸連島]]，因此有時也被稱爲「函館島」「函館山島」。函館山標高334米，周長約9公里。因其外觀形似一頭仰臥的牛，因此又名'''臥牛山'''（{{lang-ja|がぎゅうざん}}）。函館山是13座山峰的總稱，包括了建有瞭望台的御殿山（334m）、藥師山（252m）、杜鵑山（306m）、汐見山（206m）、八幡山（295m）、水元山（280m）、鞍掛山（113m）、地藏山（286m）、入江山（291m）、蝦夷立山（129m）、觀音山（265m）、牛背山（288m）、千疊敷（250m）<ref>{{cite web |url=http://contest.japias.jp/tqj1999/20197/jap/about/2.html|script-title=ja:弐,その構成 |accessdate=2015-06-10|publisher={{lang|ja|函館山詳細データ}}}}{{ja}}</ref>。函館山的基台部分是約100萬年前海底火山噴發時的產物。之後因火山噴發使得這一地區多次隆起下沉，出現了一座大島。在約5000年前。由於泥沙堆積，在島嶼和[[渡島半島|渡島半島]]之間出現了一座[[連島沙洲|連島沙洲]]，函館山遂成爲一座陸連島。而函館市的中心市區就位于連島沙洲之上。
== 歷史 ==
自1898年開始，因制定了[[要塞地带法|要塞地带法]]，日本政府開始在函館山建設要塞建設。至1905年，函館山上已建有砲台、發電厰、觀測所等17個設施（[[津輕要塞|津輕要塞]]）<ref>{{cite web |url=http://www.hakobura.jp/db/db-view/2011/03/post-36.html|script-title=ja:函館山の砲台跡（旧函館要塞） |accessdate=2015-06-10|publisher={{lang|ja|函館市}}}}{{ja}}</ref>。隨著函館山成爲要塞地區，函館山也成爲日本的軍事機密。函館山消失在官方出版的[[地形圖|地形圖]]上。普通民衆也被嚴禁進入函館山，並且禁止拍攝函館山的照片和素描函館山。測繪函館山也被嚴禁。有關函館山的話題也被嚴格限制<ref>{{cite web |url=http://pucchi.net/hokkaido/history/mt_hakodate.php|script-title=ja:地図から抹消されていた函館山 |accessdate=2015-06-10|publisher={{lang|ja|北海道ファンマガジン}}}}{{ja}}</ref>。

在二戰結束之後，函館山失去了其要塞功能，成爲國有土地。函館山的所有者[[大藏省|大藏省]]（現[[財務省_(日本)|財務省]]）免費將函館山借給函館市使用。1945年，美軍拆除了津輕要塞。1946年10月，函館市自大藏省獲得了土地使用許可。12月，函館市設立了函館山管理事務所，函館山重新對一般市民開放。1948年，函館市將這一帶326.6[[公頃|公頃]]的土地按照都市計劃法設為都市計劃緑地「函館山緑地」。

現在御殿山第一砲台遺跡位于纜車設施和停車場、瞭望台、送信所之下，但由於有崩落的危險而被禁止進入。其他的一些要塞遺跡則可以參觀。2001年，「函館山和砲台遺跡」被選為北海道遺產。

由於函館山長達半世紀都禁止普通民衆進入，因此自然環境保存良好。在函館山還生存有現在瀕臨滅絕的蝦夷蟾蜍<ref>{{cite web|url=http://www.hakodate-rc.com/main/flog/|script-title=ja:函館山生息のヒキガエル保護活動|accessdate=2015-06-10|publisher={{lang|ja|函館ロータリークラブ}}}}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}{{ja}}</ref>。但現在由於生物學上認為蝦夷蟾蜍只是日本蟾蜍的亞種，因此函館山的蝦夷蟾蜍無法得到法律保護。相反由於日本蟾蜍和貓等外來物種的入侵，函館山的固有物種面臨被捕食的危機。爲了保護函館山的自然環境，函館市政府制定了市條例，禁止擅自採取函館山花草樹木<ref>{{cite web |url=http://www3.e-reikinet.jp/hakodate/d1w_reiki/333901010005000000MH/333901010005000000MH/333901010005000000MH.html|script-title=ja:函館市都市公園条例 |accessdate=2015-06-10|publisher=e-reikinet.jp}}{{ja}}</ref>。

御殿山的山頂設有瞭望台，可在山麓搭乘[[函館山纜車|函館山纜車]]或駕車（經過[[北海道道675号立待岬函館停車場線|北海道道675号立待岬函館停車場線]]。這條道路也被稱爲函館山觀光道路。此外還有特定季節運行的巴士<ref>{{cite web|url=http://donan.net/hbus/newpage2014-101-yamaeki.html|script-title=ja:函館山登山バス|accessdate=2015-06-10|publisher={{lang|ja|函館バス}}|deadurl=yes|archiveurl=https://web.archive.org/web/20141028090059/http://donan.net/hbus/newpage2014-101-yamaeki.html|archivedate=2014-10-28}}{{ja}}</ref>。）或步行登山到達。但在4月25日至10月15日期間，自17點至22點的這段時間内，函館山的道道675号區間禁止私家車通行。同時在這段期間内全天禁止摩托車通行<ref>{{cite web|url=http://www.334.co.jp/jpn/access/|script-title=ja:函館山ロープウェイアクセスマップ|accessdate=2015-06-10|publisher={{lang|ja|函館山ロープウェイホームページ}}|deadurl=yes|archiveurl=https://web.archive.org/web/20150419011614/http://www.334.co.jp/jpn/access/|archivedate=2015-04-19}}{{ja}}</ref>。並且在10月16日開始約三週的期間，由於函館山纜車需要檢修暫停營業。為緩解交通堵塞並考慮到日落時間，限制通行的時間提前1小時，在16點至21點禁止私家車通行<ref>{{cite web|url=http://www.city.hakodate.hokkaido.jp/docs/2014022000137/|script-title=ja:夜間の観光バス登山方法（２０１５年度）|accessdate=2015-06-10|publisher={{lang|ja|函館市}}|deadurl=yes|archiveurl=https://web.archive.org/web/20150923204045/http://www.city.hakodate.hokkaido.jp/docs/2014022000137/|archivedate=2015-09-23}}{{ja}}</ref><ref>{{cite web|url=http://www.334.co.jp/jpn/news/2014/09/839.php|script-title=ja:整備点検によるロープウェイ運休のご案内|accessdate=2015-06-10|publisher={{lang|ja|函館山ロープウェイホームページ}}|deadurl=yes|archiveurl=https://web.archive.org/web/20141225212454/http://www.334.co.jp/jpn/news/2014/09/839.php|archivedate=2014-12-25}}{{ja}}</ref>。另外在冬季，函館山全面禁止汽車通行。

函館山有多條徒歩前往御殿山的路綫。大多登山客自東側的[[函館市電|函館市電]][[宝來町停留場|宝來町停留場]]出發，經函館山纜車山麓車站和函館護國神社之間的入口進山，沿舊山道登山。這一路徑長約2.8公里，耗費時間約1小時。此外還有北側的自[[函館碼頭前停留場|函館碼頭前停留場]]經[[東本願寺|東本願寺]]船見支院登山的路綫，這一路綫較短但坡較陡。這兩條路綫中途都沒有路燈，若以觀賞夜景為目的登山，需要攜帶光源或在日落時就到達山頂<ref>{{cite web |url=http://www.hakobura.jp/nightview/post-156.html|script-title=ja:函館山へのアクセス、早わかりガイド |accessdate=2015-06-10|publisher={{lang|ja|函館市}}}}{{ja}}</ref>。

函館山是瞭望的聖地。在白天的函館山頂，除了可以看到眼前的函館市區之外，還能隔[[津輕海峽|津輕海峽]]看到遠處的[[下北半島|下北半島]]。但函館山的夜景更加聞名。在夜間的函館山頂可以看到呈細腰形的函館市區的燈火。與之相對的是漆黑的大海，加上漁船的捕魚燈火，使得函館山上看到的夜景頗負盛名<ref>{{cite web |url=http://www.hakobura.jp/nightview/post-105.html|script-title=ja:函館の夜景がきれいな理由 |accessdate=2015-06-10|publisher={{lang|ja|函館市}}}}{{ja}}</ref>。據説在夜景中若能找到「喜歡」（「{{lang-ja|スキ}}」）和「心」（「{{lang-ja|ハート}}」）的文字，就能找到幸福<ref>{{cite web |url=http://tabizine.jp/2014/08/21/17481/|script-title=ja:夜間の観光バス登山方法（２０１５年度） |accessdate=2015-06-10|publisher={{lang|ja|函館市}}}}{{ja}}</ref><ref>{{cite web|url=http://www.334.co.jp/jpn/news/2014/09/839.php|script-title=ja:まるで宝石箱のよう！ミシュラン３つ星の函館夜景|accessdate=2015-06-10|publisher={{lang|ja|タビジン}}|deadurl=yes|archiveurl=https://web.archive.org/web/20141225212454/http://www.334.co.jp/jpn/news/2014/09/839.php|archivedate=2014-12-25}}{{ja}}</ref>。因此函館山也在戀人之中頗具人氣。函館山夜景和[[神戶市|神戶]][[摩耶山|摩耶山]][[掬星台|掬星台]]及[[長崎市|長崎]][[稻佐山|稻佐山]]看到的夜景並列為日本三大夜景。另外，在春季至夏季由於函館山容易有霧，因此有時無法清楚欣賞夜景。
==信號發射塔==
[[日本放送協會|NHK]]和各民營電視台都在函館山山頂瞭望台附近建有電視和FM廣播的信號發射所。各數位電視台為了不讓信號傳送至青森縣一側，均調整了天線的方位<ref>{{cite web|url=http://th-rrv.digi2.jp/tv/hakodateyama.htm|script-title=ja:函館送信所|accessdate=2015-06-10|publisher={{lang|ja|テレビ送信所めぐり}}}}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}{{ja}}</ref>。
