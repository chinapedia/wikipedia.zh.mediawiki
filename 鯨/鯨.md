'''鲸'''是海洋哺乳动物[[鲸下目|鲸下目]]中部分生物的通称。鲸目又分為[[齒鯨|齒鯨]]類（Odontoceti）和[[鬚鯨|鬚鯨]]類（Mystacoceti），前者包括[[海豚科|海豚科]]。在日常語言中，常將鲸和海豚（以及[[江豚|江豚]]）分开，但在[[動物學|動物學]]中它們同屬於一個系群。海豚科屬於[[鲸目|鲸目]]下的[[齒鯨亞目|齒鯨亞目]]，該亞目中有[[抹香鯨|抹香鯨]]及等。而以鯨鬚替代牙齒，利用[[鲸鬚|鲸鬚]]過濾水中浮游生物進食的[[鬚鯨亞目|鬚鯨亞目]]，有[[座頭鯨|座頭鯨]]及現存地球上最大的動物[[藍鯨|藍鯨]]等。

鲸的尺寸可以由只有1.7米長50公斤的[[毛伊海豚|毛伊海豚]]，到長34米重達190噸的藍鲸，後者也是地球上最大的生物。一些種類表現出性别差異，雌性體型比雄性體型更大。鲸拥有流線型的身體和鳍狀的一對前肢。雖然不像[[海豹|海豹]]一样敏捷，鲸鱼也能以20[[節_(單位)節|節 (單位)節]]的速度巡游。鬚鲸用喉咙嚨上的皺褶來擴展它們的嘴，以便吞嚥大口的海水。露脊鲸摄取的海水占牠百分之四十的體重。齒鲸則有尖锥状的牙齒，用於捕捉烏賊和魚類。鬚鲸在海洋中拥有發達的“嗅覺”，而齒鲸擁有敏锐的听觉。鲸的嗅覺之發達，可以同时活躍於水上和水下的環境，以至於有的鲸甚至在盲了以後仍能生存。

數世紀以來，鯨經常被作為桌上佳餚或是[[工業|工業]]產品的原料。然而，到了20[[世紀|世紀]]中葉，鯨的數量已經因為捕鯨工業的盛行而銳減，成為了瀕臨絕種的生物。所幸現階段大多數國家都已經在八零年代簽下全球禁捕令，停止捕鯨工業的持續發展。

==演化==
[[File:AmbulocetusNatansPisa.JPG|thumb]]

所有的鯨目生物都是陸生動物中[[偶蹄目|偶蹄目]]的後裔，而鯨目與偶蹄目在生物分類上又屬於鯨與河馬所屬的[[鯨偶蹄目|鯨偶蹄目]]。鲸也是从陆生哺乳动物经过水生的适应过程而回到海洋的，此过程大约发生在距今5200万---8300万年的[[始新世|始新世]]。

鲸，海豚和江豚，同屬[[鲸偶蹄目|鲸偶蹄目]]，和它們兩趾的有蹄动物血缘最近的近親是[[河馬|河馬]]，大约在4000萬年前分化，古生物學家相信首先由[[巴基鯨科|巴基鯨]]進化成[[走鯨|走鯨]]等水陸兩棲生活的古鯨類，再進化成[[龍王鯨科|龍王鯨]]等完全在水中生活的鯨類。

在3400万年前就分化成[[齒鲸|齒鲸]]和[[鬚鲸|鬚鲸]]兩个[[亞目|亞目]]。!

== 生物分類 ==
现存共有40种鲸魚種類。包括[[新鬚鲸科|新鬚鲸科]]（Cetotheriidae，現存的唯一成員是[[小露脊鲸|小露脊鲸]]），Balaenopteridae（[[鬚鲸|鬚鲸]]家族），Balaenidae（[[露脊鲸|露脊鲸]]家族），Eschrichtiidae（[[灰鲸|灰鲸]]家族），[[一角鲸科|一角鲸科]]（Monodontidae，[[白鲸|白鲸]]和[[獨角鲸|獨角鲸]]），Physeteridae（[[抹香鲸|抹香鲸]]家族），Kogiidae（[[侏儒鲸|侏儒鲸]]和[[小抹香鲸|小抹香鲸]]），Ziphiidae（[[喙鲸|喙鲸]]家族）。
===须鲸===
[[须鲸亚目|须鲸]]口中没有牙齿，只有须。体积庞大，体长至少6米。须鲸有两个鼻孔，换气时可以喷出两股水柱。主要以磷虾等小型甲壳类动物为食。

须鲸种类少，有[[露脊鲸科|露脊鲸科]]、[[灰鲸科|灰鲸科]]和[[须鲸科|须鲸科]]三个科，共6属11种。其中[[蓝鲸|蓝鲸]]体积庞大；[[露脊鲸|露脊鲸]]行动缓慢、头大体胖；[[座头鲸|座头鲸]]体短臂长，常在近岸活动；[[小须鲸|小须鲸]]体小吻尖。

===齿鲸===
[[齿鲸|齿鲸]]口中无须，但有牙齿，呈圆锥状。齿鲸的鼻孔只有一个。除[[抹香鲸|抹香鲸]]外，体积都较小，最小只有1米左右，最大也在20多米。主要以乌贼、鱼类为食。

齿鲸种类较多，有8个科，大约34属、72种。如[[虎鲸|虎鲸]]和[[海豚|海豚]]。

== 解剖 ==
如同所有哺乳动物，鲸用肺呼吸，是[[温血动物|温血动物]]，体温大约为37℃，心跳每分钟只有10次<ref>{{cite book|author1=李宏|title=动物与海洋（海洋与科技探索之旅 ）|date=2013-12-11|publisher=青苹果数据中心}}</ref>。其幼崽由哺乳喂养，还有少许毛发。一些種類可以潜到極深的深度，它们往往擁有厚重的脂肪層，或叫作[[鲸脂|鲸脂]]，能在冰冷的水中保持温暖。

鲸的体形是梭状的。它的前肢形成鳍，后肢完全退化，尾巴变成尾鳍，可以上下摆动，是游泳的主要器官。有些种类还有背鳍，用来平衡身体。肋骨有10～20对，胃分四个室，肾脏多为瘤状。

特别值得瞩目的是[[蓝鲸|蓝鲸]]。它是世界上现存的最大动物，体长可达30米，重达180吨。

鲸鱼一般上可活40-90年，视品种而定。

=== 头部 ===
鲸的眼睛很小，没有泪腺，视力较差。耳朵在水下环境有其特殊的适应性，没有外耳壳，外耳道也很细，但听觉灵敏，能感受超声波，靠回声定位来寻找食物、联系同伴或逃避敌害。鼻孔有一或两个，在头的上部，俗称喷气孔。一般鼻孔位置越靠后，进化程度越高。

== 繁殖及哺乳 ==
雄鲸睾丸位于腹腔内，利用输精管将精子排入雌性体内。完成受精过程后，精子和卵子在母鲸体内结合。雌鲸子宫为双角形，有一对乳房，位于生殖裂两侧的乳沟内，有细长的乳头。

母鲸通常生一只幼鲸。养育期较长，某些种类多于一年。在此期间，母子之间会建立非常强的关系。一些鲸的成熟期很晚，大约要七到十年。鲸的生殖器官在游泳时避免阻力而缩回体内。母鲸在海水中通过喷射乳汁入幼崽口内来喂养幼鯨，而幼鯨會自動將海水及奶水分離。

== 捕鲸 ==
{{Main|捕鲸}}
捕鲸是年代很久的行業。《[[新唐書|新唐書]]》卷三六《五行志三》：“開成二年三月壬申，有大魚長六丈，自海入淮，至濠州招義，民殺之。”又《新唐書》卷二一九《北狄列傳·黑水靺鞨》記載：“拂涅，亦稱大拂涅，開元、天寶間八來，獻鯨睛……”。鲸肉可食，脂肪可制油，用于医药和其他工业。鲸鱼的种类比较多，其中有不少种类的鲸鱼濒临灭绝，但有些种类的鲸鱼数量比较多。

目前捕鲸的国家有[[挪威|挪威]]、[[冰岛|冰岛]]和[[日本|日本]]，以及[[西伯利亚|西伯利亚]]、[[阿拉斯加|阿拉斯加]]和北[[加拿大|加拿大]]的一些土著部落。关于捕鲸问题，反对的一方主要是一些环保组织如[[绿色和平|绿色和平]]、[[世界自然基金会|世界自然基金会]]等。但是，近年来澳大利亚、巴西等国政府也介入反对捕鲸。

== 文化 ==
人類很早就對鯨魚有深刻的描寫。[[崔豹|崔豹]]《古今注》：“鯨，海魚也，大者長千里，小者數十丈。”“其雌曰鯢，大者亦長千里。”並有大量的文章，如《莊子》曰：“吞舟之魚失水，則螻蟻而能制之。”吞舟之鱼就是大魚，也就是鯨魚。[[張衡|張衡]]《[[西京賦|西京賦]]》有所謂“鯨魚失流而蹉跎”。[[任昉|任昉]]《述異記》：“南海有珠，即鯨魚目瞳，夜可以鑒，謂之夜光。”

《[[淮南子|淮南子]]》還有“鯨魚死而彗星出”之說。《[[論衡|論衡]]·亂龍》亦言：“夫東風至酒湛溢，鯨魚死彗星出，天道自然，非人事也。”《[[太平御覽|太平御覽]]》卷七及卷九三八引《春秋考異郵》都說到“鯨魚死而彗星出”，卷八七五引《春秋考異郵》作“鯨魚死彗星合”。

《太平御覽》卷八七〇引《[[三秦記|三秦記]]》稱“始皇墓中燃鯨魚膏為燈。”

在西方，最早的一個故事乃是《聖經》裡關於[[約拿|約拿]]的記載：水手們把約拿拋入大海，以便平息眼看就要把船毀掉的狂風惡浪。約拿被一頭鯨魚吞下，三天後，又被牠吐在陸地上。不過經文並沒有明確說是鯨魚，只說是一條大魚。《聖經》裡所說的另一種動物[[利維坦|利維坦]]，也很有可能是鯨魚。至中世紀，鯨魚的記載見諸於[[斯堪地那維亞|斯堪地那維亞]]和[[冰島|冰島]]的文獻。其中最重要的一部，當屬13世紀中葉的《{{Link-en|王室寶鑑|Speculum regale}}》，這本書描寫了生活在冰島四周海域的各種鯨魚。<ref>{{Cite book |last=Cohat |first=Yves |others=蔡鴻濱/譯 |date=1994年5月31日 |edition=1999年3月15日初版六刷 |title=《鯨魚生與死：海上帝王的生命之歌》 |series=[[發現之旅叢書|發現之旅]] |volume=[[發現之旅叢書書目列表|10]] |url=https://www.goodreads.com/book/show/36119050 |location=臺北 |publisher=[[時報文化|時報文化]] |pages=第18–19頁 |chapter=第一章：關於鯨魚的傳說—離奇古怪的記載 |language=zh-TW |isbn=978-957-13-1083-1}}</ref>
