'''琼州大桥'''，也被称为'''南渡江第九桥'''，是中国[[海南省|海南省]][[海口市|海口市]][[国兴大道|国兴大道]]跨过[[南渡江|南渡江]]的桥梁<ref>{{Template:Cite web|author = 郭祖莹|url = http://www.hinews.cn/news/system/2012/12/18/015253277.shtml|title = 琼州大桥重要部件材料缺损严重 或将大修(图)_新闻_南海网|publisher = Hinews.cn|date = 2012-12-18|accessdate = 2013-04-12}}</ref><ref>{{Template:Cite web|url = http://www.ludongpo.com/bk/qiaoliang/ShowInfo.asp?InfoID=21873|title = 海口琼州大桥 【南渡江第九桥】[语文大百科-东坡语文]|publisher = Ludongpo.com|date = 2008-12-25|accessdate = 2013-04-12}}</ref>，连接了海口市区和[[海口美兰国际机场|海口美兰国际机场]]，于2003年5月12日建成通车，造价2亿人民币<ref name="china1"><span class="citation web">[http://www.66766china.com/jingqu/xiangqing.asp?id=4269 "海南海口琼州大桥介绍"]. 66766china.com<span class="reference-accessdate">. </span></span></ref><ref>{{Template:Cite web|url = http://news.sina.com.cn/c/2003-05-13/0841118718s.shtml|title = 琼州大桥和海文高速公路海口连接线通车|publisher = News.sina.com.cn|date = 2003-05-13|accessdate = 2013-04-12}}</ref>。
