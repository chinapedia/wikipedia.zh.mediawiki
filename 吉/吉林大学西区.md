'''吉林大学西区'''，位于[[吉林省|吉林省]][[长春市|长春市]][[绿园区|绿园区]]西安大路5333号。

==历史==
;总后勤部卫生部兽医学院
*1949年，在[[北京|北京]][[丰台|丰台]]创建'''总后勤部卫生部兽医学院'''<ref name=hqj/>。
<br>
;中国人民解放军第一兽医学校
*1946年8月，在辽西省（今[[吉林省|吉林省]]）[[白城|白城]]组建'''东北民主联军兽医学校'''<ref name=hqj> {{Cite book | author =  中华人民共和国军事院校教育发展史编辑委员会| title = 中华人民共和国军事院校教育发展史后勤卷 | location =  | publisher =军事科学出版社  | date = 2001年 | pages =  | ISBN =  | accessdate =  | url =  | language =  | quote =  }} </ref>。
*1947年5月，迁至[[黑龙江省|黑龙江省]][[齐齐哈尔市|齐齐哈尔市]]，同年11月更名'''西线战勤兽医学校'''<ref name=hqj/>。
*1949年4月以后，随着解放区扩大，中国人民解放军在接收伪满大陆科学院马疫研究处和国民党第五兽医器材库的基础上，将西线战勤兽医学校迁至[[长春|长春]]，更名'''中国人民解放军东北军区兽医学校'''<ref name=hqj/>。
*1951年，更名'''中国人民解放军第一兽医学校'''。
*1952年，更名'''中国人民解放军高级兽医学校'''<ref name=jsszjsz/>。
<br>
;中国人民解放军第二兽医学校
*1904年，[[清政府|清政府]]在[[直隶|直隶]][[保定|保定]]建立'''北洋马医学堂'''，是中国现代兽医教育的发源地<ref name=xiaobao/>。
*1907年，清政府将[[兵部|兵部]]改组为[[陆军部_(清朝)|陆军部]]，该校于1907年4月归陆军部管辖，更名'''陆军马医学堂'''<ref name=xiaobao/>。
*1912年，陆军马医学堂更名'''陆军兽医学校'''<ref name=xiaobao/>。
*1928年，该校由[[南京国民政府|南京国民政府]]军政部接管，继续招收新生<ref name=xiaobao/>。
*1949年11月18日，[[贵州|贵州]][[安顺|安顺]]解放，在[[西南军区|西南军区]]党委领导下，该校改组为'''中国人民解放军西南军区兽医学校'''，设有解剖、生理、药理、病理、细菌、寄生虫、勤务、卫生、外科、内科、蹄铁、畜牧、畜产等13个系<ref name=xiaobao/>。
*1951年，更名'''中国人民解放军第二兽医学校'''<ref name=xiaobao/>。
<br>
;中国人民解放军第三兽医学校
*1949年，在[[江苏|江苏]][[淮安|淮城]]创建'''中国人民解放军华东军区兽医学校'''<ref name=hqj/>。
*1951年上半年，由华东军区后方勤务部卫生部兽医学校组成新的兽医学校。1951年10月7日，中央军委电令，该兽医学校更名'''中国人民解放军第三兽医学校'''<ref name=jsszjsz> {{Cite book | author =江苏省地方志编纂委员会  | title = 江苏省志·军事志 | location =  | publisher =  军事科学出版社 | date =2000年  | pages = 213 | ISBN =  | accessdate =  | url =  | language =  | quote =  }} </ref>。
<br>
;中国人民解放军第四兽医学校
*1951年9月3日，在[[湖北|湖北]][[武汉|武汉]]创建'''中国人民解放军中南军区兽医学校'''<ref name=hqj/>。
*1951年10月7日，更名'''中国人民解放军第四兽医学校'''。
<br>
;中国人民解放军军需大学
*1953年1月1日，根据中央军委命令，将中国人民解放军高级、第二、第三、第四兽医学校合并，在长春市成立'''中国人民解放军兽医大学'''。隶属[[中国人民解放军总后勤部|中国人民解放军总后勤部]]。是[[中国人民解放军|中国人民解放军]]唯一一所培养畜牧、兽医科技人才的高等院校<ref name=jlszjsz> {{Cite book | author = 吉林省地方志编纂委员会 | title = 吉林省志·卷十四·军事志 | location =  | publisher = 吉林人民出版社 | date = 1996年 | pages =145 | ISBN =  | accessdate =  | url =  | language =  | quote =  }} </ref><ref name=xiaobao/>。[[朱德|朱德]]题写校名<ref name=ding/>。
*1956年1月1日，[[中华人民共和国国务院|国务院]]决定将该校移交[[中华人民共和国农垦部|中华人民共和国农垦部]]<ref name=jlszjsz/>。更名'''长春畜牧兽医大学'''。
*1958年，长春畜牧兽医大学撤销，参与合并组建长春农学院（后为[[吉林农业大学|吉林农业大学]]）。
*1961年11月，国务院决定将该校交还中国人民解放军，仍称'''中国人民解放军兽医大学'''，隶属中国人民解放军总后勤部建制<ref name=jlszjsz/>。
*1964年12月，[[中国人民解放军军事医学科学院|中国人民解放军军事医学科学院]]军马卫生科学研究所由[[北京|北京]]迁到长春，转隶兽医大学<ref name=jlszjsz/>。
*1978年以后，国家恢复研究生制度，该校被批准为首批博士研究生、硕士研究生授予单位<ref name=jlszjsz/>。
*1992年8月，更名'''中国人民解放军农牧大学'''。[[江泽民|江泽民]]题写校名<ref name=ding>丁洪浩，中国人民解放军军需大学简介，吉林教育科学：普教研究2001年第3期</ref>。
*1999年4月，中央军委主席江泽民签发命令，改建为'''中国人民解放军军需大学'''<ref name=ding/>。
<br>
;吉林大学西区
*2003年6月23日，中央军委决定撤消军需大学，整体移交地方继续办学<ref name=xiaobao/>。
*2004年5月29日，《国务院办公厅　中央军委办公厅关于军需大学等四所军队院校移交归属意见的复函》（国办函〔2004〕40号）发出，决定“军需大学，移交[[中华人民共和国教育部|教育部]]管理，与[[吉林大学|吉林大学]]合并，相对独立办学。”<ref>{{Cite web |url =http://www.gov.cn/zwgk/2005-08/26/content_26165.htm  |title =  国务院办公厅　中央军委办公厅关于军需大学等四所军队院校移交归属意见的复函|author =  |date =  2005-08-26|publisher = 中国政府网 |language =  |accessdate =  |quote =  }}</ref>
*2004年8月29日，中国人民解放军军需大学移交教育部与吉林大学合并仪式在长春举行，军需大学即日起正式与吉林大学合并，更名'''吉林大学和平分校'''，相对独立办学，具有独立法人资格<ref>{{Cite web |url =  http://www.people.com.cn/GB/jiaoyu/1053/2747309.html|title = 军需大学移交教育部与吉林大学合并仪式举行 |author =  |date = 2004-08-29 |publisher = 人民网 |language =  |accessdate =  |quote =  }}</ref>。
*2004年12月30日，改称'''吉林大学农学部'''<ref name=xiaobao/>。
*2005年1月7日，根据《关于军需大学并入吉林大学后管理模式的若干意见》，明确了“军需大学并入吉林大学后，成为吉林大学的一个校区（和平校区），改称农学部。学校对学部实行统一领导和管理，按学校授权，学部相对独立管理校区事务。”由此，原军需大学成为'''吉林大学和平校区'''，改称'''吉林大学农学部'''<ref>{{Cite web |url = http://edu.people.com.cn/GB/xiaoyuan/13518596.html |title =吉林大学校友爱心基地揭牌仪式在大连举行  |author =  |date = 2010-12-18 |publisher = 人民网 |language =  |accessdate =  |quote =  }}</ref>。
*2008年9月9日，世界首例“带有抗猪瘟病毒基因的[[克隆|克隆]]猪”在吉林大学农学部诞生<ref name=xiaobao/>。
*2014年8月2日，吉林大学在和平校区召开农学部办学模式调整和管理体制改革动员会议，标志着该项调整改革工作正式启动。改革目标是调整农学部相对独立管理校区事务的办学模式，强化“校院两级”管理，优化和平校区办学布局和办学功能<ref>{{Cite web |url =http://news.jlu.edu.cn/info/1126/24686.htm  |title =  吉林大学启动农学部办学模式调整和管理体制改革工作|author =  |date = 2014-08-02  |publisher = 吉林大学新闻网 |language =  |accessdate =  |quote =  }}</ref>。
*2014年，改为'''吉林大学西区'''<ref>{{Cite web |url =http://jl.sina.com.cn/news/b/2014-12-31/detail-iawzunex8570221.shtml  |title = 吉大图书馆副馆长许春芳私设小金库被免职 |author =  |date = 2014-12-31 |publisher = 新浪 |language =  |accessdate =  |quote =  }}</ref>。设西区综合办公室，作为吉林大学的派出机构<ref>{{Cite web |url = http://www.jlu.edu.cn/jgsz/gljg.htm |title = 管理机构 |author =  |date =  |publisher =吉林大学  |language =  |accessdate = 2017-10-04 |quote =  }}</ref>。
