'''吉列爾莫·維拉斯'''（{{lang-es|Guillermo Vilas}}，{{bd|1952年|8月17日|catIdx=V}}），出生於[[阿根廷|阿根廷]]的[[布宜諾斯艾利斯|布宜諾斯艾利斯]]，阿根廷男子職業[[網球|網球]][[運動員|運動員]]，在1969年正式參加[[职业网球联合会|ATP職業巡迴賽]]，現已退役。

他是公開賽年代第一位奪得大滿貫單打（包括男女）的[[南美洲|南美洲]]球員，在其職業生涯中最高單打排名為世界第二。 生涯獲得單打929勝-286負，奖金共492,3882美元。他的綽號是「潘帕斯公牛」。

== 技術分析 ==

維拉斯是一名[[網球戰略|防禦型底線型]]球員，單手反拍的[[左撇子|左撇子]]，善長底線抽擊及上旋球。由於他善長打[[泥地球場|-{A]]賽、他被媒體稱為'''紅土王'''，他職業生涯的62個ATP單打冠軍，有48個在-{泥地}-賽場（紅-{土}-或綠土）上奪得的。 目前他是獲得ATP-{泥地}-賽事單打冠軍次多的球員。 

此外，他在其他場地如硬地、草地和地毯亦有一定成績，例如他於1974年在草地舉行的年終大師賽單打決賽中，他就擊敗[[羅馬尼亞|羅馬尼亞]]前球王[[伊利耶·納斯塔塞|納斯塔塞]]而奪標。 

1975年，他首次打入了'''法網'''男單決賽（亦是個人首個[[網球大滿貫|大滿貫]]男單決賽），以直落三盤（比分 : 6-2, 6-3, 6-4）敗於衛冕冠軍[[瑞典|瑞典]]的[[比约恩·博格|博格]]，無緣冠軍。 此後，同屬為'''紅土王'''的博格也成為了他職業生涯的剋星，'''維拉斯對博格'''擁有'''極度劣勢的對賽成績 : '''[http://www.atpworldtour.com/en/players/fedex-head-2-head/guillermo-vilas-vs-bjorn-borg/V028/B058/ '''5-18''']，所以作為紅土王的他也只贏過一次法網男單冠軍。

== 1977年 ==
[http://www.atpworldtour.com/Tennis/Players/Vi/G/Guillermo-Vilas.aspx?t=pa&y=1977&m=s&e=0 '''1977年''']，是維拉斯職業生涯中最輝煌的一年。

該年，他曾創下'''-{泥地}-場（紅-{土}-或綠土）53連勝的紀錄'''，直至2006年6月，才被[[西班牙|西班牙]]的[[拉法埃尔·纳达尔|纳达尔]]打破。 

至今他仍保持所有賽事單打最長連勝（1977年內的'''46場'''，包括[[戴维斯杯|戴维斯杯]]）、一年奪得ATP單打錦標最多（'''1977年的16個ATP單打冠軍'''，包括[[法網|法網]]和[[美網|美網]]兩項[[網球大滿貫|大滿貫]]）、唯一一位一年內在五大洲都奪得過ATP單打冠軍（'''1977年'''在[[亞洲|亞洲]]、[[非洲|非洲]]、[[歐洲|歐洲]]、[[北美洲|北美洲]]、[[南美洲|南美洲]]五洲奪標）的球員等紀錄。

當年球季（賽季橫跨1977年至1978年初），他取得'''134勝14負'''的ATP單打成績（勝率為'''90.54%'''），進入22個男單決賽並取得'''16個ATP單打冠軍'''，年終世界排名為第2。

一月，他進入了草地進行的[[澳網|澳網]]男單決賽，以直落三盤3-6敗於[[美國|美國]]球員Roscoe Tanner，獲得亞軍。 

同年的'''法網'''男單比賽，他又打入了決賽，以6-0, 6-3, 6-0擊敗了另一美國球員Brian Gottfried，首奪大滿貫冠軍。

同年的'''美網'''綠土賽男單决賽中，正是作為底線專家的他，以别出心裁地以[[發球上網|發球上網]]打法智取衛冕的美國名將[[吉米·康诺尔斯|康诺尔斯]]而首奪冠軍。  

他在'''温布尔登'''網球錦標賽後 [http://www.menstennisforums.com/showthread.php?t=95300 單打連勝46場]（包括兩場戴维斯杯對[[澳大利亚|澳大利亚]]的男單賽事），並連續奪得Kitzbuhel、華盛頓、路易維爾、南橙郡、哥倫布市、美網、巴黎共七個ATP單打冠軍，直至[[法國|法國]][[普羅旺斯埃克斯|普羅旺斯埃克斯]]的-{zh-hans:红土;zh-hk:泥地;zh-tw:紅土;}-賽男單決賽中，直落落后兩盤（1-6, 5-7）於宿敵納斯塔塞時，指控對手使用非法球拍Spaghetti String，憤而中途退出比賽，-{泥地}-場53連勝和所有賽事46場連勝的紀錄被鎖定，ATP事後亦將Spaghetti String球拍列為禁用球拍。

之後，他又在10月至11月內連奪五項ATP單打錦標（德黑蘭、波哥大、聖地牙哥、布宜諾斯艾利斯、约翰内斯堡 WCT）。

當年度的'''年终大师赛'''中，他在分組賽擊敗了宿敵康诺尔斯，卻在半决赛敗於剋星博格，無緣冠軍。 

雙打方面，他和羅馬尼亞搭擋伊翁·提里亞克（Ion Tiriac）合作，奪得四項ATP雙打錦標（巴爾的摩、布宜諾斯艾利斯、尼斯、德黑蘭）。

那年ATP單打年終世界排名上，維拉斯屈居世界第一的干納斯（當年的'''温布尔登'''和'''美網'''的單打雙料亞軍、'''年终大师赛'''冠軍）之後，但當年著名的《世界網球》雜誌給了他'''世界排名第一'''的稱號。

當年，他和年終世界第一的康诺尔斯的對戰紀錄為'''2勝0負'''。


== 蟬聯[[澳網|澳網]] ==

1978年，他首度奪得當時還在草地舉行的'''澳網'''男單冠軍。  1978年的男單決賽（在1979年1月舉行），他擊敗了東道主的John Marks而奪標。  在'''法網'''亦打入男單決賽，卻再度負於剋星博格，無緣冠軍。 當年他一共奪得漢堡、慕尼黑、格斯塔德、南橙郡、普羅旺斯埃克斯、巴塞爾和澳網共七項ATP巡迴賽賽單打冠軍，不過年終排名已降至第三位。

1979年，他又蟬聯'''澳網'''男單冠軍，當年的男單決賽（在1980年1月舉行），他擊敗了美國的John Sadri，成功衛冕。


== 後期生涯 ==

1981年，他帶領阿根廷打入'''戴维斯杯'''決賽，負於美國而奪得亞軍。

1982年，他第四度進入'''法網'''男單決賽，負於瑞典新星[[马茨·维兰德|维兰德]]而奪得亞軍。 他在當年的美網和年終大師賽亦打入了半决赛。 同年，他迴光返照式的奪得七項ATP單打錦標（布宜諾斯艾利斯、鹿特丹、米蘭、蒙地卡羅、馬德里、波士頓、Kitzbuhel），年終世界排名第四位。 其中四冠包括分別在兩次在紅土決賽擊敗紅土專家[[伊万·伦德尔|伦德尔]]，以及兩次在地毯決賽打敗該年溫布頓-美網單打雙冠王康诺尔斯，後兩者均是當年當時得令的球員。

1989年，他正式退出ATP職業巡迴賽，但直至1992年，他仍參加ATP挑戰賽。 

他亦是眾多未能登上ATP單打球王寶座（世界排名第一，以1973年 ATP電腦世界排名實施後開始計算。）的球員中，生涯奪得最多ATP單打冠軍最多的球員。 （'''62個'''，ATP史上單打冠軍最多第九位，排在'''109冠'''的康诺尔斯、'''97冠'''的[[罗杰·费德勒|费德勒]]、'''94冠'''的伦德尔、'''77冠'''的[[约翰·麦肯罗|约翰·麦肯罗]]、'''75冠'''的纳达尔、'''68冠'''的[[諾瓦克·喬科維奇|祖高域]]、'''64冠'''的[[皮特·桑普拉斯|桑普拉斯]]和博格之后，這八位名將均曾至少兩次成為ATP單打年終世界排名第一。） 

他生涯獲得ATP單打戰績為'''929勝'''286負。 （ATP史上單打獲勝最多第四位，排在康诺尔斯、伦德尔和费德勒之后。）

他在網球壇的卓越成就帶動了阿根廷甚至整個南美洲的網球發展， 阿根廷名將[[吉列爾莫·卡尼亞斯|卡纳斯]]和[[吉列尔莫·科里亚|科里亚]]均以他的名字'''吉列尔莫'''命名。

1991年，他被引薦進入[[國際網球名人堂|國際網球名人堂]]。

2004年，他在法網男單決賽後，為兩名同胞 - 冠軍[[加斯顿·高迪奥|高迪奥]]和亞軍哥利亞頒獎。

他是2005年美國[[TENNIS_Magazine#The_40_Greatest_Players_of_the_Tennis_Era|TENNIS雜誌評出最偉大的40位球員]]中的第24位。

2013年8月，維拉斯[http:///www.10sballs.com/2013/08/21/vilas-asks-atp-to-reconsider-giving-him-no-1-ranking-for-1977 要求ATP重新考慮給予他在1977年世界排名第一。]
