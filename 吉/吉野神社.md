'''吉野神社'''位於[[臺灣|臺灣]][[花蓮港廳|花蓮港廳]]底下[[吉野村|吉野村]]的宮前部落（今[[花蓮縣|花蓮縣]][[吉安鄉|吉安鄉]]慶豐村的新市場後方），是因為吉野村沒有讓日本移民奉祀的「御神靈」而興建的神社，主祀[[大國魂命|大國魂命]]、[[大己貴命|大己貴命]]、[[少彥名命|少彥名命]]、[[能久親王|能久親王]]{{r|潘}}。現僅殘留其鎮座紀念碑{{r|潘}}。

== 沿革 ==
吉野村為明治四十三年（1910年）開闢的日本移民村，由於剛開始時沒有神社，臺灣總督府遂在吉野村宮前部落取得一甲五分的地來興建神社，滿足居民的宗教信仰需求{{r|潘}}。神社於明治四十五年（1912年）1月開始動工，於5月30日落成，分靈自[[臺灣神社|臺灣神社]]{{r|潘}}。同年6月8日，臺灣神社宮司清水廣忠前來擔任齋主（主祭），花蓮港廳廳長中田直溫擔任祭典長，舉行了鎮座式{{r|潘}}。
== 祭典 ==
[[File:Yoshino_Shrine鎮座紀念2018.jpg|thumb]]
吉野村建設較久，所以神社祭典亦比豐田、林田神社來得頻繁。<ref name="張素玢">{{cite book |last=張素玢 |first= |date= |title=《未竟的殖民-日本在臺移民村》 |url= |location= |publisher=衛城出版 |page= |isbn= |author-link= }}</ref>
*元始祭 : 1月1日
*祈年祭 : 6月8日
*大祓祭 : 6月30日、12月31日
*除夜祭 : 12月31日
*始政紀念祭
*天長節
*神嘗祭
 
吉野神社的例祭日原本定在6月8日，但後來因為正值農忙，且天氣炎熱令環境衛生堪憂，所以在大正十二年（1923年）時改為10月8日{{r|潘}}。
