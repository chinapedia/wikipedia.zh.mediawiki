'''吉川 麻衣子'''（きっかわ まいこ、{{bd|1985年|3月28日|||Kikkawa Maiko}}），[[廣島縣|廣島縣]]出身<ref name="girls-record">{{Cite web|author=|date=|url=http://www.girls-record.jp/artists/kikkawa_art.html|title=公式プロフィール|work=[[GIRLS'_RECORD|GIRLS' RECORD]]|publisher=|page=|accessdate=2014-06-13|deadurl=yes|archiveurl=https://web.archive.org/web/20140618141220/http://www.girls-record.jp/artists/kikkawa_art.html|archivedate=2014-06-18}}</ref><ref>[[吳市|呉市]]にある倉橋島の出身<sup class="noprint Template-Fact">[''<span title="この記述には信頼できる情報源の提示が求められています。（2014年6月）">要出典</span>'']</sup><sup class="noprint Template-Fact"></sup>。</ref>。曾為[[日本|日本]]的[[寫真偶像|寫真偶像]]，當時隸屬在K-point事務所下，現已退出藝能界。

== 簡歷 ==
吉川麻衣子有位年紀小5歳的弟弟<ref>{{Template:Cite web|url = http://ameblo.jp/kikkawamaiko/entry-10956426069.html|title = 質問の答え☆|work = オフィシャルブログ「マイページ」|date = 2011-07-17|accessdate = 2012-12-15|archiveurl = https://archive.today/LAkfgl|archivedate = 2013-04-30}}{{Dead link}}</ref>。2005年間以身為Fitone（フィットワン）藝能公司所推出的「GIRLS' RECORD」成員之一出道。身材雖嬌小但比例很好，並且有著可愛的治癒系外貌，以此在寫真女星中獲得喜愛。

在2006年間也挑戰出演短劇，如參加了首次公開演出的「劇團水果蛋糕」（劇団ショートケーキ）等。2009年間，以減肥團體「在意卡路里會？」（カロリー気にしてる会?）」團長的身分，於網路上發表「1個月內要減重3公斤」的挑戰宣言，並順利的達成此目標。

2011年間曾轉籍到GORDIE ENTERTAINMENT藝能公司並回到演藝圈<ref>{{Template:Cite web|url = http://ameblo.jp/kikkawamaiko/entry-10927107972.html|title = 移籍しました!|work = オフィシャルブログ「マイページ」|date = 2011-06-18|accessdate = 2012-12-15|archiveurl = https://archive.today/AjOqO|archivedate = 2013-04-30}}{{Dead link}}</ref>，不過於2012年間因業務合作的緣故又轉籍到K-point藝能公司。

2014年6月12日當天，吉川所屬的藝能公司突然將她的姓名從網路上刪除，她的部落格也遭關閉。隔日，週刊「FRIDAY」刊出吉川與[[DOWN_TOWN|DOWN TOWN]]成員[[濱田雅功|浜田雅功]]的婚外情報導，浜田雅功承認了這段婚外情關係並對外謝罪。當時浜田的妻子小川菜摘於接受訪問時說：「（既然浜田）是藝人，從跟他結婚起就有他會外遇的覺悟了，真是對不起。（他與吉川）只是一時興起不是嗎？因為丈夫對我很重要，所以我完全沒有任何不滿。如果他變成不受女性歡迎的話，藝人生涯就要結束了呢。受歡迎的丈夫我覺得很帥呢。」，對婚姻完全沒有展現出任何的動搖。

在2014年6月14日這天，確認吉川與K-point間的契約已於同年4月間終止。

吉川常在網路現場直播的「ひかり荘」節目中擔任與網友聊天的主持人，該節目曾於2007年1月開始的數次節目中播放「自組電腦」（[[組裝電腦|自作パソコン]]）的企畫，藉由Intel（[[英特尔|インテル]]）公司的職員與粉絲所提供的線上意見成功的將電腦組裝完成，並經由公開募集將該電腦命名為「パソ太」。之後吉川也確實將這台電腦拿回家使用。
