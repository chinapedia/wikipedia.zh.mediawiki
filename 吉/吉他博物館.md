'''吉他博物館'''（{{lang-en|'''Guitars – the Museum'''}}）是一座位於[[瑞典|瑞典]][[西博滕|西博滕]]省[[于默奧|于默奧]]市，專門收藏[[電吉他|電吉他]]的[[樂器|樂器]][[博物館|博物館]]。博物館建築物座落於原先屬於{{tsl|en|Vasaskolan, Umeå|Vasaskolan}}學校的磚造校舍內。建築物內還有搖滾俱樂部、餐廳、音樂商店及唱片小舖。博物館選於2014年1月底開幕，與[[于默奧|于默奧]]市成為2014年的[[歐洲文化之都|歐洲文化之都]]相呼應<ref>{{cite web|title=Umeå2014 - European Capital of Culture|url=http://umea2014.se/en/|accessdate=2014-04-12|language=en}}</ref>。

博物館由音樂商店：「4Sound」，以及搖滾俱樂部：「{{tsl|en|Scharinska}}」（即使該俱樂部從{{tsl|en|Scharinska villan|Scharinska villan}}搬到Vasaskolan後，仍然保留原名）<ref>{{cite web|url=http://www.svt.se/nyheter/regionalt/vasterbottensnytt/scharinska-flyttar |title=Nöjespalatset Scharinska flyttar |publisher=[[SVT|SVT]] |accessdate=2014-03-14 |language=sv |deadurl=yes |archiveurl=https://web.archive.org/web/20140322001745/http://www.svt.se/nyheter/regionalt/vasterbottensnytt/scharinska-flyttar |archivedate=2014-03-22 }}</ref> 之幕後合夥人們參與共同營運，這些人同時也是吉他博物館的投資者<ref>{{cite web |url=http://umea2014.se/2013/08/gitarrmuseet-skruvar-upp-volymen/ |date =2013-08-29 |title=Gitarrmuseet skruvar upp volymen |publisher=Umea2014.se |accessdate=2014-03-22 |language=sv}}</ref><ref>{{cite web |url=http://www.guitarsthemuseum.com/static/sv/88/ |title=Guitars - the Museum, webbsida |accessdate=2014-03-14|language=sv}}</ref>。[[于默奧|于默奧]]市政府也提供對博物館事業的館舍裝修及房舍編配支援<ref>{{cite web |url=https://www.mynewsdesk.com/se/umea/pressreleases/18-milj-kr-foer-musik-i-vasaskolan-850091 |title=18 milj kr för musik i Vasaskolan |publisher=Umeå kommun |date=2013-03-25 |accessdate=2014-03-22|language=sv}}</ref>，並資助從2014年到2015年間達2400萬瑞典克朗的年度資金<ref>{{cite web |url=http://umea2014.se/2012/12/gitarrmuseum-blir-ett-unikt-besoksmal-i-umea/ |title=Gitarrmuseum blir ett unikt besöksmål i Umeå |publisher=Umeå2014.se |date=2012-12-19 |accessdate=2014-03-22|language=sv}}</ref>。

== 藏品 ==
博物館內的主要藏品為1950年代至1960年代的[[電吉他|電吉他]]，也有小量的電子貝斯吉他、放大器及歷史上與電吉他相關的週邊用具。這些藏品原由Mikael與Samuel Åhdén兩兄弟在1970年代所收藏。然而，這些收藏在博物館開幕前，均不對外公開展示<ref>{{cite web |url=http://www.umea.se/arkiv/profilnyheter/profilartiklar/denhemligagitarrsamlingen.5.1a5fea8a1437b3e6e52caa2.html |title=Den hemliga gitarrsamlingen |publisher=Umea.se |language=sv|accessdate=2014-03-21}}</ref>。

藏品中較為特別的有：1950年[[芬達樂器公司|芬達樂器公司]]的{{tsl|en|Fender Broadcaster|芬達廣播者}}、1958年[[吉普森吉他公司|吉普森吉他公司]]的{{tsl|en|Gibson Flying V|吉普森飛行者 V}}及1960年的[[萊斯·保羅|萊斯·保羅]]所設計的電吉他。由於博物館收藏超過五百把吉他，因此吸引國際的目光<ref>{{cite web |url=http://highlife.ba.com/News-And-Blogs/Culture-Blog/Guitars-The-Museum-opens-Sweden.html |title=Guitars - The Museum opens in Sweden |publisher=British Airways |accessdate=2014-03-21}}</ref>，同時也是世界上最豐富的吉他收藏場所<ref>{{cite web |url=http://www.esquire.co.uk/culture/article/4936/worlds-largest-guitar-museum-opens/ |title=Worlds largest guitar museum |publisher=Esquire |accessdate=2014-03-14 |deadurl=yes |archiveurl=https://web.archive.org/web/20140322001613/http://www.esquire.co.uk/culture/article/4936/worlds-largest-guitar-museum-opens/ |archivedate=2014-03-22 }}</ref> 。

此外，除了博物館本身的收藏品外，館內還開闢一個臨時展場：為了標示博物館的開幕，展出名為「''于默奧 - 歐洲核心之都 1989-2000''」（''Umeå - The European capital of hardcore 1989-2000''）的{{tsl|sv|Folkrörelsearkivet i Västerbotten|西博滕群眾運動檔案}}<ref>{{cite web |url=http://www.vk.se/plus/1106786/umeas-hardcore-tar-plats-pa-museum |publisher=Västerbottens Kuriren |title=Umeås Hardcore tar plats på museum  |language=sv |accessdate=2014-03-21}}</ref>。
