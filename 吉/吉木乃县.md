'''吉木乃县'''（[[维吾尔语|维吾尔语]]：جېمىنەي）是[[中国|中国]][[新疆维吾尔自治区|新疆维吾尔自治区]][[阿勒泰地区|阿勒泰地区]]下辖的一个[[县|县]]，位于新疆北部边陲，[[准噶尔盆地|准噶尔盆地]]北缘，[[萨吾尔山|萨吾尔山]]北麓，[[额尔齐斯河|额尔齐斯河]]南岸，地理坐标为北纬47°00′－47°59′，东经85°33′－87°09′，西与[[哈萨克斯坦共和国|哈萨克斯坦共和国]]接壤，边境线长141公里，东与[[福海县|福海县]]毗邻，南与[[塔城地区|塔城地区]]的[[和布克赛尔蒙古自治县|和布克赛尔蒙古自治县]]相连，北接[[哈巴河|哈巴河]]、[[布尔津|布尔津]]两县。

== 历史沿革 ==
“吉木乃”系氏族名，为乌古斯汗国九姓氏族之一，又名“吉别乃”，“吉木乃”为“吉别乃”的音译演变。清朝[[平定准噶尔|平定准噶尔]]后，隶属伊犁将军辖下的塔尔巴哈台参赞大臣管理，1906年科布多、阿勒泰分治时划归阿勒泰办事大臣辖。辛亥革命后，于1916年设[[县佐|县佐]]；1930年正式建立县治，县城始设在吉木乃（今吉木乃镇所在地）；1962年4月原县城划归农十师一八六团管辖，县城迁至托普铁热克镇。

== 地理 ==
地图上，吉木乃县犹如一个等腰直角梯形。境内东西长约120公里，南北宽约110公里，行政区域面积8222平方公里。

吉木乃县处于哈萨克斯坦——准噶尔板块，西[[准噶尔|准噶尔]]古生代弧盆系，塔尔巴哈台——萨吾尔岛弧带。地质构造发育，岩浆侵入及火山活动频繁，矿产比较丰富，各种矿产已达28大类107种。

吉木乃县已开发利用的矿产有：[[煤|煤]]、[[金|金]]、[[天然碱|天然碱]]、[[盐|盐]]、[[大理岩|大理岩]]、[[粘土|粘土]]、[[硭硝|硭硝]]、[[石膏|石膏]]等。 吉木乃县山川景色幽美、雄奇，草原文化古迹众多，夏季气候宜人。

== 人口与民族 ==
吉木乃县全县总人口为37733人，其中男性19090人，女性18643人；汉族13209人，全县共有少数民族15个24524人；非农业人口18530人，农业人口19203人；城镇人口总数10121人（2005年）。

[[哈萨克族|哈萨克族]]为该县世居民族，占总人口60％以上。

== 主要旅游景点 ==
[[木斯岛|木斯岛]]冰山，塔特克什岩石画，北沙窝（夏夏合特沙漠）沙漠植物园，国门，中哈界桥、界碑，吉木乃口岸，第四纪冰川遗迹，萨吾尔高山草原，托海阔拉斯岩画，森塔斯湖石人及墓群，吉恩思汗墓，拉斯特河谷风景区，哈尔交河风景区等名胜古迹。
