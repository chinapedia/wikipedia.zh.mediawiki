'''《萌王EX》''' （英语：Kawaii Strike: Cute to Kill）是由上海萌宫坊网络科技有限公司研发，网易游戏代理的一部二次元3D动作战术竞技[[手游|手游]]。在《萌王EX》里玩家能操纵历史上真实存在过的、[[性别转换|性别转换]]的帝王们，在名为[[上海|魔都]]的战场上互相交战。<ref name="腾讯游戏/游戏">{{cite news |language = zh |author =  |coauthors =  |url = http://xw.qq.com/games/20160729036963/GAM2016072903696300 |title = 萌王ex制作人陈浩：9月将对玩家开放初次测试 |publisher = 腾讯游戏 |pages = |date =   |accessdate =2016-07-29 }}</ref>

==制作背景==
制作团队企图把萌王EX打造成一款中国玩家喜爱的[[二次元|二次元]]IP（'''游戏素材的版权'''）。为了能贴合现代年轻人的审美观，打造出一群玩家热爱的有趣角色，制作团队找来了深度了解二次元文化的年轻人，以及最热爱动漫题材的玩家们为角色的塑造共同出力。<ref name="任玩堂">{{Cite web |url = http://www.appgame.com/archives/550259.html |title = 专访《萌王EX》主创团队：打造中国玩家喜爱的二次元IP |author = [http://m.appgame.com/archives/author/lichenghao 寿司] |date =  |publisher = 任玩堂 |language = zh |accessdate =2016-07-30  |quote = }}</ref>

制作团队为了可以吸引到足够多的玩家，因此游戏采用原创世界观，而且以[[性别转换|性别转换]]来当作游戏的特点——将历史课本上耳熟能详的帝王君主全部扭转性别，又以严肃而风趣的考据心态加以重塑，赋予这些帝王可爱的少女形象。这种设定不仅在国内引起反响，在日本也引起热议。<ref>{{Cite web |url = http://tieba.baidu.com/p/4665961304 |title = 天朝手游在11区大热！ 《萌王EX》性转王者日本引热议 |author = [http://tieba.baidu.com/home/main?un=%E6%84%9F%E6%80%A7%E8%88%AC%E6%B8%85%E6%BE%88&ie=utf-8&fr=pb 感性般清澈] |date =   |publisher = 百度贴吧 |language = zh |accessdate =2016-07-12  |quote = }}</ref><ref name="日本热议">{{Cite web |url = http://wap.gamersky.com/news/Content-778139_2.html |title = 《萌王EX》性转设定在日本引热议 惊呼天朝脑洞太大 |author =  |date =   |publisher = [http://www.gamersky.com/ 游民星空] |language = zh |accessdate =2016-07-12  |quote = }}</ref>

==游戏设定==
===角色===
玩家在游戏里操控名为“君王”的一类角色。君王是王权力量的拥有者。
在萌王EX世界观里出现的君王角色全都并非历史上的本人，是非现实界域里的存在，人类共识的产物。<ref>{{Cite web |url= http://www.th7.cn/game/ios/201607/228735.shtml#fromapp|title= 萌王EX手游新手必看攻略 王权量级介绍|accessdate= 2016-07-13 |work= |publisher=第七城市 |date=|language=zh }}</ref>

萌王ex中的君王分为六大类型：战士、盾卫、刺客、射手、法师还有辅助。<ref>{{Cite web |url= http://www.9game.cn/mwex/1002219.html|title= 萌王ex君王人物哪个好 萌王ex人物图鉴大全|accessdate= 2016-07-11 |work= |publisher= 九游 |date=|language=zh|author=玉米 }}</ref>
====王权量级====
王权量级是君王所持有王权力量的衡量标准。描述这些位于人类社会顶点的王者们达到高度和影响力。<ref name="评测"/>
===使魔===
《萌王EX》里没有传统[[MOBA|MOBA]]（多人在线战术竞技游戏，{{lang|en|Multiplayer Online Battle Arena Games}}）的装备体系。取而代之的是玩家需要通过在战斗中获取以太资源（类似[[蓝条|蓝条]]），来解锁和召唤使魔助阵。<ref name="评测">{{Cite web |url= http://m.ali213.net/pingce/160708/8205_2.html|title= 《萌王EX》评测：反转性别的君主君临天下|accessdate= 2016-07-08 |work= 手机游戏游戏评测 |publisher= 掌上游侠|date=|language=zh|author= }}</ref>
