'''费青'''（{{bd|1907年||1957年|}}）号'''图南'''，又号'''仲南'''，曾用笔名'''胡冈'''，[[江苏省|江苏省]][[吴江县|吴江县]]人，[[中华民国|中华民国]]及[[中华人民共和国|中华人民共和国]]法学家<ref name=ceji>{{Cite web |url =  http://mt.sohu.com/20170322/n484201465.shtml|title = 千古文章未尽才——法学家费青先生侧记 |author =  |date = 2017-03-22 |publisher = 搜狐 |language =  |accessdate =  |quote =  }}</ref><ref name=hexun>{{Cite web |url = http://news.hexun.com/2015-05-25/176126594.html |title =遥望费青：一个法科学人的背影  |author =  |date =2015-05-25  |publisher = 和讯 |language =  |accessdate =  |quote =  }}</ref>。

==生平==
1926年，费青从[[东吴大学|东吴大学]]医预科转入设在上海的[[东吴大学法学院|东吴大学法学院]]学习。1929年毕业，经东吴大学法学院院长[[吴经熊|吴经熊]]介绍，到[[四川|四川]]新改为国立的国立成都大学（[[四川大学|四川大学]]的前身，时任校长[[张澜|张澜]]）任教，讲授[[国际公法|国际公法]]、[[罗马法|罗马法]]、[[英美法|英美法]]等课程。1931年回到上海，经老师[[刘世芳|刘世芳]]介绍，进入[[暨南大学|暨南大学]]任讲师，讲授罗马法课程。同年下半年赴[[北平|北平]]，经[[刘志敭|刘志敭]]介绍，在北平朝阳学院（即[[朝阳大学|朝阳大学]]，1930年后更名为朝阳学院）任讲师，讲授英美法课程。1932年，费青曾在上海和同学[[袁仰安|袁仰安]]等人担任执业律师<ref name=ceji/><ref name=hexun/>。

1934年，参加[[清华大学|清华大学]]第二届留美公费考试，与[[张光斗|张光斗]]、[[宋作楠|宋作楠]]、[[曾炳钧|曾炳钧]]、[[赵九章|赵九章]]、[[钱学森|钱学森]]、[[夏鼐|夏鼐]]等20人被录取，费青是该年法科录取的唯一一人（国际私法门）。大多数被录取学生去了[[美国|美国]]，少数去[[英国|英国]]，仅赵九章和费青选择留学[[德国|德国]]，且同去[[柏林大学|柏林大学]]。1935年，费青到柏林大学法学研究院研究[[国际私法|国际私法]]及相关学科<ref name=ceji/><ref name=hexun/>。

1938年春，费青学成归国。曾到南洋群岛（[[印度尼西亚|印度尼西亚]]）看望大哥[[费振东|费振东]]，并在[[苏门答腊|苏门答腊]]、[[槟榔屿|槟榔屿]]休养半年多。其间，弟弟[[费孝通|费孝通]]获博士学位后经[[新加坡|新加坡]]回国，费青与大哥费振东到新加坡费孝通会面，这是兄弟三人时隔十余年后的再度相聚。1938年底，费青经[[越南|越南]][[西贡|西贡]]沿着[[滇越铁路|滇越铁路]]到中国[[云南|云南]]，在[[云南大学|云南大学]]法学院和[[西南联合大学|西南联合大学]]（[[北京大学|北京大学]]）法学院任教（费孝通已于1938年10月到云南大学社会学系任教授）。在西南联合大学（北京大学）法学院任教期间，费青历任讲师、副教授、教授，讲授过[[民法|民法]]、[[法理学|法理学]]、[[国际私法|国际私法]]、诉讼实习、[[法医学|法医学]]等课程（其中法理学1938至1939年度始由费青开设，费青1940年回上海养病后由[[燕树棠|燕树棠]]接任）。因为同时在西南联合大学及云南大学任教，费青每周授课达二十余小时。因教学劳累，且昆明海拔过高而气压低，费青的喘病加重。1940年夏，[[日军|日军]]飞机开始轰炸昆明，昆明没有防空设备，空袭警报时人们只能跑到郊外躲避，俗称“跑警报”。当时费青因喘病已无力“跑警报”，同住的费孝通夫妇也只能陪着费青不“跑警报”，这就更加危险。学校校医劝费青快离开昆明。1940年秋，费青携妻子叶筠，经越南回到上海<ref name=ceji/>。

1940年秋，费青回到上海任东吴大学法学院法律系主任。曾任《东吴法学院年刊》（第3卷，1944年）总编辑的[[潘汉典|潘汉典]]在其东吴大学法学院“1944年纪级史”（现在通称1944届）中记载：“开学后匝月，费青先生远道来沪，归母校长法律系。吾级首受陶熔，法学阶梯即由此而升。”1941年12月，[[太平洋战争|太平洋战争]]爆发。1942年1月8日，东吴大学校董会一致通过“停办学校案”。教务长[[盛振为|盛振为]]率部分师生到内地大后方。留在上海的大部分师生共200余人无所依靠。1942年2月，费青、[[张中楹|张中楹]]、[[刘哲民|刘哲民]]、[[艾国藩|艾国藩]]、[[卢于舫|卢于舫]]、[[姚启胤|姚启胤]]等教师商定“费青为教务长，鄂森为秘书长，[[吴芷芳|吴芷芳]]为副教务长，[[安绍云|安绍云]]为会计主任，刘哲民为会计系主任”。费青与张中楹、[[王遂征|王遂征]]等教师以“董法记”（东吴大学法学院的谐音）的名义，暂借[[上海法租界|上海法租界]]南昌路中华职业教育社的地址恢复上课。此后学校在10个月内四易地址，稍稳定后，学校由“董法记”改成“中国比较法学院”（为原东吴大学法学院英文名称“The Comparative Law School of China”的直译，以前一直译成“东吴法律专科”）<ref name=ceji/>。

1943年9月，费青独自离开上海，经[[蚌埠|蚌埠]]、[[界首|界首]]、[[洛阳|洛阳]]、[[宝鸡|宝鸡]]等地，到达[[重庆|重庆]]，出任[[复旦大学|复旦大学]]法律系教授和私立朝阳学院（重庆分院）教授，讲授民法、法理学、国际私法等课程，与复旦大学法学院教授、系主任[[张志让|张志让]]（曾是费青的老师）以及私立朝阳学院（重庆分院）主任[[闵刚侯|闵刚侯]]（是费青在东吴法学院1932届的校友）来往很多<ref name=ceji/>。

1945年8月[[抗日战争|抗日战争]]胜利，费青应邀回到西南联合大学（北京大学）法律系任教授，讲授国际私法、民法物权、民法债编等课程（且同时在云南大学法律系兼课）。从抗日战争胜利到1950年，费青除开设国际私法课程外，还开设有社会发展史、[[马列主义|马列主义]]法律理论等课程<ref name=ceji/>。

在[[一二·一运动|“一二·一”运动]]中，费青与[[周炳琳|周炳琳]]、[[钱端升|钱端升]]、燕树棠、[[赵凤喈|赵凤喈]]以及法律系助教被西南联合大学教授会推举组成法律委员会。费青与[[蔡枢衡|蔡枢衡]]合作撰写了两份态度强硬的告诉状，获得西南联合大学教授会通过。费青还在1945年12月13日出版的《时代评论》第7期上发表了署名文章《惨案的法律解决》。1946年西南联合大学解散，费青随北京大学复员北平。边教书，边以律师身份支持学生反饥饿、反内战运动。在1946年“[[沈崇案|沈崇事件]]”中，费青发挥过重要作用。费青还曾任《中建半月刊》（北平版）和《新建设》主编<ref name=ceji/><ref name=hexun/>。

1949年[[北平和平解放|北平和平解放]]后，费青任北京大学教授、法律系主任，北京政法学院（[[中国政法大学|中国政法大学]]的前身）教授、副教务长，是[[中华人民共和国|中华人民共和国]]当时仅有的三位法学二级教授之一。并曾任[[中央人民政府最高人民法院|中央人民政府最高人民法院]]委员、中国政法学会理事和研究部副主任、[[中央人民政府法制委员会|中央人民政府法制委员会]]专门委员及外事法规委员会主任委员、[[全国政协委员|全国政协委员]]等职。曾协助[[董必武|董必武]]起草制定1954年《[[五四宪法|中华人民共和国宪法]]》<ref name=ceji/><ref name=hexun/>。

1957年，费青在[[江苏|江苏]][[无锡|无锡]]病逝，享年51岁<ref name=ceji/><ref name=hexun/>。
