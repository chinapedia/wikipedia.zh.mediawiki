'''龍泉溪堰塞湖'''為一座位於[[台灣|台灣]][[台東縣|台東縣]][[池上鄉|池上鄉]]龍泉地區的一座天然[[湖泊|湖泊]]，該湖泊屬於[[堰塞湖|堰塞湖]]類型，現湖水已流放結束，回歸溪流狀態。<ref name="治山防災電子報 堰塞湖處理與治理"/><ref name="經濟部中央地質調查所 龍泉溪堰塞湖調查報告">{{cite web |url=http://www.ly.gov.tw/03_leg/0301_main/leg_news/newsView.action?id=12072&lgno=00142&stage=6&atcid=12072 |title= 經濟部中央地質調查所 龍泉溪堰塞湖調查報告 | accessdate=2015-02-17}}</ref><ref>{{cite web |url=http://www.moeacgs.gov.tw/info/view.jsp?info=199 |title=立法院全球資訊網 龍泉溪堰塞湖 度過一關 | accessdate=2015-02-17}}</ref>

== 歷史 ==
龍泉溪堰塞湖最初形成原因於2006年（民國95年）7月15日下午4點時，由[[中華民國|中華民國]][[行政院|行政院]][[農業委員會|農業委員會]][[林務局|林務局]][[關山鎮_(臺灣)|關山]]事業區第3林班所管轄鄰近龍泉溪的山坡地開始發生小規模土石崩落之情況。<ref name="治山防災電子報 堰塞湖處理與治理">{{cite web |url=http://engpaper.swcb.gov.tw/ENewsletter/004/EN00404.aspx 治山防災電子報 |title=治山防災電子報 堰塞湖處理與治理 |accessdate=2015-02-17 |deadurl=yes |archiveurl=https://web.archive.org/web/20150217173253/http://engpaper.swcb.gov.tw/ENewsletter/004/EN00404.aspx |archivedate=2015-02-17 }}</ref>
隔天，7月16日清晨6時05分於台東[[成功鎮|成功]][[地震站|地震站]]東北方6.2公里處測得[[芮氏規模|芮氏規模]]4.6[[地震|地震]]，因地震影響，[[龍泉溪_(台東)|龍泉溪]]崩塌處開始陸續發生多次大規模的土石崩塌，而導致河道遭土石淹沒，河水無處可去的狀況下，形成堰塞湖。<ref name="治山防災電子報 堰塞湖處理與治理"/>

在龍泉溪堰塞湖形成後林務局臺東林區管理處，根據堰塞湖應變處置標準作業程序，依短、中、長期之治理工程依序完成規劃及施工改善。<ref name="治山防災電子報 堰塞湖處理與治理"/>
有關龍泉溪堰塞湖之處置工程依區域大致可以分為堰塞湖崩塌區護坡工程、湖區溢流水道工程、河道堆石潛壩工程、梳子防砂[[壩|壩]]工程、支流過水路面[[涵洞|涵洞]]工程、無名[[橋|橋]]拓寬工程及[[水土保持局|水土保持局]]之龍泉溪河道整流工程等。<ref name="治山防災電子報 堰塞湖處理與治理"/>

龍泉溪堰塞湖自2006年（民國95年）形成開始，從緊急應變處置到中長期階段的工程整治，完整的保留堰塞湖處置的相關過程。在施工期間更設置一座土砂觀測系統，建置[[雨量|雨量]]、水位及影像等監測項目設備，相關監測資料記錄完整，可做為未來臺灣地區其他堰塞湖整治之研究案例，並做為相關堰塞湖處置之示範。 <ref name="治山防災電子報 堰塞湖處理與治理"/>

目前龍泉溪堰塞湖本身已經消失，其遺跡已逐漸演變為一[[濕地|濕地]][[生態|生態]]，未來可再透過生態工法的規劃，形成一濕地生態保留區。透過堰塞湖防災教育示範專區的成立，可以結合土砂觀測系統及堰塞湖處置工程的解說與圖示，了解堰塞湖應變處置的案例說明。<ref name="治山防災電子報 堰塞湖處理與治理"/>

== 地理 ==
龍泉溪堰塞湖發生後，[[經濟部|經濟部]][[中央地質調查所|中央地質調查所]]隨即針對該案件展開[[探勘|探勘]]，依據探勘資料，造成堰塞湖的山坡崩塌面積約12.9公頃，其中崩塌區約為7.5公頃，崩塌平均厚度約3公尺，崩塌土方量約20餘萬立方公尺，堆積區約5.4公頃，崩落之土石在河道上堆積成高約50公尺之天然[[土石壩|土石壩]]，並於土石壩上游形成堰塞湖。<ref name="經濟部中央地質調查所 龍泉溪堰塞湖調查報告"/>

由於崩落土石阻塞河道，使得崩塌地上游形成堰塞湖，造成溪水斷流，堰塞湖上游集水面積約1,234公頃，初步估計目前堰塞湖水深約在20至30公尺間，湖面長度約500公尺，目前蓄水量在100萬立方公尺左右。堰塞湖於7月25日上午因[[中度颱風|中度颱風]]凱米帶來豐沛雨量，已於壩體東南側溢流，根據當地民眾指出目前溢流之水量大致與未形成堰塞湖時之水量相當，顯示堰塞湖體已不再擴大。<ref name="經濟部中央地質調查所 龍泉溪堰塞湖調查報告"/>

崩塌區地質屬[[大南澳片岩|大南澳片岩]]之[[玉里帶|玉里帶]]（[[PM5|PM5]]），岩性主要為黑色片岩，[[葉理|葉理]]面發育良好，且此地區岩層節理發達，使得岩體較為破碎，加上地震與[[颱風|颱風]][[豪雨|豪雨]]影響，因而造成大規模崩塌，依據[[Varnes山崩分類|Varnes山崩分類]]屬[[岩石墜落|岩石墜落]]([[Rock_Fall|Rock Fall]])形式崩塌。<ref name="經濟部中央地質調查所 龍泉溪堰塞湖調查報告"/>

== 參數 ==
* 堰塞湖形成時間：2006年（民國95年）7月15日
* 溪流名稱：[[秀姑巒溪|秀姑巒溪]][[流域|流域]][[支流|支流]][[龍泉溪|龍泉溪]]
* [[集水區|集水區]]面積：12[[平方公里|平方公里]]
* 所在[[行政區|行政區]]：台東縣[[海端鄉|海端鄉]]
* 崩塌地：崩塌面積約16公頃，平均崩塌深度約3公尺。
* 天然壩體：壩頂長85公尺、底長320公尺、頂寬100公尺、底寬30公尺、壩高40公尺。
* 堰塞湖：回水長度700公尺、溢流口寬15公尺、最大水深40公尺，蓄水容量約100萬立方公尺。
* 崩塌地地質：屬於[[變質岩|變質岩]]區，岩石性質為變質岩，主要由[[矽質片岩|矽質片岩]]組成，偶夾薄層的黑色[[片岩|片岩]]。主要弱面變變質岩的片理，其方向大致與坡面相反；岩體表面的[[風化|風化]]層相當薄，而崩塌深度已到達岩盤。
