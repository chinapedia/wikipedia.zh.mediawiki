《'''龍戰在野'''》為武俠小說作家[[黃易_(香港)|黃易]]於2014年開始連載的作品，本書為《[[盛唐三部曲|盛唐三部曲]]》的第二部，《[[日月當空|日月當空]]》的續集。[[香港|香港]]由[[黃易出版社|黃易出版社]]出版，台灣則由[[蓋亞文化|蓋亞文化]]出版，並於2014年5月8日出版卷一，每月出版一卷，至2015年10月出版卷十八大結局。其續作為《[[天地明環|天地明環]]》。

小說故事背景為武周末年至[[神龍革命|神龍革命]]，但亦與描述晉代「[[淝水之戰|肥水之戰]]」後的《[[邊荒傳說|邊荒傳說]]》、隋末唐初的《[[大唐雙龍傳|大唐雙龍傳]]》，及以元、明為時空背景的《[[破碎虛空|破碎虛空]]》、《[[覆雨翻雲|覆雨翻雲]]》等作品角色連結<ref>{{cite news|author=李昀澔|title=寫作也趕下班 黃易談靈感奧秘|url=http://anntw.com/articles/20140511-RgrK|accessdate=2015年1月21日|newspaper=台灣醒報|date=2014年5月11日}}</ref>。

== 故事簡介 ==
因中宗回朝和武三思的譖言，龍鷹與朝廷關係已轉差，「大周國賓」的風光日子一去不返。龍鷹如何才可以尋回他的生活？<ref>龍戰在野卷一 序</ref>

== 出場角色 ==
本書出場角色眾多，包括中土、吐蕃、突厥、回紇、突騎施、龜茲、且末等。

== 港版台版之別 ==
* 台灣版在卷首印上了盛唐疆域圖，而香港版則沒有。
* 台版每卷首刷都會贈送水墨風畫卡，如卷六首刷加贈【符太 水墨風畫卡】<ref>[https://www.facebook.com/huangyi.gaea/photos/a.1427930360792157.1073741828.1427564250828768/1491214054463787/ 台版卷六封面]</ref>。
* 香港版封面由黃易挑選萬一鵬的作品，和《[[日月當空|日月當空]]》一樣，各卷順序放在一起可拼成一幅圖畫；但和《[[日月當空|日月當空]]》不同，香港版改由阮瑞萍負責設計。台版的封面同樣則由黃易挑選萬一鵬的作品<ref>[https://www.facebook.com/huangyi.gaea/photos/a.1429002994018227.1073741829.1427564250828768/1444125492505977 黃易作品集facebook群組]</ref>。
* 台版卷末都會附上下期預告，香港版則沒有。
* 台版每卷有目錄，香港版則沒有。
