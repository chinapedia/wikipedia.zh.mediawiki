{{Otheruses|龍泉溪|other=其它同名地名}}
'''龍泉溪'''為一條位於[[臺灣|臺灣]][[臺東縣|臺東縣]]的河川，龍泉溪另有「'''萬朝溪'''」的別稱。

該條河川發源於臺東縣[[海端鄉|海端鄉]][[崙天山|崙天山]]南側並流經池上鄉進入[[花蓮縣|花蓮縣]]成為臺灣[[東部地區|東部地區]]第一大[[河川|河川]][[秀姑巒溪|秀姑巒溪]]的[[源頭|源頭]]。溪水在流經池上鄉時，部分溪水會藉由圳道[[溝渠|溝渠]]流入池上鄉的[[大坡池|大坡池]]之中。<ref name=" 台東溯溪運動委員會 龍泉溪">{{cite web |url=http://www.keepon.com.tw/UploadFile/FileData/21786/8/22f21fdd-716e-487b-b602-26dea5525d44.pdf |title= 台東溯溪運動委員會 龍泉溪 | accessdate=2015-02-17}}</ref>

== 其他資訊 ==
===主要橋樑 ===
龍泉溪河道上，共有四座跨河[[橋樑|橋樑]]。

* '''文田橋'''（民國76年12月竣工）
* '''萬朝橋'''（民國85年1月竣工）
* '''大龍橋'''（民國81年11月竣工）
* '''無名橋'''
=== 瀑布 ===
{{Main|龍泉瀑布}}
'''龍泉[[瀑布|瀑布]]'''位於台東縣海端鄉[[廣原村|廣原村]]龍泉[[部落|部落]]，瀑布分上下兩瀑，落差逾200公尺。<ref name=" 台東溯溪運動委員會 龍泉溪"/>
=== 水利發展 ===
{{Main|萬朝圳}}
'''萬朝圳'''為一座位於臺灣臺東縣池上鄉的[[灌溉|灌溉]]水圳系統，該水圳為[[花東縱谷|花東縱谷]]臺東縣境內少數引水自秀姑巒溪水系的灌溉水圳，並且是池上鄉[[名產|名產]]池上[[米|米]]灌溉區的重要水源。<ref>{{cite web |url=http://www.ttia.gov.tw/page4_cs.php |title=台東農田水利會 池上工作站 |accessdate=2015-02-17 |deadurl=yes |archiveurl=https://web.archive.org/web/20151223043747/http://www.ttia.gov.tw/page4_cs.php |archivedate=2015-12-23 }}</ref>
== 災害事故 ==
=== 龍泉溪堰塞湖 ===
{{Main|龍泉溪堰塞湖}}
'''龍泉溪[[堰塞湖|堰塞湖]]'''最初形成原因於2006年（民國95年）7月15日下午4點時，由[[中華民國|中華民國]][[行政院|行政院]][[農業委員會|農業委員會]][[林務局|林務局]][[關山鎮_(臺灣)|關山]]事業區第3[[林班|林班]]所管轄鄰近龍泉溪的[[山坡|山坡]]地開始發生小規模土石崩落之情況。隔天，7月16日清晨6時05分於台東[[成功鎮|成功]][[地震|地震]]站東北方6.2公里處測得[[芮氏規模|芮氏規模]]4.6地震，因地震影響，龍泉溪崩塌處開始陸續發生多次大規模的土石崩塌，而導致河道遭土石淹沒，河水無處可去的狀況下，形成堰塞湖。<ref>{{cite web |url=http://www.ly.gov.tw/03_leg/0301_main/leg_news/newsView.action?id=12072&lgno=00142&stage=6&atcid=12072 |title= 經濟部中央地質調查所 龍泉溪堰塞湖調查報告 | accessdate=2015-02-17}}</ref>

目前湖水已流放完畢，恢復成河流狀態。

=== 溪水暴漲 ===
2014年8月12日因山區大雨，導致龍泉溪溪水暴漲溪水淹過水路面，傳12名工人與農民受困彼端，[[警|警]]方及[[消防|消防]]人員據報馳援，抵達現場後發現，過水路面寬10公尺、水深50公分，並已有6人自行涉水過溪，但為安全起見，立即起出拋繩槍架設繩索營救受困人員。
