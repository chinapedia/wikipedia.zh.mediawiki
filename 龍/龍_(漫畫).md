《'''龍'''》是[[日本漫畫家|日本漫畫家]][[村上紀香|村上紀香]]自1991年至2006年在[[Big_Comic_Original|Big Comic Original]]月刊連載的[[歷史漫畫|歷史漫畫]]。1995年，[[NHK|NHK]]將之改編為電視劇。日文原書名中的「Ron」（ロン）是以[[日語羅馬字|日語羅馬字]]標示「龍」的現代中文發音；[[日語|日語]]中一般將「龍」唸成「Ryuu」（りゅう），因此特別在書名中標注。

== 獲獎經歷 ==
* 1996年：第41屆[[小學館漫畫賞|小學館漫畫賞]]。
* 1998年：第2屆[[文化廳媒體藝術祭|文化廳媒體藝術祭]]漫畫部門優秀賞。

== 概要 ==
故事發生於1928年至1945年，前半以[[昭和|昭和]]初年的日本為背景，後半則描繪[[對日抗戰|對日抗戰]]時的中華民國。

主角'''押小路龍'''是一名中日混血兒，秉持著[[人道主義|人道主義]]的精神在亂世生活。在這十七年歲月裡，他遇見了韓裔移民、日本政要、中國黑道等三教九流各色人物，並一次又一次捲入爭奪戰爭資源的紛爭中－

從內藤高志、甘粕正彥到[[杜月笙|杜月笙]]，[[村上紀香|村上紀香]]靠著大量的史料，不時在故事中穿插真實歷史人物，重建了屬於龍的這個時代，並企圖站在持平的觀點看待[[第二次世界大戰|第二次世界大戰]]中的東亞。

值得一提的是，作者藉由女主角'''田鶴貞'''（田鶴てい）的電影演員身份，加入了不少描寫昭和初年[[日本電影|日本電影]]的片段（其中不少參照了真實的電影片段，如1937年的「人情紙風船」），有心的讀者可以發現[[小津安二郎|小津安二郎]]（於單行本14集出場的小田安次郎導演，即影射之）、[[溝口健二|溝口健二]]、[[山中貞雄|山中貞雄]]等名導也在故事中軋上一角。

== 主要人物 ==
; 押小路龍（李龍）
: 故事的主人公。出身京都名門，母親是中國人。青少年時代得意於劍道，被稱為'''關西之龍'''，並靠著[[劍道|劍道]]技巧，在1928年以十七歲之齡進入「大日本武徳會」的武道專門學校（通稱「武專」）。武專時期的龍，並沒有繼承家業的念頭，遇見主任教師內藤高治及同學們後，致力於'''武道'''的修練，遂有'''為道而生'''的決意。1936年因為捲入與自己出生相關的事件而流亡中國，這個事件是第二次世界大戰發展的關鍵。之後化名「李龍」，以中國人的身份行動。（有人認為龍的一部份特質來自日本知名左翼影評人{{tsl|ja|岩崎昶|岩崎昶}}。）
; 田鶴貞
: 故事的女主人公。是日本[[東北地方|東北地方]][[佃農|佃農]]的長女，十二歲時被酗酒的父親賣到大城市工作。在押小路家擔任女傭期間，受到龍父一磨支援，得以進入學校唸書。離開押小路家後，在片場煮飯為生，一次偶然的機會被發掘成為電影演員，因精湛的演技而走紅。1938年以勞軍為由來到[[上海|上海]]，與龍重逢並結婚。婚後，以[[滿州映畫協會|滿州映畫協會]](通稱「滿映」)女導演的身份繼續活躍於電影界。（一般認為田鶴貞的[[原型|原型]]是[[滿映|滿映]]的女導演[[坂根田鶴子|坂根田鶴子]]。）
; 小鈴
: 龍的青梅竹馬，是一位優秀的京都[[舞妓|舞妓]]。少年時代和龍互相愛慕，卻身不由己成為龍叔父卓磨的小妾。生下兒子冬馬後卓磨正式結婚。戰時以其交際手腕遊走於[[滿州國|滿州國]]軍政高層之間。
; 曹德豐
:中國留學生。表面上仰慕平息朝鮮人街暴動的龍，視龍為師父並與之共同行動，其實是某個組織派來監視龍的特務。因龍的友情而動搖原先的信念，最後下了個改變自己行動目標的重大決定。
; 金昇龍
:居住在[[朝鮮|朝鮮]]人街的少年。身為韓裔受盡歧視，因而憎恨日本人，唯一的例外是調停日韓人紛爭的龍。在成長為青年之後，仍和龍維持良好的友情，後來與龍一起遠渡中國，希望在[[抗日戰爭_(朝鮮半島)|朝鮮對日抗戰]]貢獻一己之力。（有人猜測是影射[[金日成|金日成]]。）
; 押小路一磨
: 龍的父親。年輕時代加入日本陸軍，派駐[[北京|北京]]時與中國籍的妻子相遇並生下了龍。不久與妻子死別，為專心育兒退伍回到日本。退伍後將家業交給弟弟卓磨處理，自己成為終日和[[祇園|祇園]]的[[藝妓|藝妓]]遊樂的放蕩父親；暗中不斷調查和妻子的死相關的[[清朝|清朝]]歷史。
; 押小路卓磨
: 一磨的親弟弟、龍的叔父。代替從軍的兄長管理家業，是押小路財閥的總帥。對和哥哥相似的龍感覺相當複雜，但察覺到龍有著群眾魅力，意圖培養他作為繼承人。
; [[内藤高治|内藤高治]]
: 真實歷史人物（1862～1929）。作為警視廳的劍道教師，被大日本武德會招聘為武專的劍道主任，是龍在學校中相當敬重的師長，對於現代劍道之中武道精神的喪失感到憂心。他對武士道的看法成為龍人生的指針。
;朱鳳花
: 優秀的青年[[京劇|京劇]]演員，掌握了龍出生的秘密。是擁有男女兩性性器官的[[陰陽人|陰陽人]]，並因此在學徒時代受到同門欺凌。與龍在北京相遇，並受邀到日本演出，引發一連串與清朝宮廷秘辛有關的陰謀事件。
; 李文龍
: 救了流亡中國的龍一命的少年。出身書香世家，與[[魯迅|魯迅]]是世交，他的名字就是魯迅取的。崇拜魯迅的他，與龍結為義兄弟並與之來到上海，在歷史的波瀾中，歷經了友情、戀愛、學業各方面的成長。
; [[甘粕正彦|甘粕正彦]]
:真實歷史人物（1891～1945）。暗中左右滿洲國政經的權力者，擔任[[滿州映畫協會|滿州映畫協會]]理事長一職。對和龍出生相關的事件抱持高度興趣，並嘗試與龍接觸，是和龍、貞夫婦密切相關的人物。

除上列者，本作尚有許多富有魅力的登場人物，其中包含不少中日軍政界、電影界、[[秘密結社|秘密結社]]中的真實人物－這些人物不一定完全以史料中遺存的面貌出現，更多的是作者加上自身對歷史的理解、順應故事行進路線詮釋之。
