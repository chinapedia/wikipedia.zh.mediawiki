'''龍笛'''是一种可能成型於唐代的传统横吹[[木管乐器|木管乐器]]，今日僅見於日本。由唐[[竹|竹]]制成，常用於演奏[[日本雅樂|日本雅樂]]。龍笛通过[[遣唐使|遣唐使]]传入日本，可能是由[[中亞|中亞]]的[[橫吹|橫吹]]演变而来，并與[[能管|能管]]、[[篠笛|篠笛]]一起被认为是其他[[和樂器|和樂器]]中所有横笛的原型和祖先。

== 特点 ==
龙笛在雅乐乐器中是音域很宽广的一种乐器，音域为E5～D7，共2个[[八度|八度]]，低音到高音的转换自由顺畅，犹如“翔起龙吟”（{{lang|ja|舞い立ち昇る龍の鳴き声}}），因此得名。龍笛的笛音代表翔起之龍，[[笙|笙]]音代表天光，而[[篳篥|篳篥]]代表地人。龍笛是日本雅樂使用的“三管”以及3種橫笛之一，多用於演奏中國風格的樂曲。

7孔，长40厘米，内径1.3厘米。演奏时，手指并不完全覆盖[[指孔|指孔]]，而是用盖半孔的演奏技法，轻抬手指可改变音调。

== 演奏形式 ==
合奏中，主旋律一般由[[篳篥|篳篥]]演奏，利用龍笛广阔的音域，主旋律能与其他乐声很好的配合。一般来说，作品的开始部分由龍笛[[独奏|独奏]]，独奏一般由樂曲的[[声部长|声部长]]（[[音頭|音頭]]）担任。

== 文化 ==
[[File:Ryuteki_player_by_Ken@Okinawa_in_Kamakura,_Kanagawa.jpg|200px]][[鎌倉市|鎌倉市]]）]]
日本古代的王公貴族和武将有很多都喜愛龍笛高亢入雲的声調，[[堀河天皇|堀河天皇]]、[[源義経|源義経]]和[[源博雅|源博雅]]的很多逸事都與龍笛的奇異傳説有關。和七絃琴一様，長期被認為是皇家秘器。平安中期女作家[[清少納言|清少納言]]的散文集《[[枕草子|枕草子]]》中也寫到「在所有樂器中，笛（龍笛）是最高尚的一種」（原意），並且列擧了皇室傳来的龍笛的名品。

20世纪末开始，龙笛开始在现代音乐作品中被使用，如[[琴古流|琴古流]]尺八奏者[[笹本武志|笹本武志]]就是一位专业用龙笛演奏新音乐的演奏者。

德国作曲家[[汉斯·维尔纳·亨策|汉斯·维尔纳·亨策]]在其作品《[[逃奴|逃奴]]》（''El Cimarrón''）用到了龙笛。
