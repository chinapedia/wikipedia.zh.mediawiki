'''米歇尔·劳德鲁普'''（{{lang-da|'''Michael Laudrup'''}}，{{bd|1964年|6月15日|catidx=Laudrup, Michael}}），是一名[[丹麦|丹麦]]前職業足球員，退役後擔任主教练，球員時期司職[[中场|中场]]，曾效力[[布隆德比足球俱乐部|布隆德比]]、[[尤文图斯|尤文图斯]]、[[巴塞罗那足球俱乐部|巴塞罗那]]、[[皇家马德里|皇家马德里]]等著名球队。教练时期曾执教[[丹超|丹超]]球队[[布隆德比足球俱乐部|布隆德比]]，[[西甲|西甲]]球队[[赫塔费足球俱乐部|赫塔费]]、[[皇家馬略卡|皇家馬略卡]]，[[俄超|俄超]]球队[[莫斯科斯巴达克足球俱乐部|莫斯科斯巴達]]和[[英超|英超]]球队[[斯旺西城足球俱乐部|史雲斯]]。

== 球员生涯 ==
球员时代的劳德鲁普曾经效力於[[尤文图斯|尤文图斯]]、[[拉齐奥足球俱乐部|拉齐奥]]、[[巴塞罗那足球俱乐部|巴塞罗那]]、[[皇家马德里|皇家马德里]]、[[阿贾克斯|阿贾克斯]]等欧洲顶级球队，并获得过7个联赛冠军和1个[[歐洲冠軍球會盃|歐洲冠軍球會盃]]冠军。凭借一个“4+1”的西甲五连冠的彪炳成绩，他还被评选为[[西班牙足球甲级联赛|西班牙]]25年（1974-1999年）最佳外援。他也是2003年丹麦足协提交[[欧洲足联|欧洲足联]]代表丹麦的50周年庆“黄金球员”。

== 技術風格 ==
米歇尔·劳德鲁普是是一位有创造力的球员，他是如此的优雅以至于人们戏称他是穿着燕尾服踢球的运动员。

1989年，这位丹麦奇才从尤文图斯转会加盟巴塞罗那，在都灵他没有给球迷留下深刻的印象。来到诺坎普之后，米歇尔卻轻松而淋漓尽致地展现他高超的技艺，也成就了這一位巴塞罗那梦之队的艺术家。

劳德鲁普是一对一的大师，也拥有惊人的视野，许多的妙传都是在不看传球目标的情况下完成的。在那个年代，劳德鲁普被认为是世界上最好的传球手。巴塞罗那球迷同样意识到这一点，多年来，有一条著名的彰显劳德鲁普神奇表演的横幅挂在诺坎普球场的看台上，在巴塞罗那效力期间，他出场288次，射入93个入球，帮助球队赢得许多冠军头衔。

在1994年的夏天，劳德鲁普和皇家马德里签约。虽然这个举动让人感到惊讶，但劳德鲁普魔法般的表演永远不会被巴塞罗那球迷忘记。

== 教练生涯 ==
退役后的劳德鲁普出任过[[丹麦国家足球队|丹麦国家队]]的助理教练，其后在国内任教[[布隆德比足球俱乐部|布隆德比]]，获得了两次丹麦年度最佳教练奖。

== 个人 ==
劳德鲁普的弟弟[[布莱恩·劳德鲁普|布莱恩·劳德鲁普]]也是著名足球运动员。
