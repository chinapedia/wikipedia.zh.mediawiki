'''米子市'''（{{jpn|j=米子市|hg=よなごし|rm=Yonago shi|f=y}}）是位於日本[[鳥取縣|鳥取縣]]西部的[[市|市]]，與[[島根縣|島根縣]][[安來市|安來市]]接壤，為[[山陰地方|山陰地方]]的主要都市之一。米子市是山陰地方人口最多的都市圈{{link-ja|米子都市圈|米子都市圏}}的中心都市，並和鄰近的{{link-ja|松江都市圈|松江都市圏|松江}}、{{link-ja|出雲都市圈|出雲都市圏|出雲}}、安來等都市圈形成了{{link-ja|中海、宍道湖经济圈|中海・宍道湖経済圏}}。自[[江戶時代|江戶時代]]初期以來，米子市就作為商業都市得到發展，有「[[山陰地方|山陰]]的[[大阪市|大阪]]」之稱<ref name="Hyakunen">{{cite book |author={{lang|ja|平岡昭利}}|script-title=ja:《中国・四国 (地図で読む百年)》 |year=1999 |publisher={{lang|ja|古今書院}} |location=東京都 |isbn=4-7722-5030-1|language=ja}}</ref>{{rp|87}}。由於米子地處山陰兩縣的中央，因此許多企業和政府將其統籌山陰兩縣的分公司設在米子市或同樣地處兩縣中間的[[松江市|松江市]]。因此米子市雖非縣廳所在地，但卻擁有[[鳥取大學|鳥取大學]]醫學院及其{{link-ja|鳥取大學醫學部附屬醫院|鳥取大学医学部附属病院|附屬醫院}}、[[山陰放送|山陰放送]]等重要機構，是山陰地方的據點都市之一。在交通方面，米子市是JR[[山陰本線|山陰本線]]和[[境線|境線]]、[[伯備線|伯備線]]的分歧點，{{link-ja|西日本旅客鐵道米子支社|西日本旅客鉄道米子支社|JR米子支社}}位於此地。公路方面有[[山陰自動車道|山陰自動車道]]和[[米子自動車道|米子自動車道]]經過，使得自米子前往[[岡山市|岡山]]、[[鳥取市|鳥取]]、[[境港市|境港]]、[[松江市|松江]]等地都十分便利，加上自[[米子機場|米子機場]]和其鄰近的境港有開往[[韓國|韓國]]和[[俄羅斯|俄羅斯]]的航路，使得米子成為山陰地方首屈一指的交通樞紐。

==歷史==
[[File:Yonago_castle_ishigaki.JPG|thumb]]
米子市的歷史可以追溯至[[彌生時代|彌生時代]]，市內的目久美遺跡和{{link-ja|妻木晩田遺跡|妻木晩田遺跡}}均證明米子地區在彌生時代已有發展至一定程度的文明存在<ref>{{cite web |url=http://www.pref.tottori.lg.jp/42521.htm|script-title=ja:妻木晩田遺跡の概要 |accessdate=2017-04-04|publisher={{lang|ja|鳥取県}} |language=ja}}</ref>。然而自[[古墳時代|古墳時代]]開始至[[戰國時代_(日本)|戰國時代]]，文獻少有關於米子一帶的記載。戰國時代末期，米子所在的[[伯耆國|伯耆國]]西部和[[出雲國|出雲國]]東部均由[[吉川廣家|吉川廣家]]統治。吉川在湊山一帶實行{{link-ja|町割|町割}}，此為米子城下町歷史的開始<ref name="Hyakunen"></ref>{{rp|90}}。

1600年（[[慶長|慶長]]5年），吉川被轉封至[[周防國|周防國]][[岩國市|岩國]]，此地則新設立{{tsl|ja|米子藩}}，由{{link-ja|中村一忠|中村一忠}}擔任藩主；但由於中村只有11歲，因此實際是由其家老{{link-ja|横田村詮|横田村詮}}執政。橫田統治期間修建了[[米子城|米子城]]，並全面擴張了米子城下町的規模，江戶時代米子的雛形就奠定於這一時期<ref name="History">{{cite web |url=http://houki.yonago-kodaisi.com/his-yonagoA.htm|script-title=ja:米子の歴史　（古代－現代まで） |accessdate=2017-04-04|publisher={{lang|ja|ファンタジ－米子・山陰の古代史}} |language=ja}}</ref>。1609年（慶長14年），中村一忠突然過世導致中村家斷絕，隨後[[加藤貞泰|加藤貞泰]]曾短暫成為米子藩的領主；直到1617年（元和3年），加藤被轉封至伊予[[大洲藩|大洲藩]]，米子藩被取消，改被納為[[鳥取藩|鳥取藩]]的領地，由池田氏治理<ref name="History"></ref>。1632年（寬永9年），池田氏家老{{link-ja|荒尾成利|荒尾成利}}被委任治理此地；此後至[[明治維新|明治維新]]為止，米子都由荒尾氏統治<ref name="History"></ref>。江戶時代米子憑藉其的地理位置而發展為山陰的商業中心，也確立的今日米子的以服務業為主的特徵。

明治維新之後，米子在1871年的[[廢藩置縣|廢藩置縣]]中劃入鳥取縣轄區。1876年，鳥取縣和島根縣合併，米子也隨之成為新島根縣的一部分<ref>{{cite web |url=http://www.pref.tottori.lg.jp/secure/991373/rekisisyousassi.pdf|script-title=ja:鳥取県ができるまで |accessdate=2017-04-04|publisher={{lang|ja|鳥取県}} |language=ja}}</ref>。在此期間米子城的部分建築於1878年被拆除<ref>{{cite web |url=http://www.city.yonago.lg.jp/4438.htm|script-title=ja:米子城の歴史 |accessdate=2017-04-04|publisher={{lang|ja|米子市}} |language=ja}}</ref>；但鳥取縣和島根縣的合併引發了鳥取縣人的巨大反彈，紛紛要求恢復鳥取縣並在1881年成功復縣。然而地處鳥取、島根兩縣交界出的米子和島根縣關係密切，因此當地人士多希望能將米子地區劃入島根縣，或再次合併鳥取、島根兩縣並將島根縣西部的[[石見國|石見地方]]劃入廣島縣，將新縣廳設在米子（{{link-ja|會見縣設置運動|会見県設置運動}}）。然而由於鳥取縣東部的[[鳥取市|鳥取]]和[[倉吉市|倉吉]]地區對此反應冷淡，此一提議遂不了了之<ref>{{cite web|url=http://hist-geo.jp/pdf/archive/kiyou/017_141.pdf|script-title=ja:島根県の行政区画制定の過程と県域をめぐる係争について|accessdate=2017-04-04|publisher={{lang|ja|歴史地理学会}}|language=ja|title=存档副本|archiveurl=https://web.archive.org/web/20170404132133/http://hist-geo.jp/pdf/archive/kiyou/017_141.pdf|archivedate=2017-04-04|deadurl=yes}}</ref>。1889年，米子正式實行町制，米子町誕生<ref>{{cite web |url=http://www.city.yonago.lg.jp/16978.htm|script-title=ja:米子歴史年表 |accessdate=2017-04-04|publisher={{lang|ja|米子市}} |language=ja}}</ref>。

1902年，米子開通了山陰地方首條鐵路<ref name="History"></ref>，隨後在1912年還開通了山陰線，1928年開通了伯備線，標誌米子成為山陰地方鐵路樞紐<ref name="Hyakunen"></ref>{{rp|92}}。1927年，米子町施行[[市制_(日本制度)|市制]]，米子市誕生<ref>{{cite web | language = ja | publisher = 米子市水道局 | title = 米子市水道の沿革 | url = http://www.city.yonago.lg.jp/8629.htm | author =  | date = 2011-03-09 | accessdate = 2011-07-19 }}</ref>。戰後米子在1963年建設了山陰地方首個工業團地，迎來了快速發展的時代<ref name="Hyakunen"></ref>{{rp|92}}。但與此同時，隨著汽車普及導致市區範圍擴大和人口向郊外遷移，市中心的空洞化問題也日益突出。2005年，舊米子市和{{link-ja|淀江町|淀江町}}合併為現在的米子市<ref>{{cite web |url=http://www.city.yonago.lg.jp/3686.htm|script-title=ja:市町村合併 |accessdate=2017-04-04|publisher={{lang|ja|米子市}} |language=ja}}</ref>。

== 地理 ==
[[File:Tottori_University_Faculty_of_medicine.jpg|thumb]]醫學部校區可遠望[[大山_(鳥取縣)|大山]]]]
米子市地處{{link-ja|日野川|日野川}}入海口附近，市區大部分位於日野川沖積形成的{{link-ja|米子平原|米子平野}}。市域西北部則是日本最長的沙嘴之一的{{link-ja|弓濱半島|弓ヶ浜半島}}。米子市兩面臨水，西側臨[[中海|中海]]，東側則是[[日本海|日本海]]<ref>{{cite web |url=http://www.yonago-navi.jp/welcome/about/|script-title=ja:よなごってどんなとこ？ |accessdate=2017-04-04|publisher={{lang|ja|米子市観光協会}} |language=ja}}</ref>。米子市域南部的是[[大山_(鳥取縣)|大山]]山麓的丘陵地帶，在米子市區亦可一望有「伯耆富士」美稱的大山的山容。米子西北的弓濱半島沒有自然河川，農業用水缺乏穩定水源。江戶時期鳥取藩主在現在米子市和[[境港市|境港市]]交界一帶引日野川水修建名為「米川」的人工用水路，全長約20公里，使得當地的農業取得飛躍發展，成為鳥取縣重要的農業地帶<ref>{{cite web |url=http://www.pref.tottori.lg.jp/253605.htm|script-title=ja:米川のあらまし |accessdate=2017-04-04|publisher={{lang|ja|鳥取市}} |language=ja}}</ref>。
===氣候===
米子市屬於[[日本海側氣候|日本海側氣候]]，相比自春季至秋季的氣候較為舒適，但冬季多陰天及雨雪天氣。年平均氣溫約為15.0℃，較為溫暖。米子市夏季雖然天氣炎熱，但夜晚溫度較低。初夏有時會發生[[焚風|焚風]]現象導致氣溫極度升高。米子在冬季是日本海側氣候地區中最為溫暖的地區，降雪量也只有鳥取縣東部[[鳥取市|鳥取市]]的約一半，過去從未出現過超過1米的積雪。米子市的最深積雪記錄是在2011年1月1日錄得的89cm<ref>{{Cite web |url=http://www.data.jma.go.jp/obd/stats/etrn/view/rank_s.php?prec_no=69&prec_ch=%92%B9%8E%E6%8C%A7&block_no=47744&block_ch=%95%C4%8Eq&year=&month=&day=&elm=rank&view= |script-title=ja:観測史上1～10位の値（年間を通じての値） 米子 (鳥取県) |accessdate=2013-08-05 |publisher={{lang|ja|気象庁}} }}</ref>。

==人口==
2017年2月28日時，米子市有人口149,242人<ref name="Population">{{cite web |url=http://www.city.yonago.lg.jp/9498.htm |script-title=ja:人口|accessdate=2017-04-04|publisher={{lang|ja|米子市}} |language=ja}}</ref>。米子市的人口在戰後經歷了大幅增加。1955年時，米子市的人口有99,446人。1970年時增加到117,056人。1985年時增加到	140,615人。此後由於[[老齡化|老齡化]]和[[少子化|少子化]]的進展，米子市人口增長趨緩。2000年時，米子市人口增加到147,837人。就市內各地區來看，市中心的人口明顯減少，而郊外地區的人口則呈現增加趨勢，顯示市區郊外化的進行<ref name="Chishi">{{cite book |author={{lang|ja|森川洋·篠原重則·奥野隆史}}|script-title=ja:《日本の地誌9　中国・四国》 |year=2005 |publisher={{lang|ja|朝倉書店}} |location=東京都 |isbn=4-254-16769-5|language=ja}}</ref>{{rp|165}}。

== 政治 ==
米子市和鳥取縣西部的[[境港市|境港市]]、[[北榮町|北榮町]]、[[湯梨濱町|湯梨濱町]]、[[琴浦町|琴浦町]]、[[日吉津村|日吉津村]]、[[大山町|大山町]]、[[南部町_(鳥取縣)|南部町]]、[[伯耆町|伯耆町]]、[[日野町_(鳥取縣)|日野町]]、[[日南町|日南町]]、[[江府町|江府町]]劃為日本眾議院選舉的{{link-ja|鳥取縣第2區|鳥取県第2区}}。在[[自由民主黨_(日本)|自由民主黨]]佔據壓倒性優勢的[[中國地方|中國地方]]，該區是少數非自民黨參選人曾通過比例代表復活當選的選區，也是中國地方非自民黨參選人得票率相對較高的地區。自2005年以來，該區的當選議員是自民黨的{{link-ja|赤澤亮正|赤沢亮正}}<ref>{{cite web |url=http://www.pref.tottori.lg.jp/secure/411167/46rikkouhosyatokuhyosu.pdf |script-title=ja:衆議院総選挙立候補者等一覧（鳥取県 第１回～第４６回）|accessdate=2017-04-04|publisher={{lang|ja|鳥取県}} |language=ja}}</ref>。米子市的現任市長是伊木隆司，他在2017年首次當選舊米子市的市長<ref>{{cite web |url=https://www.nnn.co.jp/news/170417/20170417048.html |script-title=ja:米子市政いかに前進　伊木氏初当選、手腕問われる|accessdate=2017-04-18|publisher={{lang|ja|日本海新聞}} |language=ja}}</ref>。{{link-ja|米子市議會|米子市議会}}則共有26名議員<ref>{{cite web |url=http://www.city.yonago.lg.jp/2919.htm |script-title=ja:議員の紹介|accessdate=2017-04-04|publisher={{lang|ja|米子市議会}} |language=ja}}</ref>。{{link-ja|鳥取縣議會|鳥取県議会}}中米子市則有9名議員<ref>{{cite web |url=http://www.pref.tottori.lg.jp/dd.aspx?moduleid=155189&blogmode=category&blogcategory=3057|script-title=ja:議員名簿（五十音順）|accessdate=2017-04-04|publisher={{lang|ja|鳥取県議会}} |language=ja}}</ref>。

== 經濟 ==
[[File:AEON_Yonago_Ekimae.jpg|thumb]]
江戶時代挖掘米川用水路之後，米子的農業得到快速發展。在江戶時代[[棉花|棉花]]曾是米子的名產品。但在1895年日本撤除棉花進口關稅之後養蠶取代棉花成為米子農業的重要部分，直到大蕭條導致生[[絲|絲]]價格暴跌為止<ref name="Chishi"></ref>{{rp|154}}。和日本大多數地區類似，現在農業只佔米子市經濟很小的比重。2000年時米子市的農戶數和1960年相比減少了45%<ref name="Chishi"></ref>{{rp|154}}。米子市現在農家人口有9,459人，佔米子總人口的6.4%<ref name="Data">{{cite web |url=http://www.city.yonago.lg.jp/9499.htm|script-title=ja:産業 |accessdate=2017-04-04|publisher={{lang|ja|米子市}} |language=ja}}</ref>。米子市的主要農產品是蔬菜和水果類，其中又以[[蔥|蔥]]最為重要。2015年，米子市的蔥生產額達11億3885萬日元，是第二位的胡蘿蔔的七倍以上<ref name="Data"></ref>。水果則以[[梨|梨]]的生產量金額最多<ref name="Data"></ref>。米子市是山陰地方主要工業都市之一。自明治時代以來，米子就設立了米子製鋼所<ref>{{cite web |url=http://www.yonago.co.jp/company.html|script-title=ja:会社情報|accessdate=2017-04-04|publisher={{lang|ja|米子製鋼株式会社}} |language=ja}}</ref>，形成了以鋼鐵和食品加工為主的工業結構。1966年，米子市和鄰接的[[境港市|境港市]]一併指定為中海地區新產業都市，興建眾多基礎設施以吸引企業投資設廠<ref>{{cite web |url=https://www.cgr.mlit.go.jp/vision/honpen_pdf/024-025.pdf|script-title=ja:地域開発の状況(PDF 149KB |accessdate=2017-04-04|publisher={{lang|ja|中国地方整備局}} |language=ja}}</ref>，但招商效果並不理想。米子市的工業就業人口在1995年有19,874人，但在2010年已減少至13,892人<ref name="Data"></ref>。2014年米子市工業生產額為1366億5000萬日元，不到2004年3221億1000万日元的一半，顯示米子市工業狀況嚴峻。服務業是米子市的支柱產業。由於位處於交通樞紐，米子確立了作為山陰地方商業中心的地位。米子雖然人口只有約15萬，但擁有米子[[高島屋|高島屋]]<ref>{{cite web |url=https://www.takashimaya.co.jp/yonago/#|script-title=ja:米子タカシマヤ|accessdate=2017-04-04|publisher={{lang|ja|タカシマヤ}} |language=ja}}</ref>和米子新町{{link-ja|天滿屋|天満屋}}<ref>{{cite web |url=http://www.tenmaya.co.jp/yonago/accessmap.html|script-title=ja:アクセスマップ|accessdate=2017-04-04|publisher={{lang|ja|天満屋}} |language=ja}}</ref>兩家百貨店以及[[永旺|永旺]]米子站前店等大型商業設施。然而大型商業設施的發展同時也引發了市中心的空洞化等問題<ref>{{cite web |url=http://www.city.yonago.lg.jp/secure/23366/keikaku2711.pdf|script-title=ja:米子市中心市街地活性化基本計画（新計画）|accessdate=2017-04-04|publisher={{lang|ja|米子市}} |language=ja}}</ref>。總部設在米子市的金融機構有{{link-ja|米子信用金庫|米子信用金庫}}<ref>{{cite web |url=http://www.yonagoshinkin.co.jp/about/ysk0001/page90|script-title=ja:金庫概要・沿革|accessdate=2017-04-04|publisher={{lang|ja|米子信用金庫}} |language=ja}}</ref>，其服務對象也以當地的普通市民和中小企業為主。

==交通==
[[File:Yonago_Station01bs4592.jpg|thumb]] 
鳥取憑藉其便利的交通而成為山陰地方的交通樞紐。[[山陰本線|山陰本線]]、[[伯備線|伯備線]]和[[境線|境線]]三條[[西日本旅客鐵道|西日本旅客鐵道]]的路線途徑米子市。2015年度[[米子車站|米子車站]]日均乘客數為3714人<ref name="Traffic">{{cite web |url=http://www.city.yonago.lg.jp/9504.htm |script-title=ja:交通|accessdate=2017-04-04|publisher={{lang|ja|米子市}} |language=ja}}</ref>。在米子停車的特急列車包括開往[[岡山市|岡山市]]的[[八雲號列車|八雲號列車]]、開往[[益田市|益田市]]的[[松風號列車|松風號列車]]、開往[[山口市|山口市]]的[[隱岐號列車|隱岐號列車]]以及連結[[東京都|東京都]]和[[出雲市|出雲市]]的[[出雲號列車|出雲號列車]]。連接米子車站和[[鳥取車站|鳥取車站]]的[[鳥取Liner|鳥取Liner]]則是鳥取縣內鐵路交通的大動脈。途徑米子市的高速公路則有[[米子自動車道|米子自動車道]]和[[山陰自動車道|山陰自動車道]]。當地的主要長途客運公司有{{link-ja|日本交通 (鳥取縣)|日本交通 (鳥取県)|日本交通}}和{{link-ja|日之丸自動車|日ノ丸自動車}}。[[美保飛行場|米子機場]]雖然名稱取自於米子市，但大部分用地位於[[境港市|境港市]]。米子機場暱稱鬼太郎機場，名稱取自於當地漫畫家[[水木茂|水木茂]]作品《[[鬼太郎|鬼太郎]]》。2013年米子機場的乘客數約有64萬人<ref>{{cite web |url=http://www.ocab.mlit.go.jp/about/total/report/pdf/riyou_h25d.pdf |script-title=ja:管内空港の利用状況概況集計表(平成機場25年度速報値)　NO.1|accessdate=2017-04-04|publisher={{lang|ja|大阪航空局}} |language=ja}}</ref>。現在米子機場運行有至[[東京國際機場|東京國際機場]]和[[首爾|首爾]]及[[香港|香港]]的定期航班<ref>{{cite web |url=http://www.yonago-air.com/flight |script-title=ja:フライト情報|accessdate=2017-04-04|publisher={{lang|ja|米子鬼太郎空港}} |language=ja}}</ref>。

==文化教育==
以米子市為代表的鳥取縣西部（[[伯耆國|伯耆地方]]）和[[鳥取市|鳥取市]]為代表鳥取縣東部（[[因幡國|因幡地方]]）的方言迥然不同。米子屬於{{link-ja|雲伯方言|雲伯方言}}區的{{link-ja|西伯耆方言|西伯耆方言}}，比起同屬鳥取縣的鳥取市使用的{{link-ja|因州方言|因州弁}}，米子的方言更接近島根縣東部使用的{{link-ja|出雲方言|出雲方言}}。總部設在鳥取的媒體有[[山陰放送|山陰放送]]，其播出地區橫跨島根和鳥取兩縣<ref>{{cite web |url=http://www.bss.jp/kaisya/ |script-title=ja:会社概要|accessdate=2017-04-04|publisher={{lang|ja|山陰放送}} |language=ja}}</ref>。米子市每年主要的節慶包括米子櫻花節<ref>{{cite web |url=http://www.city.yonago.lg.jp/4073.htm |script-title=ja:米子桜まつり|accessdate=2017-04-04|publisher={{lang|ja|米子市}} |language=ja}}</ref>、{{link-ja|日吉神社神幸神事|日吉神社神幸神事}}<ref>{{cite web |url=http://www.yonago-navi.jp/season/2249/ |script-title=ja:日吉神社神幸神事「よいとまかせ」5月3日（祝）14時 日吉神社をスタート |accessdate=2017-04-04 |publisher={{lang|ja|米子観光協会}} |language=ja |deadurl=yes |archiveurl=https://web.archive.org/web/20170405170732/http://www.yonago-navi.jp/season/2249/ |archivedate=2017年4月5日 |df= }}</ref>、米子大祭（{{lang|ja|米子がいな祭}}）<ref>{{cite web |url=http://www.city.yonago.lg.jp/9884.htm |script-title=ja:祭・イベント|accessdate=2017-04-04|publisher={{lang|ja|米子市}} |language=ja}}</ref>等活動。米子市內並無綜合大學，市內唯一的高等教育機構是[[鳥取大學|鳥取大學]]的醫學院<ref>{{cite web |url=http://www.tottori-u.ac.jp/1795.htm |script-title=ja:交通アクセス・キャンパスマップ|accessdate=2017-04-04|publisher={{lang|ja|鳥取大学}} |language=ja}}</ref>。米子市內主要的文化設施則有利用舊市政府大樓改建的{{link-ja|山陰歷史館|山陰歴史館}}<ref>{{cite web |url=http://yonagobunka.sanin.jp/p/yonagobunka/rekishi/information/ |script-title=ja:館内案内|accessdate=2017-04-04|publisher={{lang|ja|山陰歴史館}} |language=ja}}</ref>和{{link-ja|米子市美術館|米子市美術館}}<ref>{{cite web |url=http://www.yonagobunka.net/p/yonagobunka/y-moa/1/ |script-title=ja:施設案内 |accessdate=2017-04-04|publisher={{lang|ja|米子市美術館}} |language=ja}}</ref>等設施。

== 觀光資源 ==
[[File:Kaike_onsen01n3200.jpg|thumb]]
米子市最大的觀光資源是[[皆生溫泉|皆生溫泉]]。皆生溫泉位於米子市西北弓濱的皆生海岸，由一名漁師在1900年偶然發現海中冒泡而見諸於世<ref>{{cite web |url=http://www.yonago-navi.jp/onsen/summary/|script-title=ja:皆生温泉概要|accessdate=2017-04-04|publisher={{lang|ja|米子市観光協会]}} |language=ja}}</ref>，現已成為山陰最大的溫泉度假地之一。2015年有約40萬人在皆生溫泉入浴，是鳥取縣入浴遊客數最多的溫泉地<ref>{{cite web |url=http://www.pref.tottori.lg.jp/secure/226510/H27gaiyou.pdf|script-title=ja:平成27年観光客入込動態調査結果の概要|accessdate=2017-04-04|publisher={{lang|ja|鳥取県]}} |language=ja}}</ref>，有「山陰的[[熱海市|熱海]]」之稱<ref>{{cite web |url=http://www.lit.osaka-cu.ac.jp/geo/pdf/sanin/sanin_06.pdf|script-title=ja:皆生温泉と玉造温泉の現状とこれから|accessdate=2017-04-04|publisher={{lang|ja|大阪市立大学]}} |language=ja}}</ref>。皆生溫泉屬於鹽化物泉，源泉數達19處，湧出泉量豐富且溫度較高<ref>{{cite web |url=http://www.kaike-onsen.com/new/JP/outline.html|script-title=ja:あらまし|accessdate=2017-04-04|publisher={{lang|ja|皆生温泉旅館組合]}} |language=ja}}</ref>。夏季的皆生海岸同時還是海水浴場，被選入「日本海水浴場88選」<ref>{{cite web |url=http://furusato.sanin.jp/p/area/yonago/50/|script-title=ja:皆生温泉（かいけ）・皆生海水浴場|accessdate=2017-04-04|publisher={{lang|ja|皆生温泉旅館組合]}} |language=ja}}</ref>。皆生溫泉還是日本[[三項全能|三項全能]]的發祥地<ref>{{cite web |url=http://hamadayori.com/hass-col/sports/triathlon.htm|script-title=ja:トライアスロン発祥の地 |accessdate=2017-04-04|publisher={{lang|ja|発祥の地コレクション]}} |language=ja}}</ref>，現在每年均在這裡舉辦三項全能全日本大賽<ref>{{cite web |url=http://www.kaike-triathlon.com/rekishi.htm|script-title=ja:皆生大会の始まり |accessdate=2017-04-04|publisher={{lang|ja|皆生トライアスロン]}} |language=ja}}</ref>。除此之外，米子市還有米子城、{{link-ja|後藤家住宅|後藤家住宅}}等旅遊景點<ref>{{cite web |url=http://www.yonago-navi.jp/yonago/shitamachi/sightseeing/gotou/|script-title=ja:内町後藤家 |accessdate=2017-04-04|publisher={{lang|ja|米子観光ガイド]}} |language=ja}}</ref>。米子亦是前往附近[[境港市|境港市]]等地觀光時重要的交通據點。
