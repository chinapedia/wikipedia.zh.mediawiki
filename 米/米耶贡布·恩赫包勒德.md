'''米耶贡布·恩赫包勒德'''（{{lang-mn|Миеэгомбын Энхболд}}，Miyeegombo Enkhbold，{{bd|1964年|7月|||E}}）生于[[蒙古国|蒙古国]][[乌兰巴托|乌兰巴托]]，[[蒙古国|蒙古国]]政治人物，曾任[[蒙古国|蒙古国]][[蒙古总理|总理]]、[[国家大呼拉尔|国家大呼拉尔]]主席。

==生平==
1964年7月生于乌兰巴托。1987年自[[蒙古国立大学|蒙古国立大学]]经济学专业毕业。历任首都[[乌兰巴托|乌兰巴托市]]青格尔泰区副区长、青格尔泰区“公民代表会议”主席。1990年，加入[[蒙古人民革命党|蒙古人民革命党]]。1997年，任蒙古人民革命党乌兰巴托市委员会主席。1999年，任乌兰巴托市长。2005年7月至2007年10月，担任[[蒙古人民革命党|蒙古人民革命党]]（后来更名[[蒙古人民党|蒙古人民党]]）主席。2005年8月，在[[国家大呼拉尔|国家大呼拉尔]]补选中当选国家大呼拉尔委员。2006年1月25日至2007年11月，任[[恩赫包勒德内阁|恩赫包勒德内阁]]的政府总理。后来曾任[[第一次巴亚尔内阁|第一次巴亚尔内阁]]及[[第二次巴亚尔内阁|第二次巴亚尔内阁]]、[[巴特包勒德内阁|巴特包勒德内阁]]及[[巴特包勒德改组内阁|巴特包勒德改组内阁]]的政府副总理。2012年，当选[[国家大呼拉尔|国家大呼拉尔]]委员，并且出任国家大呼拉尔副主席<ref name=wy/><ref>{{Cite web|url =http://news.xinhuanet.com/world/2006-01/25/content_4099460.htm|title =新闻人物:蒙古国新总理米耶贡布•恩赫包勒德|author =|date =2006-01-25|publisher =新华网|language =|accessdate =|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20060207022429/http://news.xinhuanet.com/world/2006-01/25/content_4099460.htm|archivedate =2006-02-07}}</ref>。

2013年10月27日，[[蒙古人民党|蒙古人民党]]第27次代表大会乌兰巴托[[中央文化宫_(乌兰巴托)|中央文化宫]]开幕，议程包括总结蒙古人民党第26次代表大会以来党的工作、2012年国家大呼拉尔选举失利原因分析、讨论到2021年蒙古人民党到发展目标、修改党章、选举新一届领导人等等。在10月29日的会议上，米耶贡布·恩赫包勒德当选蒙古人民党主席。这是他第二次担任该党主席<ref name=wy>{{Cite web |url =  http://news.163.com/13/1029/21/9CCQ1PNB00014JB6.html|title =米耶贡布·恩赫包勒德当选蒙古人民党主席  |author =  |date = 2013-10-29 |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref>。2016年，在[[2016年国家大呼拉尔选举|国家大呼拉尔选举]]中，他领导蒙古人民党取得大胜，夺下65席，创造了自2000年以来的最好成绩，他也当选为国家大呼拉尔主席<ref>{{cite news|last1=Tian|first1=Shaohui|title=Mongolia's new government seeks stronger relations with China|url=http://news.xinhuanet.com/english/2016-07/14/c_135513611.htm|accessdate=2016-07-16|agency=Xinhua|publisher=People's Republic of China|date=2016-07-14}}</ref><ref>{{Cite web |url = http://news.ifeng.com/a/20160705/49299290_0.shtml |title = 蒙古国议会选举米耶贡布·恩赫包勒德为新议长 |author =  |date =  2016-07-05|publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref>。在[[2017年蒙古国总统选举|2017年蒙古国总统选举]]中，他代表蒙古人民党成为总统候选人参选，但未能获胜<ref>{{Cite web |url =http://news.xinhuanet.com/world/2017-05/06/c_1120929808.htm  |title = 蒙古国三大政党全部推选出总统候选人 |author =  |date =2017-05-06  |publisher =  新华网|language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://news.xinhuanet.com/world/2017-05/27/c_1121051181.htm |title = 蒙古国正式确认三名总统候选人资格 |author =  |date =2017-05-27  |publisher = 新华网 |language =  |accessdate =  |quote =  }}</ref>。
