'''米奇·黑斯曼'''（Mike Hessman，{{bd|1978年|3月5日||}}）是[[美國|美國]]職業棒球選手，目前效力於底特律老虎，擔任一壘手。2014年6月30日，米奇·黑斯曼打破Ollie Carnegie於1945年保持的[[國際聯盟|國際聯盟]]全壘打紀錄<ref name=LeagueRecord>{{cite web|url=http://www.cbssports.com/mlb/eye-on-baseball/24602356/mike-hessman-sets-international-league-home-run-record|title=Mike Hessman sets International League home run record|last=Snyder|first=Matt|work=[[CBS_Sports|CBS Sports]]|date=July 1, 2014|accessdate=July 1, 2014}}</ref>。

2015年，米奇·黑斯曼打破了1937年由[[巴茲·阿雷特|巴茲·阿雷特]]（Buzz Arlett）所創下的小聯盟全壘打紀錄（432支），成為史上在小聯盟擊出最多[[全壘打|全壘打]]的選手<ref name=Record>{{cite web |url=http://www.si.com/mlb/2015/08/03/mike-hessman-toledo-mud-hens-minor-league-home-run-record?xid=si_social |title=Mike Hessman breaks minor-league record with 433rd career home run |first=Dan |last=Gartland |date=August 3, 2015 |work=[[Sports_Illustrated|Sports Illustrated]] |accessdate=August 4, 2015}}</ref>。
