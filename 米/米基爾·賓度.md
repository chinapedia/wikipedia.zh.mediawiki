'''米基爾·安祖·賓度·謝雷斯'''（{{lang-es|'''Miguel Ángel Pinto Jerez'''}}，{{bd|1983年|7月4日|catIdx=Pinto}}）出生於[[智利|智利]][[聖地牙哥|聖地牙哥]]，智利足球員，司職[[守門員|守門員]]，現效力智利球會[[智利大学足球俱乐部|智利大學]]，並擔任球隊[[隊長_(足球)|隊長]]<ref>[http://www.udechile.cl/#/equipo.jugadores Jugadores - Primer Equipo - Universidad de Chile]</ref>。賓度於2002年首次出戰職業賽，共於2006年年初始為智利大學正選門將。

== 球會 ==
=== 智利大學 ===
賓度於2002年臨危受命獲得智利大學徵召，出戰球會宿敵[[科洛科洛竞技俱乐部|高路高路]]，當時首席門將受傷，而同場次席門將則被逐離場。賓度憑著出色的表現，使球會力保不失。一戰成名後，賓度在球會前途一片光明。<ref name="jcb">[http://jcbweb.wordpress.com/miguel-pinto/Miguel Pinto - JCBweb]</ref>賓度於2004年及2005年球季躍升為次選門將，成為莊尼·靴利拿（Johnny Herrera）的副選<ref name="jcb"/>。2006年，靴利拿轉會至[[哥連泰斯保利斯塔體育會|哥連泰斯]]，使賓度上位成為球會首席門將。2006年球季賓度因犯下低級錯誤而被受批評。但在2007年，賓度獲選為智利大學最佳球員。經過2008年成功的一季，智利大學獲得[[2009年南美解放者杯|2009年南美自由盃]]預賽入場券，賓度於兩回合賽事表現出高水準，助球會擊敗對手[[帕丘卡足球俱乐部|帕丘卡]]，昂然晉級小組賽事。<ref>[http://www.cooperativa.cl/prontus_nots/site/artic/20090122/pags/20090122144938.html Miguel Pinto desdramatizó el tema de las entradas para duelo con Pachuca. - cooperativa.cl]</ref>在小組賽作客[[阿雷格里港|阿雷格里港]]出戰[[甘美奧阿雷格里港足球會|甘美奧]]，賓度屢救險球，助球會和對手互交白卷，搶得寶貴分數。該仗賓度面對了[[巴西|巴西]]球隊射門次數多達25次<ref>{{cite web |url=http://www.adn.es/deportes/20090226/NWS-3585-Universidad-Miguel-Pinto-Chile-guardameta.html |title=存档副本 |accessdate=2009-07-07 |deadurl=yes |archiveurl=https://web.archive.org/web/20110719234336/http://www.adn.es/deportes/20090226/NWS-3585-Universidad-Miguel-Pinto-Chile-guardameta.html |archivedate=2011-07-19 }}</ref>。賓度在賽事10場賽事共失掉11球，其優秀表現獲得外界的垂青，更有傳[[墨西哥|墨西哥]]球會[[蓝十字足球俱乐部|藍十字]]於2009年年中提出以120萬[[美元|美元]]斟介他<ref>http://www.goal.com/es-la/news/689/m%C3%A9xico/2009/06/14/1324983/mercado-cerca-miguel-pinto-de-ser-jugador-de-cruz-azul</ref>。賓度助球會贏得2009年智利春季聯賽冠軍，這亦是他贏得的首座冠軍獎盃。

== 國家隊 ==
賓度代表智利出戰2003年[[南美青年足球錦標賽|南美青年盃]]，為U20賽事。2006年首次在國家一隊上陣，對戰[[科特迪瓦|科特迪瓦]]和[[愛爾蘭|愛爾蘭]]<ref>http://www.national-football-teams.com/v2/player.php?id=15287</ref>。前智利球練[[尼爾遜·阿哥斯達|尼爾遜·阿哥斯達]]（Nelson Acosta）挑選賓度為[[2007年美洲國家盃|2007年美洲國家盃]]三號門將。可是賓度因拒絕成為三號門將而沒有起程到[[委內瑞拉|委內瑞拉]]<ref name="kick">[http://www.kickoff.com/2010/teams/chile/player/2254/miguel-pinto.php Pinto Profile  - kickoff.com]{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。2007年[[阿根廷|阿根廷]]籍教練[[马塞洛·贝尔萨|比爾沙]]（Marcelo Bielsa）成為智利國家隊領隊，並將賓度選為二號門將，成為[[哥迪奧·巴禾|哥迪奧·巴禾]]的副選<ref name="kick"/>。[[2009年5月|2009年5月]]，賓度代表智利出戰[[麒麟盃|麒麟盃]]，上陣兩場失5球。<ref>[http://www.thefreelibrary.com/Chile+and+Belgium+draw+in+Kirin+Cup-a01611884463 Chile and Belgium draw in Kirin Cup - Free Online Library]</ref>

== 個人生活 ==
賓度自行設計自己的守門員球衣，仿傚前[[巴拉圭|巴拉圭]]門神[[何塞·路易斯·奇拉维特|芝拉華特]]<ref>http://www.terra.cl/deportes/index.cfm?accion=futbolnacional&id=744698</ref>。賓度將[[獅子|獅子]]和[[貓頭鷹|貓頭鷹]]也一併混入自己的設計，這其實是智利大學的[[吉祥物|吉祥物]]<ref>http://megagalerias.terra.cl/galerias/index.cfm?id_galeria=19442{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。賓度有一名孖生兄弟名為祖安·法蘭西斯高（Juan Francisco）<ref>{{cite web |url=http://www.lacuarta.cl/contenido/64_48105_9.shtml |title=存档副本 |accessdate=2009-07-07 |deadurl=yes |archiveurl=https://web.archive.org/web/20090707135847/http://www.lacuarta.cl/contenido/64_48105_9.shtml |archivedate=2009-07-07 }}</ref>。賓度亦有綽號「''Miguelito''」和「[[克利普頓石|克利普頓石]]」（''Criptonita''）<ref>[http://www.goal.com/en/news/184/player-profile/2010/05/01/1964266/world-cup-profile-miguel-pinto-chile World Cup Profile: Miguel Pinto - Chile - Goal.com]</ref>。
