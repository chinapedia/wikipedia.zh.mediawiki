'''米兰达·奥图'''（{{lang-en|Miranda Otto}}，{{bd|1967年|12月16日|catIdx=Otto, Miranda}}）是一位[[澳大利亚公民|澳大利亚]][[女演员|女演员]]，其雙親和妹妹也都是演员。奥图的演艺生涯于1986年开始，已经出演过多部独立和大型电影公司制作的电影。奥图的大银幕处女作是1986年的澳大利亚电影《埃玛的战争》（{{lang|en|''Emma's War''}}），在其中扮演一个于[[第二次世界大战|第二次世界大战]]期间搬到澳大利亚乡间生活的少女。接下来的十年里继续出演了多部澳大利亚电影，获得了广泛的好评，并在出演了[[1998年电影|1998年]]的[[战争片|战争片]]《[[细细的红线|细细的红线]]》和[[2000年电影|2000年]]的[[恐怖片|恐怖片]]《[[危机四伏|危机四伏]]》（{{lang|en|''What Lies Beneath''}}）后成为[[好莱坞|好莱坞]]影星。2000年代初，她在[[彼得·杰克逊|彼得·杰克逊]]的[[魔戒電影三部曲|《指环王》系列电影三部曲]]中饰演[[伊歐玟|伊歐玟]]一角，开始成为家喻户晓的女演员。

== 早年生活 ==
米兰达·奥托出生于澳大利亚[[昆士兰州|昆士兰州]]的[[布里斯班|布里斯班]]，她的父亲巴里·奥托（{{lang|en|Barry Otto}}），母亲林赛·奥托（{{lang|en|Lindsay Otto}}），以及同父异母的妹妹加茜·奥托（{{lang|en|Gracie Otto}}）都是演员<ref name="RightStage">{{cite web |archivedate=2013-09-21 |deadurl=no |archiveurl= https://web.archive.org/web/20130921060332/http://www.smh.com.au/news/film/the-right-stage-of-life/2005/09/25/1127586737926.html |url=http://www.smh.com.au/news/film/the-right-stage-of-life/2005/09/25/1127586737926.html |title=The Right Stage of Life |publisher=''The Sydney Morning Herald'' |date=2005-09-26 |accessdate=2014-01-17}}</ref>。米兰达在布里斯班和[[新南威尔士州|新南威尔士州]]的[[纽卡斯尔_(新南威尔士州)|纽卡斯尔]]长大，六岁那年父母[[离婚|离婚]]后，她短暂在[[香港殖民地時期|香港]]居住<ref name="BalancingAct">{{cite news |last=Scobie |first=Claire |title=Balancing Act |publisher=''The Herald Sun'' |date=2005-06-26}}</ref>。米兰达会到[[悉尼|悉尼]]与自己的父亲一起过周末和节日，并由此对表演产生了兴趣<ref name="FamilyViewing">{{cite web |last=Keenan |first=Catherine |archivedate=2013-09-21 |deadurl=no |archiveurl= https://web.archive.org/web/20130921060314/http://www.smh.com.au/news/arts/family-viewing/2005/09/29/1127804613819.html |url=http://www.smh.com.au/news/arts/family-viewing/2005/09/29/1127804613819.html |title=Family Viewing |publisher=''The Sydney Morning Herald'' |date=2005-10-01 |accessdate=2014-01-17}}</ref>。

童年时期的米兰达会在空余时间里和朋友们一起编写剧本，设计服装和传单<ref name="DoingItHerWay">{{cite web |last=Williams |first=Sue |url=http://miranda-otto.com/2003womensweekly.php |title=Doing It Her Way |publisher=''The Australian Women's Weekly'' |date=2003-07 |accessdate=2007-10-17 |archivedate=2007-10-17 |deadurl=yes |archiveurl= https://web.archive.org/web/20071017214540/http://miranda-otto.com/2003womensweekly.php}}</ref>。她在尼姆罗德剧院的多部[[舞台剧|舞台剧]]中亮相并引起了选角导演费斯·马丁（{{lang|en|Faith Martin}}）的注意。奥托随后获得了出演以第二次世界大战为背景的1986年电影《埃玛的战争》的机会<ref name="FamilyViewing" />。少女时代的奥托在学术和[[芭蕾舞|芭蕾舞]]上都有上佳表现，并一度考虑成为职业芭蕾舞演员<ref>{{cite news |author=J.W. Braun |title=''The Lord of the Films'' |publisher=ECW Press |year=2009}}</ref>。然而因患有中度[[脊椎側彎|脊椎側彎]]，她只能放弃这一理想<ref name="BalancingAct" />。1990年，奥托从悉尼的澳大利亚国家戏剧艺术研究所毕业<ref name="GoesOff">{{cite journal |last=Slee |first=Amruta |url=http://www.miranda-otto.com/1997hq.php |title=Miranda Otto Goes Off |publisher=''HQ Magazine'' |deadurl=yes |archivedate=2009-07-26 |accessdate=2009-07-26 |archiveurl= https://web.archive.org/web/20090726095522/http://www.miranda-otto.com/1997hq.php | date = 1997-09 & 1997-10}}</ref>。毕业前她还有在几部电影中出演过小配角，包括1987年的《引发》（{{lang|en|''Initiation''}}）和1988年的《异次元骇客》（{{lang|en|''The 13th Floor''}}）<ref name="Filmography">{{cite web |url=http://movies.yahoo.com/movie/contributor/1800010615/filmography |title=Miranda Otto Filmography |publisher=Yahoo! Movies |archivedate=2011-06-28 |accessdate=2011-06-28 |deadurl=yes |archiveurl=https://web.archive.org/web/20110628212714/http://movies.yahoo.com/movie/contributor/1800010615/filmography}}</ref>。

== 事业 ==
=== 早年事业 ===
奥托毕业后出演的第一部电影是1991年的《迟到女孩》（{{lang|en|''The Girl Who Came Late''}}），扮演内尔·蒂斯柯依兹（{{lang|en|Nell Tiscowitz}}）一角。这也是她事业上的突破口，为她赢得了澳大利亚电影业界和公众的注意。在这部由凯西·穆勒（{{lang|en|Kathy Mueller}}）执导的电影中，她诠释的是一位能够与[[马|马]]交流的年轻女性，并因此而于次年获得了[[澳大利亚电影学院|澳大利亚电影学院]]的最佳女主角奖提名。<ref name="awards" /><ref name="yahoo" />

奥托的下一部作品是1992年的电影《[[阖家不欢|阖家不欢]]》（{{lang|en|''The Last Days of Chez Nous''}}），该片讲述了一个澳大利亚[[家庭|家庭]]中各成员间的复杂关系。影片为她赢得了第二项澳大利亚电影学院奖提名，这一次提名的是最佳女配角奖<ref name="awards" /><ref name="yahoo">{{cite web |deadurl=yes |archivedate=2011-06-28 |archiveurl= https://web.archive.org/web/20110628212721/http://movies.yahoo.com/movie/contributor/1800010615/bio |url=http://movies.yahoo.com/movie/contributor/1800010615/bio |title=Miranda Otto Biography |publisher=Yahoo! Movies |accessdate=2011-06-28}}</ref>。1993年，奥托与[[诺亚·泰勒|诺亚·泰勒]]（{{lang|en|Noah Taylor}}）一起出演了带有性挑逗意味的[[喜剧片|喜剧片]]《[[末世恋情|末世恋情]]》（{{lang|en|''The Nostradamus Kid''}}），该片是根据作家[[鲍勃·埃利斯|鲍勃·埃利斯]]（{{lang|en|Bob Ellis}}）于1960年代撰写的回忆录改编。奥托在面对[[采访|采访]]时表示，自己接演该片是因为“对那段时期本身及其人物都感到着迷”<ref name="Interview">{{cite web |last=Bass |first=Matthew |url=http://www.tolkien-movies.com/words/2000/03-01-00b.htm |title=Miranda Otto Interview |publisher=''Cinema Papers'' |date=1997-10 |accessdate=2014-01-18 |archivedate=2013-12-12 |archiveurl=https://web.archive.org/web/20131212000443/http://www.tolkien-movies.com/words/2000/03-01-00b.htm |deadurl=no}}</ref>。到了1995年，她又在独立制片电影《性爱是个有四个字母的词》（{{lang|en|''Sex Is a Four Letter Word''}}）中出演了一个小角色<ref name="Filmography" />。

1995年，奥托因为没能得到自己参加试镜的角色而开始对自己事业上所做的选择产生疑虑<ref name="GoesOff" />。她飞回纽卡斯尔生活了近一年，粉刷[[母亲|母亲]]的房子<ref name="GoesOff" />。1996年，[[导演|导演]]雪莉·巴雷特（{{lang|en|Shirley Barrett}}）选中奥托出演电影《爱的小夜曲》（{{lang|en|''Love Serenade''}}），她诠释的是一位名叫迪米提·赫利（{{lang|en|Dimity Hurley}}）的孤独年轻女服务员，与[[姐姐|姐姐]]薇基-安（{{lang|en|Vicki-Ann}}）一起对来自布里斯班的一位知名{{lang|en|DJ}}产生了好感。影片获得了较为正面的评价<ref>{{cite web |accessdate=2014-01-18 |title=Love Serenade (1996) |url= http://www.rottentomatoes.com/m/love_serenade/ |publisher=Rotten Tomatoes |deadurl=no |archiveurl= https://web.archive.org/web/20131216153011/http://www.rottentomatoes.com/m/love_serenade/ |archivedate=2013-12-16}}</ref>，在线影评人史蒂夫·罗德斯（{{lang|en|Steve Rhodes}}）称赞奥托给出了片中最精彩的表现<ref>{{cite web |url=http://www.imdb.com/reviews/79/7981.html |title=Love Serenade (1996) |author=Steve Rhodes |accessdate=2014-01-18 |date=1997-06-19 |archivedate=2014-01-18 |deadurl=no |work=IMDb |publisher=Amazon |archiveurl= https://web.archive.org/web/20140118022322/http://www.imdb.com/reviews/79/7981.html}}</ref>。

奥托在1997年的电影《[[井_(1997年电影)|井]]》（{{lang|en|''The Well''}}）和《[[旅途歌手|旅途歌手]]》（{{lang|en|''Doing Time for Patsy Cline''}}）中开始担任主演。起初收到《井》的剧本时她并不愿打开细看，因为担心自己可能得不到这个角色。当时奥托已经30岁，所以她觉得自己来饰演片中年仅18岁的凯瑟琳（{{lang|en|Katherine}}）可能会缺乏可信度<ref name="GoesOff" />。这部由萨曼莎·朗{{lang|en|Samantha Lang}}）执导的电影讲述了奥托扮演的少女和一位年长女性之间的关系。影片所获评价褒贬不一，影评人保罗·费舍尔（{{lang|en|Paul Fisher}}）称奥托表演的可信度与她出演过的其他同类角色相比有所不及，而路易丝·凯勒（{{lang|en|Louise Keller}}）则称赞这是奥托从影以来最精彩的演出<ref>{{cite web |last1=Fisher |first1=Paul |last2=Louise |first2=Keller |archivedate=2013-12-13 |deadurl=no |archiveurl= https://web.archive.org/web/20131213152809/http://www.urbancinefile.com.au/home/view.asp?a=332&s=Reviews |url=http://www.urbancinefile.com.au/home/view.asp?a=332&s=Reviews |title=The Well Reviews |publisher=UrbanCinefile.com.au |year=1997 |accessdate=2014-01-18}}</ref>。奥托因本片获得了自己的第三次澳大利亚电影学院奖提名<ref name="yahoo" /><ref name="awards" />。同年晚些时候，她又与[[理查德·劳斯伯格|理查德·劳斯伯格]]（{{lang|en|Richard Roxburgh}}）一起出演了《旅途歌手》。奥托需要在这部低成本澳大利亚电影中表演[[乡村音乐|乡村音乐]]，影片所获评价同样褒贬不一<ref>{{cite web |archivedate=2012-11-07 |deadurl=no |url=http://www.rottentomatoes.com/m/1157715-doing_time_for_patsy_cline/ |title=Doing Time for Patsy Cline (2006) |publisher=Rotten Tomatoes |accessdate=2014-01-18 |archiveurl= https://web.archive.org/web/20121107111601/http://www.rottentomatoes.com/m/1157715-doing_time_for_patsy_cline/}}</ref>。

《井》和《旅途歌手》上映后，[[杂志|杂志]]和其它媒体都对这位女演员产生了兴趣。1997年，奥托开始与《旅途歌手》的男主演理查德·劳斯伯格约会，她也因此成为当时澳大利亚[[小型报|小报]]杂志的常规目标，并对此感到很不习惯<ref name="serene">{{cite web |last=Scobie |first=Claire |date=2005-06-26 |accessdate=2014-01-18 |archivedate=2012-11-04 |deadurl=no |archiveurl= https://web.archive.org/web/20121104163436/http://www.theage.com.au/news/film/serene-not-dreamy/2005/06/26/1119724516203.html |url=http://www.theage.com.au/news/film/serene-not-dreamy/2005/06/26/1119724516203.html |title=Serene, Not Dreamy |publisher=''The Age''}}</ref>。

奥托出演的下一部电影是1998年的浪漫喜剧《死信办公室》（{{lang|en|''Dead Letter Office''}}），她的[[父亲|父亲]]巴里在片中有短暂露面，这也是父女俩的首次合作。接下来奥托又在詹姆斯·博格尔（{{lang|en|James Bogle}}）导演的《冬日之光》（{{lang|en|''In the Winter Dark''}}）中饰演了一位名叫罗尼（{{lang|en|Ronnie}}）的女子，有孕在身但又被男友抛弃。影片在澳大利亚获得了广泛好评，奥托因此第4次入围澳大利亚电影学院奖<ref name="awards" /><ref name="yahoo" />。同年她在[[泰伦斯·马力克|泰伦斯·马力克]]执导，[[乔治·克鲁尼|乔治·克鲁尼]]和[[西恩·潘|西恩·潘]]等人主演的二战题材大成本电影《细细的红线》中出演了一个小角色，从此开始成为国际影星，在澳大利亚以外的多个国家参加演出<ref name="yahoo" />。例如她于2001年参演了[[意大利|意大利]]低成本电影《三条腿的狐狸》（{{lang|en|''La volpe a tre zampe''}}），该片于2004年2月在[[柏林国际电影节|柏林国际电影节]]上[[首演|首映]]，2005年11月5日在[[匈牙利|匈牙利]]电视频道首播，2009年3月在意大利电视台首播<ref>{{cite web |url= http://www.imdb.com/title/tt0259729/ |title=La volpe a tre zampe (2004) |publisher=Amazon |work=IMDb |accessdate= 2014-01-18 |archiveurl=https://web.archive.org/web/20120424080145/http://www.imdb.com/title/tt0259729/ |deadurl=no |archivedate=2012-04-24}}</ref>。

=== 好莱坞 ===
奥托出演的第一部[[美国电影|美国电影]]是2000年上映的[[惊悚片|惊悚片]]《危机四伏》，该片由[[罗伯特·泽米基斯|罗伯特·泽米基斯]]执导，[[哈里森·福特|哈里森·福特]]和[[米歇尔·菲佛|米歇尔·菲佛]]主演，奥托饰演的是福特和菲佛角色的神秘[[邻居|邻居]]玛丽·福尔（{{lang|en|Mary Feur}}）。影片所获评价褒贬不一<ref>{{cite web |url=http://www.rottentomatoes.com/m/what_lies_beneath/ |title=What Lies Beneath (2000) |publisher=Rotten Tomatoes |accessdate=2014-01-18 |archivedate=2013-10-26 |deadurl=no |archiveurl= https://web.archive.org/web/20131026011459/http://www.rottentomatoes.com/m/what_lies_beneath/}}</ref><ref>{{cite web |archiveurl=https://web.archive.org/web/20121023005040/http://www.metacritic.com/movie/what-lies-beneath |deadurl=no |url=http://www.metacritic.com/movie/what-lies-beneath |title=What Lies Beneath |publisher=[[CBS|CBS]] |work=[[Metacritic|Metacritic]] |accessdate=2014-01-18 |archivedate=2012-10-23}}</ref>，但商业上较为成功，全球票房超过2.91亿[[美元|美元]]<ref>{{cite web |title=What Lies Beneath |publisher= Amazon |work=Box Office Mojo |url=http://boxofficemojo.com/movies/?id=whatliesbeneath.htm |accessdate=2014-01-18 |archiveurl=https://web.archive.org/web/20140106224938/http://boxofficemojo.com/movies/?id=whatliesbeneath.htm |deadurl=no |archivedate=2014-01-06}}</ref>。2001年，奥托获选在喜剧片《[[人性_(电影)|人性]]》中饰演一位[[博物学|博物学家]]。[[编剧|编剧]][[查理·卡夫曼|查理·卡夫曼]]对她两年前参加自己电影《[[成为约翰·马尔科维奇|成为约翰·马尔科维奇]]》试镜时的表现留下了深刻印象，于是安排奥托参加试镜并与导演[[米歇尔·贡德里|米歇尔·贡德里]]见面<ref name="Venice">{{cite web |last=Landry |first=B. Jude |date=2002-04 |archivedate=2007-10-17 |deadurl=yes |url=http://www.miranda-otto.com/2002venice.php |title=Have Talent, Will Travel |publisher=''Venice: L.A.'s Arts and Entertainment Magazine'' |accessdate=2007-10-17 |archiveurl= https://web.archive.org/web/20071017214520/http://miranda-otto.com/2002venice.php}}</ref>。但是《人性》无论是在商业上还是专业评价上表现都很不理想<ref>{{cite web |url=http://boxofficemojo.com/movies/?id=humannature.htm |title=Human Nature |publisher=Amazon |work=[[Box_Office_Mojo|Box Office Mojo]] |accessdate=2014-01-18 |archivedate=2013-12-12 |archiveurl=https://web.archive.org/web/20131212114524/http://boxofficemojo.com/movies/?id=humannature.htm |deadurl=no}}</ref><ref>{{cite web |url=http://www.rottentomatoes.com/m/1113402-human_nature/ |title=Human Nature |publisher=Rotten Tomatoes |accessdate=2014-01-18 |archivedate=2013-04-30 |deadurl=no |archiveurl= https://web.archive.org/web/20130430220916/http://www.rottentomatoes.com/m/1113402-human_nature/}}</ref>，影评人杰弗里·M·安德森（{{lang|en|Jeffrey M. Anderson}}）批评了奥托的[[法国|法国]]口音，称她的角色看起来与周遭发生的事情格格不入<ref>{{cite web |last=Anderson |first=Jeffrey M. |date=2002-04 |archiveurl= https://web.archive.org/web/20131212144244/http://www.combustiblecelluloid.com/2002/humannat.shtml |archivedate=2013-12-12 |deadurl=no |url=http://www.combustiblecelluloid.com/2002/humannat.shtml |title=To Err Is 'Human' |publisher=CombustibleCelluloid.com |accessdate=2014-01-18}}</ref>。同年她还出现在[[英国广播公司|英国广播公司]]的[[电视剧|电视剧]]《[[如今世道|如今世道]]》中，该剧根据[[安东尼·特洛勒普|安东尼·特洛勒普]]的同名小说改编，奥托饰演的角色处心积虑地想让[[基利安·墨菲|基利安·墨菲]]扮演的角色和自己结婚<ref>{{cite web |url=http://www.imdb.com/title/tt0300879/ |title=The Way We Live Now (2001) |work= IMDb |publisher=Amazon |accessdate=2014-01-18 |archivedate=2013-07-27 |deadurl=no |archiveurl= https://web.archive.org/web/20130727233743/http://www.imdb.com/title/tt0300879/}}</ref>。

=== 剧场 ===
米兰达·奥托的[[劇場_(藝術)|剧场]]处女作是1986年悉尼剧院公司制作的《柏蒂娜的苦泪》（{{lang|en|''The Bitter Tears of Petra Von Kant''}}）<ref>{{cite web |url=http://www.inmyfathersden.com/cast.html#miranda |title=Cast Biographies |work=InMyFathersDen.com |accessdate=2009-02-23 |archivedate=2009-02-23 |deadurl=yes |archiveurl= https://web.archive.org/web/20090223180320/http://www.inmyfathersden.com/cast.html#miranda}}</ref>。之后从1980年代末到1990年代初，她又出演了另外三部该公司制作的舞台剧。2002年，她回归剧场舞台，在《[[玩偶之家|玩偶之家]]》中扮演诺拉·赫尔默（{{lang|en|Nora Helmer}}），与之后将成为她[[丈夫|丈夫]]的男演员皮特·奥布莱恩演对手戏。这次演出为奥托赢得了2003年[[赫尔普曼奖|赫尔普曼奖]]提名和{{lang|en|MO}}奖年度最佳舞台剧女演员奖<ref>{{cite web |url= http://www.helpmannawards.com.au/2003/past-nominees-and-winners/theatre |title=Past nominees and winners |deadurl=no |archiveurl=https://web.archive.org/web/20140118062931/http://www.helpmannawards.com.au/2003/past-nominees-and-winners/theatre |accessdate=2014-01-18 |archivedate=2014-01-18}}</ref><ref>{{cite web |url= http://unitedagents.co.uk/miranda-otto#profile-1 |title=Miranda Otto CV |work=UnitedAgents.co.uk |accessdate=2014-01-18 |archiveurl=https://web.archive.org/web/20120807113530/http://unitedagents.co.uk/miranda-otto#profile-1 |deadurl=no |archivedate=2012-08-07}}</ref><ref>{{cite web |url=http://moawards.com/awards/award-winners/2001-2007/2003-2/ |title=WINNERS - 28th 'MO' AWARDS 2003 |accessdate=2014-01-18 |archivedate=2013-12-20 |deadurl=yes |archiveurl=https://web.archive.org/web/20131220165044/http://moawards.com/awards/award-winners/2001-2007/2003-2/ }}</ref>。

她的下一步剧场作品是2005年的心理惊悚舞台剧《男孩追求女孩》（{{lang|en|''Boy Gets Girl''}}），扮演的是纽约一家杂志社的[[记者|记者]]。奥托是在发现自己怀孕前几天答应出演的，导演罗宾·内文（{{lang|en|Robyn Nevin}}）为此将制作时间从2004年12月改为2005年9月，以便她能够参加演出<ref name="RightStage" />。2005年，内文又开始了一部自己专为奥托定制舞台剧的前期制作工作<ref name="RightStage" />。

== 个人生活 ==
2003年1月1日，米兰达·奥托与[[男演员|男演员]]皮特·奥布莱恩成婚，两人是在参加《玩偶之家》演出时认识的<ref>{{cite web |url=http://www.tribute.ca/people/miranda-otto/7797/ |title=Star Bios: Miranda Otto |publisher=Tribute.ca |accessdate= 2014-01-18 |archiveurl=https://web.archive.org/web/20120806065051/http://www.tribute.ca/people/miranda-otto/7797/ |archivedate=2012-08-06 |deadurl=no}}</ref>。两人的独女达茜（{{lang|en|Darcey}}）于2005年4月1日出生。怀着女儿的奥托差点不得不放弃出演《世界大战》的机会，女儿出世后，她开始淡出影坛，以便腾出更多的时间陪伴自己身在澳大利亚的家人<ref name="BalancingAct" />。

在2004年接受采访时奥托表示，希望自己永远都不要像另一位澳大利亚女星[[妮可·基德曼|妮可·基德曼]]那样名扬天下，因为她觉得自己可能永远都无法处理好那样的生活<ref>{{cite web |url=http://miranda-otto.com/2004now.php |title=The Lord Of The Rings Star Miranda Otto Hopes She'll Never Be as Famous as Fellow Aussie Nicole Kidman |publisher=''Now Magazine'' |archiveurl=https://web.archive.org/web/20081122001427/http://www.miranda-otto.com/2004now.php |archivedate=2008-11-22 |date=2004-01 |accessdate=2008-11-22 |deadurl=yes}}</ref>。
