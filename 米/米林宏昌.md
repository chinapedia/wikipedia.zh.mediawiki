'''米林宏昌'''（{{bd|1973年|7月10日|||Yonebayashi Hiromasa}}）是[[日本|日本]]的[[動畫師|動畫師]]和[[動畫|動畫]][[導演|導演]]任職於[[吉卜力工作室|吉卜力工作室]]。

== 經歷 ==
米林宏昌於1973年出生於日本[[石川縣|石川縣]][[石川郡|石川郡]][[野野市町|野野市町]]。自[[金澤美術工藝大學|金澤美術工藝大學]]商業設計科畢業後<ref>[https://web.archive.org/web/20091219165538/http://hochi.yomiuri.co.jp/entertainment/news/20091217-OHT1T00039.htm 宮崎監督「バクチ」…来夏公開最新作にジブリ最年少監督] - スポーツ報知、2009年12月17日</ref>，於1996年進入[[吉卜力工作室|吉卜力工作室]]。以動畫師的身分參與過多部[[宮崎駿|宮崎駿]]執導的動畫作品。在《[[神隱少女|神隱少女]]》、《[[霍爾的移動城堡|霍爾的移動城堡]]》、《[[崖上的波妞|崖上的波妞]]》等作品中擔任了[[原畫|原畫]]的工作。此外，在[[宮崎吾朗|宮崎吾朗]]執導的《[[地海戰記|地海戰記]]》中以副作畫監督的身分參與。此外，短篇動畫方面，米林也擔任《[[空想的飛行機械們|空想的飛行機械們]]》的[[作畫監督|作畫監督]]和執導了《[[梅與小貓巴士|梅與小貓巴士]]》。

2008年米林在[[宮崎駿|宮崎駿]]、[[鈴木敏夫|鈴木敏夫]]、[[星野康二|星野康二]]三人的提議下、將[[瑪麗·諾頓|瑪麗·諾頓]]小說《[[地板下的小矮人|地板下的小矮人]]》改編成動畫電影《[[借物少女艾莉緹|借物少女艾莉緹]]》。<ref>{{Cite book |title={{lang|ja|借りぐらしのアリエッティ　ロマンアルバム}} |location=日本 |publisher=德間書店 |date=2010 |pages=101頁  |ISBN=978-4-19-720310-9 |accessdate=2012-09-06 }}</ref>此為米林首次擔任長篇作品的導演<ref name="sankei20091216">[http://sankei.jp.msn.com/entertainments/game/091216/gam0912161812005-n1.htm スタジオジブリ、新作アニメを来夏公開 新人監督を起用 - MSN産経ニュース] {{webarchive|url=https://web.archive.org/web/20101010115022/http://sankei.jp.msn.com/entertainments/game/091216/gam0912161812005-n1.htm |date=2010-10-10 }} - [[産経デジタル|産経デジタル]]、2009年12月16日。</ref><ref>[[共同通信社|共同通信]] [http://www.47news.jp/CN/200912/CN2009121601000726.html 宮崎駿さん企画でジブリ新作 来夏公開「アリエッティ」 - 47NEWS（よんななニュース）] - 全国新聞ネット、2009年12月16日。</ref><ref>[http://www.nikkei.co.jp/news/sangyo/20091216AT1D1606B16122009.html ジブリ、英の児童文学を下敷きにした新作――10年夏に公開 - NIKKEI NET（日経ネット）] - [[日本經濟新聞社|日本經濟新聞社]]、2009年12月16日。</ref>，並根據宮崎駿編寫的[[腳本|腳本]]進行電影的製作。米林宏昌也是吉卜力工作室歷年作品中最年輕的導演<ref name="sankei20091216"/>。

在和吉卜力影片製作人[[西村義明|西村義明]]完成2014年動畫《[[回憶中的瑪妮|回憶中的瑪妮]]》後，米林宏昌仍有繼續製作電影動畫的念頭，但當時吉卜力已開始有計劃要解散動畫製作部門。之後米林宏昌則離開吉卜力前往西村義明於2015年另成立的動畫工作室[[Studio_Ponoc|Studio Ponoc]]；繼續合作拍攝電影動畫。<ref>{{cite web |url=http://www.cinematoday.jp/page/N0088351 |script-title=ja:ジブリの血を受け継ぐ新作アニメに宮崎駿、高畑勲、鈴木敏夫の反応は？ |work=Cinema Today |date=2016年12月15日 |accessdate=2016年12月31日}}</ref>
