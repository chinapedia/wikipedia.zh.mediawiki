'''約翰·歐比·米基爾'''（{{lang-en|'''John Obi Mikel'''}}，{{bd|1987年|4月22日|catidx=Mikel}}）是一名[[奈及利亞|-{zh-hans:尼日利亚;zh-hk:尼日利亞;zh-tw:奈及利亞;}-]][[足球員|足球員]]，現效力於[[英冠|英冠]]球隊[[米德尔斯堡足球俱乐部|米德尔斯堡]]，曾效力[[中超|中超]]球隊[[天津泰达足球俱乐部|天津泰达]]，在隊中擔任[[中場|中場]]。

這名非洲球員於2005年[[世界少年足球錦標賽|世界少年足球錦標賽]]開始走紅，曾被[[英超|英超]][[曼聯|曼聯]]收購，但後來因轉會引發的爭議，最後改投了另一支球會[[車路士|車路士]]。他加盟[[車路士|車路士]]首兩年已越加受到重用，兩季各上陣逾廿場聯賽。

== 生平 ==
=== 早年 ===
約翰·米基爾出生時的名字是約翰·邁克爾·安卓古·奧賓納（John Michael Nchekube Obinna），父親是一名[[公務員|公僕]]。15歲時効力當地球會[[高原州聯隊|高原州聯隊]]（Plateau United），到2003年時首次代表國家參加在[[芬蘭|芬蘭]]舉辦的[[世界少年足球錦標賽|世少盃]]。其時因為評述員不斷把其名字串錯，把麥克（Michael）串成「米基爾」（Mikel）。所以他決定以後使用「米克尔」這個名稱，指這決定有特別的意思。

在[[南非|南非]]的[[开普敦阿贾克斯足球俱乐部|开普敦阿贾克斯]]（Ajax Cape Town）短暫効力後，米基爾在2005年開始為人熟悉，他參加了[[世界青年足球錦標賽|世界青年足球錦標賽]]，他帶領國家隊晉級至決賽，才被[[阿根廷國家足球隊|阿根廷]]以2-1擊敗。米基爾在賽後獲得銀球獎，為世青杯第二最佳球員。<ref>{{cite news |first=Matt |last=Spiro |url=http://www.uefa.com/magazine/news/Kind=512/newsId=392702.html |title=Mikel thrives on centre stage |publisher=UEFA |date=2006年1月30日 |accessdate=2006年6月3日 }}</ref>其時他效力[[挪威|挪威]]聯賽球會[[利恩|利恩]]。

=== 轉會爭議 ===
米基爾是於2006年轉投[[英超|英超]]的[[車路士|車路士]]，但轉會時卻引發了一些爭議，當中還牽涉到同聯賽球會曼聯。

2005年4月29日，[[曼聯|曼聯]]表示他們和[[利恩|利恩]]達成協議，成功收購米基爾，簽約4年。<ref>{{cite web |date = 2005年4月29日 |title = Man Utd get Chelsea target Mikel |url = http://news.bbc.co.uk/sport1/hi/football/teams/m/man_utd/4499893.stm |publisher = BBC Sport |accessdate = 2006年6月5日 }}</ref>消息隨後公佈於其官方網站。但其間米基爾經理人並不知情，而曼聯所簽的是沒約束性合約。據記者指該合約值400萬英鎊，<ref>{{cite web |date = 2005年6月18日 |title = Obi makes plea for Chelsea switch |url = http://news.bbc.co.uk/sport1/hi/football/4106482.stm |publisher = BBC Sport |accessdate = 2006年6月5日 }}</ref>而米基爾將於2006年1月加盟曼聯。與此同時曼聯死敵[[車路士|車路士]]也發表聲明，指已和米基爾經理人達成協議，但遭利恩否認。而後來的報導指出，[[車路士|車路士]]早就安排米基爾轉會事宜，只是簽約時間較後。

為作澄清，米基爾匆匆召開記者會，稱自己很高興能加入曼聯，也拍了一張手持曼聯球衣的相片，顯示的球衣號碼是21號。這時有傳聞指利恩一方收到一些匿名恐嚇電話；而在2005年5月11日[[挪威杯|挪威杯]]一場比賽進行時，米基爾突然失蹤，引來挪威傳媒大幅報導。利恩主席聲稱米基爾「被綁架」；而曼聯隨後也作相同聲明。他們都指控車路士牽涉這次綁架米基爾的行動中。<ref>{{cite web |date = 2005年5月13日 |title = Carlos Demands Authorities Act On Mikel Saga |url = http://www.manutd.com/news/fullstory.sps?iNewsId=175542&itype=466&icategoryid=120 |publisher = Manchester United website |accessdate = 2006年6月5日 |archive-url = https://web.archive.org/web/20060515133606/http://www.manutd.com/news/fullstory.sps?iNewsId=175542&itype=466&icategoryid=120 |archive-date = 2006年5月15日 |dead-url = yes }}</ref>

後來米基爾被發現和他的經理人一同到了[[倫敦|倫敦]]。米基爾在酒店接受[[天空體育|天空體育]]訪問，聲稱他是被迫和曼聯簽約。<ref>{{cite web |date = 2005年5月20日 |title = Obi desperate to sign for Chelsea |url = http://news.bbc.co.uk/sport1/hi/football/teams/m/man_utd/4564623.stm |publisher = BBC Sport |accessdate=2006年6月5日 }}</ref>他聲稱，曼聯曾給他一個星期考慮，遭他拒絕，而曼聯卻在米基爾沒有代表的情況下硬迫著簽約。曼聯於是向國際足協投訴車路士和米基爾兩位經理人之行為。當時其中一位經理人曾有違規前科。<ref>{{cite web |date = 2005年5月18日 |title = Man Utd/Lyn go to Fifa in Obi row |url = http://news.bbc.co.uk/sport1/hi/football/africa/4560453.stm |publisher = BBC Sport |accessdate = 2006年6月5日 }}</ref>

=== 最終轉會至車路士 ===
米基爾轉會僵局在其後出現突破。車路士主動向曼聯和利恩提出新的轉會框架。<ref>{{cite web |date = 2006年6月2日 |title = Chelsea to sign Jon Obi Mikel |url = http://www.chelseafc.com/article.asp?hlid=392067&m=6&y=2006&nav=news&sub=latest+news |publisher = Chelsea Football Club |accessdate = 2006年6月5日 }}{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>結果在2006年6月2日，車路士、曼聯和利恩達成協議：米基爾轉投車路士，曼聯同意終止相關合約。車路士支付曼聯1,200萬英鎊作為轉會費，同時並向利恩支付400萬英鎊。協議簽訂後，爭議隨之結束。<ref>{{cite news |url=http://www.manutd.com/news/fullstory.sps?iNewsid=335071&itype=466&icategoryid=120 |title=Reds Agree £12m Mikel Fee With Chelsea |publisher=Manchester United |date=2006年6月2日 |accessdate=2006-06-02 }}{{Dead link|date=2018年7月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

2006年7月19日，米基爾正式加盟車路士。<ref> {{cite news |url=http://soccernet.espn.go.com/news/story?id=374089&cc=5739 |title=Champs Chelsea secure Obi Mikel permit |publisher=ESPNsoccernet |date=2006年7月19日 |accessdate=2006年7月19日 }}</ref>米基爾表示希望大家把他稱呼作約翰·米基爾，而不是米基爾·歐比·約翰，因為這個用法較常聽到。<ref> {{cite news |url=http://www.chelseafc.com/article.asp?hlid=407157&m=8&y=2006&nav=news&sub=latest+news |title=July Diary: Overlapping seasons and emotions |publisher=Chelsea FC |date=2006年8月1日 |accessdate=2006年8月11日 }}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

== 國際賽生涯 ==
米基爾首次亮相[[尼日利亞國家足球隊|尼日利亞國家隊]]，是在2005年8月17日的一場友誼賽，他在下半場入替上場，最後以1-0擊敗[[利比亞國家足球隊|利比亞]]。之後米基爾要到[[2006年非洲國家杯|2006年非洲國家杯]]才再次被徵召入國家隊。尼日利亞首場分組賽是對[[加納國家足球隊|加納]]，米基爾沒有出場。不過，他在第二場上陣，面對[[津巴布韋國家足球隊|津巴布韋]]，在下半場早段的十分鐘內，分別開出角球造就隊友克里斯蒂安·奧博多（Christian Obodo）頂入，和親自射入了一球。到最後一場分組賽，他首次在正選的名單內，該場尼日利亞以2-1擊敗了[[塞內加爾國家足球隊|塞內加爾]]。截至2006年2月7日，他為國家隊上陣了四次，入了一球。在杯賽期間，他被指示不能對他在球會的事業發表意見。<ref>{{cite news |url=http://www.supersoccer.co.za/default.asp?id=168855&des=article&scat=supersoccer/nigeria |title=Obi Keeps Mum |publisher=Supersport News |date=2006年1月27日 |accessdate=2006年6月2日 }}</ref>而國際足協則調查有關米基爾接到死亡恐嚇的事宜。<ref>{{cite news |url=http://news.bbc.co.uk/sport2/hi/football/4661150.stm |title=Man Utd ask for Chelsea sanctions |publisher=BBC |date=2006年1月30日 |accessdate = 2006年6月2日 }}</ref>

== 車路士生涯 ==
[[File:Chelseaplayers.jpg|thumb]]
2006年9月12日，米基爾首次為車路士上陣，在[[歐洲聯賽冠軍杯|歐聯]]分組賽對[[索菲亞利夫斯基|索菲亞利夫斯基]]，他射了個令對方守門員無法挽救的射門，造就[[杜奧巴|杜奧巴]]截獲皮球偷襲得手，賽後米基爾獲得好評。不過，在2006年10月14日對[[雷丁足球會|雷丁]]的賽事中，他被球證趕出場。而他也因練習遲到，被罰三場閉門練習。當時會方對他在場外生活甚有微言，還傳出球會打算把米基爾售出。<ref>{{cite news |url=http://www.football365.com/story/0,17033,8652_1693417,00.html |title=Sorry Mikel attempts to save Chelski career |publisher=Football365.com |date=2006年11月17日 |accessdate=2006-11-17 |deadurl=yes |archiveurl=https://web.archive.org/web/20070909065655/http://www.football365.com/story/0,17033,8652_1693417,00.html |archivedate=2007年9月9日 }}</ref>米基爾其後被放逐一個月，其間他父親表示關注他的行為。<ref>{{cite news |url=http://home.skysports.com/list.aspx?hlid=429228&CPID=8&clid=8&lid=&title=Mikel+antics+causing+concern |title=Mikel Antics Causing Concern |publisher=Sky Sports |date=2006年11月16日 |accessdate=2006年11月23日 }}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>在他的行為有所改善後，米基爾獲重召，於2006年11月23日[[歐洲聯賽冠軍盃|歐洲聯賽冠軍盃]]分組賽作客對[[雲達不來梅|雲達不來梅]]的賽事中上陣。而在2007年1月6日[[英格蘭足總杯|英格蘭足總杯]]對[[马科斯菲尔德足球俱乐部|马科斯菲尔德]]的賽事中，他射入了加盟以來首個入球，協助球會以6-1大勝對手。在下一圈對[[諾定咸森林|諾定咸森林]]的賽事中，他也射入了一球。

米基爾於2006-07年度的球季共為車路士上陣22次，並於足總杯決賽正選上陣，為車路士贏得2007年度足總杯冠軍，表演受到一致好評；2007-08年球季也上陣了29場聯賽，只是未有入球進帳。

2012年11月22日，英格蘭足總正式指控米基爾，他涉嫌在上月對陣[[曼聯|曼聯]]的聯賽中侮辱球證[[马克·克拉滕博格|马克·克拉滕博格]]而被罰三場停場和罰款六萬英鎊。

值得一提是，直至2013年7月6日，米基爾仍然從未在英超聯賽中為車路士取得入球。

直到2013年9月22日，主場對陣富勒姆的比賽，終斬獲英超第一球，並協助球隊以2:0勝出。
