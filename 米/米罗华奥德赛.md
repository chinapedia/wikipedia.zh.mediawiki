{{vgname2|米罗华奥德赛|en=Magnavox Odyssey|cn=米罗华奥德赛|tw=美格福斯奧德賽|简称「'''奥德赛'''」}}，是全球第一款商业家用[[电子游戏机|电子游戏机]]。它的首次演示是在1972年5月24日，并且于同年八月正式发行。在日期上比[[雅达利|雅达利]]（{{lang|en|Atari}}）的《[[乓|乓]]》（{{lang|en|Pong}}）游戏平台早了三年。

奥德赛是由[[拉夫·亨利·贝尔|拉夫·亨利·贝尔]]（{{lang|en|Ralph Henry Baer}}）团队设计的，其原型机“棕盒子”设计于1966年9月到1967年2月期间，并获得了Magnavox购买其[[专利|专利]]以转化为商用游戏机。其原型现在美国[[华盛顿特区|华盛顿特区]]的[[史密森尼學會|史密森尼學會]][[美国国家历史博物馆|美国国家历史博物馆]]展出。

奥德赛在北美市場售價100美元，使用媒體為[[印刷電路板|印刷電路板]]，配件有[[光線槍|光線槍]]。

== 詳細 ==

奧德賽於1972年8月正式發售，首批生產量為50,000部。由於美格福斯認為首部電子遊戲機足以吸引消費者，因此奧德賽只會於美格福斯認可的銷售商發售，而並非所有一般零售商。於1972年，美格福斯共生產了120,000至140,000部，但是首年銷售量卻只有69,000至接近100,000部<ref name="VGHttl">{{cite web |url=https://videogamehistorian.wordpress.com/2015/11/16/1tl200-a-magnavox-odyssey/ |title=1TL200: A Magnavox Odyssey |work=They Create Worlds |last=Smith |first=Alexander |date=2015-11-16 |accessdate=2016-05-02 |archiveurl=https://web.archive.org/web/20160202091941/https://videogamehistorian.wordpress.com/2015/11/16/1tl200-a-magnavox-odyssey/ |archivedate=2016-02-02 |deadurl=no }}</ref>。主要原因可能是由於售價過高，限制銷售點以及只能在美格福斯電視上運行等。

雖然美格福斯曾考慮於當年停止生產奧德賽，但由於觀察到市場仍有需求，因此在1973年再生產27,000部。此外，奧德賽亦於12個國家發售，包括澳洲、比利時、法國、德國、希臘、以色列、意大利、蘇聯、西班牙、瑞士、美國及委內瑞拉<ref name=baerwinter/><ref name=VGHttl/>。1973年末，美格福斯推出大型廣告計劃，令奧德賽於1974年的銷售量達到129,000至150,000部，但是美格福斯於1975年正式停止生產奧德賽，因為他們決定發售2款專用遊戲機：奧德賽100及奧德賽200，並以此為基礎推出總共11款專用遊戲機。而奧德賽的正統後繼電子遊戲機，[[美格福斯奧德賽²|美格福斯奧德賽²]]，要在1978年才正式推出<ref name=VGHttl/>。

奧德賽成功展示電子遊戲機的潛能，但是它本身並不是一款十分成功的產品。因為美格福斯並沒有專注於生產大量遊戲，亦沒有進一步改進奧德賽的功能，包括增加聲效系統、增加可遊玩人數及更便宜的改進版本。而在奧德賽於1972年發售後，3年內都沒有其他電子遊戲機的推出，直至1975年由[[雅達利|雅達利]]推出''[[乓|乓]]''的家用專用遊戲機，以及1976年由{{link-en|Coleco|Coleco}}推出的[[Telstar_(遊戲機)|Telstar]]及[[快捷半導體公司|快捷半導體公司]]的[[Fairchild_Channel_F|Fairchild Channel F]]<ref name=VGHttl/>。

== 遊戲列表 ==
總計共有28款遊戲在美格福斯奧德賽上推出，其中有13款是隨主機一併發售（美國包含其中12款，其他國家包含其中10款）。6款為獨立發售，1款為玩家填寫意見卡後贈送，4款隨光線槍一併發售，而最後4款遊戲於1973年發售<ref name=VGHttl/>。
