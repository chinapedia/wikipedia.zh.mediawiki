'''米哈伊尔·伊拉里奥诺维奇·戈列尼谢夫-库图佐夫'''（{{lang-ru|'''Михаи́л Илларио́нович Голени́щев-Куту́зов'''}}，[[俄语罗马化|罗马化]]：''Mikhail Illarionovich Golenishchev-Kutuzov''；{{bd|1745年|9月16日|1813年|4月28日|catIdx=K}}），[[俄罗斯帝国|俄罗斯帝国]][[元帅|元帅]]，著名将领，[[军事家|军事家]]，1812年曾率领俄国军队击退[[拿破仑|拿破仑]]的大军，取得[[俄法战争|俄法战争]]的胜利。

== 年少時期 ==
庫圖佐夫出生於俄國一個中級貴族家庭，他的父親是一位專門研究[[堡壘|堡壘]]建築的軍事工程師，軍階中將。他小時候對[[主日學|主日學]]的課程極為厭倦。

== 對沙皇亞歷山大一世的七十則規勸 ==
此七十則規勸是庫圖佐夫為規勸[[亚历山大一世_(俄国)|亞歷山大一世]]沙皇作為歐洲霸主和生活處事態度而寫的，當中教導人生態度的句子在當今俄國成為了[[諺語|諺語]]。

'''有關國家事務：'''
# 臣自知臣壽命將盡，無緣再見陛下一面。因此向陛下作最後的進言，以下各條，請陛下務必遵守。
# 請陛下在打敗拿破崙時請勿放過拿破崙，應將他立即殺死，以防止他這個野心家捲土重來。
# [[波蘭人|波蘭人]]民最近的獨立聲音十分高，請避免運用高壓政策管治，應該建立一個[[傀儡政權|傀儡政權]]來管理。
# [[奧地利帝國|奧]]皇和[[普魯士|普魯士]]王都是膽小和極端貪心的人，不可以對他們過度信任，但反過來說，他們方便被陛下利用。
# 烏克蘭[[列昂尼德公爵|列昂尼德公爵]]在這場戰爭中顯示出非凡的才能，請陛下重用。

== 生平简介 ==
* 1745年生于[[圣彼得堡|圣彼得堡]]的中将军事工程师家庭。
* 1759年炮兵工程学校毕业后留校任教。
* 1761年晋升准尉并任阿斯特拉罕步兵团连长。
* 1762年晋升大尉。
* 1768～1774年和1787～1791年[[俄土战争|俄土战争]]期间，先后在[[鲁缅采夫|鲁缅采夫]]和[[亚历山大·苏沃洛夫|亚历山大·苏沃洛夫]]麾下任队列军官和参谋，作战勇敢，指挥有方，因战功晋升为少校。。
* 1774年在阿卢什塔以北舒马村(今[[库图佐夫卡|库图佐夫卡]])的战斗中负伤，失去右眼，得名“独眼将军”。出国治疗期间，考察[[普鲁士|普鲁士]]、[[英国|英国]]、[[荷兰|荷兰]]等国军事。
* 1776年在[[苏沃洛夫|苏沃洛夫]]的克里木第2团任职，1777年晋升为上校，任卢甘斯克长枪团团长，后任马里乌波尔轻骑兵团团长。
* 1782年晋升准将。
* 1784年晋升少将。
* 1785年任他组建的布格猎骑兵军军长。
* 1790年12月指挥第6纵队强攻并占领[[伊兹梅尔战役|伊兹梅尔]]，战功卓著，晋升为中将。
* 1792年起先后任驻[[奥斯曼帝国|土耳其]]大使、陆军贵族武备学校校长(1794年)、驻[[芬兰|芬兰]]俄军司令(1795年)、[[立陶宛|立陶宛]]督军(1799—1801）和[[圣彼得堡|圣彼得堡]]督军(1801—1802)。
* 1798年晋升为步兵上将。
* 1802年失宠，被免去军职，在自己的庄园闲居。
* 1805年俄、奥、英等国结成第三次[[反法联盟|反法联盟]]后，率俄军主力前往[[奥地利|奥地利]]，迎击[[拿破仑一世|拿破仑一世]]统率的法军。鉴于奥军在[[乌尔姆战役|乌尔姆战役]]惨败，指挥俄军主力后撤400余公里，从而保存了实力。[[奥斯特利茨战役|奥斯特利茨战役]]中，俄军由于沙皇[[亚历山大一世_(俄国)|亚历山大一世]]干预遭惨败。
* 1806年9月库图佐夫任[[基辅|基辅]]督军。
* 1809年6月调任[[立陶宛|立陶宛]]督军。
* 1806～1812年[[第七次俄土战争|第七次俄土战争]]中，于1811年出任[[摩尔多瓦|摩尔达维亚]]俄军总司令，采取退避三舍、相机反击战法，在同年8月[[鲁什丘克战役|鲁什丘克战役]]中以少胜多击败土军主力，迫使土耳其签订《[[布加勒斯特条约_(1812年)|布加勒斯特条约]]》。
* 由于亚历山大一世对库图佐夫怀有恶感，他在军队的领导职务被再次撤销。
* 1812年[[俄法战争|卫国战争]]中，8月20日，在俄军失利情况下出任总司令，面对优势法军实施战略撤退和坚壁清野，9月7日在[[博罗季诺战役|博罗季诺战役]]中重创法军后继续后撤，直至放弃[[莫斯科|莫斯科]]。后又指挥俄军从北、东、南三个方向进攻法军，歼敌于[[别列津纳河|别列津纳河]]畔鲍里索夫地区。
* 1812年12月12日因功被封为[[斯摩棱斯克|斯摩棱斯克]][[公爵|公爵]]并荣获最高战功勋章——一级[[乔治勋章|乔治勋章]]。
* 1813年4月率部向[[法国|法国]]本土推进前夕，病逝于[[西里西亚|西里西亚]]小城本茨劳（今[[波兰|波兰]][[博莱斯瓦维茨|博莱斯瓦维茨]]），他的遗体作防腐处理后运回圣彼得堡，葬于[[喀山主教座堂_(圣彼得堡)|喀山大教堂]]。
