{{日本作品|就活家族～一定會順利的～|就活家族～きっと、うまくいく～}}，日本[[朝日電視台|朝日電視台]]於2017年1月12日起，每週[[星期四|星期四]]21:00 - 21:54（[[日本標準時|JST]]）播放的電視劇，此劇由[[三浦友和|三浦友和]]主演，亦是三浦睽違17年之後再度主演連續劇<ref>{{Cite web |date=2016-11-10 |url=http://mainichi.jp/articles/20161109/dyo/00m/200/030000c |title=三浦友和 17年ぶり連ドラ主演 黒木瞳、前田敦子らと“就活家族”に |publisher=[[毎日新聞|毎日新聞]] |accessdate=2016-12-22 |deadurl=yes |archiveurl=https://web.archive.org/web/20161223132112/http://mainichi.jp/articles/20161109/dyo/00m/200/030000c |archivedate=2016-12-23 }}</ref>，描寫家庭成員面臨失業等問題的題材，日語「就活」為「就業活動」之簡稱。[[馬來西亞|馬來西亞]]觀眾可透過[[dimsum|dimsum]]觀看，台灣則由[[Line_TV|Line TV]]、[[KKTV|KKTV]]播出。

== 拍攝花絮 ==
本劇飾演[[前田敦子|前田敦子]]男友「真壁雄斗」一角原由[[成宮寬貴|成宮寬貴]]飾演，但拍攝至第四集時，成宮因個人因素突然宣布退出演藝圈，此角色由[[渡邊謙|渡邊謙]]之子{{link-ja|渡邊大|渡辺大}}代替接手，原成宮所拍攝完成的戲分全部作廢<ref>{{cite news|url=http://www.nikkansports.com/entertainment/news/1764253.html|title=成宮氏降板の「就活家族」、初回は11％|newspaper=[[日刊體育|日刊體育]]|date=2017-01-13|accessdate=2017-01-13}}</ref>。

== 登場角色 ==
=== 主要人物===
*富川洋輔：[[三浦友和|三浦友和]]（香港配音：[[張炳強|張炳強]]）
:日本鋼鐵金屬的人事部部長。
*富川水希：[[黑木瞳|黑木瞳]]（香港配音：[[許盈|許盈]]）
:洋輔妻子。私立中學的國文教師。
*富川栞：[[前田敦子|前田敦子]]（香港配音：[[張頌欣|張頌欣]]）
:富川家長女。原在珠寶店「Jewel D」上班，因受到上司的[[性騷擾|性騷擾]]而提出調職。
*富川光：[[工藤阿須加|工藤阿須加]]（香港配音：[[黃積權|黃積權]]）
:富川家長男，栞的弟弟。從三流大學畢業後，為就職感到煩惱。

===日本鋼鐵金屬===
*川村優子：[[木村多江|木村多江]]（香港配音：[[梁少霞|梁少霞]]）
:日本鋼鐵金屬的員工。對富川洋輔抱有好感。
*綿引雄二：{{link-ja|神保悟志|神保悟志}}（香港配音：[[陳欣_(配音員)|陳欣]]）
:總務部長。
*織部和久：{{link-ja|山田明郷|山田明郷}}（香港配音：[[盧國權|盧國權]]）
:常務。
*的場英喜：{{link-ja|中丸新將|中丸新将}}（香港配音：[[朱子聰|朱子聰]]）
:社長。
*橘美里：{{link-ja|菜葉菜|菜葉菜}}（香港配音：[[魏惠娥|魏惠娥]]）
:人事部職員。
*山田：{{link-ja|小松和重|小松和重}}
:人事部職員。

===Jewel D珠寶店===
*中原綾子：[[山本未來|山本未來]]（香港配音：[[黃玉娟|黃玉娟]]）
:Jewel D外商部一課課長。
*真壁雄斗：{{link-ja|渡邊大|渡辺大}}（香港配音：[[梁偉德|梁偉德]]）
:Jewel D外商部一課員工。富川栞的男友。

=== 其他 ===
*國原耕太：[[新井浩文|新井浩文]]（香港配音：[[潘文柏|潘文柏]]）
:國原求職訓練班社長。以花言巧語引誘為了求職而煩惱的大學生，使大學生支付高昂費用求職。
*戶川加奈：{{link-ja|中川知香|中川知香}}（香港配音：[[何晶晶|何晶晶]]）
:求職中的女大學生。和富川光一起在國原求職訓練班上課。
*原口亨：{{link-ja|矢島健一|矢島健一}}（香港配音：[[馮志輝|馮志輝]]）
:深澤學園校長。
*池上孝之：{{link-ja|卯ノ原圭吾|卯ノ原圭吾}}
:求職中的大學生。
*夏野久美：{{link-ja|木村綠子|キムラ緑子}}（香港配音：[[袁淑珍|袁淑珍]]）
:富川洋輔的高中同班同學。小型西式菓子店的老闆。
*天谷五郎：{{link-ja|段田安則|段田安則}}（香港配音：[[招世亮|招世亮]]）
:被公司開除，但沒能對家人說出口，在公園流連的男人。其後在公園認識了富川洋輔，兩人成為好友。
