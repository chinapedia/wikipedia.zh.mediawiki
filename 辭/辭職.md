'''辭職'''是在[[職業|職業]]上一個員工辭別自己原先的[[工作|工作]]崗位及[[组织_(社会学)|組織]]，讓出其職位予新接任人。与之词意相近的是'''离职'''、'''退职'''，但不包含[[退休|退休]]。

員工離職時，要先與下一位接手之人指導其崗位職務上需要被交接之事，之後向人事部門歸還公司所屬物品，比如[[職員證|職員證]]、門禁卡、公司[[鎖匙|鎖匙]]、公務車等。一旦完成離職手續，[[員工福利|員工福利]]會立即停止，[[薪水|薪水]]則發還最後一次（通常是下個月），便不再給付，換來則是收取[[代通知金|代通知金]]、[[長俸|長俸]]、[[強積金|強積金]]等。由于员工自动辞职或被开除，相比[[解雇|解雇]]，公司可节省支出。有些公司会突然改变工作地点，以达到强迫员工辞职或开除的目的<ref>{{Cite web |url=http://www.sohu.com/a/207936099_653357 |title=公司能调动员工工作地点吗？/人力资源法律 |accessdate=2018-01-03 |author= |date=2017-12-01 |work= |publisher=搜狐网|language=Zh-hans }}</ref>。

== 原因 ==

* 私人理由，申請通過後，便可離職。其理由很多，視個人狀況而論，例如非人為造成不可抗拒之因素（[[疾病|疾病]]、意外傷害造成工作能力喪失及死亡）、家人反對、全職[[學習|深造]]、顧家（家庭主婦、家庭主夫）、自己[[創業|創業]]、跳槽、[[移民|移民]]等。
* 组织理由，譬如被公司要求解約、[[解僱|解僱]]的情況。亦或因公司搬迁，工作地点变动而辞职。或依公司或政府法規之規定，到达一年龄后，不能再从事某些工作。更为普遍的是屆齡則強制[[退休|退休]]。
* [[問責制|問責制]]
** 在民意和政黨的巨大壓力下，政府首長在無選擇下被迫下台，如1974年的[[美國總統|美國總統]][[理查·尼克森|理查·尼克森]]，他因[[水門事件|水門事件]]政治醜聞在[[美國國會|國會]][[彈劾|彈劾]]的威脅下被迫辭職。
* [[僱傭|僱傭]][[合約|合約]]期完成，簡稱「完約」。
