'''娜芙蒂蒂胸像'''是一尊約有{{#expr:{{CURRENTYEAR}}+1345}}年歷史，以石灰岩與灰泥雕塑成的彩繪人像。這座雕像所刻的是[[古埃及|古埃及]][[法老|法老]][[阿肯那頓|阿肯那頓]]的大王后（{{lang|en|the Great Royal Wife}}）[[娜芙蒂蒂|娜芙蒂蒂]]。因為這座雕像的複製品是最廣為流傳的古埃及藝術品之一，娜芙蒂蒂也成為最有名的古代女性之一，同時也是女性美的一種指標。一般认为這尊雕像是在公元前1345年由雕刻家{{le|圖特摩斯|Thutmose (sculptor)}}（{{lang|en|Thutmose}}）所雕。

1912年時，一支由德國[[考古學|考古學]]家[[路德維希·波爾哈特|路德維希·波爾哈特]]（{{lang|de|Ludwig Borchardt}}）率領的德國考古隊在位於[[埃及|埃及]]阿瑪納（{{lang|en|Amarna}}）的圖特摩斯工作室發現了娜芙蒂蒂的半身像。這座雕像在被發現後曾被收藏於德國數個不同的地點，這其中包括位於馬克斯·基瑟巴赫（{{lang|de|Merkers-Kieselbach}}）市的鹽礦坑、[[柏林|柏林]]的達雷姆博物館（{{lang|de|Museum Dahlem}}）、[[夏洛滕堡|夏洛滕堡]]（{{lang|de|Charlottenburg}}）的埃及博物館（{{lang|de|Ägyptisches Museum}}）和[[柏林博物館島|柏林博物館島]]上的[[柏林舊博物館|老博物館]]。娜芙蒂蒂像目前藏於博物館島上的[[柏林新博物馆|新博物館]]（{{lang|de|Neues Museum}}），[[第二次世界大戰|二次世界大戰]]之前這座雕像正是在此地展出。

娜芙蒂蒂胸像已成為柏林、德國與古埃及的文化象徵。埃及要求德國歸還胸像亦一直是兩國之間激烈爭論的主題。一個由匈牙利藝術家所籌辦，名為「娜芙蒂蒂之軀青銅像」（The Body of Nerfertiti）的藝術展览，娜芙蒂蒂胸像本身的真偽亦令它捲入爭議之中。

== 歷史 ==
=== 娜芙蒂蒂的背景 ===
[[File:Akhenaten,_Nefertiti_and_their_children.jpg|thumb]]
娜芙蒂蒂（意即美麗的人來了）是公元前十四世紀（古埃及第十八王朝）法老[[阿肯那頓|阿肯那頓]]的王后。雖然阿肯那頓發起了在古埃及史上極重要的[[宗教|宗教]]改革：僅崇拜太陽神[[阿頓|阿頓]]的一神信仰運動，但娜芙蒂蒂的生平卻沒有留下多少相關記載。有些理論認為她可能出身於埃及皇室、也有可能是外族的公主，或是當時一個高級政府官員，後來在[[圖坦卡門|圖坦卡門]]過世後成為法老的艾（Ay）的女兒。娜芙蒂蒂在當時也許曾與阿肯那頓（在位期為公元前1352年到1336年）一起治理過埃及。<ref name = "who"/>而阿肯納頓與娜芙蒂蒂生下六個女兒，其中一個名叫安卡蘇納蒙（Ankhesenamen）的後來嫁給了娜芙蒂蒂的繼子，[[圖坦卡門|圖坦卡門]]為妻。阿肯納頓在朝的第十二年起便不再有关于娜芙蒂蒂本人的歷史紀錄，不過這究竟代表她是換了個新名字，還是從那時起已不在人世便不得而知。她也有可能在其丈夫過世後憑藉自己的身分繼任為[[法老|法老]]，並統治埃及一小段時間。<ref name = "who"/><ref name="nef">Silverman, Wegner, Wegner pp.130-33</ref> 

娜芙蒂蒂的半身像被認為是雕刻家圖特摩斯於公元前1345年所雕。<ref name="who">{{cite web|url=http://dsc.discovery.com/convergence/nefertiti/history/history.html |title=Who Was Nefertiti? |last=Maryalice Yakutchik |publisher=[[探索频道|探索频道]] |accessdate=2009-11-23 |deadurl=yes |archiveurl=https://web.archive.org/web/20100105162450/http://dsc.discovery.com/convergence/nefertiti/history/history.html |archivedate=5 一月 2010 }}</ref><ref name="natgeo2"/>这尊雕像上面并没有娜芙蒂蒂的名字，不过它所佩戴的[[皇冠|王冠]]同娜芙蒂蒂的相同。<ref name="fordummes">{{cite book|last=Charlotte Booth|title=The Ancient Egyptians for Dummies|publisher=for Dummies}}</ref>

=== 雕像的發現 ===
[[File:Nefertiti_berlin.jpg|thumb]]
1912年12月6日，由考古學家路德維希·波爾哈特所率領的[[德國東方學會|德國東方學會]]在阿瑪納發現了娜芙蒂蒂像。在圖特摩斯的工作室遺址内发现了這座雕像與其他一些以娜芙蒂蒂為主題的未完成半身。<ref name = "b285"/><ref name = "s115">Siehr p.115</ref>波爾哈特的[[日記|日記]]提供了這次發掘的主要文字紀錄，他在日誌中這麼寫道：「最栩栩如生的埃及藝術品就這麼突然出現在我們手裡，你簡直無法用文字形容它。你必須親眼見識這尊雕像。」<ref name="guard"> {{cite news|url=http://www.guardian.co.uk/artanddesign/2009/may/07/nefertiti-bust-berlin-egypt-authenticity|title=Is this Nefertiti – or a 100-year-old fake?|last=Kate Connolly |work=The Guardian|accessdate=2009-11-21|date = 2009-05-07}}</ref>

德國東方學會[[檔案|檔案]]處一份標示日期為1924年的文件描述了1913年1月20日時，波爾哈特與埃及一名資深[[官員|官員]]會面討論德國與埃及該如何分配1912年的發現成果。根據這份文件的作者，也就是德國東方學會當時在場的[[秘書|秘書]]的說法，波爾哈特“想要為我們留下娜芙蒂蒂像”。<ref name="nyt"/><ref name="SPIEGEL">{{cite news|url=http://www.spiegel.de/international/world/0,1518,606525,00.html|title=Archaeological Controversy: Did Germany Cheat to Get Bust of Nefertiti?|date=02/10/2009|publisher=Der Spiegel|accessdate=2009-11-17}}</ref> 

人们怀疑波爾哈特刻意隱瞞了這座雕像的價值，<ref name="times"/>不過他否認曾經這麼做。<ref name = "b288">Berger p. 288</ref> 

雖說德國[[考古學家|考古學家]][[菲利普·范德伯格|菲利普·范德伯格]](Philipp Vandenberg)以“極富冒險精神且無人能出其右”来形容德國的這次大收穫；但美國的時代雜誌卻將其列為十大被掠奪的藝術品之一。<ref name="b286">Breger p. 286</ref> <ref name = "TIME">{{cite web|url=http://www.time.com/time/specials/packages/article/0,28804,1883142_1883129_1883119,00.html|title=Top 10 Plundered Artifacts|year=2009|work=[[时代杂志|时代杂志]]|accessdate=2009-11-24}}</ref>波爾哈特對埃及官員出示了一張“未將納菲蒂蒂像最好的一面呈現出來”的[[照片|照片]]；這座雕像在埃及古文物審查首長古斯塔夫·樂菲伯（Gustave Lefebvre）前來檢查時被嚴密地包起，收在一個盒子裡。相關文件揭露，波爾哈特将納菲蒂蒂像说成是石膏製成的，从而誤導了審查員；但德國東方學會把這個事件歸咎於審查員的疏忽，並指出納菲蒂蒂像是物品交換[[清單|清單]]上的第一項物品，聲稱這整件[[交易|交易]]是公平的。<ref name="SPIEGEL"/><ref name = "mons"/>

=== 在德國時的收藏地點 ===
[[File:Neues_Museum_Berlin_EP1.JPG|thumb]], 柏林是娜芙蒂蒂像目前的所在之處]]

納菲蒂蒂像自從1913年被運往柏林<ref name="nyt"/>，然后獻給批發商，同時也是阿瑪納發掘企劃的贊助人，詹姆士·西蒙（Henri James Simon）後便藏於德國<ref name = "s115"/>。納菲蒂蒂像一直是西蒙家宅的展示品，直到1913年西蒙將雕像與其他一些從阿瑪納發現的藝術品於一同出借給柏林博物館。<ref name="chrono">{{cite web|url=http://www.nofretete-geht-auf-reisen.de/echronol.htm |title=The Bust of Nefertiti: A Chronology |year=2007 |work="Nefertiti travels" campaign website |publisher=CulturCooperation |accessdate=2009-11-22 |deadurl=yes |archiveurl=https://web.archive.org/web/20150318133459/http://www.nofretete-geht-auf-reisen.de/echronol.htm |archivedate=18 三月 2015 }}</ref> 

雖然其餘的阿瑪納藝術品在1913到1914年間进行了公開展出，但在波爾哈特的要求下却将娜芙蒂蒂像保密了。<ref name="b286" /> 1918年時，柏林博物館考慮過將雕像展出，但卻再次因為波爾哈特的要求而排除了公佈雕像的考虑。納菲蒂蒂像在1920年時被永久地捐給了柏林博物館，波爾哈特在1923年時發表的一篇文章向大眾揭曉了娜芙蒂蒂像的存在。1924年，納菲蒂蒂像終於以柏林的埃及博物館館藏的一部分的名義公開展出。<ref name = "b286"/><ref name="chrono"/>在1939年柏林博物館島的[[柏林新博物館|新博物館]]關閉之前，這座雕像一直在該處展出；而隨著[[二次世界大戰|二次世界大戰]]的爆發，柏林的博物館清空了所有的藝術品，將它们全部運往安全的場所。<ref name = "s115"/>娜芙蒂蒂像起初被藏在普魯士政府銀行的地窖裡，然後在1941年秋天被移往柏林的一個[[高射炮|高射炮]]塔。<ref name="chrono"/>[[柏林新博物館|新博物館]]在1943年時遭受到[[英國皇家空軍|英國皇家空軍]]的轟炸。<ref name = "inde"/>1945年3月6日，這座半身像被移到德國[[图林根|圖林根聯邦州]]的馬克斯·基瑟巴赫（Merkers-Kieselbach）市的鹽礦坑。<ref name = "s115"/> 

1945年3月，美軍發現了這座胸像，並將之轉交給自己的藝術品支部（Monuments, Fine Arts and Archives branch）。之後娜芙蒂蒂像被移往位於[[法蘭克福|法蘭克福]]的[[德意志帝國銀行|德意志帝國銀行]]，在八月的時候，雕像又被運至位於德國[[威斯巴登|威斯巴登]]的美國中央收藏據點（U.S. Central Collecting Point），然後在那公開展覽到1946年。<ref name = "s115"/><ref name="chrono"/>1956年時，娜芙蒂蒂像被送回[[西柏林|西柏林]]，並於達雷姆博物館展出。<ref name = "s115"/>

早在1946年，東德就堅持要將娜芙蒂蒂像送回它戰前的展覽處：位於東柏林的[[博物館島|博物館島]]。<ref name = "s115"/><ref name="chrono"/>1967年，雕像被移到夏洛登堡的埃及博物館，直到2005年時才被移送到老博物館。<ref name="chrono"/>最後，娜芙蒂蒂像終於在2009年10月，於[[柏林新博物館|新博物館]]重新開幕時再度以該館最受矚目的展品的身份回到了柏林新博物館。<ref name="times"/><ref name = "inde">{{cite news|url=http://www.independent.co.uk/news/world/europe/queen-nefertiti-rules-again-in-berlins-reborn-museum-1804370.html|title= Queen Nefertiti rules again in Berlin's reborn museum|last=Tony Paterson|date=2009-10-17|work=The Independent|accessdate=2009-11-15}}</ref><ref>{{cite news|url=http://features.csmonitor.com/globalnews/2009/11/02/germany-time-for-egypts-nefertiti-bust-to-go-home/|title= Germany: Time for Egypt’s Nefertiti bust to go home? |last=Isabelle de Pommereau|date=2-November-09|work=The Chrisitian Science Monitor|accessdate=2009-11-15}}</ref>

=== 埃及方提出的送還雕像的要求 ===
自打[[德國|德國]]1924年正式向世界揭露娜芙蒂蒂像的存在開始，[[埃及|埃及]]官方便從未停止過將雕像送回埃及的要求。<ref name = "b285"> Breger p. 285</ref><ref name="chrono"/><ref name = "s116">Sieher p. 116</ref>為此埃及在1925年的時候威脅若不把娜芙蒂蒂像歸還，便要禁止德國在埃及的所有考古發掘活動。1929年，埃及政府提出了用其他藝術品與德國交換娜芙蒂蒂像的提案，不過被德國拒絕了。而1950年時，埃及試著再次發起協商，但是德國並未對此作出任何回應。<ref name="chrono"/><ref name = "s116"/>雖然德國之前曾經強烈反對歸還雕像，但1933年時納粹的空軍元帥，[[赫尔曼·戈林|赫曼·歌林]]曾經考慮過要為了政治上的利益而把雕像送還給當時的埃及國王[[法魯克一世|法魯克一世]]。不過[[希特勒|希特勒]]反對這項提案，並且還跟埃及政府表示他會為這尊雕像蓋一棟新的埃及博物館，他說：「而就在這座博物館的中央，娜芙蒂蒂這個奇蹟會被送上王座……而我永遠不會出讓這皇后的頭像。」<ref name="times"/><ref name = "s116"/>當娜芙蒂蒂像在[[美國|美國]]人手中的時候，[[埃及|埃及]]也向[[美國|美國]]提出了交還雕像的要求；不過[[美國|美國]]拒絕了，還建議埃及應與德國的新政權進行交涉。<ref name="chrono"/>1989年，埃及[[總統|總統]][[穆罕默德·胡斯尼·穆巴拉克|穆罕默德·胡斯尼·穆巴拉克]]在德國參觀了娜芙蒂蒂像的展出，並在柏林宣佈：「這座雕像是埃及最棒的大使。」<ref name="chrono"/> 

埃及文化部分支，古蹟最高委員會（Supreme Council of Antiquities）的書記長─—[[札希·哈瓦斯|札希·哈瓦斯]]博士相信娜芙蒂蒂像的所有權屬於埃及，而既然這座雕像當初是以非法的手段被運出埃及的，那麼便應當被送還給該國政府。[[札希·哈瓦斯|哈瓦斯]]博士主張：早年[[埃及|埃及]]相關單位對於娜芙蒂蒂像當初是如何被運往[[德國|德國]]這件事可能有些误会，所以才沒有提出異議。他要求德國给出娜芙蒂蒂像是被合法運出埃及的證明。<ref name="nyt"/><ref name="nyt1">{{cite news|url=http://www.nytimes.com/2009/10/24/arts/design/24abroad.html?_r=1&hpw|title=When Ancient Artifacts Become Political Pawns|last=MICHAEL KIMMELMAN|date=2009-10-23|work=New York Times|accessdate=2009-11-15}}</ref>另外根據蘇黎世大學的寇特·希爾（Kurt G. Siehr）的看法，娜芙蒂蒂像的所有權之所以屬於埃及還有另一個理由：「考古文物的家在於其文化起源的國度，也應當被保存在那裡。」<ref>Siehr pp. 133–4</ref>是否送還雕像的議題在2003年時因為「娜芙蒂蒂之軀」這座雕像又再次興起了波瀾（見本文[[#爭議|爭議]]部分）。哈瓦斯博士於2005年提出，希望[[聯合國教育、科學及文化組織|聯合國文教組織]]介入協商以讓德國歸還雕像。<ref>{{cite journal|last=Nevine El-Aref |date=14–20 July 2005 |title=Antiquities wish list |journal=Al-Ahram Weekly |issue=751 |url=http://weekly.ahram.org.eg/2005/751/eg7.htm |deadurl=yes |archiveurl=https://web.archive.org/web/20100916194918/http://weekly.ahram.org.eg/2005/751/eg7.htm |archivedate=16 九月 2010 }}</ref>

哈瓦斯在2007年時威脅，若德國不願意將娜芙蒂蒂像出借給埃及，那麼便要禁止埃及藝術品在德國的展覽，不過此事不了了之。哈瓦斯也曾經要求世界各國杯葛德國博物館做出的出借展品的要求，以發起他所謂的“科學之戰”。哈瓦斯希望德國至少能夠為了將在2012年於[[吉薩|吉薩]][[吉薩金字塔群|大金字塔群]]附近開幕的大埃及博物館（Grand Egyptian Museum）而把雕像借給埃及。<ref name = "natgeo">{{cite news|url=http://news.nationalgeographic.com/news/2007/04/070418-nefertiti-egypt.html|title=Egypt Vows "Scientific War" If Germany Doesn't Loan Nefertiti|last=Dan Morrison|date=2007-04-18|work=National Geographic News|publisher=[[美国国家地理学会|美国国家地理学会]]|accessdate=2009-11-15}}</ref>同時，一個以德國[[漢堡|漢堡]]為根據地的文化團體─—文化合作（Cultur Cooperation）發起了「娜芙蒂蒂之旅」（Nefertiti Travels）的運動，四處發送了印有娜芙蒂蒂像及「原件歸還」字眼的明信片；該團體並向德國文化部長波恩德‧紐曼（Bernd Neumann）遞交了支持將雕像出借給埃及的公開信。<ref name = "bbc"/><ref>{{cite web|url=http://www.nofretete-geht-auf-reisen.de/ewelcome.htm |title="Nefertiti travels" campaign website |year=2007 |publisher=CulturCooperation |accessdate=2009-11-22 |deadurl=yes |archiveurl=https://web.archive.org/web/20150518042558/http://www.nofretete-geht-auf-reisen.de/ewelcome.htm |archivedate=18 五月 2015 }}</ref>當娜芙蒂蒂在2009年被移回她在[[柏林|柏林]][[柏林新博物館|新博物館]]的舊家時，將雕像選擇放在[[柏林|柏林]]的举动是否合适也曾遭受质疑。

數名德國藝術專家曾經試著反駁哈瓦斯的主張；他們指出德國東方學會那份1924年的文件討論的正是波爾哈特與埃及官員間立下的協定<ref name="nyt"/><ref name=autogenerated1>{{cite news|url=http://www.spiegel.de/international/world/0,1518,606525,00.html|title=Archaeological Controversy: Did Germany Cheat to Get Bust of Nefertiti?|date=02/10/2009|publisher=SPIEGEL|accessdate=2009-11-17}}</ref>。但正如前文所說，波爾哈特在這整件事裡有蓄意欺騙對方的嫌疑。德國官方則主張娜芙蒂蒂像太過脆弱，並不適合長途運輸；且與歸還雕像相關的法律論證立論薄弱。根據英國[[泰晤士報|泰晤士報]]的報導，德國可能是在擔心將雕像出借給埃及意味著娜芙蒂蒂像將永遠離開德國。<ref name="times">{{cite news|url=http://www.timesonline.co.uk/tol/news/world/africa/article6881619.ece|title=Neues Museum refuses to return the bust of Queen Nefertiti to Egyptian museum|last=Roger Boyes|date=2009-10-20 |work=时代杂志|accessdate=2009-11-15}}</ref><ref name = "natgeo"/>

在2009年12月，{{le|柏林埃及博物館|Egyptian Museum of Berlin}}（Berlin's Egyptian Museum）主管芙德列克·席菲德（Friederike Seyfried）展示由該館持有的文件，这份文件是有關於娜芙蒂蒂胸像的發現，內容包括一份由德國考古人員與埃及古文物部門（Egyptian Antiquities Service）所簽訂的協議。在這份文書裡，這尊雕像是以某位公主的彩繪石膏像的名義記載；但在波爾哈特的日記裡，他明確地稱之為「娜芙蒂蒂頭像」（the head of Nefertiti）。哈瓦斯指對此發表了以下意見：「這證明了波爾哈特是故意這麼寫，好讓自己國家得到這尊雕像。這些資料證實了埃及的論點——波爾哈特存心欺瞞，違反專業操守。」哈瓦斯的聲明引用了席菲德的說法，認為決定是否將雕像歸還，乃繫於普魯士文化遺產基金會（Prussian Cultural Heritage）和德國文化部長的意向。<ref name=autogenerated2>[http://www.boston.com/news/world/middleeast/articles/2009/12/20/egypt_antiquities_chief_to_demand_nefertiti_bust/ The Associated Press:Egypt antiquities chief to demand Nefertiti bust]</ref><ref name=autogenerated2 />

== 雕像的描述與檢驗 ==
{{double image|right|Nefertiti bust (right).jpg|140|Nefertiti_(Nofretete_in_Berlin).jpg|125|右側與正面照}} 
{{double image|right|Nefertiti bust atles.jpg|140|Nefertiti bust (back).jpg|125|左側與背面照}}
娜芙蒂蒂像約有47公分高，重約20公斤。整座雕像是由[[石灰岩|石灰岩]]為核心，表層則覆上彩繪的灰泥。雕像的臉孔近乎於完全地對稱無瑕，不過左眼卻缺少了右眼所有的鑲嵌物。<ref>{{cite book|last=Horst Woldemar Janson, Anthony F. Janson|title=History of art: the Western tradition}}</ref><ref name = "silver"/>右眼的瞳孔是一塊由[[蜂蠟|蜂蠟]]固定并塗有黑色顏料的[[石英|石英]]，眼窩則是未經修飾的石灰岩。娜芙蒂蒂戴著一頂現在被稱為「娜芙蒂蒂帽冠」的藍色王冠，冠上飾有一條代表皇權的金色帶子。這條飾帶像條緞帶一樣水平的環繞著王冠，在腦後打結；一條現已毀損的聖蛇（Uraeus）則裝飾在她眉毛上方。脖子上圍著有[[花朵|花朵]]圖樣的寬領裝飾<ref name = "Schultz">{{cite book|last=Schultz|title=Egypt the World of Pharaohs: The World of the Pharaohs|publisher=American Univ in Cairo Press. |pages=203}}</ref>，雕像的[[耳朵|耳朵]]有些受損。<ref name = "silver"/>藝術教科書「古今藝術」（Gardner's Art Through the Ages）對娜芙蒂蒂像有著這樣的聯想：圖特摩斯之所以誇張化了纖長如蛇般的頸項長度和戴著王冠的頭部重量，也許是有把這尊優雅的半身像類比為纖細花莖上碩大花朵的意圖。<ref name = "gard">{{cite book|last=Helen Gardner|title=Gardner's Art Through the Ages: the western perspective|pages=64|chapter=Art of Ancient Egypt}}</ref>

埃及古物學家大衛席維曼（David P. Silverman）認為娜芙蒂蒂像反映了古典的古埃及藝術風格，與[[阿肯那頓|阿肯那頓]]統治時期出現的那種線條較柔和，也較有律動感的阿瑪納風格不盡相同。這座雕像的實際功用是什麼並不清楚，但有些理論認為它有可能是雕刻家參考用的模型。<ref name="Ancient Egypt">{{cite book|last=Silverman|first=David P.|title=Ancient Egypt|publisher=[[牛津大学|牛津大学]]出版社|location=USA|year=1997|page=221|isbn=0-19-521952-X}}</ref>

=== 使用的顏料 ===
路德維希·波爾哈特當年送出了一份對雕像頭部所使用的顏料成分做化學分析的委託，<ref>{{cite book|last=Rudolf Anthes|title=Nofretete – The Head of Queen Nofretete|publisher=Verlag Gebr|location=Mann, Berlin |year=1961|pages=6}}</ref>
這份檢驗結果被附在1923年出版的《娜芙蒂蒂王后的肖像》(Portrait of Queen Nofretete)這本書裡：
* 藍色：添加了[[氧化銅|氧化銅]]的玻砂
* 膚色(淺紅)：磨得極細的石灰岩與作為染料的[[鐵氧化物|氧化鐵]]
* 黃色：雌黃(三硫化二砷)
* 綠色：加了[[銅|銅]]與[[氧化鐵|氧化鐵]]的玻砂
* 黑色：煤(加了蠟做為接著劑)
* 白色：[[白堊岩|白堊]] ([[碳酸鈣|碳酸鈣]])

=== 消失的左眼 ===
當這座半身像首次被發現時，波爾哈特原以為做為左眼瞳孔的石英在圖特摩斯的工房垮掉時從雕像上掉了下來；不過之後他們在瓦礫中做了一次徹底的搜索，卻依然沒有找到雕像的左眼。<ref>Joyce A. Tyldesley, ''Nefertiti: Egypt's sun queen'', Viking, 1999, p.196.</ref>這顆失落的眼睛讓人們作出假设：娜芙蒂蒂的眼球可能曾經受到感染，並且的確失去了一隻眼睛。不過娜芙蒂蒂的其他塑像上健全的雙眼與這個假說相矛盾。<ref>Fred Gladstone Bratton, ''A history of Egyptian archaeology'', Hale, 1968, p.223</ref> 

德國埃及古文物學家迪崔克·維爾東（Dietrich Wildung）認為這座目前在柏林的半身像，其實是用來作為娜芙蒂蒂官方肖像的模特兒，同時也被雕刻師傅用來教導他的學徒如何雕出眼睛的內部構造，因此左眼並沒有被放上去。<ref name="disc1">{{cite news | title = Scholar: Nefertiti Was an Aging Beauty | url = http://dsc.discovery.com/news/2006/09/05/nefertiti_arc.html | last = Lorenzi | first = R | date = 2006-09-05 | work = Discovery News | publisher =探索频道 | accessdate = 2009-12-18|pages = 1–2}}</ref>古今藝術和席維曼對此則抱持了類似的看法，同樣地認為這座雕像是刻意被保持在未完工的狀態。<ref name = "silver">Silverman, Wegner, Wegner pp. 21, 113</ref><ref name = "gard"/> 哈瓦斯的看法則是圖特摩斯的確製造了左眼，只是左眼之後毀壞了而已。<ref name = "digital"/>

=== 電腦斷層掃描的結果 ===
<!-- Nefertiti_Bust_CT.gif为合理使用：见http://zh.wikipedia.org/wiki/File:Nefertiti_Bust_CT.gif以获得更多信息 -->
娜芙蒂蒂像在1992年時首次接受了[[電腦斷層掃描|電腦斷層掃描]]，該次掃描每五公厘便拍攝一張斷層影像。<ref name="dsc"/><ref name="pdf">For comparative analysis between 1992 and 2006 CT scans: {{cite journal|last=Bernhard Illerhaus, Andreas Staude, Dietmar Meinel|title=Nondestructive Insights into Composition of the Sculpture of Egyptian Queen Nefertiti with CT and the dependence of object surface from image processing|url=http://www.ndt.net/article/ndtnet/2009/illerhaus.pdf|publisher = NDT Database & e-Journal of Nondestructive Testing|year = 2009}}</ref> 2006年，當時作為柏林埃及博物館主管的迪崔克·維爾東（Dietrich Wildung）在為當時於老博物館展出的娜芙蒂蒂像調整打光時發現了雕像脖子上的皺紋及眼袋，這代表雕刻家曾試著描繪出娜芙蒂蒂老化的情況。新一次的[[電腦斷層掃描|電腦斷層掃描]]證實了維爾東的發現：圖特摩斯在雕像的臉頰和眼部加上了石膏，好讓雕像看起來更完美。<ref name="disc1"/> 

2006年那次由柏林的造影科學研究所所長─[[亞歷山大·胡勃茲|亞歷山大·胡勃茲]]（Alexander Huppertz）所領導的電腦斷層掃描，揭露出了雕像內核上那張由雕刻家所雕，滿面皺紋的娜芙蒂蒂。<ref name="pdf"/>這次的掃描結果被發表在2009年四月份的放射學期刊上（Radiology）。<ref>{{cite journal|last=Alexander Huppertz  ,|coauthors=Dietrich Wildung, Barry J. Kemp, Tanja Nentwig, Patrick Asbach;  Franz Maximilian Rosche, Bernd Hamm|date=April 2009|title=Nondestructive Insights into Composition of the Sculpture of Egyptian Queen Nefertiti with CT|journal=Radiology|volume=251|issue=1|pages=233–240.|oclc=10.1148/radiol.2511081175|url=http://radiology.rsna.org/content/251/1/233.abstract|publisher=北美放射学会（Radiological Society of North America）}}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>
2006年的掃描顯示圖特摩斯在石灰岩所雕的內核上加上了許多層厚薄不一的灰泥，內部的雕像臉上在唇部與臉頰上有皺紋，鼻子上還有處發腫的痕跡。這些皺紋和鼻子上的腫塊都被最外層的灰泥給撫平了。根據胡勃茲的說法，這加工可能反映了「當代審美觀中的理想容貌」。<ref name="natgeo2">{{cite web|url=http://news.nationalgeographic.com/news/2009/03/090331-nefertiti-bust-picture.html|title=Nefertiti's Real, Wrinkled Face Found in Famous Bust?|last=Christine Dell'Amore|date=2009-03-30|publisher=国家地理学会|accessdate=2009-11-16}}</ref><ref>{{cite news|url=http://www.sciencedaily.com/releases/2009/03/090331091246.htm|title=Hidden Face In Nefertiti Bust Examined With CT Scan|date=2009-04-08|work=Science Daily|accessdate=2009-11-23}}</ref>相較起1992年的結果，2006年的掃描在某些地方，如灰泥下方一到二公厘的細微之處提供了更多的細節。<ref name="dsc">{{cite news|url=http://dsc.discovery.com/news/2009/03/31/nefertiti-face-bust.html|title=Nefertiti Bust Has Two Faces|last=Patrick McGroarty|work=Discovery News|publisher=探索频道|accessdate=2009-11-17|date=2009-03-31|pages = 1–2}}</ref>

== 爭議 ==
=== 娜芙蒂蒂之軀 ===
柏林的埃及博物館在2003年的時候批准了由安德拉斯·蓋利克（Andras Galik）和貝林特·哈華斯（Balint Havas）組成的匈牙利雙人藝術組合「{{link-en|小華沙|Little Warsaw}}」（Little Warsaw）為了完成要在[[威尼斯雙年展|威尼斯雙年展]]展出的裝置藝術錄影，而將娜芙蒂蒂像擺在一座近乎於全裸的女性青銅像上方的要求。根據這兩位藝術家的說法，這個被稱為“娜芙蒂蒂之軀”是小華沙向娜芙蒂蒂半身像致敬的一個企劃。維爾東認為這個設計呈現了遠古世界與現代藝術間持續的關聯。但是埃及的文化官員認為這是種侮辱，並宣稱這個企劃污衊了該國偉大的歷史象徵之一。因此，埃及頒下禁令，禁止讓維爾東及其妻未來在埃及進行任何考古活動。<ref name = "natgeo"/><ref name="nyt2">{{cite news|url=http://www.nytimes.com/2003/06/21/arts/nefertiti-s-bust-gets-a-body-offending-egyptians.html|title=Nefertiti's Bust Gets a Body, Offending Egyptians|last=HUGH EAKIN|date=2003-06-21|work=[[The_New_York_Times|The New York Times]]|accessdate=2009-11-21}}</ref><ref>For a picture of "The Body of Nerfertiti" see {{cite web|url=https://pantherfile.uwm.edu/wash/www/nefertiti.htm|title=Nefertiti's Bust Gets a Body, Offending Egyptians: A Problematic Juxtaposition|date=2003-06-21|work=纽约时报|accessdate=2009-11-23|deadurl=yes|archiveurl=https://web.archive.org/web/20111008194009/https://pantherfile.uwm.edu/wash/www/nefertiti.htm|archivedate=2011年10月8日}}</ref> 埃及文化部部長法洛克‧侯斯尼（Farouk Hosny）對此也發表了認為娜芙蒂蒂像並沒有被好好保管的聲明；雖然埃及因為與「兩國間的友好關係」而並未重申歸還雕像的要求，他們還是認為這個舉動讓人無法接受。<ref name="chrono"/>

=== 與雕像真偽有關的各式說法 ===
由瑞士藝術史學家{{le|亨利·史提林|Henri Stierlin}}（Henri Stierlin）所寫的法語書籍《Le Buste de Nefertiti – une Imposture de l'Egyptologie?》（娜芙蒂蒂像——埃及學上的一大詐欺？），而由柏林出身的歷史學家[[伊卓根·俄西凡|伊卓根·俄西凡]]（Erdogan Ercivan）所寫的「考古學中的失落環節」（''Missing Link in Archaeology'')同樣地主張了娜芙蒂蒂像是個由現代人製造出來的贗品這個看法。史提林宣稱波爾哈特可能根本就是為了測試古代顏料才創作出娜芙蒂蒂胸像；所以在當時的[[普魯士|普魯士]]王子[[約翰·格奧爾格一世|約翰·喬治]]（Johann Georg）大為讚賞雕像時，為了避免冒犯到王子的波爾哈特便將錯就錯地把這座雕像說成了真品。史提林認為，雕像所缺少的左眼在古埃及是會被視為不敬的，而且在據說雕像被發現的1913年到正式發布其存在的1924年間，整整11年沒有任何與娜芙蒂蒂像有關的科學記錄。而且，雖然表層上的顏料的確相當古老，但卻從來沒有鑑定過石灰岩核心的年份。俄西凡提出了波爾哈特的妻子才是雕像模特兒的理論。這兩個作者都認為娜芙蒂蒂像之所以到1924年才面世，正是因為它是個贗品的緣故。<ref name="guard"/>其他派說法則認為，現存的娜芙蒂蒂像其實是1930年代時希特勒下令製造的；原件則在二次世界大戰時不知所蹤。<ref name="mons">{{cite news|url=http://www.monstersandcritics.com/science/news/article_1473370.php/Nefertiti_s_hidden_face_proves_Berlin_bust_is_not_Hitler_s_fake |title=Nefertiti's 'hidden face' proves Berlin bust is not Hitler's fake |date=2009-04-27 |work=Science News |accessdate=2009-11-23 |deadurl=yes |archiveurl=https://web.archive.org/web/20120704110319/http://www.monstersandcritics.com/science/news/article_1473370.php/Nefertiti_s_hidden_face_proves_Berlin_bust_is_not_Hitler_s_fake |archivedate=4 七月 2012 }} For pictures, {{cite news|url=http://www.antinopolis.org/hernestus/archaeologyheadlines.html |title=NEFERTITI'S "HIDDEN FACE" PROVES FAMOUS BERLIN BUST IS NOT HITLER'S FAKE |date=2009-04-03 |accessdate=2009-11-23 }}</ref>

埃及博物館主管迪特崔克·維爾東（Dietrich Wildung）認為這些說法都只不過是譁眾取寵的手段，因為不管是放射線測試、詳細的[[電腦斷層掃描|電腦斷層掃描]]，還是用料分析都已經證明了這座雕像是真的。<ref name="guard"/>娜芙蒂蒂像上所使用的顏料與其他古埃及藝術家所使用的相吻合；另外，根據美國科學半月刊「{{le|科學新聞|Science News}}」(Science News)，2006年那次發現隱藏在灰泥底下的娜芙蒂蒂真實面貌的[[電腦斷層掃描|電腦斷層掃描]]已經毫無疑問地證明了雕像不是偽造的。<ref name="mons"/>

埃及相關單位同樣地駁斥了史提林的假說。哈瓦斯博士這麼說：「史提林並不是一個歷史學家，他根本就語無倫次。」縱使史提林以「古埃及人會水平地雕刻肩膀」這個論點來指出娜芙蒂蒂像有著垂直肩線的矛盾，但哈瓦斯認為這不過是娜芙蒂蒂的丈夫[[阿肯那頓|阿肯那頓]]所引導的新藝術風格所帶來的改變之一。哈瓦斯同時也宣稱雕刻家圖特摩斯其實有雕出左眼，只是這隻眼睛後來被毀掉罷了。<ref name = "digital">{{cite journal|last=Christopher Szabo|date=2009-05-12|title=Egypt’s Rubbishes Claims that Nefertiti Bust is ‘Fake’|journal=DigitalJournal.com|url=http://www.digitaljournal.com/article/272498}}</ref>

== 文化上的重要性 ==
[[File:Stamps_of_Germany_(Berlin)_1988,_MiNr_814.jpg|thumb]]的[[郵票|郵票]]，上面印有娜芙蒂蒂像的圖樣]]

娜芙蒂蒂半身像目前已經成為古埃及最被廣為欣賞、有著最多複製品的圖像之一，它是用來推銷柏林博物館的明星展品<ref name = "natgeo"/>，也同時被視為國際性的美感指標。<ref name = "times"/><ref name = "bbc"/><ref name="disc1"/>「這座雕像展現了一個有著纖長頸項、優雅的弓型[[眉毛|眉毛]]、高顴骨、細長鼻子，且紅唇上帶著附有活力的微笑的女性，它讓娜芙蒂蒂成為古代藝術品裡最美麗的女人之一。」<ref name="disc1"/>娜芙蒂蒂像被描述成古代藝術中最負盛名的藝術品之一，可以同[[圖坦卡門的面具|圖坦卡門的面具]]相提並論。<ref name = "Schultz"/>

娜芙蒂蒂半身像已經成為柏林文化的[[象徵|象徵]]之一，<ref name = "b285"/>每年有將近五十萬人前往參觀娜芙蒂蒂像的展出。它被這麼形容：「古埃及藝術品中最廣為人知的一件作品，甚至可以說它是所有古藝術品中最出名的一件。」<ref>Siehr p.114</ref>娜芙蒂蒂的臉孔不僅出現在[[柏林|柏林]]的明信片上，也在1989年的時候登上了德國[[郵票|郵票]]。<ref name = "bbc">{{cite news|url=http://news.bbc.co.uk/2/hi/europe/6632021.stm|title=Row over Nefertiti bust continues |last=Tristana Moore  |date= 2007-05-07|work=[[BBC|BBC]] News|accessdate=2009-11-22}}</ref><ref name="b292"/>

德國媒體在1930年的時候把娜芙蒂蒂像當成了他們新的統治者，將其人格化為女王。做為「德意志普魯士的藝術寶庫這個皇冠上最珍貴的一顆寶石」，娜芙蒂蒂像會重新建立起[[德意志帝國|德意志帝國]]在1918年滅亡後的國族認同。<ref name = "b291"/> [[希特勒|希特勒]]這麼形容這座雕像：一個獨特的傑作、一件出色的飾品，一樣真實的寶藏。他還發誓要為這座雕像建一座博物館來收藏它。<ref name="guard"/>到了1970年代時，這尊半身像還成了二次大戰後建立的東德與西德國際身分上的爭議點。<ref name = "b291">Breger p. 291</ref>到了1999年，娜芙蒂蒂像還出現在[[德國綠黨|德國綠黨]]的競選海報上，以“柏林需要強悍的女性”這個口號來允諾選民一個有著世界主義和多元文化的環境。<ref name = "b292">Breger p. 292</ref>美國[[印地安那大學|印地安那大學]]德國研究所的克勞狄亞·布雷格（Claudia Breger）助教授認為，娜芙蒂蒂像之所以會成為德國國族身分的原因在於它有著能與統治過[[埃及|埃及]]的[[英國|英國]]所擁有的[[圖坦卡門|圖坦卡門]]相關文物相抗衡的地位。<ref name = "b292"/>

這座雕像也因為傑克·皮爾斯（Jack Pierce）在[[好萊塢|好萊塢]]電影《科學怪人的新娘》（Bride of Frankenstein）中為女演員艾爾莎·蘭切絲特（Elsa Lanchester）所設計的那個以娜芙蒂蒂像為靈感來源的髮型，而在[[流行文化|流行文化]]有著相當的影響力。<ref>Elizabeth Young, "Here Comes the Bride: Wedding Gender and Race in Bride of Frankenstein"; ''Feminist Studies'', Vol. 17, 1991. 35 pgs.</ref>
