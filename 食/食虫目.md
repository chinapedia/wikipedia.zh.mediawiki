'''食虫目'''（[[学名|学名]]：{{lang|la|'''Insectivora'''}}）是一個[[棄用生物分類列表|已被棄用]]的[[哺乳綱|哺乳綱]][[動物|動物]]分類，其成員現時主要都被歸入[[鼩鼱目|鼩鼱目]]，其他的被分入其他目，又或獨立成為一目。

在過去，食虫目是一個包括一籃子小型到非常小型动物的分類。过去以为食虫目是真兽类最原始的成员，這些動物由於相對上較少分化，過往因為難以分類，所以都被綜合起來成為一類。

食虫性动物在中生代上白垩纪地层中就已出现。过去认为食肉目、翼手目、啮齿目都是由早期的食虫目分化出来的。基本就是[[鼩鼱|鼩鼱]]类、[[鼹|鼹]]类和[[刺猬|刺猬]]类，寿命一般不长，鼩鼱只能活6周，如果几小时不进食就会因无法维持体温死亡，最长的刺猬也就只能活10年左右，对人类的经济意义不大，可以控制部分虫害，但有的种也能传染[[鼠疫|鼠疫]]、[[疟疾|疟疾]]等疾病。

但后来基因研究等分子分类学的发展，得知食虫目是个[[多系群|多系群]]，其成员彼此之间并不亲近，而是来自不同演化支，如[[食蚁兽|食蚁兽]]属[[贫齿总目|贫齿总目]]；[[鼩鼱|鼩鼱]]、[[刺猬|刺猬]]属于[[劳亚兽总目|劳亚兽总目]]；[[马岛猬科|马岛猬科]]属于[[非洲兽总目|非洲兽总目]]等。

== 特征 ==
食虫目动物均为身体被以柔毛或硬刺的、外形似小[[老鼠|老鼠]]的小型有胎盘类动物。多数食虫目的成员很高度适应食虫生活，是高效率的猎手。有些种类的[[唾液|唾液]]中有毒性，可以轻松的制服对手。
== 分布 ==
全世界一共约有400种。几乎广泛分布在全世界（除了[[大洋洲|大洋洲]]和[[南极洲|南极洲]]〕。而在[[南美洲|南美洲]]，只见于南美洲最北部。
== 食物 ==
它们的[[食物|食物]]一般都是动物性的，如[[无脊椎动物|无脊椎动物]]、[[昆虫|昆虫]]、小型[[蛙|蛙]]、[[蛇|蛇]]、[[蜥蜴|蜥蜴]]、小[[鱼|鱼]]等，也部分食用水果。
== 种类 ==
食虫目在后期的大致分为9[[科_(生物)|科]]：
* [[猬科|猬科]](Erinaceidae)，10属，14种；现归于[[劳亚兽总目|劳亚兽总目]][[猬形目|猬形目]]
* [[鼹科|鼹科]](Talpidae)，15属，22种；现归于[[劳亚兽总目|劳亚兽总目]][[鼩鼱目|鼩鼱目]]
* [[鼩鼱科|鼩鼱科]](Soricidae)，24属，312种；现归于[[劳亚兽总目|劳亚兽总目]][[鼩鼱目|鼩鼱目]]
* [[沟齿猥科|沟齿猥科]](Solenodontidae)，2属，2种；现归于[[劳亚兽总目|劳亚兽总目]][[鼩鼱目|鼩鼱目]]
* [[马达加斯加猬科|马达加斯加猬科]](Tenrecidae)，9属，20种；现归于[[非洲兽总目|非洲兽总目]][[非洲猬目|非洲猬目]]
* [[獭鼩科|獭鼩科]](Potamogalidae)，2属，3种；现归于[[非洲兽总目|非洲兽总目]][[非洲猬目|非洲猬目]][[马岛猬科|马岛猬科]][[獭鼩亚科|獭鼩亚科]]
* [[金毛鼹科|金毛鼹科]](Chrysochloridae)，5属，11种；现归于[[非洲兽总目|非洲兽总目]][[非洲猬目|非洲猬目]]
* [[象鼩科|象鼩科]](Macroscelididae)，5属，28种；现归于[[非洲兽总目|非洲兽总目]][[象鼩目|象鼩目]]
* [[树鼩科|树鼩科]](Tupaiidae)，5属，15种。现归于[[灵长总目|灵长总目]][[树鼩目|树鼩目]]

有的分类法将象鼩科和树鼩科单分为[[目_(生物)|目]]，将马达加斯加猬科合併到猬科，将金鼹科合併到鼹科，将獭鼩科合併到[[象鼩目|象鼩目]]，那样食虫目就只有四[[科_(生物)|科]]。[[象鼩目|象鼩目]]和[[攀兽目|攀兽目]]曾经被置于食虫目。
