'''迟夙生'''（{{bd|1956年|4月11日||}}）女，[[黑龙江省|黑龙江省]][[齐齐哈尔市|齐齐哈尔市]]人，[[中华人民共和国|中华人民共和国]]律师<ref name=tx>{{Cite web |url =http://news.qq.com/a/20090304/001507.htm  |title = 迟夙生简历 |author =  |date = 2009-03-04 |publisher = 腾讯 |language =  |accessdate =  |quote =  }}</ref>。

==生平==
1975年到1978年，迟夙生在齐齐哈尔市塑料二厂工作。1978年到1979年，在[[哈尔滨|哈尔滨]]电机学校一重分校学习。1981年到1983年，在[[齐齐哈尔大学|齐齐哈尔大学]]中文专业、[[吉林大学|吉林大学]]学习，获得双学士学位。1979年到1994年，任齐齐哈尔市第一律师事务所律师。1994年起，任齐齐哈尔市夙生律师事务所主任、二级律师<ref name=tx/>。

她是第九、十、十一届全国人大代表<ref name=tx/><ref>{{cite web|author=新华网|title=第十一届全国人大代表|url=http://news.xinhuanet.com/ziliao/2008-03/03/content_7708230.htm?node=36628|publisher=新华网|accessdate=2014-01-07}}</ref><ref>{{cite news|title=全国人大代表信息-迟夙生|url=http://www.npc.gov.cn/delegate/viewDelegate.action?dbid=110585|accessdate=2013-04-05|newspaper=全国人大网}}</ref>。曾任第八、第九届[[龙沙区|龙沙区]]政协副主席。齐齐哈尔市人大内务司法委员会委员、齐齐哈尔市工商联执委。1986年加入[[九三学社|九三学社]]，后任九三学社第二届齐齐哈尔市委常委<ref name=tx/>。

1979年，迟夙生成为中国改革开放后首批律师。1980年代，她应邀到工厂、学校讲授法律。1985年，她为齐齐哈尔[[改革开放|改革开放]]后第一宗贪污案的被告辩护，使死刑改成了死缓。1994年，她离开齐齐哈尔市第一律师事务所成立夙生律师事务所，成为齐齐哈尔首位砸碎自己“铁饭碗”的执业律师。此后她开始长年免费提供法律咨询，在齐齐哈尔电视台、齐齐哈尔广播电台以及报纸上长年设专题节目或栏目。2011年，迟夙生将自己调查整理的“北海律师事件”调查报告交给[[全国人大常委会法制工作委员会|全国人大常委会法制工作委员会]]。2011年底，她成为[[李庄案|李庄案]]第三季的代理律师。在担任全国人大代表期间，她呼吁关注[[尘肺|尘肺]]病人，为辽宁省大鹿岛渔民保住渔业合作社并促成该村财务公开。2012年1月12日，在代理“黎庆洪涉黑案”时，因当庭被贵阳市小河区人民法院法官驱逐而晕倒。2012年，她作为全国人大代表领衔提出的修改《[[中华人民共和国刑事诉讼法|中华人民共和国刑事诉讼法]]》的提案获得采纳，但她对《全国人民代表大会关于修改〈中华人民共和国刑事诉讼法〉的决定》投了反对票<ref>{{Cite web |url = http://news.ifeng.com/opinion/gundong/detail_2012_03/18/13266564_0.shtml |title =敢言敢为迟夙生 她的行动，就是她最好的议案  |author =  |date = 2012-03-18 |publisher =  |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web|url = http://www.nfpeople.com/story_view.php?id=4320|title = 迟夙生 没永远的代表 有永远的律师|author = |date = 2013-03-15|publisher = 南方人物周刊|language = |accessdate = |quote = |archive-url = https://web.archive.org/web/20170408222318/http://nfpeople.com/story_view.php?id=4320|archive-date = 2017-04-08|dead-url = yes}}</ref>。2003年至2011年，出于防治[[艾滋病|艾滋病]]等考虑，她每年都作为全国人大代表领衔提出让[[卖淫|卖淫]]合法化的提案<ref>{{Cite web |url =  http://view.news.qq.com/a/20120312/000001.htm|title = 代表连续9年呼吁卖淫合法化 称嫖娼可减少强奸 |author =  |date =2012-03-12  |publisher =腾讯  |language =  |accessdate =  |quote =  }}</ref>。2017年5月，她在抚顺市望花区人民法院代理案件时，受到法院妨碍，并晕倒受伤，为此[[中共辽宁省委|中共辽宁省委]]政法委组成调查组调查<ref>{{Cite web |url =  http://news.163.com/17/0513/02/CK9J5AQJ00014AEE.html|title = 律师庭上与法官争吵疑遭拖拽晕倒 辽宁:正调查 |author =  |date = 2017-05-13 |publisher =网易  |language =  |accessdate =  |quote =  }}</ref>。
