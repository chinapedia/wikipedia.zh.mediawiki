'''吲哚'''（{{lang-en|indole}}）是[[芳香环|芳香杂环]][[有机化学|有机]]化合物，為双环结构，包含了一个六元[[苯|苯]]环和一个五元含[[氮|氮]]的[[吡咯|吡咯]]环。因为氮的孤对[[电子|电子]]参与形成芳香环，所以吲哚不是[[碱|碱]]，性质也不同于简单的[[胺|胺]]。

在室温，吲哚是[[固体|固体]]。自然情况下，吲哚存在于[[人|人]]类的[[粪便|粪便]]之中，并且有强烈的粪臭味。然而，在很低的浓度下，吲哚具有类似于[[花|花]]的香味<ref name="smell">http://www.leffingwell.com/olfact5.htm<!-- there might be better references for its odor properties --></ref>，是许多花香的组成部分 ，例如[[橘子|橘子]]花，吲哚也用来制造[[香水|香水]]，[[煤焦油|煤焦油]]也會有吲哚。

在很多有机化合物中能发现吲哚结构，比如[[色氨酸|色氨酸]]及含色氨酸的[[蛋白质|蛋白质]]，生物碱及[[色素|色素]]中也包含有吲哚结构。

吲哚能发生[[亲电取代反应|亲电取代反应]]，多取代于3号位。取代吲哚是许多色胺碱的基础结构，比如[[神经传递素|神经传递素]]复合胺，[[褪黑素|褪黑素]]，[[迷幻药|迷幻药]]，[[二甲基色胺|二甲基色胺]]，[[5-甲氧基-二甲基色胺|5-甲氧基-二甲基色胺]]和[[LSD|LSD]]。其他的吲哚化合物包括植物[[生长素|生长素]]（吲哚-3-乙酸），抗炎药物[[消炎痛|消炎痛]]（茚甲新）和血管舒张药物[[心得乐|心得乐]]。

吲哚的名称indole是由indigo（[[靛藍|靛藍]]）和oleum（[[发烟硫酸|发烟硫酸]]）所组成，因为吲哚首次制得是通过混合靛蓝和发烟硫酸。

== 历史 ==

[[File:Baeyer_indole_structure.png|left]]

吲哚化学是随着对[[靛青|靛青]]的研究开始发展的，继而转化为研究[[靛红|靛红]]，然后是[[羟吲哚|羟吲哚]]。在1866年，[[阿道夫·冯·拜尔|阿道夫·冯·拜尔]]用[[锌|锌]]粉将羟吲哚还原为了吲哚<ref name="baeyer1866">{{cite journal|author = [[Adolf_von_Baeyer|Baeyer, A.]]|journal = [[Ann.|Ann.]]|year = 1866|volume = 140|pages = 295|doi = 10.1002/jlac.18661400306|title = Ueber die Reduction aromatischer Verbindungen mittelst Zinkstaub|issue = 3}}</ref>。1869年，他假设出了吲哚的结构，至今仍然被大家公认<ref name="baeyer1869">{{cite journal|author = [[Adolf_von_Baeyer|Baeyer, A.]]; Emmerling, A.|journal = [[Chemische_Berichte|Chemische Berichte]]|year = 1869|volume = 2|pages = 679|doi = 10.1002/cber.186900201268|title = Synthese des Indols}}</ref>。

直到19世纪末，某些吲哚化合物也仅仅是作为重要的染料来看待。到了20世纪30年代，人们对吲哚的兴趣逐渐增强，因为吲哚作为一个核心基团出现在了很多重要的[[生物碱|生物碱]]中，例如[[色氨酸|色氨酸]]和[[植物激素|植物激素]]，到现在对于吲哚的研究仍然很活跃<ref name="vanorder1942">{{cite journal|author = R. B. Van Order, H. G. Lindwall|journal = [[Chem._Rev.|Chem. Rev.]]|year = 1942|volume = 30|pages = 69–96|doi = 10.1021/cr60095a004|title = Indole}}</ref>。

== 吲哚的化学反应 ==
=== 碱性 ===

不同于大多数[[胺|胺]]，吲哚几乎没有[[碱性|碱性]]。它的成键环境与[[吡咯|吡咯]]极为相似，只有像[[盐酸|盐酸]]这样的强酸才可能将之[[质子化|质子化]]，得到的[[共轭酸|共轭酸]]的pKa是-3.6。很多吲哚类化合物（如[[色胺|色胺]]）在酸性环境下的活性都是由此产生的。

=== 亲电取代反应 ===

吲哚上最容易发生[[亲电取代反应|亲电取代反应]]的位置是3位，它的活性是其苯环碳的10<sup>13</sup>倍。例如[[维尔斯迈尔－哈克反应|Vilsmeier-Haack酰化反应]]在室温下就能于3位碳上发生<ref name="james1959">{{cite journal|author=James, P. N.; Snyder, H. R.|year=1959|title=Indole-3-aldehyde |journal=[[Organic_Syntheses|Organic Syntheses]]|volume=39|pages=30| url=http://www.orgsyn.org/orgsyn/prep.asp?prep=cv4p0539}}</ref>。由于吡咯环上电子富集，往往在1位氮和2，3位碳都被取代后，苯环上的[[亲电取代反应|亲电取代反应]]才可能发生。

: [[Image:Indole_Vilsmeyer-Haack_Formylation.png|300px]]

[[芦竹碱|芦竹碱]]是一种由吲哚、[[二甲胺|二甲胺]]和[[甲醛|甲醛]]通过[[Mannich反应|Mannich反应]]得到的常用[[中间体|中间体]]。它是[[3-吲哚乙酸|3-吲哚乙酸]]和[[色氨酸|色氨酸]]的前体。

: [[Image:Gramine_From_Indole_Scheme.png|250px]]

=== N位氢的酸性和金属有机盐 ===

N位氢在[[二甲基亞碸|DMSO]]中电离的pKa是21，所以需要非常强的碱，诸如[[氢化钠|氢化钠]]或者[[丁基锂|丁基锂]]，才有可能在无水环境下将之[[去质子化|去质子化]]。得到的金属有机盐（具有非常强的碱性）有两种存在形式。对于难成共价键的[[钾|钾]]、[[钠|钠]]离子，负电荷集中在1位氮上；而对于可以成共价键的[[镁|镁]]（包括[[Grignard试剂|Grignard试剂]]）和（尤其是）[[锌|锌]]，负电荷则集中在3位碳上。类似的，在极性非质子溶剂比如[[DMF|DMF]]或[[二甲基亞碸|DMSO]]中，1位氮易受到亲电试剂进攻；而在非极性溶剂比如[[甲苯|甲苯]]中，则是3位碳更活泼<ref name="heaney1974">{{cite journal|author=Heaney, H.; Ley, S. V.|year=1974|title=1-Benzylindole |journal=[[Organic_Syntheses|Organic Syntheses]]|volume=54|pages=58| url=http://www.orgsyn.org/orgsyn/prep.asp?prep=cv6p0104}}</ref>。

: [[Image:Indole_anion_reactions.svg|400px]]

=== 碳上氢的酸性和2位碳的锂化 ===

2位碳上的氢酸性排第二。对于氮被保护了的吲哚衍生物，丁基锂或[[二异丙基氨基钾|二异丙基氨基钾]]就专一地在2位碳上发生去质子反应，得到非常强的[[亲核试剂|亲核试剂]]。
Bergman 和 Venemalm 开发出了在不保护氮的条件下在2位碳上取代的反应<ref name="bergman1992">{{cite journal|author = Bergman, J.; Venemalm, L.|journal = [[J._Org._Chem.|J. Org. Chem.]]|doi = 10.1021/jo00034a058|title = Efficient synthesis of 2-chloro-, 2-bromo-, and 2-iodoindole|year = 1992|volume = 57|pages = 2495|issue = 8}}</ref>。

: [[Image:Bergman_Indole_Lithiation.png|400px]]

[[Alan_Katrizky|Alan Katrizky]] 也曾开发出过类似的技术<ref>{{cite journal|title = Facile Synthesis of 2-Substituted Indoles and Indolo[3,2-''b'']carbazoles from 2-(Benzotriazol-1-ylmethyl)indole|author = Alan R. Katritzky, Jianqing Li, Christian V. Stevens|journal = J. Org. Chem.|year = 1995|volume = 60|issue = 11|pages = 3401–3404|doi = 10.1021/jo00116a026|url = http://pubs.acs.org/doi/abs/10.1021/jo00116a026}}</ref>。

=== 吲哚的氧化 ===

由于吲哚的富电子性，吲哚很容易被氧化。[[N-溴代丁二酰亚胺|N-溴代丁二酰亚胺]]可以选择性的将吲哚氧化为羟吲哚<ref name="lynch2002">{{cite journal|author = Lynch, S. M. ; Bur, S. K.; Padwa, A.|journal = [[Org._Lett.|Org. Lett.]]|doi = 10.1021/ol027024q|pmid = 12489950|title = Intramolecular Amidofuran Cycloadditions across an Indole π-Bond: An Efficient Approach to the ''Aspidosperma'' and ''Strychnos'' ''ABCE'' Core|year = 2002|volume = 4|issue = 26|pages = 4643}}</ref>。

: [[Image:Indole_NBS_Oxidation.png|600px]]

=== 吲哚的环加成反应 ===

只有2位碳和3位碳上的[[π键|π键]]能发生[[环加成反应|环加成反应]]。吲哚发生分子间环加成的倾向不高，但其衍生物的分子异变却可以有很高的产率。例如 Padwa et al 通过[[Diels-Alder反应|Diels-Alder反应]]合成了[[士的宁|士的宁]]的下一步中间体。在这个反应中，[[2-氨基呋喃|2-氨基呋喃]]作为了[[双烯体|双烯体]]，而吲哚作为[[亲双烯体|亲双烯体]]。

: [[Image:Indole_Cycloaddition_Padwa.png|250px]]

吲哚也可以发生分子内的 [2+3] 和 [2+2] 环加成反应。
