'''吲唑'''（{{lang-en|Indazole}}），即1,2-二氮杂[[茚|茚]]；苯并[[吡唑|吡唑]]。

== 性质 ==

无色针状结晶，易溶于稀酸，溶于[[醇|醇]]、[[醚|醚]]和热水。它的碱性比[[吡唑|吡唑]]弱，但氮上的氢酸性较强。

== 制取 ==

1、[[邻甲苯胺|邻甲苯胺]][[乙酰化|乙酰化]]，然后[[亚硝化|亚硝化]]生成[[亚硝胺|亚硝胺]]，并在45～50°C下在苯中重排为[[偶氮|偶氮]][[乙酰氧基|乙酰氧基]]化合物 1-乙酰氧基-2-邻甲苯基[[二亚胺|二亚胺]]，最后发生环合、消除[[乙酸|乙酸]]得到吲唑。

2、由[[邻氨基苯甲酸|邻氨基苯甲酸]]经[[重氮化|重氮化]]、还原、环合、氯化得到[[3-氯吲唑|3-氯吲唑]]，然后将3-氯吲唑与[[红磷|红磷]]和[[氢碘酸|氢碘酸]]混合回流24小时，精制后得到吲唑。

== 用途 ==

用于[[有机合成|有机合成]]。
