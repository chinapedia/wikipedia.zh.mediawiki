在[[數學|數學]]中，'''奇異點'''（{{lang-en|singularity}}）或'''-{zh-cn:奇异点; zh-tw:奇點;}-'''，是數學物件中無法處理的點。一般來說，可以分成兩種狀況：
* 這個點的值在數學上沒有定義。例如，一個[[除以零|除以零]]的點。函數<math>f(x) = 1/x</math>在<math>x=0</math>的點，是一個奇異點；這個點有個性質－它趋向于[[无穷|無限]]。然而，在數學中，無限的值是沒有定義的。在物理中，也儘量避免或除去導致無限的點，雖然在[[宇宙学|宇宙学]]中有[[引力奇点|引力奇點]]（黑洞奇點）。
* 或者，在某方面來說，這個點破壞了該數學物件的整體一致性。這個點被稱為[[病態_(數學)|病態]]的（{{lang|en|pathological}}），是[[良态|良态]]的反義。一般的例子是：
** 光滑的曲線或平面（[[光滑函数|光滑函数]]）上的[[尖點|尖點]]，它破壞了該函數的[[可微函数|可微性]]。
** 連續的曲線中一個斷掉的點，它破壞了該曲線的[[连续函数|連續性]]。

== 不可微的點 ==
就[[可微函数|可微性]]來說：
* 曲線<math>y^2 = x</math>在<math>x=0</math>的點是該曲線的奇異點，因為該點的切線是垂直的。[[垂直切線|垂直切線]]（{{lang|en|vertical tangent}}）的斜率是無限，所以該點[[可微函数|不可微]]。
* [[绝对值|绝对值]]函數<math>f(x) = \left|x\right|</math>在<math>x=0</math>的點是該函數的奇異點，因為在該點上無法決定斜率，所以該點不可微。
* 代數集合<math>\{(x,y):\left|x\right|=\left|y\right|\}</math>在<math>x=0</math>的點是奇異點，因為該點不可微。

== 不連續的點 ==
{{main| 不连续点}}
在[[实变函数论|實變數分析]]中，奇點是[[不连续点|不连续点]]，或是[[导数|导数]]的不連續點。

== 複分析 ==

在[[複分析|複分析]]中，有四类奇点，如下所述。假定''U''為[[複數集|複數集]]'''C'''的一個[[開集|開子集]]，''a''是''U''內的一元素，而''f''為定義在去心[[鄰域|鄰域]]''U'' \ {''a''}下的[[全純函數|復可微函數]]。

* [[孤立奇點|孤立奇點]]：假定''f''即使定義在''U'' \ {''a''}，但未定義於'''a'''。
** [[可去奇點|可去奇點]]
** [[極點　(複分析)|極點]]
** [[本性奇點|本性奇點]]
* [[分支點|分支點]]：扼要的說，支點通常是[[多值函數|多值函數]]的結果，諸如<math>\sqrt{z}</math>或<math>\log{z}</math>定義在確實的範圍內，使得它的呈現如同單值函數。
* [[非孤立奇點|非孤立奇點]]
