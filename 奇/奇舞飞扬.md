'''奇舞飞扬'''是由中国湖南电视台卫星频道推出的一档综艺节目，于2013年1月7日首播<ref>{{cite web|url=http://ent.163.com/13/0108/17/8KNCH90M00032DGD.html | language= | title="《奇舞飞扬》首播开门红 收获好口碑观众评论12万 |publisher=网易    |date=2013-01-08  }}</ref>，每周二22：00播出是国内首档创意舞蹈奇趣秀，节目宗旨在于展现草根的舞蹈创意与激情。

== 节目内容 ==
以展现平民舞蹈创意为宗旨的全民舞蹈节目，草根报名参加的展现舞蹈创意和激情的舞台，节目中的“out”区是一大亮点。
== 节目主持人 ==
* 2013年1月7日，节目主持人为KK和贾玲<ref>{{cite web|url=http://culture.cnhubei.com/jcfh/jcrw/201301/t2402441.shtml | language= | title="湖南卫视《奇舞飞扬》栏目7日晚首播 贾玲转型担任主持人 |publisher=荆楚网    |date=2013-01-05  }}</ref>。
* 2013年1月28日，节目主持人改为KK、杨乐乐、刘烨<ref>{{cite web |url=http://www.hn.xinhuanet.com/2013-01/23/c_114465978.htm |language= |title="杨乐乐华丽助阵《奇舞飞扬》 刘烨主持out特区 |publisher=新华网 |date=2013-01-23 |deadurl=yes |archiveurl=https://web.archive.org/web/20140517160711/http://www.hn.xinhuanet.com/2013-01/23/c_114465978.htm |archivedate=2014-05-17 }}</ref>。
