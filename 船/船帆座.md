'''船帆座'''（Vela），于十八世纪由[[南船座|南船座]]拆分得来。南船座原是南天星座之一，后被拆分为三个单独的星座，分别是船帆座、[[船底座|船底座]]和[[船尾座|船尾座]]。

== 星座神話 ==
[[托勒密|托勒密]]本稱[[船尾座|船尾座]]、船帆座及[[船底座|船底座]]為同一星座，叫做[[南船座|南船座]]（Argo Navis）。在十八世紀，天文學家認為[[南船座|南船座]]所佔之天區過大，故將其拆開。南船座即[[阿格號|阿格號]]（Argonauts），故事中[[伊阿宋|伊阿宋]]（Jason）帶著五十個人乘[[阿格號|阿格號]]到位於[[黑海|黑海]]的[[科爾基斯|科爾基斯]]（Colchis）找金羊毛。

[[伊阿宋|伊阿宋]]帶著眾多船員出航，當中包括[[雙子座|雙子座]]的[[北河二|卡斯托爾]]（Castor）和[[北河三|波呂杜克斯]]（Polydeuces / Pollux），樂師[[俄耳甫斯|奧爾普斯]]（Orpheus），建船者[[阿爾戈斯|阿爾戈斯]]（Argus），後來連[[海格力斯|赫拉克勒斯]]（Heracles）也加入旅程。劇作家把[[阿格號|阿格號]]描寫成船堅炮利，而[[牛頓|牛頓]]甚至將[[黃道十二宮|黃道十二宮]]與[[阿格號|阿格號]]扯上關係。建船之時，[[雅典娜|雅典娜]]（Athena）下令[[阿爾戈斯|阿爾戈斯]]採[[佩利翁山|佩利翁山]]（Pelion）的木材造船，[[宙斯|宙斯]]（Zeus）也指示[[阿爾戈斯|阿爾戈斯]]以[[多多納|多多納（]]Dodona）之橡木建船首，那裡的橡木賦有語言能力，在[[阿格號|阿格號]]啟航時，甚至聽到橡木的哭聲。

[[阿格號|阿格號]]在旅途中遇上重重困難，其中以[[撞岩|撞岩]]（Clashing Rocks）最為著名，撞岩又叫[[敘姆普勒加得斯|敘姆普勒加得斯]]（Symplegades），它好像自動門一樣開開合合擋著[[黑海|黑海]]的入口，當時[[伊阿宋|伊阿宋]]情急智生，放出白鴿，讓白鴿飛於船前，兩塊大石瞬時掩埋，夾斷白鴿的尾巴，船員趁兩塊大石打開再次撞擊之前，出盡九牛二虎之力，再得[[雅典娜|雅典娜]]之助，結果只是船尾受到少許損壞。

經過幾番波折進入[[黑海|黑海]]後．[[伊阿宋|伊阿宋]]偷去金羊毛回到[[希臘|希臘]]，把[[阿格號|阿格號]]泊於[[科林斯|科林斯]]（Corinth），算是對海神[[波塞冬|波塞冬]]（Poseidon）的一種感謝。

在星圖上，我們只能見到[[阿格號|阿格號]]的船尾，船頭被濃霧所覆蓋，或是被[[撞岩|撞岩]]所遮掩，有說是[[伊阿宋|伊阿宋]]晚年在船上沉思過往的歷險時，船首忽然塌下來壓死了熟睡中的[[伊阿宋|伊阿宋]]，於是[[波塞冬|波塞冬]]將船的其餘部份升上天空。

== 偽十字（南天假十字） ==
{{Main|南天假十字}}
'''[[南天假十字|偽十字]]'''是由船帆座δ、κ和[[船底座|船底座]]ι、ε四顆星組成的[[星群|星群]]，他經常會被誤認為是[[南十字座|南十字座]]，造成[[天文導航|天文航海]]上的錯誤。

== [[深空天體|深空天體]] ==
在船帆座內有趣的深空天體是[[行星狀星雲|行星狀星雲]][[NGC_3132|NGC 3132]]（暱稱為八裂星雲），也是在這個星座內令人感興趣的[[船帆座超新星殘骸|船帆座超新星殘骸]]。像信這是大約在10,000年前，在地球可以見到的一顆[[超新星|超新星]]爆炸形成的[[星雲|星雲]]，殘骸的[[脈衝星|脈衝星]]是第一顆被辨識出來的光學脈衝星。

[[古姆星雲|古姆星雲]]是一個暗淡的發射星雲，相信是數百萬年前爆炸的一顆古老的超新星殘骸。
