'''氢氧化锕'''是一种白色固体，化学式Ac(OH)<sub>3</sub>。有强放射性。

==制备==
氢氧化锕由可溶性锕盐和[[碱|碱]]或[[氨水|氨水]]反应，所得凝胶沉淀为产物：<ref name=wj>《无机化学丛书》.第十卷 锕系 锕系后元素. 张青莲 等. 科学出版社. 4.4 锕的化合物. P<sub>75</sub>. ISBN 978-7-03-030572-5</ref>
: Ac<sup>3+</sup> + 3 OH<sup>−</sup> → Ac(OH)<sub>3</sub>↓
其溶解度略大于[[氢氧化镧|氢氧化镧]]，但仍难溶于水。
==化学性质==
氢氧化锕是碱性氢氧化物，可以和酸反应产生相应的盐<ref name=wj />：
: Ac(OH)<sub>3</sub> + 3 HNO<sub>3</sub> → Ac(NO<sub>3</sub>)<sub>3</sub> + 3 H<sub>2</sub>O
