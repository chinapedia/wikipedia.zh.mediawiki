'''氢氧化钕'''是一种[[无机化合物|无机化合物]]，[[化学式|化学式]]为Nd(OH)<sub>3</sub>。
==制备==
[[硝酸钕|硝酸钕]]和[[氨水|氨水]]作用，产生氢氧化钕：<ref name=zb>《无机化合物制备手册》. 朱文祥 主编. 化学工业出版社. 【III-119】氢氧化钕（neodymium hydroxide）</ref>
: Nd(NO<sub>3</sub>)<sub>3</sub> + 3 NH<sub>3</sub>·H<sub>2</sub>O → Nd(OH)<sub>3</sub>↓ + 3 NH<sub>4</sub>NO<sub>3</sub>
若控制Nd(NO<sub>3</sub>)<sub>3</sub>浓度为40g/L，氨水浓度0.50mol/L，将氨水以1.5mL/min的速度加入Nd(NO<sub>3</sub>)<sub>3</sub>溶液，pH控制在7.35，以[[聚乙二醇|聚乙二醇]]为分散剂，则可得到氢氧化镧微粉，D<sub>50</sub>≤1μm。<ref>刘治平, 王晓铁. 氢氧化镧和氢氧化钕微粉的制备[J]. 《稀土》. Vol.25 No.3 Jun.2004</ref>
==化学性质==
氢氧化钕可溶于[[酸|酸]]，生成钕盐：
: Nd(OH)<sub>3</sub> + 3 H<sup>+</sup> → Nd<sup>3+</sup> + 3 H<sub>2</sub>O
