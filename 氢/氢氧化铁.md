'''氢氧化铁'''（[[化学式|化学式]]：Fe(OH)<sub>3</sub>或FeO(OH)·H<sub>2</sub>O）是[[铁|铁]]的[[氢氧化物|氢氧化物]]，由[[三价铁离子|三价铁离子]]（Fe<sup>3+</sup>）和[[氢氧根|氢氧根]]离子（OH<sup>−</sup>）生成：

: Fe<sup>3+</sup> + 3OH<sup>−</sup> → Fe(OH)<sub>3</sub>

也可以由[[氢氧化亚铁|氢氧化亚铁]][Fe(OH)<sub>2</sub>]氧化得来：

: 4Fe(OH)<sub>2</sub> + O<sub>2</sub> + 2H<sub>2</sub>O → 4Fe(OH)<sub>3</sub>

它是一种黄棕色沉淀，成分一般看作是铁（Ⅲ）的[[羟基|羟基]][[氧化物|氧化物]]的[[水合物|水合物]]。加热分解成[[三氧化二铁|三氧化二铁]]和[[水|水]]：

: 2Fe(OH)<sub>3</sub> → Fe<sub>2</sub>O<sub>3</sub> + 3H<sub>2</sub>O

新沉淀出来的水合氧化铁（Ⅲ）易溶于[[无机酸|无机酸]]，也略溶于[[碱|碱]]溶液中生成[[铁酸盐|铁（Ⅲ）酸盐]]（FeO<sub>2</sub><sup>−</sup>）。这类铁酸盐也可以从[[氧化铁|氧化铁（Ⅲ）]]和[[碱金属|碱金属]]氧化物、氢氧化物或[[碳酸盐|碳酸盐]]的熔融混合物中制得。

氢氧化铁（Ⅲ）是一个未确知的化合物，[[氨水|氨水]]或[[碱|碱]]溶液作用于铁（Ⅲ）盐溶液所得到的红棕色或黄棕色沉淀，经[[X射线晶体学|X射线晶体学]]研究是[[无定形体|非晶态]]的，它含有可变量的水。一般认为该沉淀含有FeO(OH)，至少有两种结晶变体：α-FeO(OH)（[[针铁矿|针铁矿]]）和γ-FeO(OH)（[[纤铁矿|纤铁矿]]），铁的正常[[生锈|生锈]]产生的是γ-变体。<ref>{{cite book|author=谢高阳、俞练民、刘本耀 |title=《无机化学丛书》 |series=第九卷 锰分族铁系铂系 |location=北京 |publisher=科学出版社 |date=1998年10月 |pages=页254}}</ref>

== 胶体 ==
=== 制备 ===
取蒸馏水于试管中在[[酒精灯|酒精灯]]上煮沸，滴入FeCl<sub>3</sub>溶液，在出现红棕色时立即停止加热，可以制得稳定的氢氧化铁胶体。

: FeCl<sub>3</sub> + 3H<sub>2</sub>O → Fe(OH)<sub>3</sub>（胶体） + 3HCl

在新制得的Fe(OH)<sub>3</sub>沉淀中加入适量的FeCl<sub>3</sub>溶液作为[[分散剂|分散剂]]并充分搅拌也可以制得稳定的 Fe(OH)<sub>3</sub>胶体。<ref>{{cite book|author=华彤文、陈景祖 等 |title=《普通化学原理》 |edition=第三版 |date=2005年7月 |publisher=北京大学出版社 |location=北京 |pages=第73页 |isbn=978-7-301-09213-2}}</ref>

=== 结构 ===
FeCl<sub>3</sub>水解产生的Fe(OH)<sub>3</sub>一部分聚集形成胶核[Fe(OH)<sub>3</sub>]<sub>m</sub>，另一部分Fe(OH)<sub>3</sub>与HCl反应生成FeOCl，后者则电离生成FeO<sup>+</sup>和Cl<sup>-</sup>。

: Fe(OH)<sub>3</sub> + 3HCl → 3FeOCl + 2H<sub>2</sub>O

: FeOCl → FeO<sup>+</sup> + Cl<sup>-</sup>

氢氧化铁胶核会吸附FeO<sup>+</sup>和一些Cl<sup>-</sup>，后两者共同组成吸附层，吸附层和胶核构成胶粒。另外一些主要是Cl<sup>-</sup>的离子会松散的分布在胶粒周围形成扩散层。扩散层和胶粒一起称作胶团。<ref>{{cite book|author=张永安 等 |title=《无机及分析化学》 |location=北京 |publisher=北京师范大学出版社 |pages=第27页 |date=2009年8月1日 |isbn=978-7-303-09990-0}}</ref>
