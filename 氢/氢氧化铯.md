'''氢氧化铯'''（[[化学式|化学式]]：[[铯|Cs]][[氧|O]][[氢|H]]）是一种无机化合物，为强碱，可以腐蚀玻璃。与其他[[碱金属|碱金属]][[氢氧化物|氢氧化物]]类似，氢氧化铯也是超强的[[碱|碱]]，极容易[[潮解|潮解]]。实验室常用的是氢氧化铯[[水合物|一水合物]]。

==制备==
氢氧化铯可由[[氢氧化钙|氢氧化钙]]或[[氢氧化钡|氢氧化钡]]和[[碳酸铯|碳酸铯]]反应得到：
: Ca(OH)<sub>2</sub> + Cs<sub>2</sub>CO<sub>3</sub> → 2 CsOH + CaCO<sub>3</sub>↓
==化学性质==
氢氧化铯具有碱的通性，如可以和酸反应，得到铯盐：
: CsOH + HCl → CsCl + H<sub>2</sub>O
氢氧化铯可以吸收空气中的[[二氧化碳|二氧化碳]]：
: 2 CsOH + CO<sub>2</sub> → Cs<sub>2</sub>CO<sub>3</sub> + H<sub>2</sub>O
