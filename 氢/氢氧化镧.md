'''氢氧化镧'''是一种[[无机化合物|无机化合物]]，[[化学式|化学式]]为La(OH)<sub>3</sub>。
==制备==
[[硝酸镧|硝酸镧]]和[[氨水|氨水]]作用，产生氢氧化镧：<ref name=zb>《无机化合物制备手册》. 朱文祥 主编. 化学工业出版社. 【III-43】氢氧化镧（lanthanum hydroxide）</ref>
: La(NO<sub>3</sub>)<sub>3</sub> + 3 NH<sub>3</sub>·H<sub>2</sub>O → La(OH)<sub>3</sub>↓ + 3 NH<sub>4</sub>NO<sub>3</sub>
若控制La(NO<sub>3</sub>)<sub>3</sub>浓度为40g/L，氨水浓度0.50mol/L，将氨水以1.5mL/min的速度加入La(NO<sub>3</sub>)<sub>3</sub>溶液，pH控制在7.80，以[[聚乙二醇|聚乙二醇]]为分散剂，则可得到氢氧化镧微粉，D<sub>50</sub>≤1μm。<ref>刘治平, 王晓铁. 氢氧化镧和氢氧化钕微粉的制备[J]. 《稀土》. Vol.25 No.3 Jun.2004</ref>

==化学性质==
氢氧化镧是一種強鹼性氫氧化物，可溶于[[酸|酸]]，生成镧盐：
: La(OH)<sub>3</sub> + 3 H<sup>+</sup> → La<sup>3+</sup> + 3 H<sub>2</sub>O
