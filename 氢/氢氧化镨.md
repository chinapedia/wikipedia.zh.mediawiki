'''氢氧化镨'''是一种[[无机化合物|无机化合物]]，[[化学式|化学式]]为Pr(OH)<sub>3</sub>。

==制备==
[[氨水|氨水]]和[[硝酸镨|硝酸镨]]反应，产生氢氧化镨沉淀：<ref name=zb>《无机化合物制备手册》. 朱文祥 主编. 化学工业出版社. P<sub>98~99</sub>. 【III-103】氢氧化镨（praseodymium hydroxide）</ref>
: Pr(NO<sub>3</sub>)<sub>3</sub> + 3 NH<sub>3</sub>·H<sub>2</sub>O → Pr(OH)<sub>3</sub>↓ + 3 NH<sub>4</sub>NO<sub>3</sub>
==化学性质==
氢氧化镨可溶于[[酸|酸]]，生成镨盐：
: Pr(OH)<sub>3</sub> + 3 H<sup>+</sup> → Pr<sup>3+</sup> + 3 H<sub>2</sub>O
