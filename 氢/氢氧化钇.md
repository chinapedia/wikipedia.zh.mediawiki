'''氢氧化钇'''是一种[[无机化合物|无机化合物]]，属于[[碱|碱]]，化学式Y(OH)<sub>3</sub>。

==制备==
氢氧化钇可由[[硝酸钇|硝酸钇]]和[[氢氧化钠|氢氧化钠]]溶液反应得到：<ref>田俐，陈稳纯，陈琳 等. 水热法合成氢氧化钇纳米管. 无机材料学报. 2009. 24(2): 335-339</ref>

: Y(NO<sub>3</sub>)<sub>3</sub> + 3 NaOH → Y(OH)<sub>3</sub>↓ + 3 H<sub>2</sub>O
==化学性质==
氢氧化钇作为碱，可以和[[酸|酸]]反应：
: Y(OH)<sub>3</sub> + 3 HNO<sub>3</sub> → Y(NO<sub>3</sub>)<sub>3</sub> + 3 H<sub>2</sub>O

: 2 Y(OH)<sub>3</sub> + 3 H<sub>2</sub>SO<sub>4</sub> → Y<sub>2</sub>(SO<sub>4</sub>)<sub>3</sub> + 3 H<sub>2</sub>O

氢氧化钇受热分解：
: 2 Y(OH)<sub>3</sub> → Y<sub>2</sub>O<sub>3</sub> + 3 H<sub>2</sub>O↑
