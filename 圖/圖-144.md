'''圖-144'''（'''Tupolev -{Tu-144}-'''，[[北約代號|北約代號]]稱為'''戰馬'''（Charger）），是世界上第一款[[超音速客機|超音速客機]]，由前[[蘇聯|蘇聯]][[圖波列夫|圖波列夫]]公司製造。

== 歷史和特徵 ==

一开始，当苏联领导层听说法国与英国将合作开发超音速飞机后就立即下令开始研发苏联自己的“超音速客机”。Tu-144的設計可能是源自[[法國|法國]]的[[協和飛機|協和飛機]]，当时苏联的设计局得到了整套协和式设计图，但为了得到这份设计图，一个完整的间谍网被破坏，一些苏联商务人员被驱逐出英法地区。Tu-144在外觀與协和式极為相似，但在設計上有很大的不同，以至於兩者幾乎是分開研製的。事实上，图-144 的机身比协和式长而且宽，图-144 式每排 5 座而不是协和式的每排 4 座。图-144 的发动机更紧贴机身，而不是协和式那样分得较开。圖-144的機頭兩側裝有細條狀的鴨翼，在起飛和著陸時使用。兩款超音速客機在外觀上最大的分別是協和號採用的S形前緣三角翼，而圖-144採用的是气动性能上不及协和式的简单雙三角翼。

可折叠的鸭式前翼大大改善了低速飞行和起落时的控制性能，这又领先于协和式，所以在某程度上，圖-144在技術上和协和式有所分別，以至于多年后英法在研制协和式的后继机时也增加了鸭式前翼，用作彌補雙三角翼在低速時的表現，而协和飞机則利用S形前緣三角翼作出同樣改善，並免除了鴨翼對飛機帶來的複雜化。而且其原型產品於1968年12月31日在[[莫斯科|莫斯科]]附近首次試飛，時間方面比協和機的試飛早兩個月。圖-144於1969年6月5日突破[[音障|音障]]，並於同年7月15日的營運中達到2[[馬赫|馬赫]]的速度。

Tu-144的主起落架有較多的輪子，而且採用低壓輪胎。原型機CCCP-68001的主起落架由兩組六輪起落架組成，類似[[波音777|波音777]]和[[圖-154|圖-154]]的設計；量產型則改為以八個輪為一組的主起落架。

另一重大差別是發動機安裝位置。協和號的發動機在機翼下側中部，圖-144飛機在翼根之下。這種差別導致了兩架飛機的氣動特徵、發動機控制完全不同。在收起起落架時，Tu-144和協和號的方式也有所不同，協和號將起落架直接收在機腹中，Tu-144來得複雜得多，首先它需要將機輪組向上180度接起，之後再收入特長的引擎筒中。
[[File:NASA_107665main_tu-144_drawing.png|thumb]]

Tu-144和協和機另一個不同之處，在於著陸時的[[煞車|制動]]方式，Tu-144采用民航飞机中罕见的阻力伞，给航班运营带来了不小的麻烦。協和號和普通民航客機一樣，著陸後利用[[推力反向器|反推力]]和制動器減速，但图-144的起飞着陆速度很高，引擎也沒有配備反推器，以至于有必要使用[[減速傘|減速傘]]協助減速，不利於商業運行，這也是世界上最後一款採用減速傘減速的客機。

由於其獨特的定位，Tu-144在登場後很快就成為蘇聯彰顯國家技術能力的工具。

== 服役 ==
Tu-144S型客機於1975年12月26日起開始服役，提供貨運及郵政服務，來往[[莫斯科|莫斯科]]與[[哈薩克|哈薩克]]的[[阿拉木圖|阿拉木圖]]之間。至1977年11月始提供載客服務，但在1978年5月23日的空難發生後，[[俄羅斯國際航空|俄羅斯國際航空]]（Aeroflot）於6月1日把圖-144退出客運行列，其載客服務只歷時半年，以及共完成55次航班，載客總數僅有3284人。

圖-144在退出客運後，貨運服務沒有終止，俄航使用新型的Tu-144D型飛機，配備更省油的RD-36-51引擎，其航線可以更長，可來往莫斯科與[[哈巴羅夫斯克|哈巴羅夫斯克]]之間。在飛機脫離商用服務以前，共完成了102次客運及貨運的正式航班。

Tu-144在1980年代仍有服役，更奇怪的是工廠也還在製造，但這些飛機大多沒有固定航線，只提供額外及非正式的航班，已知的最後一班圖-144飛於1987年，由[[克里米亞|克里米亞]]飛往[[基輔|基輔]]。

至1990年，[[NASA|美國太空總署]]（NASA）有意投入高速研究計畫（High Speed Research，HSR），作為發展次世代[[超音速運輸|超音速運輸系統]]（Supersonic Transport，SST）的基礎。圖波列夫、NASA、[[麥道|麥道]]與[[波音|波音]]等等幾大機構合作把TU-144重生，發展出TU-144LL，作為HSR的研究用機。但1999年時，因成本及經濟效益問題，NASA終止HSR計劃，TU-144再一次回到被遺棄的停機坪。在2001年左右，一名美國人在[[網路拍賣|網路拍賣]]購得了這架Tu-144LL，但由於它的引擎屬軍事設備，俄方不會向他提供零件，使得Tu-144LL只能擱置在地面上，形同廢鐵。

== 意外 ==
在1973年6月3日舉行的[[巴黎航空展|巴黎航空展]]中，協和飛機與圖-144——世界僅有的兩種超音速客機——同台進行飛行表演。一架編號77102的量產型Tu-144S在進行最後一次通場後在爬升過程中，突然轉為猛烈的俯衝，隨後在空中解體並波及15棟地面建築。機上6名機組人員與地面8人遇難，隨機表演的圖-144主任設計師[[弗拉基米爾·連杰羅夫|弗拉基米爾·連杰羅夫]]也一同遇難。苏联官方的说法是一架法国飞机紧贴图144导致飞行员操纵失误，并且隐藏黑匣子直到苏联解体才公布黑匣子数据。

== 生產 ==

圖波列夫設計局共製造出16架圖-144客機，計有1架原型機（編號：CCCP-68001）、1架預產型Tu-144S（CCCP-77101）、9架量產型Tu-144S（CCCP-77102至77110），以及5架Tu-144D型（CCCP-77111至77115）。在製造68001期間，另製造了至少1個機體，用作地面測試之用。在引擎方面，Tu-144S使用NK-144[[渦輪扇引擎|渦輪扇引擎]]，Tu-144D使用推力更大的RD-36-51引擎，比NK-144更省油，在超音速飛行時不需使用[[後燃器|後燃器]]，以及續航距離更長。

Tu-144LL的機體和量產型Tu-144大致相同，但使用來自[[Tu-160|Tu-160超音速戰略轟炸機]]的NK-321引擎，每具推力超過50,000磅，是Tu-144所使用過之中推力最大的引擎。

== 收藏品 ==
圖-144一直不被視為是成功的機種，服役期間又短，導致市場上一直缺乏圖-144的收藏品。至2011年，烏克蘭的ICM（ICM Holding Company）才推出1/144比例的Tu-144及Tu-144D的縮比模型。在之前，只有[[愛德美|愛德美]]（Academy）與[[西方模型|西方模型]]（Western Model）等少數幾家[[模型|模型]]廠商曾推出過Tu-144的縮比模型產品，比例由1/700至1/200不等。直至2012年，德国[[Herpa|Herpa]]推出了两款图-144的模型，比例为1/500，打破了图-144缺乏相关收藏品的尴尬。
Modelsvit 是保加利亞的模型開發商. 有推出1/72 的圖-144模型.
