== 概要 ==
=== 源由 ===
[[File:Washimiyajinja_haiden.jpg|thumb]]
[[File:Washimiyajinja_honden.jpg|thumb]]
據神社社傳記載，鷲宮神社是由日本[[上古時代|上古時代]]的傳說中的民族{{link-ja|出雲族}}所草創的關東最古老的大型神社，曾被稱為「鷲宮大明神」、「浮島大明神」、「大酉元祖」。

傳說在日本神話時代，天穗日命及其子[[御子神|御子神]]武夷鳥命為經略[[東國|東國]]而來到[[武藏國|武藏國]]，命其侍從昌彦、昌武父子，以及人數27人的出雲族部族和當地部族共同祀奉作為當地[[氏神#鎮守|鎮守]]的大己貴命，是為鷲宮神社之起源。當時興建的神社就是現在神社的別宮{{link-ja|神崎神社}}。後來[[日本武尊|日本武尊]]在平定東國之際，為祈願戰勝而興建祀奉天穗日命和武夷鳥命的別宮。這座別宮就是現在神社的本殿鷲宮神社。但這些傳說在《[[延喜式神名帳|延喜式神名帳]]》和《[[六國史|六國史]]》里並沒有記載。

=== 歷史 ===
鷲宮神社有一別名為「土師之宮」（日語：{{lang|ja|土師の宮}}），因其源由另有一說為[[崇神天皇|崇神天皇]]時代自[[河內國|河內國]]遷徙到[[東國|東國]]的[[土師氏|土師氏]]從[[下總國|下總國]][[淺草|淺草]]沿著[[利根川|利根川]]北上移居到當地時為祀奉祖先興建神社而得名<ref>{{Cite web 
|url = http://nire.main.jp/rouman/fudoki/14sait20.htm
|script-title  = ja : はにしの宮
|publisher = {{lang|ja|神話の森}}
|work = {{lang|ja|歌語り風土記}}
|language = ja
|accessdate = 2019-1-25
}}</ref>。之後當地口音經過多年的演變，當地「土師之宮」的讀音「{{lang|ja|はにしのみや}}」逐漸演變為「鷲宮」的讀音「{{lang|ja|わしのみや}}」。下文提到的鷲宮神社神樂「鷲宮催馬樂神樂」的正式名稱，即是基於「土師」之名的傳承而名為「土師一流催馬樂神樂」。

鷲宮神社號稱為「關東最古老的大神社」，但鷲宮神社相關史料記載的最早年份是《[[吾妻鏡|吾妻鏡]]》中記載的[[建長|建長]]三年（[[1251年|1251年]]），因此一般認為這個名號缺乏根據。但當地經{{link-ja|考古發掘|発掘調査}}發現了[[繩紋時代|繩紋時代]]的遺跡，證實當地在上古時代便已有居民。

進入[[日本歷史#權門體制的中世紀|中世]]以後，鷲宮神社本社周邊地域成為[[征夷大將軍|將軍]]直屬莊園太田莊，因鷲宮神社為該地[[總社|總社]]，被作為關東的總社、鎮護關東的神社而受到東國[[武家|武家]]崇敬。據《吾妻鏡》記載，建長三年（1251年）[[北條時賴|北條時賴]]曾前往鷲宮神社{{link-ja|奉幣}}祈願，還記載了[[藤原秀鄉|藤原秀鄉]]、[[新田義貞|新田義貞]]、[[古河公方|古河公方]]、歷代[[上杉氏|上杉氏]]的[[關東管領|關東管領]]等要人前往鷲宮神社奉納{{link-ja|幣帛 (神道)|幣帛|幣帛}}、寄進神領{{NoteTag|name=4th|神領指神社的所有地。寄進指將金錢、物品捐贈給神社或寺廟，領主賜予土地給神社或寺廟也是寄進的一種。}}、營造社殿的事跡。神社自[[鐮倉時代|鐮倉時代]]到[[室町時代|室町時代]]為太田莊的總社，由[[太田氏#藤原秀鄉流太田氏|藤原秀鄉流太田氏]]及其後裔{{link-ja|小山氏}}進行管理，在神社背後還曾興建過城池栗原城。[[應安|應安]]五年（[[1372年|1372年]]）小山氏當主{{link-ja|小山義政}}重振神社香火。位於太田莊內的[[中川_(東京)|中川]]沿岸還有許多鷲宮神社的[[分靈|分祀]]。

[[天正|天正]]十九年（[[1591年|1591年]]），[[德川家康|德川家康]]贈與神社領地400石，之後的[[江戶幕府|德川幕府]]歷代將軍亦都發給神社[[朱印狀|朱印狀]]，保障神社領地安堵（即{{link-ja|朱印地|朱印地・黒印地}}）。自[[江戶時代|江戶時代]]初起到[[明治|明治]]時代初，神社領地由秀鄉流藤原姓田原氏流[[大內氏|大內氏]]管治（所領約1000石）。江戶時代初，大內氏在[[德川家光|德川家光]]參拜[[日光東照宮|日光東照宮]]時參與利根川渡河警備，在家光不慎墜河時將其救起。這件大功本來能為大內氏加封領地，但也許是因為當時正值[[關原之戰|關原之戰]]戰後不久的幕府尚未有餘力加封大名，便以[[江戶城|江戶城]]內1萬石的形式賞賜大內氏。因此，後來大內氏不僅在登江户城拜谒将军时可以使用对应[[大名|大名]]身份的{{lj|侍候席}}，就連大內氏的{{link-ja|用人}}據說在江戶城內也享有[[家老|家老]]待遇。

[[明治|明治]]元年（[[1868年|1868年]]），被明治新政府劃為[[東京十社|准勅祭社]]（東京十二社）之一。在明治初年的[[神佛分離|神佛分離]]政策寺院整理神社統合中，神社的大乘寺別當地位被廢除{{NoteTag|name=5th|明治時代之前的日本奉行[[神佛習合|神佛習合]]思想，將一些佛寺附屬於神社管理，稱為[[神宮寺|神宮寺或別當寺等]]。而神宮寺管理職位名稱即是{{link-ja|別當|別当#寺}}。}}。

=== 重要無形民俗文化財產 （國家指定） ===
; 鷲宮催馬樂神樂
: 鷲宮神社舉行的[[神樂|神樂]]鷲宮催馬樂神樂（正式名稱：土師一流催馬樂神樂），是關東神樂的源流。神樂的作成年代並不明確，但據[[鎌倉時代|鎌倉時代]]史書《[[吾妻鏡|吾妻鏡]]》記載，鷲宮神社曾在[[建長|建長]]三年（[[1251年|1251年]]）4月舉行神樂，但無足夠證據證實其是否與今日的鷲宮催馬樂神樂是同一神樂。神樂在江戶[[享保|享保]]時代之前曾擁有三十六[[wikt:ja:座#接尾辞|座]]曲目，但經時任大宮司藤原國久改編，變為如今的十二座曲目。而除了十二座曲目以外，神樂實際上還擁有外一座曲目與番外一座曲目，以及端神樂。
: 鷲宮催馬樂神樂是鷲宮神社一社相傳的社傳神樂，由舉行神樂的家族世襲傳承，並領有神社賜予的3{{link-ja|反 (單位)|反#土地の面積の単位|反}}（約2975.208[[平方米|平方米]]）土地{{link-ja|知行}}。但自[[昭和|昭和]]時代以來，舉行神樂的神樂師越來越少，到了昭和二十年代，神樂的傳承者僅剩白石國藏一人。為挽救瀕臨消亡的神樂，[[1955年|1955年]]（昭和三十年）7月31日，以[[NHK廣播|NHK廣播]]在全國演奏神樂笛聲為契機，町內聚集十餘名年輕人組成了「鷲宮神社神樂復興會」，並由白石指導傳承神樂，總算使神樂避免了消亡的危機。後來「鷲宮神社神樂復興會」更名為「鷲宮催馬樂神樂保存會」，至今仍在為了神樂的傳承而活動著。
: 神樂會在下文標註的6場神社祭典中舉行，此外在每年的4月15日還會在八甫支社舉行1場，合計每年舉行7場。
: 關於神樂的更多資料在久喜市鄉土資料館中有展出。

=== 神輿 ===
; {{link-ja|千貫神輿}}
: 千貫神輿是一座製作於[[寬政|寬政]]年間（[[1789年|1789年]]－[[1791年|1791年]]）的{{link-ja|神輿}}。神輿長寬皆長約1.4米，重約3噸，是[[埼玉縣|埼玉縣]]內最大型的神輿。平日供奉在八坂神社後面的神輿殿內。
: 每年7月13日，即夏季祭典（八坂祭）的第1天，都會舉行千貫神輿的聖駕巡遊。最初是由擔手擔起神輿進行巡遊，但因每次巡遊都需要180名、算上替補人員則需要300名以上之多的擔手，自[[1913年|1913年]]（大正二年）起，神輿改以放上神輿車的形式進行巡遊。
: [[1982年|1982年]]（昭和五十七年），由於想再度恢復以人力擔起神輿的方式進行巡遊，當地志願者發起組織「土師祭神輿會」，並於[[1983年|1983年]]（昭和五十八年）舉辦首次「土師祭」，時隔70年再度以人力擔起神輿進行聖駕巡遊。此後，在八坂祭時千貫神輿將被放上神輿車進行巡遊，在土師祭時千貫神輿則是由人力擔起進行巡遊。
: 2018年（平成三十年），因土師祭中止，由人力擔起進行的千貫神輿聖駕巡遊也未能舉行。

; 幸運☆星神輿
: 幸運☆星神輿，是一座由前述的土師祭神輿會會長成田靖發起粉絲志願者企劃的、登場於2008年（平成二十年）土師祭的繪有漫畫《[[幸運☆星|幸運☆星]]》角色插畫的神輿。神輿原定為僅限2008年的限定企劃，之後「幸運☆星神輿神輿會」成立，改為每年都會舉行神輿的聖駕巡遊。神輿平日供奉在鷲宮神社[[鳥居|鳥居]]附近的「八坂神輿殿」中。

; 鷲宮堀內遺跡碑
: 為紀念發現[[繩紋時代|繩紋時代]]遺跡而立的碑。

: 光天之池自古時候便已位於此處，但水體在遭受多年沙土侵蝕後，長期被埋藏於地下。[[1999年|1999年]]（平成十一年），作為神社境內整備工事的其中一個項目，展開了光天之池的復原工事。傳說池子內的沙土被搬出後，形似龍一樣的霧隨著噴湧而出的水盤踞於池子上空，池子因而得名為「光天之池」。像其他境內神社一樣，光天之池設有刻有名稱的石碑和塗有朱漆的鳥居，前述的御池社坐鎮於池子後方深處。

== 交通 ==
* [[東武鐵道|東武]][[伊勢崎線|伊勢崎線]][[鷲宮站|鷲宮站]]出站后步行7分鐘
* [[東日本旅客鐵道|JR]][[東北本線|東北本線]]（[[宇都宮線|宇都宮線]]）[[東鷲宮車站|東鷲宮站]]出站后步行35分鐘
* {{link-ja|朝日自動車}}（[[公共汽車|公共汽車]]） 東鷲宮站西口 - 豐野社區中心方向 「鷲宮神社入口」下車 步行2分鐘 '''（白天時班次約為每小時2班）'''

此外，神社還設有[[停車場|停車場]]，但只能停放約20輛車，每當有祭事時會造成交通混雜。每當{{link-ja|正月的頭三天|正月三が日|正月的頭三天}}或土師祭等節日前後，神社周邊道路還會實施[[交通管制|交通管制]]，車輛亦無法駛入停車場。公共汽車在交通管制時，也會改為與平時不同的臨時迂迴路線，臨時公交站也設於遠離鷲宮神社路段之處。
