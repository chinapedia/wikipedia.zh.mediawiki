《'''終極鬥士3：贖罪'''》（{{lang-en|'''''Undisputed III: Redemption'''''}}）是一部2010年[[美國|美國]][[武俠片|武俠片]]，為[[伊薩克·佛倫亭|伊薩克·佛倫亭]]執導，為2006年電影《[[終極鬥士2|終極鬥士2]]》的續集。電影由[[史考特·艾金斯|史考特·艾金斯]]和{{le|米凱·夏儂·傑金斯|Mykel Shannon Jenkins}}主演。

該片於2010年4月17日在{{le|動作片國際影展|ActionFest}}上首映，並於2010年6月1日在美國以[[錄影帶首映|錄影帶首映]]發行。

== 劇情 ==
故事敘述繼前集的數年過後，右膝受傷的失去戰力的尤里·波卡因自卑而引退，而改至廁所打掃。但因黑幫老大兼監獄格鬥的大亨嘎卡的緣故，使得尤里再次復出參戰。後來，尤里被轉送往[[喬治亞|喬治亞]]；在那，他除了面對喜歡碎碎念並與自己不合的美國人托波外，還要和來自他國的囚犯們展開廝殺。

== 演員 ==
*[[史考特·艾金斯|史考特·艾金斯]] 飾 尤里·波卡（Yuri Boyka）：前俄羅斯監獄拳王，擅長綜合格鬥，因一級謀殺罪而入獄。數年前右膝蓋曾遭打斷而變得無法戰鬥。
*{{le|米凱·夏儂·傑金斯|Mykel Shannon Jenkins}} 飾 傑瑞科·「托波」·瓊斯（Jericho "Turbo" Jones）：來自美國的囚犯，因殺了兩個人而入獄，與尤里不合。
*{{le|馬克·伊凡諾|Mark Ivanir}} 飾 嘎卡（Gaga）：監獄格鬥賽的背後大亨，打算讓尤里參加格鬥賽。
*[[馬爾科·扎瑞|馬爾科·扎瑞]] 飾 勞爾·「哀痛」·奎涅斯（Raul "Dolor" Quinones）：來自哥倫比亞的囚犯，因走私和殺人而入獄。寇斯和雷佐的代表選手。
*{{le|赫里斯托·斯紹波夫|Hristo Shopov}} 飾 典獄長寇斯（Warden Kuss）：喬治亞監獄的典獄長。
*{{le|維納恩·多切夫|Vernon Dobtcheff}} 飾 瑞佐（Rezo）：世界監獄格鬥賽的主辦人。
*{{le|羅伯特·科斯坦卓|Robert Costanzo}} 飾 吉歐·費爾納特帝（Gio Farnatti）：嘎卡的大亨同夥，派出托波出賽。
*{{le|拉蒂夫·克勞德·多斯·桑托斯|Lateef Crowder dos Santos|拉蒂夫·克勞德}} 飾 安德里亞哥·席爾瓦（Andriago Silva）：來自巴西的囚犯。
*{{le|瓦倫丁·加內夫|Valentin Ganev}} 飾 典獄長馬爾科夫（Warden Markov）：俄羅斯監獄的典獄長。

== 續集 ==
據《''Movie Cricket''》報導，艾金斯證實，將會有第四部電影。2014年5月29日，艾金斯再度表示確實會有第四集，只是需要時間來實現，因為現在DVD電影較難生存；而他也透露自己相當喜歡飾演尤里·波卡<ref>{{cite web |url=http://www.moviecricket.com/exclusive-scott-adkins-confirms-boyka-undisputed-4-90605/ |title=Exclusive! Scott Adkins Confirms ‘Boyka: Undisputed 4′ |author=Fletcher, Billy |publisher=''Movie Cricket'' |date=2014-05-29 |accessdate=2014-06-07}}</ref>。

第四部電影《[[終極鬥士4|終極鬥士4]]》於2015年6月開始製作，艾金斯回歸擔任主演。拍攝於2015年8月初完成，並定於2016年推出。
