《'''終末的伊澤塔'''》（{{lang-ja|終末のイゼッタ}}）是一部由藤森雅也执导，亚细亚堂制作的原創動畫，于2016年10月1日在[[AT-X|AT-X]]首播。《伊泽塔》于2016年6月24日正式發表，並公開宣傳影片及官方網站，但並未提及製作團隊及聲優<ref>{{Cite web |url = http://gnn.gamer.com.tw/7/133057.html |title = 馳騁於戰場上的少女《終末的伊澤塔》原創動畫公開首波宣傳影片 |date = 2016-06-13 |publisher = 巴哈姆特電玩資訊站 |accessdate = 2016-07-16}}</ref>。直至同年9月2日，官方才首次公開本作的製作團隊及聲優名單等資訊<ref>{{Cite web |url = http://www.animatetimes.com/news/details.php?id=1472808637 |title = 『終末のイゼッタ』茜屋日海夏さん・早見沙織さんさんら主要声優16名が明らかに！　ボイス入りPV第4弾も解禁 |date = 2016-09-02 |publisher = Animate Times |language = ja |accessdate = 2016-09-02}}</ref>。

本作是藤森雅也第一次执导的长篇动画，在此之前也有不少动画制作的经验。本片的角色原案由被认为与[[abec|abec]]同一人的[[BUNBUN|BUNBUN]]设计，角色设计、总作画监督则为山下祐，Abec与山下祐皆与《刀剑神域》的制作有关联。山下祐一贯负责任的在动画制作过程中般保持人物设计出的形象。作为动作作画监督的竹内哲也所擅长表现的烟雾、物体碎片和枪支也是《伊泽塔》中常见的构成元素<ref>{{cite web|author1=Frog-kun|title=FEATURE: What's in an Animation? "Izetta, the Last Witch"|url=http://www.crunchyroll.com/anime-feature/2016/09/29-1/feature-whats-in-an-animation-izetta-the-last-witch|website=Crunchyroll|publisher=Crunchyroll, Inc.|accessdate=2016-10-02|language=英语|date=2016-09-30}}</ref>。受出资方制片人的要求，在配乐方面启用了未知瑠。

== 劇情介紹 ==
公元1939年9月1日，[[架空历史小说|架空]]的[[二战|二战]][[第二次世界大战欧洲战场|欧洲]]，专制国家日耳曼尼亞{{#tag:ref|以[[德意志帝国|德意志帝国]]和[[纳粹德国|纳粹德国]]为原形，后文简称日耳曼尼亞|group="注"}}掀起了战争，并很快侵占了欧洲大片领土。公元1940年，为配合非洲的戰事，日耳曼尼亞向南侵略，开始威胁阿尔卑斯山山麓的小国埃爾施塔特公國{{#tag:ref|后文简称公国，由今[[奥地利|奥地利]]的[[蒂罗尔州|蒂罗尔州]]和[[福拉尔贝格州|福拉尔贝格州]]以及[[列支敦士登|列支敦士登]]组成|group="注"}}。为抗击侵略，当时的公女菲涅代表大公秘密前往中立国维斯特里亚{{#tag:ref|今[[瑞士|瑞士]]|group="注"}}与不列颠尼亚{{#tag:ref|后文简称不列颠，今[[英国|英国]]|group="注"}}外相瑞德弗德会面，争取与不列颠建立同盟。

但因公国战事的爆发使得谈判未果，菲涅也被突然闯入的日耳曼尼亞军逮捕。随后菲涅与魔女伊泽塔相遇，伊泽塔是世界上最后在世的魔女，幼时便与菲涅相识。伊泽塔就帮助了菲涅安全回国。在了解当时时局后伊泽塔打算用她的魔法守卫公国。于是在魔法的介入下日耳曼尼亞的侵略计划受阻并打乱。隨後公国大公逝世，菲涅继承王位。伊泽塔同时透露魔女的魔力源自{{tsl|en|Ley line|地脉}}，只有在地脉密度较高之地才能发挥力量。在见识到魔法的力量下，日耳曼尼亞秘密开展了找寻魔女的新计划，并找到了曾經与公国结怨{{#tag:ref|中世纪[[猎巫|猎杀巫女时代]]在公国这一地域。一天佐菲救了受伤的王子，二人不久也陷入爱河。佐菲拒绝了成为妃子的请求，但许下如有需要會幫助王子的诺言。后来一次战争中佐菲信守诺言击退了敌国，二人有了短暂辛福时光。负有重伤的王子死后，王子鉴于佐菲的力量和惧于教廷的宗教权威下令处决了佐菲，佐菲最终含恨而死。|group="注"}}的白色魔女佐菲的骸骨。公国也相应的展開了宣传魔女魔法的计划，希望通过魔法的强大震慑日耳曼尼亞。伊泽塔的魔法事迹于是向全世界進行传播。此时公国暂时恢复平静。

不久在诺斯王国{{#tag:ref|今[[挪威|挪威]]|group="注"}}召开了同盟国大会，借助伊泽塔击沉日耳曼尼亞航母{{tsl|de|Drachenfels (Siebengebirge)|龙岩山|龙岩}}{{#tag:ref|以齐柏林号为原型|group="注"}}的展示，菲涅赢得了与会成员的信任，来自亚特兰大合众国的大使也向总统发电报请求出兵欧洲同時消灭日耳曼尼亞與埃爾施塔特。但同时因为魔法的强大使得人们既惊叹于魔法也产生了恐惧。在击沉航母时地脉的秘密也被日耳曼尼亞得知。日耳曼尼亞加快了进度，透过克隆技术将佐菲复活。并透过伊泽塔的血液唤醒了佐菲的意识。另一方面日耳曼尼亞秘密获取了可吸收地脉能量的半块魔石。但它的使用会减损寿命。一番准备后日耳曼尼亞卷土重来，伊泽塔与佐菲爆发战斗。因魔石的助力伊泽塔被佐菲击败。随后日耳曼尼亞攻占了公国首都，不久公国沦陷。菲涅与大臣逃往阿尔卑斯山藏匿，并奋力救回了伊泽塔。伊泽塔苏醒后，公国方齐克拿出了另半块魔石，伊泽塔为守卫公国接受了魔石。后來菲涅從投誠的貝格曼得知，日耳曼尼亞利用魔女的力量研发了类似核武器的戰略兵器，打算摧毁公国以威脅同盟國。但戰略兵器仍需要依靠地脉和佐菲的力量控制。

为阻止这场悲剧，带着魔石的伊泽塔再次跟佐菲爆发战斗。伊泽塔已和菲涅商量好，透过吸收欧洲的地脉能量使得魔法不再存在于欧洲，而菲涅赴维斯特里亚将她们的計畫告诉同盟国。这也是伊泽塔提出的作战策略。最终佐菲因身體是複製體而无法承受过多的地脉能量聚集而死亡，伊泽塔虽然活了下来但落下双腿残疾，魔法从此在欧洲消失。此后亚特兰大合众国{{#tag:ref|今[[美国|美国]]|group="注"}}与伏尔加联邦{{#tag:ref|以[[前苏联|前苏联]]为原形|group="注"}}两路出兵日耳曼尼亞。日耳曼尼亞皇帝自尽、帝国覆灭、欧洲最终恢复和平，此后菲涅致力于维护世界和平，伊泽塔则在幼时与菲涅相遇之地静养，菲涅不时会来看望伊泽塔。

== 登場人物 ==
=== 主要人物 ===
; {{nihongo|伊澤塔|イゼッタ|Izetta|聲：[[茜屋日海夏|茜屋日海夏]]}}
:生日：1925年10月6日{{#tag:ref|出自官方周邊|group="注"}}。
:祖先世代皆會承繼不可思議力量的魔女一族後裔，也是世界上最后在世的魔女。當手部接觸無機物後便會賦予魔力並能自由操縱。複數無機物的操縱也是可能的。正因自己魔女的身份，自幼便跟隨祖母於歐洲各地旅居。如果去掉魔女的身份，伊泽塔给人留下朴实、内心善良的形象。虽然略显天真，做起事来也干劲十足。童年時曾經和菲涅偶遇，因一次事件中受到菲涅的保护而对她产生了崇敬之情。两人也结为了朋友，并对菲涅有著深厚而复杂的情感（包含愛情）。可以说为了菲涅，伊泽塔愿意付出一切。

; {{nihongo|菲涅|フィーネ|Finé |聲：[[早見沙織|早見沙織]]}}

: 埃爾施塔特公國元首魯道夫三世的獨生女及繼承者。自小聰明又溫柔，得到国民的愛戴，自己认为她是依靠民众的愛而活下来的，勤于政务，致力于创造一个人人都有选择的和平世界。“Finé”一词有“最后、結束”的意思。菲涅给人留下坚强和果断的形象。同样地她对伊泽塔有深厚而复杂的情感（包含愛情）。在幼时保护过伊泽塔致使腹部留下了旧伤。每次遇到大事时会与伊泽塔进行商量讨论，平等的对待着伊泽塔。

== 本作用語 ==

:相當於現實世界的[[納粹德國|納粹德國]]及[[德意志帝國|德意志帝國]]，為了掌握歐洲的霸權、對周遭的國家行使侵略。
