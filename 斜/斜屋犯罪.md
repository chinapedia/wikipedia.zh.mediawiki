== 故事概要 ==
在[[日本|日本]]最北端、[[北海道|北海道]]的[[宗谷岬|宗谷岬]]的盡頭、可以俯瞰[[鄂霍次克海|鄂霍次克海]]的懸崖上，有座名叫「流冰館」的建筑，由於它自北向南傾斜，所以當地人又稱之為「斜屋」……

1983年飄雪的[[平安夜|平安夜]]，斜屋主人濱本幸三郎邀請親朋好友來斜屋出席聖誕派對。然而連續兩晚均發生殺人事件。詭異的是，在第一名死者遇害當晚，竟然有人目擊濱本的收藏品人偶「葛雷姆」凌空出現在三樓窗外……究竟真是受詛咒的人偶作祟，還是另有隱情？

== 登場人物 ==
=== 主要角色 ===
; {{nihongo|[[御手洗潔|御手洗 潔]]|{{ruby|御手洗|みたらい}} {{ruby|潔|きよし}}|Mitarai Kiyoshi}}
: 負責本案的牛越刑警因案情棘手，於是不得不拜托[[東京|東京]]的中村刑警。因此受中村所托前往流冰館解決事件。

; {{nihongo|[[石岡和己|石岡 和己]]|{{ruby|石岡|いしおか}} {{ruby|和己|かずみ}}|Ishioka Kazumi}}
: 和御手洗一同前往流冰館。

; {{nihongo|濱本 幸三郎|{{ruby|浜本|はまもと}} {{ruby|幸三郎|こうさぶろう}}|Hamamoto Kousaburou}}
: 「流冰館」主人，濱氏[[柴油|柴油]]股份有限公司創立人兼董事長。喜愛研究和收集西洋古董[[發條|發條]]玩具、機械人偶和[[天狗_(日本)|天狗]]面具。

: 濱本幸三郎的么女，擁有1.73米的高挑身材。性格非常強勢，是個非常任性的[[利己主義|利己主義]]者。

; {{nihongo|早川 康平|{{ruby|早川|はやかわ}} {{ruby|康平|こうへい}}|Hayakawa Kouhei}}
: 濱本家工作了二十余年的傭人。妻子是千賀子，有一個叫良江的女兒，但因交通事故去世。

; {{nihongo|早川 千賀子|{{ruby|早川|はやかわ}} {{ruby|千賀子|ちかこ}}|Hayakawa Chikako}}
: 早川康平的妻子。和丈夫一起在濱本家幫傭。

; {{nihongo|梶原 春男|{{ruby|梶原|かじわら}} {{ruby|春男|はるお}}|Kajiwara Haruo}}
: 濱本家的廚師。

; {{nihongo|菊岡 榮吉|{{ruby|菊岡|きくおか}} {{ruby|栄吉|えいきち}}|Kikuoka Eikichi}}
: 濱氏[[柴油|柴油]]的承包商——菊岡[[機軸|機軸]]公司的董事長，在幸三郎面前極盡討好之能事。
: 為了女人不惜一擲千金，但曾在周刊上因桃色糾紛而大大露過兩次面。
: 在流冰館的密室里被刺身亡。

; {{nihongo|相倉 久美|{{ruby|相倉|あいくら}} クミ|Aikura Kumi}}
: 菊岡的秘書兼情人，有著一張如小貓咪般嬌小可愛的臉龐。在流冰館和英子暗中針鋒相對。

; {{nihongo|上田 一哉|{{ruby|上田|うえだ}} {{ruby|一哉|かずや}}|Ueda Kuzuya}}
: 菊岡的司機。出生於[[岡山縣|岡山]]，在[[大阪|大阪]]長大，二十五歲時加入[[陸上自衛隊|陸上自衛隊]]，二十九歲時進入菊岡公司工作。為人不善與人交際。
: 在流冰館的密室里，被發現其被刺身亡，屍體的姿勢如同跳舞般怪異。

; {{nihongo|金井 道男|{{ruby|金井|かない}} {{ruby|道男|みちお}}|Kanai Michio}}
: 菊岡的部下。臉上一直都是滿臉堆笑的表情，使人甚至不能聯想他平常的表情。
: 妻子名叫初江。

; {{nihongo|金井 初江|{{ruby|金井|かない}} {{ruby|初江|はつえ}}|Kanai Hatsue}}
: 金井道男的妻子。

; {{nihongo|日下 瞬|{{ruby|日下|くさか}} {{ruby|瞬|しゅん}}|Kusaka Shun}}

; {{nihongo|戶飼 正樹|{{ruby|戸飼|とがい}} {{ruby|正樹|まさき}}|Togai Masaki}}
: [[參議員|參議員]]戶飼俊作的兒子，[[東大|東大]]生。

; {{nihongo|濱本 嘉彦|{{ruby|浜本|はまもと}} {{ruby|嘉彦|よしひこ}}|Hamamoto Yoshihiko}}
