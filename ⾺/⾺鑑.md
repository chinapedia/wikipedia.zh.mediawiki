'''馬鑑'''教授，{{BD|1883年||1959年|catIdx=Ma馬}}，[[字|字]]'''季明'''，[[浙江省|浙江省]][[鄞县|鄞县]]（今[[宁波市|宁波市]][[鄞州区|鄞州区]]）人，文史學者，[[香港中文大學|香港中文大學]]前校長[[馬臨|馬臨]]之父，曾任[[香港大學|香港大學]]中文系教授兼系主任。

==生平==
⾺鑑⽣於1883年，⼗七、八歲時，放棄科舉功名⽽投入[[南洋公學|南洋公學]]。1925年7⽉，年過四⼗的⾺鑑遠赴美國[[哥倫比亞大學|哥倫比亞大學]]隨[[杜威|杜威]]教授深造，獲[[教育學|教育學]]碩士學位。之後返回[[北京|北京]]，出任[[燕京大學|燕京大學]]國文系教授，並擔任國文系主任。⾺鑑為破除科舉的窒礙，特意開創格局，延聘新⼀代學者⽂⼈，如容庚、郭紹虞、沈尹默、俞平伯、錢玄同、⾺裕藻、謝婉瑩（冰⼼）等任教燕京⼤學中⽂系。<ref name="schhist">{{cite web |url=http://www.chinese.hku.hk/main/school-history/ |title=中文學院簡史}}</ref>

1935年⾺鑑教授應[[許地山|許地山]]邀請到⾹港⼤學任教。1941年，許地山逝世，教授一職由1940年到學院履新的史學大師[[陳寅恪|陳寅恪]]接任，系內行政事務則由馬鑑掌管。日佔時期，馬鑑返回內地，在成都[[燕京大學|燕京大學]]任教。戰後1946年5月重返香港大學，出任中文系教授兼系主任，至1950年退休。<ref>{{cite web |url=https://www.chunghwabook.com.hk/Public/files/upload/9789888488445.pdf |title=香港大學中文學院九十年}}</ref>

⾺鑑在任系主任期間，深覺中⽂系仍宥於記誦式之學習，倡議改⾰為系統的研究⽅式，奠下了中⽂學院的課程規模。⾹港英殖時期，教育獨尊英語，⾺鑑教授在⼤學入學試中提⾼國⽂程度的要求，從上⽽下，使中⽂得到應有的重視。為了破除⽂化隔膜，⾺鑑教授認為中⽂系的其中⼀個⽬標是溝通中西⽂化，他除在課程規劃上下功夫外，亦著⼒推動⽂化交流，包括藝術展覽、中英⽂化協會活動、中西學者學術演講等。<ref name="poster">{{cite web |url=http://web.chinese.hku.hk/poster/20170919.pdf |title=桃李不⾔，蹊徑垂蔭：馬鑑教授、馬蒙教授}}</ref>

⾺鑑退休後仍擔任中英⽂化協會榮譽會長，積極組織活動。

== 家庭==
⾺鑑有兄[[馬裕藻|馬裕藻]]、[[馬衡|馬衡]]，均為著名文史學者。[[馬衡|馬衡]]在1933年至1949年出任[[北京故宮博物院|北京故宮博物院]]的院長。<ref>{{cite web |url=https://www.hkcnews.com/article/202/%E6%95%85%E5%AE%AE%E8%88%87%E6%B8%AF%E4%BA%BA%E6%9C%89%E6%B7%B5%E6%BA%90 |title=故宮與港人有淵源 |date=2017-01-01 |publisher=眾新聞}}</ref> 弟[[馬准|馬准]]為文學家及風俗學家，弟[[馬廉|馬廉]]為民俗學家及藏書家。

馬鑑夫人鄭心如，長子馬咸、次子[[馬蒙|馬蒙]]、三子馬豫、幼子[[馬臨|馬臨]]，長女馬彤、次女馬彰、三女馬彬、幼女馬彥。其中次子[[馬蒙|馬蒙]]曾任[[香港大學|香港大學]]中文系教授兼系主任。幼子[[馬臨|馬臨]]為著名生物化學學者，曾任[[香港中文大學|香港中文大學]]校長。<ref name="ma-hku">{{cite web|url=http://paper.wenweipo.com/2005/05/04/OT0505040002.htm |title=馬鑑在港大 |publisher=[[文匯報|文匯報]] |author= |date=2005-05-04 }}</ref>
