'''丘雨勤'''（Edcon Gabriel Yau，{{bd|1979年|2月17日}}），香港體育主持人兼評述員，前加拿大[[新時代電視|新時代電視]]和[[加拿大中文電台|加拿大中文電台]]節目主持及體育新聞主播。曾任職於[[無綫電視|香港無綫電視]]，[[now寬頻電視|now寬頻電視]]體育頻道 [[Now寬頻電視體育平台|now Sports]]擔任體育記者和主持及[[樂視體育香港|樂視體育香港]]體育節目主持和及評述。現時任職於體通天下，主要負責粵語旁述[[NBA|NBA籃球]]賽事，並經常主持各個大型活動，更是[[NBA|NBA]]中國賽的御用賽場主持人。2018年加入香港香港體育播放平台[[體通天下|體通天下]]當廣東話評述員。

== 背景 ==
丘雨勤生於[[香港|香港]]，早年畢業於[[天佑小學|天佑小學]]<ref name="Edcon Yau's primary school, print screened on 2018-6-7">[https://drive.google.com/file/d/11DY1YWGTExTutc1muBihPz76UrJt_8uP/view?usp=sharing Edcon Yau's primary school, print screened on 2018-6-7]</ref>，在[[可風中學|可風中學]]完成中四課程後，隨家人移民到[[加拿大|加拿大]][[多倫多|多倫多]]，畢業於 Thornlea Secondary School 後入讀[[多倫多大學|多倫多大學]]電腦科學系，<ref name="Edcon Yau's alma mater, print screened on 2018-6-7">[https://drive.google.com/file/d/1mPdAETYqOn_AGv3beiNqOEtw-4aId46c/view?usp=sharing Edcon Yau's alma mater, print screened on 2018-6-7]</ref>曾任加拿大華人青少年體育會[http://www.ccyaa.org]([http://www.ccyaa.org CCYAA])董事會成員，其後更加入頤康基金會[http://www.yeehong.com/cn/foundation/index.php]任職通訊及公關部，協助頤康中心[http://www.yeehong.com ]籌募擴建經費，並同時在[[新時代電視|新時代電視]]主持節目。2005年回流香港，在香港[[無綫電視|無綫電視]]擔任體育記者。2007年底轉投[[now寬頻電視|now TV]]體育頻道[[Now寬頻電視體育平台|now Sports]]成為體育評述員。2016年底轉投[[樂視體育香港|樂視體育香港]]成為體育評述員。

== 傳媒生涯 ==
=== 加拿大 ===
1997年，當時仍在就讀高中的丘雨勤，得到[[新時代電視|新時代電視]]邀請加入，擔任[[多倫多猛龍|NBA多倫多猛龍隊]]首個中文電視節目《恐鳥縱橫NBA》節目主持。1999年，正式擔任《體育世界》主持。2002年，和[[林嘉華|林嘉華]]、[[潘宗明|潘宗明]]等著名體育主持人擔任[[2002年世界盃足球賽|2002年世界盃]]及[[2004年歐洲足球錦標賽|2004年歐洲國家盃]]的加拿大全國粵語評述。2005年，《體育世界》節目結束，他開始在[[加拿大中文電台|加拿大中文電台]]FM88.9擔任DJ主持著名節目《陽光早晨》，同時在[[新時代電視|新時代電視]]擔任不同電視節目的主持，包括《[[大城小聚|大城小聚]]》、《飲食男女》、《英語一分鐘》等，並兼任體育新聞主播一職，後來夥拍著名主持人葉子青拍攝旅遊節目《海洋三省自悠行》。
=== 香港 ===
==== TVB ====
2005年10月，丘雨勤從加拿大回流到香港，加入[[無綫電視|無綫電視]]綜藝及體育科，擔任體育記者和主持《體育世界》，並曾擔任多個大型運動會和比賽的現場主持及採訪，包括[[澳門東亞運動會|2005年澳門東亞運動會]]、[[NBA全明星賽|NBA全明星賽]]、[[澳門格蘭披治大賽車|澳門格蘭披治大賽車]]、超級工商杯籃球賽、[[世界女排大獎賽|世界女排大獎賽]]、[[長洲搶包山|長洲搶包山]]、2007年體操精英大匯演[http://www.gahk.org.hk/gee/main11.html]等。

==== now TV ====
2007年下旬，丘雨勤離開無綫電視，加入香港收費電視 [[now寬頻電視|now TV]] 體育頻道 [[Now寬頻電視體育平台|now Sports]] 成為體育主持和評述員。早期曾擔任[[国家橄榄球联盟|美式足球(NFL)]]、[[國家冰球聯盟|北美冰上曲棍球(NHL)]]、[[K-1|K-1綜合格鬥技]]等評述。由於他在外國長大，擁有不俗的英文溝通能力，經常被委派負責國外採訪，並多次到英國各大英超足球會進行現場記者工作。

2010-11球季 [[Now寬頻電視體育平台|now Sports]] 獲得 [[NBA|NBA]] 賽事直播權，丘雨勤夥拍星級籃球評述員[[徐嘉樂|徐嘉樂]]和[[張丕德|張丕德]]，負責 [[NBA|NBA]] 和 [[WNBA|WNBA]] 的直播粵語評述。2011年2月，更與[[張丕德|張丕德]]到[[美國|美國]][[洛杉磯|洛杉磯]]現場評述及採訪[[2011年NBA全明星赛|2011年NBA全明星赛]]。

==== 樂視體育香港 ====
2016-17球季，[[樂視體育香港|樂視體育香港]] 獲得 [[NBA|NBA]] 賽事直播權，丘雨勤夥拍星級籃球評述員[[徐嘉樂|徐嘉樂]]和[[張丕德|張丕德]]，負責 [[NBA|NBA]] 的直播粵語評述。

== 活動主持 ==
2007年開始，擔任在中國和台灣舉行的《NBA中國賽》現場主持，2008年起更成為《NBA Madness香港》推廣活動的現場主持，2014年夏天起為NBA在中國最大的球迷互動巡遊活動《NBA籃球國度》擔任現場主持，及後活動在2016年改為全國戶外五人制籃球賽，名為《NBA 5V5》。

除電視評述工作外，經常負責各類戶外大型活動的現場主持工作，主力負責外國體育明星到訪香港的活動主持，包括：

NBA 
- [[迈克尔·乔丹|Michael Jordan 米高佐敦]]、[[高比拜仁|高比拜仁]]、[[沙奎尔·奥尼尔|「大白鯊」奧尼爾]]、[[姚明|姚明]]、[[凯文·杜兰特|杜蘭特]]、[[卡梅隆·安东尼|「甜瓜」安東尼]]、[[克里斯·保罗|基斯保羅]]、[[特雷西·麦格雷迪|麥基迪]]等

足球
- [[基斯坦奴·朗拿度|C.朗拿度]]、[[歐洲足協|歐洲足協]]主席[[柏天尼|柏天尼]]、[[曼聯|曼聯]]的[[朴智星|朴智星]]和[[里奧·費迪南|里奧·費迪南]]、[[利物浦足球俱樂部|利物浦]]亞洲之旅香港站等

網球
- [[麥根萊|麥根萊]]、[[張德培|張德培]]、[[李娜_(网球运动员)|李娜]]等。

另外，亦包括在中港台舉行的體壇盛事及大型綜合性運動會，除擔任主持人，亦曾出任現場英文播報員。其中有：
－[[2010年广州亚运会|2010年廣州亞運會]]
－[[2011年世界大學生夏季運動會|2011年深圳世界大學生運動會]]
－[[2011年亞洲籃球錦標賽|2011年武漢亞洲籃球錦標賽]]
－在澳門舉行的[[2011年美國傳奇球星亞洲之旅澳門站|2011年美國傳奇球星亞洲之旅澳門站]]
－在廣州舉行的2012年[[NBA呈現：Jam_Live!_閃耀群星|NBA呈現：Jam Live! 閃耀群星]]
－[[2013澳門威尼斯人國際群星籃球對抗賽|2013澳門威尼斯人國際群星籃球對抗賽]]
－[[2013年CBA全明星週末|2013年CBA全明星週末]]
－[[汇丰冠军赛|滙豐冠軍賽高爾夫球賽]](2013-2017)
－姚基金慈善賽(2016-2017)
