'''丘逢甲'''（{{bd|1864年|12月26日|1912年|2月25日|catIdx=Q丘}}）<ref>[[雍正|雍正]]年間，不少「[[丘姓|丘姓]]」者因[[避諱|避諱]][[孔子|孔子]]之名，改成了普遍常見的「[[邱姓|邱姓]]」，故丘逢甲[[宗族|宗族]]姓「丘」或「邱」者皆有。</ref>，[[譜名|譜名]]'''秉渊'''，[[表字|字]]'''仙根'''，一字'''吉甫'''<ref> [http://books.google.com/books?id=s38sAAAAYAAJ&pg=PA60#v=onepage&q&f=false 光緒己丑科會試同年齒錄]</ref>，[[号|号]]'''蛰仙'''、'''仲阏'''、'''柏莊'''、'''华严子'''、'''痛哭生'''、'''海东遗民'''、'''南武山人'''，[[齋號|齋號]]'''蛰庵'''、'''自強不息齋'''、'''嶺雲海日樓'''。[[晚號|晚號]]'''倉海君'''，又作'''滄海君'''。出生於[[福建省|福建省]][[臺灣府|臺灣府]][[淡水廳|淡水廳]][[銅鑼鄉_(台灣)|銅鑼灣]]（今[[苗栗縣|苗栗縣]][[銅鑼鄉_(台灣)|銅鑼鄉]]竹森村），籍屬[[彰化縣_(清朝)|彰化縣]]。[[清朝|清朝]]官員、[[詩人|诗人]]和[[教育家|教育家]]。

== 生平 ==
[[File:丘逢甲進士碑旗桿座.JPG|thumb]]丘逢甲紀念公園內的進士碑旗桿座，鐫刻：「欽點工部虞衡司主事邱逢甲立」]]
因父親[[丘龍章|丘龍章]]受聘講授[[漢文|漢文]]，丘逢甲於[[同治|同治]]三年（1864年）在[[福建省|福建省]][[臺灣府|臺灣府]][[淡水廳|淡水廳]][[竹南二堡|苗栗堡]]堡銅鑼灣（今[[臺灣|臺灣]][[苗栗縣|苗栗縣]][[銅鑼鄉_(臺灣)|銅鑼鄉]]竹森村、客家庄）出生。因生逢[[甲子|甲子]]年，故稱逢甲。

幼聰穎，六歲能文，14歲即考取秀才，得到[[巡撫|巡撫]][[丁日昌|丁日昌]]的賞識，譽為“[[東寧_(台灣古名)|東寧]]才子”。25歲赴[[福州|福州]]鄉試中舉。次年（[[光緒|光緒]]十五年（1889年）），至[[燕京|燕京]]赴試，考中[[进士|进士]]，同年五月，著[[主事|主事]]，分部学习<ref>《大清德宗同天崇运大中至正经文纬武仁孝睿智端俭宽勤景皇帝实录》（卷二百七十）：光绪十五年。己丑。五月。丙午朔。……○乙卯。引见新科进士。……叶祥麟、刘如煇、鲍琪豹、士魁、徐心泰、王世琪、罗凤华、周来宾、陈庚经、陈泽霖、曹允源、夏声乔、丁宝铨、贺涛、张其镃、张景旭、高树、瑞贤、丁述曾、沈瑜宝、赖绍濂、邹洪纬、陈恩寿、鲍心增、赵蔚坊、陈春瀛、袁用宾、王铁珊、王为相、魏有声、唐烜、金蓉镜、赵继泰、陈昌昙、高增融、何文澜、孟滢、张颉辅、梁于渭、李登云、钱鸿文、林国赞、万和锡、罗厚焜、刘锦荣、张允言、曾光岷、熊起渭、王敟、陈金台、杨芾、陈宝森、郭集琛、黄榜书、刘华、朱怀新、易贞、费道纯、唐宗海、于宗潼、王祖武、魏秀琦、雷光第、陈三立、毛慈望、张用宾、刘宝森、周秉道、高永孝、任廷扬、连培型、赵曾棣、钰昶、邱逢甲、张立德、武瀛、武镳、赵金寿、周朝槐、郑篯、俱著分部学习。</ref>，被任命为[[工部|工部]]虞衡司[[主事|主事]]。但丘逢甲无意在[[京師|京師]]做官，以「親老需侍」為由返臺，受[[臺灣|臺灣]]望族，原籍[[漳州|漳州]]的[[神岡三角仔呂家|筱雲呂氏]]（今稱[[神岡三角仔呂家|神岡呂家]]）禮聘講授[[漢文|漢文]]。其兄丘先甲也因辦理[[鄉勇|鄉勇]][[團練|團練]]、開墾土地等因素，舉家遷往[[臺中|臺中]]。丘逢甲曾到[[臺中市|臺中]]「衡文書院」任主讲，後又於[[臺南市|臺南]]和[[嘉義縣_(清朝)|嘉義]]舉辦新式学堂。

===臺灣民主國===
光緒二十年（1894年），[[甲午战争|甲午战争]]爆發，丘逢甲奉旨督辦[[團練|團練]]。次年，清軍戰敗，[[李鴻章|李鴻章]]與[[日本|日本]]簽訂《[[馬關條約|馬關條約]]》，割讓臺灣，臺灣人民激憤，丘逢甲呈文，反對割臺。條約生效後，丘逢甲倡立民主國，率紳民奉[[藍地黃虎旗|旗]]、璽及總統印，獻於巡撫[[唐景崧|唐景崧]]，并聲援[[义军|义军]]反抗。5月23日，唐景崧發表《[[臺灣民主國|臺灣民主國]]獨立宣言》。5月25日，臺灣民主國成立，年號“永清”，唐景崧出任[[總統|總統]]，[[劉永福|劉永福]]為[[大將軍|大將軍]]，[[李秉瑞|李秉瑞]]為軍務大臣，丘逢甲為[[副總統|副總統]]兼[[團練使|團練使]]，統領[[義勇軍|義勇軍]]。<ref>《臺灣通史·[[s:臺灣通史/卷4|卷四]]》：初二日，紳士邱逢甲率人民等公上大總統之章。景崧受之，建元永清，旗用藍地黃虎。以兵部主事邱逢甲為義勇統領，禮部主事李秉瑞為軍務大臣，刑部主事俞明震為內務大臣，副將陳季同為外務大臣，道員姚文棟為遊說使，使詣北京，陳建國情形。</ref>唐景崧駐守[[臺北|臺北]]，丘逢甲奉命駐兵[[南崁|南崁]]策應。<ref>《臺灣通史·[[s:臺灣通史/卷4|卷四]]》：當是時，全臺之兵，土、客、新、舊為數三百數十營，每營三百六十人。景崧既駐臺北，以逢甲率所部戍附近，備策應。</ref>

5月29日，日軍登臺，進佔[[基隆|基隆]]，守軍不敵。6月4日，總統唐景崧率先棄職，乘[[德|德]]商輪船逃往[[廈門|廈門]]。丘逢甲隔天聞訊後，見局勢不可為，遂棄義勇軍死生於不顧，攜家眷逃到[[廣東|廣東]][[嘉應州|嘉應州]]。<ref>《臺灣通史·[[s:臺灣通史/卷4|卷四]]》：十三日，日軍以一大隊迫[[獅球嶺|獅球嶺]]。臺人請景崧駐八堵，為死守計，不從。營官李文魁馳入撫署，大呼曰：「獅球嶺亡在旦夕，非大帥督戰，諸將不用命」。景崧見其來，悚然立，舉案上令架擲地曰：「軍令俱在，好自為之」。文魁側其首以拾，則景崧已不見矣。景崧既入，攜巡撫印奔滬尾，乘德商輪船逃。將出口，砲臺開砲擊之；適德兵艦泊附近，以其擊己船也，亦開砲擊。當是時潰兵四出，劫藩庫，焚撫署，土匪亦乘發，鬥死者五百餘人，哭聲滿巷。如是兩晝夜。林維源、林朝棟、邱逢甲相率去。</ref>離臺前有詩：「 [[李鴻章|宰相]]有權能割地，孤臣無力可回天；扁舟去作[[范蠡|鴟夷子]]，回首河山意黯然。」丘逢甲死後十餘年，此詩方由其子[[丘念臺|丘念臺]]公布，傳頌一時，但仍無法改變其臨危脫逃之惡名。

丘逢甲內渡大陆後，先後主講[[潮州|潮州]]韓山書院、[[潮陽|潮陽]]東山書院、[[澄海|澄海]]景韓書院，並與三弟樹甲共同成立嶺東同文學堂。

[[File:丘逢甲書法.jpg|thumb]]

1896年6月丘逢甲首次到[[廣州|廣州]][[萬木草堂|萬木草堂]]，其時草堂已被清廷取締，屋宇也早也歸還丘家。丘逢甲在此先後拜見了[[廣東巡撫|廣東巡撫]][[許振禕|許振禕]]、[[翰林院|翰林院]]編修[[劉葆貞|劉葆貞]]、[[菊坡精舍|菊坡精舍]]山長梁詩五等。許仙屏上奏朝廷，陳述丘逢甲抗日護台的良苦用心和義烈舉動，請求朝廷褒揚錄用，朝廷批覆「歸籍海陽」。這也是丘逢甲第一次到廣州，在邱氏書室寫下了《[[鎮海樓|鎮海樓]]》、《珠江書感》等20多首詩。1903年，被中国现代教育奠基人何子渊等聘为兴民学堂首任监督。光緒三十年（1904年）再赴廣州，任廣東學務公所參議，長住萬木草堂。三十二年（1906年），被[[兩廣總督|兩廣總督]][[岑春煊|岑春煊]]聘為兩廣學務處視學、廣州府中學堂[[監督|監督]]、商業職業學校監督。1907年，与辛亥革命元老、同盟会嘉应州主盟人何子渊等秘密筹划潮州黄冈起义，并参与营救遇险的革命党人。三十四年（1908年），被推為廣東教育總會會長。[[宣統|宣統]]元年（1909年），[[廣東諮議局|廣東諮議局]]成立，丘逢甲當選[[議員|議員]]，既而被推舉為副議長。

丘逢甲積極支持[[姚雨平|姚雨平]]、[[邹鲁|邹鲁]]等人的反清[[革命|革命]]活动。宣統三年（1911年），[[武昌起義|武昌起義]]爆發，各省響應，廣東宣佈獨立，推[[胡漢民|胡漢民]]為都督。丘逢甲被選為代表，選舉[[孫文|孫文]]為[[中華民國|中華民國]]臨時大總統。

[[民國紀年|民國]]元年（1912年），丘逢甲因積勞成疾，請假南返。2月25日，卒於[[廣東|廣東]][[鎮平縣|鎮平縣]]（今[[梅州市|梅州市]][[蕉嶺縣|蕉嶺縣]]）員山里[[門第|門第]]，遺言「葬須南向，吾不忘臺灣也」。<ref>《民國丘倉海先生逢甲年譜》</ref>丘逢甲出殯時，“執紼而哭者數千人”，有輓聯曰：“憶當年，禍水滔天，空拼九死余生，雙手難支新建國；病今日，大星墜地，只剩二三遺老，背面同哭故將軍。”

== 紀念 ==
[[File:Chiu_Feng_Chia_Memorial_Hall,_Feng_Chia_University.jpg|thumb]]
1961年[[中臺灣|中臺灣]]士紳由丘逢甲之子[[丘念台|丘念台]]及[[監察委員|監察委員]][[楊亮功|楊亮功]]等議，在中部地區興建一所大學，為紀念丘逢甲而命名為逢甲工商學院。1961年，學校正式創建於[[臺中市|臺中市]]北屯區大坑今[[中台科技大學|中台科技大學]]校址，大坑原為丘逢甲之兄丘先甲開發居住及墓園所在，丘逢甲本人也曾短期待過此地。而後[[逢甲工商學院|逢甲工商學院]]遷往西屯區現址，1980年改制為[[逢甲大學|逢甲大學]]。

[[2006年5月|2006年5月]]，丘逢甲在广东省[[蕉岭县|蕉岭县]]文福镇逢甲村的故居被[[中華人民共和國|中華人民共和國]][[中華人民共和國國務院|国务院]]公布为[[全国重点文物保护单位|全国重点文物保护单位]]<ref>陈斌华，[http://news.xinhuanet.com/tai_gang_ao/2006-09/18/content_5106845.htm 抗日护台志士丘逢甲故居获定为全国文保单位]，新华网</ref>。

[[丘逢甲墓|丘逢甲墓]]至今尚存，和[[逢甲纪念亭|逢甲纪念亭]]，皆為[[梅州市文物保護單位|梅州市文物保護單位]]。<ref>[http://www.ccnh.cn/bl/wxyc/wenbml/3261121294.htm 梅州市各级文物保护单位] {{webarchive|url=https://web.archive.org/web/20160306163822/http://ccnh.cn/bl/wxyc/wenbml/3261121294.htm |date=2016-03-06 }} 中國遺產網</ref>

== 爭議事件 ==
*关于是否逃亡
**反割臺運動期間，丘逢甲在[[馬關條約|馬關條約]]簽訂後屢次刺血上書，反對割臺；[[台灣民主國|台灣民主國]]成立時痛哭表示：「吾臺孤懸海外，去朝廷遠，朝廷之愛吾臺，曷若吾臺人之自愛。官兵又不盡足恃，一旦變生不測，朝廷遑復相顧。惟人自為戰，家自為守耳。否則禍至無日，祖宗廬墓擲諸無何有之鄉，吾儕其何以為家耶？」<ref>清稗類鈔 忠藎類</ref>；[[乙未戰爭|乙未戰爭]]剛開戰，唐景崧逃亡。丘還痛罵唐景崧不戰而逃，宣稱：「吾臺其去矣！误我臺民一至此极！景崧之肉，其足食乎！」<ref>《丘逢甲與臺灣歷史文化學術研討會文集》，第90頁</ref>，結果在七月末（景崧六月三日內渡，逢甲六月初五）內渡<ref>丘逢甲傳p71 徐博東 黃志平</ref>逃往[[福建|福建]][[泉州|泉州]]<ref name="丘傳" />。[[連雅堂|連雅堂]]在《臺灣通史》便作總評為「成敗論人，吾所不喜，獨惜其為[[吳湯興|吳湯興]]、[[徐驤|徐驤]]所笑爾。」<ref>《臺灣通史·[[s:臺灣通史/卷36|卷三十六]]·列傳八·丘逢甲傳》：连横曰：逢甲既去，居于嘉应，自号仓海君，慨然有报秦之志。观其为诗，辞多激越，似不忍以书生老也。成败论人，吾所不喜，独惜其为吴汤兴、徐骧所笑尔。</ref>
**有中國學者在書中表示，丘逢甲曾經率兵抗戰二十日。<ref>廖隆盛〈剖雲行日：丘逢甲傳〉</ref><ref>《近代中國》48期，1985年8月，頁218-224，見頁221</ref>

*关于是否支援唐景崧
**根據<讓台記>及<中國近代史資料叢刊續編>所載述，[[日軍|日軍]]登臺後，唐景崧曾經兩次派[[電報|電報]]「千急速速赴援」、「萬急速速赴援」（讓台記中電文內容略有不同 「萬急急速赴援」 與 「千急急萬急急速赴援」）要求丘逢甲北上支援<ref>讓台記</ref>；<ref>（中国近代史资料丛刊续编）第12册p68p69</ref>，但丘未予理會。
**其一，丘逢甲已成為日軍搜捕的重要對象，藏身困難。<ref>《中日战争》（中国近代史资料丛刊续编）第12册，中华书局1996年版，第74—75页</ref>。其二，幫辦呂賡虞的叛變投敵使義軍難以為繼。<ref>林蔾《台灣名人傳》：「日軍懸賞六十萬元來購買丘逢甲的頭。」丘逢甲〈重送頌臣〉詩中言及「人情易翻覆，交舊成鬼蜮」即指舊部呂某叛投日人事</ref>。其三，也是最主要的，乃是丘逢甲深感自己无力回天，眼前又找不到像[[郑成功|郑成功]]那样的伟人，实难扭转危局，只有等待时机再图恢复。<ref>《丘逢甲文集》，花城出版社1994年版，第251—252页</ref><ref>丘逢甲離台內渡考  戚其章</ref>

*關於是否貪汙
**丘氏內渡之際，傳聞其[[貪污|貪污]]，挾帶十萬兩[[银两|白銀]]之巨額公款而行<ref name="臺灣通史">《臺灣通史·[[s:臺灣通史/卷36|卷三十六]]·列傳八·丘逢甲傳》：「...十三日，日軍迫獅球嶺，景崧未戰而走，文武多逃，逢甲亦挾款而去，或言近十萬云。」</ref><ref name="四魂集">《四魂集，魂南集》：「...林朝棟、 楊汝翼、主事邱逢甲皆擁巨資， 棄師潛逃。」</ref><ref name="讓臺記">《讓臺記》：「...丘逢甲，各領餉銀三個月。楊汝翼拔隊至[[大甲區|大甲]]，聞變逗留不前，乘間逃回福州，將餉銀席捲而去，於勇在臺輾轉無依。」</ref><ref name="臺海思痛錄">《臺海思痛錄》：「...數月之間，逢甲領去官餉銀十餘萬兩，僅有報成軍一稟而已。」</ref>；但因無實據，亦有持不同意見者<ref name="嶺海微飆68">《我的奮鬥史（嶺海微飆）》第68頁</ref><ref name="蒋君章">《丘逢甲的一生》第2-3頁</ref><ref name="郭兆華">《丘逢甲先生的生平》第35-39頁</ref>，{{or|其貪污之事，正反各具一詞。有人認為可能是由寄鶴齋詩話中所述舉家內渡後因領不到軍餉使得部隊怒燒邱家屋舍事件，跟台北官署最後還剩十餘萬兩的兩件史事經傳聞混合與誇張所致。}}

== 歷史評價 ==
=== 正面評價 ===
*丘逢甲少年得志，卻棄官返台從事教育工作；乙未割台時，他首倡獨立抗日；內渡大陸後則獻身推廣新式教育，為國家培養元氣。終其一生始終對國家抱有很高的期望，具有強烈的愛國情操<ref name="丘傳">《丘逢甲傳》第214頁 </ref>。
*丘逢甲的詩文曾獲得諸多學者高度的評價：[[錢仲聯|錢仲聯]]曾評其《嶺雲海日樓詩鈔》曰「七律一種，開滿勁弓，吹裂鐵笛，真成義軍舊將之詩<ref>《夢苕庵詩話》第89頁</ref>。」[[柳亞子|柳亞子]]謂「時流竟說[[黃遵憲|黃公度]]，英氣終輸倉海君，戰血臺澎心未死，寒笳殘角海東雲<ref>《柳亞子詩選輯》第30頁</ref>。」[[梁啟超|梁啟超]]則譽為「詩界革命鉅子」、「天下健者」<ref>《梁任公先生年譜長編初稿》第94頁</ref>。
