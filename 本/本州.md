'''本州'''（{{jpn|j=本州|hg=ほんしゅう|rm=Honshū|f=y}}，{{Audio|ja-Honshu.ogg|發音}}）是[[日本|日本]]最大[[岛|岛]]，也是[[日本地理|日本本土]]四島之一，位于[[日本列岛|日本列岛]]中部，向北与[[北海道岛|北海道岛]]隔[[津轻海峡|津轻海峡]]相望，向南与[[四国|四国岛]]隔[[濑户内海|濑户内海]]相对，向西南与[[九州_(日本)|九州岛]]以[[关门海峡|关门海峡]]和[[丰后水道|丰后水道]]相隔。身為[[島國|島國]]日本最主要的島嶼，因而得名「本州」，古代則稱為「秋津島」或「秋津洲」。假如不将[[澳大利亞洲|澳洲]]算做島嶼，本州是世界第七大岛。

== 地理 ==
{{see|日本地理}}
本州从东北向西南延伸全长约1,300[[公里|公里]]，宽度在50至230公里之间，其总面积为231,216.49[[平方公里|平方公里]]（2009年數據），占日本总面积的60%，海岸线則全长5,450公里。山岳地帶橫亙本州，構成整個島嶼的脊梁，此也導致本州北方與南方的氣候差異性大。主要气候为[[季风气候|季风气候]]，从北方的[[温带季风气候|温带季风气候]]递变为南方的[[亚热带季风气候|亚热带季风气候]]。本州是[[火山岛|火山岛]]，岛上迄今仍有活跃[[火山|火山]]，[[地震|地震]]频繁。全島最高点[[富士山|富士山]]亦為日本最高峰，高3,776米。本州全境[[河流|河流]]密布，其中[[信浓川|信浓川]]是全島乃至於日本最长的河流。[[平原|平原]]主要集中於沿海，大部分為[[沖積平原|沖積平原]]，其中[[關東平原|關東平原]]最大。

== 人文環境 ==
[[日本歷史|從古至今]]，本州都是日本的中樞地帶，無論是政治、經濟或文化的發展上，均以本州為主要舞台。根據2005年的日本[[國勢調查_(日本)|國勢調查]]資料，本州與附屬島嶼的总人口1億334萬人<ref>[http://www.e-stat.go.jp/SG1/estat/List.do?bid=000001007609&cycode=0 平成17年 都道府県・市区町村別統計表（一覧表）]（[[獨立行政法人|獨立行政法人]]{{link-ja|統計中心 (日本)|統計センター|統計中心}}「[[平成|平成]]17年國勢調査」）</ref>，逾日本人口的80%。日本28個[[日本城市人口排名|人口超過50萬的都市]]中<ref>此處將[[東京都區部|東京都區部]]劃為一個都市。</ref>，有23個位於本州。

而自關東平原起至[[關門海峽|關門海峽]]的本州[[太平洋|太平洋]]側沿岸地區，是當代日本的發展重心，日本[[三大都市圈|三大都會區]]——[[首都圈_(日本)|首都圈]]（[[東京|東京]]都會區）、[[近畿圈|近畿圈]]（[[京阪神|京阪神]]）與[[中京圈|中京圈]]（[[名古屋|名古屋]]都會區）均位於此軸線上，其中[[日本首都|首都]]東京所在的首都圈人口達3千7百萬，聚集了全日本約1/4的人口；[[太平洋工業帶|太平洋工業帶]]的大部分區域亦座落於此，其中包括[[京濱工業地帶|京濱]]、[[中京工業地帶|中京]]、[[阪神工業地帶|阪神]]等3大工業地帶。

: [[青森县|青森县]]、[[岩手县|岩手县]]、[[宫城县|宫城县]]、[[秋田县|秋田县]]、[[山形县|山形县]]、[[福岛县|福岛县]]
; [[關東地方|關東地方]]
: [[茨城县|茨城县]]、[[栃木县|栃木县]]、[[群马县|群马县]]、[[埼玉县|埼玉县]]、[[千叶县|千叶县]]、[[东京都|东京都]]、[[神奈川县|神奈川县]]

: [[新潟县|新潟县]]、[[富山县|富山县]]、[[石川县|石川县]]、[[福井县|福井县]]、[[山梨县|山梨县]]、[[长野县|长野县]]、[[岐阜县|岐阜县]]、[[静冈县|静冈县]]、[[爱知县|爱知县]]

; [[中国地方|中国地方]]

本州与北海道、九州、四国等其他日本本土島嶼之间均有[[跨海大橋|隧道或桥梁]]相连，包括[[青函隧道|青函隧道]]、三個{{link-ja|本州四國聯絡橋|本州四国連絡橋}}、[[關門橋|關門橋]]與[[關門隧道|關門隧道]]。
