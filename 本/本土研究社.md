'''本土研究社'''（{{lang-en|'''Liber Research Community'''}}），是一個[[研究|研究]]網絡、[[出版社|出版社]]<ref name="基道">{{cite web |work =[[基道|基道]] |title=出版社：本土研究社|url = http://www.logos.com.hk/bf/acms/content.asp?site=logosbf&op=search&type=product&match=exact&field=publisher&text=%u672C%u571F%u7814%u7A76%u793E|}}</ref>，創立於2009年<ref>{{cite web |author1=本土研究社 |title=專頁資訊|url = https://www.facebook.com/localresearch/info/?entry_point=page_nav_about_item&tab=page_info|agency=[[Facebook|Facebook]]|}}</ref>，由一群熱心關注[[香港|香港]][[發展|發展]]及[[社會問題|社會問題]]的[[研究員|研究者]]組成，2013年正式註冊成非牟利機構<ref>{{Cite news|url=https://theinitium.com/article/20170307-hongkong-local-research-community/|title=象牙塔外的本土研究，他們要用細節打魔鬼｜端傳媒 Initium Media|last=發自香港|first=端傳媒記者 陳倩兒|newspaper=端傳媒 Initium Media|accessdate=2017-12-12|language=zh-HK}}</ref>。

他們結合民間進步力量，以[[政治學|政治學]]、[[地理學|地理學]]、[[社會學|社會學]]、[[哲學|哲學]]、[[語言學|語言學]]、[[文化研究|文化研究]]、[[傳媒|傳媒]]、[[規劃|規劃]]等學科[[知識|知識]]組成跨學科､具政策及在地視野，並透過研究、調查及[[報告|報告]]的方式，推動[[重建|重建]]、[[規劃|規劃]]、[[發展|發展]]、[[政制|政制]]、[[鄉郊|鄉郊]]、[[保育|保育]]、融合、[[香港經濟|香港經濟]]等香港議題。<ref>{{cite web |author1=本土研究社 |title=關於我們|url = https://localresearchcommunity.wordpress.com/about|}}</ref>

過去發表的研究包括：土地供應研究、棕土、短期租約土地（Short-term Tenancy）、房屋研究、丁屋政策、農業研究、食物安全、中港關係研究等。

==傳媒報道==

=== 郊野公園土地建屋問題 ===
2017年1月18日，時任行政長官[[梁振英|梁振英]]發表施政報告，建議利用[[郊野公園|郊野公園]]內小量生態價值不高、公眾享用價值較低、位於邊陲地帶的土地用作公營房屋。本土研究社成員陳劍青指出，做法違背《[[跨越2030年的規劃遠景與策略|跨越2030年的規劃遠景與策略]]》（簡稱《2030+》）意向，皆因《2030+》中，政府從未提及任何郊野公園發展，而時任發展局局長陳茂波曾承諾，任期內不會「考慮」發展任何郊野公園，批評一旦發展郊野公園，會觸發骨牌效應。<ref name=":0">{{Cite news|url=http://www.inmediahk.net/node/10471036|title=【施政報告】郊野公園土地建屋  本土研究社憂觸發骨牌效應 {{!}} 獨媒報導 {{!}} 香港獨立媒體網|newspaper=香港獨立媒體網|accessdate=2017-12-09}}</ref>

陳劍青亦指出，施政報告提到將佔地500公頃的[[紅花嶺|紅花嶺]]納入為郊野公園，但其實紅花嶺即使不劃為郊野公園，本身已是具生態價值的山林資源，而用作起樓的郊野公園邊陲卻會受到實質破壞。<ref name=":0" />

2017年11月24日，本土研究社成員楊夏至指出，美國自然環境學家[[湯博立|湯博立]]博士（Prof. Lee M. Talbot）當初為香港訂立郊野公園版圖，湯博立早於1965年已經警告，未來或會有人打郊野公園的主意，建議政府需要用最大力度去保護郊野公園。楊夏至又指出，現時若要改變郊野公園土地用途，必須通過許多程序，而政府當初定下這些程序，正正為了保護郊野公園，故質疑政府要求房協就發展郊野公園邊陲地進行可行性研究，做法已違反設立郊野公園的原意。<ref>{{Cite news|url=https://www.hkcnews.com/article/8471/%E9%83%8A%E9%87%8E%E5%85%AC%E5%9C%92%E9%82%8A%E9%99%B2%E5%9C%B0-%E5%A4%A7%E6%AC%96%E9%83%8A%E9%87%8E%E5%85%AC%E5%9C%92-%E6%88%BF%E5%8D%94-8471/%E6%88%BF%E5%8D%94%E5%8A%83120%E5%85%AC%E9%A0%83%E9%83%8A%E9%87%8E%E5%85%AC%E5%9C%92%E9%82%8A%E9%99%B2%E5%9C%B0%E7%A0%94%E7%A9%B6%E5%BB%BA%E5%B1%8B-%E7%92%B0%E4%BF%9D%E5%B0%88%E5%AE%B6%E6%89%B9%E9%81%B8%E5%9D%80%E6%9C%AA%E8%AB%AE%E8%A9%A2%E3%80%81%E5%8A%83%E7%95%8C%E9%9A%A8%E6%84%8F|title=眾新聞 {{!}} 房協劃120公頃郊野公園邊陲地研究建屋　環保專家批選址未諮詢、劃界隨意|newspaper=眾新聞|accessdate=2017-12-09|language=zh-HK}}</ref>

=== 棕土 ===

2015年12月27日，本土研究社發表「棕土報告」。團體根據Google Earth衞星圖片，發現全港棕土面積達1192公頃，面積相當於3.7個啟德發展區，較「[[公共專業聯盟|公共專業聯盟]]」及「[[土地正義聯盟|土地正義聯盟]]」於2012年發現803公頃增加近半<ref>{{Cite web|url=https://hk.news.appledaily.com/local/daily/article/20120309/16140279|title=803公頃棕土荒廢變車場回收場　促發展代填海|accessdate=2017-12-07|work=Apple Daily 蘋果日報}}</ref>。團體發現，全港棕土集中當中元朗、北區及屯門等九個地方，大多有道路相連，惟欠規劃及管理差劣，在發現的棕土當中，用作「貨櫃場」及「停車場」的佔地面積上升至477公頃。認為棕土問題有明顯擴散迹象。<ref>{{Cite news|url=http://hk.on.cc/hk/bkn/cnt/news/20151227/bkn-20151227134701153-1227_00822_001.html|title=規劃漏招致棕地如癌擴散　團體倡整合建公屋|newspaper=on.cc 東網|accessdate=2017-12-07|language=zh-hk}}</ref>

本土研究社指出，房屋署曾於2012年進行內部研究，探討利用元朗工業邨西面的橫洲棕土發展公營房屋的可行性，但計劃最後無疾而終，其中一個困難很可能是地區鄉紳勢力反對。然而，團體認為，即計劃發展範圍涉及私人土地，政府亦有權運用《收回土地條例》賦予的權力收地，將之規劃作公共用途（如公營房屋），另外亦可考慮用官地換取私人土地，或收購某地段的私人土地等。<ref>{{Cite web|url=https://news.mingpao.com/pns/dailynews/web_tc/article/20160114/s00012/1452707845330|title=何雅心／陳劍青：土地發展  棕土包尾！ - 20160114 - 觀點 - 觀點|accessdate=2017-12-07|work=明報新聞網 - 每日明報 daily news|language=zh-tw}}</ref>

=== 棕土群發展潛力研究 ===
針對政府表示棕地分佈雜亂零散，本土研究社2018年3月15日發表《合棕連橫：新界棕土發展潛力研究》，當中用半年時間，統計出全港相鄰棕土群共1,171.5公頃，即面積約7個荃灣市中心，當中723公頃未納入發展計劃。團體整理出，全港2公頃或以上的新界相鄰棕土群達94幅，當中位於橫台山及位於坪輋的棕土群更分別達71公頃及70公頃的棕土群，另有26個相鄰棕土群大過10公頃，比起部份資助房屋佔地面積更大。<ref name=":2">{{Cite web|url=https://hk.news.appledaily.com/local/daily/article/20180316/20333261|title=團體︰723公頃棕土群遭忽略|accessdate=2018-03-17|work=Apple Daily 蘋果日報}}</ref>

本土研究社成員楊夏至批評，政府對棕土的發展潛力有雙重標準，因政府提出《元朗錦田南發展計劃》範圍同樣鄰近石崗機場，但卻指八鄉棕土因受高度限制，無發展潛力。<ref name=":2" />

本土研究社倡議修復棕土，面積較大和無法修復的棕土可用作發展公營房屋等用途，面積較小（2至10公頃）棕土群可作發展鄉村式公屋<ref name=":2" />。

根據本土研究社公布的《合棕連橫：新界棕土發展潛力研究》，全港首十六大棕土群分布如下<ref>{{Cite news|url=http://www.mingpaocanada.com/Tor/htm/News/20180316/HK-gab1_r.htm|title=本土研究社﹕723頃棕地未納規劃 可建8.4萬伙|last=Pao|first=Ming|accessdate=2018-03-17|language=zh-HK}}</ref>：

=== 短租閒置土地 ===
2017年7月20日，本土研究社整合地政總署資料，公布「香港[[短租|短租]][[閒置土地|閒置土地]]地圖」，指出政府擁有大量可供短期租用的閒置土地，一共有927幅，佔地142萬平方米，約142公頃，面積相當於7.5個維園。最大一幅位於馬灣，約8萬平方米。<ref>{{Cite web|url=https://news.mingpao.com/ins/instantnews/web_tc/article/20170720/s00001/1500563788774|title=本土研究社推可短租土地地圖籲市民善用　發展局：歡迎查詢 (23:41) - 20170720 - 港聞|accessdate=2017-12-07|work=即時新聞 instant news|language=zh-tw}}</ref>

另外，近三分之一的短租批地協議乃私人短租，被用作私人後花園，已知佔地約37.4公頃。其中有不少屬「富豪設施用地」，包括23個泳池和8個網球場。本土研究社質疑做法猶如將短期租約將官地「私有化」。相關短租富豪用地的使用者包括鷹君集團[[羅嘉瑞|羅嘉瑞]]家族、信和置業[[黃志祥_(商人)|黃志祥]]家族、澳門賭王[[傅老榕家族|傅老榕家族]]、前助理警務處長[[關綺蘿|關綺蘿]]、有「貨櫃大王」之稱的西域投資（香港）有限公司董事長[[林良成|林良成]]、廣西壯族自治區政協[[朱殿安|朱殿安]]等。<ref name=":1">{{Cite news|url=https://thestandnews.com/politics/%E6%9C%B1%E9%96%80%E9%85%92%E8%82%89%E8%87%AD-%E6%9C%AC%E5%9C%9F%E7%A0%94%E7%A9%B6%E7%A4%BE-%E5%AF%8C%E8%B1%AA-%E7%A7%81%E6%9C%89%E5%8C%96-%E5%AE%98%E5%9C%B0-%E5%B9%B3%E7%A7%9F%E5%BB%BA%E6%B3%B3%E6%B1%A0%E7%B6%B2%E7%90%83%E5%A0%B4-%E7%B1%B2%E6%94%BF%E5%BA%9C%E5%85%AC%E9%96%8B%E7%A7%9F%E7%94%A8%E6%A2%9D%E6%AC%BE/|title=【朱門酒肉臭】本土研究社：富豪「私有化」大量官地　平租建泳池網球場　政府須公開條款 {{!}} 立場報道 {{!}} 立場新聞|newspaper=立場新聞 Stand News|accessdate=2017-12-07|language=en}}</ref>

本土研究社又指出，地政總處有關短期租約的資訊封閉，政府沒有公布短租土地的確實租金水平，只對向外公布收取「市值租金」，令外界無法得知租金水平。<ref name=":1" />

2017年7月22日，本土研究社成員楊夏至指出，政府多年利用優惠價，將政府土地租給「超級富豪」，部分短期租約呎租低至0.6港元，部分政府土地批獲短租後被改建成私家路、泳池等。相反，曾有馬屎埔農戶因其田地一半是政府土地，為了繼續務農，遂向政府申請以短期租約形式繼續承租，但遭地政總署反對，及後被發展商以高價投走。楊夏至質疑，現時政府短租政策不公平，對於不同階級人士採取不同態度應對，低下階層不容租用官地，但富豪則能以低價租地自用。<ref>{{Cite news|url=http://hk.on.cc/hk/bkn/cnt/news/20170722/bkn-20170722153726620-0722_00822_001.html|title=團體商土地問題　轟政府城市規劃漠視綠化|newspaper=on.cc東網|accessdate=2017-12-09|language=zh-hk}}</ref>

=== 市建局高估開支 ===
2017年12月3日，[[市區重建局|市建局]]行政總監[[韋志成|韋志成]]發表網誌表示，若要發展油旺地區研究範圍內800多幢地積比已用盡的舊樓，按每平方呎建築面積收購價約15,700元計算，或須面對近1,380億元虧損。不過，本土研究社成員陳劍青質疑市建局「報大數」，一來市建局成立至今只重建700多幢樓，「未來重建哂油尖旺區800幾幢舊樓」的說法成疑；二來市建局過去不會選擇地積比已用盡的舊樓去重建，現在突然表示會重建這類舊樓，質疑只是假象；三來市建局所訂的收購價未必準確，皆因一旦[[地積比|地積比]]已經用盡，收購價未必如此高，加上市建局部分項目會同發展商分紅，部分收益或未計算其中，故要求市建局公開相關帳目。 <ref>{{Cite news|url=https://hk.news.appledaily.com/local/realtime/article/20171203/57534749|title=韋志成指重建油旺蝕千億非靠嚇 陳劍青質疑假象：冇提發展商分紅|author=蘋果日報|date=2017年12月3日|work=|accessdate=}}</ref>

2017年12月5日，本土研究社在facebook統計市建局預計開支與實際開支差距，指出市建局過去十年預計總開支每年平均達271億，但翻看相關年報，實際總開支平均只有137億，遠低於當初估算，十年以來累積下來一共高估589億，質疑市建局「報大數」。團體質疑，市建局錯誤高估開支，藉此合理化現時重建模式，令市建局為了收回高估後的開支，只能以「賺到盡」的方式開發土地，而拒絕其他發展方式，如公營／私營混合物業發展模式、興建公屋或其他可負擔房屋等。<ref>{{Cite news|url=http://www.inmediahk.net/node/1053717|title=市建局也「支爆」? {{!}} 本土研究社 {{!}} 香港獨立媒體網|newspaper=香港獨立媒體網|accessdate=2017-12-09}}</ref>
