==營運==
在營運上，本土新聞在艱苦中營運，由於[[資源|資源]]緊張，現在他們的[[編輯|編輯]]室的工作人員都已經半義務承擔[[工作|工作]]，並且他們背後沒有[[金主|金主]]<ref name = "a3">{{cite web |author1=《本土新聞》 |title=【請讀者持續訂閱 《本土新聞 》】|url = https://www.facebook.com/localpresshk/photos/a.1487946891421787.1073741828.1487928594756950/1783213435228463|agency=[[Facebook|Facebook]]|date=2016-07-19|accessdate=2016-08-20|}}</ref><ref name = "a1"/>，資金來源是靠民眾捐款。

==事件==
===被立法會保安阻入立法會採訪===
2014年6月20日早上，香港[[香港立法會|立法會]][[保安|保安]]阻止《本土新聞》記者[[曾焯文|曾焯文]]進入立法會採訪，而所謂的理由是「有[[議員|議員]]投訴你哋唔係[[傳媒|傳媒]]。」當曾焯文追問是哪位議員，保安卻支吾以對，最後曾焯文被人強行帶走。<ref>{{cite web |author1=《本土新聞》 |title=【採訪受阻】|url = https://www.facebook.com/video.php?v=1504608309755645|agency=[[Facebook|Facebook]]|date=2014-06-20|accessdate=2016-08-20|}}</ref>

同日下午，香港時事評論員[[陳云根|陳雲]]就這事作出批評：「離地賣港的議員，怕本土怕到要死！」。<ref>{{cite web |author1=[[陳云根|陳雲]] |title=陳雲：《本土新聞》證明已經踏入港共、財閥與泛民的賣港禁地！剛才立法會職員拒絕《本土新聞》記者曾焯文博士進入採訪，聲稱有議員投訴《本土新聞》並非傳媒，要列入立法會的黑名單。離地賣港的議員，怕本土怕得要死！|url = https://www.facebook.com/video.php?v=1504608309755645|agency=[[Facebook|Facebook]]|date=2014-06-20|accessdate=2016-08-20|}}</ref>

===總編輯遭警察拘捕===

2014年11月20號，《本土新聞》總[[編緝|編緝]][[梁錦祥|梁錦祥]]因為遭香港[[香港警察|香港警察]]以「懷疑策劃衝擊行動（衝擊立法會事件）」為由拘捕到[[觀塘警署|觀塘警署]]，之後轉送到[[中區警署|中區警署]]，而所謂的罪名是「涉嫌[[刑事毀壞|刑事毀壞]]」。<ref name = "a4"/>

2014年11月21號凌晨，《本土新聞》發表啟事澄清梁錦祥並無策劃立法會大樓衝擊行動的事，當日他只有履行[[採訪|採訪]]職務。<ref>{{cite web |author1=《本土新聞》 |title=啟事：關於《本土新聞》總編輯梁錦祥被警方拘捕之事|url = https://www.facebook.com/localpresshk/posts/1565723410310801|agency=[[Facebook|Facebook]]|date=2014-11-21|accessdate=2016-08-20|}}</ref>

梁錦祥遭警方拘捕之後，《本土新聞》翻譯主任[[曾焯文|曾焯文]]認為梁錦祥遭警方拘捕，可能與[[社民連|社民連]]前主席[[陶君行|陶君行]]旗下的網媒「[[香港花生|香港花生]]」的一篇老屈文章有關。「香港花生」曾經於2016年11月19日以《熱血新聞，四圍昆人》為題刊登文章，而這篇文章裡面附有梁錦祥在立法會衝撃現場附近採訪的[[相片|相片]]，並且說梁錦祥「尋晚（2016年11月18號）突然出現喺立法會停車場同金鐘佔領區大台附近（衝突地點），整個過程中，猶如監軍，親自督戰」。 <ref name = "a4">{{cite_news|url=http://www.passiontimes.hk/article/11-21-2014/19739/?fb_comment_id=799261013463390_799395233449968#f4cd0091ac6b14|title=無恥網媒出賣抗爭者 梁錦祥法國佬被篤涉衝撃被捕|work=《[[熱血時報|熱血時報]]》|date=2014-11-21|accessdate=2016-08-20}}</ref>

2014年11月22號，《本土新聞》發表譴責聲明，譴責警方在毫無證據下的濫權行為。《本土新聞》透過這個聲明指出，當日梁錦祥是代表《本土新聞》履行採訪職務，他身上亦佩戴了[[國際記者證|國際記者證]]，他又有將現場的消息及相片即時傳送給《本土新聞》編輯室。《本土新聞》又透過這個聲明對於這件事發生之後，多間[[傳媒|傳媒]][[報導|報導]]這件事不盡不實，沒人將事件聚焦於警方的無理拘捕，反而對涉嫌策劃的事捕風捉影，繪形繪聲，有幫助政府維穩的嫌疑表示遺憾。《本土新聞》再透過這個聲明指出警方於這件事發生之前，已經限制記者採訪[[雨遮革命|雨遮革命]]的佔領區，甚至在旺角佔領區驅逐國際戰地記者，於世界出醜。《本土新聞》最後透過這個聲明促請警方尊重新聞自由及對他們的濫權行為道歉。<ref>{{cite web |author1=《本土新聞》 |title=【譴責聲明】|url = https://www.facebook.com/localpresshk/photos/a.1487946891421787.1073741828.1487928594756950/1566331806916628|agency=[[Facebook|Facebook]]|date=2014-11-22|accessdate=2016-08-20|}}</ref>

===遭拒絕進入立法會參選人簡介會採訪===
2014年8月2日，香港[[選舉事務處|選舉事務處]]新聞主任拒絕《本土新聞》等網絡傳媒[[記者|記者]]進入[[選舉管理委員會|選舉管理委員會]]舉行的立法會參選人簡介會採訪，其中的原因是他們未獲邀請、未有登記，《本土新聞》記者要求新聞主任提供邀請名單，問可不可以即場登記及登記程序是怎樣，但是新聞主任卻不理問題，嚴重干預記者正常採訪。8月3日，[[香港記者協會|香港記者協會]]發表聲明，對於再拒絕受《本土新聞》等網媒採訪表示不滿，認為做法不合情不合理。記協說他們已經就這件事去信政府新聞處處長及選舉管理委員會主席，批評政府長期以來歧視網絡傳媒的政策，明顯有違新聞自由，亦與政府鼓勵創新科技的大[[政策|政策]]不相符，要求[[政府|政府]]確認網絡傳媒認可地位，保障市民的[[新聞自由|新聞自由]]。<ref>{{cite_news|url=http://www.localpresshk.com/2016/08/online-media-rejected-to-report/|title=《本土新聞》等網媒被拒採訪立會參選人簡介會　記協發聲明譴責|work=《本土新聞》|date=2016-08|accessdate=2016-08-20}}</ref><ref>{{cite_news|url=https://www.thestandnews.com/politics/%E7%AB%8B%E5%A0%B4%E7%AD%89%E7%B6%B2%E5%AA%92%E8%A2%AB%E6%8B%92%E6%8E%A1%E8%A8%AA%E9%81%B8%E7%AE%A1%E6%9C%83%E7%B0%A1%E4%BB%8B%E6%9C%83-%E8%A8%98%E5%8D%94%E6%89%B9%E4%B8%8D%E5%90%88%E6%83%85%E4%B8%8D%E5%90%88%E7%90%86/|title=《立場》等網媒被拒採訪選管會簡介會　記協批不合情不合理|work=《立場新聞》|date=2016-08-03|accessdate=2016-08-20}}</ref><ref>{{cite_news|url=http://www.inmediahk.net/node/1043814|title=網媒被拒採訪候選人簡介會 記協不滿|work=《香港獨立媒體》|date=2016-08-03|accessdate=2016-08-20}}</ref>

=== 營運資金不足，面臨停刊 ===
2017年1月25日，《本土新聞》社長區惠蓮發出公告，稱如在2月底前無法籌得足以支持網站營運下去的資金，《本土新聞》將會停刊。公告指他們仍希望盡全力挽救：「《本土新聞》能不能走下去，我們還是和讀者一起決定。」
