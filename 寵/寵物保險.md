'''寵物保險'''主要是償付寵物主人因寵物生病或發生意外時需要支付的費用，功能上類似人類的醫療保障，性質上則屬於財產保險。

== 起源 ==

很多材料都認為，首張寵物保單是由瑞典的Claes Virgin發出。<ref>{{cite web |url=http://www.agriapet.co.uk/our-history/default.aspx |title=存档副本 |accessdate=2009-05-10 |deadurl=yes |archiveurl=https://web.archive.org/web/20090531015945/http://www.agriapet.co.uk/our-history/default.aspx |archivedate=2009-05-31 }}</ref>但據資料顯示，Claes Virgin發出的是家畜保險，主要為當時瑞典的畜牧業減低疾病風險。<ref>[http://www.unige.ch/ses/istec/EBHA2007/papers/Broman.pdf]{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>至於真正意義上的寵物保險，應在1924年在[[瑞典|瑞典]]寫出，首隻受保寵物是一頭狗。<ref>{{Cite web |url=http://www.aspcapetinsurance.com/insurance_research/history-of-pet-insurance.aspx |title=存档副本 |access-date=2009-05-10 |archive-url=https://web.archive.org/web/20090501042317/http://www.aspcapetinsurance.com/insurance_research/history-of-pet-insurance.aspx |archive-date=2009-05-01 |dead-url=yes }}</ref>

隨著越來越多家庭飼養寵物，寵物保險於1947年登陸[[英國|英國]]，1982年登陸[[美國|美國]]。<ref>{{Cite web |url=http://www.aspcapetinsurance.com/insurance_research/history-of-pet-insurance.aspx |title=存档副本 |access-date=2009-05-10 |archive-url=https://web.archive.org/web/20090501042317/http://www.aspcapetinsurance.com/insurance_research/history-of-pet-insurance.aspx |archive-date=2009-05-01 |dead-url=yes }}</ref>現時，瑞典的寵物保單滲透率已達49%；<ref>{{cite web |url=http://press.petinsurance.com/mediakit/factsheet.aspx |title=存档副本 |accessdate=2009-05-10 |deadurl=yes |archiveurl=https://web.archive.org/web/20090430183436/http://press.petinsurance.com/mediakit/factsheet.aspx |archivedate=2009-04-30 }}</ref>英國其次，有23%；<ref>[http://mediacentre.tescofinance.com/pet-insurance/press-release/2008-08-18]</ref>美國則只有1%左右。<ref>[http://www.googobits.com/articles/1208-the-ins-and-outs-of-pet-insurance.html]{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

隨著香港越來越多家庭飼養寵物，寵物保險亦於2005年登陸，由藍十字（亞太）保險有限公司承保。其後，皇家太陽聯合保險有限公司也於2006年在香港推出寵物保險。<ref>[http://www.mpfinance.com/htm/Finance/20061205/News/wm3_wm3a1.htm]</ref>

2018年7月，PetbleCare 寵物保險正式推出，根據香港寵物主人的需要設計全新的寵物保險計劃，透過保險科技平台www.petblecare.com處理投保程序，由利寶國際保險有限公司承保、進邦匯理有限公司獨家銷售。
