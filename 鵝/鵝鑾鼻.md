'''鵝鑾鼻'''（Cape Eluanbi），為台灣最南端的[[岬角|岬角]]，又名'''南岬'''（South Cape）。其位置於[[臺灣|臺灣]][[屏東縣|屏東縣]][[恒春鎮|恒春鎮]]，為[[中央山脈|中央山脈]]盡處[[臺地|臺地]]的最南端，隔著[[巴士海峽|巴士海峽]]與[[菲律賓|菲律賓]]相望。

== 命名 ==
鵝鑾鼻之名來由，眾說紛紜。其中[[墾丁國家公園|墾管處]]廣傳的說法如下：「鵝鑾」可追溯於[[排灣語|南排灣族語]]，意為「船帆」，因該地西側香蕉灣有顆形似船帆的石頭，因而取名「鵝鑾」，西側大石被稱為[[船帆石|船帆石]]；而「鼻」在[[台灣閩南語|台語]]中意旨物體突出而像鼻子的部分，即為岬角之意（如：'''貓鼻頭'''），因而得名鵝鑾鼻。[[臺灣日治時期|臺灣日治時期]]之文獻上，此地名亦常以[[片假名|片假名]]書寫為「{{lang|ja|ガランピ}}」，或[[漢字|漢字]]「鵞鑾鼻」。

== 歷史 ==
[[File:Garanpi_Shrine.jpg|thumb]]
清代多記載臺灣最南點為「沙馬磯」或「沙馬磯頭」，例如[[黃叔璥|黃叔璥]]的《[[臺海使槎錄|臺海使槎錄]]》載：「南路界盡沙馬磯頭，相傳地脈直接呂宋。」，但現代對於沙馬磯頭是[[貓鼻頭|貓鼻頭]]或鵝鑾鼻則未有定見。

鵝鑾鼻一帶原為[[排灣族|排灣族]]龜仔甪社（Kuarut）之領地。其地形為[[珊瑚礁|珊瑚礁]]隆起之[[台地|台地]]，三面臨海，一面背山，地勢險要且沿岸多珊瑚暗礁，由於地處[[巴士海峽|巴士海峽]]與[[太平洋|太平洋]]交會之處，往來漁船頻繁，其海下之暗礁險惡甚之。

[[清朝|清]][[同治六年|同治六年]]（1867年），美國商船[[羅發號事件|羅發號]]（又譯'''羅妹號'''），於鵝鑾鼻西南方約15公里的七星礁觸礁沉沒，船長亨特（J. W. Hunt）夫婦偕同船員水手14人游至獅龜嶺（今龍磐草原一帶）上岸，因誤闖龜仔甪社領地，而被[[原住民|原住民]]誤認為侵略者，而殺害之，僅一名[[廣東人|粵籍]]水手逃生，受當地居民幫助輾轉送至[[打狗英國領事館|打狗英國領事館]]，促而使中美雙方簽訂[[南岬之盟|南岬之盟]]。但也因[[清廷|清廷]]在[[台灣|台]]消極治理之作為，並稱：「''告以生番不歸地方官管轄，嗣後請飭外國商人謹遵土牛之禁，不可擅入生番境界，以免滋事。''」，埋下日本企圖佔領台灣的野心。

[[清朝|清]][[同治十年|同治十年]]（1871年），琉球漁民因遇颱風，而漂至八瑤灣（今[[滿州鄉|滿州鄉]]九棚-港仔一帶），因船毀壞無法獨力回國，而深入異地與[[高士|高士佛社]]的[[原住民|原住民]]起了衝突，而54名琉球人進而遭原住民殺害。而日本為此為藉口，起兵討伐原住民，引發兩年後的[[牡丹社事件|牡丹社事件]]。經歷兩次大規模外國漁船觸礁案，清廷無法坐視不管，首先於[[琅嶠|琅嶠]]猴洞山（今恆春市區）興建縣城，定名「[[恆春縣|恆春縣]]」；再者委託[[英國皇家學會|英國皇家學會]]的畢齊禮（Michael Beazeley）勘查恆春半島地貌，尋求興建燈塔的地點，後於現今[[鵝鑾鼻燈塔|鵝鑾鼻燈塔]]處，以100塊銀元向龜仔甪社購買土地，興建燈塔。

== 設施 ==
在1982年成立鵝鑾鼻公園，以[[鵝鑾鼻燈塔|鵝鑾鼻燈塔]]為主體，規劃出鄰近五十[[公頃|公頃]]的[[國家風景區|風景區]]，後又劃為[[墾丁國家公園|國家公園]]管理
