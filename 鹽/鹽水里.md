'''盐水里'''為[[台灣|台灣]][[新竹市|新竹市]][[香山區|香山區]]轄下之一[[里|里]]，位於新竹市郊的香山海濱，西側臨[[臺灣海峽|臺灣海峽]]有寬達800公尺的[[潮間帶|潮間帶]]，為新竹市濱海野生動物保護區的一部份，北以[[海山川|海山川]]與海山里為界<ref>中華民國內政部 臺灣地區地名查詢系統 海山里，{{cite web |url=http://placesearch.moi.gov.tw/input/detail.php?input_id=10155 |title=存档副本 |accessdate=2007-08-09 |deadurl=yes |archiveurl=https://web.archive.org/web/20070929131641/http://placesearch.moi.gov.tw/input/detail.php?input_id=10155 |archivedate=2007-09-29 }} ，臺灣地名辭書(卷十八)新竹市，臺灣省文獻會</ref> 東南兩側與內湖裡接鄰，西南則是南港里。

==聚落概述==
鹽水里包含兩傳統聚落，鹽水港以及草厝仔，居民多為閩籍<ref>{{cite web |url=http://dep-civil.hccg.gov.tw/_file/2890/SG/23623/D.html |title=存档副本 |accessdate=2007-08-10 |deadurl=yes |archiveurl=https://web.archive.org/web/20070930045638/http://dep-civil.hccg.gov.tw/_file/2890/SG/23623/D.html |archivedate=2007-09-30 }} 新竹市政府民政局網頁，網頁中所附的日治時期1921年閩粵人口統計表，鹽水港大字閩籍人口佔97%</ref>，從事農漁業。

鹽水港聚落位於鹽水里中部，縱貫鐵路西側。其歷史最早可上溯自乾隆三十七年(1772年)陳璋琦等人入墾，1838年淡水同知在此設''鹹水港官渡''，成為[[汫水港|汫水港]]前往[[竹南堡|竹南堡]]中港的必經道路，清末1887年就以此聚落為核心建立[[鹽水港_(新竹市)|-{鹽水港庄}-]]<ref>中華民國內政部 臺灣地區地名查詢系統 鹽水港，{{cite web |url=http://placesearch.moi.gov.tw/input/detail.php?input_id=7673 |title=存档副本 |accessdate=2007-08-09 |deadurl=yes |archiveurl=https://web.archive.org/web/20070929131603/http://placesearch.moi.gov.tw/input/detail.php?input_id=7673 |archivedate=2007-09-29 }} ，臺灣地名辭書(卷十八)新竹市，臺灣省文獻會</ref>。而在鹽水港稍南的地方，是鹹水港官渡候船之地，候船所搭建的而草棚子稱為草厝子，日治時期居民撿拾貝殼建窯燒石灰，所以又稱草厝子為灰窯。<ref>中華民國內政部 臺灣地區地名查詢系統 草厝仔（灰窯），{{cite web |url=http://placesearch.moi.gov.tw/input/detail.php?input_id=7674 |title=存档副本 |accessdate=2007-08-09 |deadurl=yes |archiveurl=https://web.archive.org/web/20070929131622/http://placesearch.moi.gov.tw/input/detail.php?input_id=7674 |archivedate=2007-09-29 }} ，臺灣地名辭書(卷十八)新竹市，臺灣省文獻會</ref>

在省道臺1線的東側，則有晚近新建的鋼鐵工廠。

==交通==
臺1線及[[臺61|西濱公路]]為鹽水里主要幹道，其貫穿里境連絡臺灣西部各縣市，長興街則為里內居民的傳統的交通動線<ref name="新竹市民政局">[http://civil.hccg.gov.tw/002/s17.htm 新竹市民政局 自治行政課] {{webarchive|url=https://web.archive.org/web/20070930043751/http://civil.hccg.gov.tw/002/s17.htm |date=2007-09-30 }}</ref>。
