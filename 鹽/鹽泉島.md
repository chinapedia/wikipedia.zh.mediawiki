'''鹽泉島'''（{{lang-en|Saltspring Island}} 或 {{lang|en|Salt Spring Island}}）是[[加拿大|加拿大]][[卑詩省|卑詩省]]一座島嶼，坐落卑詩省大陸部分和[[溫哥華島|溫哥華島]]之間的[[喬治亞海峽|喬治亞海峽]]，是海灣群島（{{lang|en|Gulf Islands}}）中面積最大和人口最多的島嶼。鹽泉島隸屬[[首府地區_(卑詩省)|首府地區]]，為一[[非建制地區|非建制地區]]<ref name="CRD"/>，即島上沒有任何正式建制的地方政府架構。島上的市政事務由首府地區區域局負責<ref name="CRD">{{Cite web |url=https://www.crd.bc.ca/about/about-the-region/salt-spring-island |title=Salt Spring Island |accessdate=2014-02-07 |work=Capital Regional District }}</ref>，而全島範圍則構成首府地區的己選區（{{lang|en|Electoral Area F}}）。據2011年加拿大人口普查所示，鹽泉島上人口為10234人<ref name="2011census"/>。

==歷史==
鹽泉島原為沙利殊原住民族（Salish）的地域，該族人將此島嶼稱為“Chouan”或“Chuan”，意即“兩端”，代表該島南北兩端的高山<ref name="tour">{{Cite web |url=http://saltspringtourism.com/about/history/ |title=Salt Spring Island History |accessdate=2014-02-07 |work=Salt Spring Island Tourism }}</ref>。島上有多個鹽水泉，因此[[哈德遜灣公司|哈德遜灣公司]]高層於19世紀中期將此處名為“鹽泉島”，並計劃從此處取鹽<ref name="GeoBC">{{Cite web |url=http://apps.gov.bc.ca/pub/bcgnws/names/13666.html |title=BC Geographical Names: Saltspring Island |accessdate=2014-02-07 |work=GeoBC }}</ref>。英國船長理查斯於1859年在該帶海域進行勘探，並將此島名為“海軍將官島”（{{lang|en|Admiral Island}}）以紀念英國海軍少將貝恩斯（{{lang|en|Robert Lambert Baynes}}）<ref name="GeoBC"/>。

首批非原住民居民於1858年至1859年間抵達此地，當中包括來自美國的黑人居民<ref name="tour"/>。另外，當時的卑詩殖民地政府亦准許殖民搶佔島上的土地：殖民需自行開拓土地，其後准以低價購入由其砍伐的土地<ref name=Sandwell>{{cite book | last = Sandwell | first = Ruth Wells | title = Beyond the City Limits: Rural History in British Columbia | publisher = UBC Press | year = 1999 | isbn = 0-7748-0694-X | page = 85}}</ref>。由於當時[[維多利亞_(卑詩)|維多利亞]]一帶的地價上揚，當地失業率亦偏高，因此這項措施吸引大批殖民遷至此地<ref name="tour"/>。

島上居民傾向將此島稱為鹽泉島而非海軍將官島，加拿大地理委員會亦因此於1910年正式恢復鹽泉島一名。該委員會將此島英文名拼為單字“{{lang|en|Saltspring}}”，但島上不少居民則傾向將之拼為雙字“{{lang|en|Salt Spring}}”<ref name="GeoBC"/>。

==交通==
鹽泉島上有三座[[卑詩渡輪|卑詩渡輪]]碼頭：東南部的富爾福德港碼頭（{{lang|en|Fulford Harbour}}）提供航線通往溫哥華島的施瓦茨灣（{{lang|en|Swartz Bay}}）；東北部的長港碼頭（{{lang|en|Long Harbour}}）提供航線通往美恩島（{{lang|en|Mayne Island}}）、片打島（{{lang|en|Pender Island}}）和[[低陸平原|低陸平原]]的[[三角洲_(卑詩省)|三角洲]]；而西北部的維蘇威灣碼頭（{{lang|en|Vesuvius Bay}}）則提供航線通往溫哥華島的克羅夫頓（{{lang|en|Crofton}}）。

島上的巴士服務於2008年創立<ref>{{Cite web |url=https://www.crd.bc.ca/about/what-we-do/transportation-transit |title=Transportation & Transit (SSI) |accessdate=2014-02-07 |work=Capital Regional District }}</ref>，營運開支由首府地區區域局和卑詩-{公車局}-共同承擔<ref>{{Cite web |url=http://www.transitbc.com/regions/ssi/ |title=Salt Spring Island Transit System |accessdate=2014-02-07 |work=BC Transit }}</ref>；島上全數六條巴士路線皆以該島主要聚落恆河村（{{lang|en|Ganges}}）為總站<ref>{{Cite web |url=http://www.transitbc.com/regions/ssi/schedules/map.cfm?rmnu=dvMap&p=side&region=0& |title=Regional Map for Salt Spring Island |accessdate=2014-02-07 |work=BC Transit }}</ref>。

航空交通方面，恆河水上機場（{{lang|en|Ganges Water Aerodrome}}；IATA代碼：YGG）提供水上飛機航線前往[[溫哥華國際機場|溫哥華國際機場]]附近的水上機場，而最接近鹽泉島的主要機場則為溫哥華島上的[[維多利亞國際機場|維多利亞國際機場]]。
