'''鹽竈市'''（{{jpn|j=塩竈市|hg=しおがまし|rm=Shiogama shi|f=y}}），俗作「鹽釜市」（{{lang|ja|塩釜市}}），是[[日本|日本]][[宮城縣|宮城縣]]的一個[[市|市]]級次行政區，位于[[太平洋|太平洋]]沿岸的{{link-ja|仙台灣|仙台湾}}內、宮城縣的中央。該市是宮城縣內的市町村中人口密度最高的一個城市。

== 概要 ==
[[File:Shiogama_and_sendai_from_an_aeroplane.jpg|left]]

位於[[宮城縣|宮城縣]]接近中央、[[仙台市|仙台市]]與[[松島|松島]]之間的位置，是{{Link-ja|仙鹽|仙塩}}地区的中心之一。圍繞於松島湾和{{Link-ja|松島丘陵|松島丘陵}}間，平地幾乎都是[[填海|填海]]土地。[[市街地|市街地]]中，填海土地佔6成，丘陵地佔4成<ref>{{Cite news|url=http://www.city.shiogama.miyagi.jp/html/about/plan/kankyou/pdf/2-12-14.pdf|title=地形・地質|publisher=鹽竈市|language=ja|deadurl=yes|archiveurl=https://web.archive.org/web/20071211214108/http://www.city.shiogama.miyagi.jp/html/about/plan/kankyou/pdf/2-12-14.pdf|archivedate=2007-12-11}}</ref>。由於狹窄的可住地內聚居有密集的人口，該地幾乎無適合路邊店開店的土地。而藉由{{Link-ja|減反政策|減反政策}}獲得餘裕土地的[[利府町|利府町]]和[[多賀城市|多賀城市]]有大量路邊店鋪進駐，因此鹽竈市失去了仙鹽地區的商業中心的地位，不僅是市中心部，郊外部的商業亦變得低迷。（[[#經濟|見「經濟」章節]]）

=== 經濟 ===
經濟狀況嚴峻，市內商業街滿是關閉的店舗。為了以觀光促進經濟發展而設立的旅客船樞紐「{{Lang|ja|マリンゲート塩釜}}」也是在[[承租人|承租人]]陸續撤退的情況下赤字經營。

== 市名表記（「鹽竈」與「鹽釜」） ==
「鹽竈」（{{Lang-ja|塩竈}}）二字中，「鹽」在日本[[常用漢字|常用漢字]]中寫作「{{Lang|ja|[[Wikt:塩|塩]]}}」（[[日本新字体|新字體]]），市名用之。[[舊字體|舊字體]]「{{Lang|ja|[[Wikt:鹽|鹽]]}}」字在「-{[[鹽竈神社|鹽竈神社]]}-」以外已不常使用。「{{Lang|ja|[[Wikt:竈|竈]]}}」字有時也寫作[[擴張新字體|擴張新字體]]「{{Lang|ja|[[Wikt:竃|竃]]}}」。另外，各機關、設施中關於地名的「{{Lang|ja|鹽竈}}」「{{Lang|ja|塩竈}}」「{{Lang|ja|塩釜}}」等不同表記，皆讀作「{{lang|ja|しおがま}}」（Shiogama）。

以日文同音的「{{Lang|ja|塩釜市}}」表記的情況很常見，位於鹽竈市內的非政府機關之名稱多用「{{Lang|ja|塩釜}}」（例如[[郵局|郵局]]為「{{Lang|ja|塩釜郵便局}}」），[[JR|JR]]線站名也使用「{{Lang|ja|塩釜}}」（[[鹽釜站|鹽釜站]]、{{Link-ja|本鹽釜站|本塩釜駅}}等），「[[Wikt:釜|釜]]」即所謂「鍋・釜」的「{{Lang|ja|かま}}」，而「竈」是用於承載釜的「{{Lang|ja|かまど}}」，字義不同<ref>{{Cite web|url=http://www.city.shiogama.miyagi.jp/shiminsomu/shise/joho/shimin/zenpan/kamadonoji.html|title=「塩竈」の「竈」の字の表記について|publisher=鹽竈市|language=ja|deadurl=yes|archiveurl=https://web.archive.org/web/20140813222847/http://www.city.shiogama.miyagi.jp/shiminsomu/shise/joho/shimin/zenpan/kamadonoji.html|archivedate=2014年8月13日|df=}}</ref>。

市名來自鹽竈神社的社号，該市的公文表記統一作「{{Lang|ja|竈}}」，由於「{{Lang|ja|竈}}」字書寫困難，市公文以外常以「{{Lang|ja|釜}}」代替「{{Lang|ja|竈}}」，亦可獲承認。為了讓人們正確書寫[[漢字|漢字]]「{{Lang|ja|竈}}」，市官方網站及宣傳冊為了對外宣傳，收錄有「{{Lang|ja|竈}}」的筆順<ref>{{Cite web |url=http://www.city.shiogama.miyagi.jp/seisaku/shise/gaiyo/kamado.html |title=塩竈市公式サイト-「竈」の字について |publisher=鹽竈市 |language=ja}}</ref>。
