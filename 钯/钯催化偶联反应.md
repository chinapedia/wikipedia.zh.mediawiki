'''[[钯|钯]][[催化|催化]]的[[偶联反应|偶联反应]]'''是偶联反应的一大类，是指以钯化合物作为催化剂。它是{{le|均相催化|Homogeneous catalysis}}剂的研究和应用的活跃领域。2010年[[诺贝尔化学奖|诺贝尔化学奖]]颁发给[[理查德·赫克|理查德·赫克]]、[[根岸英一|根岸英一]]和[[铃木章|铃木章]]，三位围绕此主题进行研究的化学家<ref>http://nobelprize.org/nobel_prizes/chemistry/laureates/2010/</ref><ref>[http://www.nobelprize.org/nobel_prizes/chemistry/laureates/2010/advanced-chemistryprize2010.pdf PALLADIUM-CATALYZED CROSS COUPLINGS IN ORGANIC SYNTHESIS] details of reactions</ref>。

== 例子 ==
反应通常遵循以下化学计量：
:X-R  +  M-R'  →  MX  +  R-R'
变化基于X-R（通常为芳基溴）和M-R'的身份。 通常，反应产生盐或盐样产物（卤化锌，卤化锡，卤化硅）

例如：

* [[Heck反应|Heck反应]]：烯烃与芳卤偶联
* [[Suzuki反应|Suzuki反应]]：芳卤与烷基硼酸偶联
* [[Stille反应|Stille反应]]：卤代烃与有机锡偶联
* [[Hiyama偶联反应|Hiyama偶联反应]]：卤代烃与有机硅偶联
* [[Sonogashira偶联反应|Sonogashira偶联反应]]：芳卤与炔烃偶联，碘化亚铜作共催化剂
* [[Negishi偶联反应|Negishi偶联反应]]：卤代烃与有机锌偶联
* [[Buchwald-Hartwig胺化反应|Buchwald-Hartwig胺化反应]]：芳卤与胺偶联

== 催化剂 ==
常用的钯催化剂有：
* [[乙酸钯|乙酸钯]]
* [[四(三苯基膦)钯|四(三苯基膦)钯]]
* [[二氯双(三苯基膦)钯|二氯双(三苯基膦)钯]]
* [[(1,1'-双(二苯基膦)二茂铁)二氯化钯|(1,1'-双(二苯基膦)二茂铁)二氯化钯]]

== 工作条件 ==
未优化的反应一般用10-15 mol%的钯催化，优化的反应可仅用0.1 mol%或更少的催化剂。有些新颖配体和手性催化剂也有报道，但大多仍未商业化，故实际用途不多。带有膦配体的钯对氧气敏感，而且易脱落，所以不少人研究用其他类型的配体替代膦，如利用Arduengo类型的[[碳烯|碳烯]]配合物。

随着这些反应用途的拓展，对反应后除钯的研究也逐渐增多。传统的方法是用[[柱色谱|柱色谱]]分离，不过Smopex等除金属剂<ref>{{cite web |url=http://www.alfa.com/en/GP100w.pgm?DSSTK=044710 |title=存档副本 |accessdate=2010-10-06 |deadurl=yes |archiveurl=https://web.archive.org/web/20110707111611/http://www.alfa.com/en/GP100w.pgm?DSSTK=044710 |archivedate=2011-07-07 }}</ref> 及QuadruPure<ref>http://www.sigmaaldrich.com/chemistry/drug-discovery/medicinal-chemistry/quadrapure.html</ref> 和ISOLUTE<ref>http://www.biotage.com/DynPage.aspx?id=36161</ref> 一类的树脂也有较好的效果。
