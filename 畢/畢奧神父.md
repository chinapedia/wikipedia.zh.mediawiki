聖'''畢奧神父'''（{{lang-en|Saint Padre Pio (Pius) of Pietrelcina}}；{{lang-it|Pio da Pietrelcina}}；{{bd|1887年|5月25日|1968年|9月23日}}）是一位[[意大利|意大利]]籍[[天主教|天主教]][[方濟嘉布遣會|方濟嘉布遣會]][[神父|神父]]、受[[聖痕|聖傷]]者、[[基督教神秘主义|神秘主義]]者<ref name="Saint Pio of Pietrelcina" />，後來亦被[[羅馬天主教|羅馬天主教]]追封為[[基督教聖人|聖人]]。畢奧神父因從1918年起終身攜帶[[聖痕|聖傷]]而廣為人知，並由此引起了針對這一事件的許多研究與爭議。

畢奧神父原名方濟各·弗哲（{{lang-it|Francesco Forgione}}），出生於意大利南部小鎮[[皮耶特雷尔奇纳|皮耶垂西那]]。他於1903年1月22日入[[方济嘉布遣会|嘉布遣方濟會]]，取名「畢奧」（或譯「比約」）。畢奧神父於1968年9月23日逝世於[[圣焦万尼罗通多|聖若翰·羅通多]]，享年81歲。2002年6月16日被[[教宗|教宗]][[若望·保祿二世|若望·保祿二世]]追封為「聖人」<ref>{{cite web |url = http://www.vatican.va/news_services/liturgy/saints/ns_lit_doc_20020616_padre-pio_en.html |title="Padre Pio de Pietrelcina",- PADRE PIO DA PIETRELCINA |publisher= Vatican News Service |accessdate=}}</ref>。

畢奧神父的紀念日是9月23日<ref>[http://www.binetti.ru/collectio/liturgia/missale_files/crg9.htm 拉丁文：九月的羅馬天主教聖人歷]</ref>。

== 早年生活 ==
聖畢奧神父於1887年5月25日在[[意大利南部|意大利南部]]鄰近[[貝內文托|貝內文托]]小鎮[[皮耶特雷尔奇纳|皮耶特雷尔奇纳]]出生。他的父親 Grazio Mario Forgione (1860–1946) 是一位自食其力的農夫，母親 Maria Giuseppa Di Nunzio Forgione (1859–1929)是一位寬厚仁慈的婦女<ref name="Ruffin">{{Cite book |last = Ruffin |first = Bernard C. |authorlink = C. Bernard Ruffin |title = Padre Pio: The True Story |year = 1991 |publisher = Our Sunday Visitor |ISBN = 978-0-87973-673-6 |page = 444 |postscript = <!--None-->}}</ref><ref name = "gerhold">{{Cite journal |last = Gerhold |first = Ryan  |title = The Second St. Francis |date = 2007-02-20  | journal  = The Angelus |pages =12–18  |postscript = <!--None-->  }}</ref>。嬰兒畢奧在鎮上的聖亞納小堂（Santa Anna Chapel）[[聖洗聖事|領洗]]，[[聖名|聖名]]為[[方濟各|方濟各]]<ref name="ewtn1">{{Cite web |title = Padre Pio the Man Part 1  |url = http://www.ewtn.com/padrepio/man/biography.htm  | accessdate = 2008-01-19  | postscript = <!--None-->}}</ref>。自小他幫助父母在田中耕種，不過他的任務主要是牧放羊群，由此他的教育遲緩了幾年。據說在方濟五歲時，他已下定決心將整個生命獻給天主，甚至在那個年紀已開始了刻苦的[[補贖|補贖]]<ref name="Ruffin"/><ref name="ewtn1"/>。

皮耶特雷尔奇纳是一個人民宗教生活虔誠的小鎮。方濟的家人是熱心的天主教徒，每日參加[[彌撒|彌撒]]、誦念[[玫瑰經|玫瑰經]]，一周內三天[[守齋|守齋]]戒食肉類<ref name="ewtn1" />。雖然方濟的父母和祖父母是[[文盲|文盲]]，他們記得經文並常常講述[[福音|福音]]和聖經故事給孩子們聽。方濟的母親稱方濟從小時候起就常常看見[[耶穌|耶穌]]和[[聖母|聖母]]，還有他的[[守護天使|守護天使]]，以至於年幼的方濟竟以為大家都像他一樣能看見諸聖。

根據他的神師[[圣马尔科因拉米斯|拉密斯]]的奧思定[[神父|神父]]（Agostino da San Marco）的日記，年幼的方濟常常遭受各種疾病的困擾。6歲時他患了一場嚴重的[[胃炎|胃炎]]，使他有很長一段時間患病在床。10歲時，他患上[[傷寒|傷寒]]而高燒大病，鎮上的赤腳大夫看了之後，甚至還說他大概只有幾天好活了，但最後卻誤食一大罐辣椒而獲救<ref name="Legenda">[http://books.chinacath.org/files/article/fulltext/0/397.html 天主教在線：比約神父傳] {{webarchive|url=https://web.archive.org/web/20140806075449/http://books.chinacath.org/files/article/fulltext/0/397.html |date=2014-08-06 }}</ref>。{{NoteTag|一段趣事：方濟十歲時染上傷寒，病得很重。鎮上的赤腳大夫賈欽佗．瓜達納(Giacinto Guadagna)看了之後，搖搖頭說他大概只有幾天好活了，爸爸媽媽強忍悲痛，老實跟方濟說他快死了。方濟聽了之後，說：「如果我快死了，我想再看一眼我深愛的皮阿納．羅馬納。」爸媽考慮之後，決定完成兒子的心願，要他十五歲的哥哥米切雷把方濟搬到驢子背上，跟他一起去一趟那間小屋。當時正是收成季節，貝芭炒了很多胡椒，準備款待請來的雇工。因為這些辣椒實在太辣，所以那些大男人吃得也沒想像中多，還剩了一大罐在桌上。方濟跟米切雷進屋時，想跟哥哥要點來吃，但哥哥不讓他吃。等貝芭來了，方濟跟她說：「媽咪，把門關上好不好？ 光太刺眼了。我想一個人待一下，妳先走開一下好嗎？」等貝芭離開後，這病奄奄的孩子從床上起來，把辣椒全都給吞了，還配上半罐牛奶。貝芭回來發現辣椒沒了，驚訝之餘以為是米切雷不小心放狗進來，讓狗把辣椒全給吃了，把米切雷罵了一頓。當時方濟正在昏睡，醒來之後覺得身體好了大半，這才知道整個經過。他老實告訴媽媽說：吃了那罐辣椒的「狗」，其實就是他。<ref>[http://www.books.com.tw/products/0010614937 《聖五傷畢奧神父傳》啟示出版]</ref>}}

童年時的方濟常常經歷[[异象|神視]]和{{Link-en|神魂超拔|Religious ecstasy}}<ref name="Ruffin" />。1897年，10歲的方濟完成了為期三年的公立中小學教育後，他感受到[[天主|天主]]的[[聖召|聖召]]，被修士的樸素生活所深深吸引。他向父母表達了[[修道|修道]]的意願後，他們帶著孩子去了皮耶特雷尔奇纳以北{{convert|13|mi|km}}的[[莫尔科内|莫尔科内]]，拜訪了那裡的[[方济嘉布遣会|嘉布遣方濟會]]修院，以探知方濟是否能入會修道。會士們表示他們願意接納方濟，但要等到他接受更多教育之後<ref name="ewtn1" />。

方劑的父親為了負擔他的學費，便前往美國尋求工作，以期讓方劑接受更多學術教育<ref>{{cite web |url = https://www.americancatholic.org/Features/Saints/saint.aspx?id=1147 |title=Saints |publisher=The American Catholic |accessdate=}}</ref>。1899年9月27日，方劑接受了[[堅振聖事|堅振聖事]]。之後方劑通過了先前修會規定的教育要求。1903年1月6日，15歲的方劑加入了[[方济嘉布遣会|嘉布遣方濟會]]，領受了{{link-en|會衣|religious habit}}，成為一名初學修士，取名畢奧[[修士|修士]]（'''Fra Pio'''），並发神貧、貞潔、服從的誓願。關於選用這一名字的原因有許多推測，比較公認的是為紀念[[庇护一世|教宗聖庇護一世]]，這位聖人的[[聖髑|聖髑]]存放於畢奧領洗的聖亞納小堂<ref name="santanna">{{cite web |first = Convento |last = Pietralcina |title = La chiesetta di Sant'Anna |url = http://www.cappuccinipietrelcina.com/la-chiesetta-di-santanna/ |website = www.cappuccinipietrelcina.com |accessdate = 30 July 2014 }}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref> <ref name="ewtn1" /><ref name="Ruffin" />。
