[[File:Steel_pd_zh.svg|thumb]]為[[沃斯田鐵|沃斯田鐵]]與[[滲碳體|滲碳體]]的共熔混合物，[[珠光體|珠光體]]為[[鐵素體|鐵素體]]與[[滲碳體|滲碳體]]的層狀組織物，而α、β及δ為不同種類的[[固熔体|固熔体]]]]

'''铁碳合金相图'''是[[铁|铁]]-[[碳|碳]]二元[[合金|合金]]系统的[[相图|相图]]。

在加工完的[[钢|钢]]和[[铸铁|铸铁]]中，总会含有一定数量的[[碳|碳]]元素，这些碳元素所占的比重决定了钢/铸铁的特性。通过铁碳合金相图，人们可以直观的看出（钢铁）形成过程中的组分变化与碳元素含量、[[温度|温度]]间的相互关系。

若需要知道在不同冷卻速率下的{{link-en|微結構|Microstructure}}，一般會參考{{link-en|恆溫變態圖|Isothermal transformation diagram}}。

碳是铁碳合金中最重要的合金元素，碳含量小幅的變化也可能會對合金的材質或性質有大的影響。铁碳合金相图可以表達溫度及碳的濃度對鋼鐵的影響，不過沒有其他金屬的資訊。铁碳合金相图可以分為二部份：{{link-en|亞穩定|Metastability}}的Fe-Fe<sub>3</sub>C系統，其中的碳已和鐵鍵結，以及[[穩定性|穩定]]的Fe-C系統，其中碳以[[石墨|石墨]]的形式存在。铁碳合金相图一般會包括這兩個系統，不過Fe-Fe<sub>3</sub>C系統用到的比較多。

== 铁碳合金相图中的代表性區域 ==
[[File:IJzerkoolstofdiagram.png|thumb]]

相图中的[[X軸|X軸]]是碳的含量，[[Y軸|Y軸]]是[[溫度|溫度]]。圖中只標示一般比較常用到的部份，也就是碳含量在0%到6.67%的區域（只有少量合金的含量高過此值），碳含量6.67%約對應100%的[[碳化三铁|碳化三铁]]。相图中代表性的位置用英文字母表示，有些標示方式會省略英文字母I，改用字母J表示。

其中折線ABCD是{{link-en|液相線|Liquidus}}，在液相線以上的部份為液體。折線AHIECF為[[固相線|固相線]]，低於固相線的部份為固體。在液相線和固相線之間的是糊状、部份熔化的區域，其中包括δ铁、γ铁和[[碳化三铁|碳化三铁]]（Fe<sub>3</sub>C）．有不同的濃度及比例。若合金冷卻到液相線下時，會漸漸的[[结晶|结晶]]。

隨著碳含量的不同，铁碳合金中的鐵也會出現不同的[[同素異形體|同素異形體]]，像分別由δ铁、γ铁及α铁形成的[[固溶體|固溶體]]，對碳就有不同的溶解度，其原因是不同的晶格結構及[[晶格常數|晶格常數]]。像[[肥粒鐵|肥粒鐵]]就是由δ铁或α铁形成的固溶體，而[[奧氏體|奧氏體]]則以γ铁為主。

== 大致的分區 ==
* [[熔化|熔化]]是指形成液相的铁碳合金，{{link-en|液相線|Liquidus}}以上均為液相，在液相線和[[固相線|固相線]]之間的是固相和液相共存。
* δ [[肥粒鐵|肥粒鐵]]：體心[[立方晶系|立方晶系]]
* γ [[奧氏體|奧氏體]]：面心立方晶系
* α [[肥粒鐵|肥粒鐵]]：體心立方晶系
* [[石墨|石墨]]（穩定態）或[[雪明碳鐵|雪明碳鐵]]（Fe<sub>3</sub>C，{{le|亞穩定|Metastability}}系統）

[[波來鐵|波來鐵]]和[[萊氏體|萊氏體]]不算是特別的相，只是一種相混合狀態（{{link-en|微結構|Microstructure}}）。波來鐵和萊氏體只會出現在穩定，或是亞穩定的系統中，例如緩慢的冷卻。若是快速冷卻（例如[[淬火|淬火]]）會形成[[马氏体|马氏体]]，是一種硬脆的結構。以下是在亞穩定的系統中，一些特殊的點、線及現象。

== 等溫過程 ==
铁碳合金相图中有三個[[等温过程|等温过程]]，分別是{{link-de|包晶|Peritektikum}}（線 HIB）、[[共晶系統|共晶]]（線 ECF）及{{link-de|共析|Eutektikum}}（線 PSK）。

點 H：δ鐵素體中，最大碳溶解度的點
點 I：包晶 δ+L → γ<!--(sind im EKD nicht benannt)-->

當鋼加熱或是冷卻的時候，會出現一些特性不連續變化的情形，主要有以下幾點。
* A<sub>1</sub> – 線P-S-K，當碳含量> 0.02 %時，超過723 °C時奧氏體會分解為珠光體。
* A<sub>2</sub> – 線M-O，加熱超過769 °C（[[居里點|居里點]]）時會失去[[鐵磁性|鐵磁性]]。
* A<sub>3</sub> – 線G-O-S，冷卻時會形成含碳量較少的鐵素體，從奧氏體中游離的碳會開始累積，直到溫度到723 °C的共晶溫度為止。

== 應用 ==
配合铁碳合金相图，可以清楚的回答一些有關[[鋼|鋼]]（可鍛的鐵碳合金，碳含量小於2.6%）及[[鑄鐵|鑄鐵]]（不可鍛的鐵碳合金，碳含量大於2.6%）的特性問題。

* 鋼可以[[鍛造|鍛造]]，因為其成份為均質的[[奧氏體|奧氏體]]，而鑄鐵中的碳是以[[石墨|石墨]]或是菜氏體的形式存在，因此延展性變差，不適合鍛造，而且其相變化是在熔化時突然發生。
* 純鐵的[[熔點|熔點]]是1536 [[°C|°C]]，也可以看出鋼及鑄鐵在完全固化（或開始熔化）時的溫度（A-H-I-E線及E-C-F線），鑄鐵開始熔化的最低溫度是在1147 °C，這也說明鑄鐵比鋼更容易用在[[鑄造|鑄造]]的應用上。

基於上述原因，铁碳合金相图是在要了解铁碳合金特性時，很重要的工具。
