== 剧情 ==
该片通过一名美国女记者“露茜”在黑龙江省大庆市目睹和记载“王进喜”开采石油的故事，讲述了中华人民共和国建国之初国人乐观向上的进取开拓和奉献牺牲精神。<ref>{{cite web |url=http://www.dqdaily.com/ztxw/2012/2012-04/12/content_1094781.htm |title=电影《铁人王进喜》简介 |accessdate=2012 |publisher=大庆网 |work= |language=zh-hans}}</ref>

== 花絮 ==
该片是庆祝中华人民共和国建国60周年献礼重点影片。<ref>{{cite web |url=http://ent.sina.com.cn/m/c/2009-02-19/10242382074.shtml |title=《梅兰芳》费二爷出演铁人王进喜 尹力执导(图) |accessdate=2009 |publisher=新浪网 |work= |language=zh-hans}}</ref>

该片拍摄于[[黑龙江省|黑龙江省]][[克拉玛伊油田|克拉玛伊油田]]、[[大庆油田|大庆油田]]等地。<ref name="Tiger" >{{cite web |url=http://www.chinanews.com/yl/dyzx/news/2009/02-19/1570643.shtml |title=“费二爷”变身王进喜 刘烨、黄渤加盟《铁人》 |accessdate=2009 |publisher=中新网 |work= |language=zh-hans}}</ref>

导演尹力钦点新人“白静”配合黄渤、刘桦。<ref>{{cite web |url=http://ent.qq.com/a/20090506/000402.htm |title=导演尹力钦点新人白静 《铁人》搭档刘烨黄渤 |accessdate=2009 |publisher=腾讯网 |work= |language=zh-hans}}</ref>
