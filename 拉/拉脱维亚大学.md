'''拉脱维亚大学'''（{{lang-lv|Latvijas Universitāte}}，缩写LU），坐落在[[拉脱维亚|拉脱维亚]]首都[[里加|里加]]市区，成立于1919年，是该国最大的综合类[[大学|大学]]。<ref>{{cite web |url=http://www.mfa.gov.cn/chn/wjb/zzjg/xos/gjlb/1736/1736x0/default.htm |title=拉脱维亚国家概况 |publisher=[[中华人民共和国外交部|中华人民共和国外交部]] |date=2009年2月16日 |accessdate=2009年2月24日 |language=zh }}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>2007-08学年注册学生数达23801，其中博士生900人。<ref name="studnet"/>现任[[校长|校长]]为Mārcis Auziņš教授。<ref name="rector"/>

== 历史 ==
[[File:Riga_polytechnikcum_university_of_latvia.jpg|thumb]]
拉脱维亚大学于1919年9月28日，在原里加理工学院的基础上成立了拉脱维亚学院（Latvijas Augstskola），1923年改称“拉脱维亚大学”。<ref name="history">{{cite web |url=http://www.lu.lv/eng/general/history.html |title=History |publisher=拉脱维亚大学 |date= |accessdate=2009年2月24日 |language=en |archive-url=https://web.archive.org/web/20090227004527/http://www.lu.lv/eng/general/history.html |archive-date=2009年2月27日 |dead-url=yes }}</ref>

在[[拉脱维亚苏维埃社会主义共和国|苏联时期]]，拉脱维亚大学的一些学科独立出来，成立了[[拉脱维亚农业大学|拉脱维亚农业大学]]、[[拉脱维亚药学院|拉脱维亚药学院]]、[[里加工业大学|里加工业大学]]等高等院校。<ref name="history"/>

1991年，拉脱维亚从苏联独立后不久，拉最高委员会在9月18日通过了拉脱维亚大学章程，规定拉脱维亚大学是“一所为拉脱维亚国家和人民的需求服务的国立教育、科学、文化机构”。<ref name="history"/>

== 教学及研究机构 ==
[[File:Lotyšská_univerzita.jpg|thumb]]
拉脱维亚大学设有13个学院，分别是：[[生物|生物]]学院、[[化学|化学]]学院、[[经济|经]][[管理|管]]学院、[[教育|教育]]与[[心理学|心理]]学院、[[地理|地理]]与[[地球科学|地球科学]]学院、[[历史|历史]]与[[哲学|哲学]]学院、[[法学|法学]]院、[[医学院|医学院]]、现代[[语言学|语言]]学院、哲学与[[艺术|艺术]]学院、[[数学|数]][[物理|理]]学院、[[社会科学|社会科学]]学院、以及[[神学院|神学院]]。<ref name="faculty">{{cite web |url=http://www.lu.lv/eng/faculties/ |title=Faculties |publisher=拉脱维亚大学 |date= |accessdate=2009年2月24日 |language=en |archive-url=https://web.archive.org/web/20090306212531/http://www.lu.lv/eng/faculties/ |archive-date=2009年3月6日 |dead-url=yes }}</ref>

拉脱维亚大学还设有10多个研究所，涉及到众多不同的领域和学科。<ref name="inst">{{cite web |url=http://www.lu.lv/eng/general/structure/institutes/index.html |title=Institutes |publisher=拉脱维亚大学 |date= |accessdate=2009年2月24日 |language=en |deadurl=yes |archiveurl=https://web.archive.org/web/20090306220242/http://www.lu.lv/eng/general/structure/institutes/index.html |archivedate=2009年3月6日 }}</ref>另外，由拉脱维亚、[[瑞典|瑞典]]两国政府以及Soros基金会共同设立的[[里加法学研究生院|里加法学研究生院]]也是拉脱维亚大学的一个独立、自治单元。<ref name="rgsl">{{cite web |url=http://www.rgsl.edu.lv/index.php?option=com_content&task=view&id=54&Itemid=107 |title=About RGSL |publisher=里加法学研究生院 |date= |accessdate=2009年2月24日 |language=en |archive-url=https://web.archive.org/web/20090314032040/http://www.rgsl.edu.lv/index.php?option=com_content&task=view&id=54&Itemid=107 |archive-date=2009年3月14日 |dead-url=yes }}</ref>

== 国际交流 ==
拉脱维亚大学是IAU、EUA、[[乌得勒支网络|乌得勒支网络]]、[[欧洲首都大学联盟|欧洲首都大学联盟]]（UNICA）和波罗的海区域大学网络（BSRUN）等大学联盟的成员。<ref name="IntlOrg">{{cite web |url=http://www.lu.lv/eng/international/membership.html |title=International Organizations and Networks |publisher=拉脱维亚大学 |date= |accessdate=2009年2月24日 |language=en |deadurl=yes |archiveurl=https://web.archive.org/web/20090227004354/http://www.lu.lv/eng/international/membership.html |archivedate=2009年2月27日 }}</ref>

除了通过[[欧盟|欧盟]]的伊拉斯谟（Erasmus）等项目与欧洲范围内的大学进行交流，拉脱维亚大学还与世界上29个国家的76所大学签署了双边合作协议。<ref name="partners">{{cite web |url=http://www.lu.lv/eng/international/cooperation.html |title=University of Latvia has 76 partner institutions in 29 countries |publisher=拉脱维亚大学 |date= |accessdate=2009年2月24日 |language=en |deadurl=yes |archiveurl=https://web.archive.org/web/20090225141817/http://www.lu.lv/eng/international/cooperation.html |archivedate=2009年2月25日 }}</ref>
