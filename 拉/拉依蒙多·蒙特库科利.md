'''拉依蒙多·蒙特库科利伯爵'''（'''Raimondo, Count of Montecúccoli''' or '''Montecucculi'''，{{bd|1608年|2月21日|1680年|10月16日|catIdx=M}}），[[奥地利陆军元帅|奥地利陆军元帅]]和軍事理論家。神圣罗马帝国亲王和[[那不勒斯|那不勒斯]]梅尔菲公爵（Melfi）
== 早年的軍事生涯 ==
他的家族原是[[勃艮第|勃艮第]]人，在10世纪定居在[[意大利|意大利]]北部。在16岁开始蒙特库科利作为他的叔父Count Ernest Montecucculi（一位杰出的奥地利将军，d. 1633）的私人士兵。四年后他成为一名步兵上尉。在1631年布賴滕費爾德（Breitenfeld,1631）和呂岑（Lutzen,1632）對瑞典國王[[古斯塔夫二世|古斯塔夫二世]]（Gustavus II）作戰有功。1639年被瑞軍俘虜，在拘禁期間研究戰爭藝術，並開始寫關於戰略戰術的著作。
== 主帥指揮 ==
他在1661-1664年對土耳其的作戰中獲得勝利。当時土耳其的十二萬大軍正攻向[[维也纳|维也纳]]，蒙特库科利元帅临危受命，率领一支东拼西凑，人数处于劣势的基督教联军，成功的挡住了土耳其大軍，当土耳其軍的锐气被消磨之后，他又大胆的组织了几次进攻，成功戰勝土耳其軍。打了胜仗的皇帝[[利奧波德一世|利奧波德一世]]，与土耳其[[蘇丹|蘇丹]]签订了二十年的休战协定。

1672-1678年的[[法荷戰爭|法荷戰爭]]時，他被擢升為[[神聖羅馬帝國|神聖羅馬帝國]]的全軍主帥，統領反法的[[德意志|德意志]]聯軍反攻法國。1675年他以較多的兵力，與法軍元帥[[杜倫尼|蒂雷納]]相持卻居於下風，但蒂雷納被流彈射死後，他很快就打敗法軍、攻下大部分的[[亞爾薩斯|亞爾薩斯]]；之後在同年，他與法國元帥[[大孔代|大孔代]]正面交鋒，但無法攻下戰略目標，被迫撤軍，把[[亞爾薩斯|亞爾薩斯]]再次讓給法國。
== 退休與死亡 ==
約在[[法荷戰爭|法荷戰爭]]結束前的1676年，他向皇帝[[利奧波德一世|利奧波德一世]]請求退休。在谢绝了皇帝的再三挽留之後，他归隐到[[維也納|維也納]]乡间，撰寫許多軍事行政與科學研究的書籍。1679年他被[[利奧波德一世|利奧波德一世]]授予帝王親王的名位，不久也被[[西班牙|西班牙]]國王給予梅尔菲公爵（Melfi）的身分。1680年10月他在一場事故中過世。
== 評價 ==
他是那个时代极少数在书本中研究军事理论的将军（所谓军人中的文化人），靈巧地活用大規模調遣部隊的理論，以此獲得勝利。作為無與倫比的戰爭高手，他擅長圍攻、強行軍與切斷敵人的部隊聯繫。他和同時代的法國名將[[杜倫尼|蒂雷納]]、[[大孔代|大孔代]]，並列為17世紀中期的軍事天才。
