'''拉裕事件'''是1974年1月31日发生在[[新加坡|新加坡]]的一起[[恐怖事件|恐怖事件]]，日语中又称“新加坡事件”（{{lang-ja|シンガポール事件}}，''Shingapōru jiken''）该事件中[[日本赤軍|日本赤軍]]和[[解放巴勒斯坦人民阵线|解放巴勒斯坦人民阵线]]的四名恐怖分子袭击了位于[[新加坡|新加坡]][[毛广岛|毛广岛]]的[[荷兰皇家壳牌|壳牌]][[煉油廠|煉油廠]]，随即劫持了拉裕号（''Laju''，[[马来语|马来语]]中意为“飞驰”）驳船。

== 事件经过 ==
1974年1月31日，[[日本赤军|日本赤军]]成员[[和光晴生|和光晴生]]、[[山田义昭|山田义昭]]<ref>『ザ・ハイジャック』P.148 佐々淳行著 [[文藝春秋|文藝春秋]] 2010年</ref>以及解放巴勒斯坦人民阵线的两名成员持[[冲锋枪|冲锋枪]]入侵了新加坡毛广岛上的壳牌炼油厂，直接目的是阻止新加坡向[[越南共和國|南越]]等“不友好”国家供应[[燃油|燃油]]。解放巴勒斯坦人民阵线的一名发言人次日称，此举是为了警告所有[[垄断|垄断]]的[[石油|石油]]企业和压迫[[中东|中东]]革命的[[帝国主义|帝国主义]]者。<ref name=smod>{{cite web|author=Ong Kian Seng |url=http://www.mindef.gov.sg/imindef/about_us/history/overview/birth_of_saf/v06n01_history.html|title=1974 - The Laju Incident|publisher=[[新加坡国防部|新加坡国防部]]|date=2002-01-07|accessdate=2014-03-23}}</ref>四名袭击者以捕鱼为由搭乘一艘[[渔船|渔船]]前往毛广岛，随即袭击船主并接管这艘船，但不久之后船即触礁，他们只得要求另一位渔民将他们带上岛。<ref name=NLB>{{cite web|url=http://eresources.nlb.gov.sg/infopedia/articles/SIP_1372_2009-01-15.html|title=Laju hijacking|publisher=National Library Singapore|accessdate=2014-03-23}}</ref>

袭击者登上毛广岛后爬过[[海堤|海堤]]，到达油库门前，试图开枪逼停一辆卡车，但没有成功。岛上的一名[[工程师|工程师]]在驾车经过时也被枪击，但未受伤。炼油厂卫兵成功脱逃，发出警报。袭击者当时携带了12枚炸弹，但只能引爆三枚。其中一个储油罐被引爆起火，但火势迅速被扑灭，最终损失的燃油价值达15000[[新加坡元|新加坡元]]。四个人未能引爆剩余的爆炸物，为逃离现场而在毛广岛码头登上前往[[新加坡岛|新加坡岛]]的拉裕号渡船，并将其劫持。<ref name=NLB/>拉裕号五名船员被恐怖分子要求将船驶入[[公海|公海]]，新加坡海警随即对拉裕号展开跟踪。拉裕号停靠在东安克雷奇后，即被海警船、[[海关|海关]][[摩托艇|摩托艇]]和[[新加坡海军|新加坡海军]]炮艇包围。<ref name=smod/>在劫持五名人质之后，恐怖分子开始与海警谈判。当天夜间，两名人质跳船而逃。

数日后，劫船者以派“保证人”与四人一同乘机赴[[科威特|科威特]]为条件同意释放人质，这引发了{{Link-en|日本駐科威特大使館佔領事件|1974 attack on the Japanese Embassy in Kuwait}}。<ref name=NLB/>2月7日，四名袭击者被带往[[巴耶利峇空軍基地|巴耶利巴机场]]（当时亦称“新加坡国际机场”），投案后释放人质并登上[[日本航空|日本航空]]的[[道格拉斯DC-8|道格拉斯DC-8]]特派机（编号JA8032，曾于1968年在[[旧金山湾|旧金山湾]]发生[[日本航空2号班机事故|事故]]）。8日凌晨1时25分，载着四名劫船者、13名[[新加坡政府|新加坡政府]]官员、2名[[日本政府|日本政府]]官员和12名机组人员的日航专机离开新加坡，13名新方保证人中包括时任[[新加坡保安与情报司|新加坡保安与情报司]]司长[[塞拉潘·纳丹|塞拉潘·纳丹]]。<ref>{{cite news|url=http://news.asiaone.com/print/News/Elections/Story/A1Story20110829-296828.html|title=I wasn't sure if I'd return alive: Nathan|publisher=The New Paper|date=2011-08-29|accessdate=2014-03-23}}</ref>
