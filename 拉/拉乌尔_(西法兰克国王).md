'''拉乌尔'''（'''{{lang-fr|Raoul de France}}'''；{{bd|890年||936年|1月15日|catIdx=R}}），又名'''鲁道夫'''（'''{{lang-fr|Rodolphe}}'''），本是[[勃艮第公爵|勃艮第公爵]]（921年—923年在位），后成为[[西法兰克王国|西法兰克王国]]的[[国王|国王]]（常等同于[[法国国王|法国国王]]，923年—936年在位）。他从他的父亲、[[勃艮第公爵|勃艮第公爵]][[理查_(勃艮第公爵)|理查]]处继承了[[勃艮第|勃艮第]][[公国|公国]]。他娶西法兰克国王[[罗贝尔一世_(西法兰克)|罗贝尔一世]]的女儿[[法兰西的埃玛|法兰西的埃玛]]为妻，故得以在岳父死后，以及罗贝尔之子伟大的于格没有争夺王位意愿的情况下以[[女婿|女婿]]的身份继承西法兰克王位。他属于与[[加洛林王朝|加洛林王朝]]敌对的[[卡佩家族|卡佩家族]]，是后来的[[法兰西王国|法兰西王国]][[卡佩王朝|卡佩王朝]]诸王的祖先。

== 生平 ==

[[罗贝尔一世_(西法兰克)|罗贝尔一世]][[驾崩|驾崩]]后，没有留下男嗣。由于按照[[法兰克人|法兰克人]][[传统|传统]]的《[[萨利克继承法|萨利克继承法]]》，[[女性|女性]]不能继位，所以身为女婿的拉乌尔被[[贵族|贵族]]会议[[选举|选举]]为国王。923年7月13日[[星期天|星期天]]，[[桑斯|桑斯]][[大主教|大主教]][[沃尔特_(桑斯大主教)|沃尔特]]在[[苏瓦松|苏瓦松]]的[[圣梅达教堂|圣梅达教堂]]为拉乌尔[[加冕|加冕]]。即位后，拉乌尔将担任不到两年的[[勃艮第公爵|勃艮第公爵]]一职让给了弟弟[[黑于格|黑于格]]。

但是，被贵族们废黜的[[查理三世_(西法兰克)|夏尔三世]]仍然健在，他拒绝承认拉乌尔，并且仍然宣称自己是西法兰克的国王。不过，属于[[加洛林王朝|加洛林王朝]]的[[韦芒杜瓦伯爵|韦芒杜瓦伯爵]][[赫伯特一世|赫伯特一世]]却背叛了本家族，与身为[[博索尼德王朝|博索尼德王朝]]成员的拉乌尔结盟。赫伯特一世娶拉乌尔妻子艾玛的妹妹为妻，所以他们俩是[[连襟|连襟]]。赫伯特一世欺骗了夏尔三世，在假装与他会谈的时候逮捕了夏尔并将他投入[[监狱|监狱]]。

== 死亡 ==

935年，国王犯病，于第二年死在[[欧塞尔|欧塞尔]]。死因是“身上的[[阴虱|阴虱]]增殖，扩散到全身”。死后被葬在圣哥伦布附近的[[桑斯|桑斯]],我们现在仍然可以看见那里的保存着他被遗弃的空石棺的修道院地下室。他像他父亲一样葬在大教堂的祭坛上，但是他的墓周围明显丰富了许多。他的墓有四个国王雕像柱支撑着，在底部刻着哥特式的字符：鲁道夫国王。教堂[[宗教战争|宗教战争]]时被[[新教徒|新教徒]]破坏。在他们离开后，修道院的合唱团提出了重建计划。1721年，在教堂的重建过程中，人们在坚硬的水泥下发现了纪念墙和摇篮形拱顶。挖出来后，人们发现一些灰尘，这是拉乌尔的骨灰。最终在1792年时，最后残余的纪念碑也灰飞烟灭了。

== 配偶与子女 ==

拉乌尔与[[法兰西的艾玛|法兰西的艾玛]]结婚，有两个早夭的子女：

* 路易（？—929年6月14日）
* 朱迪丝（？—929年6月14日）
