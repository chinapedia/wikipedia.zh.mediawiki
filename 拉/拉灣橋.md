'''拉灣橋'''坐落於[[臺灣|臺灣]][[臺東縣|臺東縣]][[金峰鄉_(台灣)|金峰鄉]][[嘉蘭部落|嘉蘭部落]]及[[拉灣部落|拉灣部落]]中間跨越[[太麻里溪|太麻里溪]]，為兩部落之對外重要聯絡橋樑，由[[臺東縣政府|臺東縣政府]]進行維護與管理。<ref>{{cite web |url=http://www.taitung.gov.tw/Personnel/News_Content.aspx?n=E4FA0485B2A5071E&sms=E13057BB37942D3F&s=9723187808A57D34 |title=台東縣政府 人事處 台東金峰鄉拉灣橋10日熱鬧通車|date=2014-04-10 |accessdate=2014-04-10 }}</ref>

== 建設資訊 ==
[[莫拉克風災|莫拉克風災]]後的拉灣橋新橋全長348公尺、寬11公尺，為預力箱型梁橋型式，橋身共4跨，三座橋墩，主要溪水流槽區採長跨度設計，以避免阻礙太麻里溪水流，最大跨距達150公尺，為[[中華民國|中華民國]]第三長的預力箱型橋梁，並且也將是臺東縣境內第一座採推進工法施作的大型橋梁工程。<ref>{{cite web |url=http://news.housefun.com.tw/news/article/78344528619.html |title=台東金峰拉灣橋通車　嘉蘭不再是孤島|date=2013-04-12 |accessdate=2013-04-12 }}</ref><ref>{{cite web |url=http://www.taitung.gov.tw/News_Content.aspx?n=7F245332EA31CEB4&s=EDC49D78BD0A4E0B |title=台東縣政府 拉灣橋新建工程|date=2010-06-28 |accessdate=2010-06-28 }}</ref>

拉灣橋重建計畫經費由中華民國[[經濟部|經濟部]][[水利署|水利署]]補助工程經費1億4,500萬元重建之。<ref>{{cite web |url=http://www.pcc.gov.tw/epaper/10202/news_18.htm |title=工程會快訊 臺東縣金峰鄉拉灣橋 目標農曆年前通車 |date=2013-04 |accessdate=2013-04 |deadurl=yes |archiveurl=https://web.archive.org/web/20141013082033/http://www.pcc.gov.tw/epaper/10202/news_18.htm |archivedate=2014年10月13日 |df= }}</ref>

== 風災毀損 ==

拉灣橋舊橋為傳統的混凝土橋，在2009年8月8日莫拉克颱風來襲，遭到暴漲的太麻里溪沖毀，造成當時亦將太麻里溪下游的[[台鐵|台鐵]][[南迴線|南迴線]]太麻里溪橋與鄰近的[[嘉蘭橋|嘉蘭橋]]一併沖毀，災後，金峰鄉嘉蘭與拉灣部落當地民眾往返僅能繞道[[台9線|台9線]]太麻里溪公路橋或冒險橫渡溪底便道，而中華民國[[行政院|行政院]]重建會[[太麻里溪流域復建專案計畫|太麻里溪流域復建專案計畫]]督導管控下，重新檢討太麻里溪治理計畫及堤線，以擴大通洪斷面，降低重覆致災風險，並指示經濟部水利署編制易淹水計畫補助工程經費新台幣1億4,500萬元來辦理拉灣橋重建。<ref>{{cite web |url=http://www.ettoday.net/news/20130319/177622.htm |title=不用再繞道台9線　台東金峰拉灣橋20日通車 ETtoday 新聞雲 |date=2013-04 |accessdate=2013-04 }}</ref>
