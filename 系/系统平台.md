'''系统平台'''（{{lang-en|computing platform}}）是指在[[電腦|電腦]]裡讓[[软件|软件]]运行的系统环境，包括[[硬件|硬件]]环境和[[软件|软件]]环境。典型的系統平台包括一台電腦的[[電腦系統結構|硬件体系结构]]（computer architecture）、[[作業系統|作業系統]]、[[運行時庫|運行時庫]]等。<ref>{{cite web|url=http://www.thefreedictionary.com/Computing+platform|title=Platform|work=The Free Dictionary|publisher=Farlex|accessdate=2013-05-10}}</ref>

== 组成部分 ==
系统平台可能包括：
* 硬件本身，如一些嵌入式系统，不需要操作系统，直接访问硬件。
* 基于Web的软件使用的浏览器。浏览器本身也是在一个系统平台上运行的，但是浏览器里的应用并不关心。<ref>[http://www.informationweek.com/applications/google-redefines-browser-as-platform/d/d-id/1105162 Information Week]</ref>
* 应用程序，应用程序中可以支持一些脚本语言，比如Excel中的巨集。<ref>[http://msdn.microsoft.com/en-us/library/bb906062.aspx Microsoft]</ref>
* 提供一些功能的软件框架。
* 作为服务的云集算平台。<ref>[http://www.interoute.com/what-paas Interoute]</ref> 社交网络 Twitter 和 facebook 等也可以看作一个开发平台。<ref>[https://dev.twitter.com/ Twitter]</ref><ref>[https://www.facebook.com/notes/2207512130 Facebook]</ref>
* 虚拟机(VM)如 Java 虚拟机。<ref>[http://www.oracle.com/technetwork/topics/newtojava/intro-139083.html Oracle]</ref> 应用被编译成和机器码类似的字节码，可以被虚拟机执行。
* 完整系统的虚拟化版本。包括虚拟硬件、操作系统、软件和存储。
