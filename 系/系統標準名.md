'''系統標準名'''是一種以系統的方式為具體的獨一無二的群體、有機體、物件或[[化學物質|化學物質]]，給定名稱。系統的名字通常是[[名稱|名稱]]的一部分。<br /> 
'''半系統標準名''' 或'''半種名'''是至少有一[[種名|種]]或部分有系統標準名。
<ref>Stedman's Medical Dictionary: [http://www.mondofacto.com/facts/dictionary?semisystematic+name ''semisystematic name'']</ref><ref>{{GoldBookRef|title=semisystematic name (semitrivial name)|file=S05601}}</ref>。

系統標準名的創建可以很簡單，只是為每個物件分配[[字首|字首]] (在這種情況下，它們的類型是[[編碼制|編碼制]])，或是為每一個物體分配複雜的編碼和完整結構的名稱。許多系統會結合一些額外的序號，使之成為有關命名物件的一個[[唯一識別碼|唯一識別碼]]。

系統標準名經常會與系統建立之前的名稱共存。例如，許多常見的化合物被提到時，仍會共同使用種名，甚至化學家的名字。

== 在化學 ==
在化學，系統標準名會描述該[[化學物質|化學物質]]的化學結構，從而得知與其性質有關的一些資訊。<br />
在[[IUPAC命名法|IUPAC命名法]]，[[化學術語總目錄|金皮書]]給出了定義：<br />
''"完整的名稱組成包含專門的輔佐喊所選的音節，帶或不帶數值的字首。例如：戊烷、噁唑<ref>{{GoldBookRef|title=systematic name|file=S06236}}</ref>。
此處，種名可以是系統標準名或是它的一部分，因為它們已經成為IUPAC命名法的一部分。例如，[[苯|苯]] (cyclohexatriene ) 或 [[甘油|甘油]] (丙三醇，trihydroxypropane )。
