'''樊会涛'''（{{bd|1962年|10月18日|}}），[[河南|河南]][[汝阳|汝阳]]人，[[中国空空导弹研究院|中国空空导弹研究院]]总设计师、副院长、研究员。[[中国工程院院士|中国工程院院士]][[汉族|汉族]]，[[中华人民共和国|中华人民共和国]]政治人物、[[第十二届全国人民代表大会|第十二届全国人民代表大会]]河南地区代表。

== 生平 ==
1986年4月毕业于[[西北工业大学|西北工业大学]]发动机系发动机专业。<ref>{{Citeweb |url=http://www.avic.com.cn/cn/qywh/hkym/367869.shtml |title=赤子之心 蕴持伟力——樊会涛 |deadurl=yes |archiveurl=https://web.archive.org/web/20131219230820/http://www.avic.com.cn/cn/qywh/hkym/367869.shtml |archivedate=2013-12-19 }} </ref> 主持研制出了具有世界先进水平的空空导弹，实现了中国空空导弹由第三代向第四代的跨越，填补了中国空空导弹领域发展的多项技术空白。2007年，被中共中央、国务院、中央军委授予“高技术武器装备发展建设工程重大贡献奖”，2010年，被中央组织部、国防科工局、人力资源部、中国科学院、中国工程院授予“第二届国防科技工业杰出人才奖”，荣获第六届光华工程奖青年奖。此外，还获得国家科技进步奖一等奖1项，国防科技成果特等奖1项、一等奖1项，国防发明专利6项，出版专著3部，发表论文28篇。<ref>{{Cite web |url=http://newpaper.dahe.cn/dhb/html/2013-12/20/content_1005053.htm?div=-1| title= 洛阳“本土”科学家樊会涛当选中国工程院院士}} </ref>

2013年，担任全国人大代表<ref>{{cite news|title=全国人大代表信息-樊会涛|url=http://www.npc.gov.cn/delegate/viewDelegate.action?dbid=121519|accessdate=2013-04-05|newspaper=全国人大网}}</ref>。

2013年12月19日，当选[[中国工程院院士|中国工程院院士]]。<ref>{{Cite web |url= http://www.cae.cn/cae/html/main/col66/2013-12/19/20131219091148980103712_1.html |title= 中国工程院2013年院士增选结果 |deadurl= yes |archiveurl= https://web.archive.org/web/20131219212015/http://www.cae.cn/cae/html/main/col66/2013-12/19/20131219091148980103712_1.html |archivedate= 2013年12月19日 |df=  }} </ref>
