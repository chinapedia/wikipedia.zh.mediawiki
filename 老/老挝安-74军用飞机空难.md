2014年5月17日，一架属于[[老挝人民军|老挝军方]]的{{link-en|安-74|Antonov An-74}}飞机在从[[老挝|老挝]][[万象|万象]]飞往老挝[[川圹省|川圹省]]的途中坠毁。此机型于不久前开始在全国范围内运送重要人物。这架安-74飞机上的乘客包含[[老挝|老挝]]国防部长[[隆再·皮吉|隆再·皮吉]]、公安部长及其夫人，以及老挝首都[[万象|万象]]市长以及一些其他老挝领导层代表。他们原计划前往参加庆祝[[老挝人民军|老挝人民军]]第二师成立55周年的阅兵式<ref name="hqw">{{cite web |url = http://world.huanqiu.com/article/2014-05/4998306.html |title = 军机在老挝坠毁 俄媒称老挝防长公安部长遇难 |publisher = 环球网 |accessdate=2014-05-17}}</ref>。

该机上载有17人，其中14人遗体已送回首都[[万象|万象]]。老挝官方随后就此次空难举办新闻发布会。目前已确定罹难人员包括老挝副总理兼国防部部长[[隆再·皮吉|隆再·皮吉]]中将及其夫人、老挝公安部部长[[通班·显阿蓬|通班·显阿蓬]]、[[老挝人民革命党|老挝人民革命党]]中央宣传部部长[[征·宋本坎|征·宋本坎]]、万象市长[[苏甘·马哈拉|苏甘·马哈拉]]等。生还的3人中一人为副驾驶，一人为随机护士，另一人身份不明，而之后其中两人经抢救无效死亡<ref>{{cite web |url = http://news.163.com/14/0517/16/9SFAL2FC00014JB5.html |title = 老挝失事军用飞机罹难人员名单确认 |publisher = 国际在线 |accessdate=2014-05-17}}</ref>，仅随机护士生还。

== 坠毁 ==
2014年5月17日，当地时间6:15<ref name="Reuters">{{cite web |url = http://uk.reuters.com/article/2014/05/17/uk-laos-crash-idUKKBN0DX03320140517 |title = Laos air force plane crashes, defence minister reported dead |work = Reuters |date = 2014-05-17 |accessdate = 2014-05-17}}</ref>、6:30<ref name="Xinhuanet" />或7:00<ref name="KPL">{{cite web |url = http://www.kpl.net.la/english/news/edn1.htm |title = Plane crashes in Xiengkhouang province |work = KPL |date = 2014-05-17 |accessdate = 2014-05-17 |deadurl = yes |archiveurl = https://web.archive.org/web/20140517144431/http://www.kpl.net.la/english/news/edn1.htm |archivedate = 2014-05-17 }}</ref>左右，该机坠毁于老挝[[川圹省|川圹省]]。坠机地点距离目的地川圹省{{link-en|川圹机场|Xieng Khouang Airport}}<ref name="Xinhuanet">{{cite web |url = http://news.xinhuanet.com/english/world/2014-05/17/c_133341024.htm |title = At least five confirmed dead, three survive in air crash in northeastern Laos |work=Xinhuanet |date=2014-05-17 |accessdate = 2014-05-17 |author = Peng, Fu}}</ref><ref name="ASN" />1500米<ref name="Xinhuanet" />或2000米<ref name="Reuters" />，距离起飞地万象[[瓦岱國際機場|瓦岱國際機場]]500千米<ref name="ASN">{{cite web |url = http://aviation-safety.net/database/record.php?id=20140517-0 |title = Accident description |work = Aviation Safety Network |accessdate = 2014-05-17}}</ref>。

这是[[老挝历史|老挝历史]]上死亡人数第二多的空难事件，仅次于2013年10月的[[老挝航空301号航班空难|老挝航空301号航班空难]]<ref name="BBC">{{cite web |url = http://www.bbc.co.uk/news/world-asia-27452919 |title = Laos Defence Minister Douangchay Phichit's plane crashes |work = BBC |date = 2014-05-17 |accessdate = 2014-05-17}}</ref>。
