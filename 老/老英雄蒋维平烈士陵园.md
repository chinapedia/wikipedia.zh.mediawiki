'''老英雄蒋维平烈士陵园'''，位于[[北京市|北京市]][[房山区|房山区]][[石楼镇_(北京市)|石楼镇]]石楼村，为纪念[[中国人民解放军|中国人民解放军]]中的老英雄[[蒋维平|蒋维平]]而建立。

==简介==
蒋维平原籍[[清朝|清朝]][[河间府|河间府]][[阜城县|阜城县]]蒋庄村（今[[河北省|河北省]][[衡水市|衡水市]][[阜城县|阜城县]]蒋庄村），后离家出走从军，辗转落户在河北省房山县石楼村（今北京市房山区石楼镇石楼村）。<ref>{{Cite web |url = http://news.ifeng.com/a/20150629/44061364_0.shtml |title =劳动英雄蒋维平籍贯考  |author =  关向东|date = 2015-06-29 |publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref>[[抗日战争|抗日战争]]期间，1938年2月蒋维平60岁参加[[八路军|八路军]][[359旅|359旅]]，同年4月加入[[中国共产党|中国共产党]]。他担任军医，并积极参加[[大生产运动|大生产运动]]，成了[[陕甘宁边区|陕甘宁边区]]有名的老英雄。77岁被授予[[中国人民解放军|中国人民解放军]]中校军衔，成为[[中华人民共和国|中华人民共和国]]成立后获授军衔的最年长者。1964年，蒋维平在[[张家口|张家口]]逝世。安葬在[[察哈尔烈士陵园|察哈尔烈士陵园]]。<ref>{{Cite web |url =http://www.71.cn/2015/1203/850745.shtml  |title = 草色青青 |author =  |date = 2015-12-03 |publisher =宣讲家网  |language =  |accessdate =  |quote =  }}</ref>

老英雄蒋维平烈士陵园位于石楼镇文体广场，属于衣冠冢。有1993年兴建的老英雄蒋维平半身石雕像。雕像基座正面刻有[[杨成武|杨成武]]的题字“老英雄蒋维平”（落款“杨成武”），背面有碑文：<ref>{{Cite web |url =  http://mt.sohu.com/20160322/n441570778.shtml|title =贾岛故居石楼镇  |author =  |date = 2016-03-22 |publisher =搜狐  |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://bj.wenming.cn/qxdt/201304/t20130408_1163302.html |title = 房山：传承清明文化 倡导文明祭扫 |author =  |date =  2013-04-08|publisher = 中国文明网 |language =  |accessdate =  |quote =  }}</ref>

蒋维平系房山石楼村人。一八七八年生，一九六*四年病逝，享年八十六岁。他一生学过徒，当过长工，戎马转战，寻求救国。一九三八年三月，他携义子参加革命。同年五月，加入中国共产党。在[[延安整风|延安整风]]和大生产运动中，被评为陕甘宁边区劳动英雄、全国劳动模范，曾受到[[毛主席|毛主席]]表扬。他历任[[白求恩国际和平医院|白求恩国际和平医院]]、察哈尔军区医院副院长等职。[[中华人民共和国国防部|国防部]]授予他中校军衔。他在家乡组织自卫团，办[[红十字会|红十字会]]，建立医务室，引进资金和技术发展生产，并慷慨解囊资助家乡建设，与家乡人心心相印。他对党的事业忠心耿耿，对人民无私奉献，老当益壮德高望重，是深受人们爱戴和崇敬的老英雄。时塑像立碑，永为纪念。<br>

2012年，“老英雄蒋维平烈士陵园”被列为[[北京市房山区普查登记文物|北京市房山区普查登记文物]]。2013年1月挂普查登记文物牌保护。

老英雄蒋维平雕像后面，如今还立有两块石碑。一块是文昌碑，又称文昌帝君碑，[[清朝|清朝]][[嘉庆|嘉庆]]四年（1799年）立。青白石质，方首无座，碑高1.1米，宽0.52米，厚0.18米。碑首为祥云浮雕，额题正书“文昌碑记”，碑文为：“自古称大孝者首推[[虞舜|虞舜]][[文昌帝君|文昌帝君]]，继舜而称孝者也，其所著[[孝经|孝经]]与十八章相表里，盖木至性所发言……嘉庆四年岁在己未七月立。”2012年，“文昌碑”被列为[[北京市房山区普查登记文物|北京市房山区普查登记文物]]。

另一块是王辅三墓志碑，青白石质，方首无座，碑阳额篆题“王君辅三墓志”，碑身首题“君姓王氏讳邦屏字辅三房山石楼人也…”，落款“[[中华民国|中华民国]]二十年夏正五月榖日立”；碑阴额题“碑阴”，碑身仅在落款处刻有人名，可能是碑文内容没刻完。
