'''老锣'''（{{lang-de|'''Robert Zollitsch'''}}，直译为'''罗伯特·佐里奇'''，{{bd|1966年|4月9日}}）是中国新式[[音乐|音乐]]的[[德国|德国]][[作曲家|作曲家]]和[[音乐制作人|音乐制作人]]。老锣将自己定位成一个[[亚洲|亚洲]][[传统音乐|传统音乐]]的专家和当代中国音乐重要的作曲家之一。他最著名的作品是流传于中国大陆[[网络|网络]]的民歌《[[忐忑_(龚琳娜)|忐忑]]》。

==生平==
1966年出生德国[[慕尼黑|慕尼黑]]。早期童年他就开始学习[[巴伐利亚|巴伐利亚]]的民俗乐器，[[巴伐利亚古筝|巴伐利亚古筝]]。1986年到1990年，他就读于{{tsl|de|Hochschule für Musik „Hanns Eisler“ Berlin|汉斯·艾斯勒音乐学院}}学习音乐[[理论|理论]]。1993年他靠奖学金进入[[上海音乐学院|上海音乐学院]]师从[[龚一|龚一]]学习[[古琴|古琴]]。<ref>[http://book.people.com.cn/BIG5/n/2013/0416/c69398-21155919.html 揭龔琳娜與老鑼的愛情秘事 曾遭龔媽媽強烈反對]</ref>

老锣精通[[中国|中国]]、[[蒙古|蒙古]]和[[西藏|西藏]]音乐。他和中国的音乐家或歌手（如[[龚琳娜|龚琳娜]]、[[乌仁娜|乌仁娜]]、[[杜聪|杜聪]]、[[罗燕|罗燕]]等）一同创作了广泛使用[[和声|和声]]，融合不同文化元素的中国[[古典音乐|古典音乐]]。这些音乐听起来似乎是中国的传统音乐风格受到西方音乐的影响，有时听起来像是[[莫扎特|莫扎特]]或[[安东尼奥·维瓦尔第|安东尼奥·维瓦尔第]]的风格，有时又像[[流行音乐|流行音乐]]的风格。这些风格的集合形成了独具一格的“老锣风格”。<ref name="HKCO">香港中樂團网站</ref><ref>{{cite web |url=http://www.bayerischerhof.de/de/presse/bands/gonglinnaundrobertzollitsch/index.php |title=存档副本 |accessdate=2010-08-20 |deadurl=yes |archiveurl=https://web.archive.org/web/20100214004711/http://www.bayerischerhof.de/de/presse/bands/gonglinnaundrobertzollitsch/index.php |archivedate=2010-02-14 }} 巴伐利亚霍夫音乐会的新闻信息</ref>

迄今为止，他已经通过自己的品牌[[酷酷音乐|酷酷音乐]]制作了多个当代中国艺术音乐，在[[欧洲|欧洲]]、[[台湾|台湾]]和[[东南亚|东南亚]]轰动一时。他的中国听众将他的音乐描述为“十分中国化，但有一种新颖的触动”。<ref name="HKCO" />

他的配偶是中国歌唱家[[龚琳娜|龚琳娜]]。他为龚琳娜撰写了一些专辑如“静夜思”（2006年）和“夜雪”（2010年）。
他與龔琳娜於2004年2月结婚，并育有两个儿子海酷（2005年4月14日-）、雅酷（2007年6月17日-）。

自2003年起，老锣开始定居于中国[[北京|北京]]。
2011年元旦时，龚琳娜在[[湖南卫视|湖南卫视]]跨年晚会上演唱了他的实验性作品《[[忐忑_(龔琳娜)|忐忑]]》，使《忐忑》开始流行于中国，特别是中国[[网民|网民]]中间。

2012年至2013年新旧岁交替老锣夫妇在跨年晚会的“神曲”《法海你不懂爱》、《金箍棒》，因歌词及表演被指戏谑佛教，遭佛教界严正抗议及要求致歉，导致[[法海事件|法海事件]]。<ref>{{cite web |url=http://fo.ifeng.com/special/wendaotj1 |title=法海事件专辑1：别拿信仰开玩笑 不是道歉那点事 |accessdate=2013-03-17 |publisher= 凤凰网|archiveurl= |archivedate=  |work= |language=zh-cn}}</ref><ref>{{cite web |url= http://fo.ifeng.com/special/ganfashengsjb/ |title= 法海事件专辑2：敢发生说话是历史的进步|accessdate=2013年3月17日 |publisher= 凤凰网|archiveurl= |archivedate=  |work= |language=zh-cn}}</ref>
