'''老山'''是位于中国[[北京市|北京市]][[石景山区|石景山区]]东部的一座小山，也是当地地名。

==简介==
老山位于[[石景山区|石景山区]]东部，乃[[永定河|永定河]]冲击扇上的丘陵状岛山，海拔最高点为130.4米<ref>[http://www.bjdfz.gov.cn/ShowNewsLevel3.jsp?NewsID=1371 老山汉墓，北京市地情资料网，2009-2-25]</ref>。老山位于古金河河北侧，该河现已无存。老山东西长约1.7公里，南北宽约0.8公里。其内海拔70米以上的面积约为1.3平方公里。1990年代，其范围东隔上庄、良安村与[[八宝山|八宝山]]为邻，西南紧临八角村并与[[八角山|八角山]]相连，南隔[[石景山路|石景山路]]与焦家村、双女坟村相对<ref>北京市石景山區地名誌，北京科学技术出版社，1991年，第86页</ref>。

如今，老山以北为从[[京石高速公路|京石高速公路]][[大瓦窑桥|大瓦窑桥]]南侧延伸至[[海淀区|海淀区]]南坞的铁路支线，铁路北侧有平行的[[田村山南路|田村山南路]]。老山以东为[[上庄大街|上庄大街]]，隔街为[[八宝山人民公墓|八宝山人民公墓]]暨[[八宝山|八宝山]]。老山南麓有[[老山汉墓|老山汉墓]]。老山西侧为[[老山西街|老山西街]]。老山奥运场馆群就分布在这里。其中，老山西街5号为[[中国现代五项运动协会|中国现代五项运动协会]]、[[老山自行车馆|老山自行车馆]]、[[老山体育健身中心|老山体育健身中心]]，老山西街6号为[[中国自行车运动协会|中国自行车运动协会]]，老山西街9号为[[老山小轮车赛场|老山小轮车赛场]]。

2005年11月20日，来华访问的[[美国总统|美国总统]][[布什|布什]]在老山自行车训练场骑[[自行车|自行车]]75分钟健身<ref>{{Cite web |url = http://news.163.com/05/1120/16/231039VJ0001121S.html |title =美国总统布什在北京老山自行车训练场骑车  |author =  |date = 2005-11-20 |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref><ref name=yaoqing/>。

2006年5月31日，老山山地自行车场在老山开工建设<ref>{{Cite web|url =http://news.xinhuanet.com/olympics/2006-05/31/content_4628802.htm|title =08奥运老山山地自行车场工程开工 明年5月竣工|author =|date =2006-05-31|publisher =新华网|language =|accessdate =|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20140901191737/http://news.xinhuanet.com/olympics/2006-05/31/content_4628802.htm|archivedate =2014-09-01}}</ref>。2007年9月竣工<ref name=yaoqing>{{Cite web |url =  http://sports.sina.com.cn/o/2007-09-13/07273166074.shtml|title =老山山地自行车场通过竣工验收 9月举办国际邀请赛  |author =  |date = 2007-09-13 |publisher =  新浪|language =  |accessdate =  |quote =  }}</ref>。该场地是[[2008年夏季奧林匹克運動會自行車比賽|2008年北京奥运会自行车]]项目中的山地自行车赛事的比赛场地。该场地毗邻[[老山自行车馆|老山自行车馆]]、[[老山小轮车赛场|老山小轮车赛场]]，是北京奥运会的北京市11个改扩建体育场馆之一。此次改扩建工程包括周长大约为6.2公里的比赛赛道，8725平方米的固定设施，2000个座位的临时看台及其他竞赛服务临时设施等。工程新建2栋竞赛服务楼，山地自行车赛道长度4.6公里<ref>{{Cite web |url = http://2008.sohu.com/20060531/n243503135.shtml |title = 2008奥运老山山地自行车场改扩建工程全面启动 |author =  |date =2006-05-31 |publisher = 搜狐 |language =  |accessdate =  |quote =  }}</ref><ref name=sohu>{{Cite web |url =http://travel.sohu.com/20090812/n265898605.shtml  |title =老山山地车场变形记 体育主题公园  |author =  |date = 2009-08-12 |publisher =  搜狐|language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url =http://sports.sina.com.cn/o/2007-09-17/14503175534.shtml  |title =老山山地自行车场竣工 静待本周末国际邀请赛举行  |author =  |date =2007-09-17  |publisher = 新浪 |language =  |accessdate =  |quote =  }}</ref>。2007年9月22日，“[[好运北京|好运北京]]”国际山地自行车邀请赛在老山山地自行车场举行<ref name=yaoqing/>。

北京奥运会结束后，老山山地自行车场自2008年起开始改建为老山城市休闲公园。2009年5月1日，老山城市休闲公园建成并对外开放。这是北京市第一个自然式山地公园，免费向游人开放<ref>[http://www.sjs.gov.cn/zjsjs/xwzx/mtyzdsjs/66000.shtml 劳动午报：昔日荒山土坡变身休闲公园，中国石景山，于2013-04-06查阅]{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref>[http://www.bjyl.gov.cn/hdjl/zxts/ldxx/201204/t20120424_105540.html 关于老山城市休闲公园的水景景观问题，首都园林绿化政务网，2012-05-07]</ref><ref>[http://bj.esf.sina.com.cn/news_show.php?id=24673 老山城市休闲公园，新浪，2011-04-26]{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。该公园的前身是2008年北京奥运会老山山地自行车场<ref name=sohu/>。该公园东到[[上庄大街|上庄大街]]，东南方向隔[[老山南路|老山南路]]为上庄，西到[[老山西街|老山西街]]，南为首钢老山居民区（[[老山南路|老山南路]]横穿该居民区，居民区东北角有[[老山东街|老山东街]]），北临铁路及[[田村山南路|田村山南路]]。老山位于该公园内的北侧，[[老山汉墓|老山汉墓]]也位于公园内。2008年北京奥运会老山山地自行车场的赛道作为纪念被留在该公园内。该公园由石景山园林局设计，石景山园林局的设计者专为该公园设计了两座观景台。其中一座观景台位于山顶，海拔约120米，向北可见[[颐和园|颐和园]]、[[玉泉山|玉泉山]]。另一座观景台位于山下，海拔约80米，向西可见老山奥运场馆群、[[首钢|首钢]]总公司，向南可见石景山区人民政府大楼、[[万达广场_(中国大陆)|万达广场]]、[[京燕饭店|京燕饭店]]<ref>[http://www.sgdaily.com/Html/shsj/2009-4/30/094811904.html 老山城市休闲公园 踏青休闲的新天地，首钢日报，2009-4-30]{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

老山及周边地区属于[[老山街道|老山街道]]管辖。
