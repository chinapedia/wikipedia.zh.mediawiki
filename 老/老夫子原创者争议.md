'''老夫子原创者争议'''，是指著名漫画形象[[老夫子|老夫子]]，被质疑并非香港漫画家[[王家禧|王泽]]原创，真正原创者是民国时期的著名漫画家[[朋弟|朋弟]]。

== 事件背景 ==
《老夫子》系列漫画是香港著名漫画人物，由香港漫画家王泽（本名[[王家禧|王家禧]]）创作。《老夫子》在华人世界很受欢迎，使漫画家王泽获得巨大声誉<ref>{{Cite news|url=http://news.ltn.com.tw/news/life/paper/801241|title=老夫子 百幅價4千萬|author=湯佳玲|date=2014-08-03|work=自由时报|accessdate=}}</ref>。《老夫子》被海内外媒体称为“中国人漫画创作的奇迹”、“全世界规模最大的华人漫画”，著名的法兰克福书展也将《老夫子》作为中国图书的唯一标志<ref>{{Cite news|url=http://www.gmw.cn/01ds/2001-08/22/02-8990481E0B78A20C48256AAF0082F5A6.htm|title=冯骥才写书为朋弟说话王泽反应低调|last=|first=|date=2001-08-22|work=中华读书报|accessdate=}}</ref>。

== 事件经过 ==
1983年，著名漫画家朋弟去世。朋弟在抗战时期就创作了“老夫子”、“老白薯”等漫画形象<ref>{{Cite news|url=http://www.people.com.cn/GB/paper69/4203/487828.html|title=朋弟的“老夫子”与王泽的“老夫子”|last=|first=|date=1999年7月|work=讽刺与幽默|accessdate=}}</ref>，在三四十年代的北方很受欢迎。

九十年代初，朋弟的朋友就已经知晓王泽的抄袭行为，曾带着朋弟的老夫子作品去香港。朋弟的朋友找到香港文化圈人士[[彭志銘|彭志铭]]、杨维邦帮忙，将抄袭情况写成专题发在《漫画读物》上<ref>{{Cite news|url=http://hk.apple.nextmedia.com/supplement/culture/art/20130131/18151358|title=元氣堂：老夫子　抄足五十年？|date=2013年1月31日|work=苹果日报|accessdate=}}</ref>。

1996年，中国著名作家[[冯骥才|冯骥才]]从埃及回国途经新加坡时看到了王泽的《老夫子》漫画，发现名称和造型都与朋弟的创作相同。在1997年至1998年间，冯骥才曾在天津大树画馆接待过王泽，但王泽否认抄袭朋弟作品<ref>{{Cite news|url=http://news.sohu.com/45/12/news146271245.shtml|title=冯骥才:王泽剽窃《老夫子》是文坛最大一桩剽窃案|author=刘江华|date=2001年8月16日|work=北京青年报|accessdate=}}</ref>。

1998年，冯骥才在《今晚报》上发表文章《朋弟的“老夫子”与王泽的“老夫子”》，揭露抄袭情况。年底，藏书家姜德明也在山东画报出版社《老漫画》第六辑的《朋弟的“老夫子”》一文中指称王泽的“老夫子”纯系盗版<ref>{{Cite news|url=http://www.gmw.cn/01ds/2001-08/22/02-8990481E0B78A20C48256AAF0082F5A6.htm|title=冯骥才写书为朋弟说话王泽反应低调|last=|first=|date=2001-08-22|work=中华读书报|accessdate=}}</ref>。

2001年，冯骥才出版《文化发掘：老夫子出土》一书，并举行发行座谈会，向媒体公布[[王澤|王泽]]剽窃朋弟的情况，《老夫子》抄袭情况才在中国文化圈为众人所知<ref>{{Cite news|url=http://hk.apple.nextmedia.com/supplement/culture/art/20130131/18151358|title=元氣堂：老夫子　抄足五十年？|last=|first=|date=2013年1月31日|work=苹果日报|accessdate=}}</ref>。

冯骥才在2001年出书的活动中，曾见到朋弟的家属并建议他们打官司。时任西苑出版社社长的杨宪金也与朋弟的后人联系。但是朋弟家属始终没有组织起来为朋弟维权<ref>{{Cite news|url=http://ent.ifeng.com/zz/detail_2013_02/28/22573437_0.shtml|title=香港代表性漫画风光五十年，抄袭公案陈年未果：一个老夫子，两种人生|last=|first=|date=2013年2月28日|work=时代周报|accessdate=}}</ref>。

== 王泽版和朋弟版的异同 ==
* 朋弟版和王泽版，主角都是老夫子，名字相同，衣着打扮相同。
* 朋弟版重要配角是老白薯，王泽版改名叫大番薯，但身材造型相同。
* 媒体报道最多的一组对比图：王泽版的《真正阿飞舞》与朋弟版的《文舞带打》情节上一样<ref>{{Cite news|url=http://news.ltn.com.tw/news/world/breakingnews/1794774|title=港漫《老夫子》是抄襲？傳靈感來自中國漫畫家|last=|first=|date=2016-08-14|work=自由时报|accessdate=}}</ref>。
* 网友shihhow扫描上传了两组对比图：王泽版的《奇迹》与朋弟版的《胡为胡来》情节上一样，王泽版的《香饵钓鱼》与朋弟版的《钓伴江春》情节上一样<ref>{{Cite web|url=http://blog.omy.sg/shihhow/archives/3030|title=原型與部份橋段抄襲自40年代的老夫子&老白薯，王家禧欠原者朋弟一個澄清，其子王澤則欠一個道歉（多圖證據）|accessdate=|author=shihhow|date=2016年1月11日|publisher=|deadurl=yes|archiveurl=https://web.archive.org/web/20161220133040/http://blog.omy.sg/shihhow/archives/3030|archivedate=2016年12月20日|df=}}</ref>。

== 王泽父子的态度 ==
王泽本名叫王家禧，王泽是他的笔名，也是他长子的名字。

王家禧在八十年代已经移民美国加州，《老夫子》漫画由儿子王泽负责。

[[彭志銘|彭志铭]]提到，当他们通过中间人询问儿子王泽，王泽表示这是对他父亲的污蔑<ref>{{Cite news|url=http://hk.apple.nextmedia.com/supplement/culture/art/20130131/18151358|title=元氣堂：老夫子　抄足五十年？|last=|first=|date=2013年1月31日|work=苹果日报|accessdate=}}</ref>。

时代周报记者曾联系王泽父子询问此事，王家禧称不便接受采访。公司公关人员以邮件回复：“对于这陈年旧闻，我方不予回应。<ref>{{Cite news|url=http://book.sina.com.cn/news/c/2013-02-28/0850427186.shtml|title=“老夫子”涉嫌抄袭冯骥才为原作者打抱不平|last=|first=|date=2013年2月28日|work=时代周报|accessdate=}}</ref>”

== 外界评论 ==
* 《[[麦兜故事|麦兜故事]]》导演袁建滔听说抄袭事件后很震惊，所以拒绝了《老夫子》五十周年的画贺图邀请。他认为批评王泽已经没有意义，但是要还历史一个公道<ref>{{Cite news|url=http://hk.apple.nextmedia.com/supplement/culture/art/20130131/18151358|title=元氣堂：老夫子　抄足五十年？|author=何兆彬|date=2013年1月31日|work=苹果日报|accessdate=}}</ref>。
* 香港漫画家林祥焜也觉得王泽和朋弟的漫画有九成相似，但认为王泽父子也有自己的创作。
