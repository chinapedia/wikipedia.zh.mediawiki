《'''老友狗狗'''》（{{lang-en|'''A Watchdog's Tale'''}}；前名《好狗出更》）是[[香港|香港]][[電視廣播有限公司|電視廣播有限公司]]以[[狗|狗]]作為[[寵物|寵物]]題材拍攝製作的時裝[[劇集|劇集]]，由[[馬浚偉|馬浚偉]]、[[鍾嘉欣|鍾嘉欣]]、[[鄭則士|鄭則士]]及[[邵美琪|邵美琪]]領銜主演，並由[[黃浩然|黃浩然]]及[[唐詩詠|唐詩詠]]聯合主演，監製[[梁材遠|梁材遠]]。此劇為2009無綫節目精選第二季劇集之一。

==故事大綱==
黎俊升 ( [[鄭則士|鄭則士]]飾 ) 加入警犬隊25年來由領犬員晉升至教官，並計劃好將退休金拿來開一所狗場，為狗隻、狗主提供訓練和玩樂的場地。俊升有一乖巧女兒黎羨如 ( [[鍾嘉欣|鍾嘉欣]]飾 ) 。羨如受父親影響，愛動物如愛人，更赴澳攻讀獸醫。回港後，在一所名獸醫診所工作，深感人們養狗知識有限，立志將照顧狗隻知識廣傳開去，決定加入父親的狗場做駐場獸醫。俊升在徒弟何天佑 ( [[黃浩然|黃浩然]]飾 ) 穿針引線下，向其表哥周用恭 ( [[馬浚偉|馬浚偉]]飾 ) 租用地點偏僻的長勝村一幅地開設狗場。一次意外下，用恭被羨如弄得意外受傷，出醜人前，再加上羨如外表不修邊幅，對羨如父女更無好感，與羨如為歡喜冤家。可是，相處日久，用恭漸漸發現羨如可愛之處，頓生好感及追求之意……不過，在羨如眼中，用恭吊兒郎當、對狗沒有愛心，甚至以為他嗜食狗肉而感到極度噁心，拒絕用恭追求……另一方面用恭表弟的天佑一直被長勝村惡霸周偉威 ( [[郭峰_(演員)|郭峰]]飾 ) 之女周嘉汶 ( [[唐詩詠|唐詩詠]]飾 ) 暗戀，但天佑覺得嘉汶乃時下貪玩少女，避之則吉。及後，天佑發覺嘉汶本質不差，鼓勵她改變自己，嘉汶因愛天佑而努力配合。嘉汶一直希望知道自己的生母是誰，但結果卻是峰迴路轉……嘉汶因事與偉威吵架要離家出走，連偉威送給她的保齡球也帶走，但原來保齡球內藏毒品，偉威遂千方百計誓要奪回毒品……毒販頭目顏大強 ( [[麥長青|麥長青]]飾 ) 借急病送院途中逃獄，其手下告知毒品在新界一所狗場內，大強誓要取回屬於自己的那袋毒品，率領手下鎖起狗場，將羨如、用恭、俊升、嘉汶、天佑、Bingo、阿黃困在狗場內，要脅偉威交出毒品，否則人狗也一併燒死……

==涉嫌抄襲事件==
劇集播出後，有報章發現指此劇的劇情涉嫌抄襲[[亞洲電視|亞洲電視]]於2006年底公佈的「[[亞洲電視節目巡禮2007|亞洲電視節目巡禮2007]]」的同名劇集《老友狗狗》，該劇主演演員為[[袁文傑|袁文傑]]和[[張文慈|張文慈]]，但該劇集並無開拍<ref>[http://hk.apple.nextmedia.com/template/apple/art_main.php?iss_id=20100114&sec_id=462&subsec_id=463&art_id=13623700 《老友狗狗》劇名橋段似十足　無綫竊取亞視2007爛尾劇，《蘋果日報》，2010年01月14日]</ref><ref>[http://www.hkatv.com/v3/special_events/06/atv2007/home01.html aTV 節目巡禮 2007] {{webarchive|url=https://web.archive.org/web/20100103111324/http://www.hkatv.com/v3/special_events/06/atv2007/home01.html |date=2010-01-03 }}，亞洲電視官方網站</ref>。

[[無綫電視|無綫電視]]其後召開記者會，否認有關指控，更聲言於2007年5月已有此劇初稿，于2008年8月開拍時當時劇名為《[[好狗出更|好狗出更]]》。亞視卻聲稱該同名劇集早於2006年9月已籌拍，並於同年底的節目巡禮對外公開，兩方各據一詞。
