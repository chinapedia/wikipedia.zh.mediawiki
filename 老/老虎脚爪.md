'''老虎脚爪'''（[[汉语拼音|拼音]]：'''lǎohǔ jiǎozhǎo'''；[[吴语拉丁化方案|吴语苏沪嘉小片发音]]：'''lauhu ciahtsau'''<ref>{{cite web | url=http://wu-chinese.com/minidict/| title=吴音小字典 | publisher=吴语协会| date= | accessdate=}}</ref>
）是一種起源于[[江苏|江苏]][[镇江|镇江]]<ref name="六角铮铮的镇江美食京江脐 ">{{Cite web |url = http://api.photochina.china.cn/frag/articleInfo_2986_1.html |title = “六角铮铮”的镇江美食京江脐 |date = 2014-12-11 |publisher = 中国网 |language = zh-cn |accessdate = 2015-02-04 |deadurl = yes |archiveurl = https://web.archive.org/web/20150205044546/http://api.photochina.china.cn/frag/articleInfo_2986_1.html |archivedate = 2015-02-05 }}</ref>，流行于[[江苏|江苏]]、[[上海|上海]]一带的传统[[小吃|小吃]]，江淮一带通常称'''金刚脐'''，江苏[[盐城|盐城]]又称之为'''金刚麒'''；[[镇江|镇江]]、[[扬州|扬州]]又称为'''京江脐'''，亦简称'''江脐子'''（[[江淮官话|江淮官话]][[扬州话|扬州话]]{{IPA|/kaŋ̩ t͡ɕʰi t͡sɛ/}}）或'''脐子'''<ref name="扬州方言词典">{{cite book|author1=黄继林|author2=李荣|title=扬州方言词典|url=http://books.google.com/books?id=pMBwAAAAIAAJ|year=1996|publisher=江蘇教育出版社|page=24}}</ref>，此处"江"读"刚"音；[[南京|南京]]称为'''金刚脐'''（[[江淮官话|江淮官话]][[南京话|南京话]]{{IPA|/t͡ɕin kɔo t͡sʰi/}}），但读法特殊，老派[[南京话|南京话]]中此处"刚"读"高"音<ref name="南京方言词典">{{cite book|author1=刘丹青|author2=李荣|title=南京方言词典|url=http://books.google.com/books?id=7MVwAAAAIAAJ|year=1995|publisher=江蘇敎育出版社|page=195}}</ref>。一说因外形类似泥塑金刚的肚脐，因而叫做金刚脐<ref name="六角铮铮的镇江美食京江脐 "/>。也有说得名于起源地镇江的别称“京江”<ref name="扬州江脐子">{{Cite web |url = http://news.takungpao.com/paper/q/2013/1208/2091610.html |title = 
扬州江脐子"  |date = 2013-12-08 |author = 徐永清|publisher = 大公网 |language = zh-cn |accessdate = 2015-02-04}}</ref> 。

因正面突起六个尖状物，颜色呈焦黄，由于是手工制作不用模具，六个角时常分得很开，形状类似老虎的爪子，传入[[苏州|苏州]]、[[上海|上海]]等地后，故有了老虎爪子之名。<ref name="laohujiaozhao">{{cite web| url=http://www.daynews.com.cn/lvyou/cbsz/231785.html| title=苏州的老虎脚爪| publisher=山西新闻网 旅游频道-吃遍神州| date=2007-06-15| accessdate=2011-10-03| deadurl=yes| archiveurl=https://web.archive.org/web/20111007211134/http://www.daynews.com.cn/lvyou/cbsz/231785.html| archivedate=2011-10-07}}</ref>

==制作方法==

老虎脚爪表面脆而不硬，稍显坚实是其特色，还有略一点点碱香味，用老面发酵的传统方法制作，据说可以帮助消化。以前[[胃炎|胃病]]患者胃痛起来，都要买老虎脚爪吃。<ref name="laohujiaozhao"/>老虎脚爪过去也曾是春、秋游的必备干粮。<ref>裴伟、《“老虎脚爪”京江齐》一文</ref>

在上海一带，因相比[[面饼|大饼]]，老虎脚爪体积小而不耐饥，一般不做早饭食用，早上店家也不售卖。往往在卖完大饼之后，为节约起见，店家要封[[煤|煤球]]炉，以节约煤球。为了利用大饼炉内还有的余火余温，店主就让师傅将做烧饼多余的实心面团摘成一小块一小块，先贴在炉膛里，再封炉口。因为余温有限，所以体积不能太大，一般只有拳头盖大小。一直烘到下午三四点钟，店家会利用余温，再烤上一炉香脆饼，<ref name="laohujiaozhao3">  {{cite web | url=http://news.eastday.com/eastday/shfashion/node248/node274/node5960/userobject1ai90123.html| title=城隍庙"失传小吃"引人狂!| publisher=[[东方网|东方网]]上海新闻| date= 2006-03-28| accessdate=}}</ref>或者重新开炉，生火放煤，为[[晚餐|晚茶]]做准备。经过长达五六小时的微火烘焙，使老虎脚爪变得外脆里香，另外它个头小，算是一种消闲食品，售价也低，深受市民的欢迎。<ref name="laohujiaozhao2"> {{cite web| url=http://www.jfdaily.com/a/734334.htm| title=上海基本绝迹的点心 老虎脚爪| publisher=[[解放日报|解放日报]]牛网-美食栏目| date=2009-12-10| accessdate=}}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

而在镇江一带，店家前一天下午先将加了[[油|油]]、[[盐|盐]]、[[碱|碱]]的面粉和好，让其发酵一夜。第二天白天再把面揉成馒头状的小团，用刀切成六角形，贴进炉壁烘烤约15分钟，即可出锅。刚出炉的京江脐棱角分明，呈立体雪花状，味道香脆松软。<ref name=" 运河畔老夫妻 ">{{Cite web |url = http://news.xdkb.net/society/2014-11/22/content_731878_all.htm |title =  运河畔老夫妻做了40年"京江脐" 引网友怀念"记忆中的味道"  |date = 2014-11-23 |author = 封疆江|coauthor= 林清智|publisher = 现代快报网 |language = zh-cn |accessdate = 2015-02-04}}</ref> 

==口味与吃法==

金刚脐有甜咸两种口味<ref name="扬州江脐子"/>。咸味的金刚脐味道更像实心大饼，盐城人喜欢把它撕开，放在[[牛肉|牛肉]]汤、[[羊肉|羊肉]]汤里泡着食用。亦可泡在[[豆浆|豆浆]]或[[粥|粥]]中食用<ref name="扬州江脐子"/>。甜味的金刚脐在出锅时外表会刷一层糖水。苏州人和上海人喜好甜食，味道甜但不重，属于甜烘饼。<ref name="laohujiaozhao3"/>扬州、镇江一带亦有用[[京果粉|京果粉]]泡甜金刚脐的吃法。
==淡出消费市场==

1970年代起，大饼摊的炉子逐步改成了更为现代化的电炉，原先利用煤球炉余热制作的老虎脚爪也逐渐淡出了食品消费市场，无论在盐城、苏州还是上海都难寻踪影。1990年代，上海的一些饮食公司曾希望恢复老虎脚爪的传统工艺，但电炉的火力永远也无法调节到以往煤球炉的微温状态，也不可能烘焙那么长时间，一般只会在一些传统食品展与节日庙会中象征性的出现。<ref name="laohujiaozhao2"/>

2006年，上海老字号[[王家沙点心店|王家沙点心店]]将老虎脚爪重新恢复制作并售卖。相对于传统工艺，王家沙的烹饪流程相对简化，将老虎脚爪放入明火炉烘烤约20分钟，一炉五十个，从和面到出炉约为40分钟。<ref>{{cite web | url=http://news.sohu.com/20060227/n242028525.shtml| title=老虎脚爪重装上阵 | publisher=解放网-新闻晨报 作者：徐运| date= 2006-02-27| accessdate=2011-10-03}}</ref>

==习俗==

镇江、扬州一带有[[清明|清明]]食用京江脐的习俗<ref name="镇江市志">{{cite book|title=镇江市志|url=http://books.google.com/books?id=cjeWSwAACAAJ|year=1993|publisher=上海社会科学院出版社}}</ref>，相传京江脐是为[[寒食|寒食]]不动烟火而制作的，而两地的寒食已和清明合为一日<ref name="扬州方言词典"/>。两地民间有谚语云“清明不吃脐子馓子，死后变成猫子狗子”与“清明不吃脐儿和馓子，死后阎王老爷要打板子”<ref>  {{cite web | url=http://www.yznews.com.cn/yzwb/html/2015-03/28/content_676547.htm| title=清明为何要吃京江脐子 | publisher=扬州晚报| date= 2015年3月28日| accessdate= 2016年3月12日}}</ref>。
