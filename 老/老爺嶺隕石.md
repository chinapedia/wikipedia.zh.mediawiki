'''老爺嶺隕石'''是在1947年墬落在[[蘇聯|蘇聯]]錫霍特阿蘭山脈（老爺嶺）的隕石，這批墬落隕石的總量是最近的歷史中最大的。

== 歷史 ==
[[File:Sikhote-Alin_stamp_1957.jpg|thumb]]

在1947年2月12日上午大約10:30之際，在蘇聯濱海的[[濱海邊疆區|濱海邊疆區]]錫霍特阿蘭山脈（老爺嶺）的目擊者，看見一個比[[太陽|太陽]]還要明亮的[[火流星|火球]]出現在北方，並以41度的角度下墬。明亮的閃光和震耳欲聾的聲音在距離墬落點300公里的[[盧切戈爾斯克|盧切戈爾斯克]](Luchegorsk)和東北方440公里的[[海參威|海參威]]都感受得到。煙塵的尾巴估計長達32公里，並在天空中維持了數小時。

當這個隕石以每秒14公里的速度進入大氣層時，它開始碎裂。大約在5.6公里的高度上，最大的隕石發生劇烈的爆炸，並且這些碎片都一起墬落下來。

在1957年11月20日<ref name=stamps>[http://www.pibburns.com/catastro/metstamp.htm Meteorite Stamps and Coins] Philip R. "Pib" Burns</ref>，[[蘇聯|蘇聯]]發行了一張郵票紀念老爺嶺流星雨的十周年，重現了俄羅斯藝術家梅德韋傑夫目擊流星雨墬落的版畫：她坐在窗口看見了這個流星的出現，並且立即將看見的景象做了素描<ref>[http://www.planetarium.montreal.qc.ca/Information/Expo_Meteorites/Vedettes/sikhotealin_a.html Sikhote Alin] {{webarchive|url=https://web.archive.org/web/20060521024636/http://www.planetarium.montreal.qc.ca/Information/Expo_Meteorites/Vedettes/sikhotealin_a.html |date=2006-05-21 }} - Planetarium de Montreal</ref>。

== 軌道 ==
因為這個[[隕石|隕石]]是在白天墬落的，因此有許多的目擊者觀測到。有些觀測資料獲得極高的評價，例如[[俄羅斯科學院|俄羅斯科學院]]的隕石委員會主席[[V._G._Fesenkov|V. G. Fesenkov]]的觀測報告，被用來推測它在遭遇到[[地球|地球]]之前的軌道。它的軌道是[[橢圓|橢圓]]形的，它距離[[太陽|太陽]]最遠的位置類似其他許多穿越過地球軌道的小行星，位在[[小行星|小行星帶]]。這種軌道的小行星可能是在小行星帶內的碰撞創造出來的。

== 尺寸 ==
老爺嶺烙下的質量很大，以整體的體積估計總質量大約在900吨以下。Krinov估計在大氣層中隕石損失的質量大約是70吨，而由Tsvetkov和其他人在最近完成的估計認為有100吨<ref>[http://meteoritemag.uark.edu/604.htm Meteorite Magazine] {{webarchive|url=https://web.archive.org/web/20100612144717/http://meteoritemag.uark.edu/604.htm |date=2010-06-12 }}，Arkansas Center for Space and Planetary Sciences, University of Arkansas</ref>。

== 散播場 ==
這批隕石的[[散布區|散播場]]大約覆蓋的面積大約是1.3平方公里的橢圓形，有些碎片造成了坑穴，最大的有6米深，直徑達到26米<ref>{{cite Earth Impact DB | name = Sikhote-Alin| accessdate = 2009-08-19}}</ref>；有些隕石的碎片也嵌入了周圍的樹木內。

==成分和分類 ==
老爺嶺的隕石在分類上屬於[[鐵隕石|鐵隕石]]，化學成分屬於IIAB群，和有著最粗糙的[[銳鈦礦|八面石]]構造。它的組成大約是93%[[鐵|鐵]]，5.9% [[鎳|鎳]]，0.42% [[鈷|鈷]]，0.46% [[磷|磷]]，0.28%[[硫|硫]]，和微跡的[[鍺|鍺]]和[[銥|銥]]。存在的礦物包括[[鎳紋石|鎳紋石]]、[[合紋石|合紋石]]、[[隕硫鐵|隕硫鐵]]、[[鉻鐵礦|鉻鐵礦]]、[[錐紋石|錐紋石(隕鐵)]]、和[[隕磷鐵鎳石|隕磷鐵鎳石]]。

== 標本 ==
老爺嶺隕石的標本基本上有兩種類型<ref>[https://web.archive.org/web/20010415042416/http://www.meteor.co.nz/feb96_1.html www.meteor.co.nz: ''Sikhote-Alin'']</ref>：
# ''單獨的''、''拇指印紋''或''氣印標本''：顯示熔融的外殼和大氣[[燒蝕|燒蝕]]的痕跡。
# ''霰彈片''或''碎片標本''：銳利的邊緣顯示是在暴力下碎裂，殘缺不全的金屬碎片。

第一類可能是在下降初期碎裂的主要物件，這些碎片的特點是在每個標本的表面都有[[氣印|氣印]]（類似拇指按壓的痕跡）。第二類的碎片，要不是在大氣層爆炸的過程中被撕裂開來，或就是撞擊在冰凍的地上所受到的影響。大部分都可能是在5.6公里的海拔高度上爆炸的結果。

一個巨大的標本在[[莫斯科|莫斯科]]展示，其他許多的標本被[[俄羅斯科學院|俄羅斯科學院]]所持有，還有更多的小標本各自以自己的方式進入收藏家的市場上。
