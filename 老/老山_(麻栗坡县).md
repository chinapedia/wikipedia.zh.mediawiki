'''老山'''是[[中越边界|中越边界]]上的一处有七个山峰的小型山脉。2013年的报道称，现分属中越两国。在以数字命名的七个山峰中，主峰4号高地是中越国界骑线点，中国方面拥有1号、2号、3号高地<ref name="a1">{{Cite web |url=http://news.ifeng.com/history/zhongguoxiandaishi/detail_2013_11/29/31662294_1.shtml|title=中越划界为何“法卡山已划给越南”流言四起？ |accessdate=2019-03-03 |author=胡贲 |date=2013-11-29 |work= |publisher=凤凰网，来源：南方周末（2009年12月3日）|language=Zh-hans }}</ref>，在行政上属[[云南省|云南省]][[麻栗坡县|麻栗坡县]]。

主峰——4号高地海拔是1422.2米，因此也称之为1422.2高地。由于越南的高程起算面与中国不同，越方称之为1509高地。是中越传统边界骑线点。[[1979年中越战争|1979年中越战争]]爆发后，越南占领老山、[[者阴山|者阴山]]地区。1984年4月28日，中方从越南占领下收复4号高地。[[两山战役|两山战役]]结束后，中方占据老山几乎全部山峰<ref name="a1"/>。

1999年，中越两国开始边界谈判，划分一千多公里的陆地边界。由于中国政府相关规定，未公开详细情况。故中国舆论曾认为，[[法卡山|法卡山]]、老山被归属于越南<ref>{{Cite web |url=http://www.sohu.com/a/274628433_477856|title=探秘丨中国为什么归还法卡山 |accessdate=2019-03-03 |author= |date=2018-11-11 |work= |publisher=搜狐网|language=Zh-hans }}</ref>。2013年的报道，中国方面早已废弃老山、者阴山的前出阵地。现由[[中国人民解放军|解放军]]边防部队驻守，未移交[[武警|武警]]边防部队。解放军进行边防巡逻，未设常备阵地<ref name="a1"/>。
