'''老城-莱赫尔'''（{{lang-de|'''Altstadt-Lehel'''}}）是[[德国|德国]][[巴伐利亚州|巴伐利亚州]]首府[[慕尼黑|慕尼黑]]的一个[[市辖区|市辖区]]（[[慕尼黑市辖区列表|第1区]]）。它由[[慕尼黑老城|老城]]和[[莱赫尔|莱赫尔]]两个{{link-de|市分区|Stadtteil|分区}}组成。

== 方位 ==
该市辖区包括历史悠久的老城以及与之在东北处毗邻的莱赫尔共两个{{link-de|市分区|Stadtteil}}。

老城的边界基本上是由{{link-de|老城环路|Altstadtring}}所构成。仅有的例外是在北部以画廊大街（{{lang|de|Galeriestraße}}）－{{link-de|布里安大街|Brienner Straße}}的走向分界，以及在东南部以穆勒大街（{{lang|de|Müllerstraße}}）和鲁姆福德大街（{{lang|de|Rumfordstraße}}）分界。

莱赫尔的边界范围是从东部的[[伊萨尔河|伊萨尔河]]至西部的国王大街（{{lang|de|Königinstraße}}），以及从北部的蒂沃利大街（{{lang|de|Tivolistraße}}）至南部的双桥大街（{{lang|de|Zweibrückenstraße}}）。它在西南部则是通过老城环路与老城分隔。此外，[[博物館島_(慕尼黑)|博物馆岛]]和[[普拉特岛|普拉特岛]]内的北部地带亦属莱赫尔。

与之相邻的市辖区分别为北部的[[施瓦宾-弗莱曼|施瓦宾-弗莱曼]]，东部伊萨尔河对岸的[[博根豪森_(市辖区)|博根豪森]]和[[坳-海德豪森|坳-海德豪森]]，南部和西南部的[[路德维希近郊-伊萨尔近郊|路德维希近郊-伊萨尔近郊]]以及西北部的[[马克斯近郊|马克斯近郊]]。

== 区划 ==
老城-莱赫尔区包含有慕尼黑的两个市分区：
*[[慕尼黑老城|老城]]，对应{{link-de|历史城市核心|Historischer Stadtkern|历史性的城市核心}}，这是自中世纪起便由{{link-de|慕尼黑城防设施|Münchner Stadtbefestigung}}所环绕；
*[[莱赫尔|莱赫尔]]，自中世纪起便被纳入{{link-de|慕尼黑堡领地|Münchner Burgfrieden}}，并因此常被称为慕尼黑“最古老的近郊”。

为了便于管理和统计目的，老城-莱赫尔市辖区被划分为6个[[近鄰社區|社区]]，它们都是基于历史上分区而来：

# [[格拉根瑙社区|格拉根瑙社区]]
# [[安格尔社区|安格尔社区]]
# [[哈肯社区|哈肯社区]]
# [[十字社区|十字社区]]
# [[莱赫尔|莱赫尔]]
# [[英国花园_(慕尼黑)|英国花园]]南

前四个社区是对应老城的历史分区。

== 政治 ==
老城-莱赫尔上一次的{{link-de|区议会 (德国)|Bezirksausschuss|区议会}}选举是在2014年3月16日。其议席分配如下：[[拜仁基督教社会联盟|基社盟]]5席，[[联盟90/绿党|绿党]]4席，[[德国社会民主党|社民党]]4席以及[[自由民主党_(德国)|自民党]]2席<ref name="Wahl">[http://www.wahlen-muenchen.de/ergebnisse/wahldesbezirksausschusses/endgueltig/internet/gesamt/uebersicht_listenwahl_stadtbezirk-1-altstadt-lehel_gesamt.html ''Wahl des Bezirksausschusses – Stadtbezirk 1 – Altstadt-Lehel'']. Landeshauptstadt München. Abgerufen am 16. Juni 2014.</ref>。在老城-莱赫尔共16109名符合投票资格的居民中，有7035人行使了自己的表决权，选民投票率为43.7%。
