'''老城集市广场'''（'''''Rynek Starego Miasta'''''）是[[波兰|波兰]]首都[[华沙|华沙]][[华沙老城|老城]]的核心。直到18世纪末，此处也是整个华沙的核心。老城集市广场拥有独特的波兰传统餐厅、咖啡馆和商店。这也是华沙最古老的部分，和该市最重要的旅游名胜之一。[[华沙起义|华沙起义]]后，它曾被德国军队有系统地摧毁<ref>{{en icon}} {{cite web  |author = |url = http://www.worldheritagesite.org/sites/warsaw.html |title = Warsaw |work = www.worldheritagesite.org |publisher =  |pages =  |page =  |date = |accessdate = 2008-07-08}}</ref>，二战结束后，又被小心翼翼地恢复为原来的模样。

== 历史 ==
老城集市广场可以追溯到华沙初创的时期，13世纪末<ref name="perelki">{{en icon}} {{cite web  |author = |url = http://www.um.warszawa.pl/v_syrenka/perelki/index_en.php?mi_id=48&dz_id=2 |title = The Old Town Market Square |work = eGuide / Treasures of Warsaw on-line |publisher =  |pages =  |page =  |date = |accessdate = 2008-07-08}}</ref>。当时，行会代表和商人们聚集在市政厅，有时也执行死刑。1607年大火以前，周边是哥特式建筑；此后改建为文艺复兴式<ref name="staremiasto">{{pl icon}} {{cite web  |author = Marek Lewandowski 
|url = http://www.stare-miasto.com/ulice/rynek.html |title = Rynek Starego Miasta |work = www.stare-miasto.com |publisher =  |pages =  |page =  |date = |accessdate = 2008-07-08}}</ref>。19世纪，华沙迅速发展，老城失去商业和行政中心的地位。

在1939年[[波兰战役|波兰战役]]期间，该区域大部分被所摧毁，在可怕的空袭中，德国空军有目的地瞄准了该市的住宅区和历史地标<ref name="pbase">{{en icon}} {{cite web  |author = Tomasz Dziubinski |url = http://www.pbase.com/smok53/warsaw_oldtown_world_heritage_unesco_photography |title = Warsaw's Old Town Quarter |work = www.pbase.com |publisher =  |pages =  |page =  |date = |accessdate = 2008-07-08}}</ref>。

1944年[[华沙起义|华沙起义]]被镇压后，老城集市广场被德国军队有系统地摧毁<ref name="pbase" /><ref>{{en icon}} {{cite web  |author = |url = http://whc.unesco.org/en/list/30 |title = Historic Centre of Warsaw |work = whc.unesco.org |publisher =  |pages =  |page =  |date = |accessdate = 2008-07-08}}</ref>。今天的这些建筑重建于1948年到1953年<ref name="perelki" />，当时尽可能地使用了原来的砖块，建在原来的位置上，保持了17世纪富裕商人们最初兴建时的模样。
