}}'''老七佳部落'''，簡稱'''老七佳'''，是一個位於屏東縣[[春日鄉|春日鄉]]北側[[排灣族|排灣族]]聖山石可見山西側山腳的山中谷地排灣族聚落，當地標高約海拔570公尺。目前，該聚落內保存有近50幢完整的排灣族傳統北部式石板家屋。當地居民大多已於1960年代遷移至新聚落，但仍會於耕作時返回居住。<ref>{{Cite web|url=http://ecotour.footprint.tw/01_travel_detail.php?p=22&page=0|title=老七佳 石板屋部落 :: 東方足跡生態導覽解說團隊|accessdate=2016-12-21|work=ecotour.footprint.tw}}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref>{{Cite web|url=http://輝哥的天空.tw/mountain/nopaia/pingtung010.htm|title=老七佳部落|accessdate=2017-01-11|author=|date=|publisher=}}</ref><ref>{{Cite web|url=http://blog.xuite.net/yan9436/blog/128878564-%E4%BA%BA%E9%96%93%E7%A7%98%E5%A2%83%E7%9A%84%E8%80%81%E4%B8%83%E4%BD%B3%E7%9F%B3%E6%9D%BF%E5%B1%8B%E9%83%A8%E8%90%BD|title=人間秘境的老七佳石板屋部落|accessdate=2017-02-06|last=(yan9436)|first=阿城|work=隨意窩 Xuite日誌}}</ref>

== 歷史 ==
七佳村可分為老七佳、舊七佳及七佳。

數百年前，排灣族開始於老七佳定居生活；其居屋特色是以黑色石板作主要建材，根據統計，全村最多曾有約75幢[[石板屋|石板屋]]。

1958年，當局強制將當地居民遷移至力里溪畔的舊七佳；1972年，[[颱風麗塔_(1972年)|莉泰颱風]]破壞舊七佳，居民再度遷移至位在力里國小旁的七佳村。<ref>{{Cite news|url=http://gugugu4411.pixnet.net/blog/post/96027994-%E5%A4%B1%E8%90%BD%E7%9A%84%E5%A4%A9%E7%A9%BA%E4%B9%8B%E5%9F%8E--%E5%B1%8F%E6%9D%B1%E6%98%A5%E6%97%A5%E9%84%89%E8%80%81%E4%B8%83%E4%BD%B3%E9%83%A8%E8%90%BD|title=失落的天空之城--屏東春日鄉老七佳部落 @ 阿春仔in台南-台南旅遊攻略、景點、民宿、美食、水果 :: 痞客邦 PIXNET ::|last=(gugugu4411)|first=黃大黃=阿春爸|newspaper=阿春仔in台南-台南旅遊攻略、景點、民宿、美食、水果|accessdate=2017-02-06|language=zh-TW}}</ref>

當地居民遷村後，該部落處荒廢狀態達40餘年。<ref>{{Cite web|url=https://www.flyingv.cc/projects/8371|title=消失中的部落寶藏，重建老七佳石板屋 {{!}} flyingV|accessdate=2017-01-11|work=flyingV|language=zh-TW}}</ref><ref>{{Cite news|url=http://koeiman.pixnet.net/blog/post/42142802-%E6%98%A5%E6%97%A5%E9%84%89~%E8%80%81%E4%B8%83%E4%BD%B3%E4%B9%8B%E9%81%BA%E4%B8%96%E7%9F%B3%E6%9D%BF%E9%83%A8%E8%90%BD|title=春日鄉~老七佳之遺世石板部落 @ 屏東客阿凡端的旅行秘笈 :: 痞客邦 PIXNET ::|last=(koeiman)|first=koeiman|newspaper=屏東客阿凡端的旅行秘笈|accessdate=2017-02-06|language=zh-TW}}</ref>

2001年，在族人陸續返回整修後，老七佳逐步重現往日面貌。現在，部落內保存有約40幢石板屋。<ref>{{Cite news|url=http://www.appledaily.com.tw/appledaily/article/supplement/20140328/35729247/|title=屏東 老七佳 探遺世石板部落|last=|first=|date=|work=|newspaper=蘋果日報|accessdate=2016-12-21|language=zh-TW}}</ref>

== 文化 ==
雖然當地居民早已遷至新聚落，但其傳統文化仍有維持運作，如歲時祭儀masaljut（收穫祭、[[豐年祭|豐年祭]]）、[[五年祭|maljeveq]]（祖靈祭、五年祭）、pusaut（送靈祭），以及生命祭儀papusepi（祈福祭）、seman caucau（命名祭）、semanpulju（除喪祭）等；如此祭儀主要仍由[[女巫|女巫]]主持，並因此成為排灣族文化傳承的重要地區。<ref>{{Cite web|url=http://163.29.7.35/webno/cultureassets/Community/info_upt.aspx?p0=9899|title=文化部文化資產局-國家文化資產資料庫系統|accessdate=2016-12-21|date=2016-01-01|last=文化部文化資產局|work=文化資產局|deadurl=yes|archiveurl=https://web.archive.org/web/20161221135618/http://163.29.7.35/webno/cultureassets/Community/info_upt.aspx?p0=9899|archivedate=2016-12-21}}</ref>
