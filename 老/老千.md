'''老千'''在[[粵語|粵語]]中是騙子的意思，有組織的[[集團|集團]]則稱為'''千門'''。意思是以千變萬化的手段及千方百計的方式，去針對受騙人的[[心理|心理]]和[[信念|信念]]而施行騙術<ref>[http://mykampung.sinchew.com.my/node/187766?tid=53 老千橫行 《星洲日報》 2012年4月14日]</ref>。

== 偷訛拐騙 ==
* 偷：[[偷盜|偷盜]]、[[偷竊|偷竊]]
* 訛：虛假陳述
* 拐：拐帶，[[販賣人口|販賣人口]]
* 騙：利用受害人的[[迷信|迷信]]、[[信仰|信仰]]、好色及[[貪心|貪念]]等

== 招謠撞騙 ==
騙子尋找對象的原始方法。招謠是自我宣傳而引人注意，撞騙即無固定方式和地點。

== 賭博業 ==
精於賭博，主要以千術在賭博中賺取利益。主要代表人物：[[马洪刚|马洪刚]]、[[老鬼头|老鬼头]]、鬼手阿三、[[郭安迪|郭安迪]]、[[郑泰顺|郑泰顺]]、尧建云等。

=== 天仙局 ===
即天衣無縫的詐騙佈置，最完美的是被騙者於受害後也不知被佈局欺騙。例如：
# 民國初年的廣東、而直至1960年代至1970年代香港大[[酒樓|酒樓]]晚上[[宴席|宴席]]之前多設有[[雀局|雀局]]及[[牌局|牌局]]，參與者多被蒙騙也不自知。
# 先物色對象，再設賭局行使騙術。現在多以訂購貨物為名，相約於境外見面，稱時間尚早，說有[[二世祖|二世祖]]敗家好賭，但賭術不精，最好給他教訓。

== 老千集團 ==
有些老千集團以非賭博手法棍騙受害人。
=== 千門八將 ===
千門八將，「上八將」為「正」、「提」、「反」、「脫」、「風」、「火」、「除」、「謠」，而「下八將」乃「姦」、「淫」、「邪」、「盜」、「偷」、「呃」、「拐」及「騙」。分別在於上八將多是集團組織，運用智慧機會，順應被騙者的喜好，志在騙走錢財而不會傷害他人身體。

=== 千門上八將 ===
多為[[三合會|三合會]]成員，各司其職，是謂：「正、提、反、脫、風、火、除、謠」

# 正：主角，出面利用技術或智慧行騙。（口才及[[心理學|心理學]]等）
# 提：佈置及提議騙局的[[軍師|軍師]]。（精心策劃，有應對方案，器材安排，即佈下俗稱「天仙局」的老千）
# 反：（正寫玩，粵音反）陪伴玩樂的[[傍友|傍友]]，利用對手的愛好引其上當。提供女伴或賭伴（投其所好），亦作反[[間諜|間諜]]、專門留意一哥周圍人物。
# 脫：安排相關人等逃跑、脫身。（成功後的消失，失敗後的逃逸）
# 風：派專人望風，又或俗稱「天文台長」。等同源出香港[[三角碼頭|三角碼頭]]工人的[[術語|術語]]「[[睇水|睇水]]」，後演變成非法賭檔和鴉片煙檔的[[隱語|隱語]]。
##看風：讓人盯著你及附近環境，一發現不對勁，即通知相關人等
##收風：收取情報 
##放風：發放（虛假）消息 
# 火：武力。（在棍騙失敗，而又不能談判解決，則以武力手段平息事件，又或在棍騙過程中需要用武力時也會由這人負責）
# 除：談判。（在棍騙失敗後，以談判手法解決事件）
# 謠（招謠）：謠言或虛假宣傳。（造謠以混淆視聽，例如：聽說副食品要漲價了...結果使某人買回了滿屋子的油鹽醬醋）

=== 媒的演變 ===
由個人的招搖撞騙方式開始繁衍成有組織的集團形式。凡千不離媒，媒是引介及宣傳，提反謠也是媒。經過媒的引介及虛假失實的宣傳，從而令人相信正將的權威和專業。

# 明清之前：行走江湖之[[相術|相士]]、[[算命|算命]]和[[風水|風水]]先生、雲遊[[道人|道人]][[方術|術士]]、遊歷[[和尚|和尚]]及賣藝者等等。
# 民國初年：娛樂場所、[[茶館|茶館]]、[[酒樓|酒樓]]及[[報章|報章]][[宣傳|宣傳]]等等。
# 近代：[[電子媒體|電子媒體]]及[[電腦網路|電腦網路]]等等。

=== 特別行規 ===
受害人若發現被騙而又得悉該名老千身份，可透過江湖人士與該老千交涉，在受騙的指定日子內取回部份被騙的款項，形式頗為接近法院的上訴索償機制。

# 被騙七日內：可取回七成
# 被騙第八日至一個月內：可取回五成
# 被騙一個月後至三個月內：可取回三成
# 被騙三個月後：不能再追討
