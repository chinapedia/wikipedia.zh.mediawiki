'''貓娘姊妹'''（{{lang-ja|ねこシス}}）是日本輕小說作家[[伏見司|伏見司]]写的[[輕小說|輕小說]]，由[[神崎廣|神崎廣]]担当[[插畫|插畫]]。于2009年1月号至2009年7月号的《[[電擊文庫MAGAZINE|電擊文庫MAGAZINE]]》連載，[[電擊文庫|電擊文庫]]于2009年10月发行[[单行本|单行本]]。

== 登場人物 ==
* '''{{lang|ja|東雲 美緒}}　（{{lang|ja|しののめ みお}}）'''
:14歲,貓又一族,東雲家的三女(貓的形態是虎斑貓)。她對於過著人類生活的姊妹們一直懷抱著憧憬,在笫五次挑戰「人化之術」時才成功,是個天生吊車尾。

* '''{{lang|ja|東雲 千夜子}}　（{{lang|ja|しののめ ちやこ}}）'''
:15歲,東雲一家的次女(貓的形態是黑貓)。冷靜理性的姊姊,經常表現出冷淡的態度,言行舉止也常帶刺。常常捉弄妹妹們,藉以發洩心中積壓的鬱悶。

* '''{{lang|ja|東雲 鈴}}　（{{lang|ja|しののめ りん}}）'''
:13歲,東雲一家的四女(貓的形態是三毛貓)。雖是老么,卻很可靠,全家的家事由她一手包辦。比姊姊先學會人化之術,將美緒當成「妹妹」般溺愛她。

* '''{{lang|ja|東雲 かぐら}}　（{{lang|ja|しののめ かぐら}}）'''
