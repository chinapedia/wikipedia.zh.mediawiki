{{Vgname|貓咪收集|ja=ねこあつめ|en=Neko Atsume: Kitty Collector|aka=收集貓咪、貓咪後院}}，是[[日本|日本]]遊戲公司「Hit-Point」開發的一個[[iOS|iOS]]及[[Android|Android]]智能手機遊戲，於2014年10月推出，至2015年7月下載量突破550萬<ref name="apple">{{cite web|title=「貓咪收集」可愛遊戲App　愛貓人為之瘋狂|url=http://www.appledaily.com.tw/realtimenews/article/new/20150708/644015/|publisher=[[蘋果日報_(臺灣)|蘋果日報]]|accessdate=2015-07-09|date=2015-07-06}}</ref>，成為該公司最受歡迎的遊戲<ref name="cnn" />。

《貓咪收集》在推出初期關注度有限，直至2015年1月開始才獲得了一些[[網誌|網誌]]介紹，翌月[[南韓|南韓]]的下載量就出現了顯著上升，其後[[中國|中國]]和[[美國|美國]]的下載量也隨之上升。2015年6月開始，[[Twitter|Twitter]]上出現了為數甚多加上[[主題標籤|主題標籤]]「#nekoatsume」的推文，海外玩家不停在網上分享收集到的貓兒和遊戲攻略<ref name=a2/>，而美國[[CNN|CNN]]電視台更在2015年7月3日一篇專題報導中介紹了遊戲<ref name=cnn>{{cite web|title='Neko Atsume' is the addicting new app where you feed stray cats|url=http://edition.cnn.com/2015/07/03/asia/neko-atsume-japan-cat-game/|publisher=[[CNN|CNN]]|accessdate=2015-07-09|date=2015-07-08}}</ref>。截至2015年7月，遊戲有達四成的下載量均來自海外<ref name="itmedia150617">{{cite web|script-title=ja:「ねこあつめ」海外で人気　日本語版しかないのに3～4割が海外から　「日本ローカルネタ多くて恐縮」と開発会社|language=ja|url=http://www.itmedia.co.jp/news/articles/1506/17/news127.html|publisher=ITmedia|accessdate=2015-07-09|date=2015-06-17}}</ref><ref name=ttv/>。

== 遊戲內容 ==
[[File:nekoatsume_niwasaki.jpg|350px]]的貓兒。]]
《貓咪收集》以家中的後院為遊戲背景。遊戲介面相當簡單，玩家只需將[[貓糧|貓糧]]及各種道具放在庭院不同位置，下次再啟動遊戲的時候就會有[[流浪貓|流浪貓]]出現，然後幫貓兒拍照存進相冊，但玩家並不能操控流浪貓。貓兒離開後也會留下灰色魚乾和金色魚乾，供玩家當作代幣用來購買貓糧和各種道具，甚至擴張庭院和改變庭園風格。<ref>{{cite web|title=日本貓奴瘋玩超萌養成！《收集貓咪》家裡後院都是貓|url=http://game.ettoday.net/article/475789.htm|publisher=東森遊戲雲|accessdate=2015-07-09|date=2015-03-11}}</ref><ref>{{cite web|script-title=ja:《ねこあつめ》滿足貓別跑！超療癒的貓咪收集遊戲！|url=http://www.app01.com.tw/paper/2fa49755-c7bb-47d8-a340-48d11e242a65|publisher=app01|accessdate=2015-07-09|date=2015-05-06}}{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

玩家的目標主要是要收集全部59隻貓兒<ref name=cnn/>，每一隻的性格和外貌均有異，如有穿著[[棒球|棒球]]制服的貓、[[穿靴子的貓|穿靴子的貓]]、披著[[波斯地毯|波斯地毯]]的[[波斯貓|波斯貓]]、還有[[砂鍋|砂鍋]]貓、站長貓等等<ref>{{cite web|script-title=ja:超療癒貓咪收集(ねこあつめ)小遊戲！喵喵你有事嗎？|url=http://www.misskiri.com/2015/03/nekoatume.html|publisher=綺麗小姐|accessdate=2015-07-09|date=2015-03-10}}</ref>。遊戲中一些貓兒的外貌取自現實人物，如一隻戴太陽眼鏡的貓兒就與美國版《[[時尚_(雜誌)|時尚]]》雜誌主編[[安娜·溫特|安娜·溫特]]相似<ref name=cnn/>。

因操作介面簡單，初學者也能初步理解如何操作遊戲<ref name=ttv>{{cite web|title=療癒系手機遊戲 "貓咪收集"瘋迷全球　萌!用手機養喵星人 玩耍模樣撫慰人心　養虛擬貓 "貓咪收集"5百萬次點擊下載|url=http://www.ttv.com.tw/104/07/1040708/10407080018200I.htm?from=579|publisher=台視新聞|accessdate=2015-07-09|date=2015-07-08}}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

== 評價 ==
澳洲電子媒體《Techly》評論稱《貓咪收集》與[[他媽哥池|他媽哥池]]和《Hatchi》等遊戲形式相似，但「玩起來較沒有壓力和回報高」<ref>{{cite web|author1=Rachel Oakley|title=NEKO ATSUME: A NEW JAPANESE GAME WHERE YOU COLLECT… CATS|url=http://www.techly.com.au/2015/07/12/neko-atsume-new-japanese-game-collect-cats/|publisher=Techly|accessdate=2015-07-13|date=2015-07-12}}</ref>。[[大久保佳代子|大久保佳代子]]在[[日本電視台|日本電視台]]節目《{{link-ja|跳舞吧！秋刀魚御殿！！|踊る!さんま御殿!!}}》上介紹了遊戲，並表示《貓咪收集》是她「現在最熱愛的遊戲」<ref name="oricon"/>。《[[日本時報|日本時報]]》的遊戲介紹則指「《貓咪收集》的玩法實在太簡單，你不會感覺到語言造成的障礙」<ref>{{cite web|author1=Shusuke Murai|title=Japanese cat-collecting game is apps’ meow|url=http://www.japantimes.co.jp/news/2015/06/29/business/tech/japanese-cat-collecting-game-is-apps-meow/#.VaPzEmCEayk|accessdate=2015-07-13|date=2015-06-29}}</ref>。《[[國際財經時報|國際財經時報]]》的報導更稱「《貓咪收集》正威脅著[[小玉_(貓)|小玉]]和[[Maru_(貓)|Maru]]的公眾關注度」，並認為這種貓的遊戲「只會來自日本」，因為日本在不同層面都跟貓有著千絲萬縷的關係，如在虛擬世界中就有[[Hello_Kitty|Hello Kitty]]和[[多啦A夢|多啦A夢]]、現實世界中也有被稱為「貓島」的[[田代島|田代島]]和{{link-ja|青島 (愛媛縣)|青島 (愛媛県)|青島}}、[[化貓|化貓]]傳說和[[招財貓|招財貓]]等貓類的存在<ref>{{cite web|author1=Jayalakshmi K|title=Japanese online cat game Neko Atsume sees million downloads|url=http://www.ibtimes.co.uk/japanese-online-cat-game-neko-atsume-sees-million-downloads-1509484|publisher=[[國際財經時報|IBTimes]]|accessdate=2015-07-13|date=2015-07-06}}</ref>。

開發者高崎豐表示，「Hit-Point」本身主要開發[[電子角色扮演遊戲|RPG]]類型遊戲，而《貓咪收集》只是在「開發其他RPG大型遊戲時，用多出的時間製作的」、「本來只是預期能讓玩家高興兩、三個星期的程度」<ref name="oricon">{{Cite web|date=2015年3月6日|url=http://www.oricon.co.jp/special/47733/|script-title=ja:Twitterで人気急伸『ねこあつめ』とは？|language=ja|publisher=[[Oricon|Oricon]]|accessdate=2015-07-09}}</ref>。他對遊戲變得如此受歡迎表示驚訝：「其實我不太明白為甚麼遊戲在海外也那麼受歡迎，但我覺得可能是因為貓在全世界都一樣被愛著吧。」高崎也表示，遊戲中有不少元素是在日本國外是難以明白的，如身穿[[棒球|棒球]]服的貓其實穿著的是[[日本棒球|日本職業棒球]]球隊[[阪神虎|阪神虎]]的制服，「海外的玩家來說可能會不知道這些故事，對此我感到抱歉。」<ref name="itmedia150617"/>

== 衍生文化 ==
