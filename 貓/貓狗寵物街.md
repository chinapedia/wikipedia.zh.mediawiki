湯瑪與朋友們（''{{lang|ja|3丁目のタマ うちのタマ知りませんか?}}''）是[[日本|日本]]一部以[[寵物|寵物]]為題材的電視[[日本動畫|動畫]]。

==動畫及劇場版==
[[Sony_Creative_Products|Sony Creative Products]]的人氣[[文具|文具]]品牌「Tama & Friends」的衍生作品。故事圍繞著一個住在三丁目的小男孩岡本武與他飼養的一隻花貓斑仔所發生的生活小趣事，以及斑仔與牠的朋友麼麼、波子等寵物每天的歷險故事。

在1983年，[[日本|日本]]的廣大傳媒以一幅尋貓啟事來宣傳[[OVA|OVA]]。[[OVA|OVA]]版本只繪畫了動物的模樣，人物的樣子則不顯示出來。隨後為了慶祝OVA版十周年的播放紀念，劇場版在1993年8月14日於日本上映，其標題為《{{lang|ja|3丁目のタマ おねがい!モモちゃんを捜して!!}}》，中文標題則為《之小麼麼失蹤了》，其人物畫風有所改變，並增加了麼麼、愛美等人物。在1994年至1995年期間亦先後推出了[[RARE|RARE]]版和電視動畫版。

第1輯在1993年7月3日至8月28日期間於日本[[每日放送|每日放送]]和[[東京放送|東京放送]]播出，共9集。第1輯的第1集至第6集輯錄了的[[OVA|OVA]]版本，第7集至第9集則輯錄了[[RARE|RARE]]版本。而第2輯則在1994年4月3日至9月25日期間播出，共26集。此後故事設定和人物造型均沿用了[[RARE|RARE]]版本的設定。隨後[[香港|香港]][[無綫電視翡翠台|無綫電視翡翠台]]亦於1997年至2002年期間播放了的電視動畫版、OVA版本和劇場版。

此外，[[Animax|Animax]]在2006年推出了的衍生作品《{{lang|ja|タマ&フレンズ 探せ!魔法のプニプニストーン}}》（[[貓狗寵物街_(第三輯)|貓狗寵物街第三輯]]），角色沿用了斑仔、波子和小虎等動物，但造形和故事背景則有所變更，與原作沒有任何共通點。

===動物===
; {{nihongo|-{zh-hans:圆圆;zh-hk:斑仔;zh-tw:圓圓;}-|タマ|配音：笠原弘子；台灣：[[謝佼娟|謝佼娟]]}}
: 黃色耳朵，白色的貓，臉上有一塊黑色的圓圈，尾巴也是黑色的。富有正義感。與兄弟4隻一起在鄉下的奶奶家生的。生日是向日葵花開時的6月28日。
; {{nihongo|-{zh-hans:妙妙;zh-hk:麼麼;zh-tw:妙妙;}-|モモ|配音：{{lang|ja|[[椎名碧流|椎名碧流]]}}}}
: 亦稱作小桃，粉紅色耳朵的貓。家裡是咖啡廳。喜歡加了牛奶的紅茶和草莓蛋糕。
; {{nihongo|-{zh-hans:百吉;zh-hk:波子;zh-tw:波奇;}-|ポチ|配音：渡邊真砂子；台灣：[[陶敏嫻|陶敏嫻]]}}
: 一隻全白的狗，有著紅色項圈。在太熱的天氣太陽底下，就會變得很奇怪。'常常與貓咪們在一起。
; {{nihongo|小虎|トラ|配音：[[坂本千夏|坂本千夏]]；台灣：[[朱憶華|朱憶華]]}}
: 黃色耳朵，臉上有虎纹。個性有點衝，脾氣拗，常常會和ノラ吵架。喜歡吃鰹魚飯和螃蟹罐頭。
; {{nihongo|貝貝|ベー|配音：雨蘭咲木子<small>（第1輯）</small>、{{lang|ja|まるたまり}}<small>（第2輯）</small>}}
: 黑色的貓，嘴巴周圍、腳掌和尾巴末端是白色的。被新聞屋的河原先生所飼養。亦稱作阿貝。喜歡坐在腳踏車後面。
; {{nihongo|點點|コマ|配音：[[荒木香惠|荒木香惠]]<small>（第2輯）</small>}}
: 粉紅色耳朵，臉上有點點。住在湯屋，桶谷小姐所飼養。
; {{nihongo|小黑|クロ|配音：{{lang|ja|コヒエミオコ}}<small>（第1輯）</small>、岩永哲哉<small>（第2輯）</small>；台灣：[[朱憶華|朱憶華]]}}
: 黑色的狗，嘴巴周圍、腳掌和尾巴末端是白色的，有著灰褐色的項圈。住在三田先生家。
; {{nihongo|小黃|ゴン|配音：北島淳司<small>（第2輯）</small>；台灣：[[謝佼娟|謝佼娟]]}}
: 褐色的狗，耳朵及尾巴是黑色的，左眼也有黑色的圈圈。野田先生所飼養，家裡是家具店。曾住過大阪，講關西腔。
; {{nihongo|野貓|ノラ|配音：勝田治美<small>（第1輯）</small>、津賀有子<small>（第2輯）</small>；台灣：[[陳進益|陳進益]]、[[謝佼娟|謝佼娟]]}}
: 住在公園旁邊的空地，黑白相間的貓。善良，時而冷漠，偶爾感覺孤單。亦稱作阿浪。與小虎一樣喜歡吃螃蟹罐頭。
; {{nihongo|布魯|ブル|配音：荒川太郎；台灣：[[陳幼文|陳幼文]]}}
: 家裡很富有的狗。喜歡妙妙，把圓圓給視為對手。
