'''殷景仁'''（{{bd|390年||441年|1月7日|catIdx=Y}}）<ref name="卒年">《宋書·文帝紀》：「十一月癸丑，尚書僕射、揚州刺史殷景仁卒。」據[http://sinocal.sinica.edu.tw/ 兩千年中西曆轉換]轉。</ref>，一名'''鐵'''<ref>《史通·卷二十八》：「凡列姓名，罕见其字。苟前后互舉，則觀者自知。如《左傳》上言羋斟，則下曰叔牂，前稱子產，則次見國僑，其類是也。至裴子野《宋略》亦然。何者？上書桓玄，則下云敬道；後叙殷鐵，則先著景仁。以此而擬《左氏》，又所谓貌異而心同也。」另按《宋書·劉湛傳》：湛与殷景仁素款。及俱被时遇，猜隙漸生。湛黨劉敬文父成未悟其機，詣景仁求郡。敬文遽往謝湛曰："老父悖耄，遂就殷鐵干祿。"又《南史·范泰传》：泰卒，議赠开府。殷景仁曰：泰素望未重，不可。王弘撫棺哭曰："君生平重殷鐵，今以此為報。"
」</ref>，陳郡長平人。[[中国|中国]][[南朝宋|南朝宋]]政治人物，受[[宋文帝|宋文帝]]重用，官至揚州刺史、尚書僕射。晉太常[[殷融|殷融]]曾孫。

== 生平 ==
殷景仁年輕時即得[[司徒|司徒]][[王謐|王謐]]認為他有大成之量，將女兒嫁給他。初為[[劉毅_(東晉)|劉毅]]後軍參軍，後轉太尉行參軍, ，後屢次轉職，至太子中庶子。[[宋少帝|宋少帝]]即位後召其為[[侍中|侍中]]，但景仁多番辭讓，遂改黃門侍郎，不久轉任左衞將軍。元嘉元年（424年）宋文帝即位後對景仁加以重用，讓其以侍中兼左衞將軍，與[[謝弘微|謝弘微]]、[[王華_(劉宋)|王華]]、[[王曇首|王曇首]]及[[劉湛_(劉宋)|劉湛]]並稱五臣<ref>《宋書·謝弘微傳》</ref>，並與當時同為侍中的王華、王曇首及劉湛皆以才幹及器度，同升門下而名揚當時，無人可比。元嘉三年（426年），文帝出討荊州刺史[[謝晦|謝晦]]，景仁更與司徒[[王弘_(劉宋)|王弘]]共掌留事，事後替代[[到彥之|到彥之]]改兼中領軍。

後殷景仁喪母，母親下葬後即獲任命為領軍將軍。景仁以丁憂辭讓，文帝遂命[[綱紀|綱紀]]前去代景仁拜官，更讓中書舍人[[周赳|周赳]]開車接載。元嘉八年（431年），因王華及王曇首已死，景仁以人才凋零為由勸文帝召時任撫軍長史、南蠻校尉的劉湛回朝。元嘉九年（432年），守喪期完結，景仁遷護軍將軍、[[尚書僕射|尚書僕射]]。原本劉湛因景仁而得還朝內並受重用，且與其素有交情，二人一開始關係甚好。然而因景仁升任尚書僕射，專掌要職，劉湛就甚為妒忌，反認為是要排擠他，故聯結其時主政的司徒、揚州刺史、彭城王[[劉義康|劉義康]]，想籍義康的力量除去景仁，得享專任<ref>《宋書·劉湛傳》：「湛與景仁素欵，又以其建議徵之，甚相感說。及俱被時遇，猜隙漸生，以景仁專管內任，以為閒己。時彭城王義康專秉朝權，而湛昔為上佐，遂以舊情委心自結，欲因宰相之力以回主心，傾黜景仁，獨當時務。」</ref>。但文帝卻極度信任景仁，於元嘉十二年（435年）讓景仁兼本職進任[[中書令|中書令]]，不久更讓他以尚書僕射領吏部，權力更大。劉湛更加憤恨，義康亦在劉湛影響下屢次向文帝中傷景仁，但文帝對景仁的信任不減反加。景仁見劉湛如此對他，曾對親近說：「引他進來，便要咬人。」於是主動稱病解職。文帝屢次拒絕，反讓他留家養病，派黃門侍郎前往侍疾。劉湛見此更欲派人殺害景仁，文帝知道後特別改西掖門外的晉鄱陽公主府為護軍將軍府，因其接近禁宮以保護景仁。

景仁雖然養病在家，但仍與文帝有緊密聯繫，密通書信每日達十多次，文帝亦常派[[庾炳之|庾炳之]]往返宮中及護軍府<ref>《宋書·庾炳之傳》：「景仁稱疾不朝見者歷年，太祖常令炳之銜命去來，湛不疑也。」</ref>，故朝政大事景仁都知曉，景仁鑑於當時劉義康與劉湛的集團權傾朝野，密對文帝表示這不利於社稷，文帝亦同意。元嘉十七年（440年），文帝決意翦除以劉湛為首的劉義康黨眾勢力，景仁在行事當天就整飾好衣冠，因為平日與文帝聯絡並不為人所知，又臥病多年，故左右侍從都不明白景仁用意。當晚文帝就召景仁入宮，由他決定劉湛等一眾人的處分。最終劉湛、劉斌、孔胤秀、劉敬文等人皆被收捕誅殺，劉義康亦外調江州。

義康外調後，文帝以江夏王[[劉義恭|劉義恭]]為司徒，任命景仁為揚州刺史，仍兼尚書僕射領吏部。可是景仁不久便病重，文帝於是召他回尚書，同年十一月二十九日（441年1月7日）<ref name="卒年"/>去世，享年五十一歲，朝廷追贈侍中、[[司空|司空]]，諡'''文成'''。

== 性格特徵 ==
景仁不寫文章，但才思敏捷，口不談文義而深明道理，國典朝儀、舊章記注都曾撰錄，有識之士都看出他有當政的志向。

== 逸事 ==
*據說景仁接任揚州刺史後就性情大變，平時寛厚的他突然變得苛刻暴戾。他又顯得精神愰惚，曾問左右那年男婚女嫁的數目；又曾在冬天大雪天乘輿出門觀望，忽惊嚇道：「正門外怎麼有棵大樹？」接著又說：「我弄錯了嗎？」因為他上任揚州刺史後個多月就死了，有人就傳言他是被刘湛的鬼魂索命。
*景仁出任揚州刺史後患病，文帝曾下令不許揚州路上有車聲騷擾景仁休息<ref>[[裴子野|裴子野]]《[[宋略|宋略]]》：「殷景仁入居西州，疾篤，上為之累息，敕西州道上不得有車聲。」</ref>。
