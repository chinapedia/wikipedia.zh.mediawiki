'''自贡市盐业历史博物馆'''，位于[[四川省|四川省]][[自贡市|自贡市]][[自流井区|自流井区]]解放路107号[[西秦会馆|西秦会馆]]，是中国盐业史专业博物馆。

==简介==
1959年，自贡市盐业历史博物馆由[[邓小平|邓小平]]倡议创办，是中国建立较早的收藏、研究、陈列中国盐业历史文物的专业博物馆。[[郭沫若|郭沫若]]题写馆名<ref name=weiyi>{{Cite web|url = http://sc.china.com.cn/2015/lvyou_jingqu_0209/113118.html|title = 中国唯一的盐业史博物馆——盐业历史博物馆|author = |date = 2015-02-09|publisher = 中国网|language = |accessdate = |quote = |archiveurl = https://web.archive.org/web/20150628162706/http://sc.china.com.cn/2015/lvyou_jingqu_0209/113118.html|archivedate = 2015-06-28|deadurl = yes}}</ref>。馆址设在[[西秦会馆|西秦会馆]]<ref name=zgw>{{Cite web |url =http://www.china.com.cn/zhuanti2005/txt/2006-02/17/content_6123752.htm  |title =自贡市盐业历史博物馆 |author =  |date =2006-02-17  |publisher =  中国网|language =  |accessdate =  |quote =  }}</ref>。2017年，自贡市盐业历史博物馆被[[中国博物馆协会|中国博物馆协会]]评为第三批国家一级博物馆<ref>{{Cite web |url = http://www.chinamuseum.org.cn/a/xiehuigonggao/20170120/9351.html|title = 关于发布第三批国家一级博物馆名单的通知 |author =  |date =2017-01-20  |publisher = 中国博物馆协会 |language =  |accessdate =  |quote =  }}</ref>。

自贡市盐业历史博物馆拥有两个[[全国重点文物保护单位|全国重点文物保护单位]]（[[西秦会馆|西秦会馆]]、[[吉成井盐作坊遗址|吉成井盐作坊遗址]]），1个[[四川省文物保护单位|四川省文物保护单位]]（[[王爷庙_(自贡)|王爷庙]]），1个近现代工业遗址（大安盐厂）<ref name=guan/>，1个分馆（狮市镇盐运历史文化陈列馆）<ref name=guan/><ref>{{Cite web |url = http://scnews.newssc.org/system/20160825/000701752.htm |title = 自贡盐业历史博物馆将在富顺县狮市镇建分馆 |author =  |date = 2016-08-25 |publisher = 四川新闻网 |language =  |accessdate =  |quote =  }}</ref>。自贡市盐业历史博物馆主办历史学类期刊《盐业史研究》（属于[[CSSCI|CSSCI]]扩展版来源期刊）。自贡市盐业历史博物馆是[[自贡世界地质公园|自贡世界地质公园]]核心区、国家AAA级旅游景区、[[中国侨联|中国侨联]]爱国主义教育基地、全国科普教育基地、四川省哲学社会科学普及基地、中国自然科学博物馆协会专业科技博物馆委员会理事单位<ref name=guan/>。

自贡市盐业历史博物馆馆藏珍贵文物595件<ref name=guan>{{Cite web |url =http://www.zgshm.cn/content.jsp?id=4028813e5581453001559bae025b07f5  |title =自贡市盐业历史博物馆  |author =  |date = 2016-06-09 |publisher = 自贡市盐业历史博物馆 |language =  |accessdate =  |quote =  }}</ref>。藏有世界上唯一一套完整的古代顿钻工具<ref name=weiyi/>。
