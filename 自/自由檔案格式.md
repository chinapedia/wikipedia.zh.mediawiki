'''自由檔案格式'''是指沒有受到任何[[專利權|專利權]]或[[著作權|著作權]]保護的[[檔案格式|檔案格式]]。自由檔案格式的相容性通常更大，並且可以防止[[單一廠商束縛|單一廠商束縛]]（vendor lock-in），也是[[開放系統|開放系統]]的組成基礎。

== 檔案歸檔與壓縮 ==
* [[7z|7z]]，歸檔與壓縮（[[LZMA|LZMA]]）格式
* [[bzip2|bzip2]], [[gzip|gzip]], [[xz|xz]] ([[LZMA|LZMA]]), [[lzo|lzo]]，壓縮算法与相应的文件格式
* [[cpio|cpio]]，归档格式
* [[tar_(计算机科学)|Tar]]，歸檔格式

== 多媒體 ==
* [[PNG|PNG]]，[[非破壞性資料壓縮|非破壞性資料壓縮]]格式
* [[APNG|APNG]]，建基於PNG的動畫格式，另有[[MNG|MNG]]
* [[SVG|SVG]]，基于[[XML|XML]]的[[矢量繪圖|矢量繪圖]]格式
* [[Ogg|Ogg]]（音頻與影片格式）
** [[Vorbis|Vorbis]]，音頻編碼
** [[Theora|Theora]]，影片編碼
** [[Tarkin|Tarkin]]，影片編碼
** [[FLAC|FLAC]]，無損音頻編碼
** [[Speex|Speex]]，即時語音編碼
Ogg檔案格式與Vorbis編碼是為了與廣泛使用但卻受到[[軟體專利權|軟體專利權]]保護的[[MP3|MP3]][[音頻壓縮|音頻壓縮]]檔案格式競爭而設計的。
* [[Opus_codec|Opus codec]]

== 文字 ==
* [[HTML|HTML]]/[[XHTML|XHTML]]，[[網頁|網頁]]所用的[[標記語言|標記語言]]
* [[OpenDocument|OpenDocument]]，開放文檔格式
* [[RTF|RTF]]，格式化文字檔案格式
* [[TXT|TXT]]，原始文字檔案

== 页面描述/标记 ==
* [[PDF|PDF]]，可攜式文件格式
* [[PostScript|PostScript]]，[[頁面描述語言|頁面描述語言]]與[[程式設計語言|程式設計語言]]
* [[DjVu|DjVu]]，[[计算机|计算机]][[文件格式|文件格式]]

== 其他 ==
* [[NZB|NZB]]，[[Usenet|Usenet]]上的多部件[[二進位|二進位]]檔
* [[SQL|SQL]]，[[資料庫|資料庫]]
* [[XML|XML]]，多用途的[[標記語言|標記語言]]
* [[Descript.ion|Descript.ion]]，是一種為檔案提供注釋的文字檔案
