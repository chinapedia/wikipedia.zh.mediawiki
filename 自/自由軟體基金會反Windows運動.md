'''自由軟體基金會反Windows運動'''是針對[[Microsoft_Windows|Microsoft Windows]]作業系統反對的運動。其與針對[[数字版权管理|数字版权管理]]的[[設計不良|設計不良]]運動類似，但此運動的目標是針對微軟的作業系統而非DRM。

==Windows 7宗罪==

2009年時，自由軟體基金會發起了針對[[Windows_7|Windows 7]]的活動，其名稱為「Windows 7宗罪」{{lang|en|"Windows 7 Sins"}}<ref>{{cite web|author1=Eric Lai|title=Free software group attacks Windows 7 'sins'|url=http://www.computerworld.com/article/2527513/microsoft-windows/free-software-group-attacks-windows-7--sins-.html|website=COMPUTERWORLD|accessdate=2016-07-04|date=2009-08-26}}</ref>。該活動的網頁使用了從[[开源游戏|开源游戏]]{{link-en|XBill|XBill}}。該活動被指責使用了[[FUD|FUD]]策略。<ref name="techrep">{{cite news|url=http://www.techrepublic.com/blog/linux-and-open-source/free-software-foundation-cites-windows-7-sins/ |last=Wallen |first=Jack |title=Free Software Foundation cites Windows 7 'sins' |date=2009-08-31 |publisher=[[CBS互動|CBS互動]] |newspaper=TechRepublic |deadurl=no |archivedate=2014-01-07 |accessdate=2014-01-07 |archiveurl=https://web.archive.org/web/20140107231850/http://www.techrepublic.com/blog/linux-and-open-source/free-software-foundation-cites-windows-7-sins/ }}</ref><ref name="zdnet">{{cite news|url=http://www.zdnet.com/blog/hardware/dissecting-the-fsfs-windows-7-sins/5281 |title=Dissecting the FSF's "Windows 7 Sins" |last=Kingsley-Hughes |first=Adrian |date=2009-08-31 |newspaper=ZDNet |publisher=CBS Interactive |deadurl=no |accessdate=2014-01-07 |archivedate=2014-01-07 |archiveurl=https://web.archive.org/web/20140107232104/http://www.zdnet.com/blog/hardware/dissecting-the-fsfs-windows-7-sins/5281 }}</ref><ref name="linuxfmt">{{cite journal|url=http://www.linuxformat.com/content/win-7-sins-ftl |title=Win 7 Sins FTL |date=2009-09-21 |journal=Linux Format |last=Saunders |first=Michael |archiveurl=https://web.archive.org/web/20090927050330/http://www.linuxformat.com/content/win-7-sins-ftl |deadurl=no |archivedate=2009-09-27 |accessdate=2014-01-07 }}</ref>

==從Windows 8升級==

在2012年10月，自由軟體基金會開始了另一個稱為「從Windows 8升級」的活動，這次的目標是針對[[Windows_8|Windows 8]]。

==Windows 10==

在[[Windows_10|Windows 10]]釋出後，因其專有軟體的本質，自由軟體基金會發表聲明來勸使用者不要使用Windows 10。基金會也引用來其他與此相關的來源，像是強制付較少款項的客戶比付了較多款項的客戶先測試較不安全的更新、微軟在2013年的全球監控醜聞中扮演的角色，以及在Windows 10開始的新隱私權政策<ref name="win10">{{Cite web|url=http://www.fsf.org/news/the-fsfs-statement-on-windows-10|title=The FSF's statement on Windows 10|publisher=Free Software Foundation|date=2015-07-29|accessdate=2015-08-17}}</ref>。
