'''“自由魁北克万岁！”'''（{{lang-fr|'''"Vive le Québec libre!"'''}}{{IPA-fr|vivᵊ ləkebɛk ˈlibʁᵊ|pron}}）是[[法国总统|法国总统]][[夏尔·戴高乐|夏尔·戴高乐]]在1967年7月24日演讲中备受争议的一句话，当时他正以参加{{Link-en|蒙特利尔世博会|Expo 67}}为由正式访问[[加拿大|加拿大]]。当他在[[蒙特利尔市政厅|蒙特利尔市政厅]]的[[阳台|阳台]]上发表即兴演讲时，他高呼：“[[蒙特利尔|蒙特利尔]]万岁！[[魁北克|魁北克]]万岁！”（{{lang-fr|"Vive Montréal; Vive le Québec !"}}）在一片欢呼聲中，他又接着喊出了：“自由魁北克万岁！”（{{lang-fr|"Vive le Québec libre!"}}）这一[[魁北克独立运动|魁北克独立运动]]使用的口号。戴高乐的演讲被视为是对魁北克独立运动的支持，并引发了法加两国的外交危机。[[加拿大总理|加拿大总理]][[莱斯特·皮尔逊|莱斯特·皮尔逊]]当即谴责这一言论，并宣称：“加拿大人不需要被解放。”（{{lang-en|"Canadians do not need to be liberated."}}）在法国，尽管许多人同情魁北克民族主义的產生，但戴高乐的演讲仍然因违反外交礼节而遭到许多媒体的批评。四十多年后，它仍然被视为加拿大英语区和法语区之间的关系和政治的一个轉捩點，而法国和加拿大的外交关系一直紧张直到2008年时任法国总统[[尼古拉·萨科齐|尼古拉·萨科齐]]宣布不再支持加拿大[[魁北克省|魁北克省]]独立为止<ref>{{cite news

== 背景 ==
[[File:Charles_de_Gaulle-1963.jpg|thumb]]]]

在戴高乐到达之前，加拿大政府就为他的來訪感到担忧。1967年3月，法国政府并没有派出高规格代表团参加[[加拿大总督|总督]]{{link-en|乔治·瓦尼埃|Georges Vanier}}的葬礼<ref name="Thomson 1988pp188-9">Thomson (1988), pp. 188-189.{{en}}</ref>。这引起了加拿大方面的注意，因为瓦尼埃和他的妻子[[波林|波林]]自1940年起就與当时在[[英国|英国]][[伦敦|伦敦]]流亡的戴高乐私交甚篤<ref name="Thomson 1988pp188-9"/>。4月，戴高乐亦没有参加於加拿大举办的[[维米岭战役|维米岭战役]]胜利50周年纪念仪式<ref name="Vimy Ridge 1967">{{cite news
 | last      = Harvey

== 演讲 ==
[[File:Montreal-Canada5884.JPG|thumb]]的阳台，戴高乐就是在这里发表的演讲。]]

7月15日，在科尔贝尔级到达前，戴高乐对{{link-fr|沙维尔·丹尼歐|Xavier Deniau}}说：“当他们在那里听到我的话的时候，会激起波澜吧！”<ref name="Thomson 1988" />他同样自信地对他的女婿{{Link-fr|阿兰·德·波瓦西厄|Alain de Boissieu}}将军说：“我要打出致命一击，事情会变得世人瞩目，但这是必须的。这是挽救懦弱的法国的最后机会。”<ref name="Thomson 1988" />此言被推測是在暗示1763年法国在[[七年战争|七年战争]][[法国－印第安人战争|北美战场]]上失败后，將七万法国殖民者留给英国的棄民政策。

7月24日，戴高乐到达蒙特利尔，他沿着[[国王大道|国王大道]]到达[[蒙特利尔市政厅|蒙特利尔市政厅]]，在那里市长{{Link-en|讓·德拉波|Jean Drapeau}}和省长约翰逊正在等他。當時戴高乐並沒有在当晚就進行演講的计划，但是人群高呼他的名字；于是他对德拉波说：“我要对这些呼唤我的人们说点话。”在訪問大量法国高级官员，以及发现了一些文件以後，学者{{Link-en|戴尔·凯恩斯·汤姆森|Dale C. Thomson}}写道，戴高乐的演讲稿其實是预先准备好的，只待时机成熟他便會將它發表出來<ref name = "Thomson 1988">Thomson (1988), p. 199.{{en}}</ref>。

戴高乐步上阳台，面对庞大的人群以及广播转播发表简短的演讲。演讲中他提到他沿着[[圣劳伦斯河|圣劳伦斯河]]到达这里，一路上欢呼的人群让他想起[[巴黎|巴黎]]从[[纳粹德国|纳粹德国]]手中[[解放巴黎|解放]]時，[[凯旋门|凯旋门]]前的[[香榭丽舍大道|香榭丽舍大道]]。在演讲的最后，他总结道：“蒙特利尔万岁！魁北克万岁！”（{{lang-fr|"Vive Montréal ! Vive le Québec !"}}）接着，他高呼：“自由魁北克万岁！法國人的加拿大万岁！万岁！万岁！法国万岁！”<ref name=cbc>{{cite news
 |last        = Depoe
 |first       = Norman
 |title       = 'Vive le Québec libre!'
 |work        = Newsmagazine/On This Day
 |publisher   = CBC news
 |date        = 1967-07-24
 |url         = http://www.cbc.ca/archives/categories/politics/language-culture/language-culture-general/vive-le-quebec-libre.html
 |accessdate  = 2012-07-24
 |archiveurl  = https://www.webcitation.org/69Oj6PyT2?url=http://www.cbc.ca/archives/categories/politics/language-culture/language-culture-general/vive-le-quebec-libre.html
 |archivedate = 2012-07-24
 |deadurl     = yes
}}
</ref>（{{lang-fr|"Vive le Québec... libre ! Vive, vive, vive le Canada français ! Et vive la France !"}}）人群大声歡呼，尤其在听到“自由魁北克万岁！”之后。戴高乐特别强调“自由”（libre）一词，他凑近麦克风并且比其他单词更慢更响亮地说出了这个词。

1969年，戴高乐访问[[布列塔尼半岛|布列塔尼]]时，他在[[坎佩尔|坎佩尔]]慷慨激昂地朗诵了一首他叔叔（名字同样是夏尔·戴高乐）以[[布列塔尼语|布列塔尼语]]写的诗，以表达他对布列塔尼文化的热爱。在对{{Link-en|布列塔尼民族主义|Breton nationalism}}的镇压后，这次演讲受到了民众的好评。之前他被指责为[[双重标准|双重标准]]，一方面因为语言不同要求从加拿大英语区中获得“自由”，另一方面镇压布列塔尼的独立运动<ref name = "Ellis 1993">Ellis (1993), p. 62.</ref>。在坎佩尔演讲中，他回应这种批评说布列塔尼是自由的，而且是被布列塔尼人和其他法国军队解放的。他还说二战中布列塔尼的抵抗是法国中最为激烈的，许多布列塔尼人加入了[[自由法国|自由法国军]]<ref>[http://www.ina.fr/politique/presidents-de-la-republique/video/CAF88035484/allocution-du-general-de-gaulle-a-quimper.fr.html French National Audiovisual Institute (INA), De Gaulle's speech in Quimper]</ref>。
