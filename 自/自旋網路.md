[[量子力學|量子力學]]中，'''自旋網路'''是一種圖表，用以表示[[粒子|粒子]]與[[量子場論|量子場]]之間的的交互作用與狀態。以[[數學|數學]]的出發點來看，這些圖案是一種簡明方法，可代表[[多線性函數|多線性函數]]以及[[矩陣群|矩陣群]]眾多[[表示論|表示]]之間的關聯函數。此圖案記號往往能簡化計算，以其能代表複雜的[[函數|函數]]。自旋網路的發明一般是歸因於[[羅傑·潘洛斯|羅傑·潘洛斯]]於1971年的貢獻，<ref>[[羅傑·潘洛斯|羅傑·潘洛斯]]—[http://math.ucr.edu/home/baez/penrose/Penrose-AngularMomentum.pdf ANGULAR MOMENTUM: AN APPROACH TO COMBINATORIAL SPACE-TIME]</ref>然而在此之前已有類似的圖樣方法。

透過[[卡洛·羅威利|卡洛·羅威利]], {{link-en|李·斯莫林|Lee Smolin}}、{{link-en|霍爾黑·普林|Jorge Pullin}}, {{link-en|羅多佛·甘比尼|Rodolfo Gambini}}等多位研究者的努力，自旋網路被用於[[量子重力|量子重力]]理論。自旋網路亦可被用在[[數學|數學]]中局域[[規範轉換|規範轉換]]不變性的[[連通|連通]]空間，用以建構特定的[[泛函|泛函]]。

== 潘洛斯原始定義 ==
1971年，[[羅傑·潘洛斯|羅傑·潘洛斯]]提出一種圖形表示法，其中每個[[線段|線段]]代表一個「單元」（[[基本粒子|基本粒子]]或粒子的複合系統）之[[世界線|世界線]]。三條線段匯聚在一個[[頂點|頂點]]。頂點可以詮釋為一個[[事件|事件]]；在此事件中，一個單元分裂成兩個單元，或兩個單元[[碰撞|碰撞]]合而為一。當一圖表中所有的線段都在頂點會合，則此圖為「封閉自旋網路」。[[時間|時間]]以單一方向行進，比如從圖的底部走到圖的頂部。然而在封閉自旋網路的例子，時間行進的方向對於計算不構成影響。

每一線段標上一個稱作[[自旋量子數|自旋量子數]]的[[整數|整數]]。帶有自旋數''n''的一個單元稱作''n''-單元，其[[角動量|角動量]]為''nħ''，''ħ''是約化[[普朗克常數|普朗克常數]]。[[光子|光子]]、[[膠子|膠子]]等[[玻色子|玻色子]]，其''n''為偶數；[[電子|電子]]、[[夸克|夸克]]等[[費米子|費米子]]，其''n''為奇數。

給定一封閉自旋網路，則可計算出一個相應的非負整數的範數（norm）。範數可用來計算不同自旋值的[[機率|機率]]。當一個自旋網路的範數是零，則其發生機率為零。當範數不為零時，在頂點處則有一些約束條件如下：

若有三個單元會合在一頂點，這三單元分別帶有自旋量子數''a''、''b''、''c''，則必須滿足
* [[三角不等式|三角不等式]]：''a''必須小於或等於''b'' + ''c''，''b''必須小於或等於''a'' + ''c''，以及''c''必須小於或等於''a'' + ''b''。
* 費米子守恆（Fermion conservation）：''a'' + ''b'' + ''c''必須是偶數。

舉例來說，''a'' = 3, ''b'' = 4, ''c'' = 6的例子是不可能，因為3 + 4 + 6 = 13是奇數。''a'' = 3, ''b'' = 4, ''c'' = 9也不可能，因為3 + 4 < 9。而''a'' = 3, ''b'' = 4, ''c'' = 5則可行，因為3 + 4 + 5 = 12是偶數且滿足三角不等式。

一些標記習慣會將整數標為半整數，約束條件則變成''a'' + ''b'' + ''c''的和要是整數。
