'''自由及开放源代码软件'''（{{lang|en|'''Free and open source software'''}}，縮寫為{{lang|en|FOSS}}，或{{lang|en|'''Free/Libre and open source software'''}}，縮寫為{{lang|en|[[自由软件|F]]/[[Gratis_versus_Libre|L]]/[[开放源代码软件|OSS]]}}，{{lang|en|FLOSS}}）是一种可以归类为既是自由软件又是开源软件的计算机软件{{efn|在英語語境中，新的使用者可能會混淆“自由”這一術語<ref>在英語中，free一詞同時可表示“自由”與“免費”兩個意思。</ref>。在自由及开放源代码软件中，“自由”是指能自由複製和重新使用該軟件，而不是指軟件的價格免費。[[自由軟件基金會|自由軟件基金會]]，一個主張[[自由軟件|自由軟件]]的組織，建議要理解這個概念，“将其作为言论自由而不是免费啤酒来理解”（think of free as in free speech, not as in free beer）<ref>{{cite web 
|url = http://www.gnu.org/philosophy/free-sw.html 
|publisher = GNU.org
|title = The Free Software Definition 
|accessdate = 2010-02-04}}
</ref>。}}。也就是，任何人被授权可以自由的使用，复制，研究和以任何方式来改动软件，并且其源代码是开放和共享的，因此人们被鼓励志愿的改善软件的设计。<ref name=":1">{{Cite web|url=https://www.gnu.org/philosophy/free-sw.html|title=What is free software?  The Free Software Definition|last=|first=|date=2018-06-12|website=The GNU Project -- GNU.org|access-date=2018-09-15}}</ref> 这种软件是相对于[[专有软件|专有软件]]（{{lang-en|proprietary software}}）是在版权的严格限制之下，并且通常其源代码对于用户是不开放的。

使用FOSS的好处包括降低软件成本，提高安全性和稳定性（特别是在恶意软件方面），保护隐私，教育，并让用户更好地控制自己的硬件。免费的开源操作系统，如[[Linux|Linux]]和BSD的后代如今被广泛使用，为数百万台服务器，台式机，智能手机（例如Android）和其他设备提供支持。{{sfn|Hatlestad|2005}}{{sfn|Claburn|2007}}许多[[开放源代码软件列表|軟件包]]使用了[[自由軟件許可證|自由軟件許可證]]和[[開放源碼許可證|開放源碼許可證]]。
