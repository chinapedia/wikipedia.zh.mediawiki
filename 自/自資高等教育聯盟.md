'''自資高等教育聯盟'''，英語：'''Federation for Self-financing Tertiary Education（FSTE）'''，是一個於1994年在[[香港|香港]]成立，由香港多所大專院校組成的教育聯盟，聯盟的目標是提升本地自資專上教育的專業質素和水平。截至2016年，自資高等教育聯盟共有14所成員院校<ref>[http://www.fste.edu.hk/tc/about-us_zh/ 自資高等教育聯盟 - 關於我們]，自資高等教育聯盟，2016</ref>，包括[[大學教育資助委員會|大學教育資助委員會]]資助的八間大學、一所政府成立的自資大學、政府成立及資助的[[職業訓練局|職業訓練局]]，以及四間根據香港法律320章註冊的自資專上學院或根據香港法律279章註冊的私立院校。

== 簡介 ==
自資高等教育聯盟的成員院校提供多元化的全日制及兼讀制課程，包括副學位、學位及[[研究院|研究院]]的資歷層級，範疇涵蓋[[學術|學術]]、[[職業|職業]]及專業培訓等。自資高等教育聯盟成立之初，主要作為院校間就持續教育課程進行合作，香港於2000年開展[[毅進計劃|毅進計劃]]及發展[[副學士|副學士]]課程，其後更推出多個自資[[學士|學士]]及更高層級的學位課程，自資高等教育聯盟不但成為各成員院校間就自資課程進行合作的平台，同時也為政府的教育政策提供建議，並受政府委託為高中至專上程度的課程提供指標。自資高等教育聯盟除了是成員院校間的合作平台，同時也為成員院校制定規範<ref>[http://paper.wenweipo.com/2013/04/27/HK1304270041.htm 專上超收逾3% 或被逐出聯盟]，文匯報，2013-04-27</ref>，監察收生程序及確保課程的質素<ref>[http://www.the-sun.on.cc/cnt/news/20130416/00407_057.html 研規範銜接副學位門檻防超收生]，東方日報，2013-04-16</ref>。

== 聯盟目標 ==
以下為自資高等教育聯盟的工作目標<ref>[http://www.fste.edu.hk/tc/mission-statement_zh/ 自資高等教育聯盟 - 使命]，自資高等教育聯盟，2016</ref>：
*推動香港終身學習發展，包括自資學士課程、自資副學位課程、持續教育及非本地教育範疇。
*提升香港自資學術及職業教育課程的質素和水平。
*加強與政府、企業、專業團體及其他持分者就自資課程及持續教育的溝通。
*為自資專上教育提供交流及分享教育資訊和經驗的平台。
*鼓勵、支持及參與就學術與職業教育相關範疇的研究。
*與政府合作，推動香港發展成為亞洲以致全球的教育樞紐。

== 歷史及發展 ==
1994年「香港高等院校持續教育聯盟」成立，初期主要著重於成員院校間在持續教育的合作。

2000年，香港政府提出開辦[[毅進計劃|毅進計劃]]及發展[[副學士|副學士]]課程，教育聯盟除了向香港政府提供建議，並受政府委託參與制定副學位課程通用指標。

2009年，香港政府為配合香港教育制度推行三三四的新學制，教育聯盟受[[香港教育局|香港教育局]]委託為將設立的[[毅進文憑|毅進文憑]]進行研究及撰寫課程<ref>[http://hk.apple.nextmedia.com/news/art/20160909/19765325 毅進普通話教材用簡體]，蘋果日報，2016-09-09</ref>。

2011年，教育聯盟受政府委託參與修訂三三四新學制下[[副學士|副學士]]及[[高級文憑|高級文憑]]課程的通用指標。

2012年，由於教育聯盟已由初期集中持續教育的合作，擴展至[[毅進計劃|毅進計劃]]、[[副學士|副學士]]、[[高級文憑|高級文憑]]、自資[[學士|學士]]等範疇，因此「香港高等院校持續教育聯盟」更名為「自資高等教育聯盟」。同年，教育聯盟獲香港政府委託進行有關資歷名銜計劃及資歷[[學分|學分]]的先導計劃。

2013年，聯盟為[[通識|通識]]課程出版「通式教育指南」。同年，聯盟向成員院校發布專上課程招生及取錄程序指引，加強對收生取錄程序的監察，聯盟又於同年取得質素提升支援計劃的撥款，開展有關[[教師|教師]]專業能力理念架構及其相關專業發展課程的研究計劃<ref>[http://www.fste.edu.hk/tc/qess_zh-2/ 自資高等教育聯盟 - 質素提升支援計劃]，自資高等教育聯盟，2016</ref>。

2014年，教育聯盟獲質素提升支援計劃撥款，延續質素提升津貼計劃下有關副學位通式教育架構的研究，政府另委託教育聯盟開展關於學分累積和轉移制度的先導計劃。

== 組織架構 ==
自資高等教育聯盟下設五個委員會<ref>[http://www.fste.edu.hk/tc/chairman_zh/ 自資高等教育聯盟 - 主席]，自資高等教育聯盟，2016</ref>，包括執行委員會、會員事務及發展委員會、研究與發展委員會、招生與學生事務委員會、毅進文憑課程管理委員會，另會就特定的項目成立工作小組。
