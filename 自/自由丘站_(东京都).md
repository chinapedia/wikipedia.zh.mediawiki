'''自由丘站'''<ref>{{Cite web|url=http://www.tokyu.co.jp/global/chinese_sc/railway/ticket/pdf/rosen.pdf|title=東急線·港未來線 路線圖|publisher=東京急行電鐵}}</ref>（{{jpn|j=自由が丘駅|hg=じゆうがおかえき|rm=Jiyūgaoka eki|f=y}}）是位於[[日本|日本]][[東京都|東京都]][[目黑區|目黑區]][[自由之丘|自由之丘]]一丁目，屬於[[東京急行電鐵|東京急行電鐵]]的[[鐵路車站|鐵路車站]]<ref name="eki">{{Cite web |url=http://www.tokyu.co.jp/railway/station/info/?id=7 |title=自由が丘駅｜各駅情報 |publisher=東京急行電鉄 |accessdate=2015-01-11}}</ref>。此站是目黑區最南的車站。

== 概要 ==
此站有[[東橫線|東橫線]]與[[大井町線|大井町線]]停靠，是兩條路線的轉乘站<ref name="ensen">{{Cite web |url=http://www.tokyu.co.jp/ekitown/info/?id=7 |title=自由が丘駅｜沿線・駅周辺ガイド |publisher=東京急行電鉄 |accessdate=2015-01-11}}</ref>。東橫線的車站編號是'''TY07'''、大井町線是'''OM10'''。

此站是[[站長|站長]]所在站，管理「自由丘管內」此站與東橫線[[都立大學站|都立大學站]]、大井町線[[綠丘站_(東京都)|綠丘站]]、[[九品佛站|九品佛站]]、[[尾山台站|尾山台站]]、[[等等力站|等等力站]]、[[上野毛站|上野毛站]]6個車站。

== 歷史 ==
*1927年（[[昭和|昭和]]2年）8月28日，以東京橫濱電鐵東橫線'''九品佛前站'''（{{lang|ja|九品仏前駅}}）的站名啟用<ref name="jtb48">[[#jtb|東急の駅]]、pp.48-49。</ref>。當時的月台配置是島式1面2線<ref name="jtb48"/>。
*1929年
** 10月22日，改名為'''自由丘'''（{{lang|ja|自由ヶ丘駅}}）<ref name="jtb48"/>。
** 11月1日，大井町線開通<ref name="jtb48"/>。
* 1959年（昭和34年），東橫線月台改為島式2面4線<ref name="jtb48"/>。
* 1961年（昭和36年），站前圓環設置女神像<ref name="jtb48"/>。
* 1966年（昭和41年）1月20日，日文拼字改為[[平假名|平假名]]而成為「{{lang|ja|自由が丘駅}}」<ref name="jtb48"/>。
* 2005年度開始耐震補強與對[[無障礙環境|無障礙環境]]的改良工程<ref name="ensen"/>。
* 2005年（[[平成|平成]]17年）10月31日，北剪票口改建完成<ref>{{Cite journal|title=東横線・大井町線 自由が丘駅の正面口の改札が変わります|journal=HOT ほっと TOKYU|date=2005-10-20|issue=303|url=http://hot.tokyu.co.jp/railway/hot/0511/densya01.html#1|publisher=東京急行電鉄|accessdate=2015-01-11|deadurl=yes|archiveurl=https://web.archive.org/web/20160305101006/http://hot.tokyu.co.jp/railway/hot/0511/densya01.html#1|archivedate=2016-03-05}}</ref>。
* 2006年
** 2月7日，大井町線1號線月台，東橫線3、4與5、6號線的[[扶手電梯|扶手電梯]]竣工。
** 3月31日 - 連結東橫線月台與大井町線月台的電梯啟用<ref>{{Cite journal|title=東横線・大井町線     自由が丘駅をバリアフリー化します|journal=HOT ほっと TOKYU|date=2006-03-20|issue=308|url=http://hot.tokyu.co.jp/railway/hot/0604/densya01.html#1|publisher=東京急行電鉄|accessdate=2015-01-12}}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。
** 12月18日 - 正面口新設的售票處完工。
** 12月21日  - 正面口的資訊處與定期券售票處啟用。
* 2007年（平成19年）3月下旬 - 正面口前與東橫線月台之間的電扶梯竣工。開通新付費區。
* 2008年（平成20年）3月 - 同月28日起大井町線開始運行急行，本站完成6輛編組月台延長工程。
* 2009年（平成21年）3月 - 為了因應與[[東京地下鐵|東京地下鐵]][[東京地下鐵副都心線|副都心線]][[直通運轉|相互直通運轉]]，本站進行月台延長工程以停靠十輛編組的特急、通勤特急、急行<ref>{{Cite press release |url=http://www.tokyu.co.jp/file/111110-1.pdf |title=特定都市鉄道整備事業実施状況 |publisher=東京急行電鉄 |date=2011-11-10 |accessdate=2015-01-12 |format=pdf}}</ref>。
* 2012年（平成24年）3月30日 - 「〜智慧模型自由丘站明亮計畫〜」：東急線首座車站採取全LED照明<ref name="LED">[http://panasonic.co.jp/corp/news/official.data/data.dir/jn120330-2/jn120330-2.html 東京急行電鉄 「自由が丘駅」に、有機EL照明器具29台、LED照明器具1,174台などを納入] - パナソニック 2012年3月30日（2013年6月23日 閲覧）</ref>。。
* 2013年（平成25年）
** 3月16日 - 月台延伸區域啟用<ref>{{Cite press release |url=http://www.tokyu.co.jp/file/130515-2.pdf |title=特定都市鉄道整備事業実施状況 |publisher=東京急行電鉄 |date=2013-05-15 |accessdate=2015-01-12 |format=pdf}}</ref>。
** 3月31日 - 「〜智慧模型自由丘站明亮計畫〜」：站內照明更新為「[[OLED|OLED]]照明器具」。

東橫線自2001年3月28日特急開始運行以來，與[[菊名站|菊名站]]一樣全日進行[[緩急接續|緩急接續]]。緩急接續時，外側副本線的3、6號線停靠各站停車，內側本線的4、5號線停靠特急、通勤特急、急行。平日早晨尖峰時段可見澀谷方向月台乘客大排長龍，大量人潮湧入急行、通勤特急列車。

過去大井町線的[[九品佛站|九品佛]]側有{{link-ja|自由丘檢車區|自由が丘検車区}}。2007年11月15日檢車區廢除。

站名標看板在2012年3月實施的「〜智慧模型自由丘站明亮計畫〜」中全面LED化，使用薄型的LED看板。

平日有本站起訖的各站停車班次，早晨有三班開往菊名站，夜間元町・中華街站有一班開至本站。

另外，3號線與6號線的副本線會在深夜停放車輛，隔日早晨再回送。

本站 - [[田園調布站|田園調布]]間設有橫渡線供列車緊急折返使用。

收費座席指定列車「[[S-TRAIN|S-TRAIN]]」在周六日駛入東橫線並停靠本站<ref>[http://www.seibu-group.co.jp/railways/news/news-release/2016/__icsFiles/afieldfile/2017/01/10/s-train.pdf 2017年3月25日（土）から「S-TRAIN」運行開始！] {{webarchive|url=https://web.archive.org/web/20170113072010/http://www.seibu-group.co.jp/railways/news/news-release/2016/__icsFiles/afieldfile/2017/01/10/s-train.pdf |date=2017年1月13日 }} 西武鉄道株式会社 2017年1月10日</ref>。

== 站名由來 ==
{{See also|自由之丘}}
附近的古刹{{link-ja|九品佛淨真寺|九品仏浄真寺}}安置9座[[阿彌陀佛|阿彌陀如來像]]，因而稱作「九品佛」（{{jpn|j=九品仏|hg=くほんぶつ}}）。車站設站當時作為最近站而命名為「九品佛前站」。隨著大井町線通車，該寺附近新設「九品佛站」，原先計畫以車站所在地[[荏原郡|荏原郡]]碑衾村大字衾（ふすま）更名為「衾站」，後因大字改為自由之丘，因而直接改為「自由丘站」。
