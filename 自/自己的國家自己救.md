'''自己的國家自己救'''是出自於中華民國的愛國歌曲《莫等待》中的一句歌詞，該曲為[[中國國民黨|中國國民黨]]的政治宣傳歌曲，而其歌詞後來演變成社會運動中一句常見的標語或口號。

《莫等待》由李士英作詞、談修作曲，是1977年[[中國國民黨|國民黨]]政府下的[[國防部政治作戰局|國防部總政治作戰部]]用以傳播政治理念的歌曲<ref>{{cite news |language = 中文 |author = |coauthors =  |url = http://www.cna.com.tw/magazine/53/201509080002-1.aspx |title = 【對著道貌岸然放響屁】「自己國家自己救」背後的暗潮洶湧 |publisher = 中央通訊社 |pages =  |date = 2015-09-08 |accessdate = 2017-01-04}}</ref>。原本、「自己的國家自己救」中的「國家」是指位於台灣的國民政府，而「救」指的是「光復祖國」。但隨著臺灣獨立的意識在臺灣人民間普及，臺灣主權在國際上又屢受中華人民共和國政府打壓，使得「自己的國家自己救」的意思轉變為「擺脫困境、建立獨立自主的國家」，而此時標語也可能被簡稱為「自己國家自己救」。在2014年[[太陽花學運|太陽花學運]]期間，「自己的國家自己救」此一標語大量地被運用在各種情境下：例如身兼獨立樂團[[拷秋勤|拷秋勤]]成員的設計師陳威仲將「自己國家自己救」印在T 恤上<ref>{{cite news |language = 中文 |author = 吳學展 |coauthors =  |url = https://buzzorange.com/2014/11/17/fuck-the-government-t-shirt/ |title =【後太陽花的運動臉譜】「自己國家自己救」T 恤出自他手！－專訪「激進」創辦人陳威仲 |publisher =報橘 |pages =  |date = 2014-11-17 |accessdate = 2017-01-04}}</ref>。時任中華民國總統馬英九在談論南海議題、對於中華民國在外交上的困境表示：「自己的國家自己救」<ref>{{cite news |language = 中文 |author = |coauthors =  |url = http://news.ltn.com.tw/news/politics/breakingnews/1588028 |title = 模仿太陽花口號 馬英九高呼：自己的國家自己救！ |publisher = 自由時報 |pages =  |date = 2016-01-28 |accessdate = 2017-01-04}}</ref>；時任宜蘭縣長[[林聰賢|林聰賢]]在[[周子瑜國旗事件|周子瑜國旗事件]]發生後，在聲援周子瑜時提到「自己的國家自己救」<ref>{{cite news |language = 中文 |author =  |coauthors =  |url = http://news.ltn.com.tw/news/politics/breakingnews/1574182 |title = 林聰賢聲援周子瑜：自己國家自己救！ |publisher = 自由時報 |pages =  |date =2016-01-16 |accessdate = 2017-01-04}}</ref>。除了台灣、這一標語也被用於[[香港獨立運動|香港獨立運動]]中<ref>{{cite news |language = 中文 |author = 陳五 |coauthors =  |url = http://www.passiontimes.hk/article/01-18-2016/28214 |title = 嗌「自己香港自己救」都有錯？　李慧琼：脫離咗我哋地位實況 |publisher = 熱血時報 |pages =  |date = 2016-01-18 |accessdate = 2017-01-04}}</ref>。

當「自己國家自己救」用於不同場合時，標語中的「國家」可能會被替換成其它詞彙。例如台灣足球運動員張武業在提及改善台灣的足球環境時，將標語改為「自己環境自己救」<ref>{{cite news |language = 中文 |author =  |coauthors =  |url = http://www.cna.com.tw/news/aspt/201612060045-1.aspx |title =灌溉足球沙漠 張武業：自己環境自己救 |publisher = 中央社 |pages =  |date = 2016-12-06 |accessdate = 2017-01-04}}</ref>；在談論《國民體育法》修法時，草案連署發起人張祐銓以「自己的體育自己救」為標語<ref>{{cite news |language = 中文 |author = 張祐銓 |coauthors =  |url =http://talk.ltn.com.tw/article/paper/1067779 |title = 張祐銓／自己的體育自己救！ |publisher = 自由時報 |pages =  |date = 2017-01-02 |accessdate = 2017-01-04}}</ref>；在提及因前往台灣旅遊的中國遊客減少、因而衝擊觀光產業時，使用「自己的觀光自己救」
<ref>{{cite news |language = 中文 |author = 陳宜加 |coauthors =  |url = http://www.chinatimes.com/newspapers/20161206000430-260114 |title = 自己的觀光自己救 業者登陸盼交流常態化 |publisher = 中時電子報 |pages =  |date =2016-12-06 |accessdate = 2017-01-04}}</ref>；在談論因[[東日本大震災|東日本大震災]]導致的食品安全問題時，使用「自己的健康自己救」<ref>{{cite news |language = 中文 |author =  |coauthors =  |url = http://www.cdnews.com.tw/cdnews_site/docDetail.jsp?coluid=112&docid=103964224 |title = 南市隱匿查獲日核食 國民黨：賴清德為仕途考量的政治正確 |publisher =中央網路報 |pages =  |date = 該新聞的日期 |accessdate = 2017-01-04}}</ref>。

「自己國家自己救」更被運用在書名中，例如時報出版的保健書籍《自己的膝蓋自己救》<ref>{{cite news|language =中文|author =|coauthors =|url =http://news.sina.com.tw/article/20161230/19962764.html|title =時報出版新書:自己的膝蓋自己救|publisher =新浪新聞|pages =|date =2016-12-30|accessdate =2017-01-04|deadurl =yes|archiveurl =https://web.archive.org/web/20161230144423/http://news.sina.com.tw/article/20161230/19962764.html|archivedate =2016-12-30}}</ref>。
