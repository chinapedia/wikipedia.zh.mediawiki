'''衣蛾'''（[[学名|学名]]：''{{lang|la|Tinea pellionella}}''）的分布极广，属于[[鳞翅目|鳞翅目]]的[[蕈蛾科|蕈蛾科]]。外觀類似幾何圖形的菱形，常出現於潮濕的牆壁上，以及衣物上。衣蛾如果在城市的话，它一般都会生长在最下面一楼。

== 型態 ==
衣蛾[[幼虫|幼虫]]是一个小型白色的毛虫，藏在一个丝质的袋状物或网状物（称为[[筒巢|筒巢]]）内，在墙壁上可见到一个黏著[[水泥|水泥]]的纺锤形丝袋，内有一深褐色头的幼虫。成虫为浅黄色的虫，惧光。
它有时候会爬出来，可是我们一般看不到。

== 食性 ==
以[[羊毛|羊毛]]、[[毛髮|毛髮]]、[[毛皮|毛皮]]、[[羽毛|羽毛]]为食，幼虫行动缓慢。会破坏[[纺织品|纺织品]]，在[[图书馆|图书馆]]或[[博物馆|博物馆]]会危害动物[[标本|标本]]。其成虫并不取食。

== 生活環境 ==
[[File:Tinea_pellionella01.jpg|thumb]]
[[成虫|成虫]]将卵产在[[皮毛|皮毛]]、[[羽毛|羽毛]]、皮品、[[毛|毛]]或污秽的[[丝绸|丝绸]]上。[[幼虫|幼虫]]会吐[[丝|丝]]作[[茧|茧]]，两端开口供取食及行动。幼虫在茧中成长，化[[蛹|蛹]]时仍在茧中，直到成虫羽化为止。特别喜爱[[纺织品|纺织品]]沾有[[食物|食物]]或其他污染，[[衣领|衣领]]或衣服折叠处也可见到衣蛾的踪迹，而[[纯棉|纯棉]]的衣物较不易受害。

== 預防 ==
最适宜的方式当衣物要收好时，需先作处理。将所有的衣物自衣柜移出，将摺叠处清理，在[[阳光|阳光]]下暴露几小时。[[乾洗|乾洗]]、[[湿洗|湿洗]]时应要烫好，用[[塑胶袋|塑胶袋]]装好[[密封|密封]]。

== 防治 ==
当[[衣柜|衣柜]]空出时，以[[陶斯松|陶斯松]]等喷洒衣柜。经处理后将衣服放回，衣柜要关紧，在上层放一些[[樟脑丸|樟脑丸]]或[[萘|萘丸]]等（依指示放入所需的量），尽记勿把有沾到污泥的衣服放入，因加上[[汗|汗]]味更易吸引衣蛾来为害。

[[馆藏品|馆藏品]]收藏时应先检查，有虫为害时应先作处理，定期检查并行[[熏蒸|熏蒸]]处理；并具环境控制的设施保持低温、低湿及清洁。[[衣鱼|衣鱼]]和衣蛾不叮咬[[人类|人类]]或传染[[病原菌|病原菌]]，易被忽略，发生后再处理恐难清除完全，在平时能清洁环境，[[报纸|报纸]]定期收集整理不堆积，送往[[资源回收|资源回收]]处集中处理。换季时衣服务必清洗[[烘干|烘干]]后再放入干净的衣橱，可减少危害虫的发生。
