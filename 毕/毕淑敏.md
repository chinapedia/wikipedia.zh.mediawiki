'''毕淑敏'''（{{bd|1952年|10月|||B毕}}），中国当代女作家，[[注册心理咨询师|注册心理咨询师]]；祖籍[[山东|山东]][[文登|文登]]，出生于[[新疆|新疆]][[伊宁|伊宁]]；[[北京师范大学|北京师范大学]]研究生院文学系硕士，[[国家一级作家|国家一级作家]]。1969年入[[中国人民解放军|中国人民解放军]]服役，曾在[[西藏|西藏]][[阿里|阿里]]地区当兵11年，是一名部队医生；转业后，任内科主治医师。1986年开始发表文学作品；1989年加入[[中国作家协会|中国作家协会]]，是中国作协第五、六、七届全国委员。现任北京市作家协会副主席。 

其每部作品，都以生命和死亡为主题。这主要是来源于毕淑敏曾长期在[[西藏高原|西藏高原]]工作的生活体验和思考。曾获[[庄重文文学奖|庄重文文学奖]]等。<ref>[http://www.chinaculture.org/gb/cn_zgwh/2005-11/09/content_75543.htm 中国文化网：毕淑敏] {{webarchive|url=https://web.archive.org/web/20090701134141/http://www.chinaculture.org/gb/cn_zgwh/2005-11/09/content_75543.htm |date=2009-07-01 }}</ref>

== 经历 ==

毕淑敏生于[[新疆|新疆]][[伊宁|伊宁]]。1965年考入[[北京外国语大学|北京外语学院]]附属学校中学部，学习[[俄语|俄语]]。17岁时，正值“[[文革|文革]]”期间，毕淑敏参军入伍，被分配至[[西藏|西藏]]阿里军分区；历任卫生员、助理军医、军医；在[[西藏高原|西藏高原]]当兵11年。1980年转业回[[北京|北京]]后，曾担任内科主治医师、卫生所所长。<ref>{{Cite web |url=http://isand.net/Article/?845.html |title=山东人物:毕淑敏 |access-date=2009-07-31 |archive-url=https://web.archive.org/web/20110523110837/http://www.isand.net/Article/?845.html |archive-date=2011-05-23 |dead-url=yes }}</ref>

1986年，毕淑敏在其34岁时写出处女作——小说《昆仑殇》，并于次年发表在《昆仑》杂志上<ref name="cctv">{{cite web|url=http://www.psytopic.com/mag/post/bi-shumin-from-writer-to-psychological-counseling-division.html|title=毕淑敏：从作家到心理咨询师|publisher=[[CCTV|CCTV]]新闻会客厅|date=|accessdate=2012-02-12|deadurl=yes|archiveurl=https://web.archive.org/web/20111220143904/http://www.psytopic.com/mag/post/bi-shumin-from-writer-to-psychological-counseling-division.html|archivedate=2011-12-20}}</ref>。她于1989年加入[[中国作家协会|中国作家协会]]。1991年，她退出已从事22年的医生职业，开始全职写作<ref name="cctv"/>。她于1991年获得[[北京师范大学|北京师范大学]]文学硕士学位。1994年，短篇小说《预约死亡》发表后，奠定了毕淑敏在中国文坛的声誉。这篇作品被誉为是“新体验小说”的代表作之一；并获得[[北京市|北京市]]“庆祝建国45周年文学作品奖”。此后，毕淑敏陆续发表作品《红处方》、《拯救乳房》、《翻浆》等。

1998年她于46岁高龄进入[[北京师范大学|北京师范大学]][[心理学|心理学]][[博士|博士]]方向入读，2002年课程结业，但因为外语和论文的原因没有获得博士学位<ref name="xmwb">{{cite web |url=http://news.eastday.com/eastday/node17/node136/node6089/userobject1ai78286.html |title=提升幸福指数 毕淑敏：听心底花开的声音 |publisher=[[东方网|东方网]]，原载于《[[新民晚报|新民晚报]]》|date=2006-03-05 |accessdate=2012-02-12}}</ref>。其后，她成为注册心理咨询师，并开办诊所，后又因与写作冲突而关闭<ref name="xmwb"/>。2009年7月，受[[中国中央电视台|中国中央电视台]]邀请，在其[[中国中央电视台科教频道|科教频道]]的《[[百家讲坛|百家讲坛]]》节目中，录制《关爱心灵》系列讲座。<ref>[http://book.ce.cn/zzdt/200907/28/t20090728_19643706.shtml 毕淑敏能救《百家讲坛》吗?] 原载《深圳商报》</ref>

== 家庭 ==
毕淑敏和丈夫[[芦书坤|芦书坤]]是在[[西藏|西藏]]当兵时认识的。两人于1977年结婚；婚后育有一子，[[芦淼|芦淼]]。<ref name="bjqnb">{{cite web |url=http://news.xinhuanet.com/book/2004-03/16/content_1366961.htm |title=丈夫眼中的毕淑敏 |publisher=[[新华网|新华网]]，原载于《[[北京青年报|北京青年报]]》|date=2004-03-16 |accessdate=2012-02-12}}</ref>
