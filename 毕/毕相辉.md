'''毕相煇'''（1905年-1947年），名佩璐，字鄂华。[[安徽|安徽]][[歙县|歙县]]县城上路街人。

父亲毕醉春，名恩桂、字雪胸，为[[徽州|徽州]]教育名宿。清末废[[科举|科举]]办学校之初，即任教绩溪思诚学堂。其后，又应胡晋接校长之邀，任教安徽省立第二师范国文课，以对学生循循善诱、殷殷教诲闻名于时，曾与胡晋接校长同获教育部嘉和章之奖。有凡经毕老师教导，则务实求真，文字通顺之誉。休退居家后，徽歙各地殷殷来求学者，无不精心教导启迪，因材施教。母亲方啸云，善画，并擅武术，亦设塾教乡邻子女。对学子和儿孙常说“要吃就要做”，既读书也要劳动。毕相煇出生于这个家庭，耳濡目染，加之较早接受了现代进步思想，早年即养成爱国进步、以天下为己任的高尚品德。

毕相煇毕业于安徽二师后，考入[[清华大学|清华大学]]和[[清华国学院|清华国学院]]。为农村经济研究会会员，在南开经济研究所研究棉花、纱布、茶叶等问题。民国20年在上海曾以共党嫌疑被关押。1935年（民国24年）以官费留学日本[[东京帝国大学|东京帝国大学]]，专攻农业经济。抗战前归国，历任福建省银行研究处、重庆国民政府花纱布管理局、农本局研究室主任和[[复旦大学|复旦大学]]教授兼茶叶研究室主任。留日时译成[[列宁|列宁]]《俄国资本主义发展》。在重庆，与[[周恩来|周恩来]]、[[邓颖超|邓颖超]]、[[郭沫若|郭沫若]]、[[许滌新|许滌新]]、[[罗叔章|罗叔章]]等多有过从。与[[陶行知|陶行知]]、[[孙起孟|孙起孟]]积极参与[[黄炎培|黄炎培]]先生[[民主建国会|民主建国会]]的成立。1944年（民国33年）创建国际贸易咨询所<ref>该所当时对外公开为孔祥熙的中国国际经济协会的下属机构，实际“有苏北投资”（毕相煇告刘海尼和王世杰），是为解放上海后的经济斗争作准备的。该所总干事[[冀朝鼎|冀朝鼎]]及[[卢绪章|卢绪章]]、[[勇龙桂|勇龙桂]]和继任所长[[陈维稷|陈维稷]]等均为中共地下党员。</ref>，请国际劳工局中国分局局长[[程瀛元|程瀛元]]<ref>程瀛元，字海峰，徽州屯溪湖边人，亦省立二师毕业生。其父程仲沂亦二师教师，垂暮之年仍与毕醉春时相交往。</ref>兼任所长，自任副所长。民国36年在[[宣城|宣城]][[水东|水东]]遇害。[[吴大琨|吴大琨]]撰文悼念，称其治学切实，待人诚实，为发展民族经济忠实，其死是中国经济学术界一个不可补偿的损失！

== 毕相煇年表 ==
* 1905年生于安徽歙县县城上路街。因三姐幼逝，排行第五，前有毕大煇、毕咏莲、毕笑梅、毕品菊（滔淇）兄姐四人。后有五妹吟萱，亦早丧。
* 1923年，毕业于安徽省立二师。
* 1924－1926，天津南开大学就读。时长兄毕大煇（名璲，字腾华）在孙家鼐（安徽寿县人）系统的天津中孚公司工作。
* 1926年8月11日，年考取清华大学国学院。受教于清华国学院五大导师：梁启超、王国维、陈寅恪、赵元任、吴宓。
* 1929至1930。在无锡、上海工作，曾任教中学并为南开经济研究所研究棉花、棉纱、棉布、茶叶等经济问题。这些工作不一定有多少钱，研究是无偿奉献。1930年，柔石等五人遇害期间，毕相煇遭关押一月。
* 1935年（民国24年），以第二名的成绩考上安徽省“官费”留学生名额，去日本东京帝国大学攻读农业经济，参加中国农村经济研究会。1937年抗日战争爆发前匆匆回国。据外甥王世杰回忆，当时换洗衣服都是破的，皮鞋底也已穿了，可见其艰难困苦。
* 1937年（民国26年）回国后曾回歙县老家省亲。接郭沫若主持的“国家资源委员会”来信，赶赴重庆，先后曾在福建省银行经济研究处、重庆花纱布管理局和马寅初主持的农本局担任研究工作。
* 1940年（民国29年）秋，兼任重庆复旦大学茶叶研究室主任。
* 1943年，成立中国工业经济研究所，任副所长，吴蕴初为董事长，章乃器为所长。抗日战争后期，民营工业备受帝国主义侵略和官僚资本的摧残，面临崩溃的严峻时刻。研究所是为了挽救民族工业的危机，探索中国工业生存和发展的道路，发展民族工业，以利于抗日救国而在重庆创立的。
* 1944年，创办“国际贸易咨询所”。
* 1945年，参与发起“中国民主建国会”，为理事。
* 1947年（民国36年），在安徽宁国水东镇遇难。

== 毕相煇在清华==
1926年8月11日， 毕相煇考取清华大学国学研究院第二届学生。本届录取新生24名，备取2名，实到24名。

在清华期间，受教于[[王国维|王国维]]、[[梁启超|梁启超]]、[[陈寅恪|陈寅恪]]、[[吴宓|吴宓]]、[[赵元任|赵元任]]等大师。毕相煇对王国维先生的学问和治学精神最为钦佩，在寄父母的家信多次提及。1927年，王国维先生投湖自沉，毕相煇作为学生为一代宗师送行。

== 毕相煇在日本 ==
在日本东京帝国大学留学期间，毕相煇参加了中国农村经济研究会。当时会员有[[吴觉农|吴觉农]]、[[冯和法|冯和法]]、[[吴大锟|吴大锟]]、[[薛暮桥|薛暮桥]]、[[许滌新|许滌新]]、[[乔冠华|乔冠华]]等。毕相煇与他们志同道合，深相接纳。归国后都努力支持艰难创办国际贸易咨询所。

期间，还翻译列宁《俄国资本主义的发展》一书。

==毕相煇和民主建国会==
1945年12月16日，为积极参加新民主主义革命斗争，民主建国会在重庆成立，成员主要是与爱国民族工商业者有关的专家学者，发起人有黄炎培、胡厥文、章乃器、施复亮、孙起孟等。民建会成立选出37位理事、19位监事。毕相煇是理事之一。孙老起孟晚年还对王世杰追忆当时艰难的情景。

成立大会上召开第一次常务理事会，毕相煇被推选为秘书处言论出版组的副主任。为了宣传国是主张、推进会务和沟通成员之间的联系，会议决定创办《平民》周刊。毕相煇为编辑委员之一 。

==毕相煇和中国国际经济协会国际贸易咨询所==
1944年10月（民国34年）成立中国国际经济协会国际贸易咨询所。孔祥熙任协会理事长，冀朝鼎为协会总干事，程海锋为副总干事兼所长，毕相煇任副所长。所址设在上海江西路三一六号。

咨询所的任务主要为沟通中外贸易双方需求，开展咨询解答，代办调查、供应商情、介绍业务等服务项目。除与国内各行政机关、经济部门有广泛联系外，还与国外工商团体近数千单位有联系。

咨询所创办，正处于第二次世界大战结束时，国际贸易进入一个新的发展时期，国际上对中国贸易的需求激增。毕相煇及时主持编辑出版了《行名录》一书。该书在调查研究的基础上汇集了全国1600多个进出口工商业的有关资料，为中外进出口提供了所需的信息。除在内地发行外，尚有英文版在台湾、香港发行。

==毕相煇遇难==
1947年2月，老母病逝，毕相煇丁忧返歙<ref>毕相煇在祭母文中，赞扬老母一生重视劳动教育，周济贫苦乡邻。教育后辈要读书，也要劳动，“要吃就要做”。偷懒就是“废料”。为慰老父，则依依长谈，论国家天下大势，光明前途。在寄刘海尼家信中盛赞老父“高度的艺术天才和脚踏实地的斗争的人格”。老人早年实已取“淡泊明志，宁静致远”之义自额其宅为“澹宁轩”。内外孙辈儿女十三人亲其教泽，奋发成才，都参加革命。解放大军渡江前后，老人泰然自若，掩护外孙王世杰、汪宜楷等与上海地下党和皖南游击队联系，在歙县、屯溪一带做策反与安民的工作，并亲自迎接二野十二军政委裴光率队从北门大塘村进城。在家书中，老人盛赞解放军“仁义之师，纪律严明”。
</ref>。但因上海咨询所事务百废待兴，《行名录》等编辑工作亟待审核，毕相煇于3月27日匆匆赶回上海，途经宣城水东镇，不幸为当地土劣枪伤致死。据当年报道情况：国民政府大通公路总局一辆客车驶经宣城县水东镇，该镇商会会长包金榜和水东乡副乡长余国鼎带领四名枪兵在车站候车。客车因煞车失灵行驶过站，包、徐命士兵开枪射击，毕相煇中弹身死。中国国际经济协会会长孔祥熙致函宣城地方法院首席检察官，谓“包金榜为主犯，应依法严办”。

上海《联合晚报》亦于4月21日登载：“中国国际经济协会国际贸易咨询所副所长毕相煇先生，前因丁忧返乡，于上月二十七日乘公路车由安徽歙县经宁国、宣城返沪，不幸车经河沥溪东约二十余里水东镇地方，有该地曾在敌伪盘踞期间作威作福的土劣商会会长包德邦中途欲拦阻上车，该车因天雨地滑，刹车不住，包某以为故意拒绝乘车，遂命令枪兵立即向车开枪，共发七、八发子弹，由车后厢传入，毕氏不幸弹中腹部，当场牺牲，噩耗传来，沪上毕氏生前好友，莫不悲愤万分…”。

当时中国，社会混乱，政治黑暗，地方势力对孔祥熙的函电也是阳奉阴违，敷衍应付。毕相煇亲属毕佩珩、毕增祺和挚友张朴先后去宣城办理后事，并进行诉讼，都是举步维艰，无从下手。最后，毕相煇之妻刘海尼于万分悲恸中求助于史良律师，指导外甥王世杰再去宣城依法斗争。史良介绍芜湖律师吴麟接办此案。王世杰得到一位上海法学院毕业在宣城法院工作的青年书记官的指导，依据《[[六法全书|六法全书]]》，要求法院在刑事判决之前对包金榜的家业执行“[[假扣押|假扣押]]”。被告才慌了手脚，为死者认真办理后事（遗体送回家乡安葬），并对死者家属作出应有的经济赔偿。最终，宣地地方法院以教唆杀人罪判处包金榜、徐国鼎各有期徒刑12 年，以共同杀人罪判王宝仁、王有财、冯金榜各有期徒刑10年，包等不服上诉。高三分院审理结果，维持一审对包金榜判处12 年有期徒刑，徐国鼎由于少年轻躁，王宝仁由于受命开枪，各减处七年有期徒刑，王有财、冯金榜由于朝天鸣枪，改判无罪。据王世杰回忆，包某直到解放后才以土豪恶霸多罪并发受到严正的法律制裁。

同年，吴大琨先生在《经济周刊》专门撰文《悼毕相煇先生》。冀朝鼎、冯和发、吴觉农、何耀曾均以书信方式表示悼念。

==毕相煇的经济思想==
毕相煇遇难后，吴大锟在《悼毕相煇先生》（载于民国三十六年出版的《经济周报》）中指出：他在这许多年来，研究中国棉花、棉纱、棉布等具体经济问题过程中，切实地体验了中国必须发展民族资本的必要。而要发展民族资本，就必须真正地培养起一些真正能够负担起时代任务来的民族资本家不可。他认为作为一个中国经济问题的研究者是应当有首先负起这种为发展真正的中国民族资本事业而服务的任务的必要的。他在这一意义上才决心选择了以发展真正中国的民族国际贸易为起点，来从事这一工作…这一理论上的路线，我是万分同意的。这与当前中国人民运动的路线是完全相合的。”

==毕相煇遗留的诗稿==
诗一：1930年赠四姐毕品菊。
: 仰首问天人类何太苦？
: 天公但谓人类自束缚。
: 吁嗟闻此复何言，
: 等闲拔剑歌还哭。
: 茫茫不视尘间物，
: 恰似春霜和秋露。

诗二：夫人刘海尼所创作的剧本《伟大的失败》遭国民党当局封杀，赠此诗以慰。
碰壁回响

: 这是一篇新鲜而成熟的作品,
: 他不幸碰了两重的厚壁。
: 他是像山泉水一样的潆澈，
: 而无语地流淌着；
: 他是世俗的庸人所不能体味的了。
: 他应是像自古以来伟大的天才的
: 缄默而雍然吧！
: 我蕴着热泪向他致敬，并且说：
: ——《伟大的失败》不朽！
