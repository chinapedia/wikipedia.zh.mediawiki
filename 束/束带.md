'''束带'''是一種{{tsl|ja|平安装束}}，是公家的正装。也被称为“昼装束”。

== 组成 ==
束带从内到外由{{tsl|ja|単衣|单 (和服)|单}}、{{tsl|ja|袙|袙 (和服)|袙}}、{{tsl|ja|下襲|下襲 (和服)|下襲}}、{{tsl|ja|半臂|半臂 (和服)|半臂}}、{{tsl|ja|袍|袍 (和服)|袍}}组成，并在袍外腰部系一条皮革腰带，即{{tsl|ja|石帯|石带 (和服)|石带}}。袴分有{{tsl|ja|大口袴|大口袴 (和服)|大口袴}}、[[表袴|表袴]]两种，表袴需穿着在大口袴之外。戴{{tsl|ja|日本の冠|冠 (和服)|冠}}，穿襪（{{lang|ja|しとうず}}），怀藏帖紙（{{lang|ja|たとう}}）、[[檜扇|檜扇]]，手执[[笏|笏]]。[[公卿|公卿]]、[[殿上人|殿上人]]要在腰间挂[[魚袋|魚袋]]。

束带的穿着有文、武官之别。文官以及三品以上武官身着{{tsl|ja|袍|縫腋袍}}（{{lang|ja|ほうえきほう}}），头戴垂纓。四品以下武官身着{{tsl|ja|袍|闕腋袍}}（{{lang|ja|けってきほう}}），头戴卷纓。另外，武官、[[中務省|中務省]]官员以及获勅许的[[参议|参议]]以上官阶的文官可佩[[日本刀|大刀]]。佩大刀时，要将刀用[[平緒|平緒]]（{{lang|ja|ひらお}}）绑在腰间。

== 特征 ==
* 下襲的背面部分很长，穿着时会像燕尾一样拖在地上。这部分称作「裾」（{{lang|ja|きょ}}），是束帶的一大特征。由于裾的长短代表着身份的贵贱，部分下襲经演变变得过长，于是裾与下襲分离，称作別裾。着別裾时，先穿上下襲，再将別裾的带子系在腰间。[[天皇|天皇]]和[[储君|皇太子]]的下襲的裾并没有与本体分离。另外，还有一种很短的裾称作纔著（{{lang|ja|さいじゃく}}），用于[[地下官人|地下官人]]穿戴。
* 在冬天，由于文官所穿的縫腋袍的肋旁是封闭的，从外观上看不出里面有无穿上半臂，所以文官在着装时通常将半臂省去。在夏天，换上较薄的縫腋袍后，由于可透过袍看到里面有无穿上半臂，因此需穿上；然而在近代，通常在夏天也不穿着半臂。
