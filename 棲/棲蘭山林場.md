'''棲蘭山林場'''之名稱，是因為伐木事業所影響，才有此說法，管理單位則稱為「棲蘭山林區」。棲蘭山林區於1949年，由[[林產管理局|林產管理局]]撥給[[退輔會|退輔會]]森林開發處（今為[[國軍退除役官兵輔導委員會榮民森林保育事業管理處|森林保育處]]）安置[[榮民|榮民]]之林區。

== 範圍 ==
林區位於[[雪山山脈|雪山山脈]]之北稜為中心，稜線以西為[[淡水河|淡水河]]集水區，以東則為[[蘭陽溪|蘭陽溪]]集水區，涵括宜蘭、新北、桃園、新竹、台中等縣市，面積合計45,799公頃。<ref>行政院退輔會榮民森林保育事業管理處編，《棲蘭山森林經營計畫》（台北：退輔會，2003），頁16。</ref>

== 伐木事業 ==
[[日治時期|日治時期]]政府就注意到棲蘭山林區的資源，唯還沒開發就已戰敗撤離。戰後[[國民政府|國民政府]]大舉發展林業，棲蘭山自交撥予[[行政院國軍退除役官兵就業輔導委員會|行政院國軍退除役官兵就業輔導委員會]]森林開發處後，即著手進行林木砍伐計劃，在退輔會所轄之林區中，伐木事業就以棲蘭山林區為主（其他如[[大甲溪|大甲溪]]林區與[[立霧溪|立霧溪]]林區），不論是伐木時間或伐木數量都是第一，因此棲蘭山林區可說是退輔會伐木事業之代表。森林開發處在棲蘭山設立工作站，於1959年5月開始作業，直至1994年6月才停止伐木，為全台伐木林區最晚者。

棲蘭山事業區的伐木作業於戰後開啟，其運材的相關設施與日治時期林場有所不同，主要是運用架空索道，配合林區產業道路的卡車接駁，而不是以索道加輕便鐵道進行運材作業，自1959年至1985年間，共開闢71.8公里的林區主要幹道與支線，並銜接至[[台七線|台七線]]（[[北橫公路|北橫公路]]）與[[台七線#甲線|台七甲線]]（北橫公路宜蘭支線）。今[[明池森林遊樂區|明池森林遊樂區]]內之中國歷代神木區，即當時的林業道路，現今許多林業道路已成為保育山林資產的重要設施，<ref>行政院退輔會榮民森林保育事業管理處編，《棲蘭山森林經營計畫》，頁32-34。</ref>昔日為砍伐森林所開，現成為保育森林之用，歷史的轉折總是充滿戲劇性。

身為特許的伐木事業之一，退輔會[[森林開發處|森林開發處]]在發展棲蘭山林場的同時，還設立一系列興伐木相關之事業，如木工廠，將二級木材加入削片，製作紙漿原料，供應各大紙廠；製材廠則是承製家俱與課桌椅等，並外銷木工家具，試圖結合林業的產業鍊，打出自己的一片天。與[[大雪山林業公司|大雪山林業公司]]一般，想透過伐木（上游）與下游木業整合，最後卻皆以失敗作收。
　　

== 棲蘭國家森林遊樂區 ==
[[File:宜蘭棲蘭山檜木林20.jpg|right]]
伐木事業終止之際，森林開發處將棲蘭山部分林區規劃為[[森林遊樂區|森林遊樂區]]，將所屬太平山事業區第23-27、30等六個林班地，面積共1,717.15公頃。森林遊樂區於1992年向台灣省政府申請，1994年准予。不過根據資料顯示，1992年就已經開放遊客入園，並且收取門票，總計1992－2001年間，共有1,056,176名遊客入園參觀。<ref>行政院退輔會榮民森林保育事業管理處編，《棲蘭山森林經營計畫》，頁31。</ref>園區內有著名之神木園區，以及蔣介石行館等設施，並有兩條森林登山步道，供遊客休憩使用，自2005年起開始轉由民間業者經營，第一次得標的業者為椰子林公司，由其負責營運，十年營運到期後，2015年起轉由[[力麗集團|力麗集團]]旗下所成立的力麗明池公司負責營運<ref>江志雄，[http://news.ltn.com.tw/news/politics/breakingnews/1887344 棲蘭、明池易主經營 國庫每年進帳5000多萬]，自由時報，2016-11-15</ref>。
