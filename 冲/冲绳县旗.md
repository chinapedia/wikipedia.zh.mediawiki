'''[[冲绳县|冲绳县]]旗'''（{{jpn|j=沖縄県旗|hg=おきなわけんき|rm=Okinawa ken ki|f=}}）
冲绳县旗是[[日本|日本]]的47面[[日本都道府县旗帜|都道府县旗]]之一。该条目是对冲绳县旗以及'''冲绳县章'''（{{jpn|j=沖縄県章|hg=おきなわけんしょう|rm=Okinawa ken ki|f=}}）的解说。

==概要==
1969年，[[日美首脑会议|日美首脑会议]]上美国政府做出了[[沖繩政權移交|移交沖繩政權]]的决定。从此，开始进行县章的征集工作。1972年5月15日，[[冲绳|冲绳]]举行交接仪式，同日的第3号县告示上在186件设计方案中选出了县章的决定方案<ref>{{cite web |url=http://www.pref.okinawa.jp/reiki/34790250000300000000/34790250000300000000/34790250000300000000_blk0.html |title =沖縄県章の制定 昭和47年５月15日 告示第３号 |date = |language =ja |publisher =沖縄県|accessdate = }}</ref>。外圆表示[[海洋|海洋]]，中间白色的圆是“Okinawa”的第一个字母“O”，同时，“O”也象征着县民生活的和谐，内圆则表示[[全球局势|全球局势]]下[[冲绳|冲绳]]不断发展与进步的可能性。
县旗于1972年10月13日在第135号县告示上制定。<ref>{{cite web |url=http://www.pref.okinawa.jp/reiki/34790250013500000000/34790250013500000000/34790250013500000000_blk0.html |title =沖縄県旗の制定 昭和47年10月13日 告示第135号|date = |language =ja |publisher =沖縄県|accessdate = }}</ref>。县旗的底色是白色的，中央配有红色的县章。至于这样设计的具体理由，[[冲绳|冲绳]]政府并没有做出特定的说明。
