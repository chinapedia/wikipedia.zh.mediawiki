《'''两天一夜'''》（{{Lang-fr|'''''Deux Jours, une nuit'''''}}，{{lang-en|'''''Two Days, One Night'''''}}）是一部2014年[[达顿兄弟|达顿兄弟]]执导的法语电影，[[瑪莉安·歌迪雅|瑪莉安·歌迪雅]]和[[法布里齐奥·罗吉恩|法布里齐奥·罗吉恩]]主演。入围[[第67屆坎城影展|第67屆坎城影展]]主竞赛单元。<ref name="Cannes2014">{{cite web |url=http://www.festival-cannes.fr/en/article/60533.html |title=2014 Official Selection |accessdate=17 April 2014 |work=Cannes}}</ref>本片赢得2014年[[悉尼电影节|悉尼电影节]]最高奖，歌迪雅获得第27届[[欧洲电影奖|欧洲电影奖]]最佳歐洲女演員。代表比利时角逐第87届[[奥斯卡最佳外语片|奥斯卡最佳外语片]]，瑪莉安获得[[奥斯卡最佳女主角|奥斯卡最佳女主角]]提名。

== 剧情 ==
在比利时的[[瑟兰|瑟兰]]，女主角桑德拉是一个刚走出[[忧郁症|忧郁症]]的女工、两个孩子的妈妈，却马上面临工厂裁员和失业。在丈夫坚定的支持和陪伴下，她要在一个周末的时间，去说服同事工友们为她重新投票，以保住至关重要的工作。然而，选择支持她的工友需要是以失去1000欧元的奖金为代价。

经过不懈努力，在投票会中她只赢得刚好一半人（16人中的8位）的支持，由于没有超过一半票数，桑德拉准备收拾东西走人。此时主管突然告诉她只需要解除另外一个临时工的合同，就可以让她留下。可是这位临时工黑人Alphonse在投票时是站在了她的一边的。最终桑德拉选择了辞职，决定开始自己新的生活。

== 评价 ==
[[网易|网易]]影评认为：“影片里弱化甚至隐藏了“恶”的表现，与达内兄弟之前的作品比起来，《两天一夜》的立意略显单薄，没有了善恶的交融，变成了[[工人阶级|工人阶级]]的纯励志大片。”<ref >{{cite web|url=http://ent.163.com/14/0521/08/9SOO23ON00034TL2.html | language= | title=《两天一夜》：略显单薄的励志片 |publisher=网易娱乐 |date=2014-05-21 |accessdate= }}</ref>新浪影评：“歌迪亚扮演这个从脆弱敏感自尊到终于坚强并敢于做出决定承担后果的女人，仿佛深入到灵魂骨髓……《两天一夜》仿佛一个西部公路片，人们踏上征程，为了生存挣扎，直面利益和人性的对立做出选择。而达内兄弟带着没有偏见的镜头，诚恳写实地纪录下这个过程中人们的喜怒哀乐，他们的反应可能是温情、无奈甚至无情，影片却没有任何居高临下的道德绑架来做出审判。唯一的反面角色是代表资本家的工头和企业主，他们从资本出发的冷酷和算计，达内兄弟轻笔带过，表达的力量却毫不含糊。”<ref >{{cite web|url=http://ent.sina.com.cn/r/m/2014-05-22/21434146516.shtml | language= | title=评《两天一夜》：达内兄弟的现实关怀 |publisher= 新浪娱乐 |date=2014年5月22日 |accessdate= }}</ref>
