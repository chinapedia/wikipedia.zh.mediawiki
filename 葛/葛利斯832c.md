'''葛利斯832c'''（'''Gliese 832 c''' 或 '''Gl 832 c'''、'''GJ 832 c'''），是一顆距離地球約16光年，位於[[天鶴座|天鶴座]]的[[太陽系外行星|太陽系外行星]]，母恆星是[[紅矮星|紅矮星]][[葛利斯832|葛利斯832]]<ref name = "space.com"/>。該行星的[[地球相似指數|地球相似指數]]為0.81，是已知系外行星中該指數最高的行星之一<ref name = "phl">{{Citation|author = Abel Mendez Torres|url = http://phl.upr.edu/press-releases/gliese832|title =  A Nearby Super-Earth with the Right Temperature but Extreme Seasons|date = June 25, 2014}}</ref>。它位於母恆星的[[適居帶|適居帶]]內。

葛利斯832c發現至今是潛在適居系外行星中距離地球最近的幾個<ref name = "space.com"/>。

==概要==
葛利斯832c的質量大約是[[地球|地球]]的5.4倍。它的預期表面溫度類似地球，但是會隨著環繞恆星時與母恆星距離變動而改變。葛利斯832c的[[軌道離心率|軌道離心率]]相對地球而言較高，軌道週期約36日<ref name = "phl"/>。葛利斯832c表面的[[行星平衡溫度|平均平衡溫度]]推測是253 [[熱力學溫標|K]]（−20℃）<ref name = "phl"/>。但是它可能擁有濃密大氣層，使它的表面溫度可能遠高於平衡溫度，並且更類似[[金星|金星]]<ref>{{Citation|author = Nancy Atkinson|url = http://www.universetoday.com/112806/nearby-super-earth-is-best-habitable-candidate-so-far-astronomers-say/|title = Nearby Super-Earth is Best Habitable Candidate So Far, Astronomers Say|date = June 25, 2014|accessdate = June 26, 2014}}</ref><ref name = "dailystar">{{Citation|author = David Snelling|url = http://www.dailystar.co.uk/tech/385831/Gliese-832c-alien-planet-is-discovered-by-astronomers|title = Alien planet discovered! And it's not that far away|date = June 26, 2014}}</ref>。

葛利斯832c是位於所屬母恆星適居帶內的[[超級地球|超級地球]]。雖然它和母恆星的距離遠低於日地距離，但其母恆星是紅矮星，因此其所接收的能量相當於地球接收的太陽輻射能<ref name = "space.com">{{Citation|author = Mike Wall|publisher = [[space.com|space.com]]|url = http://www.space.com/26357-exoplanet-habitable-zone-gliese-832c.html|title = Nearby Alien Planet May Be Capable of Supporting Life|date = June 25, 2014|accessdate = June 26, 2014}}</ref>。

==發現與意义==
葛利斯832c是由[[新南威尔士大学|新南威尔士大学]]的天文學家{{link-en|羅伯特·威滕邁耶|Robert A. Wittenmyer}}（Robert A. Wittenmyer）領導的國際團隊發現<ref name = "phl"/><ref name = "space.com"/>。該行星被發現時是「[[潛在適居太陽系外行星列表|適居太陽系外行星目錄]]」內的行星中距離地球第2近的<ref name = "phl"/>。

葛利斯832c的發現者描述該行星是「至今最近且最好的適居行星候選者」<ref>{{Citation|author = Hannah Osborne|publisher = ''[[国际财经时报|International Business Times]]''|url = http://www.ibtimes.co.uk/gliese-832c-alien-most-earth-like-planet-found-16-light-years-away-1454270|title = Gliese 832 c: 'Best Habitable World Candidate' Discovered 16 Light Years Away|date = June 25, 2014|accessdate = June 16, 2014}}</ref>。

天文學家將對葛利斯832c進行更進一步研究以確認它是否適合生命存在<ref name = "dailystar"/>。
