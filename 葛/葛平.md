'''葛平'''（{{bd|1960年|6月4日{{efn|1=在葛平生日被揭晓之前，曾被网友误以为是1974年6月1日或1974年11月11日。}}||catIdx=G葛}}），[[中国|中国]][[湖南|湖南]][[长沙|长沙]]人，毕业于[[武汉音乐学院|武汉音乐学院]]，是[[中国大陆|中国大陆]]的[[配音演员|配音演员]]及[[演员|演员]]。最初由於其曾经为《[[藍貓淘氣三千問|藍貓淘氣三千問]]》系列动画中的主角“[[藍貓_(角色)|藍貓]]”配音而知名於[[藍貓_(角色)|藍貓]]粉丝圈中。

2009年，一位播客在[[优酷|优酷]]上发表了两段名为“蓝猫的配音演员：葛平”的视频<ref name="葛平2">[http://v.youku.com/v_show/id_XNDI4MDEzNzI=.html 蓝猫的配音演员－－葛平 .{{简体中文}}]</ref><ref name="葛平1">[http://v.youku.com/v_show/id_XMTExMjYxMzU2.html 蓝猫的配音演员--葛平(新版)]</ref>，该两段视频来源于某段时间在《蓝猫淘气三千问》动画播出期间插播的各配音员自我介绍。过后一段时间有些网友将葛平的两段自我介绍视频进行了重新剪辑，产生了大量恶搞[[音MAD|音MAD]]。在[[ACFUN|ACFUN]]的某视频中，一网友经过剪辑，将葛平说过“葛”字和“炮”字组合在了一起，变成了“葛炮”，葛平也因此被网友附上了“葛炮”的称谓。

KUSO视频愈来愈广，使葛平在中国大陆网友当中人气大增。后在中国大陆著名弹幕视频网站[[bilibili|bilibili]]里面，葛平被列入“'''鬼畜全明星'''”之一<ref>[http://www.bilibili.com/sp/%E8%91%9B%E7%82%AE 葛平专题]，bilibili</ref>。另外，其也是著名的网络音MAD“葛炮系列”，使其名气也传入港台地区。

葛平现时仍然在[[湖南蓝猫动漫传媒有限公司|湖南蓝猫动漫传媒有限公司]]工作，但是在“[[蓝猫_(动画)|蓝猫系列动画片]]”中，其已不再为蓝猫配音，而改配其他角色：例如《[[蓝猫平安出行|蓝猫平安出行]]》中的“警察”和《[[蓝猫龙骑团|蓝猫龙骑团]]》中的“[[巴豆_(角色)|巴豆]]”。他也曾经参加过两部[[电视连续剧|电视连续剧]]的演出，分别在1999年饰演了《[[蜜里逃生|蜜里逃生]]》中的“旦旦爸爸”<ref name="蜜里逃生">[http://movie.douban.com/subject/3071275/ 蜜里逃生]，豆瓣电影</ref>，2014年饰演了《[[女王驾到|女王驾到]]》之《[[搜查女王|搜查女王]]》中的“周沫爸爸”。

== 网络事件 ==
[[File:Ge_Ping_2.JPG|thumb]]
2009年，一位叫做篆毫的优酷播客在[[优酷|优酷]]上发表了两段名为“蓝猫的配音演员：葛平”的视频，该两段视频来源于某段时间在《蓝猫淘气三千问》动画播出期间插播的各配音员自我介绍。

其中一段视频<ref name="葛平1"/>，葛平首先介绍自己是为蓝猫配音的演员并说明自己的名字，之后介绍自己小时候兴趣很广泛{{efn|1=“兴趣很广泛”被恶搞成“性趣很管饭”等}}，包括唱歌、跳舞、绘画，并指出自己小学一年级的时候能够背诵[[毛主席语录|毛主席语录]]。再讲到当时每天需要在班上读报纸，参加学校文艺宣传队的活动，以及自己的刻苦的练习，都为其为[[蓝猫_(角色)|蓝猫]]配音打下了基础。然后便介绍自己的[[练声|练声]]方法，在10多岁的时候每天早上坚持5点钟起来到长沙市的[[天心阁|天心阁]]公园里面练声（同时还要练[[绕口令|绕口令]]），之后示范练声方法：读出「mimimimamama」{{efn|1=参见{{YouTube|Dm_b1OIcBYY|link=no|KUSO视频《葛炮的号哭》|m=2|s=11|language=简体中文}}}}。然后示范练绕口令，先「噼里啪啦噼里啪啦，把嘴皮子活动开」{{efn|1=嘴皮子，活動開。[[色情聯想|色情聯想]]}}，然后再练具体的绕口令，比如「八百标兵奔北坡，北坡炮兵并排跑。炮兵怕把标兵碰，标兵怕碰炮兵炮。」{{efn|1=著名的繞口令}}。结束时，葛平勉勵电视机前的小朋友们：将来想当配音演员吗？现在就开始学习吧！

还有一段视频<ref name="葛平2"/>，葛平先向[[秦皇岛市|秦皇岛市]]的小朋友们，[[石家庄市|石家庄市]]的小朋友们还有[[河南省|河南]][[安阳市|安阳]]的小朋友们问好，然后介绍自己在2002年10月来到了秦皇岛、石家庄，做现场的表演{{efn|1=現場的XX表演}}。特别提出在石家庄表演时发生的一件事情：
{{Cquote|当时我们在石家庄表演的时候，我说：「小朋友们，谁愿意跟我蓝猫学一句配音哪？」，只听见下面有一个（聲音）：「我来！」这個时候，我一看，小朋友们，人群当中，钻出来一个光头！哇！他全身穿的都是我们蓝猫品牌的衣服，特别显眼！我一把就把他抱了起来。我说：「你叫什么名字啊？」，他说「我叫吴克」，「你今年多大啦？」，他说他五岁。这个小朋友，非常喜欢我，抱着我呀，非要学我配音。我说「好好好，别着急啊」，我当时我就说了一句：「我是超威蓝猫！」，他也跟着我学了一句，学得非常像！}}
然后介绍他回到长沙以后收到了吴克{{efn|1=由於當時葛平表情誇張，故此人被稱爲“光頭吳克”，被虛構爲“葛平的[[炮友|炮友]]”}}和他爸爸寄来的信，信中说蓝猫淘气三千问给全中国的小朋友们做了一件大好事，因为动画里不仅得到了娱乐，同时又得到了知识。最后葛平指出：从小吴克的身上可以看到全中国的小朋友都很喜欢蓝猫，这就叫“看蓝猫，学蓝猫，我有知识我自豪！”{{efn|1=「我有知识我自豪」被恶搞成谐音：我有姿勢我自豪}}，最后表明其不会辜负众人的期望，公司会将《蓝猫淘气三千问》做的越来越精彩、越来越好看。

但是，由于在该两段视频中，葛平的表情夸张，加上视频内容与其他《[[蓝猫淘气三千问|蓝猫淘气三千问]]》的配音演员的自我介绍风格迥异，导致网友产生了大量恶搞[[音MAD|音MAD]]，在中国大陆著名弹幕网站[[ACFUN|ACFUN]]某部经过剪辑的视频中，一位网友将葛平所说过“葛”字和“炮”字组合在了一起，变成了“葛炮”，视频知名度提高以后，葛平被网友附上了“葛炮”的称谓，使葛平在中国网民当中人气大增。

之后，葛平在視頻中的聲音或影像常被网友用來重新剪辑合成，用於制作[[鬼畜_(网络视频)|鬼畜]]或歌曲演唱等类别的[[音MAD|音MAD]]视频，在中国大陆被称为“鬼畜全明星”之一，也是著名的音MAD“葛平系列”。

又因其在2011年7月15日于湖南长沙的爱好者见面会中，对观众友好的态度和称呼大家为“[[基友|基友]]”的发言使得一些年轻人关注<ref name="长沙见面会">{{Cite web |url=http://www.acfun.tv/v/ac218380/ |title=葛平长沙见面会视频 |accessdate=2012-02-21 |archiveurl=https://web.archive.org/web/20110920225402/http://www.acfun.tv/v/ac218380/ |archivedate=2011-09-20 |deadurl=yes }}</ref>，同年的10月1日至10月3日，在[[太原|太原]]的见面会中，其表示对用自己声音进行[[二次创作|二次创作]]的作品持宽容态度，在见面会签名时也多满足要求，甚至不吝写下“葛炮”等各种带有恶搞意味的留言和签名，赢得了网友的尊重和喜爱<ref name="太原见面会">[http://www.tudou.com/programs/view/Y775FPISk5w/ 葛平太原见面会视频]</ref>。

在2013年11月10日江苏省南京市的第二次超级弹幕大战见面会（次元突破）中，葛平现场演唱网友使用其音源[[KUSO|KUSO]]制作的[[音MAD|音MAD]]歌曲《[[循环_(歌曲)|循环]]》使观众吃惊，台下很明显可以听到尖叫声，网友称这又是一个“官方逼死同人”系列。

故此一些粉丝提议认为不应再以缺乏尊重的“葛炮”称呼葛平，而是应尊称其为“葛叔”等，甚至还有“中国第一配音演员”、“天朝第一歌姬”等讚美。
