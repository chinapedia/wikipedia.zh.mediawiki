'''葛西站'''（{{jpn|j=葛西駅|hg=かさいえき|rm=Kasai eki|f=y}}）是位於[[日本|日本]][[東京都|東京都]][[江戶川區|江戶川區]]{{link-ja|中葛西|中葛西}}五丁目，屬於[[東京地下鐵|東京地下鐵]][[東西線_(東京地下鐵)|東西線]]的[[鐵路車站|鐵路車站]]。[[車站編號|車站編號]]是'''T 17'''。

此站是距離{{link-ja|地下鐵博物館|地下鉄博物館}}最近的車站，因此設有副站名「地下鐵博物館前」。另外，月台側面牆壁中央也有大型地下鐵博物館招牌裝飾。

== 歷史 ==
* 1969年（[[昭和|昭和]]44年）3月29日：開業。
* 1986年（昭和61年）7月12日：地下鐵博物館開館。
* 1993年（[[平成|平成]]5年）
** 8月1日：增設位於2號月台向浦安方向的地下鐵博物館口。
** 11月4日：引入繼續定期券售票機<ref>'94営団地下鉄ハンドブック</ref><ref>與[[上野站|上野站]]、[[霞關站_(東京都)|霞關站]]、[[銀座站_(日本)|銀座站]]、[[新橋站_(日本)|新橋站]]、[[秋葉原站|秋葉原站]]、[[御茶之水站|御茶之水站]]同時導入。加上翌年1994年導入的[[後樂園站|後樂園站]]，是營團少數設置繼續定期券售票機的車站。</ref>。
* 2001年（平成13年）
** 1月1日：閘口－中二樓設置2部扶手電梯，中二樓－月台設置1部扶手電梯，合計增設3部。
** 3月8日：新設LED式月台目的地導覽顯示機。
* 2004年（平成16年）4月1日：[[帝都高速度交通營團|營團地下鐵]]民營化，東京地下鐵繼承本車站。
* 2008年（平成20年）4月1日：葛西站前地下自行停車場完工。葛西土地區劃整理組合寄贈南口風車「隨風」。

=== 站名由來 ===
來自於當地地名{{link-ja|葛西|葛西}}。

== 車站結構 ==
2面2線相對式月台的高架站。

本站是中央有2條通過線（本線），外側有2條副本線、月台面對副本線的[[側式月台|側式月台]]2面4線[[高架車站|高架車站]]。各站停車列車會在本站待避讓快速通過。浦安側本線有緊急用的橫渡線，可讓2號線月台列車折返前往西船橋方向。

剪票口有2處，一是高架下的中央剪票口，另一個是連接車站南側天橋與2號線月台浦安側的'''地下鐵博物館口'''。該剪票口在晚上11點以後關閉。

廁所在中央剪票口內。

== 使用情況 ==
2017年度1日平均上下車人次為'''106,899人'''<ref>[http://www.tokyometro.jp/corporate/enterprise/passenger_rail/transportation/passengers/index.html 各駅の乗降人員ランキング] - 東京メトロ</ref>，在東京地下鐵全部130個車站當中排名第35位<!--他鉄道との直結連絡駅及び共用している駅の乗降人員は順位から除いております-->。單一路線車站中僅次於[[東陽町站|東陽町站]]位居第2位。

由於靠近都心，加上環境良好，使用人數從1974年（昭和49年）的30,000人次急速增至1978年（昭和53年）的60,850人次。1979年10月1日西葛西站開業後曾暫時減少至53,000人，但在旺盛的住宅需求下，1989年（平成1年度）使用人數突破70,000人次<ref name="eidan" />。現在持續穩定上升。

== 巴士路線 ==
