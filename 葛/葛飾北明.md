'''葛飾北明'''（{{lang-ja|葛飾 北明}}，生歿時間不明）是[[日本|日本]][[江戶時代|江戶時代]]的女性[[浮世繪|浮世繪]]畫家。

== 生平　==
個人本名為井上政女，為浮世繪師[[葛飾北齋|葛飾北齋]]門下的子弟之一，另有{{nihongo|九九蜃|九々蜃}}、{{nihongo|畫狂人|画狂人}}等名號。<ref name="Ne">{{cite web |url=http://www.ne.jp/asahi/kato/yoshio/aiueo-zenesi/ho-zenesi/hokumei-katusika.html |script-title=ja:ほくめい　かつしか　葛飾　北明 浮世絵文献資料館 |accessdate=2015年10月24日}}</ref>

在葛飾北齋下學習繪畫技法後，於1804至1830年期間從事手繪[[美人绘|美人畫]]，或是在通俗文學讀物如{{nihongo|《復仇奇談辛物語》|復讐奇談幸物語}}、{{nihongo|《月桂新話》|月桂新話}}裡提供插圖等事務。<ref name="Ne"/><ref>{{cite web |url=http://harp.lib.hiroshima-u.ac.jp/onomichi-u/detail/724220140205094235 |script-title=ja:翻刻『復讐/奇談]幸物語』 |work=[[尾道市立大學|尾道市立大學]] |accessdate=2015年10月24日}}</ref>
