《'''等风来'''》（{{lang|en|'''''Up in the wind '''''}}）由完美世界（北京）影视文化有限公司出品，[[滕华涛|滕华涛]]导演的一部电影，改编自[[鲍鲸鲸|鲍鲸鲸]]的同名网络小说《游记,或是指南》<ref>{{cite web|url=http://ent.163.com/13/0717/08/93VK6HPJ000300B1.html | language= | title=滕华涛与鲍鲸鲸再度联手 《等风来》11月上映|publisher=网易娱乐 |date=2013-07-17 |accessdate= }}</ref>，由[[倪妮|倪妮]]、[[井柏然|井柏然]]等主演，讲述一支前往[[尼泊尔|尼泊尔]]、名为“幸福旅行团”的队伍的故事，於2013年12月31日上映<ref name=time>{{cite web|url=http://ent.qq.com/a/20131219/015894.htm | language= | title=《等风来》调档至12.31 倪妮自曝曾在片场昏倒|publisher=腾讯娱乐 |date=2013-12-19 |accessdate= }}</ref>。

==剧情==
美食专栏作家程羽蒙（[[倪妮|倪妮]] 飾）本来要去[[意大利|意大利]]的[[托斯卡纳|托斯卡纳]]工作，美梦被击碎参加了前往尼泊尔的[[旅行团|旅行团]]。总编再赠送她一道难题﹕在这个号称[[幸福指数|幸福指数]]高的国家寻找「幸福」。百般无奈的程羽蒙踏上旅程，和她一起同团的人有被父亲切断财源、想要“寻找人生意义”的富二代王灿（[[井柏然|井柏然]] 飾）、刚失恋的大学毕业生李热血（[[刘雅瑟|刘雅瑟]] 飾）、常常家长里短话题不断的台湾大姐团和冷酷沉默、以拍照为唯一目的的摄影[[驴友|驴友]]的奇怪团友，开展令人烦心的旅程。

在尼泊尔这个被认为生活水平低，但是[[幸福指数|幸福指数]]高的国度，他们经历了很多事情；遇上导游与团友大打出手、酒店停电的恐惧、市民示威暴乱等突发事件。程羽蒙在旅途中和王燦從互相彼此看不上，發展成互相依賴的伙伴，更慢慢找回失落的自我。最后，就在尼泊尔初尝滑翔伞的一刹，她恍然大悟：当站在人生的悬崖前时，不需要着急，只需静静等，等风来，静候一个合适的时机。

==花絮==
根据编剧鲍鲸鲸透露，片中女主角程羽蒙本来要去意大利的托斯卡纳，后来被“发配”至尼泊尔是情节是参考了她自己的经历；称导演滕华涛曾对鲍鲸鲸說派她去[[马尔代夫|马尔代夫]]写剧本，結果导演最後表示马尔代夫没什么可拍的，要去尼泊尔<ref>{{cite web|url=http://news.mtime.com/2013/08/06/1515925.html | language= | title=鲍鲸鲸尼泊尔《等风来》：我等的风是结婚|publisher= Mtime时光网 |date=2013-08-06 |accessdate= }}</ref>。其中片尾当地村民追求梦想的游行，以及王灿要“战死沙场”、程羽蒙举国旗开道的情节，取材于鲍鲸鲸第三次去尼泊尔采风时碰上当地人暴乱的真实经历<ref>{{cite web|url=http://news.mtime.com/2013/12/30/1522342.html | language= | title=专访"等风来"导演滕华涛：这回要治愈都市小白领|publisher= Mtime时光网 |date=2013-12-30 |accessdate= }}</ref>。

片名《等风来》来自小说中几个人滑翔时教练的一句话：“不管你有多着急，或者你有多害怕，我们现在都不能往前冲，冲出去也没用，飞不起来的。现在的我们只需要静静地，等风来”<ref>{{cite web|url=http://news.mtime.com/2013/11/25/1520973.html | language= | title=《等风来》曝倪妮新照 都市白领徜徉尼泊尔街头|publisher= Mtime时光网 |date=2013-11-25 |accessdate= }}</ref>。

《等风来》全片4/5的戏份都会在[[尼泊尔|尼泊尔]]取景拍摄<ref>{{cite web|url=http://news.mtime.com/2013/02/25/1507361.html | language= | title=倪妮携手"失恋33天"编导 新作"等风来"3月开机|publisher= Mtime时光网 |date=2013-02-25 |accessdate= }}</ref>。

== 上映 ==
《等风来》原本定於2013年12月29日上映<ref>{{cite web |url=http://ent.sina.com.cn/m/c/2013-10-30/17434033707.shtml | title=倪妮《等风来》合作滕华涛：来治疗不开心 |publisher=网易娱乐 |date=2013-10-30}}</ref>，後推遲至12月31日公映<ref name="time"/>。12月29日本片舉行了超前点映，票房為280万元，位列該周票房榜第九名<ref>{{cite web |url=http://news.mtime.com/2013/12/31/1522401.html |title="警察故事2013"开局夺冠 贺岁档周票房冲新高 |date=2013年12月31日 |publisher=Mtime时光网}}</ref>。

由於滕华涛之前执导的电影《[[失恋33天|失恋33天]]》在2011年上映时有出色的表现，本片首周六天的排片超过9万场，场均观众为21人，收获6000万元排在单周第三名<ref>{{cite web |url= http://news.mtime.com/2014/01/07/1522663.html |title=新年首周《警察故事》连冠 《等风来》收六千万 |date=2014年1月7日 |publisher=Mtime时光网}}</ref>；次周收1390万元排在第七名，累计7673万元<ref>{{cite web |url=http://news.mtime.com/2014/01/14/1523009.html |title=《神偷奶爸2》轻取周冠军 《安德的游戏》居次 |date=2014年1月14日 |publisher=Mtime时光网}}</ref>。至2014年1月26日累计票房为8059万元<ref>{{cite web | title=中国电影报2014年1月28日新浪微博 | url=http://weibo.com/2304129841/Au5HB3bnf | date=2014-01-28}}</ref>。
