在[[数学|数学]]中，「'''等距同构'''」或稱「'''保距映射'''」（isometry），是指在[[度量|度量]]空间之中保持[[距离|距离]]不变的[[同构|同构]]关系。几何学中的对应概念是[[全等|全等]]变换。

等距同构经常用于将一个空间[[嵌入|嵌入]]到另一空间的构造中。例如，[[测度空间|测度空间]]''M''的[[完备空间|完备化]]即涉及从''M''到''M' ''的等距同构，这里''M' ''是''M''上[[柯西序列|柯西序列]]所构成的空间关于“距离为零”的[[等价关系|等价关系]]的[[商集|商集]]。这样，原空间''M''就等距同构到[[完备|完备]]的[[度量空间|度量空间]]的一个[[稠密|稠密]][[子空间|子空间]]并且通常用这一空间来指代原空间''M''。 其它的嵌入构造表明每一[[度量空间|度量空间]]都等距同构到某一[[賦範向量空間|賦範向量空間]]的一个[[闭子集|闭子集]]以及每一完备度量空间都等距同构到某一[[巴拿赫空间|巴拿赫空间]]的一个闭子集。

一个希尔伯特空间上的等距、满射的线性[[算子|算子]]被称为[[酉算子|酉算子]]。

== 定义 ==
设''X'', ''Y''是两个度量空间，其中的距离分别是''d''<sub>''X''</sub> 和''d''<sub>''Y''</sub>。一个映射''f'' : ''X'' → ''Y'' 被称为“'''保距映射'''”，如果对任意的''a'',''b'' ∈ ''X''，都有
:<math>d_Y\left(f(a),f(b)\right)=d_X(a,b)</math>

保距映射一定是单射。任意两个度量空间之间的等距同构都必然是一个拓扑[[嵌入|嵌入]]。

等距同构是一一对应的保距映射，有时也被称为全局等距同构。还有一种定义是路径等距同构，指保持所有[[曲线|曲线]]长度的映射（不一定是一一对应的）。 

如果两个度量空间之间存在一个等距同构，就称它们两个为等距同构的。所有从一个度量空间到另一个的等距同构关于映射的复合运算组成一个[[群|群]]，称为'''等距同构群'''。

== 例子 ==
*所有度量空间到自身的恒等映射都是等距同构。
*在[[欧几里得空间|欧几里得空间]]中，[[平移|平移]]变换、[[旋转|旋转]]变换、[[反射|反射]]变换以及它们的复合都是等距同构。
*[[内积空间|内积空间]]'''C'''<sup>n</sup> 上的[[线性映射|线性]]等距同构是所有的[[酉矩阵|酉变换]]<ref>{{cite book|author=张贤达|title=矩阵分析与应用|publisher=清华大学出版社|year=2008|isbn=7-302-09271-0}}，第146页</ref>。

== 线性等距同构 ==
在[[賦範向量空間|賦範向量空間]]之间可以定义线性等距同构：所有保持[[范数|范数]]的线性映射：
:<math>\|f(v)\| = \|v\|</math>
线性等距同构一定是保距映射，因此如果是满射，就是（全局）等距同构。

根据[[马祖-玉兰定理|马祖-玉兰定理]]，系数域为实数的賦範向量空間上的等距同构一定是[[仿射变换|仿射变换]]。
