'''邻苯二甲酰亚胺'''，俗称'''酞酰亚胺'''，是[[邻苯二甲酸|邻苯二甲酸]]生成的[[酰亚胺|酰亚胺]]。

== 性质 ==
白色棱状结晶。能升华。溶于碱、[[苯|苯]]、沸[[乙酸|乙酸]]，微溶于水、乙醇。

== 制取 ==
#[[苯酐|苯酐]]与[[碳酸氢铵|碳酸氢铵]]在加热的条件下反应，再经结晶、冷却固化、粉碎得到成品。
#以[[尿素|尿素]]为原料合成。

== 用途 ==
用于药物、染料、农药、橡胶助剂等物质合成的中间体，例如用于生产[[苯酞|苯酞]]、[[邻苯二甲腈|邻苯二甲腈]]、[[灭菌丹|灭菌丹]]、[[亚胺硫磷|亚胺硫磷]]、[[靛蓝|靛蓝]]等。它的钾盐[[邻苯二甲酰亚胺钾|邻苯二甲酰亚胺钾]]是[[Gabriel伯胺合成|Gabriel伯胺合成]]中的试剂。
