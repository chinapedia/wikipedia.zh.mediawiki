'''邻苯二甲酸酐'''，简称'''苯酐'''，是[[邻苯二甲酸|邻苯二甲酸]]分子内脱水形成的环状[[酸酐|酸酐]]。苯酐为白色固体，分子式C<sub>6</sub>H<sub>4</sub>(CO)<sub>2</sub>O，是[[化工|化工]]中的重要原料，尤其用于[[增塑剂|增塑剂]]的制造。

== 反应 ==
同其他酸酐类似，邻苯二甲酸酐也可以进行[[羧酸|羧酸]][[衍生物|衍生物]]的相关反应，如[[酰基|酰基]]碳上的[[亲核取代|亲核取代]]反应（水解，[[醇|醇]]解，[[氨|氨]]([[胺|胺]])解），与[[有机金属化合物|有机金属化合物]]的反应，[[还原|还原]]反应等等。

== 制取 ==
目前在[[工业|工业]]生产中有两种苯酐原料路线，即[[邻二甲苯|邻二甲苯]]法（简称邻法）和[[萘|萘]]法。

:[[File:邻二甲苯制取苯酐.png|300px]]
:[[File:萘制取苯酐.png|300px]]

== 用途 ==
苯酐是最重要的[[有机|有机]]化工原料之一，其主要[[衍生物|衍生物]]有[[邻苯二甲酸二丁酯|邻苯二甲酸二丁酯]]、二辛酯和二异丁酯等，用作[[PVC|PVC]]等的[[增塑剂|增塑剂]]（容易加工及变软）；还可用于生产不饱和[[聚酯|聚酯]]树脂、[[醇酸树脂|醇酸树脂]]、[[染料|染料]]及[[颜料|颜料]]、多种[[油漆|油漆]]、[[食品添加剂|食品添加剂]]、医药中的缓泻剂[[酚酞|酚酞]]、[[农药|农药]]中的[[亚胺硫磷|亚胺硫磷]]、[[灭草松|灭草松]]以及[[糖精钠|糖精钠]]等。
