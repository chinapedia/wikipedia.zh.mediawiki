'''蓋鹵王'''（{{bd|？||475年||CatIdx=S蓋}}）亦作'''近蓋婁王'''。[[百濟|百濟]]第21代國王。455年至475年在位。名'''慶司'''，《[[宋書|宋書]]》作'''餘慶'''，《[[日本書紀|日本書紀]]》作'''加須利君'''。455年，先代[[毗有王|毗有王]]死，他以長子身份繼位。

== 治世 ==

蓋鹵王在位期間，通好于[[中國|中國]][[南朝宋|南朝宋]]，與[[新羅|新羅]]，[[倭國|倭國]]結盟，以抗[[高句麗|高句麗]]。努力維持[[百濟|百濟]]傳統的外交政策。後來招引[[北魏|北魏]]討伐[[高句麗|高句麗]]未遂，蓋鹵王因此惹禍上身。475年，高句麗攻破[[百濟|百濟]]首都[[慰禮城|慰禮城]]，蓋鹵王被殺。

== 對南朝宋通好 ==
457年（[[大明|大明]]元年）10月，蓋鹵王接受[[南朝宋|南朝宋]]皇帝[[劉駿|劉駿]]「鎮東大將軍」的封號，與宋結好。並于471年向宋朝貢。

== 與倭國結好 ==

461年，遣弟軍君[[昆支|昆支]]往倭國為人質，與倭國通好。

== 與北魏的接觸 ==

[[百濟|百濟]]自[[近肖古王|近肖古王]]以來一直與[[南朝|南朝]]通好。蓋鹵王時，開始試圖通過與[[北魏|北魏]]的接觸來牽制北方強敵[[高句麗|高句麗]]。472年（[[北魏|北魏]][[延興|延興]]2年），蓋鹵王向北魏上表，聲稱[[高句麗|高句麗]]一直在欺壓[[百濟|百濟]]，希望北魏加以討伐。但北魏與[[南朝宋|南朝宋]]的戰事正酣，無意與當時以逐漸崛起的[[高句麗|高句麗]]爆發全面軍事對抗，僅僅派出[[邵安|邵安]]為使者，前往調查情況，並斥責高句麗。

== 與高句麗的關係（百濟的衰亡及高句麗的進攻） ==

469年，蓋鹵王認為[[百濟|百濟]]的軍事實力已足夠強大，率軍突襲[[高句麗|高句麗]]，並奪回被高句麗侵佔的[[青木嶺|青木嶺]]（今[[開城|開城]]附近）。僧人[[道琳|道琳]]在高句麗[[長壽王|長壽王]]的授意下，前往[[百濟|百濟]]會見蓋鹵王。道琳以其精湛的[[圍棋|棋]]藝成功得到了蓋鹵王寵倖。在[[道琳|道琳]]的勸說下，蓋鹵王挪用大量軍費修建[[佛寺|佛寺]]，以致於國庫空虛。道琳秘密向[[長壽王|長壽王]]報告了百濟國內的現狀。475年9月，長壽王趁機率領三萬大軍討伐百濟。蓋鹵王遣其子[[文周王|文周]]往[[新羅|新羅]]求救，新羅王[[慈悲麻立干|慈悲麻立干]]遣一萬人來支援[[百濟|百濟]]，但為時已晚。百濟首都[[慰禮城|慰禮城]]被[[高句麗|高句麗]]攻陷。蓋鹵王出奔，並在[[阿且城|阿且城]]（今[[首爾特別市|首爾特別市]][[城東區|城東區]]康壯洞）被殺死。
