'''蓋伯拉許·崔普伍德'''（{{lang-en|Guybrush Threepwood}}）是電子遊戲公司[[LucasArts|LucasArts]]發行的《[[猴島小英雄|猴島小英雄]]》冒險遊戲系列主角。

在一些中文電腦遊戲雜誌專欄或遊戲代理商多以「小蓋」簡稱之。

== 作品中的描述 ==

蓋伯拉許是一名立志成為[[海盜|海盜]]的年輕人。為了能取得海盜身分資格，某日他來到[[加勒比海|加勒比海]]裡混亂島（Melee Island）上接受測試。之後在測試過程中，蓋伯拉許認識了令他心儀的對象──混亂島女領主「[[伊蓮·瑪蕾|伊蓮·瑪蕾]]」；以及同樣想追求伊蓮、俱有不死魔力的邪惡鬼船長「[[老查克|老查克]]」。而蓋伯拉許在他往後的海盜冒險過程中，皆不時與企圖從他身上奪走伊蓮的老查克對抗。

在《猴島小英雄》系列裡蓋伯拉許常被描述成有些性格帶著孩子氣、有點靠不太住的青年，而他個人沒特別厲害拿手的專長，除了「在水中憋氣十分鐘」。蓋伯拉許在遊戲中常遇到各種荒謬無厘頭的困境，但總能幸運化險為夷，並常以一些幽默的句白來挖苦對手。

作品當中，蓋伯拉許最常用的自我介紹台詞是「我是蓋伯拉許·崔普伍德，偉大的海盜™」（I am Guybrush Threepwood, Mighty Pirate™）。

== 名稱來源 ==

此角色名稱靈感源自當時在《[[猴島的秘密|猴島的秘密]]》裡負責美工的[[史蒂夫·普賽爾|史蒂夫·普賽爾]]。起初在蓋伯拉許尚未被正式命名前，在電腦上關於他的繪製草圖文件是被臨時取了一個名為「小夥子」（Guy）的檔案名稱，之後史蒂夫·普賽爾想到這些繪製文件是由[[美商藝電|美商藝電]]發行的《[[Deluxe_Paint|Deluxe Paint]]》[[點陣圖圖像編輯器|點陣圖圖像編輯器]]所完成；並且是屬於「筆刷」（Brush）形式繪製的[[精靈_(計算機圖形學)|圖形檔案]]，之後便將「Guy」與「Brush」合併成「GuyBrush」（蓋伯拉許）做為此遊戲人物名稱，並將此繪製存檔重新命名為「guybrush.bbm」。<ref name="FAQ">{{Cite web |url=http://scummbar.com/resources/faq/ |title=FAQ |publisher=ScummBar |accessdate=2014年2月14日}}</ref>

而「崔普伍德」（Threepwood）的姓氏則是後續引用自小說家[[佩勒姆·格倫維爾·伍德豪斯|佩勒姆·格倫維爾·伍德豪斯]]的著作《Blandings Castle and Elsewhere》裡登場的人物姓氏。<ref name="FAQ" />

== 配音 ==

從1997年[[LucasArts|LucasArts]]發行《猴島小英雄》系列裡首部備有角色配音的作品《[[猴島的詛咒|猴島的詛咒]]》起，往後推出的作品中關於蓋伯拉許的配音部份；皆全由[[多米尼克·阿曼托|多米尼克·阿曼托]]負責擔任。

== 反應 ==

在一些期刊報導中；蓋伯拉許於電子月刊[[Game_Informer|Game Informer]]的2011年史上前50位電子遊戲人物排行裡，列為第38名。<ref>{{Cite web |url=http://www.gameinformer.com/b/news/archive/2011/02/16/guinness-names-top-50-video-game-characters-of-all-time.aspx |title=Guinness Names Top 50 Video Game Characters Of All Time |publisher=[[Game_Informer|Game Informer]] |author=Jeff Marchiafava |date=2011年2月11日 |accessdate=2014年2月14日}}</ref>《[[帝國雜誌|帝國雜誌]]》則把蓋伯拉許列為電子遊戲史上最佳前50位角色第7名，並評析「蓋伯拉許身上一無是處的缺點，反而成為他的魅力所在」以及「電子冒險遊戲界中最叫人喜愛的人物」。<ref>{{Cite web |url=http://www.empireonline.com/features/50-greatest-video-game-characters/default.asp?film=7 |title=The 50 Greatest Video Game Characters 07：Guybrush Threepwood |publisher=[[帝國雜誌|帝國雜誌]] |accessdate=2014年2月14日}}</ref>報刊《[[觀察家報|觀察家報]]》則將蓋伯拉許評列在電子遊戲界中前10大角色之一、且表示著「即使遊戲界日新月異，蓋伯拉許仍是個難忘的存在」。<ref>{{Cite web |url=http://www.theguardian.com/technology/2010/aug/08/10-best-video-game-characters |title=The 10 best video-game characters |publisher=[[觀察家報|觀察家報]] |author=Tom Chatfield |date=2010年8月8日 |accessdate=2014年2月14日}}</ref>

而關於電子遊戲評論網站上；[[GamesRadar|GamesRadar]]的100大遊戲英雄人物裡，蓋伯拉許在獲得第10名成績中被評論著「他傻氣的模樣正是迷人的地方」及「不但是個不平凡的海盜、也是個與眾不同的英雄」。<ref>{{Cite web |url=http://www.gamesradar.com/top-100-video-game-heroes/ |title=100 best heroes in video games |publisher=[[GamesRadar|GamesRadar]] |author=GamesRadar Stuff |accessdate=2014年2月14日}}</ref>另外在UGO Networks的史上100大娛樂英雄角色排行上，得到第54名成績。<ref>{{Cite web |url=http://www.ugo.com/games/best-heroes-of-all-time?page=5 |title=Best Heroes of All Time |publisher=UGO |accessdate=2014年2月14日 |deadurl=yes |archiveurl=https://web.archive.org/web/20120825220429/http://www.ugo.com/games/best-heroes-of-all-time?page=5 |archivedate=2012年8月25日 }}</ref>[[IGN|IGN]]另將蓋伯拉許列為有資格躍上大銀幕作品的遊戲角色之一，其理由表示著「既然《[[神鬼奇航|神鬼奇航]]》系列電影都在市場上叫座，何不來些更俱趣味幽默感的？」<ref>{{Cite web |url=http://uk.ign.com/articles/2008/04/08/franchise-players?page=2 |title=Franchise Players: IGN presents console titans who deserve to be on the big screen and won't suck. |publisher=[[IGN|IGN]] |author=Jesse Schedeen |date=2008年4月8日 |accessdate=2014年2月14日}}</ref>而[[GameSpot|GameSpot]]網站另曾舉行過最佳遊戲角色投票，當時蓋伯拉許雖為票選名單成員之一，但在競選過程中輸給《[[汪達與巨像|汪達與巨像]]》的主角汪達。<ref>{{Cite web |url=http://uk.gamespot.com/greatest-video-game-hero/vote/index.html?battle_id=9 |title=Who's the Greatest Game Hero? |publisher=[[GameSpot|GameSpot]] |accessdate=2014年2月14日 |archiveurl=https://web.archive.org/web/20130402081226/http://uk.gamespot.com/greatest-video-game-hero/vote/index.html?battle_id=9| archivedate=2013年4月2日}}</ref>

== 引用作品 ==

[[LucasArts|LucasArts]]後續曾在自家推出的動作冒險遊戲《[[印地安納瓊斯：末日危機|印地安納瓊斯：末日危機]]》以及《[[星際大戰：原力釋放_II|星際大戰：原力釋放 II]]》裡，可以輸入特定的密碼讓玩家操控的角色外觀變成蓋伯拉許列的模樣。<ref>{{Cite web |url=http://kotaku.com/5650500/i-am-guybrush-threepwood-mighty-jedi-apprentice |title=I Am Guybrush Threepwood, Mighty... Jedi Apprentice?! |work=kotaku |author=Michael McWhertor |date=2010年9月28日 |accessdate=2014年2月14日}}</ref>另外由[[Cyberdreams|Cyberdreams]]出品的恐怖冒險遊戲《[[黑暗之蠱|黑暗之蠱]]》的一些場景裡，也有出現刻意寫著蓋伯拉許列姓氏的墓碑。
