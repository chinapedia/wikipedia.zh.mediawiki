'''蓋刺魚科'''為[[輻鰭魚綱|輻鰭魚綱]][[鱸形目|鱸形目]]的一個[[科|科]]。

== 分布 ==
本[[科|科]]魚類廣泛分布於全球各大洋的[[珊瑚礁|珊瑚礁]]區。

== 深度 ==
大部份種類分佈水域廣闊，淺從潮池，一直到30至50公尺深。

== 特徵 ==
本科魚類和[[蝴蝶魚科|蝴蝶魚科]]血緣相近。兩者最大的不同在於本科魚的前鰓蓋骨有一枚向後的尖銳硬棘。本科魚體側扁，口小，具尖銳的細齒。幼魚時期頭部不具骨質板，腹鰭基部無發達的腋鱗。在外表型態上，很多種類的背鰭和臀鰭的軟條部形成尖長的突出部份。尾鰭為圓形或彎月形。體色變化相當豐富，除了會隨著年齡成長而改變外，有些種類的雌雄性別之間也有不同的體色。

== 生態 ==
本科魚類主要生活於溫暖而乾淨的[[珊瑚礁|珊瑚礁]]淺海域，平常喜歡單獨出來游動，不過偶而也會成群或成對出現。生性機警，所以即使在大白天都只在洞穴或陰暗處附近逗留活動，一遇到驚嚇就馬上溜回洞裡。採一夫多妻制，一條公魚大概有三妻四妾，而母魚之間也可一定順位。奇妙的是公魚死後，第一順位的母魚在短短的幾個禮拜裡變性為公魚來取代原有公魚的位置。屬肉食性或雜食性，以[[藻類|藻類]]、[[海綿|海綿]]、[[海鞘|海鞘]]、[[珊瑚蟲|珊瑚蟲]]等為食。有時則成群的在水層中覓食[[浮游生物|浮游生物]]。

== 經濟利用 ==
本科魚類多為觀賞魚，較少人食用。
