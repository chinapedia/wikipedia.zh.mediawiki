'''嘉兴县'''，中国旧[[县|县]]名。在今[[浙江省|浙江省]]、[[江苏省|江苏省]]、[[上海市|上海市]]交界处。至近代为[[嘉兴市|嘉兴市]]前身。

秦汉时为[[由拳县|由拳县]]<ref name="a1">《[[元和郡县图志|元和郡县图志]]·卷第二十五》◎江南道一……苏州、吴郡。紧。……嘉兴县，望。北至州一百四十七里。本春秋时长水县，秦为由拳县，汉因之。吴时有嘉禾生，改名禾兴县，后以孙皓父名，改为嘉兴县也。海盐县……华亭县，上。西至州二百七里。天宝十年，吴郡太守赵居贞奏割昆山、嘉兴、海盐三县置。华亭谷……</ref><ref name="a2">《[[旧唐书|旧唐书]]·卷四十·志第二十》◎地理三……苏州，上……（武德七年）分置嘉兴县。八年，废嘉兴入吴县。九年，罢都督。贞观八年，复置嘉兴县。领吴城、昆山、嘉兴、常熟四县。……嘉兴，汉由拳县，属会稽郡。吴改嘉兴，隋废。武德七年，复置，属苏州。八年，废入吴。贞观八年，复置，属苏州。昆山……</ref>。[[三国|三国]][[孫吳|东吴]]时，因有嘉禾生，改名[[禾兴县|禾兴县]]。后[[避讳|避讳]][[孙皓|孙皓]]父亲[[孫和|孙和]]之名，改名嘉兴县<ref name="a1"/>。[[隋朝|隋朝]]时废县，[[唐朝|唐朝]][[武德|武德]]七年（624年），置嘉兴县，次年并入[[吴县|吴县]]。[[貞觀_(唐朝)|贞观]]八年（634年），复置<ref name="a2"/>。有盐官，属[[苏州_(隋朝)|苏州]]、[[吴郡|吴郡]]，为[[望县|望县]]<ref name="a3">《[[新唐书|新唐书]]·卷四十一·志第三十一》◎地理五……苏州、吴郡，雄……吴，望……嘉兴，望。武德七年置，八年省入吴，贞观八年复置。有盐官。昆山……华亭。上。天宝十载析嘉兴置。湖州、吴兴郡……</ref>。[[天宝_(唐朝)|天宝]]十年（751年），吴郡[[太守|太守]][[赵居贞|赵居贞]]奏请以[[昆山县_(南梁)|昆山县]]{{efn|《新唐书》记<ref name="a3"/>“华亭。上。天宝十载析嘉兴置。”未提及[[昆山县_(南梁)|昆山县]]、[[海盐县|海盐县]]。}}、嘉兴县、[[海盐县|海盐县]]置[[华亭县_(唐朝)|华亭县]]<ref name="a1"/>。[[五代|五代]]时为[[吴越国|吴越国]][[開元府|开元府]]府治。

[[北宋|北宋]]时，属[[秀州|秀州]]。后秀州升为[[嘉兴府|嘉兴府]]，仍属之，为望县<ref>《[[宋史|宋史]]·卷八十八·志第四十一》地理四……嘉兴府，本秀州，军事。政和七年，赐郡名曰嘉禾。庆元元年，以孝宗所生之地，升府。嘉定元年，升嘉兴军节度……县四：嘉兴，望。华亭……</ref>。[[元朝|元朝]]属[[嘉兴路|嘉兴路]]。[[明朝|明朝]]初，属嘉兴府。[[宣德|宣德]]五年（1430年）三月，分出[[秀水县|秀水县]]，[[两县同城而治|两县同城而治]]，同为嘉兴府治所。同时，又改魏塘镇巡检司为[[嘉善县|嘉善县]]<ref>《[[明史|明史]]·卷四十四·志第二十》地理五……嘉兴府，元嘉兴路，属江浙行省。太祖丙午年十一月为府，直隶京师。十四年十一月改隶浙江。领县七。西南距布政司百七十五里。嘉兴，倚。南有南湖，亦曰鸳鸯湖，流合运河。又有长水塘，西南接海宁，东北接海盐县界。又东有双溪，东出为华亭塘，南直松江府之漕舟，由此入运河。秀水，倚。宣德五年三月析嘉兴县地置。西有运河，北经闻家湖，达南直吴江县之运河。东北有杉青闸、又有王江泾二巡检司。嘉善，府东。本嘉兴县魏塘镇巡检司，宣德五年三月改为县。南有华亭塘河，东有魏塘河，东北有清风泾，皆流合焉。西北有分湖，与南直吴江县分界。又北有章练塘水，亦流合华亭塘河，达华亭县之泖湖。东北有风泾、西北有陶庄二巡检司，废。崇德，府西南……</ref>。清代仍属嘉兴府，考评：[[衝繁疲難|冲繁疲难]]<ref>《[[清史稿|清史稿]]·卷六十五·志四十》◎地理十二……嘉兴府：冲，繁，疲，难……领县七。嘉兴，冲，繁，疲，难。倚。府境之水二派，曰武林、天目，而天目派由石门、秀水入运，则合武林为一。长水塘南自桐乡、海盐界入，合练浦塘。海盐塘东南自其县入，并汇於南湖，一名鸳鸯湖，东南接滮湖。六里泾承南湖水，歧为二，一魏塘，一汉塘。合王庙、空庙、众欢诸塘，左出枝津为伍子塘。有王店、新丰、锺埭、新礼四镇。王店、新丰有汛。西水驿有丞。有铁路。秀水，冲，繁，难。明宣德四年析嘉兴置，同附郭……</ref>。

1949年4月下旬，[[中国人民解放军|解放军]]取得[[渡江战役|渡江战役]]胜利，国府败局已定。嘉兴县政府及相关机构则预备在解放军进攻后，在当地开展“游击战争”。5月6日凌晨，[[中国人民解放军第27军|解放军第27军]]79师236团、237团准备攻占嘉兴县城。经过外围战斗后，236团于7日黎明攻占县城。随后，79师开始接收嘉兴县政府，并清缴当地残余武装。次日，攻占嘉兴的解放军即向上海进军<ref>{{Cite web |url=http://zjnews.zjol.com.cn/05zjnews/system/2009/04/28/015468992.shtml |title=扫尽乌云是阳光——嘉兴各县解放纪实|accessdate=2018-12-10 |author= |date=2009-04-28 |work= |publisher=浙江在线新闻网站|language=Zh-hans }}</ref>。其后，历属[[嘉興專區|嘉興專區]]、[[嘉興地區|嘉興地區]]。1981年<ref>{{Cite web |url=http://www.jxzx.gov.cn/zlzx/jxzx/201011/t20101130_269281.html |title=政协嘉兴市委员会简介|accessdate=2018-12-10 |author= |date=2010-11-30 |work= |publisher=中国人民政治协商会议嘉兴市委员会网站|language=Zh-hans }}</ref>，嘉兴县撤县建市，是为县级嘉兴市，仍属嘉興地區。1983年，撤地建市，是为地级嘉兴市。原嘉兴县农村部分建立嘉兴市郊区，后更名为[[秀洲区|秀洲区]]<ref>{{Cite web |url=http://www.xiuzhou.gov.cn/zjxz/xzgk/201206/t20120609_442.html |title=秀洲概况|accessdate=2018-12-10 |author= |date=2018-05-17 |work= |publisher=秀洲区政府网站|language=Zh-hans }}</ref>。
