'''嘉諾撒書院'''（{{lang-en|Canossa College}}）是由[[嘉諾撒仁愛女修會|嘉諾撒仁愛女修會]]所創辦的英文女子中學，現為[[香港|香港]]政府津貼學校。嘉諾撒書院於1959年創校，前址位於[[筲箕灣|筲箕灣]][[巴士總站|巴士總站]]，後來在1984年遷往[[鰂魚涌|鰂魚涌]]海澤街10號。

嘉諾撒書院的學生統稱為「Canossian」，代表顏色為淺藍色。

== 校徽 ==
<gallery>
[[File:Logo_of_Canossa_College.gif|thumb]]</gallery>
嘉諾撒書院校徽上四個‘c’代表校訓中四種精神：謙、恭、仁、愛。中上方之百合花是純潔的象徵，而中間的十字架則體現基督精神。

== 學校設施 ==
*學校原址本在[[筲箕灣|筲箕灣]]，原名'''嘉諾撒修院學校 Canossian Convent Secondary School'''，並於1984年遷往[[鰂魚涌|鰂魚涌]]現址，能俯瞰維港景色。鄰近[[鰂魚涌公園|鰂魚涌公園]]、港鐵鰂魚涌站和太古坊。
*嘉諾撒書院校舍於1984年取得建築設計獎中的最高榮譽。為提供更多學習設施，學校改善工程計劃擴建新翼，並於2004年完成。
*學校由兩座建築物組成，當中包括學校禮堂，圖書館，小賣部，後院，兩個多用途球場，兩個操場和聖堂。
*現時學校有25間教室和18間特別室（2個美術室、2個電腦室、1個語言室、4個實驗室、4個多功能特別室、1個縫紉室個烹飪室、1個地理室及1個學生活動室）。

== 班級編制與教學課程 ==
在2016-2017學年開始，嘉諾撒書院中一至中六每級設四班合共24班。
中一，二年級設精英班

=== 中一及中二級必修科目 ===
*以英語授課：英國語文、數學、綜合科學、電腦認知、地理、歷史、家政、音樂、生活與社會、視覺藝術。
*以中文授課：中國語文、中國歷史、普通話、宗教教育、體育、議題探究學習、圖書館技巧(中一)、生活教育。

=== 中三級必修科目 ===
*以英語授課：英國語文、數學、物理、化學、生物、電腦認知、地理、歷史、家政、音樂。
*以中文授課：中國語文、通識教育、中國歷史、普通話、視覺藝術、宗教教育、體育、生活教育。

=== 中四至中六級 ===
==== 必修科 ====
*以英語授課：英國語文、數學、音樂。
*以中文授課：中國語文、通識教育、宗教教育、體育、生活教育。

==== 選修科 ====
*以英語授課：數學單元一及二、物理、化學、生物、資訊與通訊科技、經濟、地理、歷史、企業會計及財務概論。
*以中文授課：中國歷史、中國文學、倫理與宗教、視覺藝術。

== 出版刊物 ==
*由家長教師會出版之家長通訊及校報
*由學生編撰校刊，名為中文薰風及英文薰風（{{lang-en|The Breeze}}）內容圍繞學生校園生活
*收錄每級學生優秀中文作文之「擷芳集」
*每年由體育學會學幹事所編撰的體育學會會刊
*每年由音樂學會出版予有學習樂器的同學之「音樂學會年報」
*由天文學會編撰之「天文學會會刊」(Astrea)

== 校服 ==
嘉諾撒書院對校服和外表儀容的要求非常嚴格，訓導老師會經常提醒同學不容許任何「標奇立異」，亦會突擊檢查學生校裙是否及膝、耳環是否合規格(4mm純色)和書包有沒有花巧的配飾等等。學生的髮型亦受到規管，長頭髮的必須用黑色或藍色的辮子束起，而短頭髮的款式不能過短和太男性化，染髮也是嚴禁的。

===夏季校服===
白色連身百摺裙及配白色腰帶，粉藍色領帶，白色短襪和黑色皮鞋。校裙長度以挺直腰部靠牆直立直望時離膝不超過 3 吋為標準。

===冬季校服===
寶藍色連身背心百摺裙，配白色恤衫，藍色西裝領呔，白色長襪和黑色皮鞋。
