'''嘉兴子城'''又名'''子墙'''，位于[[中国|中国]][[浙江省|浙江省]][[嘉兴市|嘉兴市]][[南湖区|南湖区]]市中心，南北分别至府前街和中山东路，东西分别至建国南路和紫阳街，占地7.5万平方米。

子城即嘉兴最早的[[嘉兴城池|城垣]]，也是嘉兴建城的标志，始建于三国吴[[黄龙_(吴)|黄龙]]三年（231）改[[由拳县|由拳县]]为[[禾兴县|禾兴县]]时，唐末至五代时于[[子城|子城]]外拓建州城（亦称大城或[[罗城|罗城]]），子城遂成为历代行政中心（衙署）所在。按明万历《嘉兴府志》及崇祯《嘉兴县志》所载，子城周围二里十步，高厚均一丈二尺，城内的嘉兴府衙由府门（上建[[谯楼|谯楼]]，内有钟鼓）、仪门、戒石亭、正堂、后堂和知府、同知诸宅等组成<ref>[[明|明]]·刘应钶修、沈尧中纂，《万历嘉兴府志·卷二·城池》：“本府城隍肇自三国，吴筑有子城，唐乾宁中，镇将军曹信更筑其外，晋天福四年吴越王元瓘拓之。”又《卷二·公署》：“府治在旧子城内，中为正堂，而轩其前，后为穿堂，为后堂。……堂南立戒石亭，东西为吏廊，南为仪门，门左右各有翼室，直南为府门，上有谯楼，钟鼓咸具左右，为垛楼，宋景定中知军陈埙所葺也。府门左为土地祠，又东为宾馆。……府后堂为守宅，折而西出便门，又折西南为马道，马道之左为同知推官宅，右为通判宅者三，而经历、知事、照磨诸宅亦参互居之，皆有重门、厅轩、寝室、翼室，而守宅独宏，有楼有亭。……所皆子城包之，子城一名子墙，围二里十步，高与厚俱一丈二尺。”</ref><ref>[[明|明]]·汤齐修、黄承昊等纂，《崇祯嘉兴县志·卷二·城池》：“（旧城）唐僖宗文德元年吴越武肃王命制置使阮结筑，五代晋天福四年吴越王元瓘拓为州城，宋谓之军城，元谓之路城。……子城周围二里十步，高一丈二尺，厚一丈二尺，图经云三国吴黄龙时筑，有正门一楼曰丽谯。……子城者即子墙，唐之县治，宋之军州，元之路衙，今之府治墙也。”</ref>。辛亥革命后改为军队营房（俗称西大营），抗战胜利后为嘉兴青年中学驻地，中华人民共和国成立后改为浙江省荣军复员军人疗养院，即今浙江省荣军医院（或称嘉兴市第三医院）<ref>{{cite web |url=http://nhnews.zjol.com.cn/nhnews/system/2008/10/27/010723404.shtml|title=《子城》|accessdate=2013-11-24|work=|publisher=南湖新闻网}}</ref>。

现子城仅存元代初建、清光绪三十四年（1908）重修的府门谯楼及东西两侧长约百米的城墙，1990年重修，其谯楼为[[浙江省城墙列表|浙江省现存城墙]]上唯一的古城楼。府门为高5.4米，跨径4.5米的拱券式大门，谯楼面阔三间、进深五架梁，周围出回廊，重檐歇山顶，其西墙上嵌有《重修嘉兴府治碑记》<ref>{{cite web |url=http://www.zjww.gov.cn/unit/2006-02-23/53363058.shtml|title=《嘉兴子城》|accessdate=2013-11-24|work=|publisher=浙江省文物局}}</ref>。今城内为荣军医院，府门谯楼外为医院停车场。按照嘉兴城市发展规划，医院将在不久后迁出，原址将进行考古发掘并拟建文化公园<ref>{{cite web |url=http://www.zjol.com.cn/05jx/system/2006/02/11/006472054.shtml|title=《嘉兴老城将现最大城市广场 荣军医院原址建文化公园》|accessdate=2013-11-24|work=|publisher=浙江在线嘉兴频道}}</ref>。
