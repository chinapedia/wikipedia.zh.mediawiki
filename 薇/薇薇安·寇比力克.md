'''薇薇安·寇比力克'''（{{lang-en|'''Vivian Kubrick'''}}，全名'''Vivian Vanessa Kubrick'''，{{bd|1960年|8月5日|catIdx=V}}）是一位美國[[電影製作|電影製作人]]及[[作曲家|作曲家]]，她是著名導演[[史丹利·寇比力克|史丹利·寇比力克]]的女兒。

== 早年 ==
出生於[[加利福尼亞州|加利福尼亞州]]的[[洛杉磯|洛杉磯]]。父親是史丹利·寇比力克，母親是{{link-en|姬絲汀·寇比力克|Christiane Kubrick}}。她有一名姐姐叫安雅（Anya Kubrick）。

== 生涯 ==
青少年晚期時，薇薇安為BBC執導半小時的紀錄片《''The Making of "The Shining"''》<ref name="Guardian">{{cite news |language = en |author = Jon Ronson |url = http://www.theguardian.com/film/2010/aug/18/stanley-kubrick-christiane |title = After Stanley Kubrick |publisher = The Guardian |date = 2010-08-18}}</ref>，當中包含關於她父親的電影《[[闪灵_(电影)|閃靈]]》的多部錄影帶。她也拍攝過18小時的《[[金甲部隊|金甲部隊]]》製作特輯紀錄片，但最終未完成。<ref name="Guardian" />她用假名Abigail Mead為《金甲部隊》作曲，而她父親也請求她為《[[大開眼戒|大開眼戒]]》作曲，但後來因為薇薇安離家出走而放棄。史丹利·寇比力克曾經寫了一封40頁的信給她，試圖說服她回家但不成功。<ref name="Guardian" />最後由{{link-en|Jocelyn Pook|Jocelyn Pook}}代替薇薇安負責作曲。<ref>{{cite news |language = en |author = Jon Burlingame |url = http://articles.latimes.com/1999/jul/26/entertainment/ca-59637 |title = Her Avant-Garde Tunes Are Getting 'Wide' Notice |publisher = Los Angeles Times |date = 1999-07-26}}</ref>

2008年10月，薇薇安出席由Jules Verne Society贊助的《[[2001太空漫遊_(電影)|2001太空漫遊]]》40周年紀念放映，同場還有該電影的演員{{link-en|凱爾·杜拉|Keir Dullea}}、飾演猩猩首領的{{link-en|Daniel Richter|Daniel Richter (actor)}}，以及在她父親的電影《[[發條橙_(電影)|發條橙]]》（1971年）主演的[[馬康·麥杜維|馬康·麥杜維]]。她也代表她父親接受Society's Legendaire Award獎項。<ref>{{Cite web |url = http://www.laweekly.com/news/2001-a-space-odyssey-the-edison-downtown-2386359 |title = 2001: A SPACE ODYSSEY @ THE EDISON DOWNTOWN |author = Nicole Campos |date = 2008-10-13 |publisher = LA Weekly |language = en}}</ref>

== 個人生活 ==
2010年8月，她的家人宣佈薇薇安於1999年加入[[山達基教|山達基教]]（科學教），並且與家人疏遠，而她出席父親的葬禮時也有一名山達基教徒陪伴。2009年時，她的姐姐安雅因癌症病逝，但薇薇安卻沒有出席葬禮。<ref>{{Cite web |url = http://www.thedailybeast.com/articles/2010/08/26/stanley-kubricks-scientologist-daughter-vivian.html |title = Stanley Kubrick's Lost Daughter |author = Nicole Laporte |date = 2010-08-27 |publisher = The Daily Beast |language = en}}</ref>

2014年，她在[[Twitter|Twitter]]上載了一系列她自己和父親的相片。<ref>{{Cite web |url = http://dangerousminds.net/comments/stanley_kubricks_daughters_photographs_of_life_with_her_famous_father |title = STANLEY KUBRICK’S DAUGHTER’S PHOTOGRAPHS OF LIFE WITH HER FAMOUS FATHER |author = Paul Gallagher |date = 2014-03-11 |publisher = Dangerous Minds |language = en}}</ref>

像她的父親一樣，薇薇安是一名愛護動物的人，拯救過幾隻流浪犬。
