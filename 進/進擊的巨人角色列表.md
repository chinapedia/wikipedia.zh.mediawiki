: 本作男主角，與米卡莎·阿卡曼、阿爾敏·亞魯雷特是兒時玩伴，擁有強韌的精神力與非凡的行動力，對牆壁外的世界擁有比任何人都要高的憧憬，從小便立志要加入調查軍團。在目睹母親遭巨人吞食後，便立誓要驅逐所有巨人。他和兒時玩伴一起從軍，以第五名的成績從訓練軍團畢業。
: 托洛斯特區攻防戰中，他為了拯救摯友阿爾敏而遭巨人吞噬，不過並沒有死，而是變身為15公尺高的巨人，開始攻擊其他巨人，為其他訓練兵帶來生機，進而順利撤退到羅塞之牆內，但卻因此被視為怪物。之後協助駐紮軍團利用岩石將托洛斯特區大門堵上。奪回托洛斯特區後，被送上軍事法庭對其巨人化能力的威脅性進行審判，在調查軍團的解救下，加入調查軍團並接受監督與巨人化相關實驗。
: 第57次牆外調查中，所屬的特別作戰小組遭到「女巨人」的攻擊，慘遭全滅。艾連變身為巨人與女巨人戰鬥，卻也戰敗被抓走，事後被-{里}-維與米卡莎救回。雖然調查軍團合力將女巨人擒住，並揪出她的真實身份為亞妮·雷恩哈特，不過亞妮利用巨人的「硬質化」能力將自己冰封，因此眾人無法取得情報。
: 厄特加爾城一戰後，萊納及貝爾托特自稱自己是「盔甲巨人」和「超大型巨人」，隨後三人變成巨人並交戰，艾連遭到重創，和尤米爾一起被擄走。在聯合部隊前來救出艾連和尤米爾時，五年前吃掉母親的巨人再度現身在艾連眼前，在漢尼斯被巨人吃掉後引發了艾連的一項新能力，可以透過精神來操控其他巨人，此能力被萊納等人稱為「座標」。艾連用這項能力壓制了盔甲巨人，讓他的同伴得以撤離。
: 之後為了要修補瑪利亞之牆的破洞，努力練習「硬質化」能力，但無法成功。因艾連所擁有的「座標」能力而遭到王政府追捕，原本在調查軍團的計策下，將艾連與希絲特莉亞交給中央憲兵團，以此來追蹤他們，不料遭到識破而被囚禁。艾連從父親的記憶中了解到五年前他對雷斯家的所作所為、以及被他注射了能變成巨人的液體，並將他吞噬。悲憤之虞，希望能獻出自身，將「始祖巨人」歸還給希絲特莉亞，但被其拒絕。為了拯救即將被崩塌的洞穴壓跨的同伴，喝下了標有「盔甲」字樣的液體，成功使出「硬質化」能力並拯救了同伴。
: 於瑪利亞之牆奪回戰中，使用「硬質化」能力堵住瑪利亞之牆的破洞，隨後與調查軍團合力擊敗盔甲巨人，即將收拾盔甲巨人時，因超大型巨人的支援而被擊飛到城牆上，最後和阿爾敏聯手擊敗超大型巨人。瑪利亞之牆奪回戰成功之後，回到自己家中，在地下室裡得到父親的筆記和照片，得知牆內人類為可變成巨人並曾藉此統治世界的特殊人種-「艾爾廸亞人（尤彌爾之民）」，百年前被艾爾廸亞滅國的瑪雷於百年前戰勝後對留在瑪雷大陸的艾爾廸亞人實施嚴酷的隔離政策，之後瑪雷政府為了佔有帕拉廸島的石油等資源，以島上弗利玆王宣戰為藉口徵召艾爾廸亞孩童作為巨人之力的「容器」，也因此引發瑪利亞之牆陷落等事件。一年後，調查軍團將牆外巨人全數消滅，並一同觀看「大海」。

: 於851至854年間，與阿爾敏和吉克派遣的『反瑪雷義勇兵』合作，多次將入侵的瑪雷軍艦捕獲。於『雷貝里歐收容區之戰』的兩年前與韓吉、三笠和希斯特莉亞等人參加艾爾迪亞國與日出國之間的會見。在會談中，日出國代表季代敏提到艾爾迪亞必須在將軍事水平迎頭趕上世界水準的期間使用釋放牆內巨人的『地鳴』來震懾他國，而地鳴的可能條件就是始祖巨人與王血繼承者的巨人的存在，因此，王血繼承者必須在獲得巨人之力後『盡可能繁衍後代』以保有王血巨人的存在，當時的會議只有艾連挺身反對。
: 之後艾連因不明原因，於瑪雷與中東聯合的戰爭結束後，私自偽裝成艾爾迪亞戰士隊的傷兵，並化名為庫爾迦，成功潛入瑪雷國土。
: 在威利·戴巴發表宣戰宣言時變成巨人將威利殺死，隨後與調查軍團對戰槌巨人及瑪雷軍方發動攻擊。
: 在戰鬥中發現顎巨人的攻擊可以破壞硬質化水晶，利用顎巨人的牙齒將水晶化的戰槌巨人的本體破壞並吞噬脊髓液。本想接著吞噬顎巨人，但被盔甲巨人阻止，由於耗盡了體力，便與米卡莎一同返回撤退用的飛船，返回後因私自行動被逮捕，但調查兵團的新兵和弗洛克卻支持艾連的行動，並藉由向民眾泄漏艾連被關的消息來引發民眾對艾連的支持和聲援。
: 在110話用戰槌巨人的能力逃脫，並帶領包含新兵和弗洛克在內的擁護者尋找吉克.葉卡。
:在111話，葉卡兄弟和其擁護者等100多人被憲兵團團長奈爾稱為葉卡派。由於葉卡派成員大多來自調查兵團，加上調查兵團104期的人與艾連曾是密切的同袍，憲兵團對於調查兵團開始產生不信任。

: 本作女主角，身為艾連的青梅竹馬，兩人經常一起行動。沉默寡言、表情稀少的少女，雙親死後，被叶卡夫妇收养，对艾連有异於寻常的执念和保护欲。
: 第104期訓練兵團第1名畢業。擁有優秀的戰鬥技術，訓練兵時期便獲得教官的高度評價，被認為是百年難得一見的天才，伊安班長評價，一人的表現就足以抵過100名士兵{{notetag|出自第10話。}}。畢業後，為了和艾連一起行動，因此加入調查兵團。
: 混血兒，其父為阿卡曼一族的分支，其母為牆內中幾乎絕滅的純種東洋人，因父母都屬於在牆內居民中被迫害的民族，因此一家三口住在深山中躲避。擁有罕見端正秀麗之面貌和黑色的秀髮（約翰·基爾休坦第一次見到時即被她外表吸引）。在9歲時曾被人口販子擄走並親眼目睹雙親被人口販子殺害，之後被[[#艾·格|艾連]]救出，事件中因艾連的鼓勵，使在米卡莎體內的力量，在某個瞬間覺醒。也解釋為什麼米卡莎在104期之中極為突出。如她自己所述，只要艾連在她身邊，她便無所不能。一直寄居在艾連家中，且把艾連視為「重要的家人」，身上總是圍著與艾連初次見面時，艾連所贈的紅色圍巾。相當注意艾連的一舉一動，且給予母愛般的保護。
: 從初次上陣的托洛斯特區攻防戰起便展現極佳的作戰能力，往往能在關鍵時刻冷靜制敵，優異的身手也獲得長官極高的評價。唯獨在面臨可能「失去艾連」的情況時，會喪失原有的理性及判斷力，容易不顧後果衝動行事，這也導致她在戰鬥中幾度險些命喪戰場。
: 於托洛斯特區攻防戰中，被安排在後衛班與駐紮兵團的精銳部隊守護居民，並以強硬態度迫使高傲的商會會長撤離堵住城門的貨車，讓居民得以疏散。在前線支援同期訓練兵時，聽到阿爾敏說明艾連不幸戰死的消息，表面上想以堅強的實力帶領大家脫困，卻一度想放棄自己的生命，但在回想起艾連的事情後重新振作，且意識到自己的魯莽。
: 托洛斯特區奪回戰中表現活躍，並且在作戰的最後關頭成功掩護艾連完成任務。托洛斯特區奪回戰結束後，加入調查兵團。因為里維在法庭中痛打艾連，讓米卡莎對里維懷有不滿，在艾連遭女巨人劫走時，曾怪罪里維沒有保護好艾連。
: 第一次捕獲女巨人作戰失敗後，與里維聯手從女巨人嘴巴裡救出艾連，中間企圖擊殺女巨人使里維受傷，爾後責備自己讓調查兵團損失最高戰力。
: 第二次捕獲女巨人作戰，米卡莎要求艾連全力對付亞妮，並在最後關頭女巨人爬上城牆欲逃往牆外時，將女巨人從城牆上擊落。在擊落女巨人後，牆上出現了因女巨人攀爬而造成的缺口，米卡莎因而在洞口內發現了活著的巨人。（動畫版改為第一季最後在捕獲到亞妮後的深夜，牆上的裂痕造成牆壁外層剝落以至於出現缺口，但是否有人看到無從得知。）

== 帕拉迪斯 ==
由雷斯家族統治的帕拉迪島城牆政府，为艾尔迪亚帝国最后退守的疆土。百年前，由艾爾迪亞帝國145代國王（初代雷斯王）卡爾·弗利茲帶領大量的艾爾迪亞人來到島上，並建造了三座巨大的城牆作為保護。因為雷斯王希望牆內的居民們不要離開城牆，除了透過巨人之力消除了當時一同前來的艾爾迪亞人民記憶之外，也要求其他一同來到島上的其他種族必須保密，並建立嚴格的管控制度，阻止牆內的人民以各種方式探知世界的歷史。但也因為如此，在與世界隔絕的情況下，帕拉迪斯的人民基本上不知外界的情形，科技發展也被限制，因此整整落後外界一百年，但也因應巨人的威脅，發展出一套與世界各國不一樣的科技。如：立體機動裝置。

在島嶼的南端，有敵對的瑪雷國建造的小型瑪頭，專門在該處流放艾爾迪亞人（變成純潔巨人四處遊蕩），瑪雷政府稱帕拉迪島為「樂園」。島內的和平基本上持續了一百年，直到瑪雷嘗試奪取始祖巨人，由破壞西甘希納區的城牆和瑪麗亞之牆開始。除了覬覦始祖巨人之力外，島上豐富的資源，包含石化燃料，以及透過巨人之力創造的異種礦物（冰爆石、發光石），以及特殊植物黑金竹都是瑪雷甚至是其他國家垂涎的目標。

直到島上居民知道世界歷史與真相之後，帕拉迪斯才開始重新與世界交流，卻面臨危機重重的未來。儘管透過吉克·葉卡及瑪雷俘虜、還有希茲爾國得到新的科技（鐵路、飛船、新型武器、無線電），但畢竟尚沒有足夠的工業基礎維繫住這些新科技，因此新生的艾爾迪亞國也面臨了轉捩點，特別是雷貝利歐一戰之後更雪上加霜。

== 瑪雷帝国 ==
為進擊的巨人世界裡面的一個軍國主義的超級大國，同時也是一個有悠久歷史的古國。在巨人大戰之前，曾經遭到艾爾迪亞帝國侵犯而滅亡。然而瑪雷的人民在經過了約1700年的終於在百年前的巨人大戰得到機會，成功掌握九大巨人當中的七個，在145代弗利茲王的刻意主導下，並在戴巴家族和虛構英雄荷洛斯的帶領之下，終於打敗了艾爾迪亞帝國而成功復國。由於被世界上其他國家認為是成功打敗了大地惡魔-巨人的國度，也被稱為「英雄之國」。

然而，之後在一百年間，因為反過來利用巨人的力量，對外不斷進行擴張政策，因此與世界上多數的國家處在交惡狀態。對內，則對艾爾迪亞人實行高壓政策，將艾爾迪亞人集中於雷貝利歐收容區，並且不論男女老幼，統一需要配戴臂章以供識別（持有巨人之力者的臂章與其他人不同）。並將艾爾迪亞人視為武器使用，包括讓艾爾迪亞人在戰場上在最前線的位置，並且恣意將艾爾迪亞人變為純潔巨人用以對付敵人。對於反抗瑪雷帝國的艾爾迪亞人(大則像是組織崇拜始祖尤米爾的宗教，加入艾爾迪亞復權派；小至未經許可擅自離開雷貝利歐收容區，或是外出沒有配戴象徵艾爾迪亞人身分的臂章)，則是將他們帶到帕拉迪島進行流放，並變為純潔巨人。

由於過度依賴巨人之力，以致於科技發展及不上其他國家。又因為預見未來石化燃料的趨勢和潛力，因此打算侵略帕拉迪島奪取始祖巨人之力，同時藉以開採石化燃料，因而有超大型巨人摧毀瑪莉亞之牆的事件。在瑪莉亞之牆奪還戰中戰敗後，因為在帕拉迪島失去了超大型巨人和女巨人，因此和一向交惡的中東聯合軍進行了長達四年的戰爭，在此戰之後，世界各國因為瞭解到巨人的時代已經過去，同時瑪雷又因為100年來的過度擴張，因此瑪雷帝國正處在搖搖欲墜的狀態。<ref>漫畫86話至98話</ref>

==== 艾爾迪亞戰士隊 ====

== 巨人 ==
在一百多年前突然出現，將人類捕食殆盡。巨大的人型生物，身高在3至15公尺之間，外觀看似成年男性，不過都沒有生殖器官。擁有極高的體溫與堅硬的皮膚以及強韌的生命力，即使受到傷害也能迅速再生。每个巨人的体型都是有分别的，大多数巨人的体型都分为干瘦型和肥胖型，少数巨人拥有强壮的肌肉。能殺死巨人的方法，就是攻擊他們唯一的弱點「後頸<ref>長1m，寬10cm，適用於所有身高的巨人</ref>」，當後頸受到嚴重創傷，巨人就無法再生並死亡，而巨人的屍體則會蒸發消失。似乎不需要進食也能存活，所以推測捕食人類的行為只是殺戮，且對於人類以外的生物毫無興趣。巨人體內不具備消化系統，因此無法消化人體，在吞食人類後、體內的黏液會將之包裹住並吐出。必須要照射陽光才能維持活動力。<br />

牆內人類將巨人分為「一般巨人」和「奇行種」，一般巨人通常會被人類集中的地方吸引，奇行種和一般巨人不同，會採取特別行動。<br />

而後證實，所有巨人皆為「尤米爾的子民」艾爾迪亞人所變成的，將巨人的脊隨液注入體內就會變為「純潔的巨人」，即沒有任何智慧只會吃人的巨人。若吞食巨人之力擁有者的脊隨液，便可繼承其能力並變回人類。

=== 九大巨人 ===
根據古利夏·葉卡的日記中所記載，艾爾迪亞人的祖先「尤米爾·弗利茲」與「大地的惡魔」簽下契約而獲得了「巨人之力」，並以此創立艾爾迪亞帝國。在尤米爾死後，將巨人之力分成九大巨人。繼承了九大巨人之力的人，會在十三年之內死去，「十三年」是始祖尤米爾從得到巨人之力到死去的時間，越是到了時限，身體就會越衰弱，若是在無人繼承巨人之力的情況下死去，巨人之力會隨機出現在剛誕生的艾爾迪亞人嬰兒身上。九大巨人皆有各自的名字。

; {{nihongo|「始祖巨人」|始祖の巨人}}
: 具有統治所有巨人的「座標」能力。除了可以控制純潔巨人外，亦有消除記憶的能力，但沒有王室血統的人無法使用。原為弗利茲王室世代相傳，其後裔'''芙莉妲·雷斯'''被'''古利夏·葉卡'''奪走巨人之力，並傳給了兒子'''艾連·葉卡'''。

; {{nihongo|「進擊巨人」|進擊の巨人}}
: 身高為15公尺。據梟所述，該巨人不管在哪個時代都會為了自由而不斷前進，為了自由而戰，是個正好符合艾連個性的巨人之力。能進行全身硬質化，目前具體的能力不知為何。原為'''艾連·庫爾迦'''（梟）所持有，因壽命已到便讓'''古利夏·葉卡'''繼承，目前持有者為'''艾連·葉卡'''。

; {{nihongo|「超大型巨人」|超大型巨人}}
: 身高為60公尺。外表沒有皮膚覆蓋，因體型巨大所以動作相當遲緩。可以藉由燃燒肌肉組織來排放超高溫的蒸氣，使得立體機動裝置無法發揮作用，並灼傷甚甚至殺死敵人；也能利用變身時產生的能量引起大範圍的爆炸。瑪雷軍方稱此巨人為「破壞之神」，但也因為如此，力量是九大巨人當中消耗速度較快的，一旦力量耗盡，需要較長時間恢復。原為'''貝爾托特·胡佛'''所持有，後被變為純潔巨人的'''阿爾敏·亞魯雷特'''奪取巨人之力。

; {{nihongo|「盔甲巨人」|鎧の巨人}}
: 身高為15公尺。全身多處覆蓋了硬質化的組織，如同穿了盔甲一樣。在進行衝刺、擒抱或追擊時會自動脫落膝蓋背面的盔甲，藉此爆發出驚人的速度及破壞力，但也因此成了一大弱點。後來調查軍團的雷槍以及中東聯合艦隊的反巨人炮相繼面世，令盔甲巨人的銅皮鐵骨防禦能力造成打擊。目前持有者為'''萊納·布朗'''。

; {{nihongo|「女巨人」|女型の巨人}}
: 身高為14公尺。與其它巨人相比多了形似女性的乳房，具備高機動力與持續力，能將身體部份位置暫時硬質化。在身受多處創傷時可以決定再生的順序，也能發出吼叫吸引純潔巨人。目前的持有者為'''亞妮·雷恩哈特'''。

; {{nihongo|「野獸巨人」|獣の巨人}}

; {{nihongo|「顎巨人」|顎の巨人}}
: 體型小但動作靈敏，臉部皮膚與手指部分覆蓋了硬質化，可以抵擋子彈砲火攻擊，同時也具備有強大的殺傷力。使用銳利的爪和下顎當作武器，其銳利程度高於硬質化能力，是目前已知唯一可以破壞水晶化的巨人。原為'''馬賽·賈利亞德'''所持有，被無意識的'''尤米爾'''吞噬，巨人之力被奪走，後來歸還給瑪雷政府。目前的持有者為'''波爾柯·賈利亞德'''。

; {{nihongo|「戰槌巨人」|戦槌の巨人}}
: 能夠利用硬質化能力製作出骨頭類的物質，可以製造出地刺以及戰錘，或是任何想像的到的物品（如十字弓），殺傷力非常強大，曾一擊就將硬質化之後的進擊巨人錘爛。具有語言能力，與其他八大巨人不同，本體不需在巨人的後頸，而是可以躲藏於安全地方並使用水晶化保護本體，再透過經由水晶外部，以神經連結的方式操作巨人。原持有者為'''戴巴家族'''族長比利·戴巴之妹，之後被艾連利用顎巨人的咬合力破壞其水晶化防禦而遭到吞噬，目前的持有者為'''艾連·葉卡'''。
