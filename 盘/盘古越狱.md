'''盘古越狱'''为盘古越狱团队开发的一款[[iOS越狱|iOS越狱]]工具。支持使用在[[Microsoft_Windows|Microsoft Windows]]和[[OS_X|OS X]]中，但不支持[[Linux|Linux]]。适用于所有[[iOS|iOS]] 7.1到9.1的iOS设备<ref>{{cite web|url=http://pangu.io/|accessdate=2014-07-20|title=官方网站}}</ref>。

== Pangu越狱工具 ==
{{UTC time|中国标准时间|+8}}
盘古团队首次于2014年6月23日中午在官方微博上发布了盘古越狱最初版的截图<ref>{{cite web|author=盘古团队|title=PanguTeam的微博_新浪微博|url=http://weibo.com/p/1005055180829008/|publisher=新浪微博|accessdate=2014-07-22}}</ref>。次日，盘古团队正式发布盘古越狱1.0版，但该版本只适用于[[Microsoft_Windows|Microsoft Windows]]。随后，越狱开发者iH8sn0w在[[Twitter|Twitter]]上表态：“除了带有盗版商店和企业认证之外，盘古越狱并没有任何恶意软件”<ref name=中关村>{{cite web|author=张志成|title=国人制造 iOS 7.1.1完美越狱工具发布|url=http://pad.zol.com.cn/462/4625472.html|publisher=中关村在线|accessdate=2014-07-22}}</ref>。而另一越狱开发者Comex表示，很高兴越狱社区出现新鲜血液<ref name = 威锋观点/>。但随即遭到i0n1c的指责。

盘古越狱是第一个由中国团队发布的越狱工具。在赢得称赞的同时也因为使用了德国越狱开发者i0n1c在培训中“教给‘他们’（指盘古团队）的漏洞”，而遭致指责<ref name=威锋观点>{{cite web|author=威锋观点|title=在盘古击破iOS越狱垄断后 我们该关注什么|url=http://www.feng.com/apple/news/2014-07-01/After_dawn_break_the_monopoly_iOS_jailbreak_what_we_concern_588405.shtml|publisher=威锋网|accessdate=2014-07-20|deadurl=yes|archiveurl=https://web.archive.org/web/20140728091326/http://www.feng.com/apple/news/2014-07-01/After_dawn_break_the_monopoly_iOS_jailbreak_what_we_concern_588405.shtml|archivedate=2014-07-28}}</ref>。

随后，盘古团队在2014年6月29日发布1.1.0更新。在新的更新中，盘古修复了一些[[bug|bug]]并优化了程序结构。随1.1.0发布的还有适用于[[OS_X|OS X]]的版本<ref>{{cite news|author=vanfam|title=盘古发越狱工具更新：已支持Mac|url=http://www.feng.com/apple/news/2014-06-29/Pangu_hair_Jailbreak_Tool_Updated_Replace_vulnerability_to_enhance_stability_588238.shtml|accessdate=2014-07-22|newspaper=威锋网|date=2014-06-29}}</ref>。由于先前的版本使用了i0n1c的漏洞，在新版本中盘古更换了另外一个漏洞，引发了更大的争议。

2014年7月17日，盘古团队成员安全大会上首度公开越狱细节，同时公布了越狱成功的数据。在从发布开始的18天内，共有300万台iOS设备通过盘古越狱<ref>{{cite web|author=水木之向|title=盘古团队成员安全大会上首度公开越狱细节|url=http://www.feng.com/apple/news/2014-07-17/Pangu_team_members_on_the_first_public_jailbreak_security_conference_details_589554.shtml|publisher=威锋网|accessdate=2014-07-24}}</ref>。

=== 争议与指责 ===
在盘古越狱首次发布、大多数中国国外的越狱开发者对其表示欢迎时，盘古越狱官方网站上写到的“Thanks for infoleak bug from @i0n1c training.（感谢i0n1c培训时提供的infoleak漏洞）”引起了德国越狱开发者i0n1c的不满<ref name = 威锋观点/>。i0n1c因发布了[[iOS|iOS]] 4系列的完美越狱而闻名，他每年会参加很多安全会议，且都会办几次收费的培训。而在培训过程中，i0n1c以教学为目的使用了infoleak（直译“信息泄露”）漏洞，而学员并没有被要求签署保密协议。因此引发i0n1c的不满，并在Twitter上称“盘古是贼”<ref name = 争议>{{cite web|author=Chinaz|title=盘古越狱团队回应iOS 7.1.x完美越狱工具质疑|url=http://www.chinaz.com/news/2014/0625/357058.shtml|publisher=Chinaz|accessdate=2014-07-28}}</ref>。
盘古团队成员Daniel_K4在[[微博|微博]]上解释，近两年参加过i0n1c培训的人都知道infoleak漏洞，也没被要求签署保密协议，并且这次越狱一共使用了infoleak、[[内核|内核]]、[[代码签名|代码签名]]绕过、[[沙盒_(计算机安全)|沙盒]]绕过等漏洞，除了infoleak是来自于i0n1c的培训材料外，其他的内核和代码签名漏洞都是团队成员自己发现的<ref name = 威锋观点/>。而盘古也试图与i0n1c沟通，用自己发现的一个漏洞“赔偿”i0n1c，但没有得到回复。<ref>{{cite web|author=Daniel_K4|title=致不明真相的观众|url=http://www.weibo.com/p/1001603725265959310530|accessdate=2014-11-09}}</ref>之后，在1.1.0版本中，盘古团队使用了另外一个漏洞替换掉了原先使用的i0n1c培训时使用的漏洞。此举再次遭到指责。i0n1c等人指责盘古将原本可以用于iOS 8越狱的漏洞用在这样一个“不重要”的更新中。

盘古越狱和PP助手推出的iOS 9.2-9.3.3 的越狱工具，在越狱时需要用户输入Apple ID 帐号和密码，引发质疑<ref>{{cite web|author=黄晓闷|title=国外网友抱怨：盘古越狱工具擅闯私人帐号|url=http://www.feng.com/iPhone/news/2016-07-31/Foreign-friends-complain-pangu-jailbreak-tool-at-a-private-account_653217.shtml|date=2016-07-31}}</ref>。

=== Pangu iOS 8 越狱工具 ===
2014年10月22日，盘古越狱团队宣布已成功越狱[[iOS_8|iOS 8]]，并放出越狱工具。最初的1.0版越狱工具并未集成Cydia，因为iOS 8系统的诸多改变，Cydia以及依赖Substrate框架的许多插件都无法在iOS8下使用。

盘古越狱8支持任何能运行iOS8的机型（iPod Touch 5、iPhone 4S、iPhone 5/5C/5S、iPhone 6/6 Plus、iPad Mini/2/3以及iPad/iPad Air/Air 2）。<ref>{{cite web|title=PP越狱助手率先支持盘古iOS8.1完美越狱|url=http://games.qq.com/a/20141023/027711.htm|publisher=腾讯游戏|accessdate=2014-11-09}}</ref><ref>{{cite web|title=iOS8.1完美越狱来了！还是盘古首发iOS8.1无遗憾了|url=http://www.25pp.com/news/news_66480.html|accessdate=2014-11-09}}</ref><ref>{{cite web|title=iOS 8.1完美越狱发布！中国团队震撼世界|url=http://digi.163.com/14/1022/17/A967HMGP0016688K.html|accessdate=2014-11-09}}</ref>
