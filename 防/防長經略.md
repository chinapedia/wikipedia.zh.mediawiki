'''防長經略'''（{{Jpn|j=防長経略|hg=ぼうちょうけいりゃく|rm=BōChōkeiryaku}}）是[[天文_(後奈良天皇)|天文]]24年（1555年）10月12日至[[弘治_(後奈良天皇)|弘治]]3年（1557年）4月3日為止，[[安藝國|安藝國]]的[[戰國大名|戰國大名]][[毛利元就|毛利元就]]對[[大內氏|大內氏]]的領土[[周防國|周防國]]和[[長門國|長門國]]的侵攻作戰。

== 背景 ==
毛利元就在天文24年10月的[[嚴島之戰|嚴島之戰]]中撃破大內軍的主力[[陶晴賢|陶晴賢]]軍，於是計劃乘勢攻略周防、長門兩國。首先在10月12日把陣地從[[嚴島|嚴島]]移至安藝和周防國境之間的小方（現今[[廣島縣|廣島縣]][[大竹市|大竹市]]）並訓練作戰。

大內軍則有[[大內義長|大內義長]]和[[內藤隆世|內藤隆世]]率領3千兵力的本據地山口、山口的防衛據點‧[[椙杜隆康|椙杜隆康]]的[[蓮華山城|蓮華山城]]（現今[[山口縣|山口縣]][[岩國市|岩國市]][[周東町|周東町]]）、[[杉宗珊|杉宗珊]]、[[杉隆泰|杉隆泰]]父子的[[鞍掛山城|鞍掛山城]]（今岩國市[[玖珂町|玖珂町]]）、[[江良賢宣|江良賢宣]]、[[山崎興盛|山崎興盛]]的[[須須萬沼城|須須萬沼城]]（今[[周南市|周南市]]）、陶晴賢的居城‧由嫡男[[陶長房|陶長房]]守備的[[富田若山城|富田若山城]]（周南市）、[[右田隆量|右田隆量]]的[[右田岳城|右田岳城]]（今[[防府市|防府市]]）等城池籠城，為了撃退毛利軍而作出準備。而警戒[[石見|石見]][[三本松城|三本松城]]的[[吉見正賴|吉見正賴]]的[[野上房忠|野上房忠]]軍勢亦被配置在長門[[渡川城|渡川城]]（今[[山口市|山口市]]阿東）。

== 侵攻周防國東部 ==
=== 謀略和攻撃鞍掛城 ===
元就首先運用計略而令大內陣營內部動搖，在10月18日送出書狀而令椙杜隆康立即降伏。而隣接蓮華山城的鞍掛山城的杉隆泰亦在接報後降伏。不過這兩人在平時已經關係惡劣，椙杜隆康以杉隆泰的降伏是詐降並向元就送出証據（實際上降伏是真是假不明），因此毛利軍與隆泰的關係決裂。

10月9日，毛利軍7千人（一說為2萬人）開始攻撃，而杉軍則以2千士兵迎撃。杉軍在鞍掛城[[本丸|本丸]]佈下1千兵力，二之丸則佈下8百人，並在[[鞍掛山|鞍掛山]]東部的[[谷津原|谷津原]]和[[市頭|市頭]]各佈下4百人。杉軍相當善戰，不過在14日（或27日）因為毛利軍奇襲鞍掛城較弱的地方，於是杉氏父子和城兵1千3百人（當中有[[武士_(日本)|武士]]8百人）被討取，城池陷落。

=== 平定玖珂郡和大島郡 ===
11月（因為改元而變為[[弘治_(後奈良天皇)|弘治]]元年），毛利方的[[村上水軍|村上水軍]]進攻[[大島郡|大島郡]]的[[宇賀島|宇賀島]]（現今[[周防大島町|周防大島町]]的[[浮島_(山口縣)|浮島]]）並討伐大內方的宇賀島水軍，令到宇賀島一時間成為無人島，因為被認為是徹底掃除了當地勢力。直到弘治2年（1556年）年初，在[[玖珂郡|玖珂郡]]的[[地侍|地侍]]已經多數從屬於[[毛利氏|毛利氏]]，不過[[山代地方|山代地方]]的鄉村（[[周防山代一揆|周防山代一揆]]）在[[成君寺|成君寺]]（現今[[岩國市|岩國市]][[本鄉町|本鄉町]]）的山城中籠城並抵抗毛利軍。毛利軍從平定山代地方的[[坂元祐|坂元祐]]（坂新五右衛門）的[[高森城|高森城]]（今岩國市[[美和町|美和町]]）進攻[[成君寺城|成君寺城]]，於是在2月12日攻下城池。而同月18日亦在[[三瀨川|三瀨川]]（今岩國市[[周東町|周東町]]）撃退[[大內義長|大內義長]]軍。

此時，元就為了確保大內方失去控制的[[石見銀山|石見銀山]]而與開始行動的[[尼子氏|尼子氏]]對抗，令[[吉川元春|吉川元春]]率領的軍勢前往[[石見國|石見國]]。而為了防止[[大友氏|大友氏]]介入，亦派遣[[小寺元武|小寺元武]]前往大友的領地府內。因為這次交渉，元就承認[[九州|九州]]的大內領地由大友氏平定，而條件就是不要干涉[[周防|周防]]和[[長門|長門]]的領地（時期有諸種説法）。而且在同年秋天與[[肥前國|肥前國]]的[[龍造寺隆信|龍造寺隆信]]結盟，以便有可能與大友氏戰鬥時能出手。

== 侵攻周防國西部 ==
=== 攻略須須萬沼城 ===
[[File:Susuma_numa_castle_02.JPG|thumb]]是[[丘陵地|丘陵地]]）和毛利軍佈陣的[[道德山|道德山]]（背後）]]
在安撫佔領到的[[玖珂郡|玖珂郡]]和再次編成戰力後，把本陣移到岩國的[[永興寺|永興寺]]的毛利軍以[[都濃郡|都濃郡]]的[[須須萬沼城|須須萬沼城]]為目標。4月20日，[[小早川隆景|小早川隆景]]率領軍勢5千人進攻，不過被[[沼城|沼城]]城主[[山崎興盛|山崎興盛]]和[[大內氏|大內氏]]的援軍[[江良賢宣|江良賢宣]]撃退。沼城在三方都被[[沼澤|沼澤]]圍著，而守城方則用[[堰|堰]]把城池附近的[[小辻川|小辻川]]截著來預防水攻。籠城的兵力一說是3千人，加上玖珂郡的殘兵總共有1萬人。9月22日，[[毛利隆元|毛利隆元]]率兵再度攻撃城池，不過仍然失敗。

弘治3年（1557年）2月，元就自身率領1萬餘軍勢再次進攻沼城。元就把本陣建在沼城背後（北側）的[[綠山|綠山]]和道德山，隆元和隆景則分別在東側的[[權現山|權現山]]和南側的[[日隈山|日隈山]]佈陣，並在綠山西側的熊之尾（{{lang|ja|熊ノ尾}}）設軍以防敵方從山口前來的增援。2月19日，攻擊開始，對著毛利軍的攻撃，城兵頑強地抵抗，不過因為在3月2日早上的總攻撃，毛利軍在沼地埋下竹束和筵，於是成功攻入城中，籠城的1千5百餘男女（一說是3千人）被慘殺，此時毛利軍初次使用[[火繩銃|火繩銃]]進行戰鬥。因為毛利軍的猛攻，江良賢宣降伏，之後興盛亦從沼城中開城投降，元就希望興盛可以仕於毛利家，不過興盛拒絕並自殺。

（須須萬地區有關於沼城的戰鬥「{{lang|ja|沼を渉る女}}」的故事流傳。在沼城籠城的山崎興盛的兒子[[山崎隆次|山崎隆次]]剛剛結婚，但是在籠城前就被迫與妻子離別。不過因為妻子太掛念丈夫，於是在晩上唱著「{{lang|ja|恋う人は沼の彼方よ　濡れぬれて　わたるわれをば　とがめ給うな}}」並渡過沼的淺瀨。而因為這個女子渡過沼時被毛利軍看見，因此淺瀨的位置就被毛利軍得知並能攻入城內。）

=== 大內氏內部崩壞 ===
[[File:Daisennbou_Hofu_Tenmangu.JPG|thumb]]
就在元就侵攻周防期間，大內氏的家臣團內部亦開始崩壞。[[陶氏|陶氏]]的本據地[[富田若山城|富田若山城]]由[[陶晴賢|陶晴賢]]的嫡男[[陶長房|陶長房]]和弟弟[[陶貞明|陶貞明]]等人守備，而從[[嚴島|嚴島]]逃出的[[石見國|石見國]][[守護代|守護代]][[問田隆盛|問田隆盛]]亦滞留在城中，遭到在[[大寧寺之變|大寧寺之變]]後被晴賢殺死的[[豐前國|豐前國]]守護代[[杉重矩|杉重矩]]的遺兒[[杉重輔|杉重輔]]襲撃，一說重輔舉兵是與毛利串通，此戰中毛利軍亦有所攻略。長房等人措手不及，於是放棄城池並逃到[[龍文寺|龍文寺]]並在3月2日自殺（亦有說法指杉軍的襲撃和長房死亡是在[[嚴島之戰|嚴島之戰]]後的1555年10月）。

（一説為了擊破放棄若山城並進入要害龍文寺的長房，在[[氏神|氏神]]周方大明神的祭事[[念佛踊|念佛踊]]期間侵入寺內並消滅寺內的人。這被當作是龍文寺流傳下來的[[山口縣指定無形民俗文化財的「長穗念佛踊」的由來，根據這個故事，在要害龍文寺中死守的陶軍於門前的沼邊築起櫓和塀、把橋降下等來加強防備，這次籠城戰甚至長達數個月。）

雖然重輔成功消滅父親的敵人陶氏，不過對此感到憤怒的[[內藤隆世|內藤隆世]]亦開始討伐重輔。[[大內義長|大內義長]]對此仲裁來防止隆世，不過失敗，雙方的軍勢在山口[[後河原|後河原]]戰鬥，山口的街道在此戰中被燒燬，而戰敗的重輔在3月4日於防府被殺。

在3月8日攻下陶氏遺臣留下的富田若山城毛利軍於12日從富田若山城出發並沿[[山陽道|山陽道]]的[[浮野峠|浮野峠]]向防府進軍。防府[[天神山|天神山]]的[[松崎天滿宮|松崎天滿宮]]（防府天滿宮）有[[鷲頭隆政|鷲頭隆政]]和[[朝倉弘房|朝倉弘房]]率領大內軍2千人駐屯，不過被毛利的大軍完全消滅。此時毛利軍的兵力達到2萬人，因為情勢不利而向山口撤退的鷲頭、朝倉軍勢在[[佐波川|佐波川]]周邊被撃破。另一方面，[[右田岳城|右田岳城]]的[[右田隆量|右田隆量]]和[[野田長房|野田長房]]等人因為[[毛利元就|元就]]的勸告而降伏（一說是在元就進入富田若山城後送書狀後投降；或在鷲頭、朝倉軍勢敗北後被毛利軍壓迫而投降）。右田岳城由[[南方就正|南方就正]]以[[城番|城番]]身份入城，而向毛利軍投降的右田隆量則以進攻山口的先鋒身份而立下進陷[[冰上山|冰上山]]的城砦等戰功。制壓防府的元就把本陣移至松崎天滿宮的大専坊並親自擔任山口總攻撃的指揮。

== 終結 ==
[[File:Shimonoseki_Mt.Katsuyama.JPG|thumb]]跡（現今[[下關市|下關市]]）]]
剩下[[大內義長|大內義長]]和[[內藤隆世|內藤隆世]]軍勢的大內軍在[[嚴島之戰|嚴島之戰]]後才開始築城而且未完成的[[高嶺城|高嶺城]]內籠城，高嶺城南面的支城[[姬山城|姬山城]]則由[[宍道隆慶|宍道隆慶]]守備。不過之前[[杉重輔|杉重輔]]和內藤隆世的戰鬥而令到山口的街町已經化為焦土，而協助元就的[[吉見正賴|吉見正賴]]亦為了排除[[阿武郡|阿武郡]]渡川的[[野上房忠|野上房忠]]兵勢2千人而迫近[[宮野口|宮野口]]，於是義長和隆世等人就放棄與[[京都|京都]]同様沒有防衛據點的山口並逃亡到[[長門|長門]][[豐浦郡|豐浦郡]]（今[[下關市|下關市]]）的[[勝山城_(長門國)|且山城]]（勝山城），這個情報在15日被送到毛利本陣。

毛利軍開始侵攻山口，姬山城的宍道勢降伏，而毛利本隊亦為了佔領山口而行動，追殺大內義長的任務就交給[[福原貞俊|福原貞俊]]5千軍勢（被派遣的主要武將有[[志道元保|志道元保]]、[[桂元親|桂元親]]、[[赤川元保|赤川元保]]、[[粟屋元親|粟屋元親]]、[[兒玉就忠|兒玉就忠]]、[[阿曾沼廣秀|阿曾沼廣秀]]、[[堀立直正|堀立直正]]等）。而為了阻止大內義長的實家[[大友氏|大友氏]]的援軍，在陸路派出1千餘騎前往下關，更在長門的[[周防灘|周防灘]]直至[[關門海峡|關門海峡]]和[[豐前|豐前]]一帶派遣以[[乃美宗勝|乃美宗勝]]為主力的[[毛利水軍|毛利水軍]]和[[村上水軍|村上水軍]]進行海上封鎖。而在前年（1556年）11月19日，在元就送出的書狀中，[[堀立直正|堀立直正]]因為攻下[[赤間關|赤間關]]（下關的古名）的要害而得到賞賜，因此義長等人的退路一早就已經被切斷。還有響應毛利軍並進入山口的[[吉見正賴|吉見正賴]]亦因功被元就讚賞，在3月22日舉行宴會。

因為義長堅守的且山城是堅城，包圍城池的福原軍勢難以攻入城內（直到4月，二之丸才被攻陷，元就評價且山城為「在無雙的山上，無論如何都攻不下」（{{lang|ja|無双の山にて候、何とも手せめにならず候}}）），此時元就向貞俊送出計策，於是向城入射出繫有「對幫助[[陶晴賢|陶晴賢]]的謀反人隆世沒有可能得到原諒，不過對陶的傀儡義長沒有遺恨，因此可以保全性命並送返大友氏」（{{lang|ja|陶晴賢に荷担した謀反人である隆世を許すわけにはいかないが、陶の傀儡であった義長には遺恨は無いので助命して大友氏に送り返す}}）勸告文的箭矢。說服反對的義長後，隆世在4月2日自殺，義長在開城後離開且山城並進入長福院（[[功山寺|功山寺]]）。不過3日，貞俊軍勢包圍長福院並逼迫義長自殺。被計算的義長因為到了此刻已經無能為力，於是自殺。而在陶晴賢死後一直支持著陶氏的忠臣[[野上房忠|野上房忠]]亦在殺死[[陶長房|陶長房]]的嫡子[[陶鶴壽丸|鶴壽丸]]後自殺。

因此，大內氏和陶氏的正當後繼者被斷絶，而成功得到[[周防|周防]]、[[長門|長門]]的[[毛利元就|元就]]在4月23日於防府出發返回[[吉田郡山城|吉田郡山城]]。不過在11月10日，大內氏遺臣的[[草場氏|草場氏]]、[[小原氏|小原氏]]、[[河越氏|河越氏]]等人擁立[[大内義隆|義隆]]的遺兒[[問田龜鶴丸|問田龜鶴丸]]並在山口的[[障子岳城|障子岳城]]籠城，於是元就和[[毛利隆元|隆元]]再次出陣。11日，毛利方的[[內藤隆春|內藤隆春]]急襲障子岳城，在[[妙見崎山|妙見崎山]]撃破大內氏的殘軍（[[妙見崎之戰|妙見崎之戰]]），[[久芳賢直|久芳賢直]]、[[波多野勝實|波多野勝實]]、[[杉重輔|杉重輔]]的遺兒[[杉松千代丸|松千代丸]]等軍勢亦在各處舉兵，元就等人為了鎮壓而在富田（現今[[周南市|周南市]]）佈陣，於11月末把反亂勢力一掃而空，元就等人在12月26日返回吉田。而元就的「[[三子教訓狀|三子教訓狀]]」亦在富田[[勝榮寺|勝榮寺]]陣中的11月25日發出。

== 此後的影響 ==
;毛利氏
:因為吞併了大內氏所領的[[周防|周防]]、[[長門|長門]]而一氣擴大了勢力，於是成為與[[尼子氏|尼子氏]]並立的[[中國地方|中國地方]]大大名。因此與爭取[[石見銀山|石見銀山]]的尼子氏和爭取[[博多|博多]]權益的[[九州|九州]][[大友氏|大友氏]]之間正式開始對立。而完全掃除大內氏殘黨是在[[永祿|永祿]]12年（1569年）。

;大內氏
:因為義長的死亡而令大內氏正統斷絕，不過[[大內義隆|大內義隆]]的從弟[[大內輝弘|大內輝弘]]（[[大內義興|大內義興]]的弟弟[[大內高弘|大內高弘]]的兒子）還在，輝弘在北九州於毛利與大友戰鬥中的永祿12年，得到大友氏支援而進入山口，不過敗給毛利軍而自刃（[[大內輝弘之亂|大內輝弘之亂]]）。因此，大內氏由歷史舞台中消失。

;大友氏
:[[豐後|豐後]]的[[大友義鎮|大友義鎮]]因為對實弟[[大內義長|大內義長]]見死不救而失去在[[周防國|周防國]]的影響力，不過佔領了大內氏的舊領[[豐前|豐前]]、[[築前|築前]]。在永祿2年（1559年）6月被任命為豐前國、築前國的[[守護|守護]]。一時間與[[毛利氏|毛利氏]]結盟，不過在後來與以九州博多為目標的毛利氏在北九州激戰，並策動大內輝弘侵入山口。

;尼子氏
:在[[益田藤兼|益田藤兼]]仲介下與大內氏和解並結成同盟關係的尼子氏在肅清[[新宮黨|新宮黨]]後，在軍事和尼子宗家權力得到強化後，為了侵攻[[備前|備前]]而進攻備前[[浦上氏|浦上氏]]，不過在陶晴賢戰死後，大內氏在石見銀山的勢力開始倒退，於是乘隙奪回石見銀山，並撃破進攻銀山附近的忍原的[[吉川元春|吉川元春]]、[[宍戸隆家|宍戸隆家]]軍（[[忍原崩|忍原崩]]）。毛利氏奪回石見銀山是在永祿年間的「[[雲藝和議|雲藝和議]]」後。

;陶氏
:陶氏嫡流斷絕，不過傍流的[[宇野元弘|宇野元弘]]和[[陶隆滿|陶隆滿]]成為毛利氏家臣。

;內藤氏
:[[內藤隆世|內藤隆世]]自殺，而叔父[[內藤隆春|內藤隆春]]成為新的內藤氏當主並就任長門[[守護代|守護代]]。[[內藤氏|內藤氏]]在[[江戶時代|江戶時代]]以[[長州藩|長州藩]][[藩士|藩士]]的身份存續。
