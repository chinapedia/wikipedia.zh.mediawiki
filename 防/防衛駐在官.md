'''防衛駐在官'''（{{jpn|j=防衛駐在官|hg=ぼうえいちゅうざいかん|rm=Bōei Chūzaikan}}；{{lang-en|Defense Attaché}}）是指[[日本|日本]][[自衛隊|自衛隊]]派署在[[日本駐外機構列表|駐外使館或代表處]]中的{{Link-ja|自衛官|自衛官}}，兼具有[[外交官|外交人員]]身分，相當於其他國家的[[武官|武官]]。其主要職責是擔任日本使館的國防事務{{Link-en|專員|Attaché}}<ref name="防衛省英文">{{en}}{{lang|en|{{Cite web |title=Defense Attache |work=[[日本防衛省|Ministry of Defense, Japan]] |url=http://www.mod.go.jp/e/jdf/no41/specialfeature.html |date=2013-06 |accessdate=2014-11-28}}}}</ref>，與地主國（駐在國）政府、國防相關人員和各國駐當地武官團接觸<ref name="H25hakusho">{{ja}}{{lang|ja|{{Cite web |url=http://www.clearing.mod.go.jp/hakusho_data/2013/2013/html/nc027000.html |title=平成25年防衛白書　Q&A　防衛駐在官の役割と配置について |work=[[防衛省|防衛省]] |date=2013 |accessdate=2014-11-28}}}}</ref>，並且將所得的訊息統整為[[軍事情報|軍事情報]]<ref name="情報與反情報">{{zh-hant}}{{Cite book|title=情報與反情報 |author=張殿清 |publisher=時英出版社 |location=台北 |date=2001 |accessdate=2014-11-28 |isbn=957-8890-71-0 |page=頁409}}</ref>。除此之外，駐外防衛官也會參與日本和外邦之間在國防領域上的交流，或是國際和平合作項目<ref name="防衛省英文"></ref>。

日本的防衛駐在官由[[防衛省|防衛省]]抽調人選<ref name="情報日本">{{zh-hans}}{{Cite book|title=情報日本 |author=胡平 |publisher=東方出版中心 |location=上海 |date=2008 |accessdate=2014-11-28 |isbn=978-7-80186-837-4}}</ref>{{rp|p328}}、歸[[外務省|外務省]]管轄，外派期間均穿著佩有[[日本軍銜|階級章]]的自衛隊{{Link-ja|自衛隊制服|制服 (自衛隊)|制服}}<ref name="防衛省英文"></ref>。

==沿革==
[[第二次世界大戰|第二次世界大戰]]之前的日本即已向世界各地的主要國家派遣[[日軍|日軍]]武官<ref name="防衛省英文"></ref>，但這樣的制度卻在1945年[[日本投降|日本投降]]和日軍解體時跟著中止。1954年，日本對軍事情報的相關需求隨著自衛隊和防衛廳（防衛省前身）的創立而逐漸增加，於是以[[美國|美國]]開始作為自衛官的外派地<ref name="50年">{{ja}}{{lang|ja|{{Cite web |title = 防衛駐在官発足50年 - 平成16年版防衛白書 |url = http://www.clearing.mod.go.jp/hakusho_data/2004/2004/html/1634c1.html |work=[[防衛省|防衛省]] |date=2004 |accessdate=2014-11-28}}}}</ref>。1955年8月8日，外務省事務次官門脇季光與防衛廳次長{{Link-ja|增原惠吉|増原惠吉}}之間就駐外使館中任職的自衛官、其身分定位等相關事項簽定了[[備忘錄|備忘錄]]，當中明定這類人員必須服從[[外務大臣|外務大臣]]和所駐{{Link-en|使領館館長|Head of mission|使館館長}}的指揮和監督，不能和其派遣機構——防衛廳直接通信<ref name="備忘錄">{{ja}}{{lang|ja|{{Cite web |title = 防衛駐在官に関する覚書 |url = http://www.mod.go.jp/j/approach/hyouka/seisaku/results/15/sougou/sankou/02.pdf |format=pdf |work=[[防衛省|防衛省]] |date=2004 |accessdate=2014-11-28}}}}</ref>{{rp|p3}}<ref name="情報與反情報"></ref>。

此後，防衛駐在官的運作實際受到外、防兩個部門的雙重影響，其經費和日常管理由外務省的情報調查局（1993年改組為國際情報局）負責，而人員選派和業務指導則歸防衛廳處理。1997年1月，防衛廳之下成立[[防衛省情報本部|情報本部]]、統籌自衛隊陸、海、空情報的管理，並由曾在[[日本駐華大使館|日本駐華大使館]]當防衛官的{{Link-ja|國見昌宏|國見昌宏}}出任首屆部長。該機構成立之初，防衛廳向外務省調撥了40名自衛官，分別分發到33座駐外使館去，但這些自衛官在國外集妥的情資仍需透過外務省才能轉發給防衛廳，並由情報本部的分析部詳加評估<ref name="情報日本"></ref>{{rp|p328}}。

21世紀後，日本政府以[[恐怖活動|恐怖活動]]、[[大規模殺傷性武器|大規模殺傷性武器]]擴散、區域性衝突造成國際情勢轉變和趨於不定為由，認為日本駐外使館適時蒐集國防情報、迅速將之遞交給外務省和防衛廳的必要性越來越高，而防衛駐在官作為軍事專家，對相關訊息進行搜索、剖析和整理的角色也更加吃重。2003年5月7日，防衛廳副長官[[赤城德彥|赤城德彥]]和外務副大臣{{Link-ja|矢野哲朗|矢野哲朗}}一起修改了48年未經更動的備忘錄，表示外務省在雙方合意之下，會自動將駐外防衛官呈交回國的軍事資料確實傳達給防衛廳<ref name="備忘錄"></ref>。

==派遣情形==
防衛駐在官可由[[陸上自衛隊|陸上自衛隊]]（陸自）、[[海上自衛隊|海上自衛隊]]（海自）及[[航空自衛隊|航空自衛隊]]（空自）三大部隊中選出，階級分布以一佐（同[[上校|上校]]）佔絕大多數，其次則為二佐（同[[中校|中校]]），然而駐美首席防衛官則可由將補階級者（同[[少將|少將]]）出任<ref name="防衛省英文"></ref>。

自從1954年遣員赴美起，防衛駐在官的外派總人數逐漸上增。1979年時，共22個國家的日本大使館駐有防衛官<ref>{{ja}} {{lang|ja|{{Cite web |url=http://www.clearing.mod.go.jp/hakusho_data/1980/w1980_03.html |title=昭和60年防衛白書 |publisher=[[防衛廳|防衛庁]] |date=1980 |accessdate=2014-11-28}}}}</ref>、1989年時則增加到30國<ref>{{ja}} {{lang|ja|{{Cite web |url=http://www.clearing.mod.go.jp/hakusho_data/1990/w1990_03.html |title=平成2年防衛白書 |publisher=[[防衛廳|防衛庁]] |date=1990 |accessdate=2014-11-28}}}}</ref>，至2013年1月1日時，逾49名防衛駐在官（陸自23人、海自13 人、空自13人）派附於全球36座日本大使館和2處{{Link-de|常駐代表處|Ständige Vertretung|常駐國際組織代表處}}之下<ref name="H25hakusho"/>。2010年代初期，同時派有陸、海、空三種自衛官的國家僅有美國（共6人，各2名）、[[中華人民共和國|中華人民共和國]]、[[韓國|韓國]]和[[俄羅斯|俄羅斯]]（共3人，各1名）<ref name="2010派遣狀況">{{ja}}{{lang|ja|{{Cite web |title= 防衛駐在官の派遣状況 |url=http://www.mod.go.jp/j/approach/hyouka/seisaku/results/22/cyukan/sankou/01.pdf |format=pdf |work=[[防衛省|防衛省]] |date=2010 |accessdate=2014-11-28}}}}</ref>。

2013年1月16日，在[[阿爾及利亞|阿爾及利亞]]發生了[[因阿邁納斯人質事件|因阿邁納斯人質事件]]、導致數-{}-位日本公民遇害。日本政府於事後探討了對外加派防衛人員的對應方案，並考慮將駐有防衛官的[[非洲|非洲]]國家由原本的[[埃及|埃及]]、[[蘇丹|蘇丹]]兩國調升為9國<ref>{{ja}}{{lang|ja|{{Citation |url=http://www.kantei.go.jp/jp/singi/alg_terotaiou/kensahoukokusho20130228.pdf |format=pdf |title=在アルジェリア邦人に対するテロ事件の対応に関する検証委員会検証報告書　|date=2013-02-28 |publisher=在アルジェリア邦人に対するテロ事件の対応に関する検証委員会 |accessdate=2014-11-29 |page=pp. 6-9}}}}</ref><ref>{{ja}}{{lang|ja|{{Citation |url=http://www.mod.go.jp/j/yosan/2014/yosan.pdf |title=平成26年度防衛予算概要 |work=[[防衛省|防衛省]] |date=2014 |accessdate=2014-11-28 |format=pdf |page=p. 20}}}}</ref><ref>{{ja}}{{lang|ja|{{Cite web|url=http://www.jiji.com/jc/graphics?p=ve_pol_seisaku-anpoboei20131026j-03-w340 |title=アフリカの防衛駐在官増員へ＝軍事情報収集を強化 |author= |date=2013-10-25 |work= |publisher=時事通信 |accessdate=2014-10-26 }}}}</ref>，[[防衛大臣|防衛大臣]][[小野寺五典|小野寺五典]]也在1月26日的記者會中，指出日本的防衛官集中於歐亞，在非洲和[[南美洲|南美洲]]卻很稀少<ref>{{zh-hans}}{{Cite news|url=http://www.ribenxinwen.com/html/c/201301/26-15689.html|publisher=[[日本新聞網_(亞洲通訊社)|日本新聞網]]|title=日本计划向非洲地区增派武官|date=2013-01-26|accessdate=2014-11-29|deadurl=yes|archiveurl=https://web.archive.org/web/20141214233514/http://www.ribenxinwen.com/html/c/201301/26-15689.html|archivedate=2014-12-14|df=}}</ref>，之後又在出席6月的[[日本參議院|參議院]]{{Link-ja|日本外交防衛委員會|外交防衛委員会|外交防衛委員會}}時，表示增派人手的方針是為了在防衛官數量短缺的地域滿足情報蒐集的需求<ref>{{ja}}{{lang|ja|{{Cite web|url=http://kokkai.ndl.go.jp/SENTAKU/syugiin/186/0015/18606060015008a.html |title=第１８６回国会　安全保障委員会　第８号 |date=2013-06-06 |publisher=[[日本眾議院|衆議院]] |accessdate=2014-11-29 }}}}</ref>。

同年8月，防衛省決定向非洲的7個邦交國增派防衛官<ref>{{zh-hans}}{{Cite news |title=日本拟首次向非洲7国派遣武官 |url=http://japan.people.com.cn/n/2013/0826/c35469-22697839.html |publisher=[[人民日報|人民网-日本频道]] |date=2013-08-25 |accessdate=2014-11-28}}</ref>，當中包括了阿爾及利亞、[[吉布地|吉布地]]、[[摩洛哥|摩洛哥]]、[[衣索比亞|衣索比亞]]、[[奈及利亞|奈及利亞]]、[[肯亞|肯亞]]和[[南非|南非]]<ref>{{ja}}{{lang|ja|{{Cite web |url=http://www.shugiin.go.jp/internet/itdb_kaigiroku.nsf/html/kaigiroku/001518520131031002.htm |title=第185回国会　安全保障委員会　第2号 |date=2013-10-31 |publisher=[[日本眾議院|衆議院]] |accessdate=2014-11-29 |archive-url=https://web.archive.org/web/20141219140236/http://www.shugiin.go.jp/internet/itdb_kaigiroku.nsf/html/kaigiroku/001518520131031002.htm |archive-date=2014-12-19 |dead-url=yes }}}}</ref>。2015年度的防衛省預算請求書中也提到了朝[[波蘭|波蘭]]、[[烏克蘭|烏克蘭]]及[[澳洲|澳洲]]等地調撥防衛官的加強計畫<ref>{{ja}}{{lang|ja|{{Citation |url=http://www.mod.go.jp/j/yosan/2015/gaisan.pdf |title=平成27年度防衛予算概要 |work=[[防衛省|防衛省]] |date=2014 |accessdate=2014-11-28 |format=pdf |page=p. 18}}}}</ref>。

===爭議事件===
日本自1954年設立防衛駐在官制度後，共發生了三次駐外人員遭到地主國驅離的事件<ref>{{YouTube|id=iJzN3Q_Wnkk|title=《新闻》日驻华使馆设3武官 2人曾因刺探军情被驱 }}</ref>。第一次發生於1987年8月，[[蘇聯政府|蘇聯政府]]指控[[日本駐蘇聯大使館|日本駐蘇聯大使館]]海自防衛官竹島信博，稱他與[[三菱商事|三菱商事]]駐[[莫斯科|莫斯科]]事務所次長大谷隆男從事[[間諜|間諜]]活動，將兩人[[驅逐出境|驅逐出境]]<ref>{{en}}{{lang|en|{{Cite news |url=http://articles.latimes.com/1987-08-21/news/mn-2332_1_soviet-trade-official |title=Soviets Expel Japanese Aide, Businessman : 2 Accused of Spying; Tokyo Orders Moscow Trade Official Out |publisher=[[洛杉磯時報|Los Angeles Times]] |work=[[美聯社|Associated Press]] |date=1987-08-21 |accessdate=2014-11-29}}}}</ref>。第二次是在1996年1月，由於日本駐華大使館的空自防衛官前谷憲治和[[美國駐華大使館|美國駐華大使館]]空軍副武官格迪斯（{{lang|en|Bradley David Gerdes}}）闖進軍事禁區攝影被逮<ref name="戰後日本防衛研究">{{zh-hans}}{{Cite book |title=戰後日本防衛研究 |publisher=[[上海人民出版社|上海人民出版社]] |isbn=7-208-04653-0 |author=王少普、吳寄南 |date=2003 }}</ref>{{rp|p460}}，日方及美方在事後均應中方要求召回兩人<ref>{{zh-hant}}{{Cite book|title=外電新聞的常用語 |author=王曉寒 |publisher=正中書局股份有限公司 |location=台北|isbn=957-09-1395-9 |page=頁279}}</ref>。第三次事件則為2002年，駐華海自防衛官天野寬雅於10月26日擅入[[浙江省|浙江省]][[寧波市|寧波市]]的[[解放軍海軍|解放軍海軍]]設施拍照，被發現後遭到制止、並接受中國安全官員審問，[[數位相機|數位相機]]、[[筆記本|筆記本]]、[[錄音機|錄音機]]等隨身器材均遭到沒收。11月13日，中國政府要求天野自行離華，外務省則在翌日宣布將其召回日本<ref name="情報日本"></ref>{{rp|p329}}。
