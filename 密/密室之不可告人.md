《'''密室之不可告人'''》（{{lang-en|'''Lost in Panic Room'''}}）中国大陆悬疑惊悚片，导演张番番，主演苏有朋、吴佩慈、胡明、袁成杰、杜俊泽、史光辉。<ref>{{cite web |url=http://ent.scol.com.cn/ylgg/content/2011-10/28/content_3039621.htm?node=846 |title=苏有朋来蓉宣传《密室之不可告人》 |accessdate=2011 |publisher=四川新闻网 |work= |language=zh-hans}}</ref><ref>{{cite web |url=http://ent.163.com/10/1109/18/6L2OGSTT00032DGD.html |title=《密室之不可告人》：推理渐入佳境 |accessdate=2010 |publisher=网易 |work= |language=zh-hans}}</ref><ref>{{cite web |url=http://pic.news.sohu.com/group-238672.shtml#g=238672&p=2199268 |title=吴佩慈杀人上瘾 《密室》“杀手”升级 |accessdate=2010 |publisher=搜狐网 |work= |language=zh-hans |archive-url=https://web.archive.org/web/20140502203838/http://pic.news.sohu.com/group-238672.shtml#g=238672&p=2199268 |archive-date=2014-05-02 |dead-url=yes }}</ref>其续集是《[[密室之不可靠岸|密室之不可靠岸]]》。

==剧情==
该片讲述一宗在别墅的离奇案件。
