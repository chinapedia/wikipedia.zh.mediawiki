'''密丹公寓'''（{{Lang-en|'''Midget Apartments'''}}），位于今[[中國|中國]][[上海|上海]][[武康路_(上海)|武康路]]、[[湖南路_(上海)|湖南路]]的交叉口东北角，[[开普敦公寓|开普敦公寓]]的斜对面，是一栋5层的仿生历史建筑<ref name="dd"/>。建筑由法商[[赉安|赉安洋行]]设计，于1931年建在当时的福开森路（今武康路）。1960年代初，[[孙道临|孙道临]]曾在此居住。1994年，密丹公寓入选第二批[[上海市优秀历史建筑|上海市优秀历史建筑]]。现时公寓是[[中国历史文化名街|中国历史文化名街]]——武康路历史建筑群重要的组成部分，2003年8月18日被公布为徐汇区区级文物保护单位。

==历史==
在密丹公寓于1931年建成后，这里一直为义品放款银行（Credit Foncier d'Extreme-Orient）使用。义品放款银行是一家法国、比利时合资银行<ref name="地方志">[http://www.shtong.gov.cn/newsite/node2/node2245/node75491/node75497/node75551/node75563/userobject1ai92368.html 专业志 >> 上海金融志 >> 第二篇解放前金融机构 >> 第三章外资银行 >> 第八节 意、比、荷、挪资银行] 上海市地方志办公室.</ref>。主要业务以经营房地产押款为主<ref name="地方志"/>。在抗日战争结束后，逐渐停业<ref name="地方志"/>。1960年代初，表演艺术家孙道临曾在此居住。他就是在居住于密丹公寓时期，舍近求远，在他后来的妻子[[王文娟|王文娟]]所居住的[[枕流公寓|枕流公寓]]下乘公交车去[[上海音乐厅|上海音乐厅]]，从而同王文娟相识<ref name="新民晚报">{{Cite web |title = 我的越剧人生（31） |url = http://xmwb.news365.com.cn/lz/201210/t20121006_709290.html |publisher = [[新民晚报|新民晚报]] |date = 2012-10-06 |accessdate = 2014-01-02 |deadurl = yes |archiveurl = https://web.archive.org/web/20121103091756/http://xmwb.news365.com.cn/lz/201210/t20121006_709290.html |archivedate = 2012-11-03 }}</ref>。

== 建筑特色 ==
密丹公寓占地93.33平方米<ref name="lp">{{Cite book |title = 上海市徐汇区地名志 |publisher = 上海社会科学院出版社 |pages = 第574页 | ISBN =  | accessdate = 2014-01-02 | url = }}</ref>，现代派建筑风格，钢筋混凝土结构，共五层<ref name="dd"/>。建筑平面为三角形，并使用了深灰色水泥拉毛装饰外墙面。公寓采用了平屋顶，方形窗，局部窗台一层一层挑出<ref name="ax"/>。在窗裙、檐口、阳台等处都有[[装饰风艺术|装饰艺术]]风格的装饰。而顶部的装饰图案则是[[巴洛克|巴洛克]]风格的<ref name="dd"/>。

建筑入口采用了内凹的手法，即在大门上方逐步出挑，从而使其带有了装饰艺术特征，与古建筑的券门和柱式颇为相似<ref name="ax"/>。底层是面对路口的附屋，现为fancyfruit店铺<ref name="lp"/>。

===仿生建筑===
密丹公寓除了是一座现代主义建筑外，还是一座仿生建筑，即指其在建筑形式的仿生。密丹公寓建筑立面造型对[[大象|大象]]形体进行了仿生<ref name="dd"/>。因建筑在顶部设计了一端的连续曲线山墙和在端部设计了卷涡，从而使建筑外形与大象的外形颇为相似<ref name="ax"/>。
