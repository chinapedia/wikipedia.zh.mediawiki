《'''櫻花最前線'''》（日文：'''{{lang|ja|さくら前線}}'''）是由[[大林深雪|大林深雪]]（筆名：{{lang|ja|おおばやしみゆき}}<ref>在日本發表時使用'''{{lang|ja|[[:ja:おおばやしみゆき|おおばやしみゆき]]}}'''為作者名，尖端發行中文版時使用本名'''[[大林深雪|大林深雪]]'''作為作者名。</ref>）在[[小學館|小學館]]《[[ChuChu|ChuChu]]》連載的[[少女漫畫|少女漫畫]]及[[校園漫畫|校園漫畫]]作品，在台灣由[[尖端出版|尖端出版]]代理並於《[[甜芯少女漫畫月刊|甜芯少女漫畫月刊]]》連載<ref>[http://blog.roodo.com/itiffany/archives/3551623.html 《甜芯少女漫畫月刊》8月12日正式創刊！【尖端出版新聞稿】。]</ref>。

==簡介==
女主角中原櫻是八上中學（簡稱八中<ref>中文版單行本第一冊開頭介紹。</ref>）一年級的學生，從小就喜歡青梅竹馬的松田武彥。教室裡，坐在旁邊的是矮子齊籐由紀，武彥在籃球隊的死黨。當初由紀接近櫻是因為想知道武彥喜歡什麼類型的女孩子，但慢慢發現自己喜歡櫻。兩人感情越來越好，最後相約在某星期六下午2點去約會。但是2點時等到的不是由紀本人，而是武彥的一通電話———由紀的死訊。
因為由紀的死使櫻的好友中島 美香跟櫻斷絕關係。在自責與悲傷之時，櫻上了高中，想以籃球時時刻刻想起由紀的死。然而某天，聽到了有人大叫「ゆうき(结城)」，那個與由紀(ゆき)相似的名字會如何改變櫻的高中生活?

==人物==
; {{nihongo|中原櫻|なかはら さくら}}
: 故事的女主角，從小就喜歡青梅竹馬松田武彥，但開學後開始在意同班的矮個子齊籐由紀，而喜歡上他。
; {{nihongo|松田武彥|まつだ たけひこ}}
: 小櫻的青梅竹馬，從小就喜歡小櫻。除了學業很好以外也擅長[[籃球|籃球]]，在國一的時候成為籃球社正式隊員。相當受到女生的愛慕。也叫做「阿武」．
; {{nihongo|齊籐由紀|さいとう ゆき}}
: 喜歡與小櫻鬥嘴的同班同學，但實際上很喜歡小櫻。擅長[[籃球|籃球]]，參與籃球社。在星期六與小櫻的約會途中被卡車撞死。
; {{nihongo|中島美香|なかじま みか}}
: 小櫻的好朋友，喜歡由紀。可是卻被小櫻搶走了由紀，再加上由紀在意外死了，因此與小櫻絕交。在打工時認識了一個新男朋友。
; {{nihongo|結城伸一朗|ゆうき しんいちろう}}
: 小櫻的高中同學，擅長[[籃球|籃球]]，參加高中的籃球社。在[[日文|日文]]中「結城」（{{lang|ja|ゆうき}}）與「由紀」（{{lang|ja|ゆき}}）的發音相近，故事亦以此發展。
; {{lang|ja|二谷アイコ}}
: 小櫻的高中死黨，喜歡松田武彥。
