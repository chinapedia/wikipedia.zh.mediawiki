'''櫻真耶'''（さくら　まや、{{bd|1998年|7月26日|||Sakura Maya}}），是一名[[日本|日本]]年輕的[[女性|女性]][[演歌|演歌]]歌手，出身於[[北海道|北海道]][[帶廣市|帶廣市]]，2008年以10歲年紀出道，當時憑《{{link-ja|大漁豐收祭|大漁まつり}}》打進樂壇，2009年同時獲得[[日本唱片大獎|日本唱片大獎]]<ref>[http://jacompa.or.jp/recotai_his/51.html 2009年日本唱片大獎得獎記錄。] {{webarchive|url=https://web.archive.org/web/20121103045459/http://jacompa.or.jp/recotai_his/51.html |date=2012-11-03 }}{{ja}}</ref>及[[日本有線大獎|日本有線大獎]]<ref>[http://ranking.cansystem.info/grandprix/?md=h&ts=42 2009年日本有線大獎得獎記錄。]{{ja}}</ref>的「新人獎」，成為兩獎自設立以來最年輕的得獎者。同年度亦獲邀參與[[第60回NHK紅白歌合戰|第60回NHK紅白歌合戰]]，於特別企劃項目「小紅白歌合戰」中出演。

現時於[[東京都|東京都]]內一所[[高等學校|高等學校]]就讀一年級，同時兼顧歌唱事業及學業。

== 簡介 ==
櫻真耶原名「'''草野真耶'''」，出身於北海道帯廣市，為家中次女，家庭經營割烹料理店。2歲半起開始分別學習[[小提琴|小提琴]]、[[鋼琴|鋼琴]]、[[豎琴|豎琴]]、[[日本箏|日本箏]]等樂器，亦有學習[[樂理|樂理]]、[[歌唱|歌唱]]、[[芭蕾舞|芭蕾舞]]及[[日本舞|日本舞]]。2004年參加第19屆{{link-ja|全國童謠歌唱比賽|全国童謡歌唱コンクール}}，以一曲《母親的嬰兒》（{{lang|ja|ママが赤ちゃん}}）於兒童組奪得金獎<ref>[http://www.douyou.jp/contest/winner/index.html 全國童謠歌唱比賽歷年得獎名單。]{{ja}}</ref>。2008年獲{{link-ja|日本皇冠|日本クラウン}}簽約，成為唱片公司內最年輕的演歌歌手，並在12月推出第1首個人單曲《大漁豐收祭》，三日後於東京[[台東區|台東區]]的著名遊樂場「[[淺草花屋敷|淺草花屋敷]]」內舉行出道後的首個公開演出<ref>[http://www.barks.jp/news/?id=1000045385 小学4年生の演歌歌手、さくら まや10歳です。]BARKS音樂情報站，2008年12月3日。{{ja}}</ref>。

2009年由帶廣市搬往東京都居住<ref>{{Cite web |title=神童の法則 才能はいかにして開花したか。親たちが語る「天才少年・少女」の育った環境、育成理論 第1回 演歌歌手さくらまや「大丈夫か、うちのコ?」とさえ心配した異常な集中力 |page=2 |author=片瀬京子 |date=2011-03-18 |work=現代ビジネス |publisher=講談社|url=http://gendai.ismedia.jp/articles/-/2272?page=2 |accessdate=2014-03-12}}</ref>，並由「帶廣市立帶廣東小學校」轉至「台東區立千束小學校」就讀<ref>{{cite news|title = さくらまやさん転校 東京で歌手頑張る!|date = 2009-03-06|newspaper = WEB TOKACHI－十勝毎日新聞|url = http://www.tokachi.co.jp/news/200903/20090306-0000819.php|accessdate = 2013-07-23|deadurl = yes|archiveurl = https://web.archive.org/web/20140221021851/http://www.tokachi.co.jp/news/200903/20090306-0000819.php|archivedate = 2014年2月21日|df = }}</ref>。[[暑假|暑假]]期間舉行了首次全國性巡迴宣傳活動，也開始出席各種的電視台、電台訪問及參與電視劇活動，首齣參演的電視劇為[[TBS電視台|TBS電視台]]的「[[水戶黄門|水戶黄門]]」。年尾同時獲日本唱片大獎及日本有線大獎頒發新人獎，並獲邀參與NHK紅白歌合戰。2010年被委任為帶廣觀光大使<ref>[http://www.tokachi.co.jp/news/201008/20100802-0006222.php さくらまやさん観光大使に。] {{Wayback|url=http://www.tokachi.co.jp/news/201008/20100802-0006222.php |date=20150101091131 }}十勝毎日新聞社，2010年8月2日 。{{ja}}</ref>，亦再次在日本唱片大獎中獲得「日本作曲家協會獎勵賞」<ref>[http://jacompa.or.jp/recotai_his/52.html 2010年日本唱片大獎得獎記錄。] {{webarchive|url=https://web.archive.org/web/20150101091510/http://jacompa.or.jp/recotai_his/52.html |date=2015-01-01 }}{{ja}}</ref>。

2011年參與[[朝日電視台|朝日電視台]]，由[[關8|關8]]主持的電視節目《[[關8比賽中|關8比賽中]]》中的[[卡啦OK|卡啦OK]]對決環節，創出全勝的紀錄，被冠以「卡啦OK不敗女皇」的稱號；2012年推出了特別專輯「真耶☆卡啦OK不敗女皇的歌曲」（{{lang|ja|まや☆カラ カラオケクイーンさくらまやと歌おう!!}}），把環節中所唱過的歌曲都收錄在此，並作為出道五周年的紀念專輯<ref>{{Cite news |title=【イベントレポート】無敗のカラオケクイーン：さくらまや、これぞ圧巻のパフォーマンス |newspaper=BARKS |date= 2012-12-25 |url=http://www.barks.jp/news/?id=1000085258 |accessdate=2014-03-12}}</ref>。
