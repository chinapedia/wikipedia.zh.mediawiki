'''櫻田門外之變'''是發生於[[日本|日本]][[安政|安政]]<ref>由於同年三月十八日改元「[[萬延|萬延]]」，所以也有記為「萬延元年三月三日」者。</ref>七年三月三日（1860年3月24日）的一起[[政治|政治]][[暗殺|暗殺]]事件，不滿[[幕府|幕府]][[大老|大老]]兼[[彥根藩|彥根藩]][[藩主|藩主]][[井伊直弼|井伊直弼]]的[[水戶藩|水戶藩]]激進[[浪士|浪士]]，於[[江戶城|江戶城]][[櫻田門|櫻田門]]（今[[東京都|東京都]][[千代田區|千代田區]]的[[警視廳|警視廳]]正對面）外突襲準備進城的井伊直弼隊伍，井伊直弼當場慘死。

== 背景 ==
自[[黑船來航|黑船來航]]後，主張開國的[[彥根藩|彥根藩]]主[[井伊直弼|井伊直弼]]與主張攘夷的[[水戶藩|水戶藩]]主[[德川齊昭|德川齊昭]]對立日深，之後又為了十三代[[征夷大將軍|將軍]]的繼承人問題各自形成對立派系，政爭的結果由井伊直弼勝出，[[安政|安政]]五年（1858年）井伊直弼出任[[幕府_(日本)|幕府]][[大老|大老]]，不但獨斷地與[[美國|美國]]簽訂《[[日美修好通商條約|日美修好通商條約]]》。翌年（1859年）德川齊昭發動反攻，聯合[[越前藩|越前藩]]主[[松平慶永|松平慶永]]、[[尾張藩|尾張藩]]主[[德川慶恕|德川慶恕]]及自己的兒子[[一橋家|一橋家]]當主[[一橋慶喜|一橋慶喜]]闖入[[江戶城|江戶城]]質問井伊直弼，結果因為這種過於無禮的行為反遭井伊直弼下達永久退隱的處份。之後井伊直弼決定全力鎮壓反對勢力，大舉逮捕水戶藩及攘夷派的志士，並以嚴刑論處，釀成[[安政大獄|安政大獄]]，一時間風聲鶴唳，尤其水戶[[藩士|藩士]]的積怨已瀕臨界，有些謀畫暗殺的激進藩士為免連累本藩而事先脫藩變成[[浪士|浪士]]，反撲活動暗潮洶湧。

[[水戶藩|水戶藩]]的[[高桥多一郎|高桥多一郎]]和[[金子孙二郎|金子孙二郎]]等过激派浪人脱藩和[[萨摩藩|萨摩藩]]的[[有村次左卫门|有村次左卫门]]暗地沟通，打算带领萨摩兵上京以[[孝明天皇|孝明天皇]]敕書的名义发动[[军事政变|军事政变]]，但因萨摩藩局势變化，两藩士的计划受挫。水戶的过激派决定策划袭击大老，萨摩只有有村一人参加。

== 暗殺 ==

安政七年三月三日（1860年3月24日），這天是[[上巳|上巳]]之日，也是駐居在[[江戶|江戶]]的大名固定要進城謁拜的日子，以水戶藩浪士為主的暗殺集團決定於今日起事。

天寒未明，路上積[[雪|雪]]處處，一行人饱餐一顿由[[東海道|東海道]][[品川宿|品川宿]]（今[[東京都|東京都]][[品川區|品川區]]）的旅店出發，經過[[大木戶|大木戶]]後在[[札之辻|札之辻]]轉向，途經[[網坂|網坂]]（今[[東京都|東京都]][[港區|港區]]的[[慶應義塾大學|慶應義塾大學]]附近）、[[神明坂|神明坂]]、[[中之橋|中之橋]]，接著穿過[[櫻田通|櫻田通]]，在[[愛宕山|愛宕山]]的[[愛宕神社|愛宕神社]]與次會合，向櫻田門进发。此時共計有[[水戶|水戶]]浪士17人、[[薩摩|薩摩]]浪士1人，後世合稱「'''櫻田十八士'''」。由於井伊直弼的宅邸在[[櫻田門|櫻田門]]附近，櫻田門為井伊直弼進城的必經之路，所以暗殺集團便埋伏在櫻田門附近等候。{{Who|事先曾經警告|time=2015-9-26}}过井伊家，但是直弼没有加强防备。

当天不合时令地下着大雪，能见度很差，护卫的侍从穿着蓑衣，刀藏在裡面，对袭击方来说是有利的条件。江戶幕府开幕以来在光天化日之下胆敢袭击大名座轿是想也未曾想到的事情，造成彦根藩的大意。

[[File:Arimura_Jisaemon.jpg|thumb]]
[[上午|上午]]9時許，井伊家開門，約60人的護衛隊伍圍繞著井伊直弼的大轎出來。暗殺團的森五六郎手持訴狀，偽裝成攔轎喊冤的樣子迅速接近井伊的隊伍，在與井伊家護衛糾纏之際，冷不防地拔刀砍殺井伊的護衛，其他護衛見狀蜂湧上前，此時暗殺團的黑澤忠三郎以[[手槍|手槍]]向井伊直弼的[[轎子|轎子]]開槍，貫穿轎子，射中井伊直弼的[[腰|腰]]部與[[大腿|大腿]]，井伊直弼立刻无法动弹。這槍響同時也是暗殺團發動總攻擊的[[暗號|暗號]]，四面埋伏的暗殺份子紛紛突襲井伊隊伍。轿夫受惊吓一轰而散。数名侍卫想搬动座轿，被当场斩倒，不得不放弃座轿。彦根藩士因为刀在袋中无法拔出，有的只能用刀鞘抵抗，有的徒手抓对方刀时手指被斩落，处于不利的境地。只有二刀流高手河西忠左卫门冷静地脱掉蓑衣拔出刀来守护在座轿一侧，斩杀了稻田重藏等袭击者。但是一人难敌众人最后被杀。河西刀刃卷起的刀保存在[[彦根城|彦根城]]博物馆。

井伊的護衛雖有人數上的優勢，但因措手不及而招架不住。暗殺團的稻田重藏首先攻入井伊直弼的轎子附近，但遭井伊家護衛砍死。隨後唯一的[[薩摩藩|薩摩藩]]浪士有村次左衛門成功擊倒轎旁的護衛，一把將井伊直弼由轎中拖出[[斬首|斬首]]，與廣岡子之次郎一同帶著井伊的首級突圍，但先後被砍成重傷，最後朝向[[日比谷門|日比谷門]]方向逃逸。有村次左衛門在[[若年寄|若年寄]]遠藤但馬守的宅邸前[[切腹|切腹]]自盡，廣岡子之次郎則在[[姬路|姬路]]酒井家前切腹。暗殺團中同樣因重傷而切腹自殺的尚有山口辰之介及鯉淵要人。

通说袭击事件后彦根藩邸马上派人前往，回收了座轿和沾满鲜血的手指、耳朵；直弼的首级放在远藤家宅邸。彦根藩知道以后，伪称是年龄、体格和直弼相仿的鬥死藩士加田九郎太首级要回，在藩邸命典医和躯体缝合。

暗殺團事成後各自散逃，8人帶傷向官府[[自首|自首]]，2人於逃亡途中被捕，1人於逃亡兩年後[[自殺|自殺]]，只有2人在逃亡後得享餘年。

== 死伤者和处分 ==
袭击者中最初接近座轿的稻田重藏被河西斬倒后立刻死亡。有村次左卫门，广冈小之次郎、山口辰之介、鲤渊要人等在彦根藩士的拼死反击之下负重伤，到其他藩邸自首后[[切腹|切腹]]自刃，除此之外的要么自首要么被捕后被杀或死于狱中，只有增子金八和海后磋矶之介潜伏到[[明治|明治]]年代活了下来。

井伊家除了[[井伊直弼|井伊直弼]]另有8人死亡（当场死亡4人、伤重死亡4人）、13人负伤。死亡者准许保留武士名分，作为担当直弼护卫失败的责任。2年后1862年（文久2年）下达了对幸存者的处分决定：草刈鍬五郎等重伤者减俸並流放到藩领的下野国[[佐野|佐野]][[幽禁|幽禁]]處分，轻伤者全部[[賜死|賜死]]，无伤者和轿夫一律[[斩首|斩首]]、除去武士名分。处分不光涉及本人还连带亲族，对幕府家臣是很大的警示。

== 影响 ==
井伊試圖把幕政由「與雄藩協調」的體制拉回「由幕府主导、阻止朝廷介入政治」的政治路線出現破綻，造成幕府权威丧失，幕末的[[尊王攘夷|尊王攘夷]]运动開始激化。最终拉开依靠恐怖暗杀来达到政治目的的序幕。

1864年（元治1年）[[天狗党之乱|天狗党之乱]]中，彦根藩士口号是为[[井伊直弼|井伊直弼]]报仇，封锁[[中山道|中山道]]迎击从筑波山向京都进发的水戶藩士。天狗党人不得不走[[美濃國|美浓]]经[[飛驒國|飛驒]]进入[[越前國|越前]]，最后在[[敦賀市|敦贺]]投降，[[武田耕云斋|武田耕云斋]]等水戶浪人352人在此被处决。彦根藩士处决水戶浪人的刑场是来迎寺。

== 其后的井伊家 ==
据当时的官方记载：「井伊直弼急病发作，在与病魔斗争中提出家业继承状，受理中病死」。这是幕府为了防止谱代大名井伊家和水戶藩进一步关系激化而作的掩盖。[[井伊家|井伊家]]的菩提寺[[豪徳寺|豪徳寺]]井伊直弼墓碑上所刻死亡日期是「三月二十八日」。是为了隐瞒直弼的死讯，装作直弼在櫻田門外受伤；将军还给直弼的藩邸送来慰问品。大名们也纷纷效仿派使者来慰问，当中也有水戶藩的人，彦根藩士怒目相对。根据井伊家飞地[[世田谷|世田谷]]代官大场家的记录，为了假装直弼只不过是受伤，大场家的人还举行了祈愿直弼早日康复的仪式。然而袭击过后尾張徳川家等其他大名的座轿不断经过，很多人看到雪地上的鲜血，大老被暗杀的事情很快传扬出去。

1862年（文久2年）率兵上京的[[岛津久光|岛津久光]]拥护勅使[[大原重徳|大原重徳]]要求幕府革新，要幕府在[[松平庆永|松平庆永]]的主导下清算井伊政权，结果罢免了直弼亲信的老中[[安藤信正|安藤信正]]和[[久世广周|久世广周]]、剥夺井伊家京都守护的家职、让[[松平容保|松平容保]]担任京都守护。井伊家石高也从35万石减为25万石。彦根藩不得不把直弼心腹[[长野主膳|长野主膳]]和[[宇津木六之丞|宇津木景福]]斩首并断绝往来，但是没有对他们减封。

[[华族令|华族令]]施行时井伊家被封为[[伯爵|伯爵]]，这个爵位是按照减封后的石高作为标准。如果石高35万石，井伊家认为领有半个近江国的话，应该是[[侯爵|侯爵]]。然而减封后是9万石左右，即使不减封石高15万石也不可能是侯爵。爵位是按照华族令施行时的石高为准，跟因为[[安政大狱|安政大狱]]而遭到新政府冷遇完全没有关系。[[戊辰战争|戊辰战争]]中，[[鳥羽伏見之战|鳥羽伏見之战]]中和萨摩藩兵一同守卫东寺和大津最初就显示出勤王的姿态，在逮捕[[近藤勇|近藤勇]]的过程中因为有功赏与2万石。直弼的儿子[[井伊直宪|井伊直宪]]的妻子是[[有栖川宮|有栖川宮]]家，井伊家作为旧谱代大名已经受到很多优待。

== 和解 ==
櫻田門外之變中敌对的[[水戶市|水戶]]和[[彦根市|彦根]]在事件发生110年后的1970年缔结为友好城市。水戶以[[偕乐园|偕乐园]]的[[梅花|梅花]]、彦根以城壕中的[[天鹅|天鹅]]互赠对方。水戶和彦根和解的中介是[[敦贺市|敦贺]]，敦贺是[[天狗党之乱|天狗党之乱]]中屠杀[[天狗党|天狗党]]的地方。水戶和敦贺因为天狗党的关系，在1964年结为友好城市。
