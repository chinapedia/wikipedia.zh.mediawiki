'''各拉丹冬峰'''，是一座冰川雪山，[[青海省|青海省]]西南部靠近青藏两省交界，为[[唐古拉山脉|唐古拉山脉]]主峰，海拔6621米。各拉丹冬峰是长江源头[[沱沱河|沱沱河]]的发源地。
== 位置 ==
藏语“各拉丹冬”一词意为“高高尖尖的山峰”。地理坐标为91.0°E，33.5°N。各拉丹冬峰属于[[唐古拉镇|唐古拉镇]]，是[[海西蒙古族藏族自治州|海西州]]的一块[[飞地|飞地]]，位于唐古拉山脉中段。

各拉丹冬雪山除主峰外，周围还有超过20座海拔超过{{convert|6,000|m|0}}的山峰。各拉丹冬雪山南北长{{convert|50|km|mi|0}}，东西宽{{convert|20|km|mi|0}}，终年积雪，[[雪线|雪线]]高度{{convert|5,800|m|0}}，冰川覆盖面积约{{convert|670|km2|sqmi}}，有40条冰川。各拉丹冬雪山与[[西藏自治区|西藏自治区]][[那曲地区|那曲地区]][[安多县|安多县]]接壤。

[[青藏铁路|青藏铁路]]穿越唐古拉山脉各拉丹冬峰东部{{convert|100|km|mi|0}}。
== 气候 ==
各拉丹冬峰地区属干旱气候，年降水量仅{{convert|200|mm|in|1}}。然而，在超过{{convert|5000|m|ft|0}}的高海拔地区，局部地区有剧烈的气候循环，山麓地区年降水量在数百毫米左右。因此，该地区[[雪|雪]]、[[冰雹|冰雹]]天气很频繁，这些条件有利于冰川的形成。各拉丹冬峰地区平均气温{{convert|−5|C|F|abbr=on}}，一年中最热的月份6月到8月气温超过{{convert|20|C|F|abbr=on}}，最冷的月份在一月，气温为{{convert|−18|C|F|abbr=on}}。在超过{{convert|5000|m|ft|0}}的高海拔地区，一年四季都很冷，最低温度低于{{convert|−30|C|F|abbr=on}}。
== 野生动物 ==
各拉丹冬雪山栖息着20多种野生动物，包含[[白唇鹿|白唇鹿]]、[[藏野驴|藏野驴]]、[[藏羚羊|藏羚羊]]、[[雪豹|雪豹]]和高山[[雪鸡|雪鸡]]。
== 登山 ==
日本登山队经过三年争取获得许可后，在1985年首次登顶了各拉丹冬峰。<ref name="rackcdn">http://c498469.r69.cf2.rackcdn.com/2009/348_China_aaj2009_web.pdf</ref>他们走的是西北山脊接近东北的路线。

虽然很危险，但各拉丹冬在中国登山者中很受欢迎。1997年，[[北大山鹰社|北大山鹰社]]成为第一个到达各拉丹冬峰顶的中国登山队。
