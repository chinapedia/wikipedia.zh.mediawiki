'''甲酸'''（[[英文|英文]]：Formic acid）又称作'''蚁酸'''，[[化学式|化学式]]为'''HCOOH'''。[[蚂蚁|蚂蚁]]和[[蜜蜂|蜜蜂]]等[[膜翅目|膜翅目]][[昆蟲|昆蟲]]的分泌液中含有蚁酸，当初人们[[蒸馏|蒸馏]]蚂蚁时制得蚁酸，故有此名。甲酸无色而有刺激气味，且有[[腐蚀|腐蚀]]性，人类[[皮肤|皮肤]]接触后会起泡红肿。[[熔点|熔点]]8.4℃，[[沸点|沸点]] 100.8℃。由于甲酸的结构特殊，它的一个[[氢原子|氢原子]]和[[羧基|羧基]]直接相连。也可看做是一个[[羟基|羟基]][[甲醛|甲醛]]。因此甲酸同时具有[[酸|酸]]和[[醛|醛]]的性质。在[[化学工业|化学工业]]中，甲酸被用于[[橡胶|橡胶]]、医药、[[染料|染料]]、[[皮革|皮革]]种类工业。

== 性质 ==
甲酸与水和大多数的极性有机溶剂混溶，在[[烃|烃]]中也有一定的溶解性。在[[烃|烃]]中及气态下， 甲酸以通过以[[氢键|氢键]]结合的[[二聚体|二聚体]]形态出现。在气态下，氢键导致甲酸气体与[[理想气体状态方程|理想气体状态方程]]之间存在较大的偏差。液态和固态的甲酸由连续不断的通过[[氢键|氢键]]结合的甲酸[[分子|分子]]组成。

甲酸具有与大多数其他[[羧酸|羧酸]]相同的性质，尽管在通常情况下甲酸不会生成[[酰氯|酰氯]]或者[[酸酐|酸酐]]。直到不久以前，所有试图将甲酸转化成这些[[衍生物|衍生物]]的尝试都以产物[[一氧化碳|一氧化碳]]告终。甲酸酐可由甲酰氟和甲酸钠在零下78摄氏度反应得到。甲酰氯可由将[[氯化氢|氯化氢]]气体通过零下60度1-甲酰基[[咪唑|咪唑]]的一氯甲烷溶液得到。甲酸脫水分解为[[一氧化碳|一氧化碳]]和[[水|水]]。甲酸具有和[[醛|醛]]类似的[[还原|还原]]性，它能发生[[银镜反应|银镜反应]]，把[[银|银]][[氨|氨]][[络离子|络离子]]中的[[银离子|银离子]]还原成金属银，而自己被氧化成[[二氧化碳|二氧化碳]]和[[水|水]]：

: HCOOH + 2AgOH → 2Ag + 2H<sub>2</sub>O + CO<sub>2</sub>

甲酸是唯一能和[[烯烃|烯烃]]进行[[加成反应|加成反应]]的[[羧酸|羧酸]]。甲酸在[[酸|酸]]的作用下（如[[硫酸|硫酸]]，[[氢氟酸|氢氟酸]]），和[[烯烃|烯烃]]迅速反应生成[[甲酸酯|甲酸酯]]。但是类似于[[Koch反应|Koch反应]]的副反应也会发生，产物是更高级的羧酸。

大多数的[[甲酸盐|甲酸盐]]溶于水。

== 制备 ==
制备其他化学药品（尤其是[[乙酸|乙酸]]）的过程中，大量的甲酸作为副产物被生产出来。 然而这样的制备远远的不能满足目前对甲酸的需要，所以有些甲酸被直接生产。

在强碱作用下，甲醇和[[一氧化碳|一氧化碳]]反应生成[[甲酸甲酯|甲酸甲酯]]：
:CH<sub>3</sub>OH + CO → HCOOCH<sub>3</sub>

在工业生产中， 此反应在液态和加压的状态下进行。 典型的反应条件为80摄氏度和40个大气压。广泛使用的碱为[[甲醇钠|甲醇钠]]。[[水解|水解]]甲酸甲酯得到甲酸。 
:HCOOCH<sub>3</sub> + H<sub>2</sub>O → HCOOH + CH<sub>3</sub>OH

水解甲酸甲酯需要大量的水来保证反应顺利进行。 有些生产商使用一种间接的水解途径, 即先将甲酸甲酯和[[氨|氨]]反应产生[[甲酰胺|甲酰胺]]，然后用[[硫酸|硫酸]]水解甲酰胺得到甲酸：
:HCOOCH<sub>3</sub> + NH<sub>3</sub> → HCONH<sub>2</sub> + CH<sub>3</sub>OH
:2HCONH<sub>2</sub> + 2H<sub>2</sub>O + H<sub>2</sub>SO<sub>4</sub> → 2HCOOH + (NH<sub>4</sub>)<sub>2</sub>SO<sub>4</sub>

这种技术有其自身缺点，尤其是在对副产物[[硫酸铵|硫酸铵]]的处理上。有些生产商最近发展了一类节能的方法，即将甲酸从直接水解的大量水溶液中提取出来。在其中一种方法（由[[巴斯夫|巴斯夫]]所使用）中，甲酸在有机碱的作用下由湿法萃取得到。

在实验室制备中，甲酸可由在无水[[丙三醇|丙三醇]]中加热[[草酸|草酸]]，然后蒸汽蒸馏得到。另外一种制备方法（必须要在[[通风橱|通风橱]]中进行）是在盐酸作用下的异丙腈的水解。
:C<sub>2</sub>H<sub>5</sub>NC + 2H<sub>2</sub>O → C<sub>2</sub>H<sub>5</sub>NH<sub>2</sub> + HCOOH

异丙腈的制备由[[乙胺|乙胺]]和[[氯仿|氯仿]]反应获得。（异丙腈过于令人不快的气味使此反应必须在[[通风柜|通风橱]]中进行。）

== 化学性质 ==

甲酸具有[[羧酸|羧酸]]的通性。但甲酸同时具有一个[[醛基|醛基]]，这使它可以被进一步氧化：
:2HCOOH + {{chem|O|2}} → 2{{chem|CO|2}} + 2{{chem|H|2|O}}

甲酸在浓[[硫酸|硫酸]]或[[氧化铝|氧化铝]]催化下加热脱水生成[[一氧化碳|一氧化碳]]，这是实验室制取纯净一氧化碳的方法。
:HCOOH → CO + {{chem|H|2|O}} （在浓硫酸催化下加热）

甲酸也能在[[氧化锌|氧化锌]]或[[高锰酸钾|高锰酸钾]]催化下，进行脱氢反应：
:HCOOH → {{chem|CO|2}} + {{chem|H|2}}
== 毒性 ==
甲酸对人体的毒性除了刺激[[黏膜|黏膜]]还会抑制[[线粒体|线粒体]]中的[[细胞色素c氧化酶|细胞色素c氧化酶]]活性，导致[[呼吸链|呼吸链]]中断，代谢产物堆积，细胞坏死。
