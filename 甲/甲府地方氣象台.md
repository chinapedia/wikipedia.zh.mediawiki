'''甲府地方氣象台'''（{{Jpn|j=甲府地方気象台|hg=こうふちほうきしょうだい|rm=Kōfu chihō kishōdai}}）是[[日本|日本]][[氣象廳|氣象廳]][[東京管區氣象台|東京管區氣象台]]下屬的一個[[地方氣象台|地方氣象台]]，位於[[山梨縣|山梨縣]][[甲府市|甲府市]]，主要負責山梨縣的氣象預報、氣象觀測、地震觀測和減災防災工作<ref>{{Cite web |url = http://www.jma-net.go.jp/kofu/menu/gyoumu_info.html |title = 甲府地方気象台はこのような仕事を行っています |author = 甲府地方気象台|language = ja|accessdate = 2014-04-21}}</ref>。作為距[[富士山|富士山]]最近的地方氣象台（約40[[公里|公里]]），還負責向全國發佈富士山[[初冠雪|初冠雪]]的目測報告<ref>{{Cite web |url = http://www.jma-net.go.jp/kofu/menu/siryo_hatsukansetsu.html |title = 初冠雪の記録（甲府）：富士山 |author = 甲府地方気象台|language = ja|accessdate = 2014-04-21}}</ref>。

== 組織 ==
甲府地方氣象台受[[氣象廳|氣象廳]]和[[東京管區氣象台|東京管區氣象台]]管制，最高責任者為台長。下屬次長、業務危機管理官、防災管理官和觀測預報管理官。

== 歷史 ==
* 1894年，山梨縣立甲府測候所在甲府市成立，這是山梨縣氣象觀測的開始。
* 1938年，轉為國營。
* 1957年，升格為甲府地方氣象台。
* 1993年，新廳舍建成。<ref>{{Cite web |url = http://www.jma-net.go.jp/kofu/menu/enkaku.html |title = 甲府地方気象台の沿革と組織 |author = 甲府地方気象台|language = ja|accessdate = 2014-04-21}}</ref>
