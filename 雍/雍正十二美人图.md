'''雍正十二美人图'''或'''十二美人图'''，是[[故宫博物院|故宫博物院]]收藏的一组清代[[绢本|绢本]]设色工笔[[汉服|汉服]][[仕女圖|仕女圖]]的[[俗称|俗称]]，普遍认为此组美人图与[[雍正帝|雍正帝]]有关<ref>{{Cite web |url=http://book.ifeng.com/shuxun/detail_2013_05/30/25896965_0.shtml |title=《紫禁城》：名家共论故宫藏“美人图”|accessdate=2016-03-30 |author=责任编辑：严彬|date=2013-05-30 |work= |publisher=凤凰网，来源：凤凰读书|language=Zh-hans }}</ref>。

原系[[清宫|清宫]]旧藏<ref name="a3">{{Cite web|url=http://news.xinhuanet.com/shuhua/2012-02/17/c_122717856.htm|title=揭秘故宫馆藏"十二美人图" （组图）|accessdate=2016-03-30|author=责任编辑：谭雪莉|date=|work=|publisher=新华网，来源：国际在线|language=Zh-hans|deadurl=yes|archiveurl=https://web.archive.org/web/20120302191857/http://news.xinhuanet.com/shuhua/2012-02/17/c_122717856.htm|archivedate=2012-03-02}}</ref>，共有十二幅。1950年，故宫博物院工作人员杨臣彬、石雨村在清点库房时，发现此组绘画。1983年，[[黄苗子|黄苗子]]认为画中美人是雍正帝[[妃嫔|妃嫔]]。1986年，[[朱家溍|朱家溍]]发现[[内务府|内务府]]档案记载：[[雍正|雍正]]十年（1732年）[[圆明园|圆明园]]深柳读书堂围屏上“拆下美人绢画12张”。被认为正是画作的出处<ref name="a1">{{Cite web |url=http://news.163.com/13/1123/14/9ECFJK7U00014AED.html |title=雍正“十二美人图”之谜|accessdate=2016-03-30 |author=|date=2013-11-23  |work=祝勇 |publisher=网易，来源：北京晚报(北京)|language=Zh-hans }}</ref>。[[清宫档案|清宫档案]]记载：雍正十年八月间，传旨将这十二张绢画“着垫纸衬平，各配做卷杆”藏于宫内<ref>{{Cite web|url=http://news.xinhuanet.com/shuhua/2012-02/17/c_122717856_2.htm|title=揭秘故宫馆藏"十二美人图" （组图）|accessdate=2016-03-30|author=责任编辑：谭雪莉|date=|work=|publisher=新华网，来源：国际在线|language=Zh-hans|deadurl=yes|archiveurl=https://web.archive.org/web/20160413142724/http://news.xinhuanet.com/shuhua/2012-02/17/c_122717856_2.htm|archivedate=2016-04-13}}</ref>。2014年，故宫博物院以此组图作中的九幅为原型，创作相关动画<ref name="a2"/>。

画幅尺寸相同，均纵184厘米，横98厘米<ref name="a3"/>。每一幅画均是一位美人，根据内容，分别是：裘装对镜、烘炉观雪、倚门观竹、立持如意、桐荫品茗、抚书低吟、消夏赏蝶、烛下缝衣、博古幽思、持表观菊、倚榻观鹊、捻珠观猫<ref name="a2">{{Cite web |url=http://culture.people.com.cn/n/2014/1210/c87423-26179825.html |title=雍正女神风情万种 故宫《十二美人图》“动”起来[图]|accessdate=2016-03-30 |author=记者：许心怡|date=2014-12-10 |work= |publisher=人民网，来源|language=Zh-hans }}</ref>。

此画没有[[佚名|作者署名]]和题名。曾题名《雍正妃画像》，后改题《雍正妃行乐图》。绘画风格符合当时数位[[宫廷画师|宫廷画师]]的风格。画中背景的题字落款、印章等均为[[雍正帝|雍正帝]]本人。“裘装对镜”背景中的书法条幅落款破尘居士，这是雍正帝在[[康熙|康熙]]年间的自号。画中美人曾被认为是雍正帝妃嫔群像，但与[[清朝后宫|清宫妃嫔画像]]风格不符。曾任故宫博物院副院长的[[杨新|杨新]]推论后认为“消夏赏蝶”中美人是雍正帝[[孝敬憲皇后|皇后那拉氏]]。亦有认为画中所描绘的美女只是[[中国传统绘画|中国传统绘画]]中的格式化美人<ref name="a1"/>。
