'''善行站'''（{{jpn|j=善行駅|hg=ぜんぎょうえき|rm=Zengyō eki|f=y}}）是一個位於[[神奈川縣|神奈川縣]][[藤澤市|藤澤市]]善行一丁目，屬於[[小田急電鐵|小田急電鐵]][[江之島線_(小田急電鐵)|江之島線]]的[[鐵路車站|鐵路車站]]。[[車站編號|車站編號]]是'''OE 11'''。

因站名是良好的[[緣起|緣起]]，過去有發售紀念[[入場券|入場券]]。

== 歷史 ==
* 1960年（[[昭和|昭和]]35年）10月1日 - 隨著善行團地開發而開站，為各站停車停靠站。
* 2001年（[[平成|平成]]13年）2月 - 設置電梯。
* 2005年（平成17年） - 設置候車室。
* 2008年（平成20年）11月19日 - 車站商店改裝為OX SHOP。<!--これに伴い11月18日は改装のため売店が閉店した。-「歴史」に記載するには瑣末につきコメント化-->
* 2012年（平成24年）7月 - 啟用列車資訊顯示器。

== 站名由來 ==
本地聚落以[[江戶時代|江戶時代]]初期的「善行寺」命名為善行（寺）。

== 車站構造 ==
本站是[[側式月台|側式月台]]2面2線[[地面車站|地面車站]]，站房為[[跨站式站房|跨站式站房]]。鐵路為下沉式構造。首班車至7:15為止無站員配置。

月台從東側起為1號月台。

== 車站周邊 ==
車站旁有[[麥當勞|麥當勞]]與[[100圓店|100圓店]]、[[居酒屋|居酒屋]]、[[小菜|配菜]]店、[[拉麵|拉麵]]店、[[交番|交番]]等。<!--2010年12月まではパン屋があった。-->

另外，車站往藤澤方向有江之島線唯一的[[隧道|隧道]]。
; 西口
: 藤澤市善行市民中心、藤澤善行[[郵便局|郵便局]]、[[Ministop|Ministop]]、{{link-ja|Tsuruha藥妝|ツルハ}}、{{link-ja|相鐵ROSEN|相鉄ローゼン|SOTETSU ROSEN}}、計程車乘車處與{{link-ja|神奈川中央交通|神奈川中央交通}}巴士站。
; 東口
: [[FamilyMart|FamilyMart]]善行站東口店。步行30秒可至{{link-ja|藤澤翔陵高等學校|藤沢翔陵高等学校}}，步行5分可至{{link-ja|神奈川縣立體育中心|神奈川県立体育センター}}與{{link-ja|神奈川縣立綜合教育中心|神奈川県立総合教育センター}}善行廳舍。
; 周邊
: 藤澤中央批發市場、{{link-ja|荏原製作所|荏原製作所}}藤澤工場、荏原製作所營運的荏原湘南運動中心。往六會日大前站方向有藤澤乘馬俱樂部。車站步行20分可至藤澤市立善行小學校，15分可至藤澤市立善行中學校。由於車站位於丘陵地，周邊多斜坡。
