'''奄美魔物'''（琉球語：'''{{lang|ryu|マジムン}}'''，是[[琉球|琉球]][[奄美群島|奄美群島]][[鹿儿岛县|鹿儿岛县]]与[[冲绳县|冲绳县]]傳說中的一類[[邪靈|邪靈]]的總稱。據說奄美魔物的形態有各種各樣，很多魔物指死去的[[動物|動物]]仍以生前的姿態存在而產生的邪靈。在奄美群島也稱呼'''[[波布_(竹葉青)|波布]]'''為[[魔物|魔物]]<ref>{{Cite web|date=2008-01-07|url=http://amamiblog.yumeshima.info/?eid=587316|title=奄美の伝説「マジムン伝説」|work=[http://amamiblog.yumeshima.info/ 奄美 夢島Blog]|publisher=[http://yumeshima.info/ 奄美の観光情報サイト 夢島WEB]|accessdate=2010年8月8日}}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，傳說中認為其是神的使者，也是魔物中唯一以[[生物|生物]]形態的存在。

== 奄美魔物的種類 ==
; '''家鴨魔物''' （'''アイフラーマジムン'''）<ref>後掲『幻想世界の住人たち』IVでは「アイフラーマジムン」の名で記載されている。</ref>
:以[[鴨|家鴨]]的姿態存在的魔物<ref name="kyodokenkyu3">{{Cite journal|author=金城朝永|title=琉球妖怪変化種目|journal=郷土研究|volume=5卷|issue=3号|pages=41-42頁|publisher=郷土研究社|date=1931年7月}}</ref>。
; '''魔嬰''' （'''アカングァーマジムン'''）
: 以[[嬰兒|嬰兒]]的姿態存在的魔物<ref name="kyodokenkyu2">{{Cite journal|author=金城朝永|title=琉球妖怪変化種目|journal=郷土研究|volume=5卷|issue=2号|pages=58頁|date=1931年5月}}</ref>。
; '''牛形魔物''' （'''牛マジムン'''）
: 以[[牛|牛]]的姿態存在的魔物、同県[[島尻郡|島尻郡]]では龕（棺桶を担ぐ葬具）が牛に化けたものといわれる<ref>{{Cite book<!--|和書-->|author=日野巌・日野綏彦|editor=[[村上健司|村上健司]]校訂|title=動物妖怪譚|year=2006|publisher=[[中央公論新社|中央公論新社]]|series=[[中公文庫|中公文庫]]|isbn=978-4-12-204792-1|volume=下|pages=235頁|chapter=日本妖怪変化語彙}}</ref>。
; '''豬形魔物''' （'''ウワーグワーマジムン'''）
: 以[[豬|豬]]的姿態存在的魔物，常會以穿著衣服的形像出現<ref name="kyodokenkyu3" />。
; '''杓文字魔物''' （'''ミシゲー・マジムン'''）
: 杓文字幻化的魔物。詳見條目[[飯笥|飯笥]]，類似的還有'''杓子魔物''' （'''ナビゲー・マジムン'''，[[杓子|杓子]]幻化的魔物），都是長期使用的食器幻化而成的魔物<ref name="kyodokenkyu3" />，可視為[[付喪神|付喪神]]的一種。
