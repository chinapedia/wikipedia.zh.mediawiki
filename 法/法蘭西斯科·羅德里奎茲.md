'''法蘭西斯科·「法蘭奇」·荷西·羅德里奎茲'''（'''Francisco "Frankie" Jose Rodriguez'''，{{bd|1982年|1月7日|catIdx=R}}），暱稱'''「K-羅德」'''（K-Rod），出生於[[委內瑞拉|委內瑞拉]]，目前為[[美國職棒大聯盟|美國職棒大聯盟]]的[[費城費城人隊|費城費城人隊]]投手，司職[[終結者_(棒球)|終結者]]。


== 職棒經歷 ==
1998年，羅德里奎茲以業餘自由球員身分，透過非選秀的方式與天使隊簽約。因為肩部和手肘的傷勢，2001年球季他在小聯盟的球季有一段時間不能出賽。2002年，他從原本先發投手的位置被調到牛棚。有些人批評羅德里奎茲身材過於瘦小的且缺乏耐力，限制了他擔任先發的空間。在2002年9月中旬，天使隊牛棚裡幾個隊員掛傷號，因緣際會就登上大聯盟。羅德里奎茲在季後賽救援表現精彩，在沒有任何大聯盟勝投經驗的情況下，一口氣拿下了5勝，也讓他在國際媒體上大出風頭。

因為之前羅德里奎茲在大聯盟亮相的機會非常有限，所以季後賽時，打者還搞不清楚這位投手葫蘆裡究竟賣什麼藥，也因此情勢對他十分有利。在美聯分區系列戰(American LeagueDivision Series,ALDS)時，他封鎖了[[紐約洋基|紐約洋基]]隊的打線，在5局的投球中送出了13次三振，拿下2勝。在美聯冠軍戰(American League Chiampionship Series,ALCS)裡，他投出2勝0負，主投4局中三振7人的佳績。最後就是世界大賽了，對手是[[舊金山巨人|舊金山巨人]]。羅德里奎茲8局投球得到1勝1負，還讓對手吞下13K。當年他才20歲，不只是32年以來在世界大賽出戰的最年輕投手，而且是史上最年輕就拿下勝投的。


在大聯盟生涯第二年，羅德里奎茲真正成為一個合格的佈局投手(setup man)。他在頭15場任務中僅丟1分自責分，後來的44場也只有失9分，也被選入明星隊。在2004年8月24日，他成為繼馬克·科里爾（Mark Clear）（1980年，105次）、德魏恩·布斯(DeWayneBuice)（1987年，109次）、布萊恩·哈維(Bryan Harvey)（1991年，101次）、[[特洛依·帕西佛|特洛依·帕西佛]](Troy Percival)（1996年，100次）、[[史考特·謝爾茲|史考特·謝爾茲]]（2004年，109次）後，第六位單季三振數超過100次的非先發投手。其中謝爾茲目前擔任的正是羅德里格茲的佈局投手。

2005球季開始，原本的終結者帕西佛成為自由球員，並與[[底特律老虎|底特律老虎]]簽約而離開天使，他的位置便空下來給羅德里奎茲。改善了四壞球保送過多的毛病；以及經歷一場對[[奧克蘭運動家|奧克蘭運動家]]隊的關鍵比賽中，因為漏掉捕手回傳球，導致對方得到致勝分的糗事，羅德里奎茲真正克服了2005年降臨的艱困挑戰。他和-{[[克里夫蘭印地安人|克里夫蘭印地安人]]}-隊的當家終結者巴伯·威克曼(Bob Wickman)，在美聯以45次救援成功紀錄並列2005球季救援王。

2006年10月1日，羅德里奎茲以第47次救援成功，刷新了美聯紀錄。稍早一點的9月10日對上[[多倫多藍鳥|多倫多藍鳥]]隊時拿下生涯第100次救援成功，24歲的他成為大聯盟史上最年輕的百救援投手，取代了25歲時才獲得此殊榮的葛列格·奧森（Gregg Olson）。因為羅德里奎茲在救援上的成就，8月時被選為該月的DHL棒球快遞員（DHL Baseball Delivery Man）。

2008年7月7日，在對上遊騎兵時拿下第35場救援成功，成為大聯盟史上首位明星賽前取得35次救援的投手，原記錄是勇士[[約翰·史摩茲|約翰·史摩茲]]在2003年締造的34次。<ref>[http://mlb.com/news/gameday_recap.jsp?ymd=20080707&content_id=3086375&vkey=recap&fext=.jsp&c_id=ana Bats arrive early; K-Rod delivers late]{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>9月13日，對西雅圖水手拿下第58次救援成功，打破1990年Bobby Thigpen所保持的57次救援成功紀錄，成為單季救援成功的保持者。而球季結束後，他總共拿到62次救援成功，這項紀錄至今無人能破。

2008年12月10日，羅德里奎茲和[[纽约大都会|纽约大都会]]签下三年总价值3700万美元的合约<ref name="Mets">{{cite web|url=http://newyork.mets.mlb.com/news/article.jsp?ymd=20081210&content_id=3711163&vkey=news_nym&fext=.jsp&c_id=nym|title="K-Rod 'thrilled' to officially join Mets"|author=DiComo, Anthony|deadurl=yes|archiveurl=https://web.archive.org/web/20120222035917/http://newyork.mets.mlb.com/news/article.jsp?ymd=20081210&content_id=3711163&vkey=news_nym&fext=.jsp&c_id=nym|archivedate=2012-02-22}}</ref>。羅德里奎茲原先在天使队一直穿着57号球衣，由于[[尤汉·山塔纳|尤汉·山塔纳]]在大都会穿着57号，因此羅德里奎茲改穿75号球衣<ref name="Mets & K-Rod finalize deal">{{cite web|url=http://sports.espn.go.com/mlb/news/story?id=3760925|title="K-Rod, Mets finalize deal"|author=[[美联社|美联社]]}}</ref>。

2017年出賽28場，戰績2勝5敗及7救援成功，防禦率高達7.82創下生涯最糟的成績，因此在6月份就遭釋出。

2018年1月27日，和[[費城費城人隊|費城費城人隊]]簽下小聯盟合約，附帶春訓邀請以及375萬美金的激勵獎金。<ref>[https://tw.appledaily.com/new/realtime/20180128/1287224/ 費城人與K-Rod簽小聯盟約　延續球員生涯]，[[蘋果日報|蘋果日報]]，2018-01-28</ref>

== 球種 ==
羅德里奎茲的主戰球種是均速在94—97英哩間的快速球，其強大的尾勁常使打者把它與大角度曲球混淆。變化球系列則有兩大球路：一個是很晚轉彎、且變化幅度極大的[[曲球|曲球]]，一開始看似進壘的好球，接近後卻忽然下墜，掉出好球帶。另一個看起來像[[滑曲球|滑曲球]]的變化球，狀況好時會接近右打者，使他們屈身閃躲後，球又轉彎進壘。因為他的滑球常常因為下墜幅度太大而造成暴投次數偏高。2006球季，羅德里格茲的球種增加了[[變速球|變速球]]，用以對付左打者。

== 綽號 ==
羅德里奎茲的暱稱「K-羅德」（K-Rod）隨著他2002年9月到季後賽間的竄紅而蔚為風潮。這個暱稱取材自大聯盟巨星[[艾力士·羅德里奎茲|艾力士·羅德里奎茲]]的綽號「A-羅德」（A-Rod），而「K」代表的是棒球紀錄三振的符號。

== 特殊的形象 ==
羅德里奎茲的投球充滿著力量與熱情。在每次救援成功後，他往往用很興奮的動作與表情慶祝勝利。他投球動作的高抬腿與誇張的機制，充滿著力與美。羅德里奎茲上場時都戴著一副很像蛙鏡的透明太陽眼鏡，類似於[[洛杉磯道奇|洛杉磯道奇]]隊救援艾力克·蓋尼（Eric Gagne）和[[紐約大都會|紐約大都會]]隊的中繼杜亞納·香吉士（Duaner Sanchez）。
