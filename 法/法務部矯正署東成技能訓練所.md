'''法務部矯正屬東成技能訓練所'''是一座隸屬於[[中華民國|中華民國]][[法務部|法務部]][[矯正署|矯正署]]旗下的[[矯正機構|矯正機構]]，該機構坐落於[[臺灣|臺灣]][[臺東縣|臺東縣]][[卑南鄉|卑南鄉]][[斑鳩溪|斑鳩溪]]谷西岸平原上，核定收容額為808人，當前收容人數（2015年9月21日數據）724名。

== 沿革 ==
=== 警備總司令部時期 ===
法務部矯正署東成技能訓練所最早為民國73年（1984年）11月12日，由[[警備總司令部|警備總司令部]]、[[警政署|警政署]]、[[調查局|調查局]]、[[憲兵|憲兵]]隊等治安機關參與組成專案小組實施[[一清專案|一清專案]]中，因計劃執行到案者達3,141名，基於收容人數激增，警備總司令部趕工整建各職訓隊營房，並於隔年，民國74年（1985年）以東成305預備師的駐紮舊址設立職業訓導第三總隊附屬第四大隊二中隊，作為職業訓導第三總隊位在臺東縣東河鄉泰源總部的新收中隊，以紓解爆滿隊員，是為東成技能訓練所之始祖。<ref name=" 中華民國後備憲兵論壇 台灣警備總司令部 職業訓導處">{{cite web |url=http://mp.rocmp.org/kind/lagger/ |title= 中華民國後備憲兵論壇 台灣警備總司令部 職業訓導處 | accessdate=2015-09-30}}</ref>

民國75年（1986年）1月30日，時任[[參謀總長|參謀總長]]的[[郝柏村|郝柏村]]搭乘[[直昇機|直昇機]]來到東成進行建築工程視察時，曾因[[螺旋槳|螺旋槳]]造成[[陣風|陣風]]過大，當場把東成的新大門應聲吹倒。<ref name=" 中華民國後備憲兵論壇 台灣警備總司令部 職業訓導處"/>

民國76年（1987年）7月4日，東成職業訓導第三總隊四大二中隊擴編為職業訓導第四總隊（職業訓導第三總隊四大更易為職業訓導第四總隊三大），於當日上午由時任警備[[總司令|總司令]][[陳守山|陳守山]]宣布正式成立。<ref name=" 中華民國後備憲兵論壇 台灣警備總司令部 職業訓導處"/>

民國78年（1988年）8月22日奉[[臺灣省|臺灣省]]保安司令部依據同年4月30日公布的《台灣省戒嚴時期取締流氓辦法》，將位在當時[[臺北縣|臺北縣]]之[[坪林|坪林]]第一職訓總隊除保留職訓中心之外（番號仍隸於職一總隊），總隊部暨番號移動進駐當時東成的職業訓導第四總隊，並改為職業訓導第一總隊。<ref name=" 中華民國後備憲兵論壇 台灣警備總司令部 職業訓導處"/>

=== 法務部接管時期 ===
在臺灣解除[[戒嚴|戒嚴]]後，臺灣省政府為配合[[動員戡亂|動員戡亂]]法令終止，將警備總司令部裁撤，並於1992年（民國81年）7月1日奉[[行政院|行政院]]核定，將臺東東成職業訓導第一總隊撥交給行政院旗下之[[法務部|法務部]]接管，並更名為「臺灣東成技能訓練所」。<ref name=" 法務部矯正署東成技能訓練所">{{cite web |url=http://www.dcv.moj.gov.tw/ct.asp?xItem=5970&CtNode=7400&mp=070 |title= 法務部矯正署東成技能訓練所 歷史沿革 | accessdate=2015-09-30}}</ref>

改制後，所內建築改建工程於民國82年（1993年）2月1日正式動土開工，歷時兩年於民國84年（1995年）10月11日完工，隨即正式啟用受刑人收容。<ref name=" 法務部矯正署東成技能訓練所"/><ref name=" 「監獄建築與環境美學」">{{cite web |url=http://www.corrections-cca.org.tw/index.php?do=new_knowledge_info&id=3142 |title= 「監獄建築與環境美學」|author=吳憲璋 | accessdate=2015-09-30}}</ref>

民國100年（2011年）1月1日[[法務部矯正署|法務部矯正署]]成立，東成技能訓練所改隸在「法務部矯正署」旗下，並改稱為「法務部矯正署東成技能訓練所」。<ref name=" 法務部矯正署東成技能訓練所"/>

== 設施 ==
* 東成技能訓練所行政大樓分別設有所長室、秘書室及總務科、技訓科、人事室、會計室、統計室、政風室等各科室辦公室。<ref name=" 法務部矯正署東成技能訓練所"/>
* 戒護大樓分別有戒護科辦公室、管理人員常年教育室、備勤室、文康室及員工餐廳等，地下一層則為收容人之接見室。<ref name=" 法務部矯正署東成技能訓練所"/>
* 戒護區劃分為5個教區，二、三教區場舍建築呈菱型排列，每個菱型區再以圍牆分隔為4個獨立區，形成分區再分區，隔離再隔離，以防止串聯。<ref name=" 法務部矯正署東成技能訓練所"/><ref name=" 「監獄建築與環境美學」"/>

== 人員配置 ==
當前，東成技能訓練所員額編制共147人，並配置有所長1人，秘書1人，下分設有調查分類、輔導、技能訓練、衛生、戒護、總務等6科及人事、會計、政風等3室。<ref name=" 法務部矯正署東成技能訓練所"/>

另設所務委員會，由所長、秘書及各科室主管人員組成。<ref name=" 法務部矯正署東成技能訓練所"/>
