'''法卡山'''是[[中越边界|中越边界]]上的一处小型山脉，曾命名为'''英雄山'''<ref name="a1">{{Cite web |url=http://news.ifeng.com/history/zhongguoxiandaishi/detail_2013_11/29/31662294_1.shtml|title=中越划界为何“法卡山已划给越南”流言四起？ |accessdate=2019-03-03 |author=胡贲 |date=2013-11-29 |work= |publisher=凤凰网，来源：南方周末（2009年12月3日）|language=Zh-hans }}</ref>。现分属中越两国。中国方面，在行政上属[[广西壮族自治区|广西壮族自治区]][[凭祥市|凭祥市]][[上石镇|上石镇]]。越南方面属于[[谅山省|谅山省]][[高禄县|高禄县]]。

由五个山峰组成，以数字命名。最高峰为3号高地，海拔511.3米。3号高地曾立有中越边界26号界碑，为两国国界骑线点。1号、2号高地属中国，4号、5号高地属越南<ref name="a1"/>。[[1979年中越战争|1979年中越战争]]爆发，越军占据法卡山攻击中国。1980年[[法卡山战役|法卡山战役]]后，中国占据法卡山全部山峰。1999年，中越两国开始边界谈判，划分一千多公里的边界。由于中国政府相关规定，未公开详细情况。故中国舆论曾认为，法卡山、[[老山_(麻栗坡县)|老山]]被归属于越南。2006年，3号高地竖立1156号界碑，确认4号、5号高地划归越南<ref>{{Cite web |url=http://www.sohu.com/a/274628433_477856|title=探秘丨中国为什么归还法卡山 |accessdate=2019-03-03 |author= |date=2018-11-11 |work= |publisher=搜狐网|language=Zh-hans }}</ref>。2013年报道，中国方面早已废弃在法卡山的前出阵地。现由[[中国人民解放军|解放军]]边防部队驻守，未移交[[武警|武警]]边防部队。解放军进行边防巡逻，未设常备阵地<ref name="a1"/>。
