'''法務部矯正署泰源技能訓練所'''，簡稱'''泰源技能訓練所'''、'''泰源技訓所'''，是[[中華民國法務部|中華民國法務部]]的矯正機關，地點位於[[台東縣|台東縣]][[東河鄉_(台灣)|東河鄉]]北源村，[[海岸山脈|海岸山脈]]深山裡[[泰源盆地|泰源盆地]]之[[河階|河階]][[台地|台地]]，屬[[馬武窟溪|馬武窟溪]]支流北溪旁的河谷地，四周受北溪的[[曲流|曲流]]所包圍，易守難出，形勢險要。

== 沿革 ==
{{seealso|國防部臺南監獄}}
泰源技能訓練所前身為[[臺灣警備總司令部|臺灣警備總司令部]]所屬'''職業訓導第三總隊'''及'''泰源職業訓練中心'''，所在地於[[臺灣日治時期|臺灣日治時期]]為日本人種植[[咖啡|咖啡]]的農園，是[[台灣原住民|台灣原住民]][[阿美族|阿美族]]的部落區域，[[國防部|國防部]]接管後設立監獄，內部分為仁監與義監，大部分關押[[政治犯|政治犯]]，以思想感訓為主<ref name="泰源事件"/>。1987年（民國76年）7月政府宣布[[解嚴|解嚴]]，軍法與司法分治，原「臺灣警備總司令部」裁撤，行政院遂於1988年（民國77年）1月15日核定將「臺灣警備總司令部職業訓導第三總隊」及「臺灣警備總司令部泰源職業訓練中心」一併撥交[[中華民國法務部|法務部]]接管，並定名為「'''臺灣泰源監獄'''」<ref name="歷史沿革"/>。1992年（民國81年）1月31日，《法務部技能訓練所組織條例》公布施行，同年7月1日將「臺灣泰源監獄」正名為「'''臺灣泰源技能訓練所'''」<ref name="歷史沿革"/>。配合[[法務部矯正署|法務部矯正署]]成立，於2011年（民國100年）1月1日更名為「'''法務部矯正署泰源技能訓練所'''」至今<ref name="歷史沿革"/>。

== 歷史事件 ==
{{main|泰源事件}}
五名外役政治犯[[江炳興|江炳興]]、[[詹天增|詹天增]]、[[鄭金河|鄭金河]]、[[陳良|陳良]]及[[謝東榮|謝東榮]]，1970年2月發動監獄[[革命|革命]]，計劃結合警衛連官兵及當地青年，奪取武器，佔領廣播電台，繼而發動全島[[台獨|台獨]]革命，結果事敗、匪徒逃亡；五人被當局逮捕後，雖飽受嚴刑審判，但未供出任何共謀，最後在[[蔣介石|蔣介石]]批示下，於同年5月30日遭集體[[槍決|槍決]]<ref name="泰源事件"/><ref name="追思泰源五烈士 藍綠政要出席"/>。

== 設施 ==
泰源技能訓練所在建築方面主要區分為行政區及戒護區；戒護區由各作業工場及各職類訓練班所建構，作業工場採「田」字型建築，全所面積19.0231公頃<ref name="歷史沿革"/>。場舍計有獨居房60間、雜居房345間、懇親宿舍2間、病舍1間、工場12間、技訓班6間。收容對象為強制工作受處分人兼收容刑期7年以上之製造、運輸、販賣、持有及兼施用毒品之男性受刑人暨保安處分刑前強制工作免予繼續執行或期滿後，接續執行竊盜或贓物罪所處刑罰之男性受刑人，核定容額為1,620名<ref name="歷史沿革"/>。
