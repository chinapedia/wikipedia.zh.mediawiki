《'''法越凡爾賽條約'''》（或《'''越法凡爾賽條約'''》），是由[[越南|越南]]歷史上與[[法國|法國]]簽訂的第一個[[不平等條約|不平等條約]]<ref>[http://www.cnki.com.cn/Article/CJFDTOTAL-XSLT198103016.htm 關於《越法凡爾賽條約》的問題]，《學術論壇》，1981年03期，楊萬秀</ref>，簽訂於西元1787年11月28日。條約由[[法國|法國]][[天主教|天主教]][[巴黎外方傳教會|巴黎外方傳教會]]傳教士[[百多祿|百多祿]]代表阮王[[阮福映|阮福映]]（簽約時任，拒約時為越南／大越南帝）與法王[[路易十六|路易十六]]簽訂。

== 源由經過 ==
西元16世紀資本主義時期，[[大越|大越]]帝國（[[後黎朝|後黎朝]]政府及割據北部的[[莫朝|莫朝]]政府）與[[法蘭西王國|法蘭西王國]]（[[瓦盧瓦王朝|瓦盧瓦王朝]]／[[波旁王朝|波旁王朝]]）發生聯繫。西元1777年，盤踞大越帝國南部二百餘年的半獨立政權'''阮王'''（[[阮主|阮主]]，外國稱之'''廣南'''）直系家族為[[鄭主|鄭主]][[黃五福|黃五福]]軍及西山[[阮岳|阮岳]]軍聯合攻滅；第九任阮主兼第二任阮王[[阮福淳|阮福淳]]之姪[[阮福映|阮福映]]（阮王旁系）被推為大元帥，後於西元1780年稱王（第四任阮王）。西元1782年，阮福映於七岐江水戰為西山王國（[[西山朝|西山朝]]）軍所敗退守[[富國島|富國島]]，西元1783年企圖遣使隨百多祿到法國求援而不順風未果，輾轉流亡至[[暹羅|暹羅]]王國；西元1784年在暹羅國王[[拉瑪一世|拉瑪一世]]遣軍支援下，於[[瀝涔吹蔑之戰|瀝涔吹蔑之戰]]再度為西山軍擊敗奔逃至暹羅。西元1785年，年僅5歲的[[阮福景|阮福景]]（阮福映次子）隨百多祿前往[[法屬印度|法屬印度]]首府本地治里（今[[印度|印度]][[本地治里|本地治里]]聯邦屬地[[本地治里市|本地治里市]]）求援不成；西元1786年離開法屬印度首府本地治里而於西元1787年2月抵達法都[[巴黎|巴黎]]會晤法王。<ref name=chapuis>[http://books.google.com/books?id=Jskyi00bspcC&pg=PA175&dq=Pigneau+de+Behaine&sig=ACfU3U3rEEjyRtTiuf0SWt6zZ7UUakKZ4Q#PPA175,M1 A History of Vietnam by Oscar Chapuis, p.175]</ref>
同年11月28日，法阮雙方簽訂《法越凡爾賽條約》。<ref name=dragon>[http://books.google.com/books?id=YgA1kBqOZYgC&pg=PA86&dq=Louis+XVI+Vietnam&sig=ACfU3U2aXpDtJUqyiUsOzS37SKDsw6IsaQ#PPA86,M1 ''Dragon Ascending'' by Henry Kamm p.86]</ref>條約訂明法國派兵援助阮福映，並割讓越南的[[峴港|沱㶞港]]（峴港）和[[昆侖島|昆侖島]]。但是法國並沒有實質上幫助阮福映（这也造成越法后期的争议：法国仍然提出割让；而越南由于没有得到实质援助，拒绝割让任何领土）。因此阮福映是实质靠自己灭西山平定全國。

== 條約概述 ==
法蘭西王國：
派遣四艘軍艦、1900名士兵支援

阮王（廣南／阮主）：
割讓沱㶞（今越南[[峴港|峴港]]）、[[崑崙島|崑崙島]]給法國，並給予法國人貿易特權

== 後續形勢 ==
西元1787年12月，百多祿一行乘坐Dryade號軍艦離開法國本土，前往法屬印度首府本地治里，準備借道歸國。但27日行至本地治里時，得知法國本土發生極為嚴重的財政危機，政府瀕臨破產。法屬印度總督康韋伯爵以此為由拒絕給予援助。百多祿便通過自己的關係，招募一批法國軍官並購買一些法國武器、彈藥和軍艦。

與此同時，阮主舊領支持者通知阮福映關於西山王國內訌的情勢。阮福映見機不可失，留書辭別暹羅國王拉瑪一世並率軍東進，經過一番爭戰經營，逐漸穩固其在嘉定（今越南[[胡志明市|胡志明市]]）地區的勢力。西元1789年7月14日，百多祿與阮福景乘坐法國軍艦Méduse號，攜帶法式鎗枝彈藥返回嘉定。隨行的還有法國海軍軍官[[讓-巴蒂斯特·沙依諾|讓-巴蒂斯特·沙依諾]]、戴福桑（de Forçant）、[[讓-馬里·達約|讓-馬里·達約]]等人。阮福映任命這些人為軍官，大規模鑄造西式兵器、軍艦，阮王國實力再次壯大。根據西方史料的記載，法國Dryade號軍官[[奧利維耶·德·皮曼紐爾|奧利維耶·德·皮曼紐爾]]叛逃到阮王國，為阮福映訓練步兵和炮兵，引進西方的先進步兵思想；而原為法國海軍軍官的沙依諾、達約則負責訓練海軍。至西元1790年，阮王國陸軍中已有五萬名穿[[英|英]]式制服、行[[歐洲|歐洲]]式軍禮、裝備西式先進武器並掌握西式戰爭技術的本土士兵；到了西元1792年時，阮王國海軍已有二艘歐式軍艦和十五艘[[護衛艦|護衛艦]]。至此，阮王國無論陸軍和海軍實力都已經比二阮帝國（西山王阮岳與其弟[[阮惠|阮惠]]於西元1788年先後稱帝建國）軍強大，擁有絕對優勢。

歷經約三年（西元1790年至西元1792年）與阮岳帝國的反覆爭戰後，該國為[[阮光纘|阮光纘]]（阮惠後繼者）帝國南援軍併吞。再經西元1792年至西元1802年期間的接連戰爭，阮光纘帝國退據北河（又名[[塘外|塘外]]）地區，阮福映建立南越帝國（[[阮朝|阮朝]]）後約二個月攻滅阮光纘帝國。

== 關於履約（與法關係） ==
百多祿試圖讓阮福映與西方國家接近並貿易，但隨著西元1799年百多祿的死而以失敗告終。法國皇帝[[拿破崙一世|拿破崙一世]]有征服越南帝國（又稱'''大越南'''，西元1804年由南越帝阮福映改稱）的野心，他希望將越南作為法屬遠東殖民地以之抗衡[[英屬印度|英屬印度]]；但拿破崙忙於歐洲戰爭，無暇東顧，後來亦隨拿破崙被流放而不了了之。

法國波旁王朝復辟之後，法國首相[[阿尔芒·埃马努埃尔·德·维涅罗·迪·普莱西|阿尔芒·埃马努埃尔·德·维涅罗·迪·普莱西]]（第五代[[黎塞留公爵|黎塞留公爵]]）派Cybele號來到沱㶞港，向越南帝國贈送一艘配備52門炮的四級風帆戰列艦作為禮物，還轉交法國國王[[路易十八|路易十八]]的國書，希望越南皇帝阮福映踐行他在《法越凡爾賽條約》中的承諾；但阮福映認為自己並沒有得到法國的實際援助，故而拒絕法國的要求，卻其國書並退回法國的贈禮。
