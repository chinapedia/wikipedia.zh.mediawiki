'''法蘭士·約瑟夫冰川'''<ref>{{cite book|title=《[[外國地名譯名手冊|外國地名譯名手冊]]》（中型本）|author=中國地名委員會编|publisher=北京：商務印書館|year=1993年|isbn=7-100-00798-4|pages=第247頁}}
</ref>（[[英語|英語]]：'''{{lang|en|Franz Josef Glacier}}'''；[[毛利語|毛利語]]：'''{{lang|mi|Ka Roimata o Hinehukatere}}'''），是一個位於[[紐西蘭|紐西蘭]][[南島|南島]][[西部地區|西部地區]]的冰川。連同在冰川以南20公里的[[福克斯冰川|福克斯冰川]]，兩個冰川從[[南阿爾卑斯山脈|南阿爾卑斯山脈]]開始，直到海拔240米高的地方融化，潤澤當地的溫帶雨林。

兩個冰川的周邊地帶都屬於[[世界自然遺產|世界自然遺產]]，位於[[蒂瓦希普纳穆|蒂瓦希普纳穆]]內的[[西部區國家公園|西部區國家公園]]（{{lang|en|Westland NP}}）。而從法蘭茲約瑟冰川結束處開始的河流名為[[懷霍河|懷霍河]]（{{lang|en|Waiho}}）。

法蘭士·約瑟夫冰川的另一個特色，是由於它非常接近海洋，所以冰川所在地的氣溫亦比地球上其他冰川更溫暖。

== 命名 ==
法蘭士·約瑟夫冰川由一位德國的探險家[[尤利烏斯·馮·哈斯特|尤利烏斯·馮·哈斯特]]（{{lang|de|Julius von Haast}}）在1865年命名，用以紀念[[奧匈帝國|奧匈帝國]]的君主[[弗朗茨·約瑟夫一世|弗朗茨·約瑟夫]]。這個冰川的[[毛利語|毛利語]]名稱是「{{lang|mi|Ka Roimata o Hinehukatere}}」，意思就是「Hinehukatere的眼淚」。這個名稱源自當地的一個傳說：喜愛攀山的Hinehukatere游說了她的愛人Tawe與她一起攀山。雖然Tawe的攀山經驗沒有Hinehukatere的豐富，但他愛與Hinehukatere結伴，所以答應了。結果，Tawe被[[雪崩|雪崩]]從山頂掃下山至死，Hinehukatere十分傷心。Hinehukatere的眼淚化成了萬千水點從山頂落下，凍結了成為冰川。

== 增長與減退 ==
現時冰川有12公里長，在距離[[塔斯曼海|塔斯曼海]]19公里處完結。由於在冰川結束處已融化的冰雪與降雪所形成的{{link-en|永久雪|névé}}（{{lang|en|névé}}，又譯[[冰川雪|冰川雪]]）未必經常保持平衡，所以冰川經常都在增長與減退的週期裡循環。在2007年，由於降雪量的增加，法蘭士·約瑟夫冰川成為了紐西蘭當年唯一增長中冰川。其他地方，特別是位於南阿爾卑斯山脈東部的冰川，都因為[[全球暖化|全球暖化]]的效應而大幅縮減<ref>''[http://www.nzherald.co.nz/location/story.cfm?l_id=500592&objectid=10476881 Shrinking glaciers near crisis]'' - ''[[The_New_Zealand_Herald|The New Zealand Herald]]'', Monday 19 November 2007</ref>。

從1940年代到1980年代期間，冰川處於減退期，冰川比現在最多曾減退數公里。從1984年開始，冰川步入增長期，大約以每日0.7米的幅度增長，流量是一般冰川的10倍速度。長遠來說，冰川相比於上一個[[冰河時期|冰河時期]]已退縮了很多。地理學家估計，在1萬到1.5萬年前的冰河時期，冰川曾一度伸入海裡。

== 旅遊 ==
[[File:FranzJosefNZ_gobeirne.jpg|center]]村落的一個旅客服務中心。]]
冰川地帶是紐西蘭西岸的主要旅遊景點，旅客可以在無需導遊領導下踏上冰川。不過若旅客希望攀爬冰川的話，需要有[[冰斧|冰斧]]、[[冰鑿|冰鑿]]等特殊用具及穿着特別的靴具。
