《'''法外狂徒'''》是2003年的澳大利亚傳記片，該片是根據《{{Internal link helper/en|我們的陽光|Our Sunshine}}》改編， 《{{Le|我們的陽光|Our Sunshine}}》是作家澳洲{{Internal link helper/en|罗伯特·德鲁|Robert_Drewe}}1991年寫的小說。由约翰·迈克尔·麦克唐纳改編成電影劇本。電影描述 [[奈德·凱利|奈德·凱利]]戲劇化的人生，他出生在[[維多利亞州|维多利亚]]，並且是當地非常猖狂的亡命之徒。 

在電影中，奈德·凱利、他的弟弟丹·凱利、與兩名同夥，史蒂夫哈特和乔伯恩結夥成了幫派。故事背景是關於19世紀愛爾蘭人與英國人在澳大利亞的緊張局勢。該片由[[希斯·萊傑|希斯·萊傑]]主演，並由[[奥兰多·布鲁姆|奥兰多布鲁姆、]][[妮奥米·瓦兹|娜歐蜜·華茲]]、[[杰弗里·拉什|傑佛瑞·洛許]]飾演[[奈德·凱利|奈德·凱利]]的同夥。<ref>{{Citation|title=Ned Kelly (2003)|url=http://www.imdb.com/title/tt0277941/|accessdate=2017-07-22}}</ref><ref>{{Citation|title=Ned Kelly|url=https://www.rottentomatoes.com/m/1131148_ned_kelly|language=en|accessdate=2017-08-27}}</ref>

== 故事大綱 ==
[[奈德·凱利|奈德·凱利]]如何從一為無辜的農夫，如何成為澳洲歷史上知名的法外狂徒。<ref>{{Citation|title=Ned Kelly (2003)|url=http://www.imdb.com/title/tt0277941/|accessdate=2017-08-27}}</ref>

== 劇情 ==
在電影的開頭，年輕的奈德·凱利救了一個溺水的男孩，然後場景轉換到澳大利亞的叢林裡，奈德·凱利與他父親談話。1871年，奈德·凱利在澳洲邊疆地區醒來，看到一匹白馬，他騎馬去到鎮上卻被逮捕，隨後被以竊盜罪起訴，其後被判五年有期徒刑，但事實上，那匹白馬是他的好友懷特偷的。

兩年後，[[奈德·凱利|奈德·凱利]]（[[希斯·萊傑|希斯·萊傑]] 飾演）被假釋回家，家人熱烈歡迎他回來。奈德·凱利的家庭是天主教的爱尔兰家庭，他們靠養馬還有經營農場維生。某天晚上在酒吧，一位當地的[[維多利亞州|维多利亚巡警]] 菲茨帕翠克借請喝酒的名義向奈德·凱利的妹妹搭訕。

菲茨帕翠克窮追不捨地向凱特示好，但凱特表示她不想喝。菲茨帕翠克召來其他警員打算用強硬手段逼凱特陪酒時，奈德凱立阻止了他們。菲茨帕翠克尾隨奈德凱立回家並取走了他家飼養的馬。奈德凱立夥同他弟弟丹，還有他的朋友史蒂夫·哈特（[[菲利普·瓦尼埃|菲利普·巴藍帝尼]] 飾演）、喬伯恩（[[奧蘭多布魯姆|奧蘭多布魯姆]]飾演）、亞倫·謝里特（喬爾・埃哲頓飾演）又把馬偷了回來。 
有一天晚上，菲茨帕翠克去了凱立家，而奈德凱立正好外出。  菲茨帕翠克恐嚇屋裡的人說，他將以竊盜罪逮捕他們，雙方一言不合打了起來，因為對方人數優勢，菲茨帕翠克落荒而逃，但他返回警局並指控根本不在場的奈德凱立開槍射他。

因為奈徳凱立不在家，警方逮捕了他的母親。奈德凱立、丹、喬伯恩和史蒂夫開始逃亡。稍後，他們一行人在樹林中遇到三名警察，最後迫不得已射殺了這些警察。數個月以來,凱立為了躲避追捕、四處流浪、餐風宿露。最後政府上層派來警長弗朗西斯·哈爾（[[杰弗里·拉什|傑佛瑞·洛許]] 飾演），他逮捕了許多人，而與凱立黨關係密切的亞倫也在其中。在審訊中，警方告訴亞倫：他們只想要逮捕凱立，警方不想傷害無辜的人。為了好友喬伯恩的安全，亞倫供出了凱立黨的位置。喬伯恩提早發現了危機，後來凱立黨處決了亞倫，因為他是個告密的叛徒。隔天，凱立黨到達維多里亞並佔領了這個小鎮，挟持了居民當做人質，但他們也贏得了民心。

其中一個人質逃跑，阻止了火車意外，鎮壓的部隊包圍了凱立黨，凱立黨穿著鐵桶當盔甲朝敵人射擊，雙方爆發激烈衝突，但由於火力差距，凱立黨被逼退躲回小屋。喬伯恩拿起一個酒杯時，酒杯被子彈打破，隨後他也中彈身亡。
丹和史蒂夫用盡了他們的彈藥，知道大勢已去，兩人坐在樹下開槍自殺。奈德·凱利受了重傷，但依然靠著意志苦撐繼續向警方開槍。他最後中彈倒在地上。奈德·凱利被抬上火車。
雖然有超過3萬人聯署請求赦免奈德·凱利，但最後奈德·凱利依然被被判死刑。1880年11月11日，奈德凱立被處以絞刑。<ref>{{Citation|title=Ned Kelly|url=https://www.rottentomatoes.com/m/1131148_ned_kelly|language=en|accessdate=2017-07-22}}</ref>

== 製作 ==
[[奈德·凱利|奈德·凱利]]這個題材被拍了很多次，最早的版本是1906年由法蘭克·米斯演的《{{Internal link helper/en|凱利幫的故事(The Story of the Kelly Gang)|The_Story_of_the_Kelly_Gang}}》<ref>{{Citation|title=The Story of the Kelly Gang (1906)|url=http://www.imdb.com/title/tt0000574/|accessdate=2017-08-27}}</ref>　，後來又陸續有許多部相關的電影像《{{Internal link helper/en|奈德凱利 (1970年電影)|Ned Kelly (1970 film)}}》、《{{Internal link helper/en|最後的法外之徒(The Last Outlaw 1980|The_Last_Outlaw_(miniseries)}})》等。但導演馬修·霍姆斯認為[[奈德·凱利|奈德·凱利]]的歷史故事，從未被正確地描述過，所以打算再拍一部。他不認為其他電影拍得不好，但導演覺得應該更關注[[奈德·凱利|奈德·凱利]]的好與惡以及警察的黑暗面。為了拍攝電影，馬修·霍姆斯舉辦募款活動，希望募得兩百五十萬美元。<ref>{{Cite news|url=http://www.forbesadvocate.com.au/story/4712216/wanted-25m-for-definitive-ned-kelly-film/|title=Wanted: $2.5m for 'definitive' Ned Kelly film|last=Maddox|first=Garry|date=2017-06-06|newspaper=Forbes Advocate|accessdate=2017-08-27|language=en}}</ref>
