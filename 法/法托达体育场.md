'''法托達體育場'''（{{lang-en|'''Fatorda Stadium'''}}）正式名稱為'''賈瓦哈拉爾·尼赫魯體育場'''（{{lang-en|'''Jawarharlal Nehru Stadium'''}}），是一座位於[[印度|印度]][[果阿邦|果阿邦]][[馬爾加奧|馬爾加奧]]{{le|法托達|Fatorda}}的{{le|多功能體育場|multi-purpose stadium}}，於1989年完工。法托達體育場是果阿邦內唯一一座舉辦過國際級[[足球|足球]]與[[板球|板球]]比賽的體育場，球場內有19,000個座椅，目前由{{le|果阿体育局|Sports Authority of Goa}}管理，並曾舉行多場國家級足球及板球賽事。<ref name="Venue"/>

法托達體育場目前是多家印度足球俱樂部的主場，包括[[邱吉爾兄弟體育俱樂部|邱吉爾兄弟]]、[[丹波體育俱樂部|丹波]]、[[薩爾高卡足球俱樂部|萨尔高卡]]、[[果阿競技俱樂部|果阿競技]]、[[果阿邦足球會|果阿邦]]。<ref name="Venue"/>

==歷史==

===建造===
法托達體育場於1989年完工，建造時曾以工作天數六個月，打破以最短時間建造完工的體育場紀錄。在1989年開放時，原本只有當作足球場使用，但過沒多久便重新裝修，並開始舉行[[板球|板球]]比賽。1989年10月25日，體育場舉辦首場{{le|單日國際板球賽|One Day International}}，比賽隊伍為[[澳大利亞國家板球隊|澳大利亞]]對{{le|斯里蘭卡國家板球隊|Sri Lanka national cricket team|斯里蘭卡}}。<ref name="cricket"/>

===足球俱樂部主場===
2006年，在{{le|2007–08年印度足球甲級聯賽|2007–08 I-League}}賽季開始前，[[丹波體育俱樂部|丹波]]、[[邱吉爾兄弟體育俱樂部|邱吉爾兄弟]]、[[薩爾高卡足球俱樂部|萨尔高卡]]、[[果阿競技俱樂部|果阿競技]]四家俱樂部宣稱將定法托達體育場為其主場。

2014年，於[[印度超級聯賽|印度超級聯賽]]的[[果阿邦足球會|果阿邦]]宣佈將定法托達體育場為其主場。

==翻修==


2014年，根據[[國際足總|國際足總]]公佈的最新規格，法托達體育場開始進行翻修工程。新體育場將有19,800個座椅與全看台遮蓬，讓觀眾能在寬闊與舒適的狀況下觀賽。體育場另將座椅席次分為兩種級別，較高及別的區域被稱為VVIP區域，提供觀眾能毫不受他人干擾，並舒適的私人空間來觀賽。體育場另有高亮度球場照明、天然草皮、廣播室、電視轉播室、球員更衣室、比賽代表區、音效控制室、醫護室、貴賓休息室、團體席、媒體轉播與工作站、新聞發佈區、混合采訪區、網路監控室、游泳池、多功能健身房及停車場等。該體育場被認為是印度半島上，設施最好的體育場。<ref name="Venue2">{{cite web|title=Indian Super League Stadiums – A Walkthrough (Part-2) |url=http://indiansuperleaguefootball.in/indian-super-league-stadiums-a-walkthrough-part-2/ |publisher=Indian Super League |date=18 Sep 2014 |deadurl=yes |archiveurl=https://web.archive.org/web/20140922034958/http://indiansuperleaguefootball.in/indian-super-league-stadiums-a-walkthrough-part-2/ |archivedate=2014-09-22 }}</ref>

==足球比賽==
法托達體育場經常舉行國際級賽事，包括參加[[世界盃足球賽|世界盃足球賽]]與[[亞足聯亞洲盃|亞足聯亞洲盃]]資格賽的[[印度國家足球隊|印度國家足球隊]]等賽事，是印度國內一座重要的體育場<ref name="Venue2"/><ref>{{cite web |url=http://www.fifa.com/worldcup/preliminaries/asia/matches/round=250425/match=300033484/report.html |title=2010 FIFA World Cup Asia Group Playoffs Match Report |publisher=[[FIFA|FIFA]] |accessdate=2010-01-10}}</ref>。體育場並且是[[丹波體育俱樂部|丹波]]、[[邱吉爾兄弟體育俱樂部|邱吉爾兄弟]]、[[薩爾高卡足球俱樂部|萨尔高卡]]、[[果阿競技俱樂部|果阿競技]]、[[果阿邦足球會|果阿邦]]的主場，經常舉行[[印度足球甲級聯賽|印度足球甲級聯賽]]與[[印度超級聯賽|印度超級聯賽]]的賽事<ref name="Venue2"/>。

==板球比賽==


雖然法托達體育場原本目的是作為一座足球場地，但在過去幾年中，體育場開始舉辦越來越多地國際級板球比賽。自1989年舉辦過[[澳大利亞國家板球隊|澳大利亞]]對{{le|斯里蘭卡國家板球隊|Sri Lanka national cricket team|斯里蘭卡}}的{{le|單日國際板球賽|One Day International}}後，體育場已經舉辦共7次單日國際板球賽，最近一次舉辦的單日國際板球賽是在2007年，由{{le|印度國家板球隊|India national cricket team|印度}}對斯里蘭卡<ref name="cricket">{{cite web |url=http://stats.cricinfo.com/india/engine/records/team/match_results.html?class=2;id=1029;type=ground
|title=Nehru Stadium, Fatorda, Margao / Records / One-Day Internationals |publisher=Cricinfo|accessdate=2010-01-10}}</ref>。 

由於體育場經常性地舉辦板球比賽，導致時常與足球比賽相衝突，使得體育場有時無法舉行足球比賽。

==葡語系運動會==
法托達體育場在2014年進行整修，以迎接於[[果阿|果阿]]舉行的2014年[[葡語系運動會|葡語系運動會]]<ref name="Venue2"/><ref>{{cite web |url=https://lusofoniagamesgoa2014.wordpress.com/|title=India’s hunger for medals continues at Lusofonia Games 2014 |publisher=Lusofonia Games, Goa-2014|date=January 23, 2014}}</ref>，這屆運動會的開幕式與閉幕式，皆在此舉行<ref>{{cite web |url=http://timesofindia.indiatimes.com/sports/more-sports/others/Lusofonia-Games-opening-ceremony-venue-inaugurated/articleshow/28969605.cms|title=Lusofonia Games opening ceremony venue inaugurated|publisher=Time of Indian|date=January 18, 2014}}</ref><ref>{{cite web |url=http://www.newswala.com/Sports-News/Lusofonia-Games-2014-comes-to-an-end-56481.html |title=Lusofonia Games 2014 comes to an end |publisher=News Wala |date=January 31, 2014}}</ref>，體育場並舉辦本屆運動會的足球比賽<ref>{{cite web |url=http://www.chrispd.de/wordpress/2014/01/19/lusofonia-games-release-revised-football-fixtures/|title=Lusofonia Games release revised football fixtures |publisher=chrispd.de |date=January 19, 2014}}</ref>。
