'''法式鹹派'''（{{lang-fr|quiche}}）又稱'''洛林鄉村鹹派'''、'''洛林鹹派'''或簡譯作'''蛋批'''<ref>見[[Pacific_coffee|Pacific coffee]]的菜單。</ref>，是以雞蛋揉合牛奶或鮮奶油製成的[[糕點|糕點]]，為法國傳統爐烤佳餚。它的派皮通常先經[[盲烤|盲烤]]，再加入其他食材。熟煮的碎肉、蔬菜或起司等食材，則在送入烤爐前混入蛋液中，一同加熱。法式鹹派沒有上層派皮，被歸類為開放式[[西式餡餅|餡餅]]，但它可以以番茄切片或派邊餡料裝飾、豐富菜色。依當地風俗或個人習慣不同，法式鹹派在早餐、午餐或晚餐皆有人食用。

== 歷史 ==
雖然法式鹹派現為法國飲食文化中的經典佳餚，它的外文名「quiche」其實是源自德語的「Kuchen」，意為糕點<ref>{{cite web |url=http://www.foodreference.com/html/artquiche.html |title=Quiche Origins |work=Foodreference.com |accessdate=2011-09-09}}</ref>。自古以來，洛林北部第三區的居民便常使用[[洛林法蘭克|洛林法蘭克]]地區的德語方言，「Kuchen」因此在當地漸漸演變為「küche」。如同典型的[[阿勒曼尼語|阿勒曼尼語]]化形變，它隨後捨去「ü」（/y/），並將擦音「ch」（/ç/）轉變為「sh」（/ʃ/），形成「kische」，最後以標準法文拼寫為「quiche」，沿用至今<ref>{{cite web |work=Etymonline.com |url=http://www.etymonline.com/index.php?search=quiche&searchmode=none |title=Online Etymology Dictionary: quiche |accessdate=2011-09-09}}</ref>。
