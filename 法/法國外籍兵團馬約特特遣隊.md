'''法國外籍兵團馬約特特遣隊'''（法文：Détachement de Légion étrangère de Mayotte）為法國外籍兵團的一員，駐紮於法國海外領土[[馬約特|馬約特]]的[[藻德濟|藻德濟]]市。其工作為確保[[馬約特|馬約特]]的主權、警備其國境、保護[[法國|法國]]於此的權益以及實施島嶼戰的訓練。

== 沿革 ==
* 1956年：由馬達加斯加外籍步兵營中分出。
* 1962年：[[法國外籍兵團第3外籍步兵团|第3外籍步兵團]]駐留馬達加斯加，又另外分出一部分。
* 1975年：[[葛摩|葛摩]]宣布獨立，但[[馬約特|馬約特]]島仍屬於[[法國|法國]]。
* 1976年：馬約特特遣隊編制。
* 1984年：[[法國外籍兵團第1外籍騎兵團|第1外籍騎兵團]]又分出一部分隊員至本部隊（為最後一次此狀況）。
* 1992年：[[葛摩|葛摩]]危機爆發，部隊陷入警戒狀態，直到[[GIGN|GIGN]]介入後情況才告緩和。
[[File:DLEM_tir.JPG|250px]]
== 編制 ==
* 特遣隊本部
* 本部管理連
* 綜合戰鬥連（使用個人與小隊武器，特化為島嶼戰專門的步兵連）

* 本分遣隊由[[法國外籍兵團|法國外籍兵團]]成員混合編成，約250名成員，每4個月輪調。
