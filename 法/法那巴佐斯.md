'''法那巴佐斯''' （{{lang-el|Φαρνάβαζος }}; 約在前370年 — 約前320年），為[[亞歷山大大帝|亞歷山大大帝]]東征時的波斯[[阿契美尼德帝國|阿契美尼德帝國]]的總督，之後投降亞歷山大。 

== 早年 ==
法那巴佐斯是波斯[[赫勒斯滂弗里吉亞|赫勒斯滂弗里吉亞]]總督[[弗里吉亞的阿爾塔巴左斯|阿爾塔巴左斯]]之子。在前358年其父親和其他總督發動對[[阿爾塔薛西斯三世|阿爾塔薛西斯三世]]的叛變，但阿爾塔巴左斯的勢力終究被國王軍擊敗，於是阿爾塔巴左斯和法那巴佐斯等人逃到[[馬其頓王國|馬其頓王國]]，投靠[[腓力二世_(馬其頓)|腓力二世]]，並在馬其頓結識了少年的亞歷山大。與法那巴佐斯一同逃到馬其頓避難的一行人中，還包括與法那巴佐斯有姻親關係的[[羅德島的門農|門農]]。

法那巴佐斯等人之後因門農的兄長[[羅德島的門托耳|門托耳]]向阿爾塔薛西斯三世求情，最後得以返回波斯帝國。

== 與亞歷山大對抗 ==
當馬其頓的亞歷山大大帝繼承其父親腓力二世的遺志，開始朝亞洲進軍，並在[[格拉尼庫斯河戰役|格拉尼庫斯河戰役]]和[[哈利卡那索斯圍城戰|哈利卡那索斯圍城戰]]擊敗波斯的軍隊。為此，法那巴佐斯輔佐已是妹婿的門農統領波斯海軍，並且計畫奪取[[愛琴海|愛琴海]]諸島來迫使希臘諸城邦倒向波斯這一方，也威脅亞歷山大的補給線。門農和法那巴佐斯率領波斯艦隊成功奪下[[希俄斯島|希俄斯島]]，接著又前往[[萊斯博斯島|萊斯博斯島]]進攻，這時島上僅剩[[米蒂利尼|米蒂利尼]]尚未落入波斯軍手中，而[[雅典|雅典]]的反馬其頓派政治家[[狄摩西尼|狄摩西尼]]也開始鼓吹雅典與其他城邦反叛，這時[[斯巴達|斯巴達]]國王[[亞基斯三世|亞基斯三世]]也正在籌備與馬其頓的戰爭。然而門農在圍攻米蒂利尼時病逝，並把艦隊統帥權暫時交給法那巴佐斯<ref>[[阿利安|阿利安]] 《亞歷山大遠征記》卷二 p.057 </ref>。

在[[奧托夫拉達提斯|奧托夫拉達提斯]]的協助下，法那巴佐斯攻占了米蒂利尼和[[忒涅多斯島|忒涅多斯島]]，掌控[[赫勒斯滂|赫勒斯滂]]。法那巴佐斯更在靠近[[哈利卡那索斯|哈利卡那索斯]]處建造要塞，使亞歷山大的補給線更加岌岌可危，法那巴佐斯還攻占了[[薩莫色雷斯島|薩莫色雷斯島]]、[[錫弗諾斯島|錫弗諾斯島]]和[[安德羅斯|安德羅斯]]，並且截擊亞歷山大的補給船。

在前333年當斯巴達國王亞基斯三世與法那巴佐斯在錫弗諾斯島協商軍費以及對抗希臘本土馬其頓勢力的事務時，傳來波斯國王[[大流士三世|大流士三世]]在[[伊蘇斯戰役|伊蘇斯戰役]]戰敗的消息，法那巴佐斯立即打消入侵伯羅奔尼撒的計畫，並且回到希俄斯島鎮住反叛的希臘人。當亞歷山大轉向進攻[[腓尼基|腓尼基]]諸城，使腓尼基和賽普勒斯的艦隊支持亞歷山大，法那巴佐斯更加孤立無援。隨著亞歷山大聲勢越加浩大，法那巴佐斯治下的[[愛琴海|愛琴海]]諸島紛紛反叛，當地希臘人並引進馬其頓軍隊，於希俄斯島俘虜法那巴佐斯，但法那巴佐斯在關押期間偷偷逃走<ref>[[阿利安|阿利安]] 《亞歷山大遠征記》卷三 p.093 </ref>。

== 後期 ==
關於法那巴佐斯逃亡後的那段時期沒有相關記錄，但最終法那巴佐斯至少在前324年前投降[[亞歷山大大帝|亞歷山大大帝]]，在那一年亞歷山大在[[蘇薩|蘇薩]]舉辦集體婚禮，並把法那巴佐斯的兩個姊妹分別許配給[[歐邁尼斯|歐邁尼斯]]和[[托勒密一世|托勒密]]<ref>[[阿利安|阿利安]] 《亞歷山大遠征記》卷七 p.243 </ref>。當亞歷山大逝世後，爆發[[繼業者戰爭|繼業者戰爭]]，法那巴佐斯加入歐邁尼斯一方，並在前321年歐邁尼斯與[[克拉特魯斯_(繼業者)|克拉特魯斯]]、[[涅俄普托勒摩斯_(將軍)|涅俄普托勒摩斯]]作戰時，法那巴佐斯受命率領一支騎兵分隊，並在那戰役中歐邁尼斯獲得勝利<ref>[[普魯塔克|普魯塔克]]，《希臘羅馬英豪列傳:歐邁尼斯》 p.1051 </ref>。
