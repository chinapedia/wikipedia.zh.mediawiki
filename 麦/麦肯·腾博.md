'''麥肯·布莱·腾博'''（{{lang|en|Malcolm Bligh Turnbull}}，{{bd|1954年|10月24日|||catIdx=Turnbull}}，澳大利亞當地譯'''譚保'''），[[澳大利亚|澳大利亚]][[政客|政客]]，第29任[[澳大利亚总理|澳大利亚总理]]、[[澳大利亞自由黨|澳大利亞自由黨]]前黨魁。他曾是[[悉尼|悉尼]]东区[[溫特沃思_(澳大利亞國會選區)|溫特沃思]]选区的议员。曾於2008－2009年任澳大利亚反對黨領袖；2013-2015年担任通訊部長。

从政前他曾是大律师和投资银行家，身价过亿，曾是澳大利亚政界中的首富。<ref>{{cite news|author1=Murphy, Katharine|author2=Grattan, Michelle|authorlink2=Michelle Grattan|url=http://www.theage.com.au/national/all-a-bit-rich-as-wealthy-worlds-collide-in-the-battle-for-the-battlers-20080917-4ior.html|title=All a bit rich as wealthy worlds collide in the battle for the battlers|work=[[The_Age|The Age]]|location=[[墨爾本|Melbourne]], [[維多利亞州|Vic.]] |date=2008-09-18 |accessdate=2014-06-02 }}</ref>

2015年9月14日，腾博挑战时任总理自由党党魁艾伯特，在[[议会党团|议会党团]]表决中以54-44的票数胜出，并在2015年9月15日，获总督正式任命为澳大利亚第29任总理，至2018年反被「迫宮」下台。

== 早年经历 ==
麦肯·腾博的父亲是旅馆经纪人（即专营旅馆的地产经纪人），母亲是广播剧演员、作家和学者，腾博幼年家庭生活富裕，小学的后半段得以入读私立名小。麦肯·腾博10岁时父母离异，母亲离开澳大利亚前往[[新西兰|新西兰]]，后来移居美国，麦肯·腾博因此在单亲家庭中，由父亲抚养长大。他中学就读私校[[悉尼文法学校|悉尼文法学校]]，大学考入[[悉尼大学|悉尼大学]]，获[[文学士|文学士]]及[[法学士|法学士]]双重学位毕业。1978年，赢得[[罗德奖学金|罗德奖学金]]，赴英国[[牛津大學|牛津大學]][[牛津大學布雷齊諾斯學院|布雷齊諾斯學院]]攻读民法学士（即法学硕士）学位，1980年以荣誉学位毕业。

== 早期工作 ==
麦肯·腾博在大学时代就积极参与业余[[广播|广播]]和[[新闻|新闻]]工作，在《[[星期日泰晤士報|星期日泰晤士報]]》供职，并为美国和澳大利亚的刊物撰写稿件。

1980年回到澳大利亚后腾博成为[[讼务律师|讼务律师]]，1983年在[[凱利·帕加|帕加家族]]企业任职法律顾问。1986年腾博与布鲁斯·麦克威廉创建律师事务所，因出庭辩护被[[英國安全局|英國安全局]]（军情五处）控诉的前特工而声名大噪。

1987年与[[新南威爾士州|-{zh-cn:新南威尔士州; zh-hk:新南威爾斯州; zh-sg:新南威尔斯州; zh-tw:新南威爾斯州;}-]]前州长[[內維爾·懷恩|內維爾·懷恩]]以及[[高夫·惠特拉姆|高夫·惠特拉姆]]之子合伙创立投资银行，之后投资并经营了一系列企业。1991年腾博协助旧雇主帕加谋求收购澳大利亚另一传媒业巨头企业费尔法克斯（Fairfax）时，腾博获悉帕加准备在收购完成之后对其编辑方针进行干预，腾博为了保护新闻多元化而威胁将此计划公布，因此与帕加交恶，甚至两人互相威胁杀掉对方，最后腾博将帕加的计划告诉传媒监管机构，导致帕加被剔出收购财团。<ref>{{cite web |url = http://www.smh.com.au/national/brute-packer-threatened-to-kill-me-turnbull-20090603-bvsk.html |title = 'Brute' Packer threatened to kill me: Turnbull |work = Sydney Morning Herald |date=2009-06-04 |accessdate=}}</ref>1997年腾博加入[[高盛|高盛]]，后成为合伙人。

==投身政治==
与前总理[[陆克文|陆克文]]一样，腾博也是[[澳大利亚共和运动|澳大利亚共和运动]]的积极支持者。1993年至2000年期间，曾任澳大利亚共和运动组织的主席。1998年当选宪政会议民选代表，但由于选择支持建制派的「最小改变」共和方案（總統為虛位元首，由議會選出）在事后被共和派批评，认为他的选择导致这一缺乏选民支持的方案成为付诸公投的共和方案，因此导致了共和议题在公投中遭到否决。

[[File:Turnbull.JPG|thumb]]
1981年时麦肯·腾博曾在其居住的悉尼东区温特沃思选区寻求自由党提名，但在党内初选中被击败，之后他的党员身份逾期作废。2000年腾博再度加入自由党，在2002-3年担任新州党部干事并出任联邦党部财务部长。2004年，腾博再度寻求温特沃思选区自由党提名并获得提名，但在接下去的选举中，原来的自由党籍议员以独立议员身份参选，原为自由党稳得议席的温特沃思变成边缘选区，腾博仅以少量选票优势当选联邦议员。

2006年[[霍华德|霍华德]]总理提拔腾博入阁担任水利部长，2007年改为环境与水利部长。联盟党在2007年大选中失败，[[陆克文|陆克文]]领导的工党当政，霍华德本人失去议席，在接下来的党魁选举中腾博失败，但获委任为影子财政部长。2008年再度挑战党魁获得胜利，成为自由党党魁、反对党领袖。

== 反對黨領袖 ==
成為反對黨領袖后，腾博就一直与陆克文政府的政策唱反调，经常指责政府在应对[[2008年環球股災|经济危机]]时政策不力，而且极力批评执政党工党的经济刺激方案。惟2009年6月的统计数据显示：澳大利亚经济在2009年第一季度不跌反涨，增幅0.4%。陆克文和经济部长[[韦恩·斯万|韦恩·斯万]]称澳大利亚在工党政策之下，成功避免经济衰退，是唯一成功避免衰退的主要发达国家。<ref>{{cite news |url = http://news.bbc.co.uk/2/hi/business/8080446.stm |title = Australia able to avoid recession |publisher=BBC News|date=2009-06-03 |accessdate=2009-08-27}}</ref>

腾博任反对党领袖期间，工党经常利用他的富裕家庭背景攻击他，称他不知平民疾苦。腾博反驳时，曾指出他的父亲在离异后、家庭住所被母亲卖掉后，曾短期租房，以示他也经历过普通家庭的生活，但这种争辩反而受到对方嘲笑——和腾博相比，陆克文出身佃农，幼年父亲去世后，曾有被地主扫地出门，一家人在车里过夜的经历。<ref>{{cite news |url = http://www.theaustralian.com.au/archive/politics/im-no-silvertail-says-new-leader/story-e6frgczf-1111117505479 |title=I'm no silvertail, says new leader Malcolm Turnbull |publisher=The Australian |date=2008-09-17|accessdate=2015-09-15}}</ref>

2009年6月，腾博指称陆克文涉嫌腐败，不但接受了一名汽车销售商的赠车，而且还给这名销售商好处。他意借此机会使陆克文引咎辞职，未料到后来的调查中，腾博用来作为证据的电子邮件被证明是伪造的。此致腾博和反对党的支持率再次下滑。

同时，作为自由党内的进步派，腾博在环境议题上持有较激进的政见，因此愿意与陆克文政府合作，招来了联盟党内以托尼·艾伯特为首的保守派的不满。

2009年12月1日，艾伯特以炭税政策为契机，在党内发起领袖权挑战，腾博在随后的澳大利亚自由党议会党团内的领导权投票中，以一票之差败于[[托尼·艾伯特|托尼·艾伯特]]。<ref>{{cite web |url = http://www.smh.com.au/national/abbott-wins-liberal-leadership--by-one-vote-20091201-k1va.html |title = Abbott wins Liberal leadership - by one vote |publisher=Sydney Morning Herald |date=2009-12-01 |accessdate=2009-12-01}}</ref>

== 反對黨領袖之後 ==
[[File:Malcolm_Turnbull_-_Flickr_-_Eva_Rinaldi_Celebrity_and_Live_Music_Photographer_(1).jpg|thumb]]
2010年4月6日，腾博曾宣布不再角逐連任議員，并据说准备退出政坛，但最後又取消決定。

在[[2010年澳洲聯邦大選|同年的選舉]]中，他以11%的优势擊敗對手连任议员，並進入影子內閣任通訊部長。<ref>{{cite web |url = http://vtr.aec.gov.au/HouseDivisionFirstPrefs-15508-152.htm |title = House of Representatives Division First Preferences |publisher = [[Australian_Electoral_Commission|Australian Electoral Commission]] |date = 2010-09-15 |accessdate = 2010-09-20 |deadurl = yes |archiveurl = https://web.archive.org/web/20100905085133/http://vtr.aec.gov.au/HouseDivisionFirstPrefs-15508-152.htm |archivedate = 2010-09-05 }}</ref>腾博虽然屡次表示支持艾伯特，但作为党内偏左派系领军人，凡在保守派聯盟党魁艾伯特的支持率跌落时，即有评论是否腾博应当重夺党魁位置。

2012年7月，騰博表示支持[[民事結合|民事結合]]，其後又表示願意投票支持[[同性婚姻|同性婚姻]]，並且前提是聯盟黨魁同意聯盟黨員可就有關議題自由投票，可是保守派聯盟黨魁[[托尼·艾伯特|托尼·艾伯特]]並未容許黨員就有關議題自由投票（即只容許黨員投反對票），他又指出其他同性婚姻合法化的國家，先前都有民事結合。<ref>{{cite news |url = http://www.theage.com.au/federal-politics/political-news/turnbull-under-fire-for-gay-marriage-stance-20120707-21ngx.html |work=[[The_Age|The Age]] |title=Turnbull under fire for gay marriage stance |last = Peatling |first = Stephanie|date = 2012-07-07 |accessdate = 2013-09-30 |location = Melbourne}}</ref>

== 總理大臣 ==
由于艾伯特政府施政失误，執政期间其支持率不断下滑，執政黨更在多場地方選舉中慘敗，例如[[昆士蘭州|昆士蘭州]]自由國家黨虽然坐享压倒性多数议席仍在執政僅一屆後便下台，因此党内和党外都持续出现对艾伯特领袖地位的质疑声。

2015年9月14日，腾博宣布连同副党魁毕肖普挑战艾伯特。两人分别面见艾伯特，要求进行议会党团表决。当晚艾伯特举行表决，腾博以54-44的选票，在自由党党内选举中，击败[[托尼·艾伯特|托尼·艾伯特]]，成为自由党党魁，副党魁毕肖普亦成功連任。次日（2015年9月15日）接受[[澳大利亚总督|总督]]任命，取代艾伯特成为澳大利亚第29任總理大臣。<ref>{{cite web|url=http://www.abc.net.au/news/2015-09-14/malcolm-turnbull-beats-tony-abbott-in-liberal-leadership-ballot/6774546|title=Malcolm Turnbull topples Tony Abbott in Liberal leadership ballot|publisher=[[ABC|ABC]]|date=2015-09-14|accessdate=2015-09-14}}</ref>

2018年8月24日，时任內政部长[[彼德·達頓|彼德·達頓]]在{{en-link|2天内两度挑战党内领袖|Liberal Party of Australia leadership spills, 2018}}，在24日第二次[[议会党团|议会党团]]会议中表决中无法获得多数支持后，腾博宣布辭職並退出政壇。财政部长[[斯科特·莫里森|斯科特·莫里森]]在随后的党团竞选中击败[[彼德·達頓|达顿]]及外交部长[[朱莉·毕晓普|毕晓普]]，並即日接替他成为新一任自由黨领袖和總理。騰博於同月31日辭任議員，結束政治生涯。

== 個人生活 ==
[[File:Lucy_and_Malcolm_Turnbull_(6707565323).jpg|thumb]]
腾博1980年在[[牛津郡|牛津郡]]与[[露西·腾博|露西·腾博]]结婚。露西·腾博出身于澳大利亚世家休斯（Hughes）家族。父亲曾任联邦律政部长，曾祖父是第一任悉尼市市长。婚后，露西·腾博曾任[[悉尼市|悉尼市]]市长（Lord Mayor of Sydney）。两人育有一子一女：亚历山大（亚历克斯）和黛西。2013年，黛西诞下一子。亚历山大与妻子育有一女。

腾博曾屬[[長老宗|長老宗]]。婚后，2002年改宗為[[天主教|天主教]]，然而他對[[天主教會|天主教會]]就[[墮胎|墮胎]]、[[幹細胞|幹細胞]]研究和[[同性婚姻|同性婚姻]]議題的立場持反對意見。

2005年，麦肯·腾博与露西·腾博的财富据估算约为1.33亿澳元，<ref>{{cite news|author1=Murphy, Katharine |author2=Grattan, Michelle |authorlink2=Michelle Grattan |url = http://www.theage.com.au/national/all-a-bit-rich-as-wealthy-worlds-collide-in-the-battle-for-the-battlers-20080917-4ior.html |title=All a bit rich as wealthy worlds collide in the battle for the battlers |work=[[The_Age|The Age]] |location=Melbourne |date = 2008-09-18 |accessdate=2014-06-02}}</ref>因此他在当时是澳大利亚最富有的议员<ref>{{cite web |url = http://www.livenews.com.au/Articles/2008/09/24/The_politics_of_envy_and_the_actions_of_greed |title = The politics of envy and the actions of greed |work = Live News |date = 2008-09-24 |accessdate = 2010-04-27 |deadurl = yes |archiveurl = https://web.archive.org/web/20081216134540/http://www.livenews.com.au/Articles/2008/09/24/The_politics_of_envy_and_the_actions_of_greed |archivedate = 2008-12-16 }}</ref>，直到2013年矿业大亨、亿万富翁[[克莱夫·帕尔默|克莱夫·帕尔默]]当选议员。<ref>{{cite web |url = http://www.brw.com.au/p/entrepreneurs/clive_palmer_joins_elite_pressure_e6OlAjqPq6be7UVAlKO9cI |title = Clive Palmer joins elite club of rich-list politicians, but his fortune's under pressure |first=James |last=Thomson |date=2013-09-09|work=[[BRW_(magazine)|BRW]]}}</ref><ref>{{cite web |url=http://au.news.yahoo.com/a/20116606/clive-palmer-calls-for-company-tax-changes-in-maiden-speech-to/ |title=Clive Palmer calls for company tax changes in maiden speech to Parliament |first=Emma |last=Griffiths |date=2013-12-02 |publisher=Yahoo!7 |deadurl=yes |archiveurl=https://web.archive.org/web/20131230235623/http://au.news.yahoo.com/a/20116606/clive-palmer-calls-for-company-tax-changes-in-maiden-speech-to/ |archivedate=2013-12-30 }}</ref>

2010年，腾博曾是澳大利亚《商务评论周刊》200富豪榜中的第197名，比2009年的182名略低。当时他的个人财富被估算为1.86亿澳元，也是唯一入榜的政界人士。<ref>{{cite news |url = http://www.smh.com.au/executive-style/luxury/politicians-wealth-revealed-as-malcolm-turnbull-makes-rich-list-again-20100526-wd6y.html |title=Politicians' wealth revealed as Malcolm Turnbull makes rich list again |work=[[The_Sydney_Morning_Herald|The Sydney Morning Herald]] |date=2010-05-26 |accessdate = 2010-05-26 |first=Chris |last = Zappone}}</ref>但2014年腾博没有进入富豪榜的200强。<ref>{{cite web |url = http://www.brw.com.au/lists/rich-200/2014/ |title = BRW Rich 200 |work = [[BRW_(magazine)|BRW]] |date = 2014-06-27 |accessdate = 2014-12-26}}</ref>

=== 与中国的个人关系 ===
腾博的儿媳王怡文（Yvonne Wang）是华人，在香港长大，现居[[新加坡|新加坡]]，在旅游网站Tripadvisor（[[猫途鹰|猫途鹰]]）公关部门工作，其父亲是留学美国的中国国际关系学者王春明（Wang Chunming，音译），母亲是留学加拿大的音乐家涂崇玲（Tu Chongling，音译）。按照亚历克斯·腾博的说法，王春明在[[上海|上海]]拥有广泛人脉关系。腾博本人在1994年就曾在中國投資，在[[河北|河北]][[蔡家營|蔡家營]]創辦合資企業河北華澳礦業公司開發铅锌银矿，是中國大陸[[改革開放|改革開放]]之後的第一家中外合作礦山企業。由于腾博和中国的个人关系，以及过去对于中国的温和言论，上任伊始就在中国赢得好感，据澳大利亚媒体报道，中国媒体已开始将腾博昵称为“糖包”。<ref>{{cite web |url = http://www.news.com.au/finance/work/malcolm-turnbull-nicknamed-sugar-bomb-sweet-dumpling-in-china/story-fn5tas5k-1227530680995 |script-title=en:Malcolm Turnbull nicknamed ‘sugar bomb’, ‘sweet dumpling’ in China |work=News.com.au |date=2015-09-16 |accessdate=2015-09-18}}</ref><ref>{{cite web |url = http://www.smh.com.au/national/is-malcolm-turnbull-soft-on-china-because-of-his-family-connections-20150915-gjnbz8.html |title = Is Malcolm Turnbull 'soft' on China because of his family connections? |work=Sydney Morning Herald |date=2015-09-15 |accessdate=2015-09-18}}</ref><ref>{{cite web |url = http://www.bbc.com/zhongwen/simp/world/2015/09/150918_world_sugar_bun_aussie_pm|title=澳洲总理为何被陆媒昵称“糖包”？|work=BBC中文网 |date=2015-09-18 |accessdate=2015-09-21}}</ref>

2017年12月，因为亲中人士干预澳大利亚政治的事件曝光，特恩布尔在接受[[澳大利亚广播公司|澳大利亚广播公司]]记者采访时表示谴责，并说出“澳大-{}-利亚人民站起来”这句话。<ref>{{cite web |url = http://news.sina.com.cn/w/2017-12-10/doc-ifypnqvn2399367.shtml |title = 澳总理普通话回击中方批评:澳大利亚人民站起来了 |author=张迪 |publisher=新浪新闻 |date=2017-12-10 |accessdate=2018-12-30}}</ref><ref>{{cite web |url = https://www.abc.net.au/news/2017-12-09/malcolm-turnbull-says-he-will-stand-up-for-australia/9243274 |title = Malcolm Turnbull declares he will 'stand up' for Australia in response to China's criticism |author=Caitlyn Gribbin |publisher=abc |date=2017-12-9 |accessdate=2019-03-10}}</ref>
