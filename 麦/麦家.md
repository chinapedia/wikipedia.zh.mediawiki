'''麦家'''（{{bd|1964年|||catIdx=Jiang,蒋本浒}}），本名'''蒋本浒'''，生于[[浙江省|浙江]][[富陽市|富阳]]，先后毕业于[[解放军工程技术学院|解放军工程技术学院]]无线电系和[[解放军艺术学院|解放军艺术学院]]文学创作系。中国大陆[[作家|作家]]，电视剧[[编剧|编剧]]<ref>{{cite web |url=http://www.chinawriter.com.cn/fwzj/writer/37.shtml |title=麦家 |accessdate= 2013|publisher=中国作家网 |work= |language=zh-hans}}</ref>，浙江省作家协会主席。作品《解密》被翻译成多国语言，并入选英国“企鹅经典文库”<ref>{{Cite web|url = http://www.chinanews.com/cul/2014/03-21/5980799.shtml|title = 麦家小说《解密》入选英国“企鹅经典文库”|accessdate = 2015-09-13|author = |date = 2014年3月21日|publisher = 中国新闻网|language = zh-cn}}</ref><ref>{{Cite web|url = http://news.163.com/14/0625/03/9VI93FC100014Q4P.html|title = 21种语言追捧1本书|accessdate = 2015-09-13|author = |date = 2014-06-25|publisher = 网易新闻|language = zh-cn}}</ref>。

== 生平 ==
麦家生于浙江富阳。曾参军十七年。转业后成为专职编辑和作家。1983年毕业于[[中国人民解放军工程技术学院|解放军工程技术学院]]无线电系，1986年开始写作。1991年毕业于[[中国人民解放军艺术学院|解放军艺术学院]]文学系。1997年转业至[[成都电视台|成都电视台]]任电视剧部编剧。2008年调任[[杭州市文联|杭州市文联]]专业作家。2010年12月底，麦家当选为[[浙江省文联|浙江省文联]]副主席。2013年7月1日，在[[浙江省作家协会|浙江省作家协会]]第八届委员会第一次全体会议上，麦家当选为浙江省作家协会主席。2014年10月15日，麦家参加[[中共中央总书记|中共中央总书记]][[习近平|习近平]]在[[北京|北京]]召开的“文艺工作座谈会”，聆听习近平讲话精神“我们的文艺是社会主义文艺，要弘扬爱国爱党的真善美精神”，会后，习近平和麦家握手，称赞他：“我看过你的《暗算》《风声》，你是谍战剧第一人，歌颂的是爱国主义的精神，但是现在也有不少谍战影视剧不尊重历史，给观众造成了不良影响”。<ref>{{cite web |url=http://news.163.com/14/1017/05/A8O1ABAL00014AED.html |title= 习近平对作家麦家说：现在不少谍战剧不尊重历史|accessdate= 2014|publisher=网易 |work= |language=zh-hans}}</ref><ref>{{cite web |url= http://news.sina.com.cn/o/2014-10-17/065931002761.shtml|title=麦家：习总书记给我说“不少谍战剧不尊重历史” |accessdate=2014 |publisher=新浪网 |work= |language=zh-hans}}</ref>
