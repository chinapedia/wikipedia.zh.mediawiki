'''麦孟华'''（{{bd|1875年||1915年}}），字孺博，号蜕庵，[[广东|广东]][[顺德|顺德]]人，清末維新派人物。

光绪十四年（1888年）入[[广州学堂|广州学堂]]，光绪十七年（1891年）入[[广州|广州]][[万木草堂|万木草堂]]，成为[[康有为|康有为]]的学生，并在光緒十九年（1893年）与康有為同科中式[[举人|举人]]。

光绪二十一年（1895年）赴京应试，麦与[[梁启超|梁启超]]同寓，时常“相与规划救国政略，并助南海先生奔走国事”。中日将订《[[马关条约|马关条约]]》的消息传到[[北京|北京]]后，受康有为嘱咐，鼓动在京各省[[举人|举人]]上折拒和，参加“[[公车上书|公车上书]]”。光绪二十三年（1897年），与[[梁启超|梁启超]]、[[汪康年|汪康年]]等在[[上海|上海]]创设[[不缠足会|不缠足会]]。为《[[时务报|时务报]]》撰文时，提出“尊君权，抑民权”的主张。光绪二十四年（1898年）春，与梁启超等约同[[两广|两广]]、[[云贵|云贵]]、[[山陕|山陕]]、[[浙江|浙江]]等省举人上书，反对租让[[旅大|旅大]]给[[沙俄|沙俄]]，旋参加[[保国会|保国会]]。[[戊戌政变|戊戌政变]]后逃往[[日本|日本]]，助梁启超办《[[清议报|清议报]]》，光绪二十八年（1902年）任《[[新民丛报|新民丛报]]》[[撰述|撰述]]。光緒三十三（1907年）[[政闻社|政闻社]]在[[东京|东京]]成立时，被推为[[常务委员|常务委员]]。[[民国|民国]]四年（1915年）在[[上海|上海]]病故。著有《蜕庵诗词》。<ref>{{Cite web |url = http://search.huaxia.com/s.jsp?iDocId=483812 |title = 麦孟华 |author =  |date = 2003-06-19 |publisher = 华夏经纬网 |language = zh-hans |accessdate = 2014-12-31 |quote =  |deadurl = yes |archiveurl = https://web.archive.org/web/20121019101948/http://search.huaxia.com/s.jsp?iDocId=483812 |archivedate = 2012-10-19 }}</ref><ref>{{Cite web |url = http://www.chazidian.com/baike/807358/#1 |title = 麦孟华 |author =  |date =  |publisher =  |language = zh-hans |accessdate = 2014-12-31 |quote =  }}</ref>
