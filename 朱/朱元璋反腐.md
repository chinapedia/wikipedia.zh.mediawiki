'''朱元璋反腐'''又称'''朱元璋反贪'''<ref>[http://news.ifeng.com/history/1/jishi/200806/0626_2663_617897_3.shtml 朱元璋反贪用酷刑：贪官要剥皮] 凤凰网</ref><ref>[http://news.qq.com/a/20060331/001819.htm 朱元璋反贪] 腾讯网</ref>，指的是[[大明|大明]]开国皇帝[[朱元璋|朱元璋]]在位期间的[[政治腐敗|反腐]]事件。朱元璋反腐是下了重典的，他反腐主要以铁腕惩贪，不避亲故，法不徇私著称<ref >{{cite web|title=明太祖朱元璋惩贪不避亲：为反腐处死亲侄子与亲女婿|url=http://news.ifeng.com/a/20140701/40965124_0.shtml|publisher=凤凰网|accessdate=2014-10-10|location=北京|language=中文|date=2014-07-01}}</ref>，对贪官动则处以[[斬首|枭首]]和[[剝皮|剝皮]]示众的[[酷刑|酷刑]]。虽然有一定效果，但是并没能有效地遏制腐败<ref>{{cite web|title=不让"四风"和腐败"树倒根存"|url=http://pinglun.eastday.com/p/20141010/u1ai8383584.html|publisher=东方网|accessdate=2014-10-10|location=北京|language=中文|date=2009-01-15|deadurl=yes|archiveurl=https://web.archive.org/web/20150103061443/http://pinglun.eastday.com/p/20141010/u1ai8383584.html|archivedate=2015-01-03}}</ref>。

== 背景 ==
明朝建立以后，朱元璋发动并全程领导了史上最大最严的反腐肃贪运动<ref name="反腐存漏洞">{{cite web|title=朱元璋反腐存漏洞：以法治之名行人治之实|url=http://www.fj.xinhuanet.com/lishi/2014-07/10/c_1111553848.htm|publisher=新华网|accessdate=2015-01-03|location=北京|language=中文|date=2014-07-15|deadurl=yes|archiveurl=https://web.archive.org/web/20150103071049/http://www.fj.xinhuanet.com/lishi/2014-07/10/c_1111553848.htm|archivedate=2015-01-03}}</ref>。他以猛治国称<ref>《大诰》：“ 胡元以宽而失，朕收平中国，非猛不可”</ref>，大力打擊[[貪污|貪污]]<ref>[[叶子奇|叶子奇]]《草木子》记载：“凡守令贪酷者，许民赴京陈诉，赃至六十两以上者枭首示众，仍剥皮实草。府、州、县衙之左，特立一庙以祀土地，为剥皮之场，名曰‘皮场庙’。官府公座旁，高悬一剥皮实草之袋，使之触目惊心。”</ref>。他的“重典治国”思想不只为遏制官僚腐败<ref>{{cite web|title=朱元璋“重典治国” 思想不只为遏制官僚腐败|url=http://www.jx.xinhuanet.com/news/history/2014-10/21/c_1112905302.htm|publisher=新华网|accessdate=2014-12-30|location=北京|language=中文|date=2009-01-15|deadurl=yes|archiveurl=https://web.archive.org/web/20150103063047/http://www.jx.xinhuanet.com/news/history/2014-10/21/c_1112905302.htm|archivedate=2015-01-03}}</ref>。

== 手段 ==
=== 设立《大明律》和《大诰三编》 ===
朱元璋订《[[大明律|大明律]]》，编《[[明大诰|大诰]]》，对惩治腐败做出详细规定，并严厉实行之<ref>{{cite web |title = 历史上的朱元璋反腐：强人政治 人亡政息 |url = http://news.xinhuanet.com/ziliao/2014-05/28/c_126559003.htm |website = 新华网 |accessdate = 2015-01-03 |location = 北京 |language = zh |date = 2014-05-28 }}</ref>。朱元璋亲自参与主持的《大明律》中将《受赃》专门设为一篇，其中详尽规定了对于腐败的惩罚，为他以后的铁腕反腐奠定了基础。同时朱元璋下令将《大诰》等其他多种司法教材发到每个家庭中去，而且规定官府执法时凡是家里藏有《大诰》的，就可罪减一等，反之，要罪加一等。同时《大诰》还被列入全国各级学校的必修课程，连科举考试都从《大诰》中出题。一时间从学校到民间讲读《大诰》蔚然成风<ref name="反腐存漏洞" />。

=== 严惩腐败分子 ===
朱元璋自幼出身貧寒，對政治貪污尤其憎惡，其對貪污腐敗官員處以極其嚴厲的處罰<ref>{{cite book|author=叶子奇|title=《草木子》|quote=凡守令贪酷者，许民赴京陈诉，赃至六十两以上者枭首示众，仍剥皮实草。府、州、县衙之左，特立一庙以祀土地，为剥皮之场，名曰『皮场庙』。官府公座旁，高悬一剥皮实草之袋，使之触目惊心。}}</ref><ref name="反腐存漏洞"/>。在朱元璋主政期間，大批不法貪官被處死，包括開國將領[[朱亮祖|朱亮祖]]，女婿駙馬都尉[[歐陽倫|歐陽倫]]，其中甚至因為[[郭桓案|郭桓案]]、[[空印案|空印案]]殺死數萬名官員<ref >{{cite web|title=朱元璋反腐：全国省部级官员几乎全涉案|url=http://cul.sohu.com/20140617/n400942433.shtml|publisher=新华搜狐网|accessdate=2015-01-03|location=北京|language=中文|date=2014-06-17}}</ref>。

=== 设立监察机构 ===
朱元璋为了加大反腐力度，设立了从朝廷到地方的监察机构，其组织之严，权力之大，都是前朝历代绝无仅有的。在中央朱元璋设立最高反腐机构[[都察院|都察院]]，将刑、检、法职能集于一身。不久又设置[[六科给事中|六科给事中]]，专事纠举弹劾六部中的贪官污吏。在地方上，除了有朝廷派遣的[[监察御史|监察御史]]外，同时又在各省设按察司，掌监察之职；后来又设立“[[巡撫|巡撫]]”制度，由朝廷派亲信重臣大臣到各地出巡，监察地方官吏<ref name="反腐存漏洞"/>。

=== 广开言路鼓励民间反腐 ===
朱元璋在铁腕反腐的同时，还鼓励民间反腐。朱元璋在午门外设立“[[登闻鼓|登闻鼓]]”，号召受冤的百姓们击鼓鸣冤。为了防止此行为虚设，同时命令一名御史每天在那里负责，但凡是民间有冤情，地方官员又不受理的百姓们，当事人可以击登闻鼓，并由在那里负责的御史带着上奏；为了让下级官吏有参政机会，朱元璋打破常规，规定不论朝廷官员品级、隶属均可参加早朝上殿言事，有隐情还可单独召见；朱元璋不仅允许民告官，还鼓励百姓越级告状；他甚至鼓励普通百姓把贪官直接绑赴京城治罪，各级官员如果胆敢“截访”，就治以诛灭家族的重罪。于是，在朱元璋的反腐力度下，一群衣衫褴褛的百姓押解着贪官污吏行走在通往京城的道路上<ref name="反腐存漏洞"/>。

== 影响 ==
由於朱元璋的吏治嚴厲，在明初相當長一段時間，官員腐敗的情況得到有效遏制<ref>{{cite book|author=[[赵翼|赵翼]]|title=《廿二史剳记》|quote=明初「崇尚循良，小廉大法，几乎两汉之遗风，且驾唐宋而上哉」}}</ref>。朱元璋开展雷厉风行的肃贪运动，历时之久、措施之严、手段之狠、刑罚之酷、杀人之多，为几千年封建历史所罕见。

尽管朱元璋反贪决心大、力度猛、出奇招，使腐败现象得到一定程度的遏制，也一度取得了“阶段性的成果”，但远未达到彻底清除腐败的根本目的<ref name="反腐倡廉">{{cite web|title=从朱元璋的反腐倡廉说开去|url=http://theory.people.com.cn/GB/40537/8677605.html|publisher=人民网|accessdate=2014-01-29|location=北京|language=中文|date=2009-01-15}}</ref>。
