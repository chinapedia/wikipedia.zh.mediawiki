'''朱莉娅·菲佛·伯恩斯州立公园'''（{{lang|en|Julia Pfeiffer Burns State Park}}）是美国[[加利福尼亚州|加利福尼亚州]]的一个州立公园，位于大苏尔海岸。公园的主要景点为麦克威瀑布，从80英尺的悬崖落到[[太平洋|太平洋]]。公园生长着超过2500年，高度达300英尺（90米）的[[加州红木|加州红木]]。<ref>{{cite journal |last=Kinneberg |first=Caroline |title=America's Most Beautiful Coastal Views |url=http://www.travelandleisure.com/articles/americas-most-beautiful-coastal-views/ |journal=Travel+Lesiure |date=2010-08 |accessdate=2013-05-22}}</ref>该公园以朱莉娅·菲佛·伯恩斯的名字命名，伯恩斯是20世纪初大苏尔地区一个受人尊敬的居民和牧场主，她一生大部分时光在该地区居住，直到1928年去世。<ref name="Website">{{cite web |title=Julia Pfeiffer Burns SP |url=http://www.parks.ca.gov/?page_id=578 |publisher=California State Parks |accessdate=2012-07-04}}</ref> 公园位于[[卡梅尔海_(加利福尼亚州)|卡梅尔海]]往南37英里（60公里），成立于1962年，占地面积3,762英亩（1,522公顷）。<ref>{{cite document |title=California State Park System Statistical Report: Fiscal Year 2009/10 |url=http://www.parks.ca.gov/pages/795/files/09-10%20statistical%20report%20final%20online.pdf |publisher=California State Parks |page=24 |accessdate=2012-07-04}}</ref>
