'''朱履龢'''（{{bd|1877年||1945年|4月13日|catIdx=zhu朱}}），字'''笑山'''，[[浙江|浙江]][[嘉兴|嘉兴]]人。[[中華民国|中華民国]]政治家、法官。<ref name=kmt>{{cite book|author=刘国铭主编|title=中国国民党百年人物全书|publisher=团结出版社|year=2005}}</ref><ref name=guan>{{cite web|url=http://www.history.com.tw/pe/6/06152201.htm|title=朱履龢小传|author=[[关国煊|郑仁佳]]}}</ref><ref name=zx>{{Cite book|author = |title = 最新支那要人伝|year = 1941|publisher = [[朝日新聞社|朝日新聞社]]|isbn = }}</ref><ref name=liu>{{Cite book|author = 劉寿林等編|title = 民国職官年表|year = 1995|publisher = [[中華書局|中華書局]]|isbn = 7-101-01320-1}}</ref>

==生平==
朱履龢早年在[[英国|英国]]留学。回国后，历任[[护法军政府|护法军政府]]司法部署理秘书、总务厅厅长，法权讨论会秘书，关税特别会议委员会秘书。<ref name=kmt/><ref name=zx/><ref name=guan/><ref name=liu/>

1927年（[[民国紀元|民国]]16年）6月，朱履龢任[[南京国民政府|南京国民政府]][[中华民国外交部|外交部]]总务司司長。同年10月，总务司改为第二司，朱履龢仍任司长。1928年2月，任司法部秘書長。1928年3月，升任司法部次長（11月，任司法部政務次長）。1930年（民国19年）4月，接替[[魏道明|魏道明]]代理司法行政部長。12月，兼任[[中華民国立法院|立法院]]立法委員。1931年12月，[[羅文幹|羅文幹]]就任司法行政部長，朱履龢回任政務次長。1932年（民国21年）1月，卸任政務次長。<ref name=kmt/><ref name=zx/><ref name=guan/><ref name=liu/>

1938年（民国27年）8月，朱履龢被任命为[[中華民国維新政府|中華民国維新政府]]司法行政部次長。1939年5月，就任新設立的最高法院院長。1940年（民国29年）3月，[[汪兆銘|汪兆銘]]（汪精衛）的[[汪精卫政权|南京国民政府]]成立，朱履龢被任命为[[司法院_(汪精卫政权)|司法院]]副院長、[[中央政治委员会_(汪精卫政权)|中央政治委員会]]列席委員。同年4月，被任命为中央公務員懲戒委員会委員長。他还担任[[立法院_(汪精卫政权)|立法院]]副院长。1941年起不再任立法院副院长。<ref name=kmt/><ref name=zx/><ref name=guan/><ref name=liu/>

1945年（民国34年）4月13日，朱履龢在南京病逝。享年69岁。<ref name=kmt/><ref name=guan/>
