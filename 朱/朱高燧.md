'''趙簡王朱高燧'''{{BD|1383年|1月19日|1431年|10月5日|catIdx=Z朱}}，是明成祖[[朱棣|朱棣]]嫡第三子，乃[[仁孝文皇后|仁孝文皇后]]徐氏所出，[[明朝|明朝]]第二次封的[[趙王|趙王]]。

== 生平 ==

朱高燧於[[永乐_(明朝)|永樂]]二年四月初四<ref name="a1">《[[弇山堂别集|弇山堂别集]]·卷三十三》○亲王......赵简王高燧成祖第三子母仁孝皇后徐氏洪武十五年十二月十六日生永乐二年四月初四日封洪熙元年四月十六日之国河南彰德府宣德元年尽献三护卫六年八月（阙）日薨在位二十八年寿五十葬安阳县寿安山妃沐氏黔国公晟女永乐九年十二月二十九日册封二十一年十二月二十五日薨二十二年十月初八日册翁氏为妃常山中护卫镇抚司陆女正统五年六月二十一日薨合葬寿安山子悼僖世子瞻坺先薨无子嫡弟惠王瞻塙宣德七年自安阳王嗣......</ref>（1404年5月12日）被封為趙王，不久命居[[北京|北京]]<ref name="a2"/>。

最初，朱高燧恃寵，時常做出不法之事，又和漢王[[朱高煦|朱高煦]]合謀奪嫡，並誣陷[[太子|太子]][[朱高熾|朱高熾]]。于是太子宫寮多得罪。永樂七年（1409年），明成祖得知其不法之事，大怒，立刻誅殺他的[[長史|長史]][[顧晟|顧晟]]，同時褫奪朱高燧的[[冠服|冠服]]，幸得太子朱高熾求情才得免。後來明成祖選擇[[國子司業|國子司業]][[趙亨道|趙亨道]]及[[董子莊|董子莊]]為長史輔導他，他的行為才稍改善<ref name="a2">《[[明史|明史]]·列传第六·诸王三》</ref>。

永樂二十一年（1423年）五月，明成祖病重。[[護衛|護衛]]指揮[[孟賢|孟賢]]等人勾結[[欽天監|欽天監]][[王射成|王射成]]及[[內侍|內侍]][[楊慶|楊慶]]的養子偽造遺詔，打算毒死明成祖，廢太子朱高熾，立朱高燧為帝。[[總旗|總旗]][[王瑜|王瑜]]姻親[[高以正|高以正]]為孟賢策畫陰謀後告訴王瑜，王瑜立刻上告明成祖。不久，孟賢就被逮捕，得偽詔，伏誅，而王瑜就陞為[[遼海衛|遼海衛]]千戶。明成祖問朱高燧：「你做的嗎？」他十分害怕，嚇得說不出話。但太子極力為他辯解：「這是下人的所為，弟弟一定不知道的。」自此，朱高燧行為日益收斂<ref name="a2"/>。

[[明仁宗|明仁宗]]即位，加朱高燧歲祿二萬石，並在[[洪熙|洪熙]]元年四月十六<ref name="a1"/>（1425年5月3日）[[就藩|就藩]][[彰德府|彰德府]]，同時放棄[[常山|常山]]左右二護衛。[[明宣宗|明宣宗]]即位，又賜田園八十頃<ref name="a2"/>。

[[宣德|宣德]]元年（1426年），漢王朱高煦謀反被擒，至[[單橋|單橋]]，[[尚書|尚書]][[陳山|陳山]]迎接明宣宗。陳山建議捉拿朱高燧，因為他和朱高煦共謀逆很久，若不捉拿朱高燧，他日趙王也可能謀反，勞煩明宣宗。此時明宣宗猶豫未決，唯有[[楊士奇|楊士奇]]認為不可。明宣宗說：「先帝和二位叔叔甚為友愛。漢王謀反，朕不敢赦免。但趙王沒有謀反跡象，朕不忍心辜負先帝。」後來朱高煦至[[北京|北京]]，亦說嘗試遣人與朱高燧通謀。[[戶部|戶部]][[主事|主事]][[李儀|李儀]]及[[尚書|尚書]][[張本|張本]]請求削趙王護衛，宣宗不聽。次年，朱高燧放棄常山中護衛及[[群牧所|群牧所]]、[[儀衛司|儀衛司]]官校。宣宗命收回護衛，但無收回儀衛司<ref name="a2"/>。

朱高燧在[[宣德|宣德]]六年（1431年）八月過世，葬[[安阳县|安阳县]]寿安山<ref name="a1"/>，今属[[安丰乡_(安阳县)|安丰乡]]李家坡村<ref>{{Cite web |url=http://www.ayrbs.com/epaper/html/2011-09/21/content_52225.htm |title=安 丰: 文化丰厚之乡  |accessdate=2018-03-10 |author=贾振林 |date=2011-09-21 |work= |publisher=安阳日报电子版——安阳网|language=Zh-hans }}</ref>。[[諡號|諡號]]'''簡'''，長子悼僖世子[[趙悼僖世子|朱瞻坺]]早卒，一年後其子[[朱瞻塙|朱瞻塙]]就嗣位。

===妻===
*徐氏，[[徐章|徐章]]女，[[何福_(明朝)|何福]]外甥女，洪武三十五年（1402年）十月封为皇三子高燧妃，永乐二年（1404年）封为赵王妃，永乐八年（1410年）何福自杀后，永乐九年（1411年）三月徐氏因无子被废<ref>《[[明太宗实录|明太宗实录]]·卷114》：“敕赵王高燧曰：‘徐妃既无子，又数诳诞不悛，可令闲居善养之终其身，将别选名家女为尔之妃。徐本何福之甥，其初有过也，以福故优容之。福死，徐不悛改，遂有是命。’”</ref>
*沐氏，[[沐晟|沐晟]]女，永乐九年（1411年）十二月册封<ref>《明太宗实录》卷122：“册黔国公沐晟女为赵王高燧妃。”；《国榷》卷十五</ref>，永乐二十一年薨。谥号“恭惠”
*翁氏，武略将军、前常山中护卫镇抚翁子寿（或称翁陆<ref name="a1"/>）之女。翁氏生于洪武二十一年十月初八十月初八，永乐十年生朱瞻塙，朱瞻坺应亦为她所生。永乐二十二年册封。正统五年薨，年五十二岁。与朱高燧合葬寿安山。
