'''朱之鑫'''（{{bd|1949年|3月27日||catIdx=}}），[[浙江|浙江]][[嘉兴|嘉兴]]人，[[中华人民共和国|中华人民共和国]]政府官员，中共十六届候补中央委员、十七届中央委员<ref name=jianli/>。

==生平==
1969年参加工作，先后在[[吉林省|吉林省]][[大安县|大安县]]插队，[[长春第一汽车制造厂|长春第一汽车制造厂]]、[[中华人民共和国第一机械工业部|中华人民共和国第一机械工业部]]、[[安徽省人民政府|安徽省人民政府]]办公厅工作。1983年9月，到[[中华人民共和国国家计划委员会|中华人民共和国国家计划委员会]]工作，曾经任国家计划委员会国民经济综合司处长，曾挂职担任[[大连市|大连市]]计划委员会副主任。1993年3月，任国家计划委员会国民经济综合司副司长。1997年4月，升任国家计划委员会国民经济综合司司长<ref name=jianli>{{cite web|title=朱之鑫|url=http://cpc.people.com.cn/gbzl/html/121000144.html|publisher=中国党政领导干部资料库|date=2015-02-27|deadurl=yes|archiveurl=https://web.archive.org/web/20170216055056/http://cpc.people.com.cn/gbzl/html/121000144.html|archivedate=2017-02-16}}</ref>。

1999年9月，任[[中华人民共和国国家统计局|国家统计局]]副局长、党组副书记。2000年6月，升任国家统计局局长、党组书记。2003年3月，调任[[国家发展和改革委员会|国家发展和改革委员会]]副主任、党组成员。2005年8月，由国家发展和改革委员会党组成员升任党组副书记<ref name=jianli/><ref>{{cite web|title=中国应对全球金融危机的宏观政策|url=http://zhuzhixin.ndrc.gov.cn/zyjh/t20090402_270506.htm|publisher=中华人民共和国国家发展和改革委员会|accessdate=2012-02-19|deadurl=yes|archiveurl=https://web.archive.org/web/20120528051440/http://zhuzhixin.ndrc.gov.cn/zyjh/t20090402_270506.htm|archivedate=2012-05-28}}</ref>。2009年至2010年，兼任国务院深化医药卫生体制改革领导小组办公室主任<ref name=zggbw>{{Cite web |url = http://www.cnr.cn/2012zt/18ddbfc/zb/minsheng/zy/201211/t20121112_511335449.shtml |title =国家发展和改革委员会副主任、党组副书记朱之鑫同志简介  |author =  |date =2012-11-12  |publisher = 中国广播网 |language =  |accessdate =  |quote =  }}</ref>。2015年2月卸任[[国家发展和改革委员会|国家发展和改革委员会]]副主任、党组副书记职务<ref>{{cite web|url=http://money.163.com/15/0226/16/AJD5NLV900252G50.html|title=张勇任发改委副主任 谢振华朱之鑫徐宪平去职|publisher=网易|accessdate=2015-02-26}}</ref><ref>{{Cite web |url =  http://www.gov.cn/gongbao/content/2015/content_2835280.htm|title =中华人民共和国国务院任免人员  |author =  |date =  |publisher =国务院公报2015年第8号  |language =  |accessdate =  |quote =2015年2月9日免去朱之鑫、解振华、徐宪平的国家发展和改革委员会副主任职务。 }}</ref>。

朱之鑫是研究生学历，高级经济师，[[北京大学|北京大学]]、[[中国人民大学|中国人民大学]]兼职教授。他是中国共产党第十六、十七、十八次全国代表大会代表，第十六届中央候补委员，第十七届中央委员<ref name=zggbw/>。
