'''朱美拉棕櫚島單軌鐵路'''（{{lang|en|'''Palm Jumeirah Monorail'''}}）是[[杜拜|杜拜]][[朱美拉棕櫚島|朱美拉棕櫚島]]對外連接的[[單軌鐵路|單軌鐵路]]，於2009年4月30日正式通車，<ref name="timeout">http://www.timeoutdubai.com/knowledge/news/8723-palm-monorail-tried-and-tested</ref>並計畫在未來能連接到[[杜拜地鐵|杜拜地鐵]]紅線。<ref name="gn"/>

朱美拉棕櫚島單軌鐵路同時也是[[中東地區|中東地區]]第一條通車的單軌鐵路。<ref name="mena">{{Cite web |title=ME's 1st monorail to begin services in April |url=http://www.menafn.com/qn_news_story_s.asp?StoryId=1093207150 |publisher=''MENAFN.com'' |date=2008-08-08 |accessdate=2008-08-11 |deadurl=yes |archiveurl=https://web.archive.org/web/20110611074909/http://www.menafn.com/qn_news_story_s.asp?StoryId=1093207150 |archivedate=2011-06-11 }}</ref>

== 歷史 ==
[[File:The_Palm_Jumeirah_Monorail_Under_Construction_on_16_April_2007.jpg|thumb]]

朱美拉棕櫚島單軌鐵路於2006年[[2006年3月|3月]]在[[丸紅|丸紅公司]]的監督下正式動工<ref name="gn"/>。2008年[[2008年7月|7月]]，總長{{convert|5.45|km|mi|0}}的單軌鐵路軌道正式完工，並在同年11月開始測試列車運作情形。<ref name="ab">{{Cite web |title=Palm monorail hit by four-month delay |url=http://www.arabianbusiness.com/525611-palm-monorail-hit-by-four-month-delay?ln=en |publisher=''ArabianBusiness.com'' |date=2008-07-22 |accessdate=2008-08-11}}</ref>

單軌鐵路原本預計在2008年[[2008年12月|12月]]正式通車，但卻因故而延遲至2009年4月30日通車。<ref name="gn"/><ref name="ab"/>

建造單軌鐵路的預算為400萬[[美元|美元]]，並預留一些資金當作路線擴長{{convert|2|km|mi|0|adj=on}}來連接到[[杜拜地鐵|杜拜地鐵]]的經費。<ref name="ab2">{{Cite web |title=Quiet please for region’s first monorail |url=http://www.arabianbusiness.com/index.php?option=com_content&view=article&id=10716 |publisher=''ArabianBusiness.com'' |date=2007-04-07 |accessdate=2008-08-11}}</ref>也有人表示，建造鐵路的預算為1100萬美元<ref name="khaleejtimes.com">http://www.khaleejtimes.com/DisplayArticle.asp?xfile=data/theuae/2009/May/theuae_May169.xml&section=theuae&col=</ref>。
[[File:DubaiMonorail1.JPG|thumb]][[亞特蘭提斯酒店站|站]]]]
== 技術詳情 ==
朱美拉棕櫚島單軌鐵路採用[[日立|日立]]的[[單軌鐵路#種類|跨座式單軌鐵路]]技術建造。單軌鐵路最初預計每日會有4萬人搭乘，尖峰時刻每3分鐘一班，非尖峰時刻15～20分鐘一班。<ref name="gn"/><ref name="mena"/>

但正式通車後，平均每日僅600人搭乘<ref name="constweek"/>，列車幾乎是處在「空車」狀態下。<ref>http://www.encyclopedia.com/doc/1G1-201385494.html</ref>
