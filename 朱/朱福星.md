'''朱福星'''（1897年一1975年），又名'''芝生'''、'''步虚'''，字'''申卿'''，号'''崇德'''，晚号'''真正好笑'''。浙江东阳看芝人。近代浙江历史上一位杰出人物，是中国共产党的挚友。在二十世纪中国历史舞台上，他为民族的独立和国家的富强，为劳动大众的生存和幸福，数十年披荆斩棘，执著奋斗，谱写了一页页光辉的历史篇章。他为官清廉、光明磊落，一身正气、爱国护民，无私无畏的光辉人生，他的座右铭“造福者不为享福，善始者还当善终”彰显人格力量。

== 简介 ==
朱福星民国4年（1915年）就读浙江第七师范时，为反抗日寇强占青岛和袁世凯签订丧权辱国的“二十一条”，约同学50人投笔从戎。军旅中保持惊人的求知欲，1924年在上海中西医药函授学校毕业。1926年誓师参加北伐，战功显赫，历任连、营、团长。

[[一·二八事变|“一·二八”淞沪抗战]]为抗日总预备队防守杭嘉湖，战后任6师17旅副旅长，入中央军校高教班第一期深造。1933年毕业任6师33团上校团长，1937年参加[[淞沪会战|“八一三”淞沪抗战]]。8月31日奉令增援吴淞炮台湾守军，即向东线进击，浴血奋战数小时，于凌晨3时30分，一鼓作气扫荡孙家庄、蒲家庄、炮台湾之敌，粉碎了日寇对吴淞炮台湾的围困，迫使日寇纷纷向吴淞镇溃退。继而配合邻团左右两翼攻击，控制自[[蕰藻浜|蕰藻浜]]至[[宝山县|宝山]]的重要防线。9月4日下午，日舰集中火力向我6师阵地炮轰，两个小时间发射炮弹达一千四百余发，日寇步兵借炮火掩护多次向我33团阵地突击。朱福星团长率部冒死奋战猛力反攻将敌击退，至此，在火线指挥作战已经七天七夜。6时30分，日寇疯狂反扑，朱福星团长亲临阵前率部冲锋，士气大振，全体官兵向敌突击肉搏，终于杀退日寇，而朱福星团长不辛被日寇三颗子弹穿进了胸部，失血过多，昏倒沙场。部下拼死将他抢救下火线，用专车急送杭州陆军医院，再转国际仁爱医院抢救脱险。是年底朱福星团长出院回家乡疗伤，被当地推举为14都乡抗日自卫委员会主席、自卫队长和乡长，带伤组织抗日救亡，大义无畏，保卫家乡，抗拒日寇十万银元派款，东阳沦陷期间，唯十四都未给日伪分文粒米，迫使日军的威风难过乌竹岭。他胸襟开阔、无私为民、赤胆忠心、舍己救人，调解纠纷、保护先进，崇尚教育、办学诲人，被百姓誉为“十四都乡蔽荫大树”。

抗战胜利后，竭力保护进步青年。1949年10月至1951年5月被聘为东阳县第一至第四次各界人民代表会议代表，为建设新中国奔忙。晚年蒙冤受屈，对人民忠诚如一，爱书如命，手不释卷，抱“不为良相，则为良医”之宗旨，精研医药，无偿为群众治病；笔耕不止，著写医方医论44卷，留下珍贵的遗产。其座右铭“造福者不为享福，善始者还当善终”，是他高贵人格的写照。

== 福星亭 ==

公元二零零五年九月，适值中国人民抗日战争暨世界反法西斯战争胜利60周年，故乡人民特集资修建抗日名将朱福星先生碑亭（福星亭）于[[东阳市|东阳市]][[巍山镇|巍山镇]]眠牛山公园苛山之上，缅怀抗日英雄朱福星先生。

福星亭建于巍山镇怀鲁眠牛山脊，毗邻观音塔，北望乌竹岭，岭下即是有“华东第一枢纽”之称的怀鲁枢纽互通立交桥，南览巍山列景，村庄池塘，星罗棋布。福星亭通体为青石建筑，南北两方各有三级石阶走上亭台，亭台上是六柱六角形通透式石亭，门楣分别是许秀堂先生题写的“福星亭”和梅凯先生题写的“民族精英”，两边对联为“北伐抗日气贯长虹，抚民救国名垂青史”和江征帆先生1997年为朱福星先生百年诞辰题写的“抗日先锋功高厚，革命丹心鉴日月”。其余四面设置石栏石凳。亭阁中间立有三棱柱式纪念碑，正面镌刻有朱福星先生像和东阳市巍山镇人民政府的铭文，另外两面分别是朱福星先生的座右铭手迹：“造福者不为享福，善始者还当善终。”和楼降云先生为纪念朱福星先生诞生一百周年所作七律一首： 　

:　男儿慷慨气如云，学业文章卓不群。 
:　投笔封侯班定远，誓师御倭戚家军。 　
:　龙潭鏖战歼民贼， 淞沪忘身扫寇氛。 　
:　保障一方施德泽， 百年铭记颂功勋。
2011年公布朱福星纪念亭为东阳市“红色遗址胜迹”，成为青少年爱国主义教育基地。
