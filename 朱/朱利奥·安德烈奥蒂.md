'''朱利奥·安德莱奥蒂'''（{{lang-it|'''Giulio Andreotti'''}}）{{BD|1919年|1月14日|2013年|5月6日|catIdx=Andreotti}}[[意大利|意大利]]政治家，作家和記者。
== 生平 ==
朱利奥·安德莱奥蒂生于意大利[[羅馬|羅馬]]，在[[羅馬大學|羅馬大學]]獲[[法律|法律]]學位，[[二戰|二戰]]時在[[教廷|教廷]]擔任圖書館管理員，其間認識政壇中人並踏上從政路。他是1946至1948年制憲議會成員，1948至1991年以來的[[意大利國會|意大利國會]]眾議員，后於1991年獲[[意大利總統|總統]]任命成为[[終身參議員|終身參議員]]。有前幕僚讚他是「重建戰後意國的工程師」。

== 政治生涯 ==
安德烈奧蒂早在1940年代就步入政壇，以犀利的言辭聞名。身為天主教民主黨一員的他曾擔任多個聯合政府要職，成為意大利政壇叱吒風雲的人物。他與[[阿尔多·莫罗|阿尔多·莫罗]]是[[意大利天主教民主黨|意大利天主教民主黨]]代表人物之一。

朱利奥·安德莱奥蒂是意大利政壇在整個20世紀下半叶的重要人物。1959至1966年及1974年，出任[[意大利國防部|意大利國防部]]長。1972至1973年，1976至1979年及1989年至1992年三次出任[[意大利总理|意大利总理]]，是意大利在第二次世界大战后任职最长的总理之一，共任职七年多。1983至1989年，出任[[意大利外交部|意大利外交部]]長。

朱利奥·安德莱奥蒂與天主教民主黨籍前總理[[阿纳尔多·福拉尼|阿纳尔多·福拉尼]]及[[意大利社會黨|意大利社會黨]]籍前總理[[贝蒂诺·克拉克西|贝蒂诺·克拉克西]]在二十世紀七八十年代，被喻為義大利「三頭政治」，簡稱'''CAF'''。

安德烈奧蒂是二戰後國家重建的工程師，有著意大利政客少有的國際聲望。不過，據法新社報道，雖有人因其傑出的政治才能將他比作[[凱撒|凱撒]]，也有人認為他是意大利腐敗政治的代表。上世紀七八十年代，有人稱他為“黑色[[教皇|教皇]]”<ref>[http://news.xinhuanet.com/world/2013-05/07/c_124674899.htm 意大利七任老總理逝世]</ref>。

1991年，他獲總統委任為意大利終身參議員，1992年退下政治前線，晚年在[[梵蒂岡|梵蒂岡]]對面的豪宅居住。

雖然一生政績顯赫，但他於1990年代後期涉嫌勾結並指使黑手黨殺人，判監23年，2003年上訴得直脫罪<ref>[http://paper.wenweipo.com/2013/05/07/GJ1305070012.htm 意傳奇總理安德烈奧蒂逝世]</ref>。

== 去世 ==
安德烈奧蒂晚年健康欠佳，2012年8月起一直因心臟病住院。

2013年5月6日，安德莱奥蒂因病于罗马逝世，享年94岁。<ref>[http://www.apdnews.com/news/25766.html 意大利前总理安德烈奥蒂去世] {{webarchive|url=https://web.archive.org/web/20130508135420/http://www.apdnews.com/news/25766.html |date=2013-05-08 }}，亚太日报，2013年5月7日</ref><ref>{{cite news|author=|url=http://www.dfdaily.com/html/51/2013/5/6/991241.shtml|title=意大利前总理安德莱奥蒂逝世 享年94岁|work=[[东方早报|东方早报]]|date=2013-05-06|accessdate=2013-05-06|deadurl=yes|archiveurl=https://web.archive.org/web/20130607043423/http://www.dfdaily.com/html/51/2013/5/6/991241.shtml|archivedate=2013-06-07}}</ref>意國全國體育設施將下半旗致哀，紀念他帶領國家申辦1960年[[羅馬奧運|羅馬奧運]]。
