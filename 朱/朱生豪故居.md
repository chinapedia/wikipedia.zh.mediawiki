'''朱生豪故居'''位于[[中国|中国]][[浙江省|浙江省]][[嘉兴市|嘉兴市]][[南湖区|南湖区]]禾兴南路73号（原南大街东米棚下14号，现属[[梅湾街|梅湾街]]历史街区东片），为近代莎士比亚戏剧翻译家、诗人[[朱生豪|朱生豪]]在嘉兴的故居，朱生豪在此度过童年与少年时光，1943初新婚后携夫人[[宋清如|宋清如]]从常熟返回老家居住，次年12月病逝于此。该建筑为具有浙北风格的传统民居，坐东朝西，现存两层的楼屋五间、南北偏屋各两间和东北侧的小偏房（灶披间）两间，东西有前后院<ref>{{cite web |url=http://qjwb.zjol.com.cn/html/2012-02/05/content_1331864.htm?div=-1|title=《“我的译文比梁实秋的漂亮”》|accessdate=2014-03-19|work=|publisher=钱江晚报}}</ref>。2006年8月至2007年2月由市城投集团出资落架大修，2007年11月对外开放<ref>{{cite web |url=http://www.jxgaj.gov.cn/Html/201210/03/30334.html|title=《朱生豪故居》|accessdate=2014-03-19|work=|publisher=嘉兴市公安局}}</ref><ref>{{cite web |url=http://www.zjol.com.cn/05jx/system/2007/11/22/008992595.shtml|title=《"译界楷模"陋室译写莎翁风流 朱生豪故居正式开放》|accessdate=2014-03-19|work=|publisher=浙江在线嘉兴频道}}</ref>。
