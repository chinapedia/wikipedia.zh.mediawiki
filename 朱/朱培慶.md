'''朱培慶'''（{{bd|1948年|5月16日|catIdx=C朱}}），前著名[[電台主持|電台主持]]，前[[香港廣播處長|香港廣播處長]]，掌管[[香港電台|香港電台]]。2007年7月因一場[[桃色|桃色]]疑雲而宣佈提早退休。

==生平==
朱培慶出身於舊式徙置區，[[伊利沙伯中學|伊利沙伯中學]]畢業後，憑獎學金升讀[[香港大學|香港大學]]數學系，1971年取得學士學位後加入香港電台，任職助理節目主任。當時該職位只需中五會考資格，其高學歷令他成為重點培訓對象，1970年代曾被香港電台送往[[英國廣播公司|英國廣播公司]]受訓6個月，之後在港台與[[吳錫輝|吳錫輝]]及[[陳任|陳任]]等人合作，籌辦年青人節目《[[青春交響曲|青春交響曲]]》，對抗當時商台的著名DJ組合[[六啤半|六啤半]]。

《青春交響曲》大受歡迎，令朱培慶備受賞識，先後主持《香江之晨》、《面面觀座談會》及《太平山下漫步》等時事節目，接聽市民來電、月旦社會，被視為香港[[Phone-in|烽煙]]節目的先驅。1988年升任教育電視台長，1992年晉升助理廣播處長，1993年再獲升為副廣播處長。

1999年年底，當時廣播處長[[張敏儀|張敏儀]]調任至[[日本|日本]][[東京|東京]]，出任香港駐日本經濟貿易首席代表，朱培慶接任廣播處長。其任內的重要事務，包括公共廣播服務檢討、港台賬目混亂問題、員工被指控貪污，在節目上，港台批評政府的言論亦引起[[董建華|董建華]]及[[曾蔭權|曾蔭權]]先後責難。

在他上任之初，港台編輯自主的問題在香港引起廣泛爭議，外界曾指朱培慶作風不及前任的張敏儀硬朗，擔心未能保護港台免受政治干預，但他任內曾在多個重要事件上，支持香港電台的編輯自主和言論空間。

2001年，港台節目《[[頭條新聞_(香港電台)|頭條新聞]]》以[[塔利班|塔利班]]政權發表施捨報告比喻特首董建華，故事說很多長老要求塔利班不要留任，但塔（他）賴死不走，市民唯有跟美國人打仗，寧當俘虜，以求兩餐溫飽，還叫「北方聯盟」（北京）派人接管香港。同年10月17日，董建華批評該節目「低級趣味」，政府發言人隨後批評廣播處長作為總編輯，應切實執行「不偏不倚」的原則。<ref>{{cite web |url=http://www.hkcs.org/edcb/media/mediax/mediax11.htm |title=存档副本 |accessdate=2007-07-10 |deadurl=yes |archiveurl=https://web.archive.org/web/20070928035926/http://www.hkcs.org/edcb/media/mediax/mediax11.htm |archivedate=2007-09-28 }}</ref>事後，港台反駁節目只是嬉笑怒罵，抒發社會怨氣，堅持節目未來方針不變。
另外，港台其中一節目曾為夏金城拍攝一段音樂錄像MV「雞與龜」，但該短片卻無法引起任何熱潮。

2007年，他高調支持港台轉型公共廣播機構，同年5月更高調聲援「撐港台」遊行，在集會高呼五聲「撐」字，以示對員工的支持。

2007年，他外出尋歡時被記者拍下窩囊一面，成為新聞焦點，事後有網民將[[陳奕迅|陳奕迅]]的《[[裙下之臣|裙下之臣]]》的影像部份[[惡搞|惡搞]]，但音樂部份仍然不變，由[https://web.archive.org/web/20070622094408/http://www.hkhahaha.com/ 香港哈哈哈]剪輯，段涉及的事件有「撐港台」遊行，桃色事件以及泛民主派事後的反應。

==下台==
2007年7月5日，朱培慶與一名操非本地口音廣府話、化上艷妝的女士在[[銅鑼灣|銅鑼灣]][[伊利莎伯大廈|伊利莎伯大廈]]一間夜店MUST KARA門外挽手同行。該女士後來被揭發為一出身[[四川|四川]]，自稱21歲（有報道為自稱23歲），姓劉洋名Coco的風塵女子。

當時正值歌手[[鍾鎮濤|鍾鎮濤]]於附近舉行小型音樂會，吸引一群娛樂記者[[採訪|採訪]]。有[[記者|記者]]認出路過的朱培慶，立即舉機[[攝影|拍攝]]，朱培慶當時略有醉意，對大批記者的出現顯得錯愕，一度蹲下躲在女伴身後，忙亂間拿走女伴的棕色[[假髮|假髮]]，轉身走入一間[[餐廳|餐廳]]，舉止狼狽。這過程被記者一一拍下，翌日成為多份報章的[[頭條新聞|主要新聞]]<ref>[http://hk.news.yahoo.com/070705/60/2aw2y.html 朱培慶夜擁艷女驚惶奔逃 (星島日報2007月7月6日)] {{webarchive|url=https://web.archive.org/web/20070710184335/http://hk.news.yahoo.com/070705/60/2aw2y.html |date=2007-07-10 }}</ref><ref>[http://www1.appledaily.atnext.com/template/apple/art_main.cfm?iss_id=20070706&sec_id=4104&subsec_id=12731&art_id=7293998 朱培慶艷女同行狼狽避傳媒 (蘋果日報2007年7月6日] {{webarchive|url=https://web.archive.org/web/20080606211214/http://www1.appledaily.atnext.com/template/apple/art_main.cfm?iss_id=20070706&sec_id=4104&subsec_id=12731&art_id=7293998 |date=2008年06月6日 }}</ref>。

事後，輿論一面倒質疑他[[召妓|召妓]]。有報章指他并非首次找Coco，有報章更指控有人代他支付四千元「出街費」，惹來收受利益的揣測。在眾說紛雲中，7月9日下午，朱培慶召開記者會，他表示，雖然他原定2007年開始退休前休假，並於2008年中[[退休|退休]]，但現在已向政府提出希望儘早離職。他同時否認有人代他支付4000元費用，但卻以不涉公務為由，表明不會公開談論召妓問題。這是繼[[羅范椒芬|羅范椒芬]]後，兩個月內第二名首長級官員下台。<ref> [http://www.mpinews.com/htm/INews/20070709/gb11720c.htm 朱培慶宣告希望儘早離職 (明報2007年7月9日)] {{webarchive|url=https://web.archive.org/web/20071010123406/http://www.mpinews.com/htm/INews/20070709/gb11720c.htm |date=2007年10月10日 }}</ref> <ref>有線新聞關於朱培慶事件的報道</ref>。

對於當日狼狽的舉動，他以[[英語|英語]]向市民作出忠告：「The only comment I would make is that alcohol is not very conducive to good behaviour. So my advice has been drink less.（我只會說[[酒精|酒精]]並不導人向善，我建議少飲些。）」。而對於因被懷疑[[召妓|召妓]]醜聞而需要下台，他以「我係無悔」來作回應。<ref>[http://orientaldaily.on.cc/new/new_f12cnt.html 道亦有度：朱培慶摑記協嘴巴] {{webarchive|url=https://web.archive.org/web/20070707031529/http://orientaldaily.on.cc/new/new_f12cnt.html |date=2007-07-07 }}</ref>

2007年9月9日特區政府正式批准涉及「艷女門」事件的廣播處處長朱培慶提早退休的申請。當局認為毋須向朱培慶展開紀律聆訊，朱將可於明年8月結束所有休假，正式離開政府，屆時可根據退休金條例領取退休金。有估計朱培慶是次可領取高達900萬元的退休金，而在退休前休假的這段時間，亦可繼續領取每月16萬元的薪金。同時，當局會透過公開招聘及內部聘任的程序，招聘適當人士出任廣播處長一職，任期兩年半，而副廣播處長[[傅小慧|傅小慧]]會繼續署任廣播處長一職，直至新廣播處長上任。2008年8月，[[黃華麒|黃華麒]]出任新廣播處長。

==婚姻==
他[[太太|太太]]是[[劉家韾|劉家韾]]，出身自[[無線新聞|無線新聞部]]，曾任職[[平等機會委員會|平等機會委員會]][[公共關係|機構傳訊]]及培訓主管，因朱培慶的桃色事件而辭職。

==榮譽==
朱培慶1983年當選為[[香港十大傑出青年|香港十大傑出青年]]<ref>都市日報2007年7月10日 朱培慶「失措」落馬 提早退休</ref>，1994年3月23日獲委任為官守[[太平紳士|太平紳士]]。

朱培慶尋歡事件曝光後，獲各方封為「歡樂天地」，因為佔了很多天報紙頭版。
