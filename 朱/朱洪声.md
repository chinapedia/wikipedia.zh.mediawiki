
'''朱洪声'''，[[中国天主教|中国天主教]][[神父|神父]]，[[耶穌會|耶穌會]][[神父|神父]]。

朱洪声神父出生于中国上海，世代信仰天主教。其父朱季琳。著名企业家[[朱志尧|朱志尧]]和海门教区主教[[朱开敏|朱开敏]]均为其伯父，著名民国教育家和复旦大学创始人[[马相伯|马相伯]]也为其舅公。1933年，少年朱洪声随主教伯父去欧洲朝圣，便留在比利时入读当地的[[耶稣会|耶稣会]]中学，后来在法国进入耶稣会，1944年被祝圣为神父，曾游学爱尔兰和美国。<ref> 耶稣的知友，朱洪声和蔡石方两位神父合传，p21-23 </ref>。1947年至1949年任[[徐汇中学|徐汇中学]]理学（教务长）。1950年任[[君王堂|君王堂]]副[[本堂神父|本堂神父]]<ref>[http://www.shtong.gov.cn/newsite/node2/node4/node2249/luwan/node37494/node37507/node62749/userobject1ai22515.html 卢湾区志 >> 第三十二编民族·宗教 >> 第二章宗教]</ref>，1955年9月8日夜，朱洪声神父和[[龚品梅|龚品梅]]主教、和其他30多名神父同时被捕。1960年3月17日被判刑，刑期满后留场就业，1979年初，朱洪声和陈天祥、[[朱树德|朱树德]]等人被告秘密组织[[中国天主教地下教会|地下教会]]，1980年3月组织信徒赴上海[[佘山|佘山]]朝圣。1981年11月19日，[[上海市公安局|上海市公安局]]按照[[中华人民共和国公安部|中华人民共和国公安部]]的命令，将其作为“[[朱洪声反革命集团案|朱洪声反革命集团案]]”的首要分子执行[[逮捕|逮捕]]<ref>[http://www.shtong.gov.cn/newsite/node2/node2245/node4523/node28755/node28805/node60670/userobject1ai13638.html 中共上海党志 >> 第八篇统一战线 >> 第五章贯彻民族和宗教政策 >>第二节　宗教工作]</ref>。1983年3月21日，[[上海市中级人民法院|上海市中级人民法院]]公开审理“朱洪声反革命集团案”<ref>[http://www.shtong.gov.cn/newsite/node2/node2245/node75195/node75199/index.html 上海宗教志 >> 大事记] {{webarchive|url=https://web.archive.org/web/20160327180531/http://www.shtong.gov.cn/newsite/node2/node2245/node75195/node75199/index.html |date=2016-03-27 }}</ref>，指责朱制造“[[佘山|佘山]][[圣母|圣母]]发光”政治事件，判处有期徒刑15年。1987年9月，又“根据公安部指示”，将朱洪声减刑释放<ref>[http://www.shtong.gov.cn/newsite/node2/node2245/node4476/node58282/node59920/index.html 上海公安志 >> 第二编惩治反革命罪犯 >> 第四章打击以宗教作掩护的反革命分子] {{webarchive|url=https://web.archive.org/web/20110421035234/http://www.shtong.gov.cn/newsite/node2/node2245/node4476/node58282/node59920/index.html |date=2011-04-21 }}</ref>。
1993年7月6日，朱洪声神父在上海一家医院去世。去世前不久的一天，有关政府部门在医院向朱洪聲神父及家属宣告了撤销所有判决的通知书，并给予“平反”。
