'''裴贽'''（{{bd|9世纪||905年|7月5日|catIdx=P}}），字'''敬臣'''，[[唐朝|唐朝]]官员，[[唐昭宗|唐昭宗]]年间于900年-903年间为[[宰相|宰相]]。[[天祐_(唐朝)|天祐]]二年（905年），军阀宣武军节度使[[朱全忠|朱全忠]]准备篡位，对高官实施大清洗。裴贽没有成为第一波清洗的目标，但最终还是被贬并遭赐死。

== 家世及早期仕途 ==
裴贽生年不详，出自中眷裴，祖父[[裴稷|裴稷]]任[[虔州|虔州]][[刺史|刺史]]，父[[裴储|裴储]]任大理丞。裴稷侄[[裴坦|裴坦]]在[[唐僖宗|唐僖宗]]年间为宰相。<ref>《新唐书》卷七十一 {{cite web |url=http://www.sidneyluo.net/a/a17/071.htm |title=存档副本 |accessdate=2010-05-03 |0=2010-06-13 |deadurl=yes |archiveurl=https://web.archive.org/web/20100613021529/http://sidneyluo.net/a/a17/071.htm |archivedate=2010-06-13 }}{{cite web |url=http://www.sidneyluo.net/a/a17/table/form34.htm |title=存档副本 |accessdate=2011-10-11 |deadurl=yes |archiveurl=https://web.archive.org/web/20120301171054/http://www.sidneyluo.net/a/a17/table/form34.htm |archivedate=2012-03-01 }}</ref>

裴贽中[[进士|进士]]后，曾任司勋员外郎，[[中和_(唐朝)|中和]]二年（882年）正月，宰相[[王铎_(唐朝)|王铎]]被任为讨伐农民变军[[黄巢|黄巢]]的统帅诸道行营都都统，以裴贽为掌书记。<ref>[[:s:新唐書/卷185|《新唐书》卷一百八十五]]</ref><ref>[[:s:資治通鑑/卷254|《资治通鉴》卷二百五十四]]误作裴贽为裴坦子。</ref>裴贽历任[[右补阙|右补阙]]、[[御史中丞|御史中丞]]，[[大顺_(唐朝)|大顺]]二年（891年）正月以侍郎主管贡举放榜。<ref>[[:s:旧五代史/卷24|《旧五代史》卷二十四]]</ref>[[乾宁|乾宁]]四年（897年）十月，以[[太中大夫|太中大夫]]、前御史中丞为[[礼部尚书|礼部尚书]]、知贡举。[[光化|光化]]三年（900年）正月又任[[刑部尚书|刑部尚书]]。<ref name=BT20-1>[[:s:旧唐书/卷20上|《旧唐书》卷二十上]]</ref><ref name=NBT182>[[:s:新唐書/卷182|《新唐书》卷一百八十二]]</ref>

== 拜相 ==
九月，裴贽以正议大夫、守刑部尚书、[[上柱国|上柱国]]、'''河东县开国男'''、食邑三百户、赐紫金鱼袋被昭宗任为[[中书侍郎|中书侍郎]]，兼刑部尚书、[[同中书门下平章事|同中书门下平章事]]，充集贤殿大学士，拜为宰相，<ref name=BT20-1/><ref name=NBT10>[[:s:新唐书/卷10|《新唐书》卷十]]</ref><ref name=ZZTJ262>[[:s:資治通鑑/卷262|《资治通鉴》卷二百六十二]]</ref>不久兼[[户部尚书|户部尚书]]。昭宗曾怀疑裴贽表面道貌岸然，实则乱性，问所信任的[[翰林学士|翰林学士]][[韩偓|韩偓]]的看法。韩偓指出之所以有这样的流言，是因为裴贽和族人亲善，允许其中多人住在自己府邸，这些人和外人的关系对裴贽造成了不良影响。韩偓知道唐昭宗每听到[[咸通|咸通]]年间事都会触动，还说到裴贽是[[唐懿宗|唐懿宗]]咸通年间大臣裴坦的堂侄。唐昭宗没有处置裴贽。<ref name=NBT182/>[[天复_(唐朝)|天复]]元年（901年），昭宗刚在政变后复位，秘密问韩偓“陆扆、裴贽谁忠于我？”韩偓为陆扆辩解。<ref>[[:s:新唐書/卷183|《新唐书》卷一百八十三]]</ref>同年，昭宗被以[[韩全诲|韩全诲]]为首的当权宦官劫持到其盟友军阀[[李茂贞|李茂贞]]治下的[[凤翔|凤翔]]，于十二月诏裴贽充[[大明宫|大明宫]]留守，<ref name=ZZTJ262/>后罢免。

== 罢相与身亡 ==
后来另一军阀宣武军节度使朱全忠打败李茂贞，迫使其屠杀宦官并将昭宗交给他，带昭宗回[[长安|长安]]。宰相[[崔胤|崔胤]]贬逐多名官员，只留下被认为孤立易制的裴贽和自己一同主政。<ref>[[:s:旧唐书/卷177|《旧唐书》卷一百七十七]]</ref><ref>[[:s:新唐書/卷223下|《新唐书》卷二百二十三下]]</ref>三年（903年）十二月，裴贽被罢相为[[左仆射|左仆射]]，相位被[[独孤损|独孤损]]取代。<ref name=ZZTJ263>[[:s:資治通鑑/卷263|《资治通鉴》卷二百六十三]]</ref><ref name=NBT182/>

朱全忠为准备夺位，杀唐昭宗，立昭宗子为[[唐哀帝|唐哀帝]]。天祐二年（905年）二月，裴贽议迁庙，迁昭宗与[[唐顺宗|唐顺宗]]为一室，<ref>[[:s:请祧顺宗奏|:s:请祧顺宗奏]]</ref>获准。后以[[司空|司空]]致仕。四月，以奉山陵之劳赐裴贽等人一子八品正员官。六月，朱全忠近臣[[柳璨|柳璨]]、[[李振|李振]]说服其对唐朝来自士族的高官实施大清洗。第一波清洗导致前宰相[[裴枢|裴枢]]、独孤损、[[崔远|崔远]]、[[陆扆|陆扆]]等人丧命，裴贽并不在目标内；但同月不久，他以[[特进|特进]]、守司空致仕、上柱国、'''河东县开国公'''、食邑二千户的身份被贬为[[青州|青州]]司户参军，<ref>[[:s:贬裴贽李煦敕|:s:贬裴贽李煦敕]]</ref>裴贽甥卫尉少卿[[敬沼|敬沼]]受累获罪贬徐州萧县尉。<ref>[[:s:贬敬沼敕|:s:贬敬沼敕]]</ref>[[太子宾客|太子宾客]][[柳逊|柳逊]]因与裴贽为刎颈之交，被以本官致仕。<ref>[[:s:罢柳逊诏|:s:罢柳逊诏]]</ref>当月，裴贽等百余人被赐死。<ref name=NBT10/><ref>[[:s:旧唐书/卷20下|《旧唐书》卷二十下]]</ref><ref>[[:s:新五代史/卷1|《新五代史》卷一]]</ref><ref name=ZZTJ265>[[:s:資治通鑑/卷265|《资治通鉴》卷二百六十五]]</ref>

[[后唐庄宗|后唐庄宗]][[同光|同光]]二年（924年）六月，赠裴贽[[司徒|司徒]]。<ref>[[:s:旧五代史/卷32|《旧五代史》卷三十二]]</ref>[[后晋|后晋]][[天福_(后晋)|天福]]六年（941年）四月，监修国史[[赵莹|赵莹]]为了修史，奏请访求裴贽在任官的子孙或门生故吏。<ref>《[[五代会要|五代会要]]》</ref><ref>[[:s:论修唐史奏|:s:论修唐史奏]]</ref>
