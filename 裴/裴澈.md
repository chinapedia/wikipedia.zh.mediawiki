'''裴澈'''（{{bd|？||887年|4月6日|catIdx=P}}<ref name=NBT63>《新唐书》卷六十三</ref>），[[表字|字]]'''深源'''，[[中国|中国]][[唐朝|唐朝]]官员，[[唐僖宗|唐僖宗]]和僭位者[[李煴|李煴]]年间为宰相。李煴败亡后，他也因效力李煴而被诛。

== 家世 ==
裴澈生年不详。他出自东眷裴氏。<ref name=NBT71>《新唐书》卷七十一</ref>祖父[[裴肃|裴肃]]，[[唐德宗|唐德宗]]年间为[[浙东观察使|浙东观察使]]，父裴俅，谏议大夫，伯父[[裴休|裴休]]，[[唐宣宗|唐宣宗]]年间宰相。<ref name=NBT71/>但两唐书《裴休传》都提及裴俅而没有提及裴澈，<ref name=BT177>[[:s:舊唐書/卷177|《旧唐书》卷一百七十七]]</ref><ref name=NBT182>[[:s:新唐書/卷182|《新唐书》卷一百八十二]]</ref>可能是因为裴澈的下场。

== 效力唐僖宗 ==
[[唐僖宗|唐僖宗]][[广明_(唐朝)|广明]]元年（880年）十二月，[[黄巢|黄巢]]的大规模农民变军即将逼近京城[[长安|长安]]，首相[[卢携|卢携]]被当权[[宦官|宦官]][[田令孜|田令孜]]归咎，被罢相，因而[[自杀|自杀]]。时任翰林学士、户部侍郎的裴澈被任为工部侍郎，与新拜户部侍郎的翰林学士承旨、尚书左丞[[王徽_(京兆)|王徽]]一同授[[同中书门下平章事|同中书门下平章事]]，拜为宰相。<ref name=BT19-2>[[:s:旧唐书/卷19下|《旧唐书》卷一十九下]]</ref>但当天即传来黄巢将攻长安的消息，田令孜兄长[[陈敬瑄|陈敬瑄]]时任[[西川节度使|西川节度使]]，田令孜带僖宗逃往西川军部[[成都|成都]]。裴澈起初不及随驾，但[[中和_(唐朝)|中和]]元年（881年）二月也来到成都。以[[金紫光禄大夫|金紫光禄大夫]]、守[[中书侍郎|中书侍郎]]兼[[礼部尚书|礼部尚书]]、同中书门下平章事、[[上柱国|上柱国]]被任为[[特进|特进]]、[[门下侍郎|门下侍郎]]兼[[兵部侍郎|兵部侍郎]]、同中书门下平章事、监修国史，勋封如故。<ref>[[:s:王铎宏文馆大学士等制|乐朋龟《王铎宏文馆大学士等制》]]</ref>十一月，为检校兵部尚书、[[御史大夫|御史大夫]]、充[[鄂岳观察使|鄂岳观察使]]。<ref name=ZZTJ254>[[:s:資治通鑑/卷254|《资治通鉴》卷二百五十四]]</ref><ref name=NBT9>[[:s:新唐書/卷9|《新唐书》卷九]]</ref><ref>《[[:s:裴胤鄂岳观察使制|乐朋龟《裴胤鄂岳观察使制》]]》，[[郁贤皓|郁贤皓]]《[[唐刺史考全编|唐刺史考全编]]》怀疑“裴胤”为“裴澈”之误。</ref>但当时鄂州尚有[[刺史|刺史]][[崔绍|崔绍]]，疑裴澈未赴任。<ref>郁贤皓《唐刺史考全编》</ref>

后来黄巢兵败，唐军收复长安，三年（883年），僖宗开始准备回京，时任首相[[郑畋|郑畋]]冒犯田令孜、陈敬瑄而被罢相。七月，裴澈时任兵部尚书、判度支，被任为中书侍郎、同平章事，再度拜相。<ref name=ZZTJ255>[[:s:資治通鑑/卷255|《资治通鉴》卷二百五十五]]</ref><ref>[[乐朋龟|乐朋龟]]《[[:s:西川青羊宫碑铭|西川青羊宫碑铭]]》称裴澈为兵部尚书平章事。</ref>淮南节度使[[高骈|高骈]]命从事[[崔致远|崔致远]]给裴澈写信道贺。<ref>[[:s:度支裴彻相公|《度支裴彻相公》]]</ref>[[光启|光启]]元年（885年）十月，田令孜激怒[[军阀|军阀]][[河中节度使|河中节度使]][[王重荣|王重荣]]和[[河东节度使|河东节度使]][[李克用|李克用]]导致二人进攻长安，次年正月，僖宗再次逃离长安，奔[[兴元|兴元]]。因田令孜被认为弄权、腐败，且僖宗此次出逃在夜间，故少有官员随行，裴澈和同僚宰相工部侍郎、判度支[[萧遘|萧遘]]、[[郑昌图|郑昌图]]等文武百官不知僖宗出奔，都没有随驾。<ref name=ZZTJ256>[[:s:資治通鑑/卷256|《资治通鉴》卷二百五十六]]</ref>太子少保[[孔纬|孔纬]]奉命来催百官随驾，来见宰相，萧遘、裴澈因田令孜在僖宗左右，不想随驾，称病不见。<ref name=BT19-2/><ref name=BT179>[[:s:舊唐書/卷179|《旧唐书》卷一百七十九]]</ref><ref>[[:s:新唐書/卷163|《新唐书》卷一百六十三]]</ref>

== 效力李煴 ==
萧遘因厌恶田令孜，与裴澈合计，写信召起初联合田令孜但后也反对他的另一军阀[[静难军节度使|静难军节度使]][[朱玫_(唐朝)|朱玫]]勤王。<ref name=BT179/><ref>[[:s:新唐書/卷101|《新唐书》卷一百零一]]</ref>朱玫接管长安，准备另立僖宗曾祖父辈远亲李煴为帝。他先奉李煴为嗣襄王，四月，与萧遘、裴澈率群僚奉李煴权监军国事、承制封拜指挥，以兵部侍郎判户部郑昌图为同平章事、判度支、盐铁、户部；<ref>[[:s:舊唐書/卷175|《旧唐书》卷一百七十五]]</ref>五月，朱玫加自己[[侍中|侍中]]、诸道盐铁、转运等使，加裴澈判度支，郑昌图判户部。又以裴澈为[[门下侍郎|门下侍郎]]、右仆射、同平章事、判度支。<ref name=BT19-2/><ref name=NBT82>[[:s:新唐書/卷82|《新唐书》卷八十二]]</ref>十月，李煴称帝，裴澈和郑昌图继续为宰相。<ref name=ZZTJ256/>

== 伏诛 ==
但李煴的任期短。朱玫部将[[王行瑜|王行瑜]]受命攻兴元俘虏僖宗，却不能成功，害怕被朱玫处罚。十二月，代田令孜为左神策中尉、观军容使的[[杨复恭|杨复恭]]传檄关中，称杀朱玫者将被任为静难节度使。王行瑜因而背弃朱玫，突然回到长安。朱玫闻讯，不知王行瑜所欲，召之，责以擅归。王行瑜执杀朱玫，部下军兵劫掠京城。裴澈、郑昌图与约二百官员奉李煴经东渭桥奔河中，意欲寻求王重荣庇护。王重荣诈作奉迎，却将李煴执杀，并将裴澈、郑昌图等下狱。<ref name=BT19-2/><ref name=ZZTJ256/><ref name=NBT82/>李克用派出勤王的[[榆次|榆次]]镇将[[李承嗣|李承嗣]]安定京城，得到裴澈、郑昌图。<ref>[[:s:旧五代史/卷55|《旧五代史》卷五十五]]</ref>

三年（887年）三月，僖宗下诏就地处决裴澈、郑昌图和虽然受李煴拜相但没有积极参与建立政府的萧遘。<ref name=NBT9/><ref>[[:s:诛萧遘等敕|:s:诛萧遘等敕]]</ref>三人被斩于[[岐山县|岐山县]]。<ref name=ZZTJ256/><ref name=NBT82/>
