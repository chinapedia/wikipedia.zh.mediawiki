'''鈉依賴型葡萄糖共同運輸蛋白'''（Sodium-dependent glucose cotransporters，簡稱'''SGLT'''），為一類{{tsl|en|glucose transporter|葡萄糖運輸蛋白}}。本類蛋白質可能會分布於[[小肠|小肠]]（SGLT1）以及[[腎元|腎元]]的{{tsl|en|proximal tubule|近端小管}}，其中{{tsl|en|Proximal convoluted tubule|近曲小管}}的SGLT屬於SGLT2；{{tsl|en|Proximal straight tubule|近直小管}}則為SGLT1，可協助{{tsl|en|renal glucose reabsorption|腎臟葡萄糖再吸收|腎臟再吸收葡萄糖}}。腎小管會藉由本蛋白將濾液中的葡萄糖完全再吸收（近曲小管98%）。

== 類別 ==
SGLT家族中最知名的兩種蛋白為 SGLT1 及 SGLT2，屬於 SLC5A 基因家族。除了這兩種蛋白之外，該蛋白家族還另有其他五種蛋白<ref>[http://www.ensembl.org/Homo_sapiens/familyview?family=ENSF00000000509 Ensembl release 48: Homo sapiens Ensembl protein family ENSF00000000509<!-- Bot generated title -->]</ref>。

== SGLT2抑制劑與糖尿病的治療 ==
{{Main|鈉-葡萄糖協同轉運蛋白2抑制劑}}
SGLT2抑制劑又稱「''gliflozins''」<ref>{{cite web|url=http://www.diabetes.co.uk/diabetes-medication/sglt2-inhibitors.html|title=SGLT2 Inhibitors (Gliflozins)|publisher={{tsl|en|Diabetes.co.uk||Diabetes.co.uk}}|accessdate=2015-05-19}}</ref>，常用於治療[[2型糖尿病|2型糖尿病]]。此類藥物包含{{tsl|en|dapagliflozin|dapagliflozin}}、{{tsl|en|canagliflozin|canagliflozin}}，以及[[恩格列淨|恩格列淨]]等。

== 功能 ==
近端小管[[細胞膜|細胞]]基底膜側上的[[鈉鉀泵|鈉鉀泵]]會消耗ATP泵出鈉離子，同時將鉀細胞泵入細胞，製造細胞內外的鈉鉀濃度梯度。此時SGLT蛋白便可利用鈉離子濃度差，將[[葡萄糖|葡萄糖]]輸入細胞中。在此過程中，SGLT並沒有消耗ATP，但是乃利用鈉鉀泵主動運輸的濃度差進行運輸，因此屬於[[主動運輸|次級主動運輸]]。另外，由於鈉離子的輸入方向與葡萄糖相同，因此屬於同向運輸。
