'''齐白石'''（{{bd|1864年|1月1日|1957年|9月16日|catIdx=Q齊}}），[[湖南|湖南]][[湘潭|湘潭]]人，是当代[[中国|中国]]的[[中國畫|國畫]][[画家|画家]]。原[[名|名]]'''齐纯芝'''，字'''渭清'''，[[祖父|祖父]]取号'''兰亭'''，老师取名'''齐璜'''，号'''濒生'''，[[别号|别号]]'''寄萍老人'''、'''白石山人'''，後人常将“山人”二字略去，故后常号“白石”。齐白石也和[[张大千|张大千]]并称“'''南张北齐'''”。

== 生平 ==
齊白石於1864年[[元旦|元旦]]（同治二年十一月廿二日）出生於[[湖南省|湖南省]][[长沙府|长沙府]]湘潭县（今湖南省[[湘潭县|湘潭县]]）[[白石铺|白石铺]][[杏子坞|杏子坞]]农民家庭，原名'''齊純芝'''。自幼體弱多病。由於家族以務農為生，祖父齊萬秉認為農耕不能賺錢，於齊白石4歲時起教授寫字。於8歲時在[[公公|公公]]所開設之蒙館學習，半年後輟學，輟學後，協助家中務農。14岁起作木匠，學習雕花木工。後來兼習繪畫，並拜蕭薌陔為師。25歲時起拜名士[[胡沁園|胡沁園]]、[[陳少蕃|陳少蕃]]等為師<ref>齐白石，《白石老人自述》，第四章，第66-67页。</ref>，由胡沁園替之取名為璜，號瀕生，因家中靠近白石鋪，故取別號白石山人。學習詩、書、畫、篆刻。並開始兼以賣畫為生，不再以雕花木工賺錢。
[[File:Ci-Byshi_cikada.jpg|thumb]]
32歲時起對刻印產生濃厚興趣，開始向名家學習刻印。35歲時拜學者[[王湘綺|王湘綺]]為師。1903年39岁时第一次到北京。40岁時，開始周遊天下，以后到南北各地游历，饱览名山大川，使他開闊了眼界，師法大自然，充實了作品「造化」內容。1905年41歲時在当时广西首府[[桂林|桂林]]卖画为生，認識了[[蔡锷|蔡锷]]和張中正和尚（即[[黃興|黃興]]）。42歲時起遊覽[[廣州|廣州]]、[[廣西|廣西]]、[[重慶|重慶]]、[[越南|越南]]、[[上海|上海]]、[[南京|南京]]，期間，寫畫風格由工筆畫轉為[[寫意畫|寫意畫]]。

1917年起決定於[[北京|北京]]發展，以卖画刻印为生，並結識了[[陳師曾|陳師曾]]（[[陈寅恪|陈寅恪]]之兄），陈师曾曾对[[胡佩衡|胡佩衡]]說齐白石的作品“思想新奇，不是一般画家能画得出来的……我们应该特别帮助这位乡下老农，为他的绘画宣传”。1922年，陈师曾携齐白石的画作前往日本，在东京的中日联合绘画展览会上展出，使得齐白石在日本名噪一时，画价亦爆增。齐白石在其自传中写道：“二尺长的纸，卖到二百五十银币……还听说法国人在东京，选了师曾和我两人的画，加入巴黎艺术展览会……我做了一首诗，作为纪念：‘曾点胭脂作杏花，百金尺纸众争夸；平生羞杀传名姓，海国都知老画家。’……从此以后，我卖画生涯，一天比一天兴盛起来。这都是师曾提拔我的一番厚意”。陈病故后，齐白石对[[张次溪|张次溪]]說：“我如没有师曾的提携，我的画名，不会有今天”<ref>{{cite web|url=http://www.people.com.cn/GB/14738/28490/28491/28493/1939241.html|title=谁是齐白石的伯乐？画坛领袖陈师曾|author=林木|publisher=人民网|accessdate=2017-02-01}}</ref>。受陳師曾影響，他創造出自成一家的畫畫風格，亦即紅花墨葉的大膽風格，以原本不協調的純紅色和深墨黑用在一起，形成畫面鮮明的對比，表現了畫面上清新、樸實的感覺，实现了“衰年变法”。並學習[[徐渭|徐渭]]、[[石濤|石濤]]，在学习石涛、[[朱耷|朱耷]]、徐渭的基础上，吸收[[吴昌硕|吴昌硕]]的技法，終於自成一家。後結識[[梅蘭芳|梅蘭芳]]、[[艾青|艾青]]、[[徐悲鴻|徐悲鴻]]等人。1922年，在陳師曾協助下，其作品起於外國展出，受到重視。後受[[林風眠|林風眠]]邀請，起到[[北京藝術專門學校|北京藝術專門學校]]教授中國畫。也曾在[[张恨水|张恨水]]的[[北平私立北华美术专科学校|北平私立北华美术专科学校]]教授国画。

据说，侨居海外的[[李鐵夫|李铁夫]]曾致友人信中曾认为当代国内真正的画家只有「齐白石、[[吳昌碩|吴昌硕]]二人而已」，可見其對齊白石十分推崇。

1937年，[[日本|日本]]入侵中國，齊白石因在外國享譽，受日本人拉攏，堅拒屈服，更遭扣押。後日本當局懼怕負上迫害藝術家罪名，扣押三天後釋放。

1949年，[[中華人民共和國|中華人民共和國]]成立，齊白石因是木匠出身，以及与[[毛泽东|毛泽东]]有同乡之谊受到中共表揚。並被聘為[[中國美術學院|中國美術學院]]名譽教授。1953年他被选为中国美术家协会[[主席|主席]]。1954年被选为[[第一届全国人民代表大会|第一届全国人民代表大会]]代表。1955年，[[德意志民主共和国|東德]]總理訪問中華人民共和國，代表德國藝術科學院，授予齊白石共和国艺术科学院通讯院士的荣誉状。1956年，世界和平理事会决定他为1955年度国际和平奖金的获得者。1957年出任北京中國畫院名譽院長，同年9月16日在[[北京|北京]]逝世。享年93岁。

== 畫風 ==
[[File:墨蝦圖.jpg|thumb]]]]

齐白石强调“作画妙在似与不似之间，太似为媚俗，不似为欺世。”，在篆刻方面自称“三百石印富翁”。晚年仍然细心观察虾、蟹、青蛙的游泳姿态，他的画自成章法，人物、山水、花鸟、草虫、蔬果、玩具都可入画，和20世纪世界[[绘画|绘画]][[艺术|艺术]]的发展方向相符。

== 纪念 ==
[[File:The_Soviet_Union_1958_CPA_2116_stamp_(Qi_Baishi).jpg|thumb]]
[[File:Qibaishi_gongyuan.jpg|thumb]]
齐白石的出生地湖南湘潭和晚年的居住地分别开辟了[[齐白石故居_(湖南湘潭)|湖南湘潭齐白石故居]]和[[齐白石故居_(北京西城)|北京西城齐白石故居]]，前者是[[全国重点文物保护单位|全国重点文物保护单位]]，后者是[[北京市文物保护单位|北京市文物保护单位]]。“[[齐白石墓|齐白石墓]]”位于北京市[[海淀区|海淀区]]金山陵园，墓碑碑文为[[李苦禅|李苦禅]]所书。

另外，水星上位於4.2°S, 196.0°W的撞擊坑命名為[[齊白石撞擊坑|齊白石撞擊坑]]<ref>[http://messenger.jhuapl.edu/gallery/sciencePhotos/image.php?page=1&gallery_id=2&image_id=295 Low-angle Impacts: A Look at Qi Baishi and Hovnatanian] {{webarchive|url=https://web.archive.org/web/20140201205646/http://messenger.jhuapl.edu/gallery/sciencePhotos/image.php?page=1&gallery_id=2&image_id=295 |date=2014-02-01 }}</ref>。

== 弟子 ==
齐白石的弟子很多，其中著名的有李苦禅、李可染、王雪涛、王漱石、王铸九、许麟庐、陈大羽、李立、娄师白、董长青、张德文、陈国刚、秦慕儒等，其三子齐子如、戏剧大师梅兰芳、评剧名家新凤霞也是齐白石的弟子。

== 家庭 ==
