'''黃懷萱'''（{{bd|1997年|7月7日|||catIdx=Huang Huaixuan}}）是[[臺灣|臺灣]]的一名運動選手，擅長[[跆拳道|跆拳道]]，目前就讀[[輔仁大學|輔仁大學]]，曾獲[[2014年夏季青年奧林匹克運動會|2014年青年奧運]]女子49公斤以下級金牌<ref>{{cite news |language = zh-tw|url = http://www.appledaily.com.tw/appledaily/article/sports/20160513/37211800/ |title = 黃懷萱愛相隨 莊佳佳生日願：奪牌 |publisher = [[台灣蘋果日報|台灣蘋果日報]] |date = 2016-05-13 |accessdate = 2016-08-07}}</ref><ref>{{Cite web |url = http://www.nstc.org.tw/2016olympic/athlete08.html |title = 奧運國手小檔案 |author = 國家運動訓練中心 |language = zh-tw |accessdate = 2016-08-07}}</ref>。

她曾在2016年代表[[2016年夏季奧林匹克運動會中華台北代表團|中華台北]]參加[[2016年夏季奧林匹克運動會|2016年里約奧運]]女子跆拳道的49公斤級比賽。<ref>{{cite news |language = zh-tw|url = http://www.appledaily.com.tw/realtimenews/article/new/20160512/859658/ |title = 18歲甜心奧運國手黃懷萱　跟男友公開放閃 |publisher = [[台灣蘋果日報|台灣蘋果日報]] |date = 2016-05-12 |accessdate = 2016-08-07}}</ref>
