'''黃克競'''博士，[[OBE|OBE]]，[[CBE|CBE]]（{{Lang-en|'''Dr Haking Wong'''}}，{{bd|1906年||1996年|1月9日|catIdx=Huang黃克競}}），原名'''黃祖騋'''、字'''克競'''、以字行，[[香港|香港]]1960年代著名[[工業|工業]][[企業家|企業家]]，[[五邑|五邑]][[華僑|華僑]]，祖籍[[廣東|廣東]][[新會|新會]]。黃克競為[[香港中華廠商聯合會|香港中華廠商聯合會]]前會長，他於五十年代後期投資研發及製造光學儀器，被稱為「光學儀器大王」<ref name="rthk programme detail">{{Cite web |url=http://programme.rthk.hk/rthk/tv/programme.php?name=tv/hongkongstories19&d=2012-03-26&p=5266&e=172684&m=episode |title=香港故事 (第19輯) － 我們的品牌:廠出名堂 |accessdate=2013-09-01 |author= |date=2012-03-26 |publisher=[[香港電台|香港電台]] |language=zh-hk}}</ref>。同時，黃克競熱心於教育、公益事業，造福[[華人|華人]]社會，是一位[[慈善家|慈善家]]。

== 生平 ==
[[File:HKU_Haking_Wong_Building_foundation_stone.JPG|thumb]]捐建黃克競樓，1983年10月6日由時任[[港督|港督]]兼港大校監[[尤德|尤德]]爵士主持開幕儀式]]
黃克競在1906年生於香港，其父為著名[[泰國華人|泰國華僑]]富豪[[黄宣充|黃宣充]]，廣東新會[[雙水鎮|雙水鎮]]龍頭鄉人<ref name="appledaily news">{{Cite web |url=http://hk.apple.nextmedia.com/news/art/20100302/13776945 |title=騙80萬被追查　惱羞成怒　黃克競男孫打祖母認罪 |accessdate=2013-08-31 |author= |date=2010-03-02 |publisher=[[蘋果日報|蘋果日報]] |language=zh-hk}}</ref>。童年時居於[[泰國|泰國]]，至十一歲時返回香港，並入讀[[聖士提反書院|聖士提反書院]]。中學畢業之後，黃克競曾在包括電子、紡織、塑膠以及造船工業等多方面作出嘗試。在[[香港淪陷|香港淪陷]]時期，黃克競一度逃難至[[廣州|廣州]]，戰後再次回到香港<ref name='CPC Booklet'>{{cite book| last1 =高| first1 =添強| title =黃克競| chapter =第五章︰卜葬華人永遠墳場的著名人物| edition =1| work =高山景行 – 香港仔華人永遠墳場的建立與相關人物| publisher =[[華人永遠墳場管理委員會|華人永遠墳場管理委員會]]| date =2012-01-01| year =2012| location =香港| pages =47| url =http://www.bmcpc.org.hk/about/_doc/HillSceneChap5.pdf| accessdate =2013-09-01| deadurl =yes| archiveurl =https://web.archive.org/web/20131017155204/http://www.bmcpc.org.hk/about/_doc/HillSceneChap5.pdf| archivedate =2013-10-17}}</ref>。

[[香港1950年代|1950年代]]，香港開始發展[[香港製造業|製造業]]，黃克競和陳蕉琴女士於1956年共同創辦了「[[寶源基業有限公司|寶源基業有限公司]]」<ref name="W Haking corp info">{{Cite web |url=http://www.haking.com/home.htm |title=W.Haking Enterprises LTD |accessdate=2013-09-01 |publisher=寶源基業 |language=en}}</ref><ref name="rthk video">{{cite video |people= |date=2012-03-26 |title= 香港故事（第19輯）：廠出名堂 |url=http://programme.rthk.hk/rthk/tv/programme.php?name=tv/hongkongstories19&d=2012-03-26&p=5266&e=172684&m=episode |medium= |publisher= 香港電台 |accessdate= 2013-09-01 |time= |quote= |language=zh-yue}}</ref>。在後來的五至六年間，黃克競因想尋找較高競爭力及技術較高的工業，因由本來發展穩定的[[香港工業#塑膠業|塑膠業]]轉而製造[[光學儀器|光學儀器]]，並開始生產[[機械相機|機械相機]]，成香港首個相機製造商<ref name="rthk programme detail" /><ref name="rthk video" /><ref name="wenweipo">{{Cite web |url=http://paper.wenweipo.com/2009/08/22/HK0908220019.htm |title=黃克競男孫 涉毆祖母偷174萬 |accessdate=2013-08-31  |date=2009-08-22 |publisher=[[文匯報|文匯報]] |language=zh-hk}}</ref>。隨後，寶源基業成功研製首部百分百由香港製造的「[[夏蓮娜|夏蓮娜]]」相機，並出口到英國、美國等國際市場<ref name="rthk programme detail" />。由於黃克競在香港開創了相機及望遠鏡等的光學儀器製造的先河，他後被稱為「光學儀器大王」<ref name="rthk programme detail" />。至1993年，黃克競身家高達15億[[港元|港元]]<ref name="appledaily news" />。

黃克競一生曾任多個職銜，例如：[[東華三院|東華三院]]總理<ref>{{Cite web |url=http://www.tungwah.org.hk/upload/CH/organization/bd/bd1932.pdf |title=壬申年董事局 |accessdate=2013-09-01 |date=n.d. |format=PDF |publisher=[[東華三院|東華三院]] |language=zh-hk}}</ref>、[[保良局|保良局]]總理和[[香港中華廠商聯合會|香港中華廠商聯合會]]會長<ref>{{Cite web |url=http://www.cmass.edu.hk/index.php?option=com_content&view=article&id=9&Itemid=75 |title=創校簡史 |accessdate=2013-09-01 |date= |publisher=廠商會中學 |language=zh-hk |quote=，... 在許岳及黃克競兩位會長任內，捐款一百萬元 ...}}</ref><ref>{{Cite web |url=http://hkhiso.itsc.cuhk.edu.hk/history/node/7026 |title=香港中華廠商聯合會 廠商會訊 第146期 1978年11月 |accessdate=2013-09-01 |author= |format= |work= |publisher= |pages= |language=zh-hk |archiveurl= |archivedate=2010-02-19 |quote= ... 本會名譽會長黃克競捐港大六百萬元 ... |date=1978年11月}}</ref> 等。黃克競分別於1968年及1981年獲香港政府頒授[[OBE|OBE]]及[[CBE|CBE]]勳銜，及後在1980年獲[[香港大學|香港大學]]頒授名譽法學博士<ref name='CPC Booklet' /><ref name="wenweipo" />。

黃克競熱心於公益慈善工作，同時亦有資助教育事業；1977年他捐款成立了[[黃克競工業學院|黃克競工業學院]]（現稱為[[香港專業教育學院黃克競分校|香港專業教育學院黃克競分校]]）<ref name="appledaily news" />，復於1983年資助興建了[[香港大學|香港大學]]黃克競樓。除了在香港，黃克競亦有出資在新會興建了黃克競大橋，耗資1600萬元，在1992年11月建成通車。其後再有與弟弟黃祖棠合資興建紀念父親的黃宣充紀念大橋，以及紀念母親的黃張見紀念大橋<ref>{{Cite web |url=http://olepartner.blogspot.com.au/2011/04/blog-post_18.html |title=新會的潭江和黃克競大橋  |accessdate=2013-09-01 |date=2011-04-18 |publisher=Blogger |language=zh-hant}}</ref>。黃克競因振興新會有絕對貢獻，他於1992年獲當地政府封為「新會榮譽市民」。次年11月，他再被[[江門市|江門市]]授予「江門市榮譽市民」之稱<ref name='jmrb'>{{cite web | url = http://e.jmrb.com/m/2004/10/15/14/m_40666.shtml | title = “光学仪器大王”——黄克競 | accessdate = 2013-09-01 | date = 2004-10-15 | work = 江門日報 | language = zh-hans }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

== 家族 ==
黃克競生前有三名妻子，當中有第二任妻子湯慰珍及第三任妻子鍾海燕<ref>{{Cite web |url=http://hk.apple.nextmedia.com/news/art/20090822/13126136 |title=
黃克競孫涉盜竊打祖母11罪  |accessdate=2013-09-01 |date=2009-08-22 |publisher=蘋果日報 |language=zh-hk}}</ref>，共育有十一名子女<ref name="hkcd news">{{Cite web |url=http://www.hkcd.com.hk/content/2009-08/22/content_2374323.htm |title=黃克競孫涉毆傷繼祖母 |accessdate=2013-09-01 |date=2009-08-22 |publisher=[[香港商報|香港商報]] |language=zh-hk |deadurl=yes |archiveurl=https://web.archive.org/web/20150924030708/http://www.hkcd.com.hk/content/2009-08/22/content_2374323.htm |archivedate=2015-09-24 }}</ref>。可是在黃克競逝世後，其家族成員多次因訴訟而被報章報導。當中包括於2000年12月，黃克競的三名兒子黃允烜、黃允湛及黃允熙入稟[[高等法院|高等法院]]，申請將黃克競一手創辦的寶源基業有限公司申請[[清盤|清盤]]，最終導致公司流落到外人手中<ref name="hkcd news" /><ref>{{Cite web |url=http://orientaldaily.on.cc/cnt/news/20090822/00176_025.html |title=後人爭產斷送家族基業 |accessdate=2013-09-01 |date=2009-08-22 |publisher=[[東方日報|東方日報]] |language=zh-hk}}</ref>。

其姪子是香港80年代知名發展商商人[[黃振輝|黃振輝]]。
