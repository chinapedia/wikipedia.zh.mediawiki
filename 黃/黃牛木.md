==分佈及生境==
分佈於[[中國|中國]]、[[緬甸|緬甸]]、[[越南|越南]]、[[泰國|泰國]]、[[馬來西亞|馬來西亞]]、[[印尼|印尼]]及[[菲律賓|菲律賓]]等地，此物種的模式標本採自於越南。在東南亞，此物種的生境包括[[巽他荒原森林|巽他荒原森林]]和[[泥炭沼澤森林|泥炭沼澤森林]]。中國境內分佈於廣東、廣西、香港及雲南南部等地<ref name = "古樹名木">{{cite book|title=《香港古樹名木》|author=邢福武、丘國賢|series= |publisher=天地圖書|page=第96-97頁|isbn= 988-211-585-9|date=2006年11月}}</ref> 。多生長於丘陵或山地乾旱向陽坡上的次生林或灌木叢中<ref name = "香港植物名錄"/>，喜濕潤、酸性土壞<ref name = "古樹名木"/>，生長在[[海拔|海拔]]1240米以下，耐旱，萌發力強。<ref name = "中國植物志"/>

==形態特徵==
黃牛木是一種落葉灌木或小[[喬木|喬木]]，高約1.5－25[[米|米]]。全體無毛，樹皮光滑或有條紋，常以塊狀形式剝落使樹幹出現明顯的黃色、褐色或綠色的斑塊；樹幹底部有簇生的錐狀硬刺，枝條對生，幼枝淡紅色，無毛，幼枝略扁。葉披針形、長圓形或橢圓形，先端急尖或漸尖，基部鈍形至楔形，全緣，長約3－10.5厘米，寬約1－4厘米；單葉對生，長於枝的兩側略排列成一平面；葉柄長約2－3毫米；葉片紙質或簿革質，兩面均無毛，葉上面綠色，背面蒼白色，有透明腺點及黑點；葉側脈每邊約8－12條，兩面凸起，斜展，側脈末端不呈弧形閉合，葉中脈於葉面凹陷，下面則隆起，小脈呈網狀，兩面凸起。

[[花|花]]為聚傘[[花序|花序]]，腋生、腋外生及頂生，有花1－3朵；花兩性，輻射對稱；總花梗長約3－10[[毫毛|毫毛]]或以上；花直徑約1－1.5厘米，花梗長約2－3毫米；萼片5片，呈橢圓形，先端圓形，全面有黑色縱腺條，長約5－7毫米，寬約2－5毫米，結果時增大；[[花瓣|花瓣]]5片，粉紅、深紅至黃紅色，先端圓形，基部楔形，呈倒卵形，脈間有黑脈紋，無鱗片，長約5－10毫米，寬約2.5－5毫米；雄蕊粗短，合生成3，柄寬扁至細長，長約4－8毫米；下面肉質腺體3，長圓形至倒卵形，呈盔狀，先端增厚反曲，長約3毫米，寬約1－1.5毫米；[[子房|子房]]圓錐形，3室，無毛，長約3毫毛；花柱3，由基部叉開，呈線形，長2毫米。果為[[蒴果|蒴果]]，橢圓形，棕色，無毛，頂部開裂，有宿存花萼，長約8－12毫米，寬約4－5毫米。種子每室5－8顆，倒卵形，基部具爪，不對稱，一側有翅，長約6－8毫米，寬約2－3毫米。<ref name = "中國植物志"/> <ref name = "澳門植物園"/> <ref name = "馬灣公園"/>

==醫藥用途==
黃牛木[[根|根]]、嫩葉及[[樹皮|樹皮]]入[[藥|藥]]，味甘淡、微苦<ref>{{cite book|title=《香港草藥10徑遊》|author=李甯漢、鄭金順|series= |publisher=商務印書館 |page=第126頁|isbn= 962-07-3158-7|date=2003年7月}}</ref>
，性涼。能清熱解毒，化濕消滯，去瘀消腫，跌打損傷。可治感冒發熱、咳嗽聲嘶、腸炎、腹瀉，消化不良、黃疸等。<ref>{{cite book|title=《臺灣藥用植物名錄》|author=呂炎周|series= |publisher= |page=第451頁|isbn= 957-41-4049-0|date=1996年11月}}</ref>
<ref>[http://www.tcmlib.com/zy/html79/showdetail-373837312ce9bb84e7899be88cb62c7a79.html 黄牛茶]{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}意古中醫</ref><ref name = "中國本草彩色圖鑒"/><ref name = "實用草藥原色圖譜"/>中藥名為黃牛茶<ref name = "浸會大學"/>。
亦為[[涼茶|涼茶]][[廿四味|廿四味]]的主要材料之一<ref>[http://www.cuhk.edu.hk/scm/CMPromotionGroup/news/060626b.html 涼茶文化—廿四味]香港中文大學中醫學院中醫推廣小組，2006年6月26日</ref><ref> [http://www.tcmforum.com/forum2.php?reply=0&forumID=100691&fGroup=expert 「涼茶」何價？]，中醫大講堂</ref><ref >{{cite book|title=《香港中醫藥史記》|author=謝永光|series= |publisher=三聯書店出版|page=第80頁|isbn= 962-04-1483-7|date=1998年8月}}</ref>。

==用途==
黃牛木的木質堅硬紋理精緻，能供作雕刻及製作雀籠之用，故而成為別稱「鳥籠木」名稱的由來<ref name = "古樹名木"/><ref name = "澳門植物園"/><ref name = "馬灣公園"/><ref name = "福建植物志"/>。嫩葉可作為茶葉，幼果可供作烹調香料之用<ref name = "中國植物志"/>。樹冠圓整，枝葉較密，可作行道樹或觀賞樹<ref name = "古樹名木"/>。亦有一些遠足人士會削下細棵的黃牛木權充拐杖<ref name = "馬灣公園"/>
