'''黃葉病'''（又名'''萎蔫病'''、'''巴拿馬病'''）是出現在[[香蕉|香蕉]]根部的[[植物病理學|植物病]]。黃葉病由[[尖孢鐮刀菌|尖孢鐮刀菌]]古巴專化型（{{lang|la|''Fusarium oxysporum'' f.sp. ''cubense''}}）引起。尖孢鐮刀菌透過分泌毒素，引發香蕉出現[[細胞程序性死亡|細胞程序性死亡]]。<ref>{{Cite web |url = http://songshuhui.net/archives/50312 |title = 拿什麼保衛你，美味的香蕉（下） |author = 山要 |date = 2011年2月18日 |publisher = 科學松鼠會 |language = zh-hans |accessdate = 2014年3月2日}}</ref>此真菌對[[殺真菌劑|殺真菌劑]]有抵抗力，至今無藥可治。黃葉病又被稱為「香蕉癌症」，在2009年曾因此名稱在[[中國|中國]]傳出香蕉致癌的傳言，各地的香蕉價格慘跌。<ref>[http://news.xinhuanet.com/focus/2007-04/16/content_5968102.htm 新華網：「香蕉癌症」：原來竟然是一場價格謊言] {{webarchive|url=https://web.archive.org/web/20140201200550/http://news.xinhuanet.com/focus/2007-04/16/content_5968102.htm |date=2014-02-01 }}</ref>其實，黃葉病與食用香蕉的安全性沒有任何關聯，況且患上黃葉病的香蕉在收成前已經死亡，即使能結果，其果實也很小。

現代培植的香蕉都無法製造有用的[[花粉|花粉]]和沒有[[種子|種子]]，只能進行[[無性生殖|無性生殖]]，故現代大量種植的作物香蕉[[基因|基因]]都是一模一樣的，且無法透過[[遺傳變異|遺傳變異]]發展出對疾病的抗性。因此，黃葉病能感染蕉園之間的所有香蕉，造成大規模的失收。

1950年代，黃葉病在[[亞洲|亞洲]]以外的地區爆發，侵害當時香蕉的主要[[栽培品種|栽培品種]][[大米七|大米七]]（{{lang|en|Gros Michel}}），造成巨大損失，迫使蕉農改種其他能抵禦黃葉病的品種，主要是[[香芽蕉|香芽蕉]]（{{lang|en|Cavendish}}）。然而，1990年代黃葉病在[[東南亞|東南亞]]再度爆發，新發現在東南亞原生的4號小種能感染香芽蕉，使香蕉產業再一次面臨危機。

2014年4月，[[联合国粮农组织|联合国粮农组织]]发布消息，全球香蕉产量或因香蕉黄叶病病原菌（TR4）的扩散而大幅减少，部分香蕉出口国面临危机。<ref>{{Cite web|url = http://news.xinhuanet.com/world/2014-04/22/c_126420020.htm|title = 香蕉黄叶病肆虐致全球香蕉大幅减产|language = zh-hans|publisher = 新华网|accessdate = 2014年4月25日|date = 2014年4月22日|deadurl = yes|archiveurl = https://web.archive.org/web/20140511213722/http://news.xinhuanet.com/world/2014-04/22/c_126420020.htm|archivedate = 2014年5月11日}}</ref>
