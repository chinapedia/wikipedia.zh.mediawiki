'''黃金寶'''{{SBSHK}}{{BBSHK}}{{MH}}（{{lang-en|'''Wong Kam Po'''}}，{{bd|1973年|3月13日|catIdx=Wong黃金寶}}），[[香港單車代表隊|香港單車代表隊]]運動員，專攻公路賽，1998年[[曼谷亞運|曼谷亞運]]、2006年[[多哈亞運|多哈亞運]]、2010年[[廣州亞運|廣州亞運]]公路賽金牌得主，2007年[[馬略卡|馬略卡]][[場地單車世界錦標賽|場地單車世界錦標賽]]在不擅長的15公里[[捕捉賽|捕捉賽]]巧奪冠軍，首獲世界冠軍，得以披[[彩虹戰衣|彩虹戰衣]]，是一生贏得最頂級的大獎。其後，在2008年[[場地單車世界盃循迴賽|場地單車世界盃循迴賽]]洛磯山分站再在[[捕捉賽|捕捉賽]]摘冠，2009年則在循迴賽哥本哈根分站[[記分賽|記分賽]]摘冠。五度出征奧運，屢遭歐美選手團隊夾擊，而同國選手自顧不暇，最佳成績僅為第11名。其他獎項包括1997、2001、2009年中國[[全國運動會|全國運動會]]公路賽金牌等，1990－2000年代中國與亞洲首席公路賽運動員。

1992年曾退隊，後遇[[伯樂|伯樂]][[沈金康|沈金康]]總教練，在1994年廣島亞運公路賽獲第4名，自始光芒漸露，與1996年奧運[[滑浪風帆|滑浪風帆]]金牌得主[[李麗珊|李麗珊]]同為1990－2000年代最著名的兩位香港運動員。2007年獲世錦賽世界冠軍，時任中共總書記兼國家主席[[胡錦濤|胡錦濤]]在嘉獎「香港十大傑青」時盛讚「你比[[黃金|黃金]]還珍貴」，風頭一時無兩。

2017年11月，黃金寶離開香港單車隊，轉到港協暨奧委會任職。接受訪問時表示其實過去亦曾經「轉行」，於1992年巴塞隆拿奧運，那年黃金寶十九歲，第一次代表香港參與奧運。然而在當地集訓期間，與隊員發生衝突，事後被逐出港隊，無緣參與賽事。而於離隊的兩年間，轉行賣音響，一邊工作之餘，更曾準備移民。經過深思熟慮後，最後下定決心復出，才有後來的成就<ref>[http://bka.mpweekly.com/interview/%E5%A8%9B%E6%A8%82123/20171125-91923 【獨家專訪】黃金寶曾離隊賣音響　時代造就「亞洲第一」車神]</ref>。

== 職業生涯 ==
黃金寶與單車運動密不可分，1990年加入港隊，一年後勇奪亞青賽冠軍，但成名甚早的「阿寶」挫折亦來得早。他在[[沙田|沙田]][[瀝源邨|瀝源邨]]長大，17歲加入香港青年軍<ref>[http://hk.epochtimes.com/10/11/23/127937.htm 黃金寶亞運個人公路賽奪金]（第四段），2010年11月23日</ref>，1992年取得奧運入場券，但與隊友發生衝突被逐出港隊，無緣角逐巴塞隆拿奧運會之餘，更一度停止正規訓練，其職業生涯一度亮起警號。

直到1994年，曾任中國國家隊教練的沈金康來港掌舵，一手一腳把跌進谷底的黃金寶重新扶起來。「阿寶」立即在1994年廣島亞運會公路賽奪得殿軍，這在當時來說已是港隊歷來最好成績，也啟發了他此後衝擊金牌的不屈鬥志。

為了代表香港創造佳績，黃金寶的投入實在不少，每年也未必可以跟家人一起過農曆新年，丁父憂還一度隱瞞沈教練，強忍傷痛為京奧備戰。

其職業生涯中，最著名的是其曾分別3次獲得[[亞運會|亞運會]]公路單車項目金牌。他同時是自1996年以來[[奧運會|奧運會]]上唯一的[[華人|華人]]男性單車選手，他在華人單車手中佔領先位置<ref>〈[http://www.sport.org.cn/mingrentang/shenghuo/2004-01-07/51315.html 黄金宝:行车名将奥运为全中华争光] {{webarchive|url=https://web.archive.org/web/20061205105121/http://www.sport.org.cn/mingrentang/shenghuo/2004-01-07/51315.html |date=2006-12-05 }}〉，新華網，2004年1月7日</ref>。

黃金寶為2000、2001、2004、2005、2006、2007、2008及2010年度[[香港傑出運動員選舉|香港傑出運動員選舉]]「香港傑出運動員」的得獎者，更在2006、2007及2010年度三次獲得得票最高的最高榮譽——香港星中之星傑出運動員。同時，他亦是2002年度香港最佳風範運動員選舉的得獎者<ref>[http://sportstar2008.hkolympic.org/c_voting.php 香港傑出運動員選舉官方網站] {{webarchive|url=https://web.archive.org/web/20090209205725/http://sportstar2008.hkolympic.org/c_voting.php |date=2009-02-09 }}</ref>。

2012年，5次代表過香港出征[[奧林匹克運動會|奧林匹克運動會]]的黃金寶表示會先完成該季賽事後再決定未來動向，初步希望以陪練身份留在單車隊，再逐步轉型為[[教練|教練]]<ref>[http://hk.news.yahoo.com/黃金寶-初步希望逐步轉型做教練-231100770.html 黃金寶：初步希望逐步轉型做教練]</ref>。2012年10月5日，其教練沈金康暗示他最快28日完成環海南島單車公路賽後結束近20年單車職業生涯<ref>[http://news.sina.com.hk/news/3/1/1/2791698/1.html 沈金康令阿寶退役結婚教港隊]{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

然而，黃金寶退役不足半年就決定重出江湖，於2013年6月23日宣佈將會參加於同年8月舉行的[[中华人民共和国第十二届运动会|遼寧全運會]]，以「領導」角色率領一眾師弟衝擊男子個人公路賽金牌，於6月30日亦會出戰假[[新娘潭|新娘潭]]舉行的全港公路單車錦標賽個人公路賽，作為回歸的「第一擊」。<ref>{{cite web|url=http://orientaldaily.on.cc/cnt/sport/20130624/00286_010.html|date=2013年6月24日|title=黃金寶復出再戰全運|publisher=東方日報}}</ref>

== 退役後生活 ==
黃金寶於退役後加入港協暨奧委會的奧夢成真計劃，希望去學校推廣和幫助退役運動員發展他們事業。

== 場外生活 ==
黃金寶為成立於1988年的單車愛好者團體New Power Cycling Club車隊會長，經常帶領隊伍參加各地不同比賽<ref>{{Cite web|url=https://news.mingpao.com/ins/instantnews/web_tc/article/20170523/s00006/1495544852115|title=【亞洲車神】黃金寶辭任港單車隊教練  轉職港協暨奧委會　將告別27年港隊生涯 (21:21) - 20170523 - 體育|accessdate=2017-10-30|work=即時新聞 instant news|language=zh-tw}}</ref>。

== 全運會成績 ==
[[File:2008TourDeTaiwan_Stage1_After_Race_Media_Interview_Kam-po_Wong.jpg|thumb]]：第一站[[高雄|高雄]]站，黃金寶單站奪三冠。]]

黃金寶在[[中华人民共和国全国运动会|全国运动会]]上屢有出色表現，1997年時首次參加[[八運會|八運會]]，其便在男子個人公路單車賽項目奪得金牌，更獲「全運十佳運動員」殊榮。2001年的[[九運會|九運會]]，其再次摘金，稱霸全國。2005年時，黃金寶在[[十運會|十運會]]中面對強敵[[李富玉|李富玉]]下，終奪銅牌。於2009年[[十一運會|十一運會]]，黃金寶再次摘金，並為香港奪得該屆全運會首面金牌。

== 奧運會成績 ==
[[1992年巴塞羅那奧運會|1992年巴塞羅那奧運會]]，時年19歲的黃金寶因為打架，觸動警察，最後遭到[[國際奧林匹克委員會|國際奧林匹克委員會]]處分，喪失參加該屆奧運會的機會。

[[1996年阿特蘭大奧運會|1996年阿特蘭大奧運會]]，黃金寶在亞洲錦標賽奪得第3名，因前5名有進身資格，獲得參賽資格。可是在其他國際職業選手的競爭下，其在個人強項公路個人賽卻未能賽畢全程。

[[2000年悉尼奧運會|2000年悉尼奧運會]]，在公路賽失利的黃金寶卻獲得參與場地賽的資格。他在烏拉圭世界錦標賽B級比賽的場地賽中奪得首名，最終2000年的奧運會的場地賽項目以第11名的成績為香港隊製造紀錄。

[[2004年雅典奧運會|2004年雅典奧運會]]，黃金寶是當時唯一獲得比賽資格的男性華人選手，在此前，自1996年以來他也是奧運會上惟一的華人男性單車選手。

[[2008年北京奧運會|2008年北京奧運會]]，黃金寶於男子單車40公里場地記分賽，總成績排第15。

[[2012年倫敦奧運會|2012年倫敦奧運會]]，黃金寶於男子單車個人公路賽孤軍作戰，在沒有隊友支援下，沿途只能一直在車群末端，末段追隨英國隊於主車群作最後衝刺，以5小時46分37秒成績獲得第37名。首次在奧運完成公路賽，為其最後一次奧運會劃上完美句號。

== 世界賽成績 ==
黃金寶曾在1995年於[[國際自由車環台賽|國際自由車環台賽]]奪冠，並於1996年及1997年的賽事中成功衛冕。2007年，他在[[西班牙|西班牙]][[馬略卡|馬略卡]]舉行的[[世界場地單車錦標賽|世界場地單車錦標賽]]的15公里捕捉賽奪得冠軍，也是其單車手生涯第一個世界錦標賽冠軍以及此項榮譽的首個華人得獎者。世界單車場地錦標賽是[[場地單車賽|場地單車賽]]的最高級別比賽，冠軍得主將獲頒[[彩虹戰衣|彩虹戰衣]]{{flagiconUCI}}，是不少單車運動員夢寐以求的最高榮譽，黃金寶亦因此可在2007年-2008年期間穿上此戰衣<ref>[http://www.takungpao.com/news/07/05/16/WF-736933.htm　黃金寶慶功女友亮相]{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}，《[[大公報|大公報]]》，2007-05-16</ref>。
