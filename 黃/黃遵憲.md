'''黃遵憲'''（{{bd|1848年|4月27日|1905年|3月28日|catIdx=Huang黃遵憲}}<ref name="近代詩三百首">錢仲聯編：《近代詩三百首》，香港：天地圖書，2002年。</ref>），字'''公度'''，別號'''人境廬主人'''，生於[[廣東|廣東]][[梅州市|嘉應州]]（现广东省梅州市梅江区）。[[晚清|晚清]][[詩人|詩人]]，[[外交家|外交家]]、[[政治家|政治家]]、[[教育家|教育家]]。

== 生平 ==
黃遵憲在道光二十八年（1848年）4月27日生於廣東嘉應州（今梅州市[[梅江区|梅江区]]金山街道下市角），四歲讀書，十歲學詩。

光绪二年（1876年）中[[舉人|舉人]]。

光绪三年（1877年）隨[[何如璋|何如璋]]東渡出使[[日本|日本]]，在任期间，积极就[[琉球国|琉球]][[琉球案|问题]]同日本交涉。后又前往[[美国|美国]]、[[英国|英国]]、[[新加坡|新加坡]]等地任外交官。駐外期間，他留心觀察所在國的事物，認為中國要革新自強，必須傚法日本[[明治维新|維新]]。撰寫《日本國志》，全書共四十卷，五十餘萬字，詳細論述日本變革的經過及其得失，借以提出中國改革的主張。

光绪七年（1881年），[[黃遵憲|黃遵憲]]的《朝鲜策略》提议[[李氏朝鲜|李氏朝鲜]]与[[中國|中國]]、[[日本|日本]]、[[美国|美国]]联手对付[[帝俄|帝俄]]，为朝鮮王妃[[閔茲暎|閔茲暎]]集团所赞同，儒生們反對而掀起“[[辛巳斥邪|辛巳斥邪]]”。[[雲峴君|雲峴君]]党羽[[安骥泳|安骥泳]]等图谋政变，拥立雲峴君庶长子[[李载先|李载先]]，后因人告密而失败，安骥泳等人被[[凌遲|凌遲]]，李载先被[[賜死|賜死]]。

光绪八年（1882年）春，調任駐美国[[舊金山|舊金山]]總領事。當時美國借故逮捕大批[[華僑|華僑]]入獄，黃遵憲到監獄探望華僑，叫隨從丈量監獄。美官員無詞以對，只得釋放華僑。1894年回国。

光绪二十一年（1895年），出任湖南[[按察使|按察使]]，在巡抚[[陈宝箴|陈宝箴]]的支持下宣传[[维新变法|维新变法]]。光绪二十四年（1898年）八月，被任命为出使日本大臣。[[戊戌变法|戊戌变法]]失败后，被清政府列为“从严惩办”的维新乱党，但由于外国驻华公使等干预，清政府允许黄遵宪辞职还乡。光绪三十一年（1905年）3月28日，病逝於故乡，終年五十八歲。

== 客家詩宗 ==
[[File:Meizhou_Huang_Zunxian_Jinianguan_2013.10.02_08-55-05.jpg|300px]]]]
黃遵憲是“詩界革命”的主將，是嘉應州的一代詩宗。他曾輯錄了[[客家|客家]]民歌9首，收入《人境廬詩草》之中，大大地提高了客家歌謠的社會地位。[[梁啟超|梁啟超]]說：“近世詩人，能鎔鑄新思想入舊風格者，當推黃公度”；“公度之詩，獨闢境界，卓然自立於二十世紀詩界中，群推為大家”<ref>梁啟超：《飲冰室詩話》</ref>。

他在《人境廬詩草》中描述[[客家山歌|客家山歌]]：「瑤峒月夜，男女隔嶺唱和，興往情來，餘音裊娜，猶存歌仙之遺風，一字千回百折，哀厲而長，稱山歌。」

== 思想 ==
黄遵宪曾于1882年（[[光绪|光绪]]八年）至1885年出任清政府驻[[美国|美国]][[旧金山|旧金山]]总领事，他亲眼目睹了[[1884年美国总统選舉|1884年美国总统選舉]]，但是这并没有启迪他思想革新，反而使他愈加保守。黄遵宪作诗《纪事》<ref name=" an"  />记载了美国总统大选的情况和感触，得出了“[[共和政体|共和政体]]万不能施行于今日之吾国”的结论。<ref name=" an">{{cite web |url=http://news.ifeng.com/history/zhongguojindaishi/detail_2012_01/17/12026872_0.shtml|title=《黄遵宪看美国总统选举：大清国坚决不可学习》|accessdate=2012年1月17日|work=|publisher=凤凰网|language=zh-hans}}</ref>

== 著作 ==
* 《日本雜事詩》二卷<ref name="近代詩三百首"/>
* 1898年，《[[日本國志|日本國志]]》四十卷，1887年成書，光緒二十四年上海图书集成印书局出版、浙江書局重刊。
