《'''JoJo的奇妙冒險 第五部 黃金之風'''》（{{lang-ja|ジョジョの奇妙な冒険 Parte5 黄金の風}}）為日本漫畫家[[荒木飛呂彥|荒木飛呂彥]]所著的[[日本漫畫|日本漫畫]]《[[JoJo的奇妙冒險|JoJo的奇妙冒險]]》的第五部，自1995年起至1999年止在《[[週刊少年Jump|週刊少年Jump]]》連載，收錄於單行本47集至63集之中。由於故事的背景為義大利，而[[義大利文|義大利文]]中沒有「'''J'''」，只是Gio發音等同Jo，因此這一部的JoJo實際上名為「GioGio」。

本作在2001年、2011年分別推出一部外傳小說，並在2002年推出[[PS2|PS2]]遊戲《{{link-jp|JoJo的奇妙冒險 黃金之旋風|ジョジョの奇妙な冒険 黄金の旋風}}》。

除了在初期登場的第三部主角空条承太郎和第四部的廣瀨康一、以及在故事後期登場的第三部的J‧P‧波魯那雷夫之外，本作品和以前的角色沒有太多關聯，故事也較為獨立。

動畫版於2018年10月5日在日本開播，而臺灣則於2018年10月6日開播<ref>[https://twitter.com/anime_jojo/status/1009692466344222721 動畫版官方推特]</ref>。

== 故事大綱 ==
在[[不滅鑽石|上一部]]的故事後兩年，2001年，廣瀨康一接受承太郎的委託，到義大利尋找[[迪奧·布蘭度|DIO]]的子嗣喬魯諾‧喬巴拿，進而帶出正篇故事。喬魯諾‧喬巴拿的夢想是成為「黑道巨星」，以幫助那不勒斯成為更好的地方，特別是讓年輕人遠離毒品的危害。藉由布加拉提的推薦下而加入名為「熱情」之黑手幫的入幫測試，喬魯諾到監獄拜訪幹部波爾波，在通過他的考驗並殺死他後正式入幫，成為布加拉提小隊的一員，小隊的另外四人米斯達、納蘭迦、福葛和阿帕基起初不信任他，但後來漸漸發現他是可靠的人。

在找到波爾波的遺產後，布加拉提被提拔為幹部，他的團隊也被老闆指派了任務──護送老闆的女兒特莉休。小隊帶著特莉休從龐貝城、佛羅倫斯一路前往威尼斯的教堂，同時對抗追殺而來的黑手黨成員。在布加拉提要將特莉休交給老闆時，赫然發現老闆只是為了親手殺死特莉休才這麼做──目的是掩藏自己的身份。布加拉提試圖與老闆一戰，但因為老闆的替身過於強大而撤退，布加拉提隨後背叛組織，除了福葛以外的成員都隨他一同背叛。

他們一路對付追趕而至的組織幹部和老闆派出的直屬護衛隊，為了追查老闆的身分，抵達他的故鄉薩丁尼亞，並在當地接到神秘的信件，寄件者[[星塵鬥士|J‧P‧波魯納雷夫]]聲稱有對抗老闆的方法，並要他們前往羅馬。在羅馬競技場上，布加拉提小隊與老闆展開對決，喬魯諾的替身最終因為箭而進化，並以此擊倒老闆。故事的最後一幕是某個人跪著親吻喬魯諾的手，這表示喬魯諾已成為新任的黑手黨老大。

; {{nihongo|席菈·卡佩茲特|シィラ・カペッツート|Scilla Capezzuto}}
: 15歲的少女，十分景仰喬魯諾。擁有過人的意志力及頭腦，特技是能以敏銳的嗅覺追蹤被替身「狂躁抑鬱」所影響的人類。
: 之所以加入組織是為了向殺死姐姐的兇手復仇，也就是暗殺小隊的伊魯索。在本作中自稱為席菈E（E代表Erinni，希臘神話中的三位復仇女神）。替身攻擊的狀聲詞為「欸哩欸哩欸哩！」
:; {{nihongo|巫毒之子|ヴードゥー・チャイルド|Voodoo Child}}
:: 【破壊力 - B / 速度 - A / 射程距離 - E / 持續力 - E / 精密動作性 - B / 成長性 - B 】
:: 被毆打的地方會出現嘴唇。物體上的嘴唇會說出殘存在此處的人們的心聲，人體上的嘴唇則會痛罵該人深層心理的秘密。
:: 名稱的由來是[[吉米·罕醉克斯|吉米·罕醉克斯]]的歌曲Voodoo Child。

; {{nihongo|康諾羅·穆羅洛|カンノーロ・ムーロロ|Cannolo Murolo}}
: 32歲，穿著復古的情報小隊成員，曾向暗殺小隊洩漏過「老闆」的情報。性格自大且神經質，常對夥伴頤指氣使，也不輕易信賴他人。
: 此行目的除了消滅毒品小隊以外，還有銷毀被毒品小隊藏起來的石鬼面。
:; {{nihongo|永恆的守望塔|オール・アロング・ウォッチタワー|All Along Watchtower}}
:: 【破壊力 - C / 速度 - B / 射程距離 - A / 持續力 - A / 精密動作性 - A / 成長性 - E 】
:: 附身于撲克牌上的替身。組成牌塔後，牌會長出四肢化為人樣，並將本體想知道的事以舞台劇表演出來；不過舞台劇易受本體的精神狀態影響而變得荒誕、難以解讀。五十三張牌（黑桃、紅心、梅花、方塊各13張，加鬼牌一張）都能各自行動，是一種能用於收集情報、暗殺敵人的遠距離操縱型替身。
:: 所有受到的攻擊會分攤成五十三份而大幅減少。
:: 名稱的由來是[[吉米·罕醉克斯|吉米·罕醉克斯]]的歌曲All Along the Watchtower。

==== 毒品小隊 ====
; {{nihongo|伍拉迪米爾·科加其|ヴラディミール・コカキ|Vladimir Kocaqi}}
: 70歲老人，毒品小隊的首領，也是組織幹部之一。
: 表面上和藹可親，實際上老謀深算。
:; {{nihongo|雨天迷夢|レイニーデイ・ドリームアウェイ|Rainy Day, Dream Away}}
:: 【破壊力 - E / 速度 - B / 射程距離 - A / 持續力 - A / 精密動作性 - E / 成長性 - E 】
:: 霧雨狀的替身。能將對手的感覺固定住。
:: 名稱的由來是[[吉米·罕醉克斯|吉米·罕醉克斯]]的歌曲Rainy Day, Dream Away。

; {{nihongo|維特里奧·卡塔爾帝|ビットリオ・カタルディ|Vittorio Cataldi}}
: 全身多處自殘傷痕的16歲少年。
:; {{nihongo|洋娃娃匕首|レイニーデイ・ドリームアウェイ|Dolly Dagger}}
:: 【破壊力 - A / 速度 - A / 射程距離 - C / 持續力 - A / 精密動作性 - B / 成長性 - C 】
:: 附身在匕首上的替身。受傷時可以將七成傷害轉移到倒映在刀刃的物體。本體還是要吞下三成。
:: 受傷的種類不限，不論是槍擊或病毒感染帶來的損傷都能反彈回去。
:: 卡塔爾帝戰鬥時便是以該匕首自殘，將傷害轉移給敵人。
:: 名稱的由來是[[吉米·罕醉克斯|吉米·罕醉克斯]]的歌曲Dolly Dagger。

; {{nihongo|安潔莉卡·阿塔納西歐|アンジェリカ・アッタナシオ|Angelica Attanasio}}
: 重度毒癮的16歲少女。天生就有怪病，只能用毒品來緩解病帶來的痛苦。
:; {{nihongo|夜鳥飛翔|ナイトバード・フライング|Nightbird Flying}}
:: 【破壊力 - E / 速度 - A / 射程距離 - A / 持續力 - A / 精密動作性 - E / 成長性 - E 】
:: 小鳥形狀的自動追蹤型替身。能令對手陷入重度的毒品成癮症狀。
:: 名稱的由來是[[吉米·罕醉克斯|吉米·罕醉克斯]]的歌曲Night Bird Flying。

; {{nihongo|馬希莫·波爾沛|マッシモ・ヴォルペ|Massimo Volpe}}
: 毒品小隊的核心成員，和福葛是舊識。最終與福葛對峙，被進化過後的紫煙破音殺死。
:; {{nihongo|狂躁抑鬱|マニック・デプレッション|Manic Depression}}
:: 【破壊力 - C / 速度 - A / 射程距離 - E / 持續力 - B / 精密動作性 - B / 成長性 - C 】
:: 促進對手的生命力過剩，像嗎啡一樣強化身體，但也有可能令身體機能暴走而自滅。
:: 也可以將能力灌注於鹽，使其變成毒品，可維持約半個月。
:: 名稱的由來是[[吉米·罕醉克斯|吉米·罕醉克斯]]的歌曲Manic Depression。
