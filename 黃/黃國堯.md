'''黃國堯'''（{{bd|1956年|||}})是臺灣[[同光同志長老教會|同光同志長老教會]]牧師，來自香港，2010年至2014年4月間曾在[[基恩之家|基恩之家]]從事神職，同時也在[[am730|am730]]撰文。

== 經歷 ==
2005年，[[香港特別行政區政府|香港特別行政區政府]]本計劃就《[[性傾向歧視條例|性傾向歧視條例]]》立法進行諮詢，引來不同界別和團體反對，包括[[明光社|明光社]]。時為[[基督教宣道會|基督教宣道會]]觀塘堂主任牧師的黃國堯在報章撰文（包括：《明光社前途不光明》<ref>{{ cite web |url=http://www.inmediahk.net/node/40145 |title=明光社前途不光明 |date=2005-06-21 |publisher=明報 |author=黃國堯 }}</ref>、《別把自己當烈士》<ref>{{ cite web |url=http://hkwhite-terror.blogspot.hk/2006/04/blog-post_13.html |title=別把自己當烈士 |date=2005-07-09 |publisher=明報 |author=黃國堯 }}</ref>）批評明光社，引起教會內的爭議。<ref>{{ cite web |url=http://www.truth-light.org.hk/nt/article/%E4%B8%83%E4%B8%80%E9%81%8A%E8%A1%8C%E7%88%AD%E8%AD%B0%E4%BA%8B%E4%BB%B6%E7%B0%BF |title=七一遊行爭議事件簿  |date=2005-07-15 |publisher=明光社 |author= }}</ref> 同時，宣道會觀塘堂執事會亦不滿堂黃國堯之管理及處事手法，黃國堯最終𥾪2005年年底離職。<ref>{{cite web |url=http://eastweek.my-magazine.me/index.php?aid=25569 |title=出櫃記黃國堯 |publisher=東周刊 |date=2013-04-06}}</ref><ref>{{ cite web |url=http://silas-wong-incident.blogspot.hk/2006/05/blog-post_04.html |title=黃國堯牧師解僱信文字版 |date= |publisher= }}</ref>

2006年3月29日，宣道會香港區聯會發要聲明，指觀塘堂執事會於2004年年中起，因不滿堂黃國堯之管理及處事手法，多番溝通後感到與黃國堯在合作上仍存在很多障礙。2005年11月，執事會議決請黃國堯自動請辭，否則在徵得區聯會同意下，解聘黃牧師。區聯會同意觀塘堂執事會之議決，並應執事會之要求，委派總幹事姚添壽牧師協助勸退。及後經觀塘堂一些資深會友相勸，最後黃國堯同意於當月自動請辭。<ref>{{ cite web |url=http://christiantimes.org.hk/Common/Reader/News/ShowNews.jsp?Nid=34248&Pid=12&Version=0&Cid=220&Charset=big5_hkscs |title=宣道會區聯會就黃國堯事件聲明 |date=2006-03-29 |publisher=時代論壇 |author= }}</ref>

2010年1月1日，黃國堯受聘成為香港同志教會「基恩之家」的主任牧師。<ref>{{ cite web |url=http://christiantimes.org.hk/Common/Reader/News/ShowNews.jsp?Nid=57961&Pid=6&Version=0&Cid=150& |title=我如何參與基恩之家的事奉？ |date=2010-02-10 |publisher= |author=黃國堯 }}</ref>

2014年4月，與妻子來到台灣，受聘於[[同光同志長老教會|同光同志長老教會]]擔任駐堂牧師，帶領教會爭取接納同性戀者，並支持同性戀者作為人應有的權利與待遇。希望同志朋友能夠更勇敢，是黃國堯的信念。在他的說服之下，同光同志長老教會的會友們終於放下恐懼，從2015年1月1日起公開了地址。
