'''黃國鐘'''（{{bd|1954年|||}}），[[中華民國|中華民國]][[律師|律師]]及[[政治人物|政治人物]]，早年曾加入[[中華社會民主黨|中華社會民主黨]]，後跟隨[[朱高正|朱高正]]跳到[[新黨|新黨]]。曾代表[[新黨|新黨]]在[[高雄市|高雄市]]南區當選為第三屆[[立法委員|立法委員]]<ref>[http://www.ly.gov.tw/03_leg/0301_main/legIntro.action?lgno=00116&stage=3 立法院全球資訊網 黃國鐘委員簡介]</ref>，後改加入[[中國國民黨|中國國民黨]]<ref>{{Cite web |url =http://news.cts.com.tw/cts/general/199801/199801220012009.html |title =黃國鐘退出新黨 |author = |date =1998/01/22  |publisher =華視新聞 |language = zh-hant |accessdate = 2018年10月25日 |quote = }}</ref>
，目前為全國律師公會聯合會理事。
