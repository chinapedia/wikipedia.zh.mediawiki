'''黃鈺文'''（'''Mickey Huang'''，{{bd|1987年|1月26日|||catIdx=H黃}}），藝名'''米奇'''（'''Mi Chi'''），2005年6月畢業於[[台北市私立華興中學|台北市私立華興中學]]高中部，2009年6月畢業於[[世新大學|世新大學]]新聞系。雖非[[黑澀會美眉|黑澀會美眉]]團體成員之一，但憑著漂亮可人的外貌、活潑的個性及純真的特質，在節目中扮演娛樂觀眾的角色，無論[[RAP|RAP]]或帶動現場氣氛，都有不錯的表現，擁有不少支持者。另外，她有不錯的口才，具當主持人或記者之潛力；亦學習過[[國際標準舞|國際標準舞]]，有於公開場合表演的經驗。夢想是成為[[新聞主播|新聞主播]]。

2012年1月至2014年4月30日在[[年代新聞台|年代新聞台]]擔任氣象主播和記者；同時在[[亞洲廣播網|亞洲廣播網]]也有自己的電台節目。自從[[2008年10月|2008年10月]]《[[我愛黑澀會|我愛黑澀會]]》發生重大人事變動後，她已暫停節目的錄影。曾於《[[台灣醒報|台灣醒報]]》擔任記者撰寫新聞。主持過[[威達超舜電信|威達超舜電信]]生活台節目《輕鬆趴趴走》。

2014年4月30日米奇在臉書宣佈將離開主播台，並規畫離開台灣至澳洲的華人電視台當主播。<ref>{{Cite web |url = https://star.ettoday.net/news/352253 |title = 快訊／「天燈主播」黃鈺文閃電辭職　改戰澳洲華人市場 |author =許逸群 |date = 014-04-3 |publisher = ETtoday新聞雲 |accessdate = 2019-01-05}}</ref>[[澳洲|澳洲]][[墨爾本|墨爾本]]打工生活4年多，擔任侍酒師<ref>{{Cite web |url = https://star.ettoday.net/news/1195685 |title = 《黑澀會》米奇消失螢光幕4年　「變專業侍酒師」近況曝光！ |author = 潘慧中 |date = 2018-06-21 |publisher = ETtoday新聞雲 |accessdate = 2019-01-05}}</ref>，2018年12月19日開始重回新聞工作，擔任[[民視|民視]]VR小組主播、[[民視|民視]]董事室副主任。

[[Popu_Lady#洪詩涵|洪詩]]、[[鄭如吟|玉兔]]與'''米奇'''三位姐妹淘有個小外號：「洪玉米」。
