'''黃念祖'''（1913一1992），[[佛教|佛教]][[居士|居士]]，[[天津大學|天津大學]]、[[北京郵電學院|北京郵電學院]]教授。

== 生平 ==

黃念祖居士，法名蓮華龍尊，又字樂生、心示，別號老念、不退翁。於民國二年農曆癸丑年三月初六日（1913年4月12日）出生<ref name="baike">中国百科网 黄念祖居士（于凌波） [http://www.chinabaike.com/article/baike/fj/rw/2008/200803301303564.html]</ref><ref name="hk">黃念祖的佛教因緣（鄒相） {{cite web |url=http://www.hkbuddhist.org/magazine/599/599_21.html |title=存档副本 |accessdate=2014-04-22 |deadurl=yes |archiveurl=https://web.archive.org/web/20150924030643/http://www.hkbuddhist.org/magazine/599/599_21.html |archivedate=2015-09-24 }}</ref><ref name="xinsheng">黃念祖善知識事跡簡介（《心聲錄》） [http://www.fosss.org/JingTu/HuangNianZu/Other/hnzljssj.html] [http://www.amtb.org.tw/pdf/xinsheng.pdf]</ref>。其母是著名佛教居士[[梅光羲|梅光羲]]的胞妹，篤信佛教。早年父親逝世，由母親撫養成人。幼年在母親及舅舅影響下，常聽經聞法，參謁耆宿大德<ref name="baike"/><ref name="hk"/>。

民國二十年（1931年），黃念祖考入北京大學工學院。在校期間成績優異，長於排球、滑冰等运动，課餘研讀佛經。在第三年暑假某夜讀《金剛經》，至「應無所住而生其心」句，受啟發，以為凡夫心欲達此境，唯有念佛或持咒<ref name="baike"/><ref name="hk"/><ref name="xinsheng"/>。民國二十四年（1935年），自北京大學畢業，服務於[[開灤|開灤]]煤礦<ref name="hk" /><ref name="xinsheng"/>。並於此間，與蕭農華女士（蕭方駿居士之女，亦信佛教）結婚<ref name="baike" /><ref name="hk" />。

未幾[[七七事變|七七事變]]，[[中國抗日戰爭|抗戰]]開始，黃念祖攜妻轉赴後方工作，僑寓重慶。此期間，他曾皈依禪宗大德[[釋虛雲|虛雲]]老和尚，又皈依西藏紅教[[諾那呼圖克圖|諾那]]上師的嫡傳弟子——蓮花正覺[[王家齊|王家齊]]上師、白教大德[[第九世貢噶呼圖克圖|貢噶呼圖克圖]]學習[[密宗|密宗]]<ref name="baike" /><ref name="hk" /><ref name="xinsheng"/>。於1959年受王上師所傳蓮華精舍金剛[[阿闍黎|阿闍黎]]位<ref name="baike" /><ref name="hk" /><ref name="xinsheng"/>。

1945年，抗戰勝利。翌年，黃念祖由後方回到北京，任北京廣播電台台長。是時，佛教大德[[夏蓮居|夏蓮居]]老居士在北京宣揚淨土法門。黃念祖以舅父梅光羲居士函介，成為夏老入室弟子，自此隨侍老居士二十年<ref name="baike" /><ref name="hk" />。夏老有感於《[[無量壽經|無量壽經]]》譯本五種相差甚大，而之前會集本，有率自增文之弊﹔乃閉關三年重予會集，題名為《[[S:佛說大乘無量壽莊嚴清淨平等覺經|佛說大乘無量壽莊嚴清淨平等覺經]]》（以下稱《大經》）。黃念祖聞夏老講解此經，詳作筆記，六十年代初撰寫《大經玄義提綱》一冊，呈大師鑒核認可。夏老囑托弘揚大經，命可隨意發揮講解。<ref name="xinsheng" />

1953年（四十歲時），於[[天津大學|天津大學]]（前身為[[北洋大學|北洋大學]]）任教授。同時修法讀經。忽一日觸機成偈。呈夏大師鑒覽，肯定真開悟，稱其為「唯一心許弟子」<ref name="hk" /><ref name="xinsheng"/>。呈王上師評鑒，亦確定為開悟無疑。<ref name="hk" />

[[文化大革命|文化大革命]]動亂中，黃念祖屢遭絕境，均以安定持誦而安然渡過（又有一次遭遇龍卷風，所在室內物品被一掃而光，而黃念祖卻依然肅立自若）<ref name="hk"/><ref name="xinsheng"/><ref name="safety">臨危不懼，念佛不止（《抉擇見》） [http://www.fosss.org/JingTu/HuangNianZu/Other/lwbjnfbz.html] [http://book.bfnn.org/books3/2077.htm#a04]</ref><ref name="tornado">龍卷風中現奇相 {{cite web |url=http://www.buddhism.com.cn/jingtu/wnz/sj/002.htm |title=存档副本 |accessdate=2014-04-16 |deadurl=yes |archiveurl=https://web.archive.org/web/20150716003321/http://www.buddhism.com.cn/jingtu/wnz/sj/002.htm |archivedate=2015-07-16 }} [http://www.fosss.org/JingTu/HuangNianZu/Other/ljf.html]</ref>。正如夏大師懸記：-{zh-hans:“; zh-hant:「;}-唯艱難困苦備嘗之矣，方可成就。-{zh-hans:”; zh-hant:」;}-<ref name="baike"/>

黃念祖為弘揚淨土，自1979年經二年，閉門謝客，注解《大經》。於1981年完成初稿﹔1982年成第二稿﹔又不顧疾病纏身，终在1984年完成第三稿。於1987年，《佛說大乘無量壽莊嚴清淨平等覺經解》（以下稱《大經解》）流通於海內外。同年，黃念祖至美國華府，並將《大經解》贈與[[釋淨空|淨空法師]]<ref name="preface">《佛說大乘無量壽庄嚴清淨平等覺經解》序文（淨空法師）[http://book.bfnn.org/books/0780.htm#a02]</ref>。同時，在中國佛學院、居士林及廣化寺多次弘法或開設淨宗講座，其著作除《大經解》外，尚有《淨土資糧》、《谷響集》、《華嚴念佛三昧論講記》、《心聲錄》等。尚未完成的有《佛說大乘無量壽庄嚴清淨平等覺經白話解》等<ref name="baike" /><ref name="hk"/><ref name="xinsheng"/>。

1992年3月27日凌晨，黃念祖病逝往生<ref name="baike"/><ref name="hk"/><ref name="xinsheng"/>。4月7日[[火葬|荼毗]]、遺骨潔白、獲五色舍利百餘粒<ref name="baike" /><ref name="xinsheng" /><ref>勇猛念佛——戰勝病魔，臨終現瑞相（《抉擇見》）[http://www.fosss.org/JingTu/HuangNianZu/Other/ymnf.html] {{cite web |url=http://www.buddhism.com.cn/jingtu/wnz/sj/003.htm |title=存档副本 |accessdate=2014-04-16 |deadurl=yes |archiveurl=https://web.archive.org/web/20150716003326/http://www.buddhism.com.cn/jingtu/wnz/sj/003.htm |archivedate=2015-07-16 }} [http://book.bfnn.org/books3/2077.htm#a05]</ref>。
