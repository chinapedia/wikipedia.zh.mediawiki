'''蛾'''是[[鱗翅目|鱗翅目]]现生类群排除了[[蝴蝶|蝴蝶]]所形成的并系群中的昆蟲的總稱，静止时多數將翅膀攤開在身體兩側，且主要於夜间活动，翅膀上具有可以嚇阻敵人的似眼花紋。

== 生活史 ==
蛾屬於完全[[變態|變態]]昆蟲，第一步是从[[卵|卵]]孵化成幼虫或[[毛虫|毛虫]]，开始進食；幼虫經過多次脫皮，发育到一定程度就变成[[蛹|蛹]]，并做成[[茧|茧]]把自己缠绕起来，蛹在茧里发育成飞蛾。大部分的寿命不超过一年，它们在死亡之前进行交配。雌性飞蛾通常把卵产在寄主植物的叶子上。毛虫孵化之后，这片叶子就成了它们的第一顿食物。

== 食物 ==
幼蟲長有咀嚼式[[口器|口器]]，以植物的[[葉子|葉子]]為食物。成蟲無法咀嚼食物，而用类似吸管的长型口器吮吸樹汁、花蜜等。

== 經濟重要性 ==
蛾類的[[幼蟲|幼蟲]]在世界許多地方都被認為是農業上的[[害蟲|害蟲]]。[[舞毒蛾|舞毒蛾]]（''Lymantria dispar''）在美國東北部造成嚴重的森林損害，在當地屬於[[外來種|外來種]]。[[蘋果蠹蛾|蘋果蠹蛾]]（''Cydia pomonella''）在適當的溫度下會造成大規模的損害，特別在果園。在熱帶和亞熱帶地區，[[小菜蛾|小菜蛾]]（''Plutella xylostella''）或許是[[十字花科|十字花科]]蔬菜最嚴重的害蟲。

一些蛾類被當作經濟動物飼養，如[[家蠶|家蠶]]（''Bombyx mori''）。並非所有的[[絲|絲]]都由家蠶所製造，數個[[天蠶蛾科|天蠶蛾科]]的種類也被飼養來產絲，如[[眉紋天蠶蛾|眉紋天蠶蛾]]（''Samia cynthia''）、[[姬透目天蠶蛾|姬透目天蠶蛾]]（柞蠶，''Antheraea pernyi''）、鉤翅大蠶蛾（琥珀蠶，''Antheraea assamensis''）和大透目天蠶蛾（日本天蠶，''Antheraea yamamai''）等。人们用蚕丝纺织成精美而昂贵的[[纺织品|纺织品]]。

Mopane幼蟲（''Gonimbrasia belina''，天蠶蛾科屬的蛾類幼蟲）在非洲南部被作為重要的蛋白質食物來源之一。

== 趨光性 ==
{{see also|趋光性}}
[[File:Positive_phototaxic_of_moths.ogv|thumb]]
蛾類從旁觀者看來，好像顯然地會被[[光|光]]所吸引；但事實上更準確的說法是“因為飛蛾頭暈搞不清楚方向”而圍繞著明亮的物體盤旋。蛾類利用光線來做為羅盤導航，且進化成會利用眼睛裡固定的部分來接受光線。只要光源十分遙遠像是太陽或月亮，飛蛾眼裡所接收到的光的角度會近乎一致地平行，這時飛蛾只要大約朝直線方向行進，視覺成像就能維持不變。但當光源非常接近時，如果飛蛾依舊直線地行進，在移動的每一瞬間其所接收到的光的角度都在改變。所以飛蛾為了要去適應這種改變，就會變成從旁觀者看來好像是螺旋狀地朝光源前進。

== 文学形象 ==

因为蛾类的趋光性，经常将灯光及火光误以为月光并向其中飞去，故此[[中国|中国]][[歇后语|歇后语]]当中有“飞蛾扑火——自取灭亡”之说。
