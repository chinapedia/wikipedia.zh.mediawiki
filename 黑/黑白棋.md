'''黑白棋'''，又叫'''翻轉棋'''（{{lang|en|Reversi}}）、'''蘋果棋'''或'''奧賽羅棋'''（{{lang|en|Othello}}）。據日本棋類遊戲專家[[長谷川五郎|長谷川五郎]]在2005年的統計數據，在日本，黑白棋愛好者的數目約為六千萬人（[[日本將棋|日本將棋]]愛好者約一千五百萬人；[[圍棋|圍棋]]之愛好者約五百萬人；[[國際象棋|國際象棋]]愛好者約三百萬人）<ref>[http://dzb.sg.com.cn/fashionweekly/xss/415811.shtml 時尚週刊《風尚志》雜誌2008年第62期]{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

== 命名 ==
一般[[棋子|棋子]]雙面為[[黑|黑]][[白|白]]兩色，故稱「黑白棋」。因為行棋之時將對方棋子翻轉，變為己方棋子，故又稱「翻轉棋」（Reversi）。棋子雙面為[[紅|紅]]、[[綠|綠]]色的稱為「蘋果棋」。

== 歷史 ==
黑白棋是19世紀末[[英國人|英國人]]發明的。直到上個世紀70年代[[日本人|日本人]][[長谷川五郎|長谷川五郎]]將其發展，借用[[莎士比亞|莎士比亞]]名劇[[奧賽羅|奧賽羅]]（Othello）為這個遊戲重新命名。奧賽羅是一個[[黑人|黑人]]，妻子是[[白人|白人]]，因受小人挑撥，懷疑妻子不忠一直情海翻波，最終親手把妻子殺死。後來真相大白，奧賽羅懊悔不已，自殺而死。黑白棋就是借用這個黑人白人鬥爭的故事。<ref>http://www.beppi.it/public/OthelloMuseum/pages/history.php</ref>

類似的傳統棋類有[[西藏夾棋|西藏夾棋]]，是夾住敵棋後，換成己方的棋子。

=== 黑白棋的電子化 ===
黑白棋出現在80年代的[[任天堂|任天堂]]遊戲機和[[Apple_II|Apple II]]個人電腦遊戲裡然後就是1990年的[[Windows_3.0|Windows 3.0]]的推出，當時[[Windows|Windows]]自帶的遊戲就是黑白棋，由於當時[[電腦|電腦]]還比較少，沒等到黑白棋傳開，[[Windows_3.1|Windows 3.1]]推出，把自帶遊戲換成現在大家見到的[[踩地雷|踩地雷]]和[[接龍_(紙牌遊戲)|接龍]]。在90年代中期流行的任天堂[[Game_Boy|Game Boy]]當中也有一款黑白棋遊戲。另外，[[電子辭典|電子辭典]]中的附帶遊戲也大多包含黑白棋。

=== 網上黑白棋的興起 ===
在90年代中期互聯網開始普及，亦冒起了一些大型遊戲網站。最先出現的是[[微軟|微軟]]的Microsoft Games網站，被吸引進去下黑白棋的都是各國的好手，因為是外國網站，當時下棋的華人還是比較少。及後紛紛出現其他網站，具代表性的有：

*以台灣棋手為主：[[宏碁戲谷|宏碁戲谷]]、[[CYC遊戲大聯盟|CYC遊戲大聯盟]]
*多國棋手網站：[[Kurnik|Kurnik]]（現為PlayOK）、[[Playsite|Playsite]]、[[VOG|VOG]]、[[Yahoo!_Games|Yahoo! Games]]、[[新浪網|新浪網]]
*以中國棋手為主：[[Chinaren|Chinaren]]、[[聯眾世界|聯眾世界]]、[[中國遊戲中心|中國遊戲中心]]、[[邊峰網路遊戲|邊峰網路遊戲]]
*以香港棋手為主：[http://www.box.net/shared/a0roi7l3dp GGS (MIMOSA)]、[[Cybercity|Cybercity]]

值得一提的是微軟的[[Windows_XP|Windows XP]]裡也自帶了網路黑白棋。

== 棋具 ==
* [[棋子|棋子]]：黑白棋棋子每顆由[[黑|黑]][[白|白]]兩色組成，一面白，一面黑，共64個（包括棋盤中央的4個）。棋子呈圓餅形。
* [[棋盤|棋盤]]：黑白棋棋盤由64格的正方格組成，遊戲進行時棋子要下在格內。棋盤可分為「[[角_(黑白棋)|角]]」、「[[邊_(黑白棋)|邊]]」以及「中腹」。現今的棋盤多以8x8較為普遍。
* [[棋鐘|棋鐘]]：正式的[[比賽|比賽]]中可以使用棋鐘對選手的時間進行限制。非正式的對局中一般不使用棋鐘。

== 遊戲規則 ==
[[棋盤|棋盤]]共有8行8列共64格。開局時，[[棋盤|棋盤]]正中央的4格先置放[[黑|黑]][[白|白]]相隔的4枚棋子（亦有求變化相鄰放置）。通常[[黑|黑]]子先行。雙方輪流落子。-{只}-要落子和[[棋盤|棋盤]]上任一枚己方的[[棋子|棋子]]在一條線上（橫、直、斜線皆可）夾著對方[[棋子|棋子]]，就能將對方的這些[[棋子|棋子]]轉變為我己方（翻面即可）。如果在任一位置落子都不能夾住對手的任一顆[[棋子|棋子]]，就要讓對手下子。當雙方皆不能下子時，遊戲就結束，子多的一方勝。

== 游戏策略 ==
因为黑白棋独特的规则，很容易出现双方比分的剧烈变化，在游戏后期可能仅用几个回合就将大量对方棋子变成己方，从而扭转局势。因此，太着眼于比分是没有必要的，更重要的是占据有利位置。

中间位置的棋子最容易受到夹击，有橫、直、斜線共四个方向的可能。而边缘的棋子则只有一个可能被夹击的方向，四个角落上的位置被占据后，则完全不可能被攻击。

游戏的后期是关键位置的争夺，而前期的布局，就是为抢占关键位置作准备。比如，若不想让对方占据棋盘边缘的有利位置，那么自己就应避免在靠近边缘的那一排落子。

== 黑白棋術語 ==
[[C位_(黑白棋)|C位]]、[[星位_(黑白棋)|星位]]（C-squares and X-squares）：C位就是位於a2、a7、b1、b8、g1、g8、h2和h7的位置，星位就是位於b2、b7、g2和g7的位置。這些位置務必要小心佔用。

[[中心_(黑白棋)|中心（Center）]]：局面的中心就是[[內部子_(黑白棋)|內部子]]的集合。

[[控制中心_(黑白棋)|控制中心（Control of the center）]]：一種策略，它試圖在局面中心擁有儘可能多的棋子，沿[[邊界_(黑白棋)|邊界]]擁有儘可能少的棋子，以獲得最大可能的[[行動力_(黑白棋)|行動力]]。

[[角_(黑白棋)|角（Corner）]]：[[角_(黑白棋)|角]]就是位於a1、a8、h1和h8的位置。这些位置不可能被对方夹吃。

[[爬邊_(黑白棋)|爬邊（Edge creeping）]]：一種以[[不平衡邊_(黑白棋)|弱邊]]([[不平衡邊_(黑白棋)|不平衡邊]]……)為代價，在一條或兩條[[邊_(黑白棋)|邊]]上獲得最大數量棋步的策略。[[爬邊_(黑白棋)|爬邊]]者試圖通過將全部邊界都留給對手來快速耗盡他的棋步，但是如果爬邊不能湊效，壞邊產生的效應將使他的局面迅速變弱。

[[邊界_(黑白棋)|邊界（Frontier）]]：邊緣子的集合，也就是說那些與空位相鄰的棋子。

[[獲得餘裕手_(黑白棋)|獲得餘裕手（Gain a tempo）]]：在棋盤的某個區域內比對手多下一步棋，以迫使他在其他地方先開始下棋（從而延長他的邊界）。

[[效應_(黑白棋)|效應（Influence）]]：當棋手的棋子迫使他同時在多個方向上翻轉棋子時，我們就說這些棋子產生了效應。

[[內部子_(黑白棋)|內部子（Internal discs）]]、[[邊緣子_(黑白棋)|邊緣子（external discs）]]：內部子就是不與空位相鄰的棋子。沒有內部子在戰略上是很糟的。

[[自由_(黑白棋)|自由（Liberty）]]：非災難性的棋步。「缺少自由」：在不久的將來不得不送角。

[[多子策略_(黑白棋)|多子策略（Maximum disc strategy）]]：許多初學者所用的錯誤策略，他們每步棋都試圖翻轉最大數量的棋子。

[[行動力_(黑白棋)|行動力（Mobility）]]：棋手合法的可能棋步數量。更進一步說，當棋手擁有大量的可能棋步時，他就擁有好的行動力。

[[奇偶性_(黑白棋)|奇偶性（Parity）]]：一種在對手佔據的每個區域內都留下偶數個空位的策略。

[[安靜步_(黑白棋)|安靜步（Quiet move）]]：不翻轉邊緣子的一步棋，通常這是步好棋。

[[穩定子_(黑白棋)|穩定子（Stable discs）]]：絕對不會被翻轉的棋子。角就是一個[[穩定子_(黑白棋)|穩定子]]的實例。

[[斯通納陷阱_(黑白棋)|斯通納陷阱/四通陷阱（Stoner Trap）]]：一種針對弱邊局面強迫進行角交換的攻擊。

[[不平衡邊_(黑白棋)|不平衡邊（Unbalanced edge）]]：由非6個同色棋子組成的邊結構。

== 黑白棋軟體與人工智慧 ==

=== 黑白棋程式簡史 ===
在1980年代，[[電腦|電腦]]並不普及，在黑白棋界裡，最強的仍然是棋手（人類）。

到了1990年代初，電腦的速度以[[幾何級數|幾何級數]]增長，寫出來的黑白棋[[程式|程式]]雖然仍然有點笨拙，但由於計算深度（電腦的速度快）和尾局的準確性，所以已經很強。1990年代初最有名的程式就是[[Thor|Thor]]，在那時候是最強的程式（還是[[DOS|DOS]]模式年代），棋力能比得上世界級棋手，在這個時期的程式都是人工地加入[[行動力_(黑白棋)|行動力]]、位置策略、[[奇偶性_(黑白棋)|偶數重要性]]等等，但又因為這些策略是直接編寫在程式裡，那程式的棋力很依賴[[程式員|程式員]]本人的棋力，程式下起來比較像人類的下法。而且程式裡遺留了人類棋手的弱點、策略的不完整性等等的問題。因為早期Thor的通用性，影響到現在黑白棋界裡的統一[[棋譜|棋譜]][[資料庫|資料庫]]都是用[[Thor_database|Thor database]]。

到了1994、1995年，黑白棋程式的編寫方法有了突破性的發展，首先是有了一個稱為IOS的網站（[[GGS|GGS]]的前身），讓不同的程式同時連上去互相對局，突破了以往那種閉門造車的日子，在同期[[Michael_Buro|Michael Buro]]做出了能由程式自我學習的[[Logistello|Logistello]]，在差不多的時間，大家都學起來，用相似的方法來編寫程式。程式員不再把人工的策略和下棋方法等死硬地寫在程式裡，而是由程式自我學習，程式會記錄好、壞的形狀（patterns），根據實戰的結果自動調整策略，又會把不同的開局棋譜根據實戰的過程來評分、保存，有些程式能保存幾十萬步的開局棋譜。

就是因為Logistello在根本方法的改進、先進的算法、編寫方面的高效率和準確性等等，一直在黑白棋界保持為世界程式冠軍。在1993年~1997年的25場比賽中，Logistello有18場獲得冠軍、6場獲得亞軍。1997年8月，Logistello擊敗了1996年的世界冠軍[[村上健|村上健]]（Takeshi Murakami），從此黑白棋程式把人類棋手遠遠甩在後面。直到Michael在1998年1月宣布Logistello退休為止。現在已經有數個程式在棋力上超越已退休的Logistello，包括[[edax|edax]], [[cyrano|cyrano]], [[saio|saio]]，主要突破點在於它們把算法改善為[[多執行緒_(電腦硬體)|多線程執行（multithreading）]]。

=== WZebra ===
黑白棋程序「斑馬」（[[WZebra|WZebra]]）是[[Zebra|Zebra]]的[[Windows|Windows]]版本，除了可以下棋外，還提供了打譜、複盤、棋局分析、自我學習等功能，也可以加載[[Thor|Thor]]棋譜文件，進行針對性訓練。在標準比賽時間（2x15分鐘）內，其搜索深度可達中局18至27步、終局24至31步。[[WZebra|WZebra]]是[[自由軟體|自由軟體]]，提供了中文版本，並提供中文幫助。

=== S.A.I.O. ===
[[Saio|Saio]]是System Artificial Intelligence Othello縮寫，是黑白棋[[程式|程式]]之一。並不開放下載。

== 網路黑白棋 ==
自[[互聯網|互聯網]]興起，網路下棋成為了大眾黑白棋對弈的主要形式。網路黑白棋比賽也日漸普及起來。網路黑白棋尋找棋友非常方便，對弈後資料保留非常妥當。
