'''黑頭文鳥'''（[[學名|學名]]：''{{lang|la|Lonchura atricapilla}}''），又稱'''栗腹文鳥'''，一種居住於[[平原地帶|平原地帶]]的雀形目鳥類。主要分布在[[中國|中國]]南部、[[台灣|台灣]]以及[[南亞|南亞]]、[[東南亞|東南亞]]的[[熱帶|熱帶]]地區。該種本來學名為''Lonchura malacca''，近年與白腹型的亞種分類為兩個不同的物種。白腹型的亞種學名仍為''Lonchura malacca''，此物種列學名''Lonchura atricapilla''。

== 特徵 ==
身長約11至12公分；頭部為黑色，頭部以下皆為橘褐色。

== 習性 ==
以[[禾本科|禾本科]]種子為食，常密集群體飛行。

== 棲息地 ==
通常成群出現在平地到低海拔的[[草原|草叢]]、[[耕地|農耕]]地區。有時會和[[斑文鳥|斑文鳥]]、[[麻雀|麻雀]]等雀鳥群雜居。

== 附記 ==
在台灣，由於大量引進東南亞亞種，使得台灣原生亞種的黑頭文鳥族群急遽減少。
