《'''MIB星際戰警3'''》（'''{{lang-en|Men in Black 3}}'''）是一部于2012年5月上映的[[美国|美国]][[3D電影|3D]][[科幻片|科幻]][[动作片|动作]][[喜劇片|喜剧片]]，[[巴里·索南菲尔德|巴里·索南菲尔德]]執導。繼續由[[汤米·李·琼斯|汤米·李·琼斯]]和[[威尔·史密斯|威尔·史密斯]]主演。劇情主要以1969年為背景，[[J探员|J探员]]聯合[[K探员|K探员]]獵殺野兽波里斯，以阻止世界末日到來。<ref>http://movie.now.com/home/movie?id=meninblack3</ref>

== 剧情 ==
讲述了反派角色妖獸波里斯 (Boris the animal)（[[杰梅奈·克莱门特|杰梅奈·克莱门特]]饰演）想要回到过去杀死K探员，随之发生了一系列难以想象的事件。[[威尔·史密斯|威尔·史密斯]]扮演的J探员也回到了1969年，与年轻时代的K执行任务，共同阻止妖獸波里斯毁灭这个世界。

== 制作 ==
由[[威爾·史密斯|威爾·史密斯]]在拍攝[[MIB星際戰警2|MIB星際戰警2]]電影的前提下，首次提出在2002年，在那裡史密斯向[[巴里·索南菲爾德|巴里·索南菲爾德]]提出一個想法，就在電影上的第三部電影導演Barry Sonnenfeld，這個想法涉及史密斯的性格，特務J，時光倒流，以挽救他的合作夥伴特工K作為以及探索由[[湯米·李·瓊斯|湯米·李·瓊斯]]扮演的特工K的背景故事。據索南菲爾德的想法了很長一段時間才能真正進入製作中的情節有時間旅遊的問題。

首次公佈該片由索尼電影娛樂公司總裁羅裡·布魯爾在4月1日，2009年，索尼的ShoWest簡報。 2009年10月，伊坦·科恩曾聘請來寫劇本。截至2010年3月，威爾·史密斯仍猶豫不決是否加入這部電影或其他航行，城市。導演 Barry Sonnenfeld 證實在2010年5月由[[湯米·李·瓊斯|湯米·李·瓊斯]]和史密斯扮演主角的回報。兩人都表示在2008年再發生他們的角色. 其他人員包括樓沃爾特·帕克斯和勞裡·麥克唐納作為生產者，作為執行製片人與[[史蒂文·斯皮爾伯格|史蒂文·斯皮爾伯格]]都是前兩個電影生產者的利益。

== 花絮 ==
电影宣传的时候，说明该片中有许多国际巨星出演，不过播出之后，才知道[[姚明|姚明]]、[[大卫·贝克汉姆|大卫·贝克汉姆]]、[[女神卡卡|女神卡卡]]等国际巨星都只有几秒钟的镜头，令观众有点失望。<ref name="fish " >{{cite web |url= http://yule.sohu.com/20120526/n344132567.shtml?pvid=tc_ent&a=&b=%E3%80%8A%E9%BB%91%E8%A1%A3%E4%BA%BA3%E3%80%8B%E6%98%A8%E4%B8%8A%E6%98%A0%20%E7%BD%91%E5%8F%8B%E6%8A%B1%E6%80%A8%E6%B2%A1%E7%9C%8B%E5%88%B0%E5%A7%9A%E6%98%8E|title= 《黑衣人3》上映 网友抱怨没看到姚明|accessdate= 2012-05-26|publisher= 搜狐网|work= |language =zh-cn}}</ref>
中國公映时因部分镜头“损害中国人形象”未能被广电总局审查通过，故有3分钟的镜头被删减。删减内容基本是华人老板的戏份，中國公映版中看不出此老板是外星人。

最後一幕是K在餐廳忘記給[[小費|小費]]，電影想提醒觀眾在美國給小費的重要性。

=== 穿幫鏡頭 ===
*當鮑瑞斯將牆壁打爆而將所有的監獄護衛隊吸進太空時，原本站在二樓的護衛隊卻消失。
*J在第一次利用時空回轉跳下大樓時需要護目鏡，但在最後回去時根本不用（可以考慮到不需要了）。
*當鮑瑞斯把J靠在火箭發射平台上的欄杆上時，J對他說完話後，在下一個鏡頭裏可以發現鮑瑞斯抓住J的方向是在另一邊。
*當J對付鮑瑞斯時利用時空回轉倒回9：30分，但影片卻重回J利用時空回轉之前發生的片段（J對鮑瑞斯說“你輸定了”說了兩次）。
