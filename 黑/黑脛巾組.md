'''黑脛巾組'''（くろはばきぐみ），[[陸奧國|陸奧]]大名[[伊達政宗|伊達政宗]]所籌組的[[忍者|忍者]]組織，曾有許多活躍紀錄，但是江戶時代初期並沒有相關情報，多數記載是突然出現在江戶中期有關伊達家的軍記或文獻裡，因此也被懷疑是個虛構組織，主要活動區域在現今的[[宮城縣|宮城縣]]與[[福島縣|福島縣]]一帶。
== 特徵 ==
黑脛巾組<ref>[http://www.syuriken.com/ninja_content/ninja-troops.htm]，忍者軍團，忍者分佈</ref>得名原由是其成員必須適用黑革製的脛當做為標章，才被取名為黑脛巾組。
== 成員組織 ==
根據《伊達秘鑑》的紀錄，黑脛巾組是'''[[伊達政宗|伊達政宗]]'''命令信夫郡得鳥屋城主'''安部重定'''籌組，成員是由鄰近的出羽三山的山伏（山岳信仰者，於山中進行修煉與苦行的修練者）挑選招募，初步招集了５０名忍者，隨時日演進組織日益擴增，《老人伝聞記》記載黑脛巾組後期按照[[仙台藩|仙台藩]]的地利位置分設南、北、佐沼、石卷、本吉南方、氣仙及本吉北方六個組，主要的任務是遊走全國從事諜報任務、製造流言、敵後騷擾以及新占領地的治安警戒<ref>[http://www5f.biglobe.ne.jp/~khaos/habaki.htm]，伊達の隠密・黒脛巾(くろはばき)組(一)</ref>。
== 主要事蹟 ==
黑脛巾組為[[伊達政宗|伊達政宗]]貢獻良多，在天正13年（1585年）11月的[[人取橋之戰|人取橋之戰]]中在伊達軍被[[佐竹義重|佐竹義重]]和[[蘆名盛重|蘆名盛重]]聯合岩城常隆、石川昭光、二階堂盛行、白河義親的聯軍攻擊趨於不利時，黑脛巾組忍者[[大林坊俊海|大林坊俊海]]在佐竹軍散播後有敵人偷襲[[常陸國|常陸國]]，引誘佐竹軍撤離，替伊達軍謀得生路。後來於[[摺上原之戰|摺上原之戰]]中，大林坊俊海探知蘆名氏的備蓄情況，並向伊達政宗進言戰場氣候可能的變化，太宰金助也傳達了猪苗代家的父子相爭，讓政宗順利策反[[猪苗代盛國|猪苗代盛國]]。

根據[[新井白石|新井白石]]所寫的藩翰譜記載，天正18年（1590年）[[豐臣秀吉|豐臣秀吉]]發起[[小田原征伐|小田原征伐]]時[[太宰金助|太宰金助]]受伊達政宗派遣，偷入小田原城探查情報，據氣仙沼市史Ⅱ也記載後來伊達政宗移封至葛西、大崎領地時，正式將黑脛巾組按新領地劃分成六個組，整頓當地治安。
