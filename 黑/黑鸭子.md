'''黑鸭子合唱组'''是[[中国|中国]]歌坛的第一个[[和音|和音]]组合，发行过近50张唱片，<ref>{{Cite web |url=http://ent.tom.com/2008-03-08/000E/05039722.html |title=“黑鸭子”演唱组放声武汉剧院 称永远不解散_TOM娱乐 |access-date=2009-06-21 |archive-url=https://web.archive.org/web/20081108215414/http://ent.tom.com/2008-03-08/000E/05039722.html |archive-date=2008-11-08 |dead-url=yes }}</ref>并参与过多次大型演出。于1983年由[[徐秀霞|徐秀霞]]、[[陆莉莉|陆莉莉]]、[[尹淑占|尹淑占]]三人组成。当时三人由[[中央音乐学院|中央音乐学院]][[小提琴|小提琴]]专业毕业、分配到[[中国广播乐团|中国广播乐团]]工作。

1992年[[李蓉|李蓉]]代替移居[[新加坡|新加坡]]的尹淑占，担任一声部。1998年[[郭祁|郭祁]]代替移居[[加拿大|加拿大]]的陆莉莉，担任二声部。2002年底，黑鸭子分解成了徐氏演唱组合和李氏演唱组合。<ref>[http://ent.sina.com.cn/2004-11-23/2240577003.html 两只“黑鸭子”吵起来(组图)_影音娱乐_新浪网]</ref>

2002年12月5日，徐秀霞向国家工商行政管理总局商标局提出注册申请，注册“黑鸭子”商标，指定使用在第41类组织表演、演出等服务上。经过一些列异议、诉讼等过程，2012年5月4日，北京市高级人民法院作出终审判决，裁定该商标不予注册。<ref>[http://www.cnipr.net/article_show.asp?article_id=19341 中国广播艺术团与国家工商行政管理总局商标评审委员会第三人李蓉第3393307号“黑鸭子”商标异议复审行政纠纷案]</ref>

== 徐氏演唱组合 ==
徐氏黑鸭子演唱组合包括徐秀霞、2002年加入的[[樊桐舟|樊桐舟]]和[[杨漫|杨漫]]，2006年[[倪雅丰|倪雅丰]]代替杨漫。

== 李氏演唱组合 ==
李氏黑鸭子演唱组合包括李蓉、郭祁和[[李伟|李伟]]。

== 新“李氏”演唱组合 ==
2014年底，名为“黑鸭子演唱组”的实名新浪微博中，成员不再有李蓉。2015年2月，原“徐鸭子”的成员倪雅丰的名字出现在该微博发布的成员名单中，后者的成员变为郭祁、李伟和倪雅丰。
