== 故事概要 ==
在[[橫濱市|橫濱市]]區邊緣的黑暗坡頂上是[[江戶時代|江戶時代]]處決死刑犯的刑場。那裡矗立著一棵據說樹齡超過2000年的詭異的大[[楠樹|楠樹]]，樹旁是一棟[[二次大戰|二次大戰]]前興建，屋頂上裝了一支青銅風向雞的西式洋房。

1984年9月時[[颱風|颱風]]侵襲了橫濱市。颱風過境隔日，黑暗坡的西式洋房上卻出現了不可思議的恐怖景象：屋頂上的青銅雞消失了，取而代之的是住在洋房裡的藤並家族長男藤並卓跨坐在屋頂上、神色木然瞪著大楠樹的屍體。

凶案接二連三，受害者陸續出現！黑暗坡的大楠樹真的是傳說中的『食人樹』嗎？屢破怪案的御手洗潔如今向超越千年的古樹挑戰！<ref>[http://akijan.sakura.ne.jp/mi/kaisetu/mitarai/kurayami.html {{lang|ja|島田荘司作品解説 - 暗闇坂の人喰いの木}}]</ref>

== 登場人物 ==
=== 主要角色 ===
; {{nihongo|[[御手洗潔|御手洗 潔]]|{{ruby|御手洗|みたらい}} {{ruby|潔|きよし}}|Mitarai Kiyoshi}}
: 1984年初秋居於[[橫濱|橫濱]][[馬車道|馬車道]]的寂寂無名的[[偵探|偵探]]。因好友石岡的關係被卷入此次事件。

; {{nihongo|[[石岡和己|石岡 和己]]|{{ruby|石岡|いしおか}} {{ruby|和己|かずみ}}|Ishioka Kazumi}}
: 御手洗的同居老友，作家。以森真理子拜訪自己為契機，和御手洗兩人一同被卷入此次事件。

=== 案件相關 ===
; {{nihongo|森 真理子|{{ruby|森|もり}} {{ruby|真理子|まりこ}}|Mori Mariko}}
: 1984年秋突然拜訪石岡和巳的女性。和藤並卓保持了七年婚外情關係，而藤並卓並無和自己在一起的意思，便打算利用石岡君給藤並卓施壓。
: 首次見面給石岡君的印象是「皮膚白皙，圓滾滾的大眼睛，[[雙眼皮|雙眼皮]]，身形嬌小樣貌可愛的女性」。
: 生於[[昭和26年|昭和26年]]（1951年）的[[天蠍座|天蠍座]]。

; {{nihongo|藤並 卓|{{ruby|藤並|ふじなみ}} {{ruby|卓|すぐる}}|Fujinami Suguru}}
: 藤並八千代和詹姆斯·貝因的大兒子，生於[[昭和21年|昭和21年]]（1946年）。郁子的丈夫，和森真理子維持著不倫的婚外情。森真理子形容他是一個頭腦聰明、容貌端正、素有教養的男人。
: 1984年9月，被發現跨坐在藤並家屋頂原本應該是青銅風向雞的位置，死於[[心衰竭|心臟麻痺]]。

; {{nihongo|藤並 郁子|{{ruby|藤並|ふじなみ}} {{ruby|郁子|いくこ}}|Fujinami Ikuko}}
: 藤並卓的妻子。

; {{nihongo|藤並 讓|{{ruby|藤並|ふじなみ}} {{ruby|譲|ゆずる}}|Fujinami Yuzuru}}
: 藤並八千代和詹姆斯·貝因的二兒子，生於[[昭和22年|昭和22年]]（1947年）。研究[[死刑|死刑]]史的學者，性格有點怪異。
: 和同為怪人的御手洗志趣相投，但在御手洗為調查事件遠渡[[英國|英國]]的期間死亡，屍體被發現時全身骨折，以大頭朝下的倒栽蔥姿勢插在大楠樹的樹幹頂上。

; {{nihongo|藤並 八千代|{{ruby|藤並|ふじなみ}} {{ruby|八千代|やちよ}}|Fujinami Yachiyo}}
: 詹姆斯·貝因的前妻，卓、讓、玲王奈的母親，生於[[大正12年|大正12年]]（1923年）。在[[伊勢佐木町|伊勢佐木町]]的[[料亭|料亭]]工作的時候和貝因一見鍾情，後來結婚。[[昭和45年|昭和45年]]（1970年）離婚。在藤並卓死亡的雨夜，被發現倒臥在大楠樹下面，因[[頭蓋骨|頭蓋骨]][[粉碎性骨折|粉碎性骨折]]而命懸一線。然後在入住藤棚綜合醫院期間離開醫院，再次被發現倒在大楠樹下，但此時的她已經死亡。

; {{nihongo|[[松崎玲王奈|松崎 玲王奈]]|{{ruby|松崎|まつざき}} レオナ|Matsuzaki Leona}}
: 本名「藤並玲王奈」，藤並八千代和詹姆斯·貝因的小女兒，藤並卓和藤並讓的妹妹，生於[[昭和38年|昭和38年]]（1963年）。是個頭腦聰明的絕色美女。原本是個[[模特|模特]]，後來成為了一個娛樂明星<ref>在《[[水晶金字塔|水晶金字塔]]》中，成為了[[荷里活|荷里活]]首位日本女星。</ref>。
: 在本故事中首次登場，和御手洗、石岡一起遠赴[[英國|英國]]調查真相。之後在御手洗系列部分其他小說中均有登場。
: 在此次事件中對御手洗產生愛意，並大聲表露出來。

; {{nihongo|藤並 照夫|{{ruby|藤並|ふじなみ}} {{ruby|照夫|てらお}}|Fujinami Terao}}
: 藤並八千代和貝因離婚後的再婚對象，舊姓三本，生於[[昭和7年|昭和7年]]（1932年）。有一個於[[昭和43年|昭和43年]]（1968年）出生，名叫三幸的女兒，為照夫與前妻所生。

; {{nihongo|詹姆斯·貝因|ペイン·ジェイムズ|Payne James}}
: 卓、讓、玲王奈的父親。來自[[蘇格蘭|蘇格蘭]]的一個叫做弗塞斯的村子，二戰之後在日本和藤並八千代結婚，經營一家外國人子女的學校。
: 以前是名[[畫家|畫家]]，其父親因參與軍需物資制造行業而發家，然後在[[昭和20年|昭和20年]]（1945年）隨著[[美軍|美軍]]一起來到日本。認識他的人回憶起他，都說他是個如同繪畫於畫中的[[英國|英國]][[紳士|紳士]]一般，為人守時，生活節制有規律的過分正直的人。和八千代離婚後據說回英國去了。

; {{nihongo|淳子|{{ruby|淳子|じゅんこ}}|Junko}}
: 照夫的妹妹。
: [[昭和16年|昭和16年]]（1941年）在登上黑暗坡後失蹤，之後其屍體被發現掛在大楠樹上。
