'''黑幫博物館'''（{{lang-en|'''Mob Museum'''}}），官方名是'''國家有組織犯罪及執法博物館'''（'''National Museum of Organized Crime and Law Enforcement'''），是位於[[內華達州|內華達州]][[拉斯維加斯|拉斯維加斯]]{{link-en|拉斯維加斯市中心|Downtown Las Vegas|市中心}}的歷史博物館。該博物館於2012年2月14日開幕，致力於展出[[美國|美國]][[黑社會|有組織犯罪]]的文物、故事和歷史，還有執法部門為了預防此類犯罪而採取的行動和倡議。博物館坐落於前{{link-en|拉斯維加斯郵局和法院大樓|Las Vegas Post Office and Courthouse}}，大樓建於1933年，並在[[國家史蹟名錄|國家史蹟名錄]]上登記。博物館位於斯圖爾特道（Stewart Avenue），{{link-en|佛雷蒙特街|Fremont Street}}以北兩個街區，市中心賭場區的主要幹線。

在[[國際間諜博物館|國際間諜博物館]]和[[搖滾名人堂|搖滾名人堂]]的共同創立者Dennis Barrie的創意指導下，博物館是由一個與拉斯維加斯城市合作的非營利性董事會「300斯圖爾特道公司」（300 Stewart Avenue Corporation）管理。博物館在拉斯維加斯和整個美國的歷史背景內獻身於有組織犯罪和執法之間有爭議的關係。

[[File:Las_Vegas_Mob_Museum_2012.jpg|thumb]]

== 歷史 ==
2000年，聯邦政府將前郵政局和聯邦法院大樓以1美元的價錢出售給城市，規定將該建築物修復到原來的外觀，並用於文化目的。<ref name="friess">{{Cite web |url = http://www.nytimes.com/2009/01/10/us/10mob.html |title = Stimulus Money for a Mob Museum. Got a Problem? |author = Steve Friess |date = 2009-01-09 |publisher = The New York Times |language = en}}</ref>

然後，{{link-en|奧斯卡·古德曼|Oscar Goodman}}市長，他自己是一名前黑幫辯護律師，在2002年有一個做黑幫博物館的想法。<ref name="friess" />這個想法在早期面臨來自[[意大利裔美國人|意大利裔美國]]團體的反對，而在[[聯邦調查局|聯邦調查局]]（FBI）的支持下，包括拉斯維加斯的前頭頭探員Ellen Knowlton，加入成為博物館董事會的主席。<ref name="friess" /><ref>{{Cite web |url = https://lasvegassun.com/news/2008/aug/17/oh-irony-former-mob-lawyer-gets-fbi-support-mob-mu/ |title = Oh, the irony: The former mob lawyer gets FBI support for mob museum |author = Sam Skolnik |date = 2008-08-17 |publisher = Las Vegas Sun |language = en}}</ref>

該項目預算估計為5000萬美元，其中包括2600萬美元用於修復該建築物。<ref name="friess" />資金包括聯邦、州和當地撥款。<ref name="friess" />古德曼提議聯邦的[[2009年美國復蘇與再投資法案|刺激資金]]可以用於博物館而引起了爭議。<ref name="friess" />

博物館於2012年2月14日下午2:00開放給公眾。（早上和前一天被保留給新聞報道和高官儀式。）成人入場開始為19.95美元，內華達州居民入場開始為13.95美元。截至2016年3月，成人價錢為23.95美元，長者價錢為17.95美元，而兒童價錢為13.95美元，其他特定範疇、團體和套餐價錢則有所不同。

== 展覽 ==
[[File:Mob_Museum_exhibit_showing_an_early_slot_machine..JPG|thumb]]
黑幫博物館的核心是二樓的審判室，這是在1950年和1951年以揭發有組織犯罪而舉行的14次國家{{link-en|美國參議院州際商務犯罪特別委員會|United States Senate Special Committee to Investigate Crime in Interstate Commerce|凱弗維爾委員會}}聽證會之一。該博物館還獲得了發生[[情人節大屠殺|情人節大屠殺]]的血跡牆 。其他展覽重點是黑幫暴力，賭場瞞報贏利的運作和執法竊聽。<ref>{{Cite web |url = http://www.reviewjournal.com/jane-ann-morrison/what-will-be-mob-museums-cutting-room-floor |title = What will be on Mob Museum's cutting room floor? |author = Jane Ann Morrison |date = 2010-03-27 |publisher = Las Vegas Review-Journal |language = en |deadurl = yes |archiveurl = https://web.archive.org/web/20170227150211/http://www.reviewjournal.com/jane-ann-morrison/what-will-be-mob-museums-cutting-room-floor |archivedate = 2017-02-27 }}</ref>

在第三層開始自導參觀，這裏展出了情人節大屠殺的實際牆壁，使用電梯或樓梯可到達，並且搖動它下來到第二和第一樓。遊客可以觀看一部簡明但是非常有教育意義的電影，介紹有組織犯罪的歷史（從[[禁酒時期|禁酒令]]通過的時間開始）。展覽的主要部分包括黑幫相關照片，加上說明文字來解釋特別圖片的意義，而且說明他們與什麼類人和事件有連結。除了上述的審判室和情人節大屠殺之外，一些最有趣的物品也被展出，一把[[湯普森衝鋒槍|湯普森槍]]（可以觸摸，握著，甚至「假裝」射擊）、電椅，還有{{link-en|Albert Anastasia|Albert Anastasia}}座著被謀殺的理髮椅。

[[File:Las_Vegas_Mob_Museum_Wall_of_Mobsters.JPG|thumb]]
就收藏品而言，博物館提供了廣泛大批的主題部分，針對與黑幫牽連的非法活動，例如賭博、毒品、賣淫或非法私酒，以及執法部門努力對抗和消滅這些犯罪行為。此外，訪客能夠看到一些特別令人不安，也很少接近到的材料，例如被歸因於黑手黨的最著名謀殺罪的受害者照片（適當地命名為「黑幫的最重大謀殺」（Mob's Greatest Hits）；這些照片在本質上是相當生動的，因為他們展示的是死者的真實屍體），還有最普遍和臭名昭著的黑幫特性的圖片和簡短傳記。位於出口附近的特殊牆壁描述了所有在電影或系列中飾演著名黑幫的演員圖片。大量有意義的展覽都是互動的，還有幾個車站坐落遍布於博物館，會循環播放相關的電影片段。<ref>{{Cite web |url = http://www.deesultimatereviews.com/the%20mob%20museum.htm |title = THE MOB MUSEUM |publisher = deesultimatereviews.com |language = en}}</ref>

這裏有一間禮品店在一樓。使用了照片、文字、展品、互動技術、親身動手的展覽和其他一流博物館的方法，訪客可了解到有組織犯罪集團的歷史、禁酒令和其提供的商機。訪客還可以了解拉斯維加斯的第一個賭場、[[侯活·曉治|侯活·曉治]]、[[約翰·埃德加·胡佛|J.埃德加·胡佛]]，聯邦調查局（FBI）、[[艾爾·卡彭|艾爾·卡彭]]、[[艾略特·內斯|艾略特·內斯]]和更多的起源。訪客可以坐在複製品電椅上、聽真實的竊聽器、在警察模擬裝置內訓練，或「射擊」一把實際的[[湯普森衝鋒槍|湯普森槍]]。
