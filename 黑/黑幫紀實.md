《'''黑幫紀實'''》（{{lang-en|'''''The Making of the Mob'''''}}）是一部美國{{link-en|電視紀錄片|Television documentary|電視紀錄片連續劇}}，內容講述20世紀美國[[黑社會|黑社會]]的形成。本劇於2015年6月15日在[[AMC_(TV_channel)|AMC]]上首播，由演員[[雷·李歐塔|雷·李歐塔]]旁白。本劇也會在每集交切一些歷史檔案鏡頭，還有歷史學者、作家、演員、執法人員及黑幫家族成員的訪談。2015年7月31日AMC續訂第二季共八集，於2016年7月11日播出。<ref>{{Cite web |url = http://variety.com/2015/tv/news/making-of-the-mob-renewed-season-2-amc-1201554472/ |title = AMC Renews 'The Making of the Mob' for Season 2 Based in Chicago |author = Elizabeth Wagmeister |date = 2015-07-31 |publisher = Variety |language = en}}</ref>

== 概要 ==
=== 第一季（2015年） ===
{{main|紐約黑幫紀實}}
第一季副標題是《紐約》，主要強調[[查理·盧西安諾|查理·盧西安諾]]以及他在紐約市黑幫[[五大家族|五大家族]]的崛起。

=== 第二季（2016年） ===
{{main|芝加哥黑幫紀實}}
第二季副標題是《芝加哥》，集中於[[艾爾·卡彭|艾爾·卡彭]]的起源，他的[[芝加哥犯罪集團|芝加哥犯罪集團]]，以及在[[美國中西部|美國中西部]]的發展。<ref>{{Cite web |url = http://deadline.com/2015/07/the-west-docudrama-robert-redford-amc-1201488855/ |title = AMC Orders 'The West' Docudrama From Robert Redford, Renews 'Making Of The Mob' |author = Nellie Andreeva, Dominic Patten |date = 2015-07-31 |publisher = Deadline Hollywood |language = en}}</ref>
