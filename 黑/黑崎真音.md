'''黑崎真音'''（{{bd||1月13日}}）是[[日本|日本]][[女性|女性]][[歌手|歌手]]及[[作詞家|作詞家]]。隸屬於日本[[NBC環球娛樂|NBC環球娛樂]]公司旗下。[[血型|血型]]A型。所屬事務所為[[一二三|一二三]]，暱稱「ヲ嬢」。

== 經歷 ==

2006年秋天左右於東京‧吉祥寺一帶以團體一員開始現場活動，2008年在日本[[東京|東京]][[秋葉原|秋葉原]]的Dear Stage以個人名義演唱，被現在的音樂製作人冨田明宏相中，2010年以演唱[[電視動畫|電視動畫]]《[[學園默示錄|學園默示錄 HIGHSCHOOL OF THE DEAD]]》片尾曲集正式出道。

於TV動畫《[[魔法禁書目錄|魔法禁書目錄II]]》擔當前期和後期片尾曲，填詞配合作品世界觀而廣受好評。其後陸續演唱[[薄櫻鬼|薄櫻鬼 雪華錄]]系列片尾曲、小說改編電影『リアル鬼ごっこ3･4･5』的主題曲、女性向遊戲[[十鬼之絆_～關原奇譚～|十鬼之絆 ～關原奇譚～]]主題曲、薄櫻鬼 黎明錄的片頭曲黎鳴 -reimei-、[[軍火女王|軍火女王]]的二期片頭曲UNDER／SHAFT、[[東京闇鴉|東京闇鴉]]的片頭曲X-encounter等等。

2011年以MAON名義與[[fripSide|fripSide]]的sat和[[m.o.v.e|m.o.v.e]]的motsu組成團體「[[ALTIMA|ALTIMA]]」，演唱過《[[灼眼的夏娜|灼眼的夏娜Ⅲ]]》、[[加速世界|加速世界]]和[[噬血狂襲|噬血狂襲]]等作品的歌曲。

2011年起連續四年以個人及團體名義登上Animelo Summer Live，2010年亦開始參加Animax Musix等大型歌唱活動，除了日本國內以外，也曾去過洛杉磯、上海、廣州、香港、台北、馬來西亞、曼谷、荷蘭等各地演唱。

2016年8月ALTIMA宣布活動休止後，目前以個人名義活躍中。

==個人情報與逸聞==

暱稱為「ヲ嬢」，意思是「御宅族的大小姐」，之前擔任廣播節目A&G ARTIST ZONE 2h主持人時產生的稱呼。

自述於2008年Animelo Summer Live觀賞[[栗林美奈實|栗林美奈實]]的現場演出深受震撼<ref>{{Cite web|url = http://ceron.jp/url/mantan-web.jp/2013/04/11/20130411dog00m200033000c.html|title = 黒崎真音：栗林みな実のライブ見て歌手に　音楽のルーツ語る|publisher = [[毎日新聞デジタル|毎日新聞デジタル]]|date = 2013-04-11|5 = |deadurl = yes|archiveurl = https://web.archive.org/web/20131202235802/http://ceron.jp/url/mantan-web.jp/2013/04/11/20130411dog00m200033000c.html|archivedate = 2013-12-02}}</ref>。以此為契機朝成為動畫歌手為目標，在Animelo Summer Live 2011 -rainbow-與栗林一起合唱「翼はPleasure Line」<ref>{{Cite web|url = http://ameblo.jp/kurosakimaon/entry-11001544756.html|title = animelo summer live2011 rainbow! |publisher = [[AMEBA|AMEBA]]|date = 2011-08-29|}}</ref>
，其後在筑波大學的學園祭實現雙人演唱會的夢想。

女性向遊戲重度愛好者，喜歡的遊戲有[[純愛手札|純愛手札]]，喜歡的動漫作品有[[渡瀨悠宇|渡瀨悠宇]]的[[夢幻遊戲|夢幻遊戲]]等等。

喜歡的食物為日本蕎麥麵、梅乾、壽司、玄米茶<ref>http://profile.ameba.jp/kurosakimaon/</ref>、炸馬鈴薯。

喜歡的顏色是紫色，演唱會和活動的螢光棒多以紫色為主。

喜歡的音樂類型為[[伊凡塞斯|伊凡塞斯]]（Evanescence）、[[致命誘惑|致命誘惑]]（Within Temptation）等[[歌德金屬|歌德金屬]]系<ref>{{Cite web|url =http://www.animate.tv/news/details.php?id=1350048067|title = アニメ『ヨルムンガンドPERFECT ORDER』OPテーマ歌う黒崎真音さんにインタビュー|publisher = [[アニメイトTV|アニメイトTV]]|date = 2012-10-12|}}</ref>。

2014年迷上[[寶塚|寶塚]]、觀賞[[職業摔角|職業摔角]]還有研究[[大王烏賊|大王烏賊]]的生態。喜歡的寶塚演員有[[柚希礼音|柚希礼音]](現已退團)<ref>{{Cite web|url = https://twitter.com/kurosakimaon/status/601723856252899329|title = すきだーーーーー礼音さん}}</ref>。喜歡的摔角選手有[[中邑真輔|中邑真輔]]<ref>{{Cite web|url = https://twitter.com/kurosakimaon/status/521652559451152384|title = 大好きなプロレスラー中邑真輔さんのTシャツをGETしました}}</ref>、[[永田裕志|永田裕志]]。

業界內私交比較好的友人有[[佐咲紗花|佐咲紗花]]、[[藍井艾露|藍井艾露]]<ref>http://ameblo.jp/eir-ruru/entry-11624538203.html</ref>、[[STAR☆ANIS|STAR☆ANIS]]的WAKA([[霧島若歌|霧島若歌]])、[[電波組.inc|電波組.inc]]的[[古川未鈴|古川未鈴]]<ref>{{Cite web|url = https://twitter.com/sola_waka/status/572046524747661312|title = ディアステみりおんわかの初期メン3}}</ref><ref>{{Cite web|url = http://ameblo.jp/dolce-waka/entry-11916723620.html|title = 夢みたい|publisher = [[AMEBA|AMEBA]]|date = 2014-08-29|}}</ref>
等人，唱片公司前輩[[KOTOKO|KOTOKO]]、[[川田真美|川田真美]]、後輩[[Ray|Ray]]等也經常在個人網誌和推特等地方多次提及。

家中排行老么，有兩個姐姐，一直想要有個[[傲嬌|傲嬌]]的乾弟弟<ref>{{Cite web|url = https://twitter.com/kurosakimaon/status/595950436051324928|title = 三姉妹です！}}
</ref>。

育有兩隻[[馬爾濟斯|馬爾濟斯]]犬，名為プッチ與ココ。
