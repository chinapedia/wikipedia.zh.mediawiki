'''[[黑山|黑山]]'''的历史开始于中世纪早期。

==早期历史==
黑山的先民為[[伊利里亞人|伊利里亞人]]，公元前3世紀時被[[古羅馬|古羅馬]]征服。羅馬帝國衰落以後，伊利里亞落入[[哥特人|哥特人]]之手，後來[[拜占庭帝國|拜占庭帝國]]（東羅馬帝國）皇帝[[查士丁尼一世|查士丁尼一世]]又將該地區重新納入版圖。

早期斯拉夫民族自6、7世紀起抵達此地，並且和當地先民融合，建立自己的國家。其中，杜克里亞（{{Lang|me|Duklja}}）公國比較接近蒙特內哥羅的版圖。
==中世纪==
面臨之后保加尔人的统治，杜克里亞領土被兄弟領主分割。在10世纪，塞尔维亚的[[弗拉斯蒂米洛維奇王朝|弗拉斯蒂米洛維奇王朝]]的[[查斯拉夫·克洛尼米洛维奇|查斯拉夫·克洛尼米洛维奇]]王公将影响力扩张到杜克里亞地区。自塞尔维亚公国于960年陷落后，杜克里亞被新的拜占庭帝国占领直至11世纪。斯特凡·沃伊斯拉夫开始反抗拜占庭统治的起义，并在1042年与拜占庭军队在巴尔之战中取得大胜。至此结束了拜占庭帝国在杜克里亞的影响。1077年羅馬教皇[[額我略七世|額我略七世]]承认杜克里亞为独立国家，封米哈伊洛為杜克里亞國王（Rex Doclea）。杜克里亞於12世紀衰落的同時，拉什卡（{{Lang|me|Raška}}）大公國（又稱塞爾維亞大公國）興起了新的[[內馬尼亞王朝|內馬尼亞王朝]]，杜克里亞成為塞爾維亞的一部分。

在[[內馬尼亞王朝|內馬尼亞王朝]]統治下，杜克里亞地區又出現了澤塔（{{Lang|me|Zeta}}）公國，14世紀中時澤塔脫離了內馬尼亞的控制。

==奧斯曼帝國和黑山親王主教國==
[[Image:Montenegro1913.png|thumb]]
15世紀起[[奧斯曼帝國|奧斯曼帝國]]征服了巴爾幹半島大部分地區，澤塔于1498年被征服，併入奧斯曼帝國的斯庫台桑扎克（{{Lang|tr|İşkodra Sancağı}}）。1514年澤塔地區從斯庫台桑扎克分離，成為黑山桑扎克（{{Lang|tr|Karadağ Sancağı}}）。但奧斯曼帝國未能征服澤塔全境，例如[[洛夫琴山|洛夫琴山]]地區保持了獨立。
[[Image:Proclamation_of_the_Kingdom_of_Montenegro.jpg|thumb]]1516年，蒙特內哥羅成為以[[采蒂涅|-{采}-蒂涅]]為中心的政教合一國家，國君為集政教權利於一身的[[親王主教|親王主教]]（{{Lang|me|Vladika}}）。
==黑山公國 和黑山王國==
1852年，黑山親王主教、彼德洛維奇-涅戈什家族的達尼洛二世（{{Lang|sr|Danilo II Petrović Njegoš}}）放棄宗教頭銜，成為[[黑山公國|黑山公國]]的達尼洛一世，黑山又成為世俗公國。1860年，達尼洛二世被暗殺，[[尼古拉一世_(蒙特內哥羅)|尼古拉一世]]繼位。1876年黑山向奧斯曼帝國宣戰，並奪得大量土地。黑山於[[1878年柏林會議|1878年柏林會議]]中正式被承認為獨立國。1910年，黑山大公尼古拉一世正式稱王，建立[[黑山王國|黑山王國]]。

== 兩次世界大戰與南斯拉夫社會主義聯邦共和國 ==
[[第一次世界大戰|第一次世界大戰]]期間黑山加入了[[協約國|協約國]]以協助盟友[[塞爾維亞|塞爾維亞]]，隨即被[[奧匈帝國|奧匈帝國]]侵略。在前兩次入侵中，黑山成功抵禦了奧匈帝國的軍隊，但在1916年1月的第三次入侵中，黑山被奧匈帝國完全佔領。尼古拉一世逃亡至義大利、法國。1918年，塞爾維亞等協約國的軍隊解放黑山。

一戰結束後，在塞爾維亞的主導之下，斯洛文尼亞、克罗地亚聯合組成了第一個由南斯拉夫族群聯合的國家。同時，塞爾維亞控制下的黑山也召開會議，廢除本國國王、禁止國王回國，加入新成立'''塞爾維亞人、克罗地亚人和斯洛文尼亞人王國'''。<ref>John R. Lampe, 2002, Yugoslavia as History: Twice There was a Country. New York: Cambridge University Press, pp. 102-106.</ref>1919年，反對派發起[[聖誕節起義|聖誕節起義]]，但是被塞爾維亞軍鎮壓。1929年，國王[[亞歷山大一世_(南斯拉夫)|亞歷山大一世]]正式將王國更名為「[[南斯拉夫王國|南斯拉夫王國]]」，1939年[[二戰|二戰]]爆發，1941年4月[[納粹德國|納粹德國]]及其盟國[[入侵南斯拉夫|入侵南斯拉夫]]，[[義大利王國|義大利王國]]自[[阿爾巴尼亞意占時期|意屬阿爾巴尼亞]]入侵蒙特內可羅，4月17日義大利軍隊佔領蒙特內哥羅全境，1943年德軍取代義大利的占領，1945年第二次世界大戰結束，德軍撤出南斯拉夫。南斯拉夫民族解放反法西斯議會制憲會議宣布南斯拉夫成立共產政府，由五個民族共組聯邦，罷黜君主制。<ref>江秉彝，2007，〈南斯拉夫疆界與民族衝突(1991-2006)〉，碩士論文，東海大學政治系，頁16-22。</ref>黑山成為了[[南斯拉夫社會主義聯邦共和國|南斯拉夫社會主義聯邦共和國]]的加盟共和國之一。

1990年代，南斯拉夫聯邦各國紛紛獨立，唯有塞、黑兩國未獨立。黑山於1992年最後一次舉行獨立公投，95.96%認為應繼續留在南斯拉夫聯邦之內。當時投票率為66%，[[穆斯林|穆斯林]]、[[天主教徒|天主教徒]]以及親獨的黑山人抵制了公投，他們認為投票環境不夠民主，國營媒體過多為親統一的一方宣傳。之後黑山國內在獨立問題上頗有紛爭。
[[File:FilipVujanovic.jpg|thumb]]
1996年，[[米洛·久卡諾維奇|米洛·久卡諾維奇]]政府宣佈和[[米洛舍維奇|米洛舍維奇]]掌權的[[塞爾維亞|塞爾維亞]]斷絕關係，從此以後黑山自行制定經濟政策，改用[[德國馬克|德國馬克]]（後來德國廢除馬克，採用[[歐元|歐元]]，黑山亦改為[[歐元|歐元]]）。从此[[塞爾維亞第納爾|塞爾維亞第納爾]]在黑山不是法定貨幣，只能在一些度假村使用。

由於和[[米洛舍維奇|米洛舍維奇]]掌權的[[塞爾維亞|塞爾維亞]]斷絕關係，蒙特內哥羅在接下來的南斯拉夫和[[科索沃戰爭|科索沃戰爭]]中受到損傷並不如[[塞爾維亞|塞爾維亞]]來的大，局勢也較[[塞爾維亞|塞爾維亞]]穩定許多。

==南聯瓦解與黑山獨立==
2000年南斯拉夫聯盟大選後米洛舍維奇下台，使得歐盟與南斯拉夫聯盟恢復外交關係，西方因此由支持黑山獨立，轉向支持維持現有的聯邦狀態。但塞爾維亞與黑山間的關係並沒有因為西方國家支持維持現狀而改善，而1992年南斯拉夫聯盟的憲法中又沒有提及關於聯邦共和國實體獨立的權利及過程，從而引發爭論。<ref>江秉彝，2007，〈南斯拉夫疆界與民族衝突(1991-2006)〉，碩士論文，東海大學政治系，頁86。</ref>後來在歐盟的調停下，雙方於2002年3月14日在[[貝爾格萊德|貝爾格萊德]]簽署協議，決定將聯邦變為由兩個各自擁有經濟、貨幣以及關稅決定權的半獨立（semi-independent）國家所組成，並且更名為「[[塞爾維亞和黑山|塞爾維亞和黑山]]」。<ref>BBC News, 2002.3.14, “Yugoslav partners sign historic deal,” http://news.bbc.co.uk/2/hi/europe/1872070.stm.</ref>另外依據此協定通過了聯邦的新憲章，明定憲章通過三年後，任何一方都有權提出脫離聯邦。<ref>張志涵，2005.1，〈後冷戰時期族群衝突與血緣國干涉〉，碩士論文，台灣大學政治研究所，頁121。</ref>

黑山國會於2004年7月12日採用新的國旗、國慶日、以及國歌，以推進獨立進程。其中國旗基於前黑山王國的國旗，國慶日紀念1878年7月13日簽訂的[[1878年柏林條約|柏林條約]]承認黑山為獨立國家，以及1941年7月13日爆發的反法西斯起義。國歌則為黑山著名民歌。

2006年5月21日舉行[[2006年黑山獨立公民投票|公民投票]]，[[歐盟|歐盟]]再次介入並設定一套條件多數決的過關標準，此公投的標準為需有過半的合格選民投票，且贊成獨立的選票需達55%；合格選民包括居住於蒙特內哥羅境內兩年以上的人士，也包括居住在塞爾維亞的蒙特內哥羅公民，有別於1990年代，克羅埃西亞、波士尼亞與赫塞哥維納聯邦、斯洛維尼亞，或馬其頓脫離南斯拉夫的公投都是簡單多數決。<ref>{{Cite web |url = https://global.udn.com/global_vision/story/8663/2424823 |title = 東擴巴爾幹：蒙特內哥羅的北約賭局 |author = 徐子軒 |date = 2017年4月25日 |publisher = 轉角國際 |language = 中文 |accessdate = 2017年6月22日 }}</ref>6月3日，黑山國會正式宣佈獨立，28日黑山正式加入[[聯合國|聯合國]]。

2006年9月10日，黑山共和国举行独立后首次议会和地方选举<ref>[http://news.xinhuanet.com/world/2006-09/10/content_5074125.htm 黑山举行独立后首次议会和地方选举 ] {{webarchive|url=https://web.archive.org/web/20140221220214/http://news.xinhuanet.com/world/2006-09/10/content_5074125.htm |date=2014-02-21 }}</ref>。9月12日，选举委员会宣布，[[争取实现欧洲的黑山联盟|争取实现欧洲的黑山联盟]]获得议会81个席位中的41个议席，[[塞族名单|塞族名单]]联盟获得了12席，由[[社会主义者人民党|社会主义者人民党]]、[[人民党（黑山）|人民党]]和[[民主塞尔维亚党|民主塞尔维亚党]]组成的竞选联盟和[[变革运动党|变革运动党]]分别获得11席，其余6席由一些较小党派分获<ref>[http://news.xinhuanet.com/world/2006-09/13/content_5083286.htm 黑山执政联盟赢得独立后首次大选 ] {{webarchive|url=https://web.archive.org/web/20120502223937/http://news.xinhuanet.com/world/2006-09/13/content_5083286.htm |date=2012-05-02 }}</ref>。11月10日，新政府宣誓就职，总理为[[熱利科·什图拉诺维奇|熱利科·什图拉诺维奇]]<ref>[http://news.xinhuanet.com/world/2006-11/11/content_5315753.htm 黑山独立后首届新政府宣誓就职] {{webarchive|url=https://web.archive.org/web/20140221220210/http://news.xinhuanet.com/world/2006-11/11/content_5315753.htm |date=2014-02-21 }}</ref>。

2008年黑山當局釋出加入[[北約|北約]]的風向球，一開始反對的人遠多於支持，那時的民調顯示有六成不希望成為北約成員，2015年變為一半對一半的態勢，此時黑山當局尚無法決定要以全民公投，抑或是國會投票的方式加入北約，久卡諾維奇打算先在國會提出決議，表達執政當局的決心，2015年9月舉行投票，結果在81名議員裡有50人贊成，除執政聯盟外，也包含少數反對黨議員，同年底北約對黑山發出入盟邀請，2016年初黑山國內又發起數次抗議，並持續要求政府舉行入盟公投。<ref>{{Cite web |url = https://global.udn.com/global_vision/story/8663/2424823 |title = 東擴巴爾幹：蒙特內哥羅的北約賭局 |author = 徐子軒 |date = 2017年4月25日 |publisher = 轉角國際 |language = 中文 |accessdate = 2017年6月22日 }}</ref>
