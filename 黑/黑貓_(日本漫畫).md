《'''黑貓'''》是[[日本|日本]][[漫畫家|漫畫家]][[矢吹健太朗|矢吹健太朗]]所創作的[[漫畫|漫畫]]，以及依此改編的[[電視動畫|電視動畫]]與[[遊戲|遊戲]]。

== 概要 ==
原作從2000年開始在[[週刊少年Jump|週刊少年Jump]]（[[集英社|集英社]]）連載，於2004年1月號連載結束。單行本全20集<ref>{{cite web|url=http://acg.gamer.com.tw/acgDetail.php?s=384 | title=【漫畫】黑貓BLACK CAT|publisher=巴哈姆特|accessdate=2014-01-21 |language=zh-tw}}</ref>
。連載完畢之後2005年開始依此改編的[[廣播劇CD|廣播劇CD]]及[[小說|小說]]出版<ref>{{cite web|url=http://acg.gamer.com.tw/acgDetail.php?s=22180 | title=【輕小說】黑貓BLACK CAT|publisher=巴哈姆特|accessdate=2014-01-21 |language=zh-tw}}</ref>
。[[電視動畫|電視動畫]]於2005年]]10月起至2006年3月止於日本地區播放，一共24集<ref>{{cite web|url=http://acg.gamer.com.tw/acgDetail.php?s=18977 | title=【動畫】黑貓BLACK CAT|publisher=巴哈姆特|accessdate=2014-01-21 |language=zh-tw}}</ref>；於2007年起，分別於香港、東亞及台灣[[Animax|Animax]]播放。[[音泉|音泉]][[網路電台|網路電台]]從2006年3月30日開始每星期四「RADIO BLACK CAT」開始播放。而遊戲分別在[[PlayStation_2|PlayStation 2]]以及[[任天堂DS|任天堂DS]]推出遊戲，分別是《黑貓 機械天使》（{{lang|ja|BLACK CAT 〜機械仕掛けの天使〜}}）<ref>{{cite web|url=http://game.watch.impress.co.jp/docs/20060313/bc.htm | script-title=ja:カプコン、人気コミックを初ゲーム化!PS2「BLACK CAT ～機械仕掛けの天使～」|publisher=GAME Watch|accessdate=2014-01-21 |language=ja}}</ref>以及《黑貓 黑貓的協奏曲》（{{lang|ja|BLACK CAT 〜黒猫の協奏曲〜}}）。

== 故事簡介 ==
曾经[[暗杀|暗杀]]过无数政界要人的机密组织「克羅諾斯」前王牌[[杀手|杀手]]托雷，人称“黑貓”，相传由于背叛组织而被杀。可是，2年后，他竟然再度出现，做起了“賞金獵人”。他与拥有“預知眼”的前国际搜查官史恩搭档，再加上好财如命的变装高手琳斯雷特和[[纳米|纳米]]生体兵器伊芙，一同寻找庫利得，以了结2年前结下的恩怨……

== 登場人物 ==
=== 主要角色 ===
; {{nihongo|托雷·哈特涅特（原No.XIII）|トレイン＝ハートネット|Train Heartnet|[[配音員|配音員]]：[[近藤隆|近藤隆]]、[[高山南|高山南]]（小托雷）（日本），[[何志威|何志威]]、[[李明幸|李明幸]]（小托雷）（台灣）}}
:本作男主角，外號「黑貓」，地下世界機密組織「克羅諾斯」裡令人聞風喪膽的傳奇性殺手，槍法如神，典型的冷血殺手。十歲時父母為殺手薩奇涅所殺，後一邊跟隨薩奇涅學習殺人技巧，一邊伺機報仇。但薩奇涅在一次任務中重傷而死，托雷從此獨自過著非生即死的生活。而後在卡爾的關係加入克羅諾斯，成為克羅諾斯的殺手。
:直到托雷遇上水無月·沙耶後，開始思考自己真正想要的生活方式。而在沙耶遭庫利得殺害後，留下遭處死的假紀錄後離開組織克羅諾斯，展開了賞金獵人的生活，同時也四處尋找庫利得，以為沙耶報仇。機緣下結識同為賞金獵人的史恩·波爾菲德，由於十分投緣，在認識的第三天正式結為搭擋。
:武器是「裝飾槍-哈迪斯」，為特殊材質「奧理哈康」所製，一般的槍的[[槍托|槍托]]因不能承受托雷的連續射擊而容易損壞。與庫利得最後一戰發射炸裂電磁槍後，槍枝嚴重損毀。在兩年後由史恩修復，但破損的部分已非奧理哈康。

:作為矢吹的下一個作品的漫畫『[[出包王女|出包王女]]』出現一個外表和個性及武器都和托雷相似的角色庫羅。庫羅應該是作者有意讓他登場與主角群互動的人物，可惜過沒多久[[出包王女|出包王女]]突然在162話宣告完結，庫羅由始至終僅出場一次而已。（後在『出包王女Darkness』第70話再次登場。）

; {{nihongo|史恩·波爾菲德|スヴェン＝ボルフィード|Sven Vollfied|配音員：[[藤原啓治|藤原啓治]]（日本），[[官志宏|官志宏]]（台灣）}}
:托雷的拍擋史恩是一位風度翩翩的紳士，外表溫文有禮，談吐得體，以自己所謂的「紳士道」做為生活原則。手上的公事包為自己特製，內有許多槍械與機關。
:史恩為「國際調查局」(I.B.I)前探員，經常以眼罩遮蓋具有預知能力的「預知眼」，這是探員時期好友羅德的遺物。預知眼能看到未來幾分秒的事物，但因為會消耗大量體力，所以史恩只有在危機時才使用。後為幫助托雷對抗星之使徒，不斷訓練自己克服預知眼的使用極限，結果預知眼因而蛻變，成為全新的「支配眼」。

; {{nihongo|水無月 沙耶|サヤ  ミナツキ|Saya Minatsuki|配音員：[[豐口惠|豐口惠]]（日本），[[黃珽筠|黃珽筠]]（台灣）}}
:賞金女獵人，穿著日式和服的年輕少女，也是令托雷退出克羅諾斯的人物，非常喜歡喝牛奶。
:動畫中第一次與托雷見面時，沙耶剛好在屋頂上唱歌，引起了托雷注意，雖然托雷不承認，但他確實為了和她見面而放了瓶牛奶引誘她。
:槍法了得，其發出的「跳彈」能從各種角度擊敗敵人。被庫利得當成誘惑托雷的魔女，為了使其將托雷救出，因而遭到庫利得殺害。

; {{nihongo|伊芙|イヴ|Eve|配音員：[[福圓美里|福圓美里]]（日本）}}
:透過奈米技術及複製而成的伊芙，生來就被培養成殺人武器，但直至遇上史恩，意志開始在伊芙的內心萌發，後來在托雷與史恩的活躍下才獲得自由，之後因希望能幫上史恩的忙而跟其一同旅行。
:剛開始個性單純，在接觸外界後精神快速成長，想法變得十分成熟。有要超越托雷，成為史恩搭擋的野心。
:伊芙可透過體內數億個與自己意識同步的納米機器，隨意將身驅變成任何東西。尤其隨著伊芙知識與想像力的增加，變形的種類與精緻度也隨之上升。
:常用變形招式分別有髮束之拳、大鐵鎚、盾牌、天使翅膀、羽毛子彈，後期還有人魚型態、全身鋼鐵化與奈米切割器。
:被托雷取上「小公主」的暱稱，緣由是他認為伊芙就像被壞人囚禁起來的公主。
:作為矢吹的下一個作品的漫畫『[[出包王女|出包王女]]』出現一個外表和個性及能力都和伊芙相似的角色金色闇影（金色の闇），同作品的電視劇CD的廣播劇演員與伊芙一樣由福圓擔任。

; {{nihongo|琳斯雷特·沃克|リンスレット＝ウォーカー|Rinslet Walker|配音員：[[野上尤加奈|野上尤加奈]]（日本），[[林美秀_(配音員)|林美秀]]（台灣）}}
:琳斯雷特為惡名昭彰的小偷仲介人，因打算侵入托魯涅歐的宅邸，初次與史恩和托雷合作。
:她喜歡戴上一頭假髮，喬裝成金髮美女，而且功夫了得。

=== 時間的守護者（克羅諾斯CHRONOS）=== 
為克羅諾斯最精銳的暗殺部隊，原由12個最強的暗殺者組成，在托雷特例加入後為13人。成員皆有以名為「奧理哈康」的特殊合金所打造專用武器，以及代表為其中成員的「時間的刺青」。
; {{nihongo|賽菲莉亞·亞克斯（No.I）|セフィリア＝アークス|Sephiria Arks|配音員：[[井上喜久子|井上喜久子]]（日本），[[李明幸|李明幸]]（台灣）}}
:統領時間守護者的年輕主將。對任何人都很好，個性小心謹慎；不過為了達成任務，無論任何犧牲都在所不辭，同時具有冷酷的一面。裝備是長刀〝庫拉伊斯特〞。動畫結局為擔任克羅諾斯主導者繼續為保護世界和平努力。

; {{nihongo|貝爾傑·羅希福爾（No.II）|ベルゼー＝ロシュフォール|Belze Rochefort|配音員：[[堀秀行|堀秀行]]（日本），[[林谷珍|林谷珍]]（台灣）}}
:時間守護者的副隊長。外表看似嚴肅，其實有著替人著想的溫柔。對克羅諾斯十分忠誠，但對身為叛徒的托雷毫無敵意。使用的武器為長槍〝滾滾尼爾〞。

; {{nihongo|艾米利歐（No.III）|エミリオ＝ロウ|Belze Rochefort|配音員：[[千葉進步|千葉進步]]（日本）}}
:其手持武器是名為〝亞提米斯〞(アルテミス)的奧理哈康製長弓。(只出現於動畫中，受梅索教唆一同背叛克羅諾斯)

; {{nihongo|克蘭茲·馬多克（No.IV）|クランツ＝マドゥーク|Kranz Maduke|配音員：[[梯篤司|梯篤司]]（日本）}}
:從出生就被以守護者的姿態施予戰鬥教育。雖然在幾年前的戰鬥中失去視力，但是深信唯有戰鬥才能夠證明自己存在的他，開發出藉由感受〝聲音〞與〝空氣〞的戰術，武器是超音速震動小刀〝戰神〞。

; {{nihongo|奈薩·布萊凱瑪（No.V）|ナイザー＝ブラッカイマー|Nizer Bruckheimer|配音員：[[江川央生|江川央生]]（日本），[[官志宏|官志宏]]（台灣）}}
:賽伯拉斯的領隊。賽伯拉斯是由近距離攻擊的奈薩，中距離攻擊的傑諾斯、及遠距離攻擊的貝魯迦所組成，是重視攻擊力‧突破能力的部隊。武器是拐子〝迪歐斯庫洛伊〞。

; {{nihongo|阿努比斯（No.VI）|アヌビス|Anubis|}}
:一頭有人類智慧與說話能力的黑狼，其左眼有Ⅵ的刺青，其附屬武器是裝備在尾巴上名為〝歐西里斯〞(オシリス)的奧理哈康製長鞭。(只出現於動畫中)

; {{nihongo|傑諾斯·哈薩德（No.VII）|ジェノス＝ハザード|Jenos Hazard|配音員：[[櫻井孝宏|櫻井孝宏]]（日本），[[林谷珍|林谷珍]]（台灣）}}
:自稱是時間守護者中最珍惜淑女的人，對琳斯頗有好感。雖然是幾年前就加入時間守護者，不過好像幾乎沒見過托雷，基本上，其他的守護者都在世界各地奔走，很多人都彼此不相識。裝備是〝ECTHELON〞(銀線)。

; {{nihongo|巴魯多利亞斯·S·芬基尼（No.VIII）（通稱：巴魯多爾）|バルドリアス＝S＝ファンギーニ|Baldorias S. Fanghini|配音員：[[勝杏里|勝杏里]]（日本）}}

; {{nihongo|特比德（No.IX）|デイビッド＝ペッパー|David Pepper|配音員：[[黑田崇矢|黑田崇矢]]（日本）}}

; {{nihongo|林·夏歐利（No.X）|リン＝シャオリー|Lin Shaolee|配音員：[[齋賀光希|齋賀光希]]（日本）}}
:接替兩年前遭庫利得殺害的NO.X亞許，而成為守護者。精通快速變裝術，有個綽號叫"魔術師"，除此之外的資料都是謎。武器是〝SEIREN〞，是用奧里哈康鋼線所編織成的羽衣。似乎是中國籍。在討伐庫利得時提出僱用賞金獵人而假扮成〝古林〞組成賞金獵人同盟，在動畫中被托雷拆穿。

; {{nihongo|貝魯迦·J·哈特（No.XI）|ベルーガ＝J＝ハード|Beluga J. Heard|配音員：[[石井康嗣|石井康嗣]]（日本）}}
:賽伯拉斯部隊中的砲手。使用武器是火箭筒〝烏魯斯拉古那〞。因為攜帶武器的關係，參加的任務大多是與敵對組織抗爭或是國際間的糾紛。

; {{nihongo|梅索·奧爾特羅索（No.XII）|メイソン＝オルドロッソ|Mason Ordrosso|配音員：[[納谷六朗|納谷六朗]]（日本），[[官志宏|官志宏]]（台灣）}}
:為克羅諾斯元老，動漫最後為背叛克羅諾絲殺死三長老與醫生聯手想要藉由伊甸園計畫毀滅世界，潛伏在克羅諾斯二十五年，並吸收同伴，其武器是名為〝阿包亞庫〞(ア・バオア・クー)的奧理哈康製特殊鎧甲。最後是受托雷感化的庫利得與他對決。

; {{nihongo|亞許（原No.X）|アッシュ|Ash|配音員：[[武虎|武虎]]（日本）}}
:在漫畫為回憶中的人物，並沒呈現長相。在動畫有登場，武器是前端有尖刺的杖。他與奈薩圍攻庫利得時被迎頭斬成兩半死亡。
:因為他的死亡造成No.X位子空缺，後面才安排夏歐利遞補。

=== 星之使徒 ===
; {{nihongo|庫利得·迪斯肯斯|クリード＝ディスケンス|Creed Diskenth|配音員：[[三木真一郎|三木真一郎]]（日本），[[林谷珍|林谷珍]]（台灣）}}

== 漫畫跟動畫上的差異 ==
漫畫跟動畫在劇情上有些差異：

* 1.漫畫一開始'''托雷'''已經脫離克羅諾斯，結識'''史恩'''與'''伊芙'''，並跟前者作搭檔。動畫中托雷初次登場的時候仍然是克羅諾斯中聞風喪膽的第13號殺手"'''黑貓'''"，還是跟'''庫利得'''為搭檔。
* 2.漫畫中'''托雷'''跟'''史恩'''第一次見面是在某個小鎮的酒館。動畫則是在托雷執行任務來到一個城鎮，在要殺的目標舉辦的晚宴上與史恩第一次碰面（只是兩眼相交）。
* 3.漫畫中'''琳斯'''一開始跟一群人演戲來找合適的獎金獵人時是'''托雷'''，動畫改成'''史恩'''。
* 4.'''史恩'''在公園遇見'''伊芙'''的時候，並沒有被伊芙以變身能力刺穿身體，而是在'''托魯涅歐'''的宅院被刺穿身體。
* 5.'''托魯涅歐'''在漫畫中對'''伊芙'''使用藥物，要伊芙失控去攻擊史恩，但是被'''托雷'''阻止。動畫改為讓變身能力強化的卡片，並刪去托雷阻止的情節。
* 6.'''托魯涅歐'''的地下研究所在漫畫中是被'''托雷'''炸毀。動畫改成'''琳斯'''。
* 7.'''庫利得'''在動畫一開始就有"'''道'''"的能力並跟道士'''希奇'''有了交流，愛刀虎徹是被'''托雷'''給射斷的，而且早就計畫好要背叛'''克羅諾斯'''企圖組成個比克羅諾斯還要強大的組織（'''星之使者'''）並拉攏剛脫離克羅諾斯的托雷加入。
* 8.動畫中'''托雷'''在目睹'''沙耶'''慘死在'''庫利得'''手下之後馬上跟其打起來，但是似乎是被打敗了，把托雷救起來的是'''史恩'''跟'''伊芙'''。
* 9.動畫中'''托雷'''的鈴鐺在動畫上是托雷在祭典中救起的3個小孩（也就是'''沙耶'''救的小孩）送給他的一支黑貓娃娃上的鈴鐺（貓是沙耶給他們的），然後'''伊芙'''在把貓脖子上的鈴鐺拿下來給托雷的，也在那之後托雷成為了他們的夥伴，並換上漫畫初登場的藍色衣服。（ps.那時候已經是動畫時間過了半年了）

* 10.漫畫版時間守護者中的No.Ⅲ、Ⅵ、Ⅸ(3號、6號、9號)並沒有登場，No.ⅩⅡ(12號)在漫畫最後一集也僅僅露個臉而已。
這4位守護者的長相、名字跟武器在動畫都有交代，唯一沒交代的是3號、9號、12號的刺青位置！

* 11.由於動畫跟漫畫劇情差異極大，守護者No.Ⅳ、No.Ⅷ(4號、8號)在漫畫被定位成對克羅諾斯絕對忠誠，但在動畫後期卻背叛克羅諾斯。(No.Ⅲ與No.ⅩⅡ也是同行背叛者)

* 12.'''星之使者'''與'''塞伯拉斯'''城堡一戰，漫畫版原本只有No.ⅩⅠ(11號)死亡，動畫劇情還要再追加一名死者No.Ⅴ(5號)。

* 13.動畫15話有交代'''夏登'''與'''恭子'''分道揚鑣後的劇情。'''夏登'''跑去單挑'''賽菲利亞'''，但是打輸了。

* 14.漫畫的結局打敗庫利得後結束，但是動畫結局是卻是以原創劇情伊甸園計畫做結束。
