'''黑鑽樹蟒'''（學名：''Morelia boeleni''）是[[蛇亞目|蛇亞目]][[蟒科|蟒科]][[樹蟒屬|樹蟒屬]]下的一種無毒[[蟒蛇|蟒蛇]]，主要分布於[[新畿內亞|新畿內亞]]的山脈地區。<ref name="Meh87">Mehrtens JM. 1987. Living Snakes of the World in Color. New York: Sterling Publishers. 480 pp. ISBN 0-8069-6460-X.</ref>目前未有任何亞種被確認。<ref name="ITIS">{{ITIS|ID=634776|taxon=''Morelia boeleni''|year=2007|date=18 September}}</ref>

== 特徵 ==
黑鑽樹蟒以[[黑色|黑色]]為體色基調，身上佈有奶白色的斑紋，其唇鱗亦呈[[白色|白色]]。其軀體質地看上去像油質般滑溜，鱗片色彩鮮艷奪目，有[[閃鱗蛇|閃鱗蛇]]的特色。

== 地理分布 ==
黑鑽樹蟒主要分布於[[印尼|印尼]]、[[巴布亞新畿內亞|巴布亞新畿內亞]]等地區，標準產地為印尼西新畿內亞的「Dimija (3°56'S, 136°18'E), Wissel Lakes」。<ref name="McD99"/>

== 生態習慣 ==
黑鑽樹蟒多出沒於高山林木之間，主要捕食[[哺乳動物|哺乳動物]]及[[鳥類|鳥類]]。
