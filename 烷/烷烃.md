== 性質概論 ==
烷類皆難溶於水，在完全燃燒下可轉化為[[二氧化碳|二氧化碳]]和[[水|水]]<ref name="P=hd">{{cite book | author = 許立侖、許喆翔 | title = 《國中自然與生活科技 第四冊學習講義》P138 | publisher = 康軒文教事業}}</ref>，反應式如下：
:<math>\rm C_nH_{2n+2}+\frac{3n+1}{2}\ O_2\rightarrow  n\ CO_2+(n+1)\ H_2O</math>
烷類會因組合'''結構'''的不同，其性質可能會有極大的差異。若[[結構|結構]]相似，則性質相近；若結構不同，性質不同。<ref name="P=hd"></ref> 正因此，烷類種類繁多；但也由此許多[[化学式|化學式]]相同的物質，其性質不相同。如[[戊烷|戊烷]]（{{Chem|C|5|H|12}}）<ref name="1/2gt2">[[异戊烷|异戊烷]]</ref>：

* [[密度|密度]]皆小於一。

== 化学性质 ==
在正常情況下，烷烃性质很稳定，因为[[碳-氫鍵|碳－氫键]]和[[碳－碳键|碳－碳键]]相对稳定，难以断裂<ref>{{cite book|author=冀政勤、毛翰梅|title=有機化學|year=2001-02-22|publisher=五南|location=台北市|isbn=957-11-2276-9|url=http://books.google.com.tw/books?id=GA-TxBQfpJoC&pg=PA36&lpg=PA36&dq=%E5%9B%9B%E7%B4%9A%E7%A2%B3&source=bl&ots=Wa_h-ZGI84&sig=IxuSUqqj5gxKwP85hfuH77DGDMc&hl=zh-TW&ei=vYyZTbrkA5HEvgO_uZ3kCw&sa=X&oi=book_result&ct=result&resnum=8&ved=0CEsQ6AEwBw#v=onepage&q&f=false}}</ref> 、以及烷烴不會搶奪[[官能团|官能團]]，所以不容易發生[[反應|反應]]，除了下面三种反应，小分子的烷烃几乎不能进行其他反应。但支链多的烷烃，键角可能不同于<math>\arccos \left( -\frac{1}{3} \right)</math>即109.5度，导致其容易發生反应。
=== 燃燒反应 ===
反應通式：<math> \rm R + O_2 \rightarrow  CO_2 + H_2O </math>{{NoteTag|R即為烷烴。}}

所有的烷烃都能燃烧，而且反应[[放热反应|放热]]极多<ref name="noname">{{cite web |url = http://content.edu.tw/senior/chemistry/tp_sc/new/14/2.htm |title = 學習加油站遷移轉址到教育雲-教育大市集 |publisher=Content.edu.tw |date= |accessdate=2017-08-17 }}</ref>。烷烃完全燃烧生成[[二氧化碳|CO<sub>2</sub>]]和[[水|H<sub>2</sub>O]]<ref name="P=hd" />。如果O<sub>2</sub>的量不足，就会产生有毒气体[[一氧化碳|一氧化碳]]（CO），甚至炭黑（C）<ref name="bigref">{{cite web |url = http://highscope.ch.ntu.edu.tw/wordpress/?p=28419 |title=烷類(Alkanes)（二）──化學反應 | 科學Online |publisher=Highscope.ch.ntu.edu.tw |date=2014-06-20 |accessdate=2017-08-17 }}</ref>。

以甲烷为例：
* CH<sub>4</sub> + 2 O<sub>2</sub> → CO<sub>2</sub> + 2 H<sub>2</sub>O
O<sub>2</sub>供应不足时，反应如下：
* 2CH<sub>4</sub> + 3 O<sub>2</sub> → 2CO + 4 H<sub>2</sub>O
* CH<sub>4</sub> + O<sub>2</sub> → C + 2 H<sub>2</sub>O
分子量大的烷烃经常不能够完全燃烧，它们在燃烧时会有黑烟产生，就是炭黑。

=== 卤代反应 ===
反應通式：<math> \rm R + X_2 \rightarrow RX + HX </math>{{NoteTag|X即為[[鹵素|鹵素]]。}}

由于烷烃的结构太牢固，一般的有机反应不能进行。烷烃的[[卤素|卤]]代反应是一种[[自由基|自由基]][[取代反应|取代反应]]，反应的起始需要光能来产生自由基。

以下是甲烷被卤代的步骤。这个高度[[放热反应|放热]]的反应可以引起爆炸。
# 链引发阶段：在紫外线的催化下形成两个Cl的自由基
#* Cl<sub>2</sub> → Cl* / *Cl
# 链增长阶段：一个H原子从甲烷中脱离；CH<sub>3</sub>Cl开始形成。
#* CH<sub>4</sub> + Cl* → CH<sub>3</sub><sup>+</sup> + HCl （慢）
#* CH<sub>3</sub><sup>+</sup> + Cl<sub>2</sub> → CH<sub>3</sub>Cl + Cl*
# 链终止阶段：两个自由基重新组合
#* Cl* 和 Cl*, 或
#* R* 和 Cl*, 或
#* CH<sub>3</sub>* 和 CH<sub>3</sub>*.<ref name="powerpoint">{{cite web |url = https://docs.google.com/viewer?a=v&q=cache:UylIK0hlBXMJ:163.23.148.7/sweb/pub/downfiles.php%3Frec_id%3D176%26rel_nm%3D1%26account%3Djiahui+%E7%83%B7%E9%A1%9E+%E7%87%83%E7%87%92&hl=zh-TW&gl=tw&pid=bl&srcid=ADGEESjVn_S4UBj386pOI7NtQloiuyGfCymgttRAfWW2AE_PJB9awjuKFypwR0q1w3QyFOwcCqsao9NApFyloGNUquPoILBB4iZ2ZHgi_DLolYLdP7zuIRY4qYXTKkczeEWQbTLiXZv4&sig=AHIEtbTaNJ-0RUBSkexAGhzfaapV4QNrZQ |title= |publisher= |date= |accessdate=2017-08-17 }}</ref>

=== 裂解反应 ===
裂化反应是大分子烃在高温、高压或有催化剂的条件下，分裂成小分子烃的过程。裂化反应属于消除反应，因此烷烃的裂化总是生成烯烃。如十六烷(C<sub>16</sub>H<sub>34</sub>)经裂化可得到辛烷(C<sub>8</sub>H<sub>18</sub>)和辛烯(C<sub>8</sub>H<sub>16</sub>)。

由于每个键的环境不同，断裂的机率也就不同，下面以丁烷的裂化为例讨论这一点：
# CH<sub>3</sub>-CH<sub>2</sub>-CH<sub>2</sub>-CH<sub>3</sub> → CH<sub>4</sub> + CH<sub>2</sub>=CH-CH<sub>3</sub>
#* 过程中CH<sub>3</sub>-CH<sub>2</sub>键断裂，可能性为48%；
# CH<sub>3</sub>-CH<sub>2</sub>-CH<sub>2</sub>-CH<sub>3</sub> → CH<sub>3</sub>-CH<sub>3</sub> + CH<sub>2</sub>=CH<sub>2</sub>
#* 过程中CH<sub>2</sub>-CH<sub>2</sub>键断裂，可能性为38%；
# CH<sub>3</sub>-CH<sub>2</sub>-CH<sub>2</sub>-CH<sub>3</sub> → CH<sub>2</sub>=CH-CH<sub>2</sub>-CH<sub>3</sub> + H<sub>2</sub>
#* 过程中C-H键断裂，可能性为14%。<ref name="powerpoint" />

裂化反应中，不同的条件能引发不同的机理，但反应过程类似。热分解过程中有碳[[自由基|自由基]]产生，催化裂化过程中产生[[碳正离子|碳正离子]]和[[氢负离子|氢负离子]]。这些极不稳定的中间体经过重排、键的断裂、氢的转移等步骤形成稳定的小分子烃。

在工业中，深度的裂化叫做裂解，裂解的产物都是气体，称为[[裂解气|裂解气]]。

== 命名 ==
{{main|IUPAC命名法}}

命名烷類的首要步驟是分析其為環狀亦或是鏈狀，再進行命名步驟。<ref>參考自IUPAC命名法</ref>
=== 鏈烷烴 ===

# 找出最長的[[碳|碳]]鏈當主鏈，依碳數命名主鏈，前十個以[[天干|天干]]{{NoteTag|十[[天干|天干]]：甲、乙、丙、丁、戊、己、庚、辛、壬、癸}}代表碳數，碳數多於十個時，以[[汉语|中文]][[數字|數字]]命名，如：十一烷。
# 從最近的[[取代基|取代基]]位置編號：1、2、3...(使取代基的位置數字越小越好)。以數字代表取代基的位置。數字與中文數字之間以 - 隔開。
# 有多個取代基時，以取代基數字最小且最長的碳鏈當主鏈，並依甲基、乙基、丙基的順序列出所有取代基。

==== 具多環的環烷烴的命名法 ====
除非有[[俗名|俗名]]，否則具多環的環烷烴如[[桥环化合物|桥环]]烷烴及[[螺环化合物|螺环]]烷烴的命名較為複雜。名字包括表示環數量的前綴（如「二環」）、各環內碳原子總數的後綴以及表示各端點之間碳原子數的數字前綴（表示於中括號內。多個環公用的碳原子，即橋頭碳不計入內）。

這個環烷總碳數為七，由兩個環組成，尾為「庚烷」，頭為「雙環」。兩個被共用的碳原子間有三個連接路線：一為五元環的部分，共三個碳（兩個橋頭碳不計入內，下同）；二為四元環的部分，共二個碳；三為兩環之間共用的邊綫，只由兩橋頭碳直接連結，中間沒有碳。由此得出中括號內的數字（以降序表示數字之間用點分隔），[3.2.0]。故上圖的環烷名為二环[3.2.0]庚烷，而數字的個數總比環數多一個（在此有兩個環及三個數字）。「[3.2.0]二环庚烷」亦可接受，但環上有取代基時「二环[3.2.0]庚烷」有保留前面的位置的好處，方便加上其他含數字的前綴，以符合IUPAC命名常規。

上圖環烷總碳數為七，全為單鍵，詞尾為庚烷；兩共用碳間有兩對二碳碳鏈，也被一個碳原子連接著，故詞尾之前的數字前綴為[2,2,1]（共用碳不計入內）；這個環烷由兩個環組成，故前綴為「雙環」，最後得出「雙環[2.2.1]庚烷」。

== 结构 ==
烷烃中，每个碳原子都是四[[化合价|价]]的，采用sp<sup>3</sup>[[混成軌域|杂化]]轨道，与周围的4个碳或氢原子形成牢固的[[σ键|σ键]]，可以旋轉，故烷類無[[順反異構物|順反異構物]]。<br>

=== 太陽系的分布 ===
烷烴分布於太陽系間許多星球的大氣層。有些佔了較多的比例，例如[[天王星|天王星]]（2.3%）<ref name="abcdefg">{{cite web |url=http://news.xinhuanet.com/ziliao/2006-08/21/content_4988813.htm |title=存档副本 |accessdate=2011-09-15 |deadurl=yes |archiveurl=https://web.archive.org/web/20100906132306/http://news.xinhuanet.com/ziliao/2006-08/21/content_4988813.htm |archivedate=2010-09-06 }}</ref>、[[土卫六|土衛六]]（5%）<ref name="5%">http://www.natureasia.com/taiwan/nature/updates/index.php?i=5857&issue=7080{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，但在大多星球上，分布較少，如[[地球|地球]]、[[火星|火星]]、[[土星|土星]]等等<ref>{{cite journal | author=Courtin, R.; Gautier, D.; Marten, A.; Bezard, B. | title=The Composition of Saturn's Atmosphere at Temperate Northern Latitudes from Voyager IRIS spectra | journal=Bulletin of the American Astronomical Society | year=1967 | volume=15 | pages=831 | url=http://adsabs.harvard.edu/abs/1983BAAS...15..831C | accessdate=2007-02-04 }}</ref>。
=== 地球上的分布 ===
地球上的烷烴多為甲烷，而甲烷的濃度隨地球[[纬度|緯度]]降低而遞減，並在[[纬度|北緯]]40度及[[赤道|赤道]]附近都有明顯濃度下降。北半球與南半球平均濃度各為1.65ppm及1.55ppm。<ref name="C222224165">http://210.60.226.25/science/content/1986/00110203/0010.htm{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

== 應用 ==
=== 工業上的應用 ===
[[File:Alkane_IUPAC1.PNG|thumb]]）的结构式。]]
由于烷烃的制取成本较高（一般要用烯烃催化加氢），所以在工业上不制取烷烃，而是直接从[[石油|石油]]中提取。由于烷烃不易发生反应，所以工业上也不把它作为化工基本原料。烷烃的作用主要是做燃料。天然气和沼气（主要成分为甲烷）是近来广泛使用的清洁能源。
