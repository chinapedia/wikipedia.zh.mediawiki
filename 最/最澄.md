'''最澄'''（{{jpn|j=最澄|hg=さいちょう|rm=Saichō}}，{{bd|767年|9月19日|822年|6月26日|最澄}}），俗名'''三津首广野'''（{{jpn|j=三津 首広野|hg=みつのおびとひろの|rm=Mitsu no obitohirono}}），密教法號福聚金剛，出生于[[日本|日本]][[近江国|近江国]]（[[滋贺县|滋贺县]]）滋贺郡古市乡（今[[大津市|大津市]]），[[平安時代|平安時代]]僧人、日本[[天台宗|天台宗]]的开创者。

他也是著名書法家，作品中以[[久隔帖|久隔帖]]最為著名。

== 生涯 ==
12岁在近江[[国分寺|国分寺]]出家，座师[[行表|行表]]和尚，14岁受戒，得法名'''最澄'''。

19岁（[[延历|延历]]4年（785年））在[[奈良|奈良]][[东大寺|东大寺]]受[[具足戒|具足戒]]。同年7月，登[[比叡山|比叡山]]修行，通读[[大藏经|大藏经]]。

797年，被任命为内供奉（ないぐぶ）十禅師之一。

802年，被任命为高雄山寺（[[神護寺|神護寺]]）'''法華会'''（ほっけえ）的講師。入选入[[唐|唐]]求法的短期留学生。

804年7月、同[[空海|空海]]以及任翻译的师弟[[義真|義真]]等自[[九州_(日本)|九州]]出发。9月抵达明州（今[[中国|中国]][[浙江|浙江]][[宁波|宁波]]）。遂登[[天台山_(浙江)|天台山]]，向[[湛然|湛然]]弟子辈的[[道邃|道邃]]、[[行満|行満]]等禅师学习天台宗教义，受[[菩薩戒|菩薩戒]]。

805年5月、回日本，在[[和田岬|和田岬]]（今[[神户市|神户市]]）上陸，开设了日本最早的[[密宗|密宗]][[道场|道场]][[能福寺|能福護国密寺]]。7月入京都，在途中缮写了佛经230部460卷。适逢[[桓武天皇|桓武天皇]]卧病、最澄在宮中为天皇祈祷康复。

822年6月，最澄於比叡山中道院去世，享年56歲，866年，[[清和天皇|清和天皇]]追諡「傳教大師」。

== 日本後紀 ==
『[[日本後紀|日本後紀]]』[[嵯峨天皇|嵯峨天皇]]紀，錄有[http://applepig.idv.tw/kuon/furu/text/kouki/kouki30.htm#kou30_02 最澄薨傳]。

　癸亥，[[傳燈大法師|傳燈大法師]]位'''最澄'''，卒。云云。[[延曆|延曆]]末，入唐請益。皇太子詹事陸淳左降台州刺史，會屈[[天台宗|天台宗]]道邃和尚為座主。儻預講筵，稟學略了。良緣有感，一面為歡。助寫宗書三百餘卷訖，即復本職，拜別上京。既而隨使歸來，弘演宗義。云云。年五十有四。

== 日本國史略 ==
『[[日本國史略|日本國史略]]』[[桓武天皇|桓武天皇]]紀中，屢有[https://web.archive.org/web/20060616051506/http://applepig.idv.tw/kuon/furu/text/siryaku/siryaku02-050.htm 贊評]，言及'''最澄'''。其詞如下：

　二年，敕曰：「京畿定額諸寺，其數有限。既禁私立道場。比來，所司寛縱，曾不糺察。如經年代，無地不寺。宜嚴加禁斷。」<font size=2>【松苗曰：[[佛法|佛法]]東渡以來，世創寺塔。至前朝建[[國分寺|國分寺]]，其弊極矣。[[韓子|韓子]]所謂：『人其人，火其書，廬其居。』者，勢既不易為也。帝立此禁，寔權宜良方也。獨怪未幾，[[延曆|延曆]]七年，僧'''最澄'''，創寺於[[比叡山|比叡山]]，建[[根本中堂|根本中堂]]。安[[藥師寺|藥師寺]]像，以稱濟世醫國。其後子院眾多，遂及三千。[[鴨川_(淀川水系)|鴨水]]東北，無地不寺。民居，僅夾堂塔之間。而[[山徒|山徒]]暴虐，動起甲兵。譬由[[庸醫|庸醫]]，口稱仁術，每損人命。遂至使後世[[天子|天子]]，有：『鴨水之漲與[[山僧|山僧]]之暴，朕亦無奈之何。』之嘆。嗚呼！自[[佛法|佛法]]入我邦以降，[[蠹國|蠹國]]害政，未有如是甚者也。惜哉當時，處此良方，徒置不用，而使病勢益劇。茍有志濟世者，安得不長大息乎。後經七百餘歲，平公[[織田信長|信長]]，一怒，火攻之。天下大患，始得痊癒矣。治療之功，卻出自[[武將|武將]]。豈謂之[[下策|下策]]劫法乎。】</font>
