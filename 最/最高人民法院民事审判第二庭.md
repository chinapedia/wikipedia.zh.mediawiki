'''最高人民法院民事审判第二庭'''，位于[[北京市|北京市]][[东城区_(北京市)|东城区]][[东交民巷|东交民巷]]27号，是[[中华人民共和国最高人民法院|中华人民共和国最高人民法院]]内设机构。

==沿革==
1976年10月[[粉碎四人帮|粉碎“四人帮”]]后，刑事审判庭、民事审判庭、办公厅等业务单位、行政单位逐步恢复。到1983年，设有刑事审判第一庭、刑事审判第二庭、民事审判庭、经济审判庭、研究室、办公厅、人事厅、司法行政厅。2000年，将经济审判庭变更为民事审判第二庭<ref>{{Cite web |url = http://rmfyb.chinacourt.org/paper/html/2014-11/23/content_90796.htm?div=-1 |title =最高人民法院机构设置简史  |author =  |date =2014-11-23  |publisher = 人民法院报 |language =  |accessdate =  |quote =  }}</ref>。

==职能==
2000年12月4日，《最高人民法院机关内设机构及新设事业单位职能》（法发〔2000〕30号）发布，其中规定最高人民法院机关内设机构有民事审判第二庭，职能是<ref name=zhineng/>：
#审判第一、二审国内法人之间、法人与其他组织之间的合同纠纷和侵权纠纷案件，审判第一、二审国内证券、期货、票据、公司、破产等案件。
#审理申请撤销国内仲裁的案件。
#办理相关的申请复议案件。
#审批高级人民法院相关案件延长审限的申请<ref name=zhineng>{{Cite web |url = http://www.pkulaw.cn/fulltext_form.aspx?Gid=124743 |title =  最高人民法院机关内设机构及新设事业单位职能|author =  |date =  |publisher =北大法宝  |language =  |accessdate = 2017-04-05 |quote =  }}</ref>。

目前民事审判第二庭“主要负责最高人民法院审理的第一、二审合同、公司、证券、保险、票据等商事纠纷案件；审查和审判不服下级人民法院生效裁判的商事审判监督案件；指导有关审判工作。”<ref>{{Cite web |url =http://www.court.gov.cn/jigou-fayuanjigou-zhineng-12.html  |title = 民事审判第二庭 |author =  |date =  |publisher =中华人民共和国最高人民法院  |language =  |accessdate = 2017-04-06 |quote =  }}</ref>

民事审判第二庭主办《商事审判指导》、《商事审判裁判规范与案例指导》等刊物<ref>{{Cite web |url = http://www.court.gov.cn/shenpan-xiangqing-6182.html |title =  最高人民法院民事审判第二庭2014年工作要点|author =  |date = 2014-03-08  |publisher = 中华人民共和国最高人民法院 |language =  |accessdate =  |quote =  }}</ref>。
