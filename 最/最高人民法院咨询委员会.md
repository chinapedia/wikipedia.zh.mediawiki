'''最高人民法院咨询委员会'''，是[[中华人民共和国最高人民法院|中华人民共和国最高人民法院]]的咨询机构。

==简介==
1993年12月25日，最高人民法院咨询委员会第一次会议在[[北京|北京]][[香山_(北京)|香山]]召开<ref name=wutai>{{Cite web |url = http://www.chinacourt.org/article/detail/2013/11/id/1151269.shtml |title = 发挥余热的大舞台 法院发展的智囊团 |author =  |date = 2013-11-27 |publisher = 中国法院网 |language =  |accessdate =  |quote =  }}</ref>。

2009年4月修订的《最高人民法院咨询委员会简章》第一条规定：“咨询委员会是最高人民法院的咨询机构，在院党组领导下，提供建议、信息和接受院党组委派的工作任务。”第二条规定：“咨询委员会的主要任务是：对涉及人民法院工作的重大问题进行调查研究，并向院党组提供咨询意见；反映人民法院工作在当前形势下出现的新情况、新问题；反映人民群众对人民法院工作的意见和要求。”“受院党组的委派，对下级人民法院的审判工作、队伍建设、行政管理工作进行检查、督促、指导。”<ref name=jianzhang>{{Cite web|url =http://www.chinalaw.gov.cn/article/fgkd/xfg/sfwj/201012/20101200330539.shtml|title =最高人民法院咨询委员会简章（2009年4月修订）|author =|date =2010-12-14|publisher =国务院法制办公室|language =|accessdate =|quote =}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

2012年9月，最高人民法院院长[[王胜俊|王胜俊]]在最高人民法院咨询委员会第二十九次会议上发言称，“咨询委员会是最高人民法院党组的决策咨询机构，是最高人民法院党组贯彻执行党的路线、方针、政策，作出重大决策部署的重要参谋，是最高人民法院出台司法政策、司法解释，监督指导全国法院工作的智囊团。”<ref name=jiemi>{{Cite web |url = http://news.sina.com.cn/c/2014-02-14/012029461523.shtml |title = 最高法咨询委员会解密：退休干部组成智囊团 |author =  |date =2014-02-14  |publisher = 新浪 |language =  |accessdate =  |quote =  }}</ref>

==历次会议==
1993年12月25日，最高人民法院咨询委员会第一次会议在北京香山举行。最高人民法院党组书记、院长[[任建新|任建新]]出席并讲话。会议任命[[王怀安|王怀安]]为最高人民法院咨询委员会主任，[[林准|林准]]为常务副主任，[[华联奎|华联奎]]、[[鲁明健|鲁明健]]、[[赵石生|赵石生]]、[[黄波|黄波]]为副主任，聘请同年离任的15名[[高级人民法院|高级人民法院]]院长和[[中国人民解放军军事法院|中国人民解放军军事法院]]院长为委员。会议讨论通过了《最高人民法院咨询委员会简章》<ref name=wutai/>。

1994年5月12日到14日，最高人民法院咨询委员会第二次会议在[[浙江|浙江]][[温州|温州]]举行。会议专题研讨审判方式改革，会后形成《最高人民法院咨询委员会第二次会议纪要》，报送[[全国人民代表大会常务委员会法制工作委员会|全国人民代表大会常务委员会法制工作委员会]]作为修改《[[中华人民共和国刑事诉讼法|中华人民共和国刑事诉讼法]]》的参考<ref name=wutai/>。

1995年7月25日到29日，最高人民法院咨询委员会第四次会议在[[江西|江西]][[井冈山|井冈山]]举行。会议专题研讨民事、经济审判方式改革<ref name=wutai/>。

1996年12月23日到27日，最高人民法院咨询委员会委员参加了在北京召开的全国高级法院院长会议，并列为大会分组中的一个组，为最高人民法院咨询委员会第七次会议。会议宣布王怀安辞去最高人民法院咨询委员会主任，担任最高人民法院咨询委员会名誉主任，由华联奎接任最高人民法院咨询委员会主任<ref name=wutai/>。

1997年6月10日到15日，最高人民法院咨询委员会华北组贯彻刑事诉讼法座谈会在[[山西|山西]][[太原|太原]]举行<ref name=wutai/>。

1998年10月11日到13日，最高人民法院咨询委员会第十次会议在[[河南|河南]][[郑州|郑州]]举行。会议根据最高人民法院党组关于聘任最高人民法院咨询委员会的范围和条件的规定，修改了1993年制定的《最高人民法院咨询委员会简章》。委员们讨论了集中教育整顿和深化法院改革<ref name=wutai/>。

1999年4月，最高人民法院咨询委员会第十一次会议讨论了人民法院第一个“五年改革纲要”<ref name=wutai/>。

2000年4月20日到25日，最高人民法院咨询委员会第十三次会议在[[广西|广西]][[南宁|南宁]]举行。会议宣布华联奎不再任最高人民法院咨询委员会主任，由[[马原|马原]]接任主任<ref name=wutai/>。

2001年5月12日，最高人民法院咨询委员会第十五次会议在[[湖北|湖北]][[武汉|武汉]]召开<ref>{{Cite web |url = http://www.cnhubei.com/xwzt/2008zt/hbgg30y/dsj30y/200803/t254916_4.shtml |title = 2001 |author =  |date = 2008-03-14 |publisher = 荆楚网 |language =  |accessdate =  |quote =  }}</ref>。

==组成人员==
2009年4月修订的《最高人民法院咨询委员会简章》第三条规定：“咨询委员会由主任、副主任、秘书长和委员组成。”“咨询委员会设主任1人、副主任若干人；设秘书长1人，协助主任、副主任工作。根据工作需要，可以设副秘书长1人。”第四条规定：“咨询委员会主任、副主任由院党组在最高人民法院离任的副院长、纪检组组长、政治部主任中选任。秘书长、副秘书长由最高人民法院在职干部担任。”“咨询委员会委员由最高人民法院聘任。最高人民法院离任的省部级领导干部和各省（自治区、直辖市）高级人民法院、解放军军事法院离任院长中身体健康、本人提出申请的，经院党组研究决定聘任。”“离任后调党委、政府或者企业、[[事业单位|事业单位]]任职的，不再聘任。”<ref name=jianzhang/>
