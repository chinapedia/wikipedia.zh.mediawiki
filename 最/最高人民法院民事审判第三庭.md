'''最高人民法院民事审判第三庭'''（最高人民法院知识产权审判庭），位于[[北京市|北京市]][[东城区_(北京市)|东城区]][[东交民巷|东交民巷]]27号，是[[中华人民共和国最高人民法院|中华人民共和国最高人民法院]]内设机构。

==沿革==
1996年10月，最高人民法院成立知识产权审判庭。2000年，将知识产权审判庭更名为民事审判第三庭<ref name=jigoujsh>{{Cite web |url = http://rmfyb.chinacourt.org/paper/html/2014-11/23/content_90796.htm?div=-1 |title =最高人民法院机构设置简史  |author =  |date =2014-11-23  |publisher = 人民法院报 |language =  |accessdate =  |quote =  }}</ref>。2006年，《最高人民法院关于本院民事审判第三庭对外称“知识产权审判庭”的通知》（法发〔2006〕14号）发出<ref>《最高人民法院关于本院民事审判第三庭对外称“知识产权审判庭”的通知》（法发〔2006〕14号）</ref>。

==职能==
2000年12月4日，《最高人民法院机关内设机构及新设事业单位职能》（法发〔2000〕30号）发布，其中规定最高人民法院机关内设机构有民事审判第三庭，职能是<ref name=zhineng/>：
#审判第一、二审著作权(包括计算机软件)、商标权、专利权、技术合同、不正当竞争以及科技成果权、植物新品种权等知识产权案件。
#审查处理不服高级人民法院生效裁判的知识产权申请再审案件以及少数立案庭移送的不服基层人民法院、中级人民法院生效裁判的知识产权申请再审案件。
#办理知识产权申请复议案件。
#审批高级人民法院知识产权案件延长审限的申请<ref name=zhineng>{{Cite web |url = http://www.pkulaw.cn/fulltext_form.aspx?Gid=124743 |title =  最高人民法院机关内设机构及新设事业单位职能|author =  |date =  |publisher =北大法宝  |language =  |accessdate = 2017-04-05 |quote =  }}</ref>。

目前民事审判第三庭“主要负责最高人民法院审理的第一、二审知识产权及竞争纠纷案件；审判不服下级人民法院生效裁判的知识产权及竞争审判监督案件；指导有关审判工作。”<ref>{{Cite web |url = http://www.court.gov.cn/jigou-fayuanjigou-zhineng-13.html|title =民事审判第三庭(知识产权审判庭) |author =  |date =  |publisher =中华人民共和国最高人民法院  |language =  |accessdate = 2017-04-06 |quote =  }}</ref>
