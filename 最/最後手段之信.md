'''最後手段之信'''（{{lang-en|Letters of last resort}}），是[[英國首相|英國首相]]以相同措辭書寫並交給四名[[前衛級核潛艇|彈道導彈潛艇]]艦長的親筆信<ref name="首相的信">{{Cite web |url = http://mytv.tvb.com/tc/ed_inewscatchup/internationalnewsfile/297775 |title = 首相的信 |work = 香港無綫新聞 |date = 2017-02-22 |accessdate = 2017-03-22 |deadurl = yes |archiveurl = https://web.archive.org/web/20170323054111/http://mytv.tvb.com/tc/ed_inewscatchup/internationalnewsfile/297775 |archivedate = 2017-03-23 }}</ref>。當[[英國政府|英國政府]]因敵人的核打擊而被摧毀，且首相及其個人任命的替補決策者（通常是[[英國內閣|內閣]]高級閣員）皆死亡或失去行動能力時，艦長便會按照信中的內容採取相應行動<ref name="首相的信"/><ref name="HMAS Apocalypse">{{Cite web |url = http://www.dailymail.co.uk/news/article-1090400/HMS-Apocalypse-Deep-Atlantic-submarine-waits-alert-nuclear-missiles-end-world--.html |title = HMAS Apocalypse: Deep in the Atlantic, a submarine waits on alert with nuclear missiles that would end the world... |work = 每日邮报 |date = 2008-11-30 |language = en |accessdate = 2017-03-22 }}</ref>。無論艦長執行了哪種命令，他們所採取的行動可能是[[英國政府|女王陛下政府]]的最後一次正式行為<ref name="Doomsday"> {{Cite book | author = Martin R Mason | title = Doomsday Scenarios | publisher = lulu.com | date = 2015-14-12 | ISBN = 978-1329054523 | accessdate = 2017-03-22 | language = en}} </ref>。

== 流程 ==
當新任英國首相上任後，[[英國國防參謀長|英國國防參謀長]]便會準確告知「[[三叉戟飛彈|三叉戟飛彈]]會造成什麼損害」，然後書寫親筆信<ref name="Theresa May's first job">{{Cite web |url = https://www.theguardian.com/politics/2016/jul/12/theresa-mays-first-job-decide-on-uks-nuclear-response |title = Theresa May's first job: decide on UK's nuclear response |work = 卫报 |date = 2016-07-12 |language = en |accessdate = 2017-03-22 }}</ref>。其後，這些信件會分別交付至四艘攜帶三叉戟洲際核彈潛艇，而前任首相的信件會在確保沒有打開的情況下銷毀<ref>{{Cite web |url = http://news.sky.com/story/may-to-be-handed-keys-to-nuclear-red-button-10499943 |title = May To Be Handed Keys To Nuclear Red Button |work = 天空新闻台 |date = 2016-07-13 |language = en |accessdate = 2017-03-22 }}</ref>。

如果首相及其指定的替補決策者皆因核打擊而死亡，核潛艇的艦長（在任何時間至少會有一艘潛艇持續出海游弋）將會以一系列的檢查來確定信件有無打開的必要<ref>{{Cite web |url = http://www.politico.eu/article/the-grim-task-awaiting-theresa-may-preparing-for-nuclear-armageddon-uk-prime-minister-british-defense-letter-of-last-resort/ |title = Theresa May’s grim first task: Preparing for nuclear armageddon |author = Garrett M. Graff |date = 2016-07-15 |publisher = Politico |language = en |accessdate = 2017-03-22 }}</ref>。

根據英國歷史學家{{link-en|彼得·軒尼詩|Peter Hennessy}}《秘密國家：白廳和冷戰》一書，在1945年至1970年，潛艦艦長決定英國政府是否繼續運作的因素之一包括確認[[BBC廣播四台|BBC廣播四台]]有無繼續廣播<ref name="HMAS Apocalypse"/>。在1983年，當搭載了[[UGM-27北極星導彈|北極星A3導彈]]的[[決心級核潛艇|決心級核潛艇]]發現有明顯的核攻擊或所有英國海軍廣播已經停止四小時，便會打開信件<ref>{{Citation |url = http://discovery.nationalarchives.gov.uk/SearchUI/details/C13497321?uri=C13497321 |title = Arrangements for briefing a new Prime Minister in the event of a change of administration following the 1983 General Election |author = David Goodall |date = 1983-05-26 |publisher = 英國國家檔案館 |id = CAB 196/124 | accessdate  = 2017-03-22}}</ref>。

== 選項 ==
雖然信件的內容是機密，但根據2008年12月BBC廣播四台的紀錄片《人類的按鈕（The Human Button）》，有四個已知的選項會供首相書寫至信件內，包括：
* 使用[[核武器|核武器]]報復；
* 不採取報復行動；
* 根據形勢自行判斷；
* 可以的話，將潛艇置於盟軍國家的指揮下{{noteTag|1=紀錄片中提到的「盟軍國家」包括澳洲和美國。}}。

[[衛報|衛報]]在2016年7月的報導表示，這些選項據說還包括：「如果美國還存在，接受其命令」、「去澳洲」、「報復」<ref name="Theresa May's first job"/>。
