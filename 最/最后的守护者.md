{{Vgname|1=最后的守护者|en=The Last Guardian|ja=人喰いの大鷲トリコ|official-cn=最后的守护者|tw=食人巨鷹 TRICO|aka=食人的大鹫}}是一款由[[ICO小组|ICO小组]]、[[genDESIGN|genDESIGN]]与[[索尼电脑娱乐日本工作室|索尼电脑娱乐日本工作室]]联合开发制作的[[动作冒险游戏|动作冒险游戏]]，预定于2016年12月上市，由[[PlayStation_4|PlayStation 4]]平台独占。正式發布前以Project Trico的名义被外界所熟知，该游戏的监督是[[上田文人|上田文人]]，此前曾监督了《[[ICO|ICO]]》以及《[[汪达与巨像|汪达与巨像]]》两部作品，本作被认为和这两部作品有相似的世界设定，影像风格以及游戏設計。



== 开发与发行 ==
《食人巨鷹 TRICO》开始立项开发于2007年，在2009年的[[E3电玩游戏展|E3电玩游戏展]]上正式公布并且发布了预览动画并当时计划将于2011年发售<ref>Kennedy，Sam（2009-05-19）."http://www.1up.com/do/newsStory?cId=3174299"</ref><ref>{{cite web|last=Yin-Poole|first=Wesley|title=The Last Guardian suffers delay|url=http://www.eurogamer.net/articles/2011-04-20-the-last-guardian-suffers-delay|publisher=[[Eurogamer|Eurogamer]]|accessdate=2011-05-22|date=2011-04-20}}</ref>。不过此后关于本作的消息逐渐减少；在此期间，本作的制作人[[上田文人|上田文人]]从索尼电脑娱乐辞职也使得本作一度被传出停止开发的消息，不过索尼官方都予以了否认。

索尼电脑娱乐总裁[[吉田修平|吉田修平]]表示《食人巨鷹 TRICO》没有取消开发<ref name="吉田修平表示《食人巨鷹 TRICO》从来都没有停止开发">[http://www.3dmgame.com/news/201310/2302807.html 吉田修平表示《食人巨鷹 TRICO》从来都没有停止开发]</ref>，并暗示可能在Playstation 4上推出。<ref name="吉田暗示《食人巨鷹 TRICO》或将登陆PS4">[http://ps3.tgbus.com/zixun/201311/20131115141742.shtml 吉田再次暗示《食人巨鷹 TRICO》或将登陆PS4]</ref>

2014年的E3游戏展期间，面对本作再次被取消的流言和报道，<ref>{{cite web|url=http://www.ign.com/articles/2014/06/08/the-last-guardian-cancelled|title=RETRACTED: CONFLICTING REPORTS ON THE STATE OF THE LAST GUARDIAN|last=Butts|first=Steve|publisher=[[IGN|IGN]]|accessdate=2014-06-18|date=2014-06-07}}</ref><ref>[http://www.ign.com/articles/2014/06/09/the-last-guardian-an-apology THE LAST GUARDIAN: AN APOLOGY]Steve Butts.IGN.2014-06-08.[2014-06-18]</ref> 吉田修平再次表示「如果我们取消了《食人巨鷹 TRICO》的开发，我们会让玩家们知道。」<ref>{{cite web|url=http://www.polygon.com/2014/6/11/5801648/the-last-guardian-canceled-you-will-know|title=If we ever cancel The Last Guardian we'll tell you, Sony's Yoshida promises|last=Crecente |first=Brian |publisher=[[Polygon|Polygon]]|accessdate=2014-06-18|date=2014-06-11}}</ref>

2015年[[E3_2015|E3]]索尼展前发布会上，[[索尼電腦娛樂|索尼電腦娛樂]]（现索尼互动娱乐）正式宣布将在2016年于[[PlayStation_4|PlayStation 4]]平台推出《食人巨鷹 TRICO》，游戏开发工作也交由新成立的游戏工作室genDESIGN负责。2016年[[E3_2016|E3]]索尼展前發表會，[[索尼互动娱乐|索尼互动娱乐]]正式宣布發售日為2016年10月25日；但之后再次延期到12月6日。

== 劇情 ==
無名的少年因為遇難昏迷，醒來時發現自己位於從未見過的未知領域，身旁還有一頭受傷的貌似貓科似犬，但是又長著巨大翅膀的的巨鷹Trico。剛開始他們並不和睦，但經過一次又一次的冒險及考驗慢慢的產生信任及友誼。在某次的戰鬥中，少年意外看到了Trico的記憶，而且隨著一人一獸在未知地區的冒險，他逐漸發現了一切的真相。
