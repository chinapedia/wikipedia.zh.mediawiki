'''最高人民法院国际合作局'''，位于[[北京市|北京市]][[东城区_(北京市)|东城区]][[东交民巷|东交民巷]]27号，是[[中华人民共和国最高人民法院|中华人民共和国最高人民法院]]内设机构。

==沿革==
1988年5月7日，最高人民法院设立最高人民法院外事局，以加强外事工作<ref>{{Cite web |url = http://rmfyb.chinacourt.org/paper/html/2014-11/23/content_90796.htm?div=-1 |title =最高人民法院机构设置简史  |author =  |date =2014-11-23  |publisher = 人民法院报 |language =  |accessdate =  |quote =  }}</ref>。最高人民法院外事局是最高人民法院专门负责国际司法交流与合作的业务部门<ref name=pp>{{Cite web |url = http://www.thepaper.cn/newsDetail_forward_1465095_1 |title = 最高法外事局更名为国际合作局，专责国际司法交流与合作 |author =  |date =2016-05-05  |publisher = 澎湃新闻 |language =  |accessdate =  |quote =  }}</ref>。

2016年，经中央有关主管部门批准，最高人民法院将最高人民法院外事局更名为最高人民法院国际合作局，并且加强了人员配备<ref name=pp/>。

==职能==
2000年12月4日，《最高人民法院机关内设机构及新设事业单位职能》（法发〔2000〕30号）发布，其中规定最高人民法院机关内设机构有外事局，职能是<ref name=zhineng/>：
#负责组织人民法院与外国司法界、国际组织之间的司法交流活动。
#指导各级人民法院的外事工作。
#办理和组织、指导下级人民法院办理国际司法协助事务。
#负责最高人民法院参加外交部与外国司法协助谈判的组织、指导工作。
#编译国外有关资料<ref name=zhineng>{{Cite web |url = http://www.pkulaw.cn/fulltext_form.aspx?Gid=124743 |title =  最高人民法院机关内设机构及新设事业单位职能|author =  |date =  |publisher =北大法宝  |language =  |accessdate = 2017-04-05 |quote =  }}</ref>。

目前国际合作局“主要负责组织人民法院与外国司法界、国际组织之间的司法交流活动；审查、办理国际司法协助案件；负责最高人民法院参与的国际司法协助条约、国际公约的谈判及相关工作；协调、组织最高人民法院同国外相关机构的司法项目合作；指导各级人民法院的外事工作。”<ref>{{Cite web |url = http://www.court.gov.cn/jigou-fayuanjigou-zhineng-25.html |title = 国际合作局 |author =  |date =  |publisher =  中华人民共和国最高人民法院|language =  |accessdate =2017-04-05  |quote =  }}</ref>
