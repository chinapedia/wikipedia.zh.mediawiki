《'''最後的大亨'''》（{{lang-en|'''''The Last Tycoon'''''}}）是一部未完成的小說，作者是[[法蘭西斯·史考特·費茲傑羅|法蘭西斯·史考特·費茲傑羅]]。1941年，在他去世後由評論家、作家兼好友{{link-en|埃德蒙·威森|Edmund Wilson}}以本標題出版。<ref>{{Cite web |url = http://www.nytimes.com/books/00/12/24/specials/fitzgerald-tycoon.html |title = Scott Fitzgerald's Last Novel |author = J. Donald Adams |date = 1941-11-09 |publisher = The New York Times |language = en}}</ref>

該書於1957年改編成電視劇及[[銀色大亨|1976年的同名電影]]，由[[哈羅德·品特|哈羅德·品特]]編寫劇本。[[羅拔·迪尼路|羅拔·迪尼路]]和{{link-en|泰瑞莎·羅素|Theresa Russell}}主演。

1993年，小說的新版以《'''''The Love of the Last Tycoon'''''》的標題出版，由專門研究費茲傑羅的學者{{link-en|Matthew J. Bruccoli|Matthew J. Bruccoli|Matthew Bruccoli}}負責編輯。該版本於1998年在加利福尼亞州的洛杉磯首演舞台劇。2013年，[[HBO|HBO]]宣佈計劃改編作品。HBO取消項目和將版權送給[[索尼影視電視|索尼影視]]，於2016年由{{link-en|Amazon Studios|Amazon Studios}}製作及播出電視劇。

== 劇情總結 ==
根據《[[出版者周刊|出版者周刊]]》所說，《最後的大亨》是「整體上被認為是一本{{link-en|影射小說|Roman à clef}}」，費茲傑羅以歷史上的電影製作人[[歐文·托爾伯格|歐文·托爾伯格]]來塑造他的角色Monroe Stahr。故事內容講述Stahr在荷里活的權力崛起，以及與競爭對手Pat Brady的衝突，其角色的原型是著名電影製片廠老闆[[路易·B·梅耶|路易·B·梅耶]]。

== 主要角色 ==
* Monroe Stahr，荷里活電影製作人
* Bradogue Brady，Stahr的伙伴，也是電影製作人
* Cecelia Brady，Brady的女兒
* Kathleen Moore，Stahr的愛慕對象

== 改編作品 ==
* 1957年：[[約翰·法蘭根海馬|約翰·法蘭根海馬]]執導的電視劇《''{{link-en|Playhouse 90|Playhouse 90}}''》，{{link-en|積·皮連斯|Jack Palance}}飾演Monroe Stahr。
* 1976年：電影《[[銀色大亨|銀色大亨]]》由[[伊力·卡山|伊力·卡山]]執導，英國劇作家[[哈羅德·品特|哈羅德·品特]]編劇。[[羅拔·迪尼路|羅拔·迪尼路]]飾演Monroe Stahr，{{link-en|泰瑞莎·羅素|Theresa Russell}}飾演Cecelia Brady，而[[羅拔·米湛|羅拔·米湛]]及[[積·尼高遜|積·尼高遜]]也有份演出。
* 1998年：1993年小說新版改編的舞台劇。
* 2016年：2013年11月9日，[[HBO|HBO]]宣佈計劃改編《最後的大亨》，{{link-en|比利·雷|Billy Ray (screenwriter)}}編寫劇本。<ref>{{Cite web |url = http://deadline.com/2013/11/billy-ray-adapting-f-scott-fitzgerlds-last-tycoon-as-hbo-drama-series-638733/ |title = Billy Ray Adapting F. Scott Fitzgerald's 'Last Tycoon' As HBO Drama Series |author = Nellie Andreeva  |date = 2013-11-19 |publisher = Deadline |language = en}}</ref>後來HBO取消計劃及宣佈由[[索尼影視電視|索尼影視電視]]負責製作，{{link-en|Amazon Studios|Amazon Studios}}於2014年11月7日接手比利·雷的改編。2015年11月23日，演員[[馬修·波莫|馬修·波莫]]被選中飾演Monroe Stahr，[[莉莉·柯林斯|莉莉·柯林斯]]飾演Cecilia Brady，同時也宣佈雷將會執導試播集<ref>{{Cite web |url = http://www.darkhorizons.com/news/40121/matt-bomer-is-amazon-s-last-tycoon |title = Matt Bomer Is Amazon's "Last Tycoon" |author = Garth Franklin |date = 2015-11-23 |publisher = DarkHorizons.com |language = en}}</ref>，於2016年6月17日播出。<ref>[http://www.imdb.com/title/tt3390892/?ref_=rvi_tt The Last Tycoon]，IMDb</ref>
