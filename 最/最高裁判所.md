'''最高裁判所'''（{{jpn|j=最高裁判所|kn=さいこうさいばんしょ|rm= Saikō saibansho|f=y}}）是[[日本|日本]]的[[最高法院|最高法院]]及最高[[司法機關|司法機關]]，其根据《[[日本国宪法|日本国宪法]]》设立，其组织和运行依照《[[裁判所法|裁判所法]]》。一般在日本被简称为「最高裁」（{{jp|j=最高裁|hg=さいこうさい|rm= Saikōsai}}），有时在日本[[法学|法学]]界也直接称为「最高」。

最高裁判所的辦公大樓位於[[东京都|东京都]][[千代田區|千代田區]][[隼町_(東京都)|隼町]]，由日本[[建筑师|建筑师]][[冈田新一|冈田新一]]设计，並获得[[日本建筑学会奖|日本建筑学会奖]]表揚。

== 沿革 ==
* 1875年－日本[[司法省|司法省]]（[[法務省|法務省]]前身）裁判所改立为[[大审院|大审院]]。
* 1890年－颁布《[[裁判所构成法|裁判所构成法]]》（明治23年法律第6号）。在大审院之下，设立[[控诉院|控诉院]]、[[地方裁判所|地方裁判所]]及各地[[区裁判所|区裁判所]]。
* 1947年－随着《[[日本国宪法|日本国宪法]]》与《[[裁判所法|裁判所法]]》（[[昭和|昭和]]22年法律第59号）的施行，废除大审院，设立最高裁判所。

== 构成及组织 ==
[[File:Background_of_judges_in_superme_court_of_Japan.png|thumb]]
最高裁判所法官由一名[[最高裁判所长官|最高裁判所长官]]与十四名[[最高裁判所判事|最高裁判所判事]]组成。最高裁判所长官由[[内阁_(日本)|内阁]]提名，再由[[天皇_(日本)|天皇]]任命。最高裁判所判事由内阁任命，天皇进行确认。最高裁判所法官的法定退职年龄为70岁（见日本国宪法第79条第5项、[[裁判所法|裁判所法]]第50条）。

每位法官在获得任命后初次举行的[[众议院议员总选举|众议院议员总选举]]中，需要接受[[最高裁判所法官国民审查|最高裁判所法官国民审查]]（简称「国民审查」），此后每经过10年需要接受1次国民审查（见日本国宪法第79条第2项）。在审查中，国民可以在希望罢免的法官名字下画上×。目前，尚无任何法官以此种方式被国民罢免。

日本国宪法明文规定，最高裁判所法官的薪金在其任期内不得减少（见日本国宪法第79条第6项第2句）。然而，这一规定被理解为“在国家公务员中仅仅减少最高法院法官的薪金是违宪的”，在国家财政出现特殊情况时，通过法律修正，对于公务员全体的一般性报酬进行调整，并不会侵犯[[司法权|司法权]]的独立性或者法官的身份保障，因此属于合宪范围。在上述认识的基础上，2002年日本修改了[[裁判官报酬法|裁判官报酬法]]，首次在宪政史上减少了在任法官的薪金标准。

最高法院不仅是日本处于审判事务的最高法院，同时还肩负着领导下级法院的最高[[司法行政|司法行政]]的职能，同时，也有权制定法院诉讼程序和处理司法事务的相关规则（[[最高法院规则|最高法院规则]]）。

最高裁判所的司法行政权及规则制定权，必须通过[[最高裁判所法官会议|最高裁判所法官会议]]的决议而行使。为了协助行使上述权力，最高裁判所内设立了处理总务的[[最高裁判所事务总局|最高裁判所事务总局]]。另外，也设置了培养[[法曹|法曹]]的[[司法研修所|司法研修所]]等附属机构。

=== 审判部门的组成 ===
[[Image:The_Grand_Bench_of_the_Japanese_Supreme_Court.jpg|thumb]]
; [[大法庭|大法庭]]
: 由包括法院长官在内的15名法官组成的合议庭。
; [[小法庭|小法庭]]
: 分别由5名法官组成的3个合议庭。

最高裁判所受理的案件一般均由小法庭进行审理，但是在符合[[裁判所法|裁判所法]]第10条第1号至第3号的条件（即涉及法令是否符合宪法的判断、宪法及其他法令的解释、与此前的最高裁判所判例有差异），或者小法庭的法官意见分为人数相同的数个意见，需要通过大法庭进行审理的情况下，这类案件需要交付大法庭重新进行审理。

== 权限 ==
最高裁判所对于[[上告|上告]]，以及[[诉讼法|诉讼法]]规定的[[抗告|抗告]]案件，具有做出最终判决的权限。

最高裁判所最为重要的职能，就是在上告案件中对法令的解释进行统一，以及对于可能存在违反[[日本国宪法|日本国宪法]]的法令做出最终的宪法判断（[[违宪审查制|违宪审查制]]）（参照日本国宪法第81条）。

另外，最高裁判所对于与司法权有关的事项有权制定规则，还有权对[[司法行政权|司法行政权]]、[[下级法院|下级法院]]法官人选进行提名。

== 最高裁判所的特征 ==
; [[最高裁判所调查官|最高裁判所调查官]]制度
: 最高裁判所内部根据民事、刑事、行政等案件的性质分别设置了处理特定领域案件的法院调查官。调查官负责阅读上告案件的判决卷宗，并向最高裁判所判事作出报告。由于最高裁判所的法官仅有15人，无法处理庞大数量的案件，因此调查官可以大量减轻法官的工作量。另一方面，由于法律对于向最高裁提出上告的案件有一定限制，调查官可以先行甄别部分没有必要由最高法院进行审理的案件，从而迅速将其驳回，因此有种批判的观点指出，最高裁判所的很多审判实际上并非由最高裁判所的裁判官，而是由调查官作出的。

; 判决书中的个别意见
: 最高法院的判决书中往往可以看到与多数派意见不同的法官会写明其个别意见。这类意见包括补充意见、意见和反对意见。

另外，根据《裁判所法》第6条，最高裁判所设于东京都。因此，有一种观点认为，只要该法律不作修正，即使日本发生[[迁都|迁都]]，最高法院也将继续留在东京。

*1947年（昭和22年）9月　临时办公楼搬迁（東京地方裁判所大樓（原民事裁判所大樓）3～4层）
*1949年（昭和24年）10月　原大审院办公楼
*1974年（昭和49年）3月　现办公楼

== 名称 ==
[[File:Plate_of_Saikosaibansho.jpg|thumb]]
通常都使用日本常用漢字「{{lang|ja|最高裁判所}}」來表記，不過在廳舍所揭示的銘板上，書寫的名稱是「{{lang|ja|最髙裁判所}}」，其中的“-{髙}-”引用了[[俗字|俗字]]來表示。

一般简称为“最高裁”，但在從事[[法律工作者|法律工作者]]中也被进一步简略为“最高”。另外，由于办公大楼面对[[三宅坂|三宅坂]]，通常也以“三宅坂”指代。

由于其办公大楼的外观独特，加之最高裁判所的有关组织方针，常常被法律工作者或法学者带有揶揄意味地成为“奇岩城”（意思为“怪石头城堡”）。
