'''最高人民法院刑事审判第一庭'''，位于[[北京市|北京市]][[东城区_(北京市)|东城区]][[北花市大街|北花市大街]]9号，是[[中华人民共和国最高人民法院|中华人民共和国最高人民法院]]内设机构。

==沿革==
1949年11月1日，[[中央人民政府最高人民法院|中央人民政府最高人民法院]]正式办公。1950年1月，[[中央人民政府委员会|中央人民政府委员会]]批准了《最高人民法院试行组织条例》，条例规定最高人民法院设刑事审判庭、民事审判庭和办公厅、督导处、编纂处等单位。按照1954年《[[中华人民共和国人民法院组织法|中华人民共和国人民法院组织法]]》的规定，中华人民共和国最高人民法院设刑事审判庭、民事审判庭和研究室、督导室、顾问室、办公厅等单位。1966年“[[文化大革命|文化大革命]]”爆发，最高人民法院受冲击，机构趋于瘫痪。1976年10月[[粉碎四人帮|粉碎“四人帮”]]后，刑事审判庭、民事审判庭、办公厅等业务、行政单位逐步恢复。到1983年，设立了刑事审判第一庭、刑事审判第二庭、民事审判庭、经济审判庭、研究室、办公厅、人事厅、司法行政厅<ref name=jigoujsh>{{Cite web |url = http://rmfyb.chinacourt.org/paper/html/2014-11/23/content_90796.htm?div=-1 |title =最高人民法院机构设置简史  |author =  |date =2014-11-23  |publisher = 人民法院报 |language =  |accessdate =  |quote =  }}</ref>。

2000年12月4日，《最高人民法院机关内设机构及新设事业单位职能》（法发〔2000〕30号）发布，其中规定最高人民法院机关内设机构有刑事审判第一庭，职能是<ref name=zhineng/>：
#审判危害国家安全罪、危害公共安全罪、侵犯公民人身权利民主权利罪、妨害社会管理秩序罪、危害国防利益罪、军人违反职责罪等第一、二审案件，复核死刑案件。
#审理相关的涉外内审案件。
#审理涉外移管案件。
#审理相关的因特殊情况在法定刑以下判处刑罚的复核案件，审理因特殊情况假释的复核案件。
#处理相关的死刑喊冤案件。
#审查死刑备案材料。
#办理相关大案要案的协调、指导事宜。
#指导全国法院的减刑、假释工作<ref name=zhineng>{{Cite web |url = http://www.pkulaw.cn/fulltext_form.aspx?Gid=124743 |title =  最高人民法院机关内设机构及新设事业单位职能|author =  |date =  |publisher =北大法宝  |language =  |accessdate = 2017-04-05 |quote =  }}</ref>。

2005年6月13日，增设刑事审判第三庭、刑事审判第四庭、刑事审判第五庭，并调整了各刑事审判庭的分工<ref name=jigoujsh/>。

==职能==
目前刑事审判第一庭“主要负责所辖地区普通刑事案件的复核及有关审判指导工作。”<ref>{{Cite web |url = http://www.court.gov.cn/jigou-fayuanjigou-zhineng-6.html |title =刑事审判第一庭  |author =  |date =  |publisher =中华人民共和国最高人民法院  |language =  |accessdate = 2017-04-06 |quote =  }}</ref>

刑事审判第一庭负责的地区是：[[陕西省|陕西省]]、[[甘肃省|甘肃省]]、[[青海省|青海省]]、[[宁夏回族自治区|宁夏回族自治区]]、[[山东省|山东省]]、[[浙江省|浙江省]]、[[安徽省|安徽省]]、[[福建省|福建省]]，负责的专项是：侵害妇女儿童权益类犯罪、环境犯罪、知识产权犯罪<ref>{{Cite web |url = http://www.infzm.com/content/104788/ |title = 死刑复核权上收八年 最高法院如何刀下留人 |author =  |date =2014-10-16  |publisher =  南方周末|language =  |accessdate =  |quote =  }}</ref>。
