《'''最后命令'''》（{{lang-en|The Last Command}}）是一部1928年的美国[[无声电影|无声电影]]，由{{link-en|约瑟夫·冯·斯坦伯格|Josef von Sternberg}}导演，基于{{link-en|拉尤斯·比洛|Lajos Bíró}}的故事改编<ref name="silentera">{{cite web |url=http://www.silentera.com/PSFL/data/L/LastCommand1928.html |title=Silent Era: The Last Command |accessdate=2011-09-24 |work=silentera.com}}</ref>。主演[[埃米爾·傑寧斯|埃米爾·傑寧斯]]获得了第1届[[奥斯卡最佳男主角奖|奥斯卡最佳男主角奖]]，演员阵容中还包括[[威廉·鲍威尔|威廉·鲍威尔]]。
