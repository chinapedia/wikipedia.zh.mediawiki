'''资中县'''是[[中国|中国]][[四川省|四川省]][[内江市|内江市]]所辖的一个[[县|县]]。 因其县城座落在[[沱江|沱江]]中段，而沱江又因[[尧|尧]]的儿子“资”协助[[大禹|大禹]]治理沱江水患而称其为资江（或资水），故名为资中。总面积1734平方公里，2002年人口131万人。辖区内交通便利，[[成渝铁路|成渝铁路]]、成渝公路、[[成渝高速公路|成渝高速公路]]穿越全境。

== 历史 ==
古为巴国属地。
*前135年（[[西汉|西汉]][[建元|建元]]六年），设置资中县，治所在今资阳县。
*561-565年([[北周|北周]][[保定_(宇文邕)|保定 (宇文邕)]]年间)，析置磐石县，县址在今之资中县北门外两路口一带。
*553年（[[西魏|西魏]]废帝三年），设资州，州所在资阳。
*587年（[[隋|隋]][[开皇|开皇]]七年），资州州治由资阳迁到磐石县。辖资阳、盘石、清溪、内江、月山、龙山、银山、丹山八县。
*607年（隋[[大业|大业]]三年），改资州为资阳郡，郡治仍在磐石县。
*618年（[[唐|唐]][[武德|武德]]元年），罢资阳郡，复置资州，州治仍在磐石县。
*[[明代|明代]]废路改府，省州改县，成都路改成都府，资州改为资县，属[[中书省|中书省]]成都府。
*1727年([[清|清]][[雍正|雍正]]五年）又升资县为资州直隶州，辖资阳、内江、井研、仁寿。
*1914年改资州为资中专署，专员公署设在资中，辖资阳、简阳、资中、内江、荣县、井研、仁寿、威远八县。
*1949年12月5日成立资中县人民政府，属资中专区，仍辖资中、内江、荣县等八县。
*1950年改为内江专区，专署迁至内江，资中仍为县制。

== 地理 ==
*资中位于东经104°27′至105°07′，北纬29°34′至30°02′，地处四川盆地“腹心”，位于成渝两地之间、沱江中段。东邻[[内江|内江]][[东兴区|东兴区]]和[[安岳县|安岳县]]，南接[[威远县|威远县]]，西连[[仁寿县|仁寿县]]，北靠[[资阳市|资阳市]]。 
*全县属亚热带湿润气候，春早夏长，年均气温17.4℃，年均降雨量1007.7毫米，境内土地肥沃、物产丰富，是全国商品粮、生猪、水果、特种水产基地。
*资中自古以来就为川中商贸中心和物资集散地。

== 辖区==
辖区1734平方公里。资中县辖31个镇、2个乡（板栗垭、龙山）。 

*政府驻地：水南镇。
*镇：重龙、归德、甘露、鱼溪、铁佛、球溪、龙结、罗泉、发轮、宋家、银山、太平、双河、公民、龙江、双龙、水南、新桥、骝马、高楼、孟塘、陈家、配龙、狮子、金李井、兴隆街、顺河场、苏家湾、明心寺、走马、马鞍、龙山。
*乡:板栗垭。

== 人口 ==
全县总人口131万人，有少数民族27个，人口2351人。其中未识别民族22人。分布在全县33个乡镇，居住较多有公民、水南、归德、球溪、龙结、明心寺等乡镇。少数民族人口较多的有：彝族481人、回族208人、苗族212人、白族207人、藏族218人、傣族213人、其余100人左右。

==经济==
资中工业是以初级产品开采加工等传统产业为主的基础工业，高新技术产业的产值所占比重很小，工业产品的技术含量不高，结构也不尽合理，化工产品和建材产品是资中县的支柱产品，有四川银山化工（集团）股份有限公司和四川省重龙股份有限公司等大中型骨干企业。

主要特产为[[塔罗科血橙|塔罗科血橙]]、[[鲶鱼|鲶鱼]]、[[枇杷|枇杷]]、[[冬尖|冬尖]]。

== 旅游 ==
资中是成渝沿线唯一的省级历史文化名城，旅游资源不仅丰富，而且兼具文化气息。资中县主要景点有： 
*[[重龙山|重龙山]]，位于资中县城的东北隅，一山飞峙，登山一望，江城景致,历历可数。山上有永庆寺、山门石坊及文命诞敷木牌楼、天王殿、八角亭、天池、大雄殿、内城墙、重龙阁、唤鱼池、摩岩造像、君子泉、寿音阁等景点。
*[[白云山|白云山]]，位于资中县城西南20公里10万亩森林区，白云山雄踞于林区内，有108个山峦。山连山，林接林，巍峨苍翠，景色青幽。山上分为八大景区：竹影泉香、深谷闻莺、一路顺风、林海浮鲸、石径摩天、云端观海、潭边照影、翠湖泛舟。
*[[文庙|文庙]]、[[武庙|武庙]]--国家级文物保护单位。文庙是四川省保存最好的两庙之一，始建于北宋雍熙年间，占地7034平方米，建筑面积2191平方米。武庙，始建于明嘉靖年间，全庙面南背北，由朝贡殿、钟楼、鼓楼、大殿、武星殿、三义祠、厢房等组成。
*罗泉镇、铁佛镇是省级历史文化古镇。另外还有[[清代一条街|清代一条街]]、东岳庙、唐明渡双塔、甘露寺、大东门城楼、龙江湖、九龙峡、罗汉洞等景区、景点。[[赵逵|赵逵]]、[[骆成骧|骆成骧]]两位资中状元应了“川东川北多勇士，川西川南多文才”。

== 科教 ==
近年来，资中教育事业得到较快发展。2003年，全县拥有中小学334所，其中省示范高中1所，市示范高中2所，县示范高中2所，一般普通高中6所，省重点职中1所，一般职中3所，单设初中3所，镇乡中心学校29所，完全小学91所；社会力量办高中1所，完全小学2所；村小181所；公办幼儿园12所。全县有中小学教职工9016人，中小学生199015人。2006年高考本科上线2799人。

== 人物 ==
