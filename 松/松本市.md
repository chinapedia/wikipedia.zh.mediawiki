'''松本市'''（{{jpn|j=松本市|hg=まつもとし|rm=Matsumoto shi|f=y}}）是[[日本|日本]][[長野縣|長野縣]]中部[[中信地方|中信地方]]的城市，也是[[長野縣|長野縣]]人口第二多城市。松本市是一座以[[日本國寶|國寶]][[松本城|松本城]]為中心的[[城下町|城下町]]，是[[施行時特例市|施行時特例市]]和國際會議觀光都市。松本市在歷史上多次幸運地免於戰火摧殘，得以保留[[舊開智學校|舊開智學校]]（[[重要文化財|重要文化財]]）等歷史建築。松本市十分重視文化和教育，加上豐富的觀光資源，得以被選入{{link-ja|國際會議觀光都市|国際会議観光都市}}。松本雖然不是縣廳所在地，但卻擁有[[日本銀行|日本銀行]]松本分行、[[松本機場|松本機場]]、信州大學總部、{{link-ja|FM長野|長野エフエム放送}}、[[陸上自衛隊|陸上自衛隊]]{{link-ja|松本駐屯地|松本駐屯地}}等設施，商業銷售額居長野縣內次席，工業生產額則是縣內第三，堪稱長野縣的第二縣府。

==歷史==
[[File:Nawate04s1600.jpg|thumb]]
松本市內的岡田等地發現了3000年前的遺跡<ref>{{cite web|url=https://sitereports.nabunken.go.jp/files/attach/9806/7676_1_%E6%9D%BE%E6%9C%AC%E5%B8%82%E6%A8%AA%E7%94%B0%E3%83%BB%E5%B2%A1%E7%94%B0%E9%81%BA%E8%B7%A1.pdf　|script-title=ja:松本市横田・岡田遺跡|accessdate=2017-05-06|publisher={{lang|ja|奈良文化財研究所}}|language=ja}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。[[彌生時代|彌生時代]]之後，稻作開始普及。松本市也發現了彌生時代中期之後的遺跡<ref name="History"></ref>。[[古墳時代|古墳時代]]的松本是本州東部較早開始繁榮的地區之一，4世紀時的{{link-ja|弘法山古墳|弘法山古墳}}是東日本最古的古墳之一<ref>{{cite web|url=http://youkoso.city.matsumoto.nagano.jp/special1_index-id_60-htm　|script-title=ja:春、桜に彩られる「弘法山古墳」|accessdate=2017-05-06|publisher={{lang|ja|松本市}}|language=ja}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。[[奈良時代|奈良時代]]末期至[[平安時代|平安時代]]初期時，信濃國的國府從上田市搬遷到松本市地區<ref>{{cite web|url=http://www.city.suzaka.nagano.jp/enjoy/gakusyu/library/shinano/　|script-title=ja:信濃の由来|accessdate=2017-05-06|publisher={{lang|ja|須坂市}}|language=ja}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。市內的{{link-ja|兔川寺|兎川寺}}、{{link-ja|放光寺|放光寺 (松本市)}}、{{link-ja|牛伏寺|牛伏寺}}等古刹的歷史均可追溯自飛鳥時代至奈良時代<ref name="History"></ref>。

松本市內的{{link-ja|埴原城|埴原城}}修築於[[鎌倉時代|鎌倉時代]]初期<ref>{{cite web |url=http://takara.city.matsumoto.nagano.jp/prefecture/060.html　|script-title=ja:小笠原氏城跡 |accessdate=2017-05-06|publisher={{lang|ja|松本市}} |language=ja}}</ref>。[[南北朝時代_(日本)|南北朝]]時期，{{link-ja|甲斐源氏|甲斐源氏}}一族的{{link-ja|小笠原貞宗|小笠原貞宗}}成為松本的統治者，並在井川興建城郭<ref>{{cite web |url=http://takara.city.matsumoto.nagano.jp/city/139.html　|script-title=ja:井川城跡 |accessdate=2017-05-06|publisher={{lang|ja|松本市}} |language=ja}}</ref>。松本城的前身深志城也是由小笠原氏家臣島立氏修築。[[戰國時代_(日本)|戰國時代]]，小笠原氏本來是信濃守護。但在[[甲斐國|甲斐國]][[武田信玄|武田信玄]]攻擊信濃之後，小笠原長時被逐出信濃，松本一帶成為武田氏領地。1582年（[[天正_(日本)|天正]]10年）武田氏滅亡之後，小笠原氏恢復了原本的領地。小笠原長時之子[[小笠原貞慶|小笠原貞慶]]在[[德川家康|德川家康]]的幫助下奪回深志城，並將其改名為松本城<ref name="CastleHistory">{{cite web|url=http://www.matsumoto-castle.jp/about/history　|script-title=ja:松本城の歴史|accessdate=2017-05-06|publisher={{lang|ja|松本城}}|language=ja}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

1591年（[[天正_(日本)|天正]]19年），[[石川數正|石川數正]]轉封[[松本藩|松本藩]]，開始著手修建現在的松本城[[天守|天守]]和城下町<ref name="Hyakunen">{{cite book |author={{lang|ja|平岡昭利}}|script-title=ja:《中部 II》 |year=2000 |publisher={{lang|ja|古今書院}} |location=東京都 |isbn=4-7722-5048-4|language=ja}}</ref>{{rp|10}}。戰國時代松本藩的藩主頻繁變更，石川家、小笠原家、戶田家（松平家）、松平家（越前）、堀田家、水野家、戶田家（松平家）都曾擔任過松本藩藩主<ref name="CastleHistory"></ref>。1642年（[[寬永|寬永]]19年）之後，水野氏曾長期執掌松本藩，但在水野氏執政時期曾發生{{link-ja|貞享騒動|貞享騒動}}<ref>{{cite web|url=http://www.maff.go.jp/kanto/nouson/sekkei/kokuei/chushin/rekishi/03_1.html　|script-title=ja:「こぼれ米」と貞享騒動|accessdate=2017-05-06|publisher={{lang|ja|関東農政局}}|language=ja}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。1726年（[[享保|享保]]11年），戶田家（松平家）再次成為松本藩藩主並持續到[[廢藩置縣|廢藩置縣]]<ref>{{cite web |url=http://takara.city.matsumoto.nagano.jp/history/6.html　|script-title=ja:江戸時代 |accessdate=2017-05-06|publisher={{lang|ja|松本市}} |language=ja}}</ref>。

[[File:Former_Kaichi_School.jpg|thumb]]]]
1871年8月29日廢藩置縣之後，松本藩改為松本縣，並在11月時和附近的{{link-ja|伊那縣|伊那県}}、[[高遠藩|高遠縣]]、{{link-ja|諏訪藩|諏訪藩|高島縣}}、{{link-ja|信濃飯田藩|信濃飯田藩|飯田縣}}、{{link-ja|名古屋縣|名古屋県}}中信濃國的部分、{{link-ja|高山縣 (日本)|高山県|高山縣}}（[[飛驒國|飛驒國]]）合併，成立{{link-ja|筑摩縣|筑摩県}}<ref name="Chikuma">{{cite web |url=http://takara.city.matsumoto.nagano.jp/history/7.html　|script-title=ja:明治時代～現代 |accessdate=2017-05-06|publisher={{lang|ja|松本市}} |language=ja}}</ref>。然而在1876年，由於筑摩縣廳被燒毀，筑摩縣被併入長野縣和岐阜縣（飛驒國部分）<ref name="Chikuma"></ref>。 廢藩置縣之後，日本各地不少城郭都被拆除，松本城天守也被出售，面臨被拆除的危機。幸而因當地市民發起保留運動買回天守而得以倖存<ref name="Chikuma"></ref>。1902年，松本首條鐵路開通<ref>{{cite web|url=http://shinonoi-line.nagano.jp/about_line/　|script-title=ja:篠ノ井線概要|accessdate=2017-05-06|publisher={{lang|ja|篠ノ井線}}|language=ja}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。1907年，松本正式設市<ref name="History">{{cite web |url=http://takara.city.matsumoto.nagano.jp/history/　|script-title=ja:松本の歴史 |accessdate=2017-05-06|publisher={{lang|ja|松本市}} |language=ja}}</ref>。二戰時期，松本幸運的逃過大規模空襲。1950年代初期，松本市和鄰近市町村進行了大規模的合併，市轄區面積大幅擴大<ref name="History"></ref>。1960年代，松本市被指定為新產業都市，開通了[[松本機場|松本機場]]，並對松本車站附近地區進行大規模開發<ref name="Hyakunen"></ref>{{rp|10}}，促進了經濟發展。1994年，松本發生[[松本沙林毒氣事件|沙林毒氣事件]]。[[奧姆真理教|奧姆真理教]]信徒使用沙林毒氣對一般民眾進行恐怖攻擊，震驚社會<ref>{{cite web |url=http://www.asahi.com/topics/word/%E6%9D%BE%E6%9C%AC%E3%82%B5%E3%83%AA%E3%83%B3%E4%BA%8B%E4%BB%B6.html　|script-title=ja:松本サリン事件 |accessdate=2017-05-06|publisher={{lang|ja|朝日新聞}} |language=ja}}</ref>。2000年，松本市成為[[特例市|特例市]]。現在松本市正以成為[[中核市|中核市]]為目標<ref>{{cite web|url=https://www.city.matsumoto.nagano.jp/shisei/kikakuseisaku/tyukaku/index.html　|script-title=ja:中核市への移行について|accessdate=2017-05-06|publisher={{lang|ja|松本市}}|language=ja}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

==地理==
[[File:Mount_Hotaka_from_Mount_Yake_2002-8-6.jpg|thumb]]]]
松本市位於長野縣中央偏西處。在經過2005年的[[市町村合併|市町村合併]]之後，松本市市轄區西起[[飛驒山脈|飛驒山脈]]，東至{{link-ja|筑摩山地|筑摩山地}}，是長野縣內面積最大的自治體<ref>{{cite web|url=http://www.gsi.go.jp/KOKUJYOHO/MENCHO/201610/20_nagano.pdf　|script-title=ja:長野県|accessdate=2017-05-06|publisher={{lang|ja|国土地理院}}|language=ja}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。松本市中心位於兩個山脈之間的松本盆地中部。松本市有「岳都」之稱，日本第三高峰[[穗高岳|穗高岳]]和第五高峰[[槍岳|槍岳]]都位於松本市轄區境內。著名觀光地[[美原|美原]]和[[上高地|上高地]]也位於松本市轄區內。松本市地形複雜，市內有眾多[[沖積扇|沖積扇]]，[[糸魚川靜岡構造線|糸魚川靜岡構造線]]亦穿過松本盆地東側<ref>{{cite web |url=http://www.maff.go.jp/kanto/nouson/sekkei/kokuei/chushin/shizen/01_2.html |script-title=ja:松本盆地の複雑な地形 |accessdate=2017-05-06|publisher={{lang|ja|関東農政局}} |language=ja}}</ref>。因松本市內有較多斷層，因此也是地震較多的地區，最近一次較強的地震發生在2011年6月30日，松本市內最大震度達5強<ref>{{cite web |url=http://www.jma.go.jp/jma/press/1106/30a/kaisetsu201106301010.pdf |script-title=ja:平成23年6月30日08時16分頃の長野県中部の地震について |accessdate=2017-05-06|publisher={{lang|ja|気象庁}} |language=ja}}</ref>。松本市主要的河川有[[梓川|梓川]]、{{link-ja|奈良井川|奈良井川}}、{{link-ja|女鳥羽川|女鳥羽川}}等河流<ref>{{cite web |url=http://www.pref.nagano.lg.jp/naraigawa/jigyo/gaiyo/metobagawa.html |script-title=ja:女鳥羽川 |accessdate=2017-05-06|publisher={{lang|ja|長野県}} |language=ja}}</ref>。松本市的森林面積達74,000公頃，森林覆蓋率達81%<ref name="Geography">{{cite web |url=https://www.env.go.jp/air/sensory/jireishu/kk_jirei2009/pdf/03_2_Matsumoto.pdf |script-title=ja:松本（長野県松本市）|accessdate=2017-05-06|publisher={{lang|ja|環境省}} |language=ja}}</ref>。

[[File:150920_Mt_Yake-dake_Kamikochi_Japan01n.jpg|thumb]]
按[[柯本氣候分類法|柯本氣候分類法]]，松本市屬於[[副热带湿润气候|副热带湿润气候]]（Cfa）和{{link-ja|亞寒帶濕潤氣候|亜寒帯湿潤気候}}（Df）。按日本國內的氣候分類，松本市則經常被視為{{link-ja|中央高地式氣候|中央高地式気候}}的典型代表。松本市降水較少，日照時間較長。但山地地區則降水量略多，日照時間較少。安曇、乘鞍地區（上高地等地）的氣候則和松本盆地略有不同，屬於[[日本海側氣候|日本海側氣候]]，則冬季多雪，屬於豪雪地帶<ref>{{cite web|url=https://www.pref.nagano.lg.jp/iju/kensei/soshiki/soshiki/kencho/iju/documents/16nagano.pdf　|script-title=ja:信州移住 Q & A|accessdate=2017-05-06|publisher={{lang|ja|長野県}}|language=ja}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。松本市冬季寒冷，有時氣溫會降到零下10度以下。但由於飛驒山脈遮擋來自日本海雪雲，因此除了安曇地區之外，市內降雪較少。市中心主要降雪多是日本南岸的低氣壓帶來的。夏季的松本市則溫差較大，白天雖氣溫較高但早晚氣候涼爽，梅雨和颱風帶來的降水均較少<ref>{{cite web |url=http://www.jma-net.go.jp/nagano/kikou_tokucyou/nagano_kikou_tokucyou.html　|script-title=ja:長野県の気候 |accessdate=2017-05-06|publisher={{lang|ja|長野地方気象台}} |language=ja}}</ref>。

==人口==
[[File:Matsumoto_201706.jpg|thumb]]週邊的建築]]
1911年時，松本市有人口37,824人<ref name="Population">{{cite web|url=https://www.city.matsumoto.nagano.jp/shisei/tokei/jinkou/suii.files/suii27.xls　|script-title=ja:人口・世帯数の推移|accessdate=2017-05-06|publisher={{lang|ja|松本市}}|language=ja}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。此後至二戰為止，松本市人口穩定增加，1935年時有人口73,353人<ref name="Population"></ref>。二戰時期，松本市的人口增長一度停滯，但戰後再次開始增加。1950年代中期，因市町村合併使得松本市人口快速增加，1955年時增加到145,228人<ref name="Population"></ref>。松本市的人口增長在1980年代之後趨於緩慢。1989年，松本市人口超過20萬大關<ref name="Population"></ref>。2017年5月1日，松本市人口有240,470人<ref>{{cite web|url=https://www.city.matsumoto.nagano.jp/shisei/tokei/jinkou/jinkoh_now.html　|script-title=ja:現在の人口・世帯数|accessdate=2017-05-06|publisher={{lang|ja|松本市}}|language=ja}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。松本市的[[總和生育率|總和生育率]]約有1.5，高於日本平均值但略低於長野縣平均值<ref>{{cite web|url=https://www.city.matsumoto.nagano.jp/shisei/sousei/jinkosuikei-sogosenryaku.files/jinko-suikei.pdf　|script-title=ja:超少子高齢型人口減少社会における松本市の人口推計|accessdate=2017-05-06|publisher={{lang|ja|松本市}}|language=ja}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

==經濟==
2011年度，松本市市內生產總值有9209.6億日元，其中第一產業佔1.1%、第二產業佔19.5%、第三產業佔78.7%<ref name="Data1">{{cite web|url=https://www.city.matsumoto.nagano.jp/shisei/tokei/siryo/26matsumotoshinotoukei.files/26H.pdf　|script-title=ja:市民生活|accessdate=2017-05-06|publisher={{lang|ja|松本市}}|language=ja}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。松本市有農家7,156戶，經營耕地面積4,822公頃<ref name="Data2">{{cite web|url=https://www.city.matsumoto.nagano.jp/shisei/tokei/siryo/sihyo.html　|script-title=ja:松本市の主要指標|accessdate=2017-05-06|publisher={{lang|ja|松本市}}|language=ja}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。長野縣因地形多山而耕地面積低於日本平均值，松本市亦不例外。松本市的農業生產額達178.5億日元，其中水果、蔬菜和稻米的比重最多<ref>{{cite web|url=http://www.machimura.maff.go.jp/machi/contents/20/202/index.html　|script-title=ja:グラフと統計でみる農林水産業|accessdate=2017-05-06|publisher={{lang|ja|農林水産省}}|language=ja}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

[[File:Matsumoto_PARCO_110717.jpg|thumb]]]]
松本的近代工業開始於製絲產業。松本盆地一帶養蠶業發達且地下水資源豐富使得這裡的近代製絲產業得以較早開始發展。松本的首個西洋式機械製絲工廠興建於1873年<ref name="Chishi">{{cite book |author={{lang|ja|斎藤功·石井英也・岩田修二}}|script-title=ja:《日本の地誌6　首都圏II》 |year=2009 |publisher={{lang|ja|朝倉書店}} |location=東京都 |isbn=4-254-16766-0|language=ja}}</ref>{{rp|395}}。1890年代之後，松本的製絲工業迅速發展，成為松本的最重要產業。然而1930年代[[大蕭條|大蕭條]]之後，松本市的製絲產業遭到嚴重打擊<ref name="Chishi"></ref>{{rp|396}}。不過在二戰爆發之後，日本政府將東京和名古屋的部分工廠疏散到松本，帶動松本的工業發展<ref name="Hyakunen"></ref>{{rp|12}}。1960年代松本被指定為新產業都市之後，松本市西南部成為松本主要的工業地區<ref name="Hyakunen"></ref>{{rp|12}}。2013年，松本市的工業品生產額達4601.95億日元，其中以信息產業比重最高，其次是食品產業和電子產業<ref>{{cite web|url=https://www.city.matsumoto.nagano.jp/shisei/tokei/hokoku/genkyou/H25_kougyougenkyou.files/H25_1.xls　|script-title=ja:松本市の工業推移|accessdate=2017-05-06|publisher={{lang|ja|松本市}}|language=ja}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。總部位於松本市的大型製造業企業有{{link-ja|KISSEI藥品工業|キッセイ薬品工業}}<ref>{{cite web|url=http://www.kissei.co.jp/abouts/gaiyou/　|script-title=ja:会社概要|accessdate=2017-05-06|publisher={{lang|ja|キッセイ薬品工業}}|language=ja}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。[[樂蘭公司|樂蘭公司]]、[[富士電機|富士電機]]等企業也在松本市設有工廠。

松本市是長野縣的金融中心。[[日本銀行|日本銀行]]在松本設有其在長野縣唯一的分行<ref>{{cite web|url=https://www.boj.or.jp/about/outline/location/jp_location.htm/　|script-title=ja:本店・支店・国内事務所|accessdate=2017-05-06|publisher={{lang|ja|日本銀行}}|language=ja}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。{{link-ja|長野銀行|長野銀行}}的總部亦位於松本市<ref>{{cite web|url=http://www.naganobank.co.jp/site/aboutus/gaiyou.html　|script-title=ja:当行の概要・役員一覧|accessdate=2017-05-06|publisher={{lang|ja|長野銀行}}|language=ja}}{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。松本市也是中南信地區的商業中心，擁有[[PARCO|PARCO]]等大型商業設施。松本市的批發業和零售業銷售額在長野縣內僅次於[[長野市|長野市]]，位居次席<ref>{{cite web|url=http://www.meti.go.jp/statistics/tyo/syougyo/dms/2009/pdf/ken20.pdf　|script-title=ja:長野県|accessdate=2017-05-06|publisher={{lang|ja|經濟産業省}}|language=ja}}{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。市內主要商業區有站前地區、繩手通等地區。

|fix-attempted=yes }}</ref>。[[松本車站|松本車站]]是松本鐵路交通的中心，運行有開往[[東京|東京]]方向的[[梓號列車|梓號列車]]和開往[[名古屋|名古屋]]方向的[[信濃號列車|信濃號列車]]兩種[[特別急行列車|特急列車]]<ref>{{cite web |language=ja}}</ref>。2015年度，松本車站的總乘客數超過596萬人次，日均客流量達16,348人次<ref>{{cite |fix-attempted=yes }}</ref>。松本市內的巴士則主要由ALPICO交通運營<ref>{{cite ==觀光==
[[File:150920_Gakusawa_wetlands_Kamikochi_Japan01s3.jpg|thumb]]
松本市旅遊資源豐富，是{{link-ja|國際會議觀光都市|国際会議観光都市}}<ref>{{cite web |url=http://www.mlit.go.jp/common/000122077.pdf |script-title=ja:国際会議観光都市（52都市） |accessdate=2017-05-06|publisher={{lang|ja|国土交通省}} |language=ja}}</ref>。2013年度，松本市有超過539萬人觀光客到訪，其中以[[上高地|上高地]]和[[松本城|松本城]]遊客最多<ref>{{cite web |url=https://www.city.matsumoto.nagano.jp/shisei/tokei/siryo/26matsumotoshinotoukei.files/26K.pdf |script-title=ja:商工・観光 |accessdate=2017-05-06|publisher={{lang|ja|国土交通省}} |language=ja}}</ref>。上高地位於飛驒山脈南部的梓川上游地區，是[[中部山岳國立公園|中部山岳國立公園]]的一部分，標高約1,500米。明治時代，英國人傳教士沃爾特·威斯頓在著書中介紹了上高地，使得上高地一躍成為世界著名的山岳度假地和登山勝地<ref>{{cite web |url=http://www.kamikochi.or.jp/about/history/ |script-title=ja:上高地の歴史と年表 |accessdate=2017-05-06|publisher={{lang|ja|上高地}} |language=ja}}</ref>。上高地亦以溫泉而著稱，有眾多溫泉旅館<ref>{{cite web |url=http://www.kamikochi.or.jp/facilities/bathe/ |script-title=ja:外来入浴 |accessdate=2017-05-06|publisher={{lang|ja|上高地}} |language=ja}}</ref>。

[[松本城|松本城]]的天守是日本十二座[[現存天守|現存天守]]之一，也是現存天守城郭中唯一修建在平地上的天守，被指定為[[日本國寶|日本國寶]]<ref>{{cite web |url=http://www.matsumoto-castle.jp/about |script-title=ja:国宝 松本城について |accessdate=2017-05-06|publisher={{lang|ja|松本城}} |language=ja}}</ref>。松本城天守群由大天守、乾小天守、渡櫓、辰巳附櫓、月見櫓五棟建築構成，屬於連結複合式天守，外觀五重六層，已有超過四百年的歷史<ref>{{cite web |url=http://www.matsumoto-castle.jp/about/tower |script-title=ja:天守とその構造 |accessdate=2017-05-06|publisher={{lang|ja|松本城}} |language=ja}}</ref>。松本城在明治維新時雖然免於被拆除，但因長期陷入荒廢狀態，天守甚至一度傾斜。20世紀初期，當地中學校長小林有也發起修繕松本城的運動<ref>{{cite web|url=http://www.matsumoto-castle.jp/value/citizens　|script-title=ja:住民が守った松本城|accessdate=2017-05-06|publisher={{lang|ja|松本城}}|language=ja}}{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。在市民的募捐幫助下，松本城進行了大規模修復工程。1950年代，松本城再次進行大規模修理，得以恢復江戶時代的舊貌<ref>{{cite web|url=http://www.matsumoto-castle.jp/value/national-treasure　|script-title=ja:国宝指定と修理のあゆみ|accessdate=2017-05-06|publisher={{lang|ja|松本城}}|language=ja}}{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。松本市還擁有歷史可追溯至奈良時代的{{link-ja|美原溫泉|美ヶ原温泉}}<ref>{{cite web|url=https://www.mcci.or.jp/www/utsukushi/html/onsen.htm　|script-title=ja:美ヶ原温泉の歴史|accessdate=2017-05-06|publisher={{lang|ja|美ヶ原温泉旅館組合}}|language=ja}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>、{{link-ja|乘鞍高原|乗鞍高原}}<ref>{{cite web|url=http://norikura.gr.jp/product-category/onsen-jp/norikurakogen-onsen-jp/　|script-title=ja:乗鞍高原温泉|accessdate=2017-05-06|publisher={{lang|ja|のりくら観光協会}}|language=ja}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>等度假觀光地。
