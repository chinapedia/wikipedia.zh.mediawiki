'''松楓橋'''為[[臺灣|臺灣]][[臺東縣|臺東縣]]跨越[[瓦剛溪|瓦剛溪]]的[[橋梁|橋梁]]，由[[臺東縣政府|臺東縣政府]][[延平鄉公所|延平鄉公所]]進行管理與維護，其行經道路為[[東36線|東36線]]，作為延平鄉[[紅葉村|紅葉村]]與[[桃源村|桃源村]]對外的重要聯絡橋樑，也是臺東縣境內其中一座景觀橋梁之一。<ref>{{cite web |url=https://www.youtube.com/watch?v=qFc-TJyo0Hc |title=關山鎮導覽 一窺電光橋興建故事 }}</ref>

== 歷史 ==
松楓橋初始建造年代不可考。第一次改建時間在2003年，由於原有橋梁因紅葉村端橋臺位置地質脆弱，[[地下水|地下水]]過盛及橋梁位處[[鹿野溪|鹿野溪]][[攻擊坡|攻擊坡]]，耗時1年，於2004年1月29日完工，改建後為紅色的鋼結構拱橋，全長100公尺。<ref>{{cite web |url=http://www.econst.com.tw/ceci_asp/p09/P09_user_view_desc.asp?NO=562 |title=營建論壇 紅色鋼骨橋「松楓橋」暨東三六線拓寬工程 }}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

2009年（民國98年）08月8日莫拉克颱風來襲，鹿野溪與瓦剛溪河水暴漲，致使南端橋臺傾斜，北端橋臺基礎外露，成為危橋，因此松楓橋封閉不得通行。<ref>{{cite web |url=http://www.lnanews.com/news/%E5%8F%B0%E6%9D%B1%E7%B8%A3%E5%BB%B6%E5%B9%B3%E9%84%89%E6%9D%BE%E6%A5%93%E6%A9%8B%E9%87%8D%E5%BB%BA%E5%B7%A5%E7%A8%8B29%E6%97%A5%E6%AD%A3%E5%BC%8F%E9%80%9A%E8%BB%8A.html |title=台東縣延平鄉松楓橋重建工程29日正式通車 }}</ref><ref>{{cite web |url=http://www.ttypg.gov.tw/circulatedview.php?menu=29&typeid=1763&typeid2=&circulated_id=139%B5%B2%AE%D7 |title=臺東縣政府 延平鄉公所 延平鄉松楓橋通車典禮 }}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

2010年臺東縣政府特別向[[行政院莫拉克颱風災後重建推動委員會|行政院莫拉克颱風災後重建推動委員會]]及[[交通部公路總局|交通部公路總局]]爭取經費1億7仟萬元辦理松楓橋重建工作，是為第二次改建，由精密工程顧問有限公司設計, 重建計劃考量地質、地下水及河谷彎道等因素，橋台及橋墩都選在穩定的位置，並採防衝鋼板保護墩柱，外觀配合地方特色及人文圖騰。<ref>{{cite web |url=https://tw.news.yahoo.com/斥資近2億-松楓橋重建通車-004406489.html |title=Yahoo 新聞 斥資近2億 松楓橋重建通車 |deadurl=yes |archiveurl=https://web.archive.org/web/20160305195006/https://tw.news.yahoo.com/%E6%96%A5%E8%B3%87%E8%BF%912%E5%84%84-%E6%9D%BE%E6%A5%93%E6%A9%8B%E9%87%8D%E5%BB%BA%E9%80%9A%E8%BB%8A-004406489.html |archivedate=2016-03-05 }}</ref>
松楓橋第二次改建工程，工期約400個工作天，並費時2年，於2013年6月29日正式完工通車。<ref>{{cite web |url=http://www.taitung.gov.tw/aborigine/News_Content.aspx?n=E4FA0485B2A5071E&sms=E13057BB37942D3F&s=AAC7B2766D416AE3 |title=臺東縣政府 原住民行政處 松楓橋災修工程復建開工 }}</ref>

== 諸元 ==
松楓橋全長120公尺，橋面寬10公尺，設有雙向車道各一條，並無人行道與機車道配置，屬於造型獨特的對角拋物線鋼樑[[拱橋|拱橋]]。<ref>{{cite web |url=http://m.ltn.com.tw/news/local/paper/692706 |title=自由時報 耗資近兩億元的「松楓橋」昨通車 }}</ref>
