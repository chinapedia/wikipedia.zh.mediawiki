'''松原市'''（{{jpn|j=松原市|hg=まつばらし|rm=Matsubara shi|f=y}}）是位於日本[[大阪府|大阪府]]中部[[大阪市|大阪市]]和[[堺市|堺市]]之間，[[大和川|大和川]]南側的城市。由於緊鄰兩個大城市，也發展成這兩個城市的[[卧城|卧城]]

== 歷史 ==

在令制國時代，現在的松原市屬於[[河內國|河內國]][[丹北郡|丹北郡]]，西元5世紀時，[[反正天皇|反正天皇]]的[[皇居|皇居]]－丹比柴籬宮位於轄內上田地區。<ref name="History">{{Cite web| url=http://www.city.matsubara.osaka.jp/index.cfm/9,188,39,html| title=松田市の歷史| date=2012-12-26| publisher=松原市政府| language=ja| accessdate=2018-06-17}}</ref>

[[南北朝時代_(日本)|南北朝時代]]後，此區域曾先後出現大堀壘、丹下城、松原城、三宅砦、別所城、一津屋城等多座城堡。<ref name="History"/>[[江戶時代|江戶時代]]大部分區域屬於[[館林藩|館林藩]]的領地，另有部分區域屬於{{tsl|ja|丹南藩}}、[[狹山藩|狹山藩]]、[[小田原藩|小田原藩]]、[[狭山藩|狭山藩]]、{{tsl|ja|伯太藩}}及幕府的領地。其中丹南藩的藩廳{{Tsl|ja|丹南陣屋}}設於現在松原市最南端的丹南村，也因此名為「丹南」。

19世紀末[[明治維新|明治維新]]後，這些區域曾先後隸屬大坂裁判所司農局、大阪府司農局、大阪府南司農局、[[河內縣_(日本)|河內縣]]、[[堺縣|堺縣]]、館林縣、丹南縣、伯太縣，直到1872年第一次府縣整併中才全數整併到堺縣，但在1881年隨著堺縣被併入[[大阪府|大阪府]]，此地也成為大阪府的轄區。

1889年日本實施[[町村制|町村制]]，現在的松原市轄區在當時分屬{{tsl|ja|布忍村}}、'''松原村'''、天美村、{{tsl|ja|三宅村 (大阪府中河内郡)|三宅村 (大阪府中河內郡)|三宅村}}、{{tsl|ja|恵我村|惠我村}}、{{Tsl|ja|丹南村}}、{{tsl|ja|北八下村}}共六個行政區劃；1942年松原村改制為'''{{tsl|ja|松原町|松原町}}'''，天美村於1947年改制為{{tsl|ja|天美町}}，松原町又於1955年與布忍村、天美町、三宅村、惠我村[[市町村合併|合併]]為'''松原市'''。而丹南村在1956年原本是與{{tsl|ja|黒山村|黑山村}}和{{tsl|ja|平尾村_(大阪府)|平尾村_(大阪府)|平尾村}}合併為{{tsl|ja|美原町}}，但僅一年後即從美原町中被劃出改併入松原市；而北八下村也在1957年被廢除，其轄區內的大字河合被併入松田市，其餘區域則併入[[堺市|堺市]]。<ref name="History"/><ref>{{Cite web| url=http://www.city.matsubara.osaka.jp/index.cfm/9,192,39,html| title=市のあゆみ| date=2017-07-19| publisher=松原市政府| language=ja| accessdate=2018-06-17}}</ref>

== 交通 ==

[[近畿自動車道|近畿自動車道]]、[[西名阪自動車道|西名阪自動車道]]、[[阪和自動車道|阪和自動車道]]、[[阪神高速14號松原線|阪神高速14號松原線]]四條快速道路路線在轄內的{{tsl|ja|松原ジャンクション|松原系統交流道}}交會。<ref>{{Cite web| url=http://www.city.matsubara.osaka.jp/index.cfm/9,45474,39,html| title=松原市へのアクセス| date=2013-12-27| publisher=松原市政府| language=ja| accessdate=2018-06-17}}</ref>

鐵路方面，[[近畿日本鐵道|近畿日本鐵道]][[南大阪線|南大阪線]]自北向南進入市區，進入市區後又向東彎去，自東側穿出市區。而鐵路沿線以外的區域則仰賴同是{{tsl|ja|近鉄グループ|近鐵集團}}的{{tsl|ja|近鉄バス|近鐵巴士}}所經營的路線客運。而往堺市方向，也有{{Tsl|ja|南海バス|南海巴士}}所經營的路線客運。

== 教育 ==
