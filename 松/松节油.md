'''松节油'''，是[[精油|精油]]的一种，它是一种重要的工业原料。<ref>玉林市人民政府，《让“龙头”带活一方经济---白马农村信用社支持地方龙头企业出口创汇小记》，{{cite web |url=http://www.yulin.gov.cn/info/38431 |title=存档副本 |accessdate=2008-08-30 |deadurl=yes |archiveurl=https://web.archive.org/web/20160313084202/http://yulin.gov.cn/info/38431 |archivedate=2016-03-13 }}</ref>

松节油是通过[[蒸馏|蒸馏]]作用或其它方法从[[松科|松柏科]]植物的[[松脂|松脂]]所提取的液体，主要成分是[[萜烯|萜烯]]。松节油能以任意比例与[[氯仿|氯仿]]、[[乙醚|乙醚]]或[[醋酸|醋酸]]混合，但不溶于水。松节油属于高[[闪点|闪点]]可燃液体<ref>GB 12268-90，《危险货物品名表》</ref>，具有[[挥发|挥发]]性，燃烧时会产生大量浓烟。

刚出厂的松节油为无色、有特异臭味的液体，存储时间过久或暴露在空气中会导致颜色逐渐变黄以及臭味加重。

== 用途 ==
* 工業用途：作為[[溶劑|溶劑]]及[[有機合成|有機合成]]的原料。
* 医学用途：在[[中医学|中医学]]中，松节油可作为皮肤刺激药外用，主要的用途是治疗关节或肌肉酸痛。<ref>国家药典委员会。《中华人民共和国药典·2000年版一部》。北京：化学工业出版社，2000年：164页。ISBN 7-5025-2735-4</ref>歐洲曾作為[[利尿劑|利尿劑]]討論使用。
* 美术用途：作为[[油画|油画]]颜料的[[稀释剂|稀释剂]]。<ref>广西高校网络课程教学资源共享平台，《视觉艺术欣赏基础·术语汇编》，{{cite web |url=http://learn.gxtc.edu.cn/NCourse/sjysxsjc/web4_2.html |title=存档副本 |accessdate=2008-08-30 |deadurl=yes |archiveurl=https://archive.is/20120716194834/http://learn.gxtc.edu.cn/NCourse/sjysxsjc/web4_2.html |archivedate=2012-07-16 }}</ref>
* 其他用途：有醫學文獻紀載，古羅馬婦女喝松節油，使得她們的尿液聞起來像[[紫羅蘭|紫羅蘭]]的氣味，尿液被古羅馬人廣泛應用在洗衣及紡織品加工中；《大英百科全書》 1911年版及美國國家醫學圖書館（NLM）的Toxnet數據庫皆紀載紫羅蘭氣味相關資訊。

== 种类及生产方法 ==
根据原料和制取方式的不同，松节油可分为脂松节油、木松节油、硫酸盐松节油等。但无论是何种原料和方法，松节油的制取过程都是从原料中分离出[[松香|松香]]和松节油的过程。其中，分离出来的固体即为松香，其主要成分为[[树脂酸|树脂酸]]；而分离出来的挥发性液体即为松节油，其主要成分为萜烯。因此，松节油是松香的[[联产品|联产品]]。<ref>玉溪市林业局，《新平县松香产业稳步发展》，{{cite web |url=http://www.yxly.gov.cn/showitem.asp?id=2007083011184247 |title=存档副本 |accessdate=2008-08-29 |deadurl=yes |archiveurl=https://archive.is/20120806005304/http://www.yxly.gov.cn/showitem.asp?id=2007083011184247 |archivedate=2012-08-06 }}</ref>
=== 脂松节油 ===
{{wwv|time=2018-11-09}}
脂松节油是指利用[[蒸馏|蒸馏法]]从松脂中提取出的松节油，脂松节油采用的松脂为[[#採脂|採脂]]所得。由于脂松节油的原料是可以通过对树木进行多次採割获取，有利于充分利用森林资源，因此脂松节油占目前松节油总产量的很大比例。<ref>宋湛谦，《我国松香、松节油行业前景分析》，出自《精细与专用化学品》。2000年，15期。</ref>脂松节油的联产品是脂松香。

[[File:Gemmage1.JPG|thumb]]提取树脂。]]
==== 採脂 ====
[[採脂|採脂]]，即松脂採割，是指通过割开松柏科植物树幹外缘木质部採集[[树脂|树脂]]的方法。

目前，採脂选用的树种主要有[[赤松|赤松]]、[[黑松_(植物)|黑松]]、[[马尾松|马尾松]]、[[云南松|云南松]]、[[思茅松|思茅松]]、[[南亚松|南亚松]]、[[湿地松|湿地松]]、[[长叶松|长叶松]]等。

採脂的方法有两种，即常法採脂和化学採脂法。<ref>陶绍棣。《松脂采割》。南宁：广西科学技术出版社，1989年。ISBN 7-80565-095-0</ref>

常法採脂是指不采用化学药剂，单纯采用普通的採脂刀割开树幹外缘木质部，收取树脂的採脂方法。

化学採脂法是指使用化学採脂刀，在割开树幹外缘木质部的同时，使用化学药剂刺激树木，延长其流脂时间的方法。

化学採脂法根据其采用的药剂不同，又可分为两种，即采用亚硫酸盐低浆废液的中长期化学採脂和采用硫酸软膏的短期强度化学採脂。其中采用硫酸软膏的短期强度化学採脂由于对树木本身具有一定的伤害，一般只适用于两三年内即将砍伐的林木。<ref>国家林业局中南林业调查规划设计院松树增脂剂推广研究中心，《松树增脂剂促生丰产采脂新技术》，{{cite web |url=http://lyzny.hunan.gov.cn/zzj.htm |title=存档副本 |accessdate=2008-08-29 |deadurl=yes |archiveurl=https://web.archive.org/web/20070807142152/http://lyzny.hunan.gov.cn/zzj.htm |archivedate=2007-08-07 }}</ref> <ref>中国林业网，专家在线咨询栏目，http://www.chinaforestry.com.cn/zxzx/view.aspx?id=11068</ref>

==== 分离 ====
分离脂松节油的方法是使用水[[蒸馏|蒸馏法]]，具体的工艺有很多种，大致可分为滴水法与蒸汽法两大类。

滴水法是直接在蒸馏锅用明火加热松脂的一种生产工艺，这种生产工艺由于产品质量难以控制且容易引发火灾，已被淘汰。

现在广泛使用的方法是蒸汽法，具体工艺也有很多种，但总体可分为三个工序，即熔解松脂、净制熔解脂液和 蒸馏净制脂液。<ref>江西省林业厅，《松脂加工方法介绍》，http://www.jxly.gov.cn/lycy/mcjg/jsgy/200808/t20080808_22629.htm</ref>

=== 木松节油 ===
木松节油是指从[[明子|明子]]（即富含树脂物质的松根或树干）中提取而成的松节油。

木松节油的制取工艺有两种，即浸提法和干馏法。<ref>梧州市万德福贸易有限公司，《松香系列商品编码H.S.Code》，http://www.wonderful.com.cn/linkhscode2.html</ref>

浸提法的制取工序大约包括：切碎明子、使用有机溶剂从明子中浸提出树脂和蒸馏分离出木松节油和木松香。

干馏法的制取工序大约包括：将明子放入干馏炉，在隔绝空气的情况下加热使明子分解和从干馏产生的液体产物中分离出松节油。<ref>科学新课程，《木材干馏之二》，http://kx.kpcn.org/source/czhx/TDDZ/55_SR.asp{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

由于相比脂松节油的制取，木松节油的制取对森林资源的利用相对较差，因此目前木松节油生产量在松节油总产量中比例较低。<ref>松香网，《松香分类》，http://www.rosin-china.com/Info/GumRosinManual/RosinKnowledge/45732.htm{{Dead link|date=2019年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref>南京野生植物综合利用研究院，关于马尾松的资料，http://www.co-plant.com/wildplant/txt/09047.txt{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

=== 硫酸盐松节油 ===
硫酸盐松节油是指从木材制浆造纸工业中[[硫酸盐制浆法|硫酸盐制浆法]]产生的废液中提取而成的松节油。值得注意的是：硫酸盐松节油的联产品通常不称为硫酸盐松节油，而是被称为浮油松香。<ref>成都摩尔科技实业有限公司，《配方转让·松香、松节油（1）》，http://www.molkj.com/ProductShow.asp?ID=431{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>
