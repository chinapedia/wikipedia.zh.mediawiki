'''松江省'''，為[[中華民國|中華民國]][[中国东北地区|东北]]的一個省級行政區，是[[東九省|東九省]]的其中之一，簡稱'''松'''，省會[[牡丹江市|牡丹江]]。1945年成立，於1948年解放軍入瀋後撤銷。

== 名稱由來 ==
以東北重要河川[[松花江|松花江]]來命名。

== 地理 ==
本省東部為山嶺地帶，以丘陵山地為主體，[[長白山|長白山]]脈貫穿全境。西部於松花江中流上的大平原，是[[松遼平原|松遼平原]]的一部分，東南邊是[[圖們江|圖們江]]平原，中部有[[牡丹江|牡丹江]]河谷平原。是主要精華區。

重要的湖泊有[[鏡泊湖|鏡泊湖]]、[[長白山天池|長白山天池]]等。

== 管轄範圍 ==
轄境包括九一八事變前吉林省的中部及東南部，相當於今[[吉林省|吉林省]]的[[延邊朝鲜族自治州|延邊朝鲜族自治州]]大部，及[[黑龍江省|黑龍江省]][[牡丹江市|牡丹江市]]大部、松花江以北通河與依蘭以南地區，全省面積為85,273平方公里<ref>傅林祥，鄭寶恆著《中國行政區劃通史 中華民國卷》為84,559.31平方公里。</ref>。南傍[[朝鮮|朝鮮]]，北連[[嫩江省|嫩江省]]、[[合江省|合江省]]，西鄰[[吉林省_(中華民國)|吉林省]]、[[安東省|安東省]]，東接[[蘇聯|蘇聯]]。

== 歷史沿革 ==
自古就是[[滿人|滿人]]與[[朝鮮族|朝鮮族]]活動的地方，南方的[[長白山|長白山]]是滿人的發祥地與聖地。西元6世紀到9世紀中，是[[渤海國|渤海國]]的領土，[[寧安|寧安]]為其[[首都|首都]]。

10世紀[[遼|遼]]興起，曾為遼的領土。12世紀[[女真|女真]]在此建[[金國|金國]]，[[寧安|寧安]]曾為[[首都|首都]]。[[靖康之難|靖康之難]]時被俘虜的[[宋徽宗|宋徽宗]]、[[宋欽宗|宋欽宗]]皆被[[囚禁|囚禁]]於此地。[[元|元]]代屬[[遼陽行中書省|遼陽行中書省]]所轄，[[明|明]]代時歸[[女真|女真]][[部落|部落]]統治，[[清|清]]代時歸[[吉林將軍|吉林將軍]]管轄之地。[[光緒|光緒]]三十三年（西元1907年）設[[吉林省|吉林省]]，歸[[吉林省|吉林省]]。

1945年8月，抗戰結束後，國民政府準備接收東北。8月31日，國民政府明令將[[東三省|東三省]]劃為[[東北九省|東北九省]]，於9月4日任命松江省政府主席，省會暫駐[[哈爾濱市|哈爾濱市]]。

而[[中共|中共]]在[[蘇聯|蘇聯]]的協助下，先行進入東北。苏军委派原满洲国滨江省民政厅长[[谢雨琴|谢雨琴]]留用任省长，抗联将领[[李兆麟|李兆麟]]任副省长兼苏军卫戍副司令，[[张庭阁|张庭阁]]任哈尔滨市市长、[[周维斌|周维斌]]（抗日人士）任市公安局长。1945年10月10日，[[东北局|东北局]]派[[钟子云|钟子云]]（冀热辽区委社会部长）、[[王建中|王建中]]等到达哈尔滨市，成立了以钟子云为书记，李兆麟、王建中、张罗、张观、周维斌等为委员的中共滨江地区工委，管辖地区是哈尔滨市和满洲国[[滨江省|滨江省]]14个县。10月14日，成立由“中共北满临时省委”组织的“吉黑人民秋收自卫队”为基础经过整顿而成的哈尔滨保安总队，王建中任总队长，钟子云兼政委，刘铁南任政治部主任。不久由[[刘子奇|刘子奇]]任总队长，王建中任政委，齐渭川任政治部主任，吕天任参谋长，发展到5个大队。松江省下设：
*哈北地委、专署和军分区设在[[呼兰|呼兰]]。地委书记[[李建平|李建平]]、专员钟声、军分区司令员[[谭友林|谭友林]]；
*哈西地委、专署和军分区设在[[肇东|肇东]]。地委书记[[王建中|王建中]]、专员王效明、军分区司令员王奎先；后哈西转隶给西满分局领导。
*哈东地委、专署和军分区设在[[阿城|阿城]]。地委书记[[陈达|陈达]]、专员[[何延川|何延川]]、[[王景侠|王景侠]]，军分区司令员[[温玉成|温玉成]]，以新四军第二师的2个团的干部架子组建了哈东保安第一、二、三团；
*哈南地委、专署设在[[双城|双城]]。以后又成立了军分区，司令员[[王奎先|王奎先]]。
1945年11月中旬，陈云、张秀山等从长春飞抵哈尔滨，撤消了滨江地区工委，组建了松江省工委和松江军区。张秀山任省工委书记兼军区政委，[[钟子云|钟子云]]任省工委副书记兼军区副政委，[[卢冬生|卢冬生]]/[[聂鹤亭|聂鹤亭]]任军区司令员。

驻哈尔滨的[[苏联红旗第一集团军|苏联红旗第一集团军]]命令共产党的党政军机关部队于1945年11月23日前退出市区，由國民政府接收人員從蘇軍手中接收了[[哈爾濱|哈爾濱]]。[[中共北满分局|中共北满分局]]、松江省工委、省军区撤至[[宾县|宾县]]县城。哈尔滨市保安总队撤到三肇，司令部驻[[肇东县|肇东县]][[昌五镇|昌五镇]]。

1945年12月下旬，国民党空运100名接收官员和300名保安队员至哈尔滨。1946年1月12日，國民政府接收原[[滿州國|滿州國]]濱江省政府，並將濱江省正式改為松江省。由於本省轄區已經被[[中共|中共]]控制，故松江省政府並未能赴省境辦公。1946年4月27日[[蘇聯|蘇聯]]軍隊全數撤離哈爾濱市，各省市政府接收人員等隨苏军逃離哈爾濱撤退至[[海参崴|海参崴]]。松江省政府主席[[關吉玉|關吉玉]]率領省府人員赴瀋陽建立省政府辦事處。

1946年4月14日至25日，松江省第一次人民代表大会在[[宾县|宾县]]召开，抗联将领[[冯仲云|冯仲云]]当选为省政府主席。1946年4月28日，北满分局机关、松江省党政军机关迁入哈尔滨市。 1946年6月，中共松江省工委改为中共松江省委，直属于东北局领导，张秀山任中共松江省委书记。（1948年7月25日由[[张策|张策]]继任省委书记）

1947年6月5日，[[國民政府|國民政府]]核准內政部所擬具之東北新省區方案，規定東北新行政區域，松江省省會設於[[牡丹江市|牡丹江市]]，轄有2市、15縣。

1948年9月[[遼瀋會戰|遼瀋會戰]]開始，11月2日[[解放軍|解放軍]]攻入[[瀋陽|瀋陽]]，中華民國松江省政府行政機構徹底瓦解，前後僅三年。

1949年4月21日，东北行政委员会决定撤销[[合江省|合江省]]，并入松江省，哈尔滨市改为省辖市。1949年5月11日，两省机构正式合并，张策为省委书记，冯仲云为省政府主席，[[李范五|李范五]]、[[李延禄|李延禄]]为省政府副主席。省会哈尔滨市。 

1954年6月19日， 松江省与黑龙江省合并为黑龙江省。

== 人口 ==
人口據民國三十六年（1947年）上半年統計，為453萬5092人，下半年統計為257萬806人。

== 行政區劃 ==
民國三十六年（1947年）6月5日，[[國民政府|國民政府]]公佈「[[東北新省區方案|東北新省區方案]]」，松江省轄有2市、15縣。因省境較小，不設[[行政督察區|行政督察區]]。由於本省因抗戰結束後，全境己被中共[[東北民主聯軍|東北民主聯軍]]控制，故國民政府公佈的行政區劃並未得到執行。新設的牡丹江、延吉2市，在滿州國時期已經存在，沿革過程，詳見[[牡丹江省|牡丹江省]]、[[間島省|間島省]]。
