'''松貂'''（[[學名|學名]]''Martes martes''），又名'''林貂'''，是分佈在[[北歐|北歐]]的一種[[鼬科|鼬科]][[動物|動物]]。

== 特征 ==
牠們約有[[家貓|家貓]]的大小，體長53厘米，尾巴長25厘米。雄貂較雌貂稍大，平均重1.5公斤。毛皮呈淺褐至深褐色，在[[冬天|冬天]]會逐漸變長及淺色。[[喉嚨|喉嚨]]上有奶白色至黃色的圍兜。

== 棲息地 ==
松貂棲息在多草木的地區，會在樹穴或灌木林中築巢。牠們的爪可以半收縮，讓牠們可以生活在樹上。牠們主要在夜間或[[黃昏|黃昏]]活動。牠們的[[耳朵|耳朵]]圓而且靈敏，[[牙齒|牙齒]]峰利可以吃細小的[[哺乳動物|哺乳動物]]、[[鳥類|鳥類]]、[[昆蟲|昆蟲]]、[[青蛙|青蛙]]及[[屍體|屍體]]。牠們也會吃[[草莓|草莓]]、鳥蛋、[[堅果|堅果]]及[[蜂蜜|蜂蜜]]。牠們是地盤性的，會在邊界留下[[糞便|糞便]]。

松貂會咬[[車輛|車輛]]的[[塑膠|塑膠]]部份，目的是要磨利或清潔牙齒。

== 威脅 ==
雖然[[金雕|金雕]]及[[赤狐|赤狐]]有時會掠食松貂，但[[人類|人類]]才是牠們最大的威脅。為毛皮而被獵殺、失去棲息地、人類騷擾及[[中毒|中毒]]等都令牠們的數量大減。在[[英國|英國]]，松貂已受到法例的原全保護。

== 掠食者 ==
松貂令入侵[[英國|英國]]的[[灰松鼠|灰松鼠]]大幅減少。在松貂與灰松鼠共同的分佈地，灰松鼠的數量急速下降。估計灰松鼠較[[歐亞紅松鼠|歐亞紅松鼠]]較多在地面上活動，正好成為松貂的獵物。<ref>{{Cite news|author=Watson, Jeremy|date=2007-12-30|title=Tufty's saviour to the rescue|publisher=Scotland on Sunday|location=Edinburgh}}</ref>

== 壽命 ==
飼養的松貂壽命可達18歲，但在野外的只可以活到8-10歲。牠們要2-3歲才達至性成熟。經過7個月的[[妊娠期|妊娠期]]，幼貂一般會在3月至4月出生。牠們每次會生1-5隻幼貂。幼貂出生時重約30克，到了6月中旬就會離巢生活，到6個月大就可以完全獨立。
