'''松坂屋'''（{{jpn|j=松坂屋|hg=まつざかや|rm=Matsuzakaya}}）是[[日本|日本]]一家[[百貨公司|百貨公司]]。創立於1611年，是日本歷史最悠久的百貨公司。現為{{link-ja|J. Front Retailing|J.フロント リテイリング}}旗下'''{{link-ja|大丸松坂屋百貨店|大丸松坂屋百貨店|株式會社大丸松坂屋百貨店}}'''營運的百貨品牌。

松坂屋總店位於[[名古屋市|名古屋市]][[中區_(名古屋市)|中區]]榮，其他分店包括名古屋市[[中村區|中村區]]、[[豐田市|豐田市]]、[[東京|東京]][[上野|上野]]、[[銀座|銀座]]、[[静岡市|静岡市]]、[[高槻市|高槻市]]等。

而子会社「横濱松坂屋」的店舖位於[[横濱市|横濱市]]，始創於1864年，受金融海嘯衝擊下，也宣布於2008年10月26日[[結業|結業]]。

過去則曾在[[札幌市|札幌市]]、[[岡崎市|岡崎市]]、[[大阪市|大阪市]]、海外的[[香港|香港]]、[[巴黎|巴黎]]和[[洛杉磯|洛杉磯]]設有分店舗，但已結業。

== 概要 ==
松坂屋主力於[[東海地方|東海地方]]，在[[名古屋市|名古屋市]]等[[中京圈|中京圈]]擁有壓倒性地位。代表花為[[嘉德麗雅蘭|嘉德麗雅蘭]]，[[廣告標語|廣告標語]]為「'''結合生活與文化的松坂屋'''」（{{jpn|j=生活と文化を結ぶマツザカヤ}}）。

戰後一段時期曾超越[[三越|三越]]及[[高島屋|高島屋]]，成為日本第一銷售額的百貨店企業。現在[[三菱東京UFJ銀行|三菱東京UFJ銀行]]（舊[[東海銀行|東海銀行]]）的前身之一{{link-ja|伊藤銀行|伊藤銀行}}（曾成為東海銀行及UFJ銀行）、有「名古屋的[[帝國飯店|帝國飯店]]」之稱的{{link-ja|名古屋觀光大飯店|名古屋観光ホテル}}、名古屋[[商工會議所|商工會議所]]有所淵源，為近代名古屋的著名企業。

松坂屋是百貨業界首間採用[[電梯小姐|電梯小姐]]、制服完全洋裝化的百貨。

地方高齡市民多稱松坂屋為「[[伊藤|伊藤]]樣」。

2010年迎接百貨店開業100周年，2011年迎接「伊藤吳服店」創業400周年。

== 商號 ==
由於有時會與[[三重縣|三重縣]]「[[松阪市|松阪市]]」混淆，常誤寫為'''松阪屋'''。另外，其標誌由於酷似{{link-ja|伊勢商人|伊勢商人|松阪商人}}[[三井財閥|三井家]]家紋也是原因之一。

原先商號取自創業者{{link-ja|伊藤祐道|伊藤祐道|伊藤蘭丸祐道}}苗字命名為'''伊藤屋'''。伊藤蘭丸祐道祖先為[[織田信長|織田信長]]小姓。被稱為「伊藤丸」的標誌中間為「井桁」與「藤」，「井」＋「藤」即代表「伊藤」（讀音相同）<ref name="example">『本間之英 有名企業　社名とまーくの秘密』株式会社学習研究所、2008年4月22日。ISBN 978-4-05-403709-0）</ref>。

「松坂屋」源自於1707年伊勢松坂(現松阪市)出身的商人太田利兵衛所創立的[[和服|吳服]]店（今日上野店的前身），創業時以出身地命名為松坂屋。當時的松坂是[[棉花|木綿]]（{{link-ja|松阪木綿|松阪木綿}}）的主要供給地，也是呉服的原料之一。

1767年，伊藤屋收購松坂屋後，因松坂屋已具有知名度，因此江戶的商號持續使用「松坂屋」（相同的案例有源自橫濱的{{link-ja|松屋 (百貨店)|松屋 (百貨店)|松屋}}）。進入大阪時也以同樣理由收購「惠比須屋」。之後再以建立全國連鎖網的理由將全部店名統一為東京使用的「松坂屋」。

=== 松坂屋御家騒動 ===
[[第二次世界大戰|第二次世界大戰]]後，公司共發生兩次[[御家騷動|御家騷動]]。

首次御家騷動發生於1980年，當時社長伊藤鈴三郎遭到其兄16代伊藤次郎左衞門會長與{{link-ja|鈴木正雄|鈴木正雄}}合力逼退下台的宮廷革命。

當時在伊藤鈴三郎體制下採取擴大路線，但札幌松坂屋的失敗與橫濱野澤屋業績不佳等種種影響下，16代次郎佐衛門會長為了讓其子伊藤洋太郎提早成為繼任者而與鈴木正雄合作。

第二次御家騷動一樣有鈴木正雄參與。1985年4月，名古屋松坂屋本店召開臨時董事會，16代伊藤次郎左衞門會長去世前3個月從副社長就任會長的鈴木正雄就任社長，創業家族伊藤洋太郎（17代伊藤次郎左衞門）社長轉任無權的會長，為松坂屋370多年歷史中首次由外人掌權。鈴木本人雖然表示「我不想擔任這項工作。但大家都不願意出任」，但實際上可說是與親信合作下取得實權的政變。之後1991年連任會長，1993年起轉任顧問的鈴木繼續握有實權，直到1997年11月總會屋利益輸送事件爆發後辭職為止

== 松坂屋控股成立至經營整合 ==
2005年至2006年，資本雄厚的{{link-ja|村上基金|村上ファンド}}成為松坂屋大股東，經營權開始動搖。對此，為了建立一個迅速且靈活的管理決策系統，並讓集團各公司責任與盈利能力明確化，決定轉移股權成立「株式會社松坂屋控股」（松坂屋HD），成為[[控股公司|控股公司]]體制。

但在變更體制後不到一年，松坂屋宣布將與[[大丸|大丸]]經營整合。松坂屋控股成立1年後的2007年，共同持股公司「{{link-ja|J. Front Retailing|J.フロント リテイリング}}株式會社」 (JFR) 成立，松坂屋控股併入JFR。

* '''大阪店'''
: 1875年，松坂屋收購第1大區高麗橋1丁目（現[[中央區_(大阪市)|中央區]]{{link-ja|高麗橋|高麗橋}}1丁目）的吳服店「惠比須屋」，於第3大區新町通3丁目（現[[西區_(大阪市)|西區]]新町3丁目）開設「惠比須屋伊藤吳服店」。
: 1909年，「惠比須屋伊藤吳服店」關閉。
: 1923年，南區日本橋筋3丁目（現[[浪速區|浪速區]]{{link-ja|日本橋 (大阪市)|日本橋 (大阪市)|日本橋}}3丁目）開設「松坂屋伊藤吳服店大阪店」。
: 1938年，大阪店增建工事竣工（現[[高島屋|高島屋]]東別館）。
: 1960年代前半，市電廢止改為興建地下鐵，堺筋的市電路線廢止加上附近未開設地下鐵車站等因素，認定大阪店周邊交通環境將惡化，業績無法好轉<ref>松坂屋100年史　松坂屋発行2010年　121ページ</ref>。新大阪店預定為[[大阪城|大阪城]]西北部[[京阪電氣鐵道|京阪]][[天滿橋站|天滿橋站]]地下化後的車站大樓。
: 1964年4月，京阪電鐵、松坂屋、[[竹中工務店|竹中工務店]]3社出資的京阪大樓公司成立，公布京阪天滿橋站的地下4層、地上8層京阪大樓建設計畫<ref>松坂屋100年史　松坂屋発行2010年　121ページ</ref>。
: 1966年10月1日，移至東區京橋2丁目（現中央區天滿橋京町）。位於以「[[造幣局_(日本)|櫻花]]」與{{link-ja|天神祭|天神祭}}聞名的{{link-ja|舊淀川|旧淀川|大川}}旁，規劃可眺望景色的休息室與餐飲店為賣點，並引進結婚場地、海外旅行沙龍、文化展覽空間、體育用品中心等，展現差別化。
: 但由於人口多集中於梅田為中心的「北」，心齋橋、難波為中心的「南」，天滿橋周邊商業地域較偏向辦公商區<ref group="注">因延伸[[淀屋橋站|淀屋橋]]而地下化的天滿橋，在本店入駐時已失去過去的終點站集客能力。</ref>。總面積也不比市內其他主要百貨店<ref group="注">1966年開店時為20564㎡，1967年5月增至25864㎡，1969年12月達30750㎡，當時市內百貨店業界位居第6位</ref>，位於河岸也難以增建<ref>松坂屋70年史　132ページ</ref>。
: 1973年、1976年分別加強時尚部門。1986年起至1988年進行全館改建，整編食品樓層「美食館」。<ref group="注">悠閒挑選獨特商品的方式為全館的營業額作出極大貢獻，之後陸續導入在名古屋店等分店</ref><ref>松坂屋100年史　松坂屋発行2010年　188‐189ページ</ref>。
: 1996年，[[大阪市營地下鐵長堀鶴見綠地線|長堀鶴見綠地線]][[京橋站_(大阪府)|京橋]] - [[心齋橋站|心齋橋]]間開通，人潮流往[[心齋橋|心齋橋]]。
: 1997年至1999年，食品樓層開設「食彩館」。全館五個樓層改建為女性時尚部門，1998年4月8日全新裝修開店<ref>松坂屋100年史　松坂屋発行2010年　255ページ</ref><ref>読売新聞大阪版　朝刊1998年3月11日　10ページ</ref>。親近年輕人的裝修、引進知名租戶與業態轉換<ref group="注">引進[[淳久堂書店|淳久堂書店]]、[[HMV|HMV]]等租戶</ref>等方針未能提升來客數<ref group="注">來客數從1995年度的725萬人滑落至2002年度的670萬人。</ref>。
: 存在已久的松坂屋退出京阪側的風聲再次浮出檯面，時任社長岡田邦彥表示「競爭對手不斷出現，今後商圈將被奪走。是即使投資也不會得到回報的情況」<ref>読売新聞　大阪版　朝刊2003年10月22日9ページ</ref>，正式決定閉店。

: 1979年4月，與[[伊藤洋華堂|伊藤洋華堂]]合資，改稱為「'''York松坂屋'''」（ヨークマツザカヤ）。
: 1994年3月，脫離松坂屋集團，為「'''Robinson's百貨店'''」札幌店。
: 2009年3月26日，全館整修為時尚大樓「SUSUKINO LAFILER」（ススキノラフィラ）。
:伊藤洋華堂入駐地下食品賣場。

* '''山形松坂屋'''
: 1956年（昭和31年）11月，地方商店街出資的「丸久」於山形市七日町2-7-2開業。
: 1971年，與松坂屋業務合作
: 1973年，移至鄰地（山形市七日町2-7-10）。更名為「'''丸久松坂屋'''」。
: 1980年，更名為「'''山形松坂屋'''」。
: 2000年8月，閉店<ref name="日経2003"/>。
: 2002年6/29，東邦企業營運的「NANA BEANS」（ナナビーンズ）開幕<ref>[http://www.yamacomi.com/909.html 山形コミュニティ新聞社]</ref>。

* '''橫濱松坂屋'''
:地址：橫濱市[[中區_(橫濱市)|中區]]弁天通。
: 1864年，「'''野澤屋呉服店'''」創業。
: 1910年11月1日，「'''野澤屋'''」百貨店開業。地址為橫濱市中區伊勢佐木町1-5。
: 1968年，開始與松坂屋共同配送。商品券交換等活動與松坂屋建立起友好關係。
: 1974年（昭和49年），雙方資本合作，更名為「'''野澤松坂屋'''」。
: 受累於橫濱站西口的繁榮與伊勢佐木町的地層下陷，業績年年低迷。接著，以收購訛詐為名的{{link-ja|橫井英樹|横井英樹}}開始收購股票，引起騷動。此時松坂屋以白衣騎士的角色加入股權協防，最終松坂屋成為最大股東。
: 1977年，{{link-ja|松屋 (百貨店)|松屋 (百貨店)|松屋}}橫濱店撤退，野澤屋買下松屋土地建物作為西館，與原店（本館）之間建設連絡天橋。更名為「'''橫濱松坂屋'''」。
: 2008年（平成20年）10月26日，閉店。2010年（平成22年）拆除。
: 2012年（平成24年）2月8日，大丸COM開發營運的「'''Cattleya Plaza伊勢佐木'''」（カトレヤプラザ伊勢佐木）開幕。

== 松坂屋美術館 ==
松坂屋名古屋店南館7樓的[[美術館|美術館]]。1991年，隨著南館開幕<ref>{{Twitter|museum_matsu|松坂屋美術館}}</ref>。美術館並不擁有收藏品<ref>{{Cite web |url= http://mujam.jp/search.php?did=1219292166-322416 |title= 松坂屋美術館 |publisher=ミュージャム |accessdate=2013-09-28}}</ref>，主要為舉辦國內外各類型的展覧會。

另外，同樓層還有松坂屋史料室，舉行松坂屋歷史展覧會。
