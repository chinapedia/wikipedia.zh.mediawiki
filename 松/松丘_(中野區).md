'''松丘'''（{{jpn|j=松が丘|hg=まつがおか|rm=Matsugaoka|f=y}}）是[[東京都|東京都]][[中野區|中野區]]的地名。現行行政地名為松丘一丁目與二丁目。2013年（平成25年）10月1日為止的[[人口|人口]]有6,319人<ref>[http://www.city.tokyo-nakano.lg.jp/dept/101500/d015810_d/fil/chouchou201310.xls 中野区町丁別人口　平成25年（2013年）10月1日現在]</ref>。[[郵遞區號|郵遞區號]]165-0024。

== 地理 ==
位於中野區東北部。東鄰[[上高田|上高田]]、[[新宿區|新宿區]][[西落合_(新宿區)|西落合]]。北與[[江古田|江古田]]之間以[[新青梅街道|新青梅街道]]為界。西與[[沼袋|沼袋]]之間以[[妙正寺川|妙正寺川]]為界。南接[[新井_(中野區)|新井]]。[[中野通|中野通]]貫穿町內。此外，妙正寺川與[[江古田川|江古田川]]在此地匯流。町內多為住宅區。

== 歷史 ==
松丘在明治初期以前稱為片山村。實施住居表示時更名為松丘。

=== 地名由來 ===
實施住居表示時，因[[哲學堂公園|哲學堂公園]]周邊台地過去以[[松|松]]林聞名，因而更名為松丘。現在「片山」可見於一丁目與二丁目交界的路橋（片山橋）。

== 交通 ==
町域東南端有[[西武新宿線|西武新宿線]][[新井藥師前站|新井藥師前站]]。中野通與新青梅街道巴士眾多。

== 設施 ==
*[[哲學堂公園|哲學堂公園]]－北部新青梅街道旁。
