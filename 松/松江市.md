'''松江市'''（{{jpn|j=松江市|hg=まつえし|rm=Matsue shi|f=y}}）是一個位於[[日本|日本]][[島根縣|島根縣]]東部（[[出雲國|出雲地方]]）[[宍道湖|宍道湖]]沿岸的城市，也是島根縣廳的所在地<ref>{{cite web|url=http://www.pref.shimane.lg.jp/annai.html　|script-title=ja:県庁案内|accessdate=2017-04-06|publisher={{lang|ja|島根県}}|language=ja}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。松江市區的中心部分位於連接宍道湖和[[中海|中海]]的{{link-ja|大橋川|大橋川}}兩側。北岸的市區被稱為橋北，開發歷史較早。南岸的橋南則是新市區。兩者之間有四座大橋相連。由於松江市本就地處水岸，並且[[松江城|松江城]]的護城河系統大多得到保存，使得松江市有「{{link-ja|水都|水の都}}」之稱。加上市區的眾多傳統建築和靠近[[出雲大社|出雲大社]]的地理位置，令松江市成為[[山陰地方|山陰地方]]的觀光熱點。松江市也是山陰人口最多的城市，和[[鳥取縣|鳥取縣]]的[[米子市|米子市]]共同構成{{link-ja|中海、宍道湖經濟圈|中海・宍道湖経済圏}}的主要區域<ref>{{cite web |url=http://www.meti.go.jp/press/2016/04/20160421002/20160421002-11.pdf |script-title=ja:島根県松江市 |accessdate=2017-04-06|publisher={{lang|ja|経済産業省}} |language=ja}}</ref>。松江市在2012年成為[[特例市|特例市]]，2018年升格為[[中核市|中核市]]<ref>{{cite web |url=http://www1.city.matsue.shimane.jp/shisei/chuukakushi/chuukakushi/index.data/kihonkeikaku.pdf |script-title=ja:松江市中核市移行基本計画 |accessdate=2017-04-06 |publisher={{lang|ja|松江市}} |language=ja |title=存档副本 |archive-url=https://web.archive.org/web/20170406201450/http://www1.city.matsue.shimane.jp/shisei/chuukakushi/chuukakushi/index.data/kihonkeikaku.pdf |archive-date=2017-04-06 |dead-url=yes }}</ref>。

==歷史==
[[File:Matsue_Castle_in_Japan,_Castle_town_miniature_松江城.jpg|thumb]]

松江市所在的出雲地區是日本文明較早開始發展的地區，而松江市本身也有著悠久的歷史。松江市南郊的意宇平野一帶分佈有眾多古墳，其中山代二子塚是出雲地方最大的前方後方墳<ref name="History">{{cite web |url=https://www.kankou-matsue.jp/omoshiro/rekishi.html|script-title=ja:松江の歴史 |accessdate=2017-04-06|publisher={{lang|ja|松江観光協会}} |language=ja}}</ref>。在意宇平野還發現了出雲國府和出雲[[國分寺|國分寺]]的遺跡，顯示意宇平野自[[彌生時代|彌生時代]]至[[奈良時代|奈良時代]]是出雲國的核心地區<ref>{{cite web |url=https://www.kankou-matsue.jp/about_matsue/rekishi/history/re4-17.html|script-title=ja:出雲国庁跡 |accessdate=2017-04-06|publisher={{lang|ja|松江観光協会}} |language=ja}}</ref>。在松江市區北部的中海·宍道湖一帶也發現了西川津遺跡等彌生時代的遺跡<ref>{{cite web |url=http://www.pref.shimane.lg.jp/life/bunka/bunkazai/tayori/kankoubutu/hakkutupanhu.data/3_nisikawatu.pdf|script-title=ja:西川津遺跡 |accessdate=2017-04-06|publisher={{lang|ja|島根県}} |language=ja}}</ref>。

鎌倉幕府成立之後，{{link-ja|佐佐木義清|佐々木義清}}被任命為出雲國守護，治理包括松江市在內的出雲國。此後[[山名氏|山名氏]]和[[京極氏|京極氏]]先後統治出雲地方。[[應仁之亂|應仁之亂]]之後，京極氏的守護代[[尼子經久|尼子經久]]勢力擴大。14世紀初期時，尼子經久已控制包括松江在內的出雲國全境。經久之孫[[尼子晴久|尼子晴久]]更在14世紀中期時將尼子氏的勢力擴大的山陰大部分地區和[[山陽地方|山陽]]的部分地區<ref name="History"></ref>。然而在此之後，尼子氏和[[毛利氏|毛利氏]]發生衝突並被毛利氏吞併，山陰地方的大部分地區進入毛利氏支配之下。

1600年（[[慶長|慶長]]5年），[[江戶幕府|江戶幕府]]任命[[堀尾吉晴|堀尾吉晴]]出任出雲和[[隱岐|隱岐]]的領主<ref name="History"></ref>。堀尾吉晴最初入駐曾是尼子氏本駐的[[月山富田城|月山富田城]]。然而由於月山富田城是一座山城，生活及交通不便，堀尾吉晴決定將據點搬遷到平地的[[松江城|松江城]]並在1611年（慶長16年）竣工<ref name="Castle">{{cite web |url=http://www.matsue-tourism.or.jp/m_castle/history/index.html|script-title=ja:松江城の歴史 |accessdate=2017-04-06|publisher={{lang|ja|松江城}} |language=ja}}</ref>。堀尾吉晴治理時期不僅修築了松江城，還利用中海和宍道湖之間的大橋川為天然護城河，並確立了松江城下町的城市格局，開啟了松江城下町的歷史。堀尾氏在堀尾吉晴之後只維持了3代就家系斷絕。1634年（寬永11年），{{link-ja|京極忠高|京極忠高}}被轉封為松江藩主，但京極氏統治時期也只維持3年就告斷絕<ref>{{cite web|url=http://www.sanin-chuo.co.jp/shashin/modules/news/article.php?storyid=503067203|script-title=ja:（１０）京極氏の時代・其の一　若狭土手|accessdate=2017-04-06|publisher={{lang|ja|山陰中央新報}}|language=ja|deadurl=yes|archiveurl=https://web.archive.org/web/20170407055952/http://www.sanin-chuo.co.jp/shashin/modules/news/article.php?storyid=503067203|archivedate=2017-04-07}}</ref>。此後{{link-ja|越前松平家|越前松平家}}的{{link-ja|松平直政|松平直政}}被任命為松江藩主。松平家擔任松江的統治者直到[[明治維新|明治維新]]。在松平家的歷代藩主中，以第七代藩主，別號「不昧公」的茶道家{{link-ja|松平治郷|松平治郷}}最為著名<ref>{{cite web |url=http://ww2.sanin-chuo.co.jp/special/ocha/fumaikou.php|script-title=ja:松平不昧公 |accessdate=2017-04-06|publisher={{lang|ja|山陰中央新報}} |language=ja}}</ref>。1871年[[廢藩置縣|廢藩置縣]]之後，松江藩在同年7月改為松江縣，並在11月和廣瀬縣、母里縣及隱岐地方合併為島根縣，松江也進入了新的時代。1889年，松江實施[[市制_(日本制度)|市制]]，松江市正式誕生<ref>{{cite web |url=https://www.city.matsue.shimane.jp/jumin/shisei/kouhou/shihou_matsue/2108/120syunen.htm|script-title=ja:松江市制施行120周年 |accessdate=2017-04-06|publisher={{lang|ja|松江市}} |language=ja}}</ref>。

[[File:Lafcadio_Hearn_portrait.jpg|thumb]]在日本生活的首個城市]]

也是在明治維新時期，[[英國|英國]]作家派屈克·拉夫卡迪奧·赫恩（後入籍日本並改名[[小泉八雲|小泉八雲]]）在1890年來到松江並擔任英語教師，他雖然在松江生活的時間只有一年三個月，卻在松江和他之後的妻子相識，在松江的生活也對他之後的創作有重大影響<ref>{{cite web |url=http://www.hearn-museum-matsue.jp/hearn.html|script-title=ja:小泉八雲の生涯 |accessdate=2017-04-06|publisher={{lang|ja|小泉八雲記念館}} |language=ja}}</ref>。

1912年[[山陰本線|山陰本線]]通車和1914年松江新大橋的開通標誌松江進入近代化的新階段<ref>{{cite web |url=http://www1.pref.shimane.lg.jp/contents/kochokoho/photo/144/05a.html|script-title=ja:島根この100年[明治時代末期から大正時代] |accessdate=2017-04-06|publisher={{lang|ja|島根県}} |language=ja}}</ref>。在[[第二次世界大戰|第二次世界大戰]]期間，松江市未曾遭到空襲，使得松江得以保留江戶時代的城市景觀，成為日後松江重要的觀光資源。但另一方面，為防止在空襲之後建築延燒導致火災發生的情況出現，亦有部分建築為被拆除，其留下的空地之後被用來擴寬道路<ref>{{cite web |url=http://www1.city.matsue.shimane.jp/bunka/matsueshishi/koramubn.data/column46.pdf|script-title=ja:第 46 回 終戦前の松江市
 |accessdate=2017-04-06|publisher={{lang|ja|松江市}} |language=ja}}</ref>。日本投降之後，松江曾發生[[松江騷擾事件|松江騷擾事件]]，島根縣廳在此事件中一度被燒毀<ref>{{cite web |url=http://www.pref.shimane.lg.jp/admin/seisaku/koho/100nen/02.html|script-title=ja:島根この１００年昭和元（１９２６）年〜昭和２０（１９４５）年
 |accessdate=2017-04-06|publisher={{lang|ja|島根県}} |language=ja}}</ref>。

戰後的松江市迎來了新的發展時期。除了戰前的纖維工業之外，機械工業在松江市經濟中所佔比重也明顯提高，1966年[[松下電器|松下電器]]在松江設廠即為例證<ref name="Industry">{{cite web |url=http://www.energia.co.jp/eneso/keizai/research/pdf/MR1403-1.pdf|script-title=ja:島根県を中心とした産業発展の歴史 |accessdate=2017-04-07|publisher={{lang|ja|中国電力}} |language=ja}}</ref>。1964年，松江市所在的中海地區被指定為新產業都市<ref>{{cite web|url=http://www.lib.shimane-u.ac.jp/kiyo/a002/0011/h004.pdf|script-title=ja:日本海沿岸地域の地域開発|accessdate=2017-04-07|publisher={{lang|ja|保母武彦}}|language=ja|deadurl=yes|archiveurl=https://web.archive.org/web/20170407233158/http://www.lib.shimane-u.ac.jp/kiyo/a002/0011/h004.pdf|archivedate=2017-04-07}}</ref>，顯示松江在日本海沿岸地區的國土開發中佔據重要地位。1960年代至1980年代是松江的大規模都市開發時期<ref name="Chishi">{{cite book |author={{lang|ja|森川洋·篠原重則·奥野隆史}}|script-title=ja:《日本の地誌9　中国・四国》 |year=2005 |publisher={{lang|ja|朝倉書店}} |location=東京都 |isbn=4-254-16769-5|language=ja}}</ref>{{rp|176}}。但與此同時，經濟高速發展時期的一些計劃（如宍道湖和中海的围垦工程・淡水化計劃）也帶來了環境問題等負面影響<ref>{{cite web |url=http://www.kisuiiki.jp/old/symposium/intsem2/Intnl_Presen(J)/6.Takayasu_J_.pdf|script-title=ja:中海干拓・淡水化計画の経緯と残された課題 |accessdate=2017-04-06|publisher={{lang|ja|島根大学汽水域研究センター}} |language=ja}}</ref>。2001年，[[山陰自動車道|山陰自動車道]]的開通和[[山陰本線|山陰本線]]高速化標誌著松江與外界的交通進一步改善<ref>{{cite web |url=http://www.pref.shimane.lg.jp/admin/seisaku/koho/100nen/05.html|script-title=ja:島根この１００年平成元（１９８９）年〜平成１３（２００１）年 |accessdate=2017-04-06|publisher={{lang|ja|島根県}} |language=ja}}</ref>。然而另一方面，松江市也和日本其他眾多地方都市同樣面臨市中心空洞化等問題<ref name="Chishi"></ref>{{rp|177}}。

松江市位於島根縣的東北部，面積530.21平方公里，在日本都道府縣廳所在地城市中排名第19位<ref>{{cite web |url=http://uub.jp/rnk/cap_j.html|script-title=ja:都道府県庁所在地の人口・面積・人口密度ランキング |accessdate=2017-04-06|publisher={{lang|ja|都道府県市区町村}} |language=ja}}</ref>。島根縣地形多山，但松江市大部分地區位於山陰地方最大平原之一的{{link-ja|出雲平原|出雲平野}}之上。松江市轄內有[[宍道湖|宍道湖]]和[[中海|中海]]兩個[[潟湖|潟湖]]，兩湖中間有{{link-ja|大橋川|大橋川}}連接。宍道湖和中海分別是日本第七和第五大湖，兩湖及其形成的天然濕地有著豐富的生態環境，被列入濕地公約加以保護<ref>{{cite web |url=http://www.pref.shimane.lg.jp/infra/kankyo/kankyo/shinjiko_nakaumi/ramsar/shinji-ko_and_nakaumi-j.html|script-title=ja:ラムサール条約湿地宍道湖・中海へようこそ |accessdate=2017-04-07|publisher={{lang|ja|島根県}} |language=ja}}</ref>。松江市區位於宍道湖東岸的大橋川兩側。大橋川北側的橋北地區的歷史起源於江戶時代的[[城下町|城下町]]，[[松江城|松江城]]即位於這一地區，保留有較多歷史遺產，是松江市的舊市區。大橋川南側的橋南地區在江戶時代是寺廟集中的寺町，規模在江戶時代相對橋北較小，亦曾扮演防衛據點的角色<ref name="Hyakunen">{{cite book |author={{lang|ja|平岡昭利}}|script-title=ja:《中国・四国 (地図で読む百年)》 |year=1999 |publisher={{lang|ja|古今書院}} |location=東京都 |isbn=4-7722-5030-1|language=ja}}</ref>{{rp|104}}。現在橋南地區擁有[[松江車站_(日本)|松江車站]]，是松江的交通及商業中心，和橋北共同構成了松江市的核心部分。兩者之間自西向東有{{link-ja|宍道湖大橋|宍道湖大橋}}、{{link-ja|大橋 (大橋川)|大橋 (大橋川)|松江大橋}}<ref>{{cite web |url=http://www.ccba.or.jp/archives/pdf/matsue_oohashi.pdf|script-title=ja:松江大橋 |accessdate=2017-04-07|publisher={{lang|ja|中国建設弘済会}} |language=ja}}</ref>、松江新大橋和{{link-ja|國引大橋|くにびき大橋}}四座橋樑連接。市區以東的大橋川上還有結緣大橋（{{lang|ja|縁結び大橋}}）和{{link-ja|中海大橋|中海大橋}}<ref>{{cite web |url=http://www.pref.shimane.lg.jp/industry/norin/seibi/shimane/rekin.data/22.pdf|script-title=ja:中海大橋 |accessdate=2017-04-07|publisher={{lang|ja|島根県}} |language=ja}}</ref>。

===氣候===
松江市属于[[柯本气候分类法|柯本气候分类法]]下的[[副热带湿润气候|副热带湿润气候]]，当地夏季炎熱，冬季凉爽。迄今松江的氣溫最高極值是38.5℃，最低極值是−8.7℃<ref name="Weather">{{cite web |url=http://www1.city.matsue.shimane.jp/bunka/matsueshishi/koramubn.data/column21.pdf|script-title=ja:第 21 回 気象の記録あれこれ |accessdate=2017-04-07|publisher={{lang|ja|松江市}} |language=ja}}</ref>。在日本，松江则被归类为属于[[日本海側氣候|日本海側氣候]]區。但因松江偏南的地理位置，使這裡的氣候融合了冬季多降水的[[北陸地方|北陸地方]]和夏季多降水的[[九州_(日本)|九州]]兩者的氣候特點，全年降水較為充沛，在6月至7月的梅雨時節和9月的秋霖期間尤為明顯。松江市的全年降水日數在[[近畿地方|近畿地方]]以西地區的觀測點中排名第三，僅次於名瀨和鳥取。年平均濕度也在日本全國排名第五<ref name="Chishi"></ref>{{rp|169}}。松江市的年平均降雪量为{{convert|89|cm|in|0}}，少于同为山阴地区的[[鸟取市|鸟取市]]的{{convert|214|cm|in|0}}，並不屬於豪雪地帶。松江市过去最大积雪深度是1971年2月4日的{{convert|100|cm|in|0}}<ref name="Weather"></ref>。

== 人口 ==
2017年2月28日時，松江市有人口204,090人<ref>{{cite web |url=http://ntoukei.city.matsue.shimane.jp/ |script-title=ja:松江市統計情報データベース|accessdate=2017-04-07|publisher={{lang|ja|松江市}} |language=ja}}</ref>。松江市在戰前人口數約有12萬人，戰後至1950年代中期一度快速增加，此後人口增長一度停滯，到1970年代之後恢復增長<ref>{{cite web|url=http://www1.city.matsue.shimane.jp/shisei/toukei/info/h27kokuseichousa_suii.html|script-title=ja:国勢調査における松江市の人口および世帯数の推移|accessdate=2017-04-07|publisher={{lang|ja|松江市}}|language=ja|title=存档副本|archive-url=https://web.archive.org/web/20170407145548/http://www1.city.matsue.shimane.jp/shisei/toukei/info/h27kokuseichousa_suii.html|archive-date=2017-04-07|dead-url=yes}}</ref>。2000年時，松江市人口達到211,564人的峰值，此後人口轉入減少<ref name="Population">{{cite web|url=http://www1.city.matsue.shimane.jp/shisei/keikaku/kenkou/kosodate-shin-seido/kosodatekeikaku.data/kosodate2.pdf|script-title=ja:松江市の現状|accessdate=2017-04-07|publisher={{lang|ja|松江市}}|language=ja|deadurl=yes|archiveurl=https://web.archive.org/web/20170407144439/http://www1.city.matsue.shimane.jp/shisei/keikaku/kenkou/kosodate-shin-seido/kosodatekeikaku.data/kosodate2.pdf|archivedate=2017-04-07}}</ref>。和日本其他地區相同，松江市亦面臨嚴峻的老齡少子化及人口減少的問題，但松江市的出生率略高於日本平均水準<ref name="Population"></ref>。松江市雖有來自島根縣內地區地區的人口流入，但本身也是人口流出地區導致人口呈現社會減少。現在松江市正希望通過提高出生率和改善經濟增加僱傭來提高人口的自然增長並吸引外地人口移居，降低人口減少的速度<ref>{{cite web |url=http://www1.city.matsue.shimane.jp/shisei/keikaku/seisaku/senryaku/senryaku-yoshiki.data/01honbun.pdf |script-title=ja:松江市まち・ひと・しごと創生 ≪人口ビジョン≫ ≪第1次総合戦略≫ |accessdate=2017-04-07|publisher={{lang|ja|松江市}} |language=ja}}</ref>。

== 政治 ==
松江市和島根縣東部的[[安來市|安來市]]、[[雲南市|雲南市]]中舊[[大原郡|大原郡]]區域、[[出雲市|出雲市]]中舊[[平田市|平田市]]區域、[[奧出雲町|奧出雲町]]，[[隱岐群島|隱岐群島]]的[[海士町|海士町]]、[[西之島町|西之島町]]、[[知夫村|知夫村]]、[[隱岐之島町|隱岐之島町]]劃入[[日本眾議院|日本眾議院]]選區中的[[島根縣第1區|島根縣第1區]]。該選舉區長期由細田家經營，是[[自由民主黨_(日本)|自由民主黨]]極為強勢的保守王國。自[[小選舉區制|小選舉區制]]導入以來，自民黨的[[細田博之|細田博之]]連續當選該選區的眾議院議員，從未失手<ref>{{cite web |url=http://www.pref.shimane.lg.jp/admin/commission/senkyo/shimane_senkyo/kiroku/syuugikiroku.html |script-title=ja:衆議院議員選挙の記録（昭和２１年〜平成２6年の投票状況、開票結果） |accessdate=2017-04-07|publisher={{lang|ja|島根県}} |language=ja}}</ref>。松江市的現任市長是{{link-ja|松浦正敬|松浦正敬}}。他在2000年首次當選松江市市長<ref>{{cite web |url=http://www1.city.matsue.shimane.jp/webmayer/profile/ |script-title=ja:市長プロフィール |accessdate=2017-04-07|publisher={{lang|ja|松江市}} |language=ja}}</ref>，此後已經四次連任成功。松江市議會共有市議員34人<ref>{{cite web |url=http://www1.city.matsue.shimane.jp/gikai/giin/ |script-title=ja:議員紹介 |accessdate=2017-04-07|publisher={{lang|ja|松江市議会}} |language=ja}}</ref>。島根縣議會中松江市選舉區則有11名議員<ref>{{cite web |url=http://www.pref.shimane.lg.jp/gikai/gaido/teisu.html|script-title=ja:選挙区と議員定数|accessdate=2017-04-03|publisher={{lang|ja|島根県議会}} |language=ja}}</ref>。

== 經濟 ==
=== 農業 ===
松江市面積中約10%是農地，大多位於市域平原地區中的郊外部分<ref name="Agriculture">{{cite web |url=http://www1.city.matsue.shimane.jp/sangyou/nougyou/noushin/noushinkeikaku.data/nou_keikaku-s1.pdf|script-title=ja:松江市農業の紹介 |accessdate=2017-04-03|publisher={{lang|ja|松江市}} |language=ja}}</ref>。2005年時，松江市總農家數有5,486戶，呈減少趨勢，且農業人口老齡化情況嚴重<ref name="Agriculture"></ref>。松江市的農業生產額約佔島根縣全縣的約一成，最主要的農產品是稻米。松江農業呈現出都市近郊型農業的特點，稻米之外的主要產品多是直接面向市區生產的蔬菜和花卉等農產品。松江市的主要農特產則有西條柿、[[牡丹|牡丹]]等產品<ref>{{cite web |url=http://www1.city.matsue.shimane.jp/sangyou/nougyou/noushin/nougyououen.html|script-title=ja:松江での農業を応援します |accessdate=2017-04-03|publisher={{lang|ja|松江市}} |language=ja}}</ref>。
=== 工業 ===
[[File:SAN-IN_GODO_BANK01st3200.jpg|thumb]]
松江市屬於消費型都市，加上松江多水的地形使得松江缺乏可用來建廠的大片土地，工業並不甚發達。雖然如此，松江市在1960年代和鳥取縣[[米子市|米子市]]、[[境港市|境港市]]及島根縣[[安來市|安來市]]等地一併指定為中海新產業都市，積極招商引資。1960年代之後，松江市新開闢工業園並修建水庫確保工業用水，吸引企業在松江市設廠，使得松江市的工業有一定程度發展。但和鄰近的米子市等地相比，松江郊外的工業企業多為原本位於市區的企業將工廠搬遷至郊外，因此工業生產總量的增長並不突出<ref name="Chishi"></ref>{{rp|169}}。2010年時，松江市工業就業者人口有20,795人，佔全體就業人口的約兩成<ref name="Keikaku">{{cite web |url=http://www1.city.matsue.shimane.jp/machidukuri/toshikeikaku/toshikeikaku_gaiyou/matsueno_toshikeikaku/index.data/1_2hyousi-mokuji-matuesinogaiyou.pdf|script-title=ja:松江の都市計画 |accessdate=2017-04-03|publisher={{lang|ja|松江市}} |language=ja}}</ref>。2012年松江市的製造品出產金額數是1169億日元，在島根縣內排名第三<ref>{{cite web |url=https://pref.shimane-toukei.jp/upload/user/00017479-kxp6Q7.pdf|script-title=ja:工業統計調査結果報告書 |accessdate=2017-04-03|publisher={{lang|ja|島根県}} |language=ja}}</ref>，顯示和附近地區相比，松江市的工業並不具有突出地位。值得一提的是，[[中國電力_(日本)|中國電力]]的[[島根核能發電廠|島根核能發電廠]]位於松江市內，使得松江是日本唯一一個境內有核能發電廠的縣廳所在地城市<ref>{{cite web |url=http://www.energia.co.jp/area/shimane/shimane-h/index.html|script-title=ja:島根原子力発電所 |accessdate=2017-04-03|publisher={{lang|ja|中国電力}} |language=ja}}</ref>。 
=== 服務業 ===
松江市的消費城市和行政城市色彩強烈，是一座以服務業為主的城市。由於松江市位於山陰地方的中部，地理位置適合管轄山陰全境的業務，因此許多企業都將其山陰地方的分公司設在松江市或米子市。總部位於松江市的銀行有[[山陰合同銀行|山陰合同銀行]]和[[島根銀行|島根銀行]]。其中山陰合同銀行的業務覆蓋島根、鳥取兩縣各地，是山陰地方最大的銀行<ref>{{cite web |url=http://www.gogin.co.jp/www/contents/1000000033000/|script-title=ja:会社概要 |accessdate=2017-04-03|publisher={{lang|ja|山陰合同銀行}} |language=ja}}</ref>。島根銀行雖然以島根縣為主要服務地區，但在鳥取縣也有分行<ref>{{cite web |url=http://www.shimagin.co.jp/company/gaiyo.html|script-title=ja:しまぎんの概要 |accessdate=2017-04-03|publisher={{lang|ja|島根銀行}} |language=ja}}</ref>。松江市和其附近地區形成了以松江市為中心的商圈。但由於鄰近的[[米子市|米子市]]位於交通節點，商業集積度較高。加上西側的[[出雲市|出雲市]]也有不少大型店鋪，使得以松江市為中心的商圈規模有限。1990年代以後更由於市中心空洞化的進展，松江市中心出現大型商業設施關門的嚴峻情況<ref name="Chishi"></ref>{{rp|179}}。現在松江市內唯一的[[百貨公司|百貨公司]]是{{link-ja|一畑百貨店|一畑百貨店}}，此外市中心還有{{link-ja|永旺松江購物中心|イオン松江ショッピングセンター}}等商業設施。松江市擁有豐富的觀光資源，旅遊業也在松江經濟中佔據重要地位。1951年，松江市被指定為{{link-ja|國際觀光文化都市|国際観光文化都市}}，是日本繼[[京都市|京都市]]、[[奈良市|奈良市]]之後的第三個國際觀光文化都市<ref>{{cite web |url=http://www.jafit.jp/thesis/pdf/16_07.pdf|script-title=ja:「松江国際文化観光都市建設法」の特徴とその成立過程における住民の意識 |accessdate=2017-04-07|publisher={{lang|ja|日本国際観光学会論文集}} |language=ja}}</ref>。2015年，有超過1000萬人來自日本國內外的遊客在松江觀光，在山陰地方居於前列<ref>{{cite web |url=http://www1.city.matsue.shimane.jp/shisei/kouhou/houdou/2016/03/1022.html|script-title=ja:平成27年松江市観光客入込数・宿泊客数（国内外）集計結果（3月15日） |accessdate=2017-04-07|publisher={{lang|ja|松江市}} |language=ja}}</ref>。近年島根縣政府在松江市的[[島根大學|島根大學]]附近地區設立松江軟體商務園，信息技術產業正成為松江新的產業發展重點<ref>{{cite web |url=http://www.pref.shimane.lg.jp/industry/enterprise/richi/danchi/kougyodanchiichiran/kogyo01.html|script-title=ja:主要工業団地 |accessdate=2017-04-07|publisher={{lang|ja|島根県}} |language=ja}}</ref>。

== 觀光資源 ==
[[File:Kumano_taisha.jpg|thumb]]
松江市有不少人文觀光資源，加上鄰近日本最著名神社之一的[[出雲大社|出雲大社]]，使得松江市成為山陰地方前列的觀光都市。[[松江城|松江城]]及城下町風景是松江市最具代表性的景觀。松江城修建於1607年至1611年，可以說松江城的修建就是松江歷史的開始。[[明治維新|明治維新]]時期，松江城被賣給民間，部分建築被拆除，並且本和日本的眾多城堡一樣面臨天守甚至都將被拆除的窘境。當地有志人士無法坐視這一情況發生，集資買下天守，使得松江城天守得以倖存，成為山陰地方唯一，日本全國12座現存天守之一<ref name="Castle"></ref>。2015年，松江城天守被指定為[[日本國寶|日本國寶]]，是第五座被列入國寶的天守，可見松江城天守極高的價值<ref>{{cite web |url=http://www.pref.shimane.lg.jp/life/bunka/bunkazai/event/kokuhou/|script-title=ja:松江城天守の国宝指定 |accessdate=2017-04-07|publisher={{lang|ja|松江市}} |language=ja}}</ref>。松江城北側{{link-ja|鹽見繩手|塩見縄手}}的{{link-ja|武家屋敷 (鹽見繩手)|武家屋敷 (塩見縄手)|武家屋敷}}、護城河和松江城天守相得益彰，共同構成了松江獨特的水城景觀。現在松江的武家武敷被指定為文化財加以保護<ref>{{cite web |url=http://www.matsue-tourism.or.jp/buke/page02.html|script-title=ja:武家屋敷案内|accessdate=2017-04-07|publisher={{lang|ja|武家屋敷}} |language=ja}}</ref>，鹽見繩手地區則被指定為松江市傳統美觀保存地區，令江戶時代城下町的氣氛保存至今<ref>{{cite web |url=http://www.matsue-tourism.or.jp/buke/page03.html|script-title=ja:塩見縄手のご案内|accessdate=2017-04-07|publisher={{lang|ja|武家屋敷}} |language=ja}}</ref>。松江城的護城河上開行有遊覽船，遊客可在船上飽覽松江風景<ref>{{cite web |url=http://www.matsue-horikawameguri.jp/about/index.html|script-title=ja:堀川めぐりって？ |accessdate=2017-04-07|publisher={{lang|ja|ぐるっと松江堀川めぐり}} |language=ja}}</ref>。[[小泉八雲|小泉八雲]]使得松江成為外國人了解日本文化的窗口城市之一。現在不僅{{link-ja|小泉八雲故居|小泉八雲旧居}}仍然保存完好<ref>{{cite web|url=https://www.kankou-matsue.jp/kankou/kankou-shisetu/m-area/matsue/page3.html|script-title=ja:小泉八雲旧居（ヘルン旧居）|accessdate=2017-04-07|publisher={{lang|ja|松江観光協会}}|language=ja|title=存档副本|archive-url=https://web.archive.org/web/20170408081559/https://www.kankou-matsue.jp/kankou/kankou-shisetu/m-area/matsue/page3.html|archive-date=2017-04-08|dead-url=yes}}</ref>，松江市更開設{{link-ja|小泉八雲紀念館|小泉八雲記念館}}收藏展示與小泉八雲有關的歷史資料<ref>{{cite web |url=http://www.hearn-museum-matsue.jp/about.html#history|script-title=ja:沿革 |accessdate=2017-04-07|publisher={{lang|ja|小泉八雲記念館}} |language=ja}}</ref>。松江市內還擁有諸多頗負盛名的神社，如位於松江城內的{{link-ja|松江神社|松江神社}}、以姻緣著稱的{{link-ja|八重垣神社|八重垣神社}}<ref>{{cite web |url=http://www.shinbutsu.jp/45.html|script-title=ja:[第十四番]八重垣神社 |accessdate=2017-04-07|publisher={{lang|ja|熊野大社}} |language=ja}}</ref>、{{link-ja|美保神社|美保神社}}等<ref>{{cite web |url=http://www.mihojinja.or.jp/yuisho/|script-title=ja:ご祭神・ご由緒 |accessdate=2017-04-07|publisher={{lang|ja|美保神社}} |language=ja}}</ref>。

松江市不僅人文觀光資源豐富，自然觀光資源亦十分充沛。[[宍道湖|宍道湖]]和[[中海|中海]]是日本最大的汽水水域，水產資源亦別具特色，出產以[[鰻魚|鰻魚]]等食材為代表的「{{link-ja|宍道湖七珍|宍道湖七珍}}」<ref>{{cite web |url=http://furusato.sanin.jp/p/area/matsue/69/|script-title=ja:宍道湖七珍 |accessdate=2017-04-07|publisher={{lang|ja|神々のふるさと山陰}} |language=ja}}</ref>和以[[黃鰭刺鰕虎魚|黃鰭刺鰕虎魚]]等食材為代表的「{{link-ja|中海七珍|中海七珍}}」<ref>{{cite web |url=http://furusato.sanin.jp/p/8/16/|script-title=ja:あなたならどっち？　宍道湖VS中海　七珍MAP |accessdate=2017-04-07|publisher={{lang|ja|神々のふるさと山陰}} |language=ja}}</ref>。中海內[[大根島|大根島]]上的{{link-ja|由志園|由志園}}是島根縣內頗為著名的日本庭園。宍道湖還是觀賞夕陽的聖地<ref>{{cite web |url=http://furusato.sanin.jp/p/area/matsue/1/|script-title=ja:宍道湖 |accessdate=2017-04-07|publisher={{lang|ja|神々のふるさと山陰}} |language=ja}}</ref>。松江市溫泉資源豐沛，其中以[[玉造溫泉|玉造溫泉]]和宍道湖溫泉最為著名。玉造溫泉位於松江市區南部，在《[[出雲國風土記|出雲國風土記]]》中就有記載，歷史極為悠久<ref>{{cite web|url=http://tamayado.com/yumeguri/|script-title=ja:姫神さまの湯めぐり|accessdate=2017-04-07|publisher={{lang|ja|玉造温泉旅館協同組合}}|language=ja|title=存档副本|archive-url=https://web.archive.org/web/20170613125041/http://tamayado.com/yumeguri/|archive-date=2017-06-13|dead-url=yes}}</ref>，泉質含有多種礦物質，被認為有美肌效果<ref>{{cite web|url=http://tamayado.com/yumeguri/|script-title=ja:姫神さまの湯めぐり|accessdate=2017-04-07|publisher={{lang|ja|玉造温泉旅館協同組合}}|language=ja|title=存档副本|archive-url=https://web.archive.org/web/20170613125041/http://tamayado.com/yumeguri/|archive-date=2017-06-13|dead-url=yes}}</ref>。宍道湖溫泉地處宍道湖北岸，湧出量豐富且同樣具有多種療效，因其便利的交通而成為松江市民和觀光客的休閒名所<ref>{{cite web|url=http://www.matsue-yado.com/spa/02about/index.html|script-title=ja:松江しんじ湖について|accessdate=2017-04-07|publisher={{lang|ja|松江しんじ湖温泉組合}}|language=ja|title=存档副本|archive-url=https://web.archive.org/web/20170411143952/http://www.matsue-yado.com/spa/02about/index.html|archive-date=2017-04-11|dead-url=yes}}</ref>。
