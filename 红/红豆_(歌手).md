'''紅豆'''（原名：王立勇 {{bd|1970年|9月20日|catIdx=W王}}），中國大陆歌手，電影演員。1987年因參加“群星杯”歌唱比赛獲得亞軍而出道。<ref name="hongdou">{{Cite web|url=http://ent.people.com.cn/GB/1085/5872018.html|title=从跌倒坑里站起来有多难 红豆推单曲重返歌坛|author=[[北京青年報|北京青年報]]|date=2007-06-15|publisher=[[人民網|人民網]]|language=zh-hans|accessdate={{CURRENTYEAR}}-{{CURRENTMONTH}}-{{CURRENTDAY2}}查閱 }}</ref> 2002年，因猥褻多名男童而震驚全國。<ref name="wangfei">{{Cite web|url=http://www.southcn.com/ent/celeb/fayewong/news/200310140684.htm|title=王菲初恋男友红豆涉猥亵男童被捕|author=[[南方網|南方網]]|date=2007-06-15|publisher=[[南方網|南方網]]|language=zh-hans|accessdate={{CURRENTYEAR}}-{{CURRENTMONTH}}-{{CURRENTDAY2}}查閱 }}</ref> 2004年5月，紅豆因表現良好獲得[[假釋|假釋]]。2011年6月14日，紅豆再度因為猥褻男童，被[[內蒙古|內蒙古]][[包頭|包頭]]警方逮捕。<ref>{{Cite web|url=http://ent.163.com/11/0618/07/76QHOBR500034JAU.html|title=红豆猥亵男童案下周将公布 关押两天精神显疲惫|author=华西都市报|date=2011-06-18|publisher=[[網易|網易]]|language=zh-hans|accessdate={{CURRENTYEAR}}-{{CURRENTMONTH}}-{{CURRENTDAY2}}查閱 }}</ref>

== 紅豆簡歷 ==
1987年，紅豆在獲得由中國[[文化部|文化部]]組織的“群星杯”歌唱比賽亞軍之後，進入中國娛樂圈。紅豆最初的作品為[[電影|電影]]，他在1989年拍攝了兩部電影《煙雨情》（1993年上映）和《西行囚徒》，以及一部音樂片《好大風》。

1990年，紅豆與[[台灣|台灣]]飛揚唱片公司簽約，并發行了他的第一張個人專輯《少年個體戶》，隨後便赴[[香港|香港]]、[[泰國|泰國]]等地學習流行音樂的製作與演唱。

1992年，紅豆參加由中國[[中央電視臺|中央電視臺]]舉辦的“五洲杯”歌手大獎賽并獲得亞軍。

1994年，紅豆與香港飛圖唱片簽約并於1995年發行唱片《紅豆》，並因此獲得[[TVB|TVB]][[十大金曲|十大金曲]]1995年第一季度最佳新人獎。

1996年，紅豆與[[日本|日本]]AMUSE公司簽約并擔任AMUSE[[北京|北京]]分公司的藝術總監，並在同年參與製作了電視劇《起步停車》主題曲。同年發行同名專輯。

1997年至2001年期間，紅豆鮮有個人作品面世。

2001年，紅豆參與電影《閃靈 兇猛》的演出。

2002年，紅豆因猥褻兒童被判入獄，2004年獲[[假釋|假釋]]，最終於2005年刑滿釋放。<ref name="hongdou"></ref>

2011年6月，红豆再次因猥亵男童被捕。<ref name="hongdou2">{{Cite web|url=http://yule.sohu.com/20110616/n310365784.shtml|title=红豆再次猥亵男童落网 被捕时两男童陪伴身旁|author=[[搜狐网|搜狐网]]|date=2011-06-16|language=zh-hans|accessdate={{CURRENTYEAR}}-{{CURRENTMONTH}}-{{CURRENTDAY2}}查閱}}</ref>

== 紅豆猥褻兒童案 ==
=== 第一次 ===
* 1998年 - 1999年：紅豆在北京某舞蹈藝術學校進行舞蹈指導的時候，便開始對該校男童有猥褻情節並引發所在學校老師的不滿。<ref name="school">{{Cite web |url= http://www.southcn.com/ent/hotspot/hongdou/200206240917.htm |title= “红豆事件”猥亵男生掺杂金钱交易 |author= 王佳青，沈沉 |date=2002-01-30 |publisher= 南方網 |language= zh-hans |accessdate={{CURRENTYEAR}}-{{CURRENTMONTH}}-{{CURRENTDAY2}}查閱 }}</ref>

* 2000年1月 - 2001年6月：紅豆並沒有因校方的抗議和老師的抵制而有所收斂，反而明目張膽地將男童帶往酒店等地，並牽扯金錢交易。<ref name="school"></ref>

* 2001年9月8日：校方向[[北京|北京]][[朝陽區|朝陽區]]警方報案。<ref name="school"></ref>

* 2001年12月17日：紅豆被北京警方抓獲。<ref name="school"></ref>

* 2001年12月27日：北京檢察院正式批捕紅豆。<ref name="school"></ref>

* 2002年6月20日：紅豆猥褻男童案在北京朝陽區法院正式審理，並當庭宣判有期徒刑3年零6個月。<ref>{{Cite web |url= http://www.southcn.com/ent/hotspot/hongdou/200206240925.htm |title= 红豆猥亵定罪被判入狱三年半 |author= 张文伯  |date=2002-06-21 |publisher= 南方網 |language= zh-hans |accessdate={{CURRENTYEAR}}-{{CURRENTMONTH}}-{{CURRENTDAY2}}查閱 }}</ref> 全案終結。
