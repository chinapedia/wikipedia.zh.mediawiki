'''红木'''（'''Achiote'''，[[学名|学名]]：{{lang|la|''Bixa orellana''}}）又稱'''胭脂樹'''、'''臙脂樹'''、'''紅色樹'''，是热带[[红木科|红木科]][[红木属|红木属]]唯一一种的植物。乔木，株高达9米，花粉红色，花径5厘米，叶稍卵形，长8—18厘米，蒴果褐色，长5厘米。可生产浅红色或黄色粉末状染料[[胭脂树红|胭脂树红]]，用于黄油、乳酪及人造黄油的着色。在热带及亚热带地区广为种植。

== 形态 ==
常绿[[灌木|灌木]]或小[[乔木|乔木]]；小枝和花序有腺毛；卵形叶子基出五脉；秋季开粉红色花，圆锥花序顶生；卵形或近似球形、绿色或紫红色的[[蒴果|蒴果]]，密生长刺，两瓣裂；种子红色，可在烹調時添加進食物，為食物加添橙黃色的顏色，如[[莱斯特芝士|莱斯特芝士]]。

==用途==
具有利尿劑、抗氧化劑、抗發炎、抗菌劑以及祛痰劑等。幫助保護肝臟及腎臟，可以降低血糖值。對於消化不良、發燒、咳嗽、燒傷、皮膚問題以及體重流失等疾病很有幫助。

== 分布 ==
原产于热带美洲。分布在[[台湾|台湾]]以及[[中国大陆|中国大陆]]的[[广东|广东]]、[[云南|云南]]等地，目前已由人工引种栽培。

== 别名 ==
胭脂木（云南金平）
