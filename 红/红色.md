'''紅色'''，是以通過能量來激發觀察者的[[可見光|可見光]]譜中[[長波|長波]]末端的[[顏色|顏色]]，[[波長|波長]]大約為630到750[[納米|納米]]，類似於新鲜[[血液|血液]]的顏色，是[[三原色|三原色]]和[[心理原色|心理原色]]之一。普通人是無法看到波長長過紅色的射線，而這類射線一般被稱為[[紅外線|紅外線]]。

== 標志 ==
在許多[[國家|國家]]和一些[[民族|民族]]中，紅色有驅逐邪惡的功能。

由於紅色容易引起人們的注意，因此許多警告標記都用紅色的文字或圖像來表示。例如在[[紅綠燈|紅綠燈]]中紅色表示停止，而禁止事項和危險事項也以紅色標示。在[[足球|足球]]比賽中，裁判亮出紅牌代表嚴重犯規，犯規球員要立刻離場。

[[消防|消防]]項目與物資同樣以紅色表示；但由20世紀80年代起推出的[[緊急出口|緊急出口]]標記用色則由此前的紅色普遍換成[[綠色|綠色]]以防止與火種顏色混淆（但部份國家或地區仍然使用紅色，但在顏色象徵意義下卻有機會被部份人員視為禁止使用的緊急出口，因而造成解讀混亂），惟在汽車玻璃和建築物玻璃等上仍以紅色標示[[緊急出口|緊急出口]]標記以防止與自然背景的[[綠色|綠色]]、[[藍色|藍色]]等混淆。

== 政治 ==
紅色在政治上經常用來象徵[[革命|革命]]、[[左派|左派]]、[[社会主义|社会主义]]以及[[共產主義|共產主義]]。所以[[中华人民共和国|中国]][[公务员|公务员]]阶层及[[中國共產黨|中國共產黨]]幹部被称为“'''[[红领|红领]]阶层'''”，与左派革命及有关歌颂左派政党及政权的歌曲被称为'''[[红歌|红歌]]'''，因而在以下各国/地区的主要[[政黨|政黨]]中，[[共产党|共产党]]、[[社會民主黨|社會民主黨]]及其他传统[[左翼|左翼]][[社會民主主義|社會民主主義]]政党大多数使用'''红色'''或[[粉紅色|粉紅色]]为标志，惟少數的[[右派|右派]][[保守主義|保守主義]]政黨會使用'''紅色'''，如[[美國|美國]][[共和黨_(美國)|共和黨]]。

== 中國文化 ==
[[中國|中國]]古代，很多宮殿和廟宇的牆壁都是紅色的。

在中國的傳統文化中，五行中的[[火|火]]，所對應的顏色就是紅色，而[[八卦|八卦]]中的離卦也象徵紅色。在中國傳統文化中是四象之一[[朱雀|朱雀]]，代表的顏色是紅色。紅色也代表[[吉祥|吉祥]]的意義。

在中國[[神明誕辰|神明誕辰]]慶典，以紅色來裝飾，表示慶典之日。

在中國紅色傳統上表示[[喜慶|喜慶]]，比如在[[婚禮|婚禮]]上和[[春節|春節]]都喜歡用紅色來裝飾。

中国传统上用红色的墨水记录亏损，因此又称为“赤字”；古代[[皇帝|皇帝]]批閱[[奏摺|奏摺]]使用紅色的硃砂筆書寫，但皇帝駕崩治喪期間則改用藍色字書寫。

红色在[[京剧|京剧]]脸谱中代表忠贞、英勇的角色，如[[关羽|关羽]]。

== 其他 ==
* 紅色是[[聖誕節|聖誕節]]常用的一種顏色；
* 紅色表示[[愛|愛]]，[[勇氣|勇氣]]，[[熱情|熱情]]，[[熱血|熱血]]，[[危險|危險]]，[[男子氣概|男子氣概]]的顏色；
* 北美的股票市場，紅色表示股價的下跌；在[[東亞|東亞]]的市場，紅色表示[[股價|股價]]上升；
* [[消防車|消防車]]又稱救火車，是專門用作[[滅火|滅火]]或其他緊急用途的救災[[車輛|車輛]]大都以紅色車輛為代表色
* 自古以來，以紅色字書寫信函表示向對方嚴重侮辱，如[[收件人|收件人]]是[[寄件者|寄件者]]的相識，則更帶有絕交之意，略有[[魔法|魔法]]的意味；
* 多支著名足球俱樂部會使用紅色為球隊主色，包括[[英格蘭|英格蘭]]的[[利物浦足球會|利物浦]]及[[曼聯|曼聯]]，兩者因球隊主色而稱呼為「紅軍」及「紅魔」
* 學生的作業中，老師大多用紅筆批改。
* 按照阿濃《點心集》所說，在香港一些舊式學校當一群老師簽名報到時，校長時間到了會用紅筆劃線，遲到者只能簽在紅線以下。
* [[酸性|酸性]][[物質|物質]]用[[酸鹼指示劑|酸鹼指示劑]]變色後，為紅色。
* 红色在[[电阻|电阻]]值中代表2。
* 紅色在[[超級戰隊系列|超級戰隊系列]]中總是作品中第一男主角。近年有部分[[動畫|動畫]]作品都會用紅色為主角。
* 在決鬥時兩方往往會分红色和[[藍色|藍色]]，例如[[拳擊|拳擊]]、[[泰拳|泰拳]]和[[跆拳道|跆拳道]]。
* 紅色是戰爭的顏色，火星是紅色的星球，以[[羅馬|羅馬]]戰神瑪爾斯的名字為名；
* 在[[MediaWiki|MediaWiki]]軟體裡，紅色代表尚未撰寫的[[條目|條目]]，[[藍色|藍色]]代表已經被創建及編輯過可瀏覽內容的條目。
* 在战略游戏中，红色通常用来表示敌军，相对而言[[绿色|绿色]]或[[蓝色|蓝色]]代表友军。
*  在[[性別角色|性別角色]]裡，古代西方紅色或[[粉紅色|粉紅色]]一般是[[男性|男性]]的颜色，[[女性|女性]]則是[[藍色|藍色]]或[[粉藍色|粉藍色]]，東方[[紅色|紅色]]或[[白色|白色]]一般是[[男性|男性]]的颜色，[[女性|女性]]則是[[綠色|綠色]]或[[黃色|黃色]]，現代則一般多以[[藍色|藍色]]作為[[男性|男性]]的颜色，[[女性|女性]]則是[[紅色|紅色]]或[[粉紅色|粉紅色]]，这个习惯是从[[第一次世界大战|第一次世界大战]]后才开始的[http://answers.google.com/answers/threadview?id=238733]。有些情況下會以[[黑色|黑色]]、[[綠色|綠色]]或[[棕色|棕色]]來代表男性。至於[[淺藍色|淺藍色]]則常和男嬰聯繫在一起，因在傳統上西方父母往往會為男嬰穿上淺藍色的衣服。最主要原因是以冷色調的[[藍色|藍色]]中和男性的[[紅色|紅色]]體質，增加其穩重和克制；以暖色調的紅色中和女性的[[藍色|藍色]]體質，增加其勇氣與積極性。而在某些國家則以直觀方式分類，譬如[[日本|日本]]，男性顏色包括[[紅色|紅色]]，[[天藍色|天藍色]]，[[靛藍色|靛藍色]]；女性顏色包括，[[粉紅色|粉紅色]]，[[銀灰色|銀灰色]]，[[桃紅色|桃紅色]]；中性顏色兩者兼用的是：[[白色|白色]]，[[黑色|黑色]]，[[黃色|黃色]]，[[綠色|綠色]]，[[紫色|紫色]]，[[橙色|橙色]]等等。
* 红色在[[泰国|泰国]]是[[星期日|星期日]]的代表色。
* 红色在傳統西方神秘學中代表最具有男子氣概和勇氣的星座[[白羊座|白羊座]]，[[火星|火星]]或數字[[1|1]]與[[9|9]]，代表純粹的陽剛。
