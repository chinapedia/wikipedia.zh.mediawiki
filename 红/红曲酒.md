'''红曲酒'''，是一种以[[红曲米|红曲米]]和[[糯米|糯米]]为主要原料酿制成的[[粮食酒|粮食酒]]，为中国特产<ref>{{cite web |title=紅麴製酒過程 |url=http://anka.shulin.ntpc.gov.tw/wine_process.php |website=樹林區公所 紅麴虛擬博物館 |accessdate=2015-04-29 |archive-url=https://web.archive.org/web/20160306154738/http://anka.shulin.ntpc.gov.tw/wine_process.php |archive-date=2016-03-06 |dead-url=yes }}</ref>。[[红曲米|红曲米]]虽然具备一定的糖化和发酵能力，却不足以单独产出高浓度的酒，因此自古以来，红曲酒需另加[[酒曲|酒曲]]来拟补<ref name=酒酿>{{cite book |author1=杨建新 |author2=陈彩云 |author3=陈国灿 |title=金华酒酿造技艺 |date=2012 |publisher=浙江摄影出版社 |ISBN = 9787551401043 |pages=49}}</ref>。

== 概况 ==
[[File:Red_yeast_rice_wine_2_month.JPG|thumb]]]]

把[[红曲米|红曲米]]用于酿酒，源于何时，已甚难稽考，但不晚于[[北宋|北宋]]。北宋[[医学家|医学家]][[庄绰|庄绰]]在其作《鸡肋编》下卷中就写到「江南、闽中公私酝酿，皆红曲酒」<ref>{{cite book |author1=朱世英 |author2=季家宏 |title=《中国酒文化辞典》 |date=1990 |publisher=[[黄山书社|黄山书社]] |ISBN = 7805351112 |page = 196}}</ref>。南宋末年《[[事林广记|事林广记]]》卷八中，就提到了“造红曲法”和红曲酒的“东阳酝法”<ref name="酿造">{{cite book |editor=[[路甬祥|路甬祥]] |title=《中国传统工艺全集 酿造》|date=2007 |publisher=大象出版社 |ISBN = 7534740452 |page = 106}}</ref>。此外，元代的《居家必用事类全集》和元末明初的《易牙遗意》中则分别记录了“[[天台县|天台]]红酒”和“建昌红酒”的制作方法<ref name="酿造" />。

明朝[[李时珍|李时珍]]的《[[本草纲目|本草纲目]]》25卷，谷之四中则提到了红曲米的药用记载外，还写到「酿酒，破血行药势，杀山岚[[瘴气|瘴气]]，治打扑伤损（吴瑞）。治女人血气痛，及产后恶血不尽，擂酒饮之，良」。

== 知名种类 ==
* 福建老酒：福建老酒是[[福建省|福建省]]的传统名酒。[[闽语|闽语]]中，「糯」与 「老」发音近似，因此当地人将糯米酒读作容易发音的老米酒，略称老酒。福建老酒产自[[福州|福州]]酒厂，选用[[古田县|古田县]]一带所产的上等糯米为原料，古田红曲再加白露曲为糖化发酵剂所制<ref>{{cite book |author1=李松凌|author2=张永泰 |title=神州美酒谱 |date=1988 |publisher=贵州人民出版社 |ISBN ／= 7221002738 |pages=266}}</ref>。[[苏轼|苏东坡]]曾有“夜倾闽酒赤如凡”的诗句。

* 金华红酒：金华红酒兼用红曲、麦曲为糖化发酵剂，既有红曲酒之色、味，又有麦曲酒之鲜醇。[[朱震亨|朱震亨]]所著的《本草衍义补遗》就写到了红曲酒的习俗<ref>{{cite web|title=义乌丹溪酒：名医之后酿红曲|url=http://www.jhwwh.com/click.asp?nid=681|website=浙江金华市婺文化研究会|accessdate=2015-04-29}}{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>。李时珍的《[[本草纲目|本草纲目]]》卷25，谷之四中写到「东阳酒即金华酒，古兰陵也，李太白诗所谓兰陵美酒郁金香即此，常饮入药俱，良」。

* 老红酒：又称安溪红酒、金鸡老红酒，是福建[[诏安|诏安]]一带民间的酿造酒。安溪红酒的制法随[[郑成功|郑成功]]传入台湾，但因为当时[[红曲米|红曲米]]全部由中国大陆输入台湾，因此当时的老红酒仅供富人饮用。清代末期的[[宜兰县|宜兰县]]由于红曲制造技术的引进，降低了原料成本，使得酿酒风气兴盛起来。1909年，出身制酒世家的林青云运用他先祖在安溪学习得来的红曲米酿酒法，成立「[[臺灣菸酒股份有限公司|宜兰酒厂]]」，开始量产「安溪老红酒」<ref>{{cite journal |title=台湾的地方特产（二十二）|year =2012 |publisher=台湾周刊 |issue = 第24期 |pages=26}}</ref><ref>{{cite web |title=宜蘭『縣酒』的故事 |url = http://event.ttl-eshop.com.tw/yl/about/01main.aspx?cateid=205|website=宜蘭酒廠|accessdate=2015-04-29}}</ref><ref>{{cite web |title=紅麴做的酒－金雞陳年紅露酒 |url = http://event.ttl-eshop.com.tw/yl/about/01main.aspx?cateid=168|website=宜蘭酒廠 |accessdate=2015-04-29}}</ref>。1946年，宜兰酒厂将其改名为「红露酒」<ref>{{cite web |title=玉泉特級紅露酒 |url=http://liquor.ttl.com.tw/store_contain.asp?num=25 |accessdate=2015-04-29 }}{{Dead link|date=2018年7月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。2006年，宜兰县为促进产业观光化，提升经济效益，县政府将宜兰酒厂出品的「窖藏金鸡老红酒」称为宜兰县酒<ref>{{cite web |title=宜蘭縣議會第 16 屆第 3 次 臨時會 議案資料 |url = http://www.ilcc.gov.tw/CSource/C06/C0601M01.aspx?Fmotion_instanceOS=%C1%7B%AE%C9%B0%CA%C4%B3%AE%D7950920001&Sysno=H_08&System_work=6&btSearch=+-%B7j%B4M-&code=e&item=1 |accessdate=2015-04-29}}</ref>。
** 红露酒：1906年代，台灣商人[[黄纯青|黄纯青]]、谢道埤、陈炎等由中國福建引进红曲为原料，三人在[[树林区|树林区]]合资「树林造酒公司」以西方制酒方式以红曲制酒。其中糯米用量较多、酒龄较长之成品酒命名为「老红金鸡」，次级品为「老红酒黄鸡」。[[二战|二战]]期间，金鸡停止酿制。[[臺灣戰後時期|中華民國光復台灣後]]，黄鸡改名为「红露酒」<ref>{{cite web |title = 樹林酒廠 |url = http://anka.shulin.ntpc.gov.tw/story_winery.php |website = 樹林區公所 |accessdate = 2015-04-29 |archive-url = https://web.archive.org/web/20150925001057/http://anka.shulin.ntpc.gov.tw/story_winery.php |archive-date = 2015-09-25 |dead-url = yes }}</ref>。现在[[树林区|树林区]]有「红露酒故乡」的美名。
