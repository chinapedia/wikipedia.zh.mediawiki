'''红海'''（{{lang-ar|'''البحر الأحمر'''}}；{{lang-he|יַם סוּף}}；{{lang-fr|'''Mer Rouge'''}}；{{lang-en|'''Red Sea'''}}）位于[[非洲|非洲]]东北部与[[阿拉伯半岛|阿拉伯半岛]]之间，呈狭长型，长约2250公里，最宽355公里，均深490公尺，最深2211公尺，面积438,000平方公里。其西北面通过[[苏伊士运河|苏伊士运河]]与[[地中海|地中海]]相连，南面通过[[曼德海峡|曼德海峡]]与[[亚丁湾|亚丁湾]]相连。

紅海是[[印度洋|印度洋]]的[[陆间海|陆间海]]，实际是[[东非大裂谷|东非大裂谷]]的北部延伸。

== 名称 ==
紅海是直接由[[希臘文|希臘文]]、[[拉丁文|拉丁文]]、[[阿拉伯文|阿拉伯文]]翻譯過來。虽然红海在大多数时候并不呈红色，但是偶尔会季节性地出现大片红色[[藻类|藻类]]，这提供了其名称的一个可能的来源。另一些学者认为，“红”来源于其相对于命名者的方位（许多民族都有以颜色指代方向的传统），指南方或者西方。

另外一些该名稱的可能來源是：附近的紅色山脈；一個名稱為紅色的本地種族；紅地的海（古埃及稱沙漠為紅地）等。

== 历史 ==
最早由[[埃及人|埃及人]]發現。在《[[聖經|聖經]]·[[出埃及記|出埃及記]]》中，有[[摩西|摩西]]帶[[以色列人|以色列人]]穿過紅海的故事，但穿過的是否紅海受到質疑，因修訂版聖經中出谷記（即出埃及記）之備註是[[蘆葦海|蘆葦海]]，但沒有有關的考古證據。

[[網路謠言|網路謠言]]稱1978年，有人潛入紅海[[海床|海床]]發現一些被[[珊瑚|珊瑚]]包著的古代埃及戰車车轮，與海邊有古[[希伯来文|希伯来文]]的圆石柱，因此證明摩西穿過紅海<ref>{{cite journal |url=http://www.truth-monthly.com/issue120/0309mt01.htm |archiveurl=https://web.archive.org/web/20101104112557/http://www.truth-monthly.com/issue120/0309mt01.htm |title=史證鑿鑿聖經無誤聖經考古發現的啟示 |journal=真理報月刊 |issue=120 |date=2003-09 |archivedate=2010-11-04 |accessdate=2017-05-05 |location=[[加拿大|加拿大]][[卑斯省|卑斯省]][[大溫哥華|大溫哥華]]地區[[列治文|列治文]]市 |language=zh-hant |deadurl=yes }}</ref><ref>{{cite news |url=http://www.huaxia.com/hxhy/hywh/2011/06/2461296.html |title=摩西让红海开路 |date=2011-06-20 |language=zh-hans }}</ref>，其實是仍未證實的新聞<ref>{{cite web |url=http://www.truthorfiction.com/rumors/c/chariot-wheels.htm |title=Egyptian Chariot Wheels Found at the Bottom of the Red Sea-Unproven! |trans_title=古埃及戰車車輪在紅海海底現：未證實 |language=en }}</ref>。

在1世紀左右，[[希臘|希臘]]航海家打開了由紅海到[[印度|印度]]的航線。[[歐洲人|歐洲人]]在15世紀開始對紅海發生興趣，1798年[[法國|法國]][[拿破崙|拿破崙]]佔領[[埃及|埃及]]，並計劃建運河，但後來放棄了。1869年11月，[[苏伊士运河|苏伊士运河]]建成，由英法意分用，一戰後，美蘇也增加了對運河的影響力。1967-1975年，運河由於[[第三次中東戰爭|6日戰爭]]關閉。至今，[[紅海航線|紅海航線]]仍未超越[[好望角|好望角]]航線的地位。

== 地质 ==
滨岸水深一般浅于50公尺；红海沿岸广布着[[珊瑚礁|珊瑚礁]]；曼德海峡中散布着浅滩、岛礁；海峡下部有一道海槛；红海的[[中轴线|中轴线]]为中央海槽，大部深1500公尺。[[海槽|海槽]]中部出现几处深邃的V形[[裂谷|裂谷]]，为红海最深的地方。

[[非洲板块|非洲板块]]与[[阿拉伯板块|阿拉伯板块]]约在2000万年前开始分离，之间的裂谷沿海槽轴通过；两个板块近300万～400万年来，仍以每年约2.2毫米的速度分开。红海是未发育成熟的大洋，海底沉积物主要來自珊瑚礁和其他钙质生物碎屑，另有少量由风带来。

20世纪60年代初以来，在裂谷底层水中，发现了若干水温和盐度特别高的地点，近底层水温达34～56℃。

== 气候 ==
受[[副热带高压|副热带高压]]和[[信风|信风]]带控制，形成[[热带沙漠气候|热带沙漠气候]]；兼有[[季风气候|季风气候]]；冬半年，北部盛行西北风，南部盛行东南风；夏半年，海区多东北风，风速为3.4～10.7公尺/秒。[[降水|降水]]稀少，降水多集中于冬季，平均年降水北部28毫米，南部约127毫米，一般以[[雷暴|雷暴]]、[[沙塵暴|沙塵暴]]的形式下雨；终年高温，月平均气温2月最低（北部15.5℃），8月最高（南部32.5℃）；[[蒸发|蒸发]]量大，年平均蒸发量2100毫米，蒸发损失的水由[[印度洋|印度洋]]补充。

== 水文 ==
[[File:Dust_red_sea.jpg|thumb]]
* [[盐度|盐度]]：世界上盐度最高的海域，盐度在3.6-3.8%之間；比其他深层海水盐度高2～9倍；由于裂谷扩展涌上来的[[熔岩|熔岩]]加热沿裂隙下渗的海水，而富含溶解盐类和[[矿物质|矿物质]]的热水重新上升。

* [[洋流|洋流]]：因气候原因，红海水面比周围海域的水面要低，使得表层洋流向红海流动；海底因为盐度高，密度大，压强大，所以底层洋流是红海海水向周围海域流动。

== 交通 ==
紅海是连接[[地中海|地中海]]和[[阿拉伯海|阿拉伯海]]的重要通道，也是一条重要的[[石油|石油]]运输通道，並具有战略价值。
