'''红眼航班'''（{{lang-en|'''Red-eye flight'''}}）是指在[[深夜|深夜]]至[[凌晨|凌晨]]时段运行，並於翌日[[清晨|清晨]]至早上抵達目的地，飛航時間少於正常[[睡眠|睡眠]]需求（8小时）的[[客運航班|客運航班]]。红眼航班最早在1959年出现于[[美国|美国]]，因为乘客下飛機时多睡眼惺忪，像[[兔子|兔子]]一樣紅著眼睛上下飛機，红眼航班因此得名。但如果是約八小時以上的紅眼航班，對可以在飛機上熟睡的旅客來說反而較佳，因這種航班可以給予旅客充足睡眠，还可以節省旅客的時間及住宿費用；在另一方面，如果兩地[[時差|時差]]過大，旅客為了調整時差就產生了類似熬夜的身心負擔。

事實上，由於红眼航班的飛行時間欠佳，通常只能吸引[[經濟艙|經濟艙]]乘客選乘，對於追求生活品味、能為航空公司提供較高收益的頭等、商務旅客而言，红眼航班是相當惡劣的飛行安排。因此，红眼航班的[[頭等艙_(航空)|頭等]]、[[商務艙|商務客艙]]上座率通常極度慘淡，而一旦航空公司對經濟艙的客量預算失準，客運上則隨時有虧本的可能。然而，紅眼時段是[[貨運|貨運]]的[[黃金時段|黃金時段]]，航空公司可以以客機的腹艙裝載貨物爭取收益，尤其是運送數量少、時間短，但收益相當高的急件貨物，利用紅眼航班不僅可以減省使用大型貨機的高昂營運成本，還可以開拓經濟艙的市場，提高收益。因此，對於部分依賴貨運業務的主要航空公司（如[[國泰航空|國泰航空]]、[[大韓航空|大韓航空]]等），红眼航班是他們十分喜歡的飛行安排，一方面可賺取急件貨物的收益，另一方面則以極廉價的機票價錢安排旅行團乘坐航班，賺取更多利潤。近年來，甚至出現了以客機營運但不載客的红眼航班，以及在凌晨時分起飛/降落的載客紅眼航班等異象。

== 優點與缺點 ==
{{unreferencedsect|time=2014-3-18}}
;優點
* 提高飛機使用率
* 如果航程長（超過7小時），旅客可以在機上獲得充足睡眠，然後一抵達目的地便可開始行程。
* 價格方面，一般也要優於其他航班，至少比普通航班低兩折左右。

;缺點
* 載客量低，航空公司很可能虧本。
* 飛機起降時間過晚，聯外公共交通已收班，旅客可能需承擔額外成本才能出入機場。
* 很可能違背機組人員的生理時鐘，造成安全性下降。
* 旅客因需要提前約1至2小時抵達機場報到，因此可能無法獲得充足睡眠，甚至需要[[熬夜|熬夜]]，可能影響抵達目的地後白天的行程。
* 起降航線下方的居民在夜間及清晨對噪音的忍受力較低，接近人口密集區的機場通常設有[[宵禁|宵禁]]在夜間禁止飛機起降，因而不適合飛紅眼航班。

== 舉例 ==
===亞太區===
* [[大韩民国|南韓]]和[[日本|日本]]均比[[香港|香港]]快一小時，因此從香港前往南韓和日本的航班不乏紅眼航班。[[國泰航空|國泰航空]]、[[港龍航空|港龍航空]]、[[大韓航空|大韓航空]]或[[韓亞航空|韓亞航空]]皆有營運香港飛往[[首爾|首爾]]及[[釜山廣域市|釜山]]的凌晨航班。至於香港直航日本方面，國泰航空、[[全日本空輸|全日本空輸]]也有提供紅眼航班前往[[东京|东京]]和[[大阪市|大阪]]（[[日本航空|日本航空]]也曾開辦香港往[[東京國際機場|羽田機場]]的紅眼航班，但目前已改為日間航班）；從香港出發經停臺北的航班通常為了在臺北重新載客，故未安排紅眼航班。
* [[东南亚|东南亚]]（如[[泰国|泰国]]、[[新加坡|新加坡]]、[[马来西亚|马来西亚]]等）往返日本、南韓的航程需時約5-6小時，而日韓的時制也比東南亞快一小時（較泰國快2小時），因此從東南亞往來日韓的航班也不乏紅眼航班。
* 東亞距離[[澳大利亚|澳大利亚]]較遠（約6,000至7,500公里不等），航程需時7-10小時不等，因時差之故也存在不少紅眼航班。通常是由東亞前往澳洲東岸城市（[[悉尼|悉尼]]、[[墨尔本|墨尔本]]、[[布里斯班|布里斯班]]或[[凱恩斯_(昆士蘭州)|凯恩斯]]）的紅眼航班較為普遍（回程也有，但數量較少），國泰航空更設有在傍晚時分，由香港飛往悉尼和墨爾本的航班；來往東南亞與東北亞的部分航班會在香港或臺北中途加油與載客，如[[新加坡航空|新加坡航空]]或[[泰國國際航空|泰國國際航空]]。
* 澳洲國土面積遼闊，橫跨多個時區，由澳洲西岸飛往東岸的國內航班需時約4-5小時，且東岸城市較西岸快2小時（夏季快3小時），因此也設有紅眼航班。
* [[印度|印度]]、[[尼泊尔|尼泊尔]]、[[中东|中東地區]]直航往香港、東南亞、[[东北亚|东北亚]]的航班；[[中華航空|華航]]從[[臺北|臺北]]經[[德里|德里]]前往[[罗马|罗马]]的航班亦為紅眼航班。
* 從[[臺灣桃園國際機場|臺灣桃園國際機場]]前往歐洲的班次，若為不中停直飛的話，多半為紅眼航班；如[[荷蘭皇家航空|荷航]][[马尼拉|马尼拉]]→臺北→[[阿姆斯特丹|阿姆斯特丹]]、華航臺北→[[維也納|維也納]]或[[美因河畔法兰克福|法蘭克福]]，以及[[長榮航空|長榮]]臺北→[[巴黎|巴黎]]等班次，皆為次日早上抵達歐洲機場。部分飛往美加的航班也為了通過國際換日線而安排紅眼航班，以便於次日早上飛抵[[溫哥華|溫哥華]]、[[洛杉矶|洛杉矶]]或[[旧金山|旧金山]]；如2000年發生於桃園機場的[[新加坡航空006號班機空難|新航006空難]]，從新加坡飛抵臺北中途加油後，在深夜起飛時因天候惡劣致視線不佳發生死亡空難。另外，[[低成本航空公司|低成本航空公司]]興起之後，新增飛往日本、韓國及菲律賓等地的紅眼航班。
* 在[[伊斯兰历|伊斯兰历]][[朝觐_(伊斯兰教)|朝覲]]期間，東南亞的[[伊斯兰国家|伊斯兰国家]]如[[马来西亚|马来西亚]]與[[印度尼西亚|印尼]]，航空業者為了開闢飛往[[沙特阿拉伯|沙特阿拉伯]][[吉达|吉达]]的朝聖航班，以及飛越[[印度洋|印度洋]]的需求，部分從[[雅加达|雅加达]]、[[吉隆坡|吉隆坡]]出發的航班會在夜晚起飛，次日早上飛抵吉達，展開穆斯林的朝聖旅程。

===北美洲===
* [[美国|美国]]和[[加拿大|加拿大]]也橫跨多個時區，且來往西部與東部之間的飛行時間約為5-6小時，紅眼航班數量也為數不少。
* 從美洲飛到亞洲的航班，由於需經過[[国际日期变更线|国际日期变更线]]，故較多紅眼航班；如[[大韓航空007號班機空難|-{zh-hans:大韩航空007号班机空难; zh-hant:韓航007空難;}-]]，韓航班機從[[纽约|纽约]]經[[安克拉治|安克拉治]]加油之後飛往[[首爾|首爾]]的途中，深夜誤闖[[库页岛|库页岛]]上空遭前蘇聯戰鬥機擊墜。
* 美國方面除了服務橫越大西洋及太平洋的航線外，亦提供了往來南北美的航班，從美國前往[[阿根廷|阿根廷]]、[[巴西|巴西]]、[[智利|智利]]等南美國家的班次也會提供紅眼航班，以應付美洲長途飛行；但因為航線呈南北走向，時區跨距並不明顯，較越洋航線難發揮紅眼航班次日晨間抵達的特性。
* 美國部分航空公司從美國本土飛往[[阿拉斯加州|阿拉斯加]]及[[夏威夷州|夏威夷]]的航線，雖屬國內航班的範疇，但因為飛行距離遙遠，故會安插一部分紅眼航班。

===歐洲===
* [[芬兰航空|芬兰航空]]從[[赫尔辛基|赫尔辛基]]前往亞洲航點的航班，為了藉由[[大圆|大圓航線]]快速飛抵目的地，通常於晚間出發，並於次日早上飛抵亞洲機場。
* [[俄罗斯|俄罗斯]]方面由於擁有世界上最大的領土與跨越最多的時區，從西部城市[[圣彼得堡|圣彼得堡]]或[[莫斯科|莫斯科]]出發到遠東城市[[伯力|伯力]]、[[海參崴|海參崴]]的航班偶爾會安排紅眼航班，但因為屬國內航線，也會見到日間航班。
* [[英國航空|英國航空]]及[[法國航空|法國航空]]曾營運過的[[协和飞机|協和客機]]，由於具有超音速飛行的優勢，飛越大西洋的航線全程約兩個半小時，因此協和客機的班次較少紅眼航班。

===洲際===
* [[亚洲|亚洲]]往返[[欧洲|欧洲]]的航班；
* 歐洲往返[[美洲|美洲]]的航班。

== 其他 ==
===中国对红眼航班的禁令===
2002年，中国民航业相隔22天先后发生两起空难－[[中国国际航空129号班机空难|釜山空难]]<ref>[http://news.bbc.co.uk/chinese/simp/hi/newsid_1930000/newsid_1930100/1930141.stm  BBC中文网：中国国航客机坠毁韩国釜山]，2002年04月15日</ref>和[[中国北方航空6136号班机空难|大连空难]]<ref>[http://news.sina.com.cn/c/2002-05-08/2217569541.html 中新网：北航麦道客机5月7日在大连附近海域失事]，2002年05月08日</ref>。5月11日，[[中国民用航空局|中国民航总局]]颁布一系列安全规定<ref>[http://fol.math.sdu.edu.cn/jrxt/content/content.php?tb=alfx&id=23 21世纪经济报道：大连空难背后还有多少秘密？]{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}，2002-05-23</ref>，包括了禁止任何国内航班跨零时起降，即对红眼航班祭出禁令。由于此禁令的存在，一些长途航线的时间安排受到限制，加大了机场资源的緊繃程度。2006年，民航总局下发通知<ref>[http://news.carnoc.com/list/64/64483.html 民航资源网: 走出空难阴霾-民航总局将解禁“红眼航班”]，2006-02-24 </ref>，国内所有机场客运航班时刻从零点延长至次日凌晨二时，[[新疆维吾尔自治区|新疆地区]]延长至次日凌晨四时。2008年，中国大陆部分省市遭[[2008年中国雪灾|雪灾]]襲擊后，民航总局下发通知，取消凌晨兩点至六点禁止航空公司运行国内客运航班的限制，以帮助疏散滞留旅客。
