'''紅厚殼'''（[[学名|学名]]：{{lang|la|''Calophyllum inophyllum''}}），又名'''瓊崖海棠'''，为[[胡桐科|胡桐科]][[红厚壳属|红厚壳属]]的植物。

== 形态 ==
常绿[[乔木|乔木]]，暗褐或灰褐色树皮；椭圆形或宽椭圆形的叶子对生，顶端钝圆或微缺，厚革质有光泽，全缘或波状，侧脉细密；总状花序或圆锥花序，花白色有香气；球形[[核果|核果]]，熟时为黄色、肉质。

== 分布 ==
分布在[[斯里兰卡|斯里兰卡]]、[[臺湾|臺湾]]、[[马达加斯加|马达加斯加]]、[[菲律宾|菲律宾]]、[[中南半岛|中南半岛]]、[[波利尼西亚|波利尼西亚]]、[[马来西亚|马来西亚]]、[[安达曼群岛|安达曼群岛]]、[[印度尼西亚|印度尼西亚]]、[[澳大利亚|澳大利亚]]、[[印度|印度]]以及[[中国大陆|中国大陆]]的[[海南|海南]]等地，生长于海拔60米至200米的地区，多生在丘陵空旷地及海滨沙荒地上，目前尚未由人工引种栽培。

== 别名 ==
胡桐（中国高等植物图鉴误用），琼崖海棠树（中国树木分类学），海棠木、海棠果（广东），君子树（海南），呀拉菩（台湾）
