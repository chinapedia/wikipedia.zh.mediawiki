《'''红楼梦'''》是[[中国中央电视台|中国中央电视台]]旗下的[[中国电视剧制作中心|中国电视剧制作中心]]根据[[红楼梦|同名小说]]改编拍摄的[[电视剧|电视剧]]，1984年9月开始拍摄，1987年在中国大陆全国首播。该剧根据[[红学|红学]]家[[周汝昌|周汝昌]]的意见，没有按照[[高鹗|高鹗]]的续四十回的情节拍摄，对结尾部分作了重大改动。

该剧由多位文艺界泰斗如[[启功|启功]]、[[沈从文|沈从文]]、[[王昆仑|王昆仑]]等人担任顾问，于全国各地遴选演员，仅[[林黛玉|林黛玉]]一个角色就有三万余人参选，并专门花时间对演员进行培训。其制作极为严谨，仅为不到一分钟的抚琴镜头就专门为演员聘请[[古琴|古琴]]老师进行教学。且专门建造正定荣国府、[[北京大观园|北京大观园]]作为拍摄基地，并在[[黄山|黄山]]、[[长江|长江]]、[[苏州园林|苏州园林]]等名胜古迹处取景。该剧的拍摄集中了当时大陆电视业界和红学界的诸多力量，在大陆电视剧史上留下了浓重的一笔。

本剧播出后即红极一时，并多次重播，成为经典剧集。剧中的《[[枉凝眉|枉凝眉]]》、《[[葬花吟|葬花吟]]》等[[歌曲|歌曲]]也广为传唱。而引起广泛争议的是电视剧的[[故事|故事]]结局，没有按照高鶚續之后四十回，而是根据红学家的探佚结果，[[悲剧|悲剧]]結局更为彻底。

== 拍摄过程 ==
* 1979年，导演[[王扶林|王扶林]]提议将古典名著《[[红楼梦|红楼梦]]》以电视剧的形式搬上荧幕，此想法得到了[[中央电视台|中央电视台]]和[[红学|红学]]界的支持。
* 1981年11月，[[中央电视台|中央电视台]]召开了关于《[[红楼梦|红楼梦]]》拍摄成电视剧的专题会议，专门研究改编问题。
* 1983年2月筹备组成立，5月编剧组成立，8月剧组顾问委员会成立，《[[红楼梦|红楼梦]]》电视剧拍摄进入实质性阶段。12月，剧本初稿正式确定。
* 与此同时，剧组采取海选形式在全国各地选拔演员，引起热烈反响，一百多个演员席位争夺的异常激烈。
* 1984年春夏期间，剧组在北京圆明园举办了两期红楼梦剧组演员学习班，主要学习内容为研究原著，分析角色，亦学习琴棋书画，增强艺术修养。通过学习，才慢慢确定了最终的演员人选。同时，剧组为了拍摄的需要，按照原著的描述，在[[北京市|北京市]][[宣武区|宣武区]]建立了大观园，即现在的[[北京大观园|北京大观园]]；出于北京用地紧张的考虑，剧组另外在[[河北|河北]][[正定县|正定县]]建造了[[宁国府|宁国府]]、[[荣国府|荣国府]]和[[宁荣街|宁荣街]]等景址，如今都成为旅游风景区。同年2月9日，开始试拍。
* 1984年9月10日，本剧在[[安徽省|安徽]][[黄山|黄山]]正式开机，录下了第一组镜头，1987年上半年完成拍摄，总投资为680万[[人民币|人民币]]<ref>[http://www.people.com.cn/GB/yule/1082/2233655.html 艺术人生：“红楼”明星20年后再聚首]</ref>。整个拍摄过程历时三年之久，剧组先后走遍全国10个省市的41个地区的219个景点，拍摄镜头近万个。<ref>[http://news.qq.com/a/20060927/000687.htm 87版《红楼梦》金陵12钗选角内幕] 2006年09月27日 国际在线</ref>
* 总制片人任大惠透露，本剧片酬最高的是他和导演[[王扶林|王扶林]]，每集70元人民币。[[欧阳奋强|欧阳奋强]]、[[陈晓旭|陈晓旭]]、[[张莉_(演员)|张莉]]、[[邓婕|邓婕]]四位主演为每集60元人民币，其他演员则分别是每集50元、40元、30元、20元不等。<ref>[http://yule.sohu.com/2003/12/07/54/article216595497.shtml 红楼儿女20年后聚首 岁月不仁但真情依旧] 2003年12月7日 京华时报</ref>

== 剧本 ==
本剧剧本由[[刘耕路|刘耕路]]、[[周雷|周雷]]、[[周岭|周岭]]三位[[编剧|编剧]]根据[[红学家|红学家]][[周汝昌|周汝昌]]的意见进行编写。总体原则为“忠于原著”(指[[曹雪芹|曹雪芹]]所著的前八十回)，而对后四十回则采取“重视续作”的原则，续作中凡与原著相契合处就采用，悖谬处则摒弃，主要是根据前八十回的伏线，根据脂砚斋的提示，参考续作和红学家多年来的研究成果，采用电视剧本的形式合理构想后四十回的故事。不过限于20世纪80年代电视剧拍摄的局限，原著中的不少内容被略过，比如“贾宝玉梦游太虚境”等都没有在电视剧上反映出来，不能不说是一个遗憾。

剧本于1983年3月6日起稿，当年底完成初稿，1984年3月完成二稿，直到1986年1月才最终定稿，历时两年零二个月。在此期间，周汝昌、吴世昌、朱家溍、杨乃济、阮若琳、王立平、戴临风、导演王扶林等人都提出了许多建议和意见。全剧本最终由[[周岭|周岭]]统一修改并定稿。

=== 周边 ===
本剧演员的海选是一次真正意义上面对民间的大选拔，其中有售货员、干事、接线员、皮鞋厂工人、中学生……等。持续三年的剧组工作、生活让这一批演员深深的被《红楼梦》所影响——[[妙玉|妙玉]]扮演者姬培-{杰}-改名[[姬玉|姬玉]]，[[晴雯|晴雯]]扮演者张静林改名[[安雯|安雯]]。[[鸳鸯_(人物)|鸳鸯]]扮演者[[郑铮_(演员)|郑铮]]则说：“鸳鸯对我的影响是绝对的，影响了至少十年”。

虽说本剧的热播让这些原本是普通人的演员有了一个较高的高度，但除了[[邓婕|邓婕]]、[[郑铮_(演员)|郑铮]]等个别演员，很多人在之后的演艺道路上并不成功。一是角色被定型的原因，如[[陈晓旭|陈晓旭]]、[[欧阳奋强|欧阳奋强]]虽然演出很成功，但都因无法摆脱剧中角色的影子，一个下海经商，一个转行做导演；二是导演[[王扶林|王扶林]]带有浓厚理想主义的选材方式，为了追求演员扮相高度契合原著小说，多数演员的外表都是宜古不宜今，之后的戏路受到很大制约。

拍完红楼梦之后出国的有：[[薛宝钗|薛宝钗]]扮演者[[张莉_(演员)|张莉]]、[[贾元春|贾元春]]扮演者成梅、[[紫鹃|紫鹃]]扮演者徐丽霞、[[秦可卿|秦可卿]]扮演者张蕾、[[尤二姐|尤二姐]]扮演者張明明……等。

转行经商的有：[[林黛玉|林黛玉]]扮演者[[陈晓旭|陈晓旭]]、[[贾惜春|贾惜春]]扮演者[[胡泽红|胡泽红]]、[[贾迎春|贾迎春]]扮演者牟一、[[妙玉|妙玉]]扮演者[[姬玉|姬玉]]……等。

继续留在演艺圈的有：[[贾探春|贾探春]]扮演者[[东方闻樱|东方闻樱]]、[[鸳鸯_(人物)|鸳鸯]]扮演者[[郑铮_(演员)|郑铮]]、[[秋桐|秋桐]]扮演[[沈璐|沈璐]]、[[贾宝玉|贾宝玉]]扮演者[[欧阳奋强|欧阳奋强]]……等。

[[林黛玉|林黛玉]]扮演者[[陈晓旭|陈晓旭]]和[[贾瑞_(红楼梦)|贾瑞]]扮演者马广儒的人生境遇则真有些印证了人生如戏，尤其是后者，出演[[贾宝玉|贾宝玉]]未成，入戏太深，之后在现实生活中也是异常坎坷，角色落选、爱情失意、父亲去世。1995年，年仅32岁的马广儒酗酒而死，只留下一部《红楼梦》和一支洞箫。而[[王熙凤|王熙凤]]的有力候选樂韻为了追求一个香港演员而辞演，后来因感情不顺，在[[香港|香港]]跳楼自杀，年仅29岁。

对于这些，[[郑铮_(演员)|郑铮]]有感：“对好多人来说，那红楼老在心口上，太疼了”。<ref>[http://news.xinhuanet.com/ent/2006-10/30/content_5266538.htm 86版《红楼梦》留在心口 一入红楼"误"了终身 ] {{webarchive|url=https://web.archive.org/web/20070204063815/http://news.xinhuanet.com/ent/2006-10/30/content_5266538.htm |date=2007-02-04 }} 2006年10月30日 沈阳今报</ref>

== 音乐 ==
《红楼梦》的音乐是由著名词曲作家[[王立平|王立平]]创作而成。1984年，[[王立平|王立平]]向《红楼梦》剧组自荐获准，此后历时四年，耗尽心力完成了《序曲》、《引子》、《枉凝眉》、《葬花吟》、《紫菱洲歌》、《红豆曲》、《秋窗风雨夕》、《晴雯歌》、《聪明累》、《分骨肉》、《叹香菱》、《题帕三绝》、《好了歌》13首歌曲，其中仅一首《葬花吟》就费时一年零九个月，用他自己的话形容“这是我投入感情最多、面临压力最大、遇到困难最多的一次创作”。

歌曲《序曲》以古筝开篇，用一段幽怨、婉转的女声独唱，表达了一段催人泪下的嗟叹情感，其后用弦乐的变调引出红楼梦的主旋律；《引子》则秉承《序曲》的基调，注重给人以广阔的遐想空间，演绎了金陵十二钗的哀怨之声；《枉凝眉》的主题是宝黛悲剧爱情，其曲调极为哀怨，有动人魂魄之感。

歌曲的女声原唱者是[[陈力_(歌手)|陈力]]，男声原唱者是王洁实和叶茅，当时的陈力仅是一名普通的女工歌手，因其嗓音符合王立平对音乐的要求，故被选中。陈力当时丈夫因意外过世，或许是设身处地的感怀身世，她将自己的特殊情感完全注入到歌声之中，唱出了沧桑凄感、动人魂魄的天籁悲音。这亦是陈力的绝唱，之后因为人生变故，她不再从事歌唱事业。<ref>[http://www.360doc.com/content/10/0723/15/65663_40916185.shtml 王立平与他催人泪下的《红楼梦》插曲]</ref><ref>[http://www.cflac.org.cn/chinaartnews/2007-01/26/content_9149103.htm 王立平谈《红楼梦》音乐的创作]{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }} 2007年1月26日 中国艺术报</ref><ref>[http://www.66163.com/Fujian_w/news/fjrb/jjjj/dzb/html/2008-01/22/content_86692.htm 《红楼梦》主题曲 我没有缺憾]{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }} 2008年1月22日 晋江经济报</ref><ref>{{cite news|url=http://www.people.com.cn/GB/paper1787/10818/982869.html|title=红楼明星20年后再聚首（再聚红楼）|publisher=[[京华时报|京华时报]]|date=2003-12-07|accessdate=2015-11-09}}</ref>

== 评价 == 
本剧在1987年刚刚播出时，就受到观众极大的欢迎，但也引起不小争议。1987年6月20日至6月24日相关部门在[[北京|北京]]举行电视剧《红楼梦》学术研讨会，与会的专业人士对此剧既有肯定也有批评，认为改编成电视剧是一次可贵的尝试，但仍有不少不足之处<ref>[http://gb.cri.cn/27564/2010/09/10/342s2987672.htm 87版《红楼梦》当年评论界观点精编 与新版境遇有多相似？] 2010年9月10日 国际在线</ref>。尽管本剧相比原著不完美，但依旧红遍全国，到现在依然重播不断，成为大陆影视剧经典剧作之一，剧中多名角色（如[[陈晓旭|陈晓旭]]扮演的[[林黛玉|林黛玉]]、[[邓婕|邓婕]]扮演的[[王熙凤|王熙凤]]、[[欧阳奋强|欧阳奋强]]扮演的[[贾宝玉|贾宝玉]]）深入人心，喜爱者甚多。据电视剧发行商[[北京华录百纳有限公司|北京华录百纳有限公司]]的统计，本版《红楼梦》到2010年已重播1000余次。著名红学家[[冯其庸|冯其庸]]在当年曾这么描述过此剧的影响力：“《红楼梦》小说有史以来最大的普及。当时确实书店里的书抢购一空，这是对的。以前都是折子的、片断的表演，我们是全本搬演，第一次把《红楼梦》的人物在屏幕上定了位”。可以说87版《红楼梦》的播出对原著小说的推广起了重要作用，尤其是在中国大陆1970年代和1980年代出生的群体中有着深远的影响力。

2010年9月，[[中国青年报|中国青年报]]社会调查中心在《[[红楼梦_(2010年电视剧)|2010版红楼梦]]》热播时做了一项调查，发现逾八成读过原著者更喜欢1987版《红楼梦》。公众肯定1987版《红楼梦》的亮点有“深得原著精髓”、“叙事好、吸引人”、“服饰、道具、布景等精致逼真”、“台词白话、容易理解”、“没有拘泥于原著，有自己的探索创新”。而1987版《红楼梦》的音乐更是被大家认为无法超越的部分，如《枉凝眉》等歌曲更是成为了经典之作。<ref>[http://news.e23.cn/Content/2010-09-16/201091600200.html 逾8成读过原著者更喜欢87版《红楼梦》]{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }} 2010年9月16日 [[中国青年报|中国青年报]]</ref>

87版《红楼梦》的支持者更是列出了七大要素来说明本版剧作为何经久不衰：编剧忠于原著，敢于发挥；导演心无旁骛、潜心创作；演员形神兼备，过目难忘；歌曲浑然天成，哀怨动人；色调明快亮丽，赏心悦目；造型博采众长，深入人心；经典无须争辩，历久弥新。<ref>[http://www.360doc.com/content/10/0915/19/749013_53914720.shtml 87版《红楼梦》经久不衰的原因的七大秘诀]</ref>
