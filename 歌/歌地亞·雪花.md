'''歌地亞·雪花'''（[[德語|德語]]：'''Claudia Schiffer'''，{{bd|1970年|8月25日|catIdx=Schaffer, Claudia}}），是出生於[[德國|德國]][[北莱茵-威斯特法伦|北莱茵-威斯特法伦]]的[[超模|超模]]。在歐美時尚界她有「真人[[芭比娃娃|芭比娃娃]]」的封號。2002年，《[[福布斯|福布斯]]》估計歌地亞共有3800萬的財產。在德國所舉辦的[[2006年世界盃足球賽|2006年世界盃足球賽]]的開幕典禮上，她和巴西球王[[貝利|貝利]]一起同台揭開世界盃的序幕。

== 早年生活 ==
歌地亞在德國的一個富有的[[天主教|天主教]]家庭長大。

歌地亞起初是想當[[律師|律師]]，並和她父親一起工作。但直到1987年被人發掘成為模特兒才放棄這個念頭。

== 事業 ==
[[File:Claudia_Schiffer_Césars_1993.jpg|thumb]]举办的[[恺撒奖|恺撒奖]]颁奖礼]]
[[File:Claudia_Schiffer_Cannes.jpg|thumb]]]]
克勞迪婭在1987年時於夜店被Michel Levaton發掘，並簽約Metropolitan Model Agency。她憑其5尺11吋的高挑身材，貌似五＋年代法國明星，碧姬·芭杜的面孔，當時17歲的克勞迪婭迅速竄紅。克劳迪娅的伯樂是[[卡爾·拉格斐|卡爾·拉格斐]]，卡爾在九十年代時提拔克勞迪婭為[[香奈兒|香奈兒]]的專用模特兒，自此克勞迪婭便稱霸於伸展台上。1991年，克勞迪婭以當時史上最高金額，一千萬美元與化妝品牌，露華濃簽訂廣告合約，轟動一時。幾年間，克勞迪婭為五百多本時裝雜誌拍攝封面，資產多達三千五百萬英鎊。她最後一次走伸展台是在1998年。克勞迪婭還跟另外3位的超模，[[娜奧美·金寶|娜奧美·金寶]]、[[克莉絲蒂·杜靈頓|克莉絲蒂·杜靈頓]]及艾麗·麥花遜開有一間名為「Fashion Cafe」的餐廳。她與，[[娜奧美·金寶|娜奧美·金寶]]、[[克莉絲蒂·杜靈頓|克莉絲蒂·杜靈頓]]、[[辛蒂·克勞馥|辛蒂·克勞馥]]以及[[琳達·伊凡吉莉絲塔|琳達·伊凡吉莉絲塔]]一起被譽為「世上最偉大的超級名模」。

== 個人生活 ==
歌地亞於1994至1999年跟知名[[魔術師|魔術師]][[大衛·考柏菲_(魔術師)|大卫·科波菲尔]]相愛並訂婚，其後分手。2001年她認識了導演[[马修·沃恩|马修·沃恩]]，並於2002年5月25日結婚，2003年1月30日歌地亞產下兒子Caspar Matthew；2004年11月11日產下女兒Clementine；2010年5月14日產下女兒Cosima。克劳迪娅現居住在[[倫敦|倫敦]]的[[诺丁山|诺丁山]]。

== 慈善工作 ==
歌地亞是一位有爱心的女性，她是预防[[乳腺癌|乳腺癌]]的代言人，还是[[联合国儿童基金会|联合国儿童基金会]]的志愿者。

== 美譽及軼事 ==
*真人[[芭比娃娃|芭比娃娃]]
*美國雜誌《人物》（People），「世界上50位最美麗的人」之一（1990-1991年）
*法國雜誌《Paris Match》，「世界上最美麗的女人」（1995年）
*歌地亞曾是全球收入最高的模特兒
*第一位登上《人物》（People）、《名利場》（Vanity Fair）、《滾石》（Roilling Stone）封面的模特兒
*健力士世界記錄的「上過最多雜誌封面的模特兒」的記錄保持者（達500多本）
*歌地亞閒時喜愛[[繪畫|繪畫]]及欣賞名畫。
