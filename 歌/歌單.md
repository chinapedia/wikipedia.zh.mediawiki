'''歌單'''，又名播放列表，即是播放[[歌曲|歌曲]]的名單，在[[電台廣播|電台廣播]]及[[個人電腦|個人電腦]]中有不同的含意<ref>{{cite web|url=http://minnesota.publicradio.org/radio/services/the_current/playlist/faq.php |title=Frequently Asked Questions |publisher=Minnesota Public Radio |accessdate=11 April 2011 |deadurl=yes |archiveurl=https://web.archive.org/web/20121014120008/http://minnesota.publicradio.org/radio/services/the_current/playlist/faq.php |archivedate=14 十月 2012 }}</ref>；亦可以指[[DVD|DVD]]上的錄像檔案清單，或某些[[互聯網|互聯網]]影片分享[[網站|網站]]例如[[YouTube|YouTube]]上的播放列表。通常歌單都可以順序播放或{{link-en|隨機播放|Shuffle play}}<ref>{{cite web|title=iPod shuffle Tips and Tricks|url=http://oreilly.com/pub/a/mac/2005/02/15/shuffle.html|first=Scott|last=Knaster|date=February 15, 2005|work=Mac OS X Excerpts|publisher=O'Reilly Media|accessdate=11 April 2011|archive-url=https://web.archive.org/web/20140826120413/http://oreilly.com/pub/a/mac/2005/02/15/shuffle.html|archive-date=2014-08-26|dead-url=yes}}</ref>。

==電台==
早於1950年代，各地的電台已經開始自製（甚至發佈）最受歡迎歌曲名單，歌單的口味亦可以隨不同的{{link-en|廣播時段|Dayparting}}而改變，例如晨早時段較為輕快，半夜時段較為舒情等<ref>{{cite journal|title=Collaborative Radio Community|author=Paolo Avesani, Paolo Massa, Michele Nori, & Angelo Susi|publisher=ITC irst|location=Italy}}</ref>。

==電腦及互聯網==
隨着電腦儲存及播放音樂檔案愈趨普及，大多[[媒體播放器|媒體播放器]]已加入歌單功能，藉以更加有效地整理及控制這些檔案。使用者可以隨自己的個人喜好，按照[[歌手|歌手]]的名稱、歌曲的類別，或音樂的年份製定一個或多個歌單，再按心情選播。

===製作歌單===
製作歌單最初是帕凱特（{{lang|en|Pachet, F.}}）和羅伊（{{lang|en|Roy,P.}}）的構思<ref> Pachet, F. and Roy, P., [http://www.csl.sony.fr/publications/item/?reference=pachet%3A99m Automatic Generation of Music Programs] {{webarchive|url=https://web.archive.org/web/20120509141312/http://www.csl.sony.fr/publications/item/?reference=pachet%3A99m |date=2012-05-09 }}. Proceedings of Constraint Programming Conference, CP 99, LNCS 1713/2004, pages 331-345, Washington, VA, 1999. Springer Verlag.</ref>，透過[[人工智能|人工智能]]的{{link-en|約束補償|Constraint satisfaction}}技術，創建滿足任意的“序列約束”播放列表。自此，多項技術亦得到引入，包括[[案例推論|案例推論]]（{{lang|en|Case-based reasoning}}）<ref name="ai">{{cite journal|title=Case-Based Sequential Ordering of Songs for Playlist Recommendation|doi=10.1007/11805816_22|url=https://link.springer.com/chapter/10.1007/11805816_22|date=2006-09-04|publisher=Springer, Berlin, Heidelberg|language=en|pages=286–300|accessdate=2018-04-02|isbn=9783540368434|author=Claudio Baccigalupo, Enric Plaza}}</ref>。

===常見格式===
以下是部分歌單存檔格式：
* [[.asx|.asx]]：以[[XML|XML]]製成，可以儲存歌單上各項目的多種資訊<ref name="microsoft-formats"/>
* .bio：以文本為基礎的格式
* .fpl：為[[foobar2000|foobar2000]]所採用
* .kpl：以[[XML|XML]]製成，改善加載時間
* [[M3U|.m3u]]：以文本為基礎的格式，頗為普及
* .pla：為[[三星|三星]]及[[Winamp|Winamp]]所採用，二進制
* .aimppl、.plc：為[[AIMP|AIMP]]所採用
* [[.pls|.pls]]：與[[.ini|.ini]]類似的格式
* [[Synchronized_Multimedia_Integration_Language|.smil]]：獲[[萬維網聯盟|萬維網聯盟]]推薦的格式；亦支援視頻和屏幕佈局檔案，在[[數位標牌|數位標牌]]中使用<ref>{{cite web|url=http://www.fileinfo.com/extension/smil|publisher=FileInfo.com|title=.SMIL File Extension|accessdate=11 April 2011}}</ref>
* .vlc：為[[VLC多媒體播放器|VLC多媒體播放器]]所採用<ref>{{cite web|url=https://wiki.videolan.org/.VLC/|publisher=videolan.org|title=.vlc File Extension|accessdate=16 May 2014}}</ref>
* [[Windows_Media_Player_Playlist|.wpl]]：為[[微軟|微軟]][[Windows_Media_Player|Windows Media Player]]9-12版本所採用<ref name="microsoft-formats">{{cite web|url=http://support.microsoft.com/kb/316992|publisher=Microsoft|title=Information about the Multimedia file types that Windows Media Player supports|date=November 17, 2010|work=Microsoft Knowledge Base|accessdate=11 April 2011}}</ref>
* [[XML_Shareable_Playlist_Format|.xspf]]：以[[XML|XML]]製成，可供分享<ref>{{cite web|url=http://xspf.org/xspf-v0.html|title=The XSPF Playlist Format, version 0|publisher=The Xiph Open-source Community|accessdate=11 April 2011}}</ref>
* .zpl：為[[Zune|Zune]]可攜式媒體播放器所採用
