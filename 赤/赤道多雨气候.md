'''赤道多雨氣候'''，全年'''高溫多雨'''。[[热带雨林气候|热带雨林气候]]的一种子类型，在對應的[[柯本氣候分類法|柯本氣候分類法]]中代號為「Af」。根據[[三圈環流|三圈環流]]，由於[[地球|地球]][[赤道|赤道]]地區受[[赤道低壓帶|赤道低壓帶]]控制盛行上升[[氣流|氣流]]，而導致降水充沛。

== 成因 ==
[[大气环流|大气环流]]：[[信风带|信风带]]在此辐合形成[[赤道低压带|赤道低压带]]上升，盛行[[赤道气团|赤道气团]]<br />
[[下垫面|下垫面]]：[[海洋|海洋]]面积宽广，[[热带雨林|热带雨林]]广布<br />
[[太阳辐射|太阳辐射]]：地处赤道区域，[[太阳高度角|太阳高度角]]大，基本[[昼夜平分|昼夜平分]]<br />

== 分布 ==
大势：位于赤道区域，大致伸至[[纬度|纬度]]10度左右。<br />
具体地区：[[亞馬遜平原|亞馬遜平原]]、[[馬來群島|馬來群島]]、[[馬來半島|馬來半島]]南部、[[菲律賓群島|菲律賓群島]]南部、[[剛果盆地|剛果盆地]]、[[幾內亞灣|幾內亞灣]]北岸。

== 特征 ==
# 长[[夏|夏]]无[[冬|冬]]，无[[季相|季相]]变化。[[最冷月|最冷月]]在18[[摄氏度|摄氏度]]以上，[[最热月|最热月]]一般不超过38摄氏度，全年皆夏。
# 气温[[年较差|年较差]]小于[[日较差|日较差]]。年较差一般小于3摄氏度，而平均日较差最多可达12摄氏度。
# 全年多雨，无[[干季|干季]]。[[年雨量|年雨量]]在2000毫米以上。降水多在午后，多[[对流雨|对流雨]]，一年中有75至150天有[[雷阵雨|雷阵雨]]。
# [[天气|天气]]单调。全年受赤道海洋气团控制，[[风|风]]力微弱，季相无变化，基本每日上午闷热晴朗，接着[[积云|积云]]越积越厚，午后则[[暴雨|暴雨]]倾盆，雨后天气稍凉，第二天又复如此。
