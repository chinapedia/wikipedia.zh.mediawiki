'''赤腹鹰'''（[[学名|学名]]：{{lang|la|''Accipiter soloensis''}}）为[[鹰科|鹰科]][[鹰属|鹰属]]的[[鸟类|鸟类]]。夏季时在[[中国|中国]]东南部、[[台湾|台湾]]、[[韩国|韩国]]和[[西伯利亚|西伯利亚]]地区繁殖，冬季迁徙到[[菲律宾|菲律宾]]和[[印尼|印尼]]。在[[中国大陆|中国大陆]]，分布于[[广东|广东]]、[[海南|海南]]、[[四川|四川]]、[[陕西|陕西]]、[[河南|河南]]、[[河北|河北]]、[[安徽|安徽]]、[[江西|江西]]、[[浙江|浙江]]、[[江苏|江苏]]、[[福建|福建]]、[[贵州|贵州]]、[[广西|广西]]、[[河北|河北]]、[[山东|山东]]等地，一般栖息于山林。该物种的模式产地在印度尼西亚爪哇<ref name='id1485'/>。

== 習性 ==
* 保育等級：II級，珍貴稀有野生動物。
* 生息狀態：普遍的過境鳥。
* 偏好開闊森林環境。以[[蛙|蛙]]類、[[昆蟲|昆蟲]]、[[蜥蜴|蜥蜴]]等為食，也會捕抓小型鳥類。
* 在台灣地區赤腹鷹為普遍的過境鳥，秋季9月初即可見到，比[[灰面鵟鷹|灰面鵟鷹]]較早南遷，飛行路徑集中於[[中央山脈|中央山脈]]山區、東部[[海岸山脈|海岸山脈]]一帶，西部山區數量較少，群聚於[[恆春鎮|恆春]][[墾丁|墾丁]]過境至[[菲律賓|菲律賓]]。春季4、5月的遷移過境時期，又比灰面鵟鷹較晚北返，此時飛行路徑集中於西部山區，以中部[[八卦山|八卦山]]最為著名。山區天空可見數十隻甚至數百隻集結成群飛行，有時一天之內數量達數千隻之多。會在山區利用熱氣流盤旋上升至一定高度後，滑翔至下一個山頭，由於翼展並非寬廣，利用氣流的效益有限，因此可常見其振翅飛行。
* 繁殖期會發出一連串音調下降而快速帶有尖銳的鼻音似笛聲的鳴聲，其他時間並不常聽聞其鳴聲。

== 特徵 ==
* 體長約30公分的小型猛禽。
* 雌、雄鳥體色大致相同。
* 翼短而尖，後緣平直，尾羽短，有數條黑色橫斑。
* 成鳥頭部至背面灰黑色，胸腹部橙色，翼尖黑色，飛羽外緣黑色形成一條細黑帶，翼下及腹面沒有斑紋。
* 幼鳥及亞成鳥背面褐色至灰褐色，胸有縱紋，腹為心型斑及橫斑。雄鳥眼睛紅色、雌鳥黃色<ref name="culture_Accipiter_soloensis"/>。

== 分布 ==
* 族群：繁殖於東北亞及中國華中、華南，冬季南遷至菲律賓、[[東南亞|東南亞]]、[[新幾內亞|新幾內亞]]一帶地區度冬。
* 台灣為其過境地區，春、秋兩季過境時期數量龐大，2004年秋過境期在[[墾丁|墾丁]]共記錄到超過22萬隻以上的數量，為迄為止所觀察到的最大數量，為台灣過境候鳥數量最多的猛禽。<ref name="culture_Accipiter_soloensis" />

== 相似鳥種 ==
[[松雀鷹|松雀鷹]]雄鳥飛行時可見翼下有橫紋、翼尖無黑色，雌鳥胸、腹部及翼下滿佈橫紋。[[灰面鵟鷹|灰面鵟鷹]]飛行時翅膀較為狹長，頭部有眉線<ref name="culture_Accipiter_soloensis"/>。

== 生態威脅 ==
人為干擾以及棲地改變應是其主要生存危機<ref name="culture_Accipiter_soloensis"/>。
