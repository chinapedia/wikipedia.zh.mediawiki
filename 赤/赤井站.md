'''赤井站'''（{{jpn|j=赤井駅|hg=あかいえき|rm=Akai eki|f=y}}）是一個位於[[福島縣|福島縣]][[磐城市|磐城市]]{{link-ja|平 (磐城市)|平 (いわき市)|平}}赤井字田中，屬於[[東日本旅客鐵道|東日本旅客鐵道]]（JR東日本）[[磐越東線|磐越東線]]的[[鐵路車站|鐵路車站]]<ref name="eki">{{Cite web |url=http://www.jreast.co.jp/estation/station/info.aspx?StationCd=28 |title=JR東日本：各駅情報(赤井駅)  |publisher=東日本旅客鉄道 |accessdate=2014-10-29}}</ref>。
== 車站結構 ==
[[側式月台|側式月台]]1面1線的[[地面車站|地面車站]]<ref name="zeneki">{{Cite journal| |date = 2013-08-04 |journal = 週刊JR全駅・全車両基地 |issue = 50 |page = 26 |publisher = 朝日新聞出版  |accessdate = 2014-10-29}}</ref>。截至1986年為止曾經設有[[島式月台|島式月台]]1面2線<ref name="zeneki"/>。
