'''赤平市'''（{{jpn|j=赤平市|hg=あかびらし|rm=Akabira shi|f=y}}）是位于日本[[北海道|北海道]]中部的城市，屬[[空知綜合振興局|空知綜合振興局]]，主要市區主要位於{{Tsl|ja|空知川}}流域，除空知川沿岸區域外接為山區。

過去在煤礦產業最盛的1960年，人口數曾經達到59,430人<ref name="brief">{{cite web |url=http://www.city.akabira.hokkaido.jp/sidemenu/shisei.html |title=市勢概要 |publisher=赤平市政府 |accessdate=2008年2月12日 |language=ja |deadurl=yes |archiveurl=https://www.webcitation.org/67Sf6efdx?url=http://www.city.akabira.hokkaido.jp/sidemenu/shisei.html |archivedate=2012年5月6日 }}
</ref>，但是之後隨著煤礦產業的衰退使得人口開使減少，目前人口僅有一萬人左右，是全日本人口第四少的市級行政區劃；目前正致力發展以煤礦產業遺跡為主的觀光旅遊業。

城市名稱源自[[阿伊努語|阿伊努語]]的「aka-pira」，意思是山稜的崖。

==歷史==

在1891年開始有[[和人|和人]]到達本地進行開墾，最初在1906年設立歌志內村（現在的[[歌志內市|歌志內市]]），本地一同被劃入歌志內村，之後由於發現煤礦，在1918年起開始開採煤礦，帶動本地的發展，隨著人口的成長在1922年自歌志內村分出獨立設置為'''赤平村'''<ref name="brief" />。

此後在1930年代陸續有新的煤礦開始開採，赤平村也隨著發展在1943年2月升格改制為'''赤平町'''，1954年7月升格改制為'''赤平市'''，在當時是為北海道第18個城市。

但此後煤礦產業開始萎縮，1960年代起各個煤礦陸續停止開採，最後一座煤礦礦場也在1994年停止營運。

== 教育 ==

過去在繁榮時期轄內最多曾有4所高中、8所中學校、10所小學校；但隨著人口減少，多所學校陸續遭到整併，目前僅轄內已無任何高中，僅餘中學校2所，小學3所。
