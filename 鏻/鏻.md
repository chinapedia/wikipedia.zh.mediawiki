在[[化學|化學]]中，'''鏻'''，又稱'''鏻離子'''或'''磷鎓离子'''<ref name="phosphonium,terms.naer.edu">{{Cite Web | url = http://terms.naer.edu.tw/detail/2293725/?index=5 | title = 鏻離子 phosphonium ion | website = 國家教育研究院 | accessdate=2017-07-21}}</ref>，是由[[膦|膦]]（或[[磷化氫|磷化氫]]）分子衍生出的正一[[化合价|价]]、帶1個正電荷的[[離子|離子]]。是[[磷化氫|磷化氫]]分子，由磷提供[[孤電子對|孤電子對]]而被[[質子化|質子化]]（與[[氫離子|氫離子]][[配位|配位]]結合）後產生的正離子，其化學式為{{化學式|磷|氫|4|1}}<ref name="pubchem">{{Cite Web | url = 
https://pubchem.ncbi.nlm.nih.gov/compound/phosphonium | title = phosphonium | website = pubchem | accessdate=2017-07-21}}</ref>。其在化學反應中有類似於金屬離子的特性。

== 命名 ==
鏻離子是[[磷化氫|磷化氫]]分子[[質子化|質子化]]（與氫離子配位結合）所形成的[[鎓離子|鎓離子]]。由於化學性質類似於金屬離子，因此其名稱來自「磷」與金屬的組合。

鏻的英文名稱「Phosphonium」<ref name="www.zdic.net"/><ref name="phosphonium,terms.naer.edu"/>來自於「Phosphine」（[[膦|膦]]）和「-ium」組成，類似於銨的命名，其也加上了表示金屬元素的字尾「-ium」。而中文則是將「膦」的偏旁改成[[金部_(部首)|金部]]，變為「鏻」，做為這種陽離子的名稱<ref name="phosphonium,terms.naer.edu"/>。由于化学性质类似于[[金属|金属]][[离子|离子]]，這也是這種離子被命名為“鏻”的原因之一。而鏻（phosphonium）本身也代表化學式為{{Chem|PR|4|+}}的多原子陽離子<ref>{{cite book|first=D. E. C. |last=Corbridge |title=Phosphorus: An Outline of its Chemistry, Biochemistry, and Technology |edition=5th |publisher=Elsevier |location=Amsterdam |date=1995 |isbn=0-444-89307-5}}</ref><ref name="www.zdic.net">{{Cite Web | url = http://www.zdic.net/z/26/js/93FB.htm | title = “鏻”字的基本信息 | website = 漢典 | accessdate=2017-08-14}}</ref>
。而'''鏻鹽'''（phosphonium salt）則代表含有鏻離子（{{化學式|P|H|4|1}}）的鹽，例如碘化鏻（{{化學式|磷|氫|4|1|碘|1|-1}}）<ref>{{Cite Web | url = https://pubchem.ncbi.nlm.nih.gov/compound/Phosphonium_iodide#section=Top | title = Phosphonium iodide   | website = pubchem | accessdate=2017-08-07}}</ref>或者含有四價磷的相關有機衍生物，如季鏻鹽（四級鏻鹽），常見的例子有{{link-en|氯化四苯基鏻|Tetraphenylphosphonium_chloride}}（(C<sub>6</sub>H<sub>5</sub>)<sub>4</sub>P<sup>+</sup> Cl<sup>−</sup>）<ref>{{Cite Web | url = https://pubchem.ncbi.nlm.nih.gov/compound/164911 | title = Tetraphenylphosphonium chloride  | website = pubchem | accessdate=2017-08-07}}</ref>和碘化四甲基鏻（[P(CH<sub>3</sub>)<sub>4</sub>]<sup>+</sup>I<sup>−</sup>）<ref>{{Cite Web | url = https://pubchem.ncbi.nlm.nih.gov/compound/120511#section=Top | title = Tetramethylphosphonium iodide | website = pubchem | accessdate=2017-08-07}}</ref>。而在部分字典中，鏻則表示所有含磷的有機化合物的總稱<ref name="dict.variants.moe.edu.tw">{{Cite Web | url = http://dict.variants.moe.edu.tw/yitib/frb/frb05441.htm | title = B05441 【鏻】 | website = [[中華民國教育部|教育部]]異體字字典 | accessdate=2017-07-21}}</ref>。

== 結構 ==
鏻與[[銨|銨]]、[[鉮|鉮]]類似，都是[[氮族元素的氫化物|氮族元素的氫化物]]被質子化後的產物，其皆擁有相似的分子結構，為正四面體型的分子，並且與甲烷互為[[等電子體|等電子體]]。

== 衍生物 ==
*[[季𬭸鹽|季𬭸鹽]]：鏻的氫原子被取代的衍生物
*:直接含有{{化學式|磷|氫|4|1}}離子的鹽十分少見，但這種離子是製備有工業上用途的{{link-en|四羥甲基氯化鏻|tetrakis(hydroxymethyl)phosphonium chloride}}時所需要的[[反應中間體|反應中間體]]<ref name=Ullmann>Svara, Jürgen; Weferling, Norbert ; Hofmann, Thomas. Phosphorus Compounds, Organic. Ullmann's Encyclopedia of Industrial Chemistry.  John Wiley & Sons, Inc, 2008 {{doi|10.1002/14356007.a19_545.pub2}}</ref>：
*::PH<sub>3</sub> + HCl + 4 CH<sub>2</sub>O → {{chem|P(CH|2|OH)|4|+|Cl|−}}
*:有機鏻鹽是實驗室中的常用試劑。具有P-H鍵有機鏻鹽可透過膦的質子化製備：
*::PR<sub>3</sub> + H<sup>+</sup> → {{chem|HPR|3|+}}
*:許多有機季鏻陽離子（{{Chem|PR|4|+}}）是透過有機膦的烷基化來製備。甲基三苯基碘化鏻，即[[维蒂希反应#维蒂希试剂|维蒂希試劑]]的前體，就是利用將[[三苯基膦|三苯基膦]]與[[碘甲烷|碘甲烷]]反應來生成：
*::PPh<sub>3</sub> + CH<sub>3</sub>I → {{chem|CH|3|PPh|3|+|I|−}}
*:而四苯基鏻陽離子（{{chem|PPh|4|+}}）是一種有用的沉澱劑，其效果與用於[[相转移催化剂|相轉移催化]]的季銨鹽類似。
**[[碘化四苯基鏻|碘化四苯基鏻]]（化学式：(C{{sub|6}}H{{sub|5}}){{sub|4}}PI）
*[[碘化鏻|碘化鏻]]（化學式：{{化學式|磷|氫|4|碘}}）
