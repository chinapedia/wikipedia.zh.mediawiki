'''墨西拿海峡'''（{{lang-it|'''Stretto di Messina'''}}）是[[意大利|意大利]][[西西里|西西里]]岛和[[卡拉布里亚|卡拉布里亚]]之间的[[海峡|海峡]]。它连接了[[伊奥尼亚海|伊奥尼亚海]]和[[第勒尼安海|第勒尼安海]]，长度32[[公里|公里]]，宽度3至8[[公里|公里]]。位于[[西西里|西西里]]东北部的[[墨西拿|墨西拿]]是海峡上最重要的港口。

== 交通 ==
[[File:Stretto_di_messina_incisione.jpg|thumb]]
轮渡往返于墨西拿和卡拉布里亚的[[维拉圣焦万尼|维拉圣焦万尼]]。在墨西拿和[[雷焦卡拉布里亚|雷焦卡拉布里亚]]（Reggio Calabria）之间还有水翼船服务。

== 穿过海峡的高压电线路 ==
1957年在墨西拿海峡之间架设了220千伏的高架电线。其高压电线架是世界最高的之一。电线后来被海底电缆替代，但高压电线架被保留下来，并因其纪念意义受到保护。

== 大桥计划 ==
{{Main|墨西拿海峽大橋}}
在过去的几十年间不断有人提出计划在墨西拿海峡上建造大桥以连接西西里和意大利内陆，但是由于经济原因和该地区属地震高发地带计划始终难以实施。

2005年10月13日意大利议会批准了[[贝卢斯科尼|贝卢斯科尼]]政府和意大利建筑商''Impregilo''之间的一项大桥建筑合同。如果大桥建成它将是世界最大的单拱[[悬索桥|悬索桥]]，单拱跨度3300[[米|米]]。工程将于2006年开工，计划于2012年完工，工程造价预计将达到60亿[[欧元|欧元]]。

== 候鸟迁徙的集中地 ==
每年四五月间都有多达三万只左右的[[隼形目|隼]]鸟和[[鹳科|鹳]]鸟从北欧的[[繁殖地|繁殖地]]飞过墨西拿海峡。直到20世纪90年代在海峡两岸还有很多非法捕鸟现象。经过自然保护者和意大利森林警察的多年努力，到今天已经基本杜绝了对[[候鸟|候鸟]]的非法捕猎。
