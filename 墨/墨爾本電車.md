'''墨爾本电车'''为[[澳洲|澳洲]][[墨爾本|墨爾本]]的[[路面電車|路面電車]]系統，截至2017年，整个系统由250km长的轨道，493列列车，24条线路和1763个车站组成。以长度计算，墨尔本电车为全球最大的有轨电车系统。此系统也和[[香港輕鐵|香港轻铁]]的发展颇有渊源。

== 车辆 ==
目前共有7种型号的列车正在运行，其中每个型号还分为不同的子型号

=== W级墨尔本电车 ===
{{Main article|{{tsl|en|W-class Melbourne tram}}}}
W级电车最早于1923年作为新的标准设计被引入，这级电车拥有两个转向架以及特别的“drop centre”部分，使得电车中间的车门可以更加靠近地面。W级电车在60年的时间内都是墨尔本电车系统的主力车辆，一共有分为12个变种合计752辆被建造出来，最后一辆则于1956年建造。现在W级电车已经不在日常运营中使用，除了少部分经过改造翻新后运行在35路城市环线和电车餐厅上。

=== Z级墨尔本电车 ===
{{Main article|{{tsl|en|Z-class Melbourne tram}}}}
Z级电车建造的主要目的是为了替代老化的W级，由Comeng于1975年开始生产。1975年至1978年首先生产了100列电车，被命名为Z1级。之后在1978年至1979年又生产了15列和Z1级非常相似，只有细微外观变化的Z2级。由于Z1级和Z2级的种种问题，1979年至1983年制造了115列经过大幅改进的Z3级。Z3的改进主要有：在每侧增加了一扇车门、车窗打开方式改为下拉式，目的地显示器更换、更平稳的加减速以及改进的悬挂。三种型号中，Z1级和Z2级均已于2016年退役封存，Z3级截至2018年10月仍有111列尚在服务中。编号为116-232 <ref>{{Cite web|url=http://vicsig.net/trams/class/Z1|title=Z1 Class - VICSIG|accessdate=2018-04-01|author=|date=|work=vicsig.net|publisher=}}</ref><ref>{{Cite web|url=http://vicsig.net/trams/class/Z2|title=Z2 Class - VICSIG|accessdate=2018-04-01|author=|date=|work=vicsig.net|publisher=}}</ref><ref>{{Cite web|url=http://vicsig.net/trams/class/Z3|title=Z3 Class - VICSIG|accessdate=2018-04-01|author=|date=|work=vicsig.net|publisher=}}</ref>

=== A级墨尔本电车 ===
{{Main article|{{tsl|en|A-class Melbourne tram}}}}
A级电车于1984年至1986年间建造，其中1983年至1985年建造了28列A1级，1985到1987年建造了42列A2级。A1级和A2级几乎完全一样，区别在于A2级使用了性能更好的刹车以及出厂时就使用了[[受电弓|受电弓]]而不是A1级出厂时的[[受电杆|受电杆]]，A2也是墨尔本第一种使用受电弓的电车，不过A1级在之后也已安装了受电弓。编号为233-300 <ref>{{Cite web|url=http://vicsig.net/trams/class/A1|title=A1 Class - VICSIG|accessdate=2018-04-01|author=|date=|work=vicsig.net|publisher=}}</ref><ref>{{Cite web|url=http://vicsig.net/trams/class/A2|title=A2 Class - VICSIG|accessdate=2018-04-01|author=|date=|work=vicsig.net|publisher=}}</ref>

=== B级墨尔本电车 ===
[[墨尔本B型电车|墨尔本B型电车]]最初于1984年随着B1级原型车推出，B1级共建造了两列，第二列于1985年建造。B级电车使用和Z3以及A级电车相同的牵引设备，同时特别为了轻轨线路而建造。B级电车于1988年开始量产，直到1994年结束，共有130列由Comeng（之后变为ABB交通）制造，定名为B2级。B2级也是墨尔本第一种拥有[[空氣調節|空调]]设备的电车。目前所有B2级电车都仍在运营，而B1级已经于2017年11月在全电车网络进行告别运行后退役。编号为2003-2132 <ref>{{Cite web|url=http://vicsig.net/trams/class/B1|title=B1 Class - VICSIG|accessdate=2018-04-01|author=|date=|work=vicsig.net|publisher=}}</ref><ref>{{Cite web|url=http://vicsig.net/trams/class/B2|title=B2 Class - VICSIG|accessdate=2018-04-01|author=|date=|work=vicsig.net|publisher=}}</ref>

=== C级墨尔本电车（Citadis） ===
{{Main article|{{tsl|en|C-class Melbourne tram}}|{{tsl|en|C2-class Melbourne tram}}}}
随着墨尔本电车系统的私有化，私有运营商接收了新的电车来替换原来的Z级电车。2001年，Yarra Tram引进了低地板的C级电车，阿尔斯通在法国制造的{{tsl|en|Alstom Citadis||Citadis}}系列的一个变种。C级电车为3车厢设计，共生产了36列。<ref>{{Cite web|url=http://vicsig.net/trams/class/C/1|title=C Class - VICSIG|accessdate=2018-04-01|author=|date=|work=vicsig.net|publisher=}}</ref>C级电车为墨尔本本地的命名，根据阿尔斯通的命名为Citadis 202。<br/>5列C2级电车于2008年引进，这5列电车是从法国米卢斯租借的，被命名为“Bumble Bees”，由于他们独特的黄色涂装以及只在96路上行驶。在2010年，州政府宣布正在商讨购买5列C2级，购买已经在2013年完成。C1型编号为3001-3036，而C2级编号为5103,5106,5113，5116,和5123 <ref>{{Cite web|url=http://vicsig.net/trams/class/C2|title=C2 Class - VICSIG|accessdate=2018-04-01|author=|date=|work=vicsig.net|publisher=}}</ref><ref>{{Cite web|url=https://www.theage.com.au/national/victoria/bee-trams-to-stay-but-at-what-price-20101012-16huk.html|title=Bee trams to stay, but at what price?|accessdate=2018-04-01|date=2010-10-12|last=Lucas|first=Clay|work=The Age|language=en}}</ref><ref>{{Cite web|url=http://ptv.vic.gov.au/projects/trams/tram-procurement-program/|title=Tram Procurement Program - Public Transport Victoria|accessdate=2018-04-01|date=2013-05-13|deadurl=yes|archiveurl=https://web.archive.org/web/20130513161451/http://ptv.vic.gov.au/projects/trams/tram-procurement-program/|archivedate=2013-05-13}}</ref>C2为墨尔本本地的命名，根据阿尔斯通的命名为Citadis 302。

=== D级墨尔本电车（Combino） ===
{{Main article|{{tsl|en|D-class Melbourne tram}}}}
类似于C级电车，由[[西门子交通|西门子]]制造的D级电车也是墨尔本电车系统新一代的低地板电车，由现在已经解散的M>Tram（当时电车系统除Yarra Tram外的另一家运营商）于2002年至2004间引进。D级电车有两种子型号，分别为3车厢的D1级和5车厢的D2级。<ref>{{Cite web|url=http://vicsig.net/trams/class/D1|title=D1 Class - VICSIG|accessdate=2018-04-01|author=|date=|work=vicsig.net|publisher=}}</ref><ref>{{Cite web|url=http://vicsig.net/trams/class/D2|title=D2 Class - VICSIG|accessdate=2018-04-01|author=|date=|work=vicsig.net|publisher=}}</ref>和早一年投入使用的同样是低地板设计的C级和2008年投入服务的设计类似的C2级相比，D级电车的车内空间以及座椅空间较小，在C级电车上2个座椅的横排座椅在D级电车上只能安装一个。D级电车为墨尔本本地的编号，D级属于西门子{{tsl|en|Siemens Combino||Combino}}系列。D1编号为3501-3538，而D2编号为5001-5021.

=== E级墨尔本电车（Flexity） ===
{{Main article|E级墨尔本电车}}
E级电车为墨尔本电车系统最新的电车型号，使用了低地板设计并由3个车厢组成，由[[龐巴迪運輸|庞巴迪]]在位于墨尔本丹德农的工厂制造，这也是自1990年代B级电车以来第一款在墨尔本本地制造的电车，而电车的转向架和动力系统则由是在庞巴迪位于德国[[曼海姆|曼海姆]]和[[锡根|锡根]]的工厂制造。庞巴迪于2010年9月在竞标中被选中制造50列新一代低地板电车，同时合同也包括了到2017年的车辆维护以及进一步100列电车的选择权。编号为6打头的4位数字，由于交付并未完成，编号上限尚未确定。<ref>{{Cite web|url=https://www.bombardier.com/en/media/newsList/details.760-bombardier-wins-contract-for-50-trams-for-one-of-the-worlds-largest-tram-operations-in-melbourne-australia.bombardiercom.html|title=Bombardier Wins Contract for 50 Trams for One of the World's Largest Tram Operations in Melbourne, Australia - Bombardier|accessdate=2018-04-01|work=www.bombardier.com|language=en}}</ref><ref>{{Cite web|url=http://www.yarratrams.com.au/media-centre/news/articles/2012/melbourne%E2%80%99s-newest-tram-%E2%80%93-the-story-so-far|title=Yarra Trams - Melbourne’s newest tram – the story so far|accessdate=2018-04-01|last=Agency|first=Helsinki|work=www.yarratrams.com.au|language=en|archive-url=https://web.archive.org/web/20180407200839/http://yarratrams.com.au/media-centre/news/articles/2012/melbourne%E2%80%99s-newest-tram-%E2%80%93-the-story-so-far/|archive-date=2018-04-07|dead-url=yes}}</ref><ref>{{Cite web|url=http://www.yarratrams.com.au/media-centre/news/articles/2010/fifty-new-trams-for-melbourne's-network/|title=Yarra Trams - Fifty new trams for Melbourne's network|accessdate=2018-04-01|last=Agency|first=Helsinki|work=www.yarratrams.com.au|language=en|archive-url=https://web.archive.org/web/20180407203656/http://yarratrams.com.au/media-centre/news/articles/2010/fifty-new-trams-for-melbourne's-network/|archive-date=2018-04-07|dead-url=yes}}</ref><br/>E级电车于2013年11月底首先在96路上投入服务，随后扩展到11路和86路。在2017年3月州政府宣布之前增加的20列订单将同时对电车的设计进行升级，使电车的安全性得到提升。<ref name=":0">{{Cite web|url=https://www.theage.com.au/national/victoria/safer-sleeker-trams-ready-to-roll-in-wake-of-sharp-rise-in-injuries-20170320-gv2aup.html|title=Safer, sleeker trams ready to roll in wake of sharp rise in injuries|accessdate=2018-04-01|date=2017-03-20|last=Carey|first=Adam|work=The Age|language=en}}</ref>这些电车被命名为E2级，和E级最明显的区别是驾驶室两侧的立柱面积显著减小，不过有一些因为事故受损的E级电车在修复后也采用了E2级的设计。更新的设计主要增加了扶手、驾驶室挡风玻璃面积和降低挡风玻璃的眩光，以降低乘客摔倒的概率以及更好的使司机观察交通和乘客上下车的情况。<ref name=":0" />截至2017年3月，全部50列E级电车均已投入运营，20列E2级中12列已经投入服务。<ref>{{Cite web|url=http://vicsig.net/trams/class/E/1|title=E Class -
