'''林毓豪'''（{{bd|1940年||1997年||}}），[[海南省|海南省]][[樂東縣|樂東縣]][[黃流鎮|黃流鎮]]人，[[中國|中國]][[雕塑家|雕塑家]]、[[畫家|畫家]]。

受時代影響，林毓豪的藝術語言深受[[俄羅斯|俄羅斯]][[藝術|藝術]]及[[印象派|印象派]]所影響，他沉迷於[[雕塑|雕塑]]各行各業的人物肖像之中，大量地塑造源自普羅百姓的藝術形象，為他後來所創作的紀念性題材的大型公共雕塑作品奠定了基礎。除雕塑之外，林毓豪還涉足[[水粉畫|水粉畫]]、[[水彩畫|水彩畫]]、[[國畫|國畫]]、[[版畫|版畫]]、[[油畫|油畫]]等其他藝術門類。

林毓豪創作並落成的大、中型公共紀念性雕塑主要分佈在中國的[[陝西|陝西]]、[[南京|南京]]、[[廣東|廣東]]、海南等省市地區，代表作有《南京[[雨花臺|雨花臺]]烈士紀念碑》群像原稿、《鹿回頭》、《[[黃花崗|黃花崗]]自由神像》、《新花》、《[[李綱|李綱]]像》、《[[孫中山|孫中山]]胸像》、《鑒真登岸》、《[[黃河|黃河]]英魂冼星海》、《[[張雲逸|張雲逸]]將軍》、《省港罷工》、《智慧女神》等。水彩畫代表作有《生命之詩》、《石體浮想》、《自然》、《江河》等。他最為突出的公共雕塑作品：其一，是1975年－1977年所創作的《南京雨花臺紀念碑》群像，現稱《先驅者》，該塑像至今立於[[江蘇省|江蘇省]]南京市雨花臺陵園內；其二，是海南建省前夕為[[三亞市|三亞市]]而塑造的《鹿回頭》，該作品現今已經成為三亞市的城標。1980年7月，[[廣東電視台|廣東電視台]]的「文化生活」專欄節目播出《雕塑新人—— 林毓豪的頭像雕塑》，介紹了他從60年代到70年代所塑造的100多件肖像雕塑作品。

== 生平 ==
1940年9月5日，林毓豪出生於中國[[海南|海南]]（即現在的海南省）[[黃流鎮|黃流鎮]]一個貧窮的家庭。後來，由於戰爭，舉家逃難到海南省[[三亞市|三亞市]]。

林毓豪自幼就對[[美術|美術]]充滿熱愛，他曾在自傳裡寫道：“[[海南島|海南島]]美麗的自然環境養育了我追求[[藝術|藝術]]的熱情。我在[[沙灘|沙灘]]上畫畫，在菸盒或所有撿來的小紙片上畫畫，在學校的壁報上畫畫。只要能讓我畫，就感到無比的幸福。”孩童時期的他每每跟隨祖父去廟宇拜祭後，會隨手抓起泥土臨摹廟內供奉的佛像。他的藝術天賦在中學時期更得以全面展現，[[繪畫|繪畫]]、[[唱歌|唱歌]]、[[吹奏樂器|吹奏樂器]]、[[體操|體操]]，樣樣俱通。

1959年，林毓豪考入[[廣州美術學院|廣州美術學院]]雕塑系學習。在學期間，他的雕塑作品被選送省、市美術展覽展出；1964年畢業後，他進入廣東省工藝美術研究所工作；1970年被廣州雕塑院吸納，成為職業雕塑家。1992年因病離職；1997年離世。 

林毓豪生前是[[中國美術家協會|中國美術家協會]]會員、中國雕塑學會會員、廣東省美術家協會理事、廣州粵海書畫研究會副會長、廣州海日書畫研究會顧問、廣州市文聯委員、廣州城市雕塑藝委會委員、[[海南大學|海南大學]]客座教授。他的雕塑作品被載入《世界雕塑全集》、《中國當代美術系列》的“雕塑與環境藝術”和《中國美術史當代卷》、《中國現代美術家大辭典》的“美術辭林雕塑卷”；其個人藝術活動資料和作品被編入《中國高級專業技術人才辭典》、《中國當代美術家書法家大辭典》、《MEN OF ACHIEVEMENT1997》、《中華勞模大典》、《海南名人辭典》、《國際現代書畫篆刻家大辭典》等。

1981年，林毓豪被授予[[廣州市|廣州市]]勞動模範稱號，出席該年度的廣州市先進工作者代表大會並在大會上發言；1982年參加廣州市先進工作者代表大會及廣東省先進工作者代表大會，被評為[[廣東省|廣東省]]先進工作者，以及當選為廣東省六屆人民代表大會代表和大會主席團成員； 1983年度再度次被授予廣州市勞動模範光榮稱號，並當選為[[民盟|民盟]]廣州市委委員。   

2017年，在林毓豪去世20年之後，廣東省美術館和[[海南省博物館|海南省博物館]]分別為他舉辦名為《質樸與溫度－林毓豪回顧展》。

== 主要作品 ==
* 《南京雨花臺烈士紀念碑》原稿，[[江蘇省|江蘇省]][[南京市|南京市]]。（1975年－1977年）
* 《鹿回頭》花崗岩，高12米、寬10米、厚5米，全高15米，海南省三亞市。（1982年－1989年）
* 《廣州黃花崗自由女神像》高3.4米，廣東省廣州市。（1981年）
* 《孫中山胸像》漢白玉，高2米，廣東省[[中山市|中山市]]。（1982年）
* 《黃河英魂——冼星海》漢白玉，高2.4米，廣東省廣州市。（1983年）
* 《省港大罷工》高3.2米，寬3.3米，廣東省廣州市烈士陵園。（1985年）
* 《李綱像》花崗岩，4.2米高，海南省[[萬寧市|萬寧市]]。（1985年－1989年）
* 《鑒真登岸》花崗石，高9米，海南省三亞市。（1992年－1993年）

== 個人展覽 ==
* 1981年812月15日－1982年1月3日：美術家協會廣東分會聯合廣州雕塑院室舉辦 《林毓豪雕塑繪畫作品展覽》，展出作品193件
* 1982年7月22日至8月24日：海南省海口市舉辦《林毓豪個人作品展》，展出作品134件；
* 1982年9月1日至10月5日：廣東省、廣州市總工會在廣州市第一工人文化宮舉辦《林毓豪雕塑繪畫作品展》，展出作品154件。
* 2017年2月21日至3月26日：廣東美術館主辦《質樸與溫度－林毓豪回顧展》，展出雕塑作品86件，繪畫作品88件，文獻資料18件;
* 2017年4月18日至5月21日：中共海南省委宣傳部、海南省文化廣電出版體育廳、海南省文學藝術界聯合會主辦的《質樸與溫度－林毓豪回顧展》展出雕塑作品76件，繪畫作品68件，文獻資料20件，紀錄片。

== 個人榮譽 ==
* 1988年，作品及資料編入《中國當代美術系列》的“雕塑與環境藝術”和《中國美術史當代卷》、《中國現代美術家大辭典》的“美術辭林 雕塑卷”；
* 1989年，個人藝術活動資料編入《中國美術年鑑 1999》及《海南名人辭典》、《中國當代美術家人名錄》；林毓豪的10件雕塑作品載入《世界雕塑全集》；
* 1993年，個人藝術活動資料編入《海南名人傳略》及《中國現代書畫篆刻界名人錄》；
* 1994年10月，個人簡介傳略載入《國際現代書畫篆刻家大辭典》並獲“世界優秀獎藝術家稱號”；
* 1996年，個人簡介傳略載入《中國高級專業技術人才辭典》、《中國當代美術家書法家大辭典》、《MEN OF ACHIEVEMENT1997》、《中華勞模大典》。
