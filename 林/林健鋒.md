'''林健鋒'''{{GBSHK}}{{MBE}}{{JP}}（{{lang-en|Jeffrey Lam Kin-fung}}，{{bd|1951年|10月23日}}），生於[[香港|香港]]，籍貫[[廣東|廣東]][[南海|南海]]，[[香港立法會|香港立法會]]議員，循[[功能界別|功能界別]]商界（第一），代表[[香港總商會|香港總商會]]自動當選進入立法會，亦為[[香港特別行政區行政會議|行政會議]]成員及[[全國政協委員|全國政協委員]]。

林健鋒本身是[[永和實業|永和實業]]董事長，亦是香港總商會理事，[[香港工業總會|香港工業總會]]理事及[[珠三角工業協會|珠三角工業協會]]名譽會長。2011年他獲[[香港特區政府|香港特區政府]]授予[[金紫荊星章|金紫荊星章]]。<ref name="林健鋒葉劉淑儀任行會議員">{{cite news|url=http://hk.news.yahoo.com/林健鋒葉劉淑儀任行會議員-015805347.html |title=林健鋒葉劉淑儀任行會議員 |newspaper=《[[明報|明報]]》 |date=2012年10月17日 |accessdate=2012-10-22 |deadurl=yes |archiveurl=https://archive.is/20130106080311/http://hk.news.yahoo.com/林健鋒葉劉淑儀任行會議員-015805347.html |archivedate=2013年1月6日 }}</ref>

林健鋒之現時擔任之公職還包括[[西九文化區管理局|西九文化區管理局]]董事局成員、[[公務員薪俸及服務條件常務委員會|公務員薪俸及服務條件常務委員會]]委員、[[關愛基金|關愛基金]]督導委員會成員、[[機場管理局|機場管理局]]成員及[[撲滅罪行委員會|撲滅罪行委員會]]委員<ref name="林健鋒葉劉淑儀任行會議員"/>、[[中國人民政治協商會議全國委員會香港地區委員|第十二屆全國政協委員]]。

林健鋒早年於[[香港島|香港島]][[西營盤|西營盤]]生活，曾就讀[[聖士提反書院附屬小學|聖士提反書院附屬小學]]和[[聖若瑟書院|聖若瑟書院]]。讀至中四時轉到[[美國|美國]]入讀高中，<ref>{{cite web |url=https://eastweek.my-magazine.me/main/3221 |title=變形金剛 林建鋒 |last= |first= |date=2009-9-24 |website= |publisher=東周刊 |access-date= 2019-3-7|quote=}}</ref>並在中學畢業後考入[[美國|美國]][[塔夫斯大學|塔夫斯大學]]（Tufts University）的[[機械工程學|機械工程學]]系<ref name="林健鋒葉劉淑儀任行會議員"/>。林健鋒的[[父親|父親]][[林亮_(企業家)|林亮]]是永和實業[[創辦人|創辦人]]、代理[[變形金剛|變形金剛]]起家。

== 政治聯繫 ==
林健鋒曾為[[自由黨_(香港)|自由黨]]黨員，他在2008年9月18日獲提名接替[[田北俊|田北俊]]任自由黨主席<ref>[http://hk.news.yahoo.com/article/080917/3/89se.html 林健鋒獲提名接掌自由黨]</ref>，但在2008年10月8日突然與[[梁君彥|梁君彥]]、[[梁劉柔芬|梁劉柔芬]]宣佈退黨<ref>[http://hk.news.yahoo.com/article/081008/4/8lvv.html 林健鋒等人退出自由黨]</ref>，主席最終由[[劉健儀|劉健儀]]所擔任。

2012年10月7日，林健鋒與六名建制派立法會議員成立「[[香港經濟民生聯盟|香港經濟民生聯盟]]」，他成為創始聯盟副主席<ref>[http://www.881903.com/Page/ZH-TW/newsdetail.aspx?ItemId=564124&csid=261_341 七名建制派立法會議員組成香港經濟民生聯盟]</ref>。10月17日，林健鋒與[[葉劉淑儀|葉劉淑儀]]作為第二批獲委任加入當屆[[梁振英|梁振英]]政府[[香港特別行政區行政會議|行政會議]]的立法會議員，成為16位非官守議員之一，被指代表著工商界之權益<ref name="林健鋒葉劉淑儀任行會議員"/>。

== 爭議 ==
2014年3月21日，林健鋒在觀塘裁判法院被裁定危險駕駛罪名成立，被判罰款8,000元及停牌6個月。案情指林健鋒在2013年7月1日出席[[香港巨蛋音樂節|香港巨蛋音樂節]]時於九龍灣宏照道與祥業街交界危險駕駛，期間曾經逆線行駛200米及衝紅燈。<ref>[http://orientaldaily.on.cc/cnt/news/20140322/00176_030.html 林健鋒危駕 官斥太猖狂]</ref>

2014年4月17日審計報告批評「盛事基金」撥款「香港龍獅節」向關連組織採購服務卻無申報，涉利益衝突。獲政府注資2.5億元的盛事基金被指成效欠佳及監管混亂，盛事基金評審委員會主席、經民聯立法會議員林健鋒指有人向廉署舉報盛事基金贊助的「香港龍獅節」涉嫌欺詐是「擺政治姿態」，又聲言部分傳媒報道不盡不實。昨向廉署舉報事件的民主黨總幹事林卓廷還撃，斥林健鋒不知反省，竟不批評龍獅節召集人陳鑑林涉嫌欺詐，反而批評傳媒及其舉報，是顛倒是非，逃避監管失職的責任，令人遺憾。<ref>[http://news.sina.com.hk/news/20140419/-101-3243150/1.html 林卓廷批評林健鋒轉移視線] {{webarchive|url=https://web.archive.org/web/20140419231621/http://news.sina.com.hk/news/20140419/-101-3243150/1.html |date=2014-04-19 }}</ref>

2015年12月9日，當林健鋒在立法會大會上正準備就一項口頭質詢發言時，陳志全突然要求點算法定人數，打斷了他的發言。之後，林健鋒在座位上說出了「冚家」的粗言穢語。由於當時座位上的麥克風還未有關掉，故此該詞語在多個立法會直播節目中清楚地廣播。<ref>[http://hk.apple.nextmedia.com/news/art/20151210/19406363 隔牆有耳：點人數累發言受阻林健鋒鬧慢必冚家……]</ref>

==名下馬匹==
林健鋒為[[香港賽馬會|香港賽馬會]]會員，曾與[[陳永棋|陳永棋]]共同擁有馬匹「奇峰」。
