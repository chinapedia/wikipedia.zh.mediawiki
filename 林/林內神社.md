'''林內神社'''為[[臺灣日治時期|臺灣日治時期]]位在[[台南州|台南州]][[斗六郡|斗六郡]]（今[[雲林縣|雲林縣]][[林內鄉|林內鄉]]）的[[神社|神社]]。

== 歷史 ==
林內神社選址在台南州斗六郡斗六街林內（今雲林縣林內鄉公園路、濟公堂與孔廟一帶）<ref>{{Cite book | author = 金子展也 | title = 台湾旧神社故地への旅案内―台湾を護った神々 | location = 東京 | publisher = 神社新報社 | date = 2015-09-01 | pages = 210頁 | ISBN = 978-4-908128-02-8 | language = ja }}</ref>興建，始建於[[昭和|昭和]]14年(1939年)10月，至翌年(1940年)4月竣工<ref>{{Cite web |url = http://www.linnei.gov.tw/tour/index-1.asp?m=13&m1=9&m2=31&gp=136&sid=&keyword=&id=63 |title = 景點地圖導覽-林內神社("「林內神社」又稱「日本神社」.... "章節第2段) |author = 林內鄉公所官方網站 |language=zh-TW }}</ref>，並於昭和15年(1940年)12月20日鎮座，[[近代社格制度|社格]]是無格社。祭祀[[北白川宮能久親王|北白川宮能久親王]]、[[開拓三神|開拓三神]]([[大國魂命|大國魂命]]、[[大己貴命|大己貴命]]、[[少彥名命|少彥名命]])、[[豐宇氣毘賣神|豐受大神]]，例祭日3月3日。神社主體(拜殿、本殿)已於1957年毀壞，改建為林內濟公總堂。殘留下來的一、二之鳥居則被增加綠瓦，其中一之鳥居上的匾額其題字從"林內神社"被改為"林內公園"。近年已重建神橋、手水舍、春日燈籠數對、三之鳥居等。

== [[神社建築|神社建築]] ==
林內神社原配置有[[參道|參道]]、社號標(位於一之鳥居前南側)<ref>{{Cite web |url = https://hualeanman.tian.yam.com/album/10296870#385879188|title = 標註為"IMG_70550.jpg"的圖片上半部有拍攝到原貌|author = 花蓮ㄚ榮的寫真作品集 |date = 2013-04-06 |language=zh-TW }}</ref>、[[鳥居|明神鳥居]]4基(一-三之鳥居是混凝土造，四之鳥居是木造並作為中門<ref name="金子展也 113頁">{{Cite book | author = 金子展也 | title = 台湾旧神社故地への旅案内―台湾を護った神々(注明為"林內公學校職員"的圖片) | location = 東京 | publisher = 神社新報社 | date = 2015-09-01 | pages = 113頁 | ISBN = 978-4-908128-02-8 | language = ja }}</ref>。)，沿途景物多是由老百姓所奉獻捐贈的，有燈籠(大石燈籠6對、混凝土半木造春日燈籠16對)、揭示場(神橋旁北側)、神橋(位於二之鳥居前的樓梯下方)、下乘札(神橋旁南側)<ref>{{Cite web |url = http://www.himoji.jp/database/db04/permalink.php?id=3469|title = 林內神社二之鳥居周圍舊照(淵明中學提供)|author = 海外神社（跡地）に関するデータベース |publisher = 神奈川大學 |language = ja }}</ref>、手水舍、神社建立紀念碑、旗幟台、休憩所、石造[[石獅子|狛犬]]1對、前有[[破風|千鳥破風]]的[[入母屋造|入母屋造]]拜殿<ref name="金子展也 113頁"/>、[[流造|流造]]本殿<ref>{{Cite web |url = http://www.linnei.gov.tw/tour/index-1.asp?m=13&m1=9&m2=31&gp=136&sid=&keyword=&id=63 |title = 景點地圖導覽-林內神社("燈台"章節以下全部) |author = 林內鄉公所官方網站 |language=zh-TW }}</ref>。

建造當時規模是路寬十六尺、石階一六六級，共耗費62,300人力，其工程浩大可見一般。日據時代，建造[[神社|神社]]當時，是由日本政府下令，從級至下級分配職責，由[[保正|保正]](如今村長之位)、[[甲長|甲長]](如今鄰長之位)負責監工，每一戶支援一名義務工，稱為勤勞奉仕，勤勞奉仕必須日以繼夜，血汗建造神社<ref>{{Cite web |url = http://www.linnei.gov.tw/tour/index-1.asp?m=13&m1=9&m2=31&gp=136&sid=&keyword=&id=63 |title = 景點地圖導覽-林內神社("「林內神社」又稱「日本神社」.... "章節第2、4段) |author = 林內鄉公所官方網站 |language=zh-TW }}</ref>。

== 現況 ==
二戰後，政局混亂，神社之遺跡漸遭人破壞或拿走而日漸消失。直至民國四十六年(1957年)左右，因行政機關建設，而將拆除遺留物，經有心人士商榷，決定將匾額「林內神社」改為「林內公園」，使得鳥居(此基具匾額的大鳥居與其後的二之鳥居戰後加裝琉璃屋瓦)得以保留下來<ref>{{Cite web |url = http://www.linnei.gov.tw/tour/index-1.asp?m=13&m1=9&m2=31&gp=136&sid=&keyword=&id=63 |title = 景點地圖導覽-林內神社("燈台"章節第二段) |author = 林內鄉公所官方網站 |language=zh-TW }}</ref>。拜殿、本殿、手水舍、神橋、制札皆已不存，而石燈籠、春日燈多遭到破壞。原本殿所在之位置改建為濟公廟(1958年成立<ref>{{Cite web |url = http://crgis.rchss.sinica.edu.tw/temples/YunlinCounty/linnei/0910002-JGT|title = 財團法人濟公堂介紹 |author = 陳良旻 |date = 2012-09-05|publisher = 中央研究院地理資訊科學研究專題中心文化資源地理資訊系統|language=zh-TW }}</ref>)。原有配置僅參道、一與二之鳥居、大石燈籠2對、剩下混凝土造部分的春日燈籠數基、林內神社建立紀念碑<ref>{{Cite web |url = http://www.linnei.gov.tw/tour/index-1.asp?m=13&m1=9&m2=31&gp=136&sid=&keyword=&id=63 |title = 景點地圖導覽-林內神社("紀念碑"章節) |author = 林內鄉公所官方網站 |language=zh-TW }}</ref><ref>{{Cite web |url = http://blog.xuite.net/jin.ishin/train/23259889-%E6%9E%97%E5%85%A7%E7%A5%9E%E7%A4%BE%E8%B8%8F%E6%9F%A5 |title = 林內神社踏查 |author = 旅的火車頭 |date = 2009-04-10 |language=zh-TW }}</ref>、[[石獅子|狛犬]]1對(安置於濟公堂內)、大石燈籠構件若干(安置於濟公堂後方)等保留下來。

近年來因加強觀光產業，在雲林縣政府以及林內濟公堂的努力下，已將神橋、春日燈、手水舍以及三之鳥居<ref>{{Cite web |url = https://www.google.com.tw/maps/uv?hl=zh-TW&pb=!1s0x346ecbfebe6423bf:0x707e50d670d0e0b2!2m22!2m2!1i80!2i80!3m1!2i20!16m16!1b1!2m2!1m1!1e1!2m2!1m1!1e3!2m2!1m1!1e5!2m2!1m1!1e4!2m2!1m1!1e6!3m1!7e115!4shttps://picasaweb.google.com/lh/sredir?uname%3D108951911840948432851%26id%3D6493137820263165730%26target%3DPHOTO!5zR29vZ2xlIOaQnOWwiw&imagekey=!1e3!2s-wB5vJpE0Iic/WhxCcgZwryI/AAAAAAAAHTo/V26BV6XPjUIDHfqSp1JhA2pJ56WOLSJwACLIBGAYYCw|title = google搜尋上的林內神社"三之鳥居"|language=zh-TW }}</ref>(現為濟公廟牌樓)修復。神社周遭環境基本上已修復的非常完整，林內神社遺址是目前全台灣神社殘跡中唯一具有三副完整鳥居的神社建築群(原本的舊鳥居兩副，濟公堂鳥居一副)。
