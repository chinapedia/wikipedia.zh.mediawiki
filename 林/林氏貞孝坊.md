'''林氏貞孝坊'''是一座豎立於[[臺灣|臺灣]]臺中[[大甲區|大甲]]的清代[[貞節牌坊|貞節牌坊]]。該牌坊乃為表彰淡水廳[[大甲街|大甲街]]地方人士余榮長之[[妻|妻]]林春娘（文獻多記為「春娘」，而「娘」字為舊時對單名婦女之通稱，故其人原名應為「林春」，而櫟社詩人[[蔡子昭|蔡子昭]]所撰碑記即寫有「[[名諱|諱]]春」二字{{r|甲}}）之[[三從四德|貞孝]]事蹟，而興建於[[道光|道光]]二十八年（1848年）之[[紀念建築物|紀念性建築物]]。{{r|林}}{{r|淡}}

[[民國|民國]]七十四年（1985年）8月19日，此牌坊經[[中華民國內政部|中華民國內政部]]指定為臺中縣[[三級古蹟|三級古蹟]]。<ref>{{Cite web|url=https://www.dajia.taichung.gov.tw/982944/982950/983022/983052/1262552|title=林氏貞孝坊|accessdate=2017-04-04|date=2013-06-19|last= |work=臺中市大甲區公所}}</ref>

此外，林春娘在當地除貞孝事蹟外更有[[祈雨|禱雨]]事蹟，並被當地居民奉為「貞節媽」，還有[[神像|神像]]供奉於[[大甲鎮瀾宮|大甲鎮瀾宮]]之內，成為「大甲三神」之一。{{r|林}}

== 歷史 ==
林春娘為大安中莊人，生於清[[乾隆|乾隆]]四十四年（1779年），[[同治|同治]]三年（1864年）去世，年八十六歲{{r|林}}。

林春娘七歲時成為余家[[童養媳|童養媳]]，但[[未婚夫|未婚夫]]余榮長在乾隆五十四年（1789年）赴[[鹿港鎮|鹿港]]經商時溺斃，此時林氏只有十二歲，尚未正式結婚，而余榮長之父已逝，余榮長為家中獨子，但林氏不打算改嫁，留下來侍奉婆婆{{r|林}}。她曾撫養兩名[[過繼|過繼]]來的族子為子嗣，但一位早逝，後來撫養者則在娶巫氏為妻後不久去世，而後林氏與媳婦一同撫養幼孫{{r|淡}}。

[[道光|道光]]十二年（1832年），本地[[士紳|士紳]]劉獻廷呈請旌表，隔年[[臺灣知府|臺灣知府]][[周彥|周彥]]與[[淡水同知|淡水同知]][[李嗣鄴|李嗣鄴]]頒賜旌額，而到了道光十六年（1836年）[[朝廷|朝廷]]才下旨准許建坊旌表，但余家家境貧困，故直到道光二十八年（1848年）才在職吏張綱上呈淡水同知[[黃開基|黃開基]]後由黃開基捐[[銀元|銀元]]五十元募捐，不久[[竹塹城|竹塹城]][[門閥|門閥]][[林占梅|林占梅]]即捐助銀一百二十元，隨後各方響應捐款後，共得銀一千二百多元{{r|林}}。之後經過數月後，於[[大甲城|大甲城]]南門外建成牌坊{{r|林}}。

[[臺灣日據時期|臺灣日據時期]]，[[昭和|昭和]]八年（1933年）發生[[旱災|旱災]]，大甲地區供奉林氏神像禱雨得應，而此時牌坊已年久失修，地方人士遂有重修之議，於是便用祈雨募捐餘款另購坊地，於昭和十年（1935年）重修{{r|林}}。

[[二次大戰|二次大戰]]後，大甲鎮瀾宮曾於民國四十年（1951年）整頓環境，於周圍建有矮牆，大門位於順天路，後來在民國六十八年（1979年）大甲鎮公所重修，拆除舊牆改建新牆，並將入口改到光明路，並添設一對[[石獅|石獅]]{{r|甲}}。民國八十三年（1994年），由內政部、省民政廳、臺中縣政府補助經費再次重修，相關工程於民國九十一年（2002年）4月17日完工{{r|文}}。

== 建築 ==
[[File:大甲林氏貞孝坊_(夜).jpg|thumb]]
林氏貞孝坊為四柱三間形式{{r|文}}，而或因下旨准許建章時間到實際建坊時間有差，牌坊北邊是刻道光二十八年（1848年）的職官，南面卻是道光十三年（1833年）的職官，此外坊柱四面皆有楹聯而不同一般牌坊只刻於正反面，亦為此牌坊的特色之一{{r|林}}。

== 林氏禱雨事蹟 ==
林氏生前就文獻可考者而言，最早於道光三十年（1850年）留有禱雨事蹟，此後尚有禱雨數次，而在同治元年（1862年）的[[戴潮春事件|戴潮春事件]]時，大甲被圍，林氏曾三出禱雨，適時化解水道被斷的缺水危機（林豪《東瀛紀事》一書有記載）{{r|林}}。

==戲劇改編==
[[民視|民視]]《[[台灣奇案|台灣奇案]]》將此事蹟改編為古裝劇，收錄為中，分為三集播出，由[[白冰冰|白冰冰]]飾演林婦，也順帶介紹華南到台灣的傳播者—[[報馬仔|報馬仔]]。
