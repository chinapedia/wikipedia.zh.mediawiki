'''林旺'''（1917年10月29日－2003年2月26日），是一頭曾於[[第二次世界大戰|二次大戰]]期間在[[日軍|日軍]]、[[中國遠征軍|中國遠征軍]]中服役的[[緬甸|緬甸]]籍[[戰象|戰象]]，後來隨著[[孫立人|孫立人]]將軍遷往[[臺灣|臺灣]]，安置在[[臺北市立動物園|臺北市立動物園]]。牠有大半生都是在臺北市立動物園度過，無庸置疑的是臺灣最有人氣及名氣的動物，並且成為許多台灣民眾兒時的共同回憶，大家都叫牠「'''林旺爺爺'''」。

== 抗日戰爭 ==
1941年[[日本|日本]][[珍珠港事件|攻擊珍珠港]]之後，在1937年開打的[[抗日戰爭_(中國)|抗日戰爭]]也成為[[二次大戰|二次大戰]]的主要戰場之一。1942年當日本進軍[[英國|英國]][[殖民地|殖民地]][[緬甸|緬甸]]時，[[大元帥|大元帥]][[蔣中正|蔣中正]]下令組織[[中國遠征軍|中國遠征軍]]，[[孫立人|孫立人]]指揮其中1部- 新38師，前往緬甸作戰。在1943年的一場戰役中，[[中國|中國]]軍隊不費一槍一彈便俘虜了為[[日軍|日軍]]工作的林旺與其他十二頭[[亞洲象|亞洲象]]。日軍主要是使用這群大象來運補物資及拖拉[[大砲|大砲]]，[[盟軍|盟軍]]的用途也十分類似。此時林旺的名字是“阿妹”。

1945年，這群大象和牠們的騎師及騾馬隊經由[[滇緬公路|滇緬公路]]回到了中國，但是在這艱難的旅程中，有六頭大象不幸身亡。當他們抵達[[廣州|廣州]]時，戰爭結束了，但這群大象的任務並沒有告一段落。牠們在1946年春天時參與了“抗戰烈士紀念碑”的建造工作，同時在[[馬戲團|馬戲團]]進行表演，為[[湖南|湖南]][[飢荒|飢荒]]進行募款。之後，其中四頭大象分別被送到了[[北京|北京]]、[[上海|上海]]、[[南京|南京]]和[[長沙|長沙]]的[[動物園|動物園]]。剩下包括林旺在內的三頭大象，被重新安置在廣州[[新一軍印緬陣亡將士公墓|新一軍印緬陣亡將士公墓]]內。

== 台灣歲月 ==
[[File:_Lin_Wang_and_Sun.jpg|thumb]]將軍，攝於1947年。]]
1947年，孫立人被派遣到臺灣從事訓練新部隊的工作，並帶著包含林旺在內的兩頭大象一同前往，剩下一頭送往[[黃花崗七十二烈士墓|黃花崗七十二烈士墓]]供廣州市民觀賞。大象來到臺灣後，在[[高雄縣|高雄縣]][[鳳山區|鳳山鎮]]的軍事基地附近從事搬運原木及其他一些簡單的工作。1951年，另一頭公象阿沛因感染寄生蟲死亡之後，林旺成為當初十三頭的象群中唯一的存活者。

1954年，孫立人決定將林旺送給當時位於[[圓山|圓山]]的[[臺北市立動物園|臺北市立動物園]]，園方同時將當時年僅3歲的母象馬蘭與林旺送作堆，結為連理。此時林旺的名字仍然是“阿妹”（但軍中一般念作「阿美」），但園方覺得「阿妹」或「阿美」都太過[[女性化|女性化]]，遂請孫立人為之改名，孫表示習慣其稱呼，可叫「阿美」，或由園方自行決定。因此園方取“森林之王”之義，改名為“林王”。但是卻有一個[[記者|記者]]誤將林王錯聽成音調接近的“林旺”，並將此名字刊登出來，之後“林旺”就成為牠的名字而沿用至今，並成為臺灣家喻戶曉的動物明星。

1969年，50歲的林旺突然性格大變。管理員在他的糞便中發現血絲，發現他罹患了大腸癌。由於當時動物園沒有麻醉大象的經驗和技術，只好將他五花大綁以進行治療。後來手術雖然成功，但從此他對獸醫和管理員的態度有了180度的轉變，不再像以往那般溫馴。

自1971年起，林旺在每年11月至次年5月間都會有一段「狂暴期」，會變得具有攻擊性。為了安全與管理的考量，園方因而決定將林旺的一隻腳，用鐵環加以固定。 因為套上了腳鐐，活動空間變小，林旺開始出現奇怪的舉動─頭和身體呈現節奏性的左右搖晃。這種情形一直到了1977年，園方擴建象欄，鍊腳的日子才告結束。 

1983年，園方為林旺舉辦66歲的生日[[派對|派對]]。在此之後，每年10月的最後一個星期日，園方都會為林旺舉辦生日派對，與眾多的遊客一同為林旺“祝壽”。1986年，動物園從圓山遷往[[木柵|木柵]]，許多的臺北市民駐足在街道兩旁觀看動物們搬家，特別是林旺。由於林旺生性多疑，在動物園準備搬家之時，數十名工作人員及獸醫花了一整天的時間，才將牠“拖”進特製的大型貨櫃。在抵達木柵的新家之後，由於已經近黃昏，加上環境生疏，林旺誤將新家前大象造型的[[電話亭|電話亭]]當成了馬蘭而走入壕溝中，並且遲遲不肯走出，直到管理員將馬蘭從貨櫃中放出，林旺才肯走出壕溝。

2002年2月24日，馬蘭因腳疾惡化及淋巴癌病逝、享年54歲，園方擔心林旺不適應。然而林旺果真情緒差，就像是得了憂鬱症一樣，遲遲無法適應沒有老伴的日子，園方一直忙著安撫。

從2003年初開始林旺的左後腿因[[關節炎|關節炎]]退化而變得疼痛無力，導致出現食慾下降跟排泄困難的狀況。2003年2月中旬開始，平時不愛下水的林旺，卻不時的浸泡在水池裡，有時甚至會泡上一整天，可能是利用身體泡在水裡的浮力淺輕關節炎的疼痛。最後在2月26日凌晨被管理員發現安詳地側臥在水池邊辭世，死因為突發性休克，享壽86歲。　

== 紀念 ==
林旺來[[台北動物園|台北動物園]]前是軍用動物，在[[中華民國國軍|國軍]]內享有盛名，刁難[[士兵|士兵]]的國軍[[體罰|體罰]]「[[黃埔十道菜|黃埔十道菜]]」，亦有「我愛林旺」一項，要新兵以手勢模仿象鼻，持[[槍|槍]]罰站。

林旺去世後，為林旺舉辦的紀念活動持續了一個月，園區湧入了成千上萬的遊客，並且留下鮮花及卡片，向林旺做最後的道別。時任[[台北市長|台北市長]][[馬英九|馬英九]]授與林旺「台北市榮譽市民」的榮耀，並贈輓聯「獨領風騷半世紀，共同記憶四代人。」，橫批「世紀象瑞」。<ref>[http://old.ltn.com.tw/2003/new/mar/1/today-c1.htm 追思林旺 象迷永難忘]</ref>[[中華民國總統|總統]][[陳水扁|陳水扁]]也獻上花圈，並在卡片寫著「給我們永遠的朋友，林旺」。2004年10月，林旺被製作成全世界最大的亞洲象標本，置放在園內的教育中心，供遊客參觀紀念<ref>[http://play.tvbs.com.tw/other/490335 林旺爺爺回來了　明回木柵動物園]</ref><ref>[http://news.tvbs.com.tw/other/490237 林旺回來了！標本運回動物園　民眾爭睹]</ref><ref>[http://www.appledaily.com.tw/appledaily/article/supplement/20041016/1310244/ 林旺爺爺與大家再會]</ref>。

以一般亞洲象平均70歲的壽命而言，林旺算是相當長壽的。牠以86歲的高齡逝世，也超過在1997年去世的Lakshmikutty的84歲，成為全世界在豢養情況下最長壽的大象。

林旺的一生，宛如一部台灣近代史。從戰亂時期的中國，到經濟起飛的台灣，見證了台灣社會的變遷，更跨越了不同的世代，在人們的心中烙下難以抹滅的深刻記憶，因此林旺不單單只是一頭大象而已。

2017年8月9日，原先存放在動物園蒐藏庫角落的林旺與馬蘭全身骨骼遺骸，經由國立自然科學博物館與臺北市立動物園的共同合作，以及石尚企業股份有限公司的贊助與支持之下，完整其骨架，林旺與馬蘭站起來展區與科博館現正展出的《象群特展》，共同展至2018年3月18日<ref>[http://web2.nmns.edu.tw/Exhibits/106/Elephant 國立自然科學博物館-象群特展(2017年6月28日-2018年3月18日)]</ref>。

== 年表 ==
*1943年（民國32年），原本為[[日軍|日軍]]拖[[大砲|大砲]]的[[亞洲象|亞洲象]]林旺被國軍俘虜，時年26歲。
*1945年（民國34年），林旺跟著國軍部隊，作為軍用馱獸，征戰[[滇|滇]][[緬|緬]]地區。
*1946年（民國35年），林旺被送到[[廣州|廣州]]，加入[[孫立人|孫立人]][[將軍|將軍]]的部隊。
*1947年（民國36年），孫立人派人將林旺與另一頭母象，經海路運送到[[高雄縣|高雄縣]][[鳳山區|鳳山鎮]]的[[陸軍|陸軍]]訓練部。
*1951年（民國40年），與林旺同時到鳳山的母象阿沛病逝。
*1954年（民國43年），[[台北市長|台北市長]][[吳三連|吳三連]]牽紅線，讓圓山動物園的唯一母象「馬蘭」與林旺送作堆。馬蘭年僅三歲，這段[[老少配|老少配]][[婚姻|婚姻]]纏綿半世紀。
*1967年（民國56年），管理員在林旺的糞便中發現血絲，由於情況嚴重，經過診斷發現是直腸[[腫瘤|腫瘤]]，決定為林旺動手術。不過當時醫療技術很落後，動物園管理員只好把林旺上腳銬、五花大綁，再把[[手術|手術]]工具直接插入林旺的患部電療及上藥，所幸後來檢查直腸瘤是良性，但是林旺經此以後，心性大變，對管理員及獸醫的態度不友善。
*1977年（民國66年），林旺出現幾次狂暴期，由於擔心傷及管理員，動物園第一次為林旺的腳打上鐵環，讓牠不會狂奔，後來在欄舍另一側開闢展示場，才把腳銬取下。
*1983年（民國72年）10月，動物園為年過花甲的林旺舉行第一次慶生會，林旺時年66歲，從此每年都有慶生會，持續20年直至去世。
*1986年（民國75年），動物園遷移，幾十名工作人員用特製的大型改裝貨櫃，花費了一天的時間才讓林旺與馬蘭搬入木柵新家─白宮。
*1991年及1993年（民國80年及82年），林旺進入狂暴期，數度將馬蘭踢落壕溝內。
*1997年（民國86年），林旺狂暴期高潮，憤怒地追逐馬蘭，並用尖牙及腳對馬蘭施暴。馬蘭在情急之下，跨越一公尺高的矮牆逃命。
*2002年（民國91年）11月，馬蘭因為腳疾惡化及淋巴癌病逝。林旺失去老伴，一直很孤單，常常對著欄舍發呆。
*2003年（民國92年）2月25日，凌晨2時30分，林旺因休克病逝，享壽86歲，是目前文獻記載全世界最長壽的亞洲象。
*2004年（民國93年）10月，林旺標本費時一年8個月完成，被送回木柵動物園永遠陪伴國民。
*2017年 (民國106年) 8月，林旺與馬蘭完整骨架於[[國立自然科學博物館|國立自然科學博物館]]展出。
*2018年 (民國107年) 6月，林旺、馬蘭與阿沛的完整骨架將於臺北市二二八公園內的[[國立臺灣博物館|國立臺灣博物館]]展出。
