'''林登·B·约翰逊号驱逐舰'''（'''USS Lyndon B. Johnson (DDG-1002)'''）是[[美国海军|美国海军]][[朱姆沃尔特级驱逐舰|朱姆沃尔特级驱逐舰]]的第三艘和最后一艘，以[[美国|美国]]第36任[[总统|总统]][[林登·B·约翰逊|林登·B·约翰逊]]的名字命名。林登·B·约翰逊号是第34次利用总统名字命名战舰。<ref name=DestroyerName>{{cite press release |title=Navy Names Zumwalt Class Destroyer USS Lyndon B. Johnson |url=http://www.defense.gov/releases/release.aspx?releaseid=15190 |publisher=Defense.gov |date=16 April 2012 |accessdate=20 April 2012}}</ref>

==建造和命名==
修建林登·B·约翰逊号和[[迈克尔·蒙索尔号驱逐舰_(DDG-1001)|迈克尔·蒙索尔号]]的合同于2011年9月15日与[[巴斯钢铁厂|巴斯钢铁厂]]签订。这艘战舰和迈克尔·蒙索尔号一起总共价格为18.26亿美金<ref name=NVR>{{cite web |url=http://www.nvr.navy.mil/nvrships/details/DDG1002.htm |title=No Name (DDG 1002) |author= |date=6 February 2012 |work=[[Naval_Vessel_Register|Naval Vessel Register]] |publisher=Navy.mil |accessdate=18 April 2012 |deadurl=yes |archiveurl=https://web.archive.org/web/20160731085844/http://www.nvr.navy.mil/NVRSHIPS/DETAILS/DDG1002.HTM |archivedate=2016年7月31日 }}</ref><ref>{{cite press release |title=DDG 1001 and DDG 1002 Ship Construction Contract Award Announced |url=http://www.navsea.navy.mil/PR2011/091511DDG.pdf |publisher=[[Naval_Sea_Systems_Command|Naval Sea Systems Command]] |format=PDF |date=15 September 2011 |accessdate=18 April 2012 |deadurl=yes |archiveurl=https://web.archive.org/web/20130302104930/http://www.navsea.navy.mil/PR2011/091511DDG.pdf |archivedate=2013年3月2日 }}</ref>在2012年4月16日，海军秘书{{link-en|雷·麦伯斯|Ray Mabus}}宣布[[朱姆沃尔特级驱逐舰|朱姆沃尔特级]]第三号舰名字为林登·B·约翰逊号以纪念美国第36任总统[[林登·B·约翰逊|林登·B·约翰逊]]。<ref name=DestroyerName />
