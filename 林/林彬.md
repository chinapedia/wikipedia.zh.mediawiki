'''林彬'''（原名'''林少波'''<ref>[http://www.dimcook.com/ahsio/blog/5088604/%E8%B7%91%E9%A6%AC%E5%9C%B0%E5%A4%A9%E4%B8%BB%E6%95%99%E5%A2%B3%E5%A0%B4%E8%88%87%E6%9E%97%E5%BD%AC 跑馬地天主教墳場與林彬]</ref>，{{bd|1929年|9月24日|1967年|8月25日|catIdx=L林}}），是位[[孤兒|孤兒]]，[[林光海|林光海]]之兄，於1960年代擔任[[香港商業電台|香港商業電台]]播音員。1967年，香港[[親中共|親共]]和[[左派|左派]]人士為了響應[[文化大革命|文化大革命]]，在香港展開[[六七暴動|六七暴動]]行動。由於林彬在電台節目中抨擊有關人士的暴力行為，於同年8月24日在[[九龍|九龍]][[何文田|何文田]][[窩打老道山|窩打老道山]]嘉鳴閣對開馬路上被伏擊，其人連同座駕遭到縱火，燒至重傷，翌日不治。<ref>[http://www.inmediahk.net/node/1006870 商台、黃永 - 應記一功，值得一讚], [[香港獨立媒體|香港獨立媒體]], 2010-05-03</ref><ref>[http://www.voachinese.com/content/article-20100514-riots-stirs-fresh-memory-controversy-hongkong-93764699/476300.html 香港六七暴動勾起痛苦回憶和新爭議], [[美國之音|美國之音]], 14.05.2010</ref>

== 背景 ==
{{seealso|六七暴動}}
1967年4月中旬，香港[[新蒲崗|新蒲崗]]塑膠花廠發生工潮，當時中國大陸「[[文化大革命|文化大革命]]」在發展得如火如荼，香港親[[中共|中共]]左派介入工潮活動，把文革的舞台搬到香港，與殖民地政府在街頭上對抗。5月16日，[[工聯會|工聯會]]領導人物[[楊光_(鬥委會)|楊光]]等人成立「[[鬥委會|鬥委會]]」，公開呼籲左派人士走上街，使用暴力與殖民地政府抗爭，後來左派工人被打死射杀多名，愤怒之下暴徒在街上放置土製炸彈，嚴重影響市民生活。雖然炸彈上寫「同胞勿近」，但是受難者多數是華人。

暴動發生後，[[商業電台|商業電台]]抨擊左派人士的暴行，由林彬主持的節目《欲罷不能》及《時事評論》，抨擊左派“暴徒”，諷刺當時左派陣營中，幾位積極發動暴亂份子，指他們擾亂香港秩序，對左派的行為及目的更是諸多嘲諷。根據周奕著的《香港左派鬥爭史》記載，林彬在節目《欲罷不能》中大罵「左仔」，痛斥左派擾亂社會秩序，是「無恥無良、低能邋遢，下流賤格」。<ref>{{cite news|url=http://hk.apple.nextmedia.com/template/apple/art_main.php?iss_id=20100430&sec_id=4104&subsec=11866&art_id=13981746|title=民建聯買時段掀大風波 名嘴潘小濤哭商台對不起林彬|publisher=蘋果日報|date=2010年4月30日}}</ref>

1967年8月20日下午，暴徒在[[北角|北角]]清華街擺放炸彈，八歲女童黃綺文及其兩歲弟弟黃兆勳出來遊玩，觸摸炸彈即時被炸得肚破腸流、死狀至慘。被林彬痛罵為野獸行為，指斥左派人士喪盡天良。由於清華街慘案深入人心，加上林彬利用廣播對市民產生一定的影響力，這一段期間，林彬曾經接到過無數的恐嚇信，左派報紙《[[文滙報|文滙報]]》更把林彬的名字改為「臨殯」，甚至公開聲明要置之於死地。據知，事前曾有一神秘女子致電商台位於[[又一村|又一村]]的總部恐嚇林彬，揚言對他不利，接聽電話的著名播音員[[李我|李我]]覺得事態嚴重，並陪同林彬到[[石硤尾警署|石硤尾警署]]報案，惟林彬以有保護裝備為理由，婉拒警方提供保護。<ref>{{cite book|url=http://www.huanghuagang.org/library/1967hongkong_12-7-07.html|title=1967:中共煽動香港暴動的罪行――紀念林彬遇害四十年|work=《黃花崗雜誌》|author=林偉棠|deadurl=yes|archiveurl=https://web.archive.org/web/20080306182617/http://www.huanghuagang.org/library/1967hongkong_12-7-07.html|archivedate=2008-03-06}}</ref>

== 遇害 ==
1967年8月24日上午8時45分，林彬駕駛著車牌號碼AF7268的私家車於[[何文田|何文田]][[窩打老道山|窩打老道山]]文運道開車返回電台，剛轉入文福道近文運道處，被兩名偽裝成修路工人的兇徒攔截。兇徒向車上潑以汽油縱火，林彬全身着火，翌日傷重死亡，終年37歲，與其同車的堂弟林光海則昏迷留醫至8月30日不治。<ref>{{cite web|url=http://www.youtube.com/watch?v=v37scgsFMPc|title=《由1967開始》 第一集、第二部分|author=電視廣播有限公司}}</ref>事發翌日，《[[大公報|大公報]]》以「地下突擊隊鋤奸　敗類林彬受重傷」為題譴責他。親台灣的右派報章《[[工商日報|工商日報]]》則報道，事發當天「地下鋤奸突擊隊」司令部發表公告，指事件是「向林逆作出民族紀律的處分，嚴厲懲罰，以示儆戒」。《[[工商日報|工商日報]]》又稱，林彬當時被擲汽油彈，滾出車後再被淋電油，臉部燒焦，頭髮燒光。他在救護車一度甦醒，並向妻子大喊：「左仔害死我咯！」。<ref>{{cite news|url=http://hk.apple.nextmedia.com/template/apple/art_main.php?iss_id=20100513&sec_id=4104&subsec_id=11866&art_id=14024469|title=死亡真相 林彬送院喊：左仔害死我|publisher=蘋果日報|date=2010年5月13日}}</ref>林彬被殺後，有人自稱為「鋤奸突擊司令部」的匿名者承認責任，稱林彬為「民族敗類，[[港英|港英]]走狗」；謀殺是「執行民族紀律」，並聲稱仍會繼續「制裁其他敗類」。警方懸紅五萬元緝兇，商業電台立即增加十萬元，成為當時香港有史以來最高的花紅，可見各方對事件的重視程度，但是暴徒行兇之後，逃到[[澳門|澳門]]難以緝獲（澳門已在1966年「[[一二·三事件|一二·三事件]]」後，為共產黨勢力所控制），事件中無人被捕，至今仍為懸案。<ref>{{cite book|url=http://www.ngensis.com/mad-ref/mad-2.htm|title=林彬遇害|work=《文化大革命志 補卷一: 赤禍香港》}}</ref>

商業電台董事總經理[[何佐芝|何佐芝]]即日發表聲明：「暴徒顯然對商業電台努力揭發他們的殘害市民及破壞本港安寧的醜惡活動，害怕得要死，因而採取這種滅絕人性的手段，以圖恐嚇本台及工作人員，但左派暴徒今日的殘暴手段，決不能動搖本台及各同寅反對暴力的正義立場，我們一定會更加努力，在各方面協助當局敉平騷亂<ref>{{cite book|author=張家偉|title=香港六七暴動內情|year=2000|publisher=太平洋世紀出版社|location=香港|isbn=9628274295|pages=103-105}}</ref>。」

===公眾反應===
林彬死後，市民普遍對左派暴徒之暴行感到憤怒，多名[[市政局_(香港)|市政局]]議員亦隨之對如何處理謀殺林彬之慘案發表意見，大體上分為[[鷹派|鷹派]]和[[鴿派|鴿派]]。

鷹派方面，[[香港公民協會|公民協會]]的[[張有興|張有興]]議員主張把殺人暴徒公開執行[[絞刑|絞刑]]示眾，以消滅[[恐怖主義|恐怖主義]]行為，並相信會獲大批市民支持；但「鴿派」的[[沙利士|沙利士]]及[[黃夢花|黃夢花]]議員則反對張有興的意見，沙利士認為這樣做會使共黨暴徒作出更狠毒的還擊，並主張以「銀彈」收買一些左派人士，黃夢花認為真正需要的是教導市民盡量與警方合作。另一「鷹派」[[香港革新會|革新會]]議員[[胡寶星|胡寶星]]指中環、香港仔等地已有一些憤怒青年示威抗議共黨暴行，並提出舉辦「公開反恐怖主義示威遊行」，讓青年人一洩心中之憤，「鴿派」議員[[葉錫恩|葉錫恩]]反對青年採取過激行動，免香港陷入「內戰」危機，[[馬超常|馬超常]]議員指出香港乃法治社會，此等事件應交由法律解決，並反對示威遊行<ref>{{cite web|url=http://1967riot.wordpress.com/2012/08/31/ksyp-19670831/|title=《工商日報》社論 1967年8月31日}}</ref>。2017年，正正是林彬遇害50周年，不少媒體前輩及關心歷史的學者、研究者們到遇害現場致祭。來到文運道街頭，在林彬座駕撞牆位置以讀舊報章、社評及悼文方式紀念<ref> {{cite web|url=https://www.hkcnews.com/article/6406/%E6%B6%88%E5%A4%B1%E7%9A%84%E6%AA%94%E6%A1%88-%E5%85%AD%E4%B8%83%E6%9A%B4%E5%8B%95-%E6%9E%97%E5%BD%AC-6422/%E6%AD%B7%E5%8F%B2%E5%9C%A8%E9%80%99%E8%A3%A1%E6%B2%89%E6%80%9D-%E6%9E%97%E5%BD%AC%E9%81%87%E5%AE%B350%E5%91%A8%E5%B9%B4-|title=歷史在這裡沉思　林彬遇害50周年|publisher=[[眾新聞|眾新聞]]|date=2017-08-23|accessdate=2017-08-25}}</ref>。

===兇徒身份之謎===
殺害林彬兇徒的真正身份至今仍然是一個謎，但從[[梁慕嫻|梁慕嫻]]的文章《回憶林彬兄弟慘案》及張家偉的著作《六七暴動：香港戰後歷史的分水嶺》均有透露一點線索。

梁慕嫻指出，兇徒為「××總商會鬥委會」屬下的「戰鬥隊」成員，其中一名成員於行兇後移居[[澳洲|澳洲]]<ref>[http://www.open.com.hk/content.php?id=203 梁慕嫻：回憶林彬兄弟慘案]</ref>；而張家偉的著作則指出，一名不願透露姓名之人士向張氏透露，其中一名兇徒姓許，來自[[福建省|福建]]，是該知情人士的同鄉，犯案時年約20歲，事後逃到福建[[南安市|南安市]][[官桥镇_(南安市)|官橋鎮]]老家<ref>{{cite book|author=張家偉|title=六七暴動：香港戰後歷史的分水嶺|year=2012|publisher=香港大學出版社|location=香港|isbn=9789888083961|pages=282}}</ref>。

== 影響 ==
林彬死去的當天晚上，商業電台又在固定節目時播出《大丈夫日記》，當劇中播到林彬之死的悲痛情況時，所有播音員真的大哭起來。香港商業電台因而被香港市民視為敢言的電台，謠傳他的住所「[[十八樓C座|18樓C座]]」數十年來成為商業電台一個諷刺時弊的節目名稱，而商業電台的辦公室在1997年前一直掛有林彬及林光海的遺像，訓示同工不要忘記他們的犧牲。

林彬之死也被認為使香港的[[言論自由|言論自由]]受影響。<ref>[http://www.newtaiwan.com.tw/bulletinview.jsp?bulletinid=53526 李鵬飛揭中國封殺香港新聞自由內幕] {{webarchive|url=https://web.archive.org/web/20141006115814/http://www.newtaiwan.com.tw/bulletinview.jsp?bulletinid=53526 |date=2014-10-06 }}, [[新台灣新聞周刊|新台灣新聞周刊]], 2004/06/02</ref>

林彬身後留下妻子[[鄭潔梅|鄭潔梅]]及三名年幼女兒，後來在-{商台}-及市民捐助下，於同年9月17日赴[[台灣|台灣]]，三女兒完成學業後移居[[加拿大|加拿大]]，鄭潔梅近年定居[[法國|法國]]。[[中華民國|中華民國]]政府更將林彬封為烈士，靈位供奉於[[台北|台北]][[忠烈祠_(臺灣)|忠烈祠]]<ref>{{cite news|url=http://www.libertytimes.com.tw/2001/new/jul/9/today-int5.htm|title=平反文革暴動董建華膽大妄為？|publisher=《自由電子新聞網》|author=林保華|date=2001年7月9日|deadurl=yes|archiveurl=https://web.archive.org/web/20041211075051/http://www.libertytimes.com.tw/2001/new/jul/9/today-int5.htm|archivedate=2004年12月11日}}</ref>。

== 演出 ==
林彬除了主任電台節目主持之外，還擔任演出電影《大丈夫日記》上、下集的演員。《大丈夫日記》本來是商業電台的廣播節目，後來由[[楚原|楚原]]執導，經改編成拍成電影。於1964年完成，影片反映出當時香港白領階層年輕夫婦在同輩之間的相互影響的現象，是一套喜劇，沒有明顯的政治動機。<ref name = "赤禍香港">[http://www.ngensis.com/mad-ref/mad-2.htm 林彬遇害]《文化大革命志 補卷一: 赤禍香港》</ref>

== 謠傳 ==
商台廣播劇《[[十八樓C座|十八樓C座]]》播出後，有傳林彬生前寓所的位置是在十八樓C座，但其實林彬並非住在十八樓C座。<ref name = "蘋果">{{cite news|url=http://hk.apple.nextmedia.com/template/apple_sub/art_main.php?iss_id=20080701&sec_id=38164&subsec_id=11236448&art_id=11294678|title=十八樓C座 拗足40年|publisher=《蘋果日報》|date=2008年7月1日}}</ref>
