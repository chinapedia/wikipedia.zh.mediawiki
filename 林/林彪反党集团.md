'''林彪反党集团'''或'''林彪反革命集团'''是[[中华人民共和国|中华人民共和国]]“[[文化大革命|无产阶级文化大革命]]”时期的是一个「帮派组织」称谓。主要有[[林彪|林彪]]、[[陈伯达|陈伯达]]、[[叶群|叶群]]、[[黄永胜|黄永胜]]、[[吴法宪|吴法宪]]、[[李作鹏|李作鹏]]、[[邱会作|邱会作]]、[[林立果|林立果]]等组成。中共指控其在文革期间与[[江青反革命集团|江青反革命集团]]相互勾结夺取中国的最高权力。

== 结果 ==
由于策划杀害[[毛泽东|毛泽东]]的[[五七一工程纪要|计划]]不成，1971年9月13日，林彪乘飞机出逃，飞机失事坠毁于[[蒙古人民共和国|蒙古人民共和国]][[温都尔汗|温都尔汗]]{{参|{{cite web |title=1972年彭德怀听闻林彪反党后称“我不活了” |url = http://news.ifeng.com/history/zhongguoxiandaishi/detail_2011_09/22/9383623_0.shtml |publisher=凤凰网|accessdate=2014-01-26 |language=中文 |date=2011-09-22}}}} 。而其他骨干成员也被抓，林彪集团被粉碎{{参|{{cite web |title=退休将军谈周恩来如何布置抓捕林彪集团成员 |url = http://news.qq.com/a/20120111/001005_2.htm|publisher=腾讯网|accessdate=2014-01-26|language=中文 |date=2012-01-11}}}}。

== 评价 ==
文革期间由于林彪的出逃失败，林彪反党集团被称为「代表了被打倒的[[地主|地主]][[资产阶级|资产阶级]]的利益，代表了被打倒的[[反动|反动]]派推翻[[无产阶级专政|无产阶级专政]]、复辟资产阶级专政的愿望」{{参|{{cite web |title=论林彪反党集团的社会基础 |url = http://marxistphilosophy.org/Hongqi/75/197503-20.htm|publisher=MARXIST PHILOSOPHY |accessdate=2014-01-26}}}}{{参|{{cite journal |author=[[姚文元|姚文元]] |title=《论林彪反党集团》|journal=《思想战线》|year=1975年 |issue=02期}}}} 。
