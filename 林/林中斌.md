'''林中斌'''（{{lang-en|'''Lin Chong-Pin'''}}，{{bd|1942年|5月7日|||catIdx=Lin林}}），生於中國[[雲南|雲南]][[昆明|昆明]]，祖籍[[廣東|廣東]][[新會|新會]]，曾任[[陳水扁政府|陳水扁政府]][[中華民國國防部|國防部]]文職的軍政副部長。研究[[中國人民解放軍|中國人民解放軍]]、[[國際關係|國際關係]]與[[兩岸關係|兩岸關係]]的[[專家|專家]]，現為[[退休|退休]][[教授|教授]]，曾任國防部副部長、國安會諮詢委員、陸委會副主委、淡江大學戰研所教授、國立中正大學政治系研究所所長。畢業於[[台大|台大]]地質系，擁有三碩士、一博士學位。

== 學歷 ==
*[[國立臺灣大學|國立臺灣大學]]地質系學士（1965年）
*[[鮑林格林州立大學|鮑林格林州立大學]]（Bowling Green State University）地質碩士（1969年）
*[[洛杉磯加州大學|洛杉磯加州大學]]（UCLA）企管碩士（1975年）
*[[喬治城大學|喬治城大學]]政治系碩士、國際關係博士（1978-1986年），他的博士論文即為《China's Nuclear Weapons Strategy》 《中共核武戰略》一書。
详见：[[中华人民共和国与大规模杀伤性武器|中国大陆与大规模杀伤性武器]]

== 經歷 ==
*1969年鮑林格林州立大學地質碩士畢業後任地質探勘師，服務於加拿大姜石滿礦業公司（Canada Johns-Manville Co. Ltd.）。於1974年發現[[美國|美國]][[蒙大拿州|蒙大拿州]]靜水白金（[[鉑|鉑]]、[[鈀|鈀]]）礦第一個露頭。1975年獲企管碩士後回任地質探勘師，1977年任職滿府磊企業公司（Manville Corp.）世界總部<ref name="劍與花">劍與花的歲月：林中斌凡塵隨筆，商訊出版社，ISBN：9789866378119 
</ref>。1978年辭職轉為[[喬治城大學|喬治城大學]]政治系學生。
*1986年9月成為前[[美國常駐聯合國代表列表|美國駐聯合國大使]]、[[喬治城大學|喬治城大學]][[教授|教授]]{{Link-en|珍妮·寇派翠|Jeane Kirkpatrick}}的助理之一。
*1987年1月得到博士學位後專任喬治城大學教授[[珍妮·柯克帕特里克|珍妮·柯克帕特里克]]（{{Lang-en|Jeane Kirkpatrick}}）的助理。
*1987年夏到1995年任[[華盛頓哥倫比亞特區|華盛頓]][[共和黨|共和黨]]華府「{{Link-en|美国企业公共政策研究所|American Enterprise Institute}}」專任學者兼亞洲部副主任。同一時期並先後任喬治城大學外交學院及政治系之講座教授以及兼任教授。
*1995年返回[[台灣|台灣]]後，曾任[[國立中山大學|國立中山大學]]政治學研究所教授兼所長。
*1996年到2002年擔任[[行政院大陸委員會|行政院大陸委員會]]副主任委員，1999年底起為第一副主委兼發言人。
*2002年到2003年擔任[[中華民國國家安全會議|中華民國國家安全會議]]諮詢委員。
*2003年到2004年5月20日擔任[[中華民國國防部|中華民國國防部]]{{Link-ja|軍事行政|軍政 (行政)|軍政}}[[軍政|副部長]]，部長為[[李傑_(臺灣)|李傑]]<ref>

====中美爭鬥前景====
2019年在台北News 98電台的專訪中探討「中美必有一戰說法」認為所為目前國際上的修昔底德陷阱理論者都是發表謬論，因為[[潛艦|潛艦]]發射的[[核武器|核武器]]所形成[[相互保證毀滅|第二擊]]現象發明出來後，大國間全面戰爭已沒有可能性，另一方面這些西方視角教育的學者對中國文化歷史了解貧乏，<ref name=n98lintung>[https://www.youtube.com/watch?v=BhYDrWZCQGM News98官方頻道-林中斌2019專訪]</ref>林中斌表示北京目前已經擬定最高戰略以[[孫子兵法|孫子兵法]]和毛澤東游擊戰術為最高思想，孫子兵法名為[[軍事思想|軍事思想]]其實核心是主張儘量減低用軍力，以外交、經濟、文化為主攻手段，瓦解敵方大量戰力於無形領域，最終必要時才最少使用軍力和其代價，達成實質長久的勝利。而思維停留在[[克勞塞維茲|克勞塞維茲]]圈圈中的很多西方人無法跳出框架。

同時毛澤東游擊戰術來自孫子思想的變形，強調避開敵方來勢洶洶的鋒芒，採取假性撤退和低姿態，尋機在意料之外領域開闢新戰場，在敵方想不到的地點製造殺傷力。他預言中美貿易戰將幾年內收場與一些商界人士預言截然不同，理由是經濟理論與歷史證明從沒有打貿易戰最後能使自己強大的範例，只是百姓相互受損的煎熬，而誰能熬到最後是贏家<ref name=n98lintung/>，這點上不用[[選舉|選舉]]的習近平占優勢，美國和其他波及的相關國隨著一場場選舉鄰近或失利，將尋求快速收場，中國大陸將做出象徵性的一些退讓而將來用別的方式討回。同時長線來看旁觀的大量欠發展國家將從事件中領悟到西方是不值得信任對象，更加有動機加入一帶一路計畫為自己爭取更多保障。<ref name=n98lintung/>

==== 台灣已失去軍力質量優勢 ====
2011年1月12日，[[殲-20|殲-20]]試飛以後林中斌表示，[[中華民國空軍|中華民國空軍]]已失去[[台海|台海]]的[[空軍|空軍]]質量優勢，即使向[[美國|美國]]採購[[F-16戰隼戰鬥機|F-16C/D]]，兩岸軍力的平衡也不復存在。林中斌指出，[[北京|北京]]的[[東風-21中程彈道飛彈|東風-21中程多彈頭彈道飛彈]]，加上[[潛艦|潛艦]]以及殲-20，意義在於告訴美國，[[第一島鏈|第一島鏈]]已經守不住了，而且也不要再到[[台灣海峽|台灣海峽]]來了，[[北京|北京]]不戰而主東亞的大謀略就要實現了。並且[[經濟|經濟]]上中國的[[外匯存底|外匯存底]]已經達到2.8兆，可望突破3兆，有雄厚的實力。林中斌認為，台灣應扮演善意[[催化劑|催化劑]]的角色，與彼岸分享長年[[政治|政治]]、經濟、社會改革的實驗成果<ref>{{cite news
|url=http://www.nownews.com/2011/01/12/91-2681478.htm#ixzz1dZTiKmMc
|title= 殲20一出　林中斌：台軍質量空優沒了，買F16 C/D也沒用
|publisher= 今日新聞網 
}}</ref>。

2017年1月林中斌接受中國官媒《[[環球時報|環球時報]]》專訪時表示，兩岸存在和平統一的可能性，只是時間還沒到，同時認為若是中國若犯台，台灣撐不過72小時。<ref>[http://www.storm.mg/article/213828 中國官媒專訪我國防部前副部長：兩岸和平統一只是時間問題]</ref><ref>[http://www.appledaily.com.tw/realtimenews/article/recommend/20170117/1036821/%E4%B8%AD%E5%9C%8B%E8%8B%A5%E7%8A%AF%E5%8F%B0%E3%80%80%E5%9C%8B%E9%98%B2%E9%83%A8%E5%89%8D%E5%89%AF%E9%83%A8%E9%95%B7%EF%BC%9A%E5%8F%B0%E6%92%90%E4%B8%8D%E9%81%8E3%E5%A4%A9 中國若犯台國防部前副部長：台撐不過3天]</ref>
<br />
引發議題後林中斌又受訪表示環球時報該段未刊登完整語意：台灣只能抵擋72小時不是我核心看法，當時是以述說資料的語氣提及歷任台國防部長都曾說過抵擋時間在72小時到一個月之間。內幕可能是北京有人對反貪不滿將台灣問題操作為鬥爭習近平工具<ref>[http://www.storm.mg/article/214549 林中斌：台灣只能抵擋72小時 不是我的看法]</ref>

==== 兩岸趨勢 ====
林中斌多次在個人臉書<ref>[https://www.facebook.com/chongpin.lin facebook.com/chongpin.lin]</ref>表明[[兩個一百年|兩個一百年]]計畫的提出幾乎已經明確暗示，2049年中華人民共和國的建國一百年國慶那天之前台灣問題必須解決，但不可能在一種戰火紛飛的時刻下，所以若是武統那日程絕不是2049年當年，而是大幅提前十年廿年都有可能，和統的話則另說，台灣永不可能知道詳細的計畫與策略內容。但蔡英文上台後的諸多跡象表明，2016年後未來數年將以「窮台」為主攻手段<ref>[https://udn.com/news/story/1/1846614 對台政策是「窮台」]</ref>，逐漸讓台灣民眾深刻感受到經濟枯竭的感受以承認面對大陸的巨大影響力，同時打擊蔡政府威信，另一方面迫使一些年輕人到大陸求職從而跳脫台灣媒體環境，用真實的眼光見證大陸發展。

2017年6月7日貼文引用友人之友的看法表示中國大陸沿海大城市發展已經進入一種人文素養與社會軟體層面的提升階段，並非純粹大型硬體建設階段，台灣一些偏頗眼光還用20年前的印象在想像大陸，在自身硬體建設大大落伍時還有一種安慰想法認為自己軟體方面和人文方面還是領先，事實證明上海等城市的新現象終究會擴散到全國，屆時台灣的下一代只能自求多福。<ref>[http://www.chinatimes.com/newspapers/20170609000731-260301 林中斌-上海新體驗]</ref>

2017年7月12日發表專文表示台北生活品質及人民善良謙讓為北京太子黨所愛而上海生活便捷與文化提昇為台灣跨國企業精英所喜。雙方各講對了一半。<ref>[https://udn.com/news/story/7340/2578034 林中斌-新雙城記]</ref>

====蔡英文困境====
2018中旬接受美國記者訪問與後續臉書貼文表明，目前許多媒體人和[[柯文哲|柯文哲]]所判斷的蔡英文被深綠脅持是事實。

蔡英文當總統之前管過最大機構就是200人的[[陸委會|陸委會]]，她沒管過縣市或大部會，所以一方面行政經驗不夠二方面人才與人脈庫存都極少，加上她在民進黨中是外來者，始終與其他人隔有一層，現在在民意支持下滑中又面對選舉困境，她只能尋求靠向深綠換取最即時又廉價的黨內一派人支持。讓整個政治遊戲還能勉強運轉下去，若是敗選後失去黨主席地位又被全社會視為連任無望<ref>[http://www.chinatimes.com/realtimenews/20180703000019-260407 林中斌稱 蔡英文是民進黨外來者 被深綠夾持]</ref>，那她政治生命將迅速枯萎，屆時很可能是[[陳水扁|陳水扁]]末期的放大版，也就是在台獨路上更加飆車以尋求深綠最後的支持，或採取放手投降讓位給賴清德，只能二選一。

== 家庭 ==
*父親[[林文奎|林文奎]]，（1909年－1980年），生於[[廣東|廣東]][[新會|新會]]，[[清華大學|清華大學]]經濟與地學雙料學士，1934年[[中華民國空軍軍官學校|中央政府航空學校]]（今[[中華民國空軍軍官學校|中華民國空軍軍官學校]]）第二期第一名畢業，隨即官派赴歐留學四年，1940年任[[陳納德|陳納德]]的機要秘書與情報室主任。抗戰期間，同時具有[[國民政府軍事委員會調查統計局|軍統]]情報人員身份，直接受[[戴笠|戴笠]]領導。1945年勝利後任臺北第23地區空軍司令，曾因為看不慣[[陳儀|陳儀]]與手下紀律敗壞，上萬言書請[[蔣中正|蔣中正]]撤換陳儀。結果拂逆當道，在1946年辭去第23地區空軍司令一職，離開台灣入英國皇家空軍參謀大學，並以優異成績畢業，獲英國國王[[喬治六世|喬治六世]]嘉勉。1948年，應國防部長[[何應欽|何應欽]]電召，出任[[聯合國|聯合國]][[中華民國軍事代表團|中華民國軍事代表團]]空軍參謀。<ref>{{cite news|url=http://www.stnn.cc:82/reveal/200707/t20070711_573683.html|title=台灣首任空軍司令林文奎竟是天字號情報員|publisher=星島環球網|date=2007-07-11|deadurl=yes|archiveurl=https://web.archive.org/web/20130811104127/http://www.stnn.cc:82/reveal/200707/t20070711_573683.html|archivedate=2013-08-11}}</ref>。1953年應[[孫立人|孫立人]]邀請任陸軍總司令部情報署署長。1954年[[黃杰|黃-{杰}-]]接任[[中華民國陸軍司令|中華民國陸軍總司令]]，林文奎在歡迎宴會上公然辭職<ref name="劍與花"/>。於1963年以[[上校|上校]]軍階退役，轉任兼任教授。1980年逝世<ref name="master-c"/>。

*母親張敬（1912年－1997年1月4日），生於[[貴州|貴州]][[安順|安順]]，[[北平大学|北平女子文理學院]]畢業，[[北大|北大]]研究所肄業，任教於[[西南聯大|西南聯大]]；來台後，先後任教於東港空軍參謀學校（[[國防大學_(中華民國)|國防大學]]空軍指揮參謀學院前身）、[[國立台灣大學|國立台灣大學]]中文系<ref name="劍與花"/>。專長[[宋詞|宋詞]]、[[元曲|元曲]]及明清戲曲。1997年1月4日病逝於[[臺大醫院|臺大醫院]]。<ref name="master-c">{{cite web | url = http://www.cl.ntu.edu.tw/people/bio.php?PID=137#personal_writing | title = 張敬的個人資料 | publisher = [[國立臺灣大學|國立臺灣大學]]中國文學系 | language = zh-tw }}</ref><ref name="cl-ntu-intro2">{{cite web | url = http://www.cl.ntu.edu.tw/intro/super_pages.php?ID=intro2 | title = 系所特色 | publisher = [[國立臺灣大學|國立臺灣大學]]中國文學系 | language = zh-tw }}</ref>

*弟弟林中明，居住在美國。

*1995年7月1日林中斌與小他24歲的張家珮結婚<ref name="劍與花"/>。
