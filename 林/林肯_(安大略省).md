'''林肯'''（{{lang-en|Lincoln}}）是[[加拿大|加拿大]][[安大略省|安大略省]][[南安大略|東南部]][[尼亞加拉區|尼亞加拉區]]一鎮，坐落[[尼亞加拉半島|尼亞加拉半島]]北部，北面瀕臨[[安大略湖|安大略湖]]，東鄰[[聖凱瑟琳斯|聖凱瑟琳斯]]市，東南及皮拉姆鎮，西南臨[[西林肯|西林肯]]鄉，西面則與[[格里姆斯比_(安大略省)|格里姆斯比]]鎮為鄰。據2011年加拿大人口普查所示，林肯鎮內有22487名居民<ref name=SC11/>，而此鎮亦是聖嘉芙蓮斯-[[尼亞加拉瀑布城_(安大略省)|尼亞加拉瀑布城]]都會區的一部分。

== 歷史 ==
現林肯鎮一帶本為中立原住民族（{{lang|en|Neutral Nation}}）的聚居地，但[[易洛魁聯盟|易洛魁族]]則於1650年代向中立族宣戰，數年後中立族已全面撤離該帶<ref name="hist">{{Cite web |url=http://www.lincoln.library.on.ca/about#historytown |title=A brief history of the town of Lincoln |accessdate=2015-01-12 |publisher=Lincoln Public Library }}</ref>。[[美國獨立戰爭|美國獨立戰爭]]後，大批效忠英王的十三州殖民遷居至[[上加拿大|上加拿大]]，當中雅各布·比姆（{{lang|en|Jacob Beam}}）於1788年在此帶設立一個聚落，而該聚落亦以他命名為比姆斯維爾（{{lang|en|Beamsville}}）<ref name="hist"/>。另一方面，部分[[賓夕法尼亞州|賓夕法尼亞州]][[門諾會|門諾會]]信徒亦於1799年遷至此地，並建立佐敦（{{lang|en|Jordan}}）和葡萄園（{{lang|en|Vineland}}）兩座村落<ref name="hist"/>。葡萄園村内的第一門諾會教堂於1801年設立，為現加拿大境内歷史最悠久的門諾派教會<ref>{{Cite web |url=https://uwaterloo.ca/mennonite-archives-ontario/congregations/first-mennonite-church-vineland |title=The First Mennonite Church (Vineland, ON) |accessdate=2015-01-12 |publisher=Conrad Grebel University College }}</ref>。

1792年，[[上加拿大|上加拿大]]總督閃高（{{lang|en|John Graves Simcoe}}）將該殖民地劃分成19個縣，此帶地域成為林肯縣一部分。林肯縣再細分為12個鄉（{{lang|en|township}}），並以[[英格蘭|英格蘭]][[林肯郡|林肯郡]]内的鄉鎮命名，當中包括現林肯鎮境内的克林頓鄉（{{lang|en|Clinton}}）和婁斯鄉（{{lang|en|Louth}}）<ref name="hist"/>。1970年1月1日，安大略省政府改革此帶地域的地方政府，林肯縣和威蘭縣（{{lang|en|Welland County}}）合併成新的[[尼亞加拉區|尼亞加拉區]]，而比姆斯維爾鎮、克林頓鄉和婁斯鄉約一半地域則合併成一個新的鎮級行政區，經鎮内居民投票後取名林肯鎮<ref name="hist"/><ref>{{Cite web |url=http://www.lincoln.ca/content/about-lincoln-0 |title=LINCOLN:  The Story of an Ontario Town |accessdate=2015-01-12 |publisher=林肯鎮政府 |deadurl=yes |archiveurl=https://web.archive.org/web/20150115004627/http://www.lincoln.ca/content/about-lincoln-0 |archivedate=2015年1月15日 |df= }}</ref>。（婁斯鄉的餘下地域則併入[[聖凱瑟琳斯|聖凱瑟琳斯]]市。）

== 交通 ==
省級[[400系列公路|400系列]][[高速公路|高速公路]][[伊利沙伯皇后道|伊利沙伯皇后道]]呈東西走向貫穿林肯鎮，並在鎮内設有三座交匯處，為來往[[多倫多|多倫多]]、[[哈密爾頓_(安大略)|咸美頓]]、[[伊利堡|伊利堡]]和[[美國|美國]][[水牛城|水牛城]]之間的要道。
