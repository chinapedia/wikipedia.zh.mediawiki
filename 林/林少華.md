'''林少华'''{{BD|1952年|||||}}，[[中国|中国]][[翻译家|翻译家]]，曾任[[中国海洋大学|中国海洋大学]][[外国语学院|外国语学院]][[教授|教授]]，现已退休。因其翻译了日本文学家[[村上春树|村上春树]]的众多作品而闻名中国大陆。<ref>{{cite web |url=http://news.163.com/13/1012/14/9B0A6CRR00014Q4P.html |title=“御用翻译”林少华：我和村上春树都倾向于孤独(组图) |accessdate=2013 |publisher=网易 |work= |language=zh-hans}}</ref><ref>{{cite web |url=http://culture.people.com.cn/BIG5/n/2013/1011/c22219-23157911.html |title=諾貝爾文學獎村上敗陣 林少華依然力挺:遲早得獎 |accessdate=2013 |publisher=人民网 |work= |language=zh-hant}}</ref><ref>{{cite web |url= http://news.china.com.cn/rollnews/ent/live/2013-10/11/content_22829522.htm|title= 林少华：村上春树作品大致符合诺奖标准|accessdate=2013 |publisher=中国网 |work= |language=zh-hans}}</ref>

==生平==
林少华祖籍[[山东省|山东省]][[蓬莱市|蓬莱市]]，1952年出生于[[吉林省|吉林省]]九台。1968年，初中毕业的他适逢[[文化大革命|文化大革命]]，回到乡下务农。

后来他又到吉林大学外文系就读日文专业，并在1975年毕业。后他又继续深造并于1982年获吉林大学文学[[硕士|硕士]]学位。

1982年至1993年期间，林少华与妻子一同任教于[[暨南大学|暨南大学]]外语系。1993年他到日本[[长崎县立大学|长崎县立大学]]执教，1996年回国并返回暨南大学文学院任职教授。1999年林少华调往[[青岛海洋大学|青岛海洋大学]]任职，2002年曾到[[东京大学|东京大学]]担任特别研究员。<ref>[http://www.artsbj.com/Html/interview/wyft/zj/917198_3.html 专访村上春树“御用翻译”：林少华]</ref><ref>[http://www.jiaodong.net/wenhua/system/2008/01/24/010165171.shtml 林少华 胶东文化名人] {{webarchive|url=https://web.archive.org/web/20140108131759/http://www.jiaodong.net/wenhua/system/2008/01/24/010165171.shtml |date=2014-01-08 }}</ref>

==作品==
===翻译作品===
自1989年[[漓江文艺出版社|漓江文艺出版社]]获得版权起，林少华就开始翻译[[村上春树|村上春树]]的作品。<ref name="1Q84翻译">[http://gcontent.nddaily.com/3/32/3323fe11e9595c09/Blog/baf/afe986.html 《1Q84》译者不是林少华] {{webarchive|url=https://archive.is/20130427072220/http://gcontent.nddaily.com/3/32/3323fe11e9595c09/Blog/baf/afe986.html |date=2013-04-27 }}</ref>2001年版权易手[[上海译文出版社|上海译文出版社]]，林少华依然是翻译者。<ref>[http://www.techcn.com.cn/index.php?doc-view-135389 十年一剪西窗烛]</ref>

《当我谈跑步时，我谈些什么》首次出版时，出版方南海出版公司发动“村上‘自传’首次公开征集译者”活动，林少华也有参与，但最终[[施小炜|施小炜]]成为出版方指定翻译者。<ref>[http://www.jfdaily.com/a/1020098.htm]{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

虽林少华本人在新经典文化公司获得《[[1Q84|1Q84]]》版权时也表明有意继续担当翻译工作，但未能如愿。<ref name="1Q84翻译"/>

除此以外，林也曾翻译过[[夏目漱石|夏目漱石]]、[[芥川龙之介|芥川龙之介]]、[[川端康成|川端康成]]、[[井上靖|井上靖]]、[[东山魁夷|东山魁夷]]等作家的作品。

===研究作品===
2005年，林出版他的研究著作《村上春树和他的作品》。在失去了《当我谈跑步时，我谈些什么》的翻译作出版的机会后，林少华自称“加快了从翻译向创作和学术研究转型的步伐”。2010年出版《为了灵魂的自由——村上春树的文学世界》，总结多年来对村上春树作品翻译的感悟。

===散文集===
2006年林少华的散文集《落花之美》出版，2010年散文集《乡愁与良知》出版。

==评价==
多年对村上春树的作品翻译，使得林少华在某种程度上影响了中国大陆地区读者对村上的作品解读。<ref name="1Q84翻译"/>2007年7月25日由[[朝日新闻社|朝日新闻社]]出版的日本东京大学中文部教授藤井省三著作《村上春树心目中的中国》，认为村上作品用的是“口语体”，林少华用“文语体、书面语体”翻译未能准确传达，相较更推崇叶惠的香港版和赖明珠的台湾版。林少华回应指翻译出意境才是最忠实。<ref>[http://www.chinanews.com.cn/cul/news/2007/11-30/1092244.shtml 村上春树中译本遭日本教授恶评 林少华撰文反驳]</ref>唯翻譯忠實與否的評價，實在與使用口語抑或文語并無干係，查藤井並未以此為論據。譬如台灣作家林文月翻譯日本古典，也用貼切的口語體翻譯，意境忠于原著。日語原文中，藤井的客觀評價通過與港台譯本做比較，直指林少華的措辭不夠溫柔敦厚，用語宛如浓妝艷抹，有直譯卻要再創作意譯。
