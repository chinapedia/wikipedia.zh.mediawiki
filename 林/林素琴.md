'''林素琴'''（{{Bd|1923年|11月2日|2015年|}}），[[新竹|新竹]]人。原名林素珍，日文名 林こずゑ，外文拼音 Su-Qing Lin，臺羅拼音 Sòo-Khîm Lîm。畢業自臺北帝國大學哲學系，後任為[[國立臺灣大學|臺灣大學]]哲學系助教。專攻法國哲學及大陸理性論。1953年曾以 {{Internal link helper/en|Charles Adams|Charles Adams}} 與  {{Internal link helper/en|Paul Tannery|Paul Tannery}} 所編之法文版[[勒内·笛卡尔|笛卡兒]]全集為基礎於《臺大文史哲學報大文史哲學報》發表〈笛卡特的哲學方法〉，是「臺北帝國大學唯一臺籍哲學學士」，也是「首位受聘於臺灣大學哲學系的臺灣學者」，也是唯一一位在日治時期完成哲學訓練的臺灣女性。

== 生平 ==
1923年11月2日出生，新竹州桃園郡大園庄人，居住於臺北州臺北市御成町4之2（現址臺北市中山北路二段二十巷），其父為臺灣民主運動先驅[[林呈祿|林呈祿]]，上有哥哥林益謙、三個姊姊及一個妹妹林少薰，家中排行第五。幼時就讀[[建成小學校|建成小學]]，1937年畢業旋即考取[[臺北市立第一女子高級中學|臺北第一高等女學校]]，1941年則負笈日本[[東京女子大學|東京女子大學]]高等學部，然1943年畢業因戰事加劇回臺就讀[[臺北帝國大學|臺北帝國大學]]文政學部哲學科，為唯一臺籍哲學科本科生。1945年8月以兩年期間密集修習取得哲學科學士學位，綜觀其教育歷程惟從日本菁英培育之發展，但在戰後卻因語言學思問題成其阻礙。1946年5月起雖任[[臺灣大學|臺灣大學]]哲學系外國哲學研究室助教，所擔任職務則似行政助理，至1955年7月31日離職前，謹曾於1953年發表一篇哲學論文〈[[勒内·笛卡尔|笛卡特]]的哲學方法〉於《臺大文史哲學報大文史哲學報》。與小兒科專業李廷堅醫師於1955年6月結婚，隔年6月隨夫離臺赴美北卡羅來納習醫，期間繼續攻讀社會學碩士，其碩士論文題為《日本に於ける婦人の地位》（關於日本人的婦女的地位，日譯），然因家中長子出生則斷攻讀博士之路。隨後六零年代返臺相夫教子，育有一子一女。1984年8月12日李廷堅醫師因病溘然長逝，1990年以筆名李翠竹用日文發表半自傳《素蘭の迷路カら》從素蘭的迷路開始）。2015年歿於台北。<ref name=":0">{{Cite book|title=啟蒙與反叛─臺灣哲學的百年浪潮|last=陸品妃|first=吳秀瑾|publisher=國立臺灣大學出版中心|year=2019|isbn=|editor-last=鄧敦民|editor-first=洪子偉|location=臺北|chapter=台北帝大唯一台籍哲學學士林素琴}}</ref>{{來源請求}}

== 理論與主張 ==
受到相當完整的菁英學科訓練，堪稱為臺灣第一位女性哲學家，卻因歷史及社會現實框限其發展。唯一出版的〈笛卡特哲學的方法〉是 {{Internal link helper/en|Charles Adams|Charles Adams}} 與  {{Internal link helper/en|Paul Tannery|Paul Tannery}} 所編之法文版笛卡兒全集為基礎，班略其日語文法的影響之結構（如述語在後），清晰涵蓋了[[勒内·笛卡尔|笛卡兒]]思想體系：自我實在（我思故我在）的方法論、從有限之人其觀念來證明其存在的神學論證、及就形上學基礎以論機械論假定運動相對性之物理學 。
