== 简介 ==
林天爱出生于[[山东省|山东省]][[青岛市|青岛市]]，11岁离开了[[青岛市|青岛市]]去[[香港|香港]]读书，学的是[[舞蹈|舞蹈]]专业方面的[[爵士舞|爵士舞]]。<ref name="dbw " />

2005年，林天爱签约加入[[上海|上海]][[涂鸦文化公司|涂鸦文化公司]]。<ref>{{cite web |url=http://ent.sina.com.cn/x/2005-08-16/1459811693.html |title=四招打造新人林天爱 |accessdate= 2005年|work= |publisher=新浪网 |language =zh-cn}}</ref>
