'''林文雄'''（{{bd|1942年|2月1日|catIdx=L林}}），[[中華民國|中華民國]]政治人物，前[[臺南市_(省轄市)|臺南市]][[臺南市市長|市長]]。曾任[[台南市議員|台南市議員]]、[[台灣省議員|台灣省議員]]。

== 早年生平 ==
林文雄出身[[安平區|安平區]]，後住在西區，國小時曾就讀石門國小、永福國小畢業。中學時就讀[[台南一中|台南一中]]初、高中部，畢業後入[[臺灣大學|臺灣大學]]社會系。

大學畢業後，林文雄曾入其家族企業敬華鋼鐵公司擔任總經理。<ref>〈林文雄會是內定人選嗎?〉，《中國時報》七版，1985年6月6日。</ref>並曾任[[救國團|救國團]]臺南市中、西、安平區常委以及臺南市團管區西區輔導中心主任。<ref>林文雄，《十年問政》。</ref>

1973年林文雄在[[臺南市_(省轄市)|臺南市]][[東區|東區]]參選台南市第八屆市議員選舉，以3,455票當選市議員，為全市第二高票（第一高票為[[蘇南成|蘇南成]]）。林文雄在市議員任內即積極佈署省議員，<ref>〈張麗堂再度獲提名 實機運苦幹的結果〉，《[[中國時報|中國時報]]》七版，1977年8月11日。</ref>1977年，林文雄獲國民黨提名參選臺灣省省議員，<ref>〈縣市長暨省市議員提名〉，《中國時報》二版，1977年8月9日。</ref>最後當選。1980年林文雄出任立委候選人[[林聯輝|林聯輝]]競選總幹事一職，<ref>〈從林文雄出任林聯輝競選總幹事談起〉，《鳳凰城雜誌》第七期，見林文雄，《十年問政》，頁89-90。</ref>並為其輔選當選立法委員。<ref>〈省議員登記候選人簡介〉，《中國時報》六版，1981年10月6日。</ref>

1981年林文雄一度登記[[國民黨|國民黨]]內台南市長提名，<ref>〈縣市長提名登記名單〉，《中國時報》二版，1981年6月15日。</ref>然而最後國民黨因禮讓蘇南成而在該屆未提名。林文雄繼續參選省議員並獲得連任。

== 台南市長 ==
1985年林文雄獲國民黨提名參選[[1985年中華民國縣市長選舉|臺南市長選舉]]，最後以130,294票擊敗曾任蘇南成機要秘書的[[陳森茂|陳森茂]]，當選台南市長。

林文雄任內完成[[市立臺南醫院|市立臺南醫院]]的建設，並還清[[蘇南成|蘇南成]]市長時代的大量負債。然而任內先是[[婚外情|婚外情]]遭陳森茂揭露，林文雄與黃姓女子「夜談」的錄音帶被陳森茂公布，一時成為全市關注焦點。1988年發生與國民黨市黨部主委[[封惠南|封惠南]]擅自出售臨安路六信倉庫市有土地，封惠南更施壓阻止市議員參加議會主辦的說明會，引起議長[[張坤山|張坤山]]憤而請辭，<ref>《民進報》第11號，1988年5月21日至19885月27日。</ref>一度引起府會衝突。

1989年4月，林文雄以父親阻止參選連任為由，宣布不再尋求市長連任。<ref>〈仕途比不上父子情深 林文雄放棄競選連任〉，《[[中華日報|中華日報]]》二版，1989年4月21日。</ref>

== 家庭 ==
* 林文雄妻子楊秀敏，師大數學系畢業，曾任[[國民黨|國民黨]]臺南市黨部婦工組組長，1989年林文雄退出市長初選後曾一度登記黨內市長初選，最後遭到勸退。<ref>〈台南市市長夫人楊秀敏退出市長初選〉，《中國時報》三版，1989年6月20日。</ref>1991年代表國民黨[[1991年中華民國國民大會代表選舉|參選國民大會代表]]，敗給[[民進黨|民進黨]]籍的[[唐碧娥|唐碧娥]]。
* 林文雄父親林再傳為安平區人，<ref>〈南市市長省議員 預料將有一番苦戰〉，《中國時報》七版，1977年8月20日。</ref>，年輕時曾在「金源合」擔任學徒。<ref>〈府前街堪稱選舉街 重兵屯駐戰雲密布〉，《中國時報》七版，1981年11月4日。</ref>日後致力創業，創辦[http://jin-rong.com.tw/page_company.html 金榮]、敬華鋼鐵公司。<ref>〈「走攤之孫」深知下層疾苦 伺機而動林文雄信心十足〉，《中國時報》六版，1985年10月23日。</ref>
