'''林凱衡'''{{BD|1985年|}}目前為[[國立臺灣大學|國立臺灣大學]]社會學研究所博士班學生、[[國立臺灣大學工會|國立臺灣大學工會]]理事，曾任國立臺灣大學工會秘書長和國會助理，活躍於研究生勞動權運動。

== 生平 ==
林凱衡生於1985年，[[臺北市立建國高級中學|臺北市立建國高級中學]]、[[國立臺灣大學|國立臺灣大學]]社會學系學士、碩士畢業。碩士畢業後，進入國會任職立法委員助理工作一年，之後返校攻讀社會所博士班。

== 工會運動 ==
林凱衡於2010年開始投入研究生勞動權運動，致力於組織台大校內研究生成立和組織工會。2013年，台大校方因主張研究生助理與校方無雇傭關係（與勞委會之認定相反），故向勞委會提起訴願。11月7日，臺北高等行政法院判決台大敗訴 <ref>[http://pnn.pts.org.tw/main/2013/11/08/%E5%8F%B0%E5%A4%A7%E5%B7%A5%E6%9C%83%E6%A1%88-%E6%B3%95%E9%99%A2%E8%AA%8D%E5%AE%9A%E5%AD%B8%E7%94%9F%E5%8A%A9%E7%90%86%E6%98%AF%E5%8B%9E%E5%B7%A5/ 台大工會案 法院認定學生助理是勞工] {{Wayback|url=http://pnn.pts.org.tw/main/2013/11/08/%E5%8F%B0%E5%A4%A7%E5%B7%A5%E6%9C%83%E6%A1%88-%E6%B3%95%E9%99%A2%E8%AA%8D%E5%AE%9A%E5%AD%B8%E7%94%9F%E5%8A%A9%E7%90%86%E6%98%AF%E5%8B%9E%E5%B7%A5/ |date=20140407060640 }}公視新聞議題中心，2013/11/08</ref>。同月8日9時，林凱衡為慶祝台大敗訴，在台大行政大樓前的傅鐘廣場發放150份雞排 <ref>[http://www.coolloud.org.tw/node/76229 告勞委會 台大敗訴 工會發雞排 促校方改善助理勞動權]苦勞網，2013/11/08</ref> 。

林凱衡於2014年勞動節遊行時，表示「研究生30年前就領3000塊，至今卻還只領3000塊」，呼籲政府重視研究生勞動權益，善待人才 <ref>[http://www.epochtimes.com/gb/14/5/1/n4145094.htm%E6%B4%BE%E9%81%A3%E6%98%AF%E4%BD%8E%E8%96%AA%E5%85%83%E5%87%B6-%E5%8F%B0%E4%BA%A7%E4%B8%9A%E5%B7%A5%E4%BC%9A-%E7%BB%84%E5%B7%A5%E4%BC%9A%E5%BD%93%E5%8A%B3%E5%B7%A5%E5%90%8E%E7%9B%BE.html 派遣是低薪元凶 台产业工会：组工会当劳工后盾]大紀元，2014/05/01 </ref>。

== 太陽花學運期間 ==
[[太陽花學運|太陽花學運]]期間，2014年4月1日，林凱衡與其他台大、東吳的研究生自發性串聯前往監察院陳情，因警方強制驅離佔領行政院的民眾，要求監委彈劾當時的行政院長江宜樺與警政署長王卓鈞。對此事，林凱衡表示：「當天警方強制驅離民眾與學生，到現在沒人負起法律責任，但這場驅離行動違反了比例原則，民眾並沒有使用暴力，警方為何要強制驅離？」<ref>[http://iservice.libertytimes.com.tw/2013/specials/stp/news.php?rno=6&no=979561&type=l 學生們赴監院 要求彈劾江揆與王卓鈞] 自由時報 2014/04/01</ref>

== 台北市政府勞動局長遴選 ==
林凱衡於2014年12月3日宣布報名[[臺北市政府勞動局|臺北市政府勞動局]]局長備選人選。他主張以下三點 <ref>[https://www.facebook.com/kenny.lin.98284/posts/10153310375263362] 林凱衡個人臉書頁面 2014/12/03</ref>：
# 重新檢討政府人力需求，降低政府臨時雇用。 
# 鼓勵勞工組織產業與企業工會。
# 建立違規廠商資料庫，開放線上檢索。
林凱衡認為「即使政府權力再大，勞檢員再多，不一定可遏止違法行為。」光仰賴政府監督是不夠的，所以他希望增加工會組織，並強化其能力，防止讓勞方總落入勞、資、官三方中，比較弱勢的地位。<ref>[http://pnn.pts.org.tw/main/2014/12/06/%E5%AD%B8%E8%80%85%E3%80%81%E5%AD%B8%E7%94%9F%E3%80%81%E5%B7%A5%E9%81%8B%E4%BA%BA%E5%A3%AB-%E8%AA%B0%E6%8E%8C%E5%8B%9E%E5%8B%95%E5%B1%80%EF%BC%9F/ 【局長自己選】學者、學生、工運人士…… 誰掌北市勞動局？] {{Wayback|url=http://pnn.pts.org.tw/main/2014/12/06/%E5%AD%B8%E8%80%85%E3%80%81%E5%AD%B8%E7%94%9F%E3%80%81%E5%B7%A5%E9%81%8B%E4%BA%BA%E5%A3%AB-%E8%AA%B0%E6%8E%8C%E5%8B%9E%E5%8B%95%E5%B1%80%EF%BC%9F/ |date=20141210012909 }} 公視新聞議題中心</ref>

林凱衡通過第一階段45選23後，於同年12月8日進入第二階段23選10的名單，於同月10日參與政見發表會時，謙虛地表示有許多資歷勝過自己的前輩未通過遴選，而呼籲遴選委員會加辦額外的政見發表會，讓這23人的理念也能公開被檢視。<ref>[https://livehouse.in/channel/KPtaipei/record/-JcmYR4hhiAQY8YiC9Y6 141210 I-voting 「自己的勞動局長，自己選」5:30:00 起] {{Wayback|url=https://livehouse.in/channel/KPtaipei/record/-JcmYR4hhiAQY8YiC9Y6 |date=20141210135417 }} LIVEhouse.in</ref>
