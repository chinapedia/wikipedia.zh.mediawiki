'''贺长龄'''（{{bd|1785年||1848年|catIdx=H贺}}），[[表字|字]]'''耦耕'''，[[号|号]]'''耐庵'''，[[湖南|湖南]][[善化縣|善化縣]]人，祖籍[[浙江|浙江]][[会稽|会稽]]。清朝學者，政治人物。

== 简介 ==
[[贺熙龄|贺熙龄]]之兄，[[嘉庆|嘉庆]]十二年（1807年）舉丁卯科鄉試第一，嘉慶十三年（1808年）成戊辰科进士，选[[庶吉士|庶吉士]]，[[散馆|散馆]]授[[编修|编修]]<ref> 《大清仁宗受天兴运敷化绥猷崇文经武孝恭勤俭端敏英哲睿皇帝实录》（卷之二百九）：嘉庆十四年。……引见戊辰科散馆及补行散馆人员。得旨、此次散馆之修撰吴信中、编修谢阶树、及补行散馆之修撰吴廷琛、业经授职。其清书二甲之庶吉士高翔麟、费丙章、汉书二甲之庶吉士贺长龄、陈官俊、周之琦、钱林、张葆、刘嗣绾、史评、陶梁、沈岐、陈传经、林培厚、王耀辰、刘荣黼、董国华、朱棨、胡承珙、罗家彦、宫焕、李可琼、夏国培、饶绚春、李恩绎、于德培、恩宁、胡兆兰、俱著授为编修。汉书三甲之庶吉士王道行、冯缵、俱著授为检讨。尹济源、吴恩韶、潘恭常、赵植庭、赵光禄、沈学廉、赵维熊、何珣、吴昌龄、萧应荃、程钟灵、姚丙成、唐业谦、何增元、张美如、杨镇、戴宗沅、廖敦行、吴其浚、钱仪吉、区玉麟、隆文、龚以镗、宗室敏勤、俱著以部属用。于学宗、魏元烺、曾冠英、李聚元、郑城、杨本浚、徐江、屠倬、贾秉钟、朱廷庆、涂晋、王锡蒲、王榘曾、唐善良、郝兆钰、李建北、杨煊、李桂林、俱著以知县即用。邵凤依、赵钟彦、徐步云、施鸾坡、俱著归原班铨选。</ref>，迁[[赞善|赞善]]。[[道光|道光]]元年（1821年），出为[[江西|江西]][[南昌府|南昌府]]知府。道光四年闰七月十一，由山东兖沂曹济道道员升任[[廣西按察使|廣西按察使]]。道光四年九月初七（1824年10月28日），同江苏按察使互调<ref>{{cite news|title=《广西通志 政府志 第一篇 晚清广西省政府及农民政权 第一章 政府机构　第一节 省级机构》|url=http://www.gxdqw.com/bin/mse.exe?seachword=&K=a&A=15&rec=4&run=13|accessdate=2013-01-28|newspaper=广西地方志}}{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。后升任[[山东|山东]][[巡抚|巡抚]]、[[江宁|江宁]][[布政使|布政使]]等职，乞归养亲。道光十五年（1835年），母丧[[服阕|服阕]]，补[[福建|福建]]布政使，调[[直隶|直隶]]。道光十六年（1836年），擢[[贵州|贵州]]巡抚，治黔九载，平息民乱，振兴文教。道光二十五年（1845年），擢[[云贵总督|云贵总督]]，兼署[[云南巡抚|云南巡抚]]。期间因平定[[回族|回族]]反清运动不力，降补[[河南|河南]][[布政使|布政使]]。道光二十七年（1847年），乞病归，又被前事追究，革职。次年卒。

== 政绩 ==
贺长龄为官40年，为官清廉，为政勤勉，泽披天下。[[山东省|山东省]]多水患，贺长龄带领人民开沟洫，兴修水利，使得粮食收获倍增。贺长龄在[[贵州|贵州]]为官时，主张查禁私种[[罂粟|罂粟]]和吸食[[鸦片|鸦片]]。整顿吏治，练营伍，储粮备荒，恤孤抚幼，劝课桑棉，教以纺织，创建书院义塾，得到了当地人民的交口称赞。

== 作品 ==
委托幕友纂辑《[[皇朝经世文编|皇朝经世文编]]》120卷，著有《耐庵文存》等。
