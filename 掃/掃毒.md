《'''掃毒'''》（'''The White Storm'''）是一部於2013年上映的香港警匪动作片，由[[陈木胜|陈木胜]]监制并执导，[[刘青云|刘青云]]、[[古天乐|古天乐]]和[[张家辉|张家辉]]领衔主演。[[中國大陸|中國大陸]]於2013年11月29日上映，香港於同年12月5日上映。获得[[第33届香港电影金像奖|第33届香港电影金像奖]]最佳电影、最佳导演等8项提名。

==劇情==
马昊天、张子伟与苏建秋是从小一起长大的好兄弟，三人长大后都在[[香港警務處|香港警務處]][[毒品調查科|毒品調查科]]駐守。其中阿秋在贩毒集团里做[[卧底|卧底]]很久，不能过正常人生活让他极为烦躁，大哥阿天和探员兄弟阿伟常劝解说，捉住最后一个大毒枭他就能归队。[[香港警務處憲委級職級|警队高层]]要阿秋继续跟进以放长线钓大鱼，阿秋深感不满。最后三兄弟决定一起赴[[泰国|泰国]]抓捕大毒枭八面佛，帮阿秋解脱。不过阿秋出于安危的顾虑不愿意跟八面佛进行假的毒品交易，就偷偷用电话跟对方联系说交易取消了，导致八面佛对购买毒品一方阿秋等人的怀疑。双方在战斗后，逼得昊天不得不在张子伟与苏建秋两人之间选择一个他能救走之人，最后昊天选择拯救阿秋，而子伟则身中一枪并被推入[[鳄鱼|鳄鱼]]潭中，子伟的死让昊天和阿秋都深感愧疚，两人也负责照顾起了患病失忆住院的子伟妈妈，阿秋之妻在生下女儿后与阿秋分了手。

五年后，八面佛之子在港被尖沙咀黑帮段坤所杀，阿秋和昊天在调查这件事时发现子伟并没死，他还娶了米娜成为了八面佛的女婿。三人通过阿秋之女被绑事件，终于化解了彼此之间的怨恨，兄弟情再次得以重生、阿秋也与妻子和女儿和好，并得知原来是子伟联合段坤杀害了八面佛之子，以计划引出八面佛并除掉之。在八面佛为死去的儿子做法事之夜，三兄弟选择与八面佛决一死战。而已被八面佛怀疑的子伟首先被迫砍掉了自己的一只手，激烈枪战后只剩下阿秋、昊天和八面佛三人存活，而在子弹不足的情况下，昊天则用自己的身体当盾牌做出牺牲，八面佛终于被阿秋打死。

==票房==
=== 中國大陸 ===
首周三天排映近76000场，场均观众35人收8230万人民幣居亚，比周冠军《[[地心引力_(电影)|地心引力]]》还多出15人<ref >{{cite web|url=http://news.mtime.com/2013/12/03/1521309.html | language= | title="地心引力"内地连冠 "扫毒"三天破八千万居亚 |publisher=Mtime时光网 |date=2013年12月3日 |accessdate= }}</ref>。陈木胜执导的新片口碑不俗，第2周《扫毒》在《[[无人区_(电影)|无人区]]》和《[[四大名捕II|四大名捕2]]》两部强有力新片的夹击下，还能比上周多收12%，即9200万，两周共取下1.74亿，比去年同期的《[[王的盛宴|王的盛宴]]》高出1亿以上<ref>{{cite web |url= http://news.mtime.com/2013/12/10/1521547.html|title="无人区"首周票房登顶 "四大名捕2"报收九千万 |date=2013年12月10日 |publisher=Mtime时光网}}</ref>；第3周电影受类型和阵容相近的《[[风暴_(电影)|风暴]]》影响，折损一半，进账4550万，位列该周票房榜第4名<ref>{{cite web |url= http://news.mtime.com/2013/12/17/1521870.html|title="《风暴》首周夺冠 贺岁档前半程港产片成主力 |date=2013年12月17日 |publisher=Mtime时光网}}</ref>，第四周拿下1488万排在第五位，四周总票房為2.35亿<ref>{{cite web |url=http://news.mtime.com/2013/12/24/1522147.html |title=《私人订制》首周创纪录夺冠 掀起贺岁档热潮 |date=2013年12月24日 |publisher=Mtime时光网}}</ref>。

=== 香港 ===
《扫毒》在香港上映时间比内地稍晚，但是香港市场则比内地更佳，首日在41家戏院排330场，场均人次75，周日的场均人次攀升过百；这部电影几乎在香港市场上取得了压倒性的优势，四天便超过1000万，為該周票房冠軍<ref>{{cite web |url=http://yule.sohu.com/20131211/n391621263.shtml|title="1202-1208香港票房 《扫毒》横扫市场 |date=2013年12月11日 |publisher=搜狐娱乐 }}</ref>。次周拿到1071万的票房排在亚军位置<ref>{{cite web |url=http://news.mtime.com/2013/12/20/1522039.html|title="香港票房："霍比特人"首周夺冠 "冰雪大冒险"上榜 |date=2013年12月20日 |publisher=Mtime时光网 }}</ref>；第3周拿502万位列季軍，三周累计2723万<ref>[http://www.hkfilmart.com/boxofficedetail.asp?lang=cn&wbid=981 12月16日至12月22日票房紀錄] {{webarchive|url=https://web.archive.org/web/20131224121253/http://www.hkfilmart.com/boxofficedetail.asp?lang=cn&wbid=981 |date=2013-12-24 }}</ref>。第4周取得336萬位列第六，四周累計2977萬。第5周取得144萬，累計3121萬。

==评价==
[[网易|网易]]的影评人写道：“陈木胜的《掃毒》掉头玩起来复古了，说来说去整部电影倒跟扫毒一点关系没有，两个半小时，看的是三个大男人哭天抹泪的兄弟情谊。演的是警察故事，讲的是黑道忠义，赶的是扫毒时髦。但是也就是港片，才能把警察也拍出江湖感。陈木胜拿手的是武戏，《扫毒》的枪战和爆破亦十分精彩，一场泰国二十分钟的枪战戏，场面大到如同灾难片，直升机扫射，追车，撞车，一气呵成，毫无断层，观赏性绝对值回票价。三位演员也挑不出什么毛病，打的哭的都颇卖力气。而文戏设计则颇为情绪化，导演和演员都被剧本深深伤到了。《扫毒》里说的是一个[[江湖|江湖]]，也只有江湖才会有“义气大过天”这么天真的东西存在。可剧情天真的实在有些过分，里面处处都是上个世纪[[吴宇森|吴宇森]]用过的英雄演绎法。”<ref >{{cite web|url=http://ent.163.com/13/1128/04/9EO8H95G00034R77.html | language= | title=《扫毒》：导的好演的好编的不好 |publisher=网易娱乐 |date=2013年11月29日 |accessdate= }}</ref>

本片被[[搜狐|搜狐]]评为2013下半年十大华语佳片第五名<ref >{{cite web|url=http://yule.sohu.com/20140109/n393173811.shtml | language= | title=2013年下半年搜狐十大佳片十大烂片评选揭晓 |publisher=搜狐娱乐 |date=2014年1月9日 |accessdate= }}</ref>。
