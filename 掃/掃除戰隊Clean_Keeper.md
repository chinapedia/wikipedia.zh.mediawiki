《'''掃除戰隊Clean Keeper'''》（[[日語|日語]]：{{lang|ja|お掃除戦隊くりーんきーぱー}}）是[[日本|日本]][[Idea_Factory|Idea Factory]]、[[Lupinus|Lupinus]]<ref>{{cite web|url=http://dol.dengeki.com/data/news/2007/12/7/35c7eb17cf54305e59995221e60be31a.html|script-title=ja:アイディアファクトリー、新ブランド「ルピナス」を設立！ 新作も発表|accessdate=2008-08-03|language=ja}}</ref>所發售的[[Wii|Wii]]以及[[PlayStation_2|PlayStation 2]]平台[[遊戲軟體|遊戲軟體]]。

== 概要 ==
《'''掃除戰隊Clean Keeper'''》是日本[[Idea_Factory|Idea Factory]]、[[Lupinus|Lupinus]]於2008年4月24日<ref>{{cite web|url=http://www.akibablog.net/archives/2008/04/wii-clean-080425.html|script-title=ja:「Ｗｉｉ伝説の幕開け」　お掃除戦隊くりーんきーぱー発売|accessdate=2008-08-03|language=ja}}</ref><ref>{{cite web|url=http://news.dengeki.com/elem/000/000/076/76256/index.html|script-title=ja:女の子とコスって戦え！ 『お掃除戦隊くりーんきーぱー』|accessdate=2008-08-03|language=ja}}</ref>發售的[[戀愛冒險遊戲|戀愛冒險遊戲]]<ref>{{cite web|url=http://www.famitsu.com/game/coming/1212276_1407.html|script-title=ja:学園を汚れから守れ！ 『お掃除戦隊くりーんきーぱー』 - ファミ通.com|accessdate=2008-08-03|language=ja}}</ref>，這一部是Lupinus的第一部作品<ref>{{cite web|url=http://www.gpara.com/article/cms_show.php?c_id=5793&c_num=14|script-title=ja:萌え系ブランド「ルピナス」誕生！第1弾は『お掃除戦隊』／ゲーム情報ポータル：ジーパラドットコム|accessdate=2008-08-03|language=ja}}</ref>，也是在Wii遊戲機上的第一款[[美少女遊戲|美少女遊戲]]<ref>{{cite web|url=http://plusd.itmedia.co.jp/games/articles/0805/21/news002.html|script-title=ja:今日は何の日？　萌え曜日！　「お掃除戦隊くりーんきーぱー」ミニレビュー：Wii初のギャルゲーは戦隊モノ!?　カワイイ女の子たちがコスってキレイにしちゃいます - ITmedia +D Games|accessdate=2008-08-03|language=ja}}</ref>。於2009年10月1日發行PlayStation 2版《'''掃除戰隊Clean Keeper H'''》（{{lang|ja|お掃除戦隊くりーんきーぱーH（ - はいぱ〜）}}），並且加入了新角色以及遊戲系統<ref>{{cite web|url=http://blog.livedoor.jp/geek/archives/50897108.html|script-title=ja:お風呂でバナナ　くりーんきーぱーＨ　「日本に生まれて、本当によかった！|accessdate=2010-07-27|language=ja}}</ref>。

玩家扮演主角源九朗，在校內搜尋可能成為掃除戰隊的隊員的女孩子和搭檔的掃除精靈。途中遇到特殊事件時切換到行動模式可以得到道具。

行動模式過關後則會顯示「獎勵CG」<ref>{{cite web|url=http://www.famitsu.com/game/coming/1213770_1407.html|script-title=ja:ちょっと(かなり？)エッチなご褒美CGを公開！ 『お掃除戦隊くりーんきーぱー』 - ファミ通.com|accessdate=2008-08-03|language=ja}}</ref>，在Wii的所有遊戲當中描寫性的要素比較多<ref>{{cite web|url=http://www.akibaos.com/?p=3354|script-title=ja:ロリか悪魔か「お掃除戦隊くりーんきーぱー」エロいってレベルじゃねーぞ！ - アキバOS|accessdate=2008-08-03|language=ja}}</ref>，因此被戲稱為Wii平台的[[日本成人遊戲|成人遊戲]]<ref>{{cite web|url=http://www.akibaos.com/?p=3211|script-title=ja:Wiiでエロゲ「お掃除戦隊くりーんきーぱー」わたしとコスってキレイキレイしましょう - アキバOS|accessdate=2008-08-03|language=ja}}</ref><ref>{{cite web|url=http://www.akibablog.net/archives/2008/04/wii-clean-080411.html|script-title=ja:Wii用ゲーム：お掃除戦隊くりーんきーぱー　「ちょっｗｗこれなんてエロゲー」|accessdate=2008-08-03|language=ja}}</ref>。

== 故事簡介 ==
就讀高天原高中的主角'''源九朗'''在打掃走廊時，遇到了掌管掃除的精靈'''瑪莉'''。美化委員長九朗為了對抗污染學園的邪惡組織'''哥瓦爾斯基'''（{{lang|ja|ゴワルスキー}}），必須尋找總共五名的掃除戰隊'''Clean Keeper'''（{{lang|ja|くりーんきーぱー}}）的隊員，並且與掃除精靈簽訂契約，背負著阻止哥瓦爾斯基組織繼續污染學校，就這樣九朗開始搜尋Clean Keeper隊員以及掃除精靈……。

== 登場人物 ==
; {{nihongo|源 九朗|源 九朗（みなもと くろう）||[[聲優|聲優]]：[[代永翼|代永翼]]<ref group="注">聲優為廣播劇版，遊戲本篇則沒有聲音。</ref>}}
:本作品主角，就讀高天原高中二年級。被任命為Clean Keeper的隊長之前過著平凡的學生生活。但是決定班級委員的那一天正好缺席的緣故，被任命為美化委員長，不過他並不討厭掃除。個性認真但是總是嫌其他人麻煩，對自己的身高非常在意。

=== 掃除戰隊 ===
; {{nihongo|春日野 麻-{里}-子|春日野 麻里子（かすがの まりこ）||聲優：[[庄子裕衣|庄子裕衣]]}}
:就讀二年級，17歲，與九朗同班的青梅竹馬。個性開朗又愛管閒事，但是不喜歡強出頭。不擅長接觸九朗以外的異性。興趣是製作甜點。
; {{nihongo|夏川 京子|夏川 京子（なつかわ きょうこ）||聲優：[[田村由香里|田村由香里]]}}
:就讀二年級，17歲，麻-{里}-子的好朋友，參加演劇社。個性有點[[天然呆|天然呆]]，由於個性的緣故成績也不太好。
; {{nihongo|秋葉 瑠璃|秋葉 瑠璃（あきは るり）||聲優：[[中原麻衣|中原麻衣]]}}
:就讀一年級，16歲，參加游泳社。個性像男孩子一樣行動力旺盛，對可愛的東西沒有抵抗力，特別是精靈拉碧絲柔軟的臉頰。
; {{nihongo|冬杜 晶|冬杜 あきら（ふゆもり あきら）||聲優：[[酒井香奈子|酒井香奈子]]}}
:就讀三年級，18歲，運動神經非常好，個性坦率的女孩子。雖然是裡面年紀最大的，但是跟九朗一樣個子不高，討厭被說成像小孩子一樣。
; {{nihongo|倉木 文繪|倉木 ふみえ（くらき ふみえ）||聲優：[[高橋美佳子|高橋美佳子]]}}
:就讀一年級，16歲。個性膽怯不擅長與別人說話，。經常不去上課而待在圖書館自習，成績是全年級最高的。喜歡狹窄的地方。
:由於甜點的誘惑而加入哥瓦爾斯基幹部，但是後來背叛了哥瓦爾斯基而成為Clean Keeper的一員。

=== 掃除戰隊的精靈 ===
; {{nihongo|瑪莉|まりー||聲優：[[原阿久美|原阿久美]]}}
:麻里子的掃除精靈，是九朗最早遇到的掃除精靈，不管遇到任何事情總是全力以赴，不過有時候不會通知。
; {{nihongo|坎恩|きゃん||聲優：[[平山緣|平山緣]]}}
:京子的掃除精靈，與搭檔京子個性相反，對京子的我行我素的性格顯得焦躁。討厭被說成是「羽蝨」。
; {{nihongo|拉碧絲|らぴす||聲優：[[藍川千尋|藍川千尋]]}}
:瑠璃的掃除精靈，在九朗等人發現之前與琉璃相遇並且簽訂契約。與琉璃有強烈的信賴關係。個性非常認真，但是會漏掉重要的部分。
; {{nihongo|亞莉亞|ありあ||聲優：[[雪野梨沙|雪野梨沙]]}}
:晶的掃除精靈，個性活力充沛而且好奇心旺盛。對晶相近的人總是意氣投合。
; {{nihongo|阿妮亞|あにゃ||extra=聲優：{{lang|ja|[[高城みつ|高城みつ]]}}}}
:文繪的掃除精靈，說話方法像中國人一樣為特徵。個性有時候少一根筋，經常忽略掉周圍的狀況。與亞莉亞特別要好。
