'''城口中学'''位于[[重庆市|重庆市]][[城口县|城口县]]，是重庆市[[重点中学|重点中学]]，是城口县唯一的一所高完中。学校前身为1942年成立的城口县立初级中学，现总占地面积78.3[[亩|亩]]，教职工251人。<ref>{{cite web |url=http://www.cqckzx.com/show.asp?id=206&cid=11 |title=城口中学简介 |accessdate=2016-08-11 |publisher=城口中学  }}</ref>

== 历史沿革 ==
*1942年，城口县立初级中学成立
*1958年，城口县立初级中学更名为[[四川省|四川省]]城口县中学校
*1997年，学校更名为重庆市城口中学校
*2008年，[[重庆市|重庆市]]城口中学校升格为重庆市[[重点中学|重点中学]]<ref>{{cite web |url=http://www.cqckzx.com/show.asp?id=207&cid=11 |title=历史沿革 |accessdate=2016-08-11 |publisher=城口中学  }}</ref>
