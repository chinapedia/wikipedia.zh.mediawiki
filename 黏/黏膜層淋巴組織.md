'''黏膜相關淋巴組織'''（{{lang|en|Mucosa-associated lymphoid tissue}}，簡稱'''MALT''')，位於消化道、呼吸道、泌尿生殖道等人體各種黏膜組織中，由黏膜表皮細胞下方的淋巴小結以及表皮細胞之間的[[微皱褶细胞|微皱褶细胞]]組成。<br>MALT是一种无明确范围的低浓度的淋巴组织，它可以在你的身体各处被找到，例如：胃肠道，甲状腺，乳房，肺部，唾液腺，眼部，皮下。<br />

黏膜層淋巴組織充满淋巴细胞，如：[[T细胞|T细胞]]和[[B细胞|B细胞]]，还有[[浆细胞|浆细胞]]和[[巨噬细胞|巨噬细胞]]，每个黏膜层淋巴组织都正好位于抗原需要通过的黏膜上皮处。pithelium. 比如说肠道上的MALT，当[[微皱褶细胞|M细胞]]同时存在时，M细胞从肠道内捕捉抗原并传递给淋巴组织。

== 組成 ==
MALT的組成部份有時會再細分如下：
* '''[[腸淋巴組織|腸淋巴組織]]''' (gut-associated lymphoid tissue，GALT) [[淋巴小结|淋巴小结]]是在[[小肠|小肠]]内壁被发现的肠淋巴组织的一部分。
* '''[[支氣管淋巴組織|支氣管淋巴組織]]''' (bronchus-associated lymphoid tissue, BALT)
* '''[[鼻淋巴組織|鼻淋巴組織]]''' (nose-associated lymphoid tissue, NALT)
* '''[[咽喉淋巴組織|咽喉淋巴組織]]''' (larynx-associated lymphoid tissue, LALT)
* '''[[皮膚淋巴組織|皮膚淋巴組織]]''' (skin-associated lymphoid tissue, SALT) 
* '''[[血管淋巴组织|血管淋巴组织]]''' (vascular-associated lymphoid tissue, VALT)一种最近被承认的存在于血管中的淋巴组织，但它的免疫作用还不清楚。
* '''[[眼淋巴组织|眼淋巴组织]]''' (eye-associated lymphoid tissue, EALT)由结膜淋巴组织和泪腺淋巴组织组成。

*淋巴小結：內含B細胞、T細胞、巨噬細胞
*M細胞：負責捕捉抗原

== 在致病方面扮演的角色 ==
MALT在调节黏膜[[免疫|免疫]]力方面起着重大作用。但它也可能成为淋巴瘤，通常成为非霍奇金淋巴瘤。一个确定的实例就是在[[胃|胃]]部与[[幽門螺桿菌|幽門螺桿菌]]相接的黏膜層淋巴瘤。

== 歸巢性 ==
1964年James Gowans發現在黏膜活化的淋巴細胞經胸導管、血液又落戶于黏膜部位發揮針對同一抗原的免疫反應發揮作用，而外周組織中活化的淋巴細胞就不會進入黏膜。這種現象的一個好處是在黏膜某一處活化的淋巴細胞可以傳播到黏膜組織的各個部位，稱之為「common mucosal system」現象。
