{{engname|《汪洋中的一條船》|''He Never Gives Up''}}是依據[[台灣|台灣]]作家[[鄭豐喜|鄭豐喜]]所撰寫的[[自傳|自傳]]《[[汪洋中的一條船|汪洋中的一條船]]》改編的[[電影|電影]]，[[中央電影公司|中央電影公司]]出品。上映前，因為[[劇本|劇本]][[版權|版權]]問題，讓拍攝進度有所延宕；後來經由中影[[總經理|總經理]][[梅長齡|梅長齡]]與鄭豐喜遺孀[[吳繼釗|吳繼釗]]相談協議後，正式簽訂拍攝版權[[合約|合約]]<ref>[http://movie.cca.gov.tw/Topics/Content.asp?ID=678 台灣電影筆記]{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

上映後，旋即造成盛況空前的轟動，無人不知此部家喻戶曉的影片，觀眾皆無不動容落淚，創下極高的賣座紀錄。
[[File:CMPC_He_Never_Gives_Up_snapshot_report_20131120.jpg|thumb]]

== 概述 ==
故事劇情述說鄭豐喜奮發向上的歷程，以尊嚴克服逆境創造自己的人生，最終成為一位生命的勇士，帶給世人艱辛燦爛的榜樣；亦反映當時國家處境，及生命意義的反思。整個舞臺在[[嘉義縣東石鄉港墘國民小學|嘉義縣東石鄉港墘國民小學]]取景<ref>[http://outdoor.cyc.edu.tw/outdoor/dong/b4.htm 港墘國小]</ref>，並曾依製作需求前往[[臺北市|臺北市]][[中正區_(臺北市)|中正區]][[寶藏巖|寶藏巖]]拍攝<ref>[http://www.tvbs.com.tw/news/news_list.asp?no=blue20070131145715 電影「南國再見！南國」　取景寶藏巖]，[[TVBS|TVBS]]新聞</ref>，構築出早期台灣艱困生活環境、以及鄭豐喜不屈不饒的求學過程。

== 劇情簡介 ==

台灣[[雲林縣|雲林縣]][[口湖鄉|口湖鄉]]的一處農家裏，鄭老伯期待著第十一個孩子的出生，並取名為豐喜。

豐喜出生後，患有先天性小腿萎縮，雙足畸形。可是，家境貧寒的豐喜幾近遭到拋棄的厄運，幸得爺爺的阻止和呵護，才得以留在家裏。由於，豐喜在爺爺的呵護下，從小就很聰明懂事，博得鄰居和賣藝趙伯伯的喜歡。

六歲的豐喜，為減輕家庭生活負擔，遂成趙伯伯的徒弟，跟著四處耍猴賣藝，彼此相依為命。那段日子裏，豐喜在艱難困苦的人生際遇中，不僅學會做人的道理，還磨練堅韌的[[意志|意志]]。趙伯伯被心臟病奪去生命後，豐喜只能獨自流浪生活，一直到被偶遇的老鄉帶回為止。

返家的豐喜，並沒有跟父母一起居住，跟著五哥在郊外養起鴨子，靠著辛苦勤奮的努力工作，憑藉一百多隻鴨子養活自己。某日，一場特大暴風雨，不僅沖走所有的鴨子，還差點奪去豐喜的生命。此後，豐喜的父母再也不讓他獨自離家生活。

九歲的豐喜，進入學校求學，靠著自強不息的毅力和求存圖強的進取精神，榮獲優異的學習成績。十幾年後，高中畢業的豐喜事跡經當地報紙報導後，引起社會的強烈反響。為此，校長和醫院院長特為豐喜裝上假肢，實現終於能夠站著走進大學校門的願望。

大學期間，豐喜一方面持續打工掙錢，另一方面將自己的經歷寫成自傳《汪洋中的一條船》參加徵文比賽，引起台灣社會廣大的迴響。此書深深感動著同校法律系女學生吳繼釗，儘管她有來自父母的反對壓力，仍毅然持續交往。大學畢業後，倆人結為夫妻，並育有兩女。

1975年，鄭豐喜因[[肝癌|肝癌]]逝世，得年31歲。

== 逸事 ==
1985年，《汪洋中的一条船》和同屬中影出品的《[[源|源]]》在中国大陆各地公开上映，深获好评<ref name="黄仁">[http://www.xici.net/b509254/d37715065.htm 黄仁：两岸电影文化交流合作的影响和成果（1924-2005）]2006-5-18</ref>；這是自1949年[[國民政府|國民政府]]遷入台灣以來，首次在[[中國大陸|中國大陸]]公開放映的[[台灣電影|台灣電影]]之一<ref>1985年之前，[[白景瑞|白景瑞]]的《[[家在台北|家在台北]]》和[[李行|李行]]的《[[彩云飞|彩云飞]]》曾先後在[[广州|广州]]、[[上海|上海]]、[[北京|北京]]等地上映。</ref>。另一说，《汪洋中的一条船》是1986年10月在上海首映，之后再在大陆各地公映<ref name="黄仁"/>。但是，此时中国大陆并未获得《汪洋中的一条船》版權<ref>[http://www.pts.org.tw/special/000129/about.htm 汪洋中的一條船：鄭豐喜]，[[公共電視台|公共電視台]]</ref>。1986年7月2日，中影總經理[[林登飛|林登飛]]則譴責[[中國共產黨|中國共產黨]]未取得電影版權、逕行[[拷貝|拷貝]]影片後擅自在中國大陸公開放映，並下令追查拷貝流出地區<ref>[http://cinema.nccu.edu.tw/cinemaV2/twfilm_s_list.htm?KEYWORD=%A8L%ACv%A4%A4%AA%BA%A4@%B1%F8%B2%EE&DATE=%20&Login=%ACd%B8%DF 台灣電影史料庫]{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}，〈[[台灣電影資料庫|台灣電影資料庫]]〉</ref>。1987年，[[百花奖|百花奖]]将《汪洋中的一条船》中的[[秦漢_(演員)|秦漢]]列为最佳男主角候选人之一<ref name="黄仁"/>。1988年，台湾电影以[[转口贸易|转口贸易]]方式正式在中国大陆放映。1990年，李行等台湾电影导演代表团正式将《汪洋中的一条船》等电影带到中国大陆供观摩<ref name="黄仁"/>。

李行因此片於1978年榮獲[[中華民國總統|中華民國總統]][[蔣經國|蔣經國]]頒發[[華夏獎章|華夏獎章]]，成爲台灣電影界榮獲華夏獎章的第一人。
