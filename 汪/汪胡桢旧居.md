'''汪胡桢旧居'''又名'''湖滨小筑'''，位于[[中国|中国]][[浙江省|浙江省]][[嘉兴市|嘉兴市]][[南湖区|南湖区]][[建设街道|建设街道]]梅湾街东区帆落浜39号，地濒鸳水边，为近现代水利工程学家[[汪胡桢|汪胡桢]]在嘉兴的住所，目前正由嘉兴市政府进行对外开放前的修复和陈列布展工作<ref>{{cite web |url=http://www.zj.gov.cn/art/2013/5/31/art_13130_722831.html|title=《嘉兴市召开汪胡桢故居陈列大纲论证会》|accessdate=2013-11-13|work=|publisher=浙江省人民政府}}</ref>。1928年汪胡桢任太湖水利工程处副总工程师时在此购地筑楼，以作奉母养颐之所，后毁于抗战，1948年重建<ref>{{cite web |url=http://www.jxgaj.gov.cn/Html/201210/03/30337.html|title=《汪胡桢故居》|accessdate=2013-11-13|work=|publisher=嘉兴市公安局}}</ref>。旧居占地2466平方米，主楼为工字型的西式花园洋房<ref>{{cite web|url=http://zjbwg.zjol.com.cn/zjbwg/system/2011/12/07/018057064.shtml|title=《嘉兴市领导专题调研汪胡桢故居保护工作》|accessdate=2013-11-13|work=|publisher=浙江在线|deadurl=yes|archiveurl=https://web.archive.org/web/20131113144747/http://zjbwg.zjol.com.cn/zjbwg/system/2011/12/07/018057064.shtml|archivedate=2013-11-13}}</ref>，大门一侧有2010年所立的汪胡桢雕塑<ref>{{cite web|url=http://www.zj.xinhuanet.com/website/2010-02/09/content_18999467.htm|title=《嘉兴：汪胡桢雕塑昨在梅湾街东区落成》|accessdate=2013-11-13|work=|publisher=新华网浙江频道|deadurl=yes|archiveurl=https://web.archive.org/web/20131113151932/http://www.zj.xinhuanet.com/website/2010-02/09/content_18999467.htm|archivedate=2013-11-13}}</ref>。
