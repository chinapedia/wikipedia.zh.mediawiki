'''汪平雲'''（{{bd|1967年|11月1日|2007年|2月12日|catIdx=Wang汪平雲}}），出生於[[台灣|台灣]][[台北|台北]]，[[律師|律師]]。曾任[[中華民國行政院|行政院]][[蒙藏委員會|蒙藏委員會]]簡任秘書、參事、蒙藏委員，並擔任[[台灣智庫|台灣智庫]]諮詢委員、[[21世紀憲改聯盟|21世紀憲改聯盟]]顧問及[[國際通商法律事務所|國際通商法律事務所]]律師。

== 生平 ==
就讀[[台大|國立台灣大學]]期間，曾任「大學新聞社」總編輯，並曾參選[[台大學生會|台大學生會]]會長，為[[野百合學運|野百合學運]]領袖之一，曾任[[野百合學運|野百合學運]]第二屆七人決策小組成員，並參與同時期諸多爭取民主改革之社會運動。

汪專長為[[公法|公法]]領域，學生時代開始即協助[[民主進步黨|民主進步黨]]撰寫[[法律|法律]]相關論述，並多次主筆撰寫各項[[立法|立法]]、修法及[[釋憲|釋憲]]聲請書，包括「[[檢察官|檢察官]][[羈押|羈押]]權違憲」案、「[[副總統|副總統]]兼任[[行政院院長|行政院院長]]違憲」案、「[[全民健保|全民健保]][[地方政府|地方政府]]補助保費違憲」案（以[[行政院|行政院]]立場發表反對意見）、「[[公民投票法|公民投票法]]」立法、「[[三一九槍擊事件|319事件]][[真調會|真調會]]違憲」案、戶籍法「強制按捺[[指紋|指紋]]違憲」案、「[[國家通訊傳播委員會|NCC]]組織法違憲」案、「總統[[豁免權|豁免權]]」釋憲聲請書等，被視為民進黨最重要的法律幕僚之一。但因其[[外省人第二代|外省子弟]]身分，顧及家族親友感受，汪行事極為低調，社會大眾對其極少瞭解。

2002年因其業師[[許志雄|許志雄]]接任[[蒙藏委員會|蒙藏委員會]]委員長，隨許進入蒙藏委員會，曾任秘書、參事、蒙藏委員。

== 去世 ==
2007年2月12日，汪平雲向其妻王春芳表示心情不佳，獨自出外散心，但徹夜未歸。2007年2月15日，被人發現溺斃於[[木柵路|木柵路]]四段[[萬壽橋|萬壽橋]]下，未發現[[遺書|遺書]]。王表示，汪因壓力大，長期失眠。警方判斷疑因[[憂鬱症|憂鬱症]]厭世。

2007年3月14日，汪平雲[[公祭|公祭]]於[[臺北市第二殯儀館|臺北市第二殯儀館]]舉行，宣讀汪平雲[[褒揚令|褒揚令]]與民主進步黨褒揚狀。

== 逸聞 ==
汪1986年考入台大[[數學|數學]]系，第二年重考進入[[歷史|歷史]]系，兩年後降轉至[[哲學|哲學]]系，最後休學入伍服役，退伍後又重新考入[[法律|法律]]系，同其他優秀學長類似，汪畢業前即通過[[律師|律師]]、[[司法官|司法官]]等各種考試，並以第一名考入台大法研所獲得[[碩士|碩士]]學位。
就讀[[建國中學|建國中學]]高三時，獲邀參加英語研習社年度英語話劇公演，擔綱演出[[莎士比亞|莎士比亞]]名劇《羅密歐與茱莉葉》，扮演男主角羅密歐。
