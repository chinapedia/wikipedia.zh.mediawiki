'''嚴敏華'''（{{bd|1992年|5月18日||}}），綽號「小辣椒」，香港活躍社運人士。畢業於[[保良局胡忠中學|保良局胡忠中學]]，2010年起因曾與香港多宗抗議活動，引起香港媒體關注。她出身基層，父親是退休司機，母親區家寶任職保健推拿師；2015年年底和來自非洲多哥的丈夫註冊結婚，丈夫來港後從事工程，婚後曾小產一次，和丈夫居於大角咀的劏房。<ref name="apple20180907">{{cite news|url=https://hk.news.appledaily.com/local/realtime/article/20180907/58649518 |title=【東北抗爭者】「社運小辣椒」嚴敏華化身保險經紀 走進人生新一頁 |publisher=蘋果日報 |date=2018-09-07 }}</ref>

==生平==
===聲援艾未未===
2011年，《[[金融時報|金融時報]]》亞洲版頭版刊登嚴敏華聲援[[艾未未|艾未未]]被捕時的照片，使她成為國際新聞人物。<ref name="rfa._香港少女">{{Cite web | title=香港少女聲援艾未未成為國際新聞人物 | url = http://www.rfa.org/cantonese/news/hk_post_90s-04192011104647.html|work = 自由亞洲電台| date =2011-04-19}}</ref>

===咬人案===
嚴敏華因在2012年[[七一遊行|七一遊行]]中咬傷一名女警長，被法庭裁定襲警罪名成立，時年20的她被即時還押[[更生中心|更生中心]]。<ref>{{Cite web | title = 小辣椒咬警判更生中心 | url = http://life.mingpao.com/cfm/dailynews3b.cfm?File=20130220/nalgh/gha1.txt| work = 明報| date = 2013-02-20}}</ref>嚴敏華上訴至[[香港高等法院|香港高等法院]]被駁回。因年滿21歲被改判入獄21天。不過扣除還押的21天後獲得即時釋放。<ref>{{Cite web | title = 嚴敏華咬女警上訴被駁回 法官稱屬重罪不姑息 |url = http://news.takungpao.com.hk/hkol/politics/2013-06/1664556.html | work = 大公網 | date =2013-06-04 }}</ref>

===被拒入境中國內地===
2014年11月，嚴敏華在[[羅湖|羅湖]]企圖進入[[深圳|深圳]]時被深圳關員拒絕入境，深圳當局指她「擾亂外交及影響國家形象」。<ref>{{Cite web | title = 嚴敏華指深圳指她擾亂外交及影響國家形象而被拒入境 | url = http://www.881903.com/Page/ZH-tw/newsdetail.aspx?ItemId=759259&csid=261_341 | work = 商業電台 | date = 2014-11-09 }}</ref><ref>{{Cite web | title = 【雨傘革命】嚴敏華帶雨傘禁入深圳 海關指擾亂外交 | url = http://hk.apple.nextmedia.com/realtime/news/20141108/53109314 | work =蘋果日報 | date = 2014-11-08 }}</ref>

===反東北案===
2014年6月，她因參與[[反新界東北撥款示威|反新界東北撥款示威]]被捕。在2016年2月19日被裁定參與非法集結罪成立，被判120小時[[社會服務令|社會服務令]]，後來律政司不滿刑期過輕，向高等法院上訴法庭申請覆核刑期。2017年8月15日，法庭裁定律政司覆核刑期得直，嚴敏華被改判入獄13個月，她第二次入獄。<ref>{{cite web|title=【衝擊立會案】上訴庭准許加刑 12被告入獄13個月|url=http://hk.apple.nextmedia.com/realtime/news/20170815/57082514|publisher=蘋果日報|accessdate=2017-08-16|date=2017-08-15}}</ref> 2018年2月7日，嚴敏華、梁穎禮和朱偉聰獲准保釋等候上訴結果。2018年9月8日，反東北發展示威者向終審法院向提出上訴案審結，首席法官馬道立指上訴人行為雖屬暴力，但上訴庭量刑起點明顯過重，加上13人已服刑逾100天，裁定13名反東北發展示威者上訴得直，當庭釋放。<ref>{{cite news|url=https://hk.news.appledaily.com/local/realtime/article/20180908/58657822 |title=東北案終極上訴 13人得直獲釋 官質疑量刑起點過重 |publisher=蘋果日報 |date=2018-09-07 }}</ref>

===當保險經紀===
2018年6月，嚴敏華考獲保險代理牌照，正式開展當保險經紀的事業。她表示從無後悔過公民抗命，但若知會入獄，可能會考慮多些，因要為家人著想，抗爭成本太高，坐過兩次監，不想再坐第三第四次監。<ref name="apple20180907"></ref>
