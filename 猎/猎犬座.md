'''猎犬座'''是北天的一个小[[星座|星座]]，17世纪由波兰天文学家波兰天学家[[約翰·赫維留|約翰·赫維留]]（{{lang|en|Johannes Hevelius}}）创立，代表[[牧夫座|牧夫座]]牵的两条狗''Chara''和''Asterion''（中文称作常陈四）。是三个代表狗的星座之一，其他两个是[[小犬座|小犬座]]和[[大犬座|大犬座]]。

== 历史 ==
[[File:Bootes_&_Coma_Berenices.gif|thumb]]
在古代希腊文献中，猎犬座的星被描绘为牧夫扛的棒子。后来被阿拉伯人翻译为钩子，或牧人的带钩牧杖。再翻译到西欧文字误成了狗，最终被[[赫维留|赫维留]]定成一个独立的星座。

猎犬座包含中国古代的星座“[[常陈_(星官)|常陈]]”。《晋书·天文志》记载：“常陈七星如毕状，在帝座北，天子宿卫武贲之士，以设疆御。”

== 重要的星 ==
* 最亮的星为猎犬座α星，中文称作[[常陳一|常陳一]]，視星等2.90等。
* [[獵犬座Y|獵犬座Y]]是一颗红色半规则[[变星|变星]], 周期158天，在4.7-6.2等之间变化。
* AM CVn是视星等14的蓝色星。它属于一类特殊的[[激变变星|激变变星]]： 其伴星不是[[主序星|主序星]]而是一颗[[白矮星|白矮星]]。
* [[常陈四|常陈四]]（猎犬座β）被认为很有可能包含行星。天文学家[[瑪格麗特·杜布爾|瑪格麗特·端貝爾]]（Margaret Turnbull）建议把它列为[[SETI|SETI]]射电望远镜搜寻的目标之一<ref>[http://www.stltoday.com/stltoday/news/stories.nsf/stlouiscitycounty/story/B9223C0D00A224D38625711A00286766?OpenDocument Is Beta CVn lively]</ref>。

== 深空天体 ==
包含5个[[梅西耶天體|梅西耶天體]]。之一是[[螺旋星系|螺旋星系]]即[[渦狀星系|M51]]，包含[[NGC_5194|NGC 5194]] 和不规则星系[[NGC_5195|NGC 5195]]；后者正对地球，于1845年被William Parsons观测到，是第一个被认为有螺旋结构的星系。猎犬座还包括[[向日葵星系|向日葵星系]]（M63或NGC 5055）,螺旋星系[[M94|M94]]和螺旋星系[[M106|M106]]。[[M3|M3]]([[NGC_5272|NGC 5272]])是一个球状星团，直径18&prime，6.3等，可以用双筒望远镜看见。
