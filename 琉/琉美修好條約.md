《'''琉美修好條約'''》（'''琉米修好條約'''）為[[琉球國|琉球國]]政府和[[美利堅合眾國|美利堅合眾國]]政府代表於1854年7月11日（[[咸豐|咸豐]]4年6月17日）在[[首里城|首里城]]簽訂的[[不平等條約|不平等條約]]。正式名稱為「'''亞米利加合眾國琉球王國政府議定約'''」。琉球方面代表為[[尚宏勳|尚宏勳]]、[[馬良才|馬良才]]，美國方面的代表為[[馬休·佩里|馬休·佩里]]。

== 背景 ==
1853年5月26日（[[咸豐|咸豐]]3年4月19日），[[馬休·佩里|馬休·佩里]]初次率「薩斯凱哈那」號巡洋艦」等4隻[[美國|美國]]軍艦進入[[那霸港|那霸港]]，5月28日會見琉球國總理官[[尚大模|尚大模]]（摩文仁按司）。6月6日訪問[[首里城|首里城]]，要求會見尚泰王，遭[[攝政_(琉球)|攝政]][[尚惇|尚惇]]（大里王子朝教）拒絕，但尚惇仍派尚大模在[[首里城|首里城]]北殿會見了他。6月8日，尚泰王遣使向馬休·佩里贈送禮物，次日，佩里留下「密西西比號巡洋艦」，率艦隊前往[[小笠原群島|小笠原群島]]。6月18日歸港。7月2日，因艦隊供給不足，將「薩斯凱哈那號巡洋艦」留在那霸港，持美國總統親筆書信，率4隻軍艦往[[日本|日本]][[浦賀|浦賀]]，與[[江戶幕府|江戶幕府]]談判。7月25日回到那霸港，得到[[琉球國|琉球國]]方面的供給。在與琉球方面達成四項協議後，佩里於8月1日率艦隊離開琉球前往[[香港|香港]]。

1854年1月21日，佩里又率艦隊來至那霸港，並再度訪問首里城。2月7日，在得到琉球方面的供給後駛往日本[[江戶|江戶]]。7月1日，佩里與日本江戶幕府簽訂《[[神奈川條約|神奈川條約]]》後返回琉球。7月7日與琉球交涉，要求簽訂條約。翌日開始談判，美國就1853年達成的四項協議加以修正，要求琉球方面最遲於7月10日作出答復。7月10日，雙方再度會談，琉方承認美國提出的修正案有效。次日，美方代表馬休·佩里與琉方代表[[尚宏勳|尚宏勳]]（仲里按司朝紀）正式簽訂條約並互換文件。

== 條約內容 ==
{{Wikisource|琉美修好條約}}
*（第一条）自由貿易
*（第二条）提供美國[[船舶|船舶]]薪水
*（第三条）對美國船漂流民的救助
*（第四条）承認美國領事裁判權
*（第五条）設置美國人墓地並保護
*（第六条）琉球國[[引水人|引水人]]相關規定
*（第七条）提供美國船舶薪水相關費用等

==全權==
*琉球國　[[尚宏勳|尚宏勳]]（仲里按司）、[[馬良才|馬良才]]（棚原親方）
*美利堅合眾國 [[馬休·佩里|馬休·佩里]]

== 補足 ==

1854年10月29日（[[咸豐|咸豐]]4年9月8日），[[薩摩藩|薩摩藩]]命令將第一條修正如下：

'''原文'''

一、此後合眾國人民到琉球，須要以禮厚待，和睦相交。其國人要求買物，雖官雖民，亦能以所有之物而賣之。官員無得設例阻禁百姓。凡一支一收，須要兩邊公平相換。

'''修正文案''' ※紅色粗體字為修正部分

一、此後合眾國人民到琉球，須要以禮厚待，和睦相交，其國人{{Color|red|'''欲'''}}買物，'''{{Color|red|則雖市店之品物達官所買貨者名記于品之若賣貨者以其物送官所以價錢與官吏而後品物交易專可司令官吏預聞雖}}'''阻禁'''{{Color|red|私議而己凡}}'''一支一收，須要'''{{Color|red|兩公邊公平相換。}}'''
