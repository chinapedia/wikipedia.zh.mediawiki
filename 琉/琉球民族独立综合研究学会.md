'''琉球民族独立综合研究学会'''（{{lang-en|The Association of Comprehensive Studies for Independence of the Lew Chewans}}，{{lang-ja|琉球民族独立総合研究学会}}）是一个主張[[琉球独立|琉球独立]]组织，由[[日本国|日本国]][[冲绳县|冲绳县]]当地政治家、大学教授、社会活动家以及市民团体成员，在2013年5月15日下午在冲绳縣成立。<ref>{{Cite web |url= http://mil.chinaiiss.com/html/20135/16/a5e79a.html |title= 琉球独立组织欲向联合国请愿 |date= 2013年5月15日 |publisher= 《[[环球日报|环球日报]]》 |language = zh |accessdate= 2013年5月15日 |quote=  }}</ref>

該學會由沖繩縣[[石垣島|石垣島]]出身的[[龍谷大學|龍谷大學]]教授[[松島泰勝|松島泰勝]]和沖繩國際大學副教授[[友知政樹|友知政樹]]等人組織發起，探讨琉球独立的可能性，並只限[[琉球族|琉球族]]人加入，拒絕日本人（[[大和人|大和人]]）加入。“琉球民族独立综合研究学会”于5月15日下午在冲绳县政府举行记者会，然后在冲绳的大学里举行成立大会<ref name="琉球独立で平和な島に">{{Cite web |url = http://www.yaeyama-nippo.com/2013/05/13/%E7%90%89%E7%90%83%E7%8B%AC%E7%AB%8B%E3%81%A7%E5%B9%B3%E5%92%8C%E3%81%AA%E5%B3%B6%E3%81%AB-%E5%B0%96%E9%96%A3%E3%81%AF%E4%BA%89%E3%81%84%E6%A3%9A%E4%B8%8A%E3%81%92%E3%82%92-%E5%B8%82%E5%87%BA%E8%BA%AB%E3%81%AE%E6%9D%BE%E5%B3%B6%E6%95%99%E6%8E%88%E6%8F%90%E5%94%B1 |title= 琉球独立で平和な島に　尖閣は争い棚上げを　市出身の松島教授提唱 |date= 2013年5月15日 |publisher= 《八重山日報》 |language= ja |accessdate= 2013年5月15日 |quote=  }}</ref>。现有会员100多人。早在4月27日，该学会的筹备会就主办了“思考琉球主权国际研讨会”，当时大约300多人参加研讨会。该学会的目标是积极吸纳会员并逐步发展为地方政党，然后在恰当时候向[[联合国|联合国]]“[[脱离殖民地化特别委员会|脱离殖民地化特别委员会]]”申诉独立请求。

== 學會的主張 ==
該學會主張[[琉球民族|琉球民族]]是一個獨立的民族，[[琉球國|琉球國]]是獨立國家，1879年被日本[[琉球處分|強行併合]]。之後琉球成為日本及[[美國|美國]]的[[殖民地|殖民地]]，兩國政府皆歧視、剝削、支配琉球人，日本透過犧牲琉球來換取和平，但琉球依然時刻受到[[戰爭|戰爭]]威脅，琉球人有權自決前途，脫離日本，並撤去他國之軍事基地，與世界各國建立友好關係，實現和平的「甘世」<ref>{{Cite web |url= http://ryukyushimpo.jp/news/storyid-206634-storytopic-1.html |title = 琉球独立学会を設立　平和な『甘世』実現目指す |date = 2013年5月16日 |publisher= 《[[琉球新報|琉球新報]]》 |language= ja |accessdate = 2013年5月16日 |quote =  }}</ref>。他們認為[[釣魚臺列嶼|釣魚臺列嶼]]（日本稱[[尖閣諸島|尖閣諸島]]）是[[琉球歷史|琉球歷史]]的一部份，反對日本把[[釣魚台國有化|釣魚台國有化]]，認為任何一方宣示主權皆會引起戰爭，因此各方應分享島嶼和海權<ref name="琉球独立で平和な島に" />。

== 琉球的民意 ==
[[琉球族|琉球族]]的日本[[社民党|社民党]]国家对策委员长、[[众议院|众议院]]议员[[照屋宽德|照屋宽德]]，对该学会表示支持，他于学会成立当天，在他的博客上发表文章，标题是《终于要从大和走向独立了》，内容声称：“冲绳还是独立的好。明治以来的近现代史中，冲绳总是受到当时政权的歧视。即使现在，冲绳人也没有被看成日本国民。”<ref>{{Cite web |url= http://www.asyura2.com/13/senkyo146/msg/266.html |title= 沖縄、ついにヤマトから独立へ |date= 2013年4月1日 |publisher = 照屋寛徳 |language = ja |accessdate = 2013年4月1日 |quote =  }}</ref>

冲绳当地媒体《[[琉球新报|琉球新报]]》在5月15日发表社论，文章内容称：经历了日本政府强行配置[[MV-22鱼鹰運輸机|MV-22鱼鹰運輸机]]一事后，冲绳民意以及自主决定权希望被尊重的意愿越来越多；“琉球民族独立综合研究学会”的成立即代表了这种趋势。<ref>{{Cite web |url = http://ryukyushimpo.jp/news/storyid-206581-storytopic-12.html |title =  ＜金口木舌＞続く「祖国」との葛藤 |date = 2013年5月15日 |publisher = 《[[琉球新報|琉球新報]]》 |language = ja |accessdate = 2013年5月15日 |quote =  }}</ref><ref>{{Cite web |url = http://ryukyushimpo.jp/news/storyid-206580-storytopic-11.html |title = 本土復帰４１年　自己決定権の尊重を　揺るがぬ普天間閉鎖の民意 |date = 2013年5月15日 |publisher = 《[[琉球新報|琉球新報]]》 |language= ja |accessdate = 2013年5月15日 |quote=  }}</ref>
