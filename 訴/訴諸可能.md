
'''訴諸可能'''（Appeal to Possibility/Probability）或'''把合理當正確'''<ref>冀劍制，《邏輯謬誤鑑識班：訓練偵錯神經的24堂邏輯課》，第1課 「把合理當正確」的謬誤</ref>是一種[[非形式謬誤|非形式謬誤]]，是基於一件事有可能或相當可能是真的，就把它當作確定是真的。<ref>Bo Bennett, ''Logically Fallacious: The Ultimate Collection of Over 300 Logical Fallacies (Academic Edition)'', The fallacies > Appeal to Possibility[http://www.logicallyfallacious.com/index.php/logical-fallacies/41-appeal-to-possibility]: {{lang|en|“When a conclusion is assumed not because it is probably true, but because it is possible that it is true, no matter how improbable.”}}</ref>

真相（或事情未來的發展）是唯一的，合理的推測卻有很多，當思考或決策忽略了其他可能性，把一種合理的、可能的推測當做正確的、必然的真相，就不是恰當的推理。像例如說「[[死刑|死刑]]的[[嚇阻|嚇阻]]效果不明確，因此死刑可能無助治安」是合理的，但這不表示[[死刑|死刑]]一定無助治安。因為死刑嚇阻效果不明確，就一口咬定死刑一定無助治安，是不恰當的推理。

== 示例 ==
{{exampleH|例1}}
* 甲：你確定只申請第一大學嗎？你的成績並不算好，缺乏優異事蹟，而且大家都覺得你準備不夠充分……
* 乙：你認為我不可能上嗎？
* 甲：並非「不可能」，只是……
* 乙：既然「有可能」，那就沒問題了。
{{exampleF}}

決策不應只基於「有可能」而做，如許多客觀資訊都顯示可能性極低還堅持要做，且認為一定會成功，後果很可能不符預期。

{{exampleH|例2}}
網路上很多[[駭客|駭客]]，你上網不裝[[防火牆|防火牆]]，你的電腦一定會被駭客入侵的！
{{exampleF}}

上網不裝防火牆會增加電腦被入侵的風險，但並非一定會被駭客入侵。

{{exampleH|例3}}
小華在偵訊時結結巴巴、言詞反覆，還數度說謊，這就是他涉入了這起大規模詐欺案的明確證據！辯護律師找一堆理由幫這種惡徒脫罪，真是毫無良心！
{{exampleF}}

要判定小華涉案，通常需有充分的法律證據，小華在偵訊時的表現可作為懷疑小華涉案的線索，但不應據此咬定他必有涉案，及認定試圖為嫌疑犯辯護的律師都是昧著良心做事。

{{exampleH|例4（布佛氏論證）}}
魔術師經常能表演出讓物體浮起來的魔術，所以小華讓物體浮起來一定是變魔術，而不是因為他有[[超能力|超能力]]。
{{exampleF}}

除了超能力以外，魔術也能讓物體漂浮，小華有可能是變魔術，但不宜據此認定小華必然是變魔術。（參見：[[布佛氏論證|布佛氏論證]]）

{{exampleH|例5（訴諸陰謀）}}
九一一事件其實是美國政府自導自演的一齣戲。想想看，飛機有可能撞毀摩天大廈嗎？為什麼五角大樓的缺口直徑與飛機直徑不符？襲擊後，美國政府怎麼能立即公佈劫機者的名單？事件不久後就以消滅塔利班政權及捕捉賓·拉登為名向阿富汗宣戰？然後又以搜尋大殺傷力武器為名向伊拉克宣戰？很明顯的，九一一事件絕對是美國政府自導自演，目的就是製造藉口發動戰爭，以控制全世界！<ref>參見[[九一一事件陰謀論|九一一事件陰謀論]]</ref>
{{exampleF}}

以上資訊和推論可懷疑九一一事件是美國政府自導自演，但在有更明確的證據前，不宜輕率認定它就是事實。（參見：[[鬼祟謬誤|訴諸陰謀]]）

{{exampleH|例6}}
活摘器官的報導大多數都不可信，[[喬高-麥塔斯調查報告|喬高-麥塔斯調查報告]]也沒有直接證據，所以活摘器官的指控，只是[[法輪功|法輪功]]人士捏造的謠言罷了。
{{exampleF}}

法輪功學員捏造活摘器官事件只是一種可能性，但假報導不能代表活摘法輪功學員器官的事情就一定沒有發生。

{{exampleH|例7}}
他有可能有罪，但抓不到其他的嫌疑犯，因此不能將這人給無罪開釋，一定要給他定罪。
{{exampleF}}

有罪只是一種可能性，但這不代表這人就一定要被定罪。

{{exampleH|例8}}
雖然我和他沒見過面，但他用了女生頭像，整體給人的感覺也像女的，因此一定是個女孩子。
{{exampleF}}

用女生頭像，行為也像女生的網民，不一定在真實世界是女的。

{{exampleH|例9}}
髮禁可能會害得學生花更多時間在外表上，而且可能扼殺學生的創造力和審美觀，所以不該有髮禁。
{{exampleF}}

這些都只是可能性，而未必是百分之百成立的。

{{exampleH|例10}}
[[死刑|死刑]]可能無助治安，因此該[[廢除死刑|廢除死刑]]。
{{exampleF}}

[[死刑|死刑]]可能無助治安，表示死刑[[嚇阻|嚇阻]]效果不明確，但這也可能表示其實死刑有助治安，只是效果不足以為既有的研究所探知而已。

== 相關謬誤 ==
* [[布佛氏論證|布佛氏論證]]：假定某觀點是錯的，由此出發解釋為什麼許多人會相信它，然後斷定該觀點是錯誤的。
* [[鬼祟謬誤|訴諸陰謀]]：在未有充分證據下，主張事情的真相一定是某些人在背後進行陰謀。
