'''羌塘自然保护区'''位于[[青藏高原|青藏高原]]北部，于1993年经[[西藏自治区人民政府|西藏自治区人民政府]]批准成立，2000年4月4日经[[中华人民共和国国务院|中华人民共和国国务院]]批准晋升为[[国家级自然保护区|国家级自然保护区]]'''西藏羌塘国家级自然保护区'''。保护区面积33.4万[[平方公里|平方公里]]（12.9万[[平方英里|平方英里]]）<ref name="dezi9or">[http://dezi9or.com/c4n/wildindia/chiru/habitat.htm "Habitat of Chiru"]{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref name="icimod.org">{{Cite web |url=http://www.icimod.org/hkhconservationportal/PA.aspx?ID=54 |title="Changthang National Protected Area" |access-date=2016-08-30 |archive-url=https://web.archive.org/web/20160304023241/http://www.icimod.org/hkhconservationportal/PA.aspx?ID=54 |archive-date=2016-03-04 |dead-url=no }}</ref>，仅次于[[东北格陵兰国家公园|东北格陵兰国家公园]]和[[卡万戈赞比西跨境保护区|卡万戈赞比西跨境保护区]]，是世界上第三大陆地[[自然保护区|自然保护区]]，它的面积超过了[[世界各国和地区面积列表|183个国家]]。行政区划上，它属于[[那曲市|那曲市]]的[[安多县|安多县]]、[[双湖县|双湖县]]、[[尼玛县|尼玛县]]和[[阿里地区|阿里地区]]的[[改则县|改则县]]、[[日土县|日土县]]；其中双湖县县治位于保护区内。毗邻保护区的储备地区面积共有496,000平方千米（191,507平方英里），约等于[[西班牙|西班牙]]的面积，超过了[[世界各国和地区面积列表|197个国家]]。

== 地理 ==
羌塘（[[標準藏語|藏语]]意为“北部平原”）是一个延伸自印度[[拉達克|拉達克]]的巨大的高海拔高原，位于西藏自治区北部，[[昆仑山|昆仑山]]、[[可可西里山|可可西里山]]以南，[[冈底斯山|冈底斯山]]和[[念青唐古拉山|念青唐古拉山]]以北。羌塘自然保护区包含了中国高原的大部分地区。

平均海拔超过4,800米(16,000英尺)“低山、丘陵与湖泊构成波伏起伏的地面分布在20,000英尺（6000米）的山上。”冬季夜间极度寒冷气温可达零下40度。它是极其寒冷的冬天气温降至零下40度在夜间。 旅行在冬季更容易，因为夏天非常潮湿，车辆容易陷入泥沼。<ref>[http://www.npr.org/templates/story/story.php?storyId=7316356 "A Resurgence of Wildlife in Northern Tibet"] {{Wayback|url=http://www.npr.org/templates/story/story.php?storyId=7316356 |date=20160402212015 }} by Alex Chadwick.</ref>

== 历史 ==
羌塘自然保护区成立于1993年，由国际著名动物学家[[乔治·夏勒|乔治·夏勒]]协助，位于西藏北部人迹罕至的羌塘高原。2000年，羌塘自然保护区晋升为[[国家级自然保护区|国家级自然保护区]]。它被[[國際自然保護聯盟|國際自然保護聯盟]]分类为：VI——资源管理保护区。<ref name="icimod.org">{{Cite web |url=http://www.icimod.org/hkhconservationportal/PA.aspx?ID=54 |title="Changthang National Protected Area" |access-date=2016-08-30 |archive-url=https://web.archive.org/web/20160304023241/http://www.icimod.org/hkhconservationportal/PA.aspx?ID=54 |archive-date=2016-03-04 |dead-url=no }}</ref>

== 人口 ==
羌塘自然保护区内只有少量的藏族[[遊牧民族|游牧]]家庭和防止偷猎的保护区工作人员。

== 植物群和动物群 ==
[[File:Atilope_du_Tibet.png|缩略图]]
[[File:Equus_kiang_-_Changtang.jpg|缩略图]]

羌塘分布着一些野生[[有蹄類|有蹄类动物]]种群：野[[牦牛|牦牛]](''Bos grunniens'')，[[西藏野驴|西藏野驴]](''Equus kiang''), [[岩羊|喜马拉雅蓝羊]](''Pseudois nayaur''), [[盘羊|盘羊]] (''Ovis ammon'')，[[黄羊|蒙古原羚]] (''Procapra gutturosa'')和[[藏羚羊|藏羚羊]](''Pantholops hodgsonii'')。走私藏羚羊底绒制成的披肩称为[[沙图什|沙图什]]，是世界公认的最精美最柔软的披肩，沙图什以印度[[克什米尔|克什米尔]]的最为出名。[[捕食|掠食者]]包括[[雪豹|雪豹]](''Panthera uncia or Uncia uncia'')，[[藏狼|藏狼]] (''Canis lupus chanco'')，土耳其斯坦山猫 (''Lynx lynx isabellinus'')和[[西藏棕熊|西藏棕熊]](''Ursus arctos pruinosus'')。位于食物链的底端有大量的[[鼠兔|鼠兔]](''Ochotona spp.'')<ref>Dorje (2009), p. 390.</ref>

植被为灌丛和草原，主要是[[针茅属|针茅属]]和[[嵩草属|嵩草属]]。<ref name="icimod.org">{{Cite web |url=http://www.icimod.org/hkhconservationportal/PA.aspx?ID=54 |title="Changthang National Protected Area" |access-date=2016-08-30 |archive-url=https://web.archive.org/web/20160304023241/http://www.icimod.org/hkhconservationportal/PA.aspx?ID=54 |archive-date=2016-03-04 |dead-url=no }}</ref>
