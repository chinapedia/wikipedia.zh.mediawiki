「跆拳道」一詞，是1955年由[[崔泓熙|崔泓熙]]创造。 在南韓大邱市成立，其中「跆」指踢擊、「拳」指拳擊，「[[道|道]]」則是代表道行、自己對禮儀的修練<ref>{{english}} {{cite web |url= http://www.worldtaekwondo.com/history.htm|title=  THE HISTORY OF TAEKWONDO|accessdate=2008-12-23 |work= A Report for Recommendation Black Belt Testing 1994|publisher=Glen R. Morris |date= }}</ref>。1966年，[[國際跆拳道聯合會|國際跆拳道聯合會]]（'''ITF'''）在[[南韓|南韓]]成立，並由韓國人[[崔泓熙|崔泓熙]]擔任主席<ref>{{zh}}{{cite web|url=http://www.taekwondo-web.com/cn/tkd/tkdhist.htm|title=跆拳道歷史|accessdate=2008-12-25|work=|publisher=China I.T.F.|date=|deadurl=yes|archiveurl=https://web.archive.org/web/20070722181801/http://www.taekwondo-web.com/cn/tkd/tkdhist.htm|archivedate=2007-07-22}}</ref>。1972年，[[大韓跆拳道協會|大韓跆拳道協會]]的中央道場[[韓國國技院|韓國國技院]]開始動工。1973年5月，[[世界跆拳道聯盟|世界跆拳道聯盟]]（'''WTF'''）在南韓[[首爾|首爾]]成立，[[金雲龍|金雲龍]]當選為主席。1975年，世界跆拳道聯盟成為了[[國際體育聯合會|國際體育聯合會]]的成員，令跆拳道運動得到更多人的認識。五年後，[[國際奧委會|國際奧委會]]（IOC）正式承認了世界跆拳道聯盟。2000年，-{zh-cn:悉尼; zh-tw:雪梨; zh-hk:悉尼}-奧運會將跆拳道列入正式競賽項目；在成為奧運會項目前，跆拳道曾成為[[世界運動會|世界運動會]]的比賽項目。隨著時代不斷地變化，跆拳道將會不斷地發展、延伸下去。

== 三國及高麗時代的萌發 ==
跆拳道一般均聲稱起源於三國及高麗時代，但按古藉等記載的內容，1955年前均沒有關於「跆拳道」的記載。因此也有一說指跆拳道是由創辦人崔泓熙參考朝鮮的古武術而自創的武術<ref>{{zh}}{{cite web|url=http://www.taekwondo-web.com/cn/tkd/tkdsound/tkd_history.htm|title=歪曲的跆拳道歷史|accessdate=2008-12-25|work=|publisher=China I.T.F.|date=|deadurl=yes|archiveurl=https://web.archive.org/web/20080611051541/http://taekwondo-web.com/cn/tkd/tkdsound/tkd_history.htm|archivedate=2008-06-11}}</ref>。其中一個跟跆拳道聲稱的起源吻合，而且同樣使用腳部的武術（但實際上兩者有不少不同之處），是現稱[[跆跟|跆跟]]的「腳戲」。

朝鮮的三國時代，[[高句麗|高句麗]]、[[新羅|新羅]]及[[百濟|百濟]]三個國家為了爭奪領土，引發了戰爭。朝鮮人在這段期間創立了「腳戲」，並迅速發展<ref name="web">{{english}}{{cite web|url= http://home.att.net/~taekwondo.junkie/TKDHistory.html|title= Tae Kwon Do：Yesterday and Today|accessdate= 2008-12-25|work= |publisher= Taekwondo Junkie|date= |deadurl= yes|archiveurl= https://web.archive.org/web/20081219203021/http://home.att.net/~taekwondo.junkie/TKDHistory.html|archivedate= 2008-12-19}}</ref>。在[[新羅|新羅]]時代，一個青年武團以「事君以忠，事親以孝，交友以信，臨戰無退，殺身有擇」的宗旨，組織了一批青年人一起切磋[[花郎道|花郎道]]，為統治三國為一體而打定了根基。據《[[三國史記|三國史記]]》記載，百濟歷代國王都大力推廣武技，要求全國人民也要去學習射箭、騎馬及「腳戲」等武技。而於918年統一朝鮮半島的國家——高麗則常常要軍隊進行「[[手搏|手搏]]」、「腳戲」，並規定為軍隊的必修項目之一。據《高麗史》描述，手搏和腳戲是高麗人最喜愛的運動。[[考古學家|考古學家]]們則在[[高麗|高麗]]的皇家墓穴中發現一些類似跆拳道的壁畫；在新羅時代的石窟中則有些守門佛像和金剛力士像表現出跆拳道的姿勢；新羅史冊裡更有關於跆拳道的技巧及方法等的記載<ref>{{zh}}{{cite web |url=http://www.cnkang.com/dzjk/bjzs/jsfs/tqd/200705/91089.html |title=跆拳道歷史 |accessdate=2008-12-25 |work= 大眾健康|publisher= 中華康網|date= 2007-05-15}}</ref>；在百濟王朝的史書裡也有紀錄狀似跆拳道武術對打的教學<ref>{{zh}}{{cite web |url= http://www.dtes.chc.edu.tw/special/sportsp/new_page_5.htm|title= 跆拳道歷史|accessdate=2008-12-25 |work= |publisher= |date= }}</ref>。而在古藉《[[同濟大學|同濟大學]]》亦曾說道：「Taekwondo is the basis of martial art, enabling one to build strength by using the hand and foot freely and training arms and legs as well as the body to adaptable to any critical situations,」（跆拳道是武術的基礎，從其訓練手和腳在任何危急情況下都能適應及自由地運用）<ref>{{english}}{{cite web |url= http://www.wtf.org/site/about_taekwondo/history/ancient.htm |title=Ancient Times|accessdate=2008-12-25 |work= WTF |publisher= 世界跆拳道聯盟|date= }} </ref>。

== 李氏朝鮮的傳承 ==
1392年，王氏高麗被[[李成桂|李成桂]]所建立的[[李氏朝鮮|李氏朝鮮]]取代。由於新王朝[[重文輕武|重文輕武]]，跆拳道在官方的眼中並無重要的地位，但在民間卻不受影響，依然旺盛。同時，朝鮮王朝的軍隊依然以「手搏」及「跆跟」作為選擇士兵的要點；如果想考獲[[武官|武官]]之地位，則需在競技中使用「手搏」及「跆跟」打倒三人或以上方為合格。1790年，由[[李德懋|李德懋]]將軍、[[朴齊家|朴齊家]]和[[白東修|白東修]]三人完成《[[武藝圖譜通志|武藝圖譜通志]]》，此書詳述併圖解韓國武術的動作及多種武器的使用。

根據李朝末期美術學書籍《[[海東竹枝|海東竹枝]]》所記載，關於[[腳術|腳術]]（Taekgyeon）的記錄如下：「古代風俗中之腳術即意爲兩人對立伸腿將對方擊倒。列三等，最次者踢到對方腿部，稍強者踢到對方肩部，最優者可踢至對方頭頂。古人憑此藝洗清冤仇，或決鬥贏得美人。」<ref name="kwondo"/>。由此可見，跆拳道的雛形在很久以前已經開始發展，其規則亦跟現代之跆拳道規則大同小異。

== 殖民地時期的衝撞 ==
1910年，[[日本|日本]]佔領了朝鮮36年並在當地成立了殖民政府，一度禁止所有含有[[朝鮮文化|朝鮮文化]]色彩的活動；在這段期間，就連韓文也不能使用<ref name="web"/>；當然，跆拳道也被包括在內，這令它幾乎一度滅絕。跆拳道為了適存，融合了日本和[[中國武術|中國武術]]的元素，增添入了不少文化色彩，得到了充實的發展，並慢慢地形成了一個新的體系。[[第二次世界大戰|第二次世界大戰]]後，朝鮮獨立，社會面貌亦改觀，自衛術復興，許多在異國的人民也相續回到朝鮮，同時也將各國的武術文化融合在跆拳道裡，與之形成一體，慢慢成為了現代跆拳道的雛型<ref>{{zh}}{{cite web |url=http://games.52pk.com/2008/ziliao/20080526/308399.html |title=奧運會跆拳道歷史概況 |accessdate=2008-12-25 |work= 奧運專題|publisher= S2PK 遊戲網|date= }}</ref>。

== 現代跆拳道的誕生 ==

國際跆拳道聯盟創始人[[崔泓熙|崔泓熙]]先生在幼年時已曾跟隨書法名師學習，並掌握到古典武道如何运用。青年時則因為完成學業至日本學習，他不止完成学業，還學習了[[琉球|琉球]]傳統武術空手道，更考得了二段。1945年8月15日隨著朝鮮的解放，崔將軍在漢城與其他人創建了韓國軍隊。同時他開始了新的武道研究，即是今日的跆拳道<ref name="Park">{{english}}Park, S. W. (1993): About the author. In H. H. Choi: ''Taekwon-Do: The Korean art of self-defence'', 3rd ed. (Vol. 1, pp. 241–274). Mississauga: International Taekwon-Do Federation.</ref> 。

1955年4月11日，「跆拳道」經過韓國各界著名人士組成的名稱制定委員會，包括崔泓熙少將（韓國第三軍管區司令），[[李亨根|李亨根]]陸軍上將（陸軍參謀總長），國會副議長，著名武館館長<ref>{{cite web |url= http://ochihhaw.myweb.hinet.net/htm/1.htm|title= 簡介|accessdate=2009-01-16 |work= 跆拳道史|publisher= 永和跆拳道歐志豪韓國國技院捌段師範|date= }}</ref>等提出了各自所構思的名稱。之後透過不記名投票，跆拳道一詞正式誕生，其中「跆」指踢擊、「拳」指拳擊，「[[道|道]]」則是代表道行、自己對禮儀的修練<ref>{{english}} {{cite web |url= http://www.worldtaekwondo.com/history.htm|title=  THE HISTORY OF TAEKWONDO|accessdate=2008-12-23 |work= A Report for Recommendation Black Belt Testing 1994|publisher=Glen R. Morris |date= }}</ref>。

在四年後，跆拳道更被帶到其他國家，崔泓熙更率領了軍隊中最優秀的十九名成員，訪問了[[越南|越南]]和[[台灣|台灣]]，這令跆拳道得以在各國有所發展。[[韓國出兵越南|韓國出兵越南]]期間，曾任{{Link-ko|大韓跆拳道協會|대한태권도협회|大韓跆手道協會}}創會首屆會長的[[駐越韓國軍|駐越韓國軍]]司令官[[蔡命新|蔡命新]][[中將|中將]]，亦表示跆拳道的推廣可作為戰術的延伸，並於1965年創立越南跆拳道協會（{{lang|ko|월남태권도협회}}）。軍需參謀禹忠林（{{lang|ko|우종림}}）[[中校|中校]]在[[大韩民国首都机械化步兵师|猛虎師]]駐屯地向[[南越政府軍|南越政府軍]]、警察、學校和平民教授跆拳道<ref>{{cite web |url=http://www.dlibrary.go.kr/JavaClient/jsp/wonmun/full2.jsp?v_kw_str=%28%28%E9%A7%90%E8%B6%8A%20%E9%9F%93%E5%9C%8B%E8%BB%8D%29%E8%B7%86%E6%8B%B3%E9%81%93%E6%95%8E%E5%AE%98%E5%9C%98%E9%80%9A%E5%8F%B2%29&v_db=4&v_doc_no=17617&mode=1#cover_image |title=(駐越 韓國軍)跆拳道敎官團通史}}</ref>。

== 近代狀況 ==
1966年，國際跆拳道聯盟（'''I'''nternational '''T'''aekwondo '''F'''ederation）在韓成立，並由[[崔泓熙|崔泓熙]]擔任主席<ref>{{zh}}{{cite web|url=http://www.taekwondo-web.com/cn/tkd/tkdhist.htm|title=跆拳道歷史|accessdate=2008-12-25|work=|publisher=China I.T.F.|date=|deadurl=yes|archiveurl=https://web.archive.org/web/20070722181801/http://www.taekwondo-web.com/cn/tkd/tkdhist.htm|archivedate=2007-07-22}}</ref>。1972年，大韓跆拳道協會的中央道場[[韓國國技院|韓國國技院]]開始動工。1973年5月，世界跆拳道聯盟在[[首爾|漢城]]（即今首爾）成立，{{Link-ko|金雲龍|김운용}}當選為主席。1975年，世界跆拳道聯盟成為了[[國際體育聯合會|國際體育聯合會]]的成員，令跆拳道運動得多更多人的認識。5年後，[[國際奧委會|國際奧委會]]（IOC）正式承認了世界跆拳道聯盟。跆拳道於1988年[[1988年夏季奧運會|漢城奧運會]]成為示範項目，並於1992年的[[巴塞隆拿奧運會|巴塞隆拿奧運會]]開始進行公開競賽，但其成績並不計入獎牌榜。直到2000年，-{zh-cn:悉尼; zh-tw:雪梨; zh-hk:悉尼}-奧運會將跆拳道列入正式競賽項目，令跆拳道再往國際化的舞台踏上多一步；。隨著時代不斷地變化，跆拳道將會不斷地發展、延伸下去。亦因為跆拳道的訓練不受時間、空間、經費之限制，可隨時隨地展開訓練，韓國人自小即接受訓練，在學校、社會及軍中更是盛行<ref>{{zh}} 叢津滋（1978）：紀效新書的傳統與近代跆拳道發展。中華跆拳道季刊，中華跆拳道協會印行，36-40頁。</ref>。跆拳道的興起、蓬勃發展的原因不只是因為崔泓熙在各國積極宣傳跆拳道，更重要的是因為它集合了各國[[武術|武術]]的精髓、民族精神及道德修養，使跆拳道得以在各國有良好的發展<ref>{{zh}} 高誼、陳立人(1998)：跟專家練跆拳道。北京：北京體育大學出版社。</ref><ref>{{zh}} 劉宏偉、關鐵雲(2001)：跆拳道。台北市：國家出版社。</ref><ref name="kwondo"/>。

=== 中國大陸的發展 ===
[[中國|中國]]是一個文明古國，有很多不同的武術門派。而崔泓熙（現代跆拳道創始人）早就認同中國是宣揚跆拳道的好地方。故此，他在1986年6月隨著國際跆拳道代表團訪問了中國，並跟當時的中國奧委會主席[[何振梁|何振梁]]交談；此外團中亦有派人到中國各地（包括[[北京|北京]]、[[西安|西安]]和[[濟南|濟南]]等）進行跆拳道表演，這些活動都令跆拳道得到一個重大的幫助。隨此之後，崔泓熙將軍及國際跆拳道聯盟的高級師範們多次親自到訪[[中國|中國]]，向中國學生教授跆拳道的知識。雖然這些方案都取得了一定的成果，但由於中國政府有關方面[[領導人|領導人]]的[[經濟|經濟]]務實決策等各種客觀原因，始終沒有達到預期效果<ref>{{zh}} {{cite web |url= http://taekwondo.sport.org.cn/taekwondo/knowledge/2004-05-08/46718.html|title= 跆拳道在中國的引進和發展|accessdate=2008-12-25 |work= |publisher= 中國跆拳道協會|date= 2004-05-08 }}</ref>。

=== 台灣的發展 ===
1966年，[[中華民國國防部長|中華民國國防部長]][[蔣經國|蔣經國]]組團到了[[大韓民國|大韓民國]]參訪軍隊的戰技訓練，而當時的[[大韓民國國軍|大韓民國國軍]]把跆拳道列為必修的課程。蔣經國看到後認為這個觀念很不錯，便於一年後成立「[[中華民國海軍陸戰隊|陸戰隊]]莒拳班」，聘請了一班韓籍教官來訓練，並以[[莒光|莒光]]之名，將其定名為「[[莒拳|莒拳]]」，列入三軍戰技必修訓練課程。其「莒拳」之名定義為「爭取[[民主|民主]]、[[自由|自由]]，促進世界大同的義務與責任」。自此跆拳道開始在台灣發展<ref>{{zh}}{{cite web |url= http://www.dtes.chc.edu.tw/special/sportsp/new_page_6.htm|title= 跆拳道在台灣 |accessdate=2008-12-25 |work= |publisher= |date= }}</ref>。

1973年，[[國軍運動大會|國軍運動大會]]將跆拳道首次列為比賽項目。並於同年成立[[中華民國跆拳道協會|中華民國跆拳道協會]]，將跆拳道這項有益身心的運動推廣到社會的每一個角落，有助跆拳道於台灣的發展。後來，台灣的跆拳道發展得很迅速，在民間有不少跆拳道的訓練班，顯示其普及性；台灣於歷屆奧運會共獲兩金一銀兩銅，在金牌榜排名第三；這些事亦為台灣的跆拳道刻立了一個重要的里程碑<ref>{{zh}}{{cite web |url= http://www.ettoday.com/2004/08/27/1119-1678060.htm|title= 雅典奧運／26日頒出15金　台灣跆拳道雙金風光十足|accessdate=2008-12-25 |work= |publisher= 今日新聞 Now News|date= 2004-08-27 }}</ref>。目前，台灣各地到處也屹立著跆拳道的[[道館|道館]]，各級學校也成立了跆拳道的社團，使其成為青年男女最熱愛的武術運動之一<ref>{{zh}} 汪茂鈞（1995）：跆拳道。台北市：政治作戰學校，171-191頁。</ref><ref name="kwondo">[http://www.lhu.edu.tw/m/oaa/publish/24/9.%E6%B1%AA%E8%8C%82%E9%88%9E-%E8%B7%86%E6%8B%B3%E9%81%93%E9%81%8B%E5%8B%95%E7%9A%84%E8%B5%B7%E6%BA%90%E8%88%87%E7%99%BC%E5%B1%95%E4%B9%8B%E6%8E%A2%E8%A8%8E.pdf 跆拳道運動的起源與發展之探討－汪茂鈞（龍華科技大學體育室）]{{dead link|date=十月 2017 |bot=InternetArchiveBot }}</ref>。
