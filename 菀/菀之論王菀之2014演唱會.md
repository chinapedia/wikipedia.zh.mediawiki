《'''菀之論王菀之2014演唱會'''》是香港[[創作歌手|唱作歌手]][[王菀之|王菀之]]的第三張[[現場專輯|現場錄音專輯]]，由[[東亞唱片_(集團)|東亞唱片]]於2014年8月29日發行。

==專輯簡介==
本專輯收錄她於2014年5月17日在[[香港體育館|紅磡香港體育館]]演唱的歌曲。<ref name = CD>{{cite web |title = 菀之論 王菀之2014演唱會 (2CD) |publisher=Yesasia |url=https://www.yesasia.com/global/%E8%8F%80%E4%B9%8B%E8%AB%96-%E7%8E%8B%E8%8F%80%E4%B9%8B2014%E6%BC%94%E5%94%B1%E6%9C%83-2cd/1036619004-0-0-0-zh_TW/info.html |accessdate = 2018年3月21日}}</ref>本張專輯特別收錄三首錄音室版本歌曲：「星空」、「好時辰」及演唱會主題曲「天堂有路」。然後，於同年9月5日和11月5日分別發行[[卡拉OK|卡拉OK]] [[DVD|DVD]]及[[藍光光碟|藍光光碟]]版本。<ref name = DVD>{{cite web |title = 菀之論 王菀之2014演唱會 Karaoke (3DVD) |publisher=Yesasia |url=https://www.yesasia.com/global/%E8%8F%80%E4%B9%8B%E8%AB%96-%E7%8E%8B%E8%8F%80%E4%B9%8B2014%E6%BC%94%E5%94%B1%E6%9C%83-karaoke-3dvd/1036673539-0-0-0-zh_TW/info.html |accessdate = 2018年3月21日}}</ref><ref name = bluray>{{cite web |title = 菀之論 王菀之2014演唱會 Karaoke (Blu-ray) |publisher=Yesasia |url=https://www.yesasia.com/global/%E8%8F%80%E4%B9%8B%E8%AB%96-%E7%8E%8B%E8%8F%80%E4%B9%8B2014%E6%BC%94%E5%94%B1%E6%9C%83-karaoke-blu-ray/1037458140-0-0-0-zh_TW/info.html |accessdate = 2018年3月21日}}</ref>
