'''剑桥大学学院列表'''列举了[[剑桥大学|剑桥大学]]目前所有的成員学院（{{lang|en|Colleges}}，又译“[[學院_(學校)|書院]]”）。這些成員学院是[[剑桥大学|剑桥大学]][[本科生|本科生]]和[[研究生|研究生]]住宿的地方，他們同时也负责安排自己的本科生录取<ref name=role>{{Cite web|url=http://www.cam.ac.uk/admissions/undergraduate/colleges/role.html |archiveurl=https://web.archive.org/web/20071023055509/http://www.cam.ac.uk/admissions/undergraduate/colleges/role.html |archivedate=2007年10月23日 |title=学院的角色 |accessdate=2008年3月27日 |publisher=[[剑桥大学|剑桥大学]] |deadurl=yes }}</ref>，亦会为大学资深的研究人员提供资金和住宿方面的帮助<ref name="thecolleges" />。 惟需注意：成員學院與大學的學術學院（{{lang|en|Schools}}）性質不同，後者提供教學與研究指引及編制教材。<ref name=role/>

[[剑桥大学|剑桥大学]]目前共拥有31个学院<ref name="thecolleges">{{Cite web |url = http://www.cam.ac.uk/colleges/ |title = 大学学院 |publisher = [[剑桥大学|剑桥大学]] |year = 2009 |accessdate = 2009年4月6日 |archive-url = https://web.archive.org/web/20090404070207/http://www.cam.ac.uk/colleges/ |archive-date = 2009-04-04 |dead-url = no }}</ref>，其中[[剑桥大学彼得学院|彼得学院]]是最古老的学院，始建于1284年<ref>{{Cite web |url = http://news.bbc.co.uk/1/hi/uk/40871.stm |title = 剑桥大学学院 |date = 1997年12月19日 |accessdate = 2008年3月26日 |publisher = [[BBC|BBC]]新闻 |archive-url = https://web.archive.org/web/20090223064405/http://news.bbc.co.uk/1/hi/uk/40871.stm |archive-date = 2009-02-23 |dead-url = no }}</ref>。成立最晚的学院是[[剑桥大学罗宾森学院|罗宾森学院]]，建于1977年。<ref>{{Cite web |url = http://www.robinson.cam.ac.uk/about/ |title = 关于罗宾森学院 |accessdate = 2008年3月26日 |publisher = [[剑桥大学罗宾森学院|剑桥大学罗宾森学院]] |archive-url = https://web.archive.org/web/20080409124402/http://www.robinson.cam.ac.uk/about/ |archive-date = 2008-04-09 |dead-url = no }}</ref>

剑桥大学有三所学院只招收女性，它们分别是[[剑桥大学露西·卡文迪许学院|露西·卡文迪许学院]]、[[剑桥大学纽纳姆学院|纽纳姆学院]]和[[剑桥大学默里·爱德华兹学院|默里·爱德华兹学院]]，其它学院则招收男女学生。[[剑桥大学达尔文学院|达尔文学院]]是第一个招收男女学生的学院，而[[剑桥大学丘吉尔学院|丘吉尔学院]]、[[剑桥大学克莱尔学院|克莱尔学院]]和[[剑桥大学国王学院|国王学院]]于1972年开始招收女性[[本科生|本科生]]入学。[[剑桥大学莫德林学院|莫德林學院]]则是最后一所只招收男性的学院，并于1988年开始招收女性学生。<ref>{{Cite web |url = http://www.guardian.co.uk/obituaries/story/0,3604,976477,00.html |title = 伯纳德·威廉斯爵士讣告 |publisher = [[卫报|卫报]] |accessdate = 2009年5月8日 |date = 2003年6月13日 |archive-url = https://web.archive.org/web/20080725232703/http://www.guardian.co.uk/obituaries/story/0,3604,976477,00.html |archive-date = 2008-07-25 |dead-url = no }}</ref>剑桥大学有两所学院只招收[[研究生|研究生]]，它们是[[剑桥大学克莱尔学堂|克莱尔学堂]]和[[剑桥大学达尔文学院|达尔文学院]]；有四所学院只招收21岁以上的成年学生和研究生，它们是[[剑桥大学休斯大厅学堂|休斯学堂]]、[[剑桥大学露西·卡文迪许学院|露西·卡文迪许学院]]、[[剑桥大学圣埃德蒙学院|圣埃德蒙学院]]和[[剑桥大学沃尔森学院|沃尔森学院]]<ref>{{Cite web

== 过往的学院 ==
以上的列表并不包括那些现已不存的学院，这些学院包括：
* '''白金汉学院'''始建于1428年，初期是一間[[本篤會|本篤會]]學堂，后于1542年重建为现在的[[剑桥大学抹大拉学院|抹大拉学院]]。<ref>{{Cite web| url =http://www.magd.cam.ac.uk/about/history/early-days.html| title =学院早期| publisher =[[剑桥大学抹大拉学院|剑桥大学抹大拉学院]]| accessdate =2008年3月27日| deadurl =yes| archiveurl =https://web.archive.org/web/20090326124845/http://www.magd.cam.ac.uk/about/history/early-days.html| archivedate =2009年3月26日}}</ref>
* '''卡莱尔学堂'''是[[剑桥大学卡莱尔学院|卡莱尔学院]]于1338至1856年所采用的名字，卡莱尔学院于1966年建立了一所新的学院，又叫作[[剑桥大学卡莱尔学堂|卡莱尔学堂]]。
* '''国王学堂'''始建于1317年，后于1546年和迈克尔学院合并为现在的[[剑桥大学三一学院|三一学院]]。<ref name=trin>{{Cite web | url = http://www.trin.cam.ac.uk/index.php?pageid=20 | title = 历史回顾 | publisher = [[剑桥大学三一学院|剑桥大学三一学院]] | accessdate = 2008年3月27日 | archive-url = https://web.archive.org/web/20080329063213/http://www.trin.cam.ac.uk/index.php?pageid=20 | archive-date = 2008-03-29 | dead-url = no }}</ref>
* '''迈克尔学院'''始建于1324年，后于1546年和国王学堂合并为现在的[[剑桥大学三一学院|三一学院]]。.<ref name=trin/>
* '''冈维尔学堂'''始建于1348年，于1557年重建为现在的[[剑桥大学冈维尔与凯斯学院|冈维尔与凯斯学院]]。<ref>{{Cite web|url=http://www.cai.cam.ac.uk/college/past/index.php |title=历史 - 介绍 - 开始 |accessdate=2008-03-27 |deadurl=yes |archiveurl=https://web.archive.org/web/20080315090932/http://www.cai.cam.ac.uk/college/past/index.php |archivedate=2008-03-15 }}</ref>
* '''上帝学院'''始建于1437年，于1505年重建为现在的[[剑桥大学基督学院|基督学院]]。<ref>{{Cite web | url = http://www.christs.cam.ac.uk/college-life/history/ | title = 学院历史 | accessdate = 2009年6月9日 | publisher = [[剑桥大学基督学院|剑桥大学基督学院]] | archive-url = https://web.archive.org/web/20090531042909/http://www.christs.cam.ac.uk/college-life/history/ | archive-date = 2009-05-31 | dead-url = no }}</ref>
* '''卡文迪许学院'''建于1873年，学院建筑从[[剑桥大学哈默顿学院|哈默顿学院]]购入，因错误的尝试让贫穷的学生入学而因资金问题关闭。<ref>{{Cite web|url=http://www.queens.cam.ac.uk/Queens/Misc/Coll-Dates-Cam.html |title=剑桥学院建立时间 |publisher=[[剑桥大学王后学院|剑桥大学王后学院]] |accessdate=2008年3月27日 |deadurl=yes |archiveurl=https://web.archive.org/web/20080220151914/http://www.queens.cam.ac.uk/Queens/Misc/Coll-Dates-Cam.html |archivedate=2008年2月20日 }}</ref><ref>{{Cite web|url=http://www.homerton.cam.ac.uk/teaching/library/archive.html |title=哈默顿学院档案 |accessdate=2008年3月27日 |publisher=[[剑桥大学哈默顿学院|剑桥大学哈默顿学院]] |deadurl=yes |archiveurl=https://web.archive.org/web/20070807112143/http://www.homerton.cam.ac.uk/teaching/library/archive.html |archivedate=2007年8月7日 }}</ref>
* '''大学学院'''始建于1965年，于1972年重建为现在的[[剑桥大学沃尔森学院|沃尔森学院]]。
* '''新学堂'''始建于1954年，于2008年更名为[[剑桥大学默里·爱德华兹学院|默里·爱德华兹学院]]。
