'''韓國地域對立'''，是指[[統一新羅|統一新羅]]至[[朝鲜王朝|朝鲜王朝]]，朝鮮半島長達一千三百多年以來，[[韓國|韓國]]各道之间的矛盾。

== 全羅道—慶尚道對立 ==
韓國[[全羅道|全羅道]]人與[[慶尚道|慶尚道]]人不和，可以追溯到[[统一新羅|统一新羅]]時代，[[百济|百济]]人成为[[新罗|新罗]]臣民。在[[高丽王朝|高丽王朝]]时候，亦規定全羅道人不得為官。[[朝鲜王朝|朝鲜王朝]]在[[鄭汝立|鄭汝立]]之乱前，在这里提拔不少人才，之后视同化外。

在现代韓國，多数總統来自較富裕的慶尚道，只有[[金大中|金大中]]来自全羅道。民主化後，通過地方自治，多数全羅道人不满意庆尚道人，但已没以前嚴重。

此種現象尤以國會議員選舉與總統選舉時最為突出，慶尚道民眾傾向於支持[[保守派|保守派]]陣營，而全羅道民眾則傾向於支持[[自由派|自由派]]陣營，而[[首爾|首爾]]和其周邊範圍則存在競爭，勢力相當。<ref>[https://news.mingpao.com/pns/dailynews/web_tc/article/20170511/s00014/1494440384125 世局解碼：韓國「嶺南」總統提名「湖南」總理 文在寅望破地域政治壁壘]</ref>

== 西北—東北歧視 ==
[[朝鲜王朝|朝鲜王朝]]對半島西北和東北地方如[[平安道|平安道]]與[[黃海道|黃海道]]和[[咸鏡道|咸鏡道]]人有歧视，在咸鏡道一帶，過去最主要民族是[[契丹人|契丹人]]與[[女真人|女真人]]。[[朝鲜人|朝鲜人]]是後來才來到。

朝鲜王朝建立后，对他们招撫，並出兵控制[[鴨綠江|鴨綠江]]以南。在[[李满住|李满住]]與[[董山|董山]]戰死後咸鏡道才成为朝鲜人的地方，平安道出身的人在李朝官吏叙用上受到長年歧视，引起後來[[两西大亂|两西大亂]]。

== 濟州的歧視 ==
濟州島以前稱州胡，有自己君長與文化，與一般朝鲜文化不同，在朝鲜王朝統治後，和[[江華島|江華島]]是流配罪人(政爭敗陣的[[兩班|兩班]]與王族)的地方。
