'''韓達拉'''（{{lang-ar|حنظلة}}）是由[[巴勒斯坦|巴勒斯坦]]卡通畫家[[納吉·阿里|納吉·阿里]]所創造的漫畫角色，他常於納吉·阿里繪製的政治漫畫裡串場，並被視為影射巴勒斯坦在國際間所處困境的代表。<ref>{{cite web |url=https://www.buzzfeed.com/palestinestudies/handala-the-little-palestinian-refugee-spct |title=Handala: The Little Palestinian Refugee |work=BuzzFeed |author=PalestineStudies |language=en |date=2014年10月2日 |accessdate=2017年1月26日}}</ref>

== 背景 ==
此角色最初於1969年期間[[科威特|科威特]]的報紙《Al-Siyasa》上登場。<ref name="Page54">{{cite web |url=http://www.netanya.ac.il/englishSite/Centers/DialogueCenter/Academic-publications/Documents/Discourse.pdf |title=Discourse, Culture, and Education in the Israeli-Palestinian Conflict |work=S. Daniel Abraham Center |author=Ashraf Ajrami、John Ashley、Yuval Benziman、Dalia Fadila、Elie Friedman、Nedal Jayousi、Motti Neiger、Soli Vered |language=en |format=[[PDF|PDF]] |page=54頁 |accessdate=2017年1月26日}}</ref>身份上韓達拉是個10歲巴勒斯坦小男孩、以[[難民|難民]]的身份活著、並且穿着破舊的衣裳及赤裸著雙腳。<ref name="Jadaliyya">{{cite web |url=http://www.jadaliyya.com/pages/index/5812/handala-will-age-again-soon |title=Handala Will Age Again Soon |work= |author=Jamil Sbitan |language=en |date=2012年6月3日 |accessdate=2017年1月26日}}</ref>而他的名字在阿拉伯語則有「苦澀」的含義。<ref name="Jadaliyya"/>

在1973年後納吉·阿里開始將他的姿態弄成總是雙手於背後交叉、背對著讀者不讓人看見他的情緒、像是旁觀者般地站著。<ref name="Page54">{{cite web |url=http://www.netanya.ac.il/englishSite/Centers/DialogueCenter/Academic-publications/Documents/Discourse.pdf |title=Discourse, Culture, and Education in the Israeli-Palestinian Conflict |work=S. Daniel Abraham Center |author=Ashraf Ajrami、John Ashley、Yuval Benziman、Dalia Fadila、Elie Friedman、Nedal Jayousi、Motti Neiger、Soli Vered |language=en |format=[[PDF|PDF]] |page=54頁 |accessdate=2017年1月26日}}</ref>而他交叉的雙手，被視為對於外界向巴勒斯坦所作出的干涉；所展現的一種拒絕態度。<ref>{{cite web |url=http://runforpalestine.com.au/meet-handala.html |title=Meet Handala |work=Run for Palestine |language=en |accessdate=2017年1月26日 |deadurl=yes |archiveurl=https://web.archive.org/web/20170217060847/http://runforpalestine.com.au/meet-handala.html |archivedate=2017年2月17日 }}</ref>

== 評價 ==
作者納吉·阿里生前認為韓達拉是代表著能保護他性格不陷入墮落的一種象徵，並且是個永遠朝向巴勒斯坦的指南針。<ref>{{cite web |url=https://www.caabu.org/sites/default/files/resources/Handala%20Handout.pdf |title=Who is Handala? |work=Caabu.Org |language=en |accessdate=2017年1月26日}}</ref>而[[美國|美國]]前總統[[歐巴馬|歐巴馬]]在會見[[以色列|以色列]]總理[[班傑明·納坦尼雅胡|納坦尼雅胡]]時表示著韓達拉反應出巴勒斯坦人內心深厚的遺棄感。<ref name="Page54"/>

== 引用 ==
在2009年發生的[[伊朗綠色革命|伊朗綠色革命]]中，在網路上有流傳著繪製韓達拉圍著綠色圍巾圖示；作為支持行動的象徵。<ref>{{cite web |url=http://www.huffingtonpost.com/shirin-sadeghi/qods-day-protesters-trans_b_291220.html |title=QODS DAY: Protesters Transform Jerusalem Day Into Iran Day |work=[[哈芬登郵報|哈芬登郵報]] |author=Shirin Sadeghi |language=en |date=2009年11月18日 |accessdate=2017年1月26日}}</ref>
