'''韓簡'''（{{bd|9世纪||883年||catIdx=Han韓簡}}），封'''魏郡王'''<ref name=NBT210>[[:s:新唐书/卷210|《新唐书》卷二百一十]]</ref>，后进'''昌黎郡王'''<ref name=BT181>[[:s:旧唐书/卷181|《旧唐书》卷一百八十一]]</ref>，[[晚唐|晚唐]]军阀，控制[[魏博|魏博]]。[[黄巢|黄巢]]起义后，朝廷权威瓦解，韩简试图武力吞并邻镇，失败后，所部推举将领[[乐行达|乐行达]]取代他。韩简最后忧虑而死，一说被杀。

== 背景 ==
韩简生年不详。他的父亲[[韩君雄|韩君雄]]长期在魏博为将領，在870年兵變杀害军纪苛酷的时任[[节度使|节度使]][[何全皞|何全皞]]后，控制魏博。<ref name=ZZTJ252>[[:s:资治通鉴/卷252|《资治通鉴》卷二百五十二]]</ref>韩君雄成为节度使后，韩简成为[[节度副使|节度副使]]。<ref name=BT181/>

== 初领魏博 ==
韩君雄后被赐名[[韩允中|韩允中]]。[[乾符|乾符]]初年，韩简累官至检校工部尚书。韩允中于乾符元年（874年）十一月去世，士兵推举韩简继任，[[唐僖宗|唐僖宗]]同意了，十二月起复韩简为节度观察留后，一个多月后加检校右仆射，二年（875年）三月任为兼魏州大都督府长史即实质上的全权节度使。累官（阙文）军、守左金吾卫大将军员外置同正员、[[御史大夫|御史大夫]]、[[上柱国|上柱国]]、赐紫金鱼袋。<ref>[[:s:赠太尉韩允忠神道碑|纥干濬《赠太尉韩允忠神道碑》]]</ref>三年（876年）七月，又授[[同中书门下平章事|同中书门下平章事]]的[[宰相|宰相]]荣衔，<ref name=ZZTJ252/>中和二年（882年）闰月，又授兼[[侍中|侍中]]。<ref name=ZZTJ255>[[:s:资治通鉴/卷255|《资治通鉴》卷二百五十五]]</ref><ref>[[:s:魏博韩简侍中|崔致远《魏博韩简侍中》]]</ref>他还被封为魏郡王，又进封昌黎郡王。<ref name=BT181/>

== 试图征服邻镇失败和去世 ==
[[黄巢|黄巢]]军攻占[[长安|长安]]迫使唐僖宗逃往[[成都|成都]]后，朝廷权威不复存在，[[軍閥|軍閥]]们为了争夺领地互相攻伐。八月，韩简率军三万攻[[河阳节度使|河阳节度使]][[诸葛爽|诸葛爽]]于[[修武|修武]]。诸葛爽弃[[河阳|河阳]]，韩简留赵文弁率军戍守修武河桥，<ref name=zgs>《旧唐书·诸葛爽传》作中和元年（881年）四月魏军大败诸葛爽夺取河阳，十月诸葛爽复入河阳，十一月攻新乡，次年正月韩简被杀。</ref><ref name=NBT9>[[:s:新唐书/卷9|《新唐书》卷九]]</ref><ref name=NBT187>[[:s:新唐书/卷187|《新唐书》卷一百八十七]]</ref>北上劫掠[[昭义|昭义]]的[[邢州|邢州]]、[[洺州|洺州]]后返回魏博。<ref name=ZZTJ255/><ref name=BT19-2>[[:s:旧唐书/卷19下|《旧唐书》卷一十九下]]</ref>

九月，韩简攻打[[天平军|天平军]]军部[[鄆州|鄆州]]，[[天平军节度使|天平军节度使]][[曹存實|曹存實]]战死。<ref>《旧唐书·僖宗纪》《旧唐书·韩简传》《旧唐书·朱瑄传》《新唐书·僖宗纪》等将此时天平军节度使误作[[曹全晸|曹全晸]]。</ref>曹存實部将[[朱瑄|朱瑄]]接掌天平军，防守郓州，韩简围城半年，未能攻克。十月，诸葛爽却收復了河阳，犒劳魏军，命赵文弁率他们离开。三年（883年）二月，朱瑄求和，韩简同意，从天平撤军，<ref>《旧唐书·韩简传》作曹全晸死后郓州将[[崔君裕|崔君裕]]守城，半年后请降韩简，误，据《资治通鉴》《新唐书·僖宗纪》，崔君裕已于乾符六年（879年）为曹全晸所讨杀。《新唐书·僖宗纪》称曹全晸死后部将[[崔用|崔用]]自称留后，亦误。见[[郁贤皓|郁贤皓]]《唐刺史考全编》。</ref><ref name=NBT188>[[:s:新唐书/卷188|《新唐书》卷一百八十八]]</ref>再战诸葛爽于河阳。诸葛爽派大将[[李罕之|李罕之]]在[[武陟|武陟]]、[[新乡|新乡]]拦截从郓州来战屯兵[[获嘉|获嘉]]西的韩简。当时韩简想引魏军入关诛灭黄巢，自己图王，不听三军劝谏。诸葛爽大败韩简八万军，清水为之不流。此役后，韩简大将[[澶州|澶州]]刺史[[乐行达|乐行达]]直接返回[[魏博|魏博]]军部[[魏州|魏州]]并占据之。士兵立乐行达为留后。<ref name=NBT187/>韩简单骑奔还，一说被气得疽发背而死，<ref name=NBT210/><ref name=BT181/>一说被部将所杀。<ref name=ZZTJ255/><ref name=zgs/><ref name=NBT9/><ref name=BT19-2/><ref name=BT182>[[:s:旧唐书/卷182|《旧唐书》卷一百八十二]]</ref>
