'''韓信草'''（[[學名|學名]]：''{{lang|la|Scutellaria indica}}''，[[英文|英文]]名稱：''{{lang|en| Skullcap }}''）<ref>[http://www.producegreen.org.hk/hkwildflower/100flower/flower_71.htm 香港野花網]綠田園</ref>，別稱'''大韓信草'''、'''耳挖草'''、'''挖耳草'''、'''大力草（廣州）'''、'''煙管草（江蘇無錫）'''、'''順草（貴陽）'''、'''順筋草'''、'''順經草'''、'''調羹草'''、'''疔瘡草'''、'''笑藥草'''、'''龍游香草'''、'''偏向花（浙江）'''、'''合耳花'''、'''三合香（河南商城）'''、'''紅葉犁頭草'''、'''紅葉犁頭尖（湖南永興）'''、''' 藍花茶匙癀 '''、'''虎咬癀'''、'''鐵燈盞 '''、'''油燈盞'''、'''向天盞'''、'''立浪草（台灣）'''、'''昏病草'''、'''地蘇麻'''、'''金鎖匙'''、'''金茶匙'''、'''半枝蓮'''、'''心葉半枝蓮'''、'''大葉半枝蓮'''、'''鉤頭線'''、'''印度黃芩（台灣）'''等<ref>[http://frps.plantphoto.cn/pdf2.asp?id=29551 中國植物志]{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}第172頁</ref><ref>[http://tai2.ntu.edu.tw/PlantInfo/species-name.php?code=523%20041%2002%200 台灣植物資訊整合查詢系統]國立台灣大學生態學與演化生物學研究所</ref><ref>[http://www.cfh.ac.cn/33093.sp Scutellaria indica   Linn.  韓信草]中國自然標本館</ref><ref name="意古中醫">[http://www.tcmlib.com/zy/html73/showdetail-373234312ce99fa9e4bfa1e88d892c7a79.html 韓信草]{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}意古中醫</ref>，為[[唇形科|唇形科]][[黄芩屬|黄芩屬]]植物，[[模式種|模式種]]採自於中國<ref>[http://www.zhiwutong.com/latin/Labiatae/Scutellaria-indica-Linn.htm 韩信草]植物通</ref>。韓信草的花萼於果熟時會增大閉合，待種子飛散後剩下的下唇堅硬狀如耳挖，故而有耳挖草之稱<ref name = "金門">[http://digital-kinmen.kmnp.gov.tw/files/806-1000-2446,c79-1.php　數位典藏網]金門國家公園</ref><ref name = "香港野花">{{cite book|title=《香港野花 第一冊》|author= |series=魏遠娥、洪烈招、陳淑瓊、陳愛強.|publisher=效野公園之友會、天地圖書|page=第266-267頁|isbn= 978-988-211-344-2|date=2007年7月}}</ref>。韓信草的花朵，於風中挄動仿如一片紫藍色的波濤向前湧，故而有立浪草之稱<ref name = "香港野花"/>。一般是在[[春|春]]、[[夏|夏]]季採取，洗淨鮮用或曬乾用<ref name = "浸會大學">[http://libproject.hkbu.edu.hk/was40/detail?record=3&channelid=1288&searchword=%28alphabet%3DL%29+And+parent_latin%3D%27Scutellaria%27&sortfield=+name_formal_latin_sort 藥用植物圖像數據庫]香港浸會大學中醫藥學院</ref>
。花期4－5月，果期6－9月<ref name = "浸會大學"/>。

==分佈==
分佈於[[中國|中國]]、[[台灣|台灣]]、[[日本|日本]]、[[朝鮮|朝鮮]]、[[中南半島|中南半島]]、[[印尼|印尼]]及[[印度|印度]]等地。中國國內分佈於[[江蘇|江蘇]]、[[浙江|浙江]]、[[安徽|安徽]]、[[江西|江西]]、[[湖南|湖南]]、[[河南|河南]]、[[陝西|陝西]]、[[貴州|貴州]]、[[四川|四川]]、[[雲南|雲南]]、[[廣西|廣西]]、[[廣東|廣東]]、[[海南|海南]]及[[福建|福建]]等地。多生長於海拔1500米以下濕潤的山坡、丘陵地、郊遊草地、疏林下及路旁等<ref name = "廣東植物志"/><ref name = "意古中醫"/><ref name = "浸會大學"/><ref name = "安徽植物志">[http://www.bhl-china.org/data/index.php?name=Scutellaria%20indica 安徽植物志]第248頁</ref><ref name="秦嶺植物誌">[http://www.cvh.org.cn/difangzhi/qinling/page/1(4)/221a.pdf  秦嶺植物誌]{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}第221-222頁</ref>。

==形態特徵==
韓信草是一種多年生[[草本植物|草本植物]]，全株披毛，高約10－37厘米。根莖短，有簇生的纖維狀鬚根。
[[莖|莖]]直立或斜升 ，由根狀莖發出，四梭形，通常帶暗紫色，密披向上曲的長節毛，尤其莖上部及棱角上最密。
葉[[賢形|賢形]]或[[卵狀橢圓形|卵狀橢圓形]]，草質至近堅紙質，先端純圓或鈍尖，基部圓形或微心形，對生，葉緣反卷，邊緣密生圓鋸齒及節狀短緣毛，表面暗綠色，背面淡綠色或紫紅色，兩面密披柔毛，背面披毛較盛，有時背面具紅色腺點，長約1.5－3厘米，寬約1.2－3.2厘米;葉柄密披短柔毛，長約5－15毫米。
花輪有花2朵，頂生假總狀[[花序|花序]]，花偏向一側，長約3－12厘米;總花梗上密披節毛和節毛狀的頭狀腺毛，下部一對[[苞片|苞片]]葉狀，卵狀長圓形，上部苞片橢圓形，全緣，密披柔毛，無柄，長約3－6毫米，寬約1－2.5毫米；[[花萼|花萼]]鐘狀，外面密披節毛狀的頭狀腺毛，2唇形，全緣，長約2.5毫米，果時增大至長約5毫米，[[萼筒|萼筒]]背部生有1囊狀盾鱗，盾片高約1.5毫米，果期增大一倍；[[花冠|花冠]]藍紫色，2唇形，外面疏披短節毛和腺體，基部囊狀，[[冠筒|冠筒]]前方基部明顯膝曲，向上漸寬大，長約14－20毫米，喉部寬約4.5毫米；[[冠檐|冠檐]]上唇盔狀，先端微凹，下唇3裂片，中裂片卵圓形，先端微凹，2側裂片小，卵狀三角形，具深紫色斑點；雄蕊4枚，2強，稍外露，前對外側花藥退化，[[花絲|花絲]]扁平，中部前對外側、後對兩側披柔毛；[[花柱|花柱]]細長，稍外露；[[柱頭|柱頭]]2裂不相等；[[子房|子房]]光滑具柄，4裂，花盤肥厚。[[果|果]]為小[[堅果|堅果]]，卵球形，橫生，褐色或栗色，具瘤，腹面中下部有具柄果臍，每室[[種子|種子]]４顆，長約1毫米，直徑不及1毫米。種子黑色。<ref name = "廣東植物志"/><ref name = "意古中醫"/><ref name = "金門"/><ref name = "浸會大學"/><ref name = "安徽植物志"/><ref name = "秦嶺植物誌"/>

==醫藥用途==
{{medical}}
中藥名為韓信草，性味苦、辛，性寒，歸心、[[肺|肺]]及[[肝|肝]]經，能清熱解毒、 止血消腫、活血止痛等，主治肺癰、肺熱咳喘、皮膚瘙癢、腸癰、 牙痛、咽痛、喉痹、便血、吐血、筋骨疼痛 、創傷出血、跌打損傷及毒蛇咬傷等。<ref name = "意古中醫"/><ref name = "浸會大學"/>

==化學成份==
根含[[黃酮類|黃酮類]]、[[酚性|酚性]]成分、[[氨基酸|氨基酸]]、[[有機酸|有機酸]]等成份組成，如[[半枝蓮種素|半枝蓮種素]](tivularin)、[[半枝蓮素|半枝蓮素]](scutevurin)、[[高分黃芩甙|高分黃芩甙]](scutellarin)、[[漢黃芩素|漢黃芩素]]（wogo-nin)、[[山姜素|山姜素]]（alpinetin）等。<ref name = "意古中醫"/><ref name = "浸會大學"/>
