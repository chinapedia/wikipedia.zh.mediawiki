'''韓石泉'''（{{bd|1897年|10月27日|1963年|6月30日|catIdx=Han韓石泉}}{{r|韓}}{{r|記}}），[[號|號]]'''南陽'''，[[臺南|臺南]]人，[[臺灣|臺灣]]醫學家、社會活動家。戰後曾任台南市參議員。

== 早年生平 ==
韓石泉於丁酉年農曆十月二日（1897年10月27日）出生於臺南。父親韓子星、叔父韓子明均為前清[[秀才|秀才]]，{{r|六|page=59}}父親韓子星在1895年《馬關條約》割臺後曾一度逃往[[廈門|廈門]]。後來返台於天壇創設[[私塾|私塾]]「尚志齋書房」。{{r|六|page=58}}韓石泉七歲時曾一度就讀[[重慶寺|重慶寺]]私塾。{{r|六|page=59}}八歲時就入臺南第一公學校就讀，{{r|六|page=60}}公學校下課後至天壇私塾讀漢文。[[明治|明治]]四十四年（1911年）畢業。

[[公學校|公學校]]畢業後，由於距投考[[臺灣總督府醫學校|臺灣總督府醫學校]]報考年齡還有兩年，{{r|六|page=67}}一度考入[[臺南廳|臺南廳]]財政課擔任「給仕」（雜役），業餘時間仍勤奮自學，後於[[大正|大正]]二年（1913年）考取[[臺灣總督府醫學校|臺灣總督府醫學校]]的公費生{{r|韓}}{{r|記}}，4月北上入臺灣總督府醫學校。{{r|六|page=71}}二年級時父親病危，返家處理各項事務約一個月，1916年2月21日父親病逝。喪事完畢後方回校補考，但其成績仍保持名列前茅{{r|韓}}。

韓石泉在三年級出任學寮組長，因為不滿學校賣店小使（工友）壟斷販賣部、兼任學校情報員向舍監瀧野舉報學生行為、思想，韓石泉遂發起罷買運動，使該小使損失慘重。該小使報告舍監瀧野。使韓石泉遭舍監瀧野召至舍監室，痛罵其為流氓、與[[余清芳|余清芳]]同等人物，揚言定要開除其學籍。{{r|六|page=82}}最後在校長[[堀內次雄|堀內次雄]]介入下，使韓石泉逃過退學處分，瀧野只好降氣求和。{{r|六|page=83}}

大正七年（1918年）4月，韓石泉自臺灣總督府醫學校畢業。<ref>[http://ds3.th.gov.tw/ds3/app007/list3.php?ID1=0071021559a005 卒業證書授與（醫學專門學校）]，臺灣總督府府報資料庫查詢。</ref>

== 執業時期 ==
畢業後，韓石泉先在臺北的日本紅十字社臺灣支部醫院（現在[[臺北市立聯合醫院|臺北市立聯合醫院]]中興院區）內科工作，一年後（1919年）在堀內校長介紹下，改到[[臺灣總督府|臺灣總督府]]臺南病院（今 [[衛生福利部臺南醫院|衛生福利部臺南醫院]]）內科服務，剛開始為醫務助手，月俸二十二圓。三年後升為醫官補，任判任官。韓石泉也是臺南病院內科中首位臺籍醫官補（外科為[[王受祿|王受祿]]）。{{r|韓}}{{r|記}}{{r|六|page=97}}在臺南病院工作其間，韓石泉受到院長[[明石真隆|明石真隆]]薰陶增進醫學與醫術外，{{r|六|page=99}}也見證院中臺籍職員與日籍職員的對立，以及部分日籍職員的差別觀念。{{r|六|page=98}}

在臺南病院工作四年後，韓石泉於大正十一年（1922年）冬離職，與[[黃金火|黃金火]]於今[[民權路_(臺南市)|民權路]]上開設「共和醫院」，韓石泉負責內科，{{r|六|page=109}}黃金火負責外科{{r|韓}}。大正十五年（1926年）3月31日，韓石泉與莊綉鸞於[[臺南公會堂|臺南公會堂]]結婚。{{r|六|page=118}}[[昭和|昭和]]三年（1928年）3月，韓石泉改在對面（本町四丁目）開設韓內科醫院。{{r|韓}}{{r|六|page=118}}

而在韓石泉的醫生事業逐漸發展的同時，他也積極參與社會活動，在1921年加入[[臺灣文化協會|臺灣文化協會]]並擔任理事{{r|記}}。1923年12月爆發[[治警事件|治警事件]]時，韓石泉亦遭到逮捕。韓石泉於12月16日臨晨於睡夢中遭日警逮捕，羈留臨時收容所（後臺南空軍醫院）。{{r|六|page=110}}後被移送臺北，與[[蔡培火|蔡培火]]、[[陳逢源|陳逢源]]、[[吳海水|吳海水]]一同被解押至[[臺北刑務所|臺北監獄]]（今金山南路二段52號中華電信）。在審判中，一審被判罰金百圓，{{r|臺|page=272}}最後二審被判無罪定讞{{r|韓}}{{r|記}}。韓石泉為[[治警事件|治警事件]]18人遭起訴者中年紀第三輕者，僅年長於[[石錫勳|石錫勳]]、吳海水。

在臺灣文化協會分裂後，於1927年7月成立[[臺灣民眾黨|臺灣民眾黨]]。韓石泉擔任中央委員，負責臺南支部的事務，8月臺南支部後出任四名常務委員之一（包括王受祿）。1928年臺灣民眾黨在臺南召開第二次黨員代表大會，由王受祿出任議長、韓石泉出任副議長，共同主持大會。韓石泉參與臺灣民眾黨事務直到1931年2月18日臺灣民眾黨被強制解散為止。{{r|韓}}{{r|六|page=116}}

昭和十年（1935年）3月，韓石泉受到長子良哲夭折及臺灣民眾黨遭解散的打擊，遂決定前往日本求學。{{r|六|page=129}}適逢明石真隆時任[[九州|九州]][[熊本醫科大學|熊本醫科大學]]校長，韓石泉遂入明石內科研究室研究，{{r|六|page=129}}4月母親逝世。{{r|六|page=130}}其間因氣候不適與研究壓力等因素，曾罹患坐骨神經痛、胃出血等症狀；後於1940年完成研究工作，以《脾臟燐脂質之研究》、《關於膽汁之分泌作用及其成份》等論文取得醫學博士學位{{r|韓}}{{r|記}}。

取得學位之後，韓石泉返回臺南行醫，但在[[第二次世界大戰|第二次世界大戰]]末期，韓內科醫院毀於1945年3月1日的[[臺南大空襲|臺南大空襲]]之中，其長女韓淑英亦不幸罹難。{{r|韓}}{{r|記}}{{r|六|page=152}}

== 戰後 ==
二次大戰後，韓石泉被聘為自治指導員，協助國府接收工作，後來更擔任中國國民黨臺南市黨部指導員{{r|韓}}{{r|記}}。1946年4月15日他當選為臺灣省參議會第1屆參議員（選區[[臺南市_(省轄市)|臺南市]]），任職期間曾協助處理教育處長[[范壽康|范壽康]]的失言事件，而在[[二二八事件|二二八事件]]發生時，提出「不擴大，不流血，不否認現有行政機構，政治問題用政治方法解決」的四大原則來處理臺南市的動盪，降低臺南市所受到的影響{{r|韓}}{{r|記}}。

1947年韓石泉參加[[1947年中華民國國民大會代表選舉|區域性國大代表選舉]]，然而當時「某方面已屬意某候選人，曾力為支持，使余甚難獲得更多之票數也。」，<ref>在韓良俊編註的《六十回憶：韓石泉醫師自傳》中指出「此位某候選人是誰，該次選舉結果即是答案」，而該屆臺南市當選人為[[連震東|連震東]]。見韓良俊編註，《六十回憶：韓石泉醫師自傳》。2009年2月初版，頁206</ref>最後獲2129票落選。{{r|韓}}{{r|記}}{{r|六|page=206}}韓石泉晚年不再參與政治活動{{r|記}}。

而除了政治活動外，韓石泉在1945年到1947年間任臺南市私立光華女子中學（今[[臺南市私立光華高級中學|臺南市私立光華高級中學]]）校長，1947年到1963年任臺南市私立光華女子高級中學董事長。另外還曾擔任[[中華民國紅十字會|中華民國紅十字會]]臺南市分會分隊長、臺南市第二信用合作社理事主席等職{{r|韓}}{{r|記}}。

1963年6月29日，因天氣炎熱與終日勞累，韓石泉於夜裡突然發生腦溢血，至30日逝世{{r|韓}}。
