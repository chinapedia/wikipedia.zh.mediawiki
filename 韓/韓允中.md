'''韓允中'''（{{bd|814年||875年|1月3日|catIdx=Han韓允中}}），或作'''韩允忠'''，本名'''韓君雄'''，[[賜名|賜名]]'''允中'''，[[唐朝|唐朝]]将领，在一次兵变中控制了[[魏博镇|魏博镇]]并成为其[[节度使|节度使]]，实质独立于朝廷。

== 背景 ==
韩君雄生于[[唐宪宗|唐宪宗]]年间的814年，是魏博军部[[魏州|魏州]]人氏。父[[韓國昌|韓國昌]]长期在魏博节度使[[何弘敬|何弘敬]]手下为右职，韩君雄也因而在年轻时就得以在军中效力，升裨校，在[[唐武宗|唐武宗]]年间的833年—834年间和父亲一同参与了何弘敬协助朝廷军队讨伐不经朝廷许可，就自命[[昭義節度使|昭義節度使]]的[[刘稹|刘稹]]之战。<ref name=BT181>[[:s:旧唐书/卷181|《旧唐书》卷一百八十一]]</ref>

== 夺取魏博 ==
何弘敬的儿子[[何全皞|何全皞]]继任为节度使，[[唐懿宗|唐懿宗]]咸通十一年（870年）八月，他[[严刑峻法|严刑峻法]]，又传言他要削减士兵在衣食上的补贴。士兵兵变，何全皞逃跑时被杀。士兵推举韩君雄为[[留后|留后]]。<ref>[[:s:旧唐书/卷19上|《旧唐书》卷一十九上]]</ref><ref>[[:s:旧唐书/卷19上|《旧唐书》卷一十九上]]</ref><ref name=NBT9>[[:s:新唐书/卷9|《新唐书》卷九]]</ref><ref>[[:s:旧五代史/卷14|《旧五代史》卷一十四]]</ref><ref>[[:s:新五代史/卷39|《新五代史》卷三十九]]</ref>由于相邻的[[成德节度使|成德节度使]][[王景崇_(唐朝)|王景崇]]的支持，九月，懿宗授韩君雄[[银青光禄大夫|银青光禄大夫]]、左散骑常侍、兼[[御史中丞|御史中丞]]，充节度观察留后，<ref>[[:s:与韩君雄书|郑畋《与韩君雄书》]]</ref><ref name=ZZTJ252>[[:s:资治通鉴/卷252|《资治通鉴》卷二百五十二]]</ref>而任皇子普王[[唐僖宗|李儼]]为遥领的名义节度使。<ref name=BT181/>十二年（871年）正月，进副大使，转检校工部尚书、魏州大都督府长史、充魏博节度观察等使。<ref name=ZZTJ252/><ref name=NBT210>[[:s:新唐书/卷210|《新唐书》卷二百一十]]</ref><ref name=HYZ>[[:s:赠太尉韩允忠神道碑|纥干濬《赠太尉韩允忠神道碑》]]</ref>

三迁至检校[[司空|司空]]。十四年（873年），唐懿宗驾崩，李儼继位，即[[唐僖宗|唐僖宗]]。僖宗刚登基即授韩君雄等节度使为[[同中書門下平章事|同中書門下平章事]]，还为韩君雄赐名允中。<ref name=BT181/><ref name=ZZTJ252/><ref name=NBT210/>

[[乾符|乾符]]元年（874年）十一月，韩允中去世。士兵推举他任副节度使的儿子[[韩简_(唐朝)|韓簡]]继任，<ref name=NBT9/>唐僖宗认可了。<ref name=ZZTJ252/>韩允中得册赠[[司徒|司徒]]，累赠[[太尉|太尉]]。<ref name=BT181/><ref name=NBT210/>

== 家族 ==
*曾祖父韩朝，魏博临清镇都知兵马使
*祖父（名阙），（阙文）夫、检校太子宾客、使持节[[相州_(北周)|相州]]诸军事、守相州刺史、充本州防御使、御史中丞，赠工部尚书
*祖母张氏，追封清河郡太君
*父韩国昌，银青光禄大夫、检校国子祭酒、使持节[[贝州|贝州]]诸军事、守贝州刺史、充本州防御使，兼御史中丞，赠兵部尚书
*母吴氏，追封凉国太夫人
*妻刘氏，追封彭城郡太君，早逝
*继妻靳氏，封清河郡君，先韩允中而亡
*长子韩简，（阙文）军、守左金吾卫大将军员外置同正员、检校尚书右仆射兼魏州大都督府长史、御史大夫、[[上柱国|上柱国]]、赐紫金鱼袋
*次子韩记，前魏州都督府文学
*三子韩谏，亲事将军<ref name=HYZ/>
