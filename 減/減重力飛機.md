'''減重力飛機'''（Reduced-gravity aircraft）是一種[[固定翼飛機|固定翼飛機]]，提供短暫的近乎失重環境，可以進行[[太空人|太空人]]訓練、研究，或拍攝無重力的[[電影|電影]]畫面。

這種飛機正式的暱稱是「失重奇蹟」（Weightless Wonders）<ref>{{cite web |url=http://www.nasaexplores.com/show2_articlea.php?id=03-008 |title=存档副本 |accessdate=2008-06-12 |deadurl=yes |archiveurl=https://web.archive.org/web/20080612103946/http://www.nasaexplores.com/show2_articlea.php?id=03-008 |archivedate=2008-06-12 }}</ref>，是[[美國太空總署|美國太空總署]]低重力研究項目計畫<ref>{{cite web |url=http://jsc-aircraft-ops.jsc.nasa.gov/Reduced_Gravity/index.html |title=NASA Reduced Gravity Research Program |publisher=Jsc-aircraft-ops.jsc.nasa.gov |date=2009-03-17 |accessdate=2009-09-14 |deadurl=yes |archiveurl=https://web.archive.org/web/20090825085742/http://jsc-aircraft-ops.jsc.nasa.gov/Reduced_Gravity/index.html |archivedate=2009-08-25 }}</ref>。非官方的綽號「嘔吐彗星」也聞名於世<ref>{{cite web |url=http://www.nasa.gov/audience/forstudents/brainbites/nonflash/bb_home_vomitcomet.html |accessdate=May 15, 2012 |deadurl=yes |archiveurl=https://web.archive.org/web/20120505001314/http://www.nasa.gov/audience/forstudents/brainbites/nonflash/bb_home_vomitcomet.html |archivedate=2012年5月5日 |title= }}</ref>。
