{{vgname|惡魔獵人2|ja=デビルメイクライ2|en=Devil May Cry 2|cn=鬼泣2|tw=惡魔獵人2|大陆行货版官方译作'''恶魔猎人2'''}}，是日本[[卡普空|卡普空]]在[[PlayStation_2|PlayStation 2]]推出的第二款「[[惡魔獵人系列|惡魔獵人系列]]」作品，在2003年1月25日正式發售。

== 系統 ==

; 惡魔之心系統
: 《惡 魔獵人2》中獨特的惡魔之心系統，是把「魔石」裝備在但丁的「魔化徽章(AMULET)」上，讓魔人的能力產生變化的系統。當但丁變身為魔人時所能使用的 特殊能力會依裝備的魔石而改變。魔石有隱藏「屬性攻擊系」、「移動系」等各式各樣的特殊能力，只要裝備魔石之後進行「惡魔觸發」，就能夠獲得新的惡魔之 力。靈活驅使「惡魔之心系統」來狩獵惡魔。
; 角色選擇
: 《惡魔獵人2》中玩家可以自由選擇但丁或路西亞來進行遊戲，兩名主角的動作特性完全不一樣，擅長的地形當然也不相同。

== 情節 ==

=== 登場角色 ===
{{main|鬼泣角色列表}}

; [[但丁_(惡魔獵人)|但丁]]（Dante）
: 但丁繼續擔任續作《惡魔獵人2》的主角，具有半人半魔的身分。
; 路西亞（Lucia）
: 路西亞是《惡魔獵人2》的女主角，名字源自[[神曲|神曲]]中。路西亞是亞歷烏斯製造的惡魔，後期才知道自己的真實身分，現在是「守護之手」之繼承人。
; 亞歷烏斯（Arius）
: 亞歷烏斯表面為國際企業之總裁，對於宗教及古代魔術文化有深入的研究。為了得到邪神的力量而進行召換的儀式，但最後被但丁阻止。
; 馬蒂爾（Matier）
: 路西亞名義上的母親。曾經與斯巴達共同作戰過，有著「守護之手」的稱號。

=== 故事 ===

《惡魔獵人2》的故事是由但丁或路西亞兩人所交織出來的。經營著“Devil May Cry ”事務所的惡魔獵人Dante在博物館協助一名神秘女子Lucia消滅了忽然出現的惡魔，之後接到女子委託前來到“聖與魔的交融島嶼”杜瑪利島上，從Lucia的母親Matier得知了野心家Arius為了復活邪神的消息，同時也得知了自己父親Sparda曾經封印邪神的事蹟。為了阻止邪神的複活，Dante與Luica一同來到了Arius的總部。在此，Lucia卻得知自己是由Arius用魔力創造的魔人的事情。在Dante擊敗了Arius並阻止了儀式後，Luica要求Dante殺了自己，因為自己是一個魔人。 Dante告訴Lucia：“Devils Never Cry。”擦去了Luica的眼淚。這時，魔界的大門再次在島嶼上方開啟，Dante獨自一人，繼承父親Sparda的意志，進入了魔界。在魔界中，Dante封印了因為儀式中斷而並沒有完全覺醒的邪神ARGOSAX。而在島上，Lucia也不再為自己的身世而困擾，打敗了已成為邪神ARGOSAX媒介體的Arius。最後，在惡魔獵人事務所內，Lucia獨自拿著Dante留下的硬幣，等待著Dante的歸來。這時，事務所門外，傳來了一陣熟悉的摩托聲。

== 歷史 ==

由於前作《[[惡魔獵人|惡魔獵人]]》取得空前的成功，故[[卡普空|卡普空]]隨即宣佈開發續作。卡普空與[[卡普空第四開發部|卡普空第四開發部]]的[[神谷英樹|神谷英樹]]意見不合，《惡魔獵人2》便交由[[卡普空第一開發部|卡普空第一開發部]]負責，由[[伊津野英昭|伊津野英昭]]擔當遊戲監督，[[田中剛|田中剛]]擔任遊戲製作人。

== 爭議 ==

卡普空與[[卡普空第四開發部|卡普空第四開發部]]的[[神谷英樹|神谷英樹]]意見不合，《惡魔獵人2》便交由[[卡普空第一開發部|卡普空第一開發部]]負責，由[[伊津野英昭|伊津野英昭]]擔當遊戲監督，[[田中剛|田中剛]]擔任遊戲製作人。

也因為風格與之前的一代相差太大，所以人氣下滑許多，因此三代才繼續由[[卡普空第四開發部|卡普空第四開發部]]繼續開發。
