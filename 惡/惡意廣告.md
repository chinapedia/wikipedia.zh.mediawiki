'''惡意廣告'''是種透過線上廣告來散播[[惡意軟體|惡意軟體]]的一種[[廣告|廣告]]。<ref name="William Salusky">{{Template:Cite web|url = http://isc.sans.edu/diary.html?storyid=3727|title = Malvertising|publisher = SANS ISC|date = 2007-12-06|author = William Salusky|accessdate = 2010-08-05}}</ref>

惡意廣告透過線上[[广告网络|廣告網路]]及[[網頁|網頁]]的侵入型廣告來散播惡意軟體。<ref name="OTA">{{Template:Cite web|url = http://isc.sans.edu/diary.html?storyid=3727|title = Malvertising|publisher = SANS ISC|date = 2007-12-06|author = William Salusky|accessdate = 2010-08-05}}</ref>線上廣告提供散步[[惡意軟體|惡意軟體]]一個溫床因為可以透過其向使用者利用明顯的宣傳手法吸引點擊。<ref name="CFS">{{Template:Cite web|url = http://isc.sans.edu/diary.html?storyid=3727|title = Malvertising|publisher = SANS ISC|date = 2007-12-06|author = William Salusky|accessdate = 2010-08-05}}</ref> 因為廣告內容可以在知名的網頁上面顯示，這點使[[電腦犯罪|網路犯罪]]者有了一個平台來散布惡意軟體給一些沒有注意廣告或安全概念的[[使用者|使用者]]。<ref>{{Template:Cite web|url = http://isc.sans.edu/diary.html?storyid=3727|title = Malvertising|publisher = SANS ISC|date = 2007-12-06|author = William Salusky|accessdate = 2010-08-05}}</ref><ref>{{Template:Cite web|url = http://isc.sans.edu/diary.html?storyid=3727|title = Malvertising|publisher = SANS ISC|date = 2007-12-06|author = William Salusky|accessdate = 2010-08-05}}</ref> 惡意廣告是 對於犯罪者有吸引力並且可以透過其快速散播於大量的合法網站且不需與個別網站進行溝通。<ref name="zeltser">{{Template:Cite web|url = http://isc.sans.edu/diary.html?storyid=3727|title = Malvertising|publisher = SANS ISC|date = 2007-12-06|author = William Salusky|accessdate = 2010-08-05}}</ref>

== 如何運作 ==
網站或網路發布者會在沒有預先了解的情況下收錄了惡意廣告於他們的頁面當中。當廣告已經在頁面中並且瀏覽者已經點擊，他們的電腦可能已經[[電腦中毒|中毒]]，“用戶點擊廣告並且瀏覽廣告網頁，或者被重新導向至惡意網站，這些網站會利用技巧來讓用戶安裝看似非常類似 [[Flash_player|Flash]]播放器軟體的[[間諜軟體|間諜軟體]]。”<ref name="Bullguard">{{Template:Cite web|url = http://isc.sans.edu/diary.html?storyid=3727|title = Malvertising|publisher = SANS ISC|date = 2007-12-06|author = William Salusky|accessdate = 2010-08-05}}</ref>[[重新導向|重新導向]]非常容易出現在線上廣告中, 並且散布惡意軟體因為用戶很容易以為這就是他們原先點擊的廣告。<ref name="William Salusky">{{Template:Cite web|url = http://isc.sans.edu/diary.html?storyid=3727|title = Malvertising|publisher = SANS ISC|date = 2007-12-06|author = William Salusky|accessdate = 2010-08-05}}</ref>

== 方法及模式 ==
瀏覽網站非常容易被惡意廣告引響且非常有風險。以下列出不同種廣告或[[軟體|軟體]]於網頁中顯示的方法:
* [[弹出式广告|彈出式廣告]]是非常常見偽裝成下載軟體，甚至是假的[[防毒軟體|防毒軟體]]並安裝許多惡意軟體於電腦中<ref name="OTA">{{Template:Cite web|url = http://isc.sans.edu/diary.html?storyid=3727|title = Malvertising|publisher = SANS ISC|date = 2007-12-06|author = William Salusky|accessdate = 2010-08-05}}</ref>
* 文字廣告或段落中廣告
* [[路过式下载|路過式下載]]<ref name="OTA">{{Template:Cite web|url = http://isc.sans.edu/diary.html?storyid=3727|title = Malvertising|publisher = SANS ISC|date = 2007-12-06|author = William Salusky|accessdate = 2010-08-05}}</ref>
* [[Web_Widget|網頁小工具]]可以指定重新導向至一個惡意網站<ref name="CFS">{{Template:Cite web|url = http://isc.sans.edu/diary.html?storyid=3727|title = Malvertising|publisher = SANS ISC|date = 2007-12-06|author = William Salusky|accessdate = 2010-08-05}}</ref>
* 隱藏的[[HTML元素|iframes]]框架散播惡意軟體<ref name="CFS">{{Template:Cite web|url = http://isc.sans.edu/diary.html?storyid=3727|title = Malvertising|publisher = SANS ISC|date = 2007-12-06|author = William Salusky|accessdate = 2010-08-05}}</ref>
* 濫用[[內容傳遞網路|內容傳遞網路]]來散布惡意軟體<ref name="CFS">{{Template:Cite web|url = http://isc.sans.edu/diary.html?storyid=3727|title = Malvertising|publisher = SANS ISC|date = 2007-12-06|author = William Salusky|accessdate = 2010-08-05}}</ref>
* 惡意橫幅廣告<ref name="CFS">{{Template:Cite web|url = http://isc.sans.edu/diary.html?storyid=3727|title = Malvertising|publisher = SANS ISC|date = 2007-12-06|author = William Salusky|accessdate = 2010-08-05}}</ref>
* 網頁中的第三方廣告<ref name="readwrite">{{Template:Cite web|url = http://isc.sans.edu/diary.html?storyid=3727|title = Malvertising|publisher = SANS ISC|date = 2007-12-06|author = William Salusky|accessdate = 2010-08-05}}</ref>
* 第三方軟體，像是[[論壇|論壇]]、幫助平台、[[客户关系管理|客戶關係管理系統]]以及[[内容管理系统|內容管理系統]]<ref name="readwrite">{{Template:Cite web|url = http://isc.sans.edu/diary.html?storyid=3727|title = Malvertising|publisher = SANS ISC|date = 2007-12-06|author = William Salusky|accessdate = 2010-08-05}}</ref>
