{{vgname|狙击手：幽灵战士3|en=Sniper: Ghost Warrior 3|cn=狙击手：幽灵战士3|tw=狙擊之王：幽靈戰士3}}是由[[波兰|波兰]][[CI_Games|CI Games]]公司开发的一款[[第一人称射击游戏|第一人称射击游戏]]。《狙击手：幽灵战士3》是一个发生在被内战的鲜血染红的土地上，关于友情，信仰与背叛的故事。玩家是置身于[[格鲁吉亚|格鲁吉亚]]的[[美国|美国]]狙击兵，所处的位置靠近[[俄罗斯|俄罗斯]]边界，以终极现代军旅神射手的身份深入敌后。玩家将在一个严酷而开放的世界里履行使命<ref>{{Cite web |url = http://www.sonkwo.com/products/2324 |title = 狙击手：幽灵战士3 |author = |date = 2016-08-02 |work = 杉果游戏 |publisher = 中电博亚 |language = |accessdate = |quote = }}</ref>。本作延续了系列一贯强调的拟真狙击体验，环境条件、呼吸控制、射击姿势与子弹选择都会影响射击的轨迹。由于采用了开放世界的设定，诸如天气变幻、昼夜更替等也会对战场产生影响。枪械种类更加繁多，玩家也根据个人风格进行改装，而近年来军事题材游戏中频繁出现的无人机设备也没有缺席<ref name="multiple">{{Cite web|url =http://www.sonkwo.com/articles/2148|title =大量实机截图公布！《狙击手：幽灵战士3》抢先看|author =|date =2016-08-08|work =杉果游戏|publisher =中电博亚|language =|accessdate =|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20160907153802/http://www.sonkwo.com/articles/2148|archivedate =2016-09-07}}</ref>。

游戏確定于2017年4月25日发行，在[[中国大陆|中国大陆]]由[[杉果游戏|杉果游戏]]代理同步发行。目前游戏可进行预购，预购奖励包括一段名为“莉蒂亞的逃亡”的单人战役及“[[复合弓|复合弓]]”独占武器<ref name="multiple" />。
