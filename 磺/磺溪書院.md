'''磺溪書院'''（俗稱'''文昌祠'''）位於[[臺灣|臺灣]][[臺中市|臺中市]][[大肚區|大肚區]]磺溪里文昌路上的古蹟。書院的前身稱為「'''西雝社'''」亦稱為「'''文昌會'''」，是當代文人士子的結集之所。西雝社創建於清[[嘉慶|嘉慶]]四年（1799年），後來在[[光緒|光緒]]十三年（1887年）再擴建成為「磺溪書院」。<ref name="hs01">陽鴻榮.[https://market.cloud.edu.tw/content/junior/chinese/tc_md/wujih/con9.htm 先賢事蹟]</ref>

在民國七十五年（1986年），經[[中華民國內政部|中華民國內政部]]列為第三級古蹟。<ref name="hs02">[https://travel.taichung.gov.tw/zh-tw/Attractions/Intro/275/%E7%A3%BA%E6%BA%AA%E6%9B%B8%E9%99%A2 磺溪書院] .台中觀光旅遊網</ref>
[[File:磺溪書院（二）.JPG|thumb]]

== 歷史 ==
[[File:磺溪書院－內院中庭.JPG|thumb]]
西雝社<ref>「雝」發音同「庸」</ref>是[[清朝|清朝]]居住在今臺中市烏日區的[[舉人|舉人]][[楊占鰲|楊占鰲]]（[[清朝|清]][[道光|道光]]十九年舉人）下寮尾座（今[[磺溪里|磺溪里]]）仕紳[[趙順芳|趙順芳]]所創建，原本是一些文人士子以文會友的集會場所，同時也有許多士紳募集善款當作學租招收孤寒學子，所以也有[[義學|義塾]]的用途<ref name="hs01"/>。後來有感於該區的文教風氣尚嫌不足有待發揚，所以經趙順芳還有趙順芳之子[[趙壁興|趙壁興]]、夥同汴仔頭（今[[永順里|永順里]]）商號-蔡勝記的[[蔡燦雲|蔡燦雲]]、[[蔡瀚雲|蔡瀚雲]]兄弟商議([[蔡瀚雲|蔡瀚雲]]後擔任日治時期台中州大甲郡龍井庄/今[[龍井區|龍井區]]庄長)。由仕紳[[蔡燦雲|蔡燦雲]]、[[蔡瀚雲|蔡瀚雲]]、與[[張錦上|張錦上]]等人號召領頭，再度發動社員募集，籌建「文昌廟」，並在文昌廟中開闢「'''磺溪書院'''」，於[[光緒|光緒]]16年（1890年）竣工，廣招[[大肚下堡|大肚下堡]]地區（今[[大肚區|大肚區]]、[[龍井區|龍井區]]、[[烏日區|烏日區]]等臺中市市轄區）的學子，作育英才<ref>[http://www.mi-great.com.tw/dadu/ 磺溪書院].台中市大肚鄉公所</ref>。

[[台灣日治時期|日治時代]]雖然磺溪書院被迫停止了教育，不過[[日軍|日軍]]的近衛師團曾經在此進駐，所以都還維持著不錯的狀態。昭和十年（1935年）12月5日，以「大肚舍營所」的名義與其他的「北白川宮能久親王御遺跡」一起被[[臺灣總督府|臺灣總督府]]指定為[[臺灣史蹟名勝天然紀念物|史蹟]]{{r|林|page=220}}{{r|吳|page=28}}。戰後[[台灣|台灣]]義務教育普及，大肚國小曾暫借書院，作為鄰近村落學子上課的教室，前[[國立清華大學|國立清華大學]]校長[[陳文村|陳文村]]即曾在磺溪書院接受小學一、二年級啟蒙教育。但是戰後，磺溪書院數度遭竊與變賣其廟產，在1959年時更遭逢[[八七水災|八七水災]]，屋宇與門牆都遭到嚴重的破壞。直到1984年[[大肚區|大肚鄉]]公所聘請著名建築學者研究調查，1985年經由[[中央政府|中央政府]]與[[地方政府|地方政府]]出資進行修復工程，並且名列第三級古蹟。<ref>[http://content1.edu.tw/publish/helen7179/material/563645/7179/bookweb/repair-2.htm 磺溪書院整建修復] {{webarchive|url=https://web.archive.org/web/20071004221429/http://content1.edu.tw/publish/helen7179/material/563645/7179/bookweb/repair-2.htm |date=2007-10-04 }}</ref>

[[民國八十八年|民國八十八年]]（1999年）9月21日的[[921地震|集集大地震]]之後，原[[臺中縣|臺中縣]]（今臺中市）共有14古蹟遭到損壞，磺溪書院也曾經被外觀判定為「有立即性危險」的建物，不過所幸受損情況並不嚴重，並在[[民國九十一年|民國九十一年]]（2002年）底修復完成。

== 近代沿革與影響 ==
[[台灣日治時期|台灣日治時期]]雖然日本大力的在臺灣推行日本教育，但是剛開始書院的傳統教育並未被中止，但是當時滿清棄臺割讓，即使在書院中學到傳統的教育，也無法與當時一般的社會接軌。所以後來基過當地人向日本政府申請，於1899年（[[明治|明治]]32年）3月16日獲准成立「'''大肚[[公學校|公學校]]'''」（今[[臺中市大肚區大肚國民小學|臺中市大肚區大肚國民小學]]），首任校長是[[日本人|日本人]][[木村左市郎|木村左市郎]]，學校成立之初即以磺溪書院充當校舍。之後在1901年經過當時文昌祠的爐主趙壁等三十多人，將廟後的的十多[[甲_(單位)|甲]]土地捐贈給學校，1909年大肚公學校就在磺溪書院後方建立了新校舍。<ref name="hs03">[http://www.ttes.tc.edu.tw/ddes/ 臺中市大肚國民小學網站] 中校史沿革的介紹說明。</ref>

1938年之後大肚公學校遷址到現址，書院捐贈的土地被他人非法侵佔、變賣，導致磺溪書院只剩數千坪土地<ref name="hs03"/>。書院甚至在1945年之前，[[美國|美國]]盟機曾經空襲過[[臺灣|臺灣]]，書院的三川殿跟廂房都曾被炸彈波及。在戰後，許多書院的資產不是被外移就是被侵占，收入來源也完全中斷，[[台灣省戒嚴令|戒嚴時期]]政府與民間也無人願意妥善管理。另外在1959年發生[[八七水災|八七水災]]，[[大肚溪|大肚溪]]的溪水氾濫，夾雜大量泥沙重創磺溪書院。而當時負責管理的趙維孝等人也遠走他鄉，任由書院的建材與文物資產被偷，加速了書院的荒廢。

1984年（[[民國73年|民國73年]]）時任[[大肚區|大肚鄉]]鄉長林振祥、陳榮懷以及大肚國小的校長雒繼光等人，感到磺溪書院傳承著重要文化的資產，於是與聘請著名的[[建築學|建築學者]][[漢寶德|漢寶德]]教授勘察研究，於1985年（[[民國74年|民國74年]]）8月19日經[[臺灣省政府|臺灣省政府]]正式指定磺溪書院文昌祠為[[台灣古蹟列表#三級古蹟|國家三級古蹟]]，並動工執行修復工程。<ref name="hs03"/>

在修復工程之前，已經有大量的產物都遭竊；甚至在修復中，書院之中原本有兩個古匾「雅歌孝友」與「經天緯地」，是在光緒年間由[[御前侍衛|御前侍衛]][[廖錦華|廖錦華]]所立，其中「雅歌孝友」也讓竊賊藉由修復工程所搭起的[[鷹架|鷹架]]之便偷走了。不過雖然飽經各種事件的阻擾，也在[[行政院文化建設委員會|行政院文化建設委員會]]、[[內政部民政司|內政部民政司]]、[[臺灣省政府民政廳|臺灣省政府民政廳]]以及[[台中縣政府|台中縣政府]]等單位的協助，以及漢光建築師事務所設計監造、勝記營造公司負責施工後，於1989年（[[民國78年|民國78年]]）3月2日正式竣工，總修復費用約[[新台幣|新台幣]]一千四百九十多萬元。<ref name="hs03"/>

自從西雝社成立以來，經由文昌祠的改建一直到民國政府修復為止，磺溪書院經歷了各種不同的天災與人禍摧殘，但是因為書院的成立，帶給了[[臺中市|臺中市]]、[[彰化縣|彰化縣]]一帶的文化教育有很深的影響。

== 結構 ==
磺溪書院內祭拜有五尊神明[[文昌帝君|文昌帝君]]、[[孚佑帝君|孚佑帝君]]、[[文衡帝君|文衡帝君]]、[[朱衣星君|朱衣星君]]與[[魁斗星君|魁斗星君]]，合稱為「[[五文昌|五文昌]]」，是古時[[科舉|科舉時代]]掌管所有考試的[[神明|神明]]。內設有大殿與拜殿，所以俗稱文昌祠或文昌廟，建築的樣式屬於傳統的閩南式合院型態，是一間結合學堂與廟宇的[[四合院|四合院]]。

建築材料的使用除了紅磚瓦片之外，還有泉州白（[[花崗岩|花崗岩]]）、青石等石材搭配福杉等建造而成。門廳比地面高出三階，講堂則高出五階。在講堂之前有個高約60公分的小「月臺」<ref>這裡指的月臺並非一般說明中的車站月台，而是指一般孔廟中在祭祀中獻八佾舞的平臺，也稱為「丹樨」，[http://www.gio.gov.tw/info/festival_c/teacher/dachen.htm 大成殿] {{webarchive|url=https://web.archive.org/web/20070209193915/http://www.gio.gov.tw/info/festival_c/teacher/dachen.htm |date=2007-02-09 }}。</ref>，月臺之上築有拜亭，這是磺溪書院的特色造型。<ref>[https://market.cloud.edu.tw/content/local/taichun/mingdau/school/a12.htm 磺溪書院]中建築說明。</ref>

磺溪書院整個建築相當考究，細部裝飾到特殊的屋頂型態都是很華麗的結構。平面的設計呈現出一般民宅與孔廟之間的結合，立面上的構造也是在閩南式建築之中相當罕見。<ref name="hs02"/>
