'''學園前站'''（{{jpn|j=学園前駅|hg=がくえんまええき|rm=Gakuemmae eki|f=y}}）是一個位於[[千葉縣|千葉縣]][[千葉市|千葉市]][[綠區_(千葉市)|綠區]]{{link-ja|生實野|おゆみ野|生實野中央}}一丁目，屬於[[京成電鐵|京成電鐵]][[千原線|千原線]]的[[鐵路車站|鐵路車站]]。車站編號是'''KS63'''。

本站位於[[新市鎮|新市鎮]]{{link-ja|生實野|おゆみ野}}地區的西側。

==年表==
* 1995年（[[平成|平成]]7年）4月1日 - 車站啟用，當時車站由千葉急行電鐵經營<ref name="chibanippo199541"/>。
* 1998年（平成10年）10月1日 - 千原線路線經營權轉交京成電鐵，車站改由京成電鐵經營。

==車站構造==
本站[[相對式月台|相對式月台]]2面2線[[地面車站|地面車站]]，站房為[[跨站式站房|跨站式站房]]。大森台方向有拖上線，月台後已保留延伸空間。

== 站名由來 ==
[[明治大學|明治大學]]曾有在此地設立校區的計畫，因此站名取為「學園前」。該計畫現已終止<ref>{{Cite conference |url = http://asp.db-search.com/chiba-c/dsweb.cgi/documentframe!1!guest04!!26320!1!1!10,-1,10!3778!217842!10,-1,10!3778!217842!13,12,11!26!47!9204!43!23?Template=DocAllFrame |date = 1995-12-12 |title = 千葉市議会本会議 |conference = 平成７年第４回定例会（第６日目） |quote = 緑区おゆみ野地区への明治大学の進出の件についてお尋ねいたします }}</ref><ref>{{Cite conference |url = http://asp.db-search.com/chiba-c/dsweb.cgi/documentframe!1!guest04!!26320!1!1!10,-1,10!3778!217842!10,-1,10!3778!217842!13,12,11!26!47!16404!4!14?Template=DocAllFrame |date = 1999-12-10 |title = 千葉市議会本会議 |conference = 平成11年第４回定例会（第６日目） |quote = 明治大学が新学部の本市への進出を断念したことにより，長年に及ぶ地元の熱い思いが断ち切れてしまい，落胆は非常に大きく，まことに遺憾と言わざるを得ません }}</ref>。附近雖有千葉明德短期大學、千葉明德高等學校等，但與站名無關。
