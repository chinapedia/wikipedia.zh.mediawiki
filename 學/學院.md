'''学院'''，又可譯為'''書院'''，是[[学校|学校]]的意思。它的定義廣泛，可以指一所[[中學|中學]]、[[大學|大學]]、[[社區學院|社區學院]]或職業院校等，視乎不同地區而定。

== 臺灣 ==
在[[臺灣|臺灣]]，獨立學院提供單科[[大學|大學]]教育，如「管理學院」、「技術學院」、或現已升格為大學的[[臺北醫學院|臺北醫學院]]與[[高雄醫學院|高雄醫學院]]。其學制與大學相同，[[本科|本科]]畢業頒授[[學士|學士]][[學位|學位]]。

== 中国大陆 ==
在[[中国大陆|中国大陆]]，学院的全日制在校生规模应在5000人以上。在人文学科(哲学、文学、历史学)、社会学科(经济学、法学、教育学)、理学、工学、农学、医学、管理学等学科门类中，称为学院的应拥有1个以上学科门类作为主要学科。其主要学科门类中应能覆盖该学科门类3个以上的专业。

称为学院的在建校初期专任教师总数不少于280人。专任教师中具有研究生学历的教师数占专任教师总数的比例应不低于30%，具有副高级专业技术职务以上的专任教师人数一般应不低于专任教师总数的30%，其中具有正教授职务的专任教师应不少于10人。各门公共必修课程和专业基础必修课程 ，至少应当分别配备具有副高级专业技术职务以上的专任教师2人;各门专业必修课程，至少应当分别配备具有副高级专业技术职务以上的专任教师1人;每个专业至少配备具有正高级专业技术职务的专任教师1人。<ref>{{Cite web|url=http://www.moe.gov.cn/jyb_zwfw/zwfw_fwxx/zhfu_gk/201511/t20151104_217627.html|title=2016年高考报名季 常见五大名词解释 - 中华人民共和国教育部政府门户网站|accessdate=2017-05-11|last=李娜|work=www.moe.gov.cn}}</ref>

== 香港 ==
在[[香港專上教育|香港專上教育]]的領域，「學院」一般指提供專上教育課程而未有大學資格的高等院校，所辦的多為[[高級文憑|高級文憑]]及[[副學士|副學士]]課程，部份學院亦提供[[學士|學士]]或以上程度之[[學位|學位]]課程。香港對機構名稱使用「大學」，及對應英文“University”設有限制，除了取得立法及行政機關同意，或海外大學在香港設立的課程外，任何公司機構及學校均不能在名稱包含「大學」及英文“University”的字詞。由於「學院」稱呼在香港社會有被認為較「大學」次一等的觀感，因此除了[[香港大學|香港大學]]、[[香港中文大學|香港中文大學]]、[[香港科技大學|香港科技大學]]，這三所成立時已為大學的院校外，香港其他大專院校都一直爭取獲准正名為大學，香港教育學院爭取升格為大學期間，校長張仁良承認[[香港教育學院|香港教育學院]]學生因為「學院」名稱產生自卑感，因此期望升格為「香港教育大學」可解決此問題<ref>[http://std.stheadline.com/yesterday/edu/0928go01.html 教院「正名」後發 展 藍圖推「兩條腿」走路]，星島日報，2016-09-28</ref>。

[[香港中文大學|香港中文大學]]是採用書院制，學院（書院）可以指其成員書院，如[[崇基學院|崇基學院]]、[[新亞書院|新亞書院]]等，主要為本科生提供住宿服務及通識教育課程。對於[[香港大學|香港大學]]而言，學院可以指{{tsl|en|Residential college|住宿書院}}，如{{tsl|en|St. John's College, University of Hong Kong|香港大學聖約翰學院}}。此外，「學院」也常與「書院」及「學校」二詞通用。

== 马来西亚 ==
在[[马来西亚|马来西亚]]，多作为提供专科课程（Diploma）教育机构名称的一部份，当中大多为私立的教育机构。

== 日本 ==
在日本，多作為[[高中|高中]]學校及[[大學|大學]]名稱的一部份，當中大多為私立的教育機構。

== 美国 ==
在[[美国|美国]]，学院除了與大学（University）一词通用外，也泛指提供[[高等教育|高等教育]]課程的院校，如[[文理学院|文理学院]]、[[社區學院|社區學院]]。「學院」一詞也見於美國的[[专业技术协会|專業團體]]。

== 加拿大 ==
在[[加拿大|加拿大]]，学院一般指的是[[社區學院|社區學院]]，或提供[[職業|職業]]性、專業性課程的院校，與大学不同。它也可指中學、[[書院聯邦制|書院聯邦制]]大學的成員、[[专业技术协会|專業團體]]等。有些可授予學士或以上程度學位的高等院校（如藝術、軍事類院校），因為各種因素，沒有在名稱中使用「大學」一詞，如{{tsl|en|Royal Military College of Canada|加拿大皇家軍事學院}}。在[[魁北克|魁北克]]，學院指的是「{{tsl|en|CEGEP|普通及職業教育學院}}」（[[法語|法語]]：Collège d'enseignement général et professionnel，CÉGEP），該等公立院校（或具有同等性質、但非CÉGEP的私立院校）大多提供大學預備課程及技術課程，其招生對象為[[魁北克|魁北克]]學制下的中學畢業生（中學的第五級），CÉGEP一共有三個年級。

== 澳大利亚 ==
在[[澳大利亚|澳大利亚]]，「學院」（college）無明確定義，可指[[高等教育|高等教育]]院校（規模較大學小，有時為大學的一部份）、[[技术与继续教育学院|TAFE]]院校、大學的成員機構（如{{tsl|en|Residential college|住宿書院}}及{{tsl|en|University college|大學學院}}）、[[专业技术协会|專業團體]]、中學（視地區而定，詳見下文）等。

*在[[塔斯馬尼亞州|塔斯馬尼亞州]]及[[澳大利亞首都特區|澳大利亞首都特區]]，「學院」指是第11班及12班的課程，或是專門提供這類課程的大學預校（Matriculation College）。
*在[[維多利亞州|維多利亞州]]，大部份提供中學課程的公立學校都稱為高校（High School），但某些則稱為中等學校（Secondary College）。
*在[[西澳大利亞州|西澳大利亞州]]、[[南澳大利亞州|南澳大利亞州]]和[[北領地|北領地]]，所有自1990年代後期建立的州高校（State High School），或是一些更早建立的州高校，都會使用或改用「學院」一詞。但高校（High School）一詞仍用來指州高校，或當地官方所指，「有中學生的學校」（schools with secondary students）。不少私立、獨立學校，包括許多提供幼稚園到第7班（[[西澳大利亞州|西澳大利亞州]]、[[南澳大利亞州|南澳大利亞州]]及[[昆士兰州|昆士兰州]]的小學7年級，和澳洲其他地區的中學1年級）課程之學校，都會稱為「學院」。
*在[[新南威爾士州|新南威爾士州]]，一些高校，特別是一些因合併而擁有多個校園的高校，被稱為中等學校。
*在[[昆士兰州|昆士兰州]]，「學院」一詞用於某些私立中學，州學校（State College）則用於一些較近期建立、並會提供小學及中學課程的學校。對於只提供中學課程的學校，則稱為州高校。
