2014年6月宣布改編成電視動畫<ref>{{cite web|url=http://www.animenewsnetwork.com/news/2014-06-21/gakko-gurashi-survival-manga-gets-tv-anime/.75791|title=Gakkō Gurashi! == 內容簡介 ==
此作品以描述身為主角的女高中生在學校內與學園生活社其他成員（下述）一同於在學校內進行社團活動的快樂日常生活來展開。作品透過不同的視點來描述主角以其主觀視點進行快樂社團活動，同時也描述主角群在滿布殭屍的環境下求存的客觀情況。

== 用語 ==
; {{nihongo|它們|彼ら}}
: 受到细菌感染，已經[[喪屍|喪屍]]化的生物。不過漫畫中並沒有稱之為「喪屍」，僅稱呼為「它們」。
: 目前確定的傳染途徑是咬傷，但於 58 話中推論出傳染途徑極有可能是空氣傳染。不僅[[人|人]]，只要是[[哺乳动物|哺乳類]]都會受到感染，其他物種如[[鸟|鸟]]則未出現感染現象。依照被咬的程度不同，完全變成「它們」所花時間也不同。
: 已知擁有歸巢本能（會遵循生前的行動，因此白天住宅區較安全，晚上變成學校與商圈較安全，但也有行動未完全遵循者），不擅長爬樓梯，得打掉頭部才會死，會受光和聲音吸引，如果突然受到強音刺激會暫時失去活動能力。
: 患者能在患病的初期施打抑制病情的藥物，但仍會有體溫異常低下的副作用，且不會被「它們」視為目標，從直升機駕駛的狀況推測藥物需要接續施打才能維持效果。在过往也曾在巡之谷爆发，因此推断此病能被治癒，但细菌有变异的可能性。
: 動畫第三話以慈的角度說明了喪屍爆發的情形：當天上午自街上開始擴散，起先造成的傷亡被媒體視為大規模打鬥事件；情況嚴重到造成交通癱瘓、慈收到母親的緊急訊息時已差不多是下午放學時刻。此時因為喪屍入侵校園，由紀等人因而被困在[[天台_(建筑物)|天台]]。

; {{nihongo|巡之丘市|巡ヶ丘市}}
: 故事發生的舞台。舊名為「男土」，原是海運興起的城市。名稱由來取自[[羅馬七座山丘|羅馬七座山丘]]，原型为[[神奈川县|神奈川县]][[横滨市|横滨市]]。

; {{nihongo|巡之丘學園高等學校|巡ヶ丘学院高等学校}}
: 由紀等人所就讀的學校，校社為地上三層地下兩層的建築。樓頂設有[[太陽能發電|太陽能發電]]板與園藝社的菜園，還配有優良的淨水設備，因此災難爆發後由紀等人得以暫時有水電與食物運用，地下二樓設有避難所，似乎有為對應這次事件設有儲物間及物資，以及抑制病情的藥物。事件爆發後慈領導學園生活社在二樓階梯以課桌設置路障，確保三樓完整安全的生活空間。

=== 與原作的差異 ===
动画的剧本仍由漫画原作海法紀光负责，同时另有多名电子游戏编剧和小说家参与其中<ref>{{cite web |title=スタッフコメントを公開！ |url=http://gakkougurashi.com/news/index00110000.html |publisher=“学园孤岛”电视动画官方网站 |accessdate=2015-08-08 |language=ja }}</ref>。
* 美紀在動畫開始便已加入學園生活社，從購物商場被營救逃出的段落則改為以回憶的方式呈現。孤身一人時每日以在學時正常生活作息來保持理智的段落亦被刪去。
* 太郎丸在原作中生前並沒有和美紀接觸過，名字也是由紀擅自取的。動畫修改成與美紀在購物商場相遇，並從原主人得知名字，主人後來變成「它們」而收養牠，並引導來遠足的學園生活社救援美紀。後來疑似發覺有同伴擅自跑到地下避難所被慈姐攻擊而感染，也變成「它們」，雖然最後被注射藥物一度恢復意識，但仍因感染太久過於虛弱而死亡。
* 動畫中為了強調圭「想出去」的意境，將原先與其他生還者短暫生活的情節刪除，和美紀及太郎丸成為商場的倖存者。在避難所中待了數天後，從堅決離開的絕望改編成出去尋求救援的希望。
* 漫畫中火燒校園和全體聚集避難所的情節濃縮成為動畫版的雨天最後一戰。

与中文标题不同，本作日语标题直译意为“学园生活”，并无“孤岛”等字样对作品类型作出暗示。不仅官方网站在正式开播前包装为常见的[[空气系|日常系]]动画，没有体现本作的恐怖要素<ref>{{lang|ja|[http://natalie.mu/comic/news/153474 「がっこうぐらし！」BD＆DVD1巻9月に発売、キービジュアルは荒廃した絵に] - コミックナタリー、2015年7月10日配信}} {{ja}}</ref>，日本各大动画新闻站在报道第一话先行上映会时也只介绍了故事中“可爱”和“愉快”的一面<ref>[http://animeanime.jp/article/2015/07/05/23990.html {{lang|ja|「がっこうぐらし！」水瀬いのり、M・A・OがOP秘話も明かす　先行上映会は大盛況 | アニメ！アニメ！}}] {{ja}}</ref><ref>[http://www.animate.tv/news/details.php?id=1436244442 {{lang|ja|水瀬いのりさんら登壇『がっこうぐらし！』先行上映会レポ到着 - アニメイトTV}}] {{ja}}</ref><ref>[http://www.ota-suke.jp/news/145396 {{lang|ja|水瀬いのり＆M・A・O登壇「がっこうぐらし!」先行上映会のレポ到着 :おた☆スケ【声優情報サイト】}}] {{ja}}</ref>。因此，在第一话冲击性的结尾播出后，日本互联网上随即出现大量讨论，许多用户将其比作同样有[[nitro+|nitro+]]参与制作的《[[魔法少女小圆|魔法少女小圆]]》{{noteTag|第三话中巴麻美学姐死亡，改变作品整体基调。}}，直呼本作太吓人<ref>{{cite web|url=http://otapol.jp/2015/07/post-3350.html|script-title=ja:「騙された」「まどマギ以上に鬱」…アニメ『がっこうぐらし！』の衝撃展開にネット騒然|publisher={{lang|ja|{{tsl|ja|おたぽる}}}}|date=2015-07-15|accessdate=2015-07-18|language=ja}}</ref>。[[niconico動画|niconico動画]]上第一话也在一星期内获超过146万次播放<ref>{{cite web|url=http://news.mynavi.jp/news/2015/07/17/639/|script-title=ja:『がっこうぐらし！』7/20秋葉原で特製うちわ配布＆"学校"居酒屋とコラボ|publisher={{lang|ja|{{tsl|ja|マイナビニュース}}}}|accessdate=2015-07-18|date=2015-07-17|language=ja}}</ref>，原作第一卷销售额更达到动画开播前的约十倍<ref>{{cite web|url=http://hon-hikidashi.jp/enjoy/2968/|script-title=ja:アニメ化影響で『がっこうぐらし！』の売上前週比がなんと10倍！ニコニコ動画再生数100万超えの大ブレイク夏アニメ|publisher={{lang|ja|ほんのひきだし}}|date=2015-07-20|accessdate=2015-07-20|language=ja}}</ref>。

后续集数陆续播出后，互联网上出现诸如“看了第二话和第三话后再看第一话，看出[[伏笔|伏笔]]来了”一类的声音，称赞本作的[[叙述性诡计|叙事技巧]]<ref name="CrankIn">{{cite web|url=http://www.crank-in.net/game_animation/special/38374/2|script-title=ja:『まどマギ』の衝撃再び!?　“萌え”に隠された驚きの展開『がっこうぐらし』の魅力|publisher={{lang|ja|クランクイン!}}|author=阿部美香|accessdate=2015-08-21|date=2015年8月15日|language=ja|deadurl=yes|archiveurl=https://web.archive.org/web/20150818012345/http://www.crank-in.net/game_animation/special/38374/2|archivedate=2015年8月18日}}</ref>。
