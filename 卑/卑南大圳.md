'''卑南大圳'''，是[[臺東縣|臺東縣]]最大的[[水利工程|水利工程]]<ref name="卑南大圳水利公園風情萬千"/><ref name="台灣的古圳道"/><ref name="卑南大圳水利公園"/>，位於臺東縣[[卑南鄉|卑南鄉]]與[[臺東市|臺東市]]，最早興建於1898年（明治31年），從岩灣引台東第一大溪[[卑南溪|卑南溪]]溪水[[灌溉|灌溉]]流灌整個台東平原[[農田|農田]]，全長9公里餘，圳道呈「排梳型」的分布結構，是一條水利兼具防洪功能的灌溉溝渠<ref name="台灣的古圳道">{{cite book|author=王萬邦|title=台灣的古圳道|year=2003年4月|isbn=9572856103 }}</ref><ref name="卑南大圳水利公園">{{cite web|language=zh-tw|url=http://www.ttia.gov.tw/water_page_benan_02.php|title=卑南大圳水利公園 |author=|publisher=台東農田水利會|date=|accessdate=2014-05-29}}</ref>。

== 沿革 ==
[[台灣清領時期|台灣清領時期]]初期，[[臺灣東部|臺灣東部]]嚴禁[[漢人|漢人]]進入，直到同治13年（1874年）[[沈葆禎|沈葆禎]]奏請[[開山撫番|開山撫番]]，光緒元年（1875年）廢止禁令，才鼓勵漢人入山開墾，也才開始有小型灌溉事業<ref name="張炎銘">{{cite web|language=zh-tw|url=http://epaper.wra.gov.tw/Article_Detail.aspx?s=53EF8FB32037D07F|title=1936年3月－卑南大圳完工及內埔子、蘭潭、龍鑾潭、榮湖、瓊林、東湧等水庫竣工 |author=張炎銘|publisher=水利署電子報|date=|accessdate=2014-05-29}}</ref>。

卑南大圳最早建於明治31年（1898年），因當時於臺東市與卑南鄉的農民為增加生產、改善生活，請設卑南圳，由政府補助工程費1,300圓，民眾出勞力，拓建卑南圳，因經費不足，圳路只能灌溉約668甲土地<ref name="張炎銘"/>。1933年，日本政府為謀臺東農業發展，由[[台灣總督府|台灣總督府]]撥款88萬餘圓補助地方政府負責修建卑南圳，於1933年10月5日開工，原擬2年完工，但因1934、1935年2次颱洪災害，延至1936年3月才完成，經費增為97萬餘圓，灌溉面積達2,440甲<ref name="張炎銘"/>。1936年（昭和11年）日本人為求軍糧補給而再次整修，於1941年（昭和16年）竣工<ref name="台灣的古圳道"/>。卑南大圳工程完工後並且使當地單期作成為兩期作<ref name="張炎銘"/>。水圳規模擴大後，吸引大批漢人移入，建立聚落<ref name="台灣的古圳道"/>。

== 分布 ==
卑南大圳經過今日的[[台東新站|台東新站]]、永樂里（[[豐年機場|豐年機場]]附近），到[[利嘉溪|利嘉溪]]北岸豐源堤防，全長9公里餘，共有13條平行的支線<ref name="卑南大圳水利公園"/>。
