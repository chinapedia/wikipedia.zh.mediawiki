'''卑南上圳'''為一座位於[[臺灣|臺灣]][[臺東縣|臺東縣]][[卑南鄉|卑南鄉]]境內的[[灌溉|灌溉]]水圳系統，因该鄉另有一座灌溉水圳系統名為「[[卑南大圳|卑南大圳]]」，為區隔兩者而得名「卑南上圳」，水圳目前由[[臺東縣|臺東]][[農田水利會|農田水利會]][[卑南鄉|卑南]][[工作站|工作站]]進行維護與管理。

== 沿革 ==
=== 起源 ===
卑南上圳工程最初始於[[日治時代|日治時代]]末期當時[[日本|日本]][[台灣總督府|台灣總督府]]計畫於台東紅葉清水地區建設水壩與[[水力發電廠|水力發電廠]]進行發電的計畫，該計畫代號為'''清水計畫'''<ref name="國立台灣大學 館藏目錄 卑南上圳25週年回顧 / 賴亮郡編著" />，然而[[第二次世界大戰|第二次世界大戰]]後期[[美國|美國]]向還在日本殖民政府統治下的臺灣進行轟炸導致物資缺乏，使得日本殖民政府在離開臺灣前，清水計劃始終沒有實際執行。直到[[二戰|二戰]]後因為[[卑南鄉|卑南]][[平原|平原]]一帶之農地地勢較高，無灌溉設施。臺灣省政府為平衡東部地區發展，再加上當時臺灣社會處於與[[中國大陸|中國大陸]]的緊張關係之下，因此積極將臺灣各地之荒地改造成[[稻田|稻田]]，以做為儲備糧倉。<ref name="國立台灣大學 館藏目錄 卑南上圳25週年回顧 / 賴亮郡編著" />

=== 規劃 ===

省府委員[[南志信|南志信]]曾率領地方人士極力爭取卑南上圳<ref name="林志興">{{cite journal |author=林志興|chapter=〈南志信〉|journal=《原住民文獻》|issue=2014年12月第十八期|date=2014-12|publisher=原住民委員會|location=台灣|language=zh-tw}}</ref>。在時任總統的[[蔣中正|蔣中正]]來到該地區訪查後，指示清水地區應該建設卑南上圳工程<ref name="國立台灣大學 館藏目錄 卑南上圳25週年回顧 / 賴亮郡編著" />。

因此，1948年（民國37年）起先後省政府水利局、[[台灣電力公司|台灣電力公司]]、[[水資源局|水資會]]等機關多次調查規劃。在1970年（民國61年）11月成立第四規畫調查隊，由隊長，當時台灣省政府[[第八工程處|第八工程處]]的[[胡江東|胡江東]]與[[劉定遠|劉定遠]]隨同部分第八工程處人員以鹿野溪下游羅試地區興建進水口攔水壩為前提下進行調查工作，在1971年（民國62年）6月提出第一期的工作報告，報告指出，在經過鹿野溪羅試地區鑽探後，該區[[礫石|礫石]]堆積層太厚，非常不利於建設水壩，應重新尋找新的地點建築卑南上圳所需的攔水壩。<ref name="國立台灣大學 館藏目錄 卑南上圳25週年回顧 / 賴亮郡編著" />

1971年11月，經由省政府水利局探勘隊探勘後，選定在原羅試[[水壩|水壩]]預定地下游約0.8公里處的上里地區，該區鹿野溪的兩岸硬岩裸露，為適合興建水壩與卑南上圳進水口的極佳地點，並以這個地點進行全面調查。最終在1973年（民國64年）完成'''卑南上圳灌溉工程計畫報告書'''。<ref name="國立台灣大學 館藏目錄 卑南上圳25週年回顧 / 賴亮郡編著" />

=== 興建 ===

卑南上圳工程於1977年（民國66年）由當時臺灣省政府撥款10億元台幣正式動工。<ref name="國立台灣大學 館藏目錄 卑南上圳25週年回顧 / 賴亮郡編著" />

工程項目包括[[攔河堰|攔河堰]]暨進水口各一座，[[延平鄉|延平鄉]]與卑南鄉之間的山區，興建卑南上圳導水[[隧道|隧道]]4,086公尺、幹線12,770公尺、支線四條7,270公尺、灌溉幹管三條16,100公尺及其他灌排設施。其中4.08公里的導水隧道形式為2R（R=1.17公尺）的標準馬蹄鐵型，並且，該隧道是目前全台最長的山-{岳}-中型導水隧道。導水隧道開鑿時，因地形所限制，故隧道全線開鑿時，無法藉由設置縱坑或是橫坑來加快隧道挖掘，僅能以上下游頭尾挖掘方式開鑿。<ref name="國立台灣大學 館藏目錄 卑南上圳25週年回顧 / 賴亮郡編著" />

在卑南上圳長達10年的工期中開鑿導水隧道的工程當中一名技工胡仙吉不幸觸電身亡，另外隧道崩坍事件造成四名坑工殉職，總工程一共有五位工程人員不幸殉職。<ref name="人間通訊社 台東卑南上圳完工通水25週年回顧暨表揚有功人員慶典活動">{{cite web |url=http://www.lnanews.com/news/%E5%8F%B0%E6%9D%B1%E5%8D%91%E5%8D%97%E4%B8%8A%E5%9C%B3%E5%AE%8C%E5%B7%A5%E9%80%9A%E6%B0%B425%E9%80%B1%E5%B9%B4%E5%9B%9E%E9%A1%A7%E6%9A%A8%E8%A1%A8%E6%8F%9A%E6%9C%89%E5%8A%9F%E4%BA%BA%E5%93%A1%E6%85%B6%E5%85%B8%E6%B4%BB%E5%8B%95.html |title=人間通訊社 台東卑南上圳完工通水25週年回顧暨表揚有功人員慶典活動| publisher=[[人間通訊社|人間通訊社]]|language= zh-hant | accessdate=2012-09-10}}</ref><ref name="國立台灣大學 館藏目錄 卑南上圳25週年回顧 / 賴亮郡編著" />

==== 崩塌事件 ====

卑南上圳的導水隧道開挖過程中發生及多次崩塌事件其中以1983年（民國72年）3月15日發生之崩塌最為嚴重。<ref name="國立台灣大學 館藏目錄 卑南上圳25週年回顧 / 賴亮郡編著" />

1983年（民國72年）3月15日下午04點40分，在導水隧道第五期工程開挖時，隧道上段樁號1K435處（1.435公里處），上方斷層帶突然發生嚴重天然坍方。石屑，塊石夾帶大量泥沙從崩塌處衝到樁號1K280處（1.280公里處），而泥漿更流到樁號0K980處（0.980公里處），等同於坍方的壓力將土石往已開鑿的隧道向外推擠了455公尺。<ref name="國立台灣大學 館藏目錄 卑南上圳25週年回顧 / 賴亮郡編著" />

導水隧道崩塌事件也造成了當時正在挖掘處施工的人員柯榮安、柯瑞木、柯清春、陳仁喜四人不幸罹難。<ref name="人間通訊社 台東卑南上圳完工通水25週年回顧暨表揚有功人員慶典活動" /><ref name="國立台灣大學 館藏目錄 卑南上圳25週年回顧 / 賴亮郡編著" />

=== 完工 ===
長達4公里的導水隧道最終在1985年（民國74年）4月25日全線貫通，整體工程在1986年（民國75年）5月31日完工，並於1986年（民國75年）10月15日正式通水撥交給台東農田水利會接管，同時，台東農田水利會成立卑南工作站，以辦理卑南上圳灌溉管理營運之相關業務。<ref name="國立台灣大學 館藏目錄 卑南上圳25週年回顧 / 賴亮郡編著" />

== 改善工程 ==
* 卑南上圳進水口第一沉砂池排砂道天兔颱風災害修復工程

由於在2013年9月底[[天兔颱風|天兔颱風]]來襲並造成卑南上圳進水口遭砂石掩埋，農田水利會緊急進行搶修，總計耗費一個多月修復進水口，然而因[[鹿野溪|鹿野溪]]溪水混濁、含砂量高，導致[[沉砂池|沉砂池]]磨損嚴重，為解決溪水濁度高之問題，台東農田水利會宣布卑南上圳全線自2014年2月17日起至2014年4月28日將中斷供水70天以待溪水中的泥沙沉澱。<ref>{{cite web|url=http://www.ksnews.com.tw/newsdetail.php?n_id=0000482060&level2_id=107|title=更生日報 卑南上圳進水口淤砂 無法灌溉|publisher=[[更生日報|更生日報]]|language=zh-hant|accessdate=2013-10-15|deadurl=yes|archiveurl=https://web.archive.org/web/20141208181831/http://www.ksnews.com.tw/newsdetail.php?n_id=0000482060&level2_id=107|archivedate=2014-12-08}}</ref><ref>{{cite web |url=http://news.ltn.com.tw/news/local/paper/749588 |title=自由時報〈南部〉整修沉砂池 卑南上圳將斷水70天 | publisher=[[自由時報|自由時報]] |language=zh-hant | accessdate=2014-01-24}}</ref>

== 設施 ==

=== 進水口 ===
卑南上圳取水口坐落於鹿野溪上游南岸的[[延平鄉|延平鄉]][[紅葉村|紅葉村]][[上里部落|上里部落]]<ref>{{cite web |url=http://tour.taitung.gov.tw/zh-tw/Travel/ScenicSpot/235/%E5%8D%91%E5%8D%97%E4%B8%8A%E5%9C%B3 |title=台東縣政府 台東觀光網 卑南上圳 | accessdate=2014-12-06}}</ref>，設有六孔攔污柵進水孔，設計最大引水量為每秒7.206立方公尺，每孔寬度5.0公尺總長30公尺，檻頂標高212.5至212.85公尺，進水口底標高EL.212.0公尺。

制水閘門共設有兩座，長寬各兩公尺。

=== 攔河堰 ===

卑南上圳於進水口設有一座攔河堰，名為'''上里攔河堰'''，全長71公尺，壩高20.4公尺，壩頂標高213.4公尺，壩頂寬度5.8公尺，壩型為[[重力壩|重力壩]]，壩體內設置有廊道，廊道高2.2公尺，寬1公尺，水平長度35公尺。<ref>{{cite web |url=http://student.kmes.tc.edu.tw/taiculture/water/14/C/04/02/C-14-04-02-02.htm |title=灌排系統 台東農田水利會灌排系統 卑南上圳 |accessdate=2014-12-06 |deadurl=yes |archiveurl=https://web.archive.org/web/20150122180810/http://student.kmes.tc.edu.tw/taiculture/water/14/C/04/02/C-14-04-02-02.htm |archivedate=2015-01-22 }}</ref>

=== 調整池 ===

卑南上圳調整池最初在1985年（民國74年）開始規劃，1986年（民國75年）全面推動調整池計畫，並規劃設有兩座調整池，一座位於卑南鄉賓郎[[四格山|四格山]][[山腰|山腰]]處，該調整池即為現今的'''十股調整池'''。

第二座調整池原計畫設置於[[卑南鄉|卑南鄉]][[卑南國中|卑南國中]]後方與[[卑南文化公園|卑南文化公園]]之山腰處，後因當地土地徵收問題遲無法解決，因此取消第二座調整池之興建計畫。

十股調整池位於卑南鄉[[賓朗村|賓朗村]]四格山坡的高[[台地|台地]]，由卑南上圳幹線的上游段引水，經由四公里的管線送至調整池，運用自然落差蓄水，實施[[噴灌|噴灌]]。<ref>{{cite web |url=http://www.ttia.gov.tw/water_page_benan_01.php |title=台東農田水利會 卑南上圳 十股調整池 | accessdate=2014-12-06}}</ref>

台東農田水利會於該調整池周遭設有休憩公園。

=== 導水設施 ===

排砂隧道兩座，位於第一沉沙池的排砂隧道長86.5公尺，形式為為直徑2公尺的圓形隧道，位於第二沉沙池的排砂隧道長75.57公尺，其中，95.57公尺段隧道形式為直徑2公尺的圓形，其餘16公尺為箱涵形式。

導水隧道4,086公尺，斷面為2R（R=1.17公尺）的標準馬蹄鐵形式，初始計畫之通水量為每秒7.206立方公尺，流速為每秒1.8立方公尺，其中隧道進水口（上里端）標高210.15公尺，出水口（初鹿端）標高206.07公尺。

主幹線12,770公尺，並在幹線上設置有六座電動耙污機，而支線共四條，總長7,270公尺、灌溉幹管共三條總長16,100公尺及其他灌排設施若干座。<ref name="台東農田水利會 卑南工作站">{{cite web |url=http://www.ttia.gov.tw/page4_beinan.php |title=台東農田水利會 卑南工作站 | accessdate=2014-12-06}}</ref><ref>{{cite web |url=http://www.tjia.gov.tw/upload/publish/2004/20051230134325-特殊貢獻人員28-1.txt |title=農田水利會 獎勵人員 |publisher=[[農田水利聯合會|農田水利聯合會]] |language= zh-hant | accessdate=2014-12-06}}</ref>

=== 水利公園 ===
