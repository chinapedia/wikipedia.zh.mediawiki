'''字母會'''是華文書寫的重要實驗，涉及當代哲學、小說、評論與出版的跨領域基進實踐，時間長達五年以上。

2012年起由[[駱以軍|駱以軍]]、[[陳雪|陳雪]]、[[童偉格|童偉格]]、[[顏忠賢|顏忠賢]]、[[胡淑雯|胡淑雯]]<ref>{{cite web|url=http://www.biosmonthly.com/interview_topic/9768|title=下女的越位，專訪胡淑雯：寫作是將你的舌頭贖還|author=BIOS monthly|}}</ref>與[[楊凱麟|楊凱麟]] 共同參與。[[楊凱麟|楊凱麟]]每2個月從[[法文字母|法文字母]]A到Z依序選出代表當代思想的詞彙，寫成1000字的「思想影像」，5位小說家則據此啟發創作5000字的小說。字母會將當代哲學的提問交由台灣現役小說家創作出「小說回覆」，26個字母所圍繞的始終是同一個根本的問題：什麼是小說的當代性？

2013年6月，6人開始在《[[短篇小說_(雜誌)|短篇小說]]》雙月刊發表字母A：未來，一直持續進行到2015年12月《[[短篇小說_(雜誌)|短篇小說]]》停刊，該期發表的字母是 P：摺曲。<ref>[http://udn.com/news/story/7049/486685-%E6%96%87%E5%AD%B8%E7%9B%B8%E5%B0%8D%E8%AB%96%EF%BC%8F%E9%A1%8F%E5%BF%A0%E8%B3%A2VS%E9%99%B3%E9%9B%AA%E2%94%80%E9%97%9C%E6%96%BC%E7%9F%AD%E7%AF%87%E5%B0%8F%E8%AA%AA 文學相對論／顏忠賢VS陳雪─關於短篇小說] .聯合報</ref>

《短篇小說》停刊後，字母會寫作實驗仍持續進行，改交由衛城出版重新構想出版概念，二年後（2017年9月）26個字母分 4季上市。衛城出版總編輯莊瑞琳同時創辦《字母 Letter》，這是一本專門評論《字母會》寫作計畫與當代文學書寫的評論雜誌。

== 《字母會》與華文書寫的當代性 ==
《字母會》是一個「完全作品」的文學基進實驗，從概念、創作、出版、評論到書籍設計，甚至到專屬評論雜誌、各地書台的講座、讀書會與朗讀會。由台灣一線的青壯小說家、評論者、出版者與封面設計者攜手創造的跨領域當代文學實驗。

這是繼六○年代的「[[現代主義|現代主義]]」（[[白先勇|白先勇]]、[[王文興|王文興]]、[[陳映真|陳映真]]、[[七等生|七等生]]、[[黃春明|黃春明]]等）與九○年代的「[[後現代主義|後現代主義]]」（[[張大春|張大春]]、[[林燿德|林燿德]]等），戰後台灣文學與西方思潮的第三次相遇。

[[楊凱麟|楊凱麟]]指出：「我們重新試著進入當代思想的基底，對外部、陌異、非思、不可見、差異與虛構等當代哲學問題提出小說式的回應，試著再搖晃池子，製造喧囂與推倒圍牆，但卻是為了能再次感受位居當代思想核心的恐怖沈默與異語。」<ref>〈在字母的前沿......〉.《字母 Letter》，2017 sep. vol.1</ref>

== 《字母會》的參與者 ==
2017年9月，《字母會》由衛城出版公司分 4季出版，分別由[[王志弘|王志弘]]、[[何佳興|何佳興]]、[[林小乙|林小乙]]、[[聶永真|聶永真]]負責書籍設計。這4位設計師亦是台灣目前僅有的4位 {{link-en|AGI|Alliance Graphique Internationale}}（Alliance Graphique Internationale）會員。

衛城出版公司同時針對《字母會》 6位小說家（[[駱以軍|駱以軍]]、[[陳雪|陳雪]]、[[顏忠賢|顏忠賢]]、[[童偉格|童偉格]]、[[胡淑雯|胡淑雯]]、[[黃崇凱|黃崇凱]]）發行文學評論雜誌《字母 Letter》。<br>

== 文學評論刊物《字母 Letter》 ==
《字母 Letter》是一本圍繞著字母會創作觀念的文藝評論刊物，每兩個月以「封面人物」專輯，逐一深度評論字母會六位核心小說家：[[駱以軍|駱以軍]]、[[陳雪|陳雪]]、[[顏忠賢|顏忠賢]]、[[童偉格|童偉格]]、[[胡淑雯|胡淑雯]]、[[黃崇凱|黃崇凱]]。

衛城出版總編輯莊瑞琳在發刊詞中堅定地指出，「小說的巔峰仍舊是觀測自由時代的最佳指標......我們相信，文學的造山運動並未結束。」<ref>〈文學，成為一個自由的人——《字母 Letter》發刊詞〉，《字母 Letter》2017 Sep., Vol.1, 3</ref>
