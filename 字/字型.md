'''字型'''或'''字模'''（{{lang-en|font}}；传统英式{{lang-en|fount}}）是指[[印刷|印刷]]行業中某一整套具有同樣[[字体|樣式]]和尺碼的[[字形|字形]]，例如一整套用於內文的[[宋體|宋體]]5號字、一整套用於標題的10號字就叫一套字型。電腦早期用[[點陣字|點陣字]]，仍然有字型概念，同樣一套風格如[[中易宋體|中易宋體]]，一套字型是指一整套15×16[[像素|點數]]或一整套24×24點數的字。[[矢量圖形|向量字型]]出現後，同一套風格字型已不用製作不同點數字型，只需製作一套即可隨意縮放，「字型」與「[[字体|字体]]」之間的界限開始模楜。一般的英語使用者同樣分不清「字型」（{{lang|en|Font}}）與「字体」（{{lang|en|[[:en:Typeface|Typeface]]}}）的分別。

==字型特性==
===尺寸===
{{Main|點 (印刷)}}
字型'''尺寸'''（size）是指其高度。单位是“点”或“级”（中文也采用“号”做单位），72点相当于1英寸（1 point = 127⁄360 mm = 352.7 µm——约等于352.7微米），而12点则称为1皮卡（pica，或译“派卡”）。如：大小为12点的字可以称作“12级字”。

===字重===
[[File:Helvetica_Neue_typeface_weights.svg|thumb]] 的多种字重]]

{{anchor|粗细}}本页说明了粗细字形。下面说明了字形的粗细辨别。

'''字重'''（weight），是指相对于字高度的笔画粗细（stroke width）程度。一个字体（typeface）的某个字型（font）的字重常常至少4-6个，其中正常与黑体几乎是必备的：{{noteTag|1=以上的分類只是大致上的，並不是所有字型都按照此規則，在[[Adobe|Adobe]] Typekit中對字重劃分中，Heavy指的是800而不是900。另外，在Photoshop和Sketch裡面，Ultra Light是100，而Thin是200。}}
*100 - 淡體 Thin ( Hairline )
*200 - 特細 Extra-light ( ultra-light )
*300 - 細體 Light
*350 - 次細 Demi-Light
*400 - 標準 Regular ( normal / book / plain )
*500 - 適中 Medium
*600 - 次粗 Demi-bold / semi-bold
*700 - 粗體 Bold
*800 - 特粗 Extra-bold / extra
*900 - 濃體 Black ( Heavy )
*950 - 特濃 Extra-black ( Ultra-black )
