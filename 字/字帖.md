'''字帖'''，或'''習字帖'''，是學習寫字所用的[[臨摹|臨摹]]範本，主要分為[[毛筆|毛筆]]所用的[[書法|書法]]字帖與[[鋼筆|鋼筆]]等硬筆字帖。

== 歷史 ==
[[梁武帝|梁武帝]]蕭衍命[[殷鐵石|殷鐵石]]將「書聖」[[王羲之|王羲之]]的書法作品中拓出一千個不重複的字，供諸王臨摹，然覺字間散漫，無甚組織，又命[[周興嗣|周興嗣]]將其輯錄成有意義的文句，此即《[[千字文|千字文]]》，亦成為一套有系統的臨摹字帖。<ref>{{cite book|author=全唐文|title=徐氏法書記|pages=卷二百六十八|url=http://gj.zdic.net/archive.php?aid=16613}}</ref> 

以鐫刻文字使其持久流傳，自[[殷|殷]]以降愈盛。漢魏乃至南北朝時期，碑版極多。及至唐代，始以保存書法之美而刻石。傳唐太宗得王羲之蘭亭序後，遣臣搨摹並刻石於宮禁。唐時摹拓之法亦盛，因此一本法書可大量複製，不過因為所需費時，摹勒刻石繼之而起，此即石刻法帖之濫觴。

往後，字帖的選擇皆以書法大家作品為上，更有甚者，由宮廷選材編刻成為法帖，供士子臨摹，如《[[御刻三希堂石渠寶笈法帖|三希堂法帖]]》。

在現代出版技術出現前，學習書法不外乎取用墨蹟、金石拓本、石刻法帖，然墨蹟難得，金石易剝蝕，石刻雖近原本，卻不能絲毫無異。景印法書通行之後，對於字帖之複製與學習均有極大幫助。<ref>{{cite book|author=王壯為|title=書法研究|year=1967|publisher=臺灣商務印書館|location=臺北市|isbn=9570506180|pages=30}}</ref>

== 書法字帖 ==
學習書法，除了準備[[文房四寶|文房四寶]]外，字帖的選擇能夠決定字型的風格。如[[歐體|歐體]]字帖、[[柳體|柳體]]字帖或[[顏體|顏體]]字帖等。字帖的內容通常是將書法名家的碑刻與墨跡以拓印、[[景印|景印]]方式留存，供人學習。

字帖如以書體分類，則有[[楷書|楷書]]字帖、[[行書|行書]]字帖與[[草書|草書]]字帖等，不一而足。[[楷書|楷書]]字帖中，每一個[[漢字|漢字]]均配置在[[九宮格|九宮格]]或米字格內，並搭配[[永字八法|永字八法]]說明運筆技巧，最常見的是以描紅方式，只留字形輪廓與筆勢提示，便於學習者摹仿。知名的書法作品成為字帖出版廣為流傳，又以楷書為多。

字帖的運用方式極其多樣，最普遍的使用方法有以下幾種：

* 描紅：用毛筆在印有紅色字跡的紙上描寫。
* 臨寫：將字帖放置於旁，按照字型依樣書寫。
* 摹寫：將紙張覆蓋於字帖上，在字上書寫。

== 硬筆字帖 ==
[[File:Point_of_purchase.jpg|缩略图]]

[[鋼筆|鋼筆]]、[[原子筆|原子筆]]等硬筆也有字帖供學習參考，使個人字跡美觀，亦實用於日常。在[[設計|設計]]領域也有專屬的字帖，如[[POP字型|POP字型]]此類著重於[[麥克筆|麥克筆]]手繪字體的字帖。

== 其他字帖 ==
除了漢字書體字帖，還有一種學習[[英文字母|英文字母]]時所用的[[筆順|筆順]]字帖。此類字帖不同於漢字九宮格，是一種類似[[五線譜|五線譜]]的參考線，字體也以虛線輔以筆順表示。
