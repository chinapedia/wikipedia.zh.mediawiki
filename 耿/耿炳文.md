'''耿炳文'''（{{bd|1334年||1403年|catIdx=G耿}}），[[濠州|濠州]]（今[[安徽|安徽]][[鳳陽|鳳陽]]）人，[[中國|中國]][[明朝|明朝]]开国军事将领，官至大将军，以功封'''長興侯'''。

== 生平 ==
[[元朝|元朝]]末年，他與其父亲[[耿君用|耿君用]]追隨[[朱元璋|朱元璋]]渡過[[長江|長江]]。耿君用在1356年救援[[宜興|宜興]]時，與[[張士誠|張士誠]]交鋒，力戰而死。耿炳文承襲其父的管軍[[總管|總管]]之職，繼續[[統領|統領]]其部眾。[[至正|至正]]十七年（1357年）二月初一耿炳文從[[廣德|廣德]]攻[[長興|長興]]州，[[張士誠|張士誠]]守將[[趙打虎|趙打虎]]以三千精兵迎戰失利，逃往[[湖州|湖州]]。初三，炳文攻陷長興，獲戰船三百多艘。朱元璋改長興为长安州，在县署内设立永兴翼元帅府，炳文任[[总兵|总兵]][[都元帅|都元帅]]。炳文納儒士[[溫祥卿|溫祥卿]]於幕下，協助策劃防守。炳文固守長興十年，前後歷數十戰，多次打敗[[張士誠|張士誠]]。

朱元璋向張士誠全面進攻時，炳文帶兵攻克湖州，围[[平江|平江]]（[[苏州|苏州]]）。朱元璋建立明朝後，授镇国上将军兼右率府副使。耿炳文随大军征讨中原，攻克山东沂、峄等州，攻下[[汴梁|汴梁]]，巡视[[河南|河南]]，侍从太祖北巡。又随[[常遇春|常遇春]]攻占[[大同|大同]]，攻克晋、冀。随大将军徐达征讨[[陕西|陕西]]，打败[[李思齐|李思齐]]、张思道。洪武二年（1369年），镇守陕西，疏通泾阳洪渠十万余丈，给老百姓带来了很多好处。[[洪武|洪武]]三年（1370年）任命为秦王府左相兼陕西行省右丞，封长兴侯，[[封邑|食禄]]一千五百石<ref>《[[明史|明史]]》(卷百三十):“耿炳文，濠人。父君用，從太祖渡江，積功為管軍總管。援宜興，與張士誠兵爭柵，力戰死。炳文襲職，領其軍。取廣德，進攻長興，敗士誠將趙打虎，獲戰船三百余艘，擒其守將[[李福安|李福安]]等，遂克長興。長興據[[太湖|太湖]]口，陸通[[廣德|廣德]]，與[[宣城市|宣]]、[[歙縣|歙]]接壤，為[[江蘇|江]]、[[浙江|浙]]門戶。太祖既得其地，大喜，改為長安州，立永興翼元帥府，以炳文為總兵都元帥，守之。溫祥卿者，多智數。避亂來歸，炳文引入幕府，畫守御計甚悉。張士誠左丞[[潘元明|潘元明]]、元帥[[嚴再興|嚴再興]]帥師來爭。炳文奮擊，大敗去。久之，士誠復遣司徒[[李伯升|李伯升]]帥眾十萬，水陸進攻。城中兵七千，太祖患之，命[[陳德|陳德]]、[[華高|華高]]、[[費聚|費聚]]往援。伯升夜劫營，諸將皆潰。炳文嬰城固守，攻甚急，隨方御之，不解甲者月余。[[常遇春|常遇春]]復帥援兵至，伯升棄營遁，追斬五千余人。其明年，改永興翼元帥府為[[永興衛|永興衛]][[親軍指揮使司|親軍指揮使司]]，以炳文為使。已而士誠大發兵，遣其弟[[張士信|士信]]復來爭。炳文又敗之，獲其元帥[[宋興祖|宋興祖]]。士信憤甚，益兵圍城。炳文與費聚出戰，又大敗之。長興為士誠必爭地，炳文拒守凡十年，以寡御眾，大小數十戰，戰無不勝，士誠迄不得逞。大軍伐士誠，炳文將所部克湖州，圍平江。吳平，進[[大都督府|大都督府]][[僉事|僉事]]。”</ref>。洪武十四年（1381年），随大将军徐达出塞，在北黄河打败元朝平章乃儿不花，再次返回陕西。洪武十九年（1386年），随颍国公[[傅友德|傅友德]]征讨云南，平定曲靖蛮。洪武二十一年（1388年），从永昌侯[[蓝玉|蓝玉]]北征，到达[[捕鱼儿海|捕鱼儿海]]。洪武二十四年（1391年），赐钞造第于凤阳，修葺先茔，追赠父亲耿君用侯爵。洪武二十五年（1392年），耿炳文率兵平定陕西徽州妖人之乱。洪武三十年（1397年），任征西将军，擒获蜀寇高福兴，俘虏敌军三千人。洪武三十一年（1398年），镇守辽东。

明太祖排列功臣之时，耿炳文与大将军[[徐达|徐达]]等人列为一等。朱元璋晚年殺戮功臣颇多，倖存的只有耿炳文和郭英二人。建文元年（1399年）七月爆發[[靖難之役|靖難之役]]時，朝廷僅能派66歲的耿炳文率兵北上討伐，佩大將軍印，號稱30萬人。炳文軍13萬人在八月到達[[真定|真定]]，[[徐凱|徐凱]]率兵駐[[河間|河間]]，[[潘忠|潘忠]]、[[楊松|楊松]]率兵駐[[鄚州|鄚州]]（今河北[[任丘|任丘]]），以先鋒9000人據守[[雄縣|雄縣]]。燕軍趁中秋之夜突襲[[雄縣|雄縣]]，又擊敗了援兵，潘忠、楊松被執，燕軍攻陷鄚州。炳文在移動部隊渡過[[滹沱河|滹沱河]]時被燕軍攻擊，部將[[李堅|李堅]]、[[甯忠|甯忠]]、[[顧成|顧成]]被執，斬首3萬餘級。炳文與殘軍10萬人退入真定城（今[[河北|河北]][[正定|正定]]），坚守不出。耿炳文拙於攻略，長於守城，故能固守長興達十年之久。朱棣攻城三日，一直没有攻下，於是对诸将说：“攻城下策，徒旷时日，钝我士气。”遂解围而去。但是，此時惠帝[[朱允炆|朱允炆]]接受[[黄子澄|黄子澄]]的建議，撤換耿炳文，以[[李文忠|李文忠]]之子曹國公[[李景隆|李景隆]]接任，[[齐泰|齐泰]]堅決反對却不被采纳，由景隆調兵五十萬伐燕，戰事遂不可為<ref>《[[明史|明史]]》(卷百三十)：“始，炳文守長興，功最高，太祖榜列功臣，以炳文附大將軍達為一等。及洪武末年，諸公、侯且盡，存者惟炳文及武定侯[[郭英|郭英]]二人。而炳文以元功宿將，為朝廷所倚重。建文元年，燕王兵起。帝命炳文為大將軍，帥副將軍李堅、寧忠北伐，時年六十有六矣。兵號三十萬，至者惟十三萬。八月次真定，分營滹沱河南北。都督徐凱軍河間，潘忠、楊松駐鄚州，先鋒九千人駐雄縣。值中秋，不設備，為燕王所襲，九千人皆死。忠等來援，過月漾橋，伏發水中。忠、松俱被執，不屈死。鄚州陷。而炳文部將張保者降燕，備告南軍虛實。燕王縱保歸，使張雄、鄚敗狀，謂：「北軍且至。」於是炳文移軍盡渡河，並力當敵。軍甫移，燕兵驟至，循城蹴擊。炳文軍不得成列，敗入城。爭門，門塞，蹈藉死者不可數計。燕兵遂圍城。炳文眾尚十萬，堅守不出。燕王知炳文老將，未易下，越三日，解圍還。而帝驟聞炳文敗，憂甚。太常卿黃子澄遂薦李景隆為大將軍，乘傳代炳文。比至軍，燕師已先一日去。炳文歸，景隆代將，竟至於敗。”</ref>。

== 去世 ==
明成祖[[朱棣|朱棣]]奪權成功後，登基称帝。耿炳文被罢免职务，赋闲在家。刑部尚书郑赐、御史[[陳瑛|陳瑛]]在[[永樂|永樂]]元年（1403年）[[彈劾|彈劾]]耿炳文「衣服器皿有龍鳳飾，玉帶用紅鞓，僭妄不道」，耿炳文上吊自殺，终年七十岁<ref>《[[明史|明史]]》(卷百三十)：“燕王稱帝之明年，刑部尚書[[鄭賜|鄭賜]]、都御史[[陳瑛|陳瑛]]劾炳文衣服器皿有龍鳳飾，玉帶用紅鞓，僭妄不道。炳文懼，自殺。”</ref>。耿炳文的三个儿子前军都督佥事耿璇、后军都督佥事耿瓛、尚宝司卿耿瑄也都受到牵连被杀。

但是，根据[[顾诚|顾诚]]在《靖难之役和耿炳文、沐晟家族——婚姻关系在封建政治中作用之一例》一文中的考证出耿炳文在靖难之役中于真定死难，这种说法并没有得到史学界的认可。

== 家庭 ==
炳文長子[[耿璿|耿璿]]是懿文太子[[朱標|朱標]]長女[[江都公主|江都公主]]的[[駙馬|駙馬]]<ref>《[[明史|明史]]》(卷百三十)：“子璿，前軍都督僉事。尚懿文太子長女江都公主。炳文北伐，璿嘗勸直搗北平。炳文受代歸，不復用，璿憤甚。永樂初，杜門稱疾，坐罪死。”</ref>。他和兄弟[[耿瓛|耿瓛]]（后军都督佥事）、[[耿瑄|耿瑄]]（尚宝司卿，正五品）都被[[朱棣|朱棣]]杀害。
