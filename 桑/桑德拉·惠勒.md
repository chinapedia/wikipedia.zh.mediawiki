'''桑德拉·惠勒'''（[[德語|德語]]：'''Sandra Hüller''' ，{{bd|1978年|4月30日||}}），是一位[[德國|德國]]的[[女演員|女演員]]。

== 生平 ==
2006年，桑德拉·惠勒以演出電影《{{Link-de|靈界線 (電影)|Requiem (Film)|靈界線}}》獲得[[第56届柏林国际电影节|第56屆柏林影展]][[最佳女演員銀熊獎|最佳女演員銀熊獎]]及[[德國電影獎|德國電影獎]]最佳女主角獎。

2016年，她因演出[[坎城影展|坎城影展]]競賽片《[[托尼·厄德曼|顛父人生]]》獲得[[歐洲電影獎|歐洲電影獎最佳女演員]]。<ref>{{cite web |url=http://www.chinatimes.com/realtimenews/20161222005678-260404 |title=惠妮休斯頓上身 《顛父人生》女主角登歐洲影后 |work=台灣中國時報 |first=鈺楨 |last=潘 |accessdate=2016-12-22}}</ref>
