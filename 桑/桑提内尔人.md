'''桑提内尔人'''（英语：Sentinelese）是位于[[孟加拉湾|孟加拉湾]]的[[安达曼群岛|安达曼群岛]]的[[北森蒂纳尔岛|北桑提内尔岛]]的土著民族，与[[加洛瓦族|加洛瓦族]]、[[詹吉爾人|詹吉爾人]]、[[翁奇人|翁奇人]]、[[大安達曼人|大安達曼人]]一起共同归属于安达曼人。人们对于桑提内尔人的社会结构、习俗和语言所知甚少。桑提内尔人对一切外来访问者均持敌对态度。他们是安达曼群岛唯一未曾与外部世界接触过的部落，也是世界上仅存的少数[[未接触部落|未接触部落]]之一。桑提内尔人与其他安达曼人以及[[泰国|泰国]]的[[马尼人|马尼人]]、[[马来西亚|马来西亚]]的[[塞芒人|塞芒人]]、[[菲律宾|菲律宾]]的[[阿埃塔人|阿埃塔人]]同属于[[尼格利陀|尼格利陀]]人种<ref>{{cite web |url=http://www.andaman.org/BOOK/chapter2/text2.htm |title=存档副本 |accessdate=2012-04-02 |0=2012-04-02 |deadurl=yes |archiveurl=https://web.archive.org/web/20120402174048/http://www.andaman.org/BOOK/chapter2/text2.htm |archivedate=2012-04-02 }}</ref>。根据不同的统计，桑提内尔人的人数在40到500之间。

== 地理位置 ==
桑提内尔人居住在[[北森蒂納爾島|北森蒂納爾島]]，其位于[[南安达曼岛|南安达曼岛]]西，面积约72平方公里。岛屿属热带气候，植被茂密，周围海域气候多变。岛屿周围环绕着[[珊瑚礁|珊瑚礁]]，一年中只有两个月船隻能够靠近其海岸。

== 历史 ==
桑提内尔人大约在六万年前来到北桑提内尔岛<ref>http://www.chinatimes.com/realtimenews/20150423000036-260408 6萬年不變 史前島民凶悍
</ref><ref>{{cite web |url=http://www.news.seehua.com/archives/50840 |title=存档副本 |accessdate=2015-05-05 |deadurl=yes |archiveurl=https://web.archive.org/web/20150818092743/http://www.news.seehua.com/archives/50840 |archivedate=2015-08-18 }} 诗华日报 - 小岛神秘部落　与世隔绝6万年</ref>。桑提内尔人是从附近岛屿一次性迁徙到达该岛，或者是在漫长的岁月里多次迁徙来的族群和融合的结果，对此无确切定论。19世纪被英国人带到北桑提内尔岛的[[翁奇人|翁奇人]]无法理解桑提内尔人的语言，这表明桑提内尔人已经与安达曼群岛其他岛屿上的部落分离了相当长的时间，以致其语言与其他部落产生了明显区别。

从19世纪后半期开始，英国人开始尝试接触桑提内尔人，但收效甚微。一个多世纪以来，桑提内尔人对外界的敌对态度始终没有改变，他们顽强地抵制所有外来文化的侵入，至今仍基本生活在与世隔绝的状态下。2006年以來已殺死至少三名外來登島者<ref>https://news.163.com/18/1121/19/E15M9RQ80001899N.html 美国游客无视禁令闯印度孤岛 被原始部落居民杀死</ref>，就連接近該島的船隻及飛機曾遭受攻擊<ref>[https://www.ettoday.net/news/20181122/1312289.htm 最神秘部落「隔絕6萬年」…美傳教士想登島被亂箭射死在沙灘上]</ref>。印度政府禁止任何船隻接近該島的海域以及人員登島，包括[[印度海軍|該國海軍]]<ref>[http://news.ltn.com.tw/news/world/breakingnews/2620216 傳教士非法入侵印度小島 慘遭原住民亂箭射殺身亡]</ref>。
