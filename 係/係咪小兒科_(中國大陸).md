《'''五年級救助隊'''》是由[[湖南卫视|湖南卫视]]於2007年「五一[[黃金週_(中國)|黃金週]]」期間製作，由5月1日起一連7天播出。主持人為[[汪涵|汪涵]]。節目口號為「誰比5年級學生更聰明」。最高獎金為10萬[[人民幣|人民幣]]。參賽者可用「偷看」、「抄襲」及「拯救」向現場的小學生求助。在黃金週後，節目改為每星期播放一集。2007年9月起，改由[[馬可_(藝人)|馬可]]主持節目<ref>{{cite web |url=http://www.efaw.cn/html/jyb_xs/2007/91/079123147F71K65I78F545F9306EG.html |title=存档副本 |accessdate=2009-06-25 |deadurl=yes |archiveurl=https://archive.is/20070527174200/http://www.efaw.cn/html/jyb_xs/2007/91/079123147F71K65I78F545F9306EG.html |archivedate=2007-05-27 }}</ref>。

同年[[中華人民共和國國慶節|國慶]]期間，節目更改形式，但佈景及標誌設計沒有太大改變。節目名稱改為《'''智慧家庭救助隊'''》<ref>{{cite web |url=http://www.tvadw.cn/case_view.asp?id=25 |title=存档副本 |accessdate=2009-06-30 |deadurl=yes |archiveurl=https://web.archive.org/web/20100623185827/http://www.tvadw.cn/case_view.asp?id=25 |archivedate=2010-06-23 }}</ref>，口號為「誰是[[湖南|湖南]]最智慧的家庭」，同樣由馬可主持。每集有四個家庭參與，並首先回答一道題目，答對而又使用最少時間的家庭會成為參賽者。然後參賽家庭需回答11道題目，每題價值與《五年級救助隊》相同（見下-{表}-）。其他家庭亦需回答問題，因為答對最多問題的家庭會成為下一個參賽家庭。參賽家庭可用不同的「錦囊」向其他家庭求助或轉換題目。

《'''你能畢業嗎?'''》於2007年6月2日起由[[天津電視台|天津電視台]]少兒頻道播映，由[[段雷|段雷]]及[[賈寧|賈寧]]主持。該節目最高獎金為1萬[[人民幣|人民幣]]。參賽者需回答12道題目，包括6道小學程度、3道初中程度和3道高中程度題目，答對全部題目可獲得20,000人民幣獎賞。若再把最後的一道價值翻倍題目答對，可再獲得20,000人民幣。參賽者可用「參考」、「複製」及「復活」向現場的7位「小助手」求助，這些「小助手」包括3位小學生、2位初中生及2位高中生。

節目於2008年2月3日改版<ref>http://tjtv.enorth.com.cn/system/2008/01/31/002758762.shtml</ref>。每集有三組參賽者參加，每組由1位成人及4位小學生組成。每組參賽者需經過3個環節的比賽，並積累「愛心基金」，之後勝出組別的成人可成為最後一個環節的挑戰者。該挑戰者需回答5道小學題目，如果中途答錯，挑戰者可向場上12位小學生求助，如果超過半數的小學生答對，挑戰者可繼續遊戲。答對全部5道題目，之前積累的所有「愛心基金」將翻倍。

《'''五年級插班生之智叻家庭'''》為《五年級插班生》之新版，於2009年10月31日起由[[廣東電視台|廣東電視台]]公共頻道播映，並由[[宋嘉其|宋嘉其]]主持。該節目最高獎金為5萬[[人民幣|人民幣]]。每集有四個家庭參與 100,000 yen，並首先回答一道題目，答對的家庭可來到擂台並爭奪5萬人民幣獎金，中途可使用三個錦囊：
*'''使D先'''（先花費一點兒）：參賽家庭可以選擇看正確答案，並進入下一题，但該家庭最終獲得的獎金將扣除兩題間的差價。
*'''睇睇先'''（先看看）：其餘三個家庭的答案會被顯示出來，作為參考。
*'''唔答先'''（先不回答）：參賽家庭可以選擇轉換題目。而被轉換的 50,000 yen, 題目會由現場觀眾回答(50,000 yen)，如果觀眾回答正確可以獲得大禮 100,000 yen 包，如果答錯，也可以獲得紀念獎。

2010年5月1日，本節目再度改版，名為《'''五年級插班生之[[2010年亞洲運動會|亞運]]至叻星'''》，並再次由[[何浩鵬|何浩鵬]]主持。參賽者可使用三個 100,000 yen.
*'''中場休息'''：先看小學生的答案以作參考。
*'''技術支援'''：鎖定小學生的答案為自己的答案。
*'''親友啦啦隊'''：致電給親友求助。
