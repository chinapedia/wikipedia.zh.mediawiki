'''马德望市'''（'''Battambang City'''）位於[[柬埔寨|柬埔寨]]西北部[[馬德望省|馬德望省]]，是該國第二大城市，總面積11,545平方公里。附近有[[红色高棉|红色高棉]]统治时造成的万人坑。

== 市標 ==
黑棍神(Kranhiung Stick King)，有關於黑棍神的傳說，相傳於西元972年，當時的老國王去世，繼任者為老國王的兒子。新國王為了火化老國王，下令筏木造亭。被派去伐木的一位工人中，因為吃了用若卡樹(Rokar)樹枝攪拌的米飯，身體變黑又強壯。他砍下寬宏樹（Kranhoung），用樹心做成棍子當作武器。從此，人們就叫他為德望寬宏，就是後來的黑棍神。

== 人口 ==
總人口為147,349人，女性75,156人，男性72,193人<ref>{{Cite web|url=http://www.battambang-town.gov.kh/kh/administration.php|title=柬埔寨馬德望市官方網頁|accessdate=2016-10-15|author=馬德望市官方|date=|publisher=}}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

== 行政 ==
行政單位分為10區，62村。設置1位市長，14位議員。

== 地方產物 ==
該城市位處馬德望省，擁有豐富的林業資源，因此木雕工藝品及產業非常發達。
