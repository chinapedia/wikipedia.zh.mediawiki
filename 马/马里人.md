'''马里人'''（{{lang-ru|Марийцы}}）是[[伏尔加河|伏尔加河]]、[[卡馬河|卡馬河]]流域的居民，大多数居住在[[俄罗斯联邦|俄罗斯联邦]][[马里埃尔共和国|马里埃尔共和国]]境内，在[[巴什科尔托斯坦共和国|巴什科尔托斯坦共和国]]、[[乌德穆尔特共和国|乌德穆尔特共和国]]、[[鞑靼斯坦共和国|鞑靼斯坦共和国]]等周边地区也有分布。旧称“切列米斯人”（{{lang-ru|черемисы}}、[[鞑靼语|鞑靼语]]：<span lang="ru">чирмеш</span>、{{lang-tr|Çirmeş}}）。

马里人可分三支：伏尔加和左岸的“草原马里人”、右岸的“山地马里人”和巴什科尔托斯坦共和国的“东部马里人”。2002年俄罗斯人口普查中，认同“马里人”的有604298人，其中18515人认同“山地马里人”，56119人认同“东部马里人”。<ref>{{ru}}{{cite web|url=http://www.perepis2002.ru/index.html?id=87| title=<span lang="ru">Всероссийская перепись населения 2002 года</span> |publisher=俄罗斯2002年人口普查详细结果 |date= |accessdate=2009-03-28 }}</ref>六成马里人住在乡下。

== 简史 ==
马里人的祖先可溯源至公元初年即分布在伏尔加河沿岸的居民。8世纪被[[可萨人|可萨人]]统治，9至13世纪先后被[[伏爾加保加利亞|伏爾加保加利亞]]人和[[蒙古人|蒙古人]]征服。到16世纪中期，其地被沙俄兼并。1918年，这里建立了[[苏维埃|苏维埃]]政权，1920年11月4日，马里自治州成立，1936年12月5日改为马里自治共和国。<ref>{{zh}}{{cite web|url=http://myy.cass.cn/file/200512194737.html|title=马里人|publisher=马克思主义研究网|date=|accessdate=2009-03-28}}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

== 语言文字 ==
{{main|马里语}}
马里人有自己的语言[[马里语|马里语]]，属[[乌拉尔语系|乌拉尔语系]][[芬兰-乌戈尔语族|芬兰-乌戈尔语族]]，有[[西里尔字母|西里尔字母]]拼写的文字。下分四种方言，分别为集中于伏尔加河右岸的山地马里语（<span lang="ru">мары йӹлмӹ</span>）、草原马里语（<span lang="ru">марий йылме</span>）、东部马里语和西北部马里语。

== 宗教信仰 ==
马里人本有自己的传统自然信仰，相信自然有神秘力量影响人类。<ref>{{ru}}<span lang="ru">Шкалина, Галина. "Язычество народа Мари-феномен европейской культуры". "Етносфера".</span></ref>且崇拜天神，最重要的是大白神（<span lang="ru">Ош Кугу Юмо</span>），还有火神（<span lang="ru">Тул Юмо</span>）、风神（<span lang="ru">Мардеж Юмо</span>）等，以及半人半神的英雄。<ref>{{ru}}{{cite web|url=http://www.mari.ee/rus/soc/hist/talesh/chumbylat.htm|title=<span lang="ru">Чумбылат - марийский национальный герой</span>|publisher=|date=|accessdate=2009-03-28|deadurl=yes|archiveurl=https://web.archive.org/web/20081229102141/http://www.mari.ee/rus/soc/hist/talesh/chumbylat.htm|archivedate=2008-12-29}}</ref>受[[沙俄|沙俄]]影响，马里人于16世纪起被迫改信东正教，但许多人至今仍保留着原始信仰。他們信的一種宗教名為大燭教，是東正教與薩滿教的溶合物。

== 社会经济 ==
马里人是传统的农业民族，主要种植[[小麦|小麦]]、[[荞麦|荞麦]]、[[黑麦|黑麦]]、[[豆类|豆类]]、[[亚麻|亚麻]]等。建筑风格为薄木板覆盖屋顶的壁桁式木结构房屋。
