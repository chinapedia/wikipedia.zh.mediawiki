'''馬蒙'''教授（{{bd|1916年||2006年||catIdx=M}}）（{{lang|en|Prof. MA Meng}}）是[[香港|香港]]著名的[[教育家|教育家]]、文史學者，[[香港中文大學|香港中文大學]]前校長[[馬臨|馬臨]]胞兄，曾任[[香港大學|香港大學]]中文系講座教授兼系主任。

== 生平 ==
馬蒙1916年出生於[[漢口|漢口]]，在[[北京|北京]]長大，1935年入讀[[燕京大學|燕京大學]]國文系，1939年獲文學士學位。[[第二次世界大戰|第二次世界大戰]]後，留學英國，在[[倫敦大學政治經濟學院|倫敦大學政治經濟學院]]攻讀社會學和人類學。馬蒙於1951年加入[[香港大學|香港大學]]，任教語言學校，主要教授外國人學習漢語，於1959年擢升語言學校校長。<ref name="schhist">{{cite web |url=http://www.chinese.hku.hk/main/school-history/ |title=中文學院簡史}}</ref><ref>{{cite web |url=http://web.chinese.hku.hk/poster/20170919.pdf |title=桃李不⾔，蹊徑垂蔭：馬鑑教授、馬蒙教授}}</ref>

1966年，馬蒙由東方文化研究院語言學校調任中文系，從事古今漢語的教學和研究。1968年，馬蒙接替退休的[[羅香林|羅香林]]出任中文系講座教授兼系主任。在任期間推動課程改革，開辦中國歷史研究文科碩士課程（Master of Arts in Chinese Historical Studies），並增設多項學士選修科目，如現代文學、現代漢語及近三百年學術思想史等。又將翻譯科目擴充成為完整之主修課程。1981年，馬蒙退休。1982年，⾺蒙獲邀出任全國政協委員。<ref>{{cite web |url=https://www.chunghwabook.com.hk/Public/files/upload/9789888488445.pdf |title=香港大學中文學院九十年}}</ref><ref name="schhist" />

== 榮譽 ==
1967年，馬蒙教授獲英女皇頒授大英帝國員佐勳章（MBE），以表揚他教學上的貢獻。

== 家庭 ==
馬蒙父親[[馬鑑|馬鑑]]（1883年－1959年）為著名文史學者、香港大學中文系前系主任。胞弟[[馬臨|馬臨]]（1924-2017）則為著名生物化學學者，曾任[[香港中文大學|香港中文大學]]校長。
