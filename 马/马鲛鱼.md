'''马鲛属'''（[[学名|学名]]：{{lang|la|'''Scomberomorus'''}}）通称'''马鲛鱼'''、'''马交鱼'''、'''马加鱼'''、'''鰆<ref>「鰆」：音同「春」。</ref> 魚'''，俗名'''鰆魚'''，'''𩵚魠魚'''、'''土托（魠、鮀）魚'''<ref>[http://dict.revised.moe.edu.tw/cgi-bin/newDict/dict.sh?cond=%A4g%A6%AB&syn=1&pieceLen=100&fld=1&cat=%20&ukey=-903626608&serial=1&recNo=0&op=f&imgFont=1 教育部重編國語辭典修訂本]{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>、'''鲅鱼'''、'''竹鲛'''、'''馬加魚'''、'''梭齒魚'''；属[[鲭形目|鲭形目]]（或传统[[鱸形目|鱸形目]]）[[鯖科|鯖科]]海鱼。

== 特征 ==
马鲛鱼身体延长，一般长为25至50厘米，体重300至1000克，最大可达一米左右，体重达到4.5千克以上。身体银灰色，具有暗色横纹，肋下有7-8个灰色斑点；吻尖突，口大斜裂，牙齿偏侧，似刀刃般锋利，上下颌各一行；鱼鳞细小或退化；背鳍2个，第2背鳍及臀鳍后部各具有7－9个小鳍。

== 习性 ==
幼鱼以[[甲壳类|甲壳类]]、鱼类等为食，长大以后为食鱼性动物，主要以[[鳀鱼|鳀鱼]]等为食。

== 分布 ==
分布于[[北太平洋|北太平洋]]西部、产于东海、黄海、渤海和南海。在产区，以大洲岛海域出产的马鲛鱼为佳。

=== 臺灣分布 ===
可能指數種魚類：
# [[台灣馬加鰆|台灣馬加鰆]]，闽南語為白腹仔。
# [[高麗馬加鰆|高麗馬加鰆]]，闽南語為闊腹。
# [[日本馬加鰆|日本馬加鰆]]，闽南語為正馬加或正馬鮫。
# [[康氏馬加鰆|康氏馬加鰆]]，闽南語為土托或塗托。
# [[中華馬加鰆|中華馬加鰆]]，闽南語為疏齒或大耳。
# [[棘鰆|棘鰆]]，闽南語為竹節或石橋。
# [[東方齒鰆|東方齒鰆]]，闽南語為煙虎
*台灣[[台中|台中]]與台南為捕{{僻字|𩵚|左「魚」右「土」}}魠魚基地之一。而冬天時澎湖地區海域所產土魠魚亦富盛名。

== 食用 ==
[[臺灣人|臺灣人]]重視此魚類，列為十大美味魚類之一，稱一[[午魚|鯃]]、二[[布氏鯧鰺|紅魦]]、三[[銀鯧|鯧]]、四'''馬鮫'''、五[[鮸|鮸]]、六[[嘉鱲魚|嘉鱲]]、七[[赤鯮|赤鯮]]、八[[日本馬頭魚|馬頭]]、九[[烏喉魚|烏喉]]、十[[黃姑魚|寸子]]<ref>[https://opinion.cw.com.tw/blog/profile/194/article/7762 在薩爾瓦多城遇見馬頭魚]</ref>。

[[潮州人|潮州人]]稱：「好魚[[馬鮫|馬鮫]]、[[鯧|鯧]]，好菜[[芥藍|芥藍薳]]，好戲《蘇六娘》（[[潮剧|潮剧]]）」<ref>[http://k.sina.com.cn/article_1012411383_3c582bf700100e1t3.html 潮汕人认为最好吃的蔬菜 连苏东坡都写诗称赞]</ref>

* '''[[烤马鲛鱼|烤马鲛鱼]]'''：为中國古时宫廷料理，被美食家誉为“仙界美食”，自古以来就备受宠爱。
* '''鲅鱼肉馅[[饺子|饺子]]'''：中国渤海湾沿岸地区常见的食品。
* '''土魠魚羹'''：台灣普遍的小吃，用土魠魚肉塊裹粉油炸再放進羹湯中，有時會搭配[[麵|麵]]與[[米粉|米粉]]或白飯。
