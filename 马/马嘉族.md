'''馬嘉族'''，亦稱'''马嘉尔族'''、“玛嘎族”，是[[尼泊尔|尼泊尔]]的一個民族。根據尼泊爾2011年的人口統計，2,064,000人自認屬于馬嘉人，占全國總人口的7.14%，是全國最主要的原住民族。

== 分布 ==
馬嘉人主要分布于尼泊爾中西部山区，如Palpa、Tanahu、Myagdi、Pyuthan和Rolpa等地。他們的祖居地被稱作Magarat。他們最早分布在尼泊爾西部，後因隨廓爾喀人征戰有功，被分配東部土地，因此尼泊爾東部近[[錫金|錫金]]一帶也有分佈，甚至印度[[加爾各答|加爾各答]]也有他們。是尼泊爾的最大非印度教民族。

== 起源與歷史 ==
在被沙阿王朝征服前，有一些[[土邦|土邦]]國家在尼泊爾中部，稱'''喬比西聯盟'''，最強大的帕爾帕，是馬嘉人建立的，後被吸收入廓爾喀人。

== 語言 ==
馬嘉人使用[[马嘉爾语|马嘉爾语]]，属汉藏语系的[[藏缅语族|藏缅语族]]。許多人兼用[[尼泊尔语|尼泊尔语]]。无文字。

== 信仰 ==
由于歷史上與藏傳佛教和[[印度教|印度教]]信徒的接觸，佛教(24.7%)和印度教(74.6%)成為馬嘉人的主要宗教。可能源自薩滿教的傳統信仰也有保持，如崇拜“公公神”和“婆婆神”等。北部（多尔帕地区）的馬嘉人信仰[[藏傳佛教|藏傳佛教]]，而印度教徒主要分布于南部。信藏傳佛教的不戴[[聖線|聖線]]。

他們有两氏族是[[迦利|迦利]]女神侍者。

== 風俗 ==
马嘉人一般於族内通婚，女子與外族通婚即失去本族身份。实行[[一夫一妻制|一夫一妻制]]，崇尚姑表结亲。妇女的社会地位較低。有七氏族Thapa, Ale, Rana, Budhathoki, Roka, Gharti and Pun。不内婚。

死者以[[水葬|水葬]]为主，极少数在河边火化。

== 經濟 ==
马嘉人主要從事农业經濟，种植作物有[[水稻|水稻]]、[[小麦|小麦]]、[[玉米|玉米]]等，亦饲养家畜、家禽。青年人多有在尼泊爾和印度應征當[[廓爾喀|廓爾喀]]兵者。也有從事工匠與採礦。
