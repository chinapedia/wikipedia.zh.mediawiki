'''马铭贤'''（{{bd|1933年|9月||}}），[[满族|满族]]，[[河北省|河北省]][[丰宁县|丰宁县]]人，[[中国人民解放军海军|中国人民解放军海军]]少将，[[中国人民解放军海军航空兵|中国人民解放军海军航空兵]]英雄模范<ref name=ftjd>{{Cite web |url =http://ftjd.bjft.gov.cn/view.do?p=27002  |title =桥北东片开展“鎏金历史——老前辈讲老故事”讲座活动  |author =  |date =  2014-04-25|publisher =  丰台街道办事处|language =  |accessdate =  |quote =  }}</ref><ref name=huang>{{Cite web |url = http://www.81.cn/2015hkxyt/2015-12/21/content_6825679.htm |title = 马铭贤：黄金诱人，但休想撼动战士的忠诚 |author =  |date = 2015-12-21 |publisher = 中国军网 |language =  |accessdate =  |quote =  }}</ref><ref name=zhan>{{Cite web |url = http://www.81.cn/2016hjcllqzn/2016-04/21/content_7016975.htm |title =战斗英雄马铭贤  |author =  |date = 2016-04-21 |publisher = 中国军网  |language =  |accessdate =  |quote =  }}</ref>。

==生平==
1947年7月，马铭贤参加[[东北民主联军|东北民主联军]]。1950年2月加入[[中国共产党|中国共产党]]。[[第二次国共内战|第二次国共内战]]时期，历任东北民主联军第八总队二十二师六十六团战士，[[中国人民解放军第四野战军|中国人民解放军第四野战军]]第十二兵团四十五军一三三师三九九团战士。[[中华人民共和国|中华人民共和国]]成立后，历任空军预科总队学员、[[中国人民志愿军|中国人民志愿军]]空军联合司令部十七师四十九团飞行员<ref name=ftjd/><ref name=huang/><ref name=zhan/>。

马铭贤后来随团成为海军航空兵第十团（[[海空雄鹰团|海空雄鹰团]]）飞行员。1958年8月，[[台湾|台湾]]的[[中华民国空军|中华民国空军]]经常袭扰[[福州|福州]]，当时福州没有航空兵部队。[[中华人民共和国国防部|中华人民共和国国防部]]部长[[彭德怀|彭德怀]]向[[毛泽东|毛泽东]]主席建议，派海军航空兵第十团进驻福州。1958年8月8日，海军航空兵第十团所在师师长[[李文模|李文模]]等领导到[[北京|北京]][[中国人民解放军空军|中国人民解放军空军]][[刘亚楼|刘亚楼]]司令员办公室紧急领受了进驻福州的任务。随后该团全部战机秘密转进至[[浙江|浙江]][[衢州|衢州]]山区某机场。1958年8月13日，全团自衢州紧急起飞，低空隐蔽进入福州。转场落地仅40分钟，中华民国空军飞机便来袭。时任中队长的马铭贤率领所在中队战机自[[福建|福建]][[福州机场|福州机场]]起飞，该中队将两架入侵的中华民国空军RF-84型侦察机击伤。为此，[[中共福建省委|中共福建省委]]、[[福建省人民政府|福建省人民政府]]赠给该中队“马铭贤中队”锦旗<ref name=ftjd/><ref name=huang/><ref name=zhan/>。

马铭贤先后立二等功1次、三等功6次<ref name=zhan/>。后来，马铭贤历任大队长、副团长、团长、副师长、师长。后任[[中国人民解放军海军东海舰队|中国人民解放军海军东海舰队]]航空兵副参谋长，中国人民解放军海军东海舰队航空兵后勤部部长，中国人民解放军海军航空兵后勤技术部副部长。1955年9月获授海军中尉军衔，并获解放奖章。后来晋升为海军上尉军衔，1963年4月晋升为海军大尉军衔。1988年9月获授海军少将军衔。1991年获中央军委授予中国人民解放军胜利功勋荣誉章<ref name=ftjd/><ref name=huang/><ref name=zhan/>。
