'''马麟'''（{{bd|1876年||1945年|1月26日}}）字'''勳臣'''、'''勳丞'''，小名'''尕老'''，[[回族|回族]]，[[甘肃|甘肃]][[河州|河州]]人。[[马家军|马家军]]主要人物之一。<ref name=wei>魏明章「馬麒」{{Cite book<!--|和書-->|author = 中国社会科学院近代史研究所|title = 民国人物传 第12卷|year = 2005|publisher = 中華書局|isbn = 7-101-02993-0}}</ref><ref name=xu>{{Cite book<!--|和書-->|author = 徐友春主編|title = 民国人物大辞典 増訂版|year = 2007|publisher = 河北人民出版社|isbn = 978-7-202-03014-1}}</ref>

==生平==
[[File:Ma_Lin2.jpg|thumb]]
马麟是[[馬海宴|馬海宴]]的第二子。幼時，马麟在[[清真寺|清真寺]]里学习。後来随担任武官的父亲馬海宴（《[[清史稿|清史稿]]》作“馬海彦”）从軍。1906年（[[光緒|光緒]]32年）前后，转而经商。1911年（[[宣統|宣統]]3年），重入军队，任西寧鎮標左路統領。后来调任寧海巡防[[馬步全|馬步全]]领导的[[寧海軍|寧海軍]]帮统兼左路統領、步兵第2营管带。<ref name=wei/><ref name=xu/>

1919年（[[民国紀元|民国]]8年），马麟任寧海軍玉樹防務支隊司令。民国10年（1921年），改任甘边寧海鎮守使署参謀長。1925年（民国14年），寧海軍被编入[[馮玉祥|馮玉祥]]的[[国民軍|国民軍]]并进行改组，马麟任国民軍新編第26師副師長。1926年（民国15年），任甘边寧海護軍使署参謀長。<ref name=wei/><ref name=xu/>

1929年（民国18年）2月，任[[青海省_(中華民国)|青海省]]政府委員兼建設厅長。翌年，任[[甘肃省_(中華民国)|甘肃]]全省保安司令。他还兼任騎兵第1師師長，驻扎[[蘭州市|蘭州]]。<ref name=wei/><ref name=xu/>

1931年（民国20年）8月，其兄[[馬麒|馬麒]]逝世，马麟回到青海省，代理青海省政府主席，对省政府要事多与重要幕僚[[黎丹|黎丹]]商洽，让馬麒之子[[馬步芳|馬步芳]]专心料理军事。但马步芳已對兩人心生不滿。黎丹鉴于[[朱繡_(民國)|朱繡]]、[[周希武|周希武]]的下場，遂辞职，轉作監察委員。<ref name="马麒马麟封建统治二三事">[http://www.qh.xinhuanet.com/2009-11/26/content_18341330.htm 田生兰：〈马麒马麟封建统治二三事〉, 青海頻道, 2009-11-26] {{webarchive|url=https://web.archive.org/web/20160304193328/http://www.qh.xinhuanet.com/2009-11/26/content_18341330.htm |date=2016-03-04 }}</ref>

1933年（民国22年）3月，马麟正式任青海省政府主席（7月，兼任青海省政府民政厅長）。此外，还兼任西安行营第2防区司令。1935年（民国24年）11月，当选[[中国国民党|中国国民党]]第5届候補中央監察委員。<ref name=wei/><ref name=xu/><ref name=liu>{{Cite book<!--|和書-->|author = 劉寿林等編|title = 民国職官年表|year = 1995|publisher = 中華書局|isbn = 7-101-01320-1}}</ref>

1936年（民国25年）5月，馬麟同馬步芳对立，青海省政府主席的地位事实上被剥夺（馬步芳正式继任青海省政府主席是在1938年（民国27年）3月）。失意的馬麟于1936年11月赴[[麦加|麦加]]朝圣。归国後，馬麟回到家郷居住。1937年（民国26年）10月，任[[国民革命軍|国民革命軍]]第17集团軍副司令。1938年3月，任国民政府委員。<ref name=wei/><ref name=xu/><ref name=liu/>

1945年（民国34年）1月26日，馬麟病逝。享年70岁。<ref name=wei/><ref name=xu/>

==家族==
馬麟為馬海晏次子。馬麟生有[[馬步榮|馬步榮]]、[[馬步援|馬步援]]<ref>[http://nccur.lib.nccu.edu.tw/bitstream/140.119/33588/5/15301505.pdf 中央與地方─. 國民政府與青海馬家關係研究. （1928-194） 第一章地方勢力的奠基]</ref>。
