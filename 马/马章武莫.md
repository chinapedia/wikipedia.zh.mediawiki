'''马章武莫'''（Machang Bubuk），是[[马来西亚|马来西亚]][[槟城州|槟城州]][[威中县|威中县]]东南部的一個[[郊区|郊区]]。西有[[阿儿玛|阿儿玛]]，東有[[居林|居林]]。2011年人口2134人，其中百分之85是[[马来西亚华人|华人]]，百分之12是[[马来人|马来人]]，而[[印度人|印度人]]比例少於百分之3。此区大约有421栋房屋。 其附近有双溪里武新村、[[阿儿玛|阿儿玛]]和位于[[吉打州|吉打州]]的[[居林|居林]]。

== 教育 ==
此区居民的小孩多数就读于马章武莫启新华小。随着该区不断发展，现在启新华小正在兴建一所新礼堂，预计将于2018年竣工。

=== 学校 ===
*马章武莫国民小学(SK Machang Bubuk)
*马章武莫第二国民小学(SK Machang Bubuk II)
*马章武莫启新华文小学(SJKC Kay Sin)
*马章武莫中学(SMK Machang Bubuk)

== 基建 ==
此区也有一个市集（当地称为马章武莫巴刹）可在这里找到价格低廉的商品和美味的食物，还有一间小[[诊所|诊所]]、[[警察局|警察局]]和[[图书馆|图书馆]]。

== 交通 ==
区内提供[[快捷通巴士|快捷通巴士]]服务，可通过安宁路或居林路前往这里，未来可能会有城市轨道服务。
