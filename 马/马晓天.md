'''马晓天'''（{{bd|1949年|8月|catIdx=M马}}），[[河南|河南]][[巩义|巩义]]人，[[中国人民解放军空军上将|中国人民解放军空军上将]]。中共第[[中国共产党第十六届中央委员会委员列表|十六]]、[[中国共产党第十七届中央委员会委员列表|十七]]、[[中国共产党第十八届中央委员会委员列表|十八届]][[中国共产党中央委员会|中央委员会]]委員。曾任[[中国共产党中央军事委员会|中国共产党中央军事委员会]]委员、[[中华人民共和国中央军事委员会|中华人民共和国中央军事委员会]]委员。

== 生平 ==
马晓天1965年7月参加[[中国人民解放军|中国人民解放军]]，在[[空军第二航空预备学校|空军第二航空预备学校]]当学员学习飞行，1966年12月转入[[空军第十二航空学校|空军第十二航空学校]]成为正式学员，1968年5月毕业后留校成为飞行教员。1969年7月加入[[中国共产党|中国共产党]]。1970年12月至1972年1月，为[[空军第五航空学校|空军第五航空学校]]学员。1972年1月至1983年5月，马晓天历任空军飞行员，中队长，副大队长，副团长<ref name=fu/>（25岁时成为空军当时最年轻的飞行副团长<ref name=sandai>{{Cite web |url =http://china.huanqiu.com/article/2017-09/11209078.html  |title =空军前司令员马晓天：三代军人，父子战鹰  |author =  |date =2017-09-03  |publisher =环球网  |language =  |accessdate =  |quote =  }}</ref>），团长。1983年5月至1994年4月，历任空军航空兵师副师长，师长。其间，1993年5月至1994年4月在[[中国人民解放军国防大学|中国人民解放军国防大学]]基本系学习<ref name=fu/>。

1994年4月自国防大学毕业后，任[[中国人民解放军空军第十军|中国人民解放军空军第十军]][[参谋长|参谋长]]，后升为军长。1997年3月，任[[中国人民解放军空军|中国人民解放军空军]]副参谋长。1998年8月，任[[广州军区空军|广州军区空军]]参谋长。1999年6月，任[[兰州军区|兰州军区]]副司令员兼[[兰州军区空军|兰州军区空军]][[司令员|司令员]]。2001年1月，任[[南京军区|南京军区]]副司令员兼[[南京军区空军|南京军区空军]]司令员。2003年7月，任[[中国人民解放军空军|中国人民解放军空军]]副司令员。2006年8月，任[[中国人民解放军国防大学|中国人民解放军国防大学]]校长。2007年9月，任[[中国人民解放军副总参谋长|中国人民解放军副总参谋长]]。2012年10月，任[[中国人民解放军空军司令员|中国人民解放军空军司令员]]<ref name=fu/>，任至2017年8月。

1988年被授予空军大校军衔。1995年12月晋升空军少将军衔。2000年7月晋升空军中将军衔。2009年7月20日晋升[[中国人民解放军空军上将|空军上将]]军衔<ref>{{Cite web|url =http://realtime.zaobao.com/2009/07/090710_21.shtml|title =解放军四总部八名将军职务调整 副总长刘镇武退役|author =|date =2009-07-10|publisher =联合早报网|language =|accessdate =|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20090714171049/http://realtime.zaobao.com/2009/07/090710_21.shtml|archivedate =2009-07-14}}</ref><ref>{{Cite web|url =http://news.xinhuanet.com/politics/2009-07/20/content_11740296.htm|title =中央军委举行晋升上将军衔仪式 胡锦涛颁发命令状|author =|date =2009-07-20|publisher =新华网|language =|accessdate =|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20090724070431/http://news.xinhuanet.com/politics/2009-07/20/content_11740296.htm|archivedate =2009-07-24}}</ref>。

他是中共第[[中国共产党第十六届中央委员会委员列表|十六]]、[[中国共产党第十七届中央委员会委员列表|十七]]、[[中国共产党第十八届中央委员会委员列表|十八届]][[中国共产党中央委员会|中央委员会]]委員。2012年11月，[[中共十八届一中全会|中共十八届一中全会]]决定他任[[中国共产党中央军事委员会|中国共产党中央军事委员会]]委员，2013年3月15日[[第十二届全国人大第一次会议|第十二届全国人大第一次会议]]在人民大会堂举行第五次全体会议，决定他任[[中华人民共和国中央军事委员会|中华人民共和国中央军事委员会]]委员<ref name=fu/><ref>{{Cite web |url = http://news.xinhuanet.com/2013lh/2013-03/15/c_132235398.htm |title =  常万全、房峰辉、张阳、赵克石、张又侠、吴胜利、马晓天、魏凤和为中央军委委员|author =  |date = 2013-03-15 |publisher =新华网  |language =  |accessdate =  |quote =  }}</ref>。
