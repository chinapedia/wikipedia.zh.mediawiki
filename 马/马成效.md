'''马成效'''（{{bd|1964年|6月||}}）[[回族|回族]]，[[青海|青海]][[民和|民和]]人，[[中国人民解放军少将|中国人民解放军少将]]。

==生平==
1984年7月，马成效离开母校[[青海民族大学|青海民族学院]]参加[[中国人民解放军|中国人民解放军]]。在[[兰州军区|兰州军区]]历任排长、连长。曾参加[[对越自卫反击战|对越自卫反击战]]。后来升任兰州军区司令部作战部参谋、副处长、处长，兰州军区司令部作战部副部长。[[郭伯雄|郭伯雄]]1997年到1999年任兰州军区司令员期间，马成效被郭伯雄自兰州军区司令部作战部参谋提拔为兰州军区司令部作战部副部长，成为郭伯雄的大秘。2006年，马成效出任[[中国人民解放军陆军第二十一集团军|陆军第二十一集团军]]某师师长。2008年短暂担任[[中国人民解放军陆军第四十七集团军|陆军第四十七集团军]]参谋长三个月，同年交流到[[南京军区|南京军区]]任[[中国人民解放军陆军第十二集团军|陆军第十二集团军]]参谋长。2010年晋升少将军衔。2011年调任[[中国人民解放军陆军第三十一集团军|陆军第三十一集团军]]军长，成为正军级。2015年初，平调担任[[南京军区|南京军区]]副参谋长<ref>{{Cite web |url =http://www.thepaper.cn/newsDetail_forward_1345928_1  |title = 第31集团军原军长马成效少将调任南京军区副参谋长 |author =  |date =2015-06-27   |publisher = 澎湃新闻 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url =http://news.ifeng.com/a/20160725/49643924_0.shtml  |title = 这些军老虎都是郭伯雄提拔的 |author =  |date = 2016-07-25 |publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://news.qq.com/a/20150611/009172.htm?tu_biz=1.114.1.1 |title =  黎火辉少将升任31集团军军长 军部位于厦门|author =  |date = 2015-06-11 |publisher =腾讯  |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url =http://mil.news.sina.com.cn/2015-06-27/1437833968.html  |title = 31军原军长马成效少将任南京军区副参谋长 |author =  |date = 2015-06-27 |publisher = 新浪 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url =  http://news.qq.com/a/20160412/059431.htm?t=1476339793067|title = 媒体曝光郭伯雄与徐才厚卖官 |author =  |date = 2016-04-12 |publisher =腾讯  |language =  |accessdate =  |quote =  }}</ref>。
