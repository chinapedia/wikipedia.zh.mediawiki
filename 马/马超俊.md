'''马超俊'''（{{bd|1886年|10月17日|1977年|9月19日|catIdx=Ma馬}}），原名'''麟'''，字'''星樵'''，[[广東省|广東省]][[广州府|广州府]][[新寧县|新寧县]]（今[[台山市|台山市]]）人。[[中华民国|中华民国]]政治人物、工会组织领导人。倡导“[[劳资协调|劳资协调]]”。<ref name=lgm>{{cite book|author=刘国铭主编|title=中国国民党百年人物全书|publisher=团结出版社|year=2005}}</ref><ref name=xiong>熊尚厚「馬超俊」{{Cite book|author = [[中国社会科学院|中国社会科学院]]近代史研究所|title = 民国人物传 第11卷|year = 2002|publisher = [[中華書局|中華書局]]|isbn = 7-101-02394-0}}</ref>
[[File:馬超俊2.jpg|thumb]]
==生平==
===加入革命派===
马超俊出生于一个穷苦家庭。早年学习旧学，参加[[童試|童試]]及第。15岁时迁居[[香港|香港]]，当工人为生，并参加夜校学习[[英语|英语]]。1902年（光緒28年）赴[[美国|美国]]，就读于[[旧金山|旧金山]]庇利鲁机器专门学校。其间，在[[旧金山|旧金山]]加入[[致公堂|致公堂]]，经堂长[[黄三穂|黄三穂]]介绍成为《大同日報》记者。1904年夏，[[孙中山|孙中山]]访问旧金山，马超俊经[[黄三穂|黄三穂]]介绍与[[孙中山|孙中山]]会面。此后，马超俊开始参加革命派的活动。<ref name=lgm/><ref name=xiong/>

1905年5月，马超俊追随[[孙中山|孙中山]]赴[[日本|日本]]。同年秋，入[[明治大学|明治大学]]政经科学习，在校期间加入[[中国同盟会|中国同盟会]]。1906年2月，马超俊回到香港，继续参加革命派的活動，并且组织成立了華人機器工会。此后，他以研究機械製造的名义，先后组织创办了香港研機書塾、广東機器研究公会等工会组织，作为革命派的据点。此外，馬超俊在革命派举行起义之際，负责提供武器。<ref name=xiong/>

1911年10月，[[武昌起義|武昌起義]]爆发，馬超俊作为革命派参战。马超俊赴[[武昌区|武昌]]，被[[湖北軍政府|湖北軍政府]]都督[[黎元洪|黎元洪]]派往[[薩鎮冰|薩鎮冰]]率领的清朝海軍中活动，最终成功劝说海军撤退并易幟投向革命派。<ref name=xiong/>

===追随孙中山 ===
[[中華民国|中華民国]]成立後，馬超俊任[[黎元洪|黎元洪]]的都督府顧問。但是黎元洪不断支持[[袁世凱|袁世凱]]，馬超俊对此表示反对，并展开批判活動，乃遭到收監。幸而经[[温宗尧|温宗尧]]、[[伍廷芳|伍廷芳]]营救，馬超俊很快获释。1914年（民国2年），袁世凯支持的[[广東省_(中華民国)|广東]]都督[[龙济光|龙济光]]手下的[[馬存发|馬存发]]被馬超俊派出刺客暗殺，馬超俊随后遭[[龙济光|龙济光]]追捕，流亡[[日本|日本]]。1916年春，在[[中華革命党|中華革命党]]馬超俊认识机师[[坂本寿一|坂本寿一]]<ref>见熊尚厚「馬超俊」133頁。那时, [[梅屋庄吉|梅屋庄吉]]向他们提供资金。</ref>，并他们俩一起参与创设[[滋賀县|滋賀县]]的西京[[八日市市|八日市]]飞行学校，训练革命派人士。1917年，馬超俊归国，策划在北京创办民間航空学校，但因[[張勳復辟|張勳復辟]]发生而未能实现。<ref name=xiong/><ref name=lgm/>

1917年9月，孫中山发动[[護法運動|護法運動]]，在[[广州|广州]]成立[[护法軍政府|护法軍政府]]，馬超俊参加，从事劳工運動的組織工作。他先后参与创办了[[南洋烟草公司|南洋烟草公司]]職工同志会、粤僑工界聯合会、广東機器工人維持会等工会组织。在[[旧桂系|旧桂系]]及[[陳炯明|陳炯明]]的紛争下，馬超俊领导工会组织坚决支持孫中山。后被孙中山委任为广东东路第二路游击司令。1923年起，先后任广东石井兵工廠廠長、[[陆海军大元帅大本营|陆海军大元帅大本营]]宣传委员会委员。1924年（民国13年），任[[中国国民党广州市党部|中国国民党广州市党部]]執行委員兼工人部部長。馬超俊坚持[[劳资協調|劳资協調]]的立场，反对孫中山推進三大政策。<ref name=xiong/><ref name=lgm/>

===从劳资協調到反蒋===
[[File:MaoChaoChun_馬超俊_MayorNanking_1935.png|thumb]]
孫中山逝世後的1925年（民国14年）春，馬超俊加入反共的[[孫文主義学会|孫文主義学会]]，担任总幹事。1926年至1927年间，赴[[美国|美国]]、[[加拿大|加拿大]]考察工人运动。1927年，[[蒋介石|蒋介石]]发动[[四一二政变|四一二政变]]后归国，出任[[南京国民政府|南京国民政府]]劳工局局長。同年8月，在[[上海市|上海]]创办《劳工時報》，倡导劳資合作，反对[[中国共产党|中国共产党]]方面的工会。<ref name=xiong/><ref name=lgm/>

1928年（民国17年）1月，馬超俊改任广东省政府委员兼农工厅厅长。同年秋，农工厅改为建設厅，馬超俊仍任厅長。1929年（民国18年），兼任中国国民党广州市党部宣传部長。在广州，他还曾任[[中国国民党广东省党部|中国国民党广东省党部]]整理委员、广州市党部执行委员等职。同年5月，作为中国代表出席[[国際劳工组织|国際劳工组织]]（ILO）会议。归国後，因同广東省政府主席[[陳銘枢|陳銘枢]]对立，辞去了在广東省的各職。同年，当选中国国民党候补中央执行委员。同年8月，任[[中華民国立法院|立法院]][[立法委員|立法委員]]、劳工法起草委員会召集人。此后作为中国国民党華北党務特派員被派往[[天津市|天津]]，从事反蒋派的对抗活動。同年冬，馬超俊回到南京，任中国国民党中央训练部民众训练处处長。<ref name=xiong/><ref name=lgm/>

馬超俊支持孫中山之子[[孫科|孫科]]，蒋介石软禁[[胡漢民|胡漢民]]之后，馬超俊和孫科共同从事反蒋活動。1931年[[九一八事变|九・一八事变]]发生后，中国国民党各派大团結。1931年，馬超俊当选中国国民党中央执行委员、中央民众运动指导委员会副主任委员。1931年（民国20年）末，蒋介石一度下野，孫科出任[[行政院長|行政院長]]，馬超俊被任命为[[南京市市长|南京市市长]]。但孫科迅速被迫辞去行政院長职务，馬超俊也随之失去南京市長一职。1934年（民国23年）12月，馬超俊任国民政府委員。1935年（民国24年）3月，馬超俊再度出任南京市長。<ref name=xiong/><ref name=lgm/><ref name=liu>{{Cite book|author = 劉寿林等編|title = 民国職官年表|year = 1995|publisher = 中華書局|isbn = 7-101-01320-1}}</ref>
[[File:馬超俊和林森.jpg|缩略图]]
===抗日战争、晩年 ===
1937年（民国26年）[[抗日战争|抗日战争]]爆发，日军占领南京，馬超俊逃往[[武漢市|武漢]]。此後，馬超俊从事抗战中的民衆及社会組織工作。1938年春，馬超俊任中国儿童救济总会理事長、全国慰劳总会副会長。4月，任中国国民党中央社会部副部長。1940年（民国29年），调任中国国民党中央組織部副部長。1942年（民国30年），在[[重慶|重慶]]和[[吴铁城|吴铁城]]等人共同创办南洋華僑总会。1943年，馬超俊就任该会理事長。1945年（民国34年）5月，当选中国国民党第六届中央執行委員。<ref name=xiong/><ref name=lgm/>

[[抗日战争|抗日战争]]结束后，馬超俊复任南京市市長，办理接收事務。1946年（民国35年）12月，任中国国民党中央農工部部長。同时他还兼任中国国民党中央财务委员会委员。1947年9月，在中国国民党六届四中全会上，当选中央執行委員会常務委員。1948年（民国37年）4月，在南京创立中華民国全国总工会。1948年，当选为[[第一届国民大会代表|第一届国民大会代表]]。[[第二次国共内战|第二次国共内战]]末期，馬超俊逃往[[台湾|台湾]]。1950年，被聘为[[总统府国策顧問|总统府国策顧問]]。此后，历任中国国民党中央紀律委員会委員、主任委员，華僑協会总会理事長，[[大陆救济总会|大陆救济总会]]常务理事，[[光復大陸設計研究委員会|光復大陸設計研究委員会]]委員，广東同乡总会理事長，[[中国国民党中央评议委员会|中国国民党中央评议委员会]]委员等职。1965年（民国54年），与台湾中国文化学院共同创办劳工研究所，任该所理事長。1969年9月退休后，被聘为[[总统府资政|总统府资政]]。<ref name=xiong/><ref name=lgm/><ref name=liu/>

1975年10月24日，[[蔣經國|蔣經國]]函電[[宋美齡|宋美齡]]：
{{quote|
「今為馬超俊先生九十華誕兒曾往賀」<ref>[[周美華|周美華]]、[[蕭李居|蕭李居]]編，《蔣經國書信集——與宋美齡往來函電》（上），[[台北|台北]]「[[國史館|國史館]]」出版，2009年，第338頁。</ref>
}}

1977年（民国66年）9月19日，馬超俊在[[台北市|台北市]]病逝，享年92岁（满90歳）。<ref name=xiong/><ref name=lgm/>
