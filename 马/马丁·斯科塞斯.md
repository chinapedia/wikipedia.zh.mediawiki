'''马丁·斯科塞斯'''（{{lang-en|'''Martin Scorsese'''}}，{{IPA-it|skorˈseːze|lang}}；{{bd|1942年|11月17日||}}）原名'''马丁·查尔斯·斯科塞斯'''（{{lang-en|'''Martin Charles Scorsese'''}}），美國[[電影導演|電影導演]]、監製、編劇、演員和電影歷史學家，生涯持續超過50年。史高西斯的作品主題針對西西里裔美國人身份、羅馬天主教的內疚和[[得救|救贖]]概念<ref>{{Cite web |url = http://www.adherents.com/people/ps/Martin_Scorsese.html |title = The Religious Affiliation of Director Martin Scorsese |language = en}}</ref>、[[大男子主義|大男子主義]]、現代犯罪及幫派衝突。他的電影也以暴力描寫和大量[[粗口|粗口]]而著名。

在[[美國新荷里活電影|美國新荷里活電影]]浪潮中，他是電影史上其中一個被普遍視為具有重要影響力的電影製作人。1990年，他創立了{{link-en|電影基金會 (美國)|The Film Foundation|電影基金會}}，一個專注於{{link-en|電影保護|Film preservation}}的非營利組織。2007年，他創立了{{link-en|世界電影基金會|World Cinema Foundation}}。他是[[AFI終身成就獎|AFI終身成就獎]]的得主，也曾獲得[[奧斯卡金像獎|奧斯卡金像獎]]、[[金棕櫚獎|金棕櫚獎]]、[[最佳導演獎_(康城影展)|康城影展最佳導演獎]]、[[銀獅獎_(電影)|銀獅獎]]、[[格萊美獎|格萊美獎]]、[[艾美獎|艾美獎]]、[[金球獎|金球獎]]、[[英國電影學院獎|英國電影學院獎]]和[[美國導演工會獎|美國導演工會獎]]。

他曾執導過多部里程碑電影，例如犯罪片《[[窮街陋巷|窮街陋巷]]》（1973年）、驚悚片《[[的士司機_(電影)|的士司機]]》（1976年）、傳記運動片《[[狂牛|狂牛]]》（1980年）、黑色喜劇《[[喜劇之王_(1983年電影)|喜劇之王]]》、犯罪片《[[盜亦有道|盜亦有道]]》（1990年）和《[[賭城風雲_(1995年電影)|賭城風雲]]》（1995年），他與演員及好友[[羅拔·迪尼路|羅拔·迪尼路]]在這些電影中合作。史高西斯也與演員[[里安納度·狄卡比奧|里安納度·狄卡比奧]]合作過五部電影，由《[[紐約風雲|紐約風雲]]》（2002年）至最近的《[[華爾街狼人|華爾街狼人]]》（2013年）。

史高西斯另外的電影包括音樂會電影《{{link-en|最後華爾茲|The Last Waltz}}》（1978年）、黑色喜劇《[[三更半夜|三更半夜]]》（1985年）、《[[基督的最後誘惑_(電影)|基督的最後誘惑]]》（1988年）、心理驚悚片《[[海角驚魂|海角驚魂]]》（1991年）和《[[不赦島|不赦島]]》（2010年）、傳記片《[[娛樂大亨|娛樂大亨]]》（2004年）、歷史冒險片《[[雨果的巴黎奇幻歷險|雨果的巴黎奇幻歷險]]》（2011年）以及史詩宗教片《[[沉默_(電影)|沉默]]》（2016年）。他的作品也包括[[HBO|HBO]]電視劇《[[酒私風雲|酒私風雲]]》和《[[黑膠時代|黑膠時代]]》的首集。他憑著犯罪電影《[[無間道風雲|無間道風雲]]》（2006年）獲[[第79屆奧斯卡金像獎|第79屆奧斯卡金像獎]]最佳導演。

== 早年 ==
[[File:Cuccia-Scorsese-Deseta.jpg|thumb]]]]

馬丁·斯科塞斯的雙親是[[義大利|義大利]][[西西里島|西西里島]]人，兩人都是在[[紐約市|紐約市]]出生。父親是[[查爾斯·史高西斯|查爾斯·史高西斯]]，是一名熨衣員及演員，母親是[[嘉芙蓮·史高西斯|嘉芙蓮·史高西斯]]，是名裁縫師及女演員，他們同樣在[[曼克頓|曼克頓]]的{{link-en|服裝區|Garment District, Manhattan}}工作。<ref>{{Cite web |url = http://www.filmreference.com/film/26/Martin-Scorsese.html |title = Martin Scorsese Biography (1942-) |publisher = Film Reference |language = en}}</ref>他們在[[皇后區|皇后區]]居住並且生下馬丁，後來移居到曼克頓[[小意大利_(曼哈顿)|小意大利]]區。史高西斯從小就在[[天主教|天主教]]的環境成長，而他最初是想成為一名[[神父|神父]]。小時候患有[[哮喘|哮喘]]，不能夠與其他孩子正常玩耍，所以他的父母和哥哥經常帶他到電影院看電影，由此他在自己的人生中發展出對電影的熱情。<ref>{{cite news |language =en |url=http://movies.nytimes.com/person/110533/Martin-Scorsese/biography |title=Martin Scorsese | publisher= The New York Times}}</ref>青少年時期的史高西斯經常去商店租看{{link-en|鮑爾與普萊斯柏格|Powell and Pressburger}}導演的《{{link-en|霍夫曼的故事 (電影)|The Tales of Hoffmann (film)|霍夫曼的故事}}》（''The Tales of Hoffmann''），他是僅有兩個會租借這電影膠片的人之一，另一個人是《[[活死人之夜|活死人之夜]]》的導演[[喬治·安德魯·羅梅羅|喬治·安德魯·羅梅羅]]<ref>{{cite web|url=http://www.eyeforfilm.co.uk/feature/2015-07-08-george-a-romero-talks-zombies-religion-and-genocide-in-karlovy-vary-feature-story-by-richard-mowe|title=Romero - master of the macabre|author = Richard Mowe|publisher=Eye for Film|language =en}}</ref>。

== 生涯 ==
=== 早期生涯 ===
斯科塞斯唸[[布朗克斯|布朗克斯]]區的 {{link-en|Cardinal Hayes High School|Cardinal Hayes High School}} 中學，然後在1960年進入知名的[[紐約大學|紐約大學]]唸電影系，大學時期曾制作三部短片，《{{link-en|你這麼好的女孩在這裡幹什麼|What's a Nice Girl Like You Doing in a Place Like This?}}》（1963年）、《{{link-en|那不僅是你，穆瑞！|It's Not Just You, Murray!}}》（1964年）及《{{link-en|剃鬚記|The Big Shave}}》（1967年）。<ref>{{Cite web |url = http://www.encyclopedia.com/topic/Martin_Scorsese.aspx |title = SCORSESE, Martin |date = 2001 |publisher = Encyclopedia.com |language = en }}</ref><ref>{{Cite web |url = http://nofilmschool.com/2013/08/three-student-films-by-martin-scorsese |title = Watch Three Student Films Made by Martin Scorsese While at NYU |author = Justin Morrow |date = 2013-08-24 |publisher = No Film School |language = en}}</ref>1966年斯科塞斯拿到電影碩士，1967年他拍攝了第一部長篇的黑白電影《''I Call First''》，後來改名為《[[誰在敲我的門|誰在敲我的門]]》（''Who's That Knocking at My Door''），男演員是[[夏菲·基圖|夏菲·基圖]]，而電影剪接是{{link-en|塞爾瑪·斯昆梅克|Thelma Schoonmaker}}，之後他們兩人成為史高西斯的長期合作伙伴。在1968年到1970年之間史柯西斯留在紐約大學電影系任教。

=== 1970年代 ===
他與一些有影響力的1970年代「電影小子」成為朋友，包括[[白賴仁·龐馬|白賴仁·龐馬]]、[[法蘭斯·哥普拉|法蘭斯·哥普拉]]、[[佐治·魯卡斯|佐治·魯卡斯]]及[[史提芬·史匹堡|史提芬·史匹堡]]，而白賴仁·龐馬則介紹史高西斯給[[羅拔·迪尼路|羅拔·迪尼路]]認識。<ref> {{Cite book | author = Veronika Adams | title = Martin Scorsese Unauthorized Biography | publisher = Ebook.GD Publishing | date = 2011-01-27 | ISBN = 1-61323-010-9 | url = https://books.google.co.uk/books?id=JdSDUfiwsVQC&pg=PP3&dq=Martin+Marcantonio+Luciano+Scorsese&hl=en&ei=FyfDTem-C4Ot8QPr0MHFBQ&sa=X&oi=book_result&ct=result#v=onepage&q=Martin%20Marcantonio%20Luciano%20Scorsese&f=false | language = en | deadurl = yes | archiveurl = https://web.archive.org/web/20160101064317/https://books.google.co.uk/books?id=JdSDUfiwsVQC&pg=PP3&dq=Martin+Marcantonio+Luciano+Scorsese&hl=en&ei=FyfDTem-C4Ot8QPr0MHFBQ&sa=X&oi=book_result&ct=result#v=onepage&q=Martin%20Marcantonio%20Luciano%20Scorsese&f=false | archivedate = 2016-01-01 }} </ref>在這段期間，斯科塞斯曾經在1970年擔任傳奇演唱會[[紀錄片|紀錄片]]《{{link-en|胡士托 (電影)|Woodstock (film)|胡士托}}》的[[助理導演|助理導演]]，並遇見了演員及導演[[尊·加沙域|尊·加沙域]]，他也成為史高西斯的一個親密朋友和導師。

1972年，史高西斯為[[B級片|B級片]]監製[[羅傑·科曼|羅傑·科曼]]執導[[經濟大蕭條|經濟大蕭條]]時代的電影《[[冷血霹靂火|冷血霹靂火]]》，科曼也幫助過多名導演例如[[法蘭斯·哥普拉|法蘭斯·哥普拉]]、[[占士·金馬倫|占士·金馬倫]]和{{link-en|John Sayle|John Sayle}}開始他們的職業生涯。<ref>{{Cite web |url = http://www.nytimes.com/movie/review?res=980CE1DF1631E63BBC4052DFBE668389669EDE |title = The Screen:'Boxcar Bertha' Tops Local Double Bill |author = Howard Thompson |date = 1972-08-18 |publisher = The New York Times |language = en}}</ref>科曼教史高西斯用最少的錢或時間拍攝娛樂電影，為年輕導演將會拍攝《[[窮街陋巷|窮街陋巷]]》的挑戰而準備。在電影上映後，加沙域鼓勵史高西斯製作他想製作的電影，而不是別人的項目。

《窮街陋巷》是史高西斯、迪尼路和基圖的一個突破，電影被有影響力的影評家{{link-en|保琳·凱爾|Pauline Kael}}所擁護。到現在，史高西斯的招牌風格已經到位：男子氣概、血腥暴力、天主教罪咎和救贖、真實的紐約現場（雖然《窮街陋巷》大多數實際是在洛杉磯拍攝）、一連串剪接和用當代音樂作配樂 。雖然這部電影是很創新，它緊張的氣氛、前衛的紀錄片風格和真實的街頭指導卻向導演加沙域、{{link-en|薩繆爾·富勒|Samuel Fuller}}和早期的[[尚盧·高達|尚盧·高達]]表示感激。<ref>{{Cite web |url = http://www.washingtonpost.com/wp-dyn/articles/A99967-1991Nov24.html |title = Scorsese, Master Of The Rage |author = Hal Hinson |date = 1991-11-24 |publisher = The Washington Post |language = en}}</ref>

1974年，女演員[[艾倫·鮑絲汀|艾倫·鮑絲汀]]選擇了史高西斯在《[[愛莉絲不住在這裡|愛莉絲不住在這裡]]》中指導她，為此她贏得奧斯卡最佳女主角獎。雖然備受好評，電影在導演早期的職業生涯中仍是個反常的，因為電影是專注於一名主要的女性角色。其後他回到小意大利區探索他的種族根源，史高西斯接下來的作品《{{link-en|意大利裔美國人 (紀錄片)|Italianamerican|意大利裔美國人}}》，是一部由他的父母查爾斯和凱瑟琳·史柯西斯演出的紀錄片。

下一部就是在1976年的《[[的士司機_(電影)|的士司機]]》- 史高西斯的黑暗、城市噩夢使一個孤獨的男人慢慢地墮落到精神錯亂。這部電影讓史高西斯成為一個熟練的電影製作人，並引起了電影攝影師{{link-en|邁克爾·查普曼|Michael Chapman (cinematographer)}}的注意，他的風格傾向於高對比度、強烈的顏色和復雜的鏡頭移動。這部電影主演是[[羅拔·迪尼路|羅拔·迪尼路]]，飾演麻煩多的和有精神病的{{link-en|崔維斯·拜寇|Travis Bickle}}（Travis Bickle）。電影的聯合主演則有[[茱迪·科士打|茱迪·科士打]]，飾演一名有高度爭議的未成年妓女角色，而[[夏菲·基圖|夏菲·基圖]]飾演她的皮條男Matthew，稱為「"Sport"」。《的士司機》還標誌著史高西斯和作家[[保羅·許瑞德|保羅·許瑞德]]之間的一系列合作的開始，許瑞德受到包括將會成為刺客的{{link-en|Arthur Bremer|Arthur Bremer}}的日記和法國導演[[羅伯特·布列松|羅伯特·布列松]]的電影《''{{link-en|Pickpocket (film)|Pickpocket (film)|Pickpocket}}''》所影響。電影在上映後便立即出現了爭議，五年後，當[[約翰·欣克利|約翰·欣克利]]對當時的總統[[朗奴·列根|朗奴·列根]]企圖進行暗殺的時候，《的士司機》再次為人所矚目。欣克利後來把他的行為歸咎茱迪·科士打的《的士司機》角色（在電影，迪尼路的角色{{link-en|崔維斯·拜寇|Travis Bickle}}（Travis Bickle）企圖暗殺參議員）。<ref>{{Cite web |url = https://www.theguardian.com/film/2006/jul/06/features.geoffreymacnab |title = 'I was in a bad place' |author = Geoffrey Macnab |date = 2006-07-06 |publisher = The Guardian |language = en}}</ref>《的士司機》在1976年康城影展上獲得了[[金棕櫚獎|金棕櫚獎]]<ref>{{Cite web |url = http://www.festival-cannes.fr/en/archives/ficheFilm/id/3E0FBC86-DB8F-4AA2-8FF7-B39C1F31DB3F/year/1976.html |title = Festival Archives: TAXI DRIVER |publisher = Festival de Cannes |language = en}}</ref>，也獲得了四項[[奧斯卡金像獎|奧斯卡]]提名，包括[[奧斯卡最佳影片獎|最佳電影]]。

《的士司機》的成功鼓勵了史高西斯推動他的第一個大預算的項目：極為風格化的音樂電影《[[紐約，紐約_(電影)|紐約，紐約]]》。這對史高西斯的家鄉和經典荷里活音樂電影的致敬來說是一個票房失敗。這部電影是導演與羅拔·迪尼路的第三次合作，聯合主演則是[[麗莎·明尼利|麗莎·明尼利]]。電影在今天最值得記著的是電影標題名主題曲，由[[法蘭·仙納杜拉|法蘭·仙納杜拉]]普及宣傳。雖然具有史高西斯的慣常視覺派頭和風格化的演奏，許多評論家則認為這部電影的封閉氣氛較他的早期作品為沉重。雖然反響不佳，但電影受到一些影評人的正面肯定。Richard Brody在《[[紐約客|紐約客]]》上寫道：「對於史高西斯，一個終身的電影迷來說，紐約的精髓可以在經典的荷里活電影中找到出來的。明顯地，他對音樂電影和有營養的浪漫音樂劇的黃金時代的回顧致敬，成為他最隨心所欲和個人的電影之一。」<ref>{{Cite web |url = http://www.newyorker.com/arts/events/revivals/2008/01/28/080128gomo_GOAT_movies_brody |title = TOP OF THE HEAP |author = Richard Brody |date = 2008-01-28 |publisher = The New Yorker |language = en |deadurl = yes |archiveurl = https://web.archive.org/web/20080124000216/http://www.newyorker.com/arts/events/revivals/2008/01/28/080128gomo_GOAT_movies_brody |archivedate = 2008-01-24 }}</ref>

《紐約，紐約》令人失望的反響使史高西斯感到沮喪。在這個階段，導演也產生了嚴重的[[可卡因|可卡因]]上癮。然而，他尋找到創造的動力來製作非常受注視的《{{link-en|最後華爾茲|The Last Waltz}}》，這記錄了[[樂隊合唱團|樂隊合唱團]]最後的音樂會。它在三藩市的{{link-en|溫特蘭德舞廳|Winterland Ballroom}}舉行，並成為了在一場音樂會上有大陣容的著名嘉賓表演者出場的紀錄片之一，包括[[巴布·狄倫|巴布·狄倫]]、[[尼爾·楊|尼爾·楊]]、[[靈高·史達|靈高·史達]]、[[馬迪·沃特斯|馬迪·沃特斯]]、[[瓊尼·米歇爾|瓊尼·米歇爾]]、[[范·莫里森|范·莫里森]]、[[保羅·巴特菲爾德|保羅·巴特菲爾德]]、{{link-en|尼爾·戴門|Neil Diamond}}、[[羅尼·伍德|羅尼·伍德]]和[[艾瑞克·克萊普頓|艾瑞克·克萊普頓]]。然而，史高西斯對其他項目的承諾推遲了電影的發行，直到1978年為止。

'''1970年代的其他作品'''

另一部史高西斯導演的紀錄片名為《''{{link-en|American Boy: A Profile of Steven Prince|American Boy: A Profile of Steven Prince|American Boy}}''》，也出現在1978年，主題集中在Steven Prince上，他曾出現在《的士司機》中飾演cocky槍推銷員。史高西斯還為紀錄片《''{{link-en|Elvis on Tour|Elvis on Tour}}''》幫助提供片段。1977年，他執導百老匯音樂劇《''{{link-en|The Act (musical)|The Act (musical)|The Act}}''》，由麗莎·明尼利主演。<ref>{{Cite web |url = http://flavorwire.com/475332/martin-scorseses-weirdest-projects/2 |title = Martin Scorsese's Weirdest Projects |author = Jason Bailey |date = 2014-09-03 |publisher = Flavorwire |language = en}}</ref>

=== 1980年代 ===
根劇幾個描述（包括史高西斯的），羅拔·迪尼路實際上挽救了史高西斯的生命，他說服史高西斯踢走他的[[可卡因|可卡因]]成癮問題，以製作他的高評價電影《[[狂牛|狂牛]]》。他把精力投入到製作這部中量級拳擊冠軍{{link-en|Jake LaMotta|Jake LaMotta}}的暴力傳記中，稱之為[[特別攻擊隊|神風式]]電影製作法。該電影被廣泛地視為一部傑作，並被英國的《''[[Sight_&_Sound|Sight & Sound]]''》雜誌評選為1980年代最偉大的電影。它得到了八項奧斯卡提名，包括最佳電影，[[奧斯卡最佳男主角獎|最佳男主角]]和史高西斯的第一個[[奧斯卡最佳導演獎|最佳導演]]。迪尼路贏得男主角獎，與{{link-en|塞爾瑪·斯昆梅克|Thelma Schoonmaker}}一樣，她贏得最佳剪接，但最佳導演則去了[[羅拔·烈福|羅拔·烈福]]的《[[普通人|普通人]]》。從這部作品開始，史高西斯的電影在宣傳資料上總是標記為「馬田·史高西斯電影」（A Martin Scorsese Picture）。《狂牛》以高對比度的黑白拍攝，是史高西斯的風格達到頂峰時期：《的士司機》和《紐約，紐約》使用[[表現主義|表現主義]]的元素來復制心理學的觀點，但在這部的風格則被帶到新的極端程度，利用廣泛的慢動作、複雜的跟踪鏡頭，以及透視的過度失真（例如，拳擊臺的大小會從搏擊至搏擊中改變）。主題方面也一樣，從《窮街陋巷》和《的士司機》繼續下去：不安全的男性、暴力、內疚和贖罪。

雖然《狂牛》的劇本都歸功於保羅·許瑞德和Mardik Martin（之前他們合作編劇《窮街陋巷》），但是完成的劇本與許瑞德的原始草案有很大不同。它被包括{{link-en|傑伊·庫克斯|Jay Cocks}}在內的多位編劇重寫了好幾次（後來他繼續合作編劇史高西斯的電影《[[心外幽情|心外幽情]]》和《[[紐約風雲|紐約風雲]]》）。最終草案主要由史高西斯和羅拔·迪尼路撰寫。[[美國電影學會|美國電影學會]]選擇了《狂牛》作為排名前10大最佳體育電影中的第一位。1997年，該學會將《狂牛》評為在[[AFI百年百大電影|AFI百年百大電影]]的第24位最偉大的電影。2007年，他們將《狂牛》評選為在[[AFI百年百大電影_(2007年版)|AFI百年百大電影10周年版]]上的第4位最偉大的電影。

史高西斯的下一個項目就是他與羅拔·迪尼路第五次合作的《[[喜劇之王_(1983年電影)|喜劇之王]]》（1983年）。 它是[[諷刺|諷刺]]世界上的媒體和名人，其中心角色是一名麻煩的孤獨者，他透過犯罪行為（[[綁架|綁架]]）而諷刺地變得出名。這部電影明顯偏離了他之前很有激情堅定的電影。視覺上，它的動態比史高西斯以前所發展的風格非常少，經常使用一個靜態鏡頭和長鏡頭。他以前的作品是表現主義，在這裡給予的是幾乎完全的[[超現實主義|超現實主義]]。它仍然具有許多史高西斯的標記特徵。然而，《喜劇之王》在票房上失敗了，但自從在電影上映以來的歲月裡，電影越來越受到影評人的好評。德國導演[[溫·韋德斯|溫·韋德斯]]將其列為他最喜歡的15部電影之中。<ref>{{Cite web |url = http://www.wim-wenders.com/news_reel/2002/apr02-directorsonfilm.htm |title = Wim Wenders answers questions for Andrew J. Rausch
for his book REEL PEOPLE: 50 DIRECTORS ON MOVIES |publisher = Internet Archive |language = en |deadurl = yes |archiveurl = https://web.archive.org/web/20020618113722/http://www.wim-wenders.com/news_reel/2002/apr02-directorsonfilm.htm |archivedate = 2002-06-18 }}</ref>

《[[三更半夜|三更半夜]]》（1985年），史高西斯的美學轉變回到一個簡化的，幾乎「地下的」電影製作風格。以極低的預算拍攝，位置在晚上的曼哈頓[[紐約蘇豪區|蘇豪區]]附近，這部電影是一部黑色喜劇，關於一個溫和的紐約文字處理員（{{link-en|格里芬·鄧尼|Griffin Dunne}} 飾演）在一連串不幸的晚上的遭遇，並由迥然不同的演員客串，例如{{link-en|特瑞·加爾|}}和{{link-en|切奇與鐘|Cheech & Chong}}。

隨著1987年[[米高·積遜|米高·積遜]]音樂錄影帶"{{link-en|Bad (Michael Jackson song)|Bad (Michael Jackson song)|Bad}}"，在1986年史高西斯製作了《[[金錢本色|金錢本色]]》，這是[[罗伯特·罗森|罗伯特·罗森]]執導與[[保羅·紐曼|保羅·紐曼]]主演的《[[江湖浪子_(電影)|江湖浪子]]》（1961年）的續集，共同演員有[[湯·告魯斯|湯·告魯斯]]。雖然秉承了史高西斯的既定風格，但《金錢本色》卻是導演第一次正式初次涉足[[主流|主流]]的電影製作。這部電影最終為演員保羅·紐曼贏得了一項奧斯卡獎，並給了史高西斯影響力來支持他的一個長久目標計劃：《基督的最後誘惑》。

1983年，史高西斯開始了一個長期渴望的個人項目，《[[基督的最後誘惑_(電影)|基督的最後誘惑]]》，基於由[[尼可斯·卡山札基|尼可斯·卡山札基]]寫的1951年小說（1960年英語翻譯）。{{link-en|芭芭拉·荷西|Barbara Hershey}}回憶說，在他們拍攝《[[列車女賊|列車女賊]]》的時候，她向史高西斯介紹了這本小說。這部電影準備在[[派拉蒙電影|派拉蒙電影]]旗下拍攝，但在主要攝影開始之前不久，派拉蒙因為宗教團體的壓力而抽掉了這個項目。在這個被拋棄的1983年版本中，[[艾丹·奎因|艾丹·奎因]]被揀選飾演耶穌，而[[史田|史田]]則飾演[[本丟·彼拉多|本丟·彼拉多]]（在1988年的版本中，這些角色分別由[[威廉·達福|威廉·達福]]和[[大衛·寶兒|大衛·寶兒]]飾演）。他在1980年代中期與商業荷里活調情後，史高西斯與[[保羅·許瑞德|保羅·許瑞德]]改編過的《基督的最後誘惑》劇本於1988年重新回到個人的電影製作。根據[[尼可斯·卡山札基|尼可斯·卡山札基]]有爭議性的1960年著作，本書重述了[[耶穌|基督]]以人類身分的一生而不是神性的關係。全世界抗議電影的[[褻瀆|褻瀆]]，甚至在電影發布之前也引起了大規模狂怒，使其有效地將一個低預算獨立電影變成媒體轟動。大多數爭議集中在電影的最後段落，其中描繪了基督在十字架上期間於[[撒旦|撒旦]]引起的[[幻覺|幻覺]]中與[[抹大拉的馬利亞|抹大拉的馬利亞]]結婚和養育一家人。

回顧過去的爭議，《基督的最後誘惑》獲得了重要的好評，並仍然是史高西斯的標準中的一部重要作品：一個明確的嘗試搏鬥精神支撐了他的電影直到那一點。導演繼續得到他的第二次最佳奧斯卡導演獎提名（再次失敗，這一次敗給了《[[手足情未了|手足情未了]]》的[[巴瑞·李文森|巴瑞·李文森]]）。

* 1990年參演[[日本|日本]]導演[[黑澤明|黑澤明]]的《[[夢_(1990年電影)|夢]]》，飾演[[荷蘭|荷蘭]][[畫家|畫家]][[梵谷|梵谷]]。
* 1997年[[美国电影学会|美国电影学会]]頒[[AFI终身成就奖|AFI终身成就奖]]給斯科塞斯。
* 1998年斯科塞斯擔任[[坎城影展|坎城影展]]評審團主席。當年雖將金棕櫚獎頒給[[希臘|希臘]]導演[[安哲羅普洛斯|泰奧·安哲羅普洛斯]]的《永遠的一天》，但最驚人的影像莫過於當評審團大獎頒給[[義大利|義大利]]導演[[羅貝托·貝尼尼|羅貝托·貝尼尼]]的《[[美麗人生|美麗人生]]》時，羅貝托·貝尼尼先是在舞台上向史柯西斯下跪，然後再抱起斯科塞斯轉一圈的動人畫面。
* 2006年的影片《[[神鬼無間|神鬼無間]]》乃翻拍自香港《[[無間道|無間道]]》系列之第一部，並令其得到[[第79屆奧斯卡金像獎|第79屆奧斯卡金像獎]]最佳導演獎。
* 2010年1月18日，马丁·斯科塞斯荣获[[第67届金球奖|第67届金球奖]]终身成就奖。
* 2012年1月15日，斯科塞斯憑著《[[雨果的巴黎奇幻歷險|雨果的巴黎奇幻歷險]]》再次獲得[[第69屆金球獎|第69屆金球獎]]最佳導演獎。其後此電影更入圍[[第84屆奧斯卡金像獎|第84屆奧斯卡金像獎]]最佳電影等11項提名，成為本屆提名最多的電影。

馬丁·斯科塞斯在進行劇情長片的同時，也拍了幾部跟[[義大利|義大利]]移民、[[義大利新寫實主義|義大利新寫實主義]]、音樂（[[爵士樂|爵士樂]]、[[藍調|藍調]]）的紀錄片。

== 個人生活 ==
=== 婚姻狀況 ===
史高西斯經歷過五次婚姻。第一任的妻子是Laraine Marie Brennan，於1965年與史高西斯結婚，育有一女Catherine，在1971年離婚。1976年，與美國作家{{link-en|Julia Cameron|Julia Cameron}}結婚，育有一女Domenica Cameron-Scorsese，但婚姻只維持了一年，他們的女兒於《[[海角驚魂|海角驚魂]]》演出小角色，後來繼續擔任演員及導演。<ref>{{Cite web |url = http://www.eonline.com/news/59163/scorsese-s-family-business |title = Scorsese's Family Business |author = Marc Malkin |date = 2007-04-17 |publisher = E! |language = en}}</ref>史高西斯於1979年與女演員[[伊莎貝拉·羅塞里尼|伊莎貝拉·羅塞里尼]]結緍，至1983年離婚。<ref>{{Cite web |url = http://www.theguardian.com/film/2006/may/01/features.features11 |title = Daddy's girl |author = Dan Halpern |date = 2006-05-01 |publisher = The Guardian |language = en}}</ref>1985年，與電影監制[[Barbara_De_Fina|Barbara De Fina]]結緍，1991年離婚收場。1999年史高西斯與Helen Schermerhorn Morris結緍，其婚姻維持至今，育有一女Francesca，曾在《[[娛樂大亨|娛樂大亨]]》及《[[無間道風雲|無間道風雲]]》中出現。<ref>{{Cite web |url = http://marriage.about.com/od/entertainmen1/p/mscorsese.htm |title = Martin Scorsese and Helen Morris Marriage Profile |author = Sheri Stritof |date = 2014-12-17 |publisher = About.com |language = en}}</ref>

=== 與羅拔·迪尼路的合作 ===
史高西斯經常與[[羅拔·迪尼路|羅拔·迪尼路]]合作，目前為止已合作了八部電影。他們由1970年代初互相認識後，史高西斯開始選迪尼路演出1973年的《[[窮街陋巷|窮街陋巷]]》。三年後，迪尼路在《[[的士司機_(電影)|的士司機]]》中擔任主角。他於1977年再次演出史高西斯的電影《[[紐約，紐約_(電影)|紐約，紐約]]》，但票房慘淡。然而，他們的合作關係繼續維持到1980年代，包括《[[狂牛|狂牛]]》及《[[喜劇之王_(1983年電影)|喜劇之王]]》，前者更獲得高度成功。1990年，迪尼路主演《[[盗亦有道|盗亦有道]]》，該電影帶來一致好評，其後也在1991年的《[[海角驚魂|海角驚魂]]》及1995年的《[[賭城風雲_(1995年電影)|賭城風雲]]》中演出。<ref>{{Cite web |url = http://www.slashfilm.com/robert-de-niro-and-martin-scorsese-reteam-for-new-gangster-movie/ |title = Robert De Niro and Martin Scorsese Reteam for New Gangster Movie |author = Peter Sciretta |date = 2007-06-19 |publisher = /FILM |language = en}}</ref>他們兩人也於2004年[[配音|聲演]]動畫電影《[[鯊膽大話王|鯊膽大話王]]》<ref>{{cite news |language = en |author = Paul Clinton |url = http://edition.cnn.com/2004/SHOWBIZ/Movies/09/30/review.shark/ |title = Review: 'Shark Tale' goes swimmingly |publisher = CNN |date = 2004-10-01}}</ref>。

=== 政治立場 ===
馬田·史高西斯是美國民主黨的長期支持者。<ref>{{Cite web |url = http://www.cinemablend.com/television/Martin-Scorsese-Direct-Bill-Clinton-TV-Documentary-50511.html |title = Martin Scorsese To Direct Bill Clinton TV Documentary |author = Leslie Kasperowicz |date = 2013 |publisher = Cinema Blend |language = en}}</ref>
