'''马尼拉'''（{{lang-tl|Maynila}}，{{lang-en|Manila}}），又稱'''岷里拉'''，是[[菲律宾|菲律宾]]首都，位於菲律宾的最大岛—[[呂宋島|呂宋島]][[馬尼拉灣|馬尼拉灣]]的東岸；今為菲國第二大城，人口有150萬；乃全國經濟、文化、教育和工業中心，現為人口達1000萬的[[馬尼拉大都會|馬尼拉大都會]]區之發展核心。2013年4月，馬尼拉人口排名全球第6名，達2120萬人<ref>[http://dailynews.sina.com/bg/news/usa/usnews/chinesedaily/20130408/06114426384.html 全球人口最多30大城 洛城上榜]</ref>。在同屬此都會區的眾多城市中，僅[[奎松市|奎松市]]人口多於馬尼拉市。马尼拉之名在西班牙殖民時，因雙方語言不通的誤解而命名為：Maynilad，起源於{{lang-tl|May}}是''這裡有''的意思，{{lang-tl|nilad}}（[[瓶花木屬|瓶花木屬]]）是一种生长着白豔花朵、可作為染料的湿地树，許多觀賞過的人們都以「那些白色的花开着，便像一双双亮晶晶的眼睛」形容它，兩字合起來為{{lang-tl|Maynilad}}，本意為[[他加祿語|他加祿語]]的''這裡有白色的花''。<ref name = "馬尼拉牆中城 尋百年遺跡_2015"> {{ cite web | title = 馬尼拉牆中城 尋百年遺跡 | url = http://www.cna.com.tw/topic/newsworld/63-1/201509160002-1.aspx | publisher = 中央通訊社 | author = 文、攝影／林行健 | language = zh-tw | date = 2015-09-16 }} </ref><ref name = "異國情調 馬尼拉採擷 - 副刊 - 自由時報電子報_2006"> {{ cite web | title = 異國情調 馬尼拉採擷 - 副刊 - 自由時報電子報 | url = http://news.ltn.com.tw/news/supplement/paper/52548 | publisher =  自由時報 | language = zh-tw | date = 2006-01-11 }} </ref>她所屬的這個都會區，菲律賓官方稱為'''馬尼拉大都會'''（Metro Manila），一般外國人和非定居者通常簡稱以馬尼拉。[[馬尼拉大都會發展委員會|全區]]由17座城市和直轄市所組成。

==历史==
{{Expand-section}}
===前西班牙时期===
在[[宋代|宋代]]称为'''蒲哩噜'''，最早见宋代《[[诸番志|诸番志]]》：“蒲哩噜与三屿联属，聚落差盛，人多猛悍，好攻劫。海多卤股之石，搓牙如枯木芒刃，銛于剑戟，舟过其侧，预曲折以避之。产青琅玕、珊瑚树，然绝难得。风俗博易，与三屿同”<ref>宋《[[諸番志|諸番志]]·三嶼蒲哩嚕》</ref>。16世紀中葉，馬尼拉只是一個在帕西河（Pasig River）岸邊、信仰著[[回教|回教]]的小漁村。

===西班牙殖民地时期===
1570年，[[西班牙|西班牙]]人抵達馬尼拉。1595年，[[西班牙|西班牙]]人公告馬尼拉成為菲律賓群島的首都。1898年[[美西戰爭|美西戰爭]]，西班牙戰敗，馬尼拉改由美國治理。

===美国殖民时期===

===二战时期===
1942年1月2日，馬尼拉在[[太平洋戰爭|太平洋戰爭]]中被日軍所據。1945年2月23日美軍奪回了這座城市，不過未能阻止[[馬尼拉大屠殺|馬尼拉大屠殺]]。在二次大戰攻防期間，菲律賓大部份的城市都被摧毀，但戰後也大部份迅速被重建。
===当前===
1975年，馬尼拉和周圍城市組成[[馬尼拉大都會|馬尼拉大都會]]。今日的马尼拉被「全球化与世界级城市研究小组与网络」评为[[世界级城市|世界级城市]]，并且是全球第二十四大消费都市。

== 地理 ==
===氣候===
馬尼拉被歸類於[[柯本氣候分類法|柯本氣候分類法]]下的[[熱帶乾濕季氣候|熱帶乾濕季氣候]]。和菲律賓的其他地區一樣，馬尼拉全境位於熱帶，四季如夏。由於其緯度接近赤道，氣溫的變化範圍相當狹小，鮮少到達{{convert|20|°C|°F}}以下或{{convert|38|°C|°F}}以上。然而其溼度相當高。在十二月下旬至五月有顯著的[[旱季|旱季]]，其餘時間則為較為氣溫較為溫和的[[雨季|雨季]]。在雨季時每天的降雨量常大量集中於短暫的時間內，鮮少下一整天的雨。颱風在六月到九月的期間來襲，並會在城市的部分地區造成水災。<ref>{{cite web |url=http://jeepneyguide.com/manila |title=Manila |publisher=[[Jeepneyguide|Jeepneyguide]] |accessdate=3-12-12 |deadurl=yes |archiveurl=https://web.archive.org/web/20120621053551/http://jeepneyguide.com/Manila |archivedate=2012-06-21 }}</ref>

== 市區簡介 ==
[[File:Big_Manila.jpg|thumb]]
马尼拉位于[[吕宋岛|吕宋岛]]西部，马尼拉湾东岸、帕西河的入海口，帕西河横贯全城；位于[[纳沃塔斯|纳沃塔斯]]市、[[加洛坎市|加洛坎市]]之南；[[奎松市|奎松市]]之西南；[[仙范|仙范]]、[[曼达路永市|曼达路永市]]之西；[[马卡迪市|马卡迪市]]之西北；[[巴萨市|巴萨市]]之北。

除了[[港區_(馬尼拉)|港區]]（Port Area）地区外，馬尼拉每一個市辖区均有教堂，其中有些教堂已成为区內市民的專属禮拜教堂；和[[马尼拉大都会|马尼拉大都会]]內眾城比鄰。菲律賓總統府[[馬拉坎南宮|馬拉坎南宮]]亦位於馬尼拉市，發揮著總統辦公室和總統官邸的中樞功能。
