'''马玉涛'''（{{bd|1936年|2月|||catIdx=Ma,马}}）女，[[山西省|山西省]][[保德县|保德县]]人，[[中华人民共和国|中华人民共和国]]女高音歌唱家，[[一级演员|一级演员]]，[[中国人民解放军|中国人民解放军]]一级文职干部<ref name=baode/>。

== 生平 ==
1936年2月，马玉涛出生于[[中国|中国]][[山西省|山西省]][[保德县|保德县]]。自幼受河曲民歌以及[[二人台|二人台]]音乐的熏陶。1950年参加[[中国人民解放军|中国人民解放军]]，随后进入[[绥远军区|绥远军区]]文工团任演员。不久调到[[华北军区|华北军区]]文工团合唱队学习声乐。1955年调入[[北京军区|北京军区]]文工团任独唱演员。1956年加入[[中国共产党|中国共产党]]。此后历任该团合唱队教导员、北京军区战友歌舞团副团长。后来成为北京军区战友歌舞团国家一级演员，并任北京军区战友歌舞团艺术指导<ref name=baode>{{Cite web |url = http://www.sxbdxw.com/info/1144/3082.htm |title = 马玉涛 |author =  |date = 2016-12-15  |publisher =  保德新闻网 |language =  |accessdate =  |quote =  }}</ref><ref name=baotou>{{Cite web |url = http://www.baotou.gov.cn/info/1202/15827.htm |title =人物春秋 马玉涛  |author =  |date =  |publisher = 包头市人民政府 |language =  |accessdate =2017-08-27  |quote =  }}</ref><ref name=zqzx/>。

1950年代后期，马玉涛三次到[[上海|上海]]声乐研究所进修。师从[[林俊卿|林俊卿]]，并曾向[[沈湘|沈湘]]、[[郭淑珍|郭淑珍]]、[[王福增|王福增]]以及[[王昆|王昆]]、[[贾桂兰|贾桂兰]]、[[常香玉|常香玉]]等人学习声乐、民歌和戏曲<ref name=baode/><ref name=baotou/>。还曾向[[丑金刚|丑金刚]]、[[李桂枝|李桂枝]]学过[[河北梆子|河北梆子]]<ref name=baotou/>。

1957年，马玉涛参加在[[苏联|苏联]]举行的第六届世界青年节，获声乐比赛的金质奖。同年赴[[法国|法国]][[巴黎|巴黎]]参加大学生联欢会的演出。1959年，随[[南京军区|南京军区]]歌舞团赴[[奥地利|奥地利]][[维也纳|维也纳]]参加第九届世界青年节的联欢演出。此后先后随[[周恩来|周恩来]]等领导赴[[印度尼西亚|印度尼西亚]]、[[阿尔巴尼亚|阿尔巴尼亚]]、[[伊朗|伊朗]]、[[科威特|科威特]]、[[柬埔寨|柬埔寨]]、[[越南|越南]]、[[日本|日本]]、[[保加利亚|保加利亚]]、[[罗马尼亚|罗马尼亚]]、[[匈牙利|匈牙利]]等国访问演出。1999年，马玉涛与其他艺术家组成“华人音乐小组”赴[[美国|美国]]、[[澳大利亚|澳大利亚]]演出<ref name=baotou/>。

马玉涛曾连续获全军一至四届文艺会演独唱比赛一等奖。在[[中华人民共和国文化部|中华人民共和国文化部]]举办的全国独唱独奏会演中获一等奖。还曾获得广播金曲奖<ref name=baotou/>。马玉涛演唱的歌剧《[[小二黑结婚_(歌剧)|小二黑结婚]]》选曲、歌剧《[[刘胡兰_(歌剧)|刘胡兰]]》选曲，演唱《马儿啊，你慢些走》 、《献给敬爱的周总理》 ，分获1964年、1977年第三、第四届中国人民解放军文艺汇演演员奖。在[[八一电影制片厂|八一电影制片厂]]1976年2月摄制的彩色音乐片《红军不怕远征难——[[长征组歌|长征组歌]]》中，马玉涛、[[贾世骏|贾世骏]]、[[马国光|马国光]]和[[王克正|王克正]]担任独唱<ref name=zqzx>{{Cite web|url = http://news.cyol.com/content/2016-08/24/content_13734576.htm|title = 马玉涛|author = |date = 2016-08-24|publisher = 中青在线|language = |accessdate = |quote = |archive-url = https://web.archive.org/web/20170829131215/http://news.cyol.com/content/2016-08/24/content_13734576.htm|archive-date = 2017-08-29|dead-url = yes}}</ref>。马玉涛除参加本团的演出外，还经常到前线、边防、救灾一线参加慰问演出，参加过“[[五个一工程|五个一工程]]”及“[[心连心艺术团|心连心艺术团]]“系列演出<ref name=zqzx/>。曾出任[[全国青年歌手电视大奖赛|全国青年歌手电视大奖赛]]评委、中国音乐最高奖“[[金钟奖|金钟奖]]”的评委<ref name=baotou/>。她是[[中国文联|中国文联]]第四届委员<ref name=zqzx/>。

马玉涛是[[第四届全国人大代表|第四届全国人大代表]]，第八、九届全国政协委员。

==作品==
马玉涛的音色宽厚、圆润、明亮，气息充足绵长，歌声宏大深厚、刚健豪放，直抒胸怀<ref name=baode/>。马玉涛在演唱歌曲《马儿啊，你慢些走》时，细致处理了各部分在风格、速度、情绪上的差别，突出对比，并糅合了戏曲唱腔的叙事风格，使这首歌曲成为她最著名的代表作<ref name=baotou/>。

马玉涛演唱的歌曲代表作有：
*《马儿啊，你慢些走》
*《见了你们格外亲》
*《老房东》
*《众手浇开幸福花》
*《献给敬爱的周总理》
*《[[英雄赞歌|英雄赞歌]]》（电影《[[英雄儿女|英雄儿女]]》插曲）<ref name=baotou/>
