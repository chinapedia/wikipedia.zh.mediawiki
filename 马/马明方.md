'''马明方'''（{{bd|1905年|12月14日|1974年|8月12日|catIdx=M}}），曾用名'''马汝舟'''，号'''济民'''；[[陕西|陕西]][[米脂|米脂]]人。1925年秋加入[[中国共产党|中国共产党]]；是[[红军|红军]]时期[[陕北|陕北]]革命根据地的创始人和主要领导人之一，是中共第七届中央候补委员、第八届中央委员。[[中华人民共和国|中华人民共和国]]建国后，出任[[中共陕西省委|中共陕西省委]]书记，是首任[[陕西省人民政府|陕西省人民政府]]主席；还曾担任[[中共中央|中共中央]]副秘书长、[[中共中央组织部|组织部]]副部长、兼财贸部部长，[[中共中央|中共中央东北局]]第三书记，第一、第二届[[全国人大常委会|全国人大常委会]]委员。

因在“[[文化大革命|文化大革命]]”中，长期遭受迫害，于1974年8月12日逝世，享年69岁。

== 早期活动 ==
马明方1905年12月14日出生在[[陕西|陕西]][[米脂县|米脂县]]叶家岔村的一个农民家庭。1925年春，考入[[绥德|绥德]]省立第四师范学校，曾先后担任绥德师范中共党小组长、支部书记。而后，以教书为掩护，秘密发展和整顿中国共产党在[[陕北|陕北]]的势力；期间，还曾与[[马文瑞|马文瑞]]等领导了数万[[陕北|陕北]]农民参加“三抗”（抗粮、抗捐、抗税）“三要”（要土地、要饭吃、要自由）斗争。

1931年，马明方出任[[中共|中共]][[陕北|陕北]]特委代理书记；1934年至1937年，任[[中共|中共]][[陕北|陕北]][[省委书记|省委书记]]，并在1935年1月25日陕北召开的第一次工农代表大会上，当选陕北省苏维埃政府主席。他和[[刘志丹|刘志丹]]、[[习仲勋|习仲勋]]等[[陕北|陕北]]早期中共领袖，领导展开游击战争，推行创建革命根据地的策略和方针，为当时的[[中国共产党|中国共产党]]保留下了最后一个政权，也为[[红军|红军]][[长征|长征]]提供了最终的“落脚点”。<ref>[http://www.laorenbao.net/html/2012-10/15/content_2653425.htm 夫人带女求医，他说上班少 工资交公] {{webarchive|url=https://archive.is/20130502132736/http://www.laorenbao.net/html/2012-10/15/content_2653425.htm |date=2013-05-02 }} 《快乐老人报》</ref> 

== 战争时期 ==
[[抗日战争|抗日战争]]爆发后，马明方先后担任[[陕甘宁边区|陕甘宁边区]]政府主席团委员兼内政部代部长、民政厅厅长等职。由于疾病困扰，马明方于1938年赴[[苏联|苏联]]学习、治病疗养。1941年在回国途经[[新疆|新疆]]时，被中央任命为驻[[迪化|迪化]]八路军办事处负责人。1942年，由于[[苏联|苏联]]的介入，新疆爆发暴乱，军阀[[盛世才|盛世才]]与苏联和中共决裂，向[[国民党|国民党]]靠拢；马明方等120多名共产党员和亲共人士被捕入狱。1946年7月11日，在[[周恩来|周恩来]]、[[张治中|张治中]]等人的协调下才获释出狱。回到[[延安|延安]]后，马明方历任[[中共中央|中共中央]]西北局副书记兼[[中共|中共]]晋南工委书记，西北人民革命大学校长，西北局后方工作委员会书记等职。<ref>[http://www.shanxids.com/view.asp?id=154 马明方]{{dead link|date=六月 2017 |bot=InternetArchiveBot }} 陕西党史网</ref>

== 建国后 ==
[[中华人民共和国|中华人民共和国]]成立后，马明方出任[[中共陕西省委|中共陕西省委]]书记、[[陕西省人民政府|陕西省人民政府]]主席，[[中共中央|中共中央西北局]]第三书记、西北行政委员会副主席、西北军区副政委等职。1954年8月，调入中央工作，任[[中共中央|中共中央]]副秘书长、兼[[中组部|中组部]]副部长，协助中央秘书长兼[[中组部|中组部]]部长[[邓小平|邓小平]]的工作；1956年1月起，还兼任中央财政贸易工作部部长；1960年调任东北工作，任[[中共中央|中共中央]]东北局第三书记。<ref>[http://bjds.bjdj.gov.cn/ShowArticle.asp?ArticleID=9821 马明方]{{dead link|date=六月 2017 |bot=InternetArchiveBot }} 北京党史网</ref>

=== 文革 ===
[[文化大革命|文化大革命]]开始后，马明方受到了[[林彪|林彪]]、[[康生|康生]]、以及“[[四人帮|四人帮]]”的残酷迫害。1967年，以所谓“新疆马明方叛徒集团”头目，被押解到[[北京|北京]]，并长期秘密关押。1974年8月13日，被迫害致死，终年69岁。

===平反===
1975年7月31日，[[中共中央办公厅|中共中央办公厅]]曾下发文件称：“毛主席、党中央已经批准对‘新疆马明方案’予以平反”，“马明方等绝大多数同志在敌人监狱里表现很好，没有自首叛变”<ref>[http://www.jrxjnet.com/lishigouchen/wangshizhuisu/201103/9522_2.html “新疆马明方叛徒集团案”始末] {{webarchive|url=https://archive.is/20130426194320/http://www.jrxjnet.com/lishigouchen/wangshizhuisu/201103/9522_2.html |date=2013-04-26 }}</ref>。1979年12月29日，[[中共中央|中共中央]]为马明方举行追悼会，肯定了他所做的重要贡献。1980年6月24日，[[中共中央|中共中央]]以中发（1980）65号文件，批准了[[中组部|中组部]]和[[解放军总政治部|解放军总政治部]]联合提出的《关于为“新疆马明方案”平反报告》，为马明方等人彻底平反<ref>[http://www.zt360.cn/jgzyjl/ljjl/ssqdwsj/2012/06/12/13355716031.html 历任陕西省委书记] {{webarchive|url=https://web.archive.org/web/20150930071417/http://www.zt360.cn/jgzyjl/ljjl/ssqdwsj/2012/06/12/13355716031.html |date=2015-09-30 }} 政坛网</ref>。
