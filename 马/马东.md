'''马东'''{{BD|1968年|12月25日|||catIdx=M马}}，[[主持人|主持人]]、[[节目|节目]]制作人，相声演员[[马季|马季]]之子，生于[[黑龙江省|黑龙江省]][[哈尔滨|哈尔滨]]，祖籍[[天津|天津]][[宝坻|宝坻]]。

==经历==
*18岁时马东独自一人到[[澳大利亚|澳大利亚]]打工留学<ref name=ssy2>[https://v.qq.com/x/cover/79r40y15x4wmzcg/s0024i647oj.html 十三邀 第二季 第一期：许知远对话马东]，腾讯视频</ref>，后从事[[IT|IT]]行业，共在澳洲生活8年，并拿到了[[永久居留权|永久居留权]]<ref>{{cite web|title=澳大利亚漂泊8年 小马季终究干回父亲老本行|url=http://ent.sina.com.cn/s/m/49081.html|website=新浪|accessdate=2017-09-05}}</ref>。
*1994年决定回国，1995年报考[[北京电影学院|北京电影学院]]管理系并拿下第二个学位。
*1999年，马东接手[[湖南卫视|湖南卫视]]谈话类节目《[[有话好说|有话好说]]》。
*2001年竞聘到[[中国中央电视台|中央电视台]]任职，担任《[[挑战主持人|挑战主持人]]》、《[[文化访谈录|文化访谈录]]》等多档节目的主持人或制片人。2009年登上[[央視春晚|央視春晚]]，表演相声《[[五官新说|五官新说]]》。2011年担任央视春晚语言类节目导演。
*2013年开始主持[[河南卫视|河南卫视]]常规姓氏节目《知根知底》和季播节目《汉字英雄》。
*2013年正式离开央视，加盟门户视频网站[[爱奇艺|爱奇艺]]，担任首席内容官。2014年，马东主持在爱奇艺独播的网络节目《[[奇葩说|奇葩说]]》。
*2015年从爱奇艺离职创业，创立[[米未传媒|米未传媒]]<ref>{{cite web|title=爱奇艺首席内容官马东、优酷副总裁卢梵溪同时离职创业，视频门户模式将死？|url=https://www.huxiu.com/article/120414.html|website=虎嗅|accessdate=2017-09-05|date=2015-07-16}}</ref><ref>{{cite web|title=你所不知道的马东：他曾经当了整整10年的IT男|url=http://news.pedaily.cn/201607/20160720399886.shtml|website=投资界|accessdate=2017-09-05|date=2016-07-20}}</ref>，创立仅5个月便使估值达20亿元人民币<ref>{{cite web|title=米未传媒成立五个月估值20亿的秘密和马东“背后的女人”|url=http://news.pedaily.cn/201602/20160225393647.shtml|website=投资界|accessdate=2017-09-05|date=2016-02-25}}</ref><ref>{{cite web|title=《奇葩说》之外的米未传媒：马东还要做什么？|url=http://www.geekpark.net/topics/215866|website=极客公园|accessdate=2017-09-05|date=2016-06-20}}</ref><ref>{{cite web|title=搭上罗振宇，召唤老奇葩，你以为马东估值20亿的米未传媒只有《奇葩说》一个摇钱树？|url=http://news.pedaily.cn/201605/20160509396938_all.shtml|website=投资界|accessdate=2017-09-05|date=2016-05-09}}</ref><ref>{{cite web|title=《好好说话》之后，忘记马薇薇，记住米果|url=https://36kr.com/p/5065500.html|website=36氪|accessdate=2017-09-05}}</ref>。

== 个人生活 ==
马东的妻子彭小盛是湖南卫视原《有话好说》的编导，两人在工作中相识相恋。2001年马东前往北京发展时，彭小盛辞去在湖南卫视的工作，随马东到北京发展。两人于2004年结婚。<ref name="lion " >{{cite web |url=http://roll.sohu.com/20120308/n337080196.shtml |title=央视主持人妻子大揭秘 |publisher=搜狐网 |accessdate=2012-03-08 |work= |language =zh-cn}}</ref><ref name="shark " >{{cite web |url=http://ent.163.com/special/00032H0C/cctv1229.html |title=央视主持人的另一半 |publisher=网易 |accessdate=2010年 |work= |language =zh-cn}}</ref><ref name="duck ">{{cite web|url=http://photo.gmw.cn/2012-03/09/content_3733396.htm|title=央视主持人妻子大揭秘|publisher=光明网|accessdate=2012-03-09|work=|language=zh-cn|deadurl=yes|archiveurl=https://web.archive.org/web/20120314202654/http://photo.gmw.cn/2012-03/09/content_3733396.htm|archivedate=2012-03-14}}</ref>2008年，两人的女儿在北京降生，取名马苏拉<ref>{{cite web|title=央视主持人马东漂亮女儿照片曝光(图)|url=http://ent.sina.com.cn/v/m/2012-07-06/10023677865.shtml|website=新浪娱乐|accessdate=2017-09-05|date=2012-07-06}}</ref>。
