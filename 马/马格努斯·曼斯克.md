'''海因里希·马格努斯·曼努斯'''（{{lang-de|Heinrich Magnus Manske}}，{{bd|1974年|5月24日}}），德国[[科隆|科隆]]人，是英国[[剑桥|剑桥]][[惠康基金会|威康信托基金会]]桑格研究院的高级研究员，[[MediaWiki|MediaWiki]]软件第一版的[[软件设计师|软件设计师]]之一。

== 教育 ==
曼斯克在[[科隆大学|科隆大学]]读[[生物化学|生物化学]]，2006年凭借博士学位毕业。他的学位论文讲的是[[分子生物学|分子生物学]]开源工具[[GENtle|GENtle]]<ref name="magphd">{{cite thesis |author=Magnus Manske|year=2006 |url = http://kups.ub.uni-koeln.de/1907/ |title=GENtle, a free multi-purpose molecular biology tool |publisher=University of Cologne |degree=PhD}}</ref><ref>{{cite web |url = http://bioinformatics.knowledgeblog.org/2011/06/21/desktop-sequence-analysis-software-review/ |title=Desktop Sequence Analysis: software review |accessdate=2013-06-04 |date=2011-06-21 |author=Alastair Kerr |website=Bioinformatics Knowledge Blog}}</ref>。

== 开发MediaWiki ==
曼斯克在其学生时代是[[Nupedia|Nupedia]]计划最活跃的贡献者之一<ref name=SpiegelJansson>{{cite news|author=Kurt Jansson |title=Zehn Jahre Wikipedia – Der kurze Sommer der Anarchie|work=Der Spiegel |url = http://einestages.spiegel.de/static/authoralbumbackground/19921/der_kurze_sommer_der_anarchie.html|date=2011-01-14}}</ref>，为Nupedia提交生物学专题内容<ref name=KS />，开发工具，扩展内容<ref name=Anderson />。后来，曼克斯不满现有软件的限制，开发出后来的MediaWiki的首个版本<ref name=Anderson>{{cite book |title=Wikipedia: The Company and Its Founders |author=Jennifer Joline Anderson |publisher=ABDO |year=2011 |ISBN = 9781617148125 |page=44}}</ref><ref name=Zeit>{{cite news |author=Torsten Kleinz |title=Neues Werkzeug soll Anfänger zur Wikipedia locken |work=Die Zeit |url = http://www.zeit.de/digital/internet/2011-12/visual-editor-testversion |date=2012-12-16}}</ref>。2002年，他的新版本软件被安装<ref name=SpiegelJansson />。

软件起初在维基百科上时被称为[[UseModWiki|UseModWiki]]，利用[[Perl|Perl]]编写。伴随着维基百科的发展，一系列问题开始出现。2001年夏，曼斯克开始着手设计UseModWiki的替代者，该工具将支持数据库，包含“维基百科的专门功能”<ref name=Sumana>{{cite book |url = http://www.aosabook.org/en/mediawiki.html |title=The Architecture of Open Source Applications |chapter=MediaWiki |author1=Sumana Harihareswara|author2=Guillaume Paumier|editor=Amy Brown |editor2=Greg Wilson|publisher=Kristian Hermansen |year=2012 |ISBN = 9781105571817}}</ref>。2002年1月25日，曼斯克发布了基于[[PHP|PHP]]wiki引擎的[[MySQL|MySQL]]的首个版本“Phase II”<ref name=Kumar>{{cite news |title=Wiki Installation and Customization |author=Rajeev Kumar |url = http://ir.inflibnet.ac.in/handle/1944/990|accessdate=2013-06-04}}</ref>。曼斯克在Phase II中实行的一大创新是利用名字空间区分不同类型的页面，如“Talk”和“User”等名字空间，从而区分出没有使用不同名字空间的旧Wiki软件<ref name=Sumana />。Phase II还引入其他一些目前仍在存在的功能，其中包括文件上传、监视列表、自动签名和用户贡献列表<ref>{{cite web |url = http://en.wikipedia.org/w/index.php?title=Wikipedia:PHP_script_new_features&oldid=691 |title=Wikipedia:PHP script new features|date=2002-01-27|accessdate=2013-06-04}}</ref><ref>{{cite web |url = http://lists.wikimedia.org/pipermail/wikipedia-l/2002-January/001188.html |author=Magnus Manske|title=[Wikipedia-l] New features list|date=2002-01-27|accessdate=2013-06-04}}</ref>。曼斯克的改版也让维基百科条目更加轻松的集成图片，并创建出新的用户组：管理员，他们有权删除页面，封禁破坏者<ref name=SpiegelJansson /><ref>{{cite news|author=Erik Möller |title=Das Wiki-Prinzip|publisher=Heise online |url = http://www.heise.de/tp/artikel/14/14736/1.html |date=2003-05-09|language=de}}</ref>。

曼斯克是开源，尤其是[[GPL|GPL]]许可证的拥护者，他使用GPL许可证发布了MediaWiki的早期版本<ref>{{cite journal |author=N. A. Polukarova|title=The concept of open editing from the copyright viewpoint|journal=Automatic Documentation and Mathematical Linguistics|volume=41|issue=3 |pages=104–107|date=June 2007 |doi=10.3103/S0005105507030053 |ISSN = 1934-8371 |publisher=Allerton Press, Inc. |quote = 简单来说，尽管专用程序的开发人员利用他们的版权剥夺大多数用户的自由，但包括上文提到的马格努斯·曼斯克在内的用户能使用这些权限保证用户的自由。（In simpler words, while the developers of proprietary programs use their copyrights to deprive most of the users of most of their freedoms, GPL adherents, including above-mentioned Magnus Manske, use these privileges to guarantee these freedoms to users.）}}</ref>。

曼斯克的“Phase II”还遇到维基百科持续增长带来的负载问题，因此李·丹尼尔·克罗克创造出另一个重写版本“Phase III”。该版本于2002年6月运行，2003年改称“MediaWiki”<ref name=Kumar/>。结果是MediaWiki软件现在成为了维基百科等维基媒体姊妹项目的核心平台，还被众多组织和机构使用<ref name=Kumar/>。

曼斯克继续开发MediaWiki的工具和扩展程序，包括绘制分类归属、计算分类交集，并从[[Flickr|Flickr]]将图片导入共享资源<ref>{{cite web |url = https://toolserver.org/~magnus/ |title = Magnus' toys'n'tools |accessdate=2013-06-04}}</ref><ref>{{cite web |url=http://tools.wmflabs.org/magnustools/|title=Magnus' tools on wmflabs|accessdate=2013-06-04}}</ref>。曼斯克还开发出Cite扩展应用，给引用管理带来了[[XML|XML]]类语法<ref>{{cite conference |url=http://www.aifb.kit.edu/images/4/46/Websci09wikipedia.pdf |title=Introducing new features to Wikipedia|author1=Mathias Schindler |author2=Denny Vrandečić |accessdate=2013-06-04|date=2009-03|editor1=Jim Hendler|editor2=Helen Margetts |booktitle=Proceedings of the First International Conference on Web Sciences WebSci09 |location=Athens, Greece}}</ref>。

== 认可 ==
曼斯克被公认[[德文维基百科|德文维基百科]]首篇条目[[聚合酶链式反应|聚合酶链式反应]]的创造者，该条目由他在2001年首先编写<ref>{{cite news|title=German Wikipedia reaches landmark millionth entry|journal=Deutsche Welle |url=http://www.dw.de/german-wikipedia-reaches-landmark-millionth-entry/a-5064273|date=2009-12-28|language=de}}</ref><ref>{{cite news |author=dpa/mos|title=Deutsche Wikipedia erreicht eine Million Einträge|work=Die Zeit |url = http://www.welt.de/wirtschaft/webwelt/article5655970/Deutsche-Wikipedia-erreicht-eine-Million-Eintraege.html|date=2009-12-28|language=de}}</ref><ref>{{cite news|author=Achim Sawall|title=Deutschsprachige Wikipedia erreicht die Millionenmarke |publisher=golem.de |url = http://www.golem.de/0912/72088.html|date=2009-12-28 |language=de}}</ref>。

2002年，[[吉米·威尔士|吉米·威尔士]]将1月25日命名为马格努斯·曼斯克日，以表彰他对维基百科的贡献。吉米宣称“今晚在晚宴上，每一位维基百科应该具备庆贺马格努斯和他的众多发明创造<ref>{{cite web|url=http://lists.wikimedia.org/pipermail/wikipedia-l/2002-January/001164.html|title=[Wikipedia-l] Celebration|date=2002-01-25|accessdate=2013-06-04}}</ref>。”[[拉里·桑格|拉里·桑格]]在他的关于维基百科早年历史的回忆录中，强调曼斯克对计划的贡献，归属于给维基百科带来最终成功的核心演员小组，而曼斯克在其中扮演着重要角色，拉里·桑格這麼說：

<blockquote>维基百科开始于少数人，其中有许多人来自Nupedia。我认为Nupedia人的早期影响至关重要，尤其是马格努斯·曼斯克（曾致力于Nupedia及维基百科的软件）的坚持不懈，我们的常客、十分注重细节的路德·福彻尔（{{lang|en|Ruth Ifcher}}），以及非常聪明的扑克牌玩家程序员李·丹尼尔·克罗克——我仅举几例......不过，因为有这些善良的人启动计划，而且我们能够适应、解释和宣扬新人类的良好生活习惯和政策，Nupedia人是项目的根基，他们帮助开发一个强大、功能性的成功社区<ref>{{cite book|title=Open Sources 2.0: The Continuing Evolution|chapter=The Early History of Nupedia and Wikipedia: A Memoir|accessdate=23 June 2013|author=Larry Sanger|editors=Chris DiBona, Danese Cooper, and Mark Stone|publisher=O'Reilly Media|year=2008|isbn=9780596553890|page=317}}</ref>。</blockquote>

[[USENIX|USENIX]]高级计算机技术协会于2010年将曼斯克及其他人认定为MediaWiki的主要贡献者，当时MediaWiki和[[维基媒体基金会|维基媒体基金会]]荣膺软件工具用户组（STUG，Software Tools User Group）奖<ref>{{cite web |url = http://www.usenix.org/about/stug.html |title=STUG award|accessdate=2013-06-04}}</ref>。
