'''馬菲·基克斯'''（{{lang-en|'''Matthew Gilks'''}}，姓氏讀作''Jilks''，{{bd|1982年|6月4日|catIdx=Gilks, Matthew}}）是一名在[[英格蘭|英格蘭]]出生的[[蘇格蘭|蘇格蘭]]職業足球員，司職[[門將|門將]]，現時效力[[英冠|英冠]]球會[[黑池足球會|黑池]]。他出身家鄉球會[[罗奇代尔足球俱乐部|羅奇代爾]]，效力7年後於2007年轉投[[诺里奇城足球俱乐部|诺里奇城]]，加盟「金絲雀」（''the Canaries''）一個球季而未有獲派在一隊上陣，於2008年作為球員交換被交易到[[黑池足球會|黑池]]，亦曾被短暫借用到[[谢斯伯利足球俱乐部|梳士貝利]]；現時效力[[英超|英超]]球會[[伯恩利足球俱乐部|-{zh-hans:伯恩利; zh-hk:般尼;}-]]。

雖然在[[英格蘭|英格蘭]]出生，但基克斯以[[祖母|祖母]]血緣取得代表[[蘇格蘭足球代表隊|蘇格蘭]]參加國際賽資格，於2010年8月曾獲徵召入伍。

== 生平 ==
=== 球會 ===
;羅奇代爾
基克斯在[[英格蘭|英格蘭]][[西北英格蘭|西北部]][[大曼徹斯特郡|大曼徹斯特郡]][[羅奇代爾|羅奇代爾]]（Rochdale）出生，年僅13歲已加盟家鄉球會[[罗奇代尔足球俱乐部|羅奇代爾]]的優材中心（Centre of Excellence），於2000年3月由青年隊提升上一隊，直到2001年3月17日才首次於作客1-1賽和[[切斯特菲尔德足球俱乐部|車士打菲特]]中上陣<ref>{{cite news |language=en |url=http://news.bbc.co.uk/sport2/hi/football/eng_div_3/1224419.stm |title=Chesterfield 1-1 Rochdale |publisher=''[[BBC_Sport|BBC Sport]]'' |date=2001-03-17 |accessdate=2011-04-30}}</ref>。期間主要擔任[[Neil_Edwards_(footballer_born_1970)|尼爾·愛華士]]（Neil Edwards）的副手，2004/05年球季兩人合力創出整季有20場不失球的球會新記錄，而其中15場由基克斯把關<ref>{{cite web |language=en |url=http://www.ex-canaries.co.uk/players/gilks.htm |title=Matthew Gilks |publisher=''ex-canaries.co.uk'' |accessdate=2011-04-30}}</ref>，最終更取代尼爾·愛華士的常規正選席位成為球隊1號門將，於2005/06年及2006/07年兩個球季基克斯出席羅奇代爾整季所有賽事。他為羅奇代爾在聯賽及盃賽合共上陣197場，當他還只是24歲時，已經是隊中效力年期最長的球員<ref name="bbc6245490"/>。

;諾域治
2007年7月1日約滿羅奇代爾的基克斯以[[自由球員|自由身]]轉投[[英冠|英冠]]球會[[诺里奇城足球俱乐部|诺里奇城]]，簽約兩年<ref name="bbc6245490">{{cite web |language=en |url=http://news.bbc.co.uk/sport1/hi/football/teams/n/norwich/6245490.stm |title=Norwich to sign Rochdale keeper |date=2007-06-27 |publisher=''[[BBC_Sport|BBC Sport]]'' |accessdate=2008-11-21}}</ref>，但他未能躋身「金絲雀」（''the Canaries''）一隊陣中，於[[2007年至2008年英格蘭足球冠軍聯賽|2007/08年球季]]大部分時間坐於後備席廂<ref name="bbc7741834">{{cite web |language=en |url=http://news.bbc.co.uk/sport1/hi/football/teams/s/shrewsbury/7741834.stm |title=Gilks solves Shrews keeper crisis |date=2008-11-21 |publisher=''[[BBC_Sport|BBC Sport]]'' |accessdate=2008-11-21}}</ref>，期間更於2007年11月在操練時扭傷[[足踝|足踝]]導致[[韌帶|韌帶]]受創而缺席一段時間<ref name="bbc7121044">{{cite web |language=en |url=http://news.bbc.co.uk/sport1/hi/football/teams/n/norwich/7121044.stm |title=Gilks looks set for rapid return |date=2007-11-30 |publisher=''[[BBC_Sport|BBC Sport]]'' |accessdate=2008-11-21}}</ref>。
