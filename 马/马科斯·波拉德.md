'''马科斯·拉胡安·波拉德'''（Marcus LaJuan Pollard，生于1972年2月8日），是前[[美式足球|美式足球]]运动员，担任[[近端锋|近端锋]]。

== 大学战绩 ==
波拉德就读于[[布拉德利大学|布拉德利大学]]，在学校里打[[篮球|篮球]]。在被学校录取之前，他并未参加足球项目。他在篮球场上担任[[大前锋|大前锋]]，在转到布拉德利之前，曾在[[堪萨斯州|堪萨斯州]]里布瑞（Liberal）的苏厄德县社区大学（Seward County Community College）打过两年球。

== 职业生涯 ==
1995年，波拉德被[[印第安纳波利斯小马|印第安纳波利斯小马]]以未选秀自由球员身份签下。他在小马队成长成为一名出色的近端锋。他职业生涯最好的一年是2001年，他出场47次，共获得8次[[达阵|达阵]]，其中一次进攻距离达86码。随着{{link-en|达拉斯·卡拉克|Dallas Clark}}的崛起，波拉德在小马队进攻阵容中逐渐淡出，之后他转投[[底特律雄狮|底特律雄狮]]队。并在那里打了2年球。之后，他又先后加入[[西雅图海鹰|西雅图海鹰]]队、[[新英格兰爱国者|新英格兰爱国者]]队和[[亞特蘭大獵鷹|亞特蘭大獵鷹]]队。

== 退役后 ==
波拉德在亚拉巴马州拉奈特高中美式足球队执教。
