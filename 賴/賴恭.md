'''賴恭'''（？—？），[[東漢|東漢]]末年人物，[[零陵郡|零陵]]人，先後事奉劉表、劉備。《三国志》记载：赖恭，東漢末年，在荆州刺史刘表手下，任交州剌史。为人忠勇刚直，才华杰出，豪义很强。后来，投奔劉備属下，任镇远將軍，与诸葛亮竭诚扶持刘备，功迹卓著，他同诸葛亮等上书给汉献帝，请封刘备为汉中王。刘备重用他，封为太常。曹丕篡汉后，赖恭无比痛恨，上言给刘备请其称帝，极力扶持劉備江山。其子赖雄，也才华超群拔萃，任丞相西曹令史。

== 生平 ==
=== 管理交州 ===
[[荆州刺史部|荊州]]刺史[[劉表|劉表]]意圖染指交州，賴恭被派遣代替被手下[[區景|區景]]殺害的[[張津_(東漢)|張津]]為[[交州|交州]][[刺史|刺史]]。及後與同期到任的[[蒼梧郡|蒼梧]][[太守|太守]][[吳巨|吳巨]]不和，被吳巨舉兵逼走回到故鄉零陵。

=== 蜀漢為官 ===
劉表死後遭[[曹操|曹操]]驅逐，所以投靠[[劉備|劉備]]，劉備封賴恭為[[鎮遠|鎮遠]][[將軍|將軍]]。[[建安_(东汉)|建安]]二十四年（219年），賴恭與[[諸葛亮|諸葛亮]]一同上書[[漢獻帝|漢獻帝]]請封劉備為[[漢中王|漢中王]]，劉備遂封賴恭為[[太常卿|太常卿]]。建安二十五年（220年），[[曹丕|曹丕]]篡漢，賴上書請劉備稱帝，其後勸諫劉備出征[[孙吴|東吳]]，劉備堅持出征，命其留守在[[益州|西川]]。劉備死後與諸葛亮一同建議把劉備與[[昭烈皇后_(刘备)|昭烈皇后]]一同合葬。

== 家庭 ==
*[[賴厷|賴厷]]，賴恭的兒子。擔任過[[丞相|丞相]][[西曹|西曹]][[令史|令史]]，且深得諸葛亮喜愛。

== 同僚 ==
*黄柱，与賴恭同时任命，官拜光禄卿。《三国演义》出其姓名卻被稱為「黄权」。

== 《三國演義》中的賴恭 ==
在小說當中，賴恭唯獨在第八十回〈漢王正位續大統〉出現過。當時，他以[[太常卿|太常卿]]的身分與其他名官吏勸劉備登基為[[皇帝|帝]]。不過，其姓名卻被稱為「[[賴忠|賴忠]]」。
