[[电脑键盘|电脑键盘]]可以按照其按键开关所使用的技术进行分类。大部分电脑键盘有 80 到 110 个按键开关，其[[段落感|段落感]]、键程（输入字符所需的按键距离）因技术而异。

==种类==
===薄膜键盘===
薄膜键盘 ( Membrane Keyboard )经常由三层组成。顶层薄膜由一个框架支撑，其底部涂有一些导电的材料。底层中有一些断开的导电材料。当顶层薄膜被按下导致两层接触时，电路被接通。

[[Image:Membrane_keyboard_diagram_FULL_SCALE.png|thumb]]

最基本的平板薄膜键盘常见于[[微波炉|微波炉]]、[[复印机|复印机]]等需要防水防尘的电器。由于此类键盘没有段落感，这些电器多有按键音或按键灯提示。一些早期的个人电脑，如[[Sinclair_ZX80|Sinclair ZX80]]、[[Sinclair_ZX81|ZX81]]、[[Atari_400|Atari 400]]，也使用这类键盘。

后来的薄膜键盘通过修改顶层薄膜的结构和机制，提高了键程和段落感。这些键盘使用一个带柄的吸盘按下顶层薄膜。

===机械开关键盘===

'''机械键盘'''（ Mechanical Keyboard ）是一种[[键盘|键盘]]的类型，每一个[[按键|按键]]都由一个独立的微动开关（Switch，通称为「轴」）组成，因此按键[[段落感|段落感]]较强。

==== 机械键盘的種類與历史沿革 ====
鍵盤略分為三種型式：機械鍵盤、薄膜鍵盤、電容鍵盤；机械键盘于[[薄膜键盘|薄膜键盘]]之前被发明，是世界上最早使用的鍵盤型式，隨著 PC 的發明，機械鍵盤也隨之引進做為 PC 的標準輸入裝置，其盛行期約在 80 至 95 年，隨著橡膠帽薄膜键盘低製造成本與盛行，造价相对高昂的机械键盘逐渐被薄膜键盘所取代。

在機械鍵盤盛行期，依軸心可略分：IBM、ALPS、Cherry MX、MLK、Mitsumi或其他軸等。

;橫樑彈簧軸 ( Beam-spring Switch )
由 IBM 公司設計及生產之鍵盤採用，專利於 1971 年申請，並於同年開始生產，至 1981 年左右停產。因造價高昂，採用此科技的鍵盤多應用於大型電腦之終端機或其他高階電腦，軸體下方使用電容偵測。

;屈蹲弹簧轴 ( Buckling-spring Switch )
由 IBM 公司所設計與生產機械鍵盤，共有兩代。初代為 Model F，於 1981 年推出，使用屈蹲彈簧軸並電容偵測。第二代，亦是較為出名的一代，於 1985 年所推出之 {{link-en|IBM Model M Enhanced Keyboard|Model_M_Keyboard}} 鍵盤，此鍵盤的配位型式奠定日後所有鍵盤的基礎。據說，Model F 的設計初衷就是為克服橫樑彈簧軸 ( Beam Spring ) 的兩大問題：其一是軸體繁複、造價過高，其二是橫樑彈簧軸 ( Beam Spring ) 軸的高度過高，不符合當時逐漸抬頭的人體工學原則。

;ALPS
由日本 ALPS 公司所推出的機械鍵盤軸心，在機械鍵盤盛行期市占率最高，其中以 BIG FOOT 系列 (又稱 ALPS 原生軸) 最為廣泛使用，其外觀特色在與鍵帽接口為口字型，1985 年台灣福華與 ALPS 公司技術合作生產鍵盤及鍵盤用開關，後推出 ALPS 演生軸，如 ALPS 簡易軸。

; [[樱桃公司|Cherry]] <ref>{{Cite web |url = http://cherrycorp.com/about-us/ |title = About CHERRY |publisher = Cherry |accessdate = 2015-08-29 |deadurl = yes |archiveurl = https://web.archive.org/web/20150317000732/http://cherrycorp.com/about-us/ |archivedate = 2015年3月17日 |df =  }}</ref><ref>{{cite video  | url=http://v.youku.com/v_show/id_XNTczMjYxMTY4.html | title=CHERRY机械键盘历史 | publisher=Youku 优酷}}</ref>
創立於 1953 年美國伊利諾州 Cherry 公司，在 1984 年取得專利所推出 Cherry MX 軸，其外觀特色在與鍵帽接口為十字型，雖有不錯的製造品質，但成本高初期僅應用於高階伺服器或 PC。

;其他軸
台灣白軸，於 1988 年推出並取得專利，結合 Cherry 外觀與 ALPS 按壓力值的機械軸心，與鍵帽接口為十字型<ref>{{Cite web |url = http://kbtalking.cool3c.com/article/18452 |title = 台灣黑軸與台灣白軸的樣式 |publisher = 鍵談坊 |accessdate = 2015-08-30 }}</ref>。

==現況==
近年來電玩競技盛行，機械鍵盤的獨特手感，使得機械鍵盤的關注度正慢慢回升，目前市面上常见的机械键盘，大部份使用 {{link-en|Cherry| Cherry (keyboards)}} 公司（Cherry，通称樱桃）的MX机械轴（十字口），但也有少见的ALPS轴（方口）机械键盘，主要集中在低价位及少部分台湾产机械键盘上，另也有自行設計之機械軸，如 Logitech 推出了使用歐姆龍代工的 Romer-G 機械軸的鍵盤。<br />

隨著 Cherry MX 軸專利過期<ref>{{Cite web |url = https://www.google.com/patents/US4467160 |title = Low profile switch |language = en |accessdate = 2015-08-29|quote = A low profile switch key has a base which slidingly supports a pushbutton. The pushbutton is slidingly engaged with a plunger body by a lost motion connection }}</ref>，市場開始出現以 Cherry MX 機械軸作為設計對象之類似產品，如：凱華軸、冠泰軸、高特軸、佳達隆軸等<ref>{{Cite web |url = https://www.thekono.com/titles/pc_market/magazines/53f5eca7746d1/articles/de7d6c83-5cf1-4580-81e8-c16070948e2e |title = 新鍵軸登場　入門機械鍵盤巡禮 |author = 劉達仁 |date = 2014-08-19 |publisher = PCM電腦廣場|accessdate = 2015-08-29 }}</ref>，如中国的雷柏公司（RAPOO）推出过使用借鉴 Cherry MX轴的TCC軸机械键盘产品。

由於原廠 ALPS 原生軸與簡易軸停產多年，IROCKS 為複刻 APLS 手感，重新設計軸體 (十字口) 並取得消音專利，於 2016 Computex 年推出 APLS 手感自有軸。

=== 常见轴体 ===

触发：开关闭合，计算机接收到键盘电子信号。

总行程：开关的轴芯(十字柱)完成整个运动过程的位移长度。

触发行程：轴芯部分完成触发的位移长度。

初始压力：外物下按开关在轴芯即将发生位移时受到的压力。

触发压力：外物下按开关到触发时受到的压力。

段落压力：外物下按开关到段落点时受到的压力。

触底压力：外物下按开关到底部时受到的压力。

触发声音：外物下按开关到触发时的声音。
