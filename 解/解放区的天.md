'''解放区的天'''，又名'''解放区的天是明朗的天'''，原名'''边区的天是明朗的天'''，是小剧《逃难》的[[主题曲|主题曲]]，由[[刘西林|刘西林]]于1943年根据[[河北|冀]][[山东|鲁]][[民歌|民歌]]曲调填词而成。<ref name="people.com.cn">{{Cite web |url= http://www.people.com.cn/GB/shehui/44/20020729/787470.html |title= 歌曲《解放区的天》诉讼案终于“晴朗” |author=  |date=2002年7月29日 |publisher= 人民网 |language= zh-hans |accessdate=2011-05-14 |quote=  }}</ref>

== 词曲作者 ==
该曲为刘西林记谱作词，曲调源于民歌。

该歌曲的作者标注曾长期为“[[佚名|佚名]]词曲”或者“[[佚名|佚名]]词，[[陈志昂|陈志昂]]曲”，原因是陈志昂曾于1944年冬根据[[山东|山东]][[胶东|胶东解放区]][[海阳县|海阳县]][[郭城|郭城]][[实验区|实验区]]的秧歌调《唱十字》或《十字调》创作《'''儿童团的红旗迎风飘'''》一曲，与《解放区的天》的曲调相近，使陈志昂错以为《解放区的天》是根据《儿童团的红旗迎风飘》填词而来<ref>陈志昂. 《关于“解放区的歌声”》. 人民音乐. 1950年，1(3).</ref> 。后证实《解放区的天》是[[刘西林|刘西林]]根据[[河北|河北]]民歌直接填词而来，得知后陈志昂认可了这一说法<ref>陈志昂. 《抗日及解放战争时期我在胶东地区的音乐活动》. 齐鲁乐苑. 1986年10月.</ref> 。但仍然有多家唱片、出版公司错误标注该曲作者，刘西林去世后其遗孀曾因此事将[[中国唱片总公司|中国唱片总公司]]、[[宁波出版社|宁波出版社]]、[[四川文艺出版社|四川文艺出版社]]、[[江西人民出版社|江西人民出版社]]、[[北岳文艺出版社|北岳文艺出版社]]等告上法庭<ref> {{cite web  | title = 作者遗孀怒上法庭  | url = http://www.people.com.cn/GB/paper1787/6143/609810.html  | accessdate = 2015-11-03   }} </ref>，并达成经济赔偿调解协议<ref>{{cite news|title=《解放区的天》署名得以更正|url=http://www.gmw.cn/01ds/1999-04/28/GB/247%5EDS127.htm|agency=中华读书报|publisher=光明网|date=1999-04-28}}</ref>。

== 使用 ==
=== 东方红 ===
该曲在1964年10月2日[[人民大会堂|人民大会堂]]演出的大型音乐舞蹈史诗《[[东方红_(音乐舞蹈史诗)|东方红]]》中为“[[齐唱|齐唱]]、[[轮唱|轮唱]]”，位于第五场《埋葬蒋家王朝》<ref name="wyzxsx.com">{{Cite web|url=http://www.wyzxsx.com/Article/Class12/200902/69889.html|title=大型音乐舞蹈史诗《东方红》歌词和朗诵词|accessdate=2011-05-08|date=2009-02-14|format=|publisher=乌有之乡|language=zh-hans|deadurl=yes|archiveurl=https://web.archive.org/web/20090903035839/http://www.wyzxsx.com/Article/Class12/200902/69889.html|archivedate=2009-09-03}}</ref>。

=== 江水滔滔 ===
该曲是电影《[[江水滔滔|江水滔滔]]》插曲。
