'''佐竹氏'''（{{jpn|j=佐竹氏|kn=さたけし|rm=Sataje-shi}}）是[[日本|日本]]的[[武家|武家]]之一。本姓[[源氏|源氏]]。家系是[[清和源氏|清和源氏]]的一家系[[河內源氏|河內源氏]]流，是以[[源義光|新羅三郎義光]]為祖先的'''常陸源氏'''嫡流。與[[武田氏|武田氏]]代表的[[甲斐源氏|甲斐源氏]]是同族。[[通字|通字]]是「'''義'''」。[[家紋|家紋]]是「扇月」（{{lang|ja|扇に月}}，一般被稱為「日之丸扇」（{{lang|ja|日の丸扇}}」）），其他的有源氏香（花散里）、笹龍膽、佐竹桐、丸釘貫（{{lang|ja|丸に釘貫}}）、丁子巴、鐵線，總共7個家紋並稱「御當家七御紋」。因為屬於源氏，因此作為[[戰國大名|戰國大名]]、[[藩主|藩主]]、[[華族|華族]]而有明確的系譜流傳到現代，是一個非常大的氏族。

== 出自 ==
佐竹氏是與[[甲斐源氏|甲斐源氏]]的一族相同的[[源賴義|源賴義]]的兒子[[源義家|源義家]]的弟弟[[源義光|源義光]]的子孫義光流源氏的一族。關於佐竹氏的初代當主，一說是[[源義光|新羅三郎義光]]的兒子[[源義業|源義業]]（常陸源氏祖、進士判官）；另一說是義業的兒子[[源昌義|源昌義]]，而昌義居住在[[常陸國|常陸國]][[久慈郡|久慈郡]][[佐竹鄉|佐竹鄉]]（現今[[茨城縣|茨城縣]][[常陸太田市|常陸太田市]]）而以地名「佐竹」為姓，因此一般都是以昌義為初代當主。關於家名，傳說是昌義在常陸太田市的[[佐竹寺|佐竹寺]]中看見只有一節的[[竹子|竹子]]，以此為瑞兆並自稱佐竹氏。

== 平安時代至鎌倉時代 ==
[[平安時代|平安時代]]後期，佐竹氏支配著被稱為[[奧七郡|奧七郡]]的[[多珂郡|多珂郡]]、[[久慈東郡|久慈東郡]]、[[久慈西郡|久慈西郡]]、[[佐都東郡|佐都東郡]]、[[佐都西郡|佐都西郡]]、[[那珂東郡|那珂東郡]]、[[那珂西郡|那珂西郡]]等[[常陸|常陸]]北部7郡，與在常陸持有強大勢力的[[常陸平氏|常陸平氏]]一族[[大掾氏|大掾氏]]建立親戚關係，因此築起強大的勢力基礎。還與中央的[[伊勢平氏|伊勢平氏]]、[[東國|東國]]的[[奧州藤原氏|奧州藤原氏]]連結、積極介入常陸南部等，確立常陸的有力豪族的地位。

在[[治承壽永之亂|治承壽永之亂]]中，佐竹氏作為[[清和源氏|清和源氏]]的一族從加入[[平家|平家]]，因此被[[源賴朝|源賴朝]]沒收所領。後來跟隨賴朝參加[[奧州合戰|奧州合戰]]。此時使用純白旗參戰，因為與賴朝的旗一樣會造成紛擾，於是被命令在白旗上加上扇子圖案。這個扇子圖案中畫有[[滿月|滿月]]，以後佐竹氏的[[家紋|家紋]]就使用「扇月」（{{lang|ja|扇に月}}，一般被稱為「日之丸扇」（{{lang|ja|日の丸扇}}））。

在[[鎌倉時代|鎌倉時代]]中，奧七郡的支配權被[[宇佐見氏|宇佐見氏]]、[[伊賀氏|伊賀氏]]、[[二階堂氏|二階堂氏]]等奪去，後來[[北條氏|北條氏]]等獲得此郡的[[地頭|地頭]]職，佐竹氏在此時度過了權勢低下的時代。

== 室町時代 ==
進入[[室町時代|室町時代]]後，佐竹氏因為很早就呼應[[足利氏|足利氏]]而被任命為[[守護|守護]]職，而因為以[[鎌倉公方|鎌倉公方]]為主君的關係，於是經常被捲入[[足利將軍家|足利將軍家]]和鎌倉公方的鬥爭。佐竹氏以[[鎌倉府|鎌倉府]]重鎮的身份相當活躍，第3代鎌倉公方[[足利滿兼|足利滿兼]]在[[關東地方|關東地方]]向8個有力[[武家|武家]]賜予[[屋形號|屋形號]]而制定了[[關東八屋形|關東八屋形]]的格式，佐竹氏亦被列入其中，以後佐竹氏的當主就被尊稱為「御屋形大人」（{{lang|ja|お屋形さま}}）。

不過因為佐竹氏宗家第11代當主[[佐竹義盛|佐竹義盛]]沒有兒子，於是迎[[藤原北家|藤原北家]]的[[勧修寺流|勧修寺流]]的[[關東管領|關東管領]][[上杉氏|上杉氏]]的[[佐竹義人|佐竹義人]]為婿養子和第12代當主，在佐竹氏庶家引入佐竹男系血筋的[[山入氏|山入氏]]對此不滿並向宗家翻起反旗。山入氏連結[[室町幕府|室町幕府]]並奪取佐竹宗家的常陸守護職，於是引發[[山入之亂|山入之亂]]（山入一揆），更因為在名義上的臣下（實際上是獨立勢力）[[大掾氏|大掾氏]]和[[那珂氏|那珂氏]]（後來的[[江戶氏|江戶氏]]）的存在等，此時佐竹氏的勢力基礎可說是相當脆弱。而且亦有內紛，於是進入[[日本戰國時代|日本戰國時代]]後的佐竹氏亦在常陸統一上顯得極之困難，連成為[[戰國大名|戰國大名]]的時間亦很遲。

== 戰國時代 ==
進入日本戰國時代後，佐竹氏第15代當主‧被稱為「中興之祖」的[[佐竹義舜|佐竹義舜]]討伐山入氏，成功制壓常陸北部。不過江戶氏仍然持續著不穩的活動，還有以制覇[[關東地方|關東地方]]的[[後北條氏|後北條氏]]的侵攻等，統一常陸陷入非常困難的狀況。

義舜的曾孫‧佐竹氏第18代當主[[佐竹義重|義重]]是被稱為「鬼義重」的名將。在義重的時代，佐竹氏逐漸擊破[[江戶氏|江戶氏]]和[[小田氏|小田氏]]等，成功把常陸的大半部份置於支配之下，令佐竹氏飛躍成為戰國大名。滅亡的[[武田氏|武田氏]]浪人[[小山田氏|小山田氏]]等的仕官令家臣團得到強化亦是理由之一。

與甲斐武田氏同盟（[[甲佐同盟|甲佐同盟]]），與後北條氏在[[天正_(日本)|天正]]12年（1584年）於沼尻（現今[[栃木縣|栃木縣]][[栃木市|栃木市]]）對決（[[沼尻合戰|沼尻合戰]]）。還有進攻[[奧州|奧州]]南部、令[[白河結城氏|白河結城氏]]臣服、把[[石川氏|石川氏]]、[[岩城氏|岩城氏]]等置於影響之下、與[[三春城|三春城]]的[[田村氏|田村氏]]對抗中確立奧州國人盟主的地位。因此與義重的正室的姪子[[伊達政宗|伊達政宗]]對立，義重與[[蘆名氏|蘆名氏]]和[[二階堂氏|二階堂氏]]、[[岩城氏|岩城氏]]等結成同盟，與企圖在奧州建立覇權的政宗在天正13年（1585年）於[[人取橋|人取橋]]（現今[[福島縣|福島縣]][[本宮市|本宮市]]）對決（[[人取橋之戰|人取橋之戰]]）。佐竹方率領3萬大軍，以多於伊達方接近10倍的兵力進攻，伊達方受到極大損害，但是在一夜間撤退，結果此役成為伊達方強化奧州覇權的契機。

不過義重在[[日本戰國時代|日本戰國時代]]中擴大領國，在兒子[[佐竹義宣|義宣]]的時代參加[[豐臣秀吉|豐臣秀吉]]的[[小田原征伐|小田原征伐]]，在秀吉的[[太閤檢地|太閤檢地]]後被承認是[[常陸|常陸]]54萬5千8百石的[[大名|大名]]（但是常陸國內的[[土浦城|土浦城]]、[[下館城|下館城]]一帶被當作結城氏的所領）。[[水戶城|水戶城]]的[[江戶重通|江戶重通]]因為沒有參加小田原征伐而被沒收所領，佐竹氏把居城由[[太田城|太田城]]移到水戶城。

[[慶長|慶長]]5年（1600年），義宣在[[關原之戰|關原之戰]]中因為無法統一家中的意見而採取中立態度。戰後處理在翌年結束。慶長7年（1602年）3月，義宣[[上洛|上洛]]並在[[伏見城|伏見城]]謁見[[德川家康|德川家康]]。5月8日，家康突然命令義宣轉移至[[出羽國|出羽國]]。7月27日，在石高不明的情況下被轉封至[[秋田郡|秋田]]、[[仙北郡|仙北]]（『[[梅津政景日記|梅津政景日記]]』）。關於在關原之戰中的行動，被家康發覺與[[上杉景勝|上杉景勝]]結盟的密約。而相當接近[[德川氏|德川氏]]本據地[[江戶|江戶]]的佐竹氏和同族的多賀谷領（[[岩城宣隆|岩城宣隆]]）、岩城領（[[岩城貞隆|岩城貞隆]]）、相馬領（[[相馬義胤|相馬義胤]]）的勢力圈加上總共有80萬石以上，而因為沒有直接參加合戰而保留了實力，因此被視為威脅。於是佐竹氏就從由[[平安時代|平安時代]]後期以來先祖傳來之地的[[常陸|常陸]]離去。

在處理之際，[[細川忠興|細川忠興]]進言「大大名佐竹氏沒有出羽一國的話，家臣可能會因為奉祿不足而引發亂事」（{{lang|ja|大大名の佐竹氏には出羽一国でなければ家臣を賄いきれず変事が起きるかもしれない}}），但是德川家康的側近[[本多正信|本多正信]]、[[本多正純|正純]]父子則說「如果給與出羽一國則與常陸沒有分別，所以給與半國就可以」（{{lang|ja|出羽一国を与えるのでは常陸と変わらないから半国でよし}}），因此家康聽從正信、正純的意見（『梅津政景日記』）。後來在正純於政治鬥爭中失勢時（[[宇都宮城釣天井事件|宇都宮城釣天井事件]]），幕府把正純禁閉在佐竹氏的領地，正純被流放到出羽横手，亦因為與佐竹氏有仇的關係而沒有收到特別照顧，於是正純就這樣在[[横手城|横手城]]度過餘生。

== 江戶時代以降 ==
在[[江戶時代|江戶時代]]中支配[[久保田藩|久保田藩]]並以[[外様大名|外様大名]]的身份存續。轉封時沒有明示的石高在[[寬文|寬文]]4年（1664年）4月2日被決定是20萬5千8百石（實際是40萬石）。其中5千8百石是慶長10年（1605年）10月17日追加的[[下野國|下野國]][[河內郡_(栃木縣)|河內郡]]、[[都賀郡|都賀郡]]的[[外飛地|外飛地]]11條村（『梅津政景日記』）。

[[元祿|元祿]]14年（1701年），佐竹氏第21代當主‧久保田藩第3代藩主[[佐竹義處|佐竹義處]]向弟弟[[佐竹義長|佐竹義長]]分與2萬石，向姪子[[佐竹義都|佐竹義都]]分與1萬石，令其建立[[久保田新田藩|久保田新田藩]]（因為屬於[[新田分知|新田分知]]，所以久保田藩的石高不變）。其中以佐竹義都為初代藩主的久保田新田藩在[[享保|享保]]17年（1732年）因為義都的兒子[[佐竹義堅|佐竹義堅]]成為久保田藩第5代藩主[[佐竹義峯|佐竹義峯]]的嗣子而被廢藩，封地變回久保田藩。另一方面，以佐竹義長為初代藩主的久保田新田藩因為歷代藩主都敘任[[壹岐守|壹岐守]]而被稱為「佐竹壹岐守家」在[[明治|明治]]2年（1869年）改稱[[岩崎藩|岩崎藩]]。

明治17年（1884年），佐竹氏第30、32代當主‧舊久保田藩主[[佐竹義堯|佐竹義堯]]敘任[[侯爵|侯爵]]，佐竹壹岐守家（舊久保田新田藩）當主[[佐竹義理|佐竹義理]]敘任[[子爵|子爵]]。明治22年（1889年），佐竹壹岐守家出身並成為義堯養子並一時間繼承宗家的[[佐竹義脩|佐竹義脩]]敘任[[男爵|男爵]]。此外關於家臣佐竹四家（東西南北家），明治33年（1900年）佐竹西家的[[佐竹義遵|佐竹義遵]]、佐竹南家的[[佐竹義雄|佐竹義雄]]、佐竹北家的[[佐竹義尚|佐竹義尚]]敘任男爵；明治39年（1906年）佐竹東家的[[佐竹義準|佐竹義準]]敘任男爵。[[東洋製作所|東洋製作所]]的原[[社長|社長]][[佐竹義利|佐竹義利]]是佐竹東家出身，現今[[秋田縣|秋田縣]][[知事|知事]][[佐竹敬久|佐竹敬久]]是佐竹北家出身。

另外，關於佐竹氏宗家（舊久保田藩主）不是被封為[[伯爵|伯爵]]而是侯爵，一說指是因為在[[戊辰戰爭|戊辰戰爭]]中加入新政府軍的關係，另一說指是因為久保田藩的實際石高比起表面高石多出很多。
