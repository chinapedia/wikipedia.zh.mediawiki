'''佐烏'''（{{lang|ja|ゾウ}}，{{lang|en|Zou}}，又譯'''象島'''），是[[尾田榮一郎|尾田榮一郎]]的動漫作品《[[ONE_PIECE|ONE PIECE]]》中虛構的島嶼。

== 島嶼概要 ==
佐烏位於[[ONE_PIECE用語列表#新世界|新世界]]，座落在一頭存活長達1,000年以上的巨象背上，並隨著巨象的步伐在海面移動<ref>One Piece漫畫 805话</ref>，巨象曾在遠古時期犯錯，因此除了行走，其餘皆被禁止。這座島不是陸地，所以靠記錄指針是到不了那裡。由島上住民「純毛族」組成「摩科莫公國」，據說有將近1,000年的歷史。

然而於魯夫等人抵達兩個星期前，四皇百獸海道的心腹「旱禍」JACK為了找出[[ONE_PIECE角色列表#和之國|和之國]]忍者雷藏的下落，使用凱薩的殺戮兵器KORO將這個國家擊潰。是和之國武士錦衛門一行人的目的地，此之前哈特海賊團也曾停駐在此地<ref>One Piece700话</ref>。

=== 文化 ===
* 因為該島嶼矗立於巨象的背部，所以島上居民的生活和巨象的作息息息相關，每當巨象用鼻部取水灑在背部淋浴（當地居民稱為「噴火雨」）時，會連同海裡的一些生物捲至佐烏，為當地居民帶來海產類的食材兼灌溉作物，島上居民還特地針對此特性設置特製的排水系統。
* 當地居民純毛族因為是擁有半人半動物習性的民族，島上的居民和統治者亦分為晝行性和夜行性兩派，每逢日夜交替之時，相反晝夜習性的居民就會進入睡眠狀態。白晝期間由白晝統治者犬嵐公爵掌管政事，黑夜則由黑夜統治者貓蝮蛇掌櫃治理國家。當白晝統治者和黑夜統治者有要事要傳達和接受通知時，會指派「王之鳥」的信使傳遞訊息。
* 當地居民習慣用「佳魯丘」做為對彼此和友善訪客的招呼語，當他們判斷有外來訪客造訪時會敲響歡迎鐘，對於友善的訪客會和對方交換衣物做為友誼的證明，此外有武裝份子入侵時則會槍響敵襲鐘。
* 由於純毛族和數百年前打造歷史本文石碑的和之國家族光月一族有宣誓效忠的盟約，所以純毛族會特別留意[[和之國|和之國]]的動向。

=== 地理 ===
佐烏上的幾個重要地點：

* 城寨：純毛族於森林搭建的城寨。
* 鯨魚森林：有著像鯨魚般的大型樹木，裡面藏有其中一座指示「拉乎德爾」所在處的紅色路標歷史本文。

=== 居民 ===
* 純毛族：又譯水貂族、毛皮族，於夏波帝諸島人口販賣首次提及的種族其一。有著半人半獸的外觀，並且會有自身獸種的能力和特性（如兔子外型擁有高靈敏度和跳躍力，狗外型有強力的嗅覺等等...），除了可以像人類一樣使用工具和武器之外，由於自身有毛皮的關係，擁有獨特產生[[靜電|靜電]]的方式來附加在身體或武器上攻擊敵人。稱呼人類為「無毛族」，外傳純毛族極度厭惡人類，但其實族人會對有恩於己的人類願意敞開心防，自劇情開始的數十年甚至數百年前，便宣誓效忠和之國的光月一族。

: 目前與[[ONE_PIECE海賊列表#草帽海賊團|草帽海賊團]]和[[托拉法爾加·D·瓦特爾·羅#哈特海賊團|哈特海賊團]]組成海賊同盟，目標是擊敗四皇「百獸」海道。

=== 俠客團 ===
; <span id="佩特羅">{{nihongo|佩特羅|ペドロ|Pedro}}
: 聲優：[[三木真一郎|三木真一郎]]（日本）；[[符爽|符爽]]/連婉鈞(童年時期)（台灣）
: [[美洲豹|美洲豹]]種純毛。鯨魚森林「俠客團」的首領，貓腹蛇大哥最信任的左右手，又稱「林木上的佩特羅」，原「諾克斯海賊團」的船長，懸賞金額：3億8,200萬貝里、年齡：32歲、生日：6月16日。平時用瀏海遮住被挖去眼球的左眼。為人重視恩義，願為恩人赴湯蹈火在所不惜。
: 因為JACK帶來了凱薩所製作的殺戮兵器導致中毒，香吉士等人抵達島上時，不顧自己的傷勢向他們懇求治療奄奄一息的犬嵐公爵與貓腹蛇大哥，在萬妲將香吉士等人誤認為是「旱禍」JACK的部下欲引爆手上的炸藥同歸於盡之際，他與特里斯坦出面化解萬妲對眾人的誤會。10天後魯夫等人也登陸佐烏，包含佩特羅在內的俠客團成員與誤闖鯨魚森林的魯夫起了衝突，後因看在萬妲的面子上而制止了羅帝和BB與魯夫的打鬥。
: 波哥姆斯的舊識，也是在純毛族中唯一能夠讓因月獅子型態而失去理智的波哥姆斯恢復理智的人。過去童年時期曾見犬嵐公爵、貓腹蛇大哥和羅傑出海，懇求羅傑帶其隨行，羅傑因其當時年紀尚小未與同意，但告訴他「每個人都有屬於自己的登場時機」。
: 因為想幫助貓腹蛇大哥尋找歷史本文成為探險家，和波哥姆斯結夥、傑波、培波結成「諾克斯探險隊」出海，結果因為追尋世界歷史本文遭世界追逐誤成通緝犯，隨著貝波下象觀察大海意外飄走、探險隊出現傷員，佩特羅轉而委託波哥姆斯帶領剩餘的探險隊成員，自己則和傑波持續探索歷史本文。
: 在5年前和夥伴傑波遊歷到「BIG MOM」領地，意外和波哥姆斯重逢，才知道波哥姆斯已加入「BIG MOM」旗下，仍決意冒險要竊走BIG MOM所掌有的歷史本文，雖然佩特羅和塔馬哥男爵決戰過程中令對方失去左眼，但仍是失敗被俘虜，隨行夥伴傑波也被「BIG MOM」奪去30年壽命身亡；佩特羅原本被「BIG MOM」要脅奪去70年壽命，但由於佩特羅自剜左眼謝罪和波哥姆斯居中求情，才以付出50年壽命的代價離開「BIG MOM」的地盤。
: 當魯夫率領的「奪回香吉士小隊」出發去蛋糕島前，獲知犬嵐公爵和貓腹蛇大哥視魯夫等人必須需要歷史本文，向貓腹蛇大哥取得同意，幫助魯夫等人尋找和取得路標歷史本文。後和布魯克在「BIG MOM」領地行動找到「BIG MOM」寶藏地，決意以誘餌身分吸引BIG MOM成員，再度和塔馬哥男爵狹路相逢並用炸藥引發大規模爆炸擊敗對方。
: 魯夫等人成功救出香吉士逃離婚宴會場後，佩特羅意識到魯夫正是純毛族與光月家族的希望，最終為幫助魯夫等人逃離托特蘭暨對付「BIG MOM」的長子裴洛斯佩羅與次子卡塔克利，不惜點燃隨身攜帶的炸藥自爆，生死未知。事後曾作為對手的塔馬哥男爵亦對其表示了敬意。
