'''佐渡島'''（{{jpn|j=佐渡島|hg=さどがしま|rm=Sadogashima|f=y}}）是位於[[日本海|日本海]]東部的一個島嶼，屬[[日本|日本]][[新潟縣|新潟縣]][[佐渡市|佐渡市]]管轄，海岸線全長262.7公里。2017年10月，佐渡島有人口55,212人。佐渡市面積854.76平方公里，是日本四大主島以外島嶼中僅次於[[沖繩本島|沖繩本島]]的第二大島嶼（不含未實際控制的[[南千島群島|南千島群島]]）<ref name="area"></ref>。佐渡島距[[本州|本州]]最近處的距離是32公里<ref name="outline">{{cite web|url=http://www.pref.niigata.lg.jp/sado_kikaku/sado_gaikyou.html　|script-title=ja:佐渡島の概況|accessdate=2017-07-08|publisher={{lang|ja|新潟県}}|language=ja}}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。佐渡島上的最高峰是{{link-ja|金北山|金北山}}，標高1,172米<ref>{{cite web |url=http://www.yamakei-online.com/yamanavi/yama.php?yama_id=419　|script-title=ja:金北山 |accessdate=2017-07-08|publisher={{lang|ja|山と渓谷社}} |language=ja}}</ref>。

== 地理 ==
[[File:Sadogashima_Island_Relief_Map,_SRTM-1.jpg|left]]
佐渡島外形類似「工」字或字母「S」。島上地形可以大致分為三個部分，北部是大佐渡山地，南部是小佐渡山地，兩者之間是地形平坦的國仲平原。大佐渡山地的海拔較高，佐渡島的最高峰金北山（海拔1,172公尺）亦位於這一地區。大佐渡山地北側的海岸線地勢險峻，是著名的觀光景點，其中又以尖閣湾最為著名<ref>{{cite web|url=https://www.visitsado.com/spot/detail0144/　|script-title=ja:尖閣湾|accessdate=2017-07-09|publisher={{lang|ja|佐渡観光協会}}|language=ja}}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。小佐渡山地則地形較為平緩，有自生[[溫州蜜柑|蜜柑]]及[[茶|茶樹]]，最高峰是大注入地山，標高646米<ref>{{cite web |url=http://www.yamakei-online.com/yamanavi/yama.php?yama_id=420　|script-title=ja:大地山 |accessdate=2017-07-09|publisher={{lang|ja|山と溪谷社}} |language=ja}}</ref>。國仲平原在日本的離島中屬於大型平原，也是佐渡島的穀倉。國仲平原西側是{{link-ja|真野灣|真野湾}}、東側是{{link-ja|兩津灣|両津湾}}。國府川流經國仲平原，全長19公里，注入真野灣，其流域面積佔佐渡島的二成<ref>{{cite web|url=http://www.pref.niigata.lg.jp/HTML_Article/935/242/kokuhu_sei1.pdf　|script-title=ja:国府川水系河川整備計画|accessdate=2017-07-09|publisher={{lang|ja|新潟県}}|language=ja}}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。國仲平原的東端有新潟縣最大的湖泊[[加茂湖|加茂湖]]。加茂湖原本是淡水湖，後開通了和日本海之間的開口，變為潟湖，[[牡蠣|牡蠣]]養殖興盛<ref>{{cite web |url=https://www.visitsado.com/spot/detail0136/?no_cache=1　|script-title=ja:加茂湖 |accessdate=2017-07-09|publisher={{lang|ja|佐渡観光協会}} |language=ja}}</ref>。佐渡島部分地區屬於[[佐渡彌彥米山國定公園|佐渡彌彥米山國定公園]]<ref>{{cite web|url=http://www.pref.niigata.lg.jp/HTML_Article/537/767/map_08,0.pdf　|script-title=ja:佐渡地域区域図及び公園計画図|accessdate=2017-07-09|publisher={{lang|ja|新潟県}}|language=ja}}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

== 氣候 ==
[[File:Kameiwa_Sado.jpg|right]]
佐渡島受到途經佐渡海域的暖流[[黑潮|黑潮]]影響，冬季比新潟縣本土溫暖且積雪較少。夏季的佐渡島由於海洋的影響，晝夜溫差比新潟縣本土要小，氣溫也比新潟縣本土涼爽，但大佐渡山地西北側的相川地區等地受到自西北方向吹來的季風影響而氣溫偏高。佐渡島的降水主要集中在[[梅雨|梅雨]]末期和初冬，和新潟縣本土相比降水量較少。在低氣壓和[[颱風|颱風]]通過日本海時，佐渡島常有[[焚風|焚風]]現象發生造成氣溫急劇升高<ref>{{cite web|url=https://www.city.sado.niigata.jp/admin/vision/bousai07/07dis05.shtml　|script-title=ja:第4節：佐渡市の自然条件|accessdate=2017-07-09|publisher={{lang|ja|佐渡市}}|language=ja}}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。佐渡市是暖流和寒流的交匯點，因此植物生態非常多樣，且水產資源亦很豐富。

== 歷史 ==
[[File:Sadokinzan-doyunowareto_01.JPG|left]]]]
佐渡島歷史悠久，自一萬年之前開始就有人類居住。在金井地區的千種遺跡發現了彌生時代後期的石器及狩獵用具。《古事記》及《日本書紀》中也有關於佐渡的計數。[[大化改新|大化改新]]後，中央政權在佐渡島設立[[佐渡國|佐渡國]]，本土人口開始移民佐渡島。同時佐渡島也成為[[政爭|政爭]]失敗者或[[異議分子|異議分子]]的[[流放地|流放地]]。[[順德天皇|順德天皇]]、[[日蓮|日蓮]]等人都曾被[[流放|流放]]至佐渡<ref>{{cite web|url=https://sado-biyori.com/about/history/　|script-title=ja:佐渡の歴史|accessdate=2017-07-09|publisher={{lang|ja|佐渡日和}}|language=ja}}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。[[鎌倉時代|鎌倉時代]]之後，{{link-ja|本間氏|本間氏}}被任命為佐渡的守護代。1589年（天正17年）後，[[上杉景勝|上杉景勝]]侵入佐渡島，此後佐渡島成為上杉氏支配的地區。

1601年，採礦者在相川鶴子銀山發現了金銀礦脈，即為[[佐渡金山|佐渡金山]]，使得佐渡島的地位一躍而升。1603年（慶長8年），[[德川家康|德川家康]]在[[關原之戰|關原之戰]]勝利後迅速將佐渡島劃為幕府[[天領|直轄領地]]。此後佐渡金山發展為日本最大的金礦，至[[幕末|幕末]]的金產量達41噸，在[[幕府|幕府]]的財政中有重要地位。1620年時，佐渡島人口達約10萬人<ref>{{cite web|url=http://www.sado-ryokou.com/sadoshiru/history.html　|script-title=ja:佐渡の歴史|accessdate=2017-07-09|publisher={{lang|ja|佐渡汽船}}|language=ja}}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。[[江戶時代|江戶時代]]中期後，佐渡金山的開採量開始減少。但自[[明治時代|明治時代]]後，由於導入了先進的挖掘技術，佐渡金山的產量恢復到了江戶時代最盛時期年產量400千克的水準。1989年，佐渡金山關閉。在近四百年的開採史中，佐渡金山的金產量達78噸，銀產量達2,300噸<ref>{{cite web|url=http://carbide.mmc.co.jp/magazine/article/sado　|script-title=ja:最大級の産出量を誇った日本の金の宝庫|accessdate=2017-07-09|publisher={{lang|ja|三菱マテリアル}}|language=ja}}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

[[廢藩置縣|廢藩置縣]]後，[[日本政府|日本政府]]在佐渡島設佐渡縣（後改名為[[相川縣|相川縣]]）。1876年，相川縣被併入新潟縣<ref>{{cite web|url=http://www.pref.niigata.lg.jp/1356778597824.html　|script-title=ja:県の歴史と有名な人|accessdate=2017-07-09|publisher={{lang|ja|新潟県}}|language=ja}}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。2004年，佐渡島內的10個市町村合併為[[佐渡市|佐渡市]]<ref>{{cite web|url=https://www.city.sado.niigata.jp/admin/profile/pdf_unit/06/05.pdf　|script-title=ja:佐渡１０市町村の合併に関する経過|accessdate=2017-07-09|publisher={{lang|ja|佐渡市}}|language=ja}}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

== 人口 ==
1960年時，佐渡島曾有人口113,296人<ref name="population">{{cite web |url=https://www.city.sado.niigata.jp/admin/stat/m1_kokusei/s_01.shtml　|script-title=ja:人口の推移 |accessdate=2017-07-08|publisher={{lang|ja|佐渡市}} |language=ja}}</ref>。因離島生活不便和工作崗位較少，佐渡島長期呈現人口減少局面。2017年10月，佐渡島人口55,212人。佐渡島的[[總和生育率|總和生育率]]約有1.9，遠高於日本及新潟縣的平均值，但因人口持續外流而使得佐渡島人口長期減少<ref>{{cite web|url=https://www.city.sado.niigata.jp/topics/p_comm/pdf2015/02_2.pdf　|script-title=ja:人口ビジョン|accessdate=2017-07-09|publisher={{lang|ja|佐渡市}}|language=ja}}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

== 文化 ==
由於佐渡島是自日本海沿岸前往[[大阪市|大阪]]的{{link-ja|西廻海運|西廻海運}}路線上重要貿易據點，因此文化上受近畿、北陸等西日本文化的影響較強。由於有眾多文化人、政治家被從[[京都|京都]]流放至佐渡，帶來了京都的傳統表演文化，使得佐渡島的傳統藝能興盛。[[能劇|能劇]]大師[[世阿彌|世阿彌]]曾被流放至佐渡，使得佐渡的能劇文化特別發達<ref>{{cite web |url=https://www.visitsado.com/sado/history/?no_cache=1　|script-title=ja:佐渡の歴史 |accessdate=2017-07-09|publisher={{lang|ja|佐渡観光協会}} |language=ja}}</ref>。江戶時代的佐渡曾有超過200個能舞台。現在佐渡仍有超過30個能舞台，人均能舞台數居日本之冠<ref>{{cite web |url=http://www.niigata-kankou.or.jp/sys/data?page-id=8808　|script-title=ja:佐渡の能舞台 |accessdate=2017-07-09|publisher={{lang|ja|新潟県観光協会}} |language=ja}}</ref>。佐渡島還有「鬼太鼓」等具當地特色的表演藝術<ref>{{cite web |url=http://www.niigata-kankou.or.jp/sys/data?page-id=8830　|script-title=ja:鬼太鼓 |accessdate=2017-07-09|publisher={{lang|ja|新潟県観光協会}} |language=ja}}</ref>。

== 朱䴉 ==
佐渡島是日本最後的[[朱䴉|朱{{僻字]]棲息地。朱䴉原本分佈在日本各地，但在明治時代之後因狩獵和農藥毒害而在佐渡島以外地區絕跡。1981年，日本開始試圖對朱䴉進行人工繁殖但告失敗。2003年10月10日，日本最後一尾朱䴉死亡，日本產朱䴉至此在日本完全滅絕。但1999年，中國贈與日本的朱䴉成功實現人工繁殖。2008年，保育部門在佐渡島放生10隻朱䴉。之後保育部門每年均有放生朱䴉<ref>{{cite web |url=http://tokinotayori.com/life/　 |script-title=ja:トキの生態 |accessdate=2017-07-09 |publisher={{lang|ja|トキの森公園}} |language=ja }}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。現在佐渡的朱䴉已成功實現在野生狀況下的繁殖<ref>{{cite web |url=http://www.env.go.jp/nature/toki/　 |script-title=ja:トキ情報 |accessdate=2017-07-09 |publisher={{lang|ja|環境省}} |language=ja }}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

== 交通 ==
{{link-ja|國道350號|国道350号}}橫貫佐渡島人口密集的地區，是佐渡島的主幹道。巴士是佐渡島唯一的公共交通方式。島上巴士全部由{{link-ja|新潟交通佐渡|新潟交通佐渡}}運營<ref>{{cite web |url=http://www.sado-bus.com/route/　 |script-title=ja:路線図 |accessdate=2017-07-09 |publisher={{lang|ja|新潟交通佐渡}} |language=ja }}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。海運是現在佐渡島最主要的聯外交通方式。佐渡島和新潟縣本土之間的渡輪由{{link-ja|佐渡汽船|佐渡汽船}}運營，有兩津港至[[新潟港|新潟港]]、小木港至{{link-ja|直江津港|直江津港}}、赤泊港至寺泊港三條航線<ref>{{cite web |url=http://www.sadokisen.co.jp/timetable/　 |script-title=ja:時刻表 |accessdate=2017-07-09 |publisher={{lang|ja|佐渡汽船}} |language=ja }}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。佐渡島上有[[佐渡機場|佐渡機場]]一處機場。佐渡機場於1958年啟用，曾開設來往[[新潟機場|新潟機場]]之間的航線；但自2014年起再沒有定期航線飛行<ref>{{cite web |url=http://www.pref.niigata.lg.jp/sado_seibi/1204045245988.html　 |script-title=ja:佐渡空港 |accessdate=2017-07-09 |publisher={{lang|ja|新潟県}} |language=ja }}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。
