'''佐賀機場'''（{{jpn|j=佐賀空港|hg=さがくうこう|rm=Saga kūkō|f=y}}，{{airport codes|HSG|RJFS}}）是位於[[佐賀縣|佐賀縣]][[佐賀市|佐賀市]]的一个[[机场|机场]]，根據日本[[空港整備法|空港整備法]]被分成第三種機場。設有愛稱「有明佐賀機場」。

== 歷史 ==
[[File:Saga_Airport_Aerial_photograph.2008.jpg|thumb]]


*1998年7月18日：機場啟用。
*2000年7月：前往[[東京國際機場|東京國際機場]]夜間航班开始。
*2001年9月1日：[[日本佳速航空|日本佳速航空]]停止前往[[大阪國際機場|大阪國際機場]]的航班。
*2002年11月1日：往[[大阪國際機場|大阪國際機場]]航班增加班次(1日2往返)。
*2003年2月1日：往[[名古屋機場|名古屋機場]]航班休止。
*2004年7月8日：前往[[東京國際機場|東京國際機場]]的货运航班开始。
*2005年10月1日：往[[東京國際機場|東京國際機場]]航班增加班次(1日3往返)。
*2006年2月24日：前往[[中部國際機場|中部國際機場]]的货运航班开始。
*2006年8月1日：前往[[中部國際機場|中部國際機場]]的國際貨物運輸開始。
*2008年1月7日：前往[[關西國際機場|關西國際機場]]的夜間貨物航班开始。(往[[中部國際機場|中部國際機場]]線路改变)
*2008年7月1日：前往[[東京國際機場|東京國際機場]]和[[關西國際機場|關西國際機場]]夜間貨物航班停開。
*2008年11月1日：往[[東京國際機場|東京國際機場]]航班增加班次(1日4往返)。

== 如何前往 ==
*由[[長崎|長崎]]自動車道[[佐賀|佐賀]]大和[[高速公路|高速公路]]出入口使用[[出租汽車|出租汽車]]至佐賀機場要40分鐘。

== 交通 ==
*[[停車場|停車場]]是免費的，可以停放700辆车。

==事故==
*2007年10月5日：一架由[[中華航空|中華航空]]執行包機飛行任務的航班，該班機目的地為[[台灣桃園國際機場|台灣桃園國際機場]]，在起飛階段由於空速表失效，導致機長判斷數值錯誤，過晚拉起機首而衝過跑道，雖然起飛成功了，但破壞了[[跑道|跑道]][[子明燈|子明燈]]1盞。
