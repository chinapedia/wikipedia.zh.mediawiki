'''佐藤優樹'''（{{bd|1999年|5月7日}}），[[日本|日本]]女子組合[[早安少女組。|早安少女組。]]第十期成員，隸屬於演藝經紀公司[[UFG#UP-FRONT_AGENCY|UP-FRONT AGENCY]]旗下。

暱稱是「まーちゃん（Maachan）」、「まさき （Masaki)」，華語區多稱為「麻醬」，應援色為翡翠綠。是早安少女組中第六位[[北海道|北海道]]出身的成員。

== 經歷 ==
*札幌モデル事務所MIST K's COMPANY 模特兒。

=== 2011 ===
*5月8日 - 早安春季巡迴演唱會上宣布將舉行『早安少女組。10期成員『元氣印』選拔甄選』。
*9月29日 - 在日本武道館舉行的『コンサートツアー2011秋 愛BELIEVE～高橋愛 卒業記念スペシャル～』，宣布與[[飯窪春菜|飯窪春菜]]、[[石田亞佑美|石田亞佑美]]、[[工藤遙|工藤遙]]成為[[早安少女組。|早安少女組。]]第十期成員<ref>[http://www.helloproject.com/news/1204/ モーニング娘。10期メンバー　決定！]</ref>。

=== 2012 ===
*1月3日 - 與Hello! Project合作參演「數學♥女子學園」。演出角色為澀谷警備隊員-宮益 りおん。
*1月25日 - 發行出道曲第48單『ピョコピョコ ウルトラ』。
*9月10日 - 10期成員Offical Blog「天気組ブログ」開設。
*9月30日 - 在秋季演唱會仙台夜公演上獲得了第一次的演唱會solo演出機會，獨自演唱[[光井愛佳|光井愛佳]]的演唱曲『私の魅力に気付かない鈍感な人』。
*10月16日 - 發表由[[早安少女組。|早安少女組。]]的[[生田衣梨奈|生田衣梨奈]]、[[石田亞佑美|石田亞佑美]]、'''佐藤優樹'''以及[[S/mileage|S/mileage]]的[[竹內朱莉|竹內朱莉]]組成SATOYAMA movement中的新組合「ハーベスト」，並將發行新單曲販售<ref>[http://www.helloproject.com/morningmusume/news/682/ ハロプロメンバーらの新ユニット「GREEN FIELDS」「ハーベスト」誕生]</ref>。

=== 2013 ===
*1月2日 - 因高燒缺席「Hello! Project 誕生15周年記念ライブ2013冬 」，之後被診斷出「左頸部淋巴腺炎」而未能演出之後的冬季演唱會。
*8月22日 - 於Youtube官方網站上釋出宣傳短片「1／娘。　モーニング娘。 佐藤優樹」<ref>[http://www.youtube.com/watch?v=MNhKy4yYamk 1／娘。　モーニング娘。 佐藤優樹]</ref>。
*9月4日 - 與[[Juice=Juice|Juice=Juice]]成員[[宮本佳林|宮本佳林]]組成SATOYAMA movement新組合「ジュリン」，新單曲以DVD形式推出<ref>[http://www.helloproject.com/news/303/ 【SATOYAMA movement より 新ユニット誕生！！】 ＜ DVDシングルをリリース！＞]</ref>。

=== 2014 ===
*4月23日，與[[勝田里奈|勝田里奈]]、[[植村あかり|植村あかり]]組成SATOYAMA SATOUMI新團體《さとのあかり》並發行單曲『嗚呼、素晴らしき日々よ』<ref>[http://www.helloproject.com/news/22/ SATOYAMA SATOUMI新ユニット結成！シングル発売のお知らせ ]</ref>。

=== 2016 ===
*12月22日，因腰痛等原因就醫診斷為「急性腰痛症、腰椎椎間板ヘルニア」（[[椎間盤脫出症|椎間盤脫出症]]）。

=== 2017 ===
*1月26日，為了能夠專心療養，官方宣布不參加第63張單曲的錄製、拍攝及宣傳活動。
*2月24日，宣布於3月18日開始的春季巡迴演唱會復出。
*3月18日，正式歸隊。

== 人物 ==
*小名為「まーちゃん（麻醬）」，剛加入早安少女組。時說話都是以「麻醬」自稱。
*小學畢業時曾告訴自己不能再以「ま-ちゃん」為第一人稱，但是一直到中學時期都沒有改變；2013年開始慢慢改為「まさー」，2017年的春季巡迴演唱會中「[[女子喧嚷物語|女子喧嚷物語]]」的歌詞裡寫著「自分をまーちゃんって呼ぶの卒業します！」，但那場演唱中的MC還是會以「まーちゃん」為第一人稱。
*早安家族製作人淳君形容麻醬「非常上相的原石」、「對音樂有著一流的理解能力，如斷奏和切分音等節奏理論都能一一指出」並給與其可塑性很高的評價。<ref>Hello Project DVD Magazine vol.29</ref>
*十期成員每位成員生日中都有七這個數字，工藤遙在十期甄選發表介紹時說了十期是Lucky7。
*和家人居住在伏見神社中<ref>http://www.colorful-hp.net/files/images/2012/09/183977.jpg</ref>。
*有2個妹妹。對父母的稱謂為「父(ちち)」、「母(はは)」。
*特技為鋼琴、芭蕾、打鼓和騎馬。曾提到因爲自己從小長大養的寵物狗離世，所以媽媽讓她學習騎馬。
*與[[譜久村聖|譜久村聖]]、[[野中美希|野中美希]]同樣都是左撇子。
*分別被成員[[譜久村聖|譜久村聖]]<ref>{{Citation|last=Skoban Tapir|title=Fukumura Mizuki on Sato Masaki's scary gaze|date=2017-02-10|url=https://www.youtube.com/watch?v=1qYM-Gtcr3M|accessdate=2017-04-09}}</ref>、[[工藤遙|工藤遙]]<ref>{{Citation|last=xxholic25xx|title=モーニング娘。’17 佐藤優樹×工藤遥 (Sato Masaki x Kudo Haruka) ENG SUB|date=2017-01-21|url=https://www.youtube.com/watch?v=52adnpW2sb4|accessdate=2017-04-09}}</ref>和[[羽賀朱音|羽賀朱音]]在不同場合認證「[[傲嬌|傲嬌]]」的屬性，對喜歡的人有著彆扭的態度。
**看到工藤背著[[石田亞佑美|石田亞佑美]]玩耍後表示出強烈的不滿，工藤問是不是也想背著玩，但本人很直接的拒絕了。<ref>{{Citation|last=翔太2 今井|title=まーどぅー その5（佐藤優樹、工藤遥）|date=2016-05-22|url=https://www.youtube.com/watch?v=9CChYcqPYHo|accessdate=2017-04-09}}</ref>
**羽賀形容就像「雖然平常都很傲嬌，但只要男朋友作出點表示便會害羞臉紅的女朋友」<ref>{{Cite news|url=http://ameblo.jp/mm-12ki/entry-12254914352.html|title=[12期]モーニング娘。'17『お姉ちゃんからの、、♡だみょーん。羽賀朱音』|newspaper=モーニング娘。‘17 12期オフィシャルブログ Powered by Ameba|accessdate=2017-04-09|language=ja}}</ref>
**氣上心頭便會說「以後不理會遙了」、「麻醬才不認識遙」、「麻和遙之間玩完了」等說話。<ref>{{Cite news|url=http://ameblo.jp/mm-12ki/entry-12254914352.html|title=[12期]モーニング娘。'17『お姉ちゃんからの、、♡だみょーん。羽賀朱音』|newspaper=モーニング娘。‘17 12期オフィシャルブログ Powered by Ameba|accessdate=2017-04-09|language=ja}}</ref>
**因為工藤曾說過最喜歡的成員是隊長[[譜久村聖|譜久村聖]]，後來隊長提到被麻醬強烈的目光注射。更因為工藤幫手整理隊長的裙子，麻醬立即吐糟「啊！麻醬以後都要不理會遙了」，讓隊長感到非常不好意思。<ref>{{Cite web|url=https://www.youtube.com/watch?v=1qYM-Gtcr3M|title=Fukumura Mizuki on Sato Masaki's scary gaze|accessdate=|author=|publisher=}}</ref>
*不怕生、天真爛漫、愛惡作劇的小惡魔性格
**在Hello! Project 2012 WINTER ハロプロ天国～ファンキーちゃん～ DVD收錄中，MC環節5中被嗣永桃子爆料，跟佐藤優樹還不太熟時，被其突如期來的惡作劇嚇到了。
**四期前輩[[石川梨華|石川梨華]]曾評價麻醬為「比[[辻希美|辻希美]]和[[加護亞依|加護亞依]]更厲害的問題兒」<ref>{{Citation|last=坂本竜馬|title=石川「まーちゃんは辻加護以上の問題児。」|date=2013-04-20|url=https://www.youtube.com/watch?v=I5aZd4bc_Rc|accessdate=2017-04-09}}</ref>
**有「Hello!Project史上最強的[[天然呆|天然呆]]」的稱號<ref>{{Cite web|url=http://entame-watch.com/%E6%80%A7%E6%A0%BC/%E3%83%A2%E3%83%BC%E5%A8%98%E3%80%82%E4%BD%90%E8%97%A4%E5%84%AA%E6%A8%B9%E3%81%AE%E5%A4%A9%E7%84%B6%E9%81%8E%E3%81%8E%E3%82%8B%E3%82%A8%E3%83%94%E3%82%BD%E3%83%BC%E3%83%89%E9%9B%86%EF%BC%81/|title=モー娘。佐藤優樹の天然過ぎるエピソード集！ {{!}} 必見！エンタメウォッチ！|accessdate=2017-04-09|work=entame-watch.com|language=ja}}</ref>
*天性散漫而且好像對很多事情都迷迷糊糊的狀態，卻擁有出眾的表演力，無論歌唱還是跳舞都很上手。
**出道後多次被冠以早安少女組。的「鬼才」、「天才」等名號，但實際上是位「笨蛋天才」，對很多日常事情都非常不擅長（連衣物的大中小碼也分不清楚）<ref>{{Citation|last=Masaki sama|title=[Engsub] Which one is the smallest? - Morning Musume Sato Masaki + Kudo Haruka (MaaDuu)|date=2015-12-24|url=https://www.youtube.com/watch?v=WRTnXtUKB8o&t=1s|accessdate=2017-04-09}}</ref>。
**常有令成員陷入迷霧的發言與無法預測的動作，小時候也被附近住戶說是不可思議的少女。
*因為自小就接受由英語主導的家庭教育，剛出道時並不太會以日語對答，經常詞不達意，讓別人一頭霧水。
**自創過『心情pokupoku的緊張』等形容詞<ref>{{Citation|last=hello nice 動画|title=まーちゃん語録　緊張がポクポクしてきた|date=2016-05-02|url=https://www.youtube.com/watch?v=O6jcyP9oC_M|accessdate=2017-04-09}}</ref>，也曾將『playback』講成『French back』。
*常常被成員說天然呆和自我中心，但是本人一點也不這麼覺得，但是最近有慢慢接受的趨勢。
*很怕孤單，曾在廣播中提及如果一天不打電話給媽媽就會睡不著。
*打簡訊跟回傳訊息的速度被光井愛佳形容為：可怕的快速。
*2016年因為「急性腰痛症、腰椎椎間板突出症」必須休養而暫停活動，成為成團以來首位因傷而缺席單曲錄製的成員，休養期間為了代替無法參與的舞台活動而每天更新部落格。
*目前為止是早安少女組。歴代成員當中最不受控的成員。
*歌唱技巧高、舞蹈水準是[[中島早貴|中島早貴]]、[[宮本佳林|宮本佳林]]都認可的等級。
*在看10期徵選結果公布的影片時，一直認為是絕對不合格的影片。
*會用不一樣的叫法來稱呼成員，而且隨時都會改變。

== 關係 ==
*敢當面直言太妹形象令人畏懼的[[田中麗奈_(早安少女組)|田中麗奈]]， 天不怕地不怕的言行令周遭也驚訝。
**會把田中喊錯成「たなさたん」，但依然非常得到田中的疼愛。
**16歲生日時更得到過前輩親自設計的蛋糕祝賀。<ref>{{Cite news|url=http://ameblo.jp/tanakareina-blog/entry-12024034680.html|title=田中れいな『がんばれーな。』|newspaper=田中れいなオフィシャルブログ「田中れいなのおつかれいなー」Powered by Ameba|accessdate=2017-04-09|language=ja}}</ref>
*很得前輩疼愛，和大家拍照時總是黏得緊緊的，除了同組的田中和[[道重沙由美|道重沙由美]]外，與[[夏燒雅|夏燒雅]]、[[萩原舞|萩原舞]]的關係也很親密。
*[[Juice=Juice|Juice=Juice]]的成員[[宮本佳林|宮本佳林]]視為「不能認輸的對手」，稱是位「擁有天然魅力的人，無論是跳舞還是唱歌都很熟練的人」。
*由於同齡和同期而非常依賴[[工藤遙|工藤遙]]，很多場合中都會看到兩人形影不離，在wota的稱號為「まーどぅー」。
**一遇上困難會毫不猶疑地扯破喉嚨喊工藤過來救場。
**工藤在家族中得到不少後輩的喜愛，讓麻醬覺得自己和工藤的「まーどぅー」關係不再，讓飯們決定以後要跟「まーどぅー」還是「まーさく」（麻醬和[[小田櫻|小田櫻]]），不過飯們大比數支持前者的關係，所以提案最後不了了之。<ref>{{Citation|last=xxholic25xx|title=モーニング娘。’17 佐藤優樹×工藤遥 (Sato Masaki x Kudo Haruka) ENG SUB|date=2017-01-21|url=https://www.youtube.com/watch?v=52adnpW2sb4|accessdate=2017-04-09}}</ref>
**曾說過要把工藤變成自己私人的吊燈
*近來培養出作曲的習慣，喜歡將自己的demo錄成cd，而且會製作練習曲讓後輩用以節奏練習。
*[[牧野真莉愛|牧野真莉愛]]在博客上特別寫到跟麻醬的練習總是讓團隊再次充滿活力和歡樂。<ref>{{Cite news|url=http://ameblo.jp/mm-12ki/entry-12231343276.html|title=[12期]モーニング娘。'17『『佐藤優樹さん♡LOVEりんです♪*ﾟ』牧野真莉愛』|newspaper=モーニング娘。‘17 12期オフィシャルブログ Powered by Ameba|accessdate=2017-04-09|language=ja}}</ref>
