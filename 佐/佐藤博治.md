'''佐藤博治'''（'''Satoh Hiroji'''，{{bd|1925年|2月3日|2000年|6月4日|c=Satoh, Hiroji}}），是一名前[[日本|日本]]男子[[乒乓球|乒乓球]]運動員，出身於[[青森縣|青森縣]]<ref>{{cite web|url=http://www.ittf.com/ittf_stats/All_events3.asp?ID=6283|title=SATOH Hiroji (JPN) - ITTF|deadurl=yes|archiveurl=https://web.archive.org/web/20121017022037/http://www.ittf.com/ittf_stats/All_events3.asp?ID=6283|archivedate=2012-10-17}}</ref>。

1952年至1953年間，他在[[世界乒乓球錦標賽|世界乒乓球錦標賽]]與[[亞洲乒乓球錦標賽|亞洲乒乓球錦標賽]]斬獲數面獎牌。當時他首度使用膠皮與底板之間加入[[海綿_(材料)|海綿]]層的球拍，被視為現代乒乓球拍的起源。雖然一度被認為憑藉技術突破而在賽場上獲得優勢，不過現代乒乓球也在海綿膠皮發明以後才真正成型。

== 經歷 ==
1947年，他在全日本軟式乒乓球選手權大會獲得男子單打冠軍，雙打則和[[金田芳雄|金田芳雄]]搭檔同樣奪得冠軍<ref>{{cite web |url=http://www.jtta.or.jp/handbook/kiroku/kokunai/7_1.html |title=公式大会優勝記録 全日本軟式卓球選手権大会 |publisher=日本卓球協会 |accessdate=2012-11-15 |deadurl=yes |archiveurl=https://web.archive.org/web/20120816040835/http://www.jtta.or.jp/handbook/kiroku/kokunai/7_1.html |archivedate=2012-08-16 }}</ref>。

1952年，原田力藏阿姆斯特朗株式會社社長發明海綿膠皮，他成為首位使用的選手<ref>{{cite web |url=http://www.ittf.com/museum/demos/map_jpn.swf |title=ITTFミュージアムマップ 厚スポンジラケットとテクノロジーの時代 |publisher=ITTF |accessdate=2012-11-15 |deadurl=yes |archiveurl=https://web.archive.org/web/20160307020136/http://www.ittf.com/museum/demos/map_jpn.swf |archivedate=2016-03-07 }}</ref><ref>{{cite web |url=http://www17.ocn.ne.jp/~ya-na-ka/haradaRikizo.htm |title=谷中・桜木・上野公園路地ツアー |publisher= |accessdate=2012-11-15 |deadurl=yes |archiveurl=https://web.archive.org/web/20121103215132/http://www17.ocn.ne.jp/~ya-na-ka/haradaRikizo.htm |archivedate=2012-11-03 }}</ref>，並於[[1952年世界乒乓球錦標賽|1952年世界乒乓球錦標賽]]一舉獲得男子單打冠軍<ref>{{cite web |url=http://www.world-tt.com/images/cgima/img_rep0904i/3-1.gif |title=1926年～2008年 世界選手権および五輪の男子シングルスチャンピオンの年齢分布 |author=伊藤条太 |accessdate=2012-11-15 |deadurl=yes |archiveurl=https://web.archive.org/web/20160304112220/http://world-tt.com/images/cgima/img_rep0904i/3-1.gif |archivedate=2016-03-04 }}</ref>。

由於原本默默無聞卻獲得世界冠軍，對國際乒壇帶來巨大衝擊<ref>{{cite web|url=http://www.independent.co.uk/sport/general/others/chess-in-shorts-the-thrill-of-table-tennis-794888.html |title=Others Chess in shorts: The thrill of table tennis |publisher=インデペンデント |date=2008-03-13 |accessdate=2012-11-15 }}</ref><ref>{{cite web|url=http://www.aolnews.com/2010/12/14/how-pingpong-changed-the-world/ |title=How Pingpong Changed the World |publisher=AOL |author=David Moye |date=2010-12-14 |accessdate=2012-11-15 }}</ref>，因此同年獲頒{{link-ja|朝日體育賞|朝日スポーツ賞}}<ref>{{cite web|url=http://www.asahi.com/shimbun/award/sports/index2.html |title=朝日スポーツ賞（1974～1929年度） |publisher=[[朝日新聞|朝日新聞]] |accessdate=2012-11-15 }}</ref>。
