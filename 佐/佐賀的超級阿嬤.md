{{japanese|japanese=佐賀のがばいばあちゃん|kana=さがのがばいばあちゃん|romaji=Saga no Gabai-baachan}}
《'''佐賀的超級阿嬤'''》，是[[日本|日本]][[作家|作家]][[島田洋七|島田洋七]]於1987年發表的[[自傳|自傳]]式[[小說|小說]]，敘述自己年少時至[[九州|九州]][[佐賀縣|佐賀縣]]投靠外婆的故事，後亦翻拍為[[電影|電影]]、[[漫畫|漫畫]]。本條目僅介紹2006年改編電影。

== 書名 ==
* 中文譯名「阿嬤」為[[閩南語|閩南語]]中「[[祖母|祖母]]、[[外祖母|外祖母]]」之意。
* 原書名「{{lang|ja|佐賀のがばいばあちゃん}}」中「{{lang|ja|がばい}}」一詞出自[[佐賀腔|佐賀方言]]，相當於日語標準語中的「{{lang|ja|とっても}}」，意為「非常」、「很」，為[[副詞|副詞]]。故原書名事實上是不正確／不完整的句子。依當地方言應作「{{lang|ja|がばいすごか}}」，相當於標準語「{{lang|ja|とってもすごい}}」（非常棒、非常厲害）。

== 情節概要 ==
[[第二次世界大戰|第二次世界大戰]]之後，昭廣的媽媽因養不起兒子，不得不把8歲的他交給佐賀縣投靠外祖母，戰後的他們生活雖艱困窮苦，甚至三餐不繼；但外祖母卻以獨到的生活智慧及毅力，將昭廣扶養長大。不過最後在昭廣要離開時，外祖母卻露出了不為人知的一面…

== 衍生作品 ==
=== 電影 ===
此電影由原作者島田洋七撰寫劇本，[[倉內均|倉內均]]執導。由於拍攝經費不足，還特別發起一人一[[萬|萬]][[日元|日元]]的募款活動，募得一[[億|億]]日元的經費。電影在2006年4月先行在九州部份地區首映，同年的6月3日在全日本上映。

=== 後續作品 ===
在《佐賀的超級阿嬤》大熱賣之後，原作者再接再厲，推出了《佐賀阿嬤笑著活下去》和《佐賀阿嬤的幸福旅行箱》《媽媽，我好想妳》《佐賀阿嬤的超元氣料理：簡單而懷念的幸福滋味 》《佐賀阿嬤給我的人生禮物》《我的老相好──「佐賀的超級阿嬤」特別劇場》《阿嬤，我要打棒球！──佐賀的超級阿嬤》。

== 榮譽 ==
《佐賀的超級阿嬤》已連續多個月高踞台灣網上書店[[博客來|博客來]]的排行榜頭十位，亦榮獲2006年「博客來年度百大」年度暢銷100獎項。
