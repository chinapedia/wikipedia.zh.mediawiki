'''佐藤麻美'''（{{bd|1975年|2月23日|catIdx=Sato Mami}}），是一位[[日本|日本]]女性[[播報員|播報員]]，目前於[[北海道電視台|北海道電視台]]（HTB）服務。身高163公分，血型A型。

暱稱是「マミ」、「マミちゃん」、「マミ部長」。

== 來歷 ==
1998年，畢業於[[南山大學|南山大學]]經濟學部，之後進入北海道電視台擔任主播。他的首個工作是「遊覽夢想的土地」（{{lang-ja|遊々ゆめらんど}}）。

2003年3月開始，於[[綜藝節目|綜藝節目]]「{{link-ja|大泉洋吃遍日本|おにぎりあたためますか}}」（又譯：全國省道挑食中）演出，知名度也因此上升，再加上{{link-ja|Impress TV|インプレスTV}}的配信以及也在[[SKY_PerfecTV!|SKY PerfecTV!]]（{{link-ja|朝日電視台Channel!|テレ朝チャンネル}}）上播放，使得知名度又擴展至全國方面。另外，也同時演出[[鬧鐘電視|鬧鐘電視]]（[[富士電視台|富士電視台]]）等其他電視網的電視節目。

於節目「{{link-ja|大泉洋吃遍日本|おにぎりあたためますか}}」中，被稱作「{{lang|ja|マミ部長}}」。

興趣是[[旅行|旅行]]、觀賞[[電影|電影]]、外出用餐。對於[[料理|料理]]相當擅長，自己曾經展露可以不需[[庖丁|庖丁]]，而臨時起意運用[[碟|碟]]的尖銳處切食物。

2006年3月，取得「{{link-ja|蔬菜&水果專家|ベジタブル&フルーツマイスター}}」的資格（3階段的中級課程，及格率約兩成左右）。

2007年12月，目標取得「調味料專家」資格，透過相關過程後順利通過。

2008年1月6日，與同電視台的新人主播[[大野惠|大野惠]]組隊參加「{{link-ja|面板猜謎25|パネルクイズ アタック25}}」。

2009年5月，獲得關於研究與美容相關的蔬菜和果物等的「Beautiful美容自我顧問」的資格。。

全部的[[蔬菜|蔬菜]]和[[肉類|肉類]]都喜歡（特別關注新型蔬菜），尤其是野生肉類。

不喜歡[[包子|包子]]等餡類食物（也有例外，例如{{link-ja|白松餡餅|白松がモナカ}}）、蒸蠶[[蛹|蛹]]（一種韓國食物）。食物以外的有[[鬼屋|鬼屋]]。

2009年8月15日，在節目「{{link-ja|Skip (情報節目)|スキップ (情報番組)}}」以及同年9月4日的[[播客|播客]]，公開結婚的消息。

2012年7月31日，節目「{{link-ja|大泉洋吃遍日本|おにぎりあたためますか}}」播出後，在8月2日宣布開始休[[產假|產假]]<ref>[http://www.htb.co.jp/announcers/mami/2012/08/ 2012年08月02日（木）ご報告♪] {{webarchive|url=https://web.archive.org/web/20130612052806/http://www.htb.co.jp/announcers/mami/2012/08/ |date=2013年6月12日 }}</ref>。11月，長男出生（2013年1月23日）。

2013年6月25日，經過九個月的產假後，正式復歸。<ref>[http://www.htb.co.jp/announcers/mami/2013/06/ 2013年06月26日（水）復帰！] {{webarchive|url=https://web.archive.org/web/20130712215304/http://www.htb.co.jp/announcers/mami/2013/06/ |date=2013年7月12日 }}</ref>

== 人物 ==
* 到目前為止印象最深的工作是，到[[澳洲|澳洲]][[跳傘|跳傘]]並大喊節目名稱。
* 認為自己容易掉淚。
* 有吃下{{link-ja|碗子荞麦|わんこそば}}（一種[[蕎麥麵|蕎麥麵]]）145碗的紀錄。（達成節目「飯糰要加熱嗎」的企畫）
* 曾經在三天兩夜的外景取材中就胖了3.5公斤，於是下一次取材時特別[[節食|節食]]（但從2007年10月起，體重卻難以下滑甚至控制<ref>{{cite web |url=http://www.htb.co.jp/announcers/old_blog/mami/log_51.html |title=存档副本 |accessdate=2013-06-12 |deadurl=yes |archiveurl=https://web.archive.org/web/20141129050214/http://www.htb.co.jp/announcers/old_blog/mami/log_51.html |archivedate=2014-11-29 }} 体重増加…</ref>），正在苦戰中）。每天早晚必定要秤量體重，也因此於未婚時期表示「我不和比我吃更少的人結婚」。
* 不擅長[[大胃王|大胃王]]比賽，於節目企畫經常挑戰失敗。
* 母親的娘家賣酒，家裡都用當地語言「函館話」對話。
* 母親原姓[[佐藤|佐藤]]。
* 最得意的[[卡拉OK|卡拉OK]]歌曲是[[岩崎良美|岩崎良美]]的「[[Touch_(岩崎良美歌曲)|Touch]]」，另外對於「{{link-ja|愛的迷戀|魅せられて}}」和「[[愛情誕生的日子|愛情誕生的日子]]」也唱得不錯。此外，也覺得自己擅長模仿Bobby Ologun和[[真矢美季|真矢美季]]（儘管被認為並不像）。
* 旅行前會徹底研究相關的情報。
* 大學時期，在[[鶴舞車站|鶴舞站]]附近的「串太郎」打工兩年。
* 北海道以外，最想住在[[金澤市|金澤]]（魚相當美味、也有加賀蔬菜）。
* 也在關注[[生態運動|生態運動]]，使用「My筷」吃飯（加上節目「飯糰要加熱嗎」的相關物品）。「My筷」也是節目的商品企畫。
* 「大泉洋吃遍日本」DVD的第一波附有預約特典特別頭巾並監修。
