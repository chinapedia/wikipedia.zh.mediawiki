'''《怒海情仇》'''(前名'''《胜利者》''')，2013年上映于中国大陆的民初剧，导演金鳌勋、黄伟杰，主演戴娇倩、霍政谚、蔡妍、张天其、李解。<ref>{{cite web |url=http://yule.sohu.com/20130225/n366946276.shtml |title= 年代剧《胜利者》各地热播 刘一演绎纨绔公子哥|accessdate=2013 |publisher=搜狐网 |work= |language=zh-hans}}</ref><ref>{{cite web |url=http://ent.sina.com.cn/v/2012-02-27/11003565749.shtml |title=资料:电视剧《胜利者》主角-蔡妍饰程芸 |accessdate=2012 |publisher=新浪网 |work= |language=zh-hans}}</ref><ref>{{cite web |url=http://ent.qq.com/a/20111201/000227.htm |title=年代偶像剧《胜利者》热拍 蔡妍张天其浪漫携手 |accessdate= 2011|publisher= 腾讯网|work= |language=zh-hans}}</ref>2011年杀青，2013年2月19日[[江苏电视台|江苏城市频道]]首播。

==剧情==
该剧讲述1927年到[[抗日战争|抗日战争]]期间，程家三兄妹大哥程涛、二弟程毅、三妹程芸投身[[抗日救亡运动|抗日救亡运动]]的人生故事。
