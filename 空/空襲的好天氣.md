'''空襲的好天氣'''（'''"Viðrar vel til loftárása"'''）是[[冰岛|冰岛]]乐团[[詩格洛絲|詩格洛絲]]（[[Sigur_Rós|Sigur Rós]]）第二張錄音室專輯''[[Ágætis_byrjun|Ágætis byrjun]]''的第七首歌曲，這首歌另外也收錄在單曲"[[Svefn-g-englar|Svefn-g-englar]]"裡頭。

== 歌名 ==
[[Sigur_Rós|Sigur Rós]]非常諷刺地引述了冰島氣象家在[[科索沃|科索沃]]戰爭時說的話：「í dag viðrar vel til loftárása<ref name="trivia">{{Cite web |url=http://www.sigur-ros.co.uk/band/trivia.php |title=sigur rós - trivia |accessdate=4 January 2009 |work=sigur-ros.co.uk}}</ref> (意思是今天是一個空襲的好天氣)」。

== 音樂錄影帶 ==
'''Viðrar vel til loftárása'''的音樂錄影帶引起了電影製作的爭議<ref>{{Cite web |url=http://www.sigur-ros.co.uk/news/?m=20020415 |title=eighteen seconds before sunrise - sigur rós news » 2002» April» 15 |accessdate=7 January 2009 |work=sigur-ros.co.uk}}</ref><ref>{{Cite web |url=http://www.nme.com/blog/index.php?blog=10&title=is_this_the_most_controversial_video_eve_2&more=1&c=1&tb=1&pb=1 |title=Is This The Most Controversial Video EVER? |accessdate=7 January 2009 |work=nme.com}}</ref>。故事發生在1950年代的[[冰島|冰島]]，整部影片最重要的一幕就是兩個小男孩在[[足球賽|足球賽]]結束之後躺在地上[[親吻|親吻]]對方，後來其中一人被其父親生氣的抓走。[[Sigur_Rós|Sigur Rós]]在這部影片中都擔任了客串演出<ref>{{Cite web |url=http://www.sigur-ros.co.uk/news/?m=20010824 |title=eighteen seconds before sunrise - sigur rós news » 2001» August» 24 |accessdate=4 January 2009 |work=sigur-ros.co.uk}}</ref><ref>{{Cite web |url=http://fat-cat.co.uk/fatcat/mediaItem.php?id=3 |title=FatCat Records : Media |accessdate=7 January 2009 |work=fat-cat.co.uk}}</ref>，[[Jón_Þór_Birgisson|Jónsi]]飾演足球隊教練、[[Orri_Páll_Dýrason|Orri]]飾演記分員、[[Georg_Hólm|Georg]]飾演裁判、而[[Kjartan_Sveinsson|Kjartan]]則是飾演觀眾之一。除此之外，影片中男孩喝的飲料瓶子包裝上的胎兒圖像是來自''[[Ágætis_byrjun|Ágætis byrjun]]''專輯封面。

音樂錄影帶的製作於2001年秋天開始進行，演員的招募<ref>{{Cite web |url=http://www.sigur-ros.co.uk/news/?m=20010713 |title=eighteen seconds before sunrise - sigur rós news » 2001» July» 13 |accessdate=7 January 2009 |work=sigur-ros.co.uk}}</ref>在[[冰島|冰島]]首都[[雷克雅維克|雷克雅維克]]進行，當地也是拍攝影片的重要景點。此音樂錄影帶由[[Arni_&_Kinski|Arni & Kinski]]導演、[[Stefán_Árni_Þorgeirsson|Stefán Árni Þorgeirsson]]和[[Sigurður_Kjartansson|Sigurður Kjartansson]]指導，另外影片獲得了2002年[[冰島|冰島]]音樂獎的"最佳影片獎"<ref>{{Cite web |url=http://www.sigur-ros.co.uk/news/?m=20020210 |title=eighteen seconds before sunrise - sigur rós news » 2002» February» 10 |accessdate=4 January 2009 |work=sigur-ros.co.uk}}</ref>。
