这是一个'''[[空间望远镜|空间望远镜]]'''列表。这里列表是按[[电磁波谱|电磁波谱]]的主要频段分类的，即自高频至低频分为[[伽玛射线|伽玛射线区]]、[[X射线|X射线区]]、[[紫外线|紫外线区]]、[[可见光|可见光区]]、[[红外线|红外线区]]、[[微波|微波区]]和[[无线电|无线电区]]。有些望远镜工作在上述中的多个频段，它们会在每一个频段中都被列出。对于采集粒子（如[[宇宙射线|宇宙射线]]原子核、电子等）的空间望远镜，以及探测[[引力波|引力波]]的空间望远镜（主要是[[LISA|LISA]]）也在这个表中列出。对于探测任务仅局限于[[太阳系|太阳系]]，包括[[太阳|太阳]]、[[地球|地球]]以及太阳系中其他[[行星|行星]]的探测器则被排除在外，关于这些探测器请参见[[太阳系探测器列表|太阳系探测器列表]]。

当望远镜处在[[地心轨道|地心轨道]]上时，关于它的高度的两个参数会以千米为单位给出，分别为初始轨道的[[近地点|近地点]]和[[远地点|远地点]]，即望远镜与地球[[质心|质心]]（准确说是望远镜与地球构成的[[二体问题|两体系统]]的质心）距离的最大值和最小值。类似的，如果望远镜处在[[日心轨道|日心轨道]]上，这两个参数也会相应地给出，但此时的单位是[[天文单位|天文单位]]（{{lang|en|AU}}）。

== 伽玛射线 ==
{{see also|伽玛射线天文学}}
伽玛射线望远镜采集并测量宇宙中独立的高能伽玛射线源。[[伽玛射线|伽玛射线]]会被大气层吸收，因此对伽玛射线的观测需要依靠高纬度的气球或太空中的探测器。伽玛射线可以来自[[超新星|超新星]]、[[中子星|中子星]]、[[脉冲星|脉冲星]]和[[黑洞|黑洞]]；而具有极高能量的[[伽玛射线暴|伽玛射线暴]]也已经被探测到，但还未能被识别。<ref name="gamma ray">{{cite web | url=http://science.hq.nasa.gov/kids/imagers/ems/gamma.html | title=Gamma rays | publisher=[[美国国家航空航天局|NASA]] | accessdate=2009-06-04}}</ref>

== X射线 ==
{{see also|X射线天文学}}
X射线望远镜用於测量高能的[[X射线|X射线]]。与伽玛射线类似，在大气层中X射线会被大幅吸收，因此观测它们需要在高空或太空中进行。发射X射线的天体有很多种，[[星系团|星系团]]可以通过[[活动星系核|活动星系核]]中的[[超大质量黑洞|超大质量黑洞]]来发射X射线，而星系中的天体如[[超新星遗迹|超新星遗迹]]、[[恒星|恒星]]、带有一颗[[白矮星|白矮星]]的[[双星|双星]]（[[激变变星|激变变星]]）、中子星或黑洞（[[X射线双星|X射线双星]]）。有些太阳系中的天体也会发射X射线，而[[月球|月球]]不仅能够反射来自太阳的X射线，[[太阳风|太阳风]]中的高能粒子（主要是[[质子|质子]]）高速撞击到月球表面后还会激发月球表面的物质粒子，从而产生X射线。宇宙还存在有很多无法一一辨认的X射线源，一般认为它们发射出的X射线集体形成了观测到的[[X射线背景|X射线背景]]。

== 紫外线 ==
{{see also|紫外線天文学}}

== 红外线 ==
{{see also|红外天文学}}

== 引力波 ==
{{See also|引力波天文学}}
探测[[引力波|引力波]]的望远镜是一类新兴的空间望远镜，[[引力波天文学|引力波天文学]]的出现标志着打开了一扇有别于传统电磁天文学的探索宇宙的新的窗口。引力波是质量产生的时空涟漪，空间望远镜所能探测到的引力波来自银河系中的双星以及河外星系的超大质量黑洞合并等天体。
