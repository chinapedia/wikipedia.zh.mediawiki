《'''空军报'''》是[[中国人民解放军空军政治工作部|中国人民解放军空军政治工作部]]主办的报纸，为[[中国共产党中国人民解放军空军委员会|空军党委]]机关报，1958年创刊。

==简介==
1950年4月15日，《空军报》的前身《人民空军》杂志创刊<ref name=lujun/>。[[毛泽东|毛泽东]]、[[刘少奇|刘少奇]]、[[周恩来|周恩来]]、[[朱德|朱德]]、[[邓小平|邓小平]]等领导人为该杂志创刊题词。毛泽东的题词是：“创建一支强大的空军，歼灭残敌，巩固国防。”<ref>{{Cite web |url =http://news.163.com/13/1004/07/9AAVKIAB00014JB5.html  |title = 《悠悠蓝天情》 余启元：空军机关工作的片段记忆 |author =  |date = 2013-10-04 |publisher =网易  |language =  |accessdate =  |quote =  }}</ref>刘少奇的题词是：“强大的人民中国，必须有强大的人民空军与民航事业。”<ref>{{Cite web |url =http://www.caac.gov.cn/website/old/D1/JNTZ/200711/t20071130_9432.html  |title = 铭记重要批示 迈向民航强国 |author =  |date = 2007-11-30 |publisher =中国民用航空局  |language =  |accessdate =  |quote =  }}</ref>周恩来的题词是：“为建立人民空军而努力。”<ref>赵玉明，周恩来建国以来的题词(一)，红岩春秋2006年第1期</ref>朱德的题词是：“努力学习，掌握技术，为建设一支新式的强大的人民空军而奋斗。”<ref>{{Cite web |url = http://agzy.youth.cn/mzh/jng/zd/tcsj/201402/t20140213_4653483.htm |title = 为《人民空军》杂志创刊题词（1950年4月6日）|author =  |date =  2014-02-13  |publisher = 中国青年网   |language =  |accessdate =  |quote =  }}</ref>邓小平的题词是：“强大的空军和强大的陆军结合起来，我们将是无敌的！”<ref>{{Cite web |url =http://news.xinhuanet.com/zgjx/2007-01/14/content_5604674.htm  |title = 邓小平为《人民空军》杂志创刊题词 |author =  |date =  2007-01-14|publisher = 中国记协网 |language =  |accessdate =  |quote =  }}</ref>

1950年12月底，组建不足一年的[[中国人民解放军空军|中国人民解放军空军]]以[[中国人民志愿军|中国人民志愿军]]空军的名义到朝鲜参加[[抗美援朝战争|抗美援朝战争]]。这一时期，《人民空军》杂志开设《志愿军空军通讯》专栏，用封面、封底、图片等形式，定期发布中国人民志愿军空军战况<ref name=lujun/>。

1958年3月10日，空军党委决定即日起在《人民空军》杂志的基础上，创办《人民空军》报。1959年2月13日，《人民空军》报更名为《空军报》<ref>{{Cite web |url = http://news.xinhuanet.com/newmedia/2008-03/28/content_7875944.htm |title = 创刊50年的《空军报》：浓缩一个军种的简史 |author =  |date = 2008-03-28 |publisher =新华网  |language =  |accessdate =  |quote =  }}</ref><ref name=lujun>{{Cite web |url = http://mil.sohu.com/20160601/n452347458.shtml |title =  陆军机关报《人民陆军》问世 司令政委合写头条|author =  |date =2016-06-01  |publisher =搜狐  |language =  |accessdate =  |quote =  }}</ref>。《人民空军》报当时出至第146期，为了有别于《人民空军》杂志而更名为《空军报》<ref>{{Cite web |url = http://cjrb.cjn.cn/html/2016-01/23/content_5506157.htm |title =七大军区报宣告停刊  |author =  |date = 2016-01-23 |publisher = 长江日报 |language =  |accessdate =  |quote =  }}</ref>。

1965年3月，[[林彪|林彪]]之女[[林立衡|林立衡]]（林豆豆）到空军报社工作。1969年10月，经[[中共中央政治局委员|中共中央政治局委员]]、[[中国人民解放军副总参谋长|中国人民解放军副总参谋长]]兼空军司令员[[吴法宪|吴法宪]]安排，25岁的林立衡被任命为《空军报》副总编辑，此前不久，林立衡的弟弟[[林立果|林立果]]已被任命为空军司令部办公室副主任兼作战部副部长。林立衡一直担任《空军报》副总编辑至1971年[[九一三事件|九一三事件]]之后。《空军报》自创办起，报头便用毛泽东字体，是从毛泽东墨迹中辑录而成。林立衡在空军报社工作期间，应刚从空军政委改任空军司令员的吴法宪之请，林立衡经[[叶群|叶群]]通过[[中共中央办公厅|中共中央办公厅]]主任[[汪东兴|汪东兴]]，请[[毛泽东|毛泽东]]为《空军报》题写了报头。毛泽东题写了“空军报”三个字，并在题字上写了一行小字：“送给林彪同志的女儿林豆豆。”吴法宪将毛泽东题写的报头转交空军报社社长朱鸿。《空军报》很快启用了新报头，并沿用至今<ref name=lin>张聿温，林豆豆的十年，同舟共进2016年12期</ref>。

2009年11月11日，《空军报》正式扩期改版<ref>王斌，在变革创新中提升舆论引导力——《空军报》扩期改版的实践与思考，新闻战线2010年第4期</ref>。2010年5月，王斌任社长后，推动报网融合<ref name=baowang/>。 

《空军报》原先由中国人民解放军空军政治部主办。2016年，在[[深化国防和军队改革|深化国防和军队改革]]中，中国人民解放军空军政治部改为中国人民解放军空军政治工作部，《空军报》继续由中国人民解放军空军政治工作部主办。

==下辖媒体==
空军报社下辖的媒体除了《空军报》之外，还有《中国空军》杂志、《基层政工读物》杂志，以及空军报社网站。

*《中国空军》：是中国人民解放军空军唯一对外公开发行的大型综合性军事双月刊<ref name=jixie/>。1986年4月1日，空军党委创刊该杂志，是当时空军最有影响的3种报刊之一。1989年是中国人民解放军空军建立40周年，空军司令员[[王海|王海]]、政委[[朱光_(空军中将)|朱光]]想请中央军委主席邓小平为空军题词。经过联系，最后邓小平采用了为《中国空军》杂志题写刊名的形式，在1989年10月23日题写了“中国空军”四字刊名。半个月后，1989年11月9日，中共十一届五中全会根据邓小平的请求，同意邓小平辞去中央军委主席职务。此次题字就成了邓小平在中央军委主席任内为空军最后一次题字。1989年11月10日下午，空军建立40周年纪念大会召开，新任中央军委主席[[江泽民|江泽民]]率领新一届中央军委领导成员参加纪念大会。为纪念空军建立40周年,《空军报》发行专刊，登载了邓小平题写的“中国空军”四字刊名。《中国空军》杂志随即以邓小平的题字作为刊名，沿用至今<ref>徐秉君，邓小平致力于建设强大的人民空军，2009年第10期</ref>。
*《基层政工读物》：是由空军政治工作部主管，空军报社主办，面向全军发行的政治工作类杂志。1994年1月创刊<ref name=jixie/>。
*空军报社网站：集合了《空军报》网络版、“一报两刊”电子版、投稿中心、博客、论坛、QQ群等功能。2010年左右开始筹建，后来正式运行<ref name=baowang/>。
