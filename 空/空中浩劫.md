《'''空中浩劫'''》（{{lang-en|''Mayday''}}、{{lang|en|''Air Crash Investigation''}}、{{lang|en|''Air Emergency''}} 或{{lang|en|''Mayday: Air Disaster''}}）是一個由[[加拿大|加拿大]]製作的[[紀錄片|紀錄片]]節目，由Cineflix制作，早期节目星芒媒体（astral Media）也有参与制作，其英文原版分別在世界各地的[[國家地理頻道|國家地理頻道]]及[[探索頻道|Discovery頻道]]播出，加拿大[[魁北克省|魁北克省]]在Canal D以法文播出，中文版则由多个电视台获得其版权。此節目主要介紹1970年代[[黑盒子|黑盒子]]發明之後所發生的近代重大[[航空|航空]]事故。目前已经播至第十九季。在第三季時，有三集穿插[[鐵道|鐵道]]事故及[[航海|航海]]事故（這三集在其他國家播出时未必归为空中浩劫系列，台湾国家地理频道将此三集列为「浩劫现场」系列）。

== 名稱 ==
此節目在不同頻道播出有不同的[[英文|英文]]名稱，於加拿大的[[Discovery頻道|Discovery頻道]]播出時稱為「Mayday」，在[[美國|美國]]播出時稱為「Air Emergency」，而在[[国家地理频道|国家地理频道]]播出時稱為「Air Crash Investigation」。在[[中国|中国]]，主要称“空中浩劫”，各电视台播出时采用的译名则较为混乱。[[CCTV-9|CCTV-9]]的「寰宇地理」时段采用的译名为「空中有难」，而在[[马来西亚航空370号班机事故|马来西亚航空370号班机]]事故发生后，CCTV将本节目引进后翻译成“空难调查”，先锋纪录频道采用的译名则是「空难日」，中国教育频道则将其列为「空难奇案」系列。此外亦有不少系列节目或多或少引用空中浩劫系列纪录片的片段，但并非所有系列节目均为专门讲述空难之节目。

== 製作手法 ==
《空中浩劫》會以模擬演出的方式，分別以[[乘客|乘客]]、[[機師|機師]]等不同人的視角，从事故发生前一段时间到事故發生、後續善後及調查為止，重現事件的整個過程。<ref>{{cite web |url = http://www.dvdtalk.com/reviews/36956/mayday-air-disasters/ |title = Mayday — Air Disasters |first = Paul |last = Mavis |date = 15 April 2009 |publisher = [[DVD_Talk|DVD Talk]]}}</ref><ref name='Franck Tabouring review'>{{cite web |url = http://www.dvdverdict.com/reviews/maydayairdisasters.php |title = Mayday — Air Disasters |first = Tabouring |last = Mavis |date = 23 April 2009 |publisher = [[DVD_Verdict|DVD Verdict]]}}</ref>

在片中會穿插對於乘客及機師的訪談（若有[[生還者|生還者]]願意受訪時）或播放[[航空交通管制|航空交通管制]]錄音（若有事故發生時錄音，且相關單位允許公開時），另外也會訪問空難調查人員、目擊者等。取材以事故調查報告及相關[[新聞|新聞]]報導為主；若因調查單位不公開報告以致無法取得事故調查報告時，則會徵詢相關專家的說法以拼湊出全貌。

同時，為了讓觀眾了解相關事故的發生原因，片中會加入相關航空知識的简介。

《空中浩劫》會對於[[飛機|飛機]]外觀、[[航空公司|航空公司]]塗裝、[[空服員|空服員]][[制服|制服]]、[[機場|機場]]等做考究，但也並非100%做到與當時完全一樣，偶有不相符之處，比如飞机驾驶舱混用，飞机外貌混用，飞机涂装使用错误，宽体机事故使用窄体机机舱等。另外，早期的集數有較多重複使用的片段（如大雨中的跑道畫面）。模擬演出時，會找與當事人相貌近似的[[演員|演員]]來扮演該角色。

== 评价 ==
此节目在多个国家收到不同观众的赞扬<ref>{{cite web |url = http://www.dvdtalk.com/reviews/36956/mayday-air-disasters/ |title = Mayday — Air Disasters |first = Paul |last = Mavis |date = 15 April 2009 |publisher = [[DVD_Talk|DVD Talk]]}}</ref><ref name='Franck Tabouring review'>{{cite web |url = http://www.dvdverdict.com/reviews/maydayairdisasters.php |title = Mayday — Air Disasters |first = Tabouring |last = Mavis |date = 23 April 2009 |publisher = [[DVD_Verdict|DVD Verdict]]}}</ref>，本节目制作者也因为制作世纪大空难而获得了第25届[[双子星大奖|双子星大奖]]的“最佳纪录片画面制作奖”。此节目第五季也在双子星大奖的“最受民众喜爱的节目”中被提名<ref name='25th Gemini Award'>{{Citation |title = Best Picture Editing in a Documentary Program or Series |url = http://www.geminiawards.ca/gemini25/catpop.cfm?catid=5004 |year = 2010 |publisher = [[Gemini_Award|Gemini Award]]}}</ref>。
