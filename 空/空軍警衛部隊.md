'''中華民國空軍警衛部隊'''，為直屬[[中華民國空軍|中華民國空軍]]的一支{{Link-en|空軍步兵及特種部隊|Air force infantry and special forces units|地面武裝步兵部隊}}，主要任務為守備全國各機場與空軍各戰略要地之安全警備衛哨勤務，並肩負實行機場反劫機、反空降等特種作戰任務。另空軍警衛部隊同時具有空軍司法警察身份，於空軍各所屬基地執行維持營規軍紀規定，若遭逢空軍部隊駐地發生暴亂情形時，警衛部隊得奉命依法執行鎮暴任務。

== 沿革==
空軍警衛部隊，於1925年創建於廣州，當時名為航空掩護隊，下轄步兵1個連，隸屬於[[國民革命軍|國民革命軍]]總司令部航空處。<ref name="空軍警衛部隊歷史">{{cite book |author=空軍警衛旅司令部 |title= 空軍警衛部隊歷史 |publisher= |year=出版年不詳|month=  |isbn=  |url= http://sinica.summon.serialssolutions.com/#!/search?bookMark=ePnHCXMw42JgAfZbUzkZJJ6v3PVid--LtcteLJz9snnFy1ldz9Zuf9q_iZtB3s01xNlDtxi89S8eOk4Rn2RkYAy6mQ-0B4iQCgDkXSda}}</ref><ref name="空軍防砲警衛部隊建軍紀念特刊">{{cite book |author=周遨, 張新民 |title= 空軍防砲警衛部隊建軍紀念特刊: 防砲六十週年、警衛七十週年、防警部四週年 |publisher=空軍防砲警衛司令部 |year=1995|month=  |isbn=9570051388  |url= https://books.google.com.tw/books/about/%E7%A9%BA%E8%BB%8D%E9%98%B2%E7%A0%B2%E8%AD%A6%E8%A1%9B%E9%83%A8%E9%9A%8A%E5%BB%BA%E8%BB%8D%E7%B4%80%E5%BF%B5%E7%89%B9.html?id=xy27AQAACAAJ}}</ref>

1927年，該部隊隨著[[國民革命軍北伐|國民革命軍北伐]]而移駐南京，航空掩護隊隨著空軍發展需要，擴編為步兵3個連，工兵1個連。1930年5月，擴編為航空掩護大隊，轄步兵5個連。<br>
1934年5月，編為特務大隊，改隸[[國民政府軍事委員會|國民政府軍事委員會]]特務團，並增編重機槍連，以及接收軍委會撥交之高射炮兩連。1935年8月，再改隸[[陸軍官校|陸軍官校]][[国军德械师#.E6.95.99.E5.B0.8E.E7.B8.BD.E9.9A.8A|教導總隊]]特務大隊，下轄步兵8連、工兵1連、小砲2連、重機槍1連。同年11月，所屬之8個步兵連，改編為2個步兵營，其餘部隊直屬大隊部。 <br>
1936年11月，廣東空軍特務營，及航校掩護營，均併入掩護大隊，並改編為特務團。1937年，由於空軍之日漸擴展，警衛勤務益趨繁重，兵力擴編為8個步兵營與6個直屬連。 <br>
1938年9月，成立航空特務旅 (全中國5個空軍軍區各設1個旅，每旅下轄3個團) <ref name="羅中揚將軍回憶">{{cite book |author= |title= 驀然迴首感恩深: 羅中揚將軍回憶
 |publisher=史政編譯室 |year=2003 |month=  |isbn=9570149388  |url= https://books.google.com.tw/books?id=90IPtwAACAAJ&hl=zh-TW&num=15}}</ref>，萬用霖將軍升任旅長。
<br>
為適應空軍建軍要求，1946年7月陸續擴充至9個步兵團。1947年8月，改組為空軍地面警衛司令部，下轄2個步兵旅，第一旅駐[[上海|上海]]，第二旅駐[[漢口|漢口]]和兩個獨立團。 <br>
1949年移駐台灣而全面縮編；1950年7月1日改組為空軍警衛旅司令部，下轄3個步兵團 與1個幹部訓練班，仍隸屬於空軍總司令部，首任旅長為易國瑞將軍。<br>1968年1月，空軍警衛旅，奉令精簡團號，成為警衛營；下轄9個警衛營。
1984年7月，空軍警衛旅改番號為空軍警衛指揮部。 <br>
1991年9月1日執行「天祥四號演習」，併編[[空軍防空砲兵指揮部|空軍防空砲兵司令部]]，成立空軍防砲警衛司令部，下轄4個防砲警衛指揮部（911指揮部-台北、912指揮部-台中、913指揮部-高雄、914指揮部-花蓮）、1個三軍防空砲兵基地訓練中心、1個幹部教育訓練班、14個防砲營、11個警衛營。 <ref name="空軍防空砲兵指揮部沿革簡史">{{Cite web |url= http://air.mnd.gov.tw/Publish.aspx?cnid=1696&p=12547&Level=2 |title= 空軍防空砲兵指揮部沿革簡史 |author=  |date= 2016-02-24 |publisher= 中華民國空軍司令部 |language= zh-tw |accessdate= 2016-06-23 |archive-url= https://web.archive.org/web/20131024052332/http://air.mnd.gov.tw/Publish.aspx?cnid=1696&p=12547&Level=2 |archive-date= 2013-10-24 |dead-url= yes }}</ref><br>
2006年1月1日執行「和平專案」，空軍警衛駐機場各營移撥[[中華民國憲兵|國防部憲兵司令部]]。空軍警衛部隊自此正式走入歷史 <ref name="空軍防空砲兵指揮部沿革簡史"></ref><br>

== 編制 ==
空軍警衛部隊自[[精實案|精實案]]後核定編制爲11個營（33個連）；每營設有營部連，下有3-4個連；各連除連部及3個排，每個排下轄3個警衛班；一個警衛班編制11人，區分為二名士官及九名士兵。
另外在每連的連部設有1分鐘機動待命班，由內部各單位人員輪值輪調。
*防空警衛司令部（桃園八德懷德山莊）
**幹訓班（雲林虎尾）
**指揮部（北911、中912、南913、東914）
***本部連
***'''警衛營'''（1～11營；各約500名；營部 + 4個連）
****營部連（營長 + 副營長 + 營輔導長〖政戰〗 + 營士官長 + 参一至參六）
****軍犬班
****'''警衛連'''（120名；連部 + 3-4個排）
*****連部（連長 + 副連長 + 輔導長〖政戰〗 + 士官長 + 行政士官）
*****1分鐘機動待命班（各單位抽調輪值；2輛[[悍馬_(軍用汽車)|悍馬車]]上各架一副[[T75班用機槍|T75班用機槍]]，並配一名背77(拐拐)的通訊兵）
*****'''警衛排'''（35名；排長 + 副排長 + 3個班）
******'''警衛班'''（11名；班長 + 副班長 + 3個伍）
*******伍（伍長 + 2名士兵）

警五營（第15、16、17、18連）: [[桃園空軍基地|桃園]] （18連外防連，龜山彈藥庫，空軍懷德山莊，變電所）<BR>

警八營（第27、28、29、30連）: [[臺南機場|台南]]<BR>
警九營（第31、32、33、34連）: [[中華民國空軍軍官學校|高雄岡山]]<BR>
警十營（第35、36、37連）: [[空軍志航基地|台東志航]]<BR>

38、39連於1989年9月成立，該年12月38連開始頻繁換防，從虎尾.桃園，新竹，台中，1991年回防[[佳山計劃|花蓮佳山]]；39連在初期接防[[馬公機場|澎湖馬公]]（原單位陸軍防砲裁撤），1992年回防台東[[建安計畫|建安計畫]]，警衛38、39連初期為換防連，接替上述各警衛連整訓，後調[[花蓮機場|花蓮]]編制為警十一營。<BR>

1991年成立了40連，編制隸屬警七營第四連；1992年初，39連 調回台東後，40連接手駐防澎湖馬公機場。<BR>
*警衛獨立14連即為空軍儀隊
<P><BR>
從警一連到警四十連，分佈在全國各空軍機場、油彈庫、雷達站等所有空軍的要塞。每個連都必須輪流到[[虎尾空軍基地|虎尾指揮部]]進行整訓，但駐守松山機場的四個連不必整訓<BR>
