'''空知郡'''（{{jpn|j=空知郡|hg=そらちぐん|rm=Sorachi gun|f=y}}）為[[日本|日本]][[北海道|北海道]]的郡，同時跨越了[[空知綜合振興局|空知綜合振興局]]和[[上川綜合振興局|上川綜合振興局]]的轄區。

轄區包含6町：
*空知綜合振興局
**[[南幌町|南幌町]]
**[[奈井江町|奈井江町]]
**[[上砂川町|上砂川町]]
*上川綜合振興局
**[[上富良野町|上富良野町]]
**[[中富良野町|中富良野町]]
**[[南富良野町|南富良野町]]

==沿革==
[[Image:Hokkaido_Sorachi-gun.png|thumb]]
[[Image:Sorachi-shicho.png|thumb]] 
[[Image:Kamikawa-shicho.png|thumb]] 

*1869年8月15日：北海道設置11國86郡，[[石狩國|石狩國]]空知郡成立。
*1897年11月：北海道實施支廳制，隸屬空知支廳之管轄，此時空知郡下轄岩見澤村、沼貝村、市来知村、幌內村、幾春別村、瀧川村、奈江村、歌志內村、栗澤村、幌向村、富良野村。<ref>{{Cite web|url=http://homepage1.nifty.com/ishato/tiri/sityoson/01hokkai/01_hokkai11.htm#sorati |title=市町村的變遷 - 北海道［道央・道南］ |publisher=ISSIE |accessdate=2008年3月8日 |language=ja |deadurl=yes |archiveurl=https://web.archive.org/web/20090930122953/http://homepage1.nifty.com/ishato/tiri/sityoson/01hokkai/01_hokkai11.htm |archivedate=2009年9月30日 }}</ref>（11村）
* 1899年5月27日：音江村從瀧川村分村。（12村）
*1899年6月：空知郡的富良野村（現在的[[上富良野町|上富良野町]]、[[中富良野町|中富良野町]]、[[富良野市|富良野市]]和[[南富良野町|南富良野町]]）改隸屬上川支廳管轄，因此空知郡成為跨支廳的郡。
*1900年6月1日：蘆別村從歌志內村分村。（13村）
*1900年7月1日：北村從岩見澤村分村，同時岩見澤村成為北海道一級村。<ref>{{cite web|url=http://www.city.iwamizawa.hokkaido.jp/i/sisei/iwa/rekisi.htm |title=市的簡介 |publisher=岩見澤市政府 |accessdate=2008年1月29日 |language=ja |deadurl=yes |archiveurl=https://web.archive.org/web/20071102170912/http://www.city.iwamizawa.hokkaido.jp/i/sisei/iwa/rekisi.htm |archivedate=2007年11月2日 }}</ref>（14村）
*1902年4月1日：奈江村成立為北海道二級村。<ref>{{cite web| url=http://www.city.sunagawa.hokkaido.jp/web/PD_Cont.nsf/0/504141A303A6C3FE49256CF3004C5049?OpenDocument| title=年表回顧| publisher=砂川市政府| accessdate=2008年2月18日| language=ja| deadurl=yes| archiveurl=https://web.archive.org/web/20070322235939/http://www.city.sunagawa.hokkaido.jp/Web/PD_Cont.nsf/0/504141A303A6C3FE49256CF3004C5049?OpenDocument| archivedate=2007年3月22日}}</ref>
*1903年7月8日：富良野村分割成上富良野村和下富良野村。（15村）
*1903年8月12日：奈江村改名為砂川村。
*1906年2月1日：岩見澤村改制為岩見澤町（北海道一級町）。（1町14村）
*1906年4月1日：沼貝村<ref>{{cite web |url=http://www.city.bibai.hokkaido.jp/2006/10/532/ |title=美唄的簡史 |publisher=岩見澤市政府 |accessdate=2008年2月4日 |language=ja |deadurl=yes |archiveurl=https://web.archive.org/web/20080514114727/http://www.city.bibai.hokkaido.jp/2006/10/532/ |archivedate=2008年5月14日 }}</ref>、三笠山村（市知來村、幌內村、幾春別村合併而成）<ref>{{cite web|url=http://www.city.mikasa.hokkaido.jp/hotnews/hotnews_view.html?id=56 |title=關於三笠市 |publisher=三笠市政府 |accessdate=2008年2月12日 |language=ja |deadurl=yes |archiveurl=https://web.archive.org/web/20071007192018/http://www.city.mikasa.hokkaido.jp/hotnews/hotnews_view.html?id=56 |archivedate=2007年10月7日 }}</ref>、瀧川村、歌志內村、蘆別村、栗澤村、上富良野村分別成為北海道二級村。（1町12村）
*1907年3月12日：砂川村改為北海道一級村。
*1908年4月1日：南富良野村從下富良野村分村。（1町13村）
*1909年4月1日：（1町14村）
**[[音江村|音江村]]、幌向村分別成為二級村。
**江部乙村從瀧川村分割獨立設置為北海道二級村。
**沼貝村、瀧川村、栗澤村改為北海道一級村。
*1910年11月3日：瀧川村改制為瀧川町（北海道一級町）。（2町13村）
*1915年4月1日：（2町14村）
**江部乙村改為北海道一級村。
**[[山部村|山部村]]從下富良野村分村。<ref>{{cite web |url=http://www.city.furano.hokkaido.jp/contents/ePage.asp?CONTENTNO=493 |title=富良野市的概要 |publisher=富良野市政府 |accessdate=2007年7月26日 |language=ja |deadurl=yes |archiveurl=https://web.archive.org/web/20070807173827/http://www.city.furano.hokkaido.jp/contents/ePage.asp?CONTENTNO=493 |archivedate=2007年8月7日 }}</ref>
**下富良野村成北海道二級村。
*1917年4月1日：中富良野村從上富良野村分割獨立設置，成為北海道二級村。<ref>{{cite web |url=http://hp.town.kamifurano.hokkaido.jp/hp/19shiryo/0010shokai/rekisi.htm |title=上富良野的歷史及主要記事 |publisher=上富良野町公所 |accessdate=2007年9月1日 |language=ja |deadurl=yes |archiveurl=https://web.archive.org/web/20070929005308/http://hp.town.kamifurano.hokkaido.jp/hp/19shiryo/0010shokai/rekisi.htm |archivedate=2007年9月29日 }}</ref>（2町15村）
*1919年4月1日：（3町14村）
**[[北村_(北海道)|北村]]、南富良野村、山部村分別成為北海道二級村。
**上富良野村和歌志內村改為北海道一級村。
**下富良野村改制並改名為富良野町。
*1922年4月1日：（3町15村）
**富良野町成為北海道一級町。
**赤平村從歌志內村分割獨立設置為北海道二級村。<ref>{{cite web |url=http://www.city.akabira.hokkaido.jp/sidemenu/shisei.html |title=市勢概要 |publisher=赤平市政府 |accessdate=2008年2月12日 |language=ja |deadurl=yes |archiveurl=https://www.webcitation.org/67Sf6efdx?url=http://www.city.akabira.hokkaido.jp/sidemenu/shisei.html |archivedate=2012年5月6日 }}</ref>
*1923年4月1日：三笠山村、蘆別村、中富良野村改為北海道一級村。
*1923年6月1日：砂川村改制為砂川町（北海道一級町）。（4町14村）
*1925年6月1日：沼貝村改制為沼貝町（北海道一級町）。（5町13村）
*1926年6月1日：沼貝町改名為美唄町。
*1929年4月1日：赤平村改為北海道一級村。
*1940年4月1日：（6町13村）
**歌志內村改制為歌志內町（北海道一級町）。
**[[東山村|東山村]]從山部村分村。
*1941年4月1日：蘆別村改制為蘆別町（北海道一級町）。<ref>{{cite web|url=http://www.city.ashibetsu.hokkaido.jp/cgi-bin/odb-get.exe?WIT_template=AC02000&Cc=7D23971339C&DM=&Tp=&IM=|title=蘆別市的概要|publisher=蘆別市政府|accessdate=2008年2月5日|language=ja}}{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>（7町12村）
*1942年9月1日：三笠山村改制為町，並改名為三笠町（北海道一級町）。（8町11村）
*1943年2月11日：赤平村改制為赤平町（北海道一級町）。（9町10村）
*1943年4月1日：岩見澤町改制為[[岩見澤市|岩見澤市]]，並脫離空知郡之管轄。（8町10村）
*1943年6月1日：北海道一級二級町村制廢止，音江村、幌向村、北村改為內務省指定村。
*1944年4月1日：奈井江村從砂川町分割獨立設置。<ref>{{cite web| url=http://www.town.naie.hokkaido.jp/mati/gai/ayumi/ayumi.htm| title=奈井江的簡史| publisher=奈井江町公所| accessdate=2008年2月21日| language=ja}}</ref>（8町11村）
*1946年10月5日：指定町村制廢止。
*1949年1月1日：[[上砂川町|上砂川町]]從砂川町和歌志內町分割獨立設置。<ref>{{cite web| url=http://www2.ocn.ne.jp/~kamisuna/info_town/index.htm| title=上砂川町的概要| publisher=上砂川町公所| accessdate=2008年2月21日| language=ja| deadurl=yes| archiveurl=https://web.archive.org/web/20080222012053/http://www2.ocn.ne.jp/~kamisuna/info_town/index.htm| archivedate=2008年2月22日}}</ref>（9町11村）
*1949年4月1日：栗澤村改制為[[栗澤町|栗澤町]]。（10町10村）
*1950年4月1日：美唄町改制為[[美唄市|美唄市]]，並脫離空知郡之管轄。（9町10村）
*1950年9月1日：奈井江村改制為[[奈井江町|奈井江町]]。（10町9村）
*1951年8月1日：上富良野村改制為[[上富良野町|上富良野町]]。（11町8村）
*1952年5月5日：江部乙村改制為[[江部乙町|江部乙町]]。（12町7村）
*1953年4月1日：蘆別町改制為[[蘆別市|蘆別市]]，並脫離空知郡之管轄。（11町7村）
*1954年7月1日：赤平町改制為[[赤平市|赤平市]]，並脫離空知郡之管轄。（10町7村）
*1956年9月30日：東山村併入富良野町。（10町6村）
*1957年4月1日：三笠町改制為[[三笠市|三笠市]]，並脫離空知郡之管轄。（9町6村）
*1958年7月1日：（6町6村）
**瀧川町改制為[[瀧川市|瀧川市]]，並脫離空知郡之管轄。
**砂川町改制為[[砂川市|砂川市]]，並脫離空知郡之管轄。
**歌志內町改制為[[歌志內市|歌志內市]]，並脫離空知郡之管轄。
*1962年5月1日：幌向村改制為町，並改名為[[南幌町|南幌町]]。<ref>{{cite web| url=http://www.town.nanporo.hokkaido.jp/cyousei/machinituite/gaiyou.jsp| title=南幌町的概要| publisher=南幌町公所| accessdate=2008年2月20日| language=ja}}</ref>（7町5村）
*1963年5月1日：音江村與[[雨龍郡|雨龍郡]][[深川町|深川町]]、[[一已村|一已村]]、[[納內村|納內村]]合併為[[深川市|深川市]]，並脫離空知郡之管轄。（7町4村）
*1964年5月1日：中富良野村改制為[[中富良野町|中富良野町]]。（8町3村）
*1966年5月1日：富良野町與山部村合併為[[富良野市|富良野市]]，並脫離空知郡之管轄。（7町2村）
*1967年4月1日：南富良野村改制為[[南富良野町|南富良野町]]。（8町1村）
*1971年4月1日：江部乙町與瀧川市[[市町村合併|合併]]為新設立的瀧川市，並脫離空知郡之管轄。（7町1村）
*2006年3月27日：栗澤町、北村被[[市町村合併|併入]]岩見澤市，並脫離空知郡之管轄。（6町）
