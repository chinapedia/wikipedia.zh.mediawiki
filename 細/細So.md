'''細So'''（{{lang|-en|Danny So Yiu Chung}}，{{bd|1979年|6月30日}}），本名'''蘇耀宗'''，[[香港商業電台|香港商業電台]][[唱片騎師|唱片騎師]]、[[素食主義者|素食主義者]]。

== 簡歷 ==
細So早年就讀[[番禺會所華仁小學|番禺會所華仁小學]]和[[香港華仁書院|香港華仁書院]]<ref name="apple1">[http://hk.apple.nextmedia.com/entertainment/art/20110219/14989796 我是誰 靠森美小儀響朵]，[[蘋果日報_(香港)|蘋果日報]]，2011年2月19日</ref>，畢業於香港理工大學<ref name="apple1"/>。他是2009年[[飢饉三十|飢饉三十]]饑饉之星，其他分別是[[張敬軒|張敬軒]]和[[蔡卓妍|蔡卓妍]]；2014年[[飢饉三十|飢饉三十]]饑饉之星，其他的有[[RubberBand|RubberBand]]<ref>[https://web.archive.org/web/20171106160930/http://studentreporter.mingpao.com/cfm/com_OldContent2.cfm?Category=activitystar&Content_ID=1780&Year=17 第17屆校園記者·非洲體會 RubberBand：開飯不簡單 津國人尊重食物 一口一感恩]，[[明報|明報校園記者網]]，2014年</ref><ref>[http://orientaldaily.on.cc/archive/20090207/ent/ent_a15cnt.html 阿Sa割草餵牛變村姑]，[[東方日報_(香港)|東方日報]]，2009年2月7日</ref>。他亦是商台節目《[[森美移動|森美移動]]》成員之一，當中成員包括[[森美|森美]]、[[詹志文|詹志文]]及[[陳強|陳強]]，另外，他曾擔任單元廣播劇《[[黃金少年|黃金少年]] Season 1, 3, 4》的製作及[[監製|監製]]。除了連續8年參與[[森美小儀歌劇團|森美小儀歌劇團]]外，他也有參與其他舞台劇。

== 感情生活 ==
2006年8月6日與[[商業電台|商業電台]][[叱吒903|叱吒903]] DJ同事[[張詠妍|張詠妍]]（阿Wing）結婚。2009年年初，因性格不合，兩人低調協議離婚，女方翌年嫁給商台高層[[林若寧|林若寧]]。2011年11月1日，細so與旅遊記者[[陳穎楠|陳穎楠]]結婚。2013年9月7日，其妻於[[聖保祿醫院|聖保祿醫院]]誕下8磅重兒子蘇浪仁。2016年7月9日則再剖腹誕下次子蘇㟍仁<ref>[http://www.sundaykiss.com/magazine/sunday-family/dj%e7%b4%b0so-%e6%90%ac%e5%8e%bb%e9%9b%a2%e5%b3%b6%e9%81%bf%e5%82%b3%e7%b5%b1%e6%95%99%e8%82%b2/ DJ細So 搬去離島避 傳統教育]，[[Sundaykiss|Sundaykiss]]，2016年11月15日</ref>。
