'''港濤軒'''（{{lang-en|'''Island Lodge'''}}），為現時[[北角|北角]]最新落成的單幢住宅，物業樓高共44層，由[[中華巴士|中華巴士]]以「Forever Vitality Limited」為名出資發展，並委託[[太古地產|太古地產]]旗下的「Charming Grace Limited」負責項目設計及銷售代理。現由仲量聯行物業管理有限公司提供管理服務。

== 歷史 ==
港濤軒前身為於1956年落成的中巴員工宿舍，地盤面積17,868方呎，於1998年8月中巴喪失專營權後空置，並於2004年拆卸。其後於2007年7月19日，太古宣佈項目被命名為「港濤軒」<ref>[http://www.islandlodge.com.hk/PressReleases/news_0719.aspx?lang=zh-HK 港島東全新優質住宅項目正式命名為「港濤軒」] {{webarchive|url=https://web.archive.org/web/20090816181750/http://www.islandlodge.com.hk/PressReleases/news_0719.aspx?lang=zh-HK |date=2009-08-16 }}</ref>。

物業於2007年11月開售，當時示範單位設於[[太古城|太古城]]中心四座地下<ref>[http://www.islandlodge.com.hk/PressReleases/news_1031.aspx?lang=zh-HK 港濤軒示範單位矚目登場，展現簡約設計特色] {{webarchive|url=https://web.archive.org/web/20090816181717/http://www.islandlodge.com.hk/PressReleases/news_1031.aspx?lang=zh-HK |date=2009-08-16 }}</ref>，開售時的呎價已經超過一萬元。項目於2008年10月落成，並在2009年6月開始入伙。

== 簡介 ==
物業位於[[北角|北角]][[渣華道|渣華道]]海傍，建築物對面為前[[北角邨|北角邨]]空地，故此景觀可謂全無遮擋，即使低層單位也可以享有無敵海景。物業樓高44層，但則有49樓（不設4、14、24、34及44樓，26樓為[[隔火層|避火層]]），物業設有一個有50個車位的私人停車場（位於一樓及二樓）和一個[[會所|會所]] (位於三樓及五樓)，設施包括室外泳池、健身室、燒烤場、家庭活動室、室外兒童遊樂場等，[[平台花園|平台花園]]設於三樓。6-47樓為標準住宅單位，一梯五伙設計，提供817-834平方呎的兩房單位、1039-1120平方呎的三房單位，及1306-1327平方呎的四房單位<ref>{{cite web |url=http://www.islandlodge.com.hk/Location/location_factsheet.aspx?lang=zh-HK |title=存档副本 |accessdate=2011-05-30 |deadurl=yes |archiveurl=https://web.archive.org/web/20100106080844/http://www.islandlodge.com.hk/Location/location_factsheet.aspx?lang=zh-HK |archivedate=2010-01-06 }}</ref><ref>{{cite web |url=http://www.islandlodge.com.hk/FloorPlan/Typical_index.aspx?lang=zh-HK |title=存档副本 |accessdate=2011-05-30 |deadurl=yes |archiveurl=https://web.archive.org/web/20100605065523/http://www.islandlodge.com.hk/FloorPlan/Typical_index.aspx?lang=zh-HK |archivedate=2010-06-05 }}</ref>；而48-49樓則為1943-2265平方呎的複式單位<ref>{{cite web |url=http://www.islandlodge.com.hk/FloorPlan/Special_index.aspx?lang=zh-HK |title=存档副本 |accessdate=2011-05-30 |deadurl=yes |archiveurl=https://web.archive.org/web/20100605065610/http://www.islandlodge.com.hk/FloorPlan/Special_index.aspx?lang=zh-HK |archivedate=2010-06-05 }}</ref>。
