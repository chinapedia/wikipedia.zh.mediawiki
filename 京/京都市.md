'''京都市'''（{{jpn|j=京都市|hg=きょうとし|rm=Kyōto shi|f=y}}<small>{{Audio|Ja-Kyoto.ogg|日語發音}}</small>）是位於[[日本|日本]][[京都府|京都府]]南部的城市，為京都府[[日本都道府縣廳所在地|府治]]，也是[[政令指定都市|政令指定都市]]之一<ref>{{cite web |url=http://www.soumu.go.jp/main_sosiki/jichi_gyousei/bunken/shitei_toshi-ichiran.html |script-title=ja:指定都市一覧 |accessdate=2013-11-24|publisher={{lang|ja|総務省統計局}}|language=ja}}</ref>。京都市的面積為827.83平方公里，下轄11個區。2017年10月，京都市有人口1,472,027人<ref name="Population">{{cite web|url=http://www2.city.kyoto.lg.jp/sogo/toukei/Population/Data/Estimate/Ward/26100.xlsx|script-title=ja:京都市の人口|accessdate=2017-10-17|publisher={{lang|ja|京都市情報館}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20171017202136/http://www2.city.kyoto.lg.jp/sogo/toukei/Population/Data/Estimate/Ward/26100.xlsx|archivedate=2017-10-17}}</ref>，是[[日本城市人口排名|日本人口第八多的都市]]。京都市還和京都府南部、[[滋贺县|滋贺县]]西部及[[大阪府|大阪府]]的部份地區共同組成了“[[京都都市圈|京都都市圈]]”，其人口數約有379萬<ref>{{cite web|url=http://www.kyoto-toshiken.jp/nhomepage/nwhat-toshiken.htm|script-title=ja:京都都市圏とは|accessdate=2013-11-19|publisher=京都都市圏自治体ネットワーク|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20131203083851/http://www.kyoto-toshiken.jp/nhomepage/nwhat-toshiken.htm|archivedate=2013-12-03}}</ref>。京都市也是「[[京阪神|京阪神都會區]]」的一部份<ref>{{cite web|url=http://www.stat.go.jp/data/kokusei/2005/nihon/pdf/r07-01da.pdf|script-title=ja:参考７　大都市圏・都市圏を構成する市区町村及び距離帯|accessdate=2013-11-19|publisher={{lang|ja|総務省統計局}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20130206063734/http://www.stat.go.jp/data/kokusei/2005/nihon/pdf/r07-01da.pdf|archivedate=2013-02-06}}</ref>。

自794年[[桓武天皇|桓武天皇]][[迁都东京|遷都]][[平安京|平安京]]到1868年[[明治天皇|明治天皇]][[迁都东京|東京奠都]]為止，京都一直都是日本的[[日本首都|首都]]<ref>{{cite web |url=http://kyotoshugakuryoko.jp/study/history01.html |script-title=ja:京都修学旅行ナビ　京都の歴史 |accessdate=2013-11-19 |publisher={{lang|ja|京都市観光推進協議会}} |archiveurl=https://web.archive.org/web/20131203185104/http://kyotoshugakuryoko.jp/study/history01.html |archivedate=2013-12-03 |deadurl=yes |language=ja }}</ref>。長年的歷史積澱使得京都市擁有相當豐富的歷史遺跡，也是日本傳統文化的重鎮之一。京都的一些傳統民俗[[祭_(日本)|祭典]]，諸如[[葵祭|葵祭]]<ref>{{cite web |url=http://www.kamigamojinja.jp/kamo/index.html |script-title=ja:賀茂祭（葵祭） |accessdate=2013-11-19|publisher={{lang|ja|上賀茂神社}}|archiveurl=https://web.archive.org/web/20131203043916/http://www.kamigamojinja.jp/kamo/index.html |archivedate=2013-12-03 |deadurl=no|language=ja}}</ref>、[[祇園祭|祇園祭]]<ref>{{cite web |url=http://www.gionmatsuri.or.jp/about/ |script-title=ja:祇園祭山鉾連合会について|archiveurl=https://web.archive.org/web/20131203002143/http://www.gionmatsuri.or.jp/about/ |archivedate=2013-12-03 |deadurl=no|accessdate=2013-11-19|publisher={{lang|ja|公益財団法人祇園祭山鉾連合会}}|language=ja}}</ref>等已舉辦超過千年，成為京都獨有的風景。京都市的部份歷史建築在1994年以「[[古京都遗址|古都京都的文化財]]」的名義被列为[[世界遗产|世界文化遺產]]<ref name="UNESCO">{{cite web |url=http://whc.unesco.org/pg.cfm?CID=31&ID_SITE=688&l=EN |title=Historic Monuments of Ancient Kyoto (Kyoto, Uji and Otsu Cities) |accessdate=2013-11-19|publisher=UNESCO|archiveurl=https://web.archive.org/web/20131203043804/http://whc.unesco.org/pg.cfm?CID=31&ID_SITE=688&l=EN |archivedate=2013-12-03 |deadurl=no}}{{en}}</ref>。這些因素使得京都吸引了眾多觀光客，2008年造訪京都的日本國內外遊客數便已突破5000萬人<ref>{{cite web|url=http://www.city.kyoto.lg.jp/sogo/cmsfiles/contents/0000062/62177/20090512-1.pdf|script-title=ja:入洛観光客数５０００万人達成について|accessdate=2013-11-21|publisher={{lang|ja|京都市}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20130712005803/http://www.city.kyoto.lg.jp/sogo/cmsfiles/contents/0000062/62177/20090512-1.pdf|archivedate=2013-07-12}}</ref>。京都也是日本重要的工業城市，其中又以傳統產業<ref>{{cite web |url=http://www.city.kyoto.lg.jp/sankan/page/0000041366.html |script-title=ja:京都市の伝統産業 |accessdate=2013-11-24|publisher=京都市|archiveurl=https://web.archive.org/web/20131203013854/http://www.city.kyoto.lg.jp/sankan/page/0000041366.html |archivedate=2013-12-03 |deadurl=no|language=ja}}</ref>和電子產業最為重要。京都是日本最重要的文化教育城市之一，市內共有37所大學，每百萬人平均擁有大學數量和學生佔人口比例都在日本大都市中居首<ref name="Student Population">{{cite web|url=http://www.city.kyoto.jp/sogo/toukei/Publish/Monthly/Topics/201008-01.pdf|script-title=ja:大学のまち・学生のまち 京都市 －国勢調査でわかること（その４）－|accessdate=2013-11-19|publisher={{lang|ja|京都市}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20131203004122/http://www.city.kyoto.jp/sogo/toukei/Publish/Monthly/Topics/201008-01.pdf|archivedate=2013-12-03}}</ref>。

== 地理 ==
[[File:Kamogawa_sakura.jpg|thumb]]時節的[[鴨川_(淀川水系)|鴨川]]]]
京都市的市區大部份地區都位於[[京都盆地|京都盆地]]之內，這也造就了京都市除南方之外三面環山的環境。京都盆地被急傾斜的山地所包圍，是由[[斷層|斷層]]活動所形成的構造性內陸盆地<ref>{{cite web |url=http://jimuhp.kyokyo-u.ac.jp/wonderful/enviro/kyouto.html |script-title=ja:京都盆地の成り立ちを眺める |accessdate=2013-11-20|publisher={{lang|ja|京都教育大学}}|archiveurl=https://web.archive.org/web/20131203013532/http://jimuhp.kyokyo-u.ac.jp/wonderful/enviro/kyouto.html |archivedate=2013-12-03 |deadurl=no|language=ja}}</ref>，其地下的地層現在仍然活動。也因此京都市內有花折斷層等活斷層存在<ref>{{cite web|url=http://topo.earth.chiba-u.jp/afr/backnumber/No16/16%E5%8F%B707%E6%9D%B1%E9%83%B7%E3%81%BB%E3%81%8B.pdf|script-title=ja:花折断層の最新活動について|accessdate=2013-11-22|publisher=活断層研究|language=ja|deadurl=yes|archiveurl=https://web.archive.org/web/20131203063700/http://topo.earth.chiba-u.jp/afr/backnumber/No16/16%E5%8F%B707%E6%9D%B1%E9%83%B7%E3%81%BB%E3%81%8B.pdf|archivedate=2013-12-03}}</ref>。盆地內的地勢則呈現北高南低，東高西低的特徵。京都盆地東部也因其地勢較高且較為乾燥適宜居住，成為京都市內最早開發的地區<ref name="Kyotochizumonogatari">{{cite book |author={{lang|ja|植村善博・上野裕}}|script-title=ja:《京都地図物語》 |year=1999 |publisher={{lang|ja|古今書院}} |location=東京都 |isbn=4-7722-1685-5|language=ja}}</ref>{{rp|105}}。京都市東南部的山科區和伏見區的醍醐地区則位於{{link-ja|山科盆地|山科盆地}}，其和京都盆地之間由[[東山_(京都府)|清水山]]、[[東山_(京都府)|稻荷山]]等山峰隔開。京都市的北部地區則多為山地，其中較為知名的有[[如意嶽|大文字山]]、[[比叡山|比叡山]]、[[嵐山_(京都市)|嵐山]]等。京都市內的最高點是{{link-ja|皆子山|皆子山}}，海拔971米<ref>{{cite web|url=http://kanko.city.kyoto.lg.jp/detail.php?InforKindCode=4&ManageCode=7000061|script-title=ja:皆子山|accessdate=2017-10-27|publisher={{lang|ja|京都観光ナビ}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20171021163844/http://kanko.city.kyoto.lg.jp/detail.php?InforKindCode=4&ManageCode=7000061|archivedate=2017-10-21}}</ref>。

京都市內主要的河川有[[桂川_(淀川水系)|桂川]]及[[鴨川_(淀川水系)|鴨川]]等。鴨川<ref>{{cite web |url=http://www.city.kyoto.jp/somu/rekishi/fm/nenpyou/htmlsheet/toshi08.html |script-title=ja:鴨川 |accessdate=2013-11-20|publisher={{lang|ja|京都市}}|archiveurl=https://web.archive.org/web/20131203005210/http://www.city.kyoto.jp/somu/rekishi/fm/nenpyou/htmlsheet/toshi08.html |archivedate=2013-12-03 |deadurl=no|language=ja}}</ref>位於京都市的東部，河道兩岸廣植[[樱花|樱花]]等樹木。每逢夏季常有眾多市民和遊客在鴨川的河床灘上納涼<ref>{{cite web |url=http://www.kyoto-yuka.com/history/index.html |script-title=ja:納涼床の歴史 |accessdate=2013-11-20 |publisher={{lang|ja|京都鴨川納涼床協同組合}} |archiveurl=https://web.archive.org/web/20131203004936/http://www.kyoto-yuka.com/history/index.html |archivedate=2013-12-03 |deadurl=yes |language=ja }}</ref>，成為京都獨特的風景線。而桂川也流經[[嵐山_(京都市)|嵐山]]等著名景區。兩條河流在南區及伏見區交界處合流，注入淀川水系。和日本其他河流一樣，京都市內的河流大多短小湍急，容易發生洪水。在平安京時期，京都就多次遇洪水侵襲<ref>{{cite web|url=http://www.himoji.jp/jp/publication/pdf/symposium/No05/D003-010.pdf|script-title=ja:平安京の地形環境と災害|accessdate=2013-11-20|publisher={{lang|ja|河角龍典}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20131203004726/http://www.himoji.jp/jp/publication/pdf/symposium/No05/D003-010.pdf|archivedate=2013-12-03}}</ref>。即使在現代，每逢[[颱風|颱風]]過境造成集中豪雨時，京都市內河川的水位也常會在短時間內大幅上升造成洪水<ref>{{cite web|url=http://www.city.kyoto.lg.jp/gyozai/cmsfiles/contents/0000156/156855/091611.pdf|script-title=ja:平成２５年９月１５日の台風１８号に伴う被害状況等について|accessdate=2013-11-20|publisher={{lang|ja|京都市}}|language=ja|deadurl=yes|archiveurl=https://web.archive.org/web/20131203013709/http://www.city.kyoto.lg.jp/gyozai/cmsfiles/contents/0000156/156855/091611.pdf|archivedate=2013-12-03}}</ref>。
=== 市內地名 ===
京都在其開發早期是一座[[计划城市|计划城市]]，市內道路類似[[棋盤|棋盤]]，極為規整，且許多平安京時期的路名沿用至今<ref name="Kinki">{{cite book |author={{lang|ja|平岡昭利・野間晴雄}} |script-title=ja:《地図で読む百年　近畿Ⅰ》 |year=2006 |publisher={{lang|ja|古今書院}} |location=東京都 |isbn=978-4772230537|language=ja}}</ref>{{rp|1}}。也正因如此，和日本其他都市常用町名（類似中文世界的「里」）來表示地址相比，京都多使用道路名稱來表示地址{{efn|例如京都市政府因其位於寺町通和御池通的路口且位於寺町通北側，其註冊地址就是「{{lang|ja|京都市中京区寺町通御池上る上本能寺前町488番地}}」<ref>{{cite web|url=http://www.city.kyoto.lg.jp/sogo/page/0000013364.html|script-title=ja:市役所へのアクセス|accessdate=2013-11-24|publisher={{lang|ja|京都市}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20140414223958/http://www.city.kyoto.lg.jp/sogo/page/0000013364.html|archivedate=2014-04-14}}</ref>。}}<ref>{{cite book |author={{lang|ja|清水さとし}}|script-title=ja:《京都通になる100の雑学》 |year=2013 |publisher={{lang|ja|実業之日本社}} |location=東京都 |isbn=978-4-408-00844-8|language=ja}}</ref>{{rp|10-11}}。而京都市內的路口也和日本其他地區大多採用附近地名不同，多合成兩條道路的名稱作為路口名，如「[[四條河原町|四條河原町]]」這一地名就是得來於因其位於四條通和河原町通兩條街道的路口<ref>{{cite web |url=http://kanko.city.kyoto.lg.jp/detail.php?InforKindCode=4&ManageCode=6000050 |script-title=ja:四条河原町 |accessdate=2013-11-21|publisher={{lang|ja|京都観光ナビ}}|archiveurl=https://web.archive.org/web/20131203021131/http://kanko.city.kyoto.lg.jp/detail.php?InforKindCode=4&ManageCode=6000050 |archivedate=2013-12-03 |deadurl=no|language=ja}}</ref>。值得一提的是，京都也是日本難讀地名較多的市之一，著名的例子有天使突拔（{{lang|ja|てんしつきぬけ}}）、先斗町（{{lang|ja|ぽんとちょう}}）<ref>{{cite book |author={{lang|ja|森谷剋久}}|script-title=ja:《京都「地理・地名・地図」の謎》 |year=2013 |publisher={{lang|ja|実業之日本社}} |location=東京都 |isbn=978-4-408-45296-8|language=ja}}</ref>{{rp|140}}等。

=== 氣候 ===
京都屬於[[柯本氣候分類法|柯本氣候分類法]]下的[[副熱帶濕潤氣候|副熱帶濕潤氣候]]（在日本則被歸類為同時屬於[[瀨戶內海式氣候|瀨戶內海式氣候]]及{{link-ja|內陸性氣候|内陸性気候}}<ref>{{cite web|url=http://www.jma-net.go.jp/kyoto/kyoto6.html|script-title=ja:京都府の気象特性|accessdate=2013-11-20|publisher={{lang|ja|京都地方気象台}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20131014081222/http://www.jma-net.go.jp/kyoto/kyoto6.html|archivedate=2013-10-14}}</ref>），其特徵為四季的氣溫及降雨量變化相當顯著。而京都位於盆地這一點又讓這一特徵體現得更為明顯<ref name="Chishi">{{cite book |author={{lang|ja|石川義孝・金田章裕}}|script-title=ja:《日本の地誌8　近畿圏》 |year=2006 |publisher={{lang|ja|朝倉書店}} |location=東京都 |isbn=4-254-16768-7|language=ja}}</ref>{{rp|312}}。京都夏天炎熱且潮溼，相對地，冬天則較為寒冷且偶爾會下雪。京都的春季開始於三月中旬，這一時期櫻花盛開，和秋季並為也是京都觀光客最多的時期。從六月中旬左右開始至七月底，京都進入悶熱潮濕的[[梅雨|梅雨]]季節。然後便進入炎熱且晴朗的夏天後半。如同大部份日本[[太平洋|太平洋]]沿岸及中部地區，京都在九月及十月易遭到颱風侵襲。之後京都進入短暫的秋季。雖然相比[[日本海|日本海]]沿岸地區，京都市的降雪相對較少。然而地處內陸的京都市仍然是近畿地方冬季較為寒冷的地區<ref>{{cite web|url=http://www.city.kyoto.lg.jp/kankyo/cmsfiles/contents/0000000/197/yoko_chap2_2.pdf|script-title=ja:気象|accessdate=2017-11-02|publisher={{lang|ja|京都市}}|language=ja|deadurl=yes|archiveurl=https://web.archive.org/web/20171107011900/http://www.city.kyoto.lg.jp/kankyo/cmsfiles/contents/0000000/197/yoko_chap2_2.pdf|archivedate=2017-11-07}}</ref>。

== 歷史 ==
=== 平安京 ===
[[File:HeiankyouMapJapanese.svg|thumb]]
京都盆地自[[旧石器時代|旧石器時代]]開始就已經有人類活動<ref name="Kyoto Chronology">{{cite web |url=http://www.city.kyoto.jp/somu/rekishi/fm/nenpyou/toshi_nenpyo.html |script-title=ja:京都の歴史年表 |accessdate=2013-11-20|publisher=京都市|archiveurl=https://web.archive.org/web/20131203004136/http://www.city.kyoto.jp/somu/rekishi/fm/nenpyou/toshi_nenpyo.html |archivedate=2013-12-03 |deadurl=no|language=ja}}</ref>。在[[繩文時代|繩文時代]]，位於現今京都市東北部的{{link-ja|北白川|北白川}}地區和山科盆地的丘陵地區已經有集落出現。到了[[彌生時代|彌生時代]]，平原地區也開始有集落出現<ref name="Kyoto Chronology"/>。進入5世紀後，[[渡來人|渡來人]]貴族[[秦姓|秦氏]]開始在京都居住<ref>{{cite web|url=http://www.city.kyoto.jp/somu/rekishi/fm/nenpyou/htmlsheet/toshi01.html|script-title=ja:秦氏|accessdate=2013-11-20|publisher={{lang|ja|京都市}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20130306124228/http://www.city.kyoto.jp/somu/rekishi/fm/nenpyou/htmlsheet/toshi01.html|archivedate=2013-03-06}}</ref>，其在京都盆地興修水利，此為京都地區大規模開發之始。

[[奈良時代|奈良時代]]末期，[[桓武天皇|桓武天皇]]為了擺脫奈良[[佛教|佛教]]寺院勢力的影響，於784年將首都從[[平城京|平城京]]（今[[奈良市|奈良市]]）遷移到[[長岡京|長岡京]]（今[[長岡京市|長岡京市]]）。然而僅過了不到10年，桓武天皇在[[和氣清麻呂|和氣清麻呂]]的建議下<ref name="Hall">{{cite book|title=The Cambridge History of Japan|last=Hall|first=John Whitney|pages=|publisher=Cambridge University Press|date=1988|id=ISBN 978-0-521-22357-7}}{{en}}</ref>{{rp|516–517}}，於794年將首都又遷到位於[[山城國|山城國]]的[[平安京|平安京]]（也就是現在的京都市），自此開啟[[平安时代|平安时代]]，也開始京都作為日本首都的歷史。平安京的選址參考[[風水|風水]]思想<ref>{{cite web |url=http://www.kyotofusui.jp/02_heiankyokento.html |script-title=ja:平安京建都と風水 |accessdate=2013-11-20 |publisher={{lang|ja|京都風水}} |archiveurl=https://web.archive.org/web/20131203064718/http://www.kyotofusui.jp/02_heiankyokento.html |archivedate=2013-12-03 |deadurl=yes |language=ja }}</ref>，並且效仿中國的[[长安|长安]]和[[洛阳市|洛陽]]，是一座傳統的[[里坊制|市坊制]]都市<ref name="Jyohosei">{{cite web|url=http://www.city.kyoto.jp/somu/rekishi/fm/nenpyou/htmlsheet/toshi03.html|script-title=ja:条坊制|accessdate=2013-11-20|publisher={{lang|ja|京都市}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20131129100553/http://www.city.kyoto.jp/somu/rekishi/fm/nenpyou/htmlsheet/toshi03.html|archivedate=2013-11-29}}</ref>。平安京南北長約5.2公里，東西長約4.5公里<ref name="Jyohosei"/>。天皇居住的[[大內裏|大內裏]]位於平安京的正北<ref name="Daidairi">{{cite web|url=http://www.city.kyoto.jp/somu/rekishi/fm/nenpyou/htmlsheet/bunka07.html|script-title=ja:内裏から京都御所へ|accessdate=2013-11-20|publisher=京都市|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20130313205924/http://www.city.kyoto.jp/somu/rekishi/fm/nenpyou/htmlsheet/bunka07.html|archivedate=2013-03-13}}</ref>，以{{link-ja|朱雀大路|朱雀大路}}為中心，城市分為{{link-ja|右京|右京}}（又稱「長安」）和{{link-ja|左京|左京}}（又稱「洛陽」）兩個對稱的部份<ref name="Kamigyo">{{cite web |url=http://www.city.kyoto.lg.jp/kamigyo/page/0000012280.html |script-title=ja:上京区の成立 |accessdate=2013-11-20|publisher={{lang|ja|上京区}}|archiveurl=https://web.archive.org/web/20131203013706/http://www.city.kyoto.lg.jp/kamigyo/page/0000012280.html |archivedate=2013-12-03 |deadurl=no|language=ja}}</ref>。平安京堪稱是一座規模宏大且規劃嚴謹的東亞傳統型的計劃城市。

然而平安京實際的開發情況和計劃相比可謂是大相徑庭。平安京的市區範圍雖然十分廣大，其人口卻只有大約15萬人，因此城內並未得到充分開發，並且有大片農田<ref name="Virtual Kyoto">{{cite book |author={{lang|ja|矢野桂司・中谷友樹・磯田弦}}|script-title=ja:《バーチャル京都》 |year=2007 |publisher={{lang|ja|ナカニシヤ出版}} |location=京都 |isbn=978-4-7795-0100-5|language=ja}}</ref>{{rp|99－101}}。甚至大內裏也在經過多次火災之後不再重建<ref name="Daidairi"/>。特別是右京地區因為地形地濕，開發進度十分緩慢。左京因此成為京都主要發展的地區，「洛陽」也成為京都的代稱<ref name="Kamigyo"/>。在平安時代中期以後，貴族的住宅大多集中在左京北部，並且開始跨過鴨川向平安京以外的範圍延伸。在11世紀後期，[[太上皇|上皇]]（退位的[[天皇|天皇]]）在{{link-ja|白河 (洛外)|白河 (洛外)|白河}}、{{link-ja|鳥羽|鳥羽 (洛外)}}等地修建離宮並掌控朝政，該時期遂稱[[院政|院政]]時期<ref name="Kyotonorekishi">{{cite book |author={{lang|ja|脇田修・脇田晴子}} |script-title=ja:《物語　京都の歴史》 |year=2008 |publisher={{lang|ja|中央公論新社}} |location=東京都 |isbn=978-4-12-101928-8|language=ja}}</ref>{{rp|63}}。而離宮的修建也使得京都市區向平安京以外地區發展的趨勢更加明顯<ref name="Kyoto History Map">{{cite book |author={{lang|ja|両洋歷史研究会}}|script-title=ja:《京都歷史地図》 |year=2012 |publisher={{lang|ja|メイツ出版}} |location=東京都 |isbn=978-4-7804-1191-1|language=ja}}</ref>{{rp|102}}。

=== 傳統文化的發達 ===
[[File:Ginkakuji_Kyoto03-r.jpg|thumb]]
[[File:Kyoto_maiko_henshin_from_Flickr.jpg|200px]]]]
平安時代的基本政治制度—[[日本律令制|律令制]]在進入10世紀後變得有名無實，這也使得[[貴族|貴族]]階級在平安時代後期開始沒落。相反，[[武士_(日本)|武士]]階級在這一時期逐漸興起。12世紀中期，日本歷史上首個[[武家政權|武家政權]][[平氏政權|平氏政權]]誕生<ref name="Kyoto History Map" />{{rp|120}}。然而平氏政權僅維持二十多年就被另一大武士集團—[[源氏|源氏]]所取代。1192年，[[源賴朝|源賴朝]]在[[鎌倉|鎌倉]]創建[[幕府|幕府]]，日本進入[[鎌倉時代|鎌倉時代]]。在[[鎌倉幕府|鎌倉幕府]]時期，日本實質上的政治中心雖然移至鎌倉，京都仍然是最大都市和天皇的居住地。

1338年，隨著[[室町幕府|室町幕府]]的成立，日本進入[[室町时代|室町时代]]，京都也再次成為日本政治的中心<ref name="Kyoto Chronology" />。在這一時期，京都的都市空間發生巨大的變化。市區按南北分為[[上京區|上京]]和[[下京區|下京]]兩個部份，東西方向的二條大路取代朱雀大路成為京都最重要的街道<ref name="Chishi" />{{rp|325}}。上京集中包括天皇的居住處{{link-ja|里內裏|里内裏}}等眾多貴族的宅邸，而下京則是京都主要的工商業區域<ref name="Kyoto History Map" />{{rp|71}}。室町時代也是[[北山文化_(日本)|北山文化]]和[[東山文化_(日本)|東山文化]]兩大傳統文化的勃興時期，京都的寺院數在這一時期大幅增加，其中最為著名的就是[[鹿苑寺|金閣寺]]和[[慈照寺|銀閣寺]]<ref name="Kyoto History Map" />{{rp|78-83}}。

在1467年至1477年之間，京都是[[应仁之乱|应仁之乱]]的主戰場，市區幾乎完全被毀<ref>{{cite web|url=http://www.city.kyoto.jp/somu/rekishi/fm/nenpyou/htmlsheet/toshi14.html|script-title=ja:応仁・文明の乱|accessdate=2013-11-20|publisher={{lang|ja|京都市}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20150923222153/http://www.city.kyoto.jp/somu/rekishi/fm/nenpyou/htmlsheet/toshi14.html|archivedate=2015-09-23}}</ref>。之後京都在市民的努力下開始逐漸復興。貴族勢力的上京和市民勢力的下京<ref>{{cite web|url=http://www.city.kyoto.jp/somu/rekishi/fm/nenpyou/htmlsheet/toshi13.html|script-title=ja:上京と下京|accessdate=2013-11-20|publisher={{lang|ja|京都市}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20131203004755/http://www.city.kyoto.jp/somu/rekishi/fm/nenpyou/htmlsheet/toshi13.html|archivedate=2013-12-03}}</ref>區隔更加明顯，兩者均由土壘包圍且中間只有室町小路一條道路連接<ref name="Kyoto History Map" />{{rp|46}}。隨著產業的發達，由工商業者組成的{{link-ja|町眾|町衆}}階級日益興起，他們在都市運營方面發揮重要的作用，也使得此時的京都可說是自由度較高的自治城市<ref name="Kyotonorekishi" />{{rp|172-177}}。京都因其政治中心的地位，在這一時期也是多個重要歷史事件的舞台。1568年，[[織田信長|織田信長]]的軍隊進入京都。在五年後信長又燒毀反抗勢力強大的上京，其勢力之強盛到達頂點。然而在1582年，在京都爆發[[本能寺之变|本能寺之变]]<ref name="Kyoto History Map" />{{rp|56-57}}，信長自殺<ref>{{lang|ja|{{Citation |last1=太田|first1=牛一|last2=中川|first2=太古|year=2013|author1-link=太田牛一|series=新人物文庫|publisher=中経出版|title=現代語訳 信長公記|volume=卷十五}}}}</ref>。

=== 都市空間的變革 ===
[[File:Nijo-jo_Ninomaru-goten_2009.jpg|thumb]]
織田信長之後，實現統一日本大業的是[[丰臣秀吉|丰臣秀吉]]。秀吉對京都進行大規模的改造，其主要內容包括修建[[聚樂第|聚樂第]]，將其作為秀吉統治京都的據點。並且在聚樂第附近修建武家屋敷供[[大名_(稱謂)|大名]]居住<ref name="Odoi">{{cite web|url=http://www.city.kyoto.jp/somu/rekishi/fm/nenpyou/htmlsheet/toshi18.html|script-title=ja:聚楽第と御土居|accessdate=2013-11-20|publisher=京都市|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20150519021619/http://www.city.kyoto.jp/somu/rekishi/fm/nenpyou/htmlsheet/toshi18.html|archivedate=2015-05-19}}</ref>；修建{{link-ja|內裏|内裏}}，並強制貴族居住在其附近；將京都市內的寺廟集中搬遷至鴨川西岸的[[寺町通|寺町]]<ref name="Chishi" />{{rp|327}}，並且將在日本全國都有強大勢力的本願寺自大阪遷至京都，加強對佛教勢力的控制<ref name="Kyotochizumonogatari" />{{rp|31}}；實施{{link-ja|天正地割|天正の地割}}，在京都市中心修建南北向的道路<ref name="Kinki" />{{rp|1}}，使得京都市內的街區由正方形變為長方形，改變平安京以來京都的都市格局<ref>{{cite web|url=http://www.city.kyoto.lg.jp/kamigyo/page/0000012443.html|script-title=ja:秀吉の都市改造|accessdate=2013-11-20|publisher={{lang|ja|上京区}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20131203012431/http://www.city.kyoto.lg.jp/kamigyo/page/0000012443.html|archivedate=2013-12-03}}</ref>；建設{{link-ja|御土居|御土居}}，以此作為洛中和洛外的分界線<ref name="Odoi"/>。秀吉的大規模改造使得京都由兩個分裂的部份重新統合為一個整體的城市，城市空間的格局也從中世都市轉變為近世城下町<ref name="Chishi" />{{rp|327}}。

進入17世紀後，[[德川家康|德川家康]]取代豐臣家，成為日本的統治者。德川家康將幕府設在[[江戶|江戶]]，但京都仍然是日本的國都所在地。德川家康雖然新建[[二條城|二條城]]<ref>{{cite web|url=http://www.city.kyoto.jp/somu/rekishi/fm/nenpyou/htmlsheet/toshi21.html|script-title=ja:二条城|accessdate=2013-11-20|publisher={{lang|ja|京都市}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20150923222159/http://www.city.kyoto.jp/somu/rekishi/fm/nenpyou/htmlsheet/toshi21.html|archivedate=2015-09-23}}</ref>並以其作為京都新的都市核心，然而京都整體的都市空間仍然維持豐臣秀吉時期的格局<ref name="Chishi" />{{rp|328}}。1788年，京都市內發生[[天明大火|天明大火]]<ref>{{cite web|url=http://www.city.kyoto.jp/somu/rekishi/fm/nenpyou/htmlsheet/toshi24.html|script-title=ja:天明の大火|accessdate=2013-11-21|publisher={{lang|ja|京都市}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20131203004758/http://www.city.kyoto.jp/somu/rekishi/fm/nenpyou/htmlsheet/toshi24.html|archivedate=2013-12-03}}</ref>，是為京都史上最大的一次火災，市區幾乎完全被毀。然而之後京都很快實現復興。位於眾多街道交匯點的京都是當時日本全國的商業物流中心之一，眾多豪商巨賈聚集於此<ref name="Kyotonorekishi" />{{rp|262}}。京都也是當時日本最重要的宗教文化都市，京都的文化人在[[神道|神道]]、[[佛教|佛教]]、[[儒學|儒學]]、[[蘭學|蘭學]]等學術領域建樹頗豐<ref name="Kyotonorekishi" />{{rp|268-269}}。江戶時代的京都和[[大阪市|大坂]]、[[江戶|江戶]]並稱為「{{link-ja|三都 (日本)|三都|三都}}」，是代表日本的大都市<ref name="Chishi" />{{rp|328}}。

=== 近現代的京都 ===
[[File:Heian_jingu_gate.jpg|thumb]]
幕末時期，京都再次成為歷史舞台的焦點。1864年，幕府軍隊和倒幕勢力在京都市內發生激烈戰鬥，是為[[禁門之變|禁門之變]]<ref>{{cite web|url=http://www.city.kyoto.jp/somu/rekishi/fm/nenpyou/htmlsheet/toshi25.html|script-title=ja:蛤御門の変とどんどん焼け|accessdate=2013-11-21|publisher={{lang|ja|京都市}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20130508090318/http://www.city.kyoto.jp/somu/rekishi/fm/nenpyou/htmlsheet/toshi25.html|archivedate=2013-05-08}}</ref>，京都市內大片街區被焚毀。這也使得京都市中心現存的建築多為幕末之後修建的建築物。1867年，京都郊外發生[[鸟羽伏见之战|鸟羽伏见之战]]，倒幕軍取得勝利，是決定戊辰戰爭局勢的關鍵戰役<ref name="Kyotonorekishi" />{{rp|282}}。1868年，日本的首都從京都遷移至東京，除了皇室及[[公家_(日本)|公家]]之外大量商人也都從京都遷居至東京，京都面臨巨大的危機。京都市政財界採取大量措施試圖挽回危機局面，其主要內容包括實施學區制<ref name="Gakuku">{{cite web|url=http://www.city.kyoto.jp/somu/rekishi/fm/nenpyou/htmlsheet/toshi26.html|script-title=ja:町組改正と小学校|accessdate=2013-11-20|publisher=京都市|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20150923222206/http://www.city.kyoto.jp/somu/rekishi/fm/nenpyou/htmlsheet/toshi26.html|archivedate=2015-09-23}}</ref>；創建京都帝國大學，使京都成為西日本高等教育和研究的中心<ref name="Kinki" />{{rp|6}}；興建[[琵琶湖疏水|琵琶湖疏水]]<ref>{{cite web|url=http://www.city.kyoto.jp/somu/rekishi/fm/nenpyou/htmlsheet/toshi27.html|script-title=ja:琵琶湖疏水|accessdate=2013-11-20|publisher={{lang|ja|京都市}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20140308214520/http://www.city.kyoto.jp/somu/rekishi/fm/nenpyou/htmlsheet/toshi27.html|archivedate=2014-03-08}}</ref>作為水運路線，並進行[[水力發電|水力發電]]；鋪設日本最早的有軌電車[[京都市電|京都市電]]<ref name="Shiden">{{cite web|url=http://www.city.kyoto.jp/somu/rekishi/fm/nenpyou/htmlsheet/toshi28.html|script-title=ja:京都の市電|accessdate=2013-11-20|publisher={{lang|ja|京都市}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20130619184325/http://www.city.kyoto.jp/somu/rekishi/fm/nenpyou/htmlsheet/toshi28.html|archivedate=2013-06-19}}</ref>；舉辦第4屆內國勸業博覽會並修建[[平安神宮|平安神宮]]<ref>{{cite web|url=http://www.city.kyoto.jp/somu/rekishi/fm/nenpyou/htmlsheet/toshi29.html|script-title=ja:京都の博覧会|accessdate=2013-11-20|publisher=京都市|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20131129094118/http://www.city.kyoto.jp/somu/rekishi/fm/nenpyou/htmlsheet/toshi29.html|archivedate=2013-11-29}}</ref>。之後京都又在大正時期完成三大事業<ref name="Kinki" />{{rp|6}}（第二琵琶湖疏水、鋪設上下水道、擴寬道路及鋪設有軌電車）。這些措施使得京都跨過危機，開始轉變為現代化都市<ref name="Kyotochizumonogatari" />{{rp|49}}。1921年，京都實施其首個都市計劃事業，開始進入現代城市規劃時代<ref name="Kyotochizumonogatari" />{{rp|59}}。

在[[第二次世界大战|第二次世界大战]]時期，京都市是日本城市中較少遭到美軍空襲轟炸的都市<ref name="Kinki" />{{rp|8}}，因此京都市是日本少數仍然擁有豐富的戰前建築的城市之一。為了降低空襲造成的損害，[[堀川通|堀川通]]、[[御池通|御池通]]、[[五條通|五條通]]沿線的建築被強制拆除，卻也提供之後拓寬道路的空間<ref name="Kyoto Chronology" />。然而卻也正因為舊市區保存較好，導致市中心很難修建新的道路和公園，使得京都市中心地區的道路面積率和公園面積率極低，並且都市更新也進展較慢<ref name="Kinki" />{{rp|9}}。戰後京都的開發主要依照1950年制定的「京都國際文化觀光都市建設法」進行<ref name="Kyotochizumonogatari" />{{rp|61}}，在郊外地區修建[[新市鎮#日本|新市鎮]]以解決住宅問題，並且將京都車站以北規劃為保存地區，以南則規劃為開發地區。1956年9月1日，京都市成為[[政令指定都市|政令指定都市]]。1964年10月，[[東海道新幹線|東海道新幹線]]開始通車，極大的促進京都觀光產業的發展。1978年，京都市電被宣告廢止，地鐵取代市電成為京都市內重要的交通方式<ref name="Shiden" />。這些事件都象徵著戰後京都都市空間的巨大變化。進入[[平成|平成]]時代之後，在平安遷都1200年之際的1994年，[[古都京都的文化財|京都的多處古蹟]]被列入世界文化遺產<ref name="UNESCO" />。京都已成為日本具代表性的觀光都市和文化都市。

== 人口 ==
[[File:Kyoto_population_pyramid.svg|thumb]]
在9世紀時，[[平安京|平安京]]的人口估計約有10萬至15萬人<ref name="Murai">{{cite book |author={{lang|ja|林屋辰三郎, 村井康彦}} |script-title=ja:《京都市の地名》 |year=1979 |publisher={{lang|ja|平凡社}} |location=東京都 |isbn=4582490271|language=ja}}</ref>{{rp|31-43}}。至10-11世紀時，平安京的人口增加至17萬至20萬人。在1571年，京都的人口則估計約有20萬至30萬人<ref name="Takao">{{cite book |author={{lang|ja|高尾一彦, 林屋辰三郎, 松浦玲}} |script-title=ja:《京都の歴史 3 近世の胎動》 |year=1968 |publisher={{lang|ja|學藝書院}} |location=東京都|language=ja}}</ref>{{rp|31-43}}。而在江戶時代的大部份時期，據{{lang|ja|京都御役所向大概覚書}}等統計資料，京都的人口大致在35萬人左右，雖然低於江戶，但和大阪相當，是當時日本屈指可數的大都市<ref name="Kyotonorekishi" />{{rp|264}}。明治維新之後，隨著大量公家和商人遷至東京，京都的人口也從約35萬大幅減少至約22萬<ref name="Chishi" />{{rp|328}}。1889年時，京都有人口279,792人，僅次於東京和大阪，是當時日本的第三大都市<ref>{{cite web|url=http://www.city.osaka.lg.jp/shimin/cmsfiles/contents/0000130/130263/shiryou01-1.pdf|script-title=ja:大都市における自治権拡充運動の歴史|accessdate=2013-11-22|publisher=大阪市|language=ja|deadurl=yes|archiveurl=https://web.archive.org/web/20140912120722/http://www.city.osaka.lg.jp/shimin/cmsfiles/contents/0000130/130263/shiryou01-1.pdf|archivedate=2014-09-12|title={title}}}</ref>。1932年，京都市人口突破100萬人<ref>{{cite web |url=http://www.city.kyoto.lg.jp/kamigyo/page/0000012632.html |script-title=ja:年表　1928－1947 |accessdate=2013-11-22 |publisher={{lang|ja|上京区}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20131203013857/http://www.city.kyoto.lg.jp/kamigyo/page/0000012632.html |archivedate=2013-12-03 }}</ref>。然而在1970年代中期，京都市人口在超過140萬之後由快速增加轉入停滯狀態<ref>{{cite web|url=http://www.city.kyoto.lg.jp/templates/shingikai_kekka/cmsfiles/contents/0000153/153831/shiryo2.pdf|script-title=ja:京都市の現状と動向|accessdate=2013-11-22|publisher=京都市|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20131203013840/http://www.city.kyoto.lg.jp/templates/shingikai_kekka/cmsfiles/contents/0000153/153831/shiryo2.pdf|archivedate=2013-12-03}}</ref>。2017年10月時，京都市人口約有147萬人<ref name="Population" />
	
京都市人口的特徵之一是生育率較低，因此兒童人口佔總人口的比例較低。2015年，京都市的[[总和生育率|总和生育率]]為1.30，大幅低於日本全國的平均水準1.46<ref>{{cite web|url=http://www.city.kyoto.lg.jp/sogo/page/0000211675.html|script-title=ja:平成27年京都市の合計特殊出生率について|accessdate=2016-12-06|publisher={{lang|ja|京都市}}|language=ja|deadurl=yes|archiveurl=https://web.archive.org/web/20161220075940/http://www.city.kyoto.lg.jp/sogo/page/0000211675.html|archivedate=2016-12-20}}</ref>。低出生率也使得京都市的高齡者人口比例較高。2016年，京都市65歲以上的高齡人口數量達到40萬485人，比例達到27.2%<ref>{{cite web|url=http://www.city.kyoto.lg.jp/sogo/page/0000209116.html|script-title=ja:京都市統計月報の更新について|accessdate=2016-12-06|publisher={{lang|ja|京都市}}|language=ja|deadurl=yes|archiveurl=https://web.archive.org/web/20161220081428/http://www.city.kyoto.lg.jp/sogo/page/0000209116.html|archivedate=2016-12-20}}</ref>。京都市人口的另一特徵是18歲至24歲的人口佔人口比例較高。這一年齡的人口多為大學生，也體現出了京都市是一座大學城市<ref name="Student Population"/>。另外在2016年，京都市人口中有43,067人是外國人<ref>{{cite web|url=http://www.moj.go.jp/housei/toukei/toukei_ichiran_touroku.html|script-title=ja:在留外国人統計|accessdate=2017-10-17|publisher={{lang|ja|法務省}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20171007122307/http://www.moj.go.jp/housei/toukei/toukei_ichiran_touroku.html|archivedate=2017-10-07}}</ref>。

== 行政區劃 ==
{{File2|zh-hant=Wards of kyoto.svg|zh-hans=Wards of kyoto2.svg|thumb|left|250px|京都市行政區劃}}
1889年，京都市開始實施[[市制_(日本制度)|市制]]，成為日本最早的市之一，同時市區附近的一些農村地區也被編入京都市的範圍之內。當時京都市的範圍包括了現在上京區、中京區、下京區、東山區的全部以及左京區和北區的部份地區<ref name="Merger history of Kyoto">{{cite web|url=http://www.pref.kyoto.jp/gappei/documents/houkokusyodata-sankou1.pdf|script-title=ja:京都府内の市町村の歴史|accessdate=2013-11-20|publisher={{lang|ja|京都府}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20131203002353/http://www.pref.kyoto.jp/gappei/documents/houkokusyodata-sankou1.pdf|archivedate=2013-12-03}}</ref>。之後京都市的行政區範圍又在1902年和1918年有所擴大<ref name="Merger history of Kyoto"/>。1931年<ref name="Merger history of Kyoto"/>，京都市附近的26個町村被劃入京都市內，行政區面積因此大幅擴大至之前的4.8倍，且京都重要的[[衛星城市|衛星都市]]—伏見在這次[[市町村合併|合併]]中被併入京都市，京都進入「大京都」時代<ref name="Kyotochizumonogatari" />{{rp|50-51}}。其後京都市的行政區面積又在1948年、1949年、1950年、1957年、1959年<ref name="Merger history of Kyoto"/>和2005年<ref>{{cite web|url=http://www.pref.kyoto.jp/tiho/documents/arikataken-gappei.pdf|script-title=ja:京都府における平成の市町村合併|accessdate=2013-11-20|publisher={{lang|ja|京都府}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20131203003512/http://www.pref.kyoto.jp/tiho/documents/arikataken-gappei.pdf|archivedate=2013-12-03}}</ref>數次擴大。

== 政治 ==
[[File:Kyoto_city_hall.JPG|thumb]]
和日本其他地區相比，[[日本共產黨|日本共產黨]]等左派政黨在京都市有著較大的影響力。在[[日本众议院|日本众议院]]選舉仍然實行{{link-ja|中選舉區制|中選挙区制}}時期，{{link-ja|京都府第1區|京都府第1区 (中選挙区)}}（全區均位於京都市內）是少數日本共產黨曾有多位候選人當選的選舉區，且日本共產黨的候選人曾經4次以首位得票率在京都府1區當選<ref>{{cite web |url=http://www.jcp-kyoto.jp/prof/history/dai3syou/ |script-title=ja:第三章　第7回党大会と綱領確定による京都府党の前進 |accessdate=2013-11-22 |publisher={{lang|ja|日本共産党 京都府委員会}} |language=ja |deadurl=yes |archiveurl=https://web.archive.org/web/20131203031928/http://www.jcp-kyoto.jp/prof/history/dai3syou/ |archivedate=2013-12-03 }}</ref>。即使在改革為[[小選區制|小選舉區制]]後，日本共產黨仍然能在京都市內的選舉區內獲得比日本全國平均水準更高的得票率<ref>{{cite web|url=http://grading.jpn.org/y2408009.html|script-title=ja:衆院選小選挙区の日本共産党得票率ランキング|accessdate=2013-11-24|publisher={{lang|ja|都道府県格付研究所}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20131203093219/http://grading.jpn.org/y2408009.html|archivedate=2013-12-03}}</ref>。現在北區、上京區、中京區、下京區、南區被劃為[[京都府第1區|京都府第1區]]，左京區、東山區、山科區被劃為[[京都府第2區|京都府第2區]]，伏見區與[[向日市|向日市]]、[[長岡京市|長岡京市]]、[[大山崎町|大山崎町]]被劃為[[京都府第3區|京都府第3區]]，右京區、西京區與[[龜岡市|龜岡市]]、[[南丹市|南丹市]]、[[京丹波町|京丹波町]]被劃為[[京都府第4區|京都府第4區]]<ref>{{Cite web|url=http://www.soumu.go.jp/main_content/000240068.pdf|title=|accessdate=2017-10-22|author=|date=|publisher={{lang|ja|総務省}}|language=ja|script-title=ja:選挙区}}</ref>。

在[[地方分权|地方自治]]層面，京都市是日本最早的由左派人士執政的「{{link-ja|革新自治體|革新自治体}}」<ref>{{cite web|url=http://ousar.lib.okayama-u.ac.jp/file/41576/oer_030_4_389_412.pdf|script-title=ja:戦後日本における地方自治財政の回顧 - いわゆる ｢革新自治体｣の役割について-|accessdate=2013-11-22|publisher={{lang|ja|岡山大学経済学会雑誌}}|language=ja|deadurl=yes|archiveurl=https://web.archive.org/web/20131203021240/http://ousar.lib.okayama-u.ac.jp/file/41576/oer_030_4_389_412.pdf|archivedate=2013-12-03}}</ref>之一。1950年，在選舉中得到左派團體支持的{{link-ja|高山義三|高山義三}}當選京都市長，使得京都邁入革新自治體行列。然而之後高山義三轉為保守派<ref>{{cite web|url=http://web.kyoto-inet.or.jp/people/my10s4/cityowaku.htm|script-title=ja:近現代都市京都と都市行政の歩み-|accessdate=2013-11-24|publisher=kyoto-Inet|language=ja|deadurl=yes|archiveurl=https://web.archive.org/web/20131203042702/http://web.kyoto-inet.or.jp/people/my10s4/cityowaku.htm|archivedate=2013-12-03}}</ref>，並與革新自治體的代表人物—京都府知事{{link-ja|蜷川虎三|蜷川虎三}}對立。1967年，在選舉中得到日本社會黨和日本共產黨支持的{{link-ja|富井清|富井清}}當選京都市長<ref>{{cite web|url=http://web.kyoto-inet.or.jp/people/my10s4/mayor.htm|script-title=ja:京都市長選挙の歩み|accessdate=2013-11-24|publisher=kyoto-Inet|language=ja|deadurl=yes|archiveurl=https://web.archive.org/web/20131203034845/http://web.kyoto-inet.or.jp/people/my10s4/mayor.htm|archivedate=2013-12-03}}</ref>，京都市再次成為革新自治體。之後的兩任京都市長{{link-ja|舩橋求己|舩橋求己}}、{{link-ja|今川正彥|今川正彦}}也延續了前任市長重視社會福利的政策。這種情況直到1989年{{link-ja|田邊朋之|田辺朋之}}當選京都市長之後才開始改變。此後歷任京都市長雖然都是得到[[自由民主党_(日本)|自由民主黨]]等保守團體支持的候選人，然而共產黨支持的候選人仍能得到相當高的得票率<ref>{{cite web|url=http://www.city.kyoto.jp/senkyo/24_sityo/tositri01.html|script-title=ja:投票結了速報　|accessdate=2013-11-22|publisher={{lang|ja|京都市選挙管理委員会}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20131203014315/http://www.city.kyoto.jp/senkyo/24_sityo/tositri01.html|archivedate=2013-12-03}}</ref>。

現任京都市長是[[門川大作|門川大作]]，他在2008年的市長選舉中當選並在2012年<ref>{{cite web|url=http://www.kyoto-np.co.jp/kp/2012senkyo/kyoto/|script-title=ja:2012年　京都市長選|accessdate=2013-11-22|publisher={{lang|ja|京都新聞}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20150804022811/http://www.kyoto-np.co.jp/kp/2012senkyo/kyoto/|archivedate=2015-08-04}}</ref>和2016年連任<ref>{{cite web|url=http://www.kyoto-np.co.jp/kp/2016senkyo/kyoto/|script-title=ja:2016　京都市長選|accessdate=2016-02-21|publisher={{lang|ja|京都新聞}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20160211225806/http://www.kyoto-np.co.jp/kp/2016senkyo/kyoto/|archivedate=2016-02-11}}</ref>。現在京都市議會共有67位議員<ref>{{cite web|url=http://www2.city.kyoto.lg.jp/shikai/meibo/index.html|script-title=ja:議員名簿|accessdate=2015-06-10|publisher=京都市会|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20150610171854/http://www2.city.kyoto.lg.jp/shikai/meibo/index.html|archivedate=2015-06-10}}</ref>。市議會中最大黨派是自由民主黨，有議員20人；其他的主要政黨有日本共產黨、[[民主党_(1998年)|民主黨]]、[[公明黨|公明黨]]、{{link-ja|京都黨|京都党}}等黨派。

京都市的財政狀況極為嚴峻。京都市的財政狀況長期以來劣於其他政令指定都市<ref name="Kyoto Keizai Tankyu">{{cite book |author={{lang|ja|岡田知弘}}|script-title=ja:《京都経済の探究》 |year=2006 |publisher={{lang|ja|高菅出版}} |location=京都 |isbn=4-901793-23-3|language=ja}}</ref>{{rp|272}}。2001年10月，京都市宣布進入「財政非常事態」<ref>{{cite web |url=http://www.city.kyoto.lg.jp/sogo/page/0000013420.html |script-title=ja:市長記者会見（2001年10月31日） |deadurl=no|language=ja}}</ref>。造成京都市財政狀況不佳的主要原因有固定資產稅收較少<ref>{{cite web|url=http://www.city.kyoto.lg.jp/gyozai/cmsfiles/contents/0000105/105022/22-6-02kyoutoshizaisei.pdf|script-title=ja:他都市比較で見る京都市財政の特徴|accessdate=2013-11-24|publisher={{lang|ja|京都市}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20131203014708/http://www.city.kyoto.lg.jp/gyozai/cmsfiles/contents/0000105/105022/22-6-02kyoutoshizaisei.pdf|archivedate=2013-12-03}}</ref>、[[京都市營地下鐵|地鐵]][[東西線_(京都市營地下鐵)|東西線]]長期以來的巨額赤字等<ref>{{cite == 經濟 ==
2015年，京都市市內生產總值有63,194億日元，比上一年增加6.8%。市民所得有45,880億日元，人均市民所得有311.0萬日元。農林水產業只佔京都市生產總值的0.1%，製造業及建設業合計佔26.6%，其他部分則都是第三產業<ref>{{cite web|url=http://www2.city.kyoto.lg.jp/sogo/toukei/Economy/Accounts/Report/2017/0_gaiyo_new.pdf|script-title=ja:京都市市民経済計算　平成２７年度推計結果（概要）|accessdate=2018-09-04|publisher={{lang|ja|京都市}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20180831021935/http://www2.city.kyoto.lg.jp/sogo/toukei/Economy/Accounts/Report/2017/0_gaiyo_new.pdf|archivedate=2018-09-04}}</ref>。
=== 農業 ===
京都市的[[农业|农业]]主要集中於郊外的山區地區，是典型的都市近郊型農業<ref name="Kyotochizumonogatari" />{{rp|93}}。然而和日本其他地區相同，京都的耕地面積和農業人口在近年持續減少<ref name="Kyoto Agriculture">{{cite web|url=http://www.city.kyoto.lg.jp/sankan/cmsfiles/contents/0000060/60247/02.pdf|script-title=ja:II 業種別産業の動向 1 農林業|accessdate=2013-11-24|publisher={{lang|ja|京都市}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20131203014208/http://www.city.kyoto.lg.jp/sankan/cmsfiles/contents/0000060/60247/02.pdf|archivedate=2013-12-03}}</ref>。京都市最具代表性的農產品為[[蔬菜|蔬菜]]。京都生產的蔬菜被稱為{{link-ja|京野菜|京野菜}}<ref name="Kyotochizuemaki">{{cite book |author={{lang|ja|植村善博・香川貴志}}|script-title=ja:《京都地図絵巻》 |year=2007 |publisher={{lang|ja|古今書院}} |location=東京都 |isbn=978-4-7722-3111-4|language=ja}}</ref>{{rp|55}}，其產值佔京都市農產品總產值的84.6%<ref name="Kyoto Agriculture"/>。京野菜的主要品種包括了[[萝卜|-{zh-hans:萝卜; zh-hant:蘿蔔;}-]]、[[芜菁|蕪菁]]、[[日本蕪菁|日本蕪菁]]、[[茄子|茄子]]、[[南瓜|南瓜]]、[[葱|葱]]等<ref>{{cite web|url=http://www.pref.kyoto.jp/nosoken/html/dentoyasai.html |archiveurl=https://web.archive.org/web/20051106011242/http://www.pref.kyoto.jp/nosoken/html/dentoyasai.html |archivedate=2005-11-06 |script-title=ja:京の伝統野菜 |accessdate=2013-11-24 |publisher={{lang|ja|京都府}} |deadurl=yes |language=ja}}</ref>。以京野菜為原料製作的[[醃|日式醃菜]]（{{lang|ja|漬物}}）是京都的重要特產<ref>{{cite web |url=http://www.suguki-narita.com/suguki.html |script-title=ja:おいしい漬物「すぐき」について |accessdate=2013-11-24|publisher={{lang|ja|京なり田}}|archiveurl=https://web.archive.org/web/20131203010736/http://www.suguki-narita.com/suguki.html |archivedate=2013-12-03 |deadurl=no|language=ja}}</ref>。除了京野菜之外，[[稻|大米]]、[[水果|水果]]、[[花|鮮花]]也是京都的主要農產品<ref>{{cite web|url=http://www.machimura.maff.go.jp/machi/contents/26/100/index.html|script-title=ja:京都府京都市|accessdate=2017-11-02|publisher={{lang|ja|農林水産省}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20171107022227/http://www.machimura.maff.go.jp/machi/contents/26/100/index.html|archivedate=2017-11-07}}</ref>。

=== 工業 ===
[[File:Nintendo_office.jpg|thumb]]
京都是日本重要的[[第二产业|工業]]都市之一。2015年，京都市的工業品生產額有25,373億日元<ref>{{cite web|url=https://www2.city.kyoto.lg.jp/sogo/toukei/Publish/Analysis/News/089EconomicCensus2016E.pdf|script-title=ja:平成28年経済センサス‐活動調査（製造業）の集計結果|accessdate=2018-09-04|publisher={{lang|ja|京都市}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20180904100356/https://www2.city.kyoto.lg.jp/sogo/toukei/Publish/Analysis/News/089EconomicCensus2016E.pdf|archivedate=2018-09-04}}</ref>。傳統產業、出版印刷、食品飲料、機械儀器、電子產業是京都工業中的主要產業<ref name="Industry">{{cite web|url=http://www2.city.kyoto.lg.jp/sogo/toukei/Economy/Data/Manufact/2014/main.pdf|script-title=ja:平成26年工業統計調査結果|accessdate=2017-10-27|publisher={{lang|ja|京都市}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20171028145835/http://www2.city.kyoto.lg.jp/sogo/toukei/Economy/Data/Manufact/2014/main.pdf|archivedate=2017-10-28}}</ref>。[[和服|和服]]相關產業在京都的傳統產業中佔有相當重要的地位。[[西陣|西陣]]是京都和服產業的一大中心地，著名的[[西陣織|西陣織]]即因其主要產地位於西陣而得名<ref>{{cite web |url=http://www.nishijin.or.jp/ori/index.html |script-title=ja:西陣織について |accessdate=2013-11-24 |publisher={{lang|ja|西陣織工業組合}} |archiveurl=https://web.archive.org/web/20131023111305/http://www.nishijin.or.jp/ori/index.html |archivedate=2013-10-23 |deadurl=yes |language=ja }}</ref>。京都其他的著名傳統產業製品還包括了陶器（[[清水燒|清水燒]]）和[[友禪染|友禪染]]等。然而在近年，由於市場規模縮小和後繼者不足等問題，京都的傳統產業面臨嚴重的困境<ref name="Kyotochizuemaki" />{{rp|45}}。在現代工業方面，總部設在京都的知名企業有[[島津製作所|島津製作所]]、{{link-ja|堀場製作所|堀場製作所}}、[[京瓷|京瓷]]、[[欧姆龙|欧姆龙]]、[[日本電產|日本電產]]、[[村田製作所|村田製作所]]、[[羅姆電子|羅姆電子]]、[[任天堂|任天堂]]等<ref name="Kyotochizuemaki" />{{rp|42-43}}，這些公司大多都是電子業企業。因京都市內可供建設工場的土地面積狹小且人力成本較高，許多總部設在京都的企業將生產據點設在日本國內其他地區或者海外，而將運營總部和研發中心設在京都<ref name="Chishi" />{{rp|352}}。由於大學林立學生眾多，京都也是日本[[创业公司|创业公司]]數量較多的都市<ref name="Kyotochizumonogatari" />{{rp|87}}。京都市西部的{{link-ja|京都研究園區|京都リサーチパーク}}集中了大量創業公司<ref>{{cite web |url=https://www.krp.co.jp/about/ |script-title=ja:京都リサーチパークについて |accessdate=2018-09-04 |publisher={{lang|ja|京都リサーチパーク}} |archiveurl=https://web.archive.org/web/20180206081547/http://www.krp.co.jp/about/ |archivedate=2018-02-06 |deadurl= |language=ja |title=存档副本 }}</ref>。就企業數和生產額來看，南區、右京區和伏見區是京都市主要的工業區<ref name="Industry" />。

=== 服務業 ===
[[File:Kyoto_Gion_Matsuri_J09_136.jpg|thumb]]
[[第三产业|服務業]]是佔京都市經濟比重最大的產業<ref name="Kyoto Economic">{{cite web|url=http://www.city.kyoto.lg.jp/sankan/cmsfiles/contents/0000109/109389/2011_zentai.pdf|script-title=ja:京都市の経済 2011年版|accessdate=2013-11-24|publisher=京都市|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20131203023537/http://www.city.kyoto.lg.jp/sankan/cmsfiles/contents/0000109/109389/2011_zentai.pdf|archivedate=2013-12-03}}</ref>。在服務業之中，又以商業和旅遊業最為發達。為保存古都風景，京都市是日本國內對大型商業設施管制最為嚴格的城市<ref>{{cite web|url=http://www.city.kyoto.lg.jp/sankan/cmsfiles/contents/0000078/78346/guideplan-new.pdf|script-title=ja:京都市商業集積ガイドプラン|accessdate=2013-11-24|publisher={{lang|ja|京都市}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20131203022845/http://www.city.kyoto.lg.jp/sankan/cmsfiles/contents/0000078/78346/guideplan-new.pdf|archivedate=2013-12-03}}</ref>。這也使得京都市內[[便利商店|便利店]]和小型[[超級市場|超市]]等小規模商業設施數量眾多。京都市也是日本[[商圈|商店街]]數量較多的大都市<ref name="Kyotochizumonogatari" />{{rp|91}}。2015年，京都市的全年商品銷售額有53,632億日元<ref>{{cite web|url=https://www2.city.kyoto.lg.jp/sogo/toukei/Publish/Analysis/News/096EconomicCensus2016C.pdf|script-title=ja:平成28年経済センサス‐活動調査（卸売業・小売業）の集計結果|accessdate=2018-09-04|publisher={{lang|ja|京都市}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20180904093459/http://www2.city.kyoto.lg.jp/sogo/toukei/Publish/Analysis/News/096EconomicCensus2016C.pdf|archivedate=2018-09-04}}</ref>。京都市的[[百貨公司|百貨商店]]則多集中於[[四條河原町|四條河原町]]地區和京都站地區<ref name="Kyotochizuemaki" />{{rp|38-39}}；主要的大型超市則多集中在近郊的鐵路沿線地區<ref name="Kyotochizuemaki" />{{rp|39-41}}。

旅遊業是京都服務業的另一大支柱產業。京都市有著豐富的觀光資源，吸引了眾多來自世界各地的遊客。2017年，來訪京都的遊客人數達5,362萬人，並有1,557萬人在京都住宿。京都的遊客中有三分之二是女性，三分之一是男性；就年齡來看，以50歲以上的遊客較多，且有六成遊客到訪京都次數超過10次<ref name="Kyoto Tourism">{{cite web|url=http://www.city.kyoto.lg.jp/sankan/cmsfiles/contents/0000240/240130/kyosa29saishu.pdf|script-title=ja:平成29年（2017）京都観光総合調査|accessdate=2018-09-04|publisher={{lang|ja|京都市}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20180904092846/http://www.city.kyoto.lg.jp/sankan/cmsfiles/contents/0000240/240130/kyosa29saishu.pdf|archivedate=2018-09-04}}</ref>。有眾多日本初中和高中都選擇京都作為[[學校旅行|修學旅行]]目的地，特別是東日本的學校<ref name="Kyoto Tourism"/>。在外國人遊客方面，2017年，共有約353萬外國遊客曾在京都住宿，其中以[[中國大陸|中國大陸]]人最多，[[台灣|台灣]]人其次<ref name="Kyoto Tourism"/>。最多外國人觀光客到訪的景點依次是[[清水寺|清水寺]]、[[二條城|二條城]]、[[祇園|祇園]]、[[鹿苑寺|金閣寺]]、[[伏見稻荷大社|伏見稻荷大社]]<ref name="Kyoto Tourism"/>。旅遊業的發展給京都帶來了巨大的經濟效果，京都市有10%以上的就業人口都是從事旅遊相關產業的工作<ref>{{cite web|url=http://www.city.kyoto.lg.jp/sankan/cmsfiles/contents/0000229/229756/besshi.pdf|script-title=ja:観光消費額に係る京都市域への経済波及効果及び国との比較|accessdate=2018-09-04|publisher={{lang|ja|京都市}}|language=ja}}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，並在2017年為京都帶來超過11,268億日元的消費<ref name="Kyoto Tourism"></ref>。在其他服務業方面，京都的[[金融|金融業]]頗具特色。和日本其他大城市相比，[[都市銀行|都市銀行]]在京都的影響力相對較小，地方銀行[[京都銀行|京都銀行]]雖然在京都府內是最大金融機構但在京都市內則居於次席。而{{link-ja|信用金庫|信用金庫}}則在京都經濟中發揮著重要作用，{{link-ja|京都中央信用金庫|京都中央信用金庫}}更是京都市內最大金融機構<ref>{{cite web|url=https://www.tdb.co.jp/report/watching/press/pdf/s161202_50.pdf|script-title=ja:7 年連続で京都銀行が首位～京都市内では京都中央信金が首位～|accessdate=2017-10-24|publisher={{lang|ja|帝国データバンク}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20171025131708/https://www.tdb.co.jp/report/watching/press/pdf/s161202_50.pdf|archivedate=2017-10-25}}</ref>。和大型的都市銀行相比，信用金庫更易融資給中小企業，這也是京都孕育出眾多創業公司的重要原因<ref name="Kyoto Economic" />。

== 都市景觀 ==
[[File:MAC_Kyoto_Japan.jpg|170px]]
京都市是代表日本文化的古都，為維護古都景觀，京都有著日本最為嚴格的景觀規制法律。1930年，京都市就將一些觀光景點附近的地區指定為風致地區，對風致地區內的建築進行限制<ref name="Kyotochizuemaki" />{{rp|15}}。1964年，[[京都塔|京都塔]]完工，其現代的造型和較高的高度引發了巨大的爭議<ref name="Keikan">{{cite web |url=http://www.yomiuri.co.jp/nie/sp/2007/02/01.htm |script-title=ja:NIE特集 景観論争 |accessdate=2013-11-20|publisher={{lang|ja|読売新聞}}|archiveurl=https://web.archive.org/web/20131203043651/http://www.yomiuri.co.jp/nie/sp/2007/02/01.htm |archivedate=2013-12-03 |deadurl=no|language=ja}}</ref>，也使得保護古都景觀的呼聲高漲。1965年，受到景觀論爭的影響，日本制定了古都保存法<ref>{{cite web|url=http://law.e-gov.go.jp/cgi-bin/idxselect.cgi?IDX_OPT=1&H_NAME=%8c%c3%93%73%82%c9%82%a8%82%af%82%e9%97%f0%8e%6a%93%49%95%97%93%79%82%cc%95%db%91%b6%82%c9%8a%d6%82%b7%82%e9%93%c1%95%ca%91%5b%92%75%96%40&H_NAME_YOMI=%82%a0&H_NO_GENGO=H&H_NO_YEAR=&H_NO_TYPE=2&H_NO_NO=&H_FILE_NAME=S41HO001&H_RYAKU=1&H_CTG=1&H_YOMI_GUN=1&H_CTG_GUN=1|script-title=ja:古都における歷史的風土の保存に関する特別措置法|accessdate=2013-11-24|publisher={{lang|ja|イーガブ}}|language=ja}}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。1973年，京都市將全市設為高度限制地區，規定市中心商務地區建築高度不得超過45米（2008年又調整為31米）<ref>{{cite web|url=http://www.city.kyoto.lg.jp/tokei/cmsfiles/contents/0000146/146248/guide_takasa.pdf|script-title=ja:京の景観ガイドライン 建築物の高さ編|accessdate=2013-11-24|publisher={{lang|ja|京都市}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20131203023859/http://www.city.kyoto.lg.jp/tokei/cmsfiles/contents/0000146/146248/guide_takasa.pdf|archivedate=2013-12-03}}</ref>。然而對古都景觀的破壞並未因此而停止，京都市中心大量的傳統民居[[京町家|京町家]]被高層大樓所取代，市中心的傳統風貌已經被完全改變。而{{link-ja|京都賓館|京都ホテル}}和京都站這兩座巨大建築的建設更掀起了一波新的景觀論爭高潮<ref name="Kyotochizuemaki" />{{rp|14-15}}。2004年，隨著日本制定《景觀法》<ref>{{cite web|url=http://law.e-gov.go.jp/cgi-bin/idxselect.cgi?IDX_OPT=1&H_NAME=%8c%69%8a%cf%96%40&H_NAME_YOMI=%82%a0&H_NO_GENGO=H&H_NO_YEAR=&H_NO_TYPE=2&H_NO_NO=&H_FILE_NAME=H16F16001000100&H_RYAKU=1&H_CTG=1&H_YOMI_GUN=1&H_CTG_GUN=1|script-title=ja:景観法施行規則|accessdate=2013-11-24|publisher={{lang|ja|イーガブ}}|language=ja}}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，京都市內的景觀規制進入新的階段。現在京都市全市均按照地區特徵被指定為不同類型的景觀規制地區<ref>{{cite web|url=http://www.city.kyoto.lg.jp/tokei/cmsfiles/contents/0000114/114167/map01.pdf|script-title=ja:景観計画区域　（区域区分図）|accessdate=2013-11-24|publisher={{lang|ja|京都市}}|language=ja|deadurl=yes|archiveurl=https://web.archive.org/web/20131203023418/http://www.city.kyoto.lg.jp/tokei/cmsfiles/contents/0000114/114167/map01.pdf|archivedate=2013-12-03|title=存档副本}}</ref><ref>{{cite web|url=http://www.city.kyoto.lg.jp/tokei/cmsfiles/contents/0000114/114167/keikaku01.pdf|script-title=ja:全体計画|accessdate=2013-11-24|publisher={{lang|ja|京都市}}|language=ja|deadurl=yes|archiveurl=https://web.archive.org/web/20131203020641/http://www.city.kyoto.lg.jp/tokei/cmsfiles/contents/0000114/114167/keikaku01.pdf|archivedate=2013-12-03}}</ref>。市中心大部份地區都被指定為美觀地區<ref>{{cite web|url=http://www.city.kyoto.lg.jp/tokei/cmsfiles/contents/0000114/114167/keikaku03.pdf|script-title=ja:市街地景観の整備に関する計画|accessdate=2013-11-24|publisher={{lang|ja|京都市}}|language=ja|deadurl=yes|archiveurl=https://web.archive.org/web/20131203022704/http://www.city.kyoto.lg.jp/tokei/cmsfiles/contents/0000114/114167/keikaku03.pdf|archivedate=2013-12-03}}</ref>；大多數觀光景點的附近則被指定為風致地區或歷史風土特別保存地區<ref>{{cite web|url=http://www.city.kyoto.lg.jp/tokei/cmsfiles/contents/0000114/114167/keikaku02henkou.pdf|script-title=ja:自然・歷史的景観の保全に関する計画|accessdate=2013-11-24|publisher={{lang|ja|京都市}}|language=ja|deadurl=yes|archiveurl=https://web.archive.org/web/20131203023930/http://www.city.kyoto.lg.jp/tokei/cmsfiles/contents/0000114/114167/keikaku02henkou.pdf|archivedate=2013-12-03}}</ref>；而在[[東寺|東寺]]、[[清水寺|清水寺]]境內等有知名眺望點的地區，則制定了關於眺望景觀的限制計劃，對附近建築物的高度進行限制<ref>{{cite web|url=http://www.city.kyoto.lg.jp/tokei/cmsfiles/contents/0000114/114167/keikaku04.pdf|script-title=ja:眺望景観の創生に関する計画|accessdate=2013-11-24|publisher={{lang|ja|京都市}}|language=ja|deadurl=yes|archiveurl=https://web.archive.org/web/20131203022216/http://www.city.kyoto.lg.jp/tokei/cmsfiles/contents/0000114/114167/keikaku04.pdf|archivedate=2013-12-03}}</ref>。在美觀地區新建建築時，建築物的外觀均需要經過審查後才能修建<ref>{{cite web|url=http://www.city.kyoto.lg.jp/tokei/cmsfiles/contents/0000146/146248/guide_kenchiku.pdf|script-title=ja:京の景観ガイドライン 建築物デザイン編|accessdate=2013-11-24|publisher={{lang|ja|京都市}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20131203022355/http://www.city.kyoto.lg.jp/tokei/cmsfiles/contents/0000146/146248/guide_kenchiku.pdf|archivedate=2013-12-03}}</ref>。京都市對戶外廣告也有嚴格的規定，市區內禁止懸掛閃爍式[[霓虹燈|霓虹燈]]廣告，並且對廣告牌的大小和顏色進行限制<ref>{{cite web|url=http://www.city.kyoto.lg.jp/tokei/cmsfiles/contents/0000146/146248/guide_koukoku%28L%29.pdf|script-title=ja:広告物編|accessdate=2013-11-24|publisher={{lang|ja|京都市}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20150921062939/http://www.city.kyoto.lg.jp/tokei/cmsfiles/contents/0000146/146248/guide_koukoku(L).pdf|archivedate=2015-09-21}}</ref>。因此京都市內部分店鋪的廣告牌採用與日本其他地區不同的顏色，例如京都市內部分[[麦当劳|麦当劳]]的看板顏色並非紅色而是棕色。
=== 建築 ===
[[File:Kiyomizu-dera_in_Kyoto-r.jpg|200px]]
京都市內集合了多種不同的風格的建築。古建築中最具代表性的是被選為[[世界遗产|世界文化遺產]]的[[古京都遗址|14處建築]]，其中除[[二條城|二條城]]之外均為寺廟或神社，顯示出宗教建築在京都建築藝術中有著極為重要的地位。由於歷史上戰亂和大火頻仍，京都市內現存的大多數古建築都修建於[[應仁之亂|應仁之亂]]之後<ref>{{cite web |url=http://kyotoshugakuryoko.jp/study/culture04.html |script-title=ja:京都の文化　建築・庭園 |accessdate=2013-11-25 |publisher={{lang|ja|きょうと修学旅行ナビ}} |archiveurl=https://web.archive.org/web/20131203202848/http://kyotoshugakuryoko.jp/study/culture04.html |archivedate=2013-12-03 |deadurl=yes |language=ja }}</ref>。[[室町时代|室町时代]]京都建築的代表是[[鹿苑寺|金閣寺]]和[[慈照寺|銀閣寺]]，這兩座建築也分別代表了[[北山文化_(日本)|北山文化]]和[[東山文化_(日本)|東山文化]]兩種不同的文化藝術流派<ref>{{cite web |url=http://www.d1.dion.ne.jp/~oo14/kitayamatohigasiyamabunka.html |script-title=ja:北山と東山文化 |accessdate=2013-11-25 |publisher={{lang|ja|日本史ナビゲーション}} |archiveurl=https://web.archive.org/web/20131203005845/http://www.d1.dion.ne.jp/~oo14/kitayamatohigasiyamabunka.html |archivedate=2013-12-03 |deadurl=yes |language=ja }}</ref>。室町時代也是京都庭園藝術最為發達的一個時期，[[禅宗|禅宗]]思想融入庭園藝術當中，[[枯山水|枯山水]]即為這一潮流的體現。室町時代庭園藝術的代表作有[[天龍寺|天龍寺]]、[[龍安寺|龍安寺]]、[[西芳寺|西芳寺]]、[[等持院|等持院]]等寺院的庭園<ref>{{cite web |url=http://www.homemate.co.jp/useful/nihonkenchikushi/syasin/kamakura/ |script-title=ja:写真で見る建築と様式　鎌倉～室町時代 |accessdate=2013-11-25|publisher={{lang|ja|ホームメイト}}|archiveurl=https://web.archive.org/web/20131203054540/http://www.homemate.co.jp/useful/nihonkenchikushi/syasin/kamakura/ |archivedate=2013-12-03 |deadurl=no|language=ja}}</ref>。在[[安土桃山时代|安土桃山时代]]和[[江戶时代|江戶时代]]，隨著技術的進步，重現平安時代建築修建當時的外觀變為可能，京都寺院建築的樣式也趨於復古<ref>{{cite web |url=http://www.homemate.co.jp/useful/nihonkenchikushi/syasin/aduti/index.shtml |script-title=ja:写真で見る建築と様式　安土・桃山～江戸時代 |accessdate=2013-11-25|publisher={{lang|ja|ホームメイト}}|archiveurl=https://web.archive.org/web/20131203061819/http://www.homemate.co.jp/useful/nihonkenchikushi/syasin/aduti/index.shtml |archivedate=2013-12-03 |deadurl=no|language=ja}}</ref>。例如[[東寺|教王護國寺]]（東寺）和[[清水寺|清水寺]]都是始建於[[平安时代|平安时代]]，在江戶時代重建的寺院建築。江戶時代京都的庭園建築的代表則是採用[[數寄屋造|數寄屋造]]風格建設的[[桂離宮|桂離宮]]<ref>{{cite web |url=https://www2.city.kyoto.lg.jp/somu/rekishi/fm/nenpyou/htmlsheet/bunka11.html　|script-title=ja:桂離宮と修学院離宮　|accessdate=2017-11-02|publisher={{lang|ja|京都市}}|language=ja}}</ref>。城郭建築在這一時代的建築史上有著重要的地位。京都城郭建築的代表則為建設於江戶時代的[[二條城|二條城]]<ref>{{cite web |url=https://www2.city.kyoto.lg.jp/bunshi/nijojo/ninomaru/index.html　|script-title=ja:二の丸御殿　|accessdate=2017-11-02|publisher={{lang|ja|京都市}}|language=ja}}</ref>。

進入近代之後，京都建築的代表則是[[京町家|京町家]]。京町家出現在江戶時代後期，是京都最具代表性的日本傳統民宅，高度以兩層居多，正面寬度狹窄但長度較長<ref>{{cite web |url=http://www.city.kyoto.lg.jp/tokei/cmsfiles/contents/0000103/103136/kyomatiyatokucho.pdf |script-title=ja:京町家の特徴 |accessdate=2013-11-25 |publisher={{lang|ja|京都市}} |archiveurl=https://web.archive.org/web/20131203023522/http://www.city.kyoto.lg.jp/tokei/cmsfiles/contents/0000103/103136/kyomatiyatokucho.pdf |archivedate=2013-12-03 |deadurl=yes |language=ja }}</ref>。然而在戰後，京町家的數量已經大幅減少。現在京都市內的京町家數量有約24,000戶<ref name="Virtual Kyoto" />{{rp|12－15}}。和日本眾多都市一樣，京都也在明治至昭和前期的這段時間內修建了大量西洋風建築。京都代表性的洋館建築有舊[[日本银行|日本银行]]京都支店（現為[[京都文化博物館|京都文化博物館]]別館）、中京郵便局、京都府立圖書館、[[京都國立博物館|京都國立博物館]]等<ref name="Virtual Kyoto" />{{rp|18-21}}。京都站則是京都現代主義建築的代表作。京都站巨大的體積和現代的造型引發了巨大的爭議，然而毫無疑問的是其已成為京都的地標<ref>{{cite web |url=http://www.mediawars.ne.jp/~m921320/a_map/map_of_kyoto_04.htm |script-title=ja:JR京都駅ビル |accessdate=2013-11-25 |publisher=Architectural Map |archiveurl=https://web.archive.org/web/20131203091427/http://www.mediawars.ne.jp/~m921320/a_map/map_of_kyoto_04.htm |archivedate=2013-12-03 |deadurl=yes |language=ja }}</ref>。

=== 語言 ===
[[京都方言|京言葉]]是通行於京都市的[[日本語言|日語方言]]，也是[[近畿方言|關西方言]]的一支。京言葉在歷史上曾長期是[[日语|日语]]的標準語，且現在仍對日語標準語有著很強的影響<ref>{{cite web |url=http://kanko.city.kyoto.lg.jp/wakaru/life/words/more/file01.html |script-title=ja:京ことばの成り立ち|accessdate=2013-11-25|publisher={{lang|ja|京都観光ナビ}}|archiveurl=https://web.archive.org/web/20131203091338/http://kanko.city.kyoto.lg.jp/wakaru/life/words/more/file01.html |archivedate=2013-12-03 |deadurl=no|language=ja}}</ref>。京言葉的特徵是較多使用長音，語音節奏相對較為緩慢，且有著複雜的敬語系統。然而近年來，京言葉受到{{link-ja|東京方言|東京方言}}和{{link-ja|大阪方言|大阪弁}}的較強影響，正逐漸被日語標準語同化<ref>{{cite web |url=http://www.kyoto-np.co.jp/kp/ojikoji/kotoba/henbo.html |script-title=ja:京都弁変ぼう 阪大教授ら調査|accessdate=2013-11-25|publisher={{lang|ja|京都新聞}}|archiveurl=https://web.archive.org/web/20100129125311/http://www.kyoto-np.co.jp/kp/ojikoji/kotoba/henbo.html |archivedate=2010-01-29 |deadurl=no|language=ja}}</ref>。

=== 文學 ===
京都是[[日本文学|日本文学]]的發源地，日本歷史上首部長篇小說《[[源氏物語|源氏物語]]》的故事舞台就在京都<ref>{{cite web |url=http://kanko.city.kyoto.lg.jp/detail.php?InforKindCode=4&ManageCode=4000062 |script-title=ja:源氏物語|accessdate=2013-11-25|publisher={{lang|ja|京都観光ナビ}}|archiveurl=https://web.archive.org/web/20131203021134/http://kanko.city.kyoto.lg.jp/detail.php?InforKindCode=4&ManageCode=4000062 |archivedate=2013-12-03 |deadurl=no|language=ja}}</ref>。除此之外，京都也是《[[枕草子|枕草子]]》、《[[方丈記|方丈記]]》、《[[更級日記|更級日記]]》、《[[今昔物語集|今昔物語集]]》、《[[平家物语|平家物语]]》等眾多日本古典文學作品的舞台。京都也是許多現代日本作家的憧憬之地。以京都為舞台的知名現代日本文學作品有《[[羅生門_(小說)|羅生門]]》、《[[檸檬_(小說)|檸檬]]》、《[[金閣寺_(小說)|金閣寺]]》、《[[古都_(川端康成)|古都]]》等<ref>{{cite web |url=http://www.kyotokanko.co.jp/bungaku_butai/ |script-title=ja:京都を舞台とした文学探訪|accessdate=2013-11-25|publisher={{lang|ja|京都じっくり観光}}|archiveurl=https://web.archive.org/web/20131203121133/http://www.kyotokanko.co.jp/bungaku_butai/ |archivedate=2013-12-03 |deadurl=no|language=ja}}</ref>。現代日本最著名的作家之一[[村上春樹|村上春樹]]也出生在京都，他的代表作之一《[[挪威的森林_(小說)|挪威的森林]]》的故事舞台就設定在京都<ref>{{cite web|url=http://www.hokubu-farm.net/11.html|script-title=ja:北部農林業地域振興協議会について|accessdate=2013-11-25|publisher={{lang|ja|京都市北部農林業地域振興協議会}}|archiveurl=https://web.archive.org/web/20131203011909/http://www.hokubu-farm.net/11.html|archivedate=2013年12月3日|deadurl=yes|language=ja|df=}}</ref>。

=== 傳統藝術 ===
京都是日本傳統藝術的中心城市。在平安時代，隨著[[國風文化|國風文化]]的發達，日本開始有了獨自的繪畫藝術<ref>{{cite web|url=http://www.d1.dion.ne.jp/~oo14/kokufubunka.html|script-title=ja:国風文化|accessdate=2013-11-25|publisher={{lang|ja|日本史ナビゲーション}}|archiveurl=https://web.archive.org/web/20131203010755/http://www.d1.dion.ne.jp/~oo14/kokufubunka.html|archivedate=2013-12-03|deadurl=yes|language=ja}}</ref>，而京都成為[[日本畫|日本畫]]的一大創作中心。京都本身也是眾多日本畫的素材，最著名的例子就是[[洛中洛外图|洛中洛外图]]<ref>{{cite web|url=http://www.emuseum.jp/detail/100318?word=&d_lang=ja&s_lang=ja&class=&title=&c_e=&region=&era=&cptype=&owner=&pos=289&num=4&mode=detail|script-title=ja:洛中洛外図屏風|accessdate=2013-11-25|publisher={{lang|ja|e国宝}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20150924000847/http://www.emuseum.jp/detail/100318?word=&d_lang=ja&s_lang=ja&class=&title=&c_e=&region=&era=&cptype=&owner=&pos=289&num=4&mode=detail|archivedate=2015-09-24}}</ref>。在近代，京都又是日本最早的美術學校京都府畫學校的創辦地<ref>{{cite web |url=http://www.kcua.ac.jp/profile/history/ |script-title=ja:沿革|accessdate=2013-11-25|publisher={{lang|ja|京都市立芸術大学}}|archiveurl=https://web.archive.org/web/20131203060118/http://www.kcua.ac.jp/profile/history/ |archivedate=2013-12-03 |deadurl=no|language=ja}}</ref>。京都也是[[尾形光琳|尾形光琳]]、[[竹內栖鳳|竹內栖鳳]]、{{link-ja|堂本印象|堂本印象}}<ref>{{cite web |url=http://insho-domoto.com/insho.html |script-title=ja:印象について|accessdate=2013-11-25|publisher={{lang|ja|京都府立堂本印象美術館}}|archiveurl=https://web.archive.org/web/20131203043226/http://insho-domoto.com/insho.html |archivedate=2013-12-03 |deadurl=no|language=ja}}</ref>等日本畫家的出身地。在表演藝術方面，日本傳統表演藝術的代表—[[歌舞伎|歌舞伎]]就是發祥於京都<ref>{{cite web|url=http://kanko.city.kyoto.lg.jp/detail.php?InforKindCode=4&ManageCode=3000005|script-title=ja:歌舞伎発祥の地|accessdate=2013-11-25|publisher={{lang|ja|京都観光ナビ}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20131203091334/http://kanko.city.kyoto.lg.jp/detail.php?InforKindCode=4&ManageCode=3000005|archivedate=2013-12-03}}</ref>。{{link-ja|南座|南座}}是京都最早的歌舞伎劇場，現在仍是日本最重要的歌舞伎表演場地之一。除了歌舞伎之外，京都也有[[能剧|能]]、[[文樂|文樂]]等日本傳統藝術的劇場。在平安時代，[[遣唐使|遣唐使]]將[[日本茶道|茶道]]自中國帶到日本，京都遂成為日本茶道的中心<ref>{{cite web|url=http://www.kyotodeasobo.com/art/theme/tea/ |script-title=ja:そもそも、「茶道」とは？|accessdate=2013-11-25|publisher={{lang|ja|京都で遊ぼうART}}|archiveurl=https://web.archive.org/web/20131203053031/http://www.kyotodeasobo.com/art/theme/tea/ |archivedate=2013-12-03 |deadurl=no|language=ja}}</ref>。京都也是[[華道|華道]]的發祥地<ref>{{cite web|url=http://hamadayori.com/hass-col/culture/kado.htm|script-title=ja:生け花発祥の地(華道発祥之地)|accessdate=2013-11-25|publisher={{lang|ja|発祥の地コレクション}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20131204030004/http://hamadayori.com/hass-col/culture/kado.htm|archivedate=2013-12-04}}</ref>，以[[池坊|池坊]]派<ref>{{cite web|url=http://www.ikenobo.jp/ikebanaikenobo/history/ |script-title=ja:いけばなの歴史|accessdate=2013-11-25|publisher={{lang|ja|いけばな池坊}}|archiveurl=https://web.archive.org/web/20131204154702/http://www.ikenobo.jp/ikebanaikenobo/history/ |archivedate=2013-12-04 |deadurl=no|language=ja}}</ref>為代表的眾多華道流派的本家都位於京都。

=== 節慶 ===
[[Image:Gozanokuribi_Daimonji3.JPG|thumb]]
京都最重要的節慶活動是有「京都三大祭」之稱的[[葵祭|葵祭]]、[[祇園祭|祇園祭]]、[[時代祭|時代祭]]和[[五山送火|五山送火]]，這四大活動合稱為「京都四大行事」<ref>{{cite web |url=http://www.kyokanko.or.jp/4dai/4dai_index.html |script-title=ja:京都四大行事のご案内|accessdate=2013-11-25|publisher={{lang|ja|京都市観光協会}}|archiveurl=https://web.archive.org/web/20131203023003/http://www.kyokanko.or.jp/4dai/4dai_index.html |archivedate=2013-12-03 |deadurl=no|language=ja}}</ref>。葵祭舉辦於每年5月15日，是京都三大祭中歷史最久的祭典，起源自6世紀中期[[賀茂別雷神社|上賀茂神社]]和[[賀茂御祖神社|下鴨神社]]的例祭<ref>{{cite web|url=http://www.kyoto-np.co.jp/kp/koto/aoi/arekore/kigen.html|script-title=ja:葵祭あれこれ|accessdate=2013-11-25|publisher={{lang|ja|京都新聞}}|language=ja|deadurl=yes|archiveurl=https://web.archive.org/web/20130627133315/http://kyoto-np.co.jp/kp/koto/aoi/arekore/kigen.html|archivedate=2013-06-27}}</ref>。葵祭遊行者使用的服裝均忠實再現平安時代的原貌<ref>{{cite web|url=http://www.kyoto-np.co.jp/kp/special/omoshiro/hito11_02.php|script-title=ja:葵祭 ２ 勅使の列|accessdate=2013-11-25|publisher={{lang|ja|京都新聞}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20130627133113/http://kyoto-np.co.jp/kp/special/omoshiro/hito11_02.php|archivedate=2013-06-27}}</ref>，是充滿貴族氣息的行事。祇園祭是[[八坂神社|八坂神社]]的祭禮，起源於9世紀時期，於每年7月1日開始舉辦，因其舉辦時間長達一個月且參與人數眾多，被列入{{link-ja|日本三大祭|日本三大祭}}之一<ref>{{cite web|url=http://www.kyoto-np.co.jp/kp/koto/gion/gion.php?mode=h|script-title=ja:祇園祭とは|accessdate=2013-11-25|publisher={{lang|ja|京都新聞}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20131107122151/http://kyoto-np.co.jp/kp/koto/gion/gion.php?mode=h|archivedate=2013-11-07}}</ref>。祇園祭的最高潮是在7月16日的前祭宵山活動和7月17日的前祭[[藝閣|山鉾]]巡行<ref>{{cite web|url=http://www.gionmatsuri.or.jp/yamahoko/|script-title=ja:山鉾について|accessdate=2013-11-25|publisher={{lang|ja|祇園祭山鉾連合会}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20131203004919/http://www.gionmatsuri.or.jp/yamahoko/|archivedate=2013-12-03}}</ref>。共有33架山鉾參加7月17日的山鉾巡行，規模極為龐大。五山送火在每年的8月16日舉辦<ref>{{cite web|url=http://www.kyokanko.or.jp/okuribi/kigen.html|script-title=ja:点火の起源|accessdate=2013-11-25|publisher={{lang|ja|京都市観光協会}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20131203014949/http://www.kyokanko.or.jp/okuribi/kigen.html|archivedate=2013-12-03}}</ref>，是[[日本盂蘭盆節|盂蘭盆節]]的壓軸活動。在這一天的20點開始，京都市郊外5處山上的大文字等圖案陸續點火<ref>{{cite web|url=http://www.kyoto-np.co.jp/kp/koto/gozan/02.php|script-title=ja:送り火百科|accessdate=2013-11-25|publisher={{lang|ja|京都新聞}}|language=ja|deadurl=yes|archiveurl=https://web.archive.org/web/20131107125659/http://kyoto-np.co.jp/kp/koto/gozan/02.php|archivedate=2013-11-07}}</ref>，象徵死者的靈魂回到冥府。在江戶時代後期還曾有其他點火圖案，然現已不存<ref>{{cite web|url=http://www.kyoto-np.co.jp/kp/koto/gozan/01.php|script-title=ja:五山送り火とは|accessdate=2013-11-25|publisher={{lang|ja|京都新聞}}|language=ja|deadurl=yes|archiveurl=https://web.archive.org/web/20131107125657/http://kyoto-np.co.jp/kp/koto/gozan/01.php|archivedate=2013-11-07}}</ref>。時代祭於每年10月15日至23日舉辦，系紀念平安遷都1100年於1895年開始的祭典活動<ref>{{cite web|url=http://www.heianjingu.or.jp/03/0101.html|script-title=ja:時代祭由緒|accessdate=2013-11-25|publisher={{lang|ja|平安神宮}}|language=ja|deadurl=yes|archiveurl=https://web.archive.org/web/20130811093401/http://heianjingu.or.jp/03/0101.html|archivedate=2013-08-11}}</ref>。祭典的中心內容是10月22日的遊行，其內容是再現自平安時代至明治時代的知名歷史人物及場景<ref>{{cite web|url=http://www.kyoto-np.co.jp/kp/koto/jidai/2007/towa.html|script-title=ja:時代祭とは|accessdate=2013-11-25|publisher={{lang|ja|京都新聞}}|language=ja|deadurl=yes|archiveurl=https://web.archive.org/web/20131107200315/http://kyoto-np.co.jp/kp/koto/jidai/2007/towa.html|archivedate=2013-11-07}}</ref>。和祇園祭相同，時代祭也是有眾多普通京都市民參加的活動<ref>{{cite web|url=http://www.kyokanko.or.jp/jidai/enkaku.html|script-title=ja:時代祭について|accessdate=2013-11-25|publisher={{lang|ja|京都市観光協会}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20131203020123/http://www.kyokanko.or.jp/jidai/enkaku.html|archivedate=2013-12-03}}</ref>。

=== 飲食 ===
[[京料理|京料理]]是京都的傳統飲食<ref name="Ryori Rekishi">{{cite book |author={{lang|ja|熊倉功夫}}|script-title=ja:《日本料理の歴史》 |year=2006 |publisher={{lang|ja|吉川弘文館}} |location=東京都 |isbn=978-4-642-05645-8|language=ja}}</ref>{{rp|179}}。由於京都地處內陸，因此和日本其他地方料理不同，京料理較少使用[[海鮮|海鮮]]，且更加強調烹飪技法<ref name="Ryori Rekishi" />{{rp|152}}。加上京都佛寺眾多，[[素齋|精進料理]]和[[懷石料理|懷石料理]]（{{lang|ja|懐石料理}}）又尤其發達。[[豆腐|豆腐]]是京料理最為常用的食材，京都市內也有眾多生產豆腐的工廠<ref>{{cite web|url=http://www.toyoukeya.co.jp/ab%20tofu.htm|script-title=ja:京豆腐について|accessdate=2013-11-25|publisher={{lang|ja|とよけ屋}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20130523030654/http://www.toyoukeya.co.jp/ab%20tofu.htm|archivedate=2013-05-23}}</ref>。京都的[[和菓子|和菓子]]也十分著名。京菓子注重季節感，每個月份均有其代表性的京菓子<ref>{{cite web|url=http://kyogashi.co.jp/shiryokan/d-3-3/01index.html|script-title=ja:京菓子辞典|accessdate=2013-11-25|publisher={{lang|ja|京菓子資料館}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20140128103836/http://www.kyogashi.co.jp/shiryokan/d-3-3/01index.html|archivedate=2014-01-28}}</ref>。[[八橋|八橋]]是最為著名的京菓子<ref>{{cite web|url=http://www.yatsuhashi.co.jp/mind/yatsuhashi.html|script-title=ja:京名物 井筒八ッ橋の由来|accessdate=2013-11-25|publisher={{lang|ja|井筒八ッ橋本舗}}|language=ja|deadurl=yes|archiveurl=https://web.archive.org/web/20131202065230/http://www.yatsuhashi.co.jp/mind/yatsuhashi.html|archivedate=2013-12-02}}</ref>。京都盆地有著豐富的地下水資源<ref>{{cite web|url=http://www.kyoto-np.co.jp/kankyo/mizuno_wa/09.html|script-title=ja:地下水—京を支える|accessdate=2013-11-28|publisher={{lang|ja|京都新聞}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20130629004539/http://www.kyoto-np.co.jp/kankyo/mizuno_wa/09.html|archivedate=2013-06-29}}</ref>，優質的地下水使京都成為重要的[[日本清酒|日本酒]]產地<ref>{{cite web|url=http://www.gekkeikan.co.jp/enjoy/industry/transition/transition06.html|script-title=ja:地酒|accessdate=2013-11-25|publisher={{lang|ja|月桂冠}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20131203022555/http://www.gekkeikan.co.jp/enjoy/industry/transition/transition06.html|archivedate=2013-12-03}}</ref>。

=== 博物館 ===
[[Image:Kyoto_National_Museum_2009.jpg|thumb]]
京都是一座[[博物馆|博物馆]]之城，市內有超過200家博物館，數量之多在日本屈指可數<ref>{{cite web|url=http://www.kyohakuren.jp/|script-title=ja:京博連とは|accessdate=2013-11-25|publisher={{lang|ja|京都市内博物館施設連絡協議会}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20131203003359/http://www.kyohakuren.jp/|archivedate=2013-12-03}}</ref>，亦是政令指定都市中人均博物館數量最多的都市<ref name="Miyako Plan">{{cite book |author={{lang|ja|京都市総合企画局政策企画室}} |script-title=ja:《京プラン（京都市基本計画）》 |year=2011 |publisher={{lang|ja|京都市}} |location=京都|language=ja}}</ref>{{rp|58}}。京都市內有多種類型的博物館，其中又以[[京都國立博物館|京都國立博物館]]和[[京都市美術館|京都市美術館]]最為知名。京都國立博物館是日本四大國立博物館之一<ref>{{cite web|url=http://www.kyohaku.go.jp/jp/gaiyou/history.html|script-title=ja:目的と沿革|accessdate=2013-11-25|publisher={{lang|ja|京都国立博物館}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20131203222852/http://www.kyohaku.go.jp/jp/gaiyou/history.html|archivedate=2013-12-03}}</ref>，建設當時因其西洋式的外型曾引發爭論，現在已成為京都知名的洋館建築。博物館收藏有27件日本國寶<ref>{{cite web|url=http://www.kokuhoworld.com/kk32.html|script-title=ja:京都国立博物館|accessdate=2013-11-25|publisher={{lang|ja|国宝を訪ねて}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20131203003023/http://www.kokuhoworld.com/kk32.html|archivedate=2013-12-03}}</ref>。京都市美術館是日本第二大公立美術館<ref>{{cite web|url=http://www.city.kyoto.jp/html/bunshi/kmma/about/index.html|script-title=ja:京都市美術館について|accessdate=2013-11-25|publisher={{lang|ja|京都市美術館}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20131203004629/http://www.city.kyoto.jp/html/bunshi/kmma/about/index.html|archivedate=2013-12-03}}</ref>，展品多為明治時代之後的藝術作品。2011年其參觀者數超過100萬人<ref>{{cite === 媒體 ===
[[京都新聞|京都新聞]]是京都市唯一的一份地方[[報紙|報紙]]，創刊於1879年，1942年改為現名<ref>{{cite web |url=http://www.kyoto-np.co.jp/kp/kyo_np/about/rekishi.php |script-title=ja:京都新聞小史（年表）|accessdate=2013-11-26|publisher={{lang|ja|京都新聞}}|archiveurl=https://web.archive.org/web/20131203023137/http://www.kyoto-np.co.jp/kp/kyo_np/about/rekishi.php |archivedate=2013-12-03 |deadurl=no|language=ja}}</ref>，其發行範圍包括了京都府和滋賀縣<ref>{{cite web|url=http://www.kyoto-np.co.jp/ad/profile/busuu_01.htm|script-title=ja:発行部数|accessdate=2013-11-25|publisher={{lang|ja|京都新聞}}|language=ja|deadurl=yes|archiveurl=https://web.archive.org/web/20131126203823/http://kyoto-np.co.jp/ad/profile/busuu_01.htm|archivedate=2013-11-26}}</ref>。京都新聞的特徵是較多關於宗教、文化、藝術方面以及大學相關的新聞。除了京都新聞之外，日本五大全國性報紙（[[讀賣新聞|讀賣]]、[[朝日新聞|朝日]]、[[每日新聞|每日]]、[[日本經濟新聞|日經]]、[[產經新聞|產經]]）也都有在京都市發行。京都市內也有多家[[出版社|出版社]]，其中又以出版學術類圖書的出版社居多。以京都市為總部的[[电视台|电视台]]有兩家，分別是[[京都放送|京都放送]]（KBS京都）<ref>{{cite web|url=http://www.kbs-kyoto.co.jp/profile/|script-title=ja:会社概要|accessdate=2013-11-25|publisher={{lang|ja|京都放送}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20131111063901/http://www.kbs-kyoto.co.jp/profile/|archivedate=2013-11-11}}</ref>和[[NHK京都放送局|NHK京都放送局]]<ref>{{cite web|url=http://www.soumu.go.jp/soutsu/kinki/2011/schedule/ |script-title=ja:近畿地区の地上デジタルテレビ放送局|accessdate=2013-11-28|publisher={{lang|ja|総務省近畿総合通信局}}|language=ja}}</ref>。

京都是[[日本电影|日本电影]]文化的發祥地。1897年，日本首次[[电影|电影]]上映會就是在京都舉行<ref>{{cite web|url=http://kanko.city.kyoto.lg.jp/detail.php?InforKindCode=10&ManageCode=2122|script-title=ja:日本映画発祥の地（駒札）|accessdate=2013-11-25|publisher={{lang|ja|京都観光ナビ}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20151222113337/http://kanko.city.kyoto.lg.jp/detail.php?InforKindCode=10&ManageCode=2122|archivedate=2015-12-22}}</ref>。1901年，京都首個[[電影院|電影院]]開業<ref name="Kyoto Movie">{{cite web|url=http://www.okeihan.net/navi/kyoto_tsu/tsu201111.php |script-title=ja:京都と映画|accessdate=2013-11-25|publisher={{lang|ja|京阪電車}}|archiveurl=https://web.archive.org/web/20131203020950/http://www.okeihan.net/navi/kyoto_tsu/tsu201111.php |archivedate=2013-12-03 |deadurl=no|language=ja}}</ref>。長期以來，京都都是日本[[時代劇|時代劇]]電影的中心。1921年，{{link-ja|牧野省三|牧野省三}}在等持院附近興建了電影攝影棚，之後京都迅速發展為日本一大電影製作中心<ref name="Eiga">{{cite web|url=http://www.city.kyoto.jp/somu/rekishi/fm/nenpyou/htmlsheet/bunka24.html|script-title=ja:京都の映画産業|accessdate=2013-11-25|publisher={{lang|ja|京都市}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20131027101817/http://www.city.kyoto.jp/somu/rekishi/fm/nenpyou/htmlsheet/bunka24.html|archivedate=2013-10-27}}</ref>。右京區的太秦地區電影攝影棚林立，有「日本[[好莱坞|好莱坞]]」之稱<ref name="Eiga"/>。在1951年獲得威尼斯影展金獅獎的《[[羅生門_(電影)|羅生門]]》就是在京都拍攝<ref name="Kyoto Movie"/>。然而在1960年代之後，隨之[[电视|电视]]的興起，京都的電影產業轉入蕭條，攝影棚相繼關閉。1975年，[[東映|東映]]在京都的部分攝影棚改建為[[東映太秦映畫村|東映太秦映畫村]]並開業，現在已成為京都重要的觀光景點<ref name="Kyoto Movie"/>。

== 教育 ==
[[File:Kyoto_University.jpg|thumb]]是日本國內最頂尖的大學之一]]
京都是西日本的「學都」，在京阪神地區更有「在京都上學、在[[大阪市|大阪]]賺錢、在[[神户市|神戶]]生活」這樣的說法<ref name="Chishi" />{{rp|353}}，可見京都教育的發達程度。1869年，京都市內創建了69所小學，此為日本現代小學教育制度之開始<ref>{{cite web|url=http://www.city.kyoto.lg.jp/kamigyo/page/0000012524.html|script-title=ja:学都・京都の創造|accessdate=2013-11-26|publisher={{lang|ja|上京区}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20131203020327/http://www.city.kyoto.lg.jp/kamigyo/page/0000012524.html|archivedate=2013-12-03}}</ref>。2016年時，京都市內共有119所[[幼稚園|-{zh-hans:幼儿园; zh-hant:幼兒園; zh-tw:幼稚園; zh-hk:幼稚園; zh-mo:幼稚園; zh-sg:幼稚园;}-]]、20所{{link-ja|认定幼儿园|認定こども園}}（{{lang|ja|認定こども園}}）、184所[[小学|小学]]、101所[[初级中学|初中]]和54所[[高級中學|高中]]<ref>{{cite web|url=http://www2.city.kyoto.lg.jp/sogo/toukei/Publish/Analysis/News/085Edu2016.pdf|script-title=ja:平成28年度学校基本調査の集計結果|accessdate=2017-10-27|publisher={{lang|ja|京都市}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20171028145611/http://www2.city.kyoto.lg.jp/sogo/toukei/Publish/Analysis/News/085Edu2016.pdf|archivedate=2017-10-28}}</ref>。

京都市內共有37所[[大學|大學]]。2009年，在京都就讀的大學生多達13萬9237人，在日本所有都市中位列第二位，學生佔總人口的比例則高居政令指定都市首位<ref name="Student Population"/>。1886年帝國大學令頒發之後，日本第二個[[帝國大學|帝國大學]]—京都帝國大學即設在京都。同時民間相繼創辦了眾多私立大學<ref name="Kyoto University">{{cite web|url=http://www.city.kyoto.jp/somu/rekishi/fm/nenpyou/htmlsheet/bunka25.html|script-title=ja:京都の大学|accessdate=2013-11-25|publisher={{lang|ja|京都市}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20130927062022/http://www.city.kyoto.jp/somu/rekishi/fm/nenpyou/htmlsheet/bunka25.html|archivedate=2013-09-27}}</ref>。京都的大學中，歷史最悠久且學術地位最高的是[[京都大学|京都大学]]。京都大學創立於1897年<ref>{{cite web|url=http://www.kyoto-u.ac.jp/ja/profile/intro/history/index.htm/|script-title=ja:大学紹介 > 歷史|accessdate=2013-11-25|publisher={{lang|ja|京都大学}}|language=ja|deadurl=yes|archiveurl=https://web.archive.org/web/20131203002247/http://www.kyoto-u.ac.jp/ja/profile/intro/history/index.htm/|archivedate=2013-12-03}}</ref>，素以自主獨立的傳統和自由的校風聞名<ref name="Kyoto University" />。在18位日本人[[诺贝尔奖|诺贝尔奖]]得獎者中，有7人都畢業於京都大學或在京都大學任教<ref>{{cite web|url=http://www.kyoto-u.ac.jp/ja/profile/intro/honor/award_b/nobel.htm|script-title=ja:日本人のノーベル賞受賞者一覧|accessdate=2013-11-25|publisher={{lang|ja|京都大学}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20131202234655/http://www.kyoto-u.ac.jp/ja/profile/intro/honor/award_b/nobel.htm|archivedate=2013-12-02}}</ref>。而在私立大學中，以[[同志社大學|同志社大學]]<ref>{{cite == 交通 ==
京都是古代日本古代交通網中的中心都市。[[三條大橋|三條大橋]]是[[東海道_(路名)|東海道]]和[[中山道|中山道]]兩大街道的終點<ref>{{cite web|url=http://kanko.city.kyoto.lg.jp/detail.php?InforKindCode=1&ManageCode=3000046|script-title=ja:三条大橋|accessdate=2013-11-26|publisher={{lang|ja|京都観光ナビ}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20131203021125/http://kanko.city.kyoto.lg.jp/detail.php?InforKindCode=1&ManageCode=3000046|archivedate=2013-12-03}}</ref>，京都郊外的伏見也曾是近畿地區重要的內河港口<ref>{{cite web|url=http://kyoto-fushimi-kanko.jp/about.html|script-title=ja:伏見について|accessdate=2013-11-26|publisher={{lang|ja|京都伏見観光連携協議会}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20070423013539/http://www.kyoto-fushimi-kanko.jp/about.html|archivedate=2007-04-23}}</ref>。現在京都主要的聯外交通方式是[[新幹線|新幹線]]和公路。[[東海道新幹線|東海道新幹線]]所有級別的列車均在[[京都站|京都站]]停車<ref>{{cite web|url=http://railway.jr-central.co.jp/station-guide/shinkansen/|script-title=ja:東海道新幹線のご案内|accessdate=2013-11-26|publisher={{lang|ja|JR東海}}|language=ja|deadurl=yes|archiveurl=https://web.archive.org/web/20130923151459/http://railway.jr-central.co.jp/station-guide/shinkansen/|archivedate=2013-09-23}}</ref>。途徑京都的主要公路則有[[名神高速公路|名神高速道路]]、京阪道路（國道1號線的一部分）、第二京阪道路、京都縱貫自動車道、阪神高速8號京都線、國道8號線、國道9號線、國道24號線、國道171號線等。京都市內並無機場，距離京都最近的機場是[[大阪國際機場|大阪國際機場]]和[[关西国际机场|关西国际机场]]。由於觀光客眾多，京都也是日本[[出租車|-{zh-cn:出租车; zh-tw:計程車; zh-hk:的士; zh-sg:德士;}-]]密度較高的都市，數量超過6000輛，京都也因此存在計程車過剩的問題<ref>{{cite web|url=http://wwwtb.mlit.go.jp/kinki/tetsuzuki/taxi/keikaku-kyoutosiiki|script-title=ja:京都市域交通圏におけるタクシー適正化・活性化地域計画|accessdate=2013-11-26|publisher={{lang|ja|近畿運輸局 - 国土交通省}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20160305171806/http://wwwtb.mlit.go.jp/kinki/tetsuzuki/taxi/keikaku-kyoutosiiki|archivedate=2016-03-05}}</ref>。雖然京都聯外交通十分發達，然而市內交通卻因街道狹窄導致堵塞頻發，來訪京都的日本遊客對於京都最為不滿之處亦為其交通狀況<ref name="Kyoto Tourism"/>。

=== 市內鐵路 ===
{{File2|zh-hant=Kyoto railway map.svg|zh-hans=Kyoto railway map2.svg|thumb|right|200px|京都市以及附近地區的鐵路線路圖}}
京都市內鐵路交通發達，除了主要服務長途旅客的新幹線之外，也有多條服務中短途乘客的鐵路路線。[[西日本旅客鐵道|JR西日本]]在京都市內共有四條線路，分別是[[東海道本線|東海道本線]]（又可分為[[琵琶湖線|琵琶湖線]]和[[JR京都線|JR京都線]]）、[[嵯峨野線|嵯峨野線]]、[[湖西線|湖西線]]和[[奈良線|奈良線]]。東海道本線的琵琶湖線是連接京都和滋賀縣的重要鐵路路線，JR京都線則有眾多開往大阪市和神戶市的列車。嵯峨野線<ref>{{cite === 巴士 ===
[[File:Kyoto_City_Bus_200_Ka_1519.jpg|thumb]]
京都市內最早的巴士出現在1928年<ref>{{cite web|url=http://www.city.kyoto.lg.jp/kotsu/page/0000006730.html|script-title=ja:交通局のあゆみ|accessdate=2013-11-26|publisher={{lang|ja|京都市交通局}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20130814083403/http://www.city.kyoto.lg.jp/kotsu/page/0000006730.html|archivedate=2013-08-14}}</ref>，現在巴士已經發展為京都市內另一重要的公共交通方式。京都市內大部分巴士均由[[京都市交通局|京都市交通局]]運營，是為[[京都市營巴士|京都市營巴士]]。京都市營巴士可分為「均一系統路線」和「多區間系統路線」兩種類型。均一系統路線只在均一區間內行駛，不論乘坐距離長短票價一律為230日元<ref>{{cite web|url=http://www.city.kyoto.lg.jp/kotsu/page/0000006679.html|script-title=ja:市バスの運賃について|accessdate=2015-02-09|publisher={{lang|ja|京都市交通局}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20150210201758/http://www.city.kyoto.lg.jp/kotsu/page/0000006679.html|archivedate=2015-02-10}}</ref>。多區間系統路線則在行駛出均一區間後票價會按長度加收費用。京都市營巴士現在共有74條路線。其中急行100、急行101、急行102稱作「洛巴士」（{{lang|ja|洛バス}}）<ref>{{cite web|url=http://www.city.kyoto.lg.jp/kotsu/page/0000009141.html|script-title=ja:洛バスで京都観光|accessdate=2013-11-26|publisher={{lang|ja|京都市交通局}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20130915060810/http://www.city.kyoto.lg.jp/kotsu/page/0000009141.html|archivedate=2013-09-15}}</ref>，車體採用特別塗裝，途徑京都主要觀光地且停站較少，主要服務觀光客。京都市營巴士中還有10條路線是環線運行。此外京都市營巴士還有僅於週末及假日白天運行的100日元環行巴士<ref>{{cite web|url=http://www.city.kyoto.lg.jp/kotsu/page/0000006396.html|script-title=ja:100円循環バスのご案内|accessdate=2013-11-26|publisher={{lang|ja|京都市交通局}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20130819165205/http://www.city.kyoto.lg.jp/kotsu/page/0000006396.html|archivedate=2013-08-19}}</ref>和只在夜間運行的夜巴士（{{lang|ja|よるバス}}）<ref>{{cite web|url=http://www.kyoto-lab.jp/nightbus/nightbus.html|script-title=ja:京都よるバス|accessdate=2013-11-26|publisher={{lang|ja|京都まちづくり交通研究所}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20131006052525/http://www.kyoto-lab.jp/nightbus/nightbus.html|archivedate=2013-10-06}}</ref>等特殊類型的巴士。除了京都市營巴士之外，京都還有京都巴士、京阪巴士、西日本JR巴士、阪急巴士、八坂巴士、近鐵巴士等公司運營的巴士。
