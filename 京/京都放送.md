'''京都放送'''（{{jpn|j=株式会社京都放送|hg=きょうとほうそう|rm=Kyōto Hōsō|f=y}}，英文名稱：Kyoto Broadcasting System Company Limited）是[[日本|日本]]的一家以[[京都府|京都府]]和[[滋賀縣|滋賀縣]]為放送地域的中波[[廣播電台|廣播電台]]，以及以京都府為播出地區的[[電視台|電視台]]。簡稱'''[[KBS|KBS]]'''，播出電視及廣播節目時經常使用「'''KBS京都'''」這一稱呼，但滋賀放送局播出的節目有時會使用「'''KBS滋賀'''」這一名稱。現在的「KBS京都」這一名稱是在1981年開播30週年時制定。京都放送的總部位於京都御所{{link-ja|蛤御門|蛤御門}}對面。

京都放送開播於1951年，最初是一家廣播電台，當時的公司名稱是「{{lang|ja|京都放送株式会社}}」（和現在相反）。1964年10月，京都放送改名為'''近畿放送'''。1969年4月開始播出電視節目。1994年9月22日，京都放送申請《會社更生法》，實際上倒閉，但並未停止播出節目。1995年10月，京都放送變更公司名為「{{lang|ja|株式会社京都放送}}」至今。2006年4月，京都放送啟用廣播、電視共通[[吉祥物|吉祥物]]「{{lang|ja|カモン・ナス}}」（以賀茂茄子為概念製作），之前廣播使用「{{lang|ja|ラジタン}}」（黃色男童）和「{{lang|ja|ラジリン}}」（粉色女童）為吉祥物。京都放送廣播和電視的共通體育主題音樂是{{link-en|弗朗茨·馮·布隆|Franz von Blon}}作曲的「歐洲萬歲（Heil Europa Marsch）」。

== 沿革 ==
京都放送於1951年4月獲得籌備執照，是日本首家獲得籌備執照的民營廣播電台，並於同年6月正式設立公司。1951年12月24日，京都放送開始播出廣播節目，是日本第五家開播的民營廣播電台<ref name="history">{{cite web |url=http://www.kbs-kyoto.co.jp/profile/profile04.htm |script-title=ja:KBS京都のあゆみ |accessdate=2017-07-18|publisher={{lang|ja|京都放送}} |language=ja}}</ref>。1964年，京都放送變更公司名稱為「近畿放送」（簡稱KBS），並在翌年於[[京都新聞社|京都新聞社]]大樓內完成了新總部<ref name="history"></ref>。1969年4月1日，京都放送開始試播電視節目，並於4月1日正式開始播出電視節目，成為[[近畿地方|近畿地方]]首個[[全國獨立放送協議會|獨立電視台]]。1981年是近畿放送開播30週年，這一年近畿放送搬入了新總部（即現在地址），並將簡稱改為「KBS京都」。

1983年，近畿放送社長兼京都新聞社社長白石英司突然去世，內田和隆繼任近畿放送社長。在遺產繼承過程中，近畿放送和京都新聞社發現白石英司因[[不動產|不動產]]投資失敗而造成約70億日元的賬外債務，成為此後近畿放送倒閉的序曲。此後圍繞賬外債務處理一事，白石英司之妻白石浩子支持京都新聞社，而白石英司之母支持京都放送，京都新聞社和京都放送（及白石家族內部）的紛爭表面化。為解決債務問題，近畿放送依賴[[地下金融|地下金融]]界大佬{{link-ja|許永中|許永中}}融資。此後許永中參與了京都放送增資，並和京都放送新社長福本邦雄關係密切。同時京都新聞社和京都放送也在這一時期至參與重建為止和近畿放送絕交。1989年，福本邦雄為對新設立的子公司「KBS開發」進行融資而將京都放送總部土地和建築以146億日元的價格抵押一事被揭露。1991年，日本戰後最大的經濟案件{{link-ja|伊藤萬事件|イトマン事件}}爆發，許永中是這起案件的核心人物<ref>{{cite web |url=http://toyokeizai.net/articles/-/140379 |script-title=ja:戦後最大の経済事件｢イトマン事件｣の深奥 |accessdate=2017-07-18|publisher={{lang|ja|東洋経済オンライン}} |language=ja}}</ref>；近畿放送也被捲入這起案件，福本邦雄因此辭職。此後近畿放送經營狀況惡化。1994年，近畿放送工會以欠薪為由向[[京都地方裁判所|京都地方裁判所]]申請適用《會社更生法》，近畿放送事實上倒閉，成為日本首個倒閉的擁有執照的放送業者。但因申請之前數年即展開存續運動，因此節目播出得以持續，避免了吊銷執照的危機。此後近畿放送開始重建，並於1996年改名為「株式會社京都放送」。[[任天堂|任天堂]]、[[華歌爾|華歌爾]]、[[歐姆龍|歐姆龍]]、[[京瓷|京瓷]]等[[京都府|京都府]]的大企業和京都府、[[京都市|京都市]]均參與出資重建。

2005年，京都放送開始播出數位電視。2007年，京都地方裁判所承認京都放送完成更生手續<ref name="history"></ref>。2015年，京都放送完全清償了更生計劃的債務，宣告完成重建<ref>{{cite web |url=http://www.k-fusyoku.jp/merumaga/15melmaga/roudou/10gatu/kbs.htm |script-title=ja:会社再建を名実ともに達成 |accessdate=2017-07-18|publisher={{lang|ja|京都府職労}} |language=ja}}</ref>。

== 廣播 ==
京都放送是[[NRN|NRN]]的加盟電台，在[[近畿地方|近畿地方]]和[[大阪放送|大阪放送]]共同承擔基幹電台的角色。京都放送的廣播部門以05:00為基點進行24小時播出<ref>{{cite web |url=http://www.kbs-kyoto.co.jp/now_on_air/radio.htm |script-title=ja:週間番組表 |accessdate=2017-07-18|publisher={{lang|ja|京都放送}} |language=ja}}</ref>。過去京都放送滋賀放送局曾製作播出部分面向滋賀縣的節目，但現在除了高中棒球比賽之外，各地已統一節目播出內容。舞鶴放送局和福知山放送局過去曾有製作播出新聞節目，但現在和滋賀放送局相同，除了高中棒球比賽之外並無獨自播出的節目內容。京都放送廣播的可收聽範圍基本覆蓋了近畿地方全境<ref>{{cite web |url=http://www.kbs-kyoto.co.jp/contents/radio/area_map/index.htm |script-title=ja:受信エリア |accessdate=2017-07-18|publisher={{lang|ja|京都放送}} |language=ja}}</ref>，覆蓋人口數近2.000萬人<ref>{{cite web |url=http://www.kbs-kyoto.co.jp/contents/sales/radio.htm |script-title=ja:ラジオ |accessdate=2017-07-18|publisher={{lang|ja|京都放送}} |language=ja}}</ref>。

== 電視 ==
京都放送加盟了[[全國獨立放送協議會|全國獨立放送協議會]]，遙控器號碼是5頻道。目前京都放送的電視部門並未實施24小時播出，基本上在平日02:33~02:53之間會收播（週六為27:03（週日03:03）），05:55開始播出節目<ref>{{cite web |url=http://www.kbs-kyoto.co.jp/now_on_air/tv.htm |script-title=ja:週間番組表 |accessdate=2017-07-18|publisher={{lang|ja|京都放送}} |language=ja}}（週一到週六的收播時間皆不同）</ref>。京都放送除了自公司製作的節目之外，也播出日本各大[[電視聯播網|電視聯播網]]在大阪的系列局未播出的節目、外國電視劇等節目。

在[[大阪電視台|大阪電視台]]開播之前，京都放送曾播出較多東京12頻道（現[[東京電視台|東京電視台]]）的節目。2007年，京都放送和神奈川電視台、埼玉電視台、千葉電視台、三重電視台、SUN電視台共同加入了{{link-ja|東名阪Net6|東名阪ネット6}}<ref>{{cite web |url=http://www.tvk-yokohama.com/company/release/070305.pdf |script-title=ja:東名阪ネット６立上げ ～共同制作番組発表～ |accessdate=2017-07-18|publisher={{lang|ja|tvk}} |language=ja}}</ref>。此外京都放送還播出[[東京都會電視台|東京都會電視台]]節目[[5時夢中!|5時夢中!]]<ref>{{cite web |url=http://s.mxtv.jp/company/press/pdf/press2012_420001.pdf |script-title=ja:ＴＯＫＹＯ ＭＸの看板番組 「５時に夢中！」 が関西上陸！４月からサンテレビとＫＢＳ京都で放送開始 |accessdate=2017-07-18|publisher={{lang|ja|TOKYO MX}} |language=ja}}</ref>。京都放送電視節目包括了京都府全府、滋賀縣及大阪府的大部分地區、奈良縣及福井縣的部分地區（部分地區需通過有線電視收看）<ref>{{cite web |url=http://www.kbs-kyoto.co.jp/contents/tv/area_map/ |script-title=ja:受信エリア |accessdate=2017-07-18|publisher={{lang|ja|京都放送}} |language=ja}}</ref>。
