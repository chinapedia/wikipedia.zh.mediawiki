'''京都縱貫自動車道'''（{{lang|ja|京都縦貫自動車道、きょうとじゅうかんじどうしゃどう、KYOTO-JUKAN EXPRESSWAY}}）是[[京都府|京都府]][[宮津市|宮津市]]至同府[[乙訓郡|乙訓郡]][[大山崎町|大山崎町]]，長93.5公里的[[國土交通大臣指定高規格幹線道路（一般國道快速道路）|國土交通大臣指定高規格幹線道路（一般國道快速道路）]]（[[國道478號|國道478號]]）。此外，法律上的京都縱貫自動車道包含京滋繞道（含名神高速公路改建區間）使用的大山崎系統交流道/交流道～久御山交流道間路段。高速公路路線編號與[[鳥取豐岡宮津自動車道|鳥取豐岡宮津自動車道]]（山陰近畿自動車道）、[[山陰自動車道|山陰自動車道]]共同被編為 {{Ja Exp Route Sign|E9}}。

== 概要 ==
南北貫穿於京都府內，強化京都南北合作，振興地方區域，做為高級道路網、京都的外環道路。京都府亦表示本路為「'''京都府的脊梁'''」<ref>[http://www.pref.kyoto.jp/doro/tokuzai.html 道路特定財源制度-京都府ホームページ]</ref>。

2015年7月18日に、[[京丹波和知IC|京丹波和知IC]] - [[丹波IC|丹波IC]]間通車，至此'''全線通車'''。

== 沿革 ==
* 1988年2月17日 - 沓掛IC - 龜岡IC間 一般國道9號'''老之坂龜岡道路'''、龜岡IC - 千代川IC間一般國道9號'''龜岡道路'''通車。當時後者暫定2車線且免費。
* 1992年 - 綾部宮津道路事業化
* 1993年 - 丹波綾部道路事業化
* 1993年4月1日 - 千代川IC - 沓掛IC間改隸一般國道478號起終點方向反轉。
* 1993年9月9日 - 千代川IC - 龜岡IC間收費。
* 1996年4月27日 - 丹波IC - 千代川IC間（'''八木園部道路'''）開通。當時暫定2車線。同時千代川IC - 龜岡IC間4車線化。藉著通車，八木園部道路、龜岡道路、老之坂龜岡道路的事業名成為'''京都丹波道路'''。
* 1997年度 : 綾部JCT - 京丹波和知IC間開工。
* 1998年3月8日 - 舞鶴大江IC - 綾部JCT間通車、連接舞鶴自動車道（今舞鶴若狹自動車道）。當時為接續綾部宮津道路-舞鶴自動車道福知山方面的半JCT。

* 2001年6月17日 : 八木西IC - 千代川IC間4車線化。
* 2002年4月21日 - 園部IC - 八木西IC間が4車線化。
* 2003年3月1日 - 綾部宮津道路 - 舞鶴自動車道舞鶴方面連結匝道開通、綾部JCT成為完整JCT。
* 2003年3月2日 - 宮津天橋立IC - 舞鶴大江IC間開通。
* 2003年3月27日 - 綾部JCT - 綾部安國寺IC間開通。JCT附近設有[[綾部系統交流道|綾部北料金所]]。
* 2005年10月1日 - [[道路關係四公團|道路關係四公團]]民營化。京都丹波道路為[[行政法人|行政法人]]日本高速道路保有・債務返濟機構、管理則交予由西日本高速道路株式會社。
* 2008年8月18日 - 因綾部北料金所拆除工程，綾部JCT-綾部安國寺IC間全面禁止通行（同年9月13日結束）<ref>[http://corp.w-nexco.co.jp/corporate/release/kansai/h20/0731/ 京都縦貫自動車道 綾部JCT-綾部安国寺IC間の全面通行止について] - 西日本高速道路、2008年7月31日</ref>。
* 2008年9月13日 : 綾部安国寺IC - 京丹波和知IC間が開通。
* 2009年6月1日 - 「綾部宮津道路管理事務所」改名「京都府道路公社管理事務所」。
* 2010年6月28日 - 京都丹波道路（丹波IC - 沓掛IC間）開始免費化社會實驗。

* 2011年3月12日 - 直接連接綾部宮津道路的鳥取豐岡宮津自動車道（宮津與謝道路）宮津天橋立IC - 與謝天橋立IC開通。
