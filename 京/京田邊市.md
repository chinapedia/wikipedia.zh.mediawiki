'''京田邊市'''（{{jpn|j=京田辺市|hg=きょうたなべし|rm=Kyōtanabe shi|f=y}}）是位於[[京都府|京都府]]南部的城市，同時與[[大阪府|大阪府]]和[[奈良縣|奈良縣]]接鄰，地處三府縣三角地帶的中心，{{tsl|ja|木津川 (京都府)|木津川 (京都府)|木津川}}由轄區東北側通過，轄區西南側為{{tsl|ja|生駒山地}}，主要市區位於生駒山地和木津川之間的平原區域。

== 歷史 ==
在令制國時代屬於[[山城國|山城國]][[綴喜郡|綴喜郡]]，進入[[江戶時代|江戶時代]]後大部分區域屬於{{Tsl|ja|淀藩}}的領地，另有部分區域屬於幕府或是皇室的天領。明治維新後，先後分屬京都裁判所、京都府、淀縣，直到1871年才被全數併入京都府之管轄。

1889年實施[[町村制|町村制]]，現在的京田邊市範圍在當時分屬：{{tsl|ja|大住村}}、田邊村、{{tsl|ja|草内村|草內村}}、{{tsl|ja|三山木村}}、{{tsl|ja|普賢寺村}}，田邊村先於1906年改制為田邊町，1951年其餘4村一同併入田邊町，形成現在京田邊市的範圍。1997年田邊町改制為市，但由於當時已有[[和歌山縣|和歌山縣]][[田邊市|田邊市]]，因此取用「京都府」的「京」，以「京田邊市」作為改制後的名稱。<ref>{{Cite web| url=http://www.kyotanabe.jp/0000001666.html| title=市の概要| date=2016-12-14| accessdate=2017-09-11| publisher=京田邊市政府| language=ja}}</ref>

== 觀光資源 ==
轄內的{{tsl|ja|酬恩庵}}為15世紀時由著名僧侶[[一休|一休]]所重建，一休最終也在此過世，因此酬恩庵也被稱為一休寺。

== 教育 ==
1980年代位於[[大阪府|大阪府]]、[[京都府|京都府]]、[[奈良縣|奈良縣]]交界處的區域，包括大阪府[[枚方市|枚方市]]、[[交野市|交野市]]、[[四條畷市|四條畷市]]、京都府京田邊市、[[精華町|精華町]]、[[木津川市|木津川市]]、奈良縣[[生駒市|生駒市]]、[[奈良市|奈良市]]被規劃為{{tsl|ja|関西文化学術研究都市|關西文化學術研究都市}}，推動為學術、研究、新產業、新文化的發展重點區域；因此在1980年{{Tsl|ja|学校法人同志社|學校法人同志社}}在此成立{{tsl|ja|同志社国際中学校・高等学校|同志社國際中學校・高等學校}}，1986年同屬同志社的[[同志社大學|同志社大學]]及{{Tsl|ja|同志社女子大学|同志社女子大學}}進一步在此設立{{tsl|ja|同志社大学京田辺キャンパス|同志社大學京田邊校區|京田邊校區}}，2010年同志社又在此購入京都{{tsl|ja|厚生年金休暇センター|厚生年金休暇中心}}的土地舊建築成立多多羅校區，使得此地區成為同志社教育事業主要的據點之一。
