'''京急2100型電動列車'''（{{jpn|j=京急2100形電車|hg=けいきゅう2100がたでんしゃ|rm=Keikyū 2100 gata densha|f=y}}）是[[京濱急行電鐵|京濱急行電鐵]]於1998年3月28日投入服務的優等列車用電車。

==概要==
本系列是作為[[京急本線|京急本線]]、[[久里濱線|久里濱線]]運行的[[快速特急|快特]]列車[[京急2000型電車|2000型]]的後繼車而製造，共有8輛編組10列（80輛）在籍。同時為紀念[[京濱急行電鐵|京濱急行電鐵]]創立100周年，作為朝向21世紀的車輛而給予了「2100」型式稱號。由於本型式的引進，2000型進行了3門化的降格改造，以置換[[京急1000型電車_(初代)|1000型]]和[[京急700型電車_(二代)|700型]]，事實上作為這些的車輛的調換用而製造。

主要是京急線內的[[快速特急|快特]]列車使用，被分類為[[特急型車輛|特急型車輛]]。車身由[[鋁合金|鋁合金]]製造，全長約18m，一側兩門構造。

特記以下文中稱呼，京急本線南側為「浦賀方」、北側為「品川方」、東側為「海側」、西側為「山側」。
編組車號以浦賀方先頭車的車輛編號為代表。同時，「新1000型」為2002年(平成14年)登場的[[京急1000型電車__(二代)|1000型（二代）]]，「1000型」為1959年(昭和34年)登場的[[京急1000型電車_(初代)|1000型（初代）]]，「700型」1967年(昭和42年)登場的[[京急700型電車__(二代)|700型（二代）]]，「600型」1994年(平成6年)登場的[[京急600型電車_(三代)|600型（三代）]]。

2109編組和2133編組的一部分車廂曾試驗安裝電視屏幕，播放[[日本廣播協會|日本廣播協會]]廣播和其他各類廣告。

== 運用 ==
因2100形列車車身較長，而且每邊只有兩道車門，因此無法直通[[淺草線|淺草線]]，服務只限於[[京急本線|京急本線]]及[[久里濱線|久里濱線]]內運行快特班次及京急翼號，以及晨早的特急及機場急行班次。

== 性能 ==
* 設計最高速度：130km/h（營業運行時120km/h）。
* [[起動加速度|起動加速度]]：3.5km/h/s
* 減速度：常用 4.0km/h/s、非常時 4.5km/h/s
*主電動機（出廠時）：西門子製1TB2010-0GC02型[[鼠籠式電動機|鼠籠型三相感應電動機]] （額定輸出（連續值）190kW、輸入電壓1,050V、電流133A、頻率61Hz，額定轉速1,800rpm)
更新後：TDK6163-A三相誘導電動機
* 為了1:1的[[MT比|MT比]]下高速性能和高加速度兩立，採用了高輸出形的三相誘導電動機。

== 電裝品 ==
控制方式採用了[[德國|德國]][[西門子|西門子]]製的 [[可關斷晶閘管|GTO]][[半導體素子|素子]]「SIBAS32」的[[可變電壓可變頻控制器|VVVF逆變器]]，因而在車內的製造標籤裡有「Powered by SIEMENS」的字樣。這同時在[[東日本旅客鐵道|東日本旅客鐵道]](JR 東日本)[[JR東日本E501系電車|E501系]]和[[新幹線E2系電力動車組|E2系]]的一部分也採用。列車在啟動的時候的電動機及逆變器裝置會發出有如[[降B大調|降B大調]]的音階Fa-So-La-♭Ti-Do-Re-♭Me的聲音為其特徴。<ref> 在日本[[鐵道迷|鐵道迷]]之間稱呼為「DoReMeFaInverter」、「唱歌電車」、「DoReMeFa車輛」。參照「[[鐵路車輛、船舶的俗稱|鐵路車輛、船舶的俗稱]]」。</ref>
但是，[[回生制動|回生制動]]的失效速度在8-6km/h前後，對比5-3km/h前後的E501系為高，而在停車的時候聽不見音階。因近年日本製IGBT-VVVF性能及可靠程度較高，京急於2008年起逐步把車輛的變頻器更新為東洋電機IGBT-VVVF。最後一列使用西門子逆變器的2100形為2133編成，於2015年3月進行更新並同時髹上Keikyu Blue Sky Train塗裝。<ref>鉄道ピクトリアル 2017年8月號 京浜急行電鐵特集 </ref>


[[電動空氣壓縮機|電動空氣壓縮機]] (CP)使用了德國[[Knorr-Bremse|Knorr-Bremse]]製的Screw式(SL-22 型式)，新1000型式(到5次車)也採用了相同樣式。
