'''京丰宾馆'''，位于[[北京市|北京市]][[丰台区|丰台区]]西四环南路61号，隶属[[中央军委机关事务管理总局招待局|中央军委机关事务管理总局招待局]]。

==简介==
1981年3月京丰宾馆开业，位于北京西四环丰台镇，占地203亩<ref name=laiwu/>。京丰宾馆主楼建于1970年代末，礼堂建于1980年代初<ref name=fawan>{{Cite web |url=http://www.fawan.com/Article/rx/xc/2011/12/26/143210141007.html# |title=京丰宾馆拆楼改造，法制晚报，2011-12-26 |access-date=2015-02-25 |archive-url=https://web.archive.org/web/20160304191704/http://www.fawan.com/Article/rx/xc/2011/12/26/143210141007.html# |archive-date=2016-03-04 |dead-url=yes }}</ref>。此外，还建有办公楼。后来扩建而成的有藏画馆、宿舍等建筑<ref>[http://www.bjjs.gov.cn/publish/portal0/tab662/info63708.htm “京丰宾馆锅炉房装修改造工程”等四个项目顺利通过我市可再生能源建筑应用示范项目验收，北京市住房和城乡建设委员会，2011-06-20] {{webarchive|url=https://web.archive.org/web/20130325172204/http://www.bjjs.gov.cn/publish/portal0/tab662/info63708.htm |date=2013-03-25 }}</ref>。

京丰宾馆主要承担中央、国家、军队重大会议接待任务及社会团体会议，是[[中国共产党全国代表大会|中国共产党全国代表大会]]、全国“[[两会|两会]]”先进接待单位，是中央国家机关和北京市政府定点会议单位<ref name=laiwu/>。到2011年为止，京丰宾馆是连续24年接待[[全国人大|全国人大]]会议代表、[[全国政协|全国政协]]会议委员的定点饭店<ref name=fawan/>。1989年4月，在京丰宾馆召开了“[[改革十年：中青年理论与实践研讨会|改革十年：中青年理论与实践研讨会]]”，俗称“京丰宾馆会议”<ref>{{cite web|title=仲大军：忆改革开放第一个十年那次理论研讨会|url=http://www.21ccom.net/articles/lsjd/lccz/article_2011050334695.html|date=2011-05-03|publisher=共识网|archiveurl=https://web.archive.org/web/20160304200416/http://www.21ccom.net/articles/lsjd/lccz/article_2011050334695.html|archivedate=2016-03-04|deadurl=yes}}</ref>。

2011年12月，为进行京丰宾馆改造项目，京丰宾馆礼堂拆除<ref name=fawan/>。2012年10月京丰宾馆停业改扩建<ref name=laiwu/>。京丰宾馆改造项目于2013年4月开工，计划2015年12月完工，工程造价5亿元人民币<ref>[http://www.86gc.net/article/html/226441.html 北京京丰宾馆改造项目，中国工程网，2014-09-16]</ref><ref name=fawan/>。

2017年6月京丰宾馆重张开业，营业面积9.05万平方米，拥有客房542间、餐厅14个、会议室18个、千人多功能厅1个，以及健身、棋牌、台球、乒乓球等配套设施。开业后的京丰宾馆作为[[中央军委装备发展部|中央军委装备发展部]]机关临时办公地，承担服务保障职能<ref name=laiwu/>。

2016年[[深化国防和军队改革|深化国防和军队改革]]前，京丰宾馆是[[中国人民解放军总后勤部|中国人民解放军总后勤部]]直属企业。2016年1月，中国人民解放军总后勤部被撤销<ref name=fawan/>。此后京丰宾馆转隶[[中央军委机关事务管理总局招待局|中央军委机关事务管理总局招待局]]<ref name=laiwu>{{Cite web |url = http://www.lwvc.edu.cn/FAQ/n_info.aspx?id=100003416713468&NodeCode=105025&lowerNodeCode=105025003  |title =京丰宾馆招聘简章  |author =  |date =2017-05-20   |publisher = 莱芜职业技术学院 |language =  |accessdate =  |quote =  }}</ref>。
