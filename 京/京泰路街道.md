'''京泰路街道'''，亦称'''海陵工业园区'''，简称'''园区街道'''<ref>{{Cite web |url=http://www.tzhl.gov.cn/tzshlq/zjydt/content/4edded6d-e5c4-4aca-ac88-330ce5343f1d.html |title=陵工业园区（京泰路街道）组织学习新修订的《中国共产党纪律处分条例》|accessdate=2018-10-31 |author=|date=2018-10-30 |work= |publisher=泰州市海陵区人民政府网站|language=Zh-hans }}</ref>，是[[江苏省|江苏省]][[泰州市|泰州市]][[海陵区|海陵区]]下辖的一个[[乡级行政区|乡级行政区]]<ref>{{PRC admin/ref sgc|{{PRC admin/parent|32/12/02/007/000}}}}</ref>。京泰路街道办事处与海陵工业园区管委会为[[两块牌子一套班子|两块牌子、一套班子]]<ref name="a1">{{Cite web |url=http://hllzw.tzhl.gov.cn/sjj/sjgsgg/content/17976137-6d1b-46ac-8a7a-cf253c4ee958.html |title=海陵工业园区原党工委副书记、管委会主任，京泰路街道办事处原主任彭俊同志任期经济责任审计结果公告|accessdate=2018-10-31 |author=|date=2018-02-05 |work= |publisher=泰州市海陵区人民政府网站，来源：审计局|language=Zh-hans }}</ref>。2003年设立海陵工业园区。2006年，批准为[[省级开发区|省级开发区]]。2011年8月，增挂泰州文化创意产业园区牌子<ref name="a2">{{Cite web |url=http://www.tzhl.gov.cn/tzshlq/hlgyyq/hlgyyq.html |title=园区介绍 >>海陵工业园区 |accessdate=2018-10-31 |author=|date= |work= |publisher=泰州市海陵区人民政府网站，来源：工业园区|language=Zh-hans }}</ref>。2018年时，办事处（园区管委会）内设42个机构，共有人员182名<ref name="a1"/>。行政面积约40平方公里，管辖13个社区，总人口约6万人<ref name="a2"/>。
