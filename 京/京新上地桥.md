'''京新上地桥'''，又称“'''上地铁路立交桥'''”，是位于[[中国|中国]][[北京市|北京市]][[海淀区|海淀区]]的一座单塔[[高速公路|高速公路]][[斜拉桥|斜拉桥]]，承载[[京新高速公路|京新高速公路]]（[[北京五环路|北京五环路]]－[[北京六环路|六环路]]段），跨越[[北京地铁13号线|北京地铁13号线]]、[[京包铁路|京包铁路]]、[[京张城际铁路|京张城际铁路]]和两条城市道路，大桥全长2,296米，主塔高99米，主跨长230米，是北京市最高及跨径最长的桥梁。于2010年5月开工建设，2011年11月主体完工<ref name="bchd">{{cite news|url=http://www.bchd.com.cn/detailPage.php?id=30555|title=京新高速上地铁路立交桥主体工程完工|author=王少辉|date=2011-11-29|publisher=北京高速公路网|access-date=2016-08-08|archive-url=https://web.archive.org/web/20160826220703/http://www.bchd.com.cn/detailPage.php?id=30555#|archive-date=2016-08-26|dead-url=yes}}</ref>，2012年1月1日正式通车，有效缓解了[[京藏高速公路|京藏高速公路]]的交通压力<ref name="tmkj">{{cite news|url=http://www.tumukeji.com/article/20445.html|title=京新高速创下北京最大斜拉桥记录|date=2013-01-16|publisher=中国土木科技网}}</ref>。

== 设计 ==
大桥全长2,296米，其中主桥长510米，桥型采用五跨连续单塔单索-{面}-[[预应力钢筋混凝土|预应力钢筋混凝土]]斜拉桥<ref name="gaosu1">{{cite news|url=http://www.cngaosu.com/a/2015/0506/538682.html|title=首届中国最美景观高速路（桥）推荐路线：京新高速上地斜拉桥|date=2015-06-29|publisher=中国高速网}}</ref>，主跨230米，副跨198米；桥面宽35.5米，主塔高99米<ref name="equip">{{cite news|url=http://www.chinaequip.gov.cn/2011-12/28/c_131330556.htm|title=京城最高斜拉桥 上地铁路立交桥建成|author=李伶俐 赵凤华|date=2011-12-28|work=科技日报|publisher=国家重大技术装备网|deadurl=yes|archiveurl=https://web.archive.org/web/20120125141120/http://www.chinaequip.gov.cn/2011-12/28/c_131330556.htm|archivedate=2012-01-25}}</ref>；承台体积为1.2万立方米，厚8米<ref>{{cite news|url=http://www.tumukeji.com/article/20446.html|title=京新高速斜拉桥关键技术达国内领先水平|date=2013-01-16|publisher=中国土木科技网}}</ref>。全桥斜拉索共44对88束，规格∅7×409，其中最大索长222米，最大索重27吨，共重1,411.8吨<ref name="tmkj" />。

== 施工 ==
大桥主塔塔柱采用[[液压|液压]]爬模施工技术<ref name="gaosu1" />。主桥上跨京包铁路、北京地铁13号线和规划中的京张城际铁路，与地铁13号线成15度[[夹角|夹角]]，为了不影响[[铁路|铁路]]和[[地铁|地铁]]的正常运行，加上施工空间不足的因素，该桥最终采用了“顶推法”进行施工，顶推主梁长度212米，顶推距离213米，顶推总重量25,000吨，这是当时中国重量最大的顶推施工作业<ref>{{cite news|url=http://news.cngaosu.com/wenhua/whdt/2015/10/39370.html|title=首都高速最美的桥梁——京新高速上地斜拉桥|date=2015-10-20|publisher=中国高速网}}</ref>。

大桥顶推段的[[箱梁|箱梁]]自重25万千[[牛顿_(单位)|牛]]、梁宽35.5米、悬臂长度63米，这三项技术指标均创造了曲线混凝土箱梁顶推施工的[[世界纪录|世界纪录]]。“两点限位”顶推工法获国家级工法、[[中华人民共和国交通运输部|中华人民共和国交通运输部]]公路工程工法、北京市优秀工程设计一等奖<ref>{{cite web|url=http://finance.china.com.cn/roll/20140619/2480470.shtml|title=中铁设计咨询京新上地斜拉桥获奖|author=徐升桥|date=2014-06-19|publisher=人民铁道报}}</ref>。

京新上地桥获得第十五届[[中国土木工程詹天佑奖|中国土木工程詹天佑奖]]。<ref>{{cite web|url=http://www.crsg.com.cn/crsg/news/photonews/73384/index.html|title=中铁六局主申报首次获奖 京新高速上地立交桥工程问鼎詹天佑奖|publisher=中铁六局|date=2017-12-11|accessdate=2017-12-21}}</ref>

== 命名 ==
2010年9月17日，北京市规划委员会公布了[[北京市人民政府|北京市人民政府]]批准的《京新高速公路五环路至六环路段沿线桥梁名称命名方案》，该桥被正式命名为“京新上地桥”<ref>{{cite news|url=http://www.bjghw.gov.cn/web/static/articles/catalog_10/article_ff8080812ac7af9e012b1edf78090297/ff8080812ac7af9e012b1edf78090297.html|title=京新高速公路五环路至六环路段沿线桥梁名称命名方案确定|date=2010-09-17|publisher=北京市规划委员会|access-date=2016-08-08|archive-url=https://web.archive.org/web/20160820045119/http://www.bjghw.gov.cn/web/static/articles/catalog_10/article_ff8080812ac7af9e012b1edf78090297/ff8080812ac7af9e012b1edf78090297.html#|archive-date=2016-08-20|dead-url=yes}}</ref>。
