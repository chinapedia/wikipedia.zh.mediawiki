'''京王多摩川站'''（{{jpn|j=京王多摩川駅|hg=けいおうたまがわえき|rm=Keiō-tamagawa eki|f=y}}）是一個位於[[東京都|東京都]][[調布市|調布市]]多摩川四丁目，屬於[[京王電鐵|京王電鐵]][[相模原線|相模原線]]的[[鐵路車站|鐵路車站]]。[[車站編號|車站編號]]是'''KO35'''。

== 歷史 ==
開業時是調布站分岐路線的終點站，原以多摩川砂石運輸為主要目的，但在1927年京王閣開幕後遊客漸增。京王閣之後改建為京王閣競輪場，本站為最近的車站。車站構造仍是港灣式構造。

之後相模原線延伸至[[京王讀賣樂園站|京王讀賣樂園]]，本站成為中間站。由於作為新市鎮鐵道的相模原線是從此站開始建設，因此加收票價（加算運賃）從本站起算，本站與調布站之間不適用加收票價。

=== 年表 ===
* 1916年6月1日，作為京王電氣軌道的車站啟用，最初名為'''多摩川原站'''。
* 1937年5月1日，改稱'''京王多摩川站'''
* 1944年5月31日，京王與[[東京急行電鐵|東京急行電鐵]]合併後，京王多摩川站作為京王線的車站運行。
* 1948年6月1日，東急與京王帝都電鐵分拆，京王多摩川站成為京王的車站。
* 1968年12月21日，車站高架化。
* 1971年4月1日，自路段延伸至[[京王讀賣樂園站|京王讀賣樂園站]]開始，調布 - 京王讀賣樂園路段改名為相模原線。
* 2013年2月22日，以KO35作為車站編號。

== 車站構造 ==
本站是[[相對式月台|相對式月台]]2面2線[[高架車站|高架車站]]。剪票口一般僅於北側，但在{{link-ja|京王閣競輪場|京王閣競輪場|京王閣}}[[競輪賽|競輪賽]]與{{link-ja|調布市花火大會|調布市花火大会}}舉行時增設南側臨時剪票口。

各月台有電梯通往北側剪票口，並設有候車室。商店「{{link-ja|京王零售服務|京王リテールサービス|A LoT}}」問於付費區內。

== 車站周邊 ==
車站往南可抵達[[多摩川|多摩川]]堤岸。東邊為櫻堤通，有長達一公里的櫻花樹。車站東側有許多以競輪場觀眾為對象的餐飲店。由於靠近商業活動興盛的[[調布站|調布站]]，車站附近缺乏大型商業設施。2000年代以後，周邊陸續興建起公寓大廈。

== 站名由來 ==
車站開設當時的站名為「'''多摩川原'''站」，此站是以運輸多摩川砂石為目的。現在的「'''京王多摩川'''站」則是以車站附近的「多摩川」命名，由於當時東京橫濱電鐵（現[[東京急行電鐵|東京急行電鐵]]）[[東急東橫線|東橫線]][[多摩川站|多摩川園前站]]舊名同為「多摩川站」，因此在站名前冠上「京王」。多摩川園前站在2000年再次改名為多摩川站。
