'''京南宾馆'''（'''中央军委机关事务管理总局招待局第三招待所'''），位于[[北京市|北京市]][[西城区|西城区]][[白广路|白广路]]头条1号，隶属[[中央军事委员会机关事务管理总局招待局|中央军委机关事务管理总局招待局]]<ref name=caigou>{{Cite web |url =  http://hy.bgpc.gov.cn:82/MEETING_2009/platform/organ/selectTInfHotel.action?ofooter=P1&tibhPk=20161115JJ768825093|title = 中央军委机关事务管理总局招待局第三招待所 |author =  |date =  |publisher = 北京市政府采购会议定点综合查询系统  |language =  |accessdate = 2017-03-28 |quote =  }}</ref>。

==简介==
该宾馆原称[[中国人民解放军总参谋部管理保障部|中国人民解放军总参谋部管理保障部]]招待局第三招待所（京南宾馆），隶属中国人民解放军总参谋部管理保障部招待局<ref>{{Cite web |url = http://www.meetingschina.com/hotel9890.aspx |title = 京南宾馆 |author =  |date =  |publisher = 中国会议产业网 |language =  |accessdate = 2017-03-29 |quote =  }}</ref>。在[[深化国防和军队改革|深化国防和军队改革]]中，2016年1月组建[[中央军委机关事务管理总局|中央军委机关事务管理总局]]，该宾馆转隶中央军委机关事务管理总局招待局，称中央军委机关事务管理总局招待局第三招待所（京南宾馆）<ref name=caigou/>。

2016年，中央军委机关事务管理总局招待局第三招待所（京南宾馆）获评为中央和北京市的北京地区（城六区）2016-2018年度会议定点单位<ref name=zhbiao>{{Cite web |url = http://www.ccgp.gov.cn/cggg/dfgg/zbgg/201701/t20170117_7867097.htm |title = 北京地区（城六区）2016-2018年度会议定点单位协议采购项目中标公告：006中标公告附件2：第二包中标供应商名单 |author =  |date = 2017-01-17 |publisher =中国政府采购网  |language =  |accessdate =  |quote =  }}</ref>。
