'''京春線'''（{{kor|諺=경춘선|漢=京春線|羅=Gyeongchun seon|f=y}}）是[[大韓民國|韓國]]的一條鐵路路線，因連接舊稱京城的[[首爾|首爾]]<ref name="Gyeongui-jp">{{cite web |title=京義線 |url=http://ja.wikipedia.org/wiki/京義線 |publisher=wikipedia |language=ja |accessdate=2013-09-18}}</ref>[[城北站_(首爾)|城北]]<ref name="station-ren">{{cite web |title=首爾地鐵1號線‘城北站’更名為‘光雲大學站’ |url=http://www.visitkorea.or.kr/cht/MA/MA_CH_9_4.jsp?cid=1798337 |publisher=韓國旅遊官方網站 |language=zh |date=2013-02-25 |accessdate=2013-09-13 |archive-url=https://web.archive.org/web/20130928173802/http://www.visitkorea.or.kr/cht/MA/MA_CH_9_4.jsp?cid=1798337 |archive-date=2013-09-28 |dead-url=yes }}</ref>和[[江原道_(南)|江原道]]首府[[春川站|春川市]]而取名「京春線」。全線由[[韓國鐵道公社|韓國鐵道公社]]營運，於2010年全線電氣化[[複線鐵路|複線鐵路]]工程完成後，改為行走[[首爾|首爾]]東[[上鳳站|上鳳站]]至[[春川站|春川站]]，並納入[[韓國首都圈電鐵|韓國首都圈電鐵]]，成為[[首都圈電鐵京春線|首都圈電鐵京春線]]的一部分。

==歷史==
原京春線於1939年7月25日通車，行走[[光云大站|城北站]]（即今光-{云}-大站<ref name="station-ren"/>）與春川之間，共18站，全長87.3公里。<ref name="lineop">{{cite web|title=경영원칙 > 경영공시 > 영업현황 > 영업거리현황 |url=http://info.korail.com/2007/kra/ope/ope08000/w_ope08606.jsp |publisher=Korail |accessdate=2013-09-15 |deadurl=yes |archiveurl=https://web.archive.org/web/20110722192217/http://info.korail.com/2007/kra/ope/ope08000/w_ope08606.jsp |archivedate=2011-07-22 }}</ref>春川是學生喜愛的交遊地點，乘客眾多，為日後京春線鐵道系統提升予以充分理由。<ref name="jd-chug">{{cite web |title=Historic railway line chugs on into history |url=http://unkcs.org/wordpress/2010/12/10/historic-railway-line-chugs-on-into-history/ |date=2010-12-09 |publisher=中央日報 |accessdate=2013-09-15 |deadurl=yes |archiveurl=https://web.archive.org/web/20151208202318/http://unkcs.org/wordpress/2010/12/10/historic-railway-line-chugs-on-into-history/ |archivedate=2015-12-08 }}</ref>

==鐵道更新==
舊京春線全線於1997年至2010年進行改建，目標為可以以每小180公里行駛的雙軌電鐵線。<ref name="jd-chug"/><ref name="irj-future">{{cite web |title=Korea's railways face a bright future |url=http://www.nxtbook.com/nxtbooks/sb/irj0708/index.php?startid=23#/24 |date=2008-07-02 |publisher=International Railway Journal |accessdate=2013-09-15}}</ref><ref name="rgi-korailplans">{{cite web |title=South Korea's growing network |url=http://www.railwaygazette.com/news/single-view/view/10/south-koreas-growing-network.html  |date=2008-09-08 |publisher=Railway Gazette International |accessdate=2013-09-15}}</ref>新線棄[[城北站_(首爾)|城北站]]，改以首爾東[[中浪區|中浪區]][[上鳳站|上鳳站]]為終點，成為韓國首都圈電鐵的一部分，與多條行走首爾的[[韓國首都圈電鐵|地下鐵路]]連接。新京春線於2010年12月21日通車，而原來路線則同日棄止。

==現況==
新線通車後，京春線已成為[[韓國首都圈電鐵|韓國首都圈電鐵]]交通系統之一部分。列車由[[江原道|江原道]]經[[京畿道|京畿道]]進入[[首爾特別市|首爾特別市]]即可於[[忘憂站|忘憂站]]轉乘[[中央線_(韓國)|中央線]]<ref name="KRNA_2010-2">{{cite web|title=망우∼금곡 복선전철|url=http://www.krnetwork.or.kr/service/condition/RouteMapView.do?PA_MAP_ID=12426|publisher=Korea Rail Network Authority|accessdate=2013-09-15|deadurl=yes|archiveurl=https://web.archive.org/web/20120727091246/http://www.krnetwork.or.kr/service/condition/RouteMapView.do?PA_MAP_ID=12426|archivedate=2012-07-27}}</ref>，而終點站的[[上鳳站|上鳳站]]則可轉乘[[中央線_(韓國)|中央線]]及[[首爾地鐵7號線|7號線]]。另外新站及新連接線亦正計畫及進行，例如預計2017年與[[首爾地鐵8號線|8號線]]連接的[[別內站|別內站]]。新系統使[[江原道|江原道]]往[[首爾特別市|首爾特別市]]不同地區除更為便捷{{r|jd-chug}}，路程由兩小時縮短至89分鐘，亦受惠於新的停靠系統，能負載的客量比舊系統增加五倍，使營運成本下降，例如票價祇及從前以[[無窮花號|無窮花號]]行走京春線時的一半{{r|jd-chug}}。

新線以韓國[[現代Rotem|現代Rotem]]生產的列車行駛，最高時速達110公里{{r|rgi-korailplans}}。此外，[[韓國鐵道公社|韓國鐵道公社]]於2012年2月28日增加新型[[ITX-青春|ITX青春列車]]，由[[龍山站_(首爾)|龍山站]]經[[清涼里站|清涼里]]接京春線往[[春川市|春川市]]，最高時速達180公里，使京春線所需通行時間更為縮短例如從[[清涼里站|清涼里]]至春川祇需52到64分鐘、[[龍山站_(首爾)|龍山站]]祇需68到74分鐘{{r|rgi-korailplans|itx}}<ref>{{cite web|title=從首爾市到其他城市|url=http://www.visitseoul.net/cb/article/article.do?_method=view&m=0004007003004&p=07&art_id=53225&lang=cb|publisher=首爾市官方旅遊資訊網站|accessdate=2013-09-15|deadurl=yes|archiveurl=https://web.archive.org/web/20131226004238/http://www.visitseoul.net/cb/article/article.do?_method=view&m=0004007003004&p=07&art_id=53225&lang=cb|archivedate=2013-12-26}}</ref>，來往[[龍山站_(首爾)|龍山站]]至[[春川市|春川市]]的票價為₩3,000至₩6,900，車費可獲折扣優惠{{r|itx}}。

==未來發展==
2010年9月，南韓政府頒布計畫，於2020年把全國路程2小時以下的路線縮短百分之五行駛時間。按此計畫，京春線未來需提速至每小時230公里，甚至有可能看到[[韓國高速鐵道|高速鐵道]]列車行走。較長遠的計畫，則是把高速鐵道延伸至韓國東面的[[束草市|束草市]]。<ref name="plans2020">{{cite web |title=Bullet trains coming to a town near you by 2020 |url=http://www.rapid-rail.com/go/rapidrail/_dbe,news,_auto_7019817.xhtml |date=2010-09-01 |publisher=Rapid Rail |accessdate=2010-09-15 }}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>
