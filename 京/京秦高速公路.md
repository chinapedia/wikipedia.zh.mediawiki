'''京秦高速公路'''，简称'''京秦高速'''是[[国家高速公路网|国家高速公路网]]北京通往河北及东北地区的一条高速公路。由[[北京|北京]]至[[河北|河北]][[秦皇岛|秦皇岛]]，线路基本沿[[京哈高速公路|京哈高速公路]]北侧，西起[[北京五环路|北京五环路]][[平房桥|平房桥]]，东至[[秦皇岛|秦皇岛]]，经[[三河|三河]]、[[蓟县|蓟县]]、[[遵化|遵化]]、[[秦皇岛|秦皇岛]]等地。目前正在建設中<ref>{{Cite web|url=http://www.sohu.com/a/111131902_354650|title=京秦高速将开工!投资28.68亿 施工总承包招标|accessdate=2017-10-15|date=2016-08-18|work=www.sohu.com}}</ref>

== 分省介绍 ==
=== 北京市 ===
*东六环-潮白河大桥（京冀界）段
全长6.3公里，双向6车道，不设主线收费站<ref>{{cite web |author1=北京日报 |title=京秦高速北京段全面开建 东六环至河北三河仅需5分钟|url=http://www.beijing.gov.cn/bmfw/zxts/t1480951.htm |accessdate=2018/8/20}}</ref>，2018年8月20日通车。<ref>{{cite web |author1=北京晚报 |title=首都地区环线高速公路（通州-大兴段）和京秦高速公路北京段今日同步开通 |url=http://www.takefoto.cn/viewnews-1548241.html |accessdate=2018-8-20}}</ref>
=== 天津市 ===
=== 河北省 ===
*冀津连接线
起点位于京秦高速公路三河段一期工程终点，终点位于冀津交界处，全长3.8公里，采用双向六车道标准建设，设计时速120公里。2018年9月30日通车。<ref>{{cite web |author1=河北日报 |title=京秦高速公路冀津连接线正式通车 |url=http://hb.jjj.qq.com/a/20181003/002710.htm}}</ref>
*京冀连接线
起点位于京冀交界潮白河中心，终点为京秦高速公路三河段一期工程起点，全长1.2公里。2018年8月20日开通。
*京秦二期（大安镇津冀界-平安城）
起点位于唐山市玉田县大安镇石河村西（津冀界），终点为唐山遵化市平安城东，东接已建成的京秦高速公路平安城至遵化段和清东陵高速公路。路线全长38.4公里，双向六车道，路基宽度34.5米，设计时速120公里。2018年12月16日通车。<ref>{{cite web |author1=北京日报 |title=刚刚，京秦高速二期通车，北京1.5小时可直达清东陵 |url=https://baijiahao.baidu.com/s?id=1619976550161678982}}</ref><ref>{{cite news|url=http://tangshan.huanbohainews.com.cn/system/2018/12/17/011828138.shtml|title=京秦高速二期正式通车 遵化融入"京津冀一小时交通圈"|author1= 周宏武|author2= 刘春宝|date= 2018-12-17|agency=环渤海新闻网|newspaper=唐山劳动日报|editor=张凌}}</ref>
*京秦三期（遵化-秦皇岛）
已获河北省发展和改革委员会核准批复，预计2020年建成通车。<ref>{{cite news|url=http://bjrb.bjd.com.cn/html/2018-03/09/content_228364.htm|title=京秦高速河北段三期年内开建|author1=丰家卫|author2=张芸波|date=2018年03月09日|newspaper=[[北京日报|北京日报]]| page=15}}</ref>
