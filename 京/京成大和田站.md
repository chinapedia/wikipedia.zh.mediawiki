'''京成大和田站'''（{{jpn|j=京成大和田駅|hg=けいせいおおわだえき|rm=Keisei-Ōwada eki|f=y}}）是位於[[日本|日本]][[千葉縣|千葉縣]][[八千代市|八千代市]]大和田，屬於[[京成電鐵|京成電鐵]][[京成本線|本線]]的[[鐵路車站|鐵路車站]]。車站編號是'''KS30'''。旅客指南採用省略「京成」二字的「大和田」。

==年表==
* 1926年（[[大正|大正]]15年）12月9日 - 車站啟用，當時稱為'''大和田站'''。是八千代市內最古老的車站。
* 1931年（[[昭和|昭和]]6年）11月18日 - 車站改稱'''京成大和田站'''。

* 2018年（[[平成|平成]]30年）3月17日 - 無障礙化工程完工。下行線側臨時檢票口營業時間延長為首班車至末班車，改名為北檢票口（上行線側檢票口改稱南檢票口）<ref>{{Cite web |date=2017-03-16 |url=https://www.seiburailway.jp/railway/eigyo/transfer/2016jyoukou.pdf |title=快適に駅をご利用いただくために 京成大和田駅バリアフリー化工事が完了します 3月17日（土）より供用開始 |format=PDF |publisher=京成電鉄 |accessdate=2018-03-17}}</ref>。

==車站構造==
本站是[[相對式月台|相對式月台]]2面2線[[地面車站|地面車站]]（[[勝田台站|勝田台站]]管理) 。

[[勝田台站|勝田台站]]側有{{link-ja|拖上線|引き上げ線}}，供平日早晨與傍晚時段及周六日早晨時段的本站起迄班次使用。下行線側月台後方有保線機器用的車庫。下行線側月台後方的空間在過去是待避線，但在[[八千代台站|八千代台站]]開業後已拆除。

1996年（[[平成|平成]]8年）改點以前，午間時段平均每40分有一班本站折返的班次，2006年（平成18年）12月10日改點起周六日部分電車改以本站起訖。

本站設有自動驗票閘門。檢票口分為上行線側的南檢票口與下行線側的北檢票口。北檢票口在2018年（平成30年）3月16日以前為臨時檢票口，僅在周一至週六的首班車至早上九點與下午四點至末班車兩個時段，以及假日下午四點至晚上七點開放使用<ref>{{Cite web |date=2018-02-24 |url=http://www.city.yachiyo.chiba.jp/140500/page100106.html |title=京成大和田駅 臨時改札口（北側）の開設時間が拡大されます  |publisher=八千代市 |accessdate=2018-03-17}}</ref>。

八千代台側有站內平交道，並有遮斷桿。

== 車站周邊 ==
* {{link-ja|八千代警察署|八千代警察署}}大和田駅前交番
* {{link-ja|藥的福太郎|くすりの福太郎}}大和田店
* 八千代市立大和田圖書館
* 八千代大和田[[郵便局|郵便局]]
* 鷹之台鄉村俱樂部
* {{link-ja|千葉縣立柏井高等學校|千葉県立柏井高等学校}}
* {{link-ja|八千代市役所|八千代市役所}}
* {{link-ja|長妙寺|長妙寺}} - [[八百屋於七|八百屋於七]]之墓。
* 小板橋時平神社

過去剪票口周邊曾形成包含小型超市等的[[商店街|商店街]]，1966年（昭和41年）八千代市推動「大和田站南地區土地區劃整理事業」<ref>{{Cite web |date=2013-05-15 |url=http://www.city.yachiyo.chiba.jp/content/000030480.pdf |title=事務事業評価表 |format=PDF |publisher=八千代市都市整備部 |accessdate=2015-01-12}}</ref>後，商店退至線路南側。

雖然因換地處分與結算問題而將計畫延長至2019年度（平成31年度），但實際上區劃整理事業已完成。不過，多數商店並未遷回站前，也沒太多新商店開幕，使得站前仍有不少空地。

== 巴士路線 ==
2010年8月2日以後，本站僅停靠{{link-ja|千葉Green Bus|ちばグリーンバス}}深夜急行巴士「新橋站～京成佐倉站」班次<ref>{{Cite web |date=2010-07-24 |url=http://www.chibagreenbus.co.jp/topics/dat/100724055722.htm |title=『深夜急行バス』平成22年8月2日　運行ダイヤ変更のお知らせ  |publisher=ちばグリーンバス |accessdate=2015-03-09}}</ref>。

「'''大和田站'''」巴士站在過去有{{link-ja|東洋巴士|東洋バス}}與電鐵直營時代的[[京成巴士|京成巴士]]停靠，但隨著[[八千代台站|八千代台站]]與[[勝田台站|勝田台站]]、[[東葉高速線|東葉高速線]]開業後陸續廢除。
