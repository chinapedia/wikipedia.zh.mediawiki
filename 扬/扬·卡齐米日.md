'''揚·卡齐米日·瓦薩'''（{{lang-pl|'''Jan Kazimierz Waza'''}}；{{lang-de|'''Johann II. Kasimir Wasa'''}}；{{lang-lt|'''Jonas Kazimieras Vaza'''}}，{{bd|1609年|3月22日|1672年|12月16日|catIdx=J}}），王号'''揚·卡齐米日'''（{{lang-pl|'''Jan Kazimierz'''}}），是[[波兰立陶宛联邦|波兰立陶宛联邦]]时代的[[波兰君主列表|波兰国王]]和[[立陶宛大公|立陶宛大公]]，<ref>{{cite web|publisher=[[UNESCO|UNESCO]]|url=http://www.mb.vu.lt/unesco/rankrasciai/autografai/karalisk/jankaz/emjankaz.htm|title=Historical Collections of the [[Vilnius_University|Vilnius University]] Library - MANUSCRIPTS|accessdate=20 June 2009|deadurl=yes|archiveurl=https://web.archive.org/web/20090707053619/http://www.mb.vu.lt/unesco/rankrasciai/autografai/karalisk/jankaz/emjankaz.htm|archivedate=2009年7月7日}}</ref>[[奥波莱|奥波莱]]和[[上西里西亚|上西里西亚]]的公爵，及名义上的[[瑞典国王|瑞典国王]]，从1648年到1660年都是他的统治时期。他的双亲分别是[[齐格蒙特三世·瓦萨|齐格蒙特三世·瓦萨]]（1566年—1632年）和[[奥地利的康丝坦茨|奥地利的康丝坦茨]]（1588年—1631年）。他的哥哥，和波兰前任统治者是[[瓦迪斯瓦夫四世·瓦萨|瓦迪斯瓦夫四世·瓦萨]]。他与[[神圣罗马帝国|神圣罗马帝国]]的[[哈布斯堡王室|哈布斯堡王室]]统治者有关系，是[[瓦萨王室|瓦萨王室]]的第三位，也是最后一位波兰统治者。他是[[波兰立陶宛联邦|波兰立陶宛联邦]]最后一位真正有繼承皇室血統的統治者。

== 王室头衔 ==
* 拉丁文的官方头衔：''Ioannes Casimirus, Dei Gratia rex Poloniae, magnus dux Lithuaniae, Russie, Prussiae, Masoviae, Samogitiae, Livoniae, Smolenscie, Severiae, Czernichoviaeque; nec non Suecorum, Gothorum, Vandalorumque haereditarius rex, etc.''  

* 中文翻译：“扬·卡齐米日，蒙上帝之恩典，为波兰国王，[[立陶宛|立陶宛]]、[[鲁塞尼亚|鲁塞尼亚]]、[[普鲁士|普鲁士]]、[[马佐夫舍|马佐夫舍]]、[[萨莫吉希亚|萨莫吉希亚]]、[[利沃尼亚|利沃尼亚]]、[[斯摩棱斯克|斯摩棱斯克]]、[[瑟沃尼亚|瑟沃尼亚]]和[[切尔尼戈夫|切尔尼戈夫]]大公；也是[[瑞典人|瑞典人]]、[[哥特人|哥特人]]和[[汪达尔人|汪达尔人]]的世袭国王。”

== 传记 ==
约翰二世·卡齐米日的父亲，[[古斯塔夫一世|古斯塔夫一世]]的孙子，[[齐格蒙特三世|齐格蒙特三世]]在1592年成功从父亲那里继承瑞典王位，仅仅到了1599年就被他的叔叔[[卡尔九世|卡尔九世]]废黜。这导致了[[瓦萨王室|瓦萨王室]]的，意图占有瑞典王位的波兰国王与瑞典国王的长期不和，这也是1600年到1629年的[[波瑞战争|波瑞战争]]的开端。波兰和瑞典在[[三十年战争|三十年战争]]（1618年-1648年）中处在对立的位置上，虽然在这场战争中波兰最大程度地躲开了所有重大的军事行动。

约翰·卡齐米日在很长时间内都处在他哥哥[[瓦迪斯瓦夫四世·瓦萨|瓦迪斯瓦夫四世·瓦萨]]的阴影中。他在波兰贵族（“[[施拉赤塔|施拉赤塔]]”）中沒有什么朋友，并公然支持奥地利，展现出对[[萨尔马提亚主义|波兰文化]]的蔑视。他的时间可以分为无节制的聚会时间和宗教思考时间。他十分遮遮掩掩，也不過问政治，在宫廷中既没有强大的政治基础，也没什么影响力。但是他有军事指挥官的能力，他的才能展现在与[[莫斯科公国|莫斯科公国]]作战的[[斯摩棱斯克战争|斯摩棱斯克战争]]（1633年）。 
从1632年至1635年，瓦迪斯瓦夫四世试图通过协商，让约翰·卡齐米日与[[克里斯蒂娜女王|克里斯蒂娜·奥古斯塔]]（再后来与一位意大利公主）结婚，来提高弟弟的影响力，但不奏效。167年，约翰·卡齐米日接受了一份到[[维也纳|维也纳]]的[[外交任务|外交任务]]，让他陷入了参加[[神圣罗马帝国|神圣罗马帝国]]军队，与法国作战的泥潭。在他的军团在战场上被击败后，他在维也纳宫廷虚度光阴了一年。 

1636年他回到[[波兰立陶宛联邦|波兰立陶宛联邦]]，与巴萝内斯·古尔登滕相爱，但是他与她的爱情遭到国王瓦迪斯瓦夫的阻挠。作为回报，瓦迪斯瓦夫试图让他统治[[库尔兰|库尔兰]]，但却被联邦议会（“[[瑟姆|瑟姆]]”）反对。因为对此感到愤怒，约翰·卡齐米日在1638年来到西班牙成为葡萄牙总督，却被法国代理人俘虏，被[[黎塞留|黎塞留]]主教囚禁到1640年。他后来被[[斯摩棱斯克省_(波兰)|斯摩棱斯克省]]省长[[亚历山大·戈谢斯基|亚历山大·戈谢斯基]]解救。 

1641年，约翰·卡齐米日决定加入[[耶稣会|耶稣会]]。1642年，他再次回到波兰立陶宛联邦，陪同他的姐妹到德意志地区。1643年，他加入耶稣会，对国王瓦迪斯瓦夫的口头谴责表示反对，导致了联邦和教皇之间的外交冲突。约翰·卡齐米日成为了[[红衣主教|红衣主教]]，但在1646年12月，发现自己并不适合宗教生活后，他回到了波兰。1647年10月，他辞去红衣主教的职位，参选波兰国王。他尝试获得[[哈布斯堡王室|哈布斯堡王室]]的支持，与一位奥地利公主结婚。

1648年约翰·卡齐米日通过他在婚后的侄子，成功[[波蘭選王制|当选波兰国王]]。这位瓦萨王室中最后一位波兰国王会受到[[俄波战争_(1654年-1667年)|俄波战争]]（1654年-1667年），和[[大洪水_(戰爭)|后来与瑞典的战争]]的控制，这就是联邦前两位瓦萨王室国王所酿成的苦果。在大洪水之战中，几乎波兰全境都被瑞典人占领，这些瑞典人虽然不能完全守卫住他们的占领区，最后被迫撤军，但也使整个波蘭滿目瘡痍。

[[File:Polish-Lithuanian_Commonwealth_in_1660.PNG|266px]]

1660年，约翰二世·卡齐米日被迫放弃对瑞典王位的要求，承认瑞典对整个[[利沃尼亚|利沃尼亚]]和[[里加|里加]]的主权。

约翰·卡齐米日与他哥哥的遗孀[[卢德维卡·玛丽亚·贡扎加|卢德维卡·玛丽亚·贡扎加]]（{{lang-pl|Maria Ludwika}}）结婚，她是[[重性抑郁障碍|多愁善感]]的国王的主要支持者。玛利亚·卢德维卡死于1667年。 

1668年9月16日，约翰二世·卡齐米日放弃波兰王位，并回到法国，他在那里加入耶稣会，成为[[内维尔|内维尔]]圣马丁修道院的院长。他死于1672年。

== 影响 ==
约翰·卡齐米日没有健在的子嗣。他的所有兄弟姊妹都比他先死，且也未留子嗣，因此他是拥有[[波娜·斯福扎|波娜·斯福扎]]血统的最后一脉。在他死后，[[阿方索二世_(那不勒斯)|那不勒斯的阿方索二世]]的所有合法后代全部死去。他那不勒斯的费兰特和[[布列恩内对耶路撒冷王国的所有权|布列恩内]]的血脉被他的远表兄，[[塔尔蒙德|塔尔蒙德]]和[[塔兰托亲王|塔兰托亲王]]，[[弗雷德里克四世_(那不勒斯)|费代里戈·迪·阿拉戈纳]]（[[费迪南德一世_(那不勒斯)|弗兰特一世]]和[[塔兰托的伊莎贝拉|塔兰托的伊莎贝拉]]的第二子）的总继承人，[[亨利·德·拉·特雷德莫伊莱|亨利·德·拉·特雷德莫伊莱]]继承，这个人也是费代里戈第一位妻子，[[萨伏依的安妮|萨伏依的安妮]]的继承人。

约翰·卡齐米日继承他的哥哥，是[[瑞典的布里德耶特|瑞典的布里德耶特]]派系的族长，约翰的血缘可以追溯至布里德耶特姊妹的长子。在他死后，族长的职位由他的第二代堂姊妹，早已退位的[[克里斯蒂娜·奥古斯塔|克里斯蒂娜·奥古斯塔]]继承。

== 对艺术的赞助 ==
[[File:Rembrandt_Harmensz._van_Rijn_113.jpg|thumb]]
[[File:Rembrandt_Harmensz._van_Rijn_120.jpg|thumb]]
在1650年代的[[大洪水時代|大洪水之战]]中，波兰[[瓦萨王朝|瓦萨王室]]的藏品被掠夺[[华沙|华沙]]的瑞典人和[[勃兰登堡|勃兰登堡]]的日耳曼人所劫掠。<ref name="lileyko">Lileyko Jerzy, ''Vademecum Zamku Warszawskiego'', Warszawa, 1980. ISBN 83-223-1818-9 p. 129</ref>虽然像[[圭多·雷尼|圭多·雷尼]]的《欧罗巴的奸情》之类的一些藏品因藏在[[奥波莱|奥波莱]]而免受劫掠。

约翰二世·卡齐米日对收藏的最大贡献是，他是[[荷兰绘画|荷兰绘画]]的热心收藏家，和[[丹尼尔·舒尔茨|丹尼尔·舒尔茨]]（曾画有[[克里米亚|克里米亚]][[将军_(軍銜)|将军]]之子代代什的肖像画[https://web.archive.org/web/20110830091735/http://swiadectwotestimony.republika.pl/dedesh.html Dedesz Aga]，并曾为报答他父亲在1663年与俄作战，为王室[[捕鹰|捕鹰]]者作画）的赞助人。国王的画作藏品中，主要是在1660年获得的，这些画作又来自于[[阿姆斯特丹|阿姆斯特丹]]画商，[[亨德里克·凡·艾伦比尔赫|亨德里克·凡·艾伦比尔赫]]，和他儿子[[赫里特·凡·艾伦比尔赫|赫里特·凡·艾伦比尔赫]]，其中主要是[[伦勃朗|伦勃朗]]的作品。其中也有[[彼得·保罗·鲁本斯|鲁本斯]]、[[雅各布·乔登斯|乔登斯]]、雷尼、[[圭尔奇诺|圭尔奇诺]]、[[老布勒哲尔|老布勒哲尔]]、[[亚科波·巴萨诺|巴萨诺]]和其他人的画作。<ref name="lileyko" />. 

在约翰·卡齐米日放弃波兰王位时，他将他的大部分藏品带到法国。留在[[华沙|华沙]][[华沙皇家城堡|皇家城堡]]的藏品在大北方战争中被掠夺，或在1720年时被[[萨克森选帝侯|萨克森选帝侯]][[奥古斯特二世_(波兰)|弗雷德里克·奥古斯特一世]]盗走，譬如两幅伦勃朗的画作——《戴着黑贝雷帽的蓄胡者》（1657年，也称为《拉比像》）和《带着由珍珠装饰的帽子的人》（1667年），今天都放置于德国[[德累斯顿|德累斯顿]]的[[绘画大师画廊|绘画大师画廊]]。<ref name="lileyko" />
