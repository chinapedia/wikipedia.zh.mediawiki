'''扬州工业职业技术学院'''，是位于[[江苏省|江苏省]][[扬州市|扬州市]]的一所公办全日制普通专科学校。<ref>{{Cite web |url=http://www.ypi.edu.cn/s/1/t/8/a/813/info.jspy |title=扬州工业职业技术学院简介 |access-date=2010-10-26 |archive-url=https://web.archive.org/web/20110601224852/http://www.ypi.edu.cn/s/1/t/8/a/813/info.jspy |archive-date=2011-06-01 |dead-url=yes }}</ref>

== 历史 ==
*1978年，建立扬州化工学校。
*1981年，建立扬州建筑工程学校，隶属于[[中国核工业总公司|中国核工业总公司]]。
*1992个通过办学水平评估（省部级重点），2000年划转到省教育厅，是省教育厅直管中专校；
*2003年经江苏省教育厅批准，由扬州化工学校、扬州建筑工程学校合并筹建扬州工业职业技术学院；
*2004年7月，江苏省政府发布《省政府关于建立无锡城市职业技术学院等8所高等职业学校的通知》<ref>[http://www.flssw.com/fagui/info/8030696/ 省政府关于建立无锡城市职业技术学院等8所高等职业学校的通知] {{webarchive|url=https://archive.is/20121216062952/http://www.flssw.com/fagui/info/8030696/ |date=2012-12-16 }}</ref>：扬州化工学校与扬州建筑工程学校合并，组建扬州工业职业技术学院，同时撤销扬州化工学校、扬州建筑工程学校建制。
*截至2015年5月，学院占地总面积55.6万平方米、建筑面积30.3万平方米；学校拥有十个院系，37个专业；建有219个实验实训室，231个校外实训基地；拥有教职工600余人。

== 师资力量 ==
* 截至2015年5月，学院有教职员工近600人，拥有一批江苏省“333工  程”培养对象、江苏省教学指导委员会委员、江苏省高校“青蓝工程”中青年学术带头人和优秀青年骨干教师。专任教师中具有高级职称教师占36%，具有硕士及以上学位教师占教师总数的比例达68.6%；专业教师中双师素质比例达87.2%；同时建立了一支校外兼职教师队伍。

== 学院荣誉 ==
*截至2015年5月，学院被授予“江苏省高等学校文明学校”、“江苏省高等学校和谐校园”、“江苏省文明校园”、“江苏省平安校园”、“江苏省大学生创业教育示范校”、“江苏省大学生创业教育示范基地”、“江苏省高校毕业生就业工作先进集体”、“江苏省职业教育先进单位”等荣誉称号。

== 教学建设 ==
*截至2015年5月，学院有有中央财政支持的国家级实训基地2个（石油化工生产技术实训基地、自动化综合控制实训基地），省级高职教育实训基地2个（现代分析测试中心、机械装配制造与维修综合实训基地），省级人才培养模式创新实验基地1个（高职院“专业导师制1+1+1”人才培养模式创新实验基地），省级工程研发中心1个（江苏省生物环境工程技术研究开发中心）。  学院有中央财政支持的高等职业学校提升专业服务能力建设专业2个（石油化工生产技术、钻井技术），省级重点建设专业（群）3个（石油化工生产技术、电气自动化技术、建筑工程技术）、省级特色专业4个（工业分析与检验、应用化工技术、建筑装饰工程技术、电子信息工程技术）。  省级精品课程：化学过程与设备、图形与图像处理技术、数学应用技术、电子线路板设计与制作、化工用泵检修与维护。
