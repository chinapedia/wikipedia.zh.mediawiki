'''乳酸'''（[[IUPAC|IUPAC]][[學名|學名]]：2-羥基丙酸）是一种[[化合物|化合物]]，它在多种[[生物化学|生物化学]]过程中起作用。它是一种[[羧酸|羧酸]]，[[分子式|分子式]]是[[碳|C]]<sub>3</sub>[[氢|H]]<sub>6</sub>[[氧|O]]<sub>3</sub>。它是一个含有[[羟基|羟基]]的羧酸，因此是一个[[α-羟酸|α-羟酸]]（AHA）。在水溶液中它的[[羧基|羧基]]释放出一个[[质子|质子]]，而产生乳酸根离子 - 乳酸質(Lactate) - CH<sub>3</sub>CHOHCOO<sup>−</sup>。

乳酸有[[手性|手性]]，有两个[[旋光异构体|旋光异构体]]。一个被称为<small>L</small>-(+)-乳酸或(''S'')-乳酸，另一个被称为<small>D</small>-(-)-乳酸或(''R'')-乳酸。<small>L</small>-(+)-是在生物学上重要的异构体。

== 历史 ==
1780年[[卡尔·威廉·舍勒|卡尔·威廉·舍勒]]在[[酸奶|-{zh-cn:酸奶; zh-hk:乳酪; zh-tw:優酪乳}-]]中发现了乳酸。1808年[[贝采利乌斯|贝采利乌斯]]发现了肌肉内的乳酸，1873年[[约翰内斯·威利森努斯|约翰内斯·威利森努斯]]澄清了其结构。1895年[[勃林格殷格翰公司|勃林格殷格翰公司]]发明了使用细菌制造乳酸的方法，从而开始了工业化的生物制造技术。

== 自然来源 ==
左旋的乳酸在[[汗|汗]]、血、肌肉、[[肾|肾]]和[[胆|胆]]中出现。混合的乳酸来自酸奶制品、番茄汁、[[啤酒|啤酒]]、[[鸦片|鸦片]]和其它高等[[植物|植物]]。

== 生物学 ==
在[[发酵|发酵]]过程中[[乳酸脱氢酶|乳酸脱氢酶]]将[[丙酮酸|丙酮酸]]转换为左旋乳酸。在一般的[[新陈代谢|新陈代谢]]和[[运动|运动]]中乳酸不断被产生，但是其浓度一般不会上升。只有在乳酸产生过程加快，乳酸无法被及时运走时其浓度才会提高。乳酸运输速度由一系列因素影响，其中包括单羧基转运体、乳酸脱氢酶的浓度和异构体形式、组织的氧化能力。一般来说[[血液|血液]]中的乳酸浓度在不运动时为1-2 mmol/L，在强烈运动时可以上升到20mmol/L。

一般来说当组织的能量无法通过[[有氧呼吸|有氧呼吸]]得以满足，组织无法获得足够的[[氧|氧]]或者无法足够快地处理氧的情况下乳酸的浓度会上升。在这种情况下[[丙酮酸脱氢酶|丙酮酸脱氢酶]]无法及时将丙酮酸转换为[[乙酰辅酶A|乙酰辅酶A]]，丙酮酸开始堆积。在这种情况下假如乳酸脱氢酶不将丙酮酸还原为乳酸的话[[糖酵解|糖酵解]]过程和[[三磷酸腺苷|三磷酸腺苷]]的生产会获得抑制。产生乳酸的过程为：
:丙酮酸 + NADH + H<sup>+</sup> → 乳酸 + NAD<sup>+</sup>

这个过程的意义在于重建糖酵解所需要的[[烟酰腺嘌呤二核苷酸|烟酰腺嘌呤二核苷酸]]（NAD<sup>+</sup>）来保持三磷酸腺苷的生产。

在氧气充足的[[肌肉|肌肉]][[细胞|细胞]]中乳酸可以被[[氧化|氧化]]为丙酮酸，然后直接用来作为[[三羧酸循环|三羧酸循环]]的燃料。它也可以在肝脏内[[糖异生|糖异生]]的过程中通过[[科里循环|科里循环]]转化为[[葡萄糖|葡萄糖]]。

[[乳杆菌属|乳杆菌属]]的[[细菌|细菌]]也可以进行乳酸发酵。这些细菌可以生活在[[口|口]]内，它们产生的乳酸是导致[[齲齒|齲齒]]的原因。

在[[医学|医学]]裡乳酸常被用在[[乳酸林格氏液|乳酸林格氏液]]中。这是一种与人的血液[[等张|等张]]的[[氯化钠|氯化钠]]、[[氯化钾|氯化钾]]和乳酸在[[蒸馏水|蒸馏水]]中的溶液。在[[损伤|损伤]]、[[手术|手术]]或[[烧伤|烧伤]]失血后常使用乳酸林格氏液来补充失血。

== 运动 ==
在强烈运动的过程中人体需要大量能量。这时人体内乳酸質的生产比组织移走乳酸质的速度高，组织内的乳酸质浓度提高。这个过程保障NAD<sup>+</sup>的再生和运动的继续。不像一般错误的描述，乳酸质浓度的上升本身并不导致[[酸中毒|酸中毒]]，它也不是肌肉酸痛的原因。在人体内乳酸质无法释放质子，因此没有酸性。对人体内[[糖酵解途径|糖酵解途径]]的分析证明这个过程不会导致酸中毒。

强烈运动时造成的酸中毒有另一个原因。在三磷酸腺苷被分裂释放能量時它释放一个质子，这些质子是导致酸中毒的原因。在强烈运动时有氧新陈代谢(Aerobic Metabolism)无法保障三磷酸腺苷的生产，因此无氧新陈代谢(Anaerobic Metabolism)开始。这个过程可以产生大量三磷酸腺苷，这些三磷酸腺苷在分解时释放大量质子，降低组织内的pH值，造成酸中毒。这也许是强烈运动过程中肌肉酸痛的众多原因之一。

== 化妆品工业 ==
在化妆品工业中，乳酸常用来取代[[乙醇酸|乙醇酸]]，因为它比乙醇酸的刺激性要小。它據稱具有减缓[[皺紋|皺紋]]、减轻阳光造成的伤害、加强[[皮膚|皮膚]]结构、肤色和提高皮肤的形态等等的效果。

使用乳酸作為[[化妝品|化妝用品]]時要進行額外防範，因为乳酸可以提高對於陽光中[[紫外線|紫外線]]的敏感度。

== 食品工业 ==
在食品工业中乳酸被用来调节食品的[[酸度|酸度]]。食品中的乳酸可以是通过[[乳糖|乳糖]][[发酵|发酵]]获得的，但大多数是通过细菌对[[淀粉|淀粉]]的发酵获得的。

== 包裝工業==
乳酸為製造聚乳酸(Poly lactic acid)的單體，聚乳酸目前被廣泛地應用，由於其具有生物可分解性質。
