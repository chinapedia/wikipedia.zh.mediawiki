'''乳房疼痛'''，是一種常見而成因多的[[乳房|乳房]]症狀<ref name=hospital>{{cite web |title=乳房疼痛可能是乳癌？～乳房疼痛的原因～ |url=http://www.tahsda.org.tw/newsletters/?p=1927 |website=臺安醫院 |accessdate=2015-10-17 |archiveurl=https://archive.is/20151017135234/http://www.tahsda.org.tw/newsletters/?p=1927 |archivedate=2015-10-17 |deadurl=yes }}</ref>，泛指任何在乳房出現的不適或[[疼痛|疼痛]]<ref name=causes>{{cite web |title=Breast pain |url=https://www.nlm.nih.gov/medlineplus/ency/article/003152.htm |website=U.S. National Library of Medicine |accessdate=2015-10-17 |archiveurl=https://archive.is/20120805080233/http://www.nlm.nih.gov/medlineplus/ency/article/003152.htm |archivedate=2012-08-05 |deadurl=no }}</ref>。

== 症狀 ==
出現乳房疼痛症狀的患者可能會感覺乳房沉甸甸、酸痛、刺痛或是遭到燃燒，這些痛楚有機會蔓延至[[腋窩|腋窩]]以至[[手臂|手臂]]；乳房疼痛一般只會是輕微的痛楚，但也有可能是嚴重和長久的症狀，影響患者[[心理健康|心理健康]]<ref name=mental>{{cite web |title=Breast pain |url=http://www.nhs.uk/conditions/breastpaincyclical/Pages/Introduction.aspx |website=NHS Choices |accessdate=2015-10-17 |archiveurl=https://archive.is/20151017151944/http://www.nhs.uk/conditions/breastpaincyclical/Pages/Introduction.aspx |archivedate=2015-10-17 |deadurl=no }}</ref>。

== 成因 ==
乳房疼痛的成因包括[[青春期|青春期]]（乳房發育）、[[更年期|更年期]]、[[懷孕|懷孕]]、[[月經|月經]]、[[創傷|創傷]]、[[胸罩|胸罩]]不適合、乳房體積過大、{{link-en|乳腺炎|Mastitis}}、{{link-en|乳房纖維囊腫變化|Fibrocystic breast changes}}、服用[[藥物|藥物]]等<ref name=hospital /><ref name=causes />。雖然乳房出現[[腫瘤|腫瘤]]可能是乳房疼痛的成因，但學者認為超過90%的乳房疼痛和[[乳癌|乳癌]]沒有關係，只有大約7%的乳癌患者會出現乳房疼痛的症狀<ref name=hospital /><ref>{{cite web |author1=趙世晃 |title=乳房疼痛的知識 |url=http://www.pain-manage.org.tw/normal/p54.htm |website=台灣疼痛治療資訊網 |accessdate=2015-10-17 |archiveurl=https://archive.is/20151017140701/http://www.pain-manage.org.tw/normal/p54.htm |archivedate=2015-10-17 |deadurl=no }}</ref>。

男性和女性均有可能出現與青春期有關的乳房疼痛<ref name=male>{{cite web |title=Breast Pain |url=http://www.nationalbreastcancer.org/breast-pain |website=National Breast Cancer Foundation |accessdate=2015-10-17 |archiveurl=https://archive.is/20151017155231/http://www.nationalbreastcancer.org/breast-pain |archivedate=2015-10-17 |deadurl=no }}</ref>。與月經有關的乳房疼痛被稱為周期性乳房疼痛，一般是源於月經前的[[激素|激素]]水平轉變<ref name=mental />；女性在遇到巨大[[緊迫|壓力]]後的月經可能會出現暫停、流量增加、提早、延後等現象，而乳房疼痛亦可能會增加和改變<ref name=hospital />。其他健康狀況、創傷和藥物引起的乳房疼痛則被稱為非周期性乳房疼痛，大部分個案的成因難以辨別，這些疼痛也可能是乳房以外的創傷引起的<ref name=mental />。可能導致或增加乳房疼痛的藥物包括從[[毛地黃屬|毛地黃屬]]植物提取的藥物、{{link-en|甲基多巴|Methyldopa}}、[[安體舒通|安體舒通]]、部分[[利尿劑|利尿劑]]、{{link-en|羥甲烯龍|Oxymetholone}}和[[氯丙嗪|氯丙嗪]]<ref name=male/>。

== 治療與護理 ==
醫生會為乳房疼痛患者進行檢查（有機會包括[[乳房攝影術|乳房攝影術]]和{{link-en|乳房活體組織切片|Breast biopsy}}），考慮患者的年齡、[[病史|病史]]和乳房疼痛的嚴重性，從而提供治療；醫生可能會建議患者服食[[鎮痛藥|鎮痛藥]]或熱敷有關部位，治療方法則包括改變[[膳食|膳食]]、減少攝取[[咖啡因|咖啡因]]、減少攝取[[鈉|鈉]]、服用[[維生素E|維生素E]]和[[鈣|鈣]]補充劑、服用[[口服避孕藥|口服避孕藥]]、服用[[甲狀腺激素|甲狀腺激素]]補充劑、佩戴有承托力的胸罩等。
