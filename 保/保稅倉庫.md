== 什麼是保稅倉庫 ==
[[Image:Bonded_warehouse.JPG|right]]邊境的保稅倉庫。由羅伯特·魯尼恩所拍攝，時間介於1900年到1920年]]

=== 保稅制度 ===
保稅制度是指運抵[[國境|國境]]的[[進口|進口]]、[[轉口|轉口]]，以及其他受[[海關|海關]]監管的[[貨物|貨物]]，在通關放行前，暫免或延緩課徵[[關稅|關稅]]的[[制度|制度]]。<ref>{{Cite web |url = http://fund.bot.com.tw/z/glossary/glexp_4849.djhtm|title = 保稅制度 |author =  |date =  |publisher = [[台灣銀行|台灣銀行]] |language = zh-tw |accessdate = 2015-08-21 |quote = 保稅貨物因未完成通過手續，故[[徵稅|徵稅]]與否須視該貨物決定進口或復出口而定。 }}</ref>

保稅制度始創於[[英國|英國]]，這種海關監管制度允許企業暫時免交稅賦，有利於貿易商降低貿易[[成本|成本]]，提高[[經濟|經濟]]效益。[[改革|改革]]開放後，中國為了適應對外[[貿易|貿易]]發展的需要，從80年代開始，陸續修訂頒佈了對來料加工、進料加工、保稅倉庫、保稅區等一系列管理辦法和規定。經過一段時期的實踐，已逐步建立了以保稅加工、保稅倉儲、區域保稅為主要內容的具有中國特色的保稅制度。<ref>{{Cite web |url = http://info.hktdc.com/shippers/vol24_6/vol24_6_chi12_01.htm |title = 中國的保稅制度 |author = 連蓮 |date =  |publisher = 拓思（香港）有限公司 |language = zh-hk |accessdate = 2015-08-26 |quote = 另外，保稅制度還帶有過程監管的特點。由於保稅貨物從進口到複運出口或內銷補稅往往需要在一段時間內進行儲存、加工、裝配，在這段時間中，保稅制度對保稅貨物的管理是過程監管。這個過程的監管除了包括前期的申報、查驗、放行管理外，還包括合同的前期備案和海關下廠檢查保稅貨物的儲存、使用等情況以及後期核銷。 }}</ref>

[[中華民國|中華民國]]《營業稅法》第7條第8、9款<ref>{{zh-tw}} [http://www.acc.ncku.edu.tw/files/%E7%87%9F%E6%A5%AD%E7%A8%85%E6%B3%95.pdf 營業稅法 ] {{Wayback|url=http://www.acc.ncku.edu.tw/files/%E7%87%9F%E6%A5%AD%E7%A8%85%E6%B3%95.pdf |date=20150824065133 }}</ref>載：「八、保稅區營業人銷售與課稅區營業人未輸往課稅區而直接出口之貨物。九、保稅區營業人銷售與課稅區營業人存入自由港區事業或海關管理之保稅倉庫、物流中心以供外銷之貨物。」而《加值型及非加值型營業稅法施行細則》第7-1條提到：「本法第七條第四款所稱供營運之貨物或勞務，指供經核准在保稅區內從事保稅貨物之貿易、倉儲、物流、貨櫃（物）之集散、轉口、轉運、承攬運送、報關服務、組裝、重整、包裝、修理、裝配、加工、製造、檢驗、測試、展覽、技術服務及其他經核准經營業務所使用，或供外銷使用之貨物或勞務。」<ref>{{Cite web |url = http://law.moj.gov.tw/LawClass/LawAll.aspx?PCode=G0340081 |title = 加值型及非加值型營業稅法施行細則 |author = 中華民國財政部賦稅目 |date = 2014-05-02 |publisher = 全國法規資料庫 |language = zh-tw |accessdate = 2015-08-23 |quote = 本法第七條第八款所稱課稅區，指中華民國境內保稅區以外之其他地區。本法第五條所稱保稅貨物及第七條第八款、第九款所稱貨物，指經保稅區營業人登列於經海關驗印之有關帳冊或以電腦處理之帳冊，以備監管海關查核之貨物。 }}</ref>

根據《海關法》對「保稅貨物」的定義是：「經海關批准未辦理納稅手續進境，在境內儲存、加工、裝配後，復運出境的貨物。」《海關法》第43條規定：「經海關批准暫時進口或暫時出口的貨物，以及特准進口的保稅貨物，在貨物收、發貨人向海關繳納相當於稅款的保證金或者提供擔保後，准予暫時免納關稅。」保稅貨物未辦理納稅手續進境，屬於暫時免納，而不是免稅，待貨物最終流向確定後，海關再決定[[徵稅|徵稅]]或免稅。從[[法律|法律]]上講，保稅貨物未按一般貨物辦理進口和納稅手續，因此，保稅貨物必須以原狀或加工後產品復運出境，這既是海關對保稅貨物的監管原則，也是經營廠商必須履行的法律義務。<ref>{{Cite web|url =http://www.ssn.com.tw/eip/front/bin/ptdetail.phtml?Part=3-5-2|title =保稅與保證金台帳制度|author =史芳銘/漢邦管理顧問有限公司會計師|date =|publisher =兩岸經貿服務網|language =zh-tw|accessdate =2015-08-23|quote =大陸海關目前保稅制度的主要形式有兩類：第一類是為國際商品貿易服務的，如保稅倉庫、保稅區、寄售代銷、免稅品商店，其主要功能在於儲存；第二類是為加工製造服務的，如來料加工、進料加工、保稅工廠、保稅集團、保稅區等，其主要功能在於加工與裝配。}}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

==== 發貨中心保稅倉庫 ====
自用保稅倉庫（原發貨中心保稅倉庫）係完全存儲自行進口或自行向國內[[採購|採購]]保稅貨物，或供重整用貨物<ref name="逢甲人115期" />（檢驗、[[測試|測試]]、整理、[[分類|分類]]、[[分割|分割]]、裝配、重裝）之保稅倉庫。<ref>{{Cite web |url = http://taichung.customs.gov.tw/ct.asp?xItem=42377&ctNode=12597 |title = 存儲物流中心保稅貨物售予課稅區，是否可比照自用保稅倉庫(原發貨中心保稅倉庫)按月彙總報關？ |author =  |date = 2009-06-09 |publisher = 財政部關務署臺中關(W1) |language = zh-tw |accessdate = 2015-08-26 |quote = 進口保稅貨物必須報明稅則號列及價格，且其貨物售予課稅區廠商，並非全部均可採按月彙報，如納稅義務人非自用保稅倉庫本身或該項貨品有簽審規定者，不得申請按月彙報，應採逐批申報通關。 }}</ref>2003年7月24日發佈廢止命令，中華民國財政部關稅總局臺總局保字第0920105092號令。<ref>{{Cite web |url = http://www.rootlaw.com.tw/LawArticle.aspx?LawID=A040070061008100-0920724 |title = 海關管理發貨中心保稅倉庫業務執行注意事項 |author =  |date = 2003-07-24 |publisher = 植根法律網 |language = zh-tw |accessdate = 2015-08-26 |quote = 中華民國88年12月2日財政部(八八)臺財關第880820990號函准予備查。中華民國 92年7月24日財政部關稅總局臺總局保字第0920105092號令廢止，廢止17條。 }}</ref>「發貨中心保稅倉庫」廢止，原發貨中心在法規型式上由「自用保稅倉庫」取代。<ref>{{Cite web |url = https://eportal.stust.edu.tw/eshare/EshareFile/2014_5/2014_5_cfc287ff.pdf |title = 物流中心貨物出倉出口 C2、C3 案件不必檢附准單 |author = 劉陽柳 |date = 2014-05-21 |publisher = [[南台科技大學|南台科技大學]] |language = zh-tw |accessdate = 2015-08-27 |quote = 至本關工作手冊放行出口貨物應注意事項第 8 點規定：「一般保稅倉庫貨運出口報單及發貨中心保稅出口報單，應檢附出倉准單。」系針對保稅貨物退運或出倉出口之規定，非指物流中心 D5 出口報單之作業規定，惟因「發貨中心保稅倉庫」已更名為「自用保稅倉庫」為避免錯誤引用，本關將即進行工作手冊之修改。 }}</ref>

==== 公用型 ====
申請設立公用型保稅倉庫的[[企業|企業]]必須具有工商行政管理部門核准的[[倉儲|倉儲]]經營權，註冊[[資本|資本]]最低為300萬元[[人民幣|人民幣]]，擬建倉庫的面積不能少於2000[[平方米|平方米]]。<ref name="公用型保稅倉庫和自用型保稅倉庫的區別">{{Cite web |url = http://www.chinabiz.org.tw/News/GetJournalShow?pid=162&cat_id=174&gid=86&id=1310 |title = 公用型（公共型）保稅倉庫和自用型保稅倉庫有何區別？如何設立？ |author = 袁明仁 |date = 2006年5月 |publisher = 大陸台商經貿網 |language = zh-tw |accessdate = 2015-08-23 |quote = 公用型保稅倉庫和自用型保稅倉庫的主要區別可從五個方面加以說明：所儲存貨物的不同、功能不同、面積標準不同、經營主體的經營範圍要求不同、經營主體的性質不同。例如：取得經貿部門頒發的《[[危險化學品|危險化學品]]經營許可證》、消防部門頒佈發的《易燃易爆化學物品消防安全許可證》等……。 }}</ref>

==== 自用型 ====
在現階段，自用型保稅倉庫主要是指備料保稅倉庫，申請主體必須達到三個基本條件，一是加工[[貿易|貿易]]企業法人，二是年出口額最低為 1000 萬[[美元|美元]]，三是必須是Ａ類或Ｂ類企業，Ｃ類和Ｄ類企業不能經營備料保稅倉庫。主要是為特定加工貿易企業供應[[生產|生產]]自用的生產性[[物料|物料]]、零配件等，不能進行簡單加工服務。<ref name="公用型保稅倉庫和自用型保稅倉庫的區別" />

==== 專用型 ====
專用型保稅倉庫可以是公用型的，也可以是自用型的。專用型保稅倉庫[[專業|專業]]性強，[[行業|行業]]特徵明顯，[[海關|海關]]對經營此類特殊倉庫的要求也比較高。『經營特殊許可商品存儲的，應當持有規定的特殊許可證件』。如經營液體危險品保稅倉庫的，倉儲[[容積|容積]]不能低於 5000 立方米，而且必須符合[[消防|消防]]、[[環保|環保]]、質檢、[[安全|安全]]等方面的規定。<ref name="公用型保稅倉庫和自用型保稅倉庫的區別" />

[[上海市|上海市]][[浦東新區|浦東新區]]，其中隱藏著中國第一家[[藝術品|藝術品]]保稅倉庫，首期建成佔地5000平方米的藝術品保稅[[倉儲|倉儲]]和保稅展示[[區域|區域]]。該保稅倉庫[[負責人|負責人]]胡環中表示，目前通過該中心進出[[海關|海關]]的國內外藝術品都可以不用繳納最低為[[到岸價格|到岸價格]]（'''C'''OST, '''I'''NSURANCE, AND '''F'''REIGHT，[[縮寫|縮寫]]'''CIF'''）貨值的24.02%的藝術品暫時進出口的[[保證金|保證金]]，這大大減輕了[[拍賣公|拍賣公]]司、藝術品公司以及藏家的[[稅收|稅收]]資金壓力。<ref>{{zh-cn}} [http://www.hxsbs.com/page/1/2015-05-06/12/70581430824860667.pdf 揭開藝術品保稅倉庫神秘面紗] {{webarchive|url=https://web.archive.org/web/20160305082839/http://www.hxsbs.com/page/1/2015-05-06/12/70581430824860667.pdf |date=2016-03-05 }}</ref>
