'''保安族'''是[[中国西北地区|中国西北]][[少数民族|少数民族]]，信仰[[伊斯兰教|伊斯兰教]]，主要聚居在[[甘肃省|甘肃省]][[积石山保安族东乡族撒拉族自治县|积石山]]，少数散居青海与新疆，现代社会因工作、婚姻的频繁迁徙，绝大部分省份均有分布。中国有保安族20074人（[[中华人民共和国第五次全国人口普查#民族构成|2015年人口普查]]），其中甘肃省有15,170人，占保安族人口的91.9%。

== 历史 ==
族源是來自中亞的[[色目人|色目人]]簽軍（[[探马赤軍|探马赤軍]]）。保安族的祖先是[[元朝|元朝]]、[[明朝|明朝]]（公元13世紀至17世紀）在青海同仁附近駐墾的蒙古人與色目人，主要信奉伊斯蘭教，到明朝時建立保安城而得名。保安族的名字来自[[明朝|明朝]]时在青海省设立的[[保安营|保安营]]。明朝万历年间，曾在青海省[[同仁县|同仁县]]境内设置了保安营，修建[[保安城|保安城]]，隶属[[河州卫|河州卫]]。後因與信仰[[藏传佛教|藏传佛教]]的藏族爭拗，先走到循化，再到积石山，其地名為'''保安三莊'''，经过历史变迁，1950年按照族民的愿望被命名为保安族。大多数族民是[[农民|农民]]，少数从事[[畜牧业|畜牧业]]或[[手工业|手工业]]。

== 保安族人口分布 ==
保安族主要分布在[[甘肃省|甘肃省]][[积石山保安族东乡族撒拉族自治县|积石山保安族东乡族撒拉族自治县]]（1981年成立）的大河家和刘集两个乡，少数散居于毗邻的[[临夏回族自治州|临夏回族自治州]]、[[青海省|青海省]][[循化县|循化县]]和[[新疆维吾尔自治区|新疆维吾尔自治区]]。

== 语言 ==
[[保安語|保安語]]是保安族的语言，属[[阿尔泰语系|阿尔泰语系]]蒙古语族，通用[[汉语|汉语]]作为交往工具。

== 文化习俗 ==
保安族一直实行土葬、速葬，仪式简单。文化方面保安族书面文学空白，口头文学十分丰富，包括「[[花儿|花儿]]」、号子、小调等，擅长丝竹乐。腰刀“[[保安刀|保安刀]]”，制作技艺高超，精致美观，种类繁多，是保安族特有的手工艺品。
