'''保羅·W·斯帕克斯'''（{{lang-en|Paul W. Sparks}}，{{bd|1971年|10月16日|||Sparks}}）是一名美國演員。他知名於[[HBO|HBO]]電視劇《[[酒私風雲|酒私風雲]]》中飾演黑幫[[米奇·道爾|米奇·道爾]]<ref>{{Cite web |url = http://www.hbo.com/boardwalk-empire/episodes/02/15-a-dangerous-maid/interview/interview-with-paul-sparks.html |title = Interview with Paul Sparks |publisher = Home Box Office |language = en}}</ref><ref>{{Cite web |url = https://www.elle.com/culture/celebrities/news/a23582/paul-sparks-sparrows-dance/ |title = Boardwalk Empire's Paul Sparks on Jazz, Scorsese, and 'Sparrows Dance' |author = Lorraine Cwelich |date = 2013-08-24 |publisher = elle |language = en}}</ref>，也於[[Netflix|Netflix]]的政治劇集《[[紙牌屋|紙牌屋]]》中飾演Thomas Yates，為他提名[[黃金時段艾美獎|黃金時段艾美獎]]。他也在電影《{{link-en|玩叛遊戲|Deception (2008 film)}}》（2008年）、《{{link-en|放學後 (2008年電影)|Afterschool|放學後}}》（2008年）、《{{link-en|消失的人|The Missing Person}}》（2008年）、《[[泥土_(電影)|泥土]]》（2012年）、《{{link-en|行刺甘迺迪|Parkland (film)}}》（2013年）、《{{link-en|偷車人生|Stealing Cars}}》（2015年）、《''{{link-en|Thoroughbreds (2017 film)|Thoroughbreds (2017 film)|Thoroughbreds}}''》（2017年）和《[[大娛樂家_(電影)|大娛樂家]]》（2017年）中演出。

== 早年生活和生涯 ==
斯帕克斯在[[奧克拉荷馬州|奧克拉荷馬州]]的[[勞頓_(俄克拉何馬州)|勞頓]]出生和長大，父親Michael "Mike" Sparks是高中欖球教練，母親Jane（娘家姓Ellis）是小學老師。<ref>{{Cite web |url = https://www.nytimes.com/2010/06/20/theater/20sparks.html?_r=0&mtrref=en.wikipedia.org&gwh=C32FFB3E2E2C0980B0E57B26F69950B4&gwt=pay |title = Embracing a Tragic Masculinity |author = David Rooney |publisher = The New York Times |language = en}}</ref>他曾在他父親執教的Marlow高中上學，並於1990年畢業。在Marlow上學期間，保羅是Marlow Outlaw Speech Team（M.O.S.T.）的一員，在那裡他由Paula和Gary McConnell執教。斯帕克斯獲得獎學金到[[奧克拉荷馬州立大學|奧克拉荷馬州立大學]]攻讀化學，但後來轉到了[[紐約大學|紐約大學]]的{{link-en|蒂施藝術學院|Tisch School of the Arts}}。他於1995年畢業，獲得戲劇{{link-en|美術學士學位|Bachelor of Fine Arts}}。

2010年，斯帕克斯開始在[[HBO|HBO]]電視劇《[[酒私風雲|酒私風雲]]》中飾演常規角色[[米奇·道爾|米奇·道爾]]，該劇是由[[提倫斯·維達|提倫斯·維達]]創作，以及由[[馬田·史高西斯|馬田·史高西斯]]和[[麥克·華堡|麥克·華堡]]執行製作。<ref>{{Cite web |url = http://www.playbill.com/news/article/126203-Hedda-Gablers-Sparks-Signed-for-HBO-Pilot-Boardwalk-Empire |title = Hedda Gabler's Sparks Signed for HBO Pilot "Boardwalk Empire" |author = Adam Hetrick |date = 2009-02-11 |publisher = Playbill |language = en |deadurl = yes |archiveurl = https://web.archive.org/web/20121102150247/http://www.playbill.com/news/article/126203-Hedda-Gablers-Sparks-Signed-for-HBO-Pilot-Boardwalk-Empire |archivedate = 2012-11-02 }}</ref>該電視劇在五季後於2014年結束。<ref>{{Cite web |url = http://connect.hbo.com/events/boardwalk-empire/live-q-paul-sparks/ |title = Q&A with Paul Sparks |publisher = HBO |language = en |deadurl = yes |archiveurl = https://web.archive.org/web/20121027235740/http://connect.hbo.com/events/boardwalk-empire/live-q-paul-sparks/ |archivedate = 2012-10-27 }}</ref>

== 個人生活 ==
斯帕克斯在2005年邂逅女演員{{link-en|安妮·派瑞絲|Annie Parisse}}後便結婚了。<ref>{{Cite web |url = https://www.wsj.com/articles/SB10001424052702304023504577319662121884338 |title = The Theater Of Discomfort |author = Lizzie Simon |date = 2012-04-02 |publisher = The Wall Street Journal |language = en}}</ref>他們的第一個兒子名叫Emmett，於2009年10月出生。這對夫婦也在2014年生下一個女兒<ref>{{Cite web |url = http://www.playbill.com/article/american-horror-story-star-transitions-to-the-stage-with-the-help-of-her-buried-child-cast |title = American Horror Story Star Transitions to the Stage With the Help of Her Buried Child Cast |author = Marcus Scott |date = 2016-03-28 |publisher = Playbill |language = en}}</ref>。斯帕克斯患有[[1型糖尿病|1型糖尿病]]，並且與{{link-en|少年糖尿病研究基金會|JDRF}}合作<ref>{{Cite web |url = http://www.iconvsicon.com/2014/08/29/on-the-rise-paul-sparks-on-his-career-role-in-clark-greggs-trust-me-and-more/ |title = On the Rise: Paul Sparks On His Career, Role In Clark Gregg's 'Trust Me' & More! |date = 2014-08-29 |publisher = Icon vs Icon |language = en}}</ref>。他與演員[[米高·沙朗|米高·沙朗]]是好朋友；兩人一起在許多作品上合作過，包括《{{link-en|消失的人|The Missing Person}}》、《[[泥土_(電影)|泥土]]》、《[[酒私風雲|酒私風雲]]》和《[[通天眼|通天眼]]》。
