'''真空保温瓶（壶、杯）'''，也称为'''保溫瓶（壶、杯）'''，是一种用來保持容器內物質（通常是[[水|水]]）一定[[溫度|溫度]]的真空隔热保温容器，日常生活中常见的一种保溫瓶也称为'''熱水瓶'''、'''暖水瓶'''、'''暖瓶'''、'''暖壶'''、'''茶瓶'''。保溫瓶以減少[[热传导|热传导]]、[[热对流|热对流]]、[[热辐射|热辐射]]來达到保溫目標，主要材料大多是[[不銹鋼|不銹鋼]]或[[玻璃|玻璃]]等材質。

== 起源 ==
1892年，[[苏格兰|苏格兰]]物理学家[[詹姆斯·杜瓦|詹姆斯·杜瓦]]在[[低温物理学|低温物理学]]研究中发明了这种储存液化气体的真空夹层容器——[[杜瓦瓶|杜瓦瓶]]。1904年，[[德意志帝國|德国]][[膳魔師|膳魔師]]（THERMOS）公司制造了第一个商用保温瓶。

== 原理及構成 ==
保溫瓶運用阻絕熱傳原理達到保溫效果。阻絕[[熱傳導|熱傳導]]（直接傳導）與作用，藉由雙層抽[[真空|真空]]結構，內外管分開，阻絕一部分的熱傳導。將內外管之間的空氣抽空，阻絕熱輻射與傳導（氣體之間介質流動變化所產生的熱傳導）。[[不锈鋼|不锈鋼]]具有高強度及耐[[腐蝕|腐蝕]]和低的熱傳導性，可以減少頸部厚度，和縮小口沿部接觸部份，以降低熱傳導而造成的熱能轉移和損失。但不锈鋼和玻璃，因具有高的[[輻射放射率|輻射放射率]]，會藉由輻射作用而造成熱能散失。所以於內管外層鍍上[[銀|銀]]，藉由鏡射反射作用，來降低輻射作用造成的熱量輸送。早期內管外層鍍上銀生產的過程，近來因硝酸銀廢液會對環境有所污染，目前多改比較環保的鍍[[銅|銅]]或裹銅箔和[[鋁|鋁]]箔方式。其中以真空保溫瓶成本最高。

== 塑料件和烤漆要求 ==
外殼的塑料使用[[聚乙烯|聚乙烯]]PE，因為聚乙烯無毒且耐一般高溫。爲了增加[[密合度|密合度]]防止漏水與防止對流現象，一般會使用密封膠圈（PACKING），材料為[[矽利康|矽利康]]（或[[矽氧樹脂|矽氧樹脂]]）硅胶（Silicone）。原材料為無味透明，近來各廠商多數已經改用有顏色（染色規定可用於醫療食品的染料）。烤漆一般規定必須使用不含鉛（Lead）的油漆。
