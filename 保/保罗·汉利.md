'''保羅·杭利'''（'''Paul Hanley'''，{{bd|1977年|11月12日}}）是一位[[澳洲|澳洲]]職業[[網球|網球]]運動員，1997年轉職業，雙打方面的成績較為出色。他的雙親傑（Jay）與茱蒂（Judy）共同擁有網球中心，這也是杭利生涯追求此項運動的原因之一。

[[2005年溫布頓網球錦標賽|2005年溫布頓錦標賽]]，他與[[塔蒂亞娜·彼伊比耶尼斯|塔蒂亞娜·彼伊比耶尼斯]]搭擋，他們闖入混雙決賽，結果負於[[马赫什·布帕蒂|马赫什·布帕蒂]]與[[玛丽·皮尔斯|玛丽·皮尔斯]]組合。他現在的搭擋是[[盧卡斯·德勞希|盧卡斯·德勞希]]。然而，他與[[韋恩·亞瑟斯|韋恩·亞瑟斯]]、[[凱文·烏利耶特|凱文·烏利耶特]]也有良好的合作關係。2006年，杭利被選為澳洲台維斯盃代表隊之一，獲得3勝3勝的結果。杭利的最高雙打排名為第5。[[2011年澳洲網球公開賽|2011年澳洲公開賽]]，他與台灣選手詹詠然闖入[[混雙決賽|混雙決賽]]{{参考|1={{cite web |url=http://tw.nextmedia.com/applenews/article/art_id/33148781/IssueID/20110129 |publisher=蘋果日報 |language =  zh-hant |title=詹詠然混雙 明搶冠 |accessdate=2011-02-25}}}}，雖然逼入第三盤，但是仍然不敵[[丹尼尔·内斯特|丹尼尔·内斯特]]與[[卡塔琳娜·斯雷博特尼克|卡塔琳娜·斯雷博特尼克]]組合{{参考|1={{cite web |url=http://tw.nextmedia.com/applenews/article/art_id/33152700/IssueID/20110131 |publisher=蘋果日報 |language =  zh-hant |title=詹詠然銀了：混雙居亞「冠軍遲早是我的」 |accessdate=2011-02-25}}}}。
