'''保食神'''（{{lang|ja|うけもち の かみ}}）為《[[日本書紀|日本書紀]]》的記述，祂是[[日本神話|日本神話]]裡出現的[[女神|女神]]，別名有'''宇食持命'''、'''宇食保命'''、'''宇氣母智命'''等。
和《[[古事記|古事記]]》記述的[[大宜都比賣|大宜都比賣]]可能是同一人物，但故事細節略有不同。

== 概要 ==
=== 日本書紀之記載 ===
依照《[[日本書紀|日本書紀]]》卷第一[[伊奘諾尊|伊奘諾尊]]和[[伊奘冉尊|伊奘冉尊]]產神之故事裡，第十一種說法記錄[[天照大神|天照大神]]聽聞[[葦原中國|葦原中國]]有'''保食神'''，於是命令[[月讀尊|月夜見尊]]下降拜訪之。保食神乃回頭向著葦原中國，嘴裡吐出[[稻|米飯]]；面向[[海|海]]則嘴裡吐出大小[[魚|魚]]群；面向[[山|山]]則嘴裡吐出大小獸類。一切食物具備，擺設宴席以宴請月夜見尊。後者忿然作色曰：「真是骯髒汙鄙！祢竟然用嘴巴裡吐出的東西招待我！」於是拔劍殺死保食神，然後回去向天照大神稟告一切。天照大神聽完勃然大怒地說：「祢是惡神！我們以後不必再見面。」所以白天就和黑夜分開了。天照大神派遣天熊人（{{lang|ja|あまのくまひと}}）前往探看，保食神確實已死，只見其頭頂化為[[牛|牛]][[馬|馬]]、頭顱長出[[小米|小米]]、[[眉毛|眉毛]]長出[[蠶|蠶]]繭、[[眼睛|眼睛]]長出[[稗|稗]]、[[肚子|肚子]]長出[[稻米|稻米]]、[[生殖系统|下陰]]長出[[小麥|小麥]]、[[大豆|大豆]]和[[紅豆|紅豆]]。天熊人將這些東西帶回獻給天照大神，天照大神歡喜地說：「天下蒼生可以食用這些東西而活命」，乃以粟、稗、麥、豆為旱田種子，以稻爲水田種子。祂又任命天邑君將稻種植於天狹田及長田，到了秋天稻穗飽滿垂首，一片豐收景象甚是快活。又口裏含繭，便能抽絲，從此始有養蠶之道。

== 解說 ==
神名裡的「食（{{lang|ja|うけ}}）」代表[[食物|食物]]，其[[神話|神話]]故事和《[[古事記|古事記]]》記述的[[大宜都比賣|大宜都比賣]]一樣，皆與[[印尼|印尼]]、[[摩鹿加群島|摩鹿加群島]]一帶流傳的[[海奴韋萊|海奴韋萊女神]]（{{lang|en|Hainuwele}}）近似，為食物起源之傳說。也因為如此，一般都將保食神和大宜都比賣視為同一神。此外，代表[[伏見稻荷大社|伏見稻荷大社]]的食物神[[宇迦之御魂神|宇迦之御魂神]]亦常被視為同一神，故全國稻荷神社大半也有配祀保食神。又，故事中提及「頭頂化為[[牛|牛]][[馬|馬]]」，很多分布在東日本的駒形神社除了主祭駒形大神外，也配祀保食神，並將之與馬頭觀音（即[[馬頭明王|馬頭明王]]）視為同一神。

== 信仰 ==
在日本只要是稻荷神社，一般都會配祀保食神，其他的[[神社|神社]]主要計有下述：
* 岩內神社（[[北海道|北海道]][[岩內郡|岩內郡]][[岩內町|岩內町]]）
* 箭弓稻荷神社（[[埼玉縣|埼玉縣]][[東松山市|東松山市]]箭弓町）
* 建穗神社（[[靜岡縣|靜岡縣]][[靜岡市|靜岡市]]）
* 籰繰神社（[[愛知縣|愛知縣]][[豐川市|豐川市]]）
* 飯持神社（[[新潟縣|新潟縣]][[佐渡市|佐渡市]]）
* 保食神社（[[長崎縣|長崎縣]][[壹岐市|壹岐市]]）
