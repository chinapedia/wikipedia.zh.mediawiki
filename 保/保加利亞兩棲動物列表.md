[[File:Bulgaria-geographic_map-en.svg|thumb]]
[[保加利亚|保加利亚]]是[[東南歐|東南歐]]的一個國家，位於[[巴爾幹半島|巴爾幹半島]]。目前已知棲息於此的[[两栖动物|两栖动物]]計有20種，為該國境內脊椎動物中物種多樣性最低的一[[綱|綱]]<ref name="Biserkov 2007 34">{{harvnb|Biserkov|2007|p=34}}</ref>。保加利亞境內的7種[[有尾目|有尾目]]動物全屬[[蠑螈科|蠑螈科]]，包含{{tsl|en|newt|水螈}}及蠑螈；13種[[無尾目|無尾目]]則分屬5科：[[铃蟾科|铃蟾科]]、[[蟾蜍科|蟾蜍科]]、[[樹蟾|樹蟾科]]、[[鋤足蟾科|鋤足蟾科]]，以及[[蛙科|蛙科]]。最新的紀錄種為2005年紀錄的{{tsl|en|northern crested newt|大冠歐螈}}<ref name="biserkov37">{{harvnb|Biserkov|2007|p=37}}</ref>，以及2007年紀錄的{{tsl|en|Macedonian crested newt|馬其頓冠歐螈}}<ref name="naumov">{{cite journal|last=Naumov|first=Borislav|first2=Nikolay|last2=Tzankov|date=2008|title=First record of Triturus macedonicus (Karaman, 1922) (Amphibia: Salamandridae) in Bulgaria|url=http://www.nmnhs.com/downloads/pdfs/historia-naturalis-bulgarica/2008/019-111-114.pdf|journal=Historia Naturalis Bulgarica|publisher=National Museum of Natural History|volume=19|pages=111–114|access-date=2015-07-13|deadurl=yes|archiveurl=https://web.archive.org/web/20140714114809/http://www.nmnhs.com/downloads/pdfs/historia-naturalis-bulgarica/2008/019-111-114.pdf|archivedate=2014-07-14}}</ref>。最常見的物種則包含[[绿蟾蜍|绿蟾蜍]]、[[多彩铃蟾|多彩铃蟾]]，及[[湖侧褶蛙|湖侧褶蛙]]。

保加利亞[[爬虫两栖类学|爬虫两栖类学]]的奠基者是一位教師{{tsl|bg|Васил Ковачев|瓦希爾·科瓦契夫}}，他在19世紀末及20世紀初發表了一系列的論文，並寫了《保加利亞兩棲爬蟲動物志》（{{lang|bg|''Херпетологичната фауна на България''}}）一書<ref name="biserkov28">{{harvnb|Biserkov|2007|p=28}}</ref>。1930年代到1940年代，動物學家{{tsl|en|Ivan Buresh|伊凡·布雷什}}和他的同事約丹·岑可夫（Йордан Ценков）對於保國境內兩棲爬蟲類的多樣性及分布進行了較為深入的研究。到了20世紀後半葉，該領域的研究則以保加利亞兩棲爬蟲學家弗拉迪米爾·貝什科夫（Владимир Бешков）博士為首繼續進行<ref name="biserkov28"/>。

保加利亞具有相當的棲地多樣性。該國全境位於[[古北界|古北界]]界內，境內有六個陸相[[生態區|生態區]]，分別為{{tsl|en|Balkan mixed forests|巴爾幹混合林區}}、{{tsl|en|Rodope montane mixed forests|羅多彼山脈混合林區}}、{{tsl|en|Euxine-Colchic deciduous forests|黑海-柯爾基斯落葉林區}}、{{tsl|en|Aegean and Western Turkey sclerophyllous and mixed forests|愛琴海暨西土耳其硬葉及混合林區}}、{{tsl|en|Forest steppe|森林草原|東歐森林草原}}，以及[[東歐大草原|東歐大草原]]等<ref>{{cite web|url=http://wwf.panda.org/about_our_earth/ecoregions/ecoregion_list/|title=List of Ecoregions|publisher=WWF|accessdate=2016-10-04|archiveurl=https://web.archive.org/web/20161004155629/http://wwf.panda.org/about_our_earth/ecoregions/ecoregion_list/|archivedate=2016-10-04|deadurl=yes}}</ref>。除此之外，保國境內也有豐富的地勢變化。由北到南分別有{{tsl|en|Danubian Plain (Bulgaria)|多瑙河平原 (保加利亞)|多瑙河平原}}、[[巴尔干山脉|巴尔干山脉]]、{{tsl|en|Sub-Balkan valleys|下巴爾幹山谷}}、[[里拉山脉|里拉]]–[[罗多彼山脉|罗多彼山脉]]、{{tsl|en|Upper Thracian Plain|上色雷斯平原}}，以及[[斯特蘭賈山脈|斯特蘭賈山脈]]等等。該國也有相當緻密的水系網絡和河流，但[[多瑙河|多瑙河]]並不包含在其中，因其支流大多較短且水流不豐沛<ref>{{harvnb|Дончев (Donchev)|Каракашев (Karakashev)|2004|p=68}}</ref>。保加利亞的年降水量約為 670 mm ，且雨量隨海拔遞增。境內最乾旱的區域為多瑙河平原東北方的[[多布羅加|多布羅加]]，年降水量僅 450 mm ；雨量最豐沛的地方則為[[奧戈斯塔河|奧戈斯塔河]]位於西巴爾幹山區的上游河谷，降水量可達 2293 mm<ref>{{harvnb|Дончев|Каракашев|2004|pp=57–58}}</ref>。

==== 蝾螈科（Salamandridae） ====
[[蝾螈科|蝾螈科]]的成員遍及陸域及水域，大多分布於[[亚洲|亚洲]]及[[欧洲|欧洲]]，[[北部非洲|北非]]和[[北美洲|北美]]也有部分物種分布。本科大多數物種的皮膚分泌物具有輕微毒性，且生活史相當多元且複雜，許多物種每年會有水棲期及陸棲期。當春天來臨，進入水棲期的蠑螈會長出背鰭及尾鰭<ref>{{cite web|title=水螈和歐螈（蠑螈科）|url=http://digimuse.nmns.edu.tw/Default.aspx?tabid=414&ObjectId=0b00000181e3febf&Domin=z&Field=a1&ContentType=Exhibit&Language=CHI&FieldName=|website=自然與人文數位博物館|publisher=國立自然科學博物館|accessdate=2016-10-04|ref=NMNS|archiveurl=https://web.archive.org/web/20161004032329/http://digimuse.nmns.edu.tw/Default.aspx?tabid=414&ObjectId=0b00000181e3febf&Domin=z&Field=a1&ContentType=Exhibit&Language=CHI&FieldName=|archivedate=2016-10-04|deadurl=yes}}</ref>。本科有21屬109種，其中保加利亞境內有紀錄的共4屬7種<ref>{{cite web |url=http://amphibiaweb.org/lists/Salamandridae.shtml |title=Salamandridae|website=AmphibiaWeb|accessdate=2015-03-18|archiveurl=https://web.archive.org/web/20160601122518/http://amphibiaweb.org/lists/Salamandridae.shtml|archivedate=2016-06-01}}</ref>。

==== 铃蟾科（Bombinatoridae） ====
[[铃蟾科|铃蟾科]]為[[旧大陆|旧大陆]]的蟾蜍支系，體型較為扁平，本屬計有2屬10種，分別為[[巴蟾属|巴蟾属]]（''Barbourula''）及[[铃蟾属|铃蟾属]]（''Bombina''）。保加利亞境內有2種，全屬鈴蟾屬<ref>{{cite web |url=http://amphibiaweb.org/lists/Bombinatoridae.shtml|title=Bombinatoridae|website=AmphibiaWeb|accessdate=2015-03-18}}</ref>。

==== 蟾蜍科（Bufonidae） ====
[[蟾蜍科|蟾蜍科]]分布於除[[澳大利亚|澳洲]]及[[南极洲|南极洲]]以外的各大洲。本科物種的前肢較短，後肢較長以利於跳躍或移動。皮膚較乾，眼後方具有腮腺。本科具有50屬590種，保加利亞境內則有2種，全屬[[蟾蜍屬|蟾蜍屬]]<ref>{{cite web |url=http://amphibiaweb.org/lists/Bufonidae.shtml|title=Bufonidae|website=AmphibiaWeb|accessdate=2015-03-18}}</ref>。

==== 樹蟾科（Hylidae） ====
[[樹蟾|樹蟾科]]是兩棲動物中種類最多樣化的一科，計有51屬951種，分布於全球。大多數物種分布於氣候較濕熱的熱帶帶區，特別是在[[新热带界|新熱帶界]]。樹蟾科的體型有大有小，趾的尖端擁有特化的趾墊，末趾節間有介軟骨，以協助其攀爬。目前歐洲已知僅有[[雨蛙属|雨蛙屬]]一屬分布，該屬共有37種，其中6屬分布於歐洲，保加利亞境內有一種<ref>{{cite web |url=http://amphibiaweb.org/lists/Hylidae.shtml|title=Hylidae|website=AmphibiaWeb|accessdate=2015-03-18}}</ref>。

==== 鋤足蟾科（Pelobatidae） ====
[[鋤足蟾科|鋤足蟾科]]是相當小的一科，僅有1屬4種，分布於歐洲、[[西亞|西亞]]，及西北非。本科成員的四肢較短，體型圓胖，瞳孔垂直。另外還會分泌一種類似[[蒜|蒜]]味的氣味 ，因此在保加利亞語稱之為「切斯諾維察」（{{lang|bg|чесновица}}），意為「蒜蟾」。保加利亞境內共有2種<ref>{{cite web |url=http://amphibiaweb.org/lists/Pelobatidae.shtml|title=Pelobatidae|website=AmphibiaWeb|accessdate=2015-03-18|archiveurl=https://web.archive.org/web/20160323152152/http://amphibiaweb.org/lists/Pelobatidae.shtml|archivedate=2016-03-23}}</ref>。

==== 蛙科（Ranidae） ====
[[蛙科|蛙科]]分布的範圍相當廣，許多種類擁有長而有力的後腿，具有蹼及流線型的身體，以利於跳躍及游泳。本屬有14屬379種，保加利亞境內有2屬6種<ref>{{cite web |url=http://amphibiaweb.org/lists/Pelobatidae.shtml|title=Pelobatidae|website=AmphibiaWeb|accessdate=2015-03-18}}</ref>。
