'''保罗书信'''（{{lang-el|'''Ἐπιστολὲς Παύλου'''}}；[[天主教|天主教]]译稱'''保祿書信'''；[[正教會|東正教]]譯稱'''帕弗羅書信'''；唐朝[[景教|景教]]譯稱'''寶路法王經'''<ref>{{Cite web |url=http://www.vccstm.ca/Otm/issue88/0101cs02.htm |title=福音與中華：中文聖經的問世 |accessdate=2018年7月1日 |author=楊愛程 |newspaper=真理報 |archive-url=https://web.archive.org/web/20180701030523/http://www.vccstm.ca/Otm/issue88/0101cs02.htm |archive-date=2018年7月1日 |dead-url=yes }}</ref>）是[[使徒保羅|使徒保羅]]写给各地[[教會|教會]]就各教會內的情況或是對来信询问教义的人的回信，书信中解释了许多有关[[基督教|基督教]]教义的疑难问题及阐明教理原则，是基督教的重要文件，後在[[第一次君士坦丁堡公會議|第一次君士坦丁堡公會議]]中確認為[[正典|正典]]而被列入[[基督教|基督教]]《[[新约圣经|新约圣经]]》中。

== 作者与写作时间 ==
“保罗书信”中，《[[以弗所书|以弗所书]]》、《[[腓立比书|腓立比书]]》、《[[歌罗西书|歌罗西书]]》和《[[腓利门书|腓利门书]]》被认为是保罗第一次在[[罗马|罗马]]坐监期间所写作，故称为[[监狱书信|监狱书信]]或[[被囚書信|被囚書信]]（{{lang|en|Captivity Letters}}）；而《[[提摩太前書|提摩太前書]]》、《[[提摩太後書|提摩太後書]]》和《[[提多書|提多書]]》的對象是教會牧者，因此亦称为教牧书信（{{lang|en|Pastoral Letters}}）。

而《[[希伯來書|希伯來書]]》作者不明，雖有少數古代學者認為乃保羅所著，但近代學者多不採納此說，因其寫作風格與已確認的保羅書信大不相同。大約在1930年發現的[[蒲草紙抄本_P46|切斯特·貝蒂紙莎草紙抄本]]第2號（P46）中也收錄了《[[希伯來書|希伯來書]]》。 這份紙莎草紙抄本僅在保羅死後一百五十年左右便寫成了。 論到這份抄本， [[英國|英國]][[校勘學|校勘學家]]凱尼恩爵士説： “值得注意的是， 希伯來書放在緊接羅馬書之後的位置（這幾乎是史無前例的），由此表明在這份手抄本寫成的早期日子，沒有人懷疑這封書信的執筆者是保羅<ref>The Story of the Bible,Sir Frederic Kenyon ，1964, page 91.</ref>。”

== 失傳作品 ==
有數篇保羅書信，現已失傳。

* {{Bible|哥林多前書|5|9}}提到，在《哥林多前書》之前，他已經寫過另一封信給[[哥林多教會|哥林多教會]]。
* {{Bible|哥林多後書|2|4}}、{{Bible|哥林多後書|7|8-9}}，提到給[[哥林多教會|哥林多教會]]的第三封信。
* {{Bible|以弗所書|3|3-4}}提到，曾寫給以弗所教會另一封信。
* {{Bible|歌羅西書|4|16}}提到給老底嘉教會的書信。

== 翻譯 ==
保羅書信原文以[[希臘文|希臘文]]寫成，後被翻譯成不同的文字版本。最早的[[圣经汉语译本|中文譯本]]出現於[[中國|中國]][[唐朝|唐朝]]的[[景教|景教]]，其名稱譯作《[[寶路法王經|寶路法王經]]》。
