'''保衛香港運動'''（{{lang-en|Defend Hong Kong Campaign}}）是[[香港|香港]]的一個亲[[中国共产党|中国共产党]]、擁護[[中華人民共和國政府|中華人民共和國政府]]、[[香港特別行政區政府|香港特別行政區政府]]和[[香港警隊|香港警隊]]的[[激进建制派|激进建制派]]政治組織，創始人兼發言人為[[傅振中|傅振中]]。該組織高調打擊[[港獨派|港獨派]]、[[泛民主派|泛民主派]]、[[自決派|自決派]]及[[本土派_(香港)|本土派]]的遊行集會政黨與人士。保衛香港運動參與者經常以揮舞[[中華人民共和國國旗|中華人民共和國國旗]]、[[香港特別行政區區旗|香港特別行政區區旗]]及使用藍帽作为標誌<ref name="「愛字頭」霸場打對台">{{cite news|url=http://hk.apple.nextmedia.com/news/art/20140220/18631834|title=「愛字頭」霸場打對台|publisher=《[[蘋果日報_(香港)|蘋果日報]]》|date=2014年2月20日|accessdate=2014-05-31}}</ref>。

== 歷史 ==
[[香港行政長官|香港行政長官]][[梁振英|梁振英]]於2012年7月1日上任後，新興[[香港親共團體|親共]]政治團體相繼成立，「[[愛港之聲|愛港之聲]]」於2012年9月成立。2013年8月，愛港之聲出現內鬥，繼而分裂。由[[傅振中|傅振中]]率領的十多人宣告脫離愛港之聲，另創新團，並名為「保衛香港運動」。該組織批評及非建制示威者及記者。

== 活動 ==
[[File:Defend_Hong_Kong_Campaign_at_Kowloon_City_Magistrates'_Court_at_10-59_on_February_12,_2016.JPG|250px]]
2016年2月12日9時4分，保衛香港運動約15名成員在[[九龍城裁判法院|九龍城裁判法院]]外，唱歌恭賀[[學民思潮|學民思潮]]成員[[林淳軒|林淳軒]]被起訴，又改編生辰歌「恭祝你今日訓街頭，慶賀你監牢好受」。載有林淳軒的警車抵達法院時候<ref name = if1>{{cite web|author1=[[香港獨立媒體網|香港獨立媒體網]]|title=林淳軒今上庭 學民到場聲援|url=https://www.facebook.com/inmediahk/posts/981871655183206|agency=[[Facebook|Facebook]]|accessdate=2016-02-15|date=2016-02-12|}}</ref>，保衛香港運動成員不斷拍打押送林淳軒到庭的警車，<ref name = nna2>{{cite news|title=林淳軒美國隊長被控暴動罪 與15歲男禁足旺角 撐警團體庭外指罵|url=https://www.facebook.com/inmediahk/posts/981871655183206|work=[[蘋果日報_(香港)|蘋果日報]]|accessdate=2016-02-15|date=2016-02-13|}}</ref>又圍截警車，高呼林淳軒是「仆街仔」，又高叫口號 「港獨人渣，嚴懲暴徒」，學民思潮與保衝香港運動於入庭前，<ref name = if1/>在電梯大堂爆發衝突，保衛香港運動成員向學民思潮成員多番挑釁，一名[[熱血公民|熱血公民]]成員上前到保衛香港運動成員前爭執，隨後發生衝突，之後該熱血公民成員就被保衛香港運動成員包圍，最後被警方拘捕。<ref name = p1>{{cite news|title=熱血公民阻土共挑釁學民反被捕　學民思潮急割𥱊　黎汶洛：不會主動襲擊持不同意見的人|url=http://www.passiontimes.hk/article/02-12-2016/28713|work=[[熱血時報|熱血時報]]|accessdate=2016-02-15|date=2016-02-12|}}</ref>

其後，保衛香港運動成員仍喋喋不休地指罵學生，又投訴庭內站立的學生遮擋視線，需由法庭書記出言制止，「呢度唔係[[電影院|戲院]]」（這裏不是電影院），要求眾人互相遷就。<ref name = nna2/>

10時45分，九龍城裁判法院外，保衛香港運動成員聚集。有到場人士離開法院時直指保衛香港運動為「六七暴動」兇手，期間雙方發生指罵<ref>{{cite web|author1=[[熱血時報|熱血時報]]|title=【旺角衝突被捕者提堂Day 2】聲援人士與土共團體雙方指罵|url=https://www.facebook.com/passiontimes/videos/1047599748636525/|agency=[[Facebook|Facebook]]|accessdate=2016-02-14|date=2016-02-12|}}</ref>。

11時15分，九龍城裁判法院外，保衛香港運動成員仍然聚集。當保衛香港運動成員叫口號「支持警察」時，另一邊市民就以「呑槍自殺」回應<ref>{{cite web|author1=[[熱血時報|熱血時報]]|title=【旺角衝突被捕者提堂Day 2】土共：「支持警察」另一邊：「呑槍自殺」|url=https://www.facebook.com/passiontimes/videos/1047608351968998/|agency=[[Facebook|Facebook]]|accessdate=2016-02-15|date=2016-02-12|}}</ref>。
