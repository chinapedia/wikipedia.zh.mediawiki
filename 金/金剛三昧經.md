'''金剛三昧經'''（梵語：Vajra-samādhi），該經中敘述佛陀入金剛三昧說一乘真實之法，收於《[[大正藏|大正藏]]》法華部，西藏本乃譯自漢本<ref name="vajra">{{cite web|author1=佛光電子大辭典|title=金剛三昧經|url=https://www.buddhistdoor.org/tc/dictionary/details/%E9%87%91%E5%89%9B%E4%B8%89%E6%98%A7%E7%B6%93}}</ref><ref>《金剛三昧經》：「爾時，尊者大眾圍遶，為諸大眾說大乘經，名一味真實無相無生決定實際本覺利行。若聞是經，乃至受持一四句偈，是人則為入佛智地，能以方便教化眾生，為一切眾生作大知識。佛說此經已，結加趺坐，即入金剛三昧，身心不動」。</ref>。本經雜有許多後起內容和名相，佛教學者多主張本經為後人之作，屬於[[禪宗|禪宗]]系統的[[佛典疑偽經文獻|本土經論]]<ref name=vajra2/><ref name="va4">{{cite book|author1=水野弘元|title=佛敎文獻硏究|url=https://books.google.com.tw/books?id=XTMCprlsolUC&lpg=PA62&hl=zh-TW&pg=PA62#v=onepage&q&f=false}}</ref>。

== 題解 ==
金剛三昧，為能通達一切諸法之[[三昧|三昧]]。因其堅固能斷破一切煩惱，猶如[[金剛|金剛]]堅固能摧破他物，故稱金剛三昧<ref>{{cite web|author1=佛光電子大辭典|title=金剛三昧|url=https://www.buddhistdoor.org/tc/dictionary/details/%E9%87%91%E5%89%9B%E4%B8%89%E6%98%A7}}</ref>。

== 傳譯 ==
本經始見於《[[出三藏記集|出三藏記集]]》〈新集安公涼土異經錄〉（北涼時代，397～439），為失譯經典。《彥琮錄》卷五將其歸入「闕本(舊錄有目而無經本)」，此後，釋靜泰《大唐東京大敬愛寺一切經論目》卷五、《大周錄》卷十二均認定其有目而無本。直至《[[開元釋教錄|開元釋教錄]]》，此經重新面世，並記載為「拾遺編入」<ref name="vajra"/><ref name=va7>{{cite journal|author1=尹富|title=地藏菩薩及其信仰傳入中國時代考|journal=四川大學學報|date=2006|url=http://www.cssn.cn/zjx/201412/t20141219_1449639_2.shtml}}</ref>。

[[宋高僧傳|宋高僧傳]]記載一則本經問世的神話，聲稱本經從「鈐海[[龍王|龍王]]」宮中傳出，仗新羅國王后之病為增上緣，出此經流傳[[新羅|新羅]]，而由[[元曉|元曉]]為之作疏<ref name=va7/>。

== 內容 ==
本經闡釋法空、[[佛性|佛性]]、[[如來藏|如來藏]]等[[中國佛教|中國佛教]]於南北朝時期探討的教說，計分八品：第一序品、第二無相法品、第三無生行品、第四本覺利品、第五入實際品、第六真性空品、第七如來藏品、第八總持品，賅攝當時所流行的諸[[大乘佛教|大乘]]思想<ref name="vajra"/><ref name="va4"/><ref>《金剛三昧經》：「若有眾生持是經者，則於一切經中無所悕求。是經典法總持眾法，攝諸經要；是諸經法，法之繫宗。是經名者，名『攝大乘經』」</ref>。

[[達摩四行觀|達摩四行觀]]的「二入」說與本經頗為類似，通常認為本經與[[禪宗|禪宗]]關係密切<ref name="v3">{{cite journal|author1=-{于}-德隆|title=《金剛三昧經》真偽考|journal=圓光佛學學報|date=2012|volume=20|pages=135-192|url=http://buddhism.lib.ntu.edu.tw/DLMBS/search/search_detail.jsp?seq=395613}}</ref><ref>{{cite web|author1=佛光大辭典|title=二入四行|url=http://buddhaspace.org/dict/fk/data/%25E4%25BA%258C%25E5%2585%25A5%25E5%259B%259B%25E8%25A1%258C.html}}</ref>。禪宗僧人[[道信|道信]]、[[堪布摩訶衍|堪布摩訶衍]]、[[永明延壽|永明延壽]]的著作，皆引用此經<ref>{{cite journal|author1=釋智學|title=中國疑偽佛典研究 （一）|journal=正觀|date=2007|volume=40|url=http://www.tt034.org.tw/upload/cht/134/1095_file_1.pdf}}</ref><ref>{{cite journal|author1=樓宇烈|title=讀慧海《頓悟入道要門論》隨記|journal=中華佛學學報|date=1999|volume=12|url=http://www.chibs.edu.tw/ch_html/chbj/12/chbj1206.htm}}</ref>。

== 考證 ==
現存經本的內容，說到「本覺」、「如來藏」，「庵摩羅」，「如來禪」，「九識」，佛教學者多據此斷定本經為後代偽作<ref name=vajra2>{{cite web|author1=印順|title=中國禪宗史|url=http://www.mahabodhi.org/files/yinshun/32/yinshun32-02.html}}</ref><ref name=va7/>。

水野弘元從譯語（淮、河、江、海）以及含有中國特有的菩薩階位思想，而判定本經在中國成立，約在648-666年間<ref name="va4"/>，尹富則認為當在671年前後至702年之間<ref name=va7/>。徐文明說此經應當創自中土，與[[慧可|慧可]]一系有關。韓國金英泰、美國巴斯維爾等人，則認為此經出自[[新羅|新羅]]<ref>{{cite journal|author1=徐文明|title=金剛三昧經作者辨|journal=中國文化研究}}</ref>。

-{于}-德隆反對本經為後人偽作之說，主張[[真諦|真諦]]重譯了《金剛三昧經》，並據此弘傳[[阿摩羅識|阿摩羅識]]理論<ref name="v3"/>。
