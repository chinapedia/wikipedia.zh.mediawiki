'''金上京历史博物馆'''，旧名'''阿城县博物馆'''，位于[[中国|中国]][[黑龙江省|黑龙江省]][[哈尔滨市|哈尔滨市]][[阿城区|阿城区]]延川大街南2公里处，始建于1961年。阿城区金上京历史博物馆以[[金代|金代]]历史为主题，设九个展厅，博物馆中另设晁楣艺术馆、粘氏石器馆两馆。2009年时，金上京历史博物馆有文物1,791余件套，其中[[国家一级文物|国家一级文物]]20件。博物馆是第二批[[国家三级博物馆|国家三级博物馆]]。

== 历史 ==
1961年，[[阿城县|阿城县]]博物馆成立。1965年，阿城县博物馆改为阿城县文管所。1986年，阿城县恢复博物馆，并定名金上京历史博物馆。1987年，阿城市金上京历史博物馆异地建设新馆，投资2000万元人民币。新馆在1998年10月8日正式开放。新馆为[[齐康|齐康]]设计，占地面积5万平方米，建筑面积5,400平方米，设九个展厅，展厅总面积3,000平方米，建筑风格带有[[金代|金代]]特色。<ref name=db>{{cite web|title=阿城金上京历史博物馆|url=http://special.dbw.cn/system/2009/11/14/052213383.shtml|publisher=东北网|accessdate=2013-09-27}}</ref>2001年，以金上京历史博物馆为主体的“[[哈尔滨金源文化旅游区|哈尔滨金源文化旅游区]]”获评[[国家4A级旅游景区|国家4A级旅游景区]]<ref>{{cite news|title=金上京历史博物馆  全国惟一金代历史博物馆|url=http://media.people.com.cn/GB/22114/113747/113751/6750039.html|accessdate=2013-09-27|newspaper=人民网－传媒频道|date=2008-01-08}}</ref>。2005年8月，因馆舍老化，阿城市花费830万元人民币对博物馆进行改造。改造历时三个月完成，2006年1月6日，博物馆于哈尔滨市冰雪节阿城主题日重新开馆。<ref>{{cite web|title=金上京历史博物馆|url=http://www.harbin.gov.cn/info/news/bsfw/fwlxz/detail/193321.htm|publisher=哈尔滨市人民政府|accessdate=2013-09-27|deadurl=yes|archiveurl=https://web.archive.org/web/20130515073236/http://www.harbin.gov.cn/info/news/bsfw/fwlxz/detail/193321.htm|archivedate=2013-05-15}}</ref>2006年9月，金上京历史博物馆中新建的[[晁楣|晁楣]]艺术馆开馆。2007年6月，另一座“馆中馆”粘氏石器馆开馆。<ref>{{cite == 展览 ==
金上京历史博物馆设九个展厅，地上五个，地下四个。2005年改造后，九个展厅是：序言、[[女真|女真]]崛起、[[金上京|金上京]]的辉煌、金源文化与金源人物、[[完颜|完颜]]后裔、金上京与[[金中都|金中都]]、金源絮余、金源遗风——女真族与[[满族|满族]]、金源宝鉴——金代铜镜。<ref>{{cite news|title=九大展厅数件奇珍首次公开 阿城金上京博物馆全新亮相|url=http://news.sohu.com/20060107/n241322474.shtml|accessdate=2013-09-27|newspaper=黑龙江生活报|date=2006-01-07}}</ref>晁楣艺术馆展出晁楣捐赠的80多幅版画、20多幅国画及书法<ref>{{cite news|title=晁楣艺术馆昨落户阿城|url=http://heilongjiang.dbw.cn/system/2006/09/20/050554630.shtml|accessdate=2013-09-27|newspaper=东北网|date=2006-09-20}}</ref>。粘氏石器馆展示金代石器<ref name=rb/>。1998年时，金上京历史博物馆的“金上京历史陈列”获得了第二届[[全国博物馆十大陈列展览精品|全国博物馆十大陈列展览精品]]评选精品奖。

== 馆藏 ==
据2009年的数据，金上京历史博物馆藏有文物1,791余件套，其中[[国家一级文物|国家一级文物]]20件，[[国家二级文物|国家二级文物]]11件，[[国家三级文物|国家三级文物]]18件<ref name=db/>。主要藏品有铜坐龙，直径43厘米、12.4公斤重的双鲤鱼铜镜，承安宝货，宝严大师塔铭志，黄金带跨，战国青铜剑，鎏金佛像等。馆藏的二百余面金代铜镜是博物馆的特色之一<ref>{{cite news|title=金上京历史博物馆内文物丰富|url=http://hlj.sina.com.cn/travel/destination/2013-03-06/40485_2.html|accessdate=2013-09-27|newspaper=新浪黑龙江|date=2013-03-06|deadurl=yes|archiveurl=https://web.archive.org/web/20131002123646/http://hlj.sina.com.cn/travel/destination/2013-03-06/40485_2.html|archivedate=2013年10月2日|df=}}</ref>。
