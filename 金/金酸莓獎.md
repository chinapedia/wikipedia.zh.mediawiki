'''金-{zh-tw:酸莓;zh-cn:酸莓;zh-hk:草莓}-獎'''（{{lang-en|'''Golden Raspberry Awards'''}}，简称Razzie Awards）是模仿[[奧斯卡金像獎|奧斯卡金像獎]]的负面頒獎典禮，每年都搶先在[[奧斯卡金像獎|奧斯卡]]頒獎前揭曉，藉以向备受傳媒批評的劣片揶揄。

== 名稱 ==
港译草莓，臺灣及中國大陸译酸莓。嚴格而言，金-{zh-tw:草莓;zh-hk:草莓;zh-cn:草莓;}-獎並非正確的翻譯。「Raspberry」中文為[[樹莓|樹莓]]，狹義上直指[[覆盆子|覆盆子]]，而非[[草莓|草莓]]，且另有表示轻蔑的咂舌之意。因樹莓味道大多有點甜中帶酸，所以最初的香港譯者因此使用「酸莓」這種帶有挖苦意味的翻譯名稱。

== 歷史 ==
1981年，電影製作及宣傳人[[約翰·J·B·威爾遜|約翰·J·B·威爾遜]]在其[[美國|美國]][[洛杉磯|洛杉磯]]住所客廳中發表金酸莓獎。其後曾經在不同場地舉行，包括住宅（派對形式）、學校禮堂、劇院和酒店等。

隨著1990年代網路興起，金酸莓獎亦建立自己的[[網站|網站]]。初期網頁設計故意佈滿一些[[HTML|HTML]]標記，好像未完成似的，內容雜亂無章，影射獲獎電影跟此網頁一樣粗製濫造。

== 獎項類別 ==
金酸莓獎：於1981年首度頒發，是最為人熟悉的獎項。獎座由一個[[高爾夫球|-{zh-hans:高尔夫球;zh-hk:哥爾夫球;zh-tw:高爾夫球;}-]]被[[8厘米膠卷|8厘米膠卷]]包圍，上面有一个覆盆子，再鍍上金色，跟參與的電影一樣，粗製濫造。歷年來金酸莓獎頒發了以下幾項主要獎項：

* {{link-en|金酸莓獎最差男主角|Golden Raspberry Award for Worst Actor|最差男主角}}：1980年至現在
* [[金酸莓獎最差女主角|最差女主角]]：1980年至現在
* {{link-en|金酸莓獎最差男配角|Golden Raspberry Award for Worst Supporting Actor|最差男配角}}：1980年至現在
* {{link-en|金酸莓獎最差女配角|Golden Raspberry Award for Worst Supporting Actress|最差女配角}}：1980年至現在
* {{link-en|金酸莓獎最差搭檔|Golden Raspberry Award for Worst Screen Combo|最差搭檔}}：1994年至現在
* {{link-en|金酸莓獎最差電影劇本|Golden Raspberry Award for Worst Screenplay|最差電影劇本}}：1980年至現在
* {{link-en|金酸莓獎最差導演|Golden Raspberry Award for Worst Director|最差導演}}：1980年至現在
* [[金草莓獎最差電影|最差電影]]：1980年至現在
* {{link-en|金酸莓獎最差前傳、重拍、惡搞及續集電影|Golden Raspberry Award for Worst Prequel, Remake, Rip-off or Sequel|最差前傳、重拍、惡搞及續集電影}}：1994年至現在，1996年和1999年除外
* [[金酸梅最佳贖莓獎|最佳贖莓獎]]：於2014年（第35屆）首次頒發，皆指之前曾數次入圍金酸莓獎但本次的作品卻得以讓該得獎者「重生」，意指「正面意味」。

=== 已消失 ===
* {{link-en|金酸莓獎最差原創歌曲|Golden Raspberry Award for Worst Original Song|最差原創歌曲}}：1980年至1999年、2002年
* {{link-en|金酸莓獎最差新星|Golden Raspberry Award for Worst New Star|最差新星}}：1981年至1998年，1989年除外
* {{link-en|金酸莓獎最差配樂|Golden Raspberry Award for Worst Musical Score|最差配樂}}：1981年至1985年
* 最差視覺效果：1986年至1987年

== 其他獎項 ==
* 最差成就獎：此獎項曾經在1981年至1987年頒發，首屆獎項由曾經擔任演員的時任[[美國總統|美國總統]][[隆納·威爾森·雷根|隆納·威爾森·雷根]]奪得。
* 主席獎（The Governor's Award）：由金酸梅獎主席John Wilson頒發的特別獎項，主要是頒給不歸屬現有獎項類別的「傑出人士」。[[崔維斯·培恩|崔維斯·培恩]]（Travis Payne）于2003年憑電影《[[熱舞狂潮|熱舞狂潮]]》（''From Justin to Kelly''）奪得最差編舞獎的「殊榮」。

== 著名領獎嘉賓 ==

不少較著名的電影導演、演員及製作人都曾經獲得金酸梅獎；但由於獎項的性質，他們大部分都不願意出席領獎，歷年来只有少數人士親自接受獎項：

* 1987年：[[比爾·寇司比|比爾·寇司比]]憑《[[天才老爹拯救地球|天才老爹拯救地球]]》（''Leonard Part 6''）中飾演傻頭傻腦的[[間諜|間諜]]而一口氣獲得最差電影、最差男演員及最差編劇三個獎項。他是首位願意接受獎項的人士，並在頒獎典禮舉行後數星期在[[福克斯廣播公司|-{zh-hans:福克斯;zh-hk:福克斯;zh-tw:福斯;}-广播公司]]節目《[[午夜漫談|午夜漫談]]》（''The Late Show''）親自領獎，但他要求獎座是由足金和[[意大利|意大利]][[大理石|大理石]]造成的「豪華」版本，於是廣播公司花了27,000[[美元|美元]]打造此獎座（金酸梅獎基金會估計標准的獎座只值4.97美元）。
* 1996年：[[保羅·范赫文|保羅·范赫文]]（Paul Verhoeven）是首位親自到典禮現場領獎的人士，他憑《[[美國舞孃|美國舞孃]]》一片獲得最差導演獎。
* 1998年：[[布萊恩·海格蘭|布萊恩·海格蘭]]（Brian Helgeland）是首位於同一星期內同時獲得金酸梅獎及[[奧斯卡獎|奧斯卡獎]]的人士。他憑《[[鐵面特警隊|-{zh-hans:洛城机密;zh-hk:幕後嫌疑犯;zh-tw:鐵面特警隊;}-]]》奪得奧斯卡最佳改編劇本獎，而在前一天憑《[[2013終極神差|2013終極神差]]》（''The Postman''）獲得最差編劇獎。雖然布萊恩並未出席金草莓獎頒獎典禮，但他希望可以把兩個獎項放在一起，讓他體會到[[好萊塢|-{zh-hans:好莱坞;zh-hk:荷里活;zh-tw:好萊塢;}-]][[唐吉訶德|唐吉訶德式]]的性質，並在不久後於[[華納兄弟|華納兄弟]]的辦公室中親自接受金草莓獎。
* 2002年：[[湯姆·葛林|湯姆·葛林]]（[[en|Tom Green]]）憑《[[魔鬼老哥|魔鬼老哥]]》（[[en|''Freddy Got Fingered'']]）獲得五個獎項，包括最差電影獎，並親自領獎。
* 2004年：[[班·艾佛列克|班·艾佛列克]]憑《[[絕配殺手|絕配殺手]]》（''Gigli''）、《[[夜魔侠|夜魔侠]]》以及《[[記憶裂痕|記憶裂痕]]》中的「精彩演出」勇奪最差演員獎，並不憤地問為何沒有拿到獎座。當他一星期後在談話性節目《[[賴利金現場|賴利金現場]]》取得獎座後，馬上把它摔爛。主辦單位把毀壞的獎座在[[eBay|eBay]]網站拍賣，獲得的金額居然足夠支付來年頒獎典禮租借禮堂的費用。
* 2005年：[[哈莉·贝瑞|哈莉·贝瑞]]親身出席這屆金酸梅獎頒獎典禮可能是歷年來最「聲名狼藉」的，並震撼整個-{zh-hans:好莱坞;zh-hk:荷里活;zh-tw:好萊塢;}-。 [[哈莉·贝瑞|哈莉·贝瑞]]憑《[[貓女|貓女俠]]》獲頒最差女主角，她一手拿著數年前奪得的奧斯卡最佳女主角獎座，另一手則拿著金酸梅獎座，惹得台下一片笑聲。
* 2010年：[[珊卓·布拉克|珊卓·布拉克]]以《[[求愛女王|求愛女王]]》（"All About Steve"）獲獎。她親自出席，並攜帶一拖車的DVD要分送給評審，希望他們「認真」觀看。在隔天的奧斯卡頒獎典禮中，她以《[[攻其不備|攻其不備]]》奪得奧斯卡最佳女主角，這也創了金酸莓獎設置以來的紀錄。

奧斯卡和金酸莓曾經提名同一部電影中的主角，包括1982年《[[酸甜苦辣母女會|酸甜苦辣母女會]]》（''Only When I Laugh''）中的[[詹姆斯·科可|詹姆斯·科可]]（James Coco）和1984年《[[楊朵|楊朵]]》（''Yentl''）中的[[艾美·厄文|艾美·厄文]]（Amy Irving）。二人最終並沒奪得獎項。同年由同一人奪下兩獎的情形要到2010年才出現，由珊卓布拉克達成。
