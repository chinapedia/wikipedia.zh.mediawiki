'''金鳌玉蝀桥'''，原名'''金海桥'''，又名'''御河桥'''，俗称'''北海大桥'''，位于[[北京市|北京市]][[西城区|西城区]][[北海公园|北海公园]]和[[中南海|中南海]]之间，桥上为[[文津街|文津街]]。

==简介==
北海大桥位于[[北海及团城|团城]]脚下，横跨北海与中海之间。桥两端原有[[明朝|明朝]][[嘉靖帝|嘉靖帝]]建的牌坊，桥西牌坊的匾额为“金鳌”，桥东牌坊的匾额为“玉蝀”，故称“金鳌玉蝀桥”。此桥长150米，宽8米，是石砌七孔拱券式桥梁，中心孔券面有浮雕兽头，原栏板都是平顶方形覆莲柱头望柱。桥身如同一条玉带，是中国堤障式石拱桥的典型<ref name=fandui>{{Cite web |url =  http://news.sohu.com/20081118/n260710425.shtml|title = 北海大桥往事:郑振铎等反对拆除 周总理保住团城 |author =  |date = 2008-11-18 |publisher = 搜狐 |language =  |accessdate =  |quote =  }}</ref>。

[[中华人民共和国|中华人民共和国]]成立后，北京车辆增加，金鳌玉蝀桥桥面窄，坡陡弯急，难以适应交通需要。有人提出，中央领导的汽车自中南海出来后东行必经团城，可能撞到牌坊或撞向团城。因此有人建议拆除团城，拉直路面，另建新桥。但该建议受到诸多文物工作者及古建筑学家反对<ref name=fandui/>。

[[国家文物局|国家文物局]]得知“拆除团城”的消息后，局长[[郑振铎|郑振铎]]请来[[梁思成|梁思成]]、[[范文澜|范文澜]]、[[翦伯赞|翦伯赞]]等几位学者会商，认为如既要保留团城，又使桥面扩宽，唯一的方法是将桥身向中南海方向拓宽。郑振铎乃给[[周恩来|周恩来]]总理写了个报告。他的朋友[[沈雁冰|沈雁冰]]是文化部部长，以“特急件”直送总理办公室。梁思成还请[[苏联|苏联]]专家参观团城，希望获得支持，但并未奏效。后来梁思成直接见周恩来，当面陈述保护团城的意见。郑振铎又让[[罗哲文|罗哲文]]去团城测绘拍照，搜集资料，尽快在《文物保护参考》杂志上发表。几天后的一夜，郑振铎接到总理办公室电话，通知他和副局长[[王冶秋|王冶秋]]陪周恩来视察团城<ref name=fandui/>。

1954年6月的一天，周恩来到团城视察，先环团城走了一圈，后站在城墙边，眺望北海琼岛，观察来往车辆，并向郑振铎等文物工作者询问有关情况，最后周恩来把目光移向中南海方向说：团城的一砖、一瓦、一树、一石都不能动！把桥面向中南海方向扩展，拆掉“金鳌”、“玉蝀”两个牌坊。拆掉后的两个牌坊移到了[[陶然亭公园|陶然亭公园]]，在“[[文革|文革]]”期间因[[破四旧|破四旧]]而销毁<ref name=fandui/><ref name=youlai>{{Cite web|url =http://www.bjxch.gov.cn/BJXCHHuDongJiaoLiu/BJXCHXCBAXiangQing.ycs?GUID=561847|title =文津街的由来|author =|date =2013-11-22|publisher =北京西城报|language =|accessdate =|quote =}}{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>。

改建后的金鳌玉蝀桥保持了原桥的风貌，桥身加长至220米，桥面拓宽至34米，中间的车行道27米，两侧的人行道3.5米，桥由7孔改为9孔，中间一孔用于通水，其余封堵当作装饰。为安全起见，将石栏板改成高的铁栏杆。1972年再次加高铁栏杆，当时是因[[天安门广场|天安门广场]]发生抢花事件后，又有人从北海大桥上跳下进入中南海，所以铁栏杆再次加高<ref name=fandui/>。
