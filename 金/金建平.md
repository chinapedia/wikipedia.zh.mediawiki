'''金建平'''（{{bd|1959年|4月13日||catIdx=J}}），籍贯[[天津市|天津市]]。1983年毕业于[[天津大学建筑分校|天津大学建筑分校]]（今[[天津城建大学|天津城建大学]]），1999年[[天津大学|天津大学]]工商管理研究生毕业，[[天津燃气集团|天津燃气集团]]董事长。<ref name="中国企业联合会网站">{{cite web| url=http://www.cec-ceda.org.cn/qiye/q03/qj022.42.htm| title=金建平简历| publisher=中国企业联合会网站| accessdate=2013-09-13| deadurl=yes| archiveurl=https://web.archive.org/web/20110302104252/http://www.cec-ceda.org.cn/qiye/q03/qj022.42.htm| archivedate=2011-03-02}}</ref>

== 简历 ==
金建平1983年毕业于[[天津大学|天津大学]]建筑分校（今[[天津城建大学|天津城建大学]]）<ref>金建平1983年毕业于天津大学建筑分校，即天津城建大学前身，部分媒体误报为天津大学建筑系</ref>，1999年金建平[[天津大学|天津大学]]工商管理研究生毕业，2003年金建平受[[中共天津市委|中共天津市委]]组织部委派到[[墨尔本皇家理工大学|澳大利亚皇家理工大学]]进修三个月。<ref name="中国企业联合会网站" />

== 工作经历 ==
1983年，金建平毕业后进入天津市第一煤气厂，至1996年先后担任该厂的技术员、车间主任、副厂长。1996至1999年，金建平任天津市煤气总公司副总经理、天津煤气设计院院长。2001至2002年1月，金建平任天津市燃气集团副总经理。2002年1月，金建平升任天津市燃气集团有限公司总经理。<ref name="中国企业联合会网站" />2013年9月2日，天津津燃公用事业股份有限公司发布公告，金建平因工作重新调配已辞任董事会主席、执行董事、薪酬委员会成员、提名委员会主席及监察主任等职务。<ref name="国际金融报 ">{{cite web | url=http://paper.people.com.cn/gjjrb/html/2013-09/13/content_1297937.htm | title=天津燃气金建平涉嫌严重违纪 | publisher=国际金融报 | accessdate=2013-09-13}}</ref>

== 双规 ==
2013年9月5日，《[[21世纪经济报道|21世纪经济报道]]》从一位接近天津官方的匿名人士处得知，金建平于几日前在[[河北省|河北省]][[香河|香河]]的居所出门买早点时被天津市纪委控制，纪委在其家中查出了部分现金。其部分家人已经被转移至国外，金建平在得到风声后，潜逃出境失败。<ref>{{cite web| url=http://sztqb.sznews.com/html/2013-09/13/content_2621815.htm| title=天津燃气集团董事长 金建平正接受调查| publisher=深圳特区报| date=2013-09-13| accessdate=2013-09-13}}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

2013年9月12日，中共天津市纪委表示，天津燃气集团董事长金建平因严重违纪已被“[[双规|双规]]”。<ref name="天津燃气集团董事长金建平因严重违纪正接受组织调查">{{cite web| url=http://www.tj.xinhuanet.com/tt/jcdd/2013-09/12/c_117341550.htm| title=天津燃气集团董事长金建平因严重违纪正接受组织调查| publisher=新华网| date=2013-09-12| accessdate=2013-09-12| deadurl=yes| archiveurl=https://web.archive.org/web/20140430013846/http://www.tj.xinhuanet.com/tt/jcdd/2013-09/12/c_117341550.htm| archivedate=2014-04-30}}</ref>由于天津燃气集团与[[中国石油天然气股份有限公司|中国石油天然气股份有限公司]]有较多业务往来，因而多家媒体在报道中分析和猜测，金建平可能与中石油的腐败案相关，但这一观点未获官方证实。<ref>{{cite web | url=http://stock.jrj.com.cn/2013/09/16193415847561.shtml | title=天津燃气董事长严重违纪被查 与中石油有业务往来 | publisher=北京青年报 | accessdate=2013-09-19}}</ref>

2013年9月13日，金建平被曝违纪后，其曾经任职的天津津燃公用事业股份有限公司表声明称，金建平涉嫌违纪受查一事与公司无关。<ref>{{cite web | url=http://www.ccstock.cn/HKstock/guoqinews/2013-09-12/A1327144.html | title=津燃公用前主席涉违纪受查 事件与公司无关 | date=2013-09-13 | accessdate=2013-09-13 | archive-url=https://web.archive.org/web/20160304135858/http://www.ccstock.cn/HKstock/guoqinews/2013-09-12/A1327144.html | archive-date=2016-03-04 | dead-url=yes }}</ref>
