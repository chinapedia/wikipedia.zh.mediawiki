'''金泰希'''<ref>{{cite news| url= http://news.naver.com/main/read.nhn?mode=LSD&mid=sec&sid1=106&oid=009&aid=0002315791 |title= ‘그랑프리’ 김태희, “서른은 나를 당당해지게 했다” | language=ko |work= startoday |publisher= [[Naver_(公司)|Naver]] |date=2010-09-03}}[http://entertain.naver.com/read?oid=009&aid=0002315791 url]</ref>（{{lang-ko|김태희}}，{{bd|1980年|3月29日|catIdx=Kim}}），名字常被誤寫為'''<nowiki>金泰熙</nowiki>'''，[[韓国|韓国]][[女演员|女演员]]。[[釜山廣域市|釜山廣域市]]出生，蔚山女子高中、[[首尔大学|首尔大学]]服裝設計學系（2005年）毕业<ref name="Cine21 profile">{{cite web |url=http://www.cine21.com/Movies/Mov_Person/person_info.php?id=22347 |title=김태희 |trans_title= |publisher=Cine21 |date=2007 |archivedate=2016-03-03 |archiveurl=https://web.archive.org/web/20160303215514/http://www.cine21.com/Movies/Mov_Person/person_info.php?id=22347 |access-date=2010-02-23 |dead-url=yes }}</ref><ref name="sp-jpn"/>。

金泰希畢業于[[首爾大學|首爾大學]]，學生時期成績優異，是演藝圈內少數擁有高學歷且非戲劇[[科班|科班]]出身的演員。[[廣告|廣告]][[模特|模特]]出身的她在業內素有「廣告女王」的稱號，出道以來拍攝過許多爲觀衆所熟悉的經典廣告。<ref name="sp-jpn">[http://www.sp-kimtaehee.net/ 金泰希經紀公司（日本）官方介紹]{{ja}}{{en}}{{ko}}</ref>

== 演藝經歷 ==
金泰希2000年藉由電視廣告出道，在圈內是屈指可數的首爾大學出身的藝人。2001年，獲邀出演電影《禮物》，飾演[[李英愛|李英愛]]的少年時代；2002年又出演短篇電影《新都市人》及情景喜劇《Let's go》；2003年作爲第一主角出演了SBS電視劇《[[夢想的屏幕|夢想的屏幕]]》，飾演一名在電影制作領域心懷夢想努力生活的天才制-{}-作人。從家庭長劇《興夫家的葫蘆開了》中的乖巧少女到《[[天國的階梯|天國的階梯]]》裏的蛇蠍女人，一轉眼又成爲《[[愛在哈佛|愛在哈佛]]》裏那個奔跑在哈佛美麗校園裏的醫學院女學生。從神秘遙遠的《[[中天_(電影)|中天]]》到極富個性的《打架》，再到緊張刺激又深情款款的《[[IRIS_(電視劇)|IRIS]]》以及天真爛漫的《[[我的公主|我的公主]]》，都有不錯的表現。<ref>[http://www.mydaily.co.kr/news/read.html?newsid=201101141557551110&ext=na 김태희, 드디어 연기인생 한판승 거두나?] mydaily</ref>2011年，受邀主演日本[[富士電視台|富士電視台]]電視劇《[[明星保鑣99天|明星保鑣99天]]》。<ref>[http://movie.naver.com/movie/bi/pi/basic.nhn?code=31893 김태희] movie naver</ref>

== 社會評價 ==
金泰希畢業于韓國第一學府首爾大學，學生時期年年全校第一名的優異學習成績一直被人稱道。<ref>[http://www.etoday.co.kr/news/section/newsview.php?TM=news&SM=4101&idxno=534814 김태희 성적표 공개, 3년간 전국1등에 체력은 '특급'] etoday</ref>金泰希素來以自然美著稱，也是韓國最爲人知曉和具代表性的天然美女，是整容外科醫生點名的完全沒有整過容的女星之一。<ref>[http://sports.khan.co.kr/news/sk_index.html?cat=view&art_id=201101131845363&sec_id=540101&pt=nv 한국 최고미녀 왜 김태희인가] 스포츠경향</ref>作為頂級明星，金泰希的廣告代言費高達每年十億韓元，并連續多年被評為最受觀眾喜愛的廣告模特。<ref>{{cite web| url=http://www.mydaily.co.kr/news/read.html?newsid=200802111723061110 |title= CF톱스타들 왜 연기력 논란 거세나? |date=2008-02-11|archivedate=2008-02-18 |archiveurl= http://archive.is/www.mydaily.co.kr/news/read.html?newsid=200802111723061110 |language=ko |publisher= 마이데일리 my daily}}</ref>
