《'''金田一少年之事件簿：上海魚人傳說'''》（'''{{lang|ja|金田一少年の事件簿}}：{{lang|ja|上海魚人伝説'''}}；'''Kindaichi shonen no jikembo:Shanghai ningyo densetsu'''）即依據日本編劇家[[天樹征丸|天樹征丸]]所撰寫的小說《[[上海魚人傳說|上海魚人傳說]]》為基礎，再進行改編的電影作品。

== 概述 ==
拍攝期間，曾遠赴[[中國|中國]]取景與動用[[中國大陸|中國大陸]]與[[台灣|台灣]]演員參與演出。

== 劇情簡介 ==
七瀨美雪收到一封由筆友楊蕾麗從上海寄來的信，指她的哥哥楊小龍捲入殺人事件而大傷腦筋，於是七瀨美雪和金田一一來到上海解決事件。但是楊氏雜技團仍然發生殺人事件，楊小龍同樣沒有不在場證據，而凶器德林格手槍一直下落不明，於是金田一和楊小龍一同逃亡，並找出真相。同時，劍持勇因上海市公安李波兒的通知來到上海，原因是將金田一帶回日本。但在金田一不斷追查的案件時，卻發現這次案件與一件日本十年前的交通事故逃逸案件有關。
