'''金卡丁'''（{{lang-en|Kincardine}}）是[[加拿大|加拿大]][[安大略省|安大略省]][[南安大略|西南部]][[布魯斯縣|布魯斯縣]][[休倫湖|休倫湖]]畔一個地方自治區，為全球最大核電站之一布魯斯核電站所在。據2011年加拿大人口普查所示，金卡丁人口為11174人<ref name="cp2011"/>。

此地遠離[[上加拿大|上加拿大]]殖民地的腹地，因此較現[[南安大略|南安大略]]的其他地域遲開發，首個歐裔聚落待1848年才於彭内坦戈爾河（{{lang|en|Penetangore River}}）匯入休倫湖所在之處形成，並以該河名為彭内坦戈爾。殖民地政府則於1850年將[[布魯斯半島|布魯斯半島]]及其以南部分地域設為[[布魯斯縣|布魯斯縣]]，又在縣内劃下多個鄉（{{lang|en|township}}），當中現金卡丁自治區一帶分別劃為金卡丁鄉和布魯斯鄉，而彭内坦戈爾聚落則於1858年建制為金卡丁鎮<ref name="muni1">{{cite web |url=http://www.kincardine.net/town-kincardine.cfm |title=Town of Kincardine |publisher=金卡丁自治區政府 |accessdate=2014-05-02 }}</ref>；上述全部地名皆以在任[[加拿大省|加拿大省]][[加拿大總督|總督]][[詹姆斯·布魯斯，第八代額爾金伯爵|金卡丁伯爵詹姆斯·布魯斯]]命名。另一方面，坐落金卡丁鄉和布魯斯鄉界線之上，離休倫湖湖岸不遠處亦於1850年形成歐裔聚落，到1860年取名蒂佛頓（{{lang|en|Tiverton}}）<ref name="muni2">{{cite web |url=http://www.kincardine.net/tiverton.cfm |title=Tiverton |publisher=金卡丁自治區政府 |accessdate=2014-05-02 }}</ref>，再於1879年建制為村。

安大略省政府於1990年代末精簡轄下地方架構：蒂佛頓村先於1998年取消建制及併入布魯斯鄉，而金卡丁鎮、金卡丁鄉和布魯斯鄉則於1999年1月1日合併成金卡丁-布魯斯-蒂佛頓鄉，再於同年末改稱金卡丁自治區<ref>{{cite web |url=http://www.mah.gov.on.ca/Page5382.aspx |title=Municipal Restructuring Activity Summary Table |publisher=安大略省政府 |accessdate=2014-05-02 }}</ref>。

安大略9號和21號省道在金卡丁交匯：前者西起金卡丁，向東南經[[布羅克頓_(安大略省)|布羅克頓]]伸延至緬圖（{{lang|en|Minto}}）；後者南通[[戈德里奇|戈德里奇]]，北達[[奧雲灣|奧雲灣]]。此外，布魯斯縣的7號、15號、20號和23號縣道皆途經金卡丁。
