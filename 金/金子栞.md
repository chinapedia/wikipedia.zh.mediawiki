'''金子栞'''（{{bd|1995年|6月13日||}}），是[[日本|日本]]女性偶像團體[[SKE48|SKE48]] Team E的前成員<ref>{{Cite news|url=http://ent.sina.com.cn/y/yrihan/2014-12-11/doc-iavxeafr6748453.shtml|title=日本SKE48偶像成员金子栞或拍下海拍戏|accessdate=2018-08-15}}</ref>，[[埼玉縣|埼玉縣]]出身，隸屬於[[AKS|AKS]]。

== 簡歷 ==
; 2010年
* 9月29日，[[SKE48|SKE48]]第4期生選拔合格（應徵總人數5,888名，最終合格者16名）。
* 12月6日，隨著Team E的組成，從研究生升格，成為SKE48的正式成員。
; 2011年
* 1月1日，在[[元旦|元旦]]舉行的SKE48的元旦公演之中，以Team E成員的身分，在[[SUNSHINE_SAKAE|SUNSHINE SAKAE]]（SKE48劇場）的舞台上表演，與同Team的[[木本花音|木本花音]]一同擔任Team E的中心。
* 11月9日，成為SKE48第7張[[單曲|單曲]]《[[愛就OK|愛就OK]]》的選拔成員，首次進入選拔組。
; 2012年
* 1月25日，成為SKE48第8張[[單曲|單曲]]《[[片思いFinally|片思いFinally]]》的選拔成員。
* 5月16日，成為SKE48第9張[[單曲|單曲]]《[[アイシテラブル!|アイシテラブル!]]》的選拔成員。
; 2013年
* 6月8日，成功於[[AKB48第32張單曲選拔總選舉|AKB48第32張單曲選拔總選舉]]獲得63位，首次進入AKB[[單曲|單曲]]的Future Girls。
; 2014年
* 4月8日，於官方設立之個人Blog中宣布考取大學<ref>{{cite web |url=http://www2.ske48.co.jp/blog_pc/detail/id:20140408140312862/ |title=報告(^^) |publisher=SKE48官方網站的金子栞個人Blog |language=ja |date=2014-04-08 |accessdate=2014-04-08 |deadurl=yes |archiveurl=https://web.archive.org/web/20140408213719/http://www2.ske48.co.jp/blog_pc/detail/id:20140408140312862/ |archivedate=2014-04-08 }}</ref>，同時發表將從SKE48畢業的消息。<ref>{{cite web|url=http://www.ske48.co.jp/news/?id=1396951889 |title=SKE48 チームE 金子栞 活動辞退による卒業のご報告 |publisher=SKE48官方網站 |language=ja |date=2014-04-08|accessdate=2014-04-08}}</ref>。
* 4月29日，最後作為SKE48的一員，在該日進行的個別握手會，正式從SKE48畢業。
== 人物 ==
* 為[[SKE48|SKE48]]之中唯一一名出身自[[關東地方|關東地方]]的正式成員。
* 公演口號為「'''一～二，Happy Happy，嗯！今天也將要許多的Happy Smile傳達給大家。我是埼玉縣出生、○○歲的金子栞。'''」
* 是SKE48之中，繼[[高田志織|高田志織]]（Takada '''Shiori'''）、[[小木曾汐莉|小木曾汐莉]]（Ogiso '''Shiori'''）和[[井口栞里|井口栞里]]（Iguchi '''Shiori'''）之後的第四位「Shiori」。
* 不是本人自己報名甄選會的。
* 是家鄉[[埼玉縣|埼玉縣]]的職業足球會「[[浦和紅鑽|浦和紅鑽]]」的球迷。
* 曾在演唱會上自己爆料說，從小學開始就持續在參加競走大賽。
* 因為呆呆的個性，被成員認定是「[[天然呆|天然]]」的人物，但自己強烈否定，自稱是乾脆主義。
* 綽號「金醬（{{lang|ja|きんちゃん}}）」是由SKE的劇場支配人湯浅洋所取的。
