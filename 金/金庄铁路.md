'''金庄铁路'''是位于[[中国|中国]][[辽宁省|辽宁省]][[大连市|大连市]]的一条[[铁路|铁路]]，自[[金州区|金州区]]的[[金州站|金州站]]至[[庄河市|庄河市]]的[[庄河站|庄河站]]。金州站至[[城子坦站|城子坦站]]（又称'''金城铁路'''）由[[沈阳铁路局|沈阳铁路局]]管辖，城子坦南站至庄河站（又称'''城庄铁路'''）由庄河地方铁路局管辖。单线铁路，尚未进行[[电气化|电气化]]改造。铁路沿线盛产海[[盐|盐]]、[[油料作物|油料作物]]、[[木|木]]材及[[水果|水果]]。

铁路兴建于[[日本|日本]]治下的[[关东州|关东州]]时期，1927年10月1日开通，当时名为'''金福铁路'''，从金州站到城子疃站（1965年城子疃镇更名为[[城子坦|城子坦]]镇，车站也随之更名），私营铁路，对关东州东海岸地区起了重要作用。沿线的著名产业包括貔子窝的盐田，亮甲店的大花生，各地的花岗岩，硅石和貔子窝的海产品等。

1939年5月20日由[[满铁|满铁]]收购。

20世纪80年代，庄河市开始修建城子坦至庄河段，从城子坦站以南引出新线至庄河站，新设城子坦南站，使原城子坦站孤悬于原线终端。

2009年4月起，每天与[[大连站|大连站]]间开行一对[[中国铁路列车车次#普通旅客快车（普快）|普通旅客列车]]（6333/6334次）。每年春运期间6333/6334次车体支援南方旅客流量较大的线路，致使该趟列车停运1至2个月。

== 相连铁路 ==

== 车站 ==
金州 - 金州东门 - 牛角山 - 广宁寺 - 蚕厂屯 - 亮甲店 - 泉水屯 - 登沙河 - 旗杵底 - 杏树屯 - 大刘家 - 皮口 - 夹心子 - 城子坦南（城子坦） - 明阳 - 高阳镇 - 大郑 - 庄河

* 金州东门站为技术停车，不办理客运业务。
* 2009年后，牛角山、蚕厂屯、泉水屯、旗杵底、大刘家站不再有图定旅客列车停靠，车站现已废弃；城子坦站也已无旅客列车停靠，车站仍正常运营，从事货运业务；城子坦南站由沈阳铁路局与庄河地方铁路共同负责货运与客运业务。

== 计划 ==
* 2010年开工建设[[丹大城际铁路|丹大城际铁路]]，与金庄线共用一小段线路。
* [[海岫铁路|海岫铁路]]计划延伸至庄河，与金庄线相接。
