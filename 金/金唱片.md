'''金唱片'''或'''金唱片獎項'''是根據一張唱片（[[音樂專輯|音樂專輯]]或[[單曲|單曲]]等）的銷量而頒授的一個獎項。在世界各地也有頒授金唱片，不過獲獎所需的銷量要求各有不同，獎項通常會分為銀唱片、金唱片、白金唱片（白金唱片的兩倍為雙白金唱片，三倍為三白金唱片，如此類推）和[[鑽石唱片|鑽石唱片]]。

== 香港的金唱片 ==
在[[香港|香港]]，金唱片獎項是由「[[國際唱片業協會|國際唱片業協會]]（香港會）」（IFPIHK）頒授，當中包括金唱片、白金唱片及多白金唱片。國際唱片業協會（香港會）於1977年起開始頒發獎項，不過因本地流行唱片銷量每況愈下的關係，於2006年<ref>[http://the-sun.on.cc/channels/fea/20051122/20051121230535_0000_1.html 《太陽報》 信和賣碟指數 Janice贏開巷]</ref><ref>[http://hk.apple.nextmedia.com/entertainment/art/20051126/5430783 IFPI金唱片定新制]（《蘋果日報》）</ref>及2008年<ref>[http://news.sina.com.hk/cgi-bin/nw/show.cgi/15/1/1/602792/1.html 《東方日報》 金唱片數字再下調]</ref>兩度降低頒獎的要求。

== 台灣的金唱片 ==
[[台灣|台灣]]的金唱片獎項是由《[[時報周刊|時報周刊]]》主辦，[[中國電視公司|中國電視公司]]（中視）協辦並製播吳芳製作、[[包國良|包國良]]主持的[[綜藝節目|綜藝節目]]《金唱片》，由全台灣[[觀眾|觀眾]]票選最喜歡的唱片，以[[電腦|電腦]]分四區（東區、中區、南區、北區）[[統計|統計]]「全國唱片排行榜」，並有新星新歌新曲選票分析與介紹。<ref>[[中華民國電視學會|中華民國電視學會]]，《[[中華民國電視年鑑|中華民國電視年鑑]]第三輯：民國六十七年至七十二年》，1984年6月30日出版，第70頁。</ref>

== 日本的金唱片 ==
[[日本|日本]]的唱片認證由[[日本唱片協會|日本唱片協會]]頒發，自2003年7月開始所有唱片不分種類一律定為金唱片100,000張、白金唱片250,000張、百萬唱片1,000,000張。
