'''金坛县'''，[[中国|中国]]旧[[县|县]]名，在今[[江苏省|江苏省]][[常州市|常州市]]境。

本为[[曲阿县|曲阿县]]地。[[隋朝|隋朝]]末年，为[[金山县_(隋朝)|金山县]]。[[唐朝|唐朝]]时并入[[延陵县_(西晋)|延陵县]]。[[垂拱|垂拱]]四年（688年），复置，更名为金坛县。属[[润州_(隋朝)|润州]]、[[丹杨郡|丹杨郡]]<ref>《[[新唐书|新唐书]]·志第三十一·地理五》</ref>。[[北宋|北宋]]末年后，属[[镇江府|镇江府]]。[[元朝|元朝]]时，属[[镇江路|镇江路]]。[[明朝|明]][[清朝|清]]时，属镇江府。民国初年，镇江府废，仍属[[江蘇省_(中華民國)|江苏省]]。1937年12月2日，[[日军|日军]]第9[[师团|师团]]占领金坛<ref>{{Cite web |url=http://news.ifeng.com/mil/200612/1203_235_42801.shtml |title=战史今日：12月2日 奥斯特里茨大会战爆发 |accessdate=2015-01-28 |author= |date=2006年12月3日 |work= |publisher=凤凰网|language=Zh-hans }}</ref>。

1949年4月20日夜，[[中国人民解放军|解放军]]发起[[渡江战役|渡江战役]]。22日至23日，金坛县[[县长|县长]]李公达等党政军警人员南逃，[[中华民国政府|中华民国政府]]对金坛县的管辖结束。23日深夜，[[中国人民解放军第三野战军|中国人民解放军第三野战军]][[中国人民解放军第20军|第二十军]]六十师一七八团进入金坛县地界。24日凌晨，进入金坛县城<ref>{{Cite web |url=http://www.jsjt.gov.cn/zgjt/jtnj/Chinese%20Communist%20history%20of%20Jintan-49-78/content/01/01-1.html |title=中国共产党金坛历史第二卷（1949-1978）——第一章第一节人民政权的建立|accessdate=2014-10-22 |author= |date= |work= |publisher=金坛市人民政府门户网站，版权所有：中国共产党金坛历史编纂委员会|language=Zh-hans }}</ref>。同年，金坛县属[[常州专区|常州专区]]，1953年，属[[镇江专区|镇江专区]]。后又属常州专区。1959年，复属镇江专区。1983年，属[[常州市|常州市]]。1993年11月，[[中华人民共和国国务院|中国国务院]]批准，撤销金坛县，设立[[金坛市|金坛市]]（县级）。<ref>{{Cite web |url=http://www.jsjt.gov.cn/zgjt/jtnj/shizhi-88-07/web/01jzqhzrdl/jzqh.html |title=金坛市志1988-2007——第一章第一节建置 |accessdate=2014-10-22 |author= |date= |work= |publisher=金坛市人民政府门户网站，版权所有：金坛市地方志编纂委员会|language=Zh-hans }}</ref>
