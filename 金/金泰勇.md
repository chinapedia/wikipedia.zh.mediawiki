'''金泰勇'''（{{lang-ko|'''김태용'''}}，{{bd|1969年|12月9日}}），[[韩国|韩国]][[导演|导演]]、[[編劇|編劇]]。2000年，以《[[女高怪談2_鬼戀人|女高怪談2 鬼戀人]]》獲得[[百想藝術大賞|百想藝術大賞]]最佳新人導演獎。2006年，以《家族的誕生》獲電影界矚目<ref>[http://yule.sohu.com/20061215/n247074671.shtml 金泰勇《家族的誕生》獲得最佳導演獎]</ref>。

== 執導作品 ==
*1999年《[[女高怪談2_鬼戀人|女高怪談2 鬼戀人]]》
*2006年《家族的誕生》
*2009年《視線1318》
*2010年《[[晚秋|晚秋]]》
*2012年《你何止美麗》─短片
*2014年《與神同行》

== 獎項 ==
*2000年[[百想藝術大賞|百想藝術大賞]]最佳新人導演（《女高怪談2 鬼戀人》）
*2006年[[青龍獎|青龍獎]]最佳導演（《家族的誕生》）
*2006年[[大鐘獎|大鐘獎]]最佳編劇（《家族的誕生》）
*2007年[[青龍獎|青龍獎]]最佳影片（《世界》）
*2007年百想藝術大賞最佳電影導演（《家族的誕生》）
*2007年百想藝術大賞最佳電影編劇（《家族的誕生》）

== 個人生活 ==
2013年10月，金泰勇在拍攝[[廣告|廣告]]期間，與曾合作電影《[[晚秋|晚秋]]》的演員[[湯唯|湯唯]]成為戀人。2014年7月2日，湯唯所屬的经纪公司发布声明，声明中表示湯唯与韩国导演金泰勇已订婚。2014年7月12日，金泰勇在[[瑞典|瑞典]]和湯唯完婚。
