'''金雞納樹屬'''（[[学名|学名]]：{{Lang|la|'''''Cinchona'''''}}，又稱作'''雞納樹'''、'''金雞勒'''、'''奎寧樹'''）為[[茜草科|茜草科]]的一[[屬|屬]]，約包含25種的[[物種|物種]]，树皮和根皮是提取[[奎寧|奎寧]]和[[奎尼丁|奎尼丁]]的重要工業原料。

== 形态 ==
一般为常绿小[[喬木|喬木]]，高度約3米。矩圓狀披針形或橢圓狀矩圓形的葉片以[[對生|對生]]的方式生長，一般有7至12厘米的長度；植物長有乳白色或玫瑰色的筒狀小花，長1厘米，末端為裂片披針形，邊緣披白色長柔毛，花朵皆是[[聚傘花序|聚傘花序]][[腋生|腋生]]或[[頂生|頂生]]，常爲圓錐花的序式排列，有強烈的氣味；[[蒴果|蒴果]]為橢圓形，長度大概為12-{毫米}-，有開裂的室間；種子形狀細小，有翅作為[[風媒|風媒]]之用。

== 分佈 ==
原產於[[南美洲|南美洲]]，最初在[[安第斯山脈|-{zh-cn:安第斯; zh-hk:安第斯; zh-tw:安地斯}-山脈]]東面[[海拔|海拔]]900至2700米的山谷以內所被發現。到了19世紀中期，金雞納樹被移植至[[東南亞|東南亞]]一帶栽培，而[[印尼|印尼]]為最早引種的國家。1933年，在[[臺灣|臺灣]]及[[中國|中國]][[廣東|廣東]]、[[雲南|雲南]]以也開始有引種。而[[臺灣大學|臺灣大學]]農學院實驗林管理處旗下所管轄的[[鳳凰自然教育園區|鳳凰自然教育園區]]則栽種著[[台灣日治時期|日治時期]][[日本|日本人]]所開闢種植金雞納樹，面積共約三十餘[[公頃|公頃]]。同樣地，[[扇平|扇平]]亦被規劃為日本[[京都大學|京都大學]]演習林，為金雞納樹試驗的重要據點，當時可由金雞納樹的樹皮提煉奎寧供治療[[瘧疾|瘧疾]]使用，後因奎寧可由化學合成，才漸漸停止金雞納樹的相關作業，而現在可在扇平地區發現大批金雞納樹林，並已可在扇平天然更新。

== 用途 ==
[[File:Quinine-2D-skeletal.png|right]]，金雞納生物鹼主要成份]]
金雞納的樹皮和根皮皆蘊藏30多種[[生物鹼|生物鹼]]，其中以[[奎寧|奎寧]]最多，其次為[[奎尼丁|奎尼丁]]、[[辛可寧|辛可寧]]、[[辛可尼丁|辛可尼丁]] 、[[金雞納鞣酸|金雞納鞣酸]]、[[奎寧酸|奎寧酸]]和[[金雞納紅|金雞納紅]]等等化合物。而奎寧具有對[[瘧疾|抗瘧]]的藥效，可以[[提煉|提煉]]成為[[製劑|製劑]]能消滅各類還在[[裂殖體|裂殖體]]階段的[[瘧原蟲|瘧原蟲]]，令瘧疾停止發作，其中對[[間日|間日]]瘧療效尤其有效。
