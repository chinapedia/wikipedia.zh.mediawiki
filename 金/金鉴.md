'''金鉴'''（{{bd|1932年|10月||}}）[[满族|满族]]，[[北京市|北京市]]人，[[中国共产党|中国共产党]]及[[中华人民共和国|中华人民共和国]]官员<ref name=xhw>{{Cite web|url =http://news.xinhuanet.com/misc/2002-02/28/content_293747.htm|title =金鉴简历|author =|date =2002-02-28|publisher =新华网|language =|accessdate =|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20061018021134/http://news.xinhuanet.com/misc/2002-02/28/content_293747.htm|archivedate =2006-10-18}}</ref>。

==生平==
1948年，金鉴加入中国共产党。[[中华人民共和国|中华人民共和国]]成立后，历任[[中国新民主主义青年团|中国新民主主义青年团]]北京市东单区工委书记，[[中国共青团|中国共青团]]北京市委宣传部、组织部副部长，[[中共中央组织部|中共中央组织部]]副处长，[[中共北京市委|中共北京市委]]常委、市委副秘书长、组织部副部长，中国劳改学会第一届会长。1980年2月至1982年6月，任中国共青团北京市委书记。1984年7月至1987年12月，任中共北京市委副书记。1988年4月至1993年6月，任[[中华人民共和国司法部|中华人民共和国司法部]]副部长。金鉴是中共第十二、十三届中央候补委员，第八、九届全国政协常委，第八届全国政协民族委员会主任。第九届全国政协民族和宗教委员会主任<ref name=xhw/>。金鉴还曾任中国人权研究会副会长<ref>{{Cite web |url =http://www.gmw.cn/01gmrb/2000-05/30/GB/05%5E18436%5E0%5EGMA1-310.htm  |title = 中国人权研究会举行新一届理事会会议 |author =  |date = 2000-05-30 |publisher =光明网  |language =  |accessdate =  |quote =  }}</ref>、中国监狱学会会长<ref>{{Cite web |url =http://www.moj.gov.cn/jyglj/content/2004-03/18/content_84194.htm?node=246  |title = 中国监狱学会会长金鉴在服刑人员普法教育培训班上的讲话 |author =  |date = 2004-03-18 |publisher = 司法部监狱管理局 |language =  |accessdate =  |quote =  }}</ref>、全国党建研究会顾问<ref>{{Cite web|url = http://www.dzdj.gov.cn/n852155/n854853/c1696737/content.html|title = 全国党建研究会召开党内教育片《居安思危》座谈会|author = |date = 2008-10-30|publisher = 中共德州市委组织部|language = |accessdate = |quote = |archive-url = https://web.archive.org/web/20170103113545/http://dzdj.gov.cn/n852155/n854853/c1696737/content.html|archive-date = 2017-01-03|dead-url = yes}}</ref>。
