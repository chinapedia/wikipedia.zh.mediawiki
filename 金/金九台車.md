'''金九台車'''是[[金瓜石|金瓜石]]至[[九份|九份]]地區的一地面式纜車計畫。由於金九地區一帶地形特殊，當地政府於是計劃興建地面式纜車，全長約１公里多，沿著舊有的台車路線，將金瓜石與九份兩處觀光勝地串連起來，起點於[[水湳洞|水湳洞]]，終點至[[勸濟堂|勸濟堂]]。

== 背景 ==

金瓜石與九份地區，因腹地有限及地形變化過大的關係，導致道路在開發上不易，每逢假日觀光人潮雍現，必造成交通堵塞，一位難求的困境，也讓當地的空氣品質惡化。台北縣政府配合「魅力水金九觀光城計畫」，仿效[[歐洲|歐洲]]的Doppelmayr系統，積極推動金九地區的觀光產業。

== 興建期程及計劃中止 ==
原定自2007年起，2011年完成。但因土地汙染問題，最後於2010年3月中止計畫，並未完成興建。<ref>[http://tw.nextmedia.com/applenews/article/art_id/32386143/IssueID/20100325 土地污染 金九台車案恐夭折] 蘋果日報 2010年3月25日 {{webarchive|url=https://web.archive.org/web/20100529065544/http://tw.nextmedia.com/applenews/article/art_id/32386143/IssueID/20100325 |date=2010年05月29日 }} </ref><ref>[http://www.libertytimes.com.tw/2011/new/may/26/today-taipei1.htm 瑞芳台車夢碎 規劃早喊卡 金九居民空等待] 自由電子報 2011年5月26日</ref>

== 特色 ==

沿線地形變化大，利用舊有台車路線且須穿越兩段既有之礦石運輸窄小坑道，提供旅客礦區復古知性之旅，六坑平台景觀工程提供遊客可遠眺廢煙道、十三層等遺跡及[[陰陽海|陰陽海]]等自然景觀。
