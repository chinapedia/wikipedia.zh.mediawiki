'''金坦波瀑布'''（Kintampo waterfalls）為[[迦納|迦納]]最高的[[瀑布|瀑布]]之一。[[英屬黃金海岸|英屬黃金海岸]]時期稱之'''桑德斯瀑布'''（Sanders Falls），該瀑布坐落在[[黑沃爾特河|黑沃爾特河]]的支流蓬蓬河（Pumpum river）上、該地首府[[金坦波|金坦波]]北方約{{Convert|4|km|mi}}處。金坦波瀑布為該地區主要的自然景點之一，隱密在森林之中，金坦波瀑布主要由三個瀑布所組成，其中一個瀑布最高有{{Convert|25|m|ft}}的落差。加上三個瀑布高度後，所在河域總計約有{{Convert|70|m|ft}}的落差<ref>{{Cite web |url=http://www.ghana.photographers-resource.com/locations/Landscape/LG/Kintampo_waterfall.htm |title=Kintampo Waterfall |publisher=ghana.photographers-resource.com }}</ref><ref>{{Cite web |url=http://www.ghana.travel/touring_ghana/eco_tourism/waterfalls/kintampo_waterfalls/ |title=Kintampo Waterfalls |publisher=Ghana Tourism Authority |deadurl=yes |archiveurl=https://web.archive.org/web/20120617190509/http://www.ghana.travel/touring_ghana/eco_tourism/waterfalls/kintampo_waterfalls/ |archivedate=2012-06-17 }}</ref><ref>{{Cite web |url=http://www.ghanaexpeditions.com/regions/highlight_detail.asp?id=&rdid=44 |title=Kintampo Waterfalls |work=Wildlife and Nature Reserves |publisher=ghanaexpeditions.com }}</ref><ref>{{Cite web |url=http://kintamponorth.ghanadistricts.gov.gh/?arrow=atd&_=37&sa=321 |title=tourism attractions |work=about this district |publisher=Kintampo North Municipal Assembly }}</ref>。
