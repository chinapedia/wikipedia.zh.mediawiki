'''金铁主义'''是[[清朝|清朝]]末年[[杨度|杨度]]指出的政治理念，主张清政府召开[[国会|国会]]，实施[[君主立宪|君主立宪]]制。在杨度的心目中，西方先进国家的根源不在民权、宪政而在金铁主义，民权与立宪只是达到“金铁主义”的手段。 他认为，西方列强之所以披靡世界，主要依靠经济之势力与军事之势力，而经济势力和军事实力二者为一的[[经济军国主义|经济军国主义]]被杨度称之为“金铁主义”。<ref>{{Cite web |url=https://books.google.com/books?id=Mp4yAQAAIAAJ&q=%E9%87%91%E9%93%81%E4%B8%BB%E4%B9%89&dq=%E9%87%91%E9%93%81%E4%B8%BB%E4%B9%89&hl=zh-CN&sa=X&ved=0ahUKEwjE8J_e9NnfAhVUFogKHQCGCf0Q6AEIPTAE|title=近代中国思维方式演变的趋势|date=|publisher=Google图书}}</ref>原文为《[[金铁主义说|金铁主义说]]》，连载于《[[中国新报|中国新报]]》第一至第六期，反应了作者当时较[[系统|系统]]化的[[政治思想|政治思想]]。他在此文中提出“[[五族共和|五族共和]]”这一理念与设想，鼓吹[[五族立宪论|五族立宪论]]，宣传“五族合一”、“五族一家”。<ref>{{Cite web |url=http://www.cnki.com.cn/Article/CJFDTotal-GDSK200405033.htm|title=孙中山与辛亥革命时期的“五族共和”论|date=|publisher=CNKI}}</ref>

==思想起源==
杨度的“金铁主义”受启发于[[德意志帝国|德意志帝国]]宰相[[奥托·冯·俾斯麦|奥托·冯·俾斯麦]]奉行的“铁血主义”政策，杨度认为，中国是五千年文明古国，[[儒家|儒家]]“[[仁爱|仁爱]]”之学早已经深入人心，若是纯粹提倡德国式的“铁血主义”，则有害于社会进步与[[民智|民智]]开启，所以，必须进行改造，即在发展经济的前提下大力发展军事。“金”指黄金，金钱，引申为经济；“铁”指黑铁，铁炮，引申为军事。<ref>{{Cite web |url=http://news.163.com/17/0616/15/CN2FM8HA00018AOP.html|title=杨度的“金铁主义”|date=2017-06-16|publisher=网易新闻}}</ref>

==社会评价==
《金铁主义说》发表后受到当时[[革命|革命]]派的猛烈抨击。
