'''金龜樹'''（[[學名|學名]]：''{{lang|la|Pithecellobium dulce}}''）<ref name = "台灣行道樹">{{cite book|title=《台灣行道樹圖鑑》|author= |series=陳俊雄、高瑞卿|publisher=貓頭鷹出版|page=第58-59頁|isbn=978-986-6651-02-1|date=2008年3月}}</ref><ref name="嘉義">{{cite encyclopedia|encyclopedia=嘉義蘭潭風景區嘉義大學景觀植物圖鑑|title=金龜樹|volume=|pages=|url=http://ncyugarden.ncyu.edu.tw/show_plant.asp?id=115}}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，別稱'''牛蹄豆'''、'''羊公豆'''、'''洋酸角'''、'''洋皂莢'''、'''甜肉圍誕樹'''及'''甜肉圍涎樹'''等<ref name="NM 物種信息卡"/><ref name="嘉義"/>，為[[豆科|豆科]][[含羞草亞科|含羞草亞科]][[猴耳環屬|猴耳環屬]]植物<ref  name="NM 物種信息卡"/> 。本種開花時的特別香氣，常吸引大量[[金龜子|金龜子]]聚集停棲於此樹，據《[[臺灣通史|臺灣通史]]》第28卷中的記載「金龜樹：以金龜多宿之，故名」<ref>{{cite encyclopedia|encyclopedia=中華文化網|title=台灣通史 第28卷 木之屬|volume=|pages=|url=http://www.chinapage.com/big5/history/taiwan3.htm|deadurl=yes|archiveurl=https://web.archive.org/web/20111108024402/http://www.chinapage.com/big5/history/taiwan3.htm|archivedate=2011-11-08}}</ref><ref>{{cite encyclopedia| encyclopedia=吉林醫藥學院大學生綜合素質教育網| title=卷二十八 虞衡志 木之屬| volume=| pages=| url=http://www.chinapage.com/big5/history/taiwan3.htmhttp://china-culture.jlmpc.cn/%E5%85%B6%E4%BB%96%E5%8E%86%E5%8F%B2%E4%B9%A6%E7%B1%8D/%E5%8E%86%E5%8F%B2/%E4%B8%93%E5%8F%B2/%E5%8F%B0%E6%B9%BE%E9%80%9A%E5%8F%B2/Resource/Book/Edu/JXCKS/TS011054/0012_ts011054.htm}}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>；或因本種特殊的二大二小[[複葉|複葉]]，遠看像是一隻展翅飛舞的小金龜，故而得名金龜樹<ref name = "台灣行道樹"/>。

==分佈==

本種原產於[[墨西哥|墨西哥]]<ref name="台北">{{cite encyclopedia | encyclopedia=台北植物園資訊網 植物資料庫 | title= 金龜樹Pithecellobium dulce (Roxb.) Benth.   | volume= |pages=|url=http://tpbg.tfri.gov.tw/plants/plants_info.php?rid=999}}</ref>，現分佈於[[北美洲|北美洲]][[墨西哥|墨西哥]]；[[中美洲|中美洲]]<ref name="中國">{{cite encyclopedia| encyclopedia=中國植物誌| title=牛蹄豆| volume=39| pages=52| url=http://v2.cvh.org.cn/zhiwuzhi/page/39/052.PDF}}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>的[[洪都拉斯|洪都拉斯]]、[[尼加拉瓜|尼加拉瓜]]、[[危地馬拉|危地馬拉]]、[[薩爾瓦多|薩爾瓦多]]；[[南美洲|南美洲]]的[[哥倫比亞|哥倫比亞]]、[[委內瑞拉|委內瑞拉]]<ref name="GRIN"/>；[[亞洲|亞洲]]的[[中國大陆|中國大陆]]、[[台灣|台灣]]等熱帶乾旱地帶<ref name="中國"/>。[[中國|中國]]國內分佈於[[廣西|廣西]]、[[廣東|廣東]]、[[香港|香港]]<ref>{{cite encyclopedia | encyclopedia=香港植物標本室. | title= Pithecellobium dulce (Roxb.) Benth.| volume= | pages=|url= http://www.hkherbarium.net/Herbarium/c_search_end.asp?id=3342}}</ref>、[[福建|福建]]、[[雲南|雲南]]等省份<ref name="中國"/><ref name="福建">{{cite encyclopedia| encyclopedia=福建植物誌| title=牛蹄豆| volume=3| pages=2-3| url=http://v2.cvh.org.cn/difangzhi/fujian/page/3/002.pdf}}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。多生於曠野或林邊<ref name="海南">{{cite encyclopedia| encyclopedia=海南植物誌| title=牛蹄豆| volume=2| pages=205| url=http://v2.cvh.org.cn/difangzhi/hainan/page/2/205.pdf}}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。人工栽培可透過播種或扦插法繁殖<ref>{{cite encyclopedia | encyclopedia=烏山頭水庫風景區 生態介紹 | title= 金龜樹  | volume= |pages=|url=http://wusanto.magicnet.com.tw/introitem_list.php?seriesid=28}}</ref>。

==用途==

金龜樹的[[樹皮|樹皮]]含[[單寧|單寧]]30%<ref name="台北"/><ref name="中國"/>可供作黃皮[[染料|染料]]<ref name="嘉義"/>，[[木材|木材]]可供製作一般建築用材及箱板之用<ref name="中國"/>，心材可供製作[[車輛|車輛]]及[[農具|農具]]<ref name="台北"/>，[[葉|葉]]及[[莢果|莢果]]可供作飼料之用<ref name="台北"/><ref name="中國"/>，[[假種皮|假種皮]]於[[墨西哥|墨西哥]]供製作[[檸檬|檸檬]]水之用<ref name="中國"/>，常栽種作行道樹、綠籬、庭院植物<ref name="嘉義"/>，亦因本種耐[[鹽|鹽]]、耐風常栽種作海岸定砂之用<ref name = "台灣行道樹"/><ref name="國立中正">{{cite encyclopedia | encyclopedia=國立中正大學 校內電子地圖查詢系統 | title= 金龜樹(Pithecellobium dulce) | volume= |pages=|url=http://ccuweb.ccu.edu.tw/gis/showPlant.php?globalno=24}}</ref>。

==形態特徵==

金龜樹是一種常綠[[喬木|喬木]][[植物|植物]]，中等大，樹形不規則，高約8－15[[米|米]]<ref name = "台灣行道樹"/>。[[樹皮|樹皮]]密生皮孔，灰白色<ref name="台北"/>；[[莖|莖]]表面塊裂狀，彎彎曲曲<ref name = "台灣行道樹"/>，常具瘤狀突起<ref name="嘉義"/>，有明顯的橫紋<ref name = "台灣行道樹"/>；枝條通常下垂，細長柔軟<ref name="國立中正"/>；小枝具一對棘針新芽，嫩芽紅色<ref name = "台灣行道樹"/>。[[木材|木材]]具異味<ref name="台北"/>。<ref name="中國"/>

===葉===
[[File:Leaves_I_IMG_2854.jpg|left]]]]

[[葉|葉]]為[[二回|二回]][[羽狀複葉|羽狀複葉]]<ref name="福建"/>，由4枚小葉片組合成一片複葉<ref name = "台灣行道樹"/>；羽片一對，互生<ref name="嘉義"/>，每一羽片上有小葉一對，小葉對生<ref name="嘉義"/>，在羽片及小葉着生處均具有1－2<ref name="福建"/>枚由[[托葉|托葉]]變態為的針狀刺<ref name = "台灣行道樹"/>；小葉片堅紙質，腎形<ref name="國立中正"/>、橢圓形、卵狀橢圓形<ref name="福建"/>、長倒卵形、斜卵形<ref name="嘉義"/>、斜卵狀橢圓形<ref name="台北"/>或倒卵狀長圓形<ref name="福建"/>，基部略偏斜或鈍<ref name="台北"/>，頂端鈍或微凹<ref name="福建"/>，大小差異甚大，無毛，全緣<ref name="嘉義"/>，[[葉脈|葉脈]]顯著，中脈偏於內側，背面葉脈較表面凸起，長約2－5[[厘米|厘米]]，寬約2－25[[毫米|毫米]]；羽片柄、總[[葉柄|葉柄]]及小葉柄均披柔毛<ref name="福建"/>。<ref name="中國"/>

===花===

[[花|花]]為頭狀[[花序|花序]]，細小，常具有短的總[[花梗|花梗]]<ref name="海南"/>，於枝頂或[[葉腋|葉腋]]處排列成狹圓錐或總狀<ref name="福建"/>[[花序|花序]]式；[[花萼|花萼]]呈[[漏斗|漏斗]]狀，5裂<ref name="福建"/>，密披白色<ref name="福建"/>長柔毛，長約1[[毫米|毫米]]；[[花冠|花冠]][[漏斗|漏斗]]狀<ref name="福建"/>，5深裂<ref name="台北"/><ref name="福建"/>，淡黃、淡綠<ref name="國立中正"/>、白色或綠白色<ref name="嘉義"/>，密披白色<ref name="福建"/>長柔毛，中部以下合生，連[[雄蕊|雄蕊]]長約3－6<ref name="海南"/>[[毫米|毫米]]<ref name="福建"/>；[[雄蕊|雄蕊]]基部合生成單體<ref name="台北"/>，突出<ref name = "台灣行道樹"/>；[[花絲|花絲]]長約8－10[[毫米|毫米]]<ref name="中國"/>。

===果===
[[File:Starr_070206-4139_Pithecellobium_dulce.jpg|left]]]]

[[果|果]]為[[莢果|莢果]]，條形<ref name="福建"/>，呈[[唸珠|唸珠]]狀螺旋捲曲<ref name="嘉義"/>，膨脹，成熟時淡紅色<ref name="國立中正"/>扭曲開裂<ref name = "台灣行道樹"/>，長約10－14[[厘米|厘米]]<ref name="福建"/>，寬約1[[厘米|厘米]]；內藏[[種子|種子]]6－8枚<ref name="台北"/>；[[種子|種子]]黑色，包於粉紅或白色的肉質[[假種皮|假種皮]]內<ref name="中國"/>，[[種柄|種柄]]絲狀<ref name="福建"/>，具有白色的[[種臍|種臍]]<ref name="海南"/>。<ref name="中國"/>
