{{japanese|japanese=金鵄勳章|kana=きんしくんしょう|romaji=Kinsihgunshō}}
'''金鵄勳章'''是[[大日本帝國|大日本帝國]]對[[大日本帝國陸軍|陸軍]]、[[大日本帝國海軍|海軍]][[軍人|軍人]]、[[軍屬|軍屬]]所授與的一種[[勳章_(日本)|勳章]]。「金鵄」的名稱由來，是根據[[日本神話|日本神話]][[傳說|傳說]]中，[[神武天皇|神武天皇]][[神武東征|東征]]時，神武天皇的[[弓|弓]]停了一隻[[金鵄|金鵄]]，迷惑了[[長髓彥|長髓彥]]軍。

== 概要 ==
[[File:kinshi4.jpg|thumb]]
[[File:Prince_Fushimi_Sadanaru.jpg|thumb]]]]
[[File:Kawamura_Kageaki.jpg|thumb]]及金鵄勳章（功一級副章）[[元帥|元帥]][[陸軍大將|陸軍大將]][[川村景明|川村景明]]]]
金鵄勳章於1890年2月11日（[[紀元節|紀元節]]）制定。
金鵄勳章分為7個等級，「功一級」向下至「功七級」以'''功級'''表示的勳章。[[位階|位階]]與[[瑞寶章|瑞寶章]]或其他勳章授與工作取得一定年資為條件的[[公務員|公務員]]（當時為[[官吏|官吏]]）或民間人士不同，金鵄勳章只授與有戰功的軍人，[[大將|大將]]與[[皇族|皇族]]軍人若沒有戰功也不會被授與。（即使以[[將軍|將軍]]與[[提督|提督]]的身分獲得[[旭日章|旭日章]]或[[瑞寶章|瑞寶章]]等[[勳一等|勳一等]]或[[勳二等|勳二等]]的受勳，對金鵄章而言可能只是[[功五級|功五級]]或[[功四級|功四級]]的軍人而已。）

另外，功一級至功七級的受章者有900[[日圓|日圓]]至65日圓的年金給付。[[昭和|昭和]]初期，當時的[[二等兵|二等兵]]的月薪只有8日圓80錢，因此算是相當高額。由於此年金為終身年金，隨著戰爭擴大也擴大了受章者數量，使[[國庫|國庫]]負擔急促增加。因此1940年改以發行[[公債|公債]]的方式支付，但是因為日本戰敗使得該公債變成一元不值。另外，[[昭和15年|昭和15年]]最後因為戰爭激烈，中止授與生還者制度，變成只授與取得戰功的[[陣亡|陣亡]]者。因此前線部隊對於功勳彪炳都會以「金鵄勳章的約定」給予[[軍刀|軍刀]]與[[感謝狀|感謝狀]]、紀念品等，陸軍方面也有制定相對應的武功勳章。

[[太平洋戰爭|太平洋戰爭]]中，日本戰敗。根據[[駐日盟軍總司令|駐日盟軍總司令]]的指示，[[昭和|昭和]]22年（1947年）5月3日依據「昭和22年政令第四號」公告廢止。

在戰前的日本是個重視階級的社會，列出勳章的等級是常見的事情。金鵄勳章也不例外會被表示出來，表示順序為：1.職務、2.階級、3.位階、4.勳等、5.功級、6.爵位、7.學位以及姓名。
*例如：「內閣總理大臣陸軍大將正一位大勳位功一級公爵[[山本權兵衛|山本權兵衛]]」。
*例如：「攝政皇族大勳位功一級法學博士常陸宮正仁親王」

=== 功級 ===
金鵄勳章都會跟著表示軍人功績的等級。與[[位階_(日本)|位階]][[勳等|勳等]]的[[榮典|榮典]]並列為[[天皇|天皇]]授與的榮譽之一。各級正式名稱就像「功一級金鵄勳章」一樣以【等級 + 金鵄勳章】的形式稱呼。功二級以下的金鵄勳章不設副章。
# [[功一級|功一級]]:針對天皇直屬部隊將官（主要為大將、中將）的司令官特別審議之後授與。大綬由左肩垂掛至右腋下，副章佩戴於左肋。
# [[功二級|功二級]]:某個有功勞的將官，是佐官最高位的功級。正章（即功一級金鵄勳章之佩星）佩戴於右肋。
# [[功三級|功三級]]:將官初敘的功績。某個有功勞的佐官。是尉官最高位的功級。頸綬。
# [[功四級|功四級]]:[[佐官|佐官]]初敘的功績。有功勞的尉官。准士官、下士官的最高位的功級。加結襟綬，佩戴於左胸。
# [[功五級|功五級]]:[[尉官|尉官]]初敘的功績。准士官、下士官中功勞比較重的功級。兵的最高位的功級。襟綬，佩戴於左胸。
# [[功六級|功六級]]:[[准尉|准士官]]、[[下士官|下士官]]初敘的功績。有功勞的兵的功級。襟綬，佩戴於左胸。
# [[功七級|功七級]]:[[兵|兵]]初敘的功績。襟綬，佩戴於左胸。

== 後期的金鵄勳章復權運動 ==
1963年生還者敘勳制度重新開始，[[大勳位菊花章|菊花章]]、[[旭日章|旭日章]]、[[瑞寶章|瑞寶章]]、[[寶冠章|寶冠章]]都恢復了，但禁止於公共場所配戴已廢止的金鵄勳章。「金鵄聯盟」以恢復金鵄勳章敘勳者的名譽與年金為訴求展開活動。1985年招集[[國會議員|國會議員]]召開「有關舊勳章名譽回復座談會」，也對當時的首相[[中曾根康弘|中曾根康弘]]做出相同請求。但是隨著敘勳者的高齡化，近年活動已漸漸式微。
