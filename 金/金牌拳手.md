{{作品名稱|name=''Creed''|hk=洛奇外傳：王者之後|tw=金牌拳手|cn=奎迪}}是一部2015年美國[[運動片|運動]][[劇情片|劇情片]]，由[[萊恩·庫格勒|萊恩·庫格勒]]執導並與{{link-en|亞倫·柯文頓|Aaron Covington}}共同編劇，而{{link-en|羅伯·查特夫|Robert Chartoff}}、{{link-en|查爾斯·溫克勒|Charles Winkler}}和[[席維斯·史特龍|席維斯·史特龍]]則負責監製。

本片為「{{link-en|洛基系列電影|Rocky (film series)}}」的第七部作品，也是2006年的電影《[[洛基：勇者無懼|洛基：勇者無懼]]》的衍生作品。由[[麥可·B·喬丹|麥可·B·喬丹]]主演{{link-en|阿波羅·克里德|Apollo Creed}}之子阿多尼斯·克里德，史特龍再度飾演他{{link-en|洛基·巴波亞|Rocky Balboa}}的角色。導演曾和演員喬丹合作過《[[奧斯卡的一天|奧斯卡的一天]]》；此外，喬丹也和[[伍迪·哈里斯|伍迪·哈里斯]]共同合作過《[[火線重案組|火線重案組]]》。主要拍攝於2015年1月19日在[[利物浦|利物浦]]進行，後來在[[費城|費城]]取景洛基的家。

電影於2015年11月25日在美國上映，以纪念《[[洛基_(電影)|洛基]]》上映四十週年。

== 劇情 ==
故事敘述世界重量級冠軍{{link-en|阿波羅·克里德|Apollo Creed}}的兒子阿多尼斯獨自前往[[費城|費城]]請知名拳王{{link-en|洛基·巴波亞|Rocky Balboa}}來擔任教練，並接受嚴格的訓練來成為拳擊冠軍。

== 反響 ==
《金牌拳手》上映後好評如潮<ref>{{cite web |url=http://www.latimes.com/entertainment/movies/la-et-mn-rocky-creed-production-ryan-coogler-20151123-story.html |title= Getting the 'Rocky' spinoff 'Creed' made was a real underdog story for director Ryan Coogler |website=latimes.com|accessdate=2015-11-25}}</ref>。[[爛番茄|爛番茄]]根據146篇專業評論新鮮度高達92%，平均得分7.9／10；該網站共識：「《金牌拳手》所帶來了《洛基》系列一個令人出乎意料深刻的第七集，儘管堅持走其經典的前身的根本，而在有趣的新方向中延伸了拳擊手的傳奇」<ref>{{cite web |url=http://www.rottentomatoes.com/m/creed_2015/?search=Creed |title= Creed (2015) |website=Rotten Tomatoes |accessdate=2015-11-27}}</ref>。[[Metacritic|Metacritic]]得分82，代表「普遍好評」<ref>{{cite web |url=http://www.metacritic.com/movie/creed |title=Creed reviews |website=Metacritic |accessdate=2015-11-26}}</ref>。據[[CinemaScore|CinemaScore]]調查，觀眾在A+至F之間平均給了「A」的評價<ref name="CinemaScore">{{cite web|url=http://deadline.com/2015/11/thanksgiving-box-office-hunger-games-mockingjay-creed-the-good-dinosaur-1201639941/ |title=Katniss Rules Wednesday On Track For $78M-80M 5-Day; ‘Good Dinosaur’ Eyes $58M-$62M; ‘Creed’ Punching $39M-$42M |author=Anthony D'Alessandro|work=''[[Deadline.com|Deadline.com]]''|accessdate=2015-11-26}}</ref>。

美國首週以600萬美元的票房登當週第三。
