'''金花石蒜'''（[[学名|学名]]：{{sname|''Lycoris aurea''}}），又名'''铁色箭'''（本草纲目）、'''忽地笑'''、'''龍爪花'''、'''黄花石蒜'''、山水仙、山金針，为[[石蒜科|石蒜科]][[石蒜属|石蒜属]]的植物。{{r|張蕙芬-2006-p78}}

== 形态 ==
[[多年生草本|多年生草本]]，广卵形[[鳞茎|鳞茎]]{{r|張蕙芬-2006-p78}}；秋季抽叶，长椭圆状带形或带形叶；花茎顶生伞状花序，有5-10花{{r|張蕙芬-2006-p78}}，夏季开黄色花，花被裂片倒披针形，花被管长约1.5厘米；[[蒴果|蒴果]]近球形。

== 分布 ==
分布于[[日本|日本]]、[[台湾|台湾]]{{r|張蕙芬-2006-p78}}<ref name="sa-tw-2011-L">{{cite web|url=http://sa.ylib.com/MagCont.aspx?Unit=columns&id=1861|title=妝點馬祖的石蒜|author1=黃士元|author2=郭曜豪|author3=蔡錫鍊|date=2011|accessdate=2016-01-22|publisher=[[科学美国人|-{zh-cn:科学美国人;zh-tw:科學人;}-]]|deadurl=yes|archiveurl=https://web.archive.org/web/20160127092117/http://sa.ylib.com/MagCont.aspx?Unit=columns&id=1861|archivedate=2016-01-27}}</ref>、[[缅甸|缅甸]]以及[[中国大陆|中国大陆]]的[[湖北|湖北]]、[[湖南|湖南]]、[[广西|广西]]、[[云南|云南]]、[[广东|广东]]、[[福建|福建]]、[[四川|四川]]等地，生长于海拔600米至2,300米的地区，自生山野，见于阴湿山坡。

== 用途 ==
[[File:Lycoris_aurea-jp.jpg|thumb]]
可供观赏；鳞茎有毒，除了可以提取石蒜碱供药用，也能提取淀粉工业用。{{-}}
