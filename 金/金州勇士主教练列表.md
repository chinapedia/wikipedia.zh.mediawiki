[[金州勇士|金州勇士]]是一支位于[[美国|美国]][[加利福尼亚州|加利福尼亚州]][[奥克兰_(加利福尼亚州)|奥克兰]]的[[NBA|NBA]]篮球队，分属于[[西部聯盟_(NBA)|西部联盟]]的[[NBA太平洋赛区|太平洋赛区]]，主场为[[甲骨文體育館|甲骨文体育馆]]。其前身费城勇士于1946年加入全美篮球协会（BAA），即后来的NBA。[[埃迪·戈特利伯|埃迪·戈特利伯]]带领球队在同年夺得BAA冠军。勇士于{{nbay|1949}}赛季加入NBA，战绩仅为26胜42负，在季后赛首轮负于[[锡拉丘兹民族|锡拉丘兹民族]]<ref name="194950season">{{cite web|language=en|url=http://www.basketball-reference.com/leagues/NBA_1950.html|title=1949-50 NBA赛季总结|publisher=Sports Reference LLC|work=basketball-reference.com|accessdate=2008-06-07}}</ref>。[[弗兰克林·缪利|弗兰克林·缪利]]和[[大來國際|大來國際]]在{{nbay|1962}}赛季时组织旧金山地区的一个投资团体出资将勇士队搬迁至[[旧金山|旧金山]]。{{nbay|1971}}赛季时球队迁至[[奥克兰_(加利福尼亚州)|奥克兰]]，更名为金州勇士<ref name="warriorshistory">{{cite web|language=en|url=http://www.basketball-reference.com/teams/GSW/|title=金州勇士历史|publisher=Sports Reference LLC|work=basketball-reference.com|accessdate=2008-06-07}}</ref><ref name="mieuli">{{cite news|date=2005-05-04|url=http://www.sfgate.com/cgi-bin/article.cgi?file=/c/a/2005/05/04/SPGAVCJO0N1.DTL|title=There's still more to Mieuli than meets the eye|publisher=San Francisco Chronicle|accessdate=2008-06-07| first=David | last=Bush}}</ref>。

自从1946年成立以来，一共有25人担任过勇士的主教练。[[乔治·塞内斯基|乔治·塞内斯基]]于{{nbay|1956}}赛季率当时的费城勇士获得队史首个[[NBA总冠军|NBA总冠军]]。<ref name="warriorshistory"/> [[艾尔·阿特尔斯|艾尔·阿特尔斯]]于{{nbay|1974}}赛季率队第二次夺冠，他曾在勇士以球员和教练身份效力过25个赛季<ref name="warriorshistory"/><ref name="attles"/><ref name="attlesplayer">{{cite web|language=en|url=http://www.basketball-reference.com/players/a/attleal01.html|title=艾尔·阿特尔斯球员数据统计|publisher=Sports Reference LLC|work=basketball-reference.com|accessdate=2008-06-07}}</ref>，也是队史常规赛和季后赛胜场最多的教练。现任主帅[[史蒂夫·科尔|史蒂夫·科尔]]是队史[[胜率|胜率]]最高的<ref name="coaches">{{cite web|language=en|url=http://www.basketball-reference.com/teams/GSW/coaches.html|title=金州勇士主教练|publisher=Sports Reference LLC|work=basketball-reference.com|accessdate=2008-06-07}}</ref>，曾于{{nbay|2014}}赛季率队时隔40年再次夺得总冠军<ref name="kerr">{{cite web|language=en|url= http://www.basketball-reference.com/coaches/kerrst01c.html |title=史蒂夫·科尔执教记录|publisher=Sports Reference LLC|work=basketball-reference.com|accessdate=2017-03-24}}</ref>。[[弗兰克·麦圭尔|弗兰克·麦圭尔]]、[[亚历克斯·汉纳姆|亚历克斯·汉纳姆]]、[[唐·尼尔森|唐·尼尔森]]和[[比尔·沙尔曼|比尔·沙尔曼]]入选了[[奈史密斯篮球名人纪念堂|奈史密斯篮球名人纪念堂]]<ref name="mcguire"/><ref name="hannum"/><ref name="sharman"/>。汉纳姆、尼尔森和科尔在勇士执教期间各获得一次[[NBA年度最佳教练|NBA年度最佳教练]]<ref name="hannum"/><ref name="nelson"/>，尼尔森还被提名为[[NBA50大巨星#NBA历史十大教练|NBA历史十大教练]]之一<ref name="topten"/>。阿特尔斯、约翰斯顿、[[乔治·李|乔治·李]]和塞内斯基这4名前勇士球员曾担任过该队主教练<ref name="attlesplayer"/><ref name="johnstonplayer">{{cite web|language=en|url=http://www.basketball-reference.com/players/j/johnsne01.html|title=尼尔·约翰斯顿球员数据统计|publisher=Sports Reference LLC|work=basketball-reference.com|accessdate=2008-06-07}}</ref><ref
