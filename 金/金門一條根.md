'''金門一條根'''是[[中華民國|中華民國]][[金門縣|金門縣]]的特產之一，產於金門珠山村，為金門原生藥用作物，是種多年生豆科植物，且有一條主根因而得名。

== 一条根名称来源 ==
一條根是種多年生豆科植物，因只有一條主根直伸入土並少有支根或鬚根而得其名。

== 說明 ==
金門一條根（野生大豆）共有三種。
1. 澎湖大豆
2. 蔓草蟲豆
3. 闊葉大豆

== 特色 ==
一條根具有平肝、健體止盜汗、強壯等功能。

== 用途 ==
能用於治風濕症、強筋骨、壯身體。

== 人工栽種 ==
經金門縣農業試驗推廣之品種為闊葉大豆(Glycine tomentella Hayata.)。
