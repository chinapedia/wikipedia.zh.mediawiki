'''金門守禦千戶所城'''位於今[[中華民國|中華民國]][[福建省_(中華民國)|福建省]][[金門縣|金門縣]]，又稱'''金門城'''，是明朝[[衛所制|衛所制]]下隸屬[[永宁卫_(福建)|永寧衛]]{{NoteTag|name=許|1=《明代海禁政策下的金門及其海域》（65、66頁）一文根據後述福建都指揮使參與金門城增築事宜，認為金門千戶所除隸屬永寧衛外，也有可能直屬於福建都司，或是兩方都有管轄權。{{r|許}}}}的金門守禦千戶所（簡稱金門所）之城池，由江夏侯[[周德興|周德興]]於明洪武二十年（1387年）四月到二十一年（1388年）二月間所築成{{r|張}}{{r|楊}}，但《滄海紀遺》（1568年，洪受）載說：「金門千戶所城……洪武二十五年，江夏侯周得興來築……」，而《明史》〈卷九十一〉「兵三」亦記載這些衛所設置於洪武二十一年<ref> 《明史》〈卷九一．兵制三〉</ref>，而許志仁於《明代海禁政策下的金門及其海域》中認為金門城可能是到洪武二十五年（1392年）才落成{{r|許}}。而因為浯洲嶼上建了此城，島名遂改為金門<ref>[http://www.kinmen.gov.tw/Layout/main_ch/ArtHtml_Show.aspx?ID=4facbade-55b0-4af8-a41c-b41d35d9e436&path=10931 金門縣政府——認識金門]</ref>。

金門守禦千戶所原額操海屯種等旗軍，共一千五百三十五名<ref name="金門守禦千戶所城"/><ref>駐屯人數為建置初期的人力</ref><ref> 《泉州府志》〈卷十．武衛志上．衛所旗軍、弓兵〉</ref>。

金門城，在浯洲之南，離縣城八十里，水程一百里，一溯可至。北阻山東，西南阻海<ref> 清《金門志》〈卷三‧規制志〉，1960，頁 49</ref>。城址位居島之西南高地，海拔在六十公尺以上，居高臨下，極目東南。城之南有南磐山，背城面海，氣象萬千，為備海要地，明都督俞大猷守金時，題其石曰「虛江嘯臥」<ref>清《金門志》〈卷一‧分域略〉，1960，頁 9-10</ref>。

滿清與[[鄭成功|鄭成功]]、[[鄭經|鄭經]]交戰時，金門為鄭家的勢力範圍，兩方交戰期間金門城受到破壞，1680年原本在[[三藩之亂|三藩之亂]]期間為鄭經所控制的金門再次被清軍攻佔後，總兵[[陳龍|陳龍]]一度修築城垣，但後來總鎮署遷到後浦，金門城因而沒落{{r|張}}。

1949年後，中華民國政府遷至臺灣，[[金門|金門]]與[[馬祖|馬祖]]成為[[中華民國|中華民國]]與[[中華人民共和國|中華人民共和國]]對峙的前線，金門守禦千戶所城殘存城垣的花岡岩被中華民國國軍拆除改築其他工事，金門城因而只剩下夯土層與部分地基{{r|張}}。民國八十六年（1997年）起，藉由[[金門酒廠|金門酒廠]]的回饋金，開始重建四方城門，但其形制與原貌有別{{r|張}}，其中重建的南門現為金門酒廠的大門。而民國九十六年（2007年）時[[金門國家公園|金門國家公園]]又在[[文臺寶塔|文臺寶塔]]附近重建了一段古城垣。

== 命名 ==
而金門之得名，始於置守禦千戶所，江夏侯周德興稱此城的形勢為「固若金湯，雄鎮海門」，而得此名<ref>[《金門縣志》〈卷二‧方域〉，1968，頁 106]</ref>。

== 沿革 ==
明朝建立初期，為了防衛[[倭寇|倭寇]]而命江夏侯[[周德興|周德興]]在東南沿海一帶大量興築衛所城，金門守禦千戶所城即是在這時期所建的城池之一{{NoteTag|1=福建沿海與金門同時設置的千戶所尚有大金所、定海所、梅花所、萬安所、莆禧所、崇武所、福全所、高浦所、[[中左守禦千戶所城|中左所]]、六鰲所、銅山所、玄鍾所、南詔所{{r|許}}。}}，隸屬永寧衛，但[[福建都司|福建都司]]亦有管轄權{{NoteTag|name=許}}。

根據《明太祖實錄》卷一八一「洪武二十年四月戊子條」所載：「命江夏侯周德興往福建，以福、興、漳、泉四府民戶三丁取一為緣海衛所戍兵以防倭寇，其原置軍衛非要害之所即移置之 。德興至福建，按籍抽兵，相視要害可為城守之處，具圖以進，凡選丁壯萬五千餘人，築城一十六，增置巡檢司四十有五，分隸諸衛以為防禦」，金門守禦千戶所城為其中一座千戶所<ref>[李國祥等編，《明實錄類纂‧福建台灣卷》，武漢：武漢出版社，1993年，465頁]</ref>。

金門城的興建始於明洪武二十年（1387年）四月，約在二十一年（1388年）二月落成{{r|張}}{{r|楊}}，但《明代海禁政策下的金門及其海域》（2009，許志仁）一文據《滄海紀遺》與《泉州市建築志》（1995年）關於[[崇武古城|崇武城]]的記載而認為金門城可能是到洪武二十五年（1392年）才落成{{r|許}}。初建的金門城周長630丈（約2016公尺），城垣高1丈7尺（約5.4公尺）、寬1丈（約3.2公尺），開有四門{{r|許}}。

後來在永樂十五年（1417年）時，都指揮谷祥強化城池防務，除城牆增高3尺外，並在西、北、南三面加築月城{{r|許}}{{r|楊}}。正統八年（1443年）時，都指揮劉亮與千戶陳旺又建四門敵樓{{r|許}}{{r|楊}}。嘉靖三十七年（1558年）時，所署燬於火{{r|許}}{{r|楊}}。

明末永曆年間，金門一帶曾為鄭成功的練兵之地，而在1663年金門被清軍攻陷後，金門開始遷界{{r|許}}。[[三藩之亂|三藩之亂]]期間，金門一度回到鄭家的勢力範圍，但在1680年又被攻陷{{r|許}}。清軍兩度入金造成金門「遂墮其城，焚其屋，棄其地，遷沿海，遺眾於界內而還」<ref name="金門守禦千戶所城">[http://ebooks.lib.ntu.edu.tw/1_file/kinmen/17/92-05.pdf 金門城「金門守禦千戶所城」]</ref>。而在此後，總兵陳龍一度打算重建金門城，但1682年決定將總鎮署遷到後浦，金門城地區因而沒落{{r|張}}。

此後自1949年起，因為金門成為中華民國與中華人民共和國對峙的前線，金門城剩餘的城垣殘跡被中華民國國軍拆除以用在其他工事上，使得金門城只剩下夯土層與部分地基{{r|張}}。民國八十六年（1997年）起，藉由[[金門酒廠|金門酒廠]]的回饋金，開始重建四方城門，但其形制與原貌有別{{r|張}}，其中重建的南門現為金門酒廠的大門。而在民國九十六年（2007年）時，金門國家公園在[[文臺寶塔|文臺寶塔]]附近重建了一段古城垣<ref>{{cite news|url=http://www.libertytimes.com.tw/2007/new/nov/24/today-north9-3.htm |title=復建古城牆 重現明代金門城 |publisher=自由電子報 |date=2007-11-24}}</ref>。

兵力部署

置有正千戶一員、副千戶一員，百戶一員，試百戶一員，鎮撫一員，差操屯種旗軍（舊額）一千五百三十五名。…教場在所城北門外。營房八十六間，在所城內。軍器一千七百四十三件…。軍儲倉大使一員，攢典一員，管收支米斗級額編三名…<ref>[明《閩書》〈兵事志〉；明《滄海記遺》，；清《泉州府志》〈兵事志〉]</ref>。

官兵待遇方面

「除本縣（昔日金門隸屬同安縣）原派折色米銀一千三百二十七兩九錢六分四厘零，內解府給軍銀八百二十七兩九錢六分四厘零，又解司備餉剩銀四百七十兩，協濟昌平州馬價銀三十兩外，該本色米二千四百五十石，每石改折銀五錢五分，其五錢解府給軍銀一千二百二十五兩，其五分解司，備銀餉一百二十二兩五錢<ref>[清《金門志》〈卷四‧兵防志〉，1960，頁 77]</ref>。浯洲鹽場課折米銀七百四十八兩八錢四分零，解府支給本所官兵奉餉。…嗣又奏准以晉、同二縣鹽場將米折銀分充金門等倉軍儲，本所自徵屯田本色米三百七十八石<ref>清《金門志》〈卷四‧兵防志〉，1960，頁 78</ref>。

== 城牆 ==
初建的金門城周長630丈（約2016公尺），城垣高1丈7尺（約5.4公尺）、寬1丈（約3.2公尺），開有四門<ref>《明代海禁政策下的金門及其海域》（2009，許志仁）一文據《滄海紀遺》與《泉州市建築志》（1995年）</ref>。

永樂十五年（1417年）時，都指揮谷祥強化城池防務，除城牆增高3尺外，並在西、北、南三面加築月城。正統八年（1443年）時，都指揮劉亮與千戶陳旺又建四門敵樓。嘉靖三十七年（1558年）時，所署燬於火。

金門城周長630丈（約2016公尺），城垣高1丈7尺（約5.4公尺）、寬1丈（約3.2公尺），開有四門，後來城牆加高3尺，又增建月城與敵樓{{r|許}}。此外金門城還設有36個窩鋪、數座烽火臺，城內有864間營房{{r|許}}，城濠深丈餘（約3.2公尺多）。

《泉州府志》〈卷四〉提及：「金門千戶所，在同安浯洲嶼，北倚山，東西南阻海。洪武二十年，江夏侯周德興，築外環以壕，深廣丈餘，周圍六百三十丈，基廣一丈，高一丈七尺，窩舖三十六64，門四。永樂十五年，都指揮谷祥，增高三尺，築西北南三月城，各建樓。正統八年，都指揮劉亮、千戶陳旺，增四門敵臺<ref>《泉州府志》〈卷之四．衛所城〉</ref>。

《馬巷廳志》及民國十年的《金門縣志》則是這樣記載的：「金門鎮城... 明洪武二十年，江夏侯周德興造，周六百三十丈，高二丈五尺，窩舖三十六，為門四，各建樓其上，西、南、北月城三，城外環以濠，置千戶所於此。永樂十五年，都指揮谷祥，增高城垣三尺，並砌西、北二月城。正統八年，千戶陳旺，增築四門敵臺，萬曆二十七年毀。清康熙間重修，設總兵駐劄（馬巷廳志）。後因墮城毀舍，人煙稀少，
乃移鎮後浦，城逐漸廢<ref>民國十年修纂的《金門縣志》乃因襲《馬巷廳志》</ref>。

== 城內建築 ==
城內有一個主要的十字街，交會之處有一座跨街而建的纕帶廟{{NoteTag|1=民國五十五年（1966年）拆毀，遷至金門城南邊的寶月庵旁，為關帝古廟{{r|許}}。}}，北面祀[[真武大帝|真武大帝]]，南面祀[[關聖帝君|關聖帝君]]，而千戶所署、[[古地城隍廟|城隍廟]]等公共建築即設於附近{{r|許}}。
