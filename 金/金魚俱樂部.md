'''金魚俱樂部'''是[[日本放送協會|NHK]]所播出的[[電視劇|電視劇]]，2011年7月23日在[[NHK綜合頻道|NHK綜合頻道]]首播。原作是獲得[[日本放送協會|NHK]]大獎的同名[[手機小說|手機小說]]改編，由作家[[椿花|椿花]]所著。[[入江甚儀|入江甚儀]]與[[刈谷友衣子|刈谷友衣子]]首次演出[[電視劇|電視劇]]男女主角。

== 概要 ==
高三的柊春（[[入江甚儀|入江甚儀]]飾）在學校人氣十足，卻對異性始終不感興趣。無意間連續看見高一的學妹春川琴（[[刈谷友衣子|刈谷友衣子]]飾）被同為一年級的同班同學欺負，仔細了解後才發現因為琴的家庭環境不正常，所以被歧視、[[欺凌|欺凌]]。對缺少依靠的琴，春決定伸出援手，陪她四處散步、談心以紓解她的不愉快。一次[[撈金魚|撈金魚]]的遊戲中他們得到了[[金魚|金魚]]，最後決定安放在「秘密基地」並養著，藉由[[金魚|金魚]]組成了屬於兩人的[[俱樂部|俱樂部]]，而每天都要聚會。之後，春與琴的距離因此漸漸拉近……<ref>{{Cite web | url=http://www.nhk.or.jp/kingyo/about/index.html | title=金魚俱樂部簡介 | publisher=[[日本放送協會|NHK]] | accessdate=2011-07-01 | archiveurl=https://web.archive.org/web/20110714071514/http://www.nhk.or.jp/kingyo/about/index.html | archivedate=2011-07-14 | deadurl=yes }}</ref>

== 人物介紹 ==
=== 主要人物 === 
<ref>{{Cite web | url=http://www.nhk.or.jp/kingyo/cast/index.html | title=金魚俱樂部主要演員介紹 | publisher=[[日本放送協會|NHK]] | accessdate=2011-07-01 | archiveurl=https://web.archive.org/web/20110722142804/http://www.nhk.or.jp/kingyo/cast/index.html | archivedate=2011-07-22 | deadurl=yes }}</ref><ref>{{Cite web | url=http://www.nhk.or.jp/kingyo/actor/index.html | title=金魚俱樂部登場人物關係圖 | publisher=[[日本放送協會|NHK]] | accessdate=2011-07-01 | archiveurl=https://web.archive.org/web/20110722142759/http://www.nhk.or.jp/kingyo/actor/index.html | archivedate=2011-07-22 | deadurl=yes }}</ref>

; '''柊 春''' - [[入江甚儀|入江甚儀]] 飾
:高校三年一組，前[[籃球|籃球]]社成員，總是沒什麼情緒，個性很沉穩的人。
:似乎有許多人在注意他，但他對異性不太關心。
; '''春川 琴''' - [[刈谷友衣子|刈谷友衣子]] 飾
:高校一年四組，因為她的家庭關係，使得她時常被欺負。母親只要帶著男友回家，就會遠離他們。
; '''木田 香代子''' - [[水野繪梨奈|水野繪梨奈]] 飾
:高校三年一組，學生會副會長，春的同班同學，愛慕著春。
; '''柳橋 翔''' - [[吉澤亮|吉澤亮]] 飾
:高校三年一組，足球社副社長，與春是最要好的朋友，個性比春活潑，心裡暗戀著香代子。
; '''本井 孝哉''' - [[栗原吾郎|栗原吾郎]] 飾
:學級委員長，琴的同班同學，喜歡琴。

== 備註 ==
*該劇有解說放送人員神田愛花（第6回由塚原愛代班），於後十分鐘發表觀眾意見、與來賓對談以及針對劇情內的問題投票，並公佈結果，被取名為「金魚的尾巴」單元。
*[[日本放送協會|NHK]]於2011年7月24日12時終止使用[[NTSC制式|NTSC制式]](除[[岩手縣|岩手縣]]、[[宮城縣|宮城縣]]、[[福島縣|福島縣]])，所以第一回仍然以[[NTSC制式|NTSC制式]]播出。
