'''靳雲鵬'''（{{bd|1877年||1951年|1月3日|catIdx=Jin,靳}}），[[表字|字]]'''翼青'''，[[山东省|山东省]][[兗州府|兗州府]][[邹县|邹县]]（今邹城）苗庄人，后随父母逃荒到济宁市区南马道。[[中华民国|中华民国]]军事及政治人物，曾任[[中华民国国务总理|中华民国国务总理]]。[[段祺瑞|段祺瑞]]手下「'''四大金剛'''」之一（其他为[[徐樹錚|徐樹錚]]、[[吴光新|吴光新]]、[[傅良佐|傅良佐]]）。<ref name=lai>{{Cite book<!--|和書-->|author = 来新夏等|title = 北洋軍閥史 下冊|year = 2000|publisher =  [[南開大学|南開大学]]出版社|isbn = 7-310-01517-7}}</ref><ref name=xu>{{Cite book<!--|和書-->|author = 徐友春主編|title = 民国人物大辞典 増訂版|year = 2007|publisher = 河北人民出版社|isbn = 978-7-202-03014-1}}</ref><ref name=liu>{{Cite book<!--|和書-->|author = 劉寿林等 編|title = 民国職官年表|year = 1995|publisher = [[中華書局|中華書局]]|isbn = 7-101-01320-1}}</ref>

== 生平 ==
[[File:Jin_Yunpeng2.jpg|thumb]]
靳雲鵬毕业于[[天津武備学堂|天津武備学堂]]。1902年（[[光緒|光緒]]28年）任北洋常備軍軍政司参謀处提調。[[李经羲|李经羲]]被任命为[[雲貴总督|雲貴总督]]后，靳雲鵬随其赴[[雲南省|雲南省]]，任新軍督練公所总参議。1911年（[[宣統|宣統]]3年）10月，为呼应[[武昌起義|武昌起義]]，[[昆明市|昆明]][[重九起義|重九起義]]爆发，靳雲鵬逃离昆明。<ref name=lai/><ref name=xu/>

1912年（[[民国紀元|民国]]元年）中華民国成立，靳雲鵬被任命为[[山東省_(中華民国)|山東省]]的北洋陸軍第5師師長。其後，调任陸軍部次長。1913年（民国2年），就任山東都督。1914年（民国3年），获授泰武将軍、督理山東軍務。1915年（民国4年）12月[[袁世凱称帝|袁世凱称帝]]，靳雲鵬获封一等伯爵。1916年（民国5年），获封果威将軍。<ref name=lai/><ref name=xu/><ref name=liu/>

同年6月袁世凱死去，靳雲鵬成为[[皖系|皖系]][[段祺瑞|段祺瑞]]手下「四大金剛」之一。1917年（民国6年），赴日本考察軍事，归国後任参战軍督練。曾代表[[北洋政府|北京政府]]與[[日本|日本]]簽訂《[[中日共同防敵軍事協定|中日陸軍共同防敵軍事協定]]》。1919年（民国8年），任国務总理兼陸軍总長，因同「四大金剛」中的[[徐樹錚|徐樹錚]]对立而称病辞职。1920年（民国9年）7月[[直皖战争|直皖战争]]皖系敗北後，靳雲鵬再度出任国務总理。<ref name=lai/><ref name=xu/><ref name=liu/>

1921年（民国10年）辞職，此後赴[[天津英租界|天津英租界]]定居，與日本財閥合辦[[膠東魯大礦業公司|膠東魯大礦業公司]]，在濟南開設[[魯豐紗廠|魯豐紗廠]]，在濟寧等地開辦電燈公司、濟寧麵粉公司，並廣置不動產。其後，[[中国国民党|中国国民党]]及[[日本|日本]]曾劝其重归政治舞台，但实际未再涉足政治。晩年篤信[[佛教|佛法]]。[[中華人民共和国|中華人民共和国]]建国後，留居天津。<ref name=lai/><ref name=xu/><ref>[http://www.tianjinwe.com/tianjin/tjcj/201010/t20101012_2041831.html 两届总理擅打“政治麻将”，天津网，2010-10-12]</ref>

1951年1月3日，在天津逝世。享年75岁。<ref name=lai/><ref name=xu/>

==家庭==
张作霖五女[[张怀曦|张怀曦]]同靳云鹏之子订婚，但未结婚。1928年张作霖在[[皇姑屯事件|皇姑屯事件]]中被炸死后，双方解除婚约。<ref>刘艳宇，中国十大军阀之谜，中国社会出版社，2000年</ref>

靳云鹏的弟弟为[[靳雲鶚|靳雲鶚]]。<ref name=lai/><ref name=xu/>
