'''靳克文'''（1954年9月-），河南[[项城|项城]]人，1984年7月加入中共，1971年2月参加工作，大学文化程度。曾任中共[[漯河市|漯河市]]委书记，2013年1月当选[[河南省政协|河南省政协]]副主席。<ref>{{cite web |url=http://news.hexun.com/2013-01-28/150662077.html |language= |title=漯河市委书记靳克文当选河南省政协副主席 |publisher=河南省人民政府网站 |date=2013年1月28日 |accessdate= |deadurl=yes |archiveurl=https://web.archive.org/web/20150110092519/http://news.hexun.com/2013-01-28/150662077.html |archivedate=2015年1月10日 }}</ref>

== 简历 ==
1998年12月至2002年7月任河南省政府副秘书长。2002年8月任漯河市市长；2006年12任中共漯河市委书记。<ref>{{cite web |url=http://news.shangdu.com/584/20110906/584_428048.shtml |language= |title=靳克文简历 |publisher=人民网 |date=2011年9月6日 |accessdate= |deadurl=yes |archiveurl=https://web.archive.org/web/20150110092134/http://news.shangdu.com/584/20110906/584_428048.shtml |archivedate=2015年1月10日 }}</ref>
