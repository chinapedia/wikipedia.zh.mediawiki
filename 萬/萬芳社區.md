'''萬芳社區'''是位於[[台北市|台北市]][[文山區|文山區]]的一個山坡地示範社區，因過去在此地有[[萬隆煤礦|萬隆煤礦]]及[[芳川煤礦|芳川煤礦]]，因此命名為「萬芳社區」。社區設籍人口約15,000人<ref>{{Cite web| publisher=台北市政府| title=台北市區里鄰電子地圖| url=http://www.czone.tcg.gov.tw/tp92-1/index.htm| accessdate=2008年10月4日| deadurl=yes| archiveurl=https://web.archive.org/web/20060719210037/http://www.czone.tcg.gov.tw/tp92-1/index.htm| archivedate=2006年7月19日}}</ref>。社區範圍大致位於[[萬芳路|萬芳路]]、[[木柵路|木柵路]]四段、[[和平東路|和平東路四段]]與[[萬美街|萬美街]]之間，面積13.7[[平方公里|平方公里]]；行政區包括了文山區的[[萬芳里|萬芳里]]與[[萬美里|萬美里]]<ref>{{Cite web| publisher=台北市文山區萬芳社區發展協會| title=文山區大萬芳里簡介| url=http://sixstar.cca.gov.tw/blog/a22393399/communityAction.do?method=doSpecialView&PK=1692| accessdate=| deadurl=yes| archiveurl=https://web.archive.org/web/20101226045529/http://sixstar.cca.gov.tw/blog/a22393399/communityAction.do?method=doSpecialView&PK=1692| archivedate=2010-12-26}}</ref>。

2006年完成的「140高地公園」是社區內最大的綠地，為面積達5.4[[公頃|公頃]]的自然森林<ref>{{Cite book| title = 臺北畫刊474期(96年7月)| url = http://theme.taipei.gov.tw/cgi-bin/SM_theme?page=46b91c42| accessdate = 2008年10月4日| year = 2007| publisher = 台北市政府| chapter = 臺北南方的靜謐天堂　140高地公園| chapterurl = http://163.29.36.87/cgi-bin/SM_themePro?page=46959b75}}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

==歷史==
在社區建立之前，此地原名為[[枹子坑|枹子坑]]，又因為[[標高|標高]]140[[公尺|公尺]]的山坡地，也被稱為「140高地」。1972年[[臺北市政府|臺北市政府]]開始規劃在此地區建設山坡地開發的示範社區，並於1978年起開始進行[[土地徵收|土地徵收]]，並著手興建[[道路|道路]]及[[國宅|國宅]]。

社區範圍原屬[[博嘉里|博嘉里]]，在1990年3月12日從博嘉里中獨立劃設置[[萬芳里|萬芳里]]，但由於人口成長快速，很快的就成為擁有61個鄰的里，因此在2002年再新劃分出[[萬美里|萬美里]]，兩里合計鄰數仍為61鄰。

==交通==
社區內的道路皆以「萬」字為開頭，包括萬利街、萬樂街、萬和街、萬安街、萬美街、萬寧街。

[[台北捷運木柵線|台北捷運木柵線]]的[[萬芳社區站|萬芳社區站]]也緊臨萬芳社區。

==大事紀==
*1949年：芳川煤礦開業，創辦人為陳其志先生<ref name="blog">{{Cite web| publisher=Willy Chang|title=1000508-1文山-芳川煤礦、萬隆煤礦|url=http://ivynimay.blogspot.tw/2011/05/1000508-1.html}}</ref>。
*1952年：萬隆煤礦開業<ref name="blog"/>。
*1978年：台北市政府於1978年11月起至1984年3月止，分階段徵用民地，建設萬芳社區<ref>{{Cite web| publisher=臺北市政府地政局土地開發總隊| title=蛻變山城－木柵景美140高地| url=http://www.lda.gov.taipei/ct.asp?xItem=18005&ctNode=3246&mp=111011}}</ref>。
*1979年：萬芳路開闢，將萬芳路上之小山丘剷除，使得萬芳社區藉由萬芳路得以連通興隆路<ref name="wenshan">{{cite web |url=http://www.wenshan.org.tw/archives/index.php?option=com_k2&view=item&id=118:140&Itemid=633 |title=存档副本 |accessdate=2016-06-12 |deadurl=yes |archiveurl=https://web.archive.org/web/20160808022108/http://www.wenshan.org.tw/archives/index.php?option=com_k2&view=item&id=118:140&Itemid=633 |archivedate=2016-08-08 }}</ref>。
*1980年：萬芳國中成立<ref name="wfsh">{{Cite web| publisher=萬芳高中| title=萬芳高中簡介|url=http://web.wfsh.tp.edu.tw/intro_c/intro_history.htm}}</ref>。
*1982年：[[萬芳國小|萬芳國小]]成立。
*1984年：芳川煤礦因[[六三水災|六三水災]]歇業<ref name="blog"/>。
*1985年：萬隆煤礦撤銷公司登記<ref name="blog"/>。
*1990年：因當地人口快速增加，博嘉里劃分為博嘉里及萬芳里<ref name="wenshan"/>。
*1996年：捷運[[萬芳社區站|萬芳社區站]]及捷運[[萬芳醫院站|萬芳醫院站]]成立。
*1997年：[[萬芳醫院|萬芳醫院]]成立。
*1998年：萬芳國中改制為[[萬芳高中|萬芳高中]]<ref name="wfsh"/>。
*2002年：因當地人口快速增加，萬芳里劃分為萬芳里及萬美里<ref name="wenshan"/>。
*2005年：萬芳交流道連通[[信義快速道路|信義快速道路]]，大幅改善文山地區及信義區的交通往來。
*2011年：2011年1月初，台北市政府宣布規劃將萬利街之萬芳超市改建為公營出租國宅，引發當地民眾抗爭<ref>{{cite web |url=http://house.udn.com/mag/house/storypage.jsp?f_ART_ID=227989 |title=存档副本 |accessdate=2016-06-12 |deadurl=yes |archiveurl=https://web.archive.org/web/20160815064851/http://house.udn.com/mag/house/storypage.jsp?f_ART_ID=227989 |archivedate=2016-08-15 }}</ref>。
*2012年：2012年6月12日，萬芳路與木柵路交叉口因暴雨發生嚴重積水現象，造成大水淹入民宅，經查為操作人員操作壓力箱涵閘門時疏忽所致，屬人為疏失<ref>http://www.gov.taipei/ct.asp?xItem=25940590&ctNode=65441&mp=100003</ref>。
*2014年：2014年5月，萬芳社區東邊之軍功路更名為和平東路4段<ref>{{cite web |url=https://tw.news.yahoo.com/%E8%BB%8D%E5%8A%9F%E8%B7%AF%E6%B6%88%E5%A4%B1-%E6%94%B9%E5%90%8D-%E5%92%8C%E5%B9%B3%E6%9D%B1%E8%B7%AF-%E9%96%80%E7%89%8C%E9%8D%8D%E9%87%91-061100572--finance.html |title=存档副本 |accessdate=2016-06-12 |deadurl=yes |archiveurl=https://web.archive.org/web/20160809092417/https://tw.news.yahoo.com/%E8%BB%8D%E5%8A%9F%E8%B7%AF%E6%B6%88%E5%A4%B1-%E6%94%B9%E5%90%8D-%E5%92%8C%E5%B9%B3%E6%9D%B1%E8%B7%AF-%E9%96%80%E7%89%8C%E9%8D%8D%E9%87%91-061100572--finance.html |archivedate=2016-08-09 }}</ref>。

==社區內重要地點及場所==
*[[捷運萬芳社區站|捷運萬芳社區站]]：捷運木柵線原規劃走木新路，但因早期人們對捷運之認識不深，木新路當地居抗議木柵線走新路，經過捷運當局考量，木柵線改走萬芳社區，並設立萬芳社區站，反而造就萬芳社區的發展<ref>http://li.taipei/ws_wanmei/36166_03</ref><ref>{{Cite web |url=http://www2.dorts.gov.tw/news/newsletter/ns269/rp269_05.html |title=存档副本 |access-date=2016-06-12 |archive-url=https://web.archive.org/web/20100712073942/http://www2.dorts.gov.tw/news/newsletter/ns269/rp269_05.html |archive-date=2010-07-12 |dead-url=yes }}</ref>。
*[[捷運萬芳醫院站|捷運萬芳醫院站]]*：捷運萬芳醫院站與萬芳醫院共構，該站也是全世界第一個和醫院共構的捷運車站<ref>http://gishileh.pixnet.net/blog/post/30734873-%E5%8F%B0%E5%8C%97%E6%8D%B7%E9%81%8B%E8%BB%8A%E7%AB%99%E7%B3%BB%E5%88%97-%E7%94%9F%E8%80%81%E7%97%85%E6%AD%BB%E5%85%B1%E5%AF%AB%E6%AD%B7%E5%8F%B2-%E6%8D%B7%E9%81%8B%E6%96%87</ref>。
*[[萬芳國小|萬芳國小]]：為一所公立小學，設有小學部及附設幼稚園。
*[[萬芳高中|萬芳高中]]*：為一所公立[[完全中學|完全中學]]，設有高中部及國中部。
*[[臺北市政府公務人員訓練處|臺北市政府公務人員訓練處]]*：為一所訓練機構，專責臺北市政府所屬機關公務人員之教育訓練。

**萬福宮：位於萬芳路上，主祀福德正神，該廟毗鄰天佑宮<ref name="blog"/>。
**萬芳福德宮（土地公廟）:主祀福德正神，原名枹子坑福德宮，已有100餘年歷史，因配合萬芳社區之開發，將此宮廟重建於萬芳7號公園內<ref>http://sixstar.moc.gov.tw/blog/a22393399</ref>。
*公車總站:公車總站有10線公車，分別是0南（萬芳社區至捷運東門站）、和平幹線（原15，萬芳社區至衡陽路）、109（萬芳社區至陽明山）、298（萬芳社區至行天宮）、606（萬芳社區至榮總）、小10（萬芳社區至貓空）、小11（萬芳社區至大春山莊）、棕2（景美女中至萬芳社區）、棕3（捷運動物園至萬芳社區）、綠11（萬芳社區至台電大樓）。另有公車282、棕5、棕6等經過萬芳社區並設公車站。

**萬安街社區：性質為國宅社區，萬芳社區之中心區域，住宅及人口較為密集，此處為狹義之萬芳社區，廣義之萬芳社區則包含萬芳里及萬美里（2002年萬芳里重新劃分為萬芳里及萬美里<ref>http://www.ifansbank.com/store4/index.php5?page=selfDefine&eShopID=116022&aid=7276#</ref>）及鄰近地區。
**萬利街社區：大多為國宅之社區<ref>http://pj2011.pixnet.net/blog/post/62437095-19%E5%B9%B4%E5%89%8D%E8%90%AC%E5%88%A9%E8%A1%97%E7%9A%84%E4%B8%80%E6%A8%81%E5%BE%80%E4%BA%8B</ref>。

**萬寧山莊（萬寧街）：性質為地上權國宅社區<ref>http://mypaper.pchome.com.tw/purse_coco/post/1322449362</ref>。

*其他公務部門：萬芳游泳池、萬芳消防隊、萬芳派出所、萬美街郵局、萬芳區民活動中心、萬芳民眾閱覽室、中華電信工務站。
*140高地：萬芳社區北邊之枹子坑山山上，有一台地稱之為140高地，其範圍大致從萬美社區（萬芳社區之一個子社區）至萬寧街前半段。
*[[140高地公園|140高地公園]]：野外山區開發而成之自然森林公園，因公園最高點為海拔138公尺枹子腳山（枹子坑山），取其整數並命名為140高地公園，本公園為萬芳社區最著名公園之一。
*興隆山（馬明潭山）[https://www.google.com.tw/maps/place/%E9%A6%AC%E6%98%8E%E6%BD%AD%E5%B1%B1/@24.9932863,121.5452122,14z/data=!3m1!4b1!4m5!3m4!1s0x3442aa7258fc9dd3:0x5669f83b7265d3e9!8m2!3d24.9932877!4d121.5627218]：位於萬芳社區南方之一座小山，最高點為海拔118公尺。萬隆煤礦及芳川煤礦位於興隆山內。
*枹子坑山（枹子腳山）[https://web.archive.org/web/20160513230032/http://pkl.gov.taipei/ct.asp?xItem=113885&CtNode=46938&mp=106011]：位於萬芳社區北方之一座小山，最高點為海拔138公尺，該點在140高地公園之步道中央處。
*萬芳山*：位於萬芳高中後方之一座小山，其最高點（海拔60公尺）在進萬芳高中後門沿下坡往前約一百多公尺處附近。
*[[萬隆煤礦|萬隆煤礦]]：1952年開業，1985年撤銷公司登記，萬芳社區之命名係起源於當地曾有萬隆煤礦及芳川煤礦。<ref name="blog"/>
*[[芳川煤礦|芳川煤礦]]：1949年開業，創辦人為陳其志先生（南僑公司董事長陳飛龍之父親），1984年因六三水災歇業，萬芳社區之命名係起源於當地曾有萬隆煤礦及芳川煤礦。<ref name="blog"/>
*[[萬芳醫院|萬芳醫院]]*：為一所臺北市之市立醫院，規模為醫學中心。
*[[萬芳交流道|萬芳交流道]]*：經由本交流道可連接信義快速道路及國道3號高速公路，本交流道之通車為文山區及信義區交通帶來很大改善及便利。
*景美溪河岸：位於萬芳社區之東方，緊鄰木柵路4段。景美溪河岸已綠化，並在此設立道南河濱公園，公園內設有自行車道、廣大草地、籃球場、網球場等。
*[[信義快速道路|信義快速道路]]：用來連接信義區與台北聯絡線萬芳交流道，全長3.2公里的快速道路，於2005年5月14日通車。
