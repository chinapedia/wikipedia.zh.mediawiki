'''萬塚站'''（{{jpn|j=万ケ塚駅|kyu=|hg=まんがつかえき|hgt=|rm=Mangatsuka eki|f=y}}）位於[[宮崎縣|宮崎縣]][[都城市|都城市]]丸谷町，是[[九州旅客鐵道|九州旅客鐵道]]（JR九州）
[[吉都線|吉都線]]上的車站。

== 車站構造 ==
岸式月台1面1線的地上站。無人車站，沒有站房，而是於原站房處設立候車室。

== 車站周邊 ==
* 萬塚簡易郵局

== 历史 ==
* 1947年（昭和22年）3月1日 - 設立。
* 1985年（昭和60年）3月14日 - 無人化。
* 1987年（昭和62年）4月1日 - [[國鐵分割民營化|國鐵分割民營化]]後成為九州旅客鐵道（JR九州）轄下的車站。
