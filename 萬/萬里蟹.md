'''萬里蟹'''，台灣的[[海產|海產]]行銷品牌，為台灣[[新北市|新北市]][[萬里區|萬里區]]漁民捕獲的海蟹，佔了台灣八成以上的海蟹捕撈量，產地主要來自新北市富貴角外海的「西北漁場」，<ref>{{Cite web|url = http://wanlicrab.tw|title = 萬里蟹【為什麼叫萬里蟹?? 】|accessdate = 2014-07-13|publisher = 新北市政府新聞局|quote = "...新北市萬里區，而萬里蟹最大的產地，就來自「西北漁場」..."|year = 2013}}</ref>，萬里區漁民也發展出台灣第一的捕蟹產業。<ref>{{Cite journal|journal = 今周刊|title = 秋嘗萬里蟹|author = 高靜玉|issue = 872|url = http://www.businesstoday.com.tw/article-content-80730-102078|date = September 5, 2013|publisher = 今周文化事業(股)公司|quote = "絕大多數的專業捕蟹船，船長都是萬里人"}}</ref>因此在2012年8月，新北市政府成立「萬里蟹」品牌並積極行銷<ref>{{Cite news|title = 朱立倫行銷萬里蟹品牌 為台灣第一海蟹發聲|url = http://www.ntpc.gov.tw/ch/home.jsp?id=28&parentpath=0,6,27&mcustomize=news_view.jsp&dataserno=256629&t=null&mserno=201309160001|publisher = 新北市政府新聞局|date = 2012-08-29}}</ref>，萬里區漁民約定成俗使用螢光綠色的三股特多龍繩來紮綁海蟹，也成為萬里蟹的一大特徵。

新北市政府漁業及漁港事業管理處於2014年完成商標註冊，取得「萬里蟹」的中華民國商標註冊證<ref>{{Cite news|url=http://news.ltn.com.tw/news/politics/breakingnews/2506108|title=萬里蟹是誰功勞？ 漁業處出示商標註冊證... - 政治 - 自由時報電子報|accessdate=2018-08-02}}</ref>。

== 種類 ==
萬里蟹包含三大類海蟹：[[鏽斑蟳|花蟹(鏽斑蟳)]]、[[红星梭子蟹|三點蟹(红星梭子蟹)]]及[[善泳蟳|石蟳(善泳蟳、顆粒蟳)]]。
== 產區 ==
新北市捕蟹產業源起於石門[[富基漁港|富基漁港]]，後由萬里區漁民發展出更專精的捕蟹技術與漁船規模，因此萬里區成為台灣捕蟹漁業最重要的基地，捕撈海蟹量佔台灣八成以上，[[台灣漁港列表#新北市|萬里區四大漁港]]（野柳漁港、東澳漁港、龜吼漁港、萬里漁港）更成為台灣海蟹最大集散地<ref>{{Cite news|title = 痴情只為萬里蟹|author = 焦桐|url = http://www.appledaily.com.tw/appledaily/article/supplement/20131120/35448827/|work = 蘋果日報|publisher = NEXT Media|date = 2013-11-20}}</ref>，年產量約500噸<ref>{{cite news|title=新北市「萬里蟹」締造3億元商機|url=http://m.ftv.com.tw/newscontent.aspx?sno=2013826S08M1|accessdate=2014-08-26|newspaper=民視新聞網|date=2013-08-26}}</ref>。

萬里蟹主要產量，來自新北市富貴角外海，東經120度40分以東、北緯26度以南的「西北漁場」海域<ref>{{Cite journal|author = 張瓊方|title = 秋蟹飄香萬里|journal = Taiwan Panorama 台灣光華雜誌|pages = 95|date=November 2013}}</ref>，但萬里漁夫在春夏4~5月時分，也會移往台中外海及澎湖捕蟹，因此像台中梧棲港、澎湖馬公港、高雄興達港等<ref>{{Cite journal|author = 劉易昇|date = 2013-10-03|title = 秋蟹開吃|journal = 食尚玩家雙周刊|issue = 276|publisher = 聯意製作股份有限公司}}</ref>，也是重要的萬里蟹卸貨集散地。 

== 捕撈方式 ==

海蟹捕撈方式，通常可分為拖網、圍網、蟹籠三種，而萬里區是以蟹籠方式為主。漁民以漁線編成、僅有三個入口的鐵製圓型蟹籠，中間置入鰹魚或秋刀魚片為誘餌，然後投入海底，待螃蟹爬進籠中，就立即被困住，漁民再收起蟹籠，即可抓到海蟹。<ref>{{Cite news|title = 產量冠全台萬里蟹  新北市打造優質品牌|author = 李鋅銅|url = http://www.want-daily.com/portal.php?mod=view&aid=44604|work = 旺報|publisher = 中時集團|date = 2012-09-29|page = B4}}</ref>

捕蟹船跟一般漁船不同，必須騰出很多空間，來收納蟹籠。每艘船會攜帶約五組以上的蟹籠，每一組約有360個蟹籠，前後各有一個固定的錨，同時仔細紮成一長列。出海後，到達船長認為螃蟹聚集的海域，漁民便把一組組置入餌料的蟹籠，依續丟入海中，

蟹籠沈在海底至少16~20小時後，漁民再以起錨機，一一將籠具拉上船同時立即將捕獲的海蟹取出，並一隻隻綁好（以免螃蟹相互廝殺而死亡），置入船內的海水艙貯放。
把蟹收好後，漁民還得整理蟹籠及餌料，然後再反覆 : 放餌＞放籠＞收籠＞綁蟹的程序，整個流程費時十餘小時以上。

另外自2014年4月起，行政院制定「沿近海漁船捕撈蟳蟹類漁獲管制措施」，明文規定若捕獲甲殼寬分別未滿6至8公分之蟳蟹，至遲需於進港後12小時內放回海中，否則將開罰最重15萬元，且每年8月16日至11月15日禁止漁船捕撈抱卵母蟹。<ref>{{cite news|author=曾佳俊|title=保護海洋資源 禁捕小萬里蟹|url=http://www.appledaily.com.tw/appledaily/article/headline/20140329/35733095/|accessdate=2014-08-25|newspaper=蘋果日報|date=2014-03-29}}</ref>
