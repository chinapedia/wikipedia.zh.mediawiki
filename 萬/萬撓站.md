'''万挠站'''（{{lang-ms|Stesen Kereta Api Rawang}}）位于[[马来西亚|马来西亚]][[雪兰莪州|雪兰莪州]][[万挠|万挠]]，是[[KTM电动列车服务|电动列车服务]]和[[巴生港线|巴生港线]]的其中一个站点。[[马来亚铁道公司|马来亚铁道公司]]是该车站的主要经营者。

本站是万挠的主要公共交通车站，因为在[[KTM通勤铁路|KTM通勤铁路]]服务启用之前，这里并无任何通勤或大众运输服务系统。目前也正在提升现有设施以迎合未来发展。<ref>{{cite web|url=http://www.sinchew.com.my/node/1613700|title=隆中环及万挠提升轨道．电动火车站更改月台|date=2017-02-12|publisher=}}</ref>

== 历史 ==
2007年4月21日，马来亚铁道推出了萬撓至叻思之間的班車服務。該服務將KTM通勤鐵路網絡延伸22公里，并覆蓋了三個新車站，即{{站|双文丹}}、{{站|峇冬加里}}和{{站|叻思}}。这是万挠－怡保复线电气化项目的首个运营路线，解决长期以来铁路被群众嫌慢的问题。通勤列车服务于2008年1月5日延伸至[[新古毛|新古毛]]。2009年6月1日，服务进一步延伸至霹雳州边境的{{站|丹绒马林}}。

自KTM通勤铁路服务开始以来，万挠作为万挠－芙蓉路线的北端终点站。直到2015年，为了迎合铁道复线电气化升级而启用万挠至巴生港的试航路线。在2016年，万挠－巴生港线与万挠－丹绒马林线合并（即[[巴生港线|巴生港线]]），从而结束了万挠作为北端终点站的角色。<ref> {{cite web |url=http://www.ktmb.com.my/Article.asp?Id=2130 |title=KTM Komuter Services For Rawang - Rasa Sector Launched |accessdate=2007-05-03 |author= |date=2007-04-21 |publisher= [[马来亚铁道|马来亚铁道]] |archiveurl=https://web.archive.org/web/20070810201026/http://www.ktmb.com.my/Article.asp?Id=2130 |archivedate=2007-08-10 |deadurl=yes }}</ref>

==意外事件==
2013年11月1日，一辆从丹绒马林前往吉隆坡的[[马来亚铁道|马来亚铁道]]（KTM）通勤列车在经过万挠站附近时意外翻轨，而列车上的乘客也没有严重伤亡。<ref> {{cite news |url=http://www.thestar.com.my/news/nation/2013/11/01/train-derail-rawang/ |title= KTM Komuter train derails near Rawang station (Updated) |accessdate=2017-09-02 |author=PATRICK LEE |date=2013-11-01|publisher=[[马来西亚星报|马来西亚星报]]|language=en }}</ref>
