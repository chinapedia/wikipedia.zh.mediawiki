《'''萬能麥斯'''》（''Mighty Max''）為[[美國|美國]][[電視動畫|電視動畫]]，於1993年至1994年於美國播映，總共播出兩季四十集，一開始播映是為了推銷同名系列玩具「[[:en:Mighty_Max_(toyline)|Mighty Max]]」，在當時動畫配合玩具的行銷手法曾造成相當風潮。[[臺灣|臺灣]]由[[台視|台視]]引進，於[[台灣時間|台灣時間]]每週三下午五點播映。



== 故事大綱 ==
麥斯是年約13歲的金髮男孩，某天他收到一個[[快遞|快遞]]包裏，裡面有能夠穿梭時空的紅色[[棒球帽|棒球帽]]，並帶有指示：「你已經被選為帽子的持有人，現在就去超市等候指示，萬能麥斯。」<br/>

後來麥斯認識了戰神與智者，四處對抗各種魔物，並朝最終大魔王「骷髏魔王」的巢穴邁進。<br/>

== 角色 ==
*麥斯（Max） －主角，由[[Rob_Paulsen|Rob Paulsen]]配音
*戰神（Norman）－麥斯的保鑣，由[[Richard_Moll|Richard Moll]]配音
*智者（Virgil）－長得像貓頭鷹，智慧的化身，由[[Tony_Jay|Tony Jay]]配音
*骷髏魔王（Skullmaster）－反派角色，由[[Tim_Curry|Tim Curry]]配音

== 發行資訊 ==
萬能麥斯曾經發行過[[VHS|VHS]]（家用錄影帶）版本，不過尚未正式發行DVD，網路上販售的DVD皆是未經授權的版本。<br/>

== 週邊產品 ==
令人意外的，萬能麥斯的周邊產品甚至比[[卡通|卡通]]本身更受歡迎。每個遊戲組合幾乎都會附有麥斯、反派角色的人形，並偶爾會附有戰神及智者的人形。

幾乎每一集卡通都有相對應的玩具組合。到了1995年，連[[麥當勞|麥當勞]]都在[[快樂兒童餐|快樂兒童餐]]推出了萬能麥斯的玩具。
另外，萬能麥斯也被做成了[[電視遊戲|電視遊戲]]、[[掌上遊戲機|掌上遊戲]]以及[[漫畫|漫畫]]、[[桌遊|桌遊]]等等。<br/>
