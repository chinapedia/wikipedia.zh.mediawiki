{{日本作品|搖擺女孩|スウィングガールズ|-{Swing Girls}-|tw=搖擺女孩|hk=喇叭書院|cn=摇摆少女}}是2004年9月11日在[[日本|日本]]上映的一部音樂喜劇[[電影|電影]]，導演是執導過《[[水男孩_(2001年電影)|水男孩]]》的{{link-ja|矢口史靖|矢口史靖}}。電影描述[[東北地方|日本東北]]的偏僻鄉村，一群女子高中生意外組成[[大樂團|大樂團]]練習[[爵士樂|爵士樂]]繼而參加音樂祭的經過。外景拍攝以日本[[山形县|山形县]][[置賜地方|置賜地方]]為中心，電影對白也是使用山形縣地區的[[方言|方言]]<ref>[http://swinggirls.jan.jp/okitama.html おきたま弁講座] - 【スウィングガールズ】おきたま応援サイト{{ja}}</ref>。此外，於片中組成大樂團的成員們其實原先並沒有各自擔當樂器的演奏經驗，在開拍前參加了合宿特訓，經過努力練習後才達到之後能在各地演出的水準<ref>[https://www.toho.co.jp/movie/lineup/swinggirls/data1.html スウィングガールズ - 映画・映像] - [[東寶|東寶]]{{ja}}</ref>。

== 劇情簡介 ==
{{Quote box
|width = 32%
|bgcolor = #E0FFFF
|quote = 「世界上有兩種人，<br>　搖擺的人和不會搖擺的人。」 
|source = --《搖擺女孩》
|salign = right}}

炎炎夏日中，有一群正在學校參加暑期輔導的女高中生，一旁則是[[管樂隊|管樂隊]]正準備去幫出賽的棒球隊應援，而便當店老闆在管樂隊出發後才把[[便當|便當]]送來。補課中很無聊看著窗外打發時間的友子（[[上野樹里|上野樹里]]飾演）見狀，便靈機一動向老師提議跟其他同學一起幫忙送便當去球場給管樂隊，於是一行人就此出發。在途中眾人偷吃了一個便當，還因為忘了下車而坐過站，耽擱許久才把便當送到。不知情的管樂隊吃了由於天氣炎熱早已壞掉的便當後，發生集體[[食物中毒|食物中毒]]的情況，只有拓雄（[[平岡祐太|平岡祐太]]飾演）因為便當少一個沒吃到而逃過一劫。

管樂隊的成員除了拓雄外都住院了，在下一場比賽前可能無法康復為棒球隊應援，拓雄只好招募臨時替補成員，卻只來了吹[[直笛|直笛]]的關口（[[本假屋唯香|本假屋唯香]]飾演）跟彈吉他與貝斯的前搖滾樂團成員。在人數嚴重不足之下，拓雄要求友子她們要為便當變質的事負責，卻遭到良江（[[貫地谷詩穗梨|貫地谷詩穗梨]]飾演）跟直美（{{link-ja|豐島由佳梨|豊島由佳梨}}飾演）的反駁，友子等人雖然不情願但又不想上[[數學|數學]]補課，兩相權衡之下，最後就抱著混水摸魚的心態加入了。然而一干人卻對[[樂器|樂器]]一竅不通，在拓雄的特訓下才開始學會吹奏簡單的旋律也漸漸發現音樂的迷人之處，此時管樂隊康復並且回歸，眾人只能不捨地交還樂器離去。

不甘心的友子想盡辦法買了二手的[[薩克斯風|薩克斯風]]並再度號召大家組成爵士樂團，為了湊錢買樂器眾人便一起去[[超市|超市]]打工。友子、良江、直美與關口卻在打工時引發小事故導致被開除，幾經波折後她們終於湊齊樂器並展開練習。首次在[[柏青哥|柏青哥]]店前興致滿滿地演出時遭到路人奚落，但有個神秘的大叔指點關口她們應該要從基礎開始練起，友子覺得奇怪想要細問便追了上去，結果發現神秘大叔居然是數學老師小澤（[[竹中直人|竹中直人]]飾演）。誤打誤撞之下她們便央求小澤老師指導，隨著日子過去，程度漸有進步的幾人在超市前再度演出，這次獲得了大家的肯定，在超市打工的其他人看到也重新燃起對[[爵士樂|爵士樂]]的熱情並再次加入組成了[[大樂團|大樂團]]。

為了參加東北學生音樂祭，搖擺女孩在小澤老師的指揮下演奏《[[In_the_Mood|In the Mood]]》並進行拍攝，完成後將[[影帶|影帶]]託付給友子寄出報名。然而粗心大意的友子忘了這件事，等到她想起來才寄出時，收到太晚報名所以額滿的回覆。友子不敢跟大家說，音樂祭當天還是照常出發，在途中終於忍不住愧疚感跟拓雄講了，於是拓雄代為轉告大家，眾人聽完後便心灰意冷，且列車也因為大雪而在半路上停止前進了。此時[[音樂老師|音樂老師]]伊丹（[[白石美帆|白石美帆]]飾演）跟著巴士一起來找被困在列車上的一行人，並轉告大家因為大雪所以有間學校無法參賽，搖擺女孩因禍得福得以參加音樂祭。於是一行人在最後一刻趕到，作為[[壓軸|壓軸]]演出並博得滿堂喝采。
