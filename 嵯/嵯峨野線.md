'''嵯峨野線'''（{{jpn|j=嵯峨野線|hg=さがのせん|rm=Sagano sen|f=y}}）是[[西日本旅客鐵道|西日本旅客鐵道]]（JR西日本）所經營的一條鐵路路線，是[[山陰本線|山陰本線]]由[[京都車站|京都]]至[[園部車站|園部]]路段的別稱，雖然此段路線於19世紀末就已通車，但直到1988年3月13日才啟用了目前的別稱。由於[[島根縣|島根縣]]、[[鳥取縣|鳥取縣]]兩個位於[[山陰地方|山陰地方]]的縣市的居民希望保留「山陰」的名字作為此路線的名稱，此路線有時亦會被稱做'''嵯峨野山陰線'''或'''山陰嵯峨野線'''。
== 概要 ==
全線根據{{link-ja|旅客營業規則|旅客営業規則}}上的[[大都市近郊區間_(JR)|大都市近郊區間]]的「[[大都市近郊區間_(JR)#大阪近郊區間|大阪近郊區間]]」，與[[IC卡|IC]][[乘車卡|乘車卡]]「[[ICOCA|ICOCA]]」的近畿圈地域。
== 路線資料 ==
* 管轄（事業類別）：西日本旅客鐵道（[[鐵路公司#第一種鐵道事業者|第一種鐵道事業者]]）
* 路線距離（{{link-ja|營業距離|営業キロ}}）：34.2公里
* [[軌距|軌距]]：1067毫米
* 站數：16個（包括起終點站）
* 複線路段：全線複線（但是京都站起至{{link-ja|京都貨物站|京都貨物駅}}的分岔點附近是單線）
* 電氣化路段：全線電氣化（直流1500V）
* [[閉塞_(鐵路)|閉塞方式]]：自動閉塞式
* 保安裝置：[[自動列車停止裝置|ATS-P]]（據點P方式）與[[自動列車停止裝置|ATS-SW]]
* {{link-ja|運轉指令所|運転指令所}}：{{link-ja|大阪綜合指令所|大阪総合指令所}}
* 最高速度：
** 京都站－嵯峨嵐山站間，馬堀站－園部站間：120公里/小時
** 嵯峨嵐山站－馬堀站間：130公里/小時

全線由{{link-ja|西日本旅客鐵道近畿統括本部|西日本旅客鉄道近畿統括本部|近畿統括本部}}管轄。
== 概要 ==
基本上以每小時一班次的快速列車以及每小時三班次的普通列車運行（普通列車當中有兩班次只在京都 - [[龜岡車站|龜岡]]路段行駛），部分班次會繼續行駛至[[胡麻車站|胡麻車站]]或是[[福知山車站|福知山車站]]。此線亦是JR西日本鐵路網[[都市網路|都市網路]]（Urban Network）中唯一一條沒有由其他路線的定期普通列車直通運行的路線。

在秋季時亦會因為較多觀光客使用此線到達[[嵯峨野|嵯峨野]]、[[嵐山|嵐山]]地區欣賞紅葉美景，這段時間亦會增開臨時列車來往京都和嵯峨嵐山及增加定期列車的車輛數目。

[[日本國有鐵道|日本國有鐵道]]時代時的山陰本線，多半是以長距離特急或是長距離普通列車為主要的營運重心，並沒有特別考量京都近郊的上班通勤及上學通學的需求。在1987年的[[國鐵分割民營化|國鐵分割民營化]]，路線由JR西日本接管之後，就不斷進行現代化工程。此段路線的嵯峨嵐山 - 馬堀間之雙線工程在1989年完成，連接新開通的隧道；而全線亦於1990年完成電力化，及於1996年完成[[二條車站|二條]] - [[花園車站_(京都府)|花園]]之間的高架化。而沿線發展亦不斷進行之下，JR西日本也於1989年及2000年開設[[太秦車站|太秦]]以及[[圓町車站|圓町]]兩個新站。

但迄今為止山陰本線的在單線區域仍然比較多的情況之下，以平均每小時3 - 4班次來說，除了特急列車外，在繁忙時間以外的時間通常只會以30分鐘左右一班次來行駛，造成班次不均衡的現象，並不足以滿足都會範圍逐漸向外發展時，都市交通機能的需要。另外園部車站以北更幾乎只有特急列車行駛，普通列車與特急列車之時間相差正逐年擴大。

嵯峨野線已在2010年完成全線雙線化工程。另外花園 - [[太秦車站|太秦]]之間的高架化，以及龜岡車站和[[嵯峨嵐山車站|嵯峨嵐山車站]]的站內橋上化及自由通道（[[跨站式站房|跨站式站房]]）也正在進行工程。

同時，隨著ATS-P的導入，[[JR神戶線|JR神戶線]]、[[JR京都線|京都線]]、[[琵琶湖線|琵琶湖線]]、[[湖西線|湖西線]]中的[[JR西日本221系電力動車組|221系電力動車組]]被[[JR西日本223系電力動車組|223系電力動車組]]取代，並轉到嵯峨野線上繼續行走，加上新建造的一列4車輛的223系電力動車組，以及對應了一人操作的16列2車輛的223系電力動車組，將會取代現時行駛中的[[日本國鐵113系電力動車組|113系電力動車組]]（亦有把轉屬的221系與223系互相併結行走的方案）。另外，近畿地方交通審議會亦在討論關於利用[[丹波口車站|丹波口]] - [[梅小路車站|梅小路]]以達成往[[大阪|大阪]]方向的互相直通運轉的提案。

隨著班次增加，鐵路噪音問題逐漸浮現，在京都 - 二條附近的高架路段尤其嚴重，現階段尚未有根治方法。

嵯峨野線全線車站均可以使用{{lang|ja|Jスルーカード}}、[[ICOCA|ICOCA]]、[[東日本旅客鐵道|東日本旅客鐵道]]的[[Suica|Suica]]及{{lang|ja|スルッと}}KANSAI的PiTaPa。

路線的主要競爭對手為[[京阪京都交通|京阪京都交通]]（舊京都交通）所營運，途徑[[國道9號_(日本)|國道9號]]來往[[京都|京都]]市內及[[龜岡市|龜岡]]以西地方的公車路線。

順帶一提，JR車站通常以車站大堂旁的月臺作為一號月臺，但在嵯峨野線範圍的車站（京都車站除外）不論大堂在哪一邊，均以上行線邊作為一號月臺。

隨着位於[[梅小路公園|梅小路公園]]的[[京都水族館|京都水族館]]和[[京都鐵道博物館|京都鐵道博物館]]的開幕，加上[[梅小路公園|梅小路公園]]離京都站有1.7公里距離，交通不便，因此JR西日本計畫在[[京都站|京都站]]和[[丹波口站|丹波口站]]之間建設新站。工程於2016年9月19日開始，2018年7月20日定名為「[[梅小路京都西車站|梅小路京都西車站]]」，已於2019年3月16日啟用。<ref>{{Cite web|url=https://www.westjr.co.jp/press/article/2016/08/page_9071.html|title=嵯峨野線 京都～丹波口駅間 新駅デザインなど概要決定：JR西日本|accessdate=2016-08-15|work=www.westjr.co.jp}}</ref>
