'''内蒙古工业大学'''坐落在[[中华人民共和国|中华人民共和国]][[内蒙古自治区|内蒙古自治区]][[呼和浩特市|呼和浩特市]]的一所[[高等院校|高等院校]]，是教育部批准设立的一所以工为主，工、理、文、经、管、法、教育、艺术相结合的多科性的全日制普通高等学校，是内蒙古自治区重点建设的高校之一。校训是博学躬行，尚志明德。<ref>{{Cite web|url =http://www.imut.edu.cn/xxgl/xxjj/index.htm|title =内蒙古工业大学概览之校风校训|author =|date =|publisher =|language =zh-hans|accessdate =|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20140706235432/http://www.imut.edu.cn/xxgl/xxjj/index.htm|archivedate =2014-07-06}}</ref>学校隶属于[[内蒙古自治区人民政府|内蒙古自治区人民政府]]，主管部门为[[内蒙古自治区教育厅|内蒙古自治区教育厅]]。<ref>{{Cite web|url =http://www.imut.edu.cn/xxgl/xxjj/index.htm|title =内蒙古工业大学概览|author =|date =|publisher =|language =zh-hans|accessdate =|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20140706235432/http://www.imut.edu.cn/xxgl/xxjj/index.htm|archivedate =2014-07-06}}</ref>

内蒙古工业大学的办学历史最早可以追溯到1951年创办的'''绥远省高级工业学校'''，1958年成立了'''内蒙古工学院'''，1993年更为现名。学校现有三个校区，新城校区位于呼和浩特市新城区爱民街49号；金川校区位于呼和浩特市经济技术开发区金川工业园区；准格尔校区位于鄂尔多斯市准格尔旗大路新区。学校共有70个本科专业，下设24个院（部、中心），有博士授权一级学科3个，博士授权二级学科13个；硕士授权一级学科18个，硕士授权二级学科85个，工程硕士专业学位授权领域17个。<ref>{{Cite web |url = http://www.hneeb.cn/web/university/yxzc/1503.html |title = 内蒙古工业大学2014年普通本、专科招生章程 |author =  |date =  |publisher =  |language = zh-hans |accessdate =  |quote =  }}</ref>
