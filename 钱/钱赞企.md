'''钱赞企'''（{{bd|1988年|8月24日|catIdx=Qian钱}}）是[[星际争霸2|星际争霸2]]的职业选手，人称NO总，现任neotv风暴英雄解说<ref>{{Cite web|url = http://www.neotv.com.cn/|title = neotv牛视网|accessdate = |author = |date = |publisher = }}</ref>。

== 职业生涯 ==
*2011年，开启星际争霸2的职业生涯。
*2011年11月7日加入星际争霸2 Dream战队，和[[F91|F91]]，Top，LoveTt，Loup成为队友。
*2012年6月1日 离开Dream，加入Phoenix战队，队友是F91, Top, LoveTt, Loup<ref>{{Cite web|url = http://s.163.com/12/0813/00/88OGJET900314D0E.html|title = Courage：虫族万岁 我需要更多的勇气和支持|accessdate = 2012-08-13|author = |date = |publisher = }}</ref>。
*2012年12月27日 离开Phoenix，加入Zenith of Origin (缩写 ZOO)，同TooDming, Gemini, LoveTt, Loup, TIME, Cyan, How ,Sakya 成为队友<ref>{{Cite web|url = http://sc2.plu.cn/info/2013-08-21/61821.html|title = [SC2采访] Courage：WCG冠军奖台是我的梦想|accessdate = 2013-08-21|author = |date = |publisher = }}</ref><ref>{{Cite web|url = http://sc2.plu.cn/info/2013-09-25/63904.html|title = [SC2采访] Courage：我真不是莽夫 输Jim不甘心|accessdate = 2013-09-25|author = |date = |publisher = }}</ref><ref>{{Cite web|url = http://s.163.com/14/0917/18/A6C5A7B500314Q8T.html|title = 【洗碗钟专访】Courage：我不是谐星 对RTS有感情|accessdate = 2014-09-17|author = |date = |publisher = }}</ref><ref>{{Cite web|url = http://s.163.com/14/0502/04/9R7BVMGR00314Q8N.html|title = 趣味：毕竟钱赞企！韩国人TheStC被国手打退役？|accessdate = 2014-05-02|author = |date = |publisher = }}</ref>。
