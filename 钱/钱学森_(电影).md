《'''钱学森'''》（英語：Hsue-shen Tsien），是2012年上映的一部中国大陸政治宣传电影，展现了“中国大陸航天之父”[[钱学森|钱学森]]的一生，描述他青年赴美求学被驱逐回国等故事。该片剧本创作始于2007年，2011年2月开拍<ref>{{cite news | title=张雨绮接新片《钱学森》身价涨五成(附图) | url=http://ent.sina.com.cn/m/c/2011-02-18/11493233372.shtml | publisher=[[新浪|新浪]] | date=2011-02-18}}</ref>，3月18日在北京举行了开机发布会<ref>{{cite news | title=陈坤《钱学森》中当父亲 张雨绮称感情空白(图) | url=http://ent.sina.com.cn/m/c/2011-03-18/21393257672.shtml | publisher=新浪 | date=2011-03-18}}</ref>，4月杀青。2012年2月29日本片在中共中央军委属下的[[国防科技大学|国防科技大学]]举行了看片点映会<ref>[http://www.people.com.cn/h/2012/0302/c25408-4007795638.html 电影《钱学森》在国防科大点映]2012年03月02日</ref>，同年3月2日在中国大陆正式上映，据新浪的评论“获得广泛好评”<ref>[http://ent.sina.com.cn/m/c/2012-03-03/09123571437.shtml 观影《钱学森》 男女观众反映大不同]2012年03月03日</ref>。

影片首周三天收获150万元票房<ref>{{cite web | title=中国电影报2012年3月6日新浪微博 | url=http://weibo.com/2304129841/y8F6vbwb4 | date=2012-03-06}}</ref>，至4月1日总票房累计325万元<ref>{{cite web | title=《钱学森》艺恩主页 | url=http://m.entgroup.cn/593757/ | publisher=艺恩网}}</ref><ref>{{cite web | title=中国电影报2012年4月6日新浪微博 | url=http://weibo.com/2304129841/ydm5097Yi | date=2012-04-06}}</ref>。

== 剧情 ==
该片从[[钱学森|钱学森]]在1947年和[[蒋英|蒋英]][[上海|上海]][[结婚|结婚]]开始到[[美国|美国]]求学，讲述了钱学森被遣返回到[[中国大陆|中国大陆]]，而后协助解放军制造[[原子弹|原子弹]]经历过[[文革|文革]]一直到晚年逝世的一生。<ref>{{cite web|url=http://www.sxdaily.com.cn/data/wtxw/20120301_10019744_2.htm|title=一个国家和民族的背影|accessdate=2012年|work=|publisher=陕西日报|language=zh-hans}}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref>{{cite web |url=http://js.people.com.cn/html/2012/02/16/79763.html |title=《钱学森》3月2日全国公映 爱情戏细腻动人 |accessdate= 2012年|work=|publisher=人民网 |language = zh-hans}}</ref>

== 花絮 ==
该片中[[钱学森|钱学森]]和[[蒋英|蒋英]]的[[爱情|爱情]]故事是一大看点。晚年时期，[[蒋英|蒋英]]挽着[[钱学森|钱学森]]的手说：“[[丘吉尔|丘吉尔]]母亲曾说，‘我这一生没有什么遗憾，我为[[大英帝国|大英帝国]]生下了[[丘吉尔|丘吉尔]]！’我也没什么遗憾，因为我的丈夫是[[钱学森|钱学森]]。”而钱学森却说：“如果不是嫁给我，你会成为我们国家最好的女高音[[歌唱家|歌唱家]]。”据中共中央政宣部门称为钱学森和蒋英晚年真实对话。<ref name="uis " >{{cite web |url=http://news.mtime.com/2012/02/24/1482742.html |title=《钱学森》曝老年爱情视频 温馨场面令人飙泪 |accessdate= 2012年|work=|publisher=时光网 |language = zh-hans}}</ref>

== 幕后 ==
拍摄该片时，演员[[陈坤|陈坤]]高烧到40度，他带病坚持拍了两个月的戏，感动了剧组。<ref>{{cite web |url=http://news.mtime.com/2011/03/30/1454962.html |title=《钱学森》北京热拍 陈坤39度高烧坚持拍戏 |accessdate= 2012年|work=|publisher=时光网 |language = zh-hans}}</ref>

该片跨越叙述了[[钱学森|钱学森]]的一生，从他到[[美国|美国]]求学、在美国入狱、寻求回国、复制[[原子弹|原子弹]]、去世，中间还贯穿了钱学森和夫人蒋英62年的爱情。耗资和制作都花费了政府不少资金和心血，导演[[张建亚|张建亚]]说：“为了追求真实震撼的视觉冲击，[[西安电影制片厂集团|西安电影制片厂集团]]斥资六千万打造，力求最大限度接近真实，还原当年场景。在拍摄这些‘大场面’的戏码时，不仅斥巨资制作了两枚1：1等比的仿真导弹用于拍摄，还动用数万[[解放军|解放军]]为剧组保驾护航，邀请解放军中的军事专家及导弹专家从旁指导，规模堪比军事演习，场面相当威武壮观。”，“因为只有逼真的场景才能感染观众，才能让观众体验到那个年代的艰苦与信念，才能理解钱老的伟大，这场戏的拍摄与当年钱老进行导弹试射的心情是一样的”<ref>{{cite web |url=http://news.mtime.com/2012/03/02/1483255.html  |title=电影《钱学森》今日公映 五大看点全解析 |accessdate= 2012年|work=|publisher=时光网 |language = zh-hans}}</ref>

蒋英的扮演者[[张雨绮|张雨绮]]直言说她出演该片是因为“最重要的就是谁来演钱学森。”比较其他自己不熟悉的演员，她更倾心陈坤。<ref>{{cite web |url= http://news.mtime.com/2012/02/28/1483071.html |title=《钱学森》北京首映礼 张雨绮直言演蒋英为陈坤 |accessdate= 2012年|work=|publisher=时光网 |language = zh-hans}}</ref>

== 评价 ==
出于政治宣传目的，政府宣传机构对影片为清一色正面积极的评价。观众对影片意见主要为歪曲主要事实，过分神化钱学森。<ref>{{cite web |url= http://movie.douban.com/subject/6041200/discussion/44680921/|title=豆瓣电影：钱学森被迫回国的历史真相|accessdate= 2012年|work=|publisher=豆瓣网|language = zh-hans}}</ref>
[[新浪网|新浪网]]点出了该片具有五大看点，华丽唯美的画面、转型的演员、启用[[国家机密|国家机密]]写剧本[[间谍战|间谍战]]部分、流畅的英文和高深的数学公式和钱学森与蒋英的甲子恋。<ref>{{cite web |url=http://i0.sinaimg.cn/ent/m/c/2012-03-02/U5199P28T3D3570139F326DT20120302004628.jpg |title=电影《钱学森》公映 五大看点全解析 |accessdate= 2012年|work=|publisher=新浪网 |language = zh-hans}}</ref>

导演[[张建亚|张建亚]]说：“我们只能尽力描摹他的一生，完全没有资格去评判他。虽然现在流行电影要接地气，但我们这部戏不一样，你不能从俯视的角度去看钱学森，只能仰视。”<ref>{{cite web |url=http://edu.qq.com/a/20120303/000010.htm |title=电影《钱学森》 公映 张建亚希望观众仰视钱学森 |accessdate= 2012年|work=|publisher=腾讯网 |language = zh-hans}}</ref>

钱学森的扮演者[[陈坤|陈坤]]被誉为该片的顶梁柱，撑起了全片的看点。<ref>{{cite web |url=http://news.mtime.com/2012/02/28/1482990.html |title=《钱学森》试映 陈坤获赞一个人撑起整部电影 |accessdate= 2012年|work=|publisher=时光网 |language = zh-hans}}</ref>

[[林永健|林永健]]饰演[[聂荣臻|聂荣臻]]据称获得政府干部的赞扬：大气威武，有大将风范。<ref>{{cite web |url= http://news.mtime.com/2011/07/27/1465878.html|title=林永健《钱学森》演聂荣臻 造型被赞逼真威武 |accessdate= 2012年|work=|publisher=时光网 |language = zh-hans}}</ref>
