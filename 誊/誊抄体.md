'''誊抄体'''<ref>{{标音|字=誊|拼音=Téng|同音字=腾}}</ref>或'''盘曲体'''（{{Lang-ar|نسخ|}}，拉丁转译：{{Transl|ar|DIN|nasḫ}}），音译为'''纳斯赫体'''，是一种[[伊斯兰书法|伊斯兰书法]]的[[字体|字体]]。相比早期的[[库法体|库法体]]，誊抄体在横竖转接处没有笔锋，因此书写较为顺畅，多用来誊抄文字故而得名，也因此之后取代了库法体的地位。目前为使用最为广泛的[[阿拉伯文|阿拉伯文]]字体<ref name="qalam">{{cite web|language =en|url=http://calligraphyqalam.com/styles/thuluth-naskh.html|title=三一体与誊抄体 Thuluth and naskh scripts|publisher=古兰经书法 CalligraphyQalam|accessdate=2010年7月15日}}</ref>。

== 特点 ==
誊抄体的笔法多曲线，较为柔美而灵动自然。字母两端几乎均匀分布在中轴线上下，转角处饱满，从而使得字体清晰<ref>{{cite web|language =en|url=http://theone01.tripod.com/Calligraphy/naskh.htm|title=誊抄体书法 NASKH CALLIGRAPHY|publisher=Arabic Culture Web site|accessdate=2010年7月15日}}</ref>。学习伊斯兰书法，需先学好誊抄体才可进阶修行。因书写方便，除用于抄写伊斯兰教经典中的《[[古兰经|古兰经]]》、[[圣训|圣训]]等之外，也多抄写诗歌等<ref>{{cite web|language =en|url=http://www.ummulhasanaat.co.za/naskh-font-in-arabic-calligraphy|title=阿拉伯书法中的誊抄体 Naskh font in Arabic Calligraphy|publisher=Madrasah Ummul Hasanaat|accessdate=2010年7月15日}}</ref>。誊抄体在伊斯兰书法中的地位，相当于[[中文书法|中文书法]]中的[[小楷|小楷]]。

== 历史 ==
[[File:Basmalah.svg|thumb]] ]]
誊抄体由[[叙利亚|叙利亚]]人创造<ref>{{cite web|language =zh-hans|url=http://zhongguoysl.bokee.com/6182009.html|title=纳斯赫体|publisher=伊斯兰人文学术|date=2007年3月24日|accessdate=2010年7月15日}}</ref>，具体诞生时间不明。据说最早使用誊抄体抄写《古兰经》的是书法家[[伊本·穆格莱|伊本·穆格莱]]，他是[[阿拔斯王朝|阿拔斯王朝]]的大臣。他最先将圆角和曲线引入了原始棱角分明的[[库法体|库法体]]，而[[哈里发|哈里发]]砍断了他的右手，他仍用左手写出美妙的字体{{r|yslys|page=168}}。

誊抄体起初只是流行于民间，在[[清真寺|清真寺]]等正式场合比较少见。但由于其独特的柔美特征则吸引着广大的[[穆斯林|穆斯林]]。[[倭马亚王朝|倭马亚王朝]]时期开始发展，公元10世纪时，书法家[[伊本·阿里·巴瓦布|伊本·阿里·巴瓦布]]对字体作出了改良，使字体更为优雅<ref>{{cite web|language=en |url=http://kakayicalligraphy.webs.com/naskhscripts.htm |title=誊抄体 Neskh Script |publisher=KAKAYI - Calligraphy |accessdate=2010年7月15日 |archiveurl=https://web.archive.org/web/20091202115036/http://kakayicalligraphy.webs.com/naskhscripts.htm |archivedate=2009年12月2日 |deadurl=yes }}</ref>。与此同时，因《古兰经》被广泛传抄，这种民间的字体也得以大肆传播。随着早期库法体的发展趋于花纹化，誊抄体也开始被正统所逐渐接受。直至11世纪，誊抄体则彻底取代了库法体的专用经文字体地位，并占据阿拉伯文书法主流直至今日{{r|yslys|page=164}}。15世纪时，书法家[[赛·哈姆杜拉|赛·哈姆杜拉]]则再次对字体作出了改良<ref name="qalam" />。
