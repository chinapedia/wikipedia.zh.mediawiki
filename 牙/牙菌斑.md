'''牙菌斑'''是在[[牙齿|牙齿]]表面逐渐沉积的[[生物薄膜|生物薄膜]]，薄的牙菌斑顏色通常是透明無色，但變厚時會呈現黃色或棕黃色，如果使用牙菌斑顯示劑可以很容易看到牙菌斑在口腔內的分佈<ref name = 醫學與生活_2001>{{cite book|title=醫學與生活:實用的保健預防書|url=https://books.google.com/books?id=le2hF1kJrDIC&pg=PA28|year=2001|publisher=臺灣商務印書館|isbn=978-957-05-1719-4|pages=28–}}</ref><ref name="LiveABC編輯群2017">{{cite book|author=LiveABC編輯群|title=2017年10月號 HOW IT WORKS 知識大圖解 中文版: 超跑打造術大揭密|url=https://books.google.com/books?id=GXY3DwAAQBAJ&pg=PA40|date=27 September 2017|pages=40–}}</ref>。

== 成因 ==
牙菌斑由食物残渣、脱落的口腔[[上皮细胞|上皮细胞]]、唾液和细菌构成<ref name="林保瑩2013">{{cite book|author=林保瑩|title=牙周病，真的會要人命|url=https://books.google.com/books?id=2NLNBQAAQBAJ&pg=PA99|date=1 November 2013|publisher=大塊文化出版股份有限公司|isbn=978-986-213-469-6|pages=99–}}</ref>。牙菌斑中的细菌主要是正常口腔内-{存}-在的[[鏈球菌|鏈球菌]]、[[厭氧生物|厌氧菌]]等。牙菌斑堆积到一定厚度之后，其内部紧挨牙齿表面的细菌因为与空气隔绝开始转入[[无氧呼吸|无氧呼吸]]。无氧呼吸在此处产生的酸不能及时被[[唾液|唾液]]冲走，因此会腐蚀[[珐琅质|珐琅质]]中的矿物成分，并进一步造成[[龋齿|龋齿]]。

== 影響與徵狀 ==
牙菌斑在牙齒上會形成蛀牙，在牙根处堆积的牙菌斑也会刺激[[牙龈|牙龈]]导致牙齦炎，當發展到牙齒周圍骨頭時就稱為[[牙周炎|牙周炎]]等牙周疾病<ref name = 醫學與生活_2001 />。

如果有以下情形，可能有牙齦方面的疾病：<ref name = 醫學與生活_2001 />
# 牙齦出血、有膿且受到擠壓有膿液、萎縮、腫脹或觸碰會疼痛等徵狀
# 牙齒鬆動或移位
# 呼吸時口腔有異味

== 清潔與治療 ==
牙菌斑对牙齿的危害随其沉积时间而增强，所以应该定期清除。牙菌斑的质地很软，可以用[[牙刷|牙刷]]、[[牙膏|牙膏]]来有效清除，刷牙重點在於牙齒本身清潔、牙齦與牙齒之間的淺溝以及牙齒與牙齒之間的牙齦縫隙(齦乳頭)，將口腔內、特別是以上三個區域刷乾淨，確定沒有食物殘渣<ref name = 醫學與生活_2001 /><ref name="林保瑩2013">{{cite book|author=林保瑩|title=牙周病，真的會要人命|url=https://books.google.com/books?id=2NLNBQAAQBAJ&pg=PA99|date=1 November 2013|publisher=大塊文化出版股份有限公司|isbn=978-986-213-469-6|pages=99–}}</ref>。一般每天早晚刷牙2次即可达到较好的效果。在牙齿之间等不容易刷到的地方可以使用[[牙线|牙线]]。[[洗牙|洗牙]]可以进一步保证牙菌斑的彻底清除，洗牙有手動洗牙跟超音波洗牙兩種，手動洗牙取決於醫生技術，超音波洗牙可以提高洗牙速度降低醫生勞動。<ref name = 家庭生活保健百科_2013>{{cite book|title=家庭生活保健百科|url=https://books.google.com/books?id=v45iAgAAQBAJ&pg=PT67|date=20 December 2013|publisher=青苹果数据中心|pages=67–|id=GGKEY:1P43ANPU2F2}}</ref>

长期沉积的牙菌斑如果磷含量超過一般牙菌斑3倍、鈣含量也較高，在结合了唾液中可溶性的碳酸鹽與磷酸鹽矿物质等成分后，会钙化形成硬质的[[牙結石|牙結石]]，牙結石表面粗糙無比，更容易產生牙菌斑。牙结石一般紧密附着在牙齿表面，其去除需要由牙医等专业人士进行。<ref name="林保瑩2013" /><ref name = 家庭生活保健百科_2013 />
