'''牙香樹'''（[[學名|學名]]：''{{lang|la|Aquilaria sinensis}}''），又名'''蜜香樹'''、'''莞香樹'''、'''土沉香'''、'''女兒香'''及'''白木香'''，屬於[[瑞香科|瑞香科]][[土沉香屬|土沉香屬]]，是[[香港|香港]]原生稀有的雙子葉植物。牙香樹亦盛產於[[華南|華南]]，但由於牙香樹在[[中國大陸|中國大陸]]遭到任意砍伐，因此屬於易危品種，被列入[[中國|中國]][[國家二級保護野生樹種|國家二級保護野生樹種]]，分佈地區包括[[海南|海南]]、[[廣東|廣東]]、[[廣西|廣西]]、[[雲南|雲南]]。

[[臺灣|臺灣]]亦出產可衍生沉香木質的牙香樹。臺灣原生種被IUCN物種保育等級評估為接近威脅，分布於西南沿海[[濕地|濕地]]地區，為耐鹽的[[紅樹林|紅樹林]]伴生植物，多數生長於沿海和[[沼澤|沼澤]]地區，這些地方易受開發破壞，許多棲地已劃入[[墾丁國家公園|墾丁國家公園]]和[[台江國家公園|台江國家公園]]等及[[中華民國|中華民國]]的國家重要濕地保護中。<ref>[http://kmweb.coa.gov.tw/subject/ct.asp?xItem=722086&ctNode=8306&mp=317&kpi=0 沉香 Aquilaria sp.瑞香科], [[行政院農業委員會|行政院農業委員會]]植物園館, 2012/9/21</ref><ref>[http://theme.coa.gov.tw/suggest.php?issue=11713&id=11955 台灣的稀有及瀕危植物列表] {{webarchive|url=https://web.archive.org/web/20140221173643/http://theme.coa.gov.tw/suggest.php?issue=11713&id=11955 |date=2014-02-21 }}, [[行政院農業委員會|行政院農業委員會]]</ref><ref>[http://np.cpami.gov.tw/youth/index.php?option=com_content&view=article&id=4156:2011-04-20-07-02-13&catid=24:2009-07-06-08-27-16&Itemid=34 土沉香], 臺灣國家公園, [[內政部營建署|內政部營建署]]</ref>此外，亞洲熱帶、[[澳洲|澳洲]]、[[玻里尼西亞|玻里尼西亞]]等地區也有牙香树分布。

牙香樹與南洋出產的品種（'''''Aquilaria agallocha'''''）親緣甚近，但不完全相同，故前者為稱土沉香，後者稱為沉香，由於前者產量少，故亦以前者較為矜貴。

== 特徵 ==
牙香樹是高約10米的中等喬木，為常綠喬木，樹榦灰色，小枝及被細小及柔軟的毛。葉呈卵形及短柄，長約7厘米。葉面有革質，故底面光亮。花期為4月，果期為7月，長有黃綠色細花且有微香，每年5月聚生成簇由枝葉末端及葉腋生出，結成木質蒴果、扁及如卵形，長2.5厘米及長滿灰色短毛，成熟時轉成黑色。

== 用途 ==
牙香樹的樹皮平滑，呈淺灰色，所以又被稱為「白木香」。樹脂帶有香氣，被製成[[嶺南|嶺南]]名貴的香料，在[[宋朝|宋朝]]時，在[[沙螺灣|沙螺灣]]和[[沙田|沙田]]被大量種植作為經濟樹<ref>[http://hk.apple.nextmedia.com/news/art/20130127/18148115 盛產香木銷海外]</ref>，再用[[香港交通|俗稱「大眼雞」的艚船運]]至[[广州市|廣州]]，遁陸路經[[南雄市|南雄]]，越[[大庾岭|大庾嶺]]，過[[赣江|贛江]]至[[九江市|九江市]]，沿[[长江|長江]]送往送往[[苏州市|蘇]][[杭州市|杭]]銷售<!---<ref>《嶺海漫話》 ISBN 9882111351</ref>---><ref name="明報20150215">{{cite news|last=陳|first=嘉文|title=街知巷聞﹕新機場 / 沉香賊 偷走了沙螺灣|date=2015年2月15日|newspaper=明報|url=http://news.mingpao.com/pns/%E8%A1%97%E7%9F%A5%E5%B7%B7%E8%81%9E%EF%B9%95%E6%96%B0%E6%A9%9F%E5%A0%B4%20-%20%E6%B2%89%E9%A6%99%E8%B3%8A%20%E5%81%B7%E8%B5%B0%E4%BA%86%E6%B2%99%E8%9E%BA%E7%81%A3/web_tc/article/20150215/s00005/1423938350266}}</ref>，且具有藥用價值。製成品包括[[鈔票|鈔票]]紙、[[肥皂|肥皂]]等。

== 保護情況 ==
土沉香屬國家二級保護植物，屬於易危品種，由於在中國大陸禁止砍伐，於是有不法之徒轉為在香港砍伐，土沉香常見於本港低地，尤其是在鄉村後面的風水林中，而在郊野公園亦有發現。雖然現時本港已無大量種植土沉香，但仍有一定數量。2013年蘋果新聞報導，可能已是本港最後的一棵野生百年土沉香被砍伐。斬樹黨大多來自[[廣東|廣東]][[茂名市|茂名]][[電白縣|電白縣]]。<!---<ref>[http://www.youtube.com/watch?v=Afd7I6mcHSA 內地幫斬絕香港　百年沉香倒下來]</ref>---><ref>[http://hk.apple.nextmedia.com/news/art/20130127/18148107 內地幫斬絕香港土沉香 本港百年沉香樹王 遭「砍頭肢解」取樹脂]</ref>
而任何人非法砍伐樹木，可能會觸犯《盜竊罪條例》第九條或第二十七條，一經定罪，分別可處監管十年或三年。<ref>[http://orientaldaily.on.cc/cnt/news/20110512/00176_097.html 探射燈：砍伐土沉香賊巢曝光]</ref>
