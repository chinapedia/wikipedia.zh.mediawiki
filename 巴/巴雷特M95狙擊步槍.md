'''巴雷特M95'''（{{Lang-en|'''Barrett Model 95'''}}）是一枝由[[美国|美国]]槍械製造商[[巴雷特槍械公司|巴雷特]]所研製及生產的重型[[犢牛式_(槍械)|犢牛式]][[狙擊步槍|狙擊步槍]]（[[反器材步槍|反器材步槍]]），為取代[[巴雷特M90狙擊步槍|巴雷特M90]]的後繼產品。發射[[12.7×99mm_NATO|12.7×99毫米（.50 BMG）]][[北大西洋公约组织|北約]][[口徑|口徑]]制式[[步枪|步枪]][[子彈|子彈]]。

== 歷史 ==
[[File:PN_Barrett_M95.jpg|thumb]]採用的巴雷特M95。]]
巴雷特在推出大口徑[[半自動步槍|半自動]][[犢牛式_(槍械)|犢牛式]]狙擊步槍（反器材步槍）[[巴雷特M90狙擊步槍|M90]]以後，為了加以改進其設計以及使操作和價格比起軍用型[[巴雷特M82狙擊步槍|巴雷特M82A1]]更簡單和更便宜，而設計及生產了巴雷特M95。

== 設計 ==
[[File:US_Navy_021209-O-0000X-002_Spanish_Sailors_aboard_the_Santa_Maria-class_Frigate_Navarra_(F-85)_man_provide_cover_for_boarding_teams,_during_a_hostile_boarding_of_the_North_Korean_vessel,_So_San.jpg|thumb]]狙擊手在[[美國海軍|美國海軍]]軍艦上使用巴雷特M95練習狙擊，攝於2002年。]]
巴雷特M95和M90一樣，保留其[[砲口制動器|雙膛直角箭頭形（Ｖ形）制動器]]、可摺疊式[[兩腳架|兩腳架]]和機匣頂部的[[皮卡汀尼導軌|MIL-STD-1913戰術導軌]]、[[彈匣|彈匣]]減少至5發、沒有[[機械瞄具|機械瞄具]]，必須利用戰術導軌上安裝[[瞄準镜|光學瞄準鏡]]。但主要分別在於更符合[[人因工程学|人體工學]]，其[[握把|握把]]和[[扳機|扳機]]之間向前移25毫米（1英吋）以便更換[[彈匣|彈匣]]、縮短每發之間的時間；另外{{Link-en|拉機柄|Cocking handle}}亦重新設計，比M90更下更後，並比M90更容易拉動槍機。槍管可快速從槍上拆卸以便縮短全槍長度以及攜帶，而[[膛室|膛室]]亦鍍上鉻以防止[[铁锈|生鏽]]；最後扳機、[[撞針|擊針]]和重量亦有很小型的變化。

巴雷特M95可以在美國購買其民用型，價格約為[[美元|$]]6,000。

== 巴雷特XM107 ==
{{seealso|巴雷特M82狙擊步槍#M82到M107}}
1999年，巴雷特M95被美軍選中成為M107。有一少部份被美軍購入及測試，但是最終反而是M82被美軍選中。儘管如此，[[巴雷特槍械公司|巴雷特]]的官方網站亦宣布M95最少用於15個國家的軍隊和[[執法機關|執法機關]]。
