'''巴唐·哈久'''（{{lang-sh|'''Baton Hadžiju'''}}，{{lang-sq|'''Baton Haxhiu'''}}，{{bd|1967年|||}}<ref name=I />）是一位因為報道[[南斯拉夫戰爭|南斯拉夫戰爭]]而被威脅的[[阿爾巴尼亞|阿爾巴尼亞]]裔[[科索沃|科索沃]][[記者|記者]]。<ref name=PBS>{{cite web |url=http://www.pbs.org/newshour/media/j_awards/haxhiu_11-22.html |title=Baton Haxhiu |date=1999-11-22 |work=NewsHour |publisher=PBS |deadurl=no |archivedate=2012-09-06 |archiveurl=https://www.webcitation.org/6AU30Tixw?url=http://www.pbs.org/newshour/media/j_awards/haxhiu_11-22.html |accessdate=2013-03-28 }}</ref>

== 早期生涯 ==
哈久於[[南斯拉夫社會主義聯邦共和國|南斯拉夫]][[普里什蒂纳|普里什蒂纳]]大學主修[[社會學|社會學]]系，但其學業於1991年因為政府強行關掉大學而被逼中斷。之後，哈久參與了普里什蒂纳的地下學術運動，並繼續進修社會學。完成學業後，哈久便於阿爾巴尼亞語日報《時間》（Koha）中擔任編輯一職。<ref name=I>{{cite web |url=http://www.highbeam.com/doc/1P2-4983212.html |title=Obituary: Baton Haxhiu |author=Martyn Rady |date=1999-04-01 |work=The Independent |publisher= |accessdate=2013-03-28 }}{{Dead link|date=2019年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>隨後，哈久成為了普里什蒂納日報《日常時間》（Koha Ditore）的總編輯。<ref name=CPJ>{{cite web |url=http://cpj.org/awards/1999/awards.php |title=1999 Awards - Announcement |date=1999-11-23 |publisher=Committee to Protect Journalists |archivedate=2012-09-06 |deadurl=no |archiveurl=https://www.webcitation.org/6AU2VIQSr?url=http://cpj.org/awards/1999/awards.php |accessdate=2013-03-28 }}</ref> 根據英國報紙《[[獨立報|獨立報]]》指出，《日常時間》在哈久的管理下，成為了阿爾巴尼亞語媒體及科索沃批評家的領導者。<ref name=I />

== 1999年威脅 ==
於1999年3月，在[[北約轟炸南斯拉夫|北約轟炸南斯拉夫]]發生前不久，哈久與《日常時間》被[[塞爾維亞與蒙特內哥羅|塞爾維亞與蒙特內哥羅]]政府控告其報道違反信息法，並要求他們支付罚款。<ref>{{cite web |url=http://cpj.org/1999/03/cpj-condemns-media-crackdown-in-yugoslavia.php |title=CPJ Condemns Media Crackdown in Yugoslavia |date=1999-03-25 |publisher=Committee to Protect Journalists |archivedate=2012-09-06 |deadurl=no |archiveurl=https://www.webcitation.org/6AU3ScUPP?url=http://cpj.org/1999/03/cpj-condemns-media-crackdown-in-yugoslavia.php |accessdate=2013-03-28 }}</ref>同年3月23日，《日常時間》的頭版僅僅寫了一句「北約，做就對了」（Nato, Just Do It），並展示了[[Nike|Nike]]的標語「Just Do It」和其商標，以鼓勵北約進行轟炸南斯拉夫的行動。<ref name=PBS/> 

作為報復，政府軍隊當晚便燒毀了《日常時間》的辦公室，並殺死了一位守衛。之後，《日常時間》的律師被殺，而北約亦誤報哈久已被殺死。<ref name=CPJ />可是，哈久實際上因躲在一個地下室裡而逃過一劫。當時，他躲在地下室裡足足一個星期之久，且只有蘋果和水充飢。期間，他更聽到[[美國有線電視新聞網|CNN]]報道自己的死訊。事後，他指出自己當時「與死亡的距離有如皮膚與骨骼的距離」。<ref name=PBS/> 

同年4月2日，政府軍士兵命令哈久所有鄰里離開自己的居所，以找出哈久。因此，哈久說服了一名攜著兒子的母親讓他假裝為她的丈夫，以避過追捕。四天後，當哈久仍然嘗試逃至[[馬其頓共和國|馬其頓]]時，他逃跑的消息被傳開了。因此，阿爾巴尼亞政治家亞賓·薩費尼派遣了一隊人員去拯救哈久脫離科索沃。<ref>{{cite web |url=http://www.highbeam.com/doc/1P2-585635.html |title=Trying to Stay Alive While Dead to the World; Editor Dodges Police After His 'Demise' |author=R. Jeffrey Smith |date=1999-04-19 |work=The Washington Post |publisher= |accessdate=2013-03-28 |archive-url=https://web.archive.org/web/20180219151006/https://www.highbeam.com/doc/1P2-585635.html |archive-date=2018-02-19 |dead-url=yes }}</ref> 當哈久到達阿爾巴尼亞後，他便繼續其於《日常時間》的工作。<ref name=CPJ /> 

於1999年10月，哈久回到科索沃。在那個月內，[[科索沃解放軍|科索沃解放軍]]的領導人[[哈辛·塔奇|哈辛·塔奇]]指控了哈久和出版者[[維頓·蘇羅伊|維頓·蘇羅伊]]配合[[南斯拉夫總統|南斯拉夫總統]][[斯洛博丹·米洛舍維奇|斯洛博丹·米洛舍維奇]]，稱他們為“親塞爾維亞的吸血鬼”，並指出「像他們這樣的人……本身應該認識到，有朝一日，他們可能成為一些個人恩怨的目標，而這是可以理解。因此，維頓·蘇羅伊和巴唐·哈久，這些普通的黑手黨成員，不應該逍遙法外」。<ref>{{cite web |url=http://www.highbeam.com/doc/1P2-630697.html |title=Forces of Intolerance Threaten to Consume Kosovo |author=Peter Finn |date=1999-10-13 |work=The Washington Post |publisher= |accessdate=2013-03-28 |archive-url=https://web.archive.org/web/20160324095858/https://www.highbeam.com/doc/1P2-630697.html |archive-date=2016-03-24 |dead-url=yes }}</ref>儘管受到威脅，《日常時間》仍然堅持出版。<ref name=CPJ /> 同年，哈久獲[[保護記者委員會|保護記者委員會]]頒發[[国际新闻自由奖|国际新闻自由奖]]。<ref name=CPJ2>{{cite web |url=http://cpj.org/awards/1999/ |title=International Press Freedom Awards 1999 |publisher=Committee to Protect Journalists |deadurl=yes |archivedate=2012-08-11 |archiveurl=https://www.webcitation.org/69pU1OZLg?url=http://cpj.org/awards/1999/ |accessdate=2013-03-28 }}</ref>

== 後期生涯 ==
哈久之後成為了《科索沃快報》的編輯。<ref name=AP>{{cite web |url=http://www.highbeam.com/doc/1A1-D9300MIO2.html |title=Journalist's contempt appeal too late |date=2008-09-04 |publisher=Associated Press |accessdate=2013-03-28 }}{{Dead link|date=2019年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>於2008年7月，在[[拉穆什·哈拉迪納伊|拉穆什·哈拉迪納伊]][[戰爭罪|戰爭罪]]審訊期間，哈久因於一篇報道中揭示了受保護證人的名稱，而被[[前南斯拉夫问题国际刑事法庭|前南斯拉夫问题国际刑事法庭]]以[[藐視法庭|藐視法庭]]罪罰款$10,000美元。<ref>{{cite web |url=http://www.haguejusticeportal.net/index.php?id=9561 |title=Haxhiu, Baton |publisher=The Hague Justice Portal |deadurl=no |archivedate=2012-09-06 |archiveurl=https://www.webcitation.org/6AU4Py7ok?url=http://www.haguejusticeportal.net/index.php?id=9561 |accessdate=2013-03-28 }}</ref><ref>{{cite web |url=http://www.highbeam.com/doc/1A2-7a638c0d-33b8-4ee9-a098-00e4c5be5f7d.html |title=Netherlands: War Crimes Editor Charged |date=2008-05-21 |publisher=Associated Press |accessdate=2013-03-28 |archive-url=https://web.archive.org/web/20160314042758/https://www.highbeam.com/doc/1A2-7a638c0d-33b8-4ee9-a098-00e4c5be5f7d.html |archive-date=2016-03-14 |dead-url=yes }}</ref> 哈久之後對判決提出上訴，但法官指出他的要求超出了允許範圍，並駁回上訴。<ref name=AP />

== 個人生活 ==
哈久擁有一位妻子及一個兒子。<ref name=I />
