'''巴达玛宁布·巴特额尔登'''（{{lang-mn|Бадмаанямбуугийн Бат-Эрдэнэ}}，{{bd|1964年|6月7日||}}）[[蒙古族|蒙古族]]，[[蒙古国|蒙古国]][[肯特省|肯特省]]人，[[蒙古国|蒙古国]]摔跤冠军、政治人物。<ref name=wangyi>[http://news.163.com/13/0505/15/8U4DKNHI00014JB6.html 蒙古人民党提名巴特额尔登为总统候选人，网易，2013-05-05]</ref><ref name=ifeng>[http://news.ifeng.com/world/detail_2013_06/25/26789355_0.shtml 蒙古将举行2013年总统选举 三名候选人参与竞逐，凤凰网，2013-06-25]</ref>

==生平==
巴特额尔登1982年毕业于[[蒙古人民共和国|蒙古人民共和国]][[肯特省|肯特省]]南德日格勒县十年制中学，1990年毕业于蒙古军事学院法律学专业。他是蒙古家喻户晓的摔跤手（布赫），曾经12次获得蒙古国全国摔跤冠军。2004年以来连续三次当选[[国家大呼拉尔|国家大呼拉尔]]委员。<ref name=wangyi/><ref name=ifeng/>

2013年，[[蒙古人民党|蒙古人民党]]主席、国家大呼拉尔委员[[乌力吉赛汗·恩赫图布辛|恩赫图布辛]]、国家大呼拉尔委员[[丹增·龙戴姜仓|龙戴姜苍]]、国家大呼拉尔委员奥云浩劳尔、国家大呼拉尔委员巴特额尔登等4人角逐代表该党参加[[2013年蒙古国总统选举|2013年蒙古国总统选举]]的资格。2013年5月5日，[[蒙古人民党|蒙古人民党]]代表会议决定，国家大呼拉尔委员巴特额尔登将代表该党参加2013年6月举行的[[2013年蒙古国总统选举|2013年蒙古国总统选举]]。<ref name=wangyi/><ref name=ifeng/>

这次竞选的三位候选人分别为：[[民主党_(蒙古)|民主党]]的现任总统[[查希亚·额勒贝格道尔吉|查希亚·额勒贝格道尔吉]]，代表最大反对党参选的巴特额尔登，以及与民主党联合执政的[[蒙古人民革命党_(2010年)|蒙古人民革命党]]提名的该党总书记、健康部长[[那查格·乌德巴勒|那查格·乌德巴勒]]。在此次竞选中，巴特额尔登提出的竞选口号是“崇尚团结、扶持真善”。2013年6月26日，[[2013年蒙古国总统选举|2013年蒙古国总统选举]]举行，[[查希亚·额勒贝格道尔吉|查希亚·额勒贝格道尔吉]]成功连任总统，巴特额尔登落選。<ref name=wangyi/><ref name=ifeng/>

2016年，巴特额尔登进入[[额尔登巴特内阁|额尔登巴特内阁]]担任国防部部长<ref>{{Cite web |url =http://www.fmprc.gov.cn/ce/cemn/chn/tpxw/t1385600.htm  |title = 邢海明大使会见蒙古新任国防部长巴特额尔登 |author =  |date = 2016-07-28 |publisher = 中华人民共和国驻蒙古国大使馆 |language =  |accessdate =  |quote =  }}</ref>。
