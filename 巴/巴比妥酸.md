'''丙二酰脲'''，又称'''巴比妥酸'''（Barbituric acid），化学名：2,4,6-[[嘧啶|嘧啶]]三[[酮|酮]]。

== 性质 ==
一种[[白色|白色]]无臭结晶体，难溶于冷[[水|水]]和[[乙醇|乙醇]]，溶于热水和[[乙醚|乙醚]]。通常含有两分子结晶水。在空气中易[[风化|风化]]。水溶液呈酸性。可以与[[金属|金属]]反应生成[[盐|盐]]类。

== 制备方法 ==
可由[[丙二酸二乙酯|丙二酸二乙酯]]与[[尿素|尿素]]在[[乙醇钠|乙醇钠]]催化下发生[[缩合反应|缩合反应]]而得。下图所示的是[[丙二酸|丙二酸]]与尿素的缩合反应。

<br/>
:[[File:Barbituric_acid_synthesis.svg|500px]]

== 用途 ==
可用作分析试剂、有机合成原料、塑料和染料的中间体、[[聚合反应|聚合反应]]的[[催化剂|催化剂]]。丙二酰脲亚甲基上两个氢原子被烃基取代后的若干衍生物，称为[[巴比妥类药物|巴比妥类药物]]，是一类重要的镇静催眠药物。
