'''巴薩諾瓦'''（Bossa Nova）是一種融合[[巴西|巴西]]森巴舞曲和美國[[冷爵士樂|酷派爵士]]的一種“新派[[爵士樂|爵士樂]]”，承襲choro和samba-canção的部分特色而又自成一格，乍聽簡潔輕快。巴薩諾瓦結構複雜；樂器的音階或[[和絃|和絃]]轉換的行進方式變幻莫測，往往乍聽下以為可以掌握[[旋律|旋律]]的起落和[[節奏|節奏]]，和絃一轉換後即捕捉不及。它的旋律可以是[[行板|行板]]或[[中板|中板]] 。

== 語源 ==
巴薩諾瓦（Bossa Nova）是葡萄牙文，「Bossa」原意是演奏[[森巴|森巴]]時的自然風格與氣質，「Nova」則是新的意思。結合起來，巴薩諾瓦就是一種融合了傳統[[巴西|巴西]][[森巴|森巴]]（samba）節奏與[[啼樂|啼樂]]（choro）的一種「新派[[音樂|音樂]]」。正統的巴薩諾瓦的起源是南美洲巴西土生土長的音樂，之後流傳到北美洲之後廣為爵士樂壇所喜愛，由美國爵士樂手大力推廣。也因此「巴薩諾瓦是拉丁爵士的一種」的這個印象，已經是後來的事情了。

== 風格 ==

巴薩諾瓦的字面意思是「新節奏」，原來是一種拉丁音樂，聽起來輕鬆柔和、慵懶甜美、浪漫性感。與傳統[[南美|南美]]及[[拉丁音樂|拉丁音樂]]不同的是，巴薩諾瓦不像[[森巴|森巴]]或[[倫巴|倫巴]]（Rumba）那樣節奏強烈，拋棄了巴西音樂傳統密集的鼓點而採用旋律以表現，故除擁有南美音樂的熱情外，還帶有一份慵懶和輕鬆的感覺。

== 歷史 ==

巴薩諾瓦在1950年代末期的巴西興起。
1959年，由[[法國|法國]][[導演|導演]]Marcel Camus執導的[[巴西|巴西]][[電影|電影]]「[[黑色奧菲斯|黑色奧菲斯]]」(Black Orpheus)勇奪[[戛纳电影节|康城電影節]]、[[奥斯卡金像奖|奧斯卡]]、[[金球獎|金球獎]]三項最佳外語片獎；
帶有巴薩諾瓦風格的電影主題曲「A Felicidade」和「Desafinado」風行一時，人們開始對巴薩諾瓦產生興趣。1960年代[[美國|美國]]更掀起一陣巴薩諾瓦風潮，延燒到世界各地，影響了往後1970、80年代的電影及流行音樂。

== 在亞洲的發展 ==
[[日本|日本]]在1960年代中期就流行巴薩諾瓦，聞名國際的[[薩克斯風|薩克斯風]]手[[渡邊貞夫|渡邊貞夫]]就是日本推動巴薩諾瓦的先鋒。<ref>{{cite book|url=http://www.eslite.com/product.aspx?pgid=1004147041644296|title=渡邊貞夫: 流動現場實況終極精選|publisher=JVC VICTOR ENTERTAINMENT|date=2006-10-15|accessdate=2009-01-19}}</ref><ref>{{cite web|url=http://www.books.com.tw/exep/prod/av/av_persona_inside.php?cat=002&type=jz&source=persona&p_no=0000004975|publisher=博客來爵士音樂館|title=渡邊貞夫|accessdate=2009-01-19|deadurl=yes|archiveurl=https://web.archive.org/web/20151023152756/http://www.books.com.tw/exep/prod/av/av_persona_inside.php?cat=002&type=jz&source=persona&p_no=0000004975|archivedate=2015-10-23}}</ref>日本的許多音樂人常表演巴薩諾瓦；除渡邊貞夫外，還有[[李敬子|李敬子]]<ref>{{cite web|url=http://www.books.com.tw/exep/prod/av/av_persona_inside.php?cat=002&type=jz&source=persona&p_no=0000005704|title=李敬子|publisher=博客來爵士音樂館|accessdate=2009-01-19|deadurl=yes|archiveurl=https://web.archive.org/web/20160304113927/http://www.books.com.tw/exep/prod/av/av_persona_inside.php?cat=002&type=jz&source=persona&p_no=0000005704|archivedate=2016-03-04}}</ref>、[[小野麗莎|小野麗莎]]<ref>{{cite news|url=http://chinanews.sina.com/ent/2008/1209/01043130652.html|publisher=新浪網|title=巴薩諾瓦代表人物小野麗莎下月開唱|date=2008年12月9日|accessdate=2009-01-19}}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>、[[鈴木重子|鈴木重子]]等爵士藝人，都有不少巴薩諾瓦的作品，其中小野麗莎更被定位為巴薩諾瓦歌手。

踏入21世紀，巴薩諾瓦在[[台灣|台灣]]開始有流行的趨勢，頻頻在廣告中出現，可說是歷久彌新的「新節奏」。
