'''巴拉旺孔雀'''（''Polyplectron napoleonis''，以往稱為''Polyplectron emphanum''<ref name=Dickinson2001>{{Cite journal en|author=Dickinson, E. C.|year=2001|title=The correct scientific name of the Palawan Peacock-pheasant is ''Polyplectron napoleonis'' (Lesson, 1831)|journal=Bull. B. O. C.|volume=121|issue=4|pages=266-272}}</ref>），又名'''巴拉望孔雀雉'''，是中等身形的[[雉|雉]]。

== 特徵 ==
巴拉旺孔雀是[[灰孔雀雉屬|灰孔雀雉屬]]中最似[[孔雀|孔雀]]的雉。雄鳥有直立的冠，[[眼睛|眼睛]]上有白紋，身體呈[[金屬|金屬]]綠及黑色。尾羽有藍綠色的大眼斑點，示愛時可以展開成扇狀。雌鳥較雄鳥細小。雌鳥呈深褐色，冠較短，[[喉嚨|喉嚨]]、頰及眼眶呈白色。

== 分佈 ==
巴拉旺孔雀是[[菲律賓|菲律賓]]的[[特有種|特有種]]，分佈在[[巴拉望島|巴拉望島]]的潮濕[[森林|森林]]。雌鳥一般會生2隻蛋。

== 保育狀況 ==
由於持續失去棲息地、數量稀少及有限的分佈地，加上被捕獵，巴拉旺孔雀被[[世界自然保護聯盟|世界自然保護聯盟]]列為[[易危|易危]]<ref name=IUCN/>，且受到《[[瀕臨絕種野生動植物國際貿易公約|瀕臨絕種野生動植物國際貿易公約]]》附錄一的保護。
