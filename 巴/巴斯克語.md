'''巴斯克語'''（{{lang-eu|'''Euskara'''}}），是一种使用於[[巴斯克地區|巴斯克地區]]（[[西班牙|西班牙]]東北部的[[巴斯克|巴斯克]]和[[納瓦拉|納瓦拉]]兩個自治州，以及[[法國|法國]]西南部）的[[孤立语言|孤立语言]]。该语言在[[西班牙語|西班牙語]]稱作{{lang|es|vasco}}，在[[法語|法語]]稱作{{lang|fr|Basque}}，而巴斯克語裡則稱為{{lang|eu|euskara}}或{{lang|eu|euskera}}。巴斯克语的起源与系属争议很大。作为西欧唯一的孤立语言，巴斯克语与现存的任何语言没有[[语言发生学|发生学]]关系。

== 历史 ==
=== 起源 ===
[[File:Basque_dialects-en.svg|thumb]]
[[巴斯克人|巴斯克人]]的祖先可能是[[歐洲|歐洲]]遠古時代的居民（[[克羅馬儂人|克羅馬儂人]]）。巴斯克語的由來到目前仍然是個謎，有些人認為這種語言可能源自[[高加索|高加索]]、或是[[非洲|非洲]]北部，也有人类学家和语言学者认为它本來就诞生在[[伊比利亚半岛|伊比利亚半岛]]，是已消亡的伊比利亚语系的最后火种。儘管如此，大部分學者將巴斯克語歸類為[[孤立語言|孤立語言]]，并認為在[[羅馬人|羅馬人]]將[[拉丁語|拉丁語]]帶到[[伊比利亚半岛|伊比利亚半岛]]时，這種古老的語言已经在那里使用了很久（有一點比較肯定，巴斯克人比其他欧洲民族更早在欧洲定居）。有部份語言學家认为巴斯克語源於[[石器時代|石器時代]]，理由是该语言中的斧頭（aizkora）和石頭（haitz）[[詞根|詞根]]相同。

在[[法國|法國]]西南部[[阿基坦|阿基坦]]使用的語言（[[阿基坦語|阿基坦語]]）的拉丁注解中，找到了一些與原始巴斯克語的[[同源詞|同源詞]]，如在現代巴斯克語中分別解作女子和男人的“{{lang|eu|neskato}}”和“{{lang|eu|gizon}}”，在阿基坦語的標音被寫作“{{lang|xaq|nescato}}”和“{{lang|在xaq|cison}}”。而語言學者曾嘗試把巴斯克語與[[伊比利亞語|伊比利亞語]]、[[格魯吉亞語|格魯吉亞語]]、[[高加索诸语言|高加索诸语言]]如[[車臣語|車臣語]]，或嘗試設立[[德内-高加索语系|德内-高加索语系]]或獨立成為單獨的[[瓦斯科尼亞語系|瓦斯科尼亞語系]]，以使巴斯克語與其他語言聯繫起來。但這些假設還未有得到大多數人的認同。

=== 發展 ===
直至2017年數據，約有70萬人正在使用巴斯克語，佔[[巴斯克地區|巴斯克地區]]人口的35%。巴斯克語也曾是[[西班牙|西班牙]]獨裁者[[弗朗西斯科·佛朗哥|弗朗西斯科·佛朗哥]]攻擊的目標。在弗朗哥統治期間（1939-1975 年），他強行推廣[[西班牙語|西班牙語]]，禁止使用包括巴斯克語在內的其他語言。

== 书写 ==
現代巴斯克語使用[[拉丁字母|拉丁字母]]書寫。但根據[[考古|考古]]出土的[[文物|文物]]，可找到羅馬化之前使用[[伊比利字母|伊比利字母]]銘刻在器皿上的巴斯克語，而在納瓦拉則有中世紀使用[[阿拉伯字母|阿拉伯字母]]書寫巴斯克語的紀錄。

== 与其他语言的关系 ==
由於和鄰近地區人們交流的關係，巴斯克語在一定程度上受到[[拉丁語|拉丁語]]、[[西班牙語|西班牙語]]、[[法語|法語]]甚至[[阿拉伯語|阿拉伯語]]的影響，從它們當中借了些[[詞彙|詞彙]]。相對的，也向週邊的語言輸出[[詞彙|詞彙]]。

巴斯克語大量使用格變化，以及複雜的動詞變化。而這樣的語言特性在临近地区只有巴斯克語和有些高加索語言才具備。

== 分类 ==
根據Louis-Lucien Bonaparte的分類，巴斯克語可以區別出八種[[方言|方言]]，但並不與行政轄區重合。現時在學校教育中使用規範化的「標準巴斯克語」（{{lang|eu|euskara batua}}）則是以[[吉普斯誇省|吉普斯誇省]]的方言為基礎。

== 地位 ==
[[File:Basque_Country_Location_Map.svg|thumb]]

巴斯克語在[[西班牙|西班牙]]的[[巴斯克自治區|巴斯克自治區]]、[[納瓦拉自治區|納瓦拉自治區]]具有[[官方語言|官方語言]]地位。由於兩地{{CN|政治傾向}}的差異，使得法律稱呼巴斯克語的方法有別，前者稱{{lang|eu|euskara}}，後者稱{{lang|eu|vascuence}}。
