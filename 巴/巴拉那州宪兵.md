'''巴拉那州宪兵（[[葡萄牙语|葡萄牙语]]：''Polícia Militar do Paraná''）'''（PMPR）是[[巴西军队|巴西军队]]的後备及辅助部队，同时也是巴西[[公共安全|公共安全]]及[[社会保障|社会保障]]系统的一部分<ref>{{en icon}}[http://www.v-brazil.com/government/laws/titleV.html  巴西宪法第144条]</ref>。其成员成称为''Military's States''<ref>{{en icon}}[http://www.v-brazil.com/government/laws/titleIII.html 巴西宪法第42条]</ref>。

巴拉那州宪兵的主要任务是在[[巴拉那州|巴拉那州]]维护社会秩序方面进行简单的预防性治安管理。

== 历史 ==
1854年8月10日，巴拉那州宪兵被作为[[散兵|散兵]]单位创制，名为警卫队（''the Police Force''），规模相当于一个[[连|连]]。它的创建源于[[巴西帝国|巴西帝国]]在紧急情况下对于加强[[陆军|陆军]]部队的需要。[[巴西共和国|巴西共和国]]成立后,根据巴西宪法成为区域性的武装力量。

巴拉那州宪兵参与了一系列的军事活动：
* [[三国同盟战争|三国同盟战争]]（又称[[巴拉圭战争|巴拉圭战争]]）
* 联邦主义革命（''Federalist Revolution''）
* 回应战争（''Contestado War''）
* 1924年起义
* 1930年革命
* 1932年革命

直到1946年二战结束[[热图利奥·瓦加斯|热图利奥·瓦加斯]]的[[独裁|独裁]]政府被推翻，[[巴西|巴西]]一直笼罩在国家团结的危险境地之内。因而[[宪兵|宪兵]]自当时开始形成如今的配置，成为隶属于[[州|州]]的一类宪兵。

== 组织 ==
巴拉那州宪兵以[[营|营]]、[[连|连]]或[[排|排]]的形式组织运作；行政上分各个部门。宪兵营驻扎在主要城区中心，旗下的连与排根据人口密度驻扎于各个城市地区。[[巴拉那州|巴拉那州]]的所有城市目前均有巴拉那州宪兵设驻。

== 消防队 ==

== 制服 ==
从1854年至建立[[共和国|共和国]]的初期，巴拉那州宪兵穿着深蓝色的制服。1912年<ref>1912年8月31日，第774号法令</ref>，采用了黄褐色的新制服，并一直使用至今。除了不同的徽章外，巴拉那州宪兵消防队也穿着同样的制服。
