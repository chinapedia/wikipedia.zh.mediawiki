'''巴特尤公寓'''（{{lang|ca|Casa Batlló}}）是[[建筑师|建筑师]][[安东尼·高迪|安东尼·高迪]]同[[若热普-玛丽亚·茹若尔|若热普-玛丽亚·茹若尔]]（Josep Maria Jujol）合作装修改造的一座[[建筑|建筑]]。该建筑建于1877年，在1904年到1906年间接受改造。位于[[加泰羅尼亞|加泰羅尼亞]][[巴塞罗那|巴塞罗那]][[扩展区|扩展区]]（{{lang|ca|Eixample}}）的[[格拉西亚大道|格拉西亚大道]]（{{lang|ca|Passeig de Gràcia}}）43号。2005年被扩充入[[世界遗产|世界遗产]]安东尼·高迪的建筑作品中。

== 主厅 ==
整幢房子中最为精华的部分是巴特略全家住的地方，即，位于一楼的主厅({{lang|ca|Piano Nobile}})。它有单独的入口和楼梯体现了典型的高迪风格。高迪设计中独具一格且不可复制的建筑艺术特色，以及原汁原味的[[现代主义|现代主义]][[美术|美术]]与[[工艺|工艺]]精品，在主厅内随处可见。另外，主厅还通向一个现代主义风格的庭院，占地230[[平方米|平方米]]，为各类大型活动提供了绝佳的户外空间。

主厅占地730平方米，有内部楼梯通向''{{lang|ca|Jujol Rooms}}''和''{{lang|ca|Coach House - Coal Cellar}}''。

== 茹若尔房间（Jujol Rooms） ==
{{lang|ca|Jujol Rooms}}位于巴特略公寓的二楼，有华丽的楼梯通向这座房子裡的其他住宅单间，也有内部楼梯与主厅相连。{{lang|ca|''Jujol Rooms''}}包含两个现代主义风格的房间，一间朝向庭院，另一间俯看加西亚大街，现已由当代[[设计师|设计师]]完全復原，并达到了一种特别的效果，使其重新散發出住宅内所特有的华丽光辉与魔幻气息。

== 车房-煤窖（Coach House - Coal Cellar） ==
现代主义的[[天窗|天窗]]({{lang|ca|claraboia}})与其本身的内部[[高度|高度]]，使自然光能透进这一两层楼高的空间。从加西亚大街上，可以通过单独的入口进入这里，因而十分适合举办大型庆典或是其他活动。入口正面有12米宽，这使其成为了一处商业用途广泛的场所，比如可举行展览、陈列或其它集会。

就更为个人的庆祝活动而言，这里不仅仅可为大型招待会或节日晚宴提供场所，下层著名的''{{lang|ca|Coal Cellar}}''更是举行派对的好地方。

''{{lang|ca|Coach House - Coal Cellar}}''内部空间宽敞，总面积达1100平方米，可同时容纳600人。这里还配备有一些特殊活动所需的设施，且也有内部楼梯通往住宅内的其它地方。

== 阁楼、屋顶和烟囱 ==
阁楼在整修一新后向公众开放，以纪念巴特略公寓建成一百周年。同时开放的还有烟囱和别具一格的屋顶。前者可以说是高迪最受人喜爱的设计了；而在屋顶的露台上则能从一种独一无二的角度俯瞰巴塞罗那。这些都是原汁原味、令人着迷的典型高迪杰作。高迪在这里似乎营造了一个属于[[童话|童话]]中的世界，[[自然|自然]]以一种全新的方式展现在人们眼前，[[四大元素|四大元素]]({{lang|en|The Four Elements}})成为了舞台上的主角，想象得以自由驰骋。

一百年来首次对公众开放，这一非凡建筑的每一个角落都能给人惊喜。比如：有如[[胸腔|胸腔]]一般的华丽[[拱顶|拱顶]]、神奇的烟囱、以及屋顶上著名的[[三维|三维]][[十字架|十字架]]等。

巴特略公寓有两个阁楼开放。位于楼顶的“龙腹”("{{lang|en|The Dragon's Belly}}")中，高迪使用了其精巧的[[悬链拱|悬链拱]]。而“火的空间”("{{lang|en|The Fire Space}}")，则是由当代建筑师[[英葛·摩利尔|英葛·摩利尔]]({{lang|ca|Ingo Maurer}})设计的，是一个[[灯光|灯光]]与[[视觉|视觉]]的奇幻空间。其余各处仿佛真实的童话，愉快而怀旧，散发着巴塞罗那浓郁的现代主义生活气息。在这里，成人与孩童一样都会浮想联翩。烟囱在晚上会亮起灯光。在屋顶400m<sup>2</sup>的露台上，可以享受[[地中海|地中海]]式的户外生活和巴塞罗那的怡人[[气候|气候]]。
