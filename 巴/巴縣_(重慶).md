'''巴县'''，[[中国|中国]]旧[[县|县]]名，初设于北周，一直是[[重庆市|重庆]][[附郭县|附郭大县]]，是重庆母县，后成为重庆的近郊区县，行政区域历代不断减小。1994年撤销并划分给了多个区县，由[[巴南区|巴南区]]继承了它的主要行政区域。

== 历史沿革 ==
[[东周|周]]慎靓王五年（前316年）[[秦国|秦]]灭[[巴国|巴国]]后，在[[巴国|巴国]]旧地置[[江州縣|江州县]]，隶于[[巴郡|巴郡]]。[[南齐|南齐]][[永明|永明]]五年（487年），析江州县地置[[垫江县_(南齐)|垫江县]]，以其故治为垫江县治（江州县治徙僰溪口，位于今[[江津区|江津]]区内）。[[北周|北周]][[武成_(北周)|武成]]三年（561年）废垫江、[[枳縣|枳]]两县，以其地置巴县，以垫江县故治为巴县治。初置县时，巴县辖境为今[[重庆主城区|重庆主城区]]大部及[[长寿区|长寿]]、[[璧山区|璧山]]、[[涪陵区|涪陵]]、[[南川区|南川]]、[[武隆区|武隆]]等县的部分地区，隶[[楚州_(南梁)|楚州]][[巴郡|巴郡]]。<ref name=":0">{{Cite web|url=http://bn.cq.gov.cn/bnkq.asp?id=6360|title=行政区划|accessdate=2018-07-26|author=|date=|work=bn.cq.gov.cn|publisher=重庆市巴南区人民政府网站}}</ref>

[[隋朝|隋]][[开皇|开皇]]二年（583年）改楚州为渝州，为[[渝州|渝州]][[治所|州治]]。[[唐朝|唐]][[貞觀_(唐朝)|贞观]]四年（630年），析巴县地置[[南平县_(唐朝)|南平县]]（在今[[綦江区|綦江区]]内），并设[[南平州|南平州]]辖南平、[[清谷县|清谷]]、[[周泉县|周泉]]等七县，八年改为[[霸州_(贞观)|霸州]]，十三年州废，辖县省入南平，属渝州。贞观十一年（637年），析置[[隆化县_(重庆古县份)|隆化县]]（今[[南川区|南川区]]）属[[涪州_(唐朝)|涪州]]。[[北宋|北宋]][[崇宁|崇宁]]元年（1102年），渝州改为[[恭州_(北宋)|恭州]]，仍为州治。[[南宋|南宋]][[淳熙|淳熙]]十六年（1189年），恭州升为[[重庆府|重庆府]]，为府治。[[元朝|元]][[至元_(忽必烈)|至元]]十五年（1278年）置[[重庆路|重庆路]]，为路治。[[明朝|明]][[洪武|洪武]]四年（1376年）改重庆路为[[重庆府|重庆府]]，为府治。[[清朝|清]][[乾隆|乾隆]]二十四年（1759年）分置[[江北厅|江北厅]]。<ref name=":0" /><ref>{{Cite book|title=[[中国历史地图集|中国历史地图集]]|last=|first=|author=[[谭其骧|谭其骧]]|publisher=[[中国地图出版社|中国地图出版社]]|year=1982|isbn=9787503118449|location=北京|pages=}}</ref><ref name="府志">{{cite book |author1=王梦庚 |title=重庆府志 |date=1843（清道光二十三年）}}</ref>[[新唐书|新唐书]]对巴县评价为中下<ref>《[[新唐书|新唐书]]·志第三十二·地理六》：“巴，中下。有鹽……”</ref>，[[元史|元史]]对巴县评价为下<ref>《元史·志第十二·地理三》：“巴縣，下。倚郭……”</ref>，[[清史稿|清史稿]]对巴县评价为[[衝繁疲難|冲、繁、难]]<ref>《[[清史稿|清史稿]]·志四十四·地理十六》：“巴沖，繁，難。倚。城內巴山，縣以此名。東：塗山。又北：太華。西：逾越、縉雲。南：霖峰。縣東有明月峽者，大江逕此。大江自江津入，逕城東南，又東北入江北廳。嘉陵江即涪江，自合州入，南流至城東，與大江合。東：丹溪自綦江入，交龍溪自長壽入，俱入大江。巡司一，駐木洞鎮。西：佛圖關。驛二：朝天、白市。”</ref>。

[[辛亥革命|辛亥革命]]后一度由[[重庆蜀军政府|重庆蜀军政府]]管辖。[[民國紀年|民国]]元年（1912年）改巴县为重庆府，次年废重庆府恢复巴县，属[[川東道|川东道]]，后划分至[[四川省_(中華民國)|四川省]]第三[[行政督察區|行政监察区]]。<ref name=":0" />

1950年，属[[川东行政区|川东行区]][[璧山专区|璧山专区]]，1951年1月划归[[重庆市|重庆市]]（[[直辖市#_中华人民共和国|直辖市]]，[[西南行政委员会|西南大行政区]]代管）。1953年2月，重新划入[[四川省|四川省]][[江津专区|江津专区]]（原璧山专区）。1958年11月，巴县再度划入[[重庆市|重庆市]]。自民国十八年（1930年）重庆建市后，为适应重庆市区扩展，巴县县界曾多次调整。至1994年，县境东西宽73公里，南北长70公里，仍三面环绕重庆市区。1994年12月，经国务院批准，撤销巴县，划出长江以北的19个乡镇分属[[沙坪坝区|沙坪坝]]、[[大渡口区|大渡口]]、[[九龙坡区|九龙坡]]三区，划出长生桥等3个镇属[[南岸区|南岸区]]，划入九龙坡区2个街道与2个镇和原巴县长江以南的39个乡镇成立新的[[巴南区|巴南区]]。<ref name=":0" /><ref name=":1">{{Cite web|url=http://www.xzqh.org/html/show/sc/23428.html|title=巴南区历史沿革_行政区划网(区划地名网) www.xzqh.org|accessdate=2011.2.25|author=|date=|publisher=}}</ref><ref name=":2">{{cite book |author1=陈潮 |coauthors=陈洪玲 |title=中华人民共和国行政区划沿革地图集 |date=2003.10 |publisher=中国地图出版社 |location=北京 |isbn=978-7-5031-3184-4 |page=124-126,211 |edition=第一版}}</ref>

== 行政区划 ==
由于重庆城市发展和巴县所辖区域过大，历史上巴县曾多次划出区域，如唐朝分置[[南平县_(唐朝)|南平县]]、[[隆化县_(重庆古县份)|隆化县]]等。[[宋朝|宋朝]]时巴县下辖四镇。[[明朝|明朝]]时巴县下辖七十二里，城内分为八坊，另有附郭二庙<ref name="县志" />。县东有涂山，[[长江|大江]]、[[嘉陵江|涪江（嘉陵江）]]与城东交汇，县内有佛图关、二郎关、铜锣关、南坪关<ref>《[[明史|明史]]·志第十九·地理四》：“巴倚。東有塗山。大江經城南，又東經明月峽，至城東，與涪江合。西北有魚鹿峽，涪江所經。東南有丹溪，東北有交龍溪，俱流入大江。東有大紅江巡檢司。西有佛圖關。西南有二郎關。東有銅鑼關。又南有南坪關。”</ref>。

[[清朝|清]][[康熙|康熙]]四十六年(1707年）巴县下辖十二里，城内分为二十九坊、附郭十五庙<ref name="县志">{{cite book |author1=朱之洪 |title=巴县志 |date=1937（民国二十六年）}}</ref>。[[乾隆|乾隆]]二十四年（1759年）以巴县辖区过大分江北至义、礼二里及仁里六甲入[[江北厅|江北厅]]（理民同知管辖），分[[缙云山|缙云山]]西祥里、依来乡入[[璧山县|璧山县]]。至[[道光|道光]]年间，巴县辖境东至沙溪交[[长寿县|长寿县]]界二百里，西至走马冈交璧山县界八十里，南至黎牛铺交[[綦江县|綦江县]]界一百五里，北过江北厅界通江面五里陆路至风垭一百七十里交[[合州|合州]]界。<ref name="府志"/>

民国十八年（1929年），巴县辖10区、4镇、76乡。1949年，巴县辖12指导区、5镇、65乡。1985年，巴县辖11个区、1个区级镇（直管镇）、10个乡级镇、79个乡<ref name=":0" />：

* [[县辖区_(中华人民共和国)|区]]：铜罐驿、白市驿、虎溪、青木关、长生桥、木洞、姜家、接龙、一品、界石、鱼洞
* [[县辖区_(中华人民共和国)|区级镇]]：西彭
* [[行政建制镇|乡级镇]]：铜罐驿、白沙沱、白市驿、陈家桥、青木关、长生桥、木洞、一品、界石、鱼洞
* [[乡级行政区|乡]]：元明、含谷、走马、曾家、西永、迎龙、栋青、天赐、小观、陈家、忠兴、大窝等

1993年，巴县进行并乡建镇，撤销了全部县辖区与60个乡、1个镇，新成立了36个镇。

1994年，巴县共辖47个镇、14个乡，共1034个村、8153个合作社、106个居委会。12月，经国务院批准撤销巴县，成立巴南区，并对重庆市市辖区进行了一系列行政区划调整。撤销前行政区划与变更为<ref name=":0" /><ref name=":1" /><ref name=":2" />：

=== 县治 ===
{{seealso|重庆城墙}}
自[[东周|周]][[周赧王|赧王]]元年（公元前314年）秦在巴国旧域设[[江州縣|江州县]]至民国二十八年（1939年）止，巴县（含江州县、[[垫江县_(南齐)|垫江县]]）县治均设于今重庆市[[渝中区|渝中区]]，由巴县历来为重庆州治、路治、府治，县治在[[重庆城墙|重庆城墙]]内，今存巴县衙门为[[清朝|清]][[康熙|康熙]]六年（1667年）知县张冉重修。<ref name=":1" /><ref>{{Cite news|url=https://baijiahao.baidu.com/s?id=1589030649986082133&wfr=spider&for=pc|title=重庆珍档 - 重庆老城现存最后一座衙门——巴县衙门旧址将启动保护修缮|author=|date=|work=上游新闻|accessdate=2018-07-27}}</ref>

民国二十八年（1939年），县治外迁至人和乡（今[[九龙坡区|九龙坡区]][[华岩镇|华岩镇]]），民国三十年（1941年）迁往李家沱马王坪。1952年10月，县治迁往南温泉。1954年1月，县治迁往鱼洞镇（今[[鱼洞街道|鱼洞街道]]），直至1994年撤销。巴南区成立后区政府仍设于鱼洞镇。2009年11月，经国务院批准，巴南区人民政府驻地迁至[[龙洲湾街道|龙洲湾街道]]龙海大道。<ref name=":0" /><ref name=":1" />

== 人口 ==
按[[清朝|清]][[嘉庆|嘉庆]]元年（1796年）重庆府属州县旧额新增户口统计，巴县共75743户，男119689丁，妇99090丁，共218779丁。<ref name="府志"/>

1994年，巴县统计局年鉴报告，1994年末全县总户数39.41万户（其中农业户数34.15万户，非农业户数5.26万户），总人口126.21万人（其中农业人口108.9万人，非农业人口17.31万人；男性65.34万人，女性60.88万人）。<ref name=":0"/>
