'''巴索夫號巡防艦'''（USCGC Bertholf WMSL-750）是一艘隸屬於[[美國海岸防衛隊|美國海岸防衛隊]]的大型[[海巡艦|海巡艦]]（Cutter），是一系列{{link-en|傳奇級巡防艦|Legend-class cutter|傳奇級}}（Legend Class）遠洋巡防艦（National Security Cutter, NSC）中的一號艦。於2005年3月29日起造，2008年8月4日正式服役的巴索夫號是以第四任[[美國海岸防衛隊司令|美國海岸防衛隊司令]]、{{link-en|艾爾沃斯·巴索夫|Ellsworth P. Bertholf}}（Ellsworth P. Bertholf）准將命名，[[母港|母港]]為[[加州|加州]][[阿拉米達_(加利福尼亞州)|阿拉米達]]。

巴索夫號與其他傳奇級的姊妹艦，在分級上屬於大型海上保安艦（Maritime Security Cutter, Large）等級，因此使用「WMSL」字首的艦身編號，主要功能為擔任美國海岸防衛隊各遠洋巡護區之指揮艦(旗艦)。2005年起在{{link-en|諾斯洛普·格魯曼造船系統|Northrop Grumman Ship Systems}}（Northrop Grumman Ship Systems, NGSS）位於[[密西西比州|密西西比州]][[帕斯卡古拉_(密西西比州)|帕斯卡古拉]]的[[英戈爾斯造船廠|英戈爾斯造船廠]]起造，是美國海岸防衛隊一個稱為{{link-en|整合深水系統計畫|Integrated Deepwater System Program|整合深水系統}}（Integrated Deepwater System, IDS，或常簡稱為「深水計畫」）的25年設備汰換計畫的一環，以取代老舊的[[漢米爾頓級緝私艦|漢米爾頓級緝私艦]]（Hamilton-class）。[[排水量|排水量]]屬4500噸級，船長127.4公尺（418呎），最高航速可達28[[節|節]]，雖然是艘海巡艦但尺碼並不會比海軍使用的[[巡防艦|巡防艦]]或甚至[[驅逐艦|驅逐艦]]遜色。

巴索夫號主要的武裝是一具安裝在艦首處、由自動砲火控制系統所操作的{{link-en|波佛斯57公釐砲|Bofors 57 mm gun|57公釐波佛斯甲板砲}}，具有在海上遠距離射擊走私船的能力以達到喝阻效果。除此之外，遠較一般同類艦隻大上許多的艦尾甲板，能夠容納並提供兩架大型的{{link-en|HH-65海豚式直昇機|Eurocopter HH-65 Dolphin|MH-65C海豚式多功能海防艦載直昇機}}（MH-65C Dolphin MCH）起降，該型直昇機擁有比一般海岸防衛隊艦載直昇機還強大的武裝火力，也連帶提升了巴索夫號的總體戰力。在巴索夫號的艦尾，設有可發射與回收高速[[橡皮艇|橡皮艇]]用的艦尾坡道，這也是海岸防衛隊的巡防艦中第一艘使用此設計的範例。

巴索夫號上編制有113名乘員（包含14名軍官）。首任艦長是約翰·普萊斯上校（John F. Prince, CAPT）。
