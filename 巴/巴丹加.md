'''巴丹加'''（Badinga）為[[布吉納法索|布吉納法索]]的一座城鎮，由[[布克萊迪穆翁大區|布克萊迪穆翁大區]][[巴瓦省|巴瓦省]][[巴拉韋縣|巴拉韋縣]]管轄。2005年人口普查中該城鎮人口有1,948人<ref>[http://www.inforoute-communale.gov.bf/list_vill/bm.htm Burkinabé government inforoute communale] {{webarchive|url=https://web.archive.org/web/20080124133758/http://www.inforoute-communale.gov.bf/list_vill/bm.htm |date=2008-01-24 }}</ref>。

==氣候==
巴丹加的年均溫度為27℃，其中4月為最炎熱的月份，平均溫度可達到33℃；而8月為最寒冷的月份，平均溫度只有22℃<ref name = "nasa">{{cite web|url=http://neo.sci.gsfc.nasa.gov/dataset_index.php|title=NASA Earth Observations Data Set Index|publisher=NASA|accessdate=2016-01-30}}</ref>。該城鎮的年平均降雨量為1,063毫米，其中8月為最潮濕的月份，平均降雨量達308毫米；而12月為最乾燥的月份，平均降雨量只有2毫米<ref name = "nasarain">{{cite web|url=http://neo.sci.gsfc.nasa.gov/view.php?datasetId=TRMM_3B43M&year=2014|title=NASA Earth Observations: Rainfall (1 month - TRMM)|publisher=NASA|accessdate=2016-01-30}}</ref>。
