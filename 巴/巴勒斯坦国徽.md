'''[[巴勒斯坦国|巴勒斯坦国]]国徽'''取材类似其自治政府；[[巴勒斯坦民族权力机构|巴勒斯坦民族权力机构]]的徽章，又类似于周边[[阿拉伯世界|阿拉伯国家]]的国徽图案。目前尚为非正式国徽<ref>{{Cite web |url = http://www.arabsino.com/articles/10-05-07/974.htm |title=巴勒斯坦国家概况 |publisher=阿拉伯信息交流中心 |date=2010-11-18 |accessdate=2010-12-25 |language=zh-cn}}</ref>。采用金黄色与黑色的[[薩拉丁之鹰|薩拉丁之鹰]]。其中心为一白边盾牌，上有竖挂的[[巴勒斯坦国旗|巴勒斯坦国旗]]的红、绿、白、黑四色<ref>{{Cite web |url = http://www.crwflags.com/fotw/flags/ps.html#ver |title=Palestine |trans_title=巴勒斯坦 |publisher=[[世界旗帜网|世界旗帜网]]|date=2009-05-16 |accessdate=2010-12-25 |language=en}}</ref>，这些都是[[泛阿拉伯主义|泛阿拉伯主义]]的代表颜色。鹰抓抓住弯匾，匾额上用巴勒斯坦国的官方语言[[阿拉伯语|阿拉伯语]][[库法体|库法体]]书写通称国名：“巴勒斯坦”（{{lang-ar|فلسطين}}，转写：''Filasṭīn''）。巴勒斯坦国的政府旗上有国徽图案，具体为国徽下加一对阿拉伯式弯刀镶于[[巴勒斯坦国旗|国旗]]黑条左上方<ref>{{Cite web |url  =http://www.crwflags.com/fotw/flags/ps_state.html |title=State Flag (Palestine) |trans_title=政府旗（巴勒斯坦）|publisher=[[世界旗帜网|世界旗帜网]] |date=2009-05-16 |accessdate=2010-12-25 |language=en}}</ref>。
