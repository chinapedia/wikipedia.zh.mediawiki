[[法國|法國]]首都[[巴黎|巴黎]]市內建有許多[[橋樑|橋樑]]，當中許多是為跨越[[塞納河|塞納河]]而建，另外亦有橫渡[[烏爾克運河|烏爾克運河]]的橋。

==統計==
''來源：[https://web.archive.org/web/20081210010912/http://www.paris.fr/portail/deplacements/Portal.lut?page_id=19&document_type_id=5&document_id=4910&portlet_id=652 巴黎市政府網站]''


截至2006年，巴黎市內共有：
* 37座跨越[[塞納河|塞納河]]的橋樑
* 58座[[行車天橋|行車天橋]]
* 10座由[[巴黎大眾運輸公司|巴黎大眾運輸公司]]（RATP）使用的橋樑
* 33座[[法國國家鐵路|法國國家鐵路]]（SNCF）使用的橋樑
* 148座跨越城市邊界[[環城大道|環城大道]]的橋樑
* 49座[[行人天橋|行人天橋]]


==塞納河==
巴黎市內的塞納河上共有37座橋，其中4座為行人專用橋、2座為[[鐵路橋|鐵路專用橋]]。3座連接[[聖路易島_(巴黎)|聖路易島]]與巴黎市其他地方，8座連接[[城岛|西堤島]]（或譯“城島”），還有1座連接上述兩島。下表列出所有橋樑，由上游至下游方向排列：
===聖路易島以東===
# '''[[上游橋|上游橋]]'''（{{lang-fr|Pont amont}}），沒有官方名稱的一座行車橋，1969年啟用。承載[[環城大道|環城大道]]，位於[[巴黎第十二區|巴黎十二]]及[[巴黎第十三區|十三區]]交界、塞納河進入巴黎市範圍處。
# '''[[國家橋|國家橋]]'''（{{lang-fr|Pont National}}），道路及鐵路混合橋，1853年啟用。1870年之前稱為“[[拿破崙三世|拿破崙三世]]橋”。連接左岸（十三區）的讓-西蒙將軍大道及右岸（十二區）的波尼亞托夫斯基大道。
# '''[[托比亞克橋|托比亞克橋]]'''（{{lang-fr|Pont de Tolbiac}}），1882年啟用。1943年一架被擊落的英國飛機撞上此橋。
# '''[[西蒙·波娃行人橋|西蒙·波娃行人橋]]'''（{{lang-fr|Passerelle Simone-de-Beauvoir}}），行人專用橋，於2006年7月13日啟用，以法國著名[[作家|作家]]及[[女權主義|女權主義]]者[[西蒙·波娃|西蒙·波娃]]命名。
# '''[[貝西橋|貝西橋]]'''（{{lang-fr|Pont de Bercy}}），道路及鐵路混合橋，其中鐵路層承載了[[巴黎地鐵|巴黎地鐵]][[巴黎地鐵6號線|6號線]]。
# '''[[夏爾·戴高樂橋|夏爾·戴高樂橋]]'''（{{lang-fr|Pont Charles-de-Gaulle}}），1996啟用。以法國總統[[夏爾·戴高樂|夏爾·戴高樂]]命名。
# '''[[奥斯特利茨高架鐵路橋|奥斯特利茨高架鐵路橋]]'''（{{lang-fr|Viaduc d'Austerlitz}}），1904年啟用，一座鐵路專用[[拱橋|拱橋]]，承載巴黎地鐵[[巴黎地鐵5號線|5號線]]。
# '''[[奥斯特利茨橋|奥斯特利茨橋]]'''（{{lang-fr|Pont d'Austerlitz}}），原橋於1805年啟用，後於1854及1884年先後進行重建和修繕。

===聖路易島===

# '''[[蘇利橋|蘇利橋]]'''（{{lang-fr|Pont de Sully}}），現橋於1877年啟用。連接左右兩岸（[[巴黎第四區|四區]]及[[巴黎第五區|五區]]）與聖路易島東沿，理論上為兩段獨立的橋。
# '''[[托內爾橋|托內爾橋]]'''（{{lang-fr|Pont de la Tournelle}}），現橋於1928年啟用。連接聖路易島與左岸。前身為[[中世紀|中世紀]]時期一座木橋，1651年木橋被洪水沖毀後復於1658年建起另一座石橋。1918年拆卸後再建起現橋。法語中''Tournelle''是「炮塔」的意思，得名自十二世紀末建於[[腓力二世_(法兰西)|腓力二世]]堡壘上的一座方型炮塔。
# '''[[瑪利橋|瑪利橋]]'''（{{lang-fr|Pont Marie}}），啟用於1635年，巴黎市第二古老的橋樑，僅次於新橋。連接聖路易島與右岸。得名自提倡興建此橋的工程師克里斯托夫·瑪利。
# '''[[路易-菲利普橋|路易-菲利普橋]]'''（{{lang-fr|Pont Louis-Philippe}}），啟用於1834年。位於巴黎第四區，連接聖路易島與右岸。得名自法國國王[[路易-菲利普一世|路易-菲利普一世]]。

===聖路易島－西堤島===

# '''[[聖路易橋|聖路易橋]]'''（{{lang-fr|Pont Saint-Louis}}），橫亙在塞納河兩大天然河島－西堤島和聖路易島之間，是塞納河巴黎市段唯一一座不與左右任何一岸連接的橋。

===西堤島===

# '''[[大主教橋|大主教橋]]'''（{{lang-fr|Pont de l'Archevêché}}），建於1882年，是巴黎行車天橋中最窄的一座，橋寬11米，可用寬度更僅11米。自從[[藝術橋|藝術橋]]和[[利奧波德-塞達-桑戈爾行人橋|利奧波德-塞達-桑戈爾行人橋]]上的情人鎖被清除，情侶們開始轉為在這條橋上掛上[[爱情锁|情人鎖]]。
# '''[[道布勒橋|道布勒橋]]'''（{{lang-fr|Pont au Double}}），意譯為“雙倍橋”。現橋建於1883年。[[弗朗索瓦一世_(法兰西)|法蘭索瓦一世]]應要求興建一座橋往西堤島以方便運送病人往島上的{{link-fr|主宮醫院|Hôtel-Dieu de Paris}}。過橋費是兩個（故稱「雙倍」）丹尼爾（{{lang-fr|denier}}）銀幣，故得此名。
# '''[[阿爾科萊橋|阿爾科萊橋]]'''（{{lang-fr|Pont d'Arcole}}），啟用於1856年。啟用的最初兩年被稱為“河灘行人橋”（{{lang-fr|passerelle de Grève}}）或“[[巴黎市政廳|市政廳]]橋”（{{lang-fr|pont de l'Hôtel-de-Ville}}），改為現名的最廣為接納的來源，是1796年拿破崙於[[意大利|意大利]][[阿爾科萊|阿爾科萊]]之橋戰役擊敗[[奧地利|奧地利]]人。
# '''[[小桥_(巴黎)|小橋]]'''（{{lang-fr|Petit Pont}}），於1853年落成。連接左岸與西堤島。這座僅32米、名副其實的「小橋」命途多舛，在現存的橋於1853年落成之前至少十三次被洪水沖毀或大火燒燬。
# '''[[聖母橋|聖母橋]]'''（{{lang-fr|Pont Notre-Dame}}），現橋於1919年啟用。連接左岸與西堤島。原址是巴黎建起的第一座橋－「大橋」（{{lang-fr|Grant-Pont}}）。十五世紀時期的版本，橋上建有多達六十幢幾層高的房屋，後來橋亦因此而超重倒塌。現橋由[[法蘭西第三共和國|法蘭西第三共和國]][[法國總統|總統]][[雷蒙·普恩加萊|雷蒙·普恩加萊]]揭幕。
# '''[[圣米歇尔桥|圣米歇尔桥]]'''（{{lang-fr|Pont Saint-Michel}}），目前的橋樑修建於1857年。連接左岸與西堤島。得名於附近的聖彌額爾小堂。
# '''[[兑换桥|兑换桥]]'''（{{lang-fr|Pont au Change}}），連接右岸與西堤島。十二世紀時期版本的橋上開滿的金匠店和兌換店，因而得名。
# '''[[新橋_(巴黎)|新橋]]'''（{{lang-fr|Pont Neuf}}），連接左岸、西堤島西端和東岸。雖然名為「新」，卻是巴黎現存最古老的橋。1578年開始建造，1607年落成使用至今。它是巴黎最早的石橋，也曾經是巴黎最寬闊的橋。

===西堤島以西===

# '''[[艺术桥|艺术桥]]'''（{{lang-fr|Passerelle des Arts}}），連接[[法蘭西學會|法蘭西學會]]和[[羅浮宮|羅浮宮]]中央廣場（[[法蘭西第一帝國|第一帝國]]時稱為“藝術宮”，故得名）。原橋建成於1804年，現橋則由時任巴黎市長的[[雅克·希拉克|雅克·希拉克]]於1984年揭幕。行人專用橋。2008年底開始，情侶們在橋的欄柵上掛上鎖。2014年6月，部分柵欄抵受不住重壓而倒下，沒有造成傷亡。
# '''[[卡魯塞爾橋|卡魯塞爾橋]]'''（{{lang-fr|Pont du Carrousel}}），連接[[杜伊勒里碼頭|杜伊勒里碼頭]]和伏爾泰碼頭，鄰近[[王宮-羅浮宮站|王宮-羅浮宮站]]。原名“聖佩雷橋”（Pont des Saints-Pères）。由於此橋鄰近[[卡魯索凱旋門|卡魯索凱旋門]]，1834年由路易-菲利普一世易為現名。另由於鄰近羅浮宮，此橋又別稱“羅浮宮橋”。
# '''[[皇家橋|皇家橋]]'''（{{lang-fr|Pont Royal}}），現橋落成於1850年，是巴黎第三古老的橋樑，僅次於新橋和瑪利橋。[[路易十四|路易十四]]賜名“皇家橋”，法國大革命時易名為“國家橋”，第一帝國時期改為“杜伊勒里橋”，最後由[[路易十八|路易十八]]恢復其原名。
# '''[[利奧波德-塞達-桑戈爾行人橋|利奧波德-塞達-桑戈爾行人橋]]'''（{{lang-fr|Passerelle Léopold-Sédar-Senghor}}），行人專用橋。連接左岸的[[奧賽博物館|奧賽博物館]]和右岸的[[杜樂麗花園|杜樂麗花園]]。舊稱“蘇法利諾行人橋”（{{lang-fr|Passerelle de Solférino}}），得名自意大利[[索尔费里诺战役|索尔费里诺战役]]。2006年時改為現名，以紀念[[塞內加爾|塞內加爾]]詩人、政治家暨首任總統[[利奧波德·塞達爾·桑戈爾|利奧波德·塞達爾·桑戈爾]]。
