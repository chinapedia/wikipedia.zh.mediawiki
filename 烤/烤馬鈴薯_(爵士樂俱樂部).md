'''烤馬鈴薯'''（{{lang-en|The Baked Potato}}）是一間位於[[洛杉磯|洛杉磯]]{{link-en|影視城 (洛杉磯)|Studio City, Los Angeles|影視城}}的{{link-en|卡漢加大道|Cahuenga Boulevard}}上的爵士樂俱樂部，於1970年由{{link-en|唐·藍迪|Don Randi}}開張。多年以來，這裏招待過很多融合爵士樂手的現場錄音。1986年，[[賴瑞·卡爾頓|賴瑞·卡爾頓]]在這裏錄製"{{link-en|Last Nite (Larry Carlton album)|Last Nite (Larry Carlton album)|Last Nite}}"。2010年，烤馬鈴薯被《{{link-en|洛杉磯雜誌|Los Angeles (magazine)}}》命名為洛杉磯最佳爵士樂俱樂部。2016年5月21日，[[麥加帝斯|麥加帝斯]]樂團的{{link-en|Nick Menza|Nick Menza}}於台上表演期間心臟病發而死亡。<ref>{{Cite web |url = http://www.billboard.com/articles/columns/rock/7378350/nick-menza-former-megadeth-drummer-dies-on-stage-at-51-ohm-dave-mustaine |title = Nick Menza, Former Megadeth Drummer, Dies On Stage at 51 |author = Mitchell Peters |date = 2016-05-22 |publisher = Billboard |language = en}}</ref>

俱樂部在2016年歌舞電影《[[星聲夢裡人|星聲夢裡人]]》中被提及過。<ref>{{Cite web |url = http://dailytrojan.com/2017/02/08/misunderstood-art-theres-jazz-la-la-land/ |title = Misunderstood art: There's more to jazz than La La Land	 |author = Max Kapur |date = 2017-02-08 |publisher = Daily Trojan |language = en}}</ref>
