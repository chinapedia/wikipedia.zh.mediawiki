'''英法聯軍攻陷北京'''是[[第二次鴉片戰爭|第二次鴉片戰爭]]之一部分。當時英法聯軍於[[八里橋之戰|八里橋大敗清軍]]後進攻[[清朝|清朝]][[首都|首都]][[北京|北京]]，[[圓明園|圓明園]]也在期間被[[英國軍隊|英國軍隊]][[縱火|縱火]]燒毀<ref name="新華電" />。這是有史以來[[北京|北京]]第一次被[[歐洲|歐洲]][[軍隊|軍隊]]占領，英法聯軍一共占領北京18天<ref name="近代中國史綱">{{zh-cn}} [[郭廷以|郭廷以]]，《[[近代中國史綱|近代中國史綱]]》。格致出版社（2009年04月1日），ISBN 9787543215849</ref>。

清政府因為京城淪陷的恥辱震驚，[[恭親王|恭親王]][[奕訢|奕訢]]主張學習西方技術，出現後來的[[洋務運動|洋務運動]]。

== 背景 ==
=== 八里橋的潰敗 ===
1857年末，英國和法國共組聯軍進攻廣州，第二次鴉片戰爭正式爆發。

1858年，清朝被迫和英法聯軍議和《[[天津條約|天津條約]]》，但和談並未達成再度開戰，大沽口前後經歷了三次激戰。

1860年8月，大沽口再度陷落。聯軍接連攻下[[天津|天津]]、[[張家灣|張家灣]]直逼[[通州|通州]]。清軍都督[[僧格林沁|僧格林沁]]為保衛京師前的最後一道防線，在[[郭家墳|郭家墳]]到[[八里橋|八里橋]]一帶佈署重兵，派出3萬清軍騎兵隊和蒙古騎兵與聯軍士兵5000人決戰。9月21日，[[八里橋戰役|八里橋戰役]]開打，清軍兵分東、西、南三路迎擊英法聯軍。未料[[勝保|勝保]]部的清軍馬隊被法軍率先擊潰，西路被攻破。最後，聯軍對漢白玉的八里橋面上刺刀發起總攻。清軍3萬馬隊死傷慘重，僧格林沁同少數清軍潰退到[[北京|北京]]，就這樣通往北京的大道已經大開<ref name="新華電">{{zh-cn}}{{zh-hant}}[http://news.xinhuanet.com/mrdx/2010-10/22/c_13569527.htm 八裏橋潰敗，武器的退步與清廷的傻] {{webarchive|url=https://web.archive.org/web/20141002042920/http://news.xinhuanet.com/mrdx/2010-10/22/c_13569527.htm |date=2014-10-02 }}（繁體）、{{cite web|title=簡體|url=http://news.xinhuanet.com/mrdx/2010-10/22/c_13569527.htm|publisher=[[新華每日電訊|新華每日電訊]]|date=2010-10-22|accessdate=2011-10-07|deadurl=yes|archiveurl=https://web.archive.org/web/20141002042920/http://news.xinhuanet.com/mrdx/2010-10/22/c_13569527.htm|archivedate=2014-10-02}}</ref>。

=== 咸丰帝出逃 ===
早在1860年9月9日，清朝欽差大臣[[桂良|桂良]]確定天津和談失敗後，[[咸豐帝|咸豐帝]]頒布[[朱諭|朱諭]]稱：
{{引用|[[桂良|桂良]]等奏，夷務決裂情形。
覽奏何勝憤怒！[[朕|朕]]為近畿百姓免受荼毒，不得已勉救撫局，乃該夷屢肆要挾，勢不決戰不能。
況我[[滿人|滿]]、[[漢人|漢]]臣僕，世受國恩，斷無不敵愾同仇，共伸積忿！

朕今親統六師，直抵[[通州|通州]]，以伸滅討而張撻伐。著內廷王、御前大臣、軍機大臣、內務府大臣迅速定議。並有[[僧格林沁|僧格林沁]]密折一封，一併閱看。本日奏事之外廷大臣，並著與議。時諭。<ref>《籌辦夷務始末》，咸豐朝，第6冊，第2254頁</ref>}}

以清朝官場文化，咸豐帝並非真的想御駕親征。而是當作名義，實際要逃難，但不便明說。希望由眾大臣簇擁下勉為其難接受<ref>《苦命天子》，pp. 214–216.</ref>。

大學士[[賈楨|賈楨]]上奏：「時無[[寇準|寇準]]，[[澶淵之盟|澶淵]]之功難恃；木蘭無險，‘[[土木之變|土木之變]]堪慮’。<ref>《籌辦夷務始末》，咸豐朝，第6冊，第2255頁</ref>」

9月10日，京城頒布要調集大車500輛的命令<ref>[[翁同龢|翁同龢]]，《翁同龢日記》第6冊，中華書局，p. 66.</ref>。而整座北京城開始流傳著將要攻城的謠言：「夷人已到通州，定于二十七日（9月12日）攻城」<ref>[[吳可讀|吳可讀]]《罔極編》，《第二次鴉片戰爭》，第2冊，p. 66，{{cite web |title = 線上摘錄版本 |url = http://agzy.youth.cn/xzzh/his_data/201003/t20100326_1179939.htm |date=2010-03-26 |accessdate=2011-10-09 }}</ref>。咸豐帝一直想北逃，但各大臣紛紛上奏勸說反對巡幸木蘭（也就是逃亡），9月13日咸豐帝再下令把調集的車馬「分別發還」。9月21日，八里橋慘敗的消息傳到[[紫禁城|紫禁城]]，當晚咸豐帝同重臣緊急召開御前會議。達成了兩項決定：一、清帝到熱河暫避，官方文書稱北狩；二、恭親王[[奕訢|奕訢]]留守北京<ref name="苦命天子,p.217">《苦命天子》，pp. 217–220.</ref>。

9月22日，剛到卯時〈大概清晨5點〉，咸豐帝召見[[惠親王|惠親王]][[綿愉|綿愉]]、[[恭親王|恭親王]][[奕訢|奕訢]]、[[惇親王|惇親王]][[奕誴|奕誴]]、[[怡親王|怡親王]][[載垣|載垣]]、[[鄭親王|鄭親王]][[瑞華|瑞華]]和[[軍機大臣|軍機大臣]]等人做足安排。己正（約上午10點），咸豐帝一行人從[[圓明園|圓明園]]後門出逃，由於太過匆忙連[[御膳|御膳]][[帳篷|帳篷]]都不及帶走<ref>[[殷兆鏞|殷兆鏞]]，殷譜經侍郎自訂年譜，《第二次鴉片戰爭》第2冊，pp. 328–329.</ref>
{{noteTag|name=皇帝出巡|1=一般清朝皇帝出巡時，會在前一個月開始準備，沿路上備妥[[行宮|行宮]]、膳食。但由於這次是逃亡，並未有以上情況，甚至已有地方官先行逃跑<ref>《苦命天子》，p. 221.</ref>}}。
當時清人記載咸豐帝倉皇出逃，路途中只吃兩個雞蛋和數碗小米粥，而沒有隊伍盛大歡送<ref>{{zh-tw}}《最恨生在帝王家》，帝王篇·奕詝，[http://big5.dushu.com/showbook/101653/1053887.html 危局中的蹩腳皇帝]{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>
{{noteTag|name=清人筆記|1=《申庚英夷人寇大變記略》：「聖駕遂於初八日（9月22日）巳刻偷走……<br />
鑾輿不備，扈從無多……<br />
車馬寥寥，宮眷後至，洵迫不及待也。是日，上僅咽雞子兩枚。次日上與諸宮眷食小米粥數碗，泣行數下。」<ref>第二次鴉片戰爭，第2冊，p. 49.</ref>}}，9月30日逃到[[熱河|熱河]][[避暑山莊|避暑山莊]]<ref>《苦命天子》，p. 257.</ref>。

== 投降 ==
1860年10月13日，北京政府交出[[安定門_(北京)|安定門]]向聯軍開放，英法聯軍堂而皇之列隊入城。自此，昔日清朝[[王師|王師]]每逢出征必經的安定門就這樣落入聯軍控制<ref name="苦命天子,p.221" />。

== 圓明園的搶奪和燒毀 ==
{{參見|圓明園}}

10月13日聯軍从[[安定门|安定门]]攻入北京，聯軍发现清軍將英法使節團多人虐待至死，遂决定報復。10月18日，[[詹姆斯·布魯斯|額爾金]]下令焚燒圓明園。英軍第一師數百名士兵進入園內[[縱火|縱火]]。大火持续了两天<ref name="苦命天子,p.226">《苦命天子》，p.226</ref>，300多名[[太监|太监]]和[[宫女|宫女]]葬身火海。法国作家[[维克多·雨果|维克多·雨果]]曾对此给予强烈谴责，认为此事为“两个强盗的胜利”。事后英军贴出布告{{quote||“宇宙之中，任何人物，無論其貴如帝王，既犯虛偽欺詐之罪，即不能逃脫其應有之責任與刑罰。茲為責罰清帝不守前約及違反和約起見，決於九月初五日焚燒圓明園，所有種種違約行動，人民未參與其間，決不加以傷害，惟于清室政府，不能不懲罰之也。”<ref>《第二次鴉片戰爭》（二），第400頁，上海人民出版社，1978年版</ref>}}
