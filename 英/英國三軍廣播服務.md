'''英國三軍廣播服務'''（{{lang-en|'''British Forces Broadcasting Service''' / '''BFBS'''}}）在1943年由[[英國|英國]][[英國陸軍部|舊陸軍部]]創立<ref>[http://www.ssvc.com/bfbs/radio/index.htm 歷史] {{webarchive|url=https://web.archive.org/web/20070210001147/http://www.ssvc.com/bfbs/radio/index.htm |date=2007-02-10 }} - 英國三軍電台</ref>。今天它為駐扎在全球各地的[[英國軍隊|英國軍隊]]和[[英國海外領地|英國海外領地]]公民，提供[[電台|電台]]、[[電視|電視]]和[[衛星|衛星]]至軍艦廣播服務。另外，它也透過[[互聯網|互聯網]]向全球[[廣播|廣播]]<ref>[http://www.ssvc.com/bfbs/radio/index.htm 廣播方式] {{webarchive|url=https://web.archive.org/web/20070210001147/http://www.ssvc.com/bfbs/radio/index.htm |date=2007-02-10 }} - 英國三軍廣播服務</ref><ref>[http://www.ssvc.com/ssvc/technology/index.htm 廣播技術] {{webarchive|url=https://web.archive.org/web/20070715031728/http://www.ssvc.com/ssvc/technology/index.htm |date=2007-07-15 }} - 英國三軍廣播服務</ref>。現時廣播的地區包括[[阿富汗|阿富汗]]、[[伯利兹|伯利兹]]、[[波斯尼亞|波斯尼亞]]、[[汶萊|汶萊]]、[[加拿大|加拿大]]、[[塞浦路斯|塞浦路斯]]、[[福克蘭群島|福克蘭群島]]、[[德國|德國]]、[[直布羅陀|直布羅陀]]、[[科索沃|科索沃]]、[[中東|中東]]、[[北愛爾蘭|北愛爾蘭]]。<ref>[http://www.ssvc.com/bfbs/map_global.htm 全球廣播範圍] {{webarchive|url=https://web.archive.org/web/20070504041216/http://www.ssvc.com/bfbs/map_global.htm |date=2007-05-04 }} - 英國三軍廣播服務</ref>

它的母公司是[[軍部影音機構|軍部影音機構]]（Services Sound and Vision Corporation），是一間註冊的[[慈善|慈善]]機構<ref>[http://www.ssvc.com/ssvc/aboutus/index.htm 關於我們] {{webarchive|url=https://web.archive.org/web/20071011174527/http://ssvc.com/ssvc/aboutus/index.htm |date=2007-10-11 }} - 英國三軍廣播服務</ref>。

== 英國三軍電台 ==
現時英國三軍電台有以下3個全球廣播的電台<ref name="radio">[http://www.ssvc.com/bfbs/radio/index.htm 英國三軍電台] {{webarchive|url=https://web.archive.org/web/20070210001147/http://www.ssvc.com/bfbs/radio/index.htm |date=2007-02-10 }} - 英國三軍廣播服務</ref>：
* 英國三軍電台第1台 - 本地社區電台、播放現代音樂
* 英國三軍電台第2台 - 播放流行音樂、新聞、時事、體育
* 英國三軍電台啹喀台 - 啹喀節目

另外，英國三軍電台亦向駐守全球不同地區的英國軍人及其家屬作當地廣播。英國三軍電台,在[[比利時|比利時]]、[[伯利兹|伯利兹]]、[[波斯尼亞|波斯尼亞]]、[[汶萊|汶萊]]、[[加拿大|加拿大]]、[[塞浦路斯|塞浦路斯]]、[[福克蘭群島|福克蘭群島]]、[[德國|德國]]、[[直布羅陀|直布羅陀]]、[[科索沃|科索沃]]、[[伊拉克|伊拉克]]、[[荷蘭|荷蘭]]、[[北愛爾蘭|北愛爾蘭]]設有直播室<ref name="radio" />。在阿富汗、[[阿曼|阿曼]]、[[阿森松島|阿森松島]]的英軍可收聽到這個電台。英國皇家軍艦上的海軍可透過衛星轉播收聽到這個電台。其他接收的方法包括互聯網和英國[[天空數碼|天空數碼]]廣播電台第0184台。

英國三軍電台第2台有很多的節目都是從[[英國廣播公司|英國廣播公司]]電台第4台和第5台（體育直播台）而來、包括肥皂劇The Archers。在1997年[[香港回归|主權移交中國]]之前，'''英國三軍電台'''（BFBS Radio）在[[香港|香港]]非常受聽眾歡迎，受歡迎節目當中便包括了The Archers。英國三軍電台啹喀台為在英軍中服役的[[啹喀兵|啹喀兵]]提供啹喀語廣播。

== 英國三軍電視台 ==
英國三軍電視台在1975年開始廣播，用的都是英國廣播公司（BBC）和[[英國獨立電視台|英國獨立電視台]]（ITV）的錄影節目。現時廣播的方法則是透過衛星。另外，也有一個「英國海軍電視台」，利用[[軍事|軍事]][[衛星|衛星]]將不同時段節目，向[[英國海軍|英國海軍]][[軍艦|軍艦]][[轉播|轉播]]。

2001年後, 英國三軍電視台一分為二：
* 英國三軍電視第1台（BFBS 1）
* 英國三軍電視第2台（BFBS 2）
2009年5月，新开播2条频道：
* 英國三軍電視第3台儿童台（BFBS 3 Kids）
* 英國三軍電視第4台（BFBS 4）

由於服務對象是英國軍人及其家屬，所以除了在小部份地區外，英國三軍電視台的訊號是[[加密|加密]]了的。在[[福克蘭群島|福克蘭群島]]上，這電視台由於是島上唯一的電視台，所以訊號是在沒有[[加密|加密]]情況下向普通[[市民|市民]]廣播的。
