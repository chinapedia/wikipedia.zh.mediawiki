== 剧情 ==
1911年，[[同盟会|同盟会]]领导“[[黄花岗起义|黄花岗起义]]”，[[岭南画派|岭南画派]]画家“高剑父”以及“潘达微”二人参加起义，但是没有壮烈牺牲，他们在起义失败之后收集了烈士遗骸进行埋葬，两人也因此彪炳史册。<ref>{{cite web |url=http://news.hexun.com/2012-03-31/139944601.html |title=辛亥壮歌《英雄喋血》 |accessdate=2012 |publisher= 和讯网|work= |language=zh-hans}}</ref><ref>{{cite web |url=http://news.hexun.com/2011-09-29/133862429.html |title=电影《英雄喋血》剧情简介 |accessdate=2011 |publisher=和讯网 |work= |language=zh-hans}}</ref>
