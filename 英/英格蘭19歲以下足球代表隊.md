'''英格蘭19歲以下足球代表隊'''（'''England national under-19 football team'''，簡稱'''英格蘭U19'''）代表[[英格蘭|英格蘭]]參加這個年齡組別的國際[[英式足球|足球]]比賽，由[[英格蘭足球總會|英格蘭足球總會]]監控。現時的教練是[[诺埃尔·布莱克|诺埃尔·布莱克]]（Noel Blake）。

==競賽==
===2009年===
英格蘭U19已獲得2009年[[歐洲U-19足球錦標賽|歐洲U-19足球錦標賽]]外圍初賽出線，於5月27日至6月1日主辦第二輪外圍賽（Elite Round），與波斯尼亞、蘇格蘭及斯洛伐克爭奪夏季在[[烏克蘭|烏克蘭]]舉行的決賽週席位<small><ref>{{cite web |language=en |url=http://www.thefa.com/England/MensUnder19s/NewsAndFeatures/2009/EliteRounddatesset.aspx |title=Elite Round dates set |publisher=[[英格蘭足球總會|英格蘭足球總會]] |date=2009-04-20 |accessdate=2009-04-21}}</ref></small>。5月27日首仗2-0擊敗波斯尼亞<small><ref>{{cite web |language=en |url=http://www.thefa.com/England/MensUnder19s/NewsAndFeatures/2009/england_bosnia_report_270509.aspx |title=Winning start for Young Lions |publisher=[[英格蘭足球總會|英格蘭足球總會]] |date=2009-05-27 |accessdate=2009-05-30}}</ref></small>；5月29日次仗亦以4-1輕取斯洛伐克<small><ref>{{cite web |language=en |url=http://www.thefa.com/England/MensUnder19s/NewsAndFeatures/2009/england_slovakia_report_290509.aspx |title=Strike duo see off Slovakia |publisher=[[英格蘭足球總會|英格蘭足球總會]] |date=2009-05-29 |accessdate=2009-05-30}}</ref></small>；6月1日最後一場2-1擊敗蘇格蘭，三戰全勝取得決賽週入場劵<small><ref>{{cite web |language=en |url=http://www.thefa.com/England/MensUnder19s/NewsAndFeatures/2009/scotland_report_010609.aspx |title=England edge battle of Britain |publisher=[[英格蘭足球總會|英格蘭足球總會]] |date=2009-06-01 |accessdate=2009-06-06}}</ref></small>。

決賽週分組賽英格蘭U19以3戰1勝2和憑[[得失球差|得失球差]]壓倒主辦國烏克蘭以首名出線，2009年7月30日在準決賽遭遇法國U19，法定時間1-1賽和，[[加時賽|加時賽]]憑[[阿士東維拉足球會|-{zh-hans:阿斯顿维拉;zh-hk:阿士東維拉;}-]]小將[[纳森·德尔福内索|-{zh-hans:德尔福内索;zh-hk:迪爾科尼素;}-]]（Nathan Delfouneso）連入兩球以3-1獲勝晉身決賽。8月2日決賽重遇烏克蘭U19，雙方在分組賽打成2-2平手，但決賽在25,000名主場球迷前以0-2不敵主隊屈居亞軍。

===2010年===
2010年[[歐洲U-19足球錦標賽|歐洲U-19足球錦標賽]]外圍初賽於2009年10月在[[斯洛文尼亞|斯洛文尼亞]]舉行，英格蘭U19取得3戰3勝出線第二輪外圍賽。英格蘭U19在[[烏克蘭|烏克蘭]]舉行的第二輪外圍賽以3戰2勝1和，兩輪外圍賽不敗的成績晉級在[[法國|法國]]舉行的決賽週。

決賽週分組賽被編與主辦國法國、荷蘭及奧地利同列A組，以3戰1勝1和1負的成績，獲得小組次名出線四強淘汰賽，同時亦取得[[2011年世界青年足球錦標賽|2011年世青盃]]參賽資格。準決賽碰上西班牙，以1-3落敗出局。
