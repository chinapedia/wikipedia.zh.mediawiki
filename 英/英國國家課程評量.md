'''英國國家課程評量'''（National Curriculum assessments）為一系列的教育評估測驗，又名為 '''Sats'''<ref>[http://www.telegraph.co.uk/news/newstopics/politics/education/primaryeducation/3199156/Sats-for-14-year-olds-abolished-Teachers-and-parents-praise-decision.html Sats for 14-year-olds abolished: Teachers and parents praise decision]</ref> 或是 '''SATs'''（詳見名稱部分）<ref name="banburyguardian">[http://www.banburyguardian.co.uk/news/Headteacher-welcomes-end-of-SATs.4595469.jp Headteacher welcomes end of SATs]</ref> ，為英國教育單位評估學童在學校的表現評量。測驗方式包含教師評估以及學生的測驗表現。本評量與同名的美國大學入學申請測驗「[[SAT|SAT]]」（學術評估測試）並無關連。

== 歷史 ==
本測驗約從1991~1998年開始施測<ref name="Sentinel">{{Cite web|title = The Sentinel|url = http://www.stokesentinel.co.uk/National-SATs-proved-controversial-introduced-19-years-ago/story-12485991-detail/story.html|website = {{url|stokesentinel.co.uk/|accessdate=9}} May 2015|deadurl = yes|archiveurl = https://web.archive.org/web/20150924125401/http://www.stokesentinel.co.uk/National-SATs-proved-controversial-introduced-19-years-ago/story-12485991-detail/story.html|archivedate = 2015-09-24}}</ref> ，用於7歲，11歲以及14歲的學童，了解學生的學習程度。

之後取消14歲的測驗，而7歲的測驗改由教師評估。<ref name="Sentinel"/> 在2002年到2005年間，這些測驗逐漸在威爾斯地區淘汰。<ref name="Sentinel"/>

英國國家課程評量由國家標準測驗局(Standards and Testing Agency) (STA)定，此為英國教育部門的一間機構。該機構改制被英國政府改制為英國資格與課程開發署(Qualifications and Curriculum Development Agency) (QCDA) 以及資格及測驗規章辦公室(The Office of Qualifications and Examinations Regulation)(Ofqual)。目前由英國政府以及 Ofqual 固定辦理測驗。

== 名稱 ==
過去本測驗的稱呼包含多重用意，由於測驗由學校老師評估，因此也被稱作教師評估測驗(Teacher Assessments)。 測驗都在學習的關鍵階段的結束後進行。<ref name="directgov">{{Cite web|url = http://www.direct.gov.uk/en/Parents/Schoolslearninganddevelopment/ExamsTestsAndTheCurriculum/DG_10013041|title = National Curriculum teacher assessments and key stage tests|accessdate = 18 February 2008|publisher = H M Government|year = 2008|work = DirectGov website}}</ref>

還要經過一項考試測驗，而這個考試則被稱為國家課程測驗(National Curriculum Tests)<ref name="qca">{{Cite web|url = http://www.qca.org.uk/qca_8584.aspx|title = Tests and tasks: National Curriculum Tests|accessdate = 18 February 2008|publisher = Qualifications and Curriculum Authority|year = 2008|work = QCA website|deadurl = yes|archiveurl = https://web.archive.org/web/20080212053031/http://www.qca.org.uk/qca_8584.aspx|archivedate = 2008年2月12日}}</ref>

一般的稱呼，特別是考試，都被稱作 SATs。這是因為最初發展時的名稱為「標準評估測試」(Standard Assessment Tasks)。<ref name="RB">{{Cite book|title = Schools' Use of Data in Teaching and Learning|url = http://www.dfes.gov.uk/research/data/uploadfiles/RR671.pdf|last = Kirkup|first = C|author2 = Sizmur, J.|author3 = Sturman, L.|author4 = Lewis, K.|year = 2005|publisher = Department for Education and Skills / NFER|location = London|format = PDF|page = 27}}</ref> 這個簡稱也被當作為 Statutory Assessment Tests,<ref name="wscc">{{Cite web|url = http://www.westsussex.gov.uk/ccm/content/education/parents/exams/sats-exams.en|title = SAT Exams|accessdate = 18 February 2008|publisher = West Sussex County Council|year = 2008|work = WSCC website|archiveurl = https://web.archive.org/web/20080414184059/http://www.westsussex.gov.uk/ccm/content/education/parents/exams/sats-exams.en|archivedate = 14 April 2008|deadurl = yes}}</ref> Standard Attainment Tests,<ref name="glossary">{{Cite web|url = http://www.standards.dfes.gov.uk/onlinehelp/?view=A-Z&version=1&ltr=s|title = The Standards Site: Online Help|accessdate = 18 February 2008|publisher = Department for Children, Schools and Families|year = 2008|work = DCSF website|deadurl = yes|archiveurl = https://web.archive.org/web/20080317122534/http://www.standards.dfes.gov.uk/onlinehelp/?view=A-Z&version=1&ltr=s|archivedate = 2008年3月17日}}</ref> Standardised Achievement Tests 以及 Standard Assessment Tests<ref name="banburyguardian"/> 等類似的名稱。"SATs" 在英國是被當作一個名詞直接念，而美國的 SATs 則是分開字母，「S-A-T」的單獨稱呼。

== 批評 ==
如同其他的測驗，本測驗也招致許多批評。有兩個主要的批評為這個測驗會使學生增加壓力，並且會使學校課程以測驗為目而安排。<ref>{{cite web |url=http://www.eirenikonpublishing.co.uk/Expectations-Encouragement-Empowerment-Education-Alternative-to-Targets-Testing-and-Tables-a-Tragedy-by-Cynthia-Bartlett/ |title=存档副本 |accessdate=2011-06-25 |deadurl=yes |archiveurl=https://web.archive.org/web/20120326025034/http://www.eirenikonpublishing.co.uk/Expectations-Encouragement-Empowerment-Education-Alternative-to-Targets-Testing-and-Tables-a-Tragedy-by-Cynthia-Bartlett/ |archivedate=2012-03-26 }}</ref><ref name="sfsgroup">{{Cite web|url = http://www.sfs-group.co.uk/news-industry-18591974-fresh_criticism_for_sats.htm|title = Fresh criticism for Sats|accessdate = 31 May 2008|publisher = SFS Group|year = 2008|work = SFS Group website|deadurl = yes|archiveurl = https://web.archive.org/web/20080723223609/http://www.sfs-group.co.uk/news-industry-18591974-Fresh_criticism_for_Sats.htm|archivedate = 23 July 2008}}</ref>

1993年時，有兩個主要的教師工會公開抵制本項測驗。<ref name="Sentinel"/>

2008年時的報告指出，根據國家考試分析，[[英國下議院|英國下議院]]的教育委員會，以及學校和家長代表提出了對於本項測驗的疑慮。他們擔心教師的專業能力不足，以及「考試領導教學」(teaching to the test)的現象產生，造成課程縮限以及學生資源分配不均的現象。他們進一步建議應將英國國家課程評量分為 地方課責(local accountability)，全國施測(national monitoring)，以及個人進步量表(individual progress measurement)等分散評量的評分方式。<ref name="select">{{Cite web|url = http://www.publications.parliament.uk/pa/cm200708/cmselect/cmchilsch/169/16912.htm|title = Conclusions and Recommendations|accessdate = 31 May 2008|publisher = UK Parliament|year = 2008|work = Select Committee on Children, Schools and Families Third Report}}</ref>

兩個主要的英國教師工會 NUT 以及 NAHT (雖然沒有 NASUWT)，在2010年發起抵制本項測驗，並且造成四分之一的學校不舉辦測驗。<ref name="BBC 2010 SATS boycott">{{Cite web|title = 'A quarter of schools' boycotted Sats tests|url = http://www.bbc.co.uk/news/10521289|website = BBC News|accessdate = 17 May 2015}}</ref> 這些教師工會的訴求是希望改由教師評估取代測驗。<ref name="Guardian">{{Cite news|last1 = Shepherd|first1 = Jessica|title = Headteachers vote to boycott Sats tests|url = http://www.theguardian.com/education/2010/apr/16/headteachers-boycott-national-tests-sats|accessdate = 17 May 2015|date = 16 April 2010}}</ref>
