'''英格拉姆M6'''（{{Lang-en|Ingram Model 6}}）是一款由[[美国|美国]]槍械設計師{{Link-en|戈登·B·英格拉姆|Gordon Ingram}}所設計、1949年至1952年之間由[[加利福尼亚州|加利福尼亚州]][[洛杉矶|洛杉矶]]警用軍械公司所生產的[[冲锋枪|冲锋枪]]，發射[[.45_ACP|.45 ACP]]、[[9×19mm魯格彈|9×19毫米魯格]]和{{Link-en|.38超級彈|.38 Super}}三種[[口徑|口徑]]的[[手枪|手枪]][[子彈|子彈]]。

== 歷史 ==
英格拉姆M6的原型是在1940年，槍械設計師戈登·B·英格拉姆研製的一枝原型衝鋒槍。<ref name=wg>[http://world.guns.ru/smg/smg86-e.htm Modern Firearms]</ref>1949年至1952年之間，警用軍械公司生產了15,000至20,000枝，主要是卖给了採用它的[[古巴海军|古巴海军]]、[[秘鲁陆军|秘鲁陆军]]以及美國國內少數警察局。在秘魯，還取得了特許生產。

== 設計 ==
雖然英格拉姆M6具有類似於[[汤普森冲锋枪|汤普森冲锋枪]]的外觀，但其實它原來的研製目的是作為美國國內的執法機構市場上銷售的一款廉價替代型衝鋒槍。除了與湯普森一樣有著發射[[.45_ACP|.45 ACP]][[手枪|手枪]][[彈藥|彈藥]]的規格，還有發射[[9×19mm魯格彈|9×19毫米魯格彈]]和{{Link-en|.38超級彈|.38 Super}}的型號。總長度約為762毫米（30英吋），槍管長度則為228毫米（8.98英吋）。其[[彈匣|彈匣]]為方型設計，具有30發子彈容量。

英格拉姆M6具有兩種基本型號，一種是“軍用型”，另一種稱為“警用型”。軍用型M6類似湯普森M1和M1A1，使用木質前托型[[護木|護木]]和具有護翼的[[照準器|準星]]，還可裝上一把錐形[[剌刀|剌刀]]。<ref name=wg/>警用型M6類似湯普森M1921至M1928，槍管下方裝有與後者非常相似的[[輔助握把|前握把]]，槍管表面亦具有散熱片。兩種型號都使用相同的整體式木製槍托和手槍握把部件，以及單連發扳機（輕扣一段時為半自動，扣到底則是全自動）。

== 衍生型 ==
英格拉姆M7版本在1952年生產，不同之處在於它使用{{Link-en|閉鎖式槍機|Closed bolt}}和在扳機上方增加了一個明顯的快慢機旋柄。1950年代時少量生產。
