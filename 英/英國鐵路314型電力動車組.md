'''英鐵314型電力動車組'''（{{lang-en|British Rail Class 314}}）是一款在1979年由{{link-en|英國鐵路工程公司|British Rail Engineering Limited}}（BREL）在[[約克|約克]]製造的列車。它是由[[英國國鐵|英國國鐵]]自1972年起購置的第三批同款式設計列車<ref group="註">其他同款列車包括{{link-en|英鐵313型電力動車組|British Rail Class 313|313}}、[[英鐵315型電力動車組|315]]、{{link-en|英鐵507型電力動車組|British Rail Class 507|507}}及{{link-en|英鐵508型電力動車組|British Rail Class 508|508}}型電力動車組。</ref>。

這批列車全數配屬[[蘇格蘭鐵路|蘇格蘭鐵路]]，主要行走[[大格拉斯哥|大格拉斯哥]]地區的通勤鐵路。它們現時是蘇格蘭鐵路旗下車齡最高的列車，蘇格蘭的交通部門曾於2005年考慮以新列車取代它們<ref group="參">{{cite web|url=http://www.transportscotland.gov.uk/system/files/documents/tsc-basic-pages/replacement_of_314_units.pdf|title=Modified Priced Option Replacement of Class 314 Electric Multiple Units|date=2005-05-10|accessdate=2015-10-04|language=en}}</ref><ref group="註">有關檔案現由{{link-en|蘇格蘭交通部|Transport Scotland}}保管，但該機構於2006年方才成立。</ref>。
