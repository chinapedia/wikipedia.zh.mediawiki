'''迪欧斯捷尔VTK'''（'''Diósgyőri VTK'''）是[[匈牙利|匈牙利]]足球俱乐部，位于[[匈牙利|匈牙利]][[米什科尔茨|米什科尔茨]]。俱乐部现属[[匈牙利足球甲级联赛|匈牙利足球甲级联赛]]球队，球队的主体育场为DVTK体育场。

== 球队历史 ==
* 1910年 : '''迪欧斯捷尔VTK'''足球俱乐部建立 
* 1938年 : 兼并了'''迪欧斯捷尔AC'''重组成'''Dimávag SC 迪欧斯捷尔'''
* 1945年 : 俱乐部更名回 迪欧斯捷尔VTK
* 1951年 : 俱乐部更名为 '''迪欧斯捷尔Vasas'''
* 1956年 : 俱乐部更名为 '''米什科尔茨迪欧斯捷尔VTK'''
* 1977年 : 球队参加[[欧洲冠军联赛|欧洲杯]]赛事(1977/78年)
* 1992年 : 俱乐部更名为 '''迪欧斯捷尔足球俱乐部'''
* 2001年 : 俱乐部更名回 迪欧斯捷尔VTK
* 2004年 : 俱乐部更名为 巴拉顿足球俱乐部 重组成迪'''欧斯捷尔-巴拉顿足球俱乐部'''
* 2006年 : 俱乐部更名回 迪欧斯捷尔VTK

== 球队荣誉 ==

** 冠军: 1977年, 1980年
** 亚军 : 1942年, 1965年, 1981年
