《'''歡樂滿人間'''》（{{lang-en|'''''Mary Poppins'''''}}）是一部1964年的美國[[歌舞片|歌舞]]奇幻電影，由{{link-en|羅伯·史蒂文遜|Robert Stevenson (director)}}執導，[[華特·迪士尼|華特·迪士尼]]監製。{{link-en|比爾·沃爾什|Bill Walsh (producer)}}和{{link-en|Don DaGradi|Don DaGradi}}寫劇本，改編自作家[[P·L·卓華斯|P·L·卓華斯]]的小說系列《[[瑪麗·包萍|瑪麗·包萍]]》，是迪士尼至今[[奧斯卡獎|奧斯卡獎]]提名最多、得獎最多的電影（13項提名、5項得獎）。主演為[[茱莉·安德絲|茱莉·安德絲]]，她飾演{{link-en|Mary Poppins (character)|Mary Poppins (character)|Mary Poppins}}，而{{link-en|迪克·凡·戴克|Dick Van Dyke}}、{{link-en|大衛·湯姆林森|David Tomlinson}}和{{link-en|葛萊妮絲·強斯|Glynis Johns}}飾演配角。此片如今已被改編成為多齣電影及舞台[[音樂劇|音樂劇]]，另外2013年的電影《[[大夢想家|大夢想家]]》亦有部分篇幅描繪籌製本片的過程。本片被視為女主角茱莉的成名作之一，也常於女權運動、家庭教育、情緒調適控管等議題中被提出討論。

== 劇情概要 ==
在[[倫敦|倫敦]]的一個家庭，有一個保姆被兩孩子Jane珍及Michael麥克氣走。Mr. Banks班克斯先生是一名強調準時與標準的銀行職員，而太太專注於女權運動。專注於事業的兩個人顧不了這兩個孩子，卻完全不知道這對姐弟並不是普通的保母能夠招架的了。

主角Mary Poppins是一位仙女褓母，她來到了人間幫助Banks家的兩位孩子重拾歡樂，教導他們如何克服生活的困難，透過對不同解讀的角度，最終改變Mr. Banks對事物的看法，並且讓終日汲汲於銀行業務與[[女權運動|女權運動]]的Banks夫婦體認到親子溫情的可貴，兩個孩子也重新感受到了親情和友情。

== 評價 ==
[[爛番茄|爛番茄]]根據48條評論，持有100%的新鮮度，平均得分8.4／10<ref>{{cite web|title=Mary Poppins|url=https://www.rottentomatoes.com/m/mary_poppins/|website=[[Rotten_Tomatoes|Rotten Tomatoes]]|publisher=[[Fandango_(公司)|Fandango]]|accessdate=2019-02-08}}</ref>。在[[Metacritic|Metacritic]]上，電影獲得了88分<ref>{{cite web|title=Mary Poppins|url=https://www.metacritic.com/movie/mary-poppins|website=[[Metacritic|Metacritic]]|accessdate=2019-02-08}}</ref>，「普遍正面好評」的評價。
== 軼事 ==
在這部電影中出現了曾一度被認為是[[最長的英文單字|最長的英文單字]]：supercalifragilisticexpialidocious（在2013年電影《[[大夢想家|大夢想家]]》譯為「超級卡利法力斯帕里多斯」），意涵簡單的說就是「非常細膩的美好就像獲得[[救贖|救贖]]一樣」之意並以此為歌；常用作並傳唱於[[迪士尼樂園|迪士尼樂園]][[巡遊|巡遊]]和舞台音樂劇。

== 影音 ==
[[華特迪士尼影業|華特迪士尼影業]]在2013年推行電影《[[大夢想家|大夢想家]]》時，順勢宣布推出歡樂滿人間的[[藍光|藍光]]影音版本，以紀念這部電影上映第50周年。

== 續集 ==
