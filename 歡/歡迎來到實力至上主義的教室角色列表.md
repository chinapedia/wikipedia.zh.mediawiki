== 一年級學生 ==
=== D班 ===
* 在故事開始時為D班，在第3學期（小說第8卷起）時升為C班。
* D班的學生都是因爲能力低或擁有不良紀錄的人才會被分配到此班，所以D班裡必然聚集了一堆問題人物（不良品）。
* 原作小說中每班班級人數約為40人，動畫為25人，故部分配角在動畫中沒有出現。

; {{nihongo|[[綾小路清隆|綾小路清隆]]|綾小路 清隆／あやのこうじ きよたか|Ayanokouji Kiyotaka}}
: 本作男主角
:D班的幕後領導人物。開始時凡事欠缺積極性，並對自己的未來毫無展望，自稱是避事主義者。入學後對於結交朋友有著一定渴求，但一直沒和眾人深交。
: 在原作小說裡面內心活動豐富，但從櫛田的話語<ref name="auto9">第三卷 虛假的團隊合作</ref>和堀北與一之瀨的對話中<ref name="auto10">第四·五卷 與別班的交流會</ref>可知，實際上綾小路清隆從故事一開始就幾乎沒有任何的表情，也幾乎沒有任何的情緒波動，甚至被評為宛如人偶一般。<br/>動畫則是採用第三身視角，直接省略了綾小路清隆絕大部分的內心活動。<br/>漫畫綾小路清隆的表情和情緒波動則是跟正常人相差無幾。
: 雖然帥氣程度在全年級男生中排第五<ref>第一卷。第一名是A班的里中，第二名是平田，第三、第四名都是A班的男生。</ref>，但不善與人交流，在陰沈排行榜也是名列前茅。一直試圖交更多的朋友，D班真正的王牌與幕后推手，以堀北鈴音作为掩護隐藏了自己的實力，雖然經常暗中帮助堀北，但也會為了自己的目的讓她吃苦頭。
: 在校內時刻意隱藏各方面的實力，故意將自己的分數控制在中間偏低的程度來減少引人注目。真實實力深不可測，且擅於智謀。另外擁有深藏不露的功夫，在面對堀北鈴音將被其兄也就是學生會長堀北學施暴時出手救了堀北鈴音，被堀北會長出手卻被他閃過。入學考試時將自己的分數都以50分通過考試，因此堀北學發覺綾小路清隆是有意隱藏自己，卻不知背後原因。
: 討厭別人窺探自己的事情，就算在其宿舍房間也沒有必需物品和學校用品以外的東西（因此被經常跑去找綾小路的櫛田等人震驚），連幾本課外讀物也沒有。所以整體來說綾小路不論是在對人還是對事，完全將自己防備到滴水不漏，不讓他人輕易發現真實的想法。
: 武力數值仍是未知數(可以同時與山田阿爾貝、石崎大地、伊吹澪、龍園翔打架並輕鬆取得勝利)。智力數值極其高超，能夠隨意控分達到自己想要的分數，由於性格原因，只在暗中幕後活躍。<br/>談判能力相當高，甚至可媲美[[談判專家|談判專家]]。
: 被班導茶柱認為是D班最大的缺陷品，而自己也有自知之明，在內心表明過自己是個充滿缺陷、愚蠢、恐怖的人，所以嚮往著平凡、平靜的日常生活。茶柱佐枝藉由[[伊卡洛斯|伊卡洛斯]]之翼的故事，來對照綾小路清隆。<ref name="auto11">第三卷　茶柱佐枝的獨白。</ref>
: 因為受其父親的影響，還有'''White Room'''的教育，綾小路清隆也是一個現實主義者，認為勝利就是一切，犧牲多寡都無所謂，有著將其他人當成棋子、道具利用的另一面。
: 因為受到父親外在的威脅，以及班導茶柱的半強迫要求，不是很情願地開始以晉升A班為目標。在接下來多次活動中的行動都退居幕後，不輕易嶄露頭角，名義上帶領D班的角色轉由堀北擔任。
: 自小時候開始就在'''White Room'''中度過，並接受教育，在這不到15年的時間內，綾小路清隆已經累積了遠超越一般人一生所能學習到的知識。<ref name="auto6">第7卷　再會與別離的通知</ref>
: 在'''White Room'''學習到的除了綾小路清隆自稱學習過的[[鋼琴|鋼琴]]、[[書道|書道]]、[[茶道|茶道]]外，還有綾小路清隆的父親認為必須學習的各種各樣的東西，[[學問|學問]]、[[學術|學術]]、[[武術|武術]]、[[防身術|防身術]]、[[處世術|處世術]]……等等。<ref name="auto6"/>'''White Room'''將綾小路清隆培養成一個全才，並被稱作最高傑作品。<ref name="auto6"/>
: 故事開始的一年前，'''White Room'''因為某些原因暫停一年，在這一年間由管家松雄負責照顧綾小路清隆。<ref name="auto6"/>松雄除了照顧綾小路清隆，還告訴綾小路清隆在這日本只有一所學校，不在綾小路清隆的父親的影響範圍內，即'''高度育成高等學校'''。<ref name="auto6"/>
: 受松雄照顧一年後，松雄幫綾小路清隆辦理入學手續，並由坂柳有-{栖}-的父親獨斷讓原本無法入學的綾小路清隆入學，綾小路清隆入學就讀，成功從父親的手中逃離。<ref name="auto6"/>
: 因理事長基於某種理由，特別讓原本無法進入這所學校的綾小路入學，還為了隱藏綾小路清隆的存在而將他被分配到揶揄為次品聚集地的最底層D班。

; {{nihongo|坂上數馬|坂上 数馬（さかがみ かずま）|Sakagami Kazuma|聲：{{link-ja|山本兼平}}（日本）；[[何志威|何志威]]（台灣）}}
: 一年C班的班導師。對於學生有過分的關心和偏袒，對龍園的暴力統治放任自流，但也對龍園有欣賞的態度。

; {{nihongo|真嶋智也|真嶋 智也（まじま ともや）|Mashima Tomoya|聲：{{link-ja|杉崎亮}}（日本）；[[何志威|何志威]]（台灣）}}
: 一年A班的班導師，教授英文，過去似乎也教過其他科目。以嚴謹出名，有著如摔角選手般的體格，同時腦筋非常好。曾負責船上無人島求生特別考試的說明。在動畫與漫畫的形象略有不同。

; 校長
: 高度育成高等學校的校長，年紀大概60歲前後。面對綾小路清隆的父親，無法展現從容的樣子。

; {{nihongo|坂柳理事長／有-{栖}-的父親|坂柳理事長（さかやなぎりじちょう）/有栖の父親|Sakayanagirijichiyou}}
: 年紀大約40歲左右，高度育成高等學校的理事長，自父親手中繼承學校理事長的位置。
: 與[[綾小路清隆|綾小路清隆]]的父親互相認識<ref name="auto6"/>。稱綾小路清隆的父親為『綾小路老師（{{jpn|j=綾小路先生|hg=あやのこうじせんせい|rm=Ayanokoujisensei}}）』，並且相當尊敬綾小路清隆的父親，面對綾小路清隆的父親也能毫不膽怯<ref name="auto6"/>。
: 本來綾小路清隆無法入學，是坂柳有-{栖}-的父親自己的獨斷，讓綾小路清隆入學<ref name="auto6"/>。拒絕綾小路清隆的父親想讓綾小路清隆退學的請求，並表示只要綾小路清隆仍然是這所學校的學生，就會受到學校規則的保護<ref name="auto6"/>。
: 曾經透過玻璃觀察在'''White Room'''中的綾小路清隆<ref name="auto6"/>。向綾小路清隆透漏一些有關高度育成高等學校的內幕<ref name="auto6"/>。向綾小路清隆表示會在規則之中保護他，但是如果規則無法繼續保護，那就無能為力了，最後向綾小路清隆應援<ref name="auto6"/>。
: 把綾小路清隆分配至D班的考量是，減少讓綾小路清隆受到眾人矚目的機會，加上D班班導茶柱看起來是一個對班級競爭不怎麼關心的老師<ref name="auto6"/>。
  第十卷中因不明原因，理事長職位被停職<ref name="auto6"/>。

== 其他 ==
; {{nihongo|綾小路老師／清隆的父親|綾小路先生（あやのこうじせんせい）/清隆の父親|Ayanokoujisensei|聲：[[增谷康紀|增谷康紀]]（日本）；[[林谷珍|林谷珍]]（台灣）}}
: 年紀大約40歲左右，眼睛有如磨好的刀尖，其銳利的眼神不僅沒有衰退，反而隨著年齡的增長而越發銳利，被他盯著，大概不少人會有內心深處被看穿的壓迫感<ref name="auto6"/>。在資料上跟[[綾小路清隆|綾小路清隆]]是父子，有無血緣關係未知<ref name="auto6"/>。動畫僅出現在綾小路清隆的回憶中。
: 管理著名為'''White Room'''的地方，綾小路清隆從小一直在'''White Room'''中長大並接受教育，根據綾小路清隆的說法，'''White Room'''不是一個很有人道的地方，甚至見不得人，還有人權等諸多問題，但是這裡可以受到最好的教育。
: 綾小路清隆是他最高傑作，綾小路知道自身的敗北即是父親的敗北<ref name="auto7">第5卷</ref>。坂柳有-{栖}-認為，只要破壞掉他的{{ruby|最高傑作|'''[[綾小路清隆|綾小路清隆]]'''}}，便可以給他顏色瞧瞧<ref name="auto7"/>。
: 綾小路清隆對他的評價是，確信自己就是絕對正義的男人<ref name="auto6"/>。綾小路清隆沒有把他當成父親看待；而他把綾小路清隆當作自己的所有物，是死是活全由他自己決定<ref name="auto6"/>。因為'''White Room'''重啟後沒有出現有如綾小路清隆的人才，導致綾小路清隆的父親想用各種手段把綾小路清隆從學校帶回去<ref name="auto6"/>。認為綾小路清隆只要按照自己的道路走，便可以超越自己，掌握全日本<ref name="auto6"/>。
: 小說第7卷為了讓綾小路清隆退學，而親自來到學校裡面。很希望學校能讓綾小路被退學，認為綾小路清隆遲早會自己選擇退學。

; {{nihongo|松雄|まつお|}}
: '''White Room'''暫停後，照顧了綾小路清隆一年的管家<ref name="auto6"/>。告訴綾小路清隆在這日本只有一所學校，不在綾小路清隆的父親的影響範圍內，即'''高度育成高等學校'''<ref name="auto6"/>。幫綾小路清隆辦理入學手續，讓綾小路清隆成功從父親手中逃離<ref name="auto6"/>。被綾小路清隆的父親逼到[[自焚|自焚]]結束生命<ref name="auto6"/>。

; 松雄的兒子
: 松雄的兒子，成績學業十分優秀，與綾小路同歲。本考上了相當不錯的高中，因為松雄協助綾小路的原因被綾小路父親報復，被強制退學後在綾小路父親操控下任何學校不再錄用他，如做著艱難維持生活的工作。
