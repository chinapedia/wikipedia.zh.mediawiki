《'''歡樂皺紋'''》（{{lang-es|Arrugas}}）是一部由[[伊格納西奧·費雷拉斯|伊格納西奧·費雷拉斯]]執導、改編自漫畫家[[帕科·羅卡|帕科·羅卡]]的同名漫畫，於2011年9月19日在[[聖賽巴斯提安國際影展|聖賽巴斯提安國際影展]]首映的西班牙動畫電影。

作品中四分之三的內容於西班牙當地完成，其餘部分則有委託給[[菲律賓|菲律賓]]一帶的動畫工作室處理。<ref>{{Cite web |url=http://variety.com/2011/digital/news/wrinkles-uses-classic-methods-to-create-magic-1118047077/ |title=‘Wrinkles’ uses classic methods to create magic |publisher=Variety |author=Variety Staff |date=2011年12月8日 |accessdate=2014年1月28日}}</ref>

== 劇情 ==
故事主角艾米立歐（{{lang|es|Emilio}}）是一位從銀行部門退休的老人，因罹患[[阿茲海默症|阿茲海默症]]而被親生兒子送至養老院中。<ref name="Img">{{Cite web |url=http://www.imagineanimation.net/news/ignacio-ferreras-snaps-two-spanish-goyas-arrugas-animation/12860 |title=Ignacio Ferreras snaps two spanish goyas arrugas animation |publisher=Imagine Animation |author=Iclover |date=2012年2月21日 |accessdate=2014年1月28日 |deadurl=yes |archiveurl=https://web.archive.org/web/20130920023415/http://www.imagineanimation.net/news/ignacio-ferreras-snaps-two-spanish-goyas-arrugas-animation/12860 |archivedate=2013年9月20日 }}</ref>

而在養老院裡頭生活的艾米立歐對於自己處境及行為逐漸癡呆起來感到恐懼，之後則在室友米格爾（{{lang|es|Miguel}}）協助下，艾米立歐開始有了不一樣的年老生活並與米格爾發展出深厚的友情。<ref name="Img"/>

== 反應 ==
=== 評價 ===
[[好萊塢報導|好萊塢報導]]評析《歡樂皺紋》使用了平靜及細膩的手法來表達出人們年老時所面臨的問題，可謂該年度最成功的西班牙電影之一。<ref>{{Cite web |url=http://www.hollywoodreporter.com/review/wrinkles-arrugas-film-review-237302 |title=Wrinkles (Arrugas): Film Review |publisher=[[好萊塢報導|好萊塢報導]] |author=Neil Young |date=2011年9月19日 |accessdate=2014年1月28日}}</ref>網站Criterion Cast認為「人生中遭遇的喜怒、恐懼、友情、背叛完全展示在這部作品中，而隨著故事的發展越能將上述的內容表現出來。」<ref>{{Cite web |url=http://criterioncast.com/film-festivals/piff-film-festivals/piff-2013/ignacio-ferreras-wrinkles-piff-2013/ |title=James Reviews Ignacio Ferreras’ Wrinkles PIFF 2013 Review |publisher=CriterionCast |author=James McCormick |date=2013年3月1日 |accessdate=2014年1月28日}}</ref>日本[[吉卜力工作室|吉卜力工作室]]的動畫導演[[高畑勳|高畑勳]]則表示「《歡樂皺紋》讓我看見動畫電影能有更多未發覺的表現手法之可行性，對於同樣已是老年人、以及在動畫界工作的我要為這部作品獻上敬意。」<ref>{{Cite web |url=http://www.ghibli-museum.jp/shiwa/comment/ |script-title=ja:高畑勲監督コメント |publisher=[[吉卜力工作室|吉卜力工作室]] |accessdate=2014年1月28日}}</ref>
