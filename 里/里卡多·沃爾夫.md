'''里卡多·沃爾夫'''[[博士|博士]]（{{lang-en|'''Ricardo Wolf'''}}，{{lang-de|'''Ricardo Wolf'''}}，{{lang-he|'''ריקרדו וולף'''}}，{{bd|1887|||1981||}}）是一名具有[[猶太|猶太]]、[[德國|德國]]、[[古巴|古巴]]、[[以色列|以色列]]等國籍的[[發明家|發明家]]、[[外交官|外交官]]、[[慈善家|慈善家]]和前[[古巴|古巴]]駐[[以色列|以色列]]大使。是[[沃爾夫基金會|沃爾夫基金會]]（[[沃爾夫獎|沃爾夫獎]]）的創立者。

== 早年生活 ==
里卡多·沃爾夫於1887年出生於[[德國|德國]][[漢諾威|漢諾威]]，[[猶太|猶太]]社區的重鎮。共有14位[[兄弟姊妹|兄弟姊妹]]。

在[[第一次世界大戰|第一次世界大戰]]爆發前，里卡多·沃爾夫從[[德國|德國]]移民到[[古巴|古巴]]，並成為他的第二個家。

1924年，他和一位1920年代[[網球|網球]]冠軍Francisca Subirana結婚。

== 晚年生活 ==
里卡多·沃爾夫在煉鐵工廠上班多年後，終於開發出了一項從[[冶煉|冶煉]]的過程中回收[[鐵|鐵]]的製程。他的發明專利被全世界許多的鋼鐵廠所利用，並帶給他相當可觀的財富。

在經濟上和道義上的考量，沃爾夫決定資助[[菲德爾·卡斯楚|菲德爾·卡斯楚]]發動[[古巴革命|古巴革命]]。為感激沃爾夫堅定不移的支持和認知他的個性，被[[菲德爾·卡斯楚|菲德爾·卡斯楚]]任命作為外交官。 在1961年時擔任[[古巴|古巴]]駐[[以色列|以色列]]大使。直到1973年，[[古巴|古巴]]和[[以色列|以色列]]斷絕邦交關係沃爾夫才卸任大使的職務。

在沃爾夫生命的最後幾年，他決定保持[[以色列|以色列]]國籍並定居於以國直到逝世。1981年2月，沃爾夫博士在[[以色列|以色列]][[海爾茲利亞|海爾茲利亞]]逝世。

== 成立[[沃爾夫基金會|沃爾夫基金會]] ==
1975年，沃爾夫親手建立了此基金會。而於1978年開始頒發[[沃爾夫獎|沃爾夫獎]]。[[沃爾夫獎|沃爾夫獎]]是表彰在[[農業|農業]]、[[化學|化學]]、[[數學|數學]]、[[醫學|醫學]]、[[物理學|物理學]]和[[藝術|藝術]]等六個領域中有傑出表現的學者專家或研究人員。

其中，[[沃爾夫藝術獎|沃爾夫藝術獎]]是每年輪流在[[建築|建築]]、[[音樂|音樂]]、[[繪畫|繪畫]]和[[雕塑|雕塑]]等四個領域頒發。

每個獎項包括文憑和100,000美元獎金。
