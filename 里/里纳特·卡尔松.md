'''里纳特·卡尔松'''（{{lang-sv|'''Lennart Carleson'''}}，{{bd|1928年|3月18日|catIdx=C}}），[[瑞典|瑞典]]数学家。生于[[斯德哥尔摩|斯德哥尔摩]]。1992年因“其在[[傅立叶分析|傅立叶分析]]、[[复分析|复分析]]、[[拟共形映射|拟共形映射]]及[[动力系统|动力系统]]理论方面的基础性贡献”获得[[沃尔夫奖|沃尔夫奖]]，2006年又因“其在[[调和分析|调和分析]]和光滑动力系统方面深刻和重大的贡献”而获得[[阿贝尔奖|阿贝尔奖]]。

== 简历 ==
1950年获得[[乌普萨拉大学|乌普萨拉大学]]博士学位，1950年至1951年在美国[[哈佛大学|哈佛大学]]做博士后研究。后在乌鲁萨拉大学、斯德哥尔摩大学、瑞典皇家工学院任教。1986年兼任美国[[加州大学洛杉矶分校|加州大学洛杉矶分校]]教授，并曾担任[[斯坦福大学|斯坦福大学]]和[[麻省理工学院|麻省理工学院]]客座教授。

== 学术成就 ==
卡尔松是当代分析大师，解决了许多困难的未决问题。其中最轰动的是他在1965年证明了Lusin猜想成立，解决了困扰傅立叶分析领域多年的一个基础问题。这一结果的证明过程非常之难，以至于此后30年内很少有数学家能够应用。

他的另一重大贡献是解决了[[日冕问题|日冕问题]](Corona)，由此得到的[[卡尔松测度|卡尔松测度]]已经成为傅立叶分析和[[复分析|复分析]]的基本工具。

1972年证明在二维情形任意阶的Bochner-Riesz均值都有界（4/3<p<4），1988年与Benedicks合作证明Hénon映射对非空参数集均存在[[奇异吸引子|奇异吸引子]]从而打开了系统研究此类动力系统的大门，也是他非常有影响的贡献。

== 评价 ==
卡尔松在多个领域都有建树，有人评价他总是解决开创性的最困难的问题，然后就转到其他领域，将空间留给他人。

阿贝尔委员会这样评价卡尔松：“他的工作已经永远改变了我们对分析的看法。他不仅证明了极难的问题，而且他引入的证明方法与定理本身同等重要。”

== 社会活动 ==
卡尔松是瑞典科学院院士，美国艺术与科学院、俄罗斯科学院、英国皇家学院、法国、丹麦、挪威、芬兰、匈牙利等科学院外籍院土。1968年到1984年曾任Mttag－Leffler研究所所长。曾担任Acta Mathematica杂志编辑。 

卡尔松曾于1978年到1982年任[[国际数学联盟|国际数学联盟]]主席，期间极力促成[[中国数学会|中国数学会]]取得代表权。他还积极建立了表彰年轻理论计算科学家的[[奈望林纳奖|奈望林纳奖]]。他还热心数学普及工作，曾撰写了科普著作“Matematik för vår tid”（我们时代的数学）。他培养的26名博士，大多成为全世界各大名校的教授。
