'''里見氏'''是[[日本|日本]]的[[武家|武家]]。[[本姓|本姓]]是[[源氏|源氏]]（[[河內源氏|河內源氏]]）。出自[[新田義重|源（新田）義重]]之子・[[里見義俊|義俊]]的氏族，[[新田氏|新田氏]]之[[庶家|庶宗家]]。[[戰國時代_(日本)|戰國時代]]，一族中出現領有房總地方的[[戰國大名|戰國大名]]・'''安房里見氏'''。

== 概要 ==
里見氏是[[新田氏|新田氏]]之[[庶家|庶宗家]]，也稱「大新田」氏。[[鎌倉時代|鎌倉時代]]，是身為[[御家人|御家人]]的將軍近侍，在鎌倉時代末的動亂和新田氏共同行動。[[南北朝時代_(日本)|南北朝]]期，一族分裂，分屬南朝方・北朝方。

根據安房里見氏家傳，里見氏的嫡流[[里見義實|里見義實]]安房入國，是安房里見氏之初祖，但是系譜關係不確定。安房里見氏成長為戰國大名支配房總，在江戶時代初期成為統治安房一國的[[館山藩|館山藩]]主，但是在1614年，[[里見忠義|里見忠義]]的時代改易。

其他還有鎌倉時代至[[室町時代|室町時代]]，分支到諸國的里見氏，有出羽里見氏、美濃里見氏、越後里見氏等。

== 安房里見氏 ==
'''安房里見氏'''（あわさとみし）是[[戰國時代_(日本)|戰國時代]]掌握[[安房國|安房國]]，在[[房總半島|房總半島]]擴大勢力，成為戰國大名的氏族。自稱「關東副帥」（[[關東管領|關東管領]]之異稱）或「關東副將軍」。

== 越後里見氏 ==
'''越後里見氏'''（えちごさとみし）包括[[里見義俊|里見義俊]]之二男・[[田中義清|田中義清]]在越後獲得所領、[[里見義成|里見義成]]之二男・義繼獲得越後國波多岐莊大井田鄉（現・[[新潟縣|新潟縣]][[十日町市|十日町市]]）之所領、義成之三男・鳥山時成也獲得波多岐莊之所領。大井田氏・[[鳥山氏|鳥山氏]]・[[田中氏#里見系_田中氏|田中氏]]等里見氏的支族形成'''越後新田黨'''，在南北朝的動亂期，跟隨宗家[[新田義貞|新田義貞]]共同活躍。

== 美濃里見氏 ==
'''美濃里見氏'''（みのさとみし）是里見義成之四男・[[里見義直|義直]]在[[承久之亂|承久之亂]]後，成為美濃國圓教寺的地頭，移住當地為起始。

南北朝時代，[[里見義宗|里見義宗]]依附北朝方獲得所領，但是在[[觀應之擾亂|觀應之擾亂]]依附[[足利直義|足利直義]]後没落。

== 出羽里見氏 ==
'''出羽里見氏'''（でわさとみし）是始於里見義成之四男・[[里見義直|義直]]的家系。子孫以[[出羽國|出羽國]][[成生庄|成生庄]]（現・[[山形縣|山形縣]][[天童市|天童市]]）為據點扶植勢力，但是移住出羽的經緯不明。

=== 天童氏 ===
[[南北朝時代_(日本)|南北朝時代]]，成生庄之[[里見義景|里見義景]]無子，以[[源義國|源義國]]之家系，[[足利氏|足利氏]]一門之[[斯波家兼|斯波家兼]]三男・[[天童義宗|天童義宗]]為養子。里見義宗繼承家督稱[[天童氏|天童氏]]。

=== 水戶藩士里見氏 ===
出羽里見氏之傍流，仕於水戶藩。
