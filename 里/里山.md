'''里山'''，是指由[[住宅|住家]]、[[聚落|聚落]]、[[耕地|耕地]]、[[池塘|池塘]]、[[溪流|溪流]]與[[山|山]][[丘|丘]]等混和而成的地景。
== 涵意 ==
「里山」在[[日语|日語]]中原指定期依序採伐的[[次生林|次生林]]（如橡樹林、松樹林、竹林等），以及可供覆蓋屋頂、[[饲料|牲畜飼料]]及[[堆肥|堆肥]]等用途的草地。

如今，該詞多用於指涉位於[[山|高山]]和[[平原|平原]]之間包含[[社區|社區]]、[[森林|森林]]和[[農業|農業]]的混合地景或複合式[[农村|農村]][[生态系统|生態系統]]。
== 里山地景 ==
在里山地景之內，許多不同種類的[[生态环境|生態環境]]混合在一起，成為眾多野生[[物种|物種]][[動物棲地|棲地]]，並有助於防災、[[集水區|集水區]]保護等目的。<ref>{{Cite news|url=http://e-info.org.tw/node/78570|title=里山倡議的核心概念與國際發展現況|newspaper=台灣環境資訊協會-環境資訊中心|accessdate=2017-02-12|language=zh-hant}}</ref><ref>{{Cite news|url=https://www.newsmarket.com.tw/blog/76849/|title=台灣里山地圖 全台走透透|date=2015-10-08|newspaper=上下游News&Market新聞市集|accessdate=2017-02-12|language=zh-TW}}</ref><ref>{{Cite web|url=http://www.cna.com.tw/proj_goodbook/1984.aspx|title=里山資本主義|accessdate=2017-02-12|author=|date=|work=www.cna.com.tw|publisher=}}</ref>
== 里山倡議 ==
里山倡議以世界各地類似日本地區里山地景的複合式農村生態系統為目標，其願景為謀求兼顧[[生物多樣性|生物多樣性]]維護與[[可持续发展|資源永續利用]]之平衡。<ref>{{Cite web|url=http://www.kskk.org.tw/satoyama/?page_id=15|title=實踐里山倡議|accessdate=2017-02-12|author=|date=|work=www.kskk.org.tw|publisher=裡山塾 - 觀樹教育基金會|language=zh-TW}}</ref><ref>{{Cite web|url=http://www.coa.gov.tw/ws.php?id=2505424|title=原鄉實踐里山倡議精神之初探(農委會)|accessdate=2017-02-12|last=COA|work=www.coa.gov.tw}}</ref>
