'''穴小鴞'''（[[學名|學名]]：{{lang|la|''Athene cunicularia''}}），[[鴟鴞科|鴟鴞科]][[小鴞屬|小鴞屬]]（Athene）的一種，因住在[[洞穴|洞穴]]而得名。主要分布於[[北美洲|北美洲]]及[[南美洲|南美洲]]地區。常見於[[草原|草原]]、[[牧地|牧地]]、[[農業|農業]]地區、[[荒漠|荒漠]]，或者低[[植被|植被]]的乾燥區域<ref name = lewis2005>Lewis (2005)</ref>。有別於其他種[[貓頭鷹|貓頭鷹]]僅活動於[[夜間|夜間]]，穴小鴞於日間依舊能照常活動，亦因此，其狩獵時間自[[黃昏|黃昏]]即已開始。

== 簡述 ==
眼睛為[[亮黃色|亮黃色]]；[[額嘴|額嘴]]依種類不同，為[[暗黃|暗黃]]或[[灰色|灰色]]。成年後的穴鴞，頭部[[羽毛|羽毛]]毛色轉為[[棕色|棕色]]，並在[[翅膀|翅膀]]上有著[[白色|白色]]的[[斑點|斑點]]。前胸、腹部的羽毛為白色，並因種類不同，而有著棕色的斑點或[[條紋|條紋]]。比起其他貓頭鷹來說，穴鴞飛行的能力雖然較為不佳，但其細長的腳卻有利於補捉[[草叢|草叢]]或水邊的獵物。

雄性穴鴞與雌性穴鴞在外貌上相差無幾，雌性相對於雄性較為碩大，而雄性之翅翼與[[尾巴|尾巴]]則較雌性來的修長。成年雄性穴鴞，因較常在洞外活動，羽毛因被太陽曝曬而變色，相比於雌性穴鴞來的明亮。成年穴鴞僅大於[[美洲知更鳥|美洲知更鳥]]（American Robin）些許，身長約25公分，體重約170公克<ref name = lewis2005 />。

== 分類與系統 ==
於[[骨學|骨學]]（osteology）和[[脫氧核糖核酸序列|脫氧核糖核酸序列]]數據中顯示，穴鴞為[[小鴞屬|小鴞屬]]中的[[陸生動物|陸生動物]]（Terrestrial animal）[[變體|變體]]。在分類方面形成了為數不少的[[亞種|亞種]]，但是牠們在外觀上相異不多，並且尚有幾個需進一步研究，方能證實<ref name = holtetal1999>Holt ''et al.'' (1999)</ref>。

多數亞種被發現於[[安地斯山脈|安地斯山脈]]附近與[[安的列斯群島|安的列斯群島]]上，唯獨[[北方穴鴞|北方穴鴞]]（或稱[[西方穴鴞|西方穴鴞]]）與[[佛羅里達穴鴞|佛羅里達穴鴞]]（Florida Burrowing Owl）被發現於[[美國|美國]]。以下為穴鴞下屬的亞種列表<ref name = korfantaetal2005>Korfanta ''et al.'' (2005)</ref>：

* ''A. c. cunicularia'' <small>([[璜·伊納修·_摩里納|Molina]], 1782)</small>：[[南方穴鴞|南方穴鴞]]（Southern Burrowing Owl），分布於南[[波利維亞|波利維亞]]、南[[巴西|巴西]]和[[火地群島|火地群島]]。
* ''A. c. grallaria'' <small>([[康拉德·雅各·特明克|Temminck]], 1822)</small>：[[巴西穴鴞|巴西穴鴞]]（Brazilian Burrowing Owl），分布於[[巴西|巴西]]中部、西部。
* ''A. c. hypugaea'' <small>([[夏爾·呂西安·波拿巴|Bonaparte]], 1825)</small>：[[北方穴鴞|北方穴鴞]]（Northern Burrowing Owl），分布於[[加拿大|加拿大]]南方至[[北美大平原|北美大平原]]間。
* ''A. c. floridana'' <small>([[羅伯特·李奇微|Ridgway]], 1874)</small>：[[佛羅里達穴鴞|佛羅里達穴鴞]]（Florida Burrowing Owl），分布於[[佛羅里達|佛羅里達]]與[[巴哈馬|巴哈馬]]。
* ''A. c. guadeloupensis'' <small>([[羅伯特·李奇微|Ridgway]], 1874)</small>：[[瓜德羅普穴鴞|瓜德羅普穴鴞]]（Guadeloupe Burrowing Owl）， 分布於[[瓜羅德普島|瓜羅德普島]]與[[瑪莉嘉隆|瑪莉嘉隆]]（Marie-Galante），於1890年[[絕種|絕種]]。
* ''A. c. amaura'' <small>([[喬治·紐博爾德·羅倫斯|Lawrence]], 1878)</small>：[[安提瓜穴鴞|安提瓜穴鴞]]（Antiguan Burrowing Owl），分布於[[安提瓜島|安提瓜島]]、[[聖徒基茨希爾|聖徒基茨希爾]]和[[尼維斯島|尼維斯島]]，於1905年[[絕種|絕種]]。

* ''A. c. nanodes'' <small>([[漢斯·馮·貝勒普施|Berlepsch]] & [[Jan_Sztolcman|Stolzmann]], 1892)</small>：[[西南秘魯穴鴞|西南秘魯穴鴞]]（Southwest Peruvian Burrowing Owl），分布於[[秘魯|秘魯]]西南部。

* ''A. c. tolimae'' <small>(Stone, 1899)</small>：[[西哥倫比亞穴鴞|西哥倫比亞穴鴞]]（West Colombian Burrowing Owl），分布於[[哥倫比亞|哥倫比亞]]。
* ''A. c. juninensis'' <small>(Berlepsch & Stolzmann, 1902)</small>：[[南安地斯山穴鴞|南安地斯山穴鴞]]（South Andean Burrowing Owl），分布於[[秘魯|秘魯]]中部至[[阿根廷|阿根廷]]間。

* ''A. c. intermedia'' <small>(Cory, 1915)</small>：[[西秘魯穴鴞|西秘魯穴鴞]]（West Peruvian Burrowing Owl），分布於[[秘魯|秘魯]]西部。
* ''A. c. minor'' <small>(Cory, 1918)</small>：[[蓋亞那穴鴞|蓋亞那穴鴞]]（Guyanan Burrowing Owl），分布於[[蓋亞那|蓋亞那]]、[[羅賴馬州|羅賴馬州]]。

== 生態 ==
在[[歐洲|歐洲]][[殖民|殖民]][[美洲|美洲]]之前，穴鴞廣泛的分布於[[新大陸|新大陸]]各個適於其生存之地。但是在歐洲列國的殖民下，生態遭到破壞，於[[北美洲|北美洲]]僅於限定區域生存，如[[北美大平原|北美大平原]]、[[佛羅里達|佛羅里達]]等地區。而在[[南美洲|南美洲]]，其生存範圍，亦隨著[[森林開伐|森林開伐]]而不斷減少。

大多數的穴鴞不具[[遷徙|遷徙]]之習性，一年四季皆在其生存地區活動。他們的壽命大約9到10年<ref>OwlPages.com (2005)</ref>。天敵包括[[獾|獾]]、[[土狼|土狼]]、[[蛇|蛇]]和[[人類|人類]]（於飛行途中遭行經車輛給撞死）。

== 食性 ==
在[[狩獵|狩獵]]時，牠們會在棲息處靜靜觀察著，直到[[獵物|獵物]]出現，再從至高點像獵物猛撲，並抓往空中，使之無法逃離。其獵物組成為[[無脊椎動物|無脊椎動物]]與小型的[[脊椎動物|脊椎動物]]，分別在其飲食比例上占了三分之一和三分之二的比例。大部分來說，穴鴞通常以大型[[昆蟲|昆蟲]]與小型的[[嚙齒目|嚙齒目]]動物，而像[[地松鼠|地松鼠]]此類動物，穴鴞雖然時常接近，但捕食機率不高。

再[[有鱗目|有鱗目]]和[[兩棲動物|兩棲動物]]中，穴鴞尤其偏好[[蜥虎|蜥虎]]、[[青蛙|青蛙]]和[[蟾蜍|蟾蜍]]。通常情況下，穴鴞所獵捕之[[脊椎動物|脊椎動物]]為脫離群體，落單的個體<ref>.  Fig. 1 in Motta-Junior (2006) is misleading: it shows the average weight of both vertebrate and invertebrate components. Compare ''Tyto alba'', which feeds almost exclusively on the same sort of rodents and other small vertebrates (but not invertebrates) as ''A. cunicularia''.</ref>。而穴鴞最常獵捕的脊椎動物是小型[[鳥類|鳥類]]，如：[[哀鴿屬|哀鴿屬]]<ref name = mottajunior2006>Motta-Junior (2006)</ref>。

而在[[無脊椎動物|無脊椎動物]]方面，穴鴞並沒特別的偏好與特定獵食對象。舉凡[[白蟻|白蟻]]、[[直翅目|直翅目]]的[[螽斯科|螽斯科]]與[[沙螽科|沙螽科]]（Jerusalem cricket）、[[蟋蟀|蟋蟀]]、[[金龜子|金龜子]]、[[狼蛛|狼蛛]]、[[蠍子|蠍子]]、[[馬陸|馬陸]]、[[步行蟲|步行蟲]]（Ground beetle）皆為其狩獵之對象<ref name = mottajunior2006 />。

不同於其他[[貓頭鷹|貓頭鷹]]，穴鴞為[[雜食動物|雜食動物]]，除了[[昆蟲|昆蟲]]、小型[[脊椎動物|脊椎動物]]外，亦吃[[果子|果子]]、[[種子|種子]]，尤其是[[仙人掌|仙人掌]]的[[果實|果實]]。在[[克拉里翁島|克拉里翁島]]，因缺乏可獵食之小型[[哺乳動物|哺乳動物]]，穴鴞以[[蟋蟀|蟋蟀]]、仙人掌果實為哺育幼兒的主食，[[哀鴿|哀鴿]]等小型鳥類則為輔<ref>Brattstrom & Howell (1956)</ref>。

== 繁殖 ==
[[File:Zico_e_filhotes_2.jpg|thumb]]。]]
穴鴞的[[繁殖季節|繁殖季節]]為3月下旬或4月開始，為[[一夫一妻制|一夫一妻制]]，但亦偶有雄性穴鴞有著兩個甚至以上之配偶的情形發生，但僅為少數<ref name = lewis2005 />。他們通常於[[平原|平原]]或[[大草原|大草原]]築建繁殖的棲所，但視情形而定，亦有發生過穴鴞於[[機場|機場]]、[[高爾夫球場|高爾夫球場]]、[[農地|農地]]等開闊性場地築建棲所的案例發生。

在繁殖季節時，穴鴞會到處收集各式各樣的[[材料|材料]]來築建牠們的[[巢|巢]]。最常見的材料為[[糞便|糞便]]，並以[[牛糞|牛糞]]居多。早些時候，使用糞便裝飾此一舉動多半被認為是掩蓋氣味，使天敵未能發現他們之巢穴。然實驗顯示，此舉並未任何幫助於此，並進而發現，此一功能類似於[[捕鼠器|捕鼠器]]，吸引獵物前來，進而方便於自家門前狩獵，便於哺育幼兒<ref>Levey ''et al.'' (2004)</ref>。

[[生產|生產]]期大概1至2天，直至一窩蛋全部產完，每次生產會產下4－12顆[[蛋|蛋]]（通常為9顆）。生產完接下來是3－4個星期的[[孵化|孵化]]期，此一時間由雄性穴鴞負責在外狩獵。小穴鴞誕生後須歷經4個星期方能進行短程[[飛行|飛行]]與離開[[巢穴|巢穴]]，此段時間則由父母輪流照顧、餵養。通常到最後能安然離開巢穴的僅4－5隻。

== 保育現狀 ==
[[File:Burrowing_Owl_and_burrow.jpg|thumb]]建築巢穴的穴鴞]]
