'''耶魯學院'''（Yale College）是[[耶魯大學|耶魯大學]]自1718年至1887年的正式校名，現在是耶魯大學的本科部，包括12個[[住宿學院|住宿學院]]。 

耶魯大學1701年創立時的原名為Collegiate School，1718年為感謝[[伊利胡·耶魯|伊利胡·耶魯]]先生的捐助，將校名改為耶魯學院，1887年，耶魯學院改名為耶魯大學，而耶魯學院成為大學的本科部名稱。

耶魯學院實行住宿學院制，由12所學院組成。這種制度始於1933年，由一位非常讚賞[[牛津大學|牛津大學]]和[[劍橋大學|劍橋大學]]類似制度的校友Edward S. Harkness捐款建立。每所學院對學生都有一套完備的支持體系，包括院長（Master）、學監（Dean）、駐院學者和Fellow。每所學院都有不同的建築風格，但都包括庭院和完備的設施。雖然每所學院都設有自己的討論課程（但向所有學生開放通選），舉辦自己的社交活動和院長茶會（Master』s Tea），耶魯學生仍然積極參與全校的學習和社會活動。

耶魯大學的住宿學院都以校史中的著名人物或者校友命名，而故意避免以捐款者命名。

耶魯大學住宿學院（按字母順序排列）：
# [[Berkeley_College|Berkeley College]]：以[[喬治·貝克萊|喬治·貝克萊]]（1685-1753）之名命名，他為耶魯早期的發展作出了貢獻。
# [[Branford_College|Branford College]]：以康州一地名命名，為耶魯大學所在地區。
# [[Calhoun_College|Calhoun College]]：以[[約翰·C·卡爾霍恩|約翰·C·卡爾霍恩]]之名命名，曾就任美國副總統。
# [[Davenport_College|Davenport College]]：以John Davenport之名命名，是紐黑文市的創立人。（該學院常被學生稱為D』Port）
# [[Ezra_Stiles_College|Ezra Stiles College]]：以耶魯大學校長Ezra Stiles之名命名。常被學生稱作Stiles。
# [[喬納森·愛德華茲學院|喬納森·愛德華茲學院]]：以耶魯大學校友、神學家和普林斯頓大學創始人[[喬納森·愛德華茲|喬納森·愛德華茲]]之名命名，為耶魯第一個住宿學院。通稱為JE。
# [[Morse_College|Morse College]]：以[[摩爾斯電碼|摩爾斯電碼]]發明人[[薩繆爾·摩爾斯|薩繆爾·摩爾斯]]之名命名。
# [[Pierson_College|Pierson College]]：以耶魯首位校長Abrahan Pierson之名命名。
# [[Saybrook_College|Saybrook College]]：以康州一地名命名，是耶魯大學的創立地。
# [[Silliman_College|Silliman College]]：以耶魯大學教授，著名化學家[[本傑明·西利曼|本傑明·西利曼]]之名命名。他發明了石油的分餾法。
# [[Timothy_Dwight_College|Timothy Dwight College]]：由耶魯大學兩任校長Timothy Dwight四世和五世命名。常稱為TD。
# [[Trumbull_College|Trumbull College]]：以康州州長Jonathan Trumbull之名命名。

自1990年起，耶魯開始大規模的翻新老舊的住宿學院建築。Berkeley College是第一個結束翻新的，最近一個完成翻新的是在2006年9月重新投入使用的Trumbull College。翻新的住宿學院保持了設計者的初衷，維持了歷史感，但更新的設施使其更舒適、更符合現代居住規範。

20世紀60年代到70年代，耶魯大學曾計劃建立第13個學院，但後來由於多方的壓力並沒有實現。
