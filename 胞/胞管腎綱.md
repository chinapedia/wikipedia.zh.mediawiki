'''胞管腎綱'''（學名：{{lang|la |Secernentea}}），又名'''側尾腺綱'''或'''尾感器綱'''，是[[线虫动物门|线虫动物门]]之下的一個重要的[[綱_(生物)|綱]]。

==分類==
胞管腎綱之下的綱目計有<ref name = "ToL:2002_Nematoda">{{cite web | publisher = [[Tree_of_Life_Web_Project|Tree of Life Web Project]] (ToL) | title = Nematoda | date = 2002-01-01 | url = http://tolweb.org/Nematoda/2472/2002.01.01 | accessdate = 2008-11-02  }}</ref>：
*[[双胃亚纲|双胃亚纲]] [[Diplogasteria|Diplogasteria]]：亦作[[雙胃線蟲亞綱|雙胃線蟲亞綱]]，可能與Rhabditia為並系群<ref name = "ToL:2002_Nematoda" />
**[[雙胃目|雙胃目]] [[Diplogasterida|Diplogasterida]]
*[[小桿亞綱|小桿亞綱]] [[Rhabditia|Rhabditia]]：可能為並系群<ref name = "ToL:2002_Nematoda" />
**[[小桿目|小桿目]] [[Rhabditda|Rhabditda]]：[[秀麗隱桿線蟲|秀麗隱桿線蟲]]
**[[圓線蟲目|圓線蟲目]] [[Strongylida|Strongylida]]：亦作[[园线目|园线目]]，例如：[[廣東住血線蟲|廣東住血線蟲]]
*[[旋尾亚纲|旋尾亚纲]] [[Spiruria|Spiruria]]
**[[蛔目|蛔目]] [[Ascaridida|Ascaridida]]：[[海獸胃線蟲|海獸胃線蟲]]
**[[驼形目|驼形目]] [[Camallanida|Camallanida]]：有時被歸入旋尾目 
**[[蚓線目|蚓線目]] [[Drilonematida|Drilonematida]]：有時被歸入旋尾目
**[[尖尾目|尖尾目]] [[Oxyurida|Oxyurida]]：[[蟯蟲|蟯蟲]]
**[[Rhigonematida|Rhigonematida]]目：舊屬墊刃亚纲
**[[旋尾目|旋尾目]] [[Spirurida|Spirurida]]：[[麦地那龙线虫|麦地那龙线虫]]
*[[墊刃亚纲|墊刃亚纲]] [[Tylenchia|Tylenchia]]：可能與Rhabditia為並系群<ref name = "ToL:2002_Nematoda" />，其下兩個目的分類單元在2011年合併成為[[墊刃下目|墊刃下目]]（[[Tylenchomorpha|Tylenchomorpha]]） 
**[[滑刃目|滑刃目]] [[Aphelenchida|Aphelenchida]]
**[[墊刃目|墊刃目]] [[Tylenchida|Tylenchida]]
