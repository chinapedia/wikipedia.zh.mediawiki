'''侠岚'''（英文：'''Shalen'''，原名：'''神之伽蓝传'''）是一部中国大陆[[三维|三维]][[武侠|武侠]][[动画|动画]]，由[[北京若森数字科技有限公司|北京若森数字科技有限公司]]出品。主要讲述了拥有离奇身世的少年辗迟，为了找回幼时失散的姐姐，机缘巧合下来到了传说中的侠岚驻地玖宫岭，成为侠岚并与伙伴们不断超越自我，与‘零’抗争的故事<ref>{{Cite web|url = http://donghua.7755.com/z_xialan60712|title = 侠岚剧情简介|accessdate = 2015-07-24|author = |date = 2008-1|publisher = |deadurl = yes|archiveurl = https://web.archive.org/web/20150724073737/http://donghua.7755.com/z_xialan60712|archivedate = 2015-07-24}}</ref>。目前已播出七部，计划共播出約兩百集。

==故事概要==
故事的主人公[[辗迟|辗迟]]，是个孤儿，身世成谜，被桃源镇饺子馆的老板娘[[辣妈|辣妈]]收养。从此，他便一直与辣妈及辣妈的侄女[[墨夷|墨夷]]在一起生活。直到他五岁的一个夜晚，因为某个复杂事件，墨夷被一个坏人（[[山鬼谣|山鬼谣]]）带走。辗迟把墨夷当姐姐看待，此事对他打击很大，他认为自己没能保护姐姐，没能阻止她被带走。为了变得更强以找回姐姐，辗迟立志成为侠岚。之后，他认识了自己的伙伴[[千钧|千钧]]、[[辰月|辰月]]及老师[[弋痕夕|弋痕夕]]、[[破陣統領|破陣統領]]等，共同踏上了冒险的旅程。

== 登场人物 ==

=== 普通人 ===
*墨夷（配音：[[山新|山新]]）
饺子馆辣妈的侄女，辗迟的姐姐（無血緣關係），在辗迟五岁时因意外涉入山鬼谣的卧底计划，左师将神坠封印其体内，后被山鬼谣带走。第一季~第四季，身体里封印着左师的神坠。在无极之渊第一层封印解开时（第四季片尾）被万零之王穹奇附体，但穹奇在第六季最后一集附体辗迟后被破阵吸入自己身体里，最后和破阵同归于尽，而墨夷也和辗迟平安的回到了饺子馆，一家人紧紧地抱在一起。
*辣妈（配音：白雪岑）
辣不辣饺子馆的老板娘，收养了辗迟。为主人公辗迟的养母，性格正直、但爱贪小便宜。火爆的脾气下是一颗温柔的心，对待辗迟的照料更是无微不至，可惜辗迟总是惹她生气。辗迟小时候体质不好，老板娘用三生草救了辗迟，并且经常给辗迟吃三生草做成的饺子，在辗迟长大后，老板娘意识到辗迟不是一个普通的小孩，于是有意识地训练辗迟自己上山采集三生草，意在无形中锻炼辗迟的能力。

=== 玖宫岭 ===
一個集合了各地有俠嵐印記，並立志成為俠嵐消灭世间的零（重零、霸零、五敗、七魄、三魂、穷奇）及守護世间和平的人之聚集地。
==== 炽天殿 ====
*辗迟（配音：褚珺）
性格开朗、活泼、充满正义感。短发的阳光少年，身上总是穿一件后背印着“辣”字的背心，给人感觉唯恐别人不知道他是在桃源镇辣不辣饺子馆长大的。（其实是饺子馆老板娘强加上去的广告）名字由来為展翅（谐音辗迟，又意展翅高飞）於嬰兒時期被注入元氣與零力，从小被桃源镇饺子馆的老板娘收养。在他五岁的一个夜晚，姐姐墨夷被人带走，从此，辗迟便立志成为侠岚救回姐姐。后与千钧、辰月进入玖宫岭炽天殿，因身上具有零力而遭到他人非议，喜欢辰月。元炁属性：火。
*千钧（配音：张杰）
喜欢叫辗迟“棒槌”，沉默冷静，不苟言笑，是辗迟的好友。千钧虽然看似心如止水，有时甚至显得冷漠，但其实他也有着温柔，甚至可爱的一面。因为本质上，千钧是一个重感情的男孩。千钧平时冷静坚强，但他也有着一段悲伤的回忆。在千钧幼年时，其父为保护他在与零的战斗中牺牲，因此对零十分憎恨，但事實上是被假葉所害。喜欢辰月。元炁属性：水。
*辰月（配音：[山新]）
单纯可爱，善良坚强。出生于侠岚世家，不喜战斗。经常与辗迟、千钧一起执行任务，三人是很好的朋友。也同时在卧底山鬼谣口中得知自己是辗迟最重要的人，与辗迟互相喜欢。碧婷是她最好的朋友 辰月因三月出生，所以才得此名。身在侠岚世家。她的爷爷就是一名神坠守护者。从小辰月就在她的爷爷指导下学习侠岚术，并和爷爷约定日后要成为一名优秀的侠岚。在一次战斗中，辰月爷爷牺牲了，零夺走了他守护的神坠。辰月因为这件事情，多年来一直无法走出阴影。元炁属性：金。
*弋痕夕（配音：赵毅）
太极侠岚，炽天殿镇殿使，出身于鸾天殿，神坠守护者。思维缜密，平易近人。與山鬼謠不但是同門，也是很好的朋友（小时候与弋痕夕相识后两人一起被左师带回玖宫岭收为徒弟）。学生為辗迟、千钧、辰月。元炁属性：木。

==== 朱天殿 ====
*游不动（配音：边江）
活泼可爱，矮胖敦实，十分贪吃，为辗迟的好友，喜欢碧婷。有时候看似胆小，但关键时刻，游不动绝对是一个信得过、可以依赖的朋友。元炁属性：土。
*碧婷（配音：白雪岑）
热情真诚、自信直爽，爱恨分明，出生于侠岚世家。碧婷家世代负责驯养海东青，小时候的碧婷一直梦想着像海东青一样，能够无拘无束地翱翔天际。但一直顺风顺水长大的碧婷，从没想过自己有折翅的一天，如果真有那一天，她会怎么面对呢？碧婷在侠岚序时，强势的样子很容易让人猜到她一直无忧无虑的生活，每次上课都要坐在前面，抢着回答问题，放学之后在朋友之中最引人注目。喜欢千钧。元炁属性：金。
*归海（配音：边江）
冷静成熟，最爱说“无聊”二字。归海是一个孤儿，小时候被一个女侠岚所救，女侠岚为了救他而双目失明，而归海一年后始终摆脱不了心里阴影。最后被女侠岚封印了记忆，并且被女侠岚所照顾着，知道真相以后的归海，终于找到了他妈妈（也就是那位女俠嵐）一直要他找的那个答案。被七魄之胄附体用于夺取神坠過。元炁属性：水。
*扰龙（配音：张震）
太极侠岚，朱天殿镇殿使。刚开始由于脾气与性格，并没有像弋痕夕、山鬼谣那样成为镇殿使，而是先在侠岚序训练受训侠岚。从而，也没有获得神坠守护者的资格。但扰龙对学生的亲近胜过了任何一位镇殿使，这也被破阵统领提升为朱天殿镇殿使。在战斗中为掩护弋痕夕等人撤退，散尽自身元炁，将自己化作一座山峰，阻止了零的追击。元炁属性：土。
*相离
太极侠岚，曾是朱天殿镇殿使，辰月的爷爷，柏寒的父亲，神坠守护者。过去在一次任务中遭到昧谷势力伏击被俘，后在假叶入侵九宫岭时出现，并被零力侵入十成，经辰月等人的努力恢复清醒，为保护辰月等人使出天乾燧宇玄芒对抗假叶的魑魅森罗、魍魉万象而牺牲。元炁属性：金。

==== 阳天殿 ====
*独龙
独断专行，坚忍不拔，云丹的学生。一心想重振阳天殿在众人心中的地位，申屠牺牲后成为鸾天殿镇殿使。元炁属性：木。
*末丑
阳天殿学生，指導老師為云丹。元炁属性：土。
*纥骨
阳天殿学生，指導老師為云丹。元炁属性：水。
*云丹（配音：[[山新|山新]]）
太极侠岚，阳天殿镇殿使，外冷内热，冷静而理智，在对阵山鬼谣过程中被山鬼谣击败，后假叶要求山鬼谣将其丢入万葬洞，於第六季中期回歸。元炁属性：金。
*山鬼谣（配音：李璐）
太极侠岚，曾担任阳天殿镇殿使，神坠守护者。天资奇高，实力非凡，为完成破陣的卧底计划，而犧牲自己的师傅左师，至今仍不想想起此事。臥底在昧谷十年，後回到玖宮岭。元炁属性：金。

==== 鸾天殿 ====
*九方
自大傲气，实力不凡，被称为四象侠岚中的精英昂，指导老师為申屠。元炁属性：金。
*地连
鸾天殿学生，指导老师為申屠。元炁属性：土。
*昆吾
鸾天殿学生，指导老师為申屠。元炁属性：火。
*申屠
太极侠岚，鸾天殿镇殿使，被假叶偷袭后散尽元炁牺牲。元炁属性：金。
*左师
太极侠岚，原鸾天殿镇殿使，神坠守护者。收留了流浪侠岚山鬼谣和弋痕夕，并成为他们的老师和父亲。也是弋痕夕、山鬼瑶、云丹的师傅，后为山鬼谣能取信于假叶而被山鬼谣所杀。元炁属性：土。

==== 玄天殿 ====
*霞露	
身材娇小，少言寡语。和自己的两个姐姐终黎、荔菲一样，霞露在很小的时候就进入侠岚序学习。小时候的游不动曾想欺负霞露，却被霞露倒吊在侠岚序门口。元炁属性：水。
*终黎
玄天殿学生。是霞露的姐姐，和荔菲是双胞胎。元炁属性：金。
*荔菲
玄天殿学生。是霞露的姐姐，和终黎是双胞胎。元炁属性：木。
*天净沙（配音：赵述仁 ）
太极侠岚，玄天殿镇殿使，神坠守护者，实力超群。有一只叫小年糕的宠物（巨大蜥蜴）。也有一只老萌老萌的小蜥蜴叫小米。元炁属性：火。

==== 成天殿 ====
*木易
心胸开阔，敏捷犀利，浮丘的学生。元炁属性：木。
*石作
成天殿学生，指導老師為浮丘。元炁属性：金。
*奔水
成天殿学生，指導老師為浮丘。元炁属性：水。
*浮丘（配音：邱秋）
太极侠岚，成天殿镇殿使，神坠守护者，美貌聪慧，气质非凡。爆发实力时会改变身形。元炁属性：水。

==== 幽天殿 ====
*金齿
幽天殿学生，元炁属性：金。
*银齿
幽天殿学生，元炁属性：火。
*黑齿	
幽天殿学生，元炁属性：水。
*子言
太极侠岚，幽天殿镇殿使，神坠守护者，外形像小孩，实力不俗，但他不及山鬼谣。他年龄与山鬼谣是一辈的。元炁属性：木。

==== 暤天殿====
*巫马
暤天殿学生。指導老師為钟葵。和荔菲领悟相同的侠岚术风巽潜影。元炁属性：木。
*石牛
暤天殿学生。指導老師為钟葵。元炁属性：土。
*乐羊
暤天殿学生。指導老師為钟葵。元炁属性：木。
*钟葵
太极侠岚，暤天殿镇殿使，神坠守护者，医术高超。元炁属性：火。
==== 钧天殿====
*破阵
冷静，严肃，神秘，玖宫岭的统领。第六季成为无极侠岚，与穹奇同归于尽。元炁属性：水。

==== 其他 ====
*游刃
两仪侠岚，游不动之父，玖宫岭食堂的大厨，饺子馆辣妈的师哥。元炁属性：金。

=== 零 ===
==== 五败 ====
*破
拥有化身成他人的能力，在七星冢曾欲夺取弋痕夕神坠，失败后企图自爆，被嗅探奇磊推入悬崖。
*散
身材矮小却擅使计策，在绝炁逆空中对阵辗迟、千钧，被辗迟打成两段，最终被假叶杀死。
*害
武器弑神钟，在绝炁逆空中对阵辰月、弋痕夕，被辰月重伤后企图同归于尽，被弋痕夕踢飞，自爆身亡。
*断
拥有吸收侠岚攻击元炁的能力，被天净沙击杀。
*伤 
行为仿佛人类老者但智慧不低，与天净沙对战后被天净沙褪去记忆并借之潜入昧谷，被鬼爪发现后恢复记忆，最终被天净沙击杀。
==== 七魄 ====
*假叶
七魄之首，阴冷、狠毒、善于利用他人、为得到穹奇的力量有着不可磨灭的恶念。绝招魑魅森罗、魍魉万象。
*胄
阴险，擅于利用附体控制别人，被辗迟等人重伤后散尽零力，开启传送阵，将鬼爪、柱纹、斑狼传送过来与众人决斗。
*鬼爪
凶狠、残暴，手段毒辣，在赶回昧谷报信的途中遭遇游不动、碧亭、归海的阻击，最后在死去扰龙残留元气的保护下，游不动等人将其击杀。
*鼠尾
狡猾，习惯躲在暗处攻击，在弋痕夕等人追击假叶的过程中，被弋痕夕的风巽千叶翔龙波及，坠入悬崖而亡。
*柱纹
善于诱导对手心中的恶念，在辗迟、辰月、千钧赶往无极之渊途中阻击三人，最终被千钧心魔所杀。
*斑狼
性格贪婪，可以吸收侠岚元炁，最终被千钧等人合力击杀。
*逝炎
在生命垂危时，附体佚烁，性格也随之产生改变。在抢夺司南的过程中，被浮丘击杀。

==== 三魂 ====
*幽
三魂之首，實力在霜、汰之上，擁有着汰的零術，實力強大，已於第六季第十集出場。在劇情中復活了已死去的七魄及五敗，給它們所有一個時辰的壽命，為了奪取所有神墜，夷平玖宮岭。其後吸取了霜的金屬性零力，使其擁有火、金、土，三種屬性的零力，並且得到了霜的零術，與輾遲等人交戰，之後因體內屬性相剋，與夜陽同歸於盡，但因窮奇的零力感應，令其再次復活。
*霜
已於第六季第六集出場，實力與智力都在汰之上，可在空中步行，可同時拿取對她攻擊的人的元炁，有着逆輪迴的零術（但最後被假葉用猛力擊破），實力強大。亦曾經使用零術讓天淨沙等人陷入另一空間。曾被奪取元炁和零力共存的力量的假葉重傷及被其奪走零力，其後輾遲把其的零力歸回她且恢復零力，最後因被幽吸取了所有零力而消失。
*汰
被穹奇复活，实力强大，拥有威力强大的零煞与空间传送能力。最后在玖宫岭被破阵统领和山鬼谣联手击杀。

==== 穹奇 ====
万零之王，來自極陰世界。被上古侠岚封印在无极之渊，需九个神坠方可解除，假叶解除第一层封印后，其部分力量进入辗迟姐姐墨夷体内。於第六季最後一集死亡。
