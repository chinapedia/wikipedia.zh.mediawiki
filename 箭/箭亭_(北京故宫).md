'''箭亭'''，位于[[紫禁城|紫禁城]]东部[[景运门|景运门]]外、[[奉先殿|奉先殿]]以南的开阔平地上，是清朝皇帝及其子孙名義上练习骑马射箭之所。<ref name=jianting>[http://www.dpm.org.cn/www_oldweb/China/phoweb/BuildingPage/7/B3024.htm 箭亭，故宫博物院，于2013-10-13查阅] {{webarchive|url=https://web.archive.org/web/20150923223839/http://www.dpm.org.cn/www_oldweb/China/phoweb/BuildingPage/7/B3024.htm |date=2015-09-23 }}</ref>惟清帝及其眷屬很少待在紫禁城，實際皇子皇孫們學習騎馬、校射的地方為圓明園山高水長樓旁「西廠」或稱「西苑」。

==历史==
箭亭始建于[[清朝|清朝]][[雍正|雍正]]八年（1730年）。为防止[[满族|满族]]受到[[汉族|汉族]]同化而导致祖制湮灭，特建设箭亭，供清朝皇帝及其子孙练习骑马射箭。[[乾隆帝|乾隆帝]]和[[嘉庆帝|嘉庆帝]]均曾在箭亭射箭并且操演武艺。每逢皇帝及其子孙在此跑马射箭时，便在箭亭前摆放箭靶，列队于两侧的武士摇旗擂鼓。[[咸丰|咸丰]]元年（1851年），皇帝在箭亭阅十五善射。清朝初年规定，特许在大内骑马的王公大臣凡入[[东华门|东华门]]者，一律在箭亭前下马，所以箭亭四周的空旷之地是当年拴歇马匹之处<ref name=jianting/>。

2016年，故宫博物院新开放[[断虹桥|断虹桥]]至[[隆宗门|隆宗门]]、箭亭至[[文华殿|文华殿]]、[[故宫文物医院|西河沿区域]]这三个通道区域，故宫开放面积由此从65%增加到76%<ref>{{Cite web |url = http://news.163.com/16/0627/01/BQHI2TFO00014AED.html |title = 故宫年内开放冰库、箭亭、西河沿三大通道 |author =  |date = 2016-06-27 |publisher =网易  |language =  |accessdate =  |quote =  }}</ref>。2017年2月20日，在[[杭州|杭州]]开启铜雕五牛进故宫文化展示的起运仪式。铜雕五牛是由朱炳仁根据故宫博物院藏[[唐代|唐代]][[韩滉|韩滉]]《[[五牛图|五牛图]]》而塑造，运抵故宫后被摆放在箭亭南侧<ref>{{Cite web |url = http://news.ifeng.com/a/20170221/50715033_0.shtml |title = 唐传世名画《五牛图》立体“复活” 铜雕五牛进故宫展示 |author =  |date = 2017-02-21 |publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref>。

==建筑==
箭亭实际上是一座独立的大殿。坐北朝南，面阔五间，进深三间，黄琉璃瓦[[歇山顶|歇山顶]]，四面出廊。箭亭内的20根朱漆大柱直接承托着梁架，从而减少了[[斗拱|斗拱]]层叠的层次，这是宫殿建筑大式做法中较罕见的。箭亭东、西两侧的山墙磨砖对缝，不开窗户；南、北两面则各辟菱花槅扇门，其中南侧五扇、北侧三扇，共计八扇。箭亭中央设有宝座，宝座东侧有卧碣1座，刊刻着[[乾隆|乾隆]]十七年（1752年）上谕，内容是要求满清贵族“衣服语言，悉遵旧制”，“操演技勇，时时练习骑射”，并且告诫子孙“永垂法守”。箭亭前为宽敞开阔的平地，是[[殿试|殿试]][[武进士|武进士]]阅技勇之处，届时需要考试马步、射弓、刀石。[[嘉庆|嘉庆]]十三年（1808年），[[清仁宗|清仁宗]]又在箭亭内的西侧立石碑，再度告诫子孙“不效汉俗”。如今，箭亭建筑保存完好。<ref name=jianting/>
