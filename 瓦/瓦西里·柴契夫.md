'''瓦希里·格里高葉維奇·柴契夫'''[[上尉|上尉]]（'''{{lang-ru|Васи́лий Григо́рьевич За́йцев}}'''，{{bd|1915年|3月23日|1991年|12月15日}}），[[二戰|二戰]]時期[[蘇聯|蘇聯]]陸軍著名狙擊手，他在1942年[[史達林格勒戰役|史達林格勒戰役]]中，於11月10日至12月17日之間共擊杀225名[[德意志國防軍|德意志國防軍]]和其他[[軸心國|軸心國]]的士兵與軍官（包括11名[[狙擊手|狙擊手]]），因此一戰成名。

据悉，截至11月10日之前，瓦西里以傳統的[[莫辛-納甘步槍|莫辛-納甘步槍]]共擊殺32名[[軸心國|軸心國]]士兵；在1942年10月到1943年1月之間，瓦西里總計擊殺大約242名敵軍。不過真實數目可能更高，部分學者認為甚至可能高達400人左右。

== 生平 ==
瓦西里出生於耶勒寧斯科耶（又稱為亞列寧斯科亞），於[[烏拉山脈|烏拉山脈]]長大。瓦西里的姓氏「柴契夫」這個名字在[[俄語|俄語]]與"[[野兔|野兔]]"有相近的意思（同字根）。

在前往[[史達林格勒|史達林格勒]]前，瓦西里服役於[[蘇聯海軍|蘇聯海軍]]，擔任岸勤人員，直到他從報章雜誌得知[[史達林格勒戰役|史達林格勒戰役]]的慘烈後，自願加入這一場有史以來最殘酷的會戰。

瓦西里被分派到蘇聯陸軍第62軍第284步兵師第1047團。在某次戰鬥中，他的上级将一個800米外窗户中的一名德军作為目標，他用普通纳甘步枪一枪击杀，包括之後出來查看的德軍都被他一舉射杀，因此表现出狙击手天赋的他獲得了英勇勋章和一支狙击型纳甘步槍。<ref>http://russiapedia.rt.com/prominent-russians/military/vasily-zaitsev/</ref>他的战绩宣传给当时在斯大林格勒陷入苦战、缺衣少食的苏军带来了很大鼓舞。<ref>http://www.warheroes.ru/hero/hero.asp?Hero_id=481</ref>

由於瓦西里在史達林格勒戰役中成名，所以苏军在史達林格勒的拉祖爾化學工廠成立狙擊學校，並由他親自負責訓練。

由於扎伊采夫的姓氏涵義的關係，因此所開辦的狙擊手學校訓練出的狙击手又被暱稱為「小兔兔」（"Zaichata"）。

在{{tsl|en|Antony Beevor|安東尼·畢佛}}（前英國第11輕騎兵中隊軍官，現任歷史學家）所著的《史達林格勒》中，認為在拉祖爾化工廠所開辦的學校正是第62軍以及全蘇軍发起大规模“狙擊手運動”的结果，部隊之間開始舉辦訓練營並且宣揚狙擊法則，以批量的方式進行大規模集團訓練狙击手，士兵們熱烈地彼此互相交換關於狙擊技巧與戰術的想法以及原則等。

據估計，老柴兔瓦西里一手训练出来的小柴兔子們，一共击毙击伤了超過三千個軸心國士兵與軍官。这只是苏军在战场上发起大规模狙击运动的缩影，由于对于狙击手的高度重视，苏军在战后制订了将狙击手分配给步兵班的战斗条令，并研制专门的[[SVD狙击步枪|SVD狙击步枪]]。

1943年1月，瓦西里的眼睛因為地雷炸傷而离开前线，之後他由{{tsl|en|ladimir Filatov|維拉米爾·費拉托夫}}医生照顧，並且治療他的眼睛。瓦西里後重返戰場並參加了[[德涅斯特河|德涅斯特河]]戰役，當時的他已經是[[上尉|上尉]]了。

戰後，瓦西里曾造訪[[柏林|柏林]]，並與軍中同袍們相見。同袍們贈送一把新的狙擊槍給他，上面刻著：「敬蘇聯英雄瓦西里·扎伊采夫，在[[史達林格勒|史達林格勒]]殺死了超過300個[[法西斯|法西斯]]份子。」（目前這把槍保存在[[伏尔加格勒|伏尔加格勒]]博物館中的[[史達林格勒戰役|史達林格勒戰役]]區）。之後瓦西里便在[[基輔|基輔]]經營工廠，1991年在基辅去世，享年76歲。

== 身後 ==
2006年1月31日，瓦西里·柴契夫的遺體以隆重的軍禮被重新遷葬到馬馬耶夫崗 (''Mamayev Kurgan'')。由於瓦西里的遺願是希望將他的遺體埋在[[史達林格勒|史達林格勒]]戰役紀念碑下，所以他的的棺木被埋葬在紀念碑旁；碑上則寫著瓦西里的名言：「我們沒有任何撤退的餘地。」（事實上該語出自[[赫鲁晓夫|赫鲁晓夫]]，当时擔任史達林格勒戰役的[[政委|總政委]]。）

== 傳說 ==
瓦西里的自傳《Notes of a Sniper》中提到他在史達林格勒戰役中其實遭遇的對手應該是「[[海恩茲·托爾伐特|海恩茲·托爾伐特]]」，瓦西里是依據從被擊斃的屍首上所搜出的身分證明文件而得知。

他收走了对方的狙击枪瞄准镜作纪念。這個名字後來被作家「{{tsl|en|David L. Robbins (Virginia writer)|大衛·羅賓斯}}」用在1999年出版的小說《老鼠之戰》（War of the Rats）中使用。羅賓斯稱托爾伐特是一位[[黨衛隊|黨衛隊]]的上校，而且還根本就是位於德國[[佐森|佐森]]（Zossen）[[德意志國防軍|國防軍]]狙擊學校的總教官。

歷史學家曾經不斷質疑這個似乎虛構的高手是否真的存在,不過下面的幾個理由臆測出歷史上從無此人過：
# 沒有紀錄能指出黨衛軍曾經參加過史達林格勒會戰。
# 黨衛軍本身的狙擊手人數屈指可數，黨衛軍戰功並不將狙擊視為重要項目。
# 不會有哪個狙擊手會官晉[[上校|上校]]（[[親衛隊旗隊領袖|Standartenführer]]），狙擊手並沒有強大到能影響戰局，而且也是高傷亡的職位，基本上全都是士兵，最多是士官。
# 即便到了戰爭末期，有明文記載德軍的头号狙击手也不過是個21歲的二等兵，[[馬豪斯·海茨瑙亞|馬豪斯·海茨瑙亞]]（Matthäus Hetzenauer），一共345人命喪他的槍下。 
# 史達林格勒戰役為前蘇聯抵抗當時德軍的進攻畫下了時代的轉捩點，在異常艱苦的狀況下，前蘇聯的政宣文工為了保持龐大部隊的戰鬥精神，而會捏造充斥著一堆謊言來增進部隊士氣，尤其當時戰況在如此不明朗也不樂觀的情形下。
