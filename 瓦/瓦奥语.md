'''瓦奥语'''，通常被称为 '''Sabela''' (别称还有：''Wao，Huao，Auishiri，Aushiri，Ssabela'' ; 自称: '''Wao Terero'''；[[贬称|贬称]]：''Auka，Auca'')是生活在[[厄瓜多尔|厄瓜多尔]][[亚马逊雨林|亚马逊雨林]]的Napo河与Curaray河之间的原住民[[瓦奥人|瓦奥人]]使用的语言。该语言是一门[[孤立语言|孤立语言]]。在秘鲁，有一小群说瓦奥语的[[未接触部落|未接触部落]]。

== 音系 ==
瓦奥语区分[[鼻元音|鼻元音]]和口元音。音节结构为(C)V，[[双元音|元音丛]]比较密集。

== 方言 ==
瓦奥语有三个方言： ''Tiguacuna(Tiwakuna)方言、Tuei(Tiwi Tuei, Tiwi)方言,'' 和 ''Shiripuno方言。''

== 系属关系 ==
Sabela何任何已知语言都没有系属关系。 但是， Terrence Kaufman认为它属于 Yawan 语族。
