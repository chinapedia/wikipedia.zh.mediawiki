'''冒廣生'''（{{bd|1873年||1959年|8月10日|catIdx=M}}），字'''鹤亭'''，号'''疚斋'''，祖籍[[江苏|江苏]][[如皋|如皋]]，生於[[广东|广东]][[广州|广州]]。[[清朝|清朝]]及[[中华民国|中华民国]]、[[中华人民共和国|中华人民共和国]]文化名人、诗人、官员、收藏家。<ref name=sht>{{Cite web |url = http://www.shtong.gov.cn/newsite/node2/node4429/node4432/node70711/node70715/node70723/userobject1ai54298.html |title = 冒广生 |author =  |date =  |publisher = 上海通 |language =  |accessdate = 2017-01-14 |quote =  }}</ref>

==生平==
冒廣生因祖辈宦游流寓广州而出生于廣州，故得名“廣生”。幼时深受其七外祖[[周星誉|周星誉]]（周畇叔）喜爱，被周星誉亲切地称作“阿灵”（乳名）。<ref name=rugao/>冒廣生是[[冒襄|冒襄]]（字辟疆，号巢民）第六代侄孙，父亲是冒树楷（字子端，1878年逝世），母亲是[[周星詒|周星詒]]之女周萱（1921年逝世）。冒广生六岁丧父，由母亲养大。《[[清史稿|清史稿]]·列女传》中有其母的介绍：<ref name=sht/><ref name=ou>{{Cite web |url =http://www.wzrb.com.cn/article470446show.html  |title = 冒广生的温州旧照 |author =  |date = 2013-05-17 |publisher = 瓯网 |language =  |accessdate =  |quote =  }}</ref>

{{引用|冒树楷妻周，树楷，如皋人；周，祥符人。树楷以知县待缺[[福建|福建]]，早卒。周挈子女从舅广州，舅亦卒。侨居，日食率百钱，翼子女以长。子得官，将请旌，周拒之曰：“妇节常耳，人子於其母，奈何欲假以为名哉？” 父星诒，诸父星譼、星詧，并有文行，周刻其遗著，为父营葬，置墓田焉。}}

其父早逝後，冒廣生随母往依广州的祖父生活。清朝[[光绪|光绪]]十二年（1886年），祖父病逝，母亲带着冒廣生迁居[[苏州|苏州]]周星诒处同住。光绪二十年（1894年），冒廣生参加江南[[乡试|乡试]]，中甲午科[[举人|举人]]，时任副主考的[[黄绍第|黄绍第]]（[[温州|温州]][[瑞安|瑞安]]人）将长女黄曾葵许配给他。<ref name=ou/><ref name=rugao/>光绪二十一年（1895年）赴[[北京|北京]][[会试|会试]]，参加了“[[公车上书|公车上书]]”。清朝末年光绪、[[宣统|宣统]]两朝，历任[[刑部|刑部]]郎中、[[农工商部|农工商部]]郎中。<ref name=sht/><ref name=rugao>{{Cite web |url = http://news.cpcrugao.cn/2015/1117/97309.shtml |title = 《龙游河棹歌补笺》前言 |author =冒榕龄  |date = 2015-11-17 |publisher =中共如皋市委新闻网  |language =  |accessdate =  |quote =  }}</ref>

光绪二十四年（1898年）春，冒廣生赴北京试于[[礼部|礼部]]，未录。此后在北京逗留。同年列名[[保国会|保国会]]。<ref name=nianpu/>[[梁启超|梁启超]]称其“英姿飒爽，气咄咄如朝日。问姓氏，审邑居，辄忆其先德巢民先生……鹤亭之文，史家之文也”（见《冒巢民先生年谱》）。<ref name=rugao/>同年秋，由[[茂名|茂名]][[杨颐|杨颐]]（号蓉浦）以经济特科保送，后因[[戊戌政变|戊戌政变]]，科试被废，杨颐“亦旋归道山”。同年冒廣生回如皋。光绪二十五年（1899年）至二十六年（1900年）多次往来北京、苏州、[[扬州|扬州]]、如皋。光绪二十七年（1901年），纳貲为[[刑部|刑部]]郎中，签分刑部陕西司行走，派陕西司主稿管[[新疆|新疆]]股。同年南归。光绪二十八年（1902年）春，冒廣生携夫人黄曾葵及长子冒景玮到北京，寓居[[宣武门外大街|宣武门外大街]]南通会馆，随后迁居[[后孙公园胡同|后孙公园]]如泰会馆，冒廣生继续任刑部郎中。光绪二十八年（1902年）春，应顺天府尹[[陈璧|陈璧]]的聘请，兼任五城学堂（位于北京[[琉璃厂|琉璃厂]]，今[[北京师范大学附属中学|北京师范大学附属中学]]的前身）教员，同事有[[林纾|林纾]]。光绪二十九年（1903年），留任[[农工商部|商部]]郎中。同年秋，应试经济特科，初考中商部第四名，举主是[[钱塘|钱塘]][[吴士鉴|吴士鉴]]（字綗齋），列为一等，但覆试中因在策论引用[[法国|法国]][[卢梭|卢梭]]的《[[社会契约论|民权论]]》，被首席读卷大臣[[张之洞|张之洞]]阅後批：“论称引卢梭，奈何？”遂被摈弃。特科二百余卷中，被张之洞所批的仅有冒廣生的试卷。光绪三十年（1904年）冬，补商部平均司郎中。光绪三十二年（1906年），商部改为[[农工商部|农工商部]]，冒廣生留任为农工商部郎中。光绪三十四年（1908年）派充普岮峪定[[清东陵|东陵]]神路营房工程处监修官。同年底[[慈禧太后|慈禧太后]]病逝，冒廣生随办慈禧太后、[[光绪帝|光绪帝]]丧礼。宣统三年（1911年），授京察覆带，以道府记名。宣统三年十二月（1912年2月）[[清帝退位|清室逊位]]，冒廣生弃官，和母亲及家人一起避往[[天津|天津]]暂居。<ref name=nianpu/>

民国元年（1912年）农历十一月回北京，任[[中华民国临时政府_(袁世凯)|北京政府]]财政部顾问。同年农历十二月，冒廣生经[[袁世凯|袁世凯]]之子[[袁克定|袁克定]]推荐，任[[浙江省|浙江省]][[瓯海关|瓯海关]]监督兼外交部[[温州|温州]]交涉员。冒廣生临行之时誓言：“祖基之前，此生此官与母周太夫人为进退。”民国二年（1913年）春，即奉母赴温州。<ref name=nianpu/><ref name=ou/>民国四年（1915年），事母孝顺的冒廣生效仿《[[二十四孝|二十四孝]]》中[[老莱子|老莱子]]“戏彩娱亲”故事，在瓯海关署西首兴建了一座小楼，名为“后戏彩堂”，并在小楼对面兴建了一座戏台，聘戏班前来为母亲演出。<ref name=ou/>在温州任内，冒廣生广结温州文人名士，筑瓯隐园（今为[[墨池公园|墨池公园]]），建永嘉诗人祠堂，编《永嘉诗人祠堂丛刻》，搜罗自[[谢灵运|谢灵运]]以来迄至[[黄绍箕|黄绍箕]]的集子。<ref name=wenzhou/>

民国七年（1918年），在北京被[[北洋政府|北京政府]]农商总长[[田文烈|田文烈]]聘为全国经济调查会会长，任期不到一年。任内将留学归来的[[丁文江|丁文江]]、[[翁文灏|翁文灏]]分别任命为第一、二科科长，从而创始了全国资源调查。民国八年（1919年），接任[[镇江关|镇江关]]监督，关署在金山脚下。任内刊刻《至顺镇江志》。民国九年（1920年），调任[[淮安关|淮安关]]监督。民国十一年（1922年），以[[丁忧|丁忧]]的名义弃官（民国成立后便已废除了清朝丁忧之制）。<ref name=wenzhou/>

民国十九年（1930年），经[[胡汉民|胡汉民]]推荐给[[南京国民政府|南京国民政府]][[考试院|考试院]]的[[戴季陶|戴季陶]]、[[邵元冲|邵元冲]]，邵元冲时任考选委员会委员长，乃任命冒廣生为考选委员。民国二十年（1931年），考试院派冒廣生兼任高等考试典试委员（高等考试典试委员会主任为[[王用宾|王用宾]]）。<ref name=wenzhou/>

1934年，[[邹鲁|邹鲁]]接任广州[[国立中山大学|国立中山大学]]校长，商聘冒廣生为词学教授。同年，应[[陆嗣曾|陆嗣曾]]（陆光宇）之聘，兼任[[广东省立勷勤大学|广东省立勷勤大学]]客座教授。同年，获[[陈融|陈融]]（陈协之）推荐兼任广东通志馆纂修（当时该馆主任兼总纂为[[温廷敬|温廷敬]]）。1935年，应陆嗣曾邀请，改定广东省立勷勤大学校歌。[[抗日战争|抗日战争]]期间，1940年应[[汤国梨|汤国梨]]（[[章太炎|章太炎]]夫人）聘为太炎文学院词曲教授。1945年，[[国史馆|国史馆]]馆长[[张继_(中华民国)|张继]]函聘冒廣生为国史馆纂修。<ref name=wenzhou/>

[[中華人民共和國|中華人民共和國]]成立後，1950年6月，冒廣生因家中生活困窘，写信给旧诗友、[[中南军政委员会|中南军政委员会]]副主席[[程潜|程潜]]。程潜回信给冒廣生，并附一封致[[上海市市长|上海市市长]][[陈毅|陈毅]]的信。7月19日，[[华东军政委员会|华东军政委员会]]人民监察委员会委员[[江庸|江庸]]来看望冒廣生，廣生将程潜的信托他交陈毅。当天，陈毅和江庸来冒廣生寓所拜访。8月7日，上海市文物保管委员会送聘书至冒廣生寓所，聘请冒鹤亭为特约顾问，月薪三百二十元，无庸办事。<ref name=pp>{{Cite web |url = http://www.thepaper.cn/newsDetail_forward_1593264 |title =上海书评 冒鹤亭与陈毅、周恩来及毛泽东的交往  |author = 冒佳骐 |date =  2017-01-14|publisher = 澎湃新闻 |language =  |accessdate =  |quote =  }}</ref>后冒廣生刊刻《楚州丛书》。<ref name=wenzhou/>

1957年初，冒廣生的长孙冒怀辛考上北京的研究生。85岁的冒廣生亲自陪同33岁的冒怀辛赴北京。1957年2月抵达北京，冒廣生住在[[宣武门|宣武门]]外[[储库营胡同|储库营胡同]][[中国人民银行|中国人民银行]]总行宿舍大院内的五子[[冒舒諲|冒舒諲]]家里（冒舒諲时任中国人民银行专门委员）。来北京三个月后，冒廣生致信国务院副总理陈毅，告知自己已来北京。几天后，陈毅邀请冒廣生参观[[故宫博物院|故宫博物院]]，并在御花园座谈，同座者有[[湖南省人民政府|湖南省人民政府]]主席程潜。御花园坐谈之后，冒廣生应陈毅之请，写出《对目前整风的一点意见》一文，发表在1957年6月6日《[[人民日报|人民日报]]》上。随后《人民日报》记者[[傅冬|傅冬]]（[[傅作义|傅作义]]之女）来冒廣生住所采访，后写成《八五老人一席话——访冒廣生老先生》一文，发表在1957年6月12日《人民日报》上。6月28日，[[周恩来|周恩来]]来冒廣生住所探访，交谈2小时，并带来[[毛泽东|毛泽东]]的口信称，毛泽东看到冒廣生在《人民日报》上的文章，想见面谈谈。两天后的6月30日，冒廣生由冒舒諲陪同进[[中南海|中南海]]，与毛泽东在游泳池畔见面交谈。冒廣生将其手稿本《疚斋词话》、《宋曲章句》、《四声钩沉》等赠给毛泽东。毛泽东握着冒廣生的手说：“我过几天要到外地去。希望你老明年再来北京。”2013年[[中共中央文献研究室|中共中央文献研究室]]编写的《毛泽东年谱（1949—1976）》出版，其中记载毛泽东在1957年6月30日的活动是：“……晚八时二十分，同[[胡乔木|胡乔木]]、[[李维汉|李维汉]]、[[吴冷西|吴冷西]]谈话。十时二十分，在中南海游泳池住处召集[[刘少奇|刘少奇]]、周恩来、[[朱德|朱德]]、[[陈云|陈云]]、[[邓小平|邓小平]]、[[彭真|彭真]]、胡乔木、李维汉、吴冷西开会。晚九时二十分，会见冒鹤亭。”<ref name=pp/>此后[[中共中央|中共中央]]很快发动[[反右运动|反右运动]]，冒廣生有四个子孙被打成“右派分子”（包括儿子冒舒諲）。<ref name=nianpu/><ref>冒廣生本人并非右派（参见冒怀苏《冒鹤亭先生年谱》。当时定右派分子是由各单位确定，冒廣生不属于任何单位，根本不在其列。自1957年至1959年，冒廣生一直受到优待。）</ref>

1957年8月冒廣生与[[周谷城|周谷城]]同车返上海，[[叶恭绰|叶恭绰]]、[[文怀沙|文怀沙]]等人送行。返回上海后不久，1958年冒廣生即病倒。1959年8月10日，冒廣生在上海病逝，享年86岁。冒廣生的追悼会由[[上海市文史研究馆|上海市文史研究馆]]馆长[[江庸|江庸]]主祭，[[姚虞琴|姚虞琴]]、[[王福厂|王福厂]]、[[吴湖帆|吴湖帆]]等人陪祭，逾三百人参加，非常隆重。陈毅电嘱上海市政府送大花圈，[[中共上海市委|中共上海市委]]统战部也送了大花圈。冒廣生葬于苏州[[灵岩山|灵岩山]]五龙公墓，其墓在[[文革|文革]]时被毁。文革结束后，1993年其后人在[[北京植物园|北京植物园]]櫻桃溝建[[衣冠冢|衣冠冢]]。衣冠冢的墓碑正面是[[赵朴初|赵朴初]]题写的“如皋冒鹤亭先生之墓”，背面刻有碑文：<ref name=pp/>

冒廣生病逝後，其家屬遵其遗愿，将珍藏300余年的先祖遗存文物900多件，于1961年全部捐给[[上海博物馆|上海博物馆]]。1989年11月，为纪念冒廣生逝世30周年，上海博物馆举办《冒广生先生遗存文物展览》。冒广生收藏的文物，主要是祖上遗存（包括书画、印章、砚台），以及师友手书信札。如明[[李痴和|李痴和]]画冒起宗（冒襄之父）肖像轴、清人画冒襄肖像轴、清[[吴历|吴历]]赠冒襄行书八言联、[[戴本孝|戴本孝]]为冒襄篆刻的六面印石章以及《水绘园图》等等。<ref name=sht/>
