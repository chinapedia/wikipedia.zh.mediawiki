'''荒木宗太郎'''（{{bd|？||1636年|12月4日}}<ref name="荒木">{{Cite web |title=荒木宗太郎|url=https://kotobank.jp/word/%E8%8D%92%E6%9C%A8%E5%AE%97%E5%A4%AA%E9%83%8E-27708}}</ref>），名'''一清'''，通稱'''惣右衛門'''，[[安土桃山時代|安土桃山時代]]後期至[[江戶時代|江戶時代]]商人，原是[[肥後國|肥後國]][[熊本縣|熊本]][[武士|武士]]，本姓[[藤原氏|藤原氏]]，後移居[[長崎縣|長崎]]，從事[[朱印船|朱印船]]貿易，活躍於[[大越|大越]][[廣南國|廣南國]]{{noteTag|日本文書中稱為「[[安南|安南]]」<ref name="荒木" />}}（今[[越南|越南]]中部一帶）及[[北河國|北河國]]{{noteTag|日本文書中稱為「[[交趾|交趾]]」<ref name="荒木" />}}（今越南北部）、[[暹羅|暹羅]]（今[[泰國|泰國]]）、[[呂宋|呂宋]]等地，並娶廣南國君主[[阮福源|阮福源]]之[[養女|養女]][[阮氏玉華|阮氏玉華]]為妻，並獲賜[[越南人名|越南名]]'''阮太郎'''（{{vie|v=Nguyễn Thái Lang|hn=阮太郎}}{{noteTag|一作「阮大良」<ref name="南河">{{cite book |author=蔣國學 |title=越南南河阮氏政權海外貿易研究|publisher= 世界圖書出版社 |year= 1999 |isbn= 9787510028892 }}</ref><ref name="Ngọc Hoa">{{Cite web |title=CÔNG NỮ NGỌC HOA, NÀNG LÀ AI?|url=http://www.erct.com/2-ThoVan/3-Unna/CongnuNgocHoa.htm}}</ref>}}）<ref name="Ngọc Hoa" /><ref name="肥後">{{Cite web |title=朱印船貿易と肥後|url=http://www.city.tamana.lg.jp/q/aview/472/1587.html}}</ref><ref name="王女">{{Cite web |title=荒木宗太郎と安南国王女|url=http://www.mekong.ne.jp/linkage/arakisoutaro.htm}}</ref><ref name="アーニオ姫">{{Cite web |title=荒木宗太郎とアーニオ姫|url=http://dream-nagasaki.jp/vietnam/anio-/anio-top.htm}}</ref><ref name="Hội An">{{Cite web|title=Đặt tên đường Công nữ Ngọc Hoa ở Hội An|url=http://www.thanhnien.com.vn/van-hoa-nghe-thuat/dat-ten-duong-cong-nu-ngoc-hoa-o-hoi-an-320539.html}}{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref><ref name="人物">{{Cite web |title=荒木　宗太郎　 （？～１６３６年）|url=http://www.mekong.ne.jp/database/special/sengokugosho/arakisoutaro/}}</ref><ref name="荒木" /><ref name="墓地">{{Cite web |title=荒木宗太郎墓地|url=http://www.city.nagasaki.lg.jp/shimin/190001/192001/p000691.html}}</ref>。
==生平==
荒木宗太郎早年事跡不詳，出身於肥後國熊本，本是武士。在[[戰國時代_(日本)|戰國時代]]末期，熊本{{link-ja|高瀨町 (熊本縣)|高瀬町 (熊本県)|高瀨町}}的商人以[[荒木氏|荒木氏]]為主，荒木宗太郎亦可能出身於高瀨町<ref name="肥後" />。[[天正|天正]]十六年（1588年）移居長崎，在浦上淵村稲佐鄉[[飽之浦|飽之浦]]（{{lang-ja|飽の浦}}，今[[長崎市|長崎市]][[飽之浦町|飽之浦町]]）興建大屋居住，成為商人。[[文祿|文祿]]元年，包括荒木宗太郎在內的八位居於長崎、[[京都府|京都]]、[[堺|堺]]的商人獲[[豐臣秀吉|豐臣秀吉]]授予[[朱印狀|朱印狀]]，特許進行[[國際貿易|海外貿易]]，自此開始朱印船貿易。[[德川家康|德川家康]]建立[[江戶幕府|江戶幕府]]後上沿襲豐臣秀吉的政策，繼續向商人授予朱印狀作為特許海外貿易的憑證，荒木宗太郎也獲幕府頒發朱印狀，繼續從事朱印船貿易<ref name="荒木" /><ref name="王女" /><ref name="人物" /><ref name="墓地" />。
[[File:ArakiRedSealShip.jpg|thumb]]
與大多數派遣[[親屬|親屬]]和僱用船員出航的朱印船主不同，荒木宗太郎不但出資，還親自揚帆出海，因而受到[[水手|水手]]尊敬，稱他為「直乘船頭」（直乗り船頭）。他的船被稱為[[荒木船|荒木船]]，所用的[[船旗|船旗]]圖案是把[[荷蘭|荷蘭]][[東印度公司|東印度公司]]的船旗上下倒轉而成。在[[慶長|慶長]]十一年（1606年）至[[寬永|寬永]]九年（1632年）間曾駕朱印船出海六次，於慶長十一年到暹羅，四年後到廣南國和北河國，元和五年（1619年）至寬永九年間曾來往北河國數次<ref name="王女" /><ref name="荒木" />。當時的越南為[[黎中興朝|黎中興朝]]時期，北部的北河國為[[鄭主|鄭主]]統治，中部的廣南國則由[[阮主|阮主]]所控制，[[後黎朝|後黎朝]]皇帝權力被架空。元和五年（越南[[後黎朝|後黎朝]][[弘定|弘定]]二十年），荒木宗太郎到廣南國[[會安|會安]]（今越南[[會安市|會安市]]），獲得廣南國君主[[阮福源|阮福源]]信任，並把養女[[阮氏玉華|阮氏玉華]]嫁給他，賜名'''阮太郎'''，又加封他為廣南貴族<ref name="南河" />。翌年，他攜同妻子回國<ref name="王女" /><ref name="Ngọc Hoa" /><ref name="Hội An" /><ref name="人物" /><ref name="荒木" />。

元和八年（1622年，後黎朝[[永祚|永祚]]四年），荒木宗太郎獲江戶幕府第二代[[征夷大將軍|將軍]][[德川秀忠|德川秀忠]]頒發新的朱印狀，再次容許他到廣南國通商。翌年他出航至妻子的祖國廣南國，但他並沒有帶妻子同行。其妻一直留在日本，改名[[阮氏玉華|王加久]]，當地人又稱她為「{{lang|ja|アニオさん}}」，與他育有一女[[荒木家須|家須]]<ref name="王女" /><ref name="Hội An" />。[[日本鎖國|日本鎖國]]後，他與妻女一同居住在長崎。寬永十三年（1636年）農曆十一月初七，荒木宗太郎於長崎的房子裡逝世，葬於長崎{{link-ja|大音寺|大音寺}}後山墓地<ref name="人物" /><ref name="荒木" />。

[[元祿|元祿]]三年以後，荒木家第三代伊太郎好信至第十代惣八郎春章皆擔任長崎西築町的[[乙名|乙名]]（{{lang|ja|おとな}}）職務<ref name="荒木" /><ref name="墓地" />。

==相關史跡、文物及紀念==
[[File:NagasakiSTA_hune.JPG|thumb]]擺放[[三菱重工|三菱重工]]長崎造船所製作、以荒木船為藍本的朱印船模型]]
現時在[[長崎市|長崎市]]飽之浦公園內有荒木宗太郎住宅遺跡，位於市內鍛冶屋町大音寺後山的[[荒木宗太郎墓地|荒木宗太郎墓地]]為{{link-ja|長崎市指定文化財列表|長崎市指定文化財一覧|長崎市指定史跡}}<ref name="墓地" />。每年在[[鎮西大社諏訪神社|諏訪神社]]舉行的{{link-ja|長崎供日|長崎くんち}}的奉納踊中都有「{{lang|ja|アニオーさんの行列}}」，以荒木宗太郎和阮氏玉華的盛大[[婚禮|婚禮]]為題材，由演員扮演荒木宗太郎夫婦站在[[道具|道具]]朱印船上<ref name="人物" />。而市內尾上町的鐵路車站[[長崎站|長崎站]]擺放了[[三菱重工|三菱重工]]長崎造船所製作、以荒木船為藍本的朱印船模型<ref name="墓地" />。於[[長崎市立博物館|長崎市立博物館]]則藏有以荒木船為題材的《荒木宗太郎異國渡海船之圖》<ref name="人物" />。
