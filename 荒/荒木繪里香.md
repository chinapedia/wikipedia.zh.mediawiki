'''荒木繪里香'''（{{bd|1984年|8月4日}}），出生於[[日本|日本]][[岡山縣|岡山縣]][[倉敷市|倉敷市]]，[[日本|日本]][[排球|排球]][[運動員|運動員]]，身高186公分，於2004年入选[[日本國家女子排球隊|日本國家女子排球隊]]。於2006年[[世界排球锦标赛|世界排球锦标赛]]及[[2008年夏季奧林匹克運動會|2008年夏季奧林匹克運動會]]皆有參賽。於[[2008年9月|2008年9月]]，荒木被[[東麗箭_(女子排球)|東麗箭]]派到[[意大利|意大利]]Volley Bergamo女子排球隊成為交流選手，是繼[[高橋美雪|高橋美雪]]後第二位到意大利的日本外援選手。

== 來歷 ==
她的父親是前[[早稻田大學|早稻田大學]][[橄欖球|橄欖球]]隊的隊員，母親則為體育老師，雙親都和體育方面有關，因此小時候就同時參加了[[游泳|游泳]]與[[田徑|田徑]]的俱樂部，在此領域也表現突出。在小學五年級時，開始接觸排球運動，到了中學時期已經成為「水瓶座盃」比賽的選拔成員之一。之後高校就讀成德學園高校（今下北澤成德高校），與[[大山加奈|大山加奈]]等人奪得[[全國高等學校排球選拔優勝大會|全國高等學校排球選拔優勝大會]]（春高排球）、[[全國高等學校綜合體育大會|全國高等學校綜合體育大會]]以及[[國民體育大會|國民體育大會]]的冠軍。

2003年，加入[[東麗箭_(女子排球)|東麗箭]]。在入隊的首年，即以全聯盟攻擊成功率第二位的成績，進入最佳陣容。同年入選[[日本國家女子排球隊|日本國家隊]]，前往[[歐洲|歐洲]]進行比賽，不過並未進入[[2004年夏季奧林匹克運動會|2004年奧運會]]的國家隊名單之中。

2005年，再次入選國家隊，場上位置以副攻為主。2006年的[[2006年世界女排大獎賽|世界女排大獎賽]]，首戰面對[[古巴國家女子排球隊|古巴]]時被列於先發陣容，以攻擊成功率90%的成績，進入了隊上固定班底，並在同年舉行的[[2006年世界女子排球錦標賽|世界排球錦標賽]]出場。2007年，參與[[2007年亞洲女子排球錦標賽|亞洲排球錦標賽]]貢獻良多，最終使日本獲得相隔24年的冠軍獎盃。在[[2007年世界盃女子排球賽|世界盃排球賽]]負責副攻的位置，全部比賽皆出場。

2008年，於場上表現活躍，幫助東麗箭奪下[[天皇盃及皇后盃全日本排球選手權大會|天皇盃及皇后盃]]、2007/08賽季[[V超級聯賽|V超級聯賽]]冠軍，個人也包辦了最高榮譽選手獎、扣殺獎、攔網獎等六項最佳獎項，其中扣殺成功率54.7%，也打破1999/00賽季[[先野久美子|先野久美子]]創下的日本本土選手53.38%的最高紀錄。

2008年8月，前往[[北京|北京]]參加[[2008年夏季奧林匹克運動會女子排球比賽|2008年奧運會]]，得到最佳副攻獎。

2008年9月，被租借至義大利的職業球隊Volley Bergamo一年，共從9月效力到隔年5月<ref>{{cite web |url=http://www.sanspo.com/sports/news/080723/spk0807232142002-n1.htm |title=東レ・荒木絵里香が伊へ移籍 |accessdate=2008-07-23 |deadurl=yes |archiveurl=https://web.archive.org/web/20080724134949/http://www.sanspo.com/sports/news/080723/spk0807232142002-n1.htm |archivedate=2008-07-24 }} - 產經體育（2008年7月23日） {{ja icon}}</ref><ref>{{cite web |url=http://www.toray.co.jp/news/event/nr080724.html |title=東レアローズ・荒木絵里香選手のイタリア・セリエAへのレンタル移籍について |accessdate=2010-08-29 |deadurl=yes |archiveurl=https://web.archive.org/web/20080725072140/http://www.toray.co.jp/news/event/nr080724.html |archivedate=2008-07-25 }} - 東麗箭公式網站（2008年7月24日） {{ja icon}}</ref><ref>{{cite web |url=http://www.vivovolley.it/modules/news/article.php?storyid=2720 |title=ベルガモ移籍を報じるイタリア記事 |accessdate=2008-07-25 |deadurl=yes |archiveurl=https://web.archive.org/web/20080828035058/http://www.vivovolley.it/modules/news/article.php?storyid=2720 |archivedate=2008-08-28 }} {{it icon}}</ref>。2009年10月，正式復歸母隊東麗箭。2009/10該賽季，不僅幫助球隊完成歷史上首次聯賽三連霸，也再度獲得扣殺獎並入選最佳陣容，另外同時也對2010年[[日韓聯賽頂級賽|日韓聯賽頂級賽]]、[[黑鷲旗全日本男女選拔排球大會|全日本男女選拔排球大會]]的冠軍作出貢獻。

2009年5月，成為日本國家隊的隊長<ref>直至2013年5月，由[[木村沙織|木村沙織]]接替（[http://www.jva.or.jp/news/20130513.html 「世界一を知る」 火の鳥NIPPONが2013年シーズンに向けて始動 新キャプテンは木村沙織選手！]）。原先決定2012/13賽季結束即退休，後經過教練[[真鍋政義|真鍋政義]]的勸導下改變心意。（[[朝日新聞|朝日新聞]] 2013年5月14日朝刊14版 24面）</ref>。2010年11月，參加由[[日本|日本]]為主辦國的[[2010年世界女子排球錦標賽|世界排球錦標賽]]，最終擊敗[[美國國家女子排球隊|美國]]獲得季軍。

2011年6月，參與蒙特勒排球大賽，協助日本奪得首座賽會冠軍。之後，成為國家隊中的核心，在其他球員紛紛受傷的情形下，穩定佔有自身的位置。同年11月的[[2011年世界盃女子排球賽|2011年世界盃女子排球賽]]得到殿軍，俱樂部方面則獲得2011年度[[天皇盃及皇后盃全日本排球選手權大會|天皇盃及皇后盃]]以及2011/12賽季V超級聯賽的冠軍，個人並榮獲相隔四年以來的第二個MVP<ref>{{Cite web|date=2012-03-25|url=http://www.sponichi.co.jp/sports/news/2012/03/25/kiji/K20120325002900020.html|title=東レ2季ぶり3度目V 荒木絵里香がMVP|publisher=[[體育日本|體育日本]]|accessdate=2012-08-13|language=ja}}</ref>。此外，還獲頒攔網獎、最佳陣容。

2012年6月，被選入[[2012年夏季奧林匹克運動會女子排球比賽|2012年奧運會]]代表隊，連續兩屆奧運會皆出場<ref>{{cite web|url=http://www.jva.or.jp/news/20120625.html|title=女子日本代表選手12名決定! 第30回オリンピック競技大会(2012/ロンドン)|author=[[日本排球協會|日本排球協會]]|accessdate=2012-06-25|language=ja}}</ref>。作為隊長的身分，帶領著球隊成功獲得銅牌，這也是睽違28年日本國家隊在奧運會上再次獲頒獎牌榮譽。

2013年，於2012/13賽季V超級聯賽上，囊括扣殺獎、攔網獎、救球獎三座個人數據獎項，並得到敢鬪獎以及入選最佳陣容。同年6月18日，在球季結束後宣布與[[橄欖球|橄欖球]]選手[[四宮洋平|四宮洋平]]結婚<ref>{{cite web|url=http://www.toray-arrows.jp/women/topics/detail.html?key=7DD97209B170F3AE49257B8E000FEE5B|title=荒木絵里香選手の入籍について|author=東麗箭|accessdate=2013-06-18|deadurl=yes|archiveurl=https://web.archive.org/web/20130620083846/http://www.toray-arrows.jp/women/topics/detail.html?key=7DD97209B170F3AE49257B8E000FEE5B|archivedate=2013-06-20}}</ref>。

2013年10月3日，東麗箭官方網站上發表預計隔年春天生產的消息，確定同年10月底退隊<ref>{{cite web|url=http://www.toray-arrows.jp/women/topics/detail.html?key=E51734F888826D8849257BF6002E793F|title=荒木選手の退社について|author=東麗箭|accessdate=2013-10-03|deadurl=yes|archiveurl=https://web.archive.org/web/20131005025231/http://www.toray-arrows.jp/women/topics/detail.html?key=E51734F888826D8849257BF6002E793F|archivedate=2013-10-05}}</ref>。
  
2016年，於2015/16賽季V超級聯賽上，其在赛中的攔網非常亮眼，而[[日本女排|日本女排]]在攔網上比較欠佳，所以被[[真鍋政義|真鍋政義]]邀請她再次回到日本女排的陣營當中。其後其隨隊出戰2016里約奧運落選赛暨亞洲區資格赛。赛中其攔網得分不少。最終日本女排5勝2負積14分，獲得了寶貴的奧運資格。
