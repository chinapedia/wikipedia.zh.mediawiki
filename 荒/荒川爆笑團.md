《'''荒川爆笑團'''》（{{lang-ja|荒川アンダー ザ ブリッジ}}）是[[日本漫畫家|日本漫畫家]][[中村光_(漫畫家)|中村光]]的[[日本漫畫|日本漫畫]]作品。作品於[[YOUNG_GANGAN|YOUNG GANGAN]]的2004年創刊號開始連載<ref>{{cite web|archiveurl=https://web.archive.org/web/20041207064518/http://www.square-enix.co.jp/magazine/yg/lineup/index.html|url=http://www.square-enix.co.jp/magazine/yg/lineup/index.html|title=2004年12月3日創刊号発売|publisher=[[史克威爾艾尼克斯|史克威爾艾尼克斯]]|archivedate=2004年12月7日|accessdate=2014年8月8日|language=Ja|deadurl=yes}}</ref>，2015年漫畫連載結束，單行本全15卷。

並於2010年4月改編為電視動畫，官方公佈於同年10月播放第二季。於2011年7月開始播放改編日劇。

==劇情簡介==
故事主人翁市之宮行是企業社長的兒子，時刻準備著繼承那間世界頂尖的家族企業。從年幼的時候開始，父親就告訴他「不欠人，不求人」的道理，生活時時遵守家訓。

可是，在一次意外溺水，行被住在橋下荒川釣魚的小珊救回性命，就此欠下救命之恩。為了報恩，小珊要求行跟她談戀愛，永遠在一起。行礙於家訓惟有答應，自己也開始在橋下生活。不過，[[荒川區|荒川區]]這地方遠比市之宮行想像更為怪異，大部分居民正是被社會認定為帶著瘋狂幻想的「電波族」……

==登場人物==
; {{nihongo|市之宮行 / 小招|（いちのみや こう） / リクルート||CV：[[神谷浩史|神谷浩史]]、[[齊藤千和|齊藤千和]]（幼年）}}
: 本作的男主角，大財閥市之宮集團的名門子弟。22歲的他雖仍是大學生，但已經設立了多間公司。生活時刻遵守家訓「不欠人，不求人」，但因意外與小珊成為救命恩人和戀人的關係。被荒川村長命名為「招兵買馬（{{lang|ja|リクルート}}/Recruit）」，荒川居民通稱「小招（{{lang|ja|リク}}）」，度著我行我素的每一天。
: 若欠人情而沒法報恩就會進入緊張狀態，引發[[哮喘|哮喘]]，但在荒川生活一段日子後鮮有病發（然而在荒川曾因為修女種植的杉樹花粉引發[[花粉症|花粉症]]）。搬來荒川後住在橋柱的房間，在荒川橋底下的工作是老師，被小珊形容有教導他人的天賦，同時在荒川跟不同居民生活期間，被迫推翻過往的學識和重新思考「常識」的意義。本人也漸漸快樂起來，特別是與小珊交往的時候。不相信幽靈和外星人的存在。
: 年幼的時候受過英才教育，4歲起接受父親積的教導開始學習製作料裡，能製作出令荒川所有居民予以好評的家常菜、演奏多種樂器、[[空手道|空手道]]拿下黑帶、煙花師的資格等多方面都十分傑出。頭腦明晰，姿容端麗使其從學生時代相當受歡迎。母親春子已故。童年回憶使其一直害怕父親。討厭[[青椒|青椒]]。
: 漫畫後期開始承接荒川村長的職位兼執行職務，曾一度想為荒川居民重新命名，但由於荒川居民表態已經中意村長取的名字而作罷。故事後期得知小珊實為金星人的身份，加著小珊手足的亡靈介入的事件，體認小珊對自己的意義而決定繼續和小珊相守，最終結局返回市之宮集團成立宇宙開發部門，決定帶著小珊向她的父母提親，家族公司亦開始登陸金星地面。

; {{nihongo|小珊|ニノ||CV：[[坂本真綾|坂本真綾]]}}
: 住在荒川的神秘少女，長相可愛的[[電波族|電波族]]。自稱「金星人」。初登場時救了為取回卡在橋梁鋼柱的西裝褲而意外墜河的行，此後把行視為自己的戀人。名字的來歷一般認為從穿的運動衫號碼布「2-3」（接二連珊(三)）。愛吃番茄、魚類、修女製作的糕餅和行的料理。
: 平日帶著男性語調，穿著藍白雙色的運動衣褲（但和行首次約會期間，曾破例換上行贈與的洋裝，有時會為在水裡活動而穿著「2-3」的泳裝）。由於其在荒川橋底下的工作是捉魚給居民吃，故經常在荒川釣魚和擅長游泳，亦曾為了捕捉一條大型鯰魚而耗費時間鍛練己身實力，間接練就強力的臂力。P子在冬季外出尋覓菜苗的期間，亦會擔任代理菜園管理者的工作<ref>漫畫第305話</ref>。擁有豐富的天文知識和懂得操作不同類型的天文望遠鏡。
:容易忘記事情，所以要行陪自己在荒川居住，以免忘記他的臉。家是由瓦紙板擔建並用布廉遮掩的小屋，但內裡有張格格不入的豪華天鵝絨床。只是睡覺的時候，會進去下層的抽屜。患有[[夢遊症|夢遊症]]，經常為尋求舒適睡眠的地方而做出麻煩事情，之後修女更因此傳授了防身術。當初表情態度十分淡白，但與行交往後變得常常露出笑容。為了防範被偶然路過的警方以公然猥褻罪取締，在荒川用鐵桶入浴時會特地包裹浴巾，直至穿好衣服後才抽出來。
: 漫畫故事後期揭露小珊是純正血統的金星人，金星人有著出生時便有成雙成對的特性與概念，但由於村長的關係，間接導致原本的孿生兄弟暨戀人身亡，當時陷入昏迷狀的小珊則被村長接引至荒川生活，由村長委託修女與瑪利亞負責保護小珊。其手足的靈魂附身於龜有的身軀，本欲將其接回金星，但獲知小珊希望留在地球和行廝守而作罷。最終結局跟著行回歸正常人生活，和行修成正果。
: 作者賦與的人物花語是[[鐵線蓮|鐵線蓮]]的「純潔的心」<ref name="Arakawa_comic_Vol.8">漫畫單行本第8集附錄</ref>。

; {{nihongo|村長|村長||CV：[[藤原啓治|藤原啓治]]}}
: 荒川村長，堅決自稱河童，自稱620歲。不管外人的目光組織荒川居民，亦跟居民隔開居住。
: 事實上是穿了膠製服裝的人，但行以外的人都視其為如假包換。內裡是一名穿潮流衣著的好青年，偶然會以魔術形式彰顯出來。新居民來的話會給他們起新的名字，並舉行歡迎會，但若是新居民仍以本名自我介紹就會感到光火。喜歡飲酒，愛吃[[黃瓜|黃瓜]]和[[海膽|海膽]]壽司<ref>漫畫單行本第14集和第15集</ref>。擔負著保護小珊的責任，訛稱要到小珊的家鄉「金星」而作宇宙特別訓練，使行等眾人為其在河底建造正式的別墅，過著舒適的生活。
: 有關戀愛事跟行一樣十分遲鈍，完全沒發現P子對自己的感情。隨劇情進展開始對行產生信任感，更委託行趁自己出差時擔任荒川區的村長。和黑道組織腐葉土領導者鼴鼠是兄弟關係<ref>漫畫單行本第15集</ref>。本人看似隨興，實則具備相當的雄厚財力和影響力，曾動用權力阻止積的開發工程，以及迫使漫畫家桑原薯條改變創作路線，以維護荒川居民的生活。
: 年輕時期和鼴鼠策畫綁架案，將當時年幼的金星人龍鳳胎（小珊及其手足）擄為人質，向兩者的雙親誆稱小珊已被自己滅口，要求交出豪宅以換取另一人的性命，但由於村長不知道金星人成雙誕生和無法忍受戀人死亡，以及得知自己孤身一人就會死亡的特性，導致聽見村長發言的小珊的手足當場暴斃身亡。懷抱罪惡感的村長則接受小珊的雙親提議，將小珊接往地球和留在荒川生活，委託修女和瑪利亞保護小珊，藉此償贖當年的過錯。
: 漫畫後期以拆卸改裝彩虹大橋的代價，將龜有和小珊的手足亡靈送回金星，但也使得荒川居民頓失住所，村長遂藉此事件舉辦惜別會暨魔術秀露出自己的真面目。最終結局以卸除河童裝的真面目和小P回歸正常人生活。

; {{nihongo|星星|星||CV：[[杉田智和|杉田智和]]}}
: 24歲的藝術家，自稱是超級明星。專長樂器是[[吉他|吉他]]和[[電吉他|電吉他]]。定期在荒川召開著音樂會，但大部分歌曲內容都很奇怪。
: 星形面具下還有月亮面具，不過在外出橋上區域時則會以真面目示人，真身是紅茶色頭髮的少年。受到挫折下沉時（多數是小珊的事），會稱自己為「海星」。不受村長約束堅決外出，特別喜歡在橋上的便利店買東西。嗜煙，特別喜歡七星煙。對時尚妝扮的敏銳度高，曾負責擔任居民的時尚顧問。
: 4年前是頂尖的歌星，曾經和某位搖滾女歌手交往，但是被八卦記者巴德偷拍到兩人約會的照片，導致女友為維護工作而主動向媒體撇清兩人的關係。因此被甩的星星特地越洋找上巴德談判，偷拍巴德和女兒伊莉的照片並賣給新聞出版社作為報復<ref name="Arakawa_comic_chapter.325">漫畫第325話</ref>。雖聲稱專輯經常進佔[[Oricon公信榜|Oricon]]第1位，但對自己沒有真正的作詞和作曲才能感到困惑。就在這樣的心情下他遇見小珊，最後認清自己追求的是由自己一手創造和演奏的音樂，於是戴上象徵「明星」的星形面具，住在移動住宅。喜歡小珊，所以視行為對手。但在許多方面都跟行合得來，尤其是台詞。
: 漫畫最終結局選擇回歸正常人生活和不再配戴頭套（星星將之贈與小珊作為餞別禮），成全行和小珊的戀情，臨走前請求行好好照顧小珊。故事尾聲舉辦個人回歸演唱會，重新復出歌壇而獲得樂迷的熱烈迴響。

; {{nihongo|修女|シスター||CV：[[子安武人|子安武人]]}}
: 穿著[[修女|修女]]服裝，金髮藍眼的倔強男人，逢星期日在荒川的教會舉行數秒彌撒。修道服下是軍服，胸前配戴的金色項墜，實為戰鬥短刀。29歲，國籍是英國。
: 數年前便於荒川生活，透過當地居民協助於橋下搭建小教堂，此後便常駐於教會為荒川居民服務，或協助調解荒川居民糾紛。對於願意收留他的村長相當感激。是少數知道小珊和村長的過去的荒川居民。史黛拉搬來荒川前兼任她的監護人，經常教授錯誤的知識給史黛拉。
: 曾是英軍大兵，有著大量的軍火，常因戰鬥癖性沒消除而反射性地攻擊。經常預備第三次世界大戰的教會地下避難所，對導彈等戰鬥詞彙有異常的快速作出反應。原為姿容端正的美青年，但右臉有較大的傷痕，遭受打擊時傷痕會大量噴血。出乎意料地擅長製作[[餅乾|餅乾]]和[[蛋糕|蛋糕]]等家庭甜品，喜歡且擅長種植花卉。過去曾於雇傭兵時期和當時擔任敵方間諜的瑪利亞相戀三年且論及婚姻，惜因洩漏情報導致戰爭落敗而連帶讓兩人分手<ref>漫畫第251話</ref>，修女和瑪利亞搬來荒川後，仍情執於瑪利亞，但是瑪利亞經常使用毒舌等各種手段摧殘修女的身心，致命傷是"無趣的人"。
: 因為行在歡迎村長歸來的派對展現的廚藝，以至於修女為了博得瑪利亞的心意而跪求行傳授製作雞蛋捲的秘訣。
: 漫畫最終結局回歸正常人的生活，決定和瑪利亞帶著史黛拉和鐵人兄弟回到自己當年服務的孤兒院共同生活。最終話交代眾人後話的鏡頭中，透露修女已轉任教會[[神父|神父]]不再穿戴修女服飾，繼續和加入教會成為修女的瑪利亞共同服務民眾。

; {{nihongo|白井通 / 小白| しろい じる/ シロ||CV：[[大塚芳忠|大塚芳忠]]}}
: 經常一邊畫白線，一邊線上移動的43歲中年男子。本名'''白井通'''。
: 對白色的事物<ref>漫畫第168話</ref>和繪製白線有著異於常人的執迷，夢想是取得畫白線的冠軍。給自己定下了「到家之前都要走在白線上，否則老婆會變成白科尼什雞」的規矩，貌似回家路線道路施工，為了遵守規矩而繞路導致迷路，從北海道來到了荒川，已離家６年未曾見過妻女，但妻女仍透過書信表態支持其夢想，不時會寄送包裹慰問近況。
: 曾為一等企業的營業員，那時候培育的商業品格和知識並沒衰弱，外貌舉止可以打扮得像正常的高級職員。但因為失業遭受社會大眾的白眼，非常介意外人嘲諷其無業。雖然在荒川沒有特別工作，但會在荒川春季轉職活動擔任審查與顧問。

; {{nihongo|天狗三人幫|天狗3人衆||CV：[[中村悠一|中村悠一]]}}
: 與亞馬遜女戰士一起守護「秘寶」，二人黑髮，一人金髮的青年群。習慣將女性胸罩外穿在Ｔ恤，即便游泳也會穿著上半截的女性泳裝。在海邊經營著「天狗海之家」。
: 性格坦率，做事和表達想法都很直接，擅長[[催眠術|催眠術]]。為女戰士的願望不惜一切，曾進行「女戰士戀愛支援計劃」，使用催眠術令行跟她搭上。其後女戰士又戀慕星，便常常稱星為「好哥哥」與「卡滋卡滋認定的第二男人」等怪稱呼。

; {{nihongo|也斯|ヤス||}}
: 「國鳥會」成員，曾是比利的下級。燕子。首次登場是通知賈桂琳攸關比利有危機的消息。醉心比利的程度等於高井溺愛行。
: 過去曾在經濟拮据的時候，被比利接濟和共住一段時間，從此對比利產生超越上司和下屬的情愫。因為關心比利而頻頻造訪荒川，視賈桂琳為情敵，化解誤會後反而為筆友。

; 長尾雉
: 「國鳥會」會長，比利的前上司，賈桂琳的前任。

; {{nihongo|鼴鼠|もぐら|}}
: 黑道組織「腐葉土」的領導人，戴著鼴鼠的面罩。個性嚴謹認真，擅長捉人和佈置陷阱。真面目是留著金髮的男子。喜歡女高爾夫球手手平美寶玲。漫畫後期揭露其真實身分是村長的弟弟。
: 過去曾和村長策畫綁架小珊和其手足，試圖向兩者的雙親勒索宅邸，卻因此導致小珊的手足死亡，從此對這段往事懷有罪惡感。曾以自身經歷為創作題材多度向漫畫雜誌投稿作品，希望藉此揭露這段往事，但皆以被退稿告終，之後和地球防衛隊長一拍即合，被後者邀請成為創作搭擋，同時提供地球防衛隊長在漫畫創作方面的協助。
: 因為和地球防衛隊長干預荒川居民的隱私而不被歡迎，曾一度設局綁架小珊，阻止村長將小珊送回金星，但被從中干預而綁架未遂<ref>漫畫單行本第10卷</ref>。後試圖用漫畫投稿方式向地球防衛隊長揭露村長和小珊的過去，被地球防衛隊長以不合理為由婉拒刊登他的漫畫，但因為地球防衛隊長告訴他「你的兄長是絕不會殺人的」而解開當年心結，轉而和荒川居民合作解救小珊。
: 漫畫最終話交代眾人後話的鏡頭裡，鼴鼠不再配戴面罩，和地球防衛隊長回歸正常人的生活，兩者收看電視節目時面露興奮之情，暗示兩人的合作作品已被改編成影視作品。

; {{nihongo|神秘人|謎の男||CV：[[藤原啓治|藤原啓治]]}}
: 推倒積的開闢計劃，拯救了荒川的人。其與小珊、修女和瑪利亞居住在荒川有很大關係。有著影響大臣決定的財力，現在亦幕後策劃送小珊回金星的火箭發射計劃。（確定是河童村長，漫畫第10冊有鼴鼠回憶畫面所顯示）

; 小珊的雙親
: 漫畫後期回憶篇，小珊的雙親。
: 過去因為村長和鼴鼠綁架龍鳳胎兒女，導致兒子死亡和小珊陷入短暫昏迷狀態，小珊的雙親認為地球是許多孤獨的人們群居的星球，可藉此讓小刪減緩孤獨不適應的心情，為了保護小珊遂向村長提議將她接往地球生活，同時亦將兩人當初作為給村長的贖金的宅邸（彩虹大橋）留在荒川。
: 漫畫後期村長動用自己的積蓄準備燃料和NASA合作，將小珊的雙親留在地球的彩虹大橋，做為送走小珊的手足亡靈的載具。

; 小珊的兄弟
: 漫畫後期登場，有著金色長髮，容貌和小珊相似的金星人幽靈，小珊過去的戀人暨孿生兄弟。
: 因為金星人出生時便和伴侶成對誕生的特性，和小珊結為情人。但是小珊年紀還小的時候，村長策畫綁架兩者，向兩者的父母誆稱小珊已被滅口，以他的性命要脅要求換得宅邸，卻導致聞言的他精神遭受打擊，過於激動而驟死。過身後為了尋找小珊的下落，附身在龜有的身上，試圖將小珊帶回金星，但由於小珊決定留在地球而尊重其意願，最終在行和村長等人的努力下，被送回金星。

; 轟天&雷
: 漫畫裡登場，村長兒時崇拜的地下獨立樂團二人組，真身是[[雷神|雷神]]。尚未走紅前經常在荒川演奏創作音樂，後來星探發現兩人的創作才能，卻只招攬雷進入演藝圈，導致兩人發生衝突並解散樂團。時隔多年後，荒川居民為了幫助村長圓願和偶像再會，架設避雷針邀請兩者，而轟天和雷則以演奏自己的音樂（落雷）擊向避雷針和星星作為答謝眾人的回應<ref>漫畫第228話</ref>。

; 小洋
: 漫畫裡短暫登場，和史黛拉同間孤兒院的少年。在史黛拉搬往荒川後，仍經常致信問候史黛拉的近況。

; 巴德·金斯柏格（バド·キﾝスバガ）
: 漫畫回憶裡短暫登場，曾經偷拍還在樂壇時期的星星和時任女友的緋聞照片，導致星星被時任女友撇清關係的[[狗仔隊|狗仔隊]]記者<ref name="Arakawa_comic_chapter.325">漫畫第325話</ref>。
: 原本是喜愛自然題材的攝影師，但在成為狗仔隊記者後，轉而偷拍名人的醜聞照謀利，導致無法忍受此事的妻子簽署離婚協議書並出售一家三口的住宅。後來被巴德偷拍而被時任女友甩掉的星星伺機報復，偷拍並擅自將巴德和伊莉的合照註記標題，出售給新聞出版社。

; 紗音（さね）
: 漫畫第325話回憶篇登場，巴德的妻子。因為無法忍受丈夫成為狗仔隊攝影記者，向巴德提出離婚，並將一家人的住宅出售給繼任屋主。

; 伊莉•金斯柏格（イリ·キﾝスバガ）
: 漫畫第325話回憶篇登場，巴德和紗音的女兒，因為父母離婚，加著父親埋首工作而常感到寂寞。後來因為父親及時醒悟而和其言歸於好。

; 怪小鬼脫褲魔
: 漫畫裡登場，經常躲在荒川大橋附近，專門物色穿著長褲的路人，趁機襲擊脫下路人的褲子的小學生四人組。後來在龜有造訪荒川時，襲擊並脫下龜有的長褲，反被龜有誤認是荒川居民精心準備的歡迎式。
: 漫畫最終結局裡已成年，其中一人準備在畢業後進入市之宮的旗下事業任職<ref name="Arakawa_comic_Vol.15">漫畫單行本第15集</ref>。
