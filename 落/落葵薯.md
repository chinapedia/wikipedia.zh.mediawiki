'''落葵薯'''（[[学名|学名]]：{{lang|la|''Anredera cordifolia''}}），又稱[[川七|川七]]、藤三七、藤子三七、小年藥、土三七、藤七、馬德拉藤、洋落葵。为[[落葵科|落葵科]][[落葵薯属|落葵薯属]]的植物。分布在[[南美热带地区|南美热带地区]]以及[[中国|中国]]的[[云南|云南]]、[[福建|福建]]、[[浙江|浙江]]、[[四川|四川]]、[[广东|广东]]、[[江苏|江苏]]、[[北京|北京]]等地，目前已由人工引种栽培。

== 别名 ==
马德拉藤（经济植物手册） 藤三七（云南） 藤七（四川植物志）

== 种下等级 ==
* Boussingaultia gracilis Miers var. pseudobaselloides Bail.

== 藤三七之藥用價值 ==
藤、[[珠芽|珠芽]]、[[塊根|塊根]]可入藥，[[性味|性味]]微苦、性[[溫|溫]]，能[[滋補|滋補]]強壯、消腫[[散瘀|散瘀]]。
