'''《齊民要術》'''是[[中國|中國]]保存得最完整的一本古代農牧情況的鉅著，由[[北魏|北魏]]官員[[贾思勰|贾思勰]]所著，成書於[[東魏|東魏]]武定二年（544年），另一說為533年至544之間。收錄公元6世纪时[[中國|中國]][[黃河流域|黃河流域]]下游地區的[[農藝|農藝]]、[[園藝|園藝]]、[[造林|造林]]、[[蠶桑|蠶桑]]、[[畜牧|畜牧]]、[[獸醫|獸醫]]、[[配種|配種]]、[[釀造|釀造]]、[[烹飪|烹飪]]、儲備，以及[[治荒|治荒]]的技术。書中正文分成10卷，92篇，援引古籍近200種，所引《[[氾勝之書|氾勝之書]]》、《[[四民月令|四民月令]]》等現已失傳的[[漢朝|漢]][[晉朝|晉]]重要農書，後人只能從此書了解當時的農業運作。

該書自出版後，長期受中國歷朝政府重視，傳遍海外後亦被常成為研究古物種變化的經典，[[達爾文|達爾文]]研究[[進化論|進化論]]時曾參考一部「中國古代百科全書」，有說此書正是《齊民要術》。在[[四庫全書|四庫全書]]中為子部農家類。

== 背景 ==
《齊民要術》作者為[[東魏|東魏]]（534-550）[[賈思勰|賈思勰]]，是[[齊郡|齊郡]][[益都县|益都县]]（今[[山東|山東]][[壽光|壽光]]）人，記述[[黃河流域|黃河流域]]下游地區，即今[[山西|山西]]東南部、[[河北|河北]]中南部、[[河南|河南]]東北部和[[山東|山東]]中北部的農業生產，蓋述農、林、牧、漁、副等部門的生產技術知識。

[[贾思勰|賈思勰]]青年時代，正值[[北魏孝文帝|北魏孝文帝]]所倡[[孝文漢化|孝文漢化]]運動的高峰，朝廷議政以農為首，督辦農業，違者免官。太和九年（485年）又實行[[均田制|均田制]]，把無主荒地分給無地或少地農民耕種，規定種植[[五穀|五穀]]和瓜果蔬菜，植樹造林。

自[[東漢|東漢]]後的200年間，各地戰火頻起，農業生產受到破壞，但農業技術並未中輟。[[魏晉南北朝|魏晉]]時期，[[人字耙|人字耙]]和[[無齒耙|無齒耙]]開始出現，形成「耕-耙-磨」結合的耕作技術，加強旱地防旱的技術，[[北魏|北魏]]時又積累了一整套針對不同季節的「耕-耙-磨」經驗。

在此中背景下形成的《齊民要術》，{{Fact|初以手抄本傳播|time=2011-10-14T07:01:38+00:00}}，除了詳細記錄農業，亦抨擊商賈漠視民生，「買賤鬻貴」（平買貴賣），但書中對市場有助貨物流通、減少荒災的作用非常重視，該文資料或為後人所加。《齊民要術》推崇<ref name="自序">[http://zh.wikisource.org/wiki/%E9%BD%8A%E6%B0%91%E8%A6%81%E8%A1%93/%E5%BA%8F 《齊民要術/自序》]。</ref>[[耿壽昌|耿壽昌]]之常平倉、[[桑弘羊|桑弘羊]]之[[均输法|均输法]]皆為「益國利民，不朽之術」，並嘲笑[[孔子|孔子]]「四体不勤，五谷不分」<ref>“四体不勤，五谷不分”语出《[[论语|论语]]·微子》，贾思勰引《论语·微子》的话说，“丈人日：‘四体不勤，五谷不分，孰为夫子?’”來嘲笑當時士大夫自命高人一等。</ref>。

自《齊民要術》推出後，引起歷代[[政府|政府]]重視，[[北宋|北宋]]時期的官刊[[善本|善本]]不易看到，有「非[[朝廷|朝廷]]人不可得」之說<ref name="葛祐之">[http://zh.wikisource.org/wiki/%E9%BD%8A%E6%B0%91%E8%A6%81%E8%A1%93/%E5%BE%8C%E5%BA%8F 《齊民要術/後序》]，[[葛祐之|葛祐之]]在序文中提到，當時[[天聖|天聖]]中所刊的[[崇文院|崇文院]]板本（版本）不是尋常人可見，藉以稱頌[[張轔|張轔]]能刊行於州治，「使天下之人皆知務農重穀之道」。</ref>。明代[[王廷相|王廷相]]（1474-1594年）稱它為「惠民之政，訓農裕國之術」，唐、宋以來出現不少農書，無不以它為範本，其中，元《[[農桑輯要|農桑輯要]]》、[[王禎|王禎]]《[[王禎農書|王禎農書]]》、[[明代|明代]][[徐光啟|徐光啟]]《[[農政全書|農政全書]]》、清《[[授時通考|授時通考]]》均受其影響。

約19世紀傳到[[歐洲|歐洲]]，英國學者[[達爾文|達爾文]]（1809年 - 1882年）在其名著《[[物種起源|物種起源]]》和《植物和動物在家養下的變異》曾提及參閱了「一部中國古代百科全書」，並援引有關事例作為[[進化論|進化論]]的佐證，{{Fact|有說該書正是《齊民要術》|time=2011-10-14T04:52:19+00:00}}。該書約於[[唐|唐]]末時傳入日本，至今日本還藏有北宋最早刊印的殘本。

== 內容 ==
《齊民要術》主要记述[[北朝|北朝]]時期的生產活動。如序文所言{{quote|今采袀经传，爰爱歌谣，询之老成，验之行事，起自農耕，終於醯醢，資生之業，靡不畢書}}。書中舉凡[[五穀|五穀]]、[[瓜果|瓜果]]、[[蔬菜|蔬菜]]、[[樹木|樹木]]的栽培，[[牲畜|牲畜]]、[[家禽|家禽]]、[[養魚|養魚]]，[[酒|酒]]、[[醬|醬]]、[[醋|醋]]、[[豉脯|豉脯]]、[[羹|羹]]、臛（[[肉羹|肉羹]]）、葅（[[泡菜|泡菜]]）、[[餅|餅]]、[[飯|飯]]、[[飴|飴]]、[[糖|糖]]等的製作，以及[[煮膠|煮膠]]、[[造墨|造墨]]的方法等，均有論述，其中许多技术，不见以前的农书或文献<ref>《齐民要术·前言》第9页</ref>。

* 卷一：耕田、收種、種穀各1篇；
* 卷二：穀類、豆、麥、麻、稻、瓜、瓠、芋等13篇；
* 卷三：種葵（蔬菜）、蔓菁等12篇；
* 卷四：園籬、栽樹（園藝）各1篇，棗、桃、李等果樹栽培12篇；
* 卷五：栽桑養蠶1篇，榆、白楊、竹以及染料作物10篇、伐木1篇；
* 卷六：畜、禽及養魚6篇；
* 卷七：貨殖、塗甕各1篇（釀造）、釀酒4篇；
* 卷八、九：釀造醬、醋，乳酪、儲存22篇，煮膠、製墨各1篇；
* 卷十：非中國（指北魏以外）物產者1篇，記熱帶、亞熱帶植物100餘種，野生可食植物60餘種。

書中每一章節，由篇題、正文和經傳文獻組成。篇題援引歷史文獻，備有有註文，述說異名、別名、品種、地方名產、物種來源及其性狀特徵；該書亦有作者親身驗證的經驗，如書中指濟州以西（今魯西）的[[長轅犁|長轅犁]]不及齊人的[[尉犁|尉犁]]「柔便」；[[蠶繭|蠶繭]]用鹽殺蛹法比曝曬為好。

在此書之前，《[[呂氏春秋|呂氏春秋]]·上農》等四篇，對農學有所專述，但主要講述栽培。[[西漢|西漢]]《[[汜勝之書|汜勝之書]]》存者不過3000餘字，但內容只記有種植業。《[[四民月令|四民月令]]》雖已涉及農、林、牧、副，卻略於技術而重於農事之安排。

== 版本 ==
《齊民要術》歷來版本眾多，除了原作外，一般相信雜記內容由後人摻入，現存版本則有
* [[北宋|北宋]]崇文院刻本，孤本残卷现存[[日本|日本]]，有[[杨守敬|杨守敬]]影描本，存[[北京|北京]][[中国农业科学院|中国农业科学院]]图书馆。1914年[[罗振玉|罗振玉]]将[[杨守敬|杨守敬]]影描本以珂罗版印行。
* 北宋崇文院刻本的日本抄本，藏日本[[金澤文庫|金泽文库]]。
* [[南宋|南宋]][[張轔|張轔]]刻本，已失。
* 南宋本張轔刻本的明代抄本，十卷完整、无缺页错页；1922年[[商务印书馆|商务印书馆]]《四部丛刊》收入此版的影印本。
* 南宋本張轔刻本的元代刻本、明代[[馬直卿|馬直卿]]刻本、明末[[胡震亨|胡震亨]]——[[毛晉|毛晉]]刻本、清代[[張海鵬|張海鵬]]刻本、清末[[袁昶|袁昶]]刻本（漸西村舍-{}-本）、日本[[山田羅谷|山田羅谷]]刻本及[[仁科幹復|仁科幹復]]刻本、日本金澤文庫舊抄卷子本。

注釋上現有[[石聲漢|石聲漢]]《齊民要術今釋》、[[繆啟愉|繆啟愉]]《齊民要術译注》等。

《[[四库全书|四库全书]]》和《[[四库全书荟要|四库全书荟要]]》都收入《齊民要術》
