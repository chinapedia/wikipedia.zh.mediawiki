'''齊柏林-斯塔肯R-VI轟炸機是'''德國在[[第一次世界大戰|第一次世界大戰]]時的[[重型轟炸機|重型轟炸機]]，在大戰期間主要和[[戈塔G轟炸機|戈塔G轟炸機]]一起轟炸英國，把死亡和恐懼帶給英國民眾。

[[齊柏林-斯塔肯|齊柏林-斯塔肯公司]]就是生產[[齊柏林飛船|齊柏林飛船]]的生產商，該公司除了造飛船也造飛機而且是超巨型飛機，德國軍方用R作為其代號，意即[[巨人機|巨人機]](德:{{lang|de|Riesenflugzeug}})，該公司在R-VI之前都製造過各種巨人機但都是單獨一架，祇有R-VI得到量產18架的訂單。<ref name = "Gray">Gray, P and Thetford, O ''German Aircraft of the First World War''  1970 Putnam London 0 85177 809 7</ref>

==發展簡史和設計特點==
[[File:Zeppelin-Staaken_R.IV_model.jpg|thumb]]

齊柏林-斯塔肯R-VI轟炸機是其前身{{tsl|en|Zeppelin-Staaken R.V|齊柏林-斯塔肯R-V轟炸機}}發展出來的，R-V有5俱戴姆勒D-1Va式水冷式發動機，其中在機首有一俱而其餘4俱發動機以兩俱為一組在左右各推動在後方的一個四葉螺旋槳，R-VI則把機首那俱發動機取消並在其位置加上一個被稱為燕巢的防衛機槍座位，其餘四俱發動機以兩俱為一組在左右各推動一個兩葉螺旋槳，即是以前拉後推的方式推動飛機前進。

不同於當時大多數飛機，齊柏林-斯塔肯R-VI轟炸機在機首下方也有起落架而令它不論在飛行或在地面皆是水平的，由於它龐大的機身和重量，其主起落架的車輪要兩個重疊在一起，它也需要特別加固的機場跑道。

===R.30/16測試機===
R.VI R.30/16是最早使用[[機械增壓器|機械增壓器]]的飛機，第五個發動機－奔馳D.II被安裝在機身中央，以大約6,000轉的速度驅動Brown-Boveri四級增壓器。這使得R.30/16能夠爬升到海拔19,100英尺(5,800米)<ref>{{cite web |url=http://www.enginehistory.org/Before1925/WW1SC/WW1SC.shtml |title=Notes on WWI German Superchargers - Brown-Boveri (misspelled ''Brown-Bolvari'') |author=Paul Dempsey |date= |work=Piston Engines-Engines Before 1925 |publisher=Aircraft Engine Historical Society |accessdate=November 14, 2012 |archive-url=https://web.archive.org/web/20170427003935/http://www.enginehistory.org/Before1925/WW1SC/WW1SC.shtml |archive-date=2017-04-27 |dead-url=yes }}</ref>。直到第二次世界大戰後期{{tsl|en|Henschel Hs 130|亨舍爾Hs130E}}再次將這一想法重新用於Höhen-Zentrale-Anlage系統時，德國才再次嘗試使用額外動力裝置為增壓器供電來增加活塞發動機輸出功率的想法。R.30/16飛機後來安裝了四個可變螺距螺旋槳，據信只有在地面才可調節螺旋槳。<ref name = "Haddow"/>

==實戰==
[[File:Zeppelin-Staaken_R.VI_photo1.jpg|thumb]]

1917年6月28日，齊柏林-斯塔肯R-VI轟炸機開始裝備在[[東方戰線_(第一次世界大戰)|東線]]成立的第500和第501巨人機大隊(德：{{lang|de|Riesenflugzeug-Abteilung}})，同年8月由[[庫爾蘭|庫爾蘭]]起飛轟炸[[俄羅斯帝國陸軍|俄軍]]，之後此兩隊被調往[[西方戰線_(第一次世界大戰)|西線]]，第500巨人機大隊被部處在[[法國|法國]]加斯提尼，而501被部處在[[比利時|比利時]]根特，兩隊皆以[[英國|英國]]作為空襲目標，9月17日，齊柏林-斯塔肯R-VI轟炸機首次轟炸英國，之後在1917年12月到1918年5月，齊柏林-斯塔肯R-VI轟炸機和[[戈塔G轟炸機|戈塔G轟炸機]]一起總共11次對包括倫敦在內的英國城市進行轟炸，投下了大約27噸炸彈，期間竟然無任何齊柏林-斯塔肯R-VI轟炸機被擊落，祇有兩架在回航時墜毀。

==戰後==
戰後總共18架的完成品，僅有6架存活或於戰後被製造出來。<ref name = "Haddow"/>

==變種==
'''齊柏林-斯塔肯 R.VI'''
:為該系列第一種正式投入生產的型號，使用四具梅巴赫 MbIV引擎(每具可輸出245匹馬力)或四具{{tsl|en|Mercedes D.IVa|奔馳 D.IVa}}引擎(每具可輸出260匹馬力)，機身設計與前一代{{tsl|en|Zeppelin-Staaken R.V|齊柏林-斯塔肯R-V轟炸機}}十分相似，不過R-VI的駕駛座艙被向前加長延伸，而位於機鼻的機槍手座艙改為可封閉式，使用系列編號R25/16到R39/16、R52/16到R54/16在{{tsl|en|Luftstreitkräfte|德意志帝國空勤部隊}}中服役，但R30被用為測試平台因此並未投入服役。<ref name = "Haddow"/>

'''齊柏林-斯塔肯 R.VII'''
:和R.IV不同之處，R.VII的尾翼修改了支柱佈置。唯一的R.VII，序列號為R14/15，在運送到前線的過程中墜毀。僅一架被製造出來。<ref name = "Haddow"/>

'''齊柏林-斯塔肯 R.XIV'''
:R.XIV與以前的齊柏林-斯塔肯設計的巨人機(Riesenflugzeug)非常相似，僅在發動機安裝和細節上有所不同。五個邁巴赫MbIV發動機機艙中兩兩成對，呈推拉式，剩下的一個發動機則安裝在機頭中。<ref name = "Haddow"/>
[[File:Zeppelin-Staaken_R.XIV_WW1_aircraft_3.jpg|right]]

'''齊柏林-斯塔肯 R.XV'''
:和R.XIV一樣的引擎布局，但在尾翼上加裝了一個大型的安定翼，總共製造了三架該型號，編號為R46/16、R47/16和R48/16，但沒有相關紀錄或證據能證明他們曾經投入作戰。<ref name = "Haddow"/>

'''齊柏林-斯塔肯 L型水上飛機'''
:這架飛機基本上是一架R.VI，配有大型{{convert | 13 | m | ftin | abbr = on}}長的硬鋁合金浮筒。[[德意志帝國海軍|德意志帝國海軍]]曾分配給唯一一架該型飛機1432的編號，然而飛機在試驗期間失事。僅有一架完成。<ref name = "Haddow"/>

'''齊柏林-斯塔肯 8301型水上飛機'''
:為了進一步為[[德意志帝國海軍|德意志帝國海軍]]開發有用的大型水上飛機，齊柏林-斯塔肯使用R.VI的機翼與全新機身，該機身包含了R.XV的大安定翼，懸掛在機身的中間，並使用類似於L型水上飛機的浮筒。共三架建成，序號為8301,8303和8304，其中8301也使用過陸地起落架進行測試，8302的存在尚未確認。<ref name = "Haddow"/>
