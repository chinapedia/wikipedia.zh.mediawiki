《'''儒林外史'''》，中國[[清代|清代]][[章回小說|章回小說]]、長篇[[諷刺小說|諷刺小說]]，作者吴敬梓，全书共五十六回（也有人认为最后一回非吴所作），约40万字，描写了近二百个人物，花了他十幾年的時間完成。描写了[[康雍乾|康雍乾]]时期[[科举|科举]]制度下读书人的功名和生活。

==主題==
《儒林外史》是[[中国文学史|中国文学史]]上一部杰出的[[现实主义|现实主义]]的长篇讽刺小说，令人「百讀不厭。<ref>黃安謹《〈儒林外史〉新評》的序言說：“《儒林外史》一書，蓋出雍乾之際，我皖南北人多好之。以其頗涉大江南北風俗事故，又所記大抵日用常情，無虛無縹緲之談；所指之人，蓋都可得之，似是而非，似非而或是，故愛之者幾百讀不厭。”（1885年上海寶文閣刊本“天目山樵戲筆”《〈儒林外史〉新評》）</ref>」《儒林外史》所写内容，假托明朝，实为清朝，而且十之八九的人物都实有其人。它真实地描绘了[[康雍乾|康雍乾]]时期知识分子生活的沉浮，境遇的顺逆，功名的得失，仕途的升降，情操的高尚与卑劣，理想的倡导与破灭，出路的探索与追寻。<ref>闲斋老人《〈儒林外史〉序》：“其书以功名富贵为一篇之骨：有心艳功名富贵而媚人下人者；有依仗功名富贵而骄人傲人者；有假托无意功名富贵自以为高，被人看破耻笑者，终乃以辞却功名富贵，品地最上一层为中流砥柱。”</ref>

吴敬梓以对待功名富贵的态度来肯定或否定书中人物，如[[匡超人|匡超人]]假造文书，[[捉刀|捉刀]]代考，却被[[温州|温州]][[学政|学政]]“把他题了优行，贡入[[太学|太学]]肄业”，[[严贡生|严贡生]]无恶不作，卻被前任周学台推举为“优行”；作者无情地鞭挞丑恶事物時，同時也歌颂少數幾位正面人物，[[王冕|王冕]]是書中的第一流人物，为人“嶔崎磊落”，[[庄绍光|庄绍光]]追求“以礼乐化俗”、“以德化人”，牛老爹和卜老爹也是作者所歌頌的。程晋芳在《怀人诗》稱：“外史记儒林，刻划何工妍；吾为斯人悲，竟以稗史传!”
 
[[吳敬梓|吳敬梓]]在《儒林外史》之中，運用樸素、靈活、幽默的本地方性的語言，撰寫了[[科舉|科舉]]制度的腐朽黑暗，假名士的庸俗不堪，貪官污吏的卑鄙刻薄。[[胡適|胡適]]形容本書：「……國家天天掛著[[孔孟|孔孟]]的招牌，其實不許人說孔孟的話，也不要人實行孔孟的教訓，只要人念[[八股文|八股文]]，做試帖詩；其餘的『文行出處』都可以不講究，講究了又『那個給你官做？』」

==取材==
儒林外史中有不少是史實，部分人物真實存在于歷史。如寫「王冕畫荷鬻錢養母」，其中主角[[王冕|王冕]]是真有其人，其事也是真有其事。[[季羨林|季羨林]]指出《儒林外史》第八回的蘧公子道：“是戥子声，算盘声，板子声。”與[[尤侗|尤侗]]《艮斋杂记》、[[褚人获|褚人获]]《坚瓠集》十集卷一中[[袁箨庵|袁箨庵]]的故事雷同。《儒林外史》第四十回和第四十一回写的女诗人沈琼枝與《[[随园诗话|随园诗话]]》卷四雷同。《儒林外史》第五十三回與《[[酉阳杂俎|酉阳杂俎]]》卷一雷同<ref>《季羡林说国学》：〈《儒林外史》取材的来源〉</ref>。

==人物舉隅==
严贡生，字致中。是[[广东省|广东省]][[高要县|高要县]]的一位[[贡生|贡生]]，[[严监生|严监生]]之兄。书中第六回描写他虽然系“优贡”，却横行乡里。他强圈已经强卖给邻居王小二的[[猪|猪]]，并打伤前来讨要的王小二哥哥的腿。他又讹诈黄梦统，虽然并没有借给银子，却硬要偿付大半年的[[利息|利息]]，并强夺黄老汉的驴和米。两人告到[[知县|知县]]那里，严贡生避往[[省城|省城]]。严贡生与新婚的儿子[[儿媳|儿媳]]自[[省城|省城]]回家，却将吃剩的几片[[云片糕|云片糕]]说成治疗[[晕船|晕船]]的贵重药物，以此恐吓船家，赖掉船费。严监生死后，他又欺凌[[守寡|守寡]]又丧子的弟妇赵氏，让他新婚的二儿子继承弟弟的产业。<ref>《同体异形:严监生、严贡生比较论》，江西社会科学，2009年10期 </ref>

沈琼枝，扬州才女，善诗书，因不愿為妾，逃到[[南京|南京]]，但不幸被人捉到了衙门。《儒林外史》在第四十回及四十一回對沈琼枝的逃婚行为大加称赞。其原型为[[袁枚|袁枚]]审议松江才女[[张宛玉|张宛玉]]逃婚一案。

==藝術成就==
《儒林外史》是中國小說史上第一部蛻盡「[[話本|話本]]」文學痕跡的長篇小說，由一連串短篇、中篇性質的「段落」連綴組合而成。每個段落有大量精彩情節，全書主題明確，「雖-{云}-長篇，形同短製」。

《儒林外史》是中國[[諷刺|諷刺]]小說中的傑作，運用諷刺文學的手法，對當時傳統社會的[[科舉|科舉]]制度和「吃人」的[[禮教|禮教]]，以至官僚政治、社會風氣、人倫關係、風俗習慣等等，作出無情的抨擊和揭露；嘻笑怒罵，淋漓酣暢。

《儒林外史》能反映當時的社會現實，並深刻地反映人性，取材於耳聞目見的實際的日常生活，是不朽的[[現實主義|現實主義]]作品，一掃過去小說中那些神鬼的荒誕，玄虛縹緲的奇談以及因果輪迴的迷信。書中成功地刻畫出許多鮮明概括的反面典型，和一些正面人物，提高女性地位（如第37回）。作者把希望寄託於古代的「純儒」，有「戀舊」氣味。

《儒林外史》長於描寫人物心理，往往通過同一人物的矛盾言行，展現其虛偽與矯飾，並善於使用帶諷刺的誇張手法，描繪人物的過份言行，暴露他們的醜態，又長於以人物與人物作對比。小說文筆生動，語言洗煉，鮮用土語[[方言|方言]]，是[[國語|國語]]的文學，善用口語，切合人物身分，語彙豐富傳神。

《儒林外史》對清朝時期的小說，有很大影響，儘管此書一開始並無預先設計的結構。<ref>[[夏志清|夏志清]]《中国古典小说导论。儒林外史》所说：其“第三部分（第三十七回至五十四回）由一组形形色色的故事混杂而成，没有明确的构思。……总的看来，这一部分留给人们一个严重的不匀称的印象。”</ref>特色是结构鬆散，没有贯穿首尾的主干，“事与其来俱起，亦与其去俱讫”<ref>鲁迅《中国小说史略》</ref>。

胡適指出，“这部书是一种讽刺小说，颇带一点写实主义的技术，既没有神怪的话，又很少英雄儿女的话。况且书裡的人物又都是儒林中人，谈什么举业、选政，都不是普通一般人能了解的。因此，第一流小说之中，《儒林外史》的流行最不广”。對鞭笞社會不公，提昇人民自主思想，有一定意義。並且，對現代諷刺文學有深刻的啟迪。

== 刻本與真偽 ==
《儒林外史》原本仅55回<ref>程晋芳在《文木先生传》中记为“五十卷”，[[胡适|胡适]]《吴敬梓年谱》认为除“幽榜”一回为后人“妄增”外，“馀下的五十五回之中，大概还有后人增加的五回”。</ref>。根据[[程晋芳|程晋芳]]《怀人诗》，可以证明在吴敬梓49岁（1750年）的时候已经脱稿<ref>《春帆集》收《怀人诗》十餘首，有一首注：“全椒吴敬梓字敏轩。”詩曰：“外史纪儒林，刻画何工妍！吾为斯人悲，竟以稗说传。”</ref>，但是直到作者死后十多年，[[金兆燕|金兆燕]]官揚州府學教授時，將《儒林外史》刊刻印行，“人争传写之”<ref>程晋芳《文木先生传》</ref>。这个刻本，今已失传。

现存最早的刻本是嘉庆八年（1803年）镌刻的卧闲草堂本。现在通行的刻本是56回，其中最末一回部份的學者認為是后人伪作{{NoteTag|清代[[金和|金和]]在群玉斋本《儒林外史跋》中说“是书原本仅五十五卷，于述琴棋书画四士既毕，即接《沁园春》一词，何时何人妄增‘幽榜’一卷”。《儒林外史》的“幽榜”是否为后人所加，目前学术界仍存在不同意见。}}。

== 評點 ==
晚清名士[[张文虎|张文虎]]嗜讀《儒林外史》，同治十二年（1873年），張文虎以“天目山樵”為名開始評點《儒林外史》<ref>[[刘咸炘|刘咸炘]]《校雠述林》卷4《小说裁论》：“好坐茶寮，人或疑之，曰：‘吾温《儒林外史》也。’”</ref>。张文虎以為《儒林外史》第三十回[[杜慎卿|杜慎卿]]厭惡婦人的話出於[[蕭詧|蕭詧]]事；《儒林外史》第三十六回写孝子郭力赴西川寻父事，张文虎以為郭孝子原型为[[李保泰|李保泰]]《啬生文集·胡孝子寻亲记》之胡孝子，第三十八回郭孝子喷嚏吓虎則出于《朝野佥载》。张文虎還考出湯奏的原型為吴敬梓的朋友[[楊凱|楊凱]]，荀玫的原型“疑是姓盧”，[[平步青|平步青]]進一步指明是[[盧見曾|盧見曾]]。

==評價==
===贊揚===
鲁迅《中国小说史略》给予高度评价，“于是说部中乃始有足称讽刺之书”，“是后亦鲜有以公心讽世之书如《儒林外史》者”。鲁迅還在《中国小说的历史变迁》评价：“讽刺小说从《儒林外史》而后，就可以谓之绝响。”

[[夏志清|夏志清]]在《中國古典小說史論》第六章《儒林外史》談到“儘管《儒林》算是一部重要的反映文人學士的小說，但如果從作者對他所處的那個時代熙熙攘攘的世界所作的五光十色的描繪這方面來看，它似乎更應是一部風俗喜劇。”胡適認為後來的[[晚清四大譴責小說|晚清譴責小說]]，如《[[二十年目睹之怪現狀|二十年目睹之怪現狀]]》、《[[官場現形記|官場現形記]]》、《[[老殘遊記|老殘遊記]]》、《[[孽海花|孽海花]]》以及《[[海上花列传|海上花列传]]》，都是繼承《'''儒林外史'''》的餘緒。<ref>《[[五十年來中國之文學|五十年來中國之文學]]》</ref>

===批評===
對《儒林外史》提出批評者也不少，有學者指出《儒林外史》第十二回张铁臂吹嘘自己能将“囊中之物，加上我的药末，顷刻化为水，毛发不存矣。”的故事完全钞自[[冯翊|冯翊]]的《[[桂苑丛谈|桂苑丛谈]]》“崔、张自称侠”一条<ref>陈四益：《“人头会”传奇》</ref>。[[錢鍾書|錢鍾書]]說：“中國舊小說巨構中，《儒林外史》蹈襲依傍處最多。”同時錢鍾書指出：“近人論吳敬梓者，頗多過情之譽。”<ref>錢鍾書的上海《小說識小》：“杜慎卿访来霞士事，本之[[朱国桢|朱国桢]]《[[涌幢小品|涌幢小品]]》卷三，或《[[坚觚集|坚觚集]]》；第七回陈和甫讲李梦阳扶乩本之[[周密|周密]]《[[齐东野语|齐东野语]]》；第十三回马二先生与蘧公孙论作八股文道始于[[王充|王充]]《[[论衡|论衡]]》和《[[传灯录|传灯录]]》卷七白居易问惟宽禅师；《儒林外史》第十四回马二先生游西湖，本于[[陆次云|陆次云]]《[[湖壖杂记|湖壖杂记]]·片石居》一条；杜慎卿隔屋闻女人臭气乃《[[周书|周书]]》卷四十八萧察语；杜慎卿访来霞士事本之《[[枣林杂俎|枣林杂俎]]》。”</ref>据考，第七回有二处，第十三回一处，第十四回一处，第四十六回一处。沿袭的有情节，也有对话。另有袭用古人诗句处<ref>李国涛：〈随人也说钱与鲁〉</ref>。

==翻譯==
《儒林外史》已被翻譯成[[英語|英]]、[[法語|法]]、[[德語|德]]、[[俄語|俄]]、[[日語|日]]、[[義大利|意]]等國家的文字。可作为全世界了解中國科举制度的一部活的生動的参考。

== 作為教材 ==
《[[范進|范進]]中舉》一文是《儒林外史》的知名故事，內容取自本書第三回「[[周進|周學道]]校士拔真才，胡屠戶行凶鬧捷報」。是一名窮苦的讀書人范進因屢試不中，受盡岳父欺凌與羞辱，一夕考取[[舉人|舉人]]了，卻因為太過歡喜而瘋狂，後來眾人逼迫他岳父把范進一巴掌打醒。岳父雖救了范進，卻屢屢道歉，恭敬異常，更認為范進是天上[[文曲星|文曲星]][[轉世|轉世]]，更怕因為打了范進，死後被[[閻羅王|閻羅王]]打入[[十八層地獄|十八層地獄]]。故事充滿諷刺，非常滑稽可笑。在[[香港|香港]]，本篇是1993年至2006年度[[香港中學會考|香港中學會考]]中文科讀本篇目。在臺灣，《范進中舉》被所有版本[[高中|高中]]國文課本選入教材。

另外所有的[[國中|國中]]國文課本亦選錄《王冕的少年時代》為教材，內容取本書第一回「說楔子敷陳大義，借名流隱括全文」。

==語言學研究==
該書第一回有這麼一句“这人姓王名冕，在诸暨县乡村里住，七岁上死了父亲，……”，這句話被簡稱為“[[王冕死了父親|王冕死了父親]]”，是[[漢語語法|漢語語法]]研究爭論的重要對象<ref>[https://www.sohu.com/a/231489629_312708]</ref><ref>[http://jingxuan.guokr.com/pick/17279/]</ref><ref>[http://www.cssn.cn/zt/zt_xkzt/xkzt_yyxzt/gnzh/jblx/gnzhlxdyjcg/201609/t20160914_3202384.shtml]</ref>。
