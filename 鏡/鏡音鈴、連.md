'''鏡音鈴、連'''（{{lang-ja|'''鏡音リン・レン'''}}），通称为'''镜音双子'''，是[[CRYPTON_FUTURE_MEDIA|CRYPTON FUTURE MEDIA]]（下略作「CRYPTON」）以[[山葉株式會社|Yamaha]]的[[VOCALOID|VOCALOID 2]][[語音合成|語音合成]]引擎為基礎開發販售的虛擬[[歌手|歌手]]軟件[[角色主唱系列|角色主唱系列]]的第二作；或此軟件的印象角色（這只是軟件的象徵，不會在實際使用時出現）。於2007年12月27日發售，於2008年7月上旬發售更新版「act2」<ref name="CNET20080612">{{cite web|url= http://japan.zdnet.com/review/news/story/0,3800079080,20375189,00.htm|title= VOCALOID「鏡音鈴、連」，更新版「act2」於7月上旬提供|accessdate= 2008年6月12日|author= 岩本有平|date= 2008年6月12日|publisher= CNET Networks|deadurl= yes|archiveurl= https://web.archive.org/web/20080804083107/http://japan.zdnet.com/review/news/story/0,3800079080,20375189,00.htm|archivedate= 2008年8月4日}}</ref>。兩個版本也是開放價格，官方估計系列軟件實際價格約15,750[[日圓|日圓]]。延續著前作[[初音未來|初音未來]]的熱潮，鏡音雙子也大受歡迎。

此外，2014年12月27日，Crypton職員其一之Wat在推特表示鏡音鈴、連將被升級為[[Vocaloid|Vocaloid]] 4，同時也會增設英語聲庫。<ref>{{cite web |url=https://www.twitter.com/vocaloid_cv_cfm/status/548708196727017473 |title=Crypton職員Wat推特更新有關鏡音鈴連之相關進度 |language=ja |date=2014-12-27 }}</ref>
也會在更新act 1和Append的Power，提升控制的力度<ref>{{cite web |url=https://www.twitter.com/vocaloid_cv_cfm/status/548708731299450881|title=リンレンV4Xについて |language=ja|date=2014-12-26 }}</ref>。

鏡音鈴、連 V4X於2015年12月24日發售<ref>{{cite web |url=http://www.crypton.co.jp/mp/pages/prod/vocaloid/rinlenv4x.jsp|title=鏡音鈴、連 V4X 官網 |language=ja|date=2015-10-30 }}</ref>。

== 軟件簡介 ==
{{main|VOCALOID}}
軟件使用了[[山葉株式會社|Yamaha]]的[[VOCALOID|VOCALOID 2]][[語音合成|語音合成]]引擎。一般認為鏡音鈴、連的調整遠比初音未來困難<ref>{{cite web|url=http://gihyo.jp/lifestyle/serial/01/vocaloid/0001|title=VOCALOID（初音ミク，鏡音リン・レン）の上手な歌わせ方教えます！|trans_title= 教授讓VOCALOID（初音未來、鏡音鈴、連）唱得好的方法！|accessdate= 2008年1月13日|author= OSTER project|date=2008年1月11日 |publisher=gihyo.jp }}</ref>。

2010年12月27日發售鏡音鈴、連 Append，追加6個不同音質（感情）的聲音資料庫，鏡音鈴、連各3個，分別為鏡音鈴的power、warm、sweet，以及鏡音連的power、cold、serious。需要有鏡音鈴、連主程式才能運作。

自鏡音鈴的形像發表後，因前作[[初音未來|初音未來]]在改編作品[http://www.nicovideo.jp/watch/sm982882 Ievan Polkka]中「拿著[[葱|葱]]」的知名，而產生「鈴應該配合甚麼物品」的議論。最初比較多人提倡「黃、橙色的蔬果」如[[香蕉|香蕉]]、[[橘子|橘子]]、[[洋蔥|洋蔥]]等，但當時沒有一個物品有決定性的流行（後來兩人都有了各自的象徵物：鏡音鈴是橘子，而鏡音連是香蕉）。2007年12月1日，[[NICONICO動畫|NICONICO動畫]]用戶loadorigin上載[http://www.nicovideo.jp/watch/sm1667273 一段影片]，內容大約說鈴和黃色的[[壓路機|壓路機]]相襯，而且壓路機的擬聲「WRYYYYYYYN!!!」（取自漫畫《[[JoJo的奇妙冒險|JoJo的奇妙冒險]]》人物DIO的口頭蟬「URYYYYYYY!!!」）和鈴的名稱「{{lang|ja|リン}}（RIN）」相似。其後在CRYPTON為創作者開設的網站PIAPRO中出現大量以壓路機為題的圖片，NICONICO動畫也開始出現相關影片並流行起來，最後成為鈴（和連）的代表物品<ref>{{cite web|url= http://ascii.jp/elem/000/000/100/100009/|title= なぜ「初音ミクがネギ」で「鏡音リンがロードローラー」なのか？|trans_title= 為甚麼「初音未來是葱」、「鏡音鈴是壓路機」？|accessdate= 2008年1月17日|date= 2008年1月15日|publisher= ASCII Corporation}}</ref>。其後在公認漫畫《[[廠商非正式_初音Mix|廠商非正式 初音Mix]]》的第三話中出現鏡音鈴坐著壓路機的情節。
