{{Japanese|japanese=かがみ あきら|kana=|romaji=Kagami Akira}}
'''鏡味晃'''（本名：鏡味 晃，[[筆名|筆名]]：'''{{lang|ja|かがみ あきら}}'''，{{bd|1957年|10月22日|1984年|8月8日}}），[[日本|日本]]男性[[漫畫家|漫畫家]]。生前筆名「'''{{lang|ja|かがみ}}♪{{lang|ja|あきら}}'''」，不過通常表示是將「♪」特殊符號刪除。其它筆名還有「'''{{lang|ja|あぽ}}'''」或「'''山田榮子'''（{{lang|ja|山田栄子}}）」。

== 概要 ==
鏡味晃是1980年代前半活躍和紅極一時，英年早逝的漫畫家，投入職業創作的時間實際上只有2年半。

本名和筆名相同，不過在創作時是將漢字全部轉化成平假名，主要以[[科學幻想|科幻]]內容在{{link-ja|白夜書房|白夜書房}}的《{{link-ja|漫畫Burikko|漫画ブリッコ}}》和[[德間書店|德間書店]]的《{{link-ja|Petit Apple Pie|プチアップルパイ}}》等雑誌活躍，並以「[[美少女|美少女]][[蘿莉控|蘿莉控]]漫畫」在當年流行的漫畫題材中綻放異彩。但是，無論以哪種名義，在現在的年代而言相當於未發表的[[成人漫畫|成人漫畫]]。

[[石之森章太郎|石森Pro]]所屬時期，常受{{link-ja|鵺工作室|スタジオぬえ}}（<small>{{lang-ja|スタジオぬえ}}</small>）的邀請擔任[[機械設計_(動畫)|機械設計]]，此時積極與{{link-ja|出渕裕|出渕裕|出-{渕}-裕}}、[[結城正美|結城正美]]、[[美樹本晴彥|美樹本晴彥]]等人建立深厚關係。製作動畫電影《[[超時空要塞_愛，還記得嗎？|超時空要塞：愛，還記得嗎？]]》期間，還有跟負責該電影劇本的[[河森正治|河森正治]]進行交流，因此在機械設計的工作中曾經協助進行小道具的設計。據單行本《{{lang|ja|さよならカーマイン}}》收錄解說內容的說法，鏡味晃生前參加《[[超時空要塞|超時空要塞]]》和[[日昇動畫|日昇動畫]]作品的製作為主。另外來自[[富野由悠季|富野由悠季]]的專訪表示，曾經給鏡味晃好的評價。

1984年8月8日左右，傳出鏡味晃突然驟逝消息。根據調查結果，是在[[東京都|東京都]][[新宿區|新宿區]][[高田馬場|高田馬場]]的[[公寓大廈|自家公寓]]內發現鏡味晃的遺體，死亡時間大約是在2天前，言以之就現代的說法是孤獨死。同年，由編輯長{{link-ja|大塚英志|大塚英志}}在《漫畫Burikko》10月號發表給鏡味晃的追思文。此事對外公開之後，傳出有[[過勞死|過勞死]]、或為消除疲憊洗澡時用冰冷水[[淋浴|淋浴]]造成體溫失衡而凍死等各種說法，不過可以確定的是鏡味晃是在過度[[疲倦|疲倦]]的狀態下引發身體不適而死亡。根據本人（筆名{{lang|ja|あぽ}}）在連載的日記及短文漫畫《月刊{{lang|ja|あぽすとろふぃ}}》最終回表示「沒有工作的話會讓我陷入不安」。結果卻在26歳剛走上漫畫家之路的時候早逝。另外從親友暨作家[[佐野邦彥|佐野邦彥]]的說法，同年6月30日與鏡味晃見面時，說他的身體狀況不是很好。

== 人物 ==
鏡味晃對影音器材方面有強烈的興趣。此外對西洋音樂造詣很深，特別是[[搖滾|搖滾]]類對他影響最大，喜歡的樂團是[[是樂隊|Yes]]（[[英國|英國]][[前衛搖滾|前衛搖滾團體]]）。因此在後期的作品常在角色的衣服圖案和背景加入「Yes Lonely heart」。其它還有《{{lang|ja|はいぱぁ あくてぃぶ}}》第2回扉頁插圖來自Yes的專輯封面「{{link-ja|究極 (Yes專輯)|究極 (アルバム)|究極}}」、《{{lang|ja|レディ·キッド}}&{{lang|ja|ベヴィボウイ}}》的主角鲍伊及自稱叫大衛的設定（言以之來自於英國的歌手[[大衛·鮑伊|大衛·鮑伊]]）及許多名稱、元素大部分都來自於西洋樂曲。逝後在《{{link-ja|漫畫Burikko|漫画ブリッコ}}》1984年10月號收錄的追思文中，其連載的日記及短文漫畫《月刊{{lang|ja|あぽすとろふぃ}}》有描寫到，即使再忙，自己也會抽出時間去聽{{link-en|霍華·瓊斯|Howard Jones (musician)}}的演唱會。

還有，鏡味晃與[[結城正美|結城正美]]、{{link-ja|出渕裕|出渕裕}}等人都是[[原田知世|原田知世]]的歌迷<ref>單行本《{{lang|ja|さよならカーマイン}}》，第171頁。</ref>。

== 年譜 ==
* 1957年：10月22日出生於[[愛知縣|愛知縣]][[名古屋市|名古屋市]]。
* 1975年：{{link-ja|名古屋市立商業高等學校|名古屋市立名古屋商業高等学校}}畢業之後，進入[[獨協大学|獨協大学]][[法學部|法學部]]就讀，同時搬至[[埼玉縣|埼玉縣]][[草加市|草加市]]居住。
* 1979年：1月，首次在[[集英社|集英社]]旗下少女漫畫雜誌《{{lang|ja|{{link-ja|ぶ～け|ぶ〜け}}}}》投稿漫畫。
* 1981年：2月，進入[[石之森章太郎|石森Pro]]就職，同時也成為出道的契機。從機械設計師{{link-ja|出渕裕|出渕裕}}開始，與許多動畫和漫畫業界人士進行人脈交流。
* 1982年：在《[[魯邦三世卡里奧斯特羅之城|魯邦三世：卡里奧斯特羅之城]]大事典》（{{link-ja|Rapport|ラポート}}出版）投稿插圖正式出道。此外，在出道之前，收錄在《{{link-ja|Animec|アニメック}}》雜誌「[[戰鬥裝甲Xabungle|戰鬥機甲薩芬格爾]]曲解大辭典」的內容實質上都已刪除。12月，從石森Pro退職，全面專注漫畫的連載。
* 1983年：在雜誌《[[哥白尼21|哥白尼21]]》（[[公文式|公文式]]出版）連載的《{{lang|ja|ワンダートレック}}》是鏡味晃的第一部連載漫畫。在進行連載的前後，也在其它雜誌發表短篇漫畫或插圖。
* 1984年：8月1日<ref>來自底頁的記述。</ref>，第一本單行本漫畫《{{lang|ja|ワインカラー物語}}》（{{lang|ja|あぽ}}名義，{{link-ja|白夜書房|白夜書房}}）發行。卻是他的最後一部和唯一的著作。8月5日左右身體不適。8月8日下午被人發現陳屍在[[公寓大廈|自家公寓]]。年僅26歲。戒名「鏡譽晃彩紫雲居士」。
* 1985年：單行本《{{lang|ja|さよならカーマイン}}》發行（Rapport出版）。內容收錄了他的早期作品「{{lang|ja|故郷を求めて}}」、「{{lang|ja|クラリスメモリー未来編}}」及未完結的連載「{{lang|ja|はいぱぁ あくてぃぶ}}」等。
* 1985年～1986年：遺作集《鏡味晃精選》《鏡味晃精選2》發行（編集／發行：青柳誠）。
* 1994年：舉辦逝世10週年追思紀念會，紀念冊《{{lang|ja|かがみのくに}}》由{{link-ja|池內一二三|うたたねひろゆき}}等人進行頒布。
* 2004年：12月，發行《{{lang|ja|ワインカラー物語―かがみあきら選集}}》，由[[角川書店|角川書店]]出版。
* 2004年～2005年：發行鏡味晃特集，由《{{link-ja|漫畫新現實|コミック新現実}}》編輯部在同雜誌Vol.1～3、5收錄。
