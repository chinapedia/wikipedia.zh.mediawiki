'''刑事訴訟法'''（英語：{{lang|en|'''criminal procedure'''}}）是規範刑事案件之偵查行為、審理行為、判決行為、執行行為及其偵查組織、審理組織、判決組織、執行組織之規範。此為廣義的刑事訴訟法，也就是現行的法規範。目的為藉由程序之進行，確認國家對被告的刑罰權存在與否，並藉程序的進行，確保國家行政訴追、司法審判之正當性、合適性。

== 組織 ==

刑事訴訟法實為整體刑事程序法中之核心，而偵查、起訴、執行的檢察官組織及審判的法院組織，在無特別程序適用時，適用法院組織法之規範。刑事訴訟法在偵查、審判、執行階段，各按該階段的本質，規範行為之組織。偵查程序中原則上為兩面關係，即所謂之[[糾問式訴訟|糾問制度]]，由偵查主體之檢察官與被告組成。訴訟中為三面關係，即所謂之控訴制度，在公訴程序由公訴人、被告、法院組成；在自訴程序由自訴人、被告、法院組成。在執行程序中，由代表國家行政權之檢察官與受刑人組成。

== 行為 ==

主要包括檢察官、司法警察所為的偵查行為、追訴程行為、自訴人的自訴行為、法院裁判行為、被告防禦行為。

== 原則 ==
主要有四，為國家追訴原則、控訴原則、法定原則、調查原則。
