'''罗玉凤'''（{{bd|1985年|9月23日|||catIdx=L罗}}），昵称'''凤姐'''，[[中华人民共和国|中华人民共和国]][[重庆市|重庆市]][[綦江县|綦江县]][[赶水镇|赶水镇]]人，是中国大陆的一位[[网络红人|网络红人]]。她因在上海的征婚启事以及江蘇衛視情感類訪談節目《人間》的争议言论而成名，并在之后参与了许多公众活动，2010年，罗玉凤前往美国，并在当地多家美甲店打工謀生，直到2014年年底。

== 生平 ==
=== 早年生涯 ===
1985年9月23日，罗玉凤出生在中华人民共和国重庆綦江县赶水镇的一个农民家庭。<ref name="騰訊">{{cite web |title=凤姐的逆袭 |url=http://cq.qq.com/zt2015/luoyufeng/index.htm |website=腾讯大渝网 |accessdate=2018-08-06}}</ref>7岁時，罗玉凤父母離異，從此罗玉凤和母親一起生活，並再也沒有見過她的父親。她自稱自己“从小在人们的白眼中长大。”<ref name="環球">{{cite web |title=凤姐回应网络炒作：我出名全靠个人奋斗 |url=http://society.huanqiu.com/roll/2010-06/844582_2.html |website=环球网 |accessdate=2018-08-06}}</ref>由於家裡窮，初中畢業之後，她在[[綦江师范学校|綦江师范学校]]就讀，並获得了中师文凭，之後又在[[重庆教育学院|重庆教育学院]]教院获得大专文凭。羅玉鳳曾在重庆奉节黄泗小学做了两年的老师。但後來辞职帶上自己的全部1萬元積蓄來到上海。她表示“因为我觉得上海有更好的发展方向，还有另一个目的，就是找个男朋友结婚”。为找工作，羅玉鳳向各行各业投出了一万多份简历，最终在上海的一个[[家乐福|家乐福]]超市擔任收銀員，月收入不超過1500元。<ref name="環球"/><ref name="騰訊"/>

=== 走红历程 ===
2009年11月20日，罗玉凤在社区发表征婚帖子《我想找个北大清华男结婚》，列了“北大清华硕士、经济学专业、有国际视野、身高1米76至1米83、无生育历史、东部沿海户籍、年龄25至28岁”這七个征婚条件，并表示自己“身高1米46，平时穿高跟鞋1米53，大专文凭、博览群书、较为狂妄。”幾天後，她曬出了自己的征婚結果，聲稱許多大型金融公司驻中国区首席执行官向她表达爱意，但自己觉得他们“年老色衰”，所以不愿意。<ref name="搜狐">{{cite web |author1=宋焘 |title=网络红人凤姐征婚 前后三百年“没有人超过我” |url=http://cul.sohu.com/20110817/n316597479.shtml |website=搜狐網 |publisher=金羊网-新快报 |accessdate=2018-08-06}}</ref>之後，羅玉鳳在上海地铁站分發了1300份征婚传单。<ref>{{cite web |title=凤姐超市工作遭围观 网友为其创作首张专辑 |url=http://news.ifeng.com/society/2/201002/0218_344_1548824.shtml |website=凤凰网 |publisher=新快网 |accessdate=2018-08-06}}</ref>這些傳單引起了上海当地媒体的注意。隨後，她在接受上海媒体访问时表示她征婚条件中最重要的一条是聪明。在媒体报道之後，罗玉凤在上海本地论坛里得到了“陆家嘴征婚女”的稱號，並因此小小走红了一段時間。2010年年初，羅玉鳳登上[[江蘇衛視|江蘇衛視]]情感类访谈节目《人间》“我要嫁白马王子”一期。在節目中，羅玉鳳的男友小吳向節目組求助，稱羅玉鳳嫌棄自己條件不好，想要分手。羅玉鳳在節目中表示“没有男孩子能在智商上超越我了，只能在身高和相貌上来弥补。”“我九岁博览群书，二十岁达到顶峰。我现在都是看社会人文类的书，例如《[[知音_(杂志)|知音]]》《[[故事会|故事会]]》……往前推三百年，往后推三百年，总共六百年没有人超过我。”節目播出之後，許多網友被羅玉鳳的言論表示震驚。節目影片越傳越廣，网友们給罗玉凤取外號为“凤姐”。<ref name="现代">{{cite web |author1=李孟漪 |title=罗玉凤：1米46女生只找北大清华硕士 |url=http://news.sohu.com/20100203/n270010547.shtml |website=搜狐網 |publisher=现代快报 |accessdate=2018-08-06}}</ref>

=== 公眾活動 ===
羅玉鳳成名之後，“信凤姐，得自信”成為了網路熱詞，“罗玉凤小组”和“罗玉凤[[百度贴吧|贴吧]]”成立，百科亦出现了“罗玉凤”條目。而羅玉鳳之前發過的帖子和博客也成為了热帖。<ref name="现代"/>之后，罗玉凤的公众活动也越来越多。节目方面，她在[[星空卫视|星空卫视]]脱口秀节目《Lady呱呱》担任特别来宾。<ref>{{cite web |title=凤姐将加盟《Lady呱呱》做主持 扬言挤走天心 |url=http://ent.sina.com.cn/v/m/2010-04-29/18512945146.shtml |website=新浪网 |accessdate=2018-08-06}}</ref>她还做客青海卫视新闻脱口秀栏目《嘎嘣爆米花》，并在节目中承认《人间》的男友是演员。<ref>{{cite web |title=凤姐做客《嘎嘣爆米花》 承认两个男友都是演员 |url=http://ent.qq.com/a/20100520/000356.htm |website=腾讯网 |accessdate=2018-08-06}}</ref>选秀节目方面，她报名选秀节目《[[花儿朵朵_(青海卫视)|花儿朵朵]]》，并在沈阳赛区拿到了资格卡。<ref>{{cite web |title=罗玉凤：我永远端坐云端以上 |url=http://news.sina.com.cn/s/qw/2015-12-10/doc-ifxmpnqi6274974.shtml |accessdate=2018-08-06}}</ref>之后，她又参加了东方卫视选秀节目《[[中国达人秀|中国达人秀]]》，在比赛中，罗玉凤带着歌词“小抄”演唱歌曲《[[爱情买卖|爱情买卖]]》，但不到30秒就被评委喊停。评委方俊认为她没有为比赛做好充分准备，而罗玉凤认为自己就是最好的“达人”。最终三位评委一致裁定“凤姐”待定。赛后，罗玉凤在接受媒体采访时遭到了一男子砸鸡蛋。<ref>{{cite web |title=“凤姐”现身“中国达人秀” 遭人扔鸡蛋 |url=http://ent.sina.com.cn/s/m/2010-05-17/00172959567.shtml |website=新浪网 |accessdate=2018-08-06}}</ref>此外，罗玉凤还参与了宁夏卫视节目《时尚星达人》以及山东卫视节目《快乐向前冲》。<ref>{{cite web |title=凤姐做客《时尚星达人》 变“Q版”林志玲 |url=http://ent.sina.com.cn/v/m/2010-06-11/12272985138.shtml |website=新浪网 |accessdate=2018-08-06}}</ref><ref>{{cite web |title=凤姐着比基尼参加山东综艺节目快乐向前冲大秀身姿 |url=http://news.iqilu.com/zongyi/20100615/258270.shtml |website=齐鲁网 |accessdate=2018-08-06}}</ref>罗玉凤亦得到了广告代言的机会。1Q73渭丁香为罗玉凤摄制了A、B兩版广告，并付给罗玉凤30万酬劳。<ref>{{cite web |title=鳳姐胃藥廣告　睇完想嘔唔駛食 |url=https://hk.news.appledaily.com/international/daily/article/20100803/14305233 |website=蘋果日報 |accessdate=2018-08-06}}</ref><ref>{{cite web |title=鳳姐代言史上最雷人胃藥廣告 |url=http://news.wenweipo.com/2010/08/02/IN1008020164.htm |website=文匯網 |accessdate=2018-08-06}}</ref>她还拍摄了一组写真，被网友称赞像[[蔡依林|蔡依林]]。<ref>{{cite web |title=凤姐新造型出炉 网友盛赞像蔡依林 |url=http://ent.sina.com.cn/s/m/2010-04-05/04412919027.shtml |website=新浪网 |accessdate=2018-08-06}}</ref>

=== 移居美国 ===
2010年11月10日，罗玉凤在个人微博上表示自己的[[美国签证政策|美国签证]]通过，从上海来到纽约。罗玉凤在2015年接受纽约时报中文网的采访时表示，自己没有绿卡，但有临时居留身份、工卡和工作许可。<ref name="NYT">{{cite web |title=纽约美甲业行业起底：凤姐有话说 |url=https://cn.nytimes.com/people/20150514/tc14fengjie/ |website=纽约时报中文网 |accessdate=2018-08-06|date=2015-05-14}}</ref>同年12月，罗玉凤在[[美国中文电视|美国中文电视台]]应聘的视频在中文网络上热传。该台采访部主任张碱认为罗玉凤非常聪明，“思路特别清晰（……）还挺有想法的。”罗玉凤通过了电视台第一轮面试。<ref>{{cite web |title=凤姐应聘美国电视台被赞思想独立 网友疑炒作 |url=http://ent.sina.com.cn/s/m/2010-12-06/08223168259.shtml |website=新浪网 |accessdate=2018-08-06}}</ref>但在随后的第二轮面试中，罗玉凤应聘失败，并发微博表示：“那个所谓电视台招聘从一开始就是骗局。”<ref>{{cite web |title=凤姐想到美国当记者 疑似出局微博威胁自杀 |url=http://news.ifeng.com/society/2/detail_2010_12/07/3374085_0.shtml |website=凤凰网 |accessdate=2018-08-06}}</ref>在抵达纽约后的四年中，罗玉凤辗转于纽约多家美甲店打工谋生，直到2014年年底。<ref name="NYT"/>2015年7月15日，罗玉凤得到凤凰新闻客户端邀请，成为该平台签约主笔。<ref>{{cite web |title=罗玉凤女士成为凤凰新闻客户端签约主笔 |url=http://news.ifeng.com/a/20150716/44181997_0.shtml |website=凤凰网 |accessdate=2019-02-16}}</ref>

== 评价 ==
2010年7月2日，[[国家广播电影电视总局|国家广电总局]]办公厅主任、新闻发言人朱虹做客[[华中师范大学|华中师范大学]]桂子山文人讲坛，其中朱虹谈到，凤姐是低俗文化，应该坚决叫停<ref>{{cite news|url=http://media.people.com.cn/GB/40606/12033167.html|title=广电总局发言人：“凤姐”这种低俗文化 应叫停|publisher=人民网|date=2010-07-02|accessdate=2010-07-28}}</ref>。

2011年9月27日，美国《人物》杂志网站发表了题为《走近中国最受讨厌的现实派明星》的文章，称凤姐为中国最惹人讨厌明星<ref>{{cite news|url=http://yule.sohu.com/20110930/n321111762.shtml|title=凤姐登美杂志《人物》|publisher=搜狐娱乐|date=2011-09-30}}</ref>。

== 争议言论 ==
2011年1月22日，罗玉凤在其[[新浪微博|新浪微博]]称要与[[陈冠希|陈冠希]]结婚，引来骂声一片<ref>{{cite web|url=http://ent.ifeng.com/idolnews/detail_2011_01/22/4402064_0.shtml|title=凤姐称要和陈冠希结婚 网友：陈冠希已经给你吓死了|publisher=凤凰网|date=2011-01-22|accessdate=2011-02-17}}</ref>。

2011年7月27日，罗玉凤在其新浪微博发表言论指「[[温州动车追尾事故|温州动车追尾事故]]」遇难者是「死前都听说过大名鼎鼎的凤姐了。所以他们也死得其所了。另外，他们为中国的人口优化作出了杰出的贡献。所以他们的死绝对是重于泰山的，是死得光荣的。本人对此表示强烈的感谢」，此言论一出立即遭致网友的强烈抨击，纷纷表示反感。不久，罗玉凤又称：「自己是不怕死的，这样的微博自己还会继续发。」<ref>{{cite news|url=http://news.xiancn.com/content/2011-07/27/content_2476865.htm|title=凤姐微博称遇难者为人口优化做贡献 粉丝狂掉|publisher=西安新闻网|date= 2011-07-27}}</ref>

2011年8月17日，罗玉凤在其新浪微博称要「一把火烧了[[美国移民局|美国移民局]]」<ref>[http://www.guancha.cn/society/2013_07_28_161651.shtml 相比吴虹飞 当年要烧移民局的凤姐为什么全身而退], 华商報, 2013-07-28</ref>。
