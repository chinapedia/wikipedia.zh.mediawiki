'''罗林泉'''（{{bd|1956年|4月||}}），[[中华人民共和国|中华人民共和国]]外交官，历任中国驻希腊大使、驻爱尔兰大使。

==求学==
罗林泉1956年4月出生，1972年毕业于[[苏州市第十二中学|苏州市第十二中学]]。<ref>{{cite web |url=http://www.szs12zx.com/html/guangcaiban/20120426/30.html |title=罗林泉大使走进光彩班 |publisher=苏州市第十二中学校 |date=2010-10-14 |accessdate=2013-06-28}}</ref>1980年，毕业于[[北京外国语学院|北京外国语学院]]英语系。

==外事工作==
毕业后，罗林泉进入[[中华人民共和国外交部|外交部]]工作，任礼宾司科员。1984年，任中国驻纽约总领馆副领事。1986年至1997年，历任外交部礼宾司随员、三等秘书、副处长、处长、参赞。1997年，任外交部驻香港特别行政区特派员公署参赞。1998年后，任礼宾司副司长、司长。2007年7月，接替[[田学军|田学军]]，担任[[中国驻希腊大使列表|中华人民共和国驻希腊大使]]。<ref>{{cite web|author=中华人民共和国外交部|title=中国驻希腊历任大使|url=http://www.mfa.gov.cn/chn/pds/ziliao/wjrw/2167/2172/t9263.htm|publisher=中华人民共和国外交部|accessdate=2012-03-07}}</ref>任职期间，
恰逢[[2008年夏季奧林匹克運動會火炬接力|北京奥运会圣火传递]]，罗林泉为火炬接力的第五段的火炬手，成为首位参与奥运火炬接力的中国大使。<ref>{{cite web|url=http://news.xinhuanet.com/sports/2008-03/23/content_7840930.htm|title=罗林泉大使:希腊境内前五棒火炬手三人来自中国|publisher=新华网|date=2008-03-23|accessdate=2013-06-28|deadurl=yes|archiveurl=https://web.archive.org/web/20080326211907/http://news.xinhuanet.com/sports/2008-03/23/content_7840930.htm|archivedate=2008-03-26}}</ref>2011年，接替[[刘碧伟|刘碧伟]]，担任[[中国驻爱尔兰大使列表|中华人民共和国驻爱尔兰大使]]。<ref>{{cite web|author=中华人民共和国外交部|title=中国驻爱尔兰历任大使|url=http://www.mfa.gov.cn/chn/pds/ziliao/wjrw/2167/2172/t9246.htm|publisher=中华人民共和国外交部|accessdate=2012-03-07}}</ref>

==私人生活==
罗林泉已婚，育有一子。

2011年离任驻希腊大使前，罗林泉获授[[凤凰勋章|希腊凤凰大司令勋章]]，成为首位受勋的中国驻希腊大使。<ref>{{cite web |url=http://gr.china-embassy.org/chn/xwdt/t839759.htm |title=中国驻希腊大使罗林泉获授希腊“凤凰军团大统帅勋章” |publisher=中华人民共和国驻希腊共和国大使馆 |date=2011-07-15 |accessdate=2013-06-28}}</ref>
