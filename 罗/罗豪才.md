'''罗豪才'''（{{bd|1934年|3月|2018年|2月12日|catIdx=R}}），祖籍[[福建省|福建省]][[泉州市|泉州市]][[安溪县|安溪县]]，出生于[[新加坡|新加坡]]。他是[[中华人民共和国|中华人民共和国]]著名法学家、归国[[华侨|华侨]]、教育家和社会活动家，中国现代行政法学的开拓者与奠基人。[[中国致公党|中国致公党]]的杰出领导人，[[中国人民政治协商会议|中国人民政治协商会议]]第九届、十届全国委员会副主席，[[中国致公党|中国致公党]]第十一届、十二届中央委员会主席，[[中国共产党|中国共产党]]党员<ref name=rmw/>。

==生平==
===早年===
1934年3月，罗豪才出生于[[新加坡|新加坡]]，幼年时不少亲友被占领新加坡的日本侵略者杀害，所以他自幼对[[日本帝国主义|日本帝国主义]]心里非常痛恨。

1945年8月[[日本投降|日本投降]]后，罗豪才重获读书机会，随后家庭经济情况好转，跳级考入中学。此后他参加了反对英国殖民统治的斗争。1951年4月14日，17岁的罗豪才被新加坡的英国殖民当局逮捕，监禁一年零三个月至1952年7月出狱。1952年7月，罗豪才因为“出生证”丢失而被新加坡的英国殖民当局驱逐，他由此返回[[中华人民共和国|中华人民共和国]]<ref name=rmw>{{Cite web |url = http://www.people.com.cn/GB/shizheng/252/9667/9685/20021129/877668.html|title = 罗豪才简历 |author =  |date = 2002-11-29 |publisher =人民网  |language =  |accessdate =  |quote =  }}</ref><ref name=pp>{{Cite web |url = http://www.thepaper.cn/newsDetail_forward_1996193 |title = 法学大家罗豪才逝世：在最高法任职时曾赢得“人民法官”尊称 |author =  |date = 2018-02-12 |publisher = 澎湃新闻 |language =  |accessdate =  |quote =  }}</ref>。

1952年至1956年，他先后在[[广东省|广东省]][[广州|广州]]知用中学和[[江苏省|江苏省]][[无锡市|无锡市]]第一中学学习，1956年考入[[北京大学|北京大学]]法律系学习。

===北大生涯===
1960年大学毕业后，因为北京大学法律系计划开设[[东南亚|东南亚]]国家法律课程，罗豪才在新加坡学过[[马来语|马来语]]，所以留校任教，历任北京大学法律系助教、讲师、系副主任。1984年至1985年赴[[美国|美国]][[哥伦比亚大学|哥伦比亚大学]]进修，他的联系导师是美国行政法奠基人欧内斯特·盖尔霍恩。1985年至1986年，任北京大学法律系副主任、副教授<ref name=rmw/><ref name=pp/>。1990年代初，罗豪才与[[袁曙宏|袁曙宏]]等人率先提出“现代行政法的平衡理论”。1990年10月1日，《中华人民共和国行政诉讼法》正式施行<ref name=pp/>。

1986年至1992年，罗豪才任北京大学副校长、教授，北京市侨联主席，[[中国侨联|中国侨联]]副主席，北京市人大常委会委员。1992年至1993年，任[[中国致公党|致公党]]中央副主席，中国侨联副主席，[[中国法学会|中国法学会]]副会长，北京市人大常委会委员，北京大学副校长，北京市侨联主席。1993年至1995年，任致公党中央副主席，[[全国政协|全国政协]]副秘书长，中国侨联副主席，中国法学会副会长，北京市人大常委会委员，北京大学副校长<ref name=rmw/><ref name=pp/>。

===法官生涯===
1995年至1996年，罗豪才任致公党中央副主席，[[最高人民法院|最高人民法院]]副院长、[[最高人民法院审判委员会|审判委员会]]委员，全国政协副秘书长，中国侨联副主席，中国法学会副会长。1996年至1997年，任致公党中央副主席，最高人民法院副院长、审判委员会委员，中国侨联副主席，中国法学会副会长。1997年至1998年，任致公党中央主席，最高人民法院副院长、审判委员会委员，中国侨联副主席，中国法学会副会长。

1995年6月，罗豪才任最高人民法院副院长后，主管行政审判和法官培训等工作，常为协调、调查案件亲自出马，还亲任审判长主持审理工作。罗豪才赢得了“人民法官”的尊称。作为全国政协和中国侨联的领导，他还关心海外留学人员生活，创造条件吸引他们归国创业。他组织了中央调研组走访[[中华人民共和国科技部|中华人民共和国科技部]]、[[中华人民共和国人事部|人事部]]、[[中华人民共和国教育部|教育部]]等单位，并考察调研[[北京|北京]]、[[上海|上海]]、[[苏州|苏州]]、[[南京|南京]]、[[大连|大连]]等地的“创业园”。1999年，罗豪才在全国政协大会发言《筑巢引凤，精心培育，促进留学人员创业园的健康发展》引起中央重视<ref name=pp/>。

===晚年===
1998年3月，64岁的罗豪才当选第九届[[全国政协副主席|全国政协副主席]]，成为[[党和国家领导人|党和国家领导人]]。同时还兼任中国致公党中央主席、中国法学会副会长等职务<ref name=rmw/><ref name=pp/>，2000年当选[[中国和平统一促进会|中国和平统一促进会]]会长。

2003年3月，罗豪才在全国政协十届一次会议上当选为第十届全国政协副主席。2007年5月10日，罗豪才当选为中国[[人权|人权]]研究会会长，任期至2016年12月<ref name=rmw/><ref name=pp/>。2005年12月8日，[[北京大学法学院|北京大学法学院]]“软法研究中心”成立，罗豪才以北京大学法学院教授身份出任中心名誉主任。他是第八届全国政协常务委员，第九届、第十届全国政协副主席<ref name=rmw/>。

2008年3月，74岁的罗豪才卸任全国政协副主席职务。直到2016年，罗豪才仍然活跃在法律专业领域活动中，就“[[互联网|互联网]]时代的人权与发展”等议题发表看法<ref name=pp/>。

2018年2月12日9时02分，罗豪才因病在[[北京|北京]]逝世，享年84岁<ref name=pp/>。2月24日上午，罗豪才的遗体在北京[[八宝山革命公墓|八宝山革命公墓]]火化。
