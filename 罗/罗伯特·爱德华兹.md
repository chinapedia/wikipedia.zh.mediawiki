'''罗伯特·杰弗里·爱德华兹'''爵士，[[大英帝國勳章|CBE]]，[[皇家學會院士|FRS]]（{{lang-en|Sir '''Robert Geoffrey Edwards'''}}，{{bd|1925年|9月27日|2013年|4月10日|catIdx=Edwards, Sir Robert}}），[[英国|英国]][[生理学|生理学家]]，[[生殖医学|生殖医学]]的先驱者，因「開發[[体外人工受精|体外受精技术]]」的成就被授予2010年[[诺贝尔生理学或医学奖|诺贝尔生理学或医学奖]]。<ref name="nobel_2010">{{cite web |url=http://nobelprize.org/nobel_prizes/medicine/laureates/2010/ |title=The Nobel Prize in Physiology or Medicine 2010 |publisher=[[诺贝尔基金会|诺贝尔基金会]] |date= |accessdate=2010年10月4日 |language=en }}</ref>

== 簡介 ==
罗伯特·爱德华兹出生于英格兰曼彻斯特，曾參與[[第二次世界大戰|第二次世界大戰]]。服滿兵役后进入[[威尔士大学|威尔士大学]]和[[爱丁堡大学|爱丁堡大学]]学习[[生物学|生物学]]，1955年获博士学位，论文内容为小鼠胚胎發育。1958年成为[[英国国立医学研究所|英国国立医学研究所]]的研究人员，开始对人类授精过程的研究。<ref name="tianjimgwe">{{cite web |url=http://www.tianjinwe.com/tianjin/tbbd/201010/t20101004_1977749.html|title=2010年诺贝尔医学奖得主罗伯特·爱德华兹个人简历|publisher=天津网|date=2010-10-04|accessdate=2010年10月5日|language=zh-hans}}</ref>

爱德华兹自1963年在[[英國|英國]][[剑桥大学|剑桥大学]]供职，与[[帕特里克·斯特普托|帕特里克·斯特普托]]（1988年去世）共同研發体外人工受精技术（俗稱试管婴儿技术），1978年世界上第一个试管婴儿誕生。随后二人共同创立了全球首个体外受精研究中心——[[伯恩霍尔生殖医学中心|伯恩霍尔生殖医学中心]]。<ref>{{cite web |url=http://www.zaobao.com/wencui/2010/10/others101005se.shtml|title=试管婴儿之父获得诺奖 当初合作者1988年去世 |publisher=联合早报网|date=2010-10-05|accessdate=2010年10月5日|language=zh-sg}}</ref>

因为在人类试管授精(IVF)疗法上的卓越贡献，罗伯特·G·爱德华兹获得2010年度诺贝尔生理学或医学奖。他的贡献使治疗不育症成为可能，包括全球超过10%的夫妇在内的人类因此获益匪浅。

爱德华兹的夫人罗丝的外祖父是著名物理学家、诺贝尔奖获得者[[欧内斯特·卢瑟福|欧内斯特·卢瑟福]]。

逝世前，爱德华兹是授精研究领域多本顶尖期刊的编辑，並在[[剑桥大学|剑桥大学]]任名誉退休教授。<ref name="tianjimgwe"/>

2013年4月10日于睡眠中去世，享年87岁<ref>[http://www.bbc.co.uk/ukchina/simp/uk_life/2013/04/130410_life_tubebaby_death.shtml BBC英伦网-2013年4月10日-“试管婴儿之父”爱德华兹去世]</ref>。
