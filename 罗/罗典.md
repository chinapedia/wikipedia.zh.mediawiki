'''罗典'''（{{bd||1719年|1808年|catIdx=L}}），字'''微五'''，号'''慎斋'''，[[湖南|湖南]][[湘潭|湘潭]]人。[[清朝|清朝]]学者，乾隆丁卯鄉試解元，辛未進士。曾任[[岳麓书院|岳麓书院]]山长。

== 简介 ==
[[康熙|康熙]]五十七年（1718年），罗典生于[[湖南|湖南]][[湘潭|湘潭]]。<ref name="湖湘人物：罗典 ">{{cite web |url=http://www.library.hn.cn/hxrw/gdrw/luodian.htm|title=《湖湘人物：罗典》|accessdate=|work=|publisher=湖南图书馆官方网|language=zh-hans}}</ref>

[[乾隆|乾隆]]十二年（1747年）湖南[[乡试|乡试]]第一名。<ref name="湖湘人物：罗典 "/>

[[乾隆|乾隆]]十六年（1751年）罗典考中[[进士|进士]]，选庶吉士，授职编修，转职[[御史|御史]]，歷官吏科、工科[[给事中|给事中]]、[[四川|四川]][[学政|学政]]，升[[鸿胪寺|鸿胪寺]][[少卿|少卿]]。曾两任会试考官，两次主持[[河南|河南]]乡试。后来因为母亲年迈而辞官还乡，从此脱离官场。<ref name="湖湘人物：罗典 "/>

[[乾隆|乾隆]]四十七年（1782年），罗典出任[[岳麓书院|岳麓书院]]山长，并且五次连任，主持[[岳麓书院|岳麓书院]]长达27年。岳麓书院在他主持下，发展达到了最高峰。<ref name="湖湘人物：罗典 "/>

[[嘉庆|嘉庆]]十三年（1808年），罗典因病逝世于[[岳麓书院|岳麓书院]]，终年九十一岁。<ref name="湖湘人物：罗典 "/>

== 学术思想 ==
罗典精通[[汉学|汉学]]，采用[[汉代|汉代]][[经学家|经学家]]“字批而句疏”的[[训诂|训诂]]诠释之法，并有所创新。擅长制艺，精通[[经学|经学]]，亦能作诗。

罗典一生以培养人才为己任，曾提出“非专衡文，当以育才为本”的主张。担任[[岳麓书院|岳麓书院]]山长后，又提出“坚定德性，明习时务”的育才思想。罗典十分重视[[美育|美育]]教育和[[岳麓书院|岳麓书院]]的环境美化，在书院后面建造园林，栽植花木；挖池塘，养游鱼。

== 评价 ==
[[严如熤|严如熤]]评价罗典说：“先生虽以制艺一世，而精神专注则在经。其治经也，以古人简质文字，无间剩，即经诂经，字批而句梳之，既皆有确切注脚，则通之一章，又通之全篇。全经有所窒，则废寝食，夜以继日，必得其融贯而后安。”，“立说时出新义……实阐古人不传之秘。”
