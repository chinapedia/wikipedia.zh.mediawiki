'''罗得岛'''（{{lang-gre|Ρόδος}} <sup>{{lang|el|''Ródos''}}</sup> {{IPA-el|ˈroðos|}}）是[[爱琴海|爱琴海]]上的一个[[岛屿|岛屿]]，位于[[佐则卡尼萨群岛|佐则卡尼萨群岛]]的最东端，它也是该群岛的主岛和[[希腊|希腊]]所辖最东的岛屿，距离[[土耳其|土耳其]]仅18公里。其名来自于[[古希腊语|古希腊语]]中的「{{lang|el|Ρόδον}}」（[[玫瑰|玫瑰]]），当时指的是今天的[[朱槿|朱槿]]；島徽則是一隻跳躍的鹿。全岛面积为1,398平方公里，居民约为125,000人，其中约一半住在岛北部的[[羅得市|羅得市]]，該市也是全島的首府和旅游中心。

罗得岛是爱琴地区文明的起源地之一，有相当古老的关于[[忒尔喀涅斯|忒尔喀涅斯]]的神话。[[品达|品达]]的诗中称罗得岛是太阳神[[赫利俄斯|赫利俄斯]]和女神[[罗得|罗得]]结合的产物。在[[希腊化时期|希腊化时期]]，这个岛屿的鼎盛时期，人们竖立起一个巨大的[[羅得島太陽神銅像|太阳神銅像]]，成为古代[[世界七大奇迹|世界七大奇迹]]之一。在[[十字軍東征|十字軍東征]]期间，[[医院骑士团|医院骑士团]]占领了该岛，并改名为“罗得岛骑士团”，成为了岛屿历史上重要的存在，他们在岛上留下了许多中世纪的建筑。罗得岛以它的文学气质和独特的历史内涵无愧为爱琴诸岛中的一颗明珠。

== 地理 ==
[[File:rhodos_topo.png|250px]]
=== 位置 ===
罗得岛位于[[地中海|地中海]]中的爱琴海的南端。

罗得岛约长78千米，宽35千米，其中心离[[雅典|雅典]]约430千米。位于岛西北岸的罗得岛国际机场离土耳其的东南岸只有17.5千米距离，这是罗得岛离[[小亚细亚|小亚细亚]]最近的地点。罗得岛以西9千米处为[[哈尔基岛|哈尔基岛]]，7千米处为[[阿里米亚岛|阿里米亚岛]]，此外还有一些无人居住的小岛。

=== 地形 ===
[[File:Rhodos_Landschaft_bei_Lindos.jpg|250px]]
尤其岛内部非常陡峭，其最高点达海拔1215米。山地一直延伸到海岸，在许多地方形成悬崖入海。岛的北端和南端分别平坦。南端形成一个[[半岛|半岛]]。

== 居民点 ==
罗得岛上有10个城镇以及许多小村庄。

岛上约半数的居民住在罗得市，罗得市也是岛的首府和旅游中心，其老城、王宫以及雄伟的城墙被[[联合国教育科学文化组织|联合国教育科学文化组织]]设定为[[世界遗产|世界遗产]]。王宫里有富丽堂皇的[[鑲嵌畫|鑲嵌畫]]和一份世界著名的[[拉奥库恩|拉奥库恩]]雕像的拷贝。

== 传说 ==
按照[[希腊神话|希腊神话]]罗得岛产於海中。[[波塞冬|波塞冬]]是岛的保护人，他不在的时候他的女儿[[海仙女|海仙女]][[罗得|罗得]]照管罗得岛。

[[宙斯|宙斯]]在分封他的领域的时候太阳神[[赫利俄斯|赫利俄斯]]正在巡视天穹，普照大地，因此宙斯将他忘了。赫利俄斯回到[[奥林匹斯山|奥林匹斯山]]的时候宙斯本来打算重分他的领域，但是赫利俄斯对宙斯说，宙斯只要将他在巡视天穹的时候看到的一座美丽的小岛分给他，他就满意了。赫利俄斯以海仙女罗得命名了这座岛，因为他爱上了这位海仙女。

后来赫利俄斯与罗得结婚，两神有七个儿子，其中的长子又有三个儿子，而这三个儿子则分别建立了岛上最古老的三座城市。

== 历史 ==
[[新石器时代|新石器时代]]就已经有人在罗得岛上定居了。[[青铜器时代|青铜器时代]]（约前2800年至前2000年）岛上已经有相当大的居民点了。前16世纪[[米诺斯文明|米诺斯人]]在岛上定居，前15世纪[[亚该亚人|亚该亚人]]在岛上定居，前11世纪[[多里安人|多里安人]]来到岛上，他们在岛上建立了今天最古老的三座城市，使得罗得岛的地位大增。罗得岛上的城市与[[科斯岛|科斯岛]]以及小亚细亚的城市[[库尼多斯|库尼多斯]]和[[哈利卡纳苏斯|哈利卡纳苏斯]]联盟组成了多里安海上城市同盟。

[[荷马|荷马]]说罗得岛在[[特洛伊战争|特洛伊战争]]中提供了九艘船。

前546年[[呂底亞|呂底亞]]被[[波斯|波斯]]战败后罗得岛陷入波斯的势力范围。罗得岛参加了小亚细亚的希腊城邦反抗波斯的战争（前500年至前494年），结果被波斯吞并。前479年在[[波希战争|波希战争]]中波斯战败后罗得岛上的三座城市加入[[提洛同盟|提洛同盟]]。

前408年罗得岛上的三座城市合并，在岛的北端建立了一座新城市，即今天的罗得市。当时希腊最有名的城市规划家[[米利都|米利都]]的[[希波丹姆斯|希波丹姆斯]]规划了这座新城市。一个世纪后这座新城市就已经超过雅典成为希腊最大的商城了。
[[File:Rhodes0211.jpg|thumb]]

罗得岛上的[[羅得島太陽神銅像|太阳神-{}-雕塑]]高34米，是世界七大奇迹之一。前227年它被地震摧毁。

前404年提洛同盟解散后罗得岛完全独立。前387年它得以对波斯保持其独立。前364年它与[[底比斯_(希臘)|底比斯]]联盟抵抗[[第二次雅典联邦|第二次雅典联邦]]。

在[[马其顿|马其顿]]对希腊的战争中罗得岛站在[[腓力二世_(马其顿)|腓力二世]]的一边。[[亚历山大大帝|亚历山大大帝]]死后罗得岛重新获得独立。

前42年[[盖乌斯·卡西乌斯·朗基努斯|盖乌斯·卡西乌斯·朗基努斯]]带领的罗马人摧毁了罗得市。但是一直到74年罗得岛才被并入[[罗马帝国|罗马帝国]]，一开始它属于[[邦非力和吕底亚行省|邦非力和吕底亚行省]]。

罗马帝国分裂后罗得岛属于[[拜占庭帝国|拜占庭帝国]]。阿拉伯人曾两次短期占领罗得岛。1097年[[第一次十字军东征|第一次十字军东征]]时[[十字军|十字军]]攻占罗得岛。1204年第四次十字军东征后罗得岛归属[[威尼斯|威尼斯]]，后来又回归与[[热那亚|热那亚]]联盟的拜占庭。
[[File:Rhodos1493.png|thumb]]

1306年热那亚将罗得岛卖给了[[圣约翰骑士团|圣约翰骑士团]]，不过这个骑士团首先还得攻占罗得岛，才能在上面统治200多年。骑士团抵擋了多次穆斯林的进攻。1522年[[苏莱曼一世_(奥斯曼帝国)|苏莱曼一世]]率领[[奥斯曼帝国|奥斯曼帝国]]军队围攻罗得岛，在数月的围攻中土耳其舰队不断炮击罗得岛上的要塞，最后骑士团投降，并于1523年元旦离开罗得岛。1912年5月在[[意土战争|意土战争]]中[[意大利|意大利]]占领了包括罗得岛在内的佐则卡尼萨群岛才结束了土耳其的统治。

意大利统治时期从1912年持续至约1943年，在这段时间里在罗得市港口入港处树立的公鹿和母鹿的像成为罗得岛的新的象征。传说这两个像位于太阳神雕塑的原址。意大利人在岛上还建造了许多街道和建筑物。本来被摧毁的王宫也是这时重建的。

德国占领时期在岛上建造了许多军事设施。比如东岸的许多碉堡以及军用机场今天依然可见。

直到1947年罗得岛才与佐则卡尼萨群岛的其它岛屿被并入希腊。

== 气候 ==
罗得岛是欧洲阳光最充沛的地区，年平均日照时间达3000小时以上。从5月中至9月中基本无雨。岛上的[[喀斯特地形|喀斯特地形]]保障在这段时间里依然有足够的贮水。夏季附近的[[西米岛|西米岛]]和哈尔基岛需要船只保障供水。7月和8月岛上的温度往往高达40 °C，海风可以保障西岸的气温在28 °C左右，东岸的气温一般在35至40 °C。不过同时空气非常干燥。夜间气温可以降到23至30 °C，8月时水温可以达到27 °C。至到11月可以在海中游泳。气温最低的时候是1月和2月。白天的气温在12至18 °C之间，夜间在8至12 °C之间。岛上基本上没有霜冻，只有在最高的山顶上有时会有降雪。最低水温是2月末至3月初，在16至17 °C之间。

== 经济和交通 ==
罗得岛最重要的经济行业是[[旅游业|旅游业]]（占生产总值的85%）。农业主要是种植[[葡萄|葡萄]]和[[橄榄|橄榄]]。部分岛上产的葡萄酒出口，但当地出产的葡萄酒的质量不高。山上以[[山羊|山羊]]和[[绵羊|绵羊]]的半畜牧业为主，罗得岛的羊奶奶酪非常有名。

岛上的公共汽车服务非常好，除最南端的半岛外到处可以通过公共汽车到达，此外[[出租汽车|出租汽车]]也非常廉价。

每天有通往周边岛屿和土耳其的摆渡船。尤其老船的摆渡费非常便宜。不过有些船员的不友好性也是非常著名的。

[[罗得岛国际机场|罗得岛国际机场]]是爱琴海岸最繁忙的机场之一，尤其许多旅行社的租机飞到这里。[[希腊航空公司|希腊航空公司]]和[[爱琴航空公司|爱琴航空公司]]有班机飞往罗得岛，主要是连接罗得岛和雅典以及[[塞萨洛尼基|塞萨洛尼基]]。

== 旅游 ==
罗得岛与[[克里特|克里特]]并列希腊最重要的旅游业地区，从5月初到10月末是旺季。岛北部地区是旅游中心，而南部也於近期蓬勃發展中。

在特里安得有意大利[[方济各会|方济各会]]修道院遗址以及[[伊利索斯|伊利索斯]]（最早的三个城市之一）的遗址，从該處可以俯视岛的北端。沿西岸的公路继续向前可以到达“蝴蝶之谷”，不过谷内只有从6月中至9月才有蝴蝶。
