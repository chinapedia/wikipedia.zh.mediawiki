'''罗得市水族馆'''（Aquarium of Rhodes）是[[希腊|希腊]][[罗得市|罗得市]]的一个水族馆、研究中心和博物馆。它建于1930年代，意大利统治罗得岛期间，目前由国家海洋研究中心管理。

==历史==
这是一座[[装饰艺术|装饰艺术]]建筑，由意大利建筑师Armando Bernabiti设计，建于1934-1935年。当它于1937年开业时，名为皇家罗德生物研究所（Reale Istituto di Ricerce Biologiche di Rodi），研究爱琴海的水文，海绵和渔业。<ref name=aqua_main>{{Cite web |url=http://rhodes-aquarium.hcmr.gr/ |title=Hydrobiological Station of Rhodes |work=rhodes-aquarium.hcmr.gr |publisher=Hellenic Centre for Marine Research |accessdate=2015-11-29}}</ref>

当该岛于1945年交回希腊时，该设施成为“希腊水文学研究所”的一部分。自1963年以来，称为“罗得岛水文站”，由国家海洋研究中心管理。<ref name=faliraki>{{Cite web |url=http://www.faliraki-info.com/aquarium/ |title=The Aquarium of Rhodes |work=faliraki-info.com |publisher=Faliraki Directory |accessdate=2015-11-29 }}{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref> 1971–72年，建筑北侧添加了一个展区<ref name=aqua_main/>。

==布局==
水族馆位于建筑物的地下室，是一个设计为水下洞穴的走廊。走廊两旁的水箱装满了大约{{convert|80000|l|gal}}过滤海水，为水族们提供了一个天然环境。展品包括[[海葵|海葵]]、[[章鱼|章鱼]]、[[海胆|海胆]]、[[珊瑚|珊瑚]]、[[双壳纲|双壳纲]]、[[螃蟹|螃蟹]]、海鳌虾、[[海龟|海龟]]、[[海豚|海豚]]、[[软体动物|软体动物]]、[[鲨鱼|鲨鱼]]以及许多其他鱼类。其中的设施也用于帮助附近受伤的海龟和海豹。<ref name=greeka>{{Cite web |url=http://www.greeka.com/dodecanese/rhodes/rhodes-museums/rhodes-aquarium.htm |title=Aquarium |work=greeka.com |publisher=Greeka.com |accessdate=2015-11-29}}</ref><ref name=rhodesguide>{{Cite web |url=http://www.rhodesguide.com/travelguide/rhodes_excursions.php?ssp=4 |title=Marine Aquarium |work=rhodesguide.com |publisher=Rhodes Guide |accessdate=2015-11-29}}</ref><ref name=travel-rhodes>{{Cite web |url=http://www.travel-rhodes.com/place.php?place_id=23 |title=Rhodes Aquarium |work=travel-rhodes.com |publisher=Travel--rhodes.com |accessdate=2015-11-29}}</ref>
