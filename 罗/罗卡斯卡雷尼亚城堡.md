'''罗卡斯卡雷尼亚城堡'''是一个位于[[基耶蒂|基耶蒂]]大区罗卡斯卡雷尼亚市的防御建筑。<div>城堡修建在一个突出的岩石的顶部，主要的聚居地，在很久以前建造并在16至17世纪大量扩建。</div><div>在20世纪最后几年经过完整的修复后，城堡重新开放。<br>
</div>

== 历史 ==
[[File:Roccascalegna_(CH).jpg|left]]
城堡大约是在11至12世纪从一个[[伦巴第|伦巴第]]警戒塔扩建而成。
<div>第一次对城堡的记载可以追溯到1525年关于城堡修复的记叙。<br>
</div><div>在16世纪重修了更高的没有城垛的城墙。<br>
</div><div>在阿内基尼斯（Annechinis）统治时期，加固了现有的塔楼并添加了一个圆形建筑。<br>
</div><div>主要参与的是在1531年至1600年统治这里的卡拉法（Carafa）男爵：他在1577年修建至圣玫瑰教堂（la cappella del Santissimo Rosario）并改善了雨水蓄水池。<br>
</div><div>在之后的1600年至1717年柯勒维（Corvi）的统治下，1705年吊桥被破坏后，修建了入口的甘亭和保护主路入口的墙壁。</div><div>在接下来的岁月里，城堡被废弃，在1940年心之塔楼（这么称呼是因为主门上的徽章）不同程度地倒塌。<br>
</div><div>在1985年城堡最后的所有人——十字纳尼（Croce Nanni），把城堡赠予市政府，市政府决定在20世纪90年代至1996年修复城堡。<br>
</div>

== 构造 ==
通过从圣彼得教堂外成阶地倾斜岩石可以到达有吊桥遗迹的入口。
<div>在它的右边是一个名为哨兵之塔的塔楼。<br>
</div><div>庭院通向其他的塔楼：监禁塔，一个通过屋檐收集雨水并汇集到蓄水池的小教堂。<br>
</div><div>一个较远的斜坡通向一座用方石和四面带孔的砖修建的警戒塔。<br>
</div><div>城墙围住了悬崖上突出的岩石。<br>
</div><div>城墙像塔楼一样高，用方石，鹅卵石和砖瓦碎片修建而成。<br>
</div><div>教堂唯一的大厅没有装饰物。<br>
</div>

== 传说 ==
[[File:Roccascalegna.jpg|thumb]]
传说城堡的男爵，科勒沃·德·柯勒维斯（Corvo de Corvis），在1646年强制执行初夜权，即领地的新娘的新婚初夜是和领主度过，而不是新郎。<div>传说最后一个新娘乔装刺死了男爵，男爵在城堡的岩石上留下了永不磨灭的血手印。</div><div>但事实上，这个血手印已经消失了。</div>

== 圣彼得教堂 ==
城堡中有一座中世纪的圣彼得教堂。
