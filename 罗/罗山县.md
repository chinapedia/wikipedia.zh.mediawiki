'''罗山县'''位于[[中华人民共和国|中华人民共和国]][[河南省|河南省]]东南部，[[大别山|大别山]]北麓，[[淮河|淮河]]上游，是[[信阳市|信阳市]]下辖的一个县，邻接[[湖北省|湖北省]]。县人民政府驻城关镇。

当地自古人才辈出，蜀汉四英之一[[费祎|费祎]]、清朝[[南河总督|南河总督]]、治河功臣[[黎世序|黎世序]]、近代历史名家[[尚钺|尚钺]]都出自罗山。

== 历史沿革 ==
[[西汉|西汉]]始置[[鄳县|鄳县]]，属[[江夏郡|江夏郡]]。魏晋沿用汉制。[[南朝宋|南朝宋]]分鄳县置[[宝城县|宝城县]]，属[[义阳郡|义阳郡]]。[[南齐|南齐]]改为宝城为保城，属[[北义阳郡|北义阳郡]]。[[北魏|北魏]]时分置保城县、鄳县、东随县。保城县、鄳县属[[齐安郡|齐安郡]]。[[北齐|北齐]]合三县置[[高安县|高安县]]，仍属齐安郡。

[[隋朝|隋]]开皇三年（583年）高安县并入[[钟山县|钟山县]]。开皇十六年（596年）析钟山县地置罗山县，属义阳郡。因县境西南有小罗山，由此得名。[[唐朝|唐]]武德四年（612年）置[[南罗州|南罗州]]，领罗山一县。武德八年（625年）南罗州废，罗山县属[[申州|申州]]。[[五代|五代]]同唐制。[[宋朝|宋]]开宝九年（976年）罗山县废。雍熙三年（986年）复置，属[[信阳军|信阳军]]。

[[元朝|元]]至元二十年（1283年）信阳州改治于罗山县城，原罗山县治迁往西南，称罗山新县，属[[汝宁府|汝宁府]][[信阳州|信阳州]]。

[[明朝|明]]洪武元年（1368年）县治复旧，仍称罗山县，属信阳州。洪武七年（1374年），改属汝宁府。成化十年（1475年）复属信阳州。[[清朝|清朝]]罗山属汝宁府。

[[中华民国|中华民国]]2年（1913年）废州改县，罗山县属[[豫南道|豫南道]]，后改成[[汝阳道|汝阳道]]。民国16年（1927年）废道，直属[[河南省|河南省]]。17年（1928年）河南省分为14个行政区，罗山属第七（信阳）督察行政区。民国21年（1932年）划属河南省第九（潢川）行政督察区。民国22年（1933年）析县境南部一片，新置[[湖北省_(中華民國)|湖北省]][[礼山县_(中华民国)|礼山县]]，即今[[大悟县|大悟县]]。

1949年3月[[中国人民解放军|中国人民解放军]]占领罗山，属潢川专区。1952年潢川专区并入信阳专区，罗山随之并入。1998年信阳撤地设市，罗山为其辖县。

== 自然地理 ==
罗山县位于河南省南部，[[河南|河南]]、[[湖北|湖北]]两省交界地带，[[大别山|大别山]]北麓。地势西南高、东北低，全境属[[淮河|淮河]]流域，土地构成大致为“五山一水四分田”。

== 行政区划 ==

羅山縣辖有9[[镇_(中华人民共和国)|镇]]、10[[乡_(中华人民共和国)|乡]]：

*鎮：城关镇、周党镇、楠杆镇、竹竿镇、青山镇、子路镇、灵山镇、彭新镇、潘新镇。

*鄉：龙山乡、高店乡、尤店乡、东铺乡、莽张乡、庙仙乡、朱堂乡、铁铺乡、山店乡、定远乡。
