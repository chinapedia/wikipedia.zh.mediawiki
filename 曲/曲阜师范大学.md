'''曲阜师范大学'''（Qufu Normal University, QFNU）是[[山东省|山东省]]以[[师范教育|师范教育]]为主的省属重点大学。座落于[[中国|中国]]古代思想家、[[教育家|教育家]]、[[政治家|政治家]]，[[儒家|儒家]][[文化|文化]]的[[创始人|创始人]][[孔子|孔子]]的故里[[曲阜|曲阜]]。在山东[[日照市|日照市]]设有新校区。简称为[[曲阜师大|曲阜师大]]，[[曲师大|曲师大]]。

== 历史 ==
=== 历史沿革 ===
==== 山东师范专科学校 ====
1955年，创立[[山东师范专科学校|山东师范专科学校]]，位于[[山东|山东]]省[[济南市|济南市]]。

==== 曲阜师范学院 ====
[[File:1964-01_1964年_曲阜师范学院2.jpg|thumb]]
1956年，迁往[[山东|山东]]省[[曲阜|曲阜]]市，并更名为[[曲阜师范学院|曲阜师范学院]]。

==== 山东大学 ====
1970年，与[[山东大学|山东大学]]文科部分合并，称为[[山东大学|山东大学]]。

==== 曲阜师范学院 ====
1974年，恢复为曲阜师范学院。

==== 曲阜师范大学 ====
1985年，更名为曲阜师范大学。

== 教学及科研 ==
[[File:1964-01_1964年_曲阜师范学院.jpg|thumb]]
曲阜师范大学按照“厚基础、宽口径、大专业、小方向”的培养模式，进行教学内容与课程体系的改革，全面实施素质教育，提高教育质量。承担国家“[[863计划|863计划]]”等科研项目。在中国古代文学、应用数学、[[孔子儒学|孔子儒学]]、基础[[数学|数学]]、[[激光|激光]][[偏光|偏光]]技术、[[运筹学|运筹学]]、[[自动化|自动化]]、[[物理化学|物理化学]]、教育科学、[[体育|体育]]科学等学科领域有较高的水平。九五以来，承担各级各类科研项目838项，其中省部级以上165项，发表论文8825篇，出版著作421部，取得重要科技成果62项（其中国际水平9项，国内领先39项），获科研奖励255项（其中省部级以上105项）。
