'''遠國奉行'''是[[江戶幕府|江戶幕府]]的職稱之一。設置在[[江戶|江戶]]以外的幕府直轄領（[[天領|天領]]）內之要地，掌管該地政務之[[奉行|奉行]]。

==概要==
遠國奉行是[[京都町奉行|京都町奉行]]、[[大坂町奉行|大坂町奉行]]、[[駿府町奉行|駿府町奉行]]之各町奉行和[[伏見奉行|伏見奉行]]、[[佐渡奉行|佐渡奉行]]、[[長崎奉行|長崎奉行]]、[[堺奉行|堺奉行]]、[[山田奉行|山田奉行]]、[[奈良奉行|奈良奉行]]、[[日光奉行|日光奉行]]、[[浦賀奉行|浦賀奉行]]、[[下田奉行|下田奉行]]、[[新潟奉行|新潟奉行]]、[[箱館奉行|箱館奉行]]、[[神奈川奉行|神奈川奉行]]、[[兵庫奉行|兵庫奉行]]之各奉行的總稱。伏見奉行由[[大名_(稱謂)|大名]]擔任，其他由[[旗本|旗本]]擔任。

[[老中|老中]]之支配下，[[伺候席|芙蓉間詰]][[律令制|諸大夫]]役。[[足高制|役高]]依任地由1000[[石高|石]]至2000石不等，也有支給[[役料|役料]]者。

遠國奉行之管轄地稱為奉行知行所。[[慶應|慶應]]4年／[[明治|明治]]元年（1868年）江戶幕府被推翻，明治政府在江戶等地設置府。至明治4年（1871年）[[廢藩置縣|廢藩置縣]]之間，除江戶府（後更名東京府）、大阪府、京都府之外的箱館府、神奈川府、越後府、新潟府、度會府、奈良府、長崎府的各府改為縣。

== 京都町奉行 ==
{{main|京都町奉行}}

[[老中|老中]]之支配下，因任地的關係，在[[京都所司代|京都所司代]]指揮下執行職務。[[京都|京都]]町政之外，也掌管畿内[[天領|天領]]及'''寺社領'''之支配，所以形同兼管[[寺社奉行|寺社奉行]]、[[勘定奉行|勘定奉行]]、[[町奉行|町奉行]]之三奉行的職務。定員2名，設置東西2奉行所。役高1500石、役料現米600石。配下有[[與力|與力]]20騎和[[同心_(官职)|同心]]50人。

== 大坂町奉行 ==
{{main|大坂町奉行}}

定員2名，設置東西2奉行所。役高1500石、役料現米600石。配下有與力30騎・同心50人。

== 駿府町奉行 ==
[[駿府|駿府]]町政之外，主要負責[[駿河國|駿河國]]內[[天領|天領]]的[[公事方|公事方]]。與[[駿府城|駿府城]]代協議，負責監視通行駿府的諸大名諸士、駿河或伊豆之裁判仕置、[[久能山東照宮|久能山東照宮]]之警衛。

== 長崎奉行 ==
{{main|長崎奉行}}

[[長崎|長崎]]町政之外，也負責對[[荷蘭|荷蘭]]貿易之事務。役高1000石、役料440俵。配下有與力10騎、同心15人。

== 下田奉行・浦賀奉行 ==
下田奉行負責[[伊豆國|伊豆]][[下田市|下田]]港之警備、船舶監督、貨物檢査、當地民政。定員1～2名。役高1000石、役料1000俵。

== 山田奉行 ==
{{main|山田奉行}}

也稱作'''伊勢奉行'''、'''伊勢町奉行'''、'''伊勢郡代'''、'''伊勢山田奉行'''、'''伊勢山田町奉行'''。負責[[伊勢神宮|伊勢神宮]]之守護・營造修理和祭禮、遷宮、[[門前町|門前町]]之支配、[[伊勢國|伊勢]]、[[志摩國|志摩]]內之訴訟，[[鳥羽港|鳥羽港]]之警備、船舶檢査等事務。

定員1～2名。役高1000石、役料1500俵。配下有與力6騎・同心70人・水主40人。

== 奈良奉行 ==
設置目的為[[興福寺|興福寺]]、[[東大寺|東大寺]]等[[奈良|南都]]大寺院之監視和門前町（[[奈良町|奈良町]]・[[奈良北町|奈良北町]]）之支配。也稱作'''南都町奉行'''。老中之支配下，但在[[京都所司代|京都所司代]]指揮下執行職務，主要任務是[[春日大社|春日大社]]之警衛和神事。

定員1～2名。役高1000石、役料700俵。配下有與力7騎和同心30人、牢番1人。奉行所設置在現在的[[奈良女子大學|奈良女子大學]]校地內。

== 堺奉行 ==
{{main|堺奉行}}

統治[[堺|堺]]及[[和泉國|和泉]]、[[河內國|河內]]兩國之天領。定員1名。役高1000石、役料現米600石。配下有與力10騎和同心50人。

== 佐渡奉行 ==
{{main|佐渡奉行}}

負責[[佐渡金山|佐渡金山]]之支配。定員1名。役高1000石、役料1500俵100人扶持。配下有與力30騎和同心70人。

== 箱館奉行（松前奉行） ==
負責[[蝦夷地|蝦夷地]]之行政、防衛。幕末時也負責開港地[[箱館|箱館]]之對外事務。

== 新潟奉行 ==
[[天保_(日本)|天保]]14年（1843年）設置，目的為[[日本海|日本海]]交通要衝的[[新潟港|新潟港]]之管理。

負責[[新潟市|新潟]]之民政、出入船舶之監視、密貿易之取締、海岸警備、海防強化。老中之支配下，役高1000石、役料1000俵。定員1名。

== 羽田奉行 ==
天保13年（1842年）設置，目的為對外國船來航[[江戶灣|江戶灣]]之防禦，2年後廢止。

管理羽田（今東京都大田區）之港灣、船舶。役高1000石、役料500俵。

== 清水奉行 ==
[[元和_(後水尾天皇)|元和]]7年（1621年）設置。負責[[駿河灣|駿河灣]]之警衛，配下有水主50人。

[[元祿|元祿]]9年（1696年）廢止，清水奉行之職務由駿府町奉行繼承。

== 兵庫奉行 ==
負責[[神戶港|兵庫港]]之對外事務。役高1000石、役料現米600石。
