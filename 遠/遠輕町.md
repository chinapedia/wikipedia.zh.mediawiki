'''遠輕町'''（{{jpn|j=遠軽町|hg=えんがるちょう|rm=Engaru chō|f=y}}）位於[[北海道|北海道]][[鄂霍次克綜合振興局|鄂霍次克綜合振興局]]中部。在2005年10月1日由原本的遠輕町以及[[生田原町|生田原町]]、[[丸瀨布町|丸瀨布町]]、[[白瀧村|白瀧村]][[市町村合併|合併]]新設而成，面積為北海道內第四（次於[[北見市|北見市]]、[[足寄郡|足寄郡]][[足寄町|足寄町]]、[[釧路市|釧路市]]）；[[人口|人口]]是北海道內第五多的[[町|町]]（次於[[河東郡_(日本)|河東郡]][[音更町|音更町]]、[[龜田郡|龜田郡]][[七飯町|七飯町]]、[[中川郡_(十勝支廳)|中川郡]][[幕別町|幕別町]]、[[日高郡_(北海道)|日高郡]][[新日高町|新日高町]]）。

遠輕町著名的事項包括：
# 町內的白瀧地區是[[合氣道|合氣道]]的發源地。
# 每年初秋有盛開1,000萬個花朵的日本最大型的大波斯菊園。
# 町內盛行樂團演奏，北海道遠輕高等學校的樂團，先後進入全日本樂團競賽7次，町內的中學和小學也常進入全國性比賽。

夏冬之間溫差很大，每年5月開始會有焚風現象，氣溫甚至會超過30度，但在7月過後，又會因為[[鄂霍次克海|鄂霍次克海]][[高氣壓|高氣壓]]的緣故使溫度迅速降低；冬季期間氣溫有時則會低於零下20度，站在高地還可以遠眺鄂霍次克海的流冰。

町名的來源和町內著名的瞰望岩有關，名稱源自[[阿伊努語|阿伊努語]]的「inkar-us-i」意思為適合眺望的地方。

== 歷史 ==
* 1869年：屬於[[和歌山藩|和歌山藩]]的管轄地。<ref>{{Cite web| publisher=遠輕町公所| title=遠輕町的沿革| url=http://engaru.jp/town_introduction/enkaku/engaru.html| accessdate=2007年4月21日| language=ja| deadurl=yes| archiveurl=https://web.archive.org/web/20070111103610/http://engaru.jp/town_introduction/enkaku/engaru.html| archivedate=2007年1月11日}}</ref>
* 1896年：位於[[仙台|仙台]]的[[日本基督教會|日本基督教會]]成立了[[北海道同志教育會|北海道同志教育會]]，計畫在北海道的遠輕地區建立基督教大學，第一批移民團於隔年5月7日抵達遠輕地區，成為最初的集體移民；但最後興建大學的計畫並未實現。
* 1919年4月1日：從上湧別村（現在的[[上湧別町|上湧別町]]）分村，成為二級村遠輕村。<ref>{{Cite web| url=http://homepage1.nifty.com/ishato/tiri/sityoson/01hokkai/01_hokkai21.htm#soya| title=市町村的變遷 - 北海道［道東・道北］| publisher=ISSIE| accessdate=2008年7月12日| language=ja| deadurl=yes| archiveurl=https://web.archive.org/web/20091001194212/http://homepage1.nifty.com/ishato/tiri/sityoson/01hokkai/01_hokkai21.htm#soya| archivedate=2009年10月1日}}</ref>
* 1925年1月1日：[[生田原町|生田原村]]從遠輕村分村。
* 1934年4月1日：改制為遠輕町 。
* 1946年8月1日：[[丸瀨布町|丸瀨布村]]、[[白瀧村|白瀧村]]從遠輕町分村。
* 1953年10月1日：丸瀨布村改制為丸瀨布町。
* 1954年4月1日：生田原村改制為生田原町。
* 2005年10月1日：遠輕町、生田原町、丸瀨布町、白瀧村合併新設為遠輕町。

== 產業 ==
主要產業是商業、林業、木材加工業和農業。

由於[[遠輕車站|遠輕車站]]過去是舊[[名寄本線|名寄本線]]和[[石北本線|石北本線]]的交會站，具有重要的交通地位，帶動商業發展。
