'''鮑比·坎納瓦爾'''（{{lang-en|Bobby Cannavale}}，{{bd|1970年|5月3日}}）是美國的一位演員<ref>[http://www.telegram.com/apps/pbcs.dll/article?AID=/20070503/NEWS/705030673/1102 "This day in history"]. NewsTelegram.com. May 3, 2007</ref>。他最著名的作品有電視劇《{{link-en|城市守護者第三梯隊|Third Watch}}》中的Bobby Caffey角色，[[全國廣播公司|NBC]]喜劇《[[威爾和格蕾絲|威爾和格蕾絲]]》中的Vincent "Vince" D'Angelo角色，以及於[[HBO|HBO]]的劇集《[[酒私風雲|酒私風雲]]》第三季中飾演[[吉普·羅賽迪|吉普·羅賽迪]]<ref>{{Cite web |url = https://www.hollywoodreporter.com/news/boardwalk-empire-bobby-cannavale-287027 |title = Bobby Cannavale Joins 'Boardwalk Empire' |author = Philiana Ng |date = 2012-02-03 |publisher = The Hollywood Reporter |language = en}}</ref><ref>{{Cite web |url = https://deadline.com/2012/02/bobby-cannavale-joins-hbos-boardwalk-empire-as-new-regular-in-season-3-225646/ |title = Bobby Cannavale Joins HBO's 'Boardwalk Empire' As New Regular In Season 3 |author = Nellie Andreeva |date = 2012-02-03 |publisher = Deadline Hollywood |language = en}}</ref>，此角色為他獲得2013年[[黃金時段艾美獎|黃金時段艾美獎]]{{link-en|黃金時段艾美獎傑出男配角|Primetime Emmy Award for Outstanding Supporting Actor in a Drama Series|傑出男配角}}。2016年，他主演HBO劇集《[[黑膠時代|黑膠時代]]》，由[[馬田·史高西斯|馬田·史高西斯]]和[[米克·傑格|米克·傑格]]監製。2017年，他加入演出[[USA電視台|USA電視台]]的劇集《[[黑客軍團|黑客軍團]]》第三季。

他於2013年演出[[活地·亞倫|活地·亞倫]]的電影《[[情迷藍茉莉|情迷藍茉莉]]》，2014年的《[[滋味旅程|滋味旅程]]》及《''{{link-en|Annie (2014 film)|Annie (2014 film)|Annie}}''》，2015年《[[蟻俠_(電影)|蟻俠]]》中飾演Paxton。
