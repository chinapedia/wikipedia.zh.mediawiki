《'''-{民}-王'''》（{{jpn|j=民王|hg=たみおう|rm=Tamiō|f=y}}，）是[[日本|日本]]小說家[[池井戶潤|池井戶潤]]創作的[[政治|政治]][[小說|小說]]，2010年5月由{{link-ja|POPLAR|ポプラ社}}出版單行本，2013年6月7日由[[文藝春秋|文藝春秋]]發行[[文庫本|文庫本]]。2015年7月，[[朝日電視台|朝日電視台]]改編為[[朝日電視台周末晚間連續劇|金11]]時段的電視連續劇<ref>[http://news.walkerplus.com/article/58658/ 池井戸潤「民王」遠藤憲一＆菅田将暉W主演でドラマ化]</ref>。

== 概要 ==
某日，身为[[日本首相|日本首相]]的武藤泰山與其就讀大學的兒子武藤翔突然間交換了身體。為了不引起騷動，兩人試圖隱瞞週遭人，並以對方的身體過著對方的生活。但，武藤翔對政治完全沒興趣，連[[日本漢字|漢字]]也看不懂，在[[日本國會|國會]]答辯時簡直陷入了一番苦鬥。另一方面，父親武藤泰山在就職活動和面試主管的面談亦是另一場苦戰……。

== 登場人物 ==
*武藤泰山：日本首相，民政黨的黨代表。在當選後一度爭權逐利在選舉時就不受女性歡迎。在黨內和家庭中的信賴度皆低。因和兒子武藤翔交換身體，不得已代替兒子前往面試，但討厭對方高高在上的態度使得求職不甚順利。
*武藤翔：武藤泰山的兒子。吃喝玩樂、作息顛倒的大學生。卻很討女性的歡迎。雖然正在找工作，但八字沒一撇。對政治完全沒興趣，唸書也不行。性格純粹、有正義感。和父親交換身體後，卻因[[日本漢字|漢字]]不識幾個而陷入奮戰。
*狩屋孝司：[[內閣官房長官|內閣官房長官]]，親近武藤泰山的人士。有「變態香蕉官房長官」的[[綽號|綽號]]。
*鶴田洋輔：[[經濟產業大臣|經濟產業大臣]]。曾以醉醺醺的狀態出席記者會。
*真田武彥：[[防衛大臣|防衛大臣]]。調查武藤為何會交換身體的原因。
*藏本志郎：第一大[[在野黨|在野黨]]憲民黨的領袖，武藤泰山的對手。
*冬島一光：第二大[[在野黨|在野黨]]共和黨的領袖。
*南真衣：女大學生，武藤翔的同學。
*村野惠里香：武藤翔的同學。以成為[[政治家|政治家]]為志願的優等生。

=== 概要 ===
武藤翔（[[菅田將暉|菅田將暉]] 飾）是個擁有「[[烹飪|料理]]」、「[[美容|美容]]」等豐富知識且{{link-ja|女子力|女子力}}爆棚的男大學生，卻不慎被[[平底鍋|平底鍋]]砸到頭而與身為[[日本內閣總理大臣|內閣總理大臣]]的父親武藤泰山（[[遠藤憲一|遠藤憲一]] 飾）交換了身體。究竟，兩人如何面對接踵而至的棘手難關呢？

=== 主要角色 ===
*武藤泰山：[[遠藤憲一|遠藤憲一]]  /  菅田將暉（交換後）（粵語配音：[[羅偉亮|羅偉亮]]）
: 第100任[[日本內閣總理大臣|日本內閣總理大臣]]。超愛[[豬排|-{zh-hans:猪排;zh-tw:豬排;zh-hk:豬扒}-]][[咖哩|咖哩]]。喜歡女色，曾傳出數起桃色糾紛。
*武藤翔：[[菅田將暉|菅田將暉]]  /   遠藤憲一 （交換後） / 石塚獅櫻（幼年）（粵語配音：[[陳健豪|陳健豪]]）
: 泰山的兒子。擁有「女子力」的早背山大學大四生，對雙親皆使用[[日語敬語|敬語]]。住在打工地方「暗雲」的[[閣樓|閣樓]]上。雖是大學四年級生，但沒有相應的學習能力。
*南真衣：[[本假屋唯香|本假屋唯香]]（粵語配音：[[莎拉|莎拉]]）
: 翔的學姊。職業介紹公司社長，「暗雲」老闆夫婦的親戚。翔憧憬的人。
*村野惠里香：[[姜知英|知英]] / 草刈正雄（交換後）（粵語配音：[[朱婉雯|朱婉雯]]）
: 和翔同校的大四生。武藤泰山政治上的對手藏本志郎的女兒。尊敬真衣。
*貝原茂平：[[高橋一生|高橋一生]]（粵語配音：[[梁皓翔|梁皓翔]]）
: 泰山的公設祕書。
*闇雲渡：{{link-ja|原金太郎|原金太郎}}
: 無國籍料理餐廳「暗雲」的老闆。經常不計成本採買大量高級食材而使店家虧損，向東京第一銀行貸款來經營餐廳。招牌菜[[豬排|豬排]][[咖哩|咖哩]]飯是翔的最愛吃的料理，父親武藤泰山亦稱讚此道料理是「日本財產」。
*闇雲郁代：{{link-ja|池谷伸枝|池谷のぶえ}}（粵語配音：[[鄭家蕙|鄭家蕙]]）
: 「暗雲」的女將。
*Ram：{{link-ja|信江勇|信江勇}}
: 常去「暗雲」找翔的女常客。職業為跳[[脫衣舞|脫衣舞]]，體型較胖。
*Mutton：谷田川真理子
: 常去「暗雲」找翔的女常客。職業為陪酒女，體型較瘦。
*武藤綾：{{link-ja|峯村理惠|峯村リエ}}（粵語配音：[[陳凱婷|陳凱婷]]）
: 泰山妻子，即[[內閣總理大臣夫人|總理夫人]]。十分溺愛翔，和丈夫泰山扮演著「假面夫妻」。
*新田理：{{link-ja|山內圭哉|山内圭哉}}
: [[警視廳|警視廳]]公安第一課警視。氣質像[[黑道|黑道]]的優秀警察。
*狩屋孝司：{{link-ja|金田明夫|金田明夫}}（粵語配音：[[李家傑_(配音員)|李家傑]]）
: [[內閣官房長官|內閣官房長官]]。代代為武藤家工作，泰山的盟友。
*小中壽太郎：{{link-ja|六角精兒|六角精児}}
: [[政治評論|政治評論]]家。早背山大學的教授。
*冬島一光：{{link-ja|猪野学|猪野学}}（粵語配音：[[袁德基|袁德基]]）
: 共和黨領袖。
*藏本志郎：{{link-ja|草刈正雄|草刈正雄}}（特別演出）/ 知英（交換後）
: 憲民黨領袖。泰山的政敵。自第一次當選以來即是TOP的熟面孔，人氣議員。
*城山和彥：[[西田敏行|西田敏行]]
: 民政黨議員。泰山所屬的城山派大老。有實力的典型金權腐敗政客，在背後操控著武藤泰山。
