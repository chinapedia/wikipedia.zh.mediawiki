'''民主派（Democrats）'''字面意思是支持[[民主|民主]]制度的人士，而其對應英文'''Democrat'''可指支持民主制度的人士，或[[民主党|民主党]]人、民主党党員。

== 香港及澳門 ==
基於特殊的政治制度環境，在[[香港|香港]]及[[澳門|澳門]]衍生的一批政治人物，一般被稱為[[泛民主派|泛民主派]]（Pan-democrats）：
* '''[[民主派_(香港)|香港民主派]]'''，當中有不少[[反共主義|反共]]人士（即反對[[中國共產黨|中國共產黨]]），被[[建制派|建制派]]和[[親中媒體|親中媒體]]稱為[[反對派|反對派]]、[[非建制派|非建制派]]。
**'''[[泛民主派|泛民主派]]'''<sup>註1</sup>（簡稱泛民）支持香港全面落實[[真普選|真正的民主普選]]，要求普選行政長官和立法會，並廢除立法會的[[功能組別|功能組別]]。
**'''[[自決派|自決派]]'''，左翼兼優先顧及香港人利益或力爭[[社會正義|社會公義]]。自決派並不完全等於[[香港獨立運動|香港獨立運動]]支持者。
**'''[[本土派_(香港)|本土派]]'''，右翼兼優先顧及香港人利益或力爭[[社會正義|社會公義]]。本土派並不完全等於[[香港獨立運動|香港獨立運動]]支持者。
* '''[[民主派_(澳門)|澳門民主派]]'''（參見[[澳門政治團體|澳門政治團體]]），政治上跟隨香港民主派的政治用語，理念上亦跟香港民主派相近，支持澳門全面落實真正的民主普選，要求普選行政長官和立法會。被[[建制派__(澳門)|建制派]]和[[親中媒體|親中媒體]]稱為[[反對派|反對派]]。

<small>註1：「派」已有「泛」的意思，稱為「泛民」可能是受英文「Pan-」或台灣的「[[泛藍|泛藍]]」、「[[泛綠|泛綠]]」等名詞影響。</small>

== 台灣 ==
過去台灣的民主派多指[[黨外運動|黨外運動]]人士及其他台灣民主運動的參與者，要求[[中國國民黨|中國國民黨]]終結[[一黨執政|一黨執政]]及[[台灣省戒嚴令|解嚴]]，[[國民大會|國民大會]]和[[立法院|立法院]]（國會）全面改選，[[中華民國總統|總統]]直接民選等。民主派並不完全等於[[台灣獨立運動|台灣獨立運動]]支持者（[[獨派|獨派]]）。自黨外人士組成[[民主進步黨|民主進步黨]]後，現多指'''[[泛綠|泛綠]]'''陣營。

== 中國大陸 ==
[[中華人民共和国|中華人民共和国]][[文化大革命|文化大革命]]末期，中国共产党理论家认为「资产阶级民主派」将变为「[[走资派|走资派]]」，其论点体现于[[池恒|池恒]]文章《从资产阶级民主派到走资派》。

現時的中國民主派則指在中國支持民主及推動[[維權|維權]]的人士，多數會受中共官方的政治打壓。如被中共取締的'''[[中国民主党|中国民主党]]'''。參見中國維權人士列表。
