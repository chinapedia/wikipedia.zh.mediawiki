'''民丹线轮渡'''，原称'''民安线轮渡'''。为[[中华人民共和国|中国]][[上海市|上海市]][[黄浦江|黄浦江]]上的一条对江轮渡航线。航线北起[[杨浦区|杨浦区]][[丹东路|丹东路]]65号丹东路轮渡站，南至[[浦东新区|浦东新区]][[民生路_(上海)|民生路]]1号民生路轮渡站。<ref>[http://www.shanghaiferry.com/1-%E7%BD%91%E9%A1%B5ppt%E8%BD%AC%E5%9B%BE%E7%89%87%E8%B5%84%E6%96%99/%E8%88%AA%E7%BA%BF%E8%88%AA%E7%8F%AD%EF%BC%88%E5%90%84%E6%B8%A1%E5%8F%A3%E7%AE%80%E4%BB%8B%EF%BC%89/1-%E4%B8%B9%E4%B8%9C%E8%88%AA%E7%BA%BF%E7%AE%80%E4%BB%8B.png 上海轮渡民丹东线简介]</ref>航线始于清朝年间，为浦西兰州路港口至浦东贾家桥的民渡。1955年11月1日，航线并入上海市轮渡公司，命名为'''民安线轮渡'''，航线由浦西安东路（今[[丹东路|丹东路]]）至浦东[[民生路_(上海)|民生路]]。1958年11月10日，民安线轮渡开辟车渡航线。1965年3月16日，浦西安东路更名为[[丹东路|丹东路]]，民安线轮渡同时更名为'''民丹线轮渡'''，名称沿用至今。2006年8月21日，民丹线车渡停航。2014年1月1日，民丹线客渡暂时停航，预计2015年6月30日恢复。<ref>{{cite web |url=http://shanghai.xinmin.cn/xmjd/2014/04/03/23951239.html |title=轮渡民丹线停运 通告欠妥乘客跑冤枉路 |author=沈文林 |date=2014年4月3日 |website=新民网 |accessdate=2014年8月23日}}</ref>

== 历史 ==
民丹线轮渡开设于清朝年间，为浦西兰州路港口至浦东贾家桥的民渡。航线开设初期配置帆船2艘渡客，后期发展为4艘。清光绪三十年（1904年），英商太古公司在浦东购买土地时，贾家桥渡口被划入购买范围内，渡口向西迁至[[民生路_(上海)|民生路]]。民国19年（1930年），购入庆和轮载客航行。民国26年（1937年）[[抗日战争|抗日战争]]爆发，航线停航。航线庆和轮被侵华日军拖走，最后沉在高桥。民国34年（1945年）抗日战争胜利，同年9月17日，航线申请复航。因浦西[[兰州路|兰州路]]港口已有至浦东洋泾港的轮渡，于是将浦西渡口码头西移至安东路（今[[丹东路|丹东路]]）。民国35—37年（1946—1948年），航线先后购置民安1—4号轮投入运营。
<br>上海战役后，1951年，又购入民安5号轮运营。1955年11月1日，航线以实有资产入股方式并入上海市轮渡公司。同时将航线命名为民安线轮渡。1965年3月16日，浦西安东路更名为[[丹东路|丹东路]]，民安线轮渡同时更名为民丹线轮渡。同年，市轮渡公司对浦西丹东路轮渡站进行扩建。<ref>[http://www.shtong.gov.cn/newsite/node2/node2245/node4501/node56018/node56020/userobject1ai43385.html 上海市地方志 专业志 >> 上海内河航运志 >> 第六章渡运 >> 节] {{webarchive|url=https://web.archive.org/web/20140101093158/http://www.shtong.gov.cn/newsite/node2/node2245/node4501/node56018/node56020/userobject1ai43385.html |date=2014-01-01 }}</ref><ref>[http://www.shanghaiferry.com/1-%E7%BD%91%E9%A1%B5ppt%E8%BD%AC%E5%9B%BE%E7%89%87%E8%B5%84%E6%96%99/%E7%99%BE%E5%B9%B4%E4%BC%A0%E6%89%BF/%E8%BD%AE%E6%B8%A1%E7%AE%80%E5%8F%B2/10-%E5%90%84%E6%B8%A1%E5%8F%A3%E5%8E%86%E5%8F%B2%E6%B2%BF%E9%9D%A9.html 上海轮渡各渡口历史沿革] {{webarchive|url=https://web.archive.org/web/20140202141841/http://www.shanghaiferry.com/1-%E7%BD%91%E9%A1%B5ppt%E8%BD%AC%E5%9B%BE%E7%89%87%E8%B5%84%E6%96%99/%E7%99%BE%E5%B9%B4%E4%BC%A0%E6%89%BF/%E8%BD%AE%E6%B8%A1%E7%AE%80%E5%8F%B2/10-%E5%90%84%E6%B8%A1%E5%8F%A3%E5%8E%86%E5%8F%B2%E6%B2%BF%E9%9D%A9.html |date=2014-02-02 }}</ref>

== 车渡 ==
1958年，为缓解[[南陆线轮渡|南陆线]]车渡航线拥挤，上海市轮渡公司在民丹线客渡旁开辟车渡航线。车渡于1958年10月31日建成，11月10日开航。开航初期，航线投入市轮渡103、104号两艘车辆渡轮。1967年，又投入沪车渡7号、8号运营。70年代，又投入3艘33米14车位车辆渡轮运营。90年代，航线配备4艘车辆渡轮。<ref>[http://www.shtong.gov.cn/newsite/node2/node2245/node4501/node56018/node56020/userobject1ai43387.html 上海市地方志 专业志 >> 上海内河航运志 >> 第六章渡运 >> 节] {{webarchive|url=https://web.archive.org/web/20140101093424/http://www.shtong.gov.cn/newsite/node2/node2245/node4501/node56018/node56020/userobject1ai43387.html |date=2014-01-01 }}</ref>2006年8月21日，为配合[[外环隧道|外环隧道]]大修，民丹线车渡停航，车渡渡轮移至[[东嫩线轮渡|东嫩线车渡]]，为东嫩线增加运力，以缓解隧道大修时取道东嫩线车渡过江时的车流压力。<ref>[http://www.jt.sh.cn/infopub/xxgkml/hygl/ywxx/info_090218035505239.html 上海市港口管理局关于同意市轮渡民丹线车辆渡停航的批复] {{webarchive|url=https://archive.is/20120804025952/http://www.jt.sh.cn/infopub/xxgkml/hygl/ywxx/info_090218035505239.html |date=2012-08-04 }}</ref><ref>[http://news.eastday.com/eastday/node17/node136/node13841/u1a165821.html 民丹线渡轮调整营运 车渡停运客渡照常]</ref>
