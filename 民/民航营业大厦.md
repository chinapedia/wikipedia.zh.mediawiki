'''民航营业大厦'''，位于[[北京市|北京市]][[西城区|西城区]][[西长安街|西长安街]]15号，隶属[[中国航空集团公司|中国航空集团公司]]旗下的中国航空集团旅业有限公司。

==简介==
民航营业大厦地处[[西单|西单]]附近，[[北京图书大厦|北京图书大厦]]及[[钟声胡同|钟声胡同]]东侧，[[北京电报大楼|电报大楼]]西侧，[[北京地铁1号线|北京地铁1号线]][[西单站|西单站]]以东。建成于1980年代末，建筑高度40.9米，共7层。建成时的建筑立面以横向线条为主，屋顶东西各有一坡顶小亭，是长安街上带有[[中国传统建筑|中国传统建筑]]风貌的“中而新”建筑<ref name=guihua>{{Cite web |url =http://www.bjghw.gov.cn/web/static/articles/catalog_10/article_402853575a02e2d6015a0327ac78000c/402853575a02e2d6015a0327ac78000c.html  |title =  促进项目发展，服务中央单位 ----我处核发西单民航大厦外立面改造的函|author =  |date =2017-01-26  |publisher =  北京市规划和国土资源管理委员会|language =  |accessdate =  |quote =  }}</ref>。1993年进行了民航营业大厦续建工程<ref>任广超、刘斌、彭云，在创一流的征途上，建筑1994(5):21</ref>。

民航营业大厦内设有民航营业大厅，有多家航空公司办理国内、国际航线的客运售票业务及相关手续<ref>{{Cite web |url = http://finance.sina.com.cn/2000-11-06/20683.html |title = 民航涨价头一天：“自杀性”政策理直气不壮 |author =  |date = 2000-11-06 |publisher = 新浪 |language =  |accessdate =  |quote =  }}</ref>。2005年，民航营业大厦以“中国航空集团旅业有限公司北京民航营业大厦”的名义进行工商登记成立公司，类型是“有限责任公司分公司（法人独资）”，经营范围是“出租写字间；航空、铁路客运代理；机动车公共停车场服务；接受委托代办公用电话业务；复印；保险兼业代理”<ref>{{Cite web |url = http://qyxy.baic.gov.cn/xycx/queryCreditAction!qyxq_view.dhtml?reg_bus_ent_id=199C30933B7D45B3B593E42534DB0EBD&credit_ticket=DE09618FBC8C0B15994F87895C718CEF |title = 中国航空集团旅业有限公司北京民航营业大厦 |author =  |date =  |publisher = 北京市企业信用信息网 |language =  |accessdate =2018-02-16  |quote =  }}</ref>。

2016年3月，北京市市政市容委发布消息称，计划2017年前全面完成长安街及其延长线的市容环境景观提升工程，其中2016年的工作目标包括完成民航营业大厦等10栋建筑物外立面改造设计方案并启动实施<ref>{{Cite web |url = http://news.sina.com.cn/c/nd/2016-03-22/doc-ifxqnskh1101188.shtml |title = 长安街及沿线城市设施将更换“中国风格” |author =  |date = 2016-03-22 |publisher =  新浪|language =  |accessdate =  |quote =  }}</ref>。2017年1月，[[北京市规划和国土资源管理委员会|北京市规划和国土资源管理委员会]]根据报[[北京市人民政府|北京市人民政府]]同意的方案，向建设单位核发了同意其装修民航营业大厦外立面的函<ref name=guihua/>。 民航大厦外立面改造工程拆除工作已于2016年12月进场实施<ref>{{Cite web|url =http://www.bjmac.gov.cn/zwxx/zwdtxx/zwgzdt/201703/t20170315_38626.html|title =民航大厦外立面改造工程顺利推进|author =|date =2017-03-15|publisher =北京市城市管理委员会|language =|accessdate =|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20180216143948/http://www.bjmac.gov.cn/zwxx/zwdtxx/zwgzdt/201703/t20170315_38626.html|archivedate =2018年2月16日|df =}}</ref><ref>{{Cite web|url =http://www.bjmac.gov.cn/zwxx/zwdtxx/zwgzdt/201612/t20161216_31303.html|title =长安街环境提升办公室现场查看绿化护栏提升试点和民航营业大厦改造提升施工情况|author =|date =2016-12-16|publisher =北京市城市管理委员会|language =|accessdate =|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20180216143946/http://www.bjmac.gov.cn/zwxx/zwdtxx/zwgzdt/201612/t20161216_31303.html|archivedate =2018年2月16日|df =}}</ref>。2017年9月，据[[北京市城市管理委员会|北京市城市管理委员会]]介绍，[[建国门_(北京)|建国门]]至[[复兴门_(北京)|复兴门]]段共13栋建筑物外立面改造提升中，[[长安俱乐部|长安俱乐部]]、[[北京长途电话大楼|联通长话大楼]]、[[北京电报大楼|电报大楼]]、东单营业厅、[[北京百盛购物中心|百盛大厦]]、[[中化大厦|中化大厦]]已经完工，民航营业大厦和[[中国教育电视台|中国教育电视台]]正在施工，[[秀水街|秀水街]]大厦正办理施工手续，[[北京图书大厦|北京图书大厦]]和北京市邮政公司正进行深化设计<ref>{{Cite web |url =  http://www.takefoto.cn/viewnews-1282752.html|title =  天安门地区景观提升拟于明年5月底前完成|author =  |date = 2017-09-28 |publisher = 北京晚报 |language =  |accessdate =  |quote =  }}</ref>。

民航营业大厦原先是从[[西单|西单]]到[[首都国际机场|首都国际机场]]的机场巴士的上下车站<ref>{{Cite web |url = http://news.sohu.com/20080302/n255467887.shtml |title =首都机场大巴车程增半小时 有望据客流量增派车  |author =  |date =2008-03-02  |publisher = 搜狐 |language =  |accessdate =  |quote =  }}</ref>。2005年10月20日，又开通了从民航营业大厦至[[北京南苑机场|北京南苑机场]]的机场巴士<ref>{{Cite web |url = http://news.carnoc.com/list/58/58822.html |title = 北京西单民航大厦至南苑机场大巴20日起开行 |author =  |date =2005-10-22 |publisher =民航资源网  |language =  |accessdate =  |quote =  }}</ref>。自2016年8月16日起，机场巴士西单站从民航营业大厦迁至西单路口南侧路西，[[光大银行|光大银行]]南侧公交车站<ref>{{Cite web |url = http://www.takefoto.cn/viewnews-895028.html |title = 南苑机场多条巴士线路站点调整 各站票价均维持不变 |author =  |date = 2016-08-25|publisher =  北晚新视觉|language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://finance.ifeng.com/a/20160816/14750293_0.shtml |title = 北京机场巴士西单站迁址 更靠近公交地铁站 |author =  |date = 2016-08-16 |publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref>。
