'''外侧沟'''（英：Lateral sulcus），亦称为'''外侧裂'''或'''侧脑沟'''，是[[脑|脑]]的最为显著的一个解剖构造。外侧沟的分隔[[大脑|大脑]]三个主要“叶”。外侧沟的下方是[[颞叶|颞叶]]，上方是[[额叶|额叶]]和[[顶叶|顶叶]]。其中，额叶位于顶叶的前部。两者被[[中央沟|中央沟]]分隔。

== 解剖与发育 ==
大脑的两侧各有一个外侧沟，大体为对称。但是多数[[人类|人类]]的左侧的外侧沟较长。外侧沟有一些分支。最为显著的两个分支包括
# '''上升支'''（Ascending ramus），亦称为竖直支（Vertical ramus）
# '''水平支'''（Horizontal ramus）
外侧沟的上升支和水平支位于额叶的[[额下回|额下回]]。

从脑发育的角度来说，外侧沟是最早形成的脑解剖结构之一。研究表明，外侧沟最早形成于[[妊娠|妊娠]]第14周。<ref>

== 别名 ==
外侧沟时常被称为“Sylvian沟”。这是因为Leiden大学的医学教授Franciscus Sylvius最早对其命名。
