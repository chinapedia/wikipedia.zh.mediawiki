[[File:mc_Epicycloid.png|thumb]]
'''外摆线'''是所有形式为

:<math>x = \cos \theta  + \frac{1}{n}\cos n\theta</math>
:<math>y = \sin \theta  + \frac{1}{n}\sin n\theta</math>

的曲线，其中''n''为正实数。

== 轨迹定义 ==
[[File:mc_Epicycloid_Roll.gif|thumb]]
假设有一个定[[圆|圆]]，若有另一个[[半径|半径]]是刚才的圆形的<math>\frac{1}{{n - 1}}</math>倍的圆在上滚动，则[[圆周|圆周]]上的一定[[点|点]]在滚动时划出的[[轨迹|轨迹]]就是一条外摆线。

== 心脏线 ==
[[File:mc_Cardioid.png|thumb]]

{{main|心脏线}}

'''心脏线'''是外摆线的一种，其{{math|''n''}}为2。它亦可以极坐标的形式表示：

{{math|1=''r'' = 1 + cos ''θ''}}

这样的心脏线的[[周界|周界]]为8，围得的[[面积|面积]]为<math>{3 \over 2} \pi</math>。
心脏线亦为[[蚶线|蚶线]]的一种。

在[[曼德博集合|曼德博集合]]正中间的图形便是一个心脏线。

心脏线的英文名称「Cardioid」是de Castillon在1741年的《Philosophical Transactions of the Royal Society》发表的；意为「像心脏-{的}-」。
