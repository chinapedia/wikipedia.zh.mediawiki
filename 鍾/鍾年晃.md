'''鍾年晃'''（{{Bd|1966年|10月27日|}}），[[臺灣|臺灣]][[苗栗縣|苗栗]][[頭份市|頭份]][[客家民系|客家人]]，[[淡江大學|淡江大學]][[法文|法文]]系畢業，報社[[記者|記者]]出身，現為臺灣知名[[媒體人|媒體人]]、[[時事評論員|時事評論員]]、[[電台|電台]]與[[電視|電視]][[政論節目|政論節目]][[主持人|主持人]]。

==簡介==
*曾任[[臺北市議會|臺北市議會]]助理，曾任《[[臺灣時報|臺灣時報]]》、《[[聯合報|聯合報]]》、《[[TVBS周刊|TVBS周刊]]》、《[[臺灣蘋果日報|臺灣蘋果日報]]》[[記者|記者]]。現為媒體工作者、政論[[名嘴|名嘴]]、[[臺灣社|臺灣社]]之[[臺灣角社|臺灣角社]]社長，多從事政治評論，亦為[[三立新聞台|三立新聞台]]《[[大話新聞|大話新聞]]》節目常態來賓。目前也擔任[[綠色和平電台|綠色和平電台]]節目主持人。

*2011年鍾年晃一度被[[民進黨|民進黨]]直接徵召為[[臺北市|臺北市]][[萬華區|萬華區]]、[[臺北市中正區|中正區]][[立法委員|立法委員]]提名人選，競爭者[[陳嘉銘|陳嘉銘]]認為應以[[民意調查|民意調查]]為主。另一位競爭者[[顏聖冠|顏聖冠]]則提出，高知名度的名嘴應到無人參選的選區較佳。最後鍾年晃表示，與陳、顏兩人皆為朋友，所以自動退出[[民意調查|民意調查]]。

*近期因在電視上批評前[[中國國民黨秘書長|中國國民黨秘書長]][[金溥聰|金溥聰]]，遭到金溥聰控告[[侵權|侵權]]，鍾年晃一審敗訴；鍾年晃表示「我沒有輸掉尊嚴」，一定會上訴到底。2012年6月19日[[臺灣高等法院|臺灣高等法院]]合議庭二審判決金溥聰敗訴<ref>[http://www.taiwanus.us/MediaVideoAudio/books/dpp/index.htm 《失落的民進黨》節錄]</ref><ref>{{Cite web |url=http://www.libertytimes.com.tw/2011/new/may/3/today-p1-2.htm |title=跳過民調 徵召鍾年晃 引爆爭議 |access-date=2011-08-16 |archive-url=https://web.archive.org/web/20110828101620/http://www.libertytimes.com.tw/2011/new/may/3/today-p1-2.htm |archive-date=2011-08-28 |dead-url=yes }}</ref><ref>{{Cite web |url=http://www.libertytimes.com.tw/2011/new/may/12/today-p2.htm |title=北市綠委提名 簡余晏、顏聖冠將獲徵召 |access-date=2011-08-16 |archive-url=https://web.archive.org/web/20110516151212/http://www.libertytimes.com.tw/2011/new/may/12/today-p2.htm |archive-date=2011-05-16 |dead-url=yes }}</ref><ref>{{Cite web |url=http://www.libertytimes.com.tw/2011/new/may/7/today-p5.htm |title=金溥聰告鍾年晃 一審勝訴 |access-date=2011-08-16 |archive-url=https://web.archive.org/web/20110828212236/http://www.libertytimes.com.tw/2011/new/may/7/today-p5.htm |archive-date=2011-08-28 |dead-url=yes }}</ref><ref>{{Cite web |url=http://www.nownews.com/2011/05/07/91-2710644.htm |title=控金溥聰操弄媒體敗訴　鍾年晃：不幸遇到「暴龍法官」 |access-date=2011-08-16 |archive-url=https://web.archive.org/web/20110507224849/http://www.nownews.com/2011/05/07/91-2710644.htm |archive-date=2011-05-07 |dead-url=yes }}</ref><ref>{{Cite web |url=http://tw.news.yahoo.com/%E9%87%91%E6%BA%A5%E8%81%B0%E6%8E%A7%E9%8D%BE%E5%B9%B4%E6%99%83%E6%90%8D%E5%AE%B3%E5%90%8D%E8%AD%BD-%E4%BA%8C%E5%AF%A9%E6%95%97%E8%A8%B4-213000987.html |title=金溥聰控鍾年晃損害名譽 二審敗訴 |access-date=2012-11-29 |archive-url=https://web.archive.org/web/20120628080436/http://tw.news.yahoo.com/%E9%87%91%E6%BA%A5%E8%81%B0%E6%8E%A7%E9%8D%BE%E5%B9%B4%E6%99%83%E6%90%8D%E5%AE%B3%E5%90%8D%E8%AD%BD-%E4%BA%8C%E5%AF%A9%E6%95%97%E8%A8%B4-213000987.html |archive-date=2012-06-28 |dead-url=yes }}</ref>。

*曾在政論節目談年金改革時，未經查證說出教師「年資從15歲算」，引發眾怒後僅在臉書低調更正，並未道歉。<ref>[http://www.ettoday.net/news/20160725/741957.htm 教師「年資從15歲算」　鍾年晃臉書認錯挨批「沒誠意」].ETtoday新聞雲.2016-07-25</ref>。
