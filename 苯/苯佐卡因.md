'''苯佐卡因'''（Benzocaine），又称“阿奈斯台辛”（Anaesthesine），学名[[对氨基苯甲酸|对氨基苯甲酸]]乙[[酯|酯]]。

== 性质 ==
无色或白色[[斜方晶系|斜方]]结晶粉末。无臭，味微苦，后有麻醉感。遇光逐渐变色。难溶于水，微溶于[[脂肪油|脂肪油]]，溶于稀酸、[[乙醇|乙醇]]、[[乙醚|乙醚]]、[[氯仿|氯仿]]、[[杏仁油|杏仁油]]、[[橄榄油|橄榄油]]。<ref>{{cite web|url=http://www.cnm21.com/00new/list.asp?id=10687|title=苯佐卡因|publisher=医药在线|accessdate=2009-09-04|deadurl=yes|archiveurl=https://web.archive.org/web/20060614010928/http://www.cnm21.com/00new/list.asp?id=10687|archivedate=2006-06-14}}</ref>

== 制取 ==
可由[[对硝基甲苯|对硝基甲苯]]被[[重铬酸钠|重铬酸钠]]／[[浓硫酸|浓硫酸]]氧化为[[对硝基苯甲酸|对硝基苯甲酸]]，再与[[乙醇|乙醇]][[酯化|成酯]]并被[[铁|铁]]／[[冰醋酸|冰醋酸]]还原而得。<ref>{{cite web|url=http://www1.syphu.edu.cn/ywhx/images/b7.doc|title=实验七  苯佐卡因（Benzocaine）的合成|accessdate=2009-09-04|deadurl=yes|archiveurl=https://web.archive.org/web/20061230220448/http://www1.syphu.edu.cn/ywhx/images/b7.doc|archivedate=2006-12-30}}</ref>

== 用途 ==
用作[[局部麻醉药|局部麻醉药]]。作用持久，毒性小。主要用于皮肤病止痒，创伤、[[痔疮|痔疮]]及[[溃疡|溃疡]]面等止痛。有[[避孕套|避孕套]]公司將苯佐卡因加在避孕套上，以延缓男性在性生活中[[射精|射精]]的時間。
