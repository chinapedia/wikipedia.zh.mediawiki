'''知識的代價'''（英語：'''The Cost of Knowledge'''）是一個由學者發起的運動，抗議學術期刊出版社[[愛思唯爾|愛思唯爾]]（Elsevier）的經營模式。
這個運動發起的主因是該出版社過高的收費，他們呼籲降低購買期刊的費用，以及期望促進學術資訊的[[開放獲取|開放獲取]]。主要的行動是號召認同理念的學者簽署聲明，拒絕投稿、[[同行評審|審搞]]和協助編輯愛思唯爾旗下的期刊。

== 緣起 ==
在[[網際網路|網際網路]]興起之前，將研究成果傳遞出去是一件辛苦的事情。過去出版社肩負起[[校對|校對]]、[[排版|排版]]、[[審稿|審稿]]、[[印刷|印刷]]等工作，並將印刷品傳遞到世界各處。<ref name="Taylor">{{cite journal|last=Taylor|first=Mike|title=It's Not Academic: How Publishers Are Squelching Science Communication|journal=[[發現_(雜誌)|Discover]]|date=21 February 2012|url=http://blogs.discovermagazine.com/crux/2012/02/21/its-not-academic-how-publishers-are-squelching-science-communication/|accessdate=22 February 2012}}</ref> 然而在現代，研究工作者被認為需要給予出版社無需再經修改的數位版本。對於數位傳輸，印刷不再必要、複製也不需費用，運輸經由網路迅速傳遞。因此，網路科技讓四家主要的科學出版社：[[愛思唯爾|愛思唯爾]]、[[施普林格科學+商業媒體|施普林格]]、[[約翰威立|約翰威立]]、[[Informa|Informa]]
能夠縮減支出，並且持續的獲得超過33%的高利潤率。<ref name="Taylor"/>

2012年1月21日，數學家[[蒂莫西·高爾斯|蒂莫西·高爾斯]]在個人部落格上發表一篇文章，呼籲抵制出版社「愛思唯爾」。該篇文章吸引了相當的注意，許多人視其為此運動的開端。<ref name="occupyelsevier">{{cite web |url=http://the-scientist.com/2012/02/07/occupy-elsevier/ |title=Occupy Elsevier? |first=Bob |last=Grant |work=[[The_Scientist|The Scientist]] |date=7 February 2012 |accessdate=12 February 2012 |archive-url=https://web.archive.org/web/20120720132514/http://the-scientist.com/2012/02/07/occupy-elsevier/ |archive-date=2012-07-20 |dead-url=yes }}</ref><ref>{{cite web |url= http://www.forbes.com/sites/timworstall/2012/01/28/elseviers-publishing-model-might-be-about-to-go-up-in-smoke/ |title=Elsevier's Publishing Model Might be About to Go Up in Smoke |first=Tim |last=Worstall |work=forbes.com |date=28 January 2012 |accessdate=12 February 2012}}</ref>高爾斯主張抵制運動有三個理由：過高的個別期刊訂閱費用、不同價值及重要性期刊的訂閱捆綁、以及愛思唯爾對 [[禁止網路盜版法案|SOPA]]、 [[保護知識產權法案|PROTECT IP Act]]和[[研究著作法案|研究著作法案]]的支持。<ref name=Economist>{{cite web|url=http://www.economist.com/node/21545974|title=Scientific publishing: The price of information|work=[[The_Economist|The Economist]]|author=|date=2012-02-04|archiveurl=https://www.webcitation.org/65VHxfKJ9?url=http://www.economist.com/node/21545974|archivedate=2012年2月16日|deadurl=no}}
</ref><ref name="Guardian">{{cite news |url=http://www.guardian.co.uk/science/2012/feb/02/academics-boycott-publisher-elsevier |title=Scientists sign petition to boycott academic publisher Elsevier |first=Alison |last=Flood |work=[[The_Guardian|The Guardian]] |date=2 February 2012 |publisher=[[Guardian_Media_Group|GMG]] |location=[[London,_England|London]] |issn=0261-3077 |oclc=60623878 |archiveurl=https://www.webcitation.org/65VHiGo2Z?url=http://www.guardian.co.uk/science/2012/feb/02/academics-boycott-publisher-elsevier |archivedate=2012年2月16日 |deadurl=no }}
</ref><ref name="chronicle">{{cite web |url=http://chronicle.com/blogs/wiredcampus/elsevier-publishing-boycott-gathers-steam-among-academics/35216 |title=Elsevier Publishing Boycott Gathers Steam Among Academics |first=Josh |last=Fischman |work=[[The_Chronicle_of_Higher_Education|The Chronicle of Higher Education]] |date=30 January 2012 |archiveurl=https://www.webcitation.org/65VIHYC2S?url=http://chronicle.com/blogs/wiredcampus/elsevier-publishing-boycott-gathers-steam-among-academics/35216 |archivedate=2012年2月16日 |deadurl=no }}</ref>

愛思唯爾反对這主張，宣稱其價格低於業界標準，說明訂閱捆绑只是購買期刊的一種選項。<ref name=Guardian/> 同時出版社也聲明其高獲益率是「單純來自於公司有效率的運作<ref name=Economist/>」批評者指出2010年「愛思唯爾」的利潤率是36%，總收入32億美元。<ref>{{cite web |url= http://www.bostonglobe.com/opinion/2012/02/12/why-scientists-are-boycotting-publisher/9sCpDEP7BkkX1INfakn3NL/story.html |title=Why scientists are boycotting a publisher — Opinion — The Boston Globe |first=Garret |last=Cook |work=bostonglobe.com |date=12 February 2012 |accessdate=12 February 2012}}</ref> 愛思唯爾自己宣稱其2010年的營業利潤率為25.7%。<ref>{{cite web |url=http://reports.reedelsevier.com/ar10/overview/2010-highlights.htm |title=2010 highlights |first= |last= |work=reports.reedelsevier.com |year=2012 [last update] |quote=operating margin |accessdate=17 February 2012 |deadurl=yes |archiveurl=https://web.archive.org/web/20111221162414/http://reports.reedelsevier.com/ar10/overview/2010-highlights.htm |archivedate=2011年12月21日 }}</ref>

== 網站 ==

一個名為「知識的代價」网站誕生，邀請研究人員與學者簽署承諾聲明：不向愛思唯爾旗下的期刊投稿、为其担任同行评审和編輯。2012年2月1日，声明約有一千个署名。<ref>{{cite web |url= http://www.bloomberg.com/news/2012-02-13/bard-motorola-medicaid-bullfrog-intellectual-property-1-.html |title=Bard, Motorola, Medicaid, Bullfrog: Intellectual Property |first=Victoria |last=Slind-Flor |work=bloomberg.com |date=13 February 2012 |accessdate=13 February 2012}}</ref>稍晚在2月，簽署的人數超過 7,700 人。<ref name="Guardian"/><ref name="chronicle"/>許多署名來自數學、計算機科學和生物等領域的学者。<ref>{{cite web |url= http://newsbreaks.infotoday.com/NewsBreaks/The-Cost-of-Knowledge-Versus-Elsevier--Signatures-and-Growing-80575.asp |title=The Cost of Knowledge Versus Elsevier: 5,600 Signatures and Growing |first=Robin |last=Peek |work=[[Information_Today,_Inc.|Information Today, Inc.]] |date=13 February 2012 |accessdate=13 February 2012}}</ref>
