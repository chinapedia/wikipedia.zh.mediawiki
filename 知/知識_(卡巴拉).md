'''Da'at''' 或 '''Daas'''（英文：Knowledge；[[希伯来文|希伯来文]]：דעת [ˈdaʕaθ]；中文：知识）在[[犹太教|犹太教]]神秘主义体系（即[[卡巴拉|卡巴拉]]）中，位于[[生命树_(卡巴拉)|生命之树]]中。在Da'at 所在之处，生命之树的十个[[质点_(卡巴拉)|质点]]合为一体。

在Da'at中，所有[[质点_(卡巴拉)|质点]]都存在于无限共享的完美之处。[[生命树_(卡巴拉)|生命之树]]左列的三个[[质点_(卡巴拉)|质点]]接受并隐藏神性之光，但并不分享或显露光。由于所有[[质点_(卡巴拉)|质点]]都播散着无限而无私的神性之光，[[质点_(卡巴拉)|质点]]间将不能再区分彼此，因此十个[[质点_(卡巴拉)|质点]]合为一体。

在[[质点_(卡巴拉)|质点]]的图示表示中，Da'at通常不会被描绘，从某种意义上说，这被认为是“空位”，其它任何[[质点_(卡巴拉)|质点]]的宝石都能嵌入其中。神性之光的光辉一直闪耀，透过Da'at或隐藏或显现，但无人能见。这种或隐藏或显现的变化并非源于Da'at自身，而''似乎''是源于[[Malkuth|Malkuth]]中的人类视角。只有位于[[Malkuth|Malkuth]]（中文：王国）处才能察觉这种变化。那些如神性之光般无私的人会看到光，Da'at的神性之光的福祉将会为他们“显现”。而自私之人则看不到光，福祉也会对他们“隐藏”。

== 视Da'at为典型的质点 ==

Da'at并非质点，但它却是合为一体的十个[[质点_(卡巴拉)|质点]]。然而有时Da'at会被视为[[质点_(卡巴拉)|质点]]并取代Keter，这相当于——使用Da'at从有限创造的观点角度来表现Keter无限大的“映像”（或“内部尺寸”）。因此，Da'at沿[[生命树_(卡巴拉)|生命之树]]中轴线出现于[[质点_(卡巴拉)|质点]]构成中，位于Kether的正下方。这种关系所对应的是嵌入于人类的[[神的形象|神的形象]]（即''[http://en.wikipedia.org/wiki/Tzelem_elohim Tzelem elohim]''）<ref>{{cite web | title=bible-magazine.net | work= 聆聽聖言:Tzelem Elohim 按照神的樣式 | url=http://www.bible-magazine.net/tc/download/25/25-8.htm}}</ref>，以及[[小脑|小脑]]。

== 层次 ==

Da'at会在两个层次起作用：

#较高层次，被称为''[http://en.wikipedia.org/wiki/Da%27as_Elyon_and_Da%27as_Tachton Da'at Elyon]''（“较高层次的知识”）或者 ''Da'at hane'elam''（“隐藏的知识”），这会确保才智的两种更高层次能量 ——chochmah（中文：智慧）与Binah（中文：理解）——能够持久结合。
#较低层次，被称为''[http://en.wikipedia.org/wiki/Da%27as_Elyon_and_Da%27as_Tachton Da'at Tachton]'' （“较低层次的知识”）或者 ''Da'at hamitpashet''（“扩展的知识”），这会确保才智能够将与[[情感|情感]]世界连接为一个整体 ，从而增强信心，决心依基要真理行事，并将之融入[[意识|意识]]之中。
