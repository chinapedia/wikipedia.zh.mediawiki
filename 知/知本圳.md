'''知本圳'''為一座位於[[臺灣|臺灣]][[臺東縣|臺東縣]][[卑南鄉|卑南鄉]][[知本|知本]]地區的[[灌溉|灌溉]]水圳系統，其最早由當地[[原住民|原住民]]族[[卑南族|卑南族]]所開設，後臺灣進入[[日治時期|日治時期]]再由[[台灣總督府|台灣總督府]]接收並管理，目前由[[臺東縣|臺東]][[農田水利會|農田水利會]][[知本|知本]][[工作站|工作站]]進行管理與維護。<ref>{{cite web |url=http://www.ttia.gov.tw/page4_ziban.php |title=台東農田水利會 知本工作站 | accessdate=2014-12-15}}</ref>

== 沿革 ==

=== 知本圳 ===

知本圳最初由於知本地區土地荒蕪，因此聯合當地知本與[[建和|建和]]兩村之村民進行開設，工程於西元1899年（民國前13年，清光緒25年）動工，並取水自[[知本溪|知本溪]]，歷時1年於西元1900年 （民國前12年，清光緒26年）正式完工通水。<ref name="源遠流長　台灣農田水利圳路史 知本圳">{{cite web |url=http://theme.imita.org/htmlarea_file/web_articles/coa/13493/57.pdf |title=源遠流長　台灣農田水利圳路史 知本圳 |accessdate=2014-12-15 }}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

臺灣進入[[台灣日治時期|日治時期]]後，日本台灣總督府於西元1915年（民國4年）將知本圳納入管轄，並設立為知本[[公共埤圳|公共埤圳]]，西元1930年（民國19年）再成立知本[[水利組合|水利組合]]對知本圳進行管理，西元1940年（民國29年）併入[[卑南鄉|卑南]]水利組合，西元1943年（（民國32年）再併入台東水利組合。<ref name="源遠流長　台灣農田水利圳路史 知本圳" />

西元1924年（民國13年），由於知本圳最初由地方人士開設，在當時已有20年歷史，再加上設施簡陋，且知本溪河床時有變動，取水量不足，以至於當時的灌溉面積僅有60餘公頃，因此日本台灣總督府撥款交由[[台灣拓殖株式會社|台東拓殖株式會社]]改建，並增建第一美和圳（該圳目前為現今的[[美和圳|美和圳]]），而原知本圳則改名為'''第二美和圳'''。 爾後由於第二美和圳引自知本溪溪水，因次又改回知本圳之名。<ref name="源遠流長　台灣農田水利圳路史 知本圳" />

[[臺灣戰後時期|中華民國接管臺灣後]]，當時[[經濟部水利署|台灣省政府水利局]]接管知本圳，鑑於知本圳因無沉沙池設置，導致大量泥沙流入圳內，因此於西元1947年（民國39年）規劃建設[[沉沙池|沉沙池]]一座。<ref name="源遠流長　台灣農田水利圳路史 知本圳" />

== 設施 ==

=== 進水口 ===

一座，進水量每秒3.50立方公尺。<ref name="源遠流長　台灣農田水利圳路史 知本圳" />

=== 沉沙池 ===

一座，長70公尺，寬15公尺。<ref name="源遠流長　台灣農田水利圳路史 知本圳" />

=== 導水設施 ===

幹線總長5.6公尺，幹線[[制水門|制水門]]2座，[[分水門|分水門]]5座。<ref name="源遠流長　台灣農田水利圳路史 知本圳" />

=== 水利公園 ===

* 知本圳進水口親水公園

知本圳進水口親水公園佔地約1.4公頃，公園內花木扶疏，綠意盎然，有健康步道、休閒涼亭、觀景亭等設施，也有小橋流水、大小奇石、水車等造景。<ref>{{cite web |url=http://tour.taitung.gov.tw/zh-tw/Travel/ScenicSpot/280/%E7%9F%A5%E6%9C%AC%E5%9C%B3%E8%A6%AA%E6%B0%B4%E5%85%AC%E5%9C%92 |title=台東縣政府 台東觀光網 知本圳親水公園 | accessdate=2014-12-15}}</ref>

== 改善工程 ==

* 知本圳幹線歷年改善工程<ref name="台東農田水利會 知本圳幹線歷年改善工程">{{cite web |url=http://www.ttia.gov.tw/page6-4-10.php |title=台東農田水利會 知本圳幹線歷年改善工程 | accessdate=2014-12-15}}</ref>

知本圳惟早期進水口及圳路設施老舊，圳路多為[[土渠|土渠]]或乾砌塊石，因易[[崩壞作用|崩塌]]及雜草竹木繁生，影響灌溉甚鉅。台東農田水利會會有鑑於此，即擬定計畫分年分期予以改善，自西元1993年（民國82年）起至西元1998年（民國87年）止，分年編列預算辦理更新。<ref name="台東農田水利會 知本圳幹線歷年改善工程" />
