《'''知音漫客'''》是由知音动漫有限公司主办发行的[[漫画杂志|漫画杂志]]，创刊于2006年1月1日<ref name=":2">{{Cite web|url = http://news.xinhuanet.com/book/2011-11/22/c_122318202.htm|title = 80后出版人捣鼓《知音漫客》 团队成员个个都牛|accessdate = |author = 华西都市报|date = 2011年11月22日|publisher = 新华网}}</ref>，主要刊登、连载16开全彩页漫画<ref>{{Cite web|url = http://comic.qq.com/a/20130207/000020.htm|title = 中国梦 漫客梦知音动漫新春献词！|accessdate = |author = |date = 2013年2月7日|publisher = 腾讯动漫}}</ref>。价格统一为5[[人民币|元]]。主要发行于[[中国大陆|中国大陆]]地区，读者对象多为中学生<ref>{{Cite web|url = http://www.chinaxwcb.com/index/2009-06/08/content_174504.htm|title = 《知音漫客》拉动原创漫画走向复兴|accessdate = |author = |date = 2009-06-08|publisher = 中国新闻出版网}}</ref>。2007年，朱家君（李靖）将知音漫客改为半月刊，2008年改为旬刊每刊64页，2009年10月以后，知音漫客的出版频率改为每周一刊每刊80页<ref name=":2" /><ref name=":5">{{Cite web|url = http://news.xinhuanet.com/society/2009-11/20/content_12504562.htm|title = 我国首家全彩色连载漫画周刊《知音漫客》将发行|accessdate = |author = |date = 2009年11月20日|publisher = 新华网}}</ref>。<ref name=":6">{{Cite web|url =http://comic.qq.com/a/20090917/000040.htm|title =原创漫画杂志《知音漫客》简介|accessdate =|author =|date =2009年9月17日|publisher =腾讯动漫|deadurl =yes|archiveurl =https://web.archive.org/web/20151222115953/http://comic.qq.com/a/20090917/000040.htm|archivedate =2015年12月22日}}</ref>

== 历史 ==
知音漫客创刊于2006年1月1日[[湖北|湖北]][[武汉|武汉]]，母公司为知音传媒，创始人为朱家君（又名李靖），首印10万册，朱家君将初始价格定每册2[[人民币|元]]<ref name=":2" />。2008年，知音漫客入选[[中国新闻出版总署|中国新闻出版总署]]扶持原创动漫作品项目<ref name=":3">{{Cite web|url = http://www.changjiangtimes.com/2014/06/480648.html|title = 从品牌到产业——知音传媒玩转“传统”与“跨界”|accessdate = |author = |date = 2014-06-16|publisher = 长江商报}}</ref><ref>{{Cite web|url = http://www.chinaxwcb.com/xwcbpaper/html/2008-06/23/content_33790.htm|title = 2007年原创动漫作品扶持项目公示名单|accessdate = |author = |date = 2008年6月23日|publisher = 中国新闻出版报}}</ref>，获得20万扶持基金<ref name=":7">{{Cite web|url = http://www.chinaxwcb.com/2010-08/11/content_204322.htm|title = 《知音漫客》百万基金重奖原创动漫|accessdate = |author = |date = 2010-08-11|publisher = 中国新闻出版网}}</ref>。2010年，知音漫客入选国家新闻出版总署“原动力”中国原创动漫出版扶持计划<ref name=":3" />，2011年知音漫客得到湖北省新闻出版局分配的报刊发展专项引导资金100万元<ref>{{Cite web|url = http://news.xinhuanet.com/zgjx/2012-05/17/c_111973366.htm|title = 湖北省局整合资源扶植优质品牌报刊|accessdate = |author = |date = 2012年5月17日|publisher = 中国新闻出版网}}</ref>。
