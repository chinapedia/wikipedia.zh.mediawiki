《'''早辰。早晨'''》（{{lang-en|'''This Morning'''}}）是[[香港電台|香港電台]]電視部製作的晨早時事資訊節目，提供天氣、交通、財經消息，亦包括報章摘要、社評、重點新聞及時事熱話等，是香港第一個提供即時[[手語翻譯|手語翻譯]]的直播時事資訊節目，於2016年4月4日起，逢星期一至五早上7點半於[[港台電視31、31A|港台電視31、31A]]播出。香港其他電視台同類晨早時事節目節目有[[無綫電視|無綫電視]]的《[[香港早晨|香港早晨]]》、[[香港有線電視|有線電視]]的《[[Cable早晨|Cable早晨]]》及[[now寬頻電視|now寬頻電視]]的《[[now早晨|now早晨]]》，而手語新聞時事節目則有同台的《時事摘錄》及[[無綫電視|無綫電視]]的《[[手語新聞報道|手語新聞報道]]》。

== 歷史 ==
因[[亞洲電視|亞洲電視]]不獲[[行政長官會同行政會議|行政長官會同行政會議]]續牌，[[香港特別行政區政府|香港特別行政區政府]]將亞視的模擬頻譜轉交予港台。[[商務及經濟發展局|商務及經濟發展局]]局長[[蘇錦樑|蘇錦樑]]有意將[[港台電視|港台電視]]模擬化，取代[[亞視本港台|亞視本港台]]的模擬頻道，並增設電視新聞的可行性。廣播處長[[梁家榮|梁家榮]]隨後表示，港台能在現有資源下，提供晨早資訊節目<ref>{{cite news|url=http://news.rthk.hk/rthk/ch/component/k2/1242556-20160216.htm |title=梁家榮保證港台接手亞視頻道後提供晨早資訊節目|work=[[香港電台|香港電台]]|date=2016年2月6日}}{{zh-hant}}</ref>。當亞視牌照於2016年4月2日到期，港台隨即接手其模擬頻譜，31台並會延長播放時間到19小時，由早上6時30分廣播至凌晨1時30分，早上則會提供新聞資訊節目《'''早辰。早晨'''》。

* 2016年4月4日，節目正式啟播。
* 2017年3月9日起，節目主持由兩人改為一人。
* 2017年9月11日起，節目以全新面貌登場，採用新片頭設計，並改用虛擬場景中進行直播。

== 內容 == 
每集節目有兩名主持（2017年3月9日起改為一位主持），報道每日新聞重點、網上熱話、報章及社評摘要，亦有《講式早餐》（星期一）、《羅啟.新棟篤笑》（星期三）和戶外直播等環節。除此之外，節目亦提供即時手語翻譯。<ref>{{cite news|url=http://www.hk01.com/%E6%B8%AF%E8%81%9E/13209/%E6%B8%AF%E5%8F%B0%E6%8E%A5%E6%89%8B%E4%BA%9E%E8%A6%96%E9%A0%BB%E9%81%93-%E6%8E%A8%E6%89%8B%E8%AA%9E%E5%82%B3%E8%AD%AF%E5%8F%8A%E5%8D%B0%E5%B0%BC%E8%AA%9E%E5%AD%97%E5%B9%95%E7%AF%80%E7%9B%AE-%E7%85%A7%E9%A1%A7%E4%B8%8D%E5%90%8C%E8%A7%80%E7%9C%BE |title=港台接手亞視頻道　推手語傳譯及印尼語字幕節目　照顧不同觀眾|work=[[香港01|香港01]]|date=2016年3月23日}}{{zh-hant}}</ref>

== 節目名稱 ==
節目名稱《'''早辰。早晨'''》的「'''辰'''」意指時辰，代表節目於辰時（早上7-9時）期間播出。
