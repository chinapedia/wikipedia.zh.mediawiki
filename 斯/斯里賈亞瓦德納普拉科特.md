'''斯里賈亞瓦德納普拉科特'''（{{lang-si|ශ්‍රී ජයවර්ධනපුර කෝට්ටේ}}、{{lang-ta|ஸ்ரீ ஜயவர்த்தனபுரம் கோட்டே}}，又译'''斯里賈亞瓦德納普拉科泰'''）是[[斯里蘭卡|斯里蘭卡]]的[[首都|首都]]。因名稱较長，通常略稱為'''科特'''、'''科泰'''。

== 概要 ==
科泰與舊首都[[可倫坡|可倫坡]]同處於國境西部，在後者市中心以東約15公里處。1985年自[[可倫坡|可倫坡]][[遷都|遷都]]於此，但是從2007年以來，除[[國會|國會]]、森林局等少數機構以外，首都機能的大部份尚未從可倫坡轉移過來。

科泰的街市與可倫坡連成一片，事實上即科倫坡郊外的一角。

== 历史 ==
{{main|斯里蘭卡歷史|錫蘭}}
1415年[[科提王國|科提王國]]定都於此，同時定名為「賈亞瓦德納普拉」（僧伽羅語的“勝利”之意）。其後，先後受[[葡萄牙|葡萄牙]]、[[荷蘭|荷蘭]]、[[英國|英國]]的殖民統治。殖民地時代稱之為「科泰」。1977年，當局計劃將可倫坡的首都機能轉移至科泰，1983年國會大堂建成。當時的[[朱尼厄斯·理查德·贾亚瓦尔德纳|朱尼厄斯·理查德·贾亚瓦尔德纳]]總統將首都名稱改為與自己姓氏相同的古名。該市的正式名称為「斯里・賈亞瓦爾達納普拉・科泰」（即“神聖的・勝利之城・科泰”）。1985年，當局正式宣佈由可倫坡遷都於此。其名稱的意涵是，「斯里（光輝的）・賈亞瓦爾達納（第二屆[[斯里蘭卡總統|斯里蘭卡總統]]的名字）・普拉（城市）・科泰（舊名）」。國會大堂建立於室內人造湖中央的島上。

== 區劃 ==
科泰主要由以下5個地區構成。其中拉賈吉利亞區為市議會 (Municipal Council) 的所在。努杰哥達區為[[斯里蘭卡鐵路|斯里蘭卡鐵路]]的主要車站及[[斯里賈亞瓦納德普拉科泰大學|斯里賈亞瓦納德普拉科泰大學]]的所在。

* 耶圖爾科泰
* 納瓦拉
* 努傑哥達
* 比塔科泰
* 拉賈吉利亞

== 人口 ==
科泰是多民族、多宗教人士混居的城市。為數最多的是斯里蘭卡的主要民族[[僧伽羅人|僧伽羅人]]，其他有[[泰米爾人|泰米爾人]]、[[穆斯林|穆斯林]]、[[馬來人|馬來人]]及[[巴格爾人|巴格爾人]]等民族。穆斯林和泰米爾中的大多數居住在拉賈吉利亞區。其他如華人和馬來人，以及印度人亦有少量存在。2001年[[人口普查|人口普查]]所示的民族及宗教構成如下。

== 氣候 ==
根據[[柯本氣候分類法|柯本氣候分類法]]，科泰屬於[[熱帶雨林氣候|熱帶雨林氣候]]。氣溫變化幅度小，濕度高。歷年的2月為最乾燥月，但是其月降水量仍有63mm之多，全年沒有其他平均降水量少於60mm的月份。
