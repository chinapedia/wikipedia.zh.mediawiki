'''斯特林发动机'''（Stirling Engine）又名'''熱空氣引擎'''，一种闭循环[[往复式发动机|活塞式]][[热机|热机]]。闭循环的意思是膨胀和压缩介质一直保存在[[气缸|气缸]]内，而开循环，如[[内燃机|内燃机]]需要与[[大气|大气]]交换气体。斯特林发动机一般被归为[[外燃机|外燃机]]。

== 历史 ==
伦敦的牧师罗巴特斯特林（Robert Stirling）于1816年发明的，所以命名为“斯特林发动机”（Stirling engine）。

== 工作原理 ==
[[File:史特林引擎2種─大學生5分鐘.ogv|thumb]]

斯特林发动机通过气体吸收热源的内能，[[热胀冷缩|热胀冷缩]]做功，然后接触冷源，气体压缩，使活塞复位。发动机内的工作气体处于封闭中，因此可以用[[無活性氣體|惰性气体]]来作为工质。以简单的双缸斯特林为例，斯特林发动机的工作过程可以分为如下几个阶段<ref>[http://auto.bowenwang.com.cn/stirling-engine4.htm 斯特林发动机工作原理] {{webarchive|url=https://web.archive.org/web/20090711102256/http://auto.bowenwang.com.cn/stirling-engine4.htm |date=2009-07-11 }}</ref>：
#右侧[[气缸|气缸]]与热源接触，缸内气体受热膨胀，对右侧活塞做功，活塞上升，接近于等温膨胀过程。
#左侧活塞上升，右侧[[活塞|活塞]]下降，工作气体进入左侧气缸，接近于等容放热过程。
#左侧气缸与冷源接触，缸内气体冷却收缩，左侧活塞下降，接近于等温压缩过程。
#左侧活塞继续下降，工作气体返回右侧气缸，接近于等容吸热过程，完成一个循环。
理想情况下，等容放热量与等容吸热量相同，等温膨胀吸热量高于等温压缩放热量，输出机械能。

配气活塞式斯特林发动机的原理，配气活塞斯特林发动机有配气活塞和动力活塞两个活塞。其中配气活塞并'''不密封'''，气体可以从其和气缸间的缝隙中自由流动：
#底部气体受热，配气活塞向上运动。
#配气活塞上部气体受到压迫，一部分推动动力活塞运动，一部分从缝隙中进入配气活塞底部受热膨胀继续推动配气活塞。
#动力活塞转动，带动飞轮，飞轮带动配气活塞向下运动。
#配气活塞压迫气体进入上方，气体被冷却收缩，配气活塞上升，完成一个循环。

== 背景 ==
在將熱變成[[機械功|機械功]]的轉換上，斯特林引擎在真實的[[熱機|熱機]]中可達最高的[[熱效率|熱效率]]，至多80%，僅受[[工作氣體|工作氣體]]和引擎材料的[[不理想性質|不理想性質]]限制，例如[[摩擦|摩擦]]、[[熱導率|熱傳導性]]、抗張強度、緩慢、[[熔點|熔點]]等。

此引擎理論上可用任何足量的[[熱源|熱源]]運行，包括[[太陽能|太陽能]]、[[化學能|化學能]]和[[核能|核能]]。

與[[內燃機|內燃機]]相比，斯特林引擎往往維修需求較低，更高效、更安靜、而且更可靠。它們傾向被應用於某些特殊用途以發揚其獨特優點。特別是首要目標非減低每單位功率的[[投資成本|投資成本]]（金錢/千瓦），而是減低引擎產生每單位能量的成本（金錢/度）的時候。

在額定[[功率|功率]]下，斯特林引擎的投資成本目前比內燃機引擎高，而且通常更大更重，因此這引擎科技很少單獨以此作為競爭基準。然而在一些用途上（例如需要靜音的[[潛艇|潛艇]]<ref>[http://news.enorth.com.cn/system/2009/05/14/004025026.shtml 零距离细看日本最强AIP潜艇工艺]</ref>），適當的本益分析可令斯特林引擎優於內燃機引擎。

近年來，鑑於能源成本普遍上漲，能源短缺和[[全球暖化|全球暖化]]之類的環境問題，斯特林引擎的優點愈來愈顯著。也有人将斯特林发动机运用在了实验车辆上。<ref>[http://news.epochtimes.com.tw/7/1/25/46742.htm 萬能科大光電系周鑑恆教授成功研發出載人的史特林引擎車]</ref>
對斯特林引擎科技提高興趣促進了斯特林裝置的研發。其應用涵蓋藉由不相容於內燃機的豐富能源抽水、宇基太空航行、發電，像是太陽能、農業廢料還有家庭垃圾。

另一個斯特林引擎的潛力是，若供應機械功，它可以作為一種[[熱泵|熱泵]]。已有實驗利用[[風能|風能]]驅動斯特林熱泵作為家用冷暖[[空調|空調]]。[[File:MM-7_Stirling_Engine.jpg|thumb]]
