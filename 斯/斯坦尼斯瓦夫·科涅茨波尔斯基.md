'''斯坦尼斯瓦夫·科涅茨波尔斯基'''（{{lang|pl|Stanisław Koniecpolski}}，{{bd|1590年/1594年{{Ref_label|a|a|none}}||1646年|3月11日|catIdx=K}}）是[[波蘭立陶宛聯邦|波兰立陶宛联邦]]的[[貴族|贵族]]（[[波兰语|波兰语]]：[[施拉赫塔|szlachta]]）和[[权贵|权贵]]，曾任多个地区的[[长老_(斯拉夫人)|长老]]和城主。他自1625年起，历任[[桑多梅日省|桑多梅日省]]省长、王室陆军指挥官和王室大指挥官（[[总司令|级位仅次于国王的军事指挥官]]）。

科涅茨波尔斯基戎马一生，战绩卓著，在20岁前就已参加[[波俄战争_(1605-1618)|德米特里战争]]和[[摩尔达维亚权贵战争|摩尔达维亚权贵战争]]，曾在1620年的[[楚措拉战役|楚措拉战役]]中被俘，后来在1623年释放。1624年至1626年间，他击败了奥斯曼帝国的盟友[[鞑靼人|鞑靼人]]。

波瑞战争（1626年-1629年）中，他赶赴[[普魯士|普鲁士]]，与[[古斯塔夫二世·阿道夫|古斯塔夫·阿道夫]]麾下的[[瑞典|瑞军]]交战，虽然在兵力上占下风，却仍能战至僵持。后来这场战争进入到第二阶段，科涅茨波尔斯基在7月15日至8月9日间多次向瑞军发起进攻，但都被击退。另外，他在1634年于[[乌克兰|乌克兰]]的[[卡缅涅茨-波多利斯基|卡缅涅茨-波多利斯基]]击退了奥斯曼人的一次大规模入侵。在其他战役中，他也多次战胜起义的[[哥薩克|哥萨克人]]与来犯的鞑靼人。

他被公认为是在[[波兰|波兰]]和[[立陶宛|立陶宛]]历史上最有能力也最著名的军事指挥官。<ref name=dict>George J. Lerski, Jerzy Jan Lerski, Piotr Wróbel, Richard J. Kozicki, "Historical dictionary of Poland, 966-1945", Greenwood Publishing Group, 1996, 第262页, [http://books.google.com/books?id=S6aUBuWPqywC&pg=PA262&dq=Stanis%C5%82aw+Koniecpolski+1632&as_brr=3]</ref><ref name=WIEM/><ref name="Podhorodecki">{{cite book|author=Leszek Podhorodecki|title=Stanisław Koniecpolski ok. 1592–1646|url=http://books.google.com/books?id=B69FAAAAIAAJ|accessdate=9 March 2011|year=1978|publisher=Wydawnictwo Ministerstwa Obrony Narodowej|location=Warszawa}}</ref>{{rp|429}}

== 童年 ==
斯坦尼斯瓦夫·科涅茨波尔斯基出生于1590年至1594年间{{Ref_label|a|a|none}}，他可能出生在贵族与权贵家族[[科涅茨波尔斯基家族|科涅茨波尔斯基家族]]的发祥地——[[科涅茨波尔|科涅茨波尔]]。他的父亲[[亚历山大·科涅茨波尔斯基_(1555-1609)|亚历山大·科涅茨波尔斯基]]是一位富有的权贵，曾任[[謝拉茲省|谢拉兹省]]省长，同时也是出身于瑞典[[瓦薩王朝|瓦萨王室]]的国王[[齐格蒙特三世|齐格蒙特三世]]的有力支持者；<ref name=WIEM>{{pl icon}} [http://portalwiedzy.onet.pl/21838,,,,koniecpolski_stanislaw,haslo.html Koniecpolski Stanisław], [[WIEM_百科|WIEM 百科]], 2009年9月4日查阅</ref>而母亲是[[卡缅涅茨-波多利斯基省|卡缅涅茨-波多利斯基省]]城主[[斯坦尼斯瓦夫·斯罗奇茨基|斯坦尼斯瓦夫·斯罗奇茨基]]之女安娜·斯萝奇茨卡，也因为她，科涅茨波尔斯基在[[波多里亚|波多里亚]]获得了大量地产。斯坦尼斯瓦夫的兄弟有[[克雷什托夫·科涅茨波尔斯基|克雷什托夫·科涅茨波尔斯基]]（''chorąży koronny''，[[贝沃兹省|贝沃兹省]]省长）、[[雷米吉乌什·科涅茨波尔斯基|雷米吉乌什·科涅茨波尔斯基]]（[[赫乌姆主教|赫乌姆主教]]，在1640年逝世）、[[扬·科涅茨波尔斯基|扬·科涅茨波尔斯基]]（谢拉兹省省长及谢拉兹城主）和普雷德博尔·科涅茨波尔斯基（1611年逝世）。<ref name="Podhorodecki"/>{{rp|14-15}}

[[File:Rzeczpospolita-ch.png|thumb]]在1648年达到全盛时期时的疆域]]

科涅茨波尔斯基有语言障碍，在說語彙较长的词时总会结巴。他在15岁时利用父亲在宫廷中的影响力而成为[[维隆|维隆]]市长老（相當於市长），此时（1603年）他也就读于[[雅盖隆大学|克拉科夫专科学校]]。<ref name=dict/><ref name="Podhorodecki"/>{{rp|22-23}}读了几年书后，他被父亲送入王室，施行能邁入上流社會的教育，他於該處待了一两年。<ref name="Podhorodecki"/>{{rp|28}}據推測，他可能曾出访过[[西欧|西欧]]（主要是法国），并在那里住了几个月。<ref name="Podhorodecki"/>{{rp|28}}后来，科涅茨波尔斯基回到了他的家族庄园。<ref name="Podhorodecki"/>{{rp|28}}

== 早年生涯 ==
科涅茨波尔斯基选择投身军旅生涯，他在1610年参与了[[波俄战争_(1605-1618)|德米特里战争]]。<ref name=dict/>他随后也参与了[[克武申战役|克武申战役]]。1611年7月8日[[斯摩棱斯克围攻战_(1609-1611)|围攻斯摩棱斯克]]时，倒塌的城墙压死了他的兄弟布雷德博尔，斯坦尼斯瓦夫在战后将他兄弟的遗体运回科涅茨波尔。<ref name="Podhorodecki"/>{{rp|33-34}}同年秋，他重返军队，并在立陶宛[[盖特曼|盖特曼]]（一個僅次於國王的軍銜）[[扬·卡罗尔·霍德凯夫斯基|扬·卡罗尔·霍德凯夫斯基]]的指挥下，为被困在[[克里姆林宫|克里姆林宫]]的波军提供物资，脱离困境。<ref name="Podhorodecki"/>{{rp|35}}那时，他被指挥官委以指挥波军右翼的重要任务。<ref name="Podhorodecki"/>{{rp|35}}

科涅茨波尔斯基在1612年于乌克兰加入由指挥官[[斯坦尼斯瓦夫·若乌凯夫斯基|斯坦尼斯瓦夫·若乌凯夫斯基]]领导的联邦正规军。<ref name="Podhorodecki"/>{{rp|35}}斯坦尼斯瓦夫深受若乌凯夫斯基的影响。<ref name="Podhorodecki"/>{{rp|36}}1614年，他負責镇压由[[扬·卡尔瓦茨基|扬·卡尔瓦茨基]]领导的正规军叛乱行動。<ref name="JB">Jerzy Besala, ''Stanisław Żółkiewski'', PIW, 1988年</ref><ref name="Podhorodecki"/>{{rp|42-43}}他在5月17日与[[扬·若乌凯夫斯基|扬·若乌凯夫斯基]]一同於[[罗哈腾战役|罗哈腾战役]]中取胜，并将卡尔瓦茨基俘虏。<ref name="Podhorodecki"/>{{rp|42-43}}1615年，他与若乌凯夫斯基的女儿[[卡塔日娜·若乌凯夫斯卡|卡塔日娜]]结婚。<ref name="Podhorodecki"/>{{rp|42-43}}在这桩婚事过后，他晋升为王冠领地内侍。<ref name name="Podhorodecki"/>{{rp|44}}

[[File:Lesseur-Hetman_Żółkiewski.jpg|left]]

科涅茨波尔斯基在1615年和1616年于乌克兰得到了与鞑靼人作战的经验，但未能击溃或俘虏任何一支鞑靼军队。<ref name name="Podhorodecki"/>{{rp|45}}1616年，他的第一位妻子在她的第一个儿子安德热出生时去世了。<ref name name="Podhorodecki"/>{{rp|46-47}}1617年，他与若乌凯夫斯基一道参加[[摩尔达维亚权贵战争|摩尔达维亚权贵战争]]，与[[厄斯坎德尔·帕沙|厄斯坎德尔·帕沙]]麾下的强大土军作战。冲突最终在年内以和解告终。<ref name name="Podhorodecki"/>{{rp|46-47}}科涅茨波尔斯基也与住在[[奥尔沙尼察|奥尔沙尼察]]附近的哥萨克人谈判，[[登记哥萨克|登记哥萨克]]（在联邦军队享有特权的哥萨克）被限制在10000人以内，且不可到[[黑海|黑海]]掠夺他人财产。<ref name name="Podhorodecki"/>{{rp|48-49}}哥萨克人通过掠夺奥斯曼的富有城市敛取钱财，但也会招惹那些城市，让他们对联邦采取报复行动。<ref name="Podhorodecki"/>{{rp|40-41}}

国王[[齐格蒙特三世|齐格蒙特三世·瓦萨]]在1618年联邦议会（[[波兰共和国下议院|瑟姆]]）召开时，不顾[[克雷什托夫·兹巴拉斯基|克雷什托夫·兹巴拉斯基]]及其党羽的反对，任命斯坦尼斯瓦夫·若乌凯夫斯基为王室大指挥官，科涅茨波尔斯基为王室陆军指挥官。<ref name=dict/><ref name name="Podhorodecki"/>{{rp|51}}

不久，科涅茨波尔斯基因在极端不利的条件下贸然追击敌军，于[[奥伦|奥伦]]被鞑靼人击败，自己侥幸逃生。<ref name name="Podhorodecki"/>{{rp|56}}科涅茨波尔斯基后来与[[克雷斯蒂娜·卢博米尔斯卡|克雷斯蒂娜·卢博米尔斯卡]]结婚，她在1620年为科涅茨波尔斯基生下一名男婴[[亚历山大·科涅茨波尔斯基_(1620-1659)|亚历山大]]。<ref name name="Podhorodecki"/>{{rp|58}}

1620年，科涅茨波尔斯基和若乌凯夫斯基带兵至[[楚措拉|楚措拉]]，与坎提米尔帐国（[[铁米尔汗|铁米尔汗]]）交战。这次战役中，联邦军队至少10000人，由来自克雷茨基家族、扎斯瓦夫斯基家族、卡扎诺夫斯基家族、卡利诺夫斯基家族和[[波托茨基家族|波托茨基家族]]的权贵招募的私人武装构成。科涅茨波尔斯基在这场战役中指挥联邦军队右翼。<ref name=WIEM/>波军在9月19日被击败，但他们依然能够有组织地撤退。但波军士气低下；9月20日至21日，科涅茨波尔斯基试图阻止军队崩溃，但就在后来的撤退的过程当中，军队轰然解体，并冲向河去。在随后的战役中，若乌凯夫斯基战死，科涅茨波尔斯基和诸多权贵（[[塞缪尔·科雷茨基|塞缪尔·科雷茨基]]、[[米科瓦伊·斯特鲁西|米科瓦伊·斯特鲁西]]、斯坦尼斯瓦夫之子[[扬·若乌凯夫斯基|扬·若乌凯夫斯基]]和[[乌卡什·若乌凯夫斯基|乌卡什·若乌凯夫斯基]]）被俘。<ref name name="Podhorodecki"/>{{rp|65-100}}这些战俘们先被转移到[[比亚沃格拉德|比亚沃格拉德]]和伊斯坎德尔·帕沙那里，随后又被转移到[[君士坦丁堡|君士坦丁堡]]附近的[[七塔城堡|七塔城堡]]，囚禁在其中的黑塔。1621年，[[霍腾战役_(1621年)|奥斯曼军队在霍腾被击败]]，随后波土关系正常化，在这种情况下，这些战俘们于1623年春回到波兰。波土关系能够稳定，主要在于克雷什托夫·兹巴拉斯基的外交使节团，这个使节团将30000名[[鞑靼|鞑靼]]战俘释放。<ref name=dict/> <ref name name="Podhorodecki"/>{{rp|101-110}}

[[File:Castle_of_Seven_Towers_Istanbul.png|thumb]]

[[卡尼贝克·吉雷|卡尼贝克·吉雷]]汗尽力遵守[[霍腾条约|霍腾条约]]这一防止进一步边界冲突的条约，而[[铁米尔汗|铁米尔汗]]（坎提米尔）为了篡夺前者的权力，继续突袭边界地区。<ref name="Podhorodecki"/>{{rp|113}}他在1623年6月重新开始突袭；随后，科涅茨波尔斯基得到了当地联邦军队的指挥权，受命阻止铁米尔汗的入侵。<ref name="Podhorodecki"/>{{rp|114-115}}1623年7月，科涅茨波尔斯基在[[特雷姆布罗夫拉|特雷姆布罗夫拉]]与鞑靼人发生冲突。<ref name=WIEM/>坎提米尔的军队（''horda budziacka''）也在1624年2月左右入侵南波兰。<ref name="Podhorodecki"/>{{rp|115-120}}其中一支部队在2月6日于[[什马尼科维采|什马尼科维采]]和[[奥尔什科夫采|奥尔什科夫采]]附近被科涅茨波尔斯基拦截击溃；<ref name="Podhorodecki"/>{{rp|115-120}}而另一支在6月20日左右稍晚些时于[[马尔蒂努夫_(乌克兰)|马尔蒂努夫]]同样被击败，这种战局迫使[[阿里|阿里]]·帕沙的军队在一片混乱中撤至[[布科维纳|布科维纳]]。<ref name=Jacques>Tony Jaques, "Dictionary of Battles and Sieges: A-E", Greenwood Publishing Group, 2007, 第155页, [http://books.google.com/books?id=3amnMPTPP5MC&pg=PA155&dq=Koniecpolski+Martynow&as_brr=3]</ref><ref name="interia">{{pl icon}} [http://encyklopedia.interia.pl/haslo?hid=79491 Koniecpolski Stanisław], [[Encyklopedia_Internautica|Encyklopedia Internautica]], retrieved on 16 July 2009</ref><ref name="Podhorodecki"/>{{rp|125-134}}坎提米尔的军队在6月5日越过边界，但在6月20日被指挥官科涅茨波尔斯基击溃。科涅茨波尔斯基运用一种新战术——哥萨克轻骑兵与鞑靼骑兵速度相当，可以利用他们将敌人赶到被称为“车阵”（Tabor）的加强防御据点前，而火器和炮兵在车阵里据守。1625年，科涅茨波尔斯基因取得了这几场冲突的胜利，得到瑟姆30000[[波兰兹罗提|兹罗提]]的奖励，同时被任命为桑多梅日省省长。<ref name="Podhorodecki"/>{{rp|125-134}}

1625年，[[扎波罗热_(地区)|扎波罗热]]哥萨克在[[马尔科·日麦洛|马尔科·日麦洛]]的领导下发动起义，他们与[[尚辛·吉雷伊|尚辛·吉雷伊]]结盟，并尝试将莫斯科公国拉到他们一边（[[日麦洛起义|日麦洛起义]]）。<ref name="Podhorodecki"/>{{rp|135}}科涅茨波尔斯基想到，鞑靼人与奥斯曼中央政府之间有冲突，因此坎提米尔的布吉亚茨卡部落不会派主力支援。<ref name="Podhorodecki"/>{{rp|136-137}}他从正规军和私人军队那里收集了兵力达12000人的军队。科涅茨波尔斯基保证所有忠于联邦的哥萨克人都会享有公平的待遇，并击溃了[[马尔科·日麦洛|马克·日马伊瓦]]手下的剩余起义者。<ref name="Podhorodecki"/>{{rp|137-138}}1625年10月25日，他在克雷科夫附近向哥萨克人发起进攻，那些哥萨克人设法抵御骑兵的第一次进攻，向库鲁科夫斯基湖撤退。他们试图再抵御第二次进攻，科涅茨波尔斯基“一度处于极度的危险之中”。<ref name="Podhorodecki"/>{{rp|139-142}}这次冲突以[[库鲁科夫条约|库鲁科夫条约]]告终，双方停火；登记哥萨克人数被降至6000人；他们再次保证不会劫掠黑海沿岸地区惹怒鞑靼人了。<ref>Brian L. Davies, "Warfare, state and society on the Black Sea steppe, 1500-1700", Routledge, 2007, 第100页, [http://books.google.co.uk/books?id=XH4hghHo1qoC&pg=PA100&dq=Kurukove+Tatars&lr=%20&as_brr=3#v=onepage&q=Kurukove%20Tatars&f=false]</ref><ref name="Podhorodecki"/>{{rp|143-144}}

1626年1月末，鞑靼人带着兵力达15000人至20000人左右的军队再次入侵，劫掠了远至[[波多尔省|波多尔省]]的土地，并将这些地方彻底夷为平地，穿过[[特尔诺皮尔|特尔诺皮尔]]和[[泰莱波夫利亚|泰莱波夫利亚]]，而一些先头部队已经抵达了[[卢茨克|卢茨克]]、[[弗拉基米尔-沃伦斯基|弗拉基米尔-沃伦斯基]]和[[利沃夫|利沃夫]]城。<ref name="Podhorodecki"/>{{rp|145}}科涅茨波尔斯基聚集了13000人的军队，并开始拦截鞑靼人，但他们拒绝参与战斗。<ref name="Podhorodecki"/>{{rp|146-147}}最后科涅茨波尔斯基试图摧毁鞑靼军队的后方，该防线位于波鞑交界上，那里有很多财宝和奴隶。<ref name="Podhorodecki"/>{{rp|146-147}}在这年晚些时候，科涅茨波尔斯基因为害怕鞑靼人的再次入侵，将瑟姆声明撕毁，征募部署了8000人的军队，以抵御料想中的鞑靼第二次入侵。<ref name="Podhorodecki"/>{{rp|150-152}}在这时的很多战役中，科涅茨波尔斯基得到了一位有能力的军官[[博格丹·赫梅利尼茨基|博格丹·赫梅利尼茨基]]的帮助；在这一年科涅茨波尔斯基启程去北方，去他在[[波罗的海|波罗的海]]附近的新战场时，赫梅利尼茨基也对鞑靼人取得了一场重大胜利。<ref name="Podhorodecki"/>{{rp|150-152}}

== 战争（1626年–1629年） ==
[[File:Gustav_II_Adolf_by_Merian.jpg|thumb]]
1626年，联邦北方的硝烟盖过了南方。这一年，瑞典人越过联邦边界，重燃[[波瑞战争|波瑞战争]]的战火。<ref name=Bain>R. Nisbet Bain, "Slavonic Europe - A Political History of Poland from 1447 to 1796", READ BOOKS, 2006, 第155页, [http://books.google.co.uk/books?id=PdZBZcUbSewC&pg=PA155&dq=Polish+Swedish+war+1626+Koniecpolski&lr=%20&as_brr=3#v=onepage&q=%20&f=false]</ref>1626年5月，古斯塔夫·阿道夫率领由125艘船组成的舰队，和至少14000多名士兵逼近波兰海岸，并向经过[[格但斯克|格但斯克]]的贸易活动课税。<ref name="Podhorodecki"/>{{rp|167}}攻下[[波罗的斯克|皮瓦瓦]]<ref>Norman Davies, "God's playground", Oxford University Press, 2005, 第238页</ref>和[[布拉涅沃|布拉涅沃]]后，瑞军横扫[[波美拉尼亞|波美拉尼亚]]，先后占领[[弗洛姆博尔克|弗洛姆博尔克]]、[[托尔克米茨科|托尔克米茨科]]、[[埃尔布兰格|埃尔布兰格]]、[[马尔堡|马尔堡]]、[[格涅夫|格涅夫]]、[[特切夫|特切夫]]和[[斯塔洛格拉德|斯塔洛格拉德]]；另一支瑞军在[[普茨克|普茨克]]登陆，并占领该城（[[波兰立陶宛联邦舰队|波兰立陶宛联邦舰队]]的主要港口）<ref name="Podhorodecki"/>{{rp|168}}但是，格但斯克等主要城市即便是在瑞军势如破竹的攻势前也绝不投降。<ref name="Podhorodecki"/>{{rp|168}} 1626年9月22日至30日，联邦与瑞典在格涅夫村附近交火，在这次战役中，古斯塔夫击败了由国王齐格蒙特亲自统帅的联邦军队。齐格蒙特撤军，并从联邦的其他地区搬救兵。<ref name="Podhorodecki"/>{{rp|173-174}}科涅茨波尔斯基受命保卫[[王室普鲁士|王室普鲁士]]；但由于南方的局势还不稳定，行程推迟。他最终在10月1日到达王室普鲁士。<ref name=dict/><ref name=WIEM/><ref name="Podhorodecki"/>{{rp|175}}

科涅茨波尔斯基的军队由4200名轻骑兵，1000名[[龍騎兵|龙骑兵]]和1000名步兵构成，很快就到达了普鲁士。<ref name="Podhorodecki"/>{{rp|175-176}} 在其他部队的支援下，他手头上共有9000名士兵与兵力达20000人的瑞军相抗衡。<ref name=Bain/>但波德赫莱茨基的估计却有些许不同——是由15000余人的波军（其中包括素质不佳的格但斯克步兵）抗衡21000人的瑞军。<ref name="Podhorodecki"/>{{rp|175-176}} 他通过[[運動戰|运动战]]，利用小批机动兵力切断敌人的通信线并将敌方较小的部队消灭，来阻止瑞典人进攻，这也迫使[[阿克谢尔·奥克森斯提尔那|阿克谢尔·奥克森斯提尔那]]的部队由攻转守。<ref name=Scott>Franklin D. Scott, "Sweden", SIU Press, 1992, 第172页, [http://books.google.com/books?id=Qv8zxie3A18C&pg=PA172&dq=Oxenstierna+Koniecpolski&as_brr=3#v=onepage&q=Oxenstierna%20Koniecpolski&f=false]</ref><ref name="Podhorodecki"/>{{rp|177-178}}与此同时，瑟姆同意为这次战争筹款，<ref name="Podhorodecki"/>{{rp|179-182}}但是波军缺钱又缺粮，形势依然紧迫。<ref name="Podhorodecki"/>{{rp|183-185}}1626年12月，立陶宛军队在[[因弗兰提省|因弗兰提省]]的科克尼斯附近被击败，被迫撤到[[西德维纳河|西德维纳河]]后。<ref name="Podhorodecki"/>{{rp|185}}瑞典人计划兵分两路——奥克森斯提尔那从[[维斯瓦河|维斯瓦河]]出发，而[[约翰·斯特雷夫·冯·拉文茨泰恩|约翰·斯特雷夫·冯·拉文茨泰恩]]和[[马克西米连·丢弗|马克西米连·丢弗]]从[[瑞典|瑞占]][[波美拉尼亞|波美拉尼亚]]出兵——重创科涅茨波尔斯基。<ref name="Podhorodecki"/>{{rp|188-190}}结果因维斯瓦河泛滥，他们的计划都泡汤了，科涅茨波尔斯基得以拦截来自波美拉尼亚的敌军。<ref name="Podhorodecki"/>{{rp|191-192}}

[[File:Koniecpolski_small2.jpg|thumb]]

1627年4月2日，科涅茨波尔斯基成功收复[[普茨克|普茨克]]。<ref name=dict/><ref name=interia/><ref name=CHP>THE CAMBRIDGE HISTORY OF POLAND, CUP Archive, pg. 473, [http://books.google.com/books?id=N883AAAAIAAJ&pg=RA1-PA473&dq=Puck+Koniecpolski&as_brr=3#v=onepage&q=Puck%20Koniecpolski&f=false]</ref><ref name="Podhorodecki"/>{{rp|193-194}}1627年4月18日，波军打到[[恰尔内|恰尔内]]附近，迫使瑞军撤入城内；后者在一周后后投降，把他们的旗帜和军章都给丢掉了。<ref name=dict/><ref name=WIEM/><ref name="Podhorodecki"/>{{rp|200-204}}很多雇佣兵和一些瑞典士兵在那时都叛变了。<ref name="Podhorodecki"/>{{rp|200-204}}波军在1627年春接连告捷，使瑞军不仅失去他们位于维斯瓦河西岸的所有要塞，也失去速战速决的希望；在这种情况下，[[勃兰登堡|勃兰登堡]][[选帝侯|选帝侯]]正式宣布支持联邦。而立陶宛军队也再次发起对因弗兰提的进攻。<ref name="Podhorodecki"/>{{rp|206-207}}

5月17日，古斯塔夫·奥古斯特带领8000名援军登陆。<ref name="Podhorodecki"/>{{rp|211}}从5月22日至23日夜，古斯塔夫在[[凯日马克|凯日马克]]附近穿过维斯瓦河，在[[格但斯克|格但斯克]]附近遭遇波兰军队，因[[臀|臀部]]受伤被迫撤退。<ref name="Podhorodecki"/>{{rp|212-213}}后来，科涅茨波尔斯基决定重夺[[格涅夫|格涅夫]]并制定了一份牵制计划。<ref name="Podhorodecki"/>{{rp|212-213}}他发兵[[布拉涅沃|布拉涅沃]]，制造试图攻占这座城镇的假象，迫使古斯塔夫解围；随后古斯塔夫跟随撤退的波军，并围攻[[奥尔内塔|奥尔内塔]]。<ref name="Podhorodecki"/>{{rp|212-213}}而科涅茨波尔斯基这时预感到瑞军的动向，向他的首要目标格涅夫发动突袭。<ref name=dict/><ref name="Podhorodecki"/>{{rp|212-213}}据称，古斯塔夫对科涅茨波尔斯基的回应速度印象深刻。<ref name="Podhorodecki"/>{{rp|214}}

科涅茨波尔斯基率领大约7800名士兵（包括2500名骑兵和联邦精锐重骑兵——骠骑兵），试图阻止瑞军从[[特切夫|特切夫]]附近入侵格但斯克。<ref name="Podhorodecki"/>{{rp|215-221}}从8月7日到8月8日，科涅茨波尔斯基与瑞军（10000名士兵，其中包括5000名步兵）在[[莫乌塔瓦湖|莫乌塔瓦湖]]沼泽附近交战。<ref name="Podhorodecki"/>{{rp|215-221}}瑞军希望诱使波军进攻，再利用自己的步兵和[[炮兵|炮兵]]将其消灭，但科涅茨波尔斯基决定不去进攻。<ref name="Podhorodecki"/>{{rp|215-221}}瑞军后来先发制人，派骑兵开始进攻，但未能将波军吸引到他们的火力范围之内。瑞军进攻令波军骑兵伤亡惨重，但并没有使波军陷入瘫痪（因为科涅茨波尔斯基，波军士气高昂）。<ref name="Podhorodecki"/>{{rp|215-221}}
最终，古斯塔夫·阿道夫再次受伤，瑞典撤军，[[特切夫战役|这场战役]]终于结束。<ref name=CHP/><ref name="Podhorodecki"/>{{rp|215-221}}

战后，科涅茨波尔斯基认识到军队改革的重要性，他认识到，在这个时候，将步兵和炮兵的火力加强到瑞军的那个水平上是必要的。<ref name="Podhorodecki"/>{{rp|221-225}}而瑞军也从波军那里学到了骑兵冲锋与混战的战术。<ref name="Podhorodecki"/>{{rp|221-222}}总的来说，1627年，战局的发展对联邦有利；普茨克和格涅夫都被收复，瑞典的计划遭到挫败，瑞军有所伤亡。<ref name="Podhorodecki"/>{{rp|222-223}}1627年11月28日，在[[奥利瓦战役|奥利瓦战役]]中，实力较弱的[[波兰海军|波兰海军]]奇迹般地击败了瑞典的一支小型舰队。<ref name="Podhorodecki"/>{{rp|222-223}}

1628年，波兰军队因缺乏资金支持，被迫由攻转守。<ref name="Podhorodecki"/>{{rp|223-225}}<ref name="Podhorodecki"/>{{rp|225-227}}古斯塔夫·阿道夫占领了[[诺维|诺维]]和[[布罗德尼察|布罗德尼察]]。科涅茨波尔斯基以最有效的手段——包括快速的骑兵混战配合步炮火力的支持、游击战术、利用工程兵修筑要塞、以及巧妙运用地形优势——利用他手头上的小支军队进行反击。<ref name="Podhorodecki"/>{{rp|230, 232}}瑟姆在[[斯坦尼斯瓦夫·波托茨基|斯坦尼斯瓦夫·波托茨基]]于[[戈日诺战役|戈日诺]]战败后，决定为这次战争进一步拨款。[[奥地利|奥地利]]派[[陸軍元帥|陆军元帅]][[扬·耶日·阿尔恩海姆|扬·耶日·阿尔恩海姆]]部支援联邦。<ref name="Podhorodecki"/>{{rp|237}}但是，阿尔恩海姆拒绝受科涅茨波尔斯基指挥。<ref name="Podhorodecki"/>{{rp|238}}

1629年6月27日，最后一战在[[奇强纳战役|奇强纳]]附近爆发。<ref name=dict/><ref name=WIEM/><ref name="Podhorodecki"/>{{rp|240-246}}瑞军向[[格鲁琼兹|格鲁琼兹]]方向发起进攻，但被阻断，瑞军随后撤至[[什图尔姆|什图尔姆]]和[[马尔堡|马尔堡]]。<ref name="Podhorodecki"/>{{rp|240-246}}科涅茨波尔斯基向伦伯爵[[扬·威廉·雷恩格拉夫|扬·威廉·雷恩格拉夫]]指挥的后防线发起进攻，并将其摧毁。<ref name=Peterson>Gary Dean Peterson, "Warrior kings of Sweden: the rise of an empire in the sixteenth and seventeenth centuries", McFarland, 2007, 第149页, [http://books.google.co.uk/books?id=CPJWfVPgFawC&pg=PT157&dq=Koniecpolski+Wrangel&lr=%20&as_brr=3#v=onepage&q=Koniecpolski%20Wrangel&f=false]</ref><ref name="Podhorodecki"/>{{rp|240-246}}他也击退了瑞军的手枪骑兵的反击，后者试图向[[普乌科维采|普乌科维采]]推进，在那里古斯塔夫·阿道夫随同2000名手枪骑兵发起另一次反击。<ref name="Podhorodecki"/>{{rp|240-246}}但这次反击也被阻断，陆军元帅[[赫尔曼·弗兰格尔|赫尔曼·弗兰格尔]]率领的最后的后备军阻止了波军的进攻，而这支军队也挽救了瑞军，使他们不至于落到一败涂地的境地。<ref name="Podhorodecki"/>{{rp|240-246}}古斯塔夫·阿道夫受伤，最后勉强自由逃离战场。<ref name="Podhorodecki"/>{{rp|240-246}}<ref name=dict/>在这场战役中，有1200名瑞军士兵被杀，扬·威廉·雷恩格拉夫与几百号人被俘。<ref name="Podhorodecki"/>{{rp|240-246}}波军只有不足200人战死或受伤。<ref name="Podhorodecki"/>{{rp|240-246}}

但联邦的军事胜利并没又转化为其在政治上或军事上的有利条件。1629年10月26日，双方在斯塔雷-塔尔戈签署停火协议，这份协议对瑞典有利，<ref name="Podhorodecki"/>{{rp|251-252}}<ref>[http://books.google.co.uk/books?id=AjIntLsWsMoC&pg=PA23&dq=Truce+of+Altmark&lr=%20&as_brr=3#v=onepage&q=%20&f=false Michael Spilling, "Estonia", Marshall Cavendish, 1999, 第23页]</ref>瑞典获得了向波兰在波罗的海的贸易活动征税的权力（贸易商品价值的3.5%），并继续统治王室普鲁士的诸多城市。<ref name="Podhorodecki"/>{{rp|251-252}}<ref>Richard Bonney, "The rise of the fiscal state in Europe, 约1200-1815", Oxford University Press, 1999, 第471页, [http://books.google.co.uk/books?id=U24lRLy_qT8C&pg=PA471&dq=Truce+of+Altmark&lr=%20&as_brr=3#v=onepage&q=%20&f=false]</ref>并被广泛认为是南波罗的海岸的统治势力。<ref name="Podhorodecki"/>{{rp|251-252}}科涅茨波尔斯基在谈判中起不到主要作用，那时他被叫回乌克兰以阻止鞑靼人入侵[[科德尼察|科德尼察]]附近<ref name=interia/>，还要处理由[[塔拉斯·费多罗维奇|塔拉斯·费多罗维奇]]领导的哥萨克起义。<ref name=WIEM/>

== 王室大指挥官 ==
[[File:Koniecpolski_near_halicz_19th_century.jpg|thumb]]绘（黑白复制品，原件在[[第二次世界大战|第二次世界大战]]中遗失）]]
1630年，塔拉斯处决了反对起义的[[赫雷霍雷·恰尔内|赫雷霍雷·恰尔内]]，<ref name=Plokhy>Serhii Plokhy, "The Cossacks and religion in early modern Ukraine", Oxford University Press, 2001, pg. 46, 136, [http://books.google.com/books?id=NCzzxNisc1MC&pg=PA136&dq=Stanis%C5%82aw+Koniecpolski&as_brr=3]</ref>占领了[[科尔松|科尔松]]要塞。科涅茨波尔斯基发动对[[佩列亚斯拉夫-赫梅利尼茨基|佩列亚斯拉夫]]的围攻，但因缺少步兵和炮兵的支持，未能攻破城墙。需要物资且遭受重大伤亡的哥萨克人同意进行协商。1630年8月，双方签署[[佩列亚斯拉夫条约_(1630年)|佩列亚斯拉夫条约]]，这部条约的条款对哥萨克起义者较为宽容，其中包括对那些起义者实行赦免。<ref>{{cite book|title=Ukraine: a history|author=[[Orest_Subtelny|Orest Subtelny]]|url=http://books.google.com/books?id=HNIs9O3EmtQC&pg=PA117&dq=Koniecpolski++Pereiaslav&as_brr=3&client=opera|publisher=[[University_of_Toronto|University of Toronto]] Press|year=2000|ISBN=9780802083906|page=第117页}}</ref><ref name="Podhorodecki"/>{{rp|265-267}}通常而言，科涅茨波尔斯基<ref name="Podhorodecki"/>{{rp|383-384}}支持严惩起义者，但他也认为，从长计议，如果给哥萨克提供更公平更合理的待遇，譬如增加哥萨克士兵数，按时给予他们军饷等等，哥萨克问题会解决得更好。<ref name="Podhorodecki"/>{{rp|267-268}}不过，他支持让哥萨克沦为农奴的政策，这是乌克兰动乱的主因之一；<ref name="Podhorodecki"/>{{rp|387}}而且，他决定通过剥削当地居民维持其军队的开销，而不是给他们发放军饷；因为上述两点原因，科涅茨波尔斯基在乌克兰极不受欢迎。<ref name=BrianLD>{{cite book|author=Brian L. Davies|title=Warfare, State and Society on the Black Sea Steppe, 1500–1700|url=http://books.google.com/books?id=XH4hghHo1qoC&pg=PA100|accessdate=9 March 2011|year=2007|publisher=Routledge|location= London ; New York|isbn=9780415239868|page=100}}</ref>

1632年，齐格蒙特三世·瓦萨在他弥留之际将科涅茨波尔斯基提拔为王室大指挥官。<ref name=dict/>自[[斯坦尼斯瓦夫·若乌凯夫斯基|斯坦尼斯瓦夫·若乌凯夫斯基]]战死后，这一职位已空缺12年；也许国王恐惧如果过早给予科涅茨波尔斯基这个职位，科涅茨波尔斯基就会权高镇主。<ref name="Podhorodecki"/>{{rp|270}}国王死后，科涅茨波尔斯基在联邦的政治事务中发挥主要作用，在[[波兰国王选举|国王选举]]中支持齐格蒙特之子[[瓦迪斯瓦夫四世|瓦迪斯瓦夫四世]]登基。<ref name=dict/><ref name="Podhorodecki"/>{{rp|271}}作为报答，在选后翌年，瓦迪斯瓦夫四世任命科涅茨波尔斯基为[[克拉科夫|克拉科夫]]城主，这是联邦最有声望的地方职位。科涅茨波尔斯基成为新王具有影响力的顾问，经常支持瓦迪斯瓦夫制定针对鞑靼人的外交政策。<ref name=dict/>科涅茨波尔斯基也支持瓦迪斯瓦夫四世的军事改革。<ref name=interia/><ref name="Podhorodecki"/>{{rp|315-323}}尽管大体上被视为国王的支持者，科涅茨波尔斯基反对瓦迪斯瓦夫四世增强王权削弱[[贵族民主制|贵族民主制]]的计划。<ref name="Podhorodecki"/>{{rp|315-323}}
[[File:Kamieniec_Podolski_widok_01.jpg|thumb]]

1633年，科涅茨波尔斯基阻止奥斯曼进攻联邦，7月4日在[[萨索维-鲁格|萨索维-鲁格]]战胜他们。<ref name=WIEM/><ref name=interia/><ref name="Podhorodecki"/>{{rp|281}}这一年10月22日，他在[[卡缅涅茨-波多利斯基|卡缅涅茨-波多利斯基]]击退超过20000人的奥斯曼军队，<ref name="Podhorodecki289-295">Podhorodecki, 第289-295页</ref>，而他自己的军队只有11000人。<ref name=Peterson/><ref>J. K. Fedorowicz, Maria Bogucka, Henryk Samsonowicz, "A Republic of nobles: studies in Polish history to 1864", CUP Archive, 1982, 第186页, [http://books.google.co.uk/books?id=p7U8AAAAIAAJ&pg=PA186&dq=Koniecpolski+Kamieniec+Podolski&lr=%20&as_brr=3#v=onepage&q=Koniecpolski%20Kamieniec%20Podolski&f=false]</ref><ref name="A Republic of Nobles: Studies in Polish History to 1864">{{cite book|author1=J. K. Fedorowicz|author2=Maria Bogucka|author3=Henryk Samsonowicz|title=A Republic of Nobles: Studies in Polish History to 1864|url=http://books.google.com/books?id=p7U8AAAAIAAJ&pg=PA186|accessdate=9 March 2011|year=1982|publisher=CUP Archive|isbn=9780521240932|page=186}}</ref>
土耳其人因这几次失败和科涅茨波尔斯基的态度在1634年9月签署一则新条约。<ref name="Podhorodecki312-314">Podhorodecki, 第312-314页</ref>该条约重申1621年的[[霍奇姆条约|霍奇姆条约]]，并为[[波土战争_(1633年-1634年)|波土战争（1633年—1634年）]]画上句号。<ref name="Podhorodecki"/>{{rp|312-314}}

1635年，[[伊凡·苏利马|伊凡·苏利马]]统帅的哥萨克军队占领，摧毁了联邦的[[柯达克要塞|柯达克要塞]]（位于现在的[[第聂伯罗彼得罗夫斯克|第聂伯罗彼得罗夫斯克]]附近），<ref name=autogenerated1>Serhii Plokhy, "The Cossacks and religion in early modern Ukraine", Oxford University Press, 2001, 第37, 46, 136页, [http://books.google.com/books?id=NCzzxNisc1MC&pg=PA37&dq=Ivan+Sulyma+Kudak&lr=%20&as_brr=3], [http://books.google.com/books?id=NCzzxNisc1MC&pg=PA136&dq=Stanis%C5%82aw+Koniecpolski&as_brr=3]</ref>科涅茨波尔斯基发起了重夺要塞，惩罚起义者的远征；苏利马遭到逮捕，随后被处决。<ref name=autogenerated4>R. Nisbet Bain, "Slavonic Europe - a Political History of Poland from 1447 to 1796", READ BOOKS, 2006, 第206-207页, [http://books.google.com/books?id=PdZBZcUbSewC&pg=PA207&dq=Koniecpolski+Ochmatow&lr=%20&as_brr=3]</ref><ref name="Podhorodecki"/>{{rp|325-326}}

这一年晚些时候，科涅茨波尔斯基回到[[波美拉尼亞|波美拉尼亚]]准备另一次对瑞作战，但因波瑞双方签署[[什图姆斯卡条约|什图姆斯卡条约]]，这次备战并不需要。<ref name="Podhorodecki"/>{{rp|326-335}}

科涅茨波尔斯基明白军队需要近代化，并与瓦迪斯瓦夫四世一起为这个目标努力，例如征募经历过西式战争的雇佣军，更进一步发展炮兵（他监督位于[[柯达克要塞|库达克]]、[[巴尔_(乌克兰)|巴尔]]和卡缅涅茨-波多利斯基的兵工厂的建设情况，并在他位于乌克兰的土地上建立锻造车间）。他是很多优秀的的炮兵部队和技术人员的赞助人。<ref name="Podhorodecki"/>{{rp|315-323}}他也许还赞助了一些制图师，譬如绘制乌克兰地图的威廉·勒·瓦瑟·博普朗，<ref name=Stone />和绘制[[克里米亚|克里米亚]]地图的[[瑟巴斯蒂安·阿德斯|瑟巴斯蒂安·阿德斯]]。<ref>[[拉珀斯维尔|拉珀斯维尔]]波兰博物馆, "Works by William le Vasseur de Beauplan", [http://www.muzeum-polskie.org/muzeum/beauplan-eng.htm]{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>他也支持建立[[波兰立陶宛联邦海军|联邦波罗的海舰队]]。<ref name=Stone>Daniel Stone, "The Polish-Lithuanian state, 1386-1795", University of Washington Press, 2001, 第152, 158, 195页, [http://books.google.com/books?id=LFgB_l4SdHAC&pg=PA151&dq=Koniecpolski+Baltic+Fleet&lr=%20&as_brr=3], [http://books.google.com/books?id=LFgB_l4SdHAC&pg=PA195&dq=Koniecpolski+Podhorce&lr=%20&as_brr=3], [http://books.google.com/books?id=LFgB_l4SdHAC&pg=PA158&dq=Koniecpolski+Tatars&lr=%20&as_brr=3]</ref>

== 晚年 ==
[[File:Nagrobek_hetmana_Koniecpolskiego_w_Koniecpolu.jpg|left]]三一教堂]]

1637年后，科涅茨波尔斯基日渐衰老，体力日渐衰弱，日渐依靠年轻的指挥官[[米克瓦伊·波托茨基|米克瓦伊·波托茨基]]，后者在1637年和1638年成功地镇压了哥萨克起义，并在1639年成功地镇压了一起鞑靼人起义。<ref name="Podhorodecki"/>{{rp|384}}科涅茨波尔斯基的名气也保护了亡命之徒[[塞缪尔·瓦什奇|塞缪尔·瓦什奇]]，前者将后者视为另一位有能力的指挥官。<ref>Orest Subtelny, "Ukraine: a history", University of Toronto Press, 2000, pg. 108, [http://books.google.com/books?id=HNIs9O3EmtQC&pg=PA108&dq=Koniecpolski+Laszcz&lr=%20&as_brr=3]</ref>

[[File:Mikołaj_Potocki.PNG|upright]]
他在1644年的冬季战役中战胜了奥斯曼军队，这也是他人生中的几次大捷之一。<ref name="Podhorodecki"/>{{rp|396-405}}1644年1月30日，带着兵力达19000名士兵的大军（他们中的60%是权贵的私人军队，科涅茨波尔斯基的私人军队有2200人），<ref name="Podhorodecki"/>{{rp|401}}他在[[奥赫马图夫|奥赫马图夫]]痛击了[[图阿伊贝伊|图阿伊贝伊]]部。<ref name=autogenerated2>R. Nisbet Bain, "Slavonic Europe - a Political History of Poland from 1447 to 1796"（《斯拉夫欧洲——自1447年至1796年的波兰政治史》）, READ BOOKS, 2006, 207页, [http://books.google.com/books?id=PdZBZcUbSewC&pg=PA207&dq=Koniecpolski+Ochmatow&lr=%20&as_brr=3]</ref><ref name="Podhorodecki"/>{{rp|396-405}}因为水上的冰层塌裂，很多土军在希那-沃达溺水身亡。[[奥赫马图夫战役_(1644年)|奥赫马图夫战役]]是17世纪初联邦对土耳其人取得的最大胜利，科涅茨波尔斯基因这次胜利也拥有了国际声望，他不仅预料到奥斯曼军队进攻的时间和地点，还在他们利用他们的常用战术——将他们的主要军队分为多种高机动性的单位（''czambuls''）——前便将他们彻底击溃。<ref name="Podhorodecki"/>{{rp|396-405}}

这场胜利让瓦迪斯瓦夫四世开始策划一次针对土耳其人的进攻。<ref name="Podhorodecki"/>{{rp|406}}科涅茨波尔斯基支持有限地进攻[[克里米亞汗國|克里米亚汗国]]，但反对对整个奥斯曼帝国发起进攻，他将这个计划看做是异想天开。<ref name=Bain/><ref name="Podhorodecki"/>{{rp|414-416}}他在一部标题为《关于摧毁克里米亚鞑靼人的论述》（"Dyskurs o Zniesieniu Tatarow Krymskich"）的计划书中，详尽地说明了他的战略计划。科涅茨波尔斯基也极力主张，为了这次军事行动，应当与莫斯科公国结盟。<ref name="Podhorodecki"/>{{rp|414-416}}<ref>Stanisław Przyłęcki, Zygmunt Stefan Koniecpolski, "Pamiętniki o Koniecpolskich: przyczynek do dziejów polskich XVII. wieku" (Memoirs about the Koniecpolskis), Nakł. L. Rzewuskiego, 1842, 第301页, [http://books.google.com/books?id=UrEEAAAAIAAJ&pg=RA2-PA301&dq=Koniecpolski+Zniesieniu+Tatarow+Krymskich&lr=%20&as_brr=3]</ref>国王瓦迪斯瓦夫继续要求向土耳其发动十字军，但国内少有人支持，除了欺骗哥萨克人称他们会因参与这次讨伐而获得权力和奖金外，这个提议什么成果也没有。<ref>Daniel Stone, "The Polish-Lithuanian state, 1386-1795", University of Washington Press, 2001年, 第158页, [http://books.google.com/books?id=LFgB_l4SdHAC&pg=PA158&dq=Koniecpolski+Tatars&lr=%20&as_brr=3]</ref>

科涅茨波尔斯基在1645年妻子克雷斯蒂娜去世后，<ref name="Podhorodecki"/>{{rp|410}}于1646年1月16日又娶了16岁的索菲亚·欧帕灵斯卡为妻，<ref name=Bogucka>Maria Bogucka, "Women in early modern Polish society, against the European background", Ashgate Publishing, Ltd., 2004年, 第6页, [http://books.google.com/books?id=9n_bfZthkvEC&pg=PA6&dq=Koniecpolski+married&as_brr=3]</ref>她是后来称为王室大元帅的[[乌卡什·欧帕灵斯基_(1612年-1666年)|乌卡什·欧帕灵斯基]]的女儿。<ref name="Podhorodecki"/>{{rp|418}}科涅茨波尔斯基在1646年3月于布罗德逝世。<ref name="Podhorodecki"/>{{rp|419-421}}很多文献指出，他的新婚是他的死因。[[约阿希姆·耶尔利奇|约阿希姆·耶尔利奇]]在他的日记中写道，他服用了过量的[[壮阳药|壮阳药]]。<ref name=Bogucka/><ref name="Podhorodecki"/>{{rp|419-421}}他的葬礼在1646年4月30日于布罗德举行。<ref name="Podhorodecki"/>{{rp|421-424}}

== 权贵 ==
[[File:Caspod1.gif|thumb]]，由威廉·博普朗修建，得到科涅茨波尔斯基的支持]]
科涅茨波尔斯基一生积攒大量财富。他的大部分领地都位于乌克兰，而他也成为乌克兰实际上的统治者；有些外国人称他是“乌克兰省督”，尽管联邦从未出现这个职位。<ref name="Podhorodecki"/>{{rp|337}}在事关乌克兰的大多数政治决策上，国王瓦迪斯瓦夫都能信任他。<ref name="Podhorodecki"/>{{rp|337}}<ref name="Podhorodecki"/>{{rp|337, 375}}在国王的了解和支持下，科涅茨波尔斯基向[[伊斯坦堡|伊斯坦布尔]]发派并迎接外交使节，展开磋商，签署协议；作为指挥官，他直接统帅联邦军队中的很大一部分。<ref name="Podhorodecki"/>{{rp|337, 374, 375}}他拥有自己的私人军队和从莫斯科延伸至奥斯曼帝国的情报网。<ref name="Podhorodecki"/>{{rp|375}}

科涅茨波尔斯基从他父亲那里继承约七八个村庄。<ref name="Podhorodecki"/>{{rp|339}}去世时，他拥有12个长老区（''starostwa''），<ref name="Podhorodecki"/>{{rp|345}}，超过300个居民区，其中包括几十个城镇，这些居民区使他年收入超过50万[[波兰兹罗提|兹罗提]]。<ref name="Podhorodecki"/>{{rp|351}}他的田产与农奴数目可观；只在[[布拉茨拉夫|布拉茨拉夫]]一地他就拥有18548户人为他劳作。<ref name="brit">{{cite web|title=Stanislaw Koniecpolski|url=http://www.britannica.com/EBchecked/topic/321768/Stanislaw-Koniecpolski|publisher=[[大英百科全书|Encyclopædia Britannica]]|accessdate=2009-12-16}}</ref><ref name="The Ukrainians: unexpected nation">{{cite book|title=The Ukrainians: unexpected nation|author=Andrew Wilson|year=2002|publisher=[[Yale_University_Press|Yale University Press]]|page=60|ISBN=9780300093094}}</ref>科涅茨波尔斯基将他的大笔财产投入到他的乌克兰领地的发展之中，并支持向人烟稀少的地方移民。<ref name=WIEM/><ref name="Podhorodecki"/>{{rp|353-359}}他建立了很多城镇与城市，并支持这些城镇和城市的发展。<ref name="Podhorodecki"/>{{rp|361}}他成立[[布罗德|布罗德]]镇，这个城镇因为他的投资变得繁荣，并成为当地重要的商业中心；<ref name="Podhorodecki"/>{{rp|362-364}}为加强城防，科涅茨波尔斯基在布罗德修筑一座[[城堡|城堡]]和若干[[棱堡|棱堡]]，此外，他还设立工坊生产[[波斯地毯|波斯]]风格的[[六股丝锦缎|六股丝锦缎]]织品和[[地毯|地毯]]。<ref name=WIEM/><ref name=Stone /><ref name="Podhorodecki"/>{{rp|362-367}}他还在[[皮德希尔齐|皮德希尔齐]]建立一座设有防御工事的宫殿，这座宫殿还有美丽的[[意大利花园|意大利花园]]。<ref name=WIEM/><ref name=Stone/>和大多数权贵一样，科涅茨波尔斯基也资助艺术家——画家、雕刻家和作家。<ref name="Podhorodecki"/>{{rp|371}}他成立了很多教堂，并努力将布罗德的学校升格为学院。<ref name="Podhorodecki"/>{{rp|371}}他主持修建[[华沙|华沙]]的[[华沙总统府|科涅茨波尔斯基宫]]（现为波兰总统官邸）。<ref name="Podhorodecki"/>{{rp|370-371}}他还主持修筑防御工事（巴尔，柯达克）。<ref name="Podhorodecki"/>{{rp|371}}

科涅茨波尔斯基被认为是一位彬彬有礼而具有教养的人，他参与了所有他能够参与的[[波兰共和国下议院|瑟姆]]，尽管因为[[口吃|口吃]]，他很少公开讲话。<ref name="Podhorodecki"/>{{rp|375, 389}}他在他的[[波兰贵族|贵族]]同僚中广受尊敬而极受欢迎。<ref name="Podhorodecki"/>{{rp|430}}

== 注释 ==
'''a.''' {{Note_label|a|a|none}}科涅茨波尔斯基的出生年份并不确定，在多部文献中都具有争议。[[莱谢克·波德霍罗德茨基|莱谢克·波德霍罗德茨基]]在他的传记《1592年-1646年的斯坦尼斯瓦夫·科涅茨波尔斯基》（''Stanisław Koniecpolski ok. 1592–1646''）中，将他的出生日期定在1592年，这也可能是其他文献中出生日期的“平均数”（在各类文献中，1590年最早，1594年最晚）。在他的笔记中，1591年最常见（譬如[[大英百科全书|大英百科全书]]中的[http://www.britannica.com/eb/article-9046002/斯坦尼斯瓦夫·科涅茨波尔斯基条目]{{Dead link|date=2018年7月 |bot=InternetArchiveBot |fix-attempted=no }}），而“出生日期在1593年和1594年”这一推测在历史学家[[瓦迪斯瓦夫·恰普灵斯基|瓦迪斯瓦夫·恰普灵斯基]]的《波兰传记词典》（Polski Słownik Biograficzny）中得到他的支持，恰普灵斯基称科涅茨波尔斯基为“那个时代伟大的专家”，但是恰普灵斯基未能为此列出证据。
