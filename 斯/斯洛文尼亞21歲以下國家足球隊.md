'''斯洛文尼亞21歲以下國家足球隊'''（'''Slovenia national under-21 football team'''，'''斯洛文尼亞U21'''）是[[斯洛文尼亞|斯洛文尼亞]]的21歲以下國家足球代表隊，由[[斯洛文尼亚足球協會|斯洛文尼亚足球協會]]組織，為前[[南斯拉夫社会主义联邦共和国|南斯拉夫]]於1992年解體後分裂出來的球隊之一，參加每兩年舉辦一次的[[歐洲U-21足球錦標賽|歐洲U-21足球錦標賽]]。

== 競賽歷史 ==
2006年[[歐洲U-21足球錦標賽|歐洲U-21足球錦標賽]]斯洛文尼亞U21在分組賽10戰4勝3和3負，最後一場主場3-0擊敗[[蘇格蘭21歲以下足球代表隊|蘇格蘭]]，鎖定次名席位<small><ref>{{cite web |language=en |url=http://www.uefa.com/competitions/under21/history/season=2006/round=1910/match=74364/index.html |title=Second place suits Slovenia |publisher=[[UEFA|''uefa.com'']] |date=2005-10-11 |accessdate=2009-09-23}}</ref></small>，緊隨衛冕冠軍[[義大利21歲以下國家足球隊|義大利]]出線附加賽。附加賽對手為[[荷蘭21歲以下國家足球隊|荷蘭]]，首回合主場憑門將[[萨米尔·汉达诺维奇|漢丹洛域]]（Samir Handanovič）出色的表現守和0-0<small><ref>{{cite web |language=en |url=http://www.uefa.com/competitions/under21/history/season=2006/round=2155/match=82024/index.html |title=Handanovic heroics deny Dutch |publisher=[[UEFA|''uefa.com'']] |date=2005-11-13 |accessdate=2009-09-23}}</ref></small>；次回合作客被[[克拉斯-扬·亨特拉尔|亨特拿]]連下兩城鍛羽而回<small><ref>{{cite web |language=en |url=http://www.uefa.com/competitions/under21/history/season=2006/round=2155/match=82025/index.html |title=Huntelaar double delights Dutch |publisher=[[UEFA|''uefa.com'']] |date=2005-11-16 |accessdate=2009-09-23}}</ref></small>。晉級決賽週的荷蘭其後更贏得冠軍錦標。
