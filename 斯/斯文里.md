'''斯文里'''位于[[中華人民共和國|中華人民共和國]][[上海市|上海市]][[静安区|静安区]][[新闸路|新闸路]]北面，由[[大田路|大田路]]分割为东西斯文里，是一片[[石库门#后期|后期老式石库门]][[里弄|里弄]]。门牌号为新闸路568弄、620弄，大田路464弄、492弄、546弄和南苏州路1463-1475号<ref name="东方网">{{Cite web |title =  搬迁人家已超半数 上海最大石库门“斯文里”面临拆迁 |url = http://sh.eastday.com/m/20130508/u1a7376054.html |publisher = 东方网 | date = 2013-05-08 | accessdate = 2014-01-23 }}</ref>。其曾是上海规模最大的旧式里弄<ref name="静安区">[http://www.shtong.gov.cn/newsite/node2/node4/node2249/node4412/node17436/node18417/node62251/userobject1ai6809.html 区县志 >> 区志 >> 静安区志 >> 第八编房屋建筑 >> 第三章旧有住宅 >> 第四节　旧式里弄住宅] {{webarchive|url=https://web.archive.org/web/20140918235515/http://www.shtong.gov.cn/newsite/node2/node4/node2249/node4412/node17436/node18417/node62251/userobject1ai6809.html |date=2014-09-18 }} 上海地方志办公室.</ref>，共占地4.66公顷，有706幢房屋。

但是西斯文里现已经被拆除，而东斯文里的[[拆迁|拆迁]]工作也已经在进行之中。

==历史==
20世纪初，斯文里所在的区域地处[[上海公共租界|上海公共租界]]西区，曾是一片荒芜的乱坟地，后来在此建造了广肇山庄<ref name="静安区"/>。1914年，一名为阿谷的[[英国|英国]]籍[[犹太人|犹太妇人]]购得此地，并将其开发为商业地产。西斯文里最早被兴建起来，当时全是砖木结构二层房屋。1918年，东斯文里启建。到1920年至1921年，斯文里建成，被命名为忻（新）康里。但忻康里原是一片乱坟地，故房子销路并不好。之后，阿谷家道衰落，里弄所有权先后转手于[[通和洋行|通和洋行]]、[[益丰洋行|益丰洋行]]，最后由斯文洋行得到<ref name="文化遗产">[http://www.jingan.gov.cn/newscenter/rdzt/fwzyc/lshg/201006/t20100621_71456.htm 静安区非物质文化遗产-斯文里] 上海静安.</ref>。斯文洋行遂将里弄名称更改为东、西斯文里<ref name="静安区"/>。

[[八一三事变|八一三事变]]后，大量来自[[闸北|闸北]]和[[虹口|虹口]]的难民南下，斯文里的房屋需求陡然上升，人口继而骤增。二房东以及二、三层阁楼也开始成为司空见惯的现象<ref name="静安区"/>。斯文里一下子就从原先的“上只角”变成了凌乱、嘈杂的里弄<ref name="东方网"/>。
===改造===
1992，斯文里开始着手拆迁以改建商业住宅。1997年，西斯文里被拆除。2012年6月，东斯文里拆迁工作开始，到2013年5月已有超过一半居民搬走<ref name="东方网"/>。在后续建设中，可能会考虑保留部分建筑。<ref>[http://www.shjagh.gov.cn/news.asp?id=1624 静安区规土局成功召开“苏州河滨河区城市设计方案征集”终期成果汇报及专家评审会] {{webarchive|url=https://web.archive.org/web/20150325182332/http://www.shjagh.gov.cn/news.asp?id=1624 |date=2015-03-25 }}</ref>2015年，东斯文里（属于[[静安区|静安区]]67街坊）连同[[东八块|东八块]]中的另一地块[[静安区|静安区]]59街坊被列入上海市城市更新首批试点项目。<ref>[http://news.xinhuanet.com/local/2015-02/13/c_127490692.htm 市区6大地标启动首批“城市更新” 市规土局：倡导城市更新完善城市功能、提升城市各方面品质].[[解放日报|解放日报]]</ref><ref>[http://news.163.com/15/0215/10/AIG5PNF700014SEH.html 看上海：上海推进城市更新，具体怎么做？].[[澎湃新闻网|澎湃新闻网]]</ref>

==建筑特色==
[[File:The_Door_of_a_Shikumen_Building_in_East_Siwenli_before_demolished.JPG|thumb]]
斯文里总占地4.66公顷，建筑面积达4.8万平方米，由706幢二三层砖木结构房屋组成。其中西斯文里有318幢，东斯文里有388幢。斯文里沿大田路段长约270米，沿新闸路段长约108米。房屋与其它石库门建筑一样采取西方[[排屋|联排式]]，为行列式。东斯文里按照地形共有13排房屋；西斯文里则有11排。斯文里符合后期老式石库门的特点，多为单开间。后天井为竖向布置，且面积缩小到3平方米。毗邻住宅的后天井采取两两相对的形式并降低了院墙，以改善通风和采光的条件。有些地方如晒台地坪、过梁已经采用了混凝土结构<ref name="静安区"/>。

石库门大门门楣上有精致的巴洛克风格的雕花<ref name="东方网"/>。室内房间布置上，底层为约23平方米的客堂和厨房，两楼有等面积的卧室以及亭子间及晒台。房屋室内并不设卫生设施。
