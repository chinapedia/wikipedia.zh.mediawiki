== 臧天朔事件 ==
 
歌手斯琴格日乐在接受某媒体采访时说出的一句“我与[[臧天朔|臧天朔]]不是朋友”，引起了外界对两人关系的众多猜测。5月30日晚10点，斯琴格日乐专程委托新浪网发表公开信，就最近有关她的一系列新闻作出解释。昨天，臧天朔听说此事后，表示自己对各种传闻根本无所谓，斯琴格日乐本人也不必太当真。前不久，斯琴格日乐在湖南做宣传时，提起自己和臧天朔的关系时她回答说：“我和他不是恋人，绝对不是；也不是朋友，绝对不是朋友。我那时候在他的乐队，说好听点，是做贝司手，其实就是打工。”

在前晚发出的公开信中，斯琴格日乐解释“我与臧天朔不是朋友”这句话既是事实也是误会，自己一直把对方当作值得尊敬的前辈和老师，所以才不敢以朋友自居。而称自己为“打工”的，则只是谦逊之辞。对斯琴格日乐的这封公开信，臧天朔昨晚在接受记者采访时表示根本没有必要，“其实这些事情我根本没有放在心上。”至于有传闻说斯琴格日乐此举只为炒作新专辑，他说：“只要她高兴就可以，我不想说太多。”但与此同时，臧天朔指出音乐应该靠实力说话。

昨晚，斯琴格日乐所属的正大国际负责人江女士在电话里说，她一直在外地对事情的前因后果并不清楚，不过对于臧天朔表现出的豁达态度，她表示非常感谢。“以我对斯琴格日乐这么多年的了解，她一直很重视自己和臧天朔之间的友谊，绝对不是那种忘恩负义的人。也许是这些天的新闻太多，她才出来解释吧，她就是那种直性子的人。她憋了两年才做成这张专辑，真的挺不容易的。况且都不是小孩子了，干吗要利用绯闻来炒作呢？作为公司，我们非常感谢臧天朔曾经对斯琴格日乐的栽培。”

== 流浪美国街头事件 ==

有消息传出[[郭峰|郭峰]]、[[张咪|张咪]]、[[成方圆|成方圆]]、斯琴格日乐、[[冯晓泉|冯晓泉]]、[[曾格格|曾格格]]等6名演员从美国洛杉矶给媒体发来联合声明，称他们被一个名叫刘楠(女)自称中央台导演的人以《艺苑风景线》节目的名义骗到美国参加8月12日《[[同一首歌|同一首歌]]——精彩中国为北京喝彩》奥运倒计时一周年大型庆典文艺演出活动。随后，刘楠又要求演员们飞往西雅图，参加18日的《精彩中国为北京喝彩》商业演出。

此时众演员发觉受骗均不愿意出演，于是被刘楠逐出所住酒店，流落美国街头。昨天，早报记者联系上《同一首歌》的相关负责人，该负责人表示《同一首歌》方面也是受害方。据6名演员在发来的声明中称，他们已经通过相关渠道证明刘楠并非央视导演，西雅图的演出也并非此次活动内容，纯属刘楠的欺诈行为。随后记者联系到《同一首歌》的出品单位中国国际电视总公司中视演艺中心的相关负责人，

该负责人表示，郭峰等人于本月12日参加的活动与他们无关。“我们本月11日的确在美国洛杉矶有个名叫《同一个世界、同一个梦想、同一声喝彩、2008北京》的活动，这是我们《同一首歌》和美国美亚艺术中心签约举办的活动，孟欣导演也亲自参加了录制全过程。这个活动在8月12日通过央视《晚间新闻》节目播出过相关报道。

而8月12日那场《同一首歌———精彩中国为北京喝彩》节目与我们毫无关系，演员也不是我们邀请的。我们抵达美国后，当地侨团主办方积极邀请我们前往录制该节目，我们考虑到当地华人和爱国侨胞的奥运热情，所以答应了录制，但是一再声明不能使用《同一首歌》的标志和名称，至于最后这台晚会能不能在央视播出，还要等领导审核之后才能做决定。现在，对方对外公布的活动名称中仍然带有《同一首歌》的字样，对我们节目造成了重大影响，也是对我们商标的侵权行为。我们正在联系该活动的主办单位，希望能得到合理的解释。”  

== 江苏台封杀事件 ==
江苏媒体报道，今年元旦过后，江苏电视台一栏目与斯琴格日乐联系参加节目录制，“斯琴格日乐本人愉快地接受了邀请”，并让该栏目和负责她媒体宣传的孔小姐 联络。经商谈，双方约定了出场费及２月１１日录制。之前，该栏目可派摄制组到京进行前期沟通和拍摄。1月8日，该栏目到京对斯琴格日乐进行采访、拍摄。2 月10日，孔要求推迟到3月18日。没几天，孔称由于斯琴格日乐不在宣传期，不能随便参加节目录制。如想录制，出场费必须涨为原来的６倍。该栏目表示这个 价格他们不能承受，于是放弃录制，并称要在江苏电视台封杀斯琴格日乐。

电话中，格日乐说：“我看你电话是北京的才敢接，我好好在家待着，却突然挨了一顿骂，全国各地记者都来问我，吓得我都不敢接电话了。我的所有日程、要参加什么节目、一切费用问题都是由我的经纪人来负责的，我根本不会去直接谈这些问题。我连他们这个栏目都不知道，就更不可能去把自己的出场费‘翻６倍’了。在这样的情况下，我突然被扣了一‘屎盆子’，我感到特别的莫名其妙。有问题他们可以去找我的公司协商，也可以用法律手段来解决，但这些他们都没做，而是直接说要让江苏电视台把我封杀掉。

我想问，一个栏目能代表江苏台吗，他们这样做很明显是在炒作自己。”记者随后拨通了格日乐经纪人尹青的电话。尹青说：“我想这件事中最倒霉的就是格日乐了。《女人百分百》只向我的助手提出过做节目的申请，根本没和我直接谈过此事。他们既没有给我打过电话，也没有任何的书面文字材料，价钱的事更不可能和我们商量。我们的日程早就安排到3月份了，根本不可能有时间去参加那个栏目的拍摄。”尹青还指出：“平常到电视台做节目是不谈钱的。我们会选择合适的栏目，选择合适的采访提纲为歌手做有针对性的宣传，双方一直合作很愉快，我们为什么要平白无故地得罪电视台呢？他们如果觉得我们的操作有问题可以直接找公司反映，但他们却选择这种很有冲突的方式，让人感觉很不地道，也严重地伤害了正大公司和艺人。《女人百分百》节目的做法杀气腾腾的，实在是不太理性，并且有自我炒作的痕迹。”
