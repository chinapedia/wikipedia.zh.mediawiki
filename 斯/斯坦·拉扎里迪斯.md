'''斯坦利·"斯坦"·拉扎里迪斯'''（'''Stanley "Stan" Lazaridis'''，{{bd|1972年|8月16日|catIdx=Lazaridis, Stan}}），[[希臘|希臘]]裔[[澳大利亞|澳大利亞]]人<ref>http://www.cambridge.org/aus/catalogue/catalogue.asp?isbn=0521547431&ss=ind</ref><ref>{{cite web |url=http://www.theworldgame.com.au/socceroos/index.php?pid=st&cid=71311 |title=存档副本 |accessdate=2009-01-09 |deadurl=yes |archiveurl=https://web.archive.org/web/20071011084029/http://www.theworldgame.com.au/socceroos/index.php?pid=st&cid=71311 |archivedate=2007-10-11 }}</ref><ref>{{cite web |url=http://www.theadvertiser.news.com.au/common/story_page/0,5936,19126599%255E12428,00.html |title=存档副本 |accessdate=2009年1月9日 |deadurl=yes |archiveurl=https://web.archive.org/web/20060525123344/http://www.theadvertiser.news.com.au/common/story_page/0,5936,19126599%255E12428,00.html |archivedate=2006年5月25日 |df= }}</ref>，已退役職業職業足球員，可擔任[[左後衛|左後衛]]或[[中場|左中場]]。澳洲球會[[珀斯光輝|珀斯光輝]]是拿沙迪斯職業生涯效力的最後一家球會，拿沙迪斯共代表[[澳洲國家足球隊|澳洲國家足球隊]]上陣58場。
