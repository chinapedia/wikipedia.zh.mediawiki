《'''斯大林格勒'''》（{{lang-ru|Сталинград}}）是一部[[费多尔·邦达尔丘克|费多尔·邦达尔丘克]]执导的[[俄罗斯|俄罗斯]][[战争片|战争片]]，是首部俄罗斯[[3D电影|3D电影]]，也是首部俄罗斯[[IMAX|IMAX]]作品<ref>[http://wearemoviegeeks.com/2012/02/russias-stalingrad-to-be-released-in-imax-3d-format-october-2013/ Russia’s STALINGRAD To Be Released In IMAX 3D Format October 2013]</ref><ref>[http://www.hollywoodreporter.com/news/ambitious-russian-3d-war-drama-227730 Ambitious Russian 3D War Drama 'Stalingrad' Begins Filming.]</ref>。2013年10月10日于俄罗斯上映<ref name="rt">{{cite web |url=http://rt.com/art-and-culture/news/stalingrad-movie-3d-shootings-836/ |title=Russia’s first big-budget 3D movie shootings over |date=2012-08-16|accessdate= 2012-08-16|work=rt.com |publisher=[[RT_(TV_network)|RT]]}}</ref>、10月31日于[[中国|中国]]上映<ref >{{cite web|url=http://news.mtime.com/2013/10/29/1519909.html | language= | title="斯大林格勒"登陆内地 细数苏联十大经典战争片 |publisher=Mtime时光网 |date=2013-10-31 |accessdate= }}</ref>。影片描写了1942年11月[[斯大林格勒战役|斯大林格勒会战]]中的一场城中[[巷战|巷战]]，以及巷战中的苏德双方的两段爱情。本片代表俄罗斯角逐[[第86届奥斯卡金像奖|第86届奥斯卡金像奖]]最佳外语片<ref name="Russia">{{cite web |url=http://en.ria.ru/art_living/20130921/183626284/Russia-Nominates-WWII-Movie-Stalingrad-for-Oscars.html |title=Russia Nominates WWII Movie ‘Stalingrad’ for Oscars |accessdate=2013-09-21 |work=RIA}}</ref><ref name="HWR">{{cite web |url=http://www.hollywoodreporter.com/news/oscars-russia-selects-fyodor-bondarchuks-633730 |title=Oscars: Russia Selects Fyodor Bondarchuk's 'Stalingrad' for Foreign Language Category |accessdate=2013-09-22 |work=Hollywood Reporter}}</ref>，最终没有获得提名。

==剧情==
故事采用倒叙手法。[[2011年日本大地震|2011年日本大地震]]，一名[[德国|德国]]女孩被压在[[水泥|水泥]]巨柱下颤抖而绝望，她不停哭喊：“我很想念我爸爸。”为了稳定她的情绪，一名来自[[俄罗斯|俄罗斯]]的派遣救援队员缓缓对她讲述了自己“五个老爹”的故事：

1942年秋天的[[史達林格勒戰役|史達林-{}-格勒戰役]]，苏联军队向盘踞在[[伏尔加河|伏尔加河]]左岸的德国军队反攻受阻，只有格罗莫夫[[大尉|大尉]]率领的几名侦察兵到了对岸，并受命固守在一栋3层楼的房子里，因此结识了房子的最后一个住户、19岁的少女卡嘉，他们之间发生了一系列人性冲突和温暖故事。另一方面，一名德国[[上尉|上尉]]卡恩率领手下攻下这座房子成为他必须完成的一个任务，他还与一位长相酷似其妻的[[俄国|俄国]]中年女子玛莎发生了感情。双方战士在这场枪林弹雨中体味着战争的残酷和人性的冲突。

==制作==
首次拍摄始于2011年秋季，用900名演员持续了17天的时间拍摄了两场主要的战争。主体拍摄发生在2012年5月至7月27日。大场面主要在[[圣彼得堡|圣彼得堡]]附近进行。总预算为3000万美元，耗时6个月完成。

导演费多尔·邦达尔丘克与德国演员[[托马斯·克莱舒曼|托马斯·克莱舒曼]]以前曾演过斯大林格勒战争电影，托马斯演的是1993年的[[德国电影|德国电影]]，费多尔演的是1989年的俄国作品。

[[3D|3D]]拍摄技术由美国公司[[3ality_Technica|3ALITY digital]]提供的机器完成，采用[[IMAX|IMAX]]3D、[[3D电影|3D]]和2D普遍版三种方式上映。

== 评价 ==
影片收获了大多正面评价，认为是一部反思战争和人性的战争佳片，如战争场面宏大震撼而残酷，战争中的温暖涓细却感人，沉重缓慢的音乐使得整部影片都沉浸在一种催人泪下的悲壮诗意中。<ref >{{cite web|url=http://ent.sina.com.cn/r/m/2013-10-30/17274033701.shtml | language= | title=《斯大林格勒》：绝望中透露出爱的美好 |publisher=新民晚报 |date=2013年10月30日 |accessdate= }}</ref>尽管本片没有呈现出整个战争的大格局，3000万美元能够呈现如此的效果、如此的品质已属难能可贵。<ref >{{cite web|url=http://ent.sina.com.cn/r/m/2013-11-04/13434036275.shtml | language= | title=《斯大林格勒》：兑了可乐的伏特加 |publisher=北京青年报 |date=2013年11月4日 |accessdate= }}</ref>本片“在视听效果、人物关系等方面为这类电影赋予了新意，最值得称道的是对人物心理的细腻刻画，台词的留白为观众留下了想象空间。缺点是片头片尾的地震救援感觉和故事没有必要联系，处理简单化了；另外，一些段落煽情成分不够，在不该克制的时候选择克制会让影片有所失分。”<ref >{{cite web|url=http://ent.sina.com.cn/r/m/2013-11-04/14244036307.shtml | language= | title=《斯大林格勒》：阵地爱情削弱战争仇 |publisher= 京华时报 |date=2013年11月4日 |accessdate= }}</ref>
