{{日本作品|放學後的昴星團|放課後のプレアデス}}是由[[日本|日本]][[速霸陸汽車|速霸陸汽車]]與[[GAINAX|GAINAX]]合作推出的原創[[動畫|動畫]]，作品結合了速霸陸的[[企業識別標誌|企業識別標誌]]「[[昴宿星團|昴宿星團]]」的六顆亮星，是日本首部汽車[[置入性行銷|置入性行銷]]的[[網路動畫|網路動畫]]。

從2011年2月1日起在[[Youtube|Youtube]]上播出，片長每回4至5分左右，4回構成（全23分），公開結束未定。2011年2月7日以『Wish Upon the Pleiades（Hokago no Pleiades）』的標題，正式公開英語字幕版。主角「すばる」即為速霸陸汽車的名字，而打開展望室的鑰匙是像汽車鑰匙一樣的設計，各角色的手杖前端部分是模仿速霸陸主要車款的前端燒烤設計和車燈的表現，手杖所發出的聲音則是汽車[[引擎|引擎]]聲。

2014年12月16日，官方网站公布了本作的电视动画将于2015年春开始播放的消息，並於2015年4月播出。

== 緣起 ==
2010年9月28日，「SUBARU x GAINAX Animation Project」官方網站開張。本作品是汽車業首見的原創網路動畫，預計於2011年冬季限定公開。由於作為品牌宣傳，動畫中幾乎不會有汽車登場<ref>{{Cite web|date=2010-11-30|url=http://diamond.jp/articles/-/10260|title=若者のクルマ離れ阻止なるか？ エヴァのガイナックスと組んだスバル“こだわり”の美少女アニメ|work=|author=桃田健史|publisher=DIAMONDonline|accessdate=2015-07-05|language=ja}}</ref>。

最初，富士重工業希望GAINAX能在動畫中宣傳[[速霸陸Legacy|速霸陸Legacy]]所採用的[[汽車防撞系統|汽車防撞系統]][[EyeSight|EyeSight]]，可是GAINAX認為用動畫表現防撞實在是非常困難，而在動畫中直接讓汽車登場甚至會動搖GAINAX的自尊心。在雙方數次討論後，富士重工業認為「故事裡沒有汽車登場也沒關係」「表現出速霸陸的優點、速霸陸追求創新與挑戰」，而GAINAX認為「這不僅是速霸陸的動畫，作為GAINAX的動畫我們也要主張自己的理念。儘管沒有汽車登場，要有某事物代替汽車來表現出速霸陸的優點」，因此動畫製作便朝「體現速霸陸存在本質」的方向前進<ref>{{Cite web|date=2010-12-08|url=http://diamond.jp/articles/-/10334|title=エヴァのガイナックスはなぜスバルと組んだ?美少女アニメ「放課後のプレアデス」誕生秘話（下）|work=|author=桃田健史|publisher=DIAMONDonline|accessdate=2015-07-05|language=ja}}</ref>。

另外，2011年1月31日於東京都內舉辦的記者發表會，GAINAX的[[武田康廣|武田康廣]]與富士重工業網路行銷負責人[[鈴木曜|鈴木曜]]進行的脫口秀中提到「企劃內容曾考慮『名為{{nihongo|齊藤愛|サイトウアイ|Saitō Ai}}的冒失娘、開著車全國旅行並接觸各式各樣的人』，不過隨著6月EyeSight開賣、此企劃便取消了」<ref>{{Cite web|date=2011-02-01|url=http://bizmakoto.jp/makoto/articles/1102/01/news017.html|title=スバル×ガイナックスの異色コラボ、できたアニメはクルマなし！|work=|author=岡田大助|publisher=ITmedia ビジネスONLiNE|accessdate=2015-07-05|language=ja}}</ref>。

於是，並非於動畫中插入電視廣告、亦非將速霸陸產品置入作品中，而是將以下幾項特點融入作品以達到宣傳目的：
* 動畫名稱的{{nihongo|昴星團|プレアデス|Pleiades}}與主角名{{nihongo|昴琉|すばる}}皆取自{{nihongo|速霸陸|スバル}}。
* 瞭望室的鑰匙設計成汽車鑰匙的樣子。
* 魔法杖設計成汽車[[傳動軸|傳動軸]]的樣子，在動畫中也直接稱作{{nihongo|傳動軸|ドライブシャフト}}，前端則設計成速霸陸各車種共通的前格柵與車頭燈樣式。魔法杖發出的聲音則是汽車引擎聲。
* 公車站名來自速霸陸各工廠（[[群馬縣|群馬縣]][[太田市|太田市]]、[[邑樂郡|邑樂郡]][[大泉町|大泉町]]的矢島工廠、太田北工廠、大泉工廠）<ref>Imageboard中（可見於官方網站）、角色所站的地方，是實際存在的同縣[[吾妻郡|吾妻郡]][[中之条町|中之条町]]的[[關越交通公司|關越交通公司]]客運的、「中之條站 - 伊參 - 赤坂 - 中之條站（中之條開始向右循環行駛）」{{cite web |url=http://www.nakanojo-kanko.jp/koutu/koutu.html |title=存档副本 |accessdate=2015-07-05 |deadurl=yes |archiveurl=https://web.archive.org/web/20100608065845/http://www.nakanojo-kanko.jp/koutu/koutu.html |archivedate=2010-06-08 }}{{cite web |url=http://www.kan-etsu.net/r-bus/timetable/nakanojyo-jyunkan-1.htm |title=存档副本 |accessdate=2015-07-05 |deadurl=yes |archiveurl=https://web.archive.org/web/20100117215807/http://www.kan-etsu.net/r-bus/timetable/nakanojyo-jyunkan-1.htm |archivedate=2010-01-17 }}的希望丘公車站。</ref>。

2011年1月31日深夜（2月1日凌晨）於YouTube公開播送，亦於[[澀谷站|澀谷站]]前的四面電視牆同步播放<ref>{{Twitter status|sbr_gx|32093478322176002|空が軽くなった今夜。渋谷駅前の４面をジャックして本編放映中!!|放課後のプレアデス|2011-01-31|2015-07-05|jpn}}</ref>。

2011年6月4日至同年7月1日，與[[United_Cinemas|United Cinemas]]合作推出期間限定的觀影禮儀宣導片（United Cinemas阿久比店除外），並推出原創紀念商品與飲料。

2013年3月21日本計畫再次啟動，並於同年12月27日宣布推出電視動畫<ref>{{Cite web|date=2013-12-27|url=http://anicobin.ldblog.jp/archives/35967850.html|title=SUBARU x GAINAX「放課後のプレアデス」次はTVアニメ化!?|work=|author=|publisher=あにこ便|accessdate=2015-07-05|language=ja}}</ref>。2014年10月漫畫化並開始在[[Comic_REX|Comic REX]]連載。另外，電視動畫原本計畫為劇場版動畫，由於[[華納娛樂日本公司|華納兄弟娛樂公司]]的要求而變更為電視動畫<ref>{{Twitter status|takedayasuhiro|610777689553698817|放課後のプレアデスは、本来劇場アニメとして企画されました。|武田康廣|2015-01-16|2015-07-05|jpn}}</ref>。

2015年4月，電視動畫開始播放，並於6月播放完畢，共12話。

== 劇情簡介 ==
「抬頭仰望天空，就能發現那顆閃耀的星星。」

放課後的學校，喜歡星星的少女昴琉用鑰匙打開了展望室，在有如異空間的地方，邂逅了神秘的少年湊。正在回家下樓梯的昴琉不小心踩到普勒阿得斯星人，被拿走了重要的指南針，在尋找時偶然看見了四個穿著如魔女一樣角色扮演的少女，昴琉在少女們中發現好友（葵）的身影…
