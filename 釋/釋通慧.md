'''釋通慧'''（1927年 - 2013年），俗姓唐，名振幫，[[中國|中國]][[四川|四川]][[遂寧市|遂寧市]]蓬菜鎮人，圓寂前指示弟子三年後當眾起缸。2016年6月12日吉时開缸。起缸後，老和尚肉身完整，表[[肉身菩薩|肉身菩薩]]，現在供奉於[[中國|中國]][[四川|四川]][[廣漢|廣漢]][[龍居寺|龍居寺]]的馬祖殿<ref name=input1>{{Cite web |url = http://dzh.mop.com/a/160613204459000295659.html |title = 四川广汉龙居寺通慧老和尚成就“肉身菩萨”圆满开缸 |date = 2016-06-13 |publisher = 猫扑网}}</ref><ref name=input2>{{Cite web |url = http://news.cang.com/infos/201606/469303.html |title = 中国佛教又出现了一尊“肉身菩萨” |date = 2016-06-13 |publisher = 华夏收藏网}}</ref>。

==生平==
通慧老和尚1922年3月5日出生，從小即隨父母茹素。1931年，在蓬溪[[金文寺|金文寺]]出家。1940年於[[寶光寺|寶光寺]]從妙能法師受具足戒。1941年跟隨[[能海上師|能海上師]]於[[成都|成都]][[淨慈寺|淨慈寺]]參學<ref>{{Cite book |url = http://www.fxwz.net/fxwz/fxwz197/wz197/17.htm|title = 能海上师轶事 |author = 郑建邦| date = 2015 |publisher = 《台州佛教》第67期}}</ref>。1942年四川廣漢[[龍居寺|龍居寺]]。文革期間[[龍居寺|龍居寺]]僧人被解散，廟產被沒收，寺廟毀壞，1986年[[廣漢|廣漢]][[龍居寺|龍居寺]]開放後，通慧法師擔任第一個住持。對整個寺廟全面整修，新建了羅漢堂，大雄寶殿，藏經樓，住宿樓，恢復重建山門，修復了地藏殿，天王殿，鐘鼓樓等。2013年2月26日晚上8點40分圓寂，之後封存缸中，示寂前指示弟子三年後當眾起缸。2016年6月12日開缸，肉身完整如好，表[[肉身菩薩|肉身菩薩]]<ref name=input1/><ref name=input2/>。
