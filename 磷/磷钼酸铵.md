'''磷钼酸铵'''（化学式：(NH<sub>4</sub>)<sub>3</sub>PMo<sub>12</sub>O<sub>40</sub>·xH<sub>2</sub>O）是[[杂多酸|杂多酸]][[磷钼酸|磷钼酸]]的[[铵盐|铵盐]]。

==结构==
十二钼磷杂多酸的阴离子（[PMo<sub>12</sub>O<sub>40</sub>]<sup>3−</sup>）是一种 1:12A 型杂多酸阴离子，1826年首先由[[永斯·贝采利乌斯|贝采利乌斯]]发现。这类 1:12A 型的结构首先由 Keggin 测得，因此常称为 [[Keggin结构|Keggin结构]]。其基本结构单元是四组三个钼氧八面体，每一组中三个钼氧八面体共用顶角上的氧原子，结合成 <math>\rm \ Mo_3O_{10}</math> 单元。四个 <math>\rm \ Mo_3O_{10}</math> 单元的四个三方共用的氧原子位于中心四面体的棱角上，构成四面体穴，而磷原子位于四面体穴的中心。

:[[File:Phosphotungstate-3D-polyhedra.png|center]]阴离子的结构（Keggin结构）]]

==性质==
有光泽黄色[[单斜晶系|单斜晶系]]柱状结晶。从不同方法制得的磷钼酸铵所含结晶水的数目和组成不同，颜色也有所不同。在水中的[[溶解度|溶解度]]较小，并易被还原为蓝色的[[磷钼蓝|磷钼蓝]]。能溶于[[碱|碱]]、[[氨|氨]]、[[磷酸|磷酸]]。溶于[[乙酸|乙酸]]、[[过氧化氢|过氧化氢]]、[[联苯胺|联苯胺]]，变为蓝色。在[[硫酸|硫酸]]真空保干器中失去结晶水，得到无色的无水磷钼酸铵晶体。<ref name="chemyq">{{cite web|url=http://www.chemyq.com/xz/xz3/22070solyl.htm|title=磷钼酸铵|publisher=化工引擎|accessdate=2009-08-24}}</ref>

磷钼酸铵和[[硝酸钾|硝酸钾]]反应，钾离子可以取代一分子或两分子的铵根。<ref>周漱尘. 硝酸洗涤剂对磷钼酸铵沉淀组成的影响[J]. 湖南师范大学自然科学学报, 1987(4):59-61.</ref>

==制取==
可由{{le|正钼酸铵|Ammonium orthomolybdate}}与含[[硝酸|硝酸]]的[[磷酸|磷酸]]溶液加热反应而得。
:<math>\rm \ H_3PO_4 +12(NH_4)_2MoO_4+21HNO_3 \rightarrow (NH_4)_3PO_4\cdot 12MoO_3 \downarrow +21NH_4NO_3+12H_2O</math>

==用途==
用于在分析化学中检测[[磷酸盐|磷酸盐]]。

步骤：<ref>{{cite book|pages=367|title=分析化学手册 第二分册：化学分析|author=杭州大学化学系分析化学教研室编|edition=第2版|publisher=化学工业出版社|location=北京|year=1997|isbn=7-5025-1971-8}}</ref> 取试液（＞0.05mg [[磷|磷]]），加[[硝酸铵|硝酸铵]]（5％～10％），硝酸（5％～10％），40～45℃加热，加沉淀剂正钼酸铵（20倍过量），放置30分钟，玻璃滤器过滤，5％硝酸铵洗涤，

*500～550℃灼烧，称量形式：<math>\rm \ HPO_3\cdot 12MoO_3</math>

*105℃干燥，称量形式：<math>\rm \ (NH_4)_3PO_4\cdot 12MoO_3</math>（0.01637），括号内为[[经验换算因数|经验换算因数]]

*180℃干燥，称量形式：<math>\rm \ (NH_4)_3PO_4\cdot 12MoO_3</math>（0.01654）

*同上操作，取得沉淀，溶解于氨水（10％氨），用盐酸酸化，再按[[镁氧剂|镁氧剂]]方法测定。称量形式：<math>\rm \ Mg_2P_2O_7</math>

干扰离子：As，F，Se，Si，Te，Ti，V，W，Zr，H<sub>2</sub>SO<sub>4</sub>，HCl，OAc
