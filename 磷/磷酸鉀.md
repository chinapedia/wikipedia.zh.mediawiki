'''磷酸鉀'''是[[鉀|鉀]]的[[磷酸鹽|磷酸鹽]]，[[化学式|化学式]]为K<sub>3</sub>PO<sub>4</sub>，常见无水物和一水合物。<ref name=CB>[https://www.chemicalbook.com/ProductChemicalPropertiesCB7206120.htm 磷酸钾]. ChemicalBoook. [2018-11-21]</ref>

==制备==
磷酸钾可[[由磷酸铵|由磷酸铵]]（(NH<sub>4</sub>)<sub>3</sub>PO<sub>4</sub>）和[[氯化钾|氯化钾]]（KCl）的[[复分解反应|复分解反应]]制得，[[溶解度|溶解度]]较小的一者将[[沉淀|沉淀]]：<ref>{{Citation|title=Cyclic process for producing tripotassium phosphate and ammonium chloride|date=1968-10-15|url=https://patents.google.com/patent/US3591359A/en|accessdate=2018-04-27}}</ref>
:<chem>(NH4)3PO4 + 3KCl -> K3PO4 + 3NH4Cl</chem>

氢氧化钾和[[磷酸|磷酸]]或[[磷酸二氢钾|磷酸二氢钾]]的反应也用于制备磷酸钾：<ref name=CB />
: <chem> 3KOH + H3PO4 -> K3PO4 + 3H2O</chem>
: <chem> 2KOH + KH2PO4 -> K3PO4 + 2H2O</chem>
: <chem> KOH + K2HPO4 -> K3PO4 + H2O</chem>

==用途==
磷酸钾可用作肥料<ref>施学鹏. 磷酸钾和低氟磷酸的工业性生产[J]. 化肥工业, 1979(4):73-76.</ref>、催化剂<ref>刘亚丽, 刘德娥, 陈战国. 磷酸钾催化的α-氰基肉桂酸乙酯与二溴海因区域选择性氨溴加成反应[C]// 中国化学会学术年会. 2014.</ref>等。
