{{expand|time=2015-11-03T10:01:00+00:00}}'''淘寶售賣假貨事件'''，或'''淘宝大战工商总局'''<ref>{{cite web |url = http://finance.people.com.cn/n/2015/0130/c1004-26475714.html |title=淘宝大战工商总局余烬：三大悬疑待解 |publisher=人民网 |date=2015-01-30 |accessdate=2015-01-30 }}</ref>、'''淘宝叫板事件'''，是[[中国大陆|中国大陆]]购物网站[[淘宝网|淘宝网]]与[[中华人民共和国国家工商行政管理总局|中华人民共和国国家工商行政管理总局]]两者之间发生的事件。事件起源于2015年1月23日，国家工商行政管理总局对网上所销售的商品公布了抽查结果，其中淘宝网的正品率仅为37.25%。同月27日，淘宝网官方微博发布《一个80后淘宝网运营小二心声——刘红亮司长：您违规了，别吹黑哨！》一文，回应了工商总局的监测结果，称“抽检程序违规”，但是国家工商总局认为“抽检合法合规”<ref name="中国网">{{cite web |url = http://news.china.com.cn/2015-01/28/content_34670184.htm |title=工商总局回应淘宝网质疑：抽检合法合规 |publisher=中国网 |date=2015-01-28  |accessdate=2015-01-29 }}</ref>。此事件在中国大陆的網路上引起争议<ref>{{cite web |url = http://finance.ifeng.com/a/20150129/13466237_0.shtml |title=淘宝舌战工商总局：争议抽样方法？ |publisher=凤凰网 | date=2015-01-29 |accessdate=2015-01-30 }}</ref>。

==背景==
淘宝网是中国一家[[网络购物|网络购物]]网站，于2003年5月10日成立，由[[阿里巴巴集团|阿里巴巴集团]]投资1亿人民币创办。截止2010年12月31日淘宝网注册会员超过3.7亿人，占中国网购市场80%的份额。阿里巴巴集团董事局主席[[马云|马云]]曾于2014年11月20日在[[世界互联网大会|世界互联网大会]]发表演讲，称“说淘宝假货多的人，基本没在上面买过东西”<ref>{{cite web |url = http://finance.takungpao.com/tech/q/2014/1120/2833082.html|title=马云：说淘宝假货多的人基本没在上面买过东西 |publisher=大公网 |date=2014-11-20 |accessdate=2015-01-30 }}</ref>。但近年来中国大陆电商平台的假货问题令人担忧。2014年7月30日，[[腾讯网|腾讯网]]透露，[[聚美优品|聚美优品]]、[[京东商城|京东商城]]、[[1号店|1号店]]、[[国美在线|国美在线]]、[[亚马逊|亚马逊]]等电商平台，存在贩卖假货的行为，供应商河北祥鹏恒业通过多个电商平台销售假冒廉价奢侈品<ref>{{cite web |url = http://news.ifeng.com/a/20140730/41360236_0.shtml|title=奢侈品假货门重创电商新贵聚美优品、京东等紧急下架闭店|publisher=凤凰网 |date=2014-07-30 |accessdate=2015-01-30}}</ref>。

另外，淘宝网还开设有“消费者保障计划”，以保证消费者在购买产品（宝贝）遇到问题时能够享受到售后服务保障，但这其中并未涉及到保证所售商品为真货这一规定<ref>{{cite web |url = http://news.sina.com.cn/o/2015-01-30/143931463053.shtml |title=监管“走进去” 电商“敞开来”|publisher=新浪 | date=2015-01-30 |accessdate=2015-01-30}}</ref>。

==事件经过==
===工商总局抽查===
2014年8月到10月，中国[[国家工商行政管理总局|国家工商行政管理总局]]对网上所销售的商品进行了抽查，并于2015年1月23日向民众公布了抽查结果，其中淘宝网的正品率仅为37.25%，为此次抽查的最低值。从抽查结果来看，[[手机|手机]]、[[玩具|玩具]]、[[服装|服装]]、[[化妆品|化妆品]]、[[化肥|化肥]]，均存在假货问题<ref>{{cite web|url=http://business.sohu.com/20150127/n408112998.shtml |title=工商总局抽检购物网站 淘宝网正品率37.25% |publisher=搜狐 | date=2015-01-27  |accessdate=2015-01-29 }}</ref>。

===调查后的回应===
2015年1月27日，淘宝网官方微博发布《一个80后淘宝网运营小二心声——[[刘红亮|刘红亮]]司长：您违规了，别吹黑哨！》一文，回应了工商总局的监测结果，称“抽检程序违规”。1月27日晚，国家工商总局新闻发言人表示，称“抽检合法合规”<ref name="中国网"/>。

1月28日，国家工商总局公开了2014年《关于对阿里巴巴集团进行行政指导工作情况的[[白皮书|白皮书]]》，指出阿里系网络交易平台存在的五大突出问题，包括主体准入把关不严、对商品信息审查不力、销售行为管理混乱、信用评价存有缺陷、内部工作人员管控不严。白皮书中提到，淘宝网、[[阿里巴巴|阿里巴巴]]平台网店销售的商品或提供的服务存在大量侵犯他人注册商标专用权、商品质量不合格、无进口来源证明、国家明令禁止销售、传销等类型的商品信息，假[[烟|烟]]、假[[酒|酒]]、高仿手机、假名牌包、[[假证|假证]]、封建[[迷信|迷信]]与[[赌博|赌博]]用品、危害公共安全的物品、[[管制刀具|管制刀具]]、[[窃听器材|窃听器材]]等大量存在<ref>{{cite web|url=http://tech.sina.com.cn/i/2015-01-28/doc-iawzunex9466886.shtml |title=工商总局首披对阿里指导白皮书 指淘宝5大问题 |publisher=新浪 | date=2015年1月28日   |accessdate=2015-01-30 }}</ref>。随后淘宝官方微博发表官方声明，认为刘红亮司长在监管过程中存在程序失当、情绪[[执法|执法]]的行为，用错误的方式得到一个不客观的结论，对淘宝以及中国电子商务从业者造成了非常严重的负面影响，并决定向国家工商总局正式投诉<ref>{{cite web|url=http://news.sina.com.cn/o/2015-01-29/033931455518.shtml |title=淘宝掐架工商总局 |publisher=新浪 | date=2015-01-29 |accessdate=2015-01-29 }}</ref>。同日晚，国家工商总局官网将白皮书撤下<ref>{{cite web|url=http://tech.sina.com.cn/i/2015-01-29/02399986406.shtml |title=工商总局官网撤下白皮书 未对淘宝投诉做回应 |publisher=新浪 | date=2015-01-29 |accessdate=2015-01-30 }}</ref>。

1月29日，[[中华人民共和国商务部|中华人民共和国商务部]]表示，要求将“继续把互联网领域专项整治推向深入，健全电子商务领域法律法规”<ref>{{cite web|url=http://tech.sina.com.cn/i/2015-01-29/doc-iawzunex9522155.shtml|title=商务部表态淘宝工商总局之争:加大电商整治力度 |publisher=新浪 | date=2015-01-29 |accessdate=2015-01-30 }}</ref>。同日中午，国家工商总局官网删除了白皮书的内容，并表示其白皮书无任何法律效力<ref>{{cite web |url=http://news.163.com/15/0131/04/AH8S1E1200014Q4P.html|title=工商总局：对淘宝白皮书无法律效力 |publisher=网易 |date=2015-01-31 |accessdate=2015-01-31 }}</ref>。

1月30日，国家工商总局局长[[张茅|张茅]]会见阿里巴巴董事局主席[[马云|马云]]，马云表示阿里巴巴将配合政府打假，加强日常线上巡查和抽检<ref>{{cite web|url=http://news.mydrivers.com/1/380/380371.htm |title=工商总局与淘宝"和解" 马云称将配合政府打假 |publisher=驱动之家 | date=2015-01-31 |accessdate=2015-01-31 }}</ref>。而在[[美国东岸时间|美国东岸时间]]1月30日，美国7家[[律师事务所|律师事务所]]宣布调查阿里巴巴及其旗下平台售假事件<ref>{{cite web |url=http://tech.huanqiu.com/news/2015-01/5560927.html |title=又有两美律所宣布调查阿里巴巴 总数增至7家 |publisher=环球网 | date=2015-01-31|accessdate=2015-01-31 }}</ref>，阿里巴巴可能会面临[[起诉|起诉]]的风险<ref>{{cite web |url=http://money.163.com/15/0130/10/AH708NL200254TFQ.html |title=中国律师征集阿里巴巴投资者 欲提集体诉讼 |publisher=网易 | date=2015-01-30 |accessdate=2015-01-31 }}</ref>。

===「香港蘇文大學」===
{{main|香港蘇文大學事件}}
2月2日，香港多家傳媒揭發，淘寶網上有一間「香港蘇文大學」出售畢業證書，香港教育局證實沒有這間學校註冊，事件曝光後，淘寶網將有關產品下架，並且刊登致歉信<ref>{{cite web|url=http://www.msn.com/zh-hk/news/national/%e5%81%87%e8%87%b3%e5%bc%b7-%e5%a4%a7%e9%99%b8%e6%9c%89%e3%80%8c%e9%a6%99%e6%b8%af%e8%98%87%e6%96%87%e5%a4%a7%e5%ad%b8%e3%80%8d%e7%b6%b2%e5%94%ae%e8%ad%89%e6%9b%b8/ar-AA8Uc2H?ocid=mailsignoutmd |title=假至強 大陸有「香港蘇文大學」網侷假證書|publisher=都市日報 | date=2015年2月3日   |accessdate=2015-02-03 }}</ref>。

==影响==
受此事件影响，[[美国东岸时间|美国东岸时间]]1月28日，阿里巴巴集团股价下跌4.36%，收盘报98.45[[美元|美元]]，市值蒸发110亿美元<ref>{{cite web|url=http://money.163.com/api/15/0129/07/AH43LQOV00251LK6.html |title=淘宝大战工商总局 阿里市值一夜蒸发687亿|publisher=网易 | date=2015-01-29 |accessdate=2015-01-29 }}</ref>；由于阿里巴巴股价大跌，马云资产一夜之间缩水14亿美元，致使[[万达集团|万达集团]]董事长[[王健林|王健林]]重登中国首富宝座<ref>{{cite web| url=http://finance.cankaoxiaoxi.com/2015/0130/645933.shtml| title=彭博:阿里股价暴跌 王健林超马云重登中国首富| publisher=参考消息网| accessdate=2015-01-30| deadurl=yes| archiveurl=https://web.archive.org/web/20150402164943/http://finance.cankaoxiaoxi.com/2015/0130/645933.shtml| archivedate=2015-04-02}}</ref>。

==相关评论==
各方网友的观点不一，有网友表示“各打五十大板”；“是要杀杀阿里的威风了！一个经济萧条的年代加上一个不懂经济的领导，造就了马云！土财主膨胀了，工商局税务局人员被告也是活该，谁让你们过去对土财主的违法行为睁一眼闭一眼的。”也有网友表示支持工商总局：“上市成首富就任性了，工商总局提出的问题淘宝确实存在，认真整改就是了，意图引领舆论导向攻击执法部门就显得太过狂妄了”<ref>{{cite web |url = http://www.bbc.co.uk/zhongwen/simp/china/2015/01/150128_china_taobao_dispute |title=淘寶叫板工商總局引發網上大辯論 |publisher=[[BBC|BBC中文網]] |date=2015-01-28 |accessdate=2015-01-29}}</ref><ref>{{cite web|url=http://www.bj.xinhuanet.com/bjyw/2015-01/29/c_1114179873.htm|title=网民吐槽“淘宝小二”：消费者贪便宜和你卖假货没关系|publisher=新华网|date=2015-01-29|accessdate=2015-01-30|deadurl=yes|archiveurl=https://web.archive.org/web/20150402153204/http://www.bj.xinhuanet.com/bjyw/2015-01/29/c_1114179873.htm|archivedate=2015-04-02}}</ref>。
