'''朴時妍'''（{{lang-ko|박시연}}，{{bd|1979年|3月29日|catIdx=P朴}}），[[韓國|韓國]][[女演員|女演員]]，2000年韓國小姐第四名。<ref>{{Cite web |url=http://women.sohu.com/20130429/n374158950_21.shtml |title=2000年韩国小姐朴诗妍 |access-date=2013-11-02 |archive-url=https://web.archive.org/web/20131103153304/http://women.sohu.com/20130429/n374158950_21.shtml |archive-date=2013-11-03 |dead-url=yes }}</ref>

== 個人生活 ==
2005年5月和演員歌手[[文晸赫|文晸赫]]公開戀人關係，於2007年3月分手。並在分手後上[[演藝家中介|演藝家中介]]表示，交往期間如同他的影子般，因此被抹煞作為演員的努力與才華，感到委屈。

2011年11月19日，與年長她四歲的貿易公司職員朴某在首爾某酒店舉行非公開的結婚典禮，2013年9月24日產女，2015年11月再度產女，但於2016年5月宣佈離婚，為兩個女兒復出演藝圈。

== 染毒遭起訴 ==
2013年3月13日，朴詩妍涉嫌違法施打麻醉藥品「普洛福」異丙酚（Propofol）遭首爾檢察官起訴，朴詩妍在22個月內共注射了185次。被起诉时，朴诗妍已怀孕7个月，她最终结果为被判处8个月徒刑，2年缓刑，同时缴纳罚款370万。
<ref>[http://www.nownews.com/n/2013/03/14/310805 驚！韓四女星疑染毒遭起訴　非法注射433次]{{zh}}</ref>
