'''朴韓星'''（{{lang-ko|박한별}}，{{lang-en|Park Han-Byul}}，{{bd|1984年|11月17日|catIdx=Park}}），[[韓國|韓國]][[女演員|女演員]]。由於韓語「別」字的[[音讀|音讀]]音與表示「星」的意思的[[訓讀|訓讀]]固有詞同音（同樣讀作「{{lang|ko|별}}」），故此有時候會翻譯為'''朴寒星'''或'''朴寒別'''（為[[大中華地區|大中華地區]]的慣用稱呼）。2013年經官方公佈將中文名定為'''朴韓星'''，因為「寒」字的意思是寒冷，與她的個性不符合。

== 個人簡介 ==
自高中時起與歌手[[崔東昱|Se7en]]交往。<ref>{{Cite web | url= http://ent.sina.com.cn/m/c/k/2013-04-18/21233903546.shtml | title=  《筆仙Ⅱ》首發劇照 朴韓星更名開微博 | date=2013年4月18日 | publisher= 新浪 | language= zh-tw }}</ref><ref>{{Cite web | url= http://ent.appledaily.com.tw/enews/article/entertainment/20120403/34133115/ | title= 朴寒星毛多吸金主 笑納523萬代言費| date= 2012年4月3日 | publisher= 蘋果日報 | language= zh-tw }}</ref><ref>{{Cite web | url= http://www.libertytimes.com.tw/2009/new/jun/11/today-show10.htm | title= Se7en認了朴寒星 怒告駭客洩戀情 | date= 2009年6月11日 | publisher= 自由時報 | language= zh-tw }}</ref>2014年經雙方經紀公司證實，此段交往12年的愛情長跑正式告終。

2002年以雜誌Ceci封面模特出道。

2014年12月24日被報導與演員[[鄭恩宇|鄭恩宇]]交往中<ref>{{Cite web | url= http://news.tf.co.kr/read/entertain/1465509.htm | title= 단독] 박한별, 세븐과 결별 확인…새 '남친'은 정은우!(종합)| date= 2014年12月24日 | publisher= THE FACT | language= ko }}</ref>，但交往7個月後因壓力過大導致分手<ref>{{Cite web | url= http://www.koreastardaily.com/tc/news/64613 | title= 朴寒星鄭恩宇承認分手：日程繁忙 公開戀愛壓力大| accessdate=2018-05-02| date= 2015年7月7日 | publisher= Korea Star Daily| language= zh-tw }}</ref>。

2017年11月24日公布懷有4個月身孕並與同齡、在金融圈服務的企業家丈夫<ref group=註>YuriHoldings投資公司代表劉仁碩。</ref>於10月時完成結婚註冊<ref>{{Cite web | url= https://www.nownews.com/news/20180430/2745368 | title= 揮別舊愛！韓女歌手情定企業家男友　今傳已剖腹生子 | publisher= nownews| accessdate=2018-05-02|date=2018-04-30| language= zh-tw }}</ref><ref>{{Cite web | url=https://tw.entertainment.appledaily.com/realtime/20180430/1344577/ | title= 朴寒星奉子閃嫁金融男　喜傳剖腹產男寶寶| publisher= 蘋果日報| accessdate=2018-05-02|date=2018-04-30| language= zh-tw }}</ref>。

2018年4月30日剖腹產下長子<ref group=註>韓國新聞媒體刊登其名字為유아준。</ref><ref>{{Cite web | url= https://n.yam.com/Article/20180430826992 | title= 朴寒星升級當媽媽 30日喜得一子publisher= Yam News| accessdate=2018-05-02|date=2018-04-30| language= zh-tw }}</ref><ref>{{Cite web | url= https://star.ettoday.net/news/1160559| title= Se7en前女友朴韓星平安卸貨！產下2.67kg男嬰宣布當媽了| publisher= ettoday| accessdate=2018-05-02|date=2018-04-30| language= zh-tw }}</ref>。

2019年3月，朴韓星受[[勝利夜店事件|勝利夜店事件]]影響，因證實曾與劉仁碩、尹總警、[[崔鍾訓|崔鍾訓]]一同球聚，而讓原先支持的愛好者們感到失望，紛紛向連續劇《[[悲傷時相愛|悲傷時相愛]]》的製作單位提出刪除她所飾演的角色內容<ref>{{cite news|url=https://tw.appledaily.com/new/realtime/20190319/1535977/|title=朴寒星被拱出陪同總警打球　崔鍾訓續玩IG不知悔改|publisher=蘋果日報|date=2019-03-19|author=陳薇安|accessdate=2019-03-19|language=zh-hant}}</ref>。
