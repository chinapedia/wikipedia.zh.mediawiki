'''朴敏慶'''（{{lang-ko|박민경}}，{{lang-en|Grace Park}}，英文譯音：'''葛瑞絲·朴''' ，{{bd|1974年|3月14日|catIdx=Park}}<ref>[http://en.battlestarwiki.org/wiki/Sources:Grace_Park 資料來源：朴敏庆 於 BattlestarWiki] {{webarchive|url=https://web.archive.org/web/20160919225240/http://en.battlestarwiki.org/wiki/Sources%3AGrace_Park |date=2016-09-19 }}</ref><ref>[http://www.tv.com/grace-park-blog/the-confusion-over-grace-parks-real-birthday/topic/83722-645911/show_blog_entry.html&topic_id=645911| The confusion over Grace Park's real birthday]</ref>）是一名[[美國|美國]]及[[加拿大|加拿大]][[韓裔|韓裔]]演員，以她於電視影集《[[2005星際大爭霸|太空堡垒卡拉狄加]]》中扮演的角色夏倫·法洛利（Sharon "Boomer" Valerii），與《[[檀島警騎2.0|檀島警騎2.0]]》中所扮演的科諾·卡拉卡瓦（Kono Kalakaua）警官等角色最為人熟識。

她亦曾於[[電子遊戲|遊戲]]《[[終極動員令3：泰伯倫戰爭|命令与征服3：泰伯利亚之战]]》亮相。<ref>{{cite news |first = Mike |last = Bantick |title = EA delivers Command & Conquer 3 Tiberium Wars on PC |url = http://www.itwire.com.au/content/view/10830/1092/ylu=X3oDMTA0cDJlYmhvBHNlYwM- |publisher = iTWire |date = 2007-03-27 |accessdate = 2007-04-04 |archive-url = https://web.archive.org/web/20070927024154/http://www.itwire.com.au/content/view/10830/1092/ylu=X3oDMTA0cDJlYmhvBHNlYwM- |archive-date = 2007-09-27 |dead-url = yes }}</ref><ref>{{cite web |url = http://www.xbox.com/en-US/games/c/commandandconquer3xbox360/default.htm |title = Command & Conquer 3 Tiberium Wars |accessdate = 2007-04-04}}</ref>
