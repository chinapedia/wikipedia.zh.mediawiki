《'''如意君传'''》（又名：《'''阃娱情传'''》），[[明朝|明朝]]署名[[吴门|吴门]][[徐昌龄|徐昌龄]]的[[性文学|性文学]][[长篇小说|长篇小说]]。<ref name="multiple12 " >{{cite web |url= http://www.stnn.cc/sex/200702/t20070201_452636.html|title= 明代性小说：《如意君传》描写武则天的性爱|accessdate=2007-01-29 |work=|publisher=星岛环球网 |language=zh-hans}}</ref>

== 内容 ==
全书叙述了[[女皇帝|女皇帝]][[武则天|武则天]]一生的[[性行為|性]]和[[爱情|爱情]]关系。全书描述了早期的[[武则天|武则天]]与[[张昌宗|张昌宗]]、[[张易之|张易之]]、[[薛怀义|薛怀义]]等多角[[性关系|性关系]]，而主要着墨则在[[武则天|武则天]]晚年的情事，[[武则天|武则天]]的情夫[[薛敖曹|薛敖曹]]利用自身有利条件，护佑李唐宗室及[[唐中宗|唐中宗]]，武则天临终之际，为避免薛敖曹失去保护伞，将其送出宫，薛敖曹遁入玄门，得道升仙。<ref name="multiple12 " /><ref name="multiple65" >{{cite web |url=http://www.dahe.cn/pdjx/dushu/t20071130_1215525.htm |title=《如意君传》：性、宫廷与政治 |accessdate=2007年11月30日 |work=|publisher=大河网 |language=zh-hans}}</ref>

== 思想 ==
《如意君传》将[[性愛|性]]、[[政治|政治]]、[[国事|国事]]联系起来，写得深刻，比后世单独描写[[性行為|性]]和[[爱|爱]]的[[性文学|性文学]]作品更加具备思想性。<ref name="multiple65" />全书描写[[薛敖曹|薛敖曹]]将其道德评价和历史评价区分开来，开创了使用[[唯物辨证法|唯物辨证法]]一分为二看待问题的先河。<ref name="multiple12 " />

== 写法 ==
《如意君传》描写[[性交|性交]]场面主要有隐喻和直露、写意和写实、[[性交技巧|性交技巧]]和[[性交|性交]][[心理|心理]]等多方面结合，场景刻画非常真实。<ref name="multiple65" />

== 影响 ==
《如意君传》对后世[[性文学|性文学]]小说影响甚巨。《[[金瓶梅|金瓶梅]]》第十八、十九、二十七、二十八、二十九、五十、五十一、五十二、六十一、七十三、七十八、七十九回中的性描写，都是仿照《如意君传》写的，甚至整段抄袭《如意君传》。<ref name="multiple12 " /><ref name="multiple65" /><br />

《如意君传》中有名句：“幽禽尚知相偶之乐，可以人而不如鸟乎？”，[[明朝|明朝]][[汤显祖|汤显祖]]《[[牡丹亭|牡丹亭]]》仿写了：“睢鸠尚有洲渚之兴，可以人而不如鸟乎”。<ref name="multiple12 " />

== 评价 ==

[[黄训|黄训]]《读如意君传》：“唐之昏风甚哉！”，“兄不兄，子不子，父不父，可以为人乎？况可以为君乎？”<ref name="multiple65" /><br />

[[柳伯生|柳伯生]]《如意君传跋》：“叙事委悉，错言奇叙，比诸诸传，快活相倍”。<ref name="multiple65" />
