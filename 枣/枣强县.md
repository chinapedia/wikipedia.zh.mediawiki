'''枣强县'''地处华北平原腹地，位于[[河北省|河北省]]东南部，隶属[[衡水|衡水]]市。

==地理==
枣强县位于河北省东南部，[[环渤海经济圈|环渤海经济圈]]。东面分别与[[景县|景县]]，[[故城县|故城县]]接壤，南面与[[南宫|南宫]]市毗连，西面与[[冀州|冀州]]市接壤，北面分别连接[[桃城区|桃城区]]，[[武邑县|武邑县]]。县城枣强镇北距首都北京272公里，西距省会石家庄124公里。总面积为892.3平方公里。[[清凉江|清凉江]]流经本县东部。

==气候==
年均气温13℃，年降水量497毫米。属暖温带大陆性季风气候。

==历史==
===东周至清朝===
本县在[[春秋|春秋]]时代属齐地，[[战国|战国]]时属赵地。
[[秦朝|秦朝]]时属巨鹿郡。
[[西汉|西汉]]初置枣强县。两汉时，枣强与[[广川|广川]]两县，几度“离合废置，本为一也”<ref>{{Cite web |url=http://www.zaoqiang.gov.cn/zqfm/lsyg/index.shtml |title=枣强历史沿革 |access-date=2013-10-26 |archive-url=https://web.archive.org/web/20131029211826/http://www.zaoqiang.gov.cn/zqfm/lsyg/index.shtml |archive-date=2013-10-29 |dead-url=yes }}</ref>。因此两汉时的广川县基本上就是今天的枣强县。
[[三国|三国]]时[[曹魏|曹魏]]复置枣强县，隶属冀州清河郡。
[[西晋|西晋]]时枣强县并入广川县，隶属冀州渤海郡。
[[东晋|东晋]]时后燕复置枣强县，隶属广川郡。
[[北魏|北魏]]神瑞二年（公元415年），枣强被并入广川县。北魏 太和二十二年（公元498年），复置枣强县。
[[北齐|北齐]]天保七年（公元556）年，广川县被并入枣强县，隶属长乐郡。
[[隋朝|隋朝]]开皇二年（公元582年），枣强县治所复为今天的旧县村。开皇三年（公元583年），废长乐郡为冀州，枣强仍属之。
[[唐朝|唐朝]]贞观元年（公元627年），朝廷在州以上设道，枣强县属河北道冀州。
[[北宋|北宋]]时枣强县隶属河北东路冀州。
[[金朝|金朝]]天会四年（公元1126年），枣强县旧县城毁于洪水，县治由旧县村处迁于刘马村，即今天县城址。隶属河北东路河间府冀州。
[[元朝|元朝]]时，枣强县属燕南河北道肃政廉访司真定路冀州。元历三年（公元1330年），真定路改属中书省，枣强仍属之。
[[明朝|明朝]]时枣强县属北直隶（京师）真定府冀州。
[[清朝|清朝]]时布政使司改为省，北直隶改为[[河北省|直隶省]]，冀州为直隶州，枣强县属冀州直隶州。

===民国时期===
民国初年时沿袭清制，枣强县隶属直隶省冀直隶州。直隶省废除府、州后，枣强县直接隶属于直隶省。民国三年(1914年)，直隶省划分为四道，本县隶属大名道。民国十七年(1928年)，直隶省改名为[[河北省|河北省]]，废道，各县直接隶属于省。民国二十六年(1937年)，河北省划为17个行政督察区，本县隶属第14督察区<ref>{{Cite web |url=http://www.zaoqiang.gov.cn/zqfm/lsyg/index.shtml |title=枣强历史沿革 |access-date=2013-10-26 |archive-url=https://web.archive.org/web/20131029211826/http://www.zaoqiang.gov.cn/zqfm/lsyg/index.shtml |archive-date=2013-10-29 |dead-url=yes }}</ref>。

===中华人民共和国成立后至今===
河北省人民政府成立后，本县政府隶属河北省[[衡水|衡水]]专区。1952年改属[[石家庄|石家庄]]专区。1962年6月，衡水专区复置后，枣强县仍然隶属衡水专区。1970年衡水专区改为衡水地区，本县仍属之。1996年5月31日，衡水地区改为地级衡水市，本县仍属之<ref>{{Cite web |url=http://www.zaoqiang.gov.cn/zqfm/lsyg/index.shtml |title=枣强历史沿革 |access-date=2013-10-26 |archive-url=https://web.archive.org/web/20131029211826/http://www.zaoqiang.gov.cn/zqfm/lsyg/index.shtml |archive-date=2013-10-29 |dead-url=yes }}</ref>。

==地域经济==

===皮毛产业===
枣强县[[大营镇_(枣强县)|大营镇]]是裘皮业的发源地，经过多年的发展，已经成为国内外著名的裘皮加工、生产、集散基地。如今，裘皮业不仅是枣强县经济的支柱，而且成为全市乃至河北省重要的区域特色产业之一。 裘皮业相传为[[商朝|商朝]]宰相[[比干|比干]]在枣强大营为官时发明，明[[嘉靖|嘉靖]]年间，营皮被额定为“土贡”，名扬天下。清朝[[道光|道光]]年间，英、俄、日、荷等10多个国家的客商开始在大营设立货栈；到上世纪六七十年代，营皮曾长期作为出口免检产品远销海外。目前全县拥有加工企业摊点1.2万家，毛皮业以大营为中心，已辐射到周边5市县500多个村庄，从业人员达到15万人，形成了方圆百里的经济圈。

==地方特产==
===马莲小枣===
可生食，还可做成许多食品，如枣糕、枣卷、棕子等，另外，还加工制做枣酒、枣醋、枣酱、枣泥、枣面、蜜枣、乌枣等多种加工品。
马莲小枣树体稍大、长势偏旺，适应性强、耐旱、耐瘠薄、耐盐碱、抗涝。结果较早，稳产高产。但由于过去战乱和人为因素，现栽植面积很少

===红富士苹果===

===东方甘薯===
东方甘薯是河北省枣强县具有地方特色的名优农产品，栽培品种为脱毒北京553，该品种薯块整齐、均匀，薯形较长，表面光滑，块根皮色为黄色，维生素及胡萝卜素含量多，含糖量可达30%，鲜食熟制均可，其产品深爱广大消费者青睐。除食用外，还是工业上制造酒精和淀粉的重要原料，也可制造乳酸、味精、柠檬酸、淀粉等，还可以加工成各种各样的粉条、粉丝等，其用途非常广泛。该产品多次被评为河北省名优农产品称号。
