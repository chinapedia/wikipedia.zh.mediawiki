'''利根川'''（{{jpn|j=利根川|hg=とねがわ|rm=Tonegawa|f=y}}），长约322公里，流域面积16,840平方公里，是[[日本|日本]]国内第二长，流域面积最大的[[河流|河流]]，长度仅次于[[信浓川|信浓川]]。別名「坂東太郎」，坂東是[[关东|关东]]的旧名，而太郎则是长子的意思。亦被称为日本三大河流之一，也是[[東京|東京]]與[[首都圈_(日本)|首都圈]]的重要水源。

== 历史 ==
{{double image|right|Tone riverine system 16century.png|200|Tone riverine system 20century.png|200|16世纪的利根川。|20世纪的利根川。}} 

这条河曾一度以它不受控制的洪水而出名，每当洪水爆发的时候它都会改变河道。因此很难描绘这条河的古河道。

利根川最初的[[入海口|入海口]]位于[[东京湾|东京湾]]，它現有的[[渡良瀨川|渡良瀨川]]、[[鬼怒川|鬼怒川]]等下游支流原本都是独立的[[水系|水系]]，並沒有與利根川匯流。为了河流运输以及控制洪水，[[天正|天正]]18年（1590年）[[德川氏|德川家]]入主[[江戶|江戶]]後，從[[文祿|文祿]]3年（1594年）到[[承應|承應]]3年（1654年）間進行「利根川東遷事業」，開闢人工河道，將原向南與[[荒川_(關東)|荒川]]匯流後注入東京灣的中下游河道，改為向東與渡良瀨川和鬼怒川匯流，直接注入太平洋。利根川經過這次的大改造後，中下游的土地得以開發，也使得[[德川幕府|德川幕府]]時代作為日本發展重心的[[關東地方|關東地區]]免於水災之患。之後又進行數次河道的整治工作。

现时河道大致上确定于[[明治时期|明治时期]]，其广阔的分水岭大部分是人造的。

== 地理 ==
[[File:Tone_River_2015.jpg|thumb]]
[[File:KurihashiSuiiKansokujo2005-6.jpg|thumb]]
利根川的主流发源於在[[群马县|群马县]]和[[新潟县|新潟县]]县境的[[大水上山|大水上山]]。在千叶县[[铫子市|铫子市]]和茨城县[[神栖市|神栖市]]之间注入[[太平洋|太平洋]]。主流全长约322公里，流域面积达16,840平方公里。流域为群马县、[[栃木县|栃木县]]、[[长野县|长野县]]、[[埼玉县|埼玉县]]、[[东京都|东京都]]、[[茨城县|茨城县]]和[[千叶县|千叶县]]的一都六县，占[[关东平原|关东平原]]的大部分。
