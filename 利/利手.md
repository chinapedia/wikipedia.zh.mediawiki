'''利手'''、'''撇子'''是人类习惯所使用的[[手|手]]，某些人更习惯使用右手，称为'''[[右利手|右利手]]'''，某些人习惯使用左手，称为'''[[左利手|左利手]]'''，而能靈活的運用雙手的稱為'''[[兩手同利|兩手同利]]'''或'''雙撇子'''。

== 人數 ==
根据2013年美國《醫藥日報》的统计，[[左撇子|左撇子]]目前占全球人口的10%<ref>[http://world.huanqiu.com/exclusive/2013-08/4243596.html 世界左撇子日關注“少數派” 左撇子佔人口10%] 环球网国际新闻 2013-08-15</ref>。

邁克爾·巴斯利曾经對這個問題曾作過深入的研究。在他所著的《左撇子》<ref>Left-Handed People，Michael Barsley，英语</ref>一書中，他説：“在這10－30%的比率中，雖然該有個較準確的數字。不過，沒有一項統計數字是全球一致公認為可接受的。我們也許可以説，在一個教育普及的開明民主社會裏，左撇子的數目約佔人口14－15%左右。”

格利亞在《國際百科全書》<ref>Encyclopedia International，Grolier，英语</ref>指出，全球約有16%的人口是擅長使用左手的。即使僅是15%或16%，這便意味到全球約3億人是左撇子。左右共利者所占的比例更是少之又少。

== 成因 ==
對於[[左撇子|左撇子]]的成因，研究人員還沒得出一致的結論，左右利手的決定機轉目前還是眾說紛紜，但是這跟[[基因|基因]]和[[生态环境|環境]]的共同作用有關，重要因素可能是[[遺傳|遺傳]]<ref name="大腦的奧秘">{{cite book|author1=中村克樹|editor=高智賢翻譯|title=彩色版徹底圖解：大腦的奧秘|date=2014年6月|publisher=楓樹林出版事業有限公司|isbn=978-986-5688-00-4|pages=p.58-59|edition=初版}}</ref>。{{fact|左撇子與右撇子在基因上沒有特別大的區別，主要差別只在於兩者的左右腦能力|time=2017-02-20}}。

[[英國|英國]][[倫敦大學|倫敦大學]]的[[心理學|心理學]]和[[醫學|醫學]]教育教授-克里斯‧麥克馬納斯在接受[[美國|美國]][[科學雜誌|科學雜誌]]《科學美國人》採訪時指出：「慣用左手的人不比慣用右手的人智商高，事實上，左撇子在患誦讀困難或者言語障礙上的比例要稍微高一些，這是由於大腦結構的不同使然。左撇子使用[[右腦|右腦]]更多，而右撇子使用[[左腦|左腦]]更多。」

左撇子母親生下左撇子小孩的機率，比右撇子母親高兩倍，但若雙親都是右撇子，而且雙方家族也都是右撇子的情況下也可能誕下左撇子小孩<ref name="大腦的奧秘"/>。科學考究發現，即使雙親皆為左撇子，誕下左撇子小孩的機率 也只有一半，遺傳訊息完全相同的[[雙胞胎|同卵雙胞胎]]和遺傳訊息只有50%相同的異卵雙胞胎的左撇子發生率幾乎相同<ref name="大腦的奧秘"/>。同卵雙胞胎亦可能有不同的慣用手。有左利手的父母有較高的機會生出左利手小孩，即使最高機率僅約為1/5。根據上述結果，光用遺傳法則並無法解釋慣用手的決定機轉。

除了遺傳說，一些科學家提出更多的假說來解釋慣用手的形成，如天生、胎兒期的姿勢、受產時狀況、受產後的日常生活影響、社會環境和教育的影響等等的說法都有<ref name="大腦的奧秘"/>。

==左右利手的腦部==
關於[[腦部|腦部]]構造，70%的左撇子腦部構造是和右撇子相同的<ref name="大腦的奧秘"/>。科學家對左撇子和右撇子的[[腦部|腦部]]研究結果發現其功能運作模式略有差異，97－99%右撇子的語言中樞位於大腦左半球；而左撇子則分三種類型。一是和右撇子一樣，語言中樞位於左半球（約佔70%）；二是語言中樞位於右半球（約佔15%）；三是左右兩半球都有語言中樞（約佔15%）<ref name="大腦的奧秘"/>。

有研究人員指出左撇子可進行更複雜的[[推理|推理]]，其大腦能增加他們的[[創意|創意]]力，以不一般的方式來處理[[言語|言語]]、[[空間|空間]]和[[情感|情感]]。研究人員指這也是左撇子在[[諾貝爾獎|諾貝爾獎]]得獎者、[[作家|作家]]、[[建築師|建築師]]、[[藝術家|藝術家]]、[[數學家|數學家]]、[[音樂家|音樂家]]中的比例較高的原因之一。[[達文西|達文西]]、[[牛頓|牛頓]]、[[拿破崙|拿破崙]]和[[愛因斯坦|愛因斯坦]]等歷史名人都是左撇子。

另一方面，[[生存|生存]]的[[生态环境|環境]]使得左撇子需尋求較佳的[[解決|解決]][[方案|方案]]，這使得他們承擔更大的心理[[壓力|壓力]]。

19世紀[[法國|法國]][[神經外科|神經外科]]醫生[[保羅·布洛卡|保羅·布洛卡]]曾倡導一項理論，認為就腦部機能運用方面而言，左撇子其實是慣用右手的人的鏡子對稱影像而已。

其他人把這項理論進一步引申，認為此項理論亦能應用在身體結構方面。因此他們以為左撇子的心臟理應位於他身體的右邊。[[解剖學|解剖學]]很快便推翻這種推論。然而，大腦的模式或組織與人慣運用左手還是右手的確有密切的關係。[[神經外科|神經外科]]醫生現正就有關課題進行更廣泛研究。

人腦的每一邊均稱為[[大腦|大腦]]半球。自19世紀以來，人們明白到各大腦半球專責不同任務。若干研究認為，大部分人的左大腦半球支配[[語言|語言]]、[[數字|數字]]、[[邏輯|邏輯]]以及其他分析性功能。至於右大腦半球看來卻偏重[[藝術|藝術]]等活動。不過，慣用左手的人在處理訊息和使用兩個大腦半球方面則與一般人不盡相同。

在一對一的格鬥或某些運動項目（如[[棒球|棒球]]、[[擊劍|擊劍]]等）上，由於左利手數量較少，慣用右手的一般人不易習慣，因此在競技時有一定的優勢。但在團體性的運動如[[足球|足球]]，左利手並沒有特別突出。

動物界也有慣用手的分別，但其左、右撇子比例近乎相同，只有人類以右撇子佔大多數，這是自然界中的特例<ref name="大腦的奧秘"/>。
