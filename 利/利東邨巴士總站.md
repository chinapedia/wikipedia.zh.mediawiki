'''利東邨巴士總站'''（{{lang-en|'''Lei Tung Estate Bus Terminus'''}}）是位於[[香港|香港]][[鴨脷洲|鴨脷洲]][[玉桂山|玉桂山]]利東商場對面的露天巴士總站。

==總站路線資料==
===[[新巴93A線|新巴93A線]]===
*{{HK Bus Route HKIsland Show:93A|format=inline}}
於1991年9月2日開始服務，只在上課日早上繁忙時間服務，1998年給予新巴接手。

===[[新巴94線|新巴94線]]===
*{{HK Bus Route HKIsland Show:94|format=inline}}
1989年11月16日開辦，現由[[新巴|新巴]]營運，來往[[利東邨|利東邨]]及[[中環碼頭|中環碼頭]]，取道[[薄扶林道|薄扶林道]]，只在平日晨早繁忙時間提供服務。以東興樓的總站上落客。每當此線提供服務時，[[新巴91線|91線]]會暫停繞經利東邨。

===[[城巴96線|城巴96線]]===
*{{HK Bus Route HKIsland Show:96|format=inline}}
提供利東邨及黃竹坑來往銅鑼灣的巴士服務，於1987年2月18日投入服務，最初由中巴經營，1993年9月1日改由城巴接辦，並增派空調巴士行走，是第一批由中巴手上接辦的26條城巴路線之一。隨著空調巴士大量引入及二手Atlantean退役，本線逐漸改派全空調巴士行走。往利東方向，先繞經東興樓及利東街市兩站後，才折返總站。

===[[城巴97線|城巴97線]]===
*{{HK Bus Route HKIsland Show:97|format=inline}}
提供鴨脷洲利東邨及黃竹坑來往灣仔、金鐘及中環的巴士服務，原本計劃於1987年2月9日投入服務。

===[[城巴98線|城巴98線]]===
*{{HK Bus Route HKIsland Show:98|format=inline}}
提供利東邨、漁安苑至香港仔的穿梭巴士服務，原本計劃於1987年2月9日投入服務。

===[[過海隧道巴士171A線|過海隧道巴士171A線]]===
*{{HK Bus Route Cross Harbour Show:171A|format=inline}}
於2011年9月19日投入服務，由[[九巴|九巴]]及[[城巴|城巴]]聯合營運，取代[[過海隧道巴士171線|171線]]上午繁忙時間由海怡半島開出的班次，由東興樓開出，只於平日上午繁忙時間提供服務。

==途經路線資料（此途經路線的車站以「東興樓」站為名）==
===[[新巴91線|新巴91線]]===
*{{HK Bus Route HKIsland Show:91|format=inline}}
該路線提供鴨脷洲邨及香港仔來往薄扶林道、上環及中環渡輪碼頭的巴士服務，於1982年11月16日起投入服務。每當[[新巴94線|94線]]在服務時間結束後，此線會繞經利東邨。

===[[新巴94A線|新巴94A線]]===
*{{HK Bus Route HKIsland Show:94A|format=inline}}

===[[城巴95P線|城巴95P線]]===
*{{HK Bus Route HKIsland Show:95P|format=inline}}
為95C號線的特別班次，提供田灣及香港仔開往利東、海怡半島及鴨利洲邨的巴士服務，於2002年5月6日起投入服務，於上課日下午提供服務。

===[[城巴97A線|城巴97A線]]===
*{{HK Bus Route HKIsland Show:97A|format=inline}}
於1989年7月3日投入服務，途經[[利東邨|利東邨]]、[[黃竹坑|黃竹坑]]及[[深灣|深灣]]，只於平日繁忙時間提供服務，為穿梭[[南區_(香港)|南區]]區內的短途循環路線。

===[[城巴99線|城巴99線]]===
*{{HK Bus Route HKIsland Show:99|format=inline}}
於1994年11月28日投入服務，初因中巴搶[[城巴92線|城巴92線]]的客源，由海怡半島來往[[北角碼頭|北角碼頭]]。1995年城巴接手後，改由海怡半島至[[西灣河碼頭|西灣河碼頭]]，1997年4月14日改為海怡半島至筲箕灣，並繞經耀東邨。

===[[城巴99X線|城巴99X線]]（特別班次）===
*{{HK Bus Route HKIsland Show:99X|format=special}}
於2014年1月13日投入服務，途經利東邨，不停香港仔隧道收費廣場分站、不經永興街及炮台山一段英皇道，以西灣河（太康街）作終站；並於2015年8月17日增加一班車，只於平日上午繁忙時間提供服務。

===[[過海隧道巴士171線|過海隧道巴士171線]]===
*{{HK Bus Route Cross Harbour Show:171|format=inline}}
於1994年6月6日投入服務，由[[九巴|九巴]]及[[城巴|城巴]]聯合營運，途經[[美孚新邨|美孚新邨]]、[[長沙灣|長沙灣]]、[[深水埗|深水埗]]、[[旺角|旺角]]、[[油麻地|油麻地]]、[[佐敦|佐敦]]、[[海底隧道_(香港)|海底隧道]]、[[香港仔隧道|香港仔隧道]]、[[黃竹坑|黃竹坑]]、[[利東邨|利東邨]]及[[鴨脷洲邨|鴨脷洲邨]]。每當[[過海隧道巴士171線|171A線]]提供服務時，此線往荔枝角方向會暫停服務。

===[[過海隧道巴士671線|過海隧道巴士671線]]===
*{{HK Bus Route Cross Harbour Show:671|format=inline}}
1997年8月29日投入服務，以配合-{[[荷里活廣場|荷里活廣場]]}-開幕，可是[[鑽石山站公共運輸交匯處|鑽石山地鐵站總站]]（現稱鑽石山鐵路站巴士總站）尚未啟用，所以以[[鳳德道|鳳德道]][[龍蟠苑|龍蟠苑]]停車場對出作臨時的巴士總站；於同年11月2日鑽石山地鐵站總站啟用後遷往該總站。

===[[城巴A10線|城巴A10線]]===
*{{HK Bus Route Lantau Show:A10|format=inline}}
於2006年3月26日投入服務，途經[[海怡半島|海怡半島]]、[[利東邨|利東邨]]、[[香港仔|香港仔]]、[[田灣|田灣]]、[[華富邨|華富邨]]、[[置富花園|置富花園]]、[[薄扶林|薄扶林]]、[[摩星嶺|摩星嶺]]、[[堅尼地城|堅尼地城]]、[[石塘咀|石塘咀]]、[[西區海底隧道|西區海底隧道]]、[[西九龍公路|西九龍公路]]、[[昂船洲大橋|昂船洲大橋]]、[[青嶼幹線|青嶼幹線]]、[[香港國際機場#一號客運大樓|一號客運大樓]]及[[香港國際機場#二號客運大樓|二號客運大樓]]。

===[[城巴N90線|城巴N90線]]===
*{{HK Bus Route HKIsland Show:N90|format=inline}}
提供來往灣仔、金鐘及中環的通宵巴士服務。

===[[過海隧道巴士N171線|過海隧道巴士N171線]]===
*{{HK Bus Route Cross Harbour Show:N171|format=inline}}
提供鴨脷洲邨、利東及黃竹坑來往九龍區的通宵巴士服務，於1997年4月1日起投入服務。

===[[香港島專線小巴37線|香港島專線小巴37線]]===
*{{HK MiniBus Route HK Show:37|format=inline}}
此條路線由鴨脷洲專線小巴承辦。

===[[香港島專線小巴39C線|香港島專線小巴39C線]]、[[香港島專線小巴39S線|39S線]]===
*[[香港仔中心|香港仔中心]] ↺ '''[[利東邨|利東邨]]'''／[[漁安苑|漁安苑]]（總站）（39C線）
*[[香港仔中心|香港仔中心]] ↺ [[深灣軒|深灣軒]]（總站）／'''[[利東邨|利東邨]]'''（39S線）
此兩條路線由[[進智公交|進智公交]]承辦，39C線是先經漁安苑然後往香港仔中心，而39S線則不經漁安苑直接往香港仔中心。

===[[香港島專線小巴39M線|香港島專線小巴39M線]]===
*{{HK MiniBus Route HK Show:39M|format=inline}}
此路線由進智公交承辦，早上10點前不會服務利東邨，早上10點後則以循環線服務，漁安苑成為中途站。
