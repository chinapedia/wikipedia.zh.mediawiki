'''利家圳'''為一座位於[[臺灣|臺灣]][[臺東縣|臺東縣]][[卑南鄉|卑南鄉]]的小型[[灌溉|灌溉]][[水圳|水圳]]系統。

== 沿革 ==
利家圳最早再1896年由當地[[利嘉村|利嘉村]]與[[大南村|大南村]][[原住民|原住民]]村民所自行開鑿，[[臺灣戰後時期|二戰後]]由當時[[台灣|台灣]][[省政府|省政府]]接收，目前為[[臺東縣|臺東]][[農田水利會|農田水利會]][[知本|知本]][[工作站|工作站]]進行管理與維護。<ref name="台東農田水利會 專輯報導 鹿野大圳灌溉用水有效調配營運成果">{{cite web |url=http://www.ttia.gov.tw/page6.php |title=台東農田水利會 主要圳系建造年代表 |accessdate=2015-01-06 |deadurl=yes |archiveurl=https://web.archive.org/web/20141208020113/http://www.ttia.gov.tw/page6.php |archivedate=2014-12-08 }}</ref>

== 設施 ==
=== 進水口 ===
利家圳源分有兩座進水口，第一座建於利嘉溪支流大南溪溪口右岸處，將大南溪溪水引流至沉砂池沉澱後，再利用跨河擋水臨時土堤將利嘉溪主流與大南溪溪水匯集至設置在利嘉溪右岸的第二進水口，爾後臺東農田水利會在利嘉溪河床上興建一座倒洪吸工連結第一進水口的沉砂池與第二進水口，讓大南溪溪水不再經由利嘉溪河床表面引流至第二進水口。

=== 幹線 ===
利家圳幹線總長5.1公里，共分有15條支線，總灌溉面積186公頃。<ref name="利家圳幹線歷年改善工程">{{cite web |url=http://www.ttia.gov.tw/page6-4-9.php |title=台東農田水利會 利家圳幹線歷年改善工程 | accessdate=2015-01-06}}</ref>

== 改善工程 ==

* '''圳路翻新工程'''<ref name="利家圳幹線歷年改善工程" />

因利家圳早期進水口及圳路設施老舊，圳路多為乾砌塊石，時易崩塌，影響灌溉甚鉅。臺東農田水利會有鑑於此，即擬定計畫分年分期予以改善，自1993年起至1997年止，分年編列預算進行圳路改善工程。<ref name="利家圳幹線歷年改善工程" />

* '''鳳凰颱風災害修工程'''
利家圳進水口因[[鳳凰颱風|鳳凰颱風]]來襲導致進水口邊坡遭洪水沖毀，進水口制水門構造物毀損，故臺東農田水利會於2014年12月12日公開進行工程招標，總工程預計耗資新臺幣8,893,000元<ref>{{cite web |url=http://gov.playgroup.com.tw/gov_notice/20141211212923352271#.VKs_LCuUfxw |title=利家圳進水口鳳凰颱風災害修復工程 |accessdate=2014-12-12 }}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

* '''[[蘇迪勒颱風|蘇迪勒颱風]]災後修復工程'''
臺東農田水利會於2015年12月展開利家圳蘇迪勒颱風災後修復工程，工程包括改建進水口、河岸護坡、增建一座排砂閘門以及下河的工作便道一條<ref>{{cite web |url=http://www.ttia.gov.tw/page1.php?ACT=2150&PK=466 |title=利家圳斷水公告 | accessdate=2016-01-02}}</ref>。
