[[数学|数学]]上，'''利萨茹（{{lang|fr|Lissajous}}）曲线'''（又称'''利萨茹图形'''、'''李萨如图形'''或'''鲍迪奇（{{lang|en|Bowditch}}）曲线'''）是两个沿着互相垂直方向的正弦振动的合成的轨迹。

[[纳撒尼尔·鲍迪奇|纳撒尼尔·鲍迪奇]]在1815年首先研究这一族[[曲线|曲线]]，[[朱尔·利萨茹|朱尔·利萨茹]]在1857年作更详细研究。

== 数学定义 ==

利萨茹曲线由以下[[参数方程|参数方程]]定义：
: <math>\begin{cases}
x( \theta)=a\sin(\theta)\\
y(\theta)=b\sin(n \theta + \phi)
\end{cases}</math>
其中<math> 0\le \phi \le \frac {\pi}{2} </math>，<math>n\ge 1\,</math>。

<math>n</math>称为曲线的参数，是两个正弦振动的频率比。若比例为有理数，则<math>n=\frac{q}{p}\,</math>，参数方程可以写作：
: <math>\begin{cases}
x( \theta)=a\sin(p\theta)\\
y(\theta)=b\sin(q \theta + \phi) \\
0\le \theta \le 2\pi
\end{cases}</math>，

其中<math> 0\le \phi \le \frac {\pi}{2p}</math>。

== 性质 ==
*若<math>n</math>为无理数，曲线在长方形<math>[-a,a]\times[-b,b]</math>中[[稠密|稠密]]。
*若<math>n</math>为有理数， 
**曲线是<math>2q</math>次[[代数曲线|代数曲线]]若<math>\phi \in \left(0,\frac{\pi}{2p} \right]</math>对奇数<math>p</math>，或<math>\phi \in \left[0,\frac{\pi}{2p} \right)</math>对偶数<math>p</math>。
**曲线是<math>q</math>次[[代数曲线|代数曲线]]的一部份若<math>\phi=0\,</math>对奇数<math>p</math>，或<math>\phi=\frac{\pi}{2p}</math>对偶数<math>p</math>。

*若<math>n</math>为偶数而<math> \phi = \frac{\pi}{2} </math>，或若<math>n</math>为奇数而<math> \phi =0\,</math>，则曲线是第<math>n</math>个[[切比雪夫多项式|切比雪夫多项式]]<math>T_n</math>的曲线的一部份。

== 特别情况 ==
*若<math>a=b</math>，<math>n=1</math>，则曲线是[[椭圆|椭圆]]。
**若<math>\phi=\frac{\pi}{2}</math>，则这椭圆其实是[[圆|圆]]。
**若<math>\phi=0\,</math>，则这椭圆其实是线段。
*若<math>a=b</math>，<math>n=q=2</math>（所以<math>p=1</math>），则曲线是[[besace|besace]]。
**若<math>\phi=\frac{\pi}{2}</math>，则这besace是[[拋物线|拋物线]]一部份。
**若<math>\phi=0\,</math>，则这besace是一个[[热罗诺双纽线|热罗诺双纽线]]。

== 在電子學上的應用 ==
藉由使用'''利萨茹圖形'''可以測量出兩個[[信號|信號]]的[[頻率|頻率]]比與[[相位|相位]]差。
