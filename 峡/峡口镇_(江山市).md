'''峡口镇'''是[[中国|中国]][[浙江省|浙江省]][[衢州市|衢州市]][[江山市|江山市]]南部的一个[[镇|镇]]<ref>{{PRC admin/ref sgc|{{PRC admin/parent|33/08/81/119/000}}}}</ref>，位于和[[福建|福建]]、[[江西|江西]]两省的交界处，总面积204.7[[平方公里|平方公里]]，辖区内有[[峡口水库|峡口水库]]、[[白水坑水库|白水坑水库]]等，为[[钱塘江|钱塘江]]的重要源头之一。全镇辖18个[[行政村|行政村]]和1个[[居委会|居委会]]，总人口42449人。

== 地理 ==
峡口镇境内地势南高北低。[[仙霞岭|仙霞岭]]群山绵延，海拔1000米以上的山峰有5座，其中最高峰为靛青洋尖，海拔1384米，最低为地山岗，海拔150余米。境内溪河纵横交错，江山市境内最大的河流[[江山港|江山港]]自东南往西北方向穿境而过，以[[峡口水库|峡口水库]]大坝为[[上游|上游]]与[[中游|中游]]界点，上游名为大峦口溪，中游名为峡口溪。峡口镇境内的三卿口溪、广渡溪、王坛溪等，均为江山港的支流。全镇拥有耕地21735亩，山林面积269086亩，森林覆盖率80%。

== 历史沿革 ==
*[[新石器时期|新石器时期]]人类就在峡口周围地区繁衍生息，镇域的肩头垄、古城山有[[商周|商周]]时期的文化遗迹。
*[[北宋|北宋]]时峡口属须江县江山乡三公里。
*南宋时峡口属江山乡廿七[[都甲制|都]]。
*[[明代|明代]]和[[清|清]]初，峡口属江山乡三公里一图。
*清[[雍正|雍正]]时改为江山乡峡口庄，雍正十三年（1735年）衢州府奉文在此设峡口[[同知署|同知署]]，俗称衢州府二衙门。
*清[[宣统|宣统]]二年江山县改为23个自治乡镇，峡口为[[自治镇|自治镇]]之一。
*[[民国|民国]]二十八年改为峡口[[县辖区|署区]]，峡口镇属峡口区。
*民国三十四年，文溪、清湖、淤头三署区栽撤，保留峡口署区，峡口仍为镇制，属峡口镇。
*[[中华人民共和国|中华人民共和国]]成立初期，改为峡口乡。
*1953年改为峡口镇。
*1958年改称为峡口[[人民公社|人民公社]]。
*1961年改为峡口区。
*1983年实行政社分设，改为峡口乡。
*1985年复改峡口镇。
*1992年5月，撤区、扩镇、拼乡后，峡口镇由原王村乡、三卿口乡、峡口镇扩拼而成。
*2005年10月与原大峦口乡合并为新的峡口镇。

== 城镇建设 ==
峡口镇是江山市第三大城镇，中心镇区建成面积3平方公里，聚集人口达3万余人，规模仅次于江山市区和[[贺村镇|贺村镇]]。
目前镇区内的主要街路有：上街、中街、下街、镇前街、中横街、惠民街、兴华街、兴华北街、文教路、金益路、中心街、迎宾路、迎宾中路、迎宾西路、望江路、迎丰路、迎春路、迎新路、迎福路、定村街等。
