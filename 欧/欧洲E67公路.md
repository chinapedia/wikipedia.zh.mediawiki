'''欧洲E67公路'''（{{lang-en|European route E67}}）是一条[[欧洲高速公路|欧洲高速公路]]，起自[[捷克|捷克]][[布拉格|布拉格]]，经过[[波兰|波兰]]、[[立陶宛|立陶宛]]、[[拉脫維亞|拉脫維亞]]和[[爱沙尼亚|爱沙尼亚]]，终于[[芬兰|芬兰]][[赫尔辛基|赫尔辛基]]。<ref>{{Cite web |url = http://www.elbruz.org/eroads/E67.htm |title = E67 |publisher = Elbruz |accessdate = 20 April 2015}}</ref><ref>{{Cite web |url = https://www.unece.org/fileadmin/DAM/trans/conventn/ECE-TRANS-SC1-384e.pdf |title =  AGR 2008 revised-384en bis - unece  |publisher = UN Treaty Collection |accessdate = 20 April 2015}}</ref>它串联了布拉格、[[弗罗茨瓦夫|弗罗茨瓦夫]]、[[华沙|华沙]]、[[考那斯|考那斯]]、[[帕內韋日斯|帕內韋日斯]]、[[里加|里加]]、[[塔林|塔林]]和赫尔辛基等东欧城市。<ref>{{cite news|url=http://news.bbc.co.uk/1/hi/world/europe/6923745.stm|title=Poland halts wetlands road plan|last=Easton|first=Adam|date=2007-07-31|work=BBC News|publisher=BBC|accessdate=2008-08-30}}</ref><ref name="rasoc">[http://wyborcza.pl/1,75248,6417107,Rospuda_ocalona.html ''Rospuda ocalona'']</ref><ref>[http://news.bbc.co.uk/2/hi/europe/7962918.stm BBC - Poland reverses to spare wetland]</ref>
