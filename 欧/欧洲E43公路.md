'''欧洲E43公路'''（'''E43'''）是欧洲的一条[[欧洲国际公路|国际公路]]，起点为[[德国|德国]][[维尔茨堡|维尔茨堡]]，经过[[奥地利|奥地利]]，终点为[[瑞士|瑞士]][[贝林佐纳|贝林佐纳]]，全长约530公里。<ref>{{Cite web |url = http://www.elbruz.org/eroads/E43.htm |title =  E43  |publisher = Elbruz |accessdate = 20 April 2015}}</ref>

== 线路 ==
欧洲E43公路穿过以下主要城市：<ref>{{Cite web |url = https://www.unece.org/fileadmin/DAM/trans/conventn/ECE-TRANS-SC1-384e.pdf |title =  AGR 2008 revised-384en bis - unece  |publisher = UN Treaty Collection |accessdate = 20 April 2015}}</ref>
*德国：[[维尔茨堡|维尔茨堡]] - [[乌尔姆|乌尔姆]] - [[林道|林道]]
*奥地利：[[布雷根茨|布雷根茨]]
*瑞士：[[聖馬格雷滕|聖馬格雷滕]] - [[布克斯_(聖加侖州)|布克斯]] - [[库尔|库尔]] - {{le|聖貝納迪諾 (瑞士)|San Bernardino, Switzerland|圣伯纳迪诺}} - [[贝林佐纳|贝林佐纳]]
