'''欧鲁普雷图'''（{{lang-pt|Vila Rica do Ouro Preto}}）是巴西[[米纳斯吉拉斯|米纳斯吉拉斯]]州的一座城市，葡萄牙殖民统治时期著名的[[矿业|矿业]]城市，位于[[埃斯皮尼亚山|埃斯皮尼亚山]]附近。

1980年，因其拥有较为杰出的[[巴洛克建筑|巴洛克建筑]]，經[[联合国教科文组织|联合国教科文组织]]列入[[世界遗产|世界遗产]]。这也是[[巴西|巴西]]首座列入世界遗产的城市。

== 历史 ==
[[File:Ouro_Preto_November_2009-13.jpg|thumb]]）]]
在葡萄牙语里，{{lang|pt|Ouro Preto}}的意思是“黑色的金子”，因为这里的金矿中含有[[钯|钯]]元素，而钯能够吸收气体，从而使[[金子|金子]]的表面呈现黑色光泽，18世纪，欧鲁普雷图掀起了一场淘金热，这一时期亦是巴西发展的黄金时期，因此人们又将欧鲁普雷图称为“黑金之城”。 

采金创造的巨大财富吸引了[[欧洲|欧洲]]才智的汇集，当地[[哲学|哲学]]和[[艺术|艺术]]蓬勃发展，城市规模亦不断扩大。1711年，葡萄牙国王决定将欧鲁普雷图升格为城市。城市同时拥有保存较为完好的葡萄牙殖民时期与当代城市风格的建筑，在现代的建筑建设中，政府努力维护新的建筑与历史建筑的风格相融合。当地建于18世纪和19世纪的[[教堂|教堂]]用黄金和[[阿莱雅迪尼奥|阿莱雅迪尼奥]]的雕塑作品进行装饰，使欧鲁普雷图渐渐发展成为旅游胜地。<ref>{{cite web |url=http://www.guwh.com/s/ssss/s0704.htm |language= |title=巴西的世界遗产 |publisher=世界遗产专题集邮网 |date= |accessdate=2010年5月7日 |deadurl=yes |archiveurl=https://web.archive.org/web/20070824103725/http://www.guwh.com/s/ssss/s0704.htm |archivedate=2007年8月24日 }}</ref>

1789年，一个向[[葡萄牙|葡萄牙]]殖民主义进行独立的失败尝试——[[米纳斯阴谋|米纳斯阴谋]]在欧鲁普雷图发生，领导人被绞死，以此作为对革命家的威胁。

1876年，矿业学院（{{lang|pt|Escola de Minas}}）建立。这个学校为巴西在之后新矿产的发现提供了技术支持。

欧鲁普雷图在1822年至1897年曾作为[[米纳斯吉拉斯|米纳斯吉拉斯]]州的州政府所在地，后因政府的需求超过了这个[[山谷|山谷]]小城的能力，州政府将所在地迁至[[贝洛奥里藏特|贝洛奥里藏特]]。同时期，由于金矿资源的枯竭，欧鲁普雷图在巴西的影响力不断下降。<ref>{{cite web |url=http://www.tripdv.com/view/v/zl_34683.html |language= |title=巴西：欧鲁普雷图（古镇） |publisher=旅游视频网 |date= |accessdate=2010年5月7日 |deadurl=yes |archiveurl=https://web.archive.org/web/20100514172644/http://www.tripdv.com/view/v/zl_34683.html |archivedate=2010年5月14日 }}</ref><ref>{{cite web|url=http://whc.unesco.org/en/list/124 | language=en | title=Historic Town of Ouro Preto |publisher=UNESCO |date= |accessdate=2010年5月7日 }}</ref>

== 人口 ==
根据[[2000年巴西人口普查|2000年人口普查]]的数据，欧鲁普雷图的常住[[人口|人口]]为66,277人，市区人口56,293人、农村人口9,985人。
[[File:Panorâmica_de_Ouro_Preto.jpg|thumb]]

== 地理 ==
欧鲁普雷图全年[[温度|温度]]在6℃-28℃之间，6月和7月气温可能降至-2℃。全市平均[[海拔|海拔]]1,116米，最高点位于{{lang|pt|Pico de Itacolomi}}，海拔1,722米。城市北部与伊塔比里图和圣巴巴拉接壤，南部与皮兰加伊塔韦拉瓦接壤，东部与马里亚纳接壤，西部与孔戈尼亚斯接壤。

该地区的年平均降雨量为2018mm，降水分布不规则，但主要集中在[[夏季|夏季]]。
