'''欧洲E1公路'''（'''E1'''）是欧洲的一条[[欧洲国际公路|国际公路]]，由几段公路组成，起点为[[北爱尔兰|北爱尔兰]][[拉恩|拉恩]]，经过[[爱尔兰|爱尔兰]]、[[西班牙|西班牙]]、[[葡萄牙|葡萄牙]]，终点为[[西班牙|西班牙]][[塞维利亚|塞维利亚]]，扣除海洋路段全长约1507公里。<ref>{{Cite web |url = http://www.elbruz.org/eroads/E01.htm |title = E01 |publisher = Elbruz |accessdate = 20 April 2015}}</ref>

其中从爱尔兰[[罗斯莱尔港|罗斯莱尔港]]到西班牙[[拉科鲁尼亚|拉科鲁尼亚]]为海洋，无渡轮服务。

== 线路 ==
欧洲E1公路穿过以下主要城市：<ref>{{Cite web |url = https://www.unece.org/fileadmin/DAM/trans/conventn/ECE-TRANS-SC1-384e.pdf |title =  AGR 2008 revised-384en bis - unece  |publisher = UN Treaty Collection |accessdate = 20 April 2015}}</ref><br />
*北爱尔兰：[[拉恩|拉恩]] - [[贝尔法斯特|贝尔法斯特]] - [[利斯本|利斯本]] - [[纽里|纽里]]
*爱尔兰：[[邓多克|邓多克]] - [[都柏林|都柏林]] - [[韦克斯福德|韦克斯福德]] - [[罗斯莱尔港|罗斯莱尔港]] - （海洋）
*西班牙：（海洋）  - [[拉科鲁尼亚|拉科鲁尼亚]] - [[圣地亚哥-德孔波斯特拉|圣地亚哥-德孔波斯特拉]] - [[蓬特韦德拉|蓬特韦德拉]] - [[維戈|維戈]] 
*葡萄牙：[[瓦伦萨_(葡萄牙)|瓦伦萨]] - [[布拉加|布拉加]] - [[波尔图|波尔图]] - [[阿威罗|阿威罗]] - [[科英布拉|科英布拉]] - [[圣塔伦_(葡萄牙)|圣塔伦]] - [[里斯本|里斯本]]  - [[塞图巴尔|塞图巴尔]] - [[阿尔布费拉|阿尔布费拉]] - [[法鲁|法鲁]] - [[马林堡|马林堡]]  - [[圣安东尼奥雷阿尔城|圣安东尼奥雷阿尔城]]
*西班牙：[[韦尔瓦|韦尔瓦]] - [[塞维利亚|塞维利亚]]
