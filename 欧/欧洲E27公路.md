'''欧洲E27公路'''（'''E27'''）是欧洲的一条[[欧洲国际公路|国际公路]]，起点为[[法国|法国]][[贝尔福|贝尔福]]，经过[[瑞士|瑞士]]，终点为[[意大利|意大利]][[奥斯塔|奥斯塔]]，全长约328公里。<ref>{{Cite web |url = http://www.elbruz.org/eroads/E27.htm |title = E27 |publisher = Elbruz |accessdate = 20 April 2015}}</ref>

== 线路 ==
欧洲E27公路穿过以下主要城市：<ref>{{Cite web |url = https://www.unece.org/fileadmin/DAM/trans/conventn/ECE-TRANS-SC1-384e.pdf |title =  AGR 2008 revised-384en bis - unece  |publisher = UN Treaty Collection |accessdate = 20 April 2015}}</ref><br />
*法国：[[贝尔福|贝尔福]]
*瑞士：[[德莱蒙|德莱蒙]] - [[伯尔尼|伯尔尼]] - {{le|馬爾蒂尼 (瓦萊州)|Martigny|馬爾蒂尼}} - [[大圣伯纳德山口|大圣伯纳德山口]]
*意大利：[[奥斯塔|奥斯塔]]
