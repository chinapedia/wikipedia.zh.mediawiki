'''歐嘉隆'''（大陸譯作「欧-{加}-农」、臺灣譯作「歐-{嘉}-隆」）是一家以[[荷蘭|荷蘭]][[奧斯|奧斯]]及[[美國|美國]][[新澤西州|新澤西州]]Roseland作共同基地的藥廠，亦是總部設於荷蘭[[阿納姆|阿納姆]]的[[阿克蘇諾貝爾|阿克蘇諾貝爾]]集團旗下的健康產品組成機構。歐嘉隆的主要產品在三方面，分別為：[[避孕藥|避孕藥]]、[[生殖藥物|生殖藥物]]及更年期適應藥物。此外，歐嘉隆亦有生產心理健康、麻醉和心血管疾病的藥物。

== 簡介 ==

歐嘉隆的主要生產基地位於荷蘭，行銷亦是歐洲總部的主要功能。在美國，歐嘉隆除了位於Roseland的總部以外，在[[賓夕法尼亞州|賓夕法尼亞州]]的Allentown亦有一個佔地約1.6萬平方米的包裝及分銷中心，並在[[麻薩諸塞州|麻薩諸塞州]]的劍橋有一所生物科技研究設施。歐嘉隆在[[蘇格蘭|蘇格蘭]]Newhouse的研究設施是它在歐洲的研究所在地，另外在其他三個國家亦有研發隊伍。它的產品在全球超過100個國家都有發售，而當中超過60個國家都設有分公司或附屬公司、15個國家設有生產線(主要生產線設於[[中華人民共和國|中]]、[[英國|英]]、[[美國|美]]、[[日本|日]]、[[德國|德]]及[[荷蘭|荷]]六國)。歐嘉隆現時在全球僱有13,000名員工，單是美國本身就已有超過1800名員工。歐嘉隆在台灣及中國大陸都設有分公司。

== 歷史 ==
歐嘉隆於1923年由Dr. Saal van Zwanenberg在荷蘭成立，名叫“Zwanenberg-Organon”。當時的產品主要是從動物副産品中生産出[[胰島素|胰島素]]和其他藥品，供人類使用。<!-- With Van Zwanenberg if commercial director, and Laqueur as scientific control Organon developed into an international company, which had been represented in 1934-->1930年代成功使[[雌激素|雌激素]]商品化。 

[[第二次世界大戰|第二次世界大戰]]之後，歐嘉隆積極開拓新市場。1953年開始生產[[可的松|可的松]](又譯[[皮質酮|皮質酮]])。1962年，歐嘉隆收購了Lyndiol。Lyndiol是世上第一家生產[[避孕丸|避孕丸]](口服避孕藥)的公司。

== 爭議 ==

2004年，歐嘉隆藥廠曾被指控涉嫌透過濫用《[[藥品價格競爭及專利回復法|藥品價格競爭及專利回復法]]》，以不法手段擴大壟斷其產品[[樂活憂|樂活憂]](Remeron)在[[抗憂鬱藥|抗憂鬱劑]]的市場佔有率，而被[[史夫仁和巴洛衛律師事務所|史夫仁和巴洛衛律師事務所]]帶上法庭。結果雙方以3600萬美元達成和解協議<ref>http://www.sbclasslaw.com/chinese_traditional/achievements.cfm#g， {{webarchive|url=https://web.archive.org/web/20060529214149/http://www.sbclasslaw.com/chinese_traditional/achievements.cfm |date=2006-05-29 }}[[樂活憂|樂活憂]][[反壟斷法|反壟斷]]訴訟，編號02-CV-2007 (D.N.J. 2004)</ref>。
