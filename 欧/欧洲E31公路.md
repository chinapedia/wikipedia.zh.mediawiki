'''欧洲E31公路'''（'''E31'''）是欧洲的一条[[欧洲国际公路|国际公路]]，起点为[[荷兰|荷兰]][[鹿特丹|鹿特丹]]，终点为[[德国|德国]][[路德维希港|路德维希港]]，全长约538公里。<ref>{{Cite web |url = http://www.elbruz.org/eroads/E31.htm |title =  E31  |publisher = Elbruz |accessdate = 20 April 2015}}</ref>

== 线路 ==
欧洲E31公路穿过以下主要城市：<ref>{{Cite web |url = https://www.unece.org/fileadmin/DAM/trans/conventn/ECE-TRANS-SC1-384e.pdf |title =  AGR 2008 revised-384en bis - unece  |publisher = UN Treaty Collection |accessdate = 20 April 2015}}</ref><br />
*荷兰：[[鹿特丹|鹿特丹]] - [[霍林赫姆|霍林赫姆]] - [[奈梅亨|奈梅亨]]
*德国：[[戈赫|戈赫]] - [[克雷费尔德|克雷费尔德]] - [[科隆|科隆]] - [[科布伦茨|科布伦茨]] - [[萊茵河畔賓根|萊茵河畔賓根]] - [[路德维希港|路德维希港]]
