'''欧洲E39公路'''（'''E39'''）是欧洲的一条[[欧洲国际公路|国际公路]]，起点为[[挪威|挪威]][[特隆赫姆|特隆赫姆]]，终点为[[丹麦|丹麦]][[奥尔堡|奥尔堡]]，全长约1330公里。<ref>{{Cite web |url = http://www.elbruz.org/eroads/E39.htm |title =  E39 |publisher = Elbruz |accessdate = 20 April 2015}}</ref>

== 线路 ==
欧洲E39公路穿过以下主要城市：<ref>{{Cite web |url = https://www.unece.org/fileadmin/DAM/trans/conventn/ECE-TRANS-SC1-384e.pdf |title =  AGR 2008 revised-384en bis - unece  |publisher = UN Treaty Collection |accessdate = 20 April 2015}}</ref><br />
*挪威：[[特隆赫姆|特隆赫姆]] - [[奥勒松|奥勒松]] - [[卑尔根|卑尔根]] - [[斯塔万格|斯塔万格]] - [[克里斯蒂安桑|克里斯蒂安桑]] - （渡轮） 
*丹麦：（渡轮）- [[希茨海尔斯|希茨海尔斯]] - {{le|约灵|Hjørring}} - {{le|诺勒松比|Nørresundby}} - [[奥尔堡|奥尔堡]]
