'''欧洲E9公路'''（'''E9'''）是欧洲的一条[[欧洲国际公路|国际公路]]，起点为[[法国|法国]][[奥尔良|奥尔良]]，终点为[[西班牙|西班牙]][[巴塞罗那|巴塞罗那]]，全长约967公里。<ref>{{Cite web |url = http://www.elbruz.org/eroads/E09.htm |title = E09 |publisher = Elbruz |accessdate = 20 April 2015}}</ref>

== 线路 ==
欧洲E9公路穿过以下主要城市：<ref>{{Cite web |url = https://www.unece.org/fileadmin/DAM/trans/conventn/ECE-TRANS-SC1-384e.pdf |title =  AGR 2008 revised-384en bis - unece  |publisher = UN Treaty Collection |accessdate = 20 April 2015}}</ref><br />
*法国：[[奥尔良|奥尔良]] - [[维耶尔宗|维耶尔宗]] - [[瓦唐_(安德爾省)|瓦唐]] - [[沙托鲁|沙托鲁]] - [[利摩日|利摩日]] - [[布里夫拉盖亚尔德|布里夫拉盖亚尔德]] - [[卡奥尔|卡奥尔]] - [[蒙托邦|蒙托邦]] - [[图卢兹|图卢兹]] - [[欧特里弗|欧特里弗]] - [[帕米耶|帕米耶]] - [[富瓦|富瓦]] - [[阿克斯莱泰尔姆|阿克斯莱泰尔姆]] - {{link-fr|皮莫朗山口|Col de Puymorens}} - [[布爾馬當|布爾馬當]] 
*西班牙：[[普伊格塞爾達|普伊格塞爾達]] - [[阿爾普|阿爾普]] - [[巴加_(西班牙)|巴加]] - [[贝尔加|贝尔加]] - [[希罗内利亚|希罗内利亚]] - [[曼雷萨|曼雷萨]] - [[塔拉薩_(加泰羅尼亞)|塔拉萨]] - [[魯維|魯維]] - [[巴塞罗那|巴塞罗那]]
