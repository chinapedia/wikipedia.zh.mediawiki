'''欧洲E13公路'''（'''E13'''）是欧洲的一条[[欧洲国际公路|国际公路]]，全程均在[[英国|英国]]，起点为[[唐卡斯特|唐卡斯特]]，终点为[[伦敦|伦敦]]，全长约328公里。该公路主要经过英格兰[[M1高速公路_(英格兰)|M1高速公路]]。<ref>{{Cite web |url = http://www.elbruz.org/eroads/E13.htm |title = E13 |publisher = Elbruz |accessdate = 20 April 2015}}</ref>

== 线路 ==
欧洲E13公路穿过以下主要城市：<ref>{{Cite web |url = https://www.unece.org/fileadmin/DAM/trans/conventn/ECE-TRANS-SC1-384e.pdf |title =  AGR 2008 revised-384en bis - unece  |publisher = UN Treaty Collection |accessdate = 20 April 2015}}</ref><br />
*英国：[[唐卡斯特|唐卡斯特]] - [[谢菲尔德|谢菲尔德]] - [[诺丁汉|诺丁汉]] - [[莱斯特|莱斯特]] - [[北安普敦|北安普敦]] - [[卢顿|卢顿]] - [[伦敦|伦敦]]
