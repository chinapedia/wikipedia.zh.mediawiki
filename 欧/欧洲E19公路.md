'''欧洲E19公路'''（'''E19'''）是欧洲的一条[[欧洲国际公路|国际公路]]，起点为[[荷兰|荷兰]][[阿姆斯特丹|阿姆斯特丹]]，经过[[比利时|比利时]]，终点为[[法国|法国]][[巴黎|巴黎]]，全长约551公里。<ref>{{Cite web |url = http://www.elbruz.org/eroads/E19.htm |title = E19 |publisher = Elbruz |accessdate = 20 April 2015}}</ref>

== 线路 ==
欧洲E19公路穿过以下主要城市：<ref>{{Cite web |url = https://www.unece.org/fileadmin/DAM/trans/conventn/ECE-TRANS-SC1-384e.pdf |title =  AGR 2008 revised-384en bis - unece  |publisher = UN Treaty Collection |accessdate = 20 April 2015}}</ref><br />
*荷兰：[[阿姆斯特丹|阿姆斯特丹]] - [[海牙|海牙]] - [[莱斯韦克|莱斯韦克]] - [[鹿特丹|鹿特丹]] - [[布雷达|布雷达]] 
*比利时：[[安特卫普|安特卫普]] - [[布鲁塞尔|布鲁塞尔]] - [[蒙斯|蒙斯]] 
*法国：[[瓦朗谢讷|瓦朗谢讷]] - [[康布雷|康布雷]] - [[贡比涅|贡比涅]] - [[巴黎|巴黎]]
