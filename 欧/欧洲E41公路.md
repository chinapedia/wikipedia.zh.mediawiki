'''欧洲E41公路'''（'''E41'''）是欧洲的一条[[欧洲国际公路|国际公路]]，起点为[[德国|德国]][[多特蒙德|多特蒙德]]，终点为[[瑞士|瑞士]][[阿尔特多夫|阿尔特多夫]]，全长约788公里。<ref>{{Cite web |url = http://www.elbruz.org/eroads/E41.htm |title =  E41  |publisher = Elbruz |accessdate = 20 April 2015}}</ref>

== 线路 ==
欧洲E41公路穿过以下主要城市：<ref>{{Cite web |url = https://www.unece.org/fileadmin/DAM/trans/conventn/ECE-TRANS-SC1-384e.pdf |title =  AGR 2008 revised-384en bis - unece  |publisher = UN Treaty Collection |accessdate = 20 April 2015}}</ref><br /><br />
*德国：[[多特蒙德|多特蒙德]] - [[哈根|哈根]] - [[奧爾珀|奧爾珀]] - [[锡根|锡根]] - [[韦茨拉尔|韦茨拉尔]] - [[哈瑙|哈瑙]] - [[阿沙芬堡|阿沙芬堡]] - [[维尔茨堡|维尔茨堡]] - [[海尔布隆|海尔布隆]] - [[斯图加特|斯图加特]] - [[伯布林根|伯布林根]] - [[黑倫貝格|黑倫貝格]] - [[菲林根-施文宁根|菲林根-施文宁根]] - [[巴特迪尔海姆|巴特迪尔海姆]] - [[辛根|辛根]]
*瑞士：[[沙夫豪森|沙夫豪森]] - [[温特图尔|温特图尔]] - [[苏黎世|苏黎世]] - [[施维茨|施维茨]] - [[阿爾特多夫_(瑞士)|阿爾特多夫]]
