'''欧洲塞浦路斯航空'''（Eurocypria Airlines），是[[塞浦路斯共和国|塞浦路斯共和国]]的一家[[航空公司|航空公司]]，总部设在[[拉纳卡|拉纳卡]]。该公司以[[拉纳卡国际机场|拉纳卡国际机场]]为[[枢纽机场|枢纽]]、[[帕福斯国际机场|帕福斯国际机场]]为次要枢纽，主营飞往[[欧洲|欧洲]]各国的定期与[[包机|包机]]航班。<ref>{{En icon}}{{cite journal en |date=2007年4月3日 |title=Directory: World Airlines |journal=Flight International |pages=p78 }}</ref>

== 历史 ==
[[File:Eurocypria_b737-800_5b-dbu_arp.jpg|thumb]]降落]]
欧洲塞浦路斯航空成立于1991年，是塞浦路斯首家包机航空公司。<ref name="facts">{{cite web |url=http://www.eurocypria.com/main/default.aspx?tabid=113 |title=Eurocypria Facts |publisher=欧洲塞浦路斯航空 |date= |accessdate=2009年8月17日 |language=en }}{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>1992年，公司以两架全新的[[空中客车A320|空中客车A320]]客机开始运营。首航日期是1992年3月25日，从塞浦路斯拉纳卡飞往[[英国|英国]][[伯明翰|伯明翰]]。这一年，欧洲塞浦路斯航空开通了飞往6个[[国家|国家]]、15个[[机场|机场]]的航班，全年共执行675对往返班次，运送195650名乘客。次年，公司引进了第三架空客A320。<ref name="history">{{cite web |url=http://www.eurocypria.com/main/default.aspx?tabid=130 |title=Our History |publisher=欧洲塞浦路斯航空 |date= |accessdate=2009年8月17日 |language=en }}{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>

2001年起，欧洲塞浦路斯航空在包机业务的基础上，也开通了定期航班。2003年上半年，公司对机队进行全面更新，用4架[[波音737|波音737-800W]]客机更换了原有的空中客车飞机。后来，又分别在2006年和2007年引进第5和第6架波音737-800W。<ref name="history"/>

欧洲塞浦路斯航空在成立时是[[塞浦路斯航空|塞浦路斯航空]]的全资子公司。但在2006年，塞浦路斯航空将欧洲塞浦路斯航空以1345万[[塞浦路斯镑|塞浦路斯镑]]的价格完全转让给塞浦路斯政府。<ref>{{cite web |url=http://www.allbusiness.com/operations/shipping-air-freight/1180840-1.html |title=Cyprus Airways sells off Eurocypria |publisher=Airline Industry Information  |date=2006年6月26日 |accessdate=2009年8月17日 |language=en }}</ref>

== 目的地 ==
截至2009年8月17日，欧洲塞浦路斯航空的定期航班通达塞浦路斯、[[奥地利|奥地利]]、[[德国|德国]]、[[希腊|希腊]]、[[俄罗斯|俄罗斯]]和[[英国|英国]]等6个国家的17个目的地<ref>{{cite web |url=http://www.eurocypria.com/main/default.aspx?tabid=98 |title=Flight Schedules & Timetable |publisher=欧洲塞浦路斯航空 |date= |accessdate=2009年8月17日 |language=en }}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>（见下表），而其包机航班则覆盖更多欧洲国家以及部分[[中东|中东]]国家<ref>{{cite web |url=http://www.eurocypria.com/main/default.aspx?tabid=99 |title=Our Destinations for Summer 09 |publisher=欧洲塞浦路斯航空 |date= |accessdate=2009年8月17日 |language=en }}{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>。
