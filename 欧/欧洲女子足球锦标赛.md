'''歐洲女子足球錦標賽'''（'''UEFA European Women's Championship'''，又称UEFA Women's Euro），是[[欧洲足联|欧洲足联]]各成员国[[女子足球|女子足球]]代表队之间的最高级别比赛，每四年举办一届，对应男子足球中的[[欧洲足球锦标赛|欧洲足球锦标赛]]。

== 历史 ==
[[File:UEFA_Women's_Euro_2009_final_(ceremony_before_the_match).jpg|thumb]]
欧洲女子足球锦标赛的前身是'''欧洲女子足球代表队竞赛'''（UEFA European Competition for Representative Women's Teams），创办于1982年。首届比赛共有16支球队参加，分成4个小组展开预选赛，揭幕战于1982年8月18日举行，由[[芬兰国家女子足球队|芬兰队]]在[[瓦马拉|瓦马拉]]迎战[[瑞典国家女子足球队|瑞典队]]。<ref name="h1984">{{cite web |url=http://www.uefa.com/competitions/woco/history/season=1984/intro.html |first=Paul |last=Saffer |title=1984: The Swede smell of success |publisher=[[UEFA|UEFA]] |date= |accessdate=2009年9月16日 |language=en }}</ref>决赛于1984年5月在瑞典队和[[英格兰女子足球代表队|英格兰队]]之间进行，采取主客场两回合赛制。双方均在各自的主场以1-0战胜对手，最后瑞典经点球大战夺冠。<ref name="h1984"/><ref name="history">{{cite web |url=http://www.uefa.com/competitions/woco/history/index.html |title=History |publisher=[[UEFA|UEFA]] |date= |accessdate=2009年9月16日 |language=en }}</ref>接下来的两届比赛分别于1987年和1989年在[[挪威|挪威]]和[[联邦德国|联邦德国]]举行，并且均由东道主夺冠。<ref name="history"/><ref name="h1987">{{cite web |url=http://www.uefa.com/competitions/woco/history/season=1987/intro.html |first=Paul |last=Saffer |title=1987: Norway victorious in Oslo |publisher=[[UEFA|UEFA]] |date= |accessdate=2009年9月16日 |language=en }}</ref><ref name="h1989">{{cite web |url=http://www.uefa.com/competitions/woco/history/season=1989/intro.html |first=Paul |last=Saffer |title=1989: Germany arrive in style |publisher=[[UEFA|UEFA]] |date= |accessdate=2009年9月16日 |language=en }}</ref>

1991年起，这项赛事更名为欧洲女子足球锦标赛，仍保持两年一届。<ref name="h1991">{{cite web |url=http://www.uefa.com/competitions/woco/history/season=1991/intro.html |first=Paul |last=Saffer |title=1991: Dominant Germany stride on |publisher=[[UEFA|UEFA]] |date= |accessdate=2009年9月16日 |language=en }}</ref>其中，1991年和1995年两届比赛也同时作为[[女子世界杯足球赛|女子世界杯足球赛]]的欧洲区预选赛。1997年起，赛事改为四年一届。<ref name="history"/>更名后的欧洲女子足球锦标赛至今已举办8届，最成功的球队是[[德国国家女子足球队|德国队]]，夺得了其中的7次冠军，并且在最近6届赛事中实现6连冠。<ref name="history"/>最近一届比赛于2013年7月10日至28日在[[瑞典|瑞典]]举行。
