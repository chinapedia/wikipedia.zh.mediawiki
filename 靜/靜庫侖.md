在[[厘米-克-秒制|厘米-克-秒單位制]]（CGS制）裏，'''靜庫侖'''（statC）或（esu）是[[電荷|電荷]]的[[測量單位|物理單位]]。它是一種衍生出來的單位，表達為
: 1 statC = 1 g<sup>1/2</sup> cm<sup>3/2</sup> s<sup>-1</sup> = 1 erg<sup>1/2</sup> cm<sup>1/2</sup>。

在[[國際單位制|國際單位制]]裏，則採用[[庫侖|庫侖]]（C）為電荷的物理單位。轉換公式為
: 1 C = 2997924580 statC。

這公式乃精確的（但是，請參閱後面正確使用方法的警告）。在公式右邊的數值是[[光速|光速]]（CGS制）的十分之一。雙方向轉換大約為
: 1 C ≈ 2.99792{{e|9}} statC。
: 1 statC ≈ 3.33564{{e|−10}} C。

靜庫侖定義為：假若兩個固定的，相距1 cm的點電荷各自帶有電量1 statC，則它們彼此互相排斥的[[靜電力|靜電力]]是1 [[達因|達因]]（dyne）。這排斥力是由[[庫侖定律|庫侖定律]]給出，在CGS制裏，表達為
:<math>F=\frac{q_1q_2}{r^2}\,\!</math>。

其中，<math>F\,\!</math>是力量，<math>q_1\,\!</math>、<math>q_2\,\!</math>分別是兩個點電荷的電量，<math>r\,\!</math>是它們之間的距離。

== 因次分析 ==
做一個[[因次分析|因次分析]]。<math>F\,\!</math>的因次是[質量][長度][時間]<sup>-2</sup>。所以，在CGS制裏，電荷的因次必須是[質量]<sup>1/2</sup> [長度]<sup>3/2</sup> [時間]<sup>-1</sup>。

在國際單位制裏，這句話並不正確。因為，庫侖定律的形式不同：
:<math>F=\frac{1}{4\pi\epsilon_0}\frac{q_1q_2}{r^2}\,\!</math>；

其中，<math>\epsilon_0\,\!</math>是[[真空電容率|真空電容率]]。

由於<math>\epsilon_0\,\!</math>本身有因次。與靜庫侖不同，庫侖的因次不是[質量]<sup>1/2</sup> [長度]<sup>3/2</sup> [時間]<sup>-1</sup>。實際上，我們無法只用質量、長度和時間來表達庫侖的因次。

因此，以下這公式必須非常小心地使用，公式兩邊的單位並不一致：
: 1 C = 2997924580 statC。

我們不能直接地使用這轉換因子在庫侖與靜庫侖之間做轉換，就好像我們在公分與公尺之間做轉換一樣。更確切地說，這公式應該詮釋為："1 coulomb"對應於"2997924580 statC"。換句話說，假若某一個帶電體的電量是1 coulomb，那麼，它的電量也是  2997924580 statC。

特別注意，以下這轉換的因次完全一致，對於國際單位制和CGS制之間的轉換，非常有用：
:<math>\frac{1 \; \mathrm{C}}{\sqrt{4 \pi \epsilon_0}} = 2997924580 \; \mathrm{statC}\,\!</math>；

其中，<math>\epsilon_0\,\!</math>是CGS制的真空電容率，大約為8.85{{e|−21}} [[安培|A]]<sup>2</sup>s<sup>4</sup> [[克|g]]<sup>-1</sup>[[厘米|cm]]<sup>−3</sup>。
