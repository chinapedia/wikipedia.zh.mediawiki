'''靜思精舍'''（[[英文|英文]]：Jing Si Abode）位於[[臺灣|臺灣]][[花蓮縣|花蓮縣]][[新城鄉_(臺灣)|新城鄉]][[康樂村|康樂村]]，是[[佛教慈濟功德會|佛教慈濟功德會]]（今[[慈濟基金會|慈濟基金會]]）所在地。

「靜思」是[[證嚴法師|證嚴法師]]走出俗家、現[[出家|出家]]相前，自取的名字；「精舍」則是修行人自修、清修的道場。

靜思精舍的修行者，自力更生，秉持一日不做、一日不食的叢林生活。

== 緣起 ==

1964年，證嚴法師與其弟子借住普明寺，過著耕讀的生活。1966年，佛教克難慈濟功德會成立不久，普明寺附近剛好有人要賣地，法師俗家母親王沈月桂女士聽到消息，於是出錢買下了現今靜思精舍的土地。1968年，因功德會救濟的會務愈來愈多，於是用土地向銀行抵押貸款蓋靜思精舍，大殿採[[唐|唐]]式建築風格。1969年，靜思精舍落成使用。

最早只有大殿及旁邊加蓋的廚房，大殿內供奉[[本師釋迦牟尼佛|本師釋迦牟尼佛]]、[[觀世音菩薩|觀世音菩薩]]、[[地藏王菩薩|地藏王菩薩]]。靜思精舍屋頂本為[[日本黑瓦|日本黑瓦]]所舖，因花蓮多[[颱風|颱風]]、[[地震|地震]]，早先的屋瓦多已破損；證嚴法師決定用[[水泥|水泥]]砌以防止[[天災|天災]]帶來的損害，並親自與工人一同施工、調整屋脊的斜度。

== 自力耕生 ==

靜思精舍不接受供養，為了維持精舍營運，從證嚴法師現出家相後，僧俗二眾做了二三十種手工藝與工作，維持精舍的開支，未取用功德會的善款。手工藝如下：<br>
1.織毛線<br>
2.種田<br>
3.糊水泥袋<br>
4.嬰兒鞋<br>
5.紗棉手套(手動機器製作)<br>
6.成衣加工<br>
7.背心代工<br>
8.木雕<br>
9.漏電斷路器零件<br>
10.女裝代工<br>
11.紗棉手套(自動機器製作)<br>
12.布手套<br>
13.高週波尿褲<br>
14.種菊花<br>
15.串項鍊<br>
16.塑膠花<br>
17.猴子爬樹玩具<br>
18.手拉坏<br>
19.化妝品瓶蓋內墊<br>
20.薏豆粉<br>
21.蠟燭<br>

== 今日之精舍 ==
靜思精舍主要是作為女眾或常住眾修行之所，從1968年年至今，因人數及功能的增加，前前後後共增建了十幾期的工程。[[慈濟|慈濟]]之會眾稱靜思精舍為「心靈的故鄉」<ref>{{cite web|url=http://www.tzuchi.org.tw/index.php?option=com_content&view=article&id=47%3Ajingsi-abode-and-tzuchi&catid=119%3A2009-03-27-06-10-09&Itemid=429&lang=zh|title=靜思精舍與慈濟世界：心靈故鄉|publisher=慈濟全球資訊網|accessdate=2015年4月1日|archive-url=https://web.archive.org/web/20150402151509/http://www.tzuchi.org.tw/index.php?option=com_content&view=article&id=47%3Ajingsi-abode-and-tzuchi&catid=119%3A2009-03-27-06-10-09&Itemid=429&lang=zh|archive-date=2015-04-02|dead-url=yes}}</ref>。
