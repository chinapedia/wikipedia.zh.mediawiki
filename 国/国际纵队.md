'''國際縱隊'''（{{lang-es|Brigada Internacional}}、{{lang-en|International Brigades }}）是[[西班牙內戰|西班牙內戰]]時（1936年至1939年）[[西班牙第二共和國|西班牙第二共和國]]的軍事單位。由[[英國|英國]]、[[法國|法國]]、[[美國|美國]]等56个國家的[[志願兵|志願兵]]組成，以對抗西班牙法西斯主義者（[[法西斯主義|法西斯主義]]勢力）。縱隊成員还包括一批[[詩人|詩人]]、[[藝術家|藝術家]]、[[醫生|醫生]]和[[記者|記者]]，如[[阿爾貝·加繆|加缪]]、[[聶魯達|聶魯達]]、[[海明威|海明威]]、[[喬治·歐威爾|奥威尔]]、[[畢卡索|畢卡索]]等。

== 國際縱隊的成立 ==

在1936年9月間，[[共產國際|共產國際]]人士Willi Münzenberg首先提議由各國[[共產黨|共產黨]]招募志願兵加入西班牙內戰。在[[蘇聯|蘇聯]][[國防部|國防部]]的協助下，[[意大利共产党_(1921年)|意大利共产党]]和[[法國共產黨|法國共產黨]]隨後即著手成立縱隊。 

[[巴黎|巴黎]]是主要的招募中心，志願者在此獲得金錢及[[護照|護照]]方面的協助，並由鐵路或乘船前往西班牙。志願者主要來自義大利、[[德國|德國]]與其它當時的專制國家，隊伍中也有許多[[失業者|失業者]]（主要來自法國）與冒險者；此外，有大約五百名之前遭流放至俄羅斯的共產黨人來到西班牙，其中包括許多活躍於[[第一次世界大戰|第一次世界大戰]]的軍事領袖。

共產主義者熱烈支持這個計畫，[[無政府主義者|無政府主義者]]則持懷疑態度。起初，控制法國邊界的無政府主義人士拒絕讓共產黨志願兵入境，遭抗議後才不情願地放行。

第一批的500名志願者在1936年10月14日抵達[[阿爾瓦塞特|阿爾瓦塞特]]，他們在此會見已在西班牙戰鬥的國際志願人士，並依各人之經驗與出身進行編成。為了溝通上的便利，成員依國籍或語言編組。阿爾瓦塞特隨後成為國際縱隊的總部及主要倉庫，並由共產國際的三位重要人士André Marty、Luigi Longo與Giuseppe Di Vittorio領導。

法國共產黨為縱隊提供制服。縱隊的內部管理十分嚴格：成員必須在基地裡待上數週，進行嚴格的軍事訓練。

== 成員比例 ==
首批國際縱隊的成員大多來自法國、[[比利時|比利時]]、義大利及德國，此外還有一批來自法國北部及比利時的[[波蘭|波蘭]][[礦工|礦工]]。第一批編成的縱隊是第11、12及13縱隊，第14及15縱隊由老手與新手混編而成。規模較小的第86、129與150縱隊在1937年底及1938年成立，主要是出於短期的戰術目的。國際縱隊各營多以著名的人物或事件命名。

總計約有32,000人參與國際縱隊<ref name="Thomas 2003">Thomas（2003）,pp 941-3</ref>，其中不乏第一次世界大戰的老兵。他們的能耐在[[馬德里包圍戰|馬德里包圍戰]]中展露無遺。

== 馬德里戰役 ==
發生於1936年10月至1937年1月的[[馬德里|馬德里]]保衛戰是共和政府的重大勝利，此役粉碎了[[佛朗哥將軍|佛朗哥將軍]]的速勝迷夢。國際縱隊在此役中的重要性廣為人知，有時甚至被誇大。例如，當時的[[英國|英國]][[大使|大使]]Henry Childon [[爵士|爵士]]宣稱馬德里守軍中沒有西班牙人；事實上，四萬名守軍中只有三千多人不是西班牙人。

馬德里的戰略要地之一是Casa de Campo，共和政府的[[第三縱隊|第三]]及[[第四縱隊|第四縱隊]]在此阻擋了叛軍的攻勢。進攻此地的叛軍為[[摩洛哥|摩洛哥]]軍隊，他們長於在開闊地戰鬥，但不擅[[巷戰|巷戰]]。

在1936年11月9日，由1,900人組成的國際縱隊第十一縱隊進駐此地，並於當晚對叛軍發動攻擊。戰鬥持續至隔日早晨，結果叛軍不得不撤退，並放棄直接由Casa de Campo進攻馬德里的計畫，而[[第十一縱隊|第十一縱隊]]則損失了三分之一的人員。

在11月13日，由1,550人構成的[[第十二縱隊|第十二縱隊]]進攻Cerro de Los Angeles高地的叛軍。由於語言與通訊問題、與裝甲部隊的協調不良及缺乏砲兵支援，這波攻勢以失敗告終。

11月19日，共和軍下屬的無政府主義者部隊被迫撤離，而叛軍在納粹[[禿鷹軍團|禿鷹軍團]]的掩護下進據大學城。第十一縱隊受命將叛軍逐出[[大學城|大學城]]。戰鬥十分激烈，雙方以手雷與[[刺刀|刺刀]]逐室相搏，並夾雜著砲擊與[[飛機|飛機]]轟炸。大學裡的戰鬥持續到叛軍掌控四分之三的地盤為止，然後雙方各自構築陣地。此時無論由任一方進攻，都會付出嚴重代價。叛軍將領不得不放棄直接攻城的計畫，並準備[[圍城|圍城]]。

在1936年12月13日，18,000名叛軍企圖進攻馬德里外圍的Guadarrama，此役稱為[[Corunna路戰役|Corunna路戰役]]。共和軍派出一個蘇聯的[[裝甲部隊|裝甲部隊]]，及第十一、十二國際縱隊。在激烈的戰鬥後，他們阻止了叛軍的攻勢。

叛軍在[[耶誕節|耶誕節]]後的圍城計畫受挫，但共和軍也付出了慘重代價。在1937年1月6日，國際縱隊下屬的[[台爾曼營|台爾曼營]]抵達Las Rozas，並在此堅守至全軍覆沒為止。在1月9日，叛軍的進展只有十公里，而第十三、十四國際縱隊抵達馬德里。隨後共和軍企圖奪回失地，戰況激烈但收效不大。1月15日，雙方各自構築陣地，戰局陷入僵持。

叛軍直到戰爭尾聲的1939年3月才取得馬德里，其後幾個月間仍存有零星的反抗區。

== 哈拉瑪戰役 ==
隨著[[馬拉加|馬拉加]]失守，叛軍在1937年2月進攻[[馬德里|馬德里]]-[[安達魯西亞|安達魯西亞]]的道路。雙方都付出嚴重代價，戰事在二月底陷入僵持，雙方都自稱獲勝，但雙方都遭遇了失敗<ref>Thomas（2003）,579</ref>。

在此期間，[[國際聯盟|國際聯盟]]於2月22日宣佈禁止外國志願兵介入戰局。

== 瓜達拉哈拉戰役 ==
由於叛軍在哈拉瑪的進展有限，遂改道馬德里東北方50公里的[[瓜達拉哈拉|瓜達拉哈拉]]。[[墨索里尼|墨索里尼]]派出含35,000人、80輛戰車與200門大砲的特遣隊，以為義大利記上一筆戰功。叛軍在3月9日攻破共和軍的防線，共和軍方面遂組織包含第十一、十二國際縱隊在內較有戰力的單位。

在3月10日[[凌晨|凌晨]]，國際縱隊的[[加里波第|加里波第]]營發起反攻。叛軍在3月11日突破防線，但台爾曼營與[[加里波底營|加里波底營]]仍守住了要地。共和軍的飛機與戰車在3月12日發起了攻擊。台爾曼營奪回了Trijuete鎮，並俘獲大量叛軍。

== 國際縱隊的解散 ==
為了讓叛軍外援撤離，並讓英、法等國解除禁運，共和政府於1938年9月21日宣稱依[[國際聯盟|國際聯盟]]之決議，解散國際縱隊。此時仍有大約一萬名外國志願者在西班牙戰鬥，其中過半來自德國、義大利及[[匈牙利|匈牙利]]等當時的右翼法西斯政權；這些人無法安全返國，西班牙政府遂授予他們榮譽公民資格，並納入人民軍。其餘人員則被遣送回國。

國際縱隊於10月底在[[巴塞隆納|巴塞隆納]]進行了最後的遊行，[[總統|總統]]與[[总理|总理]]皆到場致意；[[西班牙共產黨|西班牙共產黨]]代表[[多洛雷斯·伊巴露丽|多洛雷斯·伊巴露丽]]（人稱“热情之花”）更發表了慷慨激昂的演說<ref>[http://www.english.uiuc.edu/maps/scw/farewell.htm La Pasionaria's Farewell Address], Barcelona, November 1, 1938</ref>，成為西班牙內戰的名篇之一。
