'''国民政府资源委员会旧址'''位于[[中国|中国]][[江苏省|江苏省]][[南京市|南京市]][[鼓楼区|鼓楼区]][[中山北路_(南京)|中山北路]]200号（原虹桥20号），今为[[南京工业大学|南京工业大学]]虹桥校区办公楼。该建筑建于1947年，由[[杨廷宝|杨廷宝]]设计，原由大门、警卫亭、办公楼和辅助用房组成，其中大门临中山北路、面朝西南，单开间，两个立柱内各设有一个警卫室，大门两侧各有一个警卫亭（今存西北侧警卫亭），大门与警卫亭均为砖木结构，屋顶覆以绿色琉璃瓦，警卫亭采用庑殿顶仿古建筑风格，梁枋施有彩绘。办公楼为两层混合结构建筑，面朝东南，红砖墙，屋顶覆以灰色水泥板瓦，建筑面积约2600平方米<ref>{{cite web |url = http://www.njmgjz.cn/xzjz/b238 |title = 《国民政府资源委员会》 |accessdate = 2013-08-12 |work =  |publisher = 南京民国建筑网 |archive-url = https://web.archive.org/web/20141021205640/http://www.njmgjz.cn/xzjz/b238 |archive-date = 2014-10-21 |dead-url = yes }}</ref>。
