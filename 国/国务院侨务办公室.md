'''国务院侨务办公室'''，简称'''国务院侨办'''或'''国侨办'''，是[[中共中央统一战线工作部|中共中央统一战线工作部]]对外加挂的牌子，為中央统战部统一领导海外统战工作，管理侨务行政事务，负责拟订侨务工作政策和规划的機構。

== 机构沿革 ==

1949年10月22日，[[中央人民政府委员会|中央人民政府委员会]]根据《[[中华人民共和国政府组织法|中华人民共和国政府组织法]]》第8条规定，设立'''[[中央人民政府华侨事务委员会|中央人民政府华侨事务委员会]]'''。由[[政务院|政务院]]直接领导，[[何香凝|何香凝]]为主任委员。1954年，改称中华人民共和国华侨事务委员会，简称'''中侨委'''。[[文化大革命|文化大革命]]期间的1969年，中侨委被撤消，侨务工作归属[[中华人民共和国外交部|外交部]]。设立的中侨委留守处，由外交部代管，直到1978年1月，成立'''國務院僑務辦公室'''<ref>{{cite journal|title=《“中侨委”初创新中国侨务》|author=麦尚文|url=http://qwgzyj.gqb.gov.cn/qwhg/139/1030.shtml|journal=侨委工作研究|publisher=国务院侨务办公室|accessdate=2018-09-30|issue=6|year=2007|location=北京|issn=1672-8831}}</ref>。2018年3月21日，中共中央印发了《深化党和国家机构改革方案》，中央统战部合併並保留国务院侨务办公室的牌子，不再保留单设的国务院侨务办公室。

== 职责权限 ==
=== 主要职能 ===
国务院侨务办公室的主要职能是：<ref>{{cite web|url=http://www.gqb.gov.cn/node2/node3/node22/node23//index.html|title=主要职能|accessdate=2017-09-01|publisher=国务院侨务办公室}}</ref>
# 统一领导海外统战工作
# 管理侨务行政事务
# 负责拟订侨务工作政策和规划
# 调查研究国内外侨情和侨务工作情况
# 统筹协调有关部门和社会团体涉侨工作
# 联系香港、澳门和海外有关社团及代表人士
# 指导推动涉侨宣传、文化交流和华文教育工作

国务院侨务办公室海外华人华侨社团联谊等职责划归[[中国侨联|中国侨联]]行使。
