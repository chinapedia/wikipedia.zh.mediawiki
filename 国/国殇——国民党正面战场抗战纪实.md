《'''国殇——国民党正面战场抗战纪实'''》，是一本於2005年在[[中国大陆|中国大陆]]上市出版的书籍，主要描写了从1931年[[九一八事變|九一八事變]]到1938年10月[[武汉会战|武汉会战]]期间，在[[蒋中正|蒋中正]]等[[中国国民党领袖|中国国民党领袖]]及将领的领导下，[[国民革命军|国民革命军]]将士在正面战场进行的大规模[[中国抗日战争|抗日作战]]。<ref>{{cite news |url = http://data.book.hexun.com/book-921.shtml |title =《国殇：国民党正面战场抗战纪实》目录 |publisher = 和讯网 |accessdate = 2013-06-06}}</ref>该书是自[[中华人民共和国|中华人民共和国]][[国共内战|成立之后]]，为数不多的在[[中国大陆|中国大陆]]出版的以描述[[中国国民党|中国国民党]]正面战场抗战为题材的书籍之一，还有评论认为“该书是第一本全面反映[[蒋介石|蒋介石]][[国民政府|政府]]抗战的专著”<ref name="tianwei">{{cite web|url=http://news.skykiwi.com/world/dl/sh/2005-06-17/10549.shtml|title=《国殇》上市, 正面描写国军抗日引起海内外关注|publisher=瞭望东方周刊|date=2005-06-17|accessdate=2013-09-12|deadurl=yes|archiveurl=https://web.archive.org/web/20130921055349/http://news.skykiwi.com/world/dl/sh/2005-06-17/10549.shtml|archivedate=2013-09-21}}</ref>。

==创作与出版==
由于意识形态等种种原因，中国大陆早先对于[[中国抗日战争|中国抗日战争]]一般较多宣传[[中国共产党|中国共产党]]所领导的游击战，而对于以蒋中正为首领的中国国民党及其武装力量—国民革命军所进行了大量抗日作战则一直采取淡化、回避的态度，导致不少大陆民众在这种片面宣传教育的影响下产生了所谓“国民党不抗日”的错误观点。但随着时间的推移，历史的真相逐步得到还原，由蒋中正和国民党领导的国民革命军所进行的正面战场抗日作战逐步得到重视<ref name="guoshangjieshao">{{cite web | url=http://history.people.com.cn/n/2013/0527/c205478-21626655.html | title=国殇：国民党正面战场抗战纪实 | publisher=人民网-文史频道 | date=2013年5月27日 | accessdate=2013-09-11}}</ref><ref name = "xuezhanhouji">{{cite book |title = 《血战—国民党军正面战场抗战纪实》后记 |publisher = 乌兰文艺出版社 |pages = 430—432 |author = 胡兆才 |year = 2010年9月 |isbn = 978-7-5098-0145-1}}</ref>。该书的作者张洪涛，是一名军旅作家，[[陕西|陕西]][[西安|西安]]人，[[国防大学|国防大学]][[硕士|硕士]]，著有多部战争类书籍<ref name="guoshangjieshao"/>。张洪涛表示，他撰写此书的原因之一正是因为30岁以下的年轻一代大陆民众对国军抗日的事迹所知不多<ref name="miandianlaobing">{{cite web | url=http://news.ifeng.com/history/minjianshuoshi/sunchunlong/detail_2010_09/10/2485342_0.shtml | title=缅甸国军老兵：不抗日 我那么多弟兄怎么死的？ | publisher=凤凰网历史 | date=2010年9月10日 | accessdate=2013-09-11}}</ref>。其实该书早在1994年曾出版过，当时的名字是《燃烧的太阳》，出版后加印过一次，之后没能再版，2005年此书以《国殇》为书名再次获得出版被认为和[[连战|连战]]、[[宋楚瑜|宋楚瑜]]相继访问大陆有关<ref name="xinlangwangdebaodao">{{cite web | url=http://news.sina.com.cn/c/2005-06-17/09346199309s.shtml | title=正面描写国民党抗日书籍上市 | publisher=生活报 | date=2005年6月17日 | accessdate=2013-09-11}}</ref>。

==主要内容==
该书主要记述了从1931年九一八事变至1938年10月武汉会战结束期间中国抗日战争中由蒋中正及[[国民政府|国民政府]]所领导的国民革命军进行的抗日战事，另外也穿插描写了日本政局的变故，[[国共合作|国共合作]]的波折，蒋中正对于抗日的看法和准备等等，并且对部分人物性格与言行方式等也进行了一定的刻画。
*书中主要叙述的抗日战争期间的重大战事和其他事件有：[[九一八事变|九一八事变]]、[[江桥抗战|江桥抗战]]、[[一二八淞沪抗战|一二八淞沪抗战]]、[[长城抗战|长城抗战]]、[[五一五事件|五一五兵变]]、[[五一五事件|二二六兵变]]、[[七七事变|七七事变]]、[[淞沪会战|淞沪会战]]、[[四行仓库保卫战|四行仓库保卫战]]、[[八一四空战|八一四空战]]、[[南京保卫战|南京保卫战]]、[[南京大屠杀|南京大屠杀]]、[[太原会战|太原会战]]、[[忻口会战|忻口会战]]、[[徐州会战|徐州会战]]、[[捕杀韩复榘|韩复榘被正法]]、[[台儿庄战役|台儿庄战役]]、[[兰封会战|兰封会战]]、[[花园口决堤事件|花园口决堤事件]]、[[武汉会战|武汉会战]]、[[万家岭战役|万家岭战役]]、[[中华民国空军|中华民国空军]]抗日作战、[[中华民国海军|中华民国海军]]抗日作战等。
*书中主要描写的中日双方政治军事人物有：[[蒋中正|蒋中正]]、[[李宗仁|李宗仁]]、[[张学良|张学良]]、[[薛岳|薛岳]]、[[冯玉祥|冯玉祥]]、[[佟麟阁|佟麟阁]]、[[张自忠|张自忠]]、[[汤恩伯|汤恩伯]]、[[庞炳勋|庞炳勋]]、[[王铭章|王铭章]]、[[孙连仲|孙连仲]]、[[白崇禧|白崇禧]]、[[阎锡山|阎锡山]]、[[郝梦龄|郝梦龄]]、[[卫立煌|卫立煌]]、[[程潜|程潜]]、[[张发奎|张发奎]]、[[胡宗南|胡宗南]]、[[裕仁天皇|裕仁天皇]]、[[近卫文磨|近卫文磨]]、[[松井石根|松井石根]]、[[土肥原贤二|土肥原贤二]]、[[冈村宁次|冈村宁次]]、[[東久邇宮稔彥王|東久邇宮稔彥王]]、[[板垣征四郎|板垣征四郎]]等。

==反响及后续==
[[File:国民革命军第十军在长沙城内与日军进行激烈巷战.jpg|210px]]《国殇》于2005年在中国大陆各地上市后，引发了媒体和社会关注，不少学者和专家认为，国军在抗日战争期间进行了大量抗日作战，仅[[湖南|湖南]]一地就有[[长沙会战|长沙会战]]等6次大规模的战役，国军抗战的功绩是不能抹杀的。由于此次出版恰逢时任中国国民党主席连战[[连战访问大陆|率团访问大陆]]和抗日战争胜利60周年，因此也被认为有更深层次的意义。此外，该书还被认为还原了过去被中共视为敌人的国军将领，肯定了其抗战的功绩，对这些将领进行了较过去相对客观的叙述。<ref name="xinlangguoshang">{{cite web| url=http://book.sina.com.cn/excerpt/sz/2005-06-17/1157185818.shtml| title=不该忘记的历史 国民党正面抗战纪实| publisher=中青在线| date=2005年6月17日| accessdate=2013-09-13| deadurl=yes| archiveurl=https://web.archive.org/web/20080422213318/http://book.sina.com.cn/excerpt/sz/2005-06-17/1157185818.shtml| archivedate=2008年4月22日}}</ref>

同年，时任中共中央总书记、国家主席、中央军委主席[[胡锦涛|胡锦涛]]在抗日战争胜利60周年纪念活动中发表讲话肯定了[[中国国民党|中国国民党]]在正面战场抗战的作用<ref name = "hujintaokending gmd">{{cite web |url = http://news.sina.com.cn/c/2005-09-07/04516878935s.shtml |title = 胡锦涛肯定国民党抗日正面战场贡献大 |publisher = 青年参考 |date = 2005年9月7日 |accessdate = 2013-09-04}}</ref>。之后有媒体报道《国殇》之续集，以描写抗战相持阶段为内容的第二部《浴血》和以描写抗战反攻阶段为内容的第三部《曙光》将在2015年抗战胜利70周年之前出版完成<ref name="guoshangxuji">{{cite web | url=http://vip.book.sina.com.cn/books/39071.html | title=国民党正面战场抗战纪实：国殇 | publisher=新浪读书 | accessdate=2013-09-13}}</ref>。
