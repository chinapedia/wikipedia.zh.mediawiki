'''国书'''是当某国向别国派遣[[大使|大使]]、[[公使|公使]]时，由派遣国[[国家元首|国家元首]]致接受国国家元首的正式[[文件|文件]]。即派遣国元首给接受國（地主国）元首的大使、公使介绍信或委任状。[[中文|中文]]当中，把召回大使、公使时，大使母国的元首致召回国元首的信件也称为国书。[[代办|代办]]使用的相同性质的文件一般称为介绍书。

==国书的签发及内容==
大使级及公使级外交代表的国书一般由国家元首签发，[[外交部长|外交部长]]副署；代办级外交代表的介绍书，由外交部长签发。以[[英王|英王]]为国家元首的[[英联邦国家|英联邦国家]]，国书由英王签发，派遣国政府首脑（閣揆）副署。国书的抬头一般写国家元首的称呼及姓名。2004年12月29日，[[加拿大政府|加拿大政府]]宣布，自2005年1月1日起，[[加拿大|加拿大]]派驻他国的使节所持国书由[[加拿大总督|加拿大总督]]签发，而外国使节呈递加拿大的国书的抬头亦改为总督[[阁下|阁下]]。

国书为一式两份。正本加封，副本不封口。各国的国书有各自较统一的格式，国与国之间的国书则格式有所不同。尽管格式有所差异，但各国国书记载事项均类似，包括：外交代表的任命及其等级，该使节代表本国元首或政府的言行，请驻在国国家元首予以信任等。以中華人民共和國為例，[[中华人民共和国主席|中华人民共和国主席]]致[[共和制|共和制]]国家[[总统|总统]]的国书格式如下<ref>李家发，《外交外事知识及国际交往礼仪》，广西师范大学出版社，2008年3月第一版，第79页~80页。</ref>：

== 呈递国书 ==
[[File:Mao_Zedong_with_Kavalam_Madhava_Panikkar.jpg|thumb]]向中国中央人民政府主席[[毛泽东|毛泽东]]呈递国书。]]
=== 呈递国书的意义 ===
外交代表到达驻在国[[首都|首都]]的第一件事就是呈递国书。在呈递国书之前，外交代表除了不能进行任何外交步骤、讲演、会谈及[[谈判|谈判]]等之外，礼节上拜会驻在国的高级官员及[[大使|外交使团]]成员也是不可以的。呈递和接受国书的法律意义在于：
* 国书接受国承认呈递国书者为驻本国外交代表，承认其所享有的各种外交权利、义务和[[外交豁免|豁免]]；
* 确定外交代表派遣国及驻在国之间的权利及义务；
* 确定外交代表正式就任日期，并根据国书呈递日期确定外交团中同级外交官之间的位次{{NoteTag|大使、公使因本国政体变更或君主离世需要重新呈递国书的，其在外交团中的位次不变，仍按原呈递时间排定位次。}}。

因为由呈递国书先后次序来决定优先位次，所以各国外交代表都会尽快与驻在国协商呈递事宜。[[法国|法国]][[路易十四|路易十四]]时代，有一次[[意大利|意大利]]和[[普鲁士|普鲁士]]两国公使同时到[[巴黎|巴黎]]上任，就谁先向[[法国国王|法国国王]]呈递国书的问题发生争执，法国[[宫廷|宫廷]]的意见为，两国公使同一天觐见国王，先到[[凡尔赛宫|凡尔赛宫]]者先呈递。于是普鲁士公使半夜就到凡尔赛宫等候天明立即呈递国书。意大利公使到达凡尔赛宫时见普鲁士公使正在假寝，就从侧门进入。普鲁士公使随即赶紧跟进。等普鲁士公使到达路易十四的寝宫时，意大利公使已经读完了颂词，正准备向国王呈递国书，普鲁士公使见状不妙，急忙将国书呈放在法国国王的御床上<ref>李家发，《外交外事知识及国际交往礼仪》，广西师范大学出版社，2008年3月第一版，第177页。</ref>。

通常外交代表抵达驻在国首都后，即可将国书副本及颂词副本递交给驻在国[[外交部|外交部]]，表明其已抵达驻在国首都。同时与外交部的相关部门（如[[礼宾司|礼宾司]]、交际司）协商呈递国书正本事宜及安排。通常认为，外交代表驻在国尽快安排国书呈递事宜是重视两国关系的表现，如无特殊原因迟迟不安排国书呈递，外交代表则会被认为受到冷遇，两国关系冷淡。特殊原因一般为，国家元首不在首都或休假；部分[[穆斯林国家|穆斯林国家]]的[[斋月|斋月]]等。也有国家习惯一次集中接受几位大使的国书。

1923年，[[直系|直系]][[民国军阀|军阀]][[曹锟|曹锟]]将[[中华民国大总统|中华民国大总统]][[黎元洪|黎元洪]]赶下台，总统职权由摄政内阁代行。此时，[[日本|日本]]新派驻中国的公使[[芳泽谦吉|芳泽谦吉]]抵达[[北京|北京]]，他认为[[日本天皇|日本天皇]]写给[[中华民国总统|中华民国总统]]的国书不应呈递给摄政内阁，就以中国现今没有总统，无法递交国书为由，没有按惯例安排礼节性拜访外交部，协商呈递国书事宜。随即就以日本驻华公使身份拜会各国驻华公使及一些内阁总长。时任外交总长的[[顾维钧|顾维钧]]表示，芳泽谦吉未拜访外交部，外交部不知他来华身份，亦不承认其具有任何身份，不与其交往。只有当他向外交总长递交国书副本后，外交总长才与其打交道。后来，芳泽谦吉不得不照办<ref>中国社会科学院近代史研究所编，《顾维钧回忆录》（1），中华书局1983年版，第323-326页。</ref>。

1999年，中華民國與[[马其顿共和国|馬其頓]]建交，但主導建交的[[马其顿共和国总理|马其顿共和国总理]]與反對建交的[[马其顿共和国总统|马其顿共和国总统]]意見相左，以致馬其頓總統一直不接受中華民國駐馬其頓大使的到任國書，直至2001年馬國與中華民國斷交。

根据1961年《[[维也纳外交关系公约|维也纳外交关系公约]]》第十三条的规定，外交代表呈递国书确定正式就任日期，有两种方式：一是向驻在国国家元首正式呈递国书时算起；二是向驻在国外交部递交国书正式副本时算起。采取哪一种方式，由驻在国自定，但必须一致对待。一些国家规定，大使向外交部递交国书副本后，即可以开始其外交工作，但到任日期仍以向驻在国国家元首呈递国书正本之日算起。

=== 呈递国书的方式及礼仪 ===
国书正本必须由大使、公使亲自向驻在国元首呈递。在以[[英国君主|英国君主]]（国王或女王）为元首[[英联邦|英联邦]]国家中，国书由大使亲自呈递给驻在国总督，再由总督转呈给英国君主。由于[[安道尔公国|安道尔公国]]国家元首为两位，一位为[[西班牙|西班牙]][[乌赫尔主教|乌赫尔主教]]，一位为[[法国总统|法国总统]]（之前先后为弗阿伯爵、法国君主），所以使节到达安道尔履任的时候，必须分别前往西班牙乌赫尔市的[[主教座堂|主教座堂]]及法国巴黎呈递国书。而呈递安道尔的国书，抬头则为“安道尔两大公阁下”。

一般情况下，国书只在使节到任时呈递一次即可，但下列情况则需重新递交国书：
* 外交代表或外交代表机构等级发生变化。如公使升为大使，公使馆升为[[大使馆|大使馆]]。
* 派遣国国家制度发生根本变化。如由[[君主制|君主制]]变为共和制。
* 驻在国国家制度发生根本变化。
* 任命该使节的君主死亡或退位。
* 驻在国的君主死亡或退位。

以上需要重新呈递国书的情况多与国家元首为君主有关。共和制国家元首发生变更是否需要重新呈递国书，则有不同看法。生于德国的英籍[[国际法|国际法]]学者奥本海认为，共和制国家元首有派遣和接受使节的权力，因此其死亡、辞职、下台及任期届满卸任，其派遣及接受使节的权力也随之终止，所以除了像[[瑞士联邦|瑞士联邦]]这样以[[瑞士联邦委员会|瑞士联邦委员会]]构成的集体元首的国家，都应该呈递新的国书；而英国人萨道义的观点相反，认为共和制国家元首变更不影响外交使节的使命，无需重新呈递国书。1928年[[美洲|美洲]]国家签订的《[[哈瓦那外交官公约|哈瓦那外交官公约]]》中的相关规定，既与后一种观点一致。[[中华人民共和国主席|中华人民共和国主席]]（国家主席）[[中华人民共和国主席存废之争|被废除]]时，不要求别国使节重新呈递国书。

大使递交国书的[[礼仪|礼仪]]规定较严格，一般有成文的规定，同时也较隆重。呈递国书时，使馆全体外交官要随同大使前往，但使节夫人一般不出席仪式。通常的程序为：
# 驻在国礼宾司司长和元首侍从武官以国家元首派遣的名义，乘悬挂驻在国[[国旗|国旗]]或两国国旗的礼车前往大使馆迎接大使。在[[英国|英国]]、[[荷兰|荷兰]]、[[西班牙|西班牙]]等部分以君主为国家元首的国家，礼车为皇家高级马车。
# 大使乘礼车抵达元首府邸、宫殿或其举行仪式的场所{{NoteTag|例如，[[图瓦卢|图瓦卢]]接受国书呈递的仪式是在一家[[旅馆|旅馆]]公开举行，而不是[[总督府|总督府]]。}}时，武装士兵或仪仗队应向大使致敬。有时也会奏[[国歌|国歌]]，检阅[[仪仗队|仪仗队]]。通常是外交使节抵达时奏使节派遣国国歌，离开时奏驻在国国歌。
# 大使正式向驻在国国家元首呈递国书。在很多国家，外交代表同时要致颂词，国家元首要致答词。通常在此之前，颂词原文会随国书副本一同递交给驻在国外交部长。颂词的内容一般为大使转达本国元首的问候，向驻在国人民及政府致意，恰如其分的评价和赞扬两国关系及驻在国在国际舞台上的作用，表达使节本人愿为两国关系做出努力的意愿等。过去，大使呈递国书要口头致颂词。现在由于外交礼仪的日趋简化，一些国家在呈递国书的时候，以互换讲话稿代替口头致颂词。
# 大使应邀进入客厅会谈。内容纯属礼仪性的，一般是共叙两国友好关系，互致问候，谈些[[天气|天气]]等无关紧要的话题。此时谈及任何实质性特别是不愉快的问题，均属失礼。

有些国家在仪式结束后还设小型酒会欢迎使节及参礼人员。

== 特殊身份 ==
有些國家接受國書的不一定是[[最高領導人|最高領導人]]，而是代表國家負責禮儀事務的[[虚位元首|名義國家元首]]。

2010年3月8日[[中国驻朝鲜/韩国大使列表|中國駐朝鲜大使]][[劉洪才|劉洪才]]上任時，負責接受國書的是[[最高人民会议常任委员会委员长|最高人民会议常任委员会委员长]][[金永南|金永南]]，不是作为[[朝鲜民主主义人民共和国领导人列表|朝鲜最高领导人]]的[[朝鲜劳动党|朝鲜劳动党]]总书记[[金正日|金正日]]<ref>[http://www.fmprc.gov.cn/mfa_chn/zwbd_602255/wshd_602258/t662587.shtml 驻朝鲜大使刘洪才向金永南委员长递交国书]</ref>。[[中华人民共和国|中华人民共和国]]在1975年至1982年期间[[中华人民共和国主席存废之争|没有设立国家主席]]，也是由[[全国人民代表大会常务委员会委员长|全国人民代表大会常务委员会委员长]]代表[[全国人民代表大会常务委员会|全国人大常委会]]行使[[中华人民共和国国家元首列表|国家元首]]职权接受国书，不是作为实际[[中華人民共和國最高領導人|最高领导人]]的[[中国共产党中央委员会主席|中国共产党中央委员会主席]]。
