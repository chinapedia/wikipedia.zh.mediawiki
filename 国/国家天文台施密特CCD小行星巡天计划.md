'''国家天文台施密特CCD小行星巡天计划'''（英文：BAO Schmidt Asteroid Program，或简称'''SCAP'''<ref name="SCAP">SCAP 為 BAO Schmidt CCD Asteroid Program 或 Beijing Schmidt CCD Asteroid Program 的簡稱，即国家天文台施密特CCD小行星巡天计划。{{Cite web|url=http://batc.bao.ac.cn/work/work10.htm#二. BATC小行星巡天的概况|title=二. BATC小行星巡天的概况|accessdate=2015-07-07|work=大视场CCD小行星巡天及物理研究|publisher=BATC课题组}}</ref>），是由[[中国科学院|中国科学院]]出资，于1995年5月在[[北京天文台|北京天文台]]设立的[[近地小行星|近地小行星（NEAs）]]与[[彗星|彗星]]观测计划。

==仪器==
SCAP配备有一台60/80厘米的[[施密特摄星镜|施密特摄星镜]]。该仪器装备一组2048x2048[[像素|像素]]的[[电荷耦合元件|CCD]][[照相机|照相机]]，现被安装在位于[[河北|河北]][[承德|承德]]的[[国家天文台兴隆观测基地|国家天文台兴隆观测基地]]。

== 科学发现 ==
在1995到1999年间，SCAP共发现了1颗彗星，2460颗小行星并对43860颗已知小行星实施过观测任务，是当时世界第五大小行星观测计划。在它的发现中有5颗近地小行星（NEAs），其中2颗被认定为有潜在危险的小行星（PHAs）。它还在2002年时于[[月球|月球]]附近发现了一颗近地小行星。

由于一些原因，[[小行星中心|小行星中心]]将约半数SCAP的发现记名为“BAO Schmidt”，余下的则为“Beijing Schmidt”。但两者应指同一机构。

== 近况 ==
在与[[Space.com|Space.com]][[撰稿人|撰稿人]]Michael Paine的谈话中，SCAP主任朱进透露为了给其他观测让路，本计划可支配的观测时间已经明显少于以往。
