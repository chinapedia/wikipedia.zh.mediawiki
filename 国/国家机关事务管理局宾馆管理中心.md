'''国家机关事务管理局宾馆管理中心'''，是[[国家机关事务管理局|国家机关事务管理局]]直属[[事业单位|事业单位]]。

==沿革==
1998年，'''国务院机关事务管理局宾馆管理中心'''成立，是经[[中央机构编制委员会办公室|中央机构编制委员会办公室]]批准设立的[[国务院机关事务管理局|国务院机关事务管理局]]直属正司级事业单位，承担政治接待、服务保障、宾馆管理等职能，具备独立法人资格。截至2009年，该中心下属6家宾馆：五星级的[[首都宾馆|首都宾馆]]、[[北京友谊宾馆|友谊宾馆]]，四星级的[[国谊宾馆|国谊宾馆]]、[[国二招宾馆|国二招宾馆]]，以及[[达园宾馆|达园宾馆]]、[[紫金宾馆|紫金宾馆]]，总资产17亿元人民币，职工5000多人<ref>{{Cite web|url =http://www.ggj.gov.cn/gzdt/200912/t20091208_269173.htm|title =国务院机关事务管理局宾馆管理中心简介|author =|date =2009-12-08|publisher =国务院机关事务管理局|language =|accessdate =|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20170328195753/http://www.ggj.gov.cn/gzdt/200912/t20091208_269173.htm|archivedate =2017-03-28}}</ref>。随着2013年国务院机关事务管理局更名为国家机关事务管理局，该中心也更名为'''国家机关事务管理局宾馆管理中心'''<ref>{{Cite web |url = http://www.gjsy.gov.cn/cxzl/ |title =  	国家机关事务管理局宾馆管理中心|author =  |date =  |publisher =事业单位在线 |language =  |accessdate =2017-03-28  |quote =  }}</ref>。

2018年，达园宾馆、紫金宾馆改为达园服务中心、紫金服务中心，与宾馆管理中心仍同为事业单位；首都宾馆、友谊宾馆、国谊宾馆、国二招宾馆为企业<ref>{{Cite web |url = http://www.sohu.com/a/135031692_543956 |title = 国家机关事务管理局公开招聘工作人员 |author =  |date =2017-04-19  |publisher = 搜狐 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url =http://www.mohrss.gov.cn/SYrlzyhshbzb/fwyd/SYkaoshizhaopin/zyhgjjgsydwgkzp/zpgg/201803/t20180328_290922.html  |title = 国家机关事务管理局宾馆管理中心及所属企事业单位2018年度公开招聘公告 |author =  |date = 2018-03-28  |publisher =中华人民共和国人力资源和社会保障部  |language =  |accessdate =  |quote =  }}</ref>。
