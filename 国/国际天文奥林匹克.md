'''国际天文奥林匹克'''（International Astronomy Olympiad，IAO）是一个为高中生(14-18岁)组织的国际性年度[[天文|天文]]科教活动，包含一个在这些学生间的智力竞赛，它属于[[国际科学奥林匹克|国际科学奥林匹克]]的一员。

== 日程 ==
一般来说，IAO的日程长9天，分别为：<ref>{{Cite web|url=http://issp.ac.ru/iao|title=The International Astronomy Olympiad: main page|accessdate=2018-08-01|work=issp.ac.ru}}</ref>

* 第一天：参赛队伍抵达比赛地，入住酒店。
* 第二天：开幕式，选手破冰。
* 第三天：理论考试。
* 第四天：观测考试（如天气晴好），领队翻译选手的答卷。
* 第五天：外出观光，观测考试（如前一天阴天），领队开始改卷。
* 第六天：实测考试，领队改卷。
* 第七天：所有领队及学生外出放松。
* 第八天：闭幕式，颁奖。
* 第九天：参赛队伍离开。

== 试题 ==
IAO的试题分为三部分，分别为：

# 理论：该部分通常包含5道解答题，其中会有一道需要选手画图的题目，时长4个小时。该部分的题目会由领队在考试当天的凌晨翻译好，学生的答案也将由领队翻译为英语。
# 观测：该部分题型不定，通常为2题在10~30分钟内完成，至少1题需要使用[[望远镜|望远镜]]。该部分的题目会由领队事先翻译好，而答案则不会经过翻译，因此必须用英语答题。
# 实测：该部分通常包含2道数据分析与处理的题目，时间4小时，题目会经过翻译，而答案则不会经过翻译。

理论试题将由所有参赛国家的领队共同裁判，而观测和实测试题则将由本地组委会裁判。在最后一晚，领队将开会决定金银铜奖的分数线，进而决定奖牌的归属。

在以下方面获得第一的选手将额外获得特别奖：

* 总分（BR）
* 理论分数（BT）
* 观测分数（BO）
* 实测分数（BP）
* 画图（BD/BB）
* 总分最高的第一次参与国家学生（BN）
