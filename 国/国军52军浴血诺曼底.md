'''“国军52军浴血诺曼底”'''是一则[[网络谣言|网络谣言]]（或“[[钓鱼文|钓鱼文]]”）中提及的所谓“[[历史|历史]]事件”。内容主要是[[国民革命军第五十二军|国民革命军第五十二军]]在[[诺曼底战役|诺曼底战役]]中浴血奋战，令世界刮目相看，最终使[[中国|中国]]获得[[联合国安理会常任理事国|联合国“五常”席位]]。

== 起源 ==
据一名和此文作者过往甚密的网友说，此文撰写的初衷是用来调侃和讽刺一些“[[国粉|国粉]]”（意即[[中华民国|中华民国]]的粉丝）。原作者认为，国军在[[抗战|抗战]]中的积极一面不可否认，但是近年来[[国军|国军]]在抗战战场上的表现已被部分“國粉”拔高，甚至已经到了神话的地步。在这样的背景下，就有了“国军52军浴血诺曼底”一文<ref name="guancha">{{cite web | url=http://www.guancha.cn/society/2014_04_20_223534.shtml | title=钓鱼文“国军血战诺曼底”登上湖北高考模拟题 网友：侮辱考试严肃性 | publisher=观察者网}}</ref>。

== 主要内容 ==
文章先以一大段历史背景（[[斯大林格勒战役|斯大林格勒战役]]、盟军商议开辟第二战场、反法西斯战争迎来胜利曙光）开篇，之后谈到时任[[美国总统|美国总统]][[富兰克林·罗斯福|富兰克林·罗斯福]]为了让中国成为联合国安理会常任理事国，建议国军派兵参加诺曼底战役，[[蒋中正|蒋中正]]遂决定派国军第52军参战。经过严苛训练，52军在诺曼底浴血奋战，立下奇功。而文章末尾称，此事被[[美国|美国]]及[[中共|中共]]封杀。

== 漏洞 ==
该则谣言包含许多漏洞，以下为部分纰漏：
* 52军自1941年到抗战结束，一直驻守在[[云南|云南]]<ref name="52jun">{{cite book |url=http://www.google.com.hk/books?id=o1uEAAAAIAAJ&lr=&hl=zh-CN | title=国民革命军陆军第五十二军军史 | publisher=国防部史政编译室 | author=孙建中 | year=2003 | location=台北}}</ref>；
* 文中称52军的体能训练“要求所有人的万米成绩必须达到十八分钟”，而截至2014年4月20日，[[田徑紀錄列表|男子万米世界纪录]]也只是26分17秒53（由[[埃塞俄比亚|埃塞俄比亚]]运动员凯内尼萨·贝克勒于2005年创造）<ref>{{cite web | url=http://www.iaaf.org/records/toplists/middlelong/10000-metres/outdoor/men/senior | title=男子10000米世界纪录 | publisher=国际田联 | language=en}}</ref>，士兵不可能跑得那么快；
* 当时担任52军军长的是[[赵公武|赵公武]]，而不是文中提及的所谓“Shir Wong”<ref name="52jun"/>。

此外，文章中“Chung-Go Sun”托起炸药包高喊“为了[[中华民国|中华民国]]，前进 ”、炸毁[[德军|德军]]碉堡的情节则是戏仿了[[中国大陆|中国大陆]]小学课本中《[[董存瑞|董存瑞]]舍身炸[[碉堡|碉堡]]》的情节。

== 影响 ==
尽管漏洞百出，不少人仍信以为真。甚至在2014年[[湖北|湖北]]七市（州）高三毕业班联考[[历史|历史]]考试的试题中都出现了此内容，被指侮辱了考试的严肃性和科学性<ref name="guancha"/>。

而中国军方报纸《[[解放军报|解放军报]]》的官方微博“军报记者”更将该文称作“网络造史”的一部分<ref>{{cite web | url=http://weibo.com/2280198017/BfkM7kywb | title=网上读史：你有一双慧眼吗 | publisher=军报记者的微博 | accessdate=2014-07-27}}</ref>，被网友戏称“钓上大鱼”。
