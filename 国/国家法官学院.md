'''国家法官学院'''（最高人民法院法官国际交流中心、最高人民法院司法案例研究院），位于[[北京市|北京市]][[丰台区|丰台区]][[北京四环路|南四环西路]]111号，是[[中华人民共和国最高人民法院|中华人民共和国最高人民法院]]直属[[事业单位|事业单位]]，是中国[[法官|法官]]教育培训的主要机构<ref>{{Cite web |url = http://www.gjsy.gov.cn/cxzl/ |title = 国家法官学院、最高人民法院法官国际交流中心、最高人民法院司法案例研究院 |author =  |date =  |publisher =  事业单位在线|language =  |accessdate =2017-04-05  |quote =  }}</ref><ref name=jj/>。

== 沿革 ==
1985年，创建'''全国法院干部业余法律大学'''。1988年，升格为'''中国高级法官培训中心'''。1997年，正式成立'''国家法官学院'''。国家法官学院原址在北京市[[通州区|通州区]]天成桥甲1号，占地面积75亩，可同期培训300人。2015年9月15日，最高人民法院院长[[周强|周强]]出席了国家法官学院秋季开学典礼并为新校区建设落成揭牌。国家法官学院位于北京市丰台区南四环西路111号的新校区正式启用。新校区占地面积248亩，建筑面积将近6万平方米，可同期培训750人<ref name=jj>{{Cite web|url = http://njc.chinacourt.org/article/detail/2010/05/id/1472172.shtml|title = 国家法官学院简介|author = |date = 2016-09-06|publisher = 国家法官学院|language = |accessdate = |quote = |archive-url = https://web.archive.org/web/20170406021732/http://njc.chinacourt.org/article/detail/2010/05/id/1472172.shtml|archive-date = 2017-04-06|dead-url = yes}}</ref>。

截至2015年底，国家法官学院在全国设立北京、上海、江苏、海南、青岛海事等25个分院，另在筹备8个分院。国家法官学院分院同时也是各[[高级人民法院|高级人民法院]]的法官教育培训机构，承担辖区内法官培训任务。经最高人民法院批准，截至2015年底，先后有[[北京市第一中级人民法院|北京市第一中级人民法院]]、[[天津市第一中级人民法院|天津市第一中级人民法院]]等77个法院被确定为全国法院法官培训现场教学基地<ref name=jj/>。

国家法官学院是最高人民法院对外交流窗口，兼有“最高人民法院法官国际交流中心”职能<ref name=jj/>。

2016年9月30日，最高人民法院举办最高人民法院司法案例研究院揭牌暨“中国司法案例网”开通活动。最高人民法院院长周强出席并讲话<ref>{{Cite web |url =http://mt.sohu.com/20161001/n469468654.shtml  |title = 最高人民法院司法案例研究院揭牌 |author =  |date = 2016-10-01 |publisher = 搜狐 |language =  |accessdate =  |quote =  }}</ref>。
