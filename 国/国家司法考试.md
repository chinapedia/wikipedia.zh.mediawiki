[[中华人民共和国|中华人民共和国]]的'''国家司法考试'''是依据《中华人民共和国法官法》、《中华人民共和国检察官法》、《中华人民共和国律师法》、《中华人民共和国公证法》及《国家司法考试实施办法》的有关规定並由[[中华人民共和国司法部|司法部]]與各地司法厅（局）统一组织的从事特定法律职业的资格考试，考试合格授予《[[法律职业资格证书|法律职业资格证书]]》。

国家司法考试始于2001年，在此之前僅有律师资格考试。根据现行的[[制度|制度]]，除非在司法考试制度建立前就已经取得执业资格，否则要担任[[中華人民共和國律师|律师]]、[[法官|法官]]、[[检察官|检察官]]和[[公证员|公证员]]，就必须通过统一司法考试。<ref name=rule>{{Cite web |url= http://www.moj.gov.cn/sfks/content/2009-05/31/content_1097978.htm?node=8014 |title= 《国家司法考试实施办法》 |author= 司法部国家司法考试司，司法部国家司法考试中心 |date= 2008年8月15日 |publisher= [[中国普法网|中国普法网]] |language= zh-hans |accessdate= 2012年12月8日 |quote= 初任法官、初任检察官，申请律师执业和担任公证员必须通过国家司法考试，取得法律职业资格。法律、行政法规另有规定的除外。 }}</ref>

根據[[新華社|新華社]]報導，2017年8月28日，包括法官法等在内的８部法律修正案草案提请十二届全国人大常委会第二十九次会议审议。这８部法律修正案草案所涉及的司法考试改革將停辦国家司法考试，改為[[国家统一法律职业资格考試|国家统一法律职业资格考試]]，律师法修正案草案明确规定，舊有的国家统一司法考试合格证书、律师资格凭证，与国家统一法律职业资格证书具有同等效力。<ref>{{cite web|url=http://news.xinhuanet.com/legal/2017-08/28/c_1121558064.htm|title=司法考试改革将引发哪些法律职业资格变化？|publisher=新华社|location=北京|author1=丁小溪|author2=杰文津|author3=王茜|date=2017-08-28}}</ref>


==有关机构==
每年度的司法考试工作主要由[[司法部国家司法考试司|司法部国家司法考试司]]与[[国家司法考试中心|国家司法考试中心]]负责。

===司法部国家司法考试司===
[[司法部国家司法考试司|司法部国家司法考试司]]由考试指导处、综合处、教育处三个部门组成<ref>{{Cite web |url = http://www.moj.gov.cn/sfks/jg/sfkssjgsz.htm |title = 国家司法考试司机构设置  |author = 司法部国家司法考试司、司法部国家司法考试中心  |date=  |publisher= 中国普法网 |language= zh-hans |accessdate= 2012年12月9日 |quote=  }}</ref>，主要负责<ref>{{Cite web |url= http://www.moj.gov.cn/sfks/jg/sfksszn.htm |title= 国家司法考试司职能  |author= 司法部国家司法考试司、司法部国家司法考试中心  |date=  |publisher= 中国普法网 |language= zh-hans |accessdate= 2012年12月9日 |quote=  }}</ref>：
* 研究制定司法考试的工作规则和规范，制定国家司法考试的工作规章制度； 
* 负责司法考试大纲编审； 
* 管理指导司法考试考务工作，对考试工作进行监督； 
* 指导开展司法考试的理论研究和国际交流； 
* 审查授予《[[法律职业资格证书|法律职业资格证书]]》，对取得资格证书者进行管理； 
* 开展国家司法考试工作的信息化建设，指导地方司法行政机关组织国家司法考试工作。

===国家司法考试中心===
[[国家司法考试中心|国家司法考试中心]]有由办公室、命题处、考务处、信息统计处和综合处组成<ref>{{Cite web |url= http://www.moj.gov.cn/sfks/jg/sfkszxjgsz.htm |title = 国家司法考试中心机构设置  |author= 司法部国家司法考试司、司法部国家司法考试中心  |date =  |publisher = 中国普法网 |language= zh-hans |accessdate = 2012年12月9日 |quote=  }}</ref>，主要负责<ref>{{Cite web |url= http://www.moj.gov.cn/sfks/jg/sfkszxzn.htm |title= 国家司法考试中心职能   |author= 司法部国家司法考试司、司法部国家司法考试中心  |date=  |publisher= 中国普法网 |language= zh-hans |accessdate= 2012年12月9日 |quote=  }}</ref>：
* 参与研究、草拟司法考试的工作计划和年度工作方案与国家司法考试工作规章、规范；
* 参与、协助司法考试大纲编审工作；
* 承担年度报名工作，汇总、分析全国报名数据和信息； 
* 指导地方考区设置，汇总考点、考场设置； 
* 建设司法考试题库； 
* 制定命题方案，承担试卷的组拼、发送工作； 
* 组织进行试卷的评判、成绩登录、分数复核和分数统计、分析工作； 
* 承担司法考试考务工作的信息化建设；

==历史==
*1994年，律师资格考试首次向[[港澳台|港澳台]]地区开放，三地共有359人参加，台湾有三人通过考试。不久律考大门因为“配套设施没跟上”而关闭。<ref name=taiwan>{{Cite web |url= http://infzm.com/content/18041 |title= 14年后，大陆对台重开司法考试大门 |author= 赵蕾，江飒 |date= 2008年10月2日 |publisher= 《[[南方周末|南方周末]]》 |language= zh-hans |accessdate= 2012年12月9日更新 |quote=  }}</ref>
*2001年7月15日，[[中华人民共和国最高人民法院|最高法]]、[[中华人民共和国最高人民检察院|最高检]]、[[中华人民共和国司法部|司法部]]就国家对初任法官、检察官和取得律师资格实行统一司法考试制度有关问题联合发表公告。<ref name=timeline>{{Cite web |url= http://www.moj.gov.cn/sfks/content/2009-05/31/content_1097884.htm?node=8019 |title= 2001年——2002年司法考试大事记 |author= [[中华人民共和国司法部|中华人民共和国司法部]]国家司法考试司，司法部国家司法考试中心 |date= 2009年5月31日 |publisher= [[中国普法网|中国普法网]] |language= zh-hans |accessdate= 2012年12月8日 |quote=  }}</ref>
*2001年10月30日，《国家司法考试实施办法（试行）》正式发布，自2002年1月1日起施行。<ref name=timeline/>
*2001年12月30日，司法部发布公告宣布首次国家司法考试定于2002年3月30、31日举行。<ref name=timeline/>
*2002年3月30日、31日两日，全国司法考试统一举行，共有36万余人参加考试。<ref name=timeline/>
*2005年9月17日，2005年国家司法考试顺利开考，港澳首设国家司法考试考场。<ref name=timeline2>{{Cite web |url= http://www.moj.gov.cn/sfks/content/2009-05/31/content_1097886.htm?node=8019 |title= 2005年司法考试大事记 |author= [[中华人民共和国司法部|中华人民共和国司法部]]国家司法考试司，司法部国家司法考试中心 |date= 2009年5月31日 |publisher= [[中国普法网|中国普法网]] |language= zh-hans |accessdate= 2012年12月8日 |quote=  }}</ref>

*2008年，2008年国家司法考试报名人数突破37万人，实际参考人数近32万人。当年，中国国家司法统一考试向[[台湾|台湾]]居民再次开放，据官方数据，当年有818名台湾人获准参加考试。<ref name=taiwan/>
*2018年，新華社報導司法考试改革相關訊息，表示国家司法考试將停辦，改為国家统一法律职业资格考試。

==考试资格==
根据2008年颁布的《国家司法考试实施办法》，符合以下条件的人员，可以报名参加国家司法考试：<ref name=rule/>
# 具有[[中华人民共和国国籍|中华人民共和国国籍]]；
# 享有选举权和被选举权；
# 具有完全民事行为能力；
# 高等院校法律专业本科毕业或者高等院校非法律专业本科毕业或应届毕业生，部分地区可以放宽至高等院校法律专业专科毕业；<ref>{{Cite web |url= http://www.moj.gov.cn/sfks/content/2009-05/31/content_1097976.htm?node=8014 |title= 司法部关于确定国家司法考试放宽报名学历条件地方的意见 |author= 司法部国家司法考试司，司法部国家司法考试中心 |date= 2009年5月31日 |publisher= 中国普法网 |language= zh-hans |accessdate= 2012年12月8日 |quote= }}</ref>
# 品行良好。
但是有下列情形之一的人员，不能报名参加国家司法考试，已经办理报名手续的，报名无效：<ref name=rule/>
# 因故意犯罪受过刑事处罚的；
# 曾被国家机关开除公职或者曾被吊销律师执业证、公证员执业证的；
# 被处以二年内不得报名参加国家司法考试期限未满或者被处以终身不得报名参加国家司法考试的。

==考试时间==
国家司法考试每年举办一次，2002年第一届司法考试因为[[全国初任法官考试|全国初任法官考试]]、[[初任检察官考试|初任检察官考试]]和律师资格考试停考和首次考试过渡性的原因确定在3月举行，其余均选取每年九月的第三个星期六、星期日作为考试时间。<ref name=whytime>{{Cite web |url= http://www.moj.gov.cn/sfks/content/2009-05/31/content_1098036.htm?node=8010 |title= 今年国家司法考试的时间是如何确定的 |author= 司法部国家司法考试司，司法部国家司法考试中心 |date= 2009年5月31日 |publisher= 中国普法网 |language= zh-hans |accessdate= 2012年12月8日 |quote=  }}</ref>

出于保证大学应届生参加考试的原因，故将每年考试的报名时间在七月。因为考试的的准备工作环节，如试卷运送、保管等工作可以避开“十一”假期，所以考试通常在每年九月举行（2008年受到[[汶川大地震|汶川大地震]]影响，[[四川省|四川省]]部分地区的司法考试单独推迟考试）。<ref name=whytime>{{Cite web |url= http://www.moj.gov.cn/sfks/content/2009-05/31/content_1098036.htm?node=8010 |title= 今年国家司法考试的时间是如何确定的 |author= 司法部国家司法考试司，司法部国家司法考试中心 |date= 2009年5月31日 |publisher= 中国普法网 |language= zh-hans |accessdate= 2012年12月8日 |quote=  }}</ref>

==通过率==
* 以前的律师资格考试，主要考察法条，相对近年的全部为案例式的题目较为容易。但是通过率仍然不高。
* 在2006-2010这些年间，为了配合经济形势，通过率特殊地高（俗称“放水年”<ref>http://blog.sina.com.cn/s/blog_49a9538c0101mcbd.html</ref><ref>http://edu.takungpao.com.hk/zcpx/q/2013/0527/1636947.html{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>）。其中2008-2009的通过率逼近30%。实际上，考试并非按照考生的绝对分数给予总分的。例如，在低通过率的年份考获低分的，在高通过率年度就相当于高分通过。<ref>{{cite web |url=http://edu.gmw.cn/2013-05/17/content_7666929.htm |title=存档副本 |accessdate=2013-11-27 |deadurl=yes |archiveurl=https://web.archive.org/web/20131203082347/http://edu.gmw.cn/2013-05/17/content_7666929.htm |archivedate=2013-12-03 }}</ref><ref>http://blog.sina.com.cn/s/blog_49a9538c0101mcbd.html</ref>
* 2011年开始通过率急剧下降（从20%降了一半到10%左右），尽管当年（2011年）的考试题目较前明显容易。
* 2012-2014年期间，考试难度突然提高，但是通过率反而再次下降到约7-8%以下。因此，这几年最难通过。
* 2015年，普遍发现卷四的成绩特高，许多卷二、三的成绩不及格的，卷四分数几乎达120-130，导致民法刑法不合格的却可以高分通过的奇特现象。官方一直不会正式公布通过率，但是，依据卷四的分数，2015年是又开始“放水”的年度。（备注：卷四实践发现是用来调节通过率的）<ref>{{cite web |url=http://sifakaoshi.abang.com/od/sifakaoshigxm.--3b/a/sifakaoshigxm.--B3.htm |title=存档副本 |accessdate=2015-11-21 |deadurl=yes |archiveurl=https://web.archive.org/web/20151121194941/http://sifakaoshi.abang.com/od/sifakaoshigxm.--3b/a/sifakaoshigxm.--B3.htm |archivedate=2015-11-21 }}</ref>

港澳地区的通过率是1-2%（该数据包括了刚来自大陆地区的移民考生，真正香港澳門本地受教育的考生通过司法考试者非常少）。<ref>http://emuch.net/fanwen/329/46861.html</ref>相對而言，台湾地区的考生的通过率比港澳高得多。

==成绩查询==
司法考试的成绩通常于11月中下旬公布，理论上每年司法考试的合格分数线需要与最高人民法院与最高人民检察院商议。<ref>{{Cite web |url= http://www.moj.gov.cn/sfks/jg/sfkss.htm |title= 国家司法考试 |author= 司法部国家司法考试司、司法部国家司法考试中心 |date=  |publisher= 中国普法网 |language= zh-hans |accessdate= 2012年12月9日 |quote=  }}</ref>但每年实际的合格分数线均为360分，放宽报名学历条件地方合格分数线为315分。<ref>{{Cite web |url= http://www.moj.gov.cn/sfks/content/2009-05/31/content_1098024.htm?node=8011 |title= 关于2008年国家司法考试成绩公布、合格分数线及申请授予法律职业资格等事项的公告 |author= 中华人民共和国司法部国家司法考试办公室 |date= 2008年11月20日 |publisher= 中国普法网 |language= zh-hans |accessdate= 2012年12月9日 |quote=  }}</ref><ref>{{Cite web |url= http://www.moj.gov.cn/sfks/content/2009-11/20/content_1184854.htm?node=8011 |title= 关于2009年国家司法考试成绩公布、合格分数线及申请授予法律职业资格等事项的公告 |author= 中华人民共和国司法部国家司法考试办公室 |date= 2009年11月20日 |publisher= 中国普法网 |language= zh-hans |accessdate= 2012年12月9日 |quote=  }}</ref><ref>{{Cite web |url= http://www.moj.gov.cn/sfks/content/2010-11/18/content_2357277.htm?node=8011 |title= 关于2010年国家司法考试成绩公布、合格分数线及申请授予法律职业资格等事项的公告 |author= 中华人民共和国司法部国家司法考试办公室 |date= 2010年11月18日 |publisher= 中国普法网 |language= zh-hans |accessdate= 2012年12月9日 |quote=  }}</ref><ref>{{Cite web |url= http://www.moj.gov.cn/sfks/content/2011-11/16/content_3095698.htm?node=8011 |title= 关于2011年国家司法考试成绩公布、合格分数线及申请授予法律职业资格等事项的公告 |author= 中华人民共和国司法部国家司法考试办公室 |date= 2011年11月16日 |publisher= 中国普法网 |language= zh-hans |accessdate= 2012年12月9日 |quote=  }}</ref><ref>{{Cite web |url= http://www.moj.gov.cn/sfks/content/2012-11/21/content_3999193.htm?node=8011 |title= 中华人民共和国司法部国家司法考试办公室公告 |author= 司法部 |date= 2012年11月20日 |publisher= 中国普法网 |language= zh-hans |accessdate= 2012年12月9日 |quote=  }}</ref>
