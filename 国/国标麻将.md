'''国标麻将'''，是[[麻将|麻将]]的一种玩法，其规则为[[中华人民共和国|中国]][[国家体育总局|国家体育总局]]于1998年7月所制定，故被称为国标麻将。國標麻將的牌含有花牌，共144張，每人各持有13張手牌。国标将番种增加至81种，增加趣味性与竞技性。国标麻将8番起胡，增加比赛的难度，同时也增加趣味性。国标正在逐渐被大众所接受，成为一种时尚的娱乐方式。

== 国标麻将计分方式 ==

和牌后的各方得分： 
:底分=8分
#胜方：
#:自摸和牌：(底分+番種分数)×3
#:别人点炮和：(底分×3)+番種分数
#负方：
#:点炮者：-(底分+番種分数)
#:非点炮者：-底分
#:自摸和牌：-(底分+番種分数)


除了國標麻將之外，有些古老麻將規則、[[香港麻將|香港麻將]]或是其他地方規則也會使用類似這種計分方式，但在這種規則之下，若是有人胡牌，放銃者除了要付上點數以外，所有玩家（包括放銃者）都要付上底分給放銃者，使得某些人會利用這個規則向別人詐賭（兩個說好的人相互胡牌以騙取受害者的金錢），甚至這種計分方式也在某些電影當中出現（例如香港電影《[[洪福齊天|鬼賭鬼]]》當中，[[洪金寶|洪金寶]]與孟海利用這種計分規則在[[麻將館|麻將館]]對一位老婆婆詐賭，後被人趕出去）。

== 国标麻将規則要點 ==
#八番起和：不足8番时报和牌算作诈和
#一場比賽打四圈，不設連莊，因此每場一定是打16局
#沒有任何和牌限制：即使下家同巡跟打你剛打過的牌，一樣可以和
#第三圈（西圈）開始前，東家跟南家，西家跟北家對調座位，由原本的南家先做莊
#暗槓時，4隻牌牌面向下，不用給其他人看，唯在牌局完結後，不論有否和牌，都需打開供對手查閱

==国标麻将计分注意点==
===不重复原则===
当某个番种，由于组牌条件所决定，在其成立的同时，必然并存着其它番种，则其它番种不重复计分。

例：（明杠）[[File:0101一萬.svg|35px]][[File:0101一萬.svg|35px]][[File:0101一萬.svg|35px]][[File:0101一萬.svg|35px]]（明杠）[[File:0102二萬.svg|35px]][[File:0102二萬.svg|35px]][[File:0102二萬.svg|35px]][[File:0102二萬.svg|35px]]（碰）[[File:0104四萬.svg|35px]][[File:0104四萬.svg|35px]][[File:0104四萬.svg|35px]]（碰）[[File:0108八萬.svg|35px]][[File:0108八萬.svg|35px]][[File:0108八萬.svg|35px]]（手牌）[[File:0109九萬.svg|35px]]（点和）[[File:0109九萬.svg|35px]]

此牌型正确计分为'''“清一色24分+碰碰和6分+全求人6分+双明杠4分+幺九刻1分=41分”'''，其中清一色必然存在无字，全求人必然存在单钓将，双明杠必然存在明杠，故不重复计算。

===不拆移原则===
确定一个番种后，不能将其自身再拆开互相组成其它番种计分。

例：[[File:0109九萬.svg|35px]][[File:0109九萬.svg|35px]] [[File:0201一餅.svg|35px]][[File:0201一餅.svg|35px]] [[File:0202二餅.svg|35px]][[File:0202二餅.svg|35px]] [[File:0203三餅.svg|35px]][[File:0203三餅.svg|35px]] [[File:0307七條.svg|35px]][[File:0307七條.svg|35px]] [[File:0308八條.svg|35px]][[File:0308八條.svg|35px]] [[File:0309九條.svg|35px]][[File:0309九條.svg|35px]]

此牌型正确计分为'''“七对24分+无字1分=25分”'''，七对是七个对子，所以不可以拆开重新组成四个顺子加一对将的形式，继而不得计平和、一般高的番种，全带幺也不得计。

但有趣的是，國標麻將沒有某些規則有的番種「兩般高（又稱雙龍抱、[[二杯口|二盃口]]）」，若是以[[日本麻將|日本麻將]]的角度來計算這副牌的番種的話，至少有6翻（兩般高3番+純全帶幺3番）以上，若是不聽九萬單吊，則再計平胡1番（若以七對子的角度計，於日本麻將中僅2番，即七對子本身的番數）。

若是以廣東麻將來看這副牌的話，雙龍抱的10番是比七對的7番（有些玩法視為10番，番數持平）還要來得高的。

===不得相同原则===
凡已组合过某一番种的牌，不能再同其它一副牌组成相同的番种计分。

例：[[File:0106六萬.svg|35px]][[File:0106六萬.svg|35px]][[File:0106六萬.svg|35px]] [[File:0207七餅.svg|35px]][[File:0207七餅.svg|35px]][[File:0207七餅.svg|35px]] [[File:0308八條.svg|35px]][[File:0308八條.svg|35px]][[File:0308八條.svg|35px]] [[File:0109九萬.svg|35px]][[File:0109九萬.svg|35px]][[File:0109九萬.svg|35px]] [[File:0401東風.svg|35px]][[File:0401東風.svg|35px]]

此牌型正确计分为'''“三色三节高8分+碰碰和6分+幺九刻1分=15分”'''，要么6万、7饼、8条组成三色三节高，要么7饼、8条、9万组成三色三节高，而7饼、8条在这两种番種都出现过，所以只能任选其一不得两个番种重复计算。

===就高不就低原则===
有两副以上的牌，有可能组成两个以上的番种，而只能选其中一种计分时，可选择分值较高的番种计分。

例：[[File:0102二萬.svg|35px]][[File:0102二萬.svg|35px]][[File:0102二萬.svg|35px]][[File:0103三萬.svg|35px]][[File:0103三萬.svg|35px]][[File:0103三萬.svg|35px]][[File:0104四萬.svg|35px]][[File:0104四萬.svg|35px]][[File:0109九萬.svg|35px]][[File:0109九萬.svg|35px]][[File:0302二條.svg|35px]][[File:0303三條.svg|35px]][[File:0304四條.svg|35px]]点和[[File:0104四萬.svg|35px]]

此牌型正确计分为'''“一色三同顺24分+门前清2分+平和2分+喜相逢1分+缺一门1分=30分”'''，要么计算成“一色三节高24分+门前清2分+双暗刻2分+缺一门1分=29分”，要么计算成正确计分，则取较高的30分来计算。

===套算一次原则===
如有尚未组合过的一副牌，只可同已组合过的相应的一副牌套算一次。

例：[[File:0101一萬.svg|35px]][[File:0102二萬.svg|35px]][[File:0103三萬.svg|35px]] [[File:0204四餅.svg|35px]][[File:0205五餅.svg|35px]][[File:0206六餅.svg|35px]] [[File:0307七條.svg|35px]][[File:0308八條.svg|35px]][[File:0309九條.svg|35px]] [[File:0201一餅.svg|35px]][[File:0202二餅.svg|35px]][[File:0203三餅.svg|35px]] [[File:0405中.svg|35px]][[File:0405中.svg|35px]]

此牌型正确计分为'''“花龙8分+喜相逢（或连六）1分=9分”'''，花龙组合好之后（123万、456饼、789条），只剩下123饼与一对红中，而123饼要么和123万组成喜相逢，要么和456饼组成连六，而规定只能套算一次，所以只能任选其一，不得两種番種分數重复计算。

==最大和牌番数==
在国标麻将的规则下，最大和牌番数为：大三元 88番 + 四杠 88番 + 字一色 64番 + 四暗刻 64番 + 杠上开花 8番 + 妙手回春 8番 + 门风刻 2番 + 圈风刻 2番 + 花牌×8 8番 = 332番。
