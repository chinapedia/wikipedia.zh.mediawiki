'''硒化锌'''是一种[[无机化合物|无机化合物]]，为黄红色的晶体，化学式为ZnSe。它是一个[[本征半导体|本征半导体]]，[[能隙|能隙]]2.70 [[电子伏特|eV]]（25 °C）。硒化锌很少出现在自然界，它出现在[[方硒锌矿|方硒锌矿]]中。

==制备==
将[[乙酸铵|乙酸铵]]缓冲的[[硫酸锌|硫酸锌]]稀溶液滴入[[硒化氢|硒化氢]]饱和溶液制得，此溶液需要持续通入被稀释的硒化氢（如用氮气或氢气稀释）。尾气用浓[[硝酸|硝酸]]吸收。<ref name=zb>《无机化合物制备手册》. 朱文祥 编. 化学工业出版社. P<sub>378~379</sub>. 【XII-6】硒化锌（zinc selenide）</ref>
: Al<sub>2</sub>Se<sub>3</sub> + 6 HCl → 2 AlCl<sub>3</sub> + 3 H<sub>2</sub>Se↑

: H<sub>2</sub>Se + ZnSO<sub>4</sub> → ZnSe↓ + H<sub>2</sub>SO<sub>4</sub>
用干法也能制备硒化锌，如将[[硫化锌|硫化锌]]和[[亚硒酸|亚硒酸]]的混合物加热至800℃，可以得到该产物：<ref name=wj />
: ZnS + H<sub>2</sub>SeO<sub>3</sub> → ZnSe + SO<sub>2</sub>↑ + H<sub>2</sub>O↑
另一种方法是利用[[氧化锌|氧化锌]]、[[硫化锌|硫化锌]]和[[硒|硒]]单质的反应。具体步骤是：取4g[[氧化锌|ZnO]]、2.5g[[硫化锌|ZnS]]和6g[[硒|Se]]，均匀混合，放入石英坩埚中，在800[[摄氏度|℃]]加热15[[分钟|min]]，即得到产物：<ref name=zb />
: 2 ZnO + ZnS + 3 Se → 3 ZnSe + SO<sub>2</sub>↑

==化学性质==
硒化锌在湿空气中易被氧化。它可溶于稀酸，放出剧毒的[[硒化氢|H<sub>2</sub>Se]]气体：<ref name=wj>《无机化学丛书》.第六卷 卤素 铜分族 锌分族. P<sub>543~544</sub>. (2)硒化锌</ref>
: ZnSe + 2 H<sup>+</sup> → Zn<sup>2+</sup> + H<sub>2</sub>Se↑
