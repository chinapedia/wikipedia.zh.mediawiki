'''硒代硫酸钾'''是一种[[无机化合物|无机化合物]]，[[化学式|化学式]]為{{化學式|鉀|2|硒|硫|氧|3}}<ref name=wc5>《无机化学丛书》.第五卷.氧 硫 硒分族.科学出版社. 张青莲 主编. 2012年3月第四次印刷. P<sub>344</sub>. 3.硫代硒酸盐和硒代及碲代硫酸盐. ISBN 978-7-03-030544-2</ref>。

==结构==
在硒代硫酸钾中，阴离子SeSO<sub>3</sub><sup>2-</sup>的Se为中心原子。
==性质==
硒代硫酸钾可溶于水，其固体受热分解，产生[[硒化钾|硒化钾]]、[[多硫化钾|多硫化钾]]和[[硫酸钾|硫酸钾]]的混合物<ref name=wc5 />。

硒代硫酸钾也可以被酸分解，如<ref name=wc5 />：
:K<sub>2</sub>SeSO<sub>3</sub> + 2 HCl → 2 KCl + Se↓ + SO<sub>2</sub>↑ + H<sub>2</sub>O

也可以和某些金属的化合物（如[[银|银]]、[[汞|汞]]的化合物等）反应<ref name=wc5 />：
: K<sub>2</sub>SeSO<sub>3</sub> + Ag<sub>2</sub>O → Ag<sub>2</sub>Se + K<sub>2</sub>SO<sub>4</sub>
: K<sub>2</sub>SeSO<sub>3</sub> + Hg(CN)<sub>2</sub> + H<sub>2</sub>O → Hg<sub>2</sub>Se + K<sub>2</sub>SO<sub>4</sub> + 2 HCN
其产物是它们的[[硒化物|硒化物]]。

它有还原性，可以被单质[[碘|碘]]氧化：
:I<sub>2</sub> + 2 K<sub>2</sub>SeSO<sub>3</sub> → 2 KI + K<sub>2</sub>Se<sub>2</sub>S<sub>2</sub>O<sub>6</sub>（二硒代连四硫酸钾）
