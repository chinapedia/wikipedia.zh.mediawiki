'''急性乳腺炎'''( {{lang-en|acute mastitis}} )主要原因是细菌感染导致的一种急性[[乳腺|乳腺]]疾病。

== 症状表现 ==
短期内出现肿胀现象，按压会出现疼痛、硬块、红肿等情况；局部有波动感；腋窝[[淋巴结|淋巴结]]变大，几天内就会出现化[[脓|脓]]现象；

== 形成原因 ==
没有做好平日[[乳房|乳房]]自检工作，自身机体[[免疫力|免疫力]]下降，[[情绪|情绪]]波动大，恶劣的生活习惯，等是形成急性乳腺炎的原因可能。哺乳期时3到4周要特别注意。
== 预防 ==
# 争取产后尽早开始喂奶，一时吃不完，一定要用[[吸乳器|吸奶器]]将余下的乳汁吸出，防止乳汁淤积。
# 切忌让宝宝含着乳头入睡。因为宝宝长时间吮吸易造成乳头破损，导致细菌容易入侵；再者，宝宝的唾液中含有消化酶，可将乳汁变成“小乳酪”，容易堵塞乳管。宝宝吸完奶后如果不松口切不可强行将乳头拔出，应用手指轻轻按压宝宝下颌，使其慢慢松口。
# 哺乳期一定要注意乳头的护理。应经常用温水清洗乳头、乳晕，不要用肥皂类清洗，这样会洗去了乳房皮肤的油脂，乳房容易出现皲裂或破损。每次哺乳后可挤出一滴乳汁或用芝麻油涂在乳头上起保护作用。
# 哺乳期间一定要佩戴合适的胸罩。对乳房起到良好的承托作用，以保持乳房良好的血液循环。
# 注意饮食，进食过多的肉、鸡、鱼、蛋，滋补过量会造成乳汁中蛋白、脂肪含量增多，如果饮水不足，乳汁粘稠，容易导致排出不畅。<ref>{{cite web |url=http://www.ycwb.com/ePaper/ycwb/html/2015-01/29/content_639994.htm |title=新手妈妈不懂喂奶护乳易得急性乳腺炎 |deadurl=yes |archiveurl=https://web.archive.org/web/20150226110350/http://www.ycwb.com/ePaper/ycwb/html/2015-01/29/content_639994.htm |archivedate=2015-02-26 }}</ref>

== 治疗方法 ==
使用[[抗生素|抗生素]]，通畅乳腺管，排挤[[乳汁|乳汁]]，或进行[[手术|手术]]治疗。
