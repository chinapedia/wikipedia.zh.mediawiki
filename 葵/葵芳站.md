'''葵芳站'''（{{lang-en|Kwai Fong Station}}）是[[港鐵|港鐵]][[荃灣綫|荃灣綫]]的一個[[架空車站|架空車站]]，位於[[新界西|新界西]][[葵青區|葵青區]]，[[中葵涌|中葵涌]]與[[下葵涌|下葵涌]]之間的分界綫，但主要的是[[下葵涌|下葵涌]][[醉酒灣|醉酒灣]]葵仁路旁，於1982年5月10日啟用。

==結構==
{{ multiple image | align = right | direction = vertical | width = 220
| image1 = Kwai Fong Station 2014 04.JPG | caption1 = 車站外觀
}}
葵芳站是一個架空車站。其與葵興站之間的一段的架空路軌設有混凝土隔音屏障，是首項地鐵在興建時已經規劃的減低噪音工程，減低列車嘈音對列車於葵芳邨中間通過時的影響以及防止葵芳邨居民將垃圾丟進軌道。

===大堂===
{{ multiple image | align = right | direction = vertical | width = 220
| image1 = Kwai Fong Station 2018 06 part4.jpg | caption1 = 大堂
}}
葵芳站大堂的非收費區是被收費區分隔，因此乘客不能由大堂北面（A、D出入口）直接走至大堂南面（B、C出入口）或E出入口大堂，需要途徑路面往返兩個大堂，此特點與[[荃灣站|荃灣站]]相似。

葵芳站大堂現時有不同類型的車站商店，包括便利店、麵包糕餅店、外賣店、診所、藥房等，以及各樣自助服務，包括自動櫃員機和自動售賣機<ref name="shop"/>。大堂內亦有「iCentre」免費上網服務設施供乘客瀏覽[[全球資訊網|萬維網]]的內容，乘客亦可以設有[[Wi-Fi|Wi-Fi]]功能的電子設備於「iCentre」範圍內享5節各15分鐘的免費無線寬頻服務<ref>{{cite web|url=http://www.mtr.com.hk/chi/getting_around/icentre.html|title=鐵路四處遊－iCentre互聯網中心|publisher=港鐵公司|work=港鐵網頁|accessdate=2012-01-29}}</ref>。此外，於非收費區近A和D出入口設有「-{會員服務站}-」，供乘客登記為港鐵友禮會會員、查閱獎分及換領獎賞<ref>{{cite web|url=http://www.mtr.com.hk/ch/customer/services/csp.html|title=鐵路四處遊－「-{會員服務站}-」|publisher=港鐵公司|work=港鐵網頁|accessdate=2016-09-26}}</ref>。

===月台===
{{ multiple image | align = right | direction = vertical | width = 220
| image1 = Kwai Fong Station 2018 06 part1.jpg | caption1 = 1號月台﹙往荃灣方向﹚
| image2 = Kwai Fong Station 2018 06 part5.jpg | caption2 = 2號月台﹙往中環方向﹚
}}
月台採用[[側式月台|側式月台]]設計，兩個月台的候車位置被路軌分隔，並裝有月台閘門。乘客如果往返兩個月台，需要途徑大堂。根據車站啟用時的月台設計，在1號月台牆身較低位置，由於面向當時的葵芳[[臨時房屋區|臨時房屋區]]（即現時[[新都會廣場|新都會廣場]]及[[葵涌廣場|葵涌廣場]]的位置），所以裝了全身落地玻璃板或設有幼鐵絲網欄的全身結構開口以便自然通風。但2號月台因為太靠近建築物（好爵中心），故此牆身沒有通風窗口或玻璃窗。兩個月台的牆身較高位置設有大型廣告燈箱，而每個燈箱以車站主色（綠色）的纖維板相間，纖維板上有車站名稱和月台號碼。後來在纖維板上方印上月台列車方向及目的地。而車站經歷多次月台翻新工程，現今月台的形像與車站啟用時大有不同。

葵芳站車站的整體佈局及設計跟葵興站非常相似。

==鄰接車站==
葵芳站是荃灣綫的中途站，乘客可在此乘坐往中環或荃灣列車，往返[[香港島|香港島]]、[[九龍|九龍]]及[[新界|新界]]等地。
{{荃灣綫}}

==利用狀況==
葵芳站是葵青區[[葵涌|葵涌]]的大型交通中轉站，亦有多條巴士線連接和新界西倒數旺地，車站又設有大型巴士總站，供乘客轉乘巴士前往新界西區；車站附近集合了葵涌工業區、大型商場 [[新都會廣場|新都會廣場]]、[[葵涌廣場|葵涌廣場]]、住宅樓宇及[[葵青劇院|葵青劇院]]、[[葵涌運動場|葵涌運動場]]等康樂設施，另外有不少就讀[[香港專業教育學院青衣分校|香港專業教育學院青衣分校]]的學生，都是乘搭港鐵荃灣綫到此站轉乘88C小巴或香港專業教育學院青衣分校校巴上下課，因此此站成為區內最繁忙的港鐵車站之一。

===命名===
葵芳站位於葵涌垃圾灣（即[[醉酒灣|醉酒灣]]）堆填區，在1967年的地底集體運輸系統建議書中，此站命名為'''垃圾灣'''，在通車時據附近興建有政府廉租屋[[葵芳邨|葵芳邨]]而命名為葵芳站。

===與西鐵交匯的建議===
現時[[西鐵綫|西鐵綫]]（前稱九廣西鐵）在葵芳站南面100米處的葵福路地底路過，但不在葵青區設站。惟1990年代後期，[[九廣鐵路公司|九廣鐵路公司]]在規劃西鐵時，曾有意見認為西鐵應在葵芳設站，與此站接駁。然而，九廣鐵路公司以興建技術因難及轉乘安排不便為由否決有關建議。<ref>[https://web.archive.org/web/20070303041745/http://www.sp1900.net/railposthk/RPHK05_p2.jpg 港鐵誌：九廣東鐵葵涌貨運線 (二之二) － 客運篇(2)]（Web Archive備份）</ref>

===增建E出入口===
葵芳站1號月台以前沒有與往新都會廣場的行人天橋連接，當時往來新都會廣場的乘客需要經過大堂及出入口才可出入車站。後來因人流增多而決定興建新出入口接駁1號月台及行人天橋，於2006年5月24日下午4時啟用，並編定為E出入口。

===月台閘門===
現時葵芳站所有月台已完成裝上月台閘門，並已投入運作。月台閘門約高1.5米。整個計劃於平日深夜終止服務的時間加建。此站與[[葵興站|葵興站]]及[[觀塘綫|觀塘綫]][[九龍灣站|九龍灣站]]、[[牛頭角站|牛頭角站]]屬於第二期進行月台閘門加裝工程，該站所有月台的月台閘門加裝工程經已於2011年8月完工。

===翻新月台===
早期的月台改善工程要牽涉在廣告燈箱的位置，原本月台設計是每個在月台的大型廣告燈箱與綠色站名纖維板梅花間竹地掛在牆身高位，1號月台共有34個廣告燈箱，2號月台共有36個廣告燈箱。後來於1990年代，基於工作安全理由，將部份在樓梯或扶手電梯之上的廣告燈箱封上綠色膠布，這些燈箱不再作廣告燈箱使用，這使1號月台可用廣告燈箱減至25個，2號月台減至24個燈箱。

與市區線很多非地下車站一樣，葵芳站月台由2011年起也開始展開翻新工程，當中包括安裝假天花、改善照明，亦翻新了深綠色站名纖維板，原「往荃灣/中環」的行車方向資訊則改為張貼在月台閘門之上。但由於假天花進一步阻擋大型廣告燈箱的視線，故此港鐵公司再大幅度減少可使用的月台廣告燈箱數目，使1號月台只剩下22個可用，2號月台剩下8個可用，這些可繼續使用的燈箱的位置都略為降低，而其他棄用的燈箱則以綠色膠布封掉。
