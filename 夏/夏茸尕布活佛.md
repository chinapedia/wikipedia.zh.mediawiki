'''夏茸尕布活佛'''（{{bo|z=ཞབས་དང་དཀར་པོ}}），又译'''「夏容嘉布活佛」'''，[[汉语|汉语]]通常称“'''白佛'''”，又称“'''察罕诺们汗'''”、“'''拉茂活佛'''”（又译“拉莫活佛”）、“'''阿齐图诺们汗'''”等等，[[藏传佛教|藏传佛教]][[格鲁派|格鲁派]]活佛，是[[青海省|青海省]]较大的藏传佛教[[转世|转世]][[活佛|活佛]]系统之一，驻锡[[拉莫德钦寺|拉莫德钦寺]]<ref name=kao>[http://www.tibet.cn/periodical/xzyj/1994/04/200805/t20080523_395822.htm 考洛巴·陈列奥赛，夏茸尕布传，西藏研究1994年第4期]</ref><ref name=xzzx>{{Cite web |url = http://www.tibetol.cn/html/2013/zdyg_0405/393.html |title =夏茸尕布活佛  |author =  |date = 2017-05-23 |publisher =  西藏在线|language =  |accessdate =  |quote =  }}</ref><ref name=cairang>才让扎西，拉茂德乾寺历史与现状研究，青海民族大学法学硕士学位论文，2016年</ref>。

==沿革==
夏茸尕布活佛是中国[[青海省|青海省]][[尖扎县|尖扎县]]拉莫德钦寺寺主。原先辖有今青海省尖扎县、[[化隆县|化隆县]]、[[贵德县|贵德县]]、[[贵南县|贵南县]]、[[同德县|同德县]]、[[刚察县|刚察县]]、[[海晏县|海晏县]]等地的众多部落及寺院<ref name=xzzx/>。

[[明朝|明朝]][[万历|万历]]五年（1577年），应[[蒙古|蒙古]][[土默特部|土默特部]]首领[[俺答汗|俺答汗]]之邀，[[三世达赖|三世达赖]]率随从到[[青海湖|青海湖]]畔同俺答汗会晤，措尼嘉措随行。三世达赖与俺答汗临别时，应[[火落赤|火落赤]]的请求，派[[措尼嘉措|措尼嘉措]]为代表到二十四旗讲经，获得当地蒙藏信众崇奉。火落赤把阿噶绛曲林寺（今[[尖扎县|尖扎县]]阿哇多杰桑波）奉献给他。此后不久，[[明朝|明朝]][[万历帝|万历帝]]封他为“阿噶格杰尕布”（{{bo|z=ར་སར་ས་སེས་དཀར་པོ}}）并敕赐金印。“格杰”藏语意为转世，“尕布”藏语意为白色，因此当地民众称他为“夏茸尕布”（又譯「夏容嘉布」，意为善业白色活佛）。此后，因弘法有功，他又获得“察汗[[诺们汗|诺们汗]]即夏茸尕布”的尊号。此即夏茸尕布名称之由来。他也是第一世（不计追认）夏茸尕布活佛<ref name=kao/><ref name=xzzx/><ref name=cairang/>。

[[清朝|清朝]][[康熙|康熙]]四十四年（1705年），第三世夏茸尕布活佛到[[北京|北京]]受封，清朝颁发“察罕诺们汗”诏书并封其为[[扎萨克喇嘛|扎萨克喇嘛]]<ref name=xzzx/>。

清朝康熙五十三年（1714年），[[七世达赖|七世达赖]]封第三世夏茸尕布活佛为 “阿齐图诺们汗”<ref name=xzzx/>。

清朝[[雍正|雍正]]三年（1725年），[[青海蒙古|青海蒙古]]划旗定界，夏茸尕布活佛所属部落被单独划为“察罕诺门汗旗”，夏茸尕布活佛由此成为今贵南县、同德县西部、贵德县南部、尖扎县、[[泽库县|泽库县]]、化隆回族自治县西部以及海北藏族自治州海晏县的政教双重统治者<ref name=xzzx/>。

察罕諾們汗是藏化的[[青海蒙古|青海蒙古]]，其所轄[[察罕諾們汗旗|察罕諾們汗旗]]是清朝[[外札薩克蒙古|外札薩克蒙古]][[青海蒙古|青海蒙古]]三十旗之一，授扎薩克喇嘛，轄四佐領，歸[[西寧辦事大臣|西寧辦事大臣]]統轄。19世紀以來，[[青海湖|青海湖]]周圍的[[蒙古族|蒙古族]]中，靠近[[藏族|藏族]]的青海湖南岸的蒙古各旗和察罕諾們汗旗也相繼藏族化。察罕諾門汗旗的形成與變異是蒙藏文化融合、蒙古藏化的典型事例<ref>南文淵，可可淖爾蒙古 走向邊緣的歷史，遼寧民族出版社，2007年5月，第694頁</ref>。

夏茸尕布活佛迄今共十六世，其中前七世系追认。如不计追认，则迄今共九世<ref>{{cite book|title=藏族大辞典|publisher=甘肃人民出版社}}</ref><ref>{{cite web|url=http://www.cnr.cn/wcm/qinghai/zjsy/zcfj/t20031104_141404.html|title=青海地区主要活佛|publisher=中广网}}</ref><ref name=kao/>。
