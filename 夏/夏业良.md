'''夏业良''' （{{bd|1960年|9月4日|catIdx=X}}）中国[[经济学|经济学]]家，曾任[[北京大学经济学院|北京大学经济学院]]教授。北京大学外国经济学说研究中心副主任，中国经济体制改革研究会公共政策研究所副所长，兼任[[上海大学|上海大学]]、[[上海财经大学|上海财经大学]]、[[西安交通大学|西安交通大学]]教授，美国[[卡托研究所|卡托研究所]]客座研究员。

夏业良提倡自由民主、法治、个人自由和自由市场竞争，也是《[[零八宪章|零八宪章]]》的首批签名者之一。<ref>[http://www.appledaily.com.tw/realtimenews/article/international/20131019/277437/%E6%89%B9%E8%A9%95%E4%B8%AD%E5%9C%8B%E6%94%BF%E6%B2%BB%E9%AB%94%E5%88%B6%E3%80%80%E5%8C%97%E5%A4%A7%E6%95%99%E6%8E%88%E5%A4%8F%E6%A5%AD%E8%89%AF%E9%81%AD%E8%A7%A3%E8%81%98 批評中國政治體制　北大教授夏業良遭解聘], [[蘋果日報_(台灣)|蘋果日報 (台灣)]], 2013年10月19日</ref><ref>[http://www.bbc.co.uk/zhongwen/simp/china/2014/02/140212_xia_yeliang.shtml 北大解雇教授夏业良加入加图研究所 BBC 2014年2月12日]</ref>

== 求學 ==
1960年9月4日夏业良生于安徽[[芜湖|芜湖]]，1984年获得[[安徽大学|安徽大学]]学士学位。此后先后获得[[复旦大学|复旦大学]]经济学硕士和博士学位。

== 經歷 ==
夏业良曾在政府部门、研究机构和高等院校工作，参与[[联合国开发计划署|联合国开发计划署]]和[[世界银行|世界银行]]在华项目工作。其他兼职包括[[中国改革基金会|中国改革基金会]]国民经济研究所研究员、[[天则经济研究所|天则经济研究所]]特邀研究员、中国金融教育发展基金会理事、九鼎公共事务研究所财经研究中心主任、[[中央人民广播电台|中央人民广播电台]]经济顾问<ref>[http://www.cser.org.cn/news/136.aspx 公共政策评论人物系列 夏业良 博士，中国经济体制改革研究会]</ref>。

== 思想 ==
===经济学===
夏业良被称为[[自由主义经济学|自由主义经济学]]家，他认为“计划思维”的回归与资源的逆市场化权力配置导致了社会财富向少数利益集团集中，收入分配的扭曲和贫富差距的扩大。<ref>[http://www.cser.org.cn/news/703.aspx 夏业良：中国如何跨越“风险社会”,中国经济体制改革研究会]</ref>他指出近10年来，中国地区、城乡、行业、群体间的收入差距明显加大，收入分配格局失衡导致社会财富向少数利益集团集中，尤其是向官僚利益集团和垄断利益集团集中。中国1%的家庭掌握了全国41.4%的财富，财富集中度远远超过了美国，成为全球两极分化最严重的国家之一。<ref>[http://www.cnki.com.cn/Article/CJFDTotal-ZSTW201013003.htm 《中国财富集中度超过美国》，《商周刊》 2010年13期 ]</ref>他指出近几年的政策取向就是使垄断国企越来越强势，这种"国进民退"其实就是最大的风险。<ref>[http://www.cnki.com.cn/Article/CJFDTotal-LUYE200908008.htm “国进民退”下的国家经济风险，《绿叶》 2009年08期]</ref>

夏业良对通过税收调节来缩短贫富差距、解决分配不均问题表示了担忧：
{{quote|通过税收调节，最终缩小的可能是袁隆平和普通农民工之间的差距，从而削弱了中国未来的创新能力和财富的创造能力。实际上老百姓深恶痛绝的并不是依靠科技发明、诚实劳动致富的人，而是那些通过不当的[[权利垄断|权利垄断]]获得巨额财富的人。<ref>[http://www.cser.org.cn/news/382.aspx 改革到了向纵深推进的时候]</ref>}}

夏业良提出了要把[[共产党|共产党]]的党产和国家的财产分开的理论，指出中国经济改革，无法回避政治体制、政党、宪政民主以及权力制衡等基本核心问题：
{{quote|要讲政治体制改革，大家最关心的问题就是宪政民主的问题。即使你现在不能实现，那又没有一个时间表，或是朝这个方向努力。从权利制衡来讲，不管是三权分立、四权分立，对权力一定要有个制衡。对这个根本问题，中央领导人从来没有回答过。<br>
对人文社会科学的思想，还是由党的意识形态部门加以严格控制。举个简单例子，一个国家进行人文社会科学研究，应当有自由的空间，可是在中国，所有的社会科学研究的重大课题，都是由[[社会科学规划办公室|社会科学规划办公室]]作出的。这个办公室就设在中共中央的宣传部里。<ref>[http://www.voachinese.com/content/a-21-w2006-09-03-voa39-57874047/1060490.html 夏业良：须加快政改停止钳制思想]</ref>}}

{{quote|这一个是北大人的悲哀，是一种现象，它的根源是上边的，一切的控制舆论，压制言论自由的渠道，这个根是在中宣部。北大是听命于上边的，北大这个学校是[[党委制|党委制]]，[[党委书记|党委书记]]、[[党委|党委]]说了算，不是按照北大的学术地位和宪法来说事。<ref>[http://www.voachinese.com/content/outspoken-beijing-u-professor-threatened-to-be-fired-20130704/1694945.html 美国之音：北大夏业良教授遭威胁 去留将考验北大人风骨]</ref>}}

台湾[[立法委员|立法委员]]、[[时代力量党|时代力量党]]首任主席[[黃國昌|黃國昌]]博士在《[[台灣蘋果日報|台灣蘋果日報]]》即時新聞專欄撰文〈自我作賤的北京大學〉批判：「當[[北大|北大]]決定不再續聘長期批判中共極權、要求中國進行政治改革的夏業良教授時，就已自我對全世界正式宣告：我們不僅墮落為極權的附庸，也早已將[[蔡元培|蔡元培]]所曾標榜「研究學問、思想自由」的北大精神糟蹋殆盡，更已配不上「大學」這個稱號。...10月11日的投票，不過是結果早已決定的過場秀。北大經濟學院嗣後試圖自清所提出的解釋，不僅是在侮辱眾人的智商，更是在羞辱北大自己的智商。...喊的再大聲的「中國夢」口號，也無法掩蓋中共極權殘暴的本質。」

已故的台湾前[[立法委员|立法委员]][[李敖|李敖]]之子[[李戡|李戡]]博士曾于2013年3月29日在其[[新浪微博|新浪微博]]中写道：“昨晚和夏业良老师吃羊蝎子，时光飞逝，上次到他家里，已是两年半前的事了。许久不见，老师的言论缓和许多，但仍不失尖锐的批判思维。我们相谈甚欢，尽管在部分议题上存在意见分歧。我认为夏老师不应该追随“他朋友”的团队，事实上他比他们任一人都优秀，相貌堂堂且学识丰富，应该做独立的思想播种者。” 李戡也曾在《[[时代周报|时代周报]]》专访中表示：“（北大）比较有特色的是教《经济学原理》的夏业良教授，他蛮有学问，我看到他有很多著作，上过很多[[央视|央视]]、[[凤凰台|凤凰台]]的节目。他上课不只是告诉你一些经济学原理，他会评论很多社会现象。”

美國知名學府[[衛斯理學院|衛斯理學院]]，有超過130名教授聯名發表公開信予以譴責，要求衛斯理學院停止與北大的合作關係。<ref>[http://www.appledaily.com.tw/realtimenews/article/new/20131021/278343/ 【黃國昌專欄】​自我作賤的北京大學], [[蘋果日報_(台灣)|蘋果日報 (台灣)]], 2013年10月21日</ref>

== 著作 ==
主要研究领域为[[新制度经济学|新制度经济学]]、公共选择与公共政策、[[劳动经济学|劳动经济学]]、中国转型与改革。

在《经济研究》、《经济学动态》、《经济科学》、《管理世界》、《世界经济与政治》、《战略与管理》、《社会科学战线》、《改革》、《经济评论》、《经济理论与经济管理》、《中国人口科学》、《复旦学报》等期刊上发表论文40多篇。已出版个人文集包括《制度是怎样炼成的》、《公共问题的经济解读》、《经济学不能承受之重》、《经济学能做什么》、《选择之道》等。

主要期刊论文包括《中国的人口结构变化与就业前景》、<ref>[http://www.cqvip.com/qk/81923x/200102/12347930.html 《战略与管理》2001年 第2期]</ref>《境外投资者参与债权转股运作的可行性分析》、<ref>[http://www.cqvip.com/qk/90392x/200007/4375459.html 《国际经济合作》2000年 第7期]</ref>《全球化对中国高等教育的影响》、<ref>[http://www.cqvip.com/qk/96773x/200105/5220366.html 《上海改革》2001年 第5期]</ref>《劳动经济学理论前沿》、<ref>[http://www.cqvip.com/qk/93743x/200104/5124988.html 《经济学动态》2001年 第4期]</ref>《就业机制的内涵与中国城市就业机制》<ref>[http://old.ccer.edu.cn/workingpaper/paper/c2000011.pdf 就业机制的内涵与中国城市就业机制] {{webarchive|url=https://web.archive.org/web/20051027073838/http://old.ccer.edu.cn/workingpaper/paper/c2000011.pdf |date=2005-10-27 }}</ref>等。

译著包括《劳动经济学》、《城市土地经济学》、《我们身边的经济学》、《经济学方法论新论》、《体验经济》、《全球化及其不满》、《盐》、《经济学百科全书》等。
