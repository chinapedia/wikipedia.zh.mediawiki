
'''夏勇'''（{{bd|1961年|11月||}}），[[湖北|湖北]][[江陵|江陵]]人，法学博士，研究员，教授。曾任中国社会科学院法学研究所所长、国际法研究所所长，中共中央办公厅调研室副主任，中共中央保密办主任、国家保密局局长，国务院法制办副主任。曾任中共十七大代表、十八大代表，第十届全国人大常委会香港基本法委员会委员，第十二届全国政协委员。

==生平==
1978年考入西南政法学院（现西南政法大学）法律专业，1982年7月毕业留校，任法制史教研室助教，后攻读该校外国法制史硕士研究生，[[北京大学|北京大学]]法律学系博士研究生。哈佛大学法学院做访问学者，哈佛大学甘乃迪政府学院做博士后。

曾在[[西南政法大学|西南政法大学]]、[[中南财经政法大学|中南财经政法大学]]任教，任[[中国社会科学院|中国社会科学院]]法学研究所研究员，研究室副主任、主任，之后升任社科院研究生院法学系主任、教授、博士生导师、法学研究所所长，国际法研究所所长，两所联合党委书记（兼）<ref>{{cite news|title=逢物言道 遇事阐法  ———记当代杰出中青年法学家夏勇|url=http://www.legalinfo.gov.cn/moj/lsgzgzzds/content/2005-06/15/content_155341.htm?node=286|accessdate=2014-10-13|newspaper=司法部|deadurl=yes|archiveurl=https://web.archive.org/web/20141026091200/http://www.legalinfo.gov.cn/moj/lsgzgzzds/content/2005-06/15/content_155341.htm?node=286|archivedate=2014-10-26}}</ref>。

曾兼任国务院学科评议组成员、中国法律史学会会长、中国法学会副会长、<ref>{{cite web |title=法治人物：夏勇 |url=http://www.dffyw.com/sifashijian/jj/201310/33904.html |website=东方法眼 |accessdate=2018-11-29}}</ref>中国法学会法理学研究会常务副会长、<ref>{{cite news |author1=《新京报》 |title=《法学家夏勇入主国家保密局》 |url=http://news.sina.com.cn/c/2005-09-20/03227813421.shtml |accessdate=2018-11-28 |work=新京报 |publisher=新浪网}}</ref>《环球法律评论》主编、全国青联常委、中国与欧盟人权学术交流机制中方主席、北京东方公益诉讼与法律援助律师事务所主任等职。<ref>{{cite web |title=学者文集——夏勇 |url=http://www.iolaw.org.cn/showScholar.aspx?id=86 |website=中国法学网 |accessdate=2018-11-28}}</ref>

曾获1993年第七届“中国图书奖”，1995年被授予首届“中国十大杰出中青年法学家”称号。<ref>{{cite news |author1=吴江 |title=认认真真做事做人（归国留学人员风采录） ———记中国社科院法学研究所副所长夏勇 |url=http://www.people.com.cn/GB/paper39/1940/311926.html |accessdate=2018-11-28 |work=《人民日报》海外版 |agency=人民日报 |issue=第三版 |publisher=人民网 |date=2000年11月4日}}</ref>1998年留学回国工作后，曾获中国社会科学院优秀学术著作奖、全国社科青年优秀论文奖、教育部吴玉章学术著作奖、“改革开放35年来影响中国的十部优秀法学著作”等作品奖，被授予“有突出贡献的中青年专家”称号并享受政府特殊津贴、全国普法先进个人、中组部等六部委联合授予“留学回国人员成就奖”、全国宣传文化系统首批“文化名家”及“四个一批”人才。曾被《中国青年》杂志评选为“可能影响中国二十一世纪的百名青年人物。”2000年担任江泽民主持的中南海法制讲座主讲人。<ref>{{cite web |author1=人民网 |title=中共中央举办法制讲座 |url=https://www.people.com.cn/GB/paper1172/1537/249505.html |website=人民网 |accessdate=2018-11-28}}</ref>

之后从政，<ref>{{cite web |title=胡锦涛开始清算江理论 |url=http://www.epochtimes.com/gb/5/8/11/n1014809.htm |website=大纪元 |publisher=《开放杂志》2005年8月 |accessdate=2018-11-29}}</ref>历任[[中央办公厅|中央办公厅]]调研室副主任；<ref>{{cite web |title=夏勇简介——新任国家保密局局长夏勇是人权与宪政专家 |url=http://blog.sina.com.cn/s/blog_67c53c970102wj04.html |website=新浪网 |publisher=瞭望东方周刊/中国网2005 年9 月20 日 |accessdate=2018-11-29}}</ref>[[中央保密委员会办公室|中央保密委员会办公室]]主任、[[国家保密局|国家保密局]]局长<ref>{{cite news|title=夏勇任国务院法制办副主任|url=http://news.sina.com.cn/c/2005-09-20/03227813421.shtml|accessdate=2014-10-13|newspaper=新浪网}}</ref>。2013年3月，调任[[国务院法制办公室|国务院法制办公室]]副主任<ref>{{cite news|title=夏勇同志简历|url=http://politics.people.com.cn/GB/shizheng/252/9667/9670/6583864.html|accessdate=2014-10-13|newspaper=人民网}}</ref>。2016年7月，不再担任[[国务院法制办公室|国务院法制办公室]]副主任<ref>{{cite news||title=夏勇不再担任国务院法制办副主任，曾任职中央办公厅|url=http://www.thepaper.cn/newsDetail_forward_1501364|accessdate=2016-07-20|newspaper=澎湃新闻}}</ref>。2016年7月29日被撤销政协委员资格<ref>{{cite web|url=http://www.thepaper.cn/newsDetail_forward_1505755|title=夏勇被撤销全国政协委员资格，曾任国家保密局局长|publisher=澎湃新闻|accessdate=2016-07-28}}</ref>。
