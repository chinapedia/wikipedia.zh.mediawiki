'''夏森美'''（英文名：Sammy Haze），是[[香港|香港]]一名前經理人，目前失業。最為人熟悉的是曾經擔任香港著名女歌手[[謝安琪|謝安琪]]的經理人，以及其火爆性格為謝安琪製造大量麻煩，2011年更因其關係令謝安琪與環球緣盡<ref name="hk8n_[忽然一">{{Cite web |title=[忽然一周.直接睇]環球出招獎 謝安琪隨時飛起經理人 |trans-title= |author= |work=hk8news.blogspot.com |date= |access-date=17 October 2018 |url= http://hk8news.blogspot.com/2011/12/blog-post_1326.html |language=zh |quote=}}</ref>。夏森美於2003年與[[周博賢|周博賢]]及[[謝安琪|謝安琪]]合組公司[[Ban_Ban_Music|Ban Ban Music]]<ref>[http://the-sun.on.cc/cnt/lifestyle/20120108/00485_001.html?facebook=y 周博賢如果命運能選擇- 太陽報]</ref>。同時，他亦與[[謝安琪|謝安琪]]合組音樂公司[[邁亞音樂|邁亞音樂]]，旗下藝人包括[[梁心頤|梁心頤]]及[[goldEN|goldEN]]等。2016年，由於謝安琪退出樂壇，停止與夏森美合作至今<ref name="hk.o_唔急傾續">{{Cite web |title=唔急傾續約 夏森美同Kay再合作睇緣份 |trans-title= |author= |work=on.cc東網 |date= |access-date=17 October 2018 |url= http://hk.on.cc/hk/bkn/cnt/entertainment/20160416/bkn-20160416205942924-0416_00862_001.html |language=zh |quote=}}</ref>，並且在2018年復出時選擇加盟[[麥浚龍|麥浚龍]]而沒有與夏森美再度合作，因此夏森美正在待業中。

==簡介==
2003年，夏森美與[[周博賢|周博賢]]合作，開立獨立唱片及經理人公司Ban Ban Music，而旗下藝人包括J.O.Y.、[[Seasons_Lee|Seasons Lee]]、英師傅、[[Cousin_Fung|Cousin Fung]]及謝安琪等。<ref>[http://profileengine.com/groups/profile/426322468/ban-ban-music Ban Ban Music : The Profile Engine]</ref>

2014年5月，他與合作多年的歌手[[謝安琪|謝安琪]]開立新公司[[邁亞音樂|邁亞音樂]]<ref>[http://hk.apple.nextmedia.com/enews/realtime/20140524/52509848 【樂壇黑仔王？】謝安琪正式簽內地公司 仲會唔會唱反洗腦歌？]</ref>。其後亦相繼簽下謝安琪的師弟[[goldEN|goldEN]]及[[梁心頤|梁心頤]]，而夏森美則擔任goldEN的經理人。

2016年，由於謝安琪因計劃生第二胎，在《山林道》及「拾回演唱會」後便宣佈暫別樂壇，並且由於沒有回歸時間表，並宣佈將所有經理人公司及音樂製作公司結業，當中包括Ban Ban Music及邁亞音樂，因此兩人的合作到此為止。其後於2017年10月謝安琪宣佈全面簽約麥浚龍的[[幻國文化|幻國文化]]，並無論在音樂還是其他工作都獲得Juno的最高級別支持，令謝安琪於2018年的回歸計劃進行得非常順利之下，夏森美亦終於正式失業。

==爭議==
自謝安琪出道直至退出樂壇，夏森美已被指多次製造大量爭端，更被視為香港樂壇最麻煩的經理人之一<ref name="next_麻煩經理">{{Cite web |title=麻煩經理人 囂槳張丹惡鬥B餐夏森美｜壹週刊 |author= |work=壹週刊 |date= |access-date=17 October 2018 |url= https://nextplus.nextmedia.com/news/other/20150405/178664 |language= |quote=}}</ref>，更多次令謝安琪無辜受牽連。

2011年，謝安琪與當時的唱片公司環球唱片起爭執，有傳媒曾指出夏森美與環球方面「勢成水火」，更曾為留Kay繼續合作不斷重金投資推出新企劃務求Kay脫離夏森美，但Kay顧念舊情未有選擇，只能被雪藏直至等待完約<ref name="hk8n_[忽然一">{{Cite web |title=[忽然一周.直接睇]環球出招獎 謝安琪隨時飛起經理人 |trans-title= |author= |work=hk8news.blogspot.com |date= |access-date=17 October 2018 |url= http://hk8news.blogspot.com/2011/12/blog-post_1326.html |language=zh |quote=}}</ref>。環球高層Duncan曾指出與謝安琪的不歡而散，「完全唔關Kay事」，「只係經理人太不可理喻」，令雙方不能再合作。

2014年，夏森美曾因謝安琪與雜誌的拍照尺度問題在網絡上炮轟，換來對方的長篇回應。<ref>[http://hk.apple.nextmedia.com/enews/realtime/20140605/52554023 【人妻Kay性感出事】《Jet》回應夏森美：謝安琪著小背心冇say no]</ref>

2014年，由於謝安琪因公司關係未能出席好友[[Eric_Kwok|Eric Kwok]]的作品展音樂會，惹來對方微言。而由於謝安琪的首本名曲《[[囍帖街_(歌曲)|囍帖街]]》正是由Eric作曲，Eric對她的事業發展有一定幫助。但當Eric批評相關安排的報導出街後，夏森美卻惡言相向，令雙方產生衝突。而事實上，謝安琪與Eric份屬好友，Eric亦表明只是開玩笑，並非真心批評她<ref>[http://hk.apple.nextmedia.com/enews/realtime/20141218/53244326  【Iron Man淆底？】「永不寫歌俾謝安琪」 Eric Kwok：玩啫，唔好咁認真！]</ref>，而她亦多番表示想出席，但因公司方面安排而未能出席。此事令不少歌迷對夏森美不滿。<ref>[https://hk.celebrity.yahoo.com/news/%E7%B6%B2%E6%B0%91%E5%96%AA%E6%8F%92%E7%B6%93%E7%90%86%E4%BA%BA-%E7%B4%AF%E6%AD%BBkay-bb-225043829.html 網民喪插經理人 累死Kay BB]</ref>

由於很多謝安琪的歌迷都認為夏森美處理謝安琪的事務的表現欠奉，例如2011年因他與環球唱片不和而令謝安琪犧牲（此事件獲環球高層Duncan認證），令謝安琪只能與環球解約。在失去大公司的支持下的謝安琪在整個2012年處於真空狀態，演唱會亦因欠缺宣傳及贊助合作等而只能開兩場，較2009年獲環球唱片大搞而大開7場大相逕庭。謝安琪只好在2013年加盟細公司星煥，其後卻因公司改組成星夢只能再度解約，再次浪費一年，且在整個2013年只能推出一首主打歌《最好的時刻》。

2014年，謝安琪的《[[Kontinue|Kontinue]]》大熱，當中的歌曲《[[雞蛋與羔羊|雞蛋與羔羊]]》更在推出不久並獲過百萬點擊，卻因夏森美的個人決定否決了將此曲派台，令謝安琪未能乘勢而再上高峰，此事令謝安琪的歌迷感到相當惋惜。2016年謝安琪終於找到機會，推出大熱《[[山林道|山林道]]》後便宣佈退出，並將所有經理人合約解約，終於終結了與夏森美的合作<ref name="hk.o_唔急傾續">{{Cite web |title=唔急傾續約 夏森美同Kay再合作睇緣份 |trans-title= |author= |work=on.cc東網 |date= |access-date=17 October 2018 |url= http://hk.on.cc/hk/bkn/cnt/entertainment/20160416/bkn-20160416205942924-0416_00862_001.html |language=zh |quote=}}</ref>，使得夏森美失業至今。
