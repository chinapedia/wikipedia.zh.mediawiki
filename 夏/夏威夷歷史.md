'''夏威夷歷史'''是構成[[美利堅合衆国|美利堅合衆国]]第50個州的[[夏威夷州|夏威夷州]]的[[夏威夷群島|夏威夷群島]]的歷史。

夏威夷在史前時代，渡過[[太平洋|太平洋]]的[[波利尼西亞人|波利尼西亞人]]在這里過著傳統的生活。1778年[[詹姆斯·庫克|詹姆斯·庫克]]發現夏威夷群島之後，夏威夷被捲入了近代化的大潮之中。經過了各島之間的內戰，建立了歷史達100年的統一國家[[夏威夷王國|夏威夷王國]]，隨著和歐美人開始接觸，社會也開始快速變化。19世紀前期確立了宗教基礎和希望實現經濟發展的歐美殖民者及他們的後裔，逐漸開始爲了確保經濟權益而開始追求政治權力，其影響力也逐漸加強。在甘蔗種植園和其交易中獲得大量土地和收益的成功者們為追求產業發展開始從[[日本|日本]]等地大量召集廉價勞動力，夏威夷社會混合了多種人種，孕育出複雜的文化。

1893年白人勢力[[夏威夷王國滅亡|推翻夏威夷王國]]，以近代化為藉口于1900年將夏威夷合併入[[美利堅合衆國|美利堅合衆國]]領土。並且受美國和日本勢力的抬頭及其對立的激化的影響，夏威夷作為太平洋上重要的軍事據點得到了開發，以勞動者身份前往夏威夷的大量[[夏威夷日本人移民|日本人移民]]受到了嚴重的歧視。夏威夷雖然作為現代化觀光都市實現了發展，然而夏威夷也面臨開發伴隨的環境污染、歷史遺跡的破壞以及人口增加引發的地價和物價高漲，[[夏威夷事務協會|夏威夷事務協會]]所提倡的夏威夷人獲得自治權等眾多問題。夏威夷也在摸索這些問題的解決方案。

== 史前時代 ==
夏威夷的島嶼是因火山活動而從海底隆起形成的，西北部較早形成的島嶼約在500萬年前至100萬年前，[[夏威夷島|夏威夷島]]等較新的島嶼則是在約50萬年前形成<ref>[[#後藤2004|後藤，2004，p.16]]</ref>。

== 夏威夷王国 ==
1795年[[卡美哈梅哈一世|卡美哈梅哈一世]]建立夏威夷王国，是为卡美哈梅哈王朝。1893年[[夏威夷王國滅亡|夏威夷王國滅亡]]。

== 美国领土 ==
1894年[[桑福德·多尔|杜亨]]任夏威夷共和國臨時政府的首任總統，并希望能够加入美国。克利夫兰总统很快就与夏威夷新政府代表达成兼并条约并其递交联邦参议院批准。

1897年3月，[[威廉·麦金莱|威廉·麦金莱]]担任美国总统。他开始着手准备合并夏威夷。次年美國正式將夏威夷合併。

1941年12月7日[[日本|日本]]海军航空兵袭击夏威夷的美国太平洋舰队基地[[珍珠港|珍珠港]]，导致[[太平洋战争|太平洋战争]]爆发、美国对日宣战，随后美国在夏威夷设立军政府。

1946年9月爆发长达11年的民主革命。

1959年8月21日夏威夷正式成為美國第50個州。

2013年11月13日夏威夷通過同性婚姻法案，成為美國第15個同性婚姻合法的州，得到美國總統[[歐巴馬|歐巴馬]]的讚賞：「我一直都以在夏威夷出生為榮，今天的表決結果讓我倍感光榮！」

2018年1月13日發生[[2018年夏威夷彈道飛彈誤報|彈道飛彈誤報事件]]，引起公眾恐慌，州長[[大衛·伊藝|大衛·伊藝]]公開道歉。
