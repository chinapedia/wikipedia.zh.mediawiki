{{engname|《夏天協奏曲》|Summer Times}}是一部由[[黃朝亮|黃朝亮]]執導的台灣電影，並由[[張睿家|張睿家]]、[[林逸欣|林逸欣]]和[[林美秀_(演員)|林美秀]]等人主演。本片在[[台灣|台灣]]於2009年11月27日上映。下片後於2010年4月23日起於台灣各影視出租店開始出租，並於2010年5月18日起於台灣各通路開始販售。

== 劇情大綱 ==
夏天是夏日精靈回到[[金門|金門]]的季節，阿寬在[[網路|網路]]上看到全國鋼琴大賽第一名的陳文青。
透過在酒廠上班的阿輝伯，阿寬偶然見到了來金門度假的小青，彼此產生了初戀的情愫。

小青喜歡用[[手機|手機]]紀錄拍照，在金門期間，和阿寬、沙蟲三人玩在一起，
一起去[[坑道|坑道]]探險，一起在鄉間道路上騎[[腳踏車|腳踏車]]奔馳。
小青每天練琴的時候，阿寬就在屋外聆聽，那是他每天最幸福的時刻。

阿寬生活清苦，自覺無法配上小青，他刻意的逃避，最後終於讓小青勇敢去愛的純真給融化了。
小青跟阿寬過完最美好的暑假，回去之前，小青承諾她會天天給阿寬[[E-mail|E-mail]]，
也承諾明年夏天，她一定會回到他身邊。

隔年夏天，小青按照約定回來了！但是阿寬卻對小青開始有著不同的感覺……。
一個夏天的真愛，是否勝過一輩子的等待？

== 拍攝契機 ==
導演黃朝亮於製作八大電視台《大冒險家》外景節目時，因偶然接觸到金門棲息的[[鱟|鱟]]及[[栗喉蜂虎|栗喉蜂虎]]這兩種皆具有一公一母成雙成對習性的特有生物，故使其開始有把這樣的浪漫的習性幻化成愛情故事的構想。

== 相關資訊 ==
本片全程在金門取景拍攝，劇情中結合了金門的特殊生物、文化產業及傳統聚落等外島的夏天美景。同時本片也受前金門縣長[[李炷烽|李炷烽]]推崇肯定，並親身代言行銷金門。原預定於2009年8月8日-9日於金門社福館演藝廳當地公開播映，但遇到[[莫拉克颱風|莫拉克颱風]]侵台，致使影片未能如期運抵金門，延期播映。

== 劇情特有戰地風情 ==
由於金門一地曾發生過[[八二三砲戰|八二三砲戰]]為主的大小戰役，在早期為[[中華民國|中華民國]]與[[中共|中共]]的對峙前線，因此形成了特有戰地景點觀光文化，片中也受其影響呈現出許多戰地風情。例如，小青與阿寬在海邊對話時，台詞中出現金門當地特有的「反兩棲登陸椿」及「地雷」等軍武工事比喻、及小青不離身的項鍊，中心部為彈殼所製等等。包括劇中坑道裡沙蟲跟阿寬耍寶的《[[金門高中|金門高中]]校歌》，亦充滿濃厚的軍教意味。

== 劇中出現的金門動物==
=== [[栗喉蜂虎|栗喉蜂虎]] ===
擁有色彩鮮豔的羽毛，金門當地稱為「夏日精靈」的候鳥。每當夏季時飛渡至金門避寒。栗喉蜂虎的天性為一公一母的一起行動，每當飛至金門，也提示著高梁成熟的日子接近。

=== [[鱟|鱟]] ===
亦為一公一母行動的天性，公的會附在母的甲殼上行動。金門當地漁民都知道，抓鱟一定要抓雙不抓單，因為不論抓走公的還是母的，另外一隻會因為找不到伴侶而死去，故在金門有「夫妻魚」的美喻。

=== [[方格星蟲|沙蟲]] ===
在金門沙岸中棲息的一種生物。為當地作為菜餚的[[食材|食材]]之一。

== 相關影響事件 ==
西元2010年6月《夏天協奏曲》受海峽兩岸論壇的邀約，至大陸作巡迴展映後大受好評，因此契機促成兩岸（台灣獨角先傳播及大陸海峽世紀傳播）攜手將電影改編為30集的電視[[偶像劇|偶像劇]]的合作計劃。電視劇主要拍攝地仍以'''金門'''為主、場景並增加了[[台北|台北]]、[[廈門|廈門]]、[[福州|福州]]等地，將呈現兩岸四地的地方文化特色。電視劇《[[戀夏38℃|戀夏38℃]]》於西元2011年9月開始拍攝，並於2012年10月31日播映。
