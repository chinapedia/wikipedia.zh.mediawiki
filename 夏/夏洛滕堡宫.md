'''夏洛滕堡宫'''（{{lang-de|Schloss Charlottenburg}}）是[[德国|德国]][[柏林|柏林]]现存最大的宫殿。它座落于夏洛特堡和威尔默斯多夫大区下辖的[[夏洛特堡|夏洛特堡]]区。

== 历史 ==
[[File:Schloss_Charlottenburg_Garten.JPG|thumb]]
夏洛滕堡宫最初名为“Lietzenburg”，为[[巴洛克建筑|意大利巴洛克式风格]]，由[[勃兰登堡选帝侯|勃兰登堡选帝侯]][[腓特烈一世_(普鲁士)|腓特烈一世]]的妻子[[索菲·夏洛特|索菲·夏洛特]]委托建筑师Arnold Nering设计。1701年，腓特烈一世加冕为普鲁士国王，称腓特烈一世，夏洛特成为他的王后。这个最初作为柏林郊外的夏天休息地，由建筑师Eosander von Göthe 扩建为雄伟的建筑。夏洛特在1705年去世后，腓特烈一世为了纪念她，将这座宫殿和附属产业命名为夏洛滕堡。从1709年到1712年进行了进一步扩建，这一时期出现了有特色的转塔和橘园。

夏洛滕堡宫内部曾被描述为“世界第八大奇迹” — [[琥珀室|琥珀室]]这个房间的墙壁装饰着[[琥珀|琥珀]]。这个想法来自[[但泽|但泽]]和[[柯尼斯堡|柯尼斯堡]]，1701至09年哥特弗里德 Wolffram、恩斯特沙赫特和哥特弗里德 Turau 在那里编写计划。这个房间在安德烈 Schlüter的监督下完成。

1713年腓特烈一世去世后，[[腓特烈·威廉一世|腓特烈·威廉一世]]继位，夏洛滕堡宫进入了一个新的时期。1716年，他将琥珀室作为礼物赠送给[[沙皇|沙皇]][[彼得大帝|彼得大帝]]。然而，他在1740年去世后，新国王[[腓特烈二世_(普鲁士)|腓特烈二世]]允许格奥格·克诺贝尔斯多夫扩建夏洛滕堡的东翼。随后，腓特烈二世对夏洛滕堡宫的兴趣转向了[[波茨坦|波茨坦]]的[[无忧宫|无忧宫]]（完成于1747年）。

夏洛滕堡宫最终完成于[[腓特烈·威廉二世|腓特烈·威廉二世]]时期，完成了西部宫廷剧院和卡尔哥达汉斯的小橘园。

== 花园 ==
[[File:Schloss-Charlottenburg_Schlosspark.jpg|thumbnail]]
17世纪末，其花园为法国风格，18世纪末，部分花坛改成英式园林。花园内设有1788年卡尔哥达汉斯的宫廷茶馆，1824年至1825年那不勒斯别墅式的新馆，和1810年建造的路易丝王后的陵墓。

== 现代用途 ==
在第二次世界大战中，夏洛滕堡宫受损严重。1945年以后重建，改为一座博物馆（或用作观光用途）。过去的剧院是史前和早期历史博物馆，小橘园用作餐厅。

其花园是一个大型娱乐公园，服务于附近人烟稀少的夏洛滕堡老城。不过，有计划建立一个门票的公园。

从2004年到2006年初，当[[貝爾維尤宮|貝爾維尤宮]]装修期间，夏洛滕堡宫用作德国总统府。
