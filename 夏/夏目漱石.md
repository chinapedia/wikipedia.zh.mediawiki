'''夏目漱石'''（{{bd|1867年|2月9日|1916年|12月9日}}<ref name=":0">{{cite book |author=夏目漱石|others= 陳苑瑜譯|title=心|year=2010|publisher=立村文化|isbn=9789866283093}}</ref>），本名'''夏目金之助'''（{{jpn|j=夏目 金之助|hg=なつめ きんのすけ|rm=Natsume Kinnosuke|link=no}}），[[日本|日本]][[明治|明治]]至[[大正|大正]]時代的[[作家|作家]]、時事評論家、英文學者。

== 名字的由來 ==
'''漱石'''這個名字取自於唐代《[[晋書|晋書]]》中關於[[西晉|西晉]]時期[[孫楚_(西晉)|孫楚]]「[[漱石枕流|漱石枕流]]」的故事。最初是作為[[號|號]]使用，後來直接用作名字使用。

== 生平 ==
夏目漱石，生於[[江户|江户]]的牛込馬場下橫町（今 [[東京都|東京都]][[新宿區|新宿區]][[喜久井町|喜久井町]]）一个小吏家庭，是家中末子。1874年，七岁，入浅草寿町的户田学校。夏目自幼喜欢汉学，14岁开始学习中国古籍，少年时曾立志以汉文出世。<ref name=":0" />

23岁，夏目进入[[东京帝国大学|东京帝国大学]]英文系（现[[东京大学|东京大学]]）。1889年，就学期间的夏目因受好友[[正冈子规|正冈子规]]等人影响而开始写作<ref name=":0" />；同年9月9日，他的汉文暑假游记《木屑录》脱稿。《木屑录》不仅是漱石最早汇集成册的作品，且署名为“漱石顽夫”。漱石二字来源于唐代《[[晋书|晋书]]》的故事漱石枕流，一开始是子规的笔名，为夏目借用，并最终成为了他的正式笔名“夏目漱石”。1895年，在[[爱媛县|爱媛县]]松山中学任教，后又去[[熊本县|熊本县]]执教<ref name=":0" />；这段经历后来体现在他的小说《少爺》中。1896年，夏目在家人的安排下与贵族院书记官长中根重一的长女镜子结婚。<ref name=":0" />这一时期夏目在俳句界较为活跃声名鹊起。

1899年10月28日，33岁的夏目抵达[[伦敦|伦敦]]，开始了在英国[[倫敦大學學院|倫敦大學學院]]的留学生活。1902年12月5日，夏目啟程归国，結束为期两年的留学生活。夏目归国后，在东京帝大讲授英文，并开始文学创作。1905年的《[[我是猫|我是猫]]》令他一举成名。1907年，开始为《[[朝日新闻|朝日新闻]]》写连载小说（包括《虞美人草》、《三四郎》）。

暮年的夏目漱石追求“则天去私”的理想。1911年，他曾拒绝接受政府授予的博士称号。<ref name=":0" />

1916年，因[[胃溃疡|胃溃疡]]去世。夏目死后，家屬将他的脑和胃捐赠给东京帝大的医学部。他的脑至今仍保存在东京大学。1984年，他的头像被印在日元1000元的纸币上。

== 地位 ==
[[File:1000_yen_Natsume_Soseki.jpg|thumb]]上的夏目漱石頭像]]

夏目漱石在日本近代文学史上享有很高的地位，被称为“国民大作家”。他对东西方的文化均有很高造诣，既是英文学者，又精擅[[俳句|俳句]]、[[汉诗|汉诗]]和[[书法|书法]]。写作小说时他擅长运用[[对句|对句]]、[[迭句|迭句]]、幽默的语言和新颖的形式。他对个人心理的精确细微的描写开了后世[[私小说|私小说]]的风气之先。他的门下出了不少文人，[[芥川龍之介|芥川龍之介]]也曾受他提携。
