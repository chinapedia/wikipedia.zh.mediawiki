《'''夏色'''》（なついろ）是[[日本|日本]]民谣二重唱、[[日本流行音樂|J-Pop]]组合[[ゆず|柚子]]（ゆず）的首张单曲。1998年6月3日由其所属公司{{link-ja|Senha&Company|セーニャ・アンド・カンパニー}}发售。<ref>{{Cite web|url =http://www.senha-yuzu.jp/discography/single/index10.php|title =ゆず Official Website {{!}} ディスコグラフィ {{!}} シングル|accessdate =2015-07-05|author =|date =|publisher =|pages =10|language =ja|deadurl =yes|archiveurl =https://web.archive.org/web/20150706144939/http://www.senha-yuzu.jp/discography/single/index10.php|archivedate =2015-07-06}}</ref>

本曲也被用作[[京濱急行電鐵|京急]][[上大岡車站|上大岡站]]的列車接近時的提示音樂。<ref>{{Cite web|title = くるり、ゆず、EXILE他の名曲が京急線ホームに響く|url = http://natalie.mu/music/news/11054}}</ref>

== 销量 ==
同年7月的第一周，销量进入[[Oricon公信榜|oricon]]榜前30位；9月第一周进入前20位，第三周达到最高17位。<ref>{{Cite web|url = http://www.oricon.co.jp/prof/30824/products/225292/1/|title = 夏色 {{!}} ゆず {{!}} ORICON STYLE|accessdate = 2015-07-05|author = |date = |publisher = |language = ja}}</ref>

2011年 [[iTunes_Store|iTunes Store]] 年度销量仍达79位。<ref>{{Cite web|url = http://www.barks.jp/news/?id=1000075375&page=2|title = 2011年に最もダウンロードされた作品は？ 「iTunes Rewind 2011」発表 {{!}} Apple {{!}} BARKS音楽ニュース|accessdate = 2011-12-09|author = |date = |publisher = |language = ja}}</ref>
