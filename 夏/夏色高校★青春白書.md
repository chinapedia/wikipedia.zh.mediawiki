{{Vgname|夏色高校★青春白書|ja=夏色ハイスクル★青春白書(略)}}<ref group="註">原名稱為《夏色ハイスクル★青春白書〜転校初日のオレが幼馴染と再会したら報道部員にされていて激写少年の日々はスクープ大連発でイガイとモテモテなのに何故かマイメモリーはパンツ写真ばっかりという現実と向き合いながら考えるひと夏の島の学園生活と赤裸々な恋の行方。〜》</ref>是[[開放世界|開放世界]]類型遊戲，由{{le|Tamsoft|Tamsoft}}開發，{{le|D3 Publisher|D3 Publisher}}發行。2015年6月4日，於[[PlayStation_3|PlayStation 3]]和[[PlayStation_4|PlayStation 4]]平台發售，遊戲定位為「開放世界高中戀愛冒險」。

== 玩法 ==
玩家將扮演在「夢之島高中」就讀的男生，其高中在虛構島嶼夢之島上。玩家要完成島上其他人給予的任務，拍攝女學生的內褲照片給校內的報導部。玩家若被其他人發現，他們會將告訴警察或老師，最終會被關押或輔導室進行輔導<ref name="GamePlay"/>。

== 消息 ==
遊戲最早於2014年11月20日在《[[Fami通|Fami通]]》發佈消息，稱遊戲將於[[PlayStation_3|PlayStation 3]]與[[PlayStation_4|PlayStation 4]]平台發售<ref name="FamiTsu-1"/><ref name="D3P-PS3PS4"/>。約一星期後，{{le|D3 Publisher|D3 Publisher}}公佈遊戲大綱與四位女主角信息<ref name="D3P-4Girls"/>。2014年12月4日出版的《Fami通》公佈三位新女性角色<ref name="D3P-3Girls"/>。2014年12月10日，D3 Publisher在YouTube發佈第一部作品預告片，以及由[[三上枝織|三上枝織]]與[[洲崎綾|洲崎綾]]演唱的歌曲「{{lang|ja|夏色バタフライ☆}}」<ref name="D3P-Youtube"/>。2015年2月，遊戲公佈發售日期，並公佈十三名新角色以及配音員等情報<ref name="Game-13"/>。2015年5月15日發佈長達8分鐘的新預告片<ref name="Game-8min"/>。而製作人岡島信幸表示，遊戲的靈感來源於角色扮演以及開放世界類型的[[上古卷轴V：天际|上古卷轴V：天际]]<ref name="OkajimaP"/>。

== 音樂 ==
遊戲的片頭曲是「{{lang|ja|夏色バタフライ☆}}」，片尾曲為「{{lang|ja|記憶の中のリグレット}}」，兩首歌曲都是三上枝織與洲崎綾演唱，並於2015年6月4日發售CD<ref name="ThemeMusic-OPED"/>。

== 評价與發售 ==
{{Video game reviews
| Fam = 30/40<ref name="FamiTsu-Score"/>
}}
日本遊戲雜誌《Fami通》以40分滿分，《夏色高校★青春白書》其取得了30分<ref name="FamiTsu-Score"/>。

[[PlayStation_4|PlayStation 4]]與[[PlayStation_3|PlayStation 3]]版在遊戲發售首週內，於日本的銷售榜分別排名第三（13,868）以及第十（6,772）<ref name="PS3PS4-R"/>。而遊戲發售首週，PlayStation 4版於日本[[PlayStation_Network|PlayStation Network]]中的下載量為首位<ref name="PS4-R1"/>，PlayStation 3版於首週下載量為第四<ref name="PS3-R4"/>。
