'''夏綬草'''（[[学名|学名]]：''{{lang|la|Spiranthes aestivalis}}''）是[[蘭科|蘭科]][[綬草屬|綬草屬]]的[[多年生|多年生]][[宿根性|宿根性]][[草本|草本]]植物，是一种较为稀有的原生兰，分布於欧洲中部和南部以及欧洲北部的[[荷兰|荷兰]]、[[英格兰|英格兰]]南部，南至[[摩洛哥|摩洛哥]]，生长在海拔1,800米以下的地区，常见於在贫瘠的沼泽和荒野上。

== 特征 ==
夏綬草由2－6个贮藏块根，长5－8厘米，茎高10－35厘米，生长有3－5片玫瑰形叶。

基生叶3－6片，淡绿色，线披针形，长5－14厘米，宽0.6－1.2厘米，修长挺直。花序长3－10厘米，由5－20枚小白花松散地构成，螺旋盘绕在花茎上。

花期为7月初至8月末，盛花期为7月末至8月初。

== 栖息地 ==
夏綬草喜生长在湿润[[钙华|钙华]]腐殖土和湿地植物[[草甸|草甸]][[群丛|群丛]]中。典型的共生植物有[[赤箭莎属|赤箭莎属]]的''Schoenus ferrugineus''和''Schoenus nigricans''。

== 保护和威胁 ==
夏綬草和其他兰花一样受到欧洲和国家法律的保护。在[[德国|德国]]，[[巴伐利亚|巴伐利亚]]以外的地区很少能见到夏綬草，仅能在[[巴登-符腾堡|巴登-符腾堡]]见到。

此物种有灭绝威胁，主要原因是泥炭地及湿草甸开垦造成的栖息地丧失。
