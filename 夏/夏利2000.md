'''夏利2000'''（又称'''TJ7136U型轿车'''，或称'''夏利2000世纪广场轿车'''）是中国[[天津一汽夏利汽车股份有限公司|天津一汽夏利汽车]]与日本[[丰田汽车|丰田汽车]]合作生产的一款四门三厢轿车，产品周期自2000年起到2007年止，并先后发展出雅酷、威乐一共三个轿车产品。根据'''天津一汽'''先前与日本[[大发工业株式会社|大发工业株式会社]]合作生产的并命名的'''夏利'''品牌和投产年份，为中国市场生产的这款[[w:Toyota_Platz|Toyota Platz]]命名夏利2000。

== 历史 ==

夏利2000型轿车于2000年12月14日下线，2001年1月在全国正式上市销售。该车由天汽集团公司和日本丰田汽车公司联合开发，日本丰田汽车公司提供技术援助，天津汽车夏利股份有限公司组织生产，天津汽车工业销售公司负责销售<ref>{{cite web |url=http://www.faw.com.cn/faw_online/news//xwzx/mtjj/20120919084600042.htm?pros=2 |publisher=中国一汽 |title=持久创新 夏利塑造品牌经典 |accessdate=2014-09-09 |archive-url=https://web.archive.org/web/20160304122814/http://www.faw.com.cn/faw_online/news//xwzx/mtjj/20120919084600042.htm?pros=2 |archive-date=2016-03-04 |dead-url=yes }}</ref>。夏利2000采用丰田在当时天津生产的最先进的8A发动机，其它基本采用进口零部件组装生产。

夏利2000轿车装用天津丰田发动机公司生产的8A-FE型发动机，排量1.342L，为双顶置凸轮轴、16气门、闭环多点燃油喷射技术发动机<ref name="ISBN 7-111-09804-8">{{cite book|title=夏利2000轿车使用与维修手册 |author=张凤山 |coauthors=王蕾、张风亭、静永臣、王颖 |publisher=机械工业出版社 |ISBN=7-111-09804-8 |editor=2002年3月第1版第1次印刷 |date=2002年3月 |pages=第1页}}</ref>。

2007年所有天津一汽合作生产的丰田NBC产品全部停产<ref>{{cite web|url=http://news.xinhuanet.com/auto/2007-07/24/content_6421211_4.htm |title=不符合乘用車燃料消耗量標準的産品 |publisher=新华网 |accessdate=2014-09-29 }}</ref>，产能主要转换为天津一汽丰田威驰。

== 配置本地化 ==

=== 减配 ===

=== 方向盘 ===
夏利2000减去了方向盘助力<ref>{{cite web|url=http://www.hangzhou.com.cn/20030101/ca307161.htm |title=夏利2000的三大失误 |publisher=杭州网 |accessdate=2014-09-30}}</ref>，令该车倒桩、进车位等操作不便。

==== 发动机 ====
Platz 在北美、日本等市场使用丰田 VVT-I 发动机，但是天津一汽却使用了相对落后的丰田8A发动机。

==== 气囊 ====
夏利2000和其后来的雅酷仅在驾驶员侧配备一个气囊，然而在丰田全球其他市场同款车配备驾驶员、副驾驶员2气囊。

=== 本地化配备===

在天津投产的初期，夏利2000原配备六辐钢圈轮毂。随着国产化率的提高，后改用天津一汽自行研发和测试的八辐铝合金轮毂。

== 市场反馈 ==

夏利2000的营销令其在中国上市后销量一直不佳，有人认为它的命名未能摆脱夏利低端车的形象而影响购买意欲<ref>{{cite web|url= |title=夏利2000难改夏利形象,品牌知名不等于成功 |accessdate=2014-09-29 |publisher=新浪网 |url=http://auto.sina.com.cn/news/2001-02-05/4238.shtml}}</ref>。

按2000年至2001年间人民币对日元汇率<ref>{{cite web|url=http://srh.bankofchina.com/search/whpj/search.jsp |title=中国银行外汇牌价}}</ref>计算，100日元大约为6-7元人民币，当时售价150万日元的 Platz 应对应销售人民币9-10.5万元，但是天津一汽却按照132980元定价<ref>{{cite web|url=http://auto.sohu.com/20001215/file/0224,020,100010.html |title=夏利2000型明年1月正式发售 |publisher=搜狐网 |accessdate=2014-09-29}}</ref>，明显高于其实际价值。

== 后续产品 ==
===天津一汽雅酷===
与前作夏利2000相比，雅酷为仅有配置有自动变速器的换代车型。该车在2002年10月<ref>{{cite web |title=天津一汽雅酷就是夏利2000自动挡 |url=http://auto.enorth.com.cn/system/2002/10/18/000436827.shtml |publisher=北方网 |accessdate=2014-09-29 |deadurl=yes |archiveurl=https://web.archive.org/web/20141006102406/http://auto.enorth.com.cn/system/2002/10/18/000436827.shtml |archivedate=2014-10-06 }}</ref>至2003年12月之间投产，弃用夏利车标并使用天津一汽标识，全部使用8辐铝合金轮毂，但依然使用8A发动机并且仅有一具安全气囊。

===天津一汽威乐===

威乐（FAW VELA CA7156）于2004年2月上市<ref>{{cite web|url=http://auto.163.com/special/y/yiqivela_testdrive.html |title=网易试驾：卡通“胖小子” 天津一汽威乐1.5自动挡 |publisher=网易汽车版 |accessdate=2014-09-29}}</ref>。此车有4速自动变速器和5速手动变速器两款，均使用天津一汽生产的1.5升5A-FE发动机，并配备防盗芯片一体式遥控钥匙控制车门。
