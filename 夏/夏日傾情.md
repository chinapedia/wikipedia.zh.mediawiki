《'''-{SUMMER NUDE}-'''》為日本[[富士電視台|富士電視台]]於[[富士電視台週一晚間九點連續劇|月九時段]]（[[星期一|週一]] 21:00 - 21:54，[[日本標準時|JST]]）播出的[[電視連續劇|電視連續劇]]，由2013年7月8日起開始播出。此劇由[[山下智久|山下智久]]主演。

《-{SUMMER NUDE}-》以由{{link-ja|真心Brothers|真心ブラザーズ}}在1995年發行的著名歌曲《-{[[SUMMER_NUDE_(歌曲)|SUMMER NUDE]]}-》（{{lang|ja|サマーヌード}}）為印象而製作。這部戀愛連續劇的故事描述將無法忘懷的過往戀愛藏著心中的年輕人，在[[夏日|夏日]]發生的愛情故事。<ref name="fujitv130614">{{cite web |url=http://www.fujitv.co.jp/fujitv/news/pub_2013/i/130614-i097.html |language=ja |title=<span lang="ja">山下智久フジテレビ主演ドラマで初の主題歌！「サマーヌード」をカバー『-{SUMMER NUDE}-』</span> |publisher=[[富士電視台|富士電視台]] |date=2013-06-14 |accessdate=2013-07-09 |deadurl=yes |archiveurl=https://web.archive.org/web/20130719125259/http://www.fujitv.co.jp/fujitv/news/pub_2013/i/130614-i097.html |archivedate=2013-07-19 }}</ref>這首歌也獲起用為主題曲，由主演此劇的山下智久[[翻唱|翻唱]]。<ref name="fujitv130614"/>值得一提的是，香港譯名《夏日傾情》亦取自歌名，為[[黎明|黎明]]於1993年發行的同名歌曲，雖然與原名意思不同，但歌曲內容卻與故事不謀而合。

== 概要 ==
《-{SUMMER NUDE}-》是[[山下智久|山下智久]]繼2010年1月至3月的《[[Code_Blue救護直升機醫生|Code Blue救護直升機醫生]]》第二季後，再次主演[[富士電視台週一晚間九點連續劇|富士電視台月九劇]]。山下智久跟在劇中飾演女主角的兩名演員[[香里奈|香里奈]]和[[戶田惠梨香|戶田惠梨香]]都有合作經驗。山下跟香里奈曾於2001年在《[[正義代書戰士|正義代書戰士]]》一劇中飾演戀人，《SUMMER NUDE》是他們第四次合作；山下智久跟戶田惠梨香上一次合作則是在《Code Blue救護直升機醫生》，這次合作也是第四次<ref name="oricon130527">{{cite web|url=http://www.oricon.co.jp/news/movie/2024972/full/ |language=ja |title=<span lang="ja">山下智久、3年ぶり“月9”主演　香里奈・戸田恵梨香と王道ラブストーリー</span> |publisher=[[Oricon|Oricon]] |date=2013-05-27 |accessdate=2013-07-09}}</ref>。香里奈和戶田皆有月九的主演經驗，香里奈曾主演2011年10月期的《[[我無法戀愛的理由|我無法戀愛的理由]]》，亦在2012年10月期的《[[Priceless無價人生|Priceless無價人生]]》劇中與木村拓哉共演，飾演女主角；戶田則曾主演2011年1月期的《[[教我愛的一切|教我愛的一切]]》，以及在2012年4月期的《[[上鎖的房間_(電視劇)|上鎖的房間]]》飾演女主角。<ref name="oricon130527" />

劇中故事圍繞着山下智久、香里奈和戶田惠梨香的三角關係進行。在劇中飾演主角前女友的[[長澤雅美|長澤雅美]]前一次演出月九劇已是6年前的《[[求婚大作戰|求婚大作戰]]》，當時跟山下智久飾演情侶，長澤因此表示「相當懷念」。<ref>{{cite web |url=http://www.sponichi.co.jp/entertainment/yomimono/drama/2013/07_09/summer_nude/kiji/K20130611005996150.html |language=ja |title=<span lang="ja">長澤まさみ　６年ぶり“月９”出演！山Ｐと共演「懐かしいなぁ」</span> |publisher=Sponichi Annex |date=2013-06-11 |accessdate=2013-07-09 |deadurl=yes |archiveurl=https://web.archive.org/web/20130728123807/http://www.sponichi.co.jp/entertainment/yomimono/drama/2013/07_09/summer_nude/kiji/K20130611005996150.html |archivedate=2013-07-28 }}</ref>此外，飾演CAFE & Bar店長的{{link-ja|高橋克典|高橋克典}}是繼1996年的《{{link-ja|天使之愛|ピュア (テレビドラマ)}}》後，相隔17年半再次出演月九劇，也是首次跟山下智久和飾演他妻子的[[板谷由夏|板谷由夏]]等人同台演出。<ref>{{cite web |url=http://www.sponichi.co.jp/entertainment/yomimono/drama/2013/07_09/summer_nude/kiji/K20130617006031560.html |language=ja |title=<span lang="ja">高橋克典、１７年半ぶり「月９」出演！山Ｐらと初共演</span> |publisher=Sponichi Annex |date=2013-06-17 |accessdate=2013-07-09 |deadurl=yes |archiveurl=https://web.archive.org/web/20130728150544/http://www.sponichi.co.jp/entertainment/yomimono/drama/2013/07_09/summer_nude/kiji/K20130617006031560.html |archivedate=2013-07-28 }}</ref>
