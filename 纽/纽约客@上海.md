《'''纽约客@上海'''》（{{lang-en|'''Shanghai Calling'''}}）中国大陆和美国联合制作的浪漫爱情片，导演和编剧都是夏伟，主演丹尼尔·海尼、伊丽莎·库普。<ref>{{cite web |url=http://ent.sina.com.cn/m/c/2012-07-10/14233681247.shtml?bsh_bid=107061669 |title= 朱珠处女秀《纽约客@上海》挑暑期档|accessdate=2012 |publisher=新浪网 |work= |language=zh-hans}}</ref><ref>{{cite web |url=http://news.hexun.com/2013-09-18/158113621.html |title=《纽约客@上海》： 人生如鸭 |accessdate=2013 |publisher= 和讯网|work= |language=zh-hans}}</ref><ref>{{cite web |url=http://ent.sina.com.cn/m/c/2012-08-24/16493721295.shtml |title=《纽约客@上海》：美国人在上海的故事 |accessdate=2012 |publisher=新浪网 |work= |language=zh-hans}}</ref>

==剧情==
该片讲述21世纪的[[上海市|上海市]]的两名外国商人的浪漫爱情故事。
