'''纽卡斯尔机场'''（{{lang-en|'''Newcastle Airport'''}}，{{airport codes|NCL|EGNT}}）是[[英国|英国]][[泰恩河畔纽卡斯尔|泰恩河畔纽卡斯尔]]的一座国际机场，位于市中心西北约9.8公里的[[沃尔辛顿|沃尔辛顿]]。<ref>{{cite web |url=http://www.nats-uk.ead-it.com/public/index.php%3Foption=com_content&task=blogcategory&id=104&Itemid=153.html|title=Newcastle - EGNT|publisher=NATS|language=en|accessdate=2010-03-29}}</ref>该机场在2009年英国最繁忙的机场中排名第12位。<ref name="stats">{{cite web |url=http://www.caa.co.uk/default.aspx?catid=80&pagetype=88&sglid=3&fld=2009Annual |title=UK Airport Statistics: 2009 - annual |publisher=Civil Aviation Authority|language=en|accessdate=2010-03-29}}</ref>

虽然纽卡斯尔机场位于纽卡斯尔，但该机场实际由7个地方当局（51%）和[[哥本哈根机场|哥本哈根机场]]（49%）所拥有。7个地方当局分别为：[[达勒姆郡|达勒姆郡]]、[[盖茨黑德|盖茨黑德]]、[[泰恩河畔纽卡斯尔|泰恩河畔纽卡斯尔]]、[[北泰因赛德|北泰因赛德]]、[[诺森伯兰郡|诺森伯兰郡]]、[[南泰因赛德|南泰因赛德]]和[[桑德兰|桑德兰]]。

纽卡斯尔机场拥有[[英国民航管理局|英国民航管理局]]颁发的公共机场运营牌照（第P725号），可执行公共旅客飞行运输和飞行指令。它也在一项调查中被评为英国北部最受欢迎的机场。<ref>{{cite web |url=http://www.wanderlust.co.uk/|title=WANDERLUST TRAVEL AWARDS|publisher=Wanderlust|language=en|accessdate=2010-03-29}}</ref>

== 历史 ==
纽卡斯尔机场于1935年7月26日正式启用，由时任[[空军大臣|空军大臣]]的[[菲力浦·坎利夫-李斯德爵士|菲力浦·坎利夫-李斯德爵士]]揭幕。当时的建造成本花费了3万5千[[英镑|英镑]]，包含了会所、机库、车间、燃料库和草地[[跑道|跑道]]等设施。

尽管在[[二战|二战]]期间该地区的主要机场是位于[[诺森伯兰郡|诺森伯兰郡]]的[[克拉姆灵顿|克拉姆灵顿]]，但战争结束后当局却决定集中发展现在的机场。因此在1950年代，前[[英国皇家空军|英国皇家空军]][[战斗机|战斗机]]飞行员吉姆·德尼尔被任命为机场经理，并在此后数年间每年都有超过5000名旅客使用该机场前往[[泽西岛|泽西岛]]和[[怀特岛|怀特岛]]等目的地。

纽卡斯尔机场的旅客人数在1960年代得到大幅度的增长。这主要得益于英国国民开始热衷于到海外度假而非本土的趋势。一条新的跑道、一座新的停机坪和塔台被建成。新设施由时任[[英国首相|英国首相]]的[[哈罗德·威尔逊|哈罗德·威尔逊]]揭幕启用。1970年代，随着年发送旅客数量接近百万，纽卡斯尔机场的等级被改为B级，使之成为区域性国际机场。1980年代机场进一步投资完善了办票、餐饮和免税商店等设施。1991年，机场地铁站启用，实现了机场与纽卡斯尔市中心的轨道连接，并在2002年可通过[[泰恩-威尔地铁|泰恩-威尔地铁]]连接至桑德兰市中心。2000年，首家[[廉价航空公司|廉价航空公司]]进驻机场，Go Fly和后来倒闭的吉尔航空一同运营往返于[[伦敦斯坦斯特德机场|伦敦斯坦斯特德机场]]的航线。2001年，哥本哈根机场购入了纽卡斯尔机场49%的股权。

2004年，机场的航站楼完成扩建和翻新并启用。扩建的区域达到3000平方米，其中包括增加了新的商店、咖啡厅和1200个座椅。

根据英国民航管理局的数字显示，纽卡斯尔机场在2006年的旅客发送量达到创纪录的540万人次。这一数字预计在2009年达到700万，但由于受到[[2007年－2010年环球金融危机|2007年－2010年环球金融危机]]的影响，实际数字可能会比预计低约250万。

客运量的快速增长使得机场南端也用作了商业航班用途，原本这里是[[通用航空|通用航空]]的区域，现在则供[[货运|货运]]、[[邮件|邮件]]和公司航班使用。作为机场总纲计划的一部分，机场南端区域的设施必须扩建和修缮，包括新的飞机库和扩建停机坪。[[纽卡斯尔民航学院|纽卡斯尔民航学院]]也位于这一区域。

== 未来计划 ==
纽卡斯尔机场在2006年公布了直到2016年的机场发展总纲提案。短期内，这项计划包括在机场南端新建一个以取代目前的短停留车位的多层停车场、一个拥有187张铺位的[[酒店|酒店]]（目前已在建）和扩建货运设施。而关于长期的发展计划，目前正在进行可行性研究，这些建议包括：
* 延长其跑道东部的末端；
* 将A696交界处的[[平面道路交叉|平面道路交叉]]转换成[[立体交叉桥|立体交叉桥]]，以应付未来预期增加的交通流量。
* 建造一条重型铁路以与国家铁路网连接。

[[2007年10月|2007年10月]]，一个耗资820万英镑建设的全新[[塔台|航空交通控制塔]]在机场北侧竣工。<ref>

== 服务区域 ==
纽卡斯尔机场主要服务于[[诺森伯兰|诺森伯兰]]、[[泰恩赛德|泰恩赛德]]和[[维尔赛德|维尔赛德]]地区。该机场还与一座更为小型的[[达勒姆蒂谷机场|达勒姆蒂谷机场]]争夺前往或来自[[达勒姆郡|达勒姆郡]]及[[蒂赛德|蒂赛德]]地区的客源。前往或来自[[坎布里亚郡|坎布里亚郡]]，[[北约克郡|北约克郡]]和[[苏格兰|苏格兰]]南部的乘客也可以使用该机场，邻近的类似规模的机场有南部的[[利兹布拉德福德机场|利兹布拉德福德机场]]和北部较大的[[爱丁堡机场|爱丁堡机场]]及[[格拉斯哥国际机场|格拉斯哥国际机场]]。在客运量方面，纽卡斯尔机场是英格兰北部仅次于[[曼彻斯特机场|曼彻斯特机场]]和[[利物浦机场|利物浦机场]]的第三大机场。

== 地面交通 ==
=== 轨道交通 ===

== 地面服务 ==
纽卡斯尔机场主要的地面服务代理为[[Swissport|Swissport]]和[[Servisair|Servisair]]。
