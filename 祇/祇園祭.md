'''祇園祭'''是[[日本|日本]][[京都|京都]]每年一度舉行的[[節慶|節慶]]，<ref>[http://www.welcome2japan.tw/indepth/cultural/matsuri/gion.html 專欄：祇園祭]，日本政府觀光局</ref>不但為「京都三大祭」（[[葵祭|葵祭]]、祇園祭、[[時代祭|時代祭]]）之一，也為「日本三大祭」（祇園祭（京都）、[[天神祭|天神祭]]（[[大阪|大阪]]）、[[神田祭|神田祭]]（[[江戶|江戶]]））與「三大曳山祭」、「三大美祭」（[[秩父夜祭|秩父夜祭]]（[[埼玉縣|埼玉縣]][[秩父市|秩父市]]）、祇園祭（[[京都府|京都府]][[京都市|京都市]]）、[[高山祭|高山祭]]（[[岐阜縣|岐阜縣]][[高山市|高山市]]））之一。並被認為是其中一項最大規模及最著名的祭典。整個祇園祭長達一個月，在7月17日（前祭）和7月24日（後祭）則進行大型巡遊，京都的33個區，每區均會設計一個裝飾華麗的花轎參加巡遊。

== 歷史 ==
祇園祭起源自平安時代西元869年<ref name=":0">{{Cite book|title = 京都原來如此：千年古都的風俗、習慣、飲食文化解謎|last = 博學堅持俱樂部|first =|publisher = 遠足文化事業股份有限公司|year = 2015|isbn = 9789869189620|location =|pages = 第212到213頁}}</ref>，當時人口急增疫症爆發，醫學知識不足將疾病原因歸咎於怨念作祟，此想法來自於印度的[[牛頭天王|牛頭天王]]傳說影響，為了平息怨靈，作法供祀為神祇。<ref name=":0" />於是在京都東山的祇園社（現今[[八坂神社|八坂神社]]）把[[牛頭天王|牛頭天王]]供為神祇祭祀後，瘟疫就平息下來。<ref name=":0" />在當時只有發生瘟疫才會舉行法事，到了西元970年起才成為慣例，在容易發生流行病的6月7日跟6月14日固定舉行。<ref name=":0" />到了日本改制新曆後，則改在7月在京都市內舉辦例行祭典<ref name=":0" />，祈求潔淨及消除瘟疫。

== 祇園祭活動時程表 ==
* 7月1日 - 吉符入、長刀鉾町御千度。祭典開始。
* 7月2日 - 籤取儀式、山鉾町社参。決定山鉾巡行順序。從[[室町時代|室町時代]]以來為避免競争事先已定好順序。接著、帶頭巡行（前祭）之先頭長刀鉾、第5號函谷鉾、第21號放下鉾、第22號岩戸山、第23號（帶頭巡行（前祭）之最後）船鉾。後頭巡行（後祭）之先頭橋辨慶山、接次之北観音山、後頭巡行（後祭）第6號南観音山、後頭巡行（後祭）之最後大船鉾「籤置儀式」與呼叫、順序均已予先決定。[[京都|京都市役所]]市委員會議場完成儀式。
* 7月7日 - 綾傘鉾稚兒社参
* 7月10日 - 幣切、鉾立、神事用水清祓、御迎提灯、神輿洗
* 7月10日到13日 - （前祭）宵山建立、鉾的建立。山之分解收納・鉾之組合、進行懸装。
* 7月13日 - 高橋町社参、稚児社参、久世稚児社参、（前祭）鉾引初、山建、舁初
* 7月14日 - 中之町御供、（前祭）宵々々山
* 7月15日 - 斎竹建、式包丁奉納、伝統芸能奉納、（前祭）宵宮祭、宵々山
* 7月16日 - 献茶祭、豊園社御真榊建、手洗井戸開き、神剣拝戴、石見神楽  、（前祭）宵山、宵宮神賑奉納神事、日和神楽 
* 7月17日 - （前祭）山鉾巡行。祇園祭之重要節目。
* 7月17日 - 神輿渡御出發式。
* 7月18日到20日 - （後祭）宵山建立
* 7月20日 - 宣状式
* 7月21日 - （後祭）宵々々山
* 7月22日 - （後祭）宵々山
* 7月23日 - 斎竹建、献茶祭、琵琶奉納、（後祭）宵山
* 7月24日 - （後祭）山鉾巡行、花傘巡行、御真榊御祭、還幸祭。上述後祭的変化
* 7月25日 - 狂言奉納
* 7月28日 - 神事用水清祓、神輿洗
* 7月31日 - 疫神社夏越祭。祭典結束。

在7月舉辦各種活動期間，相關祭典人士會遵守不能吃小黃瓜的戒律，小黃瓜雖然是夏季盛產蔬菜，但因小黃瓜的斷面長得太像[[八坂神社|八版神社]]的社紋而不敢吃小黃瓜。<ref name=":0" />
