《'''樹大根深'''》<ref>原劇名直譯為《根深蒂固的樹》，原著小說在中國大陸出版，書名被人民文學出版社訂為《樹大根深》，亦有直接以此為劇名的作法。</ref>（{{lang-ko|뿌리깊은 나무}}）為[[韓國|韓國]][[SBS_(韓國)|SBS]]自2011年10月5日起播出的[[SBS水木迷你連續劇|水木連續劇]]，由[[金榮昡|金榮昡]]、[[朴尚淵|朴尚淵]]共同負責劇本寫作，《[[錢的戰爭|錢的戰爭]]》、《[[風之畫員|風之畫員]]》導演[[張太侑|張太侑]]執導。根據韓國作家{{link-ko|李正明（作家|이정명|李正明}}的同名歷史推理小說改編，以《[[訓民正音|訓民正音]]》頒布前七日在[[景福宮_(朝鮮)|景福宮]]發生的[[集賢殿|集賢殿]]學士連鎖殺人事件為題材，對[[朝鮮世宗|世宗]]重新解說的電視劇。此作為[[韓石圭|韓石圭]]自1995年[[文化廣播_(韓國)|MBC]]《酒店》後，時隔16年再次出演的電視劇。

== 原著／劇名出處 ==
取自《[[龍飛御天歌|龍飛御天歌]]》第二章其中一段唱詞，被印在10,000元韓國鈔票上，漢字文言文為：「根深之木，風亦不扤，有灼其華，有蕡其實。源遠之水，旱亦不竭，流斯為川，于海必達。」因此正確的名稱應為「根深之木」。

== 引用典故 ==
*空饌盒：君主賜空饌盒代表「不必再食，讓其自盡」之意。相傳[[曹操|曹操]]贈食給[[荀彧|荀彧]]，荀彧打開食器卻空無一物，覺得曹操認為他已無用，被迫服毒自盡。

== 劇情簡介 ==
展示《[[訓民正音|訓民正音]]》創造過程中，人們不曾瞭解的背景、反對派所持的理由以及[[朝鮮世宗|世宗]]與[[百姓|百姓]]對《訓民正音》有怎樣的想法。焦慮孤獨的王、被復仇蒙蔽的[[奴婢|奴婢]]、把朝鮮建國理念當做自己使命的[[士大夫|士大夫]]以及無意中成為訓民正音中心的宮女，通過這四個人之間憐憫、信賴及對決展開的故事。原著小說中本是連環殺人事件的推理型態，在電視劇中弱化推理調查要素，著重於世宗一派和密本間的鬥爭。

== 演員陣容 ==
=== 主要角色 ===
*世宗[[朝鮮世宗|李祹]]：[[韓石圭|韓石圭]]<small>（青年：[[宋仲基|宋仲基]]、幼年：[[姜山_(演員)|姜山]]）</small>
:朝鮮第四代國王，即位前為忠寧大君。為百姓著想的自虐型工作狂人，也是失眠症患者。外表看起來充滿威嚴，卻是一個急躁甚至會說民間[[粗話|粗話]]的人物，對各種繁文縟節感到厭煩，不喜歡形式的實用主義者，並非偉人傳記常出現的完美聖君形象。
*姜彩允（小福）：[[張赫_(韓國)|張赫]]<small>（少年：[[呂珍九|呂珍九]]、幼年：[[蔡相宇|蔡相宇]]）</small>（粵語配音：[[陳健豪|陳健豪]]）
:最愛的父親無故被捲入鬥爭而去世，心中一直視李祹為仇人。在北方時遇見絕世高手李方志，從他身上學得高超武藝。抱持著復仇之心活著，終於來到王宮，脫去奴婢身分成為兼司僕，精心盤算如何殺了李祹。他的執著無人能敵，膽識過人，為達目的不擇手段，能屈能伸。
*咲梨（丹兒）：[[申世景|申世景]]<small>（幼年：[[金賢秀|金賢秀]]）</small>
:廣平大君的宮女，小時候經歷巨大變故的衝擊，患上了失語症。參與《訓民正音》創製計畫的宮女組織核心成員。比李祹更熱衷於《訓民正音》，在這份熱情中還包含年幼時的創傷，努力去理解身為一國之君的李祹心中的痛苦、害怕。
*鄭基準：謎樣人物，於故事中揭曉。<!--注意：請勿將此飾演者寫出--><small>（青年：[[崔宇植|崔宇植]]、幼年：[[申東基|申東基]]）</small>
:[[鄭道傳|鄭道傳]]的弟弟鄭道光之子，密本第三代本元。延續三峯之志，「君主是花，宰相是根，士大夫才是國家的根基」，將來朝鮮依然有王，卻無法支配一草一木。在父親死後，大隱隱於市24年，靜待時機實現密本。

==== 初期主要角色（第1－4集）====
*世宗[[朝鮮世宗|李祹]]（青年時期）：[[宋仲基|宋仲基]]
:一個擁有與生俱來才能的天才型人物，在暴君父親的壓制下，度過了暗潮洶湧的青年時期，對權力的虛無感早已認清，並決定以文治世。
*太宗[[李芳遠|李芳遠]]：[[白潤植|白潤植]]
:曾以兩次王子之亂的方式，剷除異己，進而繼位為王。個性想擁有權力，阻擋者殺無赦。晚年把王位讓給第三子李祹，不過在宮內仍保留一定權力，並掌握軍事大權直到去世。為李祹剷除所有可能危害朝鮮王權的勢力，為李祹日後的發揮留下大幅空間。
