'''樹袋鼠'''是生活在樹上的[[袋鼠科|袋鼠科]][[動物|動物]]。牠們分佈在[[新畿內亞|新畿內亞]]、[[昆士蘭|昆士蘭]]極東北區及鄰近[[島嶼|島嶼]]的[[雨林|雨林]]。雖然大部份都是生活在山區之中，當中也有幾個[[物種|物種]]棲息在低地中，如[[皂樹袋鼠|皂樹袋鼠]]。大部份樹袋鼠都因獵殺及失去棲息地而被列為[[瀕危|瀕危]]。

現時已知約有12種樹袋鼠，一些分類卻仍未明。牠們的毛色及大小有顯著不同，[[頭部|頭部]]及身體長41-77厘米，尾巴長40-87厘米，重14.5公斤。雌性較雄性細小。

== 演化 ==
樹袋鼠相信是[[演化|演化]]自與現今[[袋鼠|袋鼠]]及[[小袋鼠|小袋鼠]]相似的祖先，仍保有一些[[袋鼠科|袋鼠科]]在平原上生活的適應性，如巨大的後腳及窄長的腳掌。樹袋鼠發展出很長的尾巴，用以平衡身體；前肢強壯，適合攀樹。腳稍短及闊，有很長的爪，腳掌具彈性，都是適合抓樹的特徵。

相信所有袋鼠的祖先都是細小的樹棲[[有袋類|有袋類]]，就像現今一些[[澳洲|澳洲]]的[[負鼠|負鼠]]。自這些祖先走到地上來，開始演化出適合地上頻繁運動的身體後，就分支出最早期的袋鼠科。樹袋鼠的祖先為何又回到樹上則不詳，但相信是因缺乏廣闊的地方所致。

== 運動 ==
樹袋鼠在地上行動很緩慢及笨拙。牠們會以行走的步伐及笨拙的跳躍方式來運動，並會將身體向前傾來平衡重重的尾巴。但當牠們走回樹上，牠們卻十分靈活。牠們以前肢抱著樹幹，並以強壯的後肢跳躍來攀樹。牠們是飛躍的能手，可以跳逾9米，更可以從高18米的樹上跳落地上。

== 食性 ==
樹袋鼠主要吃[[葉子|葉子]]及[[果實|果實]]，另外也會吃[[穀物|穀物]]、[[花朵|花朵]]、[[樹汁|樹汁]]、[[樹皮|樹皮]]、[[蛋|蛋]]及雛鳥。牠們的[[牙齒|牙齒]]適合撕開葉子而非砌斷[[草|草]]。牠們的[[胃部|胃部]]很大，內裡的[[細菌|細菌]]可以幫助分解[[植物|植物]][[纖維|纖維]]。

== 分類 ==
[[Image:kanga2031.jpg|thumb]]。]]
以下是[[樹袋鼠屬|樹袋鼠屬]]下的[[物種|物種]]：<ref name=MSW3/>
* [[灰樹袋鼠|灰樹袋鼠]]（''Dendrolagus inustus''）：分佈在[[新畿內亞|新畿內亞]]西部及北部，與及[[亞彭島|亞彭島]]、[[薩拉瓦蒂|薩拉瓦蒂]]及[[衛古島|衛古島]]。
* [[盧氏樹袋鼠|盧氏樹袋鼠]]（''Dendrolagus lumholtzi''）：分佈在[[澳洲|澳洲]]的[[昆士蘭|昆士蘭]]。
* [[班氏樹袋鼠|班氏樹袋鼠]]（''Dendrolagus bennettianus''）：分佈在澳洲的昆士蘭。
* [[熊形樹袋鼠|熊形樹袋鼠]]（''Dendrolagus ursinus''）：分佈在新畿內亞的[[佛吉克普半島|佛吉克普半島]]（Vogelkop Peninsula）。
* [[赤樹袋鼠|赤樹袋鼠]]（''Dendrolagus matschiei''）：分佈在新畿內亞的[[休恩半島|休恩半島]]（Huon Peninsula）。
* [[多麗樹袋鼠|多麗樹袋鼠]]（''Dendrolagus dorianus''）：分佈在新畿內亞西部、中部及東南部。
* ''Dendrolagus stellarum''：分佈在新畿內亞中西部的高原。
* [[古氏樹袋鼠|古氏樹袋鼠]]（''Dendrolagus goodfellowi''）：分佈在新畿內亞的中部及東南部。
* [[金披風樹袋鼠|金披風樹袋鼠]]（''Dendrolagus pulcherrimus''）：分佈在新畿內亞的[[福賈山脈|福賈山脈]]（Foja mountains）及[[托里切利山脈|托里切利山脈]]（Torricelli Mountains）。
* [[皂樹袋鼠|皂樹袋鼠]]（''Dendrolagus spadix''）：分佈在[[巴布亞新畿內亞|巴布亞新畿內亞]]西南部的低地。
* [[白腹樹袋鼠|白腹樹袋鼠]]（''Dendrolagus mbaiso''）：分佈在新畿內亞中西部的高原。
* [[史氏樹袋鼠|史氏樹袋鼠]]（''Dendrolagus scottae''）：分佈在新畿內亞的[[桑道恩省|桑道恩省]]。

樹袋鼠的分類，尤其是多麗樹袋鼠及古氏樹袋鼠的超種十分複雜。''stellarum''以往被描述為多麗樹袋鼠的[[亞種|亞種]]<ref name=Flannery>{{Cite book en|author=Flannery, T.|year=1995|title=Mammals of New Guinea|publisher=Reed Books|id=ISBN 978-0-7301-0411-7}}</ref>可能是一個有效的物種<ref name=mayri>[http://www.tenkile.com/wondiwoi_tree_kangaroo.html Wondiwoi Tree Kangaroo] {{webarchive|url=https://web.archive.org/web/20080425232225/http://www.tenkile.com/wondiwoi_tree_kangaroo.html |date=2008-04-25 }} Tenkile Conservation Alliance</ref>，但大部份文獻都因缺乏認識而將之分類為多麗樹袋鼠的亞種。<ref name=MSW3/>金披風樹袋鼠的情況與''stellarum''相似。牠們最初被認為是古氏樹袋鼠的亞種<ref name=Flannery/>，但亦被提升為獨立的物種。<ref name=MSW3/>史氏樹袋鼠在[[貝瓦尼山脈|貝瓦尼山脈]]（Bewani Mountains）發現的一個群落有可能是未被描述的亞種。<ref name=Tenkile>[http://www.tenkile.com/tenkile_tree_kangaroo.html Tenkile Tree Kangaroo] {{webarchive|url=https://web.archive.org/web/20140225060902/http://www.tenkile.com/tenkile_tree_kangaroo.html |date=2014-02-25 }} Tenkile Conservation Alliance</ref>
