'''樹杞林文化館'''，位於[[台灣|台灣]][[新竹縣|新竹縣]][[竹東鎮|竹東鎮]]，為[[竹東鎮|竹東]]鎮民舉辦藝文活動之主要場所。1991年建造完成，原為新竹縣農產品展售中心的“農產品展售中心”；後因使用欠佳且閒置狀態，當地鎮公所配合2003年(民國92年)[[文化部|文化部]][[文建會|文建會]]所補助推動之“地方文化館計畫”，並於2004年7月10日正式啟用<ref>[http://www.hcctt.gov.tw/zhudong/home.jsp?menudata=ZhudongMenu&mserno=201108020019&contlink=content/cn01.jsp&serno=201108030016 關於]樹杞林文化館 新竹縣竹東鎮公所</ref>，打造出具有人文、藝術、鄉土文化之文化館。

== 命名由來 ==
竹東之舊名為“橡棋林”，亦稱名為“樹杞林”；此命名來由為先民至當地開墾之初期，發現當地長滿橡棋樹且成林下，故而得名之<ref>[http://www.hcctt.gov.tw/zhudong/home.jsp?menudata=ZhudongMenu&mserno=201108020019&contlink=content/cn01.jsp&serno=201108030016 命名] 樹杞林文化館 新竹縣竹東鎮公所</ref><ref>[http://superspace.moc.gov.tw/hall/local_culture_page.aspx?oid=d87195e1-72c2-4b93-85f9-c2501b413205 關於本館] 樹杞林文化館 文化部地方文化館</ref>；亦即該文化取名之緣故。

== 建築運作 ==
由當地鎮公所所主導全年度之藝文展演活動，並擬以社區營造的精神概念下，定位為結合藝文表演、展覽、教育活動及社區集會等功能公共場所；一樓為150坪的展覽館，展覽的文物多半以[[竹東鎮|竹東鎮]]在地之人文鄉土歷史與創意藝術為主；二樓為588座位的表演廳，以不定時會有具地方特色之展演活動。並且，運用該棟作為現代化之建築，透過各類展演活動之規劃<ref>[http://superspace.moc.gov.tw/hall/local_culture_page.aspx?oid=d87195e1-72c2-4b93-85f9-c2501b413205 運用] 樹杞林文化館 文化部地方文化館</ref><ref>[http://travel.hsinchu.gov.tw/page.aspx?wtp=1&wnd=57&id=216 竹東藝文空間‧樹杞林文化館] {{webarchive|url=https://web.archive.org/web/20140407090117/http://travel.hsinchu.gov.tw/page.aspx?wtp=1&wnd=57&id=216 |date=2014-04-07 }} 新竹縣旅遊網- 古蹟文化</ref>。並在室內增設咖啡館及藝文展售中心，以作為藝文交流的空間，設罝咖啡座椅、提供書報藝術資料，配合不定期展演之活動<ref>[http://www.railway.gov.tw/Jhudong/CP.aspx?sn=13428 樹杞林文化館] 台灣鐵路管理局</ref>；提昇竹東鎮在地文化與藝術生活之水準品質。

== 展覽活動 ==
該文化館常推出[[客家戲曲|客家戲曲]]與陶藝研習等之具有地方特色活動，用以強化竹東在地文化與藝術生活品質性；該館將定位為溪南地區藝術文化活動中心，一方面可滿足居民多元之藝文需求，展覽主題將涵蓋竹東人文與鄉土歷史、溪南八鄉鎮藝術創作等人文景觀以及自然地理資源，如橫山野薑花、峨眉茶香醉月、五峰賽夏果香、寶山糖筍爭豔等，並可提供鄰近八鄉鎮的藝文展演活動場所<ref>[http://superspace.moc.gov.tw/hall/local_culture_page.aspx?oid=d87195e1-72c2-4b93-85f9-c2501b413205 展覽活動] 樹杞林文化館 文化部地方文化館</ref><ref>[http://travel.hsinchu.gov.tw/page.aspx?wtp=1&wnd=57&id=216 竹東藝文空間‧樹杞林文化館] {{webarchive|url=https://web.archive.org/web/20140407090117/http://travel.hsinchu.gov.tw/page.aspx?wtp=1&wnd=57&id=216 |date=2014-04-07 }} 新竹縣旅遊網- 古蹟文化</ref>。以推廣文化展演、藝文教育，並兼具休憩與社會交誼、觀光與資訊導覽之多功能且多元化之藝術整合中心。
