'''樹幹'''，亦寫作'''樹榦'''，是[[樹木|樹木]]的主要莖幹，木質，由地面開始，向上到分為樹枝為止。樹幹負責承拓整株植物，樹幹裡的[[木質部|木質部]]負責[[運輸|運輸]]供[[水|水]]及[[營養|營養]]以助[[光合作用|光合作用]]和[[新陳代謝|新陳代謝]]。樹幹是[[木本植物|木本植物]]的[[莖|莖]]特化而來，通常外包有[[樹皮|樹皮]]。不同樹種的樹幹外觀不同，可以作為植物分類的依據。

樹幹是大多數[[木材|木材]]的來源，

== 結構 ==
[[File:Taxus_wood.jpg|250px]]
樹幹是木質化的[[莖|莖]]，通常經過[[次級生長|次級生長]]。狹義的樹是由多年的次級生長累積而成，只出現在[[雙子葉植物|雙子葉植物]]和[[裸子植物|裸子植物]]。由外而內為[[木栓層|木栓層]]、[[木栓形成層|木栓形成層]]、綠皮層、[[韌皮部|韌皮部]]、[[維管束形成層|維管束形成層]]、[[木質部|木質部]]、[[木髓|木髓]]。木質部通常佔最大比例，可再細分為邊材和心材。邊材在外側，較年輕，具水份運輸的功能，結構相對疏鬆，顏色較淡；心材較老，不具水份運輸的功能，主要提供結構支持，顏色較深。由韌皮部以外合稱為[[樹皮|樹皮]]。

在氣候變化明顯的地方，樹幹中會有[[年輪|年輪]]。有些樹，例如[[櫻|櫻]]，在樹幹上會有[[皮孔_(植物)|皮孔]]協助[[氣體交換|氣體交換]]。在熱帶地區，有些樹的樹幹上會直接長出[[花|花]]和[[果實|果實]]，稱為幹生花和幹生果，例如[[水同木|水同木]]、[[可可樹|可可樹]]等。

===單子葉植物===
有些[[棕櫚|棕櫚]]、[[芭蕉|芭蕉]]、[[龍舌蘭|龍舌蘭]]、[[龍血樹|龍血樹]]、[[蘆薈|蘆薈]]等[[單子葉植物|單子葉植物]]也有樹幹。這類樹幹主要由大量的[[薄壁細胞|薄壁細胞]]堆積而成，通常沒有形成層和樹皮。有些種類的單子葉植物有分散的{{link-en|單面形成層|unifacial cambium}}。

===蕨類===
{{link-en|樹蕨|tree fern}}大多屬於[[桫欏目|桫欏目]]，沒有次級生長，樹幹是由大量的根和葉柄聚集而成。已滅絕的[[石松門|石松門]]植物[[鱗木|鱗木]]也有樹幹。

== 演化 ==
目前已知最古老有樹幹的植物是約3億8500萬年前（[[泥盆紀|泥盆紀]]中期）的[[枝蕨綱|枝蕨綱]][[瓦蒂薩屬|瓦蒂薩屬]]植物，高度可達八公尺。至[[石炭紀|石炭紀]]時，[[鱗木|鱗木]]、[[封印木|封印木]]、[[蘆木|蘆木]]等樹已經形成廣大的森林。

更早之前，志留紀晚期出現的真菌或地衣[[原杉藻|原杉藻]]已經能形成數公尺高的主幹，但因為不是植物，嚴格來說不能稱為樹幹。

相較於沒有木質樹幹的植物，樹幹的主要功能是增加植物的高度，有助於散布孢子或種子以及競爭陽光。

== 經濟用途 ==
樹幹為人類利用製成[[木材|木材]]，用作[[建築材料|建築材料]]、[[船|運輸工具]]、[[傢俱|傢俱]]等。木材也可以當作燃料。
