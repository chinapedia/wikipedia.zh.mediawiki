{{日本作品|草莓之夜|ストロベリーナイト|Strawberry Night}}是[[日本|日本]]作家[[譽田哲也|譽田哲也]]的警察[[推理小說|推理小說]]。

== 登場人物 ==
=== 姬川班 ===
;{{nihongo|姬川玲子|姫川 玲子（ひめかわ れいこ）}}
: 警視廳搜查一課殺人犯搜查十組主任，警部補。唯一的女性班長，率領「姬川班」。畢業於女子大學，雖非菁英組，卻破例快速晉升，27歲就成為警部補。在男性體系的警界，一路與偏見戰鬥至今。擁有敏銳的直覺，並經常成為破案的關鍵。
;{{nihongo|菊田和男|菊田 和男（きくた かずお）}}
: 搜查十組姬川班的刑警，巡查部長。嚴謹誠實，沉默寡言。擅長各種武術。過著單身生活，家事也難不倒他。剛開始面對女性上司不知所措，但漸漸發現玲子不僅是優秀的刑警，更是個有魅力的人，而心生好感。興趣是打柏青哥。
;{{nihongo|大塚真二|大塚 真二（おおつか しんじ）}}
: [[巡查長|巡查長]]。玲子的部下。最初發現網站「草莓之夜」的刑警。
;{{nihongo|石倉保|大石倉 保（いしくら たもつ）}}
: 搜查十組姬川班的刑警，巡查部長。姬川班當中最資深的刑警，深受大家信賴。從基層苦幹上來，在警界人面很廣，在外也有自己的情報網與人脈。原本有點不爽年輕女性當上司，但之後發現玲子是優秀的刑警，便決心輔助她。
;{{nihongo|湯田康平|湯田 康平（ゆだ こうへい）}}
: 搜查十組姬川班的刑警，巡查長。今泉組長將他從地方警局調派進來，是姬川班最年輕的成員。個性開朗又率直，深受姬川班同事的疼愛。願意不辭勞苦，腳踏實地進行搜查。
;{{nihongo|葉山則之|葉山 則之（はやま のりゆき）}}
: 搜查十組姬川班的刑警，巡查長。因為其痛苦的過去，高中畢業後加入了警隊，做事認真，沉默寡言。剛進入姬川班時有些不合群，後來也慢慢認同了玲子，接受了姬川班。

=== 警視廳 ===
;{{nihongo|勝俣健作|勝俣 健作（かつまた けんさく）}}
: 警部補。捜查一課殺人犯捜查五組（勝俣班）主任。玲子的死對頭。為了破案而採取一些違法手段，遊走法律邊緣。
;{{nihongo|今泉春男|今泉 春男（いまいずみ はるお）}}
: [[警部|警部]]。捜查一課十係長。
;{{nihongo|橋爪俊介|橋爪 俊介（はしづめ しゅんすけ）}}
: [[警視|警視]]。
;{{nihongo|日下守|日下 守（くさか まもる）}}
: 警部補。捜查一課殺人犯捜查十係（日下班）主任。
;{{nihongo|北見昇|北見 昇（きたみ のぼる）}}
: 警部補。
;{{nihongo|井岡博滿|井岡 博満（いおか ひろみつ）}}
: 巡查長。對玲子有好感。
