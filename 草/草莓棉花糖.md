《'''草莓棉花糖'''》（{{lang-ja|苺ましまろ}}），又被简称為'''-{莓乱扔}-'''（logo上的“{{lang|ja|苺ましまろ}}”形似“-{苺乱扔}-”），是[[日本|日本]]漫畫家[[Barasui|Barasui]]（{{lang|ja|ばらスィー}}）在《[[月刊Comic電擊大王|月刊Comic電擊大王]]》連載的漫畫，原是投稿插畫的不定期連載，是一部[[萌系漫畫|萌系漫畫]]。

在2005年7月播出[[電視動畫|電視動畫]]，同年8月改編成[[電子遊戲|電子遊戲]]。另外发售了两套[[OVA|OVA]]，2007年发售第一套共三话，2008年发售第二套共两话。

== 作品簡介 ==
故事的主角們是五位11歲至16歲女孩，經由她們的視野觀察這個看似普通的世界。以每一話各自獨立的[[單元劇|單元劇]]形式進行，宣傳口號為{{nihongo|「可愛即正義！」|かわいいは、正義！}}。<ref name="vol7_4year">{{cite web|title=苺ましまろ：4年ぶり新刊　「かわいいは、正義！」でおなじみの人気マンガ - MANTANWEB（まんたんウェブ）|url=http://mantan-web.jp/2013/03/23/20130322dog00m200067000c.html|accessdate=2016-05-28}}</ref><ref name="animeanime">{{cite web|title=「苺ましまろ」がBD‐BOXに　シリーズ全話をHDリマスター化 {{!}} アニメ！アニメ！|url=http://animeanime.jp/article/2015/10/31/25504.html|website=animeanime|accessdate=2016-05-28}}</ref>

作者在作品中隐藏许多其对摇滚乐的喜爱，许多作品中人事物与摇滚乐有关，其中标题中的「{{lang|ja|ましまろ}}」就是启发自[[奥田民生|奥田民生]]。<ref>{{cite web|url=http://www.manganomori.net/00000001/mag67.htm|title=Barasui Interview|publisher=Manga no Mori|accessdate=2006-08-28|language=ja|archiveurl=https://web.archive.org/web/20060813083637/http://www.manganomori.net/00000001/mag67.htm|archivedate=八月 13, 2006|deadurl=yes}}</ref>

故事的設定取用很多確實存在的地方與事物，主角們的住處位於[[日本|日本]][[靜岡縣|靜岡縣]][[濱松市|濱松市]]（作者的出身地），[[濱松車站|濱松車站]]、[[佐鳴湖|佐鳴湖]]、一家[[牧場|牧場]]的牛奶廠牌「{{lang|ja|都田のおいしい牛乳}}」、美羽等四人就讀的小學等。不過濱松市的氣候溫和，少有[[降雪|降雪]]的情形發生。<ref name="漫城第7卷">{{cite web|title=《草莓棉花糖》时隔4年单行本第7卷2月发售_国际_漫域_中国动漫综合门户|url=http://news.comicyu.com/2012/12/31/98908.html}}</ref>在改编动画中也如实参照了当地实景来改编创作。<ref>{{Cite book|title=苺ましまろ完全ビジュアルブック|publisher=ASCII MediaWorks|year=|origyear=2006|isbn=|language=ja|pages=p60}}</ref>

濱松市成為許多《草莓棉花糖》觀眾前往[[朝聖|朝聖]]的「聖地」，卻屢屢引起當地居民困擾；《月刊Comic電擊大王》曾經就此勸導讀者不要打擾到居民的生活。

==登场角色 ==
===主要角色===
; {{nihongo|伊藤伸惠|伊藤 伸恵（いとう のぶえ）||[[配音員|配音員]]：[[生天目仁美|生天目仁美]]}}
: 漫画版为16歲的短髮高中生，A型，生日為10月17日。<ref>{{cite book|author1=ばらスィー|title=草莓棉花糖 第一卷|publisher=台湾角川|location=台湾|isbn=9867993985|pages=p5|accessdate=2016-05-23|quote=顺带一提，我叫伊藤伸惠，16岁。是个很平凡的高一学生。}}</ref>動畫版则更改为20岁的短大生，生日为10月18日。<ref>{{cite web|url=http://www.ichigomashimaro.com/character.html|title=Official home page for the anime edition - Characters|publisher=[[ASCII_Media_Works|ASCII Media Works]]|accessdate=2006-11-21|language=ja}}</ref>动画版第一话时也以此为[[哏|哏]]，伸惠起初说出自己在漫画设定的16岁高一生，然后以开玩笑地说出自己真实年龄是20岁短大生。
:最初連載是染金留長髮並有大量壞習慣的少女，之後剪短染黑，是除了妹妹千佳之外三個小蘿莉崇拜的對象。喜好抽煙（偏好[[七星香煙|七星]]），也喜歡在洗澡後來上一瓶啤酒，经常去妹妹千佳那里“借”钱，试过多次戒烟但都失败。
:在作品中有一點[[蘿莉控|蘿莉控]]的傾向，特別寵愛茉莉與安娜。對付過動的美羽很有一套，虽然很烦美羽过分的行为，但是实际上仍十分痛爱美羽。

; {{nihongo|伊藤千佳|伊藤 千佳（いとう ちか）||配音員：[[千葉紗子|千葉紗子]]}}
: 12歲的小學六年級生，A型、身高145公分、體重35公斤，生日為5月11日，通称为“小千”。髮型為單邊結髮。伸惠的妹妹，個性算是比較正常的。
:其特徵就是沒有什麼特徵。常常被姊姊跟鄰居美羽玩弄，美羽的雙關語笑話往往讓她笑得翻來翻去，也经常吐槽美羽。很擅長做點心，養了一隻叫佐竹的狗（長得像[[柴犬|柴犬]]）。

; {{nihongo|松岡美羽|松岡 美羽（まつおか みう）||配音員：[[折笠富美子|折笠富美子]]}}
: 跟千佳同年的鄰居，B型，身高144公分，生日在9月8日，通称为“小美”。常做一些無厘頭的舉動。最初連載是黑髮[[雙馬尾|雙馬尾]]的造型，之後染啡色。
:常常直接從房間的窗戶跑到千佳房間玩。在可愛的外表下有著小惡魔般的智慧和天馬行空的幻想。以搞笑搭檔來說，美羽是負責發動而千佳負責吐嘈她（美羽自稱）。
:因為是獨生女的緣故，內心覺得很寂寞而搞笑或作弄人讓別人來注意她，但往往玩過頭而讓其他人氣得半死，是個天生的顽皮小淘氣，而且更經常性地被人毆打至臉朝地暈倒地上（或倒在屋簷、雪地上）。
:不知為何很了解一些早期[[特攝|特攝]]動作跟宅語。　

; {{nihongo|櫻木茉莉|桜木 茉莉（さくらぎ まつり）||配音員：[[川澄綾子|川澄綾子]]}}
: 住在伊藤家附近的11歲小學五年級生，O型，生日是1月25日。個性內向害羞短髮戴著一付眼鏡，很容易被唬得團團轉，是伸惠跟美羽惡作劇的好對象，是一位極度愛哭的不得了的小女孩（例如因為爸爸吃了她的布丁而大哭一頓），身上重要的部位已經被伸惠和美羽看完了。跟英籍轉學生安娜成為好友。鼓勵安娜學英文，自己也跟著學，最後英文卻比安娜還好。
:有一隻叫約翰的[[雪貂|雪貂]]。

; {{nihongo|安娜·柯普拉|アナ·コッポラ|Ana Coppola|配音員：[[能登麻美子|能登麻美子]]}}
:跟茉莉同年的[[英國|英國]]人，老家是在英國康威爾，6月18日生，AB型。金色長髮藍眼像[[洋娃娃|洋娃娃]]一樣，除了外觀跟姓名外都很日本的英國小女孩，很不幸的名字跟日語眼窩洞（穴（{{lang|ja|あな}}））同音，所以很忌諱別人拿名字開玩笑，但是冒失的美羽從沒顧忌這點，更曾因為這樣而弄得安娜大哭一頓。
:儘管表面上嘗試像英國人一般地生活，卻無法掩飾自身對日本文化的了解與喜好。日語流利，卻幾乎不會英文，處處依賴茉莉學習英語。
:家裡養著一隻叫福爾西安堤的大型犬。

===次要角色===
; {{nihongo|笹塚|笹塚（ささづか）||配音員：[[皆川純子|皆川純子]]}}
: 跟茉莉和安娜同班的小男生，名字不明，常常被老師以各種匪夷所思的理由罰站。

;{{nihongo|5年2班老师|5-2担任||配音：[[近藤孝行|近藤孝行]]}}
:茉莉和安娜所在班的男老师，带着眼镜。在安娜转学后，经常无缘无故地笹塚罚站。在动画篇中唯一一次被笹塚拒绝后自行出去罚站。<ref>动画篇，第12话</ref>

;{{nihongo|6年2班老师|6-2担任||配音：生天目仁美}}
:千佳和美羽所在的女老师，同样带着眼镜，对美羽有时在班中熟视无睹的搞怪行为习以为常，然后淡定地将其处置。

; {{nihongo|老爺爺|おじいさん||配音員：[[長_(日本配音員)|長島雄一]]}}
:千佳他們上學的路上的文具店（或者是點心店）「明文堂」的老闆。经常出现在主角群出现的地方。

; {{nihongo|绀野爱子|紺野 愛子||配音：[[福圓美里|福圓美里]]}}
:伸惠的好朋友，当伸惠在学校场景时会经常出现。动画篇则改为伸惠在便利店打工时的工友。

; {{nihongo|伊藤高文|伊藤高文（いとう たかふみ）}}
: 不定期連載時出現過伊藤姊妹的兄長，單行本沒有收錄，18歲。角色笹塚出现后就少见登场。具体容貌作品中少有提及。

== 遊戲 ==
2005年8月11日發售對應[[PS2|PS2]]平台上的全年龄向视觉小说遊戲。<ref>{{Cite web|url=http://www.amazon.co.jp/dp/B0009EK67Y/|title=|accessdate=2013-10-20|author=|date=|publisher=[[Amazon.com|Amazon.co.jp]]|language=ja|script-title=ja:苺ましまろ}}</ref>遊戲中套用動畫的設定，主角是一個對伸惠有好感的大學生，最终要得到作品中五名女主角的仰慕。另外于2005年8月27日发售相应的游戏指导手册。

Adventures|accessdate=2007-01-18|last=Santos|first=Carlo|date =October 10, 2006|publisher=[[Anime_News_Network|Anime News Network]]}}</ref>{{tsl|en|The_Comics_Journal|漫畫卷}}的Dirk
