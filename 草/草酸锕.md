'''草酸锕'''是[[锕|锕]]的[[草酸盐|草酸盐]]，化学式为Ac<sub>2</sub>(C<sub>2</sub>O<sub>4</sub>)<sub>3</sub>。
==制备==
草酸锕可由[[草酸铵|草酸铵]]和可溶性锕盐反应得到<ref name=wj />：
: 2 Ac<sup>3+</sup> + 3 C<sub>2</sub>O<sub>4</sub><sup>2-</sup> → Ac<sub>2</sub>(C<sub>2</sub>O<sub>4</sub>)<sub>3</sub>

==物理性质==
草酸锕是难溶于水的固体。

==化学性质==
草酸锕在高温下分解，产生[[氧化锕|氧化锕]]，放出[[一氧化碳|一氧化碳]]和[[二氧化碳|二氧化碳]]<ref name=wj />：
: Ac<sub>2</sub>(C<sub>2</sub>O<sub>4</sub>)<sub>3</sub> → Ac<sub>2</sub>O<sub>3</sub>+ 3 CO↑ + 3 CO<sub>2</sub>↑
