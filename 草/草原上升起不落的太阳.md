《'''草原上升起不落的太阳'''》，内蒙著名作曲家[[美丽其格|美丽其格]]词曲，1952年作于[[中央音乐学院|中央音乐学院]]（当时24岁的美丽其格为该学院大一学生），该歌曲是一首歌颂[[内蒙古草原|内蒙古草原]]（具体指[[科尔沁草原|科尔沁草原]]）、歌颂领袖[[毛泽东|毛泽东]]、歌颂[[中国共产党|中国共产党]]的歌曲。[[吴雁泽|吴雁泽]]、[[楼乾贵|楼乾贵]]、[[朱崇懋|朱崇懋]]、[[王宏伟|王宏伟]]、[[刘秉义|刘秉义]]等都曾演唱过该曲。<ref name="Bunny_1stbm"/>

== 历史 ==
1951年，美丽其格考入[[中央音乐学院|中央音乐学院]]（当时该校在[[天津|天津]]）。1952年，学校布置作业，要求每个学生拿出一首作品在“学生作品演唱、演奏会”上展示，而《草原上升起不落的太阳》正是24岁的大一学生美丽其格的作业。<ref name="Bunny_1stbm">{{Cite web |url= http://www.1stbm.com/Html/infor/20094/200941018258765.shtml |title= 1952草原上升起不落的太阳 |author= 李尧整理 |date= 2009-04-10 |publisher= 第一品牌网 |language= zh-hans |accessdate= 2011-6-19导入 |quote=  |deadurl= yes |archiveurl= https://web.archive.org/web/20120317215344/http://www.1stbm.com/Html/infor/20094/200941018258765.shtml |archivedate= 2012-03-17 }}</ref>

== 使用及改编 ==
*中央音乐学院、上海音乐学院、山东音乐学院等多所院校将这首歌曲纳入声乐教材。
*2006年，《草原上升起不落的太阳》入选[[嫦娥一号|嫦娥一号月球探测卫星]]播放曲目，并在卫星绕月期间择机播放。
=== 收录专辑 ===
*2007年6月21日，[[瀛洲文化|瀛洲文化]]发行的《旋风 新音乐 长征路》收录该曲。
*2007年7月1日，[[瀛洲文化|瀛洲文化]]发行的《新音乐长征路 红色经典系列一》收录该曲，由[[高登|高登]]演唱。
== 歌曲影响 ==
=== 所获荣誉 ===
*1954年，《草原上升起不落的太阳》获全国群众歌曲评选一等奖；
*1989年，在[[国家文化部|国家文化部]]举办的“建国四十年唤起我美好回忆的那些歌”征歌活动中，该歌曲获优秀作品奖；<ref name="Bunny_Min">{{cite journal| quotes =  | author = 巴义尔 | coauthors= |date = 1997年 | title = 经典的魅力──美丽其格和他创作的《草愿上升起不落的太阳》| journal = 民族团结 |volume= |issue= 7 |pages= 32-33 | issn = | url =  | language = zh-hans | format =  |accessdate = |quote =  }}</ref>
*1992年，《草原上升起不落的太阳》被评为“20世纪华人音乐经典”（该曲是其中唯一的作者为少数民族的作品）。
