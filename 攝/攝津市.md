'''攝津市'''（{{jpn|j=摂津市|hg=せっつし|rm=Settsu shi|f=y}}）是位於日本[[大阪府|大阪府]]北部，[[淀川|淀川]]右岸的城市，距離[[大阪市|大阪市]]市區僅約12公里。<ref>{{Cite web| url=http://www.city.settsu.osaka.jp/shisei/shoukai/profile/3951.html| title=市のプロフィール| date=2018-03-30| publisher=攝津市政府| accessdate=2018-11-15| language=ja}}</ref>

== 歷史 ==

過去在令制國時代屬於[[攝津國|攝津國]][[島下郡|嶋下郡]]，9世紀末現在的鳥飼地區曾被律令政府設置「{{Tsl|ja|鳥養牧}}」，在此放牧牛馬，後來也成為在淀川往來的貴族中停休憩的場所，[[宇多天皇|宇多天皇]]甚至也在此建立了鳥養院離宮。<ref name="History">{{Cite web| url=http://www.city.settsu.osaka.jp/shisei/shoukai/profile/3953.html| title=市のおいたち| date=2018-03-30| publisher=攝津市政府| accessdate=2018-11-15| language=ja}}</ref>

16世紀末，[[織田信長|織田信長]]之弟[[織田長益|織田長益]]（有樂齋）成為[[豐臣秀吉|豐臣秀吉]]的[[御伽眾|御伽眾]]後並領有本地味舌2千石的領地，[[關原之戰|關原之戰]]後由於功績受封位於[[大和國|大和國]][[山邊郡|山邊郡]]的新領地，成為[[大名_(稱謂)|大名]]，但由於織田長益仍將[[陣屋|陣屋]]設於味舌，因此稱為{{Tsl|ja|味舌藩}}。但在17世紀初，織田長益將大和國的領地分給四男、五男後，位於味舌的大部分領地因無人繼承而被幕府收回。

至19世紀[[江戶時代|江戶時代]]結束時，現在攝津市的轄區西半部大多為幕府直屬和{{Tsl|ja|芝村藩}}的領地，東半部為{{Tsl|ja|高槻藩}}的領地，其他另有少部分區域屬於[[古河藩|古河藩]]、{{Tsl|ja|淀藩}}和[[宮家|宮家]][[閑院宮|閑院宮]]。[[明治維新|明治維新]]後，這些區域曾先後隸屬大阪府司農局、大阪府北司農局、[[攝津縣|攝津縣]]、[[豐崎縣|豐崎縣]]、[[兵庫縣|兵庫縣]]、高槻縣、淀縣、芝村縣、古河縣；1871年第一次府縣整併後，才全數被整併至大阪府。

1889年實施[[町村制|町村制]]，現在的攝津市範圍在當時分屬：{{Tsl|ja|味生村 (大阪府)|味生村 (大阪府)|味生村}}、{{Tsl|ja|鳥飼村 (大阪府)|鳥飼村 (大阪府)|鳥飼村}}、味舌村和{{Tsl|ja|三宅村 (大阪府三島郡)|三宅村 (大阪府三島郡)|三宅村}}共四個行政區劃；味舌村先於1950年升格為{{Tsl|ja|味舌町}}後，於1956年與味生村、鳥飼村[[市町村合併|合併]]為'''三島町'''，三宅村在1957年併入茨木市後，又陸續將原屬三宅村南部的區域自茨木市改併入三島町。1966年，三島町升格改制為市，由於當時在靜岡縣已經有名為「[[三島市|三島市]]」的市級行政區劃，因此便以舊的國名「攝津國」命名，更名為'''攝津市'''。<ref name="History"/>

== 行政 ==

=== 歷任市長 ===

# 深田丈夫：原三島町町長，自1956年10月12日至1966年10月31日期間擔任共三屆的三島町町長，1966年三島町改制為攝津市後，繼續擔任市長職務至1968年10月11日卸任。
# {{Tsl|ja|井上一成}}：1968年10月12日 - 1976年10月11日、共2任。
# 井上信也：1976年10月12日 - 1988年10月11日、3任，為前任市長井上一成之弟。
# 森川薫：1988年10月12日 - 2004年10月11日、4任。
# {{Tsl|ja|森山一正}}：2004年10月12日 - 今、現為第4任任期。

== 交通 ==

[[阪急電鐵|阪急電鐵]][[京都本線|京都本線]]和[[西日本旅客鐵道|西日本旅客鐵道]][[JR京都線|JR京都線]]以南北向穿過轄區西部，[[大阪高速鐵道|大阪高速鐵道]][[大阪單軌電車線|大阪單軌電車線]]和[[東海旅客鐵道|東海旅客鐵道]]的[[東海道新幹線|東海道新幹線]]則是穿過轄區中心，其中東海道新幹線在轄內設有{{Tsl|ja|鳥飼車両基地|鳥飼車輛基地}}，一旁同時也有[[日本貨物鐵道|日本貨物鐵道]]的{{Tsl|ja|大阪貨物ターミナル駅|大阪貨物總站}}。[[近畿自動車道|近畿自動車道]]以南北向從轄區中央穿過。

轄區東部和南部由於未設有鐵路車站，主要依賴{{Tsl|ja|阪急バス|阪急巴士}}、{{Tsl|ja|近鉄バス|近鐵巴士}}、{{Tsl|ja|京阪バス|京阪巴士}}、{{Tsl|ja|大阪シティバス|大阪城市巴士}}的客運路線。

== 觀光資源 ==

[[File:新幹線公園.JPG|thumb]]

轄內主要綠地分別位於[[淀川|淀川]]及{{Tsl|ja|安威川}}沿岸，在[[日本貨物鐵道|日本貨物鐵道]]{{Tsl|ja|大阪貨物ターミナル駅|大阪貨物總站}}北側安威川沿岸區域設有{{Tsl|ja|新幹線公園}}，其中有展示{{Tsl|ja|新幹線車両|新幹線車輛}}及[[鐵路機車|鐵路機車]]。在淀川沿岸則有屬於{{Tsl|ja|国営公園|國營公園}}的{{Tsl|ja|淀川河川公園}}。

== 教育 ==

[[File:Osaka_Kun-ei_Senior_High_School.JPG|thumb]]

轄內的高等教育學校包括同是隸屬{{Tsl|ja|学校法人薫英学園|學校法人薫英學園}}的[[大阪薰英女子短期大學|大阪薰英女子短期大學]]和{{Tsl|ja|大阪人間科学大学|大阪人間科學大學}}，該學院在攝津市內還設有{{Tsl|ja|大阪薫英女学院中学校・高等学校|大阪薫英女學院中學校、高等學校}}。
