'''陆东福'''（{{bd|1955年|11月||}}），男，汉族。中华人民共和国[[江苏|江苏]][[丹阳市|丹阳]]人，现任[[中国铁路总公司|中国铁路总公司]]总经理，党组书记<ref name=zgs>{{Cite web |url =  http://www.china-railway.com.cn/zgsgk/gsld/201611/t20161125_61601.html|title =陆东福　中国铁路总公司总经理、党组书记  |author =  |date =2016-11-25  |publisher = 中国铁路总公司 |language =  |accessdate =  |quote =  }}</ref>。

==生平==
陆东福出生于1955年11月，1973年11月参加工作，1979年4月加入[[中国共产党|中国共产党]]，[[上海大学|上海大学]]成人教育学院（夜大）管理工程专业毕业，在职大学学历，高级工程师。1973年11月至1981年8月，任[[中华人民共和国交通部|交通部]]、[[中华人民共和国铁道部|铁道部]][[上海铁路局|上海铁路局]]基建工程处第五工程段学徒工、线路工，第三工程段线路工、团委副书记、工长。1981年8月至1986年9月，任铁道部上海铁路局团委宣传部副部长、团委副书记兼宣传部部长、团委书记。其间，1981年9月至1982年2月在中央团校第二十期宣传干部班学习；1982年9月至1983年2月在上海市工交党校学习。1986年9月至1988年7月，在[[复旦大学|复旦大学]]国际政治系政治学（行政管理）专业干部专修班学习<ref name=zgs/>。

1988年7月至1990年1月，任铁道部上海铁路局政治部办公室副主任、宣传部副部长。1990年1月至1993年12月，任铁道部上海铁路局纪委副书记、纪委副书记兼监察处处长。其间，1990年3月至1990年5月在[[中共上海市委|中共上海市委]]党校纪检干部班学习。1993年12月至1995年3月，任铁道部上海铁路局上海工程总公司党委书记。1995年3月至1995年7月，任铁道部上海铁路局副总经济师。1995年7月至1997年1月，任铁道部上海铁路局党委副书记。1997年1月至1998年1月，任铁道部上海铁路局党委副书记兼上海市轨道交通明珠线工程指挥部副指挥，上海铁路局轨道交通开发总公司总经理。1998年1月至1999年10月，任铁道部上海铁路局副局长、党委常委兼上海市轨道交通明珠线工程建设指挥部副指挥，上海铁路局轨道交通开发总公司总经理。1999年10月至2000年11月，任铁道部上海铁路局副局长兼上海铁路分局分局长。2000年11月至2003年7月，任铁道部上海铁路局局长、党委副书记。其间，1998年9月至2001年7月在上海大学成人教育学院（夜大）管理工程专业学习<ref name=zgs/>。

2003年7月至2013年3月，任铁道部副部长、党组成员。其间，2006年3月至2006年7月在[[中央党校|中央党校]]进修班学习。2013年3月至2016年9月，任[[中华人民共和国交通运输部|交通运输部]]副部长、党组成员，[[国家铁路局|国家铁路局]]局长、党组书记<ref name=zgs/>。他是国家铁路局首任局长<ref>{{cite news |title = 陆东福任国家铁路局局长 张业遂任外交部副部长 |url = http://news.163.com/13/0326/14/8QT9AA5J0001124J.html |date=2013-03-26 |website = 新华网}}</ref>。2016年9月至2016年10月，任[[中国铁路总公司|中国铁路总公司]]党组书记，交通运输部副部长、国家铁路局局长。2016年10月起，任中国铁路总公司总经理、党组书记<ref name=zgs/><ref>{{cite news |title = 陆东福接替盛光祖任中铁总总经理 |url = http://www.ce.cn/cysc/newmain/pplm/qyrw/201610/09/t20161009_16574168.shtml |date = 2016-10-09 |publisher = 中国经济网}}</ref><ref>{{cite web |language = |publisher = 网易财经 |title = 陆东福任中铁总总经理 盛光祖退休 |url = http://money.163.com/16/1009/15/C2UOSGAI00258105.html |author = 网易 |date = 2016-10-09 |accessdate = }}</ref>。

2008年，当选[[第十一届全国人大代表|第十一届全国人大代表]]<ref>{{cite web |title = 全国人大代表信息-陆东福 |url = http://www.npc.gov.cn/delegate/viewDelegate.action?dbid=110902 |accessdate=2013-04-05 |website=全国人大网}}</ref>。2017年，当选[[中共十九大|中共十九大]]代表，并进入中共十九大主席团<ref>{{Cite web |url =  http://cpc.people.com.cn/19th/n1/2017/1018/c414305-29592998.html|title = 十九大主席团名单 |author =  |date = 2017-10-18 |publisher =中国共产党新闻网  |language =  |accessdate =  |quote =  }}</ref>。
