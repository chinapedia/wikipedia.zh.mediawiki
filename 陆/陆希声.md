'''陆希声'''（{{bd|9世纪||895年|catIdx=L}}），[[表字|字]]'''鸿磬'''，自号'''君阳遁叟'''（一称'''君阳道人'''）。[[中国|中国]][[唐朝|唐朝]][[官员|官员]]，[[唐昭宗|唐昭宗]]年间短暂拜为[[宰相|宰相]]。

== 家世 ==
陆希声生年不详。祖上多代为官。五世祖[[陆元方|陆元方]]为[[武周|武周]]宰相，高伯祖[[陆象先|陆象先]]为[[唐睿宗|唐睿宗]]和[[唐玄宗|唐玄宗]]宰相。陆希声的高祖父为[[工部尚书|工部尚书]][[陆景融|陆景融]]。祖父陆孟儒，官至[[苏州|苏州]]司士参军。父陆翱，没有仕官记载。<ref>《[[新唐书|新唐书]]》卷七十三</ref>

== 早期仕途 ==
陆希声博学，善作文，精通《[[易经|易经]]》、《[[春秋_(史书)|春秋]]》、《[[老子|老子]]》，论-{著}-甚多，主张儒道合一。<ref>[[张跃|张跃]]《唐代后期儒学》，第102页，[[上海人民出版社|上海人民出版社]]，1994年</ref>曾评价已故太子校书郎[[李观|李观]]和被认为与李观不相上下的[[韩愈|韩愈]]：“李观尚文辞，所以文辞胜于说理；[[韩愈|韩愈]]尚思辨，所以说理胜于文辞。即使韩愈老了，也终究不能胜过李观的文辞；即使李观比韩愈后死，也不能达到韩愈的思辨。”<ref>[[:s:新唐书/卷203|《新唐书》卷二百零三]]</ref>曾为[[岭南|岭南]]从事，在[[洪州_(隋朝)|洪州]]石亭观音院遇到仰山大师（法名慧寂，号仰山），成为其门人，并被其称赞为门人之首。大约[[大中_(唐朝)|大中]]七年（853年），陆希声辞别大师。[[咸通|咸通]]初年，[[商州|商州]][[刺史|刺史]][[郑愚|郑愚]]表他为属僚。<ref>郑愚为商州刺史的任期考证详见[[郁贤皓|郁贤皓]]《[[唐刺史考全编|唐刺史考全编]]》。</ref>离职后去[[东平|东平]]辞别仰山大师，隐居于[[宜兴市|义兴]]。久后，于[[乾符|乾符]]初年被召为[[右拾遗|右拾遗]]。<ref>[[:s:周易传序|:s:周易传序]]</ref>当时贪腐官员掌权，连年歉收，以[[汴州|汴州]]、[[宋州|宋州]]最为严重。陆希声见州县凋敝，上言谨防盗贼。次年，如其所警，农民变军[[王仙芝|王仙芝]]发起大规模民变。<ref>据《[[资治通鉴|资治通鉴]]》，王仙芝之乱开始于乾符元年（874年）十二月；则陆希声之表上于同年。见[[:s:資治通鑑/卷252|《资治通鉴》卷二百五十二]]。</ref>陆希声曾在[[襄州_(西魏)|襄州]]，应香严大师门人所请为香严大师作碑文，有“仰山龙从於江西，大安（大师）雨聚於闽越，香严霰於南阳”语。<ref name=YS>[[:s:仰山通智大师塔铭|:s:仰山通智大师塔铭]]</ref>累次升官至[[歙州|歙州]]刺史。陆希声曾教僧人蛩光双钩写字法，后来蛩光进入长安为翰林供奉，而陆希声尚未显达，就写诗给蛩光，即《寄蛩光上人》：“笔下龙蛇似有神，天池雷雨变逡巡。寄言昔日不龟手，应念江湖洴澼人。”蛩光为其所感，推荐陆希声。<ref>《[[鉴诫录|鉴诫录]]》</ref>昭宗闻其名，召为[[给事中|给事中]]。<ref name=NBT116>[[:s:新唐書/卷116|《新唐书》卷一百一十六]]</ref>[[景福_(唐朝)|景福]]年间，曾对尚书郎[[颜荛|颜荛]]说：“颜公（颜上人，名[[薛茂圣|薛茂圣]]）从荆门来拜访我，兴尽而去。无以赠其行，请用知交赋送别。”颜荛应其命，并另写一卷送别，陆希声为其作序。<ref>颜荛《[[:s:颜上人集序|颜上人集序]]》</ref>

== 拜相前后 ==
乾宁二年（895年）正月，昭宗任陆希声为[[户部侍郎|户部侍郎]]、[[同平章事|同平章事]]。但陆希声在相位无所作为。三月，应仰山大师门人光昧所请，抱病为仰山大师撰塔铭。<ref name=YS/>四月，罢为[[太子少师|太子少师]]。<ref>[[:s:新唐书/卷10|《新唐书》卷一十]]</ref>七月，昭宗因为[[凤翔节度使|凤翔节度使]][[李茂贞|李茂贞]]和[[静难节度使|静难节度使]][[王行瑜|王行瑜]]所攻，从长安出逃到[[秦岭|秦岭]]，<ref>[[:s:資治通鑑/卷260|《资治通鉴》卷二百六十]]</ref>陆希声带病随驾，出奔期间去世，赠[[尚书左仆射|尚书左仆射]]，谥'''文'''。<ref name=NBT116/>

== 著作 ==

*《春秋通例》三卷<ref>[[:s:新唐书/卷57|《新唐书》卷五十七]]</ref>，传至[[清朝|清朝]]只余一卷<ref>[[:s:清史稿/卷145|《清史稿》卷一百四十五]]</ref>

*《李观集》三卷，《全唐文》收录有[[:s:唐太子校书李观文集序|《唐太子校书李观文集序》]]，作于[[大顺_(唐朝)|大顺]]元年（890年）十月，时陆希声已为给事中
*《颐山诗》（或名《颐山录》《颐山录诗》）一卷<ref>[[:s:新唐书/卷60|《新唐书》卷六十]]</ref><ref>[[:s:宋史/卷206|《宋史》卷二百零六]]</ref>

*《[[:s:君阳遁叟山集记|君阳遁叟山集记]]》一卷<ref>[[:s:宋史/卷208|《宋史》卷二百零八]]</ref>
