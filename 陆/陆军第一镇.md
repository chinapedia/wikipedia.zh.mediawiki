'''陆军第一镇'''，[[清朝|清朝]]末年军队现代化改革之后的[[新建陆军|新建陆军]]编-{制}-之一，相当于现在的师的规模。

==历史==
[[光绪|光绪]]二十八年十一月初七日（1902年12月6日），[[上谕|上谕]]「现因[[八旗|八旗]]挑选[[兵丁|兵丁]]，著先派三千人交[[袁世凯|袁世凯]]认真训练，期成劲旅。」<ref name=yuan>袁世凯奏议2，天津古籍出版社，1987年，第704页</ref>

[[光绪|光绪]]二十八年十二月二十四日（1903年1月22日），[[袁世凯|袁世凯]]奏《[[遵旨训练旗兵拟订开办章程折|遵旨训练旗兵拟订开办章程折]]》，「为遵旨训练[[旗兵|旗兵]]，谨将拟订开办章程缮单恭折具陈，仰祈[[圣鉴|圣鉴]]事」，上呈袁世凯擬定的《[[訓練旗兵開辦章程|訓練旗兵開辦章程]]》，并奏请添派[[內閣學士|內閣學士]][[鐵良|鐵良]]為[[京旗|京旗]][[練兵翼長|練兵翼長]]。<ref name=yuan/><ref name=liu>劉鳳翰，中國近代軍事史叢書（第一辑）清末与辛亥革命，黃慶中，2008年，第201-202页</ref>蒙准後，即由袁世凯主持，在[[保定|保定]]開始編練'''京旗常備軍'''。该军最初仅有[[步隊|步隊]]4營，[[砲隊|砲隊]]1營、[[馬隊|馬隊]]1營、[[工程|工程]]2隊。編練该军所需的幹部，從袁世凯编練的[[武衛右軍|武衛右軍]]與[[自強軍|自強軍]]調来。<ref name=liu/>

光緒二十九年五月初八日（1903年6月3日），袁世凱以「[[鐵良|鐵良]]到營以來，選用兵丁，聯絡將士，遇事和衷商榷，深資臂助。現蒙恩補授[[戶部|戶部]]右侍郎，職位較崇，似未便仍令參佐軍事，可否請旨特派該侍郎會辦京旗練兵事宜。」翌日，上諭「戶部右侍郎鐵良，著會同袁世凱辦理京旗練兵事宜。欽此。」自此，鐵良由參佐軍事（翼長）變為了會辦京旗練兵大臣。五月初九日（1903年6月4日），又調五品京堂[[那晉|那晉]]襄理軍務。<ref name=liu/>

光绪二十九年十二月二十七日（1904年2月12日），[[袁世凱|袁世凱]]與[[鐵良|鐵良]]会奏该军添营成协，获得清廷批准。<ref name=bjz>北京志: 军事卷 军事志，北京出版社，第244页</ref><ref name=xinjun>中华民国史资料丛稿 专题资料选辑 第二辑 清末新军编练沿革，中华书局，1978年，第93-99页</ref>光绪三十年六月十七日（1904年7月29日），[[袁世凱|袁世凱]]與[[鐵良|鐵良]]会奏称，该军已经完成了一个協的編制。光绪三十一年四月二十八日（1905年5月31日），[[袁世凱|袁世凱]]與[[鐵良|鐵良]]再度上奏获准，添兵成鎮。光绪三十一年五月，改称'''陆军第一镇'''，與北洋常備軍同時改編完成，兵額與[[陆军第四鎮|陆军第四鎮]]相同。<ref name=liu/><ref name=bjz/><ref name=xinjun/>

陆军第一镇辖步兵第一协、第二协，主要由京城旗兵组成。该镇起初驻扎[[保定|保定]]，光绪三十一年十二月十五日（1906年1月9日），[[袁世凱|袁世凱]]與[[鐵良|鐵良]]会奏，拟调该镇一个协移驻[[北京|北京]][[仰山洼|仰山洼]]，“以资拱卫而便征调”，获得清廷批准。光绪三十二年十一月初九日，铁良奏请将陆军第一镇全镇调驻[[北京|北京]][[仰山洼|仰山洼]]，十一月十四日奉旨依议。光绪三十三年九月十二日（1907年10月18日），专司训练近畿陆军各镇副都统[[鳳山_(清朝人物)|凤山]]奏称，陆军第一镇拟于九月二十日、二十一日自保定开拔，十月初二日到达[[北京|北京]][[仰山洼|仰山洼]]新建的兵营驻扎。<ref name=bjz/><ref name=xinjun/>

==后续==
[[中华民国|中华民国]]成立后，该镇更名为'''陆军第一师'''（即中央陆军第一师），首任师长[[何宗莲|何宗莲]]，1914年[[蔡成勳|蔡成勳]]接任师长。<ref>{{Cite book<!--|和書-->|author = 来新夏等|title = 北洋軍閥史 下冊|year = 2000|publisher =  [[南開大学|南開大学]]出版社|isbn = 7-310-01517-7}}</ref><ref>{{Cite book<!--|和書-->|author = 徐友春主編|title = 民国人物大辞典 増訂版|year = 2007|publisher = 河北人民出版社|isbn = 978-7-202-03014-1}}</ref>

1922年6月，陆军第一师入江西，师长[[蔡成勳|蔡成勳]]受命“节制江西全省军队”。陆军第一师辖第一、二旅，驻[[南昌|南昌]]、[[九江|九江]]。1924年12月，[[蔡成勳|蔡成勳]]被赣南镇守使[[方本仁|方本仁]]驱走，该师第二旅旅长[[冯绍闵|冯绍闵]]继任师长。1925年春，中央陆军第一师改编为'''江西陆军第三师'''（[[冯绍闵|冯绍闵]]于1925年1月21日被任命继续担任该师师长）；另一方面则以江西陆军第九混成旅及一个省防步兵团改编为'''中央陆军第一师'''，[[邓如琢|邓如琢]]于1925年1月22日任师长，全师约8000人。1926年，[[国民革命军|国民革命军]]北伐入江西，陆军第一师被歼，邓如琢率残部离开江西。<ref>江西省军事志，江西省军事志编纂委员会，1997年，第58页</ref><ref>章伯锋主编，北洋军阀1912-1928，武汉出版社，1990年</ref><ref>中華民國史檔案資料滙编 第3卷 第3部分，江苏人民出版社，第617页</ref>
