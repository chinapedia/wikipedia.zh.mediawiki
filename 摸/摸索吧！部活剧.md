'''摸索吧！部活剧'''（{{lang|ja|てさぐれ!部活もの}}，{{lang|ja|てさぐれ ぶかつもの}}）是[[日本|日本]]原创[[日本电视动画|动画]]作品。2013年10月至12月在[[日本电视台|日本电视台]]播放第一季，2014年1月播出第二季《'''摸索吧！部活剧 Encore'''》（{{lang|ja|てさぐれ!部活もの あんこーる}}）。

2015年4月与另一部[[四格漫画|四格漫画]]作品《[[见习女神_噗噜噗噜夏露姆|见习女神 噗噜噗噜夏露姆]]》合作改编为第三季《'''摸索吧！部活剧 Spinoff 和噗噜噗噜夏露姆游玩吧'''（{{lang|ja|てさぐれ!部活もの すぴんおふ プルプルんシャルムと遊ぼう}}）》播出<ref>{{Cite web |url=http://comic.qq.com/a/20150302/043350.htm |title=《摸索吧！部活剧》第三季4月4日正式开播 |accessdate=2015-03-03 |work= |publisher= 腾讯动漫|date= 2015-03-02}}</ref>，並延長為30分鐘。

== 概要 ==
[[gdgd妖精s|gdgd妖精s]]演出及[[直球表题機械人動畫|直球表题機械人動畫]]导演[[石馆光太郎|石馆光太郎]]與制片人[[福原慶匡|福原慶匡]]再次合作，全部基于[[计算机图形学|电脑动画]]制作作品<ref name="animate">{{Cite web|url=http://www.animate.tv/news/details.php?id=1376232322&p=1|title=テレビアニメ『てさぐれ！部活もの』石舘監督&福原プロデューサーにインタビュー|work=[[アニメイトTV|アニメイトTV]]|publisher=[[フロンティアワークス|フロンティアワークス]]|date=2013-08-12|accessdate=2013-10-06}}</ref>。本作使用[[三维计算机图形软件|三维计算机图形软件]][[Cinema_4D|Cinema 4D]]和在“直球”一样的[[MikuMikuDance|MikuMikuDance]]进行[[卡通渲染|卡通渲染]]<ref name="animate" />，在官方网站也发布了相应角色的MMD模型文件<ref name="animate" />。同“直球”一样为15分钟片长，在完成配音后再制作相应画面。<ref name="animate" />

故事内容为以主角群在“探索部”进行社团活动，每话分为两部分，分别为讨论“其他社团的特点”与“探索新的社团的特点”，每话针对一个社团进行讨论（除了第一期第11话，为2个）。其中第二部分“探索新的社团的特点”採事前錄音，沒有劇本，而是完全由声优现场憑著自由讨论发挥，即興演出並直接录制成配音，包括了笑场和出错情况<ref>{{Cite web|url=http://news.nicovideo.jp/watch/nw791908|title=声優の“素の声”が満載!? 新アニメ「てさぐれ！部活もの」の見どころはココだ！|work=[[ニコニコニュース|ニコニコニュース]]（[[ザテレビジョン|ザテレビジョン]]）|date=2013-10-05|accessdate=2013-10-06}}</ref>，錄音完成後在依照當時演出完成動畫製作。

第三季起長度延長至30分鐘，除了原本探索部的部分，新增了見習女神們的即興表演，此外雙方的角色也會串場至另一方進行即興表演。下集預告則由萌舞子的家族擔當。

== 登场角色 ==
部员的[[日本人名|苗字]]取自“日本苗字常用排行（{{lang|ja|日本の苗字ランキング}}）”头四个，名字取自“2012年度新生儿儿名常用排行女子组（{{lang|ja|2012年度新生児名前ランキング（女の子）}}）”头10个<ref name="gajet">{{Cite web|url=http://getnews.jp/archives/395874|title=今度のMMDアニメは日常系で“部活”がテーマ！ 2013秋アニメ『てさぐれ！部活もの』制作決定|work=[[ガジェット通信|ガジェット通信]]|date=2013-08-10|accessdate=2013-10-06}}</ref>。

== 相关产品 ==
第一期[[Blu-ray|Blu-ray]]和[[DVD|DVD]]于2013年12月25日分三卷三期发售。第二期的于2014年4月23日分三卷三期发售。第三期的于2015年6月24日分六卷发售。

第二期各卷分别收录四名主角声优邀请其好友来评价本作品的副声轨，分别为：
* 第一卷：西明日香与[[洲崎綾|洲崎綾]]（同为广播节目「[[洲崎西|洲崎西]]」主播）
* 第二卷：明坂聪美与[[本多真梨子|本多真梨子]]（同为[[Amuleto|Amuleto]]配音员同事）
* 第三卷：大桥彩香与[[大坪由佳|大坪由佳]]（动画[[Project575|GO!GO!575]]相关配音）、荻野可鈴与[[志田友美|志田友美]]（[[夢みるアドレセンス|夢みるアドレセンス]]同事）

本系列发售了两卷[[CD|CD]]专辑，分别对应第一期和第二期。第一期名为《探索吧！部活剧 相关音乐集 摸索吧！歌曲剧（{{lang|ja|てさぐれ!部活もの 関連楽曲集 てさぐれ!歌もの}}）》，收录第一期片头曲、片尾曲和四人角色曲与相应伴奏版本；第二期名为《探索吧！部活剧 Encore音乐集 摸索吧！歌曲剧 Encore <br/>（{{lang|ja|てさぐれ!部活もの あんこーる関連楽曲集 てさぐれ!歌もの あんこーる}}）》，收录了片头曲和片尾曲的改编曲、萌舞子角色曲与上述的伴奏版本，另收录本动画作品的OST。

== 漫画版 ==
2013年10月30日至12月20日，由{{lang|ja|鈴音ことら}}作画的四格漫画在《{{tsl|ja|まんがライフWIN|漫画人生WIN}}》（[[竹書房|竹書房]]）连载，共三集。
==现实影响==
第三期第10集中，在关于讨论主题「新颖的理发店」，西明日香所扮演的结爱提出全裸躺在洗头床上洗头，之后更有在分娩床洗头的点子，引发日本网民恶搞，相关的关键词组「西明日香 分娩台」成为了当日yahoo日本搜索热门第一名兼搜索备选词之一和twitter热门关键词。<ref>{{cite web|title=《西明日香的低級梗》聲優還是要慎選台詞 免得自己被惡搞？ {{!}} yam蕃薯藤新聞|url=http://n.yam.com/gamme/otaku/20150616/20150617097031.html}}</ref>
