== 位置 ==
西隔[[丹麦海峡|丹麦海峡]]与[[格陵兰岛|格陵兰岛]]相望，东临[[挪威海|挪威海]]，北面[[格陵兰海|格陵兰海]]，南界[[大西洋|大西洋]]。属[[北欧|北欧]]国家。

== 地质地貌 ==
早中新世晚期以来，由[[大西洋中洋脊|大西洋中洋脊]][[裂谷|裂谷]]溢出的[[上地幔|上地幔]]物质堆积而成，属于[[火山岛|火山岛]]。组成冰岛的[[岩石|岩石]]都是[[火山岩|火山岩]]，以[[玄武岩|玄武岩]]分布最广，还有[[安山岩|安山岩]]、[[流纹岩|流纹岩]]等。

由於冰島處於歐亞板塊和北美洲板塊之間，因此岛上坐擁多座火山，並以“极圈火岛”之名著称，共有[[火山|火山]]200至300座，有40至50座[[活火山|活火山]]。主要的火山有[[拉基火山|拉基火山]]、[[华纳达尔斯火山|华纳达尔斯火山]]、[[海克拉火山|海克拉火山]]與[[卡特拉火山|卡特拉火山]]等等。1963年至1967年在西南岸的火山活動形成了一個約2.1平方-{zh:千米;zh-hans:千米;zh-hk:千米;zh-tw:公里;}-的小島。冰島溫泉的數量是全世界之冠，全島約有250個鹼性溫泉，最大的溫泉每秒可產生200升的泉水。

基本地勢方面，整個冰島是个碗状[[高地|高地]]，四周为海岸[[山脉|山脉]]，中间为一[[高原|高原]]。大部分是[[台地|台地]]，台地高度大多在400至800-{zh:米;zh-hans:米;zh-hk:米;zh-tw:公尺;}-之间，个别[[山峰|山峰]]可达1300至1700-{zh:米;zh-hans:米;zh-hk:米;zh-tw:公尺;}-，冰岛最高峰是[[华纳达尔斯赫努克山|华纳达尔斯赫努克山]]（2119-{zh:米;zh-hans:米;zh-hk:米;zh-tw:公尺;}-）。[[低地|低地]]面积很小，西部和西南部分布有[[海成平原|海成平原]]和[[冰水冲积平原|冰水冲积平原]]，平原面积占全岛的7％左右。無冰川流過的海岸線不規則，多峽灣、小海灣。其他沿海地區主要為沙灘，岸外的沙洲形成潟湖。

[[地热|地热]]资源丰富，[[温泉|温泉]]广佈。

== 气候 ==
地处[[高纬|高纬]]，南部属于[[溫帶海洋性氣候|溫帶海洋性氣候]]，北部属于[[苔原气候|苔原气候]]。

深受[[北大西洋暖流|北大西洋暖流]]影响，主要从其南面流过，并有一支绕行于它的西面和北面。所以虽然地处[[北极圈|北极圈]]附近，[[冬季|冬季]]气温并不低，[[夏季|夏季]]气温全岛在7至12摄氏度之间。

处于[[冰岛低压|冰岛低压]]中心附近，[[天气|天气]]多变。[[气旋|气旋]]给冰岛帶来丰沞的[[降水|降水]]，西南部和西部[[年均降水量|年均降水量]]在1000至2000-{zh:毫米;zh-hans:毫米;zh-hk:毫米;zh-tw:公釐;}-之间，北部和东北部较少，为400至600-{zh:毫米;zh-hans:毫米;zh-hk:毫米;zh-tw:公釐;}-。无论什么[[季节|季节]]，都有可能下[[雨|雨]]和下[[雪|雪]]。

== 水文 ==
河流呈辐射状由中部高原向外流出，主要的河流有：[[菲约德勒姆冰河|菲约德勒姆冰河]]等。

冰岛不仅是[[第四纪|第四纪]][[冰盖|冰盖]]的中心，而且高原上仍有[[现代冰川|现代冰川]]分布，主要是盾形的[[冰帽冰川|冰帽冰川]]，也有少量的[[冰斗冰川|冰斗冰川]]，面积占全岛的11.5％。主要有[[瓦特纳冰原|瓦特纳冰原]]、[[朗格冰原|朗格冰原]]、[[霍夫斯冰原|霍夫斯冰原]]及[[米达冰川|米达冰川]]，其中瓦特纳冰原面积达8450平方-{zh:千米;zh-hans:千米;zh-hk:千米;zh-tw:公里;}-，厚度在几百-{zh:米;zh-hans:米;zh-hk:米;zh-tw:公尺;}-到2千米之间，是除[[南极|南极]]和格陵兰之外世界最大的冰川。

== 植被 ==
[[苔原|苔原]]广布，草地面积占24%，故畜牧业较为发达。森林面积占1.37%左右，分布在背风和向阳的山坡和谷地中，以桦树灌木林为主，近几十年引进不少欧美耐寒松柏，长势良好。
