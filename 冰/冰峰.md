'''冰峰'''也称'''冰峰汽水'''，是西安市糖酒集团有限公司冰峰食品饮料分公司生产的一种桔子味汽水，1953年诞生于[[西安|西安]]。<ref name=legend> {{Cite web |url= http://xian.qq.com/a/20121029/000352.htm#p=2 |title= 揭秘冰峰汽水传奇 |author=  |date= 2012年10月29日 |publisher= 腾讯大秦网 |language= zh-hans |  accessdate= 2013年5月15日 |quote=  }}</ref>冰峰深受陕西人民喜爱，<ref> {{Cite web |url= http://xian.qq.com/a/20121031/000187.htm |title= 冰峰成吃货心头最爱 哪些又是你的“菜” |author=  |date= 2012年10月31日 |publisher= 腾讯大秦网 |language= zh-hans |  accessdate= 2013年5月15日 |quote=  }}</ref>冰峰经常与[[凉皮|凉皮]]和[[肉夹馍|肉夹馍]]合称为“三秦套餐”。<ref name=china> {{Cite web |url= http://finance.china.com.cn/roll/20130402/1364520.shtml |title= 草根明星冰峰冲击全国 |author= 王赫 |date= 2013-04-02 |publisher= 中国网 |language= zh-hans |  accessdate= 2013年5月15日 |quote=  }}</ref><ref> {{Cite web |url= http://xian.qq.com/a/20130401/000046.htm |title= “陕西味道”引爆蓉城 西安冰峰亮相全国糖酒会 |author=  |date= 2013年4月1日 |publisher= 腾讯大秦网 |language= zh-hans |  accessdate= 2013年5月15日 |quote=  }}</ref><ref> {{Cite web |url= http://www.changjiangtimes.com/2012/09/413161.html |title= 品西北风味 读厚重历史 |author=  |date= 2012-09-28 |publisher= 长江商报 |language= zh-hans |  accessdate= 2013年5月20日 |quote=  }}</ref>

==历史==
[[File:Hulutou.jpeg|thumb]]与冰峰]] 
冰峰诞生于1953年，其生产商为西安市食品厂，冰峰是该厂的其中一个生产车间，被厂里人称为“汽水车间”。<ref name=legend/>

20世纪90年代末，冰峰曾和深圳“深宝”有过短暂合作，期间冰峰汽水披过“深宝”的包装进行销售。<ref name=china/>1997年生产冰峰的工厂改名“西安市冰峰食品饮料公司”。

2008年7月，西安成立“西安糖酒集团”，冰峰食品饮料公司成为其分公司。<ref> {{Cite web |url= http://news.cnwest.com/content/2009-11/24/content_2594191.htm |title= 西安冰峰汽水：一个草根饮料明星的本地化生存 |author=  |date= 2009-11-24 |publisher= 西部网 |language= zh-hans |  accessdate= 2013年5月15日 |quote=  }}</ref>

2012年4月，冰峰汽水推出易拉罐包装的汽水。
