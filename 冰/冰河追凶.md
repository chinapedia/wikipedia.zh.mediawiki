{{engname|《冰河追凶》|Lost In White}}，徐伟自编自导，[[梁家辉|梁家辉]]、[[佟大为|佟大为]]主演的一部侦探悬疑电影。中国于2016年4月15日上映。

== 剧情概述 ==
北方冰河里发现了一具男尸，其胃里一种罕见化学原料引起了老警察周鹏的注意。同时南方年轻警察汪豪正在调查一起绑架案。新的尸体让他们相遇，线索直指十年前的事件，两位警察一同找出隐藏在背后的秘密。

== 制作 ==
本片是是摄影师出身的徐伟担任导演的首部作品<ref>{{cite news|url=http://jjckb.xinhuanet.com/invest/2014-07/08/content_511869.htm|title=《冰河追凶》下一部"白日焰火"? |date=2014-07-08 |publisher=21世纪经济报道|accessdate=2015-02-14 }}</ref>。2015年2月8日在[[黑龙江省|黑龙江省]][[伊春市|伊春市]]开机<ref>{{cite news|url=http://ent.163.com/15/0210/07/AI30UDDH000300B1.html|title=《冰河追凶》黑龙江开机 梁家辉佟大为现身片场|date=2015-02-10|publisher=网易|accessdate=2015-02-14}}</ref>。

在伊春市拍摄时，温度为零下40度左右，不少工作人员吐槽温度太低手机也冻死机<ref>{{cite news|url=http://yule.sohu.com/20150213/n408990541.shtml|title=《冰河追凶》零下40度热拍 曹卫宇挨冻直呼刺激|date=2015-02-13|publisher=搜狐|accessdate=2015-02-14}}</ref>。

佟大为表示因梁家辉邀请，接演本片<ref>{{cite news|url=http://ent.ifeng.com/a/20160303/42584120_0.shtml|title=佟大为因梁家辉接演“冰河” 自曝3岁玩真枪终圆警察梦|date=2016-03-03|publisher=凤凰网|accessdate=2016-03-03}}</ref>。

== 宣传 ==
2015年6月13日于[[上海国际电影节|上海国际电影节]]上举行首次新闻发布会，发布了“破冰版”概念海报<ref>{{cite news|url=http://www.chinajilin.com.cn/jlnews/content/2015-06/13/content_3682069.htm|title=长影70周年悬疑大戏《冰河追凶》首发概念海报|date=2015-06-13|publisher=中国吉林网|accessdate=2015-06-14|deadurl=yes|archiveurl=https://web.archive.org/web/20150913221624/http://www.chinajilin.com.cn/jlnews/content/2015-06/13/content_3682069.htm|archivedate=2015-09-13}}</ref>。导演徐伟，主演佟大为、周冬雨、邓家佳、魏晨等出席并亮相红毯<ref>{{cite news|url=http://ent.qq.com/a/20150613/027562.htm|title=周冬雨亮相《冰河追凶》发布会 首演悬疑片|date=2015-06-13|publisher=腾讯|accessdate=2015-06-13}}</ref>。

2016年3月2日片方宣布定档4月15日并发布一款先导预告及“倒春寒”版海报<ref>{{cite news|url=http://ent.ifeng.com/a/20160303/42584204_0.shtml|title=《冰河追凶》定档415 梁家辉佟大为双雄玩命缉凶|date=2016-03-03|publisher=凤凰网|accessdate=2016-03-03}}</ref>，然而预计于当日下午在北京举办的发布会因安全隐患推迟<ref>{{cite news|url=http://weibo.com/5453121809/DkpqIzStD|title=声明|date=2016-03-02|publisher=微博|accessdate=2016-03-03}}</ref>。

3月17日发布主题曲《破绽》<ref>{{cite news|url=http://ent.163.com/16/0317/09/BIBO5LJH000300B1.html|title=《冰河追凶》曝主题曲海报 魏晨献唱《破绽》|date=2016-03-17|publisher=网易|accessdate=2016-03-18}}</ref>和终极海报。

3月29日在北京举办“通缉令”主题发布会，导演徐伟携梁家辉、佟大为亮相。并发布动作特辑曝光了大量动作场面<ref>{{cite news|url=http://news.mtime.com/2016/03/29/1553933.html|title=《冰河追凶》举办冰上发布会|date=2016-03-29|publisher=时光网|accessdate=2016-03-29}}</ref>。

4月111日在北京举行首映发布会，导演徐伟携主演梁家辉、佟大为、周冬雨、邓家佳、魏晨等出席<ref>{{cite news|url=http://ent.ifeng.com/a/20160411/42603794_0.shtml|title=《冰河追凶》首映 梁家辉致敬“尸体”群演|date=2016-04-11|publisher=凤凰网|accessdate=2016-04-14}}</ref>。

9月1日在第73届[[威尼斯国际电影节|威尼斯国际电影节]]市场放映<ref>{{cite news|url=http://finance.ifeng.com/a/20160902/14853788_0.shtml|title=威尼斯电影节开幕片赢掌声 还有哪些影片值得看？|date=2016-09-02|publisher=凤凰网|accessdate=2017-01-02}}</ref>。

== 奖项 ==
2016年9月18日，作为首届意大利中国电影节主竞赛单元入围影片获得评委会特别奖<ref>{{cite news|url=http://finance.ifeng.com/a/20160919/14888611_0.shtml|title=首届意大利中国电影节落幕 搭建中意电影市场桥梁|date=2016-09-19|publisher=凤凰网|accessdate=2017-01-02}}</ref>。
