'''冰島發酵鯊魚肉'''（{{lang-is|Hákarl}}，發音[ˈhauːkʰadl̥]；或{{lang|is|Kæstur Hákarl}}）是一種起源自[[冰島|冰島]]的食品，亦為[[冰島人|冰島人]]在冬季節慶裡會採用的傳統料理，其帶有強烈、類似[[氨|氨]]的刺激氣味<ref name="WS">{{Cite web |url=http://online.wsj.com/news/articles/SB10001424052970204661604577186843056231170?mg=reno64-wsj&url=http%3A%2F%2Fonline.wsj.com%2Farticle%2FSB10001424052970204661604577186843056231170.html |title=You Eat That? |work=The Wall Street Journal |author=Rachel Herz |date=2012年1月28日 |accessdate=2014年3月1日}}</ref>。

== 製作方法 ==
[[File:Hakarl_near_Bjarnahöfn_in_Iceland.JPG|thumb]]

主要製作方法是將[[小頭睡鯊|小頭睡鯊]]身上的肉切下、之後經由四至五個月時間的發酵及晾乾處理所完成。會採用發酵的方法原因是新鮮的小頭睡鯊肉塊裡帶有著大量的[[尿素|尿素]]、或[[三甲胺氧化物|三甲胺氧化物]]，而若讓肉塊擺著發酵腐敗便能慢慢清除該成份；以較適宜食用。<ref>{{Cite web |url=http://voices.yahoo.com/the-mystery-hakarl-rotten-shark-meat-delicacy-from-617158.html?cat=16 |title=The Mystery of Hakarl: Rotten Shark Meat Delicacy From Iceland |work=[[Yahoo|Yahoo]] |author=David Yuen |date=2007年10月26日 |accessdate=2014年3月1日 |deadurl=yes |archiveurl=https://web.archive.org/web/20131029212303/http://voices.yahoo.com/the-mystery-hakarl-rotten-shark-meat-delicacy-from-617158.html?cat=16 |archivedate=2013年10月29日 }}</ref>

傳統發酵方法一般採用在佈滿礫石的海灘上挖個淺洞，之後將去頭切身後的睡鯊放置在淺洞後，用大量的小石頭給蓋起來，讓被石頭壓著的睡鯊肉慢慢將身上殘留的汁液擠出。而現今較現代的方法則有改將鯊魚肉放置具排水功能的塑膠容器裡發酵。

在等待6到12週時間後便會將先前埋在海灘上的鯊魚肉挖出來，之後放在專門處理的地方裡晾乾。而在晾乾的過程中鯊魚肉會在外表產生棕色的痕跡，這些棕色的外皮會在晾乾處理過程結束後切除。

== 食用 ==
在冰島有將發酵鯊魚肉分為帶著淺紅色、較有嚼勁的品種「{{lang|is|glerhákarl}}」；以及以鯊魚腹部一帶肉塊製成、外型較白的「{{lang|is|skyrhákarl}}」。而冰島也多採用將發酵鯊魚肉切成小型正方塊來插著吃<ref>{{Cite web |url=http://gourmetwonderland.com/hakarl-brennivin |script-title=en:Iceland's Wild Culinary Traditions: Hákarl + Brennivín |work=Gourmet Wonderland |author=David Yuen |date=2007年10月26日 |accessdate=2014年3月1日}}</ref>。

== 反應 ==
美國主廚兼作家[[安東尼·波登|安東尼·波登]]曾食用過發酵鯊魚肉後表示著「這是我所嚐過味道最恐怖的東西了」。<ref name="WS"/>而另一英國主廚[[戈登·拉姆齊|戈登·拉姆齊]]在節目《The F Word》上挑戰過試吃發酵鯊魚肉，之後全吐了出來。<ref>{{YouTube|-xhfJRdwHnU|James May eats Bull's Penis and Rotten Shark|m=2|s=48|language=英語}}</ref>

在[[旅遊生活頻道|旅遊生活頻道]]節目《[[古怪食物|古怪食物]]》第2季中，主持人[[安德魯·席莫|安德魯·席莫]]嘗試發酵鯊魚肉時表示著「這氣味是我這輩子遇過最糟的事了」以及「味道嚐起來不像聞起來那麼可怕，不過確實不太適合一些初次想挑戰奇怪食物的人」。另[[探索頻道|探索頻道]]節目《河中巨怪》主持人傑瑞米·韋德試吃時則表示「我的喉嚨好像燒起來了」。
