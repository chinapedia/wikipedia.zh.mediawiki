'''圓廳別墅'''（{{lang-it|La Rotonda}}）是興建於16世紀晚期、地處[[義大利|義大利]][[威欽察|威欽察]]郊外莊園中的一座獨棟式貴族住宅，中文稱作「別墅」。圓廳別墅是[[文藝復興|文藝復興]]時期建築巨匠[[安德烈亞·帕拉第奧|安德烈亞·帕拉第奧]]的代表作<ref name="簡介">{{Cite web |url=http://vr.theatre.ntu.edu.tw/fineart/architect-wt/palladio/rotonda.htm |title=圓廳別墅 |accessdate=2013-07-09 |work=視覺素養學習網 |publisher=[[國立台灣大學|國立台灣大學]] |language=zh }}</ref>，1994年作為[[威尼托的帕拉第奧式別墅|威尼托的帕拉第奧式別墅]]的一部分登錄[[世界遺產|世界遺產]]<ref name="遺產">{{Cite web |url=http://whc.unesco.org/en/list/712 |title=威欽察城和威尼托的帕拉第奧式別墅 |accessdate=2013-07-09 |work=世界遺產名錄 |publisher=[[聯合國教科文組織|聯合國教科文組織]] |language=en }}</ref>。

== 設計及建造歷史 ==
1565年，主教保羅·阿爾梅里科（{{lang|it|Paolo Almerico}}）從梵蒂岡退休返回威欽察，決定在家鄉威尼斯公國定居<ref name="建造史">{{Cite web |url=http://www.villalarotonda.it/Genesi.pdf |title=圓廳別墅歷史 |accessdate=2013-07-09 |format=PDF |work=圓廳別墅官方網站 |language=it }}</ref>。當時'''莊園府邸'''（{{lang|it|Villa}}，又稱'''別墅'''）已經流行起來，貴族和富裕階層紛紛在農村購地興建莊園{{sfn|陳志華|1997|pp=137}}。帕拉第奧設計過相當數量的這類別墅，集中在威欽察的四周郊區，迄今整體或部份保留的仍然有20餘座<ref name="遺產"/>。他很快完成了設計和結構主體的建造工程：根據記錄1571年主教已經在別墅生活了<ref name="建造史"/>（另有一說是1569年）。

建築裝修工程在結構完工後持續了很久，也可能是再次期間不斷調整改建，直至1589年主教去世。1591年別墅被轉手給卡普拉兄弟（{{lang|it|Capra}}），有證據表明此時依然有工程營造。室內部份有多人參與：[[濕壁畫|濕壁畫]]由{{lang|it|Alessandro Maganza}}和{{lang|it|Lodovico Dorigny}}；雕塑由{{lang|it|Lorenzo Rubini}}；線腳抹灰由{{lang|it|Lorenzo Rubini}}、{{lang|it|Ruggero Bascape}}和{{lang|it|Domenico Fontana}}；石雕小裝飾由{{lang|it|G. B. Albanese}}；壁爐由{{lang|it|Bartolomeo Ridolfi}}<ref name="介紹">{{Cite web |url=http://www.boglewood.com/palladio/rotonda.html |title=阿爾梅里科別墅 |accessdate=2013-07-09 |work=帕拉第奧的義大利別墅 |publisher=Boglewood |language=en }}</ref>

== 建築簡介 ==
圓廳別墅坐落於莊園中央的高地上。平面呈正方形，按照縱橫兩條軸線嚴格地對稱佈置，四個立面完全相同。

室外的大臺階直接通往二層。二層的入口[[門廊|門廊]]在立面上特意強調，以6棵素面的[[愛奧尼柱式|愛奧尼]]圓柱支撐起三角形山花，遠遠超出結構上的實際需要。二層正中央是一個直徑12m的圓廳，其穹頂裝飾得很華麗。
