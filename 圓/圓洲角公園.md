'''圓洲角公園'''（[[英文|英文]]：'''Yuen Chau Kok Park'''）位於[[香港|香港]][[新界|新界]][[沙田|沙田]][[城河東|城河東]]，1993年建成，由[[康樂及文化事務署|康樂及文化事務署]]管理，是居民[[晨運|晨運]]的熱門地點。公園原址是[[沙田海|沙田海]]中的[[圓洲角|圓洲角]]一小島，因為發展[[沙田新市鎮|沙田新市鎮]]時進行了填海工程，小島已經成為[[城河東|城河東]]的一部份。

== 設施 ==
公園環繞圓洲角小山綠化地帶興建，佔地1.42[[公頃|公頃]]，山下公園設有卵石步行徑、遊樂場地和流水景色。登山小徑可登上山丘最高點，山腰有觀景亭供遊人休息，山頂設有觀景地圖及日規，山丘最高點有[[地政總署|地政總署]]的三角測量站。

== 事件 ==
