《'''在黑夜與冰山'''》（{{lang-de|'''In Nacht und Eis'''}}；也被稱為'''Der Untergang der Titanic'''）是一齣 1912年6月拍攝並在同年上畫的[[德國電影|德國電影]]，講述同年4月15日鐵達尼號在[[大西洋|大西洋]]沉沒的經過。

== 電影公司 ==
此片由[[德意志帝國|德國]][[柏林|柏林]]的大陸電影公司製作。

== 拍攝 ==
此片大部分鏡頭在[[柏林|柏林]]片廠拍攝，但部分片段在1912年6月的夏季在[[德意志帝國|德國]]城市[[漢堡|漢堡]]的[[船塢|船塢]]及遠洋[[郵輪|郵輪]][[奥古斯特·维多利亚號|奥古斯特·维多利亚號]]拍攝，約請了三十位演員。

[[柏林|柏林]][[消防局|消防局]]為片廠提供[[水|水]]，以便拍攝船隻下沉片段。電影公司特地製造一艘縮小版的鐵達尼號[[模型|模型]]，用於拍攝從船外觀看鐵達尼號下沉的橋段。此片有不少橋段是[[虛構|虛構]]，如片中鐵達尼號下沉時，[[煙囪|煙囪]]冒火，[[鍋爐|鍋爐]]房[[爆炸|爆炸]]，乘客同唱[[聖詩|聖詩]]等場面。雖然拍攝的科技遠不及[[鐵達尼號_(1997年電影)|鐵達尼號]]，但片長已達35分鐘，遠超同期[[默片|默片]]，在當時的拍攝技術而言也算高超。

== 遺失 ==
1914年，[[第一次世界大戰|第一次世界大戰]]爆發，德國進入戰爭狀態，《在黑夜與冰山》一如同時代其他默片，也被認為永久遺失，無法被尋回。

== 重見天日 ==

在1998年2月初，一名[[德國|德國]][[電影|電影]]收集者突然發現自己有此片的私人珍藏，遂將該片公開，並廣為全球媒體報道<ref>[[路透社|路透社]]，1998年2月</ref><ref>[[香港蘋果日報|香港蘋果日報]]，1998年2月9日C版(娛樂版)：史上第一齣鐵達尼號影片　菲林失傳多年終重現人間</ref>。現時，此片的不同鏡頭可見於[[紀錄片|紀錄片]][[鐵達尼號|鐵達尼號]]以及網路視訊，如[[YouTube|YouTube]]。

== 片中重大錯誤 ==

* 事實上，[[鐵達尼號|鐵達尼號]][[愛德華·約翰·史密斯|船長]]並無在沉船前一夜參與位於巴黎咖啡廳的晚宴，而是在 A la carte 餐廳參與[[晚上|晚上]][[派對|派對]]。

* 事實上，[[鐵達尼號|鐵達尼號]]船首只有一枝[[桅桿|桅桿]]，但在片中，船頭卻出現三枝桅桿，分別是中間的主桅桿和兩側用於支撐的桅桿。

* 事實上，[[樂隊|樂隊]]並無在巴黎咖啡廳演出，而是在頭二等艙的晚餐間演出。

* 事實上，鐵達尼號的散步甲板只位於A層並且在近船頭處閉合，但片中出現三層散步甲板，位於A 至 C 層，且所有部分均和[[奧林匹克號|奧林匹克號]]一樣，不存在閉合結構。

* 事實上，[[鐵達尼號|鐵達尼號]]的[[墩堠|瞭望台]]沒有[[望遠鏡|望遠鏡]]，此儀器卻在電影出現。

* 事實上，[[鐵達尼號|鐵達尼號]]的[[無線電|無線電]][[通訊|通訊]]室並無[[窗戶|窗戶]]，但片中的無線電室卻有一扇窗。
