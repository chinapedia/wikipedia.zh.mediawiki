'''笠置町'''（{{jpn|j=笠置町|hg=かさぎちょう|rm=Kasagi chō|f=y}}）是位於日本[[京都府|京都府]]南部的行政區劃。轄內約八成區域為山林地，{{tsl|ja|木津川 (京都府)|木津川}}自東向西由轄區中間通過主要是區位於木津川與其支流布目川、打瀧川、横川匯流處。

目前人口不到2,000人，為京都府內人口最少的行政區劃，也是目前全日本人口第二少的[[町|町]]級行政區劃（因[[福岛第一核电站事故|福岛第一核电站事故]]而須撤離的區域除外）。

== 历史 ==
西元7世紀時，[[天武天皇|天武天皇]]在此建立了{{tsl|ja|笠置寺}}；8世紀時，為興建[[平城京|平城京]]，從木津川上游運送大量木材至奈良，而在此地建立了港口。<ref name="History">{{Cite web| url=http://www.town.kasagi.lg.jp/contents_detail.php?co=cat&frmId=137&frmCd=1-1-0-0-0| title=沿革| publisher=笠置町公所| date=2008-10-28| accessdate=2017-09-30| language=ja}}</ref>

14世紀[[後醍醐天皇|後醍醐天皇]]為推翻[[镰仓幕府|镰仓幕府]]引起[[元弘之亂|元弘之亂]]，期間後醍醐天皇曾逃至此地，幕府軍追擊此地後發生[[笠置山之戰|笠置山之戰]]並逮捕後醍醐天皇，此後也進一步造成[[南北朝時代_(日本)|南北朝時代]]的開始。<ref name="History"/>

[[江戶時代|江戶時代]]後原先為{{Tsl|ja|伊賀上野藩}}的領地，在1615年伊賀上野藩遭到廢藩後，成為[[津藩|津藩]]的領地。1871年[[廢藩置縣|廢藩置縣]]後改隸屬津縣，但僅約四個月後此地即被整併到[[京都府|京都府]]。1889年實施[[町村制|町村制]]，正式設立笠置村，1934年笠置村改制為現在的笠置町。<ref name="History"/>

== 交通 ==
1897年[[關西鐵道|關西鐵道]]開通了{{tsl|ja|伊賀上野駅|伊賀上野車站|上野車站}}（現已更名為伊賀上野車站）至{{Tsl|ja|加茂駅_(京都府)|加茂車站_(京都府)|加茂車站}}路段，同時在此設立了{{tsl|ja|笠置駅|笠置車站}}。

== 觀光資源 ==
在1932年笠置山周邊沿著木津川一的區域已被列為國家史蹟名勝地，此地也被京都府劃為「府立笠置山自然公園」，周邊主要景點包括後醍醐天皇行宮遺跡以及笠置寺。
