'''移民加拿大'''是指人们迁徙至[[加拿大|加拿大]]，并最终在该国永久定居的过程。这些移民中的大部分成为[[加拿大公民|加拿大公民]]。1947年后，加拿大国内移民法经历了很多重大的修改，特别值得一提的是[[1976年加拿大移民法案|1976年加拿大移民法案]]和2002年开始沿用至今的[[移民和避难者保护法案|移民和避难者保护法案]]。

在加拿大的移民可以分为四种：家庭原因移民（有[[直系亲属|直系亲属]]是加拿大居民），[[经济移民|经济移民]]（熟练工人和行业人士），其他（因[[人道主义|人道主义]]或体恤理由而接受的移民）以及避难者（[[政治庇護|逃避迫害、酷刑或残酷惩罚]]的人）。

目前，加拿大以具有宽松的移民政策而闻名，这一特点体现在加拿大的[[种族多元性|种族多元性]]上。根据[[加拿大统计局|加拿大统计局]] 2001年的统计数据，加拿大十万人以上的[[族群|族群]]有34个，其他小族群不计其数。16.2%的人口属于[[有色人种|有色人种]]：人数最多的是[[南亚裔|南亚裔]]（占总人口的4.0%），[[华裔|华裔]]（3.9%），[[黑人|黑人]]（2.5%）和[[菲律宾人|菲律宾裔]]（1.1%）。而比例上超过有色人种的少数族裔是[[不列颠|不列颠]]，[[爱尔兰|爱尔兰]]和[[法国|法国]]以外的[[白色人种|白色人种]]，其中最多的是[[德国人|德国裔]]（10.18%）和[[意大利人|意大利裔]]（4.63%），另外3.87%是[[乌克兰人|乌克兰裔]] ，3.87%是[[荷兰人|荷兰裔]]，3.15%是[[波兰人|波兰裔]]。（“[[印第安人|北美印第安人]]”是指可能包括来自[[美国|美国]]和[[墨西哥|墨西哥]]的群体，但其中大部分不属于移民，占全国人口的4.01%。）其他白色少数族裔包括[[俄罗斯人|俄罗斯裔]]（1.60%），[[挪威人|挪威裔]]（1.38%），[[葡萄牙人|葡萄牙裔]]（1.32%）和[[瑞典人|瑞典裔]]（1.07%）<ref name="census-proportion">{{cite web

== 历史 ==
{{main|加拿大移民史|加拿大国籍法历史}}
[[File:Canada_immigration_graph.png|400px]] |accessdate=2010-10-18 |deadurl=yes |archiveurl=https://web.archive.org/web/20080108010629/http://www40.statcan.ca/l01/cst01/demo03.htm?sdi=immigration |archivedate=2008-01-08 }}{{en icon}}</ref>]]
经过早期的不列颠、法国[[殖民|殖民]]统治， 在接近两个世纪的时间里，共有四波主要的移民定居潮。目前正在进行的是第五波。

第一波显著的加拿大外来移民潮延续了近两个世纪，这波移民潮主要是法国裔移民在[[魁北克省|魁北克省]]和[[阿卡迪亚|阿卡迪亚]]缓慢有序地定居下来，同时也有小部分的欧美[[企业家|企业家]]和不列颠军人。随着逃离美国[[独立战争|独立战争]]的[[亲英分子|亲英分子]]的大量涌入，这波移民潮才停息下来，这些移民主要是从[[美国中部濒临大西洋诸州|美国中部濒临大西洋诸州]]迁徙到今天的[[安大略省|安大略省]]南部，魁北克省[[东城|东城]]，[[新不伦瑞克省|新不伦瑞克省]]和[[新斯科舍省|新斯科舍省]]。因清拆土地而来的[[苏格兰高地|苏格兰高地人]]也在此期间成为加拿大的土地所有者。

第二波移民潮来自不列颠和[[爱尔兰|爱尔兰]]。[[1812年战争|1812年战争]]以后，加拿大殖民统治者鼓励移民在加拿大定居，包括曾在战争期间服役的[[英国军队|不列颠正规军]]军人。殖民统治者担心美国人入侵，并且意图抵制魁北克的法语势力的影响，所以急于促使移民在有序地带沿新建成的木板路定居，这些地方属于人口稀疏的乡村，大部分在[[上加拿大|上加拿大]]（今日的安大略省）。第二波移民潮中爱尔兰人越来越多地迁徙到加拿大，并在1846年到1849年的[[爱尔兰大饥荒|爱尔兰大饥荒]]时达到顶峰，这次饥荒致使数十万爱尔兰人来到加拿大海岸，尽管其中很大一部分先后改道去了美国。

加拿大人外迁到美国的数量超过美国迁入加拿大的移民，这一点有据可查。但历史上也曾有短暂的几个时期，情况相反。举例来说，独立战争时期的亲英逃难者；卡里布/弗雷泽淘金热和之后的[[克朗代克淘金热|克朗代克淘金热]]期间也有很多美国淘金者在[[不列颠哥伦比亚|不列颠哥伦比亚]]和[[育空|育空]]定居；20世纪初期和某些政治动乱、战争时期（如[[越南战争|越南战争]]）有移民者从北部平原迁徙至北美大草原。

第三波移民潮主要来自欧洲本土，在[[第一次世界大战|第一次世界大战]]之前的1911年-1913年达到顶峰（1913年有超过40万移民）。第四波移民潮同样来自欧洲，这次发生在1957年，有28万2千人移民，这批数量可观的非英语、非法语人口使加拿大的文化更加多元化。例如， [[乌克兰|乌克兰裔]]加拿大人占据乌克兰和[[俄罗斯|俄罗斯]]以外最多的乌克兰人口。移民数量的下降也时有发生，特别是第一次世界大战和[[第二次世界大战|第二次世界大战]]期间，此外还有[[大萧条|美国大萧条]]时期 。

[[File:Come_To_Stay.png|thumb]]》，内容涉及移民到“[[自治领|加拿大自治领]]”]]20世纪70年代开始，来自[[第三世界|第三世界]]的有色族裔成为移民中的压倒性多数。[[1976年加拿大移民法案|1967年移民法案]]进行了修订，这一情况受到了很大的影响，这一法案作为加拿大官方政策沿用多年。[[马丁·布赖恩·马尔罗尼|马尔罗尼]]执政时期，移民的许可标准被提高。到20世纪80年代末，第五波移民潮和之前相比略有增长（每年22万5千到27万五千）。目前大多数移民来自[[南非|南非]]的英國裔與同樣是大英國協的印度人和來自中国、韓國的東亞移民，这一趋势有望继续保持。

1885年以前，限制移民主要是为了应对大股的移民潮而不是有计划的政治决策，并且至少在官方政策里没有专门针对某个群体或者民族。而在1885年，[[人头税|加拿大华人人头税]]法令首次通过，以应对为[[加拿大太平洋铁路|加拿大太平洋铁路]]工作的华人劳工的增长。1900年和1903年，人头税相继增高，限制了华人来到加拿大。1923年，政府通过了《华人移民法案》，在1923年到1947年间禁止华人进入加拿大。因历史时期对华人移民的歧视，2006年6月22日，加拿大政府进行了官方道歉声明，并宣布将对华人移民作出补偿。

2017发布的《胡润报告》称，中国富裕人群海外移民目的地国家中，美国是中国富豪最青睐的移民目的地，加拿大排名第二，英国名列第三。<ref>{{Cite news|url=http://www.bbc.com/zhongwen/simp/business-40640591|title=胡润：中国富人移民对英国兴趣下降|date=2017-07-18|newspaper=BBC 中文网|accessdate=2017-08-15|language=en-GB}}</ref>
