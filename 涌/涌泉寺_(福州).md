'''福州涌泉寺'''，又称'''国师馆'''、'''鼓山白云峰涌泉禅院'''，素有“'''闽刹之冠'''”的称誉。位于[[中國|中國]][[福建|福建]][[福州|福州]]郊區的[[鼓山|鼓山]]上，為中國[[汉族地区佛教全国重点寺院|汉族地区佛教全国重点寺院]]之一。
== 歷史 ==
涌泉寺始建于[[唐|唐]][[建中|建中]]四年（783年），[[五代|五代]][[后梁|后梁]][[开平|开平]]二年（908年），[[神晏|神晏]]主持重修，并改名为“国师馆”。[[后梁|后梁]][[乾化|乾化]]五年（915年），改名为鼓山白云峰涌泉禅院。[[明|明]][[永乐_(明朝)|永樂]]五年（1407年）称涌泉寺。后经过历代修葺，[[清|清]][[康熙|康熙]]三十八年（1699年），[[康熙皇帝|康熙皇帝]]御赐“涌泉寺”金匾<ref>[http://www.tourunion.com/spot/jd/4216.htm 涌泉寺]</ref>。

台湾传统的四大佛教法脉[[月眉山|月眉山]]系、[[观音山|观音山]]系、[[法云寺|法云寺]]系、[[大岗山|大岗山]]系皆与涌泉寺有法脉联系。

== 建筑 ==
涌泉寺現有建筑多為明清兩代所建，基本維持了明朝[[嘉靖|嘉靖]]年間的格局，現有殿堂25個，包括了天王殿、大雄宝殿、法堂、方丈室、祖堂等，還有钟鼓楼、放生池、歲寒樓等建筑，全寺占地面積為{{convert|16650|m2}}，整個寺廟建筑群依山勢逐級升高，錯落有致。<ref name="涌泉寺">[http://www.fjnet.com/msgc/nr/200812/t20081229_98822.htm 涌泉寺]，佛教在線網站</ref>

== 文物 ==
涌泉寺曾为重要印经所，保留有[[明朝|明]]、[[清朝|清]]及近代的[[雕版印刷|雕板]]2万多块。其中包括[[元朝|元]]《[[延祐藏|延祐藏]]》、《[[麻沙藏|麻沙藏]]》、《[[永乐北藏|永乐北藏]]》、《[[永乐南藏|永乐南藏]]》、《[[清朝梵本|清朝梵本]]》、《[[书本藏|书本藏]]》、《[[日叙本藏|日叙本藏]]》等<ref>[http://tw.people.com.cn/GB/26741/26746/26786/879369.html 福州涌泉寺]</ref>。涌泉寺前有建于11世紀北宋[[元豐_(北宋)|元豐]]五年（1082年）的兩座千佛陶塔，為中國[[全國重點文物保護單位|全國重點文物保護單位]]。<ref name="第五批重点文物保护单位：鼓山摩崖石刻(石窟寺及石刻)">[http://www.godpp.gov.cn/wmzh/2007-09/18/content_11182537.htm 第五批重点文物保护单位：鼓山摩崖石刻(石窟寺及石刻)]，文明中華網站</ref>

== 高僧 ==

*灵峤禅师：涌泉寺前身华严寺（唐[[建中_(唐朝)|建中]]四年，783年）开山祖师，唐代[[洪州宗|洪州宗]]僧。受[[马祖道一|马祖道一]]印可。
*神晏法师：[[五代|五代]]唐末高僧。[[闽国|闽国]]国师。谥号‘兴圣国师’。在华严寺旧址上重修涌泉寺（国师馆）。
*永觉元贤禅师：[[明|明]]末高僧，[[曹洞宗|曹洞宗]][[寿昌县|寿昌]]系鼓山法脉（亦称鼓山禅、鼓山系）一代传人。
*为霖道霈禅师：字为霖，法讳道霈，号旅泊，两任涌泉寺住持，[[明末清初|明末清初]]著名的禅宗大师，时人称之为古佛再世。
*妙莲禅师：[[清|清]]代僧。福建[[归化县_(福建)|归化]]人，俗姓冯。名地华，别号云池。[[临济|临济]]龙池系法派四十二世。创建[[槟城极乐寺|槟城极乐寺]]、[[漳州南山寺|漳州南山寺]]等。
*古月禅师：古月为近代福建最有影响的僧人之一。以一身主鼓山涌泉寺、雪峰[[雪峰寺|崇圣寺]]、象峰崇福寺、怡山[[西禅寺|西禅寺]]、瑞峰[[林阳寺|林阳寺]]五大禅林。
*[[虚云法师|虚云法师]]：中国近代禅宗泰斗。中国[[曹洞|曹洞]]四十七代，[[临济|临济]]四十三代，[[云门|云门]]第十二代，[[法眼|法眼]]第八代，[[沩仰|沩仰]]第八代。一人延续禅宗五家法脉。
*[[释圆瑛|圆瑛法师]]：中国近代佛教领袖，[[中国佛教协会|中国佛教协会]]第一任会长。
