'''幾何化單位制'''（{{lang|en|geometrized unit system}}），不是一種完全定義或唯一的單位制。在這單位制內，只規定光速與重力常數為1，即<math> c = G = 1</math> 。這樣留出足夠空間來規定其它常數，像[[波茲曼常數|波茲曼常數]]或[[库仑常數|库仑常數]]：

:<math> k_B = 1</math>、
:<math> \frac{1}{4 \pi \epsilon_0} = 1 </math>。

假若約化普朗克常數也規定為 <math> \hbar = 1</math>，則幾何化單位制與[[普朗克單位制|普朗克單位制]]完全相同。

另外，我們也可以不定義庫侖常數為1，而改定義更自然的[[電常數|電常數]]<math>\epsilon_0</math>為1，此時，庫侖常數就會變成<math>\frac{1}{4 \pi}</math>，這是比較自然的有理化幾何單位制，而如果是定義庫侖常數為1，則是非理化的幾何單位制。（我們通常會選擇比較自然的常數定義為1，例如我們不會把原始的[[普朗克常數|普朗克常數]]<math>h</math>定義為1，而是會把[[約化普朗克常數|約化普朗克常數]]<math>\hbar</math>定義為1，因為約化普朗克常數比較自然）

在[[廣義相對論|廣義相對論]]中，<math>G</math>經常會與<math>4 \pi</math>合併，故此時的幾何單位制定義為：

:<math> c = 4 \pi G = \hbar = \epsilon_0 = \mu_0 = Z_0 = k_B = 1</math>
:<math> e = \sqrt{4\pi\alpha}</math>

注意此時的[[萬有引力常數|萬有引力常數]]與[[庫侖常數|庫侖常數]]的值相同，都是<math>\frac{1}{4 \pi}</math>。

== 单位换算 ==
转变 kg, s, C, K 至 m :

*''G''/''c''<sup>2</sup> [m/kg]
*''c'' [m/s]
*((''G''/(4''πε''<sub>0</sub>))<sup>0.5</sup>)/''c''<sup>2</sup> [m/C]
*(''Gk'')/''c''<sup>4</sup> [m/K]

转变 m, s, C, K 至 kg :

*''c''<sup>2</sup>/''G'' [kg/m]
*''c''<sup>3</sup>/''G'' [kg/s]
*1/(''G''×4''πε''<sub>0</sub>)<sup>0.5</sup> [kg/C]
*''k''/''c''<sup>2</sup> [kg/K]

转变 m, kg, C, K 至 s :

*1/''c'' [s/m]
*''G''/''c''<sup>3</sup> [s/kg]
*((''G''/(4''πε''<sub>0</sub>))<sup>0.5</sup>)/''c''<sup>3</sup> [s/C]
*(''Gk'')/''c''<sup>5</sup> [s/K]

转变 m, kg, s, K 至 C :

*''c''<sup>2</sup>/((''G''/(4''πε''<sub>0</sub>))<sup>0.5</sup>) [C/m]
*(''G''×4''πε''<sub>0</sub>)<sup>0.5</sup> [C/kg]
*''c''<sup>3</sup>/((''G''/(4''πε''<sub>0</sub>))<sup>0.5</sup>) [C/s]
*(''k''×(''G''×4''πε''<sub>0</sub>)<sup>0.5</sup>)/''c''<sup>2</sup> [C/K]

转变 m, kg, s, C 至 K :

*''c''<sup>4</sup>/(''Gk'') [K/m]
*''c''<sup>2</sup>/''k'' [K/kg]
*''c''<sup>5</sup>/(''Gk'') [K/s]
*''c''<sup>2</sup>/(''k''×(''G''×4''πε''<sub>0</sub>)<sup>0.5</sup>) [K/C]
