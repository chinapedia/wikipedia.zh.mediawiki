'''潇湘职业学院'''是位于[[湖南省|湖南省]][[娄底市|娄底市]][[经济技术开发区|经济技术开发区]]，为[[湖南省|湖南省]]的一所民办高级[[专科大学|专科大学]]。<ref name=" gfw1" >{{cite web |url=http://www.hnxxc.com/about.asp |title=学校概况 |accessdate= 2011年|work= |publisher=潇湘职业学院官方网 |language =zh-cn}}</ref>

== 历史 ==
潇湘职业学院始建于1992年，位于[[湖南省|湖南省]][[娄底市|娄底市]]，紧靠[[涟钢集团|涟钢集团]]基地。<ref name=" gfw1" />

== 学校院系 ==
潇湘职业学院拥有四大院系：
*有机电工程学院
*汽车工程学院
*大汉城建工程学院
*潇湘商学院

== 学校专业 ==

潇湘职业学院开设了28个专业，主要专业有：
*汽车检测与维修
*数控技术
*广告设计
*动漫
*软件技术
*模具设计与制造
*应用电子技术
*市场营销
*商务英语
