*本表為[[恆星天文學|恆星天文學]][[年代學|年表]]，以下譯自英文維基同名條目。
== 18世紀之前 ==
* [[前134年|前134年]] - [[喜帕恰斯|喜帕恰斯]]（{{lang|el|Ἳππαρχος}}）創造[[視星等|恆星的視亮度]]（[[光度|光度]]）的概念，並編制了有1,025顆[[恆星|恆星]]的[[星圖|星圖]]。
* [[1596年|1596年]] - [[大衛·法比利薩斯|大衛·法比利薩斯]]注意到[[米拉|米拉]]的光度變化。
* [[1672年|1672年]] - [[Geminiano_Montanari|Geminiano Montanari]]注意到[[大陵五|大陵五]]的光度變化。
* [[1686年|1686年]] - [[Gottfried_Kirch|Gottfried Kirch]] 注意到[[天鵝座χ|天鵝座χ]]的光度變化。

== 18世紀 ==
* [[1718年|1718年]] - [[愛德蒙·哈雷|愛德蒙·哈雷]]經由比較希臘時代的天體位置記錄，發現恆星[[自行|自行]]。
* [[1782年|1782年]] - [[约翰·古德利克|约翰·古德利克]]注意到[[大陵五|大陵五]]的光度變化，並推測是[[食雙星|食雙星]]。
* [[1784年|1784年]] - [[约翰·古德利克|约翰·古德利克]]發現第一顆[[造父變星|造父變星]]（[[仙王座|仙王座]]的[[造父一|造父一]]）。

== 19世紀 ==
* [[1838年|1838年]] - [[Thomas_James_Henderson|Thomas James Henderson]]、[[瓦西里·雅可夫列维奇·斯特鲁维|瓦西里·雅可夫列维奇·斯特鲁维]]和[[弗里德里希·威廉·贝塞尔|白塞耳]]測量出恆星的[[視差|視差]]。
* [[1844年|1844年]] - 白塞耳解釋[[天狼星|天狼星]]和[[南河三|南河三]]的位置擺動是因為有看不見的暗淡伴星。

== 20世紀 ==
* [[1906年|1906年]] - [[亚瑟·爱丁顿|亚瑟·爱丁顿]]開始用統計學研究恆星的運動。
* [[1908年|1908年]] - [[亨丽爱塔·勒维特|李維特]]發現[[造父變星|造父變星]]的[[周光關係|周光關係]]。
* [[1910年|1910年]] - [[埃希纳·赫茨普龙|赫茲史普]]和[[亨利·诺利斯·罗素|羅素]]各自研究恆星的光度和[[光譜|光譜]]的關聯。
* [[1924年|1924年]] - [[亚瑟·爱丁顿|亚瑟·爱丁顿]]發展出[[主序星|主序星]]的質光關係。
* [[1929年|1929年]] - [[乔治·伽莫夫|乔治·伽莫夫]]提出[[氫|氫]]的[[核聚變|核融合]]是恆星能量的來源。
* [[1938年|1938年]] - [[汉斯·贝特|汉斯·贝特]]和[[Carl_von_Weizsacker|Carl von Weizsacker]]詳述恆星內部的[[pp鏈|pp鏈]]和[[碳氮氧循環|碳氮氧循環]]。
* [[1939年|1939年]] - [[Rupert_Wildt|Rupert Wildt]]認為負氫是使恆星變得不透明的重要[[離子|離子]]。
* [[1952年|1952年]] - [[沃尔特·巴德|沃尔特·巴德]]區分出造父變星有造父Ⅰ和造父Ⅱ兩類。
* [[1953年|1953年]] - [[弗雷德·霍伊尔|弗雷德·霍伊尔]]預測[[碳|碳]]12在恆星內部的高溫下可以經由[[三氦反應|三氦反應]]生成。
* [[1961年|1961年]] - [[林忠四郎|林忠四郎]]發表恆星是完全對流體時，演化路徑的[[林軌跡|林軌跡]]。
* [[1963年|1963年]] - [[弗雷德·霍伊尔|弗雷德·霍伊尔]]和[[William_A._Fowler|William A. Fowler]]提出超重質量恆星的想法。
* [[1964年|1964年]] - [[苏布拉马尼扬·钱德拉塞卡|钱德拉塞卡]]和[[理查德·费曼|理查德·费曼]]發展出脈動星在[[廣義相對論|廣義相對論]]下的理論，並且表示超重質量恆星的不穩定性受到廣義相對論的支配。
* [[1967年|1967年]] - [[Eric_Becklin|Eric Becklin]]和[[Gerry_Neugebauer|Gerry Neugebauer]]在10微米的波長發現[[BN天体|BN天体]]。
