'''亨德里·佩奇有限公司'''（{{lang-en|'''Handley Page Limited'''}}）又译'''汉德利·佩季'''，由[[弗雷德里·亨德里·佩奇|弗雷德里·亨德里·佩奇]]（Frederick Handley Page）於1909年創立，它是[[英國|英國]]第一家公開交易的飛機製造公司。公司總部設在[[赫特福德郡|赫特福德郡]]的[[Radlett|Radlett]]機場（Aerodrome），公司於 1970年[[自愿停止營業|自愿停止營業]]（voluntary liquidation）。公司以生產重型轟炸機和干線飛機聞名於世。

==歷史==
弗雷德里克·罕德列·貝吉最先在[[Woolwich|Woolwich]], [[Fambridge|Fambridge]]和[[Barking_Creek|Barking Creek]]這幾個地點實驗和建造了幾架[[雙翼機|雙翼機]]和[[單翼機|單翼機]]。他的公司成立於1909年6月17日，成為了英国首家公开上市的航空企业。

1912年罕德列·貝吉在从Barking搬到[[Cricklewood|Cricklewood]]后建立了航空器工厂，从那以后直到公司退出航空领域，航空器都是在这家工厂里建造的。

===一戰===
在[[一戰|一戰]]期間，罕德列·貝吉為[[英國皇家空軍|英國皇家空軍]]生產了一系列[[重型轟炸機|重型轟炸機]]（heavy bomber），它們被用來轟炸[[德國|德國]]的[[齊柏林飛艇|齊柏林飛艇]]塢，其最終目的是為了轟炸柏林以報復齊柏林飛艇對倫敦的轟炸。[[英國海軍部|英國海軍部]]要求罕德列·貝吉生產一種“使敵人失去勇氣的飛機”（bloody paralyser of an aeroplane）。這些飛機包括1915年產的[[Handley_Page_O/100|O/100]]、1918年產的[[Handley_Page_O/400|O/400]]和航程可達柏林的四引擎飛機[[Handley_Page_V/1500|V/1500]]，V/1500於戰爭即將結束的[[1918_in_aviation|1918]]年才進入服役。
===两次世界大戰之間時期===
在戰後不久，罕德列·貝吉公司改造了一些O/400型飛機，使之成為客運飛機，這些飛機做為'''Handley Page Transport''公司的航班，飛行在[[倫敦|倫敦]]—[[巴黎|巴黎]]航線上。考慮到V/1500型飛機對當時來話實在是太大了，但是一些V/1500型飛機的設計元素不久以後還是融合在O/400型飛機的機身上並推出了公司的第一個專門的客機設計——[[Handley_Page_W.8|W.8]]。1924年Handley Page Transport與2個其它地區性航線合併成[[Imperial_Airways|Imperial Airways]]（提供了英國第一個全國性航線服務）。罕德列·貝吉公司開發了幾種大型[[雙翼機|雙翼]]客機。包括用於[[大英帝國|大英帝國]]到非洲和印度的定期航線上的8座豪華型[[Handley_Page_H.P.42|Handley Page H.P.42]]。

罕德列·贝吉也自主开发了著名的“罕德列·贝吉前缘缝翼”（Handley Page Slat，又可称为slot,参见[[前缘缝翼|前缘缝翼]]（Leading edge slats）），就是一个切入机翼前缘的小槽，可以在高[[攻角|攻角]]（angle of attack）情况下改善空气流。这个设计是如此成功，以至于别的公司给的生产许可费成了他们公司在1920年代前几年的主要收入。

在1929年，位于Cricklewood的机场被关闭，同时在[[Radlett|Radlett]]建了一个新的，在那里有最多的飞机在建造，但是在Cricklewood制造飞机的工作直到1964年才停止，这那一年，那个生产场地被卖了，并成为Cricklewood trading estate。

===二戰===
当[[二战|二战]]的阴云初现，罕德列·貝吉回到轟炸機設計的工作上，並生產出[[Handley_Page_Hampden|HP.52 Hampden]]，它參加了第一次從英國空襲[[柏林|柏林]]的戰鬥，做為政府要求載重更大、航程更遠的飛機的回應，罕德列·貝吉設計了裝了兩台[[Rolls-Royce_Vulture|Rolls-Royce Vulture]]引擎的HP.56 ，但後來被證實有很多問題未解決，因此生產本機的計劃被取消，但是這個設計很多優點被帶到了四引擎的[[Handley-Page_Halifax|HP.57 Halifax]]，它是排在[[蘭開斯特轟炸機|蘭開斯特轟炸機]]後的英國產量最大的重型轟炸機之一，這雖然與蘭開斯特轟炸機的研發路徑相同，哈利法克斯還是需要一些更多的後續開發工作，很快它就達到了成熟階段，並被看成是一種優秀的飛機。{{Fact|date=March 2007}}.

===戰後===
戰後英國政府發出標書，需要一種能帶攜帶國家[[核威懾|核威懾]]的噴氣轟炸機。那就是被稱為[[V_bomber|V型轟炸機]]的三種飛機。罕德列·貝吉貢獻了其中一種——[[Handley_Page_Victor|HP.80 Victor]]，它是一架四引擎、月牙型機翼設計。這種飛機在建造它的公司停業後還在服役（做為[[空中加油機|空中加油機]]）。

在[[1947_in_aviation|1947]]年，罕德利·貝吉購買了破產的[[Miles_Aircraft|Miles Aircraft]]公司的一些資產，這些資產包括已存在的設計、工具、模板（最重要的是[[Miles_M.52|Miles M.52]]超音速研究飛機）和位於Woodley的[[Miles_Aircraft|Miles Aircraft]]公司[[Reading,_Berkshire|Reading]]的場地，整個商業操作是這樣進行的：由'''Handley Page (Reading) Ltd'''購買並運營在法律上還存在實際上已經停業的[[Handley_Page_Transport|Handley Page Transport]] Ltd。最明顯的繼承性設計就是[[Handley_Page_Herald|Herald]]客機，從reading設計室設計出來的產品型號的前綴都是HPR(來自於 "Handley Page (Reading) Limited" )。

===結束===
不像別的大型英國飛機製造商，罕德列·貝吉反抗政府將公司合併到更大的公司裡的壓力。到了1960年代晚期，整個英國航空工業由僅有的兩家企業控制：[[Hawker_Siddeley|Hawker Siddeley]] 和[[British_Aircraft_Corporation|British Aircraft Corporation]]。

由於無法競爭到政府訂單或生產大型商用飛機，罕德利·貝吉生產出了它最後著名的罕德利·貝吉設計——[[Handley_Page_Jetstream|Jetstream]]，這是一個小型的使用[[涡轮螺桨发动机|涡轮螺桨发动机]]的通勤班机，它有著[[座舱增壓|增壓座艙]]（cabin pressurization），可載運12-18人，它設計的首要目標是為[[美國|美國]]的[[支線客機|支線客機]]（feederliner）市場。

Jetstream出現的太晚了，因此沒能拯救罕德利·貝吉公司，公司於1970年3月進入了[[自動清盤|自動清盤]]流程，並於創建61年後停止營業。但Jetstream飛機做為一款成功的產品繼續生存，設計被位於[[Prestwick|Prestwick]]的[[Scottish_Aviation|Scottish Aviation]]購買並生產。後來Scottish Aviation於1977年合併到[[British_Aerospace|British Aerospace]]。

Radlett機場（[[Radlett_Aerodrome|Radlett Aerodrome]]）做為罕德列·貝吉民航的草地機場於1929年起用。為了生產[[Handley_Page_Halifax|Halifax]]轟炸機，跑道於1939年進行了延伸，大部分的塔台、[[停機庫|停機庫]]（hangar）和跑道在1970年公司停業後被破壞。現在M25高速公路從場地的南邊通過。[[拉法基|拉法基集團]]（Lafarge）目前擁有剩餘的部分。

==產品==
===型號===
罕德列·貝吉最初使用遞進的單一字母做為機型的型號（如：''R'', ''S'', ''T''等）並結合一個數字，它可能有也可能沒有確定的意義（如O/100表示該型的翼展為100英尺）。

1924年罕德列·貝吉改用字母''HP''加上一個數字來表示型號。如O/400變成了HP.16，[[Handley_Page_Type_W|W.8]]變成了HP.18。

當它接收了Miles Aircraft公司的資產後，Reading設計辦公室開始使用''HPR.''（表示Handley Page Reading）加上一個數字表示型號，它表示原始設計是由Reading的設計室進行的。（如HPR.1 Marathon）。
