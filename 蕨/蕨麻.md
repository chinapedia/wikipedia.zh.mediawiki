'''蕨麻'''（[[学名|学名]]：{{lang|la|''Potentilla anserina''}}）是[[蔷薇科|蔷薇科]][[委陵菜属|委陵菜属]]的植物。

== 形态 ==
蕨麻是多年生草本。肉质[[根|根]]肥厚，呈纺锤状；葡萄[[茎|茎]]细长，节上生根；基生叶为[[羽状复叶|羽状复叶]]，小叶3-12对，椭圆形，背部密布白色棉毛；夏季开花，花单生较大，鲜黄色。

== 分布 ==
蕨麻分布在[[大洋洲新西兰|大洋洲新西兰]]、[[南美智利|南美智利]]、[[塔斯马尼亚岛|塔斯马尼亚岛]]、[[欧亚美三洲北半球温带|欧亚美三洲北半球温带]]以及[[中国大陆|中国大陆]]的[[云南|云南]]、[[甘肃|甘肃]]、[[西藏|西藏]]、[[辽宁|辽宁]]、[[四川|四川]]、[[山西|山西]]、[[吉林|吉林]]、[[青海|青海]]、[[黑龙江|黑龙江]]、[[宁夏|宁夏]]、[[新疆|新疆]]、[[内蒙古|内蒙古]]、[[陕西|陕西]]、[[河北|河北]]等地，生长于海拔500米至4,100米的地区，目前尚未由人工引种栽培。

== 用途 ==
蕨麻全株含[[鞣质|鞣质]]，可提取栲胶，茎叶可制取黄色染料，根富含[[淀粉|淀粉]]，可煮食或酿酒。

== 别名 ==
人参果、延寿草、蕨麻委陵菜（秦岭植物志）、莲花菜、鹅绒委陵菜（中国高等植物鉴）
