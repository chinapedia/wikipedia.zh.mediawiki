'''紅軍勝利紀念碑'''位於[[四川省|四川省]][[江油市|江油市]]纪念碑大街与解放街交汇处<ref name="a2" />。原为“'''中国革命军第十九军一路司令[[董宋珩|董珩]]德政碑'''”，[[坐北朝南|坐北朝南]]<ref name="a1">{{cite web |url = http://www.my.gov.cn/MYGOV/150607825712709632/20110517/566747.html |title=红军在江油（故事篇）（四）|accessdate=2017-04-13 |author= |date=2011-05-17 |website = 绵阳市人民政府网站|language = zh-hans }}</ref>。

1935年4月17日，[[中国工农红军第四方面军|中国工农红军第四方面军]]攻占江油县中坝镇。第四方面军总指挥部决定将德政碑改为红军胜利纪念碑，总政治部副主任[[傅鍾|傅鍾]]负责改建工作。正面碑文：“'''百战百胜的工农红四方面军光荣胜利纪念碑'''”。背面碑文：“为争取独立自由与领土完整的苏维埃[[新中国|新中国]]而战”。西侧碑文：“[[土地改革运动|彻底没收地主阶级土地平分给贫苦农民]]，[[扩红|坚决做好扩大红军的工作]]”。东面碑文：“铲除封建势力，消灭卖国贼蒋介石，坚决[[赤化|赤化]]全四川”。数月后，[[国民政府|国民政府]]收复中坝，铲去碑文，重新刻字。1949年后，中共政权对其进行修复，南北两侧文字保存较好，东西两侧照原样恢复<ref name="a1"/>。1976年12月，江油县革命委员会公布为江油县文物保护单位<ref name="a2">{{cite web |url = http://www.myrb.net/html/2016/news/4/45206_2.html |title=央视纪录片《永远的长征》重温"红军在江油" |accessdate=2017-04-13 |author= |date=2016-04-18 |website = 中国绵阳新闻网 |language=zh-hans }}</ref>。1980年7月7日列為四川省第一批重新確定公布的省級文物保護單位，文物遺址年代判定為1935年<ref>『關於重新公布全省文物保護單位名單的通知』，四川省人民政府，一九八〇年七月七日，川府發（1980）154號文件</ref>。
