'''紅尾鯰'''（[[学名|学名]]：''{{lang|la|Phractocephalus hemioliopterus}}''）又名'''紅尾鴨嘴鯰魚'''、'''紅尾護頭鱨'''、'''狗崽鯨'''、'''狗仔鯨'''、'''紅尾貓'''、'''亚马逊河鲶鱼'''、'''枕頭鯰'''，是[[輻鰭魚綱|輻鰭魚綱]][[鯰形目|鯰形目]][[油鯰科|油鯰科]]的一[[種|種]]，為常見的觀賞魚種。為'''護頭鱨屬'''（''Phractocephalus''）下[[單種屬|唯一的物種]]。

== 分布 ==

本魚原產[[奧里諾科河|奧里諾科河]]以及[[亞馬遜河|亞馬遜河]]流域所經過的[[巴西|巴西]]、[[圭亞那|圭亞那]]、[[委內瑞拉|委內瑞拉]]、[[祕魯|祕魯]]等[[國家|國家]]，之後被引進[[亞洲|亞洲]]的一些國家以及[[佛羅里達州|佛羅里達州]]，但卻未在佛羅里達州成為普遍魚種。

== 特徵 ==

本魚有十分寬廣的[[口部|口部]]以及[[頭部|頭部]]，[[背鰭|背鰭]]上總共有7個鰭刺，口部上並有三對觸須，而上顎骨則不延伸至背鰭。紅尾護頭鱨主要體色為深灰色，並在上面會有許多微小不明顯的深黑色的斑點，體中間有一條明顯的白色側線，延伸到雪白的[[腹部|腹部]]，其紅色的[[尾鰭|尾部]]是紅尾護頭鱨最明顯的特徵。體型十分巨大，一般在[[水族館|水族館]]的紅尾護頭鱨就可達到90[[公分|公分]]，在野外，甚至可見到長達120公分左右的大型個體。而目前體型最大的紅尾護頭鱨可長至1.34[[公尺|公尺]]，重達44.2[[公斤|公斤]]。

== 生態 ==

本魚棲息於[[酸碱度|酸碱度]]5.5 - 6.8dH、水溫[[攝氏|攝氏]]20至26度的[[熱帶|熱帶]][[河川|河川]]以及[[池塘|池塘]]等水域，並喜好於水域底部游動。屬於[[雜食性|雜食性]]動物，幾乎無所不吃，從[[水果|果子]]、[[甲殼綱|甲殼綱]][[動物|動物]]到[[淡水魚|淡水魚]]等等，都被列入紅尾護頭鱨的[[菜單|菜單]]之內。

== 价值 ==

=== 经济价值 ===

可食用，也是重要的游釣魚。為高經濟價值得觀賞魚，觀賞用的幼年紅尾護頭鱨可用[[金魚|金魚]]餵食，但最好的食物是活[[蚯蚓|蚯蚓]]、[[冷凍|冷凍]]蚯蚓等食物，而[[成年|成年]]與[[青少年|青少年]]時期的紅尾護頭鱨，則喜歡[[小龍蝦|小龍蝦]]及[[螃蟹|螃蟹]]之類大型甲殼綱動物。飼養紅尾護頭鱨的[[水族箱|水族箱]]裝備及環境，攸關到紅尾護頭鱨生長狀況，在安裝設備時，必須注意照明設備不可以安裝過多，因為紅尾護頭鱨不喜歡太多[[光|光]]線；水族箱基部的選擇與需不需要加蓋蓋子皆是比較不用注重的；選擇洞穴形狀時，建議選擇分支狀的[[洞穴|洞穴]]；而外部過濾器則可以裝置多一些。
