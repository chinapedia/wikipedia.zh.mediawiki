{{Japanese|japanese=赤い蝋燭と人魚|kana=あかいろうそくとにんぎょ|romaji=Akairōsoku to Ningyo}}
《'''紅蠟燭與人魚'''》（[[日語|日語]]：{{lang|ja|赤い蝋燭と人魚}}）是1921年（[[大正|大正]]10年）時被[[小川未明|小川未明]]發表的創作[[童話|童話]]。把人們潛藏的[[利己主義|利己主義]]與非人類抱持著怨念為主題的作品。

== 概述 ==
認為人類是善良而懷抱幻想的[[人魚|人魚]]的母親，把孩子託付給老夫婦後，在事後卻遭到背叛的人魚姑娘的故事。大正10年1月在[[東京朝日新聞|東京朝日新聞]]上連載，是未明的成名作。

這個作品，被認為是構思於[[新潟縣|新潟縣]][[上越市|上越市]][[大潟區|大潟區]]的[[雁子濱|雁子濱]]（がんこはま）的人魚傳說。雁子濱當時所創業的，蠟燭店現在還存在著，一般被認為其原形是是這裡。

== 故事概要 ==
很久很久以前，有個懷了身孕的人魚棲息在某個北方黑海。人魚看了看附近的環境後，思考過為了不希望讓自己的孩子在海裡寂寞的生活。而有著像人類一般善良的心的人魚，聽說附近城鎮的生活是非常快樂的，於是牠就前往城鎮附近海邊的[[神社|神社]]，並決定在那裡把孩子生了下來。

第二天早晨，人魚在神社捨棄的孩子，被一對開蠟燭店的老夫婦撿到。而這一對老夫婦，老爺爺通常在家中負責做蠟燭，而老奶奶則負責在店前賣蠟燭，他們把那個孩子給細心的養育著，並在其後成為一個很美麗的女孩。長大後的人魚姑娘，為了答謝老夫婦的養育之恩，就在白蠟燭上用紅色顏料畫上圖案，而那蠟燭立刻便成為很高的評價，並讓這間蠟燭店變的生意興榮。如果在神社點了這蠟燭發出來的燈火而出海捕魚的話，就算是遇到[[暴風雨|暴風雨]]也還是能平安地返回，因此這事讓這評價越來越廣為流傳了。

某一天聽到這評價的一個[[流動小販|流動小販]]（[[賣香燭的|賣香燭的]]）來到蠟燭店看上了那個人魚，希望老夫婦把那個女孩賣給他。一開始這個家的老夫婦不肯賣這女孩，不過街頭藝人對他們說「很久以前人魚就被視為不吉利的東西。」這話後，並且願意與老夫婦相當的金錢，而見錢眼開的老夫婦就答應了。女孩眼看自己將要被關到鐵籠裡，就懇求老夫婦不要讓她離開，可是被利益熏心蒙蔽雙眼的老夫婦並沒有聽見她的請求。因此女孩把剩下的蠟燭給染上[[真紅|真紅]]色之後，馬上就被帶走了。

在賣了人魚那個夜晚，有一個令人毛骨悚然的女子突然來到老夫婦的店裡，買下那些真紅的蠟燭。不久後海上突然颳起狂風，讓很多船隻翻覆，而載著關著女孩的籠子的船也一起沉沒了。在此後不論是好天氣，只要在神社點上紅蠟燭的燈火的話，也會有很強烈的暴風雨來，必定會讓人在海上遇難死亡。事後老夫婦認為這是神給予的懲罰，於是就把蠟燭店給收掉了。

由於蠟燭店被收掉了，詛咒也不被平息，山上的神社被人們害怕和討厭而斷絕往來，在數年後繁榮的城鎮已經完全消失了。

== 雁濱的人魚傳說 ==
這個傳說，被認為是起源於袴形村還有[[住吉神社|住吉神社]]，聽說是某個[[水難事故|水難事故]]的原形。袴形的神社在一個有點高的山丘上的松林裡面，在[[佐渡島|佐渡島]]面向海的牌坊南方常常都會點著一排[[常夜燈|常夜燈]]，就算是壞天氣點燈也從來沒有斷過。貢獻這個點燈的光，是為了前來佐渡島一位不可思議的女子。雁子濱的一個年輕人和女子相戀而成為戀人，每晚都會溜出來幽會。不過這個年輕人有個[[未婚妻|未婚妻]]，然而事情被年輕人的母親發現後而責備了兩人的相戀，因此之後年輕人有一夜就停止了點燈。然後沒想到那個女子居然遭到遇難而被淹死，而這年輕人為了這個女子而向袴形的懸崖下跳水自盡。村裡的人很同情兩人的遭遇，就在長夜燈旁邊做了個[[比翼塚|比翼塚]]，並安置個小地藏神像。但不知不覺的人們不再點燈了，山丘上的住吉神社因此就此沉寂了，[[明治|明治]]41年發生了山崩而倒塌。<ref>大瀉工商會宣傳手冊</ref>

這個故事變成人魚傳說而流傳後世，成為未明童話故事的主線。現在神社的地基被平整後，只剩下石碑和一座[[燈籠|燈籠]]。
