'''紅頰獴'''（'''''Herpestes javanicus'''''）又叫'''紅臉獴'''、'''印度獴'''或'''爪哇獴'''，是[[灵猫科|灵猫科]][[獴属|獴属]]]的一個[[品種|品種]]，廣泛分佈[[亞洲|亞洲]]多個不同的地方，分為不同[[亞種|亞種]]。獴在世界很多地方均可找到，但紅頰獴的體型比[[非洲|非洲]]的亞種細型。

== 分佈及習性 ==
紅頰獴主要分佈於[[亞洲|亞洲]]南部，由[[伊朗|伊朗]]到[[中國|中國]][[華南|華南]]地區，再分佈至[[印尼|印尼]]。這個品種的適應能力很強，可以於不同的自然環境生存，包括[[濕地|濕地]]、[[草原|草原]]、[[森林|森林]]等，但較偏向空曠地方。

该物种的模式产地在爪哇。<ref name='id489'/>

== 獵食 ==
紅頰獴是[[機會主義|機會主義]]者，任何有能力捉到的小動物均成為牠們的食物，例如[[昆蟲|昆蟲]]、[[兩棲類|兩棲類]]、[[蝸牛|蝸牛]]、[[蜥蜴|蜥蜴]]、[[雀|雀]]、[[老鼠|老鼠]]等，甚至[[蛇|蛇]]虱可以捕食到。

== 習性 ==
紅頰獴一般為獨行性，但到了繁殖季節時均會以2至4隻群居。牠們為[[晝行性|晝行性]]動物，喜歡玩耍。

== 外來入侵種 ==
紅頰獴曾被認為是[[人類|人類]]刻意引入[[農田|農田]]以防治[[鼠|鼠]]害，但獴亦有捕捉[[雀|雀]]及其[[蛋|蛋]]，對部份地方的生態環境帶來嚴重威脅。[[國際自然保護聯盟|國際自然保護聯盟]][[物種存續委員會|物種存續委員會]]的入侵物種專家小組（ISSG）列為[[世界百大外來入侵種|世界百大外來入侵種]]。
