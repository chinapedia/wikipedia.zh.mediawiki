《'''紅男綠女'''》是由[[吉林電視台|吉林衛視]]都市頻道播出的[[東北話|東北方言]][[情景喜劇|情景喜劇]]。

該劇於2004年首播，每週一到週六18：00—18：30分播出，現已播出300餘集。该剧的导演兼编剧是[[许爱军|许爱军]]。

紅男綠女的拍攝地點在[[吉林省|吉林]][[長春|長春]]，[[取景|取景]]全部於[[辦公室|辦公室]]內進行，劇情主要圍繞辦公室青年男女以及老闆在廣告公司內外發生的軼事。由於演員使用東北話，使得劇情與表達更加風趣、幽默，具感染力，成為在[[吉林省|吉林省]]內，乃至[[東北地區|東北地區]]甚至更廣泛的北方地區觀眾喜聞樂見的都市情景戲劇。

== 演員介紹 ==

===主要演員===

* [[張楠_(中國演員)|張楠]]：美女主持，身材高挑，皮膚白皙。與張柏鑫談戀愛。
* [[袁媛|袁媛]]：不算漂亮，但很受看，聰明賢惠，知書達理，較有心計，愛吃零食，外冷內熱。與劇中後期偶爾出現的一位博士（小金子）處對象。
* [[張柏鑫|張柏鑫]]：身材高大，帥氣，人較為實在，頭腦簡單，愛吃。在追求張楠。
* [[周全|周全]]：個子不高，年齡最小，非常小氣，喜歡溜縫，善於拍馬，也經常受氣。最初暗戀袁媛，後與袁媛介紹的文文。
* 楊總（[[羅旭文|羅旭文]] 飾）：外號大羊（其妻稱），年輕時曾為痞子學生，愛打架。現在是一個與員工打成一片的好領導，但同時是一個“氣管炎”，愛打麻將。

===客串人物===

* 文文（[[張聞博|張聞博]] 飾）：周全劇中的對象
* 大玲子（[[竇玲|竇玲]] 飾）：是楊總的媳婦，原名叫竇玲，現改名叫竇海文。劇中用原名竇玲。
* 小红（[[李丁|李丁]] 饰）：从事过非法传销，后自己经营小饭店。一直对张柏鑫抱有好感。
* 小金子（[[金柏安|金柏安]] 飾）：是袁媛的劇中的對象，扮演一個博士角色，是吉視都市中《[[守望都市|守望都市]]》的主持人。
* [[劉洋|劉洋]]、[[苗強|苗強]]：是最新紅男綠女中咖啡廳的人，苗强为老板，刘洋为员工，且是杨总的亲戚，苗強暗恋劉洋。

== 劇集發展 ==

隨著此劇逐漸熱播，劇集的發展也分為幾個階段
# 最初是在居民樓裡拍攝，大概幾十集（每集大概10分鐘）
# 隨後是在一個較小的寫字樓裡，大概是近200集（每集大概15分鐘）
# 後來是在一個高層寫字樓拍的，大概近100集（每集大概20分鐘）
# 現在是在一個更高的寫字樓裡，多了一個咖啡廳，現在只在吉林都市頻道播出（每集大概30分鐘）

== 續集 ==
《新紅男綠女》於2017年3月18日在吉林衛視播出。
