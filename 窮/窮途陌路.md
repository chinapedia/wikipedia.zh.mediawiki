《'''窮途陌路'''》是一檔由中國大型女子偶像團體[[SNH48|SNH48]]為主要參演成員的綜藝節目，於[[騰訊|騰訊]]網站播放。

== 簡介 ==
本節目是一檔[[SNH48|SNH48]]的旅遊探險類真人秀節目，由上海星四芭、上海久尚及絲芭影視共同製作<ref name="yxw">{{Cite web |url = http://www.yx-w.com/yingshixingwen/21346.html |title = SNH48综艺节目《穷途陌路》 9月24日剧场首曝 |date = 2015-09-22 |accessdate = 2015-10-20 |publisher = 影俠網 }}</ref>。參與節目的三名成員前不同國家地區，於限制花費的條件下，開發最省錢的方式遊玩當地景點。

第一季12集，外加2集先導片，總共14集<ref>{{Cite web |url = http://www.snh48.com/html/allnews/zixun/2015/1012/1436.html |title = 国庆假期不尽兴 SNH48《穷途陌路》带您续航 |date = 2015-10-12 |accessdate = 2015-10-20 |publisher = SNH48官方網站 }}</ref>。首集先導片於2015年9月24日在[[SNH48星夢劇院|SNH48星夢劇院]]公演前先行播放<ref name="yxw" />。
