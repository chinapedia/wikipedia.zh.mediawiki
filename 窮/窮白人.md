《'''窮白人'''》（{{lang-en|'''''Poor White'''''}}）是[[舍伍德·安德森|舍伍德·安德森]]所寫的美國小說，於1920年出版。<ref>{{Cite web |url = http://www.publishersweekly.com/978-0-8112-1242-7 |title = Poor White |publisher = Publishers Weekly |language = en}}</ref>

== 情節 ==
內容講述一名發明家休·麥克維（Hugh McVey），在[[密西西比河|密西西比河]]岸的貧窮生活中成功的故事。小說展示出[[工業革命|工業革命]]在美國農村中心區域的影響力。
