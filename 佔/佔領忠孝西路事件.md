'''佔領忠孝西路事件'''是指2014年4月27日至4月28日，由中華民國[[全國廢核行動平台|全國廢核行動平台]]主導的佔領[[臺北市|臺北市]][[忠孝西路_(臺北市)|忠孝西路]]。活動最早是為了呼應前[[民主進步黨|民主進步黨]]主席[[林義雄|林義雄]]主張反對興建[[龍門核能發電廠|龍門核能發電廠]]、重新修正公民投票而發起的[[2014年林義雄反核四禁食行動|禁食行動]]，進而召集聲援反核運動的民眾前往[[凱達格蘭大道|凱達格蘭大道]]與周遭道路參與[[426反核大遊行|426反核大遊行]]以表達訴求。而在4月26日凱達格蘭大道前的晚會上，主辦單位首次宣布將在遊行行經[[台北車站|台北車站]]前的忠孝西路時，發動佔領行動以癱瘓車站前的交通。

下午3時佔領忠孝西路行動正式展開，然而遭到警方事先以路障方式攔阻示威群眾，而只開放2個車道讓民眾進入。但在下午4時指揮臺下令群眾跨過路障，忠孝西路10個車道隨即被數萬名群眾佔領，警方在舉牌3次要求解散不成之後決定暫時收隊。另一方面，[[公投護台灣聯盟|公投護台灣聯盟]]也從立法院方向進入佔領忠孝西路和中山南路口，佔領兩方在警方撤守後重新集結。下午5時，全國廢核行動平台依照原訂計畫在忠孝西路進行核災模擬，稍後帳篷、睡袋等物資陸續進駐準備進行長期抗戰。

不過到了晚上10時30分[[臺北市市長|臺北市市長]][[郝龍斌|郝龍斌]]召開記者會，強調在4月28日清晨前會動用一切方式使生活與交通恢復常態。到了4月28日凌晨3時，警方出動鎮暴警察和2臺鎮暴水車自館前路一端開始強制驅離，並且與抗議民眾多次發生推擠。大約在早上7時左右，忠孝西路和周遭路段則完全清空。

== 經過 ==
在4月26日凱道前的晚會上，主辦單位首次宣布將會在遊行行經台北火車站前的忠孝西路時，發動佔領行動、癱瘓火車站前的交通。4月27日下午3點遊行正式展開，而警方也事先以路障方式，只開放2個車道讓民眾進入，但在下午4點，指揮台下令群眾跨過路障，忠孝西路10個車道隨即被數萬名群眾佔領，警方在舉牌三次要求解散不成之後，決定暫時收隊。另一邊公投盟也從立院方向進入佔領忠孝、中山南路口，兩邊在警方撤守後連成一線。下午5點，廢核平台依原訂計畫，在忠孝西路進行核災模擬。而後，帳篷、睡袋等物資陸續進駐準備進行長期抗戰<ref>[http://www.coolloud.org.tw/node/78415 428廢核佔領忠孝西路即時報導]</ref>。晚間10點半，台北市長郝龍斌召開記者會，強調在28日清晨前，「會用一切方式，讓台北市民的生活及交通恢復常態，任何可能的事情都會做」<ref>[http://udn.com/NEWS/BREAKINGNEWS/BREAKINGNEWS1/8639841.shtml 驅離忠孝西路民眾？ 郝龍斌：不排除]</ref>。
<br>約在凌晨3點，警方出動鎮暴警察及兩台鎮暴水車，自館前路一端開始強制驅離，並與抗議民眾多次發生推擠，約在早上7點完全清空，期間鎮暴水車至少噴水50次。

== 爭議 ==

*4月28日，由於強制驅離時，有民眾帶兒童上街抗爭<ref>[http://www.ettoday.net/news/20140428/351063.htm 快訊／忠孝西反核現場有小孩　群眾不滿批警沒人性]</ref>，國民黨立委[[王育敏|王育敏]]因此提案修改《兒童及少年福利與權益保障法》，禁止任何人以迫使或誘使的方式，使兒少處於易發生立即危險或傷害之環境<ref>{{Cite web |url=http://www.stormmediagroup.com/opencms/news/detail/65cc69c7-cebe-11e3-9580-ef2804cba5a1/?uuid=65cc69c7-cebe-11e3-9580-ef2804cba5a1 |title=藍委王育敏修法 禁止午夜帶兒童上街頭抗爭 |access-date=2014-05-12 |archive-url=https://web.archive.org/web/20140430125944/http://www.stormmediagroup.com/opencms/news/detail/65cc69c7-cebe-11e3-9580-ef2804cba5a1/?uuid=65cc69c7-cebe-11e3-9580-ef2804cba5a1 |archive-date=2014-04-30 |dead-url=yes }}</ref>。

*4月30日，消防員工作權益促進會認為在驅離過程中，鎮暴水車要求消防水箱車協助加水，有違反《消防法》的嫌疑，但台北市消防局隨即回應，認為當天支援不影響勤務執行，而無違法問題。<ref>[http://www.appledaily.com.tw/realtimenews/article/new/20140430/389329/ 水箱車補鎮暴車水源　警消轟「這不是噴人的」]</ref>
*5月12日，民進黨立委[[尤美女|尤美女]]與婦女團體召開記者會，批評警方在驅離時，以強力水柱對女學生的腰部及背部猛沖，導致女學生背部受傷、腎臟受損及血尿情形，已違反比例原則，並構成傷害罪、重傷害未遂，將提出刑事告訴與國家賠償訴訟，而警方內部流傳女學生[[走光_(行為)|走光]]影片，已也經構成性騷擾行為，要求警政署應依《性騷擾防治法》提出糾正，並將提出民事賠償訴訟<ref>[http://iservice.ltn.com.tw/2014/specials/nonukes/news.php?rno=1&type=l&no=1005600 驅離走光影片外流 女學生：警察不是人民保姆嗎？]</ref>。
