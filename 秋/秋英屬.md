'''秋英屬'''（[[学名|学名]]：''{{lang|la|Cosmos}}''），又名'''大波斯菊屬'''、'''波斯菊屬'''、'''秋樱屬'''，屬於[[菊目|菊目]][[菊科|菊科]]，約有20至26種，除此之外还有众多的[[变种|变种]]和[[栽培种|栽培种]]。主要產地在[[墨西哥|墨西哥]]、[[美國|美國]]南部、[[中美洲|中美洲]]及[[南美洲|南美洲]]等地，品種大多為[[多年生|多年生]][[草本|草本]]植物，[[花期|花期]]大都在8－10月，主要作為觀賞用途。

“秋英”一名通常指[[大波斯菊|大波斯菊]]（''Cosmos bipinnatus''）。

== 特征 ==
秋英屬植物花色绚丽，有白、红、黄、粉、紫、橙等多种颜色，亦有复色品种，植株株高0.3－2米，叶为一回或二回[[羽状复叶|羽状复叶]]，对生。[[花序|花序]]外环为舌状花宽环，中心为盘状花。秋英屬植物大多是[[短日照植物|短日照植物]]，但也有6月开花的早生品种。

大部分秋英屬植物的花朵对[[鸟类|鸟类]]和[[蝴蝶|蝴蝶]]都很有吸引力。

== 命名 ==
学名''{{lang|la|Cosmos}}''是由西班牙[[牧师|牧师]]和[[植物学|植物学]]家[[安東尼奧·何塞·卡瓦尼列|安東尼奧·何塞·卡瓦尼列]]命名的，源於[[希腊语|希腊语]]“{{lang|el|κόσμος}}”一词，意为秩序、和谐、世界、宇宙、饰物、勋章<ref>{{zh}}{{cite book|editor=罗念生、水建馥 |title=《古希腊语汉语词典》 |date=2004年6月1日 |publisher=商务印书馆 |location=北京 |isbn=7100028361 |pages=第476页}}</ref><ref>{{en}}{{cite web|url=http://www.greekbible.com/l.php?ko/smos_n-----asm-_s | title=Kosmos |publisher=希腊语圣经 |accessdate=2009-01-21 }}</ref><ref>{{zh}}{{cite web |url=http://dome.ycool.com/post.1168599.html |title=《“世界”的圣经希腊语源考证》 |date=2006-03-29 |publisher=荆棘百合（歪酷博客） |accessdate=2009-01-21 |deadurl=yes |archiveurl=https://web.archive.org/web/20081026091419/http://dome.ycool.com/post.1168599.html |archivedate=2008-10-26 }}</ref>，而英语中表示“[[宇宙|宇宙]]”的词之一也是源於这一希腊语单词，拼写与秋英屬的学名''{{lang|la|Cosmos}}''完全相同，易引起混淆。

== 主要种 ==
* [[巧克力秋英|巧克力秋英]] ''Cosmos atrosanguineus''
** 黒紫色花朵，有[[巧克力|巧克力]]香气，[[耐寒性|耐寒性]]一般
* [[大波斯菊|大波斯菊]]（秋英） ''Cosmos bipinnatus''
** 株高0.3－2米，细茎，叶狭长
* [[帝皇乌蓝|帝皇乌蓝]] ''Cosmos caudatus'' 
* [[异叶秋英|异叶秋英]] ''Cosmos diversifolius'' 
* ''Cosmos herzogii'' 
* [[小花秋英|小花秋英]] ''Cosmos parviflorus'' 
* [[前胡叶秋英|前胡叶秋英]] ''Cosmos peucedanifolius'' 
* [[蓝盆花秋英|蓝盆花秋英]] ''Cosmos scabiosoides'' 
* [[硫華菊|硫華菊]] ''Cosmos sulphureus''
** 主要在夏季开花，花色为黄色和橙色
