'''秋山真之'''（1868年4月12日 - 1918年2月4日）[[大日本帝国海军|大日本帝国海军]][[將領|將領]]。最高军衔海军[[中将|中将]]。幼名淳五郎（じゅんごろう）。母亲是贞。哥哥[[秋山好古|秋山好古]]是陆军[[大将|大将]]。妻子[[秋山季子|秋山季子]]（[[稲生すゑと|稲生すゑと]]）。育有4男2女。参议院议员[[大石尚子|大石尚子]]是真之的外孙女（次女宜子的长女）。

== 经历 ==
[[File:麹町区土手三番町.jpg|250px]][[佐久间正节|佐久间正节]]的宅第所在。麹町区土手三番町(现在[[千代田区|千代田区]]五番町)]]
[[File:市ケ谷橋.jpg|250px]]宅第眺望[[秋山好古|秋山好古]]的母校[[陆军士官学校_(日本)|陆军士官学校]]]]

秋山真之出生于伊予国[[松山城|松山城]]中徒町（现在[[爱媛县|爱媛县]][[松山市|松山市]]）的下级[[武士|武士]]家庭。父亲是秋山久敬。母亲贞是松山藩士山口家的女儿。在乡里的[[汉学|汉学]]塾上学。受到好友[[正冈子规|正冈子规]]上京的刺激，5年级时从爱媛县第一中学（现在[[爱媛县立松山东高等学校|爱媛县立松山东高等学校]]）退学。[[明治|明治]]16年（1883年）前往东京，先在共立学校（现在[[开成中学校|开成中学校]]）学习英语，接着进东京大学预备门（后来的[[第一高等学校|第一高等学校]]），最后在[[东京大学|东京大学]]学习教养课程。

大学预备门的时候一直接受哥哥好古的接济，毕业后和以立志于文学的正冈子规分道扬镳。明治19年（1886年）进[[海军兵学校|海军兵学校]]17期班, 成為[[廣瀨武夫|廣瀨武夫]]的學弟。明治23年（1890年）以第一名的优异成绩毕业后授予[[少尉|少尉]]候补军衔在海防舰[[比叡|比叡]]上服役，其间救助了触礁搁浅的[[土耳其|土耳其]]军舰。日清[[甲午战争|甲午战争]]中在[[筑紫|筑紫]]舰上服役担任侦查和救助任务。战后转[[和泉|和泉]]舰服役。明治29年（1896年）年1月划归横须贺，并且在海军水雷术练习所（[[海军水雷学校|海军水雷学校]]）学习布雷技术，毕业后担任横须贺水雷团第2水雷队副队长。后来在通讯舰[[八重山|八重山]]服役升任[[大尉|大尉]]。同年11月被[[军令部|军令部]]作为谍报员派往[[中国|中国]]东北活动。

明治31年（1898年）日本海军选拔公费留学生，秋山真之落选，即以私费身份留学[[美国|美国]]。在美期间师从海军大学校校长军事思想家[[马汉|马汉]]。在华盛顿海军大学秋山利用大学图书馆和海军文库钻研兵术理论的研究。这期间正好爆发[[美西战争|美西战争]]，秋山作为观察员提交了报告。秋山参观了美国海军封锁[[古巴|古巴]]圣地亚哥港的作战并提交报告，<REF>在米国海軍大尉秋山真之「サンチャゴ・ヂュ・クバ之役（極秘諜報第百十八号）」</REF>以此为参考在日俄战争中提出了封锁[[旅顺|旅顺]]港的作战方案。明治32年（1899年）1月前往英国视察8月归国。明治33年（1900年）入[[海军省|海军省]]军务局第1课，成为常备舰队参谋。

[[File:MIKASAPAINTING.jpg|right]]舰桥<br />中央三人从左至右是[[加藤友三郎|加藤友三郎]]，[[东乡平八郎|东乡平八郎]]，秋山真之]]
明治35年（1902年）成为[[海军大学校|海军大学校]]教官。明治36年（1903年）8月结婚。明治37年（1904年）升任海军[[第一舰队|第一舰队]][[少佐|少佐]][[参谋|参谋]]。因[[朝鲜半岛|朝鲜半岛]]问题日本和沙皇[[俄国|俄国]]关系恶化，同年作为联合舰队司令长官[[东乡平八郎|东乡平八郎]]的[[作战参谋|作战参谋]]登上第1舰队[[旗舰|旗舰]][[三笠|三笠]]参加[[日俄战争|日俄战争]]。为了歼灭沙俄太平洋舰队用水雷封锁旅顺港。并且制定了阻击沙俄前来救援的[[波罗的海舰队|波罗的海舰队]]的作战方案，取得[[日本海海战|日本海海战]]的胜利，组成日俄战争战略胜利的重要一环。

明治38年（1905年）12月联合舰队解散后担任巡洋舰舰长，第1舰队参谋长。大正元年（1912年）12月1日担任[[军令部|军令部]]第1班长，后任军令部第1部长。

大正3年（1914年），在军舰建造问题上发生[[西门子事件|西门子事件]]。随后发展成对政府的批判，引起对军界在事件中暗箱处理不满，众议院议员[[花井卓藏|花井卓藏]]召集响应者要求公开军法会议。同年1月设置调查委员会秋山被指名参加。3月[[山本权兵卫|山本权兵卫]]内阁总辞职，[[大隈重信|大隈重信]]当选新总理，任命[[八代六郎|八代六郎]]为新的[[海军大臣|海军大臣]]，秋山任军务局长辅佐八代六郎。11月成立治罪法改正委员会，同花井卓藏等人辩论。大正5年（1916年）2月调出军令部，委员资格由[[铃木贯太郎|铃木贯太郎]]继任。

担任军务局局长期间，秋山真之乘坐巡防舰[[音羽|音羽]]访问[[上海|上海]]对中国进行实地考察，向中国提出招收留日学生的建议。和[[孙文|孙文]]也有接触并在背后支持中国民主革命。和[[小池张造|小池张造]]结成“小池部屋”。通过实业家[[久原房之助|久原房之助]]等人开展革命活动。明治44年（1911年）中国发生[[辛亥革命|辛亥革命]]推翻清朝成立[[中华民国|中华民国]]。大正4年（1915年）[[袁世凱|袁世凱]]复辟，中国各地倒袁之声四起，日本政府和其他西方国家也表示抗议。[[川岛浪速|川岛浪速]]等浪人和参谋本部[[田中义一|田中义一]]副参谋长等人主导了第二次[[满蒙独立运动|满蒙独立运动]]，秋山作为外务省政务局长也有参与。直至从军令部转出才离开对中政策的智囊团。

大正5年（1916年）3月，前往视察[[第一次世界大战|第一次世界大战]]。从朝鮮半岛经西伯利亚铁道到到了俄国，芬兰等东欧国家。5月到达英国。随后又前往法国、意大利，大正6年（1917年）9月前往美国，10月回国。回国后马上被任命为[[第二舰队|第二舰队]]水雷司令官，因为身体原因不久辞职。同年7月选为[[海军将官会议|海军将官会议]]议员荣誉职位。

大学校教官时代，被邀请参加[[佐藤铁太郎|佐藤铁太郎]]的「天晴会」研究经典，晩年醉心于灵学和宗教研究。皈依被多数军人接受的[[日莲宗|日莲宗]]。还跟从神道家[[川面凡儿|川面凡儿]]学习神道，两人设立了皇典研究会。在好友[[海军机关学校|海军机关学校]]教官[[浅野和三郎|浅野和三郎]]的介绍下参加了新兴宗教[[大本教|大本教]]。大正7年（1918年）去世前钻研[[般若心经|般若心经]]。

晩年因为[[腹膜炎|腹膜炎]]在[[箱根|箱根]]疗养，最后病情恶化在[[小田原|小田原]][[山下龟三郎|山下龟三郎]]的别墅去世。享年49岁。

墓所在[[东京|东京]][[港区|港区]]的[[青山墓地|青山墓地]]。

== 人物 ==
日俄战争中[[日本海海战|日本海海战]]是其人生的最高点，此后除了准确预测了[[第一次世界大战|第一次世界大战]]的结果并没有什么特别表现。但是海军因为他在日俄战争中的功绩加以神话化。

东乡平八郎评价秋山智如潮涌。在日俄战争后变得十分迷信，[[戴季陶|戴季陶]]在其著作《日本论》中将秋山描述为神机妙算的军师，还提到去见他时经算卦认为应往高处去住，后来戴原先的住处被台风水淹的事。

诗人[[正冈子规|正冈子规]]是秋山少年时代的好友，在东京的共立学校也是同学经常一起游玩。电视剧《[[坂上之云|坂上之云]]》中有秋山考试时给子规递条子的表现，实际上递纸条的另有别人。后来成为[[大藏省|大藏省]]官员的[[胜田主计|胜田主计]]也是秋山和子规在松山时的好友。

[[日本海海战|日本海海战]]时秋山的电报『本日天気晴朗ナレドモ浪高シ』明文是"今日天气晴朗但是浪高"，天氣晴朗代表可清楚目視俄國艦隊，浪高代表長期在太平洋的日本艦隊佔有著主場優勢。短短13个字就表达了很完整的意思。得到高度评价。此外旗语「皇国ノ興廃此ノ一戦ニ在リ、各員一層奮励努力セヨ」（皇国兴废在此一战，诸君当愈益奋勵努力）也是出自他的手笔。

[[日本海海战|日本海海战]]胜利后联合舰队解散时东乡平八郎的训示也是秋山起草。因为受到文章感动美国总统[[罗斯福|罗斯福]]派人将其全文翻译后分发到美国海军各单位。「秋山文学」也由此名噪天下。

秋山长期身為參謀，一直在东乡平八郎身后不为外人所知，直到战后才透过[[岛田谨二|岛田谨二]]的介绍為人所知。[[昭和|昭和]]47年（1972年）[[司马辽太郎|司马辽太郎]]的小说《[[坂上之云|坂上之云]]》使秋山家喻户晓。

== 轶事 ==
* 秋山小时候很调皮，领着邻居小孩玩打仗游戏。学着书里自己制作焰火。由于太顽皮母亲贞拔出刀说「お前も殺して私も死ぬ」（杀了你我也自杀算了），秋山才有所收敛。喜欢画图，游泳和跑步。
* 少尉候补的时候，喜欢吃饭时候用面包屑做成[[俾斯麦|俾斯麦]]、[[拿破仑|拿破仑]]和[[丰臣秀吉|丰臣秀吉]]的头像。
* 候补生时期，后辈问他为什么不见他用功成绩还是那么好。秋山回答参考过去考试出的题目，猜测教官的出题心理。
* 喜欢吃炒豆，口袋里一直放着。所以描写他喜欢放屁。
* 喜欢用军服的袖口擦鼻涕，想作战方案时可以几天不洗澡忘我投入。
* 从美国归国途中遭遇赌博骗子。输光的秋山用刀逼骗子把骗去的钱还了出来。
