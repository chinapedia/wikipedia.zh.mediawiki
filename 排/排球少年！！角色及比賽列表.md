:; 烏野「'''最強的誘餌'''」
: 擁有超群的反應、速度、彈跳力等能力，彌補了基礎技術和身高上的不足。目標是成為烏野高中的王牌。
: 很容易緊張，有點[[天然呆|天然呆]]而率直的性格。學習方面不是很好，在東京合宿時也因為英文科期末考不及格需要補考而比其他人晚到。
: 因為身高的原因，不時被其他人稱呼為「小不點（{{lang|ja|チビちゃん}}）」。
: 小時候看電視中播放的排球比賽裡面，全場個子都很高，唯獨有一名烏野高中的小個子身影大顯身手，就像「小巨人」一樣，當時翔陽對此留下深刻印象，努力想成為「小巨人」一樣的人，並從初中接觸排球。
: 以前是雪之丘中學三年級的排球社主將。初一時因為排球社人數不夠，日向總是一個人練習，初三時終於有三名初一的成員加入，日向並邀請好友組成臨時小隊，參加中學第一次也是最後一次正式比賽。在比賽中對上強豪── 北川第一中學 而落敗，並在那次比賽中認識影山飛雄。日向更想在之後的比賽一雪前恥，巧合地兩人都入讀了烏野高中，一開始兩人針鋒相對著，後來兩人慢慢建立互信關係，更成為最佳組合。
: 信任著影山的托球，日向能閉上眼揮下手也能扣殺成功的「怪人快攻」。
: 在比賽中不斷用快攻迅速得分，使對手把注意力集中放在日向身上，讓其他主攻手在對手缺乏防備時活躍起來，因而被稱之為「最強的誘餌」。
: 最可怕的事：有次緊張到發球失誤，不慎打中影山的後腦勺，而影山卻因發怒而失控，更利用這下子來提醒日向，說：「連打爆他的後腦勺這種可怕的事都做過了，就沒什麼可以緊張的理由了。」後來這一幕成了日向舒緩緊張的方法，因為今後不可能會比這個更可怕的事了。
: 到過白鳥澤學院高中參觀後，牛若(牛島若利)斷言及川徹是「卓越的果實」，而青葉城西高中是「貧瘠的土地」，令日向不甘，想以敗過給青葉城西高中的「混凝土」出身的身份在全國比賽打敗牛若。
: 因為想要變強，認為只靠閉上眼打出的殺球是不能進步的，想要強大得可以靠自己戰鬥，但影山卻不認同這種想法，兩人一度發生衝突。
: 在與影山發生衝突、第一次的東京合宿完結後受到烏養前教練的指導，接受了「與任何人配合都可以打出第一節奏」的扣球和球感等練習，希望「一個人也能在空中與攔網戰鬥」。在第二次東京合宿中和影山成功做到了托球能在撃球點附近「落下來」、能在空中有更多擊球點選擇的「新怪人快攻」。
: 在對上稻荷崎，最後與影山合力攔下宮氏兄弟的「雙胞胎快攻」，助烏野贏得比賽勝利。

:; 烏野「球場上的'''王者'''」
: 擁有超群的球感、控球能力、和敏銳的判決力。從小學二年級開始打排球。
: 雖然在球場上頭腦很好、很會思考對策，但在學習方面不是很好，在東京合宿時也因為期末考不及格需要補考而比其他人遲到。
: 以前是北川第一中學的排球隊舉球員，人稱'''「球場上的王者」'''，但本人卻十分討厭這名字。
: 精準的托球和球感，配合日向翔陽的高敏捷性和彈跳力，發揮出高水準的「怪人快攻」，並且只有這兩人的組合才能做到。
: 原本報了白鳥澤學院高中，可惜沒有考上。認為如果及川徹是縣內最強的二傳手，那麼超越他的人就是自己。
: 認為舉球員這個位置就像支配者，是隊伍中的指揮官，因而經常想以個人之力贏得比賽，只顧自己所追求的實力，就像暴君一樣的專橫、獨裁，使其在中學時被人稱之為「球場上的王者」，後來更因為過份自我中心，令隊友無法忍受，漸漸遠離他。高中在加入烏野排球社後，遇上日向和其他可靠部員，並在前輩的指導下，日漸了解到一個人的話是贏不了的，也學會如何配合別人及信賴別人，放下單獨取勝、自我中心的想法，逐漸蛻變成為真正的王者。
: 在與日向發生衝突、第一次的東京合宿完結後練習了「能停下來的托球」，並在第二次東京合宿中和日向成功做到了托球能在撃球點附近「落下來」、能在空中有更多擊球點選擇的「新怪人快攻」。
: 曾在與條善寺高中的比賽發生被打中臉引致流鼻血的突發情況，需要讓菅原接替其上場擔任二傳手。
: 於春高前被邀請參加全日本青年強化合宿。
: 在進入IH階段後，與伊達工業高中進行練習賽時，影山的球感伴隨隊員的能力變化逐漸提升，並希望覓到對球隊更易得分的可能性，於漫畫223話開始對隊員提出抱怨，例如是西谷接球後阻礙了後排進攻的助跑、月島能在攔網時進行2回跳躍等，令影山驚覺自己可能變回初中時的「王者」。教練指出：「讓攻手打出輕鬆的托球就是最棒的托球、之前的影山會主動配合攻手的壞習慣，現在則會讓攻手將擊球點拉得更高」，同時，隊員也認為影山的抱怨是合理可行的話也會嘗試接納，令不再是專橫、獨裁，而是為人設想的影山以「新·球場上的王者」再次誕生。
: 在對上稻荷崎，最後與翔陽合力攔下宮氏兄弟的「雙胞胎快攻」，助烏野贏得比賽勝利。

== 烏野高中 ==
位於[[宮城縣|宮城縣]]的縣立高中。曾經有著全國級的實力，但5年前開始衰退，外界因而稱其為「'''沒落的強豪——無法飛翔的烏鴉'''」。
