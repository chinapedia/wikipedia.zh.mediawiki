
'''濠頭鄭氏大宗祠'''，又稱為'''綸絅二祖祠'''，位於[[廣東省|廣東省]][[中山市|中山市]][[火炬開發區|火炬開發區]]濠頭一村（火炬開發區第三小學旁）。始建于[[明朝|明]]天启四年（1624），[[清朝|清]]乾隆四十年（1775）重修，2005年再修，保留了清代建筑风格。中山市不可移动文物。<ref>{{cite web|url=http://www.zsmuseum.cn/bwg/news_show.asp?id=577|title=中山市第三次全国文物普查 工作简报 第二十八期|deadurl=yes|archiveurl=https://web.archive.org/web/20150418081348/http://www.zsmuseum.cn/bwg/news_show.asp?id=577|archivedate=2015-04-18}}</ref> <ref>{{cite web|url=http://www.zs.gov.cn/main/zwgk/paperview/index.action?id=1560|title=关于公布中山市第三次全国文物普查 不可移动文物名录的通知 日期：2012年02月16日　　中山市人民政府政报 第203期　　文号：中府〔2012〕8号}}</ref>

== 建築 ==
建築為五間三進，門前有'''滎陽派遠'''，'''浙水支長'''對聯，彰顯先祖來歷，正堂為'''崇善堂'''，供有祖先牌位。

祠堂前原有兩對'''明代石獅'''，現存一對立於祠堂前，另外一對則立於濠頭村委大樓前。

== 浦江世澤坊 ==
祠堂前立有[[浦江世澤牌坊|浦江世澤牌坊]]始建於明崇禎九年，清雍正八年重修。正面刻有'''浦江世澤'''，背面刻有'''昭代褒崇'''。為中山市現存始建時間比較早的牌坊建築之一。<ref>{{cite web|url=http://www.yododo.com/area/photo/0135B98F28AF3C93FF80808135B5EEA9|title=实拍濠头浦江世泽牌坊}}</ref>

2010年5月10日公布为'''广东省文物保护单位'''。<ref>{{cite web|url=http://www.yododo.com/area/photo/0135B98F28AC3C92FF80808135B5EEA9|title=广东省文物保护单位石碑}}</ref>

== 姓氏源流 ==
據濠頭[[義門鄭氏家譜|義門鄭氏家譜]]記載，濠頭鄭氏又稱[[義門鄭氏|義門鄭氏]]起源于[[河南|河南]][[滎陽|滎陽]]，後遷至[[浙江|浙江]][[浦江|浦江]]，于南宋紹定年間由浙江遷入粵。
