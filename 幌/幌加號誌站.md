'''幌加號誌站'''（{{jpn|j=ホロカ信号場|hg=ホロカしんごうじょう|rm=Horoka shingōjō|f=y}}）位於[[日本|日本]][[北海道|北海道]][[勇拂郡|勇拂郡]][[占冠村|占冠村]]，是[[北海道旅客鐵道|北海道旅客鐵道]]（JR北海道）[[石勝線|石勝線]]上的[[號誌站|號誌站]]。

名稱來源於[[阿伊努語|阿伊努語]]的「horka-nay」，意思是倒流的河流<ref>{{Cite web|url=http://www.pref.hokkaido.lg.jp/ks/ass/grp/111120P.pdf|title=アイヌ語地名リスト|accessdate=2017-08-17|author=北海道庁|date=|publisher=北海道庁}}</ref>。

== 車站構造 ==
擁有2線的號誌站。號誌站兩端設有[[防雪廊|防雪廊]]（snow shelter）以避免[[轉轍器|轉轍器]]凍結。

== 車站周邊 ==
號誌站周圍是完全沒有人煙的廣大[[森林|森林]]地帶。
*鵡川

== 历史 ==
*1981年（[[昭和|昭和]]56年）10月1日 - 設立，原本稱為'''苫鵡號誌站'''（{{lang|ja|トマム信号場}}）。
*1987年（昭和62年）2月1日 - 由於原本的石勝高原站改名為苫鵡站，因此苫鵡號誌站改名為'''幌加'''。
*1987年（昭和62年）4月1日 - [[國鐵分割民營化|國鐵分割民營化]]後，由JR北海道接收管理。
