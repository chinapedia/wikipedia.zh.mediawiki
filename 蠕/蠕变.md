'''潛變'''（{{lang-en|'''Creep'''}}），也稱'''蠕變'''，是在[[應力|應力]]作用下固体材料缓慢且永久的變形。它的发生是低于材料[[屈服强度|屈服强度]]的應力长时间作用的结果。当材料长时间处于高温或者在[[熔点|熔点]]附近时，潛變会更加剧烈。潛變速率常常随着温度升高而加剧。

潛變速率与材料性质、加载时间、加载温度和加载结构應力有关。取决于加载應力和它的持续时间，这种變形可能變得很大，以至于一些部件可能会失效。例如，[[涡轮叶片|涡轮叶片]]的潛變将会使叶片与外壳相接触，导致叶片的失效。潛變常常是工程上和冶金上评价在高應力或高温下工作的部件所需要关注的。潛變可能是组成失效模型的變形机制，也可能不是。混凝土中适中的潛變有时是受欢迎的，因为它会减轻可能另外引发断裂的拉應力。

不像脆性断裂，潛變變形并不会随着應力作用而突然出现。相反，[[應變|應變]]会在长时间的應力作用下积累。因此，潛變是一种“与时间相关的”變形。

潛變變形发生的温度范围因材料不同而不同。例如，钨需要几千度才能发生潛變變形，然而冰可以在冰点下潛變。通常，在金属熔点的大约30%和陶瓷熔点的40%-50%时，潛變的影响开始變得显著。事实上，任何材料在接近其熔点的时候都会发生潛變。由于潛變的最低温度和熔点有关，潛變可以在相对较低的温度下在一些材料上发生，如塑料和低熔点金属，包括许多焊料。室温潛變可以很明显的发生在旧的铅热水管上。[[冰河|冰河]]流也是個常見的潛變例子。

除了在需要保持高温的系统中，例如核电站、喷气发动机和热交换机，对于许多日常用品的设计，考虑潛變變形也是很重要的。例如，金属纸夹比塑料强度大，因为塑料在室温下发生潛變。老化的玻璃窗常常错误的被用来当成这个现象的例子：可观测的潛變仅仅在高于玻璃转變温度（900°F/500°C）下发生。尽管玻璃在正确的条件下展现出潛變，然而旧窗户上明显的下垂现象可能来自废弃的制造工艺，例如用于制造冕牌玻璃而引发不均一厚度的工艺。

一个潛變變形應用的例子是钨灯丝的设计。支柱之间灯丝圈的下垂随时间不断增长，原因是灯丝自身重量而引发的潛變變形。如果过多的變形发生，邻近圈的灯丝相互接触，将引发短路和局部过热，从而很快导致灯丝失效。因此灯丝形状和支柱被设计用来限制由灯丝重量引发的應力，而且一种掺杂了氧在晶界中的特殊的钨被用来减缓Coble潛變的速率。

在蒸汽涡轮发电站中，管道在高温（566°C/1050°F）和高压（24.1MPa/3500psi或更高）下运输蒸汽。在喷气发动机中，温度可以达到1400°C/2550°F，会在[[涡轮叶片|涡轮叶片]]上引发潛變變形。因此，理解材料的潛變變形行为是很重要的。

== 潛變的阶段 ==
在初始阶段，或者初步潛變，应變速率一般较大，但是随着时间增加而放慢。这是由于形變硬化。形變率最后达到一个最小值并接近常数。这是由于形變硬化和退火（热软化）的一个平衡。这个阶段就是第二阶段或者稳态潛變。这个阶段最被了解。“潛變应變率”就是指这一阶段的應變率。应力和应變率的关系随潛變机制不同而不用。在第三阶段，由于颈缩现象，应變率随着应變增大指数性的增长。

== 潛變机制 ==
潛變机制随温度和应力不同而不同。这些不同的方法是：

[[体扩散|基体扩散]]——又被称为Nabarro-Herring潛變

[[攀移|攀移]]——这里应变變际上由攀移实现

攀移辅助[[滑移|滑移]]——这里攀移是一种可能的机制，让[[位错|位错]]绕过障碍

[[晶界扩散|晶界扩散]]——又被称为Coble潛變

热驱动[[滑移|滑移]]——例如，通过交滑移

== 主要潛變公式 ==
: <math> \frac{d\varepsilon}{dt} = \frac{C\sigma^m}{d^b} e^\frac{-Q}{kT}</math>

ε是潛變应變，C是一个依赖于材料和特别潛變机制的常数，m和b是依赖于潛變机制的指数，Q是潛變机制的激活能，σ是加载应力，d是材料的晶粒尺寸，k是波尔兹曼常数，T是绝对温度。

=== 位错潛變 ===
在高应力（相对于剪切模量）下，潛變是一个受位错控制的运动。当应力加载在材料上时，由于滑移面中的位错移动而塑性變形发生。材料中含有大量的缺陷，例如固溶原子，它们可以作为位错运动的障碍。因为位错攀移现象而潛變发生。在高温下晶体中的空位可以扩散到位错中，引起位错移动在最近的滑移面。通过攀移到邻近滑移面位错可以绕过障碍继续运动，从而使进一步的變形发生。空位扩散到位错的位置需要一定时间而导致了时间依赖的變形，就是潛變。

位错潛變中<math>Q=Q_{\text{self diffusion}}</math>, m = 4-6, b=0。因此位错潛變强烈依赖于加载应力而不依赖于晶粒尺寸。

一些合金表现出很大的应力指数（n>10），这已经由引入初始应力<math>\sigma_{th}</math>而解释，低于初始应力时无法测量。这样，修正后的公式就变成：

: <math>\frac{d\epsilon}{dt} = A \left(\sigma-\sigma_{th}\right)^n e^\frac{-Q}{\bar R T}</math>

A、Q和n可以用传统机制解释（3≤n≤10）。

=== Nabarro-Herring潛變 ===
Nabarro-Herring潛變是扩散控制潛變的一种形式。在N-H潛變中，原子通过晶格扩散，造成晶粒沿着应力轴伸长。N-H潛變中，k和原子通过晶格的扩散系数有关，Q = Q<sub>self diffusion</sub>，m = 1，b=2。因此N-H潛變是一种弱应力依赖、中等晶粒尺寸依赖的潛變，它的潛變形變率随着晶粒尺寸增长而降低。

N-H潛變有强烈的温度依赖性。因为材料中发生原子的晶格扩散，晶体结构中附近的晶格点或者空隙点是自由的。一个给定的原子将会克服能量势垒从当前位置（处于一个能量[[势阱|势阱]]当中）移动到邻近的空穴位（另一个势阱）。扩散公式的主要公式是D = D<sub>0</sub>exp(Ea / kT)，D<sub>0</sub>和尝试跳跃频率、最近邻位的数目和这些位成为空位的概率有关。因此它对温度有双重依赖性。在更高的温度下，扩散由于公式的直接温度影响、通过肖特基缺陷的空位增加和材料中原子平均能量的增加而增大。N-H潛變主要发生在相对于材料熔点的很高温度下。

== Coble潛變 ==
Coble潛變是扩散控制潛變的第二种形式。在Coble潛變中原子沿着晶界扩散而使晶粒沿着应力轴拉长。这使得Coble潛變比N-H潛變有更高的晶粒尺寸依赖性。Coble潛變中k和晶界附近的原子的扩散系数有关，Q = Q<sub>grain boundary diffusion</sub>，m=1，b=3。因为Q<sub>grain boundary diffusion</sub><nowiki> < Q</nowiki><sub>self diffusion</sub>，Coble潛變主要发生在比N-H潛變更低的温度。Coble潛變也是温度依赖性的，温度上升，晶界扩散增强。但是，由于最近邻的数目被有效的限制在晶粒的表面，而且空位在晶界的热发生不普遍，Coble潛變的温度依赖性并不如N-H潛變。它和N-H潛變一样体现出和应力的线性相关。

=== [[聚合物|聚合物]]的潛變 ===
潛變可以发生在聚合物和金属这样的粘弹性的材料中。聚合物在力作用下的行为可以用Kelvin-Voigt模型模拟。在这个模型中，材料由一个Hookean弹簧和一个平行的Newtonian阻尼器所表示。潛變应變由下式给出：

:<math>\varepsilon(t) = \sigma C_0 + \sigma C \int_0^\infty f(\tau)(1-\exp[-t/ \tau]) \,d \tau</math>

其中：

σ=加载应力

C<sub>0</sub>=瞬时潛變柔度

C=潛變柔度系数

τ=延迟时间

f(τ)=延迟时间的分配

[[Image:Creep2.JPG|right]]


当在一个阶段稳定的应力下，粘弹性材料的应變经过一个时间依赖的增长。这个现象称为粘弹性潛變。

在时间t<sub>0</sub>时，一种粘弹性材料在一个能够稳定足够长时间的应力下加载。这种材料的应變在应力作用下不断增长直到材料最终失效。当应力持续较短周期时，材料在t<sub>1</sub>之前承受一个初始的应潛，然后应力减轻，应变立刻减小（不连续），然后连续逐渐减小到剩余应變。

粘弹性潛變数据可以通过两种方式表达。给定温度下，全部的应變可以作为时间的函数被画出来。在加载应力的一个特定值下，材料会表现出线粘弹性。高于这个特定应力，潛變应變率将不成比例的更快增长。第二种图形表示材料粘弹性潛變的方式是把潛變模量（特定时间下稳定加载应力除以全应變）作为时间的函数画出来。在它的特殊应力之下，粘弹性潛變模量是和加载应力无关的。如果加载应力地用于材料的特殊应力值时，在不同加载应力作用下的一系列应变对时间的曲线可以被一个粘弹性潛變模量对时间的曲线表达。

另外，聚合物的分子重量影响它的潛變行为。增长的分子重量的效果倾向于提高聚合物链之间的二次成键，使聚合物具有更高的潛變抗性。类似的，芳香聚合物由于其环的额外刚度而具有更高的潛變抗性。分子重量和芳香环增加聚合物的热稳定性，增强了聚合物的潛變抗性（Meyers和Chawla，1999，573）。

聚合物和金属都可以潛變。聚合物可以在高于-200°C的所有温度发生明显的潛變，但是聚合物潛變和金属潛變之间有三点主要区别。金属潛變：不是线性潛變；不可恢复；只在高温下明显。

== 其它例子 ==
尽管主要是因为高温下屈服应力的降低，世界贸易大厦的倒塌部分是由于高温下的潛變。

运行中的核反应堆中的热压成分的潛變应變率可以被极大的从设计上限制，因为潛變应變率可以被高能粒子的流出而提高。
