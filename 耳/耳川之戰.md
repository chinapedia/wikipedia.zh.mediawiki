'''耳川之戰'''是在[[天正|天正]]6年（1578年），[[豐後國|豐後國]]的[[大友宗麟|大友宗麟]]與[[薩摩國|薩摩國]]的[[島津義久|島津義久]]以[[日向|日向]][[高城川原|高城川原]]（[[宮崎縣|宮崎縣]][[木城町|木城町]]）為主戰場發生激烈衝突的合戰。別稱「'''高城川之戰'''」、「'''高城川原之戰'''」。與[[今山之戰|今山之戰]]、[[沖田畷之戰|沖田畷之戰]]同屬[[九州|九州]]三強爭霸間的著名戰役。

== 起因 ==
天正5年（1577年），日向大名[[伊東義祐|伊東義祐]]被[[島津氏|島津氏]]擊敗。義祐被流放，於是寄身於有友好關係的[[大友氏|大友氏]]。翌年，[[大友宗麟|大友宗麟]]和[[大友義統|義統]]決定與宿敵島津氏決戰，於是率領3～4萬大軍遠征日向。

== 概要 ==
在大友家内部，因為宗麟狂熱地信奉[[基督教|基督教]]而與家臣團之間產生不和。[[立花道雪|立花道雪]]等人在開戰前表示強烈反對。而且在合戰軍議中，大友軍中的[[田北鎮周|田北鎮周]]主張交戰，但是大將[[田原親賢|田原親賢]]則在背後與島津軍進行和睦交渉而沒有理會。田北鎮周不服並前往攻撃島津軍，大友軍於是情非得已地與島津軍展開戰鬥。而事前主張「未到戰鬥時機」的大友軍軍師[[角隈石宗|角隈石宗]]看清了結果，於是把秘傳的奥義書燒掉並突入敵陣中戰死。

當初大友軍的兵力比島津軍強很多，但是因為大友軍徐徐而行而導致士兵開始疲勞，而且大友軍為了追撃敵軍而把陣形伸長，島津軍見狀立即突擊並扭轉戰況，最後大友軍敗走。大友軍的戰死人數接近3千人，但是大半是敗走時在渡過耳川的急流時溺死，或是渡河時被突擊的島津軍兵士所殺。

== 影響 ==
[[大友氏|大友氏]]因為此戰，失去了直臣佐伯宗天、田北鎮周、[[築後|築後]]國人蒲池鑑盛等多數重臣和幕下有力武將以及兵力。更令到後來在大友支配地内的[[秋月種實|秋月種實]]（[[築前國|築前國]]）反抗和[[龍造寺隆信|龍造寺隆信]]（[[肥前國|肥前國]]）謀反，引起國內有很多有力國人離反，大大削弱了勢力和領國。而且此戰後，[[立花道雪|立花道雪]]彈劾擔任軍監的[[志賀親守|志賀親守]]。

[[島津氏|島津氏]]在一連串戰爭後，在[[九州|九州]]内已經再沒有敵人能阻止，確立在九州南部（[[薩摩|薩摩]]、[[大隅|大隅]]、[[日向|日向]]）的支配。島津氏在這次大勝利後，令到失去大友氏為後盾的[[球磨|球磨]][[相良氏|相良氏]]降伏，肥後[[阿蘇氏|阿蘇氏]]和肥前[[龍造寺氏|龍造寺氏]]在後來的「[[沖田畷之戰|沖田畷之戰]]」戰敗，九州内的國人們漸漸投向島津方。島津氏更開始侵攻大友氏的本據地[[豐後|豐後]]，一度迫向大友氏首府。但是因為宗麟的邀請而導致[[豐臣秀吉|豐臣秀吉]]介入，於是無可奈何地撤退，遂向秀吉表示恭順。
