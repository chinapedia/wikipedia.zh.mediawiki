'''沪杭甬铁路'''是[[清朝|清朝]]末年计划建设的一条铁路线，原名'''苏杭甬铁路'''，起点改为[[上海|上海]]后，更名为沪杭甬铁路。线路始于[[上海|上海]]，经[[嘉兴|嘉兴]]至[[杭州|杭州]]，并延长至[[绍兴|绍兴]]和[[宁波|宁波]]，其路线大致相当于现在的[[沪昆铁路|沪昆铁路]][[上海南站|上海南站]]至[[萧山西站|萧山西站]]段（原[[沪杭铁路|沪杭铁路]]和[[浙赣铁路|浙赣铁路]]的一部分）和[[萧甬铁路|萧甬铁路]][[杭州南站|杭州南站]]至[[庄桥站_(宁波市)|庄桥站]]。其中，沪杭段于1906年正式开工，并于1909年6月通车<ref>{{cite book |title = 上海地名志 |chapter = 第五篇 交通地名 |isbn = 7-80618-527-5|date=1998年12月}}</ref>。杭甬段于1910年开工，但在1959年前从未完工，[[抗日战争|抗日战争]]时期更遭到拆除<ref name="ningbo-rail-history">{{cite web|url = http://nbwb.cnnb.com.cn/forum.php?mod=viewthread&tid=70992 |title = 宁波铁路的百年沧桑 |publisher = 宁波晚报 |date = 2012-05-06 |accessdate = 2013-06-09}}</ref>。

== 历史 ==
=== 筹建 ===
[[File:苏省铁路公司股票.JPG|thumb]]]]
沪杭甬铁路的历史始于19世纪末。[[中日甲午战争|中日甲午战争]]之后，英、法、俄等国争夺在华铁路修建权利。曾一度落后于法俄的英国在争夺[[卢汉铁路|卢汉铁路]]路权失利后，转而着手实施“长江流域铁路系统”计划。[[光绪|光绪]]二十四年九月初一日（公历1898年10月15日），时任清政府[[铁路督办大臣|铁路督办大臣]][[盛宣怀|盛宣怀]]与英商[[怡和洋行|怡和洋行]]在上海签订《苏杭甬铁路草约》，拟将从[[苏州|苏州]]至杭州并延长至宁波的铁路路权交与英国。但到了1905年，《铁路简明章程》鼓励华商兴办铁路。同时，全国兴起“[[收回利权运动|收回利权运动]]”。在这种情形下，江浙两省绅商提出自办铁路并成立[[浙路公司|浙路公司]]和[[苏路公司|苏路公司]]并获得商部奏准自办铁路。在英商的反对下，清政府拟向英商借款并转借浙路公司修建铁路，遭到民间强烈抵制，史称“沪杭甬铁路风潮”<ref>{{cite article |title = 晚清沪杭甬铁路对英借款刍议 |author = 黄文 |publisher = 牡丹江师范学院学报（哲社版）|year = 2007 |issue = 4}}</ref>。江浙商人成立“国民拒款会”抵制清政府对外借款，同时民间纷纷出资认股，在两月内筹集资金三千六百余万元。1908年2月4日，中英《沪杭甬铁路借款合同》及苏路和浙路公司与[[邮传部|邮传部]]间的《存款章程》签订，铁路借款由邮传部向英方租借，再由邮传部租借给两路公司，且未将路权作为借款抵押而改用[[京奉铁路|关内外铁路]]余利作保，但清政府的这一让步仍未被苏路和浙路公司接受。1911年，两路公司宣布废除原有存款章程，辞退英方工程师，铁路最终完全自办<ref>{{cite article |title = 论沪杭甬铁路的商办历程 |author = 黄文 |publisher = 扬州大学 |year = 2008}}</ref>。1914年，沪杭甬铁路全线收归国有，由中华民国交通部沪杭甬铁路管理局管理<ref name="huhang-his">{{cite web |url = http://ccnews.people.com.cn/GB/13051090.html |title = 沪杭铁路历史变迁 |publisher = 人民网 |date = 2010-10-26 |accessdate = 2013-06-12}}</ref>。

=== 沪杭段 ===
{{See also|沪杭铁路}}

沪杭甬铁路上海至杭州段大部分与现有的[[沪杭铁路|沪杭铁路]]重合。铁路最早修建的部分为杭州湖墅途径{{站|艮山门}}、[[杭州站|城站]]、{{站|南星桥}}至钱塘闸口的路段，称为江墅铁路，于1906年开工建设，1907年竣工通车，是浙江省最早的铁路<ref>{{cite article |title = 浙江境内第一条铁路——江墅铁路建造始末 |author = 杨宝林 |publisher = 今日浙江 |year = 1998 |issue = 20}}</ref>。随后，从[[上海南站_(1908年–1937年)|上海南]]至杭州的线路开工。其中，由浙路公司负责的杭州至枫泾段于1906年9月开工，1909年4月通车。苏路公司负责的上海至枫泾段于1907年2月开工，1909年3月通车。1909年6月，两线接轨，沪杭铁路全线建成。1915年3月，沪杭铁路接轨[[沪宁铁路|沪宁铁路]]线路开工，1916年12月建成通车，沪杭铁路终点改为[[上海北站|上海北站]]<ref name="huhang-his" />。

=== 杭甬段 ===
{{See also|萧甬铁路}}

沪杭甬铁路杭州至宁波段大部分与现有的[[萧甬铁路|萧甬铁路]]重合。杭甬段施工始于1910年6月，1914年建成宁波至曹娥江东岸的铁路，跨曹娥江大桥桥墩也已建设完成。但适逢[[第一次世界大战|第一次世界大战]]，建设曹娥江大桥的德国公司无法继续完成施工，铁路建设中断。1937年[[八一三事变|八一三]]事件发生后，浙江省政府和铁道部全力赶拼工程进度，使钱塘江大桥在9月26日通车<ref name=":0">{{Cite web|title=【铁路抗战故事】抗战中的华东铁路|url=https://mp.weixin.qq.com/s/jL7le-otKT7RFbNa3o0EPQ|work=[[中国铁路总公司|中国铁路总公司]]公众号|accessdate=2019-04-14|author=[[上海铁路局|上海铁路局]]|date=2015-09-06|publisher=}}</ref>。1936年10月，沪杭甬铁路萧绍段续建，并于1937年11月通车，经[[钱塘江大桥|钱塘江大桥]]与沪杭铁路贯通。但曹娥江大桥仍未修建，过江需要依靠渡船接驳。

大桥通车后便担负起铁路和公路的运输任务，使物资得以快速经由[[浙赣铁路|浙赣线]]运往后方，[[沪宁铁路|京沪]]、沪杭甬、[[苏嘉铁路|苏嘉铁路]]的机车车辆和器材也得以转运到浙赣、[[粤汉铁路|粤汉]]、[[湘桂铁路|湘桂]]等线继续使用，居民也得以疏散<ref name=":0" />。此时已进入抗日战争时期，为阻断日军进军路线，通车一个月的沪杭甬铁路萧绍段与曹甬段一起遭到拆除，直至中华人民共和国成立，更名为萧甬铁路的沪杭甬铁路杭甬段方才得到重建<ref name="ningbo-rail-history" />。
