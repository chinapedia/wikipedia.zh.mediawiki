'''沪西礼拜堂'''是中国[[上海|上海]]的一座[[基督教|基督教]][[新教|新教]][[教堂|教堂]]，原名'''沪西神召会耶稣堂'''，位于[[长宁区|长宁区]][[长宁路|长宁路]]1465号。沪西礼拜堂是文革以后上海市区第一座进行重建的新教教堂。

沪西礼拜堂起源于1918年[[倪桂珍|倪桂珍]]、[[黄佩文|黄佩文]]和[[神召会|神召会]]的[[瑞典|瑞典]]女[[传教士|传教士]][[克利生|克利生]]、[[安德生|安德生]]建立的[[北新泾|北新泾]]'''沪西布道所'''。1925年起，由[[汤荣福|汤荣福]]担任[[牧师|牧师]]。1927年迁入[[周家桥|周家桥]][[三角场|三角场]]的租用房屋，命名为'''沪西神召会耶稣堂'''。1937年[[淞沪会战|淞沪会战]]期间，房屋被毁，1940年，[[汤荣福|汤荣福]]牧师发动信徒捐款，在[[白利南路|白利南路]]（今[[长宁路|长宁路]]）建造教堂。<ref>{{cite web

1958年9月，在[[献堂献庙|献堂献庙]]高潮中，沪西神召会耶稣堂被保留，作为[[长宁区|长宁区]]的3座[[联合礼拜|联合礼拜]]教堂之一，[[凯旋路|凯旋路]][[士每拿堂|士每拿堂]]、[[北新泾|北新泾]][[耶稣堂|耶稣堂]]、[[长宁路|长宁路]][[救恩会耶稣堂|救恩会耶稣堂]]和[[中山北路|中山北路]][[恩基堂|恩基堂]]信徒并入该堂聚会。1960年，[[新恩堂|乌鲁木齐北路聚会所]]、[[锡安堂_(上海)|锡安堂]]均被划入[[静安区|静安区]]，沪西神召会耶稣堂成为[[长宁区|长宁区]]唯一的[[联合礼拜|联合礼拜]]地点。<ref>{{cite web
