'''东京都立滨离宫恩赐庭园'''（{{lang-ja|東京都立浜離宮恩賜庭園}}），又称'''滨离宫恩赐庭园'''或'''旧滨离宫庭园'''，是位于[[日本|日本]][[东京都|东京都]][[中央区_(东京都)|中央区]]的一座都立庭园。庭园面积约25万平方米，内有潮入之池和两处猎鸭场，是[[江户时代|江户时代]]具有代表性的[[大名|大名]][[日本庭園|庭园]]。滨离宫恩赐庭园及其周边的水面是日本的国家指定[[特别史迹|特别史迹]]、[[特别名胜|特别名胜]]。

== 庭园历史 ==

庭园的所在地最初是[[江户幕府|江户幕府]]在[[宽永|宽永]]年间开拓的[[征夷大將軍|将军]]家的猎鹰场，是一整片的芦苇地。1654年（[[承应|承应]]3年），第三代将军[[德川家光|德川家光]]的儿子、[[甲府|甲府]]宰相[[德川纲重|德川纲重]]从其兄长、第四代将军[[德川家纲|德川家纲]]处拜领了这片土地，在这里建立了别庄，名为{{nihongo|'''甲府滨屋敷'''|甲府浜屋敷}}，又称{{nihongo|'''海手屋敷'''|海手屋敷}}。在德川纲重的儿子[[德川家宣|德川家宣]]成为第六代将军时，将这片宅地再次作为将军家的别庄，改名为{{nihongo|'''滨御殿'''|浜御殿}}，并进行了大幅度的改修。此后，经过历代将军几次的建设和改修，在第十一代将军[[德川家齐|德川家齐]]时，这片宅地基本有了如今庭院的模样<ref>{{cite web |url=http://teien.tokyo-park.or.jp/contents/outline028.html |title=浜離宮恩賜庭園：概要 |publisher=東京都公園協会 |language=ja |accessdate=2010-03-27 }}</ref>。

幕府末期，庭园所在地成为幕府海军的基地，称为{{nihongo|'''海军所'''|海軍所}}，幕府海军的{{nihongo|军舰训练基地|軍艦操練所}}也迁移至此处。同时，日本的第一座石造西洋建筑——名为{{nihongo|'''延辽馆'''|延遼館}}的幕府海军教学基地——开始建设<ref>{{cite web |url=http://www.komazawa-u.ac.jp/~kazov/Nis/study/castles/hamarikyu-brief+.ppt |title=浜離宮：江戸幕府の海軍基地 |publisher=中世城郭研究会　西村 和夫 |language=ja |accessdate=2010-03-27}}</ref>。1869年（[[明治|明治]]2年），延辽馆建设完成，建筑面积约1,400平方米，成为明治天皇接待国外宾客的迎宾馆。直到1889年（明治22年）[[鹿鸣馆|鹿鸣馆]]建成后取代了延辽馆成为新的迎宾馆，老旧的延辽馆被拆除为止，包括美国第18任总统[[尤利塞斯·S·格兰特|格兰特]]夫妇等人都曾经在此居住<ref>{{cite web |url=http://www.tokyochuo.net/issue/city/0205_hama/0205_ver1.html |title=浜離宮恩賜庭園観光編 |publisher=NPO法人東京中央ネット |language=ja |accessdate=2010-03-27 }}</ref>。除此以外，庭园的其他部分在明治维新中作为皇室的离宫，划归[[宫内厅|宫内省]]管辖，改名为{{nihongo|'''滨离宫'''|浜離宮}}。

1945年（[[昭和|昭和]]20年）11月3日，[[二战|二战]]战败的日本政府以[[駐日盟軍總司令|盟军总司令部]]的要求将滨离宫下赐给[[东京都|东京都]]。次年4月1日，修整后的滨离宫作为都立庭园向一般民众收费开放。1949年（昭和23年）12月，根据日本的《文化财产保护法》，{{nihongo|'''旧滨离宫庭园'''|旧浜離宮庭園}}被指定为国家名胜及史迹。1952年（昭和27年）11月，庭园及其周边共32.4[[公顷|公顷]]的面积被指定为日本的国家[[特别名胜|特别名胜]]及[[特别史迹|特别史迹]]。

== 主要景观 ==
;{{nihongo|潮入之池|潮入の池、しおいりのいけ}}
: 潮入之池是将海水导入庭园中的水池，利用[[潮汐|潮汐]]的涨落使得水池产生不同的情趣，是海边的日本庭园常用的设计样式。[[旧芝离宫恩赐庭园|旧芝离宫恩赐庭园]]、[[清澄庭园|清澄庭园]]和[[旧安田庭园|旧安田庭园]]原先也设有潮入之池，但如今滨离宫恩赐庭园的潮入之池是东京都唯一有海水实际出入的水池<ref>{{cite web |url=http://www.kensetsu.metro.tokyo.jp/kouen/kouenannai/park/hamarikyu.html |publisher=东京都建设局 |title=都立公園·庭園案内 |language=ja |accessdate=2010-03-26 }}</ref>。

;{{nihongo|猎鸭场|鴨場、かもば}}
: 庭园内有庚申堂鸭场和新钱座鸭场两座猎鸭场，分别建于1778年（[[安永|安永]]9年）和1791年（[[宽政|宽政]]3年）。猎鸭场是由3米的土丘围绕的水池和树林，土丘上种植的常绿树和竹子将猎鸭场与外界隔断，使得鸭群可以在其中休憩。从池边挖有几条浅壕，狩猎时从细缝中观察鸭的动向，用谷物将鸭引向浅壕，再伺机用网兜从隐蔽处将鸭捕获<ref>{{cite web |url=http://www.tokyo-park.or.jp/park/format/view028.html |title=浜離宮恩賜庭園 - 見どころ |publisher=东京都公园协会 |language=ja |accessdate=2010-03-26 }}</ref>。

;{{nihongo|鸭冢|鴨塚、かもづか}}
: 1935年（[[昭和|昭和]]10年）11月5日，为了祭慰庭园内成为猎鸭的猎物的鸭的灵魂，在新钱庄鸭场的东侧修建了一座鸭冢。

;{{nihongo|三百年之松|三百年の松、さんびゃくねんのまつ}}
: 位于大手门侧，是江户[[幕府|幕府]]第6代将军[[德川家宣|德川家宣]]在对庭园进行大型改造时为了宣扬自己的伟业而种植的。松树高约18米，特点是粗大的树枝低低的垂下，树姿显得端庄大方，是东京最大规模的[[黑松_(植物)|黑松]]之一。

;{{nihongo|中岛的茶屋|中島の御茶屋、なかしまのおちゃや}}
: 潮入之池的中央位置有一个小岛，和池岸以长118米的扁柏木制成的{{nihongo|传桥|お伝い橋、おつたいはし}}相连。岛上建有一间[[茶屋|茶屋]]，水面倒影着茶屋是庭园的一大风景。茶屋最早于1707年（[[宝永|宝永]]4年）落成，向海眺望曾经可以看见房总半岛，如今则是彩虹大桥和台场的风景。现在的茶屋是1983年（[[昭和|昭和]]58年）重建的，游客可以在其中休息<ref>{{cite web |url=http://teien.tokyo-park.or.jp/cn/hama-rikyu/map.html |title=滨离宫恩赐庭园 - 园内介绍 |publisher=东京都公园协会 |language=zh |accessdate=2010-03-26 }}</ref>。
