'''“滨州”号导弹护卫舰'''（[[舷号|舷号]]515）简称'''“滨州”舰'''，是[[中国人民解放军海军|中国人民解放军海军]]第二十三艘[[054A型导弹护卫舰|054A型导弹护卫舰]]，可以单独或者协同海军其他[[兵力|兵力]]攻击敌[[水面舰艇|水面舰艇]]及[[潜艇|潜艇]]，具有较强的远程警戒和[[防空作战|防空作战]]能力。滨州舰由[[沪东造船厂|沪东造船厂]]建造，2015年12月13日下水，2016年12月29日入列，服役于[[东海舰队|东海舰队]][[驱逐舰第六支队|驱逐舰第六支队]]。<ref>{{cite web |url=http://www.msa.gov.cn/html/xinxichaxungongkai/gkml/HXTG/SHHSJ/20151210/2BF6792F-ACD4-4112-A413-1A01A154D606.html |title=黄浦江浦东侧沪东船厂新船出坞，部分江面禁止通航 |publisher=中国海事局 |date=2015-12-10 |deadurl=yes |archiveurl=https://web.archive.org/web/20151222075936/http://www.msa.gov.cn/html/xinxichaxungongkai/gkml/HXTG/SHHSJ/20151210/2BF6792F-ACD4-4112-A413-1A01A154D606.html |archivedate=2015-12-22 }}</ref>滨州舰舰名取自[[山东省|山东省]][[滨州市|滨州市]]，其入列命名授旗仪式于2016年12月29日在[[浙江省|浙江省]][[舟山市|舟山市]]某军港举行。<ref>{{cite web |url=http://www.wifizs.cn/folder1/folder127/2016-12-29/99456.html |title=滨州舰入列命名授旗仪式在我市举行 |publisher=舟山广电海洋宽频网 |date=2016-12-29 |deadurl=yes |archiveurl=https://archive.is/20161229034820/http://www.wifizs.cn/folder1/folder127/2016-12-29/99456.html |archivedate=2016-12-29 }}</ref>
