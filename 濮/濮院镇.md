'''濮院镇'''是[[中华人民共和国|中华人民共和国]][[浙江省|浙江省]][[嘉兴|嘉兴]][[桐乡市|桐乡市]]下属的一个镇，位于市境东部，境内有[[京杭大运河|京杭大运河]]、[[申嘉湖高速公路|申嘉湖高速公路]]和[[320国道|320国道]]经过。全镇总面积64.71平方公里，总人口20万人，其中常住人口4.6万人<ref>{{cite web |url=http://www.cnjxol.com/xwzx/jxxw/qxxw/tx/content/2007-10/25/content_533290.htm|title=《我市对部分行政区划进行调整》|accessdate=2015-05-31|work=|publisher=嘉兴在线}}</ref><ref>{{Cite book|title=中国共产党浙江省嘉兴市组织史资料 第4卷|last=中共嘉兴市委组织部编|first=|publisher=杭州：浙江人民出版社|year=2009年12月|isbn=978-7-213-04176-1|location=|pages=300-301}}</ref>。

== 概述 ==
此地古称'''槜李'''、'''梅泾'''，南宋时著作郎[[濮凤|濮凤]]随宋高宗南渡后卜居于此，其六世孙[[濮斗南|濮斗南]]拥立宋理宗有功，擢吏部侍郎，赐第曰“濮院”，遂以此名。清代时东北属[[秀水县|秀水县]]，西南属[[桐乡县|桐乡县]]，两县各得其半<ref name="县志">清·严辰纂修，《光绪桐乡县志·卷一·疆域上·市镇》：濮院镇古名梅泾，在县城东北十八里，东西广三里许，南北袤二里许，西南为桐界，东北为秀界，各得其半。……水自运河来，其分三道，西南由永新港来，至梧桐桥入市，东北由陡门塘来，至灵官桥入市，西北自塘口来，至秀桐桥入市。去水只两路，一由镇南之聚星桥，一由东南之幽湖桥，形家谓此地受乾艮之水，利于上中二元，而五黄运为独盛，故乾隆时日出万绸，至道光十年后绸业渐衰，迨遭咸丰庚申之劫，市廛零落，克复后渐有转机，聚市之处桐界为多，至大庙宇及官廨、书院、善堂均在秀界也。</ref>。民国时则以市河为界，东北属嘉兴县，称嘉濮镇，西南属桐乡县，称桐濮镇。1950年两镇合并，称濮院镇。1958年改公社，1959年复改镇，1965年改称卜院镇，1989年复名濮院<ref>{{cite book |author=戴均良、刘保全、邹逸麟等|title=《中国古今地名大词典》|year=2010|publisher=上海辞书出版社|location=上海|pages=}}</ref>。古镇街区仍保存完好，历史遗迹众多，如濮院古桥群（含众安桥、女儿桥、栖凤桥、大德桥、大有桥、大积桥、定泉桥、秀桐桥及王板桥<ref>{{cite web |url=http://www.jiaxing.gov.cn/mljx/jxwh_36090/lswh/wzwhyc/sjwwbhdw/201401/t20140114_308698.html|title=《濮院古桥群》|accessdate=2015-05-31|work=|publisher=嘉兴市政府}}</ref>）、[[大有桥街章宅|大有桥街章宅]]、[[翔云观|翔云观]]等。

该镇在历史上是[[杭嘉湖平原|杭嘉湖平原]]的工商业名镇，号称明清时期“江南五大名镇”之一，也是著名的[[濮绸|濮绸]]发源地，明清时丝绸业鼎盛，乾隆时曾日出万绸<ref name="县志" />。现状以毛针织产业著称，被誉为“中国羊毛衫名镇”<ref>{{cite web|url=http://txnews.zjol.com.cn/txnews/system/2014/12/15/018809232.shtml|title=《濮院镇简介》|accessdate=2015-05-31|work=|publisher=濮院镇政府|deadurl=yes|archiveurl=https://web.archive.org/web/20150512075804/http://txnews.zjol.com.cn/txnews/system/2014/12/15/018809232.shtml|archivedate=2015-05-12}}</ref>。

==辖区==
该镇现辖5个居委会，14个行政村<ref>{{Cite book|title=浙江建置区划沿革|last=俞志壮主编|first=|publisher=杭州：浙江大学出版社|year=2009年4月|isbn=978-7-308-06705-8|location=|pages=387}}</ref>：包括新生社区、梅园社区、锦苑社区、双龙社区、永乐社区、新河村、永越村、永乐村、新濮村、新星村、永联村、新东村、新妙智村、新联村、新港村、红旗漾村、油车桥村、星旗村。
