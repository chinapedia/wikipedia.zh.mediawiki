{{Vgname|突袭2|en=Sudden Strike 2|official-cn=突袭II}}是一款以[[二战|二战]]为背景的即时战术游戏，是突袭系列的第二部作品。<ref>{{Cite web |url = http://www.eurogamer.net/articles/p_sudden2 |title = Sudden Strike II  |author =  |date = 2002-12-06 |publisher = Eurogamer.net |language = en |accessdate = 2017-08-12 }}</ref>相比[[突袭_(游戏)|前作]]中的苏军、德军、盟军（英美军）三方势力，本作中追加了日军这一势力。<ref>{{Cite web |url = http://games.sina.com.cn/newgames/2003/05/05191624.shtml |title = 组图：五国军队任你指挥　二战重现《突袭 2》 |author =  |date = 2003-05-19 |publisher = 新浪 |language = zh-cn |accessdate = 2017-08-12 }}</ref>

==突袭：资源战争==
《突袭：资源战争》（{{lang-en|Sudden Strike: Resource War}}）是基于《突袭2》引擎制作的独立资料片（无需突袭2原版游戏支持）。<ref>{{Cite web |url = http://games.sina.com.cn/newgames/2004/08/081740777.shtml |title = 《突袭2》最新资料片网上曝光(组图) |author =  |date = 2004-08-18 |publisher = 新浪 |language = zh-cn |accessdate = 2017-08-12 }}</ref>于2004年11月在北美发售。<ref>{{Cite web |url = http://www.ign.com/games/sudden-strike-resource-war/pc-708223 |title = Sudden Strike: Resource War |author =  |date = 2004-11-09 |publisher = IGN |language = en |accessdate = 2017-08-12 }}</ref>《突袭：资源战争》中包含有地图编辑器。
