'''表字'''，又称'''字'''，是[[漢字文化圈|漢字文化圈]]傳統上在[[本名|本名]]之外的一個正式代稱。通常基於禮貌性質，一般習慣間接性叫喚他人的表字而不直稱其本名，以表示客氣避免失禮（本名則可以是當事人自述或由尊長直稱）。

表字的取法，一般与本名意义相关。歷史上有些人的表字廣為人知，本名反而隱沒於世，稱為「[[#以字行|以字行]]」，如：[[孟浩然|孟浩然]]，名'''浩'''，字'''浩然'''。

== 簡介 ==
根據記載，古代男子20歲行[[冠禮|冠禮]]（[[唐代|唐代]]稱為[[元服禮|元服禮]]）時取字，女子15歲[[笄禮|及笄]]（日本稱為{{tsl|ja|裳着|裳著禮}}或[[染黑齒|鐵漿禮]]）或[[许嫁|许嫁]]時取字。如[[項籍|項籍]]字羽，[[孔丘|孔丘]]字仲尼，[[司马迁|司马迁]]字子長，[[李白|李白]]字太白。

字與名通常有關。如[[杜甫|杜甫]]字子美，甫為古代男子美稱；[[韓愈|韓愈]]字退之，為互補；[[杜牧|杜牧]]字牧之，為相似；[[王維|王維]]字摩詰，為欣賞[[佛教|佛教]]人物[[維摩詰|維摩詰]]而名。另外，亦有将伯、孟、仲、叔、季加於字中表以示排行，如[[孙坚|孙坚]]4子（[[孙策|孙策]]字伯符，[[孙权|孙权]]字仲谋，[[孙翊|孙翊]]字叔弼，[[孙匡|孙匡]]字季佐），孟為庶長子專用，如馬超字孟起。

根據《[[禮記|禮記]]·檀弓》<ref>「幼名，冠'''字'''，五十以伯仲，死謚，周道也。」參見《禮記》〈[[:s:禮記/檀弓上|檀弓上第三]]〉篇。</ref>上的說法，在人[[成年|成年]]后，需要受到社会的尊重，同輩人直呼其名會显得不恭敬；于是需要取一个字，用来在社會上與别人交往时使用，以示相互尊重。因此，古人在成年以后，名字只供[[長輩|長輩]]恩師和自己稱呼，自稱其名表示謙遜，而字才是用來供社會上的人彼此稱呼的。

[[北齊|北齊]][[顏之推|顏之推]]認為，[[人名|人名]]用以區別彼此，字則是用以體現一個人的德行。大部分人的名与字在意义上都是有關聯的，如[[岳飛|岳飛]]字鵬舉（飛、鵬相關），[[關羽|關羽]]字雲長（羽、雲相關）。

字起源於[[商朝|商朝]]，盛行于[[周朝|周朝]]，後來逐漸形成了一種制度。直到近代仍然被許多知識界和文化界的人使用，如：[[胡適|胡適]]字適之，[[孫文|孫文]]字載之，[[毛澤東|毛澤東]]字潤之，[[李敖|李敖]]字敖之。隨著社會風氣上使用表字的慣例逐漸減少，現代社會的[[户口|户口]]登記的發展中也並未規範納入，表字的文化傳統在20世紀逐漸式微，已經很少人於日常生活使用，只在一些較注重文化修養層面和道德自律嚴格的家庭、宗族、團體和個人仍然保持和沿用。現代由於[[中國人口|中國人口]]眾多，既有十三億人漢字同名同姓重複率過高，一些民間人士與學者為此建議參酌文化基礎，支持在現代戶政的管理制度上建立「姓、名、字」三大系統以合理的相關辦法增加个人的識別度。

字也有一言字（如[[項籍|項籍]]字羽，[[屈原|屈原]]字原、名平）跟三言字的形式（如[[李建成|李建成]]字[[毗沙門|毗沙門]]）。最常見的是两言字。

字的构成的一些形式：
* 在字前加男子的尊称“子”字：如[[公孙侨|公孙侨]]字子产，杜甫字子美。
* 在字前加排行字表示排行：孔丘字仲尼；也有-{只}-以排行作为字的：如[[劉邦|劉邦]]字季。

== 特點 ==
[[先秦|先秦]][[女子|女子]]的字构成方式和[[男子|男子]]不太一样，稱呼女子時是先字後姓，另外許多是在[[姓氏|姓氏]]上冠以[[排行|排行]]字作为字，如[[季姬|季姬]]（[[姬姓|姬姓]]人家的幼女）、[[孟姜|孟姜]]（[[姜姓|姜姓]]人家的長女）、[[叔隗|叔隗]]（[[隗姓|隗姓]]人家的三女）等。

另外在[[古文|古文]]中，男性名与字连称时，习惯上先称字后称名。如《[[史记|史记]]》中称呼[[孔子|孔子]]的[[父亲|父亲]]为“[[叔梁纥|叔梁纥]]”，其中“叔梁”是字，“纥”是名；女性的完整姓名則是「字、姓、名」的次序，如[[季芈|季芈]]稱為「季芈畀我」，「季」是字，[[芈姓|芈姓]]，「畀我」是名。

== 以字行 ==
'''以字行'''是“以字行于世”的意思，是一個關於稱謂的術語。

[[漢字文化圈|漢字文化圈]]中，[[中国|中国]]、[[朝鮮半島|朝鮮半島]]、[[越南|越南]]等地之古人，一般有“[[名|名]]”有“字”，「以字行」即是因为种种原因，多僅稱呼此人的“字”，而不熟悉其“[[名|名]]”。

例如[[明朝|明朝]]著名阁臣[[杨士奇|杨士奇]]，名'''寓'''，字士奇，但世人通常僅知其字，所以称之为「以字行」<ref>《明史·列传第三十六》</ref>。

反例，如'''[[李白|李白]]'''，字'''太白'''，'''[[蒋中正|蒋中正]]'''，字'''介石'''，因名、字都為世人所熟悉，就不能說是「以字行」。
