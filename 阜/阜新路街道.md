'''阜新路街道'''是[[中华人民共和国|中国]][[山东省|山东省]][[青岛市|青岛市]][[市北区|市北区]]下辖的一个街道。

== 行政区划 ==

阜新路街道下辖南山社区、阜康社区、小村庄社区、康平路社区、东太平社区、海泊桥社区、西太平社区、南宁路社区、抚顺支路社区。<ref>{{Cite web|url=http://www.stats.gov.cn/tjsj/tjbz/tjyqhdmhcxhfdm/2015/37/02/03/370203026.html|title=统计用区划和城乡划分代码/山东省/青岛市/市北区/阜新路街道办事处|accessdate=2017-04-23|author=国家统计局|date=2015-09-30|publisher=}}</ref>
== 人口 ==

2010年[[中国第六次人口普查|中国第六次人口普查]]时，阜新路街道共有人口94170人。共有家庭29743户，平均每户2.64人。14岁以下的少年儿童共8158人，占总人口8.663%；15-64岁人口共76800人，占总人口81.55%；65岁及以上的老年人共9212人，占总人口的9.782%。男性共计48181人，占总人口51.16%；女性共计45989，占总人口48.83%。本地居住的人口中，拥有本地户籍的人口为55325人，占58.75%。<ref name="人口普查2010">{{cite web| url =http://www.yearbookchina.com/navipage-n2013030152000017.html| title =《中国2010年人口普查分乡、镇、街道资料》| author =中国国务院人口普查办公室、中国国家统计局人口就业统计司等| year =2010| website =统计年鉴分享平台| page =表15 山东省乡、镇、街道人口| format =收费| access-date =2018-07-19| archive-url =https://web.archive.org/web/20180719084356/http://www.yearbookchina.com/navipage-n2013030152000017.html| archive-date =2018-07-19| dead-url =no}}</ref>
