'''阜阳县'''，[[中国|中国]]旧[[县|县]]名。

[[清朝|清朝]][[雍正|雍正]]十三年（1735年），[[清朝政府|清朝政府]]升[[颍州_(北魏)|颍州]]为[[颍州府|颍州府]]，增设阜阳县，为[[附郭县|附郭县]]。评价：[[冲繁疲难|繁，疲，难]]。民国时仍属[[安徽省_(中華民國)|安徽省]]。

1947年6月25日，[[中国人民解放军|中国人民解放军]][[豫皖苏军区|豫皖苏军区]]独立旅和中原独立旅从[[太和县|太和县]]原墙集（今安徽省太和县[[原墙镇|原墙镇]]）出发，占领阜阳县城。6月27日，解放军撤出阜阳，[[国民政府|国民政府]]继续占据阜阳。10月11日，[[华东野战军|华东野战军]][[华东野战军第6纵队|六纵]]十七师四十九团再度占领阜阳。第三天，解放军撤离。1948年3月20日，[[晋冀鲁豫野战军|晋冀鲁豫野战军]]一纵与十一纵[[会师|会师]]后，于25日奉命东进，围攻阜阳。4月7日，[[國民革命軍第七十四軍|国军整编七十四师]]进入阜阳，后于7月29日夜（一说7月27日夜<ref name="a1">{{Cite web |url=http://www.ahage.net/fuyang/fuyangdiqudashiji_9.html |title=阜阳地区历史大事记（商朝至解放初）(9) |accessdate=2014-10-04 |author=安史 |date=2008-07-22 |work= |publisher= |language=Zh-hans |archive-url=https://web.archive.org/web/20141006153223/http://www.ahage.net/fuyang/fuyangdiqudashiji_9.html |archive-date=2014-10-06 |dead-url=yes }}</ref>）奉命撤离。次日，国民政府在阜阳的守军全部逃窜<ref>{{Cite web |url=http://www.ahage.net/fuyang/jiefangfuyang.html |title=阜阳地区战争历史——解放阜阳 |accessdate=2014-10-04 |author=安史 |date=2008-07-22 |work= |publisher=安徽文化网 |language=Zh-hans |archive-url=https://web.archive.org/web/20141006124008/http://www.ahage.net/fuyang/jiefangfuyang.html |archive-date=2014-10-06 |dead-url=yes }}</ref>，豫皖苏军区四分区独立三团进入阜阳<ref name="a1"/>，至此国民政府对阜阳县的管辖彻底结束。

1949年后，属[[阜阳专区|阜阳专区]]（属[[皖北行署区|皖北行署区]]）。1950年，将[[阜城市|阜城市]]并入阜阳县。1964年，析阜阳县和[[涡阳县|涡阳]]、[[蒙城县|蒙城]]、[[凤台县|凤台]]三县置[[利辛县|利辛县]]。1971年为[[阜阳地区|阜阳地区]]所辖，1992年11月20日撤销阜阳县，并入当时的县级[[阜阳市|阜阳市]]，以原县级阜阳市和阜阳县的行政区域为新县级阜阳市的行政区域。县级阜阳市于1996年1月1日撤销，其行政区域被划分为地级[[阜阳市|阜阳市]]三个[[市辖区|市辖区]]即[[颍州区|颍州区]]、[[颍东区|颍东区]]和[[颍泉区|颍泉区]]，小部分劃入[[亳州市|亳州市]]。
