{{札沼線RDT}}
'''札沼線'''（{{jpn|j=札沼線|hg=さっしょうせん|rm=Sasshō sen|f=y}}）是連結[[北海道|北海道]][[札幌市|札幌市]][[中央區_(札幌市)|中央區]]的[[桑園站|桑園站]]與北海道[[樺戶郡|樺戶郡]][[新十津川町|新十津川町]]的[[新十津川站|新十津川站]]，屬於[[北海道旅客鐵道|北海道旅客鐵道]]（JR北海道）的鐵路線（[[地方交通線|地方交通線]]）。設有愛稱「'''學園都市線'''」（{{lang|ja|学園都市線}}）。

== 路線概要 ==
根據{{link-ja|鐵道敷設法|鉄道敷設法|改正鐵道敷設法}}{{link-ja|鐵道敷設法別表第136號|鉄道敷設法別表第136号|別表第136號}}規定有「[[石狩國|石狩國]]札幌開始途經[[當別町|當別]]至[[沼田町|沼田]]的鐵路」（{{lang|ja|石狩國札幌カラ當別ヲ經テ沼田ニ至ル鐵道}}」，全線通車時是一條由桑園站開始至可以與[[留萌本線|留萌本線]]連接的[[雨龍郡|雨龍郡]][[沼田町|沼田町]]的[[石狩沼田站|石狩沼田站]]的路線。[[新十津川站|新十津川站]]－石狩沼田站間在1972年（昭和47年）6月19日因應「[[赤字83線|赤字83線]]」而廢除，成為現在的營業路段<ref name="tanaka1 130-131">[[#tanaka1|『写真で見る北海道の鉄道』 上巻 国鉄・JR線 130-131頁]]</ref><ref name="jrhokkaido-press-20090909-2" group="報道" />。但是，雖然路線的起點是桑園站，但是該站沒有始終着的列車，作為運行系統的「札沼線（學園都市線）」再停靠1個站[[札幌站|札幌站]]。並行於鋪設[[函館本線|函館本線]]的複線旁，直通札沼線系統專用的單線<ref name="rj339-98">[[#RJ339|『鉄道ジャーナル』通巻339号 98頁]]</ref>。

現在的營業路段以中途的[[北海道醫療大學站|北海道醫療大學站]]為界，有很大差異。南部的桑園站－北海道醫療大學站間自1980年代後半以後的沿線開發與輸送強化，成為{{link-ja|札幌都市圈|札幌都市圏}}的通勤、通學路線，作為JR的[[地方交通線|地方交通線]]卻有比較多的列車班次和使用人數，也可使用[[IC卡|IC]][[乘車卡|乘車卡]]「[[Kitaca|Kitaca]]」。另外，此路段由{{link-ja|北海道高速鐵道開發|北海道高速鉄道開発}}為事業主體進行電氣化工程，2012年（平成24年）6月1日完成[[交流電|交流電]][[電氣化鐵路|電氣化]]
<ref group="報道" name="jrhokkaido-press-20120314-1">{{Cite press release|url=http://www.jrhokkaido.co.jp/press/2012/120314-1.pdf|format=PDF|title=学園都市線電化開業に伴う電車の投入（第一次）について|publisher=北海道旅客鉄道|date=2012-03-14|accessdate=2012-09-25|archiveurl=https://web.archive.org/web/20120925203437/http://www.jrhokkaido.co.jp/press/2012/120314-1.pdf|archivedate=2012年9月25日|deadlinkdate=2017年10月}}</ref>。另一方面，北部的北海道醫療大學站－新十津川站間是列車班次和使用人數都非常少的閑散線區，故而繼續維持{{link-ja|非電氣化|非電化}}。另外，末端路段形成夾着[[石狩川|石狩川]]與函館本線並行的情況<ref group="注釋">起初石狩當別站－石狩沼田站間作為「鄰近的並行路線，可進行代替輸送的路線」而在二戰時納入[[不要不急線|不要不急線]]，列車暫時停止運行。</ref>。

=== 廢線問題 ===
2016年11月18日，JR北海道由於嚴峻的經營環境，發表10條路線共13個區間為「難以單獨維持正常營運的路線」<ref name="abandon_20161118A">{{cite web |url=http://www.jrhokkaido.co.jp/press/2016/161118-3.pdf |language=ja |title=当社単独では維持することが困難な線区について |publisher=JR北海道 |date=2016-11-18 |accessdate=2017-07-06 |archiveurl=https://web.archive.org/web/20161118050351/http://www.jrhokkaido.co.jp/press/2016/161118-3.pdf |archivedate=2016-11-18 |deadurl=yes }}</ref>。而札沼線的情況，桑園站至北海道醫療大學站路段屬於JR北海道能夠獨自繼續營運的路線。另一方面，北海道醫療大學站至新十津川站則屬於「為了能夠持續營運的交通體系，需要向地方政府商討轉變成巴士等交通工具的可能性」，JR北海道會與沿線的地方政府商討廢線及替代巴士的相關事宜<ref name="abandon_20161118A"/><ref name="abandon_20161118B">{{cite web |url=http://www.jrhokkaido.co.jp/press/2016/161118-4.pdf |language=ja |title=当社単独では維持することが困難な線区について（PowerPoint版本） |publisher=JR北海道 |date=2016-11-18 |accessdate=2017-07-06 |archiveurl=https://web.archive.org/web/20161118121205/http://www.jrhokkaido.co.jp/press/2016/161118-4.pdf |archivedate=2016-11-18 |deadurl=yes }}</ref>。

== 沿線概況 ==
由桑園車站往北伸展、經過八軒至新琴似之間的高架橋雙線路段。非電氣化，這類橋在日本中是很少見到，在JR專團中是唯一一個路段，其他鐵道中，例子包括[[東海交通事業城北線|東海交通事業城北線]]及[[伊勢鐵道伊勢線|伊勢鐵道伊勢線]]。

新川及新琴似之間會跨越[[札樽自動車道|札樽自動車道]]之上。在高架路段之後就在住宅街之中穿透，之後便經過北海道內最長的石狩川橋，跨越石狩川進入[[當別町|當別町]]。過橋之後便會經過札幌近郊的住宅地及農地。與剛才的市街大幅度改變。札比內車站至新十津川站之間都是經過一些平坦的農地。

== 運行狀況 ==
現時並沒有行走全線的定期班次。電化後，由札幌開出的列車全以[[愛之里公園站|愛之里公園]]、石狩當別及北海道醫療大學為總站，繁忙時間每小時開出4班，其餘時間則每小時開出2-3班；非電化區域則全部以石狩當別為起訖站，其中每天有1.5個短程來回至[[石狩月形站|石狩月形站]]、5個來回至[[浦臼站|浦臼站]]、1個來回至新十津川站，當中浦臼站至新十津川站為[[JR集團|JR集團]]中最少客運列車服務的路段。所有行經非電化區域的列車，都是[[一人控制|一人駕駛列車]]。

2012年10月27日時間表改正後，加開了早上由[[苫小牧車站|苫小牧]]開往愛之里公園、石狩當別開往[[新千歲空港站|新千歲空港]]，晚上由新千歲空港開往石狩當別，以及由石狩當別開往[[千歲站|千歲]]的班次。同一時間，亦把自2000年起開設直通[[函館本線|函館本線]]至[[江別車站|江別車站]]的服務取消了。

路線內的列車會[[夜間滯泊|夜間滯泊]]。

函館本線[[瀧川車站|瀧川車站]]和新十津川車站的距離只有3公里，只隔了一條[[石狩川|石狩川]]。由於由2016年3月26日起該站僅在早上安排一來回列車服務，如無法乘搭該班列車可以乘坐[[北海道中央巴士|北海道中央巴士]]之路線往來兩車站。

== 歷史年表 ==
* 1931年（昭和6年）10月10日：札沼北線開始營業，路線為石狩沼田至中德富，新設北龍車站、碧水車站、和車站、石狩追分車站、雨龍車站、上德富車站、石狩橋本車站及中德富車站（第一代）。
* 1934年（昭和9年）10月10日：路線由中德富延長至浦臼，新設下德富車站及浦臼車站。
** 11月20日：札沼南線(25.9km：)開始營業，路線為桑園至石狩當別，新設新琴似車站、篠路車站、石狩太美車站、石狩當別車站。
* 1935年（昭和10年）10月3日：石狩當別－浦臼之路段開通，札沼北線及札沼南線合併為札沼線，新設石狩金澤車站、本中小屋車站、中小屋車站、石狩月形車站、札比內車站及晩生內車站。
* 1943年（昭和18年）10月1日：石狩月形－石狩追分之路段停用。停用車站：札比內、晩生內、浦臼、下德富、中德富、石狩橋本、上德富及雨龍。

== 電氣化工程 ==
為了要提高路線的便利性、解決交通堵塞和減少行車時間，<ref>[http://www.hobidas.com/news/article/106105.html JR北海道　札沼線中部分路段電氣化] {{webarchive|url=https://web.archive.org/web/20120513060136/http://www.hobidas.com/news/article/106105.html |date=2012-05-13 }} - 貓出版 Hobidasu新聞、2009年9月9日。</ref><ref>[http://www.dbj.jp/ja/topics/dbj_news/2009/html/0000003880.html 札沼線（愛稱：學園都市線）電氣化工程對於融資契約與北海道高速鐵道開發（株）結論] - 日本政策投融資銀行、2010年01月29日。</ref>JR北海道在2009年（平成21年）9月9日、公佈將在桑園－北海道醫療大學之間一段長28.9[[公里|公里]]的路段進行電氣化工程，之後、工程開始進行。整個電氣化工程主要由[[北海道高速鐵道開發|北海道高速鐵道開發]]負責。起工儀式在2009年12月10日開始，於2012年（平成24年）春季完成。

電氣化後、札幌車站 - 石狩當別車站之間的所要時間由46分鐘縮短至39分鐘，三門列車投入後每天早上的擁塞率會由130%降至104%，空調化列車會由63%提升至100%。

在工程進行中已預備42輛新車，其中新車包括[[JR北海道733系電力動車組|733系]]與[[JR北海道735系電力動車組|735系]]，兩種新車供本路線専用，2012年10月，全數電化列車已投入服務。
