'''九龍巴士73線'''是由[[九巴|九巴]]營運的一條[[香港|香港]][[新界|新界]]巴士路線，來往粉嶺[[華明邨|華明]]及[[大埔工業邨|大埔工業邨]]／[[大埔中心|大埔中心]]。

==歷史==
*1973年7月16日：本線投入服務，來往[[上水|上水]]及[[大埔墟|大埔墟]]。
*1977年3月1日：早上至黃昏班次延長至[[文錦渡|文錦渡]]。
*1980年9月16日：配合[[大埔工業邨|大埔工業邨]]發展，本線延長至該處。
*1982年12月24日：配合[[彩園邨|彩園邨]]入伙而遷往該處，上水至文錦渡的一段由73A線（現稱[[九龍巴士73K線|73K線]]）取代。
*2006年11月18日：本線延長至[[太平巴士總站|太平巴士總站]]（[[北區醫院|北區醫院]]附近），同時繞經[[華明邨|華明邨]]及[[雍盛苑|雍盛苑]]。但加價之餘，班次更疏落，乘客也難以預算候車時間，因為平均班次為21－29分鐘一班，無法像「廿分鐘一班／半小時一班」那樣容易預算，容易被乘客投訴班次失準，反而流失乘客。
*2008年12月7日：本線提早頭班車及延長尾班車開出時間，以配合[[九龍巴士70線|70線]]永久停駛<ref>[http://www.kmb.hk/tc/news/press/archives/news20081124257.html 九巴加強八達通轉乘優惠 配合70線停辦]，2008年11月24日，[[九巴|九巴]]</ref>。
*2011年8月8日：本線換入一輛空調巴士行走，是[[北區_(香港)|北區]]及[[大埔區|大埔區]]最後一條加入空調巴士的路線<ref>[http://www.kmb.hk/tc/news/press/archives/news201108041379.html 九巴73線提供空調巴士服務]，2011年8月4日，[[九巴|九巴]]</ref>。
*2011年9月19日：本線提升至全空調巴士服務。<ref>[http://www.kmb.hk/tc/news/press/archives/news201109171443.html 九巴73線提供全空調巴士服務]，2011年9月17日，[[九巴|九巴]]</ref>
*2013年8月24日：配合「[[北區區域性巴士路線重組|北區區域性巴士路線重組]]」第二階段實施，北區總站遷往[[華明巴士總站|華明]]，全程收費調整至$6.5，並改經[[廣福道|廣福道]]而不再途經[[南盛街|南盛街]]；同時於星期一至五19:00後，以及星期六、日及公眾假期全日改以[[大埔中心|大埔中心]]為終點站<ref>[http://www.kmb.hk/tc/news/press/archives/news201308201907.html 《北區巴士路線重組本周六實施第二階段》]，九巴新聞稿，2013-08-20</ref><ref>[http://www.kmb.hk/loadImage.php?page=1377135983_8840_0.jpg 73更改行車路線]，九巴乘客通告，2013-08-22（[http://www.peeep.us/055f61d0 存檔] {{Wayback|url=http://www.peeep.us/055f61d0# |date=20160308035616 }}）</ref><ref>[http://www.kmb.hk/loadImage.php?page=1377136078_7412_0.jpg 73短程特別車]，九巴乘客通告，2013-08-22（[http://www.peeep.us/940a144d 存檔] {{Wayback|url=http://www.peeep.us/940a144d# |date=20160308114953 }}）</ref>。
*2013年11月2日：更改粉嶺行車路線，繞經一鳴路，星期六早上部份班次以大埔工業邨為終點站，星期一至五延長由大埔工業邨尾班車開出時間。
*2014年10月4日：往華明方向抵和興路後，改經迴旋處、華明路（北行）、一鳴路、華明路（東行）、迴旋處、雷鳴路返回華明巴士總站，不再途經百和路<ref>[http://www.kmb.hk/loadImage.php?page=1411996587_8656_0.jpg]</ref>。
*2016年9月24日：來往粉嶺及大埔中心的班次更改行車路線如下：
**往大埔方向抵達南運路後，直接左轉安埔路前往[[大埔中心巴士總站|大埔中心巴士總站]]<ref>[http://kmb.hk/loadImage.php?page=1474519551_3650_0.jpg 九巴乘客通告] {{webarchive|url=https://web.archive.org/web/20160924091117/http://www.kmb.hk/loadImage.php?page=1474519551_3650_0.jpg |date=2016-09-24 }}</ref>。
**往粉嶺方向抵達安埔路後，右轉往南運路南行線，後返回原有路線<ref>[http://www.kmb.hk/loadImage.php?page=1474603056_3020_0.jpg 九巴乘客通告]</ref>。

===八達通轉乘優惠===
乘客以同一張八達通卡於登上首程巴士後150分鐘內轉乘以下路線，次程可獲優惠。詳情如下：

;73 ←→ 71K，次程可獲$4.2折扣優惠
*'''73往大埔''' → 71K任何方向。
*71K任何方向 → '''73往華明'''。

;73←→72、74A
*'''73往大埔''' → 72或74A往九龍，回贈首程車資。
*72或74A往太和 → '''73往華明'''，次程車資全免。
**乘客於同一日內來回程均使用上述轉乘，回程可額外獲$2.4（74A線）／$2.9（72線）折扣優惠。

;73←→271／271P
*'''73往大埔''' → 271／271P往尖沙咀，次程可獲$6.1折扣優惠。
*271往富亨 → '''73往華明'''，次程可獲$4.7折扣優惠。
**乘客於同一日內來回程均使用上述轉乘，回程可額外獲$1.1折扣優惠。

;73←→74X／75X
*'''73往大埔''' → 74X／75X往九龍，回贈首程車資。
*74X／75X往大埔 → '''73往華明'''，次程車資全免。

;73←→277X
*'''73往華明''' → 277X往[[藍田站|藍田站]]，回贈首程車資。
*277X往聯和墟 → '''73往大埔'''，次程車資全免。

;73←→70K
*'''73往華明''' → 70K任何方向，次程車資全免。
*70K任何方向 → '''73往大埔'''，回贈首程車資。

;73←→273A
*'''73往華明''' → 273A任何方向，次程車資全免。
*273A任何方向 → '''73往大埔'''，回贈首程車資。

==用車==
本線間歇亦有晚年「長牛」[[丹拿CVG6巴士|丹拿E型]]行走，並於八十年代初以「雞車」[[利蘭勝利二型巴士|利蘭勝利二型]]為掛牌車。在勝利二型逐漸淡出此路線後，先後以[[利蘭奧林比安|利蘭奧林比安]]9.7米（BL）、[[都城嘉慕威曼都城巴士|都城嘉慕都城型]]11米（S3M）為主帥，期間更出現曾創下九巴超速紀錄的一代車皇[[平治O305|平治O305]]，於2002年底退役前仍然經常出沒於此路線，成為不少巴士迷追捕的目標。

現時本線及73A共有2輛[[富豪超級奧林比安|富豪超級奧林比安]]12米及9部[[亞歷山大丹尼士Enviro_500|亞歷山大丹尼士Enviro 500]]12米，均屬[[九巴上水車廠|上水車廠]]。<ref>[http://1005.idv.hk/index.php?page=01&search=sch&word=73]</ref>

==客流量及未來==
現時主要的競爭對手為[[港鐵|港鐵]][[東鐵綫|東鐵綫]]及來往上水至[[大埔_(香港)|大埔]]的[[香港小巴|紅色小巴]]，小巴班次頻密，本線班次顯得相當疏落，因此本線客量一直處於偏低水平，就算在平日繁忙時間都甚少出現客滿的情況，九巴已對本線採取放棄態度。

隨著[[九龍巴士70線|70線]]於2008年12月7日起永久停駛，九巴由當日起增闢本線與[[九龍巴士72線|72線]]、[[九龍巴士74A線|74A線]]及[[九龍巴士271線|271線]]的轉乘優惠以取代70線，並加密班次，本線有望走出困局。但另一方面，[[清河邨|清河邨]]於2008年下半年全面入-{伙}-後，於2008年9月26日政府憲報中，公開招標由清河邨經[[北區醫院|北區醫院]]及[[粉嶺|粉嶺]]南至[[大埔那打素醫院|大埔那打素醫院]]的專線小巴路線<ref>[http://www.gld.gov.hk/cgi-bin/gld/egazette/gazettefiles.cgi?lang=c&year=2008&month=9&day=26&vol=12&no=39&gn=6621&header=1∂=0&df=1&nt=gn&newfile=1&acurrentpage=12&agree=1&gaz_type=mg 第6621號公告　九龍及新界的公共小型巴士（專線）路線]，2008年9月26日，[[香港政府憲報|香港政府憲報]]</ref>，編號為[[新界區專線小巴502線|502]]，並已於2009年6月17日投入服務，以較頻密的班次及全線空調服務與本線競爭，本線的客量因此再度出現流失。

[[運輸署|運輸署]]於2003-2004年路線發展計劃中建議永久停駛本線，以專線小巴取代，但受到居民及[[北區區議會|北區區議會]]反對而攔置。而近年路線發展計劃中，運輸署建議九巴加入空調巴士行駛，但本線於2011年8月8日才加入空調巴士行走，成為[[北區_(香港)|北區]]及[[大埔區|大埔區]]最後一條加入空調巴士的路線。

隨著巴士服務空調化是大勢所趨，九巴加快將巴士路線轉為全空調服務的速度，於2011年9月19日起，一口氣將7輛普通巴士換成空調巴士行走，提升至全線空調服務，由於另一條北區路線[[九龍巴士73A線|73A]]亦於同日起改為全空調服務，象徵以北區作總站的所有專利巴士路線已經改為全空調服務，亦令到大窩西支路及整個北區再沒有普通巴士行走。

特區行政長官梁振英於《2013 -2014年度施政報告》中，決定引入「區域性巴士路線重組」，北區成為首個採用此模式重組巴士路線的地區，並於2013年計劃及實行（即《2013-2014年北區巴士路線發展計劃》），鑑於本線行車路線迂迴、沿途上落客站多（由上水至大埔沿途經過50多個巴士站）、行車速度緩慢，再加上與類近九巴巴士路線／其他交通工具平行，遂導致乘客抗拒使用本線，以致乘客量及使用率均偏低，因此在該項區域性重組計劃內，建議本線北區總站縮短至華明，並增加70K及273A的八達通轉乘優惠替代本線於北區區內的服務，另外增加接駁大埔、北區來往市區路線的轉乘優惠，以替代客量偏低的70X線；同時星期一至五晚間、星期六及假日大埔區總站縮短至大埔中心，並降低全程車費。整個重組建議最終於2013年7月8日於北區區議會交通及運輸委員會會議上獲得通過，本線隨即於2013年8月24日起，配合北區「區域性巴士路線重組」第二階段實施，實施以上改動，往大埔方向改經一鳴路而不再經華明路（雍盛苑），於大埔工業邨改為來回程取道大貴街，與同日縮短路線的73A線，於大窩西支路提供10-15分鐘一班的聯合班次來往太和及大埔墟。

自從本線的北區總站縮短至華明後，客量開始有所上升。
