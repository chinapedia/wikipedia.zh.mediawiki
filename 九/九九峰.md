'''九九峰'''，亦稱'''九十九尖峰'''，是[[臺灣|臺灣]][[中臺灣|中部地區]]擁有火炎山特殊地理景觀的地區，諸峰獨立，鮮有稜脈相連，九九峰全境最高峰為海拔779.4公尺（[[921地震|921地震]]前高度777.5公尺），與[[三義鄉_(台灣)|三義]][[火炎山_(苗栗縣)|火炎山]]、[[六龜區|六龜]][[十八羅漢山|十八羅漢山]]並稱為「台灣三大火炎山地形」{{r|Caotun}}。九九峰地質含有[[赭土|赭土]]，赭土本身具有[[鐵|鐵]]、[[鋁|鋁]]等[[氧化物|氧化物]]成份，在晨間、黃昏之時，遠觀九九峰，宛如火紅般的山色在燃燒，故得火炎山之美譽{{r|Caotun}}。

九九峰於921地震之後，植被隨之土石崩落，使山頭失去植被而裸露土壤。由[[國道六號_(中華民國)|國道6號]]沿[[烏溪|烏溪]]向東橫越九九峰南側，規劃草屯服務區可供旅客觀賞九九峰。九九峰對於草屯是別具意義的指標，因此炎峰國小前校長簡金間對九九峰比喻：「尖峰宛如筆架，能鍾聚靈氣，培才毓秀。」取「炎峰毓秀」命名[[南投縣草屯鎮炎峰國民小學|炎峰國小]]，是象徵[[草屯鎮|草屯]]第一名景{{r|Yienfong}}。

== 自然保留區 ==
經歷921地震後，[[農委會|農委會]]依照[[文化資產保存法|文化資產保存法]]第49條細則第72條規定，以地震造成崩坍、斷崖等特殊地景為保育對象，因此由[[林務局|林務局]]於1999年10月提出九九峰嚴重崩坍區域劃為[[自然保留區|自然保留區]]，同年12月完成規劃報告書，將一千兩百[[公頃|公頃]]所屬嚴重崩坍地且[[人類|人類]]不易到達之區劃設，供學術研究與戶外教學之用，翌年5月22日正式成立「'''九九峰自然保留區'''」{{r|Hong-Zhong}}{{r|Jyun-Hong}}。

九九峰自然保留區設立後，已明確地將九九峰範圍納定，位於[[南投縣|南投縣]][[草屯鎮|草屯鎮]]、[[國姓鄉|國姓鄉]]及[[臺中市|臺中市]][[霧峰區|霧峰區]]、[[太平區_(臺中市)|太平區]]，面積達1198.4466[[公頃|公頃]]，屬於台灣[[文化資產保存法|文化資產保存法]]中的「自然地景」，由[[農委會林務局|農委會林務局]][[埔里鎮|埔里]]事業處第8～13與15～20林班管轄{{r|Zhou-Chin}}。

== 成因 ==
受到古烏溪的上游帶來的岩塊、土壤等沉積物質，經由搬運與堆積的作用，在現今九九峰位置形成沖積扇，加上長時間堆積、擠壓，將地層內的礫石彼此膠結，逐漸地形成卵礫石岩層，這便是火炎山層地質構造，但此時仍是處於埋沒在地表之下，經過造山運動後，這些卵礫石構成的岩層得以露出地表{{r|Hong-Yu}}。九九峰除了地質構造是火炎山層，還挾帶著[[香山砂岩|香山砂岩]]與[[頁岩|頁岩]]，後來長時間經歷雨水沖刷，將脆弱的頁岩沖蝕流失，成了現在看到黃山群起，諸峰挺立的火炎山景觀，九九峰與鄰近[[頭嵙山|頭嵙山]]同屬頭嵙山層地質{{r|Caotun}}。

== 地理環境 ==
[[烏溪|烏溪]]自九九峰東側、南側流經，境內由東向西有仙人指坑溪、田尾坑溪、油車坑溪、乾溪等四條河川貫穿{{r|Hong-Yu}}{{r|Jin-Song}}。依[[中央氣象局|中央氣象局]]雙冬測候站，自1995年－2002年記錄顯示雨季在5月～8月，此時才會顯現大河之勢，相對地在10月～翌年1月是為乾季，這期間大河變成細流，當地農民為了適應此處環境而發展出鑿井灌溉，故可見得許多交錯水管穿越烏溪上方{{r|Jyun-Hong}}。

九九峰河谷景觀相當陡峭、開闊，平均坡度皆有60～70度，最陡達到85度峭壁，因此區地質屬於卵礫石岩層，受到風化、雨襲是顯得地表狀況不穩定，許多河谷都呈現V字形侵蝕溝，如此山景美色，卻無法種植耕地，在[[地理學|地理學]]喻為「[[惡地|惡地]]」，與[[田寮區|田寮月世界]]、[[利吉月世界|利吉月世界]]屬不同地質構造{{r|Jyun-Hong}}{{r|Hong-Yu}}{{r|Hong-Yu}}。

== 地震影響 ==
九九峰周邊有雙冬斷層、龜蒲斷層，但並未由這兩斷層所觸發地震，而是受到車籠埔斷層引發921地震而使表土裸露。在自然保留區境內崩坍面積總達729.36公頃，其中以第15林班崩坍區域最為廣泛，相對位居國姓鄉第20林班，受到地震影響並不大，是全部崩坍面積最小的林班地{{r|Jin-Song}}。實際上，多數崩坍地分布於山頭，平均海拔皆在400公尺以上，而且所造成地形景觀宛如一根根玉筍佇立，因此地震之後，對於有立即危害之區域進行整治、植栽，避免[[土石流|土石流]]危害周邊民居與損害範圍擴大，在九九峰境內的河川可見得有箱涵工法或蛇籠築壩整治{{r|Jin-Song}}{{r|Hong-Zhong}}。
