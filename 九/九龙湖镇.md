'''九龙湖镇'''，是[[中国|中国]][[浙江省|浙江省]][[宁波市|宁波市]][[镇海区_(宁波市)|镇海区]]的一个[[建制镇|建制镇]]。东临[[澥浦镇|澥浦镇]]，南面与[[骆驼街道|骆驼街道]]相接，西面与[[江北区_(宁波市)|江北区]][[慈城镇|慈城镇]]、[[庄桥街道|庄桥街道]]相连，西面与[[骆驼街道|骆驼街道]]接壤，北面与[[慈溪市|慈溪市]]相接。总面积65.3平方公里，人口6.49万，是镇海区面积最大的乡级行政区。[[街道办事处|街道办事处]]位于九龙东路1号。

== 沿革 ==
九龙湖镇原名河头，古名鸿落河头。1983年恢复为河头乡，2001年，根据所辖地旅游景点九龙湖（原名[[十字路水库|十字路水库]]）更名为九龙湖镇。

== 下辖 ==
目前，九龙湖镇辖有思源社区1个社区。辖有九龙湖、田杨陈、杜夹岙、西经堂、河头、西河、长石、长宏、汶溪、中心、田顾共11个行政村。

== 交通 ==
九龙大道穿过九龙湖镇，连接[[329国道|329国道]]。

== 旅游 ==
宁波新十景之一的[[十字路水库|九龙湖]]是九龙湖镇最重要的旅游景点。
