'''九份老街'''主要範圍以聚集在基山街、豎崎路及輕便路等街道為主，是全台灣最熱門的觀光景點之一，全年觀光客絡繹不絕。若以客源分析，平日以來自[[日本|日本]]、[[韓國|韓國]]等地的外籍觀光客以及中國大陸、香港、澳門的遊客和本國遊客各占一半，假日則以國民旅遊為主，是[[北台灣|北台灣]]國際級的觀光景點。商圈定位鮮明，集歷史、天然景致、美食文化於一爐。

==商圈特色==
[[File:Jiufen_Panolama.JPG|thumb]]
人文歷史景觀如台陽礦業事務所、彭園、五番坑口 、[[昇平戲院|昇平戲院]]等，而位於九份上，往下可俯視[[基隆嶼|基隆嶼]]、[[八斗子|八斗子]]、[[番仔澳|番仔澳]]、[[鼻頭角|鼻頭角]]等地的海景，往上可眺望[[雞籠山|雞籠山]]景。[[九份|九份]]日夜氛圍不同，於白日可俯瞰海景，感受九份老街旺盛人氣，於夜晚一切歸於寧靜，靜靜坐在[[茶樓|茶樓]]間，細品九份寧靜氛圍，往外看，僅剩盞盞漁火於海面上。

== 基山街 ==
[[File:九份基山街賴阿婆芋圓店.JPG|thumb]]
昔時[[九份|九份]]一帶最繁華熱鬧的商業街，由於採礦全日無休，因此礦工也三班制輪班，因此基山街上徹夜燈火通明，當時甚至是全台消費能力最高的商業區，至今，基山街仍維持舊觀，是九份老街最主要的商店街。

== 豎崎路 ==
九份最早的交通要道，全線由石頭砌成，約三、四百階石梯，依山勢興築，向上貫穿九份地區，延著石級而上，與重要道路汽車路、輕便路及基山街的路口，形成九份最熱鬧的區域。
[[File:九份徛崎路(豎崎路)佮輕便路路口.JPG|right]]

== 輕便路 ==
[[日治時期|日治時期]]，於1931年在此興築[[輕便車|輕便車]]軌道，以加強九份到瑞芳間的貨物運輸。一直到1954年廢道拆除，輕便路的路名便是由此而來。
