'''九世之乱'''是[[商朝|商朝]]自商王[[中丁|中丁]]后，连续发生王位纷争，又屡次迁都，使王朝中衰、诸侯离叛的事件。这一动乱历经中丁、[[外壬|外壬]]、[[河亶甲|河亶甲]]、[[祖乙|祖乙]]、[[祖辛|祖辛]]、[[沃甲|沃甲]]、[[祖丁|祖丁]]、[[南庚|南庚]]、[[阳甲|阳甲]]九王，故名“九世之乱”。

九世之乱延续近百年，直到[[盘庚迁殷|盘庚迁殷]]后才最终结束。

== 过程 ==
中丁是商王[[大戊|大戊]]的儿子，在位时将商朝首都自[[亳|亳]]{{NoteTag|关于商朝亳都的具体位置，共计有六种说法，分别为西亳说、南亳说、北亳说、杜亳说、垣亳说及郑亳说，其中以西亳说和郑亳说被广泛接受。西亳说认为亳都位于河南省偃师市，而郑亳说则认为亳都位于河南省郑州市的商城遗址。}}迁至[[嚣|嚣]]{{NoteTag|《史记·卷三·殷本纪》称隞。}}（今[[河南省|河南省]][[郑州市|郑州市]]附近，一说在今河南省[[荥阳市|荥阳市]]东北），又出兵攻打[[蓝夷|蓝夷]]。<ref>《古本竹书纪年·殷纪》：仲丁即位，元年，自亳迁于嚣。仲丁即位，征于蓝夷。</ref>中丁死后，他的兄弟们凭借个人的势力争夺王位。中丁的弟弟外壬在中丁死后继位，造成商朝一百多年王位继承的混乱。<ref name="殷本纪"/>

外壬在位时，商朝国力开始有衰落的迹象，邳和侁的部落发生叛乱。<ref>《今本竹书纪年·殷纪》：邳人、侁人叛。</ref>外壬死后，他的弟弟河亶甲继位。河亶甲是商王大戊的儿子、商王仲丁、外壬的弟弟。河亶甲在位期间多次发动对外战争，致使商朝国力再度衰落。<ref>《史记·卷三·殷本纪》：帝外壬崩，弟河亶甲立，是为帝河亶甲。河亶甲时，殷复衰。</ref>他曾迁都于[[相|相]]（今河南省[[内黄县|内黄县]]境内），出兵征伐蓝夷和[[班方|班方]]，<ref>《古本竹书纪年·殷纪》：河亶甲整即位，自嚣迁于相。征蓝夷，再征班方。</ref>并先后派[[彭伯_(商朝人物)|彭伯]]、[[韦伯_(商朝人物)|韦伯]]征服背叛的邳和侁部落。<ref>《今本竹书纪年·殷纪》：（河亶甲）三年，彭伯克邳。五年，侁人入于班方。彭伯、韋伯伐班方，侁人來賓。</ref>河亶甲死后，他的儿子祖乙继位。<ref>《史记·卷三·殷本纪》：河亶甲崩，子帝祖乙立。</ref>祖乙先将国都由相迁至[[耿|耿]]{{NoteTag|《史记·卷三·殷本纪》称邢。}}（今河南省[[温县|温县]]东），后因洪水毁坏又迁都于[[庇|庇]]（今[[山东省|山东省]][[定陶县|定陶县]]）。<ref>《今本竹书纪年·殷纪》：（祖乙）元年己巳，王即位，自相遷于耿…二年，自耿遷于庇。</ref><ref>《尚书书序》：祖乙圮于耿。《尚书正义》：圮，毁也；河水所毁曰圮。</ref>祖乙在位时重用贤臣[[巫贤|巫贤]]，使商朝国运再度中兴。<ref>《史记·卷三·殷本纪》：帝祖乙立，殷复兴。巫贤任职。</ref>祖乙死后由他的儿子祖辛继位。祖辛死后由他的弟弟沃甲继位。沃甲死后由他的侄子、祖辛的儿子祖丁继位。祖丁死后由他的堂弟、商王沃甲的儿子南庚继位，<ref>《史记·卷三·殷本纪》：祖乙崩，子帝祖辛立。帝祖辛崩，弟沃甲立，是为帝沃甲。帝沃甲崩，立沃甲兄祖辛之子祖丁，是为帝祖丁。帝祖丁崩，立弟沃甲之子南庚，是为帝南庚。</ref>南庚在位时将国都由庇迁至[[奄|奄]]（今山东省[[曲阜市|曲阜市]]）。<ref>《古本竹书纪年·殷纪》：南庚更自庇迁于奄。</ref>南庚死后由祖丁的儿子阳甲继位。阳甲在位时，商朝国力再度衰落。<ref>《史记·卷三·殷本纪》：帝南庚崩，立帝祖丁之子阳甲，是为帝阳甲。帝阳甲之时，殷衰。</ref>

== 影响 ==
九世之乱使商朝[[兄终弟及|兄终弟及]]与[[父死子继|父死子继]]相结合的王位继承制度遭到破坏，商朝统治力量遭到严重削弱，无力再顾及四方诸侯、[[方国|方国]]，诸侯不再向商朝朝见纳贡。<ref name="殷本纪">《史记·卷三·殷本纪》：自中丁以来，废适而更立诸弟子，弟子或争相代立，比九世乱，于是诸侯莫朝。</ref>少数民族趁机发动叛乱，日益威胁着商朝的统治。<ref>《后汉书·卷八十七·西羌传》：及殷室中衰，诸夷皆叛。</ref>混乱使商朝贵族内部矛盾更为激化，王室贵族或倨傲放肆、或淫逸奢侈，离心力日增。<ref>《尚书正义·卷九·盘庚上》：郑玄云：祖乙居耿后，奢侈逾礼，土地迫近山川，尝圮焉。至阳甲立，盘庚为之臣，乃谋徙居汤旧都。又序注云：民居耿久，奢淫成俗，故不乐徙。</ref>

== 后续 ==
阳甲的弟弟的[[盘庚|盘庚]]继位后，迁都至[[殷|殷]]（今河南省[[安阳市|安阳市]]西北[[殷都区|殷都区]][[小屯村|小屯村]]），<ref>《古本竹书纪年·殷纪》：盘庚旬自奄迁于北蒙，曰殷。</ref>使商朝不长期定居、到处迁移的历史从此结束。商朝国力得到恢复，人民生活开始安定，诸侯又来朝见。<ref>《史记·卷三·殷本纪》：（盘庚）乃遂涉河南，治亳，行汤之政，然后百姓由宁，殷道复兴。诸侯来朝，以其遵成汤之德也。</ref>据《[[竹书纪年|竹书纪年]]》记载，直到[[商纣王|商纣王]]亡国，整整二百七十三年商朝再未迁都。<ref>《古本竹书纪年·殷纪》：自盘庚徙殷，至纣之灭，两百七十三年，更不徙都。</ref>

== 质疑 ==
九世之乱的说法仅见于《[[史记|史记]]·卷三·殷本纪》，[[梁玉绳|梁玉绳]]所著《[[史记质疑|史记质疑]]》认为九世之乱期间有[[祖乙中兴|祖乙中兴]]，而中丁、外壬、河亶甲三代符合商朝兄终弟及的传统，中丁之前的[[雍己|雍己]]、大戊，阳甲之后的盘庚、[[小辛|小辛]]、[[小乙_(商朝)|小乙]]也是如此继承，所以九世之乱的说法存在质疑。<ref>{{cite book|author=梁玉绳|coauthors= |title=《史记志疑》第57页|year=1981年4月|publisher=中华书局|location= |isbn=}}</ref>
