《'''凝望 — 君看雙眼色，不語似無愁'''》一書收錄了已故聾人教師[[李菁_(聽障教師)|李菁]]過往在刊物及網上日誌，講述有關「聽障」的文章。

==成書背景==
2008年3月，[[失聰|失聰]]女教師李菁逝世。[[香港嶺南大學|香港嶺南大學]]為了喚起社會對聾人及弱聽人士的關注<ref>{{cite web |language=zh-hk |url=http://www.takungpao.com.hk/news/08/09/05/JX-956767.htm |title=嶺大文集籲關注聽障者 |publisher=[[大公報|大公報]] |date=2008-09-05 |accessdate=2012-02-20 }}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，成立「[[李菁紀念基金|李菁紀念基金]]<ref>{{cite web |language=zh-hk |url=http://paper.wenweipo.com/2008/09/10/ED0809100007.htm | title=嶺南大學 成立「李菁紀念基金」 |publisher=[[文匯報_(香港)|文匯報 (香港)]] |date= 2008-09-10|accessdate=2012-02-20}}</ref>」，並決定將李菁生前的生活照及文章，結集成《凝望 — 君看雙眼色，不語似無愁》一書<ref>{{cite web |language=zh-hk |url=http://news.sina.com/oth/mingpao/301-101-101-105/2008-09-04/16003274518.html | title=嶺大為自殺聽障教師出文集 |publisher=[[明報|明報]] |date= 2008-09-04|accessdate=2012-02-20}}</ref>。

這本書集不會公開發售，只供有興趣的讀者聯絡嶺南大學索取。此外，這本書集亦推出了網上版本，並上載於[[慈善機構|慈善機構]]「[[龍耳|龍耳]]」的網頁，供有興趣的讀者下載。
