'''旺宏電子股份有限公司'''（{{lang-en|Macronix International Co., Ltd}}，英文簡寫「Macronix」；{{twse|2337}}）是提供[[非揮發性記憶體|非揮發性記憶體]]整合元件解決方案（IDM）製造廠商，其總部設於[[臺灣|臺灣]][[新竹科學園區|新竹科學園區]]，並在[[美國|美國]]、[[歐洲|歐洲]]、[[日本|日本]]、[[韓國|韓國]]、[[新加坡|新加坡]]、中國大陸[[蘇州|蘇州]]設有營業據點。目前在唯讀記憶體（ROM）市佔率為世界第一，（NOR Flash）為全球第一，更致力於擴大NAND Flash市場佔有率。

==公司簡介== 
1989年12月，旺宏電子創辦人暨現任董事長[[吳敏求|吳敏求]]號召數十位[[美國|美國]][[矽谷|矽谷]]半導體設計精英團隊集體返臺，與臺灣高科技人才共同創立旺宏電子<ref>{{Cite web |url= http://www.bnext.com.tw/article/view/id/8346 |title= 快閃 後PC的天空 |author= |date= 2000年5月1日|publisher= 數位時代|language= |accessdate= |quote= }}</ref>，是[[臺灣|臺灣]]第一家以第三類股上市的高科技公司，同時也是第一家在美國[[NASDAQ|NASDAQ]]上市的臺灣企業。

從創辦至今，不同於臺灣其他[[半導體|半導體]]製造公司以[[代工|代工]]為主體，旺宏電子堅持自有技術，每年約投資營業額的20%於研發工作，擁有逾7,100項的國際關鍵技術及專利等智慧財產權。根據[[美國|美國]]權威調研諮詢機構「專利委員會」(The Patent Board)2011年公佈的半導體企業專利排名結果，旺宏電子在專利實力的評比列名全球第18名，為臺灣企業的第一名<ref>{{Cite web |url= http://www.nownews.com/n/2011/11/01/462904 |title= 旺宏專利實力台灣第一、全球排名十八　台積電全球排第23 |author= 楊伶雯|date= 2011年11月1日|publisher= NOWnews今日新聞|language= |accessdate= |quote= }}</ref>。　

目前旺宏擁有一座12吋晶圓廠(晶圓五廠)、一座8吋晶圓廠(晶圓二廠)及一座6吋晶圓廠(晶圓一廠)，晶圓五廠及晶圓二廠主要生產旺宏自有非揮發性記憶體產品，晶圓一廠則以利基型邏輯產品的晶圓代工業務為主。

==產品線==
旺宏專注於非揮發性記憶體，產品以ROM(唯讀記憶體)、NOR Flash(NOR快閃記憶體)與NAND Flash(NAND快閃記憶體)為主。

===ROM(唯讀記憶體)===
ROM的特點為資料儲存後無法修改，主要優勢為記憶體容量大且成本低，其應用集中於電子遊戲卡匣、電子玩具與印表機等，產業已趨於特定應用導向。旺宏全球市佔率超過八成，，排名全球第一。

===NOR Flash(NOR型快閃記憶體)===
NOR Flash可以重複讀寫，廣泛應用於消費性、通訊、資訊及手機等電子產品，近年來的規格發展以序列式介面(Serial Interface)為主流。旺宏全球市占率約24%，全球第一。旺宏電子從2009年開始決定進入車用電子記憶體元件供應商的領域，緊鑼密鼓地通過一連串包括TS16949與AEC-Q100的認證，並逐漸開展市場，車用IC出貨量已達1億顆，搶當車用NOR龍頭。 <ref>[http://www.digitimes.com.tw/tw/dt/n/shwnws.asp?cnlid=13&id=0000470016_KIV1Z92I0DFVFV7EPV2L6#ixzz4VdGqzngH]。</ref>，<ref>[http://udn.com/news/story/7253/1864480]。</ref>，<ref>[http://www.appledaily.com.tw/realtimenews/article/new/20161115/988836/]。</ref>

===NAND Flash(NAND型快閃記憶體)===
與NOR Flash相較，NAND Flash的容量較大，改寫速度較快，主要應用在大量資料的儲存。旺宏在SLC NAND的市場占有率為全球第三，旺宏也是第一家可以供給完全符合AEC-Q100車用規範的SLC NAND記憶體的廠商，目前全球主要Tier 1汽車電子供應商已經是旺宏主要的客戶。[https://www.moneydj.com/KMDJ/wiki/WikiViewer.aspx?Title=NAND%20FLASH <ref>1</ref>][http://www.chinatimes.com/realtimenews/20180207003353-260410 <ref>2</ref>][https://www.digitimes.com.tw/iot/article.asp?cat=130&id=0000520407_MJG56A0ULQO28387RK9LB <ref>3</ref>]

==發展簡歷==
旺宏電子長期致力於開發自有技術與產品，歷年發表的技術論文，持續入選IEDM及ISSCC等多項國際學術會議。旺宏目前擁有超過7,100項國際關鍵技術及專利等智慧財產權，並與全球高科技業界領導廠商成立技術合作聯盟，共同進行次世代記憶體先驅技術的研究。

除了專注於提昇自有研發能力與製造技術，旺宏堅持嚴謹的公司治理，榮登台灣證券交易所第一屆公司治理評鑑排名前5%企業，並持續深化環境保護與社會參與，以具體行動體現企業社會責任。旺宏於2008年成為竹科首家通過「SA 8000企業社會責任管理系統」驗證的半導體公司，並持續獲得[[中華民國勞動部|中華民國勞動部]]「國家工安獎」、[[天下雜誌|天下雜誌]]「天下企業公民獎」及[[遠見雜誌|遠見雜誌]]「企業社會責任獎」等肯定。

==旺宏教育基金會==
為提昇臺灣知識經濟價值，培育科技人才，關懷兒童與青少年身心發展，提昇社會祥和與生活品質，旺宏電子於2001年捐贈成立了旺宏教育基金會。每年定期舉辦針對大專院校學生的旺宏金矽獎，與針對高中生的旺宏科學獎等活動，以期提昇臺灣學生的研發創新能力。

旺宏教育基金會長期深耕科普教育，致力提昇國內科技創新，2013年更榮獲教育部績優教育基金會評鑑肯定，評選為最佳成績之「特優」<ref>{{Cite web |url= http://edn.udn.com/news/view.jsp?aid=688671&cid=47 |title= 旺宏教育基金會獲教部特優評鑑 |author=  |date= 2013年12月11日|publisher= 經濟日報|language= |accessdate= |quote= }}</ref>。

===旺宏金矽獎===
有鑒於高科技人才養成不易，並為鼓勵大學院校學生創新研究的精神，促進產業與學術領域良性交流互動，旺宏電子於2000年創辦第一屆「旺宏金矽獎－半導體設計及應用大賽」，持續獲得各大專院校教授及學生的熱烈迴響，參賽學生已將旺宏金矽獎視為電機、電子相關領域的重要指標性獎項，與全國好手一較高下，以評量自己所學。

===旺宏科學獎===
旺宏教育基金會為啟發高中職學生對自然科學與應用科學的興趣，並鼓勵其探索科學的精神與創造發明的潛力，於2002年舉辦第一屆「旺宏科學獎」，提供一個與其他學校同學良性競爭的環境及成果發表的園地。為了延續「旺宏科學獎」的精神，基金會於2004年9月為每屆獲獎同學成立「旺宏科學獎聯誼會」，舉辦各項活動與年會，加強交流與成長。
