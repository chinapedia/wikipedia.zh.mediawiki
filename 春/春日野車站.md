'''春日野站'''（{{jpn|j=はるひ野駅|hg=はるひのえき|rm=Haruhino eki|f=y}}）是一個位於[[神奈川縣|神奈川縣]][[川崎市|川崎市]][[麻生區|麻生區]]{{link-ja|黑川 (川崎市)#黑川下·春日野地區|黒川 (川崎市)#黒川下・はるひ野地区|春日野}}五丁目，屬於[[小田急電鐵|小田急電鐵]][[多摩線|多摩線]]的[[鐵路車站|鐵路車站]]。[[車站編號|車站編號]]是'''OT 04'''<ref name="春日野站導覽"/>。開業時導入通用化設計<ref name="k-press20041210"/>與風力、太陽能混合式發電設備<ref name="RF527p192"/>。兩月台之間的天橋上有大型屋頂覆蓋為本站外觀特色<ref name="Shinoda"/>，並入選2005年鐵道建築協會獎<ref name="ARA"/>。

== 歷史 ==
多摩線建設時已有本站周邊的開發計畫，但小田急、[[京王電鐵|京王]]未能與地主達成共識，造成開發計畫難以進行，導致1977年（昭和52年）兩公司宣布終止開發<ref name="開發p29"/>。現在春日野站的位置在多摩線計畫時便是設站地點之一<ref name="開發p33"/>。1991年（平成3年）3月，車站周邊地區開發計畫核准，翌年動工<ref name="開發p29"/>。歷經多次計畫變更後<ref name="開發p31"/><ref name="開發p32"/>，小田急於2000年（[[平成|平成]]12年）12月宣布設站計畫<ref name="開發p33"/><ref group="註釋">在如同春日野開發正史的[[#開發|《春日野開發與地域紀錄》 p33]]中記載「包含川崎市或當地地主都未想到（設站）」，意指小田急是在沒有機構、地主的委託下獨立建設新站，但申請核准時的報導資料為「受到都市基盤整備公團委託設置新站」，兩者說法不同。</ref>。在舉行地方說明會後<ref name="開發p26"/>，2003年（平成15年）2月申請設站許可<ref group="註釋">實際上是小田急電鐵提出事業基本計劃變更的許可申請。</ref>，2003年11月動工<ref name="開發p33"/>。本站位於黑川特定土地區劃整理事業地中央（之後的「春日野」）<ref group="註釋">[[#開發|《春日野開發與地域紀錄》 p29]]為初期的土地利用計畫圖，[[#開發|《春日野開發與地域紀錄》 p33]]為最後的土地利用計畫圖，本站實際建設地點並非原先的保留用地，這是為了保存小田急永山方向北側京王相模原線與小田急多摩線之間的三澤川支流水源地（[[#開發|《春日野開發與地域紀錄》 p35]]），而將車站改到現在的位置。</ref>，由小田急設計顧問{{link-ja|篠田義男|篠田義男}}建築研究所設計，小田急建設施工，總工程費用約25億日圓，面積5,300平方公尺<ref name="Shinoda"/><ref name="k-press20041210"/>。2004年（平成16年）12月11日開業，是小田急第70座車站<ref name="RF527p192"/><ref name="RP829p212"/>。依據[[都市再生機構|都市再生機構]]、[[川崎市|川崎市]]等的備忘錄，一開始未規劃站前廣場<ref name="k-press20041210"/>，開業當時也未決定都市計畫<ref name="都市計畫公告"/>。在都市計畫決定後，2008年（平成20年）11月完成南口站前廣場<ref name="站前完成"/>。廣場規劃時，由於周邊巴士業者未有停靠春日野站的計畫，因此設計上未考慮大型巴士的容納空間<ref>{{cite web

== 站名由來 ==
站名源自於都市基盤整備公團（之後的[[都市再生機構|都市再生機構]]）開發的住宅區「'''黑川春日野'''」（くろかわはるひ野）<ref name="開發p31"/>。為了避免因「黑川」造成與其他車站之間的混淆，選擇以「春日野」作為站名<ref name="開發p34"/>。2006年3月13日，本地實施住居表示，都市再生機構開發地區的町名也改為春日野<ref name="開發p27"/>。至2009年為止，本站是小田急車站中唯一使用平假名的車站<ref name="2009-167"/>。

地名「春日野」本身是2001年（平成13年）4月地主與都市基盤整備公團兩方從「春日野」、「水木野」、「萬葉之丘」三案中選出<ref>『黒川地区「愛称」選考について』 平成13年4月22日</ref>。「春日野」的「春」取自對新市街成長的期待，「日」代表本地面東向陽的斜坡面，「野」意為廣大土地<ref>「黒川特定土地区画整理区域の新町界・町名(案)について」 新町名地番検討会 2004年6月発行</ref>。為維持其獨特性，「春日野」由都市再生機構登記為商標<ref>{{cite web
|url = http://www1.ipdl.inpit.go.jp/syutsugan/TM_DETAIL_A.cgi?0&1&0&1&1&133640316471047204124828|title = 商標出願・登録情報検索
|accessdate = 2012-05-07
|publisher = 特許庁
}}</ref>。

== 車站構造 ==
[[File:Windmill_at_Haruhino.JPG|thumb]]
本站是當時2004年日本國內新站中少見的[[地面車站|地面車站]]<ref name="春日野站導覽"/><ref name="Shinoda"/>，可停靠10輛編組列車的2面2線[[側式月台|側式月台]]構造。兩月台間的天橋在考慮通風、採光的因素下增加大型屋頂<ref name="Shinoda"/><ref name="OER20031211"/>。剪票口分為北口與南口兩處，北口在新百合丘方向月台（以下稱上行月台），南口在唐木田方面向月台（以下稱下行月台）<ref name="春日野站導覽"/>。站務室位於南口，站員也僅配置於南口<ref name="春日野站導覽"/><ref name="RP829p212"/>。值勤職間為7:30～11:00與13:00～末班車，其他時間無人執勤<ref name="春日野站導覽"/><ref group="註釋">除本站外，多摩線全站皆為跨站式站房或高架車站。另外，本站動工的2003年當時全部車站都只有一處剪票口。多摩中心站東口剪票口在2004年11月設置，新百合丘站中央東剪票口在2008年3月設置，因此只有本站在開業時擁有2處剪票口。</ref>。本站設置由10座額定輸出功率400kW的風力發電機與8座額定輸出功率136kW的[[太陽能電池|太陽能電池]]組成的混合型發電系統，為車站設備電力來源之一<ref name="RF527p192"/><ref name="hybrid"/>，而上行月台屋頂作有部分綠化工程<ref name="k-press20041210"/><ref name="Shinoda"/>。兩月台之間的天橋設有電扶梯與電梯，上行月台電梯可通至北口<ref name="春日野站導覽"/>。下行月台有無障礙廁所，上行月台為一般廁所<ref>{{cite web
|url = http://www.odakyu.jp/station/facilities/
|title = 各駅設備一覧
|publisher = 小田急電鉄
|accessdate = 2012-09-02
}}</ref><ref name="春日野站導覽"/>。上行月台的候車室配備空調<ref name="k-press20041210"/>。2010年度於站內設置列車資訊顯示器<ref>{{cite web
|url = http://www.odakyu.jp/csr/safety_report/2011/8/
|accessdate = 2012-09-06
|title = 駅や車内、ホームページなどで列車運行に関する情報提供を行っています。
|publisher = 小田急電鉄
}}</ref>。本站在2005年入選鐵道建築協會獎<ref name="ARA"/>。

== 車站周邊 ==
=== 周邊設施 ===
1992年（平成4年）11月開發以前，本站周邊多為未開發之山林<ref name="開發p30"/>。此地的開發構想起於1973年（[[昭和|昭和]]48年）<ref name="開發p29"/>，但由於遲遲無法與地主達成共識<ref name="開發p29"/>，重新規劃後也遭到民眾反對<ref name="開發p30"/>，使得本地在構想提出30年後的2003年（平成15年）11月才開始開放入住<ref name="開發p32"/>。
