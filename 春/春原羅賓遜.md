'''春原ロビンソン'''（はるはら ろびんそん、{{bd|1986年|3月12日}}<ref>{{Cite web |author=春原ロビンソン|date=2013-07-27|url=https://twitter.com/haruhara/status/360794302382751745|title=Twitter / haruhara: ボク明日１００００日目だから麻里子様ボクと１日違いなのか...|publisher=twitter|accessdate=2013-07-27}}</ref>）[[日本|日本]][[漫畫家|漫畫家]]，男性。代表作品為『[[戰勇。|戰勇。]]』。

== 人物簡介 ==
曾於動畫公司擔任過製片助理，期間因本身體質與壓力等原因，後來辭去工作轉為自由漫畫家<ref>出自[http://www.harvest-inc.jp/label_other/html/222.html アニメ95.2]{{Dead link|date=2018年8月 |bot=InternetArchiveBot |fix-attempted=yes }}一書。</ref>。在[[NICONICO動畫|NICONICO動畫]]以MAD動畫作品成名<ref>{{Cite web|author=春原ロビンソン|date=2008-8-26～|url=http://www.nicovideo.jp/mylist/81486955|title=春原ロビンソン さんの公開マイリスト シリーズ}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，後受到該平臺母公司株式會社[[多玩國|多玩國]]邀請，在當時剛創立的[[NICONICO漫畫|NICONICO漫畫]]連載『[[戰勇。|戰勇。]]』出道。不過該作品本篇已完結，目前半永久連載外傳性質的短篇集《戰勇+》，並於每個月第二與第四個禮拜五在網路更新。

後來更於2013年動畫化，同時在電視與[[NICONICO動畫|NICONICO動畫]]都有播出，目前共兩季，合計26集。發行過OST一張與DVD四卷。

除了NICONICO漫畫外，亦於月刊[[Jump_Square|Jump Square]]連載了『戰勇。』，已完結，單行本共兩冊。但必須注意的是，SQ（Jump Square）與WEB（NICONIO漫畫）版本雖標題相同，且有著相似的角色與世界觀，兩者的走向卻完全不同，作者本人更直接表示過，SQ版本是和WEB版本為平行世界的搞笑漫畫<ref>{{Cite web |author=春原ロビンソン|date=2012-12-04|url=http://seiga.nicovideo.jp/comic/3731|title=戦勇。〔ジャンプSQ.〕 単行本 試し読み！/ 春原ロビンソン - ニコニコ静画 (マンガ) / SQ戦勇はweb戦勇とパラレル世界のギャグ漫画なんです！|publisher=ニコニコ静画}}</ref>。而後又在NICONICO漫畫以[[水曜日のシリウス|水曜日のシリウス]]名下作品連載了『[[戰勇。|戰勇。]]Main Quest』，是依循『WEB戰勇。』內容的紙本規格版本，會有些許的修改，但劇情走向大致與WEB原版本相同，於網路更新，且由[[講談社|講談社]]出版單行本漫畫。

== 逸話 ==
* 有一個曾夢想過成為聲優的姊姊，在『戰勇。』第一季動畫中以「春原麻里子」一名為角色獻聲過<ref>{{Cite web |author=春原ロビンソン|date=2013-02-27|url=http://ch.nicovideo.jp/haruhara/blomaga/ar136153|title=春原のブロマガ / 来週のアニメ戦勇のゲスト声優の話}}</ref>。
* 不幫角色做過於詳細的設定，是因為希望能保持著「感覺他們真的是在哪個世界活著的，而不僅僅是我筆下的創作物。」的心情繼續執筆。
* 不只一次公開表示過，是支持自己的作品被同人二次創作的。但因為覺得害羞所以並不會主動去看。
* 與『[[陽炎Project|陽炎Project]]』作者[[Jin|Jin]]（じん，自然の敵P）關係友好，曾一起開過兩人結婚了的玩笑<ref>{{Cite web |author=春原ロビンソン|date=2013-05-05|url=https://twitter.com/haruhara/status/330892273980170240|title=Twitter / haruhara: じん(自然の敵P)と結婚した|publisher=twitter}}</ref>。
** 曾自稱有七個愛人或者已婚並育有兩子等，在生放送中被讀者問到：「那Jin呢？」的時候表示：「我都忘了這個設定。」
** 曾說過：「感覺要是哪天真的結婚了，在twitter上報告也不會有人相信我了！」
*目前經確認為已婚，並有一個孩子（性別未確認），在部落格上表示當年結婚時有發表一則推特「天氣好熱啊，我稍微出去結個婚。」<ref>{{Cite web |author=春原ロビンソン|date=2013-08-21|url=https://twitter.com/haruhara/status/370058863862501376|title=Twitter / haruhara: 外が凄まじく暑そうだけどそろそろ出かけないと。　ちょっと結婚してくるわ　 |publisher=twitter}}</ref>但因為常做類似的言論，當時似乎沒有半個人相信他是真的去結婚。
* 和『[[青之驅魔師|青之驅魔師]]』的作者[[加藤和惠|加藤和惠]]友好，『戰勇。』主角還曾於該作品露面過一格。
* 與『戰勇。』主角之聲優[[下野紘|下野紘]]會出去一起喝酒，還購買過其寫真書<ref>{{Cite web |author=春原ロビンソン|date=2013-06-06|url=https://twitter.com/haruhara/status/342427378846408704|title=Twitter / haruhara: 4月くらいに下野さんと飲んでたとき、その場でAmazonに注文したスマイルデイズさいきんやっと届いた　 |publisher=twitter}}</ref>。
* 不時會在twitter或生放送「表明」自己的職業、身份與住所等，例如突然說自己還是個要放暑假的高中生，而且是個偵探，或者其實已經四十多歲等。讀者們多半也很配合。
** 因生放送時的發言，維基條目中曾被寫下「從事過與建築相關的工作」，本人看到後在推特上表示困惑並否定（「至少訂正成是現任高中生偵探吧！」），結果這段話在日文版中仍被保存了下來。
* 與[[株式会社スタジオモリケン|株式会社スタジオモリケン]]的[[森井ケンシロウ|森井ケンシロウ]]為好友，跟對方說了：「幫我做動畫啦。」後得到：「好啊。」這樣的答案，於是促成了[[がくモン!〜オオカミ少女はくじけない〜|がくモン!〜オオカミ少女はくじけない〜]]的Flash動畫化。
