《'''春天的故事'''》是一首纪念[[邓小平|邓小平]]的歌曲，以1979年[[改革开放|改革开放]]和1992年[[邓小平南巡|南方讲话]]为两段歌词的内容。由[[蒋开儒|蒋开儒]]、[[叶旭全|叶旭全]]作词，[[王佑贵|王佑贵]]作曲，[[董文华|董文华]]演唱。1994年10月在[[中央电视台|中央电视台]]播出，由此成为知名歌曲。歌曲MV由[[張國立|張國立]]執導。歌曲亦成為1997年元旦在[[中央電視台|中央電視台]]首播的大型文獻紀錄片《[[鄧小平_(紀錄片)|鄧小平]]》的主題曲，同年2月6日播出的[[1997年中国中央电视台春节联欢晚会|1997年中国中央电视台春节联欢晚会]]亦包括此歌曲。

== 创作背景 ==
1992年3月26日，曾在1月份[[邓小平|邓小平]]视察[[深圳|深圳]]时跟随的《[[深圳特区报|深圳特区报]]》副总编辑[[陈锡添|陈锡添]]发表了记录[[邓小平南巡|邓小平南巡]]的长篇通讯《东方风来满眼春》，该通讯一经发表在全国引起了巨大的影响，时任[[黑龙江省|黑龙江省]][[穆棱县|穆棱县]]县[[政协|政协]]副主席的[[蒋开儒|蒋开儒]]就是被这篇文章所震动的人们中的一员。蒋开儒1979年去[[香港|香港]]探亲时曾路过当时还仅仅是个小渔村的深圳，看到报道后，蒋再次前往了深圳，被深圳13年间的变化深深震撼了。为此，蒋开儒创作了《春天的故事》最初的歌词，得到了《[[深圳特区报|深圳特区报]]》的发表。<ref name=dlxww>{{Citeweb |title=唱着“春天的故事”“走进新时代” |url = http://szb.dlxww.com/dlfzb/html/2011-07/01/content_503602.htm |date=2011年7月1日 |publisher=大连法制报}}</ref>

蒋开儒写好歌词后很快就交给了作曲家[[王佑贵|王佑贵]]。一年半之后，王佑贵完成了谱曲。1994年3月，在广东省青春歌曲创作大赛的深圳分赛区，由蒋开儒作词、王佑贵作曲的参赛歌曲《春天的故事》落选。由于不甘心这首歌就此泯灭，蒋、叶打算找人来完善这首歌，他们找到了时任东深供水局副局长、省青联常委的[[叶旭全|叶旭全]]。叶旭全在结构、立意和遣词造句等方面对歌曲进行了二度创作。经过多次修改，面目全新的《春天的故事》得以再生。叶旭全把二度创作后的歌曲直接交到了广东青春歌曲创作大赛评委会。虽然最初在评委中出现了两种截然不同的意见，但最终该歌曲还是在广东“94青春歌曲创作大赛”中独占鳌头，被评为金奖，一举成名。<ref name=dlxww/>

为使该曲能够在全国范围内流行开来，叶旭全等又筹集款项为该曲拍摄了音乐电视片。1994年10月，由[[孟欣|孟欣]]总创意、策划，[[张国立|张国立]]总导演兼摄像的音乐电视片在[[中央电视台|中央电视台]]播出后一炮而红，并获得了多个奖项。<ref name=dlxww/>

== 影响 ==
此曲成为以邓小平为核心的中国共产党第二代[[中国共产党集体领导制度|中央领导集体]]的音乐代表曲目，成为了[[改革|改革]]和“[[改革开放|改革开放]]”的一个代名词<ref>[http://cpc.people.com.cn/pinglun/n/2013/1112/c241220-23519171.html 十八届三中全会再次唱响“春天的故事”]</ref>，

== 获奖 ==
* 1994年，广东“94青春歌曲创作大赛”金奖
* 1994年12月，中央电视台第二届[[音乐电视大赛|音乐电视大赛]]中荣获金奖。
* 被选入大型文献纪录片《邓小平》，获得了1995年中宣部“五个一工程奖”
* 1996年中国音乐最高奖“[[金钟奖|金钟奖]]”
* 广东省鲁迅文艺奖、音乐奖等奖项
