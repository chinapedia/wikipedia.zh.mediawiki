'''藍鰓太陽魚'''（學名：{{lang|la|''Lepomis macrochirus''}}），[[輻鰭魚綱|輻鰭魚綱]][[鱸形目|鱸形目]][[太陽魚科|太陽魚科]]的其中一個[[種|種]]。

== 分布 ==
本魚分布於[[北美洲|北美洲]][[聖勞倫斯河|聖勞倫斯河]][[五大湖|五大湖]]區與[[密西西比河|密西西比河]]流域至[[墨西哥|墨西哥]]北部間的溪流及湖泊中。

== 特徵 ==
本魚為棕色帶淺藍色，身上有許多垂直條紋，兩條紋之間有淺色線隔開。鰓蓋邊緣有淺藍色，因此而得名。「耳蓋」為黑色；尾柄與腹鰭之間的腹部顏色較淺；腰部為黃色。大多數鰭為黃中戴藍色，有少許深色圖案。體長可達41公分。

== 生態 ==
本魚棲息在湖、池塘、水庫與緩動性的溪流中。活躍的時間在薄暮與破曉的時候。屬肉食性，成魚以[[蝸牛|蝸牛]]、小龍蝦、[[昆蟲|昆蟲]]、蠕蟲與魚類等為食。幼魚則捕食[[甲殼動物|甲殼動物]]，[[昆蟲|昆蟲]]與蠕蟲。

== 經濟利用 ==
為重要的遊釣魚，做為觀賞魚及食用魚，具商業價值。並引入某些國家，但造成當地的生態破壞。
