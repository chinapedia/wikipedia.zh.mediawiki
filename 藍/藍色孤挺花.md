'''藍色孤挺花'''，是[[石蒜科|石蒜科]]'''藍色孤挺花屬'''的一種球根植物，依據IUCN3.1歸類為CR。原產於[[南美洲|南美洲]]的[[巴西|巴西]]東部<ref>{{Cite web |title= Worsleya procera |url= http://apps.kew.org/wcsp/namedetail.do?name_id=291432 |publisher= World Checklist of Selected Plant Families: Royal Botanic Gardens, Kew |accessdate= 2011-12-05 |language=en}}</ref>，是巴西[[特有種|特有的植物]]，地底的[[鱗莖|鱗莖]]很大，地上莖可以長到1.5公尺高，花朶很大，開花時非常美麗壯觀，俗名為'''巴西女王'''（empress of Brazil）。花瓣顏色為淡紫色至藍色，花序及花形與[[孤挺花屬|孤挺花]]類似，因此被稱為藍色孤挺花（blue amaryllis）。

== 形態 ==
多年生草本植物，鱗莖很大，是石蒜科石蒜亞科的植物中極為罕見及最高大的植物之一，地上莖高約1.5公尺 <ref>{{Cite web |title= Worsleya |url= http://www.pacificbulbsociety.org/pbswiki/index.php/Worsleya |publisher= Pacific Bulb Society |accessdate= 2011-12-05 |language= en}}</ref>。[[葉|葉]]彎曲呈鐮刀狀，綠色。[[花|花]]淡紫色至藍色，花瓣上有小斑點。花徑約15公分，[[花序|花序]]頂端著生有6朶以上的花<ref>{{Cite web |title= Worsleya procera (= W. rayneri, Hippeastrum procerum) - The Blue Amaryllis |url= http://www.strangewonderfulthings.com/184.htm |publisher= Strange Wonderful Things |accessdate= 2011-12-05 |language= en}}</ref>。[[種子|種子]]黑色，半圓形。

野生的藍色孤挺花喜歡生長在極端及[[潮濕|潮濕]]的環境中，常被發現生長在[[日照|日照]]充足，靠近[[瀑布|瀑布]]含有肥沃[[土壤|土壤]]的[[花崗岩|花崗岩]]上，這也是它被認為是一種[[岩生植物|岩生植物]]的原因。

== 栽培 == 
藍色孤挺花的栽培非常困難，栽培介質以同時滿足透氣及保溼為主，日照以全日照最佳，盆器以素燒陶盆(需勤澆水以保濕，因為陶盆非常透氣)或塑膠盆(需於盆側鑽洞以透氣)為佳。播種時，通常以[[浮石|浮石]]作為播種介質，有時也可以用[[水苔|水苔]]，不過用水苔作介質時種子爛掉不發芽的機率較高。播種後大約四個星期後開始發芽，由播種到第一次開花大約需要九年的時間<ref>{{Cite web |title= Worsleya rayneri |url= http://www.bulbsociety.org/GALLERY_OF_THE_WORLDS_BULBS/GRAPHICS/Worsleya/Worsleya_rayneri3/Worsleya_rayneri3.html |publisher= International Bulb Society |accessdate= 2011-12-05 |language=en}}</ref>。
