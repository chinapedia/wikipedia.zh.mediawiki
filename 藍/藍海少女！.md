《'''藍海少女！'''》（{{lang-ja|あまんちゅ!}}）是[[天野梢|天野梢]]的[[日本|日本]][[漫畫|漫畫]]作品。於2008年11月29日發售的[[Mag_Garden|Mag Garden]]雑誌《[[月刊Comic_Blade|月刊Comic Blade]]》2009年1月號<ref name="ann-end">{{cite web|url=http://www.animenewsnetwork.com/news/2008-10-30/aria-amano-to-launch-amanchu-manga-on-november-29|title=Aria's Amano to Launch Amanchu! Manga on 29 November 2008|publisher=[[動畫新聞網|動畫新聞網]]|date=2008年10月30日|accessdate=2010年1月21日|language=en}}</ref><ref>{{cite web|url=http://www.amazon.co.jp/dp/B001KY1TWS/|script-title=ja:月刊 COMIC BLADE (コミックブレイド) 2009年 01月号|publisher=[[日本亞馬遜|日本亞馬遜]]|accessdate=2016年5月1日|language=ja}}</ref>開始連載。

== 故事簡介 ==
喜歡海和潛水的活潑少女——小日向光，與從東京搬到鄉下的內向少女——大木雙葉，兩名少女在夢丘高中入學時相遇並加入潛水社。以自然豐饒的伊豆半島為舞台，兩名少女透過潛水享受四季的美好日常生活。

== 登場人物 ==
=== 主要角色 ===
; {{nihongo|小日向光|小日向 光（こひなた ひかり）|聲：[[鈴木繪理|鈴木繪理]]<ref name="character">{{cite web|url=http://amanchu-anime.com/character.html|script-title=ja:キャラクター|accessdate=2016年10月25日|language=ja}}</ref>（日本）；[[陳頴琪|陳頴琪]]（香港）}}
: 本作主角。夢丘高中一年級生→二年級生（56話）。
: 綽號「閃光（{{lang|ja|ぴかり}}）」，15歲，身高148cm，1月31日生，血型O型，水瓶座。
: 個性非常積極活潑。興趣是水肺潛水，將來的夢想是做當地的嚮導。身上常常帶著奶奶給的口哨。

; {{nihongo|大木雙葉|大木 双葉（おおき ふたば）|聲：[[茅野愛衣|茅野愛衣]]<ref name="character"/>（日本）；[[王慧珠|王慧珠]]（香港）}}
: 本作另一主角。夢丘高中一年級生→二年級生（56話）。
: 綽號「小點（{{lang|ja|てこ}}）」（因其眉毛星星點點），16歲，身高170cm，4月3日生，血型AB型，牡羊座。
: 有機車駕照。初次搬家到伊豆，人際關係完全重來，使她對新生活有些許畏懼，個性相較內向，所以學期剛開始時不知該如何是好。
: 常常被光的活潑氣場牽著走，很感謝有像光這樣的朋友。
: 對夢境相當敏感，自稱為「夢境專家」，非常了解夢的運作規則，有時心境也會反映在夢境之中。

; {{nihongo|二宮愛|二宮 愛（にのみや あい）|聲：[[大西沙織|大西沙織]]<ref name="character"/>（日本）；[[莎拉|莎拉]]（香港）}}
: 二年級生→三年級生（56話）→畢業生（81話）。
: 通稱「姐姐前輩(あねちゃん先輩)」，潛水部長，16歲，身高160公分，1月7日生，血型A型，魔羯座。
: 個性非常直率，與弟弟二宮誠是雙胞胎。畢業後會上同縣的美術系大學。<ref name="mangach80">漫畫80話</ref>

; {{nihongo|二宮誠|二宮 誠（にのみや まこと）|聲：[[梅原裕一郎|梅原裕一郎]]<ref name="character"/>（日本）；[[張振熙|張振熙]]（香港）}}
: 二年級生→三年級生（56話）→畢業生（81話）。
: 通稱「弟弟前輩(弟くん先輩)」，16歲，身高178公分，1月7日生，血型A型，魔羯座。
: 與姐姐二宮愛是雙胞胎，總是被姐姐欺負。畢業後會到東京上大學，一個人生活。<ref name="mangach80" />

; {{nihongo|火鳥真斗|火鳥 真斗（かとり まと）|聲：[[伊藤靜|伊藤靜]]<ref name="character"/>（日本）；[[蔡雁紅|蔡雁紅]]（香港）}}
: 主角的班導師，同時是潛水部的顧問老師。
: 是一名潛水員，知識豐富，性格大喇喇，認為學生無論是要在高中三年間好好念書還是成日玩到樂此不疲都沒關係，最重要的是要過得快樂。
: 在入學時，為了歡迎學生，將黑板佈置的十分漂亮。

=== 夢丘高中 ===
; {{nihongo|校長先生|校長先生|聲：[[立木文彥|立木文彥]]}}
: 夢丘高中的校長。

; {{nihongo|永遠野守|永遠野 守（とわの まもる）|聲：[[高橋廣樹|高橋廣樹]]（日本）；[[譚禹晉|譚禹晉]]（香港）}}
: 夢丘高中的老師，二宮愛的班導，長髮，總是瞇瞇眼，像是還沒睡醒一般。
: 在還是嬰兒時，曾被母親遺留在大雪中的神社。

=== 小日向家 ===
; {{nihongo|小日向紀乃|小日向 きの（こひなた きの）|聲：[[井上喜久子|井上喜久子]]<ref name="character"/>（日本）；[[吳小藝|吳小藝]]（香港）}}
: 小日向家的奶奶，66歲，身高142公分，7月29日生，血型O型，獅子座。
: 現在在海邊經營著店家「海女人屋」，所煮的豬肉湯有口皆碑。
: 年輕時為海女，後轉行當潛水員，獨自發現了伊豆地區的潛水名勝「富戶岩穴」，可說是伊豆潛水運動的先驅。

; {{nihongo|小日向木靈|小日向 こだま（こひなた こだま）|聲：[[洲崎綾|洲崎綾]]（日本）；[[何凱怡|何凱怡]]（香港）}}
: 小日向家的么妹，初中三年級→夢丘高中一年級生（56話）。
: 從小開始就跟閃光一起進行潛水運動。
: 常會罵人「小呆瓜」來表達自己的不滿。

; {{nihongo|小日向疾風|小日向 はやて（こひなた はやて）|聲：[[杉田智和|杉田智和]]}}
: 小日向家的爺爺，68歲，身高200公分，5月5日生，血型A型，金牛座。
: 被稱作「漆黑的人魚」，是伊豆地區傳說中的潛水員。

; {{nihongo|小日向翼|小日向 つばさ（こひなた つばさ）}}
: 小日向家的長男。

=== 親友 ===
; {{nihongo|水無茜|水無 茜（みずなし あかね）|聲：[[葉月繪理乃|葉月繪理乃]]（日本）；[[石梓晴|石梓晴]]（香港）}}
: 雙葉中學時期的朋友。

; {{nihongo|姬野千鶴|姫野 ちずる（ひめの ちずる）|聲：[[齋藤千和|齋藤千和]]（日本）；[[何凱怡|何凱怡]]（香港）}}
: 雙葉中學時期的朋友。

; {{nihongo|岬心|岬 こころ（みさき こころ）|聲：{{tsl|ja|山本亜衣 (声優)|山本亞衣}}}}
: 常被誤認為女生的男生，3月3日生，雙魚座。
: 在浮潛時跟閃光相識，很喜歡章魚。
: 喜歡閃光，所以對雙葉有著競爭意識。

; {{nihongo|岬小鳥|岬 ことり（みさき ことり）|聲：[[加隈亞衣|加隈亞衣]]}}
: 岬心的姐姐，木靈的朋友，跟木靈兩人一起考入了夢丘高中。
: 曾在入學前的某個下午跟雙葉在同一家咖啡廳休息，結果兩人同時入睡時作了相連的夢。
: 非常憧憬雙葉所表現出的優雅穩重。

=== 永遠的國度 ===
; {{nihongo|彼得（Peter）|ピーター|聲：[[入野自由|入野自由]]}}
: 夢丘高中七大不可思議之一，原型來自於知名劇作「彼得潘（Peter Pan）」。
: 喜歡在像祭典這樣的特別日子現身，誘拐不願長大的少年少女進入夢中的世界，永遠都不會衰老的少年。
: 本來是個被遺棄在神社裡的嬰兒，為了延長自己的生命，把自己封閉在了夢的世界之中。此後不時會有人，來自於不同時代、不同地點，在睡夢之中誤入這個世界。

; {{nihongo|凱特西（Cait Sith）|ケット シー}}
: 依偎在嬰兒身旁，幫助他作夢的黑貓。

=== 其他 ===

; {{nihongo|公主|お姫（おひめ）|聲：[[洲崎綾|洲崎綾]]<ref>{{Cite web||url=http://amanchu-anime.com/?p=272|title=『お姫』キャスト　洲崎綾さんに決定！|accessdate=2016-09-03|language=ja}}</ref>}}
: 被閃光和雙葉在神社撿到的雌性小貓，之後被校長收養。

== 電視動畫 ==
2015年10月宣佈製作電視動畫<ref>{{Cite news |url=http://comic.qq.com/a/20151005/014376.htm |title=《海之少女》2016年TV动画化决定 |work=騰訊動漫 |date=2015-10-05}}</ref>，翌年2016年7月至9月於[[AT-X|AT-X]]、[[東京都會電視台|TOKYO MX]]等台播出第1期。2017年11月宣布製作第2期<ref name="dengekionline20171114">{{Cite web|url=http://dengekionline.com/elem/000/001/628/1628625/|title=TVアニメ『あまんちゅ！』第2期が制作決定。2018年4月より放送開始|work=電擊Online|publisher=[[KADOKAWA|KADOKAWA]]|date=2017-11-14|accessdate=2017-11-14|language=ja}}</ref>，標題名叫《'''藍海少女！～Advance～'''》<ref name="dengekionline20171114" />，於2018年4月至6月首播。

作品標語為「'''{{lang|ja|さあ、新しい物語を始めましょう}}。'''」。
