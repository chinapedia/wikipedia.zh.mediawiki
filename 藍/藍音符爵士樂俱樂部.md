'''藍音符爵士樂俱樂部'''（{{lang-en|'''Blue Note Jazz Club'''}}）是一家爵士樂俱樂部和餐廳，位於[[紐約市|紐約市]][[格林尼治村|格林尼治村]]第三街西131號。該俱樂部於1981年9月30日由業主和創始人Danny Bensusan開張，而{{link-en|Nat Adderley|Nat Adderley}}五重奏樂團為當晚擔任表演者。<ref>{{Cite web |url = https://books.google.com.hk/books?id=MxIEAAAAMBAJ&pg=PA44&redir_esc=y#v=onepage&q&f=false |title = Billboard |date = 2001-11-24 |language = en}}</ref>俱樂部的表演時間安排每天晚上8:00和晚上10:30，而週日爵士早午餐的表演在下午12:30和下午2:30。該場館還開始了一個每兩週一次的夜間劇場演奏系列，提供紐約有前途的爵士樂、靈魂樂、嘻哈、R&B和放克樂表演者在星期六和星期日凌晨12:30展示他們的天分。俱樂部目前繼續營運的位置有日本（[[東京都|東京]]、[[大阪市|大阪]]和[[名古屋|名古屋]]）、[[意大利|意大利]]（[[米蘭|米蘭]]）<ref>{{Cite web |url = http://www.thembj.org/2008/03/interview-with-mega-owner-steven-bensusan/ |title = Interview with Mega-Owner Steven Bensusan |author = Alex Beram |date = 2008-03-? |publisher = Music Business Journal |language = en}}</ref>、美國（[[威基基|威基基]]和[[納帕_(加利福尼亞州)|納帕]]）<ref>{{Cite web |url = http://bluenotehawaii.com/about.html |title = ABOUT BLUE NOTE HAWAII |publisher = Blue Note Jazz Clubs |language = en}}</ref>、[[中國|中國]]（[[北京|北京]]）和[[斯洛伐克|斯洛伐克]]（{{link-en|瓦赫河畔新梅斯托行政區|Nové Mesto nad Váhom District}}）。
