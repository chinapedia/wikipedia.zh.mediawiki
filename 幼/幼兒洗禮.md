'''嬰兒洗禮'''（{{lang-en|infant baptism}}<ref name="CoE">{{cite web |url = http://www.cofe.anglican.org/lifeevents/baptismconfirm/baptism1.html |title = The Baptism Service |quote = Q. What's the difference between a baptism and a christening?<br />A. None, they are just different words for the same thing. |publisher = [[Church_of_England|Church of England]] |accessdate = 2007-06-08 |deadurl = yes |archiveurl = https://web.archive.org/web/20070716125528/http://www.cofe.anglican.org/lifeevents/baptismconfirm/baptism1.html |archivedate = 2007-07-16 }}</ref><ref name="UMC">{{cite web |url = http://www.umc.org/site/apps/nlnet/content3.aspx?c=lwL4KnN1LtH&b=2043509&ct=3006211 |title = Can I have my infant christened? |quote = Christening is not a separate or different service. It is the same thing as baptism. |publisher = [[United_Methodist_Church|United Methodist Church]] |accessdate = 8 June 2007 |deadurl = yes |archiveurl = https://web.archive.org/web/20110607051218/http://www.umc.org/site/apps/nlnet/content3.aspx?c=lwL4KnN1LtH&b=2043509&ct=3006211 |archivedate = 2011年6月7日 }}</ref>）指[[基督宗教|基督宗教]]的[[洗禮|洗禮]]儀式中對嬰兒及兒童的洗禮。該詞有時和信仰洗禮（{{lang-en|credobaptism}}，拉丁語{{lang|la|credo}}意為「我相信」）有相同意義。

大多數主流的基督宗教信徒所屬的教派均承認嬰兒洗禮，包括[[天主教|天主教]]、[[東正教|東正教]]、[[聖公宗|聖公宗]]和[[路德宗|路德宗]]在內<ref>[http://www.nph.com/vcmedia/2369/2369939.pdf So... You Want to Be Baptized? Leaders’ Guide]</ref>，但也有不少教會不承認嬰兒洗禮者為[[基督徒|基督徒]]，如[[浸信宗|浸信宗]]等不少[[基督新教|基督新教]]宗派和[[福音派|福音派]]。基於基督教「因信得教」和「[[因信稱義|因信稱義]]」的基督教基本教義，不少基督教宗派認為雖然作為教徒的父母為其子女帶到教會進行洗禮，但其子女尚處於孩童階段，不一定知道及認同基督教教義，因此不承認嬰兒洗禮者為[[基督徒|基督徒]]。
