'''复兴门南大街'''是位于中国[[北京市|北京市]][[西城区|西城区]]中部的一条大街。[[北京二环路|北京二环路]]的组成部分。

==简介==
复兴门南大街北起[[复兴门内大街|复兴门内大街]]、[[复兴门北大街|复兴门北大街]]和[[复兴门内大街|复兴门内大街]]交汇处的[[复兴门桥|复兴门桥]]，南向弧行至[[宣武门西大街|宣武门西大街]]。此处原为[[北京内城|北京内城]]西城墙及护城河旧址。1960年代末，因修建[[北京地铁|北京地铁]]拆除了城墙。地铁修竣之后，地面上辟筑为道路，成为北京二环路的组成部分及交通干道之一。因地处[[复兴门|复兴门]]以南，而得名“复兴门南大街”。<ref name=bjdmd>王彬、徐秀珊主编，北京地名典，北京：中国文联出版社，2008年，第207页</ref>

==雕塑==
*《'''金花阿鹏'''》：位于复兴门南大街东侧、天银大厦西侧的绿地内。2003年9月28日，作为友好区州的[[云南省|云南省]][[大理白族自治州|大理白族自治州]]人民政府与北京市西城区人民政府联合在北京举办仪式，大理的形象标志“金花、阿鹏”青铜雕塑落户[[西单文化广场|西单文化广场]]。仪式上大理白族自治州民族歌舞团还表演了一台文艺节目。金花、阿鹏青铜雕塑通高3.2米，重1.5吨，展示了一对[[白族|白族]]青年男女载歌载舞的情景。由原[[四川美术学院|四川美术学院]]院长[[叶毓山|叶毓山]]创作<ref>{{Cite web |url =http://www.people.com.cn/GB/paper39/10283/939800.html  |title = “金花阿鹏”雕塑落户北京西单 |author =  |date =  |publisher = 人民日报海外版2003年9月29日，第四版 要闻·社会 |language =  |accessdate =  |quote =  }}</ref>。2000年代中期，因西单文化广场施工，迁至复兴门南大街东侧。
