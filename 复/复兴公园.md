'''复兴公园'''是位于[[中国|中国]][[上海市|上海市]][[黄浦区|黄浦区]][[雁荡路|雁荡路]]105号的一座公园，东至[[重庆南路|重庆南路]]、北接[[雁荡路|雁荡路]]，西接[[香山路_(上海)|香山路]]、[[皋兰路_(上海)|皋兰路]]，南至[[复兴中路|复兴中路]]，目前占地7.3公顷。<ref name=a>{{ cite web |url=http://218.242.36.250/Ghj_CMS/uploads/zdykx/2007-1211/tuze/2007-1211%201031%E3%80%81032%E3%80%81033%E3%80%81034-1.jpg |title=淮海社区控制性详细规划 |author=上海规土局 |accessdate=2014年1月27日 |deadurl=yes |archiveurl=https://web.archive.org/web/20150601094223/http://218.242.36.250/Ghj_CMS/uploads/zdykx/2007-1211/tuze/2007-1211%201031%E3%80%81032%E3%80%81033%E3%80%81034-1.jpg |archivedate=2015年6月1日 }}</ref>复兴公园拥有超过一百年的历史，曾是上海租界内最大的公园，<ref name=f>{{cite web |url=http://218.22.190.134/FK_ArticelSearch.aspx?ID=45619255# |title=上海复兴公园与中山公园空间变迁的比较研究 |author=《中国园林》2013年 第5期 |accessdate=2014年8月20日 |deadurl=yes |archiveurl=https://web.archive.org/web/20140821015241/http://218.22.190.134/FK_ArticelSearch.aspx?ID=45619255 |archivedate=2014年8月21日 }}</ref>也是上海最著名一座法式公园，以安静高雅著称。

== 历史 ==

复兴公园原址为顾家宅村。[[上海法租界|上海法租界]]第二次扩张后，该村紧邻法租界西边界。1900年，[[法租界公董局|法租界公董局]]以7.6万两规银购买土地152亩建造顾家宅兵营。1901年[[辛丑条约|辛丑条约]]签订，兵营内法军逐步减少。1904年，法国总会向公董局申请租用兵营土地建造网球场、停车场等设施，得到了批准。网球场落成后经营状况良好。1908年7月1日，公董局董事会决定将兵营全部、分批改建为公园。公园由法国园艺师柏勃（Papot）设计，在法国[[里昂|里昂]][[金头公园|金头公园]]的蓝图的基础上建造该公园。<ref>{{cite book|title =上海的公园和古典园林|author =惜珍|publisher =东方出版中心|isbn =9787547301487}}</ref> 1909年6月29日，公园建成，并于同年7月14日正式对外开放。正式名称为'''顾家宅公园'''（{{Lang-fr|parc de koukaza}}），俗称法国公园。公园仅限法国侨民进出。之后随周边道路陆续建成，剩余兵营搬迁，公园不断改、扩建，至1925年逐渐形成今日的规模。这部分工程由法国工程师约少默（Jousseaume）设计，中国设计师[[郁锡麒|郁锡麒]]也参与其中。1928年，公园允许华人出入。<ref>{{cite web|url = http://www.cnki.com.cn/Article/CJFDTotal-ZLXI199512021.htm|title = “华人与狗,不得入内”牌子的由来及其拆除|author = 张河成 |accessdate = 2014-08-15}}</ref>抗战爆发后，公董局董事会决定接受上海市立动物园内受战火威胁的动物，在园内建设动物园，并于1938年开放。1943年[[汪精卫政府|汪精卫政府]]收还法租界，1944年随辣菲德路（今复兴中路）更名大兴路，将公园更名为大兴公园，1946年又随路名更名而更名为复兴公园。1963年，公园内动物园撤销，原有动物则转移至西郊公园（现[[上海动物园|上海动物园]]）。文革期间，公园一度更名为红卫公园，但不久即改回原名至今。<ref name=f></ref><ref name=b>{{ cite web|url=http://www.shtong.gov.cn/newsite/node2/node2245/node69854/node69860/node69913/node69923/userobject1ai69560.html |title=复兴公园 |author=上海地方志办公室 |accessdate=2014年1月27日 }}</ref><ref name=c>{{ cite web |url=http://www.yplib.org.cn/structure/jdsh/bnsz/dg_100769_2.htm# |title=复兴公园 |author=杨浦区图书馆 |accessdate=2014年1月27日 |deadurl=yes |archiveurl=https://web.archive.org/web/20140203100634/http://www.yplib.org.cn/structure/jdsh/bnsz/dg_100769_2.htm |archivedate=2014年2月3日 }}</ref>现为[[黄浦区文物保护单位|黄浦区文物保护单位]]。

2014年，正值复兴公园建成105周年之际，公园被[[貓途鷹|貓途鷹]]评为中国十大最受欢迎的公园之一。<ref>{{cite web |url=http://sh.xinhuanet.com/2014-08/13/c_133551664.htm |title=复兴公园度过105岁生日 位列中国十大受欢迎公园 |author=新华网 |accessdate=2014年8月15日 |deadurl=yes |archiveurl=https://web.archive.org/web/20140819083214/http://sh.xinhuanet.com/2014-08/13/c_133551664.htm |archivedate=2014年8月19日 }}</ref>

== 特色 ==
作为上海历史最久的公园之一，也是中国现在保存最完整的法式园林，复兴公园造园采取规则式与自然式相结合的布局：中北部采取规则式，其中的沉床式花坛体现了法式风格；西南部则以自然式为主，在建造中亦融入了中式元素，体现在公园南部亭子与池塘、假山的布置上。公园内现存景观有<ref name=b></ref>：
; 毛毡花坛 
位于公园中部，占地面积2742平方米，为一方形沉床式花坛，由6个小花坛组成。花坛中心面积154平方米，原设有环龙纪念碑，以纪念1911年在上海飞行表演中罹难的环龙，纪念碑于1950年拆除，现设置喷水池。
; 马、恩雕像
位于花坛北部，原为草坪和道路，1985年建成马、恩雕像。
; 大草坪
位于公园东南部，面积8000余平方米，由[[悬铃木|悬铃木]]环绕。
; 月季园
位于公园西部，占地面积2741平方米，为一椭圆形沉床式花坛，由椭圆形环路分割为四个部分，分别种植不同花卉。
; 中国园
处于公园西南部，是公园的扩建部分。园内开凿荷花池，在池南布置长廊。另设置有假山瀑布等景观。<ref name=d>{{ cite web|url=http://www.shtong.gov.cn/newsite/node2/node71994/node82435/node82444/index.html |title=复兴公园 |author=上海地方志办公室 |accessdate=2014年1月31日 }}</ref><ref>{{cite web|url=http://www.cqvip.com/qk/71226x/201306/46352196.html |title=探讨城市综合性公园植物景观设计——以复兴公园为例 |author=《建筑知识：学术刊》2013年 第6期 |accessdate=2014年8月20日}}</ref>

== 商业 ==
复兴公园是上海最有情调的公园之一，商业气息浓郁，公园边缘有[[钱柜KTV|钱柜KTV]]等娱乐场所，园内大草坪经常举办各种嘉年华、演唱会活动。而官邸、97酒吧等已经成为世界各地明星们来上海后经常光顾的地点。复兴公园也一度成为上海的时尚地标，承载着80后上海人的[[集体记忆|集体记忆]]。<ref>{{cite web|url = http://culture.ifeng.com/a/20140430/40107155_0.shtml|title = 复兴公园钱柜停业与一代人的城市记忆|publisher = 凤凰网|accessdate = 2014-08-15}}</ref>

==周边景观==

公园靠西有[[中华民国|中华民国]]国父[[孙中山|孙中山]]的[[上海中山故居|纪念馆]]（香山路7号），和[[张学良|张学良]]寓所（皋兰路1号）。靠南则有[[思南公馆|思南公馆]]和[[周公馆|周公馆]]（思南路73号）。
