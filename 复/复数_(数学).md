'''複數'''，為[[實數|實數]]的延伸，它使任一[[多項式|多項式]][[方程|方程]]都有[[根_(數學)|根]]。複數當中有個「[[虛數單位|虛數單位]]」<math>i</math>，它是<math>-1</math>的一个[[平方根|平方根]]，即<math>i ^2 = -1</math>。任一複數都可表達為<math>x + yi</math>，其中<math>x</math>及<math>y</math>皆為實數，分別稱為複數之「實部」和「虛部」。

複數的發現源於[[三次方程|三次方程]]的根的表達式。數學上，「複」字表明所討論的[[數體|數體]]為複數，如[[矩陣|複矩陣]]、[[複變函數|複變函數]]等。

== 歷史 ==
最早有關[[負數|負數]]方根的文獻出於公元1世紀希臘數學家[[希罗|希罗]]，他考慮的是平頂金字塔不可能問題。16世紀意大利數學家（請參看[[塔塔利亞|塔塔利亞]]和[[卡爾達諾|卡爾達諾]]）得出[[三次方程|一元三次]]和[[四次方程|四次方程式]]的根的表達式，並發現即使只考慮實數根，仍不可避免面對負數方根。17世紀[[笛卡兒|笛卡兒]]稱負數方根為[[虛數|虛數]]，「子虛烏有的數」，表達對此的無奈和不忿。18世紀初[[棣莫弗|棣莫弗]]及[[歐拉|歐拉]]大力推動複數的接受。1730年，棣莫弗提出[[棣莫弗公式|棣莫弗公式]]：

:<math>(\cos \theta + i \sin \theta)^{n} = \cos n \theta + i \sin n \theta</math>，

而歐拉則在1748年提出[[分析學|分析學]]中的[[歐拉公式|歐拉公式]]：

:<math>\cos \theta + i \sin \theta = e ^{i \theta}</math>，

18世紀末，複數漸漸被大多數人接受，當時[[卡斯帕尔·韦塞尔|卡斯帕尔·韦塞尔]]提出複數可看作平面上的一點。數年後，[[高斯|高斯]]再提出此觀點並大力推廣，複數的研究開始高速發展。詫異的是，早於1685年[[約翰·沃利斯_(數學家)|約翰·沃利斯]]已經在''De Algebra tractatus''提出此一觀點。

卡斯帕尔·韦塞尔的文章發表在1799年的''Proceedings of the Copenhagen Academy''上，以當今標準來看，也是相當清楚和完備。他又考慮[[球體|球-{zh-hans:体; zh-hant:體;}-]]，得出[[四元數|四元數]]並以此提出完備的[[球面三角學|球面三角學]]理論。1804年，Abbé Buée亦獨立地提出與沃利斯相似的觀點，即以<math>\pm\sqrt{-1}</math>來表示平面上與實數軸垂直的單位線段。1806年，Buée的文章正式刊出，同年[[讓-羅貝爾·阿爾岡|讓-羅貝爾·阿爾岡]]亦發表同類文章，而阿岡的複數平面成了標準。1831年高斯認為複數不夠普及，次年他發表了一篇備忘錄，奠定複數在數學的地位。[[柯西|柯西]]及[[阿贝尔|阿贝尔]]的努力，掃除了複數使用的最後顧忌，後者更是首位以複數研究著名的。

複數吸引了著名[[數學家|數學家]]的注意，包括[[库默尔|库默尔]]（1844年）、[[克罗内克|克罗内克]]（1845年）、[[Scheffler|Scheffler]]（1845年、1851年、1880年）、[[Bellavitis|Bellavitis]]（1835年、1852年）、[[喬治·皮科克|喬治·皮科克]]（1845年）及[[德·摩根|德·摩根]]（1849年）。[[莫比乌斯|莫比乌斯]]發表了大量有關複數幾何的短文，[[約翰·彼得·狄利克雷|約翰·彼得·狄利克雷]]將很多實數概念，例如[[質數|質數]]，推廣至複數。

[[費迪南·艾森斯坦|費迪南·艾森斯坦]]研究<math>a + bj</math>，其中<math>j</math>是<math>x^3 - 1 = 0</math>的複根。其他如<math>x^k - 1 = 0</math> (<math>k</math>是質數)亦有考慮。類以推廣的先鋒為库默尔的[[完美數|完美數]]理論，經由[[菲利克斯·克莱因|菲利克斯·克莱因]]（1893年）以幾何角度加以簡化。[[伽羅華|伽羅華]]其後提出更一般的推廣，解決了五次以上多項式的根不能表達問題。

== 定義 ==

=== 符号表示 ===
尽管可以使用其他表示法，复数通常写为如下形式

:<math> a + bi</math>，

这裡的<math>a</math>和<math>b</math>是[[实数|实数]]，而''i''是[[虛數單位|虛數單位]]，它有着性质<math>i^2=-1</math>。实数'''<math>a</math>'''叫做复数的'''实部'''，而实数'''<math>b</math>'''叫做复数的'''虚部'''。实数可以被认为是虚部为零的复数；就是说实数''<math>a</math>''等价于复数<math>a+0i</math>。实部为零且虚部不为零的复数也被称作“纯虚数”；而实部不為零且虚部也不为零的复数也被称作“非純虚数”或“雜虛數”。

例如，<math>3+2i</math>是复数，它的实部为3虚部为2。如果<math>z=a+ib</math>，则实部（''<math>a</math>''）被指示为<math>\operatorname{Re}(z)</math>或<math>\Re(z)</math>，而虚部（''<math>b</math>''）被指示为<math>\operatorname{Im}(z)</math>或<math>\Im(z)</math>。

在某些领-{域}-（特别是[[电子工程|电子工程]]，这裡的''i''是[[电流|电流]]的符号）中，虚部<math>i</math>被替代写为<math>j</math>，所以复数有时写为<math>a+jb</math>。

所有复数的[[集合|集合]]通常指示为<math>C</math>，或者用{{Link-en|黑板粗体|Blackboard bold}}写为<math>\mathbb{C}</math>。实数<math>\mathbb{R}</math>可以被当作'''<math>\mathbb{C}</math>'''的[[子集|子集]]，通过把实数的所有成员当作复数：<math>a=a+0i</math>。

=== 等量关系 ===
复数中的虚数是无法比较大小的，即两个虚数-{只}-有相等和不等两种等量关系。

两个复数是相等的，[[当且仅当|当且仅当]]它们的实部是相等的并且它们的虚部是相等的。就是说，設<math>a</math>,<math>b</math>,<math>c</math>,<math>d</math>為實數，則<math>a+bi=c+di</math>当且仅当<math>a=c</math>并且<math>b=d</math>。

=== 运算 ===
通过形式上应用[[代数|代数]]的[[结合律|结合律]]、[[交换律|交换律]]和[[分配律|分配律]]，再加上等式<math>i^2=-1</math>，定义复数的加法、减法、乘法和除法:

:* [[加法|加法]]：<math>\,(a + bi) + (c + di) = (a + c) + (b + d)i</math>
:* [[减法|减法]]：<math>\,(a + bi) - (c + di) = (a - c) + (b - d)i</math>
:* [[乘法|乘法]]：<math>\,(a + bi) (c + di) = ac + bci + adi + bd i^2 = (ac - bd) + (bc + ad)i</math>
:* [[除法|除法]]：<math>\,\frac{(a + bi)}{(c + di)} = \frac{(a+bi)(c-di)}{(c+di)(c-di)} =\frac{ac+bci-adi-bd i^2}{c^2 -(di)^2}=\frac{(ac+bd)+(bc-ad)i}{c^2+d^2}=\left({ac + bd \over c^2 + d^2}\right) + \left( {bc - ad \over c^2 + d^2} \right)i</math>

=== 複數體 ===
複數可定義為實數<math>a, b</math>組成的[[有序對|有序對]]，而其相關之[[和|和]]及[[積|積]]為：

* <math> ( a , b ) + ( c , d ) = ( a + c , b + d )</math>，

* <math> ( a , b ) \cdot ( c , d ) = ( ac - bd , bc + ad )</math>，

複數數系是一個[[域_(数学)|體]]，複數體常以<math>\mathbb{C}</math>來表示。

一個實數<math>a</math>等同於複數<math>(a, 0)</math>，故實數體為複數體的[[子體|子體]]。虛數單位<math>i</math>就是複數<math>(0,1)</math>。此外，還有:

*加法[[单位元|单位元]]（“零元”）: <math> (0, 0)</math>
*乘法单位元（“幺元”）: <math> (1, 0)</math>
*<math>(a,b)</math>的[[加法逆元|加法逆元]]: <math>(-a,-b)</math>
*非零<math>(a,b)</math>的乘法[[逆元|逆元]]（倒数）: <math>\left({a\over a^2+b^2},{-b\over a^2+b^2}\right)</math>。

複數體亦可定為[[代數數|代數數]]的[[拓撲閉包|拓撲閉包]]或實數體的[[代數閉包|代數閉包]]。

=== 複數平面 ===
{{main|複數平面}}
先把坐标轴画出来，横的叫实轴，竖的叫虚轴，然后确定0的位置，<math>z=a+bi</math>可以用二维空间来表示出来。<br />
复数<math>z</math>可以被看作在被称为'''阿甘得图'''（得名於[[让-罗贝尔·阿冈|让-罗贝尔·阿冈]]，也叫做[[高斯|高斯]]平面）的二维[[笛卡尔坐标系|笛卡尔坐标系]]内的一个点或[[矢量|位置向量]]。这个点也就是这个复数''<math>z</math>''可以用笛卡尔（直角）坐标指定。复数的笛卡尔坐标是实部<math>x=\Re z</math>和虚部<math>y=\Im z</math>。复数的笛卡尔坐标表示叫做复数的“笛卡尔形式”、“直角形式”或“代数形式”。

=== 絕對值、共軛與距離 ===
<math>z = re^{i\phi}</math>，则<math>|z|=r</math>是<math>z</math>的'''「[[绝对值#复数的绝对值|絕對值]]」'''（'''「[[模|模]]」'''、'''「[[幅值|幅值]]」'''）。如果<math>z=a+bi</math>，則<math>|z| = \sqrt{a^2+b^2}</math>.

對所有<math>z</math>及<math>w</math>，有
:<math> | z | - | w | \leq | z + w | \leq | z | + | w |</math>

:<math> | z w | = | z | \; | w |</math>

:<math> \left| \frac{z}{w} \right| =\frac{| z |}{| w |}</math>

當定義了距離<math>d ( z , w ) = \left| z - w \right|</math>，複數體便成了[[度量空间|度量空间]]，我們亦可談[[極限|極限]]和[[連續|連續]]。加法、乘法及除法都是連續的運算。

<math>z=a+ib</math>的[[共軛複數|共軛複數]]定義為<math>z=a-ib</math>，記作<math>\overline{z}</math>或<math>z^*</math>。如圖所示，<math>\overline{z}</math>是<math>z</math>关于實數轴的「对称点」。有

:<math>\overline{z+w} = \overline{z} + \overline{w}</math>

:<math>\overline{zw} = \overline{z}\cdot\overline{w}</math>

:<math>\overline{\left( \frac{z}{w} \right)} =\frac{ \overline{z}}{\overline{w}}</math>

:<math>\overline{\overline{z}}=z</math>

:<math>\overline{z}=z</math> 當且僅當<math>z</math>是實數

:<math>|z|=|\overline{z}|</math>

:<math>|z|^2 = z\overline{z}</math>

:<math>z^{-1} = \overline{z}|z|^{-2}</math> 若<math>z</math>非零。這是計算乘法逆最常用的等式。

對於所有代數運算<math>f</math>，共軛值是可交換的。這即是說<math>f(\overline z) = \overline{f(z)}</math>。一些非代數運算如[[正弦|正弦]]「<math>\sin</math>」亦有此性質。這是由於<math>i</math>的不明確選擇——<math>x^2 = -1</math>有二解。可是，共軛值是不可微分的（參見[[全纯函数|全纯函数]]）。

一複數<math>z = re^{i\phi}</math>的「幅角」或「相位」為<math>\phi</math>。此值對模<math>2\pi</math>而言是唯一的。

對於乘法和除法分別有：
:<math>\,re^{\alpha i}se^{\beta i}=(rs)e^{(\alpha + \beta)i}</math>（即“模值相乘，幅角相加”）
:<math>\,\frac{re^{\alpha i}}{se^{\beta i}}=\frac{r}{s}e^{(\alpha - \beta)i}</math>（即“模值相除，幅角相减”)

== 极坐标形式 ==
作为替代，复数<math>z</math>可以用[[极坐标系|极坐标]]来指定。极坐标是叫做'''绝对值'''或'''模'''的<math>r=\left \vert z \right \vert \geq 0</math>和叫做''<math>z</math>''的'''辐角'''的<math>\varphi = \arg z</math>。对于<math>r=0</math>，任何值的<math>\varphi</math>都描述同一个数。要得到唯一的表示，常规的选择是设置<math>\arg 0 =0</math>。对于<math>r>0</math>辐角<math>\varphi</math>[[模运算|模以]]<math>2\pi</math>后是唯一的；就是说，如果复数辐角的两个值只相差精确的<math>2\pi</math>的整数倍数，则它们被认为是等价的。要得到唯一表示，常规的选择是限制<math>\varphi</math>在区间<math>(-\pi,\pi]</math>内，就是<math>-\pi <\varphi \leq \pi</math>。复数的极坐标表示叫做复数的“极坐标形式”。

=== 从极坐标形式到笛卡儿坐标形式的转换 ===
:<math>x = r \cos \varphi</math>
:<math>y = r \sin \varphi</math>

=== 从笛卡尔坐标形式到极坐标形式的转换 ===
:<math>r = \sqrt{x^2+y^2}</math>

:<math>\varphi =
\begin{cases}
\arctan(\frac{y}{x}) & \mbox{if } x > 0\\
\arctan(\frac{y}{x}) + \pi & \mbox{if } x < 0 \mbox{ and } y \ge 0\\
\arctan(\frac{y}{x}) - \pi & \mbox{if } x < 0 \mbox{ and } y < 0\\
+\frac{\pi}{2} & \mbox{if } x = 0 \mbox{ and } y > 0\\
-\frac{\pi}{2} & \mbox{if } x = 0 \mbox{ and } y < 0\\
\mathrm{undefined} & \mbox{if } x = 0 \mbox{ and } y = 0.
\end{cases}</math>

前面的公式要求非常繁杂的情况区分。但是很多编程语言提供了经常叫做[[atan2|atan2]]一个变体的[[反正切|反正切]]函数来处理这些细节。使用反余弦函数的公式要求更少的情况区分:
:<math>\varphi =
\begin{cases}
+\arccos\frac{x}{r} & \mbox{if } y \geq 0 \mbox{ and } r \ne 0\\
-\arccos\frac{x}{r} & \mbox{if } y < 0\\
\mathrm{undefined} & \mbox{if } r = 0.
\end{cases}</math>

=== 极坐标形式的符号 ===
极坐标形式的符号
:<math> z = r\,(\cos \varphi + i\sin \varphi )</math>，
被叫做“三角形式”。有时使用符号[[cis函數|cis]] φ简写[[餘弦|<span style="text-decoration: underline;">c</span>os]]φ + <span style="text-decoration: underline;">''i''</span>[[正弦|<span style="text-decoration: underline;">s</span>in]]φ。
使用[[欧拉公式|欧拉公式]]还可以写为
:<math> z = r\,\mathrm{e}^{i \varphi}\,,</math>
这叫做“指数形式”。

=== 极坐标形式下的乘法、除法、指数和开方根 ===
在极坐标形式下乘法、除法、指数和开方根要比笛卡尔形式下容易许多。

使用[[三角恒等式#角的和差恒等式|三角恒等式]]得到

:<math>r_1\,e^{i\varphi_1} \cdot r_2\,e^{i\varphi_2}
= r_1\,r_2\,e^{i(\varphi_1 + \varphi_2)}</math>，

和

:<math>\frac{r_1\,e^{i\varphi_1}}{r_2\,e^{i\varphi_2}}
 = \frac{r_1}{r_2}\,e^{i (\varphi_1 - \varphi_2)}</math>。

依据[[棣莫弗定理|棣莫弗定理]]做整数幂的指数运算，

:<math>\big(r\,e^{i\varphi}\big)^n = r^n\,e^{in\varphi}</math>。

任意复数幂的指数运算在条目[[指数函数|指数函数]]中讨论。

两个复数的加法只是两个向量的[[向量空间|向量加法]]，乘以一个固定复数的可以被看作同时旋转和伸缩。

乘以<math>i</math>对应于一个逆时针旋转90 [[角度|度]](<math>\frac{\pi}{2}</math> [[弧度|弧度]])。方程<math>i^2=-1</math>的几何意义是顺序的两个90度旋转导致一个180度(<math>\pi</math>弧度)旋转。甚至算术中的<math>(-1)\times(-1)=+1</math>都可以被在几何上被理解为两个180度旋转的组合。

任何数的所有[[方根|方根]]，实数或复数的，都可以用简单的[[算法|算法]]找到。<math>n</math>次方根给出为

:<math>  \sqrt[n]{r e^{i\varphi}}=\sqrt[n]{r}\ e^{i\left(\frac{\varphi+2k\pi}{n}\right)}</math>

对于<math>k=0,1,2,\ldots,n-1</math>，这裡的<math>\sqrt[n]{r}</math>表示<math>r</math>的主<math>n</math>次方根。

== 一些特性 ==

=== 矩陣表達式 ===
這是個實用價值不大，但具數學意義的表達式，是將複數看作能旋轉及伸縮二維位置矢量的2×2實數[[矩陣|矩陣]]，即是
:<math>a+ib \leftrightarrow \begin{pmatrix}
  a &   -b  \\
  b & \;\; a
\end{pmatrix} = r\begin{bmatrix} 
    \cos\varphi & -\sin\varphi \\
    \sin\varphi & \cos\varphi 
  \end{bmatrix} = r\exp\left(\varphi\begin{bmatrix} 
    0 & -1 \\
    1 & 0 
  \end{bmatrix}\right),</math>

其中<math>a</math>及<math>b</math>為實數。可算出此類矩陣的和、積及乘法逆都是此類矩陣。此外
:<math>\begin{pmatrix}
  a &     -b  \\
  b & \;\; a
\end{pmatrix}
=
a \begin{pmatrix}
  1 & \;\; 0  \\
  0 & \;\; 1
\end{pmatrix}
+
b \begin{pmatrix}
  0 &     -1  \\
  1 & \;\; 0
\end{pmatrix}</math>
即實數1對應着單位矩陣
:<math>\begin{pmatrix}
  1 & \;\; 0  \\
  0 & \;\; 1
\end{pmatrix}</math>，
而虛數單位<math>i</math>對應着
:<math>\begin{pmatrix}
  0 &     -1  \\
  1 & \;\; 0
\end{pmatrix}</math>。

此矩陣令平面作逆時鐘90度旋轉，它的平方就是-1。

複數的絶對值就是[[行列式|行列式]]的[[平方根|平方根]]。這些矩陣對應相應的平面變換，其旋轉角度等於複數的徧角，改變比例等於複數的絶對值。複數的軛就是矩陣的[[轉置|轉置]]。

若矩陣中的<math>a</math>和<math>b</math>本來就是複數，則構成的代數便是[[四元數|四元數]]。由此，矩陣代表法可看成代數的[[凯莱-迪克森结构|凱萊-迪克森結構法]]。

=== 實向量空間 ===
<math>\mathbb{C}</math>可以視作二維[[實數|實]][[綫性空間|綫性空間]]。<ref>{{cite web | url=http://episte.math.ntu.edu.tw/articles/mm/mm_01_1_02/ | title=從實數到複數 | publisher=數學知識 | accessdate=2014-10-22 | author=繆龍驥}}</ref>不同於實數體，複數體上不可能有與其算術相容的[[全序|全序]]：'''<math>\mathbb{C}</math>'''並非[[有序體|有序體]]。

<!--//对上面的C域是个二维线性空间表示严重质疑，理由如下：1，数域是数域，空间是空间；2，复平面（表示复数域）只是和二维实线性空间（即R^2）有相同拓扑结构；3，复数域的乘法是两个复数相乘，结果仍在复数域上，而R^2上的乘法是两个向量相乘，结果可能是一个实数（点乘）或垂直于R^2的向量（叉乘），由此可见C域（复数域）根本不是二维实线性空间。-->
=== 多項式的根 ===
滿足<math>p(z)=0</math>的複數''z''是[[多項式|多項式]]<math>p</math>的“根”。[[代數基本定理|代數基本定理]]指出，所有<math>n</math>次多項式，不管實數系數抑或複數系數的，都剛好有''<math>n</math>''個複數根（<math>k</math>重根按''<math>k</math>''个计算）。這定理等價於複數體是[[代數閉體|代數閉體]]。

事實上，複數體是實數體的[[代數閉包|代數閉包]]。它是[[多項式|多項式]][[環|環]]<math>\mathbb{R}[X]</math>經由[[理想_(环论)|理想]]<math>\left \langle X^2+1 \right \rangle</math>顯生出的[[商環|商環]]：

:<math> \mathbb{C} = \mathbb{R}[ X ] / ( X^2 + 1)</math>。

這是一個體因為<math>X^2+1</math>為[[不可約多項式|不可約多項式]]，而<math>X</math>在商環內對應着虛數單位<math>i</math>。

=== 代數特徵 ===
複數體'''<math>\mathbb{C}</math>'''唯一（就[[體同構|體同構]]來說）的體擁有三項代數特征:

*它的[[特徵值|特徵值]]是0
*它對[[質數體|質數體]]的[[超越度|超越度]]是[[連續統|實數的基數]]
*它是[[代數閉包|代數閉]]的

而然，'''<math>\mathbb{C}</math>'''包含很多與'''<math>\mathbb{C}</math>'''同構的子[[體_(數學)|體]]。

===不可排序===
在'''<math>\mathbb{C}</math>'''上不可能建立與其加法及乘法相容之[[全序關係|全序關係]]，即不存在一全序<math>\preceq</math>使得對於任意複數<math>z_1, z_2</math>，有<math>0 \preceq z_1, z_2 \Rightarrow 0 \preceq z_1+z_2, 0 \preceq z_1 z_2</math>。

===复指数幂===
{{main|幂#正实数的复数幂}}
计算一个实数的复数幂是可以的。<math>a^z</math>可以定义为<math>e^{z\cdot \ln(a)}</math>。

== 複分析 ==
研究複變函數的理論稱為[[複分析|複分析]]。它在[[應用數學|應用數學]]和其他數學分支上都有許多實際應用。[[實分析|實分析]]和[[數論|數論]]的結果，最自然的証明經常是以複分析的技巧完成（例子可見[[素數定理|素數定理]]）。

複變函數的圖像是四維的，所以不像實變函數般可以用平面圖像表示。要表示複變函數的圖像，可以用有顏色的三維圖像表達四維資訊，或者以動畫表示函數對複平面的動態變換。

== 應用 ==

=== 系统分析 ===
在[[系统分析|系统分析]]中，系统常常通过[[拉普拉斯变换|拉普拉斯变换]]从[[时域|时-{域}-]]变换到[[频域|频-{域}-]]。因此可在複平面上分析系统的[[极点|极点]]和[[零点|零点]]。分析系统稳定性的[[根轨迹|根轨迹法]]、[[奈奎斯特图|奈奎斯特图法]]和[[尼科尔斯图|尼科尔斯图法]]都是在複平面上进行的。

无论系统极点和零点在左半平面还是右半平面，根轨迹法都很重要。如果系统极点
*位于右半平面，则[[因果系统|因果系统]]不稳定；
*都位于左半平面，则因果系统稳定；
*位于虚轴上，则系统为[[临界稳定|临界稳定]]的。

如果稳定系统的全部零点都位于左半平面，则这是个[[最小相位系统|最小相位系统]]。如果系统的极点和零点关于虚轴对称，则这是[[全通系统|全通系统]]。

=== 信号分析 ===
[[信号分析|信号分析]]和其他领-{域}-使用复数可以方便的表示周期信号。模值<math>\left \vert z \right \vert</math>表示信号的[[幅度|幅度]]，辐角<math>\arg z</math>表示给定[[频率|频率]]的[[正弦|正弦波]]的[[相位|相位]]。

利用[[傅里叶变换|傅里叶变换]]可将实信号表示成一系列周期函数的和。这些周期函数通常用形式如下的複函數的实部表示：
:<math> f ( t ) = z e^{i\omega t}</math>，

其中<math>\omega</math>对应[[角频率|角频率]]，复数<math>z</math>包含了幅度和相位的信息。

[[电路|电路]]分析中，引入[[电容|电容]]、[[电感|电感]]与频率有关的虚部可以方便的将[[电压|电压]]、[[电流|电流]]的关系用简单的线性方程表示并求解。（有时用字母<math>j</math>作为虚数单位，以免与电流符号''i''混淆。）

=== 反常積分 ===
在應用層面，複分析常用以計算某些實值的[[反常積分|反常積分]]，藉由複值函數得出。方法有多種，見{{le|圍道積分方法|Methods of contour integration}}。

=== 量子力學 ===
[[量子力學|量子力學]]中複數是十分重要的，因其理論是建基於複數體上無限維的[[希尔伯特空间|希尔伯特空间]]。

=== 相對論 ===
如將時間變數視為虛數的話便可簡化一些[[狹義相對論|狹義]]和[[廣義相對論|廣義相對論]]中的[[時空|時空]]度量 (Metric)方程。

=== 應用數學 ===
實際應用中，求解給定[[差分方程|差分方程]]模型的系統，通常首先找出線性差分方程對應的[[特徵方程|特徵方程]]的所有複特徵根''r''，再將系統以形爲''f''(''t'')= ''e''<sup>''rt''</sup>的基函數的線性組合表示。

=== 流-{zh-hans:体; zh-hant:體;}-力學 ===
複函數於[[流體力學|流-{zh-hans:体; zh-hant:體;}-力學]]中可描述[[二維|二維]][[勢流|勢流]]。

=== 分形 ===
一些[[分形|分形]]如[[曼德博集合|曼德博集合]]和[[茹利亚集|茹利亚集]]（Julia set）是建基於複平面上的點的。

== 复数的平方根 ==
{{main|平方根#负数与複數的平方根}}
复数的平方根是可以计算的。其公式为<math>\sqrt{x+iy} = \sqrt{\frac{\left|x+iy\right| + x}{2}} \pm i \sqrt{\frac{\left|x+iy\right| - x}{2}}</math>。
