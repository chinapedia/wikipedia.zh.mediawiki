'''复兴门桥'''，位于[[北京市|北京市]][[西城区|西城区]]，是[[北京二环路|北京二环路]]的第一座立交桥。

==简介==
复兴门桥位于原[[复兴门_(北京)|复兴门]]处，[[长安街|长安街]]延长线上的[[复兴门外大街|复兴门外大街]]－[[复兴门内大街|复兴门内大街]]东西向从桥上通过，而属于[[北京二环路|二环路]]的[[复兴门北大街|复兴门北大街]]－[[复兴门南大街|复兴门南大街]]南北向穿过立交桥。桥长43.35米，宽51.5米，桥梁面积2233平方米，有4条圆形匝道。复兴门桥同[[北京地铁|地铁]]二期同期施工，于1972年3月开工，1974年9月竣工，次年12月正式竣工验收<ref>{{cite book  | title = 北京地方志·市政卷·道桥志| publisher = 北京出版社| year = 2001|isbn=7-200-04454-7}}</ref>。

1997年6月，为庆祝[[香港回归|香港回归]]祖国，在复兴门桥东侧、[[建国门桥|建国门桥]]西侧，分别建立一座彩虹门。2017年，这两座彩虹门的主体结构大部分杆件已有锈蚀，内部线路老化，经[[中共北京市委|中共北京市委]]、[[北京市人民政府|北京市人民政府]]批准，[[北京市城市管理委员会|北京市城市管理委员会]]将两座彩虹门拆除重建。重建的彩虹门位置、外形均不变，基础是钢筋混凝土独立基础，主体结构是钢桁架结构，灯具由[[霓虹灯|霓虹灯]]改为[[LED灯|LED灯]]。2017年11月25日凌晨开始拆除工作，2018年1月31日前重建竣工<ref>{{Cite web |url =http://www.xinhuanet.com/photo/2017-11/25/c_1122010060.htm  |title =北京长安街彩虹门拆除重建  |author =  |date = 2017-11-25 |publisher =新华网  |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url =http://www.sohu.com/a/206282178_203914  |title = 长安街彩虹门拆除重建 2018年1月31日前竣工 |author =  |date =2017-11-24  |publisher =搜狐  |language =  |accessdate =  |quote =  }}</ref>。

1999年，由[[首都绿化委员会|首都绿化委员会]]建议，经北京市人民政府批准同意，开展了林木绿地认养活动。[[张国立|张国立]]、[[邓婕|邓婕]]夫妇与北京园林局绿化处联系后，认养了复兴门桥匝道内的4块合计10000平方米的绿地，双方签订了绿地认养协议，夫妇每年为此缴纳4万元人民币的养护、维护费用。园林绿化部门在四块绿地中树立了认养宣传牌，牌上写有张国立、邓婕的名字。到2011年，夫妇已连续认养这4块绿地13年，此后他们每年仍继续认养这4块绿地<ref>{{Cite web |url =http://ent.sina.com.cn/s/m/2011-09-29/14373431200.shtml  |title = 张国立邓婕勤于公益 认养复兴门绿地达13年 |author =  |date =2011-09-29  |publisher = 新浪 |language =  |accessdate =  |quote =  }}</ref>。

==雕塑==
复兴门桥四角匝道之外不远处各有一座雕塑：
*《'''和平少女'''》：位于复兴门桥西北方向绿地内，复兴门外大街北侧。为大型[[汉白玉|汉白玉]]雕像，由[[潘鹤|潘鹤]]、[[王克庆|王克庆]]、[[郭其祥|郭其祥]]、[[程允贤|程允贤]]合作<ref>{{Cite web|url = http://www.zhongdiaowang.com/2013/01/26299685.html|title = 和平少女|author = |date = |publisher = 中雕网|language = |accessdate = 2018-02-17|quote = |archive-url = https://web.archive.org/web/20160820191633/http://www.zhongdiaowang.com/2013/01/26299685.html|archive-date = 2016-08-20|dead-url = yes}}</ref>。1983年11月，[[中共中央总书记|中共中央总书记]][[胡耀邦|胡耀邦]]访问[[日本|日本]]， 答应以国家名义赠送一尊塑像给[[长崎|长崎]]{{link-ja|平和公园|平和公園|平和公园}}永久陈列。潘鹤设计的《和平少女》初稿中选。但日方要求雕像基座不高于2米<ref>{{Cite web |url = http://www.chinawriter.com.cn/n1/2016/0722/c404005-28576713.html |title =潘鹤作品背后的秘密——访雕塑家潘鹤  |author =  |date = 2016-07-22 |publisher =中国作家网  |language =  |accessdate =  |quote =  }}</ref>。1985年初，潘鹤在日本工作半年，完成了日本长崎《和平少女》的创作<ref>{{Cite web |url = http://news.163.com/11/1116/10/7IVOP7PK00014JB6.html |title =  一尊“母亲”铜像 落户闽粤三地|author =  |date = 2011-11-16 |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref>。1985年，中国政府将一对相同的《和平少女》雕塑中的一座赠送给日本长崎平和公园，以表达中国人民热爱和平、期待中日友好的愿望<ref>两言，日本长崎平和公园雕塑，神州学人2006(2):43-43</ref>。同年，这对《和平少女》雕塑中的另一座被安放在复兴门桥西北角<ref name=zhm/>。1986年中国发行的《[[国际和平年|国际和平年]]》金银纪念币图案便采用了《和平少女》<ref>我国发行国际和平年纪念币，中国金融1986(10)</ref>。2004年4月，一位日本老人捐款清洗了该少女像，以纪念已去世的和该少女像很相像的女儿。2005年1月，该少女像的右手食指前端断裂掉落，随后被修复<ref name=zhm>{{Cite web |url = https://news.qq.com/a/20050117/000228.htm |title = 北京著名塑像遭破坏 “和平少女”成断指 |author =  |date = 2005-01-17 |publisher = 腾讯 |language =  |accessdate =  |quote =  }}</ref>。2006年4月8日，北京城市雕塑建设管理办公室开始清洗长安街、王府井、二环等沿线的雕塑，清洗工作从《和平少女》开始。自2006年起，4月第二个星期六定为北京市的“雕塑清洗日”，城市雕塑将每年清洗一次<ref>{{Cite web |url = http://news.163.com/06/0409/01/2E7SRQMS00011229.html |title = 长安街蒙尘雕塑洗刷刷 |author =  |date = 2006-04-09 |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref>。
*《'''海豚与人'''》：位于复兴门桥西南方向绿地内，复兴门外大街南侧，电教大楼（[[中国教育电视台|中国教育电视台]]所在地）正北。为大型[[钛|钛]]雕塑。由[[崔玉琴|崔玉琴]]、[[孙贤陵|孙贤陵]]、[[王引家|王引家]]、[[赵文煜|赵文煜]]合作<ref> {{Cite book | author = 全国城市雕塑建设指导委员会 编 | title = 中国城市雕塑50年 | location =  | publisher = 陕西人民美术出版社 | date =1999年  | pages =  | ISBN =  | accessdate =  | url =  | language =  | quote =  }} </ref>。[[宝鸡|宝鸡]]有色金属加工厂制作。1989年[[中华人民共和国国庆节|中华人民共和国国庆节]]前夕落成。高6米，宽4.5米，造型复杂，线条流畅，表达了人与自然亲密和谐的情感。这座雕塑开创了中国城市钛景观制作的先河<ref>钛雕塑“海豚与人”，稀有金属材料与工程1992(6):53</ref>。
*《'''树'''》：又称《生命树》，位于复兴门桥东南方向绿地内，复兴门内大街南侧。1998年12月，由首都城市雕塑艺术委员会、中共北京市委宣传部、全国城市雕塑建设指导委员会共同组织的“新世纪的希望”城市雕塑设计方案竞赛开始向全国征稿，征得稿件750多件，邀请入选的120多件做成立体方案，并于1999年4月2日起在[[北京国际会议中心|北京国际会议中心]]举办“新世纪的希望”城市雕塑方案展览，由建筑、规划、园林、雕塑专家及长安街沿线单位讨论。经评审及优选，推荐其中8件作品在长安街设置，1999年9月中旬全部建成，以庆祝中华人民共和国成立五十周年。其中之一是《树》，为大型[[不锈钢|不锈钢]]雕塑，由[[鲁迅美术学院|鲁迅美术学院]]雕塑系[[霍波洋|霍波洋]]、[[屈东群|屈东群]]、[[张沈|张沈]]、[[洪涛|洪涛]]等人设计，北京市贵龙雕塑有限责任公司制作。《树》由四根稳定的树干支撑248只鸽子组成的树冠，高6米，象征着和平和教育事业枝繁叶茂、祖国繁荣昌盛<ref>{{Cite web|url = http://www.bjghw.gov.cn/web/static/articles/catalog_66/article_558/558.html|title = 六、城市雕塑|author = |date = 2002-07-05|publisher = 北京市规划和国土资源管理委员会（城乡规划）|language = |accessdate = |quote = |archive-url = https://web.archive.org/web/20180217202932/http://www.bjghw.gov.cn/web/static/articles/catalog_66/article_558/558.html|archive-date = 2018-02-17|dead-url = yes}}</ref>。
*《'''马踏飞燕'''》：位于复兴门桥东北方向绿地内，复兴门内大街北侧，[[北京百盛购物中心|北京百盛购物中心]]（原为[[中国工艺美术馆|中国工艺美术馆]]）正南。1999年建成。高2.5米，铸铜材质。是1969年出土于[[甘肃|甘肃]][[武威|武威]]雷台、被[[郭沫若|郭沫若]]定名为《[[马踏飞燕|马踏飞燕]]》的青铜雕塑的大型复制品<ref>{{Cite web|url = http://www.bjghw.gov.cn/web/static/articles/catalog_66/article_553/553.html|title = 四、绿化美化（上）|author = |date = 2002-07-05|publisher = 北京市规划和国土资源管理委员会（城乡规划）|language = |accessdate = |quote = |archive-url = https://web.archive.org/web/20180217202917/http://www.bjghw.gov.cn/web/static/articles/catalog_66/article_553/553.html|archive-date = 2018-02-17|dead-url = yes}}</ref>。
