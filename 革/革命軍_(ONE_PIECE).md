'''革命軍'''是漫畫《[[ONE_PIECE|ONE PIECE]]》裡虛構的反政府的革命[[组织_(社会学)|組織]]。

== 組織簡介 ==
《ONE PIECE》的世界裡，[[世界政府_(ONE_PIECE)|世界政府]]將之視為世界穩定的最大威脅；然而，其性質卻有別於[[海賊_(ONE_PIECE)|海賊]]，因一般海賊並不會主動去攻擊政府或海軍，但革命軍會策動推翻國家的政權、幫助遭海賊掠奪的貧困國家擊退來犯的海賊、協助遭遇天災人禍的國家居民避難等，故長期與世界政府互有勝負，世界政府甚至動用CP9去暗殺革命軍地方領導幹部。

革命軍作風神秘，一般人並不知曉他們的行動；惟首領「革命家」蒙其·D·多拉格卻為世人所認知，更被政府認定為「世界最兇惡的罪犯」。革命軍擁有很強的情報網，能掌握及分析世界各地發生的事件與後續，例如：白鬍子死後的世界變局。成員大多都戴著如老鼠、長頸鹿等各種動物造型的帽子。

被革命軍收容或志願加入的對象不分身分、種族、階級，動畫第737話薩波的回憶中，透露革命軍也會收容並訓練孤苦無依的孩童成為革命軍的戰力<ref name="ONEPIECE動畫_737">《ONE PIECE》漫畫第737話。</ref>。

=== 根據地 ===
革命軍的根據地原本位於偉大航道上的白土之島{{nihongo|巴爾迪可|バルティゴ}}<ref>《ONE PIECE》漫畫第440話。</ref>，但受到黑鬍子海賊團的襲擊重創，後遷至「卡馬帕卡」王國<ref>動畫EP752</ref>。

=== 船隻 ===
革命軍的主船名「畢特·格拉馬號」，是一艘船首為黑色西方龍的黑色大船。

=== 經歷 ===
* 10年前革命軍曾現身於東海哥雅王國「不確定物終點站」，協助因火災而毀掉家園的災民撤離。 
* 6年前在「聖地」馬力喬亞舉行的「世界會議」中，伊路西亞王國的國王修恩達·魯卡斯發表了革命軍首領蒙其·D·多拉格存在的危險性<ref name="ONEPIECE漫畫_142">《ONE PIECE》漫畫第142話。</ref>。
* 在草帽海賊團現身在羅格鎮時，魯夫被斯摩格抓住，而蒙其·D·多拉格及時救出魯夫讓魯夫一行人繼續航行。
* 在草帽海賊團大鬧司法之島前後，革命軍在南海成功推翻了國家{{nihongo|聖多雷亞|セントウレア}}的政權<ref>ONEPIECE漫畫第440話</ref>。
* 革命軍幹部暨卡瑪帕卡女王艾波里歐·伊娃科夫率領推進城新人類們幫助魯夫等人逃出推進城，參與「頂點戰爭」。
* 革命軍視「歐哈拉事件」的唯一生還者妮可·羅賓為「革命之燈」，多年來一直尋找她的下落，並想邀請她加入。在[[妮可·羅賓|妮可·羅賓]]被「暴君」巴索羅繆·大熊的果實能力彈飛到特奇拉沃爾夫期間，革命軍動員解放此地，釋放了包含羅賓在內因建造大橋而被囚禁勞役的奴隸。事後，革命軍接引羅賓並共同行動一段時間。
* 兩年後的「多雷斯羅薩事件」裡，間接協助魯夫等人對抗[[唐吉訶德·多佛朗明哥|唐吉訶德·多佛朗明哥]]的黨羽，但在多雷斯羅薩事件落幕後，總部巴爾迪可被黑鬍子海賊團一號船船長吉札士·伯吉斯找到，爾後被黑鬍子海賊團襲擊，嚴重損毀。在草帽海賊團偕同杰爾瑪66、火戰車海賊團、太陽海賊團大鬧四皇之一「BIG MOM」[[夏洛特·莉莉|夏洛特·莉莉]]的總領地圓蛋糕島後，革命軍將總部遷移至桃色島的卡馬帕卡王國，同時魯夫與薩波互為義兄弟的關係亦公諸於世<ref name="ONEPIECE漫畫_904">《ONE PIECE》漫畫第904話。</ref>。
* 本屆「世界會議」開始前，革命軍的4名軍隊長幫助露露西亞王國擊敗並制服隸屬黑鬍子旗下的桃鬍子海賊團。

== 角色介紹 ==
=== 總司令官 ===
; <span id="蒙其·D·多拉格"></span>{{Anchor|蒙其·D·多拉格|多拉格}}{{nihongo|蒙其·D·多拉格|モンキー·D·ドラゴン|Monkey·D·Dragon}}
: 聲優：[[柴田秀勝|柴田秀勝]]（日本）；[[符爽|符爽]]〈ep.52-324、737-738〉／[[孫中台|孫中台]]〈ep.501-510〉（台灣）
: 年齡：55歲／生日：10月5日／星座：天秤座／身高：256公分／血型：F型／出生地：東海「哥亞王國」／喜歡的食物：波卡迪托牛奶蛋糕<ref>海賊王20周年雜誌《ONE PIECE magazine VOL.3》</ref>。
: 革命軍的總司令官，別名「世界最兇惡的罪犯」。身穿深綠色大衣，左脸有红色的方紋刺青。為海軍[[海軍_(ONE_PIECE)#蒙其·D·卡普|卡普]]的兒子、海賊[[蒙其·D·魯夫|魯夫]]的父親。
: 多年來倡導顛覆世界政府的革命思想，能夠直接對抗世界政府的革命軍領導人，從10多年前就開始倡導革命思想，甚至已經推翻好幾個王國，所以被世界政府視為必須特別注意的極兇惡人物。
: 10年前與伊娃柯夫等人，在東海目擊故鄉哥亞王國的大火災，在街上觀察情況時偶遇求助的薩波，對於哥亞王國的腐敗導致孩子如此吶喊而感到震驚，與革命軍幹部救出為數不少的居民<ref>《ONE PIECE》漫畫第586話。</ref>。為儲備糧食，曾在[[索隆|索隆]]小時候修煉的村子做短暫停留，隨後在天龍人抵達哥雅王國當晚救走在該海域被襲擊的薩波並回到「白土島」巴爾迪可<ref>《ONE PIECE》漫畫第589話。</ref>。
: 在東海羅格鎮成功地從斯摩格手上救走魯夫，使魯夫得以前往偉大航道。在頂點戰爭時海軍正式宣布魯夫是多拉格之子一事，伊娃詢問革命軍的軍心是否會被動搖，但多拉格認為這樣讓一向神祕的他較為人性化反而是好事<ref name="ONEPIECE漫畫_593"/>。
: 頂點戰爭過後，他對部下表示希望能親自會見[[妮可·羅賓|妮可·羅賓]]，委託部下將羅賓帶到革命軍的本部。
: 多雷斯羅薩事件後，因需要龐大力量，指派可亞拉召集分散在全世界的革命軍幹部。圓蛋糕島事件落幕後，閱報得知魯夫為首的聯合軍大鬧圓蛋糕島的消息。
: 名字多拉格（Dragon），是英文「[[龍|龍]]」的意思。

=== 參謀總長 ===
[[File:薩波_(ONE_PIECE).jpg|thumb]]
; <span id="薩波"></span>{{Anchor|薩波}}{{nihongo|薩波|サボ|Sabo}}
: 聲優：[[古谷徹|古谷徹]]、[[竹內順子|竹內順子]]〈童年時期〉（日本）；[[孫中台|孫中台]]、[[錢欣郁|錢欣郁]]〈童年時期〉、[[劉如蘋|劉如蘋]]〈童年時期，ep.737-738〉（台灣）；[[杜景煜|杜景煜]]（香港）
: 年齡：22歲／生日：3月20日<ref>「薩波噢！」的日文諧音</ref>／星座：雙魚座／身高：100公分（童年）→187公分（成年）／血型：／出生地：東海「哥雅王國」／喜歡的食物：[[拉麵|拉麵]]<ref>海賊王20周年雜誌《ONE PIECE magazine VOL.3》</ref>。
: 革命軍參謀總長，地位僅次於身為領袖的多拉格，被人們稱為「革命軍的第二把交椅」。懸賞金額：6億200萬貝里，是[[蒙其·D·魯夫|蒙其·D·魯夫]]的義兄暨[[波特卡斯·D·艾斯|波特卡斯·D·艾斯]]的義兄弟。於魯夫的回憶篇中首次登場時，頭戴掛有防風鏡的大禮帽，金色短髮，嘴邊缺一顆牙；「大事件」落幕兩年後已成年，蓄著波浪般的金色短髮，左眼殘留年幼時遭天龍人炮擊的燒燙傷疤。由於他只聽重點的個性，身為其部下的可亞拉與哈克對此非常生氣。
: 原為生在哥雅王國的貴族之子，年幼時因為厭倦貴族的生活和發現父母關心的是地位和財產而非他的時候，逃出高區定居於灰色車站生活，和年幼的艾斯為了存當海賊的儲金而一同犯壞勾當。在10歲經歷「波爾謝米事件」後與魯夫和艾斯喝下交杯酒成為結拜兄弟，共同生活數年。
: 後來薩波的父親得知了他的下落，花錢請布魯賈姆海賊團對付魯夫和艾斯，薩波爲了保護他們兩人而被父親強行帶回去，在從乾弟弟斯特里和其他貴族那得知布魯賈姆海賊團與皇族勾結，準備對非確定物終點站縱火的真相後，薩波為了知會魯夫與艾斯而再次逃走，卻遭高城士兵強制驅趕，哥亞王國發生大火的當晚，偶遇革命家多拉格，對多拉格揭露火災背後真相的同時，感嘆自己身處哥雅王國，卻無法得到自由。火災過後，為了能早日脫離貴族世界的控制，薩波偷走一艘小漁船獨自出航，卻因為座船行經天龍人巡訪東海的路徑，遭到天龍人砲擊船隻因而沉海，其後被多拉格暗中帶上船急救，但因炮擊後遺症導致失憶與臉部殘留燒燙傷疤。便跟隨革命軍行動，並一路升至參謀總長，可見實力之強大。在頂點戰爭過後某日，在閱讀報紙時得知艾斯去世的消息，因心底過度悲痛而恢復記憶。
: 薩波在出海時，曾遺留一封信交到魯夫與艾斯的手上，信中提到自己爲了得到自由想早一步出海，但約好3人將來一定會再大海上見面，最後拜託艾斯照顧他們唯一的弟弟魯夫。由於薩波出航時的旗幟標誌為一個打叉的S，艾斯為了紀念薩波，在左臂上自己名字的刺青「ACE」中央也多加了一個打叉的「S」刺青。薩波曾說過，貴族只有到了18歲才能被人們真正地稱為「貴族」，所以他要在17歲出海。於是艾斯和魯夫為了紀念薩波，定下了他們的17歲生日就是出海日。
: 艾斯過世兩年後以革命軍參謀總長身分正式現身，此時的薩波為了找出並摧毀[[多雷斯羅薩|多雷斯羅薩]]的武器工廠，完成革命軍的其中一項任務，與可亞拉一同現身於多雷斯羅薩的鬥牛鬥技場。不過中途在多雷斯羅薩的競技場的決賽中以「魯西」之名，代替魯夫上場爭奪火焰果實，戰鬥中僅以一擊擊潰競技場，趁著混亂中搶奪並立即食用火焰果實後，獲得了火焰果實的能力，用月步登上競技場上空，施展艾斯的經典招式「火拳」，將鬥技場徹底擊毀<ref>ONE PIECE 薩波特別篇 〜3兄弟的羈絆 奇跡的再會和被繼承的意志〜</ref>。
: 比賽結束後，薩波向[[多雷斯羅薩#蕾貝卡|蕾貝卡]]展示自己的真實身份，向蕾貝卡禮貌地自我介紹和拜託後者關照魯夫。隨後被唐吉訶德·多佛朗明哥在多雷斯羅薩內宣布其為3星賞犯（懸賞3億貝里）。其運用火焰果實的能力和武裝色霸氣，以一人之力阻擋海軍上將「藤虎」一笑、數名中將及數千海軍，徒手捏斷海軍中將巴士底的「斬鯊刀」並將其輕鬆擊敗。與藤虎展開決鬥和得知對方的本意之後，在地下港幫助無法脫逃的傷兵，隨後聽到[[革命軍_(ONE_PIECE)#可亞拉|可亞拉]]的訊息，趕往中心街區，救了奄奄一息的魯夫，隨即與伯吉斯展開戰鬥並擊敗對方。在整起事件結束後，隨魯夫等人前往居魯士的住所休養，為魯夫製作了生命紙，自己取下一小片後交予索隆等人，之後便騎乘烏鴉離去。<ref>ONE PIECE動畫EP 737</ref>
: 圓蛋糕島事件落幕後，和革命軍重要幹部閱報獲知魯夫等人偕同杰爾瑪66、火戰車海賊團、太陽海賊團暗殺「BIG MOM」未遂而大幅提升懸賞額的消息，為魯夫等人的成就感到高興。
: 世界會議期間，和摩利、卡拉斯、林德伯格潛入聖地馬力喬亞，為了拯救已無自我意識的[[革命軍_(ONE_PIECE)#巴索羅繆·大熊|巴索羅繆·大熊]]而與海軍上將產生衝突。
: 是惡魔果實自然系「[[ONE_PIECE用語列表#惡魔果實|火焰果實]]」能力者，擁有[[ONE_PIECE用語列表#見聞色霸氣|見聞色霸氣]]和[[ONE_PIECE用語列表#武裝色霸氣|武裝色霸氣]]，使用的武器為鐵製[[水管|水管]]，他所使用出的招式有龍鉤爪以及從火焰果實中得到的「火焰能力」。
: 作者尾田在與同是漫畫家的松本大洋的一場對談中，提到很喜歡對方的《[[惡童當街|惡童當街]]》，該作品影響了魯夫幼時三結義時期的生活描寫，以及薩波使用長水管當武器的設定。
