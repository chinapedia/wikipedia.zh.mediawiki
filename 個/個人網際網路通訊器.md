'''個人網際網路通訊器'''原文是英文的'''Personal Internet Communicator'''（以下簡稱：'''PIC'''），是[[美國|美國]][[超微|超微]]公司於2004年10月提出[https://web.archive.org/web/20051101025215/http://50x15.amd.com/zh-CN/ 50x15計畫]{{Ref|R01}}時，呼應此計畫所推出的個人上[[網際網路|網]]機。PIC強調簡單、低廉，並以[[開發中國家|開發中國家]]為主要的目標市場，如[[金磚四國|金磚四國]]的[[巴西|巴西]]、[[俄羅斯|俄羅斯]]、[[印度|印度]]、[[中國|中國]]等以及[[墨西哥|墨西哥]]，之後也將[[加勒比海|加勒比海]]一帶的中美洲各國包括在內。

== 硬體組件 ==
為了更合乎開發中國家的需求及市場環境，PIC的設計必須盡可能精簡，因此其機內的主要三項硬體組件為：

#超微公司（AMD）的[[Geode|Geode]] GX[[處理器|處理器]]（366MHz運作時脈頻率）
#[[三星電子|三星公司]]（Samsung）的[[DDR_SDRAM|DDR SDRAM]][[記憶體|記憶體]]（128MB容量空間）
#[[希捷公司|希捷公司]]（Seagate）的3.5吋[[硬碟|硬碟]]（10GB容量空間）

其中超微的Geode GX處理器相當省電，典型功耗僅1瓦特，且運作上不需要倚賴散熱風扇，進而使PIC全機可以不使用風扇，無風扇電腦（fanless PC）一直是許多人的期望，特別是過去經常有消費者抱怨電腦風扇的吵雜聲影響使用心情。

除此之外，PIC的週邊硬體規格還包括：

*1個視訊輸出埠（[[VGA|VGA]]介面，最高支援1600x1200[[解析度|解析度]]，以及85Hz垂直更新率）
*1個[[耳機|耳機]]音源輸出孔（AC97介面，Stereo雙聲道）
*1個[[麥克風|麥克風]]音源輸入孔（AC97介面）
*1個傳統類比撥接[[數據機|數據機]]（ITU v.92標準，傳輸率為56kbps）
*4個[[USB|USB]]外接埠（USB 1.1版標準，傳輸率為12Mbps）

值得注意的是，PIC的USB埠並不像一般[[PC|PC]]的USB埠能連接各種週邊裝置，而是有限定的使用，此也是簡化概念下所考量的一部份。依據PIC最初的設定，僅允許PIC的USB埠連接[[印表機|印表機]]、USB介面的外接式硬碟、[[闪存盘|USB隨身碟]]以及USB介面的網路卡（多是指：[[无线局域网|無線區域網路]]的網路卡），若接入上述之外的裝置則不會有動作反應，而上述支援的USB週邊裝置，其對應的[[驅動程序|裝置驅動程式]]也早在PIC出廠前就已預先安裝，不再需要消費者自行假手，如此的設計是期望達到更完全的隨插即用（[[:en:Plug-and-play|Plug and Play]]，[[:en:Plug-and-play|PnP]]）。{{Ref|R02}}

== 軟體組件 ==
PIC的軟體主要有[[Microsoft|微軟公司]]的[[Windows_CE|Windows CE]][[嵌入式作業系統|嵌入式作業系統]]以及[[Internet_Explorer|Internet Explorer]] 6.0[[網頁瀏覽器|網頁瀏覽器]]，除此之外的相關預裝軟體有：

*電子信件收發程式
*[[Macromedia|Macromedia公司]]的[[Flash|Flash]]播放程式
*微軟公司的視窗媒體播放程式（[[Windows_Media_Player|Windows Media Player]]）
*微軟公司的[[PowerPoint|PowerPoint]]電子簡報閱讀程式
*[[PDF|PDF]]檔案的閱讀程式
*微軟公司的即時通訊軟體：視窗傳訊家（[[Windows_Messenger|Windows Messenger]]）
*微軟公司的視窗平面遊戲套包（Windows 2D Game Pack）
*人類軟體公司（[http://www.humansoftware.com/index.shtml HumanSoftware]）的[http://www.humansoftware.com/pages1200/StarMaker/HSstarm11.html SoftMaker]文書處理軟體與試算表軟體，能相容微軟公司的[[Microsoft_Word|Word]]、[[Microsoft_Excel|Excel]]檔案格式。
*圖片影像的閱讀程式，支援.jpg、.bmp、.png及.gif等圖片影像的檔案格式
*壓縮與解壓縮的公用程式（Zip/Unzip）

== 關於價格 ==
在價格上，以上述的PIC軟硬體組合，再加上[[滑鼠|滑鼠]]與當地[[語系|語系]]的[[鍵盤|鍵盤]]等，如此約為185[[美元|美元]]，若還有[[阴极射线管|映像管]]顯示器的需求則總花費約為249[[美元|美元]]。不過價格方面並非是硬性規定，仍可由各地推行業者再行增減調整。

== 營運模式 ==
超微公司對PIC的推行策略是與各國當地的[[網際網路服務提供商|網際網路服務提供商]]（ISP）合作，將超微的PIC硬體產品搭配在地的上網服務一起推廣，而且也允許PIC硬體以該地業者的品牌來銷售。

== 推行成效 ==
超微的推廣方式已在世界各地有諸多成效，如墨西哥的Telmex公司將PIC硬體改名成自屬的Telmex Prodigy後，再搭配Telmex公司的上網服務一同銷售，此外南非的Ucoms公司也將PIC改稱為Ucoms i-khuluma，以此方式與該公司的上網服務一同推廣，其他還有印度的Tata集團（[[:en:Tata_Group|Tata Group]]）、墨西哥的CRC公司、加勒比海地區的[[:en:Cable_&_Wireless|Cable & Wireless]]公司、Doğan Online (DOL)公司、FIC公司、美國Radio Shack公司等也都比照此種模式來推廣，如今PIC已成功打入[[印度|印度]]的五個城市。

值得注意的是，除了上網之外，各地營運業者在更具體的推行上也各有不同取向、不同訴求，例如CRC公司還搭配教學教育軟體一同推行，以教學運用電腦為號召，而Cable & Wireless公司則是作為災變發生時的應急型電腦，可作為救助聯絡之用。

== 新機預覽 ==
2006年的[[台北國際電腦展覽會|台北國際電腦展覽會]]（COMPUTEX）中，超微公司展示了新一代的PIC設計，處理器部分依然是使用Geode GX系列，但記憶體部分增加了一倍的容量，達256MB，硬碟方面使用更嬌小的2.5吋硬碟，且容量增加了三倍，達40GB，此外過去需額外選用擴充的無線區域網路功能，也在新機種上成為基本功能，新的PIC機具有IEEE 802.11g的[[WiFi|WiFi]]無線通訊能力。更重要的是，單機的價格依舊維持在180美元左右。
