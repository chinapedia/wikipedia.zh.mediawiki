# 间壁分离（见于部分[[动物|动物]]和[[植物|植物]]线粒体）：线粒体内[[线粒体内膜|线粒体内膜]]首先形成隔，随后[[线粒体外膜|线粒体外膜]]部分内陷，插入到隔的双层膜之间，将线粒体一分为二。
# 收缩分离（见于[[蕨类植物|蕨类植物]]和酵母菌线粒体）：线粒体中部先缢缩同时向两端不断拉长然后一分为二。
# 出芽分离（见于[[藓类植物|藓类植物]]和酵母菌线粒体）：线粒体上先出现小芽，小芽脱落后成长、发育为成熟线粒体。

线粒体分裂与[[线粒体融合|线粒体融合]]一般保持[[动态平衡|动态平衡]]，<ref>{{cite journal
