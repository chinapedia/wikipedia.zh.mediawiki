'''线列步兵'''（{{Lang-en|Line Infantry}}, {{Lang-fr|Infanterie de ligne}}），或譯'''戰列步兵'''、'''線性步兵'''，在17至19世紀間，歐洲國家在[[野外戰|野戰]]時主要的歩兵運用形態。

同樣的兵種在當時的歐洲各國、中國、印度、新大陸([[大陸軍|大陸軍]])，世界各地都有所編制。

== 創立 ==
线列步兵最早出现在17世纪。在17世纪初，瑞典国王[[古斯塔夫二世·阿道夫|古斯塔夫二世]]决定以[[簧輪槍|簧輪式火器]]武装他的陆军，但只有騎兵裝備了這種結構複雜的武器。他在[[三十年戰爭|三十年戰爭]]期間的優異戰績使得[[線式戰術|線式戰術]]得到全歐洲的矚目。在他逝世後的一段时间，瑞典步兵便以新式簧輪式火枪（Musket）武装，与旧式火枪相比，新式火枪比较轻，更加容易发射，不需要额外帮助。另外，制作新式火枪，需要的铁更少，意味着在大量生产时，更加便宜。這種火器的出现，使得需要大量装备火枪的线列步兵的創立变得可能。

== 战术 ==
[[File:Charge_of_the_French_Cuirassiers_at_Waterloo.jpg|400px]]中向英军方阵冲锋。]]

线列步兵在大多数时候，使用这三种阵型：线列（Line），方阵（Square）与纵阵（Column）。

从17世纪中期开始，步兵广泛采用了手持枪械，<ref>手持枪械（Hand Guns），与[[加农炮|加农炮]]（Cannon）相反；并非[[手枪|手枪]]（Handgun）。</ref>占主导地位的战术是线式战术，将步兵排成宽而浅的线列，然后齐射。一条线列，有二至四排士兵。

因为在近距离内，火枪的准确度能够相对地提高，加上装弹速度的缓慢，<ref>每分钟二至三发。</ref>所以，只有密集阵型的射击才能最大限度地造成敌军的伤亡。线列以发挥最猛火力为目的排列。以鬆散（Skirmish）阵型排列的步兵，虽然能够寻找掩护，而且相对地灵活，但是，在对抗骑兵时十分脆弱，也不能守住阵地，抗击前进中的步兵纵阵。线列步兵向受到威胁撤退的散兵与骑兵提供了支持。

在遭到骑兵包围时，线列步兵能够迅速转变为方阵。这种方阵中间是空的，不同于长矛兵与旧式火枪手的方阵。

即使是训练有素的步兵，以线列阵型行进时，速度仍然会非常缓慢，而且，在行进时，阵型很容易裂开，特别是在通过灌木、森林时。因此，在固定时，步兵通常采用线列阵型，而在行进时，通常采用纵阵。在冲锋、近战时，也会采用纵阵。

== 训练 ==
线列步兵以步兵操典（Manual of arms）为训练指南，主要内容是快速排成一列，快速射击与机动。

线式战术需要严明的纪律与简单的移动，需要士兵熟练指令到下意识的程度。在训练中，演习与体罚经常进行。

线列步兵很快成为了欧洲国家的常见步兵兵种。火枪手（Musketeers）与精锐部队，掷弹兵，逐渐变为线列步兵的一部分，采用起线列战术。

== 武器装备 ==
在17世纪中期，线列步兵的火枪加上了[[刺刀|刺刀]]。刺刀挂在火枪的枪口，在近战时使用。它对抗骑兵时也有帮助。

在17世纪晚期，火枪手被更轻更便宜的、燧发枪兵（Fusilier）代替，他们装备了[[燧发枪|燧发枪]]，重五千克，口径17.5毫米，出现在法国，后来传播到其他国家。在很多国家，燧发枪仍然称为火枪。但是，无论是火枪抑或是燧发枪，都是无膛线的，这使得两者的准确度低、射程短。但是，相应地，由于无膛线，弹丸对人体的创伤比有膛线枪的弹丸高。

线列步兵没有任何防护装备。只有一些精锐部队，能够保留一些有传统元素的防护装备，比如掷弹兵的铜帽。

== 同时代的其他步兵兵种 ==
[[File:Kurhess_Jäger.jpg|200px]]

起初，线列步兵只是步兵的一小部分，因为，它在对抗敌军骑兵时，显得很脆弱。长矛兵（Pikiner）是主力步兵，也被称为重步兵。旧式火枪手是步兵中具有重大意义的部分，但是，当时并未使用线式战术。然而，到了17世纪中叶，在大多数欧洲国家的陆军中，已经有一半是线列步兵。在发明了刺刀后，线列步兵能保护自己，对抗敌军骑兵。长矛兵在陆军中所占的比例大幅下降。在1689年，奥地利陆军废除了长矛兵。在1703年，法国陆军也效法前者，废除长矛兵。在1721至1769年间，[[彼得大帝|彼得大帝]]几乎将所有俄国步兵转为线列步兵。线列步兵成为了欧洲陆军的主力步兵。

当时的兵种，除了线列步兵外，还有精锐部队<ref>皇家卫队与其他精锐步兵团，如英国的冷杉卫队。</ref>与轻步兵。与线列步兵的紧密阵型相反，轻步兵采用松散阵型。在18世纪后期开始，大多数欧洲国家的陆军都有来-{}-福枪兵，<ref>比如德意志国家的猎兵（Jäger）。</ref>以昂贵的线膛[[卡宾枪|卡宾枪]]武装，接受精确射击、寻找掩护的训练。线列步兵装上刺刀的火枪比起卡宾枪重，被称为重步兵，用作主力部队。

法国在大革命与拿破仑战争时期，将步兵分为卫队、线列步兵与轻步兵，但是，线列步兵与轻步兵的武器与战术相同。{{来源请求|拿破仑不信任线膛火器}}。然而，线列步兵与轻步兵都有引入了散兵（Tirailleurs）或尖兵（Voltigeurs），既能以散兵方式作战又能以线列步兵方式作战。

在1853年，俄国步兵由110个团组成。其中52个团是线列步兵团，10个团是卫队，另外46个团是轻步兵团<ref>其中42个团是猎兵团、四个团是卡宾枪团。</ref>。然而，只有部分俄国轻步兵装备了来-{}-福枪。

在19世纪，来-{}-福枪兵在欧洲各国陆军中占的比例提升，而装备火枪的线列步兵的比例下降。在美国内战中的南北双方只有几个线列步兵团。然而，在法国，因为崇拜[[拿破仑一世|拿破仑一世]]的[[拿破仑三世|拿破仑三世]]的干预，在1870年仍有300个线列步兵团<ref>在法国步兵中占压倒性多数。</ref>。即使法国线列步兵在1866年装备了切西普来-{}-福枪（Chassepot Rifle），他们仍被教导以紧密阵型作战：线列、纵阵与方阵，直到拿破仑三世退位后，才改变战术。

== 废除 ==

拿破仑战争后，线列步兵仍然是主力部队，而轻步兵则提供火力支援、掩护行进中的部队。俄国、英国、法国等国家，在19世纪晚期仍使用线式战术，例如[[克里米亚战争|克里米亚战争]]、[[普法战争|普法战争]]。

在新武器发明后，线列步兵学说开始衰落。米林弹（Minie Ball）大大增加了枪械的射程和精确度。紧密阵型中的士兵成为了显著的目标，南北战争就证明了这一点。这场战争后，[[后膛装填步枪|后膛装填步枪]]被采用，使得步兵的射程大大提高，射击的速度也大大加快。在19世纪60年代，大多数德意志国家与俄国都将自己的线列步兵与来福枪兵结合在一起，使用来复枪与散兵战术。在普法战争后，德国、法国也效法前者。然而，英国保留了“线列步兵”的名字，即使他们在1853年开始使用来福枪，在1867年开始使用后膛装填步枪，在布尔战争时由紧密线列改用松散阵型。

来复枪不断提升的射速与在1883年发明的[[马克沁机枪|马克沁机枪]]使得步兵在接近敌军前就会承受巨大的伤亡。骑兵也因为步兵武器的改进而衰落。在20世纪初，步兵开始慢慢接受以轻步兵散兵阵型作战，但是，仍然以线列步兵战术训练。

== “线列步兵”名称的保留 ==
虽然，线式战术在19世纪晚期已经变得过时，但是，在欧洲一些陆军中，步兵团继续归类为“线列步兵”。这一名称意味着这个团是正规军，而不是特种或精锐部队。以1914年的比利时陆军为例，有14个线列步兵团（Infanterie de Ligne），三个轻步兵团（Chasseurs a pied），一个掷弹兵团（Grenadiers）与一个卡宾枪兵团（Carabiniers）。同时期的欧洲陆军也类似于比利时陆军。现代的英国陆军保留了传统名称，如“线列步兵”与“来福枪兵”，即使他们都已现代步枪装备，不采用线式战术。同样地，大多数21世纪的陆军仍然会操练钻型阵型，作为灌输纪律与提高单位凝聚力的一种方式。
