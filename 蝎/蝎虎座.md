'''蝎虎座'''（拉丁文Lacerta，意思是[[蜥蜴|蜥蜴]]）是现代的88[[星座|星座]]之一。在1687年为[[波蘭|波蘭]]天文学家[[赫维留|赫维留]]创立。

没有很亮的星，不包含[[梅西耶天体|梅西耶天体]]、[[视星等|视星等]]亮于14.5等的[[星系|星系]]、[[球状星团|球状星团]]或有独立名字的星。

蝎虎座位于北天，其北部和[[银河|银河]]相交。整个星座位于[[天鹅座|天鹅座]]，[[仙后座|仙后座]]和[[仙女座|仙女座]]之间，和仙后座一样呈W形。包含中国古代星座：车府，螣蛇。

== 神话传说 ==
没有有关的古代传说。[[赫维留|赫维留]]创立本星座之前，有些人把这片星空称为“正义之杖”或者“[[腓特烈二世_(普鲁士)|腓特烈]]的荣耀”。

== 有名的星 ==
*[[蝎虎座α|蝎虎座α]]：A1V型[[主序星|主序星]]，光学[[双星系统|双星系统]]，[[视星等|视星等]]3.77等。
*[[Roe_47|Roe 47]]：实际是由五颗星组成（星等分别是：5.8, 9.8, 10.1, 9.4, 9.8）。

== [[深空天体|深空天体]] ==
*[[NGC_7243|NGC 7243]]：是一个[[疏散星团|疏散星团]]，亮度大约6.4m。
*[[蝎虎座BL|蝎虎座BL]]：[[蝎虎座BL型天体|蝎虎座BL型天体]]的原型。以前曾经被误认为是变星系统。处于活动星系的中心。中心可能是巨大的黑洞。亮度在几天可于12-16等的范围内变化。
