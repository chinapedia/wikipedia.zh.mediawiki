'''諾福克州立大學'''（'''Norfolk State University'''，縮寫：NSU），是位於美國[[弗吉尼亞州|弗吉尼亞州]][[諾福克_(弗吉尼亞州)|諾福克]]的一所[[州立大學|州立大學]]、[[傳統黑人大學|傳統黑人大學]]。2015年《[[美國新聞與世界報道|美國新聞與世界報道]]》將其列在“傳統黑人大學”中的第34位。<ref name="USnews">{{cite web|title=Norfolk State University|url=http://colleges.usnews.rankingsandreviews.com/best-colleges/norfolk-state-university-232937/overall-rankings|publisher=[[美國新聞與世界報道|美國新聞與世界報道]]|accessdate=2015-04-22}}</ref>  

1935年9月18日創立時是[[弗吉尼亞聯合大學|弗吉尼亞聯合大學]]的一個分校區，1942年獨立。<ref name="AboutNSU">{{cite web|url=https://www.nsu.edu/provost/emeriti-faculty/history|title=History|accessdate=2015-01-12}}</ref> 現因財政問題和管理不當而遭到[[南方學院和學校協會|南方學院和學校協會]]的警告。<ref>{{cite web |url=http://www.insidehighered.com/quicktakes/2013/12/11/southern-accreditor-clears-virginia-fisk-florida-am |title=Southern Accreditor Clears Virginia, Fisk, Florida A&M |publisher=[[Inside_Higher_Ed|Inside Higher Ed]] |date=December 11, 2013 |accessdate=December 24, 2013}}</ref>
