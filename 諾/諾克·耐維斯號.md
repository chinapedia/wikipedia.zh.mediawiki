'''諾克·耐維斯號'''（{{lang|en|'''Knock Nevis'''}}）是一艘曾屬於[[挪威|挪威]]籍、已經拆除的[[超大型原油運輸船|超大型原油運輸船]]（Ultra Long Crude Carrier，ULCC）等級之[[超級油輪|超級油輪]]。改建后，它是世界上最長的船隻與人工製造水面漂浮物，船長超過1/4英里（458公尺），比橫躺下來的[[艾菲爾鐵塔|艾菲爾鐵塔]]還長；而其全載重量則高達657,019噸，是[[載重噸位|載重噸位]]（Deadweight Tonnage, DWT）的世界記錄保持者，并从DWT小于50万吨一跃成为有史以来DWT超过50万吨的7艘人工製造水面漂浮物之首（另有4艘Batillus级和2艘Esso atlantic级）。在2004年經營業主易手改名為「諾克·耐維斯」之前，它又曾名為'''海上巨人號'''（{{lang|en|'''Seawise Giant'''}}，1979年），'''快樂巨人號'''（{{lang|en|'''Happy Giant'''}}，1989年），與'''亞勒維京號'''（{{lang|en|'''Jahre Viking'''}}，1991年）。2004年3月該輪進入[[杜拜|杜拜]][[乾塢|乾塢]]進行改裝工程並改名為諾克·耐維斯號，改裝後作為{{link-en|浮動式生產、儲存與卸載油駁|Floating production storage and offloading|浮動式儲存與卸載油駁}}（Floating Storage and Offloading unit，FSO），曾長期停泊在[[卡達|卡達]]的{{link-en|夏辛油田|Al Shaheen Oil Field}}（Al Shaheen Oilfield）做為外海儲油用途。2009年12月，該船被出售给[[印度|印度]]的[[拆船廠|拆船業者]]，改名为'''Mont'''。2010年1月4日，于印度[[古吉拉特邦|古吉拉特邦]]{{link-hi|亞蘭市|अलांग}}（Alang）拆解{{R|Alang}}。其36噸重的[[船錨|船錨]]則保存於[[香港海事博物館|香港海事博物館]]作展覽{{R|Anchor}}。

== 歷史 ==
[[File:Bateaux_comparaison2.svg|thumb]]；3.豪華郵輪[[瑪麗皇后二號|瑪麗皇后二號]]；4.鐵砂散裝貨輪{{link-en|柏格斯達號|MS Berge Stahl}}；5.航空母艦[[企業號航空母艦_(CVN-65)|企業號]]]]
諾克·耐維斯號最早是在1976年12月時，於{{link-ja|住友重機械工業|住友重機械工業}}位於[[日本|日本]][[橫須賀市|橫須賀市]]的追浜造船所（Oppama Shipyard，目前已改名為[[橫須賀製造所|橫須賀製造所]]）起造，船體編號1016號，並獲得海上巨人號（Seawise Giant，1979年）的命名。海上巨人號原本的訂購者是一名希臘船運業者，但他在擁有該船三年後，在船隻尚未完工之前就因破產之故，將這艘船轉賣給了香港籍的船王董浩雲。

{{fact|time=2018-02-05|此船于1974年正式获得订单，原设计为418,611噸（DWT）单船壳(Single Hull)油轮，1975年3月19日起建，仅不到6个月后，1975年9月4日即船壳下水，1976年舾装完成并暂以1016号工厂船壳号为名进行了海试。原计划可与法国“Batillus”级超级油轮竞争，553,662吨（DWT）级的“Batillus”级共4艘，首船Batillus号的建造几乎与1016号同步，于1975年铺设龙骨，1976年舾装完成，随即投入运营（后亦因单船壳船体结构问题，于1983提前退役，1985年拆解）。1016号下水和海试中发现诸多技术问题尤其是船体多处发生振颤，被原希腊订购方拒收，所以也未能以希腊方原先设想的“porthos”号冠名并投入运营，1976年5月26日 住友重機械工業 将其归档为“已交付/Delivered”并以船厂所在地追浜将其首次命名为“追浜/Oppama”号后搁置。

在旷日持久的诉讼中，原希腊订购方因既不能将其投入运营又压滞巨额资金宣告破产，住友重機械工業 不得不寻找新的买家，并于1979年 將此船折价轉賣香港船东董浩云，后首次更名为“海上巨人/Seawise Giant”号，当年投入了一次往返日本与波斯湾的运营。运营中意识到船体问题难以回避，于是又进行了长达2年的改建，主要改建项为将船体改为双船底(Double Bottom)以增强船体结构。最大改建项是针对航行时受激波脱离影响最大的船舯前段，在1980年7月至10月于日本津市耗费3个月加装了一个长达81米的新增的双船壳(Double Hull)舱段；同时希望以增加运营酬载挽回改建损失；但这也使船体过长却未相应加宽加深超出原来应对海浪波长的计算，因此在甲板铺设大量管道，以便在航行中应时调整各舱配载（尽管如此，于1983年仍因结构问题在满载原油航行时船体受损，并因适逢国际油价低迷，一度暂改为海上原油存储平台）。改建期间董浩雲建造資金困難、結局获得中国政府支援。最终形成主要为双船底(Double Bottom)，而小段为包含了双船底的双船壳(Double Hull)结构。}}改建後原本已有418,611噸（DWT）水位的海上巨人號加長了“ 数”公尺(376.7m to 458.45m)，而增加了146,152 噸的水位達564,763噸（DWT），其高達825,614噸的總承載後重量（Gross Loaded Weight，GLW）使海上巨人號正式成為當時世界上最巨大的船隻。兩年後，1981年時，海上巨人號終於改建完成并开始正式运营，比“Batillus”号迟了5年。主要的任務是在[[墨西哥灣|墨西哥灣]]與[[加勒比海|加勒比海]]一帶運輸[[原油|原油]]。

當時正進行中的[[兩伊戰爭|兩伊戰爭]]嚴重地影響了[[波斯灣|波斯灣]]地區[[石油|石油]]的生產與運輸，因為[[伊拉克|伊拉克]]宣佈任何載運[[伊朗|伊朗]]（或其他[[阿拉伯國家|阿拉伯國家]]）所生產原油的船隻都將成為伊拉克襲擊的目標，但海上巨人號仍然冒險進入波斯灣海域進行運輸工作，終於招致不幸，在1988年5月14日航經[[荷姆茲海峽|荷姆茲海峽]]時，遭伊拉克戰機以法製[[飞鱼反舰导弹|飛魚反艦飛彈]]攻擊重創，坐底沉在伊朗的[[哈爾克島|哈爾克島]]（Kharg Island）海岸外之淺海海域。所幸当时所处海域水深较浅，海底也较平坦，海上巨人号仍有部分露于水面，也可说处于搁浅状态。且正处于前往伊朗装载原油途中，为空舱，仅具数千吨燃料用重油，未酿成严重生态灾难。沈沒後的海上巨人號就以這狀態一直存留在海底，直到兩伊戰爭結束。

1989年，海上巨人號被以3千5百萬美元的代價轉賣給[[挪威|挪威]]的聯合私募基金諾曼國際（Norman International），船隻在[[打撈|打撈]]起來後被拖至[[新加坡|新加坡]]的[[吉寶企業|吉寶造船廠]]（Keppel shipyard），自1990年10月起進行大規模修復工作，這次的維修需要3200噸的[[鋼|鋼]]原料，更換掉的管線長達32公里，需要6千萬美元的修繕費用。在經過一場堪稱全世界最大規模的船隻修復工程後，這艘船被改名為快樂巨人號（Happy Giant）後重新復出。

諾曼國際實際營運快樂巨人號的時間並不久，1991年時他們將這艘船以3千9百萬美元的價格賣給另一家挪威船公司{{link-no|約根·亞勒|Jørgen Jahre|亞勒海運}}（Jahre），而該船也同時改名為亞勒維京號（Jahre Viking）。在歷經十餘年的服務後，亞勒維京號被轉賣給了新加坡籍的{{link-en|第一奧森油輪|First Olsen Tankers}}（First Olsen Tankers），並且改名為諾克·耐維斯號（Knock Nevis），該公司與卡達的{{link-da|快桅石油|Maersk Oil}}（Maersk Oil）簽訂一紙合約，租賃這艘船三到五年的時間，停在夏辛油田作為海上儲藏與裝卸原油的設施。

為此諾克耐維斯號在2004年3月進入卡達乾塢進行FSO化的改裝，改裝項目包括主要管線的重設，新的泊靠系統，與一個[[直昇機停機坪|直昇機停機坪]]的裝設等，並於6、7月間開始啟用。

2009年12月，結束FSO用途後，船體被卖给印度，命名为“Mont”。2010年1月4日，于印度古吉拉特邦的亞蘭市拆解。

== 相關數據 ==
[[File:Comparison_of_Knock_Nevis_with_other_large_buildings.svg|thumb]]之高度比較]]
諾克耐維斯號能夠容納將近650,000立方公尺的原油（將近410萬桶），利用其3.5吋厚、由住友造船開發的雙層船體（Double Hull）技術，來防止原油意外滲漏污染環境的可能。然而，滿載後的這艘船吃水超過24公尺，因此無法通過水深較淺的世界主要航道，例如[[英吉利海峽|英吉利海峽]]，[[蘇伊士運河|蘇伊士運河]]就更不用說了。另外，由於這樣的吃水深，滿載時的諾克·耐維斯號並無法進入世界大部分的主要港口，而需要特殊的接駁設施，在外海直接卸載原油。

諾克耐維斯號是世界上最巨大的船隻，其高度的自動化設計與電腦輔助，僅需用35至40名的船員就能順利航行。
