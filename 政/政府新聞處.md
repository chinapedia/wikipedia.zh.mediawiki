'''政府新聞處'''（英文：'''I'''nformation '''S'''ervices '''D'''epartment，縮寫：'''ISD'''）是[[香港|香港]][[民政事務局|民政事務局]]轄下的[[香港特別行政區政府部門|部門]]，專責處理[[公共關係|公共關係]]。

== 歷史 ==
1951年'''政府公共關係處'''成立，成立初期總部設在舊[[香港郵政總局|香港郵政總局]]（現址為[[環球大廈|環球大廈]]）。1954年4月改為'''政府新聞統籌處'''，總部搬到[[中區政府合署|中區政府合署]]西座，直至1959年改為現稱，1963年6月8日搬遷到中環[[皇后大道中|皇后大道中]][[拱北行|拱北行]]（[[長江集團中心|長江集團中心]]前身）一共渡過了接近35年時光。隨著1995年12月開始拱北行的清拆，總部又遷移到[[美利大廈|美利大廈]]。1997年初，政府新聞處正式接管[[資訊統籌署|資訊統籌署]]資源，政府新聞處處長亦負責掌管這個合併後的資訊架構<ref>http://www.legco.gov.hk/yr97-98/chinese/panels/ip/papers/ip2301_4.htm</ref>。2012年8月29日，總部再遷至[[海港政府大樓|海港政府大樓]]。

== 職能 ==
[[File:HK_Central_政府新聞處_Information_Services_Department_ISD_poster_box.JPG|thumb]]
政府新聞處作為政府的公共關係顧問，負責政府的[[出版|出版]]、[[政府宣傳片|政府宣傳片]]及[[新聞|新聞]]工作，是政府與香港及外國傳播媒介間溝通的橋樑。新聞處亦透過傳媒把政府政策向市民進行宣傳，及提高市民對政府工作的認識。

政府新聞處亦負責[[香港品牌|香港品牌]]飛龍標誌，及宣傳香港作為「亞洲國際都會」[[品牌|品牌]]宣傳計劃的統籌及管理工作。

== 旗下網站 ==
;香港政府新聞網
香港政府新聞網由政府新聞處開設，包含網上新聞、特寫及評論等內容。網站由政府新聞處的撰稿員及編輯負責製作，用以透過特寫、圖片、短片等多媒體內容及連結其他網頁，提供背景及相關資料，在發佈新聞資訊的同時，促進政府與工商界、學術界及社會各界人士之間進行交流。編輯小組會及時將香港政府的消息及背景資料在網站上發放，盡快更新有關內容<ref>{{cite web|url=http://www.news.gov.hk/tc/about/index.shtml|title=關於我們：關於本網站|work=[[香港政府新聞網|香港政府新聞網]]|accessdate=2013-02-07}}</ref>。

;香港政府新聞公報
{{main|香港政府新聞公報}}
香港特區政府新聞公報（或稱新聞公報），是香港政府新聞處所發行的刊物，曾以印刷模式發行，現時則主要經互聯網發佈，該署於1995年12月起設立網頁上載新聞公報內之新聞稿，並於1998年起負責製作、管理、開設網上新聞稿發佈網站，當新聞稿發送到傳媒之後，便會同時上傳至該網頁，供公眾人士查閱。<ref>{{cite web|url=http://www.isd.gov.hk/chi/notices.htm|title=重要告示：關於本網站|work=政府新聞處|date=2010年10月6日|accessdate=2013-02-07}}</ref>

== 架構 ==
新聞處屬下設有本地公共關係科、宣傳及推廣科、香港以外地區公共關係科及行政科。新聞處設有一位處長，另設有兩名副處長及6位助理處長，並有兩名新聞秘書，分別協助[[政務司司長|政務司司長]]及[[財政司司長|財政司司長]]處理對傳媒的工作。巧合的是，3名前處長現已成為林鄭月娥班子的局長。

本地公共關係科屬下的新聞組，新聞組負責向傳媒及公眾發放政府消息；而大眾傳播研究組則負責向政府各部門反映香港各主要[[報章|報章]]、[[電視|電視]]及[[電台|電台]]等傳媒發表的[[社論|社論]]等意見。
