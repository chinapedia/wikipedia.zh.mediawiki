'''政治指導區'''，是民國二十八年（1939年）[[西康省|西康省]]寧屬屯墾委員會成立後，經省府批准，陸續在各縣彝族聚居區或彝漢雜居區設置了一批政治指導區，為建縣准備。有的由屯委會直轄，有的屬縣或設治局，不時設、並、撤。共分三種。<ref>「寧屬的政治指導區」，[[李仕安|李仕安]]，中國人民政治協商會議涼山彝族自治州委員會文史資料委員會編.《涼山彝族自治州文史資料選輯》（第9輯）,1991年03月第1版.第159-162頁</ref>

== 模範政治指導區 ==

北山模範政治指導區，其轄境南接[[西昌|西昌]]城，1950年後為[[紅毛螞姑彝族自治區|紅毛螞姑彝族自治區]]，是涼山的第一個區域自治地區。現劃歸[[喜德縣|喜德縣]]。

== 特別政治指導區 ==

# 拖烏特別政治指導區，區署設在樂西公路線上的<u>拖烏</u>。從[[冕寧縣|冕寧縣]]北部劃設。
# 瀘寧特別政治指導區，區署設<u>瀘寧營</u>(俗呼二四營)。是從[[冕寧縣|冕寧縣]]北西部劃設的。後改為設治局、解放初並設[[金礦縣|金礦縣]]。金礦縣治設木里庄，後來併入[[冕寧縣|冕寧縣]]。
# 普雄特別政治指導區，因受軍政力量的局限，區長進不了普雄，只好把區署設在昭覺西北與普雄隔一座大山的林密地，而僅在下普雄距[[越嶲|越嶲]]城只十多里的<u>胎西</u>地方設辦事處。
# 腴田特別政治指導區，由[[越嶲|越嶲]]東北部靠近田壩場附近的彝村劃設，區署設腴田村。
# 寧西特別政治指導區，由[[西昌縣|西昌縣]]的普格區同<u>忠良</u>、<u>馬雄</u>兩個指導區合併而成。區署設普格場；以後升為[[普格設治局|普格設治局]]，後改[[普格縣|普格縣]]。
# 黃草特別政治指導區，區署設在[[鹽源縣|鹽源縣]]中所地方的黃草壩。1942年尚是政治指導區，後升為特別政治指導區。

== 政治指導區的作用 ==

{{Fact}}設區後，彝族頭人當上了鄉長、大隊長之類，有屯委會給他們撐腰，不再怕鄧秀廷剿殺。百姓在區內來往買賣也不成問題。而未設區的地方彝人被列為「野彝」連附近鄉場都不敢去，怕官府找麻煩。設區地方的治安秩序也比漢區好得多，很少發生搶劫事件。
