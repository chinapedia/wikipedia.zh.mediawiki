'''《閃耀維港》3D 光雕匯演'''為由[[香港旅遊發展局|香港旅遊發展局]]所舉辦的[[3D|3D]][[光雕|光雕]]匯演，於2016年及2017年特定日子假[[香港|香港]][[九龍|九龍]][[油尖油區|油尖油區]][[尖沙咀|尖沙咀]]舉行，直至同年2月14日為止；透過3D[[投影|投影]]技術將動畫投射在[[前九廣鐵路鐘樓|前九廣鐵路鐘樓]]及[[香港文化中心|香港文化中心]][[外牆|外牆]]上演，展現香港特色事物，包括期間是[[香港FUN享夏日禮|香港FUN享夏日禮]]、[[香港繽紛冬日節|香港繽紛冬日節]] 及 [[香港新春節慶|香港新春節慶]]  盛事，附以燈光以及音樂（現場播放以及透過[[流動應用程式|流動應用程式]]收聽），為在[[尖沙咀文化中心露天廣場|尖沙咀文化中心露天廣場]]和隔岸在[[中環海濱長廊|中環海濱長廊]]及[[中環海濱公園|中環海濱公園]]的觀眾帶來[[視覺|視覺]]及[[聽覺|聽覺]]享受。在香港舉辦盛事及正值特定假期或者特別日子期間，匯演亦會配以相關主題內容和應。

== 演出時間、主題內容 ==
閃耀維港舉辦期間每日晚上上演4次，場次分別為晚上8時20分、8時45分、9時15分及9時45分，每節約8分鐘。
*2016年8月5日至28日
[[香港FUN享夏日禮|香港FUN享夏日禮]]
*2016年11月25日至12月28日[[香港繽紛冬日節|香港繽紛冬日節]]
*2017年1月17日至2月14日 (由1月27至29日除外)[[香港新春節慶|香港新春節慶]]

== 歷史 ==
為了提升[[香港旅遊業|香港旅遊業]]的競爭力及[[香港|香港]]的吸引力，[[香港旅遊發展局|香港旅遊發展局]]於2015
年耗資2,500萬[[港元|港元]]籌備閃耀维港3D光雕匯演<ref>[http://www.singpao.com/xw/gat/201412/t20141224_542419.html 3D光雕匯演 讓維港更閃耀] {{webarchive|url=https://web.archive.org/web/20150402115845/http://www.singpao.com/xw/gat/201412/t20141224_542419.html |date=2015-04-02 }} 《成報》 2014年12月24日</ref>，由參與過[[2008年夏季奧林匹克運動會開幕式|2008年夏季奧林匹克運動會開幕式]]及[[中國2010年上海世界博覽會開幕式|中國2010年上海世界博覽會開幕式]]燈光設計的[[澳大利亞|澳大利亞]]公司Spinifex Group創作。

== 環境保護 ==
閃耀維港融入低碳排放[[環境保護|環境保護]]措施<ref>[http://news.takungpao.com.hk/paper/q/2014/1025/2801591.html 3D光雕下周三闪耀维港] 《大公報》 2014年10月25日</ref>，[[香港旅遊發展局|香港旅遊發展局]]委託「低碳亞洲」的碳排放審計師於活動期間實地監察其碳排放量，並且提供減少碳排放量的建議。至於無可避免的碳排放量，則透過支持雲南芒里的水電項目為賠償安排<ref>[http://www.singpao.com/xw/gat/201410/t20141025_533713.html 3D光雕匯演閃耀維港] {{webarchive|url=https://web.archive.org/web/20150402093738/http://www.singpao.com/xw/gat/201410/t20141025_533713.html |date=2015-04-02 }} 《成報》 2014年10月25日</ref>。
