'''東海大學圖書館'''（[[英文|英文]]：Tunghai University Library），是[[東海大學_(台灣)|東海大學]]附設的圖書館，於1955年設立，提供東海大學教職員生中西文書籍借閱、[[期刊|期刊]]、[[電子書|電子書]]、視聽資料等服務，並設有閱覽室、研討室、附設餐廳等，是[[臺灣|臺灣]]最早實施開架制度的[[大學圖書館|大學圖書館]]。

圖書館舊址為現今的行政中心，1985年位於文理大道頂端的新圖書館落成啟用，此外還包括2008年落成、位於第二教學區的「管院分館」。

== 歷史沿革 ==
* 1955年：創立。<ref name="history">{{cite web|language=zh-tw|url=http://www.lib.thu.edu.tw/web/about/detail.php?scid=1&sid=1|title=歷史沿革|author=|publisher=|date=2013-06-11|accessdate=2013-12-06}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>
* 1983年12月：新館開工。<ref name="history"/>
* 1984年10月：新館落成。<ref name="history"/>
* 1985年4月：新館啟用。<ref name="history"/>
* 2008年9月：圖書館管院分館開館。

== 館藏特色 ==
總館館藏以配合學校師生教學研究需要為主，並兼顧休閒活動所需。此外，古籍的庋藏，其中不乏手稿、善本，尤以刊於[[宋朝|宋]][[紹興_(南宋)|紹興]]十年（西元1140年）之西漢文類，乃罕見之善本，彌足珍貴。管院分館館藏則以[[管理學|管理學]]類科、餐旅管理學類科之中、西文圖書與期刊，以及[[音樂學|音樂學]]類科[[樂譜|樂譜]]和視聽資料為主。

== 服務項目 ==
*館內閱覽
*圖書外借
*參考諮詢
*電子資料庫檢索
*指定參考書
*縮影資料
*館際合作
*複印服務
*資訊檢索服務
*有線及無線上網服務
