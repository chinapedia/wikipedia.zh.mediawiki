'''東鹿越車站'''（{{jpn|j=東鹿越駅|hg=ひがししかごええき|rm=Higashi-shikagoe eki|f=y}}）是一座位於[[日本|日本]][[北海道|北海道]][[空知郡|空知郡]][[南富良野町|南富良野町]]字東鹿越，由[[北海道旅客鐵道|北海道旅客鐵道]]（JR北海道）與日本貨物鐵道（JR貨物）所共同經營的[[鐵路車站|鐵路車站]]。東鹿越是[[根室本線|根室本線]]沿線車站之一，車站編號'''T35'''，{{link-ja|電報略號|電報略号 (鉄道)}}為'''{{lang|ja|ヒコ}}'''（HiKo）。

== 歷史 ==
* 1941年（[[昭和|昭和]]16年）12月29日 - [[日本國有鐵道|日本國有鐵道]]開設'''東鹿越信號場'''，開始提供專用線上貨運列車的貨物收發服務。
* 1946年（昭和21年）3月1日 - 升格為'''東鹿越車站'''，為{{link-ja|一般車站|日本の鉄道駅#一般駅}}。
* 1982年（昭和57年）11月15日 - 除了行李及專用軌道的貨物收發外，停辦{{link-ja|車攜貨運|車扱貨物}}業務。
* 1987年（昭和62年）4月1日 - [[國鐵分割民營化|國鐵分割民營化]]後，本站的經營由JR北海道繼承。
* 1994年（[[平成|平成]]6年）4月1日 - 由{{link-ja|北海道旅客鐵道釧路分社|北海道旅客鉄道釧路支社}}改為北海道旅客鐵道總社直屬的{{link-ja|北海道旅客鐵道鐵道事業本部|北海道旅客鉄道鉄道事業本部|鐵道事業本部}}管轄。
* 1997年（平成9年）3月 - 伴隨貨運列車的停駛，車站無人化。

== 車站構造 ==
東鹿越車站為擁有1座[[島式月台|島式月台]]及2條鐵路路線的地面車站，由於站內並未設置[[人行天橋|跨線橋]]，因此月台間是用[[平交道|平交道]]連接。東鹿越雖已降級為無人車站，但站內仍保留著當初設有站員時代時所使用的站房構造。

== 貨物處理 ==
東鹿越目前僅用於{{link-ja|車攜貨運|車扱貨物}}的臨時收發站用途，既無貨運列車的起訖，也沒有配置任何貨運{{link-ja|專用鐵路線|専用鉄道|專用線}}。但在過去，本站曾配置有連接{{link-ja|日鐵礦業|日鉄鉱業}}東鹿越採礦場的專用線，除了礦石的運送外，該專用線也被用於輸送[[石灰岩|石灰岩]]至[[中斜里車站|中斜里車站]]，以供{{link-ja|北連農業協同組合聯合會|ホクレン農業協同組合連合会|北連}}（{{lang|ja|ホクレン農業協同組合連合会}}）中斜里製糖工廠製造[[食糖|砂糖]]時使用。在專用線營運末期每日尚有1班往返，但相關業務已於1997年3月廢除。

此外，本站也曾配置有連往{{link-ja|王子木材綠化|王子木材緑化}}鹿越採礦場的專用鐵路線。

== 車站周邊 ==
車站前面為廣闊的[[湖泊|湖泊]]，而另一面是工廠。
* {{link-ja|金山湖|金山ダム}}（{{lang|ja|かなやま湖}}）：由{{link-ja|空知川|空知川}}上游{{link-ja|金山水壩|金山ダム}}所圍城的人工湖泊。
* 日鐵礦業東鹿越採礦場
* 王子木材綠化鹿越採礦場
* [[南富良野町|南富良野町]]營循環巴士「東鹿越車站前」站牌
