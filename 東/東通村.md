'''東通村'''（{{jpn|j=東通村|hg=ひがしどおりむら|rm=Higashidōri mura|f=y}}）是[[青森縣|青森縣]][[下北半島|下北半島]]東北部的村。

== 歷史 ==

* 1889年（明治22年）施行町村制，大利村、目名村、蒲野澤村、野牛村、岩屋村、尻屋村、刷勞村、猿森村、小田野澤村、白糠村、砂子又村、田屋村合併成'''東通村'''。村公所設置於田名部町（現在的[[陸奧市|陸奧市]]）。
* 1988年（昭和63年）東通村100周年記念，村公所的辦公大樓改為設置於村內。

== 地理 ==
位於面向太平洋及津輕海峽的下北半島的東北。東北的盡頭為尻屋崎，岬附近為桑畑山。村內佈滿地形平坦且標高很低的山。東面海岸附近有很多沼澤，亦有很多流向大海的河川。
村落鬆散地於本村聚集，但以內陸部份作為本村的中心地帶，而核心為本村的行政施設及體育館，並形成住宅團地。此外，本村的東部的太平洋側有一座東通核能發電所。
* 山：桑畑山、片崎山
* 河川：田名部川
* 湖沼：小沼、妹沼、長沼、大沼、タテ沼、左京沼
=== 鄰近的自治體 ===
* [[陸奧市|陸奧市]]
* [[上北郡|上北郡]]：[[横濱町|横濱町]]、[[六所村|六所村]]

== 行政 ==
* 以前以本村的道路未準備好作為主要原因，而將村公所設置於本村之外（[[陸奧市|陸奧市]]），1988年，即本村成立100周年記念，以此為契機將村公所搬遷至本村。
* 由於本村興建[[東通核電廠|東通核能發電所]]，因此有較穩定的財政來源，並採取不與周邊市町村合併為本村方針。但本村現在的面對的問題是興建不少住宅但不能售出，造成大幅度的財政赤字，令原定想發展為町的計劃不能夠繼續進行。
