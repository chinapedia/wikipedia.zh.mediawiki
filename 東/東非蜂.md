==生物特性==
和[[歐洲蜜蜂|歐洲蜜蜂]]（Apis mellifera）相比，最顯著的是，東非蜂（Apis mellifera scutellata）圍繞[[蜂巢|蜂巢]]、成群叮咬[[敵人|敵人]]的[[防守|防守]]行為。東非蜂體型也稍微小一些，其差異只能使用[[顯微鏡|顯微鏡]]確定。歐洲的蜜蜂比東非蜂更能承受寒冷的[[溫度|溫度]]。[[歐洲蜜蜂|歐洲蜜蜂]]一般每年能[[繁殖|繁殖]]兩次至三次，但東非蜂每年繁殖次數則高達17次。<ref name="Apis mellifera scutellata (Lepeletier)">{{Cite web |url = http://www.issg.org/database/species/ecology.asp?si=325&fr=1&sts=&lang=TC |title = Apis mellifera scutellata (昆蟲) |author = Mark L. Winston, Professor and Fellow, Morris J. Wosk Centre for Dialogue, Canada. |date = 2010-09-21 |publisher = National Biological Information Infrastructure (NBII) & IUCN/SSC Invasive Species Specialist Group (ISSG)
Updates on management information with support from the Overseas Territories Environmental Programme (OTEP) project XOT603, a joint project with the Cayman Islands Government - Department of Environment |language = en |accessdate = 2015-08-03 |quote = 不同於歐洲蜂（Apis mellifera），東非蜂（Apis mellifera scutellata）不囤積蜂蜜越冬。相反地，他們依靠蜂群流動（遷徙與分群），以尋找新的食物來源，水或空間。 }}</ref>

東非蜂需面對惡劣乾燥多變的[[環境|環境]]與大自然許多的[[掠食者|掠食者]]而演化出非常強烈的性情，環境不利容易棄巢[[遷徙|遷徙]]。東非蜂相對於其他蜜蜂亞種投注更多的資源在蜂群的增殖上，即使[[天災|天災]]使大多數的蜂群滅亡，少數存活的蜂群仍能逐漸恢復蜂群數量。東非蜂體質強健，對大部分的蜜蜂[[疾病|疾病]]都有抵抗性，這些特性非常受到養蜂者看重。

東非蜂雖然個性兇猛，但兇猛的特性在蜜蜂亞種內不算特別，兇猛的蜜蜂亞種不少，例如[[歐洲黑蜂|歐洲黑蜂]]的雜交種也很兇猛。東非蜂在生物特性上最重要且最不為人注意的特性就是：含有東非蜂血統的[[蜂王|蜂王]]從卵成長至羽化只需15天，而其他亞種的蜂王需16天才可羽化，而這少一天的成長期令東非蜂在美洲的擴散如入無人之境，也才造成美洲的蜜蜂非洲化的問題。

=== 外觀及解剖 ===
學者 Francoy 等人利用新形態的分析技術，分析[[工蜂|工蜂]]的前翅，以鑑別歐洲蜜蜂和東非蜂的差異<ref name="Apis mellifera scutellata (Lepeletier)" />。東非蜂比[[義大利蜂|義大利蜂]]稍微小一些，但外觀與義大利蜂相同，工蜂腹部也是黑黃相間，蜂后的腹部顏色是橘色，腹部背板有大塊的黑斑。

=== 亞種爭議 ===
東非蜂，學名A．m．scutellata Lepeletier。廣泛分布於[[中非|中非]]、[[東非|東非]]大[[草原|草原]]和[[南非|南非]]的大部分地區。分蜂性強，經常[[遷徙|遷徙]]，在[[洞穴|洞穴]]和露天營巢。[[西非蜂|西非蜂]]，學名A．m．adansonii Latreille。發現於[[西非|西非]]。在許多習性上與東非蜂相似。關于東非蜂和西非蜂是否為兩個不同的[[亞種|亞種]]，以及1956年引入[[巴西|巴西]]的非洲蜜蜂到底是哪一個亞種，在[[學術界|學術界]]尚有分歧。<ref>{{Cite book | author = 中國農業百科全書總編輯委員會 養蜂卷編輯委員會 編 | title = 中國農業百科全書：養蜂卷 | location = 北京市 | publisher = 农业出版社 | date = 1993-06-01 | pages = 50 | ISBN = 7109024911 | accessdate = 2015-08-04 | url = http://art.tze.cn/Refbook/entryPrint.aspx?bi=m.20080317-m300-w001-071&ei=5EE15FC2740914DFB1D2681CFE1A65AE90C2F4311874F2E27DA1E42A3B855528201E42512E5719D4&cult=US&et=&Token=122D040E2283F72A55B596CCF0D41E5B | language = zh-cn | quote = 其他非洲蜜蜂，即摩洛哥大蜂(A．m．major Ruttner)、撒哈拉蜂(A．m．sahariensis Baldensperger)、蘇丹小蜂(A．m．nubica Ruttner)、坦桑海濱蜂(A．m．littorea Smith)、單色蜂(A．m．unicolor Latreille)和也門蜂(A．m．jementica Ruttner)等，僅發現于某些局部地區，只測定了少數標本，對它們的生物學特性尚在研究之中，關于它們的分類地位還不能作肯定的結論。 }}</ref>

== 一般影響 ==
[[File:Apis_mellifera_subsp_scutellata,_nesingang,_Groenkloof_NR.jpg|thumb]]

東非蜂是具攻擊性的蜜蜂，[[防衛|防衛]]蜂巢時離巢50[[英尺|英尺]]叮咬入侵者，追逐入侵者到一[[英里|英里]]遠。[[人類|人類]]和其他[[動物|動物]]，通常只有在無法脫身的情況下才會被蜜蜂殺[[死|死]]。遭受東非蜂攻擊的受害者，通常會承受比歐洲蜂攻擊，多十倍以上的叮刺，東非蜂對[[干擾|干擾]]的[[反應|反應]]比歐洲蜂快十倍。公眾對蜜蜂和養蜂行業的偏見，造成不幸的結果，使[[媒體|媒體]]過度恐懼，一再攻擊東非蜂。這可能導致呼籲禁止在市區飼養蜜蜂，因而空出蜜蜂合適的棲地。東非蜂會填補這些空隙，並建立大量的族群，造成更多的問題。<ref name="Apis mellifera scutellata (Lepeletier)" />

養蜂業的[[勞動力|勞動力]][[成本|成本]]提高，因為[[工人|工人]]採收東非蜂的意願低。有些[[生態系統|生態系統]]可能會處於危險之中，因為東非蜂會侵入[[領域|領域]]，競逐本地授粉[[昆蟲|昆蟲]]。東非蜂最大的影響是對[[經濟|經濟]]和[[生態|生態]]的影響。東非蜂的[[授粉|授粉]]效果也比歐洲蜂差。東非蜂數量的增加，會使蜂蜜業（在[[美國|美國]]每年 1.4億[[美元|美元]]產值）處於[[風險|風險]]之中，以及造成[[農作物|農作物]]和[[果園|果園]]，普遍減少授粉，在美國每年估計損失100億至140億美元以上。<ref name="Apis mellifera scutellata (Lepeletier)" />

==雜交品種的利用==
{{see also|非洲化蜜蜂}}

東非蜂及含有其血統的雜交種最著名的特性就是兇猛的性情，最著名的例子就是美洲的[[殺人蜂|殺人蜂]]<ref>{{cite news |language = zh |author = 錢美臻 |coauthors =  |url = http://dailynews.sina.com/bg/news/usa/uslocal/singtao/20120623/07263499730.html |title = 殺人蜂攻擊 南加狗群遭狂叮 |publisher = 星島日報 |pages =  |date = 2012-06-23 |accessdate = 2015-08-05}}</ref>。東非蜂在非洲原產地已是蜜蜂養殖的品種，非洲農民也早已適應與管理非洲蜂，美洲的養蜂者遽然接觸兇猛的非洲蜂很難以接受，但也了解無法根除含有非洲蜂血統的蜜蜂，經過多年的適應後已漸漸學會如何管理養殖的東非蜂，許多國家都有品種改良計畫，[[巴西|巴西]]已出口不少改良蜂王至其他中南美國家。中南美洲國家適應了與非洲化蜜蜂相處並利用他的優點，造成產蜜量大增，算是達成當初引入的目的。<ref>{{Cite web |url = http://www.jiaer.cn/shownews.asp?id=168 |title = 巴西養蜂業 |author =  |date =  |publisher = 重慶佳爾生物技術研究所 |language = zh-cn |accessdate = 2015-08-04 |quote = 最近20年里，巴西的养蜂设备和蜂产品迅速增长，不但能自给而且还出口不锈钢摇蜜机、割蜜刀、巢础以及喷烟器、工作服等。蜂产品有特种蜜、蜂胶、花粉用于化妆品和保健品。养蜂协会纷纷成立而且活动很多。庆祝养蜂协会成立25周年的刊物发行5000册，每册印有“甜蜜的信息”字样，还有格言：“我们并不富足，我们组织起来”。 }}</ref>

為防止東非蜂（Apis mellifera scutellata）與大規模商業飼養的[[歐洲蜜蜂|歐洲蜜蜂]]（A. mellifera）[[雜交|雜交]]，人們釋放大量歐洲蜜蜂的[[雄蜂|雄蜂]]，以降低與東非蜂[[交配|交配]]的可能性。也經常讓養殖蜜蜂分群重新產生[[蜂后|蜂后]]，確保其[[族群|族群]]持續維持。另外一個方法是：把歐洲蜜蜂的[[精子|精子]]注入東非蜂的蜂后，產生較不具[[攻擊|攻擊]]性的蜜蜂。也有許多國家試圖使用[[化學品|化學品]]和[[陷阱|陷阱]]來摧毀東非蜂族群，但[[效果|效果]]不佳。<ref name="Apis mellifera scutellata (Lepeletier)" />

學者 Ojar 表示，經由[[交配|交配]]產生[[近交|近交]]系的族群。這些自交系蜂群的攻擊性增加，能耐[[冬天|冬天]]低溫，更會頻繁地棄巢分群。學者 Masterson 認為，東非蜂的棄巢率，比歐洲蜂高很多，已被記錄的棄巢率高達10％。東非蜂比歐洲蜂投入在儲存蜜蜂的[[能量|能量]]較少，它把更多的能量投入在[[生育|生育]]。由於這些屬性，東非蜂不適用於養蜂業。<ref name="Apis mellifera scutellata (Lepeletier)" />

== 海角蜂與東非蜂 ==
東非蜂體質強健，對大部分的蜜蜂[[疾病|疾病]]都有抵抗性，但對於[[海角蜜蜂|海角蜜蜂]]（A. m. capensis）<ref name="海角蜂與東非蜂" />的入侵卻無能為力，常造成整群蜜蜂滅亡。

[[海角蜜蜂|海角蜜蜂]]滅亡東非蜂的機制是，[[侵入|侵入]]東非蜂蜂群內[[產卵|產卵]]，並培育出海角蜂的新蜂王取代東非蜂原蜂王，造成蜂群混亂，互相鬥殺，最後導致蜂群毀滅。這些未經[[授精|授精]]的[[卵|卵]]，很大部分能發育成雌性蜂，並能培育出[[蜂王|蜂王]]來。而少數孵化的海角蜂又會飛至鄰近的蜂巢重覆以上的[[循環|循環]]，造成更多的蜂群滅亡。<ref>{{zh-cn}} [http://www.360doc.com/content/11/0131/04/5043743_90068487.shtml 开普敦的海角蜜蜂 - A.m.capensis]</ref>

1990年發生海角蜂毀燒事件，自然狀況下海角蜂與東非蜂因[[山脈|山脈]]的阻隔彼此相安無事，但[[人類|人類]]轉地放蜂的養蜂模式使海角蜂入侵東非蜂的[[領地|領地]]並造成嚴重的損失。使東非蜂原產地的蜂農無法採收[[蜂蜜|蜂蜜]]和[[蜂蜡|蜂蜡]]，也因缺少蜜蜂的授粉，農業損失3.5億美元，南非政府農業局已經採取強烈措施，對受到海角蜂侵擾一律銷毀，未經批准，蜂群不得搬出指定地區，並對重新購買蜂群者給予[[補助|補助]]。<ref>{{zh-tw}} [http://www.101975.tw/tab/435/id/8026/ 蜜蜂生態-福昶養蜂育種場]</ref>

== 疾病與病蟲害 ==
艾留德‧穆立是國際昆蟲生理學與生態中心的[[蜜蜂養殖|蜜蜂養殖]]專家，他與美國[[賓州州立大學|賓州州立大學]]的研究人員合作，調查[[肯亞|肯亞]]所有主要生態系的[[蜂巢|蜂巢]]，包括[[草原|草原]]、[[山區|山區]]、[[熱帶|熱帶]][[海岸|海岸]]與[[沙漠|沙漠]]。他們測量蜂巢大小與裡面的蜜蜂[[數量|數量]]，並進行[[寄生蟲|寄生蟲]]與[[農藥|農藥]][[汙染物|汙染物]]的檢測。2009年，穆立與共同撰寫[[論文|論文]]的賓大研究人員瑪麗安‧佛瑞茲爾首度在肯亞蜜蜂身上發現[[瓦蟎|蟹蟎]]。那裡從前沒有發現[[孢子蟲|孢子蟲]]，但在這次[[研究|研究]]中，除了最偏遠的調查區域以外，兩種[[害蟲|害蟲]]都普遍出現。研究人員指出肯亞的蜜蜂身上同樣帶有在其他地方造成蜂群消失的害蟲與[[疾病|疾病]]，讓人驚訝的是，這些蜜蜂沒有被擊倒。即使在有多種[[病原體|病原體]]的蜂群中，族群數量依然充足，該次研究結果發表於2014年4月16日線上期刊 PLOS ONE 的論文中。<ref name="國家地理雜誌">{{Cite web |url = http://www.ngtaiwan.com/4760 |title = 東非蜜蜂不受致命病原體影響 |author =  |date = 2014-04-18 |publisher = [[國家地理雜誌|國家地理雜誌]] |language = zh-tw |accessdate = 2015-08-05 |quote = 有關東非蜜蜂的驚人發現，讓西方國家對抗「蜂群崩潰症候群」的努力，有了新的希望。科學家發現，肯亞的蜜蜂對於在世界其他地方造成數十億蜜蜂死亡的病原體，有極高的抵抗力。 }}</ref>

有一些[[報告|報告]]指出，入侵[[美國|美國]]西南的[[非洲化蜜蜂|非洲化蜜蜂]]對害蟲的抵抗力比其他種類的蜜蜂高。牠們源自1950年代晚期從[[非洲|非洲]]被帶到[[巴西|巴西]]的[[品種|品種]]，後來又與歐洲蜜蜂[[雜交|雜交]]，因此不應與穆立、格洛辛格和他們的同僚所研究的東非蜜蜂混為一談。<ref name="國家地理雜誌" />
