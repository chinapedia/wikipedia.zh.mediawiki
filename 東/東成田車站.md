'''東成田車站'''（{{jpn|j=東成田駅|hg=ひがしなりたえき|rm=Higashi-narita eki|f=y}}）是一個位於[[千葉縣|千葉縣]][[成田市|成田市]]古込字込前，屬於[[京成電鐵|京成電鐵]]與[[芝山鐵道|芝山鐵道]]的[[鐵路車站|鐵路車站]]。京成電鐵車站編號為'''KS44'''。

== 概要 ==
此站是由京成電鐵管轄的[[共構與共站|共同使用站]]。（[[機場第2大樓站|機場第2大樓站]]管理。）

此站有京成電鐵的[[東成田線|東成田線]]與芝山鐵道的[[芝山鐵道線|芝山鐵道線]]停靠，既是東成田線的終點又是芝山鐵道線的起點站。兩線的列車以此站進行[[直通運行|互相直通運行]]，運用上形成一體。但是芝山鐵道線不能使用[[PASMO|PASMO]]、[[Suica|Suica]]等IC卡。除[[押上線|押上線]]的[[押上站|押上站]]外，京成電鐵與其他公司共用的共同使用站都只編入車站實際管理公司的車站編號，因此本站由京成給予[[車站編號|車站編號]]'''KS44'''。

站體位於[[成田國際機場|成田國際機場]]用地內，1978年起成為成田機場旅客候機樓最近的車站「（舊）成田機場站」，但因為仍需步行一段路才能到達航廈，因此[[石原慎太郎|石原慎太郎]]在擔任[[運輸大臣|運輸大臣]]（今 [[國土交通大臣|國土交通大臣]]）時要求京成與JR東日本延伸路線到航廈下方設立新站以便利旅客。1991年現時的[[成田機場站|成田機場站]]（以下稱作「新站」）完工開業之際改為現名，新站與[[機場第2大樓站|機場第2大樓站]]（1992年開業）成為新的機場轉運站。雖然搭機旅客不再利用此站搭車，不過通勤時刻仍有許多在機場工作的相關人士使用。現在閘外大堂有連通成田機場第2候機樓的地下聯絡通道。現在站內仍可見到當時遺留的「成田機場站」字樣。

== 歷史 ==
* 1978年（[[昭和|昭和]]53年）5月21日：第一代成田機場站開業，是京成本線的終點站。由於當局只計畫興建[[成田新幹線|成田新幹線]]直通成田機場，沒有其他直達的鐵路交通，故本站設有來往成田機場的收費接駁巴士（由成田機場交通經營）。

== 車站結構 ==
本站是一[[地下車站|地下車站]]，月台採[[島式月台|島式月台]]2面4線設計。其中1、2號月台為特急列車專用月台，3、4號月台為普通列車專用月台。新的成田機場站啟用後，特急列車專用月台被封閉及停用，而原有的3、4號月台則改稱為1、2號月台。但直到今日仍可看到特急列車專用月台上一些「成田機場站」的站名牌及以往特急列車的廣告。
站內沒有電梯，但月台設有一座上行電扶梯通往付費區大廳。本站剪票口外設有連接[[機場第2大樓站|機場第2大樓站]]的換乘通道，全長約500米（開放時間為5時20分至23時15分）。本站亦有機場第2大樓站的時刻表，連絡通道外的自動售票機還可購入該站的乘車券。

1997年4月1日博物館動物園站停止營業（2004年4月1日廢站）起至2010年7月17日成田機場線（成田Sky Access）通車並新設成田湯川站為止，本站是京成電鐵停靠班次最少的一站。芝山鐵道線通車以前，本站以上野、都營淺草線方向列車為主，通車後則縮減至早晨時段與傍晚以後，午間時段以成田 - 芝山千代田區間列車為主。

=== 巴士路線 ===
成田機場站時代，本站可搭乘第1航廈的收費連絡巴士。更改站名後曾有往京成成田站、[[多古町|多古]]方向的路線巴士，但在芝山鐵道線通車後全部移往芝山千代田站。

之後，乘客可改搭航廈聯絡巴士來往東成田站與各航廈，但2018年10月1日現在已取消停靠<ref>[https://www.narita-airport.jp/jp/access/shuttlebus ターミナル間の移動] - 成田国際空港</ref>。現在僅剩日本航空等部分機場相關企業會開設至本站的接駁巴士。
