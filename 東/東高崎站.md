'''東高崎站'''（{{jpn|j=東高崎駅|kyu=|hg=ひがしたかさきえき|hgt=|rm=Higashi-Takasaki eki|f=y}}）位於[[宮崎縣|宮崎縣]][[都城市|都城市]][[高崎町|高崎町]]，是[[九州旅客鐵道|九州旅客鐵道]]（JR九州）
[[吉都線|吉都線]]上的車站。車站片假名中的「{{lang|ja|たか'''さ'''き}}」 ，與町名「{{lang|ja|たか'''ざ'''き}}」有些微差異。

== 車站構造 ==
位於山區內，岸式月台1面1線的地上站。無人車站，沒有站房，但月台上有候車室。

== 車站周邊 ==
* 東霧島神社

== 历史 ==
* 1963年（昭和38年）12月1日 - 設立。
* 1987年（昭和62年）4月1日 - [[國鐵分割民營化|國鐵分割民營化]]後成為九州旅客鐵道（JR九州）轄下的車站。
