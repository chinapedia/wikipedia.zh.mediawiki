'''東皋心越'''（{{jpn|j=東皐心越|hg=とうこう しんえつ}}，1639年-1696年10月25日） 俗姓[[蔣|蔣]]，字'''心越'''，法名'''兴俦'''，初名'''兆隐'''，號'''東皋'''、'''樵雲'''、'''越道人'''、'''鹜峰野樵'''，是在[[江戶時代|江戶時代]]初期從[[中国|中国]]東渡到[[日本|日本]]的一位禅僧。他在日本傳播中國的[[詩畫|詩畫]][[篆刻|篆刻]]以及[[古琴|古琴]]，被稱為“日本琴樂中興之祖”和“日本篆刻之祖”。<ref name="人民網">{{Cite web |url=http://paper.people.com.cn/rmrbhwb/html/2006-07/28/content_9862633.htm|title=心越：让东瀛盛开篆刻花（金石鉴赏）|accessdate=2015-02-01|author= |date= 2006-07-28 |work= |publisher= [[人民網|人民網]]}}</ref><ref>{{Cite web |url=http://www1.cncm.ne.jp/~seifu/sinetu.htm|title=日本琴楽中興の祖、東皐心越|accessdate=2015-02-01|author= |date= |work= |publisher=株式会社長崎ケーブルメディア}}</ref>

== 生平==
[[File:吟立軸，東皋心越.jpg|thumb]]
東皋心越出生於[[浙江省|浙江省]][[金华府|金华府]][[浦江县|浦江县]]，相傳為西漢[[蔣翔|蔣翔]]後裔。<ref name="浦江網">{{Cite web |url=http://pjnews.zjol.com.cn/pjnews/system/2005/12/03/000509021.shtml|title=旅日琴僧东皋心越传略|accessdate=2015-02-01|author= 谢孝苹 |work=2005-12-03 |publisher= 今日浦江}}</ref> 八歲時在[[蘇州|蘇州]][[報恩寺_(蘇州)|報恩寺]]剃髮，曾雲遊[[江浙|江浙]]一帶。二十歲奉[[曹洞宗|曹洞宗]]寿昌派觉浪道盛為師，道盛圓寂之後從[[杭州|杭州]]皋亭山崇光显孝寺阔堂大文，阔堂[[圓寂|圓寂]]之後，驻锡[[杭州|杭州]][[永福寺_(杭州)|永福寺]]。他雖然已經出家，卻仍然在[[康熙|康熙]]十三年（1674）年參與響應[[吳三桂|吳三桂]]的浙闽一带[[反清|反清]][[起事|起事]]，然而最終失敗。

他只得在康熙十五年（1676），应[[长崎|长崎]][[興福寺_(長崎)|兴福寺]][[法主|法主]]澄一道亮之邀，以'''杜多俦'''或'''越杜多'''的化名自杭州东渡日本，<ref name="國學網">{{Cite web |url=http://www.confucianism.com.cn/html/A00030007/11052249.html|title=东皋心越琴学研究纲要|accessdate=2015-02-01|author= |date=2010-06-21 |work= |publisher= 中國國學網}}</ref>，十二月抵達[[舟山|舟山]]，<ref name="浦江網"/> 同月三十日抵达[[九州|九州]]，次年正月抵达[[长崎|长崎]]。<ref>敖運梅、趙元云，<[http://www.nssd.org/articles/article_detail.aspx?id=46977550 论明末东渡僧人心越詩歌的禪理與畫意]>，《文藝評論》，2013年08期</ref>因日本的[[日本锁国|鎖國令]]，他在[[延宝|延宝]]五年（1677）正月至延宝七年（1679）十二月一直居住于專供外國人停留的[[長崎|長崎]]，駐興福寺。實際上因為他不屬於澄一道亮的[[臨濟宗|臨濟宗]]，為了入境還被迫向澄一行弟子礼。延寶七年夏，和[[水戶藩|水戶藩]]主[[德川光国|德川光国]]（[[水戶黃門|水戶黃門]]）使臣、[[朱舜水|朱舜水]]之学生今井弘济會面，後者表示德川有意邀請東皋離開此地，於是按照二人的計劃，心越在延寶八年（1680）五月到[[京都|京都]]附近[[黄檗山|黄檗山]]万福寺向主持木庵性瑶贺七十壽誕，作《上黄檗木和尚书》向木庵請求幫助。<ref name="浦江網"/> 未果，之後再以访[[瑞圣寺|瑞圣寺]]之名前往[[江户|江户]]，但最終被人揭發而被迫返回長崎，兴福寺也被關閉。<ref name="國學網"/>

在長崎的年歲中，東皋心越在宗教、艺术領域十分活躍，以禪學和藝術結交幕府人士，因此再次得到德川光国的幫助，[[天和_(日本)|天和]]元年（1682）東皋先移居長崎皓台寺，同年六月离开长崎，之後寓居于德川光国在水戶的别墅中。期間廣交各界人士、教授[[古琴|古琴]]。[[元禄|元禄]]四年（1691）入住水户岱宗山天德寺（今寿昌山祗园寺），元禄五年（1692）十月於天德寺正式开堂说法。元禄八年（1695）九月圆寂，後來被奉为寿昌山祗园寺和少林山达摩寺的开山祖师。<ref name="國學網"/><ref name="人民網"/>

==評價==

*[[高罗佩|高罗佩]]：“夫中国七弦琴者，往昔日本人徒听其名，而未尝知其实，日本旧籍多载‘琴’一 字，而其意殆指[[和琴|和琴]]、[[伽倻琴|新罗琴]]、皆筝课之一种，非中国先圣之制。”<ref name="國學網"/>
*《日本琴史》：“琴学盛于日本，实师（东皋）之功也。”<ref name="國學網"/>
*[[三谷阳子|三谷阳子]]：“清初，如果没有心越亡命日本，就不会有日本琴乐的发展。”<ref name="國學網"/>

他东渡日本时带有古琴五床，其中有記載的是“虞舜”、“素王”、“万壑松”這三床。“虞舜”由德川家收藏，[[藤田东湖|藤田东湖]]在其琴匣上作記，現兩者都收藏在[[東京國立博物館|東京國立博物館]]。此外他還把《[[松弦馆琴谱|松弦馆琴谱]]》、《[[理性元雅|理性元雅]]》、《[[伯牙心法|伯牙心法]]》、《[[琴学心声|琴学心声]]》和自撰《谐音琴谱》等[[古琴譜|古琴譜]]帶到了日本。在日本東皋心越尤以授琴出名，著名弟子包括[[杉浦正职|杉浦正职]]等，他的弟子將琴学传入了日本的儒官及学者之中，並且搜集心越的琴曲，集成《東皋琴譜》，於[[宝永|宝永]]年间出版。<ref name="國學網"/>
