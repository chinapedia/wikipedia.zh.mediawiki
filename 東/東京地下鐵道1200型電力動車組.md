'''東京地下鐵道1200型電力動車組'''（{{jpn|j=東京地下鉄道1200形電車|hg=とうきょうちかてつどう1200がたでんしゃ|rm=Tōkyō Chikatesudō 1200 gata densha|f=y}}），是供[[帝都高速度交通营团|帝都高速度交通营团]][[银座线|银座线]]使用的[[通勤型电力动车组|通勤型电力动车组]]，以2~3編組為主。于1934年至1984年间使用。

== 概要 ==
本列車繼承了[[東京地下鐵道1000型電力動車組|1000、1100型]]的基本特點，是[[東京地下鐵道|東京地下鐵道]]擁有最後製造的新型列車。[[帝都高速度交通營團|帝都高速度交通營團]]成立后繼續擁有此款列車，并一直在[[銀座線|銀座線]]使用。<br/>列車共製造24輛，其中[[汽車製造|汽車製造]]東京支店在1933年至1934年生產了21輛（編號1231-1251），[[川崎重工車輛公司|川崎重工]]兵庫工場在1934年生產了3輛（編號1252-1254）。

== 車體 ==
本列車車體構造基本延續[[東京地下鐵道1000型電力動車組|1000、1100型]]，不過由於焊接技術的發展，車體外壁的鉚釘有所減少，車頂曲線變得相對更加平滑。另外，車廂編號在車廂兩端均有標示，以便在連掛期間也能看清。

== 搭載設備 ==
列車採用了[[通用電氣|通用電氣]]製造的機器設備，取代了以往常用的[[三菱電機|三菱電機]]製造的機器設備。主電動機GE-259C因性能較好，代替了[[三菱電機|三菱電機]]和[[西屋公司|西屋公司]]合作製造的電動機MB-231AF。<ref group="注">端子電圧600V時定格出力90kW/658rpm。</ref>另外，控制器也採用了[[通用電氣|通用電氣]]製造的PC電空凸輪控制器，取代了[[西屋公司|西屋公司]]製造的ABF電空單位開關式控制器。不過列車也預留了PC電空凸輪控制器與ABF<ref group="注">末尾的M是通用电气公司产的M控制系统的控制器兼容功能说明。此外，三菱电机的命名规则是指ABFM是ABF多段化。</ref>電空單位開關式控制器之間互換或兼容的需要。這些搭載設備在之後的[[銀座線|銀座線]]列車中繼續有所體現。

== 運用 ==
列車在出廠後，長期和既有車型一起在[[銀座線|銀座線]]上使用。<br/>1964年，[[營團2000型電力動車組|2000型]]的先頭車開始採用固定編組的模式，隨後1200型、[[營團1300型電力動車組|1300型]]、[[營團1400型電力動車組|1400型]]開始[[駕駛台撤銷車|駕駛台撤銷車]]改造工程，1200型、1300型、1400型中間的[[動車組動車|動力車]]及[[營團1500型電力動車組|1500N型]]的[[動車組拖車|拖車]]一起進行了相關改造。車廂內飾部改為了和[[營團2000型電力動車組|1000型]]一樣的內飾，部分客室的車窗亦改為了和[[營團1300型電力動車組|1300型]]一樣的小窗戶。<br/>在電路改造方面，列車安裝了高壓母線，使得電動发电機的低壓輔助電源及備用燈的蓄電池可以有所減少，但1200型和1300型的中間的[[動車組拖車|拖車]]的[[集電靴|集電靴]]和主蓄電池依然在改造中有所保留。因[[銀座線|銀座線]]採用[[第三軌供電|第三軌供電]]，在道岔位置無法使供電軌道連貫鋪設，因此在1200型列車通過較長的道岔期間，仍會出現車內照明瞬間停電的現象，這種現象直到後來的[[營團1500型電力動車組|1500N型]]和[[營團01系電力動車組|01系]]列車上得到了解決。<br/>1978年，列車更換為了[[阿爾斯通公司|阿爾斯通公司]]製造FS-387型[[轉向架|轉向架]]。不過隨著1984年[[營團01系電力動車組|01系]]列車的服役，1200型列車開始陸續報廢，直至1986年全數報廢完畢<ref>{{Cite journal|title= {{link-ja|铁道杂志|鉄道ジャーナル}} |date = 1987-01 |volume = 21 |issue = 1 |page = 128 |publisher = 鉄道ジャーナル社 }}</ref>。也是營團地下鐵的歷代車輛中服役最久的。
