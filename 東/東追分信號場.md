'''東追分信號場'''（{{jpn|j=東追分信号場|hg=ひがしおいわけしんごうじょう|rm=Higashi-oiwake shingōjyo|f=y}}）位於[[日本|日本]][[北海道|北海道]][[勇拂郡|勇拂郡]][[安平町|安平町]]追分美園，曾經是[[北海道旅客鐵道|北海道旅客鐵道]]（JR北海道）[[石勝線|石勝線]]沿線的車站，只停靠部份普通列車。由於近年使用人數極低，2016年3月26日起遭廢站，並降格為[[信號場|信號場]]。

== 廢站前的車站構造 ==
設站時已採用簡易車站模式，不設站房，亦沒有站員駐守，只設置相對式[[側式月台|側式月台]]2個，擁有2面2線的月台配置。月台之間以[[天橋|天橋]]連接，並將兩端的出入口加設趟門及放置椅子等設施，作為簡單的[[候車室|候車室]]。

== 車站周邊 ==
很難找到車站的聯外道路，事實上由附近的[[平交道|平交道]]有道路可通往東追分站。車站附近是廣大的農村以及數間農舍。
* [[北海道道462號川端追分線|北海道道462號川端追分線]]

== 歷史 ==
*1965年（[[昭和|昭和]]40年）3月1日 - 設立。只有辦理客運業務。
*1981年（昭和56年）6月1日 - 將車站往追分方向移動300公尺。
*1987年（昭和62年）4月1日 - [[國鐵分割民營化|國鐵分割民營化]]後成為JR北海道轄下的車站。
*2012年（[[平成|平成]]24年）2月16日 - 一列由釧路貨物站行駛的貨物列車因沒有依照訊號導致出軌。直至19日才回復正常服務。
*2016年（平成28年）3月26日 - 因使用者減少廢止車站<ref>{{Cite press release|url=http://www.jrhokkaido.co.jp/press/2015/151218-3.pdf|format=PDF|title=平成28年3月ダイヤ改正について|publisher=北海道旅客鉄道|date=2015-12-18|accessdate=2015-12-18}}</ref>。

==軼事==
