'''安東尼·羅拔·杜利高'''（'''{{lang-en|Anthony Robert Dorigo}}'''，{{bd|1965年|12月31日|catIdx=Dorigo ,Tony}}），簡稱'''東尼·杜利高'''（'''{{lang-en|Tony Dorigo}}'''），出生在[[墨爾本|墨爾本]]，是一名已退役的[[英格蘭|英格蘭]]足球員，曾效力[[阿士東維拉足球會|阿士東維拉]]、[[車路士足球會|車路士]]、[[列斯聯足球會|列斯聯]]和[[英格蘭足球代表隊|英格蘭]]效力，司職[[足球後衛|左閘]]。

==生平==
杜利高出身於[[阿士東維拉足球會|阿士東維拉]]青訓。1984年5月12日，他在對[[葉士域治足球會|葉士域治]]的賽事中後備入替而處子上陣<ref>{{en}}{{cite web 
|url=http://www.sporting-heroes.net/football-heroes/displayhero_club.asp?HeroID=39584
|title=Tony Dorigo
|publisher=Sporting-heroes. 
|date=1984-05-12 |accessdate=2010-03-14}}</ref> 。

1987年7月3日，他以47.5萬[[英鎊|英鎊]]身價轉投[[車路士足球會|車路士]]。8月15日，他在對[[錫周三足球會|錫周三]]的賽事中處子上陣。該季結束後，他雖然贏得車路士年度最佳球員，但球會最終亦降班。他協助球會升班重返甲組聯賽。1991年，他的轉會請求被撤回，時任領隊{{Link-en|卜比·甘保|Bobby Campbell (footballer born 1937)}}亦不重用他。

最終他在同年6月6日以130萬英鎊身價艱難地離開[[史丹福橋球場|史丹福橋球場]]並加盟[[列斯聯足球會|列斯聯]]<ref>{{en}}{{cite web 
|url=http://www.independent.co.uk/sport/football-barnet-players-sign-new-contracts-1559918.html
|title=Football: Barnet players sign new contracts
|publisher=The Independent 
|date=1992-10-27 |accessdate=2010-03-14}}</ref>。8月20日，他在對[[諾定咸森林足球會|諾定咸森林]]的賽事中處子上陣<ref>{{en}}{{cite web 
|url=http://www.sporting-heroes.net/football-heroes/displayhero_club.asp?HeroID=39585
|title=Tony Dorigo
|publisher=Sporting-heroes. 
|date=1991-08-20 |accessdate=2010-03-14}}</ref> 。他加盟的首季，列斯聯就贏得甲組聯賽冠軍。

1997年8月20日，杜利高加盟[[意大利|意大利]]球會[[拖連奴足球會|拖連奴]]<ref>{{en}}{{cite web 
|url=http://www.independent.co.uk/sport/football-city-open-to-pounds-7m-offers-for-kinkladze-1246430.html

|url=http://www.sporting-heroes.net/football-heroes/displayhero_club.asp?HeroID=39586

2000年7月12日，杜利高加盟[[史篤城足球會|史篤城]]並成為隊長<ref>{{en}}{{cite web

退役後，他曾於2006/07球季在[[布拉沃|布拉沃]]和2007/08球季在{{Link-en|5頻道|Channel 5}}擔當意大利足球球的評述員，現在他忙於房地產開發的工作。2002年，他獲通知取得{{Link-en|歐洲足協專業牌照|UEFA Pro Licence}}。


==國際賽==
杜利高雖然在[[澳洲|澳洲]]出世，但他持有[[英國|英國]][[護照|護照]]。他曾代表[[英格蘭21歲以下足球代表隊|英格蘭U21]]上陣11次、[[英格蘭足球代表乙隊|英格蘭B隊]]7次和[[英格蘭足球代表隊|英格蘭]]15次。
1988年，由於[[史超活·皮雅斯|史超活·皮雅斯]]受傷，杜利高因而被徵召以{{Link-en|堅尼·森臣|Kenny Sansom}}的替補身份被列入[[1988年歐洲國家盃|1988年歐洲國家盃]]英格蘭隊的名單中。

1989年12月13日，他在對[[南斯拉夫國家足球隊|南斯拉夫]]的賽事中首次代表英格蘭隊上陣<ref>{{en}}{{cite web 
|url=http://www.thefa.com/England/MensSeniorTeam/Archive.aspx?x=652
|title=Yugoslavia - England
|publisher=[[英格蘭足球總會|英足總]]
|date=1989-12-13 |accessdate=2010-03-14}}</ref>。

杜利高亦有在[[1990年世界盃|1990年世界盃]]以1-2的比分不敵[[意大利國家足球隊|意大利]]的季軍戰中上陣<ref>{{en}}{{cite web 
|url=http://www.thefa.com/England/MensSeniorTeam/Archive.aspx?x=664

|date=1990-07-07 |accessdate=2010-03-14}}</ref>。1993年10月13日，他在[[1994世界盃外圍賽|1994世界盃外圍賽]]對[[荷蘭國家足球隊|荷蘭]]的賽事中最後一次代表英格蘭隊上陣<ref>{{en}}{{cite web
