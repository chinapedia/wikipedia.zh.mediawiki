'''東山線'''（{{jpn|j=東山線|hg=ひがしやません|rm=Higashiyama sen|f=y}}）為連接從[[日本|日本]][[愛知縣|愛知縣]][[名古屋市|名古屋市]][[中川區|中川區]]之[[高畑車站|高畑站]]至同市內[[名東區|名東區]]之[[藤丘車站_(愛知縣)|藤丘站]]之[[名古屋市營地下鐵|名古屋市營地下鐵]]的路線,也是地下鐵最繁忙的路線。正式名稱為'''名古屋市高速度鐵道第1號線'''。本路線顏色為[[黃色|黃色]]。

所有的車站均可使用[[manaca|manaca]]（已於2011年2月11日引進使用）進出。

== 概要 ==
本路線為名古屋市內最早開業之地下鐵路線。

本線呈東西向橫貫名古屋市內，通過[[名古屋站|名古屋站]]與市中心「榮」地區後連結市內東側，本線為名古屋市營地下鐵之中乘客人數最多之路線，也成為名古屋站至[[千種車站|千種站]]之間[[JR東海|JR東海]]之重要替代路線。由一社站通往上社站之間，以及通往藤丘站從地下轉往地上區間，成為高架路線。而本郷站與藤丘站之間高架橫跨[[東名高速道路|東名高速道路]]。由東端之[[藤丘車站_(愛知縣)|藤丘站]]，連接[[愛知高速交通東部丘陵線|愛知高速交通東部丘陵線]]（Linimo）。在[[日本|日本]]國內之公營地下鐵之中，為僅次於[[大阪市營地下鐵|大阪市營地下鐵]][[御堂筋線|御堂筋線]]之賺錢路線。

於2005年所開辦的[[2005年日本國際博覽會|愛知萬博]]為契機，車內廣播系統除了[[日語|日語]]之外，還有[[英語|英語]]、[[華語|華語]]、[[葡萄牙語|葡萄牙語]]（[[巴西葡語|巴西葡語]]）、[[韓語|韓語]]共計5國的語言來播送。當在名古屋站發車時（前往藤丘）及抵達時（前往高畑），以及抵達藤丘站時播送上述5種語言，在萬博閉幕後也持續至今。也因如此，名古屋市內的地下鐵、[[名古屋鐵道|名鐵]]等主要車站或是幹線道路沿線中，成為各交通機關標識裡以5國語言顯示之先驅。

並預計在2015年度（[[平成|平成]]27年度）中，於東山線所有車站月台上設置可動式[[月台門|月台門]]。

== 路線資料 ==
* 路線距離（{{link-ja|營業距離|営業キロ}}）：20.6公里
* [[軌距|軌距]]：1435毫米
* 站數：22個（包括起終點站）
* 複線路段：全線
* 電氣化路段：全線（直流600V、[[第三軌供電|第三軌供電]]）
* 地面路段：一社－藤丘間
* [[閉塞_(鐵路)|閉塞方式]]：[[自動列車運行裝置|ATO]]
* 最小曲線半徑：125公尺（名古屋－伏見間）
* 最高速度：65公里/小時

== 營運型態 ==
原則上為全線運行方式，但在首班車與末班車之發車站分別為以下區間（由2008年到現在）。
*往藤丘
**首班車：由[[岩塚車站|岩塚站]]出發
**末班車：止於[[星丘車站_(愛知縣)|星丘站]]
*往高畑
**首班車：由[[池下車站|池下站]]出發
**首班車後：由星丘站出發
**末班車：止於岩塚站

車班間隔則是平日早晨通勤通學時間帶以2分鐘為間隔，白天時間則以4到5分鐘為間隔，傍晚返家尖峰時段則為3分鐘間隔，清晨夜間時段則大約8分鐘為間隔。周六假日則為早、晚以4分為間隔，白天時間則以4到5鐘為間隔。但是在假日之白天時段至傍晚間，比平常日快大約為4分鐘為間隔。

本路線在末班車時段，會在榮站停留較久一點。另外在岩塚站之首班車發車時，有時也會利用到第2月台來發車。
