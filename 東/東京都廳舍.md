'''東京都廳舍'''（{{jpn|j=東京都庁舎|hg=とうきょうとちょうしゃ|rm=Tōkyōto chōsha|f=h}}）是[[日本|日本]][[東京都廳|東京都廳]]（東京都政府）及[[東京都議會|東京都議會]]的[[地方政府大樓|所在地]]，目前位於[[東京都|東京都]][[新宿區|新宿區]]<ref name="TMGB SCP">{{cite web |url=http://skyscraperpage.com/cities/?buildingID=210 |title=Metropolitian Government Building |accessdate= |publisher=Skyscraperpage.com}}</ref><ref name="TMBG EMP">{{cite web |url=http://www.emporis.com/en/wm/bu/?id=tokyocityhall,toweri-tokyo-japan |title=Tokyo City Hall, Tower I |accessdate= |publisher=Emporis.com}}</ref>的建築為第三代廳舍，1990年12月落成，1991年正式啟用，為[[西新宿|新宿新都心]]最高的[[摩天大樓|摩天大樓]]。其在時任[[東京都知事|東京都知事]][[鈴木俊一|鈴木俊一]]任內籌建，由[[建築師|建築師]][[丹下健三|丹下健三]]設計。原第一、二代廳舍位於[[東京都|東京都]][[千代田區|千代田區]]丸之內

== 概要 ==
1970年代，為了解決[[丸之內|丸之內]]舊東京都廳舍老舊、狹窄、分散等既有的問題，1979年當時的[[東京都知事|東京都知事]][[鈴木俊一|鈴木俊一]]強烈建議將東京都廳舍遷移至新宿。[[東京都議會|東京都議會]]於1985年9月通過了「東京都政府設置位置條例」並決定將新都廳設立在[[新宿副都心|新宿副都心]]，同年10月舉辦「新都廳舍大樓設計比賽」，隔年（1986年）4月[[丹下健三|丹下健三]]（結構設計為[[武藤清|武藤清]]）的設計方案被選中。

東京都廳舍於1988年4月起造，1990年12月完工。隔年（1991年）4月1日從丸之內的舊辦公樓遷移，並開始作為[[東京都廳|東京都廳]]的用途。東京都廳舍由第一本廳舍、第二本廳舍、都議會議事堂等3棟建築構成，在1993年時被高296米的[[橫濱地標大廈|橫濱地標大廈]]打破。東京都廳舍第一本廳舍近年來已成為東京的[[觀光|觀光]]勝地之一，到45樓的展望室參觀的訪客絡繹不絕。

東京都廳舍的原址為舊淀橋汙水處理廠，位於新宿西口超高層摩天大樓群的西南側，地址為二丁目8番1號、4號及5號共三塊地。1號為第二本廳舍，4號為第一本廳舍，5號為[[東京都議會|東京都議會]]議事堂及都民廣場。此外，第一及第二本廳舍面對[[新宿中央公園|新宿中央公園]]西側。

=== 第一本廳舍・設施===
第一本廳舍高度為243米，落成時打破[[豐島區|豐島區]][[池袋|池袋]]的[[太陽城60|太陽城60]]維持了12年的高度紀錄，成為日本第一高樓，但此紀錄並沒有維持很久，在1993年時被高296米的[[橫濱地標大廈|橫濱地標大廈]]打破，東京第一高樓名的號也於2007年時被248米高的[[中城大廈|中城大廈]]超越。

==== 87展望台 ====
廳舍的[[觀景台|展望台]]設於第一本廳舍南、北棟45樓，可從1樓乘搭直通[[電梯|電梯]]直達，時間約為55秒。開放時間為上午9時半至下午11時(南棟至下午5時)，除[[新年|新年]]期間外全年無休，免費入場，北棟展望台設有[[咖啡室|咖啡室]]及[[伴手禮|伴手禮]]店。東京都廳舍雖然高達48層，但是其43樓、44樓、46樓-48樓為機房，一般訪客不能進入（但可使用44樓之廁所）。

== 設計與構造 ==
東京都廳舍是丹下健三的代表作之一，設計風格採[[後現代主義|後現代主義]]路線，被認為是引用了[[哥特式|哥特式]]教堂的設計作為藍本，採用了和[[巴黎聖母院|巴黎聖母院]]相似的横三段和豎三段式立面，切面為[[正八邊形|正八邊形]]，而建築物的外表設計啟發自[[中村弘道|中村弘道]]，他讀了諾曼·卡佛的《日本建築的形與空間》（日本建築の形と空間），書中記載留在大阪的富農之房屋·吉村邸的天井為細長的格子狀圖案，因此他使用此作為大樓外型的設計，丹下健三曾表示此外型設計像[[印刷電路板|印刷電路板]]；但也有人批評這種設計，表示其造型像是[[石碑|石碑]]。此外，第二本廳舍及同為丹下健三設計的[[新宿公園塔|新宿公園塔]]造型極為相似，可以看得出來是同一位建築師之作品。

== 資料 ==
* 層數：地面48層、地下3層（第一本廳舍）
* 高度：243.4米
* 建築面積：11,042平方米
* 動工：1988年4月／竣工：1990年12月
* 建設費：1569億日圓
* 每年維修管理費：40億日圓
* 由第一、第二本廳舍及都議會議事堂三座組成。
