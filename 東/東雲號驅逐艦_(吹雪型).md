'''「東雲」號驅逐艦'''（{{lang|ja|しののめ}}）是[[大日本帝國海軍|大日本帝國海軍]]第40號[[驅逐艦|驅逐艦]]，「[[吹雪型驅逐艦|吹雪型驅逐艦]]」的六號艦。於[[佐世保海軍工廠|佐世保海軍工廠]]動工，在1928年至1941年間服役。其命名繼承自[[東雲型驅逐艦|東雲型驅逐艦]]的同名艦，是第二艘被命名為「東雲」的戰艦。雖然編號為「吹雪」型六號艦，但事實上較同屬「吹雪」型的五號艦「[[叢雲號驅逐艦_(吹雪型驅逐艦)|叢雲]]」號更早動工、下水及服役。

== 簡介 ==
[[吹雪型驅逐艦|吹雪型驅逐艦]]是日本海軍在簽署《[[華盛頓海軍條約|華盛頓海軍條約]] 》後，透過強化不受條約限制的輕巡洋艦以下補助艦艇，以彌補和英美海軍之間的差距而設計及建造的新型軍艦。[[藤本喜久雄|藤本喜久雄]]造船[[大佐|大佐]]參照「[[夕張號輕巡洋艦|夕張]]」號[[輕巡洋艦|輕巡洋艦]]，大幅輕量化後完成設計。當中封閉式艦橋、砲塔式主砲等都成為後來日本海軍[[驅逐艦|驅逐艦]]的基本設計。

「吹雪型」由於建造期間長，不同階段的設計都有所變更，其中「東雲」號驅逐艦屬「吹雪型」的「特I型」，屬於先行設計及建造的十艘戰艦之一。但由於「東雲」號在戰鬥中一度失去聯絡，因此其結局並不清楚，分別有「被敵軍戰機擊沉」{{refTag| 1=雑誌「丸」編集部（1997），《'''日本海軍艦艇写真集16 駆逐艦 吹雪型［特型］'''》{{ja}}，頁179，光人社}}及「觸動水雷沉沒」{{refTag| 1=外山操（2005），《'''艦長たちの軍艦史'''》{{ja}}，頁269，光人社，ISBN 4-7698-1246-9 }}等推測。

== 特點 ==
「特I型」吹雪型驅逐艦備有高干舷、長首樓船型及飛剪式艦首，並採用封閉式艦橋。相較過去的驅逐艦，「吹雪型」擁有較大的居住區，大幅改善船員的起居生活。此外，「吹雪型」為日本海軍首次在艦上安裝50倍口徑三年式雙聯裝炮的驅逐艦。但由於大幅輕量化，「吹雪型」艦體強度不足，重心較高影響了其穩定性，在經歷[[第四艦隊事件|第四艦隊事件]]後才作出改造。

== 建造經緯 ==
於1926年8月12日在[[佐世保海軍工廠|佐世保海軍工廠]]動工，被命名為「第40號驅逐艦」，後來於1928年8月1日，改名為「東雲」號，級別屬於一等驅逐艦。

於1928年7月25日正式完工，於同年12月1日加入[[第2艦隊|第2艦隊]][[第2水雷戰隊|第2水雷戰隊]]第12驅逐隊正式服役。

==艦歷==
於1936年12月1日轉為予備役，返回母港進行整修及改造。

於1940年 [[中國抗日戰爭|中國抗日戰爭]]期間，主要在[[華南|華南]]沿岸一帶作戰，並參與[[日軍入侵法屬印度支那|日軍入侵法屬印度支那]]戰役。[[大東亞戰爭|大東亞戰爭]]爆發後，與同屬[[第3水雷戰隊|第3水雷戰隊]]的「[[白雲號驅逐艦_(吹雪型驅逐艦)|白雲]]」號及「[[叢雲號驅逐艦_(吹雪型驅逐艦)|叢雲]]」號一同參與[[南方侵略作戰|南方侵略作戰]]。

於1941年12月13日參與[[婆羅洲|婆羅洲]]攻略作戰，為登陸[[美里|美里]]的部隊提供掩護而向[[金蘭灣軍事基地|金蘭灣軍事基地]]進發。「東雲」號在12月15日到達美里附近，離開船團單獨行動後下落不明。然而，於同月17日上午9時左右，日本登陸戰船團收到不明來歷的報告表示「正與敵軍戰機交戰」{{refTag| 1=志賀博（1985），《'''海軍兵科将校'''》{{ja}}，頁103-104，光人社，ISBN 4-7698-0264-1}}，在中午12時左右曾觀察到美里附近海面出現水柱，「叢雲」號受命前往調查，於附近海面發現大量燃油，推斷為「東雲」號沉沒地點。於1942年1月15日除籍。

==搜尋==
一隊以美里為基地的搜索隊在[[荷蘭|荷蘭]]的幫助下，自2004年開始在推斷沉沒地點一帶進行搜索，但時至今日，「東雲」號的確實沉沒地點仍未被發現。
