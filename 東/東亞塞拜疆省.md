'''東阿塞拜疆省'''（[[波斯語|波斯語]]：آذربایجان شرق）是[[伊朗|伊朗]]三十一個省份之一。面積46,929平方公里，在所有省份中排行第十。人口約3,500,000（2005年數據）；首府位於[[大不里士|大不里士市]]。

== 地理位置 ==
東阿塞拜疆省位於[[伊朗|伊朗]]西北部；北與[[阿塞拜疆共和国|阿塞拜疆共和国]]及[[亞美尼亞共和國|亞美尼亞共和國]]接壤，東面、南面、及西面則分別與[[阿爾達比勒省|阿爾達比勒省]]、[[贊詹省|贊詹省]]、及[[西阿塞拜疆省|西阿塞拜疆省]]為鄰。境內多高山，其中位於[[大不里士|大不里士市]]南面的薩蘭德峰是全省的最高點，[[海拔|海拔]]超過3,700公尺。

東阿塞拜疆省的[[氣候|氣候]]因[[山區|山區]]和[[平原|平原]]地區而異。[[山區|山區]]較乾燥及寒冷；[[平原|平原]]地區則較溫和。

== 歷史 ==
在[[馬其頓|馬其頓]][[亞歷山大大帝|亞歷山大大帝]]逝世後，在阿塞拜疆地區有一名叫[[阿特羅巴特斯|阿特羅巴特斯]]的總督；他於公元前320年宣布獨立，帶領這個地區脫離[[馬其頓|馬其頓]]人的管治。因此，這個地區名稱為Atorpatkan，最後被改成阿塞拜疆（[[英語|英語]]：Azarbaijan）。

== 大地震 ==
{{main|2012年大不里士地震}}
根據國際媒體從[[德黑蘭|德黑蘭]]傳出的消息，2012年8月11日，當地時間16時53分，在伊朗東阿塞拜疆省連續發生了兩起地震<ref>[http://www.bbc.co.uk/zhongwen/simp/world/2012/08/120811_iran_quakes.shtml 伊朗强烈地震造成至少220人丧生-BBC中文网]{{zh-cn}}</ref>，至少有60座村莊受災，當中6座村莊全毀，而6座中有4座在阿哈爾市，兩起地震間隔約11分鐘。<ref>[http://www.hkcd.com.hk/content/2012-08/13/content_3024652.htm 伊朗地震近3千死傷 6村莊夷為平地- 香港商報] {{webarchive|url=https://web.archive.org/web/20140808055207/http://www.hkcd.com.hk/content/2012-08/13/content_3024652.htm |date=2014-08-08 }} {{zh-hant}}</ref>

[[德黑蘭大學|德黑蘭大學]]地球物理研究所報告，地震強度為[[芮氏|芮氏]]6.2與6.0，而[[美國地質調查局|美國地質調查局]]（USGS）說是6.4與6.3；地震深度僅約10公里，所以造成了227死與1380人傷。

地震發生的第一時間，土耳其、新加坡、德國、台灣等表示願意支援，但[[红十字会与红新月会国际联合会|伊朗紅新月會]]（Red Crescent）稱有能力處理災情，故婉拒其他國家的援助。<ref>[http://tw.news.yahoo.com/%E6%9C%89%E5%8A%9B%E8%99%95%E7%90%86%E7%81%BD%E6%83%85-%E4%BC%8A%E5%A9%89%E8%AC%9D%E5%8F%B0%E7%81%A3%E6%95%91%E6%8F%B4-012612591.html 有力處理災情 伊婉謝台灣救援-Yahoo!奇摩新聞]{{zh-tw}}</ref>

伊朗內政部長[[那賈爾|那賈爾]]（Moustafa Mohammad-Najjar）說：「搜救行動現已結束。我們正努力確保滿足生還者在食物與住宿上的需要。」那賈爾及其他官員說，瓦礫磚石堆下已經沒有其他生還者，沒有進一步搜救必要。

目前有超過16000人無家可歸，地震還波及了[[大不里士|大不里士]]與瓦爾扎甘。
