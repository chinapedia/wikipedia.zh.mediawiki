'''東伏見宮依仁親王'''（ひがしふしみのみや よりひとしんのう、[[庆应|庆应]]3年旧暦9月19日（1867年10月16日） - 1922年（[[大正|大正]]11年）6月27日），[[大日本帝国|日本]]的[[皇族|皇族]]、[[大日本帝国海軍|海軍]][[軍人|軍人]]。

[[伏見宮邦家親王|伏見宮邦家親王]]王子。官位至[[日本元帥|元帥]][[海軍大将|海軍大将]]・[[菊花章|大勲位]]・[[金鵄勲章|功三級]]。妃[[岩倉具定|岩倉具定]][[公爵|公爵]]長女[[依仁親王妃周子|周子（かねこ）]]。

==生平==
初名定麿王。留学[[英国|英国]]後，1885年（明治18年）12月，成为[[小松宮彰仁親王|小松宮彰仁親王]]的養子，1886年（明治19年）5月[[親王宣下|親王宣下]]，以[[明治天皇|明治天皇]]猶子被赐名为'''依仁'''。

1887年（明治20年）[[法国布列斯特海军学院|法国布列斯特海军学院]]留学，1890年（明治23年）被授予海軍少尉。此後历任[[横須賀鎮守府|横須賀鎮守府]]司令長官、[[第二艦隊_(日本海軍)|第二艦隊]]司令長官。

1903年（明治36年），[[小松宮彰仁親王|小松宮彰仁親王]]去世，依仁親王由于和彰仁亲王生前关系很坏，不想继承小松宮，因此新设立[[東伏見宮|東伏見宮]]<ref>[[小松宮彰仁親王|小松宮彰仁親王]]はもと東伏見宮[[嘉彰親王|嘉彰親王]]を名乗っていた</ref>。他曾参加英国[[乔治五世_(英国)|乔治五世]]的加冕礼，[[东乡平八郎|东乡平八郎]]、[[乃木希典|乃木希典]]作为随从人员参加。历任大日本水産会总裁、日法协会名誉总裁等。1918年（[[大正|大正]]7年）7月，晋升为[[海軍大将|海軍大将]]。1922年（大正11年），56歳去世。追赠[[大勲位菊花章頸飾|大勲位菊花章頸飾]]和元帥称号。由于无后，東伏見宮废绝。

== 与夏威夷王国的公主亲事 ==
1885年（明治18年），[[夏威夷王国|夏威夷王国]]的国王访问日本。这是外国元首首次访日。明治天皇在[[迎賓館|赤坂離宮]]与夏威夷国王会谈时建议联姻，即依仁親王（当时被称为山阶宮定麿王）与国王的侄女结亲<ref>{{cite web |url=http://www.isis.ne.jp/mnn/senya/senya1232.html |title=反米大陸 |publisher=松岡正剛の千夜一夜・遊蕩篇 |author=[[松岡正剛|松岡正剛]] |date= |accessdate=2012-04-07 |archive-url=https://web.archive.org/web/20160423171126/http://www.isis.ne.jp/mnn/senya/senya1232.html |archive-date=2016-04-23 |dead-url=yes }}</ref>。日本国力増強后，明治新政府就对此事闭口不谈了。

== 血缘 ==
* 父母：[[伏見宮邦家親王|伏見宮邦家親王]] - 妃[[鷹司景子|鷹司景子]] - （生母）女房[[伊丹吉子|伊丹吉子]]

* 兄弟：[[山階宮晃親王|晃親王]] - [[聖護院宮嘉言親王|嘉言親王]] - [[曼殊院宮譲仁入道親王|譲仁親王]] - [[久邇宮朝彦親王|朝彦親王]] - 男子 - [[伏見宮貞教親王|貞教親王]] - 男子 - [[小松宮彰仁親王|彰仁親王]] - [[北白川宮能久親王|能久親王]] - 男子 - 男子 - [[華頂宮博経親王|博経親王]] - [[北白川宮智成親王|智成親王]] - [[伏見宮貞愛親王|貞愛親王]] - [[清棲家教|清棲家教]] - [[閑院宮載仁親王|載仁親王]] - '''依仁親王'''（女子省略）

* 妻：山内八重子（侯爵[[山内豊信|山内豊信]]的三女。后離婚，嫁给[[秋元興朝|秋元興朝]]为継室）
* 後妻：[[依仁親王妃周子|岩倉周子]]（公爵[[岩倉具定|岩倉具定]]的長女）

[[久邇宮邦彦親王|久邇宮邦彦親王]]第三親王子[[東伏見慈洽|東伏見宮邦英親王]]，1919年（[[大正|大正]]8年）養育在東伏見宮邸，親王去世时，担任喪主主持葬礼。事实上，邦英王可以说是依仁親王的养子。

此后，东伏见的姓氏，由伯爵东伏见邦英继承。
