東港跨年晚會於2007年起舉辦，舉辦地點於東港河堤旁並邀請藝人以及社區及學校團體一起參與 並於活動途中施放段落性煙火，並於倒數後施放主煙火通常施放時間在10～15分不等，其跨年舞台為全臺唯一搭在水面上的水上舞台。

== 2007 ==
為東港鎮第一次舉辦跨年晚會，本次煙火施放長達30分鐘於進僑大橋橋下沙洲施放

== 2008 ==
== 2009 ==
以牛轉先機慶豐年為主題，施放30分，18種不同種類之煙火。本次跨年首次於進德大橋橋面施放煙火。
== 2010 ==
本次煙火施放10分鐘，本次施放中可見到魚型煙火、2010字樣、微笑等…… 煙火施放上密度增高

== 2011 ==
為慶祝民國一百年的到來 本次煙火施放15分鐘來慶祝2011的來臨 為當年度最長的跨年煙火

== 2012 ==
本次煙火施放13分鐘(段落性3分 主秀10分)配合著水面爆破煙火並於施放過程中出現愛心和微笑等造型煙火，本次壓軸藝人為羅時豐。

== 2013 ==
本次跨年晚會結合屏東縣政府、鵬管處、墾管處及恆春半島觀光產業聯盟今年首度攜手合作，推出「BIKE訪鵬灣，HIGH翻東港，LIGHT在墾丁」跨年系列活動，本次煙火一共施放18分鐘(段落性4分 主秀14分)，煙火施放尺寸歷年最大並配著江南Style音樂施放。本次煙火創下全台最長跨年煙火秀的記錄

== 2014 ==
本次跨年晚會藝人有黃文星、愷弟、向蕙玲、方炯鑌、藍天等演唱主題以「愛戀東港」為題<ref>http://www.appledaily.com.tw/realtimenews/article/new/20131231/318215/</ref>，在本次壓軸跨年煙火主秀以「完美、極限、激情」三大元素，呈現東港漁民的力與美，<ref>http://www.dbnsa.gov.tw/user/Article.aspx?Lang=1&SNo=04004032</ref>本次煙火更結合進德大橋與河岸邊的高空煙火同步施放，在最後階段以煙火瀑布之美來迎接2014年的到來。
== 2015 ==
本年度'''無跨年煙火施放及晚會'''，改於2014年東港聖誕節晚會中施放<ref>http://news.ltn.com.tw/news/local/paper/840638</ref>，本次施放長達13分鐘煙火並結合音樂節奏搭配，開場時搭配冬季戀歌，營造河畔浪漫氣氛，接著穿插各種特效煙火施放，最後以東港特有的水景及彩花流水瀑布搭配3D立體旋轉煙火壓軸，結合進德大橋壯闊場景，炫麗場景將驚豔所有觀眾，本次施放首次設計風火輪特效煙火設計。晚會嘉賓有 : 郭靜、許博安、孫淑媚、許志豪等藝人接力，在寒冬中熱力開唱。
== 2016 ==
今年以飛躍東港為主題，邀請在地社團及藝人來演出，本次的嘉賓有 :王彩樺、羅小白、藍天、王妃、謝佳純、雅妮、陳隨意等歌星來為東港助陣。在煙火施放上，今年特別搭建山巒造型固定器材，用特效煙火呈現不同於傳統跑動式特效煙火，創新特效噴泉花束將山巒的美峰峰相連到天邊的壯闊真實呈現，盡情揮灑在東港溪沿岸的夜空中，並施放全台唯一跨年場16吋煙火彈於東港溪上綻放,打破2014年時12吋煙火的紀錄<ref>http://www.akau.tw/?p=38193</ref>
