'''東漢末年'''（189年－220年，東漢[[中平|中平]]元年－[[建安|建安]]二十五年）是指[[黃巾之亂|黃巾之亂]]起，直至[[曹丕篡漢|曹丕篡漢]]或[[三國|三國]]鼎立（[[東吳|東吳]][[孫權|孫權]]於229年稱帝）結束，持續36年（或以東吳建國計，為45年），这段时间内军阀割据，彼此混戰。

[[東漢|東漢]]末年，由於土地兼并越加尖銳，地主豪強擁有自己的私人武力，加上中央政府政治黑暗，皇權虛弱無力，對地方過度放權，因而造成群雄割據的局面。

而這些割據勢力又有著不同的興起過程與發展脈絡，反映出那個動亂時代的多元面向。

东汉末年被归入广义的[[三国时期|三国时期]]。

== 背景 ==
東漢末年，農業生產出現嚴重的土地兼并現象，直接導致地方豪強勢力的崛起<ref name="童超8.1"/>。

由於中央政府政治黑暗，[[十常侍|十常侍]]橫行朝野，對地方的控制越加衰弱，造成土地兼并問題日益惡化，而人民飽受此現象的苦果，加上天災與瘟疫的雙重打擊，他們於是紛紛起來造反，發動大規模的叛亂<ref name="童超8.1"/>，最著名者是[[黃巾之亂|黃巾之亂]]。

時代的動盪不安，也使地方豪強有了崛起的機會，他們一開始靠著自己的財力組織自己的武裝軍隊保衛家園，後來就逐漸演變成擁有私人武力的軍閥，如[[曹操|曹操]]、[[劉備|劉備]]等人都是在這樣的背景下嶄露頭角，中央政府由於不修政事，對於這樣的情況，也是無能為力<ref name="童超8.1"/>，反而需要他們維持地方穩定。

地方豪強發展到後期，又逐漸分化為[[士族|士族]]與[[寒門|寒門]]兩個陣營。其中，士族在土地與權勢，佔有了優勢，也因此擁有壟斷政府高層的實力，形成了[[三國|三國]][[兩晉|兩晉]]時期特殊的士族政治<ref name="童超8.1"/>。

社會現況，當時的年輕人，熱中用言語攻擊前輩。<ref>[[孔融|孔融]]《論盛孝章書》，收錄於《[[文選|文選]]》第四十一卷：「今之少年，喜謗前輩，或能譏評孝章。」</ref>

== 刺史改州牧 ==
除了嚴重的土地兼并問題，是造成地方豪強興起的誘因之外，刺史制度的變動，則是造成群雄割據的主因<ref name="童超8.2"/>。

[[刺史|刺史]]制度源自[[西漢|西漢]][[漢武帝|武帝]]時期，本意是用等級低的官員監察等級高的官員，以達到制衡的目的（以[[六條問事|六條問事]]制衡官員），儘管刺史的權力越來越大，但是始終沒有統領一州的法定權力和信度。不過到了東漢[[漢靈帝|靈帝]]時，他接受[[劉焉|劉焉]]的建議，將各州刺史改稱州牧。於是，刺史成為地方的官員，從原本的中央和郡、縣之間加入州一級，成為中央、州、郡、縣的四級結構。州牧位居郡守之上，掌握一州的軍政大權<ref name="童超8.2"/>。

這樣的情況持續的結果，使各個州牧紛紛利用天下大亂，名正言順在地方割據勢力，而那些未被改為州牧的刺史和郡守，也趁機擴大權利和武裝力量，還有人在戰亂中自封為州牧和刺史，如此一來，等於宣告東漢政府的實質統治力已經名存實亡<ref name="童超8.2"/>。

== 割據情況 ==
當時東漢共分為十三個州，每個都有豪強割據。各方勢力彼此互相交戰，百姓生活自然苦不堪言<ref name="童超8.3"/>。

在這些割據勢力中，[[曹操|曹操]]、[[袁紹|袁紹]]、[[孫堅|孫堅]]父子與[[劉備|劉備]]等人為著名的勢力外，另外還有一些勢力的堀起經過，更能呈現出亂世所發展出的不同模式<ref name="童超8.3"/>。

首先是[[劉表|劉表]]，他就是典型的受專於刺史改州牧政策的地方統治者。劉表成為荊州牧，掌握荊州的軍政大權。他在亂世之中，先殺[[孫堅|孫堅]]，後抗曹操，成為天下舉足輕重的力量，不過劉表過於保守多疑，寵信奸佞。他死後，其中[[劉琮|劉琮]]繼位後，曹操統一北方，開始南征，劉琮便舉眾投降<ref name="童超8.3"/>。

接下來是[[劉焉|劉焉]]、[[劉璋|劉璋]]父子，劉焉向漢靈帝提出刺史改州牧的建議，而後主動申請赴任益州牧，到了該地後，一方面掃除地方勢力，加強自身的統治；另一方面公然斷絕益州與中央的聯繫，成為益州的一方之霸，他死後，由其子劉璋繼承其位，最後由劉備所降服<ref name="童超8.3"/>。

領兵的武將如[[韓遂|韓遂]]、[[馬騰|馬騰]]等人，也是東漢末年群雄割據來源的一大途徑。他們曾一再擊敗中央政府的軍隊，而聲名大噪，不過後來發生內鬥情況，最終被曹操所分化擊敗<ref name="童超8.3"/>。

此外，民間出現了很多以宗教形式出現的武裝集團，例如[[張角|張角]]領導的[[黃巾賊|黃巾賊]]就是其中最有影響力的一支。另外還有一支持續時間最長的民間割據勢力，那就是[[張魯|張魯]]在[[漢中|漢中]]地區領導的[[五斗米道|五斗米道]]教眾<ref name="童超8.3"/>。涼洲[[枹罕|枹罕]]的[[宋建|宋建]]則自立為「河首平漢王」，設百官改年號，並割據三十年；宋建以王的身份建國（有百官制及年號），為歷史之先。

==後世影響==
===文化===
正史《[[三国志|三国志]]》所记载的历史时期从黄巾之乱开始，包括了这一时期，直到三分归晋而结束。後世的文化作品因而把東漢的群雄割據時代的歷史視為[[三國|三國]]歷史的一部份，兩者的歷史關係非常緊密。《[[三國演義|三國演義]]》的内容也以《三国志》为本，橫跨群雄割據和三國鼎立的兩個時期，直到[[晉朝|晉]]灭吴為止。[[三國遊戲|三國系列遊戲]]則在這個基礎上對歷史再進行加工。
