'''東門前站'''（{{jpn|j=東門前駅|hg=ひがしもんぜんえき|rm=Higashimonzen eki|f=y}}）位於[[日本|日本]][[神奈川縣|神奈川縣]][[川崎市|川崎市]][[川崎區|川崎區]]中瀨三丁目，是[[京濱急行電鐵|京濱急行電鐵]][[大師線_(京濱急行電鐵)|大師線]]的[[鐵路車站|鐵路車站]]。[[車站編號|車站編號]]是'''KK24'''。

==年表==
* 1925年（[[大正|大正]]14年）8月15日 -  海岸電氣軌道車站啟用。
* 1937年（[[昭和|昭和]]12年）12月1日 - 車站停用。
* 1944年（昭和19年）6月1日 - 以[[東京急行電鐵|東京急行電鐵]]（大東急）車站重新啟用。
* 1948年（昭和23年）6月1日 - 成為京濱急行電鐵車站。
* 2023年（令和5年）度 - 預定地下化<ref>{{Cite web |url=http://www.city.kawasaki.jp/980/cmsfiles/contents/0000067/67733/280212machi3-2.pdf |title=京浜急行大師線連続立体交差事業　都市計画事業認可の変更について |format=PDF |publisher=川崎市 |accessdate=2016-04-28}}</ref>。

[[側式月台|側式月台]]2面2線[[地面車站|地面車站]]。月台間無天橋，小島新田方向有站內平交道。

車站下方有地下道。

== 車站周邊 ==
周邊為住宅區。川崎大師（[[平間寺|平間寺]]）在步行圈內（約10分），雖然比從[[川崎大師站|川崎大師站]]前往的時間較久（約5分），但初詣等人潮較多的時後仍有乘客選擇從本站前往。

本站步行2分鐘左右的國道409號北側在2005年至2010年進行大型公寓群建設。
* {{link-ja|國道409號|国道409号}}
* 川崎大師自動車交通安全祈禱殿
* {{link-ja|大師公園|大師公園}}
* 川崎區役所大師支所
* 川崎市立東門前小學校
* 川崎市立大師小學校
* 川崎大師[[郵局|郵局]]
* 川崎昭和郵局
* {{link-ja|島忠|島忠|Home's}}川崎大師店
* [[小松製作所|小松教習所]]神奈川中心

=== 巴士路線 ===
步行3分的國道409號上有「小松製作所前」巴士站，可搭乘{{link-ja|川崎鶴見臨港巴士|川崎鶴見臨港バス}}2系統（快速 浮島橋系統、川01系統）。川01系統平日2僅有兩班往返。<!--詳細はリンク先へ-->

== 站名由來 ==
因位於[[平間寺|平間寺]]（川崎大師）東門前（實際在東北方）而命名。
