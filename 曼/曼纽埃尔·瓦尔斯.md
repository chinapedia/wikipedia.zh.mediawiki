'''曼纽埃尔·瓦尔斯'''（{{lang-fr|''' Manuel Valls'''}}，{{lang-ca|'''Manuel Carles Valls i Galfetti'''}}，{{IPA-fr|ma.ny.ɛl vals|lang}}，{{bd|1962年|8月13日|catIdx=V}}）[[法国|法国]]和[[加泰隆尼亞|加泰隆尼亞]][[政治人物|政治人物]]，前任[[法國總理|法國總理]]。

2014年被总统[[弗朗索瓦·奥朗德|弗朗索瓦·奥朗德]]任命为法国总理。此前他自2012年开始担任[[法国内政部|內政部长]]。2001年至2012年任[[埃夫里_(埃松省)|埃夫里]]市长并自2002年一直是[[法國國民議會|國民議會]]成员。他是[[社会党_(法国)|社会党]]成员。2016年12月6日，瓦爾斯為了參加[[2017年法國總統選舉|2017年法國總統選舉]]，辭去總理職務。2018年，宣布參選[[加泰隆尼亞|加泰隆尼亞]]首府[[巴塞羅拿|巴塞羅拿]][[市長|市長]]選舉。

== 早年岁月 ==
瓦尔斯1962年生于[[加泰隆尼亞|加泰隆尼亞]][[巴塞罗那|巴塞罗那]]，1982年根据法国[[归化|归化]]政策加入法国国籍。其父亲是一个[[加泰罗尼亚|加泰罗尼亚]]画家，母亲是瑞士籍意大利人。

1986年，他从[[巴黎第一大学|巴黎第一大学]]历史系毕业，获得优异评语。此前，他在法国陆军120师团服兵役，驻扎在枫丹白露。在巴黎第一大学大学时期，他认识了他的第一任妻子并育有4个儿女。2010年，他与小提琴家安娜·格拉梵结婚。

== 总统大选 ==
2012年，瓦尔斯参加社会党内初选。首轮选举告负后，他转投[[弗朗索瓦·奥朗德|弗朗索瓦·奥朗德]]并被任命为竞选团队的媒体发言人。他对媒体的牢牢掌控，除总统候选人之外唯一的高出镜率为他赢得了“军官”的称号。

== 内政部长 ==
2012年5月，他在[[让-马克·埃罗|让-马克·埃罗]]政府内阁任内政部长。

== 总理 ==
2014年3月，由于社会党在法国市镇选举中失利，来自社会党的总理[[让-马克·埃罗|让-马克·埃罗]]为此承担败选的责任选择辞职，法国总统[[弗朗索瓦·奥朗德|弗朗索瓦·奥朗德]]任命瓦尔斯担任新一任总理<ref>[http://www.bbc.com/news/world-europe-26826292 French President Hollande names Valls as new PM] ''BBC News.'' March 31, 2014.</ref><ref>{{fr}}[http://www.liberation.fr/politiques/2014/03/31/hollande-oblige-de-remanier-aujourd-hui-selon-le-depute-ps-thierry-mandon_991891]''Libération (Libération avec AFP)'' March 31, 2014</ref>。新内阁于4月2日正式组建， 由来自于社会党的16名部长和一名来自于[[左派激进党|左派激进党]]的部长组成。

2014年8月25日，瓦尔斯因与经济部长[[阿诺·蒙特布尔|阿诺·蒙特布尔]]发生激烈冲突，向总统奥朗德递交辞呈，奥朗德随后宣布解散现政府，并要求瓦尔斯组建新政府。26日，奥朗德任命新一届内阁成员，经济部长蒙特布尔被[[埃马纽埃尔·马克龙|埃马纽埃尔·马克龙]]取代。马克龙代表社會黨政府推行勞工及經济改革，削減退休金及社會福利，受到不少左派人士的批評。法國政府於2016年7月20日動用憲法中第49-3條，即政府可以不通過國會投票表決而通過某項法案，通過新勞工法，飽受爭議的「限制勞資調解委員會津貼」一條被廢除，但「經濟型解僱」被確定下來。35小時工作制被解禁，而帶薪假期也可以在剛入職時就享有，不需等待曾經的「權利開啟」。這是在總理瓦爾斯任期內第二個被通過的重要法案，另一項是經濟改革的馬克隆法案。<ref>[http://www.epochtimes.com/b5/16/7/25/n8135791.htm 法國勞動法改革「長跑」5個月終得通過] 2016-07-25 大紀元</ref>
