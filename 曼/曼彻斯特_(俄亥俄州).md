'''曼彻斯特'''是美国[[俄亥俄州|俄亥俄州]][[亞當斯縣_(俄亥俄州)|亚当斯县]]的一座城镇<ref group="注">在俄亥俄州，人口在五千人以下的城市称为Village，与其他地方的Village意思不同，故译为城镇而非乡村。{{cite web|url=http://codes.ohio.gov/orc/703.01 |title=Ohio Revised Code Section 703.01(A) |accessdate=2007-09-12 }}</ref>，位于[[俄亥俄河|俄亥俄河]]畔， [[2010年美國人口普查|2010年美國人口普查]]时共有2,023人。

曼彻斯特位于[[曼彻斯特高中_(俄亥俄州曼彻斯特)|曼彻斯特本地学区]]，[[亚当斯县_(俄亥俄州)#图书馆|亚当斯县图书馆]]的在曼彻斯特有一座分馆。曼彻斯特有一座警察局，共有三名警察。曼彻斯特火警/营救/紧急服务为曼彻斯特城镇区、曼彻斯特乡镇及斯普里格乡镇服务，共有50名成员，其中三人为全职紧急服务人员。

== 历史 ==
{{link-en|纳撒尼尔·马西|Nathaniel Massie}}于1791年在地图上标记了曼彻斯特。1833年时，这个小镇有一个邮局，三家商店和25间住户。<ref>{{Cite web|url = http://books.google.com/books?id=dt48AAAAYAAJ&pg=PA291#v=onepage&q&f=false|title = The Ohio Gazetteer, or, a Topographical Dictionary|publisher = Scott and Wright|date = 1833|accessdate = 12 December 2013|author = Kilbourn, John|pages = 291}}</ref>

== 人口 ==

=== 2010年人口普查 ===
根据2010年[[人口普查|人口普查]]<ref name="FactFinder">{{Cite web|title = American FactFinder|url = http://factfinder2.census.gov/faces/nav/jsf/pages/index.xhtml|publisher = [[美国人口调查局|美国人口调查局]]|accessdate = 2013-01-06}}</ref>结果，曼彻斯特共有2,023人，818户和527个家庭。人口密度为{{Template:Convert|1580.5|PD/sqmi|PD/km2|1}}。1,032间住房单位密度为{{Convert|806.3|/sqmi|/km2|1}}。曼彻斯特总人口的96.2%是[[白种人|白种人]]，0.2%是[[非洲裔美国人|非洲裔美国人]]，0.4%是土著美国人， 0.1%是[[亚裔美国人|亚裔]]，0.7%来自其他种族，还有2.4%是混血。西班牙裔和拉丁美洲裔分别占总人口的1.7%。

曼彻斯特的818户中34.2%有18岁以下的孩子和他们居住，39.4%为已结婚的夫妇，19.2%已婚丧夫的家庭，5.9%为已婚丧妻家庭，还有35.6%户是非家庭。818户中的30.3%由个人组成，14.1%居住着65岁及以上的独居老人。平均每户有2.47人，每个家庭3.05人。

曼彻斯特的年龄中位数为38.1岁，18岁以下的少年儿童及青少年占总人口的26.7%，7.5%的人口居于18岁到24岁之间，26.6%的人口居于25岁到44岁之间，19.8%的人口居于45岁到64岁之间，还有15.5%的人口是65岁以上的老年人。曼彻斯特的人口由48.2%的男性和51.8%的女性组成。

根据2000年[[人口普查|人口普查]]<ref name="GR2"/>结果，曼彻斯特共有2,043人，882户和555个家庭。人口密度为{{Template:Convert|1894.5|PD/sqmi|PD/km2|1}}。1,027间住房单位密度为{{Template:Convert|952.4|/sqmi|/km2|1}}。曼彻斯特总人口的97.50%是[[白种人|白种人]]，0.15%是[[非洲裔美国人|非洲裔美国人]]，0.44%是土著， 0.24%是[[亚裔美国人|亚裔]]，0.05%来自其他种族，还有1.62%是混血。西班牙裔和拉丁美洲裔分别占总人口的0.34%。

曼彻斯特的882户中29.0%有18岁以下的孩子和他们居住，42.0%为已结婚的夫妇，15.3%已婚丧夫的家庭，还有37.0%户是非家庭。818户中的33.3%由个人组成，18.8%居住着65岁及以上的独居老人。平均每户有2.32人，每个家庭2.92人。

曼彻斯特的年龄中位数为37岁，18岁以下的少年儿童及青少年占总人口的25.0%，9.5%的人口居于18岁到24岁之间，27.6%的人口居于25岁到44岁之间，21.1%的人口居于45岁到64岁之间，还有16.7%的人口是65岁以上的老年人。平均每100名女性对应89.2名男性。18岁以上的每100名女性对应85.1名男性。

曼彻斯特城市住户年收入中位数为23,140[[美元|美元]]，家庭年收入中位数为28,641美元。男性年收入中位数为29,539美元，女性则是20,179美元。全市人均年收入为13,535美元。大约19.1%的家庭和23.3%的人口低于贫困线，这其中的29.7%未满18岁，23.3%则已65岁及以上。
