'''曼尼托巴省旗'''是[[加拿大|加拿大]][[曼尼托巴省|曼尼托巴省]]的官方旗幟，設計圖案為[[紅船旗|紅船旗]]加上曼尼托巴省徽章中的盾牌。此旗幟的設計源自[[加拿大紅船旗|加拿大紅船旗]]；隨著楓葉旗於1965年正式成為[[加拿大國旗|加拿大國旗]]，加拿大紅船旗亦告棄用。為了讓紅船旗上的[[米字旗|米字旗]]圖案可繼續在曼尼托巴省境内飄揚，該省議會同於1965年通過以紅船旗加曼尼托巴省徽盾牌為省旗設計<ref name="CHA"/><ref name="Fraser">{{Cite web |last=Fraser |first=Alistair B. |url=http://fraser.cc/FlagsCan/Provinces/Manitoba.html |title=Manitoba |work=The Flags of Canada |accessdate=2015-04-04 }}</ref><ref name="MBGov">{{Cite web |url=http://residents.gov.mb.ca/reference.html?d=details&program_id=5908 |title=Manitoba's Flag |accessdate=2015-04-04 |publisher=曼尼托巴省政府 }}</ref>。此設計包含米字旗圖案，因此需先獲[[加拿大君主|加拿大君主]]御准才可使用<ref name="Fraser"/>。女王[[伊利沙伯二世|伊利沙伯二世]]於1965年10月批准此設計<ref name="MBGov2">{{Cite web |url=http://news.gov.mb.ca/news/archives/1965/10/1965-10-08-queen_approves_manitoba_flag.pdf |title=Queen Approves Manitoba Flag |accessdate=2015-04-04 |date=1965-10-08 |publisher=曼尼托巴省政府 }}</ref>，而這面旗幟則於1966年5月12日正式成為曼尼托巴省旗<ref name="CHA"/><ref name="MBGov"/>。（[[安大略省議會|安大略省議會]]也是在相似的情況下通過[[安大略省旗|安大略省旗]]的設計。）省旗的設計於2010年11月15日列入加拿大紋章局的紋章及旗幟名冊內<ref name="CHA">{{Cite web |url=http://reg.gg.ca/heraldry/pub-reg/project.asp?lang=e&ProjectID=2096&ShowAll=1 |title=Province of Manitoba [Civil Institution] |accessdate=2015-04-04 |publisher=加拿大紋章局 }}</ref>。

盾牌中的白底紅十字來自[[哈德遜灣公司|哈德遜灣公司]]的紋章，代表該公司由英國人建立；該公司過去曾操控包括現曼尼托巴省地域在内的[[魯珀特地|魯珀特地帶]]。盾牌底部則繪有一隻站在石上的水牛，代表居於當地並在過往以捕獵水牛為生的[[加拿大原住民|原住民族]]<ref name="CHA"/>。
