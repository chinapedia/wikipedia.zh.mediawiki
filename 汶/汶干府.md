'''汶干府'''（{{thai|t=จังหวัดบึงกาฬ|r=Changwat Bueng Kan}}，{{IPA-th|t͡ɕāŋ.wàt bɯ̄ŋ kāːn}}）是[[泰国|泰国]]的一个[[泰國行政區劃|府]]。2011年3月23日，根据国王BE 2554 (2011)令，从原[[廊開府|廊開府]]中分离出几个[[泰國次級行政區劃|縣]]，設置汶干府。<ref name = Gazette>{{cite journal|journal=Government Gazette|volume=128|issue=18 A|pages=1|title=Act Establishing Changwat Bueng Kan, BE 2554 (2011)|url=http://www.ratchakitcha.soc.go.th/DATA/PDF/2554/A/018/1.PDF|date=2011-03-22|language=th}}</ref>汶干府是的第76个[[泰國行政區劃|府]]，也是第77个府级行政单位。

汶干府位于泰国东北部，隔[[湄公河|湄公河]]与[[老挝|老挝]]相邻。

== 行政區劃 ==

汶干府共分為8個縣，該等縣份轄下再分成共54個區和599條村。

== 名勝古蹟 ==
* 甫托山（ภูทอก，Phu Thok）：位於汶干縣，山頂上建築有「哇庶尼雅其里威漢寺」，面積約28.8平方公里，花了五年時間才建成七層環山螺旋橋至山頂，寺裡供奉有佛陀[[舍利子|舍利子]]，周邊環境秀麗，佈滿叢密樹林。
* 哇沙旺阿隆寺（วัดสว่างอารมณ์ หรือ，Wat Sawang Arom）：位於巴卡縣，寺裡供奉有一尊神聖佛像，它的周邊有很多美麗石景，涓涓流水，風景怡人。
