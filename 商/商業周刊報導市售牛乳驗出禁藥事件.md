[[2013年臺灣|2013年]]11月底，臺灣雜誌《[[商業周刊_(台灣)|商業周刊]]》大篇幅報導市售的多家知名[[牛乳|牛乳]]品牌的成分中含有不明化學物質殘留，指涉牛乳產品篩選過於寬鬆，牛乳安全性低。這篇報導刊出後引起社會關注，但被[[酪農|酪農]]、[[乳業|乳業]]、[[農委會|農委會]]及[[食藥署|食藥署]]等抨擊其引述數據的檢驗方法、儀器與判讀方式不嚴謹、報導不詳盡與全面。最後，《商業周刊》於2013年11月27日在官網發出道歉聲明。<ref> [http://news.ltn.com.tw/news/society/breakingnews/908756 牛奶駭人風波 商周補交資料道歉] 自由時報 </ref>

==起源==
2013年11月20日，《[[商業周刊_(台灣)|商業周刊]]》封面專題報導揭露台灣大部分的牛乳產品都檢測出不明化學物質殘留<Ref>{{Cite web|url = http://www.businessweekly.com.tw/KWebArticle.aspx?ID=52485|title = 牛奶駭人|accessdate = 2013-11-25|author = 吳美慧|date = 2013-11-20|work = 商業周刊 第1358期|publisher = 商周集團}}</ref>，報導中引用[[銘傳大學|銘傳大學]]生物科技學系副教授[[陳良宇_(學者)|陳良宇]]的報告<ref>{{cite web|url = http://www.businessweekly.com.tw/KWebArticle.aspx?ID=52490|title = 編者的話：一份不夠全面的檢驗報告|accessdate = 2013-11-25|author = 郭弈伶|date = 2013-11-20|work = 商業周刊 第1358期|publisher = 商周集團|language = zh-tw}}</ref>，指出市面上超過六成以上的牛乳製品有不該出現的化學物質。頓時之間引起媒體的爭相引用。
