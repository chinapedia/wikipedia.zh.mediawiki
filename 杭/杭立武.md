'''杭立武'''（{{bd|1903年|1月26日|1991年|2月26日|catIdx=H杭}}），[[中華民國|中華民國]]教育家、政治学家、政治家、外交家、社会活动家。于[[安徽省|安徽省]][[滁州|滁州]]出生，[[籍貫|籍貫]][[浙江省|浙江省]][[杭州|杭州]]。一生历任教育、文化、外交、社团等各方要职，晚年仍积极不懈地从事国民外交、保障人权和救助难民等工作。

== 早年經歷 ==
1923年，毕业于[[金陵大学|金陵大学]]文学院。同年以第一名的成绩考取安徽省公费留学名额，赴英国[[伦敦大学|伦敦大学]]深造，其间赴美国[[威斯康辛大学|威斯康辛大学]]担任名誉研究员，获硕士学位后返英国继续学习，1929年获伦敦大学政治学博士学位。回国后任[[国立中央大学|国立中央大学]]政治系教授，曾兼任系主任。

1931年，任中英庚款董事会总干事。1932年，创立中国政治学会并兼任总干事。1933年，在南京成立中英文化协会，此后中法、中比、中瑞、中美等文化协会亦相继成立。

== 中年經歷 ==
[[中國抗日戰爭|抗战]]期间，任[[国民参政会|国民参政会]]参议员、美国联合援华会会长。1944年，任[[国民政府|国民政府]]教育部常务次长，1946年转任政务次长。

1947年6月2日，教育部次長杭立武抵武漢大學調查。<ref name="民國史大事記">{{cite book|editor=李新總主編，[[中國社會科學院|中國社會科學院]]近代史研究所[[中華民國史|中華民國史]]研究室編，韓信夫、姜克夫主編|title=《中華民國史大事記》|location=[[北京|北京]]|publisher=[[中華書局|中華書局]]|date=2011年7月}}</ref>{{rp|8366}}7月28日，[[交通大學|交通大學]]校長[[吳保豐|吳保豐]]辭職後，是日成立整理委員會，教育部次長杭立武擔任主任委員，決定限期一個月，將校務整理就緒。<ref name="民國史大事記"/>{{rp|8390}}9月3日，[[聯合國教科文組織|聯合國教科文組織]]遠東區基本教育研究會在南京召開，中國代表杭立武當選為主席；會議主題是關於推進基本教育。<ref name="民國史大事記"/>{{rp|8406}}同年任[[國立中央博物院|國立中央博物院]]籌備處主任，并在[[安徽省|安徽省]][[滁縣|滁縣]]當選為[[第一屆國民大會安徽省代表|第一屆國民大會安徽省代表]]。

1948年8月16日，教育部次長杭立武到[[漢口|漢口]]，處理湖北師範學院及流亡學生問題<ref name="民國史大事記"/>{{rp|8660}}。

1949年，就任教育部长。4月25日，教育部長杭立武到達廣州<ref name="錄" >{{cite book |year=2011 |title=《1949大流亡：美國外交檔案室密錄》 |editor=王景弘編譯 |location=台北市 |publisher=玉山社出版 |isbn=978-986-294-000-6 }}</ref>{{rp|127}}。5月10日，教育部長杭立武向美國註廣州領事館證實，互不信任繼續損害反共力量之團結，蔣在李宗仁證明有死戰到底之決心前，不敢完全信任李<ref name="錄" />{{rp|146}}。政府撤台后籌組並出任[[國立中央博物圖書院館聯合管理處|國立中央博物圖書院館聯合管理處]]主任委員。1949年5月至1950年2月，任[[國立編譯館|國立編譯館]]館長。

1953年，参与美国教会联合会筹办[[東海大學_(台灣)|东海大学]]，作为大陆十三所教会大学之延续。1956年起，历任中華民国驻[[泰国|泰国]]、[[寮國|寮國]]、[[菲律宾|菲律宾]]及[[希腊|希腊]]诸国大使，兼[[联合国教科文组织|联合国教科文组织]]会议首席代表。

== 晚年經歷 ==
1965年，发起成立“亚洲与世界社”（AWI）。其间改组“港澳之友会”为“中华港澳之友协会”，协助港澳同胞继续拥有自由权益；同时积极维护及争取[[台湾原住民|台湾原住民]]权益。1972年，任[[國立政治大學國際關係研究中心|國立政治大學國際關係研究中心]]主任。1979年，创立[[中国人权协会|中国人权协会]]。

1991年2月26日，逝世於[[台北市|台北市]]。
