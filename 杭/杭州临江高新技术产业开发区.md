'''杭州临江高新技术产业开发区'''位于[[杭州市|杭州市]][[萧山区|萧山区]]，原名'''萧山临江工业园区'''，2013年9月22日更名为'''临江高新技术产业园区'''<ref>{{cite web |url=http://www.parkchina.net/park/news_view.php?id=500&park_id=1121&st=1 |language= |title=临江工业园区成功更名为临江高新技术产业园区 |publisher=萧山临江高新区 |date=2013-10-10 |accessdate= }}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，2015年4月经[[国务院|国务院]]批准升级为[[国家高新技术产业开发区|国家高新技术产业开发区]]，成为[[浙江省|浙江省]]第六家国家高新区<ref>{{cite web |url=http://www.zj.xinhuanet.com/finance/region/2015-04/14/c_1114957683.htm |language= |title=杭州临江国家高新区昨挂牌成立 |publisher=今日早报 |date=2015-04-14 |accessdate= |deadurl=yes |archiveurl=https://archive.is/20150803030808/http://www.zj.xinhuanet.com/finance/region/2015-04/14/c_1114957683.htm |archivedate=2015-08-03 }}</ref>。现任高新开发区管委会主任为杨军。

== 主要企业 ==
临江的主导产业包括节能和新能源汽车及零部件、机械装备、中科院产业、新能源、新材料等。拥有国家重点扶持高新技术企业26家、省级以上高新技术（研发）中心25家。

*[[新能源|新能源]]产业：[[东方电气|东方电气]]
*[[机器人|机器人]]产业：[[沈阳新松|沈阳新松]]
*整车生产企业：[[长安福特|长安福特]]、[[东风裕隆|东风裕隆]]
*高端先进装备制造业：[[西子航空|西子航空]]、[[三一重工|三一重工]]、[[杭齿前进|杭齿前进]]

== 交通 ==
开发区位于[[萧山|萧山]]市区东北方向，[[钱塘江|钱塘江]]南岸，通过[[钱江六桥|钱江六桥]]与江北的[[下沙开发区|下沙开发区]]相连。[[杭甬高速|杭甬高速]]和[[杭州绕城高速|杭州绕城高速]]经过该地。距离东边的[[萧山机场|萧山机场]]也不远。
