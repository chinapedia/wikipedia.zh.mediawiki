'''澀谷梓希'''（{{bd|1993年|8月11日|catIdx=Shibuya Azuki}}<ref name="profile"/>）是[[日本|日本]]的女性[[聲優|聲優]]、[[歌手|歌手]]，女性偶像團體[[i☆Ris|i☆Ris]]的成員。[[埼玉縣|埼玉縣]]出身。血型O型<ref>{{Cite web |url=https://www.facebook.com/media/set/?set=a.434484659927931.98683.429637317079332&type=3 |title=澁谷梓希|accessdate=2014-02-08}}</ref>。[[81_Produce|81 Produce]]所屬<ref>{{Cite web |url=http://www.81produce.co.jp/list.cgi?lady+2253710123120 |title=所屬俳優プロフィール |publisher=81 Produce |accessdate=2014-02-08 |archive-url=https://web.archive.org/web/20140222041837/http://www.81produce.co.jp/list.cgi?lady+2253710123120 |archive-date=2014-02-22 |dead-url=yes }}</ref>。

== 人物 ==
在[[avex|avex]]與聲優事務所[[81_Produce|81 Produce]]共同舉辦的「動畫歌曲主唱試鏡」中合格，並和其他五名成員組成團體「[[i☆Ris|i☆Ris]]」<ref>{{Cite web |url=http://www.girlsnews.tv/voice/62241 |title=i☆Ris 声優を目指す私たちを応援して欲しい GirlsNews! accessdate=2012-11-26}}</ref>。

與偶像團體「[[Tokyo_Cheer2_Party|Tokyo Cheer② Party]]」的團員蘆原優愛是高中時代的同學<ref>{{Cite web|url=http://ameblo.jp/ashihara-yua/entry-11948859335.html|title=楽しい放送でした|publisher=チアチア 芦原優愛オフィシャルブログ「your ashihara・・」|date=2014-11-06|accessdate=2014-11-07}}</ref>。

在2016年的第10回[[聲優獎|聲優獎]]中以i☆Ris一員的身分獲得歌唱賞<ref>{{Cite web|publisher=アニメイト|work=アニメイトTV|url=http://www.animate.tv/news/details.php?id=1457773181|title=「第十回 声優アワード」受賞者発表!助演男優賞&パーソナリティ賞に鈴村健一さん、最多得票賞に5年連続で神谷浩史さん|date=2016-03-12|accessdate=2016-03-13}}</ref>。
