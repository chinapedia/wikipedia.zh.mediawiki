'''澀谷'''{{efn|「{{lang|ja|渋谷}}」中文譯名應為「-{zh-tw:澀;zh-cn:澀(涩)}-谷」，但常誤寫作「涉谷」。「{{lang|ja|渋谷}}」原本所對應的傳統漢字（[[旧字体|旧字体]]）寫法為「{{lang|ja|'''澁'''谷}}」，按照日本漢字簡化規則，「{{lang|ja|澁}}」的簡化字為「{{lang|ja|渋}}」。「{{lang|ja|渋}}」字下的四個小點即代表上面的「止」字重覆兩次（「{{lang|ja|摂}}」亦同此理，為「-{zh-tw:攝;zh-cn:攝(摄)}-」的[[新字体|新字体]]）。在中文漢字裡，「-{澁}-」為「-{zh-tw:澀;zh-cn:澀(涩)}-」的[[異體字|異體字]]。「涉谷」則是錯用別字，因「{{lang|ja|渋}}」字形似「涉」字而混淆（但「涉」的對應日本簡化漢字實為「{{lang|ja|渉}}」）。<ref>[http://www.letsgojp.com/archives/4775/ 是「澀谷」還是「涉谷」？5個常見日式漢字大解析，讓你不只看得懂還唸得出來]，《樂吃購！日本》</ref>}}（{{jpn|j=渋谷|hg=しぶや|kyu=澁谷|rm=Shibuya|f=y}}）是[[日本|日本]][[東京|東京]]的地名，包含以下定義：

# 位於[[東京都|東京都]][[澀谷區|澀谷區]]的[[町|町]]。依照其{{link-ja|住居表示|住居表示}}，分為澀谷一丁目至四丁目。人口3,283人。<ref>{{Cite web |url=http://www.city.shibuya.tokyo.jp/data/statics/tokei/02jinko.html |title=渋谷区／住民基本台帳・外国人登録による人口 |accessdate=2015-04-27 |archiveurl=https://web.archive.org/web/20150923204033/http://www.city.shibuya.tokyo.jp/data/statics/tokei/02jinko.html |archivedate=2015-09-23 }}</ref>
# 以[[澀谷站|澀谷站]]為中心的廣域地名。包括澀谷一丁目至四丁目，以及鄰近的[[道玄坂|道玄坂]]、[[宇田川町|宇田川町]]、[[神南|神南]]、[[櫻丘町_(澀谷區)|櫻丘町]]等[[街區|街區]]。
# 東京都[[東京都區部|特別區]]之一的澀谷區。
# [[明治時代|明治時代]]與[[大正時代|大正時代]]設置的澀谷村→{{link-ja|澀谷町 (東京府)|渋谷町 (東京府)|澀谷町}}。1932年編入[[東京市|東京市]]，並與周圍地區合併為澀谷區。

一般所稱的「澀谷」是指第1、2項，即以澀谷站為中心的[[市區|市街地帶]]。澀谷與[[新宿|新宿]]、[[池袋|池袋]]並列為東京三大副都心，是東京最具代表性的[[商圈|商圈]]之一，乃日本[[流行|流行]]與[[時尚|時尚]]的重要發信地，又以「[[音樂|音樂]]之街」、「年輕人之街」著稱；同時是東京通往[[首都圈_(日本)|首都圈]]東南郊區的重要[[轉運站|轉運點]]，擁有多家連鎖[[百貨公司|百貨公司]]及綜合[[商場|商場]]，為{{link-ja|東急集團|東急グループ}}的大本營。澀谷站前的[[八公|八公]]銅像是澀谷的[[地標|地標]]；澀谷站前[[交叉口|交叉口]]的[[行人保護時相|行人保護時相]]，則是東京知名的城市地景之一。

== 概要 ==
如同[[新宿|新宿]]、[[池袋|池袋]]，澀谷也是以[[山手線|山手線]]轉運車站為中心發展的商圈。以「年輕人之街」而聞名，「[[西武百貨店|西武百貨店]]」、「[[東急百貨店|東急百貨店]]」、「[[巴而可|巴而可]]」、「[[109百貨|109百貨]]」等[[百貨店|百貨店]]、專門店、餐廳林立。澀谷站前有[[忠犬八公|忠犬八公]]銅像，前方設有[[行人保護時相|行人保護時相]]。

舉辦[[1964年夏季奧林匹克運動會|東京奧運]]的1964年（[[昭和|昭和]]39年）左右，當時的澀谷以{{link-ja|百軒店|百軒店}}的電影院街與[[圓山町_(澀谷區)|圓山町]]的[[花街|花街]]為中心，充滿大人文化，後方的[[松濤|松濤]]與[[南平台町|南平台町]]是高級住宅街。

澀谷道路呈放射狀，加上多處斜坡與單行道使得道路拓寬十分困難，讓開發業者敬而遠之，造成此地比[[新宿|新宿]]、[[池袋|池袋]]相比發展較為遲緩<ref>荷風Vol.17，15頁，日本文芸社，2008年</ref>。

1973年（昭和48年），[[巴而可|巴而可]]開幕吸引許多青少年前往，1975年（昭和50年）以後取代[[新宿|新宿]]成為流行基地。

== 地理 ==
如同其名，澀谷是擁有眾多谷地地形的地方。相較之下新宿位在[[甲州街道|甲州街道]]旁的脊線而有「丘上之街」之稱，澀谷位於侵蝕[[武藏野台地|武藏野台地]]的{{link-ja|澀谷川|渋谷川}}（{{link-ja|穩田川|穏田川}}）、{{link-ja|宇田川|宇田川}}匯流處，而有「谷底之街」之稱{{efn|新宿周邊標高35m - 40m，澀谷周邊約15m左右，差距20m以上。}}。谷地兩側高低差距極大，例如{{link-ja|澀谷Mark City|渋谷マークシティ}}1樓出入口是在谷底，但位於谷上的出入口是在四樓。也因地形關係，澀谷擁有相當多的「坂」（即{{link-ja|坡道|坂}}），眾多道路與建築沿山坡順勢興建。

澀谷原有穩田川（澀谷川上游）、宇田川等河川，現在已不存在{{efn|全已暗渠化成為下水道（千駄谷幹線與宇田川幹線），澀谷站周邊水路進行大幅改造，澀谷川現與下流下水道連結。}}。兩河的下游澀谷川失去源頭，起點改到澀谷站北方地底，幾乎沒有自然水源注入。

現在在行政區劃上與[[代代木|代代木]]、[[神宮前_(澀谷區)|神宮前]]、[[青山_(東京都港區)|青山]]、[[代官山町|代官山町]]等地區鄰接。

== 歷史 ==
根據[[吾妻鏡|吾妻鏡]]等史書記載，此地在[[平安時代|平安時代]]末期至[[鎌倉時代|鎌倉時代]]是武將{{link-ja|澀谷氏|渋谷氏}}的活動範圍，[[河內源氏|河內源氏]][[源義朝|源義朝]]的近臣{{link-ja|土佐坊昌俊|土佐坊昌俊|澀谷金王丸}}也是本族族人。

[[江戶時代|江戶時代]]，{{link-ja|大山街道|大山街道}}（大約等同與現在[[國道246號|國道246號]]）沿線形成繁榮的聚落。

[[明治時代|明治時代]]，1885年（[[明治|明治]]18年）[[山手線|山手線]]開通，1911年（明治44年）都心方向的[[東京都電車|東京市電]]、1907年（明治40年）西側的{{link-ja|東急玉川線|東急玉川線|玉川電鐵}}（今[[東急田園都市線|東急田園都市線]]）通至此地，成為交通節點。1927年（[[昭和|昭和]]2年）{{link-ja|東京橫濱電鐵|東京横浜電鉄}}（今[[東京急行電鐵|東京急行電鐵]][[東急東橫線|東橫線]]）、1933年（昭和8年）帝都電鐵（今[[京王電鐵|京王電鐵]][[京王井之頭線|井之頭線]]）、1938年（昭和13年）{{link-ja|東京高速鐵道|東京高速鉄道}}（今[[東京地下鐵|東京地下鐵]][[東京地下鐵銀座線|銀座線]]）陸續通車。

{{link-ja|五島慶太|五島慶太}}領導的東京橫濱電鐵仿照{{link-ja|小林一三|小林一三}}領導的阪神急行電鐵（今[[阪急電鐵|阪急電鐵]]）在[[梅田站|梅田站]]的作法，在1934年（昭和9年）開設東橫百貨店（今{{link-ja|東急百貨店東橫店|東急百貨店東横店}}東館），在關東是次於{{link-ja|池上電氣鐵道|池上電気鉄道}}（今[[東急池上線|東急池上線]]）[[五反田站|五反田站]]{{link-ja|白木屋 (百貨公司)|白木屋 (デパート)|白木屋}}（1928年）、[[東武鐵道|東武鐵道]]淺草雷門站（今[[淺草站|淺草站]]）{{link-ja|松屋百貨|松屋 (百貨店)|松屋}}（1931年）後的第三例，也是全國第四例的轉運總站百貨店。交通上的便利成功吸引了東急沿線的居民。

1954年（昭和29年）東急會館（舊玉電大樓，今東急百貨店東橫店西館）、1957年（昭和32年）{{link-ja|東急文化會館|東急文化会館}}（今[[澀谷Hikarie|澀谷Hikarie]]），1965年（昭和40年）澀谷東急大樓（今澀谷[[東急Plaza|東急Plaza]]）、1967年（昭和42年）{{link-ja|東急百貨店本店|東急百貨店本店}}等陸續設立，澀谷成為「東急之街」。

1975年（昭和50年）以後，為了搶奪澀谷，{{link-ja|季節集團|セゾングループ|季節}}系列的[[西武百貨店|西武百貨店]]在1968年（昭和43年）進駐澀谷，接著1973年（昭和48年）[[巴而可|巴而可]]開店。之後，東急與西武展開開發競爭。

1970年代，澀谷取代[[新宿|新宿]]成為成為[[青年|年輕世代]]的流行基地；這不只影響澀谷本身，還影響了整個澀谷區。1970年代以前，新宿是年輕人文化與流行時尚的先驅。1973年（昭和48年）澀谷PARCO開幕後，年輕人文化基地從原本的新宿移至澀谷，之後擴展到包含[[原宿|原宿]]在內的澀谷區全體。

=== 地名由來 ===
地名由來主要由以下說法<ref>[http://www.city.shibuya.tokyo.jp/shibuya/profile/uraig.html 渋谷区／地名の由来] {{webarchive|url=https://web.archive.org/web/20140112044529/http://www.city.shibuya.tokyo.jp/shibuya/profile/uraig.html |date=2014-01-12 }}</ref>：
* 古時此地稱為「鹽谷之里」。後「鹽谷」（{{lang|ja|しおや}}）演變為「澀谷」（{{lang|ja|しぶや}}）。
* [[平安時代|平安時代]]末期的此地領主河崎重家，抓住闖入[[京都御所|京都御所]]的盜賊澀谷權介盛國，因此賜姓「澀谷」，領地也改稱為「澀谷」。
* 此地河水富含鐵質，呈現紅色的「澀柿色」（{{lang|ja|シブ色}}），稱為「{{lang|ja|シブヤ川}}」。
* 澀谷川流域低地稱為「{{lang|ja|しぼんだ谷}}」。

== 特色 ==
=== 文化 ===
[[File:PARCO_Shibuya_Part1-2.jpg|thumb]]
澀谷PARCO劇場、Club Croisee、CINESAISON澀谷、Studio PARCO等[[音樂展演空間|音樂展演空間]]、[[劇場|劇場]]、[[電影院|電影院]]林立，常舉行電影祭與音樂祭。也有許多上映各種個性作品的電影院。像是舉辦[[篠山紀信|篠山紀信]]寫真展等多項企劃展的PARCO MUSEUM（2007年閉館）、新資訊發送基地的PARCO FACTORY、LOGOS GALLERY等常舉辦具有高度社會性的藝術展覽。「TOKYO FM澀谷西班牙坂工作室」等眾多廣播公司的衛星工作室也聚集於此。東急計畫在東急文化會館舊址建設都內最大規模的[[音樂劇|音樂劇]]劇場。

=== 流行 ===
[[File:Center_Gai-dusk-20071001.jpg|thumb]]
1970年代起，PARCO、OIOI的進駐與[[109百貨|SHIBUYA 109]]的誕生，使得澀谷與[[原宿|原宿]]並列為年輕人的[[流行|流行]]文化基地。1990年代，「澀谷辣妹」（ギャル）風潮引起大眾媒體的注目。以20、30歲OL為主力的百貨店是成人時尚的領導者，10多歲的少女則信奉以109與PARCO、路邊小店引領的辣妹時尚。受到辣妹時尚影響的「辣男」（ギャル男）與「兄系」（お兄系）等風潮也從澀谷推廣到全國各地。澀谷辣妹的大型活動{{link-ja|澀谷Collection|渋谷コレクション}}極為知名。此外澀谷也有許多[[古着|古著]]店。

=== 資訊科技 ===
[[資訊科技|資訊科技]]相關的[[創業公司|創業公司]]以車站南邊的[[櫻丘町_(澀谷區)|櫻丘町]]為中心。在[[國土交通省|國土交通省]]的調查中，本地軟體相關的IT企業數量僅次於[[千代田區|千代田區]]與[[港區_(東京都)|港區]]，排名第三，以站別來看[[澀谷站|澀谷站]]僅次於[[秋葉原站|秋葉原站]]名列第二，有「Bit Valley」之稱{{efn|「澀谷」直譯成英文的「bitter valley」，加上[[位元|位元]]的「Bit」，成為宛如[[美國|美國]][[矽谷|矽谷]]（Silcon Valley）的「'''Bit Valley'''」（ビットバレー）（[http://pc.nikkeibp.co.jp/word/page/10021438/#DIC PC Online]，[http://d.hatena.ne.jp/keyword/%A5%D3%A5%C3%A5%C8%A5%D0%A5%EC%A1%BC ?ダイアリー]）。}}。
