'''指南针'''，又稱'''羅盤'''或'''司南'''，是一种用于指示方向的工具，广泛应用于各種方向判讀，譬如[[航海|航海]]、[[野外探险|野外探险]]、城市道路、地圖閱讀等领域。

指南針可分為依靠磁力與不依靠磁力兩類。依靠磁力的指南針又可再細分為兩類：一是根据地球[[磁场|磁场]]的有极性制作的地磁指南针，但这种指南针指示的南北方向与真正的南北方向不同，存在[[磁偏角|磁偏角]]；二是电子指南针，采用磁场传感器的[[磁阻|磁阻]]（MR）技术，可很好地修正磁偏角的问题，现已大量用于[[GPS|衛星]]定位装置中。至於不依靠磁力的則純粹依靠器件內部的物理方法指示方向，如[[指南車|指南車]]或[[慣性導航系統|慣性導航系統]]。

指南針又稱指北針，因為當代指向針常在指向北方的一端漆上顏色標記，以便使用者尋找北方。

== 司南 ==
有一种观点认为中国古代记载的“[[司南|司南]]”就是指南针最早形式（戰國時[[韓非子|韓非子]]《有度篇》即已出現[[司南|司南]]一詞）。东汉王充（公元27～约97）《论衡》（明嘉靖版）卷十七是应篇第五十二说“[[司南|司南]]之杓，投之於地，其柢指南”，这个记载虽不是最早，但是最早表述清楚的一个。[[司南|司南]]具体形式却有很大争议，根据《论衡》以及唐韦肇《瓢赋》中的记载，王振铎判定[[司南|司南]]为勺型天然磁石配合地盘<ref>王振铎《司南指南针与罗经盘》，《中国考古学报》第3册，1948</ref>，虽有出土汉地盘和河南南阳东汉墓出土石刻[[司南|司南]]勺图佐证，但毕竟无完整配合出土。1950年代钱临照试图以天然磁石制做勺形[[司南|司南]]，但因天然磁石磁距小、底部摩擦大而未成功<ref>李志超《再议司南》。http://lizhichao35.bokee.com/view.b?diaryId=10098399</ref>。

将[[司南|司南]]理解为磁勺从古文献考证及实验方面在学术界一直存有异议。其中东北师范大学教授刘秉正在1956年考证出指南鱼是中国人最早地进行人工磁化的应用，同时也对[[司南|司南]]是磁性指南工具提出质疑<ref>刘秉正《我国古代关于磁现象的发现》。《物理通报》，1956第8期。{{cite web |url=http://www.ylss.cn/lsqk/201101/20110121.htm |title=存档副本 |accessdate=2011-05-18 |deadurl=yes |archiveurl=https://web.archive.org/web/20131101160315/http://www.ylss.cn/lsqk/201101/20110121.htm |archivedate=2013-11-01 }}</ref>。近些年来他进一步提出《论衡·是应篇》以及《瓢赋》中的[[司南|司南]]是天上的北斗， 而其它的文献中的[[司南|司南]]或指南有的是指南车如《鬼谷子》记载的司南，有的是行事的准则如《抱朴子·外篇·疾谬》中的指南，还有是指官职如《韓非子》中的记载<ref>刘秉正《再论司南是磁勺吗？》。《自然科学史研究》，2006，第25卷第三期。http://www.ylss.cn/lsqk/201101/20110127.htm{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。2005年中国国家博物馆研究员孙机近根据前北平历史博物馆旧藏残宋本《论衡》认定“[[司南|司南]]之杓”杓字实为酌，那么看来“其柢指南”之柢也非指勺柄<ref>孙机 《简论“司南”兼及“司南佩”》。《中国历史文物》，2005，第4期</ref>。根据最新的文献考证表明，司南是北斗的别称，同时司南的其他用法也都与北斗有关<ref>刘亦丰, 刘亦未, 刘秉正 《司南指南文献新考》 自然辩证法通讯 2010年05期。http://blog.sina.com.cn/s/blog_629342ab0100qzm7.html</ref>。此文通过对北魏·温子升《定国寺碑》中“幽隐长夜，未睹山北之烛；沉迷远路，讵见司南之机。”、梁·吴均《酬萧新浦王洗马诗二首》中的“独对东风酒，谁举指南酌”等唐以前文献的分析，认为这些文献中司南只能解释成北斗，而《论衡》，《瓢赋》中的司南也应是北斗。而元稹《加裴度幽镇两道招抚使制》中的司南应是官职，宋·释正觉《颂古》：“妙握司南造化柄，水云器具在甄陶。”中的司南应是权力，唐·宋暠《獬廌赋》中“守法者仰之以司南，疾恶者投之於有北。”的司南是法律，这些词意都来源于北斗。另外《韩非子·有度》中的司南也应解释成法律，其中的“朝夕”不是方向，而是早晚朝见或君王早晚听政。其他司南引申为定时间（杜甫《咏鸡》中的司南）、指德高望重的人(张九龄《祭张燕公文》)、以及指导准则等意也来源于北斗。此文认为目前司南是磁性指南工具的论据还远不充分。

== 指南魚 ==
[[File:武经总要_(888).jpg|thumb]]
[[曾公亮|曾公亮]]成书于1044年的《[[武经总要|武经总要]]》前集卷十五中有关于水浮指南鱼的记载：{{quote|若遇天景曀霾，夜色螟黑，又不能辨方向，则当纵老马前行，令识道路，或出指南车或指南鱼以辨所向。指南车法世不传。鱼法以薄铁叶剪裁，长二寸、阔五分，首尾锐如鱼形，置炭火中烧之，候通赤，以铁钤钤鱼首出火，以尾正对子位，蘸水盆中，没尾数分则止，以密器收之。用时，置水碗于无风处，平放鱼在水而令浮，其首常南向午也。}}李约瑟指出，铁皮从高于[[居里点|居里点]]（600℃-700℃）的温度急速降温，就会被地磁场磁化成为磁铁。这种利用地磁场制作磁铁的方法，虽然磁性较弱，但具有不需天然磁铁的优点<ref>[[李约瑟|李约瑟]]著，王玲协助 鲁宾孙 特别协助《中国科学技术史》第四卷 第一分册《物理学》235页 科学出版社2003 ISBN 7-03-011232-6</ref>。上文中的“道路”、“指南鱼以辨所向”等字句，说明指南针在1044年已经用于陆路交通了。

世界上最早的关于用天然磁石磨钢针的方法造磁针，以及人造磁针指南的记载，出现在[[北宋|北宋]][[沈括|沈括]]《[[梦溪笔谈|梦溪笔谈]]》<ref>[[沈括|沈括]]《[[梦溪笔谈|梦溪笔谈]]》卷二十四 杂志一</ref>：{{quote|方家以磁石磨针锋，则能指南，然常微偏东，不全南也。水浮多荡摇。指爪及碗唇上皆可为之，运转尤速，但坚滑易坠，不若缕悬为最善。其法取新纩中独茧缕，以芥子许蜡，缀于针腰，无风处悬之，则针常指南。其中有磨而指北者。余家指南、北者皆有之。}}

其中记载：
* 方家用磁石磨针锋，制造指南针。
* 四种支撑磁针的方法：水浮（水磁针）、放在指甲上、放在碗边，用一根蚕丝悬挂磁针（旱磁针）。
* 磁偏角现象。
* 亦有人制造指北针

== 旱罗盘 ==
元初《[[事林广记|事林广记]]》记载<ref>[[李约瑟|李约瑟]]原著 柯林·罗南改编 《中华科学文明史》 3 上海交通大学科学史系翻译 上海人民出版社 16-17页 ISBN 7-208-03966</ref>：{{quote|以木刻鱼子，如拇指大，陷好磁石一块子，却以蜡填满，用针一半佥从鱼子口中钩入，令没水中，自然指南。以手拨转，又复如此。以木刻龟子一个，一如前发制造，但于尾边敲针入去，用小板子，上安以竹钉子，如箸尾大，龟腹下微陷一穴，安钉子上，拨转常指北。}}

== 元、明航海罗盘的運用 ==
[[File:郑和航海图五.jpg|thumb]]
最早见于航海用途的指南针记载于[[北宋|北宋]][[朱彧|朱彧]]撰寫的《[[萍洲可谈|萍洲可谈]]》<ref>《[[萍洲可談|萍洲可談]]》卷二 </ref>：{{quote|[[舟师|舟师]]识地理，夜则观星，昼则观日，阴晦观指南针，或以十丈繩鉤，取海底泥嗅之，便知所至。海中無雨，凡有雨則近山矣。……}}《萍洲可谈》成书於1111年-1117年间，但所叙述的是1086年的事<ref>《中华科学文明史》第三卷，[[李约瑟|李约瑟]]原著，柯林，罗南改编，上海[[交通大学|交通大学]][[科学史|科学史]]系译ISBN 7-208-03966-6</ref>。这比Alexander Neckam可能写於1190年的《论器具（De naturis rerum）》中说法：航海时“白天云遮太阳，或夜间黑暗不辨方向时，使用磁铁摩擦铁针，针停时指南北，”早约一个世纪。与英国几乎同时法国和稍晚一点的意大利也有了首次指南针的记载。

1123年宋朝派遣使臣取海路出使朝鲜，《宣和奉使高丽图经》记载用水浮指南针导航<ref>[[李约瑟|李约瑟]]原著 柯林·罗南改编 上海交通大学科学史系翻译 第3卷 第33 页 上海人民出版社 2002 ISBN 7-208-03966-6</ref>：{{quote|是夜洋中不可住，维观星斗前迈。 若晦冥，则用指南浮针， 以揆南北。}}

宋代的航海指南针，除“南北”，还未见有其他方位，如：東、西、西北等方向。

四十八方位罗盘用于航海的文献纪录，最早见于元代[[周达观|周达观]]《[[真腊风土记|真腊风土记]]<ref>元代[[周达观|周达观]]《[[真腊风土记|真腊风土记]]》总序15页 中华书局2006 ISBN 7-101-02028-3</ref>》：{{quote|自温州开洋，行[[海道针经#双针罗经|丁未针]]。历闽、广海外诸州港口，过[[七洲洋|七洲洋]]，经交趾洋到[[占城|占城]]。又自占城顺风可半月到[[真蒲|真蒲]]。又自真蒲行坤申针，过昆仑洋入港。}}其中丁未针= 202.5°、坤申针= 232.5°<ref>同上 25页，夏鼐注⑤</ref>。

明朝[[永乐_(明朝)|永乐]]年间随[[郑和下西洋|郑和下西洋]]的[[巩珍|巩珍]]在《[[西洋番国志|西洋番国志]]》的《自序》中叙述应用水浮罗盘的情况<ref>明[[龚珍|龚珍]]《[[西洋番国志|西洋番国志]]·自序》 向达校注本 第5页 中华书局 2000 ISBN 7-101-02025-9</ref>：{{quote|往还三年，经济大海，绵邈弥茫，水天连接，四望迥然，绝无纤翳之隐蔽。惟日月升坠，以辨西东，星头高低，度量远近。斫木为盘，书刻干支之字，浮针于水，指向行舟。}}

《[[郑和航海图|郑和航海图]]》中纪录109道[[海道针经|航海针路]]。例如：
* “[[太仓港|太仓]]港口开船，用丹乙针，一更，船平[[吴淞江|吴淞江]]。用乙卯针，一更，船到[[南汇|南汇]]嘴。平[[镇海|招宝]]”。（郑和航海图五）
* “[[苏门答腊|苏门答腊]]开船，用[[海道针经#双针罗经|乾戌针]]，十二更，船平[[龙涎屿|龙涎屿]]。四十更，船又用辛酉针。五十更，船见[[锡兰山|锡兰山]]”；（郑和航海图十八 乾戌= 307.5° ，辛酉= 277.5°）
* “[[官溜屿|官溜屿]]用庚酉针，船收[[木骨都束|木骨都束]]”。（郑和航海图二十。庚酉= 262.5°）
明代《[[顺风相送|顺风相送]]》、《[[指南正法|指南正法]]》、《[[东西洋考|东西洋考]]》等文献都有针路记录：
* [[广东|广东]]往[[磨六甲|磨六甲]]：南亭门放洋，用坤未针 （217.5°）五更船取乌头山。用单坤针（西南 225°）十三更取七洲洋。坤未 （217.5°）针 七更船平独猪山。……乾亥 （322.5°）针五更船平昆宋屿，单亥针（330°）五更船取前屿，乾（315°）针五更取五屿；沿山使取磨六甲。<ref>[[向达|向达]]校注 《[[顺风相送|顺风相送]]》 55页 中华书局 ISBN 7-101-02025-9</ref>
* [[台湾|台湾]]往[[日本|日本]]从[[大港|大港]]出。东南风可用丁未 （202.5°）及单未 （210°）过茄老湾线。南到青水乌水乾，可牵舵及用壬（345°）及壬子（352.5°），转变取[[澎湖|澎湖]]东过。<ref>向达校注 《指南正法》 133页 中华书局 ISBN 7-101-02025-9</ref>。
* [[七洲洋|七洲洋]]用坤未针，三更取铜鼓山。<ref>明[[张燮|张燮]]东西洋考 173页 中华书局 ISBN 7-101-02029-1</ref>

明代[[舟师|舟师]]使用的罗盘，用水浮针，有一定的规矩<ref>向达校注 《[[顺风相送|顺风相送]]》 25页 ISBN 7-101-02025-9</ref>：{{quote|取水下针，务要阳水，不取阴水”，“安罗经，下指南（针），须从乾宫下”。下针之前，舟师必须诵读《地罗经下针神文》，拜[[周公|周公]]圣人，[[李淳风|李淳风]]仙师，定针童子、转针神郎、换水神君、下针力士、走针神兵、罗经坐向守护尊神、护国庇民妙灵昭应明著[[天妃|天妃]]等神灵，祈求“今日良辰下针，青龙下海永无灾。}}

近来曾在青岛和海南发现明代瓷质水浮指南针，尺寸大至为：最大外径9.5厘米；上有盛水同心圆孔径4.4厘米，孔深2.3厘米，外围青花釉绘刻度，底座、四壁厚实坚固，估计是去水收藏，用时加水。现代液体罗盘基本原理类似，不过改进采用了密封技术。

== 现代指南针 ==
中国在12世纪已经发明旱[[罗盘|罗盘]]即现代指南针基本形式。经过阿拉伯人有效改进后才传入欧洲。但是欧洲现存关于指南针的记载早于阿拉伯，阿拉伯语中指南针（al-konbas）一词也似源于古意大利语。近来在江西临川一座葬于公元1198年的宋墓里发现一俑（标写“[[张仙|张仙]]人”），手持一似风水罗盘“亦是旱罗盘”物，有观点认为旱罗盘中国也可能早于西方使用，但可能先发明于江南沿海（如泉、广）一带应用，故博学如沈括也未见到，今后尚须考古文物工作者努力。内陆后世旱罗盘则多用于风水罗盘和校时罗盘，如174/l189曾三异《同话录》“地螺，或有子午正针，或用子午丙壬间缝针。天地南北之正，当用于午。”“地螺”可能是一种校时罗盘，主要用途是校时。
[[File:放在士林公民會館永福磁石上而狀態相反的指北針.jpg|thumb]]<ref name="呂一銘">{{Cite news|url=|title=反經石怪而不奇 勞動了兩位博士 固體物理學家趕往現場求證 祇是一堆小磁鐵組成的石頭|author=呂一銘|language=zh-tw|publisher=《聯合報》|date=1974-09-23|accessdate=}}</ref>。]]

另外，越靠近[[兩極地區|兩極地區]]，指南針就越沒辦法發揮功效。在[[北極點|北極點]]時，地表上的任何方向都可以被視為南方；在[[南極點|南極點]]時，地表上的任何方向都可以被視為北方。
