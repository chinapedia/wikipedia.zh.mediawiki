《'''洛克人零'''》系列（日文：{{lang|ja|ロックマンゼロ}}；英文：{{lang|en|Mega Man Zero}}），是[[日本|日本]]電子遊戲《[[洛克人|洛克人]]》系列第五部作品。於2002年發行，屬動作遊戲，共四集。

== 遊戲 ==

=== 對戰系統 ===

《洛克人Zero》跟《洛克人EXE》系列一樣有屬性相剋系統。裡面有四種不同的屬性，分別為火屬性、冰屬性、雷電屬性與無屬性。屬性大大影響著敵方的力量或者是抵抗能力，當特定屬性的攻擊擊中對該屬性有較低抵抗力的屬性對象時，該屬性的攻擊力則會變成兩倍，但相反用對手有較高抵抗力的屬性時，攻擊力則變成零。屬性的相剋關係則可以用下方的圖示表示（箭頭所指到的屬性表示該屬性被箭頭另外一方的屬性給剋到）：

== 專有名詞介紹 ==
'''思考型機器人(レプリロイド/Repliod)'''：自久遠以前被發明出來，與人類相近的機器人。由高度科技製造，灌輸與人類相似的人格；原先是做為人類的夥伴而設計，與人類一同工作。<br>
'''Irregular(イレギュラー)'''：有過襲擊人類記錄的Repliod機器人，被人類所畏懼。在X系列中譯作非正規機器人。<br>
'''Neo Arcadia(ネオ・アルカディア)'''：曾經是人類的理想鄉，為了保護人類不被Irregular襲擊的城市。但原本統領國家的艾克斯失蹤，後由雪兒製造的複製艾克斯統治(Z1)。<br>
'''反抗軍(レジスタンス/Resistance)'''：一個對抗Neo Arcadia的私人武裝軍隊，由雪兒管轄，之後她和一夥人失去了連絡後，由普蕾莉接任。<br>

== 主要人物 ==

; [[傑洛|傑洛]]（{{lang|ja|ゼロ}}╱ {{lang|en|Zero}}）
: 配音員：[[風間勇刀|風間勇刀]]
: 本作主人公，曾跟[[艾克斯_(洛克人)|艾克斯]]在100年前的非正規品戰爭中活躍的英雄。被雪兒發現，從100年的沉睡中蘇醒的傳说的思考型機器人。因為沒有任何去向，又了解反抗軍的苦況，而成為反對派一方的戰士，長期接受反抗軍給予的任務，幫助被迫害的反對派,與Neo Arcadia對戰。其實傑洛現在的身體並不是他真正的身體，他真正的身體在妖精戰爭前被拜魯博士奪走，之後艾克斯將他的靈魂放進複製的身體，與他一起解決妖精戰爭。

; 雪兒（{{lang|ja|シエル}}╱ {{lang|en|Ciel}}）
: 配音員：[[田中理惠_(聲優)|田中理惠]]
: 本作女主角。在胎兒時期，接受了DNA改造，使她有開發機器人與兵器的優秀技術。在艾克斯失蹤後，製造了複製艾克斯擔任Neo Arcadia的新領導人，但是複製艾克斯卻以不正當的理由大量屠殺無辜的思考型機器人。無法接受的她離開了Neo Arcadia，聚集被視為非正規機器人的逃亡者組成了反抗軍(レジスタンス/Resistance)，進行著拯救準備以能源浪費為罪名而處分的思考型機器人的活動。不過，因為被Neo Arcadia鎮壓，被追到絕境的反抗軍來到了在遙遠過去被封印，被人稱做「英雄」的傑洛所在的遺跡。

; 電子艾克斯（Cyber-X）
: 電子精靈，與其他的電子精靈不同,帶着神秘的氣氛，把劍交給傑洛，和他好像有着很密切的關係……。
: 根據官網說明，他就是在100年前與傑洛在非正規品戰爭一起戰鬥的[[艾克斯_(洛克人)|艾克斯]]。

=== Neo Arcadia ===

; 複製艾克斯（{{lang|ja|コピーエックス}}╱ {{lang|en|COPY X}}）
: 配音員：[[今井由香|今井由香]]
:在艾克斯為了封印闇之妖精失蹤後，Neo Arcadia上級命令雪兒尋找代理人，而被雪兒製造出來的、模仿艾克斯的思考型機器人。但是追求和平的理念卻與艾克斯不同；為了成為英雄，他不惜處死大量無辜的思考型機器人。平常是藍色裝甲和黑色皮膜的機器人，戰鬥時會在身上變出金色、白色相間的天使型裝甲，原型是一架沒有腳的六翼天使型機器人。
:在第一代被傑洛打敗，但是在第三代被拜魯復活，並且對拜魯有極大的依賴性；為了奪走闇之妖精，甚至在人類生存區投下帶著奧米加的炸彈。在第三代再次輸給傑洛後，企圖變回原型，可是被引爆拜魯埋藏在體內的炸彈而炸死。

; 赫爾琵亞（{{lang|ja|ハルピュイア}}╱ {{lang|en|Harpuia}}）
: 配音員：[[緒方惠美|緒方惠美]]
: 是以艾克斯的DNA製造的X-bioroid（X-人造人），是為了再生環境與人類的生活圈而製造的天氣控制用思考型機器人，能操做局部的氣象。Neo Arcadia四大天王之一队长賢將，稱號是翠綠的斬擊，率領烈空軍團。一開始是個性格激動的不良少年，但在複製艾克斯死後，轉任Neo Arcadia的代理指揮，性格漸趨成熟。与Leviathan合作中，主要任务是扩大人类环境的生活，Harpuia前名字的由来是，希腊神话中登场的妖鸟"哈比"的别名。使用的武器是兩把光束劍 是十大闪光武器之一（音速刀锋），第二型態能夠變成一台鳥型的戰鬥機。

; 法布尼爾（{{lang|ja|ファーブニル}}╱ {{lang|en|Fefnir}}）
: 配音員：[[中井和哉|中井和哉]]
: 是以艾克斯的DNA製造的X-bioroid（X-人造人），是為了淨化被汙染的土地而製造的思考型機器人。Neo Arcadia四天王之一的鬥將，稱號是紅蓮的豪腕，率領塵炎軍團。性格單純而好戰，渴望有個能全力一戰的對手。使用的武器是兩具重型火砲，第二型態能變成一台有四個龍頭砲口的戰車。

; 蕾薇亞丹（{{lang|ja|レヴィアタン}}╱ {{lang|en|Leviathan}}）
: 配音員：[[今井由香|今井由香]]
: 是以艾克斯的DNA製造的X-bioroid（X-人造人），與赫爾琵亞的製造理念相同，是為了淨化海洋而製造的思考型機器人，有操縱水流與水溫的能力。Neo Arcadia四天王之一的妖將，稱號是蒼海的海神，率領冥海軍團。是四天王中最早被製造出來的，有一盯上獵物便會窮追不捨的個性，因此對傑洛非常執著，為了打倒傑洛連原本的目的都能擱置。使用的武器是能發射飛彈的長槍，第二型態能變成魟魚型的潛水艇。

; 幻影（{{lang|ja|ファントム}}╱ {{lang|en|Phantom}}）
: 配音員：[[稻田徹|稻田徹]]
: 是以艾克斯的DNA製造的X-bioroid（X-人造人），並非為了再生地球而製造的思考型機器人，具有光學迷彩和高機動力。Neo Arcadia四天王之一的隱將，稱號是漆黑的幻影，率領斬影軍團。沉默寡言的溫厚性格，說話時用古老的語言，所以其他三天王都甚少和他聯繫，總是孤軍奮戰，但對X的忠誠心沒有動搖。玩家認為用信念這詞來形容他最合適不過了。在第一代最終關卡敗給傑洛之後，企圖與傑洛同歸於盡而自爆身亡。但是死後靈魂在電子空間活動，可以在第三代看到他。使用的武器是飛鏢和忍者刀，雖然遊戲中沒出現（加上本人在第一代自爆身亡），不過官方表示幻影的第二型態是架蝙蝠型的隱型戰鬥機。

=== 主要敵方 ===

; 拜魯博士 （{{lang|ja|Dr.バイル}}）
: 配音員：[[大塚周夫|大塚周夫]]
: 本作的最終魔王。在過去的非正規品戰爭結束後，奪走傑洛原本的身體改造成奧米加，並且將馬薩妖精(マザーエルフ)改造成闇之妖精(ダークエルフ)，使用複製自黑暗妖精製造的幼兒妖精(ベビーエルフ)控制大量的思考型機器人，發起了名為『妖精戰爭』的戰爭，造成了九成的思考型機器人和六成的人類死亡。戰爭結束後被追究戰爭的責任，記憶被轉換成數據資料，身體被安裝了具有再生能力的盔甲後，與奧米加一同被放逐到宇宙漂流了一百年。在闇之妖精復活後歸來，先是復活了複製艾克斯重歸Neo Arcadia，接著設計複製艾克斯自爆奪走Neo Arcadia的政權，之後用奧米加和闇之妖精控制了大部分的思考型機器人，企圖發起第二次妖精戰爭，但是被傑洛阻止。第四代依然掌握Neo Arcadia的大權，為了控制所有人類，發起破壞自然的Ragnarok作戰。所做的一切，都是為了向放逐他的人類進行報復。

; 奧米加（Omega）
: 配音員：[[諏訪部順一|諏訪部順一]]
:拜魯製造出來的惡魔思考型機器人，是以傑洛原本的身體改造而來的，與闇之妖精結合能發揮出無窮的力量。但是拜魯如何入手傑洛的身體，官方沒有做出解釋。

; 厄爾畢斯（Elpizo／Elpis）
: 於《洛克人Zero2》登場，初登場時為反對派新領袖。原為阿卡迪亞代號TK31的機器人，在某次任務中因為閱覽了妖精戰爭時的資料，而被Neo Arcadia認定為非正規品，之後加入反對派，並改名為「厄爾畢斯」(有『希望』之意)。在一代到二代的過度中，領導反對派重整旗鼓並建立了新的基地，因著這份功績被忙於研究的雪兒推舉為新的反對派領導者，性格因此膨脹起來。
: 在發起『正義的一擊』作戰失敗後，因為無法接受作戰失敗並造成部下死亡的愧疚與罪惡感，還有對傑洛的自卑感，以及對力量的渴望下，決定解放一百年前的黑暗妖精以獲得無人能及的力量。最後雖然破壞了艾克斯的身體並解放黑暗妖精，還是不敵傑洛而戰敗。在死前終於坦然面對自己的過錯，並被黑暗妖精變為電子妖精離去。
