'''洛桑大学'''（{{lang-fr|Université de Lausanne}}，缩写UNIL），是[[瑞士|瑞士]]的一所[[大学|大学]]，位於西南部[[沃州|沃州]]的[[洛桑|洛桑]]市。该校於1537年成立，1890年升格为大学。<ref name="history">{{cite web |url=http://www.unil.ch/central/page43591_en.html |title=Yesterday... and today |publisher=洛桑大学 |date= |accessdate=2008年12月24日 |language=en }}</ref><ref name="swissinfo">{{cite web|url=http://www.swissinfo.org/chi/education/detail.html?siteSect=525&sid=1264634&cKey=1158935530000 |title=洛桑大学(Université de Lausanne) |publisher=瑞士资讯网 |date= |accessdate=2008年12月24日 |language= }}</ref>洛桑大学现有2000多名教职人员和1万多名学生。<ref name="stat"/>

2005年以前，洛桑大学采用[[法国教育|法国教育]]模式，但有细微差别。目前已实施[[博洛尼亚进程|博洛尼亚进程]]。<ref name="history_21">{{cite web |url=http://www.unil.ch/central/page43771_en.html |title=The UNIL in the 21st century |publisher=洛桑大学 |date= |accessdate=2008年12月24日 |language=en }}</ref>洛桑大学与[[洛桑联邦理工学院|洛桑联邦理工学院]]一起在[[莱芒湖|莱芒湖]]畔组成了巨大的校区。

== 历史 ==
[[File:University_of_Lausanne.jpg|thumb]]
洛桑大学的前身是成立于1537年的Schola Lausannensis，首要任务是为[[教堂|教堂]]培养神职人员。该校当时拥有全国惟一一所法语[[新教|新教]][[神学院|神学院]]。<ref name="history"/><ref name="history_16">{{cite web |url=http://www.unil.ch/central/page43776_en.html |title=The UNIL in the 16th century |publisher=洛桑大学 |date= |accessdate=2008年12月24日 |language=en }}</ref>1711年，让·巴贝拉克（Jean Barbeyrac）破除了当时在大学里使用[[拉丁语|拉丁语]]的传统，在其就职演说上使用法语。<ref name="history_18">{{cite web |url=http://www.unil.ch/central/page43774_en.html |title=The UNIL in the 18th century |publisher=洛桑大学 |date= |accessdate=2008年12月24日 |language=en }}</ref>1833年，[[教育|教育]]学院成立。1853年，[[工程|工程]]学院的前身“洛桑特别学院”成立。1873年，[[药学|药学]]院成立。1890年5月10日，历经数百年不断壮大发展的学校获得大学地位及名称，由亚历山大·毛雷尔（Alexander Maurer）任校长，注册学生约有300人。[[医学院|医学院]]同期成立。三年后，[[物理|物理]]与[[化学|化学]]学院成立。<ref name="history_19">{{cite web |url=http://www.unil.ch/central/page43773_en.html |title=The UNIL in the 19th century |publisher=洛桑大学 |date= |accessdate=2008年12月24日 |language=en }}</ref>

20世纪上半叶，[[社会|社会]]与[[政治科学|政治科学]]学院、[[经济|经济]]与[[工商管理|工商管理]]学院、[[建筑学|建筑]]学院（隶属于工程学院）相继成立。<ref name="history_20">{{cite web |url=http://www.unil.ch/central/page43772_en.html |title=The UNIL in the 20th century |publisher=洛桑大学 |date= |accessdate=2008年12月24日 |language=en }}</ref>1970年，洛桑大学逐步从洛桑老城区搬迁到现在位于莱芒湖畔的新校区，当时学生数达到3000人。<ref name="history"/><ref name="history_20"/>20世纪末，瑞士法语区内洛桑、[[日内瓦|日内瓦]]、[[纳沙泰尔|纳沙泰尔]]等地使用法语教学的各大学进一步加强院校间的合作。<ref name="history"/>2001年，洛桑大学、[[日内瓦大学|日内瓦大学]]和洛桑联邦理工学院共同签署了三校之间的合作协议，进行科研和教学上的合作。<ref name="history_21"/>2003年，洛桑大学增设了[[生物|生物]]与医学院和[[地球科学|地球科学]]与[[环境|环境]]学院。<ref name="history"/><ref name="history_21"/>

== 教学机构 ==
洛桑大学现有7个学院，具体为：[[神学|神学]]与[[宗教|宗教]]研究学院、[[法律|法律]]与[[司法|司法]]学院、[[艺术|艺术]]学院、社会与政治科学学院、[[工商|工商]]与经济学院（[[洛桑高等商学院|洛桑高等商学院]]）、地球科学与环境学院、生物学院和医学院。<ref name="swissinfo"/><ref name="faculties">{{cite web |url=http://www.unil.ch/central/page2863_en.html |title=Faculties |publisher=洛桑大学 |date= |accessdate=2008年12月24日 |language=en }}</ref>法律与[[犯罪学|犯罪学]]、生物学、医药学、[[地理学|地理学]]、环境学等学科是洛桑大学实力较强的学科。<ref name="swissinfo"/>

== 附属研究所 ==
洛桑大学下辖的职业与健康学院（Institute for Work and Health, IST）<ref>http://www.i-s-t.ch/en/</ref>位于洛桑大学医院，是由洛桑大学<ref>http://www.unil.ch</ref>和日内瓦大学<ref>http://www.unige.ch/</ref>共同管理的一所附属研究所。其研究经费由瑞士沃州和日内瓦州州政府共同分担。其宗旨为增进职业健康领域的教学实力和科研水平，并为外界提供专业咨询。

IST是瑞士唯一一所在职业健康领域具有一定规模的研究机构，其专业水平在瑞士国内和国际上享有良好的声誉。在职业健康领域，IST亦是世界卫生组织合作中心伙伴之一<ref>{{cite web |url=http://www.i-s-t.ch/en/who-we-are/ |title=存档副本 |accessdate=2015-07-12 |deadurl=yes |archiveurl=https://web.archive.org/web/20150924032107/http://www.i-s-t.ch/en/who-we-are/ |archivedate=2015-09-24 }}</ref>。

== 校区 ==
[[File:UNIL-p1010656.jpg|thumb]]
[[File:Bibliothèque_cantonale_et_universitaire_de_Lausanne.jpg|thumb]]

洛桑大学的校区位于莱芒湖畔，与洛桑联邦理工学院毗邻。校区各栋建筑间建有公园和[[植物园|植物园]]。[[图书馆|图书馆]]位于中心位置，视野开阔，可以眺望到莱芒湖以及[[阿尔卑斯山脉|阿尔卑斯山脉]]，天气晴朗时甚至可以看到[[勃朗峰|勃朗峰]]。

== 声誉 ==
洛桑大学在瑞士本國、[[欧洲|欧洲]]及至世界上都具有良好的声誉。2014/15年[[QS世界大学排名|QS世界大学排名]]将洛桑大学排在瑞士第5位、世界第105位<ref name="qs">{{cite web |url=http://www.topuniversities.com/university-rankings/world-university-rankings/2014#sorting=rank+region=+country=+faculty=+stars=false+search= |title=QS World University Rankings® 2014/15 |publisher=QS Quacquarelli Symonds Limited |date= |accessdate=2015年7月12日 |language=en }}</ref>。莱顿大学基于科学文献引用率的排名，将洛桑大学排在世界前250所大学中的第41位（不考虑学校规模）<ref name="Leiden">{{cite web |url=http://www.leidenranking.com/ranking/2015 |title=CWTS Leiden Ranking 2015 |publisher=Centre for Science and Technology Studies, Leiden University, The Netherlands |date= |accessdate=2015年7月12日 |language=en }}</ref>。[[英国|英国]]《[[泰晤士高等教育|泰晤士高等教育]]》2014-2015年世界大学排名将洛桑大学排在世界第136位<ref name="the_eu">{{cite web |url=http://www.timeshighereducation.co.uk/world-university-rankings/2015/world-ranking/#/ |title=World University Rankings 2014-15 |publisher=[[泰晤士高等教育|泰晤士高等教育]] |date= |accessdate=2015年7月12日 |language=en }}</ref>，其中生命科学领域名列世界第56位<ref name="the_life">{{cite web |url=http://www.timeshighereducation.co.uk/world-university-rankings/universite-de-lausanne?ranking-dataset=1087 |title=University of Lausanne - Life Sciences |publisher=[[泰晤士高等教育|泰晤士高等教育]] |date=2015年7月12日 |accessdate=2015年7月12日 |language=en }}</ref>。[[上海交通大学|上海交通大学]]2014年[[世界大学学术排名|世界大学学术排名]]将洛桑大学排在瑞士第6-7位、世界第151-200位。<ref name="arwu">{{cite web |url=http://www.shanghairanking.cn/ARWU2014.html |title=2014世界大学学术排名五百强 |publisher=世界大学学术排名（AWRU） |date= |accessdate=2015年7月12日 |language= }}</ref>
