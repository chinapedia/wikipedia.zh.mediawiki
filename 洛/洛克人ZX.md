{{vgname|洛克人ZX|ja=ロックマンゼクス|en=Mega Man ZX}}是[[洛克人|洛克人]]第六系列、[[洛克人ZX系列|洛克人ZX系列]]的第一集。2006年[[洛克人系列|洛克人系列]]第二線製作。

新的系列承繼了[[洛克人零系列|洛克人零系列]]的故事，但兩系列的人物連接則較間接。今次是有史以來，洛克人的遊戲中出現女性主角。

== 故事大綱 ==
[[地球|地球]]的環境經歷屢次思考型機械人大戰後被嚴重破壞，經過前人的努力終於露出大規模復甦的曙光。國家們在魔蛇公司的協助下成功建立都市、將非正規機械驅在國外，但仍在各國境間出現，世界因此而分成內界和外界。

作為送貨員的主角梵╱艾兒與他的老闆齊爾威，兩人為對抗非正規機械的邊境軍隊「守衛軍」運送物品時，發現了內藏過去世界戰鬥紀錄、很像洛克人零年代主要人物的神秘變身組件──生命金屬，並用此和野生機械們戰鬥。

== 遊戲系統 ==
* 結晶知識重生系統（日版）╱變換容存晶體意識系統（美版）（ {{lang|en|R.O.C.K. (Rebirth Of Crystallised Knowledge) / M.E.G.A. (Meta-Encapsulated Granule Awareness)}} ）
: 單純是使用生命金屬「變身」。名字分別是取了「洛克」日式和英式名稱的前頭部份，故此能夠使用生命金屬者均被稱為「洛克人」。

* 超限發動系統（ {{lang|en|O.I.S. (Overdrive Invoke System)}} ）
: 將四天王形態（H型、F型、L型和P型）的力量爆發的系統，到時能夠使用四天王各自的特別攻擊技，但會耗用生命金屬能量。

* 技術性擊倒系統（ {{lang|en|T.F.S. (Technical Finish System)}} ）
: 與類洛體機械人戰鬥時，若以生命金屬的所在位置作為攻擊重點，敵人的生命值扣減會較大，但隨之而然所得的生命金屬的級數便會降低。要獲得高級數生命金屬便要「技術性擊倒」對方，即是避開打中死穴。
: 不過，即使「誤傷」生命金屬，玩家仍然可能返-{回}-首領所在地再戰，或者以能量水晶為其升級。

* 生命金屬畫面系統
: 四天王形態另外擁有在副畫面使用的功能。

== 結局 ==
經過一番苦戰，梵／艾兒終於打倒了塞爾邦，並且點破了他瘋狂追求力量的本質及態度，以及殘害無辜人民的行為，就跟在外面游晃沒腦袋的非正規機械沒啥兩樣，由於激烈的戰鬥緣故魔蛇公司開始崩潰，生命金屬們只好使用力量把主人公拖出去，留下滿腹疑問跟不服氣而落敗的塞爾邦，隨著爆炸而灰飛湮滅。

被力量保護著的主人公。思考著自己會不會走上與塞爾邦一樣的墮落之道，此時一個電子精靈靠近主人公，那是齊爾威的靈魂，他反而只是輕輕數落了學弟／妹幾下而已，接著鼓勵他／她繼續擁有力量，保護一切的生物，使得更多的無辜人民不受敵人威脅而能安居樂業，齊爾威與後輩道別後，生命金屬們也鼓勵著它們的擁有者繼續為善，接著梵／艾兒回到高速公路，見到了昔日的守衛軍夥伴，投入了夥伴們的懷抱，繼續著無止盡的、守衛人民生命及財產的的旅程。
