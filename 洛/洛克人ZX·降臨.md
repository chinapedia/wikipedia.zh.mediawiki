{{vgname|洛克人ZX·降臨|ja=ロックマンゼクスアドベント|en=Mega Man ZX Advent}}是紀念[[洛克人系列|洛克人系列]]二十周年的第二部作品，也是[[洛克人ZX系列|洛克人ZX系列]][[洛克人ZX|洛克人ZX]]的續集。本作的舞台是《[[洛克人ZX|洛克人ZX]]》的數年後，科技高度發達的未來世界，人類和人類創造的機械生命體“レプリロイド”之間的漫長戰爭已經結束，開始了共存之路。

本集與上集同樣推出男女主角的選擇，但故事發展將會為分歧路線。

== 遊戲系統 ==
* 結晶知識重生系統（日版）╱變換容存晶體意識系統（美版）（ {{lang|en|R.O.C.K. (Rebirth Of Crystallized Knowledge) / M.E.G.A. (Meta-Encapsulated Granule Awareness)}} ）
: 和上集一樣有該系統，不過已更新至2.12版本，系統更多元化，不但作為使用生命金屬「變身」的系統，還新增了地圖系統，在畫面下方有區域地圖，並顯示玩家身處的位置，觸動左下角的地圖標記有該區域的簡易地圖（以小方格來砌成該區的範圍）。
* 超限發動系統（O.I.S. (Overdrive Invoke System)） 
: （跟上集一樣）四天王形態（Ｈ型、Ｆ型、Ｌ型和Ｐ型）及ＺＸ形態的力量爆發的系統，到時能夠使用各自的特別攻擊技，但會耗用生命金屬能量。
* 生命金屬畫面系統 
: （跟上集一樣）四天王形態擁有副畫面使用的功能。

== 新鮮事 ==
* 自由進行任務
: 以往ZX系列是在機器上接取任務，一旦不接任務不會產生劇情，最後又要回頭再接任務。本集會自動產生任務，並不會強迫玩家到達某區域。玩家完成任務後回到傳輸機報告任務完成後，就會得到賞金。下一任務就會產生，同時可在選單中查閱現在和過往的任務。

* 初學玩家教學
:遊戲開始前，玩家可選取容易和專家版本，選取容易後，在故事初段，會教導玩家如何移動，讓玩家容易上手。

* 互動下屏
: 上部ZX除幾種型態下屏可用作攻擊輔助外，其他型態（例人類和Ｘ型態等）時，NDS的下屏只是顯示一幅圖，沒有任何作用。今部在人類、A型和八大首領狀態，下屏會顯示區域地圖，觸動下屏左下方的地圖標記有該區域的簡易地圖，而觸動右下方畫面會轉為型態列表，觸碰任何一個會變成該形態。

* 槍類武器
: 今作的Ａ形態主要武器為槍，不像往時[[洛克人零系列|洛克人Zero]]和[[洛克人ZX|洛克人ZX]]那樣以劍作為主要武器，不過在複製敵人能力仍然可以利用劍（包括ＺＸ形態）。兩支槍的攻擊分為五種︰
:# 普通射擊︰直接按下射擊按鈕。（注︰男女主角的攻擊力和連射力有所不同　男為1傷害　女為1.5傷害，不過在首腦戰時攻擊力相同）
:# 蓄力（短）攻擊︰按著射擊按鈕短時間。
:# 蓄力（長）攻擊︰和蓄力（短）一樣，不過按著時間較長。（兩位主角的槍擊皆不同，男主角為普通炮擊，女主角則是線條型炮擊，碰到牆邊或地下會反彈，一段時間後消失。）
:# 追蹤攻擊︰此是另一支槍的攻擊，按著攻擊按鈕會放出上下約45°的電磁光線（男女主角有所差別，男主角的比較闊，女主角的追蹤性較高），碰到敵人會作出鎖定，放下攻擊按鈕會一次性放出攻擊（男為光束，女為電磁砲），會消耗少量能量。
:# 雙槍連擊（全屏攻擊）︰此乃最新型攻擊，主角會用雙槍連環射擊，必殺技，具攻擊性，不過需要用盡全部能量計。

* 八大首領獎牌
: 在打八大首領時，利用一些特別方式來打敗它，會得到金、銀或銅獎牌，每個皆是利用不同方法去得到。

* 區域轉移
: 基於今作的區域較為散亂，不是一區連接到另一區，今次在每個區域開頭都設置一部傳送記憶裝置，簡易版本是免費開啓，專家版本則要付100EC（金錢單位），困難版本需要500EC。到時在各區域盡頭或獵人營地04室都可以用傳送裝置把主角傳送到記憶裝置。

== 結局 ==
在Albert總部緊急撤退後，世界隨即發生巨大災難，Albert打算用新基地去侵略世界，葛雷／雅雪打算用獵人基地的飛船飛往Albert的基地阻止，但所有飛船已被炸毀，正當主人公為此煩惱時，前作主角梵／艾兒在這時現身，說有最後方法，當葛雷／雅雪作好準備就通知梵／艾兒。

當葛雷／雅雪準備好後，梵／艾兒就把手舉向上面，竟然是守衛軍飛船。於是去到的基地，梵／艾兒和一群人在後面看守，主人公就去找Albert，途中遇到四天王阻撓，但梵／艾兒上前解決他們，叫主人公繼續前進。找到Albert後，展出最後苦戰。經過一番苦戰，葛雷／雅雪終於打倒了Albert。

（葛雷篇）葛雷說自己雖是製造出來的，但拯救世界是自己的決定和命運，目的是和人類和平共存，說完Albert就毀滅了。之後，飛船準備爆炸了，生命金屬Ａ型提示要趕快離開，但葛雷已筋疲力盡，暈倒。艾兒方面，生命金屬Ｚ型提示她去救葛雷，並自己一個去解決四天王他們，艾兒照著做了。基地爆炸了，葛雷和艾兒安全回去了，葛雷醒後感謝生命金屬Ａ型的一切。走出室外與艾兒會面，並了解對方的未來去向，艾兒會繼續乘守衛軍飛船去遊覽世界，而葛雷會四處走尋找自己的背景，之後兩位依依不捨地道別了。

（雅雪篇）雅雪責備Albert並不是甚麼神，說出沒人可改變人類的命運和故事，包括自己，說完Albert就毀滅了。之後，飛船準備爆炸了，生命金屬Ａ型提示要趕快離開，雅雪答應了，但說出身體突然無法移動，之後暈倒。梵方面，生命金屬Ｚ型提示他去救雅雪，並自己一個去解決四天王他們，梵照著做了。基地爆炸了，雅雪和梵安全回去了，雅雪醒後感謝生命金屬Ａ型的擔心和一切。走出室外與梵會面，並了解對方的未來去向，梵會繼續乘守衛軍飛船去遊覽世界，而雅雪會做回遺跡獵人，去探索世界不同的地方，學習新鮮事物和提升知名度，讓世界人民認識她，之後兩位依依不捨地道別了。
