'''洛城四部曲'''（{{lang-en|'''L.A. Quartet'''}}）是由{{link-en|詹姆士·艾洛伊|James Ellroy}}所寫的四部[[犯罪小說|犯罪小說]]系列，背景設定在1940年代晚期至1950年代晚期期間的洛杉磯。<ref>{{Cite web |url = http://www.nytimes.com/2006/09/15/movies/15dahl.html |title = In a Noir Los Angeles, Murder Most Lurid |author = Manohla Dargis |date = 2006-09-15 |publisher = The New York Times |language = en}}</ref><ref>{{Cite web |url = http://www.winnipegfreepress.com/arts-and-life/entertainment/books/ellroys-newest-compulsively-readable-62082277.html |title = Ellroy's newest compulsively readable |author = Randall King |date = 2009-09-27 |publisher = Winnipeg Free Press |language = en}}</ref><ref>{{Cite web |url = http://www.thehindu.com/features/metroplus/article2336788.ece |title = Videowatch: This fortnight at indiaplaza.in |date = 2011-08-08 |publisher = The Hindu |language = en}}</ref>它們分別是：
* [[黑色大理花_(小說)|黑色大理花]]（''The Black Dahlia''，1987年）
* [[絕命之鄉|絕命之鄉]]（''The Big Nowhere''，1988年）
* [[鐵面特警隊_(小說)|鐵面特警隊]]（''L.A. Confidential''，1990年）
* {{link-en|白色爵士|White Jazz}}（''White Jazz''，1992年）
幾個來自洛城四部曲的角色，最知名的Dudley Smith，曾於艾洛伊的1983年小說《''{{link-en|Clandestine (novel)|Clandestine (novel)|Clandestine}}''》中出現，內容發生在1951年及1955年之間，也提及到[[黑色大理花懸案|黑色大理花懸案]]。
