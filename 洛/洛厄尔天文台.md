'''洛厄尔天文台'''是位於[[亞利桑那州|亞利桑那州]][[弗拉格斯塔夫_(亞利桑那州)|旗杆鎮]]的[[天文學|天文]][[天文台|觀測台]]。在[[美國|美國]]，洛厄尔天文台是最老的天文台，因此在1965年被指定為[[美国国家历史名胜|美国国家历史名胜]]。最初，這個天文台只有一架迄今仍開放給民眾使用24英吋[[艾爾文·克拉克|克拉克]]望遠鏡，每年大約有70,000名遊客在白天先參加天文台安排的導遊活動，晚間再透過克拉克望遠鏡以及其他的望遠鏡觀賞夜空中奇妙的美景。它是由波士頓的望族[[洛厄尔家族|洛厄尔家族]]的天文學家[[帕西瓦尔·洛厄尔|帕西瓦尔·洛厄尔]]建立的，並且有一陣子是由他的第三代堂兄[[盖伊·洛厄尔|盖伊·洛厄尔]]掌管，現在受委託的管理人是威廉·洛厄尔·普特南，是建立起長遠委託制度的[[羅傑·普特南|羅傑·普特南]]的兒子與創建者帕西瓦尔·洛厄尔的孫外甥。

天文台目前有在旗杆鎮的兩個地點設置了一些望遠鏡，主要的機構設置在旗杆鎮正西方的火星丘，最早的24英吋克拉克折射望遠鏡和建築至今仍開放做為公共教育的場所，但已經不從事研究工作了。這架望遠鏡在1896年建造時花費了20,000美金，是由在波士頓的[[克拉克父子光學望遠鏡公司|克拉克父子光學望遠鏡公司]]組裝好，再經由鐵路運送到旗杆鎮。 同樣位於火星丘上的還有發現冥王星的13英吋望遠鏡，[[克萊德·湯博|克萊德·湯博]]用這一架望遠鏡在1930年發現了[[冥王星|冥王星]]。

本天文台用在研究的四架望遠鏡都在旗桿是西南方12英里，有著漆黑夜空的安德森台地，包括72英吋的帕金斯望遠鏡（與波士頓大學共管）和42英吋的約翰S.海爾望遠鏡。本天文台和美國海軍天文台是合作夥伴，因此NRL的[[海軍原型光學干涉儀|海軍原型光學干涉儀]]（Navy Prototype Optical Interferometer, NPOI）就位於此地。天文台還有一些較小的研究用望遠鏡分別位在有歷史性的火星丘和澳洲、智利等地方。本天文台目前正在建造的4.2米[http://www.lowell.edu/DCT 發現頻道望遠鏡]是與發現傳播集團合作的。

== 探索頻道望遠鏡 ==

本天文台正在與[[探索頻道|發現傳播集團]]合作，在亞利桑納北方鄰近快樂傑克之處興建一架新的望遠鏡。這架望遠鏡座落於這架望遠鏡座落於莫可揚山脈的[[可可尼諾國家森林|可可尼諾國家森林]]保護區邊緣，估計這是在美國本土上第五大的森林，是特別允許洛威爾天台的天文學家進入的新研究區域和在現行的管理下能更有效的高效率利用。發現傳播集團和研究的項目將聚焦在情報資料和教育上，並持續的在電視節目上做天文學、科學和工藝的傳播[http://www.discoverychannel.com.tw/_home/index.shtml 發現傳播網（中文）]。另一方面，本天文台的[[探索頻道望遠鏡|探索頻道望遠鏡]]預期會對教育和經濟的狀態有著意義重大的衝擊。

== 進行中的研究 ==
本天文台的天文學家使用地基的望遠鏡、飛機和太空中的望遠鏡，在太陽系和尖端的天文物理等領域內，廣泛的主導著研究。在許多仍在進行的研究中，像[[近地小行星|近地小行星]]、[[海王星|海王星]]外的[[古柏帶|古柏帶]]巡天、[[太陽系外行星|太陽系外行星]]的搜尋（[[跨大西洋系外行星搜尋計畫|跨大西洋系外行星搜尋計畫]]）、長達十年的太陽光度穩定性研究、和在遙遠的星系內各種不同類型恆星形成和演化的研究等等。另一方面，天文台的成員為顧客設計和建造望遠鏡或是使用本天文台和世界各地的望遠鏡。例如，本天文台的員工為[[同溫層紅外線天文台|同溫層紅外線天文學望遠鏡]]（{{lang|en| Stratospheric Observatory for Infrared Astronomy，SOFIA}}）建造了尖端的高速照相機。SOFIA是美國和德國太空總署合作的計畫，將一架波音747 SP改裝，並在上面安裝了一架2.5米的望遠鏡。

== 著名的發現 ==
 
* [[克萊德·湯博|克萊德·湯博]]在1930年發現[[冥王星|冥王星]]，在2006年之前認為是太陽系的第九大行星。
* [[維斯托·斯里弗|維斯托·斯里弗]]在1912至1914年間，廣泛的研究[[星系|星系]]的[[退行速度|退行速度]]，這項研究最終導致[[宇宙膨脹|宇宙膨脹]]的認知。
* 在1977年共同發現[[天王星|天王星]]的環。
* 發現[[哈雷彗星|哈雷彗星]]光度變化的週期。
* 發現已知的最大三顆[[恆星|恆星]]。
* 發現[[冥王星|冥王星]]有[[大氣層|大氣層]]。
* 確認冥王星兩顆衛星的軌道：[[冥衛二|Nix]]和[[冥衛三|Hydra]]。
* 在[[木衛三|甘尼米德]]上發現氧氣。
* 在天王星的三顆衛星上發現冰凍的二氧化碳。
* 發現第一顆海王星特洛伊小行星。
* 證實HD 209458 b的大氣層含有水蒸氣。
