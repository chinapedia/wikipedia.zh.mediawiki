'''莫洛夫'''{{BD|1928年|5月11日|2018年|3月19日|catIdx=Luo洛}}<ref>[http://www3.nmtl.gov.tw/writer2/writer_detail.php?id=999 2007台灣作家作品目錄-洛夫]</ref>，原名莫運端，後改'''莫洛夫'''<ref>[http://www.upmedia.mg/news_info.php?SerialNo=37173 焦點 台灣文壇再失大將　「詩魔」洛夫病逝享壽89歲]</ref>，筆名'''洛夫'''、'''野叟'''，[[臺灣|臺灣]][[現代詩|現代詩]]詩人，生於中國[[湖南省|湖南省]][[衡陽|衡陽]]，退休後移民[[加拿大|加拿大]]。於2016年夏天，返台定居。曾任教[[東吳大學_(臺灣)|東吳大學]]外文系，後任[[北京師範大學|北京師範大學]]、[[華僑大學|華僑大學]]、[[廣西民族大學|廣西民族大學]]、[[中北大學|中北大學]]等大學的客座教授。 1954年與[[張默_(詩人)|張默]]、[[痖弦|痖弦]]共同創辦《創世紀》詩刊，歷任總編輯多年。作品被譯成英、法、日、韓、荷蘭、瑞典等文，並收入各種大型詩選，包括《中國當代十大詩人選集》。

== 生平 ==
洛夫，1928年生於湖南衡陽，父莫逢春，母羅賢春<ref>《向秋天最甜的那颗石榴致敬——华语诗坛泰斗洛夫先生访问记》，「訪問前記」章節，張天男，2013。</ref>。1938年，舉家搬遷至[[衡阳市|衡阳市]]，就讀「中心小学」。1943年入私立成章中学初中部<ref>[http://hunan.ifeng.com/hunanspecial/hnzhongguomeng/detail_2014_08/13/2758876_0.shtml 洛夫：乡愁是一种治不好的病_湖南频道_凤凰网]</ref>。初中三年級時，因閱讀了大量[[俄國文學|俄國文學]]作品，將自己名字由「莫運端」改為[[俄國|俄國]]風味的「莫洛夫」<ref>[https://www.hk01.com/%E6%B8%AF%E8%81%9E/169659/-%E8%A9%A9%E9%AD%94-%E8%8E%AB%E6%B4%9B%E5%A4%AB%E4%BB%8A%E6%99%A8%E9%80%9D%E4%B8%96-%E4%BA%AB%E5%B9%B491%E6%AD%B2 「詩魔」莫洛夫今晨逝世 享年91歲]</ref>。1948年考入[[湖南大學|湖南大學]]外文系，但1949年國軍在湖南招考青年入伍，洛夫隨軍隊抵達台灣。1951年考入[[政工幹校|政工幹校]]第一期，1953年畢業，分發至左營[[中華民國海軍陸戰隊|海軍陸戰隊]]，1954年與[[張默_(詩人)|張默]]、[[瘂弦|瘂弦]]合辦了[[創世紀詩社|創世紀詩社]]，後隨軍轉赴[[金門|金門]]。[[越戰|越戰]]後期奉命參加[[中華民國駐越軍事顧問團|駐越軍事顧問團]]擔任英文秘書，期間發表了作品《西貢詩鈔》。返台後，就讀於[[淡江文理學院|淡江文理學院]]英文系，1973年畢業，並在同年8月以中校軍階退役<ref>《二十世紀臺灣新詩史》，頁229，張雙英，2006。ISBN 957114391X</ref>。軍職退役後轉為教師，曾任[[東吳大學_(臺灣)|東吳大學]]外文系副教授，1996年移居[[加拿大|加拿大]][[溫哥華|溫哥華]]，2016年夏返台定居，2017年獲[[國立中興大學|國立中興大學]]頒贈名譽文學博士。2018年3月初出版最後一本詩集《昨日之蛇》。同月19日凌晨3點21分，洛夫病逝，終年89歲。<ref>{{cite news|title=台灣當代詩人洛夫今晨病逝北榮　享壽89歲|url=https://tw.appledaily.com/new/realtime/20180319/1317532|accessdate=2018-03-19}}</ref><ref>[http://www.cna.com.tw/news/firstnews/201803190099-1.aspx 洛夫病逝享壽91歲 18歲開始寫詩人稱「詩魔」],中央社，2018年3月19日</ref>

== 創作 ==
洛夫創作甚豐，出版詩集《時間之傷》等三十七部，散文集《一朵午荷》等七部，評論集《詩人之鏡》等五部，譯著《雨果傳》等八部。他的名作《石室之死亡》廣受詩壇重視，四十多年來評論不輟，英譯本已於1994年10月由美國舊金山道朗出版社出版。 1982年他的長詩《血的再版》獲中國時報文學推薦獎，同年詩集《時間之傷》獲台灣的[[中山藝文獎|中山藝文獎]]。 1986年獲[[吳三連文藝獎|吳三連文藝獎]]。 1991年復獲[[國家文藝獎|國家文藝獎]]。2003年獲[[中國文藝協會|中國文藝協會]]贈終生榮譽成就獎章。2004年獲北京「新詩界首屆國際詩歌獎」。1999年，洛夫詩集《魔歌》被評選為台灣文學經典之一。2001年三千行長詩《漂木》出版，震驚華語詩壇，同年被評選為台灣當代十大詩人之一，名列首位。

研究洛夫作品之專著頗多，已出版者有：《詩魔的蛻變：洛夫詩作評論集》、《洛夫與中國現代詩》、《洛夫評傳》、《一代詩魔洛夫》、《漂泊的奧義》、《洛夫：詩，魔，禪》、《洛夫長詩<漂木>十論》、《大河的雄辯-洛夫詩作評論集(第二部)》等。洛夫早年為超現實主義詩人，表現手法近乎魔幻，被詩壇譽為“'''詩魔'''”。 洛夫近年沈潛於書法之探索，不僅長於魏碑漢隸，尤精於行草，書風靈動蕭散，境界高遠。曾多次應邀在台北、台中、菲律賓、馬來西亞、溫哥華、紐約、北京、濟南、南寧、深圳、杭州、衡陽、石家莊、太原等地展出。

== 主要作品 ==
《靈河》、《石室之死亡》、《外外集》、《無岸之河》、《詩人之鏡》、《時間之傷》、《漂木》、《因為風的緣故》、《魔歌》、《昨日之蛇》
