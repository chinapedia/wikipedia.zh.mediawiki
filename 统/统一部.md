'''统一部'''（{{朝|'''통일부'''|統一部|}}），[[大韩民国|大韩民国]]中央行政十七部門之一。部门首长称'''统一部长'''（{{Kor|諺=통일부 장관|漢=統一部 長官}}），同时兼任国务委员，其任務在於实现[[朝鲜半岛|朝鲜半岛]]和平并构建统一基础，以此开创“朝鲜半岛统一时代”。<ref>{{cite web|url=http://cn.unikorea.go.kr/content.do?cmsid=79|title=統一部成立目的|publisher=大韓民國統一部|location=首爾特別市|language=zh|accessdate=2016-09-30|deadurl=yes|archiveurl=https://web.archive.org/web/20170305011830/http://cn.unikorea.go.kr/content.do?cmsid=79|archivedate=2017-03-05}}</ref>

== 沿革 ==
統一部根據法律第2041号的新设国土统一院等相关政府组织法中法律修订案中設立，由总统令第3574号在1969年1月29日確立編制，並於3月1日正式设立国土统一院。其後隨南北情勢發展與事實需要，而有組織上的變化。<ref>{{cite web|url=http://cn.unikorea.go.kr/content.do?cmsid=69|title=統一部沿革|publisher=大韓民國統一部|location=首爾特別市|language=zh|accessdate=2016-09-30|deadurl=yes|archiveurl=https://web.archive.org/web/20161001214308/http://cn.unikorea.go.kr/content.do?cmsid=69|archivedate=2016-10-01}}</ref>
*1990年12月27日 － 改称统一院。院长升为副总理级。
*1998年2月28日 － 副总理制废止，统一院改组为统一部。
*1999年7月8日 － [[朝鲜民主主义人民共和国|北韩]][[脱北者|离脱住民]]固定支援事务所设立。
*2003年11月20日 － 南北出入事务所设立。
*2014年8月27日 － 设立韩半岛统一未来中心。
*2015年1月7日 － 新设统一文化课、人道开发合作课。
*2016年9月23日 －新设共同体基础建设局、和平政策课、公关专员、北韩人权档案中心

== 职责 ==
*南北统一及南北对话、交流合作、人道支援等相关政策的制定。
*统一事业的教育、宣传等相关事务的管理。
