'''襤褸'''（{{lang-ja|'''ぼろ'''}}，今通稱「'''ボロ'''」、「'''BORO'''」）是[[日本|日本]]傳統[[拼布|拼布]]文化和工藝<ref name=nhk></ref><ref name=am></ref><ref name=b></ref>，是{{link-ko|百家布|조각보}}的一種，源於[[江戶時代|江戶時代]]的日本[[青森縣|青森縣]]（當時為[[津輕藩|津輕藩]]與[[盛岡藩|南部藩]]）的村莊<ref name=nhk></ref><ref name=am></ref><ref name=b></ref>。這種拼布工藝製作的服飾也稱為襤褸，為[[百德着物|百德着物]]的一種，與其他百德着物不同之處在於它使用的是粗布。BORO與青森另一種縫紉工藝{{tsl|ja|裂織|裂織}}頗有淵源<ref name=nhk></ref>。

== 簡介 ==

當時布匹稀少，村民們將常用的[[麻布|麻布]]或其他粗舊布碎片拼湊成[[織物|織物]]，當做寢具（如被褥、床墊）或服裝，世代相傳使用、穿著，並繼續將布料一層層織染繡補。<ref name=am>{{cite web|title=BORO EXHIBITION|url=http://www.amusemuseum.com/boro/|publisher=AMUSE MUSEUM（[[東京|東京]]布文化と浮世絵の美術館）|accessdate=2013-12-30|language=ja}}</ref> <ref name=b>{{cite book|author=小出由紀子、都築響一|title=《BORO つぎ、はぎ、いかす。青森のぼろ布文化》|publisher=アスペクト|language=en|date=2008年12月}}</ref>

青森氣候寒冷环境嚴酷，而且直到[[明治时代|明治时代]]才开始栽種[[棉花|棉花]]，因此在當時爲了禦寒，村民們使用厚麻布拼湊成BORO。經年累月，其呈現出或深或淺的靛藍色<ref name=nhk>{{cite web|title=file159　「青森のBORO（ぼろ）」｜NHK 鑑賞マニュアル 美の壺|url=http://www.nhk.or.jp/tsubo/program/file159.html|publisher=[[NHK|NHK]]|language=ja}}</ref>。

青森女性將麻布和貴重的棉線編制在一起編織成有著漂亮花紋的服裝，這種工藝被稱為[[刺子|刺子]]（{{lang|ja|刺し子}}，Sashiko）<ref name=nhk></ref><ref name=am></ref>。青森的女孩子從五六歲就開始學習，十五六歲時可以獨當一面，會帶著兩三件完成的服裝出嫁<ref name=nhk></ref>。

== 研究 ==

日本民俗學家{{tsl|ja|田中忠三郎|田中忠三郎}}關注到了其收藏的裂織、BORO布的藝術性和生態環保概念，發表了寫真集《BORO》（都築響一撮影，2009年1月出版），並於東京[[上野之森美術館|上野之森美術館]]、[[京都|京都]]{{tsl|ja|思文閣美術館|思文閣美術館}}、[[旭川市|旭川]]{{tsl|ja|国際染織美術館|国際染織美術館}}、東京{{tsl|ja|青山ブックセンター|青山圖書中心}}舉辦了「田中忠三郎珍藏 『BORO』」（{{lang|ja|田中忠三郎コレクション 『BORO』}}）展覽。

== 影響 ==

BORO的縫紉工藝、藝術設計得到了世界現代[[時裝|時尚]]界的關注<ref name=am></ref>，[[匡威|匡威]]等品牌推出了各種以BORO為靈感並命名的設計和產品。<ref>{{cite web|title=CONVERSE FIRST STRING JACK PURCELL BORO|url=http://www.milk.com.hk/content/converse-first-string-jack-purcell-boro|publisher=MILK.COM|language=zh-hant|deadurl=yes|archiveurl=https://web.archive.org/web/20140101092755/http://www.milk.com.hk/content/converse-first-string-jack-purcell-boro|archivedate=2014年1月1日|df=}}</ref> <ref>{{cite web|title=体现日本工艺之独到，KAPITAL Kountry 牛仔裤、古布披肩、拼接蓝染卫衣|url=http://www.efu.com.cn/manager/newsview-203214-1.html|publisher=中國服裝網|language=zh-hans}}</ref>
