'''影像掃描機'''-{zh-tw:，常簡稱'''掃描機'''，;zh-hk:，;zh-cn:，;}-是一個能夠把[[相片|相片]]、[[印刷|印刷]]-{[[文件|文件]]}-或手寫-{文件}-等影像，或裝飾品等小物件掃描、分析並化成[[數位|數位]]影像的器材。

現在大部份影像掃描機都是桌面掃描機的變種。以手動作掃描的手提掃描器曾一度流行，但由於很難獲得理想影像質素，現已被淘汰。這些掃描器都使用[[電荷耦合器|電荷耦合器]]（{{lang|en|Charge-coupled device，CCD}}）作為影像感應器，而鼓式掃描器則採用[[光電倍增管|光電倍增管]]（{{lang|en|Photomultiplier Tube}}）作為影像感應器。影像掃描機在我們日常生活中十分重要，隨處都有它的存在。

其他種類的掃描器包括planetary掃描器，可為書籍及文件拍攝影像，及激光距離掃描器（{{lang|en|Laser Range scanner}}），可製作立體[[三維空間|三維空間]]物件的模型。

現今普遍使用的[[多功能事務機|多功能事務機]]（MFP）即整合了影像掃描器、[[印表機|印表機]]、[[影印機|影印機]]及[[傳真機|傳真機]]。

== 桌面掃描機 ==
桌面掃描機通常裝設有一塊[[玻璃|玻璃]]面，在玻璃下面設有光源，通常是[[氙|氙]]或冷光[[光管|光管]]照耀著玻璃面，另外有一個可單方向移動的CCD 感光單元。彩色掃描器通常有3排CCD 感光元件，分別有紅，綠及藍色[[濾鏡|濾鏡]]。使用者把需要掃描的影像朝下放在玻璃上，光管會亮著，感光單元及光源則會一起由[[馬達|馬達]]移動，把整個需要掃描面積掃描一次。因為光源的照射，影像便會被CCD 感光元件感應到。透明物料的原稿，例如[[菲林|菲林]]底片則不能採用此方法掃描，需要另加特別的附件從上面照亮物件才可被掃描。

一些型號的掃描機設有自動文件掃描功能(automatic document feed, ADF)，讓使用者可以把一疊文件放進紙匣，每頁紙將被自動吸進掃描器內並進行掃描。在掃描時CCD 感光元件不會被移動，而源文件則會由捲筒以固定的速度在掃描器內移動，在文件出口另設有紙匣收集已掃描的文件。

== 掃描器質素 ==
[[File:Personal_computer,_exploded_6.svg|thumb]]
掃描器通常可透過感光[[CCD|CCD]] 閱讀紅綠藍[[3原色|3原色]](RGB) 資料，以一些專利[[演算法|演算法]]對不同[[曝光|曝光]]情況進行處理，並透過掃描器的輸入/輸出界面把影像數據傳送到電腦，傳輸-{界面}-包括[[SCSI|SCSI]]、[[USB|USB]]或較早期的[[並列埠|並列埠]]。[[色深|色深]]將由CCD的特性決定，通常最少為24 bit。高級型號則有48 bit甚至更高。

另一個影響影像質素的指標就是影像的[[解像度|解像度]]，以每吋的點數(dpi)量度，或可以更準確地以samples per inch (spi)量度。掃描器廠商通常喜歡以插值解像度，而非真實的光學解像度宣傳其產品，但插值解像度會因為軟件[[插值法|插值法]]而大大提高，造成夸大解像度的假象。在2004年，一部家用級數的影像掃描器可達到1600–3200 dpi的光學解像度，高檔桌面掃描器最高可以5400 dpi掃描，鼓式掃描器則更可達到8000–14,000 dpi的光學解像度。掃描器生產商通常聲稱插值解像度可達到19,200 dpi，但這些數字並無實質意義，而理論上可能的插值像素是[[無限大|無限大]]。

== 輸出資料 ==
掃描機在掃描後最終得到的是未經壓縮的RGB三色影像數據，通常會傳送至電腦的[[記憶體|記憶體]]。影像可以使用[[位图|位图]]編輯程式，例如[[Adobe_Photoshop|Adobe Photoshop]]或[[GIMP|GIMP]]等進行處理及編輯，並儲存在[[硬碟|硬碟]]等儲存設備。

== 與電腦連接 ==
掃描機製造出來的影像數據資料可以十分大：一張600[[dpi|dpi]]，9吋x11吋，稍大於[[A4紙|A4紙]]面積的未經壓縮24bit彩色影像，將相等於100 MByte的未壓縮數據。較新型號的掃描機可在幾秒內創造出這麼大量的數據，因此快速的連接方式十分重要。

早期掃描器使用[[并口|并口]]連接電腦，速度緩慢，最高傳送速度只有每秒70 [[KB|KB]]。專業型號採用[[SCSI|SCSI]]-II界面，優點是傳送速度快，達每秒數[[MB|MB]]，但缺點是價格高昂，亦需要在電腦內加裝一張[[SCSI|SCSI]]擴充卡，對一般用家構成不便。

[[IEEE1394|IEEE1394]]（火線，Firewire）已開始取代[[SCSI|SCSI]]作為專業製作級，大量文件掃描器的傳送標準。

近年較經濟的掃描機型號大部份採用[[USB|USB]]作為連接界面，在初期的[[USB|USB]] 1.1，傳送速率有每秒1.5 [[MB|MB]]，較新配備USB 2.0 的型號，傳送速率可達每秒60 [[MB|MB]]，消除了傳輸的樽頸。

市場上兩種主要的掃描機傳送界面包括：
* [[TWAIN|TWAIN]]主要用於低價位或家用器材。
* [[ISIS|ISIS]]由Pixel Translations創立，仍然使用[[SCSI|SCSI]]-II以提供高速度，被大型专业機器使用。

== 紅外線清理 ==
紅外線清理(Infrared cleaning)是一種為[[菲林|菲林]]底片清除[[塵埃|塵埃]]及花痕的技術。大部分新型號的掃描器已設有此功能。[[紅外線|紅外線]]清理會以紅外線光束掃描菲林，可以掃描出阻隔紅外線通過的塵埃及花痕，並以其位置、大小、形狀等資料自動去除塵埃及花痕。

掃描機生產商通常會各以自己的命名形容此技術。例如[[Epson|Epson]]、[[尼康|尼康]]、[[Microtek|Microtek]]等廠商便使用[[柯達|柯達]]開發的Digital ICE技術，而[[佳能|佳能]]就使用自己開發的FARE (Film Automatic Retouching and Enhancement)系統。
