'''崔珙'''（{{bd|？||854年|catIdx=C}}），'''安平郡公'''，[[唐朝|唐朝]]官员，[[唐武宗|唐武宗]]年间为[[宰相|宰相]]。

== 家世 ==
崔珙出自[[博陵崔氏|博陵崔氏]]最显赫的博陵第二房。祖父[[崔意|崔意]]是博州聊城县县令，父亲[[崔頲|崔頲]]官至[[同州|同州]][[刺史|刺史]]。<ref>{{cite web |url=http://www.sidneyluo.net/a/a17/072.htm |title=存档副本 |accessdate=2008-10-03 |0=2008-11-20 |deadurl=yes |archiveurl=https://web.archive.org/web/20081120085821/http://www.sidneyluo.net/a/a17/072.htm |archivedate=2008-11-20 }}[http://www.sidneyluo.net/a/a17/table/form63.htm 《新唐书》卷七十二]  {{webarchive|url=https://web.archive.org/web/20100620180624/http://sidneyluo.net/a/a17/table/form63.htm |date=2010-06-20 }}</ref>崔頲有八子，皆至达官，时人比于[[汉朝|汉朝]][[荀氏八龍|荀氏，也号为“八龙”]]。<ref name=BT177>[[:s:旧唐书/卷177|《旧唐书》卷一百七十七]]</ref><ref name=NBT182>[[:s:新唐书/卷182|《新唐书》卷一百八十二]]</ref>

崔珙与同样仕途显赫的哥哥[[崔琯|崔琯]]同母。崔珙在书判拔萃科中取得高等，在地方[[节度使|节度使]][[幕府|幕府]]为僚佐。性格威严持重，尤精于吏术。<ref name=BT177/><ref name=NBT182/>

== 唐文宗年间 ==
[[唐文宗|唐文宗]][[太和_(唐文宗)|太和]]（827年-835年）初年，崔珙为[[泗州|泗州]]刺史，[[陆畅|陆畅]]有诗《夜到泗州酬崔使君》。后入[[长安|长安]]为[[太府卿|太府卿]]。太和七年（833年）正月，拜[[广州|广州]]刺史、[[岭南节度使|岭南节度使]]。赴任前，他去[[延英殿|延英殿]]面圣谢恩，文宗问他治理南海的计划，他回答得明确而具体，文宗很欣赏。当时，[[武宁军节度使|武宁军节度使]][[高瑀|高瑀]]部卒骄横悖乱。高瑀不能制，时议认为徐州应由有威望的节度使管领，文宗也这么想，对宰相说：“崔珙言事，神气精爽，此可以临徐人。”七月，调崔珙为兼[[检校|检校]][[工部尚书|工部尚书]]、[[徐州|徐州]]刺史兼[[御史大夫|御史大夫]]，充武宁军节度、徐泗濠观察使。<ref>[[:s:旧唐书/卷162|《旧唐书》卷一百六十二]]</ref>派[[王茂元|王茂元]]去岭南代崔珙。<ref name=BT17-2>[[:s:旧唐书/卷17下|《旧唐书》卷一十七下]]</ref><ref>《册府元龟》误作崔珙为“新授镇南节度使”。</ref>崔珙到任后，在任两年，<ref>开成元年（936年）十二月，[[薛元赏|薛元赏]]被任为武宁军节度使，可见当时崔珙已离任。</ref>宽猛适宜，徐州人得以安宁。<ref name=BT177/><ref name=NBT182/><ref>[[:s:資治通鑑/卷244|《资治通鉴》卷二百四十四]]</ref>

[[开成|开成]]初年，加检校[[兵部尚书|兵部尚书]]。二年（837年），改检校[[吏部尚书|吏部尚书]]、右金吾大将军，充街使。六月，迁[[京兆尹|京兆尹]]。<ref name=BT17-2/>当时其兄崔琯官居尚书左丞，兄弟俩都位居显官。当年，京畿闹旱灾。次年（838年）正月，崔珙奏请差少尹去终南广惠公庙祈祷降雪，<ref>[[:s:祷雪终南广惠庙奏|:s:祷雪终南广惠庙奏]]</ref>及将引入宫中的[[浐水|浐水]]减去十分之九，改给贫民灌溉田地，文宗同意。当月，发生了一般认为是当权宦官[[仇士良|仇士良]]主使的对宰相[[李石_(唐朝)|李石]]的刺杀，<ref name=ZZTJ246>[[:s:資治通鑑/卷246|《资治通鉴》卷二百四十六]]</ref>崔珙因不能抓获凶手，被罚俸，<ref name=BT177/>声望也略有衰落。<ref name=NBT182/>四年（839年）三月之前卸任京兆尹。<ref>当时京兆尹为[[郑复|郑复]]。</ref>

== 唐武宗年间 ==
唐文宗于开成五年（840年）正月驾崩，弟唐武宗继位。崔珙和实质上为首相的李德裕相善，累迁[[户部侍郎|户部侍郎]]，进[[刑部尚书|刑部尚书]]、充诸道盐铁转运等使。<ref name=BT177/>进阶[[银青光禄大夫|银青光禄大夫]]。五月，崔珙被授[[同平章事|同平章事]]，为[[实质宰相|实质宰相]]，<ref name=NBT8>[[:s:新唐书/卷8|《新唐书》卷八]]</ref><ref>《旧唐书·武宗纪》《册府元龟》作二月误作以[[户部尚书|户部尚书]]判度支守本官拜相。</ref>兼盐铁转运使，<ref>[[:s:旧唐书/卷49|《旧唐书》卷四十九]]</ref><ref>[[:s:授崔珙同平章事制|:s:授崔珙同平章事制]]</ref>兼[[门下侍郎|门下侍郎]]（《新唐书》作拜[[中书侍郎|中书侍郎]]），后又兼尚书[[左仆射|左仆射]]。<ref name=NBT182/>增江淮茶税。<ref>[[:s:新唐书/卷54|《新唐书》卷五十四]]</ref>[[会昌_(唐朝)|会昌]]元年（841年）三月，武宗认为前宰相[[杨嗣复|杨嗣复]]和[[李珏|李珏]]反对他继位而分别意图拥立当时已被[[赐死|赐死]]的安王[[李溶|李溶]]和陈王[[李成美|李成美]]，想赐死杨、李，崔珙和同僚宰相[[李德裕|李德裕]]、[[崔郸|崔郸]]、[[陈夷行|陈夷行]]介入，请求开延英殿论事，李德裕、崔珙等并指出杨嗣复、李珏意欲另立他人事真假难辨，武宗虽坚称杨、李确有拥立他人之事，但也同意宽恕，于是杨、李虽然被贬，却得以免死。<ref name=ZZTJ246/><ref>[[:s:旧唐书/卷176|《旧唐书》卷一百七十六]]</ref><ref>[[:s:新唐书/卷174|《新唐书》卷一百七十四]]</ref>当月，李德裕、陈夷行、崔珙、[[李绅|李绅]]等奏[[唐宪宗|唐宪宗]]有恢复中兴之功，请为百代不迁之庙，武宗也认为此论至当并继续议论此事，但最终没有实行。<ref name=BT18-1>[[:s:旧唐书/卷18上|《旧唐书》卷一十八上]]</ref>

会昌二年（842年）正月，祀九宫贵神，崔珙奉敕摄[[太尉|太尉]]行礼。<ref>[[:s:旧唐书/卷24|《旧唐书》卷二十四]]</ref><ref>[[:s:请定祀九宫仪注奏（会昌二年正月太常礼院）|:s:请定祀九宫仪注奏（会昌二年正月太常礼院）]]</ref>当月与陈夷行奏定左右仆射上事仪注。<ref>[[:s:新唐书/卷181|《新唐书》卷一百八十一]]</ref>二月，崔珙奏荐右散骑常侍[[柳公权|柳公权]]为集贤院学士、判院事。李德裕不满此事恩非己出，找事由贬柳公权为太子詹事。<ref name=ZZTJ246/><ref>[[:s:旧唐书/卷165|《旧唐书》卷一百六十五]]</ref><ref>[[:s:新唐书/卷163|《新唐书》卷一百六十三]]</ref>四月，李德裕、时为银青光禄大夫、守右仆射、门下侍郎、平章事的崔珙、宰相李绅、[[金紫光禄大夫|金紫光禄大夫]]检校[[司徒|司徒]]兼[[太子太保|太子太保]][[牛僧孺|牛僧孺]]等上章，请加武宗尊号仁圣文武至神大孝皇帝。崔珙以右仆射兼中书侍郎为摄[[中书令|中书令]]读册官，获赐一子出身。<ref>[[:s:加尊号赦文|:s:加尊号赦文]]</ref>盐铁判官姚勖曾平反冤案，崔珙奏请嘉奖，得任为权知职方员外郎，因尚书右丞[[韋溫_(宣歙觀察使)|韦温]]不奉诏，改检校礼部郎中。<ref>[[:s:旧唐书/卷168|《旧唐书》卷一百六十八]]</ref>崔珙又和官员[[崔铉|崔铉]]不和，会昌三年（843年）二月，崔珙因兄[[山南西道节度使|山南西道节度使]]崔琯去世且本人患病而请辞，罢相为[[右仆射|右仆射]]，<ref name=NBT8/><ref>《旧唐书》本传作崔琯因崔珙罢相受累被罢免节度使，归东都，卒于五年（845年），疑误。</ref>崔铉继为宰相兼盐铁转运使，弹劾他乱用[[宋州|宋州]]、[[滑州|滑州]]院盐铁钱九十万贯及曾保护军阀[[刘从谏|刘从谏]]。四年（844年）六月，崔珙因而被贬为[[澧州|澧州]]刺史，当月又以银青光禄大夫、澧州刺史、[[上柱国|上柱国]]、安平郡开国公、食邑二千户的身份被再贬[[恩州_(广东)|恩州]]司马员外置。<ref name=BT177/><ref name=NBT182/><ref name=BT18-1/>

== 唐宣宗年间 ==
会昌六年（846年）三月，唐武宗崩，叔[[唐宣宗|唐宣宗]]继位，赦免武宗时期被贬的崔珙和其他四位前宰相[[李宗闵|李宗闵]]、[[牛僧孺|牛僧孺]]、杨嗣复、李珏，调近京城，八月，崔珙被任为[[安州_(西魏)|安州]]长史。[[杜牧|杜牧]]有《上安州崔相公启》，提及自己曾于三年八月给崔珙写过信。约[[大中_(唐朝)|大中]]元年（847年）徙[[商州|商州]]刺史，又为[[太子宾客|太子宾客]]，<ref>但当时并无[[皇太子|皇太子]]，这完全只是荣衔。</ref>当年又被起用为[[凤翔节度使|凤翔节度使]]。<ref name=BT177/><ref name=NBT182/>当年在[[桂管观察使|桂管观察使]][[郑亚|郑亚]]幕府为书记的[[李商隐|李商隐]]作《为荥阳公上凤翔崔相公贺正启》。大中二年（848年），十一月，奏周公祠有泉水涌出，<ref>[[:s:周公祠灵泉奏状|:s:周公祠灵泉奏状]]</ref><ref>《周公祠灵泉记》</ref>宣宗手诏赐名润德泉。<ref>[[:s:答崔珙奏敕|:s:答崔珙奏敕]]</ref>崔珙上表谢手诏。<ref>[[:s:谢赐手诏表|:s:谢赐手诏表]]</ref>十二月，立灵泉碑。<ref>《周公祠灵泉碑并题奏状及敕批答》</ref>当月，唐军在收复[[安史之乱|安史之乱]]后被[[吐蕃|吐蕃]]占领的领地时，崔珙战胜吐蕃军，收复[[清水縣_(天水市)|清水]]。<ref name=ZZTJ248>[[:s:資治通鑑/卷248|《资治通鉴》卷二百四十八]]</ref>

大中三年（849年），崔铉再任宰相。崔珙闻讯，称病请辞，被任为[[太子少师|太子少师]]，分司东都，<ref>[[:s:贬崔珙太子少师分司东都制|:s:贬崔珙太子少师分司东都制]]</ref>五年（851年）正月前就地拜留守，<ref>《[[张季戎|张季戎]]墓志》</ref>判尚书省事。<ref>《芝田录》</ref>六年（852年）又以守[[太子少保|太子少保]]为凤翔节度使，<ref>[[:s:授崔珙凤翔节度使制|沈珣《授崔珙凤翔节度使制》]]</ref>八年（854年）前，卒于任上。<ref name=NBT182/><ref>其去世当在大中八年[[裴识|裴识]]被任为凤翔节度使前。</ref><ref>《旧唐书·杨收传》载崔珙罢相为[[淮南节度使|淮南节度使]]时以太常博士[[杨收|杨收]]为观察支使。崔珙未曾为淮南节度使，误，应为崔铉。可参考《新唐书·杨收传》。</ref>他的侄孙[[崔远|崔远]]后在唐朝末年[[唐昭宗|唐昭宗]]、[[唐哀帝|唐哀帝]]年间为宰相。<ref name=BT177/>
