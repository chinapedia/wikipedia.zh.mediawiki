'''崔元綜'''（{{bd|7世纪||8世纪|catIdx=C}}），[[唐朝|唐朝]]和[[武周|武周]]官员，在[[武则天|武则天]]年间任[[宰相|宰相]]。

== 家世 ==
崔元綜出自于[[清河崔氏|清河崔氏]]-{定著}-六房之一的郑州崔氏，曾祖父[[崔君肅|崔君肅]]在[[唐高祖|唐高祖]]年间为[[黄门侍郎|黄门侍郎]]、鸿胪卿、<ref name=BT90>[[:s:旧唐书/卷90|《旧唐书》卷九十]]</ref>秦王府长史、使持节[[襄州_(西魏)|襄州]]诸军事、襄州刺史。<ref>两唐书都称崔君肅是崔元综的祖父，见《[[旧唐书|旧唐书]]》卷九十和《[[新唐书|新唐书]]》卷一百一十四。但《新唐书·宰相世系表》又称崔元综的祖父为崔君肅的弟弟[[崔君宙|崔君宙]]。崔君宙和崔元综的父亲[[崔千里|崔千里]]都不仕官且没有爵位，见《新唐书》卷七十二下。当以《崔杰墓志》为是，崔君肃当为崔元综曾祖父。</ref>祖父崔思约，[[和州_(北齐)|和州]]刺史。父崔哲，任太原祁县令，赠国子司业，武周年间为中大夫行[[并州_(太原)|并州]][[盂县|盂县]]令。<ref>见《全唐文补遗》第一缉的《唐故信王府士曹崔君（杰）墓志铭并序》。崔哲与崔千里或为一人，史料有阙。</ref>

== 武则天年间 ==
武则天[[天授_(武周)|天授]]年间（690年 - 692年），崔元综累转[[秋官侍郎|秋官侍郎]]。[[如意|如意]]元年（692年）八月，武则天任他为[[鸞臺侍郎|鸞臺侍郎]]，授[[同鳳閣鸞臺平章事|同鳳閣鸞臺平章事]]，为[[实质宰相|实质宰相]]。<ref>[[:s:旧唐书/卷6|《旧唐书》卷六]]，作'''崔元琮'''</ref>崔元综为政勤勉，经常工作到很迟，几乎不休息。他行事谨慎，二十余年不享用美食大餐。但他外表宽容，内心严酷，对人们吹毛求疵课以重罚，人们因此畏惧且鄙视他。次年（693年）八月，他被弹劾有罪，流放[[振州|振州]]，<ref>[[:s:新唐书/卷4|《新唐书》卷四]]</ref><ref>[[:s:资治通鉴/卷205|《资治通鉴》卷二百零五]]</ref>朝野都为此欢欣鼓舞。不久他被赦免，被召回任[[監察御史|監察御史]]。<ref name=BT90/>

== 唐中宗年间 ==
[[神龙_(年号)|神龙]]元年（705年），武则天被政变推翻，她曾登基为帝的儿子[[李显|李显]]复辟，即[[唐中宗|唐中宗]]。唐中宗年间，崔元综任[[尚書左丞|尚書左丞]]，[[唐睿宗|唐睿宗]][[景云|景云]]二年（711年）六月，任[[蒲州_(北周)|蒲州]]都督，<ref>《[[唐会要|唐会要]]》卷六十八，作'''崔元琮'''</ref>不久因老病而[[致仕|致仕]]。<ref>当年十一月蒲州都督府废。次年二月蒲州刺史为[[萧至忠|萧至忠]]，崔元综告老当在此前。</ref>退休后，他学习[[道家|道家]]的[[炼金术|炼金术]]，以求延年益寿。他活到九十余岁，<ref name=BT90/>但卒年不详。

== 轶闻 ==
《[[定命录|定命录]]》有两则崔元综的故事：

1.崔元综曾任益州参军，想结婚，已经定了吉日，忽然梦见有人对他说：“这家的女子不是你的妻子，你的妻子今天才出生。”他便在梦中随此人来到洛阳履信坊十字街西道北的一户人家，进到室内的东屋旁，看到一个妇女正好生了一个女儿，那人说：“这才是你的妻子。”崔元综惊醒，不相信。这时传来消息，他正要娶的女子暴亡。崔元综直到五十八岁官居四品才娶了侍郎[[韦陟|韦陟]]十九岁的堂妹。<ref>如此轶闻属实，因韦陟生于697年，可见当时不早于715年，即崔元综出生不早于658年。但715年崔元综已辞去蒲州都督，此后没有继续仕官的记载，待考。</ref>韦氏虽然觉得崔元综年纪大，但还是嫁给了他，就在履信坊韦家宅院成亲，新娘子原来果然是住在东屋的。推算起来，她出生的年月，正是崔元综做梦的那一天，崔元综后来官至三品官，活到九十岁。韦夫人与他白头偕老，共同生活四十年，享尽了荣华富贵。

2.武则天年间，崔元综官至宰相，令史奚三儿说：“您六十天以内要被流放到南海；六年之中有三次该死，但最终不会死。从这以后，你将更换官职，最后还会官复原职，寿数是一百岁，最终要饿死。”经过六十天，果然获罪被流放到南海以南，几年后得了血痢病，病了百日，病情极为危重却没有死。遇赦回京，乘船渡海遭遇大风浪，船被淹没，一同乘船的人都死了，只有崔元综一个人抱住一块木板，随波漂荡，忽上忽下，漂泊到一个小岛上，进入芦苇丛。他抱的木板上有一个大钉子，正好从脊背扎进身体数寸，木板压在钉子上，他只能在泥水中昼夜忍痛呻吟罢了。这时忽然有一船人来到这个岛上，听到呻吟声，出于可怜把他救起来扶着上了船，并给他止血拔钉，很长时间才苏醒过来。盘问他的姓名，他说是原来的宰相，众人更可怜他并给他粮食，他一路讨饭吃。有一天他正在船上躺着，看见一个穿青绿色衣服的官员，他认出那官员是他当宰相时的令史，便招呼官员和自己说话，那官员又给他一些粮食，他才得以回到京城。六年以后，收录司回来，选曹司把原宰相情况上奏，武则天下令破格给崔元综官职。等到进宫拜谢那天，他被带到殿堂上问话，因崔元综穿着青绿色的衣服，则天见到后认出来曾见过，问他得到什么官职，他就把实情说了。武则天下诏吏部，令任命崔元综为赤尉。又等到进宫拜谢那天，武则天又特敕任他为御史。以后，他从御史做到郎官，多次升迁直到当了[[中书侍郎|中书侍郎]]。他九十九岁时，子侄都死了，<ref>崔元综子崔志廉卒于[[唐玄宗|唐玄宗]][[天宝|天宝]]八载（749年），如此轶闻属实，则崔元综去世不早于749年，出生不早于651年。</ref>只有独身一人病卧在床上，唤奴婢拿饭粥，奴婢们欺他年老病重，都笑而不动。崔元综已没有能力责罚他们了，他们也都受不到处分。崔元综感叹气愤之下不吃东西，几天后死了。

== 子孙 ==
*子：[[崔志廉|崔志廉]]，[[银青光禄大夫|银青光禄大夫]]、[[太子左庶子|太子左庶子]]，历任洺、魏、襄、泽、仙等五州刺史。
**孙：崔杰，14岁以[[五经|五经]]擢第，20岁补授太子校书。历任棣王府法曹、信王府士曹。天宝十一载（752年）终于东都洛阳，年五十一。
