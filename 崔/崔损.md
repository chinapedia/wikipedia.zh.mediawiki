'''崔损'''（{{bd|8世纪||803年|11月27日|catIdx=C}}），字'''至无'''，[[唐朝|唐朝]]官员，[[唐德宗|唐德宗]]年间为[[宰相|宰相]]。

== 家世 ==
崔损出自[[博陵崔氏|博陵崔氏]]-{定著}-四房之一的博陵大房，崔损的高祖[[崔行功|崔行功]]为[[秘书监|秘书监]]，但曾祖[[崔晷|崔晷]]以下开始衰微，崔损的祖父和父亲甚至没有在《新唐书》的宰相世系表上留下名字。<ref>{{cite web |url=http://www.sidneyluo.net/a/a17/072.htm |title=存档副本 |accessdate=2008-10-03 |0=2008-11-20 |deadurl=yes |archiveurl=https://web.archive.org/web/20081120085821/http://www.sidneyluo.net/a/a17/072.htm |archivedate=2008-11-20 }}[http://www.sidneyluo.net/a/a17/table/form63.htm 《新唐书》卷七十二]  {{webarchive|url=https://web.archive.org/web/20100620180624/http://sidneyluo.net/a/a17/table/form63.htm |date=2010-06-20 }}</ref>

== 早期仕途 ==
崔损曾与[[柳镇|柳镇]]交游。<ref name=LZY>[[:s:先君石表阴先友记|柳宗元《先君石表阴先友记》]]</ref>[[唐代宗|唐代宗]][[大历|大历]]（766年-779年）末年，崔损中[[进士|进士]]，登博学宏词科，授[[秘书省|秘书省]]校书郎，后又授[[咸阳|咸阳]]尉。但后来他的舅舅[[王翃|王翃]]做了[[京兆尹|京兆尹]]，不宜作为他的上司，他因而被改任为大理评事，<ref>王翃于建中三年（782年）成为京兆尹，四年（783年）已经离任，故崔损调任应在此年间。见《旧唐书》[http://www.sidneyluo.net/a/a16/012.htm 卷一十二]  {{webarchive|url=https://web.archive.org/web/20081023231250/http://www.sidneyluo.net/a/a16/012.htm |date=2008-10-23 }}、[http://www.sidneyluo.net/a/a16/157.htm 卷一百五十七] {{webarchive|url=https://web.archive.org/web/20071210091002/http://www.sidneyluo.net/a/a16/157.htm |date=2007-12-10 }}。</ref>累迁兵部郎中，[[贞元_(唐朝)|贞元]]十一年（795年），迁右谏议大夫。<ref name=BT136>[http://www.sidneyluo.net/a/a16/136.htm 《旧唐书》卷一百三十六] {{webarchive|url=https://web.archive.org/web/20080621164458/http://www.sidneyluo.net/a/a16/136.htm |date=2008-06-21 }}</ref><ref name=NBT167>[http://www.sidneyluo.net/a/a17/167.htm 《新唐书》卷一百六十七] {{webarchive|url=https://web.archive.org/web/20090202173116/http://sidneyluo.net/a/a17/167.htm |date=2009-02-02 }}</ref>

== 拜相 ==
十二年（796年），宰相[[门下侍郎|门下侍郎]][[同平章事|同平章事]][[赵憬|赵憬]]新亡，崔损在朝议郎、右谏议大夫任上为之作《[[:s:祭成纪公文|祭成纪公文]]》。同时，[[中书侍郎|中书侍郎]]同平章事[[卢迈|卢迈]]风疾严重，不能参政，十月，德宗授崔损和[[给事中|给事中]][[赵宗儒|赵宗儒]]同中书门下平章事，<ref name=NBT7>[[:s:新唐书/卷7|《新唐书》卷七]]</ref>赐紫金鱼袋。<ref>[[:s:旧唐书/卷167|《旧唐书》卷一百六十七]]</ref>崔损拜相是因同年新亡的德宗近臣户部尚书[[裴延龄|裴延龄]]所荐。<ref name=BT13>[[:s:旧唐书/卷13|《旧唐书》卷一十三]]</ref><ref>[[:s:資治通鑑/卷235|《资治通鉴》卷二百三十五]]：冬，十月，甲戌，以諫議大夫崔損、給事中趙宗儒並同平章事。損，[[崔玄暐|玄暐]]之弟孫也，嘗為裴延齡所薦，故用之。</ref>因两位宰相或故或病，[[中书省|中书省]]长达十天无人主事，人们希望有德者为相，而崔损并无佳声，人们失望。崔损为相后，行为龌龊，谨慎以求不冒犯皇帝，当德宗会见宰相时，他几乎不说话。<ref name=BT136/><ref name=NBT167/>

十四年（798年）四月，[[唐太宗|唐太宗]][[唐昭陵|昭陵]]旧宫寝殿被野火烧毁，德宗任时任左谏议大夫的崔损为八陵修奉使，重建昭陵及[[唐高祖|唐高祖]][[唐献陵|献陵]]、[[唐高宗|唐高宗]][[乾陵|乾陵]]、[[唐中宗|唐中宗]][[唐定陵|定陵]]、[[唐玄宗|唐玄宗]][[唐泰陵|泰陵]]、[[唐睿宗|唐睿宗]][[桥陵|桥陵]]、唐代宗[[唐元陵|元陵]]的旧宫房屋，并修葺[[唐肃宗|唐肃宗]][[唐建陵|建陵]]。七月，崔损转任[[门下侍郎|门下侍郎]]，仍居相位。<ref name=BT136/><ref name=NBT167/>同月，修复八陵寝宫完毕，群臣于宣政殿称贺。<ref name=BT13/>

十六年（800年），德宗命崔损承旨令宰相[[中书侍郎|中书侍郎]][[齐抗|齐抗]]兼修国史，齐抗让[[中书舍人|中书舍人]][[权德舆|权德舆]]上表辞让。<ref>[[:s:为齐相公让修国史表|:s:为齐相公让修国史表]]</ref>

崔损任职后期，常生病在家，德宗赐绢二百匹（《新唐书》《册府元龟》作三百匹）以为药用。崔损历任南北两省要职乃至拜相，却无可称。他不扩建也不商议迁移母亲的坟墓，不参加为尼的姐姐的葬礼，为士人君子所批评。[[建中_(唐朝)|建中]]以后，宰相少有久在位的，才数年就获罪被黜，而崔损因迎合德宗之意，在相位八年。德宗也知道公众认为崔损不堪为相，但因崔损事他谨慎，他继续宠信崔损，以为长者。<ref name=LZY/>十九年（803年）三月，[[唐献祖|唐献祖]]、[[唐懿祖|唐懿祖]]庙成，崔损奉敕摄[[太尉|太尉]]，告[[太庙|太庙]]。<ref>[[:s:旧唐书/卷26|《旧唐书》卷二十六]]</ref><ref>[[:s:遣宰臣告庙诏|:s:遣宰臣告庙诏]]</ref>闰十月，崔损卒，<ref name=NBT7/><ref name=BT13/><ref>[[:s:資治通鑑/卷236|《资治通鉴》卷二百三十六]]</ref>赠[[太子太傅|太子太傅]]，谥'''靖'''，<ref name=NBT167/>赙布帛五百端、米粟四百石。<ref name=BT136/>
