'''崔彦昭'''（{{bd|9世纪||9世纪|catIdx=C}}），字'''思文'''，[[中国|中国]][[唐朝|唐朝]][[官员|官员]]，[[唐僖宗|唐僖宗]]年间为[[宰相|宰相]]。

== 家世 ==
崔彦昭出自显赫的[[清河崔氏|清河崔氏]]小房，但祖父崔秩、父崔岂都没有任官。<ref>《[[新唐书|新唐书]]》卷七十二（下）</ref><ref name=BT178>[[:s:舊唐書/卷178|《旧唐书》卷一百七十八]]</ref>崔彦昭中进士前，曾拜访姐夫或妹夫[[王凝_(唐朝)|王凝]]，<ref name=EZT>《新唐书》《资治通鉴》均作王凝母为崔彦昭母之妹，即二人为姨表兄弟。但据崔母郑太素墓志，王凝娶崔母次女，故王凝应系崔彦昭姐夫或妹夫；且王凝母虽系宰相[[郑肃|郑肃]]的姐妹（见《旧唐书·王凝传》），和崔母的家族坟墓却不在一处，虽然崔母和王母都出自[[荥阳郑氏|荥阳郑氏]]，但关系已经疏远，不可能是亲姐妹，故崔彦昭和王凝为姨表兄弟系史书讹误。见[http://www.cssn.cn/kgx/zmkg/201504/P020150424394658506335.pdf 李建华《唐宰相崔彦昭之母<郑太素墓志>研究] {{webarchive|url=https://web.archive.org/web/20160304132606/http://www.cssn.cn/kgx/zmkg/201504/P020150424394658506335.pdf |date=2016-03-04 }}</ref>王凝不敬地不着冠带，只是穿着衩衣相见，还戏言：“君不如举[[明经|明经]]。”崔彦昭感到很受冒犯，<ref name=ZZTJ252>[[:s:資治通鑑/卷252|《资治通鉴》卷二百五十二]]</ref>因为明经考试，被认为較簡單，远不如进士考试。即便此时，崔彦昭也因深谙[[儒术|儒术]]闻名。<ref name=NBT183>[[:s:新唐書/卷183|《新唐书》卷一百八十三]]</ref>[[大中_(唐朝)|大中]]三年（849年），崔彦昭中进士，数次响应[[节度使|节度使]]们的征辟奏授，精明于吏治，在各任所有声望且政绩都拔尖。<ref name=BT178/><ref name=NBT183/>

== 唐懿宗年间 ==
[[唐懿宗|唐懿宗]][[咸通|咸通]]三年（862年），崔彦昭作为户部员外郎，与吏部侍郎[[郑处诲|郑处诲]]、[[蕭倣_(唐朝)|萧仿]]、吏部员外郎[[杨俨|杨俨]]试宏词选人。咸通年间（860年—874年），崔彦昭累迁兵部员外郎，转郎中、知制诰，拜[[中书舍人|中书舍人]]，再迁[[户部侍郎|户部侍郎]]，判本司事。<ref name=BT178/>八年（867年）十月，崔彦昭时为户部侍郎判度支，奏请下诸道州府场监院依限送纳及给还商人，不得托称截留，<ref>《[[册府元龟|册府元龟]]》</ref>懿宗准奏。<ref name=BT19-1>[[:s:旧唐书/卷019上|《旧唐书》卷一十九上]]</ref><ref>[[:s:请禁占留商人换牒奏|:s:请禁占留商人换牒奏]]</ref>

咸通十年（869年），任[[检校|检校]][[礼部尚书|礼部尚书]]、[[孟州|孟州]]刺史、[[河阳节度使|河阳节度使]]，进阶金紫。十一年（870年），<ref>《旧唐书》本传作十二年。此从《旧唐书·懿宗纪》，否则按下文，崔彦昭三年后任满受代时，唐懿宗已崩，不可能准老人请其留任之奏。</ref>在河阳三城节度、孟怀泽观察使、[[中散大夫|中散大夫]]、检校礼部尚书、孟州刺史、[[御史大夫|御史大夫]]任上被任为[[金紫光禄大夫|金紫光禄大夫]]、检校[[刑部尚书|刑部尚书]]、[[太原|太原]]尹、北都留守、河东节度管内观察等使。当地[[沙陀|沙陀]]部落不遵法纪，崔彦昭恩威并施，三年间，境内大治，军民歌颂。任满受代时，数千老人诣阙上书乞求让他留任，<ref name=BT178/>获准。<ref name=NBT183/><ref>[[:s:答河东军民乞留节度使崔彦昭诏|:s:答河东军民乞留节度使崔彦昭诏]]</ref>十四年（873年）二月，因[[振武节度使|振武节度使]][[李国昌|李国昌]]作乱杀害云州防御使[[段文楚|段文楚]]，懿宗诏崔彦昭与[[幽州节度使|幽州节度使]][[张公素|张公素]]率师讨之，<ref name=BT19-1/>无功。<ref>[[:s:新唐書/卷218|《新唐书》卷二百一十八]]</ref>

== 唐僖宗年间 ==
十四年（873年），懿宗崩，年轻的僖宗继位。<ref name=ZZTJ252/>宰相之一[[赵隐|赵隐]]与崔彦昭同科进士，推荐崔彦昭长于治财赋。<ref>《旧唐书》还称另一宰相[[高璩|高璩]]也和崔彦昭同科进士并推荐了他，但高璩早在咸通六年（865年）就已去世。见[[:s:資治通鑑/卷250|《资治通鉴》卷二百五十]]。</ref>[[乾符|乾符]]元年（874年）三月，崔彦昭被召为[[吏部侍郎|吏部侍郎]]，充诸道盐铁转运使，<ref name=BT178/>当月又改授尚书[[兵部侍郎|兵部侍郎]]。八月，被任为[[中书侍郎|中书侍郎]]，加衔[[同平章事|同平章事]]，为[[实质宰相|实质宰相]]。<ref name=ZZTJ252/><ref>《旧唐书·僖宗纪》作四月加同平章事，十一月改中书侍郎。《册府元龟》作四月守本官加同平章事。</ref><ref name=NBT9>[[:s:新唐書/卷009|《新唐书》卷九]]</ref><ref>[[:s:授崔彦昭中书侍郎判度支制|:s:授崔彦昭中书侍郎判度支制]]</ref>因近期宰相[[杨收|杨收]]、[[路岩|路岩]]、[[韦保衡|韦保衡]]都因在任内腐败而新遭贬杀，高级宰相萧仿改革弊政，崔彦昭协助，二人将素日厌恶的宰相[[刘邺|刘邺]]罢免。数月后，百职修举，察不至苛，为士人君子所称。<ref name=NBT183/>十一月，迁[[门下侍郞|门下侍郞]]兼[[刑部尚书|刑部尚书]]。乾符二年（875年）正月，率文武百僚上尊号，僖宗御正殿受册。<ref name=BT19-2>[[:s:旧唐书/卷019下|《旧唐书》卷一十九下]]</ref>僖宗下诏嘉奖他，比他为[[尧|尧]]、[[舜|舜]]时期的[[皋陶|皋]]、[[夔|夔]]。<ref name=BT178/>六月，累兼[[尚书右仆射|尚书右仆射]]兼门下侍郎。<ref>《新唐书》卷六十三</ref>

崔彦昭侍奉母亲[[郑太素|郑太素]]至孝，虽然官拜宰相，退朝侍膳与家人杂处时，顺色柔声，承奉左右，如同平民，为士人所嘉。<ref name=BT178/>庆贺过年时，公卿拜席，时人以为荣。<ref>[[:s:新唐书/卷182|《新唐书》卷一百八十二]]</ref>但郑太素却害怕他处罚他时任兵部侍郎的姐妹夫王凝，在崔彦昭能听到的场合对侍婢说：“为我多制作鞋袜，王侍郎母子肯定要被放逐了，我要和王家妹子（即王母）一起走。”<ref name=EZT/>崔彦昭哭着下拜：“我一定不敢这么做。”于是王凝免遭报复。<ref name=ZZTJ252/><ref name=NBT183/>

伶人[[李可及|李可及]]仗着懿宗宠爱，很是横行，在崔彦昭弹劾下被放逐[[岭南|岭南]]而死。<ref name=BT181>[[:s:舊唐書/卷181|《旧唐书》卷一百八十一]]</ref>

三年（876年）三月，崔彦昭充太清宫使、弘文馆大学士，与[[郑畋|郑畋]]、[[李蔚_(唐朝)|李蔚]]同知政事，六月兼左仆射，前后共三次加兼官，都如前判度支。九月，进阶[[特进|特进]]。四年（877年）正月，为[[司空|司空]]，闰二月罢。<ref name=NBT9/>五年（878年）十月，罢相为[[太子太傅|太子太傅]]。<ref name=ZZTJ252/><ref name=NBT183/><ref name=BT19-2/><ref>《资治通鉴》作三年三月罢为太子太傅，但此与其其他仕途记载不符。</ref>卒于任上，时间不详。<ref name=NBT183/>

== 子 ==
按《旧唐书》本传，崔彦昭有子崔保谦。《新唐书·宰相世系表（崔氏表）》则只列出崔彦昭子崔砺，字殷诰。未详崔保谦与崔砺是否一人。
