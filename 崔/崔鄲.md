'''崔鄲'''（{{bd|780年||850年|8月31日|catIdx=C}}），字'''晋封'''，<ref name=epitaph>《唐故淮南节度副大使知节度事管内营田观察处置等使金紫光禄大夫检校司空兼扬州大都督府长史御史大夫上柱国清河郡开国公食邑二千户赠司徒崔公墓志铭并序》</ref>[[唐朝|唐朝]]官员，在[[唐文宗|唐文宗]]、[[唐武宗|唐武宗]]兄弟年间担任宰相。

== 家庭背景 ==
崔鄲出自[[清河崔氏|清河崔氏]]-{定著}-六房之一的清河小房，曾祖父崔综，[[醴泉|醴泉]]县令，赠吏部侍郎；祖父[[崔佶|崔佶]]曾任[[太子中允|太子中允]]，赠左仆射；父亲[[崔倕|崔倕]]任检校吏部郎中兼[[御史中丞|御史中丞]]，赠[[太师|太师]]。<ref>[[s:舊唐書/卷155|舊唐書‧卷155]]</ref><ref name=NBT72>《新唐书》卷七十二</ref>母夫人[[陇西李氏#姑臧房|姑臧李氏]]，赠秦国太夫人。<ref name=epitaph/>

崔鄲至少有七个兄弟：哥哥[[崔邠|崔邠]]、[[崔鄷|崔鄷]]、[[崔郾|崔郾]]、[[崔郇|崔郇]]、[[崔邯|崔邯]]、[[崔鄯|崔鄯]]，弟弟[[崔鄜|崔鄜]]。<ref name=epitaph/><ref name=NBT72/><ref>墓志铭则称崔郸为第六子。</ref>在众兄弟中，崔邠最为知名，事迹在《[[旧唐书|旧唐书]]》《[[新唐书|新唐书]]》中有大量记载；<ref name=BT155>[[:s:旧唐书/卷155|《旧唐书》卷一百五十五]]</ref><ref name=NBT163>[[:s:新唐书/卷163|《新唐书》卷一百六十三]]</ref>除崔邯、崔鄜外，崔氏六兄弟都在唐朝政府中身居高位。<ref name=BT155/>

== 拜相前 ==
[[贞元_(唐朝)|贞元]]十九年（803年），崔郸中[[进士|进士]]。调补秘书省正字，再调以书判入高等，授[[渭南|渭南]]县尉。[[浙西观察使|浙西观察使]][[李翛|李翛]]辟崔郸为观察推官，授监察里行，旋征入朝。拜正监察御史，转左补阙，迁起居舍人，改司勋员外郎、<ref>《唐尚书省郎官石柱题名考》卷八《司勋员外郎》下有崔郸题名。[[岑仲勉|岑仲勉]]据此考订崔郸在唐穆宗、[[唐敬宗|唐敬宗]]年间曾任此官。</ref>刑部郎中。[[元和_(唐朝)|元和]]十五年（820年）六月，他曾因宗庙事务被罚一季俸料，削两阶。<ref>[[:s:责降宗正少卿李子鸿等敕|:s:责降宗正少卿李子鸿等敕]]</ref>前宰相[[杜元穎|杜元穎]]为[[西川节度使|西川节度使]]，请崔郸为自己效力，崔郸被诏授检校司勋郎中兼[[御史中丞|御史中丞]]，仍赐金紫。<ref>根据《旧唐书》卷一百六十三，杜元穎在823年 - 829年年间担任此职，崔鄲只能在其间在他手下做事。</ref>后征为兵部郎中，转考功郎中。唐文宗好文，崔郸因能作文且见识广，被充[[翰林学士|翰林学士]]，加知制诰，拜中书舍人，依前充学士，且文宗认为他有宰相器。崔郸不想久居要职，屡次上疏请辞，[[大和_(唐朝)|大和]]六年（832年），罢翰林学士。<ref name=BT155/>文宗出其守本官，不久改工部侍郎，充集贤殿学士、判院事。七年（833年）九月，以中书舍人为监内官考使。<ref name=CFYG>《册府元龟》</ref>八年（834年）十二月，拜礼部侍郎。<ref name=epitaph/><ref>因两唐书对崔郸仕途记载有分歧，此从墓志铭。</ref>

九年（835年），改[[兵部侍郎|兵部侍郎]]。[[甘露之变|甘露之变]]后，大臣多躲在家，崔郸却不顾子弟劝谏，坚持上朝，时公卿上朝的只有三四人。[[开成|开成]]元年（836年）春，知吏部选事。四月，文宗召崔郸和尚书右丞知铨事[[郑肃|郑肃]]商议选官标准时，向崔鄲询问他是如何处理不合格的候选人的。崔鄲说他会把这些人送到边疆任职，文宗不赞同，认为这是苦了边疆的百姓。但他仍然在当年任命崔鄲为[[吏部侍郎|吏部侍郎]]。二年（837年）十一月，崔鄲离开长安，任[[宣歙觀察使|宣歙觀察使]]、[[宣州|宣州]]刺史，兼御史大夫。当月，奏[[溧阳县|溧阳县]]百姓陈埏五代同爨，请免赋税，获准。三年（838年）十二月，奏请旌表陈埏门闾，获准。<ref>[[:s:旌陈埏五代同爨奏|:s:旌陈埏五代同爨奏]]</ref>当时茶法还由地方掌管，地方茶业的生产还是由州县严格控制，从事茶叶生产的茶户必须在地方官府规定的茶园制作茶叶，茶法实行专卖制度，官府不再收购茶户的茶叶，而是向茶户发放公帖，茶户取得公帖后才可以出售茶叶。一些百姓由于没有公帖而私下出售少量茶叶遭到官府的严惩，而参与缉拿百姓的官吏则得到奖赏。地方长官以下都从税收方面的盈余中得利，长官甚至将二千万钱茶税收入私囊。崔郸到任后不受贿，四年（839年）二月，放松禁令，<ref name=CFYG/>编户心安。他建议徵收茶户两税时还徵缴茶税，未获准。<ref>[[:s:答崔郸奏必茶法诏|:s:答崔郸奏必茶法诏]]</ref>同年，他又被召回长安，任[[太常卿|太常卿]]。<ref name=BT155/><ref name=BT17-2>[[:s:旧唐书/卷017下|《旧唐书》卷一十七下]]</ref>很多人拥辕遮路望着他，哭喊说：父母走了！我们不知道依靠谁了。<ref name=epitaph/>

== 拜相 ==
当年七月二十五日，崔鄲以[[太中大夫|太中大夫]]、守太常卿、[[上柱国|上柱国]]、赐紫金鱼袋守本官，授[[同中書門下平章事|同中書門下平章事]]，为[[实质宰相|实质宰相]]。<ref name=BT17-2/><ref name=ZZTJ246>[[:s:资治通鉴/卷246|《资治通鉴》卷二百四十六]]</ref><ref name=NBT8>[[:s:新唐书/卷008|《新唐书》卷八]]</ref><ref>[[:s:授崔郸平章事制|:s:授崔郸平章事制]]</ref>随后他又任[[中書侍郎|中書侍郎]]，授[[銀青光祿大夫|銀青光祿大夫]]荣衔，<ref name=BT155/>监修国史，五年（840年）二月复兼礼部尚书，依前平章中书修史。<ref name=epitaph/>同年，文宗驾崩，弟唐武宗继位，宰相[[杨嗣复|杨嗣复]]、[[李珏|李珏]]被罢撤，[[李德裕|李德裕]]被任命为首席宰相，而崔鄲的宰相地位不变，仍以本官同平章事。<ref name=ZZTJ246/><ref name=BT18-1>[[:s:旧唐书/卷018上|《旧唐书》卷一十八上]]</ref>[[会昌_(唐朝)|会昌]]元年（841年），武宗认定杨嗣复、李珏不希望自己继位，想处决他们。李德裕在[[杜悰|杜悰]]建议下，和崔鄲及其他宰相[[陈夷行|陈夷行]]、[[崔珙|崔珙]]一同介入，武宗饶过了杨、李的性命，改为流放到距京城很远的地方。<ref name=ZZTJ246/><ref name=BT176>[[:s:旧唐书/卷176|《旧唐书》卷一百七十六]]</ref><ref>[[:s:新唐书/卷174|《新唐书》卷一百七十四]]</ref>

== 拜相后 ==
李德裕忌惮崔郸正直且受宠，崔郸因而乞求去职，<ref>《旧唐书》则称李德裕和崔氏兄弟有长期的交情。此从墓志铭。</ref>十一月，罢相离京，<ref name=NBT8/>任[[剑南西川节度使|剑南西川节度使]]、检校吏部尚书、平章事。<ref name=ZZTJ246/>人们都可惜他去职，而权臣则更横行。崔郸治蜀，政令明具，宽而有制，仁而必断。当时西川的税法只征缗钱，百姓甚至废业以供赋。崔郸请求改变，允许征用货物，蜀人受益，作歌谣歌颂。[[吐蕃|吐蕃]]、[[南诏|南诏]]也敬畏不敢窥境。又废止前任[[李固言|李固言]]广置的淫祠，诛杀流放巫者。武宗的叔父[[唐宣宗|唐宣宗]]继位后，因他累次上表请求面圣，征为检校尚书右仆射，<ref name=BT18-2>[[:s:旧唐书/卷018下|《旧唐书》卷一十八下]]</ref>不久又以他检校[[司空|司空]]，充[[淮南节度使|淮南节度使]]，均仍[[同中书门下平章事|同中书门下平章事]]。淮南大治。卒于淮南任上。<ref name=NBT163/>根据墓志铭，崔郸官终淮南节度副大使知节度事、管内营田观察处置等使、金紫光禄大夫、检校司空、兼扬州大都督府长史、御史大夫、上柱国，'''清河郡开国公'''，食邑二千户。赠[[司徒|司徒]]。<ref name=epitaph/>
