'''淮海公寓'''原名'''万国储蓄会公寓'''、'''盖司康公寓'''或'''戤司康公寓'''（{{lang-en|'''Gascogne Apartments'''}}），是位于[[上海市|上海市]][[徐汇区|徐汇区]][[淮海中路|淮海中路]]1202～1218号的一处公寓住宅，由沿街五层辅楼（淮海中路1204-1218号）和其后的十三层主楼（淮海中路1202号）组成。公寓处于[[淮海中路|淮海中路]]北侧，华亭路、[[东湖路_(上海)|东湖路]]之间，东临原为亨利公寓的[[淮中大楼|淮中大楼]]。<ref name=cl>{{cite book|title =《老上海经典公寓》|publisher =同济大学出版社|author =娄承浩、薛顺生|url =http://www.yplib.org.cn/structure/jdsh/bnsz/jz_73648_1.htm|isbn =9787560831541|deadurl =yes|archiveurl =https://web.archive.org/web/20140813233955/http://www.yplib.org.cn/structure/jdsh/bnsz/jz_73648_1.htm|archivedate =2014-08-13}}</ref>
==历史==
20世纪20年代起，随着上海人口增长，地价飙升，高层公寓开始兴起，淮海公寓也建造于这一时期。20世纪30年代，[[上海法租界|上海法租界]]内的金融机构[[万国储蓄会|万国储蓄会]]投资建造该公寓。公寓由[[赉安|法商赉安洋行]]设计，中法营造厂承建。<ref name=whj>[http://whj.xh.sh.cn/xhwh_website/HTML/xhwh/xhwh_wwbh_wbdw/Info/Detail_25752.htm 盖司康公寓]徐汇区文化局</ref>1935年公寓建成，以万国储蓄会大班盖司康的名字命名为'''盖司康公寓'''。<ref name=dfz>[http://shtong.gov.cn/newsite/node2/node71994/node81772/node81774/node81784/userobject1ai108979.html 特色志 >> 上海名建筑志 >> 欧美风情的沪滨广厦 >> 高档的盖司康公寓与亨利公寓（淮海公寓和淮中公寓）]上海地方志办公室</ref><ref name=p>{{cite book|title =Shanghai Style: Art and Design Between the Wars|author =潘玲|publisher =Long river press|isbn =9781592650781|page = 226}}</ref>1946年，公寓由国民政府接收，提供政府官员使用。中华人民共和国建国后，由上海市房地产部门接管,规定为外国驻沪领事馆租用房屋。<ref name=mz>{{cite book|title = 《上海百年名楼名宅: 上海百年名宅》|publisher = 光明日报出版社 |author = 《上海百年名楼・名宅》编撰委员会| page = 53-55 |isbn = 9787802062368}}</ref>1953年，公寓更名为'''勤工公寓'''。1954年，又因地处淮海路而改称'''淮海公寓'''。<ref name=s>[http://shtong.gov.cn/newsite/node2/node2245/node69543/node69549/node69574/node69592/userobject1ai68200.html 专业志 >> 上海建筑施工志 >> 工程篇 >> 第二章住宅建筑 >> 第三节　公寓]上海地方志办公室</ref>1994年，公寓主楼被选为第二批[[上海市优秀历史建筑|上海市优秀历史建筑]]，2003年又入选[[徐汇区登记不可移动文物列表|徐汇区登记不可移动文物]]。公寓辅楼则分别于1999年和2003年，被列为第三批[[上海市优秀历史建筑|上海市优秀历史建筑]]和[[徐汇区登记不可移动文物列表|徐汇区登记不可移动文物]]。

公寓建成后曾是上海有名的高档公寓，建成之初多为外侨，尤其是领事馆工作人员租住。当时公寓内一室二厅的住宅月租金为200[[美元|美元]]，而且住户必须接受面试才能入住。历史学家[[周谷城|周谷城]]、经济学家孙曜东曾在此居住，<ref name=mz></ref>瑞士、印度尼西亚驻沪领事馆也曾在此办公。<ref name=whj></ref>目前公寓仍为住宅，辅楼底层开设店铺。21 世纪初,公寓主楼被美国驻上海总领事馆租用,作为领事馆工作人员宿舍。<ref name=mz></ref><ref>{{cite web|url =http://old.jfdaily.com/gb/node2/node17/node162/node787/node791/userobject1ai1349.html|title =淮海公寓——曾是驻沪领事们的“家”|publisher =解放日报|accessdate =2014-08-16|deadurl =yes|archiveurl =https://web.archive.org/web/20140819083954/http://old.jfdaily.com/gb/node2/node17/node162/node787/node791/userobject1ai1349.html|archivedate =2014年8月19日|df =}}</ref>

==建筑特色==
===总体设计===
公寓用地面积0.5公顷，包含南侧临街的辅楼和北侧的主楼。主楼和辅楼之间设有绿地和回车道，汽车库和其余附属用房则位于主楼北侧。如此设计可能是为了充分利用公寓所属地块狭长的地形。<ref name=cl></ref><ref name=dfz></ref>
===主楼设计===
公寓主楼是一幢13层建筑，高53米，建筑面积9062平方米。建筑[[桩基|桩基]]深达31米，采用洋松桩和混凝土灌注桩相结合的混合桩基支承。其地上部分则是[[钢筋混凝土|钢筋混凝土]]框架结构，建筑平面成“一字形”。其结构设计巧妙,传力体系沿高度变化复杂而又不失明确,且充分利用了材料的强度。<ref>[http://www.cqvip.com/qk/96024x/200405/9790384.html 淮海公寓房屋质量检测]住宅科技</ref>

该建筑是赉安洋行最具代表性的作品之一，其建筑风格为现代风格，带有明显的[[装饰艺术|装饰艺术]]特征。公寓采取这种风格，是设计者赉安洋行与业主协商的结果。<ref name=l>[http://www.docin.com/touch_new/preview_new.do?id=353333613 赉安洋行在上海作品的研究]</ref>建筑立面中轴对称，呈现由中部向两翼跌落的形态，以米黄色砖铺设。南立面中央设有三条白色的水泥装饰带，两翼对称地布置向内凹进的阳台，阳台栏板亦为白色，每侧的阳台又以一条白色的水泥装饰带分割。如此形成简洁而又强烈的虚实对比，使得该建筑颇具现代风格的特点，<ref name=cl></ref><ref name=whj></ref><ref name=dfz></ref>对称元素的应用则展示了装饰艺术的特点。<ref name=l></ref><ref name=p></ref>

主楼入口位于底层中部，设有雨棚，并与门厅相连。除门厅外，主楼底层还设有接待室等公共用房。二楼部分为车库。三楼以上为住宅。其中二至六层可分为三个单元。中间单元为两个三室户，一个一室户，根据户型设相应阳台。七层以上居住标准更高，每层设有两个五室户。<ref name=cl></ref>共有居室62套，大小在18～24平方米之间；客厅面积较大，为60平方米。室内铺设硬木地板，墙面装有柚木护墙板，卫浴设施齐全，装饰考究。主楼设有四部电梯，其中中间单元设一部客梯，一部货梯；两边单元各设一部客梯。<ref name=whj></ref><ref name=dfz></ref>

===辅楼设计===
公寓辅楼高5层，建筑面积3318平方米。就建筑结构和内、外部设计而言，辅楼均与主楼十分相似。辅楼底层为商铺，二层以上每层设6套居室，共24套。<ref name=dfz></ref><ref name=s></ref>
