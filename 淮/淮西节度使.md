'''淮西節度使'''，又称'''淮宁節度使'''、'''申蔡節度使'''、'''彰义節度使'''，是[[唐朝|唐朝]]在今[[淮南道|淮南道]]西部设置的[[節度使|節度使]]，是割据的藩镇，在817年，平定[[吴元济|吴元济]]后废除。

== 淮南西道 ==
淮西節度使原称'''淮南西道節度使'''，是唐朝为了防止[[安禄山|安禄山]]叛军南下设置的方镇，初期辖区很大。许州（今[[河南|河南]][[许昌|许昌]]）、申州（今河南[[信阳|信阳]]）、光州（今河南[[潢川|潢川]]）、[[郑州|郑州]]、蔡州（今河南[[汝南|汝南]]）、陈州（今河南[[淮阳|淮阳]]）、颍州（今[[安徽|安徽]][[阜阳|阜阳]]）、[[亳州|亳州]]、寿州（今安徽[[寿县|寿县]]）、安州（今[[湖北|湖北]][[安陆|安陆]]）、沔州（今湖北[[安陆|安陆]]）、蕲州（今湖北[[蕲春|蕲春]]）、黄州（今湖北[[黄冈|黄冈]]）、汴州（今河南[[开封|开封]]）、曹州（今[[山东|山东]][[菏泽|菏泽]]）、宋州（今河南[[商丘|商丘]]）、[[徐州|徐州]]、泗州（今[[江苏|江苏]][[盱眙|盱眙]]）、隋州（今湖北[[随州|随州]]）、唐州（今河南[[泌阳|泌阳]]），这二十个州都一度受过淮南西道的節度。[[安史之乱|安史之乱]]结束后，淮南西道的多数州郡被分割。

== 李氏割据 ==

[[李忠臣|李忠臣]]接任[[节度使|节度使]]后，开始不聽[[朝廷|朝廷]]的[[敕令|敕令]]，[[大历|大历]]十四年（779年）三月，李忠臣被族侄[[李希烈|李希烈]]驱逐，此时淮西辖区只剩六州——[[申州|申州]]、[[蔡州_(隋朝)|蔡州]]、[[光州|光州]]、[[寿州_(隋朝)|寿州]]、[[安州_(西魏)|安州]]、[[唐州_(唐朝)|唐州]]。[[唐德宗|唐德宗]][[建中_(唐朝)|建中]]年间，李希烈平定[[山南东道节度使|山南东道节度使]][[梁崇义|梁崇义]]，开始反叛唐朝，称[[皇帝|皇帝]]，国号[[李楚_(政权)|楚]]，与[[朱泚|朱泚]]遥相呼应。

== 吴氏割据 ==
786年，李希烈被[[陈仙奇|陈仙奇]]所杀，陈仙奇归附朝廷。陈仙奇旋即被[[吴少诚|吴少诚]]所杀，重新割据。朝廷在这期间收回了寿州、安州、唐州三州。吴少诚死後，其義從弟[[吴少阳|吴少阳]]繼位，少陽傳子[[吴元济|吴元济]]，多次侵略河南州县。[[元和_(唐朝)|元和]]十二年（817年），[[裴度|裴度]]、[[李愬|李愬]]平定淮西，生擒吴元济，次年[[唐宪宗|唐宪宗]]下诏废除淮西節度使。

== 唐末 ==
881年，以忠武军节度使属下的蔡州为奉国军防御使，次年升为奉国军节度使，897年，以光州、申州归属奉国军节度使，907年，废除奉国军节度使。[[秦宗权|秦宗权]]、[[赵犨|赵犨]]、申丛、郭璠、崔洪、[[朱延寿|朱延寿]]等人曾经担任奉国军节度使。
