'''淮陰三范'''，是指出身于[[江苏省|江苏省]][[淮阴县|淮阴县]]的[[范尉曾|范尉曾]]（[[范耕研|范耕研]]）、[[范紹曾|范紹曾]]（[[范農研|范農研]]）、[[范希曾|范希曾]]（[[范耒研|范耒研]]）三兄弟，學術卓著，世稱「淮陰三范」。三范皆卒業于[[國立南京高等師範學校|國立南京高等師範學校]]（[[南京大學|南京大學]]），故又稱「南雍三范」。

==簡介==

*[[范尉曾|范尉曾]]，字冠東，號耕研，著作有《章實齋年譜》、《管子集證》、《辯經疏證》、《呂氏春秋疏證》等。曾和[[平湖|平湖]][[胡士瑩|胡士瑩]]、[[淮陰|淮陰]][[張煦侯|張煦侯]]、[[孫雨霆|孫雨霆]]合編《中學國文教科書》。

*[[范紹曾|范紹曾]]，字農研，畢業于南京高師數理化部，教育家。

*[[范希曾|范希曾]]，字耒研，號穉露，目錄學家、校-{讎}-學家。

國學大師[[柳詒徵|柳詒徵]]：「淮陰三范，俱以抗志績學聞于南雍。伯冠東，治周秦諸子；仲紹曾，攻物理化學；叔希曾，為歸、方散體文。」三范出身[[清江浦|清江浦]]書香世家，曾祖父范裕源、祖父范冕、父范逵儀，均為縣諸生，以文章著稱。

== 故居及纪念 ==

原位于淮安市[[清河区_(淮安市)|清河区]]（今[[清江浦区|清江浦区]]）东长街272号<ref name="a1">{{Cite web |url=http://www.jshatour.com/shop-126 |title=里运河文化长廊景区 |accessdate=2018-02-12 |author= |date= |work= |publisher=淮安市旅游局公共服务官方平台网站|language=Zh-hans }}</ref>的三范故居，于2010年5月迁址重建<ref name="a2">{{Cite web |url=http://jsnews2.jschina.com.cn/system/2011/11/09/012036571.shtml |title=里运河畔新增一处免费开放景点 |accessdate=2018-02-12 |author= |date=2011-11-09 |work=编辑：杨玉兰 |publisher=中国江苏网，来源：淮安新闻网|language=Zh-hans }}</ref>。与相邻的[[程莘农|程莘农]]故居，是淮安市文管会保护的“淮安市名人故居”。成为位于轮埠路<ref name="a1"/>的[[里运河|里运河]]文化长廊景区的景点之一。由此设立三范故居纪念馆，2011年11月对公众开放<ref name="a2"/>。
