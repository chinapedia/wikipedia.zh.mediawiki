《'''終極鬥士2'''》（{{lang-en|'''''Undisputed II: Last Man Standing'''''}}，或記作{{lang|en|'''''Undisputed 2'''''}}）是一部2006年[[美國|美國]][[武俠片|武俠片]]，由[[伊薩克·佛倫亭|伊薩克·佛倫亭]]執導，為2002年電影《[[終極鬥士|終極鬥士]]》的續集。由[[邁克·賈·懷特|邁克·賈·懷特]]和[[史考特·艾金斯|史考特·艾金斯]]主演。該片於2006年4月11日和2007年1月16日分別在荷蘭與美國以[[錄影帶首映|錄影帶首映]]發行。

== 劇情 ==
故事敘述在前集過後的數年，前世界重量級拳王喬治·「冰人」·錢伯為了在退休時間賺點錢，便而前往[[俄羅斯|俄羅斯]]拍廣告。在造訪當地進行的一系列的拳擊比賽時，他遭他人誣陷持有[[古柯鹼|古柯鹼]]，而再度被送往監獄，但那裡的氣候卻比以前待的監獄還要寒冷。

在那他發現了該監獄官員安排的非法[[綜合格鬥|綜合格鬥]]賽，以助在外的大亨賺取賭金。為此，冰人與背後的大亨達成協議，只要打倒獄中最強的戰士尤里·波卡就能重獲自由。但身手高超的幫派頭目尤里·波卡為了成為世界最強的拳手，而與冰人不合，使得雙方的暴力戰役即將展開。

== 演員 ==
*[[邁克·賈·懷特|邁克·賈·懷特]] 飾 喬治·「冰人」·錢伯（George "Iceman" Chambers）：前世界重量級拳王，退休期間前往俄羅斯，被誣陷持毒後入了獄。
*[[史考特·艾金斯|史考特·艾金斯]] 飾 尤里·波卡（Yuri Boyka）：監獄殺人犯，囚犯中最具權勢的人，擅長綜合格鬥。
*{{le|艾力·丹克|Eli Danker}} 飾 克羅特／尼可拉（Crot／Nikolai）：半身不遂的年邁的囚犯，幫助冰人學習綜合格鬥。
*[[班·克羅斯|班·克羅斯]] 飾 史蒂芬·帕克（Steven Parker）：冰人的監獄室友。
*{{le|馬克·伊凡諾|Mark Ivanir}} 飾 嘎卡（Gaga）：監獄格鬥賽的背後大亨，迫使冰人打贏尤里來為自己賺錢。
*{{le|瓦倫丁·加內夫|Valentin Ganev}} 飾 典獄長馬爾科夫（Warden Markov）：俄羅斯監獄的典獄長。
*{{le|黛西·朗|Daisy Lang}} 飾 史韋特蘭娜（Svetlana）：嘎卡的手下。

== 評價 ==
Kung Fu Cinema的馬克·波拉德給了《絕地鬥士》四星的好評（滿分五星），並稱其為「2007年第一偉大的武術電影，以及伊薩克·佛倫亭迄今最好的」<ref name="KFCinema">{{cite web|url=http://www.kungfucinema.com/reviews/undisputed-2-last-man-standing-2007 |title=Review: ''Undisputed II: Last Man Standing''|last=Pollard|first=Mark|work=Kung Fu Cinema|date=|accessdate=2018-11-22|archive-url=https://web.archive.org/web/20150410061519/http://www.kungfucinema.com/reviews/undisputed-2-last-man-standing-2007|archive-date=2015-10-04|dead-url=yes}}</ref>。

== 續集 ==
續集《[[終極鬥士3：贖罪|終極鬥士3：贖罪]]》於2010年推出，由[[史考特·艾金斯|史考特·艾金斯]]所飾演的尤里·波卡擔任主角。下一部續集《[[終極鬥士4|終極鬥士4]]》於2016年推出，這再次以艾金斯為主角。
