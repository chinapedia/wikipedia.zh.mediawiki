'''數位落差'''（{{lang-en|'''Digital divide'''}}），亦作'''數碼鴻溝'''、'''數碼隔膜'''或'''數碼差距'''，是指社會上不同性別、種族、經濟、居住環境、階級背景的人，接近使用數位產品（如電腦或是網路）的機會與能力上的差異。簡約來說，觀察數位落差可以從接近使用（Access）電腦及網路的機會、以及對於電腦及網路的使用能力（亦可稱之為[[資訊素養|資訊素養]]，Information Literacy）這兩大面向來看。

最早有系統觀察數位落差的國家是[[美國|美國]]，自1995年起，由[[美國商務部|美國商務部]][[國家通信及資訊管理局|國家通信及資訊管理局]]（NTIA）陸續發布數位落差調查報告。

一方面，[[資訊科技|資訊科技]]使大眾的生活質素得到改善，並把繁瑣的日常工作簡化。然而，對於未能享用同等資訊科技的人，卻要繼續沿用舊有的方式去工作，不能透過資訊科技去獲取資訊或把資訊增值。這當中的差異，有可能會使社會的兩極化更趨激烈，從而令社會財富更為不平均。

== 落差的維度 ==
數碼落差並不是一個山峽般模樣、把社會大眾一分為二的分隔。根據研究員的報告，數碼落差的成因可能是以下其中一個或多個：
# 使用效能較低的電腦
# 使用連線較差或價錢過高的互聯網服務（例如：撥號網接）
# 比大眾較難得到電腦技術的支援
# 較難取得需要訂閱的網上內容<!--

== 教學上的意義 ==
由於學生之間的數碼差距，學校在設計課業時，要回應學生之間的數碼差距，以便家裡沒有上網設施、甚或連電腦也沒有的學生，不會因為數碼差距而造成學生成績上的差異。舉例說，有些學校會在小息及午飯時間開放電腦室與學生使用，並讓沒有電腦設備的學生優先使用電腦室；又或對資訊科技有不同掌握的學生訂定不同的課業，使學生不會因為數碼差距而影響課業成績。

== 社會學上的意義 ==
一方面，政府可以利用資訊科技簡化工序，並利用網上服務方便市民。但另一方面，政府亦應透過各種途徑為沒有電腦設備的市民提供免費上網服務，讓他們亦能享用政府的嶄新服務。

此外，香港政府亦曾在主權移交後為中小企提供資助，以協助他們更新或安裝電腦設備，以便使用政府的電腦報關、報稅服務等。[[中华民国行政院|中华民国行政院]]環境保護署則有二手電腦回收轉贈計畫。

==  臉書分隔  ==
'''[[臉書分隔|臉書分隔]]'''<ref>{{Cite news|url=http://www.zaobao.com/forum/views/opinion/story20170623-773468|title=面簿分隔的社会|author=容振声|date=23/06/2017|work=新加坡聯合早報|accessdate=21/3/2018}}</ref> （{{lang-en|Facebook Divide}} ）為數碼鴻溝衍生的概念。臉書分隔的現象在於人使用臉書和不使用臉書所帶來的影響。在2017年印度的新經濟管理學術會議 ICMAPRANE<ref>{{Cite web|url=http://www.jaipuria.ac.in/icmaprane2018/past-conference/|title=ICMAPRANE 2017|accessdate=21/3/2018|author=|date=|publisher=}}</ref>提出了臉書分隔社會的問題，並帶出相關的臉書原住民，臉書移民，臉書邊緣人等概念。臉書移民在使用臉書時候會相應的增加其個人的社會資本。臉書原住民，臉書移民，臉書邊緣人等構成臉書不公義的現象。在2018年 ICMAPRANE<ref>{{Cite web|url=http://www.jaipuria.ac.in/icmaprane2018/icmaprane-2018/|title=ICMAPRANE-2018|accessdate=21/3/2018|author=|date=|publisher=}}</ref>中提出了臉書分隔現象指數。臉書分隔現象指數可作為分析世界各個社交網路的基礎。
