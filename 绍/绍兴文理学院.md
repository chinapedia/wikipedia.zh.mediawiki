'''绍兴文理学院'''（英文：Shaoxing University）位于中国[[浙江省|浙江省]][[绍兴市|绍兴市]]市区的一所全日制本科院校，其前身为绍兴师专、绍兴高专等地方性大专院校，1996年4月，在上述学校基础上，正式成立绍兴文理学院。2005年通过教育部本科教学工作水平评估，成为硕士学位授权建设单位，为升格为省级重点大学——"绍兴大学"作准备。其下属二级学院——"兰亭书法艺术学院"为中国首家开设全日制[[书法学|书法学]]本科专业的院校，同时也是浙江首家外国留学生书法教育基地。

== 学校历史 ==

1909年，[[绍兴山会初级师范学堂|绍兴山会初级师范学堂]]（[[鲁迅|鲁迅]]曾任校长）

1915年，[[浙江省第五师范学校|浙江省第五师范学校]]

1923年，绍兴五师并入[[浙江省第五中学|浙江省第五中学]]。[[抗战|抗战]]胜利前夕，在会稽山区重建[[绍兴简易师范|绍兴简易师范]]

1948年，绍兴简易师范改名为绍兴县立师范学校

1951年8月，省文教厅决定新办“浙江省绍兴初级师范学校”

1956年9月，建立[[绍兴中等师范学校|绍兴中等师范学校]]

1970年，绍兴中师改名为[[绍兴地区师范学校|绍兴地区师范学校]]

1978年4月，改制大专，开始招收大专生

1980年月8日，经[[国务院|国务院]]批准建立[[绍兴师范专科学校|绍兴师范专科学校]]

1984年12月，建立[[绍兴高专|绍兴高专]]

1986年3月，绍兴高专、省电大绍兴分校、绍兴职工大学联合办学

1993年10月，绍兴市委市府决定在绍兴师专、绍兴高专基础上筹建“[[绍兴大学|绍兴大学]]”

1995年8月，绍兴师专与绍兴高专正式合并

1996年4月，经国家[[教育部|教育部]]批准，正式成立绍兴文理学院

== 学校现状 ==

截止2008年秋季，学院有全日制在校学生2万余人。教职员工方面：教职员工总计1800人，其中专职教师1000余名，具有高级职称教师450余名，其中正高职称118名，博士生导师、硕士生导师15人，博士、硕士320余人，8位教授享受政府特殊津贴，另有外聘[[院士|院士]]8人。

== 近景目标 ==

目前学院为一所地方性的普通综合大学，计划在2010年前建成一所：以全日制本科教育为主，拥有20个以上硕士授予点，占地3000亩，建筑总面积65万平方米，全日制在校学生超过2万人，从事高水平教学和科研，具有江南风韵和越文化特色，并在全国有知名度的省级重点大学——绍兴大学。

== 对外交流 ==

学院目前已与美国、德国、法国、奥地利、丹麦、日本、韩国等国数所高校建立学术交流和友好合作关系，学校下属独立二级学院[[元培学院|元培学院]]则与法国凡尔赛大学等4所高校全面合作。
