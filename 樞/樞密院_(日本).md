[[日本|日本]]的'''枢密院'''，是由枢密顾问（顾问官）组成的[[天皇_(日本)|天皇]]的谘询机构。因为处理了许多[[宪法|宪法]]问题，也被称为“宪法的守护者”。1888年设立，在1947年被废止。简称为'''枢府'''，议长被称为'''枢相'''。

== 沿革 ==
1888年为了审议[[宪法|宪法]]草案，根据枢密院官制创设了枢密院。在1889年公布的[[大日本帝国宪法|大日本帝国宪法]]中，也明确规定其为[[天皇|天皇]]的最高谘询机关。首任议长为[[伊藤博文|伊藤博文]]。

长期以来，枢密院是掌控日本国家大政方针的显要部门，但在1931年[[九一八事变|九一八事变]]之后，日本国内政党势力衰退，军部逐渐强硬，枢密院的影响力也日渐式微，最后随着[[日本国宪法|日本国宪法]]的施行，于1947年结束了其历史使命。

== 组织 ==
枢密院设议长1名、副议长1名、顾问官24-28名。任命资格为40岁以上、国家元勋的人物。另外，各国务大臣都自然成为顾问官，可以参加表决。在东京的成年亲王也可以参加会议。

枢密院建筑位于东京都皇居内，建于1921年，二战后供[[皇宫警察|皇宫警察]]使用至今。

== 权限 ==
大日本帝国宪法第56条规定了“枢密顾问根据枢密院官制规定，接受天皇谘询，审议重要的国务”。

向枢密院进行谘询的事项如下。
* [[皇室典范|皇室典范]]、[[皇室令|皇室令]]中属于枢密院权限的事项
* 宪法条款的草案及质疑
* 宪法附属法令的相关草案及质疑
* 枢密院官制及事务规程的改正
* [[紧急勅令|紧急勅令]]、紧急财政处分
* 国际[[条约|条约]]的缔结
* [[戒严|戒严]]的宣告
* 教育的相关重要勅令
* 行政各部的官制等与官规有关的勅令
* 荣典、恩赦等相关勅令
* 其他被谘询的事项

枢密院禁止参与施政（枢密院官制第8条），也不得与大臣以外的人进行公务上的交涉（枢密院事务规程第3条）。

== 与政府的关系 ==
一般当枢密院与政府的政策发生对立时，多数通过对话协商，由某一方作出让步。但1927年为救济[[台湾銀行|台湾銀行]]，[[第1届若槻内阁|第1届若槻内阁]]作出的紧急勅令案被枢密院以19对11否决，从而导致了内阁总辞职。这是唯一一次由枢密院压倒政府内阁的实例。

尽管如此，但实际上并没有明文规定被枢密院否决的情况下，内阁必须总辞职。一种观点认为，这次偶然事件只是由于首相的软弱造成的。

例如历史上还有一次类似的事件。1930年，[[濱口雄幸|濱口雄幸]]內閣要求批准[[倫敦海軍條約|倫敦海軍條約]]。当时，希望批准条约的政府（[[民政党|民政党]]执政）为一方，枢密院、军部、[[鸠山一郎|鸠山一郎]]等为主的在野党[[政友会|政友会]]为对立的另一方。当时有人主张，内阁断然实行军缩是干涉天皇统帅权的行为，枢密院内部也积极开展反对滨口内阁的行动。但是，滨口首相得到了[[元老|元老]][[西园寺公望|西园寺公望]]和舆论的支持，坚决抵制枢密院，甚至拒绝了枢密院核心人物[[伊东巳代治|伊东巳代治]]的调查要求。当时的[[东京日日新闻|东京日日新闻]]为首的各大报纸也都猛烈抨击枢密院。最后枢密院一方只能让步，批准了这一条约。
