'''紗倉真菜'''（{{lang-ja|紗倉 まな}}，{{bd|1993年|3月23日|}}），[[日本|日本]]的[[AV女優|AV女優]]<ref name="SOD"/>、[[日本電視藝人|藝人]]、[[歌手|歌手]]、[[小說家|小說家]]、[[YouTuber|YouTuber]]，[[千葉縣|千葉縣]]出身。現為「[[Soft_on_Demand|SOD Create]]」的當家王牌女優<ref name="SOD"/><ref>{{cite web|url=http://www.playno1.com/portal.php?mod=view&aid=26931#.W8SCpGgzaUk |title=Rio退位！現在AV女優最崇拜的對象是… |publisher=AVNo.1-PLAYNO.1 |date=2018-05-15}}</ref>，所屬於「{{link-ja|Mine's|マインズ (芸能事務所)}}」事務所<ref>{{Cite web |url=http://mines-pro.jp/pc/blog.html |title=Blog・Twitter - モデルプロダクション[マインズ] |publisher={{link-ja|マインズ|マインズ (芸能事務所)}} |accessdate=2018-10-14}}</ref>。

== 簡歴 ==
2011年11月，拍攝《{{lang|ja|工場萌え美少女 紗倉まな 18歳}}》的寫真影片、並進行[[凹版雜誌|凹版雜誌]]的活動。

2012年，在就讀高等專門學校時<ref name="lite-ra">{{Cite web |date=2015-01-21 |url=http://lite-ra.com/2015/01/post-805.html |title=「14歳でAV女優を志した」さんまもハマったAV女優・紗倉まなが抱える闇 |work=リテラ |publisher=サイゾー |page=1 |accessdate=2015-04-24|language=ja}}</ref>，以作品《{{lang|ja|紗倉まな AVDebut}}》在「[[Soft_on_Demand|SOD Create]]」[[AV|AV]][[出道|出道]]<ref name="cyzo">{{cite web|url=http://www.cyzo.com/2012/03/post_10086.html |title=「激しいのをいっぱいされたいです♪」Mで工場萌えな大型新人、紗倉まなが登場!! |work=日刊サイゾー  |publisher= |date=2012-03-08 |accessdate=2013-10-26|language=ja}}</ref><ref>{{Cite book|editor= |title=ショートヘアの究極美女 |date=2012-5 |publisher=ミリオン出版 |series=ミリオンムック |isbn=978-4813066033 |volume=3 |pages=19-22|language=ja}}</ref>。同年4月16日在成人網站《[[DMM.com|DMM.com]]》得到作品類及女優類的銷售第一名<ref>{{cite journal|date=2012-05-22 |publisher=GOT |title=月刊DMM |volume=2012年7月号 |pages=104-106|language=ja}}</ref>。

2013年3月1日登上青年漫畫雜誌《{{lang|ja|ヤングアニマル嵐}}》的封面，為史上第二人<ref>成為繼[[Rio_(AV女優)|Rio]]第二位擔任封面人物的AV女優。</ref>。

在SOD大獎2012，總共得到：最優秀女優獎、優秀女優獎、最優秀銷售作品獎、最優秀包裝作品獎、優秀包裝作品獎第3名、最優秀出租作品獎、一萬部達成獎3作品，總共得到六個冠軍。

2013年3月9日與6月11日，在《{{link-ja|FM大阪|エフエム大阪}}》廣播節目《{{lang|ja|TENGA Presents Midnight World Cafe 〜TENGA茶屋〜}}》的錄音中擔任來賓。當時被[[宮川大輔|宮川大輔]]說溜嘴承認是他最喜歡的AV女優。同年的6月28日上映的電影《{{lang|ja|ゴッドタン キス我慢選手権 THE MOVIE}}》、飾演[[劇團一人|川島省吾]]的妹妹。同年，得到了《[[成人廣播獎|成人廣播獎]]》2013的「新人女優獎」以及「FLASH獎」兩個獎<ref>{{cite web|url=http://adult-awards.com/archives/2013/ |title=2013年の受賞者 |work=スカパー!アダルト放送大賞 |year=2013 |accessdate=2013-03-08|language=ja}}</ref>。

[[明石家秋刀魚|明石家秋刀魚]]在2012年7月的節目《{{lang|ja|FNS27時間テレビ笑っていいとも！}}》獲選為Lovemate 10的第7名<ref name="nikkan-spa20120811-1">{{Cite web |date=2012-08-11 |url=https://nikkan-spa.jp/268953 |title=27時間テレビで注目のAV女優・紗倉まな「さんまさん、新作見てください！」 |work=日刊SPA! |publisher=[[扶桑社|扶桑社]] |page=1 |accessdate=2014-05-09|language=ja}}</ref>。在2013年10月18日發售的寫真雜誌《{{lang|ja|フライデー}}》報導了他們秘密約會的消息<ref name="lite-ra"/><ref>{{cite web|url=http://www.playno1.com/article-12002-1.html#.W8SOAmgzaUk |title=搭上明石家さんま! 紗倉まな‧20歳、性欲、覚醒 |publisher=AVNo.1-PLAYNO.1 |date=2013-10-28}}</ref>。

在[[豐田汽車|豐田汽車]]的汽車資訊網站「GAZOO」進行了專欄的連載。由於是特例的起用而在網際網路上造成話題<ref name="j-cast"/>。身為專欄作家的她在2015年為四個專欄擔任寫作工作。

2014年，在[[SEGA|SEGA]]出品的遊戲《[[人中之龍0_誓言的場所|人中之龍0 誓言的場所]]》舉辦的性感女優人氣投票活動得到第2名<ref>{{Cite news|url=http://www.appledaily.com.tw/realtimenews/article/new/20140906/465046/ |title=AV女優爭遊戲女角　人氣投票結果出爐 |publisher=[[蘋果日報_(台灣)|蘋果日報]]|date=2014-09-06}}</ref>。

2015年1月15日，發表自身初次的[[隨筆|隨筆]]《身為高專生的我和世界邂逅只是個天職》（{{lang|ja|高専生だった私が出会った世界でたった一つの天職}}）創下了大賣的紀錄<ref>{{Cite web |url=https://natalie.mu/owarai/news/136236 |title=バカリズム、紗倉まなの自伝エッセイに帯コメント |publisher=ナタリー |date=2015-01-15 |accessdate=2018-10-15|language=ja}}</ref>。分別在網路商店[[亞馬遜日本|亞馬遜日本]]的日本演員書類（綜合）得到銷售第1名和[[樂天_(Rakuten)|樂天]]的電影（日本）類別的銷售第1名。作品發售初期在各地的書店都銷售一空，並形成完全沒有庫存的熱賣盛況<ref>{{Cite web |url=https://www.oricon.co.jp/news/2047309/full/ |title=バカリズムも夢中 人気セクシー女優の初著書が大ヒット |publisher=[[Oricon|ORICON NEWS]] |date=2015-01-17 |accessdate=2018-10-15|language=ja}}</ref>。

2015年3月3日，再度獲得《[[成人廣播獎|成人廣播獎]]》2015的「女優獎」、及第二次的「FLASH獎」。加上2013年的「新人女優獎」，達成了史上最初的三冠王<ref>{{Cite web |url=https://www.tokyo-sports.co.jp/entame/entertainment/373408/ |title=「スカパー！ アダルト放送大賞」女優賞の紗倉まなが涙 |publisher=[[東京體育|東スポWeb]] |date=2015-03-03 |accessdate=2018-10-15|language=ja}}</ref>。

2016年2月12日，她發表了自己的原創小說《最低。》，描繪了生活在AV界中女人的姿態<ref>{{Cite web |url=http://thetv.jp/news_detail/72448/ |title=紗倉まなインタビュー前編「AV女優の処女作誕生!?」  |publisher=Smartザテレビジョン |date=2016-02-12 |accessdate=2016-03-26|language=ja}}</ref>，並拿下日本暢銷書排行榜第一名<ref>{{Cite news|url=https://www.setn.com/News.aspx?NewsID=366999 |title=這AV女優不簡單！　紗倉真菜小說稱霸日本暢銷榜第一名 |publisher=[[三立新聞|三立新聞]]|date=2018-04-10}}</ref>。

2016年5月4日，獲得《[[DMM成人獎|DMM成人獎]]》2016「媒體獎」<ref>{{Cite web|url=http://www.dmm.co.jp/adultaward/archive/winner/2016_html/=/navi=none/ |title=DMM.R18アダルトアワード 2016 受賞結果  |publisher=[[DMM.com|DMM.R18]] |date=2016-05-04 |accessdate=2018-10-15|language=ja}}</ref>。

2017年3月18日，發表第二部長篇小說《凹凸》<ref>{{Cite web|url=https://eiga.com/news/20170318/11/ |title=紗倉まな、断酒して執筆の第2弾小説「凹凸」発売「飲まない寂しさを主人公にぶつけた」  |publisher=映画ニュース |date=2017-03-18 |accessdate=2018-10-15|language=ja}}</ref>，發行第3天便宣布再版<ref>{{Cite web|url=https://www.tokyo-sports.co.jp/entame/entertainment/678438/ |title=紗倉まなの小説「凹凸」は発売３日で重版決定の好スタート |publisher=東スポWeb |date=2017-04-26 |accessdate=2018-10-15|language=ja}}</ref>。

2017年9月，宣布個人處女作《最低。》決定改編成真人電影<ref>{{Cite news|url=https://tw.appledaily.com/new/realtime/20171124/1247172/ |title=SOD頭牌女優籲男性　「AV都是演的！不要學」 |publisher=[[蘋果日報_(台灣)|蘋果日報]]|date=2017-11-24}}</ref>，並在10月26日，入圍2017年[[東京國際影展|東京國際影展]]<ref>{{Cite news|url=https://www.setn.com/News.aspx?NewsID=438131 |title=AV女優職業縮影寫成小說！改編電影《最低》獲影展肯定 |publisher=[[三立新聞|三立新聞]]|date=2018-10-04}}</ref>。

2017年12月，應[[筑波大學|筑波大學]]邀請，到資訊學系「內容應用理論」課程擔任特別講師，以她跨媒體的實務經驗，分析出版、廣告、動畫等媒體業界的現況與發展<ref>{{Cite news|url=https://udn.com/news/story/6810/2889076 |title=AV女優到大學當講師 超人氣引發大學生暴動 |publisher=[[聯合報|聯合新聞網]]|date=2017-12-22}}</ref>。

2018年10月13日，日本成人網站《FANZA》（原《[[DMM.com|DMM.R18]]》）公布網站統計資料，紗倉真菜在男性及女性的「最受歡迎AV女優」排行榜都獲得第2名<ref>{{cite web|url=http://www.playno1.com/article-28204-1.html#.W8SA6mgzaUk |title=誰是日本最紅的AV女優？網站檢索數告訴你！ |publisher=AVNo.1-PLAYNO.1 |date=2018-10-13}}</ref><ref>{{Cite news|url=http://ent.ltn.com.tw/news/breakingnews/2580365 |title=最愛AV女優出爐！ 她超狂男女榜通殺 |publisher=[[自由時報|自由時報]]|date=2018-10-14}}</ref>。
