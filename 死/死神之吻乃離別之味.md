《'''死神之吻乃離別之味'''》（{{lang-ja|死神の接吻は別離の味}}）為[[ALcot|ALcot シトラス]]社2009年4月24日發表的[[戀愛冒險遊戲|戀愛冒險遊戲]]。<ref>{{cite web|url=http://www.getchu.com/soft.phtml?id=619585|title=死神の接吻は別離の味|publisher=Getchu.com|accessdate=2014年3月20日|language=ja}}</ref>

== 世界觀 ==
作品開始地點為臨海小鎮，小鎮中有著名為八潮學園的學校以及名為「ここのか」的影像出租店，小鎮中有著可以將凡人引渡的死神，而死神一般是看不見的，除非是將死之人。死神必須保持平和，不能輕易展露喜怒哀樂，因此看起來總是面無表情。

== 故事 ==
住在在臨海的小鎮的少年少女，自小喜歡著彼此，好似兩人之間有著強烈的羈絆，然而在3月14日，白色情人節，少年為了傳達自己的對少女的心意，約少女到海邊玩，然而由於意外事故，在海邊遊玩的少年及少女，還來不及將彼此的思念訴說給對方便迎向了死別，被死神帶走的是少女，而活下來的是少年。少女成為少年臨海的小鎮生活時，難以忘懷的遺憾。
長大後，當年的少年，也是遊戲的主人公天宮誠，進入八潮學園就讀，目前為二年級學生，與小他一歲妹妹雫一起生活，平常放學後就到同學穂乃香父親開的影像出租店打工。某天，天宮誠的同學里中凛的哥哥，遭遇了意外事故而死亡，在事故現場，天宮誠發現了一名手持巨大鐮刀的少女，神奇的是除了自己外其他人都沒查覺到少女突兀的存在，手持鐮刀的少女對天宮誠說：「你能看見我的身姿嗎？」「原來如此。你也快死了呢。」少女的身分，為名為「琥珀」的死神。

== 登場角色 ==
=== 主要角色 ===
; {{nihongo|天宮誠|天宮 誠（あまみや まこと） || }}
: 身高：176ccm。體重：65kg。
: 本作主人公，八潮學園的2年級學生，與小他一歲妹妹雫一起生活，平常放學後就到同學穂乃香父親開的影像出租店「ここのか」打工。有著端正的性格同時也愛於助人，甚至連明顯對自己不公平的事情依然會幫忙。在學校很受女生的歡迎，被告白了許多次。對於多年前在海岸發生的不幸事件的少女深感自責，眼神毫無生氣，為自己還活著這個事實感到無奈，因此對自己身邊的安危不太注意。

; {{nihongo|新島穂乃香|新島 ほのか（あらしま ほのか） || 聲：[[鮎川ひなた|鮎川ひなた]]<ref>{{cite web|url=http://www.h-comb.biz/product/02/page_ch/ch_01.html|script-title=ja:新島 ほのか|publisher=[[ALcot|ALcot]]|accessdate=2017年2月18日|language=ja}}</ref>}}
: 身高：153cm。體重：42kg。三圍：B83/ W52/ H80。
: 八潮學園的2年級學生，也是該班的委員長，與天宮誠同班。放學後即回到父親開的影像出租店「ここのか」幫忙。性格端正，待人平和，在學校中很受男生的歡迎，被告白了許多次。在影像出租店與天宮 誠一同工作時，總是被父親揶揄說「你們阿，趕快結婚好了。」，對於父親毫無節操的湊合發言感到慌亂。對於成人的東西感到不好意思，因此很少打理出租店中的成人區。

; {{nihongo|天宮雫|天宮 雫（あまみや しずく） || 聲：[[五行薺|五行薺]]<ref>{{cite web|url=http://www.h-comb.biz/product/02/page_ch/ch_02.html|title=天宮雫|publisher=ALcot|accessdate=2017年2月18日|language=ja}}</ref>}}
: 身高：162cm。體重：45kg。三圍：B81/ W51/ H79。
: 天宮誠妹妹，但是沒有血緣關係，是繼父的小孩。在海岸發生的不幸事件發生後，天宮誠的父母不久也離婚，因此自很小的時候就跟天宮誠同住，然而剛開始時難以接受多了一個哥哥的現實。很少說話，經常趁哥哥不在房間時賴在哥哥房間看女性雜誌，每天都要哥哥叫他起床。自稱「我不在時哥哥什麼都做不到」不過實際上完全顛倒。對於自己長得很高，屁股很大的事情特別在意。

; {{nihongo|琥珀|琥珀（こはく） || 聲：[[夏野こおり|夏野こおり]]<ref>{{cite web|url=http://www.h-comb.biz/product/02/page_ch/ch_03.html|title=琥珀|publisher=ALcot|accessdate=2017年2月18日|language=ja}}</ref>}}
: 身高：146cm。體重：38kg。三圍：B81/ W51/ H79。
: 在里中凛的哥哥死亡的事故現場發現的手持巨大鐮刀少女，自稱是死神，除了天宮誠外其他人都沒查覺到的存在，自從被天宮誠發現後，對天宮誠眼神毫無生氣的將死狀態感到興趣，因此不時的會出現在天宮誠的生活。由於是死神因此可以穿牆來去自如，所以有時候會在很尷尬的時間出現，特別是天宮誠洗澡的時候，而且特別盯著天宮誠的胯下看，使天宮誠感到困擾。由於死神總是面無表情，因此也無法由表情看出喜怒哀樂。
