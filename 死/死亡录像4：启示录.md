《'''死亡录像4：启示录'''》（{{lang-es|''[REC] 4: Apocalipsis''}}，某些地区写作“Apocalypse”<ref>{{cite web|url=http://www.aullidos.com/pelicula/7285/rec-4-apocalipsis/|title=REC 4|author=Aullidos|date=2014-10-31|work=Aullidos.COM|accessdate=2015-04-24}}</ref><ref>{{cite web|url=http://www.aullidos.com/videos/3574/rec-4-entrevista-balaguero/|title=Entrevista Jaume Balagueró REC 4|author=Aullidos|work=Aullidos.COM|accessdate=2015-04-24}}</ref>）是一部2014年[[西班牙|西班牙]][[恐怖片|恐怖片]]，《[[死亡录像|死亡录像]]》系列的第四部及最後一部作品<ref>{{cite web |url=http://twitchfilm.com/2013/05/new-poster-for-balagueros-rec4-apocalypse.html|title=Balaguero's [''REC'']''4: Apocalypse''!|accessdate=2013-10-07}}</ref>。电影是《[[死亡录像2|死亡录像2]]》的正式续集，发生在第二部电影的事件过后不久<ref>{{cite web |url=http://www.shocktillyoudrop.com/news/21154-fantastic-fest-11-exclusive-balaguero-talks-rec-3-and-rec-4|title=Fantastic Fest '11 Exclusive: Balaguero Talks Rec 3 and Rec 4|accessdate=2013-04-13}}</ref>。前两部作品的导演{{le|豪梅·巴拿盖鲁|Jaume Balagueró}}与女主角{{le|曼奴拉·沃拉斯科|Manuela Velasco}}回归，曼奴拉再度出演处在危险中的女记者安其拉·维达尔<ref>{{cite web |url=http://www.shocktillyoudrop.com/news/171221-the-teaser-trailer-for-rec-4-apocalypse-is-upon-us|title=The Teaser Trailer for ''REC 4: Apocalypse'' is Upon Us|accessdate=2013-04-16}}</ref>。

影片于2014年9月9日在[[多伦多国际电影节|多伦多国际电影节]]首映，于2014年10月3日在[[锡切斯电影节|锡切斯电影节]]上映，10月31日在西班牙上映，12月5日登陆[[台湾|台湾]]，2015年1月8日登陆[[香港|香港]]。2015年1月2日，影片在美国由{{le|Magnet Releasing|Magnet Releasing}}以剧院上映和[[视频点播|视频点播]]方式发行<ref>{{cite web|url=http://bloody-disgusting.com/news/3321447/magnet-acquires-rec4-apocalypse/|title=Magnet Acquires '[REC]4: Apocalypse'! -|work=bloody-disgusting.com|accessdate=2015-04-24}}</ref>。

== 剧情 ==
影片开头，一支{{le|特别行动小组|Grupo Especial de Operaciones}}队员前去援救安其拉。他们开始在整座公寓里设置炸弹时，遭到受感染的老妇人马努的伏击。之后队员们将他杀掉，还杀了受感染的队友。之后他们听到安其拉求助的尖叫声，告诉她是时候出去了。

醒来时安其拉在一个房间里，吉纳德医生正对她进行测试。他询问她是否记得些事情，她说没有。吉纳德需要血液样本以确保她未受感染。一名小组队员古兹曼找到了一位上了年纪的女士，据透露她是[[死亡录像3|第三部电影的事件]]中的唯一幸存者。古兹曼还遇到了他的同伴卢卡斯。安琪拉尝试逃离实验室成功后，跑向古兹曼。两人发现他们在一艘船上。医生利卡特和士兵们告诉他们情况正得到控制，安其拉的血液未收到任何感染。

古兹曼介绍了最后一次出航的奥尔特加船长，还有负责通讯的尼克。尼克尝试修复安吉拉摄像机里的录像。那天晚上，停电发生。古兹曼发现安其拉在船的走廊上来回走动。她声称很害怕，古兹曼让她对一个秘密感到疑惑。第二天下午，安其拉遇见了她的超级粉丝尼克。尼克向她展示了录像带，安其拉开始记得崔丝塔娜·梅代鲁斯。尼克还把文件发给实验室让他们看看。

利卡特和他的团队发现他们的实验对象，一隻受感染的猴子，在停电期间被释放了。猴子袭击并感染了船上的厨师。他们的血液污染了船上的食物供给，这让摄入食物的大多数船员受到感染。在厨子身上的血清测试失败后，他们在饭堂留下安琪拉、古兹曼、卢卡斯和老妇女。事实证明卢卡斯从公寓里拿走了血液样本，还知道梵蒂冈负责此事的人士。用菜刀武装自己并杀死两位受感染的人员和一只猴子后，老妇女走散了，卢卡斯志愿前往寻找，而古兹曼和安琪拉去向船长和尼克寻求帮助。

利卡特做什么事都无法阻止感染后放弃，然后决定继续进行“协议”。在这样做之前，他的助手告诉他收到了传来的视频。利卡特目击了崔丝塔娜将一只蠕虫状的寄生虫插入到安其拉的口中，这让他决定迫使安琪拉做手术来提取寄生虫。安其拉恳求称她不知道他在说什么，但在利卡特做切口前，受感染的二副闯入进来，咬了利卡特的助手，让安其拉得以逃脱。利卡特和他的团队之后前往寻找安其拉，但利卡特在厨师被释放并袭击了他的团队后将自己锁了起来，却遭到安其拉的袭击。安琪拉咬了利卡特，让他去检验她的血，证明他是错的。古兹曼、卢卡斯和尼克之后找到武器和重新启动轮船的方式。尼克告诉他们，安其拉还活着，古兹曼肯定会找她。与此同时，卢卡斯和尼克前往锅炉房。当时实验结果呈阴性后，利卡特证实了安其拉的说法。虽然如此，利卡特还是相信安琪拉玩弄了他们，于是去质问安琪拉。但安其拉反问古兹曼是否一直在欺骗他们。据透露古兹曼在援救安琪拉时，寄生虫已进入到他的体内。利卡特之后将船设置为自曝状态后逃跑，而古兹曼一直在底下给安琪拉设陷阱。

在锅炉房被感染的卢卡斯之后被杀害，尼克在一艘救生艇上发现了利卡特。尼克呼叫安琪拉，但利卡特告诉他安琪拉已是求生不得，求死不能了。尼克一气之下把他击倒，拿走救生艇并搜寻安琪拉。后来他发现了她，两人用船的引擎杀死了受感染的猴子。他们在食堂躲避受感染者，但遭到古兹曼的袭击。古兹曼企图将他体内的寄生虫传给安琪拉，但被安其拉用尼克提供的钓竿杀死了他。安琪拉和尼克在一大波受感染的追击下逃离轮船。尼克紧跟着安琪拉跳入大海，逃离轮船的爆炸。在水下，寄生虫获得意识进入到一条鱼的口中。

片尾字幕期间，尼克和安琪拉乘坐出租车回家。

== 制作 ==
2010年5月3日，{{le|恶心血腥網站|Bloody Disgusting}}宣布，“......Filmax将在未来几年制作两部新的《死亡录像》电影《死亡录像3：创世纪》和《死亡录像4：启示录》，起用前两部电影的导演<ref name="Bloody Disgusting Sequel">{{cite web|url=http://www.bloody-disgusting.com/news/20053 |title=''&#39;Rec&#39;'' Directors Return for Not One, But TWO Sequels!!! |publisher=Bloody-disgusting.com |accessdate=2012-01-06}}</ref>。”2012年3月30日《创世纪》上映，该片由系列的联合缔造者豪梅·巴拿盖鲁导演，豪梅与摩奴·迪茨编剧<ref>{{cite web|url=http://www.dreadcentral.com/news/83358/rec-4-apocalypse-character-videos-off-quick-fix/|title=REC 4: Apocalypse Character Videos Offer a Quick Fix|work=Dread Central|accessdate=2015-04-24}}</ref>。卡洛斯·费尔南德斯、胡里奥、费尔南德斯和阿尔贝托·马里尼第四次回归担任执行制片人<ref>{{cite web|url=http://www.dreadcentral.com/news/90361/exclusive-reveal-uk-dvdblu-ray-trailer-rec-apocalypse/|title=Exclusive Reveal: UK DVD/Blu-ray Trailer for [REC] Apocalypse|work=Dread Central|accessdate=2015-04-24}}</ref>。

在接受《疯哥利牙》（{{lang|en|''Fangoria''}}）采访时，导演巴拿盖鲁表示影片推测不会采用世界末日风格，表示，“《死亡录像》系列所有电影的预算都是一样的，所以你不会看到巴塞罗那遍布僵尸的大场面；他们无意做这种事。这必须是一个得到控制和强有力的故事。”《启示录》设置在第二部电影之后的事件，不会以“短片”风格制作。制片预计于2013年启动。在2012年的[[锡切斯电影节|锡切斯电影节]]上，据宣布电影将于2013年10月全球首映。11月下旬，公告预告片发布，曼奴拉·沃拉斯科确认回归。

然而在2013年初，巴拿盖鲁声称电影的制作已被推迟，将于2014年中旬的某个时间首映，但证实曼奴拉·沃拉斯科将主演第四部电影。5月上旬，首支预告片和更新过的首映日起被公布。2013年7月13日，影片于[[大加那利岛|大加那利岛]]<ref>{{cite web |url=http://bloody-disgusting.com/news/3242617/rec-4-apocalypse-started-filming-today/|title=[''REC 4''] ''Apocalypse'' Started Filming Today!!|accessdate=2013-06-13}}</ref>一艘俄罗斯商人的海军舰艇上开机<ref>{{Cite web  | title = Behind The Scenes Of ''&#39;[REC] 4: Apocalypse&#39;'' With Manuela Velasco!  | date =2013-08-07  | publisher = Bloody Disgusting  | url = http://bloody-disgusting.com/news/3247571/behind-the-scenes-of-rec-4-apocalypse-with-manuela-velasco/  | accessdate = 2013-08-10}}</ref>。2014年4月，影片的剧院版预告片发布<ref>{{Cite web  | title = The Apocalypse Commences in a New Trailer for REC 4: Apocalypse  | date = 24 April 2014  | website = Beyond Hollywood  | url = http://www.beyondhollywood.com/the-apocalypse-commences-in-a-new-trailer-for-rec-4-apocalypse/  | accessdate =2014-05-05}}</ref>。

== 发行 ==

影片于2014年12月5日在[[台湾|台湾]]上映，首周末票房78万新台币，位列票房榜第九位<ref>{{cite web|title=《星际穿越》台北重回榜首 《太平轮（上）》居亚 《饥饿游戏3》落至第三|url=http://news.mtime.com/2014/12/12/1535147.html|website=Mtime时光网|accessdate=2015-10-05|date=2014-12-12|language=zh-cn}}</ref>。第二周票房36万新台币，总票房达148万，票房榜排名跌至第十位<ref>{{cite web|title=《法老与众神》台北登顶 《阿呆与阿瓜2》第五 《星际穿越》降至次席|url=http://news.mtime.com/2014/12/19/1535419.html|website=Mtime时光网|accessdate=2015-10-05|date=2014-12-19|language=zh-cn}}</ref>。

影片于2015年1月8日在[[香港|香港]]上映，首周票房144万港币，位列票房榜第七位<ref>{{cite web|title=《星际穿越》香港票房三连冠 "饥饿游戏3"首周居亚 "黄飞鸿"试映上榜|url=http://news.mtime.com/2014/11/28/1534597.html|website=Mtime时光网|accessdate=2015-10-05|language=zh-cn|date=2014-11-28}}</ref>。

== 制作书籍 ==
电影制作书籍《{{lang|es|REC：El Libro Oficial}}》包含几张幕后照片、故事版和剧照。该书还提供了对整个系列的神话的洞悉，解答影片留下的疑团。该书于2014年10月在西班牙发行<ref>{{cite web|url=http://www.planetadelibros.com/rec-el-libro-oficial-libro-164766.html|title=[REC] El libro oficial|date=2014-10-02|work=PlanetadeLibros|accessdate=2015-04-24}}</ref>。
