'''庶路車站'''（{{jpn|j=庶路駅|hg=しょろえき|rm=Shoro eki|f=y}}）是位於[[北海道|北海道]][[白糠郡|白糠郡]][[白糠町|白糠町]]庶路甲區的[[北海道旅客鐵道|北海道旅客鐵道]]（JR北海道）[[根室本線|根室本線]]的[[鐵路車站|鐵路車站]]。車站編號為'''K49'''。電報略號為'''ヨロ'''。

車站名稱來自通過本地的[[庶路川|庶路川]]的阿伊努語的「so-oro/{{lang|ja|ショ・オロ}}（瀑布之場所）」或「so-ri-oro/{{lang|ja|ショ・リ・オロ}}（高瀑布之場所）」。「瀑布之場所」是指位於庶路川上流的眾多瀑布。

== 歷史 ==
* 1901年（[[明治|明治]]34年）7月20日 - 以[[北海道官設鐵道|北海道官設鐵道]]車站開始營業。為一般車站。
* 1905年（明治38年）4月1日 - 被編入[[日本國有鐵道|官設鐵道]]。
* 1971年（[[昭和|昭和]]46年）10月2日 - 廢除貨物及行李處理。車站簡易委託化。
* 1976年（昭和51年）4月13日 - 在鄰近車站內部，特急「大空」3號列車出軌翻側。
* 1987年（昭和62年）4月1日 - [[國鐵分割民營化|國鐵分割民營化]]後，由JR北海道繼承。
* 1992年（[[平成|平成]]4年）4月1日 - 車站簡易委託化，完全無人化。

== 車站構造 ==
* 庶路車站是1面1線、[[側式月台|側式月台]]的地面車站。原本連接側式月台的[[架空道路|架空道路]]，現在變成可以自由通行的道路。
* 為無人車站。
* 月台上設有列車接近指示的設施。

== 車站周邊 ==
車站的南北兩邊都是廣闊的庶路聚落，由於成為釧路市的通勤者都市，增加了不少住宅。
* [[國道38號|國道38號]]
* [[釧路警察署|釧路警察署]]庶路派出所
* 庶路郵政局（併設[[日本郵政|日本郵政]]釧路西分店庶路配送中心）
* 釧路白糠工業村
* 釧路巴士「庶路車站」車站
