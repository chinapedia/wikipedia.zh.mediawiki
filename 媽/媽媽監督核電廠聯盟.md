'''媽媽監督核電廠聯盟'''，常簡稱為'''媽盟'''，是[[台灣|台灣]]一個監督核電廠以及反對[[龍門核電廠|核能四廠]]的團體，於2012年12月開始運作。2013年3月8日，由[[陳藹玲|陳藹玲]]、[[吳淡如|吳淡如]]、[[陶晶瑩|陶晶瑩]]、[[徐璐|徐璐]]、[[李烈|李烈]]、[[方念華|方念華]]、[[葉樹姍|葉樹姍]]、[[丁乃竺|丁乃竺]]、[[胡茵夢|胡茵夢]]、[[殷琪|殷琪]]、[[郭采潔|郭采潔]]和[[陳嘉樺|陳嘉樺]]等人共同成立的團體<ref>{{cite news |author = 林秀姿 |url = http://www.cw.com.tw/article/article.action?id=5046470 |title = 陳藹玲：我要孩子有安全的未來 |publisher = [[天下雜誌|天下雜誌]] |date = 2013-01-12 |accessdate = 2014-05-15}}</ref><ref>{{cite news |url = http://www.appledaily.com.tw/realtimenews/article/new/20130206/165153/ |title = 「媽媽監督核電廠聯盟」將成立　陶晶瑩、郭采潔力挺 |publisher = [[蘋果日報_(台灣)|蘋果日報]] |date = 2013-02-06 |accessdate = 2014-05-15}}</ref><ref name=母親節反核四>{{cite news|author=朱孝貞|title=不一樣的母親節 3萬人簡訊連署關注核能安全|url=http://www.epochtimes.com.tw/n58233/%E4%B8%8D%E4%B8%80%E6%A8%A3%E7%9A%84%E6%AF%8D%E8%A6%AA%E7%AF%80-3%E8%90%AC%E4%BA%BA%E7%B0%A1%E8%A8%8A%E9%80%A3%E7%BD%B2%E9%97%9C%E6%B3%A8%E6%A0%B8%E8%83%BD%E5%AE%89%E5%85%A8.html|accessdate=2014-05-15|newspaper=大紀元時報|date=2013-04-26}}</ref>。

==訴求及官方回應==
該團體之主要訴求為<ref>{{Cite web |url = https://www.facebook.com/momlovestaiwan |title = 媽媽監督核電廠聯盟臉書粉絲專頁 }}</ref>：
# 台電公開所有真實狀況與數據，成立與公民之對話平台。
# 凡有危險疑慮之核電廠皆不應運作，核四廠立刻停止興建。
# 檢討並研擬新能源政策。

對於相關訴求，[[原能會|原能會]]的回應如下<ref>{{cite press release |url = http://www.aec.gov.tw/newsdetail/headline/2834.html |title = 針對「媽媽監督核電廠聯盟」之組成乙事，原能會堅持安全管制與資訊公開 |publisher = [[行政院原子能委員會|行政院原子能委員會]] |date = 2013-01-11 |accessdate = 2014-05-15}}</ref>：
# 國內核電廠運轉與興建安全品質與管制工作，持續對外透明公開。
# 堅守監督立場，嚴格執行龍門電廠（核四廠）建廠品質與安全管制。
# 加強與民眾溝通，提高民眾對政策信心。

==爭議==
===監督非反核===
* 2013年3月31日，媽媽監督核電廠聯盟發起人之一的陳藹玲女士與總統馬英九、核能相關官員座談時澄清：「該聯盟不是廢核團體，也不是反核團體，而是『中立團體』。」<ref>{{cite news |author = 王鼎鈞 |url = http://www.nownews.com/n/2013/03/31/301252 |title = 陳藹玲：媽媽監督核電廠聯盟不是廢核、反核團體 |publisher = [[今日新聞網|今日新聞網]] |date = 2013-03-31 |accessdate = 2014-05-15}}</ref>

* 2013年4月1日，資深媒體人[[楊憲宏|楊憲宏]]發表《陳藹玲欠台灣反核英雄們一個大大的道歉》的文章，認為廢核與反核者應考慮退出<ref>{{cite news |url = http://www.ettoday.net/news/20130403/187843.htm |title = 陳藹玲：媽媽監督核電聯盟非反核團體　楊憲宏：欠道歉 |publisher = [[ETtoday_東森新聞雲|東森新聞雲]] |date = 2013-04-03 |accessdate = 2014-05-15}}</ref>。

===北捷差別待遇台電擁核廣告、媽盟反核四廣告爭議===
2014年初，媽盟在臉書質疑[[臺北捷運公司|臺北捷運公司]]對台電的擁核廣告與媽盟的反核四廣告差別待遇，媽盟、北捷以及居間協調廣告內容的廣告商「合和國際」各執一詞。

2014年2月19日，媽盟於臉書粉絲專頁表示，在[[台北捷運|台北捷運]]張貼的廣告受到北捷壓力被迫去除「廢除核四吧」字樣<ref name="北捷事件-01">{{cite news |author = 李鴻典 |url = http://www.nownews.com/n/2014/02/20/1125510 |title = 只許擁核？北捷遭疑禁登「廢除核四吧！」 |publisher = [[今日新聞網|今日新聞網]] |date = 2014-02-20 |accessdate = 2014-05-15}}</ref><ref>{{cite news |url = http://www.ettoday.net/news/20140221/327584.htm |title = 禁登反核廣告字？北捷：沒有強迫修正 |publisher = [[東森新聞雲|東森新聞雲]] |date = 2014-02-21 |accessdate = 2014-05-15}}</ref>。隔日，[[台北捷運公司|台北捷運公司]]發表聲明稿，稱廣告內容係由委刊者與廣告廠商協調定案，該五個字為廣告廠商在送樣前便自行撤下，北捷並未干涉，且該廣告於2月1日就已張貼{{r|北捷事件-01}}，網路上也開始質疑媽盟刻意捏造假新聞<ref>{{cite news |language =zh-tw | author = | url =http://www.appledaily.com.tw/realtimenews/article/new/20140221/348788/ | title =廢核廣告遭修改　網友質疑自導自演掀筆戰 | publisher =[[台灣蘋果日報|台灣蘋果日報]] | date =2014-02-21 | accessdate =2014-11-30}}</ref>。
