《'''隨著風吹'''》（{{lang-ja|風が吹いている}}）是日本樂團[[生物股長|生物股長]]的第24張[[單曲|單曲]]，於2012年7月18日由[[Epic_Records_Japan_Inc.|Epic Records Japan Inc.]]發售。

== 概要 ==
本作跟前作《[[春之歌|春之歌]]》相隔將近3個月發售，是[[生物股長|生物股長]]在2012年發行的第3張單曲。單曲在[[限定版|初回期間]]附送「生物卡片030」一張。這張單曲延續前作，都是以諧音訂定<!--遠低於市價的-->價格，<!--僅-->售[[日圓|日圓]]610元，與[[2012年夏季奧林匹克運動會|2012年奧運]]主辦地[[倫敦|倫敦]]的日語讀音相近<ref name="WHAT's IN? WEB">{{cite web |url=http://www.musicnet.co.jp/whatsin/news/20674 |script-title=ja:いきものがかり、NHK ロンドン2012放送 テーマソングはバンド史上最長曲 |language=ja |publisher=WHAT's IN? WEB |date=2012 |accessdate=2012-08-03 }}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。封面中，背景為[[東京|東京]]地標[[東京鐵塔|東京鐵塔]]<ref>[https://twitter.com/mizunoyoshiki/status/222244229391458304 水野良樹 (mizunoyoshiki) on Twitter 2012年7月9日のツイート]{{ja icon}}</ref>。

== 歌曲 ==
《隨著風吹》被[[NHK|NHK]]選為[[2012年夏季奧林匹克運動會|2012年奧運]]與[[2012年夏季帕拉林匹克運動會|2012年帕運]]的[[主題曲|主題曲]]。此前NHK遴選的主題曲都成為經典歌曲，例如同是[[神奈川縣|神奈川縣]]出身的前輩[[音樂團體|音樂團體]][[柚子_(組合)|柚子]]在2004年推出的《光榮之橋》（{{lang|ja|[[:ja:栄光の架橋|栄光の架橋]]}}）、2008年[[Mr.Children|Mr.Children]]的《[[GIFT_(Mr.Children單曲)|GIFT]]》等。

此歌曲也為生物股長至今最長的一首<ref name="WHAT's IN? WEB" />。[[音樂錄影帶|PV]]於[[埼玉縣|埼玉縣]][[熊谷市|熊谷市]]熊谷運動文化公園路上競技場拍攝，三人走向[[體育場|體育場]]中央，在中間的綠色草皮上表演。

本曲獲得當年度的[[日本唱片大獎|日本唱片大獎]]優秀作品獎，並在[[第63回NHK紅白歌合戰|第63回NHK紅白歌合戰]]中以紅組壓軸身分演唱。
