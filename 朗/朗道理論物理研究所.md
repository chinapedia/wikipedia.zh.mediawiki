'''朗道理論物理研究所'''（{{lang-ru|'''Институт теоретической физики им. Л. Д. Ландау РАН'''}}），是一個以研究[[理論物理學|理論物理學]]為主的世界著名[[研究中心|研究中心]]，坐落在[[俄羅斯|俄羅斯]]首都[[莫斯科|莫斯科]]附近的一個小鎮[[切爾諾戈洛夫卡|切爾諾戈洛夫卡]]。

== 研究領域 ==
其主要研究領域包括：
* [[數學物理|數學物理]]
* [[計算物理學|計算物理學]]
* [[非線性|非線性]]物理學
* [[凝聚態物理學|凝聚態物理學]]
* [[粒子物理學|粒子物理學]]
* [[原子核物理學|原子核物理學]]
* [[量子場論|量子場論]]

== 歷史 ==
[[File:Landau_Institute_for_Theoretical_Physics,_Moscow,_Russia._Main_entrance_view.JPG|300px]]
[[File:Landau_Institute_for_Theoretical_Physics,_Moscow,_Russia._View_on_the_building_from_Kosygina_street.JPG|300px]]
朗道理論物理研究所命名之是為紀念[[俄國|俄國]]著名[[物理學家|物理學家]][[列夫·達維多維奇·朗道|列夫·達維多維奇·朗道]]。1965年成立之後，即迅速增長至百名[[科學家|科學家]]，成為全球最知名和領先的理論物理研究所之一。

不像許多其他的科學中心，在俄羅斯，朗道理論物理研究所的有能力面對上個世紀​​九十年代的人員出走危機。雖然其中有一半左右的科學家在國外領先的科學中心和大學任職，他們大多保持與朗道理論物理研究所機構的聯繫，以形成了科學的網絡和支持年輕的理論物理學家發展。

== 傑出成員 ==
1965年至1992年，由{{link-en|伊薩克·馬克維奇·哈拉尼科夫|Isaak Markovich Khalatnikov}}領導研究所。之後由{{link-en|弗拉基米爾·扎哈羅夫|Vladimir E. Zakharov}}擔任所長。多數成員皆是[[俄羅斯科學院|俄羅斯科學院]]院士。

眾多傑出的[[科學家|科學家]]、[[物理學家|物理學家]]和[[數學家|數學家]]包括：

* [[謝爾蓋·彼得羅維奇·諾維科夫|謝爾蓋·彼得羅維奇·諾維科夫]]
