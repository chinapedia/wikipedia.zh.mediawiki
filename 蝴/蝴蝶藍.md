== 簡歷 ==
2012年3月，他的完结作品《[[网游之近战法师|网游之近战法师]]》首次在大陆正式出版。

2013年11月，他的网游竞技小说《全职高手》纪念画册出版众筹项目以181%完成度成功<ref>{{Cite web|url = http://www.qingju.com/projects/13|title = 蝴蝶蓝作品《全职高手》纪念画册出版|accessdate = 2013-11-21|deadurl = yes|archiveurl = https://web.archive.org/web/20141129221309/http://www.qingju.com/projects/13|archivedate = 2014-11-29}}</ref>。其后相关周边产品继续引爆市场，成为中国大陆网络文学界周边产品领域的成功首例<ref>{{Cite news|url = http://news.sohu.com/20141202/n406568092.shtml|title = 网络文学商业化十周年风云录|last = |first = |date = 2014-12-02|work = 搜狐网|accessdate = }}</ref>。

2014年8月，他的首部东方玄幻小说《天醒之路》参拍中国首个网络文学作品游戏版权拍卖会，以465万元成交<ref>{{Cite web|url = http://weibo.com/1659820261/BggLHiMNC|title = #国内首个网络文学作品游戏版权拍卖会直播# 第四部拍品为《天醒之路》作者：蝴蝶蓝，起点中文网大神级作家。蝴蝶蓝作品《全职高手》等都深受读者喜爱。起拍价：400万。主持人邀请盛大数据分析专家对作品进行数据分析后竞拍。成交价：465万。现场竞价激烈啊！|accessdate = 2014-08-01}}</ref><ref>{{Cite news|url = http://tech.21cbh.com/2014/8-5/2NMDA0MTVfMTI1Njk2NA.html|title = 哄抢IP：网络文学和游戏步入蜜月期？|last = |first = |date = 2014-08-05|work = 21世纪经济报道|accessdate = |deadurl = yes|archiveurl = https://web.archive.org/web/20150403004120/http://tech.21cbh.com/2014/8-5/2NMDA0MTVfMTI1Njk2NA.html|archivedate = 2015年4月3日|df = }}</ref>。9月24日，《全职高手》盟主<ref>{{Cite web|url = http://wwwploy.qidian.com/HelpCenter/default.aspx?type=0&categoryid=85&parentid=21|title = 作品粉丝值系统_起点中文网|accessdate = }}读者在一部作品得到了100,000粉丝积分（即消费约￥1,000），就能成为这部作品的盟主</ref>数超过一千，创造了网络文学史上的新记录<ref>{{Cite news|url = http://book.163.com/14/0926/15/A730PBH300923M2M.html|title = 起点“周边模式”发功：《全职高手》破千盟|last = |first = |date = 2014-09-26|work = 网易|accessdate = }}</ref><ref>{{Cite web|url = http://wwwploy.qidian.com/ploy/20140930/index.aspx|title = 全职高手千盟庆专题 - 起点中文网|accessdate = 2014-09-30|deadurl = yes|archiveurl = https://web.archive.org/web/20150216084420/http://wwwploy.qidian.com/ploy/20140930/index.aspx|archivedate = 2015年2月16日|df = }}</ref>。12月，蝴蝶蓝以纸质图书版税及游戏改编授权总收入800万登上当年网络作家榜第17位<ref name=":0">{{Cite news|url =http://www.wccdaily.com.cn/shtml/hxdsb/20141217/262585.shtml|title =在集体创富的路上 唐家三少5000万版税领跑|last =|first =|date =2014-12-17|work =华西都市报|accessdate =}}{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>。

2016年9月，《全职高手》入选中国网络小说排行榜半年榜，居已完结作品第4位<ref>{{Cite web|url=http://www.chinawriter.com.cn/n1/2016/0913/c404023-28712895.html|title=2016年中国网络小说排行榜半年榜揭晓|accessdate=2016-11-04|author=|date=2016-09-13|last=|first=|work=|publisher=中国作家网}}</ref>，虚拟网游与现实竞技双线互动的表层叙述结构被视为最大亮点<ref>{{Cite web|url=http://www.chinawriter.com.cn/n1/2016/0913/c404023-28712898.html|title=2016年中国网络小说排行榜半年榜（已完结作品）上榜作品评语|accessdate=2016-11-04|author=|date=2016-09-13|last=|work=|publisher=中国作家网|quote=《全职高手》是一部网络游戏题材作品。虽然文学性不是很强，但能够通过游戏让读者产生极强的身份代入感。作品采用了典型的“英雄归来”模式，主角从高处跌落，蜗居小镇，陷入谷底，后又凤凰涅槃，王者归来，符合读者的审美感受。小说最大亮点在于，网络游戏“打怪升级”的虚拟人生和现实生活竞技拼搏的奋斗之旅，在整个故事叙述中嵌合得非常紧密，构成了故事双线互动的表层叙述结构。}}</ref>。10月，蝴蝶蓝入选微博2016十大影响力文化大V，被誉为“千盟大神”、“现象级”作家<ref>{{Cite web|url=http://ent.chinadaily.com.cn/2016-11/01/content_27241576.htm|title=2016“V影响力峰会”文化大V奖项落定 他与近半数获奖者均有合作|accessdate=2016-11-04|author=|date=2016-11-01|last=舒靓|work=|publisher=中国日报网|quote=第二位 网络文学“千盟大神”，蝴蝶蓝  　　网络文学界首位千盟盟主，是足以让全国粉丝为之疯狂的“现象级”作家。著有《全职高手》《天醒之路》《网游之江湖任务行》等多部高人气作品。其中最具代表性的《全职高手》可谓打破二次元“次元墙”的里程碑级作品，新浪话题阅读量高达23亿，一举打入日本青春图书市场。}}</ref>。
