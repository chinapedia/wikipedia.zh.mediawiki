'''蝴蝶酥'''（[[法语|法语]]：'''Palmier'''；[[中式英文|中式英文]]：'''Butterfly Cracker'''）是一款流行于[[德国|德国]]、[[西班牙|西班牙]]、法国、[[意大利|意大利]]、[[葡萄牙|葡萄牙]]和[[犹太人|犹太人]]之间的經典西式甜點。人们普遍认为是法国在20世纪早期发明了这款甜点，也有观点认为首次烘焙是在[[奥地利|奥地利]]的[[维也纳|维也纳]]，所以这款甜点没有一个确切的起源地。一般认为，蝴蝶酥的发展是基于对果仁蜜饼等类似的[[中东|中东]]甜点烘烤方法的一次改变。<ref>{{cite web | url=http://www.wisegeek.com/what-is-a-palmier.htm| title=What Is a Palmier?| publisher=wisegeek.com| date= 2011-09-23| accessdate=2011-10-02}}</ref>因其外形，在西方又有「棕榈树叶」、「[[象|象]]耳朵」、「眼镜」等形象的说法。<ref>{{cite web | url=http://southendfoodies.blogspot.com/2009/03/elephant-ears-palmiers.html| title=elephant ears palmiers | publisher=southendfoodies| date=2009-03 | accessdate=2011-10-01}}</ref>在德国，这一甜点又被称作「Schweineohren」，意为猪耳朵；因其外形似蝴蝶，在[[汉语|汉语]]中则被称作蝴蝶酥。

==做法与享用==
[[蝴蝶|蝴蝶]]酥有很多種類，其中比較常見的為做的比較小的千層酥皮甜點，蝴蝶酥外觀為淡黃或焦黃有很多層皺摺，有一些外表上有[[白砂糖|白砂糖]]，一般制作过程为把千层酥皮擀薄，切成长方形状，刷上蛋液，撒糖和其他配料，随后两边分别向中间折叠，最中间留个空， 两面叠起并切成小条，最後經由烘箱做出來。<ref>{{cite web| url=http://www.cakeok.cn/knowledge/2008/1120/article_1322.html| title=蝴蝶酥的配方与制作| publisher=好的蛋糕网| date=2008-11-20| accessdate=2011-10-01}}{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>。蝴蝶酥常被用作早餐食用，在[[法國|法國]]可作为正餐吃完後的甜點，脆口香甜[[奶油|奶油]]味濃厚也很適合下午茶，一般搭配[[茶|茶]]或[[咖啡|咖啡]]享用。

在西方国家中，蝴蝶酥可以做出许多不同的[[开胃菜|开胃菜]]品种，用料包括有各种[[奶酪|奶酪]]、[[草药|草药]]和[[香料|香料]]，以及[[大蒜|大蒜]]、[[西红柿|西红柿]]乾、[[蒜|香蒜]]。也可用于其他不同配料的组合。这类蝴蝶酥通常作为开胃菜或配菜，而不是甜点食用。

==流行于中国==
蝴蝶酥在中國的西式面包房中十分常見，[[上海|上海]]是最早接觸西式蝴蝶酥并成功将其本土化的中国城市，现已成為上海的經典海派小點心，其中以[[國際飯店|國際飯店]]西餅屋和有百年歷史的老西餐店[[德大食品專賣店|德大食品]]最有名气，國際飯店西餅屋的蝴蝶酥以酥脆和奶味重而广受欢迎，其蝴蝶酥的年销量超过了一千万元[[人民币|人民币]]。<ref>{{cite web | url=http://video.sina.com.cn/p/finance/20110309/090061276369.html| title=上海传统名点蝴蝶酥年销量飞过千万| publisher=[[中国中央电视台|CCTV-2]]「第一时间」栏目| date=2011-03-09 | accessdate=2011-10-01}}</ref>

蝴蝶酥也是[[北京|老北京]]历史悠久的傳統風味小食，融入了[[中國文化|中國文化]]後非常適合過年時的消遣。其口感松脆香酥，香甜可口，具有浓郁的[[桂花|桂花]]香味。<ref>{{cite web| url=http://www.yxdaily.com/yxnews/cate/dfmc/2006_11_6/97856.shtml| title=北京蝴蝶酥| publisher=玉溪日报| date=2006-11-06| accessdate=2011-10-01}}{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>在中国的其他地方，如成都、天津、哈尔滨、南京、杭州、昆山、合肥等地都有结合当地特色的蝴蝶酥，在一些明显带有传统中国特色的食店中也会有所售卖，<ref>{{cite web| url=http://xmwb.news365.com.cn/hczk/201107/t20110707_3080999.htm| title=蝴蝶酥| publisher=[[新民晚报|新民晚报]] 作者:公输于兰| date=2011-07-07| accessdate=2011-10-01| deadurl=yes| archiveurl=https://web.archive.org/web/20110711213319/http://xmwb.news365.com.cn/hczk/201107/t20110707_3080999.htm| archivedate=2011年7月11日| df=}}</ref>可以说蝴蝶酥是在中国本土化非常成功的一款西点。

==健康问题==
由于蝴蝶酥属于高糖、高热量的起酥类甜点，多食宜引发[[肥胖|肥胖]]、高脂血症、[[心血管疾病|心血管疾病]]。
