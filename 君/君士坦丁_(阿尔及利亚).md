}}'''君士坦丁'''（[[阿拉伯语|阿拉伯语]]：'''{{rtl-lang|ar|قسنطينة}}'''）位于[[阿尔及利亚|阿尔及利亚]]东北部，是[[君士坦丁省|君士坦丁省]]的首府，西距首都[[阿爾及爾|阿爾及爾]]322公里。根据2008年人口普查，它有448,374名居民，是阿尔及利亚第三大城市。它是一座工业城市和交通枢纽，地毯紡織等手工業盛行，附近還有[[非洲|非洲]]最大的瓷器場。君士坦丁有一座大学、一座[[伊斯蘭學校|伊斯蘭學校]]以及不少[[古羅馬|古羅馬]]與中世紀時期的古蹟，如城區周圍的石砌圍牆。

阿尔及利亚民族英雄[[阿布德·卡迪尔|阿布德·卡迪尔]]墓就位于君士坦丁。

== 地理 ==
君士坦丁位于阿尔及利亚东部的一座[[高原|高原]]上，海拔650米。只有使用跨越[[古迈勒河|古迈勒河]]谷的桥梁才能进入城市。

== 历史 ==
[[文件:Expeditionconstantine.jpg|thumb]]
在古代君士坦丁是[[努米底亞|努米底亞]]最富有、最兴旺的城市，拥有非常重要的作用。它的[[迦太基|迦太基]]名称是{{lang|la|Karta}}（城市），[[古罗马|罗马人]]称之为{{lang|la|Cirta}}<ref name="WDL">{{cite web |url = http://www.wdl.org/zh/item/8789/ |title = 阿尔及利亚康斯坦丁全景 |website = [[World_Digital_Library|World Digital Library]] |date = 1899 |accessdate = 2013-09-26 }}</ref>。[[马西尼撒|马西尼撒]]的儿子[[米奇普撒|米奇普撒]]在[[古希腊|希腊殖民者]]的帮助下于前203年把它建造成努米底亞的首都。

它的公共建筑比[[非洲|非洲]]北部所有其它城市都要辉煌，人口比所有其它城市多。前113年[[尤古尔塔|尤古尔塔]]占领君士坦丁。罗马将军[[昆图斯·卡埃基利乌斯·梅特鲁斯·努米底库斯|昆图斯·卡埃基利乌斯·梅特鲁斯·努米底库斯]]和[[盖乌斯·马略|盖乌斯·马略]]用它作为主要据点。前107年马略在君士坦丁击败尤古尔塔。

前46年亲庞贝的[[朱巴一世|朱巴一世]]在非洲失败后[[恺撒|恺撒]]把君士坦丁的部分地区分给他的一名亲信，君士坦丁成为一座罗马殖民地，获得了特权和{{lang|la|Sittlanorum Colonia}}的名称。

从此老城开始衰败。311年[[马克森提乌斯|马克森提乌斯]]与自立为罗马皇帝的[[阿非利加_(羅馬帝國行省)|非洲行省]]总督[[多米提乌斯·亚历山大|多米提乌斯·亚历山大]]作战时老城完全被毁。312年[[君士坦丁一世_(罗马帝国)|君士坦丁一世]]重建该城，并把它命名为{{lang|la|Constantina}}<ref name="WDL"/>。

430年[[汪达尔人|汪达尔人]]占领君士坦丁。从534年至697年它是[[拜占庭帝国|拜占庭帝国]]的一部分。从8至15世纪[[柏柏尔人|柏柏尔人]]和[[阿拉伯人|阿拉伯人]]王朝统治这里。从1529年它受[[奥斯曼帝国|奥斯曼帝国]]统治，土耳其总督的驻地设立在这里。

1837年10月13日[[法国|法国]]军队占领君士坦丁以及今天阿尔及利亚的整个北部。1851年这个地区被并入法国海关区，1865年被政治和经济上被并入法国。从1962年开始君士坦丁是独立的阿尔及利亚的一部分。

== 名胜 ==
君士坦丁最重要的名胜包括博物馆、清真寺、老城、大学、王宫、古罗马水道和君士坦丁一世的塑像。

== 经济和基础结构 ==
君士坦丁是一座工业城市，市内有水泥和金属工业、轻工业和食品工业。它也是一座交通枢纽，拥有一座大学、一座伊斯兰学校和多座技校和图书馆。

阿尔及利亚独立后一些企业被[[国有化|国有化]]，从1980年代初开始经济又开始放松，国家促进私有企业。
