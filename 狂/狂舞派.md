{{engname|《狂舞派》|''The Way We Dance''}}為2013年[[香港電影|香港電影]]，[[黃修平|黃修平]]導演，[[陳心遙|陳心遙]]與王日平監製，由[[顏卓靈|顏卓靈]]、[[蔡瀚億|蔡瀚億]]、[[楊樂文|楊樂文]]、范穎兒及 Tommy "Guns" Ly主演，[[高先電影|高先電影]]投資兼發行，獲[[電影發展基金|電影發展基金]]<ref>[http://www.fdc.gov.hk/tc/projects/film_financing.htm 電影發展基金電影製作融資計劃]</ref>及[[太陽娛樂文化|太陽娛樂文化]]融資，於第37屆[[香港國際電影節|香港國際電影節]]首映，並於2013年8月公映。

== 剧情 ==
成長於祖傳荳品店的花（[[顏卓靈|顏卓靈]]飾），從小熱愛跳舞，是Hip-hop天才少女，但被家人注重學業的傳統思想下埋沒了舞蹈發展的理想。

按家人意願考入讀大學後，花終擺脫了陳舊思想，憑著出色舞技加入大學跳舞學會，成為舞隊「BombA」的超新星，更獲萬人迷會長Dave（[[楊樂文|楊樂文]]飾）垂青。「BombA」是大學學界很受歡迎的舞隊，可是卻屢屢在跳舞比賽「舞林大會」中輸給勁敵「Rooftoppers」。

在苦練創出新舞招對付宿敵「Rooftoppers」之際，花卻遭隊友，曾於中學舞蹈表演中在眾人面前被花奚落的Rebecca（[[范穎兒|范穎兒]]飾）報復。Rebecca不惜橫刀奪愛和挑撥離間，嘲笑花的舞姿，令愛情理想雙失的花受到羞辱後憤然離隊退出「BombA」。

一天，[[太極拳|太極]]學會會長柒良（[[蔡瀚億|蔡瀚億]]飾）無意中看中花的心靈手巧，拉攏她加入太極學會。花原以為太極與良一樣陳舊老套，直至發現良不為人知的過去，知道曾經進入懲教所的阿良被太極改變並感化別人，洗心革面後，花對太極與良完全改觀，二人日久生情擦出愛火花，更受太極啟發自創全新舞招。

此時，Rebecca為[[Cosplay|Cosplay]]理想改名Momoko，離隊競選「動漫寶寶」希望重奪昔日舞台最耀眼目光，並與Dave分手。「BombA」及Dave因八卦周刊的醜聞報導而名譽掃地，隊員士氣低落如一盤散沙，Dave苦苦哀求花回歸舞隊應付即將舉行的舞林大會，花心動希望重返舞林一圓夢想，但卻被暗自呷醋的良無意間弄傷腳部，無法參賽。花感失意絕望，瀕臨放棄理想之時，宿敵「Rooftoppers」的老大Stormy（Tommy "Guns" Ly飾）卻現身花的輪椅面前，當面揭開一個「Rooftoppers」稱霸舞壇的秘密......

== 劇本原型 ==
導演及編劇黃修平與及其拍檔，本片的監製陳心遙，在報章訪問中提到自己是[[香港中文大學|香港中文大學]][[新亞書院|新亞書院]]的畢業生，一個唸[[藝術|藝術]]，一個唸[[人類學|人類學]]。畢業後，兩人並沒有從商，反而堅持從事藝術創作，甘心過著顛沛流離的生活。
導演及編劇黃修平稱他們當時仍然沒有自己的辦公室，需要經常四周飄泊，席地度橋，例如晚上在[[香港理工大學|香港理工大學]]的邵逸夫樓的[[7-Eleven|7-Eleven]]便利店討論電影劇本。

理大學生會因應學生投訴邵逸夫樓（VA）大學書局旁的空地，有太多街舞者動作大、阻塞通道，安裝了兩塊「請勿在此進行跳舞活動」的告示牌，這個舞蹈學會(Dance Soc)禁止在校內練習的情況，令黃修平想起了一個為理想而奮鬥到底的故事劇本。<ref>{{cite web|url=http://hk.apple.nextmedia.com/news/art/20130910/18416297|title=《狂舞派》電影靈感之源理大解禁　准校內跳街舞}}</ref>

== 中國大陸上映 ==

== 票房 ==

導演黃修平則表示，很多時候開拍續集未必比首集好，當中包括很多因素，例如急功近利。追捧熱潮容易走下坡，所以他打算不拍續集，以原班人馬，直接開拍《[[狂舞派3|狂舞派3]]》<ref>{{cite web|url=http://hk.apple.nextmedia.com/entertainment/art/20130905/18409261|title=《激戰》《狂舞派》爆出　鬥快開續集|publisher=[[蘋果日報_(香港)|蘋果日報 (香港)]]|date=2013年9月5日}}</ref>。
黃修平表示「因為人是需要熱血，但亦要小心容易變成一些廉價口號，今次開拍《狂舞派3》，會在意作出這方面的反思。」<ref>[https://www.mings-fashion.com/%E7%8B%82%E8%88%9E%E6%B4%BE-%E9%BB%83%E4%BF%AE%E5%B9%B3-%E9%A1%8F%E5%8D%93%E9%9D%88-210770/ 黃修平與他的狂舞派們｜Unexpected Fashion Icons]</ref>

黃修平與監製陳心遙在《狂舞派》公映後，正著手另一電影《[[哪一天我們會飛|哪一天我們會飛]]》(前稱：愛的根源)的準備工作，開拍《[[狂舞派3|狂舞派3]]》至少是2014年下半年以後的事，也許最快要到2015年才能與觀眾見面<ref>{{cite web|url=http://www.hkfilmblog.com/archives/31122|title=《狂舞派》膺福岡國際電影節觀賞大獎|publisher=講。鏟。片|date=2013年9月19日}}</ref>。
