'''盆地'''（[[英语|英语]]：'''Basin'''），[[地貌学|地貌学]]的盆地的特徵为盘地四周地形的水平高度要比盆地自身高，在中间形成一个低地，常為一地形（[[平原|平原]]、[[高原|高原]]）被[[山|山]]所圍繞也是盆地，因此是盆地是[[地形|地形]]分支的一种。在[[云南省|云]][[贵州省|贵]][[四川省|川]]等地的山间盆地也叫'''坝子'''。

依形成原因可區分為以下4種：

* 侵蝕盆地
:: [[地質|地質]]不同，產生差異侵蝕所造成。
* 褶曲盆地
:: [[褶曲|褶曲作用]]所造成。
* 斷層盆地
:: [[斷層|斷層作用]]所造成。
* [[沉積盆地|沉積盆地]]
:: 來自[[海洋|海洋]]及[[大陸|大陸]][[板塊|板塊]]擠壓的交界。[[地质学|地质学]]意义上的盆地是长期稳定沉降的区域。因此，地理学或地貌学认为[[鄂尔多斯高原|鄂尔多斯]]是高原，地质学与石油天然气勘探认为[[鄂尔多斯盆地|鄂尔多斯]]是盆地。
