'''勾菊石亞目'''（[[學名|學名]]：Ancyloceratina）是一類多樣化的菊石，[[菊石目|菊石目]]下[[弛菊石亞目|弛菊石亞目]]的近親。牠們於[[侏羅紀|侏羅紀]]晚期開始[[演化|演化]]，至[[白堊紀|白堊紀]]期間非常普遍及多樣化。牠們遍佈各[[大洲|大洲]]，可以作為[[指準化石|指準化石]]。大部份勾菊石亞目的殼都不是螺旋狀，故屬於異形菊石。

== 特徵 ==
[[Image:Hoploscaphites_ammonite.jpg|left]][[南達科他州|南達科他州]]發現的[[裝甲船菊石|裝甲船菊石]][[化石|化石]]。]]
較原始的勾菊石亞目的形狀接近完整的螺旋，只是在最末及外端開放，像螺旋下的一個鉤。以這種形狀而言，螺旋的部份就是浮起的部份，而鉤就是住室。例子包括[[鉤菊石|鉤菊石]]、''[[Protanisoceras|Protanisoceras]]''及''{{tsl|en|Tropaeum|Tropaeum}}''。較後期的形態與之有甚大的差別。例如''[[Ptychoceras|Ptychoceras]]''的殼就有三或四個擠在一起及互相連接的筒。[[鉤角石|鉤角石]]的形狀就像是一個[[回形針|回形針]]。

到了[[白堊紀|白堊紀]]晚期，牠們的形狀更為多變，甚至有完全筆直的[[桿菊石|桿菊石]]及''[[Sciponoceras|Sciponoceras]]''、扭轉螺旋狀的[[塔菊石|塔菊石]]、奇形怪狀的[[日本菊石|日本菊石]]等。一些形態是綜合了多個形狀，例如{{tsl|en|Nostoceras|念珠菊石}}，開始是像塔菊石的螺旋形，底下是像鉤菊石較水平的鉤。

並非所有勾菊石亞目的形狀都如此特殊的。很多早期的形態甚至很難於普遍的[[菊石目|菊石目]]分辨出來。一些展開的分支後來也取回螺旋的形狀，最著名的包括[[船菊石|船菊石]]、{{tsl|en|Hoploscaphites|裝甲船菊石}}及其親屬。

== 生態 ==
勾菊石亞目的生態不明，但有一點可以肯定的是展開形態的並不適合游泳。一些擁有開放式殼的物種游動時很容易製造出漩渦，而其殼也是浮在軟體之上，對游泳造成極大的阻礙。所以，牠們很有可能只是浮游的，以長觸手來捕捉細小獵物；又或是底棲的。

== 大小及分佈 ==
勾菊石亞目的大小差異很大：最細小的''Ptychoceras''只有約3厘米長；而[[桿菊石|桿菊石]]及''[[Diplomoceras|Diplomoceras]]''就可以長達1-2米。一些[[物種|物種]]如[[鉤角石|鉤角石]]就廣泛分佈在[[歐亞大陸|歐亞大陸]]、[[南美洲|南美洲]]及[[南極洲|南極洲]]；而另一些就可能只限於[[西部內陸海道|西部內陸海道]]。
