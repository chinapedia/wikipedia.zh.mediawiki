'''池上圳'''為一座位於[[臺灣|臺灣]][[臺東縣|臺東縣]][[池上鄉|池上鄉]]境內的[[灌溉|灌溉]][[水圳|水圳]]系統，其水圳擁有'''大陂圳'''，'''新開園圳'''的舊名，該水圳引水自[[新武呂溪|新武呂溪]]出口接[[卑南溪|卑南溪]]左岸，由[[台東農田水利會|台東農田水利會]][[池上工作站|池上工作站]]進行管理與維護。<ref name="池上鄉公所 池上圳介紹">{{cite web |url=http://www.cs.gov.tw/pagea.php?Act=page5e-1 |title=池上鄉公所 池上圳介紹 | accessdate=2014-12-07}}</ref>

== 沿革 ==

池上圳最初由[[鳳山區|鳳山]]移民[[陳枝和|陳枝和]]等18名和[[恆春鎮|恆春]]移民數名，以及[[原住民|原住民]]族[[阿美族|阿美族]]人於西元1878年（清[[光緒|光緒]]4年）合力開鑿建設，而成為當時池上地區的第一條水圳，並且，當時的池上圳命名為大陂圳，亦稱新開園圳。<ref name="池上鄉公所 池上圳介紹" />

大坡圳引自新武呂溪的溪水，圳道呈西北走向，長約10里，始於[[新開園|新開園]]，尾於[[大坡|大坡]]，灌田二百餘甲，為[[台灣清治時期|清治時期]]，台東地區灌溉面積最廣的最大圳道，亦是今日池上圳的主幹道。<ref name="池上鄉公所 池上圳介紹" />

而後臺灣進入[[台灣日治時期|日治時期]]，當時[[台灣總督府|台灣總督府]]又分別在西元1906年（明治39年）新設了[[萬安圳|萬安圳]]以及西元1908年（明治41年）開鑿[[大坡圳|大坡圳]]，依舊不敷使用。 <ref name="池上鄉公所 池上圳介紹" />

西元1921年（大正10年），當地墾民為擴大灌溉範圍，以新開園圳為一主體，向臺灣總督府申請成立[[新開園圳|新開園圳]][[公共埤圳組合|公共埤圳組合]]，而後地方人士[[林福|林福]]、[[王明經|王明經]]等181人加入，並向[[勸業銀行|勸業銀行]][[貸款|貸款]]進行舊圳的補強、設新圳道[[盛土圳|盛土圳]]（即為浮圳）以及延長舊圳道等工程。<ref name="池上鄉公所 池上圳介紹" />

今日現代化的池上圳，完成於西元1993年（民國82年），水圳進水口進入圳道交接處，設有電腦配置進行水流監控。<ref name="池上鄉公所 池上圳介紹" />

== 設施 ==

=== 進水口 ===

池上圳進水口共有3個。其中最老的進水口興建於[[台灣日治時期|日治時期]]，進水口位置自[[池上大橋|池上大橋]]下，新武呂溪往上游約5.84公里距離處之北側溪岸，懸崖下端，遺有池上圳[[鋼筋|鋼筋]][[水泥|水泥]]建造的圳口，圳口建築依然完好，上面的「池上圳」「大正十年三月」之圳名與建造年代刻字目前依然保存完好。<ref name="生態的池上圳進水口水利公園">{{cite web |url=http://blog.xuite.net/yan9436/blog/128879083-%E7%94%9F%E6%85%8B%E7%9A%84%E6%B1%A0%E4%B8%8A%E5%9C%B3%E9%80%B2%E6%B0%B4%E5%8F%A3%E6%B0%B4%E5%88%A9%E5%85%AC%E5%9C%92 |title=生態的池上圳進水口水利公園 | accessdate=2011-02-10}}</ref>

進水口大小約有1.2丈寬，1丈高，出水口分兩道口，現已被泥砂堵住。<ref name="生態的池上圳進水口水利公園" />

第二個進水口位於新武呂溪出口，接卑南溪左岸，在池上大橋上游不遠處，設有長53.4公尺的[[通舟形|通舟形]][[魚道|魚道]]、48.1公尺的[[水道式|水道式]][[魚道|魚道]]。<ref name="生態的池上圳進水口水利公園" /><ref>{{cite web |url=http://www.walkingtaiwan.org/content/county/city_poi.asp?bid=8&jid=81 |title=走讀台灣 池上鄉：池上水利公園 |accessdate=2014-12-07 }}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

第三個進水口設於西元2005年（民國94年），位於第二座進水口上游450公尺處，設有[[魚骨形|魚骨形]][[魚道|魚道]]。<ref name="生態的池上圳進水口水利公園" />

=== 沉砂池 ===

沉沙池於西元1989年（民國78年），設置在第二進水口旁。<ref>{{cite web |url=http://www.ttia.gov.tw/page4_cs.php |title=台東農田水利會 池上工作站 | accessdate=2014-12-07}}</ref>

=== 水利公園 ===

水利公園於2005年（民國94年）進行規劃與建設，其中包含管理道路整治、庭園景觀、涼亭設置及植栽花草樹木，並有[[荷花|荷花]]池及仿古[[水車|水車]]搭配渠道設施，配合池上鄉環鄉[[自行車|自行車]][[步道|步道]]的設立，在經過灌排渠道段落施設休憩區。<ref>{{cite web |url=http://www.ttia.gov.tw/water_page_cs.php |title=台東農田水利會 池上圳水利公園 | accessdate=2014-12-07}}</ref>

=== 導水設施 ===

主幹線總長約5.5公里 及10條支線、29條幹給水路。

== 池上浮圳 ==
'''池上浮圳（池上圳第六支圳、盛土圳）'''

西元1921年（日治大正10年），池上平原墾民以原有新開園圳為主，向臺灣總督府申請成立新開園圳公共埤圳組合。地方人士林福、王明經等人加入，向勸業銀行貨款進行舊圳補強、新設圳道等工程。其中新設盛土圳，俗成浮圳。浮圳起點為新興村，終點靠近錦園村，與池上大圳相連。由於圳溝位於土提上，高過農田，稱為浮圳。<ref name="文化部 文化資產局 池上浮圳（池上圳第六支圳、盛土圳）">{{cite web |url=http://www.boch.gov.tw/boch/frontsite/cultureassets/caseBasicInfoAction.do?method=doViewCaseBasicInfo&iscancel=true&caseId=VA09602000984&version=1&assetsClassifyId=1.2&menuId=302 |title=文化部 文化資產局 池上浮圳（池上圳第六支圳、盛土圳） |accessdate=2014-12-07 |deadurl=yes |archiveurl=https://web.archive.org/web/20141209220600/http://www.boch.gov.tw/boch/frontsite/cultureassets/caseBasicInfoAction.do?method=doViewCaseBasicInfo&iscancel=true&caseId=VA09602000984&version=1&assetsClassifyId=1.2&menuId=302 |archivedate=2014-12-09 }}</ref>

浮圳原為土溝，日治末期改成砌石混凝土圳溝。1963年（民國52年），臺東農田水利會改建圳道為RC造。浮圳全長1,195.5公尺，最寬處約10公尺，最高處近6公尺，目前浮圳中段建有木造休憩涼亭。<ref name="文化部 文化資產局 池上浮圳（池上圳第六支圳、盛土圳）" />
