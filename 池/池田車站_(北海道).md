'''池田車站'''（{{jpn|j=池田駅|hg=いけだえき|rm=Ikeda eki|f=y}}）是位於[[北海道|北海道]][[中川郡_(十勝支廳)|中川郡]][[池田町_(北海道)|池田町]]字東一条的[[北海道旅客鐵道|北海道旅客鐵道]]（JR北海道）[[根室本線|根室本線]]的[[鐵路車站|鐵路車站]]。車站編號為'''K36'''。電報略號為'''イケ'''。包括特急「[[超級大空號列車|超級大空]]」在內，全部列車都會停靠此車站。

車站名稱是來自附近的池田農場。

== 歷史 ==
* 1904年（[[明治|明治]]37年）12月15日 - 以[[北海道官設鐵道|北海道官設鐵道]]利別～浦幌之間的車站開始營業。一般車站。
* 1905年（明治38年）4月1日 - 被編入[[日本國有鐵道|國有鐵道]]。
* 1910年（明治43年）9月22日 - 位於網走線的池田車站（後來的[[網走本線|網走本線]]、[[北海道池北高原鐵道故鄉銀河線|池北線]]）開始營業。
* 1984年（[[昭和|昭和]]59年）2月1日 - 廢除貨物處理。同時廢除機車庫並將車輛轉移至釧路車輛管理處。
* 1985年（昭和60年）3月14日 - 廢除行李處理。
* 1987年（昭和62年）4月1日 - [[國鐵分割民營化|國鐵分割民營化]]後，由JR北海道繼承。
* 1989年（[[平成|平成]]元年）6月4日 - 廢除池北線，並由[[北海道池北高原鐵道|北海道池北高原鐵道]]繼承。
* 2006年（平成18年）4月21日 - 廢除北海道池北高原鐵道故鄉銀河線。
* 2010年（平成22年）4月20日 - 檢票口及音樂盒的聲效將會使用在池田町出身的[[吉田美和|吉田美和]]（[[DREAMS_COME_TRUE|DREAMS COME TRUE]]）的聲線。

== 車站構造 ==
池田車站為2面3線的地面車站，包括單式（1號線）及[[島式月台|島式月台]]（2、3號線）。上行本線（往[[帶廣車站|帶廣]]、[[瀧川車站|瀧川]]、[[札幌站|札幌]]方向）在1號線，而下行本線（往[[浦幌車站|浦幌]]、[[釧路車站|釧路]]方向）則在2號線行駛，而3號線則上下行列車都可以使用。此外，當特急列車行駛時，將會使用鄰近檢票口的1號線，而兩個方向的普通列車則會使用其他路軌。

3號線往瀧川方向的路軌有切出的軌道，為於2006年（[[平成|平成]]18年）4月被廢除的[[北海道池北高原鐵道故鄉銀河線|故鄉銀河線]]的4號線的痕跡。此外，根室本線直通列車亦主要使用3號線。

由於在故鄉銀河線廢除前，該車站為池北高原鐵道的運行據點，因此設置池田操作分所，並配置職員（配置職員車站）。[[綠窗口|綠窗口]]營業時間為7時00分至20時20分，並設有售票亭。

== 車站便當 ==
以前有2間店舖售賣便當，而現在只有在車站前的1間餐廳售賣。出名的便當有「親子便當」、「十勝紅酒牛排便當」及「香蕉饅頭」。此外，牛排便當必須預約訂購，並提供乘坐的列車資料。之後在該列車附近收取。<ref>JR時間表2010年9月號（交通新聞社出版）717頁</ref>
