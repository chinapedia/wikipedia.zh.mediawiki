'''田子町'''（{{jpn|j=田子町|hg=たっこまち|rm=Takko machi|f=y}}）是[[三戶郡|三戶郡]]西南部，南與[[岩手縣|岩手縣]]、西與[[秋田縣|秋田縣]]接壤的町，亦是青森縣內最南的町。本町大部份地區為山地，而主要居住地區是鄰近熊原川、相米川及大川的範圍。田子町的名字來自[[阿伊努語|阿伊努語]]的tapkop（低山）。由於山岳地帶的緣故，日夜的溫差十分大。此外，本町的[[大蒜|大蒜]]產量為日本第一，因此有「大蒜之町」之稱。

== 歷史 ==
由於出土不少繩文時代的遺跡，因此確定在古時有聚集地。[[鎌倉時代|鎌倉時代]]時，由於奧州合戰的戰績，將糠部五郡（糠部郡、岩手郡、閇伊郡、鹿角郡、津經郡）賜予南部光行，成為南部氏的支配地。並在田子建築支城作為鹿角的緩衝樞紐。在[[江戶時代|江戶時代]]為南部藩尾去澤銅山（今[[鹿角市|鹿角市]]）的野邊地，並發展出蓬勃的貿易路線。

* 1889年（明治22年） - 施行町村制，田子村、相米村合併為'''田子村'''。
* 1928年（昭和3年）11月10日 - 施行町制，改為'''田子町'''。
* 1955年（昭和30年）3月1日 - 田子町與上鄉村合併為'''田子町'''。

== 地理 ==
位於[[青森縣|青森縣]]的最南部，[[岩手縣|岩手縣]]及[[秋田縣|秋田縣]]的邊境。

由於田子町位於八甲田山山系，本町大部份地區為山岳地帶，自然資源亦十分豐富。1992年（平成4年）被環境廳認定為擁有日本第一的星空。

* 山：大黑森（719m）、雷鉢森（874m）、朝日奈岳（720m）、猿平（679m）、黑森（727m）、大倉森（897m）、四角岳（1003m）
* 河川：熊原川、相米川、杉倉川、種子川、鍵掛川

== 經濟 ==
田子町的主要產業為以高地為中心的農業。尤其大蒜的生產量為日本第一，因此集中在町內生產大蒜。

== 觀光 ==
=== 景點、遺跡 ===
* 大黑森
: 在頂峰附近的山肌生長了數萬株麒麟杜鵑。在山頂上亦能觀賞360度的全景。
* 和平高原
: 在1992年（平成4年）稱為耀眼的日本第一星空。亦飼養不少田子牛。
* 彌勒瀑布
: 位於熊原川的源頭。被編制成旅遊路線。
* 田子館址
: 三戶城支城
* 日之澤館址
: 三戶城支城
* 舊橋本家住宅（田子字川代ノ上ミ）
: 縣貴重資產，為19世紀後期的南部作曲家的住宅，已搬遷至タプコプ創遊村。[http://www.pref.aomori.lg.jp/culture/juho_kenzoubutu/38.htm]
* 真清田神社
* 田子神樂
: 青森縣的無形民俗文化財產。南部藩御用神樂。
* タプコプ創遊村
: 對傳統產業及文化的體驗設施，並設立高山滑雪聖地「創遊村229滑雪地」。
* タプコピアンプラザ
: 共同設立圖書館、野外舞台、文化大廳、田子町有線電視的電視藝術家工作室。為適合鎮上的居民所設立的文化發信基地。[http://www.town.takko.aomori.jp/txt/f03.htm]{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}
