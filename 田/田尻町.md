'''田尻町'''（{{jpn|j=田尻町|hg=たじりちょう|rm=Tajiri chō|f=y}}）是位於日本[[大阪府|大阪府]]西南部的行政區劃，臨[[大阪灣|大阪灣]]，[[關西國際機場|關西國際機場]]人工島位於其海岸線外約5公里處。<ref name="Brief">{{Cite web| url=http://www.town.tajiri.osaka.jp/kanko/gaiyou/1292561393665.html| title=歴史・地勢| publisher=田尻村公所| language=ja| accessdate=2018-04-17}}</ref>

關西國際機場人工島位於田尻町及相鄰的[[泉佐野市|泉佐野市]]、[[泉南市|泉南市]]外海，整座人工島以這三個行政區劃在海岸線上的分界點延伸線劃分，分別歸屬這三個行政區劃，因此該人工島的中央約三分之一的區域被劃為本町的轄區，雖然機場北部與本土相連的[[關西國際機場聯絡橋|關西國際機場聯絡橋]]屬於泉佐野市，但位於機場中央的航廈等旅客設施皆位於本町轄區，也因此機場旅客在機場產生的消費成為本町主要的稅收來源。[[廉價航空|廉價航空]][[樂桃航空|樂桃航空]]也因為將總部設於關西國際機場內，而成為田尻町內的企業。

除關西國際機場外，本地主要產業為農業，以[[洋蔥|洋蔥]]及[[茄|茄]]為主要農作。

在2006年位於[[岐阜縣|岐阜縣]]的{{tsl|ja|墨俣町|墨俁町}}被[[市町村合併|併入]][[大垣市|大垣市]]後，曾一度以3.86平方公里的面積成為全日本面積最小的[[町|町]]級行政區劃，不過在同一年10月[[国土地理院|国土地理院]]更新的資料中，由於填海造地的關係，轄內土地面積增加至4.77平方公里，因此全日本面積最小的町級行政區劃變成位於[[愛知縣|愛知縣]]的[[春日町|春日町]]。（目前全日本面積最小的町級為同樣位於大阪府的[[忠岡町|忠岡町]]）

== 歷史 ==

現在田尻町的轄區在令制國時代屬於[[和泉國|和泉國]][[日根郡|日根郡]]，在[[江戶時代|江戶時代]]末期，在此區域內包含{{tsl|ja|吉見 (田尻町)|吉見 (田尻町)|吉見村}}和{{tsl|ja|嘉祥寺 (田尻町)|嘉祥寺 (田尻町)|嘉祥寺村}}兩村，其中位於東北的嘉祥寺村為{{tsl|ja|岸和田藩}}領地，位於西南的吉見村為{{tsl|ja|三上藩}}領地，其中三上藩在1870年將藩廳遷自[[近江國|近江國]]移至吉見村，也因此更名為{{tsl|ja|吉見藩}}。

1871年實施[[廢藩置縣|廢藩置縣]]後，這些區域被併入[[堺縣|堺縣]]，1881年再被併入[[大阪府|大阪府]]；1889年實施[[町村制|町村制]]，吉見村和嘉祥寺村合併為'''田尻村'''，1953年改制為現在的'''田尻町'''。<ref name="Brief"/>

== 交通 ==

[[關西國際機場|關西國際機場]]的主要航站設施雖然屬於本町轄區，但由於位於海外的人工島上，必須經由北方[[泉佐野市|泉佐野市]]才能抵達；町內主要市區則有[[南海電氣鐵道|南海電氣鐵道]][[南海本線|南海本線]]通過，並設有{{tsl|ja|吉見ノ里駅|吉見之里車站}}，往北可前往[[大阪市|大阪市]]、[[堺市|堺市]]，往南可至[[和歌山市|和歌山市]]，位於關西國際機場內的[[關西機場車站|關西機場車站]]雖然也是位於本町轄區，但要從町內前往也須先至泉佐野市的[[泉佐野車站|泉佐野車站]]轉乘。

== 觀光資源 ==
[[File:Tajiri_historic_house04s3200.jpg|thumb]]

「田尻歴史館」為20世紀初創立了大阪合同紡績的本地企業家谷口房藏當時在本地的別墅，為紅磚西式建築，現被登錄為大阪府有形文化財，現由田尻町公所管理，作為博物館對外開放。

== 教育 ==

由於面積不大且人口不多，轄內僅有{{tsl|ja|田尻町立小学校|田尻町立小學校}}及{{tsl|ja|田尻町立中学校|田尻町立中學校}}兩間學校；在1990年代由於配合關西國際機場啟用，沿海區域被規劃為填海造地後的新市鎮[[臨空鎮|臨空鎮]]，{{tsl|ja|大阪府警察学校|大阪府警察学校}}也配合計畫於2012年從[[交野市|交野市]]遷至此。
