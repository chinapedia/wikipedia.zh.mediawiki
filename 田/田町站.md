'''田町站'''（{{jpn|j=田町駅|hg=たまちえき|rm=Tamachi eki|f=y}}）是位於[[日本|日本]][[東京都|東京都]][[港區_(東京都)|港區]][[芝_(東京都港區)|芝]]五丁目，屬於[[東日本旅客鐵道|東日本旅客鐵道]]（JR東日本）的[[鐵路車站|鐵路車站]]。

== 停站路線 ==
本站的軌道名稱為[[東海道本線|東海道本線]]（詳情參見路線條目與{{link-ja|鐵道路線的名稱|鉄道路線の名称}}），停靠的運轉系統則有[[電車線與列車線|電車線]]的[[京濱東北線|京濱東北線]]列車與[[山手線|山手線]]列車兩種，旅客資訊完全不提及「東海道（本）線」。依據[[特定都區市內|特定都區市內]]制度，本站屬於「[[特定都區市內#設定區域一覧|東京都區內]]」與「[[東京山手線內|東京山手線內]]」<ref group="広報" name="ticket">{{Cite web |url=https://www.jreast.co.jp/kippu/11041.html |title=JR東日本：きっぷに関するご案内>特定の都区市内駅を発着する場合の特例 |publisher=東日本旅客鉄道 |accessdate=2014-07-22}}</ref>。 - 車站編號「'''JK 22'''」（[[File:JR_JK_line_symbol.svg|15px]] [[京濱東北線|京濱東北線]]）、「'''JY 27'''」（[[File:JR_JY_line_symbol.svg|15px]] [[山手線|山手線]]）。

=== 站名由來 ===
田町是取自於三田口（西口）周邊一帶的舊町名。根據《{{link-ja|文政町方書上|文政町方書上}}》』，該地在[[江戶時代|江戶時代]]從田地轉變成町屋，因而稱為田町。

明治初期冠上[[芝_(東京都)|芝]]稱作「芝田町」（之後1911年5月再省略冠稱「芝」）。當時町域面海且狹長，鐵道沿著海岸線的海上防波堤鋪設。

1909年，芝田町一丁目設置田町站。現在芝浦口（東口）周邊一帶在當時還不是陸地，直至1913年開始填埋後轉變成工業地帶。之後芝浦口周邊設立新芝町（之後的西芝浦一丁目）。

站名來由的田町，在[[港區_(東京都)|港區]]成立的1947年再更名為芝田町。之後隨著{{link-ja|住居表示|住居表示}}實施，1964年7月部分劃為芝五丁目，1967年4月剩餘町域劃為三田三丁目，地名的田町從此消滅。

== 車站構造 ==
本站屬[[島式月台|島式月台]]2面4線[[地面車站|地面車站]]，設有[[跨站式站房|跨站式站房]]<ref group="広報" name="ecomo">{{Cite web |url=http://www.ecomo-rakuraku.jp/stationmap/22807.html |title=らくらくおでかけネット - 駅案内図一覧（田町駅・三田駅） |publisher=公益財団法人交通エコロジー・モビリティ財団 |accessdate=2014-07-22}}</ref>。
京濱東北線與山手線的[[田端站|田端站]]至本站區間，是同方向電車停靠同島式月台的[[複複線|方向別複複線]]配置<ref group="広報" name="ecomo"/>。[[東京站|東京]]方向發出的京濱東北線南向列車（橫濱方向）要與山手線外回列車相互轉乘時，在本站轉乘會比在南邊採線路別複複線配置的[[品川站|品川站]]便利。因此，京濱東北線南向列車與山手線外回列車的車內廣播會在本站播放轉乘通知。

以前月台狹小，使用者多且集中在平日，早晨尖峰時刻月台非常擁擠且危險，加上站內沒有電扶梯與電梯等無障礙設施，因此3、4號線月台進行月台拓寬、階梯增設、大廳拓寬等站內改良工程，並於2004年完工<ref name="SED">{{Cite journal|author = 日野 忠春（JR東日本 東京工事事務所 鉄道設備 課員）、 吉田 和俊（ジェイアール東日本建築設計事務所 構造設備部 部長） |date = 2005-02-01 |title = 田町駅改良と既存建物耐震補強 |journal = SED : Structural engineering data |volume = 23 |publisher = 東日本旅客鉄道構造技術センター |pages = 90-97 }}</ref>。

=== 檢票口 ===
* 本站有北檢票口與南檢票口兩處，但兩檢票口之間有自由通道連結<ref group="広報" name="ecomo"/>。
* 改良工程進行以前，各月台品川側的階梯通道可通往設於三田口側地面的臨時檢票口（出口專用）<ref name="SED"/>。
* 三田口側設有[[綠色窗口|綠色窗口]]與指定席售票機。

== 車站周邊 ==
{{See also|三田 (東京都港區)|芝|芝浦}}
本站設有「三田口（西口）」與「芝浦口（東口）」2個出口<ref group="広報" name="ecomo"/>。

=== 三田口 ===
站前是交通量大的[[第一京濱|第一京濱]]（[[國道15號|國道15號]]），也鄰近與日比谷通的路口與都營三田線、淺草線三田站，交通十分便利。另外，周邊有森永製菓、{{link-ja|森永乳業|森永乳業}}與[[三菱汽車|三菱汽車工業]]、[[日本電氣|日本電氣]]、[[萬代南夢宮控股|萬代南夢宮控股]]等知名大企業本社，[[飛雅特克萊斯勒汽車|日本FCA]]與[[亞培|日本亞培]]、{{link-ja|SBJ銀行|SBJ銀行}}等外資系企業辦公室，以及政府設施等，高層、超高層商業大廈甚多。

三田口周邊有[[慶應義塾大學|慶應義塾大學]]、[[戶板女子短期大學|戶板女子短期大學]]、{{link-ja|普連土學園中學校、高等學校|普連土学園中学校・高等学校}}、{{link-ja|東京女子學園中學校、高等學校|東京女子学園中学校・高等学校}}等多所教育設施，呈現學生街的樣貌，並有多間餐飲店等商業設施。[[都市銀行|都市銀行]]大多數分行都設於三田口側。

另外，與港區其他地區相同，本站鄰近[[波札那|波札那]]、[[匈牙利|匈牙利]]、[[義大利|義大利]]、[[科威特|科威特]]等國的[[駐日本外交機構列表|駐日大使館]]。除了芝稅務署、三田勞動基準監督署、港勤勞福祉會館、障害者福祉會館等政府公共設施外，也可見到許多佛寺、神社、教會等宗教設施。離站稍遠的丘陵地是高級住宅區，近年三田口側附近也開始興建高層高級公寓。

另外，由於站前圓環狹小，除了[[出租汽車|出租汽車]]乘車處，一般車輛難以進入，巴士乘車處也設於第一京濱。

=== 芝浦口 ===
1926年，當時田町站一日上下車人數約4萬人，其中三分之一是是芝浦方向埋海地的工廠勞工，因此鐵道省投入17,000日圓開設芝浦口<ref name="asahi1926" />。

芝浦口側幾乎都是20世紀以後完成的填海地，地勢平坦，多[[運河|運河]]。填海地多作為工業區，工廠、倉庫、辦公大樓眾多，與三田口側展現完全不同的樣貌。除了都營住宅與2000年代的新開發高層公寓以外，居住人口不多。

芝浦口與三田口相比，商業活動較不活躍，但隨著近年的工業區與新填海地的再開發，陸續興建了{{link-ja|芝浦島|芝浦アイランド}}等大型高層公寓，居住人口急增。

== 其他 ==
戰前，國鐵在品川 - 東京間增設[[複線|複線]]時，曾規劃一條行駛京濱東北線部分區間與橫須賀線的「京濱急行線計畫」。該計畫將本站東京側的內側線作為留置線，京濱東北線、山手線行駛外側線，並在田町 - 濱松町間與京濱急行線於內側匯合。該計畫隨著[[太平洋戰爭|太平洋戰爭]]而中止，之後京濱東北線改與山手線分離（『東京縱貫複複線化工程』）。
