'''田園調布站'''（{{jpn|j=田園調布駅|hg=でんえんちょうふえき|rm=Den'en-chōbu eki|f=y}}）是位於[[日本|日本]][[東京都|東京都]][[大田區|大田區]][[田園調布|田園調布]]三丁目，屬於[[東京急行電鐵|東京急行電鐵]]的[[鐵路車站|鐵路車站]]<ref name="eki">{{Cite web |url=http://www.tokyu.co.jp/railway/station/info/?id=8 |title=田園調布駅｜各駅情報 |publisher=東京急行電鉄 |accessdate=2015-01-13}}</ref>。此站為大田區最西端的車站。

== 概要 ==
此站有[[東橫線|東橫線]]與[[目黑線|目黑線]]停靠<ref name="ensen">{{Cite web |url=http://www.tokyu.co.jp/ekitown/info/?id=8 |title=田園調布駅｜沿線・駅周辺ガイド |publisher=東京急行電鉄 |accessdate=2015-01-13}}</ref>。在軌道名稱上，此站是目黑線的正式終點，而東橫線從此站起至[[日吉站_(神奈川縣)|日吉站]]都是[[四線鐵路|複複線]]，不過其中2線是由目黑線的列車使用至日吉站，因此此站－日吉站區間視為兩條不同的路線。東橫線的車站編號是'''TY08'''、目黑線是'''MG08'''<ref name="eki"/>。

[[站長|站長]]所在站。管理田園調布管內此站與[[多摩川站|多摩川站]]以及[[東急多摩川線|東急多摩川線]][[沼部站|沼部站]]。

==車站結構==
車站為[[島式月台|島式月台]]2面4線[[地下車站|地下車站]]<ref name="map">{{Cite web|url=http://www.tokyu.co.jp/railway/station/yardmap/?id=8|title=田園調布駅｜駅構内図|publisher=東京急行電鉄|accessdate=2015-01-13|deadurl=yes|archiveurl=https://web.archive.org/web/20150420151856/http://www.tokyu.co.jp/railway/station/yardmap/?id=8|archivedate=2015-04-20}}</ref>。其中月台外側為東橫線，內側為目黑線<ref name="map"/>。橫濱方向與日吉方向、澀谷方向與目黑方向可在同一月台轉乘<ref name="map"/>。目黑線月台已設置[[月台門|月台門]]<ref name="meguro"/>。

剪票口位於地上，僅有一處，設有人工通道「See through剪票口」<ref name="map"/>。

東急管理的廁所在付費區內<ref name="map"/>。付費區外則有地方自治設置的廁所<ref name="map"/>。

改良工程以前，本站月台為3面4線形態，東側屬於[[目蒲線|目蒲線]]，西側屬於東橫線，站房位於地面<ref name="jtb50"/>。改良工程後改建為[[跨站式站房|跨站式站房]]，月台改為現有配置。

== 特色 ==
車站周邊地形為西高東低。

西側的洋館風舊站房因東橫線複複線化事業一環的田園調布 - 多摩川間改良工程，於1990年（[[平成|平成]]2年）<ref name="hot"/>9月4日停止使用後解體<ref name="news99-41"/>，2000年（平成12年）1月15日復原<ref name="news99-41"/><ref name="hot"/>。復原後的建築已沒有站房機能<ref name="news99-41"/><ref name="hot"/>，入口內側設有電梯<ref name="map"/>。

周辺是寧靜的住宅區，無法夜間施工，因此工程難度很大。此改良工程最初未規劃在田園調布、多摩川兩站設置電梯，但在地方居民的請願下增設，開啟了日本鐵道站的無障礙化浪潮。

[[埼玉高速鐵道線|埼玉高速鐵道線]][[浦和美園站|浦和美園站]]與[[都營地下鐵三田線|都營地下鐵三田線]][[高島平站|高島平站]]發出的臨時列車「港未來號」會從目黑線轉入東橫線。2006年（平成18年）8月以前是在[[武藏小杉站|武藏小杉站]]轉線，同年12月改至日吉站，最後2011年12月24日改從本站轉線。

2013年（平成25年）3月16日起東橫線直通[[東京地下鐵副都心線|東京地下鐵副都心線]]，為了停靠十輛編組的優等列車（急行、特急、通勤特急），本站在地下化時將月台往澀谷方向延伸<ref name="kaishi"/>。

東橫線、目黑線的多摩川站側有往[[東急多摩川線|東急多摩川線]]的連絡線，直通該線的多摩川站5、6號線月台。該連絡線在2000年（平成12年）8月5日以前是由[[目蒲線|目蒲線]]使用。

== 站名由來 ==
{{See also|田園調布}}
設站時的1923年（大正12年），本地地名為[[荏原郡|荏原郡]]'''調布'''村大字上沼部字旭野，因此以村名命名為「調布站」。「田園」則是因東急在此打造[[田園都市|田園都市]]而冠上。之後1889年（明治22年）地名改為荏原郡東調布町，1932年（昭和7年）編入東京市時成為大森區「[[田園調布|田園調布]]」。

「調布」則是源自於古時此地上繳手作的'''布'''作為[[朝廷|朝廷]]的[[租庸調#調|'''調'''稅]]。
