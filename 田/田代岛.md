'''田代島'''（{{jpn|j=田代島|hg=たしろじま|rm=Tashirojima|f=y}}）是[[日本|日本]]的一个岛屿，隸屬[[宫城县|宫城县]][[石卷市|石卷市]]，位於[[牡鹿半岛|牡鹿半岛]]以西。该岛总面积3.14km²，海岸線總長11.5km。島上最高峰為正島山，海拔約96米。岛上人口约70人。该地经济以[[渔业|渔业]]为主。岛内有猫神社。当地居民奉[[猫|猫]]为神明，堪称“'''猫岛'''”<ref>{{cite news|title=日本網友實地走訪《田代島》貓島需要大家來觀光 |url=http://news.gamme.com.tw/669152 |language=zh-tw |publisher=宅宅新聞 |date=2014年2月27日 |accessdate=2015年6月30日 }}</ref><ref>{{cite news |title=喵星人大集結！日本8大貓島 |url=http://www.weekendhk.com/home/%E7%9E%84%E6%98%9F%E4%BA%BA%E5%A4%A7%E9%9B%86%E7%B5%90%E6%97%A5%E6%9C%AC8%E5%A4%A7%E8%B2%93%E5%B3%B6/ |language=zh-hk |publisher=新假期周刊 |date=2015年3月11日 |accessdate=2015年6月30日 |deadurl=yes |archiveurl=https://web.archive.org/web/20150708133313/http://www.weekendhk.com/home/%E7%9E%84%E6%98%9F%E4%BA%BA%E5%A4%A7%E9%9B%86%E7%B5%90%E6%97%A5%E6%9C%AC8%E5%A4%A7%E8%B2%93%E5%B3%B6/ |archivedate=2015年7月8日 }}</ref>。

== 歷史 ==
田代島曾生產絲布。當地人為了治[[鼠|鼠]]，便飼養了貓。隨時代轉變，田代島已經不再生產絲布，但島民依然視貓兒為島上的成員。全島只有70人，卻有幾百隻貓，貓比人多，成為當地一大特色。

== 設施 ==
島上設有[[郵局|郵局]]，但島民甚少到那裡領取[[郵件|郵件]]和包裹。因為島民以老人為主，郵差會直接把郵件和包裹送到府上，郵差甚至會為老人做些雜務，例如水電維修，人情味相當濃厚。

除了郵局，島上還有十多間民宿，為旅客提供食宿和導遊服務。平日的田代島相當寧靜，只見貓兒到處流連。可是到了假日，遊客就會湧到島上。

== 景點 ==
島上有一貓神社，供奉貓神。據說，往日的[[漁民|漁民]]會藉觀察貓的行為來預測[[天氣|天氣]]。某日，有一隻貓被石頭砸死，幸好有善心的漁民把牠好好安葬。想不到的是，那位漁民竟然走大運，連年漁獲豐收。

== 黑傑克 ==
島上有一隻花貓，黑白色，名叫黑傑克。根據[[都市傳奇|都市傳奇]]，只要摸到牠，就會走運一整年。

== 注意事項 ==
田代島又稱貓島，在那裡狗是不受歡迎的，因此除了[[導盲犬|導盲犬]]等工作犬因工作需要外，禁止登島者攜帶狗隻入島。
