《'''湯神君沒有朋友'''》（{{lang-ja|湯神くんには友達がいない}}）是{{tsl|ja|佐倉準|佐倉準}}原作的[[日本|日本]][[漫畫|漫畫]]作品，在2012年7月號到2013年11月號在《{{tsl|ja|週刊少年サンデーS|周刊少年Sunday S}}》上連載，後轉移至《[[週刊少年Sunday|週刊少年Sunday]]》（2013年48號）開始連載。

== 故事簡介 ==
轉校生綿貫千尋座位旁邊的湯神，是學校棒球隊的王牌，但是卻沒有融入棒球隊中，而且在班上更是獨來獨往，湯神就這樣獨自享受著自己一個人的校園生活。

== 主要人物 ==
聲優在「星期天CM劇場」和「落語ドラマCD版」是一樣的<ref>{{Cite web|url=http://natalie.mu/comic/news/138868|title=「湯神くんには友達がいない」6巻、内山昂輝ら出演のドラマCD付き特別版も|website=[[natalie|natalie]]|accessdate=2017年2月9日|date=2015年2月18日}}</ref>。
; {{nihongo|湯神 裕二|ゆがみ ゆうじ|聲：[[内山昂輝|内山昂輝]]}}
:本作主角。[[左撇子|左撇子]]。2年A組。總是一個人。棒球隊的王牌兼隊長。課業優秀。被周圍的人視為麻煩人物。常會說出很有道理的話。不會欠別人人情。非常喜歡[[落語|落語]]以及落語家燦燦亭平樂的大粉絲。
; {{nihongo|綿貫 千尋|わたぬき ちひろ|聲：[[米澤円|米澤円]]}}
:以前因爸爸的關係常常轉學，漸漸地不再觸碰人群。很想交朋友。認為裕二是個麻煩人物。非常擅長撈金魚和戳羊毛氈。也會聽落語。手藝部部長。
; {{nihongo|門田 春樹|かどた はるき|聲：[[阿部敦|阿部敦]]}}
:一年級。棒球部。與湯神在少年時期就認識。裕二的專用捕手。常常幫湯神跑腿。認為裕二很麻煩。
; {{nihongo|久住 若奈|くずみ わかな}}
:棒球部經理，常常滿腦子棒球部的事。與裕二從一年級就同班，但沒說過幾句話(因為太麻煩)。跟香織從小認識。跟綿貫是非常好的朋友。
; {{nihongo|藤沢 梨緒|ふじさわ りお}}
:2年D組。因一年級時裕二出手幫忙她完成體育祭的美工作業而喜歡上裕二，但知曉裕二的性格後，感到非常後悔。非常熱愛棒球，還指出棒球隊內部問題。跟綿貫是很好的朋友。
; {{nihongo|百瀬 香織|ももせ かおり}}
:田徑隊。佔有慾極其嚴重。原本討厭綿貫，因她與自己最討厭的裕二說話，經過畢業旅行的相處後，向綿貫道歉並且和好。相當敵視湯神到讓綿貫誤以為是喜歡，認為裕二是讓久住煩惱的人。
; {{nihongo|八重樫 良太|やえがし りょうた}}
:2年A組。歴史研究部部長。
:身體相當虛弱，不太適合喝咖啡。非常崇拜裕二。
; {{nihongo|渡邊 大翔|わたなべ ひろと}}
:2年A組的班長。說話輕浮，不負責任。常把自己說得很厲害。
; {{nihongo|城戸|きど}}
:與門田同班也是棒球隊的下任經理。唯一不覺得裕二麻煩的人。
; {{nihongo|林山 真咲|はやしやま まさき}}
:天城高中棒球隊的王牌。與裕二在以前就認識。一直把裕二當成對手，但裕二已經把他忘了。喜歡綿貫，曾大聲對她告白。
