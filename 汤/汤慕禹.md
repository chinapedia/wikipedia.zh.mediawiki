'''汤慕禹'''{{BD|1904年||1932年|10月}}，又名'''汤茂如'''。[[四川省|四川省]][[巴縣_(重慶)|巴县]]木洞双河乡人，[[黄埔军校|黄埔军校]]第四期毕业。

== 生平 ==

1925年，汤慕禹考入黄埔军校第四期。毕业后，参加[[國民革命軍北伐|國民革命軍北伐]]，在军中加入中国共产党，同年参加[[南昌起义|南昌起义]]<ref>{{Cite web |url=http://www.cqbnda.com/E_ReadNews.asp?NewsID=752 |title=巴南历史名人 |accessdate=2015-09-21 |author= |date=2012-06-25 |work= |publisher=巴南区档案方志信息网　 |language=Zh-hans |deadurl=yes |archiveurl=https://web.archive.org/web/20160307123707/http://www.cqbnda.com/E_ReadNews.asp?NewsID=752 |archivedate=2016-03-07 }}</ref>。后赴[[苏联|苏联]][[高级步兵学校|高级步兵学校]]学习。1930年回国，到洪湖根据地任[[中国工农红军第二军团|中国工农红军第二军团]]参谋长，11月任[[红六军|红六军]]军长。1931年3月任[[红三军|红三军]]教导团团长，中共鄂西临时分特委执行委员。后任红二十五师师长，兼独立第三团团长<ref>{{cite book|editor=中国人民政治协商会议四川省巴县委员会文史资料委员会编|title=巴县文史资料 第8辑|pages=51-52|accessdate=2015-09-05}}</ref>。1932年，先后任红军军事政治学校第二分校总队队长，[[红三军|红三军]]九师二十五团团长、师参谋长，红八师师长等职<ref>{{cite book|author1=《中国人民解放军历史辞典》编委会编|title=中国人民解放军历史辞典|date=1990年5月|publisher=北京：军事科学出版社|isbn=7-80021-189-4|pages=243|accessdate=2015-09-05}}</ref>。同年10月在[[湖北|湖北]][[荆门|荆门]]半边街战斗中阵亡<ref>{{cite book|author1=廖盖隆|title=中国共产党历史大辞典 总论 人物|date=2001年6月|publisher=北京：中共中央党校出版社|isbn=7-5035-2234-8|pages=207|accessdate=2015-09-05}}</ref>。

== 其他 ==
2009年，电视剧《[[人间正道是沧桑|人间正道是沧桑]]》播出后，汤慕禹家属因电视剧中一国军高级将领也叫汤慕禹，而于2009年9月将制作、播放该剧的江苏省广播电视总台、中央电视台以及出版同名图书的江苏文艺出版社等4家单位告上法庭，索赔百万元。后原被告双方达成庭外和解，被告方补偿3万元后，原告方在法院撤诉。而“汤慕禹”也在字幕中被更换为“汤沐雨”<ref>{{cite news|author1=刘璐璐|title=烈士汤慕禹后人状告《人间正道是沧桑》侵权索赔百万|url=http://www.chinacourt.org/article/detail/2009/09/id/373800.shtml|accessdate=2015-09-05|work=中国法院网}}</ref>。
