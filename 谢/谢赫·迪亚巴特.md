'''謝赫·蒂安尼·迪亞比堤'''（{{lang-fr|'''Cheick Tidiane Diabaté'''}}，{{bd|1988年|4月25日|catidx=Diabaté, Cheick}}）是一名[[馬里|馬里]][[足球|足球]][[運動員|運動員]]，司職[[前鋒_(足球)|前鋒]]，現時被[[土耳其足球超级联赛|土耳其足球超级联赛]]球會[[奧斯曼利斯普足球會|奧斯曼利斯普]]外借至[[意甲|意甲]][[贝内文托足球俱乐部|班尼雲度]]。

==生平==
===球會===
;波爾多
迪亞比堤於2007/08年球季在[[波尔多足球俱乐部|波爾多]]預備隊有出色的表現，上陣35場射入18球，翌季獲提升上一隊，但由於隊內有更多富有經驗的球員，時任領隊[[羅倫特·白蘭斯|羅倫特·白蘭斯]]將他外借到位於[[科西嘉岛|科西嘉岛]]上的[[法乙|法乙]]的球會[[阿雅克肖足球俱乐部|阿些斯奧]]汲取上陣經驗。

===國家隊===
迪亞比堤最初代表馬里U17參賽2005年非洲U17錦標賽，於每場分組賽各射入1球，但球隊未能晉身淘汰賽階段<ref>{{cite web |url=http://www.rsssf.com/tablesa/afr-u17-05.html |title=African U-17 Championship 2005 |author=Díaz Rubio, Julian; Kalumiana, Kalumiana |publisher=RSSSF |accessdate=24 January 2012 }}</ref>。賽事結束後不足1個月，迪亞比堤已首次取得[[馬里國家足球隊|成年國家隊]]的上陣機會，於[[2006年世界盃外圍賽_(非洲區)|世界盃外圍賽]]4-1擊敗[[利比里亞國家足球隊|利比里亞]]中85分鐘後補處子登場<ref>{{cite web |url=http://www.fifa.com/worldcup/archive/germany2006/preliminaries/preliminary=7381/matches/match=35805/report.html |title=Match Report: Mali - Liberia |publisher=[[FIFA|''FIFA.com'']] |accessdate=2012-01-24 }}</ref>。迪亞比堤是馬里奪得[[2012年非洲國家盃|2012年非洲國家盃]]季軍的功臣之一，他以3個入球成為賽事的神射手之一，包括2012年2月11日的季軍戰對[[迦納國家足球隊|加納]]包辦兩個入球<ref>{{cite web |url=http://www.monitor.co.ug/Sports/Soccer/-/690266/1325676/-/n9l6uh/-/ |title=Mali beat Ghana to finish third|author=Agencies |publisher=monitor.co.ug|accessdate=13 February 2012 }}</ref>。
