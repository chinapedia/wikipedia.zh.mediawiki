'''谢尔·韦斯特'''（{{lang-sv|'''Kjell Westö'''}}，1961年8月6日-）是[[芬兰瑞典族|芬兰瑞典族]]作家、小说家。曾使用笔名安德尔斯·赫德。使用[[瑞典语|瑞典语]]写作。<ref>http://www.randomhouse.de/author/author.jsp?per=95095 Portrait von Kjell Westö</ref>

== 生平介绍 ==
1961年8月6日，韦斯特出生于[[芬兰|芬兰]]首都[[赫尔辛基|赫尔辛基]]的一个瑞典族家庭，他的弟弟是作家[[莫尔腾·韦斯特|莫尔腾·韦斯特]]（Mårten Westö）。1986年发表处女作《Tango orange》。2006年，韦斯特凭借其小说《[[我们曾去往何方|我们曾去往何方]]》赢得当年的[[芬兰文学奖|芬兰文学奖]]。他曾多次获得国际[[都柏林文学奖|都柏林文学奖]]的提名。2014年凭借长篇小说《幻影38》获得[[北欧理事会文学奖|北欧理事会文学奖]]。截至2014年，韦斯特已出版了十四部作品。<ref>http://www.norden.org/da/nordisk-raad/nordisk-raads-priser/nordisk-raads-litteraturpris norden.org, Kjell Westö vinder Nordisk Råds Litteraturpris</ref>

== 作品主题 ==
韦斯特的小说主要描写生活在芬兰的瑞典族与芬兰族之间因生活习惯等方面的文化差异而产生的矛盾。<ref>{{cite web |url=http://www.impacdublinaward.ie/2007/Titles/Westo.htm |title=存档副本 |accessdate=2006-12-07 |date=2007-09-28 |deadurl=yes |archiveurl=https://web.archive.org/web/20070928063325/http://www.impacdublinaward.ie/2007/Titles/Westo.htm |archivedate=2007-09-28 }} Westö on the International IMPAC Dublin Literary Award site</ref>
