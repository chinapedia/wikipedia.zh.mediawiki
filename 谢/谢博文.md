'''谢博文'''（{{lang-en|'''Pok Man Tse'''}}，{{bd|1931年|10月1日|2015年|9月3日}}），[[中国大陆|中国大陆]]话剧表演艺术家，早年从事话剧表演和影视演出，晚年从事影视教育。1947年加入解放军任[[文艺兵|文艺兵]]，1949年加入中国煤矿文工团，任一级演员，从事[[话剧|话剧]]表演和几百部的电影电视剧的演出，并多次与西方影视机构合作，出演[[古今大战秦俑情|古今大战秦俑情]]、[[末代皇帝|末代皇帝]]、[[宋家皇朝|宋家皇朝]]、[[鸦片战争|鸦片战争]]、[[汉武帝|汉武帝]]、[[十岁的天空|十岁的天空]]。1989年与香港合作合拍摄經典電影《[[古今大战秦俑情|古今大战秦俑情]]》<ref>{{Cite web |url = http://tech.qq.com/a/20060404/000390.htm |title = 张艺谋巩俐倾情演绎《古今大战秦俑情》 }}</ref> ，在影中飾演[[徐福|徐福]]<ref>{{Cite web |http://www.dvd-vov.com/DVD/581375079.html|title=古今大戰秦俑情(電影版) }}</ref>/导游。2015年9月3日观看[[抗战胜利70周年阅兵|抗战胜利70周年阅兵]]心情激动引发[[心肌梗塞|心肌梗塞]]离世，享年85歲。
