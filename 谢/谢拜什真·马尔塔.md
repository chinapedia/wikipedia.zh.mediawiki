'''谢拜什真·马尔塔'''（{{lang-hu|'''Sebestyén Márta'''}}，{{bd|1957年|8月19日|||Sebestyén Márta}}）是出身於[[匈牙利|匈牙利]][[布達佩斯|布達佩斯]]的女性民謠歌手。

== 簡介 ==
瑪塔出身於父親為經濟學作家及母親為作曲家的背景，其中她母親有在音樂家[[佐爾坦·高大宜|佐爾坦·高大宜]]下受訓。而瑪塔的父親在她小時曾前往美國擔任客座教授，之後返回匈牙利時有從[[史密森尼學會|史密森尼學會]]帶來大量民族音樂方面的資料作品給瑪塔欣賞學習。

就讀語文學校後瑪塔開始與民謠樂團「{{lang|hu|Muzsikás}}」（音樂家樂團）展開長達二十五年以上的合作關係，<ref name="UNESCO">{{Cite web |url=http://www.unesco.org/new/en/unesco/about-us/who-we-are/goodwill-ambassadors/artists-for-peace/marta-sebestyen/ |title=Márta Sebestyén |work=[[聯合國教科文組織|聯合國教科文組織]] |accessdate=2014年2月24日}}</ref>

於樂團中瑪塔是負責擔任女主唱及笛樂演奏的角色。

後續瑪塔曾在1996年發行的電影《[[英倫情人|英倫情人]]》中擔任歌曲的演唱。<ref name="UNESCO"/>而法國音樂二人組Deep Forest則有在專輯《Boheme》中引用過瑪塔演唱的歌曲；<ref>{{Cite web |url=http://www.deepforest.co/df02.htm |title=Discography At A Glance |work=Deep Forest |accessdate=2014年2月24日 |deadurl=yes |archiveurl=https://web.archive.org/web/20141109202844/http://www.deepforest.co/df02.htm |archivedate=2014年11月9日 }}</ref>另日本[[吉卜力工作室|吉卜力工作室]]推出的動畫電影《[[兒時的點點滴滴|兒時的點點滴滴]]》裡，也採用了三首瑪塔演唱的作品為插曲。<ref>{{cite book |title={{lang|ja|おもひでぽろぽろ ロマンアルバム}} |date=2002年11月22日 |publisher=[[德間書店|德間書店]] |isbn=978-419-720159-4 |page=101頁 |accessdate=2014年2月24日 }}</ref>

在2010年6月1日[[聯合國教科文組織|聯合國教科文組織]]則肯定瑪塔在民謠樂界的貢獻，將她評列為該年度的和平藝術家。<ref name="UNESCO"/>
