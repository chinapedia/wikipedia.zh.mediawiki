'''谢添'''（{{bd|1914年|6月18日|2003年|12月13日|catIdx=X谢}}），原名'''谢洪坤'''，曾用名'''谢俊'''，[[中国|中国]]著名演员、导演。[[籍贯|籍贯]][[广东|广东]][[番禺|番禺]]，生于[[天津|天津]]，父亲是铁路员工。早年就读于天津英文商务专修中学。1933年在天津开始业余话剧演出。

== 演艺生涯 ==
1936年参演《[[马路天使|马路天使]]》。1949年之后入[[北京电影制片厂|北京电影制片厂]]。1959年他曾主演电影《[[林家铺子|林家铺子]]》等多部影片，也经常饰演反面人物。1959年开始做导演，导演了《[[水上春秋|水上春秋]]》等。1980年，在第三届“[[百花奖|百花奖]]”评选中，谢添因《[[甜蜜的事业|甜蜜的事业]]》而荣获“最佳导演”奖。1981年，在第四届“百花奖”评选中，他导演的《[[七品芝麻官|七品芝麻官]]》荣获“最佳戏曲片”奖。1982年，谢添成功将[[老舍|老舍]]的名著《[[茶馆_(话剧)|茶馆]]》搬上了银幕，获1983年第三届中国电影[[金鸡奖|金鸡奖]]特别奖和文化部1982年优秀影片特别奖。1989年因导演的电视连续剧《[[那五|那五]]》获中国电影制片厂优秀电视剧一等奖。

== 家庭 ==
夫人杨雪明。
共有子女四人：
*[[谢绵绵|谢绵绵]]，长女，毕业于[[北京外国语大学|北京外国语大学]]，曾在[[北京航空航天大学|北京航空航天大学]]任外语教师，后留学[[加拿大|加拿大]]，现任[[加拿大爱德蒙顿孔子学院|加拿大爱德蒙顿孔子学院]]文化部主任。 
*[[谢钢|谢钢]]，长子，1953年1月出生，[[北京电影制片厂|北京电影制片厂]]演员、导演。
*谢联，次女，[[总政话剧团|总政话剧团]]一级演员。
*谢平，次子，[[中影集团|中影集团]]摄像师。
