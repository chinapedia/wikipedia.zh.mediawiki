'''谢旭人'''（{{bd|1947年|10月|catIdx=X谢}}），[[浙江|浙江]][[宁波|宁波]]人。1980年7月加入[[中国共产党|中国共产党]]。大专学历。[[中国共产党第十六届中央委员会候补委员列表|第十六届中共中央候补委员]]，[[中國共產黨第十七屆中央委員會委員列表|中共十七屆中央委員會委員]]。曾任[[中华人民共和国财政部|中华人民共和国财政部]]部长、[[全国社会保障基金理事会|全国社会保障基金理事会]]理事长。

== 简历 ==
曾长期在家乡任职；从底层做起。1967年，进入[[宁波|宁波]]镇海机械厂，逐渐由设备科副科长被提拔为副厂长。1981年进入[[浙江大学|浙江大学]]学习。1981年9月至1984年1月在[[浙江大学|浙江大学]]工业经济管理专业干部专修科学习。毕业后，即步入政坛，历任[[浙江省|浙江省]][[余姚县|余姚县]]副县长，[[中共|中共]][[鄞县|鄞县]]县委副书记、县长等职。1985年9月升任[[浙江省|浙江省]]计划经济委员会党组成员、计委副主任，兼浙江省经济信息中心主任等职。

1990年5月，43岁的谢旭人从[[浙江|浙江]]上调中央，出任[[中华人民共和国财政部|中华人民共和国财政部]]综合计划司副司长。此后的八年间，他先后担任预算司副司长，综合计划司司长，综合与改革司司长，直至部长助理和财政部副部长。1998年，谢旭人出任[[中国农业发展银行|中国农业发展银行]]行长；2000年任[[中央金融工委|中央金融工委]]副书记；2001年11月出任[[国家经贸委|国家经贸委]]副主任<ref>[http://news.xinhuanet.com/ziliao/2003-04/02/content_810860.htm 谢旭人简历] {{webarchive|url=https://web.archive.org/web/20071012005214/http://news.xinhuanet.com/ziliao/2003-04/02/content_810860.htm |date=2007-10-12 }} 新华网</ref>。

2003年4月，在[[温家宝|温家宝]][[内阁|内阁]]中担任[[国家税务总局|国家税务总局]]局长。在他执掌[[国家税务总局|国家税务总局]]的4年中，[[中国|中国]]年税收收入几乎翻了一番，平均年税收增长幅度达到15.5%，远远超过了[[国内生产总值|国内生产总值]]的增长速度。2003年，[[中国|中国]]税收收入更是突破了2万亿元大关。2007年8月30日，[[第十届全国人民代表大会常务委员会|第十届全国人民代表大会常务委员会]]第二十九次会议任命谢旭人为[[中华人民共和国财政部|财政部]]部长。

2010年4月10日，[[中国财政学会|中国财政学会]]2010年会暨第18次全国财政理论讨论会于北京召开，会上谢旭人当选为中国财政学会第八届理事会会长。

== 执掌财政部 ==
对于出掌财政部，外界评价谢旭人是位严谨、细致、务实的新财长。他直接参与了1993年-1994年[[中国|中国]]的财税体制改革，有着大量基层实践经验和经济专业背景。2009年10月，谢旭人被英国《新兴市场》杂志评为年度“亚洲最佳财长”<ref>[http://www.dwnews.com/2009_10_6_16_34_19_721.html 谢旭人当选“亚洲最佳财长”]{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

2013年3月26日，国务院任命谢旭人为[[全国社会保障基金理事会|全国社会保障基金理事会]]理事长<ref>http://news.xinhuanet.com/politics/2013-03/26/c_115163570.htm</ref>。

2016年11月，不再担任[[全国社会保障基金理事会|全国社会保障基金理事会]]理事长<ref>{{cite web|accessdate=2016-11-24|url=http://www.thepaper.cn/newsDetail_forward_1567429|title=楼继伟正式出任社保基金会理事长、党组成员，谢旭人不再担任|publisher=澎湃新闻}}</ref>。

== 个人爱好 ==
读书和爬山是谢旭人的两大爱好。他坚持每周抽出半天时间爬香山；而更多的业余时间则是用在读书上<ref>[http://www.people.com.cn/GB/news/37146/45767/3794466.html 谢旭人：中国第一税官] 《[[人民日报|人民日报]]》社 - 人民网</ref>。
