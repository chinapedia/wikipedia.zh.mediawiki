'''谢混'''（{{bd|4世纪|?|412年||catIdx=X}}），[[表字|字]]'''叔源'''，小字'''益寿'''。[[陈郡|陈郡]][[阳夏|阳夏]]（今[[河南|河南]][[太康县|太康]]）人，[[谢琰|谢琰]]幼子，[[谢安|谢安]]之孙。[[东晋|东晋]]官員、文学家。在晉官至尚書僕射，但因依附劉毅而被劉裕誅殺。

== 生平 ==
謝混年輕時就有美譽，長於寫文章。後娶[[晉孝武帝|晉孝武帝]]女[[晉陵公主|晉陵公主]]，並承襲父親'''望蔡縣公'''的爵位。

謝混在朝廷历任[[中书令|中书令]]、中领军及[[尚书左仆射|尚书左仆射]]領選等職。

谢混在政治上支持[[北府軍|北府軍]]将领[[劉毅_(東晉)|劉毅]]，與[[劉裕|劉裕]]抗衡，謝混堂兄[[謝澹|謝澹]]，常為此擔憂，因而逐漸跟謝混疏遠<ref>《南史·卷十九》：「初，澹從弟混與劉毅昵，澹常以為憂，漸疏混，每謂弟璞、從子瞻曰：『益壽此性，終當破家。』混尋見誅，朝廷以澹先言，故不及禍。」</ref><ref>《宋書·武帝紀中》：「毅既有雄才大志，厚自矜許，朝士素望者多歸之。與尚書僕射謝混、丹陽尹郗僧施並深相結。」</ref>。[[义熙_(东晋)|义熙]]八年（412年），劉裕決定消滅劉毅勢力，遂收谢混下獄，並於獄中賜死他，晉陵公主也被迫脫離謝家<ref>《南史·卷二十》：「義熙八年，混以劉毅黨見誅，混妻晉陵公主改適琅邪王練。公主雖執意不行，而詔與謝氏離絕。」</ref><ref>《宋書·武帝紀中》：「九月，藩入朝，公命收藩及謝混，並於獄賜死。自表討毅。」</ref>。

刘裕於永初元年（420年）接受晉室禪讓時，謝混堂侄[[謝晦|謝晦]]對劉裕說：「陛下順應天命，登壇之時可惜沒有謝益壽奉上蠒紱。」刘裕也很後悔，說：「我也很後悔，這讓後生一軰無法看到他的風采！」<ref>《晋书·谢混传》：「及宋受禪，謝晦謂劉裕曰：『陛下應天受命，登壇日恨不得謝益壽奉璽紱。』裕亦歎曰：『吾甚恨之，使后生不得見其風流！』益壽，混小字也。」</ref>

== 性格特徵 ==
*謝混曾为时人认为“风华[[江左|江左]]第一”，工於詩<ref>檀道鸾《续晋阳秋》说玄言诗“至义熙中谢混始改”。</ref>，[[钟嵘|钟嵘]]《诗品》评谢混诗，谓“其源出于张华，才力苦弱，故务其清浅，殊得风流媚趣”，《南齐书·文学传论》稱“谢混情新”。作品早佚，今存诗3首，收入《先秦汉魏晋南北朝诗》。
*謝混為人孤高，不常與其他人交際，但就和幾個有文才的族子[[謝靈運|謝靈運]]、[[謝瞻|謝瞻]]、[[謝晦|謝晦]]、[[謝曜|謝曜]]及[[謝弘微|謝弘微]]經常在一起。而他特別重視謝弘微，在謝混死後弘微亦為其代管家業，直至晉亡後晉陵公主得回謝家。

== 逸事 ==
*謝琰被帳下督張猛所殺，謝混的兩個哥哥亦在該戰被殺。後來劉裕在討伐孫恩時生擒了張猛，將之送去謝混那裏，謝混就劏開張猛，割下其肝臟生吃，以報父兄之仇<ref>《晉書·謝琰傳》：「琰帳下都督張猛於後斫琰馬，琰墮地與二子肇峻俱被害，寶亦死之。後劉裕左里之捷生擒猛，送琰小子混，混刳肝生食之。」</ref>。
*晉孝武帝為晉陵公主招夫婿時就向親近的[[王珣|王珣]]說：「公主的夫婿只要像[[劉惔|劉真長]]、[[王獻之|王子敬]]那樣，我就滿意了；[[王敦|王處仲]]、[[桓溫|桓元子]]這種確實好，但才剛得富貴就要干預別人家事。」王珣就答：「謝混雖然比不上真長，但不比子敬差呀。」孝武帝於是說：「這就好了。」不久孝武帝去世，[[袁山松|袁山松]]打算嫁女給謝混，王珣就向袁山松說：「你不要想染指禁臠呀。」正以孝武帝已為公主選定了謝混，他就如專門進獻給皇帝獨佔的那塊最好的肉，其他人不能打其主意。此即“禁脔”一名的由来<ref>《晉書·謝混傳》 ：「初，孝武帝為晉陵公主求婿，謂王珣曰：『主婿但如劉真長、王子敬便足。如王處仲、桓元子誠可，才小富貴，便豫人家事。』珣對曰：『謝混雖不及真長，不減子敬。』帝曰：『如此便足。』未幾，帝崩。袁山松欲以女妻之，珣曰：「卿莫近禁臠。」初元帝始鎮建業，公私窘罄，每得一豚，以為珍膳，項上一臠尤美，輒以薦帝，羣下未嘗敢食，于時呼為禁臠，故珣因以為戲。」</ref>。
*[[桓玄|桓玄]]當政時曾打算改建[[謝安|謝安]]舊宅，謝混就說：「[[召公奭|召伯]]的仁心尚能惠及一棵甘棠樹，而文靖（謝安諡號）的德行卻不能保住五畝的屋宅嗎？」桓玄聽後慚愧不已，遂打消念頭<ref>《晉書·謝混傳》：「桓玄嘗欲以安宅為營，混曰：『召伯之仁猶惠及甘棠，文靖之德更不保五畝之宅邪？』玄聞慙而止。</ref>。
