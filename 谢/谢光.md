
'''谢光'''（{{bd|1929年||2017年|5月23日|catIdx=}}），[[江苏|江苏]][[无锡|无锡]]人，[[中国人民解放军中将|中国人民解放军中将]]<ref name=pp>{{Cite web |url = http://www.thepaper.cn/newsDetail_forward_1696581 |title =原国防科工委副主任谢光中将逝世，享年88岁  |author =  |date =2017-05-29  |publisher = 澎湃新闻 |language =  |accessdate =  |quote =  }}</ref><ref name=gw>{{Cite web |url =  http://www.guancha.cn/politics/2017_05_29_410703.shtml|title =原国防科工委副主任谢光中将逝世 享年88岁  |author =  |date =2017-05-29   |publisher =观察者网  |language =  |accessdate =  |quote =  }}</ref>。

==生平==
1948年，谢光考入北京铁道学院（现[[北京交通大学|北京交通大学]]），1949年参加[[中国人民解放军第四野战军|中国人民解放军第四野战军]]，历任南下工作团学员、随营学校区队长、后勤部宣传股文化教员、职位。1953年谢光进入[[哈尔滨军事工程学院|哈尔滨军事工程学院]]航空工程系飞机设计专业就读，毕业后留校任教。后离开学校，历任国防部第六研究院研究室主任、研究所副所长，[[国务院国防工业办公室|国务院国防工业办公室]]副局长，[[国防科工委|国防科工委]]科技部副部长，国防科工委副主任等职位<ref name=pp/><ref name=gw/>。

1988年授予[[中国人民解放军少将|少将]]军衔，1990年晋升中将<ref name=pp/><ref name=gw/>。

2017年5月23日，谢光在[[北京|北京]]逝世，享年88岁。5月27日在[[北京|北京]][[八宝山殡仪馆|八宝山殡仪馆]]东礼堂举行告别仪式<ref name=pp/><ref name=gw/>。
