'''章宗祥'''（{{bd|1879年||1962年|10月1日|章宗祥}}），字'''仲和'''，[[浙江|浙江]][[吳興縣_(中華民國)|吴兴]][[荻港|荻港]]（今[[湖州|湖州]][[南浔区|南浔区]][[和孚镇|和孚镇]]） 人。中华民国政治人物。<ref name=zheng>鄭則民「章宗祥」{{Cite book<!--|和書-->|author = [[中国社会科学院|中国社会科学院]]近代史研究所|title = 民国人物传 第3卷|year = 1981|publisher = [[中華書局|中華書局]]|isbn = }}</ref><ref name=xu>{{Cite book<!--|和書-->|author = 徐友春主編|title = 民国人物大辞典 増訂版|year = 2007|publisher = 河北人民出版社|isbn = 978-7-202-03014-1}}</ref>

== 生平 ==
[[File:Zhang_Zongxiang.jpg|thumb]]

章宗祥本為[[清朝|清朝]][[秀才|秀才]]，清[[光绪|光绪]]二十五年（1899年）留学[[日本|日本]]，先入[[第一高等学校|第一高等学校]]，后入[[东京帝国大学|东京帝国大学]]法科，并结识了[[曹汝霖|曹汝霖]]。后获[[明治大学|明治大学]][[法学士|法学士]]学位。在日本期间，他曾参加[[译书汇编社|译书汇编社]]。光绪二十九年（1903年）回国，在[[北京|北京]][[京师大学堂|京师大学堂]]任教，清廷赐[[进士|进士]]出身。他曾历任[[進士館|進士館]]教習、[[法律馆|法律馆]]纂修官、[[农工商部|农工商部]]候补主事、[[民政部|民政部]]财例局提调等职务。1905年（光緒31年）他和[[董康|董康]]共同翻译了日本的刑法。后来他帮助商部尚書[[載振|載振]]编纂[[商法|商法]]，又升任[[憲政編査館|憲政編査館]]編制局副局長。1907年（光緒33年），[[徐世昌|徐世昌]]就任[[東三省总督|東三省总督]]，他作为随从跟随赴任。1909年（[[宣統|宣統]]元年），他任北京内城巡警厅丞。此後，他历任法律編纂局編修、内閣法制院副使。1911年（宣統3年）10月[[辛亥革命|辛亥革命]]爆发后，他作为[[唐紹儀|唐紹儀]]的随从参加[[南北议和|南北议和]]。<ref name=zheng/><ref name=xu/>

[[中華民国|中華民国]]成立後，章宗祥任总統府秘書兼法制局局長。1912年（[[民国紀元|民国]]元年）7月，他被发表为[[陸徵祥内閣|陸徵祥内閣]]司法总長，但未获得批准，遂改任[[中华民国大理院|大理院]]院長。1914年（民国3年）2月，他被任命为[[孫宝琦臨時内閣|孫宝琦臨時内閣]]司法总長。<ref name=zheng/><ref name=xu/><ref name=liu>{{Cite book<!--|和書-->|author = 劉寿林等編|title = 民国職官年表|year = 1995|publisher = 中華書局|isbn = 7-101-01320-1}}</ref>

翌年，袁世凱称帝，章作为司法总長，否定了[[籌安会|籌安会]]的違法性。袁世凱死後的1916年（民国5年）6月末，章宗祥接替[[陆宗舆|陆宗舆]]改任駐日公使，到日本赴任。在日本，他和[[西原龟三|西原龟三]]进行交涉以缔结[[西原借款|西原借款]]，并同日本政府缔结軍事協定。他还承认了日本方面在山東半島的权益。他还和[[陸宗輿|陸宗輿]]一起就对德国宣战向[[段祺瑞|段祺瑞]]進言。<ref name=zheng/><ref name=xu/><ref name=liu/>

1919年（民国8年）[[巴黎和会|巴黎和会]]上山東問題成为中国关心的焦点问题。由于对日本采取让步态度，章宗祥、[[曹汝霖|曹汝霖]]、[[陸宗輿|陸宗輿]]被中国舆论纠弹。同年[[五四運動|五四運動]]中，[[曹汝霖|曹汝霖]]的宅邸被学生焚毁，史称[[火烧赵家楼|火烧赵家楼]]。当时归国后正在曹宅訪問的章宗祥被学生殴打致[[脑震荡|脑震荡]]。<ref name=zheng/><ref name=xu/><ref>[http://cul.cn.yahoo.com/ypen/20120925/1330180.html 反思五四暴力行为：章宗祥被打成脑震荡，雅虎，2012-09-25] {{webarchive|url=https://web.archive.org/web/20121101130802/http://cul.cn.yahoo.com/ypen/20120925/1330180.html |date=2012-11-01 }}</ref>6月10日，章宗祥、[[曹汝霖|曹汝霖]]、[[陸宗輿|陸宗輿]]被罷免。<ref name=zheng/><ref name=xu/><ref name=liu/>事后，章宗祥释放了打人学生，此事在历史上鲜为人知。

以後，章宗祥从政界引退，转入实业界，任[[中华汇业银行|中华汇业银行]]（中日合办的銀行）总理、[[北京通商銀行|北京通商銀行]]总理。[[北洋政府|北洋政府]]倒台後，他長期寓居[[青島市|青島]]。1942年（民国31年）3月，他应[[王揖唐|王揖唐]]之邀，任[[華北政務委員会|華北政務委員会]]諮詢委員。后来他还任電力会社的[[董事長|董事長]]。[[抗日战争|抗日战争]]结束後，他被[[国民政府|国民政府]]逮捕，未受裁判即被释放，后寓居上海。[[中華人民共和国|中華人民共和国]]成立後，他留在上海。<ref name=zheng/><ref name=xu/>

1962年10月1日，章宗祥病逝于上海。享年84岁。<ref name=zheng/><ref name=xu/>
