'''梨山建設管理局'''，曾經是[[中華民國|中華民國]][[臺灣省|臺灣省]]管轄的[[管理局|管理局]]，存在於1973年至1981年約8年的時間，以原[[臺中縣|臺中縣]][[和平區_(臺灣)|和平鄉]]的梨山、平等兩村為轄區，包含青山、達見（德基）、佳陽、福壽山、[[梨山|梨山]]、松茂、環山、志良、武陵、勝光、思源等地，受[[臺灣省政府|臺灣省政府]]指揮監督，辦理轄區內的建設及行政管理事宜。局址設於[[梨山賓館|梨山賓館]]旁，即今日的'''交通部觀光局參山國家風景區管理處梨山管理站'''。

== 沿革 ==
*1967年7月11日，梨山地區建設委員會成立，隸於[[臺灣省政府民政廳|臺灣省政府民政廳]]之下。 
*1967年7月12日，[[劉大中|劉大中]]、[[蔣碩傑|蔣碩傑]]、[[顧應昌|顧應昌]]、[[費景漢|費景漢]]在[[中華民國經濟部|經濟部]]部長[[李國鼎|李國鼎]]陪同下，於[[福壽山農場|福壽山農場]][[梨山行館|梨山行館]]會見[[中華民國總統|總統]][[蔣中正|蔣中正]]，是為「梨山會議」。
*1971年5月15日，臺灣省政府通過設立梨山管理局。
*1972年5月1日，[[行政院|行政院]]核定名稱改為梨山建設管理局。
*1973年2月19日，臺灣省政府的改制[[公文|公文]]正式發出。
*1973年2月22日，行政院正式核定通過改制。
*1973年7月1日，梨山建設管理局正式升格，直轄於臺灣省政府。
*1973年12月29日，梨山建設管理局與[[臺中縣政府|臺中縣政府]]辦理業務交接。
*1980年12月2日，設立梨山風景特定區，並發佈梨山建設管理局降編辦法。
*1981年3月1日，降編為梨山風景特定區管理所，改隸於[[臺灣省政府交通處|臺灣省政府交通處]]之下。
*1994年4月1日，降編為[[梨山風景區|梨山風景區]]管理所，改隸於臺灣省政府交通處旅遊事業管理局之下。
*2001年3月16日，降編為梨山管理站，改隸於[[交通部觀光局|交通部觀光局]][[參山國家風景區管理處|參山國家風景區管理處]]之下，無對外獨立發文權。

== 編制 ==
===初建時期===
管理局置[[簡任|簡任]]局長一人，由臺灣省政府任命，並設主任秘書一人襄理政務。

直屬單位：
*秘書室
*行政科
*建設科

附屬單位：
*警察所
*衛生所
*自來水廠

===裁撤前夕===
管理局置簡任局長一人，由臺灣省政府任命，並設主任秘書一人襄理政務。

直屬單位：
*秘書室
*行政科
*建設科

附屬單位：
*[[臺中縣警察局|臺中縣警察局]]梨山分駐所
*臺中縣[[和平區_(臺中市)|和平鄉]]梨山衛生所
*梨山自來水廠
*地政處
*都市計劃委員會
*梨山國民小學
*平等國民小學

== 特色 ==

梨山建設管理局設立時，人口僅2000多人，面積683平方公里，但年平均[[國民所得|國民所得]]超過新臺幣4萬元（相當於1050美元），為臺灣當時整體平均國民所得的兩倍，當地並有定期直升機班機，自[[福壽山農場|福壽山農場]]的直升機場往返[[水湳機場|水湳機場]]。

照[[陽明山管理局|陽明山管理局]]慣例，梨山建設管理局轄區內選出的縣議員、[[鄉民代表|鄉民代表]]等均參加[[臺中縣議會|臺中縣議會]]、和平鄉民代表會行使職權，並邀該局人員列席報告並備詢，但實務上該局局長常是直接在[[臺灣省議會|臺灣省議會]]備詢。

梨山建設管理局直轄梨山地區內戶籍、合作業務、教育、地籍管理及[[都市計劃|都市計劃]]工作，並兼轄[[德基水庫|德基水庫]]、[[福壽山農場|福壽山農場]]與[[武陵農場|武陵農場]]，所屬的梨山、平等國小均改冠該局頭銜；另該局人口、土地等統計數字雖併列於當時臺中縣內，惟臺中縣需另外括號單獨併列註記。
