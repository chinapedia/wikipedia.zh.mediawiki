'''玫瑰盃球場'''位於[[美國|美國]][[洛杉磯|洛杉磯]][[郊區|郊區]]的[[帕萨迪纳_(加利福尼亚州)|帕薩迪納市]]，它現在可容納92,542人，建於1921-1922年。玫瑰盃球場亦是[[1984年夏季奧林匹克運動會|1984年奧運會]]和[[1994年美國世界盃|1994年世界盃]]決賽的場地。

== 歷史 ==
玫瑰盃球場在[[美國|美國]]國內因每年舉行大學[[美式足球|美式足球]]盃而著名，球賽被稱為[[玫瑰盃|玫瑰盃]]。[[玫瑰盃|玫瑰盃]]是[[大學|大學]][[美式足球|美式足球]]盃賽中歷史最悠久和最著名的盃賽。除了盃賽外，玫瑰盃球場也舉行過其他運動項目，自1982年起這球場是[[洛杉磯加利福尼亞大學|洛杉磯加利福尼亞大學]][[美式足球|美式足球]]隊的主場，亦是[[美國職業足球大聯盟|美國職業足球大聯盟]]球隊[[洛杉磯銀河|洛杉磯銀河]]自1996年建隊以來的主場，直至2003年銀河隊搬到英式足球球場Home Depot Center；1998年舉辦美國職業足球大聯盟決賽，以及1984年[[奧運|奧運]]、[[1994年世界盃|1994年世界盃]]和[[1999年女子世界杯足球赛|1999年女足世界杯]]的比賽地點。

== 世界盃 ==
玫瑰盃球場是1994年男子[[世界盃足球賽|世界盃]]及1999年女子世界盃的決賽球場，另外一個辦過男女子世界盃決賽的球場是[[瑞典|瑞典]]的Råsunda Stadium，曾於1958年舉行男子世界盃決賽及1995年舉行女子世界盃決賽。玫瑰盃球場的兩場世界盃決賽都是[[加時|加時]]後0:0打和要互射[[十二碼|十二碼]]決勝，1994年世界盃中[[巴西國家足球隊|巴西]]擊敗[[意大利國家足球隊|意大利]]，而1999年女子世界盃中美國勝中國。

1999年女子世界盃是史上最多觀眾的女子運動比賽，官方統計的入場人數是90,185。

== 超級碗 ==
玫瑰碗球場舉行過[[超級盃|超級盃]]5次，分別在1977, 1980, 1983, 1987, 1993。
