《'''战狼'''》（曾經用名《'''兵锋'''》及《'''特种兵之战狼'''》）為由[[中國|中國]][[演員|演員]][[吳京_(演員)|吴京]]自导自演的一部中国现代军事题材电影，于2015年4月2日以[[二維|二維]]及[[三維|三維]]/[[全景声|全景声]]等多個版本在[[中国大陆|中国大陆]]院线上映。續集《[[戰狼2|戰狼2]]》於2017年上映。

== 剧情 ==
在一次与公安部联合捣毁制毒窝点的行动中，冷锋违反命令将国际犯罪组织的骨干武吉狙杀，导致被关禁闭，后得到战狼中队指挥官龙小云赏识，保释他并让他加入战狼中队参与军事演习。

在冷锋狙杀武吉时，武吉的哥哥、国际犯罪组织的首领敏登正在与武吉通话并亲眼目睹了武吉被冷锋所杀害。其后敏登被国际刑警逮捕但被一股雇佣兵救走，后敏登向雇佣兵首领“老猫”下达命令让其杀死冷锋为弟报仇。

演习中，冷锋带领一支小分队出其不意地使红军指挥官石青松退场，但在返程路上小分队遭到狼群袭击，众人艰难击退狼群。殊不知突然出现的狼群是雇佣兵组织的圈套，其后分队长俞飞遭到雇佣兵袭击阵亡，雇佣兵入侵中国领土。演习变为实战，所有解放军人員均整装待发准备驱逐雇佣兵为战友报仇，殊不知雇佣兵早有准备，解放军伤亡惨重，后冷锋众人施计脱困后将雇佣兵逐一击杀，最后在国境线上冷锋与“老猫”对峙并将其刺死。与此同时，敏登等人也化装为解放军军医，试图以新研发的针对中国人的基因武器暗杀冷锋，但事情败露，反被冷锋逮捕在国境线内。

== 制作 ==
《战狼》涉及[[特种部隊|特种部隊]]题材，为在[[中国電影|中国電影]]中比较少见。吴京为此筹备历时7年，期间出演了电视剧《[[我是特種兵之利刃出鞘|我是特種兵之利刃出鞘]]》预热，并且利用三維技术呈现战争场面，亦开展了[[中国大陸|中国大陸]]电影的先河<ref>{{cite news|url=http://ent.163.com/14/1211/08/AD610N4H000300B1.html|title=《战狼》题材受关注 揭秘奇兵“东方之狼”|date=2014-12-11|publisher=网易|accessdate=2015-04-05}}</ref>。关于剧本设定，吴京坦言剧本审查非常严格：“因为拍现代战争片会涉及到军事机密，而且在外交层面上也十分敏感，所以审查会很严格，《战狼》最初版本的剧本就曾被毙过，我把片中外国军人角色更改为雇佣兵，这才避免了不少麻烦。”<ref>{{cite news|url=http://news.mtime.com/pix/2013/08/09/395704.html|title=视频探班吴京新片"战狼" 现役坦克群集结出动|date=2013-08-09|publisher=时光网|accessdate=2015-04-05}}</ref>

本片拍摄得到了[[中国人民解放军南京军区|中国人民解放军南京军区]]的大力支持，动员了多款新式武器装备，采用近万发子弹，上百辆[[坦克|坦克]]以及[[武直-10|武直-10]]直升机进行实战拍摄<ref>{{cite news|url=http://movie.mtime.com/203613/behind_the_scene.html|title=关于《战狼》你应该知道的5件事|publisher=时光网|accessdate=2015-04-05}}</ref>。吴京还经过特批在[[特种部队|特种部队]]体验生活18个月<ref>{{cite news|url=http://news.sina.com.cn/c/p/2015-03-29/210831658507.shtml|title=吴京为拍《战狼》在特种部队呆了18个月|date=2015-03-29|publisher=新浪|accessdate=2015-04-05}}</ref>。

《战狼》於2013年6月15日在[[南京|南京]]开机<ref>{{cite news|url=http://ent.sina.com.cn/m/c/2013-06-18/12173944766.shtml|title=《战狼》南京开机 吴京荷枪实弹霸气回归|date=2013-06-18|publisher=新浪|accessdate=2015-04-05}}</ref>，9月5日杀青<ref>{{cite news|url=http://ent.sina.com.cn/m/c/2013-09-06/17514003795.shtml|title=电影《战狼》艰难杀青 吴京望天咆哮|date=2013-09-06|publisher=新浪|accessdate=2015-04-05}}</ref>。

== 宣传 ==
[[File:Wujing_in_Nanjing,20150401,01.JPG|thumb]]
本片于2014年12月和2015年3月举行两次看片场，赢得了不错的口碑。在影片宣传期间，吴京亲自与影城、院线方面沟通，尤其在一些二三线城市如[[淄博|淄博]]等地的路演，营造了影片的声势<ref>{{cite news|url=http://media.people.com.cn/n/2015/0405/c40606-26800539.html|title=《战狼》三天破亿将拍续集 吴京自导自演成大黑马|date=2015-04-05|publisher=人民网|accessdate=2015-04-05}}</ref>。从2015年3月16日至4月4日，为宣传本片，吴京已在中国内地跑了18个城市，100多家影院<ref name=a >{{cite news|url=http://ent.163.com/15/0405/03/AMDIUV5000032DGD.html|title=《战狼》三天票房过亿这是吴京跑出来的|date=2015-04-05|publisher=网易|accessdate=2015-04-05}}</ref>。

== 票房 ==
本片自4月2日在中国大陆上映后，三天票房过亿，成为清明节档的一匹“黑马”，口碑票房双丰收<ref name="a" />。首周四天获得2.08亿人民币居冠<ref>[http://www.1905.com/news/20150407/877331.shtml#p1 内地票房：清明3天吸金5亿 《战狼》登顶成黑马]</ref>。最终累计5.45亿。

== 评价 ==
《[[海南日报|海南日报]]》评论称：“国产电影里似乎很少这种类型的影片，描写的是特种兵题材，全片一个劲儿地“打”，包括武器战和肉搏战，而且节奏紧凑到一气呵成，紧张得让人喘不过气来，堪称最能让人看嗨的一部电影了。”<ref>{{cite news|url=http://news.163.com/15/0405/07/AME179UK00014Q4P.html|title=《战狼》让人看得嗨|date=2015-04-05|publisher= 网易|accessdate=2015-04-05}}</ref>

专业影评人对影片有不同声音，支持者认为本片作为国产当代军事题材在多个方面都有突破，武戏场面和文戏剧情上能看到美式（或好莱坞）战争片的影子。批评者认为剧情简单不足，人物塑造脸谱化，一些动作简单粗暴。<ref >{{cite web|url=http://www.1905.com/news/20150405/877139.shtml | language= | title=影评人热议激辩：《战狼》火了，当真好看吗？ |publisher=腾讯娱乐 |date=2015-04-05 |accessdate= }}</ref>
