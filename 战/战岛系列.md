'''''Battle Isle''''' 是一系列[[回合策略|回合策略]][[电脑游戏|电脑游戏]]，由[[Blue_Byte_Software|Blue Byte Software]]自1990年开始开发。描述 Chromos 这颗星球上的战争。

== 故事主线 ==
Chromos星球是一个科技高度发达的社会，深入研发[[机器人|机器人]]和[[人工智能系统|人工智能系统]]但是太空科技只能支持到达卫星[[殖民地|殖民地]]。游戏围绕着殖民地战争进行。

== 玩法 ==
前三个游戏使用[[六边形|六边形]]地图。玩家控制战斗单位 (从 [[步兵|步兵]] 、[[坦克|坦克]]到[[直升机|直升机]]，[[战斗机|战斗机]]、[[轰炸机|轰炸机]]、[[装甲火车|装甲火车]]、 [[军舰|军舰]]和[[潜艇|潜艇]]、碉堡等)， 后勤辅助单位有不同的武器和经验，玩家可以控制某些要塞，生产新单位。 ''Battle Isle 2''起，天气改变影响单位移动 (如冰封的海洋河流，阻止船只，但是允许轻型单位通过)。[[战争迷雾|战争迷雾]]要求玩家执行侦察任务。

''Battle Isle''第一作把屏幕一分为二，每一部分对应一个玩家，回合指令是移动和攻击两种。当一个玩家指挥移动时，另外一个玩家己的部队执行原有命令。 ''Battle Isle 2'' 则使用全屏幕，并融合了两种指令，也引入了[[三维|三维]]战斗。 ''Battle Isle 3''类似''Battle Isle 2''，画面效果和单位多样性都有所改善。''Battle Isle: The Andosian War''是全三维环境下融入即时因素的新作。

== 历史 ==
''Battle Isle'' 是Blue Byte首次在[[德国|德国]]和[[欧洲|欧洲]]都取得成功的作品。完成于1991年，深受[[PC_Engine|PC Engine]]平台[[日本|日本]]游戏''[[月面基地|月面基地]]''启发， ''Battle Isle''拥有大量的插件add-ons，续作(著名的 ''Battle Isle 2'' 1993 和''Battle Isle 3'' 1995)和仿制作品。仿制作品中最著名的是BlueByte's自己的''[[History_Line:_1914-1918|History Line: 1914-1918]]''1993，使用[[第一次世界大战|第一次世界大战]]背景。其他的包括[[开源|开源]]的''[[Advanced_Strategic_Command|Advanced Strategic Command]]''和''[[Crimson_Fields|Crimson Fields]]''。

Blue Byte 曾用''Battle Isle''招牌研发新游戏，[[MMORPG|MMORPG]]''[[DarkSpace|DarkSpace]]'', 并传出''Battle Isle V: DarkSpace''的消息，可不久Blue Byte被[[Ubisoft|Ubisoft]]收购，''DarkSpace''成了独立项目。

发售平台开始是[[Amiga|Amiga]]和[[MS_DOS|MS DOS]]，后来是[[Microsoft_Windows|Microsoft Windows]]。

现在没有新的战岛系列正在开发，不过[[Advanced_Strategic_Command|Advanced Strategic Command]]吸引了不少老玩家。另外，[[韦诺之战|韦诺之战]]也受到''战岛系列''影响。
