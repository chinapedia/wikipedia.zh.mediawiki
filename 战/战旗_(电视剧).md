《'''战旗'''》是[[江苏卫视|江苏卫视]]于2012年热播的一部[[抗日战争|抗日战争]]题材电视剧，11月16日起播出<ref>[http://ent.sina.com.cn/v/m/2012-11-16/10553789328.shtml 《战旗》江苏卫视开播 上演战地版欢喜冤家]2012-11-16</ref>。该剧自称“偶像式”战争剧，以泼辣的女游击队长戴金花和留洋海归军官金戈联合抗日为主线，两人在“一边互掐一边合作”的过程中，慢慢发现彼此互补是天生的一对，成为不可分离的欢喜冤家<ref>[http://ent.qq.com/a/20121123/000238.htm 高富帅英雄正当红 《战旗》四板斧讨好小年轻]2012-11-23</ref>。

== 剧情概述 ==
金戈从美国弗吉尼亚军校毕业后，回国加入国民党，任晋绥军852团特战大队队长。他擅用奇谋，对日军作战神勇。然而，晋绥军852团团长肖百川暗中与日军长官龟井次郎勾结，假意让金戈回家成亲，实际上设下陷阱要除掉他。金戈并未回家而是带人奇袭了日军重火力炮兵阵地，炸死了龟井次郎，与其兄长龟井一郎结下仇怨。金戈与未婚妻柳文婷完婚之日，肖百川弃城逃跑，引日军入城，金戈的母亲被日军坦克压死，柳文婷为免受辱被迫跳楼。血战之后，金戈被土匪于化龙、于大海父子所救，於是，炸死埋名。而戴金花卻把他從土匪中救出，二人就一起打鬼子。

八路军冀中游击队队长戴金花发现金戈就是上级要她护送的军事专家，二人从此一路前行，开始了联手抗击日本人的旅途。二人一个是留洋归来家道殷实的军事天才，一个是风里来雨里去混在男人堆里的“穆桂英”；一个是用惯了西式战术战法的国民党高级军官，一个是童养媳苦出身的女游击队长，对对方完全没有好感。但命运却紧紧地将他们纠缠在一起，两位性格迥异的英雄儿女，竟也在这战火硝烟的洗礼下惺惺相惜，爱情意外闯入两人心扉。而金戈更建议打出自己的战旗震慑鬼子，第九大队就此成立，他们神出鬼没的出击令日本人心惊胆战。金戈的死敌龟井一郎受发誓要消灭这支游击队，但第九大队无论遭受了多大损失，鲜艳的红旗始终牢牢的扎在敌后游击战场……

1945年，抗日战争已进入尾声，金戈受命回到家乡—东安城，粉碎日军的“死海计划”，而负责这项计划的正是死对头龟井一郎。在东安城，金戈遇到了昔日同窗钱柏豪和投水未死的柳文婷，他们的目标也是“死海计划”。金戈、戴金花、钱柏豪、柳文婷，个人的情与义，家国的恨与爱，在龟井一郎的天罗地网下，他们如何拯救东安城，而他们的命运又会走向何方……

== 相关 ==
* 导演毛卫宁说，为了吸引更多年轻人观剧，加入了很多偶像化的元素，但拒绝恶搞，“战争绝对不能游戏化，不能恶搞”。<ref>[http://ent.ifeng.com/tv/news/mainland/detail_2012_11/28/19603304_0.shtml 《战旗》收视破2 导演毛卫宁：有偶像元素但不恶搞]2012-11-28</ref>

* 随着本剧的热播，金戈和戴金花的战地情感成为亮点，网友戏称“泼辣女八路戴金花倒追海龟军官金戈，一路追随的欢喜冤家，如同爆灯牵手成功的男女嘉宾，这不是战争版的《非诚勿扰》爆灯吗？”<ref>[http://news.xinhuanet.com/ent/2012-11/29/c_124021636.htm 《战旗》成战争版“非诚勿扰” ]2012-11-29</ref>

* 本剧使用了多种偶像化的拍摄手法，打造“年轻态”的观众体验。<ref>[http://ent.sina.com.cn/r/m/2012-11-29/12063799532.shtml 评论：《战旗》偶像战争剧迈向成熟]2012-11-29</ref>

* 本剧剧集数因播出后收视率的一路攀升，被江苏卫视采取上集回放的方式拉长到43集。12月2日的大结局则在新剧《与狼共舞》第一集后播出。
