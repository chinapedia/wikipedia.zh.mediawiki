'''殺手鱷'''（{{lang-en|Killer Croc}}，又譯作'''鱷魚人'''、'''殺人鱷'''）是一名登場於由[[DC漫畫|DC漫畫]]出版的漫畫書中的虛構[[超級反派|超級反派]]，是[[蝙蝠俠|蝙蝠俠]]的主要敵人之一。首次於《[[偵探漫畫|偵探漫畫]]》#523（1983年2月）中短暫登場。該角色的創作者為作家[[葛瑞·康威|葛瑞·康威]]與藝術家[[吉恩·科蘭|吉恩·科蘭]]。

== 人物 ==
殺手鱷的本名為'''韋倫·瓊斯'''（{{lang-en|Waylon Jones}}）。瓊斯因[[返祖现象|返祖现象]]賦予了他爬行動物般的外表，他由姑姑撫養長大。儘管瓊斯只是名青少年，姑姑卻常因其醜陋的外表而虐待他及為他取難聽的外號<ref>{{cite web |url=http://www.pulpinterest.com/entertainment/movies/5-batman-villains-yet-see-movies/ |title=5 Batman Villains We Have Yet To See In The Movies |publisher=Pulpinterest.com |date=2014-02-05 |accessdate=2014-05-13 |archive-url=https://web.archive.org/web/20140327060758/http://pulpinterest.com/entertainment/movies/5-batman-villains-yet-see-movies/ |archive-date=2014-03-27 |dead-url=yes }}</ref>。後來，他殺了姑姑，並成為了犯下無數殺人罪的罪犯、以及蝙蝠俠與[[羅賓_(漫畫)|羅賓]]的敵人。

== 能力 ==
由於返祖現象的緣故，殺手鱷的身體開始鱷魚化，並擁有了利爪、尖牙，以及可以彈開手槍子彈的鱗片，並擁有如動物般的直覺和感官（像是嗅覺與聽覺變得十分靈敏等等）；他也因此能在水中生活和行動。

殺手鱷具有超越常人的力量、速度與體能，加上他接受過一段時間的摔角技巧訓練，這使他在正面作戰時幾乎無人能擋。

殺手鱷具有癒合因子，這使他在受傷後能迅速痊癒。
