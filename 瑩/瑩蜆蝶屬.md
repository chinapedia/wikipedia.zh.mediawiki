'''瑩蜆蝶屬'''（學名：'''''Lucillella'''''）是[[蜆蝶科|蜆蝶科]][[蜆蝶亞科|蜆蝶亞科]][[樹蜆蝶族|樹蜆蝶族]]裡的一個[[屬|屬]]。共有7個物種，分佈於[[哥倫比亞|哥倫比亞]]、[[厄瓜多爾|厄瓜多爾]]和[[秘魯|秘魯]]的[[安第斯山脈|安第斯山脈]]中海拔地區<ref>Brower, Andrew V. Z. 2012. Lucillella Strand 1932. Lucilla Hewitson 1870 invalid junior homonym of Lucilla Gimmerthal, 1842 (Diptera). Version 12 March 2012 (under construction). http://tolweb.org/Lucillella/105301/2012.03.12 in The Tree of Life Web Project, http://tolweb.org/ {{en}}</ref>。

== 形態 ==
翅膀黑色，前翅配有橙或紅色的警告色橫紋。外表或是[[擬態|擬態]]模仿[[優蜆蝶亞科|優蜆蝶亞科]]的褐雅蜆蝶（''Hades hecamede''），[[束珍蝶屬|束珍蝶屬]]、[[燈蛾科|燈蛾科]]、[[尺蛾科|尺蛾科]]和[[透翅舟蛾亞科|透翅舟蛾亞科]]的物種都可能是此屬的混合擬態對象<ref>Hall, J.P.W.,Willmott, K.R. 2010. Description of a new Lucillella species (Riodinidae: Symachiini) discovered in the eastern Andes of Ecuador using the single rope canopy access technique. J. Lep. Soc. 64: 139-146.</ref>。
