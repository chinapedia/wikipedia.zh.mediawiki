'''三氟化钛'''（[[钛|Ti]][[氟|F<sub>3</sub>]]）是钛元素的一种[[氟化物|氟化物]]。该物质难溶于水、稀酸、稀碱溶液，但会被浓酸分解，因此需避免与强酸接触。

== 制备 ==
三氟化钛最早于1954年制得，当时将金属钛与[[氟化氢|氟化氢]]反应制得了该物质：

:<math>\rm \ 2Ti + 6HF \xrightarrow{\Delta} 2TiF_3 + 3H_2</math>

实际生产中将氢气与氟化氢按1：4混合后在700℃下与金属钛反应4-5小时，产率约为90%，需要严格控制温度，因为温度如果太高则会生成[[四氟化钛|四氟化钛]]。

在[[氩|氩]]气气氛保护下，在一支[[镍|镍]]舟中用金属钛还原四氟化钛也可制得三氟化钛，一开始先在550℃下处理3-4小时，然后再900℃下反应6-8h，产率也可达到90%。然后在1000℃下真空[[升华|升华]]提纯产物：

:<math>\rm \ Ti + 3TiF_4 \xrightarrow{\Delta} 4TiF_3</math>

另一种方法是[[三氯化钛|三氯化钛]]与氟化氢气体进行[[复分解反应|复分解反应]]：

:<math>\rm \ TiCl_3 + 3HF \xrightarrow{\Delta} TiF_3 + 3HCl</math>

一开始温度控制在425℃，然后迅速升温至700℃，最后迅速冷却，最佳产率只有30%，因此不太常用。但是不需要用成本较高的金属钛。用氢气在650℃时还原[[六氟钛酸铵|六氟钛酸铵]]也得三氟化钛，产率更低，约为17%：

:<math>\rm \ 2(NH_4)_2TiF_6 + H_2 \xrightarrow{\Delta} 2(NH_4)_2TiF_5 + 2HF</math>
:<math>\rm \ (NH_4)_2TiF_5 \xrightarrow{\Delta} NH_4F + TiF_3</math>

因为反应温度下部分原料未发生还原早已升华，因此产率低。<ref>{{cite book|title=《无机化学丛书》第八卷：钛分族、钒分族、铬分族|pages=P24|author=张青莲|isbn=7-03-002238-6|location=北京|publisher=科学出版社}}</ref>

== 用途==
它可用于制取[[钛氟玻璃|钛氟玻璃]]。
