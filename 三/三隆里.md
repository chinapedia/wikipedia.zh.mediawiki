三隆里，位於[[高雄市|高雄市]][[大寮區|大寮區]]中部的一個里，北和永芳里、琉球里為鄰，東及東南分以為[[上寮里|上寮里]]、[[大寮里|大寮里]]為鄰，南近[[過溪里|過溪里]]，西及會社里、過溪里相靠<ref>高雄縣大寮鄉行政區域圖，內政部，2005年11月</ref>。全里分48鄰，人口約5500人，戶數約1850戶，面積3267361平方公尺。

== 地名由來 ==
三隆里由芎蕉腳、洪厝程、拷潭寮三個聚落結合來命名,而【隆】取它繁榮興旺之意,為當時永芳公學校校長張簡德瑞集合村内人士命名而成。

== 歷史 ==
1920年全臺行政區劃大改制，原小竹上里所轄之[[磚仔磘庄|磚仔磘庄]]、[[翁公園庄|翁公園庄]]、[[山仔頂庄|山仔頂庄]]，及[[小竹下里|小竹下里]]所轄的拷潭庄、赤崁庄、大藔庄合併成立大寮庄，隸屬高雄州鳳山郡。<ref>《臺灣堡圖》，臺灣總督府，1904年</ref>1946年（民國35年），大寮庄改制為大寮鄉，屬於高雄縣；其下屬的大寮則分拆為上寮村（頂大寮）、大寮村（下大寮、後壁寮）及三隆村（芎蕉腳、考潭寮、洪厝埕）三個村。2010年12月25日五都改制，三隆村改為[[高雄市|高雄市]][[大寮區|大寮區]]三隆里。<ref>《新舊對照管轄便覽》</ref>

== 主要街道 ==
鳳林三路、二路(省道25)、[[大寮路|大寮路]](縣道188)、光明路2段，則三隆路貫穿本里，東接上寮路。
芎蕉腳:北興路。
洪厝程:永芳路。
拷潭寮(為三隆人口主要集中地):(橫向)大信、昌隆、瑞隆、秀隆、文隆、明隆、永隆、義隆、仁隆、忠隆街(縱向)慈隆、立隆、安隆、富隆、正隆、東隆、華隆、佳隆、福隆、忠隆街、興隆路。<ref>高雄縣大寮鄉行政區域圖，內政部，2005年11月</ref>
