《'''三字集'''》，於[[1931年|1931年]]9月發行，為[[台灣日治時期|台灣日治時期]]出版之[[無產階級|無產階級]]宣傳刊物，以[[台灣話文|台灣話文]]寫成。《三字集》的編作與發行者是[[共產主義|共產主義]]提倡者[[陳德興|陳德興]]、[[陳神助|陳神助]]、[[林水福|林水福]]、[[張城|張城]]與[[簡吉|簡吉]]等知名[[台灣共產黨|台灣共產黨]]政治人物。《三字集》為[[台灣|台灣]]無產階級或共產主義於台灣發行的首份通俗宣傳品，與同時間出品的《[[真理_(雜誌)|真理]]》、《[[二字集|二字集]]》相同，都是以「喚醒台灣[[階級意識|階級意識]]」為目標的政治刊物。不過因為涉及反日，旋即遭封禁，而與著作相關人士，均於[[日本|日本]]政府清[[共產黨|共]]的過程中，遭逮捕入獄。

== 特色 ==
以宣揚共產主義為目的《三字集》特色為[[文字|文字]]通俗易懂，並深含政治目的，這兩項特色均有其背景。就文字普羅化方面，深受1920年末期[[台灣話文論戰|台灣語文論戰]]影響，以讓農工等基層族群閱讀及使用起來方便為由，捨棄[[文言文|漢古文]]寫作，強調[[臺灣話|臺灣話]]與文字合一。另外就政治目的方面，作為[[台灣共產黨|台灣共產黨]]週邊組織機關報之一的《三字集》，字裡行間皆為宣達無產階級之偉大。

== 背景 ==
1929年10月，世界[[經濟大恐慌|經濟大恐慌]]，社會主義思想風起雲湧，這風潮亦影響當時為日本殖民地的台灣。同時，同情[[社會主義|社會主義]]的相關文學刊物亦相繼於台灣出刊，如《[[台灣文學_(雜誌)|台灣文學]]》、《[[伍人報|伍人報]]》、《[[明日|明日]]》、《[[現代生活|現代生活]]》、《[[赤道|赤道]]》、《[[新台灣戰線|新台灣戰線]]》等。

1931年，因應[[日本|日本]]本土國內日趨緊張政經情勢，台灣[[大正民主|大正民主]]時期的政治放鬆態勢亦隨之宣告結束。日人組成的統治台灣機器──[[台灣總督府|台灣總督府]]為了壓制台灣本土政治運動所造成的反日風潮，大舉搜捕台灣社會主義組織成員，另一方面，如[[台灣文化協會|台灣文化協會]]的臺灣社會主義團體及相關許多文學刊物亦遭禁止運作與發行。

其中，文化協會成員[[詹以昌|詹以昌]]、[[陳崑崙|陳崑崙]]與週邊組織[[台灣農民組合|台灣農民組合]]主要幹部簡吉、[[顏錦華|顏錦華]]等人開會研商相應對策。與會台灣社運團體成員均認為，要對抗日本[[資本主義|資本]]與[[帝國主義|帝國主義]]的迫害，必須捨棄如文化協會的消極溫和做法，而改為較激烈的組織與社會活動來對抗。於具體做法上，他們亦認為發展共產組織是較適當做法。

在此認知下，同年8月，可算台灣共產黨後續週邊組織的[[臺灣赤色救援會|臺灣赤色救援會]]成立，會中的會務發展決議裡面的重點之一，就是發行該組織機關報來宣揚擴大組織規模。9月末，赤色救援會機關報之一的《三字集》分上下兩卷發行，全文以通俗易懂為編輯要點。

== 發行 ==
三字集以三字一句，兩句一押韻，以臺灣話寫成。因為字句通俗，貼近民眾，造成一定風潮。易琅上口的文章內容，不但有助於向無產大眾宣傳共產主義思想，於文學方面，該《三字集》也宣告1930年代臺灣話文運動達到了文藝大眾化的實際成果。

三字集發行不久後，隨即遭台灣總督府查禁，同年年末，三字集發行人與編者均遭駐台日警逮捕入獄，如農民組合、赤色救援會暨三字集發行人簡吉就旋即被宣判十年徒刑。

== 目的 ==
*激發階級意識為主
*描寫無產大眾受壓迫的苦況
*控訴台灣[[地主|地主]]、[[資本家|資本家]]與[[帝國主義|帝國主義]]相互勾結，剝削無產大眾
*宣揚[[蘇維埃|蘇維埃]]為無產階級剝削的理想世界
*倡導共產主義革命的必要
