'''三硫化二砷'''是一种无机化合物，化学式为As<sub>2</sub>S<sub>3</sub>。三硫化二砷在自然界以[[雌黄|雌黄]]矿物的形式存在。<ref name=whdx>武汉大学 吉林大学等. 无机化学（第三版）下册. 高等教育出版社, 2011. pp 704-706. ISBN 978-7-04-004880-3</ref>

==结构==
三硫化二砷存在晶体和无定形体。<ref>Wells, A.F. (1984). Structural Inorganic Chemistry, Oxford: Clarendon Press. ISBN 0-19-855370-6.</ref>单斜的As<sub>2</sub>S<sub>3</sub>是层状结构，每个As和3个S成键，As-S为224pm，∠S-As-S 99°；而其蒸气以二聚体As<sub>4</sub>S<sub>6</sub>的形式存在，结构与[[三氧化二砷|As<sub>4</sub>O<sub>6</sub>]]类似。<ref>无机化学丛书. 第四卷 氮 磷 砷分族. 科学出版社. pp 408. 三硫化二砷</ref>
==制备==
三硫化二砷可由单质化合得到，或者从As(III)的溶液中由H<sub>2</sub>S沉淀。<ref>{{lang|ru|Справочник химика. Редкол.: Никольский Б.П. и др.. (3-е изд.). испр. — Л.: Химия, 1971. Т. 2.}} pp 1168.</ref>
: 2 As + 3 S → As<sub>2</sub>S<sub>3</sub>
==化学性质==
三硫化二砷在空气中灼烧，可以产生[[三氧化二砷|三氧化二砷]]：<ref name=whdx />
: 2 As<sub>2</sub>S<sub>3</sub> + 9 O<sub>2</sub> → 2 As<sub>2</sub>O<sub>3</sub> + 6 SO<sub>2</sub>
它可以被[[硫化钠|硫化钠]]转化为[[三硫代亚砷酸钠|三硫代亚砷酸钠]]，被多硫化钠转化为[[四硫代砷酸钠|四硫代砷酸钠]]：
: As<sub>2</sub>S<sub>3</sub> + 3 Na<sub>2</sub>S → 2 Na<sub>3</sub>AsS<sub>3</sub>
: As<sub>2</sub>S<sub>3</sub> + 3 Na<sub>2</sub>S<sub>2</sub> → 2 Na<sub>3</sub>AsS<sub>4</sub>
而在和[[氢氧化钠|氢氧化钠]]溶液的反应中，会同时产生亚砷酸盐和硫代亚砷酸盐：
: As<sub>2</sub>S<sub>3</sub> + 6 NaOH → Na<sub>3</sub>AsO<sub>3</sub> + Na<sub>3</sub>AsS<sub>3</sub> + 3 H<sub>2</sub>O
此[[溶液|溶液]]在经过脱硫、还原等操作可以制得三氧化二砷。<ref>郑雅杰, 刘万宇, 白猛. 采用硫化砷渣制备三氧化二砷工艺. 中南大学学报:自然科学版, 2008. 39(6): 1157-1163</ref>当溶液中OH<sup>-</sup>不足时，则产生偏亚砷酸盐和硫代偏亚砷酸盐：<ref name=chen>陈寿椿. 重要无机化学反应. 上海科学技术出版社, 1994</ref>
: 2 As<sub>2</sub>S<sub>3</sub> + 4 NaOH → NaAsO<sub>2</sub> + 3 NaAsS<sub>2</sub> + 2 H<sub>2</sub>O
和[[碳酸钠|碳酸钠]]溶液反应，得到同样的砷化合物：
: 2 As<sub>2</sub>S<sub>3</sub> + Na<sub>2</sub>CO<sub>3</sub> → NaAsO<sub>2</sub> + 3 NaAsS<sub>2</sub> + 2 CO<sub>2</sub>↑
和[[氯气|氯气]]反应，产生棕色液体：<ref name=chen />
: As<sub>2</sub>S<sub>3</sub> + 6 Cl<sub>2</sub> → (AsCl<sub>3</sub>)<sub>2</sub>·3SCl<sub>2</sub>
和[[碘|碘]]反应，产生[[三碘化砷|三碘化砷]]和[[硫|硫]]单质：
: As<sub>2</sub>S<sub>3</sub> + 3 I<sub>2</sub> → 2 AsI<sub>3</sub> + 3 S
和[[氯化亚铜|氯化亚铜]]、[[氯化银|氯化银]]等金属氯化物共热，均会产生[[三氯化砷|三氯化砷]]：<ref name=chen />
: 3 As<sub>2</sub>S<sub>3</sub> + 6 CuCl —<sup><u>200~300℃</u></sup>→ Cu<sub>6</sub>As<sub>4</sub>S<sub>9</sub> + 2 AsCl<sub>3</sub> （CuCl过量则产生Cu<sub>4</sub>As<sub>2</sub>S<sub>5</sub>）
: 2 As<sub>2</sub>S<sub>3</sub> + 3 AgCl —<u><sup>170℃</sup></u>→ 3 AgAsS<sub>2</sub> + AsCl<sub>3</sub>
: 2 As + 3 Hg<sub>2</sub>Cl<sub>2</sub> → 2 AsCl<sub>3</sub> + 6 Hg
: 2 As + 3 HgCl<sub>2</sub> → 2 ACl<sub>3</sub> + 3 Hg

==用途==
三硫化二砷可以用于制备[[砷|砷]]单质<ref>侯汉娜, 陈甜甜. 硫化砷渣全湿法制备单质砷的研究. 环境保护科学, 2014(6):42-45</ref>及[[三氧化二砷|三氧化二砷]]<ref>孟文杰, 施孟华, 李倩 等. 硫化砷渣湿法制取三氧化二砷的处理技术现状. 贵州化工, 2008, 33(5):26-28</ref>、砷酸盐<ref>赖建林, 李勤. 用硫化砷渣制取砷酸铜. 有色金属:冶炼部分, 2001, 1(2):42-44</ref>等其它砷化合物。

此外，三硫化二砷还能用作防腐剂，或在[[医药|医药]]领域有所应用。<ref>曹梦晔, 巩江, 高昂 等. 雌黄药学研究概况. 辽宁中医药大学学报, 2011(3):54-56</ref>
