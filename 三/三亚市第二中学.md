'''三亚市第二中学'''，位于[[海南省|海南省]][[三亚市|三亚市]][[解放路|解放路]]426号，为全日制公立中学。

== 沿革 ==
1963年，崖县红旗小学建立，它是三亚市第二中学的前身。后来学校更名为“三亚中学”，现用名“三亚市第二中学”。

== 学校文化 ==
=== 理念 ===
办有灵魂的学校，育有思想的学生

=== 特色 ===
三亚市第二中学除了完成[[中华人民共和国教育部|中华人民共和国教育部]]规定的课程之外，还开了特色课程，如：“体育与健康”、“艺术”、“技术”。<ref>{{cite web |url=http://news.k618.cn/xy_37043/201305/t20130518_3243764.html |title=海南省校本课程“各显神通” |accessdate=2013 |publisher=未来网 |work= |language=zh-cn}}</ref>
