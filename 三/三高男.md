'''三高男'''是起源於[[日本|日本]]的詞語，其中「三高」意指一些[[女性|女性]]對擇偶的主要三項條件：「高學歷」、「高收入」、「高身長」（{{lang-ja|高身長}}，[[身长|身长]]即是身高），後在[[大中华地区|大中华地区]]均有使用。[[中國大陸|中國大陸]]近來也有同義的'''[[高富帥|高富帥]]'''一詞，意為“[[身长|身材]]高、[[財富|財富]]多、[[長相|長相]]帥”。在[[臺灣|臺灣]]也有近義詞'''[[溫拿|溫拿]]'''（英文winner的諧音，成功者、胜利者）來形容人帥、有車有房有錢的「人生勝利組」。在[[拜金主義|拜金主義]]的推動下許多[[小姐|小姐]]都希望被[[包養|包養]]，笑貧不笑娼風氣正盛行，在網路做裸露直播的錢正好賺，沒錢買性的會被當作[[魯蛇|魯蛇]]。<ref>[http://tw.news.yahoo.com/%E7%A4%BE%E7%BE%A4%E6%96%B0%E8%81%9E-ptt%E8%87%AA%E5%89%B5%E8%A9%9E-%E9%AD%AF%E8%9B%87%E6%AD%A3%E5%A4%AF-093158868.html Yahoo新聞-Ptt自創詞 「魯蛇」正夯]</ref>。

== 名稱起源 ==
“三高男”之名稱起源於日本[[明治時代|明治時代]]，本來是指「高志」、「高德」、「高潔」。直至1980年代末，即[[日本泡沫經濟|日本泡沫經濟]]的全盛時期，「三高」被賦予新意義，指當時部分女性的擇偶條件为「高學歷」、「高收入」、「高身長」，是當時的一項流行術語。後随着网络传开，流行于中华地区。

== 發展 ==
{{cn|日本女性的「三高」擇偶條件，到了後期更進一步增加「不是長子」的條件（在日本，[[長子|長子]]多數與[[雙親|雙親]]同住）。}}正因為如此，「三高」結果導致了自從[[明治|明治]]時代以後，日本第一次出現[[男性|男性]]難以找到[[結婚|結婚]]對象的情況。這一現象充分反映了日本社會在[[泡沫經濟|泡沫經濟]]時期對[[物質|物質]]上的需求遠超於心理素質的現象。近年於[[東亞|東亞]]其他[[都市化|都市化]]地區如[[北京|北京]]、[[上海|上海]]、[[台灣|台灣]]、[[廣州|廣州]]、[[香港|香港]]、[[韓國|韓國]]等亦有這種現象。

然而，泡沫經濟崩潰後，[[日本經濟|日本經濟]]低迷，絕對的三高便順理成章地成為過去。然而男性擇偶困難這一點卻未有明顯改變，[[心理學|心理學]]者[[小倉千加子|小倉千加子]]指出，在泡沫經濟爆破後的日本，女性向男性要求的條件逐漸轉變，但要求依然相對地高，相對的三高依然存在，東亞其他[[都市化|都市化]]程度較高地區的男性亦遇到同樣問題。

此現象除反映[[物質主義|物質主義]]、[[拜金主義|拜金主義]]盛行外，與女性地位提高亦有關係。近20年來，不少女性的學歷、收入、職位都比過往的女性高，物質享受也比過往的女性高，也吸收較多食物營養和較多時間運動，因此身高亦較過往的女性高。但傳統文化中的「男強女弱」觀念仍然存在，令女性的擇偶條件亦相應提高。另外，女性經濟獨立，令她們不需要依靠婚姻來維持生存，對婚姻品質的要求亦大大提高，而家庭經濟、夫妻[[配合度|配合度]]、配偶外表亦多少會影響婚姻素質，收入影響家庭經濟，學歷影響夫妻溝通和配合，身高影響觀感（有些女性認為身材高的男性較有安全感），因此對某些女性來說，「三高」的條件頗為重要。

此外，近年「三高」在[[大中華地區|大中華地區]]亦用來形容「高收入」、「高學歷」、「高職位」的女性（三高女性），這種女性亦面臨難以找到結婚對象的問題。部分「三高女性」的對結婚對象的要求亦是「三高」。

== 影響 ==
男性只要有低學歷、低收入或身高較矮三條件之一，就難以找到結婚或戀愛對象，產生[[自卑|自卑]]心態，可能產生心理問題，影響與異性相處、戀愛或將來的婚姻生活。而男性因為自卑出現的心理問題又更令女性卻步，於是更鞏固「三高」的條件。「三高」的觀念又令年輕人尤其是女性偏向重視物質和外表，忽略精神和[[心靈|心靈]]上的交流，令愛情變質。

由於「三高」的觀念令不符合三高的男性被[[標籤|標籤]]為次等，一些女性會瞧不起他們。有些自認為是次等的男性會以低聲下氣討好女性的方式追求，有些居心不良的女性就把他們當作[[馱獸_(人物)|馱獸]]利用。也加深了不少男性對女性的不滿、甚至造成[[女性貶抑|貶抑]]心理及[[仇恨|仇恨]]，令性別對立加劇，甚至連符合三高條件的男性都可能出現這種心態。「三高」又是出於「男強女弱」的觀念，讓男性感受到很明顯的[[逆向歧視|逆向歧視]]，不利[[性別平等|性別平等]]，某程度上亦強化了社會上既有的[[大男人主義|大男人主義]]（如果女性不願意選擇比自己弱的男人，那就應該把工作機會及薪資留給男性，以免高未婚率及高無後率造成一群自私無遠見的國民）。而一些希望結婚的女性不是堅持「三高」擇偶條件就是個性過為強勢，亦難以找到結婚對象，以至過了適婚年齡依然未婚，甚至要孤獨終老；更多男性也會因為非頂尖而終身與婚姻感情無緣，一些男性條件雖然良好、但不擅長討好及分辨女人，他們也可能因為女性的「強勢化、過度現實、高要求及性開放」而對婚姻愛情裹足不前。「三高」的擇偶條件令男女雙方的權利皆受到損害（尤其是三高要求加劇[[少子化|少子化]]問題，而少子化造成[[下流老人|下流老人]]問題 ），許多女性（包括[[女權主義|女權主義]]者）、總是喊著打倒父權，卻不知道、或拒絕承認，三高要求反而製造了更惡劣的父權。
