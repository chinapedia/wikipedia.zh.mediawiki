{{about|毛泽东于1970年代提出的[[国际关系理论|国际关系理论]]|其它名为“三个世界”的理论|三个世界}}
{{Maoism sidebar|概念}}
'''三个世界理论'''指[[毛泽东|毛泽东]]在[[冷战|冷战]]背景下，根据[[第二次世界大战|第二次世界大战]]后[[国际关系|国际关系]]的新格局把[[国际社会|国际社会]]划分为3个部分的[[国际关系理论|国际关系理论]]。

毛泽东这一思想萌芽于1940年代的[[中间地带论|中间地带论]]思想，雏形于1960年代的中间地带论外交战略，形成于1970年代。{{citation needed}}

== 提出 ==
1973年6月22日，毛泽东在会见[[马里|马里]]国家元首[[特拉奥雷|特拉奥雷]]时提到：“我们都是叫做第三世界，就是发展中国家”。1974年2月22日，毛泽东在会见[[赞比亚总统|赞比亚总统]][[卡翁达|卡翁达]]时首次公开提出这一思想，“我看美国、苏联是第一世界。中间派，日本、欧洲、澳大利亚、加拿大，是第二世界。咱们是第三世界。”“亚洲除了日本，都是第三世界，整个非洲都是第三世界，拉丁美洲也是第三世界。”

1974年4月，邓小平在联合国第六届特别会议上发言，第一次向世界全面阐述了划分“三个世界”的战略思想。

== 内容 ==
三个世界理论中的“三个世界”的涵义是：
*[[第一世界|第一世界]]是指[[美国|美国]]和[[苏联|苏联]]两个超级大国，在世界范围内推行[[霸权主义|霸权主义]]的[[超级大国|超级大国]]。
*[[第二世界|第二世界]]是指处于这两者之间的[[发达国家|发达国家]]，如[[英国|英国]]、[[德国|德国]]、[[日本|日本]]、[[加拿大|加拿大]]、[[澳大利亚|澳大利亚]]等国。
*[[第三世界|第三世界]]指[[亚洲|亚洲]]、[[非洲|非洲]]、[[拉丁美洲|拉丁美洲]]和其他地区的[[发展中国家|发展中国家]]和[[未開發國家|未開發國家]]。

其中以第三世界國家最多，而且彼此大有差別（除最貧窮的國家外，還包括盛產石油的國家），因此無法給出適當的定義。有些人（包括聯合國）還承認存在一個包括25個最貧窮國家的第四世界。

第三世界國家往往採取中立的立場，因而從政治上把世界一分為三。

{{fact|毛泽东认为[[超级大国|超级大国]]之间争夺世界[[霸权|霸权]]地位是世界局势动荡不安的主要根源。}}第三世界是反对[[帝国主义|帝国主义]]、[[殖民主义|殖民主义]]、[[霸权主义|霸权主义]]的主要力量。

== 批评 ==
* [[阿尔巴尼亚|阿尔巴尼亚]]领导人[[恩维尔·霍查|恩维尔·霍查]]强烈反对三个世界理论，他认为三个世界理论掩盖了时代的基本特征，即[[帝国主义|帝国主义]]和[[无产阶级革命|无产阶级革命]]时代的基本矛盾，鼓吹阶级合作，从本质上说是[[阶级斗争|阶级斗争]]熄灭论的变种。三个世界理论和“不结盟世界”理论、“南北世界”理论、“[[发展中国家|发展中国家]]”理论一样，都不是以[[阶级|阶级]]为标准来划分的，因此不是[[马列主义|马列主义]]的理论<ref>{{cite web |url = http://news.dayoo.com/china/200812/03/53868_4781451.htm |title=霍查曾骂毛泽东是"种族主义,要奴役全人类"|deadurl=yes|archiveurl=https://web.archive.org/web/20150318182110/http://news.dayoo.com/china/200812/03/53868_4781451.htm |archivedate = 2015-03-18 }}</ref>。霍查这一批评得到了很多[[反修正主义|反修正主义]]政党和组织的支持，它们因此放弃[[毛主义|毛主义]]，转而追随[[阿尔巴尼亚劳动党|阿尔巴尼亚劳动党]]的政治路线，形成了[[国际共产主义运动|国际共产主义运动]]中的[[霍查主义|霍查派]]。<ref>[http://www.wyzxwk.com/Article/guoji/2009/09/14526.html 当代国际共运史的主线：毛主义和现代修正主义的斗争]</ref>
* 美国革命共产党刊物《革命》1978年发表文章认为，中国修正主义者将三个世界理论称为“面向国际无产阶级和被压迫人民的全球战略”，这绝非毛泽东的本意，将三个世界理论归于毛泽东是一种伪造，三个世界理论是反革命路线。<ref>{{Cite web|url=https://www.marxists.org/history/erol/ncm-5/rcp-3.htm|title=“Three Worlds” Strategy: Apology for Capitulation|accessdate=|author=Revolutionary Communist Party|date=|publisher=}}</ref>
