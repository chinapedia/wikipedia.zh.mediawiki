'''三毛入野命'''（{{jpn|j = 三毛入野命|hg = みけ<small>（或みか）</small>いりの<small>（或いりぬ、いるぬ）</small> の みこと|rm = Mikeilino no mikoto|kyu = |hgt = }}），一般稱作'''三毛野命'''（{{jpn|j = 三毛野命|hg = みけの<small>（或みけぬ、みつけの、みつけぬま）</small> の みこと|rm = Mikeno no mikoto|kyu = |hgt = }}，御毛沼命）是[[神武天皇|神武天皇]]（日本第1代[[天皇|天皇]]）的皇兄。

== 名稱 ==
人名中的「三毛（{{lang|ja|みけ}}）」意為[[食物|食物]]。

== 事績 ==
=== 日本書紀的記載 ===
依據《日本書紀》，[[神武東征|神武東征]]過程中，大軍行至熊野（今[[和歌山縣|和歌山縣]]、[[三重縣|三重縣]]南部）外海突遇暴風雨襲擊，二兄[[稻飯命|稻飯命]]拔劍躍入海中後，'''三毛入野命'''亦道：「我母<ref group="注">指[[豐玉姬|豐玉姬]]。</ref>及姨<ref group="注">指[[玉依姬|玉依姬]]。</ref>並是[[海神|海神]]。何爲起波瀾，以灌溺乎？」於是祂也躍入海裡，踏浪花渡往[[常世國|常世鄉]]。

=== 古事記的記載 ===
依據《古事記》上卷，御毛沼命踏上浪頭，渡坐至[[常世國|常世國]]。

=== 父母 ===
* 父：[[彥波瀲武鸕鶿草葺不合尊|彥波瀲武鸕鶿草葺不合尊]]（彥波瀲尊、彥澰尊） - [[彥火火出見尊|彥火火出見尊]]之子。[[天照大神|天照大神]]之四世孫。
* 母：[[玉依姬|玉依姬]] - [[海童|海童]]之小女。彥波瀲武鸕鶿草葺不合尊之[[姨|姨]]・[[乳母|乳母]]。

三毛入野命，是彥波瀲武鸕鶿草葺不合尊的第三子也<ref group="注">
# 《日本書紀》本文三毛入野命為三男。
# 同書第一種異說三毛入野命為三男。
# 同書第二種異說三毛野命為次男。
# 同書第四種異說三毛入野命則變成么男。
# 《古事記》御毛沼命為三男。</ref>。

=== 同母兄弟姉妹 ===
* 兄：[[彥五瀨命|彥五瀨命]]（五瀨命） - 中賊矢薨于軍。
* 兄：[[稻飯命|稻飯命]] - 爲妣国而沒海爲[[鋤持神|鋤持神]]。爲[[新羅|新良國]]國主。[[新羅|新羅]]國王之祖。
* 弟：[[稚三毛野尊|稚三毛野尊]]（神日本磐余彥尊、神倭天皇） - 神武天皇（日本第1代[[天皇|天皇]]）。生母[[豐玉姬|豐玉姬]]（入[[海|海]]而歸[[龍宮|龍宮]]）。

== 解說 ==
依照[[宮崎縣|宮崎縣]][[西臼杵郡|西臼杵郡]][[高千穗神社|高千穗神社]]的說法，三毛入野命並未渡往常世之國，反而因為和祂的兄弟走散了，返回大軍[[神武東征|東征]]的出發地高千穗。彼時當地出現一個名為「鬼八（{{lang|ja|きはち}}）」的凶神，三毛入野命驅趕之；故後人將祂入祀高千穗神社，其妻兒、子孫成為[[十社大明神|十社大明神]]，一同被祭祀。

== 信仰 ==
供奉三毛入野命的[[神社|神社]]主要有：
* [[高千穗神社|高千穗神社]]（[[宮崎縣|宮崎縣]][[西臼杵郡|西臼杵郡]][[高千穗町|高千穗町]]）：該社奉祀的[[十社大明神|十社大明神]]乃三毛入野命、妻、兒女等共十位神祇。
* [[阿古師神社|阿古師神社]]（[[三重縣|三重縣]][[熊野市|熊野市]]）
