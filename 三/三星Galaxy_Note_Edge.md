'''三星Galaxy Note Edge'''是[[韩国|韩国]][[三星电子|三星电子]]制造的最新[[平板手机|大屏幕]]旗舰级[[智能手机|智能手机]]，於2014年9月3日在[[德国|德国]]举行的[[柏林国际广播展|柏林国际广播展]]上发布。与同期发布的[[三星Galaxy_Note_4|Galaxy Note 4]]相比，Note Edge的最大卖点是右侧附有一块可独立运作的曲面屏幕，该屏幕运用三星最新的柔性[[AMOLED|AMOLED]]工艺，与主屏幕是无缝一体化。<ref name=engadget-noteedgehandson>{{cite web|title=The Galaxy Note Edge: Samsung's first smartphone with a bent display|url=http://www.engadget.com/2014/09/03/samsung-galaxy-note-edge-hands-on/|website=Engadget|date=2014年9月3日}}</ref>它的顯示器在設備右側彎曲顯示，可用作邊欄顯示應用程序快捷方式，通知和其他信息。

== 開發和發布 == 
在2013年消費電子展上，三星展示了“Youm”概念 - 用於融合柔性顯示屏的智能手機原型。一個原型機的屏幕沿著手機的右邊緣彎曲，另一個原型機的屏幕在手機底部彎曲。三星解釋說，可以使用額外的“strip”來顯示附加信息，例如通知或新聞摘要。<ref>{{Cite web|url=https://www.oled-info.com/samsung-youm|title=Samsung YOUM flexible OLEDs: technology and status! OLED-Info|accessdate=2018-04-29|work=www.oled-info.com|language=en}}</ref><ref>{{Cite news|url=https://www.cnet.com/news/eyes-on-samsungs-youm-flexible-display-tech-at-ces-2013/|title=Eyes-on: Samsung's Youm flexible-display tech at CES 2013|date=2013-01-09|newspaper=CNET|accessdate=2018-04-29|language=en}}</ref><ref>{{Cite news|url=https://www.theverge.com/2013/1/9/3855960/samsung-flexible-oled-phone-prototype-hands-on|title=Samsung shows off flexible OLED phone prototype (hands-on)|newspaper=The Verge|accessdate=2018-04-29}}</ref>“Youm”概念將作為Galaxy Note Edge的一部分出現，Galaxy Note Edge 將於2014年9月4日與Galaxy Note 4一同推出。<ref>{{Cite news|url=https://www.engadget.com/2014/09/03/samsung-galaxy-note-edge-hands-on/|title=The Galaxy Note Edge: Samsung's first smartphone with a bent display|newspaper=Engadget|accessdate=2018-04-29|language=en-US}}</ref>三星策略師Justin Denison解釋說，該公司喜歡在其產品中冒風險，繼續說“我們不是一家做一次性事業的公司,我們喜歡做大事，並承擔責任。<ref>{{Cite news|url=https://www.cnet.com/news/with-galaxy-note-edge-samsung-returns-to-comfort-zone-hardware/|title=With Galaxy Note Edge, Samsung returns to comfort zone: hardware|date=2014-09-03|newspaper=CNET|accessdate=2018-04-29|language=en}}</ref>

== 產品規格 ==

=== 硬件和設計 ===
Galaxy Note Edge在設計上與Galaxy Note 4不同是按照Galaxy Note 3設計，配有金屬框架和塑料皮革後蓋。該器件具有[[Exynos|Exynos 5 Octa 5433]]（韓國版）或[[Snapdragon|Qualcomm Snapdragon 805]]（國際版），3 GB RAM以及32或64 GB可擴展存儲。與其他Galaxy Note系列設備一樣，它包含一個S Pen [[手寫筆|手寫筆]]，可用於筆輸入，繪圖和手寫。與其他最近的三星旗艦設備類似，它還包括心率傳感器和指紋掃描儀。Galaxy Note Edge採用5.6英寸 “[[显示分辨率列表|Quad HD +]][[Super_AMOLED|Super AMOLED]]”顯示屏，其中包含一個160像素寬的色譜柱，可在曲線上環繞設備的側面。<ref>{{Cite news|url=https://www.engadget.com/2014/09/03/samsung-galaxy-note-edge-hands-on/|title=The Galaxy Note Edge: Samsung's first smartphone with a bent display|newspaper=Engadget|accessdate=2018-04-29|language=en-US}}</ref><ref>{{Cite news|url=https://www.cnet.com/news/with-galaxy-note-edge-samsung-returns-to-comfort-zone-hardware/|title=With Galaxy Note Edge, Samsung returns to comfort zone: hardware|date=2014-09-03|newspaper=CNET|accessdate=2018-04-29|language=en}}</ref><ref>{{Cite news|url=https://www.gsmarena.com/samsung_galaxy_note_edge-review-1165.php|title=Samsung Galaxy Note Edge review: The other side|newspaper=GSMArena.com|accessdate=2018-04-29|language=en-US}}</ref><ref>{{Cite web|url=https://www.gsmarena.com/samsung_galaxy_note_edge-6631.php|title=Samsung Galaxy Note Edge - Full phone specifications|accessdate=2018-04-29|work=www.gsmarena.com|language=en-US}}</ref>該設備包括有一個1600萬像素的後置攝像頭，具有背照式傳感器，[[光学防抖|光學防抖]]和4K視頻錄製功能，以及一個3.7萬像素的前置攝像頭。在2017年，現在'''Edge'''仍舊可以在二手市場上獲得，並且是最後一款具有可更換電池的手機之一。

=== 軟件 ===
Galaxy Note Edge附帶[[Android_4.4|Android 4]].4.4“KitKat”和三星的[[TouchWiz|TouchWiz]]界面和軟件套件，與Note 4類似。屏幕的弧形邊緣用作各種用途的[[側邊欄|側邊欄]]：可用於顯示不同的面板，包括常用應用程序的快捷方式，通知，[[新闻|新聞]]，[[股票|股票]]，[[体育|體育]]，[[社交網路服務|社交網絡]]，音樂和視頻播放器的播放控制，相機控制，數據使用和小遊戲的顯示。工具也可通過面板使用，包括標尺，秒錶，計時器，錄音筆和手電筒按鈕。一個[[软件开发工具包|軟件開發工具包]]可用於開發人員編碼面板; 其他面板可以通過[[Galaxy_Apps|Galaxy Apps]]獲得。熄屏顯示“時鐘”模式允許邊沿屏幕在預定時間段內在不使用時顯示[[Always_on_Display|Always on Display]]。由於AMOLED顯示器的性質（根本不打開像素而呈現黑色），此模式不會顯著消耗電池電量，但根據軟件限制，它一次不能超過12小時。
