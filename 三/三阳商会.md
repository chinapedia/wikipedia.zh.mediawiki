'''三阳商会'''（{{jpn|j=株式会社 三陽商会|hg=さんようしょうかい|rm=SanyoShokai}}），是[[日本国|日本国]][[东京都|东京都]][[新宿区|新宿区]]一家服装的设计制造商，旗下有48个品牌，其商品主要通过[[百货公司|百货公司]]进行销售。该公司的理念是“通过时尚，创造一个美好和丰富的文化生活并对社会的发展贡献”<ref>{{Cite web|url = http://www.sanyo-shokai.co.jp/company/corporate/philosophy.html|script-title = ja:SANYO 株式会社三陽商社{{!}}企業理念|accessdate = 2015-07-12|author = {{lang|ja|三陽商社}}|date = |publisher = |quote = {{lang|ja|ファッションを通じ、美しく豊かな生活文化を創造し、社会の発展に貢献します。}}}}</ref>。三阳商会同时也在中国大陆开设有独资子公司。<ref name=":1">{{Cite web|url = http://www.sanyo-shokai.co.jp/company/corporate/profile.html|script-title = ja:SANYO 三陽商会 会社概要|accessdate = 2015-07-14|author = |date = |publisher = |quote = {{lang|ja|上海三陽時装商貿有限公司}};{{lang|ja|「BURBERRY」 コートの輸入販売開始}};{{lang|ja|バーバリー社と契約し「BURBERRY」を発売}};}}</ref>

== 简介 ==
{{link-ja|吉原信之|吉原信之}}原本从事电气相关的各种工业产品以及纤维制品的加工制造和销售。1943年（[[昭和|昭和]]18年），日本时值战争年代，他在东京都[[板桥区_(日本)|板桥区]]建立了资本金为5万日元公司，来进行机器工具的维修加工和销售，该公司即株式会社三阳商会<ref>{{Cite web|url = http://gendai.ismedia.jp/articles/-/39553|script-title = ja:バーバリーに逃げられた!名門・三陽商会の「苦悩と決断」売り上げの半分を失う……会社は大丈夫なのか|accessdate = 2015-07-12|author = {{lang|ja|週刊現代}}|date = 2014-06-16|publisher = {{lang|ja|週刊現代}}}}</ref>。三阳的三来自当时日本的大财阀“[[三井|三井]]”和“[[三菱财阀|三菱]]”，“阳”则是纪念创始者吉田的父亲“吉原阳”。<ref name=":0">{{Cite journal|url = |script-title = ja:「三陽商会におけるブランドの発展」|journal = {{lang|ja|立命館経営学}}|issue = 第44卷第5号93-119|doi = |others = |year = 2006|volume = |page = |pmid = |author = {{lang|ja|木下明浩}}}}</ref>

[[二战|二战]]结束后的1945年（昭和20年）8月，板桥的工厂被出售，总部迁移至[[银座|银座]]，主营业务则变成了销售[[雨衣|雨衣]]<ref>{{Cite web|url = http://biz-journal.jp/2014/06/post_5103.html|script-title = ja:バーバリー、三陽と契約解除の舞台裏　相次ぐ海外ブランドの一方的解除でアパレル業界岐路|accessdate = 2015-07-12|author = {{lang|ja|福井晋}}|date = 2014-06-12|publisher = {{lang|ja|biz-journal.jp}}|coauthors = {{lang|ja|フリーライター}}}}</ref>。1949年(昭和24年)，第一通商（现[[三井物产|三井物产]]）开展了“全国出口商品展销会”（{{lang|ja|全国エキスポートバザー}}）的企划，三阳获得了大量雨衣定做缝制的订单，作为“雨衣制造销售大公司”而初露头角<ref name=":0" />。与此同时，日本橡胶工业（后来与冈本橡胶工业合并），通过诸如藤仓橡胶工业等的关系，被迅速推进成为雨衣业界的大公司。日本橡胶工业成产雨衣，而三阳则负责拓展雨衣销售的业务<ref name=":0" />。1951年(昭和26年)，三阳注册了“三阳雨衣”的商标，同时开始向全国的各大百货商店供货。在1953年(昭和28年)的银座松屋百货的竞赛企划中，三阳的“防尘衣”被评选为最优秀产品。1959年(昭和34年)时，当时成为话题的法国电影“[[三月出生|三月出生]]”于2月25日上映<ref>{{Cite web|url = http://www.allcinema.net/prog/show_c.php?num_c=9208|script-title = ja:映画 三月生れ|accessdate = 2015-07-12|author = |date = |publisher = {{lang|ja|Allcinema}}}}</ref>，电影中女演员Jacqueline·Sassard身着三阳的Sasaru大衣（{{lang|ja|ササールコート}}），从此三阳从雨衣制造商正式转变为了综合服饰制造商。

== 合作关系 ==
三阳在1965年（昭和40年）开始和英国的[[巴寶莉|巴宝莉]]开始合作，开始在日本本土代理销售巴宝莉生产的大衣。1969年(昭和44年)以后，三阳通过与巴宝莉其他海外的大衣制造厂的技术协作，获得了在日本国内制造、设计的许可。此后，这两家公司一起开发了众多时尚品牌，逐出现“EPOCA”、“FRAGILE”等品牌。<ref name=":1" />

三阳与巴宝莉的许可协议在2015年（平成27年）6月30日结束<ref>{{Cite web|url = http://www.47news.jp/CN/201405/CN2014051901002069.html|script-title = ja:三陽商会、バーバリー販売終了へ　１５年夏に|accessdate = 2015-07-13|author = {{lang|ja|共同通信}}|date = 2014-05-19|publisher = 47news|deadurl = yes|archiveurl = https://web.archive.org/web/20150108084935/http://www.47news.jp/CN/201405/CN2014051901002069.html|archivedate = 2015-01-08}}</ref>。英国巴宝莉将在日本成为独立法人<ref>{{Cite web|url = http://toyokeizai.net/articles/-/38333|script-title = ja:三陽商会はなぜバーバリーを失ったのか - 屋台骨喪失の内幕と､激化する跡地争奪戦|accessdate = 2015-07-13|author = {{lang|ja|秦 卓弥}}|date = 2015-05-22|publisher = {{lang|ja|東洋経済}}}}</ref>，从此两家公司分道扬镳<ref>{{Cite journal|url = |script-title = ja:三陽商会、英バーバリーとの契約が30日終了　後継事業育成急ぐ|journal = {{lang|ja|日本経済新聞電子版}}|issue = |doi = |others = |year = 2015-06-30|volume = |page = |pmid = |author = }}</ref><ref>{{Cite web|url = http://www.nikkei.com/article/DGXNASDZ210GN_R20C14A5TI0000/|script-title = ja:バーバリー、3年で日本の売上高4倍に　直営を拡大|accessdate = 2015-07-13|author = {{lang|ja|黄田和宏}}|date = 2014-05-21|publisher = {{lang|ja|日本経済新聞}}}}</ref>。

== 在中国的发展 ==
2006年5月26日，三阳商会在上海成立独资子公司“上海三阳时装商贸有限公司”，子公司本社位于[[上海市|上海市]][[南京西路_(上海)|南京西路]]。<ref>{{Cite web|url = http://www.sanyo-shokai.co.jp/company/corporate/history.html|script-title = ja:SANYO 株式会社三陽商会 沿革|accessdate = 2015-07-14|author = |date = |publisher = |quote = {{lang|ja|上海に現地法人設立}}}}</ref>
