'''三英戰呂布'''，是小說《[[三國演義|三國演義]]》所虛構劇情之一。

== 背景 ==
此詞彙出現於《三國演義》第五回「發矯詔諸鎮應曹公 破關兵三英戰呂布」。背景是[[東漢|東漢]]末年，當時[[丞相|丞相]][[董卓|董卓]]把持[[朝政|朝政]]，引致各地[[郡守|郡守]]、[[州牧|州牧]]和[[義士|義士]]不滿。其中有一個漢官[[曹操|曹操]]，刺殺董卓不成，於是逃亡至[[陳留|陳留]]，並發[[矯詔|矯詔]]給各地[[州牧|州牧]]，以聚集[[士兵|士兵]]共討董卓。

== 事件開頭 ==
董卓得知後，派大將[[華雄|華雄]]出戰[[汜水關|汜水關]]，討董卓盟盟主[[袁紹|袁紹]]指派[[孫堅|孫堅]]作先鋒，而[[袁術|袁術]]作糧草供應。[[濟北|濟北]]相信[[鮑信|鮑信]]恐怕孫堅立取大功，連派其弟[[鮑忠|鮑忠]]抄小路偷襲華雄，華雄引五百鐵騎應戰，鮑忠不足一個回合戰死。而孫堅初戰華雄，大敗董卓軍。入夜後，華雄領兵夜襲，孫堅敗退，唯孫堅部隊[[祖茂|祖茂]]引開華雄，方可保命。

袁紹再派[[大將|大將]][[俞涉|俞涉]]、[[潘鳳|潘鳳]]應戰華雄，但全部不足三個回合戰死。[[關羽|關羽]]則[[毛遂自薦|毛遂自薦]]，說：「小將軍願往斬華雄頭，獻於帳下！」袁術反對說：「汝欺吾眾諸侯無大將耶？量一弓手，安敢亂言！與我打出！」曹操則指可以讓關羽嘗試，並賜關羽一杯酒，關羽要斬完華雄再飲。結果關羽一刀將華雄斬下。回來時，酒還是熱的！（這亦是民間戲曲「溫酒斬華雄」等劇之來源。）

== 內容 ==
董卓得知華雄被斬，再派[[飛將軍|飛將軍]][[呂布|呂布]]出戰。呂布一出戰，連斬大將[[方悅|方悅]]、[[穆順|穆順]]，殺傷[[武安國|武安國]]、[[公孫瓚|公孫瓚]]。[[張飛|張飛]]見公孫瓚打不過呂布，連忙接替公孫瓚，大喊：「[[三姓家奴|三姓家奴]]休走！燕人張飛在此！」，之後連鬥五十餘回合，不分勝負。（此為毛宗崗本，有些早期版本為張飛槍法漸漸散亂，呂布越添精神。）<ref>三英战吕布的不同版本有八種，

就此三個人圍住呂布不停廝殺，十八路人馬都看得呆了。不久，呂布開始感到力不从心，飛馬撤退，但三人追趕著呂布不肯捨，直至遇見董卓的兵馬才撤退。

==史實==
《三國志》有記載，斬華雄破呂布是孫堅所為，華雄是於陽人被斬殺，呂布之敗的原因是董卓派呂布與[[胡軫|胡軫]]來守，呂布與胡軫不和而發生惟亂，所以胡軫被孫堅擊退，後來孫堅與董卓開戰，董卓敗走，後來孫堅再進擊去洛陽宣揚城門，擊退呂布。

自從《演義》記載了「溫酒斬華雄」及「三英戰呂布」之後，此二事便廣泛流傳至今。
