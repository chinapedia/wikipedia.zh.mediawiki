'''帕杰罗'''（[[英文|英文]]：'''Mitsubishi Pajero'''）是一款由[[三菱|三菱]]汽车开发制造的运动型多功能越野车。帕杰罗的名称来源于阿根廷南部巴塔哥尼亚高原的一种山猫。在一些国家和地区也是用Montero和Shogun的名称。

== 发展历史 ==
帕杰罗最早的原型车出现在1973年11月的东京汽车展，1978年第二款原型车出现。1983年1月，帕杰罗首次出现在巴黎达喀尔拉力赛上。

===第一代（1982年-1991年）===
[[File:1986_Mitsubishi_Pajero_(NC)_Commercial_wagon_(2015-07-10)_01.jpg|thumb]]
第一代帕杰罗在1981年10月的东京汽车展上首次亮相，1982年正式发布。最初发布的是3门短轴版，提供金属硬顶和帆布软顶，以及3种不同引擎的选择。

推出时为求宣传，三菱于1983年参加第五届[[巴黎-达喀尔汽车拉力赛|巴黎-达喀尔汽车拉力赛]]，並派出了帕杰罗进行改装，以适合沙漠比赛。最后帕杰罗取得了优胜的成绩，从此帕杰罗一炮成名。

===第二代（1991年-1999年）===
[[File:1998_Mitsubishi_Pajero_(NL)_GLS_wagon_(2015-06-15)_01.jpg|thumb]]
第二代帕杰罗主要包括V31、V33、V43等几款主要车型，外形大抵跟第一代差不多，只是比较浑圆。同时第二代还引进了三菱自家的Super Select 4WD (SS4) 和多项模式的ABS系统。两者都是第一次于日本越野车上採用。

在1997年10月三菱还在第二代基础上改良，推出了帕杰罗进化版 (The Pajero Evolution)。三菱的用意是配合巴黎-达喀尔赛事的新要求。<br />

===第四代（2006年-2015年）===
[[File:Mitsubishi_Pajero_4th.JPG|thumb]]
第四代帕杰罗主要包括V83、V87、V88、V93、V97、V98几种车型。其中V83、V87是3门短轴版的，V93和V97是5门全尺寸版的。V93配备了6G72 V6 3.0L汽油发动机，V97配备了6G75 V6 3.8L汽油发动机。<br />

== 中国产版本和衍生车型 ==
[[File:Mitsubishi_Pajero_IV_LWB_facelift_China_2015-04-13.jpg|thumb]]
中国产帕杰罗是由三菱汽车和广汽长丰（原湖南长丰）合资生产，使用三菱标示，主要车型为V73。
衍生车型是由长丰汽车购买了三菱汽车的帕杰罗V31\V33生产线和流程工艺等要素，生产猎豹黑金刚车型，悬挂猎豹标示。
