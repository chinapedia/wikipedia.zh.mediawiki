'''Samsung Galaxy Note 3'''是一款由[[韓國|韓國]][[三星電子|三星電子]]所生產的一款建構於[[安卓|安卓]]平台的高端大螢幕平板式[[智慧型手機|智慧型手機]]（[[平板手機|平板手機]]），於2013年9月4日於[[德國|德國]]柏林[[柏林國際廣播展|IFA]]展上的三星產品發表會上發佈<ref name=verge-gn3>{{cite web|title=三星的Galaxy Note 3更大，更快，更薄，更輕，但它更好嗎？|url=http://www.theverge.com/2013/9/4/4693206/samsung-galaxy-note-3-specs-features-release-date-price|work=The Verge|accessdate=4 September 2013}}</ref>。同期的最大競爭對手是Apple [[iPhone_5|iPhone 5]]及[[HTC_One_(M7)|HTC One (M7)]]。

繼承上一代機型（[[Galaxy_Note_II|Galaxy Note II]]），Note 3被設計成一部更輕巧的手機，背殼是[[塑膠|塑膠]]材質的以及擁有皮質的觸感。

== 規格 ==

=== 硬件 ===
與之前的三星設備相比，Galaxy Note 3的設計旨在提供更高檔的“高級”外觀。儘管Galaxy Note 3 採用了[[聚碳酸酯|聚碳酸酯]]的設計，但與其他近期的三星設備相同，Galaxy Note 3具有[[縫紉|縫合]]人造金屬圈和塑料皮革製成的後蓋。厚度為8.3毫米，比 Galaxy Note II稍薄，並且稍微輕一些。

在Galaxy Note 3（N9005）的[[欧洲联盟|歐盟]]和北美銷售的[[長期演進技術|LTE]]版本使用2.3 GHz四核[[高通驍龍|Snapdragon 800]]芯片，而在上述地區以外銷售的僅限於[[GSM|GSM]]的型號（N9000 / N9006）使用八核[[三星Exynos|Exynos 5420]]由四核1.9 GHz [[ARM_Cortex-A15_MPCore|Cortex-A15]]內核和四核1.3 GHz [[ARM_Cortex-A7_MPCore|Cortex-A7]]內核組成。兩種模型的測試都表現出相似的性能。<ref>{{Cite news|url=https://www.phonearena.com/news/Galaxy-Note-3-fights-self-barely-wins-Exynos-5-Octa-vs-Snapdragon-800-versions-benchmarked_id47879|title=Galaxy Note 3與自我競爭，勉強獲勝：Exynos 5 Octa與Snapdragon 800版本進行了基準測試|newspaper=Phone Arena|accessdate=2018-04-29}}</ref>該設備還包括3 GB RAM，5.7英寸[[1080p|1080p]] [[AMOLED|Super AMOLED]]顯示屏，1300萬像素後置攝像頭，能夠以60 fps的速度拍攝1080p視頻和 [[4K解析度|4K分辨率]]，30 fps（5分鐘限制），16,32或64 GB內部存儲，以及3200 mAh電池。<ref name=":0">{{Cite news|url=https://www.theverge.com/2013/9/4/4693206/samsung-galaxy-note-3-specs-features-release-date-price|title=三星的Galaxy Note 3更大，更快，更薄，更輕，但它更好嗎？|newspaper=The Verge|accessdate=2018-04-29}}</ref><ref>{{Cite news|url=https://www.cnet.com/products/samsung-galaxy-note-3/|title=新 Note 3 的速度，功能大|date=2013-10-01|newspaper=CNET|accessdate=2018-04-29|language=en}}</ref>

Galaxy Note 3 Duos（N9002）基於LTE版本，但增加了第二張SIM卡插槽

與其他Galaxy Note系列設備一樣，Galaxy Note 3附帶S Pen [[数码绘图板|手寫筆]]。筆尖貼近屏幕寫<ref name=":0" /> <ref>{{Cite news|url=https://www.cnet.com/products/samsung-galaxy-note-3/|title=新 Note 3 的速度，功能大|date=2013-10-01|newspaper=CNET|accessdate=2018-04-29|language=en}}</ref>Galaxy Note 3也是第一款支持[[USB_3.0|USB 3.0]]的智能手機，它可以在連接到兼容端口時實現更快速的數據傳輸和充電。<ref>{{Cite news|url=https://www.theverge.com/2013/9/5/4696874/galaxy-note-3-usb-3-0-power-connector-explained|title=Galaxy Note 3首款手機具有醜陋的新型Micro USB 3端口|newspaper=The Verge|accessdate=2018-04-29}}</ref>

Galaxy Note 3首次推出黑色，白色和粉紅色。2013年12月，三星為特定市場推出了三種新色彩方案; 黑色飾有金色飾邊，白色飾有玫瑰金色飾邊，紅色飾有銀色飾邊。<ref>{{Cite news|url=https://www.pcmag.com/article2/0,2817,2427850,00.asp|title=Samsung 展示玫瑰金，梅洛紅色Galaxy Note 3|newspaper=PCMAG|accessdate=2018-04-29|language=en}}</ref>

=== 軟件 ===
Galaxy Note 3配備了[[Android|Android]] 4.3“Jelly Bean”和三星專有的[[TouchWiz|TouchWiz NatureUX 2.5]]用戶界面和軟件。Note 3的軟件中增加了手寫筆導向的功能; 從它的機抽出，用筆針按壓手寫筆按鈕手寫筆懸停在顯示激活的懸浮指令(Air Command) [[環形選單|環形選單]]提供快捷方式手寫筆導向功能，如[[便利貼|行動記事]]（屏幕上的便簽使用[[手写识别|手寫識別]]來檢測其內容並提供相關操作，例如在[[Google地图|Google地圖]]上查找地址和撥號電話號碼），Screen Write（註釋工具），Pen Window（允許用戶繪製彈出窗口來運行某些應用程序），搜索工具S Finder和Scrapbook。多窗口功能也通過擴展的應用程序支持進行更新，能夠運行單個應用程序的多個實例以及在應用程序之間[[拖放|拖放]]內容的功能。該設備還附帶一個稱為My Magazine 的[[聚合器|新聞聚合器]]應用程序，可通過從屏幕底部向上滑動以及S Note的更新版本進行訪問。<ref name=":0" /><ref>{{Cite news|url=https://www.engadget.com/2013/09/04/samsung-galaxy-note-3-officially-debuts-with-5-7-inch-screen-and/|title=Samsung Galaxy Note 3於9月25日正式亮相，配備5.7英寸1080p屏幕和人造皮革背面|newspaper=Engadget|accessdate=2018-04-29|language=en-US}}</ref>

2014年1月13日，[[Android|Android]] 4.4“KitKat”更新首次通過[[Samsung_Kies|Samsung Kies]]在[[波兰|波蘭]]的LTE型號提供。<ref>{{Cite news|url=https://www.engadget.com/2014/01/13/samsung-galaxy-note-3-begins-android-4-4-kitkat-rollout-in-polan/|title=Samsung Galaxy Note 3開始在波蘭推出Android 4.4 KitKat|newspaper=Engadget|accessdate=2018-04-29|language=en-US}}</ref>該更新增加了用戶界面調整，例如鎖定屏幕角落的相機快捷方式，用於設置默認啟動器和短信應用程序的選項，支持打印以及用於跟踪和控制位置使用的新位置設置菜單跟踪應用程序。<ref>{{Cite news|url=https://www.zdnet.com/article/kitkat-rolls-out-for-galaxy-s4-note-3-in-uk-and-us/|title=KitKat在英國和美國推出Galaxy S4，Note 3 |last=Tung|first=Liam|newspaper=ZDNet|accessdate=2018-04-29|language=en}}</ref><ref>{{Cite news|url=https://www.gsmarena.com/android_44_kitkat_now_seeding_on_korean_galaxy_s4_ltea-news-7754.php|title=Android 4.4 KitKat現在到更新韓國版本Galaxy S4 LTE-A|newspaper=GSMArena.com|accessdate=2018-04-29|language=en-US}}</ref><ref>{{Cite web|url=http://www.pocket-lint.com/phones/news/samsung/127845-what-s-new-in-the-samsung-galaxy-note-3-and-galaxy-s4-android-4-4-2-kitkat-update|title=Samsung Galaxy Note 3和Galaxy S4 Android 4.4.2 KitKat更新有哪些新功能？|accessdate=2018-04-29|work=www.pocket-lint.com|language=en}}</ref>[[Android|Android]] 4.4出於安全原因也對設備上的輔助存儲的處理做出重大改變; 應用程序對[[SD卡|SD卡]]的訪問訪問權限關的，特定於應用程序的目錄，而仍然允許對內部主存儲的完全訪問權限。雖然自從[[Android|Android]] 3.0“Honeycomb”以來，這種行為已經存在，但三星等原始設備製造商以前修改了Android的發行版以保留以前的行為，允許應用程序無限制地訪問SD卡內容。<ref>{{Cite news|url=https://www.anandtech.com/show/7859/examining-microsd-changes-in-android-44|title=在Android 4.4中檢查MicroSD更改|last=Ho|first=Joshua|accessdate=2018-04-29}}</ref>

三星于2015年1月開始推出[[Android|Android]] 5.0“Lollipop”更新。<ref>{{Cite news|url=https://www.gsmarena.com/samsung_galaxy_note_3_gets_lollipop_update_in_russia-news-10957.php|title=Samsung Galaxy Note 3在俄羅斯獲得Lollipop更新|newspaper=GSMArena.com|accessdate=2018-04-29|language=en-US}}</ref>

== 問題 ==
Galaxy Note 3的美國和歐洲模式在'''某些實施地區域'''[[SIM卡|SIM卡]][[區域限制|區域限制]]系統; 要求在歐洲和北美模式中使用的SIM卡來自該地區的運營商。雖然三星發言人聲稱，使用有效的SIM卡，鎖就會被移除，但用戶報告說情況並非如此。儘管[[XDA_Developers|XDA Developers]]用戶開發了一個清除系統使用的運營商[[黑名單|黑名單]]的工具，但它需要[[Root_(Android)|Root]]電話。一位發言人稱該系統主要是為了防止[[灰色市場|灰色市場]]轉售，雖然一些批評者懷疑運營商可能已經要求三星實施該功能，以便強制用戶在旅行時[[漫遊|漫遊]]，防止他們使用本地SIM卡。三星還表示，它對2013 年 7月之後生產的Galaxy S III，Note II，S4和S4 Mini型號實施了類似[[政策|政策]]。<ref>{{Cite news|url=https://www.theverge.com/2013/9/26/4772820/samsung-region-locks-european-galaxy-note-3-within-europe|title=三星地區鎖定note 3以避開經銷商|newspaper=The Verge|accessdate=2018-04-29}}</ref><ref>{{Cite web|url=https://gigaom.com/2013/10/07/app-rescues-galaxy-note-3-from-its-region-lock-but-samsungs-motives-remain-unclear/|title=應用程序從其區域鎖中拯救Galaxy Note 3，但三星的動機仍不清楚|accessdate=2018-04-29|date=2013-10-07|last=Meyer|first=David|work=gigaom.com|language=en-US}}</ref>

Note3的型號為：SM-N900（GSM），N9000（3G），SM-N9005（4G / LTE），N9002（LTE雙卡），SM-N9006（中國/ WCDMA），SM-N900A美國AT＆T），SM-N900T（美國T-mobile），SM-N900P（美國Sprint），SM-N900V（美國Verizon），SM-N900R4（美國數據服務），SM-N900W8（加拿大的一些運營商拉丁美洲）

== 爭議 ==
=== 效能測試作弊 ===
美國科技網站Ars Technica報導指出Note 3與[[LG_G2|LG G2]]使用相同的4核心處理器，但是在效能測試時Note 3的得分總是比G2高，背後原因是Note 3運行效能測試軟體Geekbench時，處理器的4個核心會同時啟動，並鎖定用最高速度運轉，但在正常使用情況下為節省電力，手機通常僅使用以節能模式運作的一個核心，其他三個則閒置，因此Note 3的效能分數未能反映正常使用情況的效能，而Note 3的正常結果應比原本的低20％。另一網站AnandTech亦指出Note 3在7個效能測試軟件當中有6個分數屬「作弊」<ref>[http://www.chinatimes.com/realtimenews/Galaxy-Note-3跑分作弊？其實幾乎所有Android廠商都在作弊啦！-20131005001474-260412 Galaxy Note 3跑分作弊？其實幾乎所有Android廠商都在作弊啦！] {{webarchive|url=https://web.archive.org/web/20141128014928/http://www.chinatimes.com/realtimenews/Galaxy-Note-3%E8%B7%91%E5%88%86%E4%BD%9C%E5%BC%8A%EF%BC%9F%E5%85%B6%E5%AF%A6%E5%B9%BE%E4%B9%8E%E6%89%80%E6%9C%89Android%E5%BB%A0%E5%95%86%E9%83%BD%E5%9C%A8%E4%BD%9C%E5%BC%8A%E5%95%A6%EF%BC%81-20131005001474-260412 |date=2014-11-28 }}</ref>

=== 手工差劣 ===
Note 3被曝出做工問題，部分Note 3的國外用家於網上抱怨自己的Note 3做工不結實，在使用一段時間後發現，手機的Home按鍵「有鬆動、並開始移位」。其中一張網民上載的[http://cdn.androidpolice.com/wp-content/uploads/2013/10/nexusae0_note3button_thumb.gif 圖片]顯示使用僅3個小時的Note 3的Home按鍵在手指四處按動時，Home按鍵也會跟著上下左右流動。在androidpolice進行的投票中，有2017位Note 3用戶反映自己的Home按鍵會位移，而僅1150名​​用戶沒有遇到這種情況<ref>[http://digital.sina.com.hk/news/-7-1424298/1.html 做工堪憂！Galaxy Note 3被曝Home鍵移位] {{webarchive|url=https://web.archive.org/web/20131010031506/http://digital.sina.com.hk/news/-7-1424298/1.html |date=2013-10-10 }}</ref>。
