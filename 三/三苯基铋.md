'''三苯基铋'''是一种[[金属有机化合物|金属有机化合物]]，[[化学式|化学式]]为Bi(C<sub>6</sub>H<sub>5</sub>)<sub>3</sub>，或简写为BiPh<sub>3</sub>。
==制备==
* 三苯基铋可以由[[氯化铋|氯化铋]](III)和[[格氏试剂|格氏试剂]]反应得到：<ref name=w /><ref name=synj />
::<math>\mathsf{C_6H_5Br + Mg \ \xrightarrow{}\ C_6H_5MgBr }</math>
::<math>\mathsf{BiCl_3 + 3C_6H_5MgBr \ \xrightarrow{}\ Bi(C_6H_5)_3 + 3MgBrCl }</math>

* 铋钠合金和[[氯苯|氯苯]]作用，可以得到三苯基铋：
::<math>\mathsf{Na_3Bi + 3C_6H_5Cl \ \xrightarrow{}\ Bi(C_6H_5)_3 + 3NaCl }</math>

* 通过氯化铋和[[二苯基汞|二苯基汞]]的反应也可以制备三苯基铋：
::<math>\mathsf{2BiCl_3 + 3Hg(C_6H_5)_2 \ \xrightarrow{}\ 2Bi(C_6H_5)_3 + 3HgCl_2 }</math>
==化学性质==
*三苯基铋在空气中稳定，可以和浓[[盐酸|盐酸]]反应：<ref name=w />
::<math>\mathsf{(C_6H_5)_3Bi + 3 HCl \ \xrightarrow {}\ 3C_6H_6 + BiCl_3 }</math>
*和[[硝酸|硝酸]]反应产生(C<sub>6</sub>H<sub>4</sub>NO<sub>2</sub>)<sub>3</sub>Bi(NO<sub>3</sub>)<sub>2</sub>，产物为淡黄色晶体。
*也可以和[[卤素|卤素]]反应：
::<math>\mathsf{(C_6H_5)_3Bi + Cl_2 \ \xrightarrow {}\ Bi(C_6H_5)_3Cl_2 }</math>
::<math>\mathsf{(C_6H_5)_3Bi + Br_2 \ \xrightarrow {}\ Bi(C_6H_5)_3Br_2 }</math>
::但碘只能产生(C<sub>6</sub>H<sub>5</sub>)BiI<sub>2</sub>和(C<sub>6</sub>H<sub>5</sub>)<sub>2</sub>BiI。
==用途==
三苯基铋可用作固化催化剂。<ref>鲁国林 夏强 杜娟.三苯基铋对高燃速丁羟推进剂的催化固化作用研究[J].《含能材料》.1999年02期</ref><ref name=synj>刘训恩 等.三苯基铋的合成研究及应用[J].《北京理工大学学报》.1992年S1期</ref>
