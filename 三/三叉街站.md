'''三叉街站'''位于[[福州市|福州市]][[仓山区|仓山区]]上三路、六一南路、则徐大道交叉口，是[[福州地铁|福州地铁]][[福州地铁1号线|1号线]]的一座中间站，2016年5月18日随1号线南段（三叉街-福州火车南站）开通试运营，曾为1号线北端终点站，于2017年1月6日北段开通后恢复为中间站。<ref name=":0">{{cite news|author=邱泉盛|url=http://news.fznews.com.cn/fuzhou/20160517/573a5f29de4d4.shtml|title=地铁1号线南段明天试运营 服务时间早7点至晚8点|publisher=[[福州晚报|福州晚报]]|date=2016-05-17|accessdate=2016-06-03}}</ref>

== 站名来源 ==
据《榕城考古略》记载，下藤路、埔头路、埔尾路三条路交叉于此，且往南前往白湖亭的路段店铺密集，后被当地居民习惯性称为“三叉街”。<ref>[http://mt.sohu.com/20160518/n450194481.shtml 地铁1号线 9站名起底]</ref>
== 车站结构 ==
三叉街站为地下二层车站，地下一层为站厅层；地下二层为站台层，岛式站台，全长483.5m，宽23.2~35.5m。<ref name=":1" />

=== 站厅 ===
{{ multiple image | align = right | direction = vertical | width = 220
| image1 = 三叉街站站厅-20160924 02.jpg | caption1 = 三叉街站站厅
| image2 = 20161126 Citizen corner of Sanchajie Station.jpg | caption2 = 车站市民角
}}
三叉街站站厅位于上三路与六一路交叉路口下方。车站站厅采用白色直线型吊顶与白色搪瓷板装修，设置客服中心2处，设有智能导乘机、自动售货机等服务设施，C出入口通道设有洗手间。站厅已预留3号线车站的换乘段，目前换乘段作为“市民角”使用。
==== 市民角 ====
车站付费区设有市民角，墙面设有少儿绘画展示板以及心愿柱用于展示收录的儿童画及市民建议；市民角中央设有福州地铁吉祥物“榕榕”立板供市民合影，亦设有舞台作为地铁公司活动举办场所；市民角旁设有意见箱与心愿箱，用于收集市民对福州地铁运营与发展的意见或建议。

=== 站台 ===
{{ multiple image | align = right | direction = vertical | width = 220
| image1 = 三叉街站站台-20160924.jpg | caption1 = 三叉街站站台
| image2 = 20161126 Platform of Sanchajie Station.jpg | caption2 = 处于南北贯通试运行时的站台

三叉街站共设置2个分别来往于[[象峰站|象峰]]、[[福州火车南站站|福州火车南站]]的1号线站台，采用岛式站台设计。车站站台采用白色直线型吊顶与白色搪瓷板装修，设有智能导乘机、自动售货机等服务设施。
==== 停靠模式 ====
* 2016年5月18日至2016年11月25日，三叉街站作为南段开通的北端临时终点站，列车停靠采用站前折返模式，即仅使用2号站台，1号站台不使用，作为备用停车轨道以便非运营列车临时停靠。
* 2016年11月25日至2016年11月25日，1号线南北段开始在运营时段贯通试运行，三叉街站恢复为中间站，往象峰方向列车使用2号站台，往福州火车南站方向列车使用1号站台，站台中央设置[[铁马_(路障)|隔离护栏]]引导乘客至2号站台候车，避免乘客误入尚未开通的试运行区段。但该期间偶尔有恢复南段初期的站前折返模式停靠，与北段试运行隔离。
* 2016年12月25日至2017年1月3日，1号线全线进行为期十天的试乘，三叉街站作为中间站使用，但由于南段正常试运营，在白湖亭站前往三叉街站时司机会通过广播提醒未参加试乘活动的乘客于三叉街站下车。
* 2017年1月6日起，1号线全线开通试运营，三叉街站作为中间站使用。

== 历史 ==
* 2011年4月29日，省发改委批复了福州地铁1号线（一期）的初步设计，其中在在则徐达道、三高路路口设'''三叉街站'''。<ref>
[http://www.fjdpc.gov.cn/zfshow.aspx?id=27917 福建省发展和改革委员会关于福州市轨道交通1号线一期工程初步设计的批复]{{Dead link|date=2019年2月 |bot=InternetArchiveBot |fix-attempted=yes }}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}
</ref>
* 2011年12月17日，三叉街站开始一期围挡施工，封闭了六一南路。<ref name=":1">
[http://digi.dnkb.com.cn/dnkb/html/2011-12/16/content_194699.htm 福州地铁三叉街站明晚开始围挡]{{Dead link|date=2019年2月 |bot=InternetArchiveBot |fix-attempted=yes }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}
</ref>
* 2012年5月5日，三叉街站开始二期围挡施工，占用了则徐大道由北至南机动车道及西侧部分人行道。<ref>
[http://digi.dnkb.com.cn/dnkb/html/2012-05/04/content_213088.htm 榕地铁三叉街站二期明晚围挡施工]{{Dead link|date=2019年2月 |bot=InternetArchiveBot |fix-attempted=yes }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}
</ref>
* 2013年3月29日，三叉街站开始三期围挡施工。<ref>[http://dzb.hxnews.com/2013-03/29/content_99133.htm 地铁三叉街站 今起三期围挡]</ref>
* 2014年9月14日，三叉街站开始四期围挡施工。<ref>[http://news.fznews.com.cn/fuzhou/2014-9-13/2014913DylJynDPvv7756.shtml 地铁1号线三叉街站今起四期围挡施工　至明年8月]</ref>
* 2015年8月13日，三叉街站开始五期围挡施工。<ref>
{{Cite web |url=http://mag.fznews.com.cn/shtml/fzwb/20150814/19960.shtml |title=地铁三叉街站实施五期一阶段施工围挡 |access-date=2016-11-25 |archive-url=https://web.archive.org/web/20161126003539/http://mag.fznews.com.cn/shtml/fzwb/20150814/19960.shtml |archive-date=2016-11-26 |dead-url=yes }} {{webarchive|url=https://web.archive.org/web/20161126003539/http://mag.fznews.com.cn/shtml/fzwb/20150814/19960.shtml |date=2016-11-26 }}
</ref>
* 2016年5月18日，三叉街站随地铁1号线南段开通而启用。<ref name=":0" />
* 2016年9月，三叉街站C出入口投入使用。
* 2016年11月26日，三叉街站启动南北贯通试运行，取消原先站前折返至2站台操作而采用列车继续停靠1站台，经清客后继续直接往北段试运行的措施。
* 2017年1月6日，三叉街站A出入口投入使用。

== 争议 ==
* 2016年5月18日三叉街站开通后，陆续有市民质疑“三叉街”的“叉”字读音应为“chā”而非地铁报站中的“chà”，随后三叉街街道办事处相关人员也证实了这一点，随后地铁公司在后续报站更新中将读音改正。<ref>[http://www.dnkb.com.cn/archive/info/20160805/110534316668255_1.shtml 脑补！福州三叉街站的“叉”读chā还是chà？]</ref>
