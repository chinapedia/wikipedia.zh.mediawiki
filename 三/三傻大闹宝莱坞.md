《'''三個傻瓜'''》（{{lang-en|'''''3 Idiots'''''}}）是2009年一部[[宝莱坞|宝莱坞]]喜剧影片，改编自[[印度|印度]]畅销书作家{{le|奇坦·巴哈特|Chetan Bhagat}}的处女作小说《{{le|五分生|Five Point Someone}}》（Five Point Someone），由[[拉库马·希拉尼|拉库马·希拉尼]]执导。该片一上映就打破了印度电影的票房纪录，创下了[[宝莱坞|宝莱坞]]电影上映首週、上映第一天和全球票房最高纪录；至今位列印度电影本土票房第三位、海外票房第二位和全球票房第三位。

== 剧情 ==
法罕（[[马德哈万|马德哈万]]饰）、拉加（[[沙曼·乔希|沙曼·乔希]]饰）与藍丘（[[阿米尔·汗|阿米尔·汗]]饰）進入了[[印度|印度]]首屈一指的帝国理工大学（ICE，以[[印度理工學院|印度理工學院]]為藍本），三人共居一室，结为好友。在成绩决定一切的学校裡，藍丘鼓勵法罕追求理想，去當一名野生动物摄影师，鼓励拉加放下思想包袱，做真实的自己，还劝说韋魯博士的二女儿佩雅（[[卡琳娜·卡浦尔|卡琳娜·卡浦尔]]饰）离开满眼铜臭的未婚夫蘇哈斯。

為了說服拉加不要像死背硬記學習的模范学生查托一樣偏執，藍丘和法罕趁機竄改了大會的演講稿，而查托正是大會演講的推薦獻詞人，因被竄改演講稿而得罪教育部的重要來賓還有韋魯博士，引起對他們的不满。查托於是跟藍丘打赌，十年後的今日再來比較誰的人生更成功，故事的開頭即是以此展開倒敘，開始了三人尋找藍丘的旅程。

== 创作 ==
2008年7月28日开拍。拍摄地有[[德里|德里]]、[[班加罗尔|班加罗尔]]、[[孟买|孟买]]、[[拉達克|拉達克]]、[[Chail|Chail]]和[[西姆拉|西姆拉]]。Aamir等几位主演于2008年9月上旬进剧组，于同年12月完成拍摄<ref>{{cite web|url=http://www.indiafm.com/news/2008/07/28/11730/index.html |title=Rajkumar Hirani's 3 Idiots goes on floors |publisher=indiafm.com |deadurl=yes |archiveurl=https://web.archive.org/web/20080804233117/http://www.indiafm.com/news/2008/07/28/11730/index.html |archivedate=2008-08-04 }}</ref><ref>{{cite web |url=http://ia.rediff.com/movies/2008/aug/28look1.htm |title=Shooting for 3 Idiots underway |publisher=rediff.com}}</ref>。主要拍摄场地有[[印度管理研究所邦加羅爾分校|印度管理研究所邦加羅爾分校]]，剧组在此花了33天<ref>{{cite web |url=http://www.thaindian.com/newsportal/uncategorized/shooting-three-idiots-is-like-holidaying-for-aamir-khan_100143291.html |title=Shooting ‘Three Idiots’ is like holidaying |publisher=thaindian.com}}</ref>。其中有一幕在[[印度理工學院|印度理工學院]]孟買校區拍摄。

*台灣部份，上映四個月全台票房1600萬<REF>[http://ent.ltn.com.tw/news/paper/485786 三個傻瓜登台長跑，熱映破4個月] 2011/04/20 自由時報</REF>，[[台北市|台北]]票房最終獲得新台幣1,518萬元<ref>[https://web.archive.org/web/20121028171300/http://www.atmovies.com.tw/movie/film_f1in41187043_now.html 三個傻瓜台北票房] 開眼{{zh}}</ref>好成績，映期更是成為台灣電影史上第二長映期，從2010年12月17日在台北市上映，直到2011年7月7日才正式下片，一共長達6個月又28天。僅次於《[[侏儸紀公園|侏儸紀公園]]》的8個月又29天，第三長映期為《[[鐵達尼號|鐵達尼號]]》的6個月又21天。直到2017年電影《[[我和我的冠軍女兒|我和我的冠軍女兒]]》上映12天獲得$3000萬元新台幣票房，打破印度在台電影票房《三個傻瓜》7年冠軍紀錄<REF>[http://www.cna.com.tw/news/amov/201704050420-1.aspx 我和我的冠軍女兒 全台票房破3千萬],2017.4.5 中央通訊社</REF>。

== 穿帮镜头 ==
1.影片開頭起飛的飛機外景拍攝時是[[空中巴士|空中巴士]]的A321型飛機（[[窄體客機|單通道窄體]]），而拍攝艙內時看到的明顯是雙通道的[[寬體客機|寬體客機]]，駕駛艙亦為波音777的駕駛艙，電門及燃油幫浦均為關閉。

2.Farhan將假的骨灰罐的蓋子扔到了馬桶裡，可是後來真正Rancho送走Farhan一行人的時候手中的骨灰罐是有蓋子的。虽蓋子沒被沖走所以可能撈起來，但片中未有相关镜头且Rancho手中骨灰罐的盖子并不是湿的。不過由於鏡頭不連貫，有可能中途抹過。

3.六年前（2003年）Rancho為Pia的姐姐接生的時候，與Pia進行了視訊通話，其中Pia還放映了在Youtube上的影片，但是Youtube是在2005年誕生的。

4.在2小時40分16秒介紹Raju的部落格時，左上角卻有“資源回收筒”的圖示，顯然部落格只是一張桌面。

5.在影片2小時42分57秒處，在Rancho和Pia親吻完的一剎那，鏡頭轉到了帳篷那一側。雖然由於背景被非焦點化，但依然可以看出是一排人站在那裡，有兩個人頭上似乎戴著红綢。由於前面的鏡頭中孩子們曾離開鏡頭範圍，可能是將劇組人員拍攝入了鏡頭中。但此幕開段遠攝時曾拍到帳篷，有可能眾人在此露營，也有可能的確是孩子們。
