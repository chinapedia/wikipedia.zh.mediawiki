'''三视教育運動'''是[[中华人民共和国|中华人民共和国]]建政初期，在[[朝鲜战争|朝鲜战争]]中为了肃清社会上存在的“親美、崇美、恐美”思想，面向全国展开的以“'''仇视'''”、“'''鄙视'''”及“'''蔑视'''”[[美帝國主義|美帝國主義]]的[[政治|政治]][[思想工作|思想教育]]运动。

== 历史背景 ==
1950年6月25日，朝鲜战争爆发，[[中共中央|中共中央]]高层決定進入[[朝鲜民主主义人民共和国|北韩]]参战之前，存在意见分歧。在10月4日下午召开的[[中共中央政治局扩大会议|中共中央政治局扩大会议]]上，“多数人不赞成出兵或者对出兵存有种种疑虑”<ref>{{cite book |author=逄先知 |coauthors=李捷 |title=《毛泽东与抗美援朝》 |year=2000-10-01 |publisher=中央文献出版社|isbn=9787507308921 }}</ref>。朝鲜战争的爆发尤其是[[美国|美国]]的出兵朝鲜, 中国民众“思想认识很不一致”<ref>{{cite book |author= 侯松涛 |title= 抗美援朝运动与民众社会心态研究 |publisher= 中共党史研究 |year= 2005年第2期 |location= 第19页}}</ref>，“曾引起各阶层群众一定的思想波动和混乱”<ref>{{cite book |author=《当代中国》丛书编辑部编辑 |title=《当代中国的山东》上册 |year=1989年 |publisher=中国社会科学出版社 |location=第103页 }}</ref>。

而当时中国刚经历过[[中国抗日战争|抗日战争]]及[[国共内战|国共内战]]，社會上存在許多“親美、崇美、恐美”的思想<ref>{{cite book |author=侯松涛 |title=抗美援朝运动中的"三视"教育---宏观视角下的回顾与反思 |year=2007年第6期 |publisher=《党史研究与教学》 }}</ref>。“恐美”情绪在多数民众的心理中普遍存在，并对中国介入朝鲜战争持悲观情绪，认为“美国经济实力强大、武器精良，而[[解放军|解放军]]是土包子，没法跟美国打，[[志愿军|志愿军]]赴朝参战是惹火烧身”<ref>{{cite book |author=当代中国丛书编辑部编 |title=《当代中国的山东》上册 |year=1989 年版 |publisher=中国社会科学出版社 |location=第 103 页 }}</ref>。“崇美”和“亲美”心理主要是在当时的[[民族资产阶级|民族资产阶级]]、上层知识分子和部分学生中比较普遍。据当时的调查，“在一部分[[教授|教授]]、[[资本家|资本家]]、[[信徒|教徒]]和[[教会学校|教会学校]]中仍存在着浓厚的亲美思想, 需要用极大的努力来肃清”<ref>{{cite book |author=中共北京市委政研室 |title=北京市抗美援朝运动报告(一) |year=1950年11月5日 |publisher=《北京工作》第8期 }}</ref>；“在教授和工商界中, 很多人还有崇美、恐美心理或对[[抗美援朝|抗美援朝]]的军事行动口是心非”<ref>{{cite book |author=中共北京市委政研室 |title=北京市抗美援朝运动报告(二) |year=1950年11月12日 |publisher=《北京工作》第8期 }}</ref>。

这些因素不利于调动社会资源参战，而且从[[共产主义国家|共产主义国家]]意识建构的角度来看，在[[二元对立|二元对立]]的结构中确立一個对立面是必需的，因此这样的思想必须肃清<ref>{{cite book |author= 萨支山 |title= 新爱国主义与国家意识之建立——以抗美援朝爱国运动为例 |publisher= http://cc.shu.edu.tw/~atss1/Seminar/20120905/F1/F1-1.pdf}}</ref>。但是即使沒有朝鮮战争，肃清美国对中国的影响问题，[[中国共产党|中共]]也是非常重视的<ref>{{cite book |author= 萨支山 |title= 新爱国主义与国家意识之建立——以抗美援朝爱国运动为例 |publisher= http://cc.shu.edu.tw/~atss1/Seminar/20120905/F1/F1-1.pdf}}</ref>。1949年[[美国国务院|美国国务院]]发表[[中美關係白皮書|中美關係白皮書]]，[[毛泽东|毛泽东]]就少见地对此接连写了数篇文章批驳、回应<ref>这几篇文章分别是：《丢掉幻想，准备战斗》、《别了，司徒雷登》、《为什么要讨论白皮书？》、《“友谊”，还是侵略？》</ref>。而通过抗美援朝运动，逐步肃清“亲美”、“崇美”、“恐美”的思想后，[[周恩来|周恩来]]就认为“这更是一个无价的收获”，因为“这种侵略毒素不是一天侵入的，是长期地不知不觉迷惑、麻醉侵入的”<ref>{{cite book |author=周恩来 |title=《周恩来军事文选》（第四卷）《巩固和加强国防力量是头等重要任务》 |year=1997 年 |publisher=北京：人民出版社 |location=第231-232页  }}</ref>。可见其意义是远在单纯的战争动员之上的<ref>{{cite book |author= 萨支山 |title= 新爱国主义与国家意识之建立——以抗美援朝爱国运动为例 |publisher= http://cc.shu.edu.tw/~atss1/Seminar/20120905/F1/F1-1.pdf}}</ref>。

== 内容 ==
“仇美、鄙美、蔑美”是“三视”宣传教育中最重要的內容<ref>{{cite book |author= 萨支山 |title= 新爱国主义与国家意识之建立——以抗美援朝爱国运动为例 |publisher= http://cc.shu.edu.tw/~atss1/Seminar/20120905/F1/F1-1.pdf}}</ref>。为了配合宣传教育运动的展开，中共宣传部门专门编写了一本名为《怎样认识美国（宣传提纲）》的小册子，其主要论述逻辑分为三个部分：

第一部分是“仇视美国，因为它是中国人民的死敌”，从“目前”到历史，共列数十条事例说明美国对中国的侵略。

第二部分是“鄙视美国，因为它是腐朽的[[帝国主义|帝国主义]][[国家|国家]]，是全世界[[反动|反动]]墮落的大本营”，指出美国被少数资本家控制，对內政策是压迫和剥削人民，绞杀[[民主|民主]]，绞杀文化；对外则是侵略和战争的政策。

第三部分是“蔑视美国，因为它是[[纸老虎|纸老虎]]，是完全可以打败的”，这是因为它在政治上孤立，在军事上也存在战线过长、兵力不足等弱点，虽然有[[原子弹|原子弹]]，但並不可怕等等<ref>{{cite book |author=中国人民抗美援朝总会宣传部编 |title=《伟大的抗美援朝运动》《怎样认识美国（宣传提纲）》（1950年11月5日）|year=1954年 |publisher=北京：人民出版社 }}</ref>。

== 宣传及推广形式 ==
1950年[[中国人民志愿军|中国人民志愿军]]参战后不久，[[中共中央|中共中央]]就发出关于时事宣传的指示，提出“为了使全体人民正确地认识当前形势，确立胜利信心，消灭恐美心理，各地应即展开关于时事的宣传运动”<ref>{{cite book |author=中国人民解放军国防大学党史党建政工教研室编 |title= 《中共党史教学参考资料》第19册 |publisher= 国防大学出版社 |year= 1986 |location= 第211页}}</ref>。1950年11月4日《各[[民主党派|民主党派]]联合宣言》发表后，“三视”教育运动宣传攻势正式在中国大陆范围内普遍展开<ref name="侯松涛 2005年第期">{{cite book |author= 侯松涛 |title= 抗美援朝运动与民众社会心态研究 |publisher= 中共党史研究 |year= 2005年第期 |location= 第22页}}</ref>。

通过新闻媒体传输一定量的教育信息是宣传教育的重要形式。从1950年11月到1953年，关于[[朝鲜战争|朝鲜战争]]的宣传成了中国新闻机构的中心主题。《[[人民日报|人民日报]]》、[[新华社|新华社]]、[[中央人民广播电台|中央人民广播电台]]等媒体通过发表相关[[社论|社论]]、评论，特辟专刊等方式对战争进行宣传<ref>{{cite book |author= 侯松涛 |title= 抗美援朝运动与民众社会心态研究|publisher= 中共党史研究第二期 |year= 2005年 |location= 第22页}}</ref>。据统计，从1951年到1954年，《人民日报》关于朝鲜战争的宣传文字平均每月就达两万字左右<ref>{{cite book |author= 张涛 |title= 《中华人民共和国新闻史》|publisher= 经济日报出版社 |year= 1996年 |location= 第61页}}</ref>。同时，[[中国共产党|中国共产党]]着手建立了宣传员和报告员制度, 在共产党的每一个支部设立宣传员，并在共产党的各级领导机关设立报告员，以配合新闻媒体进行时事宣传<ref name="侯松涛 2005年第期"/>。此种宣传员报告员制度，实际上成为新闻媒介的重要辅助和补充。

通过各种会议及控诉会，以“群众亲历、亲闻、亲见的活生生事实，揭露美帝国主义的残暴罪行，使他们对美帝国主义的理性认识与自己的真情实感相结合，树立起深刻的仇美思想”<ref>{{cite book |author= 萨支山 |title= 新爱国主义与国家意识之建立——以抗美援朝爱国运动为例 |publisher= http://cc.shu.edu.tw/~atss1/Seminar/20120905/F1/F1-1.pdf}}</ref>。1950年12月中共就要求“在工厂、机关、学校、街道、团体、里弄举行集会，控诉美帝国主义的罪恶，批判亲美、崇美、恐美的错误思想”<ref>{{cite book  |title= 《继续扩大与深入抗美援朝保家卫国运动》|publisher= 人民日报 |year= 1950年12月28日}}</ref>。

组织不同阶层民众在不同地区进行[[遊行|遊行]]也是“三视”教育运动推广的主要手段。1950年11月30日，[[天津市|天津市]]4万多[[工商业者|工商业者]]举行了“抗美援朝”示威游行大会，并在会后致电毛泽东，称他们“已经认清了美国的侵略本质，表示要与各阶层人民团结一致”<ref>{{cite book |author= 侯松涛 |title= 抗美援朝运动与民众社会心态研究 |publisher= 中共党史研究 |year= 2005年第2期 |location= 第23页}}</ref>，“为完成抗美援朝保家卫国的神圣任务而奋斗到底”<ref>{{cite book |author= 《当代中国》丛书编辑部编辑 |title= 抗美援朝战争 |publisher= 中国社会科学出版社 |year= 1990年 |location= 第343页}}</ref>。1950年12月26日，上海工商界举行了有15万人参加的示威游行。其他如[[北京|北京]]、[[哈尔滨|哈尔滨]]、[[青岛|青岛]]、[[西安|西安]]、[[武汉|武汉]]等城市的工商业者也在1950年12月上半月相继举行了数万人至十几万人的示威游行。在有各界民众参加的集会游行中，1951年“[[五一|五一]]” 节期间举行的示威游行规模最大，参加的各阶层民众共有18643万余人，占了全国总人口的39%以上，如果加上“五一”节前参加游行的人数，全国就有一半以上的人口参加了游行 <ref>{{cite book |author= 廖盖隆 |title= 《爱国运动论集》|publisher= 新华书店 |year= 1951年 |location= 第17页}}</ref>。

艺术性宣传是“三视”教育运动推广的另外一种手段。通过通俗性的大众艺术如[[杂文|杂文]]、[[漫画|漫画]]、戏剧、歌曲等，调动起群体的情感共鸣从而提高凝聚力，并激发群体强烈义愤。1950年11月，[[中华全国文学艺术界联合会|中华全国文学艺术界联合会]]发出《关于文艺界展开[[抗美援朝|抗美援朝]]宣传工作的号召》，号召[[文联|文联]]所属各分会及各地文艺组织一致行动起来，以文艺形式“揭露美帝国主义的罪行”、“肃清帝国主义流毒对一部分人的影响”<ref>{{cite book |author= 侯松涛 |title= 抗美援朝运动与民众社会心态研究 |publisher= 中共党史研究 |year= 2005年第期 |location= 第24页}}</ref>。中国大陆各地的文艺性刊物和各地的主要报纸，都以大量的版面，先后刊登了成千上万关于朝鲜战争的文艺作品和指导性的文章。据[[上海市|上海市]]1950年11月3日至12月5日一个月零两天的统计，已经发表和演出的文艺作品就达890篇。全国各地的出版社出版了大量的朝鲜战争题材的文艺丛书。1950年11月到1951年3月仅4个月出版的有关丛书就达数十种之多<ref>{{cite book |author= 刘宏煊 |title= 抗美援朝研究 |publisher= 人民出版社 |year= 1990年 |location= 第370页}}</ref>。

== 对[[中国基督教|中国基督教]]的影响 ==
[[天主教|天主教]]和[[新教|新教]]在中国有着深厚的影响，当时大约有400万信众，其中新教团体多与美国有着密切的联系，受美国教会资助、控制，而他们举办的一些慈善、教育事业，对开展仇视美国的思想教育运动来说，又具有极大的“迷惑性”<ref>{{cite book |author= 萨支山 |title= 新爱国主义与国家意识之建立——以抗美援朝爱国运动为例 |publisher= http://cc.shu.edu.tw/~atss1/Seminar/20120905/F1/F1-1.pdf}}</ref>。如何处理它们，成为“三视”教育的一个重点。在中共的推动下，1950年9月，基督教方面[[吴耀忠|吴耀忠]]等人推出《中国基督教在新中国建设中努力的途径的宣言》，12月，[[四川|四川]][[广元|广元]]天主教[[王良佐|王良佐]]等发表《自立革新运动宣言》，号召教徒切断与帝国主义国家的联系，实行“自治、自养、自传”（[[三自|三自]]运动）。朝鲜战争开战后，宣传重点转为对外国[[传教士|传教士]]及从事间谍的[[神职人员|神职人员]]的控诉。其中比较有影响的是教会所办的[[育婴堂|育婴堂]]的虐婴事件的控诉。其中一个是[[广州圣婴育婴院|广州圣婴育婴院]]的“血案”，《人民日报》指控其是在“[[慈善|慈善]]事业”、“[[宗教|宗教]]事业”掩护下的“杀人场”<ref>{{cite news  |title= 打碎美国伪善的“救济”招牌|publisher= 人民日报 |date= |accessdate=1951-04-27 }}</ref>。因由对教会育婴堂的控诉具有良好的效果，1951年全国各地就发掘出许多类似的事件，大量关于育婴堂的罪行被控诉，而且其控诉的步骤也基本相同。对他们的指控，更容易激起仇恨，而且，这种仇恨很自然地就引向了美帝国主义并同时激起爱国热情。这是“三视”教育中最为成功的地方<ref>{{cite book |author= 萨支山 |title= 新爱国主义与国家意识之建立——以抗美援朝爱国运动为例 |publisher= http://cc.shu.edu.tw/~atss1/Seminar/20120905/F1/F1-1.pdf}}</ref>。

==影响及评价==
“三视”教育运动重新塑造了[[中国大陆|中国大陆]]民众对于美国的认知。美国在许多中国大陆人心中的形象，由抗日战争时期的民主、文明、对中国友好的国家，成为了“全世界[[反动|反动]]堕落的大本营”<ref>{{cite book |author=杨玉圣 |title= 《中国人的美国观》 |publisher= 复旦大学出版社 |year= 1996 |location= 第235页}}</ref>。

其一，美国是中国人民的死敌，“是打劫起家的两头蛇”，“是中国人民的[[吸血鬼|吸血鬼]]”<ref>{{cite book |author=《北京工作》第8期 |title= 《北京市抗美援朝运动报告(一)》》(1950年11月5日) |publisher= 中共北京市委政研室 |year= 1950}}</ref>，美帝的侵略是“伪善”的、“杀人不见血的侵略”<ref name="中共中央中南局政策研究室编印 1951年">{{cite book |author=中共中央中南局政策研究室编印 |title=《武汉市委关于抗美援朝以来资产阶级动态及统战工作情况的报告》(1950年10月23日) |year=1951年 |publisher=《城市资料》第3期 |location=第33页 }}</ref>，因此，美国“是我们中国人民最危险的敌人，最凶恶的敌人”；

其二，“作为帝国主义侵略阵营的魁首”，美国是“世界上最反动的、最野蛮的、最富于侵略性的帝国主义国家”，是“屠杀人类的老窝”，因此，它也是“全世界爱好和平[[民主|民主]]人民的最凶恶的敌人”<ref>{{cite book |author= 曲爱国 |title=《抗美援朝战争纵横谈——访柒成文将军问答录》 |year=1992年 |publisher=《人物》第5期 }}</ref>，是“世界的反动中心”；

其三，美国是最为腐朽的帝国主义国家，是“[[特务|特务]]、[[流氓|流氓]]、[[抢劫|强盗]]的[[国家|国家]]”<ref name="中共中央中南局政策研究室编印 1951年"/>，美国是“狗的天堂”、“人的[[地狱|地狱]]”、“疯子的天下”。美国人民生活贫困、流离失所，不愿作战<ref>{{cite book |author=李维汉 |title= 《回忆与研究》下 |publisher= 中共党史资料出版 |year= 1986 |location= 第715页～第717页}}</ref>，美国的科学研究正在走向末路<ref>{{cite book |author= 曲爱国 |title=《抗美援朝战争纵横谈——访柒成文将军问答录》 |year=1992年 |publisher=《人物》第5期}}</ref>。

“三视”教育运动所形成的观念，潜移默化地影响中国人。1951年7月1日朝鲜停战谈判消息传到北京后，“很多群众对媒体中称[[马修·李奇微|李奇微]]为‘将军’、为‘[[联合国军|联合国军]]总司令’甚为不满”<ref>{{cite book |author=中共中央宣传部秘书处编印 |title=《北京及其他地区各阶层人民对朝鲜停战谈判的反映》 |year=1951年7月 |publisher=《宣传通讯》 |location=第6页 }}</ref><ref>{{cite book |title=《北京市抗美援朝运动资料汇编》 |year=1993年版 |publisher=知识出版社 |location=第156页 }}</ref><ref>{{cite book |author=中共北京市委党史研究室编 |title=《北京市各阶层人民对朝鲜停战谈判的反应》 |year=1951年7月 |publisher=中共北京市委向毛主席、中央、华北局的报告}}</ref>，因为据“三视”教育中接受的观念，“李奇微乃是美帝国主义的代表，又怎能尊称其为[[军衔|将军]]呢？”新闻稿件和发言稿中，到处都是匪类、帝国主义、恶魔、[[法西斯|法西斯]]等“刺激性的词语”，1952年5月周恩来曾指示要“避免或少用不必要的刺激性语句”<ref>{{cite book |author=中共中央文献研究室编 |title=《周恩来年谱(一九四九—一九七六)》上卷 |location=第238-239页 }}</ref>。直到1972年，中国政府宣布[[尼克松|尼克松]]访华的消息，中国的普通群众“在感情上一时无法接受”<ref>参见何妍：《20世纪60年代中国媒体宣传中的美、苏形象——以<人民日报>为个案》，2005年香港中文大学第一届国际博士研究生研讨班论文。</ref>，当他们在报刊上看到[[毛泽东|毛泽东]]与“美帝国主义头子”[[理查德·尼克松|尼克松]]“握手言和”的时候，他们普遍感到惊愕，以致目瞪口呆<ref>{{cite book |author=杨玉圣 |title= 《中国人的美国观》 |publisher= 复旦大学出版社 |year= 1996 |location= 第238页,第258页|ISBN= 9787309016031}}</ref>。

“三视”教育运动通过新闻媒体的宣传教育及组织各种会议与集会等方式，成功地激起并引导了大陆民众对美国的仇恨情绪。以“仇美”、“[[反美|反美]]”为核心的对美[[意识形态|意识形态]]形成并影响了此后数代中国大陆的人们<ref>{{cite news |author=洪振快 |title=洪振快：历史如何让亲美变成仇美|url=http://comments.caijing.com.cn/2013-03-27/112624699.html |publisher=财经网 |date= |accessdate=2013-03-27 }}</ref>。這影響在1967年香港暴動時亦被領導者用於鼓動參與者對抗港英政府的統治<ref name="singtao">{{cite book |chapterurl=http://lib-nt2.hkbu.edu.hk/sca_fb/book_pub/clippingindex1/data/assets/basic-html/page83.html |chapter=〈中共對港虛聲恫嚇. 號召左派份子展開三視運動〉，《星島日報》，1967年6月11日。|title=港澳問題剪報索引 (1950-1973)|page=83 |language=zh-hant }}</ref><ref>{{cite news |url=http://www.hk1967.org/thoseyears.asp?this=6706 |title=1967年6月6日, 《文匯報》社論號召以「三視運動」(仇視、鄙視、蔑視)反擊「緊急法令」。}}</ref>。
