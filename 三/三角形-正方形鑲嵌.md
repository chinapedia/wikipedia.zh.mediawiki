== 二階正方形-三角形鑲嵌 ==
二階正方形-三角形鑲嵌是一種[[複合正多邊形密鋪|複合正多邊形密鋪]]<ref name="圖解數學辭典">《圖解數學辭典》天下遠見出版 [[複合正多邊形密鋪|複合正多邊形密鋪]] ISBN 986-417-614-5</ref>，其為Krötenheerdt提出的較有系統的[[不均勻半正鑲嵌圖|不均勻半正鑲嵌圖]]之一<ref name = "Krötenheerdt">Krötenheerdt, O. "Die homogenen Mosaike n-ter Ordnung in der euklidischen Ebene. I." Wiss. Z. Martin-Luther-Univ. Halle-Wittenberg, Math.-Natur. Reihe 18, 273-290, 1969.</ref><ref name = "Grünbaum">Grünbaum, B. and Shephard, G. C. Tilings and Patterns. New York: W. H. Freeman, 1986.</ref>。其中二階並非是[[二階無限邊形鑲嵌|二階鑲嵌]]的二階，而是指該鑲嵌是由二排正方形和一排三角形交錯排列而成。

二階正方形-三角形鑲嵌是一種異扭稜正方形鑲嵌變體，又稱異扭稜正方形柱鑲嵌，因為它可以當作異扭稜正方形鑲嵌拆開後加入無限角柱。
=== 對偶鑲嵌 ===
二階正方形-三角形鑲嵌的對偶鑲嵌是正方形-柱形五邊形鑲嵌，也可以視為柱形五邊形鑲嵌的變體，又稱異扭稜正方形柱鑲嵌柱形五邊形柱鑲嵌，因為它可以當作柱形五邊形鑲嵌拆開後加入無限角柱。

二階正方形-柱形五邊形鑲嵌是一種柱形五邊形鑲嵌的變體。

正方形-二階三角形鑲嵌是一種[[複合正多邊形密鋪|複合正多邊形密鋪]]<ref name="圖解數學辭典"/>，其為Krötenheerdt提出的較有系統的[[不均勻半正鑲嵌圖|不均勻半正鑲嵌圖]]之一<ref name = "Krötenheerdt"/><ref name = "Grünbaum"/>。其中二階並非是[[二階無限邊形鑲嵌|二階鑲嵌]]的二階，而是指該鑲嵌是由一排正方形和二排三角形交錯排列而成。

正方形-二階三角形鑲嵌是一種異扭稜正方形鑲嵌變體，又稱異扭稜正方形反柱鑲嵌，因為它可以當作異扭稜正方形鑲嵌拆開後加入無限角反柱。

正方形-二階三角形鑲嵌與二階正方形-三角形鑲嵌相似，只差再重複的重負擔圓從正方形變為正三角形。此類鑲嵌也可以視為正方形鑲嵌的局部與正三角形鑲嵌的局部交錯組合。
=== 對偶鑲嵌 ===
正方形-二階三角形鑲嵌的對偶鑲嵌是六邊形-柱形五邊形鑲嵌，也可以視為柱形五邊形鑲嵌的變體。
