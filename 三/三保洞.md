'''三保洞'''（{{lang-id|Klenteng Sam Po Kong}}），也称'''三宝洞'''或'''三宝庙'''，是[[印度尼西亚|印度尼西亚]][[三宝垄|三宝垄]]最古老的中国寺庙建筑群，最初由[[明代|明代]][[航海家|航海家]][[郑和|郑和]]建立。尽管其最初建立时是为了[[伊斯兰教|伊斯兰教]]的[[礼拜|礼拜]]使用，现在[[佛教|佛教]]等其他宗教的活动也会在此地举行，其使用者也包括[[汉族|汉族]]和[[爪哇族|爪哇族]]等多个种族{{r|Sudiarno 2001, Sam Poo Kong}}{{r|Suherdjoko 2005, Temple gears up}}。

==历史==

三保洞最初建立的具体年代存在争议，大致处于公元1400年到1416年之间{{r|Sudiarno 2001, Sam Poo Kong}}{{r|Setiakusuma2006|page=57}}，但一般认为其是由明代航海家[[郑和|郑和]]通过[[卡琅河|卡琅河]]到达今天的三宝垄西部时建立的{{r|Sudiarno 2001, Sam Poo Kong}}{{r|印尼僑聲雜誌}}{{r|Suherdjoko 2005, Sam Poo Kong}}。在郑和的船队登陆后，作为[[穆斯林|穆斯林]]的郑和在一个石山附近找到一个洞穴并以其作为礼拜的场所使用{{r|Sudiarno 2001, Sam Poo Kong}}。他在离开[[爪哇岛|爪哇岛]]之前建立了一个小的庙宇，同时有说法认为其出于对这片地方的喜爱留下了他的副手[[王景弘|王景弘]]{{r|Sudiarno 2001, Sam Poo Kong}}{{r|Setiakusuma2006|p=57}}{{r|印尼僑聲雜誌}}。洞中安放了一尊郑和雕像{{r|Setiakusuma2006|page=59}}。据记载，郑和建立的庙宇在1704年的一次山崩中被毁{{r|Sudiarno 2001, Sam Poo Kong}}{{r|Setiakusuma2006|page=59}}，并于1724年重建完成{{r|Sudiarno 2001, Sam Poo Kong}}。重建包括新的洞穴，和原有的洞穴相邻{{r|Sudiarno 2001, Sam Poo Kong}}。

18世纪中期，[[犹太裔|犹太裔]]地主约翰内斯（{{lang-en|Johanes}}）取得了三保洞及其附近的土地的所有权，开始向祭祀者收取费用{{r|Setiakusuma2006|page=61}}{{r|Suherdjoko 2005, Semarang set for Sam Po}}。由于费用过高，华人社团只好每年支付2000[[荷兰盾|荷兰盾]]来进行祭祀；这一数额后来通过还价降低到了500盾{{r|Setiakusuma2006|page=61}}{{r|印尼僑聲雜誌}}。由于这一数额仍然过高，华人社团转而祭祀供奉于{{convert|5|km|mi}}外的大觉寺中的保生大帝雕像，并称之为“小三宝”{{r|印尼僑聲雜誌}}。其后，当地华人又集资从中国引进了另一尊郑和雕像，并供奉于大觉寺中{{r|Suherdjoko 2005, Semarang set for Sam Po}}{{r|人民日报}}。1879年，当地的富商黃志信从逝世的犹太裔地主手中购得三保洞及其附近的土地，并将之免费开放{{r|印尼僑聲雜誌}}。作为回应，当地华人进行了游行以示庆祝，并开始重新回到三保洞祭祀{{r|Setiakusuma2006|page=63}}{{r|Suherdjoko 2005, Semarang set for Sam Po}}。1924年，三保洞的所有权被转交给了三保洞基金会{{r|Setiakusuma2006|page=61}}。

三保洞的历史上曾多次整修。1937年，三保洞进行了修缮。在[[印度尼西亚日据时期|印度尼西亚日据时期]]，日本侵略者在寺庙中給郑和像配上了负面的说明文字{{r|Sudiarno 2001, Sam Poo Kong}}。此后，在印尼独立战争中，三保洞遭到破坏{{r|Sudiarno 2001, Sam Poo Kong}}，并于1950年重新修缮。1960年代，印尼国内不稳定的政局使得三保洞再次受到忽视{{r|Sudiarno 2001, Sam Poo Kong}}。2002年至2005年，三保洞再次进行了重新修缮{{r|Suherdjoko 2008, Semarang's Old City}}。

==布局==

三保洞建筑群包括五座主要建筑，由中式建筑和爪哇风格组成。五座建筑包括历史最久的三保洞和土地洞、船舡爷庙、铁锚庙和福兴庙{{r|Sudiarno 2001, Sam Poo Kong}}，占地面积{{convert|3.2|ha|acre}}{{r|Suherdjoko 2005, Temple gears up}}。

土地洞位于建筑群北入口大门内，供奉了[[土地公|土地公]]。紧邻土地洞的是船舡爷庙，供奉了王景弘的棺椁。这里常被用于祈求生意上的顺利{{r|Sudiarno 2001, Sam Poo Kong}}。在船舡爷庙以南是供奉有郑和像的三保洞大殿以及殿后的三保洞。在2002年修缮前，三保洞大殿一边长约{{convert|16|m|ft}}，修缮后长约{{convert|34|m|ft}}{{r|Suherdjoko 2005, Temple gears up}}。洞中有一个祭坛，用于求签；祭坛下是三保井，传说中永不乾涸且包治百病{{r|Sudiarno 2001, Sam Poo Kong}}。再往南是铁锚庙，供奉有郑和船队使用的铁锚，用于祭祀牺牲的船员。最南端是福兴庙，供奉郑和使用过的关刀{{r|Sudiarno 2001, Sam Poo Kong}}。

==庆祝活动==
每年农历六月三十，为了纪念郑和到达三宝垄，印尼华人都会举着供奉于大觉寺的三宝公圣像（郑和像）游行至三保洞并返回{{r|Suherdjoko 2005, Semarang set for Sam Po}}。这一活动在华人中象征着郑和巡视三宝垄，表现了对郑和的尊敬{{r|人民日报}}。在当代，这一活动已不仅限于华人参加，而成为了不分宗教、种族的共同的庆祝活动{{r|Suherdjoko 2005, Semarang set for Sam Po}}。
