《'''三月，七日。'''》是由{{lang|ja|[[森橋ビンゴ|森橋ビンゴ]]}}所著的輕小說，並獲得第三屆[[娛樂大賞|娛樂大賞]]戲劇企畫部門優秀獎。

== 故事簡介 ==
從[[廣島|廣島]]搬至[[東京|東京]]的'''宮島七日'''在到東京那天，因為父親跳下鐵軌救女兒而死亡。另一方面，澀谷三月努力成為好學生而成為學生會會長。這兩人，在同一所高中相遇……

== 人物 ==
; {{nihongo|宮島七日|宮島 七日|Nanoka Miyajima}}
: 故事的女主角，家住在[[廣島|廣島]]到[[東京|東京]]來的普通科一年級學生。
: 受到當到宿舍舍監操的青睞，接任成為舍監‥‥‥
: 和三月是雙胞胎兄妹，第二卷把姓氏換回「澀谷」
; {{nihongo|澀谷三月|渋谷 三月|Sangatsu Shibutani}}
: 故事的男主角，升學科一年級學生，學生會會長。
: 和七日是雙胞胎兄妹。
: 在七日、真希和母親面前才會顯示真面目：第一人稱是「俺」，性格散悶；其餘時間都是表現得像乖孩子。
; {{nihongo|藤井真希|藤井 真希|Maki Fujii}}
: 升學科一年級學生，是三月的同班同學。
; {{nihongo|羽住海里||Kairi Hazumi}}
: 升學科的保健室老師。
; {{nihongo|澀谷彌生|渋谷 弥生|Yayoi Shibutani}}
: 三月七日的母親。
; {{nihongo|岸直美||Naomi Kishi}}
: 與七日同樣住在女生宿舍的普通科一年級學生。
; {{nihongo|東山操|東山 操|Misao Higashiyama}}
: 藝術科三年級學生，故事開始時是女生宿舍的舍監。
