'''龙柏街道'''是[[上海市|上海市]][[闵行区|闵行区]]曾经存在的一个乡级行政區劃，管辖范围东到新泾港、虹桥镇界、虹井路，西到北横泾，北到长宁区界，南到七宝镇、虹桥镇，存在于2000年至2009年，已於2010年拆分并撤销<ref>{{cite web|url=http://www.stats.gov.cn/tjsj/tjbz/tjyqhdmhcxhfdm/2009/31/01/310112.html|title=2009年统计用区划代码和城乡划分代码：闵行区|website=[[中华人民共和国国家统计局|中华人民共和国国家统计局]]}}</ref>。

== 区划沿革 ==
=== 街道建立之前 ===
1949年5～7月，[[上海市军事管制委员会|上海市军事管制委员会]]新泾区接管委员会先后设5个办事处，下辖6镇、69个行政村、386个自然村。1950年6月，因龙华区面积过大，虹桥镇在蒲汇塘以南部分划入新泾区，设虹桥乡和新泾乡。当时虹桥乡管辖关港、龙华、漕溪、新龙华、石龙等十五个村（上述村今大多属于[[徐汇区|徐汇区]]）；新泾乡管辖虹桥、虹二等十个村；当时的西郊村属于新泾乡。<ref name="jz">{{cite web|title=上海县志 >> 第一篇建置 >> （三）行政区划|url=http://www.shtong.gov.cn/newsite/node2/node4/node2250/shanghai/node53607/node53615/node53617/userobject1ai35420.html|website=上海通网站|publisher=上海市地方志办公室}}</ref>

1956年3月，新泾、虹桥两镇随新泾区撤销，并入[[西郊区|西郊区]]<ref>{{cite web|title=上海县志 >> 第一篇建置 >> （三）行政区划   上海市新泾区、龙华区、西郊区行政区划|url=http://www.shtong.gov.cn/newsite/node2/node4/node2250/shanghai/node53607/node53615/node53617/userobject1ai35421.html|website=上海通网站|publisher=上海市地方志办公室|accessdate=2017-08-04}}</ref>。1958年8月，上海市[[西郊区|西郊区]]虹桥乡部分地区划入[[上海县|上海县]]境<ref name="shxz-jjbh">{{cite web|title=上海县志 >> 第一篇建置 >> （二）县域 >>  境域变化|url=http://www.shtong.gov.cn/newsite/node2/node4/node2250/shanghai/node53607/node53612/node53614/userobject1ai50766.html|website=上海通网站|accessdate=2017-07-31}}</ref>。

1992年7月，虹桥乡西郊村划入长宁区，归新泾镇，农副工业管理也同时划属长宁区<ref name="mzz" />。同年，随着上海县撤销改[[闵行区|闵行区]]，虹桥乡撤乡改镇。1994年，为解决西郊村“一块土地，两个政府”的问题，西郊村复归虹桥镇，闵行、长宁东按[[徐虹支线|徐虹支线]]、西按[[沪青平公路|沪青平公路]]中心线重划区界。<ref name="mzz" />

=== 街道建立===
1997年9月29日，龙柏新村街道在原西郊村地域上成立，以原虹桥镇西郊村、先锋村、井亭村（部分）地块为基础，辖区包括龙柏新村、金汇花园和原华光村<ref name="mzz">{{cite web|title=上海民政志 >> 第二章行政区划管理 >>  第二节  解放后区划管理|url=http://www.shtong.gov.cn/newsite/node2/node2245/node65977/node65987/node66011/userobject1ai61577.html|website=上海通网站|publisher=上海市地方志办公室|accessdate=2017-08-03}}</ref><ref>{{cite web|title=年鉴 >> 上海年鉴1998 >> 四十三、区 >> 节  (一)综述|url=http://www.shtong.gov.cn/newsite/node2/node19828/node19945/node29672/node29674/userobject1ai59017.html|website=上海通网站|publisher=上海市地方志办公室|accessdate=2017-08-03}}</ref>，边界为沿唐家泾{{NoteTag|在青杉路北侧，大致与青杉路平行，是金汇花园片区的最南侧边界。原通过航华大港河与七宝镇的唐家浜衔接，与唐家浜原是同一条河流，后被填没}}、虹井路—老虹井路、[[吴中路|吴中路]]、[[上海外环高速公路|外环线]]、[[沪青平公路|沪青平公路]]、[[徐虹支线|徐虹支线]]、[[新泾港|新泾港]]至唐家泾止{{NoteTag|这里指的是实际辖区之和，参见“龙柏新村”“金汇花园”段落。}}<ref>{{Cite book|title=《闵行区地名志》|author=上海市闵行区地名志编纂委员会|publisher=上海市社会科学院出版社|isbn=7-80618-780-4|year=2002}}</ref>{{rp|10-11}}。目前，龙柏新村和金汇花园有数个小区以西郊{{NoteTag|龙柏一村西郊公寓小区（红松路绿苑路、虹井路）}}、华光{{noteTag|金汇华光城小区（红松路金汇路）}}、先锋{{NoteTag|龙柏三村先锋公寓西区（红松路白樟路西侧）、龙柏三村先锋公寓东区（红松路虹井路西南）}}等虹桥镇原有行政村名字命名，虹桥镇也保留了先锋街{{NoteTag|[[红松路|红松路]]、吴中路之间的东西向乡道，虹井路—金雨路}}、华光路{{NoteTag|[[延安西路|延安西路]]、[[红松路|红松路]]之间的东西向乡道，程家桥（支）路—[[虹许路|虹许路]]}}两个路名。

2000年10月8日，[[上海市人民政府|上海市人民政府]]下发文件<ref>沪府〔2000〕71号</ref>，批准闵行区镇、街道行政区划调整方案。按照该方案，撤销龙柏新村和航华两街道建立新的龙柏街道。<ref name="mhb-kd">{{cite web|title=时间的刻度 历史的脚步|url=http://mhb.shmh.gov.cn/content/2012-09/07/content_8624.htm|website=闵行报|publisher=闵行报社|archiveurl=https://web.archive.org/web/20171014042425/http://mhb.shmh.gov.cn/content/2012-09/07/content_8624.htm|archivedate=2017-10-14|date=2017-10-14|deadurl=yes}}</ref>

=== 拆分撤销 ===
2009年9月，根据“大虹桥”地区发展方案，龙柏街道办事处消息人士表示，拟以[[沪青平公路|沪青平公路]]为界，建立“新虹桥街道”，撤销龙柏街道。<ref>{{cite web|url=http://why.eastday.com/q/20090909/u1a626863.html|title="大虹桥"所涉行政区拟作调整 龙柏街道或"消失"|publisher=青年报社|website=[[东方网|东方网]]|date=2009-09-09|author=王佳敏}}</ref>

[[上海市人民政府|上海市人民政府]]下发文件后<ref name=hf>沪府〔2009〕100号</ref>，闵行区于10月24日举行[[新虹街道|新虹街道]]成立大会，宣布撤销龙柏街道，以龙柏街道的班子为基础建立新虹街道，暂驻航西路1号。新虹街道处级干部由区委安排，原龙柏街道机关科级及科级以下公务员和直属事业单位工作人员整建制划转新街道。新虹街道和七宝镇、虹桥镇以[[沪渝高速|沪渝高速]]为界<ref name="xmw">{{cite web|title=现场：闵行区举行新虹街道成立大会_新民网_为民分忧 与民同乐|url=http://sh.xinmin.cn/minsheng/2010/01/23/3433205.html|website=新民网|publisher=新民网|accessdate=2017-07-21|archiveurl=https://web.archive.org/web/20170804174409/http://sh.xinmin.cn/minsheng/2010/01/23/3433205.html|archivedate=2017-08-04}}</ref>。根据这一拆分方案：龙柏新村、金汇花园和虹光村全数归入[[虹桥镇_(上海市)|虹桥镇]]；而航华新村片区一分为二：[[沪渝高速|沪渝高速]]以南的区域并入七宝镇；外环线以西、[[沪渝高速|沪渝高速]]以北全数归[[新虹街道|新虹街道]]<ref name=新虹街道介绍>{{cite web|url=http://xhjd.shmh.gov.cn/front/article-one.action?q.category=101001|title=走进新虹|publisher=上海市闵行区人民政府驻新虹街道（社区）办事处|deadurl=yes|archiveurl=https://web.archive.org/web/20170731233748/http://xhjd.shmh.gov.cn/front/article-one.action?q.category=101001|archivedate=2017年7月31日|df=}}</ref>。

==部分知名人物==
===“开心小屋”===
{{main|鲍美利}}
鲍美利是一名声乐教师，退休前工作于[[上海乐团|上海乐团]]。1996年，鲍美利一家从[[黄浦区|黄浦区]]原[[人民廣場街道_(上海市)|人民广场街道]]被搬迁到龙柏三村。2006年，居委会组织社区老年人开座谈会。鲍美利有感于子女远离，空巢独居，在家中开设了一周一次的“爱心小屋”活动，给当地居民提供文娱活动并进行心理疏导，深受当地居民好评。活动模式和经验被扩展开去，并被认为“开创邻里养老新路”<ref>{{cite web|url=http://www.shanghai.gov.cn/nw2/nw2314/nw38605/nw41659/nw41749/nw41773/nw41774/u21aw1204670.html|title=鲍美利们探索出邻里互助式养老新路子|website=www.shanghai.gov.cn}}</ref>。鲍美利夫妇于2012年4月被列入[[中央文明办|中央文明办]]发布的“中国好人榜”<ref>{{cite web|title=鲍美利:创办"开心小屋"点亮社区老人心火|url=http://www.wenming.cn/sbhr_pd/xiaozhuan/jlaq/201304/t20130416_1177123.shtml|website=中国文明网}}</ref>。
