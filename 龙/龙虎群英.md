'''龙虎群英'''（{{lang|ja|キイハンター}}，''KEYHUNTER'' ) 是[[日本|日本]]的[[电视剧|电视剧]]。讲述日本的首都东京变成了国际罪犯和间谍逃的地方，所以国际刑警成立了国际警察特别室来应对的故事{{reftag|[[朝日新聞|朝日新聞]]，1968年4月6日朝刊}} {{reftag|name = keyhunterCS|{{Cite web |url = http://www.toeich.jp/CS/information.php?info_id=1T0000000242 |title = キイハンター |publisher = [[东映|东映频道]] |archiveurl = https://web.archive.org/web/20100304002621/http://www.toeich.jp/CS/information.php?info_id=1T0000000242 |archivedate = 2011-12-24 |accessdate = 2010-03-06 }}}}。由[[TBS电视台|TBS电视台]]和[[東映|東映]]制作。

于1968年4月6日至1973年4月7日每周星期六21：00 - 21：56放送{{notetag|1972年10月之后，缩短1分钟，至21：55结束。}}，全262話，收视率最高为30%{{reftag|name = "sonny"|{{Cite book |author = JJサニー千葉 |authorlink = 千葉真一 |year = 2010 |title = 千葉流 サムライへの道 |publisher = [[ぶんか社|ぶんか社]] |pages = 20 - 29 |isbn = 4821142694 |ref = 千葉2010 }}}}{{reftag|name = 深作欣二|{{Cite journal  |date = 2012-11-27 |title = 千葉真一、深作欣二の初監督の怒号に驚いた |journal = [[アサヒ芸能|アサ芸＋]] |publisher = [[徳間書店|徳間書店]] |url = http://www.asagei.com/9361 |accessdate = 2012-12-5 <!--|archiveurl = http://liveweb.archive.org/http://www.asagei.com/9361 |archivedate = 2012-12-3--> }}}}。
