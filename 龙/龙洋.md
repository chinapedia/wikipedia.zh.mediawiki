'''龙洋'''，也称'''龍銀'''，近代东亚地区铸有[[龙|龙]]的图案一类的[[银元|银元]]的俗称，[[中国|中国]]、[[日本|日本]]、[[安南國|安南]]和[[朝鮮王朝|朝鲜]]均曾在各自国家发行。其中龙的形象在日本、朝鲜龙洋上出现在正面，在中国、安南龙洋上出现在背面。

龙洋的形制源自于从16世纪开始流入东亚地区的[[西班牙銀圓|西班牙银元]]，西班牙银元每枚重27.22克，成色为90.0%，也就是每元含银24.5克，各国龙洋均参照了这一重量和成色。

[[日本|日本]]从[[明治|明治]]三年（1870年）开始鑄造龙洋，中國从[[清|清]][[光绪|光绪]]十五年（1889年）开始铸造。

== 背景 ==
[[File:1888_México_8_Reals_Trade_Coin_Silver.jpg|thumb]]的墨西哥银元]]
随着[[西班牙|西班牙]]于16世纪在[[菲律宾|菲律宾]]建立起{{le|菲律宾历史 (1521-1898)|History of the Philippines (1521–1898)|殖民统治}}，[[市中市|市中市]]（[[马尼拉|马尼拉]]的前身）成为一个[[转口港|转口港]]，西班牙的船队每年定期往返于菲律宾马尼拉和[[新西班牙总督辖区|新西班牙总督区]][[阿卡普尔科|阿卡普尔科]]（今[[墨西哥|墨西哥]]阿卡普尔科）之间，横穿[[太平洋|太平洋]]将大量的[[中国|中国]]商品，经菲律宾中转，运往[[美洲|美洲]]甚至[[西欧|西欧]]，同时也将大量在墨西哥铸造的[[银元|银元]]经菲律宾中转输入到中国，这就是所谓的“[[马尼拉大帆船|马尼拉大帆船]]”贸易<ref>{{cite book |author=Williams, Glyn |year=1999 |title=The Prize of All the Oceans |publisher=Viking |location=New York |isbn=0-670-89197-5 |page=4}}</ref>。

马尼拉大帆船贸易给西班牙人带来大量财富的同时，也使西班牙的“[[八里亚尔|八里亚尔]]”银元大量流入[[东亚|东亚]]各国，并被广泛应用于国际贸易和日常交易，西班牙银元成为远东贸易中的一个货币标准。以致于西班牙在美洲的殖民统治崩溃后，西方国家发现同中国进行贸易时，首选的支付方式就是使用西班牙银元的继承者[[墨西哥银元|墨西哥银元]]<ref name="张宁2003">{{cite journal |author1=张宁 |title=墨西哥银元在中国的流通 |journal=中国钱币 |date=2003 |issue=4 |pages=26-31}}</ref>。

多国家因此发行了与西班牙银元相同的重约27.22克，含银量900‰的[[贸易银元|贸易银元]]、本国货币或殖民地货币，如[[英国|英国]]的[[叻币|叻币]]和[[港币|港币]]，日本龙洋，中国龙洋，[[法属印度支那元|法属印度支那元]]等等。其中，英国于1866年5月在[[香港|香港]][[銅鑼灣|銅鑼灣]]成立[[香港鑄幣廠|香港鑄幣廠]]，铸造一圆等面值的银币，是“圆”作为货币单位首次出现在钱币上{{efn|并非“圆”首次被用作货币单位，此前在中国已有使用，如1841年魏源《圣武记》中就有“凡商船出洋者，勒税番银四百圆”的记载<ref>{{cite web |author1=丁骋骋 |title=中国货币单位为何称“元” |url=http://money.163.com/16/1017/01/C3HS8PON002580S6.html |website=网易-上海证券报·中国证券网 |accessdate=2018-11-25 |date=2016-10-17}}</ref>。}}<ref name="香港造币厂">{{cite journal |author1=严绍林 |title=香港造币厂 |journal=钱币博览 |date=2003 |issue=3 |page=92}}</ref>。

== 安南龙洋 ==
安南王室在1830年代手制了几种背面带有龙图的明命通宝银元。

== 日本龙洋 ==
[[File:1yen-M3.jpg|thumb]]
{{seealso|日本银元|1日元银币}}
1868年，日本[[明治天皇|明治天皇]]上台，开启[[明治維新|明治維新]]，推行[[现代化|现代化]]改革。维新政府于[[明治|明治]]三年（1870年），从[[香港|香港]]购得造币机器，开始铸造本国银圆<ref name="简说日龙">{{cite journal |author1=杨觉 |title=简说日本龙洋银币 |journal=收藏界 |date=2010 |issue=6 |pages=102-103}}</ref>。这批机器来自于1866年开设，1868年就关停的[[香港造币厂|香港造币厂]]，日本政府以6万港元的价格购买了该厂的全套造币机械<ref name="香港造币厂"/>。

明治四年（1871年）5月10日，日本政府颁布《{{link-ja|新货条例|新貨条例}}》（明治4年太政官布告第267号），废止[[江户幕府|江户幕府]]时代的{{link-ja|一分银|一分銀}}等银币，制造重416[[格令|格令]]（约26.96克），含银900‰的[[1日元银币|贸易一圆银币]]，同时制作50钱以下的银辅币。为防止银辅币流失海外，成色减为800‰，质量约减少7.2%。

面额一圆的银币均带有龙图，铸造发行始于明治三年（1870年），止于[[大正|大正]]三年（1914年），45年间共铸造了34个年号，依次是明治三年、七到三十年、三十四到三十九年、四十一年、四十五年和大正三年，分为三大版别，既正面龙图背面[[太阳|太阳]]的明治三年版，正面龙图背“一圆”的明治七年版，正面龙图背“[[贸易银元|贸易银]]”的明治八年版。明治三年版仅发行于明治三年，明治七年版发行于明治七年和明治十一年之后，明治八年版仅发行于明治八至十年<ref name="简说日龙" />。

== 中国龙洋 ==
[[File:China-7sen2hun-1904.jpg|thumb]]
传统上，[[中國|中國]]硬币多为利用铸造法制造的铜钱，明清时期白银的使用日渐广泛，但多为铸造的银锭或碎银。清朝[[乾隆年间|乾隆年间]]，在[[西藏|西藏]]仿照[[中尼银钱贸易|尼泊尔银币]]制造发行的[[藏币|藏币]]，是中国最早的仿照欧洲打压法制作的银币。

[[道光|道光]]年间，[[臺灣|臺灣]]亦曾仿制银圆，称为'''银饼'''，但規模不大。

由於銀幣攜帶便利，每枚成色、重量穩定，不似元寶需要澆灌融鑄，較為方便，廣受歡迎。尤其[[晉商|晉商]]、[[徽商|徽商]]等全國性的商業人士，更加愛用。一般情形下，一個銀元約重0.72兩，九成銀，故僅含有銀0.648兩。但民間大致上以0.7兩白銀，兌換一個銀元。（有時以0.69兩兌換，稱六九銀；亦有時以0.72兩兌換，稱七二銀。）由於[[鷹洋|鷹洋]]、[[佛銀|佛銀]]等外國[[銀幣|銀幣]]的流入，嚴重影響中國的幣制和[[金融|金融]]，同時也促進中國的幣制改革。

[[光緒|光緒]]十四、十五年间（1888年-1889年）在[[廣東省|廣東省]]起開始以接近的重量、成色鑄造銀幣。其後均開始自鑄銀幣，抵抗外國貿易銀。銀幣因為上有蟠龍像，因此被稱為「[[龍銀|龍銀]]」，各[[行省|行省]]纷纷仿制。1889年，廣東地方當局正式設立正式設立銀元局，彷照[[鷹洋|鷹洋]]的重量、成色和式樣，用機器大量鑄造，開啟[[中國本土|中國本土]]自製銀幣的先例。<ref name="萬">{{cite book | language = zh-tw | author = 名鑑文化編輯部 | coauthors =  | title = 《萬物簡史》| url =  | date = 2007年 | location = 台灣 | publisher = 名鑑文化 | id =  | isbn = 986828421X | pages = }}</ref>同時，[[張之洞|張之洞]]眼見外國銀幣佔據[[中國|中國]]市場，並換回十足的[[銀兩|銀兩]]，造成[[中國人|中國人]]的貿易損失，故提議設局鑄幣，獲得朝廷同意，[[光緒|光緒]]十六年（1890年）鑄出[[龍洋|龍洋]]。各地各自鑄造，皆以龍為記，故稱龍銀、龍洋。

[[宣統|宣統]]二年（1910年），清朝廷颁布《币制则例》，规定银为本位币，以西制「標準重量為27.47克；成色90.2%，含銀24.76克」每枚重[[库平银|库平银]]七钱二分，含纯银九成，合六钱四分八厘。翌年5月开始铸造大清銀幣。旋因[[辛亥革命|辛亥革命]]发生，未正式发行。
