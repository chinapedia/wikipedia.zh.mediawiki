'''近藤喜文'''（{{lang-ja|'''近藤 喜文'''}}，{{bd|1950年|3月31日|1998年|1月21日|catIdx=Kondō Yoshifumi}}）是出生於[[日本|日本]][[新潟縣|新潟縣]][[五泉市|五泉市]]的[[動畫師|動畫師]]、[[動畫|動畫]]導演和[[人物設計師|人物設計師]]。生前與[[宮崎駿|宮崎駿]]、[[高畑勳|高畑勳]]兩人長期合作。他导演了动画《[[侧耳倾听|侧耳倾听]]》，并且是《[[清秀佳人|清秀佳人]]》、《[[名偵探福爾摩斯|名偵探福爾摩斯]]》、《[[魔女宅急便|魔女宅急便]]》、《[[岁月的童话|岁月的童话]]》和《[[幽灵公主|幽灵公主]]》等作品的作画监督。近藤喜文是[[吉卜力工作室|吉卜力工作室]]顶尖的动画导演，也是吉卜力工作室核心成員之一，在去世之前被認為是[[宮崎駿|宮崎駿]]和[[高畑勳|高畑勳]]的接班人。

近藤喜文於1998年因[[主动脉夹层|主动脉夹层]]去世。他的去世被认为是由于过度工作，也被懷疑是宮崎駿在1998年宣布退休的主要原因。

== 經歷 ==
近藤喜文於1965年4月進入了新潟縣立村松高中就讀，並加入校內的美術社。還是高中生的近藤喜文因在電影院看了高畑勳及宮崎駿合作的劇院動畫《[[太陽王子_霍爾斯的大冒險|太陽王子 霍爾斯的大冒險]]》大受感動，而產生了從事動畫師工作的想法<ref name="Ameba">{{Cite web |url=http://ameblo.jp/kokkoippan/day-20100717.html |script-title=ja:「耳をすませば」の監督・近藤喜文さんインタビュー |work=Ameba |date=2010-07-17 | accessdate=2012-08-28}}</ref>。

在高中畢業後近藤喜文前往東京，之後在1968年10月1日進入了動畫室[[SHIN-EI動畫|A Production]]，首次職務為擔任棒球題材動畫《[[巨人之星|巨人之星]]》的原畫<ref name="RomanAlbum">{{Cite book |script-title=ja:ロマンアルバム 耳をすませば Whisper Of The Heart |location=日本 |publisher=[[德間書店|德間書店]] |date=1995 |pages=39頁  |ISBN=T1061578240488 |accessdate=2012-08-28 }}</ref>。1971年高畑勳和宮崎駿同時從[[東映動畫|東映動畫]]跳槽到A Production<ref>{{Cite book |title=出發點 1979～1996 |location=台灣 |publisher=[[台灣東販|台灣東販]] |author=[[宮崎駿|宮崎駿]] |date=2006 |pages=542頁  |ISBN=978-957-473-843-4 |accessdate=2012-08-28 }}</ref>，近藤喜文則與他們兩人一同參與《[[魯邦三世_(TV第1期)|魯邦三世]]》第一期動畫的製作。1978年6月20日，近藤喜文轉移至[[日本動畫公司|日本動畫公司]]，參與《[[高立的未來世界|未來少年柯南]]》、《[[清秀佳人|清秀佳人]]》等製作<ref>{{Cite book |script-title=ja:スタジオジブリの軌跡 1984-2011 |location=日本 |publisher=德間書店 |date=2011 |pages=39頁  |ISBN=978-4-19-720326-0 |accessdate=2012-08-28 }}</ref>。之後1980年12月16日則向日本動畫公司辭職，轉移至Telecom Animation Film擔任《[[名偵探福爾摩斯|名偵探福爾摩斯]]》裡的角色設計及作畫監督。

1985年3月16日近藤喜文從Telecom Animation Film退出後，曾在6月到8月期間因[[氣胸|自發性氣胸]]入院，之後暫以從事[[自由職業|自由職業]]為主<ref name="RomanAlbum" />。

1987年1月轉移至高畑勳和宮崎駿專用的工作室[[吉卜力工作室|吉卜力工作室]]，參與了《[[螢火蟲之墓|螢火蟲之墓]]》、《[[魔女宅急便|魔女宅急便]]》等作品。數年後在宮崎駿的提案下接手了將少女漫畫家[[柊葵|柊葵]]作品《[[心之谷#原作漫畫|心之谷]]》改編成同名劇院動畫的導演任務，並在1995年夏天公開上映<ref>{{Cite book |title=-{心之谷}-－－宮崎駿V.S.製作過程 |location=台灣  |publisher=尚登出版 |date=1996 |pages=4頁 |ISBN=957-8475-10-1 |accessdate=2012-08-28 }}</ref>。

1998年1月21日，近藤喜文因主动脉剝離去世，享年47歲。葬禮中將近藤喜文遺體送至火葬場前，播放了他生前執導的《心之谷》主題曲《[[鄉村路|鄉村路]]》作為告別曲<ref>{{Cite web |url=http://www.yk.rim.or.jp/~rst/rabo/news/kondoufuhou.html |script-title=ja:さようなら、近藤喜文さん |work={{lang|ja|近藤喜文さんの世界}} |author=葉 精二 |date=1998-01-22 | accessdate=2012-08-28}}</ref>。

相隔16年後。2014年7月4日至8月31日在他的家鄉由縣立萬代島美術館舉辦「來自新潟的吉卜力動畫師 近藤喜文展」紀念活動。

== 其它 ==
* 因近藤喜文擅長描繪寫實人物肢體動作，在高畑勳和宮崎駿分別同時進行製作《螢火蟲之墓》、《[[龍貓|龍貓]]》時都希望近藤能成為團隊的一員<ref>{{Cite book |script-title=ja:映畫道樂 |location=日本 |publisher={{lang|ja|びあ}} ||author=[[鈴木敏夫|鈴木敏夫]] |date=2005 |pages=107頁  |ISBN=4-8356-1540-9 |accessdate=2012-08-28 |language= ja}}</ref>，高畑勳更曾表示若近藤不願投入《螢火蟲之墓》製作；「這部作品就無法完成」<ref>{{Cite book |script-title=ja:映畫道樂 |location=日本 |publisher={{lang|ja|びあ}} ||author=鈴木敏夫 |date=2005 |pages=106頁  |ISBN=4-8356-1540-9 |accessdate=2012-08-28 |language= ja}}</ref>，近藤最後加入了《螢火蟲之墓》團隊。
* [[高野文子|高野文子]]及[[諾曼·洛克威爾|諾曼·洛克威爾]]分別是近藤喜文欣賞的漫畫家及插畫家。
* 近藤喜文生前在[[機關報|機關報]]{{nihongo|「國公勞新聞」|国公労新聞}}第943號上的採訪，表示有著想做出像兒童作家[[灰谷健次郎|灰谷健次郎]]著作{{nihongo|《天之瞳》|天の瞳}}世界觀的動畫念頭<ref name="Ameba" />。
