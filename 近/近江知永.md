'''近江知永'''（{{jpn|j=近江 知永|hg=おうみ ともえ|rm=Tomoe Ohmi}}，{{bd|1982年|10月13日<ref name="blog-pro">{{Cite web|date=|url=http://ohmi003.jugem.jp/?pid=1|title=【PROFILE】近江知永公式ブログ“003”|work=|author=近江知永|publisher=近江知永的官方個人部落格|accessdate=2017-12-21|language=ja}}</ref>|catIdx=Ohmi Tomoe}}），[[日本|日本]]女性[[配音員|配音員]]、[[歌手|歌手]]、[[舞台|舞台]][[演員|演員]]。出身於[[埼玉縣|埼玉縣]]。身高155cm<ref name="blog-pro" />。A型[[血型|血]]<ref name="blog-pro" />。

== 簡介 ==
* [[日本旁白演技研究所|日本旁白演技研究所]]出身。以前所屬的經紀公司有{{link-ja|RAMS Professional Education|RAMS Professional Education}}、{{link-ja|RAMS|ラムズ}}，並有一段時間以[[自由職業|自由身]]持續參與，現隸屬於[[Kitty_film|株式會社Kitty]]（2016年3月進入）。

=== 年譜 ===
; 2004年
* 以[[電視動畫|電視動畫]]《[[校園迷糊大王|校園迷糊大王]]》為登場配角，城戶圓配音成為聲優出道作。
; 2005年
* 經由廣播節目「奧井雅美O-Live」舉辦的「Animelomix Audition」徵選成功勝出，並以[[奧井雅美|奧井雅美]]監製的歌曲「泡沫（{{lang-ja|ウタカタ}}）」成為踏入歌壇的出道作及代表作。
; 2008年
* 7月，從{{link-ja|RAMS|ラムズ}}退所。從此以[[自由職業|自由聲優]]身份持續參與。
; 2009年
* 4月，加入經紀公司M Friend（現改名FIRE WORKS）。
; 2013年
* 1月，從FIRE WORKS離所。又再度成為自由聲優。
; 2014年
* 與[[原田謙太|原田謙太]]（前樂團{{link-ja|Rey (樂團)|Rey (バンド)|Rey}}主唱）組成雙人演唱組合「'''AFIVE'''」後。同年12月26日，在阿佐谷閣樓A舉辦的演唱會『{{lang|ja|おーみ}}&{{lang|ja|はらけん}}新{{lang|ja|ユニット}}設立{{lang|ja|総会}}』中正式結成。
; 2016年
* 3月，加入經紀公司[[Kitty_film|株式會社Kitty]]。
; 2017年
* 1月27日，與藝人{{link-ja|TAIGA|TAIGA}}正式宣布結婚<ref>{{Cite web|url=http://natalie.mu/owarai/news/218424|title=TAIGAが声優の近江知永と本日結婚「僕のやる事によく笑う方」|work=[[Natalie_(網站)|搞笑Natalie]]|publisher=Natasha|date=2017-01-27|accessdate=2017-11-28|language=ja}}</ref>。
* 4月，成為舞台劇團{{link-ja|劇團東京High-beam|劇団東京ハイビーム|東京High-beam}}的團員。

=== 人物 ===
* 有一個哥哥、一個妹妹。妹妹結婚後於2006年生下一子{{來源請求}}。
* 喜歡的東西是[[咖哩|咖哩]]，這在近江的部落格日記上經常出現。
* 興趣有[[作詞|作詞]]、彈[[鋼琴|鋼琴]]&[[吉他|吉他]]、畫[[水彩畫|水彩畫]]<ref name="blog-pro" />。有關鋼琴的部分，可在現場演唱會的即彈即唱中顯露出她的才藝{{來源請求}}。
* 特長有[[游泳|游泳]]、打[[籃球|籃球]]{{來源請求}}。
* 年齡在官方網站上沒有正式公開，但是在定期主持的電台節目《{{link-ja|動畫Para音樂館|アニぱら音楽館}}》中，有一次同行聲優[[鈴木達央|鈴木達央]]當特別來賓，透露出『近江比起鈴木（1983年生）還長一歲』的訊息；另外在2007年的生日當天，她直接在自己的部落格說出年紀與《[[超時空要塞|超時空要塞]]》系列是一樣的，因此是1982年出生則不言可喻。
* 近江她曾經是高中棒球迷，後來有一次前往東京巨蛋看球賽時深深被職棒所吸引，之後她自己自稱是巨人隊的球迷。喜歡的選手是[[矢野謙次|矢野謙次]]{{來源請求}}。
