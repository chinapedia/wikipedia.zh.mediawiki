'''近藤史惠'''（{{bd|1969年|5月20日|||}}），是一名[[日本|日本]]著名[[推理小說|推理小說]]作家，畢業於[[大阪藝術大學|大阪藝術大學]]，1993年憑《冰凍之島》獲得{{link-ja|鮎川哲也獎|鮎川哲也賞}}出道，2008年獲得[[大藪春彥獎|大藪春彥獎]]。除撰寫小說外，近藤史惠又曾在母校擔任[[客座教授|客席副教授]]。

==生平==
近藤史惠為[[大阪府|大阪府]]出身，[[大阪芸術大学|大阪芸術大学]]文芸学科畢業後、1993年以『凍える島』獲得第4回[[鮎川哲也賞|鮎川哲也賞]]<ref name=dokusyomichi>{{Cite web |url = http://www.webdoku.jp/rensai/sakka/michi71.html |title = 作家の読書道 |publisher = WEB本の雑誌 |date = 2007-09-28 |accessdate = 2013-03-04 }}</ref>。2006年、近藤史惠擔任大阪芸術大学文芸学科[[客員准教授|客員准教授]]。
