
'''逄国'''{{efn|{{标音|char=逄|pinyin=páng}}}}，或记作'''逢国'''{{efn|“逄”字作为氏名、地名，根据甲骨文和金文可隶定为“夅”、“夆”，多被记作“逄”，也有记作“逢”的，除引用资料原文，本条目统一记作“逄”}}，中国[[先秦时期|先秦时期]]的一个[[诸侯国|诸侯国]]，其国君为[[伯爵|伯爵]]，公族为姜姓。

== 文献记载 ==
[[春秋时期|春秋]]末年成书的《[[左传|左传]]·昭公二十年》记载有[[齐国|齐国]]大夫[[晏婴|晏婴]]对于[[齐国|齐地]]古史的一段话：“昔爽鸠氏始居此地，季荝因之。有逄伯陵因之，蒲姑氏因之，而后太公因之。”<ref>[[:s:春秋左氏傳/昭公#昭公二十年|左传·昭公二十年]]，維基文庫</ref>又按[[东汉|东汉]]时[[班固|班固]]所编撰的《[[汉书|汉书]]·地理志下》记载，[[爽鸠氏|爽鸠氏]]与[[少昊|少昊]]是同一时期，[[季荝|季荝]]为[[舜|虞]][[夏代|夏]]时期，[[逄伯陵|逄伯陵]]与[[商汤|商汤]]同时，蒲姑氏，即[[薄姑氏|薄姑氏]]，为[[商代|商代]]末年的诸侯，太公则是指齐太公[[姜子牙|姜子牙]]。<ref>[[:s:漢書/卷028下|汉书·地理志下]]，維基文庫：“齊地……少昊之世有爽鳩氏，虞、夏時有季崱，湯時有逢公柏陵，殷末有薄姑氏，皆為諸侯，國此地。至周成王時，薄姑氏與四國共作亂，成王滅之，以封師尚父，是為太公。”</ref>从商汤到商末，此处所说的逄国是一个存在时间几乎与商朝相始终的诸侯国。<ref>{{Citeweb|title=大国夹缝中生存的西周小国 |url=http://jnsb.e23.cn/shtml/jnsb/20150702/1456380.shtml |publisher=舜网-济南时报 |date=2015-07-02}}</ref>

[[战国时期|战国]]初期成书的《[[國語_(書)|国语]]·周语下》“景王问钟律于伶州鸠”一文中有“我姬氏出自天黿，及析木者，有建星及牽牛焉，則我皇妣大姜之姪、伯陵之後逄公之所憑神也”，[[三国时期|三国时期]][[孙吴|吴国]]史学家[[韦昭|韦昭]]注释称“伯陵，大姜之祖有逄伯陵也。逄公，伯陵之後，大姜之姪，殷之諸侯，封於齊地”<ref>{{Citeweb|title=國語注·國語卷第三 |url=http://www.guoxuedashi.com/a/1751o/110032a.html |publisher=国学大师}}</ref>，其中“大姜”是指周太王[[古公亶父|古公亶父]]的妻子、[[周文王|周文王]]昌的祖母[[太姜|太姜]]，由于逄公是太姜的侄子，由此可知这位逄公以及他的先祖逄伯陵也都是[[姜姓|姜姓]]。《山海经·海内经》中也有伯陵为[[炎帝|炎帝]]之孙的记载<ref>{{Citeweb|title=《山海经》 卷十八 海内经 |url=http://www.guoxue.com/book/shanhaijing/0018.htm |publisher=国学网 |quote=炎帝之孙伯陵，伯陵同吴权之妻阿女缘妇，缘妇孕三年，是生鼓、延、殳。始为侯，鼓、延是始为钟，为乐风。}}</ref>。

其后史家多沿用了这些观点：晋代[[杜预|杜预]]注《左传》“有逄伯陵因之”一句时，称“逢伯陵，殷诸侯，姜姓”<ref>{{Citeweb|title=春秋左传正义·卷四十九 昭公二十年 |url=http://gj.zdic.net/archive.php?aid=3020 |publisher=汉典古籍 |author=《左传注疏》杜预、孔颖达}}</ref>；元代[[于钦|于钦]]撰《[[齐乘|齐乘]]》有“逢伯陵，商之诸侯，封于齐”<ref name=yuqin/>。

== 都邑考释 ==
南宋[[郑樵|郑樵]]著《[[通志|通志]]·[[氏族略|氏族略]]·以国为氏》“逄氏”一条：“逄氏，音庞，商诸侯，封于齐土，至商周之间有蒲姑氏代之。至武王伐商后封太公焉。其地在今临淄。”《齐乘·古迹》“逄陵城”一条：“逄陵城，般阳府东北四十里。逢伯陵，商之诸侯，封于齐，薄姑氏因之，后太公又代之”<ref name=yuqin>(元)于钦纂.齐乘.元至元五年(1339)纂,清乾隆四十六年(1781)刻本.中華書局影印本《宋元方志叢刊》.1990年5月第1版</ref>{{rp|580}}，此处“般阳府”是指元代在[[山东省|山东省]]北部的行政区划[[般阳府路|般阳府路]]，治所为淄川县（今[[淄博市|淄博市]][[淄川区|淄川区]]）。

== 文物佐证 ==
从1979年开始，考古人员对山东省[[济阳县|济阳县]]刘台子遗址多个墓葬进行了发掘，出土了[[卷鼻象足方鼎|卷鼻象足方鼎]]等多件铭文中带有“夅”字的青铜器。
