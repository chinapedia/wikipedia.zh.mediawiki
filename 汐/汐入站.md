'''汐入站'''（{{jpn|j=汐入駅|hg=しおいりえき|rm=Shioiri eki|f=y}}）是一個位於[[神奈川縣|神奈川縣]][[橫須賀市|橫須賀市]]汐入町二丁目，屬於[[京濱急行電鐵|京濱急行電鐵]][[京急本線|本線]]的[[鐵路車站|鐵路車站]]。[[車站編號|車站編號]]是'''KK58'''。

==年表==
* 1930年（[[昭和|昭和]]5年）4月1日 - 車站啟用，當時車站稱為'''橫須賀軍港站'''。
* 1940年（[[昭和|昭和]]15年）10月1日 - 為隱藏軍事設備地點，車站改稱'''橫須賀汐入站'''。
* 1941年（昭和16年）11月1日 - 湘南電氣鐵道與京濱電氣鐵道合併，本站成為京濱電氣鐵道車站。
* 1942年（昭和17年）5月1日 - 京濱電氣鐵道與東京橫濱電鐵合併。本站成為[[東京急行電鐵|東京急行電鐵]]（大東急）車站。
* 1948年（昭和23年）6月1日 - 京濱急行電鐵成立，本站成為京濱急行電鐵車站。
* 1961年（[[昭和|昭和]]36年）9月1日 -隨著町名改正，車站改稱'''汐入站'''。
* 2020年3月 - 加註副站名「橫須賀藝術劇場」（{{lang|ja|横須賀芸術劇場}}）<ref>{{cite web|url=https://www.keikyu.co.jp/company/news/2018/20190125HP_18229TS.html|title=沿線地域の活性化を目的に2020年3月 4駅の駅名を変更します|publisher=京浜急行電鉄株式会社|date=2019-01-25|language=ja}}</ref>。

==車站結構==
本站為[[側式月台|側式月台]]2面2線[[地面車站|地面車站]]。月台大部分位於邊堤上，[[浦賀|浦賀]]方向緊鄰隧道。

剪票口僅有一處，設有階梯與電梯通往月台。

本站的簡易自動廣播分別由1號線的{{link-ja|關根正明|関根正明}}、2號線的[[大原沙耶香|大原沙耶香]]配音。

== 車站周邊 ==
位於橫須賀市[[横須賀中央站#車站周邊|市中心]]西側。

車站附近可分為有悠久商店街的山側與再開發地區的海側。再開發地區過去是{{link-ja|住友重機械工業|住友重機械工業}}橫須賀分工廠與[[在日美軍|在日美軍]]軍人俱樂部，1980年代中期開始都市更新。

== 巴士路線 ==
站前有巴士總站，可至周邊住宅區與[[三浦半島|三浦半島]]中央地帶。另外步行2分左右可至國道16號線上的「汐留」巴士站，可前往橫須賀市北部、東部、西部方向。雖然本站起訖的路線，但大多數都從橫須賀站起訖。汐留的班次與路線較多，多以快特停車站橫須賀中央為轉乘站。
; 1號乘車處
: 池上線 - 池上十字路、[[衣笠站|衣笠站]]方向
; 2號乘車處
: 湘南国際村線 - 池上十字路、湘南國際村方向
; 3號乘車處
: 西来寺循環 - 西來寺、鶴丘方向
; 4號乘車處
: 往橫須賀站各線
: 池上線 - 橫須賀中央站、安浦二丁目方向
; 汐留
: 請參照[[橫須賀站#巴士路線|橫須賀站#巴士路線]]。
