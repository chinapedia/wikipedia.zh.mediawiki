'''拳擊道'''是一個由傳統[[韓國武術|韓國武術]]的風格及[[泰拳|泰拳]]的混合的技擊運動；在某程度上拳擊道被稱為[[韓國|韓國]]版本的拳擊。在某些[[城市圈|城市圈]]裡亦有被稱為「Muay Kwon」（泰道）及「Kwon Ho」。「拳擊道」一詞有幾個意思，分別是「拳擊及攻擊的藝術」、「第一個攻擊」及「拳擊的藝術」。「拳擊道」中的「[[拳擊|拳擊]]」代表著拳打及拳擊，而「[[道|道]]」即是其藝術或是修練。現在其總部位於韓國的斧山。

== 創立過程 ==
該武術是由[[鍾道武|鍾道武]]（音譯，其[[諺文羅馬化|諺文羅馬化]]為「Jung Do-Mo」）看到一個擅長[[跆拳道|跆拳道]]的人跟泰拳練習者砌磋後所創立的。創始人表示這個混合體比起單獨使用[[跆拳道|跆拳道]]或是[[泰拳|泰拳]]更有破壞性。拳擊道的左右銘是「To attack offensively, never to surrender」（意譯為「永不放棄狠狠的攻擊」）。其教義為自信、實踐、忠誠、尊敬及責任。

== 技術的混合及改變 ==
[[File:KungekdoSpinningHookKick.jpg|200px]]
這個武術的基本技術是跆拳道的基本[[踢擊|踢擊]]技巧，再加上一些泰拳的技巧，例如是一些防衛踢擊（類似[[散手|散打]]）及一些使用[[膝蓋|膝蓋]]及[[手肘|手肘]]的攻擊。另外，一些[[泰拳|泰拳]]中部分步法也被刪除掉，就好像是泰拳中的[[步進|步進]]（Stepping）改為跆拳道裡的[[滑步|滑步]]（Slides）及[[快步|快步]]（Quick steps）。對於拳擊的閃避（類似[[美式|美式]]風格對於拳擊的急忙低頭避開）已加進這個[[技擊運動|技擊運動]]以代替泰拳裡有缺點的「Back up」，這令到使用者可以閃避及還擊得快點。現在，這種風格的混合武術已經進步到站在[[戰鬥|戰鬥]]的角度其中一個最有實力的[[武術|武術]]之一。

拳擊道比較少有類似[[掌|掌]]（即是做出一個張開手掌的動作）的技術；練習者認為，這是因為在對手用力的防護或者是擊打的這個情況下，[[手指|手指]]會很容易夭斷或脫臼。這個武術的足部技巧是混合自[[跆拳道|跆拳道]]及Kickboxing（有氧搏击操）的動作。跆拳道的基礎能夠在其[[混合武術|混合武術]]中看到，例如是[[前踢|前踢]]（因為踢出時比其他的[[踢|踢擊]]都要快）、[[後踢|後踢]]、[[側踢|側踢]]及[[勾踢|勾踢]]等等。另外一個被混合至拳擊道的是泰式的[[旋踢|旋踢]]，但在[[跆拳道|跆拳道]]裡該[[踢|踢擊]]是被禁止的，這是因為這種踢擊有著驚人的[[速度|速度]]。

== 滅絕之路 ==
儘管在泰國比較少有拳擊道的習訓者用其武術來對抗[[泰拳|泰拳]]的練習者，只不過他們的[[技術|技術]]也有一定的水平。不幸地拳擊道還有很多的進步及發展空間，不能廣泛被大眾公認為這是一種武術；有見及此，一些人認為這種武術在不少人知道其生存的時候，已經走到去[[滅絕|滅絕]]的邊緣。
