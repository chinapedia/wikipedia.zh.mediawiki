'''“衡水”号导弹护卫舰'''（[[舷号|舷号]]572）简称'''“衡水”舰'''，是[[中国人民解放军海军|中国人民解放军海军]]第十一艘[[054A型导弹护卫舰|054A型导弹护卫舰]]，可以单独或者协同海军其他[[兵力|兵力]]攻击敌[[水面舰艇|水面舰艇]]及[[潜艇|潜艇]]，具有较强的远程警戒和[[防空作战|防空作战]]能力。衡水舰由[[黄埔造船厂|黄埔造船厂]]开工建造，于2011年下水，2012年7月9日入列，服役于[[南海舰队|南海舰队]][[驱逐舰第九支队|驱逐舰第九支队]]。衡水舰舰名取自[[河北省|河北省]][[衡水市|衡水市]]，其入列命名授旗仪式于2012年7月9日在[[海南省|海南省]][[三亚市|三亚市]]某军港举行。<ref>{{cite web|url=http://www.hengshui.gov.cn/HSXW/JRHS/2012/07/13/content_239523.shtml |title=“衡水”舰入列南海舰队 |publisher=衡水日报 |date=2012-07-13 }}</ref>

== 历史 ==
2013年3月19日，“衡水”号导弹护卫舰与[[兰州号导弹驱逐舰|“兰州”号导弹驱逐舰]]、[[玉林号导弹护卫舰|“玉林”号导弹护卫舰]]及[[井冈山号综合登陆舰|“井冈山”号综合登陆舰]]组成远海训练编队驶离三亚市某军港，赴[[南海|南海]]、[[西太平洋|西太平洋]]等海域进行战备巡逻远海训练。<ref>{{cite web|url=http://www.chinanews.com/mil/2013/03-19/4657766.shtml |title=南海舰队组织联合机动编队战备巡逻远海训练 |publisher=中新网 |date=2013-03-19 }}</ref>联合机动编队先后航经[[西沙|西沙]]、[[南沙|南沙]]、[[曾母暗沙|曾母暗沙]]、[[巴士海峡|巴士海峡]]、[[西太平洋|西太平洋]]等海区，先后巡视了[[渚碧礁|渚碧礁]]、[[南薰礁|南薰礁]]、[[东门礁|东门礁]]、[[赤瓜礁|赤瓜礁]]、[[永暑礁|永暑礁]]、[[华阳礁|华阳礁]]等南沙岛礁。<ref>{{cite web|url= http://gb.cri.cn/27824/2013/04/03/6611s4073228.htm|title=中国海军南海舰队联合机动编队完成任务返抵三亚 |publisher=国际在线| date=2013-04-03 }}</ref>该次远海训练历经16昼夜、航程近5千海里。联合机动编队于4月3日上午返回驻地。<ref>{{cite web|url=http://navy.81.cn/content/2013-04/04/content_5289766.htm |title=南海舰队联合机动编队十六天远海训练精彩回顾 |publisher=中国海军网 |date=2013-04-04 }}</ref>

2013年4月9日，“衡水”号导弹护卫舰与[[岳阳号导弹护卫舰|“岳阳”号导弹护卫舰]]代表驱逐舰第九支队接受了[[中共中央军委主席|中共中央军委主席]][[习近平|习近平]]检阅。<ref>{{cite web|url=http://www.chinanews.com/gn/2013/04-11/4723279.shtml |title=习近平视察海军驻三亚部队 登井冈山舰和新型潜艇 |publisher=中新网 |date=2013-04-11 }}</ref>

2013年4月10日，“衡水”号导弹护卫舰与“兰州”号导弹驱逐舰重返西太平洋海域，开展包括防空、反潜和对海打击等内容的远海攻防演练。编队于4月17日《[[马关条约|马关条约]]》签订118周年之际抵达[[钓鱼岛|钓鱼岛]]附近海域巡航。<ref>{{cite web|url=http://news.gmw.cn/newspaper/2013-04/18/content_1255496.htm |title=我舰队马关条约纪念日巡航钓鱼岛 |publisher=光明网 |date=2013-04-18 }}</ref>该次远海训练历时11天，总航程3000余海里，衡水舰等于4月20日返回三亚市某军港码头。<ref>{{cite web |url=http://news.xinhuanet.com/mil/2013-04/20/c_124606661.htm |title=海军南海舰队远海训练编队完成任务返回三亚 |publisher=新华网 |date=2013-04-20 |deadurl=yes |archiveurl=https://web.archive.org/web/20150822103727/http://news.xinhuanet.com/mil/2013-04/20/c_124606661.htm |archivedate=2015年8月22日 |df= }}</ref>

2013年8月8日，“衡水”号导弹护卫舰、“井冈山”号综合登陆舰及[[太湖号综合补给舰|“太湖”号综合补给舰]]组成中国海军第十五批护航编队从[[广东省|广东省]][[湛江市|湛江市]]启航，赴[[亚丁湾|亚丁湾]]、[[索马里|索马里]]海域接替第十四批护航编队执行护航任务。<ref>{{cite web|url=http://www.chinanews.com/mil/2013/08-08/5140705.shtml |title=中国海军第十五批护航编队奔赴亚丁湾 |publisher=中新网 |date=2013-08-08 }}</ref>该次护航历经169个日夜，衡水舰等在完成46批181艘中外船舶的护航任务护航与出访[[坦桑尼亚|坦桑尼亚]]、[[肯尼亚|肯尼亚]]、[[斯里兰卡|斯里兰卡]]3国任务后，于2014年1月23日上午抵达湛江市某军港。<ref>{{cite web |url=http://news.xinhuanet.com/mil/2014-01/23/c_119104627.htm |title=中国海军第十五批护航编队凯旋 |publisher=新华网 |date=2014-01-23 |deadurl=yes |archiveurl=https://web.archive.org/web/20151223164619/http://news.xinhuanet.com/mil/2014-01/23/c_119104627.htm |archivedate=2015-12-23 }}</ref>

2014年1月中旬，“衡水”号导弹护卫舰与[[柳州号导弹护卫舰|“柳州”号导弹护卫舰]]、[[岳阳号导弹护卫舰|“岳阳”号导弹护卫舰]]及[[三亚号导弹护卫舰|“三亚”号导弹护卫舰]]等组成编队在训练海域完成多个课目的攻防演练。<ref>{{cite web|url=http://www.chinanews.com/mil/2014/01-20/5753781.shtml |title=南海舰队某驱逐舰支队4艘舰艇组织攻防演练 |publisher=中新网 |date=2014-01-20 }}</ref>

2014年5月5日，“衡水”号导弹护卫舰与“兰州”号导弹驱逐舰在[[黄岩岛|黄岩岛]]附近海域执行任务时与[[美国海军|美国海军]][[第七舰队_(美国海军)|第七舰队]][[蓝岭号两栖指挥舰_(LCC-19)|“蓝岭”号指挥舰]]相遇。<ref>{{cite web|url=http://news.xinhuanet.com/world/2014-05/11/c_126485826.htm |title=美媒：美军自曝与中国军舰黄岩岛“偶遇”  |publisher=新华网 | date=2014-05-11 }}</ref>

2014年6月5日，琼洋浦13073号渔船失联。接到搜救命令后，正在南海某海域执行训练任务的某驱逐舰支队衡水舰紧急奔赴渔船失联海域展开搜救。[[中国人民解放军海军航空兵|海军南海舰队航空兵]]某部出动的搜救飞机率先发现该渔船并将情况通报衡水舰。衡水舰舰载直升机抵达事发海域后，向乘坐在小艇和救生筏上的14名遇险渔民投放了物资。衡水舰与[[南海救115轮|“南海救115”轮]]赶到后一同将遇险渔民全部安全救起、送往三亚。<ref>{{cite web |url=http://www.chinamil.com.cn/jfjbmap/content/2014-06/07/content_78132.htm |title=南海舰队军地联合救起14名遇险渔民 |publisher=中国军网 |date=2014-06-07 }}{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>
