'''河北衡水中学'''，简称'''衡水中学'''、'''衡中'''，是一所位于[[中国|中国]][[河北省|河北省]][[衡水市|衡水市]][[桃城区|桃城区]][[河东街道_(衡水市)|河东街道]]的[[寄宿学校|寄宿]]制[[高级中学|高级中学]]<ref name="hbhz" />，该校因其优异的高考成绩闻名中国。但于此同时，其军事化管理模式和流水线式教学方法也使其备受争议<ref name="renminribao">{{cite news |author=杨柳 |coauthors= |title=衡水中学：没对手 有隐忧 |url=http://cpc.people.com.cn/n/2013/0718/c83083-22235213.html |work= |publisher=[[人民日报|人民日报]] |date=2013-07-18 |accessdate=2014-07-02 }}</ref><ref name="infzm">{{cite news |author=雷磊 |coauthors=藏瑾 |title=衡水中学的“封神”之路 超级高考工厂 |url=http://www.infzm.com/content/94916 |work= |publisher=[[南方周末|南方周末]] |date=2013-10-10 |accessdate=2014-07-02 }}</ref>。

== 简介 ==
衡水中学成立于1951年，本部校园占地223亩，有600余名教职工，6000余名学生<ref name="wangjianyong" />。衡水中学招收高中三个年级与复读班学生，非复读班级约每班70人。从1992年李金池任校长开始，衡水中学成为封闭化、准军事化的全寄宿制学校，此后高考升学率迅速上升至全省前列<ref name="history1993">{{Cite web |url=http://www.hbhz.net/Article_Show.asp?ArticleID=934 |title=河北衡水中学大事记(1993-1995) |accessdate=2014-07-01 |author=张宗谦 |date=2004-02-21 |work= |publisher=河北衡水中学 }}</ref>。

1999年衡水中学成立了公办民助的高中'''衡水滏阳中学'''，主要招收复读生与非本地学生<ref name="infzm" /><ref name="fuyang">{{Cite web |url=http://gaokao.chsi.com.cn/zx/sch/zxgkinfo.action?id=178372807 |title=衡水市滏阳中学 学校概况 |accessdate=2014-07-01 |author= |date= |work= |publisher=阳光高考 中学版 }}</ref>。滏阳中学有100余名教职工，约3000名学生，其中复读班约有一千多人<ref name="kangxinjiang" /><ref name="hsfyzx">{{Cite web |url=http://www.hsfyzx.cn/Article_Show.asp?ArticleID=1118 |title=衡水市滏阳中学简介 |accessdate=2014-07-02 |author=外宣中心 |date=2011-02-21 |work= |publisher=衡水滏阳中学 }}</ref>。

2013年衡水中学与河北泰华锦业房地产开发有限公司合作成立民办学校'''衡水第一中学'''，于衡水市滨湖新区横一路建设新校区<ref name="hebgcdy">{{cite news|author=扈闻 张京生|coauthors=|title=衡水市第一中学经民政部门许可成立为衡中联办|url=http://www.hebgcdy.com/2014/0623/56721.html|work=|publisher=河北共产党员网|date=2014-06-23|accessdate=2014-07-02|deadurl=yes|archiveurl=https://web.archive.org/web/20140708225019/http://www.hebgcdy.com/2014/0623/56721.html|archivedate=2014年7月8日|df=}}</ref>。

2017年4月，衡水中学联办的衡水第一中学在[[浙江省|浙江省]][[平湖市|平湖市]]设立分校，但该校涉嫌提前招生，一时陷入舆论漩涡<ref name="qjwb">{{cite news |author= |coauthors= |title=衡水中学浙江分校涉嫌违规招生 教育厅责令调查|url=http://news.qq.com/a/20170410/003786.htm |work= |publisher=钱江晚报 |date=2017-04-10  |accessdate=2017-04-10  }}</ref>。

== 历史 ==
* 1951年8月，[[桃城区|衡水县]]成立第一所中学并取名为「'''衡水县中学校'''」，招收初中学生，校舍暂借用衡水镇河西街办河西完小西院<ref name="established" />。
* 1952年5月，学校自河西完小西院迁至衡水县城南门口外河东街办现校址<ref name="established" />。
* 1952年11月，学校划归[[石家庄|石家庄]]专区专员公署领导，改名为「'''衡水县初级中学'''」<ref name="established" />。
* 1956年8月，学校增设高中部，并更名为「河北衡水中学」<ref name="established" />。
* 1962年7月，衡水专署成立，暂借学校校舍办公，学校迁往县城西南疙疸头村[[衡水学院|衡水师范]]旧址<ref name="history1962">{{Cite web |url=http://www.hbhz.net/Article_Show.asp?ArticleID=832 |title=河北衡水中学大事记(1962-1970) |accessdate=2014-07-01 |author=张宗谦 |date=2004-01-01 |work= |publisher=河北衡水中学 }}</ref>。
* 1964年10月，学校由疙疸头村搬回现址<ref name="history1962" />。
* 1966年，[[文化大革命|文化大革命]]开始，学校停招新生<ref name="history1962" />。
* 1971年10月，学校改名为「'''衡水县五七技校'''」<ref name="history1971">{{Cite web |url=http://www.hbhz.net/Article_Show.asp?ArticleID=833 |title=河北衡水中学大事记(1971-1978) |accessdate=2014-07-01 |author=张宗谦 |date=2004-01-01 |work= |publisher=河北衡水中学 }}</ref>。
* 1972年初，学校招收高中1-8班、初中1-4班，班级编号一直沿续至今<ref name="history1971" />。
* 1976年3月，「衡水县五七技校」更名为「'''衡水县五七大学'''」<ref name="history1971" />。
* 1978年2月，「河北衡水中学」校名恢复，面向全县统一招生<ref name="history1971" />。
* 1992年8月，李金池任校长，学校撤初中部，初中部新建为'''衡水市第八中学'''<ref name="history1990">{{Cite web |url=http://www.hbhz.net/Article_Show.asp?ArticleID=933 |title=河北衡水中学大事记(1990-1992) |accessdate=2014-07-01 |author=张宗谦 |date=2004-02-21 |work= |publisher=河北衡水中学 }}</ref>。
* 1995年（一说为1993年），学校开始实行全寄宿、准封闭管理。1995年高考升学率开始位列衡水地区第一<ref name="history1993" /><ref name="infzm" />。
* 1999年成立公办民助衡水滏阳中学<ref name="fuyang" />。
* 2004年3月，张文茂任衡水中学校长，李金池调任衡水市教育局长<ref name="history2004">{{Cite web |url=http://www.hbhz.net/Article_Show.asp?ArticleID=2027 |title=河北衡水中学二00四年大事记 |accessdate=2014-07-01 |author=张宗谦 |date=2005-03-23 |work= |publisher=河北衡水中学 }}</ref>。
* 2018年5月，张文茂卸任衡水中学校长，由郗会锁任衡水中学党委书记、校长<ref name="xihuisuo"/>。

== 校园 ==
[[File:Hengshui_High_School_Panoramic_View.JPG|center]]

衡水中学校园本部（西校区）位于衡水市区东南角老城区南部，桃城区英才路228号，东南邻京衡南大街，占地223亩<ref name="wangjianyong" />。校园主要教学楼有格物楼（高一教学楼）、揽月楼（高二教学办公楼）和明志楼（高三教学办公楼），其中格物楼为仿制[[北京四中|北京四中]]教学楼而建，教室均为[[六边形|六边形]]，揽月楼与明志楼相连接，一到四层为教室，五、六层为办公区<ref name="sinahebei">{{cite news |author=中国青年网 |coauthors= |title=衡中：军事化管理 充满各种规矩 |url=http://hebei.sina.com.cn/news/yz/2013-08-07/162663242_2.html |work= |publisher=新浪 |date=2013-08-07 |accessdate=2014-07-02 }}</ref>。其它建筑物有求真馆（办公楼）、科教馆（实验室、机房等）、莘元馆（礼堂）、图书馆、体育馆、滏阳楼、餐饮中心和学生宿舍公寓楼四栋。校园内有中心花园、池塘和运动场，周边建有教职工生活区。<ref name="map">{{Cite web |url=http://www.hbhz.net/Article_Show.asp?ArticleID=9447 |title=衡水中学校园平面图 |accessdate=2014-07-02 |author=高一级部 |date=2012-08-21 |work= |publisher=河北衡水中学 }}</ref>

2010年8月，衡水中学东侧紧邻的衡水市第八中学迁离原址，校园交由衡水中学使用，包括教学楼、宿舍楼和操场，两校园间围墙被拆除，滏阳中学在此招收新生<ref name="history2010">{{Cite web |url=http://www.hbhz.net/Article_Show.asp?ArticleID=7019 |title=河北衡水中学2010年大事记 |accessdate=2014-07-01 |author=外宣中心 |date=2011-02-08 |work= |publisher=河北衡水中学 }}</ref>。

衡水滏阳中学（东校区）位于衡水市桃城区程富路6号，西距京衡南大街约200米，距本部（西校区）校园约1000米，占地约100亩<ref name="hsfyzx" />。校园原属于私立学校平原中学，滏阳中学于2006年搬迁至此。校园主要建筑物有教学楼一栋和学生公寓楼多栋<ref name="fuyangcampus">{{Cite web |url=http://www.hsfyzx.cn/Article_Show.asp?ArticleID=2574 |title=衡水滏阳中学平面图 |accessdate=2014-07-02 |author=张宗谦 |date=2013-06-28 |work= |publisher=衡水滏阳中学 }}</ref>。

衡水中学于2013年启动新校区「衡水市第一中学」建设，将于距离本部约10公里的衡水滨湖新区横一路路北建设新校区，占地约600亩<ref name="hsrb2013">{{Cite web |url=http://www.hsrb.com.cn/a/news/huchengyaowen/2014/0107/81839.html |title=衡水中学2013年十大亮点回眸 |accessdate=2014-07-02 |author=衡水新闻网—衡水日报 |date=2014-01-07 |work= |publisher=衡水新闻网 }}</ref>。新校区一期占地约400亩，预计2014年底竣工<ref name="soufun">{{Cite web |url=http://newhouse.hs.soufun.com/2013-08-21/10821192_all.htm |title=衡水中学新校区落户滨湖新区 催热周边楼盘 |accessdate=2014-07-02 |author= |date=2013-08-21 |work= |publisher=[[搜房网|搜房网]] }}</ref>。

== 校园生活 ==
[[File:Hengshui_High_School_Running_Exercise.JPG|300px]]
衡水中学的校园规章与活动颇具特色，成为衡水中学形象的重要组成部分。

=== 作息时间 ===
衡水中学严格规定学生的作息时间。每天5:30起床，洗漱整理内务，5:40到操场开始跑操，跑操结束开始早读，6:30开始各年级分批吃早饭，早饭后有「早预备」自习，然后开始上课，每节课40分钟。上午有五节课，第三节课后跑操（课间操）。12:45到13:45为午休（南校区提前十分钟）。下午有五节课，各年级分批吃晚饭后于18:50到19:10收看新闻（《[[新闻30分|新闻30分]]》录播），晚上有三节课为自习，21:50结束，22:10熄灯就寝。各节课开始时间与就寝时间前两分钟为预备时间，要求「进入状态」，例如上课前两分钟开始学习，不得说笑、喝水，就寝前两分钟须躺在床上进入就寝状态。<ref name="kangxinjiang" /><ref name="infzm" />

=== 纪律 ===
对班级纪律、卫生、德行、宿舍内务的考量量化为分数并加权汇总，简称为「量化」。量化由各年级以规定的方式评定与计算，与考试成绩一起决定了对班级的奖惩和老师的绩效。<ref name="lianghua">{{Cite web |url=http://edu.qq.com/a/20131015/010516.htm |title=衡水中学的量化教育：吃饭40分钟不是4分钟 |accessdate=2014-07-02 |author=现代快报 |date=2013-10-15 |work= |publisher=[[腾讯|腾讯]]教育 }}</ref><ref name="weiji">{{Cite web |url=http://sh.qq.com/a/20140627/008817.htm |title=神一样的衡水中学有何“秘密” |accessdate=2014-07-02 |author=上海观察 |date=2014-06-27 |work= |publisher=腾讯·大申网 }}</ref>

衡水中学规定了详细的违纪行为惩罚措施，违纪行为如自习或上课预备时说话、回头、喝水，就寝后交谈、学习、进食、裸睡、上厕所会扣除量化分数或发放违纪通知单<ref name="vista" />。而多次违纪或携带手机等电子产品、「男女生非正常接触」等行为则会被要求回家反省若干星期。「男女生非正常接触」简称「非触」，定义为男女生单独相处、共餐等行为，这些行为被学校认为具有[[早恋|早恋]]倾向<ref name="weiji" />。衡水中学教室安装有[[摄像头|摄像头]]，所有老师可以控制各摄像头并录制视频<ref name="beijing518">{{Cite web|url=http://hebei.beijing518.com/html/2012/xykb_0108/5382.html|title=话说衡水中学|accessdate=2014-07-03|author=|date=2012-01-08|work=|publisher=石家庄中考网|deadurl=yes|archiveurl=https://web.archive.org/web/20140715014753/http://hebei.beijing518.com/html/2012/xykb_0108/5382.html|archivedate=2014年7月15日|df=}}</ref>。

=== 作业与考试 ===
衡水中学施行「自习考试化，考试高考化」，即学科自习课布置试题，以考试标准要求<ref name="dwyz">{{Cite web|url=http://www.dwyz.com.cn/n2742c47.aspx|title=赴衡水中学考察报告|accessdate=2014-07-02|author=校办室|date=2013-01-11|work=|publisher=大悟县第一中学|deadurl=yes|archiveurl=https://web.archive.org/web/20140714145939/http://www.dwyz.com.cn/n2742c47.aspx|archivedate=2014年7月14日|df=}}</ref>。衡水中学每周有一次综合测验，每月有一次高考标准的大型调研考试。从小测试到考试均有学生、班级排名<ref name="theotherside">{{Cite web |url=http://view.163.com/special/reviews/gaokao1119.html |title=超级县中：十大工序打造的高考集中营 |accessdate=2014-07-02 |author=王蕾 |date=2013-11-19 |work= |publisher=网易新闻 }}</ref>。作业与考试试题皆由老师编写<ref name="xinjingbao">{{cite news |author=宋识径 |coauthors= |title=衡水模式：大学生的加工厂？(1) |url=http://epaper.bjnews.com.cn/html/2012-06/09/content_345360.htm?div=-1 |work= |publisher=[[新京报|新京报]] |date=2012-06-09 |accessdate=2014-07-03 }}</ref>。

=== 假期 ===

学期中衡水中学约三周放假一次，时间从高一年级的约一天到高三的一晚不等，[[中华人民共和国国庆节|国庆]]等节日放假时间一般短于[[中华人民共和国节日与公众假期|法定假期]]<ref name="jiaqi">{{Cite web |url=http://edu.qq.com/a/20131015/011228.htm |title=国庆只放假两天 超级中学是典型还是“怪胎” |accessdate=2014-07-02 |author=和讯网 |date=2013-10-15 |work= |publisher=腾讯教育 }}</ref>。

=== 跑操 ===
跑操一般以班级为单位，同学紧密排列为方阵快速绕圈跑动，由体育委员引导步伐。衡水中学的学生每天至少进行早晨与上午课间两次跑操，有时体育活动课或下午课间也会跑操<ref name="infzm" />。在跑动期间各班级会高喊口号，每个班级都有自己的口号，称为「班号」，多为一组四字词语，如「信心百倍，斗志昂扬，全力以赴，铸我辉煌」<ref name="vista">{{Cite web |url=http://edu.sina.com.cn/zxx/2012-07-10/0946347888_5.shtml |title=衡水中学：“第三世界”如何完成高考神迹 |accessdate=2014-07-02 |author=王鹏 |date=2012-07-10 |work= |publisher=新浪教育 }}</ref>。每个班级队伍旁均有[[學生會|學生會]]成员根据队伍紧密、整齐程度和口号响亮程度等标准打分。在整队等待跑操时，同学携带书籍纸条诵读。<ref name="chongqing">{{cite news|author=林祺|coauthors=匡怡蒙|title=“超级高考工厂”争议中前行|url=http://cqcbepaper.cqnews.net/cqcb/html/2014-04/23/content_1737126.htm|work=|publisher=重庆晨报|date=2014-04-23|accessdate=2014-07-03|deadurl=yes|archiveurl=https://web.archive.org/web/20140714183026/http://cqcbepaper.cqnews.net/cqcb/html/2014-04/23/content_1737126.htm|archivedate=2014年7月14日|df=}}</ref>

据传跑操为衡水中学自[[江苏|江苏]][[南通|南通]]的学校学习而来<ref name="nantong">{{Cite web|url=http://yanzhao.yzdsb.com.cn/system/2010/07/05/010576004.shtml|title=衡水二中跑操非个例 衡水好中学都这样|accessdate=2014-07-02|author=李海菊|date=2010-07-05|work=|publisher=[[燕赵都市报|燕赵都市网]]|deadurl=yes|archiveurl=https://web.archive.org/web/20140714183753/http://yanzhao.yzdsb.com.cn/system/2010/07/05/010576004.shtml|archivedate=2014年7月14日|df=}}</ref>。衡水中学宣传处称，不做广播体操而跑操，因学生多而场地不够，为锻炼身体只能跑步<ref name="xinjingbao" />。

== 影响与评论 ==
[[File:Banner_of_Hengshui_High_School_on_G45_Hengshui_section_(20170114175715).jpg|thumb]]衡水境内设立的广告标语<ref>{{cite news|title=衡水：中学高速路上做广告|url=http://gaozhong.eol.cn/news/201505/t20150509_1257000.shtml|date=2015-05-09|publisher=中学教育在线}}</ref>]]
衡水中学规模巨大，高考成绩突出，被认为是典型的「[[超级中学|超级中学]]」<ref name="superschool">{{Cite web |url=http://learning.sohu.com/s2012/superschool/ |title=中国“超级中学”铁骨铮铮 |accessdate=2014-07-03 |author=富芳芳 |date=2012-07-09 |work= |publisher=搜狐教育 }}</ref><ref name="superschool2">{{Cite web |url=http://www.infzm.com/content/82926 |title=造“神” 超级中学的源起与隐忧 |accessdate=2014-07-03 |author=赵一海 |date=2012-11-15 |work= |publisher=南方周末 }}</ref>。衡水中学以具有优势的高考成绩和特殊的管理方式受到了广泛的关注，而衡水中学也利用知名度承办或举办各类会议、举办校园开放日，以便外界参观交流。<ref name="infzm" />。

衡水中学宣传其倡导[[素质教育|素质教育]]，并且总结经验为「素质教育更能提高升学率」<ref name="suzhijiaoyu">{{cite news|author=|coauthors=|title=素质教育更能提高升学率|url=http://www.he.xinhua.org/test/2006-11/27/content_8627514.htm|work=|publisher=[[新华网|新华网]]河北频道|date=2006-11-22|accessdate=2014-07-03|deadurl=yes|archiveurl=https://web.archive.org/web/20130405080919/http://www.he.xinhua.org/test/2006-11/27/content_8627514.htm|archivedate=2013年4月5日|df=}}</ref>，但外界多认为衡水中学高升学率靠的是「半军事化的封闭管理，让师生长期处于亢奋状态以及大量做题挖掘应试潜力」<ref name="renminribao" />。衡水中学受到了广泛的争议与批评，被称为「高考巨无霸」、「[[血汗工廠|血汗工廠]]」。<ref name="theotherside" /><ref name="superschool" />

=== 交通与经济 ===
衡水中学学生人数多、招生范围广、放假间隔时间长，每次放假时对周边的京衡南大街等街道造成了严重的拥堵。衡水交警部门制定《关于衡中放假期间联勤工作方案》，派出工作人员保障衡水中学附近通畅。<ref name="hssgaj" /><ref name="infzm" />

衡水中学放假时间短，非本地学生多，很多学生家长在放假时赴衡水居住以与子女见面。另外，在衡水中学召开会议期间有大量赴会参观者在衡水就住，甚至导致衡水市区的宾馆短缺而需要到几十公里以外住宿。这些人员的住宿、交通、餐饮需求对衡水市的经济作出贡献<ref name="infzm" />。衡水中学受到衡水地方政府重视与支持，被称为「衡水对外开放的亮丽窗口」<ref name="chengui">{{cite news |author=衡水日报 |coauthors= |title=杨新丽副市长参加衡水中学扩建工作的调研 |url=http://www.hengshui.gov.cn/HSZW/SZZC/YXL/LDHD7/2009/09/03/content_30116.shtml |work= |publisher=衡水市人民政府 |date=2009-09-03 |accessdate=2014-07-03 }}</ref>。一些人则认为衡水中学是政府以教育成绩为政绩，默许学校违背规律发展的典型<ref name="chaojichengjidan">{{Cite web |url=http://www.jyb.cn/opinion/jcjy/201406/t20140625_587604.html |title=“超级成绩单”折射教育生态失衡 |accessdate=2014-07-03 |author=汪明 |date=2014-06-25 |work= |publisher=中国教育新闻网 }}</ref>。

衡水市在衡水中学外的多所中学，包括[[衡水市第二中学|衡水市第二中学]]、衡水市第十三中学、河北冀州市中学、河北武邑中学、河北枣强中学等，其高考升学率均在河北省内较为突出，且均采用与衡水中学相似的全寄宿制、准军事化管理，有相似的校园文化，被认为是衡水中学的模仿者，也成为一些学生在衡水中学之下的选择，其中衡水市第二中学提出了「低进优出」口号<ref name="infzm" />。据统计，有4万非本地学生和陪读者生活在衡水市。<ref name="hbrb">{{cite news|author=王博|coauthors=|title=被教育军团改写的城市|url=http://hbrb.hebnews.cn/html/2014-06/18/content_25857.htm|work=|publisher=河北日报|date=2014-06-18|accessdate=2014-07-03}}{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>

=== 生源 ===
据规定衡水中学的招生范围限于桃城区<ref name="edunews">{{Cite web |url=http://www.edunews.net.cn/porj6/tbch/11306zgm/zb09.htm |title=河北衡水：一所中学引发的改革 |accessdate=2014-07-02 |author=《教育》旬刊记者 |date= |work=《教育》 |publisher=教育新闻网 }}</ref><ref name="jichujiaoyu">{{cite journal |author=杨占巷 唐文博 张永 董安民 |coauthors= |year=2006 |month= |title=解读一个教育神话——河北省衡水中学探秘 |journal=《基础教育》 |volume= |issue=4 |pages=4-11 |id= |url= |accessdate=2014-07-01 |quote=衡水中学其实“名不符实”，招生范围只限衡水市的桃城区，全区人口只有30多万，是河北的重点中学中招生人口范围最小的学校之一，而河北省重点中学中招生人口范围最大的学校达到了将近1000万人。 }}</ref>，河北省教育厅基础教育处的工作人员则表示衡水中学「只能在衡水市的市域范围内招生」<ref name="60ban">{{cite news |author=吴喆华 |coauthors= |title=河北衡水中学高一拟招60班 县区中学陷生源流失窘境 |url=http://www.chinanews.com/edu/2014/06-19/6296403.shtml |work= |publisher=中国新闻网 |date=2014-06-19 |accessdate=2014-07-03 }}</ref>。而衡水中学（包括滏阳中学）生源事实上来自全河北省各地市，甚至远达[[山东|山东]]、[[北京|北京]]、[[内蒙古|内蒙古]]，并且在吸收生源上形成了「[[马太效应|马太效应]]」<ref name="infzm" />。近年来河北省高中招生竞争激烈，[[邢台|邢台]]市区、[[赞皇|赞皇]]等地[[中考|中考]]高分学生升入本地高中比例只有百分之三四十，出现了针对外地招生老师的暴力事件，[[河北辛集中学|河北辛集中学]]等省内学校在生源上采取了提前录取、提前考试、提前开学、阻止中考、隐瞒成绩等措施<ref name="zanhuang">{{cite news |author=刘刚 |coauthors= |title=河北高中“抢生源大战”(1) |url=http://epaper.bjnews.com.cn/html/2012-07/16/content_356942.htm?div=-1 |work= |publisher=新京报 |date=2012-07-16 |accessdate=2014-07-03 }}</ref><ref name="xinji">{{cite news |author=吴喆华 |coauthors= |title=河北辛集一中争夺生源提前中考 催生五大“怪招” |url=http://china.cnr.cn/yaowen/201406/t20140618_515680656_1.shtml |work= |publisher=中国广播网 |date=2014-06-18 |accessdate=2014-07-03 }}</ref>。一些人认为衡水中学造成了其它学校空心化，导致了河北省各高中在生源上的恶性争夺，造成教育生态的失衡<ref name="renminribao" /><ref name="ping1">{{cite news |author= |coauthors= |title=衡水中学成超级高考工厂 评：破坏均衡求奇迹不值得 |url=http://china.cnr.cn/yaowen/201404/t20140426_515379522.shtml |work= |publisher= |date=2014-04-26 |accessdate=2014-07-03 }}</ref><ref name="chenfang">{{Cite web |url=http://view.news.qq.com/a/20140624/027907.htm |title=衡水中学的喜报不是教育的荣耀 |accessdate=2014-07-03 |author=陈方 |date=2014-06-02 |work= |publisher=新京报 }}</ref>。

另一方面，慕名衡水中学的学生众多，为进入衡水中学就读，一些学生将户口迁至衡水以便于当地参加中考<ref name="hbrb" />，而进入衡水中学的外地官员子女与亲属为衡水中学提供了地方政府外的[[关系_(社会学)|关系]]资源<ref name="infzm" />。

2017年6月1日，河北省教育厅印发了《河北省教育厅关于对河北衡水中学衡水第一中学办学行为进行整改的通知》，该通知明确指出，两所学校程度不同存在不规范甚至违规办学招生情况。通知还要求，河北衡水中学法人代表不得代行衡水第一中学法人代表权力<ref>{{Cite web |url=http://news.163.com/17/0616/12/CN25LM1G000187VE.html|title=河北教育厅：衡水中学、衡水一中存在不规范甚至违规办学招生 |accessdate=2017-06-16 |author= |date=2017-06-16 |work= |publisher= 澎湃新闻网(上海) }}</ref>。

=== 管理 ===
衡水中学的严格的时间管理被很多人评论为限制了学生的自由，缺乏人性，会扼杀创造力<ref name="jungui">{{Cite web |url=http://www.chinanews.com/edu/2013/07-19/5060008.shtml |title=河北衡水中学“军规”：不能退步不能说话不能笑 |accessdate=2014-07-03 |author= |date=2013-07-19 |work= |publisher=中国广播网 }}</ref>。衡水中学副校长康新江对此回应说，精确的时间管理以及午休不得看书等措施保证了学生每天有8小时睡眠，且衡水中学每年提供了几十项大型活动<ref name="kangxinjiang" />。有人分析认为衡水中学实际上是在使用现代公司的[[績效管理|绩效]]量化等管理手段<ref name="infzm" />。此外，有衡水中学毕业生认为衡水中学的严格管理磨练了学生的意志，培养了隐忍坚强的品格<ref name="xmu">{{cite news |author=刘恩会 |coauthors=陈若晗 李辰 林方舟 蔡凉凉 周自扬 薛熙来 |title=【专题聚焦】衡水中学：没那么残酷也没那么不得人心 |url=http://tw.xmu.edu.cn/s/134/t/410/43/59/info148313.htm |work= |publisher=共青团[[厦门大学|厦门大学]]委员会 |date=2014-05-01 |accessdate=2014-07-03 }}</ref><ref name="pinglun">{{cite news|author=卜松竹|coauthors=|title=“超级中学”有点儿不合时宜？|url=http://gzdaily.dayoo.com/html/2014-07/03/content_2677678.htm|work=|publisher=[[广州日报|广州日报]]|date=2014-07-03|accessdate=2014-07-03|deadurl=yes|archiveurl=https://web.archive.org/web/20140703165424/http://gzdaily.dayoo.com/html/2014-07/03/content_2677678.htm|archivedate=2014年7月3日|df=}}</ref>。

衡水中学重视学校文化的建设，提出「打造『精神特区』」的口号<ref name="jingshentequ">{{cite news |author=易鑫 |coauthors= |title=衡水中学为学生打造幸福的“精神特区” |url=http://hs.hebnews.cn/2011-09/25/content_2281980.htm |work= |publisher=河北新闻网 |date=2011-09-25 |accessdate=2014-07-03 }}</ref>。康新江认为，校园文化是衡水中学高考成功的关键，并且是不可复制的<ref name="kangxinjiang" />。对此也有人批评「精神特区」实质上是控制学生的精神，使其完全服从校方<ref name="wulichuan">{{Cite web |url=http://dz.xdkb.net/html/2013-10/15/content_299622.htm |title=伍里川：超级中学只是怪胎，何时成了“典型” |accessdate=2014-07-03 |author=伍里川 |date=2013-10-15 |work= |publisher=[[现代快报|现代快报]] }}</ref>。

也有一些人认为，衡水中学顺应了学生考入[[重点大学|重点大学]]的需求，是对学生及其家庭负责的表现。[[胡锡进|胡锡进]]评论说：「在衡水那样的地方，这所中学帮那么多孩子改变了命运，对他们的家庭来说，我相信这比素质教育更重要。」<ref name="pinglun" />

此外，有人认为衡水中学是高考规则的必然产物，故衡水中学的问题根源在于教育体制。[[人民日报|人民日报]]微博称，衡水中学「根植于[[应试教育|应试教育]]的土壤」，「是中国教育无可奈何却又真实存在的现状」<ref name="pinglun" />。衡水中学工会主席梁辉也说「除非高考发生重大改变 否则衡中模式不会改变」<ref name="chongqing" />。有人指出，衡水中学反映了中国教育资源不均衡的特点，河北省缺乏重点大学，高考人数多，竞争激烈，学生只能寄希望于严格训练<ref name="weiji" />。

=== 争议===
虽然衡水中学以其独特的教学模式闻名于全国，但是这种教育模式也受到了多方的质疑。

衡水中学是最早在[[河北省|河北省]]实行全封闭[[军事化管理|军事化管理]]的监狱，学生在校期间自由受到限制，诸如中午禁止午休上厕所等管理规定引起了某些人士的厌恶。河北省高中在衡水中学的影响下，纷纷向衡水中学学习全封闭化军事管理的经验，在其他河北省高中学生中带来了负面评价。

也有分析人士认为，衡水中学的封闭式[[应试教育|应试教育]]培养出的学生只是高分应试机器，创新能力、社交能力和学术能力令人堪忧。也有人认为这是在中国高考大环境下和河北省考生人数多的条件下产生的必然结果。

在衡水中学的影响下，以河北[[唐山一中|唐山一中]]为首的一批实行宽松管理的河北高中均纷纷开始采取封闭式军事化管理，对河北省的高中教育带来了一定影响。

衡水中学一直大批量地向[[北京大学|北京大学]]、[[清华大学|清华大学]]等名校输送高分生源，但有媒体认为，这种升学率极高的现象是由该校严苛的应试教育制度导致的，亦有网友认为，这种严苛的应试教育制度会扼杀学生们的创新思维，并践踏教育精神<ref>[http://news.xinhuanet.com/edu/2012-06/09/c_123258056.htm 衡水中学军事化备战高考引争议 大学生的加工厂?] {{Wayback|url=http://news.xinhuanet.com/edu/2012-06/09/c_123258056.htm |date=20120613173752 }}</ref>。

在衡水中学进入浙江后，浙江省教育厅基础教育处处长方红峰也对衡水中学的教学模式作出质疑。他认为，衡水中学落户浙江，办学程序是合法的，但是衡水中学作为“高考工厂”，与浙江教育理念相抵触。他表示“这所学校是应试教育的典型，眼睛里只有分数没有人，跟我们浙江以人为本的素质教育理念不符合，我们浙江不需要<ref name="qjwb"/>。”
