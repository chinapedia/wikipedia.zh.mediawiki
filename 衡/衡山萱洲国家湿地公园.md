'''衡山萱洲国家湿地公园'''，位于[[湖南省|湖南省]][[衡阳市|衡阳市]][[衡山县|衡山县]]和[[衡东县|衡东县]]境内<ref name=":0">{{Cite news|url=http://news.163.com/15/0118/11/AG856OTV00014SEH.html|title=衡山萱洲国家湿地公园项目获批试点|last=李|first=春祺|date=2015年1月18日|work=红网|accessdate=}}</ref>。

== 范围 ==
衡山萱洲国家湿地公园规划总面积占地2740公顷，范围涉及衡山、衡东两个县共5个乡镇50个村<ref name=":0" />。

== 生态 ==
衡山萱洲国家湿地公园位于湖南两条鸟类迁徙通道的交接处，已经发现[[琵鷺|琵鹭]]、[[白鹭|小白鹭]]、[[灰头麦鸡|灰头麦鸡]]、[[翠鸟|翠鸟]]、[[苍鹭|苍鹭]]、[[白头鹎|白头翁]]等85种鸟类，其中3种列入近危级别，[[国家二级重点保护野生动物名录|国家二级保护动物]]9种<ref name=":1">{{Cite news|url=http://www.chinanews.com/sh/2016/11-01/8050116.shtml|title=湖南衡山发现“鸟类中的大熊猫”中华秋沙鸭|last=周|first=勉|date=2016年11月1日|work=新华网|accessdate=}}</ref>。

2016年10月27日，公园工作人员在日常监测中发现4只[[中华秋沙鸭|中华秋沙鸭]]，为国家一级保护动物<ref name=":1" />。

== 公园建设 ==
衡山萱洲国家湿地公园于2014年12月31日被[[国家林业局|国家林业局]]批准成为[[中国湿地公园|国家湿地公园]]试点<ref name=":0" />。

2017年，衡山县政府十三届二次常务会议召开，会议原则同意出台《湖南衡山萱洲国家湿地公园保护管理办法》，要求高度重视萱洲国家湿地公园试点建设工作<ref>{{Cite news|url=http://news.eastday.com/eastday/13news/auto/news/china/20170122/u7ai6431715.html|title=衡山县政府十三届2次常务会议召开|last=李|first=佳辉|date=2017年1月22日|work=红网|accessdate=}}</ref>。

衡阳市人大代表李兰英建议，要将萱洲国家湿地公园等[[衡山县|衡山县]]的旅游资源和[[南岳区|南岳区]]的旅游胜地[[衡山|衡山]]有效整合，优势互补<ref>{{Cite news|url=http://www.hyqss.cn/2017/january/290EDA54.html|title=整合南岳区、衡山县的旅游资源|last=王|first=丹|date=2017年1月11日|work=衡阳日报|accessdate=}}</ref>。

== 旅游活动 ==
2016年，衡山萱洲国家湿地公园举行首届摄影大奖赛，以“打造绿色河流廊道，推进生态文明建设”为主题<ref>{{Cite news|url=http://hn.rednet.cn/c/2016/07/08/4030170.htm|title=衡山萱洲国家湿地公园首届摄影大奖赛启幕|last=汪|first=洋|date=2016年7月8日|work=红网|accessdate=}}</ref><ref>{{Cite news|url=http://www.shidi.org/sf_3BF32DC3559F4F6FA85EA9AAEECFBFB4_151_19DB8D08617.html|title=湖南衡山萱洲国家湿地公园 首届摄影大奖赛作品评选活动圆满结束|last=|first=|date=2016年12月15日|work=湿地中国|accessdate=}}</ref>。
