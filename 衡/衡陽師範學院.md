'''衡阳师范学院'''位于[[湖南省|湖南省]][[衡阳市|衡阳市]]。<ref>{{cite web |url=http://news.sina.com.cn/o/2013-11-22/104128782015.shtml |title=湖湘大学堂走进衡阳师范学院 院长刘沛林谈诗意的栖居 |accessdate=2013 |publisher=新浪网 |work= |language=zh }}</ref><ref>{{cite web |url=http://www.hunan.gov.cn/zwgk/hnyw/szdt/201212/t20121210_742042.html |title=教育部正式批准衡阳师范学院承担“国培计划” |accessdate=2009 |publisher= |work= |language=zh |deadurl=yes |archiveurl=https://web.archive.org/web/20140318194133/http://www.hunan.gov.cn/zwgk/hnyw/szdt/201212/t20121210_742042.html |archivedate=2014-03-18 }}</ref>校园占地2166亩。其前身可追溯到1904年的湖南官立南路师范学堂。1958年，成立衡阳师范专科学校，是全国办学历史最悠久的师专之一。1999年由原'''衡阳师范高等专科学校'''与原'''衡阳教育学院'''合并而成。2001年，原'''湖南第三师范学校'''（其前身为湖南官立南路师范学堂）并入。拥有17个系（院），39个本科专业，有1个国家特色专业建设点，3个省级重点建设专业，7个省普通高校特色专业，12门省级精品课程，3个省级重点建设学科，3个省级研究基地，1个省级重点实验室，2个省级基础课示范实验室，1个省级实践教学示范中心，1个省级高校产学研合作示范基地。学院现有在编教职工1100余人，专任教师700余人，其中教授、副教授240余人，有省级科技创新团队1个、省级教学团队4个，现有在校学生16000余人。
