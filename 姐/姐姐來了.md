《'''姐姐來了'''》（{{lang-ja|お姉ちゃんが来た}}），是[[安西理晃|安西理晃]]在[[竹書房|竹書房]]《漫畫人生》雜誌連載的[[四格漫畫|四格漫畫]]作品，簡體中文電子漫畫版由[[布卡漫畫|布卡漫畫]]代理。在《漫畫人生》2013年12月號封面宣佈了動畫製作決定消息<ref>{{cite web|url=http://www.animenewsnetwork.com/news/2013-10-16/oneechan-ga-kita-sisterly-love-4-panel-manga-gets-tv-anime|title=''Oneechan ga Kita'' Sisterly Love 4-Panel Manga Gets TV Anime|publisher=[[Anime_News_Network|Anime News Network]]|date=2013年10月16日|accessdate=2014年3月6日|language=en}}</ref>。2014年1月[[電視動畫|電視動畫]]開始播放，為[[泡麵番|泡麵番]]。<ref>{{Cite web |url= http://natalie.mu/comic/news/101524 |script-title= ja:弟大好きな変人姉描く「お姉ちゃんが来た」がTVアニメ化 |author=  |date= 2013年10月17日 |publisher= [[ナタリー_(ニュースサイト)|コミックナタリー]]  |language= ja |accessdate= 2014年1月9日 |quote=  }}</ref>

== 故事劇情 ==
水原朋也是個普通的國中生，由於父親突然再婚的關係，家裏來了一個超級戀弟的姐姐水原一香，清純善良的朋也和沒有血緣關係的戀弟狂姐姐，兩人從此在同一屋簷下的同居生活會產生什麼樣的變化。這是圍繞名為水原朋也的少年與姐姐水原一香之間的日常展開。

== 角色介紹 ==
; {{nihongo|水原朋也|水原 朋也（みずはら ともや）|配音：[[愛美|愛美]]}}
: 本作的主角，13歲的初中生。
: 因父親再婚而多了一個姐姐水原一香，也為此感到困擾。
: 經常因為姐姐的親暱而感到害羞。
: 擅長做點心。

; {{nihongo|水原一香|水原 一香（みずはら いちか）|配音：[[長妻樹里|長妻樹里]]}}
: 本作女主角，17歳的女校高中生。
: 從小時候就希望有個弟弟。
: 喜歡沒有血緣關係的弟弟，房間裡有很多朋也樣子的玩偶。經常拍他的近照，相冊數量一直增加

; {{nihongo|藤咲美奈|藤咲 美奈（ふじさき みな）|配音：[[木戶衣吹|木戶衣吹]]}}
: 朋也的同學，暗戀着孝喜，外表純潔可愛，內心極為腹黑高傲。
: 每次朋也的姐姐一來就會被搶風頭，因而討厭水原一香，也進而不太喜歡水原朋也。但在游泳園感受到了朋也的溫柔，對朋也的感情有了變化。動畫第10話中，表現出暗戀孝喜。

; {{nihongo|早坂琉璃|早坂 ルリ（はやさか ルリ）|配音：[[南央美|南央美]]}}
: 早坂孝喜的姐姐，喜欢对弟弟各种揉虐过肩摔。
: 最初誤會藤總一郎是個變態，後來在眾人的解釋下，終於解開誤會。
: 現在劇情暗示她對望月麻里奈有好感，也一直知道她對藤總一郎的感情。
; {{nihongo|望月麻里奈|望月 マリナ（もちづき マリナ）|配音：[[井上麻里奈|井上麻里奈]]}}
: 巨乳，曾经因为朋也红脸看了几眼，被一香扇了一击奶光。
: 雖然看上去很可靠，但是非常少女心和喜歡可愛的東西。
: 為四分之一的俄羅斯混血兒。喜歡自己的宅男表哥藤總一郎。

; {{nihongo|花園美鶴|花園 美鶴（はなぞの みつる）|配音：[[小林優|小林優]]}}
: 朋也的好友兼邻居，略猥琐，喜欢漂亮姐姐们，被一香鉴定为可爱值一般的普通男孩。
: 成績在全級20名內，運動神經好，唱歌也好聽，但從外表看不出來。

; {{nihongo|早坂孝喜|早坂 孝喜（はやさか こうき）|配音：[[富樫美鈴|富樫美鈴]]}}
: 成绩优秀，硬汉派，经常被姐姐过肩摔揉虐，而且是个隐藏姐控。

; {{nihongo|水原夕子|水原 夕子（みずはら ゆうこ）|配音：{{tsl|ja|森澤芙美}}}}
: 水原一香的母親，水原朋也的繼母。
: 在一次偶遇水原正也後，對他一見鍾情。在跟蹤後發現他在百貨公司的美容院工作和住處，於是經常光顧並在他家前埋伏，然後假裝偶遇。
: 會教水原一香奇怪的道理。

; {{nihongo|水原正也|水原 正也（みずはら まさや）|配音：[[濱田賢二|濱田賢二]]}}
: 水原朋也的父親，一香的繼父。百貨公司的美容院的店長。

; {{nihongo|藤總一郎|藤 総一郎（ふじ そういちろう）|配音：[[高橋孝治|高橋孝治]]}}
: 麻里奈的表哥，很宅，只是長得帥而已。俄羅斯混血兒。
: 小時候被經常更「外國人」的人欺負，所以沉迷於漫畫的世界裏，因為沒有人會欺負他。雖然被欺負，但很溫柔，不會欺負人。

; {{nihongo|朋也媽媽|朋也の母|配音：小林優}}
: 在朋也幼稚園時因病去世。
