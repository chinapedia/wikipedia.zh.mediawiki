'''衞生防護中心'''（[[英文|英文]]：'''C'''entre for '''H'''ealth '''P'''rotection，[[縮寫|縮寫]]：'''CHP'''）是[[香港特別行政區|香港特別行政區]][[衞生署_(香港)|衞生署]]屬下的流行病防護中心，於2004年6月1日正式成立，負責加強[[香港市民|香港市民]]對香港預防及控制各種流行疾病，包括傳染病的認識，並會調查各種流行疾病感染個案，監察疾病通報情況，以及與其他國家交換有關疾病感染個案情況。現任衞生防護中心總監為[[黃加慶|黃加慶]]醫生，顧問醫生為[[張竹君|張竹君]]醫生。

== 標誌 ==
衞生防護中心的標誌以[[深藍色|深藍色]][[英文|英文]][[縮寫|縮寫]]「CHP」組合而成，「HP」兩個[[羅馬字母|羅馬字母]]被「C」字母所包圍，意喻健康防護任務放在最中心的位置。採用深藍色意喻「簡潔清晰、專業」，前總監[[梁栢賢|梁栢賢]]就任時亦有份參與構思創造標誌<ref>[http://orientaldaily.on.cc/cnt/news/20130513/00176_029.html 「CHP」標誌 設計有段古] 《東方日報》 2013年5月13日</ref>。

== 總部 ==
衞生防護中心辦公室選址位於前九龍醫院護士宿舍，毗連[[醫院管理局|醫院管理局]]總部[[醫院管理局大樓|醫院管理局大樓]]，理由為直接及方便即時溝通。

== 歷史 ==
2003年4月1日，[[香港|香港]]爆發[[嚴重急性呼吸系統綜合症|嚴重急性呼吸系統綜合症]]後，時任[[香港行政長官|香港行政長官]][[董建華|董建華]]隨即著手研究在香港設立一個疾病控制及預防中心組織，以對抗、治療及預防任何[[傳染病|傳染病]]在[[香港社會|香港社會]]爆發之可能性。香港政府專家報告建議成立類似美國的疾病控制中心，以統一防護及控制傳染病。2003年10月，[[梁栢賢|梁栢賢]]臨危受命籌備組織衞生防護中心，初期所面對最大挑戰為與[[醫院管理局|醫院管理局]]建立呈報機制。翌年6月1日，[[香港賽馬會|香港賽馬會]]撥款5億港元，正式成立衞生防護中心<ref>[http://orientaldaily.on.cc/cnt/news/20130513/00176_028.html 沙士催生衞生防護中心] 《東方日報》 2013年5月13日</ref>。

== 總部 ==
衞生防護中心的總部設在[[九龍|九龍]][[亞皆老街|亞皆老街]]147C號衞生防護中心大樓，由前[[九龍醫院|九龍醫院]]護士宿舍改裝而成。而屬下的公共衞生化驗服務處及公共衞生服務處則設在九龍石硤尾南昌街382號公共衞生檢測中心。
