《'''禁酒令'''》（{{lang-en|'''''Prohibition'''''}}，又名《'''美國禁酒令'''》）是一部2011年在[[電視電影|電視]]上播放的[[紀錄片|紀錄片]]，導演是[[肯·伯恩斯|肯·伯恩斯]]和{{link-en|林恩·諾維克|Lynn Novick}}，旁白是{{link-en|彼得·柯尤特|Peter Coyote}}。本片在2011年10月2日和2011年10月4日之間播放<ref>{{Cite web |url = http://www.imdb.com/title/tt1950799/episodes |title = Prohibition (2011 TV Mini-Series): Episode List |publisher = IMDb |language = en}}</ref>。本片是大幅取材自{{link-en|丹尼爾·奧克倫特|Daniel Okrent}}的2010年著作《''Last Call: The Rise and Fall of Prohibition''》。<ref>{{Cite web |url = http://articles.latimes.com/2011/oct/01/entertainment/la-et-prohibition-20111001 |title = A long and sober look at Prohibition history |author = Robert Lloyd |date = 2011-10-01 |publisher = Los Angeles Times |language = en}}</ref>

== 概要 ==
《禁酒令》講述[[酒精飲料|酒精飲料]]的消費和效果如何與美國許多不同的文化力量有關，包括{{link-en|移居至美國|Immigration to the United States|移民入境}}、[[美國婦女選舉權|美國婦女選舉權]]和{{link-en|美國入息稅|Income tax in the United States|入息稅}}。最終[[禁酒運動|禁酒運動]]導致[[美國憲法第十八修正案|美國憲法第十八修正案]]的[[禁酒時期|禁酒令]]獲得通過。這是不穩定，不受歡迎的[[沃爾斯泰德法|強制實施]]，普遍大眾公然違抗法律，而與非法酒精交易關聯的暴力犯罪活動引起對修正案的不滿日益增加，最終令修正案於十三年後被廢除。

== 集數 ==
# '''酒鬼的國度'''（'''"A Nation of Drunkards"'''）（1小時34分）講述移民入境、酗酒、[[女性參政權|女性參政權]]和[[禁酒運動|禁酒運動]]如何導致第十八修正案的禁酒令獲得通過。
# '''蔑視法律的國度'''（'''"A Nation of Scofflaws"'''）（1小時50分）針對禁酒令如何強制實施而變得反常，以及引起意想不到的後果，包括造成大部份居民的犯罪活動。
# '''偽君子的國度'''（'''"A Nation of Hypocrites"'''）（1小時45分）當[[經濟大蕭條|經濟大蕭條]]被受矚目時，緊接也逐漸傾向禁酒令的廢除。<ref>{{Cite web |url = http://www.pbs.org/kenburns/prohibition/about/episode-guide/ |title = ''Prohibition'': Episode Guide |publisher = PBS |language = en}}</ref>
