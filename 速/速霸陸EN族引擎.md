'''速霸陸EN族引擎'''為[[日本|日本]][[富士重工業|富士重工業]]在1989年至2012年之間開發製造、搭載於[[輕型車|輕型車]]的水冷式[[直列四缸|直列四缸]][[汽油|汽油]][[往復式發動機|往復式發動機]]，除了D、X二型為[[顶置凸轮轴#双顶置凸轮轴（DOHC）|DOHC]]設計外，其餘皆為[[顶置凸轮轴#单顶置凸轮轴（SOHC）|SOHC]]。在2012年停產時，此族引擎為所有輕型車車種唯一、亦是日本全車系唯一搭載[[機械增壓器|機械增壓器]]的引擎。

== EN05A型 ==
[[排氣量|排氣量]]547c.c.、缸徑56.0mm、衝程55.6mm、壓縮比10.0：1的'''EN05A型'''引擎為[[自然进气发动机|自然進氣式]]，安裝了可變喉管式化油器，屬於大缸徑短衝程型引擎（oversquare engine）。最大馬力38ps / 7,500rpm，扭力峰值4.5kg·m / 4,500rpm。車型：
# 1989年-1992年：[[速霸陸Rex#第三代（1986年-1992年）|第三代Rex / Rex Combi]]

== EN05Z型 ==
規格和EN05A型一樣的'''EN05Z型'''引擎裝置了[[機械增壓器|機械增壓器]]，最大馬力達61ps / 6,400rpm、扭力峰值7.6kg·m / 4,400rpm。車型：
# 1989年-1992年：[[速霸陸Rex#第三代（1986年-1992年）|第三代Rex]]

== EN07A型（自然進氣、化油器） ==
'''EN07A型'''之[[排氣量|排氣量]]為658c.c.，缸徑56.0mm，衝程66.8mm，不過壓縮比比EN07C型高，達到10.0：1。最大馬力42ps / 7,000rpm，最大扭力5.3kg·m / 4,500rpm。車型：
# 1990年-1992年：[[速霸陸Rex#第三代（1986年-1992年）|第三代Rex]]
# 1992年-1998年：[[速霸陸Vivio|Vivio]]

== EN07C型（自然進氣、化油器） ==
'''EN07C型'''之[[排氣量|排氣量]]為658c.c.，缸徑56.0mm，衝程66.8mm，壓縮比9.8：1，SOHC、8V。最大[[馬力|馬力]]40ps / 6,500rpm，扭力峰值5.5kg·m / 3,500rpm。車型：
# 1990年-1995年：[[速霸陸Sambar#第五代（1990年-1999年）|第五代Sambar]]

== EN07L型（LPG、化油器） ==
承襲EN07C型的'''EN07L型'''引擎，[[燃料|燃料]]改成LPG[[液化石油氣|液化石油氣]]，其餘規格同前者。車型：
# 1990年-1999年：[[速霸陸Sambar#第五代（1990年-1999年）|第五代Sambar]]

== EN07E型（自然進氣、EMPi） ==
此具658c.c.的'''EN07E型'''引擎乃是將[[化油器|化油器]]型式的EN07A型改成EMPi電子多點噴射系統（Electric Multi-point Injection之縮寫），缸徑、衝程未變，壓縮比達到10.0：1。最大馬力是52ps / 7,200rpm，最大扭力是5.5kg・m / 5,600rpm。車型：
# 1990年-1992年：[[速霸陸Rex#第三代（1986年-1992年）|第三代Rex]]
# 1992年-1998年：[[速霸陸Vivio|Vivio]]
# 1998年-2009年：[[速霸陸Pleo#第一代（1998年-2009年）|第一代Pleo]]

== EN07S型（自然進氣、SPI） ==
此具658c.c.的'''EN07S型'''引擎的供油方式改成單點噴射系統（single point injection，縮寫成SPI），缸徑、衝程未變，壓縮比達到10.0：1。最大馬力是52ps / 7,200rpm，最大扭力是5.5kg・m / 5,600rpm；此外搖臂改成附有滾針軸承的滾輪搖臂。車型：
# 1998年-2009年：[[速霸陸Pleo#第一代（1998年-2009年）|第一代Pleo]]

== EN07F型（自然進氣、EMPi） ==
此具658c.c.的'''EN07F型'''引擎乃是將[[化油器|化油器]]型式的EN07C型改成EMPi電子多點噴射系統（Electric Multi-point Injection之縮寫），缸徑、衝程未變，壓縮比達到10.2：1。最大馬力是48ps / 6,400rpm，最大扭力是5.9kg·m / 3,200rpm。車型：
# 1995年-1999年：[[速霸陸Sambar#第五代（1990年-1999年）|第五代Sambar]]
# 1999年-2012年：[[速霸陸Sambar#第六代（1999年-2012年）|第六代Sambar]]

== EN07V型（自然進氣、SPI） ==
此具658c.c.的'''EN07V型'''乃是將EMPi型式的EN07F型改為單點噴射系統（single point injection，縮寫成SPI），壓縮比為10.1：1。最大馬力46ps / 6,400rpm，最大扭力5.9kg・m / 4,000rpm。車型：
# 1999年-2012年：[[速霸陸Sambar#第六代（1999年-2012年）|第六代Sambar]]
# 2003年-2010年：[[速霸陸R2|R2]]
# 2005年-2010年：[[速霸陸R1|R1]]

== EN07Y型（機械增壓、EMPi） ==
此具658c.c.的'''EN07Y型'''引擎配合[[機械增壓器|機械增壓器]]，卻沒有中冷器之設計，缸徑56.0mm、衝程66.8mm；最大[[馬力|馬力]]55ps / 6,200rpm，峰值扭力7.1kg・m / 3,800rpm。初期的引擎點火系統為[[分電盤|分電盤]]，直到1996年以後採改為DLI（distributorless ignition之縮寫）無分電盤點火系統。跟EN07F型一樣，製造給「{{link-ja|全國紅帽汽車運輸協會|全国赤帽軽自動車運送協同組合連合会}}（{{ja}}全国赤帽軽自動車運送協同組合連合会）」的車輛引擎也有紅色頭蓋，並加入許多專用零件以延長使用壽命。此外2001年8月以降，搖臂改成附有滾針軸承的滾輪搖臂，故[[活塞|活塞]]頂部的氣門凹槽不再使用。後來最大馬力變成58ps / 6,000rpm，最大扭力7.5kg·m / 4,400rpm。車型：
# 1990年-1999年：[[速霸陸Sambar#第五代（1990年-1999年）|第五代Sambar]]
# 1999年-2012年：[[速霸陸Sambar#第六代（1999年-2012年）|第六代Sambar]]

== EN07W型（mild charge機械增壓、SPI） ==
排氣量628c.c.、缸徑56.0mm、衝程66.8mm、壓縮比8.9：1的'''EN07W型'''引擎搭配CVT[[無段自動變速器|無段自動變速器]]，配置於[[速霸陸Pleo|Pleo]]上。為了更佳的油耗表現，此具引擎具有單點噴射系統（single point injection，縮寫成SPI）和低壓式機械增壓器（原廠稱之為「mild charge」），卻沒有中冷器。最大馬力58ps / 6,400rpm，最大扭力7.3kg·m / 4,000rpm。車型：
# 1998年-2009年：[[速霸陸Pleo#第一代（1998年-2009年）|第一代Pleo]]

== EN07U型（mild charge機械增壓、EMPi） ==
承襲EN07W型而來的'''EN07U型'''，供油方式改成EMPi電子多點噴射系統（Electric Multi-point Injection之縮寫），且加裝一個小型中冷器。最大馬力60ps / 6,400rpm，扭力峰值7.6kg·m / 4,000rpm。車型：
# 1998年-2009年：[[速霸陸Pleo#第一代（1998年-2009年）|第一代Pleo / Pleo NICOT]]

== EN07Z型（機械增壓附中冷器、EMPi） ==
這具658c.c.的'''EN07Z型'''引擎為EN族中最具代表性，搖臂改成附有滾針軸承的滾輪搖臂，且EMPi電子多點噴射系統也修改成低轉速時能輸出動力。最大馬力64ps / 6,400rpm或64ps / 6,000rpm，最大扭力8.6kg·m / 4,400rpm或9.1kg·m / 3,600rpm。車型：
# 1990年-1992年：[[速霸陸Rex#第三代（1986年-1992年）|第三代Rex]]
# 1992年-1998年：[[速霸陸Vivio|Vivio]]
# 1998年-2009年：[[速霸陸Pleo#第一代（1998年-2009年）|第一代Pleo]]

== EN07D型（DOHC、AVCS） ==
排氣量658c.c.、缸徑56.0mm、衝程66.8mm的'''EN07D型'''引擎具備了[[顶置凸轮轴#双顶置凸轮轴（DOHC）|DOHC]]和[[AVCS主動閥門控制系統|AVCS主動閥門控制系統]]，且點火為直接點火系統。最大馬力為54ps / 6,400rpm，最大扭力6.4kg·m / 4,400rpm。跟後述的EN07X型相異的是，正時皮帶用的[[滑輪|滑輪]]和一般DOHC引擎相同，進氣側和排氣側的凸輪各自獨立，並採用附有滾針軸承的滾輪搖臂。車型：
# 2003年-2010年：[[速霸陸R2|R2]]
# 2005年-2010年：[[速霸陸R1|R1]]
# 2006年-2011年：[[速霸陸Stella#第一代（2006年-2011年）|第一代Stella]]

== EN07X型（DOHC、機械增壓附中冷器） ==
排氣量658c.c.、缸徑56.0mm、衝程66.8mm的'''EN07X型'''引擎具有附中冷器的[[機械增壓器|機械增壓器]]，閥門從SOHC的搖臂驅動方式改成直推式，排氣側的凸輪軸以正時皮帶驅動，進氣側的凸輪軸則以毛邊防止齒輪帶動。由於658c.c.直列四缸的缸徑太小，為了使DOHC的閥門夾角變大，縮小凸輪軸的間隙使燃燒室變得緊湊。為了使[[每分鐘轉速|轉速]]達到9千轉，採用兩個點火線圈同時點火之方式，汽門舉桿則為直動式。

至於燃料方面，上市初期加入普通無鉛[[汽油|汽油]]即可，[[速霸陸Vivio|Vivio RX-R]]車型曾改成高[[辛烷值|辛烷值]]無鉛汽油，[[速霸陸Stella#第一代（2006年-2011年）|第一代Stella]]上市時又改回普通無鉛汽油。自EN07D型以來變更[[汽缸蓋|汽缸蓋]]的冷卻管路，以便改善散熱器距離汽缸太遠、無法有效散熱的問題。[[速霸陸R2|R2]]推出後，藉著削薄汽缸體的壁厚而輕量化，縮短活塞側緣以減少摩擦、提升燃油經濟性。車型：
* 高辛烷值汽油型：最大馬力64ps / 7,200rpm、最大扭力10.8kg·m / 3,600rpm（Vivio RX-R　E型）或最大馬力64ps / 6,000rpm、最大扭力10.5kg·m / 3,200rpm（Pleo RS、R1、R2）
# 1992年-1998年：[[速霸陸Vivio|Vivio RX-R]]
# 1998年-2009年：[[速霸陸Pleo#第一代（1998年_-_2009年）|第一代Pleo]]
# 2003年-2010年：[[速霸陸R2|R2]]
# 2005年-2010年：[[速霸陸R1|R1]]
* 普通汽油型：最大馬力64ps / 6,000rpm、最大扭力9.5kg·m / 4,000rpm（R1、R2、Stella）或最大馬力64ps / 7,200rpm、最大扭力9.0kg·m / 4,000rpm（Vivio RX-R　A-D型）
# 1992年-1998年：[[速霸陸Vivio|Vivio RX-R]]
# 1998年-2009年：[[速霸陸Pleo#第一代（1998年_-_2009年）|第一代Pleo]]
# 2003年-2010年：[[速霸陸R2|R2]]
# 2005年-2010年：[[速霸陸R1|R1]]
# 2006年-2011年：[[速霸陸Stella#第一代（2006年-2011年）|第一代Stella]]

== EN08型 ==
排氣量758c.c.、缸徑56.0mm、衝程77.0mm、壓縮比9.5：1、採SOHC設計的'''EN08型'''引擎專門搭載於外銷版本的Rex。原先Rex外銷版本裝置了544c.c.的[[速霸陸EK族引擎#EK23型|EK23型]]和665c.c.的[[速霸陸EK族引擎#EK42型|EK42型]]，日本市場推出EN族引擎後，外銷版就改以此型引擎替代。因此，外銷車名依序是：速霸陸M60（EK23型）、速霸陸M70（EK42型）和速霸陸M80（EN08型）。此具引擎的最大馬力42ps / 6,000rpm，最大扭力6kg·m / 3,600rpm。車型：
# 1990年-1992年：[[速霸陸Rex#第三代（1986年-1992年）|速霸陸Fiori]]（[[澳洲|澳洲]]）
# 1990年-1992年：[[速霸陸Rex#第三代（1986年-1992年）|速霸陸Mini Jumbo]]（[[歐洲|歐洲]]）
# 1990年-1992年：[[速霸陸Rex#第三代（1986年-1992年）|速霸陸M80]]（其他[[國家|國家]]）
