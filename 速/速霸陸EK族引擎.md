'''速霸陸EK族引擎'''為[[日本|日本]][[富士重工業|富士重工業]]早期（1960年代至1980年代）開發製造、搭載於[[輕型車|輕型車]]的直列二缸[[汽油|汽油]][[往復式發動機|往復式發動機]]。自1958年該公司以[[速霸陸360|速霸陸360]]切入[[汽車|汽車]]市場起，該族引擎曾為旗下輕型車的主要動力來源，並發展出[[氣冷式引擎|氣冷式]]、水冷式、[[两冲程循环|二衝程]]、[[四冲程循环|四衝程]]等型式；甚至到1980年代後期配合[[渦輪增壓器|渦輪增壓器]]及[[機械增壓器|機械增壓器]]以榨出更大的動力，對接手的[[速霸陸EN族引擎|速霸陸EN族引擎]]產生重大影響。

== EK31型 ==
[[File:Gasoline_Engine_Type_EK31_2C_Subaru.jpg|thumb]]]]
此具'''EK31型'''引擎乃[[富士重工業|富士重工業]]最初發展的[[汽車|汽車]]引擎，[[排氣量|排氣量]]356c.c.，缸徑61.5mm、衝程60.0mm，壓縮比6.5：1，採用強制氣冷式[[两冲程循环|二衝程]]直列二缸設計。最大[[馬力|馬力]]方面，初期（1958年5月-1960年2月）為16ps / 4,500rpm，中期（1960年2月-1964年7月）為18ps / 4,700rpm，末期（1964年7月-1968年8月）則為20ps / 5,000rpm。車型：
# 1958年-1968年：[[速霸陸360|360]]
# 1961年-1966年：[[速霸陸Sambar#第一代（1961年-1966年）|第一代Sambar]]

1968年8月起將燃燒室形狀自偏心渦流式改成半圓形，馬力提升至25ps / 5,500rpm<ref>參考{{ja}}[http://www.nipponstyle.jp/exhibition/spot8/kata_052g.html スバル360 52型]。</ref>。車型：
# 1968年-1970年：360 Young S

== EK51型 ==
排氣量423c.c.的'''EK51型'''引擎乃自EK31型擴缸而來，缸徑67.0mm、衝程60.0mm、壓縮比6.5：1、最大馬力為23ps / 5,000rpm，其餘設定同EK31型。這具引擎專門為了放大版的[[速霸陸450|450]]和[[北美洲|北美]]、[[澳洲|澳洲]]外銷版的[[速霸陸450|Maia]]而開發的，也曾搭載於1961年第8屆[[東京車展|全日本汽車展]]公開的[[概念車|概念車]]「速霸陸Sports（{{ja}}スバル・スポーツ）」。可惜始終難以解決引擎過熱的問題，曾令澳洲進口商法蘭克·奧布萊恩（Frank O'Brien）損失一筆[[金錢|金錢]]<ref>參看{{en}}[http://www.mysubaru360.com/Subaru_360s_in_Australia.html The Story of the Subaru 360 in Australia]，第四段。</ref>。車型：
# 1960年-1966年：[[速霸陸450|450 / Maia]]
# 1961年：[[概念車|概念車]]速霸陸Sports

== EK32型 ==
同為排氣量356c.c.的'''EK32型'''引擎，壓縮比為7.5：1，其他設定同EK31型。不過因為搭配[[法國|法國]][[:en:Solex|Solex]]製雙[[化油器|化油器]]，最大馬力達25ps / 5,500rpm或36ps / 7,000rpm，扭力峰值3.8kg·m / 6,400rpm。車型：
# 1968年-1970年：360 Young SS

== EK33型 ==
排氣量356c.c.、壓縮比6.5：1的'''EK33型'''為EK32型的改良版本，改採[[鋁合金|鋁合金]]製的[[汽缸本體|汽缸本體]]和舌簧閥。為了兼顧引擎的性能與耐用度，潤滑方式採速霸陸獨家開發的「完全分離潤滑方式（{{ja}}スバルマチック）」<ref>參看{{ja}}[http://members.subaru.jp/know/museum/subarur2/01.php スバル博物館：スバルR2]，「エンジン・システム　強制空冷2サイクル2気筒」之段落。</ref>。性能表現：
* 最大[[馬力|馬力]]30ps / 6,500rpm、峰值扭力3.7kg·m / 5,500rpm：搭載於[[速霸陸R-2|R-2]]、[[速霸陸Sambar#第二代（1966年-1973年）|第二代Sambar]]。
* 最大[[馬力|馬力]]36ps / 7,000rpm、峰值扭力3.8kg·m / 6,400rpm：搭載於R-2 Young SS，加裝[[東京都|東京都]]三國工業（{{ja}}株式会社[[:ja:ミクニ|ミクニ]]）製造的36PHH型雙阻風門式[[化油器|化油器]]（twin choke carburetor），並具備專用的排氣管與膨脹箱（expansion chamber）。
* 最大[[馬力|馬力]]32ps / 6,500rpm、峰值扭力3.8kg·m / 5,500rpm：搭載於R-2 Sporty Delux，加裝動力噴嘴的化油器。

車型：
# 1969年-1972年：[[速霸陸R-2|R-2]] / R-2 Young SS / R-2 Sporty Delux
# 1970年-1973年：[[速霸陸Sambar#第二代（1966年-1973年）|第二代Sambar]]

== EK34型 ==
[[File:Gasoline_Engine_Type_EK34_Subaru_with_3rd_Sambar.jpg|thumb]]
同為排氣量356c.c.、缸徑61.5mm、衝程60.0mm、壓縮比6.5：1的'''EK34型'''乃延續EK33型而來，不過[[汽缸本體|汽缸本體]]改成水冷式。此具引擎在[[怠速|怠速]]時會發出啪啦啪啦的[[噪音|噪音]]，原廠額外加裝「速霸陸ISV」（{{ja}}アイドリング・サイレンス・バルブ）裝置以解決此問題<ref>在引擎內加裝防鏽閥門以抑制噪音，詳情見{{ja}}[http://www.geocities.jp/misamima90/t01.htm スバルＩＳＶ]內文。</ref>。隨著不同的調校方式而出現不同的性能表現：
* 最大馬力28ps / 5,500rpm：搭載於Sambar K71/K72/K81等車型。
* 最大馬力32ps / 6,000rpm：搭載於R-2、[[速霸陸Rex|Rex]]。
* 最大馬力35ps / 6,500rpm：搭載於Rex TS。
* 最大馬力36ps / 7,000rpm：搭載於R-2 GSS。
* 最大馬力37ps / 6,500rpm：搭載於Rex GSR。

車型：
# 1971年-1972年：R-2
# 1972年-1973年：[[速霸陸Rex#第一代（1972年-1981年）|第一代Rex]]
# 1973年-1976年：[[速霸陸Sambar#第三代（1973年-1982年）|第三代Sambar]]

== EK21型 ==
排氣量358c.c.、缸徑66.0mm、衝程52.4mm、壓縮比9.5：1的'''EK21型'''，採用水冷式四衝程[[顶置凸轮轴#单顶置凸轮轴（SOHC）|SOHC]]之設計，乃是針對1976年頒布的日本汽車廢氣排放標準新制而生。1975年12月以降，為了因應[[昭和|昭和]]51年度汽車廢氣排放標準（{{ja}}昭和51年度排出ガス規制），加裝「SEEC-T（Subaru Exhaust Emission Control-Thermal & Thermodynamic System之縮寫）」<ref>詳請可參考{{ja}}[http://www.geocities.jp/misamima90/t03.htm ＳＥＥＣ－Ｔ]。</ref>的二次空氣導入裝置。車型：
# 1973年-1976年：[[速霸陸Rex#第一代（1972年-1981年）|第一代Rex]]
# 1976年2月-5月：[[速霸陸Sambar#第三代（1973年-1982年）|第三代Sambar]]

== EK22型 ==
此具'''EK22型'''自EK21型擴缸而來，排氣量490c.c.、缸徑74.0mm、衝程57.0mm、壓縮比9.0：1，亦附有SEEC-T二次空氣導入系統。不過此型引擎的產品生命周期並不長久，隨即被EK23型取代。車型：
# 1976年-1977年：[[速霸陸Rex#第一代（1972年-1981年）|第一代Rex / Rex 5]]
# 1976年-1977年：[[速霸陸Sambar#第三代（1973年-1982年）|Sambar 5]]

== EK23型 ==
'''EK23型'''是EK族最後一型的引擎，因應當時的日本輕型車規格已擴缸至544c.c.；另為了符合昭和53年度汽車廢氣排放標準，加裝了SEEC-T二次空氣導入系統和EGR[[排氣再循環|排氣再循環]]系統。為了減少振動，水泵和油泵以平衡軸連動，除了定期保養外，也要定期更換內部零件。第一代Rex為[[後置後驅|後置後驅]]的佈局，因引擎安置於底盤後半，進氣岐管的位置也跟[[前置前驅|前置前驅]]車不同，平衡軸僅在吸氣側安裝一支而已。第二代Rex改成前置前驅後，平衡軸改成吸、排氣兩側各一支以抑制振動。到了第三代Rex除修改汽缸蓋外，也變成當時流行的3閥門設計。後來甚至加裝[[機械增壓器|機械增壓器]]及[[渦輪增壓器|渦輪增壓器]]，特別是機械增壓的EK23-Z型附中冷器和EMPi電子多點噴射裝置（Electric Multi-point Injection之縮寫），在當時車壇算是相當少見的豪華配備。

富士重工業也曾為了1975年設立、1978年全國組織化的「全國紅帽汽車運輸協會（{{ja}}[[:ja:全国赤帽軽自動車運送協同組合連合会|全国赤帽軽自動車運送協同組合連合会]]）」生產專用的EK23型引擎，這類引擎的特徵是紅色頭蓋，並加入許多專用零件以延長使用壽命。

=== EK23型（自然進氣、二閥門） ===
排氣量544c.c.、缸徑76.0mm、衝程60.0mm的'''EK23型'''引擎，其進氣、排氣閥門各一，依不同調校則性能表現各異：
* 最大馬力31ps / 6,200rpm、最大扭力4.2kg·m/3,500rpm：第一代Rex轎車型
* 最大馬力28ps / 6,200rpm、最大扭力4.2kg·m/3,500rpm：第一代Rex廂型車型
* 最大馬力31ps / 6,000rpm、最大扭力4.4kg·m/3,500rpm：第二代Rex、Rex Combi
* 最大馬力30ps / 6,000rpm、最大扭力4.2kg·m/3,500rpm：第三代Rex
車型：
# 1977年-1989年：第一、二、三代Rex
# 1977年-1989年：第三、四代
# 1983年-1987年：速霸陸600 / M60 / Mini Jumbo / Sherpa（外銷版）

=== EK23-T型（渦輪增壓、二閥門） ===
這具'''EK23-T型'''加裝了[[日立|日立]]供應的36mm渦輪，使得最大[[馬力|馬力]]達41ps / 6,000rpm、峰值扭力5.9kg·m / 3,500rpm。為了對應馬力提升，原廠亦將前[[煞車|煞車]]系統升級成通風式碟煞，此為日本輕型車之首創。車型：
# 1983年-1986年：[[速霸陸Rex#第二代（1981年-1986年）|第二代Rex Combi]]

=== EK23型（自然進氣、三閥門） ===
[[File:Gasoline_Engine_Type_EK23_Subaru_with_4th_Sambar.jpg|thumb]]
三閥門版本的'''EK23型'''採進氣側2閥門、排氣側1閥門的設計，馬力更為提升，達到36ps / 7,000rpm、最大扭力4.4kg·m / 4,500rpm。車型：
# 1986年-1989年：[[速霸陸Rex#第三代（1986年-1992年）|第三代Rex]]
# 1987年-1990年：[[速霸陸Sambar#第四代（1982年-1990年）|第四代Sambar]]

=== EK23-Z型（機械增壓、三閥門） ===
'''EK23-Z型'''乃EK23型的最後一個版本，仍採進氣側2閥門、排氣側1閥門的設計外，燃油供給方式改成EMPi電子多點噴射裝置。另外，富士重工業也是1987年以來繼[[鈴木_(公司)|鈴木汽車]]、[[大發工業株式會社|大發工業]]和[[三菱汽車|三菱汽車]]後，第四家將輕型車引擎加上[[機械增壓器|機械增壓裝置]]的日系車廠，使得該具引擎55ps / 6,400rpm的馬力、7.4kg·m / 4,400rpm的扭力創下輕型車直列二缸引擎史上的記錄。車型：
# 1988年-1990年：[[速霸陸Rex#第三代（1986年-1992年）|第三代Rex]]

== EK42型 ==
[[排氣量|排氣量]]665c.c.、缸徑78.0mm、衝程69.6mm、壓縮比9.5：1的'''EK42型'''是此族引擎容積最大的，專門搭載於外銷版本的車輛，最大馬力27ps / 6,000rpm、峰值扭力54N·m / 3,000rpm。車型：
# 1982年-1989年：第二、三代速霸陸700 / M70 / Mini Jumbo / Sherpa
# 1982年-1989年：速霸陸Small Van（即第四代Sambar之外銷版本）
