'''速霸陸EZ族引擎'''為[[日本|日本]][[富士重工業|富士重工業]]開發製造的一系列[[水平對臥六缸引擎|水平對臥六缸]][[汽油|汽油]][[往復式發動機|往復式發動機]]，其中EZ36型目前為該公司[[排氣量|排氣量]]最大的[[水平對臥引擎|水平對臥引擎]]。

== 概要 ==
1996年隨著[[速霸陸Alcyone_SVX|速霸陸Alcyone SVX]]停產後，該公司未再推出搭載[[水平對臥六缸引擎|水平對臥六缸引擎]]的車種。2000年原廠追加[[速霸陸Legacy#第三代BE/BH系（1998年-2003年）|第三代Legacy]]新車型「Lancaster 6」，搭載最新研發的EZ30D型引擎。後者乃基於[[速霸陸EJ族引擎|EJ型引擎]]重新開發，包括變更缸徑和衝程、縮短缸心距離、採用正時鏈條、旋繞式附件驅動等，跟同屬水平對臥六缸構造的[[速霸陸EJ族引擎#EG33型|EG33型]]是截然不同的設計<ref>請看{{ja}}[http://www.fhi.co.jp/news/00_4_6/5_24_3.htm 富士重工業コンパクトな水平対向6気筒エンジンを新開発スバル レガシィ「ランカスター６」に搭載]。</ref>。

== EZ30D型 ==
[[File:Subaru_EZ30_engine_001.JPG|thumb]]
'''EZ30D型'''的[[排氣量|排氣量]]為2,999c.c.、缸徑89.2mm、衝程80.0mm、壓縮比10.7：1。最大馬力220ps / 6,000rpm，
最大扭力29.5kg·m / 4,400rpm。車型：
# 2000年-2003年：[[速霸陸Legacy#第三代BE/BH系（1998年-2003年）|第三代速霸陸Legacy]]
# 2000年-2003年：第二代[[速霸陸Outback|速霸陸Outback]]

== EZ30-R型 ==
[[排氣量|排氣量]]、缸徑、衝程、壓縮比皆和EZ30D型相同的'''EZ30-R型'''，在汽缸頭進氣側加入[[AVCS主動閥門控制系統|AVCS主動閥門控制系統]]的可變氣門正時，並搭配VVL氣門揚程控制（Variable Valve Lift之縮寫），三種進氣揚程分別為2.8mm、6.0mm和9.55mm，以對應低、中、高三種轉速域。此外，曲軸支撐點嵌入鋁合金鑄鋼片，以保持正確間隙來抑制熱膨脹，進一步降低震動和運轉噪音；還有電子節氣門、樹脂進氣歧管、每邊三合一型式芭蕉段等元件。最大馬力為245hp / 6,600rpm、最大扭力30.3kg·m / 4,200rpm。車型：
# 2003年-2009年：[[速霸陸Legacy#第四代BL/BP系（2003年-2009年）|第四代速霸陸Legacy]]
# 2003年-2009年：第三代速霸陸Outback
# 2005年-2008年：[[速霸陸Tribeca|速霸陸Tribeca]]

== EZ36型 ==
[[File:SUBARU_EZ36_001.jpg|thumb]]
'''EZ36型'''的排氣量為3,629c.c.，缸徑92.0mm、衝程91.0mm、壓縮比10.5：1，採DOHC、自然進氣的配置。性能表現依各地市場不同的特性做調校，最大馬力分別為256hp（[[北美洲|北美洲]]）、258ps（[[歐洲|歐洲]]）、260ps（日本）/ 6,000rpm，扭力峰值則為247lb·ft（北美洲）、35.7kg·m（歐洲）、34.2kg·m（日本）/ 4,400rpm。奇特的是，這具引擎使用的[[活塞|活塞]]連桿是非對稱式，但活塞運動方向依舊是水平的<ref>參考{{en}}[http://wardsauto.com/ar/subaru_develops_boxer WardsAuto: Subaru Develops More Powerful, Efficient Boxer Engine] {{Wayback|url=http://wardsauto.com/ar/subaru_develops_boxer |date=20131002112042 }}。</ref>。車型：
# 2008年迄今：[[速霸陸Tribeca|速霸陸Tribeca]]
# 2010年迄今：第五、六代速霸陸Legacy
# 2010年迄今：第四代速霸陸Outback
