'''速霸陸Impreza'''（{{lang-ja|スバル・インプレッサ}}）為[[日本|日本]][[富士重工業|富士重工業]]於1992年起製造、販售的[[緊湊型轎車|緊湊型轎車]]，[[中國|中國]]的官方翻譯名稱為'''翼豹'''，[[臺灣|臺灣]]的車迷則習稱'''硬皮鯊'''。自1992年推出以來即為[[速霸陸|速霸陸]]汽車的主力車種之一，搭載了該公司向來標榜的[[水平對臥引擎|水平對臥引擎]]及全時[[四輪驅動|四輪驅動]]系統，並衍生出[[速霸陸Forester|速霸陸Forester]]、[[速霸陸XV|速霸陸XV]]、[[速霸陸WRX|速霸陸WRX]]等車款。車名「Impreza」乃為[[英語|英語]]「impresa（箴言、徽章）」的造詞。

== 歷史與概要 ==
自1992年問世以來，原廠替該車款推出許多衍生車型，本條目一併記敘下列：
# 四門轎車型：Sedan WRX、Anesis、B4。
# 五門旅行車型：Sports Wagon、Sports Wagon WRX。
# 五門掀背車型：Gravel EX、Sports、XV（另請參照[[速霸陸XV|速霸陸XV]]）。
# 雙門轎跑車型。

身為[[緊湊型轎車|緊湊型轎車]]級的「世界戰略車」而推出，目標特別鎖定[[歐洲|歐洲]]市場。不過真正打響Impreza名號的卻是其性能表現，以及自1995年至1997年連續三年在[[世界拉力錦標賽|世界拉力錦標賽]]（World Rally Championship，以下簡稱WRC）賽場上奪得車隊冠軍。尤其這部車和對手[[三菱Lancer_Evolution|三菱Lancer Evolution]]不論在賽場上的過招，抑或是現實生活中銷售的競爭，一直都是車迷們津津樂道的話題。「Impreza WRX STI」乃由[[速霸陸技術國際|速霸陸技術國際]]以Impreza WRX車型為藍本，加以改裝調校出的高性能四門跑車。速霸陸車隊使用此車的賽車版來參加[[世界拉力錦標賽|世界拉力錦標賽]]，並且在賽車手[[科林·麥克雷|科林·麥克雷]]、[[理查德·伯恩斯|理查德·伯恩斯]]、{{link-en|皮特·索伯格|Petter Solberg}}等人的努力下連續3屆蟬聯車隊冠軍。

1992年第一輛速霸陸Impreza WRX STI問世後，原本只限日本國內市場；因同為右駕，另有代理商意美汽車以及平行進口車行導入[[香港|香港]]當地；1999年起開始出口至[[澳大利亞|澳大利亞]]，2001年銷入[[歐洲|歐洲]]，直到2003年夏季才正式進駐[[北美洲|北美洲]]市場。加上富士重工業的產能有限，每年的生產數量極為有限，其中某些限量版本更為罕見，例如SPEC-C型的限量版本每年僅生產800輛，而S204型限量版總共只有600輛<ref>{{cite web|author=saltdesign.co.nz |url=http://www.subaru.co.nz/About_Subaru/News/index.php?article=887 |title=Subaru New Zealand - News |publisher=Subaru.co.nz |date=2006-04-26 |accessdate=2010-04-03}}</ref>。

2000年代由於[[美國|美國]][[通用汽車|通用汽車]]分別握有富士重工業和[[瑞典|瑞典]][[紳寶汽車|紳寶汽車]]的[[股票|股份]]，在通用汽車的主導下，紳寶汽車以第一代Impreza為基礎，修改車身外觀、[[懸吊系統|懸吊系統]]和內裝後推出[[紳寶9-2X|紳寶9-2X]]（由富士重工業的[[太田市|太田市]]工廠代工製造），成為相當罕見的兄弟車款。後來通用汽車將持股出售給[[豐田汽車|豐田汽車]]，才結束這段合作關係。

2010年代後富士重工業調整行銷策略，高性能版本獨立成[[速霸陸WRX|WRX]]；而Impreza車系則轉型為家庭用車之定位。

1992年 - 11月2日正式發售四門轎車、五門旅行車與WRX等三種車型，其中旅行車（station wagon）刻意命名為「sports wagon」，乃為了強調其運動化性能。而「WRX」之名來自先前運動化設定的[[速霸陸Leone|Leone RX]]，故WRX車型也搭載[[馬力|馬力]]達240ps的2.0L水平對臥四缸DOHC [[速霸陸EJ族引擎#EJ20G型|EJ20G型]][[渦輪增壓器|渦輪增壓]]引擎。[[底盤|底盤]]以[[速霸陸Legacy#第一代BC/BF系（1989年-1993年）|第一代Legacy]]的底盤作基礎，全長200mm、[[軸距|軸距]]縮短60mm、高度增加10mm，故車身[[重量|重量]]比第一代Legacy減輕80[[公斤|公斤]]。

關於動力來源，此代車款全部採用[[水平對臥四缸引擎|水平對臥四缸]]構造的|[[速霸陸EJ族引擎|EJ族引擎]]，日本市場計有1.5L、1.6L、1.8L、2.0L等四種[[排氣量|排氣量]]選擇；其他[[國家|國家]]則隨著市場特性與道路環境的不同，另有2.2L和2.5L可供選擇；共同搭配的[[变速器|變速箱]]則有四速自動排檔、五速手動排檔等二種。前輪的懸吊為L型下臂式[[麥弗遜懸吊|麥花臣懸吊]]，後輪則為六連桿式麥花臣懸吊。此外WRX車型的手排版為了輕量化，前輪懸吊的下臂改用鋁合金鍛造材質。

1993年 - 9月改良某些缺失，並追加WRX車型的五門旅行車。本來WRX車型僅有五速手排之設定，現追加四速自排。

[[File:1995_WRX_STI_RA_GC8.jpg|thumb]]
1994年 - 1月推出原廠代號「GC8A/B」的「WRX STi」性能車，所搭載的[[速霸陸EJ族引擎#EJ20G型|EJ20G型]]單渦輪增壓引擎可輸出最大馬力250ps / 6,500rpm、最大扭力31.5kg·m / 3,500rpm。屬於第一版的該款車車重為1,230[[公斤|公斤]]，先由太田工廠生產出WRX車型後再讓[[速霸陸技術國際|STI]]升級換裝，不過自第二版起改由STI之獨立[[大量生產|生產線]]負責製造。10月「WRX RA STi」上市，最大馬力達270ps / 6,500rpm，並具備DCCD[[駕駛者可控式中央差速器|駕駛者可控式中央差速器]]。該款接單生產的限量車大量使用賽車部件，且不裝備空調、音響等，主要針對各拉力賽車隊發售，而非個人[[消費者|消費者]]。

[[File:1st-Subaru-Outback-Sport.jpg|thumb]]
原廠另針對[[北美洲|北美洲]]市場推出名為「Impreza Outback Sport」的衍生車款，以五門旅行車為基礎，搭載2.2L水平對臥四缸SOHC [[速霸陸EJ族引擎#EJ221型及EJ222型|EJ221/EJ222型]]引擎，車身塗裝採上下雙色之方式，並配置車頂行李架等具有休旅風味的部件。

1995年 - 1月原本只有外銷的雙門[[轎跑車|轎跑車]]車型以「Impreza Retna」（{{ja}}インプレッサリトナ）之名回鍋日本市場<ref>{{ja}}[http://www.goo-net.com/catalog/SUBARU/IMPREZA_RETNA/ Goonet: スバル インプレッサリトナ　1995年1月(平成7年1月) 発売モデル]。</ref>，共有1.5L[[水平對臥四缸引擎|水平對臥四缸]]SOHC [[速霸陸EJ族引擎#EJ15型|EJ15型]]、1.6L水平對臥四缸SOHC [[速霸陸EJ族引擎#EJ16型|EJ16型]]兩種引擎動力可選購。10月日本市場推出具有濃厚休旅味道的「Gravel EX」車型，配置防撞桿和車尾備胎（備胎殼罩寫有「Gravel Express」字樣）。此款車即為美規版的「Impreza Outback Sport」，但是換裝成2.5L水平對臥四缸DOHC [[速霸陸EJ族引擎#EJ25D型|EJ25D型]]引擎。雖然總生產數量僅有1,313部，卻是[[速霸陸Forester|速霸陸Forester]]的濫觴。隨後原廠代號「GC8C」的「STi Version2 555」上市，車身塗裝同為馳騁WRC比賽用車的寶藍色、輪轂為金色，其中四門轎車限量555部、五門旅行車限量100部。自此版起，STI將進氣岐管漆成紅色，車迷俗稱為紅頭引擎。

[[File:SUBARU_IMPREZA_GC8V.JPG|thumb]]
1996年 - 1月為了紀念奪下1995年WRC賽車手、廠商車隊的雙料冠軍，發售限定版的「V-Limited」，限量1,000輛並以拉力賽車隊的經典藍色作為內飾的主色調。另外「STi Version2 555」同樣限定555輛，乃紀念主要贊助商[[英美烟草|555牌香菸]]。9月發表小改款（中期型），變更前防撞桿設計，尾燈之[[方向燈|方向燈]]改為清澈狀，輪轂也換上新造型。WRX車型之引擎為了追求高轉速、高出力，改為馬力高達280ps / 6,500rpm的[[速霸陸EJ族引擎#EJ20K型|EJ20K型]]；而EJ15型和EJ18型也經過重新調校。雙門轎跑車在日本市場不受歡迎，故停止發售，但外銷版仍可見到。原廠代號「GC8D」的第三版WRX STi除了EJ20K型引擎外，前輪剎車系統升級成對向四活塞卡鉗，車重比前一版再增加10公斤，達到1,250公斤。同時也增加了「WRX STI Type R」雙門車型，由於1,190公斤的車重比四門車型輕盈，被[[英國|英國]]著名賽車改裝團隊Prodrive選用參加WRC。除了DCCD[[駕駛者可控式中央差速器|駕駛者可控式中央差速器]]之外，並配備新開發的密齒比變速箱，加大的中冷器更增加水霧噴嘴，以強化其工作效率。

1997年 - 9月將駕駛座[[安全氣囊|安全氣囊]]列為標準配備，助手席氣囊則為選配；內裝大幅變更成與[[速霸陸Forester#第一代SF系（1997年-2002年）|第一代Forester]]共用。原廠代號「GC8E」的第四版WRX STi同時推出，其中「WRX RA STi」、「WRX R STi」的後輪煞車也升級成對向四活塞卡鉗。10月舉辦的第32屆[[東京車展|東京車展]]上，富士重工業展出以Impreza為基礎所打造的復古車「Casa Blanca（{{ja}}カサブランカ）」。

[[File:Subaru_Impreza_22B_STi-Version.jpg|thumb]]
1998年 - 3月原廠為了紀念成立40週年及連續蟬聯三屆WRC車隊冠軍，在日本市場推出寬車體、雙門的「22B-STi Version」<ref>參見{{ja}}[http://www.sti.jp/product/22b/index.html IMPREZA 22B STi VERSION]。</ref>，限量399部，且[[車輛識別號碼|車輛識別號碼]]含有「GC8E2SD」的字樣。此限定車款搭載[[速霸陸EJ族引擎#EJ22G型|EJ22G型]]引擎，搭配[[IHI|石川島播磨重工業]]的VF23型滾珠軸承渦輪增壓器，可分別輸出280ps / 6,000rpm、37.0kg·m / 3,200rpm的馬力和扭力。車體特別加寬至1,770mm，車重則為1,270公斤。避震系統由倍適登（Bilstein）承製，輪轂則為BBS製17吋鋁合金輪圈，[[輪胎|輪胎]]升級成[[倍耐力|倍耐力]]的235/40 ZR17 P Zeros，[[離合器|離合器]]也升級為雙盤式並配備陶瓷離合片，前後剎車卡鉗均為[[白色|白色]]，並加上[[紅色|紅色]]「Subaru」字樣。關於「22B」的命名緣由一直眾說紛紜：有人認為22代表2.2L的[[排氣量|排氣量]]，而B代表配備了Bilstein避震器；有人則認為555（速霸陸車隊的主要贊助商）的[[十六進制|十六進制]]剛好是22B；更有人認為22B代表2.2 Boxer。

同年9月進行後期型（「F」型）的小改款，全車系的[[車頭燈|車頭燈]]改為多折射式，WRX車型的前保桿造型也再度變更；另外新增搭載自然進氣[[速霸陸EJ族引擎#EJ204型|EJ204型]]引擎的「SRX」車型，附有[[AVCS主動閥門控制系統|AVCS主動閥門控制系統]]。原廠代號「GC8F」的第五版WRX STi改搭載全新研發的[[速霸陸EJ族引擎#EJ207型|EJ207型]]渦輪增壓引擎，雖然最大馬力280ps / 6,500rpm、最大扭力36.0kg·m / 4,000rpm，但其工作效率及排放標準相對提高，且改善低轉速域扭矩的輸出表現，使其爆發更加線性。此版之內裝重新設計過，包括賽車座椅、[[方向盤|方向盤]]、KYB工業製倒立式麥花臣懸吊系統等<ref>詳情見{{ja}}[http://www.fhi.co.jp/news/98_7_9/09_03.html 富士重工業：スバル インプレッサシリーズを一部改良]。</ref>。12月則正式推出復古外觀的「Casa Blanca」車型，限量5,000部。

同年年底速霸陸英國分公司進口16輛「22B-STi Version」，並委託著名賽車改裝團隊Prodrive加以改造。這批被命名為「22B Type UK」的特仕車和坊間水貨商進口的22B不同，不僅提供3年原廠保固，齒輪比也重新調整以適應英國的道路狀況，因為Prodrive將最終傳動比從3.90改成4.44。在外觀上和「22B-STi Version」並無二致，但車尾多了「Type UK」及「Prodrive」的銘牌。此款特仕車的售價為39,950[[英鎊|英鎊]]，且登記[[車輛號牌|牌照]]的年份為1999年，因為前一年的50輛進口額度已經被水貨商捷足先登。此外，這16輛車並不符合[[歐盟|歐盟]]認證的[[噪音|噪音]]及廢氣排放標準<ref>參看{{en}}[http://www.rb320oc.co.uk/specialeditionsubaru22b.asp The RB320 Owners Club: 22B Type-UK (1998)]。</ref>。

1999年 - 9月小改款成「G」型，「WRX STi」和「STi Type R」二種車型則追加前下巴；「WRX STi」五門和「WRX」車型的鋁合金鋼圈造型改成六幅式；。「WRX type RA STi」和「WRX type RA」可選購砲彈型後照鏡。原廠代號「GC8G」的第六版WRX STi在性能上沒有重大改變，不過變更了大型尾翼的斷面形狀，藉由提升下壓力使高速行駛更為穩定。此版共生產了2千輛四門轎車、5百輛五門掀背車，而「Type R」雙門轎跑車則為1千輛、「Type RA」發售2千輛。

2000年 - 8月23日日本市場率先發表大改款的第二代Impreza，區分成四門轎車（此代起統一名稱為「WRX」）和五門旅行車二種外型，前者搭載2.0L水平對臥四缸DOHC [[速霸陸EJ族引擎#EJ204型|EJ204型]]引擎，配合[[四輪傳動|四輪傳動]]；後者則搭載1.5L[[水平對臥四缸引擎|水平對臥四缸]]SOHC [[速霸陸EJ族引擎#EJ15型|EJ15型]]引擎，為[[前置前驅|前置前驅]]之方式。2000年代車壇興起一股「車體撞擊安全」的概念，此款車亦強調「新環狀力骨車體」<ref>參看{{ja}}[http://www.subaru.jp/about/technology/spirit/safety/psafety02.html 富士重工業：新環状力骨構造ボディ]。</ref>，在車體前方加設副車架以吸收迎面而來的衝擊力道。也因為此點，雖然[[底盤|底盤]]剛性比起前一代提升148%，車重卻多了近200[[公斤|公斤]]。

關於動力心臟部分，四門轎車「WRX」車型搭載2.0L[[自然进气发动机|自然進氣版]]與渦輪增壓版，其中[[速霸陸EJ族引擎#EJ205型|EJ205型]]渦輪增壓引擎導入TGV進氣控制系統（Tumble Generated Valve），提升燃燒效率、降低廢氣含碳量；且在進氣側加裝[[AVCS主動閥門控制系統|AVCS主動閥門控制系統]]，加強馬力輸出。五門旅行車（原廠稱「Sport Wagon」）除了前述二種2.0L引擎外，另有1.5L[[水平對臥四缸引擎|水平對臥四缸]]SOHC [[速霸陸EJ族引擎#EJ15型|EJ15型]]引擎，可選擇[[前置前驅|FF]]或[[四輪傳動|4WD]]。不論四門或五門，變速系統皆為四速自排、五速手排兩種設定。在外觀設計上，改為圓形的[[車頭燈|車頭燈]]和強調性能表現的WRX及STi特仕車相互組合下，顯得相當突兀。非但引起車迷[[愛好者|愛好者]]的反彈，也連帶影響到銷售成績與人氣<ref>參考[http://subaru.autonet.com.tw/cgi-bin/file_view.cgi?a907035621014 AutoNet: 哪輛車是麥可傑克森的變臉翻版？SUBARU第二代Impreza是也]{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}。</ref>。

同年10月24日推出「WRX STi」（原廠代號GD Rev. A）特仕車<ref>詳情見{{ja}}[http://www.fhi.co.jp/news/00_10_12/10_31_1.html 富士重工業：スバルインプレッサSTiシリーズを発売]。</ref>，且改用[[英文字母|英文字母]]來區分出產年份。由於這種官方命名方式不易區別版本，眾多車迷改以車頭燈樣式來命名：2000年至2003年的被稱為「蟲眼（臺灣則稱作圓燈）」、2004年至2005年的被稱為「淚眼」、2006年至2007年的則被稱為「鷹眼」。比起上一代的WRX STi，此代GD平台的[[底盤|底盤]]增加了120%的抗扭剛性，輪轂尺寸升級至17" x 7.5"，五速手排變速箱也升級為六速手排。在內裝方面，座椅及[[方向盤|方向盤]]等處加上了STi標誌，轉速表新設紅色換檔提示燈；剎車制動系統改良成Brembo製前四活塞、後對向活塞的卡鉗。除了加大引擎蓋進氣口外，DCCD[[駕駛者可控式中央差速器|駕駛者可控式中央差速器]]亦改成可以在ABS[[防鎖死煞車系統|防鎖死煞車系統]]作動時自動切換成解鎖模式。此版五門旅行車型的引擎輸出較小，峰值馬力下調為275ps。雖然2000年起WRX STi開始陸續投放至[[亞洲|亞洲]]、[[大洋洲|大洋洲]]及[[歐洲|歐洲]]市場，雖搭載2.0L [[速霸陸EJ族引擎#EJ207型|EJ207型]]渦輪增壓引擎，但動力輸出略有所調降，以符合各國的廢氣排放標準，最大馬力從261ps至280ps不等。

2001年 - 11月小改良成「B」型，包括調整各等級之配備和價格、新增車色塗裝、進氣壩造型等。同年12月6日發表「WRX STi type RA spec C」<ref>請看{{ja}}[http://www.fhi.co.jp/news/01_10_12/01_12_06_02.htm 富士重工業：スバルインプレッサＷＲＸ「ＳＴｉ ｔｙｐｅ ＲＡ ｓｐｅｃ Ｃ」を発売]。</ref>（原廠代號GD Rev. B），由名汽車設計師{{link-en|彼特·史蒂芬斯|Peter Stevens (car designer)}}（Peter Stevens）操刀重新設計進氣壩及車體，霧燈燈罩開始使用粉紅色STi標誌。避震器的硬度進一步強化，重新調校EJ257型引擎，使扭矩輸出獲得小幅度提升。此款特仕車取消了前防撞內鐵、駕駛座及助手席[[安全氣囊|安全氣囊]]、[[座位安全帶|安全帶]]緊縮器等元件，車身更為輕量化。此外延長15mm的[[軸距|軸距]]及使用不同的控制臂，使得車輪的後傾角從原先的3.5度上調至5度，以提升操控性能。同時自此版起，終止生產WRX STi五門休旅車型。

2002年 - 6月發售STI操刀改造的「S202 STi Version」<ref>參看{{ja}}[http://www.sti.jp/s202/top.html IMPREZA S202 STI version] {{webarchive|url=https://web.archive.org/web/20130727022903/http://www.sti.jp/s202/top.html |date=2013-07-27 }}。</ref>，透過重新編寫行車電腦程式、修改進氣系統、更換高轉速取向的排氣系統等，最大馬力一舉推至320ps / 6,400rpm，扭力峰值也達39.2kg·m / 4,400rpm<ref>詳情見[http://culture.modiauto.com.cn/historic/1150/21698_2.html 無敵汽車網：斯巴魯翼豹STi的GD時代史S202 STi Version]。</ref>；其他尚包含黃色桶型賽車座椅、輕量化的空調壓縮機、劃線剎車碟、Rays Volk製輪圈等配備。

同年11月1日宣布小改款，是為「C」型。最大的改變是車頭造型，將俗稱「蟲眼」的圓形車頭燈改成圓潤的倒三角形（俗成「淚眼」）<ref>參考[http://subaru.autonet.com.tw/cgi-bin/file_view.cgi?a2090201S4004013 AutoNet: 2003年式小改Impreza變臉成功] {{webarchive|url=https://web.archive.org/web/20131206102639/http://subaru.autonet.com.tw/cgi-bin/file_view.cgi?a2090201S4004013 |date=2013-12-06 }}。</ref>，進氣壩的開口也加大，不論四門轎車或五門旅行車皆換成17吋輪框。內裝部分沒有太大變動，僅更換MOMO製三幅式真皮[[方向盤|方向盤]]、內裝飾板與配色。進氣歧管改成等長化，所以引擎怠速時特有的「吱哆哆哆」聲響變得較小聲。原廠代號「GD Rev. C」的WRX STi特仕車改變幅度不大，仍沿用上一版大部分的設計。

2003年 - 7月19日臺灣速霸陸（Subaru of Taiwan，縮寫成SOT）引進Impreza WRX STi，除最大馬力265ps / 6,000rpm、最高扭力35.0kg·m / 4,000rpm的引擎外，額外加裝油壓計/油溫計/增壓計之三連表、碳纖維中控台面板、STi車室腳踏墊等配備<ref>參見[http://subaru.autonet.com.tw/cgi-bin/file_view.cgi?a3070178S4004012 AutoNet: 台灣速霸陸正式宣布引進Impreza WRX STi]{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}。</ref>。9月3日小改良成「D」型<ref>請閱覽{{ja}}[http://www.fhi.co.jp/news/03_07_09/03_09_03.htm 富士重工業：スバルインプレッサシリーズを一部改良あわせて セダン「１５ｉ」を追加設定]。</ref>，新增1.5L水平對臥四缸SOHC [[速霸陸EJ族引擎#EJ15型|EJ15型]]引擎。

2007年 - 4月4日開幕的第107屆[[紐約國際車展|紐約國際車展]]上，原廠公開全新改款的第三代Impreza<ref>參考{{ja}}[http://www.fhi.co.jp/news/07_04_06/07_04_03_01.html 富士重工業：新型スバル インプレッサを公開]。</ref>，並於[[北美洲|北美洲]]發售；6月4日在日本市場上市。此代改以五門[[掀背車|掀背車]]車型取代以往的[[旅行車|旅行車]]，並做為主力車型，而四門轎車車型則未在日本市場推出。另外取消性能取向的「WRX」等級，改以「S-GT」代替。日規版的動力方面含有1.5L水平對臥四缸DOHC [[速霸陸EL族引擎#EL15型|EL15型]]、2.0L水平對臥四缸SOHC [[速霸陸EJ族引擎#EJ20E型|EJ20E型]]及2.0L水平對臥四缸DOHC [[速霸陸EJ族引擎#EJ20X型與EJ20Y型|EJ20Y型]]渦輪增壓等三種引擎設定，搭配Sportshift E-4AT四速手自排，另外渦輪增壓版可配六速手排變速系統。1.5L為[[前置前驅|前置前驅]]，其餘排氣量之引擎則搭配全時[[四輪驅動|四輪驅動]]。美規版則以2.5L水平對臥四缸DOHC [[速霸陸EJ族引擎#EJ254型|EJ254型]]和2.5L水平對臥四缸DOHC [[速霸陸EJ族引擎#EJ255型|EJ255型]]渦輪增壓等兩種引擎為主，另外[[歐洲|歐洲]]市場還有一具2.0L水平對臥四缸DOHC [[速霸陸EE20型引擎|EE20型]]渦輪增壓[[柴油引擎|柴油引擎]]可供選擇。底盤懸吊為全新開發的SI Chassis（{{en}}Subaru Intelligent Chassis）結構，除了更輕量化，也比前一代更為靜肅。為了提升操控性和乘坐舒適性，首度在後輪懸吊系統採用[[雙橫臂懸架|雙A臂懸吊系統]]。自9月份起，陸續在[[澳洲|澳洲]]、[[西班牙|西班牙]]等地開賣。

同年7月31日美國速霸陸除維持原先「WRX」車型外，新設升級版的「GT」車型。後者同樣裝置了[[速霸陸EJ族引擎#EJ257型|EJ257型]]渦輪增壓引擎，配置四速手自排變速箱。經重新調校後，最大馬力從原先的224hp提升至265hp / 6,000rpm，峰值扭力則從31.3kg·m進步至33.7kg·m / 4,000rpm。同時重新設定[[懸吊系統|懸吊系統]]，前軸彈簧的彈性係數從26.5N/mm增加到38N/mm，後軸則從24N/mm增加到34N/mm；而前後防傾桿也同時加粗，前軸[[直徑|直徑]]從20mm增至21mm，後軸則從15mm提升至16mm。輪胎尺碼從205/50R17升級至225/45R17，至於ABS[[防鎖死煞車系統|防鎖死煞車系統]]、EBD[[電子制動力分配系統|電子制動力分配系統]]、VDC車輛動態控制系統、六顆[[安全氣囊|安全氣囊]]等則被列為標準配備。此外，這兩種車型從尾燈、排氣管等不同的設計可分辨得出來。

2011年 - 原廠在4月19日開幕的[[上海國際汽車工業博覽會|上海國際汽車工業博覽會]]發表「XV Concept」<ref>參考{{ja}}[http://www.fhi.co.jp/news/11_04_06/11_04_19.html 富士重工業 2011年上海モーターショーで「SUBARU XV CONCEPT」を公開]。</ref>，翌日開幕的[[紐約國際車展|紐約國際車展]]正式發表第四代大改款的Impreza四門、五門車型<ref>詳情請閱覽{{ja}}[http://www.fhi.co.jp/news/11_04_06/11_04_21.html 富士重工業：新型スバル インプレッサを公開]。</ref>，車身尺碼並沒有太大變動，但內裝方面則有所精進。同年11月30日日本市場宣布12月20日開賣，五門掀背車稱為「Sports」，四門轎車則稱「G4」。內裝鋪陳上使用黑色皮革與銀色金屬飾片打造座艙的質感，車用多媒體系統、4.3吋[[液晶显示器|液晶螢幕]]、[[衛星導航|衛星導航]]系統、行動通訊、恆溫空調、電動天窗等皆在配備清單中；外觀上則以鷹眼型頭燈和大面積蜂巢式水箱護蓋營造出該車款的性能風格。

動力方面搭載新開發的1.6L水平對臥四缸DOHC [[速霸陸FB族引擎#FB16型|FB16型]]引擎及2.0L水平對臥四缸DOHC [[速霸陸FB族引擎#FB20型|FB20型]]引擎，各有115ps / 5,600rpm、15.1kg·m / 4,000rpm以及150ps / 6,200rpm、20.0kg·m / 4,200rpm的輸出表現。搭配[[AVCS主動閥門控制系統|AVCS主動閥門控制系統]]、[[無段自動變速器#Lineartronic_CVT|Lineartronic CVT]]無段自動變速系統、TGV進氣控制系統（Tumble Generated Valve）、EGR[[排氣再循環|排氣再循環]]系統（Exhaust Gas Recirculation）等，經日本{{link-ja|JC08模式|JC08モード}}測試，FB16型的油耗表現為17.6km/L，FB20型則為17.2km/L。和前一代的底盤懸吊系統同樣採用前麥花臣、後雙A臂之設計，輔以Symmetrical AWD全時[[四輪驅動|四輪驅動]]系統、VDC[[电子稳定程序|車輛動態控制系統]]、[[防鎖死煞車系統|ABS]]+[[電子制動力分配系統|EBD]]+BA煞車輔助系統等。此外，2.0L車型追加[[EyeSight|EyeSight]]行車安全輔助系統，提升駕駛安全性。

2012年 - 雖然4月20日XV先行在[[臺灣|臺灣]]開賣，6月19日日本才宣告衍生車款[[速霸陸XV|XV]]將自成一格，10月5日正式上市。9月25日宣布Sports、G4兩種車型小改良（「B」型），將HAC上坡行車輔助系統（Hill-start Assist Control）列為標準配備，以及新增車色；同年10月1日四門及五門兩種車型同時獲得[[好設計獎|好設計獎]]。

2013年 - 11月12日開始發售改良的「C」型<ref>請閱覽{{ja}}[http://www.fhi.co.jp/news/13_10_12/13_10_29_92217.html 富士重工業：インプレッサ SPORT/G4を改良]。</ref>，加強車室靜肅性、變更動力方向機之設計等。11月21日在[[洛杉磯國際車展|洛杉磯國際車展]]公開美規WRX車型<ref>詳見{{ja}}[http://www.fhi.co.jp/news/13_10_12/13_11_21_93577.html 富士重工業：スバル新型「WRX」を発表]。</ref>，以Impreza G4為基礎換上新設計的水箱罩、頭燈、保險桿、側裙等，輪拱亦更為外擴突出以容納大尺碼[[輪胎|輪胎]]。所搭載的2.0L水平對臥四缸DOHC [[速霸陸FA族引擎#FA20_DIT型|FA20 DIT型]]渦輪增壓引擎，可輸出268ps / 5,600rpm的最大馬力與35.7kg·m / 2,000-5,200rpm的最大扭力。變速系統則提供六速手動排檔、[[無段自動變速器#Lineartronic_CVT|Sport Lineartronic CVT]]無段自動變速系統兩種，前者經重新調校後齒比範圍更寬，可提升油耗表現及高速行駛時的靜肅性；至於後者經重新調校後可模擬8速，換檔反應更靈敏。車身及懸吊系統經過調整，加上多模式VDC車輛動態控制系統和主動式扭力分配系統（Active Torque Vectoring）的輔助，提高車身的轉向反應。新款WRX更採用電動動力輔助方向盤，轉向機構剛性也提高，提供更直接的轉向手感回饋。

2014年 - 1月15日原廠在[[底特律車展|底特律車展]]上宣布新一代WRX STI即將在北美市場面世<ref>請看{{ja}}[http://www.fhi.co.jp/news/14_01_03/14_01_15_97393.html 富士重工業：スバル新型「WRX STI」を2014年北米国際自動車ショーにて世界初公開]。</ref>。相較於早兩個月發表的WRX車型，WRX STI的水箱護罩和前輪輪拱加貼了STI銘牌，車尾也多了大型擾流尾翼。至於18吋BBS製金色鋁合金輪圈則將以限量1千輛的方式，標配在美國販售的WRX STI Launch Edition特仕車上。內裝部分除了專屬紅黑配色麂皮座椅、D形平底式方向盤、STI飾板等，且配上了軟質材料和黑色頂篷，以便與WRX區隔。動力方面仍使用2.5L水平對臥四缸DOHC [[速霸陸EJ族引擎#EJ257型|EJ257型]]渦輪增壓引擎，可輸出最大馬力305hp / 6,000rpm、最大扭力290lb-ft / 4,000rpm。除了搭配6速手排變速箱、對稱式全時四輪驅動系統（Symmetrical AWD）、DCCD[[駕駛者可控式中央差速器|駕駛者可控式中央差速器]]、[[SI-DRIVE|SI-DRIVE]]駕駛輔助系統等，更具有主動式扭力引導系統（Active Torque Vectoring），可在車輛轉彎時對內側輪胎施以剎車力道，調整轉向反應。WRX STI採用了13.0:1的轉向比例，使得其左右死點僅有2.5圈；比起WRX的轉向比例14.5:1、左右死點2.8圈還要少。

5月8日台灣意美汽車正式引進性能版WRX和WRX STI共四種車型<ref>參考[http://news.u-car.com.tw/23161.html U-car車壇新聞：性能圖騰再起，Subaru WRX & WRX STI國內發表]。</ref>，前者在外觀方面擁有六角形水箱護罩、四出式排氣尾管、後保桿下方擾流鰭片、17吋金屬黑色鋁圈等；後者的外觀則具有全車紅色STI專屬徽飾、大型車尾擾流翼、18吋鋁圈等。在內裝方面，WRX以黑色為基礎，加上三輻式平底真皮[[方向盤|方向盤]]、紅光設計雙圓儀表板、紅色車縫線座椅等；WRX STI車型則具有紅色STI字樣的碳纖維飾板、紅色手動排檔桿頭等。在動力輸出的部分則和美規一樣，WRX為2.0L水平對臥四缸DOHC [[速霸陸FA族引擎#FA20_DIT型|FA20 DIT型]]渦輪增壓引擎、WRX STI為2.5L水平對臥四缸DOHC [[速霸陸EJ族引擎#EJ257型|EJ257型]]渦輪增壓引擎。

2015年 - 繼[[速霸陸XV|速霸陸XV]]之後，原廠於6月19日發表[[油電複合動力系統|油電複合動力]]車型'''Impreza Sport Hybrid'''<ref>{{ja}}[http://www.fhi.co.jp/news/15_04_07/15_06_19_116187.html 富士重工業株式会社：スバル インプレッサSPORT HYBRIDを発表]。</ref>，動力來源為2.0L水平對臥四缸DOHC [[速霸陸FB族引擎#FB20型|FB20型]]引擎，配合可輸出10kW（約13.6hp） / 6.6kgf·m功率之MA1型[[电动机|電動馬達]]與[[鎳氫電池|鎳氫電池]]模組，經日本{{link-ja|JC08模式|JC08モード}}測試平均油耗為20.4km/L。車型分成二種等級，皆將[[EyeSight|EyeSight]]行車安全輔助系統列入標準配備，且針對Hybrid動力做調整。新增ECO巡航控制機能，在啟用主動式定速巡航時能盡量以電能動力優先，長途駕駛時可省下可觀的燃油。

同年10月6日部分改良<ref>參看{{ja}}[http://www.fhi.co.jp/press/news/2015_10_06_1176/ 富士重工業株式会社：スバル　インプレッサ　SPORT／G4を改良～1.6Lモデルにアイサイト（ver.3）を設定～]。</ref>，[[EyeSight|EyeSight]]行車安全輔助系統擴及至「1.6i-L EyeSight」、「1.6i-S EyeSight」等車型，另改換[[紫色|紫色]]車色。

==== 安全性召回 ====
2015年7月15日美國速霸陸宣佈召回32,400輛在2011年4月21日至2012年2月16日之間生產的四門轎車與五門掀背車<ref>{{en}}[http://blog.caranddriver.com/subaru-recalls-32000-imprezas-for-airbag-sensors/ Car & Driver: Subaru Recalls 32,000 Imprezas for Airbag Sensors]。</ref>，由於設定在助手席的感應器可在無人時自動關閉其[[安全氣囊|安全氣囊]]系統，可是[[美國國家公路交通安全管理局|美國國家公路交通安全管理局]]卻收到26件報告指出，當助手席的乘客使用[[USB|USB]]裝置或碰觸到座椅調整手把等金屬物件，形成接地狀態時，偵測系統會誤判助手席沒有坐人。此時若發生交通事故，助手席之安全氣囊並不會作動而造成危險。美國速霸陸將通知車主回廠處理，以更換助手席的控制元件。此外[[台灣意美汽車|台灣意美汽車]]也表示臺灣地區有144輛車包含在這波召回行動中，該公司將主動通知車主回廠更換感應器。

=== 競速賽事 ===
==== 拉力賽 ====
自富士重工業發表第一代Impreza以來，該車款常被各級競速賽事的車隊使用。特別是[[國際汽車聯盟|國際汽車聯盟]]舉辦的[[世界拉力錦標賽|世界拉力錦標賽]]（以下簡稱WRC）中，常見到速霸陸與[[標緻汽車|標緻]]、[[雪鐵龍|雪鐵龍]]、[[福特汽車|福特]]等車隊一爭高下的身影。在2.0L級別的拉力賽競技用車輛中，速霸陸Impreza最常與同屬日系的[[三菱Lancer_Evolution|三菱Lancer Evolution]]拿來相提並論，也為速霸陸奠定性能與速度的品牌形象。不過受到經濟不景氣的影響，2008年12月16日富士重工業宣告退出WRC賽事，是2009年繼[[鈴木公司|鈴木汽車]]後第二家退出該比賽的車廠<ref>參見[http://www.auto-online.com.tw/news/14466 汽車線上：SUBARU宣告退出WRC系列賽事]。</ref>。
# 第一代四門 / {{link-en|A組賽車|Group A}}（1993年 - 1996年）：1993年[[科林·麥克雷|科林·麥克雷]]駕駛速霸陸Legacy在紐西蘭分站賽中奪冠，而[[理查德·伯恩斯|理查德·伯恩斯]]則在英國站稱雄；1994年科林·麥克雷駕駛速霸陸Impreza WRX STI在紐西蘭、澳大利亞及英國站稱雄；1995年速霸陸555車隊首次獲得廠商車隊總冠軍，科林·麥克雷則拿下車手總冠軍。1996年該車隊蟬聯廠商車隊總冠軍，科林·麥克雷獲得車手亞軍。
# 第一代雙門 / WRC（1997年 - 2000年）：1997年速霸陸WRC車隊再次蟬聯廠商車隊總冠軍，科林·麥克雷、皮爾洛·里亞蒂及埃里克森等車手共聯手拿下8個分站賽的冠軍；1998年科林·麥克雷兩度稱雄分站賽；1999年理查德·伯恩斯三奪分站賽冠軍，並斬獲年度車手亞軍。當年度速霸陸WRC車隊敗給豐田車隊，屈居廠商車隊亞軍。2000年[[理查德·伯恩斯|理查德·伯恩斯]]奪取年度車手亞軍，速霸陸WRC車隊則獲年度廠商車隊季軍。
# 第二代四門 / WRC（2001年 - 2008年）：2001年理查德·伯恩斯奪下年度車手冠軍；2002年[[托米·馬基寧|托米·馬基寧]]和{{link-en|皮特·索伯格|Petter Solberg}}分別在蒙地卡羅站和英國站稱霸，速霸陸WRC車隊獲得年度廠商車隊季軍。2003年皮特·索伯格首次奪得年度車手冠軍，速霸陸WRC車隊則為年度廠商車隊季軍；2004年皮特·索伯格拿到年度車手亞軍，速霸陸WRC車隊為年度廠商車隊季軍。2005年皮特·索伯格再次獲得年度車手亞軍；2006年速霸陸WRC車隊獲年度廠商車隊季軍；2007年速霸陸WRC車隊得到年度廠商車隊季軍。
# 第三代五門 / WRC（2008年-）：2008年12月16日富士重工業宣告退出WRC賽事。

至於美國拉力錦標賽，Impreza WRX STI於2005年正式參與至今，除2010年由車手安托萬·里斯坦（Antoine L'estange）駕駛[[三菱Lancer_Evolution|三菱Lancer Evolution]]奪冠外，其餘年度均奪下冠軍。

==== 場地賽 ====
在日本汽車聯盟賽事方面，2004年由車手克里斯·托雷斯駕駛WRX STI獲得全日場地錦標賽冠軍。美國跑車俱樂部賽事方面，2005年、2006年的Tire Rack SCCA Solo Nationals賽事中均奪得冠軍，2005年、2006年在STU級賽事中奪冠，2006年在AS級賽事中奪冠。

==== 公路賽 ====
在國際汽車聯盟賽事方面，2005年克里斯·迪波（Chris Deable）駕駛WRX STI在A6級賽事中奪得亞軍，並在紐博格林賽道24小時挑戰賽中排名第14名<ref>請參考{{en}}[http://www.subaru-msm.com/global/news/050511.html Subaru Global: IMPREZA finished 2nd in the class and 14th overall in its first Nürburgring challenge] {{Wayback|url=http://www.subaru-msm.com/global/news/050511.html |date=20110930184544 }}。</ref>。2006年英國的Deeble-Bellchambers Racing車隊使用WRX STI Spec-C參加紐博格林賽道24小時挑戰賽SP6級的比賽，比賽中總共燒毀兩個渦輪增壓器，最終在全部255輛參賽車輛中只獲得第115名<ref>請看{{en}}[http://www.subaru-msm.com/global/news/060619.html Subaru Global: SUBARU IMPREZA spec C Goes the Distance at the Nurburgring] {{Wayback|url=http://www.subaru-msm.com/global/news/060619.html |date=20121008234822 }}。</ref>。

[[File:Cusco_Dunlop_Subaru_Impreza_2008_Fuji_GT_500km.jpg|thumb]]
美國跑車俱樂部賽事中，Icy Racing車隊連續在2005和2006年使用WRX STI分別獲得俱樂部T2分級賽的冠軍及亞軍；ESX Motorsports車隊則於2005年使用WRX STI參加世界GT挑戰賽。

===== 超級GT賽 =====
自1990年代末期起，Cusco車隊便使用WRX STI參加超級GT賽。2008年該車隊在[[馬來西亞|馬來西亞]]的雪邦賽道初次奪冠，也造就了該比賽史上第一次由[[四輪驅動|四輪驅動]]賽車奪冠的記錄<ref>參考{{en}}[http://www.cusco.co.jp/en/whats_new/cusco_dunlop_subaru_impreza_dr.html CUSCO DUNLOP SUBARU IMPREZA Driver information updated]。</ref>。正因為WRX STI於2008年賽季的出色表現，超級GT賽委員會決定四門賽車可以參與往後的賽事（原本WRX STI得以參賽，全因超級GT賽委員會的破例）。

==== 漂移賽 ====
[[Image:Kumascooby.jpg|thumb]]
在D1國際汽車大獎賽方面，2005年Team Orange車隊第一次使用WRX STI參加D1國際汽車大獎賽，翌年該車隊奪得賽季冠軍。
