'''速霸陸Stella'''（{{lang-ja|スバル・ステラ}}）為2006年起由[[日本|日本]][[富士重工業|富士重工業]]製造販售的[[輕型高頂旅行車|輕型高頂旅行車]]。自2011年發售的第二代開始，改由同樣隸屬[[豐田汽車|豐田汽車]]集團的[[大發工業株式會社|大發工業]][[贴牌生产|OEM代工]]，以第五代[[大發Move|大發Move]]為原型[[換牌工程|換上速霸陸的廠徽銘牌]]而成。

== 概要 ==
此款車乃是以[[速霸陸R2|速霸陸R2]]為基礎而發展的[[輕型高頂旅行車|輕型高頂旅行車]]，並於2006年推出。有鑑於日本市場逐漸萎縮，富士重工業在[[輕型車|輕型車]]市場遇上瓶頸，基於成本和獲利考量，決定忍痛退出輕型車這塊市場<ref>參考[http://subaru.autonet.com.tw/cgi-bin/file_view.cgi?b0030680S4007 AutoNet: 退出輕型車市場，SUBARU日本銷售目標下修至10萬輛]{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}。</ref>。隨著[[速霸陸Pleo|Pleo]]、[[速霸陸R2|R2]]、[[速霸陸R1|R1]]等車款陸續宣布停產，第一代Stella曾一度成為該公司自製的輕型車。為了找回流逝的[[市場佔有率|市場佔有率]]，2008年4月10日豐田汽車、大發工業、富士重工業等三方曾公開發表合作開發的意向<ref>參看{{ja}}[http://www.fhi.co.jp/news/08_04_06/08_04_10-1.html 富士重工業：トヨタ・ダイハツ・富士重、開発・生産における新たな協力関係に合意]。</ref>，富士重工業遂委託大發工業代工，是以第二代Stella便在這樣的情況下登場。不過，該公司自製的第一代Stella曾一度在2009年推行[[電動車|電動車]]計劃，縱然後來此計劃遭到凍結中止。至於車名的由來，「Stella」在[[義大利語|義大利語]]裡意為[[恒星|星星]]。

2006年 - 6月14日正式上市，分成Standard型、Custom型、Revesta型等三種車型，最高端的Revesta型配備了[[發光二極管|LED]][[車頭燈|頭燈]]附燈光高低可調裝置。動力來源則有658c.c.直列四缸DOHC [[速霸陸EN族引擎#EN07D型（DOHC、AVCS）|EN07D型]]引擎及658c.c.直列四缸DOHC [[速霸陸EN族引擎#EN07X型（DOHC、機械增壓附中冷器）|EN07X型]][[機械增壓器|機械增壓]]引擎兩種選擇，搭配五速[[手動變速器|手動排檔]]或[[無段自動變速器|i-CVT]]等二種變速系統。為了刺激銷售量，特意找來[[澤尻英龍華|澤尻英龍華]]作為產品代言人。

2007年 - 當年度[[東京改裝車展|東京改裝車展]]曾展出Revesta型改裝車，獲得好評，11月29日遂正式市售。具有專屬水箱護罩、前後保險桿、鋁合金輪圈等，座椅也覆上專用材質，另有一套「Music CD Server and Welcome Sound Audio」系統，在引擎啟動前已經設定好駕駛人喜愛的[[音樂|音樂]]，一上車便能享受聽覺效果。

2008年 - 原廠在[[第34届八国集团首脑会议|第34屆G8八大工業國高峰會]]上公開以Stella為基礎而開發的[[電動車|電動車]]「Plug-in Stella（{{ja}}プラグインステラ）」，這款車乃是原廠與[[東京電力|東京電力]]共同合作的成果，極速可達80km/hr、空車[[重量|重量]]1,060kg、可乘坐4人，無記憶效應的高效能鋰錳電池每充飽一次電可行駛約80[[公里|公里]]的[[距離|距離]]。原廠提供5輛車給高峰會使用，另一輛則提供給{{link-ja|郵政事業|郵便事業}}使用。同年10月24日[[武田藥品工業|武田藥品工業]]和住友三井汽車服務共同倡導的電動車計劃則宣布，翌年起引進Stella電動車先在[[東京都|東京都]]和[[神奈川縣|神奈川縣]]作為業務用車，2010年再推廣到其他地區。11月4日原廠決定將原先僅Custom型與Revesta型可選的658c.c.直列四缸DOHC [[速霸陸EN族引擎#EN07X型（DOHC、機械增壓附中冷器）|EN07X型]][[機械增壓器|機械增壓]]引擎推廣到全車系，另外亦將藍光LED地圖燈、室內燈、一體成形的頂置式收納空間、尾翼、皮革座椅等列為Custom型與Revesta型的標配或選配。

2009年 - 6月4日宣布「Plug-in Stella」自7月起開始交車銷售。

2010年 - 因為充電設備不普及，[[消費者|消費者]]對於電動車興趣缺缺，再加上[[電池|電池]]造價成本昂貴，而且「Plug-in Stella」的知名度不如[[日产聆风|日產Leaf]]和{{link-ja|三菱i-MiEV|三菱・i-MiEV}}，使得電動車計劃就此凍結。

2011年 - 3月4日正式停產。

2011年 - 5月24日發表大改款的第二代Stella，其實與第五代大發Move幾乎完全相同。空調系統、自動收折[[倒後鏡|後視鏡]]、[[巡航定速|巡航定速]]、倒車顯影、駕駛輔助系統、LDWS車道偏移輔助系統、VSC車身動態穩定系統與FSRCC雷達輔助自動巡航系統、ABS[[防鎖死煞車系統|防鎖死煞車系統]]、EBD[[電子制動力分配系統|電子制動力分配系統]]、六具SRS[[安全氣囊|安全氣囊]]等隨車型不同而配備。第二排座椅能前後移動240mm，後廂門採用側開式。另外尚有Custom車型，具有專屬空力套件、四圓[[車頭燈|頭燈]]、15吋[[鋁合金|鋁合金]]輪圈、黑色車室內裝等。

動力來源為一具658c.c.直列三缸DOHC KF-VE3型引擎，可輸出52hp / 7,200rpm的最大馬力、6.1kg·m / 4,000rpm的最大扭力，搭配CVT[[無段自動變速器|無段自動變速箱]]，並藉由i-EGR廢氣回收系統與idle-stop[[怠速熄火系統|怠速熄火系統]]之輔助，可達到27.0km/L的油耗表現。此外，延續上一代的Custom車型並未搭載[[機械增壓器|機械增壓]]引擎，反而是[[自然进气发动机|自然進氣版]]。直到同年8月1日「Custom RS」車型上市，才追加658c.c.直列三缸DOHC KF-DET3型[[渦輪增壓器|渦輪增壓]]引擎，可輸出64ps / 6,400rpm的馬力、9.4kg・m / 4,000rpm的扭力。

2012年 - 5月21日跟隨第五代[[大發Move|大發Move]]的腳步，「Custom RS」車型導入大發工業最新開發的eco IDLE引擎怠速熄火系統，按照日本{{link-ja|JC08模式|JC08モード}}，[[前置前驅|FF]]車之平均油耗表現達24.2km/L、[[四輪驅動|4WD]]車則為22.2km/L。

同年12月20日宣布小改款，原CVT[[無段自動變速器|無段自動變速箱]]新增變速適溫控制系統，且輔以eco IDLE引擎怠速熄火系統、動能回收充電系統、Cooled i-EGR油氣回收系統等新科技，依日本{{link-ja|JC08模式|JC08モード}}下的平均油耗，自然進氣版達29.0km/L、渦輪增壓版則是25.2km/L。此外，[[懸吊系統|懸吊系統]]、[[煞車|煞車]]系統、轉向系統等皆經過改良，並新增能偵測自身與前車的安全距離及時煞車的Smart Assist系統。

2013年 - 7月1日原廠宣布「Custom RS」車型亦新增Smart Assist系統。10月28日部分改良<ref>{{ja}}[http://www.fhi.co.jp/news/13_10_12/13_10_28_92214.html 富士重工業株式会社：スバルステラシリーズを改良～ ステラにターボエンジン搭載グレード追加設定 ～]。</ref>，追加附Smart Assist系統的「LS Smart Assist」車型，並將緊急煞車時尾燈會閃爍的Emergency Stop Signal系統列入全車系標準配備。

2014年 - 5月22日發售特仕車「G」、「GS」、「Custom R type S」・「Smart Assistα」等四款。

2014年 - 12月12日與[[大發Move|大發Move]]同步發表大改款的第三代車款，共分Stella和Stella Custom兩種車型，水箱護罩重新設計。動力心臟為658c.c.直列三缸DOHC KF-VE3型引擎，分成自然進氣版和渦輪增壓版，前者具有52hp / 6,800rpm的[[馬力|馬力]]和6.4kg·m / 5,200rpm的扭力，後者則提升為64hp / 6,400rpm、9.4kg·m / 3,200rpm的輸出[[功率|功率]]；兩種引擎都搭配[[無段自動變速器|CVT]]無段變速箱。Smart Assist安全輔助系統、Push Start引擎啟閉按鈕、Keyless免鑰匙感應門鎖系統、[[怠速熄火系統|怠速熄火系統]]、行駛模式切換等皆列入配備清單，按不同等級而配置。

2015年 - 5月20日實施部分改良<ref>{{ja}}[http://www.fhi.co.jp/news/15_04_07/15_05_20_115796.html 富士重工業株式会社：スバル ステラを改良]。</ref>，Smart Assist安全輔助系統新設一個攝影鏡頭，可提供防衝撞警報、防衝撞剎車輔助、[[車道偏離警示|車道偏離警示]]等功能。

2016年 - 再度於6月21日實行部分改良<ref>{{ja}}[http://www.fhi.co.jp/press/news/2016_06_21_2513/ 富士重工業株式会社：スバル ステラを改良　～ステラカスタムに新グレード追加～]。</ref>，追加「Custom F」車型，具有[[LED|LED]]頭燈組、雙環式[[汽車儀表|儀表板]]等配備，同時廢除「GS」、「GS Smart Assist」二種車型。
