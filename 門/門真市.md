'''門真市'''（{{jpn|j=門真市|hg=かどまし|rm=Kadoma shi|f=y}}）是位於[[日本|日本]][[大阪府|大阪府]][[大阪市|大阪市]]東北側的城市，屬於大阪市周邊的[[通勤城市|通勤城市]]。

[[松下電器|松下電器]]總公司登記的地址位於該市，但由於該公司的廠區橫跨門真市與[[守口市|守口市]]兩市的市界，最初松下電器的總公司辦公室確實也是位於門真市的轄區內，但經過搬遷後，目前實際上松下電器的總公司辦公室是位於守口市；而該公司創辦人[[松下幸之助|松下幸之助]]其生前的住所也位於總公司的廠區內，正好就緊鄰兩市市界靠門真市的一側，因此其也成為門真市的居民，1964年獲頒為該市的榮譽市民。

== 歷史 ==

此地過去位於{{Tsl|ja|河内湖|河內湖|河內灣}}之中，直到[[繩文時代|繩文時代]]才開始逐漸成為陸地。在令制國時代屬於[[河內國|河內國]][[茨田郡|茨田郡]]。

[[平安時代|平安時代]]後，此地開始出現大量朝廷、寺院、武家的莊園，包括大和田莊、馬伏莊、岸和田莊；由於鄰近[[大坂|大坂]]及[[京都|京都]]，在[[江戶時代|江戶時代]]屬於幕府直屬領或是旗本領。<ref name="History">{{Cite web| url=http://www.city.kadoma.osaka.jp/shisei/k_shokai/rekishi.html| title=歷史| publisher=門真市政府| language=ja}}</ref>19世紀末[[明治維新|明治維新]]後，曾先後隸屬大坂裁判所（後來的[[大阪府|大阪府]]）、[[河內縣_(日本)|河內縣]]、[[堺縣|堺縣]]，在1881年才又隨著堺縣一同被併入大阪府。

1889年實施[[町村制|町村制]]，現在的門真市轄區在當時分屬：'''門真村'''、{{Tsl|ja|大和田村 (大阪府)|大和田村 (大阪府)|大和田村}}、{{Tsl|ja|四宮村}}、{{Tsl|ja|二島村}}四個行政區劃；門真村在1939年先改制為'''門真町'''，1956年再併入其餘三個村，形成現今的轄區範圍，1963年門真町再改制升格為現在的'''門真市'''。<ref name="History"/>

江戶時代末期起，本地開始栽種[[蓮藕|蓮藕]]，之後一度更以｢河內蓮藕」之名在日本全國打出知名度。<ref name="History"/>

1910年[[京阪電氣鐵道|京阪電氣鐵道]]在此開設[[京阪本線|鐵路]]，開始帶動本地工業的發展；1933年[[松下電器|松下電器]]的前身松下電器器具製作所也將總部遷移至此，隨著松下電器規模的日益成長，逐漸成為門真最主要的企業，甚至市政的主要稅收也都來自松下電器。<ref name="History"/>

但在1990年代[[日本泡沫经济|日本泡沫经济]]破裂，松下電器的業績開始低迷，連帶影響門真市政府的稅收，造成財政困難，與門真市與相鄰的[[守口市|守口市]]過去也同樣仰賴松下電器及[[三洋電機|三洋電機]]的稅收，也面臨相同困境；兩市在2002年為解決財政困境，開始商議進行合併已取得發行特例公債的權力，經商討後兩市一度取得共識規劃合併為{{Tsl|ja|守口門真市}}，但在2004年的公民投票中，未獲得居民的同意，合併案因而告吹。

== 交通 ==

轄內的主要鐵路路線為[[京阪電氣鐵道|京阪電氣鐵道]]的[[京阪本線|京阪本線]]，其以東西向通過轄內主要市區，該線上的{{Tsl|ja|門真市駅|門真市車站}}也是轄內主要的鐵路車站，[[大阪高速鐵道|大阪高速鐵道]]的[[大阪單軌電車線|大阪單軌電車線]]自北側進入市內後，在此與京阪本線交會，現在同時也是大阪單軌電車線的終點站。

在市區最南端還有[[大阪市高速電氣軌道|大阪市高速電氣軌道]][[長堀鶴見綠地線|長堀鶴見綠地線]]的終點站[[門真南車站|門真南車站]]，目前大阪單軌電車線已計畫將自門真市車站門真南站。

高速公路部分，[[近畿自動車道|近畿自動車道]]則從北往南通過轄區，在南部的[[門真系統交流道|門真系統交流道]]同時也是快速道路{{Tsl|ja|第二京阪道路}}往北的起點，此條快速道路可通往[[京都市|京都市]]。

客運部分則有{{Tsl|ja|京阪バス|京阪巴士}}、{{Tsl|ja|近鉄バス|近鐵巴士}}、{{Tsl|ja|大阪シティバス|大阪城市巴士}}等多家業者經營門真市的客運路線。

== 觀光資源 ==
[[File:P1000208.JPG|thumb]]

[[松下電器|松下電器]]將最初的公司總公司辦公室所在地改設為公園，公園由[[安藤忠雄|安藤忠雄]]所設計，種植大量[[櫻花|櫻花]]，因此命名為{{Tsl|ja|さくら広場|櫻花廣場}}，[[松下幸之助|松下幸之助]]過去在門真的住所也位於此公園北側，在公園的東側還有由松下電器公司所設立的松下電器博物館暨松下幸之助歷史館。

日本歷史上有名的[[一休和尚|一休和尚]]的母親死後被埋葬在該市，其墳墓位於門真南车站附近的下三島公園内。<ref>{{Cite web| url=http://www.city.kadoma.osaka.jp/shisei/k_shokai/meisho.html| title=名所、史跡、文化財| date=2016-04-01| publisher=門真市政府| language=ja| accessdate=2018-07-25}}</ref>

== 教育 ==

[[File:Osaka_Prefectural_Kadoma_Namihaya_High_School.jpg|thumb]]

門真市雖然是個人口超過十萬的城市，但是轄內並沒有任何高等教育學校，甚至也沒有私立高中，僅有{{Tsl|ja|大阪府立門真西高等学校|大阪府立門真西高等學校}}和{{Tsl|ja|大阪府立門真なみはや高等学校|大阪府立門真浪速高等學校}}兩間公立高中。
