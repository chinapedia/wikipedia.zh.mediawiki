'''路得·卡特琳娜·馬塔·普福'''（{{lang-de|Ruth Katherina Martha Pfau}}，{{bd|1929年|9月9日|2017年|8月10日|df=y}}），是一位[[德國|德國]][[天主教|天主教]][[修女|修女]]和[[醫生|醫生]]，她一生在[[巴基斯坦|巴基斯坦]]幫助[[痲瘋|痲瘋]]病患者。

==生平==
1929年出生在德國[[萊比錫|萊比錫]]，路得·普福是德國出生的巴基斯坦[[人道主義|人道主義]]者, 她有五個姐妹和一個弟弟。當[[第二戰世界大戰|第二戰世界大戰]]結束後，[[蘇聯|蘇聯]]軍隊佔領[[德意志民主共和國|德意志民主共和國]], 她同她家人逃到[[德意志聯邦共和國|德意志聯邦共和國]]，並選擇[[醫藥|醫藥]]為她未來的[[職業|職業]]生涯。1949年時路得·普福在在[[美茵茨|美茵茨]]讀[[醫學|醫學]]，但她感覺她自己的生活不夠滿足。1962年時，路得·普福加入了一間[[宗教|宗教]]秩序到[[印度|印度]]，但因[[簽證|簽證]]問題，她停留在[[巴基斯坦|巴基斯坦]]。她來到[[卡拉奇|卡拉奇]]並視察了[[火車站|火車站]]背後之[[痲瘋|痲瘋]]病患者聚居地位於麥克里奧德路。在那邊路得·普福決心把她終生用來照顧那些[[病人|病人]]。她開始給痲瘋病患者在貧民區小屋接受[[治療|治療]]。因此「瑪麗·阿德萊德·麻風病中心」<ref>{{cite web
|title = Marie Adelaide Leprosy Centre
|url = http://www.malc.org.pk/ab_drprofile.htm
|deadurl = yes
|archiveurl = https://web.archive.org/web/20090816074619/http://www.malc.org.pk/ab_drprofile.htm
|archivedate = 2009-08-16
}}</ref> 的痲瘋病患者社會組織機構被創建，1963年四月買了一間麻風病診所，然而患者從[[卡拉奇|卡拉奇]]、[[巴基斯坦|巴基斯坦]]及[[阿富汗|阿富汗]]都來接受治療<ref>{{cite web |title = Leprosy in Pakistan contained, says Dr Ruth Pfau
| url= http://www.dawn.com/2004/02/02/nat24.htm}}</ref>。 
自那時以來工作增長迅速, 小治療中心在卡拉奇與巴基斯坦各個地區成立，醫務工作者和社會工作者培訓被提供，[[健康|健康]][[教育|教育]]漸漸克服了任何傳統的妨礙或恐懼。
路得·普福有時到巴基斯坦遙遠地區, 那裡經常沒有痲瘋病患者的醫療設施，所以她經常收集德國和巴基斯坦捐款並與[[拉瓦爾品第|拉瓦爾品第]]和[[卡拉奇|卡拉奇]]醫院合作。她在巴基斯坦和國外被確認為一個傑出的人，並已獲得了許多獎項和獎章。
