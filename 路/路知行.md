'''路知行'''{{BD|1983年|11月8日||}}，中国大陆男性[[配音演员|配音演员]]。

== 简介 ==
'''路知行'''的网名叫'''路知知'''。<ref># {{Cite web |url= http://weibo.com/luzhizhi |title= 路知行的新浪微博 |author= 路知行 |publisher= 新浪微博 |quote= 路知行的微博名为路知知 }} </ref>2008年毕业于[[北京大学|北京大学]]新闻传播学院广告专业<ref># {{Cite web |url= http://www.cztv.com/s/2013/zsyjt/scopes/2013/02/2013-02-163748020.html |title= 电视节目中的个人介绍 |author= 浙江卫视 |date=2013年2月2日 |publisher= 中国网络电视台 |accessdate= 2013年9月2日  }} </ref>

大学毕业后在[[北京电视台|北京电视台]]担任出镜记者，[[配音员|配音员]]。<ref>{{ Cite visual| author =北京人民广播电台| url =http://audio.rbc.cn/play.form?programId=415&start=20130728080000| title =故事大全栏目| publisher =北京故事广播| pages =36分30秒| date =2013年7月28日}}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }} </ref>

因喜爱创作，动画以及配音，于2007年离开北京电视台，开始从事影视剧，动画，栏目，专题片，等各种题材的配音以及后期音频制作，动画制作等工作。同时致力于原创动画的创作，以及文化产品的交流活动。其在配音方的特点是声音年轻帅气可塑性强，性格直率有自己的感悟，为诸多影视剧中的人物完成后期配音并塑造了诸多动画人物的声音形象<ref># {{Cite web|url=http://audio.rbc.cn/play.form?programId=1434&start=20130730210000|title=北京故事广播《故事恳谈会》|author=主持人小丹 嘉宾路知行|date=2013年7月30日|publisher=北京人民广播电台|quote=访谈介绍了配音演员路知行的一些个人经历工作经历以及配音作品}}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }} </ref>
