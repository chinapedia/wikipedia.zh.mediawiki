'''宝伊局'''，[[清朝|清朝]]在[[新疆|新疆]][[伊犁|伊犁]]设立的[[铸钱局|铸钱局]]，为清朝在新疆设立的7个铸钱局之一{{efn|另外六个分别是[[叶尔羌局|叶尔羌局]]、[[阿克苏局|阿克苏局]]、[[乌什局|乌什局]]、[[库车局|库车局]]、[[喀什噶尔局|喀什噶尔局]]和[[宝迪局|宝迪局]]}}，厂址位于原[[伊犁将军|伊犁将军府]]所在地惠远城，即今新疆伊犁州[[霍城县|霍城县]][[惠远镇|惠远镇]]。宝伊局始设于1775年，前后铸钱91年。<ref name=ylrb>{{Citeweb |title=清代伊犁的铸币局：宝伊局 |url=http://www.jibi.net/News/gbzs/8_24_29_112.html |publisher=中国集币在线-伊犁日报 |date=2012-06-21}}</ref>

宝伊局的建立，结束了长期以来新疆天山以北没有铸币基地的状况，促进了北疆经济的发展。<ref name=ylrb/>

== 背景 ==
[[File:Mints_in_Xinjiang.svg|缩略图]]
清代，伊犁一直是新疆的政治、经济中心，自[[乾隆|乾隆]]年间清军就在伊犁驻守。虽然靠实行开垦屯田弥补部分军费，但每年仍需靠内地供银50余万两，绸缎数万匹与当地[[中國哈薩克族|哈萨克]]牧民交换牛羊马匹<ref name="ylrb" />。由于当地缺乏[[制钱|制钱]]，给清军与牧民的交易造成了很大的不便。乾隆三十六年，曾担任过阿克苏参赞大臣，对[[阿克苏局|阿克苏]]、[[乌什局|乌什]]设局铸钱甚为熟悉的[[舒赫德|舒赫德]]调任伊犁将军后，开始筹划在伊犁设局铸钱。他认为“伊犁为[[准部|准部]]汇总之区，兵屯甚众，商贩尤殷，需用钱文较他处为更要”，因而“鼓铸一事必应筹办”。但由于当时伊犁未发现[[铜矿|铜矿]]，宝伊局并未及时开设。<ref name=xjtvs>{{citeweb|title=风雨沧桑宝伊局 走过百年铸币路 |url=http://www.xjtvs.com.cn/culture/yx/2013-1-9/1357728157116.shtml |date=2013-01-09}}</ref>

== 铸币历史 ==
乾隆四十年（1775年），按[[伊犁将军|伊犁将军]][[伊勒图|伊勒图]]的建议，为巩固边疆而在伊犁开设宝伊局铸造铜钱。宝伊局的开铸结束了北疆长期没有铸币基地的历史，促进了北疆经济的发展<ref name=ylrb/>。宝伊局设炉2座，开始时每年铸钱1100余串{{efn|一串1000文}}，后每年铸钱1722串，所用铜料除在当地[[巴彦岱镇|巴彦岱]]开挖之外，主要靠南疆各地调拨。宝伊局所铸造的乾隆通宝钱数量大，版式丰富<ref name="daquan">{{cite book|author1=杜坚毅|author2=顾佩玉|title=新疆红钱大全图说|date=1996-03|publisher=中华书局|isbn=9787101015300}}</ref>{{rp|60}}。

[[嘉庆|嘉庆]]年间，新疆仅有[[阿克苏局|阿克苏]]、宝伊两局铸钱，且所铸钱币中乾隆通宝两成，嘉庆通宝八成。宝伊局所铸嘉庆通宝，按面文“嘉”字中“吉”的大小可分为为“小吉嘉”与“大吉嘉”两种版式，其中“小吉嘉”铸造的数量大大超过“大吉嘉”<ref name=xjtvs/><ref name="daquan"/>{{rp|99}}。

[[道光|道光]]年间，仍按旧例，所铸钱币中乾隆通宝两成，道光通宝八成<ref name="daquan"/>{{rp|129}}。道光二十年（1840年），宝伊局因铜少而暂停<ref name=xjtvs/>。

[[咸丰_(年号)|咸丰]]元年（1851年），为镇压[[太平天国|太平天国]]，清政府军费大增，而战乱导致的赋税收入减少、铸币材料短缺，则使得财政愈加困难，为弥补财政亏空，清政府开始发行纸币，并令各地铸钱局开铸大钱。1853年，宝伊局开铸大钱，成为当时铸行大钱的大局之一，也是新疆第一家铸造大钱的铸钱局。咸丰六年（1856年），因造成钱法混乱，大钱被废。咸丰年间，宝伊局除小部分平钱外，所铸大钱分为当四、当十、当五十、当百、当五百、当千数种<ref name=xjtvs/>。按《咸丰钱的版式系列》一书，当百、当五十、当十、当四和小平等均有实物发现，而当五百仅见记载<ref name=qzy>{{cite book|author=齐宗佑编著 |coauthors= |title=中国钱币丛书甲种本之五-咸丰钱的版式系列 |year=2002 |publisher=中华书局出版社 |location=北京 |isbn=7-101-02416-5}}</ref>{{rp|344}}。当五百、当千大钱则可见于零星的奏章中，如[[奕山|奕山]]咸丰四年正月十五日奏文中中曾提及“宝伊局铸重二两的当千大钱”，次年又在奏文中提到“当千、当五百不能行使，奏明改铸当百以下大钱”。最早披露的一枚当五百原为[[张絅伯|张絅伯]]旧藏，后归[[陈仁涛|陈仁涛]]，现为[[中国国家博物馆|中国国家博物馆]]展出<ref>{{cite web|title=咸丰元宝（宝伊当五百）|url=http://www.chnmuseum.cn/tabid/212/Default.aspx?AntiqueLanguageID=5984|website=中国国家博物馆|accessdate=2018-03-04}}{{Dead link|date=2018年9月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref name="artron">{{cite web|title=0084 清 “咸丰元宝”背“宝伊当五百”|url=http://auction.artron.net/paimai-art0024130084/|website=雅昌艺术网|accessdate=2018-03-04}}</ref>。[[阿克苏博物馆|阿克苏博物馆]]的馆藏文物中亦有当五百大钱<ref>{{cite web|title=咸丰元宝•宝伊局•当五百|url=http://www.aksmuseum.com/jpdc/qianbi/2012/0705/240.html|website=阿克苏博物馆|accessdate=2018-03-04}}</ref>。[[马定祥|马定祥]]著《[[咸丰泉汇|咸丰泉汇]]》中收录有一枚[[陈鸿禧|陈鸿禧]]旧藏的当五百，曾在2013年进行拍卖<ref name="artron"/>。咸丰初年，宝伊局曾铸造了少量的小平钱，后大多被回收铸造大钱，或被沙俄搜刮回国，因此存世不多<ref name="daquan" />{{rp|182}}。

[[同治|同治]]初年，宝伊局铸造了与咸丰重宝当四钱一脉相承的同治重宝当四钱<ref name="daquan" />{{rp|279}}。同治三年（1864年），伊犁发生[[同治新疆回变|农民起义]]，同治五年（1866年），起事者占领惠远城，宝伊局被迫停炉。延续近百年的宝伊局遂成历史。<ref name=ylrb/><ref name=xjtvs/>

1871年[[俄罗斯帝国|俄国]]侵占伊犁后，由于宝伊局铸钱含铜量较高，在侵占伊犁10年中，原有宝伊局制钱被沙俄大量搜刮回国，改铸铜戈比或用作军工原料。1881年2月24日清政府和俄罗斯签订《改订陆路通商章程》，其中内容有专门提到禁止将中国铜钱贩运出口<ref>[[:s:伊犁条约|伊犁条约]].[[维基文库|维基文库]]</ref>。光绪十五（1902年）年二月十三日，伊犁将军[[色楞额|色楞额]]的奏折追录了当时情形称“俄国代收代守，纵有制钱，俱贱价买去，消灭无迹……”。<ref name=xjtvs/>

== 钱文 ==
宝伊局所铸铜钱的钱文与[[清代造币机构列表|中国内地各局]]格式相同。面文为四字[[汉字|汉文]]：小平钱的面文为年号加“通宝”两字，当五十及以下大钱的面文为年号加“重宝”两字，当百及以上大钱的面文为年号加“元宝”两字。背文也与新疆[[阿克苏局|阿克苏局]]、[[乌什局|乌什局]]等局所铸的背文左为[[满文|满文]]，右为[[老维文|维文]]局名的格式不同，而是与内地制钱的格式相同，不用维文，而用满文“{{ManchuSibeUnicode|ᠪᠣᠣ<br>ᡳ}}”（宝伊），在穿左右。

== 材质 ==
宝伊局所铸造钱币与内地各局所铸钱币等价流通，但含铜量更高一些，内地制钱材质为“铜六铅{{efn|此处的“铅”指“白铅”，即金属[[锌|锌]]}}四”的[[黄铜|黄铜]]，呈黄色，而宝伊局制钱为“铜七铅{{efn|此处的“铅”指“黑铅”，即金属[[铅|铅]]}}三”的[[青铜|青铜]]，稍显灰涩，为暗红色。由于[[果子沟|果子沟]]天堑的阻挡，道路难行，宝伊局铸币用的材料多经[[夏特古道|夏特古道]]从[[南疆|南疆]]输入伊犁<ref name=ylrb/>。

宝伊局历代铸币多为红铜，有少量黄铜等其他材质的铸钱，尤以咸丰年间铸币的材质种类最多，不仅有青铜、黄铜、红铜，还有铁质和铅质的铸币。咸丰重宝当四钱多为红铜，铁质的较少，黄铜材质的罕见；而当十以上的大钱多为黄铜材质，红铜的较少，铁质的罕见<ref name="daquan" />{{rp|182}}。
