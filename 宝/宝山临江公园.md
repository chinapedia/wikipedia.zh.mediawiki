'''宝山临江公园'''位于[[上海市|上海市]][[宝山区|宝山区]]，是一座具备[[历史|历史]][[人文|人文]]特色的城市[[公园|公园]]，公园建成于1956年，是在清代宝山庙宇古建筑原址和周边土山基础上，由宝山县和上海市北郊区的团员青年，通过[[义务|义务]]劳动的方式而建立。最初定名为“共青公园”。建成同年8月，改名为“友谊公园”，后因园区濒临[[长江|长江]]，于1962年改名为“临江公园”。是宝山的标志性公园<ref name="multiple">{{Cite web|url =http://service.shanghai.gov.cn/newdtbs/QueryPlatform/PublicFacilitiesQuery2_107_10023.html|title =宝山临江公园|date =2011-11-07|publisher =上海市人民政府|access-date =2014-05-22|archive-url =https://web.archive.org/web/20140522192525/http://service.shanghai.gov.cn/newdtbs/QueryPlatform/PublicFacilitiesQuery2_107_10023.html|archive-date =2014-05-22|dead-url =yes}}</ref>。

公园东邻长江，南临宝林八村和临江佳园等住宅小区，西靠东林路，北倚宝城三村(海景苑)。其正门位于友谊路1号，在公园东部望江楼附近和公园东南角与临江佳园之间，另设有两个小出入口。

临江公园的部分建筑具备[[中国|中国]]古典[[江南|江南]][[园林|园林]]风格，园内残存着一段宝山古[[城墙|城墙]]和一座水关桥<ref name="shtong">{{Cite web |url =http://shtong.gov.cn/newsite/node2/node71994/node82435/node82463/index.html|title =临江公园|publisher =上海市地方志办公室}} </ref>。公园中心区域为面积约1万平方米的人工湖，湖的东侧为[[上海淞沪抗战纪念馆|上海淞沪抗战纪念馆]]，北侧为[[陈化成|陈化成]]纪念[[广场|广场]]，西南侧有一座明代古桥<ref name="shtong" />，宝善桥。

==大事记==
*1956年，公园建成；
*1962年10月，公园改名为“临江公园”；
*1966年4月，公园并入相邻的[[海滨|海滨]]浴场；
*1978年，公园恢复原名开放；
*1991年，公园进行全面扩建；
*1992年，公园在原宝山[[孔庙|孔庙]]大成殿，建立了[[陈化成纪念馆|陈化成纪念馆]]<ref>{{Cite web |url =http://lhsr.sh.gov.cn/sites/lhsr/neirong.aspx?infid=91bb901b-7cc4-4389-98e2-2e774925c55f&ctgid=94bd5dca-f2ae-43e0-b2ef-68015bd12f83|title =宝山临江公园|date =2011-11-07|publisher =上海市绿化和市容管理局}} </ref>；
*1993年5月，在公园东侧临江位置建起了望江楼；
*2000年1月28日，上海淞沪抗战纪念馆在园内落成开放；
*2004年，公园西北部扩建；
*2007年5月，公园进行局部改建，新增成河林带木[[栈道|栈道]]；
*2008年1月，被命名为上海市三星级公园<ref>{{Cite web |url =http://lhsr.sh.gov.cn/sites/lhsr/neirong.aspx?infid=c25076c1-5d8e-4d8e-a343-a531f9c9c0cd&ctgid=fc45fb5b-3ac3-474b-8458-16c20bd2fc17|title =上海市绿化管理局关于命名2007年度上海市星级公园、优美绿地景点和绿化景观道路的通知|date =2008-02-01|publisher =上海市绿化和市容管理局}} </ref>；
*2010年6月21日，被命名为上海市文明示范公园<ref>{{Cite web |url =http://lhsr.sh.gov.cn/sites/lhsr/neirong.aspx?ctgid=bbda8a85-68d3-4da8-80ff-98133fac8103&infid=694df102-a6ee-46de-98ef-ade5ed7ca9d0 |title =关于命名第二批上海市文明示范公园的通知 |date =2010-06-21 |publisher =上海市绿化和市容管理局}} </ref>。

==公园特色==
*古城墙 - 位于公园东南角，距围墙内侧约50米。城墙建于明[[嘉靖|嘉靖]]十六年(1537年)，屡经战乱和灾害，1956年又经人为拆除，园内现在仅留有很短一段土丘，上面植有树木。
*宝山城水关桥 – 建于清[[顺治|顺治]]年间，为单孔石拱结构，[[花岗石|花岗石]]材质，现遗址位于公园古城墙西侧。
*宝善桥 – 现位于公园人工湖东南侧，桥面由九块石板铺成。该桥最初建于明[[天启|天启]]五年(1624年)，后又于清[[嘉庆|嘉庆]]五年（1800年）、清[[光绪|光绪]]二十八年(1902年)和[[中华民国|中华民国]]2年(1913年)先后重修。
