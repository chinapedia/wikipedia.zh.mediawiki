'''宝船'''，又名宝舟，是[[郑和下西洋|郑和下西洋]]时的船只，為[[中國帆船|中國帆船]]的一種。

== 歷史文獻记載 ==

[[伊本·白图泰|伊本·白图泰]]所著的《游记》有记載對宋元时代來往印度洋的中国海船的描述：「大者张12帆，载1000人，其中六百名水手，400名士兵弓箭手等。大船用20至30摇橹，每橹15至30人摇，单是操作摇橹，需要300至900人。」

随[[郑和|郑和]]下西洋的翻译官[[马欢|马欢]]所著的《[[瀛涯胜览|瀛涯胜览]]》有三种定稿本，分别是《说集本》，《淡生堂本》和《三宝征夷集本》。明抄本《说集本》：「宝船六十三号，大者长四十四丈四尺，阔一十八丈，中者长三十七丈，阔一十五丈。」。<ref>明 [[马欢|马欢]]原著 万明校注 明钞本 《[[瀛涯胜览|瀛涯胜览]]》校注  第207页 海洋出版社 2005 ISBN 7-5027-6378-3 </ref>《瀛涯胜览》明抄本《淡生堂本》：「宝船六十三号，大者长四十四丈四尺，阔一十八丈，中者长三十七丈，阔一十五丈。」。<ref>明 [[马欢|马欢]]原著 万明校注 明钞本 《[[瀛涯胜览|瀛涯胜览]]》校注 第207页 海洋出版社 2005 ISBN 7-5027-6378-3 </ref>《瀛涯胜览》明抄本《三宝征夷集本》：「宝船六十三只，大者长四十四丈四尺，阔一十八丈，中者长三十七丈，阔一十五丈。」。<ref>明 [[马欢|马欢]]原著 万明校注 明钞本 《[[瀛涯胜览|瀛涯胜览]]》校注 第5页，第207页 海洋出版社 2005 ISBN 7-5027-6378-3 </ref>

1434年随郑和下西洋的[[龚珍|龚珍]]在《[[西洋番国志|西洋番国志]]》中记录：「其所乘之宝舟，体势巍然，巨无以敌。蓬、帆、摇橹，非二三百人莫能举动。」。<ref>龚珍《[[西洋番国志|西洋番国志]]·自序》，第6页，中华书局向达校注，ISBN 7-101-02025-9</ref>

《郑和家谱·下西洋船舶》：「拨舡六十三号，大船长四十四丈，阔一十八丈；中船长三十七丈，阔十五丈」。<ref>郑鹤声 郑一钧 编《郑和下西洋文选》87页 海洋出版社 2005 ISBN 7-5027-6376-7</ref>

明[[顾起元|顾起元]]《[[客座赘语|客座赘语]]·宝船厂》：「宝船六十三号，大船长四十四丈四尺阔一十八丈，中船长三十七丈，阔一十五丈。」。<ref>[明[[顾起元|顾起元]]《[[客座赘语|客座赘语]]》第29页《宝船厂》 凤凰出版社江苏地方文献丛书 ISBN 7-80643-979-X]</ref>

《明史稿》《郑和传》：「和等造大船，修四十四丈，广十八丈，中容数百人。」。<ref>《明史稿》 郑和传</ref>清代《[[明史|明史]]·鄭和傳》记载：「造大舶，修四十四丈，广十八丈者六十二。」。

[[费信|费信]]的《[[星槎胜览|星槎胜览]]》：「永乐七年己丑，上命正使太监郑和等统领官兵，驾使海船四十八号，往诸番国开读赏赐。是岁秋九月，自太仓刘家港开船，十月到福建长乐太平港泊。十二月，福建五虎门开洋，张十二帆，顺风十昼夜，至占城国」。<ref>[[费信|费信]]《[[星槎胜览|星槎胜览]]·占城国》</ref>

明[[罗懋登|罗懋登]]《[[西洋通俗演义|西洋通俗演义]]》对郑和下西洋的船队有较详细的叙述，郑和船只，最大的一号船，上有九桅杆，名为宝船，长四十四丈四尺，阔一十八丈，共36号：二号船立八道桅杆，名为马船，长三十七丈，阔一十五丈，共700号；三号上立七桅杆，船名为粮船，，长二十八丈，阔一十二丈，共240号；四号船上立六桅杆，名为坐船，长二十四丈阔九丈四尺，共300号，五号船上立五桅杆，名为战船，长一十八丈，阔六丈六尺，共180号。

明[[祝允明|祝允明]]《前闻记》记载，郑和下西洋所用的海船，有一号二号等，有清和号、惠康号、长宁号、安济号、清远号等，有以船上用橹之大小，分别为大八橹、小八橹<ref>^[[向达|向达]]校注 《[[西洋番国志|西洋番国志]]》附录六：祝允明《前闻记下西洋》55-56页 ISBN 7-101-02025-9 
</ref>。

《明史·兵志》记：「宝船高大如楼，底尖上阔，可容千人」。

== 爭論 ==
关于鄭和宝船是不是如《[[明史|明史]]》所记载的那样大，有两派不同的观点。

*肯定派：
**曾有學者以簡單的材料力學理論計算四十四丈長的寶船所需強度與結構厚度等，而其所得在一合理範圍內。{{Citation needed}}
**曾有學者以現代船舶的統計回歸資料略估鄭和寶船的船型及各種流體動力學方面的性能，並且推估其阻力與動力來源尚稱合理。{{Citation needed}}
**[[伊本·白图泰|伊本·白图泰]]游记中早已纪录中国巨大的12張帆可载千人的海船。白图泰的纪录可作为12帆载千人的宝船的旁证。
**南京郑和造船厂的船坞宽可容20丈，與寶船尺寸相符。
**南京静海寺郑和残碑所记“两千料海船”，不是“宝船”，而是较小的“战座船”。

*质疑派：
**认为《明史·鄭和傳》記载出自《三宝太监西洋通俗演义记》。{{Citation needed}}
**根据[[南京|南京]][[静海寺|静海寺]]郑和所立残碑记载：「永樂三年，將領官軍乘駕二千料海船並八擼船……清海道。永樂四年，大船駐於舊港港口，即古之三佛齊。……首陳祖義、金志名等，於永樂五年七月內回京。由是永樂七年，將領官軍乘駕一千五百料海船並八擼船。……國王亞烈苦奈柰兒謀劫錢糧船隻」；根据推论，折合长约为长十五到二十丈，宽六到八丈左右，载重量约为450吨（15料為一噸）。
**造船厂出土的15米长舵桿僅適合用於四到五倍長的福船，船长约为60到75米。（若寶船是較不耐風浪的沙船，則尺寸相符。）
**至今未有人复制能够实际航行的44丈“宝船”。（目前复制中的宝船多采用质疑派的说法。）
**认为木材强度有限，过大的船体无法保证水密性，难以做长时间的航行。<ref>[https://en.wikipedia.org/wiki/List_of_longest_wooden_ships 直至20世紀根據科學驗證和鋼鐵加固的木製船，沒有在92米以上海中航行遇到結構問題的。]</ref>

== 寶船還原 ==
2010年，由中國南京鄭和寶船廠所製造的一艘可進行遠洋航行的仿明代鄭和寶船的寶船主體，經已完成三分一；該船長逾71米，排水量達一千六百噸，自上而下共五層。寶船以中國的傳統工藝方式建造，預料於2011年完成船體的安裝，2013年進行航行試驗。<ref>[http://hk.news.yahoo.com/article/101021/4/kuds.html 「鄭和寶船」2013年啓航]</ref>2012年7月：“仿古鄭和寶船船體落成儀式”舉行，船體主體框架基本完成。但是在船身主體已經完工，龍骨、船舷側外板也已安裝到位后，寶船復建工作就已暫停。[http://ccnews.people.com.cn/n/2014/0611/c141677-25133042.html南京：「鄭和寶船」耗資千萬美元仿建8年後停工]{{Dead link|date=2018年7月 |bot=InternetArchiveBot |fix-attempted=no }}

== 模拟航海性能 ==

=== 有效馬力與速率 ===

若是在速率五節時，寶船模型比[[福船|福船]]的阻力小92%。以接近一般帆船的速率來看，若是能提供 50kW 的功率，寶船能達到約 4.4 節。福船之浸水表面積大於寶船，所以使得福船阻力大於寶船，使寶船船速會大於福船。按文獻上的航行路线和时间推算，宝船每日航行大概一百海里，航速平均大概四至五节。<ref>[http://myweb.ncku.edu.tw/~chenjh/research/zheng-he/equal-disp.PDF 陳政宏、許智超，“鄭和寶船復原模型與等排水量福船及沙船的性能比較”，[[國立成功大學|國立成功大學]]造船及船舶機械工程學系] {{webarchive|url=https://web.archive.org/web/20150513102122/http://myweb.ncku.edu.tw/~chenjh/research/zheng-he/equal-disp.PDF |date=2015-05-13 }}</ref><ref>[http://webcache.googleusercontent.com/search?q=cache:pFr6DziJTS8J:myweb.ncku.edu.tw/~chenjh/research/zheng-he/prelim.PDF+&cd=3&hl=en&ct=clnk&gl=us 許智超、陳政宏，“鄭和寶船復原模型與典型福船及沙船性能之初步比較研究”，[[國立成功大學|國立成功大學]]造船暨船舶機械工程學系]</ref>

=== 橫搖穩度與安全性 ===

寶船模型的穩度消失角約為 84 度。穩度消失角相當大，但並非不合理或不可能。一些現代船隻也常有 超過 90 度的穩度消失角。寶船稜塊係數適中，狀似圓型；使其能够有大的穩度消失角。最大扶正力臂及其對應的橫傾角為衡量大傾角穩度之重要指標。以最大值發生的角度來看。寶船其船寬大、乾舷也大，使其扶正力臂最大值較早出現，故仍具有足夠之扶正力矩，表示其能承受大的橫傾角，在海上有良好的運動性。寶船模型橫搖週期的數值約為 2.2 秒到 2.8 秒左右。
