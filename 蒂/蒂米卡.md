'''蒂米卡'''（Timika）是[[印度尼西亚|印度尼西亚]][[巴布亚省|巴布亚省]][[米米卡县|米米卡县]]的行政中心，位于巴布亚省西部。2010年统计人口106,529<ref>https://www.citypopulation.de/Indonesia-CU.html</ref>，2014年统计人口130,000<ref>{{Cite news|url=http://www.insideindonesia.org/faultlines-and-fractures|title=Faultlines and fractures - Inside Indonesia|newspaper=Inside Indonesia|accessdate=2017-01-08}}</ref>，从人口来看是[[巴布亚省|巴布亚省]]第二大城市以及[[西新几内亚|西新几内亚]]的第三大城市<ref>{{cite web |url=http://www.citypopulation.de/Indonesia-CU.html |title=INDONESIA: Cities & Settlements |last= |first= |date=2017-04-22 |website=citypopulation.de |publisher= |access-date=2017-07-16 |quote=}}</ref>。[[天主教蒂米卡教区|天主教蒂米卡教区]]隶属于[[天主教马老奇总教区|马老奇总教区]]，主教座堂三贤士主教座堂位于城内。

== 人口 ==

蒂米卡的大部分居民是来自于印尼各地的移民，蒂米卡也是巴布亚省外国人最多的城市，他们大部分都在[[印尼自由港公司|印尼自由港公司]]（Freeport Indonesia）工作。大约有2000人来自[[美国|美国]]，其余的外国人来自[[日本|日本]]、[[加拿大|加拿大]]、[[澳大利亚|澳大利亚]]、[[德国|德国]]<ref>{{Cite news|url=http://www.papua.us/2015/10/jumlah-wna-di-timika-terbanyak-di-tanah.html|title=Jumlah WNA di Timika Terbanyak di Tanah Papua|newspaper=Papua Untuk Semua|accessdate=2017-01-08}}</ref>。

== 经济 ==

[[File:Grasberg_mine.jpg|thumb]]

[[印尼自由港公司|印尼自由港公司]]隶属于[[费利浦·麦克莫兰铜金公司|费利浦·麦克莫兰铜金公司]]，出产[[金|金]]、[[银|银]]、[[铜|铜]]的精矿，工作区在蒂米卡北部的[[毛克山脉|毛克山脉]]南麓。该公司在米米卡县拥有全球储量最大的[[格拉斯伯格矿场|格拉斯伯格矿场]]铜金矿，占印尼全国铜产量的70%以上<ref name=":0">{{Cite web|url=http://www.mofcom.gov.cn/article/i/jyjl/j/201507/20150701041819.shtml|title=印尼自由港公司将归还印尼约60%的铜金矿开采权。中华人民共和国商务部网站|accessdate=2017-01-08|work=www.mofcom.gov.cn}}</ref>。

2015年的谈判中，印尼自由港公司同意将现有6座铜金矿中的4座归还给印尼，印尼自由港公司持有的矿权面积将从原来的21.3万公顷减少为9万公顷，减少幅度高达约60%。印尼能矿部长苏迪尔曼·萨义德表示，上述约60%的矿权将归还给印尼国家，然后由印尼地方政府运作经营，如果地方政府无法自主运营，中央政府将通过国企协助其经营<ref name=":0" />。

== 交通 ==

[[莫泽什吉朗因机场|莫泽什吉朗因机场]]位于城市北部，有飞往印尼各大城市和周围村镇的航班。[[跨巴布亚公路|跨巴布亚公路]]经过蒂米卡。
