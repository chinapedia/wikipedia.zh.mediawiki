'''蒂宾根行政区'''（Regierungsbezirk Tübingen）是[[德国|德国]][[巴登-符腾堡州|巴登-符腾堡州]]的4个[[行政区_(德国)|行政区]]之一，首府[[蒂宾根|蒂宾根]]。蒂宾根行政区的行政级别隶属于巴登-符腾堡州政府，行政区下设8个县和1个直辖市。

== 行政区划 ==

图宾根行政区下设8个县：

# [[比伯拉赫县|比伯拉赫县]]（Biberach），首府[[比伯拉赫|比伯拉赫]]（Biberach）
# [[博登湖县|博登湖县]]（Bodenseekreis），首府[[腓特烈港|腓特烈港]]（Friedrichshafen）
# [[拉芬斯堡县|拉芬斯堡县]]（Ravensburg），首府[[拉芬斯堡|拉芬斯堡]]（Ravensburg）
# [[罗伊特林根县|罗伊特林根县]]（Reutlingen），首府[[罗伊特林根|罗伊特林根]]（Reutlingen）
# [[锡格马林根县|锡格马林根县]]（Sigmaringen），首府[[锡格马林根|锡格马林根]]（Sigmaringen）
