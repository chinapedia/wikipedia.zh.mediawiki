'''蒂莫西·迈克尔·“蒂姆”·凯恩'''（{{lang-en|'''Timothy Michael "Tim" Kaine'''}}，{{bd|1958年|2月26日|catIdx=Kaine}}）生於[[明尼苏达州|明尼苏达州]][[圣保罗_(明尼苏达州)|圣保罗]]，[[美国|美国]]律师、政治人物，現任[[維吉尼亞州|維吉尼亞州]][[美國參議院|聯邦參議員]]。

曾任传教士，從政後曾擔任[[里士满_(弗吉尼亚州)|里士满]]市长、弗吉尼亚副州长、[[弗吉尼亚州|弗吉尼亚州]]州长。2016年獲[[希拉里·克林顿|希拉里·克林顿]]选择作为[[民主党_(美国)|民主党]]副总统参选人。

== 早年及教育 ==
凱恩出生於[[明尼苏达州|明尼苏达州]][[圣保罗_(明尼苏达州)|圣保罗]]市一個愛爾蘭裔的[[天主教|天主教]]家庭，他的父親是一名焊接工及小鐵器行老闆。凱恩在[[堪薩斯州|堪薩斯州]][[堪薩斯城_(堪薩斯州)|堪薩斯城]]都會區長大。

凱恩於[[密蘇里大學|密蘇里大學]]攻讀經濟，1979年獲[[文學士|文學士]]學位，而後進入[[哈佛大學法學院|哈佛大學法學院]]就讀，期間凱恩曾經前往[[宏都拉斯|宏都拉斯]]從事傳教工作，因此他能說流利[[西班牙語|西班牙語]]，1983年獲[[法學博士|法學博士]]學位。他在維吉尼亞州與家庭參加一間非裔美國人為主的[[教會|教會]]。

== 弗吉尼亚州州长 ==
凱恩于2005年當選弗吉尼亚州州长，任期四年。因弗吉尼亚州的州憲法禁止州長連任，他在卸任後，轉為擔任任期兩年的美國民主黨全國委員會主席。

== 联邦参议员 ==
2012年，凱恩當選聯邦參議員。

== 2016年总统大选 ==
2016年7月22日，[[希拉里·克林顿|希拉里·克林顿]]宣布选择蒂姆·凯恩作为副总统参选人角逐[[2016年美国总统选举|2016年美国总统选举]]<ref>{{cite news|url=http://www.nytimes.com/2016/07/23/us/politics/tim-kaine-hillary-clinton-vice-president.html|title=Hillary Clinton Selects Tim Kaine, a Popular Senator From a Swing State, as Running Mate|work=[[The_New_York_Times|The New York Times]]|last=Chozick|first=Amy|date=July 22, 2016|accessdate=July 22, 2016}}</ref>。
