'''越前花堂站'''（{{jpn|j=越前花堂駅|hg=えちぜんはなんどうえき|rm=Echizen-hanandō eki|f=y}}）是位於[[福井縣|福井縣]][[福井市|福井市]]花堂中一丁目的[[西日本旅客鐵道|西日本旅客鐵道]]（JR西日本）[[越美北線|越美北線]]（九頭龍線）鐵路車站。

[[米原站|米原站]]發車的北陸本線、至本站為初次非電化路線合流車站（2003年3月14日為[[敦賀站|敦賀站]]）。

== 利用可能的鐵道路線 ==
* 西日本旅客鐵道（JR西日本）
** [[北陸本線|北陸本線]]
** [[越美北線|越美北線]]（九頭龍線） ※線路名稱以本站為起點、運行系統上的起點為[[福井車站_(福井縣)|福井站]]。

另外、越美北線於旅客資料上優先使用愛稱「九頭龍線」、故以下亦以「九頭龍線」記載。

== 車站構造 ==
北陸本線為[[對向式月台|對向式月台]]2面2線、九頭龍線為[[側式月台|側式月台]]1面1線、計3面3線月台的地上車站。北陸本線月台與九頭龍線月台為扇形展開。站舎設於北陸線下行月台側、北陸線上行月台需經[[跨線橋|跨線橋]]移動、九頭龍線月台與北陸線上行月台有連絡通路通行。

無人車站、設有自動售票機。

== 車站周邊 ==
西方約300m為[[福井鐵道福武線|福井鐵道福武線]]的[[花堂站|花堂站]]。

車站北方為[[南福井站|南福井站]]（貨物站）。北陸本線東側為JR西日本金澤支社[[福井地域鐵道部|福井地域鐵道部福井運轉中心]]。

* [[Sakai_Ovex|Sakai Ovex]]花堂合繊工場

== 历史 == 
* 1960年（[[昭和|昭和]]35年）12月15日 - [[日本國有鐵道|日本國有鐵道]]越美北線勝原站開業同時、同線單獨站設置。
* 1968年（昭和43年）10月1日 - [[國民體育大會|國民體育大會]]福井秋季大會、北陸本線月台設置。
* 1984年（昭和59年）4月1日 - 無人化。
* 1987年（昭和62年）4月1日 ： [[國鐵分割民營化|國鐵分割民營化]]成為西日本旅客鐵道車站、越美北線起點變更為本站。
