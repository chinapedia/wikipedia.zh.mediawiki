'''越西县'''（原名'''越嶲'''（{{標音|字=嶲|拼音=suĭ}}）或'''越巂'''（{{標音|字=巂|拼音=xī}}），1959年更名为“越西”<ref>http://www.scnjw.gov.cn/scfuz/liangshan/lvyou/2008101020050321.html{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>；{{lang-ii|ꃺꄧꑤ vyt tuo xiep}}），是[[中国|中国]][[四川省|四川省]][[凉山彝族自治州|凉山彝族自治州]]所辖的一个[[县|县]]。越西县位于四川省西南部,凉山彝族自治州北部,东邻[[美姑县|美姑县]],南接[[昭觉县|昭觉县]]、[[喜德县|喜德县]],西界[[冕宁县|冕宁县]],北连[[甘洛县|甘洛县]]、[[石棉县|石棉县]]。全县幅员面积2256.47平方公里,最高海拔4791米,最低海拔1170米.气候系康定-雅江暖温带和西昌-巴塘亚热带区的过渡带,年平均气温13.3度,年降水量1113毫米,无霜期247天.全县辖40个乡镇,289个行政村,4个社区居民委员会,总人口35.6622万人,其中彝族27.8612人,占78.13%.是一个以彝族为主,彝、汉、藏、回、苗、布依族等十多个民族杂居的农业县。

== 历史 ==
越嶲的名字来源于嶲鸟，即[[杜鹃|杜鹃]]，秦汉时称[[蜀人|蜀人]]为嶲，当地亦有[[望帝|望帝]]化为嶲鸟的传说<ref>徐中舒《<交州外域记>蜀王子安阳王史迹笺证》，载《论巴蜀文化》，四川人民出版社，1981:157页。</ref>。越西县地汉朝置[[越嶲郡|越嶲郡]]，南梁属[[嶲州|嶲州]]，北周先后改为西宁州、严州，隋唐时属嶲州，南诏、大理时期属[[建昌府|建昌府]]邛部州，元朝属[[罗罗斯宣慰司|罗罗斯宣慰司]]建昌路邛部州。明朝正式建越嶲卫，属四川[[建昌府|建昌府]]。清朝改建昌府为[[宁远府|宁远府]]，改越嶲卫为越嶲厅。民国改越嶲厅为越嶲县。中华人民共和国改越嶲县为越西县。

===汉晋时期===
[[西汉|西汉]][[汉武帝|武帝]]元鼎六年（前111年）发兵征西南夷，在此地设[[越嶲郡|越嶲郡]]，治所在[[邛都|邛都]]，地在今[[西昌市|西昌市]]境内，今越西县境内设有[[阐县|阐县]]<ref>常常误作“阑县”，实际上是“阐县”</ref>。[[王莽|王莽]]建国元年（9年）改越嶲为领戎，后又改为集嶲，[[东汉|东汉]]又改回越嶲。[[西晋|西晋]]时，越嶲郡仅辖五县，其中并无阐县，越西县地属[[台登县|台登县]]。

===南北朝隋唐时期===
[[萧齐|齐]]改越嶲郡为[[越嶲獠郡|越嶲獠郡]]，废邛都县。[[南梁|南梁]][[梁武帝|武帝]]大同三年（537年）武陵王[[萧纪|萧纪]]镇蜀，置嶲州，设越嶲郡。[[北周|北周]][[周武帝|武帝]]保定五年（565年）改嶲州为[[西宁州|西宁州]]，在原阐县址设[[邛部郡|邛部郡]]，后因战乱废西宁州，周武帝天和五年（570年），大将军[[郑恪|郑恪]]率军征越嶲，改西宁州置[[严州|严州]]。[[隋|隋]]开皇六年（586年）复为西宁州，开皇十八年（598年）又改为[[嶲州|嶲州]]，[[大業_(年號)|大业]]初，改州为郡，嶲州改为越嶲郡，原邛部郡改为[[邛部县|邛部县]]。[[唐|唐]]武德元年(618年)，改越嶲郡为嶲州，邛部县不改，天宝元年（742年）改州为越嶲郡，[[唐肃宗|肃宗]]至德元年 (756年)[[吐蕃|吐蕃]]、[[南诏|南诏]]联兵攻陷越嶲郡，[[唐德宗|德宗]]贞元十三年(797年)，[[剑南西川节度使|剑南西川节度使]][[韦皋|韦皋]]发兵收复嶲州。因南诏连年入侵嶲州，[[唐文宗|文宗]]大和六年（832年）五月，[[西川节度使|西川节度使]][[李德裕|李德裕]]移州治至[[台登县|台登县]]。[[唐懿宗|懿宗]]咸通六年（865年）五月，嶲州为南诏所据。<ref name="doc">{{cite web |url=http://szb.lsz.gov.cn/read.aspx?id=20262 |title=存档副本 |accessdate=2013-08-20 |deadurl=yes |archiveurl=https://web.archive.org/web/20150510165031/http://szb.lsz.gov.cn/read.aspx?id=20262 |archivedate=2015-05-10 }}，凉山史志，古代</ref>

===南诏、大理、宋元时期===
南诏在嶲州境设[[建昌府|建昌府]]（为郡一级），属[[會川府|会川都督府]]（治今[[会理县|会理县]]），设三郡，今越西县地属[[建昌郡|建昌郡]]。[[宋朝|宋]]时一度收复建昌北部，在今越西县地设[[邛部州|邛部州]]，后又为大理国所据。[[大理国|大理国]]时期，升建昌郡为建昌府，今越西县地属邛部州。[[元|元]][[元世祖|世祖]]中统五年（1264年），设[[邛部川安抚招讨司|邛部川安抚招讨司]]，至元十二年（1275年）平定[[罗罗斯|罗罗斯]]诸部，设[[罗罗斯宣慰司|罗罗斯宣慰司]]，下辖四路总管府，今越西县地属[[建昌路|建昌路]]邛部川安抚招讨司，隶[[四川行省|四川行省]]，至元十九年(1282年)，罗罗斯宣慰司拨属[[云南行省|云南行省]]，至元二十一年（1284年）改邛部川安抚招讨司为[[邛部州|邛部州]]。[[元顺帝|元顺帝]]至正十三年(1353年)，[[红巾军|红巾军]][[明玉珍|明玉珍]]率军攻占建昌。<ref name="doc"/>

===明朝时期===
[[明|明]]洪武四年(1371年)，罗罗斯宣慰使[[安配|安配]]率部归顺明朝，授土指挥使，仍守建昌卫，洪武十五年（1382年）置建昌府，辖[[邛部州|邛部州]]，洪武二十五年（1392年），建昌卫指挥使[[月鲁贴木儿|月鲁贴木儿]]联合诸部叛乱，占据境内，十一月，凉国公[[蓝玉|蓝玉]]率师平定叛乱，废建昌府，在今越西县治[[越嶲卫|越嶲卫军民指挥使司]]，这是今越西县治正式称越嶲之始，邛部州隶之，自此越西不再称北周以来的“邛部”，而改称“越嶲”。永乐元年（1403年）邛部州改设长官司，称邛部长官司，属[[四川行都司|四川行都司]]越嶲卫。<ref name="doc"/>

===清朝、民国时期===
顺治十六年（1659年），清军进兵建昌，同年，改四川行都司为[[四川总镇府|四川总镇府]]。康熙元年(1662年)改设建昌监理厅，主管五卫政事，越嶲卫属之。雍正六年（1728年），平定凉山诸部落叛乱，裁建昌监理厅置[[宁远府|宁远府]]，所辖越嶲卫改置[[越嶲厅|越嶲厅]]。<ref name="doc"/>民国二年（1913年），废府存县，撤宁远府，改厅、州为县，越嶲厅改为越嶲县。
原宁远府所属地区称宁属，越嶲县为宁属，隶属上川南道（[[建昌道|建昌道]]）。民国十八年（1929年）撤道，各县直隶四川省府。民国二十四年（1936年），四川推行[[行政督察区|行政督察区]]制，宁属划为第十八行政督察区，越嶲县属之。民国二十八年（1940年），划宁属隶属[[西康省|西康省]]，越嶲县属之。<ref name="doc2">{{cite web |url=http://szb.lsz.gov.cn/read.aspx?id=20263 |title=存档副本 |accessdate=2013-08-20 |deadurl=yes |archiveurl=https://web.archive.org/web/20140913133059/http://szb.lsz.gov.cn/read.aspx?id=20263 |archivedate=2014-09-13 }}，凉山史志，中华民国时期</ref>

===中华人民共和国时期===
1950年，[[中华人民共和国|中华人民共和国]]攻占越嶲县，属[[西昌专区|西昌区专员公署]]。1952年4月30日,设立凉山彝族自治区(州)，首府[[昭觉县|昭觉县]]，1955年4月，改区为凉山彝族自治州。1956年1月，越嶲县划属凉山州。1956年，析越嶲县、昭觉县部分地区设[[普雄县|普雄县]]，驻越西贡莫。1959年，普雄、越隽两县合并置“越西县”。1960年，将越西县[[海棠区|海棠区]]划归[[甘洛县|甘洛县]]。<ref name="doc3">{{cite web |url=http://szb.lsz.gov.cn/read.aspx?id=20264 |title=存档副本 |accessdate=2013-08-20 |deadurl=yes |archiveurl=https://web.archive.org/web/20150510170155/http://szb.lsz.gov.cn/read.aspx?id=20264 |archivedate=2015-05-10 }}，凉山史志，中华人民共和国时期</ref><ref name="doc4">[http://www.xzqh.org/html/show/sc/21817.html]，越西县历史沿革</ref>

===行政区划===
越西县辖5个镇：越城、普雄、乃托、新民、中所，35个乡：河东、西山、大花、南箐、大瑞、马拖、新乡、板桥、瓦岩、梅花、白果、铁西、拉普、尔觉、德吉、尔赛、保石、拉吉、贡莫、书古、大屯、丁山、古二、拉白、四甘普、乐青地、五里箐、瓦普莫、瓦里觉、瓦曲觉、瓦曲乃乌、依洛地坝，1个民族乡：保安藏族乡。2011年6月中旬暴雨泥石流导致境内普雄段成昆铁路暂时中断运行。
越西是文昌帝君张亚子的诞生地，历史文化悠久。

===农业===
越西县是传统农业大县,农业占具基础性地位,农产品因小区气候环境而独具特色。全县已形成了以“大凉山”越西甜樱桃、苹果、贡椒、豆腐乳、萝卜干、马铃薯淀粉、苦荞茶、菜籽油、核桃、鲜蔬菜等为重点的特色农产品品牌系列.其中,甜樱桃、苹果、贡椒、马铃薯4个农产品获得“国家地理标志”认证;辣椒、萝卜、白菜、黄瓜、苹果、马铃薯等6种农产品获无公害食品认证;甜樱桃、苹果获绿色食品认证。
===金融业===
全县有农业银行、凉山商业银行、邮政储蓄银行、农商银行等4家金融机构。

==教育==
全县有各级各类中小学179所,其中:完全中学2所,职业技术学校、初级中心7所、九年一贯制学校3所、直属小学3所、乡镇中心小学39所.全县在校生   78720人,其中:小学在校学生46356人,初中在校学生 11076 人,高中在校学生 2798人,职业技术学校在校学生  410 人,学前教育在校幼儿 18080人。
==医疗==
全县有公立医疗卫生单位共45个(县医院1所,县疾控中心1所,县妇幼保健计划生育服务指导中心1所,中彝医院1所,社区卫生服务中心1所,中心卫生院8所,乡镇卫生院32所),民营医院4家,村卫生室289个.全县医院床位700张。
==交通==
县城距州府西昌130公里,距省会成都市395公里。国道[[G248|G248]]经过越西县。
