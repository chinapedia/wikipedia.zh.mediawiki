'''越南基督教新教'''是一个较小的宗教社群，占总人口的0.5%到2 %<ref>[http://www.state.gov/g/drl/rls/irf/2006/71363.htm US State Department 2006 country report on Vietnam]</ref>，根据2006年政府人口统计，大约有100万人（1%）<ref>[http://www.vietnamembassy.us/news/story.php?d=20070202130018 Religion in Vietnam 2006]</ref><ref>[http://www.vietnamembassy.us/docs/Vietnam%20White%20Paper%20on%20Religion.pdf Vietnam Affirms Consistent Policy on Religion: White Paper] {{webarchive|url=https://web.archive.org/web/20070224223005/http://www.vietnamembassy.us/docs/Vietnam%20White%20Paper%20on%20Religion.pdf |date=2007-02-24 }}</ref>。不过，更正教是越南增长最快的宗教，在过去10年间增长率达到600%<ref name="liberty">{{cite web |author=| title =Annual Report on International Religious Freedom for 2005 - Vietnam | work = U.S. Department of State| date = 2005-06-30| url = http://www.unhcr.org/cgi-bin/texis/vtx/rsd/rsddocview.html?tbl=RSDCOI&id=437c9cdd2&count=0| accessdate = 2006-07-21 }}</ref>。 

== 起源 ==
更正教于1911年由加拿大[[宣道会|宣道会]]传教士[[翟辅民|翟辅民]]传入越南[[岘港|岘港]]。[[宣道会|宣道会]]共派遣了100多名传教士到越南。第一个被政府承认的更正教机构是越南福音教会（北方）（1963年）<ref name="liberty">{{cite web |author=| title =Annual Report on International Religious Freedom for 2005 - Vietnam | work = U.S. Department of State| date = 2005-06-30| url = http://www.unhcr.org/cgi-bin/texis/vtx/rsd/rsddocview.html?tbl=RSDCOI&id=437c9cdd2&count=0| accessdate = 2006-07-21 }}</ref>。 

到1967年，更正教团体主要位于南越境内，包括法国归正会、[[圣公会|圣公会]]、[[宣道会|宣道会]]、[[浸信会|浸信会]]、基督会、环球福音会和[[基督复临安息日会|基督复临安息日会]]。1967年，在南越有15万名更正教信徒，占总人口的大约1%<ref>[http://www.sacred-texts.com/asia/rsv/rsv11.htm Protestantism in South Vietnam, US Navy, Bureau of Naval Personnel, Chaplains Division, 1967]</ref>。 

1975年共产党接管南越时，没收了一些更正教会的财产，包括[[芽庄|芽庄]]和[[河内|河内]]的神学院。数以百计的更正教教堂在这期间被摧毁<ref name="conf">{{cite web |author = Compass Direct| title =Vietnam Protestants Call Conference 'Miraculous' | work = Christianity Today| date = 2002-09-20| url = http://www.christianitytoday.com/ct/2001/107/36.0.html| accessdate = 2006-07-21 }}</ref>。

1975年，越南所有的更正教教派被当地共产党政府组合成一个单一的机构，称为越南福音教会，到1997年有50万正式成员。不过，在1988年，当一些活跃的牧师被逐出官方教会时，[[家庭教会|家庭教会]]运动开始了。其中最突出的一位牧师是 Tran Dinh Ai，他拥有 16,000 信徒和165 个教会（1997年）<ref>[http://www.fva.org/0297/pastors.htm FVA News]</ref>。

== 越南圣经公会 ==
[[新教|新教]]翻译了几种圣经译本 in 1926年 and 1991年，并将[[旧约|旧约]]和[[新约|新约]]的若干卷翻译成越南语<ref>[http://www.vnbaptist.net/Tai_Lieu/History_of_VN_bible.htm History of the Vietnamese Bible]</ref>。

联合圣经公会在越南的工作开始于1890年。1966年成立了越南圣经公会。2005年，圣经公会在越南分发了53,170册越南文[[圣经|圣经]]和120,170册[[新约|新约]]<ref>[http://www.biblesociety.org/bs-vnm.htm Bible Society work in Vietnam]</ref>。2年前，2003年，得到政府的允许，在越南印刷了6万册越南文圣经和5万册新约（all in Vietnamese）。同年，在越南还为当地华人社团印刷了1万册[[中文|中文]]圣经。其中7,555册 在几个月内销售一空<ref>[http://ubs-ao.org/VietnamAnnRep.dsp United Bible Societies, Asia Pacific - Laos & Vietnam] {{webarchive|url=https://web.archive.org/web/20080612150210/http://ubs-ao.org/VietnamAnnRep.dsp |date=2008-06-12 }}</ref>。

== 现状 ==
目前，超过一半的越南新教徒属于[[家庭教会|家庭教会]]<ref name="state">{{cite web | title =Vietnam | work = US State Department| date = 2002-09-07| url = http://www.state.gov/g/drl/rls/irf/2002/13916.htm| accessdate = 2006-07-21 }}</ref>。信仰的增长最具有戏剧性的是在山地少数民族中，例如Mnong、Ede、Jarai和Bahnar<ref name="conf">{{cite web |author = Compass Direct| title =Vietnam Protestants Call Conference 'Miraculous' | work = Christianity Today| date = 2002-09-20| url = http://www.christianitytoday.com/ct/2001/107/36.0.html| accessdate = 2006-07-21 }}</ref>。2005年，据传一些西北部村庄的更正教徒受到压力放弃信仰，不过这些报告已经比以往减少<ref name="liberty">{{cite web |author=| title =Annual Report on International Religious Freedom for 2005 - Vietnam | work = U.S. Department of State| date = 2005-06-30| url = http://www.unhcr.org/cgi-bin/texis/vtx/rsd/rsddocview.html?tbl=RSDCOI&id=437c9cdd2&count=0| accessdate = 2006-07-21 }}</ref>。未经批准的教会聚会通常会被取缔，信徒会被拘留。2001年4月，政府正式承认了越南南部福音教会<ref name="state">{{cite web | title =Vietnam | work = US State Department| date = 2002-09-07| url = http://www.state.gov/g/drl/rls/irf/2002/13916.htm| accessdate = 2006-07-21 }}</ref>。2005年，数百个在2001年下令关闭的家庭教会，已经允许重新开放<ref name="liberty">{{cite web |author=| title =Annual Report on International Religious Freedom for 2005 - Vietnam | work = U.S. Department of State| date = 2005-06-30| url = http://www.unhcr.org/cgi-bin/texis/vtx/rsd/rsddocview.html?tbl=RSDCOI&id=437c9cdd2&count=0| accessdate = 2006-07-21 }}</ref> 100多名难民在抗议没收土地和缺乏宗教自由遭到镇压后逃往柬埔寨<ref>{{cite web |author=| title =Minorities fleeing Vietnam to Cambodia | work = BBC| date = 2001-05-04| url = http://news.bbc.co.uk/2/hi/asia-pacific/1312730.stm| accessdate = 2006-07-21 }}</ref> 。2001年，一座当地基督徒团体请求收回的建于1936年的古老的更正教堂遭到拆毁<ref>{{cite web |author=Name Hidden| title =Vietnam's Hidden Tragedy | work = Christianity Today| date = 2002-09-09| url = http://www.christianitytoday.com/ct/2002/010/4.62.html| accessdate = 2006-07-21 }}</ref>。至少54人仍然因信仰原因被关押，包括一些更正教徒<ref>{{cite web| author = | title = RELIGIOUS FREEDOM IN VIETNAM| work = 美国之音| date = 2004-09-27| url = http://www.voanews.com/uspolicy/archive/2004-09/a-2004-09-27-5-1.cfm| accessdate = 2006-07-21| deadurl = yes| archiveurl = https://web.archive.org/web/20071128011943/http://www.voanews.com/uspolicy/archive/2004-09/a-2004-09-27-5-1.cfm| archivedate = 2007-11-28}}</ref>。新生命团契，寻求官方承认已经8年，2005年被拒绝在[[胡志明市|胡志明市]]聚会<ref>{{cite web |author=| title =Vietnam bans Protestant Group | work = ABC Radio Australia| date = 2005-09-09| url = http://www.abc.net.au/ra/news/stories/s1457325.htm| accessdate = 2006-07-21 }}</ref>。一位更正教牧师被迫在精神病院度过12个月<ref>{{cite web |author=| title =Vietnam releases protestant pastor from hospital | work = ABC Radio Australia| date = 2005-09-19| url = http://www.abc.net.au/ra/news/stories/s1463217.htm| accessdate = 2006-07-21 }}</ref>。2007年3月，合法承认的越南福音教会（北方）河内大教堂信徒阮文岱因宣传宗教自由而被捕<ref>{{cite web |author=| title =Encourage the Wife of Imprisoned Vietnamese Lawyer | work = Persecution blog | date = 2007-04-25| url = http://www.persecutionblog.com/2007/04/encourage_the_w.html| accessdate = 2006-04-27 }}</ref>。

2007年10月，[[浸信会|浸信会]]和[[门诺会|门诺会]]得到了河内政府的正式承认，被认为该国的宗教自由有所改进<ref name="ASN">[http://www.asianews.it/index.php?l=en&art=10466&size=A Hanoi officially recognises Baptists and Mennonites, AsiaNews]</ref>。[[浸信会|浸信会]]牧师阮 Thong说，1989年以来，他的教会已经吸引了超过18,400名信徒，在全国23 城市和省份有135个聚会<ref name="ASN">[http://www.asianews.it/index.php?l=en&art=10466&size=A Hanoi officially recognises Baptists and Mennonites, AsiaNews]</ref>。

2007年，制作了一部关于越南地下教会以及继续受到地方政府的迫害的电影<ref>[http://www.persecution.com/news/index.cfm?action=fullstory&newsID=526 The Voice of the Martyrs News] {{webarchive|url=https://web.archive.org/web/20080612165559/http://www.persecution.com/news/index.cfm?action=fullstory&newsID=526 |date=2008-06-12 }}</ref>。2007年4月，据传一个拒绝否认基督教信仰的Hroi族年轻人在受审时受伤而死<ref>[http://www.christianpersecution.info/news/vietnam-christian-dies-after-torture-for-refusing-to-recant-faith-reports/ BosNewsLife News Center]</ref>。

根据2008年估计，许多越南山地部落的基督徒仍被当作“美国间谍”，遭到拷打、监禁和挨饿，尽管政府保证宗教自由<ref>[http://www.christiantoday.com/article/vietnam.war.rumbles.on.christians.branded.the.enemy/17594.htm Christian Today Magazine]</ref>。
