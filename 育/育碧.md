'''育碧娛樂公司'''（{{lang-en|'''Ubisoft Entertainment SA'''}}），2003年前名為「'''育碧娛樂軟體'''」（{{lang|en|'''Ubi Soft Entertainment Software'''}}），簡稱「'''育碧'''」（{{lang|en|'''Ubisoft'''}}），是一家[[總部|總部]]設於[[法國|法國]][[雷恩_(伊勒-维莱讷省)|雷恩]]<ref name="Ubisoftsiège">{{cite web|url= http://www.societe.com/societe/ubisoft-entertainment-335186094.html |title= Siège social d'Ubisoft Entertainment S.A. |website=societe.com|accessdate=2015-11-10}}</ref>的[[電子遊戲|電子遊戲]]開發商和發行商，1986年成立。育碧在全世界擁有23間[[遊戲開發|遊戲開發]][[工作室|工作室]]，是2008年[[歐洲|歐洲]]第三大[[獨立遊戲|獨立遊戲]]開發商、[[北美|北美]]第四大獨立遊戲出版商。

== 历史 ==
* 1986年，法國Guillemot五兄弟開創了育碧。
* 1991年在英国、德国、美國等重要國際市場建立了銷售部。 
* 1992年到1996年，在法國和羅馬尼亞首次建立了自主遊戲工作室。在日本、意大利、澳大利亞建立了銷售分部，并在1996年成功並入巴黎二板股票交易市場。
* 1997年到1999年，在中國、加拿大、西班牙、意大利、摩洛哥，以及後來在Morocco和Annecy（法國）成立了工作室。
* 2000年到2001年，成功收購了2家發行公司和幾間遊戲工作室： Redstorm Entertainment（美國），以製作Tom Clancy系列遊戲而聞名；Blue Byte Software（德国）以及有名的《[[工人物语|工人物语]]》，《[[战斗之岛|战斗之岛]]》系列遊戲 ；美國Learning公司旗下的工作室，以及包括《[[波斯王子系列|波斯王子]]》、《[[象棋大师|象棋大师]]》、《[[迷霧之島|-{zh-hant:迷霧之島;zh-cn:神秘岛}-]]》。
* 2002年到2004年，在韓國、芬蘭、加拿大、瑞士建立了發行部，收購了法國的Tiwak工作室。公布了新的LOGO，宣布遊戲銷售量突破一億套。
* 2004年到2005年，進軍[[好萊坞|好萊坞]]。
* 2007年，育碧成為全球（除亞洲外）排名第三的獨立遊戲經銷商；在保加利亞、中國、新加坡、印度和烏克蘭成立新的工作室；在墨西哥和波蘭新建經銷分公司；成功收購Reflections Interactive（英國）、Massive Entertainment（瑞典），以及日本的一些開發工作室；2007年11月發布[[刺客教條|刺客教條]]，該遊戲作品成為美國和英國有史以來最暢銷的視訊遊戲品牌之一；成功收購[[孤岛惊魂_(游戏)|極地戰嚎]]和[[纪元系列|大航海世紀系列]]遊戲相關版權，之後獲得Tom Clancy系列视频游戏相关的所有知识产权和相关产品。

== 评价 ==
虽然育碧的游戏制作质量多为赞许，但也有一些运营或游戏设计上的问题备受批评或讽刺，例如：网络连线游戏服务器有时会出现故障或连线缓慢，而被讽刺为“土豆服务器”（在《湯姆克蘭西：全境封鎖》初始运营时就出现服务器访问问题，<ref>{{Cite web |url=http://games.sina.com.cn/t/n/2016-03-08/fxqaffy3756338.shtml |title=《全境封锁》首发日遇服务器故障 玩家连线困难_电视游戏-新闻_新浪游戏_新浪网 |accessdate=2019-04-18 |work=games.sina.com.cn}}</ref>而被讽刺为“全境排队”）；或游戏中经常存在一些匪夷所思的BUG，而被讽刺为“买BUG送游戏”（在[[刺客信条|刺客信条]]系列中出现角色模型突然扭曲而面目狰狞，或地图模型建构问题而导致游戏角色卡在模型中；《[[彩虹六号：围攻|彩虹六号：围攻]]》也存在地图模型建构问题而导致玩家卡进地图模型内非正常空间而死亡，等）。

对于批评，育碧也尝试着力改善，例如收购i3D.net公司以改善服务器质量<ref>{{Cite web |url=http://www.donews.com/news/detail/3/3029967.html |title=服务器公司i3D.net被育碧收购 - iDoNews |accessdate=2019-04-18 |work=www.donews.com}}</ref><ref>{{Cite web |url=http://www.joyyang.com/keji/20190418/04187173.html |title=彩虹六号：UBI联合i3D.NET改善服务器质量_骄阳网 |accessdate=2019-04-18 |work=www.joyyang.com}}</ref>，游戏发售后仍然提供修复BUG的补丁<ref>{{Cite web |url=http://www.joyyang.com/news/20190418/04187135.html |title=彩虹六号Y4S1.3补丁在主机板与PC板中先后更新_骄阳网 |accessdate=2019-04-18 |work=www.joyyang.com}}</ref>等。
