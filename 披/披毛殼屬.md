'''披毛殼屬'''（[[學名|學名]]：{{lang|la|''Calvapilosa''}}，意為「毛茸茸的頭皮」）為[[雙神經亞門|雙神經類]][[軟體動物|軟體動物]]的基幹成員，屬於[[似楯殼螺目|似楯殼螺目]]的[[棱管殼科|棱管殼科]]，生存於[[奧陶紀|奧陶紀]]早期的{{link-en|特馬豆克階|Tremadocian}}，化石發現於[[摩洛哥|摩洛哥]]東部<ref name = Vinther>{{ cite journal | author = Vinther, J.; Parry, L.; Briggs, D. == 特徵描述 ==
成體全長最大可達 12 公分，亞成體化石全長小於 2 公分<ref name = theguardian/>。外型大致呈長橢圓狀，約在頭部的位置上方有一個明顯的大型殼板（anterior shell plate），但大部分表面都被[[外套膜|外套膜]]和{{link-en|骨片|Sclerites}}覆蓋。背部生有細小而密集的體刺。具有[[齒舌|齒舌]]構造，分為鉤狀（uncinal teeth）及外側的齒舌（lateral teeth）<ref name = Vinther/>。

== 命名與詞源 ==
披毛殼的屬名由兩個字結合衍化而來，分別是指[[頭皮|頭皮]]的「''calva''」，以及「''pilosus''」（意為「多毛的」），兩個字都是來自[[拉丁文|拉丁文]]，形容這種身上密布細針狀物的原始軟體動物。種名則是為了感謝在[[美國|美國]][[耶魯大學|耶魯大學]]的館藏標本中「發現」了這種奇特生物化石的[[古生物學家|古生物學家]] Björn Kröger，故取其姓氏命名<ref name = Vinther/>。

== 發現歷史 ==
目前已知的披毛殼屬化石共有七具，它們都是由化石收藏家穆罕默德·班莫拉（Mohamed 'Ou Said' Ben Moula）在2000年代後期於[[摩洛哥|摩洛哥]]東部的{{link-en|札戈拉|Zagora, Morocco}}一帶，一個被稱為{{link-en|費札瓦塔組|Fezouata Formation}}（Fezouata Formation）的地層中發現的，該地層是典型的[[伯吉斯頁岩|伯吉斯頁岩]]型頁岩層（Burgess shale-type deposits），在[[奧陶紀|奧陶紀]]時期曾擁有[[生物相|生物相]]豐富的費札瓦塔[[生物群|生物群]]（Fezouata Biota），也因此班莫拉在這裡發現了數百件，甚至數千件已[[化石#化石化作用|化石化]]的[[軟組織|軟組織]]標本。而班莫拉和[[美國|美國]][[耶魯大學|耶魯大學]]的[[古生物學家|古生物學家]]有工作上的關係，因此這些科學家有幸得以將這些化石標本運回耶魯大學進行進一步的研究<ref name = Livescience>{{ cite news | url = http://www.livescience.com/57768-ancient-spiky-slug-explains-mollusk-evolution.html | author = Laura Geggel | work = Live Science | title = "478-Million-Year-Old Spiky Slug Solves Long-Held Mollusk Mystery" | date = February 6, 2017 }}</ref>。

班莫拉提供的化石中，共有七具披毛殼化石，但當時它們並沒有立即被注意到，而是如上所述，是後來才在耶魯大學眾多的館藏標本中被發現的。在這七具披毛殼化石中，有兩具分別為成體和[[亞成體|亞成體]]的標本被完整的保存下來，足以讓科學家仔細研究其生理上的構造和細部特徵。從化石中明顯能看見披毛殼的頭部具有類似頭盔的殼，顯示早期的軟體動物可能就已經具有殼的構造<ref name = Livescience/>。

另一個引起不少科學家關注的特徵是，部分披毛殼標本的[[齒舌|齒舌]]構造被完整保存下來，這向來是軟體動物特有且普遍的器官，上面有超過125排的「小牙齒」<ref name = theguardian/>，對於披毛殼而言，這些所謂的「小牙齒」能用來翻找食物，像是附著在岩石上的海藻。佩瑞表示，披毛殼的齒舌原本是位在其頭部的殼下面的，但殼已在化石形成的過程中消失，可以看見這些「小牙齒」被印壓在殼上<ref name = Livescience/>。
