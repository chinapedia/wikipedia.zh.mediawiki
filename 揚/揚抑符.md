'''揚抑符'''（{{lang-en|'''circumflex'''}}；{{IPA|◌̂}}）是某些文字[[羅馬化|羅馬化]]和[[音譯|音譯]]方案所使用的[[拉丁字母|拉丁字母]]和[[希臘字母|希臘字母]][[附加符號|附加符號]]。其英文名稱來自[[拉丁文|拉丁文]] ''circumflexus''「繞過」 – 譯自希臘文 {{lang|el|περισπωμένη}} (''{{lang|en|{{tsl|en|perispomenon|perispōménē}}}}'')。拉丁文字的揚抑符是個{{tsl|en|Chevron (insignia)|山形}}符號 ( '''{{lang|la|ˆ}}''' )、而希臘文揚抑符則可以是[[波浪號|波浪號]] ( '''{{lang|la|˜}}''' ) 或[[反短音符|反轉的短音符]] (   ̑ )。

在[[漢語拼音|漢語拼音]]中，{{lang|la|ê}} 用來代表獨立的 {{IPAslink|ɛ}}，通常用於「欸、誒」等[[歎詞|歎詞]]。

在[[英文|英文]]，揚抑符就如其他附加符號一樣用於[[外來語|外來語]]，例如 ''{{lang|en|[[wikt:rôle|rôle]]}}''。

此符號在[[數學|數學]]也有使用，稱為 '''hat'''（帽子）或 '''roof'''（屋頂）。

==用途==
=== 音高 ===
{{See also|古希腊语}}
在语言学，扬抑符用来表示希腊语等语言的音高。

In a similar vein, the circumflex is today used to mark [[tone_contour|tone contour]] in the [[International_Phonetic_Alphabet|International Phonetic Alphabet]].

===數學===
在數學中，揚抑符用來修飾變量名稱；讀如「hat（帽子）」，例如 '''{{lang|la|î}}''' 就是「i hat」。''ƒ'' 函數的[[傅立葉變換|傅立葉變換]]通常寫作 <math>\hat f</math>。 

在矢量理論中，字母加上帽子代表[[單位矢量|單位矢量]]（[[量_(數學)|大小]]是 1 的[[矢量|矢量]]）。例如 '''î''' 代表在[[笛卡儿坐标系|x-軸]]方向的單位矢量。

===音樂===
在[[樂理|樂理]]和[[音樂學|音樂學]]，在數字上的揚抑符用來指代特定[[音級|音級]]。

在[[樂譜|樂譜]]，在音符上的{{tsl|en|Chevron (insignia)|山形}}符號代表[[加強音|加強音]]，是特別形式的加強或 {{tsl|en|accent (music)|accent}}。

== 在數碼字符集的揚抑符 ==
[[ISO-8859-1|ISO-8859-1]] 收羅了一堆[[预组字符|预组字符]]如 ''{{lang|la|Â/â}}''、''{{lang|la|Ê/ê}}''、''{{lang|la|Î/î}}''、''{{lang|la|Ô/ô}}'' 和 ''{{lang|la|Û/û}}''（皆包含揚抑符），[[Unicode|Unicode]] 收羅的更多。此外 Unicode 還有 {{unichar|0302|Combining circumflex accent|cwith=◌|nlink=組合字元}}，可將揚抑符附加在任何字母。

由於歷史原因，有個源自 [[ASCII|ASCII]]，現在又收納進 Unicode 的較大字符 {{unichar|005E|circumflex accent}}，通常稱為[[脱字符|脱字符]]（caret）。但現在已不適宜用作附加符號，因其會移格。另一個較小的 Unicode 移格字符是 {{unichar|02C6|modifier letter circumflex|link=Modifier letter|use=IPA}}，主要用作[[語音學|語音學]]符號 – 或用來表示一個獨立的附加符號。
