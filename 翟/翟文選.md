'''翟文选'''（{{bd|1878年||1950年|4月23日|catIdx=Z翟}}） 字'''熙人'''，[[吉林省|吉林将軍管轄区]][[阿城区|阿勒楚喀副都統管轄区]][[双城市|双城堡]]人<ref>《双城县志》作此。現在属于[[黑龙江省|黑龙江省]]，清末属于[[吉林将軍|吉林将軍]]管轄。徐友春主編《民国人物大辞典 増訂版》则作[[伊通满族自治县|伊通县]]人。</ref>，[[清朝|清朝]]及[[中华民国|中华民国]]政治人物。<ref name=zhang>张学继，张作霖幕府与幕僚，浙江文艺出版社，2011年</ref><ref name=xu>{{Cite book|author = 徐友春主編|title = 民国人物大辞典 増訂版|year = 2007|publisher = 河北人民出版社|isbn = 978-7-202-03014-1}}</ref><ref name=heb>[http://218.10.232.41:8080/was40/detail?record=398&channelid=39963 《双城县志》] [[哈尔滨|哈尔滨]]地情網（哈尔滨市人民政府地方志办公室网站）</ref><ref name=liu>{{Cite book|author = 劉寿林等編|title = 民国職官年表|year = 1995|publisher = [[中華書局|中華書局]]|isbn = 7-101-01320-1}}</ref>

==生平==
翟文选是清朝1901年、1902年（[[光緒|光緒]]27年、28年）辛丑、壬寅併科[[举人|举人]]。清朝末年，曾任[[黑龙江将军|黑龙江将军]]府全省文案处提调官、[[安达厅|安达厅]]抚民通判、[[呼伦厅|呼伦厅]]抚民同知等职务。<ref name=zhang/><ref name=xu/><ref name=heb/>

[[中華民国|中華民国]]成立後的1913年，翟文选任黑龙江省会警察厅长。此后，成为[[张作霖|张作霖]]领导的[[奉系|奉系]]成员。1917年，当选为[[第二次中华民国临时参议院|第二次中华民国临时参议院]]议员。1918年，当选为[[安福国会|安福国会]]参议院议员。1920年9月，翟文选被[[北洋政府|北京政府]]任命为东三省盐运使，任内解决了东三省盐政混乱、吃盐困难问题。1928年（民国17年），任東三省保安会委員。<ref name=zhang/><ref name=xu/><ref name=heb/><ref name=liu/>

1928年7月，[[张学良|张学良]]继承其亡父[[张作霖|张作霖]]的职位后，任命翟文选为[[奉天省|奉天省]]省长，同时兼任东三省交通委员会副委员长。1928年12月31日，[[东北易帜|东北易帜]]后，翟文选被[[南京国民政府|南京国民政府]]任命为奉天省政府委员兼主席。1929年1月29日，奉天省改为辽宁省。1929年（民国18年）1月，翟文选兼任東北政務委員会委員、国民政府首都建設委員会委員。1930年1月，翟文选托病辞去辽宁省主席一职，由[[臧式毅|臧式毅]]继任辽宁省主席。後来，曾任[[万国道德总会|万国道德总会]]理事。<ref>郭卿友主编. 中华民国时期军政职官志. 甘肃人民出版社. 1990年</ref><ref name=zhang/><ref name=xu/><ref name=heb/><ref name=liu/>

1931年[[九一八事变|九一八事变]]后，翟文选拒绝了[[关东军|关东军]]提出的请其参加[[满洲国|满洲国]]的邀请，举家迁往天津，从此远离政坛。後来，迁居[[北京市|北平]]。晚年，翟文选笃信佛教，在家中修行。<ref name=zhang/><ref name=xu/><ref name=heb/><ref name=liu/>

1950年4月23日，翟文选病逝，享年73岁。<ref name=xu/><ref name=heb/><ref name=liu/>1950年8月安葬在北京[[东北义园|东北义园]]。<ref>冯其利，东北义园名人补遗，北京档案史料2007年第4期</ref>
