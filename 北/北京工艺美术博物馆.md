'''北京工艺美术博物馆'''，是北京专业的工艺美术博物馆，隶属[[北京工美集团有限责任公司|北京工美集团有限责任公司]]<ref name=llzh>李江、吕林雪、张威，当代工艺美术品的新媒体展示及设计思考，湖南社会科学2014, 32:116-117</ref>。

==简介==
1987年8月29日，北京工艺美术博物馆在[[北京市|北京市]][[东城区_(北京市)|东城区]][[沙滩后街|沙滩后街]]30号开馆。该馆是中国第一家专业的工艺美术博物馆<ref>龚伯，北京工艺美术博物馆正式开馆，美术1987年11期</ref>。隶属于1980年经[[北京市人民政府|北京市人民政府]]批准成立的北京市工艺美术品总公司（1993年改组为北京工美集团总公司，2001年2月成立北京工美集团有限责任公司）<ref>{{Cite web |url = http://sdjsb.bjd.com.cn/html/2010-08/30/content_311879.htm |title = 30年积淀 北京工美集团打造未来产业航母 |author =  |date = 2010-08-30 |publisher = 首都建设报 |language =  |accessdate =  |quote =  }}</ref>。后来，该博物馆迁入[[北京工美集团有限责任公司王府井工美大厦|王府井工美大厦]]，面积550平方米<ref>{{Cite web|url =http://www.beijing.gov.cn/rwbj/bwgzxfw/bwgfl/whys/bjgymsbwg/|title =北京工艺美术博物馆|author =|date =|publisher =首都之窗|language =|accessdate =2017-02-23|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20170224054526/http://www.beijing.gov.cn/rwbj/bwgzxfw/bwgfl/whys/bjgymsbwg/|archivedate =2017-02-24}}</ref>。2012年，新建落成了[[奥林匹克公园_(北京)|奥林匹克公园]]主馆、德胜门工美大厦分馆<ref name=zoujin> {{Cite book | author =  
北京市文物局、首都博物馆联盟 编| title =走进博物馆——北京地区博物馆大全  | location =  | publisher =北京出版社  | date = 2013年 | pages = 157 | ISBN =  | accessdate =  | url =  | language =  | quote =  }} </ref><ref>{{Cite web |url =http://www.chinanews.com/cul/2012/09-12/4178428.shtml  |title =北京将打造博物馆中心区 重点抓"一轴一线"建设  |author =  |date = 2012-09-12 |publisher = 中国新闻网 |language =  |accessdate =  |quote =  }}</ref>。原王府井工美大厦的旧馆改称王府井工美大厦分馆<ref>{{Cite web|url =http://bjwb.bjd.com.cn/html/2016-05/18/content_35382.htm|title =北京的特色博物馆|author =|date =2016-05-18|publisher =北京晚报|language =|accessdate =|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20170224052857/http://bjwb.bjd.com.cn/html/2016-05/18/content_35382.htm|archivedate =2017-02-24}}</ref>。2013年，位于东四环[[红领巾桥|红领巾桥]]西南的北京城工美大厦开业，其内的北京工艺美术博物馆北京城分馆同时开馆，并举办“北京工艺美术博物馆馆藏精品展”<ref>{{Cite web |url = http://news.163.com/13/0928/02/99R0K06F00014AED.html |title = 工美 红领巾桥新店开门迎客 |author =  |date = 2013-09-28 |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url =http://fashion.163.com/13/0925/17/99KRRKDI00264MK3.html  |title = 北京工艺美术博物馆北京城分馆举行开馆展览 |author =  |date =2013-09-25 |publisher =网易  |language =  |accessdate =  |quote =  }}</ref>。2016年12月21日，经过半年升级改造的奥林匹克公园主馆正式开放，主馆以展示北京传统工艺美术“四大名旦”即牙雕、玉雕、景泰蓝、雕漆工艺品为主<ref>{{Cite web |url = http://www.bj.xinhuanet.com/bjyw/2016-12/22/c_1120165298.htm |title = 三千余件精品齐聚 北京工艺美术博物馆新馆开馆 |author =  |date = 2016-12-22 |publisher =新华网  |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://news.xinhuanet.com/politics/2017-01/10/c_129438655.htm |title = 北京工艺美术博物馆新馆落成 |author =  |date = 2017-01-10 |publisher = 新华网 |language =  |accessdate =  |quote =  }}</ref>。

北京工艺美术博物馆现设有1个主馆，3个分馆（各场馆均兼有商业销售功能）：
*[[奥林匹克公园_(北京)|奥林匹克公园]]主馆：位于北京市[[朝阳区_(北京市)|朝阳区]][[天辰东路|天辰东路]]8-10-12新奥工美大厦。主要展出[[中华人民共和国|中华人民共和国]]成立以来北京地区工艺美术大师的作品<ref name=llzh/>。
*德胜门分馆：位于北京市[[西城区|西城区]][[安德路|安德路]]83号德胜门工美大厦4层。主要展出青铜、瓷器<ref name=llzh/>。
*北京城分馆：位于北京市朝阳区东四环中路37号北京城工美大厦，主要展出民间工艺品<ref name=llzh/>。
*王府井分馆：位于北京市东城区[[王府井大街|王府井大街]]200号[[北京工美集团有限责任公司王府井工美大厦|王府井工美大厦]]，主要展出北京传统工艺美术“四大名旦”即玉雕、牙雕、景泰蓝、雕漆等<ref name=llzh/>。
