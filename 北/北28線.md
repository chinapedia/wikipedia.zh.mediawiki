'''北28線 萬里－北市縣界'''，是位於[[新北市|新北市]]的一條鄉道，東起[[新北市|新北市]][[萬里區|萬里區]]萬里，南至[[新北市|新北市]][[汐止區|汐止區]]北市縣界，全長24.303公里。{{r|thb}}有支線北28-1線、北28-2線。

== 路線說明 ==

本線為[[新北市|新北市]]最長之鄉道，也為[[新北市鄉道|新北市鄉道]]中唯一擁有兩條支線之鄉道（北28-1、北28-2）。目前原有之24.5公里及25公里的里程牌雖然牌面已撤除，但原先設置牌面之鋁杆仍存。{{r|buslover}}

;[[新北市|新北市]]
*[[萬里區|萬里區]]：'''萬里''' 0.0k（起點，[[File:TW_PHW2.svg|23px]][[台2線|台2線]]舊線岔路）→ 萬崁公路 → '''中幅'''（[[File:TW_THWtp28-1.png|30px]]北28-1線岔路） → '''崁腳'''（[[File:TW_THWtp28-2.png|30px]]岔路） → '''溪底'''（[[File:TW_THWtp28-2.png|30px]]北28-2線岔路） → 萬崁路 → 萬溪右線產業道路 → （區界）
*[[汐止區|汐止區]]：（區界）→ 五指山產業道路 → 長青路 →'''北市縣界'''24.303k（終點）
