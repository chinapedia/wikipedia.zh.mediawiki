'''北洛磯山脈地區自治市'''（{{lang-en|Northern Rockies Regional Municipality}}）是[[加拿大|加拿大]][[卑詩省|卑詩省]]東北部内陸地區的一個[[不列顛哥倫比亞省行政區劃|行政區]]，於2009年2月6日成立<ref>{{Cite web |url=http://www.northernrockies.ca/EN/main/visitors/about-municipality.html |title=About the Municipality |accessdate=2014-01-29 |work=北洛磯山脈地區自治市 |deadurl=yes |archiveurl=https://web.archive.org/web/20151208200241/http://www.northernrockies.ca/EN/main/visitors/about-municipality.html |archivedate=2015-12-08 }}</ref>，行政中心設於納爾遜堡（{{lang|en|Fort Nelson}}）。據2011年人口普查，區内共有5290名居民<ref>{{Cite web |url=http://www12.statcan.gc.ca/census-recensement/news-nouvelles/corr/cgen004-eng.cfm |title=Corrections and updates: Population and dwelling count amendments, 2011 Census |accessdate=2013-08-13 |work=[[Statistics_Canada|Statistics Canada]] |date=2014-01-08 }}</ref>，總面積85,014.52平方公里<ref name="2011census">{{Cite web |url=http://www12.statcan.gc.ca/census-recensement/2011/dp-pd/prof/details/page.cfm?Lang=E&Geo1=CD&Code1=5953&Geo2=PR&Code2=59&Data=Count&SearchText=fraser%20fort%20george&SearchType=Begins&SearchPR=59&B1=All&Custom=&TABID=1 |title=Census Profile - Census Division |accessdate=2014-01-29 |work=[[Statistics_Canada|Statistics Canada]] |date=2014-01-08 }}</ref>，佔全省面積約9%。

卑詩省政府從1965年起着手整理省內的地方行政區劃，並於1967年10月31日在省內東北部設立和平河-利亞德地區（{{lang|en|Peace River-Liard Regional District}}）<ref name="GeoBC">{{Cite web |url=http://apps.gov.bc.ca/pub/bcgnws/names/54454.html |title=BC Geographical Names: Northern Rockies Regional District |accessdate=2014-01-29 |work=GeoBC }}</ref>。該地區於1987年10月22日分拆成納爾遜堡-利亞德地區（{{lang|en|Fort Nelson-Liard Regional District}}）以及[[和平河地區|和平河地區]]<ref name="GeoBC"/>。前者於1999年3月26日改稱北洛磯山脈地區（{{lang|en|Northern Rockies Regional District}}），再於2009年2月6日與其下轄的納爾遜堡鎮合併成新的北洛磯山脈地區自治市，成為卑詩省内首個單層次行政區。此自治市的管轄範圍大致與前北洛磯山脈地區相同，但不包括後者的原住民保留區<ref name="GeoBC"/>。
