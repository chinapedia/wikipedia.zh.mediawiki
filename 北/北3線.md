'''北3線 水源里－吳仔厝'''，是位於[[新北市|新北市]]的一條鄉道，北起[[新北市|新北市]][[淡水區|淡水區]]水源里，南至[[新北市|新北市]]淡水區吳仔厝，全長6.139公里。{{r|thb}}

== 路線說明 ==
{{expand|time=2011-01-31}}
;'''[[新北市|新北市]]'''
*[[淡水區|淡水區]]：'''水源里'''（起點，[[File:TW_CHW101.svg|23px]][[縣道101號|101線]]岔路）→ 小坪頂路 → '''糞箕湖'''（[[File:TW_THWtp4.png|30px]][[北4線|北4線]]岔路）→ '''小坪頂'''（[[File:TW_THWtp2.png|30px]][[北2線|北2線]]岔路）→ '''吳仔厝'''（終點，[[新北市|新北市]]、[[台北市|台北市]]市界）
**早期路線：進入台北市[[北投區|北投區]] → 稻香路 → '''復興崗'''（早期終點，中央北路口）
