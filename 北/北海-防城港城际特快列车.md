'''北海-防城港城际特快列车'''是[[中国铁路|中国铁路]]运行于[[廣西|廣西]][[北海市|北海市]]至[[防城港|防城港]]之间的三对特快旅客列车，车次分别为T8712/8713次、T8714/8711次列车，T8716/8717次、T8718/8715次列车，和T8720/8721次、T8722/8719次列车，由[[广西沿海铁路股份有限公司|广西沿海铁路股份有限公司]]负责客运任务。列车使用[[中国铁路25T型客车|中国铁路25T型客车]]，沿[[钦北高速铁路|钦北高速铁路]]、[[钦州-钦州东联络线|钦州-钦州东联络线]]、[[钦防铁路|钦防铁路]]运行，途经[[北海市|北海]]、[[合浦县|合浦县]]、[[钦州|钦州]]、[[防城港|防城港]]。全程139公里。

该列车在开行初期，只有两对四趟车次运行，且没有停靠合浦站的车次（2015年11月1日起，T8712/8713次、T8718/8715次列车加停合浦站）。

后调整时刻，加开一对列车，启用车次T8720/8721次、T8722/8719次列车。

该列车目前的运营方式为三对六趟模式：其中在上午时段，北海站至防城港北站区间，运行1小时24分，使用车次为T8712/8713次（本车次停靠合浦站）；防城港北站至北海站区间运行1小时19分，使用车次为T8714/8711次。在中午-下午时段，[[北海站_(广西)|北海站]]至[[防城港北站|防城港北站]]区间，运行1小时19分，使用车次为T8720/8721次；防城港北站至北海站区间运行1小时21分，使用车次为T8722/8719次。在下午-晚间时段，[[北海站_(广西)|北海站]]至[[防城港北站|防城港北站]]区间，运行1小时19分，使用车次为T8716/8717次；防城港北站至北海站区间运行1小时24分，使用车次为T8718/8715次（本车次停靠合浦站）。

== 车底编组 ==
在开通运营初期，北海-防城港城际特快列车采用9节25T型硬座车编组。但由于宣传力度不够，以及客流不旺等原因，目前该城际特快列车的编组缩短为5节25T型硬座车编组。
== 机车交路 ==
自开通运营以来，北海-防城港城际特快列车全程采用由[[南宁铁路局|南宁铁路局]][[柳州机务段|柳州机务段]]租借给[[广西沿海铁路股份有限公司|广西沿海铁路股份有限公司]]的1组[[东风11G型柴油机车|东风11G型柴油机车]]，机车可直接[[列车供电|向列车提供电能]]，用于列车内空调和水炉等设备需要，因此列车不需要再额外加挂一节[[发电车|空调发电车]]。这也是广西境内第二列使用东风11G型柴油机车牵引的特快列车<ref>广西境内第一列使用东风11G型柴油机车牵引的特快列车为[[T285/286次列车|T285/286次列车]]。在开行初期，T285/286次列车的柳州到南宁区间经由[[湘桂铁路|湘桂铁路]]运行，同样使用配属于南宁铁路局柳州机务段的[[东风11G型柴油机车|东风11G型柴油机车]]牵引（参见这个视频：[http://v.youku.com/v_show/id_XNjU0NzAyMzMy.html 宁局DF11G牵引T286次通过东站便线]），后在2014年1月中旬改由[[柳南城际铁路|柳南城际铁路]]运行，成为首列运行于柳南城际铁路的普速列车。</ref>。

2015年4月，因应钦州东至钦州站间线路电气化改造完成，北防特快列车改用配属沿海铁路公司的[[和谐3D型电力机车|和谐3D型电力机车]]（HXD3D-7001、7002）牵引，是華南地區僅有的兩台HXD3D。<ref>{{cite news|url=http://www.gx.xinhuanet.com/dtzx/fcg/2015-05/01/c_1115131573.htm|title=广西沿海城际旅客列车全部实现电力牵引|publisher=[[防城港日报|防城港日报]]|date=2015-05-01|accessdate=2015-05-02|deadurl=yes|archiveurl=https://web.archive.org/web/20150729064940/http://www.gx.xinhuanet.com/dtzx/fcg/2015-05/01/c_1115131573.htm|archivedate=2015-07-29}}</ref>

== 评价 ==
北海-防城港城际特快列车的开通运营，将北海、钦州、防城港三市连为一线，大大方便广大市民和游客的出行、旅游，并且促进北部湾地区各城市旅游业的联动发展，进一步加快北部湾地区的钦、北、防三市“同城化”发展进程<ref name="测试" />。

相比较同一路线运行的动车组列车以及客运班车而言，该城际特快列车全程票价仅为21.5元[[人民币|人民币]]，占据票价优势；而单单与在同一路线运行的客运班车相比，该城际特快列车更具有运行时间上的优势<ref>[http://news.gxnews.com.cn/staticpages/20140530/newgx53880248-10405601.shtml 防城港北至北海火车票价仅为21.5元_广西新闻网_2014-05-30_12:00:00]</ref>。
