《'''北京青年报'''》是[[中国共产主义青年团北京市委员会|中国共青团北京市委]]的机关报，于1949年创刊，由[[北京青年报社|北京青年报社]]编辑出版，是北京地区发行量最大的报刊之一，也是北京地区最受欢迎的都市类报纸。<ref>［http://www.bjqnbw.com/bjqnb.html {{Wayback|url=http://www.bjqnbw.com/bjqnb.html |date=20131005095613 }} 北京青年报］</ref> 自创刊以来，它历经三次停刊，三次复刊，1981年第三次复刊以来，“北京青年报”有限公司进入稳定发展阶段。

== 发展历程 ==
1949年，[[北平和平解放|北平和平解放]]后不久，共产党认识到青年在夺取全国胜利的关键作用，想要创办一份联系青年教育青年的报纸。战争年代缺乏独立创报的条件，于是北京市委决定在由《[[人民日报|人民日报]]》（北平版）改版的《北京解放报》第五版上以专版形式创办《北京青年》。<ref>［http://media.people.com.cn/GB/10290042.html］{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref> 3月21日，《北京青年》作为北平团市工委的机关刊物正式创刊。

此后三十多年间，《北京青年报》历经风雨，由于向筹建中的其它报纸输送人力物力支援，及自然灾害等原因，三次停刊。

1980年代十一届三中全会后拨乱反正，1980年中共北京市委常委会批准团市委对恢复《北京青年报》的申请。1981年7月3日第三次复刊时，《北京青年报》只是一个处级的小报，地位很低。当时，《北京青年报》是只有26万元开办费，四开四版的周报，发行量很小，邮局订阅量也仅为29000份。复刊初期条件艰苦，在一个半地下室旧库房办公。

经历了1980年代的准备阶段，进入到1990年代，《北京青年报》也迎来了起飞时期。1990年底，《北京青年报》提出以日报为目标的五年规划作为发展纲要。<ref>［{{cite web |url=http://lib.cnki.net/cjfd/CMGC200409006.html |title=存档副本 |accessdate=2013-10-06 |deadurl=yes |archiveurl=https://web.archive.org/web/20150402172600/http://lib.cnki.net/cjfd/CMGC200409006.html |archivedate=2015-04-02 }} 解读《北京青年报》的守业术］</ref>1992年邓小平南巡讲话，提出发展市场经济，进行改革开放的发展路线。《北京青年报》迎合这一历史机遇，尝试市场运作方式，先后创办了《青年周末》和《新闻周刊》。《青年周末》四刊即发行10万份，吸引众多广告商，被首都报界誉为“周末四杰”，也常被读者与《南方周末》相提并论。《青年周末》的发行带动了《北京青年报》从机关报向市场报的转变。1994年7月，《北京青年》正式改为日报发行。1995年，《北京青年报》适应中国房地产业，IT等新兴产业的发展，在《新闻周刊》开设了“汽车时代”，“广厦时代”，“电脑时代”等专刊。这开创了中国报业屈从广告主的经营模式，使编辑，发行，广告，行政四轮驱动，提高了工作效率，加深了市场化的程度，也创造了巨大的经济效益。1996年，《北京青年报》改为自主合作发行，并成立北京小红帽报社发行服务有限责任公司。1997年起，《北京青年报》成为北京第一家自主合作发行的日报，其经营模式更是得到读者广泛认可，邮发迅速升至十几万份 。<ref>［{{cite web |url=http://www.jhmed.cn/Artical/SearchDetail.aspx?id=31803268 |title=存档副本 |accessdate=2013-10-06 |deadurl=yes |archiveurl=https://web.archive.org/web/20150320145822/http://www.jhmed.cn/Artical/SearchDetail.aspx?id=31803268 |archivedate=2015-03-20 }} 《北京青年报》：与改革开放同路］</ref>

经过几年的平稳发展，进入到21世纪《北京青年报》危机乍现，骨干层老化，内部凝聚力下降，2001年征订量更是下降20％以上，发行量在北京首次跌落30万份。为了应对危机，《北京青年报》开始全面改版，分成了A,B,C三大叠，形成规范整齐的版组，使得报纸内容更加清晰合理，便于读者阅读。报社组织结构也经历了改革，采取以编辑为主导，采编分离的做法。《北京青年报》常务副总编辑孙伟表示，“编辑部的组成结构，对报社的高速运作来说是非常重要的”，“还有一个重要原因，这就是防止采编合一会导致的发一些走后门，以权谋私的关系稿的可能性” . <ref>［《北京青年报：从地下室走向市场》唐元恺《今日中国》］</ref>在全国文化体制改革的北京下，《北京青年报》也在2001年为上市做准备，成立了股份公司，并在2004年12月22日于香港上市，成为中国内地主流传媒企业上市的“第一股”。近几年来，《北京青年报》更是通过投资，收购，兼并等方式成为拥有10报2刊2网及若干下属公司传媒集团，成为名副其实的现代传媒集团 。<ref>［《从报纸到现代传媒集团： 北京青年报对文化体制改革的初步探索 张延平《传媒》］</ref>

== 北京青年报办报追求十个最 ==
《北京青年报》总编辑总结了报社发展的十个“最”的追求 ：<ref>［《北京青年报办报追求十个最》，包灿］</ref>

（一）瞄准最活跃的读者群体
（二）开发最丰厚的新闻资源
（三）佔领最前沿的思想高地
（四）策划最醒目的正面宣传
（五）参与最激烈的新闻拼抢
（六）寻找最热情的报道角度
（七）鼓励最个性的版面创造
（八）讲究最新颖的形象包装
（九）提倡最现代的采访手段
（十）营造最浓郁的竞争氛围
