'''北京使館區'''是在今[[北京市|北京市]]的外国驻[[中国|中国]][[大使馆|大使馆]]分布区，曾位於[[北京内城|北京内城]][[正阳门|正阳门]]内的[[東交民巷|東交民巷]]，现分布在[[北京市|北京市]][[朝阳区_(北京市)|朝阳区]]，分为第一至第四使馆区。

== 简介 ==
东交民巷原名東[[江米|江米]]巷，是老北京城[[米|米]]的集散地。1901年[[清政府|清政府]]签订《北京各國使館界址四至》和《[[辛丑和約|辛丑和約]]》，此巷劃成中國第一個“使館區”。从[[清朝|清朝]]到[[中华民国|中华民国]][[北洋政府|北京政府]]时期，这里一直是中国的使馆区，直到1928年[[北伐战争|北伐战争]]结束，北京政府倒台为止。2001年，[[东交民巷使馆建筑群|东交民巷使馆建筑群]]已被列为[[第五批全国重点文物保护单位|第五批全国重点文物保护单位]]。

1949年[[中华人民共和国|中华人民共和国]]成立之後，再次定都北京。新使館區漸漸形成於北京[[建国门_(北京)|建國門]]外和[[朝陽門|朝陽門]]外，分別稱為“第一使館區”和“第二使館區”<ref>李潜虞：[http://www.cnki.com.cn/Article/CJFDTOTAL-WJXY201305006.htm 试论新中国对外国驻华使馆的管理(1949—1965)]，外交评论(外交学院学报)2013年05期</ref>。第一使館區主要是一些[[社會主義|社會主義]]陣營的國家使館。主要为[[中華人民共和國|中華人民共和國]]建國後，較早承認中华人民共和国并建立[[外交關係|外交關係]]的國家<ref name="bds-cn">{{cite web |url = http://www.bds-cn.com/html/home/shiguanjianshe/ |language = zh-hans |title = 使馆区建设 |publisher = 北京外交人员服务局 |date = |accessdate = }}</ref>。第二使館區又稱為北使館區，由[[周恩来|周恩来]]制定区域并建设于1960年代初期，西方[[發達國家|發達國家]]使館雲集此處，環境幽靜。

1980年，[[北京市城市规划设计研究院|北京市城市规划设计研究院]]按上级指示，将[[亮马河|亮马河]]下游过三环路的区域规划为“第三使馆区”。直到2000年代，才开始有国家在此建设驻华使馆。部分国家如[[美国|美国]]、[[韩国|韩国]]等于2000年代将大使馆迁入本区<ref>{{cite web |url =  http://news.163.com/06/1005/09/2SLL33450001121M.html |title = 中日韩掀起外交旋风 |author =  |date =2006-10-05 |website = 网易 |language =  |accessdate = |quote =  }}</ref>。

==使馆区建设==
[[北京外交人员服务局|北京外交人员服务局]]归口负责北京使馆区建设工作<ref>{{Cite web|url =http://www.china.com.cn/info/2013-05/15/content_28830733.htm|title =北京外交人员服务局亮马桥外交办公大楼结构封顶|author =|date =2013-05-15|publisher =中国网|language =|accessdate =|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20170101095328/http://www.china.com.cn/info/2013-05/15/content_28830733.htm|archivedate =2017-01-01}}</ref><ref name=shiguan>{{Cite web |url =http://www.bds-cn.com/html/home/shiguanjianshe/  |title =使馆区建设  |author =  |date =  |publisher =北京外交人员服务局|language =  |accessdate = 2016-12-31 |quote =  }}</ref>。
*第一使馆区：1950年代末开始开发，1970年代初建设完成。东至[[秀水街|秀水街]]，西至[[北京二环路|东二环]]，南至建国门外大街，北至[[日坛北路|日坛北路]]<ref name=shiguan/>。
*第二使馆区：1960年代开始开发，1980年代建设完成。东至东三环北路，南至[[东直门外大街|东直门外大街]]，西至[[工人体育场北路|工人体育场北路]]，北至[[亮马河南路|亮马河南路]]<ref name=shiguan/>。
*第三使馆区：1987年获得建设用地许可。东至[[亮马桥东街|亮马桥东街]]，西至[[燕莎桥|燕莎桥]]，南至[[亮马桥路|亮马桥路]]，北至[[莱太花卉市场|莱太花卉市场]]<ref name=shiguan/>。
*第四使馆区：东至[[首都机场第二高速公路|首都机场第二高速公路]]，西至[[北小河东路|北小河东路]]，南至[[坝河北滨河路|坝河北滨河路]]，北至规划路，目前正处于开发阶段<ref name=shiguan/>。
