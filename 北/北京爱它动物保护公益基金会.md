'''北京爱它动物保护公益基金会'''（英文名：'''Beijing Loving Animal Foundation'''；简称：'''它基金'''），2011年5月19日创建于[[中国大陆|中国大陆]][[北京市|北京市]]，其创建目的是为保护动物权益。它的性质是[[非公募基金会|非公募基金会]]。<ref>{{cite web |url=http://www.cnr.cn/newscenter/gnxw/201108/t20110829_508432056_2.shtml |title=朱茵出任它基金“爱它大使” |accessdate= 2011年|work=|publisher=中国广播网 |language = zh-hans}}</ref><ref name="smex ">{{cite web|url=http://www.tajijin.cn/about.php|title=关于：北京爱它动物保护公益基金会|accessdate=2011年|work=|publisher=北京爱它动物保护公益基金会官方网|language=zh-hans|deadurl=yes|archiveurl=https://web.archive.org/web/20120327195708/http://www.tajijin.cn/about.php|archivedate=2012-03-27}}</ref>

== 宗旨 ==
*理念：“善待动物、尊重生命”。<ref name="smex " />
*人与自然[[和谐|和谐]]共处。<ref name="smex " />

== 目的 ==
*推动社会[[公民|公民]]认知“善待动物、尊重生命”。<ref name="smex " />
*帮助和资助保护动物[[公益|公益]]项目和[[慈善|慈善]]行动。<ref name="smex " />
*呼吁和推动[[政府|政府]][[立法|立法]]保护动物法律工作进程。<ref name="smex " />
