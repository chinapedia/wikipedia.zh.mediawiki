'''北京老字号'''是由北京老字号协会在[[北京市商务委员会|北京市商务委员会]]指导和支持下认定的北京市的老字号。

在“北京老字号”认定前，[[中华人民共和国商务部|中华人民共和国商务部]]曾分两批认定了北京市117个[[中华老字号|中华老字号]]<ref name=shoubo>{{Cite web |url =http://www.capitalmuseum.org.cn/other/content/2014-10/20/content_61326.htm  |title = 北京老字号简表 |author =  |date = 2014-10-20 |publisher = 首都博物馆 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url =http://www.batb.com.cn:7001/batb/street/news.do?id=1153  |title = 国家商务部认定第一批中华老字号（北京） |author =  |date =2011-04-19  |publisher = 北京老字号协会 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://www.batb.com.cn:7001/batb/street/news.do?id=1154 |title = 国家商务部颁布第二批保护与促进的中华老字号名录（北京） |author =  |date = 2011-04-19 |publisher = 北京老字号协会 |language =  |accessdate =  |quote =  }}</ref>。

北京老字号协会2005年8月成立<ref name=dongdan>{{Cite web |url = http://www.bj.xinhuanet.com/bjyw/2016-02/02/c_1117968527.htm |title =166个品牌获评“北京老字号”   |author =  |date =2016-02-02  |publisher = 新华网 |language =  |accessdate =  |quote =  }}</ref>。2013年9月16日，北京老字号协会启动“北京老字号”认定工作<ref name=yi/>。

北京老字号自2014年认定第一批起，已认定三批：
*第一批：2014年4月4日认定，共145家企业（151个品牌）（见下表1至151号）<ref name=yi>{{Cite web |url = http://www.batb.com.cn:7001/batb/street/news.do?id=1805 |title = 关于认定第一批北京老字号企业的决定 |author =  |date = 2014-04-04 |publisher = 北京老字号协会 |language =  |accessdate =  |quote =  }}</ref>。
**2015年4月8日调整：第一批认定的原北京方庄酒厂“德兴勇”字号，更换为北京隆兴号方庄酒厂有限公司“隆兴号”<ref>{{Cite web |url = http://www.batb.com.cn:7001/batb/street/news.do?id=1918 |title = 北京老字号协会通告 |author =  |date = 2015-04-08  |publisher =北京老字号协会  |language =  |accessdate =  |quote =  }}</ref>。
*第二批：2015年认定，共14家企业（14个品牌）（见下表152至165号）<ref>{{Cite web |url =  http://www.batb.com.cn:7001/batb/street/news.do?id=1917|title = 第二批进入“北京老字号”企业名录公示公告 |author =  |date = 2015-04-08 |publisher = 北京老字号协会 |language =  |accessdate =  |quote =  }}</ref>
**2016年2月1日增补：[[东单菜市场|北京东单菜市场有限公司]]（注册商标：东单菜市场）（见下表166号）<ref>{{Cite web |url = http://www.batb.com.cn:7001/batb/street/news.do?id=2002 |title = 北京老字号协会公示公告 |author =  |date = 2015-09-29 |publisher = 北京老字号协会 |language =  |accessdate =  |quote =  }}</ref><ref name=dongdan/>
*第三批：计划2017年认定<ref>{{Cite web |url = http://www.batb.com.cn:7001/batb/street/news.do?id=2111 |title = 关于开展第三批北京老字号认定工作的通告 |author =  |date =  2016-03-24  |publisher =  北京老字号协会|language =  |accessdate =  |quote =  }}</ref>
