'''北61線 保佑－中路'''，是位於[[新北市|新北市]]的一條鄉道，起自[[新北市|新北市]][[蘆洲區|蘆洲區]]保佑，迄於[[新北市|新北市]][[蘆洲區|蘆洲區]]中路，全長4.528公里。{{r|thb}}

== 路線說明 ==

*'''[[新北市|新北市]]'''
;[[蘆洲區|蘆洲區]]：
保佑（[[File:TW_THWtp53-1.png|30px]][[北53線#支線|北53-1線]]岔路）→ 仁愛路 → '''和尚港''' → 民族路436巷 → 民族路426巷 → 民族路327巷 → 復興路105巷 → [[File:TW_THWtp64.png|30px]][[北64線|北64線]]（短暫共線） → 信義路 → '''溪墘'''（[[File:TW_CHW103a.svg|23px]][[縣道103號#甲線|103甲線]]） → 信義路 → 成功路 → 和平路 → '''中路'''（終點，[[File:TW_THWtp64.png|30px]][[北64線|北64線]]岔路）
