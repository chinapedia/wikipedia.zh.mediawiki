'''北二區區域教學資源中心'''（'''Second Northern Taiwan Teaching Resource Center '''，簡稱「'''北二區'''」、「'''N2 '''」）成立於2010年，並獲得[[教育部|教育部]]支持。

以[[國立臺灣大學|國立臺灣大學]]為中心學校，協同十二所夥伴學校，共同建構教學資源共享平台，互相流通各種資源、人才與制度；期能在各校精進教學成長機制，並打造全區為永續校園。雖然分布很廣，但由地緣及交通關係，發現可以分成「淡水線」(大致沿[[台北捷運|臺北捷運]][[台北捷運淡水線|淡水線]]─[[台北捷運新店線|新店線]]路廊)及「雪山線」(大致沿[[臺鐵|臺鐵]][[西部幹線|西部幹線]]─[[北宜直鐵|北宜直鐵]]─[[東部幹線|東部幹線]]路廊)兩個脈絡，而其交會點([[臺北市|臺北]]市區)集中了不少夥伴學校。而各校之編號乃根據教育部分配學校時的區域編號(區域A及區域B)<ref>{{cite web |url=http://www.n2.org.tw/about_detail.php?gid=2&nid=3 |title=存档副本 |accessdate=2013-11-03 |deadurl=yes |archiveurl=https://web.archive.org/web/20131104145151/http://www.n2.org.tw/about_detail.php?gid=2&nid=3 |archivedate=2013-11-04 }}</ref>。

發展目標為：校際交流，共享資源；建佈機制，永續校園。每年暑假期間辦理[[夏季學院|夏季學院]]跨校通識課程與生技課程，供全體北二區學生選修<ref>{{cite web |url=http://www.n2.org.tw/resources.php?gid=1 |title=存档副本 |accessdate=2013-11-03 |deadurl=yes |archiveurl=https://web.archive.org/web/20131104145327/http://www.n2.org.tw/resources.php?gid=1 |archivedate=2013-11-04 }}</ref>。於台灣大學、臺灣師範大學、臺灣藝術大學設置「北二區基地營」，提供各校師生討論空間<ref>{{cite web |url=http://www.n2.org.tw/base_camp.php |title=存档副本 |accessdate=2013-11-03 |deadurl=yes |archiveurl=https://web.archive.org/web/20131104145432/http://www.n2.org.tw/base_camp.php |archivedate=2013-11-04 }}</ref>。

== 中心規劃 ==
'''資源整合分享計畫'''：合力建構，共同分享。

Ｔ：教師成長平台

Ｒ：教學資源共享

'''主題式合作計畫'''：各校主題，在地打造。

Ａ：教學助理制度

Ｉ：教學課程建制

Ｌ：學習輔導機制

Ｓ：建立制度人才
