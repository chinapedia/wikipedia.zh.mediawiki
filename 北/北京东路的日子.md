《'''北京东路的日子'''》是一首于2010年创作并在网络上流行的中文歌曲，词、曲的作者均为当时就读于[[南京外国语学校|南京外国语学校]]的高三学生汪源，<ref name="jstv">{{Cite web |url = http://www.jstv.com/s/nanking/news/201505/t20150513_15752801.shtml |title = 南外少年把《北京东路的日子》唱给青春岁月 |deadurl = yes |archiveurl = https://web.archive.org/web/20150526200521/http://www.jstv.com/s/nanking/news/201505/t20150513_15752801.shtml |archivedate = 2015-05-26 }}</ref>原唱者汪源、胡梦源、徐逸昊、许一璇、梁竞元、鲁天舒、姜玮珉、张鎏依、刘千楚、游彧涵、金书媛、张夙西等十二人均为同班同学。<ref name="dsqq">{{cite news |author = 陈曦|url = http://dz.xdkb.net/html/2010-07/04/content_19940.htm  |title = 一首歌送别花季——北京东路的日子 |publisher = 现代快报 |date = 2010年7月4日 }}</ref>

==歌词内容==
[[北京东路_(南京)|北京东路]]是南京外国语学校的所在地，歌词的内容主要是怀念作者在南京外国语学校就读的岁月。<ref name="jstv" />据媒体报道，歌词中所提及的“方丈”是指其[[班主任|班主任]]刘洪雷；“凡哥”是指其数学课的授课教师；“乌龟大师”是指其地理课的授课教师邓奕；“顾萍”则是其历史课的授课教师。<ref name="dsqq"/>至于歌词中提及的“学乐先”，则为一款主要面向中学生群体进行推广的饮料品牌。<ref>{{Cite web |url = http://news.163.com/10/0627/03/6A5EL3T000014AED.html |title = “北京东路的日子” |author = 张磊}}</ref>

==影响==
该歌曲在录制后不久被上传到网络，随后在网络走红。据媒体报道，该歌曲在上传后不久，其在[[优酷网|优酷网]]上的点击次数已经达到六百多万人次。尽管歌曲的内容是描述南京外国语学校的生活，但仍在网友中取得共鸣。<ref>[http://daxue.163.com/11/0603/10/75KAM3F800913KM9.html 毕业歌《北京东路的日子》走红网络]</ref>其后，该歌曲在中国大陆的校园中广泛流行，是毕业季经常出现的歌曲，亦有不少学校的学生将该歌曲进行改编，把歌词中的一些元素替换为自己学校的一些元素，如《九如东路的日子》<ref>[http://news.zzedu.net.cn/zzjyxw/05/1353593.shtml 《九如东路的日子》——郑州47中学子创作歌曲献给母校]</ref>、《厦门兴才的日子》<ref>[http://dw.xmxc.com/news/dwdt/2014/512/news145121127227DG92I7JIBJI7BJ2HCAA.html 艺术与建筑学院“点亮中国梦”班歌比赛八强出炉] {{webarchive|url=https://web.archive.org/web/20150527232100/http://dw.xmxc.com/news/dwdt/2014/512/news145121127227DG92I7JIBJI7BJ2HCAA.html |date=2015-05-27 }}</ref>、《华电的日子》<ref>[http://news.ncepu.edu.cn/hdbd/hdbd3/9145.htm 自动化系举办送毕业生晚会]</ref>等等，但均没有原歌曲知名。

2013年，配合南京外国语学校50周年校庆<ref>[http://www.njupco.com/ubook/index.php?action=detail&bid=4132 南京大学出版社网站上，书籍《北京东路的日子--南京外国语学校毕业生成长记》的信息页]</ref>，由[[南京大学出版社|南京大学出版社]]出版<ref>[https://books.google.com/books/about/?id=95fZoAEACAAJ&hl=zh-CN Google图书上有关《北京东路的日子--南京外国语学校毕业生成长记》一书的书目信息]</ref>的书籍《北京东路的日子--南京外国语学校毕业生成长记》亦以该歌曲的名称作为书名的一部分。
