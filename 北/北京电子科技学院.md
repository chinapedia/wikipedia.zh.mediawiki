'''北京电子科技学院'''，简称'''电科院'''，位于[[北京市|北京市]][[丰台区|丰台区]]富丰路7号，是为党政机关培养[[信息安全|信息安全]]、[[办公自动化|办公自动化]]专业技术人才的普通高等学校，隶属[[中共中央办公厅|中共中央办公厅]]<ref name=jj>{{Cite web |url = http://www.besti.edu.cn/frontmenu/menulist?id=4d46d39267304716a18be881694eee2d&menuLevel=1 |title = 学院简介 |author =  |date =  |publisher = 北京电子科技学院 |language =  |accessdate = 2017-11-04  |quote =  }}</ref>。

==沿革==
===北京电子科技学院===
北京电子科技学院的前身是'''中央工委青年训练班'''，1947年8月在[[河北省|河北省]][[平山县|平山县]][[西柏坡|西柏坡]]成立。1949年，并入[[张家口|张家口]]的'''[[西安电子科技大学|中央人民政府人民革命军事委员会工程学校]]'''<ref name=lishi>{{Cite web |url =http://www.besti.edu.cn/frontmenu/menulist?id=2721f1a130814352b9cc6425ee189070&menuLevel=1  |title =  历史回顾|author =  |date =  |publisher = 北京电子科技学院 |language =  |accessdate = 2017-11-04  |quote =  }}</ref>。1952年6月，部分院系再次分出，与[[南京|南京]]的'''[[华东军区|华东军区]]、[[第三野战军|第三野战军]]青年干部学校'''（1950年3月成立）合并，成立'''中央人民政府人民革命军事委员会机要干部学校'''（'''南京机校'''），归[[中央人民政府人民革命军事委员会|中央人民政府人民革命军事委员会]]建制。后改为'''[[中央人民政府委员会|中央人民政府委员会]]机要干部学校'''。1954年12月，改称'''中国人民解放军第一机要学校'''。1955年5月，改为'''国务院第一机要学校'''，归[[中华人民共和国国务院|国务院]]建制<ref name=jssdfz>{{Cite web |url =http://www.jssdfz.gov.cn/book/jsz1/D3/D1J4.html  |title =  四、人民解放军陆军学校|author =  |date =  |publisher = 江苏省地方志 |language =  |accessdate =2017-11-04  |quote =  }}</ref>。后来该校迁至北京，简称'''北京机校'''。此后又迁至[[河北|河北]][[宣化|宣化]]，简称'''宣化机校'''<ref name=lishi/>。

1981年经国务院批准，成立'''北京电子专科学校'''，校址设在北京[[小汤山|小汤山]]。1983年下半年迁至北京半壁店，并正式对外公开招收大专生。1992年经[[国家教委|国家教委]]批准，在北京电子专科学校的基础上，建立'''北京电子科技学院'''，开始招收本科生，并增设函授教育。1993年，[[中共中央办公厅|中共中央办公厅]]决定将中南海业余大学并入北京电子科技学院。1999年2月，学院迁至[[丰台|丰台]]新校园，位于[[中关村科技园区|中关村科技园区]]丰台园内<ref name=lishi/>。

学校成立以来，[[毛泽东|毛泽东]]、[[邓小平|邓小平]]、[[江泽民|江泽民]]、[[胡锦涛|胡锦涛]]、[[习近平|习近平]]先后题词，江泽民还为学院题写校名。[[乔石|乔石]]、[[温家宝|温家宝]]、[[曾庆红|曾庆红]]、[[王刚_(政治人物)|王刚]]、[[栗战书|栗战书]]等中央领导为学院的发展作过指示并来学院视察<ref name=jj/>。

===中南海业余大学===
1948年春，毛泽东在[[河北省|河北省]][[平山县|平山县]][[西柏坡|西柏坡]]成立'''中央直属机关业余学校'''。当时中央领导为此作了专门研究，决定由[[刘少奇|刘少奇]]兼任校长，具体教务工作由宣教科[[刘火|刘火]]负责，课程有政治理论及语文、数学等文化课，无专职教师，聘请文化理论水平高的干部任教<ref name=liulun>柳伦，中南海机关文化学校轶事，档案春秋2014(4):24-26</ref>。

1949年，[[中共中央|中共中央]]进驻[[北平|北平]][[中南海|中南海]]。该校教学工作交给[[中共中央办公厅警卫局|警卫局]]，警卫局用中南海分散闲置的老旧平房作为教室，对外称'''中南海机关文化学校'''。[[中华人民共和国|中华人民共和国]]成立后，中南海内单位增加，工作人员文化程度不一，需要开设许多课程和班级，领导乃从[[华北人民革命大学|华北人民革命大学]]结业的学员里调来十余位有大学、专科学历的青年任教。1953年时，学员共900多人，校长是毛主席办公室党支部书记、中央警卫局副局长[[王敬先|王敬先]]，副校长是警卫局宣教科长[[马达_(中央警卫局)|马达]]，当时该校开办初中和小学文化课程，加上速成识字共26个班。由于[[中共中央直属机关工作委员会|中共中央直属机关委员会]]规定，不够中学文化水平的工作人员一律到该校学习，所以在中南海办公的中央直属机关如[[中共中央办公厅|中共中央办公厅]]、[[中共中央军委|中共中央军委]]、[[中共中央宣传部|中共中央宣传部]]、警卫局、秘书局、机要室等单位都有工作人员入学，其中警卫局人员最多。在中南海办公的国务院及其系统另有规模庞大的业余学校，但教学地点不在中南海<ref name=liulun/>。

1950年代末，该校半数学员达到初中毕业文化水平，需要上高中。这时该校课程却减少，学员也减至400多人。当时中共中央军委、中共中央宣传部机关迁出了中南海，而中南海内其他机关也因1955年、1957年、1959年三次机关整风整编，工作人员减少。校长王敬先因常随毛泽东出差而无暇过问校务，副校长马达已调到[[新疆|新疆]]工作，正副教导主任及许多教职人员陆续离职，教员仅剩5人，无力办高中<ref name=liulun/>。

1960年初，毛泽东得知上述情况后，指示警卫局领导，要办一所名副其实的完全中学：（一）一定要重视和搞好业余文化教育工作，由[[汪东兴|汪东兴]]局长兼校长，（二）要增加教员，把业余学校办成完全中学，和普通中学一样，开设文、史、地、数、理、化、生物、生理卫生、外语、政治十门课程，外语课必须有，历史课要多讲一点，（三）中学的课程讲完后，要接着办业余大学，（四）由他自己任名誉校长，亲自审批每年的教学计划和总结。他还派秘书[[胡乔木|胡乔木]]来校与教员一起研究古典文学的教学方法。当时该校已从[[怀仁堂|怀仁堂]]对面的[[四合院|四合院]]迁入南海边的政事堂，离毛泽东住地很近<ref name=liulun/>。

为落实毛泽东指示，机关调来四位教员，增开三门高中课程，另聘两位兼课教师，分别开大学历史班（[[北京大学|北京大学]]老师讲授）、普通英语班（警卫局办公室的干部孙守民讲授）<ref name=liulun/>。

[[文化大革命|文化大革命]]冲击了中南海机关的文教工作，该校停办<ref name=liulun/>。1978年[[中共十一届三中全会|中共十一届三中全会]]后，由[[胡耀邦|胡耀邦]]倡议恢复，1981年正式成立了'''中南海文化补习学校'''，给中学未毕业的人员补习高中课程。1983年，开办'''中南海业余大学'''，设秘书系等学科。1985年第一批学生毕业典礼前夕，胡耀邦为学校题写校名<ref name=liulun/><ref name=wy>{{Cite web |url =  http://news.163.com/15/0202/14/AHF35ELO00014Q4P.html|title = 中南海里曾经有所“业余大学” |author =  |date = 2015-02-02|publisher =网易  |language =  |accessdate =  |quote =  }}</ref>。

1988年时，中南海业余大学参与组建[[首都联合职工大学|首都联合职工大学]]。1993年该校又从首都联合职工大学独立出来，再度恢复为'''中南海业余大学'''。1993年，中共中央办公厅决定将中南海业余大学并入北京电子科技学院，[[合署办公|合署办公]]<ref name=wy/>。2007年，中南海业余大学被[[中华人民共和国教育部|教育部]]红牌警告，不得安排招生<ref>{{Cite web |url = http://china.rednet.cn/c/2007/09/21/1327183.htm |title = 教育部对60所成人高校亮红牌 要求其停止招生 |author =  |date =2007-09-21  |publisher =红网  |language =  |accessdate =  |quote =  }}</ref>。截至2017年5月31日，据教育部网站显示，中南海业余大学仍在成人高等学校名单中<ref>{{Cite web |url = http://www.moe.gov.cn/srcsite/A03/moe_634/201706/t20170614_306900.html |title =	全国高等学校名单  |author =  |date = 2017-06-14 |publisher = 中华人民共和国教育部 |language =  |accessdate =  |quote =  }}</ref>。
