'''北條時子'''（{{jpn|j=北条時子|hg=ほうじょう ときこ|rm=Hōjō Tokiko}}，{{bd|？||？|catIdx=Hōjō Tokiko}}）是日本[[平安時代|平安時代]]末期至[[鎌倉時代|鎌倉時代]]前期[[北條氏|北條氏]]一族的女性<ref name="houjou">[[#北条|北条（2001）]]</ref>，[[鎌倉幕府|鎌倉幕府]]首任[[執權|執權]][[北條時政|北條時政]]之女<ref name="houjou"/>，出身自[[北條氏|北條氏]][[得宗|得宗]]義時流<ref name="houjou"/>，[[御家人|御家人]]{{Link-ja|足利義兼|足利義兼}}[[正室|正室]]，{{Link-ja|足利義氏 (足利家第3任當主)|足利義氏 (足利家3代目当主)|足利義氏}}和野田朝氏室等人之母<ref name="houjou"/>。時子是[[北條政子|北條政子]]的同母妹<ref name="chida033">[[#千田|千田（1985）pp.33-40]]</ref>，其他的兄弟分別是{{Link-ja|北條宗時|北条宗時}}、[[北條義時|北條義時]]、{{Link-ja|北條時房|北条時房}}、{{Link-ja|阿波局 (北條時政之女)|阿波局 (北条時政の娘)|阿波局}}和{{Link-ja|北條政範|北条政範}}等等。真名不明，時子亦是流傳下來的稱呼，根據同樣不明<ref name="houjou"/>，法號是'''智願院殿'''。

== 生涯 ==
按《[[吾妻鏡|吾妻鏡]]》記載，[[治承|治承]]5年（1181年）2月1日，時子遵照其姊政子之丈夫首任[[鎌倉幕府將軍列表|鎌倉將軍]][[源賴朝|源賴朝]]之意，嫁給義兼<ref name="houjou"/>。義兼與賴朝同樣出身自[[河內源氏|河內源氏]][[源義家|義家]]，與賴朝是同一{{Link-ja|門葉|門葉}}，加上義兼之母是賴朝之母的姪女，因此時子嫁予義兼後，義兼與賴朝的關係更為緊密，有助其在幕府取得高位<ref name="chida033"/>。

賴朝死後，時政將他的女兒分別嫁給其他有力的御家人，例如門葉的{{Link-ja|阿野全成|阿野全城}}、{{Link-ja|稻毛重成|稲毛重成}}、{{Link-ja|平賀朝雅|平賀朝雅}}、{{Link-ja|畠山重忠|畠山重忠}}和{{Link-ja|河野通信|河野通信}}等人，北條得宗在爭奪幕府權力的過程中逐漸失勢，同時[[足利氏|足利氏]]開始在時子之子義氏一代崛起，儘管北條得宗對幕府的影響不大，但是足利氏在源家將軍絕後後仍然與執權北條氏通婚，代代迎娶北條氏女性為妻，因此足利氏在鎌倉時期的歷代當主均是北條氏所生的嫡子繼承[[家督|家督]]<ref name="chida033"/><ref group="註解">第6代任當主{{Link-ja|足利家時|足利家時}}和第8任當主[[足利尊氏|足利尊氏]]則是由側室[[上杉氏|上杉氏]]所生。家時在[[霜月騷動|霜月騷動]]後[[切腹|切腹]]，尊氏則將北條氏消滅</ref>。

時子與義兼之間生下後來成為足利家當主的{{Link-ja|足利義氏 (足利家第3任當主)|足利義氏 (足利家3代目当主)|足利義氏}}以及一度為第3任將軍[[源實朝|源實朝]]正室候補人選的[[熱田神宮|熱田]]{{Link-ja|宮司|宮司|大宮司}}野田朝氏室等人<ref name="chida033"/>，早夭的瑠璃王和藥壽御前據稱也是時子之子<ref group="註解">2008年3月，在[[真如苑|真如苑]]於紐約的拍賣會中成功投得兩尊據稱由[[運慶|運慶]]製作的[[大日如來|大日如來]]像中，存在義兼的發願，而寫有「[[鑁阿寺|鑁阿寺]]樺崎緣起並佛事次第」的台座處供納了瑠璃王和藥壽御前的遺骨。</ref><ref name="minegishi013">[[#峰岸|峰岸（2009）pp.13-24]]</ref>，而按照系圖{{Link-ja|足利義純|足利義純|畠山義純}}也是時子所生<ref name="houjou"/>。

雖然時子的去世年份不明，但是為了憑弔時子而建立的位於[[栃木縣|栃木縣]][[足利市|足利市]]的法玄寺中的的「蛭子傳說」則指是[[建久|建久]]7年（1196年）6月8日<ref name="hougenji">[http://www.hogenji.net/history.html 法玄寺の歴史-開山の由来-]</ref>。

== 蛭子傳說 ==
有關時子之死，在足利市流傳以下的傳說：有一次義兼前往[[鎌倉|鎌倉]]期間，讓時子留在{{Link-ja|足利莊|足利荘}}時，時子在野外飲用了侍女藤野從井拿來的生水，不久後肚子便發漲，形似懷孕婦女。後來，藤野向義兼告發指時子與{{Link-ja|足利宗綱|足利忠綱}}通姦才懷上孩子，義兼便對時子起疑心，時子留下「我死後你就調查我的身體吧。」的遺言，在建久7年自殺。其後，義兼按照時子的遺言，檢查其遺體時，在其腹部發現大量[[水蛭|水蛭]]，對此義兼後悔莫及，誠心憑弔時子的同時將該口井封掉，並且對藤野處以{{Link-ja|牛裂|牛裂き}}之刑<ref name="bannaji">[http://books.google.co.jp/books?id=AHRPpbXEdy0C&pg=PT53&dq=%22%E8%9B%AD%E5%AD%90%22%E3%80%80%E8%B6%B3%E5%88%A9&hl=ja&sa=X&ei=JRrNT9C5HKfPmAXKwcybAQ&ved=0CEwQ6AEwBQ#v=onepage&q=%22%E8%9B%AD%E5%AD%90%22%E3%80%80%E8%B6%B3%E5%88%A9&f=false 忍空山越『鑁阿寺小史』鑁阿寺、1888年、pp20-27。]</ref><ref name="hougenji"/>。

在[[昭和|昭和]]年間的工事時，法玄寺發掘出的鎌倉時代[[五輪塔|五輪塔]]乃時子之墓，將其稱為御水蛭大人（{{lang|ja|お蛭子さま}}），並且獲足利市指定為[[重要文化財|重要文化財]]<ref name="hougenji"/>，而在義兼建立的[[鑁阿寺|鑁阿寺]]內亦有與蛭子傳說相關的「蛭子堂（智願寺殿御靈屋）」和不能解封的井（{{lang|ja|開かずの井戸}}），作為安產之神受到敬拜，同樣獲足利市指定為有形文化財<ref>{{Cite web |url=http://www.city.ashikaga.tochigi.jp/site/bunkazai/bunkazai-shishitei.html |title=足利市指定文化財 |publisher=足利市 |accessdate=2012-07-07}}</ref>。

從傳說流傳下來的不同版本中可見，藤野誣告時子可能是因為對忠綱因愛成恨或其實藤野是義兼的側妻<ref name="hougenji"/><ref>日向野徳久：編『日本の民話　37巻　栃木の民話』未来社。</ref>。

法玄寺內的「蛭子塚」和鑁阿寺的「蛭子堂」看解說板均指，藤野稱時子的不倫對像是{{Link-ja|足利氏 (藤原氏)|足利氏 (藤原氏)|藤姓足利氏}}的足利又太郎忠綱，忠綱雖然逃走，但是誤信謠言的義兼仍然窮追不捨，並且將其斬殺或逼使其自殺，留下足利市「逆藤天滿宮」的逆生藤天神（{{lang|ja|逆さ藤天神}}）的傳說<ref name="bannaji"/>。
