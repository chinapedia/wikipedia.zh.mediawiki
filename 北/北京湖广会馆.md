'''北京湖广会馆'''，位于[[北京市|北京市]][[西城区|西城区]][[虎坊路|虎坊路]]3号，是[[清朝|清朝]]时设在[[北京|北京]]的[[湖广|湖广]]同乡联谊场所。北京湖广会馆戏楼至今仍在演出传统戏曲、曲艺，是北京有名的传统戏楼。

==历史==
北京地区的湖广同乡会馆，最早可追溯到[[明朝|明朝]]。明朝[[万历|万历]]年间，[[内阁首輔|内阁首輔]][[张居正|张居正]]将私宅捐建全楚会馆<ref name=wange>{{Cite web |url = http://news.xinhuanet.com/mrdx/2015-03/27/c_134101980.htm |title =同乡会馆的挽歌  |author =  |date = 2015-03-27 |publisher = 新华网 |language =  |accessdate =  |quote =  }}</ref>。[[沈德符|沈德符]]《[[万历野获编|万历野获编]]·补遗·凶宅》记载，“京师全楚会馆，故[[张居正|江陵相]]第，壮丽不减王侯，特[[严嵩|分宜]]旧第四分之一。右一小房，为京师富人徐性善所得，后坐他事籍没。自严及张迄徐，未三十年，三遭抄没，其为凶宅可知。”可见此宅原先为[[严嵩|严嵩]]的宅第，后归张居正，此后才改为全楚会馆。根据考证，这处全楚会馆的地址在[[南半截胡同|南半截胡同]]、[[北半截胡同|北半截胡同]]以东，[[菜市口大街|丞相胡同]]（今[[菜市口大街|菜市口大街]]）以西，与今天的北京湖广会馆不在一地<ref name=pei>裴效维，吴趼人生于“分宜故第”考，江苏师范大学学报(哲学社会科学版)2006, 32(1):26-30</ref>。

北京湖广会馆的原址曾是[[清朝|清朝]]官员和名人的故居。据史料记载，[[乾隆|乾隆]]、[[嘉庆|嘉庆]]年间大臣在此居住过，如[[刘权之|刘权之]]、[[王杰_(清朝)|王杰]]等等<ref name=yy>{{Cite web |url = http://www.beijinghuguang.com/comcontent_detail/&FrontComContent_list01-1385012606357ContId=5&comContentId=5.html |title =源远流长  |author =  |date =  |publisher =  北京湖广会馆大戏楼|language =  |accessdate =  |quote =  }}</ref>。[[嘉庆|嘉庆]]十二年（1807年），在京官员刘权之（刘云房，[[湖南|湖南]][[长沙|长沙]]人）、[[李均简|李均简]]（李秉和，[[湖北|湖北]][[黄冈|黄冈]]人）等人倡议在此修建湖广会馆<ref name=yy/>。[[道光|道光]]十年（1830年）年初团拜时，[[蒋祥墀|蒋祥墀]]（蒋丹林）、[[何凌汉|何凌汉]]（何仙槎）倡议集资扩建戏楼，作为湖广同乡集会、公宴的场所<ref name=zgw/><ref name=xinwen/>。同年，[[叶继雯|叶继雯]]（叶云素）将监馆私宅捐给湖广会馆，使湖广会馆面积扩大<ref>{{Cite web |url =http://www.chinajsb.cn/gb/content/2010-01/25/content_300314.htm  |title = 湖广会馆风云际会两百年 |author =  |date =2010-01-25  |publisher =  中国建设报|language =  |accessdate =  |quote =  }}</ref>。道光二十九年（1849年），[[曾国藩|曾国藩]]等人倡议重修，设置亭榭等建筑，总面积扩大到约4700多平方米。[[同治|同治]]九年（1870年），[[曾国藩|曾国藩]]在这里办六十大寿。[[光绪|光绪]]二年（1876年），长沙人搜集[[李东阳|李东阳]]（李文正公）手书的《丛桂堂诗》、《致仕诗》、《慈恩寺诗》以及“旧时风景”四个字摹临上石嵌在湖广会馆的宝善堂墙壁内。光绪十八年（1892年），[[茶陵|茶陵]][[谭钟麟|谭钟麟]]（谭文卿）制军、[[江夏|江夏]][[张仲炘|张仲炘]]（张次珊）通参主持重修湖广会馆。[[光绪|光绪]]二十六年（1900年）[[庚子事变|庚子事变]]，[[八国联军|八国联军]]侵入北京，八国联军中的美军提督一度将湖广会馆作为[[美军|美军]]司令部。光绪三十四年（1908年），第四次留学毕业生考试及弟诸君在湖广会馆举办团拜宴会<ref name=zgw>{{Cite web |url = http://www.china.com.cn/culture/txt/2009-02/17/content_17291415.htm |title =  逛北京 湖广会馆的故事|author =  |date = 2009-02-17 |publisher =  中国网|language =  |accessdate =  |quote =  }}</ref><ref name=xinwen>{{Cite web |url = http://www.chinaxwcb.com/index/2008-05/09/content_119094.htm |title =京都会馆三戏楼  |author = 周简段 |date = 2008-05-09 |publisher = 中国新闻出版网 |language =  |accessdate =  |quote =  }}</ref>。

[[中华民国|中华民国]]成立后，北京湖广会馆经常承办各种会议和活动。民国元年（1912年）春，财政部印刷局（今541厂）工人因受[[辛亥革命|辛亥革命]]影响，部分人参加“印刷同业进化会”，李鹤岩、王梦兰、贝允卿、张贻等人在湖广会馆开会，决定“工人要团结要组织起来，有话敢说，研究技术，改善生活。”民国元年五月七日，[[统一党_(中国)|统一党]]人在湖广会馆开会欢迎[[章太炎|章太炎]]，报纸报道称；“北京统一党日前假湖广会馆开会，在京党员五六百人到会，首由章炳麟演说该党宗旨之正大，次由[[唐文治|唐蔚芝]]演说进行方法，后由徐、龚诸君相继布告各处党员日见增多，合计达三四千以上，并述该党归入[[民社|民社]]之理由等。”民国五年（1916年），[[梁启超|梁启超]]在湖广会馆内宣讲宪法纲领<ref name=zgw/>。

[[孙中山|孙中山]]曾先后五次来湖广会馆。1912年8月24日，孙中山经[[天津|天津]]抵达北京。1912年8月25日上午10时，在北京的旧[[同盟会|同盟会]]员在湖广会馆为孙中山开欢迎大会<ref name=sun>{{Cite web |url =http://news.163.com/11/1003/08/7FE6CDRF0001124J.html  |title = 孙中山留在北京的印迹 |author =  |date = 2011-10-03 |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref>。1912年8月25日下午，[[国民党_(1912–1913)|国民党]]成立大会在湖广会馆举行，孙中山在会上演说约两小时，大会推举孙中山、[[黄兴|黄兴]]、[[宋教仁|宋教仁]]等9人为理事，[[阎锡山|阎锡山]]、[[张继_(中华民国)|张继]]、[[李烈钧|李烈钧]]、[[柏文蔚|柏文蔚]]、[[于右任|于右任]]等30人为参议<ref name=sun/><ref name=zgw/><ref name=xinwen/>。1912年8月30日下午，孙中山来湖广会馆参加北京学界欢迎大会。1912年9月4日下午，孙中山参加[[共和党_(中国)|共和党]]本部的欢迎会并发表演说，解释国民党的[[三民主义|三民主义]]。1912年9月15日，孙中山在离开北京前两日，又到湖广会馆参加国民党办的欢迎会，会后参观了[[广安门内大街|彰仪门大街]]国民党本部<ref name=sun/>。

民国九年（1920年）五月，两湖旅京人士在湖广会馆筹设两湖善后协会，以便救济两湖的灾民。民国十四年（1925年）六月十四日，湖北旅京同乡全体，在湖广会馆开紧急大会，讨论[[汉口|汉口]][[英军|英军]]惨杀同胞案，决定组织湖北旅京同乡后援会。民国十六年（1927年），湖广会馆成立120周年纪念，正月公祭湖广乡贤并举办团拜。民国十六年五月初五公祭[[屈原|屈原]]，同年刊行《湖广会馆纪略》。民国十九年（1930年）八月，董事会改组，推举[[甘鹏云|甘鹏云]]（甘药樵）为董事长，[[吴家驹|吴家驹]]（吴子昂）等人为董事。民国三十二年（1943年）十月，修竣湖广会馆内的子午井并且建井栏<ref name=zgw/>。

湖广会馆戏楼还是北京重要的戏曲演出场所。[[京剧|京剧]]名家[[谭鑫培|谭鑫培]]、[[余叔岩|余叔岩]]、[[梅兰芳|梅兰芳]]、[[程砚秋|程砚秋]]、[[陈德霖|陈德霖]]等人以及京剧名票[[王直君|王直君]]等人都曾在此演戏<ref name=zgw/><ref name=yy/>。据说清朝每年正月，两湖旅京人士都要在湖广会馆戏楼举办团拜并请名伶演剧三日，京剧名家谭鑫培、余叔岩等都曾在此演戏。中华民国成立后，这种习惯逐渐废除<ref name=zgw/>。

[[中华人民共和国|中华人民共和国]]成立后，北京的各个会馆起初保留其产权，成立会馆财产管理委员会，以取代馆长、同乡会行使管理权。1956年，会馆的会产全部移交政府，变为北京市国有资产<ref name=wange/>。

1984年，湖广会馆被公布为[[北京市文物保护单位|北京市文物保护单位]]。1994年，北京市天桥投资开发公司对湖广会馆全面修复。其中包括大戏楼、文昌阁、楚畹堂、风雨怀人馆、宝善堂等建筑。1996年5月大戏楼对社会开放，每晚由[[北京京剧院|北京京剧院]]演出。文昌阁、乡贤祠辟为北京市戏曲博物馆<ref name=zgw/>。

==建筑==
湖广会馆的主要建筑有：
*大门：嵌有砖雕<ref name=zgw/>。
*戏楼：位于前院。后台十间，前台北、东、西三面为看楼，上下合计四十间，中有广场<ref name=zgw/>。
*乡贤祠、文昌阁：位于中院（今为后院）。乡贤祠为北屋三间。文昌阁在乡贤祠楼上，阁内供奉“[[文昌帝君|文昌帝君]]神位”<ref name=zgw/>。
*宝善堂：位于后院中院，坐北朝南，面阔五间，东西有长廊<ref name=zgw/>。
*楚畹堂：位于西院，前后各三间<ref name=zgw/>。
*风雨怀人馆：位于乡贤祠和文昌阁的后室，面阔三间，建在高台上，两侧有斜廊可上下，传为曾国藩设计<ref name=zgw/>。
*花园：内有假山、亭榭，种植草木<ref name=zgw/>。
*子午井：位于乡贤祠前。口径约2尺，深过7丈。过去每天逢子时、午时，井内清泉上涌，水的味道比平时清甜。子午井于1970年代填平，其上盖起小房，井口挪到不远处砌渗水池用。1996年修复，汉白玉高井台是重修时新砌。井台四周有护栏，刻有铭文，[[傅岳棻|傅岳棻]]撰序，作于民国三十二年（1943年）淘井竣工时，其中记述子午井的得名和其地始末。兹节录一段<ref name=zgw/>：
{{引用|湖广馆文昌阁之下有井曰子午，[[纪昀|纪文达]]《[[阅微草堂笔记|阅微草堂笔记]]》云‘子午二时汲则甘，余时则否，其理莫明，或曰阴起午中，阳生子半，与地气应也。‘然二时何以水味独甘，其理究不可知，或说亦末足据。其地为[[徐乾学|徐司冠乾学憺园]]，[[岳锺琪|岳襄勤锺琪]]、[[刘权之|刘文恪权之]]、[[张惟寅|张运使惟寅]]、[[纪昀|纪文达昀]]、[[王杰_(清朝)|王文瑞杰]]相继居此，最后归[[汉阳|汉阳]]叶氏平安馆。自云素给谏传孙[[叶名琛|昆臣]]、[[叶名澧|润臣]]，两广事覆，乃捐为两湖会馆，地既历为名贤所居，而井泉又灵异莫测，则斯井亦宣南一掌故也。岁久不修，井以湮塞，癸未之秋，[[阳新|阳新]][[石荣暲|石荣暲荩年]]襄理馆事，言于总董[[吴家驹|吴家驹子昂]]，重加修葺，淘其秽积，浚其污垫，于是泉脉复通，其水味之应时而甘果不虚也，……斯井在文昌阁下，在久不改而灵异殊常，今兹塞而复通，井养而不知其为吾两湖文明大启之象乎，爰瓷石为阑属予铭之，垂际久远，余故序其缘起如此。}}

如今，湖广会馆仅存前院和中院及部分附属群房，原大门保存尚完整。后院建筑已于1970年代拓宽[[骡马市大街|骡马市大街]]时拆除<ref name=zgw/>。
