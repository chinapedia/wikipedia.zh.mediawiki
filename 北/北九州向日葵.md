'''北九州向日葵'''（{{lang-ja|ギラヴァンツ北九州}}）是一個[[日本|日本]]的職業[[足球|足球]]隊，主場設在[[福岡縣|福岡縣]][[北九州市|北九州市]]。前名'''北九州新浪'''，於2009年1月1日起改名。<ref>[http://www.j-league.or.jp/release/000/00003312.html {{nihongo|News Release:New Wave Kitakyushu became a member of J. League(Japanese)|ニュースリリース ニューウェーブ北九州がJリーグ入会}}] {{webarchive|url=https://web.archive.org/web/20091201181520/http://www.j-league.or.jp/release/000/00003312.html |date=2009-12-01 }} J. League official website posted 2009-11-30</ref>2009年在[[日本足球聯賽|日本足球聯賽]]奪得升級資格，2010年在[[日本職業足球聯賽|J2]]出賽，2014年日乙獲得了歷史最佳戰績第5名。

== 歷史 ==
成立於1947年，是由[[三菱化學|三菱化學]]中的黑崎工廠人員組成。剛開始大多參加地方聯賽，成為[[北九州|北九州]]代表。他們的對手是同城的[[八幡製鐵|八幡製鐵]]。

自2001年起，他們開始從地方崛起。2007年，在[[日本全國足球區域決勝大會|日本全國足球區域決勝大會]]上奪得第二名，並取得[[日本足球聯賽|日本足球聯賽]]參賽權。2008年1月，遞出了加入[[日職聯成員|日職聯成員]]，於2008年2月19日的會議通過。

2009年4月1日，由於[[商標權|商標權]]爭議，決定改名。2009年10月2日，確定新名為北九州前進葵花，來自[[義大利文|義大利文]]'''Girasole'''和'''Avanzare'''，意思是'''向日葵'''以及'''向前邁進'''。<ref>[http://newwave-k.co.jp/news/index/0/291 {{nihongo|Announcement of the new team name and its emblem|新チーム名称・エンブレム発表}}] {{webarchive|url=https://web.archive.org/web/20091007144843/http://newwave-k.co.jp/news/index/0/291 |date=2009-10-07 }} New Wave Kitakyushu official website news release : dated 2 October 2009</ref>

2009年11月23日，客場2比1擊敗[[高崎藝術|高崎藝術]]後，確定拿下前四名，也意味著升級至[[日本職業足球聯賽|J2]]。
<ref>[http://newwave-k.co.jp/news/index/0/353 Official website (Japanese only)] {{webarchive|url=https://web.archive.org/web/20091203112618/http://newwave-k.co.jp/news/index/0/353 |date=2009-12-03 }}</ref><ref>{{cite web |url=http://www.jsgoal.jp/official/00093000/00093326.html |title=存档副本 |accessdate=2009-11-30 |deadurl=yes |archiveurl=https://web.archive.org/web/20091204150153/http://www.jsgoal.jp/official/00093000/00093326.html |archivedate=2009-12-04 }} J's Goal Website (Japanese only)</ref><ref>[http://www.tvq.co.jp/news/news.php?did=5162]</ref>

2014年日乙球季，他們的表現有很大突破，以創球會史上成績的第5名完成球季，由於進入前6名本應能參與升班附加賽，但他們由於未有[[日職聯發牌制度|J1牌照]]而未能參戰。
