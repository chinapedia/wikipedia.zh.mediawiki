'''北京华润饭店'''，位于[[北京市|北京市]][[朝阳区_(北京市)|朝阳区]][[建国路_(北京)|建国路]]35号。现正计划改造为华润康复医院。

==简介==
1987年初春，高碑店停车场门前竖起“中国南洋有限公司”和“南洋饭店筹建处”（华润饭店的前身）两块牌子。经过五年建设，1992年5月华润饭店正式开业。当时，华润饭店是北京东部的标志性建筑
<ref>{{Cite web |url =http://crchat.crc.com.cn/2002/200206/hua.htm  |title = 华润饭店的建设者们 |author = 唐林鹤 |date = |publisher = 华润2002年6月总58期 |language =  |accessdate =  |quote =  }}</ref>。华润饭店由[[香港|香港]][[华润集团|华润集团]]管理。2001年重新装修<ref>{{Cite web |url = http://travel.fengniao.com/357/3579507_all.html |title = 两会进行时 看明星代表与委员的下榻酒店 |author =  |date = 2013-03-07 |publisher = 蜂鸟网 |language =  |accessdate =  |quote =  }}</ref>。

北京华润宾馆经常接待每年参加“全国[[两会|两会]]”的[[全国政协|全国政协]]委员<ref>{{Cite web |url =http://news.163.com/15/0228/20/AJIODKH000014SEH.html  |title = 今年全国人大代表政协委员开会时住哪儿 |author =  |date = 2015-02-28 |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref>。

2008年9月，华润饭店停业，计划重新发展为酒店兼服务式公寓。2011年，[[北京市规划委员会|北京市规划委员会]]发布公告称，华润饭店改扩建项目设计方案再作调整，将采纳群众建议，取消华润饭店东北部建筑，并且拉开华润饭店同西北部住宅的间距<ref>{{Cite web |url = http://news.cntv.cn/20110827/101940.shtml |title =华润饭店扩建区“瘦身”  |author =  |date = 2011-08-27 |publisher = 央视网 |language =  |accessdate =  |quote =  }}</ref>。此后华润饭店改扩建项目长期停滞。

2013年，华润医疗集团有限公司首次披露医院投资规划，计划将华润饭店改造为华润康复医院<ref>{{Cite web|url =http://yq.stcn.com/2013/1227/11039972.shtml|title =华润医疗首次披露医院投资规划|author =|date =2013-12-27|publisher =证券时报网|language =|accessdate =|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20140218133338/http://yq.stcn.com/2013/1227/11039972.shtml|archivedate =2014-02-18}}</ref>。不过此后改造工作并无大的进展。
