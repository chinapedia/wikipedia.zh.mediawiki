== 简介 ==
国二招宾馆的前身是1969年12月建成的'''国务院第二招待所'''<ref>{{Cite web |url = http://dangjian.people.com.cn/n/2013/0304/c117092-20665706-2.html |title = 盘点人大代表团所住宾馆：无鲜花红毯 不安排高档菜肴 |author = |date = 2013-03-04 |publisher = 中国共产党新闻网 |language = |accessdate =  |quote =  }}</ref>。国二招宾馆经常接待每年“全国[[两会|两会]]”的代表团<ref>{{Cite web |url = http://www.npc.gov.cn/pc/12_5/node_31919.htm |title = 十二届全国人大五次会议驻地 |author = |date = |website = 中国人大网 |language =  |accessdate = 2017-03-28 |quote =  }}</ref>。

国二招宾馆是四[[星级酒店|星级酒店]]<ref>{{Cite web|url =http://www.bjta.gov.cn/tsfwzt/qyml/373570.htm|title =星级饭店|author =|date =2015-04-29|publisher =北京市旅游发展委员会|accessdate =|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20160317085035/http://www.bjta.gov.cn/tsfwzt/qyml/373570.htm|archivedate =2016-03-17}}</ref>。2002年，国二招宾馆进行ISO质量体系认证<ref>{{Cite web |url = http://www.zgg.org.cn/zhtbd_5658/gwxzjnds/gwlzp/gjjgswglj/201412/t20141223_491141.html |title = 41-2 北京国二招宾馆建章立制工作调研报告 |author =  |date = 2014-12-23 |website = 紫光阁网 |accessdate = |quote =  }}</ref>。
