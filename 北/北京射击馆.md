'''北京射击馆'''，位于[[北京市|北京市]][[石景山区|石景山区]]福田寺甲3号院，隶属[[国家体育总局|国家体育总局]]射击射箭运动管理中心。

==简介==
1955年，北京射击场建成，当时因条件简陋，运动员只能在棚子内比赛和训练。1987年，北京射击场重建，条件获得很大改善<ref name=wyss>{{Cite web |url = http://money.163.com/07/0730/11/3KL89J0E00251RJ2.html |title = 射手，融于自然中 |author =  |date = 2007-07-30 |publisher =  网易|language =  |accessdate =  |quote =  }}</ref>。

北京射击馆位于原国家体育总局射击射箭运动管理中心园区南侧，南临[[香山南路|香山南路]]，东为[[北京五环路|西五环路]]，西为亚洲电视制作中心，背依[[北京西山|北京西山]]翠微山脉，总用地面积约6.45公顷，总建筑面积47626平方米（其中地上45372平方米，地下2254平方米），是[[2008年北京奥运会|2008年北京奥运会]]主要场馆之一。2003年3月20日，“2008年奥运会北京射击馆和[[老山自行车馆|老山自行车馆]]建筑设计方案竞赛”开始，最终[[清华大学建筑设计研究院|清华大学建筑设计研究院]]建筑师[[庄惟敏|庄惟敏]]、[[祁斌|祁斌]]、汪曙等人的北京射击馆设计方案获胜。北京射击馆建设项目包括资格赛馆、决赛馆、永久枪弹库、室外配套设施。资格赛馆和决赛馆之间有空中连廊连接，空中连廊下方是可容纳近万人的室外观众集散广场。建筑设计理念为“林中狩猎”，建筑中引入阳光、风、绿树等自然元素<ref>{{Cite web |url =  http://sports.sina.com.cn/s/2003-06-12/1328460477.shtml|title =北京奥运：射击馆、自行车馆初见模样  |author =  |date = 2003-06-12 |publisher =  |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://www.people.com.cn/GB/channel21/200303/21/141929.html |title = 北京奥运会射击馆和自行车馆设计方案竞赛开始 |author =  |date =2003-03-21  |publisher = 人民网 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://news.sina.com.cn/o/2003-12-30/17011467139s.shtml |title = 北京射击馆由清华人设计 |author =  |date = 2003-12-30 |publisher =  |language =  |accessdate =  |quote =  }}</ref><ref name=jianzhuzx>{{Cite web|url =http://www.archreport.com.cn/show-6-511-1.html|title =2008北京奥运会射击馆|author =|date =|publisher =中国建筑中心官网|language =|accessdate =2017-05-17|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20140701043954/http://www.archreport.com.cn/show-6-511-1.html|archivedate =2014年7月1日|df =}}</ref><ref name=wyss/>。

2003年12月28日，北京奥运会老山自行车馆、北京射击馆奠基典礼举行<ref>{{Cite web |url =  http://sports.sohu.com/2003/12/28/29/news217542956.shtml|title =  领导出席奥运会自行车馆奠基仪式|author =  |date = 2003-12-28 |publisher = 搜狐 |language =  |accessdate =  |quote =  }}</ref>。2007年7月16日，北京射击馆开始接受验收，是北京所有新建奥运场馆中第一个开始接受验收的<ref name=jungong>{{Cite web |url =http://gb.cri.cn/11830/2007/07/17/1945@1680121.htm  |title = 奥运首个新建场馆竣工 北京射击馆今起验收 |author =  |date =  2007-07-17|publisher = 国际在线 |language =  |accessdate =  |quote =  }}</ref>。2007年7月28日，北京射击馆正式验收交付，是北京第一个验收交付的奥运比赛场馆<ref>{{Cite web |url = http://news.xinhuanet.com/photo/2007-07/28/content_6442191.htm |title =北京射击馆验收交付 首个奥运新建场馆竣工  |author =  |date = 2007-07-28 |publisher =  新华网|language =  |accessdate =  |quote =  }}</ref>。

北京射击馆承担了[[2008年夏季奧林匹克運動會射擊比賽|2008年北京奥运会射击项目]]室内射击小项，包括10米、25米、50米步枪、手枪11个项目的比赛。此后在[[2008年夏季殘奧會|残奥会]]中承担[[射擊_(殘奧會)|射击]]的专项。赛后北京射击馆可用於承担国际、国内赛事，并且是中国国家射击队的常年训练基地，还对社会部分开放<ref name=jianzhuzx/><ref name=jungong/>。

在北京射击馆建设期间，在资格赛馆和决赛馆北侧拆除了一座老办公楼和一座运动员宿舍楼后，原址建设了综合训练馆、科研楼<ref name=wyss/>。

==北京射击场飞碟靶场==
2006年3月24日，北京射击场飞碟靶场正式开工。属于2008年奥运会11个改扩建场馆之一，位于原国家体育总局射击射箭运动管理中心园区北侧。建筑面积6170平方米，设座席5000个（固定座席1047个，临时座席3953个）。承担2008年北京奥运会射击项目飞碟小项，包括男女飞碟双向、多向、双多向6个项目的比赛<ref>{{Cite web |url = http://news.sina.com.cn/c/2006-03-25/10218526661s.shtml |title = 北京射击场飞碟靶场开建 计划2007年8月完工 |author =  |date =  2006-03-25|publisher =新浪  |language =  |accessdate =  |quote =  }}</ref>。飞碟靶场建筑由主体建筑、室外竞赛部分组成。主体建筑包括设在二层的室外看台及看台下部的赛事管理运营部分。室外赛区由6个独立靶位组成，各靶位间由约370余米的地下靶壕贯通，地下共设15台自动抛靶机，6个独立靶位各设2个抛靶房，另外还建有6个地下抛靶房。飞碟靶场由清华大学建筑设计研究院设计，中国航空港建设总公司施工。2007年8月，北京射击场飞碟靶场竣工。2008年奥运会后，飞碟靶场可承担国际、国内射击比赛，并作为中国国家射击队常年训练基地、青少年培训基地<ref>{{Cite web |url =http://news.sina.com.cn/o/2007-08-10/020012357356s.shtml  |title = 奥运飞碟靶场“很中国” |author =  |date = 2007-08-10 |publisher =新浪  |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://2008.qq.com/a/20070912/000155.htm |title =2008年奥运会场馆介绍—北京射击场飞碟靶场  |author =  |date =2007-09-12  |publisher =腾讯  |language =  |accessdate =  |quote =  }}</ref>。
