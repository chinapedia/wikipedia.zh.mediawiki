[[北領地|北領地]]至2008年7月1日已劃定了17個[[地方政府區域|地方政府區域]]，即是有17個地方政府直接負責地方政務。北領地對於地方政府的設置最早可追溯至20世紀初期<ref>[http://www.lgant.nt.gov.au/home/nt_local_government/history/ Northern Territory LGA History] {{webarchive|url=https://web.archive.org/web/20041026010323/http://www.lgant.nt.gov.au/home/nt_local_government/history/ |date=2004-10-26 }}</ref>。

北領地於2008年7月1日以前的地方行政區域原本有61個地方政府，曾分為4種：
# 6個自治市 
# 51個社區政府
# 3個組織協會（[[澳大利亞聯邦|聯邦]]）
# 1個特殊用途城鎮

根據2008年7月的地方行政改革，地方政府共有2種分類：
# 自治市：原屬[[达尔文_(澳大利亚)|達爾文]]等城市的郊區和其他小型的農村型城鎮 
# 郡：原為郊外或偏遠地帶

== 地方政府分類 ==
[[北領地|北領地]]曾經是個多數人口聚居於非自治區域的地方。2006年，於地方行政改革推動以前，領地內92%的土地上（1,349,130 km²之1,237,999）住著領地內16%的人口（192,898之30,523）居於非自治區域。

多數的非自治區域於2008年後已歷經整併，目前僅有19,428 km²（佔領地內1.45%的土地）的土地屬非自治<ref>{{cite web |url=http://www.bushtel.nt.gov.au/northern_territory/region1 |title=存档副本 |accessdate=2009-07-24 |deadurl=yes |archiveurl=https://web.archive.org/web/20090711190142/http://www.bushtel.nt.gov.au/northern_territory/region1 |archivedate=2009-07-11 }} and {{cite web |url=http://www.bushtel.nt.gov.au/northern_territory/region3 |title=存档副本 |accessdate=2009-07-24 |deadurl=yes |archiveurl=https://web.archive.org/web/20090711185910/http://www.bushtel.nt.gov.au/northern_territory/region3 |archivedate=2009-07-11 }}, http://www.censusdata.abs.gov.au/ABSNavigation/prenav/ProductSelect?newproducttype=Community+Profiles&collection=Census&period=2006&areacode=UCL708200&breadcrumb=LP&currentaction=201&action=401</ref>，人口佔領地內的比例為4.0%<ref>http://www.abs.gov.au/ausstats/abs@.nsf/Products/3218.0~2007-08~Main+Features~Northern+Territory?OpenDocument#LOCALGOVERNMENTAREAPOPULATIONS</ref>。

相對之餘，居於[[新南威爾斯州|新南威爾斯州]]非自治區域的人口僅佔0.02%，[[維多利亞州|維多利亞州]]佔0.002%，[[南澳大利亞州|南澳]]佔0.6%。其餘州份未見非自治區域。
