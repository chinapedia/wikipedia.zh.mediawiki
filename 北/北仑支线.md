'''北仑支线'''又名'''北仑铁路'''，是[[中国|中国]][[浙江省|浙江省]][[宁波市|宁波市]]一条货运[[铁路|铁路]]线。线路起自[[萧甬铁路|萧甬铁路]][[宁波站|宁波站]]，终于[[北仑站|北仑站]]，全长37.2千米。铁路始建于1983年12月25日，建成于1984年4月15日。电气化改造于2016年1月4日开工<ref>{{cite web | url=http://zjnews.zjol.com.cn/system/2016/01/04/020976012.shtml | title=宁波穿山港铁路支线开工 同时启动北仑支线电气化改造 | publisher=浙江在线 | date=2016-01-04 | author=胡昊}}</ref>。

== 线路走向 ==
北仑支线从萧甬铁路宁波站出发，与[[甬台温铁路|甬台温铁路]]并行至[[宁波东站|宁波东站]]，此后向东经过宝幢站向东北方向，经过大碶站后向北，终于[[北仑港|北仑港区]]。

== 经过地区 ==
*宁波市[[海曙区|海曙区]]
*宁波市[[鄞州区|鄞州区]]
*宁波市[[北仑区|北仑区]]

== 未来发展 ==
未来，北仑支线将与[[宁波铁路枢纽北环线|宁波铁路枢纽北环线]]相连，并通过[[华峙支线|华峙支线]]连接[[协和石化专用线|协和石化专用线]]、[[穿山港支线|穿山港支线]]、[[大榭支线|大榭支线]]和[[梅山支线|梅山支线]]。
