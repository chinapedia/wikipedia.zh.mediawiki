'''北65線 分子尾－二重埔'''，是位於[[新北市|新北市]]的一條鄉道，北起[[新北市|新北市]][[三重區|三重區]]分子尾，南至[[新北市|新北市]][[三重區|三重區]]二重埔，全長5.675公里。{{r|thb}}

== 路線說明 ==

;'''[[新北市|新北市]]'''
*[[三重區|三重區]]：'''分子尾'''（起點，[[File:TW_THWtp62.png|30px]][[北62線|北62線]]）→ 溪尾街 → [[File:TW_CHW103.svg|23px]][[縣道103號|103線]]岔路 → 後竹圍街 → '''大有'''（[[File:TW_CHW108.svg|23px]][[縣道108號|108線]]岔路） → 頂崁街 → 十號越堤道 → 疏洪一路 → 疏洪五路一段 → 十三號越堤道 → 五谷王北街 → '''二重埔'''（終點，[[File:TW_PHW1a.svg|23px]][[台1線#.E7.94.B2.E7.B7.9A|台1甲線]]岔路）
