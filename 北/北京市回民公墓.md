'''北京市回民公墓'''，位于[[北京市|北京市]][[丰台区|丰台区]][[芦井路|芦井路]]1号，隶属[[北京市回民殡葬管理处|北京市回民殡葬管理处]]<ref>{{Cite web |url =http://www.bjethnic.gov.cn/ssdw/hmbzglc/index.shtml  |title = 北京市回民殡葬管理处 |author =  |date =  |publisher =北京市民族事务委员会  |language =  |accessdate =2017-02-03  |quote =  }}</ref>。

==历史==
1972年，为解决北京市[[回族|回族]]、[[维吾尔族|维吾尔族]]等10个信仰[[伊斯兰教|伊斯兰教]]的少数民族居民的丧葬问题，北京市将原北京市第三人民公墓改为北京市回民公墓，划归[[北京市民族事务委员会|北京市民族事务委员会]]管理<ref name=zgmzb>{{Cite web|url =http://www.mzzjw.cn/zgmzb/html/2015-04/03/content_104973.htm|title =北京回民公墓调查|author =|date =2015-04-03|publisher =中国民族报|language =|accessdate =|quote =}}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

1996年，北京市颁布实施《北京市殡葬管理暂行条例》。同年，[[北京市民政局|北京市民政局]]出台《关于〈北京市殡葬管理暂行条例〉实施中一些具体问题的执行办法》，其中第四条称：“本市少数民族土葬的仅限于回族、维吾尔族、[[哈萨克族|哈萨克族]]、[[乌孜别克族|乌孜别克族]]、[[塔塔尔族|塔塔尔族]]、[[塔吉克族|塔吉克族]]、[[东乡族|东乡族]]、[[撒拉族|撒拉族]]、[[保安族|保安族]]和[[柯尔克孜族|柯尔克孜族]]。”<ref name=zgmzb/>

北京市回民公墓为北京市上述10个信仰伊斯兰教的少数民族居民提供殡葬服务，并为在北京逝世的外地穆斯林以及外国穆斯林提供就地安葬服务。<ref>{{Cite web |url = http://www.seac.gov.cn/art/2010/3/11/art_7194_193223.html |title = 北京市芦井回民公墓 |author =  |date = 2010-03-11 |publisher = 中华人民共和国国家民族事务委员会 |language =  |accessdate =  |quote =  }}</ref>。

到2010年代初，北京市回民公墓的423.55亩墓穴用地已快用完。为此，2013年北京市财政拨出8.3亿元人民币，在北京市回民公墓周边又扩建390亩墓穴用地<ref name=zgmzb/>。根据2013年1月[[北京市发展和改革委员会|北京市发展和改革委员会]]网站发布的《关于批准北京市回民公墓扩建工程项目建议书的函》，北京市回民公墓扩建项目在[[丰台区|丰台区]][[长辛店镇|长辛店镇]]，东到现状回民公墓，西到山丘与沟西村，北到[[京原路|京原路]]，南到规划河道北侧的隔离带<ref>{{Cite web|url =http://zhengwu.beijing.gov.cn/gzdt/bmdt/t1328349.htm|title =回民公墓扩建项目配套设施开工|author =|date =|publisher =首都之窗|language =|accessdate =2017-02-03|quote =}}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。
