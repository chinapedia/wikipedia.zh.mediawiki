'''北极村'''位于中国[[黑龙江省|黑龙江省]][[漠河市|漠河市]][[北极镇_(漠河县)|北极镇]]，號稱中国最北的城镇，不過其實仍然次於[[北紅村|北紅村]]。北极村位于北纬 53°28′45.61″，东经122°21′36.88″（神州北極碑經緯度），是中国唯一观测[[北极光|北极光]]的最佳地点。中国“[[北方第一哨|北方第一哨]]”北极哨所处于北极村。<ref>http://ccdv.people.com.cn/GB/66982/5519141.html</ref>

== 地理特色 ==

因北极村处于[[北纬|北纬]]53°以上的所谓[[高纬度地区|高纬度地区]]（尚不足通常定义的60°），所以在夏季白天通常长达17个小时以上，而[[冬季|冬季]]则夜晚时间长，是北极村的一大特色。

北极村冬季平均气温在-30℃，极端最低气温在-50℃以下。<ref>http://www.mohe.ccoo.cn/gaiK/dili.asp </ref><ref>http://www.china.com.cn/chinese/TR-c/459719.htm</ref>

此外，北极村的“中国最北”仅为文化上的象征意义。实际上，中国地理最北点在[[图强镇|图强镇]]北端的乌苏里浅滩附近，黑龙江中心航道（北纬53°33′37″，东经123°16′12″），位于北极村东偏北约61公里。而中国地理最北村镇则是[[北红村|北红村]]。

== 白夜现象 == 
北极村不在[[北极圈|北极圈]]内，因此不会出现[[极昼|极昼]]。但由于地处北纬49.34度以北，每年[[夏至|夏至]]（6月21日）前后约一个月的时间段内，在晴朗的夜晚，可观测到[[白夜|白夜]]现象，[[曙暮光|天光]]整晚可见。其中北京时间23:52为光线最暗的时刻，此时太阳位于地平线下的角度最大，约为-13°，处于所谓的“天文曙暮光”阶段；而且和低纬度地区居民的日常经验不同，此时天光中心在正北方向。

此外，白夜时段常常可见[[夜光云|夜光云]]。夜光云是中高纬度地区特有的现象。

== 北极光争议 ==
北极村将每年的夏至当天定为“极光节”，并在旅游文案中声称该段时间内可见[[北极光|北极光]]，并常常将[[白夜|白夜]]和[[夜光云|夜光云]]当作极光对外宣传。虽然在整个中国范围内，漠河是最有可能看到极光的，但观测效果远不如靠近极地的地区，例如美国[[阿拉斯加|阿拉斯加]]、[[加拿大|加拿大]]育空地区、[[冰岛|冰岛]]、[[挪威|挪威]]等，并且需要在冬季前往（夏季为极昼）。从美国空间天气预报中心所公布的极光观测指数看，漠河Kp指数为9，概率上讲即每11年仅有4天才有可能被肉眼直接观测到。事实上，漠河气象站自1957年建立以来到2010年，仅观测到58次北极光<ref>http://www.airitilibrary.com/Publication/alDetailedMesh?docid=hljqx201601015</ref>，几率极低。

此外，极光一年四季都有可能发生，而夏至由于昼长夜短，反而不利于观测。
