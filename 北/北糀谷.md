'''北糀谷'''（{{jpn|j=北糀谷|hg=きたこうじや|rm=Kita-kōjiya|f=y}}）是[[東京都|東京都]][[大田區|大田區]]的町名。現行行政地名為北糀谷一丁目與北糀谷二丁目。2013年8月1日為止的[[人口|人口]]有4,584人<ref>[http://www.city.ota.tokyo.jp/kuseijoho/suuji/jinkou/setai_jinkou/oota_suji2508.html 大田区ホームページ：平成25年8月1日現在]</ref>。[[郵遞區號|郵遞區號]]為144-0032。

== 地理 ==
位於[[東京都|東京都]][[大田區|大田區]]東南部。北鄰[[大森中|大森中]]。東至[[國道131號|國道131號]]（[[產業道路|產業道路]]），接[[大森南|大森南]]。南至[[呑川|呑川]]，接[[西糀谷|西糀谷]]。西鄰[[東蒲田|東蒲田]]。町域內為[[住宅地|住宅地]]與工業用地。

== 歷史 ==
1969年（昭和44年），北糀谷町實施住居表示，大部分町域更名為'''北糀谷'''。
=== 地名由來 ===
因為屬舊麴谷村北部而得名。

== 交通 ==
町域內沒有鐵路車站，可利用西方的[[京急本線|京急本線]][[梅屋敷站_(東京都)|梅屋敷站]]與[[京急蒲田站|京急蒲田站]]，還可利用[[大森站_(東京都)|大森站]]與[[蒲田站|蒲田站]]出發的巴士路線。
