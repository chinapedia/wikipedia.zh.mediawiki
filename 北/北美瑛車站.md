'''北美瑛車站'''（{{jpn|j=北美瑛駅|hg=きたびえいえき|rm=Kita-biei eki|f=y}}）是一由[[北海道旅客鐵道|北海道旅客鐵道]]（JR北海道）所經營的[[鐵路車站|鐵路車站]]，位於[[日本|日本]][[北海道|北海道]][[上川郡|上川郡]][[美瑛町|美瑛町]]境內，是[[地方交通線|地方交通線]][[富良野線|富良野線]]沿線一個無人駐守的車站，由[[北海道旅客鐵道旭川支社|JR北海道旭川支社]]負責管轄。

== 車站構造 ==
北美瑛是個只設有[[側式月台|側式月台]]一座、單線配置的地面車站，且其月台長度僅能容納單節列車。月台旁的[[候車室|候車室]]因為空間太小無法設置[[公共廁所|公共廁所]]{{R|JRH}}，雖然車站本身設備簡陋，但站前的廣場卻有經過作為裝飾用途的花圃。

== 車站周圍 ==
車站本身位於三條河流互相交匯所形成的三角地帶中，附近大多為農田，車站正面有北海道道213號通過。在車站南側約300公尺處、北海道道213號與543號交匯的路口，有一個小型的農村聚落（下宇莫別）。而車站東南方向隔著北海道道213號，則為一片狹長狀朝東方蔓延的[[森林|森林]]。

* [[國道237號|國道237號]]
* 宇莫別簡易郵局
* 美瑛町立美進小學
* {{Link-ja|道北巴士|道北バス}}「宇莫別入口」巴士站

== 歷史 ==
* 1958年（[[昭和|昭和]]33年）3月25日：[[日本國有鐵道|日本國有鐵道]]北美瑛車站開始營運，提供客運服務。
* 1987年（昭和62年）4月1日：配合[[日本國鐵分割民營化|日本國鐵分割民營化]]，車站的所有權與業務由JR北海道承接。
