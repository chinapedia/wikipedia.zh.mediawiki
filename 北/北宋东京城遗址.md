'''东京城遗址'''[[中国|中国]][[北宋|北宋]]都城[[东京|东京]][[开封府|开封府]][[汴梁|汴梁]]城的遗址<ref>{{cite web|author=李合群|title=北宋东京布局研究|url=http://www.cnki.net/KCMS/detail/detail.aspx?QueryID=2&CurRec=1&recid=&filename=2005139142.nh&dbname=CDFD9908&dbcode=CDFD&pr=&urlid=&yx=&v=MDEyNjRHN0s3RjlESXJaRWJQSVI4ZVgxTHV4WVM3RGgxVDNxVHJXTTFGckNVUkw2ZmIrWm9GQ25rVjd2QVYxMjc=5yWTlGWjRSOGVYMUx1eFlTN0RoMVQzcVRyV00xRnJDVVJMNmZiK1pvRnl2Z1ZMN04=|accessdate=2005年}}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。北宋时以五代[[开封府|开封府]]为基础，进行了大规模扩建。使东京成为与[[洛阳|洛阳]]一样的帝都规模。

== 历史沿革 ==
[[天祐_(唐朝)|天祐]]四年（907年），[[朱温|朱温]]代唐称帝，建国号梁，将其所在的[[汴州|汴州]]升为开封府，建名东都。改汴州使治所为建昌宫，并对其进行改建，成为北宋东京城的范本。[[開平_(後梁)|开平]]三年（909年），[[后梁|后梁]]迁都洛阳。至936年〈[[乾化|乾化]]三年至[[龙德|龙德]]三年（913-923年）除外〉，洛阳代开封成为行政中心，降开封府为汴州宣武军。[[天福_(后晋)|天福]]二年（937年），[[石敬瑭|石敬瑭]]东巡汴州，以其衙城为行阙，改称大宁宫。此后历[[后汉|后汉]]、[[后周|后周]]开封保持了东京的地位。

== 北宋都城 ==
[[建隆|建隆]]三年至[[乾德_(北宋)|乾德]]四年（962-966年），[[宋太祖|宋太祖]]对开封皇宫进行重建。先后建大庆、文德、紫宸、垂拱四大殿，[[崇文院|崇文院]]、六阁、景福和国史院、都堂等建筑。新皇城东西宽约1050米，南北长约1090米（合七里余）。后虽又进行改建，而皇城布局未作重大变化。[[政和_(北宋)|政和]]三年（1113年），[[宋徽宗|宋徽宗]]新建延福宫，皇城范围增至九里十三步。政和七年（1117年），建[[艮岳|艮岳]]。

== 现况 ==
[[靖康|靖康]]二年（1127年）[[金朝|金]]灭北宋。[[貞元_(金)|贞元]]三年（1155年），北宋故宫毁于火。[[正隆|正隆]]三年至六年（1158-1161年），金重建开封皇宫，改北宋东京为金南京。[[明朝|明朝]]改金皇宫为周王府。古城在[[崇祯|崇祯]]十五年（1642年）和[[道光|道光]]二十一年（1841年）黄河两次洪水中被彻底毁坏。
