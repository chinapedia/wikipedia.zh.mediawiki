'''北京交通大学附属中学'''，简称'''北京交大附中'''或'''交大附中'''，是一所隶属于[[海淀区|海淀区]]教委的综合性[[中学|中学]]。现分为初、高中部，其中高中部被评为[[北京市示范高中|北京市示范高中]]。学校位于著名历史遗迹大钟寺南侧，毗邻交通干线北三环路。创建于1957年，是一所海淀区教委直接领导的区属重点中学，并于2004年获得了北京市示范高中校的称号。
北京交大附中占地共65亩，分南北两个校区，现有初、高中76个教学班，3300多名学生。

== 历史 ==

== 地理位置 ==
分南、北两个校区。北校区位于[[北京市|北京市]][[海淀区|海淀区]][[皂君庙路|皂君庙路]]12号，[[邮编|邮编]]为100081，为初一初二、高一高二年级所在校区；南校区位于北京交通大学校园内，为毕业年级所在校区。

== 特色 ==
