'''北宋镇'''是[[中华人民共和国|中国]][[山东省|山东省]][[东营市|东营市]][[利津县|利津县]]下辖的一个镇。

== 区域概况 ==
北宋镇位于县域西南部，总面积106平方公里，耕地面积7.3万亩，辖72个行政村，总人口3.7万人。地理位置优越，南临黄河、西接滨州，国道220、永莘路横穿全境；油气资源丰厚，胜利油田滨南采油厂二、三矿均在北宋境内。2011年，全镇经济社会实现了平稳较快发展，完成地方财政收入2176万元，农民人均纯收入9056元，固定资产投资达6.2亿元。

== 人口 ==

2010年中国第六次人口普查时，北宋镇共有人口34309人。共有家庭10811户，平均每户3.15人。14岁以下的少年儿童共5179人，占总人口15.09%；15-64岁人口共25269人，占总人口73.65%；65岁及以上的老年人共3861人，占总人口的11.25%。男性共计17191人，占总人口50.10%；女性共计17118，占总人口49.89%。本地居住的人口中，拥有本地户籍的人口为33378人，占97.28%。<ref name="人口普查2010">{{cite web| url =http://www.yearbookchina.com/navipage-n2013030152000017.html| title =《中国2010年人口普查分乡、镇、街道资料》| author =中国国务院人口普查办公室、中国国家统计局人口就业统计司等| year =2010| website =统计年鉴分享平台| page =表15 山东省乡、镇、街道人口| format =收费| access-date =2018-07-19| archive-url =https://web.archive.org/web/20180719084356/http://www.yearbookchina.com/navipage-n2013030152000017.html| archive-date =2018-07-19| dead-url =no}}</ref>
