'''劝业场'''，位于[[北京市|北京市]][[西城区|西城区]][[廊房头条|廊房头条]]17号，是一家老字号商场<ref name=ifeng>[http://news.ifeng.com/mainland/detail_2011_12/05/11095422_0.shtml 北京百年劝业场将重现，凤凰网，2011年12月05日]</ref><ref name=jhw>[http://epaper.jinghua.cn/html/2012-05/19/content_790788.htm 北京劝业场年底恢复原貌，京华网，2012-05-19] {{webarchive|url=https://web.archive.org/web/20150510033638/http://epaper.jinghua.cn/html/2012-05/19/content_790788.htm |date=2015-05-10 }}</ref><ref name=xhw>[http://www.bj.xinhuanet.com/bjpd_sdwm/2010-07/13/content_20324826.htm 老北京劝业场筹划原貌重张，新华网，2010-07-13] {{webarchive|url=https://web.archive.org/web/20120510112357/http://www.bj.xinhuanet.com/bjpd_sdwm/2010-07/13/content_20324826.htm |date=2012-05-10 }}</ref><ref name=sina>[http://bj.house.sina.com.cn/biz/news/2010-07-14/083115260.html 北京劝业场有望重新开张，新浪，2010年07月14日]</ref><ref name=fzwb>[http://www.fawan.com.cn/html/2012-02/15/content_352506.htm 百岁北京劝业场 整装待变，法制晚报，2012-02-15]{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

== 历史 ==
“京师劝工陈列所”于1905年由[[清政府|清政府]][[農工商部|商部]]创办，是官办工艺局产品展销馆，设在[[廊房头条|廊房头条]]。1908年（光绪三十四年），陈列所遭火灾，随后搬迁到[[广安门内大街|广安门内大街]]与[[牛街|牛街]]交叉路口的东北角，现[[北京市回民学校|北京市回民学校]]西院。1912年，陈列所改为“商品陈列所”。1914年在[[廊房头条|廊房头条]]原址重建，不久再次遭火灾焚毁。1918年重建，而后第三次遭焚毁。1923年，该所在[[廊房头条|廊房头条]]原址再度重建开业，此次重建的建筑留存至今，即如今的劝业场旧址。1928年，该所更名为“工商部国货陈列馆”，馆址迁到[[正阳门|正阳门]]箭楼上<ref name=fzwb/><ref name=xhw/><ref name=qlw>[http://beijing.qianlong.com/3825/2012/02/15/3042@7703889.htm 大栅栏劝业场年内恢复原貌，千龙网，2012-02-15]</ref>。

1936年后，政府将其划归北平市政府，在[[廊房头条|廊房头条]]原址重建，称“北平市国货陈列馆”，又名“劝业场”<ref name=fzwb/>。“劝业场”意为“劝人勉力，振兴实业，提倡国货”。规定私人可在此租地设摊，但仅许卖国货，禁止卖洋货<ref name=sina/>。但此后劝业场遭多次火灾，逐渐萧条；日本占领时期仅能勉强维持<ref name=fzwb/>。

劝业场顾名思义“劝人勉力、振兴实业”，主营商业百货。根据史料记载，劝业场大楼内曾设有百余个摊位，吃、穿、用俱全，另外楼内还设有电影院、剧场、舞厅、台球厅，所以劝业场集商贸、餐饮、娱乐为一体<ref name=xhw/>。当时，一层出售日用百货，二层出售文物及艺术品，三层设有几家照相馆、理发馆，四层是一个小型剧场<ref name=ifeng/>。

1949年之后，劝业场一度关张。1956年[[公私合营|公私合营]]后，改为国营商场，主要经营珠宝玉器、金属器皿、刺绣、丝织品、棉麻织品、土特产品等商品，还出售日用百货、皮货、五金电器等。1975年，劝业场被改为“新新服装店”，共有三层楼，营业面积4000余平方米，以款式新颖、品种齐全而闻名全北京。2000年左右，又被改作“新新宾馆”<ref name=sina/><ref name=ifeng/>。由于被宾馆使用，所以劝业场大楼被被分隔出许多房间，而且外观已被广告牌等物遮盖<ref name=jhw/>。

1995年，“劝业场旧址”被列为[[北京市文物保护单位|北京市文物保护单位]]。2006年，劝业场旧址作为“大栅栏商业建筑”的一部分，被列为[[第六批全国重点文物保护单位|第六批全国重点文物保护单位]]。

在21世纪初进行的[[前门大街|前门大街]]及[[大栅栏|大栅栏]]的大规模改造中，劝业场处在“大栅栏C地块”。该地块位于正阳门西南侧，地处[[北京历史文化保护区|北京历史文化保护区]]，东到[[珠宝市街|珠宝市街]]，西到[[煤市街|煤市街]]，南到[[廊房二条|廊房二条]]，北到[[前门西河沿街|前门西河沿街]]，总占地面积大约3.74公顷。以劝业场旧址为主的核心文物保护区是“大栅栏C地块”保护整治的重点。2011年底，劝业场旧址的腾退整治工作启动<ref name=jhw/>。此次由国有企业“北京大栅栏永兴置业有限公司”负责该项文物建筑的修缮。按照规划，2012年内劝业场将恢复原貌<ref name=qlw/>。

2013年2月6日，由于北京房地集团有限公司未经[[北京市人民政府|北京市人民政府]]批准，擅自对全国重点文物保护单位劝业场旧址进行修缮，[[北京市文物局|北京市文物局]]依法对该单位处以罚款人民币20万元的行政处罚，并责令其依法办理修缮手续<ref>[http://www.bjww.gov.cn/2013/12-13/1386903080031.html 北京市文物局2013年一季度文博事业大事记，北京文博，2013-12-13] {{webarchive|url=https://web.archive.org/web/20160801042300/http://bjww.gov.cn/2013/12-13/1386903080031.html |date=2016-08-01 }}</ref>。

2014年9月，在“2014北京国际设计周”期间，劝业场作为“大栅栏会馆”正式对外开放。劝业场被开辟为文化艺术中心，举办各种艺术展览，同时也具备商业功能，引入服饰、娱乐、饮食等方面的知名品牌。修复后的劝业场内，大玻璃屋顶、大理石地砖、白色雕花楼梯尽显西洋气息<ref>[http://money.163.com/14/0929/14/A7AM4RUQ00254TI5.html 700多年金融老街“重生”：老北京劝业场全新亮相，网易，2014-09-29]</ref><ref>[http://bj.people.com.cn/n/2014/0919/c82840-22364375.html 百年劝业场成“大栅栏会馆” 西城区发布大栅栏新规划，人民网，2014-09-19]</ref>。

2017年，[[北京坊|北京坊]]建筑集群落成，劝业场成为北京坊的一部分<ref>{{Cite web |url = http://www.bj.xinhuanet.com/bjyw/2017-01/17/c_1120325244.htm |title =北京坊：打造北京文化新地标  |author =  |date = 2017-01-17 |website = 新华网 |language =  zh |accessdate =  |quote =  }}</ref><ref>{{cite web |url = http://www.sohu.com/a/196098599_211987 |title = 大栅栏历史文化名城保护提升探索与实践展 |author =  |date = 2017-10-03  |website = 搜狐 |language = zh |accessdate =  |quote =  }}</ref>。

==建筑==
劝业场建筑始建于1905年前后，现存的建筑则建于1923年，是北京首幢大型综合商业楼，也是北京第一家带厢式电梯、游乐场的大楼，曾被誉为“京城商业第一楼”<ref name=fzwb/><ref name=xhw/>。

劝业场原来外观为四层，内部实为三层，另外有地下一层。该楼为钢筋混凝土砖混结构加钢屋架，在当时属于最先进的建筑技术。<ref name=fzwb/>一层的正门位于[[廊房头条|廊房头条]]北侧。二层和三层采用壁柱、窗套、阳台装饰立面，檐上作女儿墙。建筑背面位于[[前门西河沿街|前门西河沿街]]上，看似体量较小，但非常规整，采用古希腊的[[爱奥尼柱式|爱奥尼柱]]、花瓶栏杆阳台、圆拱形山花等西洋古典装饰。劝业场内部纵深方向设有三个大厅，四周是三层回廊，设开敞式商店，内部装修极为豪华。房顶设有巨大的玻璃天窗，以便为营业厅采光<ref name=xhw/>。

2014年修缮完工后，劝业场外立面保留民国时期建筑风貌，内部加固结构，恢复了原有的三个中庭，在保留原貌的基础上加装符合文化艺术中心功能需求的设施及装饰<ref name=chuancheng>[http://bjrb.bjd.com.cn/html/2013-11/18/content_126083.htm 传承老街历史韵味 展现胡同现代魅力，北京日报，2013-11-18] {{webarchive|url=https://web.archive.org/web/20141006131629/http://bjrb.bjd.com.cn/html/2013-11/18/content_126083.htm |date=2014-10-06 }}</ref>。
