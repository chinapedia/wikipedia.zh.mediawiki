'''北京大成路九号'''（'''中央军委联合参谋部警卫局大成路招待所'''），位于[[北京市|北京市]][[丰台区|丰台区]][[大成路|大成路]]9号，隶属[[中央警卫局|中央军委联合参谋部警卫局]]。

==简介==
中国人民解放军总参谋部警卫局大成路招待所（北京大成路九号）<ref name=diyi>{{Cite web |url = http://www.ccgp.gov.cn/cggg/dfgg/zbgg/201701/t20170117_7867097.htm |title = 北京地区（城六区）2016-2018年度会议定点单位协议采购项目中标公告：006中标公告附件1：第一包中标供应商名单 |author =  |date = 2017-01-17 |publisher =中国政府采购网  |language =  |accessdate =  |quote =  }}</ref>，于2007年开业。2016年，[[中国人民解放军总参谋部|中国人民解放军总参谋部]]撤销，该招待所更名为中央军委联合参谋部警卫局大成路招待所（北京大成路九号）。

大成路九号在经营管理上借鉴军事化管理方式。2012年“全国[[两会|两会]]”期间，大成路九号接待[[全国人民代表大会|全国人民代表大会]]江苏和甘肃代表团，这是大成路九号接待的首批“全国两会”代表团。会议闭幕后，两代表团均向大成路九号发来感谢信。2013年，大成路九号接待了四川代表团。2014年，接待了河北和重庆代表团。2015年，大成路九号退出“全国两会”接待工作<ref>{{Cite web |url =http://news.163.com/15/0228/20/AJIODKH000014SEH.html  |title = 今年全国人大代表政协委员开会时住哪儿 |author =  |date = 2015-02-28 |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref>。

2014年，[[西安翻译学院|西安翻译学院]]与北京大成路九号签订了校企合作协议，成立“北京大成路九号·西安翻译学院大成班”。2014年11月3日，西安翻译学院举行了北京大成路九号就业欢送会暨“北京大成路九号·西安翻译学院2014年大成班”成立仪式<ref>{{Cite web |url =http://www.huaue.com/mxdt2014/2014115170042.htm  |title =西安翻译学院举行北京大成路九号就业欢送会暨大成班成立仪式  |author =  |date =  2014-11-05|publisher =华禹教育网  |language =  |accessdate =  |quote =  }}</ref>。
