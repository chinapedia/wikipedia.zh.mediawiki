== 歷史 ==
1895年4月17日，於[[甲午戰爭|甲午戰爭]]中戰敗的[[大清帝國|大清帝國]]和[[大日本帝國|大日本帝國]]簽訂《[[馬關條約|馬關條約]]》，清朝割讓臺灣以及附屬島嶼給與日本。不過早在1月14日，日本政府便以[[無主地|無主地]]先佔原則宣布正式將[[釣魚臺|釣魚臺]]以及周遭的島嶼編入[[沖繩縣|沖繩縣]]管理範圍，而這也包括作為釣魚臺列嶼之一的北小島<ref>{{Cite web |url= http://www.economist.com/news/christmas/21568696-behind-row-over-bunch-pacific-rocks-lies-sad-magical-history-okinawa-narrative |title= The Senkaku or Diaoyu Islands Narrative of an empty space |author=  |date= 2012年12月22日 |publisher= 《[[經濟學人|經濟學人]]》 |language= en |accessdate= 2013年2月11日 |quote=  }}</ref>。1896年時，[[古賀辰四郎|古賀辰四郎]]向日本政府免費租借[[釣魚臺|釣魚臺]]、[[黃尾嶼|黃尾嶼]]、北小島和[[南小島|南小島]]，並且雙方同意在租借30年後改以付費的方式繼續出借土地。然而1932年7月15日，古賀辰四郎的兒子古賀善次便向日本政府提議繼續購買北小島和南小島等土地，而更早之前還付費買下了釣魚臺和黃尾嶼。

1945年[[第二次世界大戰|第二次世界大戰]]正式宣告結束後，[[美國|美國]]於1946年2月2日開始對北緯30度以下的日本領土實施軍事統治<ref name = "共同通訊社">{{Cite web |url= http://www.japantimes.co.jp/text/nn20101208f2.html |title= Ishigaki fishermen fret over Senkaku encroachment |author= [[共同通訊社|共同通訊社]] |date= 2010年2月17日 |publisher= 《[[日本時報|日本時報]]》 |language= en |accessdate= 2012年2月17日 |quote=  }}</ref>。而在1952年4月28日《[[舊金山和約|舊金山和約]]》正式生效以後，北小島則改歸為[[琉球政府|琉球政府]]進行管理。在1970年7月時，琉球政府開始於釣魚臺、黃尾嶼、[[赤尾嶼|赤尾嶼]]、北小島和南小島等釣魚臺列嶼的島嶼上架設禁止非法登陸的警告牌<ref>{{Cite web |url= http://www.y-mainichi.co.jp/news/14231/ |script-title= ja:尖閣諸島「警告板の設置作業者を」　与那国の関係者探す |author=  |date= 2009年8月15日 |publisher= 《{{Link-ja|八重山每日新聞|八重山毎日新聞}}》 |language= ja |accessdate= 2012年2月17日 |quote=  }}</ref>。1972年5月15日時，美國決定連同琉球群島以及釣魚臺列嶼都[[沖繩返還|歸還給]]日本政府進行管理<ref>{{Cite web |url= http://select.nytimes.com/gst/abstract.html?res=F20714FB3D591A7493C3A8178AD95F458785F9&scp=44&sq=1972+okinawa&st=p |title= SENATE ENDORSES OKINAWA TREATY; Votes 84 to 6 for Island's Return to Japan |author= John W. Finney |date= 1971年11月11日 |publisher= 《[[紐約時報|紐約時報]]》 |language= en |accessdate= 2012年2月17日 |quote=  }}</ref>。然而在美國決定將釣魚臺列嶼移交給日本並且讓後者展開[[實際統治|實際統治]]後，這一項決定馬上促使得[[中華人民共和國|中華人民共和國]]與[[中華民國|中華民國]]紛紛主張包含北小島在內的島嶼皆為中國固有的領土；其中中華人民共和國和中華民國政府皆表示日本必須歸還因為《馬關條約》而被割讓的釣魚臺列嶼，但是對此要求日本政府則認為釣魚臺列嶼本身並不存在有主權爭議<ref>{{Cite web |url= http://www.mofa.gov.tw/official/Home/Detail/0105968e-9474-4674-8845-93998d670c14?arfid=7f013c3f-f130-44a9-905f-84cbaba2eca6&opno=907477b5-1d95-4205-a89d-320ed4806d4b |title= 外交部嚴正抗議日本政府將釣魚台列嶼赤尾嶼附近的「北小島」登載為「國有財產」 |author=  |date= 2012年3月27日 |publisher= [[中華民國外交部|中華民國外交部]] |language= zh-hant |accessdate= 2012年2月17日 |quote=  }}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref>{{Cite web |url= http://news.sina.com.cn/c/2012-09-11/080625146354.shtml |title= 外交部就日本“购买”钓鱼岛发表声明(全文) |author= SN010 |date= 2012年9月11日 |publisher= [[新浪|新浪]] |language= zh-hans |accessdate= 2012年11月16日 |quote=  }}</ref><ref>{{Cite web |url= http://news.sina.com.cn/pl/2012-09-11/031625143759.shtml |title= 人民日报：中国钓鱼岛岂容他人肆意“买卖” |author= SN047 |date= 2012年9月11日 |publisher= [[新浪|新浪]] |language= zh-hans |accessdate= 2012年11月16日 |quote=  }}</ref>。

== 地理 ==
北小島其相對位置位在[[西表島|西表島]]北方約160公里、釣魚臺列嶼主島[[釣魚臺|釣魚臺]]東南東方約5公里處，而絕對位置則為[[北緯|北緯]]25度43.6分、[[東經|東經]]123度32.3分左右。整個島嶼寬約540公尺、長約880公尺、海岸線長約3.5公里，而面積則有0.31平方公里。北小島周边包括有[[鸟巢岛|鸟巢岛]]、[[鸟卵岛|鸟卵岛]]、[[小鸟岛|小鸟岛]]等若干小岛礁，並且在南邊隔著寬度約200公尺的[[橄欖門|橄欖門]]與[[南小島|南小島]]的[[拳头岭|拳头岭]]相望。兩者之間的橄榄门[[海峽|海峽]]長約150公尺，過去在[[東海|東海]]遭遇較強風浪時臺灣漁民多會將船隻駛於此水道以躲避風浪<ref>{{Cite web |url= http://news.sina.com.cn/c/2012-09-22/091025229896.shtml |title= 钓鱼岛海域部分地理实体标准名称公布 |author= [[國際在線|國際在線]] |date= 2012年9月22日 |publisher= [[新浪|新浪]] |language= zh-hans |accessdate= 2012年11月16日 |quote=  }}</ref>。北小島最高峰為位於南部、海拔高度135公尺的[[狮峰|狮峰]]，而第二高峰則為位於中部偏北的[[鹰峰|鹰峰]]，其他較高處還有[[孔明石|孔明石]]以及[[莲花石|莲花石]]。北小島島嶼上棲息著各種鳥類，[[高良鐵夫|高良鐵夫]]在1953年8月發現北小島附近海域有大量出被列為[[瀕危物種|瀕危物種]]的[[短尾信天翁|短尾信天翁]]<ref>[http://pinacles.zouri.jp/senkakukenkyuu_ge.htm 尖閣研究 高良学術調査団資料集（下巻）]pp213-250. 尖閣諸島文献資料編纂会編. 2007年10月1日刊</ref><ref>[http://pinacles.zouri.jp/bunken/morita.htm 北小島の洋上に居たアホウドリ―尖閣列島生物調査（1953年8月）で実見―]. 尖閣諸島文献資料編纂会</ref><ref>[http://pinacles.zouri.jp/bunken/zadankai.htm 尖閣諸島調査団座談会]. 尖閣諸島文献資料編纂会</ref>，而在2002年時也有報告指短尾信天翁開始於島嶼上繁殖<ref>{{Cite web |url= http://ryukyushimpo.jp/news/storyid-103214-storytopic-86.html |script-title= ja:アホウドリ繁殖を確認/尖閣諸島北小島にも分布 |author=  |date= 2002年1月8日 |publisher= 《[[琉球新報|琉球新報]]》 |language= ja |accessdate= 2012年2月17日 |quote=  |deadurl= yes |archiveurl= https://web.archive.org/web/20131202223008/http://ryukyushimpo.jp/news/storyid-103214-storytopic-86.html |archivedate= 2013年12月2日 }}</ref>。1970年12月10日，九州、長崎大学合同学術調査團也在北小島發現數隻[[黑腳信天翁|黑腳信天翁]]<ref>沖縄の秘境を探る. 高良鉄夫. 沖縄新報社. 1980年刊. pp106-107</ref>。
