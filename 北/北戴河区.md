[[File:Pei_tai_ho_East_Cliff_1903.jpg|thumb]]
'''北戴河区'''，濒临[[渤海湾|渤海湾]]，是[[中国|中国]][[河北省|河北省]][[秦皇岛市|秦皇岛市]]的一个[[市辖区|市辖区]]。面积为70.3[[平方千米|平方千米]]，[[海岸线|海岸线]]22.5千米，人口6.1万。北戴河属[[温带|暖温带]]、半湿润、[[季风|季风]]型[[大陆性气候|大陆性气候]]，受海洋的调节，又具有的[[海洋性气候|海洋性气候]]特点。森林覆盖率70%以上，暑期平均气温只有24.5℃。

[[清朝|清朝]][[光绪|光绪]]年间，许多住在[[北京市|北京]]的外国人要求在这里建造[[别墅|别墅]]。清光绪二十四年（1898年），清政府正式将北戴河开辟为“各国人士避暑地”。1938年，已有别墅700多栋，中央和许多大型单位在北戴河又新建了200多所疗养院。

== 行政区划 ==
北戴河区辖2个街道、3个镇：

*街道：[[西山街道_(秦皇岛市)|西山街道]]、[[东山街道_(秦皇岛市)|东山街道]]。
*镇：[[海滨镇|海滨镇]]、[[戴河镇|戴河镇]]、[[牛头崖镇|牛头崖镇]]。

2015年7月[[抚宁区|抚宁县]]设区，将牛头崖镇划归北戴河区管辖。<ref>{{cite web|url=http://202.108.98.30/description?dcpid=2015|title=中华人民共和国二〇一五年县级以上行政区划变更情况|accessdate=2018-06-16|publisher=全国行政区划信息查询平台}}</ref>

== 人文 ==
1954年7月26日，[[毛泽东|毛泽东]]在北戴河写了一首称颂北戴河的词——《浪淘沙·北戴河》
 
{{quote|大雨落幽燕，白浪滔天，秦皇岛外打鱼船。
一片汪洋都不见，知向谁边？

往事越千年，魏武挥鞭，东临碣石有遗篇。

== 政治 ==
{{main|北戴河办公制度}}
如同[[俄羅斯|俄羅斯]]的[[索契|索契]]或[[扎維多沃|扎維多沃]]、[[美國|美國]]的[[大衛營|大衛營]]一般，许多[[中華人民共和國|中国]]的[[党和国家领导人|党和国家领导人]]经常到这里避暑、休养，并在此召开一些具有影响力的[[北戴河会议|会议]]，因此北戴河又具有了一定的政治含义。

== 物种保护 ==
{{seealso|北戴河鸟类列表}}
北戴河是[[东亚|东亚]]最重要的[[鸟类迁徙|候鸟]]迁徙通道之一，每年都有大量[[涉禽|涉禽]]和各种其它鸟类迁徙经过北戴河，北戴河[[湿地|湿地]]被[[观鸟|观鸟]]者称为“观鸟麦加”。北戴河鸟类环志站是目前中国[[环志|环志]]鸟类最多的环志站点之一，目前北戴河市辖区内有北戴河市级候鸟保护区一处。
