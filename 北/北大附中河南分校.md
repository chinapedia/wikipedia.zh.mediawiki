'''北大附中河南分校'''是2001年创建的位于[[河南|河南]][[新郑市|新郑市]]的一所[[高级中学|高级中学]]，是河南省人民政府与[[北京大学|北京大学]]省校合作协议中的重点项目，属于[[宇华教育集团|宇华教育集团]]旗下的民办中学，为河南省示范性普通高中、全国先进办学单位和中国百强中学<ref>{{cite web |url=http://www.bdfz-hn.com/ArticleShow.asp?ArticleID=2518 |language= |title=北大附中河南分校2015年高考喜报！ |publisher=北大附中河南分校 |date=2015-06-25 |accessdate= }}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。学校毗邻[[新郑国际机场|新郑国际机场]]，占地218亩，建筑面积9万平米；现有在校生5000多人，教职工500余人。<ref name="简介">{{cite web |url=http://www.bdfz-hn.com/gaikuang/xuexiaogaikuang.asp |language= |title=学校简介 |publisher=北大附中河南分校 |date= |accessdate=2015-08-05 |deadurl=yes |archiveurl=https://web.archive.org/web/20150811135635/http://bdfz-hn.com/gaikuang/xuexiaogaikuang.asp |archivedate=2015-08-11 }}</ref>

== 简介 ==

从2004年至2014年的高考中，学校共培养出10名河南省、[[郑州市|郑州市]][[高考状元|高考状元]]，累计有239名学生考入[[北京大学|北京大学]]、[[清华大学|清华大学]]和[[香港大学|香港大学]]（2009年22人、2010年32人、2011年35人、2012年34人、2013年27人、2014年18人），连续四年考入北大、清华、港大总人数位列全省第一位。<ref name="简介"/>

== 学校文化 ==
*校训：“孝敬、诚信、勤奋、创新”<ref name="简介"/>
*德育思路：“富家寒教”、“铁纪柔情”
*新课堂教学模式：“以学生为主体、以教师为主导”
*优良学风：“自主学习、自主管理”
