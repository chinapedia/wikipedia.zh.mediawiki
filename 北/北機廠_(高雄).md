'''北機廠'''，亦稱'''岡山機廠'''，代號CD3，是[[高雄捷運紅線|高雄捷運紅線]]的兩座[[機廠|機廠]]之一，位於[[高雄市|高雄市]][[岡山區|岡山區]]及[[橋頭區|橋頭區]]兩區交界，並於其中設有[[南岡山站|南岡山站]]。

==概要==
* 位置：北機廠位於高雄捷運紅線北端，跨越[[高雄市|高雄市]][[岡山區|岡山區]]及[[橋頭區|橋頭區]]2個行政區，在[[高雄市岡山文化中心|高雄市岡山文化中心]]東南側<ref>{{Cite web |url= http://www.krtco.com.tw/customer_info/subbusiness_land.aspx |title=高雄捷運土地開發資訊 |language= zh-tw }}</ref>。
* 規模：三級修護廠。
* 面積：34[[公頃|公頃]]，包含維修機廠、車站、道路共24.4公頃及商業開發使用面積為9.6公頃。

==歷史==
* 2008年3月9日，北機廠隨[[高雄捷運紅線|高雄捷運紅線]]通車投入運用。
* 2010年9月19日，因[[中度颱風凡那比_(2010年)|凡那比颱風]]侵襲，北機廠積水達30公分，造成20多個轉轍器進水故障，沾滿污泥無法使用，事後已全數更換。
* 2011年3月22日，高雄捷運與[[和春紀念醫院|和春紀念醫院]]簽約，承租北機廠土地26年，作為醫院及照護機構，投資金額約[[新臺幣|新臺幣]]2億元<ref>{{Cite web |url= http://www.appledaily.com.tw/appledaily/article/headline/20110323/33266272/ |title= 和春醫院租高捷機廠 建安養院 |date=2011-03-23 |language= zh-tw }}</ref>。
* 2012年12月23日，北機廠內的[[南岡山站|南岡山站]]通車<ref>{{Cite web |url=http://www.krtco.com.tw/act_news/flourishing-2_detail.aspx?ID=3d1a9c91-8d83-4908-b54f-b219cc7fcc12 |title=南岡山站12月23日正式通車，R24南岡山站到R22A橋頭糖廠站三個月免費搭乘 |date=2012-12-19 |language= zh-tw }}</ref>。
* 2013年4月25日，和春紀念醫院開工動土，預計於2014年7月落成啟用，將提供岡山、路竹及鄰近北高雄地區優質醫療及安養照護服務<ref>{{Cite web |url= https://www.krtco.com.tw/news/flourishing-1_detail.aspx?ID=dce71bcc-e300-45be-858a-83a65e08597a |title= 和春紀念醫院動土開工推動地方經濟脈動，高捷南北機廠開發案金額上看55億元 |date=2013-04-25 |language= zh-tw }}</ref>。
* 2014年12月31日，和春護理之家落成啟用。
* 2016年7月13日，高雄捷運與高雄醫學院附設醫院簽約，約3.2公頃的土地興建高醫岡山醫院<ref>[http://www.cna.com.tw/news/ahel/201607130224-1.aspx 高醫投資33億元 高捷北機廠打造岡山醫院]</ref>。
* 2017年12月25日，高醫岡山醫院開工動土，預計最快2020年完工<ref>[http://www.chinatimes.com/realtimenews/20171225004841-260405 高醫岡山醫院25日動土 搶攻北高雄醫療商機]</ref>。
* 2018年8月2日，高雄捷運北機廠所在的9.2公頃北機廠開發區，最後4.2公頃土地，昨也宣布招商成功，開發率達百分之百。高雄市政府、高捷公司、達麗米樂公司於高雄市政府簽署三方合約，達麗米樂公司將進駐高捷北機廠開發區，投資新台幣約二十億元打造複合式休閒娛樂購物廣場，基地面積4.2公頃，樓地板面積1.6萬坪的複合式休閒娛樂購物廣場，提供影視娛樂、主題商店、特色餐飲、文創市集等。達麗米樂公司將攜手秀泰影城設立高雄第一個影城據點，將接軌國際引進最新放映設備，並打造高雄最大廳高15米的4K雷射「超級巨幕廳」，容納14至15廳。預計明年動工、2021年開幕<ref>[https://udn.com/news/story/11322/3287602 高醫岡山醫院25日動土 搶攻北高雄醫療商機]</ref><ref>[https://www.krtco.com.tw/news/flourishing-1_detail.aspx?ID=3f3fdf86-0f5a-4bc8-bd2c-e67784a1ca1c 〈新聞稿〉高捷北機廠開發全「達」標 達麗米樂商場進駐]高雄捷運公司</ref>。
