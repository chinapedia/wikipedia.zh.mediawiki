'''北京燕翔谭阁美饭店'''，位于[[北京市|北京市]][[朝阳区_(北京市)|朝阳区]][[将台路|将台路]]甲2号。

==简介==
该酒店的前身是北京燕翔饭店。燕翔饭店于1979年始建，1980年11月3日开业。该饭店由[[北京市旅游发展委员会|北京市旅游局]]与美国联翔旅运有限公司共同投资，“燕翔”一名即由“[[燕京|燕京]]”和美国联翔两个意思组成。1981年，北京燕翔饭店成为北京市旅游局直属企业，1985年被北京市旅游局评为定点涉外饭店，1986年被北京市旅游局评为三星级旅游商务饭店。1992年转隶北京首汽集团公司，1999年改制重组为北京首都旅游集团下属企业，是北京首都国际酒店集团的核心企业。2004年，由首旅建国有限公司管理。2004年被北京奥组委签署为[[2008年北京奥运会|2008年北京奥运会]]接待单位之一<ref>{{Cite web |url = http://job.veryeast.cn/93410 |title = 北京燕翔饭店 |author =  |date =  |publisher =  最佳东方|language =  |accessdate = 2017-03-29 |quote =  }}</ref>。 

北京燕翔饭店曾接待过参加“全国[[两会|两会]]”的[[全国政协|全国政协]]委员<ref>{{Cite web |url =http://news.163.com/15/0228/20/AJIODKH000014SEH.html  |title = 今年全国人大代表政协委员开会时住哪儿 |author =  |date = 2015-02-28 |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref>。

2011年11月，燕翔饭店改扩建项目开工，2015年6月投入运营。燕翔饭店改扩建项目总建筑面积166293平方米，总投资近30亿元人民币。2016年11月27日，[[中国建筑业协会|中国建筑业协会]]公布年度[[中国建设工程鲁班奖|中国建设工程鲁班奖]]入选名单，燕翔饭店改扩建项目入选<ref>{{Cite web |url =http://money.163.com/16/1207/06/C7LNTR4O002580S6.html  |title =  燕翔饭店项目获鲁班奖|author =  |date = 2016-12-07 |publisher =  网易|language =  |accessdate =  |quote =  }}</ref><ref>戴建民、孟祥武，北京燕翔饭店改扩建项目，安装2014(8)</ref>。

2015年6月5日，北京谭阁美饭店管理有限公司北京燕翔谭阁美饭店开业，成为谭阁美在中国地区的旗舰店。北京燕翔谭阁美饭店由[[法国|法国]]朗涛设计团队设计，设有264间客房，以及餐厅、健身房等设施，并提供社交平台<ref>{{Cite web |url = http://news.sina.com.cn/o/2015-06-16/133031956321.shtml |title = 谭阁美品牌旗舰店——北京燕翔谭阁美饭店全新亮相 |author =  |date =2015-06-16  |publisher = 新浪 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://news.163.com/15/0617/02/AS9ERIHD00014AED.html |title = 北京燕翔谭阁美饭店开业 |author =  |date = 2015-06-17 |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref>。
