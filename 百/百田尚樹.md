{{Japanese|japanese=百田 尚樹|kana=ひゃくた なおき|romaji=Hyakuta Naoki}}
'''百田尚樹'''（{{bd|1956年|2月23日|||Hyakuta Naoki}}）[[男|男]]，日本右翼[[作家|作家]]、編劇作家，出生於[[大阪市|大阪市]][[東淀川區|東淀川區]]。2013年以小說《{{link-ja|被稱作海盜的男人|海賊とよばれた男}}》獲得日本[[本屋大賞|本屋大賞]]。
被稱為'''電網右翼的魁首'''。

== 經歷 ==
1956年2月23日，百田尚樹生於大阪市東淀川區。就讀[[同志社大學|同志社大學]]法學部期間，他參加[[朝日放送|朝日放送]]的[[戀愛|戀愛]][[綜藝節目|綜藝節目]]《LOVE ATTACK!》，結識同節目的參加者[[百田理香|百田理香]]並結婚。他在就讀大學第五年時[[退學|退學]]，成為一名撰寫電視及廣播節目[[劇本|劇本]]的「放送作家」。

2006年，他發表了以[[第二次世界大戰|第二次世界大戰]]為題材、描寫一群[[零式戰鬥機|零式戰鬥機]][[飛行員|飛行員]]故事的小說處女作《[[永遠的0|永遠的0]]》，正式出道為小說家。《永遠的0》在2012年銷售突破一百萬本，由[[東寶|東寶]]確定改編電影<ref>{{Cite web |url = http://www.oricon.co.jp/news/ranking/2017604/full/|title = 『永遠の0』、歴代13作目の文庫100万部超え |publisher = [[オリコン|オリコン]]| date = 2012-10-11 |accessdate = 2013-03-19 }}</ref>。

2009年，他的青春小說《[[Box！熱血鬥陣|Box！熱血鬥陣]]》入圍第30屆[[吉川英治文學新人獎|吉川英治文學新人獎]]，並獲第六屆[[本屋大賞|本屋大賞]]第5名、第七屆早午餐BOOK大獎新人獎，熱銷突破40萬冊感動無數日本讀者；改編電影由《[[重金搖滾雙面人|重金搖滾雙面人]]》導演[[李鬥士男|李鬥士男]]執導、[[市原隼人|市原隼人]]主演<ref name=honya>{{Cite news| title = 本屋大賞：百田尚樹さん作家7年目で文学賞を初受賞「直木賞以上の最高の賞」 | newspaper = [[毎日新聞|MANTANWEB]]| date = 2013-04-10| url = http://mantan-web.jp/2013/04/10/20130409dog00m200072000c.html| accessdate = 2013-06-10}}</ref>，於日本上映首週末票房即突破日幣一億八百萬大關，[[朝日新聞|朝日新聞]]、[[讀賣新聞|讀賣新聞]]、[[產經新聞|產經新聞]]、[[週刊文春|週刊文春]]、[[週刊新潮|週刊新潮]]等各大報章雜誌讚不絕口：「本年度No.1的體育青春小說！」

2013年，他以《{{link-ja|名叫海賊的男人|海賊とよばれた男}}》（{{lang|ja|海賊とよばれた男}}）榮獲第十屆本屋大賞<ref>{{Cite news| title = 本屋大賞：百田尚樹さんの「海賊とよばれた男」が受賞 | newspaper = [[毎日新聞|MANTANWEB]]| date = 2013-04-09| url = http://mantan-web.jp/2013/04/09/20130409dog00m200043000c.html}}</ref><ref name=tairiku>{{Cite news | title = 情熱大陸:出す本全てが売れまくる…50歳でデビューした異色の作家、百田尚樹に密着| newspaper = [[毎日新聞|毎日jp]]| date = 2013-06-04| url = http://mainichi.jp/feature/news/20130604org00m200053000c.html| accessdate = 2013-06-10}}</ref>。

目前他是編劇作家，製作許多節目，其中朝日放送綜藝節目《[[偵探！夜間獨家|偵探！夜間獨家]]》曾榮獲[[日本民間影視協會聯盟獎|日本民間影視協會聯盟獎]]首獎。另著有《24號的奇蹟》、《影法師》、《風中的瑪莉亞》、《RING》（暫譯）等書。
