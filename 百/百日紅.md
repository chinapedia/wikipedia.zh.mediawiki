《'''百日紅'''》（{{lang-ja|百日紅}}，さるすべり）是由[[杉浦日向子|杉浦日向子]]執筆的歷史[[日本|日本]][[漫畫|漫畫]]系列，自1983年至1987年連載於{{link-ja|實業之日本社|実業之日本社}}發行的《{{link-ja|漫畫Sunday|漫画サンデー}}》雜誌上。

作品內容描寫江戶時期的風俗與庶民的生活。並融入恋愛、人情、驚悚等多元要素。

最初出版的單行本是共3冊內容，後續由[[築摩書房|築摩書房]]以上下卷方式發行。

== 原作漫畫 ==
=== 內容 ===
漫畫內容是短篇[[單元劇|單元劇]]的形式，採用日本江戶時代實有的浮世繪繪師[[葛飾北齋|葛飾北齋]]、其女兒阿榮（[[葛飾應為|葛飾應為]]），以及年輕時期的池田善次郎（[[溪齋英泉|溪齋英泉]]）等人作為主要人物，描寫他們平時所遭遇的生活百態；劇情並融入戀愛、人情、驚悚、奇聞異事等多項成份。而漫畫的首篇便是以善次郎告知北齋鎮上出現了一顆被斬下、模樣不自然的女人頭作為開始。

=== 角色 ===
==== 主要人物 ====
* {{nihongo|'''[[葛飾北齋|葛飾北齋]]'''|葛飾北斎}}
: 俗稱鐵藏，是個雖有名氣但收入不多的55歲浮世繪繪師，平時住在長屋裡一間窄小雜亂工作室。
: 過著與女兒阿榮一同接案繪畫的生活，常不拘細節、憑著自己的喜好來行事。

* {{nihongo|'''[[葛飾應為|阿榮]]'''|お栄}}
: 北齋的女兒，因從小受父親北齋影響而習得一定程度的繪畫技巧，常接手替北齋不想作的案件代筆。
: 有著如男性般地強勢個性，也與北齋同樣常不在乎細節。

* {{nihongo|'''[[溪齋英泉|池田善次郎]]'''|池田 善次郎}}
: 寄住在北齋工作間的青年，為往後的繪師溪齋英泉。平時向北齋學習繪畫技法。
: 個人雖尚未闖出名氣，但因平時喜好女性及在聲色場所玩樂，其經驗讓他繪製的[[春宮圖|春宮圖]]有獲得一些買方的好評。

==== 其他 ====
* {{nihongo|'''[[魚屋北溪|岩窪初五郎]]'''|岩窪初五郎}}
: 是出師於北齋門下的弟子；為往後的繪師魚屋北溪。為個性端正溫和的男性，是阿榮心儀的對象。

* {{nihongo|'''[[葛飾北明|井上政女]]'''|井上政女}}
: 北齋門下的女弟子，個人的名號為葛飾北明。在作品裡被描述為會向同門的岩窪初五郎搭訕、以及私下與北齋互相幽會的女性。

* {{nihongo|'''[[歌川國直|歌川國直]]'''|歌川国直}}
: 為[[歌川豐國|歌川豐國]]門下的年輕繪師，雖隸屬與葛飾北齋對立的歌川派人物，但因自身欣賞北齋的藝術造詣而私下與他及阿榮等人有所往來。

* {{nihongo|'''[[歌川國芳|歌川國芳]]'''|歌川国芳}}
: 寄居在歌川國直家中與他學習繪畫並時常一同行動的青年。

* {{nihongo|'''都八造'''|都八造}}
: 帶著眼鏡，常前來拜訪北齋的表演家。

* {{nihongo|'''萬字堂'''|萬字堂}}
: 接洽北齋、阿榮或萬次郎等人繪畫案件的出版商老闆。

* {{nihongo|'''琴女'''|こと}}
: 北齋的第二任妻子，阿榮的生母。平時不待在北齋和阿榮的工作室而一人居住。

* {{nihongo|'''多吉郎'''|多吉郎}}
: 阿榮的弟弟，長大後以[[御家人|御家人]]的身分成為他人的養子，是身材高大個性憨直的青年。

* {{nihongo|'''阿猶'''|お猶}}
: 阿榮的妹妹，因天生眼盲因素而被送至寺院裡學習琵琶演奏，有著虛弱的體質。

* {{nihongo|'''小夜衣'''|小夜衣}}
: [[吉原|吉原]]裡有名的[[花魁|花魁]]，被外人流傳著她有晚上入睡脖子便會像妖怪般伸長的傳說。

* {{nihongo|'''吉彌'''|吉弥}}

2014年4月，製片方宣布製作長篇[[動畫電影|動畫電影]]。電影名為《'''百日紅 〜Miss HOKUSAI〜'''》（{{lang-ja|さるすべり ミス・ホクサイ}}）。

導演由拍攝過《[[河童之夏|河童之夏]]》、《[[Colorful_多彩奇幻之旅|Colorful 多彩奇幻之旅]]》的[[原惠一|原惠一]]執導，動畫製作室為[[Production_I.G|Production I.G]]。主角「[[葛飾應為|阿榮]]」由演員[[杏_(演員)|杏]]配音，「[[葛飾北齋|葛飾北齋]]」由[[松重豐|松重豐]]配音，2015年5月9日上映<ref>{{Cite web |url=http://natalie.mu/comic/news/115198 |title=杉浦日向子「百日紅」宣布製作動畫電影，導演由原惠一指導，隔年公開上映。 |publisher=[[Natalie_(網站)|Natalie]] |date=2014年4月25日  |accessdate=2015年1月13日 |language=ja}}</ref><ref>{{cite web |url=http://natalie.mu/comic/news/136552|title=杉浦日向子原作動畫電影「百日紅 〜Miss HOKUSAI〜」主角阿榮由杏飾演|publisher=[[Natalie_(網站)|Natalie]] |date=2015年1月20日 |accessdate=2015年1月21日 |language=ja}}</ref><ref name=cast>{{cite web|url=http://www.cinematoday.jp/page/N0071075|title=杉浦日向子原作的長篇動畫，公開新追加配音陣容，葛飾北齋由松重豐配音|publisher=シネマトゥデイ|date=2015年2月27日 |accessdate=2015年4月12日 |language=ja}}</ref>。

=== 劇情概要 ===
動畫版劇情是挑選原作漫畫裡各章節內容後；同樣獨立成各單元劇的模式來表現，而部份短篇則有彼此協接、貫穿內容的成份。

另動畫版裡加強阿榮的妹妹阿猶之戲份，以及增加阿榮對她的關懷描寫。其他一些原作漫畫的角色如歌川國芳、或葛飾北明等人雖未被提及，但漫畫裡葛飾北明搭訕岩窪初五郎、讓阿榮內心吃味的畫面，則有改編成動畫裡的一小幕。

=== 花絮 ===
在原惠一拍攝完《[[Colorful_多彩奇幻之旅|Colorful 多彩奇幻之旅]]》後，某日[[Production_I.G|Production I.G]]動畫室的社長石川光久向原惠一表示過去公司裡曾有將杉蒲百合子《百日紅》改編成動畫電影的念頭、但之後無疾而終，便希望原惠一接手先前無法完成的計畫。之後原惠一答應嘗試進行將《百日紅》改編動畫，而原惠一年輕時也喜好著杉蒲百合子的{{nihongo|《風流江戶雀》|風流江戸雀}}之類漫畫作品<ref name="Aniko1">{{cite web |url=http://ani-ko.com/08-hara01 |script-title=ja:江戸で生きる人々の日常と非日常を描く『百日紅 〜Miss HOKUSAI〜』監督 原　恵一（前編） |work=Aniko |date=2015年5月14日 |accessdate=2015年10月24日}}</ref>。

因原作漫畫是單元劇形式，原惠一因此需要一個能貫穿整部動畫的主題，便想到原作漫畫中描述阿榮的妹妹阿猶死去之章節《野分》，因此讓阿榮與阿猶之間姊妹關係成為動畫版的重要內容<ref name="Aniko1"/>。

在角色設計方面是請曾在動畫《[[電腦線圈|電腦線圈]]》裡擔任作畫指導的板津匡覽負責，雖然原作漫畫中阿榮並非美形角色，原惠一仍讓板津匡覽將阿榮給予美化，而原作漫畫裡因有著突出下巴的阿榮、而被北齋戲稱為「阿顎」的橋段便沒出現在動畫版裡<ref name="Aniko1"/>。

替此片阿榮與北齋配音的[[杏_(演員)|杏]]和[[松重豐|松重豐]]，為第三度一同飾演父女關係角色，其中前兩次是在連續劇《[[約會～戀愛為何物～|約會～戀愛為何物～]]》及電視廣告上<ref>{{cite web|url=http://www.crank-in.net/game_animation/special/36765|script-title=ja:松重豊、杏は「自慢の娘」 3度の父娘役で"親のような気持ち" |work=Crank-In |date=2015年5月8日 |accessdate=2015年6月3日|language=ja}}</ref>而為片中美型的男妓（[[陰間_(男妓)|陰間]]）吉彌配音的[[入野自由|入野自由]]表示著，在此之前不曉得江戶時代有以「陰間」稱呼男妓一事，也因此私下特別挪出時間查閱相關文獻<ref name=animatetv/>。
