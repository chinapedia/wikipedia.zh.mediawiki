'''百色站'''位于[[中国|中国]][[广西壮族自治区|广西壮族自治区]][[百色市|百色市]][[右江区|右江区]]城东进站大道，建于1995年<ref>{{Cite web |url=http://www.nntlj.com/qtfw/czfc/201707/t20170705_572.html|title=百色站|date=2016-02-15 }}</ref>，为[[南宁铁路局|南宁铁路局]]管辖的[[二等站|二等站]]，离南宁站234公里，离昆明站594公里。经过铁路有[[南昆铁路|南昆铁路]]、[[南昆客运专线|南昆客运专线]]等主要铁路干线。

==历史==
*1995年：百色站建成启用；
*2014年10月：百色站新建综合站房开工建设，于2015年7月底投入过渡性使用。<ref>{{Cite web |url=http://www.bbrtv.com/2015/0703/211959.html|title=百色站新站房本月将投入过渡性使用 年底可通高铁|date=2015-07-03 }}</ref>

== 站外交通 ==
*公交1路：凤凰巷 - 火车站
*公交4路：城西高速路口 - 火车站
*公交8路：火车站 - 森林广场 - 火车站 （环线）
*公交17路：火车站 - 华润超市 - 火车站 （环线）<ref>{{Cite web |url=http://www.nntlj.com/qtfw/czfc/201707/t20170705_572.html|title=百色站|date=2016-02-15 }}</ref>

2016年12月19日，本站在[[凌云县|凌云县]]体育馆设立高铁无轨站（类似机场的“城市候机楼”），在全国为业界首创。旅客可在高铁无轨站内购票（大巴票和动车票）、在无轨站内候车，然后乘坐无轨站的专线大巴前往百色站，返回凌云县的人士也可以在百色站乘坐大巴返回高铁无轨站，此举可以节省不少时间成本和经济成本。而南宁铁路局也将在广西各地市开通高铁的基础上，在没有高铁的县份推行高铁无轨站模式，以带动整个广西更快地融入高铁经济生活圈<ref>{{Cite web |url=http://news.ifeng.com/a/20161220/50445580_0.shtml|title=全国首个高铁无轨站在百色凌云启用！|date=2016-12-20}}</ref>。
