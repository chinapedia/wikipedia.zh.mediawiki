'''百灵杯'''，[[中国|中国]]女子[[围棋|围棋]]比赛（第3届混双赛、5届男女对抗），由[[中国棋院|中国棋院]]和[[贵州|贵州]]百灵企业集团主办。 
各届的比赛方式不同，第1届百灵杯是新老女子棋手对抗，第2届是女子围棋个人赛，第3届是混双赛，第4届是女子新老对抗赛，从第5届开始改为男女超级对抗赛。

== 历届冠军 ==

== 第1届百灵杯 ==
2006年举行，采取新老女子棋手对抗、升降棋格的形式。结果青队6-4胜蓝队。
*蓝队：[[张璇|张璇]]、[[华学明|华学明]]、[[叶桂|叶桂]]
*青队：[[王祥云|王祥云]]、[[李赫|李赫]]、[[曹又尹|曹又尹]]

== 第2届百灵杯 ==
第2届百灵杯是女子围棋个人赛
*决赛：2007年1月25日-27日，[[李赫|李赫]] 2-1 [[叶桂|叶桂]]

== 第5届百灵杯 ==
于2009年举行，百灵杯从第5届开始改为男女超级对抗赛，比赛地点在[[贵州|贵州]]的[[安顺|安顺]]、[[贵阳|贵阳]]和[[遵义|遵义]]。
*男棋手指定最新等级分前6名，分别是：[[古力|古力]]、[[常昊|常昊]]、[[孔杰|孔杰]]、[[谢赫|谢赫]]、[[朴文垚|朴文垚]]、[[王檄|王檄]]，每一站比赛随机出3人
*女棋手通过选拔产生3人，替补1人，分别是：[[王晨星|王晨星]]、[[李赫|李赫]]、[[宋容慧|宋容慧]]，(替补：[[王祥云|王祥云]])

=== 比赛规则 ===
分三个阶段，每站下二场。每方2小时，保留五分钟读秒。采取升降制，从男棋手授先开始，依次为：白倒贴3目半、7目半、二子……
每队各出3人，以团体分胜负。比赛当年的成绩明年依然有效，此比赛形式将持续下去。
