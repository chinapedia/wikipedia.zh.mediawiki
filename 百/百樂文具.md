'''百樂文具股份有限公司'''（{{lang-ja|株式會社パイロットコーポレーション}}），簡稱'''百樂文具'''，是一家生產[[文具|文具]]的[[日本|日本]]公司，總部位於[[東京|東京]]，是日本最大的文具生產商之一。

== 簡史 ==
* 1918年1月27日：[[東京高等商船學校|東京高等商船學校]]教授[[並木良輔|並木良輔]]成立'''株式會社並木製作所'''。
* 1938年：並木製作所正式改名為'''百樂萬年筆株式會社'''。
* 1989年：百樂萬年筆正式改名為'''株式會社百樂'''。
* 2002年1月：'''株式會社百樂集團控股'''成立。

== 主要類別產品 ==
* [[自動鉛筆|自動鉛筆]]、[[橡皮擦|橡皮擦]]、[[鋼筆|鋼筆]]、[[原子筆|原子筆]]、[[中性筆|中性筆]]、[[鋼珠筆|鋼珠筆]]、[[螢光筆|螢光筆]]、[[修正帶|修正帶]]

== 特色 ==
向中國市場推出高檔鋼筆的盤算中提及，日本產鋼筆在加工時會微調筆尖，使之容易寫「丨」，相對歐美高檔品牌，適合漢字文化圈書寫細小漢字的需求<ref>{{cite news|language=zh-hant|author=山田遼太郎|date=2018年11月16日|accessdate=2018年11月16日|url=http://zh.cn.nikkei.com/industry/tradingretail/32904-2018-11-16-05-00-40.html|title=日本高檔鋼筆盯上中國人錢包|publisher=日經中文網}}</ref>。

== 特色產品 ==
[[File:Pilot_ball_pens.jpg|thumb]]

* [[Pilot_Frixion|Pilot Frixion]]：是由百樂（Pilot）所用Metamocolor技術所生產的中性筆。其特點為用其所寫的字跡能透過筆桿末端的膠粒擦拭。
