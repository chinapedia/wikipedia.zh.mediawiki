'''则天文字'''或'''则天新字'''，也称'''武后新字'''，是[[中国历史|中国历史]]上[[武周|武周]]的[[皇帝|皇帝]][[武则天|武则天]]所创造的新[[汉字|汉字]]的总称，在今天看来属于[[异体字|异体字]]范畴。按照汉字的六种构造条例——[[六书|六书]]来划分，这些字都属于[[象形|象形]]和[[会意|会意]]字。

== 概述 ==
武则天代唐為周，除了改服易帜、频繁[[改元|改元]]、变更[[职官|职官]]名称外，迷信[[文字|文字]]具有統治思想的力量，所以发明了一些全新的文字，替代原有的文字，以除旧佈新、建立權威，后世称为「则天文字」。

则天文字不純粹憑空创造，也有依托古字而改，例如「天」被改成了「[[File:则天文字之天手写.svg|20px]]」，就是利用了「天」字的[[篆书|篆书]]，{{fact|表明武则天效法古代先贤，有「法古」之意；但更多的是全新創造，强调了她发动的是一场[[革命|革命]]（参看[[武周革命|武周革命]]），不但在[[政治|政治]]上革了[[唐朝|李唐王朝]]的命，在[[文化|文化]]上也要革命，所以要「自我作古」，创造一个全新的时代。}}

关于则天文字究竟有多少个，由于武则天命人所作的、收录有所有则天文字的《字海》已经失传，因此历来众说纷纭，有12个、16个、17个、18个、19个和21个等之说，而目前有记载的则天文字则有30个（參見下方[[#文字列表|文字列表]]一節）。因为有些单字的新写法有两种，或者因为印刷、手写不同而产生错误，形成[[变体字|变体字]]，按原字则算1个字，按新字则算2个字或更多，所以造成了统计的不同。

则天文字在全[[中国|中国]]僅通行十五年，虽然字并不是很多，但都是常用字，所以造成了一定的混乱。文物鉴定上也常常以则天文字的出现视为武周时期的一个特有现象<ref> {{cite journal zh|title=论韩国发现的印本《无垢净光大陀罗尼经》|journal=科学通报|date=1997|first=潘吉星|last= |coauthors=|volume=42|issue=10|pages=1009-1028|id= |url=|format=|accessdate=}}</ref>。随着女皇帝的退位，[[神龍_(年號)|神龙]]元年[[二月初四|二月初四]][[甲寅|甲寅]]日（705年3月3日）<ref>从两《唐书·则天后本纪》和《资治通鉴》，《唐会要·帝号上·中宗》作「[[二月初五|二月初五]]」（3月4日），误。</ref>[[唐中宗|唐中宗]]复[[国号|国号]]为[[唐朝|大唐]]，武周王朝告终，原字恢复，则天文字虽未被立刻废止，但由于书写困难，渐渐被人们淡忘，最终停止使用。然而，由于武则天的影响力，則天文字不但在中國本土流傳開來，還傳到国外。

== 发展 ==
[[File:A_Tang_Dynasty_Empress_Wu_Zetian.JPG|150px]][[女皇帝|女皇帝]][[武曌|武曌]]]]

=== 创制与颁行 ===
据《[[新唐书|新唐书]]》和《[[资治通鉴|资治通鉴]]》等史书记载，这套文字并不是武则天本人亲自发明，而是授意其表外甥[[宗秦客|宗秦客]]所造。

起初颁布通行的则天文字只有12个，即「照」、「天」、「地」、「日」、「月」、「星」、「君」、「臣」、「载」、「初」、「年」、「正」，被改成则天文字——「曌」、「[[File:则天文字之天二.SVG|20px]]（𠀑）」、「埊」、「 [[File:则天文字之日.SVG|20px]]（𡆠）」、「囝」、「〇」、「[[File:则天文字之君正字.SVG|20px]]」、「[[File:则天文字之臣.SVG|20px]]（𢘑）」、「[[File:则天文字之载正字.SVG|20px]]（𡕀）」、「[[File:则天文字之初.SVG|20px]]（𡔈）」、「[[File:则天文字之年正字.SVG|20px]]」、「[[File:则天文字之正.SVG|20px]]（𠙺）」<ref>[[s:新唐書/卷076|s:新唐書/卷076]]</ref>，其中「[[File:则天文字之载正字.SVG|20px]]（𡕀）」、「[[File:则天文字之初.SVG|20px]]（𡔈）」二字合成了新年号「[[载初_(武则天)|载初]]」的全新字形。武则天以[[临朝稱制|临朝稱制]]的大唐[[皇太后|皇太后]]身份颁布的《改元载初敕》中说：
{{cquote|……[[朕|朕]]宜以曌为名……特创制一十二字，率先百辟，上有依于古体，下有改于新文，庶保可久之基，方表还淳之意。……}}

从此，[[武则天|武氏]]的[[名字|名字]]由武媚改为武曌，而为了[[避讳|避讳]]，把[[诏书|诏书]]（诏[[发音|发音]]同照和曌）改为[[制书|制书]]。同时，废帝庐陵王（即[[唐中宗|唐中宗]]李哲）的长子[[李重照|李重照]]（曌同照）为了避祖母武曌的名讳，改名[[李重润|李重润]]。同年[[正月初八|正月初八]][[丁亥|丁亥]]日（689年12月25日），则天文字正式颁布通行。

而《[[旧唐书|旧唐书]]·艺文志》也有类似的记载，说-{武后}-（即武则天）-{著}-字海百卷，创造了不少新字，所以叫武后新字或则天文字。

=== 传播与新增 ===
载初元年（690年）七月，圣母神皇<ref>「圣母神皇」简称「神皇」，是武则天称帝前所加的[[尊号|尊号]]，实际身份仍是大唐的[[皇太后|皇太后]]。</ref>武曌的[[洛阳|洛阳]][[白马寺|白马寺]][[住持|住持]][[薛怀义|薛怀义]][[和尚|和尚]]献上为《[[大云经|大云经]]》所作的注解（即《[[大云经疏|大云经疏]]》），神皇便将则天文字援引入内，将经和经疏内的原字换成新字，并在全国各地颁布发行。《大云经》中有「弥勒降生」、「女子为王」的内容，《大云经疏》更注解说，神皇武曌即[[弥勒菩萨|弥勒菩萨]]降生，当以女身取代李唐王朝，成为[[女皇帝|女皇帝]]和全[[天下|天下]]之主。《大云经》和《大云经疏》是武曌利用[[佛教|佛教]]为其称帝做舆论准备的重要工具，在全国广为流传，则天文字也随之更广泛地传播开来。

载初元年[[九月初九|九月初九]][[壬午|壬午]]日[[重阳节|重阳节]]（690年10月16日），神皇武曌称帝，改元[[天授_(武则天)|天授]]。在天授年间，又出现了「授」字的则天文字——「[[File:则天文字之授二.SVG|20px]]𥠢」，而本来「天」字已经改为则天文字——「[[File:则天文字之天一.SVG|20px]]𠀑」了，这样年号中的两个字都成了则天文字；改元[[证圣|证圣]]后，又把「证」、「圣」二字改为「𤪉[[File:则天文字之证正字.SVG|20px]]」、「[[File:则天文字之圣.SVG|20px]]𨲢」，也都成了则天文字；证圣元年（695年）六月后，出现了「圀」字取代原来的「国」字；[[圣历|-{zh:圣历; zh-hans:圣历; zh-hant:聖曆;}-]]年间（698年），又出现了「[[File:则天文字之人.SVG|20px]]𤯔」字取代原来的「人」字。这样，加上原来的十二个则天文字，总共便有十七个则天文字。至于其他四个则天文字，有学者认为是以讹传讹，也有学者认为确有其事。

=== 使用与废止 ===
则天文字在武则天称帝期间的十五年（690年—705年）中得到了广泛使用，很多[[石刻|石刻]]、[[碑帖|碑帖]]、[[佛经|佛经]]等都有则天文字的身影，现存的武周时期的[[碑刻|碑刻]]、[[墓志铭|墓志铭]]等都充分证明了这一点。

唐中宗[[复辟|复辟]]后，曾一度颁布制书，将所有制度都恢复到已故父皇[[唐高宗|唐高宗]][[永淳|永淳]]以前，则天文字当然也停止不用了。但是不久，[[武三思|武三思]]等前武周皇族就展开反击，勾结[[韋皇后_(唐中宗)|韋皇后]]和[[上官婉儿|上官婉儿]]，将发动[[五王政变|五王政变]]推翻女皇的[[张柬之|张柬之]]等五位重臣（即[[五王_(唐朝)|五王]]）悉数诬告杀害，使武氏家族再次控制了大唐的朝廷，而且权势比武则天在世时更盛。这时，时任[[左补阙|左补阙]]的[[权若讷|权若讷]]上奏中宗，称则天文字仍是武则天的伟大创举，让它们保存下来可以体现出皇帝对已故母后的孝心。看到奏报后，中宗特地颁布制书予以嘉奖权若讷，并保留了母后所造的则天文字<ref>[[权若讷|权若讷]]上奏恢复则天文字之事见[[清代|清代]][[洪迈|洪迈]]所著《[[容斋随笔|容斋随笔]]·容斋续笔》。</ref>。以后一直到[[唐文宗|唐文宗]][[开成|开成]]二年（837年）十月，才再次颁布[[诏书|诏书]]，废除则天文字，一律改用本字<ref>[[唐文宗|唐文宗]]颁诏废除则天文字之事见[[北宋|北宋]][[王钦若|王钦若]]等所编《[[册府元龟|册府元龟]]》。</ref>。这样，则天文字在武则天逝世后，实际又沿用了132年，总共通行了近150年。

但是由于武则天的影响很大，她所创造的新字也流传甚广，群臣的奏章和天下的书籍都使用新字，大家对于新字的使用已经习惯，并非一纸诏书可以革除的，因此在正式废除新字以后，仍有不少书籍使用新字，又过了百余年后才销声匿迹。[[宋代|宋代]]以后，已经没有人使用则天文字了，以至于宋代人对于它们的演变发展过程也知之甚少。

== 日月当空“曌” ==
“曌”作为则天文字中最出名的一个，其意义在千百年来一直指武则天一人，成为了女皇帝的标志之一，然而对于它的来历，则众说纷纭<ref>雷家骥, 《武则天传》，人民出版社, 2001年11月第1版ISBN 7-01-003453-2</ref>。
*日月当空为曌：最普遍的说法。
*改照为曌：有学者指出，武则天本名武照，所以后来改为同音字“曌”，但此说缺乏根据，虽然“曌”的原字就是“照”，但没有证据表明武则天本名为武照。
*明空为曌：有学者指出，武则天本名武约，[[表字|字]]明空，或者本名就叫武明空<ref>见《[[太平御览|太平御览]]》引《[[唐书|唐书]]》：“则天皇后武氏讳明空。”</ref>，或者她[[出家|出家]]时[[法号|法号]]明空，所以后来把“明”、“空”二字合在一起成为“曌”字，此说较为合理，且有证据，但终是一家之言。
*日月子女为明：有学者追究日、月的来历，发现武则天最宠爱的小儿子[[唐睿宗|唐睿宗]]和小女儿[[太平公主|太平公主]]与其名“曌”上半部的“明”有关。因为唐睿宗本名[[李旭轮|李旭轮]]，后改名[[李旦|李旦]]，都和“日”有关；而太平公主名[[李令月|李令月]]<ref>《[[全唐文|全唐文]]·代皇太子上食表》：“臣某言：伏见臣妹太平公主妾李令月嘉辰，降嫔公族。诗人之作，下嫁於诸侯；易象之兴，中行於归妹。又臣铜楼再惕，常荷荫於中慈；金屋相惊，忽承恩於内辅。周官典瑞，旁稽聘女之仪；晋朝加玺，兼采纳妃之制。圣怀感尉，皇泽?濡，愿垂扶木之光，俯遂甘芹之请。谨上礼食若干举如列。尊师四学，虽有谢於温文；问竖三朝，窃无违於视膳。谨遣某奉表以闻。”</ref>与月有关（但是太平公主是否名为令月亦有争议）；“日”、“月”合而为“明”。但这也是一家之说，更有人用此说反证太平公主名令月。
*其他：[[中國中央電視台|中國中央電視台]]首播的[[电视连续剧|电视连续剧]]《[[大明宫词|大明宫词]]》中描绘[[术士|术士]]明清远（以历史上的真人[[明崇俨|明崇俨]]为原型）将“曌”字献给当时还是[[唐高宗|唐高宗]][[皇后|皇后]]的武则天，暗示助其称帝<ref>见《大明宫词》剧本第十集。</ref>。

== 更改“国”字的传说 ==
据《[[朝野佥载|朝野佥载]]》所载，[[天授_(武则天)|天授]]年间，已经成为女皇帝的武则天喜欢改造旧字成为新字，但又多有忌讳。有一个[[幽州|幽州]]（治所在今[[北京市|北京市]]西南）人叫寻如意，向女皇呈上了一封信，上面写到：“‘-{國}-’字里面是‘或’，而‘或’是天下大乱的象征{{efn|“或”[[通假字|通]]“惑”，指惑乱。}}。请在‘囗’中安上陛下的姓‘武’字来镇住它。”女皇大喜，发下[[制书|制书]]立即执行，即改“-{國}-”为“[[File:则天文字之国最初.svg|20px]]”。过了一个多月，又有一人向女皇呈上一封信，上面说：“‘武’退在‘囗’中，与‘囚’没什么差别，非常不吉利。”女皇大惊，立即追归制书，又把“囗”中的“武”改为“八方”，即为“{{unihan|5700}}”。但是这么一改仍然没解决根本问题。后来，中宗复辟，果然把武则天幽禁在上阳宫，而“上阳”二字就是“照临八方”的意思，武则天搬起石头砸自己的脚，终究逃不出被囚禁的命运，成为后人的笑柄<ref>《[[太平广记|太平广记]]·幽州人》</ref>。

这则传说虽然很动听逼真，但却没有多少根据，因为据考证，“{{unihan|5700}}”字的出现是在[[证圣|证圣]]年间，而不是在天授年间<ref name="王三庆" />。

== 影响 ==
[[File:升仙太子碑拓片局部.jpg|200px]]
虽然唐王朝的[[诏书|诏书]]通告天下废除则天文字，但实际上这些文字仍然在各地使用，流传很广，说明武则天的统治是非常有效的，尤其是[[河西走廊|河西走廊]]（在今[[甘肃省|甘肃省]]）和[[西域|西域]]（今天的[[新疆|新疆]]、[[中亚|中亚]]）等边境地区，大约沿用了二百多年（则天文字在全国通行15年不到），比如在[[敦煌|敦煌]]，就发现了大量以则天文字撰写的佛经，其中就包括早已失传的《[[大云经|大云经]]》和《[[大云经疏|大云经疏]]》，为学者们研究则天文字提供了宝贵的素材。

部分则天文字还传到[[日本|日本]]、[[韩国|韩国]]，甚至成为某些[[日本人|日本人]]的人名用字，如[[江户时代|江户时代]]的[[大名_(称谓)|大名]]和主修《[[大日本史|大日本史]]》的著名[[历史学家|历史学家]][[德川光国|德川光国]]（{{lang|ja|徳川光國}}）就根据则天文字中的“圀”字改名为“[[德川光圀|德川光圀]]”，因為他認為「-{國}-」字裡的「或」字[[通假字|通]]「惑」，不吉，所以要改名。而为德川光圀[[剃度|剃度]]的[[本国寺|本'''国'''寺]]也因此改名为[[本圀寺|本圀寺]]。这样，“圀”字便成为日本最有名的则天文字。

则天时期的[[碑刻|碑刻]]传世很多，最有名的是她亲笔书写的《[[升仙太子碑|升仙太子碑]]》，被誉为“千古美文”，其中就有很多则天文字；如右图该碑的[[拓片|拓片]]局部所示，其中“大周天册金轮圣神皇帝”（武则天当时的[[尊号|尊号]]）中的“[[File:则天文字之天手写.svg|20px]]”（则天文字之“天”字）、“[[File:则天文字之圣.SVG|20px]]”（则天文字之“圣”字）二字就是则天文字。而收藏有大量唐代墓志石刻的[[千唐志斋石刻|千唐志斋石刻]]也刻载了许多则天文字，是汇集则天文字的宝库。

武则天以后，[[南汉|南汉]]的开国君主[[刘䶮|刘岩]]也学她改字，创造了“[[Wikt:龑|-{龑}-]]”（[[汉语拼音|拼音]]：Yǎn，[[注音符號|注音符號]]：ㄧㄢˇ）字，意为“飞龙在天”，并作为自己的名字。该字現常類推简化为“-{䶮}-”字<ref>见1999年版的《[[辞海|辞海]]·缩影本》。</ref>。

在近代东亚各国進行漢字整理以來，大部分则天文字已极少在日常生活中使用。然而1980年代后，在[[中国大陆|中国大陆]]以曌、龑等字取名的现象又有所增加。

Unicode 3.1新增的[[中日韩统一表意文字扩展区B|中日韩统一表意文字扩展区B]]中收录了大部分则天文字，而其中的“〇”早已作为标点符号单独收录于Unicode 1.0里面的[[中日韩标点符号|中日韩标点符号]]区块里。
