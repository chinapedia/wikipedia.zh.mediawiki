'''还亮草'''（[[学名|学名]]：{{lang|la|''Delphinium anthriscifolium''}}）为[[毛茛科|毛茛科]][[翠雀属|翠雀属]]一年生草本植物。

== 别名 ==
浙江天目山一带别称鱼灯苏、车子野芫荽。<ref name=":2">{{Cite book|title=《浙江天目山药用植物志》上集|author=浙江省卫生厅|publisher=浙江人民出版社|year=1965|isbn=|location=杭州|pages=426}}</ref>[[清代|清代]]也叫还魂草、对叉草、蝴蝶菊。<ref name=":0">{{Cite book|title=《植物名实图考》上册|author=吴其濬|publisher=中华书局|year=1963|isbn=|location=北京|pages=|chapter=卷十三·湿草类}}</ref>[[日语|日语]]名称为芹葉飛燕草。<ref name=":3">{{Cite web|url=http://www.atomigunpofu.jp/ch6-foreign%20flowers/seribahienso.htm|title=せりばひえんそう (芹葉飛燕草)|accessdate=2019-02-23|author=嶋田英誠|date=2006|publisher=跡見群芳譜巻六 外来植物譜}}</ref>

== 分布与生境 ==
还亮草原产[[甘肃|甘肃]]、[[陕西|陕西]]、[[山西|山西]]、[[河南|河南]]、[[安徽|安徽]]、[[江苏|江苏]]、[[浙江|浙江]]、[[江西|江西]]、[[福建|福建]]、[[湖南|湖南]]、[[湖北|湖北]]、[[广东|广东]]、[[广西|广西]]、[[贵州|贵州]]、[[四川|四川]]、[[云南|云南]]以及[[越南|越南]]北部，生长于海拔0~1,700米间丘陵或低山的山坡草丛或溪边草地。<ref name=":1">{{Cite web|url=http://www.efloras.org/florataxon.aspx?flora_id=2&taxon_id=200007779|title=''Delphinium anthriscifolium''|accessdate=|author=Flora of China|date=|publisher=Missouri Botanical Garden}}</ref>[[明治时代|明治时代]]传入[[日本|日本]]。<ref name=":3" />

== 用途 ==
还亮草全草供药用，味辛有毒，外涂治痈疮癣癞。<ref name=":2" />

== 民俗 ==
[[唐代|唐代]]《[[本草拾遗|本草拾遗]]》记载：“桃朱術生園中，細如芹，花紫，子作角，以鏡向旁敲之，則子自發。五月五日乃收子，帶之令婦人為夫所愛。”[[吴其濬|吴其濬]]认为桃朱术即还亮草。<ref name=":0" />

== 变种 ==
还亮草有三个变种：<ref name=":1" />

*'''还亮草''' ''D. a.'' var. ''anthriscifolium''
*'''[[卵瓣还亮草|卵瓣还亮草]]''' ''D. a.'' var. ''savatieri'' (Franch.) Munz
*'''[[大花还亮草|大花还亮草]]''' ''D. a.'' var. ''majus'' Pamp.
