'''孫中山紀念館'''（{{lang-en|Dr. Sun Yat-sen Museum}}）位於[[香港|香港]][[香港島|香港島]][[中環|中環]][[衛城道|衛城道]]7號，前身是[[何東|何東]]胞弟[[何甘棠|何甘棠]]的住宅'''甘棠第'''，於2010年11月12日（孫中山誕辰144週年）列為[[香港法定古蹟列表|法定古蹟]]<ref>{{cite news |language=zh-hk |url=http://www.info.gov.hk/gia/general/201011/12/P201011120223.htm |title=四幢歷史建築列為法定古蹟 |publisher=''info.gov.hk'' |date=2010-11-12 |accessdate=2010-11-13}}</ref>。紀念館是[[香港歷史博物館|香港歷史博物館]]的分館，介紹[[中華民國|中華民國]]國父孫中山先生的生平及他與香港的關係。紀念館於2006年12月12日（[[孫中山|孫中山]]誕辰140周年後1個月）正式開幕。

[[File:孫中山銅像.jpg|200px]]

==建築歷史==
甘棠第建於1914年，由香港富商[[何東|何東]]之二弟[[何甘棠|何甘棠]]興建，何甘棠亦於1950年在此終老。1960年，甘棠第售予鄭氏家族，其後鄭氏又轉售予[[耶穌基督後期聖徒教會|耶穌基督後期聖徒教會]]，作教會的聚會場所。1990年，甘棠第被[[古物諮詢委員會|古物諮詢委員會]]列為二級歷史建築。2002年，教會曾計劃拆卸改建，結果引起中西區區議會及居民強烈反對，政府亦認同甘棠第的歷史價值，最終出手以5,300萬元收購，並預算動用9,100萬元進行修葺，作為孫中山博物館。<ref>[http://www.alumni.cuhk.edu.hk/magazine/200412/html/p12-13.htm 中大校友-再造「甘棠第」]</ref>甘棠第在2010年被列為[[香港法定古蹟|香港法定古蹟]]。

==建築特色==
[[File:Kom_Tong_Hall_Stair.jpg|200px]]
甘棠第樓高4層，屬[[愛德華|愛德華]]時代的古典建築風格。一、二樓設有弧形露台，以[[希臘|希臘]]式巨柱承托。外部色彩斑斕的玻璃窗、陽臺牆身的瓷磚，以及室內的柚木樓梯均保存良好。甘棠第是香港其中一座最早以鋼筋構建，並裝設供電線路鋪設的私人住宅，亦是香港現存少數的建於20世紀初建築物。

房間的天花均飾有以金箔點綴的灰塑鑲板，而主樓梯及其他當眼位置亦裝設了色彩斑斕的玻璃窗，並以當時流行的新藝術風格圖案作為裝飾。

==設施與展覽==
[[File:Dr_Sun_Yat-sen_Museum_Exhibit_201007.jpg|200px]]

* 兩個常設展覽廳，主題分別為'''「孫中山與近代中國」'''及'''「孫中山時期的香港」'''
* '''「甘棠第的歷史與修繕」'''展覽廳
* 專題展覽廳暨演講廳
* 閱覽室
* 兩個視像室
* 互動學習室：透過互動電腦遊戲形式介紹孫中山的事蹟及[[辛亥革命|辛亥革命]]等的專題
* 活動室

==展品==
* 展館展出約150件藏品，7成是市民捐贈及館藏，其餘由[[英國|英國]][[倫敦|倫敦]]（10件）及[[中國大陸|中國大陸]]的博物館（48件）借出，逾半屬一級國家文物，借出展品的展期由半年至兩年不等。[[中國國民黨|中國國民黨]]黨史館亦有借出文物展出。

* 重點展品：
# 孫中山在1886年至1887年在廣州習醫時使用的顯微鏡（中國國家博物館藏）
# 孫中山於香港習醫的考試答題卷
# [[香港西醫書院|香港西醫書院]]畢業典禮的晚宴菜單
# 1901年1月1日留日學生團體勵志會在東京舉行新年慶祝會首次公佈的「[[中華民國|中華民國]]萬歲」印章
# 孫中山於1909年向[[香港殖民地時期#香港政府|香港政府]]申請回港的親筆信
# 孫中山為[[關景良|關景良]]擔任證婚人的結婚證書
# [[中華民國大總統|中華民國大總統]]孫文宣言書（1912年1月1日；廣州博物館藏）
# [[宣統|宣統]]皇帝[[溥儀|溥儀]]退位詔書（1912年2月12日；中國國家博物館藏）
# 孫中山贈予香港海員的「博愛」親筆題幅
# 孫中山於1912年贈予[[黃興|黃興]]的對聯
# 孫中山於1923年11月26日批示的大元帥令
