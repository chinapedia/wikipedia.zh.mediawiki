'''孙布拉蚌'''（{{lang-my|ဆွမ်ပရာဘွမ်မြို့}}）是[[缅甸|缅甸]]最北端[[克钦邦|克钦邦]]下辖的一个镇。孙布拉蚌战略地位十分重要，1944年[[中国驻印军|中国驻印军]]反攻[[密支那|密支那]]时曾经过此地。2011年至2013年，孙布拉蚌亦为{{le|缅甸陆军|Myanmar Army}}和[[克钦独立军|克钦独立军]]反复争夺的战略重地，沿途多处桥梁被炸毁<ref>{{Cite web |url = http://blog.sina.com.cn/s/blog_3c6c43870101ibit.html |title = 克钦邦西北部行记1——“孙不拉蚌！孙布拉蚌！” |author = |date = 2014-05-06|publisher = |language = 中文|accessdate = 2015-05-08| quote = }}</ref>。
