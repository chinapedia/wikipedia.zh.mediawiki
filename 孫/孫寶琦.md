'''孙宝琦'''（{{bd|1867年|4月26日|1931年|2月3日|catIdx=S孫}}），字'''慕韩'''，[[浙江省|浙江省]][[杭州府|杭州府]][[钱塘县|钱塘县]]（今[[杭州市|杭州市]]）人。[[清末|清末]][[中華民國|民初]]政治家、外交官，曾任中华民国外交总长、[[中华民国国务总理|国务总理]]。

== 生平 ==

清末大臣[[孫詒經|孫詒經]]之子。成[[廕生|廕生]]。曾任[[直隶省|直隶]][[道员|道员]]。1900年[[八国联军|八国联军]]进京，随[[光绪帝|光绪帝]]至[[西安|西安]]。<ref>光绪朝实录 卷之四百九十二</ref>1901年（[[光緒|光緒]]二十七年）起，历任驻[[德国|德国]]、[[日斯巴尼亚|日斯巴尼亚]]（今称[[西班牙|西班牙]]）、[[葡萄牙|葡萄牙]]、[[法国|法国]]等国使馆随员。1902年7月，升任[[中国驻法国大使列表|出使法国大臣]]。1905年（光緒31年）归国，署[[順天府|顺天]][[府尹|府尹]]。1907年4月，出任[[中国驻德国大使列表|出使德国大臣]]，筹措讨还[[青岛|青岛]]事宜。1908年（光緒三十四年）回国，充帮办[[津浦铁路|津浦铁路]]大臣。1911年（[[宣統|宣統]]三年），转任[[山东巡抚|山东巡抚]]。任内阻止德国取得山东矿权。<ref name=xu>{{Cite book<!--|和書-->|author = 徐友春主編|title = 民国人物大辞典 増訂版|year = 2007|publisher = 河北人民出版社|isbn = 978-7-202-03014-1}}</ref>同年，[[辛亥革命|辛亥革命]]爆发，孫寶琦宣布[[山东|山东]]独立，随后又取消独立。<ref name=xjb>{{cite news|title=孙宝琦 左右摇摆 力保平安|url=http://www.bjnews.com.cn/ent/2011/08/11/143145.html|accessdate=2013-02-25|newspaper=新京報網}}</ref>

1913年（[[民国紀元|民国]]二年）9月，孫寶琦任[[北洋政府|北京政府]][[熊希龄内阁|熊希龄内阁]]外交总长。1914年2月12日，兼代国务总理。1914年5月1日辞去国务总理，专任外交总长（外交次长为[[曹汝霖|曹汝霖]]）。同年7月[[第一次世界大战|第一次世界大战]]爆发，孫寶琦支持[[袁世凯|袁世凯]]的中立政策。1915年1月，因日本提出[[二十一条|二十一条]]，辞去外交总长（[[陆征祥|陆征祥]]接任），改任[[中華民國审计院|审计院]]院长。1916年（民国5年）4月，任[[第一次段祺瑞內閣|段祺瑞内阁]]财政总长。同年6月兼[[汉冶萍公司|汉冶萍公司]]董事长。1919年任[[招商局|招商局]]董事会会长。1920年（民国9年）春，任经济调查局总裁（税务督办）。1922年（民国十一年）1月，任[[扬子江|扬子江]]水道讨论委员会会长；同年4月任外交部[[太平洋会议|太平洋会议]]善后委员会副会长。<ref name=xu/><ref name=liu>{{Cite book<!--|和書-->|author = 劉寿林等編|title = 民国職官年表|year = 1995|publisher = 中華書局|isbn = 7-101-01320-1}}</ref>

经[[中华民国大总统|大总统]][[曹锟|曹锟]]提名，[[中華民國第一届国会|国会]]通过，1924年（民国十三年）1月12日，孫寶琦被任命为国务总理，1月15日[[孙宝琦内阁|孙宝琦内阁]]就职。1924年，孫寶琦还兼任外交委员会委员长。任内，孫寶琦提出「奉行宪法」、「和平统一」的施政方针，同[[苏联|苏联]]建立了外交关系，向[[德国|德国]]索赔成功。同年7月2日，因与[[曹锟|曹锟]]的嫡系、财政总长[[王克敏|王克敏]]不和，提出辞职。由外交总长[[顾维钧|顾维钧]]代理国务总理。<ref name=xu/><ref name=liu/>

1925年（民国十四年）2月，孫寶琦被[[段祺瑞|段祺瑞]]任命为淞滬商埠督办。同年8月，被任命为[[中国驻俄国大使列表|驻苏联大使]]，但未就任。1926年（民国15年），任[[中法大学|中法大学]]董事長。<ref name=xu/><ref name=liu/>

1931年（民国二十年）2月3日，孫寶琦在上海逝世，享年65岁。<ref name=xu/>

孙宝琦在北京的故居位于[[北京市|北京市]][[东城区_(北京市)|东城区]][[东四六条|东四六条]]。

== 逸事 ==
1904年冬，[[孙中山|孙中山]]携[[朱和中|朱和中]]赴[[德国|德国]][[柏林|柏林]]，访问中国留德学生。在[[宾步程|宾步程]]的带动下，[[王发科|王发科]]、[[王相楚|王相楚]]等留学生加入了[[兴中会|兴中会]]。在柏林逗留数日后，孙中山与[[朱和中|朱和中]]赴[[法国|法国]][[巴黎|巴黎]]，留法学生[[唐豸|唐豸]]、[[胡瑛|胡瑛]]、[[汤芗铭|汤芗铭]]、[[向国华|向国华]]等十多人应邀入[[兴中会|兴中会]]。[[孙中山|孙中山]]在[[巴黎|巴黎]]住在[[利倭尼街|利倭尼街]]的瓦克拉旅馆。一天，[[汤芗铭|汤芗铭]]、[[向国华|向国华]]、[[王发科|王发科]]、[[王相楚|王相楚]]四人结伴来找孙中山去附近的咖啡馆喝咖啡，王发科、王相楚二人中途离席，到瓦克拉旅馆盗走了孙中山的行箧中的党人名单及入盟书、法国殖民大臣致安南总督的信函。此为王发科、王相楚、汤芗铭、向国华四人合伙设计。得手后，四人到清国驻法国公使馆告密，公使孙宝琦训斥四人称：“你们加入革命党，是叛清朝；今来自首，又叛革命党。且陷害同学，人格何在！”孙宝琦收阅了四人盗取的文件后称，“今后你们要好好念书，安分守己，不要胡闹。”留法学生[[夏坚仲|夏坚仲]]是孙宝琦的亲戚，得知消息后赶到使馆向孙宝琦疏通。孙宝琦乃将入盟者的宣誓书烧掉，将其余文件交给夏坚仲，夏坚仲通过邮局将文件寄还孙中山。但是，孙宝琦从法国殖民大臣致安南总督的信函中得知孙中山的安南起义计划后，即赴法国外交部交涉，导致孙中山的起义计划未能实现。<ref>[http://history.people.com.cn/GB/205478/15499794.html 胡为雄，帝制的终结——辛亥天变，当代中国出版社，2011年]</ref>
