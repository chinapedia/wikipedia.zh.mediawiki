'''广东实验中学美国分校'''，简称省实美国分校，是[[广东实验中学|广东实验中学]]在[[美国|美国]][[加利福尼亚州|加利福尼亚州]][[河滨市|河滨市]]的一座分校，原定于2011年9月30日落成。<ref name="nfdaily1">{{Cite web |url=http://nf.nfdaily.cn/nfrb/content/2011-10/02/content_30801975.htm |title=广东实验中学美国分校在加州洛杉矶河滨市落成 |accessdate=2011-10-14 |work=南方报业网 |publisher=[[南方日报|南方日报]] |date=2011-10-02 }}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref> 这所学校同时也被认为是美国第一所中国人建立的的海外实体[[中学|中学]]。<ref name="ycwb1">{{Cite web |url=http://www.ycwb.com/ePaper/ycwb/html/2011-10/02/content_1224302.htm |title=广东学校 办到美国 |accessdate=2011-10-14 |work=金羊网 |publisher=[[羊城晚报|羊城晚报]] |date=2011-10-02 |deadurl=yes |archiveurl=https://web.archive.org/web/20111011111325/http://www.ycwb.com/ePaper/ycwb/html/2011-10/02/content_1224302.htm |archivedate=2011-10-11 }}</ref>

然而，自从学校声称落成之后，陆续有[[媒体|媒体]]及[[网民|网民]]怀疑这所学校的真实性。目前已确定这所学校是不存在的，省实方面也声称“暂不在美国开设分校”。<ref name="hexun">{{Cite web |url=http://news.hexun.com/2011-10-14/134228186.html |title=未迈出的“第一步” |accessdate=2011-10-15 |work=和讯网|publisher=经济观察报 |date=2011-10-14 }}</ref><ref name="qiao0">{{Cite web |url=http://www.chinapressusa.com/2011-10/19/content_1072525.htm |title=广东实验中学就在美建分校事件发函道歉 |accessdate=2011-10-20 |work=侨报网 |publisher=侨报 |date=2011-10-19 |deadurl=yes |archiveurl=https://web.archive.org/web/20120425082013/http://www.chinapressusa.com/2011-10/19/content_1072525.htm |archivedate=2012-04-25 }}</ref>

==背景==
{{See Also|广东实验中学}}
广东实验中学成立于1888年，曾与[[华南师范大学附属中学|华师附中]]属于同一间中学。学校初中校区位于[[越秀区|越秀区]][[中山四路|中山四路]]，高中校区位于[[荔湾区|荔湾区]]坑口省实路。学校为[[广东省|广东省]]重点中学，曾培养包括[[姜伯驹|姜伯驹]]、[[钟南山|钟南山]]在内共七名[[中国科学院|中国科学院]]院士和[[中国工程院|中国工程院]]院士。

2011年3月份，首次有媒体报导有关省实建立美国分校的消息。在报道中，学校校长[[鄭熾欽|鄭熾欽]]称其在2009年与美国“加州教育委员会主任”陈立见面时，曾大胆向陈立提出建立美国分校的设想，得到了对方的赞同。随后，美国[[凯麦斯教育基金会|凯麦斯教育基金会]]寄来函件声明愿意合作。<ref name="nfdaily2">{{Cite web |url=http://news.sina.com.cn/c/2011-03-25/082022179224.shtml |title=广东实验中学将在美国开设分校9月开学 |accessdate=2011-10-14 |work=[[新浪网|新浪网]] |publisher=南方日报 |date=2011-03-25 }}</ref>

2011年5月份，“加利福尼亚州教育厅厅长”戴维·朗格（Dave Long）对省实进行了访问。报道中，陈立称凯麦斯教育基金会接受到的[[捐款|捐款]]已经超过1.5亿美元，{{Link-en|加州政府|Government of California}}也拨地300亩。<ref name="newhua2">{{Cite web |url=http://news.xinhuanet.com/edu/2011-05/21/c_121441983.htm |title=广东实验中学将在美国加州开办公立学校 |accessdate=2011-10-14 |work=新华科教 |publisher=新华网 |date=2011-05-21 }}</ref>

2011年9月30日，学校的牌匾被运到了河滨市，同时举行了落成典礼。学校领导、广东省教育厅厅长罗伟其以及省实校友和美国当地有关负责人参加了典礼活动。<ref name="qiao1">{{Cite web |url=http://www.chinapressusa.com/2011-10/02/content_1054752.htm |title=广东实验中学美国分校落成 中国迈出基础教育“走出去”第一步 |accessdate=2011-10-15 |work=侨报网 |publisher=新华社 |date=2011-10-02 }}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

==学校介绍== 
据报道称，学校占地60公顷，投资额1.26亿美元（约合人民币8亿元），其中80%由[[美国政府|美国政府]]出资，20%由美国凯麦斯教育基金会出资。<ref name="newhua1" /> 学校依山而建，位于居民区附近以及公路旁，交通方便。<ref name="ycwb1" /> 学校按功能划分，有教学区和运动区两部分，其中运动区拥有标准[[田径|田径]]场、[[棒球场|棒球场]]、[[网球场|网球场]]、[[游泳|游泳]]馆等。<ref name="nfdaily1" /> 校园建设目前已大部分完成，预计于2012年1月份竣工，并招生大概1300人。 <ref name="newhua1" /> 同时，考虑到加利福尼亚州[[地震|地震]]频发的情况，学校充分考虑了安全性，整个分校最高建筑物楼层不超过4层。<ref name="nfdaily1" />

===媒体关注度===
虽然对于美国来说，这是首例与外国合办的公立中学，然而当地媒体似乎对此反应冷淡。省实校友称，当天到场的媒体仅有省实从广州邀请的7名记者，加州当地没有任何一家媒体出席。<ref name="hexun" /> 在世界日报于10月6日提出质疑之前，美国当地唯一一家报道了此次事件的是[[华人|华人]]报纸[[侨报|侨报]]，而其中引用的来源是来自于大陆的[[新华社|新华社]]。<ref name="qiao1" /> 直至10月21日，美国当地才出现第一篇非华人报纸的报道，内容为[[洛杉矶时报|洛杉矶时报]]对此事长篇的质疑。<ref name="latimes">{{Cite web |url=http://www.latimes.com/news/local/la-me-chinese-school-20111021,0,464658,full.story |title=Chinese delegation's visit to Riverside school stirs confusion |accessdate=2011-10-22 |work=latimes.com |publisher=Los Angeles Times |date=2011-10-21 }}</ref>

国内唯一对此事进行负面报道的经济观察报网站上的文章已经被删除。<ref name="qiao3">{{Cite web |url=http://www.chinapressusa.com/2011-10/20/content_1073232.htm |title=《侨报》刊省实道歉函 引网上热烈反响 |accessdate=2011-10-21 |work=侨报网 |publisher=侨报 |date=2011-10-20 }}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

===当地学校及部门反应===
在当日学校落成典礼派发的材料中，就有人发现该学校示意图为属于河滨县Alvord联合学区的{{Link-en|Hillcrest高中|Hillcrest High School (Riverside, California)}}。同时，典礼的照片背景也是这一所美国的公立高中。<ref name="world1">{{Cite web |url=http://la.worldjournal.com/view/full_la/15922907/article-%E5%BB%A3%E6%9D%B1%E5%90%8D%E6%A0%A1%E8%A8%AD%E7%BE%8E%E5%88%86%E6%A0%A1-%E5%AD%B8%E5%8D%80%E5%82%BB%E7%9C%BC?instance=la_story1 |title=廣東名校設美分校 學區傻眼 |accessdate=2011-10-15 |work=世界新闻网 |publisher=世界日报 |date=2011-10-06 |deadurl=yes |archiveurl=https://web.archive.org/web/20160309014309/http://la.worldjournal.com/view/full_la/15922907/article-%E5%BB%A3%E6%9D%B1%E5%90%8D%E6%A0%A1%E8%A8%AD%E7%BE%8E%E5%88%86%E6%A0%A1-%E5%AD%B8%E5%8D%80%E5%82%BB%E7%9C%BC?instance=la_story1 |archivedate=2016-03-09 }}</ref> 但是联合学区却声称当日是“有中国教育界人士来学校参观”，学区负责人，商務部主任西蒙森（Mike Simonson）同时还表示自己“自己从未和中国谈合作”。<ref name="sun">{{Cite web |url=http://the-sun.on.cc/cnt/china_world/20111007/00429_001.html |title=鍾南山母校涉造假 |accessdate=2011-10-15 |work=东方日报网|publisher=太阳报 |date=2011-10-07 }}</ref> 典礼上被宣称为“广东实验中学美国分校校长”的康奈利（Cyrill Connelly）则声称自己仅仅是一所私立小学的校长，因为想拓展中国生源才特地飞来加州参加这个仪式。<ref name="world1" />

凯麦斯教育基金會負責人陈立在一开始对此辩称分校只是作“[[教师|教师]]培训基地”，并表示“当地居民反对声浪大因此决定迁址”。<ref name="sun">{{Cite web |url=http://the-sun.on.cc/cnt/china_world/20111007/00429_003.html |title=辯稱作教師培訓基地 |accessdate=2011-10-15 |work=东方日报网|publisher=太阳报 |date=2011-10-07 }}</ref> 随后国内媒体采访时，他却表示这个学校是“合作办学”，学校只是提出租借，出现这样的问题是中美国情不同的误解。至于网上有人提出的“参观半小时就把美国学校拿下，实为创举”，他声称美国分校筹备已有两年时间。<ref name="ycwb2">{{Cite web |url=http://news.ycwb.com/2011-10/08/content_3590474.htm |title=学校不真实？ 省实美国分校系省实授权合作办学 |accessdate=2011-10-15 |work=金羊网 |publisher=羊城晚报 |date=2011-10-08 }}{{Dead link|date=2018年8月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

对于陈立的答复，西蒙森在10月12日通过邮件回复“关于租借Hillcrest高中，与外界组织没有任何形式的协议，不管是书面的还是口头的”，“Hillcrest高中是納稅人的錢建設，為加州公立學校，供本地學生使用”。<ref name="world2">{{Cite web |url=http://www.worldjournal.com/view/full_news/16026747/article-%E5%BB%A3%E6%9D%B1%E5%90%8D%E6%A0%A1%E8%A8%AD%E7%BE%8E%E5%88%86%E6%A0%A1-%E6%80%A5%E5%96%8A%E5%81%9C?instance=instant |title=廣東名校設美分校 急喊停 |accessdate=2011-10-15 |work=世界新闻网 |publisher=世界日报 |date=2011-10-13 }}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref> 他甚至还说“如果有人说我们已合作，这并非事实，这是[[诈骗|诈骗]]（it's not true, it's fraud）”。<ref name="hexun" />

===对于陈立及基金会的质疑===
美国分校成立，主要的疑点均在凯麦斯基金会负责人陈立身上。在此前省实官网上的报道以及国内媒体的报道中，陈立的身份均为美国“加州教育委员会主任”。然而加州教育委员会办公厅声明，他们并未听说陈立此人，并认为陈立在利用此身份诈骗。<ref name="world3">{{Cite web |url=http://la.worldjournal.com/view/full_la/15972313/article-%E7%89%BD%E7%B7%9A%E8%A8%AD%E7%BE%8E%E5%88%86%E6%A0%A1-%E9%99%B3%E7%AB%8B%E5%AE%98%E6%96%B9%E9%A0%AD%E9%8A%9C%E9%81%AD%E8%B3%AA%E7%96%91?instance=lainstant |title=牽線設美分校 陳立官方頭銜遭質疑 |accessdate=2011-10-15 |work=世界新闻网 |publisher=世界日报 |date=2011-10-08 |deadurl=yes |archiveurl=https://web.archive.org/web/20160308023515/http://la.worldjournal.com/view/full_la/15972313/article-%E7%89%BD%E7%B7%9A%E8%A8%AD%E7%BE%8E%E5%88%86%E6%A0%A1-%E9%99%B3%E7%AB%8B%E5%AE%98%E6%96%B9%E9%A0%AD%E9%8A%9C%E9%81%AD%E8%B3%AA%E7%96%91?instance=lainstant |archivedate=2016-03-08 }}</ref> 至于与陈立同行的所谓“加利福尼亚州教育厅厅长”戴维·朗格，则早已在2008年11月份辞职。<ref name="hexun" />

陈立本人对有关问题的答复也十分模棱两可。最开始，他的说法是省实办一所公办民助性质的分校，凯麦斯基金会提供经费支持，加州政府提供土地出地，省实负责管理和运营；事件曝光后，陈立又改口说因当地居民反对，将寻其它学区办一所私立学校，冠以广东实验中学的名字；随后又变成以基金会名义注册一所私立学校，暂时将省实的学生委托给其它高中培养。<ref name="hexun" /> 目前，陈立发表了《关于广东实验中学美国分校（简称省实美分校）建立的说明》，并表示省实美国分校是他基金会获得了省实的冠名权后在美注册成立的私立学校，与当地Hillcrest高中及Alvord联合学区没有任何关系。<ref name="qiao2">{{Cite web |url=http://www.chinapressusa.com/2011-10/12/content_1067552.htm |title=广东省实验中学造假 中方官员不知情？ |accessdate=2011-10-15 |work=侨报网 |publisher=侨报 |date=2011-10-12 }}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref> 同时他表示，他“没有对省实和省厅说出此‘实情’”，有关人士参加典礼之时还不知道分校是根本没有建成的。<ref name="hexun" />

对于美国凯麦斯教育基金会，其官方网站声称基金会成立于1998年。<ref>{{cite web|url=http://www.chemaxeducation.org/?p=6|title=Introduction简介|accessdate=2011-10-05|work=chemaxeducation.org|publisher=Chemax Education Foundation|date=2011-01-15}}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref> 然而，网上的“Chemax Education Foundation Inc”的搜索结果却显示该基金会成立于2009年12月，同时陈立还是另外一家名称类似的的销售汽车配件的“Chemax International Inc”的主席，且这两家公司的地址完全相同。<ref>{{cite web |url=http://www.corporationwiki.com/California/Chino/chemax-education-foundation-inc/45412103.aspx |title=Chemax Education Foundation Inc Profile|accessdate=2011-10-05|work=corporationwiki.com|publisher=corporationwiki|date=2011-03-30 }}</ref><ref>{{cite web |url=http://www.manta.com/c/mm5mhc6/chemax-international-inc |title=Chemax Education Foundation Inc |accessdate=2011-10-15|work=manta.com|publisher=Company Profile}}</ref> 甚至，在一个旅行社的网站上也声称基金会属于该旅行社名下企业。<ref>{{cite web |url=http://www.apexus4u.com/page/html/index.php |title=公司简介 |accessdate=2011-10-05|work=apexus4u.com|publisher=華美假期  APEX  TRAVEL}}</ref> 而基金会网站上关于美国分校的所有报道全部已经被删除。

现在为止，大部分证据都显示省实美国分校是陈立及美国凯麦斯教育基金会主导的一个骗局。<ref name="qiao0" /><ref name="world4">{{Cite web |url=http://la.worldjournal.com/view/full_la/16026995/article-%E5%88%B0%E5%BA%95%E8%AA%B0%E5%BF%BD%E6%82%A0%E4%BA%86%E8%AA%B0-?instance=lainstant |title=到底誰忽悠了誰? |accessdate=2011-10-15 |work=世界新闻网 |publisher=世界日报 |date=2011-10-13 |deadurl=yes |archiveurl=https://web.archive.org/web/20160308072120/http://la.worldjournal.com/view/full_la/16026995/article-%E5%88%B0%E5%BA%95%E8%AA%B0%E5%BF%BD%E6%82%A0%E4%BA%86%E8%AA%B0-?instance=lainstant |archivedate=2016-03-08 }}</ref> 陈立本人已遭到加州社區學院及大學獨立協會副会长歐克斯（Robert Oakes）向[[奇諾|奇諾]]市警局及[[聖伯納汀諾縣|聖伯納汀諾縣]]地方[[檢察官|檢察官]]辦公室舉報。<ref name="world5">{{Cite web |url=http://la.worldjournal.com/view/full_la/15997655/article-%E9%99%B3%E7%AB%8B%E9%A0%AD%E9%8A%9C%E5%8F%97%E8%B3%AA%E7%96%91-%E9%81%AD%E8%88%89%E5%A0%B1?instance=lainstant |title=陳立頭銜受質疑 遭舉報 |accessdate=2011-10-15 |work=世界新闻网 |publisher=世界日报 |date=2011-10-11 |deadurl=yes |archiveurl=https://web.archive.org/web/20160308192001/http://la.worldjournal.com/view/full_la/15997655/article-%E9%99%B3%E7%AB%8B%E9%A0%AD%E9%8A%9C%E5%8F%97%E8%B3%AA%E7%96%91-%E9%81%AD%E8%88%89%E5%A0%B1?instance=lainstant |archivedate=2016-03-08 }}</ref>

===学校及教育厅反应===
相对于质疑提出之前的高调，作为受骗者，广东实验中学及广东省教育厅官方直至事件发生后差不多两周后才有回应。同凯麦斯教育基金会网站一样，学校网站上的有关报道在数日之后已经全部被删除。

10月12日，广东实验中学校长在学校全體教師會上講話時表示：“經校長辦公會議決定，暫停美國分校建設”。<ref name="world2" />

10月17日，学校表示在美国分校事件上出现了“出乎意料的情况”，学校正在通过多方面渠道调查，并承诺会尽快公布结果。<ref>{{Cite web |url=http://news.uschinapress.com/2011-10/17/content_1070898.htm |title=建美国分校“涉嫌造假” 广东实验中学：正在调查 |accessdate=2011-10-19 |work=侨报网 |publisher=侨报 |date=2011-10-17 }}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

10月18日，广东省教育厅称，美国分校事件“正在处理中”。<ref>{{Cite web |url=http://chinese.uschinapress.com/2011-10/18/content_1071252.htm |title=广东省教育厅：省实美分校事件正在处理中 |accessdate=2011-10-19 |work=侨报网 |publisher=侨报 |date=2011-10-18 }}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

10月19日，学校对此事做出了第一次正式的回应。在向侨报发出的信函中，学校表示由于经验不足，受到了陈立及基金会方面的欺骗，向校友道歉；同时还宣布中止了与陈立的合作关系。<ref name="qiao0" /> 学校网站上发布了内容为“经广东省教育厅批准，我校决定中止与美国凯迈斯教育基金会的合作”的公告。<ref>{{Cite web |url=http://www.gdsyzx.edu.cn/ifshow.php?id=1319024804 |title=公  告 |accessdate=2011-10-20 |publisher=广东实验中学 |date=2011-10-19 }}</ref>

===网民态度===
[[新浪微博|新浪微博]]上最早引用世界日报报道此事是一位自称为所谓资深美国[[移民|移民]][[律师|律师]]的用户“马强移民律师”。而对于此报道，网民有两种看法：一部分人认为学校是真实的，存在误解或者报道错误（包括大部分省实校友）；另一部分人则认为这是一个骗局。

不过，随着可疑点的逐渐出现，学校校友也开始在质疑此事的真实性，并且开始怀疑学校是遭到了陈立的欺骗。<ref name="hexun" /><ref name="qiao2" />

在侨报刊出了省实官方道歉函之后，网络上少数人对此感到安慰，但大多数人抱怨看不到有关事件的报道。还有网民质疑校方道歉的动机，认为此事中仍有“尚未揭开的黑幕”。<ref name="qiao3" />
