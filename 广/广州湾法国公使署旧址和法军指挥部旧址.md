'''广州湾法国公使署旧址和法军指挥部旧址'''位于中国[[广东省|广东省]][[湛江市|湛江市]][[霞山区|霞山区]]海滨一路3号和延安路1号，两者隔延安路相对。1986年广州湾法国公使署旧址被列为[[湛江市文物保护单位|湛江市文物保护单位]]，2002年被列为[[广东省文物保护单位|广东省文物保护单位]]。2013年法国公使署旧址和法军指挥部旧址被列为[[第七批全国重点文物保护单位|第七批全国重点文物保护单位]]。

1899年11月16日，[[法国政府|法国]]与[[清政府|清政府]]签定《[[广州湾租界条约|广州湾租界条约]]》，将[[雷州府|雷州府]][[遂溪县|遂溪县]]和[[高州府|高州府]][[吴川县|吴川县]]交界处的部分陆地、岛屿和两县之间的麻斜湾（今[[湛江港|湛江港]]湾）划为[[法国租界|法国租界]]（其中陆地面积518平方公里），称为“[[广州湾|广州湾租界]]”。广州湾法国公使署旧址（俗称“公使堂”）建于1903年，广州湾法军指挥部旧址（俗称"“绿衣楼”）建于1905年。1943年2月至1945年9月，日军侵占广州湾期间，军部设于“公使堂”，宪兵驻“绿衣楼”。1946年1月15日，[[国民政府|国民政府]]以原广州湾租界范围设立湛江市，市政府驻“公使堂”，“绿衣楼”为警察署。1950年至1954年，湛江市人民政府亦驻于"公使堂"办公。1955年以后，"公使堂"先后为湛江市少年宫、华南工学院湛江分院、湛江无线电一厂等单位使用。“绿衣楼”在1949年以后一直由公安局使用。<ref>{{cite news |language = zh-hans |author = 钟莹 |url = http://news.southcn.com/d/2012-10/17/content_56329963.htm |title = 充分保护利用广州湾法国公使署旧址及广州湾法军指挥部旧址，推动广州湾文化的研究 |publisher = 南方网 |date = 2012年10月17日 |accessdate = 2014年8月20日}}</ref>

2013年旧址周边被清拆，将建设为霞山“法式风情街”。<ref>{{cite news |language = zh-hans |author = 卓朝兴;骆国和 |url = http://www.zhanjiang.gov.cn/fileserver/newshtml/c2ff4378-df1b-4174-b936-90574be2ba2b.htm |title = 广州湾警察署旧址周边被清拆 |publisher = 湛江日报 |date = 2013年8月5日 |accessdate = 2014年8月20日 |deadurl = yes |archiveurl = https://web.archive.org/web/20140808045457/http://www.zhanjiang.gov.cn/fileserver/newshtml/c2ff4378-df1b-4174-b936-90574be2ba2b.htm |archivedate = 2014年8月8日 }}</ref>
