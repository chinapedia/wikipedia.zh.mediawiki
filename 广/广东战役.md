'''广东战役'''，是[[第二次国共内战|第二次国共内战]]中，[[中国人民解放军第二野战军|中国人民解放军第二野战军]]第4兵团、[[中国人民解放军第四野战军|中国人民解放军第四野战军]]兵团等部南下部队，与[[广东|广东]]境内[[中華民國國軍|国军]]的一场[[战役|战役]]。<ref name="军事I，第307页">中国大百科全书总编辑委员会《军事》编辑委员会，《中国大百科全书•军事I》，北京，中国大百科全书出版社，1989年5月上海印刷，第307页，ISBN 7500002424</ref>

== 背景 ==
1948年12月26日，廣東全境實施戒嚴。<ref name="民國史大事記">李新總主編，[[中國社會科學院|中國社會科學院]]近代史研究所[[中華民國史|中華民國史]]研究室編，韓信夫、姜克夫主編，《中華民國史大事記》（全十二卷，共十二冊），[[北京|北京]]：[[中華書局|中華書局]]，2011年7月</ref>{{rp|8761}}1949年8月，解放军进占[[江西|江西]]，准备进军广东。8月24日晨，[[蔣中正|蔣介石]]召見[[顧祝同|顧祝同]]、[[刘安祺|刘安祺]]及[[余漢謀|-{余}-漢謀]]等十餘人，嚴囑不使廣州防衛空虛。<ref>《事略稿本》，1949年8月24日，「總統府」事略室編，《蔣中正總統檔案》，台北「國史館」藏</ref>[[华南军政长官公署|华南军政长官公署]]军政长官余汉谋指挥驻广东3个兵团10个军及保安团队，共约15万人，在其国防部巩固粤北、确保广州指令下，以7个军梯次配置在[[韶关市|曲江]]至广州铁路沿线，2个军驻守[[潮安区|潮安]]、[[汕头市|汕头]]地区，1个军驻[[湛江|湛江]]，企图阻止解放军入粤。第四野战军根据中共中央军委关于解放华南地区的统一部署，决定以第4、第15兵团和[[两广纵队|两广纵队]]共22万余人，由第4兵团[[司令员|司令员]]兼[[政治委員|政治委员]][[陈赓|陈赓]]指挥，于9月底集结湘赣粤边境，在粤赣湘边纵队（即[[林平|林平]]纵队）、闽粤赣边纵队和粤中纵队共4万余人配合下，于[[衡宝战役|衡宝战役]]同时，分3路向广东进军。<ref name="军事I，第307页"/>拟在歼灭余汉谋主力，攻占广州后，第4兵团再西进至粤桂边境，对[[白崇禧|白崇禧]]集团战略迂回。<ref name="军事I，第307页"/>

== 经過 ==
10月2日，[[中国人民解放军|中国人民解放军]]分3路向广东进攻。10月7日，右路第4兵团占领曲江，中路第15兵团进占[[翁源|翁源]]，左路[[两广纵队|两广纵队]]和[[粤赣湘边纵队|粤赣湘边纵队]]占领[[和平县|和平]]、[[河源縣|河源]]等地，直插[[博罗县|博罗]]。<ref name="军事I，第307页"/>在解放军强大攻势下，粤北守军纷纷向广州撤退。<ref name="军事I，第307页"/>解放军长驱直入，在[[佛冈|佛冈]]、[[花都区|花县]]、[[英德市|英德]]、[[清远市|清远]]、[[增城区|增城]]等县城歼灭守军一部。<ref name="军事I，第307页"/>此时，余汉谋部主力向[[粤西|粤西]]溃退。<ref name="军事I，第307页"/>10月14日夜，第15兵团进占广州，两广纵队和粤赣湘边纵队进至[[珠江三角洲|珠江三角洲]]。<ref name="军事I，第307页"/>10月15日，第4兵团以6个师在粤中纵队配合下分3路向粤西追击，西路由[[三水区|三水]]经[[高要区|高要]]、[[阳春市|阳春]]迂回[[阳江市|阳江]]西侧，切断国军入桂道路;东路由[[佛山市|佛山]]经[[鹤山市|鹤山]]、[[台山市|台山]]插向阳江南侧，防止国军由海上撤退；中路由三水经[[高明区|高明]]直出[[恩平市|恩平]]，向阳江追击。<ref name="军事I，第307页"/>该兵团克服种种困难，昼夜兼程猛追，至10月24日将第二十一兵团部及4个军残部共4万余人包围于阳江。<ref name="军事I，第308页">中国大百科全书总编辑委员会《军事》编辑委员会，《中国大百科全书•军事I》，北京，中国大百科全书出版社，1989年5月上海印刷，第308页，ISBN 7500002424</ref>10月25日，第二十一兵团集中3个师向西突围未逞；第4兵团主力趁势从三面攻击，于10月26日将其全歼。<ref name="军事I，第308页"/>第4兵团一部继续西进，至11月4日，先后占领[[罗定市|罗定]]、[[信宜市|信宜]]、[[廉江市|廉江]]等地，切断白崇禧部向海南岛的退路。<ref name="军事I，第308页"/>此时，驻[[潮安区|潮安]]、[[汕头市|汕头]]之[[胡璉|胡璉]]兵团已由海上撤走，闽粤赣边纵队随即解放潮汕地区，战役结束。

== 结果 ==
此役，解放军共歼灭国军6.2万余人。解放军缴获各种炮710余门，轻重机枪2200余挺，各种枪3万余支、汽车150余辆，各种枪弹260余万发<ref name="军事科学">《中国人民解放军全史》军事历史研究部 编，[[军事科学出版社|军事科学出版社]]，2000年，ISBN 7-80137-315-4，卷“中国人民解放军战役战斗总览”。</ref>。除沿海岛屿及[[雷州半岛|雷州半岛]]外，解放军占领广东。
