'''廣島新交通1號線'''（{{jpn|j=広島新交通1号線|ひろしましんこうつういちごうせん|rm=Hiroshima Shinkōtsū Ichigō sen|f=y}}）是一條連結[[日本|日本]][[廣島縣|廣島縣]][[廣島市|廣島市]][[中區_(廣島市)|中區]]的{{link-ja|本通站|本通駅}}至同市[[安佐南區|安佐南區]]的{{link-ja|廣域公園前站|広域公園前駅}}，屬於[[廣島高速交通|廣島高速交通]]的{{link-ja|自動導引捷運系統|自動案内軌条式旅客輸送システム|AGT}}路線。愛稱是'''Astram線'''，帶有日語的「明日」和英語的「Tram」（路面電車）的意思<ref>{{Cite web |url=http://www.astramline.co.jp/q&a.html#a_q4 |title=Q4　アストラムライン」という名前の意味は何ですか？ |website= |work=よくあるご質問（FAQ） |publisher=広島高速交通 |accessdate=2018-12-21 |quote=日本語の「明日」と英語の「トラム（電車）」を組み合わせて、路線を意味する「ライン」を加えました。 全国から応募いただいた15,986件の中から選ばれた愛称で、…}}</ref>。當地簡稱'''Atram'''。

== 概要 ==
=== 路線資料 ===
* 方式：{{link-ja|自動導引捷運系統|自動案内軌条式旅客輸送システム}}
* 路線距離（{{link-ja|營業距離|営業キロ}}）：18.4公里
* 導覽軌條：側方導引式
* 站數：22個（包括起終點站）
* 複線路段：全線
* 電氣化方式：直流750V
* [[閉塞_(鐵路)|閉塞方式]]：車內信號式

== 車站列表 ==
[[File:Astramline_Station_Color.jpg|200px]]
全線位於[[廣島縣|廣島縣]][[廣島市|廣島市]]內。城北站－白島站間的[[新白島站|新白島站]]在2015年（平成27年）3月14日開業<ref name="mainichi-np-2015-3-15" />。除此以外的車站都在路線開業時開業。

Astram線開業時各站，由起點本通站依次序確定車站顏色，分別為{{Color|pink|■}}粉紅色、{{Color|orange|■}}橙色、{{Color|yellow|■}}黃色、{{Color|yellowgreen|■}}黃綠色、{{Color|#66cdaa|■}}綠色、{{Color|lightblue|■}}水藍色、{{Color|#9e9ee2|■}}紫色、（之後重覆）<ref name="living hiroshima" />，官方路線圖等亦有記載車站顏色<ref>{{cite web |url=http://www.astramline.co.jp/route.html |title=アストラムライン路線図 |accessdate=2015年3月16日 |archiveurl=https://web.archive.org/web/20140722072134/http://www.astramline.co.jp/route.html |archivedate=2014年7月22日 }}</ref>，但新白島站開業以後官方網站的路線圖刪除了車站顏色<ref>[http://astramline.co.jp/route.html アストラムライン路線図] - 広島高速交通株式会社</ref>。包括新白島站，各站的車站顏色也不再對外公佈處理。而該站開業後長樂寺站的副站名引入了{{link-ja|廣島市交通科學館|広島市交通科学館}}的{{link-ja|命名權|命名権}}，改稱為「Numaji交通博物館」，牛田站的{{link-ja|廣島市綜合室內游泳池|広島市総合屋内プール|廣島Big Wave}}，改稱為「廣新Big Wave」，官方網站上並沒有更改舊有的標記。
