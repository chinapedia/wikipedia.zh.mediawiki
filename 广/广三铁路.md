'''廣三鐵路，'''又稱'''省三支線，'''是廣東省内的一條鐵路，連接[[廣州市|廣州市]][[石圍塘站|石圍塘站]]及[[佛山市|佛山市]][[三水區|三水區]][[三水南站|三水南站]]，總長49公里。此線是[[廣州|廣州]]同[[廣東|廣東]]最早建成的一條[[鐵路|鐵路]]。

== 歷史 ==
[[File:Map_of_Guangzhou_1942.jpg|thumb]]
據[[三水縣志|三水縣志]]記載，1898年美商{{tsl|en|American China Development Company|華美合興公司}}在承修[[粵漢鐵路|粵漢鐵路]]時，發現廣州至三水沿線地勢平坦，位於[[珠江三角洲|珠江三角洲]]的魚米之鄉，修建鐵路的投資少而收益快，於是提出將廣三鐵路（當時稱為'''省佛支路'''）作為[[粵漢鐵路|粵漢鐵路]]支線並先修建的議案，得到了[[滿清政府|滿清政府]]的支援。 

廣三鐵路自廣州[[珠江|珠江]]南岸[[石圍塘|石圍塘]]，經[[三眼桥站|三眼橋]]、[[佛山|佛山]]、[[小塘|小塘]]至[[三水|三水]]，分兩段先後修築，由華美合興公司投資修築。1901年12月起，開始修築廣州石圍塘至佛山一段，長16.5公里，是中國最早的[[複線|複線]]鐵路。 

1903年10月5日，佛山至三水一段建成通車後，標誌著共耗資4000萬美元的廣三鐵路全線竣工，當時[[兩廣總督|兩廣總督]][[岑春煊|岑春煊]]主持了盛大的通車典禮。此為[[廣東|廣東]]與[[廣州|廣州]]建造鐵路的開端。

民國18年（1929年），國民政府將廣三鐵路併入[[粵漢鐵路|粵漢鐵路]]管理局，改稱'''粵漢鐵路廣三段'''。

民國27年（1938年）)10月21日，日軍进占廣州後，將[[石围塘站|石圍塘]]至佛山右線，及佛山至三水單線被全部拆除，其中部分軌料急運他處鋪設外，剩餘軌料被集中堆放在黃沙車站待運。軌道拆除後的路基被改作公路。到1945年光復後，鐵路由廣東區特派員辦事處接收，對剩餘鐵路進行整修，維持行車。

民國35年（1946年）7月，被拆除的鐵路線全部按原式重新鋪軌通車。

1982年，[[中华人民共和国铁道部|鐵道部]]、[[中华人民共和国化学工业部|化工部]]和[[广东省人民政府|廣東省當局]]聯合投資興建三腰雲段鐵路，為三茂幹線第一期工程。

1987年1月，鐵道部和廣東省當局合作，續建腰茂段鐵路，路線自腰古起，途經新興、天堂、春灣、陽春、石錄、那霍到達茂名，全長231公里。同年3月三茂鐵路公司成立，負責三茂段鐵路建設管理、運營、資金借貸和償還事宜。至1990年聖誕節時，腰茂段鐵路完成鋪軌。

== 發展變化 ==
在三眼橋連接[[广茂联络线|广茂联络线]]可通往[[廣州站|廣州站]]和[[京廣線|京廣線]]，而在三水連接[[三茂铁路|三茂线]]。主要經過廣州、佛山兩市，為[[中國|中國]]連接東西的鐵路線路之一。

2004年2月29日起，因[[广州铁路集团|广州铁路集团]]控股的三茂铁路股份有限公司管辖范围增加原由广铁集团羊城铁路总公司管辖的广三铁路，'''原广三铁路'''和'''[[三茂铁路|三茂铁路]]'''正式合并為'''[[广茂铁路|广茂铁路]]'''。同时[[中华人民共和国铁道部|中华人民共和国铁道部]]将广茂铁路由二级线路调整为一级线路<ref>[http://www.people.com.cn/GB/shizheng/14562/2365538.html 广茂铁路联为一体 铁道部对三茂铁路实现控股]</ref>。全線由廣州—三水、三水—腰古、腰古—茂名三段分期建成。營運權由[[廣州鐵路集團|廣州鐵路集團]]拥有。

在線路上行走之客車主要開往[[廣西|廣西]]、[[雲南|雲南]]、[[貴州|貴州]]、[[四川|四川]]、[[重慶|重慶]]方向。由2007年4月18日起，鐵路線開通了往[[海南|海南]]到京、滬方向客車。每日亦有兩班[[肇九直通車|肇九直通車]]來往肇慶及香港[[紅磡站|紅磡站]]之間，中途停靠[[佛山站|佛山站]]。

原线路未进行[[電氣化鐵路|電力化]]，所有列車需由[[柴油機車|柴油機車]]牽引，2017年国务院批复纳入电力化改造项目。<ref>{{Cite web |url=http://finance.sina.com.cn/money/future/indu/2017-05-25/doc-ifyfqqyh8347279.shtml |title=广茂铁路电化改造列入国家交通基础建设项目 |accessdate=2018-04-08 |work=finance.sina.com.cn}}</ref>

=== 路線資料 ===
* 管轄：[[广州铁路集团|广州铁路集团有限公司]]
* 區間（距離）：三眼桥－三水南(旧站)共49公里
* 軌距：1,435 mm
* 車站數目：12（包括起點及終點站）
* [[复线铁路|雙線]]區間：廣州西站至三眼橋站（其餘為單線）
* [[電氣化鐵路|電化]]區間：廣州站至廣州西站（其餘為非電化）
* 最高速度：95 km/h

==== 速度 ====
全線限速80—100公里
