'''广教寺'''，位于[[江苏省|江苏省]][[南通市|南通市]][[崇川区|崇川区]][[狼山镇|狼山镇]]的[[狼山_(江苏)|狼山]]，[[汉族地区佛教全国重点寺院|汉族地区佛教全国重点寺院]]之一。<ref>{{Cite book|title=江苏省志：文物志|publisher=江苏古籍出版社|year=1998|isbn=9787805199085|location=南京|pages=296-297}}</ref>

== 历史沿革 ==
广教寺始建于[[唐朝|唐]][[總章|总章]]二年（669年）<ref>《通州志》记载：“唐总章二年，山上即建大雄宝殿、殿阁、方丈室。山在巨浸中，设舟以济，号慈航院，后改广教寺。”</ref>，其开山祖师为[[僧伽|僧伽]]，[[唐中宗|唐中宗]]尊为[[国师|国师]]。[[宋朝|宋朝]][[太平兴国|太平兴国]]年间[[智幻法师|智幻法师]]住持广教寺，其弘法修寺，建造大圣殿、支云塔。

==建筑风格 ==
广教寺的寺院建筑遍布狼山。山顶有山门、萃景楼、圆通宝殿、大圣殿，还有葵竹山房、三仙祠、云塔等。山下建筑群有大佛殿、轮藏殿、大悲殿、金刚殿、藏经楼、晒经楼、枕山楼、方丈室及僧宏等。其旁有[[骆宾王|骆宾王]]墓、望江亭、御亭、平倭碑亭、[[白雅雨|白雅雨]]墓等景观，亦有双眼石、鸽子岩、磊落矶、寒玉泉等名胜<ref>[http://www.foz.cn/Article/mingsi/huadong/jiangsu/Article_1067.html 南通广教寺] {{webarchive|url=https://web.archive.org/web/20070830011309/http://www.foz.cn/Article/mingsi/huadong/jiangsu/Article_1067.html |date=2007-08-30 }}</ref>。

支云塔屹立在狼山之巅，是广教寺最有特色的建筑。塔建于北宋太平兴国年间，高35米，木质结构，五级四层，四面正方，各层有小间，由下向上次第收缩<ref>{{Cite web |url=http://www.gutrip.com/html/yuyin/shanghai/guangde/2009/0205/263.html |title=广教寺双塔 |access-date=2009-11-12 |archive-url=https://web.archive.org/web/20160305063952/http://gutrip.com/html/yuyin/shanghai/guangde/2009/0205/263.html |archive-date=2016-03-05 |dead-url=yes }}</ref><ref>[http://www.foyin.org/plus/view.php?aid=68932 狼山广教寺-江苏] {{webarchive|url=https://web.archive.org/web/20090922085114/http://www.foyin.org/plus/view.php?aid=68932 |date=2009-09-22 }}</ref>。
