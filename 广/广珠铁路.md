'''廣珠鐵路'''是一條縱貫[[珠三角|珠三角]]西岸的[[鐵路|鐵路]]，連接[[廣州|廣州]]、[[佛山|佛山]]、[[江門|江門]]、[[珠海|珠海]]四市，線路全長186.23公里，以貨運為主。[[廣州鐵路集團|廣州鐵路集團]]的子公司[[廣珠鐵路有限責任公司|廣珠鐵路有限責任公司]]於2007年1月在珠海掛牌成立。

== 歷史 ==
廣珠鐵路是[[中華人民共和國鐵道部|鐵道部]]、[[廣東省|廣東省]]合作建設的項目，1993年經國家計委立項，1997年施工，1999年因資金、股權、沿線土地等問題而停工。2005年12月，廣東省人民政府、鐵道部聯合向國家發改委上報了廣珠鐵路復工可研報告，2006年3月鐵道部、廣東省政府、珠海市政府、江門市政府的出資代表共同簽署了合資修建廣珠鐵路的《合同》和《章程》。為了推進廣珠鐵路復工，2006年3月7日，廣珠鐵路籌備組成立了項目工作小組開展前期籌備工作。<ref>[http://www.nanfangdaily.com.cn/southnews/dd/nfrb/A07/200701160150.asp 广珠铁路最快春节前全面复工]</ref>2007年，廣珠鐵路正式復工，<ref>{{Cite web |url=http://www.ycwb.com/xkb/2007-01/16/content_1351461.htm |title=停工八年广珠铁路复建　在新轨运行不停广州 |access-date=2009-04-05 |archive-url=https://web.archive.org/web/20101123005942/http://www.ycwb.com/xkb/2007-01/16/content_1351461.htm |archive-date=2010-11-23 |dead-url=yes }}</ref> 并于2012年12月30日正式通車。<ref>[http://news.sohu.com/20121229/n362014561.shtml 广珠铁路首趟货运列车从珠海高栏港站首发(图)]</ref>

== 线路 ==
自[[京广鐵路|京广鐵路]]的[[江村站|江村站]]南端引出，并行[[佛山一环高速公路|佛山一环高速公路]]、[[广州绕城高速公路|广州绕城高速公路]]至西江，西江至虎跳门段线路沿[[江肇高速公路|江肇高速公路]]至江门，经古井镇、跨越虎跳门水道进入[[珠海市|珠海市]][[斗门区|斗门区]]，沿[[珠港大道|珠港大道]]南行至[[高栏港|高栏港]]。

為免與[[廣珠城際軌道交通|廣珠城際軌道交通]]競爭，广珠铁路按照国铁一级标准建设，设计时速120公里，广珠铁路开通初期货运为主，预留了客运条件，設計年貨運量6000萬噸。

未来运行时速将达120公里<ref>[http://news.takungpao.com.hk/mainland/topnews/2016-07/3342148_print.html 珠江西岸將新添客運動車線 廣珠鐵路客運改造年底開工]</ref>

== 建設 ==
廣珠鐵路建設自有資本金為總投資的40%，其中鐵道部出資40%，廣東省出資25%，珠海市出資30%，江門市出資5%。其餘部分通過銀行融資。總投資估算為45.8億元。

《广州枢纽新建丹灶至佛山西联络线及广珠铁路客运改造工程环境影响报告书》在中铁第四勘察设计院网站公布，披露了不少新细节：新建丹灶至佛山西联络线将有大约10公里路段是双线并行，而广珠铁路客运改造工程则将增建西樵客运站，未来该线路将会是运营时速达120公里的动车。建设单位表示，该线路今年将动工开建，要通车起码要到2017年或之后，其中工程总投资高达46亿多元。

环评报告书显示，广州枢纽丹灶至佛山西联络线位于佛山市南海区，起自广珠铁路丹灶站至官窖站区间路基，终至佛山西站西端预留接轨点。其主要作用是连通深茂铁路经广珠铁路至佛山西、广州的客车运输径路。而广珠铁路[[江门南站|江门南站]]（不含）至[[丹灶站|丹灶站]]（含）客运改造工程，位于江门市和佛山市。

从环评来看，丹佛联络线推荐广珠铁路区间接轨沿南广铁路新建联络线引入[[佛山西站|佛山西站]]方案，联络线左线（上行线）自广珠铁路上行线出岔，右线（下行线）广珠铁路下行线接岔，左线沿广珠铁路西侧北行，右线上跨广珠铁路后与左线并行，后两线并行沿南广线南侧前行，跨南沙涌、北江及桂丹路后，左上跨贵广（南广）铁路、[[广茂铁路|广茂铁路]]，沿站前路跨桂丹路及其匝道，上跨佛山一环公路，至联络线左线终点接入[[佛山西站|佛山西站]]；右线并行于南广铁路南侧继续前行，上跨[[桂丹路|桂丹路]]、[[佛山一环公路|佛山一环公路]]、广茂铁路至联络线右线终点接入佛山西站。

此外，新建广珠铁路[[丹灶站|丹灶站]]（不含）至[[佛山西站|佛山西站]]（不含）联络线，左线长 17.36 公里，右线长 18.111 公里，其中双线并行地段长 9.60 公里。广珠铁路江门南站（不含）至丹灶站（含）客运改造工程，长约 73.13 公里，包括既有鹤山站增建客运设施及增建西樵客运站等改造工程。<ref>[http://www.sohu.com/a/73143160_115401 广珠铁路将进行客运改造 并新建联络线接深茂铁路]</ref><ref>[http://www.sohu.com/a/205730763_212619 广珠铁路将要改造！江门将直达佛山西 融入广佛都市圈]</ref>
