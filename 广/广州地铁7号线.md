'''广州地铁7号线'''是广州地铁已通车的路线之一，屬於城市軌道交通系統。代表色為嫩綠色。現時全线均為地下線，大致呈西南─東北走向，连接[[广州南站_(地铁)|广州南站]]至[[大学城南站_(广州)|大学城南]]。车厢编制为6节B型编组列车。

由于目前首期工程全线均位于[[番禺区|番禺区]]内，因此7號線是廣州地鐵目前全線皆在同一行政區內的兩條路線之一（另一條為[[广州地铁8号线|8号线]]）。

七號綫亦是廣州首條使用國產化信号系统（鐵科院MTC-I CBTC）的軌道交通路綫，由广州市地下铁道总公司和铁科院（中国铁道科学研究院）联合研制<ref>{{cite news| url=http://news.southcn.com/gd/content/2016-12/14/content_161615415.htm| title=广州3条新地铁线月底开通 大学城南站换乘仅需1分钟| publisher=南方网| date=2016-12-14}}</ref>。

== 概要 ==
[[File:GZMTRLine7InScale.png|350px]]
七号线分为首期、首期西延段和二期。首期已于2016年12月28日开通。

'''七号线首期'''工程路线长约21.1km，设9座车站，線路从广州南站引出，緊貼二号线東側平行前進，經過石壁站后沿汉溪大道敷设，經過鍾村、漢溪長隆和規劃當中的番禺萬博商圈，之后转向兴南大道繼續往東行進，經過雅居樂劍橋郡等小區之后大致轉向北，过珠江后航道，到达[[广州大学城|广州大学城]]并经过广东工业大学，到达首期終點[[大学城南站|大学城南站]]。

本綫在2號綫大洲停車場附近设车辆段1处，兩者中間夾住廣州南車廠；七号线控制中心设在大石与三号线合用；扩建三号线金山主变电站及四号线兴业主变电站对七号线供电。

== 沿革 ==
=== 规划 ===
==== 首期／二期工程 ====
20世纪90年代，当时的[[广州市政府|广州市政府]]计划将番禺撤市建区，由于有反对声音，当局承诺以后将[[广州地铁3号线|广州地铁3号线]]通往番禺，并且重点发展番禺作为做条件。2000年，[[番禺|番禺]]正式撤市设区，番禺进入新的发展阶段。撤市设区后，广州市政府亦落实[[广州大学城|广州大学城]]和广州新火车站落户番禺，因应发展需要，这条全线在番禺区境内的地铁线路亦应运而生。

在广州市轨道交通规划2002年的备选方案中，出現了現有七號綫首期工程段的雛型，当时该线的名称为'''芳村-番禺-黄埔线'''，从西朗开始，经由番禺区的南浦岛、大石、新造、化龙之后，到达东端终点站开发区<ref>[http://www.guangzhou.gov.cn/zhuanti/2002/metro-fanan/index.htm 广州地铁规划方案征集市民意见] {{webarchive|url=https://web.archive.org/web/20170411140411/http://www.guangzhou.gov.cn/zhuanti/2002/metro-fanan/index.htm |date=2017-04-11 }}</ref>。随后在2003年方案中，大石以东的走向改经由沙溪、大学城、长洲岛之后进入黄埔，同时二期走向基本与现时一致，但未初步定型。

随后在2003年的局部修改方案中，本站西端起訖站改于当时落实兴建的广州新客站（现[[广州南站_(地铁)|广州南站]]）。官堂站（现[[员岗站|员岗站]]）以东的走向改由与现在基本一致以外，当初的线路走向是经由塘步西、大石、[[长隆欢乐世界|香江野生动物园]]一带，并在[[大石站|大石站]]规划与现在的[[广州地铁3号线|3号线]]进行换乘。

2004年1月6日，七号线两端起訖站之一的南亭站（现[[大学城南站|大学城南站]]）动工；随后于同年的12月30日，另一端起訖站——广州新客站随着铁路站动工而开始建设。两个车站分别为现在[[广州地铁4号线|4号线]]和[[广州地铁2号线|2号线]]/[[佛山地铁2号线|佛山地铁2号线]]的换乘车站，加上大石站换乘3号线，方便乘客换乘三条线前往广州市区以及市桥、南沙等地。不过在当时的规划中，本线采用4节L型编组地铁列车，因此两个车站的预留结构均按照4节L型编组的規模進行預留。不过后来受到当时3号线使用“小编组，高密度”运营模式失败的影響，本线改為采用6节B型编组的地铁列车设计，因此两个车站的预留结构均要进行改建，以配合本线的运营需要。

[[File:Guangzhou_Metro_Line_2-CSR_L4_train-MapInfomation.JPG|thumb]]
後來2003年方案局部修改，本线與4號線的換乘站一度改為[[新造站|新造站]]，并在新造站开始向东延伸至化龙，最后过江到达[[黄埔区|黄埔区]]设终点[[大沙东站|大沙东站]]，并与当时落实兴建的[[广州地铁5号线|5号线]]交汇，意味着本线将跨过番禺区范围，形成一條廣州市區外圍近郊的切向線。另外与3号线的换乘站改为[[汉溪长隆站|汉溪长隆站]]，并将官堂站以西走向改由现在沿汉溪大道的走向。後來於2010年規劃當中，7號線確定改回在其预留换乘车站大学城南站與4號線交匯，以及7号线与2号线增加[[石壁站|石壁站]]為換乘站，本线首期工程线路亦基本定型。同时大学城南－大沙东一段的二期工程，亦改经由[[长洲岛|长洲岛]]然后过江。

2012年，首期工程獲得國家發改委批覆立項。

2015年，因應原黃埔區和[[蘿崗區|蘿崗區]]合併為新的黃埔區，二期線路規劃亦從大沙東站繼續往北大幅延長至[[蘿崗站|蘿崗站]]，與當時仍然在建的[[广州地铁6号线|6号线]]二期交匯。7號線二期因此也直接串連起原黃埔和蘿崗兩地的中心區域，解決东部片区南北走廊的交通需求。2016年，二期線路再次北延至[[水西站|水西站]]，與在建的[[广州地铁21号线|21号线]]交匯；其後確定規劃再度延長至[[水西北站|水西北站]]，作為21號線[[水西停車場|水西停車場]]上蓋物業的配套。2018年，根據广州市环保局公布广州市轨道交通七号线二期工程环境评估報告，停車場選址由飛鳳山更改為科林路以南、科丰路以西的地块<ref>[http://news.sina.com.cn/o/2018-01-16/doc-ifyqtycw8442982.shtml 广州又一地铁项目获批，可换乘7条线，这两个区最受益]廣州日報 2018-1-16</ref>。

2017年第一季，二期工程獲國家發改委批覆立項。2018年1月，廣州市發改委正式批覆二期工程可行性研究報告<ref>[http://news.ycwb.com/2018-08/24/content_30072970.htm 广州又一地铁项目获批，可换乘7条线，这两个区最受益]金羊網-新快報 2018-8-24</ref>，同年11月19日正式開工建設。

==== 西延至佛山顺德 ====
在7号线首期的规划中，本线位于广州南站的一端起訖站曾一度改为临近廣佛邊界的[[韦涌站|韦涌站]]。

广州南站正式启用后，受到[[廣佛同城化|廣佛同城化]]的影響，以及佛山地鐵線網規劃改動，因此[[佛山|佛山市政府]]提议将本线直接西延至[[顺德区|顺德区]]的陈村、北滘，并在[[北滘新城站|北滘新城站]]与[[佛山地铁3号线|佛山地铁3号线]]换乘。最后广州地铁采纳佛山方面的意见。随后在提议规划中的西延段局部修改，北滘新城站由原来起訖站改为中途站，并延伸至美的大道处设[[美的大道站|美的大道站]]为起訖站。

2015年10月9日，西延顺德段项目建设方案调整正式获得国家发改委批复同意<ref>[http://gd.qq.com/a/20151010/051974.htm 广州地铁7号线确定延至顺德！设哪7站？带你先睹为快！]腾讯新闻 2015-10-10</ref>。随后於2016年6月23日正式動工。

=== 建造 ===
==== 首期工程 ====
*2013年4月1日，石壁站、钟村站、汉溪长隆站、鹤庄站（现[[南村万博站|南村万博站]]）动工，标志着七号线首期工程建设的正式开始。稍后在6月18日和12月18日，南村站（现[[板桥站_(广州)|板桥站]]）和[[谢村站|谢村站]]亦开始动工建设。官堂站（現[[員崗站|員崗站]]）最后于2014年11月28日动工。

*2014年5月27日，广州南站连接大洲车辆段的入段线长达1238米的盾构隧道贯通，成为七号线首期工程首个贯通的隧道<ref>[http://www.gzmtr.com/ygwm/xwzx/gsxw/201405/t20140530_35353.html 广州地铁七号线首个盾构隧道贯通]</ref>。2015年1月13日，鹤庄站主体结构封顶，成为7号线首个封顶的车站<ref>[http://gzdaily.dayoo.com/html/2015-01/13/content_2841876.htm 鹤庄站主体结构封顶] {{webarchive|url=https://web.archive.org/web/20150113075716/http://gzdaily.dayoo.com/html/2015-01/13/content_2841876.htm |date=2015-01-13 }}</ref>。

*2015年1月5日，广州南站至石壁盾构区间左线贯通。同年1月18日和28日，板桥站和石壁站封顶。2015年3月，汉溪长隆至南村万博站开始进行区间挖掘，至此七号线全面进入区间盾构掘进的新阶段<ref>[http://epaper.xxsb.com/showNews/2015-03-19/220977.html 七号线土建工程累计完成52%] {{webarchive|url=https://web.archive.org/web/20150402094434/http://epaper.xxsb.com/showNews/2015-03-19/220977.html |date=2015-04-02 }}</ref>。

*2016年2月4日，钟村站主體結構封頂，標誌着七號線全線車站完成主體結構工程；随后于同年2月28日，汉溪长隆-鹤庄盾構隧道完工，標誌着七號綫全線隧道貫通；5月4日，七号线首批列車交付廣州地鐵<ref>[http://news.dayoo.com/guangzhou/201605/04/139995_47421102.htm 七号线一期首批列车今日交付广州地铁调试 首条地铁全部列车“广州造” ] {{webarchive|url=https://web.archive.org/web/20160506210652/http://news.dayoo.com/guangzhou/201605/04/139995_47421102.htm |date=2016-05-06 }}</ref>；8月26日，七号线全线长轨顺利贯通。

*2016年9月28日，一期全部車站及軌行區從建造事業總部將属地管理权、调度指挥权、设备操作维护使用权移交給運營事業總部，隨即於29日進入運營調試階段<ref>[http://www.gd.xinhuanet.com/newscenter/2016-09/29/c_1119648240.htm 广州地铁七号线一期进入运营调试阶段 年底开通试运营]</ref>；11月20日，一期開始全線進行營運時刻表演練，不載客試運行。

*2016年12月26日，廣州地鐵在微博公佈7號線一期將於12月28日中午12時開始載客試運營<ref>[http://weibo.com/2612249974/EnUDpur1A?from=page_1001062612249974_profile&wvr=6&mod=weibotime&type=comment#_rnd1482770041543 【定了！12.28见】]廣州地鐵官方微博</ref>。

*2016年12月28日，廣州市地下鐵道總公司舉辦七號線首期開通見證會。同日中午12時，七號綫首期正式開通試營運。

== 车站 ==
七号线共設有28個車站，首期9個車站，全線位於番禺區內；二期11個車站，通向[[黄埔区|黄埔区]]；西延段8个车站，其中7个车站位于佛山市順德區。

=== 換乘站 ===
* '''[[广州南站_(地铁)|广州南站]]'''：换乘[[广州地铁2号线|2号线]]。本站亦是2号线和7号线的终点站，兩線共用一岛两侧的月台佈置，但因為只有2號線始发站台和7号线到达站台位于岛式站台上，其餘兩個方向分別位於兩個側式月台上。因此只有7号线抵站乘客换乘2号线能够同台换乘，2号线抵站乘客如需换乘7号线需通过站厅步行到另一个侧式站台。
* '''[[石壁站|石壁站]]'''：换乘[[广州地铁2号线|2号线]]。因建造时未作预留，且两线站台相距较远、站體错开及并不完全平行，需经站廳通道换乘。
* '''[[汉溪长隆站|汉溪长隆站]]'''：换乘[[广州地铁3号线|3号线]]。因建造时未作预留，需经通道换乘。
* '''[[大学城南站_(广州)|大学城南站]]'''：换乘[[广州地铁4号线|4号线]]。4号线建造时已同步兴建下层7号线的岛式站台，该站台在设计上同时亦承担了连接4号线两个方向的侧式站台之用，为十字型节点式换乘，惟預留結構僅可供4節編組L型車使用，現時7號綫改為使用6節編組B型車，故動工時需要對預留結構進行擴建。

==== 將會成為換乘站的車站 ====
* '''[[南村萬博站|南村萬博站]]'''：换乘[[广州地铁18号线|18号线]]及[[佛山地铁4号线|佛山地铁4号线]]。因建造时未作预留，预计需经通道换乘。

==== 將會擴建的換乘站 ====
* '''[[广州南站_(地铁)|广州南站]]'''：换乘[[广州地铁22号线|22号线]]和[[佛山地铁2号线|佛山地铁2号线]]。建設時均未作預留，廣州22號線設於廣州南站東廣場；佛山2号线車站則設置在廣州南站西廣場，兩線均會設有通道連接既有2號線和7號線車站。
* '''[[大學城南站_(廣州)|大学城南站]]'''：换乘[[广州地铁12号线|12号线]]。建造時未作預留，預計為站廳通道換乘。

==== 規劃二期和西延段換乘站 ====

==行车安排==
七號綫將採取單一路線營運，所有列車皆往返大学城南站及廣州南站之間。

開通初期高峰期行車間隔為4分50秒一班。

2018年春运返程期间的2月27日至3月5日，廣州南站往大学城南站的常規尾班車開出之後，會於23:45和翌日00:45再開出兩班加班車，供廣州南站夜間高鐵抵站乘客接駁進入番禺區南村及大学城各地。加班車中途只停靠汉溪长隆、员岗，並以大学城南為終點站<ref>[http://gd.sina.com.cn/news/2018-01-30/detail-ifyqzcxi1584138.shtml?from=gd_cnxh 春节前后广州地铁部分线网将延长服务时间(详细)]</ref>。乘客抵站後均無法換乘其它地鐵線路，只能出站轉乘其它交通工具。

== 列车 ==
{{main|廣州地鐵七號線列車}}
路线采用六节编组B型车辆，车辆旅行速度为43km/h，最高运行速度80km/h，高峰最高断面为3.06万人/小时。

首期相关列车合约已被[[中国南车|中国南车]]中标，共138辆（23列），除首列列車（07x001-07x002）于南车株洲电力机车公司生产外，其餘列車均在[[南车株洲电力机车|南车株洲电力机车公司]]属下的廣州中車軌道交通裝備公司生产（首列车于南车株洲电力机车公司试制，已于2015年2月开始制造）。

七號線原計劃使用4節編組L型列車進行營運，但後來由於僅使用3節編組B型列車的3號線和4節編組L型列車的6號線過於擁擠，意味着「小編組高密度」模式無法滿足骨幹線路的需要，故地鐵公司其後把7號線改為使用6節編組B型列車。

== 未來發展 ==
===西延段===
'''七號線首期西延順德段'''（以下簡稱西延段）線路從廣州南站引出，大致往西南方向行走，設韦涌站後跨境进入佛山市顺德區，下穿广珠西线及广明高速後大致轉向南沿陳村大道行走；隨後線路通過南涌大橋東側，轉入105國道，通過林頭大橋南側後轉入林上北路向西北方向行走，最後轉向東北方向進入益豐路抵達終點站美的大道站<ref>广州市軌道交通七號線一期工程西延順德段項目環境影響報告書</ref>。

西延段將在美的大道站後增設益豐停車場，位于潭州水道西南侧，君兰高尔夫练习场北侧，规划益丰路西侧地块。該停車場將設置主變電站供本綫供電。

===二期===
二期路线从大学城南站东端引出，下穿华南理工大学、广州中医药大学后，沿中环东路、广中医路、金洲南路行走，在深井村南侧设置深井站与穗莞深琶洲支线大学城东站换乘；綫路沿金洲北路敷设，在下庄村前设长洲站与规划八号线东延线换乘；其后线路穿越大片农用地及下穿珠江，在洪圣沙站货场内设洪圣沙站；线路向东北斜穿珠江，下穿黄埔港后进入丰乐南路，在金逸雅居小区前设裕丰围站（原丰乐南路站）与十三号线换乘；出站后继续向北敷设，之后折向东北，穿越地块后，在镇东路与大沙地东路交叉口北侧设大沙东站与五号线换乘；綫路其后向北延伸，下穿广九铁路后折向西北进入丰乐北路，在姬堂路与丰乐北路交叉口处设姬堂站与规划十九号线换乘，站后接出入场线；之后沿科丰路行走，在南云五路和科丰路交叉口设加庄站；出站后线路继续向北敷设，在科丰路与开泰大道交叉口处设科丰路站，之后线路下穿地块折向东北进入开达路，在香雪三路与开创大道交叉口北侧，广州国际体育演艺中心前设萝岗站与六号线换乘，线路继续向北，在水西保障房地块内设水西站与二十一号线换乘，出站后轉向西北，在二十一号线水西停车场旁邊设终点站水西北站。

二期工程线路长22.0km，共设11座车站，平均站间距2.02km，最大站间距3.28km，為大学城南站至深井站區間，最小站间距0.96km，為水西站至水西北站區間。本工程將增設上堂停車場，位於黃埔區科林路以南、科豐路以西。擴建五號線魚珠主變電站和四號線興業主變電站對七號線供電。

2016年11月，二期规划已通过专家评估<ref>[http://cydb.gdd.gov.cn/view.asp?id=40992 两条地铁规划通过专家评估]創業導報，2016年11月22日</ref>。目前是廣州市新一輪軌道交通（2015-2025）建設規劃方案的候補線路之一，2017年3月下旬獲國家發改委批覆立項，2018年1月，七號線二期工可報告已獲市發改委批复，可以動工建設，同年11月正式開工。
