'''广东工业大学'''，简称'''广工'''，是[[中国|中国]][[广东省|广东省]]一所以工为主、工理经管文法结合的、多科性协调发展的重点大学，始建于1952年。主校区位于[[广州市|广州市]][[广州大学城|广州大学城]]。初期校本部设在原广东工学院的校址——广州市[[东风路_(广州)|东风东路]]729号，直到2005年校本部搬至广州大学城。<ref>{{cite web|title=广东工业大学简介|url=http://www.gdut.edu.cn/xxgk1/xxjj.htm|date=2018-05-01}}</ref>

== 历史沿革 ==
广东工业大学由[[广东工学院|广东工学院]]、[[广东机械学院|广东机械学院]]和[[华南建设学院|华南建设学院]]东院于1995年合并而成。

广东机械学院前身为创立于1956年的广东省机械学校。文革期间，1968年被迫停办。于1978年更名广东机械学院，简称“广机”。
华南建设学院东院创立于1956年。也曾于1963年和1968年停办。简称“华建东”。

广东工学院为广东工业大学的主体，办学历史可以追溯到1952年<ref>[http://www.gdut.edu.cn/newgdut/bgfw/xxjj.htm 广东工业大学历史沿革] {{webarchive|url=https://web.archive.org/web/20120510190337/http://www.gdut.edu.cn/newgdut/bgfw/xxjj.htm |date=2012-05-10 }}</ref>：
*1952年，广州水利土木工校创立。
*1955年，广州水利学校。
*1959年，广州水利电力学院：由广东电力学校，广东省水利科学研究院和广东省电业局广州中心试验所合并组成。
*1961年，中南科技学院：由广州工学院和广东科技学院合并成立。
*1962年，广东工学院：由中南科技学院合并广州水利电力学院。
*1970年，广东矿冶学院（韶关），广东工学院（广州五山）：[[文化大革命|文化大革命]]期间，广东工学院被迫停办。部分师生迁至位于广东[[韶关市|韶关市]]市郊的[[南华寺|南华寺]]办学，并改名广东矿冶学院。同年，位于广州五山的华南工学院（现[[华南理工大学|华南理工大学]]）更名为广东工学院，直到1978年。<ref>{{cite web|title=学校沿革|url=http://www.scut.edu.cn/new/9016/list.htm|website=华南理工大学}}</ref>。
*1982年，广东工学院：广东矿冶学院由韶关南华寺迁回广州原[[东山区_(广州)|东山区]]的校园（即现东风路校区）复办广东工学院。但是由于文革期间校园大量土地遭受到严重破坏和被民居占用，以至于复校的时候只剩下300多亩。校园更被文革期间建成的环市东路分隔成南北两个校园。
*2018年，拟将广东工业大学华立学院部分专业和广东省华立技师学院搬迁至江门校区。

== 学校概况 ==
=== 校区分布 ===
* 大学城校区（校本部）：位于广州大学城外环西路100号。占地2000多亩，是大学城十校当中面积最大，人数最多的高校，邻近[[广州地铁|广州地铁]][[大学城南站|大学城南站]]。容纳了现在广东工业大学70%的院系。

* 东风路校区：位于[[越秀区|越秀区]]（原[[东山区_(广州)|东山区]]）东风东路729号。是前广东工学院的校园，占地300多亩。位于广州市中心黄金地段，邻近[[羊城晚报|羊城晚报]]社和[[广州地铁|广州地铁]][[区庄站|区庄站]]。是广州闹市当中少有的大学校园，横跨广州市区两大主干道路，东风路和环市路。邻近天河商业圈，与广州三大商业街之一的[[农林下路|农林下路]]相连，交通生活非常方便。东风路校区被环市东路分成南北两部分。南面称为南苑，为教学区和运动场等。北面称为北苑，为生活区，有学生宿舍，教工宿舍和饭堂等。南北苑由一'''工'''字型天桥连接。自1995年广东工业大学合并以来，东风路校区一直是广东工业大学的校本部，直到2005年校本部搬到广州大学城。东风路校区现在设有建筑和城市规划学院，艺术与设计学院，政法学院和继续教育学院。

* 龙洞校区：位于[[天河区|天河区]]龙眼洞，迎龙路161号，占地面积589亩。邻近[[华南植物园|华南植物园]]。[[龙洞街道_(广州市)|龙洞]]校区1999年落成，最初目的是作为大学本科一年级和二年级学习的校区，目的是为了缓解当时两大主要校区东风路和五山校区的教学资源与用地负担。后因2004年广州大学城落成，广工取得其中一块占地最大，2000多亩的新校区，而取消龙洞校区作为大学本科一年级和二年级学习的功能，现在设有管理学院、经济贸易学院和应用数学学院。

* 番禺校区：位于[[番禺区|番禺区]][[钟村镇|钟村镇]]。原为'''广东工业大学番禺学院'''，番禺校区现设有国际教育学院。

* 五山校区（已撤销）：位于天河区五山[[广园路|广园东路]]，占地面积203亩，有楼高十四层的综合楼，原建设学院、材料能源学院和信息工程学院设于该校区。五山校区是原来广东机械学院的校园，与华南理工大学相邻。1995年广东工业大学合并以后，一直作为五山校区。至2006年，所有的师生和教学设备都搬到其他校区，校园转售给[[广东外语艺术职业学院|广东外语艺术职业学院]]。

* 沙河校区：位于天河区[[沙河大街|沙河大街]]。[[沙河街道_(广州市)|沙河]]校区是原来华南建筑学院东院的校园。1995年合并到广东工业大学以后，一直作为沙河校区。目前作为成人教育及开办培训课程的校区，不再承担一般教学任务。

* 广东工业大学华立学院(现为独立学院): 位于广州[[增城市|增城市]]，于2001年4月成为广东工业大学的二级学院；2004年4月，经国家教育部批准，成为广东省首批本科民办[[独立学院|独立学院]]。

* 江门校区：位于江门市新会区。华立科教城內。

=== 现况 ===
目前全日制在校生45000余人，其中本科生38000余人，研究生7000余人，并招有不同层次的成人学历教育学生、港澳台生和外国留学生，已形成“学士-硕士-博士”完整的人才培养体系。
校园占地总面积3348亩，有大学城、东风路、龙洞以及番禺等多个校区。设有19个学院、一个独立学院——华立学院和3个公共课教学部（中心），68个本科专业以以及全日制在校生45000余人，并招有港澳台生和外国留学生，是广东省最大的大学（按学生规模）。
学校有职称自主评审权，现有专任教师2000多人，其中正高级职称300多人，副高级职称700多人。2011年以来，学校已引进“百人计划”特聘教授百余名和3名学院院长。其中“长江学者”9人、国家“杰青”9人、国家“千人计划”教授17人、国家“优青”5人、国家”青年千人”教授11人、教育部“新世纪优秀人才”5人、广东省“领军人才”7人、广东省“珠江学者”23人、广东省“杰青”10人等，同时还聘有外籍院士4人、中国工程院院士4人，已组建并入选广东省“创新团队”9个。

== 专业概况 ==
广东工业大学现有19个学院，包含机电工程学院、自动化学院、轻工化工学院、信息工程学院、土木与交通工程学院、管理学院、计算机学院、材料与能源学院、环境科学与工程学院、外国语学院、应用数学学院、物理与光电工程学院、艺术设计学院、政法学院（思想政治理论课教学部）、建筑与城市规划学院、经济与贸易学院、国际教育学院、继续教育学院等。本科专业共有84个，[[硕士|硕士]]专业有88个，博士专业则有27个<ref name=数据/>。学校还设有4个公共课教学部(中心)，5个博士后科研流动站，5个一级学科博士学位授权点，17个一级学科硕士学位授权点，7个学科教授评审权，3个省攀峰重点学科一级学科，7个省优势重点学科一级学科，5个省特色重点学科二级学科，有工程硕士（17个领域）、工商管理硕士、工程管理硕士3种专业学位授予权。2012年以来，工程学位居ESI世界排名前1%行列。2017年7月，材料科学进入ESI世界排名前1%行列。

据《广东省教育厅关于广东工业大学等5所普通高等学校调整招生录取批次的意见》（粤教考函[2014]15号），广东工业大学所有专业自2014年起在广东省普通高等学校第一批本科录取批次招生。

== 其他 ==
广东工业大学继2011年首次问鼎全国大学生创新“挑战杯”金奖后，2012年在全国大学生创业“挑战杯”中再夺1金2银；在全国大学生电子商务大赛暨全国高校“创意、创新、创业”电子商务挑战赛总决赛中获得特等奖和一等奖；在第三届中国大学生方程式汽车大赛中，广工大FSAE车队获得赛会唯一“新闻传播大奖”、“ANSYS视频传播奖”、营销报告第二名、赛车设计第八名和静态项目总分第十名的奖励；1名学生荣获第七届中国青少年科技创新奖等。学校篮球队连续三年荣获全国大超联赛总冠军，2011年荣获第八届亚洲大学篮球锦标赛冠军等；学校舞蹈节目获全国大学生艺术展演一等奖等。2015年，学校承办了第十四届“挑战杯”全国大学生课外学术科技作品竞赛，摘得两项特等奖、两项一等奖、两项二等奖，并以团体总分全国高校第二、广东高校第一的好成绩捧得“优胜杯”。
2017年，在第十五届“挑战杯”全国大学生课外学术科技作品竞赛中，摘得两项特等奖、一项二等奖、三项三等奖，并以特等奖总数全国第二、团体总分广东高校第一的好成绩捧得“优胜杯”。

== 体育强校 ==
广工在体育方面名声颇高，并且以篮球强校著称。广工建有篮球和足球的国家级高水平运动队；乒乓球，网球，田径，羽毛球的广东省级高水平运动队。在篮球方面，分别获得2004和2007年全国大学生运动会男子篮球冠军，2009年中国大学生篮球超级联赛冠军，2011年中国大学生篮球超级联赛冠军。2008年北京奥运中国篮球国家队成员就有三名广工校友，分别是[[易建联|易建联]]、[[杜锋|杜锋]]和[[朱芳雨|朱芳雨]]。另外，[[广东华南虎篮球俱乐部|广东华南虎篮球俱乐部]]（前宏远篮球队）也有不少球员来自广工。

== 南国飘香BBS(已停止运行) ==
南国飘香是广东工业大学校级BBS，是学校官方论坛，始建于1997年。飘香有着最新最全的广工校园生活信息，提供了校园电视、工大FTP及魔兽对战、反恐精英对战平台等校园服务，内有校园生活、新生专版、师生互动、乡情校谊、网友日记、不吐不快等特色版块，是广工人气最好的校内信息交流平台。
现在BBS已经停止运行。
