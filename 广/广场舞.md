'''广场舞'''，或称'''广场健身舞'''，是一種行進間[[有氧健身操|有氧健身操]]，是居民自發地以[[健身|健身]]為目的在廣場、院壩等開敞空間上進行的富有韻律的舞蹈，通常伴有高分贝、节奏感强的音乐伴奏，多为徒手健身，也有一小部分手持轻器械，近年被批评嚴重擾民。

广场舞在[[中国大陆|中国大陆]]无论南北皆十分普遍，参与者多为經歷過[[文革|文革]]世代的中老年人，因此也被视为是一种中国的社会现象。在中国，跳广场舞的中老年婦女经广泛宣传被冠以“广场舞大妈”的名号。而对于广场舞的确切研究，无论在社会学界还体育界目前还没有达成共识<ref name="广场健身舞研究现状综述与趋势展望">{{cite journal | title=广场健身舞研究现状综述与趋势展望 | author=牟顶红 | journal=科技风 |date=2010年12月}}</ref>。因为广场舞中的高分贝音乐会造成噪音滋扰，许多小区业主及居民反对在小区中进行广场舞；广场舞参与者也逐渐使用无线耳机以避免高音扰民<ref>[http://news.qq.com/a/20141122/006081.htm 郑州大妈戴耳机跳广场舞 网友：场景诡异(图)]</ref><ref>[http://www.qianhuaweb.com/2015/0910/2941266.shtml?edjn8 大妈戴耳机跳广场舞 小细节折射“爱”的大情怀] {{Wayback|url=http://www.qianhuaweb.com/2015/0910/2941266.shtml?edjn8 |date=20170111083157 }}</ref>。

== 起源 ==
[[File:Chinese_fitness_dancing_in_Yiheyuan.JPG|thumb]]的一種廣場舞]]

关于广场舞和类似活动的起源说法众多，有研究者甚至将其历史追溯至古代的祭祀活动。其中一种说法是，有文字记载的广场舞可以追溯至战国晚期，是古代统治阶级用来进行宗教祭祀的一种形式和手段，是原始生活和古代社会最为经常的智力活动之一。《管子·轻重己》曾记载“以春日至始，数九十二日，谓之夏至，而麦熟。天子祀于太宗，其盛以麦……”<ref>《[[管子|管子]]·轻重己》</ref><ref name="广场健身舞研究现状综述与趋势展望" />另一种说法是，广场舞起源自近代海外舞蹈的发展<ref>{{cite web | url = http://news.163.com/13/1010/02/9APSQOH700014Q4P.html | title=能阻挡广场舞的也许只剩下台风了 | publisher=青年时报 | accessdate=2013-10-11}}</ref>。还有人认为广场舞是文革时期[[忠字舞|忠字舞]]的延伸。<ref>{{cite web |title=网上流传的许多忠字舞录像和照片都妖魔化了 |url=http://www.caogen.com/blog/infor_detail.aspx?ID=40&articleId=67408 |accessdate=2018-09-10}}</ref><ref>{{cite web |title=从忠字舞到广场舞 |url=http://yuge.blog.caixin.com/archives/73227 |accessdate=2018-09-10}}</ref><ref>{{cite web |title=从忠字舞到广场舞，宿命式的舞者们 |url=http://gy.qq.com/original/guyu_report/report158.html}}</ref>而广场舞中最得到[[国家体育总局|国家体育总局]]推荐和推广的佳木斯快乐舞步则是根据[[秧歌舞|秧歌舞]]的灵感而编排的。

2015年3月23日，中华人民共和国[[国家体育总局|国家体育总局]]宣布，将在全国范围内推出由[[专家|专家]]创编、适合不同人群的12套广场健身操舞优秀作品，并对其进行推广和培训，其中包括《[[小苹果_(歌曲)|小苹果]]》和《[[最炫民族风|最炫民族风]]》两首脍炙人口的歌曲<ref>{{cite web | url = http://tech.gmw.cn/newspaper/2015-03/26/content_105447690.htm | title=12套广场舞推向全国| publisher=光明网 | accessdate=2015-03-26 }}</ref>。

== 参与群体 ==
[[File:Haikou_People's_Park_-_Chinese_fitness_dancing_-_01.jpg|thumb]]

有研究者以[[武汉市|武汉市]]为例，对中国大陆的广场舞活动进行了研究，认为参与广场舞者大部分年龄在45至65岁之间的中老年人，属于中年人和年轻的老年人群体，65岁以上的参与广场舞的老年人并不多。在广场舞的参与者以女性居多。同时，参与广场舞的中老年人中，有八成的参与者表示喜欢跳广场舞，而他们参与广场舞的动机又各不一样，以健身目的居多。中老年人参与广场舞具有练习次数多、持续时间长的特点。<ref name="武汉市主城区中老年人广场舞开展现状调查与分析研究">{{cite journal | title=武汉市主城区中老年人广场舞开展现状调查与分析研究 | author=代敏 | journal=华中师范大学硕士毕业论文 | year=2011}}</ref>

== 开展地点和条件 ==
[[File:People_in_Chinese_park_1.JPG|thumb]]
广场舞对场地设施要求不高，一般选择在社区广场、公园等空地上进行。这类活动由于基本无可能涉及向公众传达特定议题，事实上是少数无须获得管制机构“批准”的活动。广场舞的舞种多，参与费用低，对于一般参与者也没技巧要求，因此在中老年群体中很受欢迎。中老年人对健康的科学认知程度并不高，但对体育锻炼和广场舞的认知情况良好。<ref name="武汉市主城区中老年人广场舞开展现状调查与分析研究" />

== 健身效果 ==
根据问卷调查与统计，练习广场舞能有效帮助中老年人预防和减少疾病的发生率，在生理上减轻了练习者的身体痛苦，进而在经济上减轻了中老年人的医疗负担。同时，通过长期练习广场舞，许多中老年受访者表示在心理上也更加健康。<ref name="武汉市主城区中老年人广场舞开展现状调查与分析研究" />广场舞练习时多数情况下有指导员，但指导员专业水平不齐，无法对广场舞练习者的健康状况进行科学的分析与指导。<ref name="武汉市主城区中老年人广场舞开展现状调查与分析研究" />
==引发事件==
*2013年3月 南京雨花南路邓府山村小区，广场舞活动场地被泼粪。
*2013年4月 成都一小区住户向跳舞人群扔水弹。
*2013年8月 北京昌平区56岁的施某放出三只藏獒冲进跳舞人群，将跳舞大妈吓跑。
*2013年10月 武汉一小区内正在跳广场舞的人群也遭楼上业主泼粪。
*2013年11月 郑州郑汴路与未来路交叉口的一个小区，跳广场舞大妈遭到居民泼水警告。
*2014年3月 温州市中心新国光大厦业主委员会自发垫资26万元买来高音炮「还击」广场噪音。
*2014年4月 吉安市吉州区香榭丽都小区前面的街心花园，广场舞引发争议，街心花园被泼废机油、碎玻璃。<ref>{{cite web|title= 复旦发明广场舞“降噪神器”：声音只定向传播|url= http://news.xinhuanet.com/local/2014-04/28/c_1110433299.htm|publisher=[[新華網|新華網]]|date=2014-04-28|accessdate=2016-01-13}}</ref>
*2017年4月 澳門祐漢公園跳舞人群被附近居民以改裝氣槍掃射，有人受傷。

== 评价 ==
=== 正面 ===
有评论表示，广场舞风靡在中国大陆的风靡，改变了人们对[[汉族|汉族]]不能歌善舞的印象<ref>{{cite news|author=熊慕零|title=广场舞为什么这么火 |url = http://paper.people.com.cn/rmrbhwb/html/2014-11/28/content_1503998.htm|accessdate=2014-11-28 |newspaper=[[人民日报|人民日报]]海外版 |date=2014-11-28}}</ref>。还有报道称，2014年访问[[广州|广州]]的[[悉尼|悉尼]]市长有认为广场舞有助于激活城市公共空间，甚至考虑将之移植悉尼<ref>[http://www.guancha.cn/society/2014_05_26_232748.shtml 悉尼市长访问广州 称悉尼要引广场舞入社区]</ref>。

=== 负面争议 ===
广场舞主要存在噪声扰民的争议。广场舞的音乐设备声效[[分贝|分贝]]高，广场舞活动时间一长，对周围的住户和游客会造成噪声滋扰{{efn|1=有编友留言认为解决方法可以叫舞者用wifi耳塞接收音乐，认为花费不大比较合适。}}。而此类活动由于并无特别传播讯息的目的，亦非与城市风貌管制存在较大利益冲突，市政部门少有强力执法的诱因，使居民与广场舞参与者之间的矛盾愈演愈烈。2013年起“广场舞”、“广场舞大妈”等都成为网络搜索热门词汇。<ref>{{cite web |url=http://henan.sina.com.cn/news/s/2014-01-20/0735-122630.html |title=广场舞成市民噩梦 泼粪鸣枪放藏獒奇招频出 |accessdate=2014-02-17}}</ref>此外，大规模的广场舞被指可能加重社区不同群体间的矛盾，据媒体报道中国大陆各地的广场舞参与者因可能滋扰周边居民而且官方无作出必要措施，受困扰的居民多会以泼粪、藏獒饲养者放犬袭击、鸣枪恐吓<ref>{{cite web |url=http://www.gywb.cn/content/2014-01/22/content_403607_all.htm | title=广场舞影响社会稳定 中国大妈多次遭泼粪藏獒鸣枪恐吓 | accessdate=2014-02-18}}</ref>、拉带刺铁丝网<ref>[http://news.sina.com.cn/s/2014-05-26/035930229927.shtml 物业拉带刺铁丝网阻广场舞 大妈剪断铁丝重新跳]</ref>等以示不满。
2013年，[[武汉市|武汉市]]、[[福州市|福州市]]、[[长沙市|长沙市]]等地相继出现向广场舞场地被泼粪的事件。<ref>{{cite web |url=http://www.chinanews.com/sh/2013/12-03/5574363.shtml | title=经常在福州三县洲大桥下空地跳广场舞的老人们发现 |accessdate=2014-02-18}}</ref><ref>{{cite web |url=http://news.ifeng.com/society/2/detail_2013_11/12/31163013_0.shtml |title=长沙一小区广场舞大妈频遭业主反对：曾被泼粪3次 |accessdate=2014-02-18}}</ref>

2013年有华人大妈在[[纽约市|纽约市]][[日落公园|日落公园]]跳广场舞，因周围居民投诉其制造噪音，被警察拘捕。<ref name="中国“广场舞大妈”在纽约因“扰民”两次被铐">{{cite web | url=http://www.chinanews.com/fz/2013/08-09/5142830.shtml |title=中国“广场舞大妈”在纽约因“扰民”两次被铐 |accessdate=2014-03-03}}</ref><ref name="华裔新移民不断融入 广场舞扰民越演愈烈">{{cite web |url=http://topic.uschinapress.com/damabk/ |title=华裔新移民不断融入 广场舞扰民越演愈烈 |accessdate=2014-03-03}}</ref>2014年在[[莫斯科|莫斯科]][[红场|红场]]也被警察制止。<ref>[http://www.guancha.cn/europe/2014_06_16_238027.shtml 中国大妈“远征”莫斯科红场跳广场舞 引来警察]</ref>

目前有[[昆明市|昆明市]]公安局就广场舞可能扰民问题表示，如警员执法时屡次对对方警告无效，警方可以依据《[[:s:中华人民共和国治安管理处罚法|中华人民共和国治安管理处罚法]]》第五十八条“违反关于社会生活噪音污染防治的法律规定、制造噪声干扰他人正常生活的、处警告，警告后不改正的、处二百元以上五百元以下罚款进行处罚。”条文对跳广场舞扰民者采取行动。<ref>{{cite web |url = http://news.ifeng.com/society/1/detail_2014_02/15/33835570_0.shtml |title = 昆明：跳广场舞扰民最高可罚500元 |accessdate=2014-02-17|language=zh-hans}}</ref>

而在2018年4月，[[澳門|澳門]][[祐漢公園|祐漢公園]]有廣場舞參與者被附近居民以氣槍打傷，此事亦反映了廣場舞在港澳地區構成擾民的情況。

广场舞也被某些人蔑称为[[绝经|绝经]]舞<ref>{{cite web |author = 赵妍 |title=广场舞，一场集体主义怀旧 |url = http://www.21ccom.net/articles/zgyj/gqmq/article_2013120496470.html |website=[[共识网|共识网]] |publisher=时代周报 |accessdate=2015-06-01|language=zh-hans|date=2013-12-04|archiveurl=https://web.archive.org/web/20140213171453/http://www.21ccom.net/articles/zgyj/gqmq/article_2013120496470.html|archivedate=2014-02-13|deadurl=yes}}</ref><ref>{{cite news |author =陈翔 |title = 咋办？你的骨骼 我的耳朵 |url = http://gzdaily.dayoo.com/html/2014-06/30/content_2674537.htm |accessdate=2015-06-01 |newspaper = 《[[广州日报|广州日报]]》|publisher=大洋新闻 |date=2014-06-30 |location=广州 |page=A14 |language=zh-hans|archiveurl=https://web.archive.org/web/20140630074120/http://gzdaily.dayoo.com/html/2014-06/30/content_2674537.htm|archivedate=2014-06-30}}</ref>，品味低俗。
