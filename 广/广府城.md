'''广府城'''，又称'''永年城'''，是[[中华人民共和国|中国]][[河北省|河北省]][[永年区|永年区]]原政府所在地，也是原广平府府治、永年县县治所在地。城址位于[[永年区|永年区]]东南60公里处的[[永年洼|永年洼]]中，永年洼面积28平方公里，而古县城坐落于延伸至湖中的半岛上。

== 历史 ==
[[File:Guangfucity.jpg|thumb]]
广府城城址始建于何年，史不可考。城墙初为土城，古为曲梁县，属[[冀州|冀州]]域，见于《[[左氏传|左氏传]]》记载。[[唐高祖|唐高祖]]武德二年（公元619年），河北军阀[[窦建德|窦建德]]攻陷洺州后，即建都于此。原来城池狭小，周长仅六里二百四十步。[[元朝|元朝]]时，郡守王伟扩周长至九里十三步，即4.5公里。[[明朝|明朝]][[嘉靖|嘉靖]]二十一年（公元1542年），[[广平府|广平府]]知府陈俎调集周遍十三县民工，历时十三年将土城改为砖式建筑。城墙高12米，宽（厚）8米，嘉靖四十三年（公元1564年）广平府知府崔大德在原城址的基础上又在外围建起大堤及瓮城，使得城池免受水患侵扰。原四座城门，后改为8座，设重门楼。民国三十六年（公元1947年），永年县被[[中国人民解放军|中国人民解放军]]攻克，为了交通方便，将南北瓮门拆毁，仅留东西瓮二门。城外围有长达5公里的护城河，深3.5米，宽百米。

== 地理位置 ==
广府城古为冀中南北交通要道，北近[[邢台|邢台]]，南接[[邯郸|邯郸]]，早期建有[[临洺关|临洺关]]，是河北交通咽喉。


== 城址现状 ==
广府城原建有城楼四座，角楼四座，铺舍二十六座，垛口一千七百五十二处，现已全毁。南城墙西段发生坍塌3处，仅外墙砖保存良好，内墙砖均无。北城墙和东城墙顶面保存良好，西城墙和南城墙顶面毁坏严重，表层单砖残缺不全。西城墙最南端上半部坍塌，东城墙北段有两处坍塌很严重。

== 城局布置 ==
城池坐东南而面西北，四座城门分于四角，城池形状颇似牛状，故又称卧牛城。
*'''街道''':城内有街道三十多处，有四大街、八小街、七十二道小拐弯之称。
[[File:Yang_Luchan's_former_residence_1.jpg|260px]]外的[[杨露禅故居|杨露禅故居]]。]]
*'''政府类''':在东西两城门处尚保存护城碑2块。东门碑文为：“府县告示、严禁刨挖城根、垦种庄稼、盗伐树株、偷取鱼藕，违者重究”。西门碑文为:“大明嘉靖二十一年知府陈俎修砌此城”。
*'''建筑类''':[[弘济桥|弘济桥]]、[[毛遂墓|毛遂墓]]、[[黑龙潭|黑龙潭]]、[[紫山书院|紫山书院]]、[[文庙大殿|文庙大殿]]、[[杨露禅、武禹襄故居|杨露禅、武禹襄故居]]等，城内尚保存大量明清故居。
*'''文人骚客类''':唐代[[李白|李白]]、晚唐[[司空曙|司空曙]]、元代[[王磐|王磐]]、清代[[郑板桥|郑板桥]]等都曾在此驻留，有大量诗篇传世。清代[[直隶总督|直隶总督]]方观承有诗云:“稻引千畦苇岸通，行来襟袖满荷风。曲梁城下香如海，初日楼边水近东。 拟放扁舟尘影外，便安一榻露光中。 帷堂患气全消处，清兴鸥鱼得暂同。”

== 太极拳之乡 ==
广府城[[太极拳|太极拳]]发源于清朝道光年间。主要代表人物有[[杨露禅|杨露禅]]、[[武禹襄|武禹襄]]等。由于永年人广习太极拳来锻炼身体，而众多太极拳名家又出自永年，所以被称为太极之乡。

== 战役 ==
城池处在[[永年洼|水洼]]之中，易守难攻，最著名战役为1945年[[中国共产党|中国共产党]]解放军攻打城池时，当地土匪[[许铁英|许铁英]]在[[国民政府|国民政府]]支持下，守城两年，引起美国军事调停人员高度重视，并亲临视察，一时轰动民国。
