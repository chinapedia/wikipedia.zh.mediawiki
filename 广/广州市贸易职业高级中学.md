'''广州市贸易职业高级中学'''简称'''贸易职中'''，位于[[中华人民共和国|中国]][[广州市|广州市]][[越秀区|越秀区]][[中山路_(广州)|中山六路]]瑞兴新街7号，前身是于1980年创立的广州市财经职业高级中学（市第八十五中学）及1989年创立广州市贸易职业高级中学（市第三十八中学）合并而成。学校现为广东省重点中等职业学校、广州市现代教育技术实验学校，有在校学生约2500人。

== 历史 ==
* 1980年，根据广州市教育局中学结构调整规划，'''广州市第八十五中学'''高中部改办为'''广州市财经职业中学'''。
* 1992年3月，广州市财经职业中学被广州市教育局确认为“广州市重点中等职业学校”。
* 2002年，'''广州市第三十七中学'''职中部与'''广州市服装职中'''（即原广州市第三十八中学）等学校合并成立新的'''广州市贸易职业高级中学'''。
* 2005年，广州市财经职业学校被广州市教育局调整为完全职业高级中学，并改名为'''广州市财经职业高级中学'''，初中部（广州市第八十五中学）停办。
* 2010年，经越秀区教育局调整，广州市财经职业高级中学（原市八十五中）与广州市贸易职业高级中学（原市三十八中）合并。保留“广州市贸易职业高级中学”校名。原财经职中校区开设金融事务、会计专业，原贸易职中校区开设服装设计、美术设计、商务英语、国际商务专业。
* 2012年学校通过广东省教育专家组的评估，正式被确认为“广东省重点中等职业学校”，同时文德校区调整招生方向，只招收高职方向学生。

== 师资 ==
广州市贸易职业高级中学师资，高学历（研究生或具硕士学位）教师和中高级（包括特级）以上职称教师超过全部教师总数80%。学校还是全国商业职业高中研究会副会长学校和常务理事学校、全国职教服装纺织指导委员会理事学校、省服装专业教学委员会副主任学校和全国教育科学“十五”规划重点课题“创新学习与实验研究”优秀实验学校。学校曾先后获得广州市教育系统先进单位、广州市职业高中毕业班工作评估一、二等奖及广州市中学生合唱节比赛一、二等奖等多项办学荣誉，学生在国家、省、市各级各类中职学生技能大赛中屡获团体和个人一、二等奖，学校办学成绩更多次被报刊、电台及电视台报道。

== 教学理念 ==
学校以“做人为上，能力为本”，“人人有才，人人成才”为育人理念，在培养学生 “就业有专长，升大有优势，创业有本领，发展有基础”综合能力方面积累了大量经验。特别是结合该校优势，探索实行中职、电大大专、本科“3+2+2”一条龙教育服务模式，学生中职第三年经申请续读电大大专课程，从读中职第一年算起最快5年获大专文凭，6年获本科文凭，为学生提供一条成本低、时间短的捷径。

== 开设的专业 ==
学校现开设有会计、金融事务、商务英语、美术设计与制作、服装设计与制作、物流服务与管理、国际商务和电子商务8大专业，其中，会计专业是省重点专业，金融事务、服装设计与制作专业是广州市重点专业。
