《'''聖界的奇蹟'''》（{{lang|ja|フラン2 〜聖界の奇跡〜}}）是由EXE-Create發售的PC單機遊戲，是Frane系列的第二作。繁體中文版由台灣光譜代理。

== 故事簡介 ==
發生於一代的20年後的故事。主角莉拉是一代的主角的女兒。由於天界的三神器被人奪去，天神下界尋求主角的父親—塞迪幫忙。其後莉拉也被給予試練，到各地旅行，同時也卷入人類與魔物的戰事裡。

== 登場角色 ==
===主角===
;{{nihongo|莉拉. 席林|リラ | CV：吉田 增美 }}
:人類塞迪與天使芙蘭所生的混血兒。富有正義感，在塞迪和芙蘭因幫助露比而離開家後開始冒險，慢慢開始成長。

===家人朋友===
;{{nihongo|塞迪|セディ| CV：土橋 信介}}
:莉拉的父親，在20年前成為王國英雄。與芙蘭一起對抗庫魯傑洛，後來一同失去音信。
;{{nihongo|芙蘭|フラン| CV：山下 真美}}
:莉拉的母親，原為天界的天使。與塞迪一起對抗庫魯傑洛，後來一同失去音信。被庫魯傑洛以聖杯吸去力量來創造新魔族，其後被奈拉救出。
;{{nihongo|露比|ルピー| CV：與島 ぬぐみ}}
:天神的化身。因失去三神器—神劍、神杖與聖杯而尋求塞迪幫忙。不能干涉人間的事情。
;{{nihongo|奈拉|ナイラ| CV：中 香識}}
:莉拉的老師，教導莉拉劍術、魔法和禮儀。其後因某原因而幫忙庫魯傑洛。
;{{nihongo|莫古|ポルン| CV：岡橋 弓子中 香識}}

===旅途中認識的人們===
;{{nihongo|瑪魯卡|マルカ | CV：中野 絵梨香 }}
:瑪魯卡-魔物少女。在皮柯特村被村民當成是偷銳金的小偷而被捉走。
:雷伊的青梅竹馬。曾試過幫過人類小孩，但被小孩的母親厭棄，趕跑瑪魯卡。
;{{nihongo|雷伊|レイ| CV：上野 賢一 }}
:人類少年。小時被父母遺棄，由魔物養大。在期望與人類和平共存的瑪魯卡的父親死亡後成為該族首領。
;{{nihongo|露比亞|ルピア | CV：吉岡 小百合 }}
:拉帕斯村的村民。
;{{nihongo|霸恩|パーン | CV：大森 素生}}
:魔物之王，統治全部魔物。
;{{nihongo|路希亞|リュシアン| CV：藤原 光 }}
:哈利亞王國第十三任國王。因莉拉的父親是英雄，而希望與莉拉達成政治婚姻。為討伐魔物而派軍隊到各地，認為人類與怪物只有一方能生存。
;{{nihongo|拓|リンギット| CV：中川陽介 }}
:哈利亞王國的一等騎士，被路希亞所信任。劍術高超，認為解決戰爭不是其中一方獲勝，而是共存。
;{{nihongo|阿汀|アデン | CV：赤坂 将太 }}
:瘋人塔的囚犯，前哈利亞王國二等騎士，已困在塔內20年。希望和魔物和平共存，建做一個理想王國。其中一隻魔物朋友名叫波可林。
===敵方===
;{{nihongo|庫魯傑洛|クルゼーロ | CV：国清 稔}}
:惡魔族。持有聖杯。
;{{nihongo|庫洛內|クローネ| CV：樫根 留美子 }}
:惡魔族女性，為庫魯傑洛效力。持有「奪」的神器—迪凡因華斯，失去了神器就不能殺害人類。
===其他===
;雷亞路
:拉尼安村的村長，於20年前建立了人類和怪物和平共存的拉尼安村。
;{{nihongo|吉魯達|ギルダー | CV：赤坂 将太}}
:吉魯達 王城的丞相。設計陷阱令莉拉進入瘋人塔。

==遊戲系統==
; 合成系統
: 分為料理和工作，除了可送給同伴外，部份能用作回復HP、MP、以及裝備
; 戰鬥系統
: 裝備不同武器有不同的攻擊模式。劍能用劍技，杖能用魔法。
