'''聖安娜餅屋'''是[[香港|香港]]一家售賣[[麵包|麵包]]及[[西餅|西餅]]的[[連鎖店|連鎖店]]，成立於1972年，1991年在[[香港聯合交易所|香港聯合交易所]]上市（前港交所上市編號：0192），香港店舖共90間、[[廣州|廣州]]38間、[[深圳|深圳]]2間、[[澳門|澳門]]8間、合共138間。

== 歷史==
聖安娜餅屋在1972年於[[香港島|香港島]][[跑馬地|跑馬地]]開設其第一間分店。截至2012年，聖安娜餅屋共有90間分店。1991年，聖安娜餅屋被[[香港飲食管理有限公司|香港飲食管理有限公司]]（{{hkex2|0668}}）收購，成為其附屬機構。1992年，聖安娜餅屋於[[澳門|澳門]]開設首間分店，同年開始設立其月餅品牌，推銷至香港、[[北美洲|北美洲]]及[[中國大陸|中國大陸]]的市場。1993年，聖安娜餅屋於[[九龍|九龍]][[九龍灣|九龍灣]]開設生產工場，1995年則亦於[[中國|中國]][[廣東省|廣東省]][[深圳市|深圳市]]設立生產工場。到了1999年，聖安娜餅屋開設首間中式食肆[[小城知味|小城知味]]，售賣中式小食。

2007年1月，[[利豐|利豐]]旗下的[[利亞零售|利亞零售]]（{{hkex2|8052}}）全面收購聖安娜餅屋股份，成功將其[[私有化|私有化]]。

== 產品 ==
聖安娜餅屋售賣多種[[麵包|麵包]]，當中不少為日式麵包，此外也售賣[[蛋撻|蛋撻]]、[[蛋糕|蛋糕]]和[[西餅|西餅]]。其牛油蛋撻為[[橢圓形|橢圓形]]設計，成為該公司的特色產品。聖安娜餅屋也推出各種節日產品，如賀年禮盒、[[粽|粽]]及[[月餅|月餅]]等。

=== 品质 ===
2014年，有不少香港市民投訴連鎖餅店的麵包縮水，有媒體記者分別在不同地方的六間連鎖麵包店及一間麵包小店，隨機購買餡包调查，其中将不同五間店買到的兩款蜜汁[[叉燒包|叉燒包]]及[[吞拿魚包|吞拿魚包]]切開作比拼，量度由麵包內側頂部至餡料面的高度，結果發現聖安娜兩款麵包的空心程度最嚴重，當中蜜汁叉燒包空心位置高度長達3厘米，而且在五個同款麵包中重量最輕；聖安娜另一款吞拿粟米蘑菇包，空心高度亦達2厘米。<ref>{{cite web|title=6大連鎖麵包店逾半產品餡料不足 聖安娜空心包窿大過乒乓波 |url=http://hk.apple.nextmedia.com/news/art/20140520/18726168 |publisher=蘋果日報 |date=2014-05-20 |accessdate=2014-05-20}}</ref>

== 謠傳結業 ==
1997年11月24日下午，聖安娜餅店被謠傳員工收到解僱信，當日不用上班。結果結業的消息傳到香港不同角落，不少市民吸取翡翠印花的教訓，紛紛到聖安娜分店排隊，將餅卡全部拿來換餅。結果造成多間分店出現混亂場面，旺角餅店一名職員分餅時更體力不支暈倒，需送院處理。[[八佰伴|八佰伴]]國際飲食有限公司發聲明表示改名為「[[香港飲食管理有限公司|香港飲食管理有限公司]]」，而市面傳出聖安娜餅店倒閉的消息實屬謠言。
