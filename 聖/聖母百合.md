'''聖母百合'''（[[学名|学名]]：''{{lang|la|Lilium candidum}}''）是[[百合科|百合科]][[百合屬|百合屬]][[多年生|多年生]][[草本植物|草本植物]]，又名'''白百合'''、'''瑪丹娜百合'''（Madonna Lily）。原產於[[巴爾幹半島|巴爾幹半島]]與[[西亞|西亞]]。[[種小名|種小名]]''candidum''是「純白色」的意思<ref>Helmut Genaust: ''Etymologisches Wörterbuch der botanischen Pflanzennamen'', S. 122, 2005, ISBN 978-3-937872-16-2</ref>，形容本種花瓣的顏色是純白色的。純白色的花被視為是純潔的象徵，中世紀時期的基督徒常拿這種百合花來供奉[[聖母瑪利亞|聖母瑪利亞]]，因此被稱為聖母百合、瑪丹娜百合。

== 形態特徵 ==
[[File:Lil_candidum_01EB_Griechenland_Tembeschlucht_31_05_00_clean_edit.jpg|thumb]]
株高50至130[[公分|公分]]，地下莖為[[鱗莖|鱗莖]]，鱗莖直徑可達九公分寬，鱗片為寬卵形。[[莖|莖]]表面光滑，莖粗約1.4公分。[[葉|葉]]在莖上呈螺旋狀排列，葉面光亮無毛，葉形為披針形，葉長八公分。[[花序|花序]]為總狀花序，一枝花莖著花2至12朵，有時可達20朵，[[花|花]]純白色，漏斗形，芳香。花梗綠色，長四公分。花被六片，花被片長55至65公分，寬6至13公分，花盛開時花被片會在先端向後反捲。五月至六月開花，花莖長1.2[[公尺|公尺]]，最高可以長至2公尺。

百合屬植物在冬天時地上部通常會全部枯萎，只留下生長在地底的鱗莖來越冬。聖母百合與其他百合不同，冬季時雖然地上部也會枯萎，不過會由鱗莖長出一些簇生狀的基生葉，葉長約22公分，以這些基生葉來越冬。

== 分布 ==
聖母百合分布於[[地中海|地中海]]東部地區，由[[希臘|希臘]]（[[馬其頓|馬其頓]]）至[[以色列|以色列]]（[[加利利|加利利]]）、[[敘利亞|敘利亞]]與[[克里特島|克里特島]]。生長在石灰質的土壤中，生長地夏季極為乾燥。

== 栽培 ==
聖母百合的種植已經有很長的一段歷史，百合毒素病及真菌性病害[[灰黴病|灰黴病]]是栽培時較常見的病蟲害，要避免[[病毒#植物|病毒]]的感染，一個可能的辦法就是使用[[種子|種子]]播種來繁殖。

== 象徵 ==
[[File:Flag_of_Quebec.svg|200px]]
聖母百合曾經是[[加拿大|加拿大]][[魁北克省|魁北克省]]的省花，在[[魁北克旗幟|省旗]]上就畫有這種白色的[[百合花飾|百合花飾]]。不過聖母百合並不是魁北克的[[本土植物|原生植物]]，再加上這種百合在魁北克生長的並不是很好，魁北克省的省花現在已經改成[[鳶尾科|鳶尾科]]的[[變色鳶尾|變色鳶尾]]，這種鳶尾是魁北克的原生種。<ref>[http://www.gouv.qc.ca/portail/quebec/pgs/commun/portrait/drapeau/?lang=en National Flag and Emblems]</ref>

== 文化與信仰 ==
[[File:Botticelli,_Maria_mit_dem_Kind_und_Singenden_Engeln,_1477._Gemaldegalerie.jpg|right]]1477年的作品，後面的白色花朵即是聖母百合。]]
約在公元前2000年左右，克里特島上的[[米諾斯文明|米諾斯文明]]遺址，[[克諾索斯|克諾索斯]]米諾恩皇宮（Minoan Palace）的壁畫上就繪有聖母百合。

聖母百合純白色的花朵被認為是純潔的象徵，歐洲歷史上不同時期的畫家在畫聖母瑪利亞的畫像時，通常都會同時畫上這種百合花，且在畫這種百合花時大多不會畫出花的[[雄蕊|雄蕊]]與[[雌蕊|雌蕊]]。
