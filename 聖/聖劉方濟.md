'''聖劉方濟'''（{{lang-es|Francisco Fernández de Capillas}}, [[道明會|O.P.]]；{{bd|1607年|8月15日|1648年|1月15日}}）。西班牙语名为“'''方济各·嘉彼来'''”，生在[[西班牙帝國|西班牙帝國]][[帕倫西亞|帕倫西亞]][[巴克林德坎波斯|巴克林德坎波斯]]，是首位[[中華殉道聖人|在中國殉道]]的歐洲[[傳教士|傳教士]]。

== 生平 ==
17歲入[[道明會|道明會]]。當他是執事時被送到菲律賓做傳教工作。於1631年二月間到達馬尼拉。過了不久晉升神父。在菲律賓十年的福傳洗鍊，是為到中國傳教工作的準備。

1641年他和會友Francisco Diez神父獲准去台灣。不久後被荷蘭人驅逐。他於1642年三月到達福建省，和早期教難生還下的道明會友會和。聖方濟兩年多的熱心事主。成立了道明三會。

1644年11月4日他的會友神父，Francisco Diez生病過世。同天清軍攻入福安城，消滅了明朝政權。滿洲人敵視傳教士，開始對他們的迫害。1647年11月13日，聖方濟為教友病人行傅油聖事，返回途中被捕入最壞的監獄，受到嚴刑拷打。聖方濟忍耐痛苦，平靜如常，驚動了法官和獄卒。後來他被送入死牢，獄卒欽佩他，甚至秘密送食物給他。在獄中聖方濟寫道，“我在這裏和其他犯人建立了友誼。他們問我關於福音和天主的問題。我不在乎出獄，我知道我是在行天主的旨意。我不可以在夜間不去睡覺而在祈禱。我就躺在床上祈禱直到天亮。我在這裏有很多的喜樂，無所牽掛。因為我知道這是為耶穌基督。在這裏的日子中，我找到了不容易找到的珍珠。”

1648年一月十五日聖方濟被判死刑，罪名為傳播邪教，煽動民心背叛新皇，同天被斬首殉道。<ref>{{cite web |url = http://saints.sqpn.com/saintf0u.htm |title = Saint Francis Ferdinand de Capillas |publisher=Patron Saints Index |accessdate=2008-04-24 }}</ref>

== 封圣 ==
{{Details|中華殉道聖人}}

聖方濟在五月二日1909年被教宗聖碧岳十世升為真福和十月一日2000年被教宗聖保祿二世升為聖人。梵蒂岡認可他是第一位[[中華殉道聖人|中華殉道聖人]]。他的[[瞻禮|瞻禮]]是一月十五日。
