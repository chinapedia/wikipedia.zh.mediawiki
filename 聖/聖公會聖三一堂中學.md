'''聖公會聖三一堂中學'''（{{lang-en|SKH Holy Trinity Church Secondary School}}；簡稱「聖三一」或「HTCSS」）為香港政府[[津貼中學|津貼]]男女[[香港中文授課中學|中文中學]]，由[[香港聖公會|香港聖公會]]於1978年創辦。學校的辦學宗旨是藉著基督教的辦學理念，教導學生建立正確的人生觀，使學生能自立自愛，愛主愛人。同時也幫助學生自我增值，終身學習；同時培養自律精神，建立嚴謹純樸的校風。<br />
現任校長陳倩君女士曾任港島中西區傳統名校[[聖嘉勒女書院|聖嘉勒女書院]]副校長，引入多項計劃訓練學生的英語能力，例如每月舉辦大型嘉年華，每次皆設有不同主題，寓學習於娛樂，讓學生感受學英文的樂趣。<ref>{{cite web|url=http://www1.hkej.com/features/article?q=%23%E6%A0%A1%E9%95%B7%E7%B3%BB%E5%88%97%23&suid=1163882863|title=校長訪談：聖公會聖三一堂中學校長 轉工獲前學生鼓勵|website=http://www1.hkej.com|date=2015-07-06|author=鄭志珩}}}}</ref>更把學校變身成為機場，讓學生學習相關的英語表達，提升英語能力，同時展開生涯規劃。<ref>{{cite web|url=http://paper.wenweipo.com/2016/06/30/ED1606300005.htm|title=中學變「機場」  學英文「識飛」 - 香港文匯報|website=paper.wenweipo.com}}</ref>另外，該校亦正推行[[STEM教育|STEM教育]]，讓學生在科學、科技及數學等範疇建立穩固的知識基礎。<ref>{{cite web|url=http://paper.wenweipo.com/2017/03/03/ED1703030008.htm|title=【校園放大鏡】禮堂變駕駛艙  聖三一堂學生做機師 - 香港文匯報|website=paper.wenweipo.com}}</ref><br />
陳校長擔任不同的政府公職，如曾任課程發展議會[[香港考試及評核局|香港考試及評核局]]英語文學委員會主席、現任環境局可持續發展委員會成員<ref>{{cite web|url=http://www.info.gov.hk/gia/general/201702/28/P2017022800325.htm|title=政府委任新一屆可持續發展委員會成員|website=www.info.gov.hk}}</ref>[[行政長官卓越教學獎|行政長官卓越教學獎]]督導委員會評審工作小組組員<ref>{{cite web |url=http://www.ate.gov.hk/tchinese/organisation.html |title=存档副本 |accessdate=2015-07-06 |deadurl=yes |archiveurl=https://web.archive.org/web/20150706182325/http://www.ate.gov.hk/tchinese/organisation.html |archivedate=2015-07-06 }}</ref> 、[[語文教育及研究常務委員會|語文教育及研究常務委員會]]（語常會）委員<ref>{{cite web|url=http://www.language-education.com/chi/scolar_member.asp|title=語文教育及研究常務委員會|website=www.language-education.com}}</ref>、中學高級行政人員教育行政及管理課程導師等。

該校一共有8位中層管理人員（包括副校長、中文科、通識科科主任等）獲得由[[教育局|教育局]]及[[香港浸會大學|香港浸會大學]]教育學系主辦的中層人員領導培訓課程之「傑出學員獎」。<ref>{{cite web |url=http://applications.chsc.hk/ssp2015/sch_detail8.php?lang_id=2&sch_id=294 |title=存档副本 |accessdate=2015年12月11日 |deadurl=yes |archiveurl=https://web.archive.org/web/20151222104405/http://applications.chsc.hk/ssp2015/sch_detail8.php?lang_id=2&sch_id=294 |archivedate=2015年12月22日 |df= }} </ref>該校中文科教師多次接受傳媒訪問，公開教授學生有關DSE的考試技巧。<ref>https://life.mingpao.com/chi/article?issue=20170217&nodeid=1508735303602</ref><ref>https://jump.mingpao.com/resources/detail/s00012/1551843394272/daily-tips-%E8%80%83%E8%A9%95%E5%B1%80%E5%8F%A3%E8%A9%A6%E6%94%BB%E7%95%A5-%E5%8B%BF%E7%A9%BA%E8%82%9A%E4%B8%8A%E9%99%A3-%E6%95%99%E5%B8%AB-%E8%AB%96%E9%BB%9E%E5%AE%9C%E5%AF%AC%E5%BB%A3-%E5%8B%BF%E5%8F%AA%E9%A1%A7%E6%94%BB%E7%A0%B4%E5%88%A5%E4%BA%BA</ref><ref>https://skypost.ulifestyle.com.hk/article/1763170/closest</ref>

==學校歷史==
1968年，聖公會聖三一堂（現為聖公會[[聖三一座堂|聖三一座堂]]）有見當時學生求學艱難，決定辦學。政府撥出[[十二號山|十二號山]][[愛民邨|愛民邨]][[孝民街|孝民街]]地段，於1978年奠基施工。距離建校至今已經超過35年，在眾師生的共同努力下，構建成一個「以主為基、以人為本」的關愛校園。<ref>{{cite web|url=http://www.skhhtcss.edu.hk/untitled-ceif|title=建校史略|website=聖公會聖三一堂中學}}</ref>

==公開考試成績==
學校在歷年的香港高級程度會考中，絕大部分科目的合格率皆高於全港日校平均；中國語文及文化科、中國文學科、歷史科和數學統計科的優良率更高於全港平均。
在香港中學文憑考試中，學生在中文科、英文科、數學科、通識科、數學延伸單元一和生物科取得5*級成績，而數學延伸單元一，學生取得5級或以上的比率遠較全港平均為高。

==課外活動==
學校一共有三十多種課外活動。學校的女子手球隊、田徑隊和男子足球隊在本港學界廣為人知。
*女子手球隊：為學界勁旅，曾連續六屆取得學界冠軍、分齡賽冠軍，隊員多次獲選為港隊青年軍遠赴歐洲的捷克及南韓集訓及比賽。
*男子田徑隊：在學界田徑賽中勇奪團體總冠軍，學生亦在各項田徑比賽中勇奪三甲位置。
*男子越野隊：全港學生男子乙組、丙組冠軍。
*男子足球隊：創造了最短時間由學界第三組別升上第一組別之佳績，每年在各項比賽獲獎無數。
*男子籃球隊：北區三人籃球賽（青少年組）冠軍。
*音樂方面：學校合唱團曾在香港學校音樂節中級組中文歌曲混聲合唱勇奪冠軍。多位學生在第六十四屆香港學校音樂節中取得優良獎狀、在第六十五屆香港學校音樂節中，學生取得1項冠軍、2項亞軍及1項季軍。
*服務團隊方面：學校有童軍、女童軍、義工組、公益少年團、紅十字會青年團及圖書館管理員。
*本校讓學生擔任學生會、學生長、四社以及學生輔導員之領袖職位，以培養學生的領袖才能。
*其他活動：棋藝組、觀鳥組、創藝學會、承文學會（中國文學）等。

==宗教教育==
學校宗教教育處藉著不同的活動，例如崇拜、早會祈禱和分享、學生團契聚會、團契宿營、福音月活動以及佈道會等），協助學生建立正確的人生觀，使學生能自立自愛、愛主愛人。

==學生成長支援==
學校推行各項成長支援活動，具體措施包括：
*學生自評計劃、學長支援計劃、領袖訓練計劃、義工組、國內外體驗活動、性教育話劇表演及講座、環保節能活動、到台灣考察環保實施情況等。
*學校舉辦「學習紀律教育營」、以及推行「五星級的班」班級經營計劃，以增進同學情誼、訂立人生目標、提高學習動機、明白對父母、對學習要懷有感恩之心。
