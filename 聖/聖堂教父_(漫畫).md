《'''聖堂教父'''》（原書名：サンクチュアリ／Sanctuary），是一部由[[武論尊|史村翔]]原作，[[池上遼一|池上遼一]]作畫的[[青年漫畫|青年漫畫]]作品，於1990年至1995年之間，在[[小學館|小學館]]所發行的漫畫雙月刊[[Big_Comic_Superior|Big Comic Superior]]上連載，單行本全12集。系列漫畫發行量達到700萬本以上<ref>https://booklive.jp/feature/index/id/sanc161028</ref>。

台灣最初由[[時報出版|時報出版]]取得授權出版，在版權過期後，[[東立出版社|東立出版社]]的重新取得授權版本將書名改為《'''聖堂風雲'''》，香港玉皇朝版則譯為《'''-{英雄本色}-'''》。

1995年曾拍成真人電影版，由[[藤由紀夫|藤由紀夫]]執導，[[永澤俊矢|永澤俊矢]]、[[阿部寬|阿部寬]]、[[中村梓|中村梓]]等主演。另外有改編為[[OVA|OVA]]。

2003年[[台灣|台灣]]作家[[侯文詠|侯文詠]]將本書作為[[危險心靈|危險心靈]]一書的串場著作。

==故事概要==
自[[柬埔寨|柬埔寨]][[紅色高棉|紅色高棉]]的人間地獄存活下來的兩名[[日本|日本]]小孩，北条彰與淺見千秋，在回到了日本後，發現日本社會缺乏希望。因此兩人決定從黑白兩道來徹底革新已經腐敗的日本社會。一場[[剪刀石頭布|剪刀石頭布]]，決定了兩人的發展——北条彰進入了黑社會，而淺見千秋走上了從政之路。

== 登場人物 ==
=== 主要角色 ===
; {{nihongo|北条 彰|ほうじょう あきら|Hojo Akira}}
* [[聲優|聲]]：[[速水獎|速水獎]]／電影版：[[永澤俊矢|永澤俊矢]]
: 兩位主角之一，自黑道這條路線來改變日本政界。黑道組織「北彰會」首領。
: 從地獄回來的男人。和淺見一起逃出柬埔寨而成為生死之交。在高中獲得優異成績，但為了成為黑社會而在高二的時候退學，進入了黑道組織「渡海興業」。之後殺害了相樂聯合的第一代總長相樂，自己成為了第二代總長。目標是令相樂聯合稱霸日本，重新建造黑暗世界。

; {{nihongo|淺見 千秋|あさみ ちあき|Asami Chiaki}}
* 聲：[[中田和宏|中田和宏]]／電影版：[[阿部寬|阿部寬]]
: 兩位主角之一，利用從政來改變日本政界。民自黨[[眾議員|眾議員]]・佐倉秀一的秘書。
: 從地獄回來的男人。和北條一起逃出柬埔寨而成為生死之交。高中畢業後入讀[[東京大學|東京大學]]法學部。大學畢業後進入現職，之後被選為[[國會議員|國會議員]]。後來與仙石、吉川、狩谷等人組織了名為「新日本」的[[政黨|政黨]]。為奪取日本政權而奔走。

=== 黑道相關人物 ===

====相樂聯合====
; {{nihongo|渡海|とかい|Tokai}}
* 聲：[[土師孝也|土師孝也]]
: 黑道組織「渡海興業」的組長。其妻悠子在黑道復仇中被槍殺。
: 將要求加入「渡海興業」的北條帶入組織。雖曾憎惡出色的北條趕超自己的老大地位，但被北條感化，在北條成為相樂聯合總長之後成為他的監護人及左右手。為達成相樂聯合稱霸日本的目的，北條委托他向南方擴展勢力，向[[香港|香港]]出發。

; {{nihongo|田代 憐二|たしろ れんじ|Tashiro Renji}}
* 聲：[[關智一|關智一]]
: 北條的心腹。因北條救了他的獨生子光一，於是跟隨北條成為黑道人物。在北條成為相樂聯合總長後，成為北彰會第二代首領。

; {{nihongo|相樂|さがら|Sagara}}
* 聲：[[笹岡繁蔵|笹岡繁蔵]]
: 黑道組織相樂聯合的總長，暗中與眾議員佐倉秀一有來往。利用渡海和北條的矛盾讓他們互相廝殺，來達到打壓北條的目的。後被北條殺掉。

; {{nihongo|村田|むらた|Murata}}
* 聲：[[佐藤正治|佐藤正治]]
: 相樂聯合的幹部。前總長死後成為北條的心腹。擁有由經驗而生的冷靜判斷力。

====武山聯合====
; {{nihongo|宮村 政信|みやむら まさのぶ|Miyamura Masanobu}}
: [[關東|關東]]三大勢力之一武山聯合的「宮村組」組長。武山聯合之中最能冷靜觀察的人物。年輕的時候曾受渡海和北條幫忙。
: 殺害了「德田組」的組長，成立了武山聯合和相樂聯合的同盟。

;徳田貞幸
:德田組組長，個性火爆，中北條之計而向土門會開戰，後被宮村政信所殺。

====土門會====
;河津
:土門會總長。

====神戶山王會====
; {{nihongo|緒方 榮蔵|おがた えいぞう|Ogata Eizo}}
: 神戶山王會第三代總長。與北條對峙過程中通過和他直接對話，明白到自身的弱點，為了守護山王的代紋而引退，將一切交托於伊吹。

; {{nihongo|伊吹 英明|いぶき ひであき|Ibuki Hideaki}}
: 神戶山王會第二把交椅。受緒方任命繼承山王會成為第四代總長。不久後協助北條共同致力於黑道世界的改革。


====其他====
; {{nihongo|大西 英二|おおにし えいじ|Oonishi Eiji}}
: 自己的組織因反抗[[神戶|神戶]]山王會而潰散。和渡海是獄中好友，因而幫忙勸動[[九州|九州]]第一的勢力「小倉岩田組」（組長・岩田浩德）。

; {{nihongo|中城 規介|なかぐすく きすけ|Nakagusuku Kisuke}}
: [[沖繩|沖繩]]的黑道首領。為了將沖繩的旗幟立於本州，答應和渡海、黃志陽聯手。

; {{nihongo|黄 志陽|こう しよう|Wong Chi-Yeung}}
: [[香港|香港]]某個黑社會的第二把交椅。對六年後的[[香港回歸|香港回歸]]感到危機，圖謀進軍日本而和渡海聯手。手段狠辣。

; {{nihongo|市島 陽介|いちじま ようすけ|Ichijima Yousuke}}
: 黑暗世界幕後首腦，已退休在馬場養馬。想在日本培育人才，賞識繼承相樂聯合的北條，時常給予幫助。

=== 警察相關人物 ===
;{{nihongo|石原 杏子|いしはら きょうこ|Ishihara Kyoko}}
* 聲：[[鶴弘美|鶴弘美]]
: [[警視廳|警視廳]]·[[六本木|六本木]]署的副署長，27歲（生日：1963年4月1日）。東京大學畢業後通過國家公務員上級職稱考試。為了得到北條的罪證而開始調查他，在接觸之下卻對他產生好感。

;{{nihongo|尾崎|おざき|Ozaki}}
* 聲：[[石塚運昇|石塚運昇]]
: 六本木署的警察，石原的部下。戴太陽鏡，留著不修邊幅的鬍鬚。諳熟黑道，和渡海認識很久。為了引出北條故意對渡海煽風點火，在渡海洗心革面後多番幫忙他。

;{{nihongo|熊谷|くまがい|Kumagai}}
: 警視廳搜查四課（暴力團對策）的課長。頂住抗爭策劃要一鼓作氣瓦解日本的極道組織。

;{{nihongo|平山 隆|ひらやま たかし|Hirayama Takashi}}
: [[兵庫縣|兵庫縣]]警局搜查四課的課長。體型相當短小。被上級認為是極優秀的人才，卻拒絕進入[[大藏省|大藏省]]和[[通產省|通產省]]任高職而自願選擇當警官，被人認為是個古怪的傢伙。和熊谷一同為破滅黑道組織而行動。

=== 政界相關人物 ===
; {{nihongo|仙石 慎一郎|せんごく しんいちろう|Sengoku Shinichiro}}
: [[北海道|北海道]]選出的新人國會議員。父親是民自黨伊佐岡派系的議員，後在伊佐岡的策略下被逼至自殺。被後援會推舉與淺見同時當選。言行出位，<!--氣焰囂張地說出「一旦進入國會可不能這樣就完結」的話語。-->進入國會第一天和淺見不打不相識，之後和吉川三人以一年級議員的身份致力政治改革。

; {{nihongo|吉川 秀丸|よしかわ ひでまる|Yoshikawa Hidemaru}}
: [[高知|高知]]選出的新人國會議員。和淺見、仙石同時當選，共同進退。

; {{nihongo|狩谷 久雄|かりや ひさお|Kariya Hisao}}
: 民自黨眾議員。雖然在由「[[團塊世代|團塊世代]]」組成的超黨派組別「70年會」裡最早當選，並以核心人物的身份活動著，但一直被民自黨內部的老狐狸議員打壓。一開始立場與淺見對立，經歷過比絲桃事件後了解自己作為政治家的問題和「應該做的事」，於是和淺見聯手進行政治改革。

; {{nihongo|佐倉 秀一|さくら しゅういち|Sakura Shuichi}}
* 聲：[[岡部政明|岡部政明]]
: 民自黨眾議員。68歲。[[東京|東京]]地方區連任7次的大政治家，與黑道組織相樂聯合有援助關係。有一個在醫院療養中的妻子千代和名叫由香的女兒。目標是登上下任大臣的寶座，卻在淺見和北條的計策下失足。

; {{nihongo|原口 賢二|はらぐち けんじ|Haraguchi Kenji}}
: 民自黨伊佐岡派系的[[國會議員|國會議員]]，通稱「笑面虎」。是一個時常依附於當時勢力最大的派系的野心家<!-- ，「寝業師」「カメレオン」と称される -->。與導致仙石父親自殺的「北海道未來樂園度假村」開發計劃有關係。

; {{nihongo|伊佐岡 紀元|いさおか のりもと|Isaoka Norimoto}}
* 聲：[[渡部猛|渡部猛]]
: 民自黨幹事長，擁有操縱日本政治的強大權力。利用國民對政治的不關心來構築現在的地位。敵視由淺見率領的一眾年輕政治家，絞盡腦汁要保護自己的權力。<!-- 年の割に絶倫で、話が進むたびに身長の低さが強調されるようになる。 -->

; {{nihongo|比絲桃|ビセット|Bisset}}
: [[美國|美國]][[白宮幕僚長|白宮幕僚長]]，32歲的對日強硬派。為了糾正日美汽車貿易不平衡問題，擴大美國汽車在日本市場的份額而來日交涉。但後來被自薦作為談判代表的淺見說服而回國。

; {{nihongo|克林頓|クリント|Clint}}
: 時任[[美國|美國]][[總統|總統]]。命比絲桃訪問日本。其理念是「美日兩國要改革成彼此能明確說出YES或NO地坦誠對話的關係」。
: 在渡假期間暗中訪日，在正在拉票的淺見的拉票車上當眾與他握手。

=== 其他 ===
; {{nihongo|青木|あおき|Aoki}}
: 投機公司「藍色世界」的會長。協助北條充當聯繫北都銀行和相樂聯合的[[融資|融資]]窗口。
