'''聖痕鍊金士'''（{{lang-ja|聖痕のクェイサー}}）是[[吉野弘幸|吉野弘幸]]原作、[[佐藤健悅|佐藤健悅]]作畫的校園格鬥漫畫，在《[[Champion_RED|Champion RED]]》連載中並改編成動畫；[[台灣|台灣]]方面則由[[長鴻出版社|長鴻出版社]]出版[[繁體中文|繁體中文]]版漫畫，DVD則由[[木棉花國際|木棉花國際]]發行。

== 概要 ==
[[2006年9月|2006年9月]]在[[秋田書店|秋田書店]]的月刊漫畫雜誌《Champion RED》2006年10月號連載到2016年9月號，全118話、[[單行本|單行本]]24卷，標榜非[[成人漫畫|成人漫畫]]的「吸奶的漫畫」（{{lang|ja|乳房を吸う漫画}}）引發話題。於2010年1月開始播出動畫版，目前共計兩季。[[OVA|OVA]]《聖痕鍊金士 女帝的肖像》于2010年10月20日随漫画單行本第十卷發行。

== 故事簡介 ==
來自俄國的少年沙夏，其左臉上有十字架形聖痕，因幼年時的恐怖經歷，使他變成對任何人都冷淡對待的冷酷個性。這位冷酷的俄國少年是有著「致命者」稱號的鍊金士，他能夠透過吸吮聖乳而自由地操縱鐵元素的能力。

其後沙夏接受命令進入「私立翠玪學園」，翠玪學園是女校，對於鍊金士來說便是源源不斷的魔力供應之地，隨便抓一個都可隨時隨地的補充能量，但真正目的是找到始原元素迴路（Hyancient circuit）之一的「雷之攜香女」。和沙夏一樣同時轉入的還有女王卡恰的「生神女」桂木華，看來卡恰將作為沙夏的後盾、在暗處和他共同執行任務。沙夏在扮成女性潛入學園後，在這間充滿女學生的女校內，遇見了天乃翼、瀨田深雪、皋月綾女，在找尋「雷之攜香女」中佔有舉足輕重的角色。

== 電視動畫 ==
2010年1月9日开始於[[每日放送|每日放送]]、[[TOKYO_MX|TOKYO MX]]播放，分为[[和諧社會|通常版（TV放送版）]]和导演剪辑版。导演剪辑版的进度落后通常版约一周时间，可以[https://web.archive.org/web/20100122064638/http://anime.biglobe.ne.jp/title/3513 在线]付费收看。两者相比，导演剪辑版的画面和情节更加完整一些。在[[DVD|DVD]]和[[蓝光光碟|蓝光光碟]]之中收录的是导演剪辑版。全24話。

2011年4月11日推出第二季，由TOKYO MX深夜時段放送。分為電視版本及導演剪輯版本。
