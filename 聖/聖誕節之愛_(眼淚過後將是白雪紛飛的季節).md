{{nihongo|《'''聖誕節之愛 （眼淚過後將是白雪紛飛的季節）'''》|クリスマス・ラブ （涙のあとには白い雪が降る）}}，是日本樂團[[南方之星|南方之星]]1993年發行的第34張[[單曲|單曲]]。

== 概要 ==
為南方之星繼1980年推出的單曲《[[Sha_La_La／對不起_查理|Sha La La／對不起 查理]]》後、再次以聖誕節為題材的作品。日本連鎖百貨[[丸井|丸井]]公司曾採用此曲作為聖誕節期間的廣告歌曲。<ref>{{Cite web |url= http://www.sas-hour.com/single/57.html |title= 南方之星單曲《聖誕節之愛 （眼淚過後將是白雪紛飛的季節）》介紹 |author=SAS-hour |date=2007-10-18 |publisher= |language= ja |accessdate=2011-12-23}}</ref>
 
當時在錄製歌曲時因南方之星原先的貝斯手[[關口和之|關口和之]]在做休養當中、而請另一名樂手根岸孝旨暫時頂替他的位子。

歌曲末段內容為眾人的大合唱，與南方之星主唱[[桑田佳祐|桑田佳祐]]音喜愛的歌手[[約翰·藍儂|約翰·藍儂]]一首以聖誕節為主題的單曲《[[Happy_Xmas_(War_Is_Over)|Happy Xmas (War Is Over)]]》相似的風格。本作的音樂錄音帶則在北海道[[函館市|函館市]]一帶拍攝。

作品的週內銷售最高成績為第3名，<ref>{{Cite web |url= http://www.musictvprogram.com/corner-ranking-1993-11-29.html |title= Oricon1993年11月末週單曲銷售 |author=  |date=  |publisher= MusicTVProgram |language= ja |accessdate= 2011-12-23 |quote=  |archive-url= https://web.archive.org/web/20110520134238/http://musictvprogram.com/corner-ranking-1993-11-29.html |archive-date= 2011-05-20 |dead-url= yes }}</ref>並在2005年重新發行。
