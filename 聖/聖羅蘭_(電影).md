《'''聖羅蘭'''》（[[法語|法語]]：''{{lang|fr|Yves Saint Laurent}}''）是一套[[2014年電影|2014年]]的[[傳記片|傳記]][[劇情片|劇情片]]，由{{tsl|en|Jalil Lespert|加里·勒斯培}}導演並與{{tsl|en|Jacques Fieschi|賈克·費許}}合作編劇，電影基於[[伊夫·聖羅蘭|伊夫·聖羅蘭]]1958年之後的生平改編。<ref name="Yves Saint Laurent Biopic Wins Pierre Bergé's Approval">{{cite web | url=http://www.wwd.com/eye/people/yves-saint-laurent-biopic-wins-pierre-bergs-approval-7333885?src=nl/wkEye/20140110 | title=Yves Saint Laurent Biopic Wins Pierre Bergé's Approval | publisher=WWD | date=10 January 2014 | accessdate=10 January 2014 | author=Diderich, Joelle}}</ref>

== 反響 ==
《聖羅蘭》獲得褒貶不一的評價，在[[爛番茄|爛番茄]]上錄得45%的新鮮度，以10為滿分的平均分中獲得5.3分。網站上的一條共識評論寫道：「雖然演員的表演無比精彩，但是綜合來看還是流於平淡和公式化而令人失望，尤其考慮到電影本身擁有一個華麗耀眼的主題。」<ref>{{cite web |url=http://www.rottentomatoes.com/m/yves_saint_laurent/ |title=Yves Saint Laurent (2014) |work=Rotten Tomatoes |accessdate=22 August 2014}}</ref>在[[Metacritic|Metacritic]]上以100為滿分的平均分中獲得51分，<ref>{{cite web |url=http://www.metacritic.com/movie/yves-saint-laurent |title=Yves Saint Laurent |work=Metacritic |accessdate=22 August 2014}}</ref>可見是比較兩極化的評價。
