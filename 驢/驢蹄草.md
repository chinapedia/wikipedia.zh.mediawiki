'''驢蹄草'''（[[学名|学名]]：''{{lang|la|Trollius paluster}}''）是一種屬於毛茛科驢蹄草屬的[[多年生|多年生]][[草本植物|草本植物]]，又名'''驢蹄菜'''、'''沼澤金盞花'''（Marsh Marigold）、[[立金花|立金花]]。原產於[[北半球|北半球]]的[[溫帶|溫帶]]地區，包括[[歐洲|歐洲]]的[[冰島|冰島]]與[[俄羅斯|俄羅斯]][[北極|北極]]地區，[[亞洲|亞洲]]溫帶與北極地區及[[北美洲|北美洲]]。

== 名稱與詞源 ==
種小名''palustris''，名稱來自[[拉丁文|拉丁文]] palus，是「沼澤」的意思，形容驢蹄草多生長在[[沼澤|沼澤]]、[[溼地|溼地]]等潮濕泥濘的地方。

在[[拉脫維亞|拉脫維亞]]，驢蹄草被稱為''Gundega''，這個名稱也用作女孩子的名字，是「火」的象徵。''Gundega''這個字是由''uguns''（火）和''dega''（燃燒）兩個字組合而成的，形容某些人在碰到驢蹄草的汁液時會產生灼熱感。

== 形態特徵 ==

驢蹄草是多年生草本植物，株高15至60[[公分|公分]]，最高可以生長至80公分。[[莖|莖]]中空。[[葉子|葉子]]形狀為圓形至腎臟形，具有鈍鋸齒狀的葉緣，葉片厚，葉表面光亮蠟質。[[花|花]]黃色，花徑2-5公分，具有4-9片（大多為5片）[[花瓣|花瓣]]狀的[[花萼|萼片]]，[[雄蕊|雄蕊]]黃色，雄蕊多數。在早春至夏末時開花。 在心皮表面各有一個小凹窩，花蜜就是這些凹窩分泌出來的。[[果實|果實]]為綠色囊狀的[[蓇葖果|蓇葖果]]，蓇葖果長1公分，成熟時會裂開釋放出一些[[種子|種子]]，種子深褐色，長約0.25公分。

花含有豐富的[[花粉|花粉]]和[[花蜜|花蜜]]，會吸引許多種類的[[昆蟲|昆蟲]]，包括[[甲蟲|甲蟲]]、[[蒼蠅|蒼蠅]]及[[蜜蜂|蜜蜂]]等來採集，這些昆蟲也會幫助驢蹄草傳播花粉。

== 栽培 ==
驢蹄草喜歡生長在潮濕泥濘的地方，例如濕地、沼澤地、溝渠及潮濕的林地，不過很少出現在[[泥炭|泥炭]]地上。它在有部分遮蔭的環境下生長的最好，植株長的非常茂盛。

生長在黏質[[土壤|土壤]]裏的驢蹄草繁殖力很強，只要一小段的[[根狀莖|根狀莖]]就可生長繁殖，蔓延散布至花園各處，因此在某些花園裏被當做是[[雜草|雜草]]。但是在氣候溫暖且排水良好的土壤裏，它卻不大會蔓延，很容易枯死。

驢蹄草和許多毛茛科植物一樣，整株植物體都可能含有刺激性或是有毒的物質。已經有報告指出，以手過度的觸摸植物體會導致[[皮膚|皮膚]]發炎及發生皮疹。
