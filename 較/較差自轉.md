'''較差自轉'''，又名差動自轉。是指一个天体在[[自转|自转]]时不同部位的[[角速度|角速度]]互不相同的现象。較差自轉在大多数非固体的天体中存在，比如[[星系|星系]]、[[恒星|恒星]]、[[類木行星|巨型气体行星]]等等；太陽系內則在太陽、木星和土星的表面出現。

== 歷史 ==
據歷史記載，[[伽利略·伽利萊|伽利略·伽利萊]]在觀測[[太陽黑子|太陽黑子]]時首度察覺到此一現象，成為第一位觀察到較差自轉者。而後，克里斯托夫·賽因那(Christoph Scheiner)於1630年左右指出太陽在極區與赤道區的自轉週期差異，與現今觀測結論並無太多差別。

== 成因 ==
星體的自轉來自於其在前恆星(prestellar)的吸積階段(accretion phase)，以及對角動量的守恆而來。而較差自轉的成因主要來自於星體自身結構內部的對流；由於恆星內部有溫度梯度等影響，對流會使得內部及外部的物質進行類似置換的動作，而小區塊物質本身帶有恆星的部份角動量，不同區域的對流造成了恆星內部角速度分佈的重新配置，而形成了較差自轉的現象。需要注意的是，有時[[恆星風|恆星風]]也是恆星損失角動量的來源。

== 银河系的较差自转 ==
{{see also|银河系|奥尔特常数}}
盘状星系不像固体一样旋转，但会发生较差自转。星系旋转速度随半径的变化关系称为旋转曲线，它通常被看作是对星系质量轮廓的一种测量：

:<math>v_{c}(R)=\sqrt{\frac{GM(<R)}{R}}</math>

其中
* <math>v_{c}(R),</math> 是半径 <math>R</math> 处的旋转速度
* <math>M(<R),</math> 是半径 <math>R</math> 以内包含的总质量
