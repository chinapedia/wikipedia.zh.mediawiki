'''元代代木町'''（{{jpn|j=元代々木町|hg=もとよよぎちょう|rm=Moto-yoyogichō|f=y}}）是[[東京都|東京都]][[澀谷區|澀谷區]]的町名。2017年12月1日為止的[[人口|人口]]有3,609人<ref>[http://www.city.shibuya.tokyo.jp/data/statics/tokei/02jinko/h25r01.xls 渋谷区／住民基本台帳・外国人登録による人口] {{webarchive|url=https://web.archive.org/web/20131013081847/http://www.city.shibuya.tokyo.jp/data/statics/tokei/02jinko/h25r01.xls |date=2013-10-13 }}</ref>。[[郵遞區號|郵遞區號]]為151-0062。

== 地理 ==
位於澀谷區西北部。北臨[[初台|初台]]，東接[[代代木|代代木]]，南連[[上原_(澀谷區)|上原]]，西通[[西原_(澀谷區)|西原]]。町內主要為住宅區，一部分是連接[[代代木上原|代代木上原]]的高級住宅區，並有大使館坐落於此。町域東南端有[[小田急小田原線|小田急線]][[代代木八幡站|代代木八幡站]]與[[東京地下鐵千代田線|東京地下鐵千代田線]][[代代木公園站|代代木公園站]]。町域東邊有南北向的幹道[[東京都道317號環狀六號線|山手通]]。

== 歷史 ==
元代代木町在町名變更前稱「代代木本町」，當時町域還包括[[代代木八幡宮|代代木八幡宮]]（現在[[代代木|代代木]]5丁目）等地區。

== 設施 ==

* 元代代木郵便局
* 元代代木保育園
* シオン幼稚園
* 應慶寺
