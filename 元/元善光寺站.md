'''元善光寺站'''（{{jpn|j=元善光寺駅|hg=もとぜんこうじえき|rm=Motozenkōji eki|f=y}}）是位於[[日本|日本]][[長野縣|長野縣]][[飯田市|飯田市]]座光寺的[[東海旅客鐵道|東海旅客鐵道]]（JR東海）和[[日本貨物鐵道|日本貨物鐵道]]（JR貨物）[[飯田線|飯田線]]車站。

== 車站構造 ==
[[側式月台|側式月台]]1面1線與[[島式月台|島式月台]]1面2線，可進行列車交會的[[地面車站|地面車站]]。各月台以站內平交道聯絡，側式月台側（北側）設有站舍。在島式月台側有一條[[側線|側線]]。島式月台南側設有1條側線。

[[飯田站|飯田站]]管理的[[無人站|無人站]]。至2013年3月為止[[東海交通事業|東海交通事業]]的職員擔當業務，是業務委託站，一度設置[[綠窗口|綠窗口]]（早上、夜間為無人站）。

== 利用狀況 ==
在本站步行5分鐘便到達[[長野縣飯田工業高等學校|長野縣飯田工業高等學校]]，在上學時間，學生會利用此車站。以下就是上車乘客人數。
* 2003年 - 550人
* 2004年 - 553人
* 2005年 - 529人
* 2006年 - 491人
* 2007年 - 474人
* 2008年 - 506人
* 2009年 - 485人

== 車站周邊 ==
在本站步行7分鐘便到達[[元善光寺|元善光寺]]，沿途都有許多賣土產的店鋪。元善光寺南側是麻績之里、[[麻績神社|麻績神社]]，社內有一棵400年的樹，稱舞台櫻。
* [[長野縣飯田工業高等學校|長野縣飯田工業高等學校]]
* [[太平洋水泥|太平洋水泥]]飯田服務站
* [[平安堂|平安堂]]座光寺店

== 歷史 ==
* 1923年（[[大正|大正]]12年）3月18日 - [[伊那電氣鐵道|伊那電氣鐵道]]線山吹站至元善光寺站延伸段啟用。新設'''元善光寺站'''終點站。
* 1923年（大正12年）8月3日 - 伊那電氣鐵道線飯田站至元善光寺站延伸段啟用。本站成為中途站。
* 1943年（[[昭和|昭和]]18年）8月1日 - 伊那電氣鐵道線國有化，成為[[日本國有鐵道|國鐵]]飯田線車站。並改名為'''座光寺站'''。
* 1950年（[[昭和|昭和]]25年）5月20日 - 車站再次改名為'''元善光寺站'''。
* 1971年（昭和46年）12月1日 - 行李・専用線出發外的貨物起卸服務停止。
* 1986年（昭和61年）11月1日 - 晚上無人站。
* 1987年（昭和62年）4月1日 - 因[[國鐵分割民營化|國鐵分割民營化]]，本站成為東海旅客鐵道和日本貨物鐵道的車站。
* 1996年（[[平成|平成]]8年）9月30日 - 貨物列車最後運作日。
* 1998年（平成10年）3月31日 - 開始業務委託。
