'''元朗廣場'''（'''{{lang-en|Yuen Long Plaza}}'''）是[[香港|香港]][[新界|新界]][[元朗|元朗]]的[[住宅|住宅]]物業和[[零售|零售]][[購物商場|購物商場]]，位於[[青山公路元朗段|青山公路元朗段]]249-251號，於1989年10月落成。<ref name="official_site">[http://www.yuenlongplaza.com/index.php?option=com_content&view=article&id=14&Itemid=12&lang=tc] Yuen Long Plaza</ref>

== 背景 ==
元朗廣場原址為[[九巴|九巴]]車廠，到1980年代末遷走。九龍巴士的大[[股東|股東]][[新鴻基地產|新鴻基地產]]在原地興建一座[[單幢樓|單幢樓]][[住宅|住宅]]。

== 位置 ==
商場位於青山公路元朗段的北側，鄰近元朗民政事務處、[[香港輕鐵|輕鐵]][[豐年路站|豐年路站]]及[[元朗西巴士總站|元朗西巴士總站]]。

== 設施和配套 ==
商場總面積15萬[[平方呎|平方呎]]，地面的一層設有佔地1,700平方英呎的展覽場地可作為推廣活動使用。場內集合各[[商舖|商舖]]及中西日式[[食肆|食肆]]多家<ref name="official_site"/>。而配合政府延長[[朗屏邨|朗屏邨]]對外的[[行人天橋|行人天橋]]計劃，商場2樓打通了兩條行人通道，北通朗屏邨及[[西鐵綫|西鐵綫]][[朗屏站|朗屏站]]，南可橫過[[青山公路元朗段|青山公路元朗段]]及到達[[香港輕鐵|輕鐵]][[豐年路站|豐年路站]]。

== 住宅 ==
住宅為單棟式物業，樓高28層，合共提供224伙單位，分3房2廳及2房1廳單位。於平台花園擁有一個25米的游泳池。現時由康業物業管理公司負責管理，平均管理費為$12000元。
