《'''私人订制'''》（{{lang-en|''Personal Tailor''}}）是中國導演[[冯小刚|冯小刚]]执导的一部讽刺喜剧電影，由[[王朔|王朔]]编剧，[[葛优|葛优]]、[[郑恺|郑恺]]、[[白百何|白百何]]和[[李小璐|李小璐]]主演，中国于2013年12月19日上映。<ref>[http://www.apdnews.com/photo/56129.html 冯小刚电影《私人订制》上海交大公映] {{webarchive|url=https://web.archive.org/web/20131220125207/http://www.apdnews.com/photo/56129.html |date=2013-12-20 }}，亚太日报，2013年12月20日</ref><ref >{{cite web|url=http://news.mtime.com/2013/06/16/1513292.html | language= | title=华谊公布2014年十部新片 冯小刚"私人订制"首露真容 |publisher=Mtime时光网 |date=2013-06-16 |accessdate= }}</ref>

== 剧情 ==
愿望规划师杨重、情境设计师小白、梦境重建师小璐与心灵麻醉师马青四人组成的公司“私人订制”，以替他人圆梦为自身业务，专门为不同客户量身订制“圆梦方案”。无论客户的白日梦多奇葩、要求多严格，「圆梦四人组」统统来者不拒,甘愿满足客户的任何需求,正如同公司的口号:「成全别人，恶心自己」!<br /><br />
许多心怀「奇葩梦」的客户因此纷纷找上门，公司也接连面临各种挑战: <br /><br />一心想当烈士的陕西女青年；<br /><br />号称全球最庸俗的导演却立志追求高雅品味；<br /><br />想要当清官的司机自愿接受钱色诱惑；<br /><br />生日愿望是想变成有钱人的河道清洁工人丹姐……<br /><br />「圆梦四人组」绞尽脑汁，誓要为每一位客户度身订造最贴心的圆梦方案!

== 创作 ==
最初片名为《贵族》，2013年在[[海南|海南]]取景拍摄。<ref >{{cite web|url=http://news.mtime.com/2013/06/19/1513478.html | language= | title="私人定制"台词已火 "恶心自己"是冯导心态？ |publisher=河南商报 |date=2013-06-19 |accessdate= }}</ref>冯小刚在接受采访时表示，拍本片是为了让华谊兄弟赚到钱，因为上部作品《[[一九四二|一九四二]]》赔了钱，而且影片仅几个[[广告|广告]]植入高达8000万元就已回本<ref name="死磕">{{cite web|url=http://ent.163.com/13/1230/06/9HASEBMI00031H2L.html?f=resysBvalid1#apage_resys | language= | title=冯小刚：跟影评人死磕 逮着他们就骂 |publisher= 新京报 |date=2013-12-30 |accessdate= }}</ref>。他觉得这是一部“好玩又胡扯”的电影，能让观众好好放松放松；<ref >{{cite web|url=http://ent.huanqiu.com/yuleyaowen/2013-12/4677937.html | language= | title=冯小刚谈拍《私人订制》原因：想让华谊赚钱 |publisher=千山晚报 |date=2013-12-19 |accessdate= }}</ref>影片“[[剧本|剧本]]是冯小刚和王朔喝完酒之后聊出来的，充满了王朔的各种碎片，《甲方乙方》的结构，《[[顽主|顽主]]》里的段子，以及早年一个名叫《贵族》剧本里的[[台词|台词]]”。冯小刚表示：“《私人订制》由三个故事组成。第一个说的是[[土壤|土壤]]，是《[[一九四二|一九四二]]》的现代版，说得还是人民性，如出一辙。第二个故事是[[嘲笑|嘲笑]]，但嘲笑的不是俗。第三个故事是白给的，为的是让第一个故事能存活下来。最后的道歉和王朔没有关系，是我内心对这个世界仅存的一点敬畏。”<ref name="发飙"/>

== 反响 ==
根据[[影评人|影评人]]和观众反响来看，影片备受争议。一些差评认为此片为“怒剧”，“不尊重电影，瞧不起观众，没有人物和结构，只有[[广告|广告]]，唯一有价值的东西是编几个[[笑话|笑话]]逗你玩”，“冯小刚今年拍[[春晚|春晚]]是拿《私人订制》练手，练了仨[[小品|小品]]一[[朗诵|朗诵]]，三俗一雅终归一烂”<ref>{{cite web |url=http://travel.taiwan.cn/list/201312/t20131219_5396360.htm |language= |title=《私人订制》遭遇滚滚差评 影评人称烂到怒不可赦 |publisher=凤凰网娱乐 |date=2013-12-19 |accessdate= |archiveurl=https://web.archive.org/web/20131219193017/http://travel.taiwan.cn/list/201312/t20131219_5396360.htm |archivedate=2013-12-19 |deadurl=yes }}</ref><ref>{{cite web |url=http://ent.xinmin.cn/2013/12/19/23000701.html |language= |title=《私人订制》获零好评 网友讽冯小刚：成全自己恶心别人 |publisher=四川在线 |date=2013-12-19 |accessdate= |deadurl=yes |archiveurl=https://web.archive.org/web/20131219212426/http://ent.xinmin.cn/2013/12/19/23000701.html |archivedate=2013-12-19 }}</ref><ref >{{cite web|url=http://ent.china.com.cn/live/2013-12/19/content_23961992.htm | language= | title=《私人订制》公映 被指更像三段小品 |publisher=中国网 |date=2013-12-19 |accessdate= }}</ref><ref >{{cite web|url=http://ent.sina.com.cn/m/c/2013-12-19/15184064353.shtml | language= | title=冯小刚独家回应负评：尊重观众所有的感受 |publisher= 新浪娱乐 |date=2013年12月19日 |accessdate= }}</ref>。

[[网易|网易]]的“独家影评”栏目的匿名文章对影片和《[[甲方乙方|甲方乙方]]》进行了对比，对剧本和影片的结构，及人物塑造方面提出非常严厉的批评。<ref >{{cite web|url=http://ent.163.com/13/1219/07/9GEL6O4H00034R77.html | language= | title=《私人订制》：降级版《甲方乙方》 |publisher=网易独立影评 |date=2013年12月19日 |accessdate= }}</ref>

影评人周黎明对本片持肯定态度；更有人表示，冯导作品被骂很可能跟他一贯的口无遮拦、不太注重个人形象有关。<ref name="形象">{{cite web|url=http://ent.163.com/13/1230/06/9HAS64HU00031H2L.html| language= | title=影评人回应冯小刚：冯导不太注意形象 |publisher=京华时报 |date=2013-12-30 |accessdate= }}</ref>

[[纽约时报|纽约时报]]中文网的署名文章批评影片“有些地方比较散乱，片子也过于冗长”，同时也肯定了演员的演技，影片的思想和拍摄技术<ref>{{cite web |url=http://cn.tmagazine.com/film-tv/20131221/t21tailor/ |language= |title=“私人订制”个中国梦 |publisher=纽约时报中文网 |date=2013年12月21日 |accessdate= |archive-url=https://web.archive.org/web/20140101212620/http://cn.tmagazine.com/film-tv/20131221/t21tailor/ |archive-date=2014-01-01 |dead-url=yes }}</ref>。[[Variety|Variety]]的署名评论认为影片节奏缓慢而轻松，肯定了演员的演技和剧本的思想和内容，但片子冗长，而且并未在人物塑造方面下大工夫<ref >{{cite web|url=http://variety.com/2013/film/reviews/film-review-personal-tailor-1200982211/ | language= | title=Film Review: ‘Personal Tailor’ |publisher=variety.com |date=2013年12月20日 |accessdate= }}</ref>。

面对影评人的恶评，冯小刚导演连续发[[微博|微博]]痛斥绝大多数影评人浅薄，“别现眼了”，“从一九四二到私人订制，你们的嘲笑和狂欢恰恰反映了你们的浅薄，我看不起你们，别再腆着脸引领观众了，丢人”，称“观众热衷爆米花喜剧我理解，但我无心伺候”，“观众买了票怎么骂我都行。影评人收钱还跟这儿骂，他们有权利骂我，我也有权利骂他们。我是跟他们死磕了，逮住他们就骂他们。当然他们也没有上岗证，街上卖煎饼的都有上岗证，他们没有”，“有一帮人、尤其什么[[豆瓣|豆瓣]]上的，都些大尾巴[[狼|狼]]，假装懂电影，狗屁！还在那还说什么他的镜头[[蒙太奇|蒙太奇]]，你们知道什么是蒙太奇呀？”<ref name="死磕"/><ref >{{cite web|url=http://ent.sina.com.cn/m/c/2013-12-29/21414070966.shtml | language= | title=《私人订制》冯导轰影评人是大尾巴狼 |publisher= 新浪娱乐 |date=2013年12月29日 |accessdate= }}</ref>；并表示“《私人订制》突破了对[[权力|权力]]的[[讽刺|讽刺]]”，“就对现实的批判性来说，我给它打9分”<ref name="发飙">{{cite web|url=http://ent.163.com/13/1229/10/9H8MU5P2000300B1.html | language= | title=冯小刚发飙驳影评人差评:别TM现眼了 |publisher=网易娱乐 |date=2013-12-29 |accessdate= }}</ref>，“《私人订制》可能都不用10年，5年内就成经典。” 在第15届中国电影[[华表奖|华表奖]]领奖时，冯小刚表示“我随随便便拍一部电影就卖钱，我认认真真拍一部电影就不卖钱！这让我产生了很大的困惑！”<ref name="形象"/>

本片被[[搜狐|搜狐]]评为2013下半年十大华语烂片第二名，仅次于《[[小时代：青木时代|小时代2]]》<ref >{{cite web|url=http://yule.sohu.com/20140109/n393173811.shtml | language= | title=2013年下半年搜狐十大佳片十大烂片评选揭晓 |publisher=搜狐娱乐 |date=2014年1月9日 |accessdate= }}</ref>。

== 票房 ==
本片在中国大陆上映后，首周四天收获创纪录的3.3亿元票房，位列该周票房榜第一名，打破了电影《[[人再囧途之泰囧|人再囧途之泰囧]]》在2012年12月创下的国产片首周票房纪录（3.1亿）<ref>{{cite web |url=http://news.mtime.com/2013/12/24/1522147.html |title=《私人订制》首周创纪录夺冠 掀起贺岁档热潮 |date=2013年12月24日 |publisher=Mtime时光网}}</ref>；次周收获2.202亿元列第二名，比首周四天下滑三分之一，场均31人比首周的48人下滑35%<ref>{{cite web |url=http://news.mtime.com/2013/12/31/1522401.html |title="警察故事2013"开局夺冠 贺岁档周票房冲新高 |date=2013年12月31日 |publisher=Mtime时光网}}</ref>；第三周收1.164亿元仍居亚军，累计6.67亿元<ref>{{cite web |url=http://news.mtime.com/2014/01/07/1522663.html |title=新年首周《警察故事》连冠 《等风来》收六千万 |date=2014年1月7日 |publisher=Mtime时光网}}</ref>。至2014年2月2日，累计票房将近7.2亿元<ref>{{cite web | title=中国电影报2014年2月11日新浪微博 | url=http://weibo.com/2304129841/AwfVB4898 | date=2014-02-11}}</ref>。
