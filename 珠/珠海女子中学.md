'''珠海女子中学'''（{{lang|en|'''Zhuhai Girl's Middle School'''}}，简称'''珠海女中'''）是由[[珠海恒隆集团|珠海恒隆集团]]兴办的一所民办全寄宿制女子完全中学。设有33个班，在校学生约1000人，于2011年9月1日正式成立的[[广东省|广东省]]首个私立[[女子中学|女子中学]]。<ref>[http://news.sina.com.cn/o/p/2011-03-22/061222157463.shtml 广东首家女子中学揭开面纱]新浪新闻,2011-03-22</ref>

== 概况 ==
珠海女子中学创建于2011年，是建国以来[[广东省|广东省]]、[[华南|华南]]地区[[珠海市|珠海市]]第一所完全女子中学，是中国首个民办完全女子中学。学校占地面积2.5万平方米，建筑面积3.7万平方米。<ref name="intro">[http://www.zhuhaiyunv.com/xuexiao/zhgirlschool.html 珠海女子中学简介]</ref>

鉴于世界女子中学的普遍特性的研究，珠海女中的定位是走高端线路，主要为满足具有良好经济条件并对女孩培养有更高要求的家庭需求，培养具有独立、知性、平和、关爱、美丽特质的，知书达礼、秀外慧中的国际化现代女性。

珠海女中的校园建筑是经典欧式学院风格，线条流畅、雕梁画栋、美轮美奂；珠海女中的办学思想来之于水的特性和灵感。

== 办学宗旨 ==
依据现时女生的思维发展特点，提供符合女生学习成长的教学课程和方式，为女生创造更有针对性的优质教育。通过提高女生综合素质，为其终身发展打下基础。打造一所“特色鲜明、国内一流、国际知名”的女子中学<ref name="intro">[http://www.zhuhaiyunv.com/xuexiao/zhgirlschool.html 珠海女子中学简介]</ref>。

== 培养目标 ==
培养知书达礼、秀外慧中的现代女性。
