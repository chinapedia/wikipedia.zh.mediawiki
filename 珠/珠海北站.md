'''珠海北站'''，前称'''金鼎站'''，位于[[中国|中国]][[广东省|广东省]][[珠海市|珠海市]][[珠海国家高新技术产业开发区|珠海国家高新技术产业开发区]]金鼎工业片区内、金鼎北圍北霎村的附近，是[[广珠城际铁路|广珠城际铁路]]的一个车站，也是广珠城际进入珠海市境内的第一站。

珠海北站于2008年动工，2010年5月开始站房主体结构的施工，同年8月底主体工程完工，并开始进行装修、钢结构雨棚等工程。2010年11月20日完成所有工程，正式交付给轨道运营公司。珠海北站作为一个中间站，原预计广珠城际开通时不承担列车始发或终到，但受位於[[拱北街道|拱北]]的[[珠海站|珠海站]]工程延误影响，广珠城轨把位于[[中山市|中山市]]与[[珠海市|珠海市]]交界的珠海北站作为临时终点站，于2011年1月7日与广珠城轨同时投入运营。

== 车站设计 ==
珠海北站由[[中铁二十五局|中铁二十五局]]承建，是一个[[高架车站|高架车站]]，[[站台|站台]]距地面高度为11.3米，采用[[钢筋混凝土|钢筋混凝土]]框架结构设计<ref>{{Cite web |url=http://search.zhnews.net/tqb/text.php?ud_key=43892&ud_date=2010-11-08 |title=珠海北站月底建成 西岸首条客运轨道专线直通珠海 |accessdate=2010-11-25 |publisher=珠海特區報 |date=2010-11-08 }}{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。车站总建筑面积达12万平方米，地面以上共分为两层，另设地下两层[[停车场|停车场]]，一楼地面为建筑面积为3653.4平方米的站厅层，设有站厅、售票中心和设备管理用房；二楼的站台层，站内设有四股[[铁轨|轨道]]，顶棚使用钢桁架结构，无柱雨棚结构投影面积达7500多平方米。车站外墙以[[玻璃幕墙|玻璃幕墙]]为主，除了保证站厅充分采光，旅客还能在二楼站台观看到[[凤凰山|凤凰山]]风景<ref name=zhtv>{{Cite web |url=http://www.zhtv.com/news/gbxw/201010/99948.html |title=城轨金鼎站月底完工 今年年底列车将开进珠海 |accessdate=2010-11-25 |publisher=珠海电台 |date=2010-10-13 |deadurl=yes |archiveurl=https://web.archive.org/web/20101017093951/http://www.zhtv.com/news/gbxw/201010/99948.html |archivedate=2010-10-17 }}</ref>。整个车站的基础都采用冲孔灌注桩，这种桩基础和大量的钢桁架结构设计令站點能抗8级[[地震|地震]]。在车站两边的楼梯旁边设有扶手电梯和升降电梯各两部，另方便残障人士乘车，设有无障碍通道<ref>{{Cite web |url=http://gcontent.oeeee.com/8/0a/80a8155eb153025e/Blog/e4e/73a510.html |title=城轨珠海北站20日完工 |accessdate=2010-11-25 |publisher=南方都市報珠海版 |date=2010-11-05 |deadurl=yes |archiveurl=https://web.archive.org/web/20160304091628/http://gcontent.oeeee.com/8/0a/80a8155eb153025e/Blog/e4e/73a510.html |archivedate=2016-03-04 }}</ref>。

珠海北站共设有4股轨道，包括2条正线及2条到发线，以及2个侧式站台。设计高峰小时客流量为1760人，属于小型车站规模，未来来往[[广州|广州]]和珠海的直达列车和长途列车以[[拱北|拱北]]的[[珠海站|珠海站]]为终点站，不停靠珠海北站。珠海北站成为广珠城轨开通初期临时终点站的主因，是因为站内设有4股轨道，并具备车辆调度始发的功能<ref>{{Cite web |url=http://gzdaily.dayoo.com/html/2010-03/22/content_904939.htm |title=广珠轻轨年底或通至珠海金鼎 |accessdate=2010-11-25 |publisher=广州日报 |date=2010-03-22 |deadurl=yes |archiveurl=https://web.archive.org/web/20100327100946/http://gzdaily.dayoo.com/html/2010-03/22/content_904939.htm |archivedate=2010-03-27 }}</ref>，而非[[明珠站|明珠站]]、[[前山站|前山站]]这样的无配线车站（仅有[[正线|正线]]，无[[到发线|到发线]]）<ref name=zhtv />。而另设到发线的主要原因，是为将来从珠海北站引出支线预留空间。而出于同样的原因，珠海北站的站台也与中山市境内几个车站不同，其站台与轨道之间没有设计安全门或安全围栏<ref name=zhtv />。

珠海北站在规划及建设初期稱為金鼎站。根据规定流程，广珠城轨公司在2010年中向沿线地方政府发出征询意见函，询问是否更改辖区内城轨站名，并把地方政府覆函中站名一起报送国家铁道部批覆<ref>{{Cite web |url=http://search.zhnews.net/zjwb/text.php?ud_key=37428&ud_date=2010-10-19 |title=城轨通车在即 珠海两站拟改“户口”金鼎站拟更名珠海北站金塘站拟更名唐家湾站 |accessdate=2010-11-25 |publisher=珠江晚报 |date=2010-10-19 }}{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。2010年10月中旬，金鼎站改称珠海北站<ref>{{Cite web |url=http://old.zhtv.com/Article/ShowArticle.asp?ArticleID=85583 |title=广珠城轨金鼎站更名 珠海北站月底完工 |accessdate=2010-11-29 |publisher=珠海電視台 |date=2010-10-13 |deadurl=yes |archiveurl=https://archive.is/20120714210822/http://old.zhtv.com/Article/ShowArticle.asp?ArticleID=85583 |archivedate=2012-07-14 }}</ref>。

== 交通配套 ==
珠海市委副书记[[钱芳莉|钱芳莉]]与常务副市长霍荣荫在2010年7月16日视察珠海北站施工现场时要求尽快寻找最有效、节约的交通解决方案，保证城轨通车后的交通接驳<ref>{{Cite web |url=http://search.zhnews.net/tqb/text.php?ud_key=29257&ud_date=2010-07-17 |title=广珠城轨广州南站至珠海金鼎站年底正式运营 |accessdate=2010-11-25 |publisher=珠海特區報 |date=2010-07-17 }}{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref>{{Cite web |url=http://search.zhnews.net/zjwb/text.php?ud_key=35174&ud_date=2010-09-29 |title=广珠城轨金鼎站将不再“有站无路” |accessdate=2010-11-25 |publisher=珠江晚报 |date=2010-09-29 }}{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

为了在广珠城轨首期通车时对珠海北站提供必要的交通配套，珠海北站的交通配套项目包括一条连接金峰北路的临时道路、一个面积为2100平方米的停车场、7000多平方米的绿化工程、公共卫生间等设施。其中，承担珠海北站临时接驳道路的金峰北路北围段，之前已被纳入高新区路网规划<ref>{{Cite web |url=http://search.zhnews.net/tqb/text.php?ud_key=39369&ud_date=2010-09-28 |title=广珠城轨金鼎站交通配套工程今招标 |accessdate=2010-11-25 |publisher=珠海特区报 |date=2010-09-28 }}{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。该临时道路从珠海北站引出，总长800多米，与中珠渠桥和金峰北路相连，为双向二车道设计，未来将扩建为双向四车道，成为永久性公路。
