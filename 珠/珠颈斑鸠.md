'''珠颈斑鸠'''（[[学名|学名]]：{{lang|la|''Spilopelia chinensis''}}），又名'''鸪雕'''、'''鸪鸟'''、'''中斑'''、'''花斑鸠'''、'''花脖斑鸠'''、'''珍珠鸠'''、'''斑颈鸠'''、'''珠颈鸽'''、'''斑甲(閩南語)'''、'''布鴣仔(客家語)'''  ，是分布在[[南亚|南亚]]、[[东南亚|东南亚]]地区以及[[中国|中国]]南方广大地区的一种常见的[[鸠鸽科|鸠鸽科]][[副斑鸠属|副斑鸠属]]物种。

== 特征 ==
珠頸斑鳩尾较長，體長在28－32公分，体重125－180克，比[[鴿子|鴿子]]略小。背部、翅膀和尾部為灰褐色，尾羽外側為黑色，尾端為白色，頭部為深藍色，頸後為黑色，帶有白色斑點，下腹部為暗紅色，喙為暗褐色，腳紅色。

人們常把它和[[鴿子|鴿子]]混淆，但只要由顏色和啄的構造來分辨就可以簡單看出兩者的差別。

[[File:斑鴿咧叫的聲.ogg|File:斑鴿咧叫的聲.ogg]]

發聲時頸部的羽毛會拱起，叫聲低沉，重音靠後，類似“咕-咕-咕”、“咕-咕-咕～咕”；驅趕入侵者或保護幼鳥時會發出“咕-咕”、“咕咕咕”。

== 習性 ==
[[File:Spotted_Dove_(Streptopelia_chinensis)_on_a_Kapok_(Ceiba_pentandra)_tree_in_Kolkata_W_IMG_3476.jpg|thumb]]（''Ceiba pentandra''）树上鸣叫的珠頸斑鳩，摄於[[印度|印度]][[西孟加拉邦|西孟加拉邦]][[加尔各答|加尔各答]]。]]
珠頸斑鳩的主食是果實、穀物和其他植物的種子，也會捕食昆蟲。主要在地面上覓食。有水時也會喝水，他們喝水的方式是俯身吸水，與其他鳥類不同。如果地點許可，它們也會用水清洗身體。它们是典型的陆生鸟类，一般在草地和农田中觅食。全年都会繁殖，常能在树木、建筑物的边沿甚至是地面上发现它们的巢穴。

珠頸斑鳩与同属其他斑鸠一样，一般不会群居，而是单独或成对出现。它们是温驯的鸟类，但是突然的噪声会使它们惊起。它们的飞行姿势与[[鳳頭鳩|鳳頭鳩]]类似，着陆时尾巴会上倾。

==分类==
依据其形态，本种过去一直被归类于[[斑鸠属|斑鸠属]]，但后来分子遗传学研究显示，本种实际上比其它斑鸠属成员更接近[[鸽属|鸽属]]，尤其接近[[粉鸽属|粉鸽属]]，而且[[棕斑鸠|棕斑鸠]]是本种的[[姐妹群|姐妹群]]，于是便将本种与棕斑鸠一起放在独立的[[副斑鸠属|副斑鸠属]]。

== 分布 ==
廣泛分佈在[[東南亞|東南亞]]至[[南亞|南亞]]的開闊林地、農地、都市等地區，從[[印度|印度]]、[[斯里蘭卡|斯里蘭卡]]東部到中國南部、東南亞等地。該種還被引入[[美國|美國]]的[[洛杉磯|洛杉磯]]地區，還於1860年代被引入[[澳大利亞|澳大利亞]]的[[墨爾本|墨爾本]]，以及[[印度尼西亞|印度尼西亞]]、[[紐西蘭|紐西蘭]]地區。

珠頸斑鳩有兩個亞種。'''[[指名亚种|指名亚种]]'''（''S. c. chinensis''）分布於[[中国|中国]][[華中|華中]]至[[華南|華南]]大部分地区、[[中南半島|中南半島]]北部以及[[台灣|台灣]]，'''南方亚种'''（''S. c. tigrina''）則分布於[[印度東北部|印度東北部]]、[[孟加拉|孟加拉]]、[[中南半島|中南半島]]南部、[[菲律賓|菲律賓]]、[[巽他群島|巽他群島]]。

为常见的一种[[留鸟|留鸟]]。

== 繁殖 ==
[[File:Spotted_Dove_Ic4-_Kolkata-_preening_&_sunning.jpg|left]]

求偶的雄性在表演时身体会极度倾斜，并在绕圈飞行时舒展自己的双翅和尾巴以吸引雌性。

珠颈斑鸠通常為一夫一妻制，一年繁殖一次，繁殖期为5－7月，通常由雄鳥尋找合適的地方，再帶雌鳥去選，選一個雙方都認為可以的地方，築巢是一起築的，用樹枝築巢；但也可能「鵲巢鳩佔」，佔用之前其他珠頸斑鳩或其他鳥類的巢。

珠颈斑鸠一次生兩顆白色的蛋，前後大約差一兩天。約兩星期後，小斑鳩就會同一天破蛋而出。孵蛋、養育小斑鳩則是由雄鸟和雌鸟輪流照料，喂食它们[[鸽乳|鸽乳]]。再过約兩星期，小斑鳩就必須離巢。
