'''御嶽山'''<ref name="tizu-kokudo">{{Cite web |url=http://watchizu.gsi.go.jp/watchizu.html?longitude=137.47451083333&latitude=35.894789055556 |title=地図閲覧サービス・御嶽山（岐阜県下呂市） |publisher=国土地理院 |accessdate=2013-02-25 |archive-url=https://web.archive.org/web/20130502192256/http://watchizu.gsi.go.jp/watchizu.html?longitude=137.47451083333&latitude=35.894789055556 |archive-date=2013-05-02 |dead-url=no }}</ref>（{{jpn|j=御嶽山|hg=おんたけさん|rm=Ontake san}}），又稱'''木曾御嶽山'''或'''御嶽'''，是位於[[日本|日本]][[長野縣|長野縣]][[木曾郡|木曾郡]][[木曾町|木曾町]]、[[王瀧村|王瀧村]]和[[岐阜縣|岐阜縣]][[下呂市|下呂市]]、[[高山市|高山市]]邊界的複合[[成層火山|成層火山]]，是日本47個監控的[[活火山|活火山]]之一，為山麓原野廣大的獨立峰<ref name="深田久弥 (1982)、227-230頁">[[#深田久弥|深田久弥 (1982)、227-230頁]]</ref><ref name="地質ニュースv306">{{Cite journal |date=1980 |url=http://www.gsj.jp/data/chishitsunews/80_02_01.pdf |title=御岳山1979年噴火 |author=曽屋龍典、近藤善教、下坂康哉 |format=PDF |journal=地質ニュース |volume=306 |issue=1980年2月号 |pages=6-13 |publisher=産業技術総合研究所 |accessdate=2013-02-13 |archive-url=https://web.archive.org/web/20141006233159/https://www.gsj.jp/data/chishitsunews/80_02_01.pdf |archive-date=2014-10-06 |dead-url=no }}</ref>。[[標高|標高]]3,067公尺<ref name="日本山名辞典 (1992)、116頁">[[#日本山名辞典|日本山名辞典 (1992)、116頁]]</ref><ref name="新日本山岳誌 (2005)、972-973頁">[[#新日本山岳誌|新日本山岳誌 (2005)、972-973頁]]</ref>，在日本活火山中的標高僅次於[[富士山|富士山]]。[[日本百名山|日本百名山]]之一。

== 概要 ==

[[File:Ontake_Volcano_Relief_Map,_SRTM-1.jpg|thumb]]
[[File:Mount_Ontake_(Morgenrot_zoom).JPG|thumb]]
御嶽山又稱木曾御嶽山、御嶽、王嶽<ref name="新日本山岳誌 (2005)、972-973頁" />、王御嶽<ref name="現代日本名山図会 (2003)、146-149頁">[[#現代日本名山図会|現代日本名山図会 (2003)、146-149頁]]</ref>，也以[[新字體|新字體]]寫作御-{岳}-山、或簡稱御岳<ref name="深田久弥 (1982)、227-230頁" />。是日本國內超過3000公尺的山岳中最西的一座，也是日本眾多同名山岳（御嶽山、御-{岳}-山）中的最高峰。山頂有[[三角點|一等三角點]]（3,063.<small>61</small>公尺，三角點名「御岳山」）<ref name="kijyun">{{Cite web |url=http://sokuseikagis1.gsi.go.jp/ |title=基準点成果等閲覧サービス |publisher=国土地理院 |accessdate=2015-02-14 |archive-url=https://web.archive.org/web/20150209192522/http://sokuseikagis1.gsi.go.jp/ |archive-date=2015-02-09 |dead-url=no }}</ref>與{{link-ja|御嶽神社|御嶽神社}}奥社。

自古以來便是山岳信仰的名山，是{{link-ja|木曾谷|木曽谷|木曾}}的代表山岳<ref name="日本三百名山 (1997)、148頁">[[#日本三百名山|日本三百名山 (1997)、148頁]]</ref>。[[東海地方|東海地方]]，特別是[[尾張|尾張]]地方，由於幾乎全境皆可看見其山容，因此將之稱為「木曾的御嶽山」<ref>[[#木曽のおんたけさん―その歴史と信仰|木曽のおんたけさん―その歴史と信仰(2009)、125頁]]</ref>。

御嶽山是[[日本百名山|日本百名山]]<ref name="深田久弥 (1982)、227-230頁" />、{{link-ja|新日本百名山|新日本百名山}}<ref>[[#新日本百名山|新日本百名山 (2006)、49-51頁]]</ref>、{{link-ja|花之百名山|花の百名山}}<ref name="花の百名山 (1997)、274-277頁">[[#花の百名山|花の百名山 (1997)、274-277頁]]</ref>、{{link-ja|岐阜百山|ぎふ百山}}<ref>[[#ぎふ百山|ぎふ百山 (1987)]]</ref>之一。而作為舊開田村代表山岳的飛驒頂上、舊三岳村代表山岳的{{link-ja|劍峰|剣ヶ峰}}則入選「信州故鄉120山」<ref>[[#信州ふるさと120山|信州ふるさと120山 (2011)]]</ref>。1927年（[[昭和|昭和]]2年），{{link-ja|大阪每日新聞社|大阪毎日新聞社}}與{{link-ja|東京日日新聞社|東京日日新聞社}}等將御嶽山選為{{link-ja|日本二十五勝|日本二十五勝}}之一。

御嶽山與[[木曾山脈|木曾山脈]]是木曾[[日本扁柏|扁柏]]生長地，林業興盛，因此未劃入[[國定公園|國定公園]]，僅被劃入長野縣的{{link-ja|御岳縣立自然公園|御岳県立自然公園}}<ref name="Nagano-shizenkouen">{{Cite web |url=https://www.pref.nagano.lg.jp/shizenhogo/kurashi/shizen/koen/syoukai.html |title=長野県の自然公園の紹介 |publisher=長野県 |accessdate=2016-11-17 |archive-url=https://web.archive.org/web/20161117145115/https://www.pref.nagano.lg.jp/shizenhogo/kurashi/shizen/koen/syoukai.html |archive-date=2016-11-17 |dead-url=no }}</ref>與岐阜縣的{{link-ja|御嶽山縣立自然公園|御嶽山県立自然公園}}<ref name="Gifu-shizenkouen">{{Cite web |url=http://www3.e-reikinet.jp/gifu-ken/houkishu/41190250027300000000/41190250027300000000/41190250027300000000.html |title=県立自然公園の指定（平成11年4月1日告示第273号） |publisher=岐阜県 |accessdate=2013-02-17 }}{{Dead link|date=2018年9月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。御嶽山腰森林茂密，多瀑布，是[[木曾川|木曾川]][[水系|水系]]的源頭。

在1979年（昭和54年）10月28日噴發以前，御嶽山曾被認為是[[死火山|死火山]]或[[休火山|休火山]]<ref name="地質ニュースv306" />。2008年（平成20年）3月31日，[[氣象廳|氣象廳]]發布[[火山噴發警戒等級|火山噴發警戒等級]]1（留意活火山）與火山噴發預報<ref name="御嶽山の火山活動解説資料">{{Cite web |url=http://www.data.jma.go.jp/svd/vois/data/tokyo/STOCK/monthly_v-act_doc/tokyo/13m01/312_13m01.pdf |format=PDF |title=御嶽山の火山活動解説資料（平成25 年1月） |publisher=気象庁 |date=2013-01 |accessdate=2013-02-12 |archive-url=https://web.archive.org/web/20140318135708/http://www.data.jma.go.jp/svd/vois/data/tokyo/STOCK/monthly_v-act_doc/tokyo/13m01/312_13m01.pdf |archive-date=2014-03-18 |dead-url=no }}</ref>。
2014年（平成26年）9月27日，噴發的[[火山碎屑流|火山碎屑流]]沿南側斜面流下，火山噴發警戒等級提升至3（入山管制）<ref name="jma_20140927_lv3">{{cite web|url=http://www.jma.go.jp/jp/volcano/forecast_03_20140927123646.html|title=噴火警報・予報： 御嶽山 平成26年9月27日12時36分|accessdate=2014/09/27|author=[[気象庁|気象庁]]地震火山部|archive-url=https://web.archive.org/web/20161209021109/http://www.jma.go.jp/jp/volcano/forecast_03_20140927123646.html|archive-date=2016-12-09|dead-url=no}}</ref>，火山口周邊4公里區域禁止進入。2015年6月，火山噴發警戒等級下調至2（火山口周邊管制）<ref name="jma_20150626_lv2">{{cite web|url=http://www.data.jma.go.jp/svd/vois/data/tokyo/STOCK/volinfo/VJ20150626170002_312.html|title=噴火警報・予報： 御嶽山 平成27年6月26日17時00分|accessdate=2017/10/10|author=[[気象庁|気象庁]]地震火山部|archive-url=https://web.archive.org/web/20170618090426/http://www.data.jma.go.jp/svd/vois/data/tokyo/STOCK/volinfo/VJ20150626170002_312.html|archive-date=2017-06-18|dead-url=no}}</ref>，禁止進入區域縮小至火山口周邊1公里。2017年8月，火山噴發警戒等級下調至1（留意活火山）<ref name="jma_20170821_lv1">{{cite web|url=http://www.data.jma.go.jp/svd/vois/data/tokyo/STOCK/volinfo/VJ20170821150000_312.html|title=噴火警報・予報： 御嶽山 平成29年8月21日17時00分|accessdate=2017/10/10|author=[[気象庁|気象庁]]地震火山部|archive-url=https://web.archive.org/web/20171012150310/http://www.data.jma.go.jp/svd/vois/data/tokyo/STOCK/volinfo/VJ20170821150000_312.html|archive-date=2017-10-12|dead-url=no}}</ref>。現在禁止進入區域維持在火山口周邊1公里，但依據「岐阜縣北阿爾卑斯地區及活火山地區山岳事故防止相關條例<ref name="Gifu Pref Regulations for Mountaineer">{{cite web|url=http://www.pref.gifu.lg.jp/kurashi/bosai/sangaku/11115/jourei.html|title=岐阜県北アルプス地区及び活火山地区における山岳遭難の防止に関する条例|accessdate=2017/10/10|author=岐阜県庁|archive-url=https://web.archive.org/web/20180902115819/http://www.pref.gifu.lg.jp/kurashi/bosai/sangaku/11115/jourei.html|archive-date=2018-09-02|dead-url=no}}</ref>」，若要進入御嶽山火山口周邊4公里以內的範圍，需提出登山申請書。

== 山名由來 ==
自古有王嶽藏王權現坐鎮<ref>[[#よみがえる木曽人の物語|よみがえる木曽人の物語 (1996)、117頁]]</ref>，修驗者尊稱本山為「王之御嶽」（おうのみたけ），簡稱「王嶽」（おうたけ）<ref name="新日本山岳誌 (2005)、972-973頁" /><ref>[[#御岳の信仰と登山の歴史|御岳の信仰と登山の歴史 (1988)、7頁]]</ref><ref name="いちどは行ってみたい日本の聖地(2010)、36-37頁">[[#いちどは行ってみたい日本の聖地|いちどは行ってみたい日本の聖地(2010)、36-37頁]]</ref><ref group="注釋">有一說是「王嶽」（おうたき）在室町時代中期演變為「おんたけ」。</ref>。之後演變為「御嶽」<ref name="hinoseiyaku">{{Cite web |url=http://hino-seiyaku.com/try/020.html |title=御嶽山整備事業 |publisher=日野製薬 |accessdate=2011-10-05 |archive-url=https://web.archive.org/web/20120224002833/http://hino-seiyaku.com/try/020.html |archive-date=2012-02-24 |dead-url=no }}</ref>。因此其念法「おんたけ」與源自[[山伏|修驗者]]總本山[[金峰山_(山梨縣和長野縣)|金峯山]]「金之御嶽」（かねのみたけ）的{{link-ja|甲斐駒岳|甲斐駒ヶ岳|甲斐御嶽}}、{{link-ja|御岳山 (東京都)|御岳山 (東京都)|武藏御嶽}}「みたけ」不同<ref>[[#御岳の信仰と登山の歴史|御岳の信仰と登山の歴史 (1988)、5頁]]</ref><ref name="花かおる御嶽山 (2000)、4頁">[[#花かおる御嶽山|花かおる御嶽山 (2000)、4頁]]</ref>。在日本全國眾多山岳中，有「山有[[富士山|富士]]、嶽有御嶽」之稱<ref name="いちどは行ってみたい日本の聖地(2010)、36-37頁" /><ref>[[#よみがえる木曽人の物語|よみがえる木曽人の物語 (1996)、118頁]]</ref>。

== 火山、地勢 ==
[[File:Mount_Ontake_from_Nomugitoge_ski_resort_2008-04-25.jpg|thumb]]
御嶽山是日本標高第14高的山，在火山中則僅次於[[富士山|富士山]]<ref group="注釋">第3高的火山是[[乘鞍岳|乘鞍岳]]（3,026公尺）。日本標高超過3000公尺的活火山只有富士山、御嶽山、乘鞍岳三座。</ref>。最高點劍峰屬長野縣，另有摩利支天山（2,959.2公尺）、繼子岳（2,858.9公尺）、繼母岳（2,867公尺） 等[[破火山口|外輪山]]，山頂部呈現[[梯形|梯形]]，南北長約3.5公里。北端的繼子岳是較新的山體，從北側山麓所見，由於看不見其他山峰，山體呈現如富士山的[[圓錐|圓錐]]形，因而被稱為「日和田富士」<ref>[[#ふるさとの富士200名山|ふるさとの富士200名山 (1996)、52頁]]</ref>。另外，長野縣側有[[寄生火山錐|寄生火山]]三笠山（2,256公尺）、小三笠山（2,029公尺）。王瀧口登山道至外輪山的會合處稱「王瀧頂上」（標高2,936公尺）<ref>{{Cite web |url=http://portal.cyberjapan.jp/site/mapuse4/?crs=1&lon=137.4825056&lat=35.88869167&z=16#crs=1&lon=137.48251&lat=35.88869&z=16&zoom=16&layers=BTTT |title=地理院地図（王滝頂上） |publisher=国土地理院 |accessdate=2014-09-27 |archive-url=https://web.archive.org/web/20141006100805/http://portal.cyberjapan.jp/site/mapuse4/?crs=1&lon=137.4825056&lat=35.88869167&z=16#crs=1&lon=137.48251&lat=35.88869&z=16&zoom=16&layers=BTTT |archive-date=2014-10-06 |dead-url=no }}</ref>，小坂口登山道至外輪山的會合處稱「飛驒頂上」（標高2,811 m）。

由於古時的噴發活動，[[火山灰|火山灰]]遍布周邊山麓原野，如長野縣側山麓斜坡耕地可見深色的火山灰。另外，屬長野縣的開田高原以[[蕎麥|蕎麥]]產地聞名<ref name="新日本山岳誌 (2005)、972-973頁" />。岐阜縣側地形較長野縣側複雜，平坦地少，脊線彎斜<ref name="新日本山岳誌 (2005)、972-973頁" />。

2007年（平成19年）5月10日，地質百選選定委員會將御嶽山選入「[[日本地質百選|日本地質百選]]」第1期（全國83處）<ref>[[#日本列島ジオサイト地質百選|日本列島ジオサイト地質百選 (2007)、90-91頁]]</ref>。

=== 火山活動 ===
御嶽山位居東日本火山帶的西端，基盤由古生層與[[中生代|中生代]]濃飛[[流紋岩|流紋岩]]類組成，基盤上是高1,400-1,900公尺、由[[橄欖石|橄欖石]]、複輝石、[[安山岩|安山岩]]等組成的成層火山<ref name="日本山名辞典 (1992)、116頁" /><ref name="地質ニュースv306" /><ref name="砂防と治水・第199巻">{{Cite journal |date=2011-03-20 |author=山岡耕春 |url=http://www.mlit.go.jp/common/001017612.pdf |title=日本の活火山(12) 御嶽山 |format=PDF |journal=砂防と治水 |issue=第199巻 |pages=109-110 |publisher=国土交通省 |accessdate=2016-11-17 |archive-url=https://web.archive.org/web/20160513234714/http://www.mlit.go.jp/common/001017612.pdf |archive-date=2016-05-13 |dead-url=no }}</ref>。過去噴發時，熔岩會往各方向流動，其中往西流的摩利支天山第6溶岩流遠最達約17公里。其末端是現今岐阜縣的安山岩大岩壁{{link-ja|巖立|巌立}}<ref name="ganndate">{{Cite web |url=https://www.pref.gifu.lg.jp/kyoiku/bunka/bunkazai/17768/tennen/ganndate.html |title=巌立 |publisher=岐阜県教育委員会社会教育文化課 |date=1957-12-19 |accessdate=2016-11-17 |archive-url=https://web.archive.org/web/20161117211237/https://www.pref.gifu.lg.jp/kyoiku/bunka/bunkazai/17768/tennen/ganndate.html |archive-date=2016-11-17 |dead-url=no }}</ref>。

以一之池為中心，連結摩利支天山、繼母岳、王瀧頂上的外輪山內側被認為是[[破火山口|破火山口]]。而在破火山口形成以前，御嶽山推測是高度可與[[富士山|富士山]]匹敵的成層火山。因火山大爆發而崩塌的土石形成[[土石流|土石流]]沿著河川流至岐阜縣[[各務原市|各務原市]]的{{link-ja|各務原台地|各務原台地}}。該次大噴發也形成劍峰、摩利支天山、繼母岳等山峰，使得御嶽山成為複成火山，其山容類似[[非洲|非洲]]的[[吉力馬札羅山|吉力馬札羅山]]<ref name="日本の名山 161-165頁">[[#日本の名山|日本の名山(1998)、161-165頁]]</ref><ref name="日本200名山 (1987)、166頁">[[#日本200名山|日本200名山 (1987)、166頁]]</ref>。
1970年代以前，一般認為最後的岩漿噴發是發生在約2萬年前，但在2006年（平成18年）岐阜縣的調査與2008年（平成20年）[[國土交通省|國土交通省]]多治見砂防國道事務所及{{link-ja|產業技術綜合研究所|産業技術総合研究所}}的調査發現，包含約5200年前爆發的火山碎屑流在內，御嶽山在兩萬年間共發生了四次（約1萬年前、約9000年前、約5200年前、約5000年前）岩漿噴發<ref name="砂防と治水・第199巻" /><ref name="kazanyochiren103">{{Cite web |url=http://vivaweb2.bosai.go.jp/v-hazard/sabo/53ontake_sabo.pdf |title=御嶽山火山噴火緊急減災対策砂防計画検討会 |publisher=御嶽山火山噴火緊急減災対策砂防計画検討会 |format=PDF |date=2011-07 |accessdate=2013-02-15 |archive-url=https://web.archive.org/web/20141006123624/http://vivaweb2.bosai.go.jp/v-hazard/sabo/53ontake_sabo.pdf |archive-date=2014-10-06 |dead-url=no }}</ref>。2007年（平成19年）4月30日，{{link-ja|信濃每日新聞|信濃毎日新聞}}報導指出，[[岐阜縣|岐阜縣]]在劍峰西北方6公里的下呂市小坂町發現約5200 - 6000年前的[[火山碎屑流|火山碎屑流]]所堆積的地層，並確認是五之池火山口的噴出物。

1979年的水蒸氣噴發六個月前，三之池出現池水混濁與冒泡現象。噴發六小時前，御嶽山發生火山口下方地震，被認為是噴發的前兆現象<ref name="ontake_kazan04">{{Cite web |url=http://www.cbr.mlit.go.jp/tajimi/sabo/ontake/data/ontake_kazan04.pdf |title=御嶽山火山防災だよりvol.4 |publisher=国土交通省中部地方整備局多治見砂防国道事務所 |date=2010-12 |accessdate=2013-02-15 |archive-url=https://web.archive.org/web/20130123082013/http://www.cbr.mlit.go.jp/tajimi/sabo/ontake/data/ontake_kazan04.pdf |archive-date=2013-01-23 |dead-url=no }}</ref>。1979年以後，御嶽山一直有（1991年、2007年）小規模噴氣活動<ref name="kazan" /><ref name="及川輝樹" />。2014年，氣象廳將其指定為「為了火山防災需充實監視、觀測體制的火山」<ref>{{Cite web |url=http://www.data.jma.go.jp/svd/vois/data/tokyo/STOCK/kaisetsu/CCPVE/hyoka/hyoka_houkoku.pdf |format=PDF |title=中長期的な噴火の可能性の評価について |publisher=火山噴火予知連絡会火山活動評価検討会 |pages=7 |date=2009-06 |accessdate=2013-02-12 |archive-url=https://web.archive.org/web/20141110035315/http://www.data.jma.go.jp/svd/vois/data/tokyo/STOCK/kaisetsu/CCPVE/hyoka/hyoka_houkoku.pdf |archive-date=2014-11-10 |dead-url=no }}</ref>，山頂周邊設有觀測火山活動的[[地震儀|地震儀]]、空氣振動測量儀、傾斜儀、火山氣體監測器、[[GPS|GPS]]觀測裝置、[[閉路電視|監視鏡頭]]等設備<ref>{{Cite web |url=http://www.cbr.mlit.go.jp/tajimi/sabo/ontake/data/ontake_kazan05.pdf |format=PDF |title=御嶽山火山防災だよりvol.5 |publisher=国土交通省中部地方整備局多治見砂防国道事務所 |date=2011-02 |accessdate=2011-03-24 |archive-url=https://web.archive.org/web/20120113005549/http://www.cbr.mlit.go.jp/tajimi/sabo/ontake/data/ontake_kazan05.pdf |archive-date=2012-01-13 |dead-url=no }}</ref><ref name="御岳山霊なる山の素顔 58頁">[[#御岳山霊なる山の素顔|御岳山霊なる山の素顔(2010)、58頁]]</ref>。2001年起，[[名古屋大學|名古屋大學]]大學院環境學研究科配合「岐阜、長野兩縣火山噴發警戒避難對策事業」進行御岳火山災害觀測<ref>{{Cite web |url=http://ngypub.seis.nagoya-u.ac.jp/INTRO/report/jishinyochiren/167_1.pdf |title=御嶽山近傍における地震活動の推移 |publisher=名古屋大学大学院環境学研究科 |format=PDF |date=2006-02-28 |accessdate=2016-11-17 }}{{Dead link|date=2018年9月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。2011年（平成23年）7月27日，「御嶽山火山噴火緊急減災對策砂防計畫檢討會」擬定御嶽山火山噴火緊急減災對策砂防計畫<ref name="kazanyochiren103" />。

==== 活火山定義 ====
1979年（昭和54年）水蒸氣爆發以前，眾多學者與一般民眾皆將御嶽山視為[[死火山|死火山]]。因此當時媒體將此次爆發宣傳為『死火山大爆發』<ref>[http://dx.doi.org/10.5026/jgeography.89.4_247 諏訪彰：1965〜79年の本邦火山活動と観測・研究の発展] 地学雑誌 Vol.89 (1980) No.4 P247-255</ref>。然而，19世紀前半的文獻就有記載實際的噴氣活動現象<ref name="及川輝樹" />。而氣象廳在1968年（昭和43年）發行的「火山觀測指針」 已將御嶽山（御岳山）列在63座活火山之一，1975年（昭和50年）發行的《日本活火山要覧》也將本山列入77座活火山之一<ref>{{PDFlink|[http://www.jma.go.jp/jma/kishou/books/kenshin/vol71p043.pdf 活火山カタログの改訂と火山活動度による活火山の分類（ランク分け）について]}} 林豊・宇平幸一　験震時報71巻　pp.43-57</ref>。

1979年，當時山上沒有建立穩定的觀測體制，因此無法觀測出明確的前兆。同年10月28日，水蒸氣噴發出約1,000公尺高的噴煙<ref name="kazan">{{Cite web |url=http://www.data.jma.go.jp/svd/vois/data/tokyo/312_Ontakesan/312_history.html |title=御嶽山 記録に残る火山活動 |publisher=気象庁 |date=2007-06-21 |accessdate=2011-03-24 |archive-url=https://web.archive.org/web/20140318150821/http://www.data.jma.go.jp/svd/vois/data/tokyo/312_Ontakesan/312_history.html |archive-date=2014-03-18 |dead-url=no }}</ref><ref name="気象研究所研究報告12">{{Cite journal |date=1984 |url=http://www.mri-jma.go.jp/Publish/Technical/DATA/VOL_12/12_172.pdf |title=御岳山の1979年噴火による降灰分布と川水のpH分布 |format=PDF |journal=気象研究所研究報告 |issue=第12号 |pages=172-178 |publisher=気象研究所 |accessdate=2011-03-24 |author= |archive-url=https://web.archive.org/web/20110713201330/http://www.mri-jma.go.jp/Publish/Technical/DATA/VOL_12/12_172.pdf |archive-date=2011-07-13 |dead-url=no }}</ref>。同日5點火山開始噴發，到下午兩點達到最大，之後逐漸減緩。本次噴發總量約二十多萬噸。噴煙延伸至東北方向的[[輕井澤町|輕井澤町]]與[[群馬縣|群馬縣]][[前橋市|前橋市]]一帶<ref name="地質ニュースv306" /><ref name="Ontakesan kazan">{{Cite web |url=http://www.data.jma.go.jp/svd/vois/data/tokyo/312_Ontakesan/312_index.html |title=火山・御嶽山 |publisher=気象庁 |accessdate=2013-02-12 |archive-url=https://web.archive.org/web/20140318083125/http://www.data.jma.go.jp/svd/vois/data/tokyo/312_Ontakesan/312_index.html |archive-date=2014-03-18 |dead-url=no }}</ref><ref name="気象研究所研究報告12" /><ref>[[#中部・近畿・中国の火山|中部・近畿・中国の火山 (2000)、59頁]]</ref>。

這次火山爆發促使日本對火山分類（死火山、[[休眠火山|休眠火山]]、活火山定義）的審查<ref name="地質ニュースv306" /><ref name="及川輝樹" />。日本現在已不再使用「活火山」以外的用語<ref>{{Cite web |url=http://www.data.jma.go.jp/svd/vois/data/tokyo/STOCK/kaisetsu/katsukazan_toha/katsukazan_toha.html |title=活火山とは |publisher=気象庁 |accessdate=2013-02-15 |archive-url=https://web.archive.org/web/20140318140628/http://www.data.jma.go.jp/svd/vois/data/tokyo/STOCK/kaisetsu/katsukazan_toha/katsukazan_toha.html |archive-date=2014-03-18 |dead-url=no }}</ref>。

; 休止期（約39萬年前至10萬年前）
約30萬年的活動休止期間，山體受到侵蝕，形成深谷與和村泥流。

; 新期御嶽火山（約10萬年前至2萬年前）

* 繼母岳火山群 - 約9萬年前噴發出遠至[[關東地方|關東地方]]的御嶽第1輕石層與鹽尻輕石層、Pm-I火山噴發碎屑<ref>{{Cite journal |date=1967 |url=https://www.jstage.jst.go.jp/article/geosoc1893/73/6/73_6_291/_article/-char/ja/ |doi=10.5575/geosoc.73.291 |title=御嶽火山第一浮石層:御嶽火山第一浮石層の研究 その1 |author=小林国夫ほか |accessdate=2017-12-25 |journal=地質学雑誌 |volume=73 |issue=6 |pages=291-308 |archive-url=https://web.archive.org/web/20171226130419/https://www.jstage.jst.go.jp/article/geosoc1893/73/6/73_6_291/_article/-char/ja/ |archive-date=2017-12-26 |dead-url=no }}</ref>。本次活動為[[普林尼式火山噴發|普林尼式火山噴發]]，使得古期御嶽火山中央形成直徑5-6公里的破火山口，並形成新谷熔岩層、湯之谷熔岩層、濁瀧火山碎屑流堆積物、三浦山熔岩層。總噴發量50 km<sup>3</sup>（30 km<sup>3</sup> DRE）。
* 摩利支天火山群 - 破火山口形成後的約8萬年前至約2萬年前活動，由摩利支天火山群的濁河火山、金剛堂火山、奧之院火山、草木谷火山、繼子岳火山、與大略呈南北方向的小火山群四之池火山、一之池火山、三之池火山等火山口噴發出安山岩質的熔岩、火山碎屑<ref name="ontake_kazan11">{{Cite web |url=http://www.cbr.mlit.go.jp/tajimi/sabo/ontake/data/ontake_kazan11.pdf |format=PDF |title=御嶽山火山防災だよりvol.11 |publisher=国土交通省中部地方整備局多治見砂防国道事務所 |date=2012-03 |accessdate=2013-02-15 |archive-url=https://web.archive.org/web/20130123081700/http://www.cbr.mlit.go.jp/tajimi/sabo/ontake/data/ontake_kazan11.pdf |archive-date=2013-01-23 |dead-url=no }}</ref>。噴出量以四之池火山口最多。火山群的噴發形成現在御嶽山最高點劍峰火山口丘與南北山頂群。
* 約6萬年前 - 噴發摩利支天熔岩。摩利支天第6熔岩沿著濁河川支流大俣川流動形成柱狀節理的「{{link-ja|巖立|巌立}}」（所在地為下呂市小坂町落合中崎平），並於1957年（昭和32年）指定為岐阜縣[[天然紀念物|天然紀念物]]<ref name="ganndate" />。
* 約5萬年前 - 大規模山體崩塌，土石由東側山麓西野川流向王瀧川、木曾川流域，最遠至約200公里下游的[[愛知縣|愛知縣]][[犬山市|犬山市]]。

; 有史以來

《御嶽山 地質與噴火記錄》記載774年與1892年曾有噴火活動，然而之後的研究認為這兩次噴發並不存在<ref name="及川輝樹">{{Cite journal |date=2008 |author=及川輝樹 |url=http://www.gsj.jp/data/bulletin/59_05_01.pdf |title=御岳火山の歴史噴火記録の再検討と噴気活動の歴史記録 |format=PDF |journal=地質調査報告 |issue=第59巻、第5/6号 |pages=203-210 |publisher=気象研究所 |accessdate=2011-03-24 |archive-url=https://web.archive.org/web/20141006132249/https://www.gsj.jp/data/bulletin/59_05_01.pdf |archive-date=2014-10-06 |dead-url=no }}</ref><ref>{{PDFlink|[http://www2.jpgu.org/meeting/2009/program/session/pdf/J237/J237-006.pdf 及川輝樹、奥野充：御嶽火山の最近の火山活動史]}} 日本地球惑星科学連合2009年大会予稿集 セッションJ237</ref>。

=== 長野縣西部地震 ===
[[File:Ontakesan_from_kohideyama.jpg|thumb]]

1927年（昭和4年），[[京都大學|京都大學]]地理學者{{link-ja|田中阿歌麿|田中阿歌麿}}進行湖沼調査<ref name="日本の名山 246頁">[[#日本の名山|日本の名山(1998)、246頁]]</ref><ref name="御岳の信仰と登山の歴史 (1988)、272頁" />，確認御嶽山有五座火山口湖，分別命名為一之池至五之池。呈現[[祖母綠|祖母綠]]色的二之池與三之池常年有水，其中二之池是日本最高的湖泊（2,905公尺），深度達3.5公尺<ref name="日本の山1000 (1992)、350-351頁" />。與{{link-ja|御庫裏池|ミクリガ池}}等飛驒山脈火水口湖相比，由於日夜溫度差造成融雪量差異，二之池的一日水位變化可達40公分左右<ref name="日本の名山 161-165頁" />。1979年火山噴發將大量硫磺帶入二之池，使得池水呈現[[酸性|酸性]]。現在，周邊山小屋利用湖水作為住宿者的洗澡水<ref name="tv-tokyo">{{Cite web |url=http://www.tv-tokyo.co.jp/sat/backnumber/110903.html |title=天空の宿へ 〜にっぽん山小屋物語〜 |publisher=テレビ東京土曜スペシャル |date=2011-09-03 |accessdate=2011-10-05 |archive-url=https://web.archive.org/web/20120111211324/http://www.tv-tokyo.co.jp/sat/backnumber/110903.html |archive-date=2012-01-11 |dead-url=no }}</ref>。三之池是御嶽山最大的池<ref name="岐阜県の山 (2009)、79-81頁">[[#岐阜県の山|岐阜県の山 (2009)、79-81頁]]</ref>，湖盆平均斜度14.4度、水深13.3公尺，8月初湖表平均水溫為9.75{{℃}}，低層為9.5{{℃}}<ref name="日本の名山 161-165頁" />。由於三之池畔祭祀{{link-ja|荒神|荒神}}與白龍王初春姬大神等神明<ref name="木曽のおんたけさん―その歴史と信仰(2009)、97頁">[[#木曽のおんたけさん―その歴史と信仰|木曽のおんたけさん―その歴史と信仰(2009)、97頁]]</ref>，因此三之池的湖水被信徒視為御神水（ごしんすい）<ref name="長野県の山 (1998)、88-89頁">[[#長野県の山|長野県の山 (1998)、88-89頁]]</ref><ref name="tisana tabi">{{Cite web |url=http://www.nhk.or.jp/kotabi/archive/2010/20100828.html |title=山の歌　祈りの峰 いまも 〜御嶽山〜 |publisher=[[NHK|NHK]]小さな旅 |date=2010-08-28 |accessdate=2016-11-17 |archive-url=https://web.archive.org/web/20160407212839/http://www.nhk.or.jp/kotabi/archive/2010/20100828.html |archive-date=2016-04-07 |dead-url=no }}</ref>，並與王瀧村御嶽神社里宮御神水選為「信州之名水、秘水」之一<ref>{{Cite web |url=http://www.pref.nagano.lg.jp/mizutaiki/kurashi/shizen/mizukankyo/kankyo/mesui.html |title=信州の名水・秘水一覧 |publisher=長野県環境部 |date=2013-08-27 |accessdate=2016-11-17 |archive-url=https://web.archive.org/web/20161117212628/http://www.pref.nagano.lg.jp/mizutaiki/kurashi/shizen/mizukankyo/kankyo/mesui.html |archive-date=2016-11-17 |dead-url=no }}</ref>。四之池是高地濕原，是[[高山植物|高山植物]]生長地。另外，二之池西北斜坡與賽之河原之間有個小窪地，在多雨期會形成小水池，稱為六之池。

=== 瀑布 ===
由於以御嶽山為源頭的河川有許多瀑布，因此御嶽山又被稱為瀑布之山<ref>{{Cite web |url=http://www.kankou-kiso.com/miru/takimeguri.html |title=御嶽山に登ろう・滝めぐりコース |publisher=木曽町観光協会 |accessdate=2011-03-24 |archiveurl=https://web.archive.org/web/20160304211757/http://www.kankou-kiso.com/miru/takimeguri.html |archivedate=2016-03-04 }}</ref>。這是因為御嶽山地形陡峭、高低差大，加上獨立峰山體廣大，降雨量多，茂盛的森林提供高度的保水能力，使得此地水源源源不絕。雖然有許多瀑布難以接近，但從黑澤口至油木尾根的遊步道可看見百間瀑布（西野川支流南俣川）、開田高原的尾之島瀑布、王瀧口瀑布、清瀧、{{link-ja|濁河溫泉|濁河温泉}}附近的仙人瀑布、緋之瀑布、入選[[日本瀑布百選|日本瀑布百選]]的{{link-ja|根尾瀑布|根尾の滝}}等。新瀧與清瀧是御嶽教的行場，新瀧由於有[[洞窟|洞窟]]，因此成為信眾斷食修行的場所<ref name="木曽のおんたけさん―その歴史と信仰(2009)、97頁" />。冬季時，新瀧（高度差約30公尺）與清瀧還會形成冰柱<ref>{{Cite news |title=厳冬氷の芸術 長野県王滝村の新滝 |page=1 | date=2013-02-20 |newspaper=[[中日新聞|中日新聞]] |location=[[愛知縣|愛知縣]][[名古屋市|名古屋市]]}}</ref>。黑澤口四合目的靈神場周邊有日之出瀑布、明榮瀑布、大祓瀑布、松尾瀑布等，是祭祀[[不動明王|不動明王]]等神靈的行場<ref name="木曽のおんたけさん―その歴史と信仰(2009)、107-108頁">[[#木曽のおんたけさん―その歴史と信仰|木曽のおんたけさん―その歴史と信仰(2009)、107-108頁]]</ref>。下呂市小坂町境內共有200座高低差達五公尺以上的瀑布<ref name="ontake_kazan04" />，不僅有專門網站介紹，並規劃了眾多觀賞路線供民眾選擇<ref name="osaka-taki">{{Cite web |url=http://www.osaka-taki.com/about/index.html |title=小坂の滝めぐり |publisher=小坂の滝めぐり |accessdate=2013-02-15 |archive-url=https://web.archive.org/web/20130509074652/http://www.osaka-taki.com/about/index.html |archive-date=2013-05-09 |dead-url=no }}</ref>。

=== 御嶽山與飛驒山脈 ===
[[File:Mount_Ontake_from_Mount_Norikura.jpg|thumb]]南部的[[乘鞍岳|乘鞍岳]]眺望御嶽山]]
御嶽山位在北阿爾卑斯（[[飛驒山脈|飛驒山脈]]）的延長線上，因此有人將御嶽山視為北阿爾卑斯的一部份，但由於與北側[[乘鞍岳|乘鞍岳]]之間並沒有稜線與山峰連結，一般將之視為不同山系。不過，兩山之間也沒有河川隔閡，因此至今仍未有定論。而在指南書中，可見將御嶽山與[[木曾山脈|中央阿爾卑斯]]一同介紹的情況<ref>[[#中央アルプス_御嶽山・白山|中央アルプス 御嶽山・白山 (2009)]]</ref>，因此有許多民眾將御嶽山視為中央阿爾卑斯的一部份，然而御嶽山與中央阿爾卑斯之間有木曾川分隔，很明顯的不屬同一山系。在[[國土地理院|國土地理院]]的日本主要山岳標高一覧中，到{{link-ja|鎌峰|鎌ヶ峰}}為止是列在「飛驒山脈南部」，御嶽山則是列在「御嶽山與其周邊」<ref name="kokudo" />。御嶽山與鎌峰之間的[[鞍部|鞍部]]為長峰峠（標高約1,350公尺）<ref name="tizu-kokudo" />。

== 歷史、信仰 ==
御嶽山是[[山岳信仰|山岳信仰]]的名山。日本一般將[[富士山|富士山]]、[[白山_(日本)|白山]]、[[立山_(日本)|立山]]列為[[日本三靈山|日本三靈山]]，但有時也以御嶽山取代白山或立山。在日本的山岳信仰史中，御嶽山的{{link-ja|御嶽講|御嶽講}}與富士山的{{link-ja|富士講|富士講}}同為日本庶民的重要信仰<ref>[[#御岳の信仰と登山の歴史|御岳の信仰と登山の歴史 (1988)、前書]]</ref><ref>[[#日本アルプスの登山と探検|日本アルプスの登山と探検 (1997)、285頁]]</ref>。御嶽山也是{{link-ja|教派神道|教派神道}}之一的{{link-ja|御嶽教|御嶽教}}信仰對象。最高點的劍峰上有祭祀[[大國主神|大己貴尊]]與[[惠比壽|惠比壽神]]的{{link-ja|御嶽神社|御嶽神社}}奧社。[[鎌倉時代|鎌倉時代]]，御嶽山一帶曾是修驗者的行場<ref name="新日本山岳誌 (2005)、972-973頁" />，但在之後衰退<ref name="御岳の信仰と登山の歴史 (1988)、95頁">[[#御岳の信仰と登山の歴史|御岳の信仰と登山の歴史 (1988)、95頁]]</ref>。[[室町時代|室町時代]]中期，神澤杜口的[[隨筆|隨筆]]《翁草》<ref>巻162：御山禅定は百日精進せずしては上り得ず、其間は行場に入りて修行をなす、昼夜光明真言を誦し、水垢離をとるなり。其の料金三両二分百日間の行用とす。斯くのごとくなれば軽賦の者は登り得ず生涯大切の旨願ならねば籠らずとなり。</ref>記載了參拜的情況。當時人們要前往山頂的御嶽神社奥社登拜時，需在山麓進行75日或100日嚴格的精進潔齋，結束後才可進行一年一次的上山登拜<ref name="日本三百名山 (1997)、148頁" /><ref name="御岳山霊なる山の素顔 12-14頁">[[#御岳山霊なる山の素顔|御岳山霊なる山の素顔(2010)、12-14頁]]</ref>。1560年（[[永祿|永祿]]3年）6月13日，[[木曾義昌|木曾義昌]]與隨從為了祈願，先在御嶽神社里宮進行100日的精進潔齋後登山參拜<ref>[[#御岳の信仰と登山の歴史|御岳の信仰と登山の歴史 (1988)、128頁]]</ref>。江戶時代前期，行腳僧[[圓空|圓空]]也曾來到御嶽山參拜，現在周邊[[寺院|寺院]]留有許多當時的[[雕刻|木雕]][[佛像|佛像]]<ref>[[#御岳の信仰と登山の歴史|御岳の信仰と登山の歴史 (1988)、132頁]]</ref>。1785年（天明5年），[[尾張國|尾張]]春日井郡出身的覺明行者與當地信徒打造登山口黑澤口的登拜道<ref name="現代日本名山図会 (2003)、146-149頁" />，使得登拜更為普及，也開始出現捨棄嚴格修行僅做{{link-ja|垢離|垢離|水行}}的參拜客<ref name="日本三百名山 (1997)、148頁" /><ref name="御岳の信仰と登山の歴史 (1988)、142頁">[[#御岳の信仰と登山の歴史|御岳の信仰と登山の歴史 (1988)、142頁]]</ref>。之後，普寛行者開設另一個登山口王瀧口。江戶時代，王瀧口、黑澤口與小坂口三個登山道的開通，使得御嶽教擴展至全國，成為大眾化的信仰名山<ref name="日本三百名山 (1997)、148頁" /><ref name="日本百名山地図帳(2005)、158-161頁">[[#日本百名山地図帳|日本百名山地図帳(2005)、158-161頁]]</ref>。江戶時代末期至明治初期，在御嶽講的熱潮下，每年有數十萬人前往登拜<ref name="日本の山1000 (1992)、350-351頁" />。江戶時代末期的《{{link-ja|信濃奇勝錄|信濃奇勝録}}》記有當時的參拜狀況<ref>信州一の大山なり、嶽の形大抵浅間に類して、清高これに過ぐ、毎年6月諸人潔斎して登る、福島より十里、全く富士山に登るが如し。</ref><ref name="日本の山1000 (1992)、350-351頁">[[#日本の山1000|日本の山1000 (1992)、350-351頁]]</ref>。1868年（明治元年），黑澤口8合目設置第一個[[山小屋|山小屋]]「女人堂」，並限制女人繼續向前登山<ref name="日本百名山地図帳(2005)、158-161頁" />，1872年，太政官通達宣布御嶽山解除女性禁令<ref name="新日本山岳誌 (2005)、972-973頁" />。

王瀧口與黑澤口的山道有許多靈場與修行場跡<ref name="日本山名辞典 (1992)、116頁" />。御嶽信仰有在自然石塊上刻印靈神（れいじん）名稱的習俗，稱為「靈神碑」<ref>[[#御岳の信仰と登山の歴史|御岳の信仰と登山の歴史 (1988)、84頁]]</ref>。現在黑澤口參道有大約5,000座靈神碑<ref name="日本200名山 (1987)、166頁" />。另外，距離御嶽山較遠的{{link-ja|鳥居峠 (長野縣)|鳥居峠 (長野県)|鳥居峠}}與{{link-ja|和田峠 (長野縣)|和田峠 (長野県)|和田峠}}等遙拜所，可見到御嶽信仰的石碑與祠堂<ref name="新日本山岳誌 (2005)、972-973頁" />。江戶時代後期的繪師[[谷文晁|谷文晁]]在《日本名山圖會》中，將御嶽山列為名山介紹<ref name="現代日本名山図会 (2003)、146-149頁" />。隨著林道黑石線與白崩林道等[[收費道路|收費道路]]與{{link-ja|御岳纜車|御岳ロープウェイスキー場}}的通車與啟用，吸引大量登山客來到御嶽山<ref>[[#木曽のおんたけさん―その歴史と信仰|木曽のおんたけさん―その歴史と信仰(2009)、2頁]]</ref>。1985年（昭和60年）起，山腰陸續開設四座滑雪場。

=== 御嶽神社 ===
御嶽神社祭祀稱為御嶽大神的[[國之常立神|國常立尊]]、[[大國主神|大己貴命]]、[[少彥名|少彥名命]]<ref>[[#木曽のおんたけさん―その歴史と信仰|木曽のおんたけさん―その歴史と信仰(2009)、80頁]]</ref>。王瀧口設有里宮，黑澤口設有里宮與若宮，{{link-ja|木曾御嶽神社王瀧口|木曽御嶽神社王滝口}}的奧社在王瀧頂上，木曾御嶽神社黑澤口的奥社在山頂的劍峰<ref>[[#木曽のおんたけさん―その歴史と信仰|木曽のおんたけさん―その歴史と信仰(2009)、111頁]]</ref>。1882年（明治15年）6月，小谷分喜出版《御嶽神社社略緣起》<ref name="御岳の信仰と登山の歴史 (1988)、271頁">[[#御岳の信仰と登山の歴史|御岳の信仰と登山の歴史 (1988)、271頁]]</ref>。1944年（昭和22年），御嶽教等教團與御嶽神社成立「木曾御嶽山奉贊會」，之後更名為「御嶽山奉贊會」，負責[[神社|神社]]的營運<ref>[[#木曽のおんたけさん―その歴史と信仰|木曽のおんたけさん―その歴史と信仰(2009)、91頁]]</ref>。1984年（昭和59年）9月14日，御嶽山發生地震（長野縣西部地震），一合目里宮的拜殿與末社半毀，而作為行場的清瀧，其瀑潭遭到土石掩埋<ref>[[#御岳山霊なる山の素顔|御岳山霊なる山の素顔(2010)、47頁]]</ref>。御嶽神社黑澤口有[[神樂|太太神樂]]獻演活動。

=== 御嶽的四門 ===
從遠方前來登拜御嶽時，最早能遠望御嶽山的地點稱為「御嶽的四門」<ref name="よみがえる木曽人の物語 (1996)、70頁">[[#よみがえる木曽人の物語|よみがえる木曽人の物語 (1996)、70頁]]</ref>，設有鳥居等作為御嶽山的遙拜所<ref>松平君山『御嶽縁起』</ref><ref name="御岳の信仰と登山の歴史 (1988)、62-66頁">[[#御岳の信仰と登山の歴史|御岳の信仰と登山の歴史 (1988)、62-66頁]]</ref>。此四門對應[[佛教|佛教]]四門，分別為岩鄉村神戶的發心門、長峰峠的菩薩門、三浦山中的修行門、鳥居峠的涅槃門<ref>[[#木曽御嶽信仰|木曽御嶽信仰 (2002)、130頁]]</ref>。另外，木曾福島往黑澤口的古道，在合戶峠與姥神峠等地也設有御嶽遙拜所<ref name="山と高原地図 (1993)、地図表面">[[#山と高原地図1993年版|山と高原地図 (1993)、地図表面]]</ref><ref>[[#よみがえる木曽人の物語|よみがえる木曽人の物語 (1996)、258頁]]</ref>。
* 岩鄉村神戶 - 從[[京都|京都]]方向沿[[中山道|中山道]]前來的第一個能遙望御嶽山的地點<ref name="御岳の信仰と登山の歴史 (1988)、62-66頁" />，在現在的木曾町。御嶽山頂東南東方19.8公里。
* 長峰峠 - 位於與鎌峰之間的鞍部，空明行者在此設立御嶽大權現石碑<ref name="よみがえる木曽人の物語 (1996)、70頁" /><ref name="御岳の信仰と登山の歴史 (1988)、62-66頁" />。御嶽山頂北北東方10.4公里。
* 三浦山中 - 因戰亂而破壞<ref name="御岳の信仰と登山の歴史 (1988)、62-66頁" />。位於阿寺山地的拜殿山。御嶽山頂西南方17.2公里。
* {{link-ja|鳥居峠 (長野縣)|鳥居峠 (長野県)|鳥居峠}} - 位於中山道。{{link-ja|木曾義元|木曾義元}}祈求戰勝在此建造遙拜御嶽的鳥居。此地有一心行者的石像。御嶽山頂東北東方29.2公里。

=== 宣揚御嶽信仰的行者 ===
==== 覺明行者 ====
[[File:Kakumei_gyoja.JPG|thumb]]
'''覺明行者'''（{{jpn|j=覚明行者|hg=かくめいぎょうしゃ}}）生於1718年（[[享保|享保]]3年）3月3日，是[[尾張國|尾張國]]春日井郡牛山村<!--（田楽村）-->皿屋敷的農夫丹羽清兵衛（左衛門）與千代之子，[[幼名|幼名]]源助，後改名為仁右五衛門。小時生活在新川村土器野新田的農家<ref name="御岳の信仰と登山の歴史 (1988)、15頁">[[#御岳の信仰と登山の歴史|御岳の信仰と登山の歴史 (1988)、15頁]]</ref><ref name="木曽のおんたけさん―その歴史と信仰(2009)、27-28頁">[[#木曽のおんたけさん―その歴史と信仰|木曽のおんたけさん―その歴史と信仰(2009)、27-28頁]]</ref>。1818年（[[文政|文政]]元年）10月的《連城亭隨筆》記載覺明行者在結婚後開設餅店<ref name="木曽のおんたけさん―その歴史と信仰(2009)、27-28頁" /><ref group="注釋">一說是賣魚。</ref>。之後，覺明行者成為在各地[[巡禮|巡禮]]修行的[[行者_(佛教術語)|行者]]<ref name="木曽のおんたけさん―その歴史と信仰(2009)、27-28頁" />，並在木曾谷各村進行布教<ref name="よみがえる木曽人の物語 (1996)、125頁">[[#よみがえる木曽人の物語|よみがえる木曽人の物語 (1996)、125頁]]</ref>。1782年（[[天明|天明]]2年），覺明行者向管轄御嶽山的[[神職|神職]]武居家與尾張藩木曾{{link-ja|代官|代官}}山村氏提出登山許可，但認為會破壞維持數百年的登拜型式（精進潔齋）而拒絕<ref name="木曽のおんたけさん―その歴史と信仰(2009)、30-33頁">[[#木曽のおんたけさん―その歴史と信仰|木曽のおんたけさん―その歴史と信仰(2009)、30-33頁]]</ref>。1785年（天明5年），在未取得登山許可的狀況下，覺明行者於6月8日帶領八名地方居民、6月14日帶領39名尾張信眾、6月28日帶領約80名信眾登拜<ref name="木曽のおんたけさん―その歴史と信仰(2009)、30-33頁" />，因此被拘留21天<ref name="木曽のおんたけさん―その歴史と信仰(2009)、30-33頁" />。1786年（[[天明|天明]]6年），覺明行者帶著眾多同志登拜並整修黑澤的登山道，但在整修途中於6月20日<ref group="注釋">一說是7月23日。</ref>病倒二之池畔，之後葬在黑澤口九合目的覺明堂宿舍岩場<ref name="木曽のおんたけさん―その歴史と信仰(2009)、30-33頁" /><ref name="御岳の信仰と登山の歴史 (1988)、134-143頁">[[#御岳の信仰と登山の歴史|御岳の信仰と登山の歴史 (1988)、134-143頁]]</ref>。現在山小屋「覺明堂」旁有覺明行者的[[靈場|靈場]]。之後，繼承覺明行者意志的信徒完成了黑澤口登山道的整修。隨著信眾的增加，也帶動了{{link-ja|福島宿 (中山道)|福島宿 (中山道)|福島宿}}的發展。1791年（寛政3年）6月，山麓{{link-ja|庄屋|庄屋}}一同向武居家提出輕精進登拜，並於1792年（寛政4年）1月1日取得許可<ref name="木曽のおんたけさん―その歴史と信仰(2009)、30-33頁" /><ref name="御岳の信仰と登山の歴史 (1988)、134-143頁" /><ref name="よみがえる木曽人の物語 (1996)、125頁" />。1850年（[[嘉永|嘉永]]3年），上野東叡山日光御門主授予菩薩號給覺明行者<ref name="御岳の信仰と登山の歴史 (1988)、270頁">[[#御岳の信仰と登山の歴史|御岳の信仰と登山の歴史 (1988)、270頁]]</ref>。

此外，覺明行者在山麓的開田西野教導村人種植，成為當地地名「開田」的由來。1806年（文化3年），此地設立稗田石碑作為紀念<ref name="よみがえる木曽人の物語 (1996)、125頁" />。

覺明行者被視為是御嶽山[[中興|中興]]開山的先驅者<ref>[[#木曽のおんたけさん―その歴史と信仰|木曽のおんたけさん―その歴史と信仰(2009)、25頁]]</ref>。

==== 普寛行者 ====
'''普寛行者'''（{{jpn|j=普寛行者|hg=ふかんぎょうしゃ}}）在1731年（享保16年）於[[武藏國|武藏國]][[秩父郡|秩父郡]]大瀧村落合出生（本名本明院普寛），傳聞青少年時期仕於酒井[[雅樂_(日本)|雅樂頭]]家<ref name="木曽のおんたけさん―その歴史と信仰(2009)、40-44頁">[[#木曽のおんたけさん―その歴史と信仰|木曽のおんたけさん―その歴史と信仰(2009)、40-44頁]]</ref>。1764年（[[明和|明和]]元年）於{{link-ja|三峯神社|三峯神社}}入門，成為本山派的修驗者<ref name="木曽のおんたけさん―その歴史と信仰(2009)、40-44頁" />。1792年（寛政4年）5月，普寛行者帶領江戶等地的信眾前往御嶽山，6月8日入山並在各地進行御座（おざ）<ref group="注釋">「御座」是普寛行者以普及的巫術所進行的祈禱儀式。</ref>，6月10日開闢王瀧口<ref name="木曽のおんたけさん―その歴史と信仰(2009)、40-44頁" />。之後，在江戶一帶組織御嶽講，宣揚御嶽信仰<ref name="木曽のおんたけさん―その歴史と信仰(2009)、40-44頁" />。1794年（寛政6年）在[[上野國|上州]]{{link-ja|武尊山|武尊山}}、1795年（寛政7年）在{{link-ja|八海山|八海山}}進行開山活動，1801年（[[享和|享和]]元年）9月病倒於武州{{link-ja|本庄宿|本庄宿}}<ref>[[#木曽のおんたけさん―その歴史と信仰|木曽のおんたけさん―その歴史と信仰(2009)、47頁]]</ref><ref name="御岳の信仰と登山の歴史 (1988)、267頁" />。現在王瀧口3合目的清瀧有普寛行者的墓塔<ref name="木曽のおんたけさん―その歴史と信仰(2009)、108-110頁">[[#木曽のおんたけさん―その歴史と信仰|木曽のおんたけさん―その歴史と信仰(2009)、108-110頁]]</ref>。1850年（嘉永3年），上野東叡山日光御門主授予普寛行者菩薩號<ref name="御岳の信仰と登山の歴史 (1988)、270頁" />。1890年（明治23年），王瀧村舉行普寛行者百年祭，並設立紀念碑<ref name="御岳の信仰と登山の歴史 (1988)、272頁">[[#御岳の信仰と登山の歴史|御岳の信仰と登山の歴史 (1988)、272頁]]</ref>。普寛行者的直系弟子如廣山行者、泰賢行者、順明行者等人，及之後眾多行者持續宣揚御嶽講<ref>[[#御岳の信仰と登山の歴史|御岳の信仰と登山の歴史 (1988)、182-193頁]]</ref>。

=== 御嶽講與參拜 ===
最初，普寛行者與其後人在江戶等關東地方開設普寛行者體系的御嶽講社<ref name="花かおる御嶽山 (2000)、4頁" />。之後，普寛行者弟子儀覺行者（きかくぎょうじゃ、1769-1841年）在東海地方開設宮丸講。覺明行者體系的講社則以愛知縣為中心向西日本擴展<ref>[[#木曽のおんたけさん―その歴史と信仰|木曽のおんたけさん―その歴史と信仰(2009)、127-131頁]]</ref>。其中，濃尾平野的農民將作為木曾川源頭的御嶽山視為[[水分神|水分神]]之山<ref name="よみがえる木曽人の物語 (1996)、119頁">[[#よみがえる木曽人の物語|よみがえる木曽人の物語 (1996)、119頁]]</ref>。而木曾谷地域也有許多普寛行者系的講社<ref>[[#御岳の信仰と登山の歴史|御岳の信仰と登山の歴史 (1988)、19-22頁]]</ref>。御嶽信仰相信，各講的先達（指導者）靈魂將作為靈神附於[[石碑|碑]]上（靈神碑）鎮守於御嶽山登拜道<ref name="いちどは行ってみたい日本の聖地(2010)、36-37頁" />，也就是「死後我的御靈將回歸山中」<ref>[[#よみがえる木曽人の物語|よみがえる木曽人の物語 (1996)、76頁]]</ref>。

在過去，御嶽山的登拜有一種形式是由行者帶領信徒一起進行聖地巡禮之旅（御嶽參拜<ref name="八ヶ岳・御岳と中央アルプス(1983)、164-165頁">[[#八ヶ岳・御岳と中央アルプス|八ヶ岳・御岳と中央アルプス(1983)、164-165頁]]</ref>）<ref>[[#御岳の信仰と登山の歴史|御岳の信仰と登山の歴史 (1988)、113頁]]</ref>，而講者在此多做為引導者帶領團隊前進。

江戶時代開始，關東與尾張等地的民眾多利用[[中山道|中山道]]前往御嶽山，1919年（大正8年）[[中央本線|中央本線]]全線開通後逐漸改從[[木曾福島站|木曾福島站]]開始登拜<ref name="木曽のおんたけさん―その歴史と信仰(2009)、72頁">[[#木曽のおんたけさん―その歴史と信仰|木曽のおんたけさん―その歴史と信仰(2009)、72頁]]</ref>。1923年（大正12年）木曾森林鐵道鋪設後，木曾福島站可搭乘{{link-ja|御嶽交通|おんたけ交通}}的巴士前往黑澤與王瀧<ref name="木曽のおんたけさん―その歴史と信仰(2009)、72頁" />。1966年（昭和41年），收費道路林道黑石線全線開通，可搭乘巴士直接到達王瀧口的田之原，1971年（昭和46年），收費道路白崩林道全線開通，可搭乘巴士直接到達黑澤口的中之湯<ref>[[#木曽のおんたけさん―その歴史と信仰|木曽のおんたけさん―その歴史と信仰(2009)、74頁]]</ref>。現在，在天氣穩定的7月下旬至8月中旬，民眾常選擇兩天一夜或三天兩夜的登拜活動<ref name="御岳の信仰と登山の歴史 (1988)、115頁">[[#御岳の信仰と登山の歴史|御岳の信仰と登山の歴史 (1988)、115頁]]</ref>，主要路線有從黑澤口經8合目的女人堂至山頂後折返下山、從黑澤口經山頂至王瀧口下山等路線，或是從王瀧口至山頂後折返下山、或是從王瀧口經8合目的女人堂至黑澤口下山的路線<ref name="御岳の信仰と登山の歴史 (1988)、115頁" />。

== 環境 ==
1757年（[[寶曆|寶曆]]7年），松平君山的《吉蘇志畧》記載了御嶽山的環境<ref>その東の峰に三つの池有り、一の池は水涸る、一の池は水少し、一の池は水満る、その水は流れて西のに至る。この北を地獄谷と云う硫黄多し、その水流れて王滝に至る、濁川と云う、往々硫黄を取得す、その水甚だ臭し</ref>與[[偃松|偃松]]、駒草、御蓼、[[白鼬|白鼬]]等動植物，並有[[星鴉|星鴉]]、[[岩雷鳥|岩雷鳥]]等鳥類的素描<ref>[[#御岳の信仰と登山の歴史|御岳の信仰と登山の歴史 (1988)、203-204頁]]</ref>。標高1,500m以下的山麓可見[[日本扁柏|日本扁柏]]、[[羅漢柏|羅漢柏]]等{{link-ja|木曾五木|木曽五木}}<ref name="新日本山岳誌 (2005)、972-973頁" />。現在，王瀧口登山道的田之原天然公園周邊約830公頃的土地已被指定為木曾御岳自然休養林<ref name="tanohara" />。而1836年（[[天保|天保]]7年）在西山麓的下呂市小坂町赤沼田所種植的日本扁柏人工林也在1993年（平成5年）由{{link-ja|林野廳|林野庁}}指定為「赤沼田天保日本扁柏植物群落保護林」<ref>{{Cite web |url=http://www.rinya.maff.go.jp/chubu/gijyutu/gyomu/sikengero/gero29.html |title=赤沼田（あかんた）ヒノキ人工林成長試験地 |publisher=林野庁中部森林管理局 |accessdate=2013-01-31 |archive-url=https://web.archive.org/web/20130201220007/http://www.rinya.maff.go.jp/chubu/gijyutu/gyomu/sikengero/gero29.html |archive-date=2013-02-01 |dead-url=no }}</ref>。
{{link-ja|田中澄江|田中澄江}}在著作《花之百名山》』中將[[北極花|北極花]]列為御嶽山的代表花<ref name="花の百名山 (1997)、274-277頁" />。另外，木曾町三岳也生產被稱為「御嶽黑光真石」的安山岩，是御嶽信仰的靈神碑材料。

=== 動物 ===
3萬年前的[[日本舊石器時代|舊石器時代]]，御嶽山有大角鹿棲息，而東山麓的開田高原則是大角鹿的狩獵場<ref name="よみがえる木曽人の物語 (1996)、12頁">[[#よみがえる木曽人の物語|よみがえる木曽人の物語 (1996)、12頁]]</ref>。另外，在開田高原西野的柳又遺跡曾挖掘出石器與土器<ref name="よみがえる木曽人の物語 (1996)、12頁" />。

山頂附近的登山道可見生活在高山地帶的[[星鴉|星鴉]]、[[岩雷鳥|岩雷鳥]]、[[孔雀蛺蝶|孔雀蛺蝶]]。其中，岩雷鳥（雷鳥）已被列為日本特別天然紀念物<ref>{{Cite web |url=http://bunka.nii.ac.jp/db/heritages/detail/162818 |title=文化遺産データベース「らいちょう」 |publisher=文化庁 |accessdate=2016-11-17 |archive-url=https://web.archive.org/web/20161117212044/http://bunka.nii.ac.jp/db/heritages/detail/162818 |archive-date=2016-11-17 |dead-url=no }}</ref>，環境省將其列為[[國際自然保護聯盟瀕危物種紅色名錄|紅色名錄]]極危物種（絕滅危懼IA類）<ref>{{Cite web |url=http://www.env.go.jp/press/file_view.php?serial=20551&hou_id=15619 |title=【鳥類】環境省第4次レッドリスト（2012）＜分類群順＞ |format=PDF |publisher=環境省 |date=2012-08-28 |accessdate=2013-01-31 |archive-url=https://web.archive.org/web/20121102134405/http://www.env.go.jp/press/file_view.php?serial=20551&hou_id=15619 |archive-date=2012-11-02 |dead-url=no }}</ref>，岐阜縣將其列為絕滅危懼I類、長野縣則列為絕滅危懼II類<ref>{{Cite web |url=http://jpnrdb.com/search.php?mode=map&q=02080010130 |title=日本のレッドデータ検索システム「ライチョウ」 |publisher=（エンビジョン環境保全事務局） |accessdate=2016-11-17 |archive-url=https://web.archive.org/web/20161117213438/http://jpnrdb.com/search.php?mode=map&q=02080010130 |archive-date=2016-11-17 |dead-url=no }} - 「都道府県指定状況を一覧表で表示」をクリックすると、出典元の各都道府県のレッドデータブックのカテゴリー名が一覧表示される。</ref>。2008年的調査發現棲息地從1981年的50處減少至28處<ref name="御岳山霊なる山の素顔 119頁">[[#御岳山霊なる山の素顔|御岳山霊なる山の素顔(2010)、119頁]]</ref>。現在，山中棲息的動物有[[鼬|鼬]]、[[野豬|野豬]]、[[貉|貉]]、[[亞洲黑熊|亞洲黑熊]]、[[日本獼猴|日本獼猴]]、[[赤狐|日本赤狐]]，以及[[花雀|花雀]]、[[桑鳲|桑鳲]]、[[綠雉|綠雉]]、[[旋木雀|旋木雀]]<ref>{{Cite web |url=http://www.pref.gifu.lg.jp/kurashi/kankyo/shizenhogo/c11265/kibasiri.html |title=キバシリ |publisher=岐阜県 |accessdate=2016-11-17 |archive-url=https://web.archive.org/web/20161117211519/http://www.pref.gifu.lg.jp/kurashi/kankyo/shizenhogo/c11265/kibasiri.html |archive-date=2016-11-17 |dead-url=no }}</ref>、[[煤山雀|煤山雀]]、[[佛法僧|佛法僧]]等[[鳥類|鳥類]]<ref name="日本山名辞典 (1992)、116頁" />。東南山麓的長野縣木曾郡木曾町的「三岳佛法僧繁殖地」已被指定為國家天然紀念物<ref>{{Cite web |url=http://bunka.nii.ac.jp/db/heritages/detail/161218 |title=文化遺産データベース「三岳のブッポウソウ繁殖地」 |publisher=文化庁 |accessdate=2016-11-17 |archive-url=https://web.archive.org/web/20161117211908/http://bunka.nii.ac.jp/db/heritages/detail/161218 |archive-date=2016-11-17 |dead-url=no }}</ref>。
開田高原是{{link-ja|木曾馬|木曽馬}}的飼育地，已被列為長野縣天然紀念物<ref>{{Cite web |url=https://www.pref.nagano.lg.jp/kyoiku/bunsho/bunka/rekishi/documents/15ken-tennenkinenbutu.pdf |title=【長野県】県指定等文化財　15 県天然記念物 |format=PDF |publisher=長野県 |accessdate=2016-11-17 |archive-url=https://web.archive.org/web/20160921132841/http://www.pref.nagano.lg.jp/kyoiku/bunsho/bunka/rekishi/documents/15ken-tennenkinenbutu.pdf |archive-date=2016-09-21 |dead-url=no }}</ref>。

[[鱗翅目|鱗翅目]][[夜蛾科|夜蛾科]]的御嶽夜蛾（''Apamea ontakensis'' Sugi）是僅生活在御嶽山高山地帶的高山蛾<ref group="注釋">岐阜縣列為絕滅危懼I類。</ref><ref>{{Cite web |url=http://www.pref.gifu.lg.jp/kurashi/kankyo/shizenhogo/c11265/ontakekuroyotou.html |title=オンタケクロヨトウ |publisher=岐阜県 |accessdate=2016-11-17 |archive-url=https://web.archive.org/web/20161117212034/http://www.pref.gifu.lg.jp/kurashi/kankyo/shizenhogo/c11265/ontakekuroyotou.html |archive-date=2016-11-17 |dead-url=no }}</ref>。而只棲息在[[本州|本州]]山岳地帶的鱗翅目[[尺蛾科|尺蛾科]]內城尺蛾（''Dysstroma truncata fusconebulosa'' Inoue）已確認在岐阜縣內僅棲息在御嶽山<ref group="注釋">岐阜縣列為準絕滅危懼。</ref><ref>{{Cite web |url=http://www.pref.gifu.lg.jp/kurashi/kankyo/shizenhogo/c11265/utizironamisyaku.html |title=ウチジロナミシャク |publisher=岐阜県 |accessdate=2016-11-17 |archive-url=https://web.archive.org/web/20161117212446/http://www.pref.gifu.lg.jp/kurashi/kankyo/shizenhogo/c11265/utizironamisyaku.html |archive-date=2016-11-17 |dead-url=no }}</ref>。

江戶時代末期，御嶽山的高山植物被稱為珍貴的「御神草」，生長於山頂的{{link-ja|駒草|コマクサ}}因此大量消失<ref name="花かおる御嶽山 (2000)、58頁">[[#花かおる御嶽山|花かおる御嶽山 (2000)、58頁]]</ref><ref group="注釋">東鄰的{{link-ja|木曾駒岳|木曽駒ヶ岳}}也因御神草採集而滅絕。</ref>。

[[明治時代|明治時代]]，知名[[植物學|植物學者]]{{link-ja|白井光太郎|白井光太郎}}與{{link-ja|三好學|三好学}}分別在1887年（明治20年）7月與1889年（明治22年）登頂採集高山植物<ref name="御岳の信仰と登山の歴史 (1988)、271頁" />。1933年（昭和8年），植物學者中野治房前往御嶽山進行植物生態調査<ref name="日本の名山 246頁" /><ref name="御岳の信仰と登山の歴史 (1988)、272頁" />。

御嶽山在[[林木線|林木線]]以上的高山帶有{{link-ja|青栂櫻|アオノツガザクラ}}、{{link-ja|岩梅|イワウメ}}、{{link-ja|裏白七竃|ウラジロナナカマド}}、{{link-ja|大瓢簞木|オオヒョウタンボク}}、{{link-ja|東北岩高蘭|ガンコウラン}}、[[牛皮杜鵑|牛皮杜鵑]]、[[篤斯越橘|篤斯越橘]]（黑豆木）、[[越橘|越橘]]（苔桃）、{{link-ja|米葉栂櫻|コメバツガザクラ}}、{{link-ja|高嶺七竃|タカネナナカマド}}、{{link-ja|珍車|チングルマ}}、[[偃松|偃松]]、{{link-ja|蔓山鵑|ミネズオウ}}、深山榛木（ミヤマハンノキ）等植物與{{link-ja|山桔梗|イワギキョウ}}（岩桔梗）、[[多花繁縷|多花繁縷]]（岩爪草）、{{link-ja|御蓼|オンタデ}}、{{link-ja|雲間草|クモマグサ}}<ref group="注釋">分布於御嶽山與{{link-ja|白馬岳|白馬岳}}的稀少植物。</ref><ref>[[#日本の高山植物|日本の高山植物 (1988)、398頁]]</ref>、{{link-ja|黑百合|クロユリ}}、駒草（奇妙荷包牡丹）、[[鹿蹄草葉白珠|鹿蹄草葉白珠]]（白玉木）、{{link-ja|千島桔梗|チシマギキョウ}}、[[高山龍膽|高山龍膽]]（當藥龍膽）、[[伏毛銀蓮花|伏毛銀蓮花]]（白山一花）、{{link-ja|深山秋麒麟草|ミヤマアキノキリンソウ}}、{{link-ja|深山金梅|ミヤマキンバイ}}、{{link-ja|深山大根草|ミヤマダイコンソウ}}、{{link-ja|紅葉唐松|モミジカラマツ}}等[[高山植物|高山植物]]<ref name="山と高原地図1993年版">[[#山と高原地図1993年版|山と高原地図 (1993)、地図裏面]]</ref><ref name="花かおる御嶽山">[[#花かおる御嶽山|花かおる御嶽山 (2000)]]</ref><ref>{{Cite web |url=http://www.vill.otaki.nagano.jp/ontake_tozan/kisoontakesan_no_plant/tozan03_plant_top.html |title=木曽御嶽山（王滝口）登山道の植物 |publisher=王滝村 |accessdate=2011-03-24 |archive-url=https://web.archive.org/web/20110517071939/http://www.vill.otaki.nagano.jp/ontake_tozan/kisoontakesan_no_plant/tozan03_plant_top.html |archive-date=2011-05-17 |dead-url=yes }}</ref>。其中，日本[[特有種|特有種]]御蓼即是在御嶽山上發現<ref>[[#日本の高山植物|日本の高山植物 (1988)、502頁]]</ref><ref>[[#花かおる御嶽山|花かおる御嶽山 (2000)、63頁]]</ref>。另外，山上還可見到裏白七竃與高嶺七竃的[[雜交種|雜交種]]御嶽七竈（''Sorbus'' x ''yokouchii'' M.Mizush. ex T.Shimizu）<ref>[[#日本の高山植物|日本の高山植物 (1988)、388頁]]</ref>。山頂南側現在由於有火山活動，植物稀少，北側則有較為豐富的植物相<ref name="岐阜県の山 (2009)、79-81頁" />。

御嶽山中段的亞高山帶，有{{link-ja|大白時冷杉|オオシラビソ}}（大白檜曾）、[[花楷槭|花楷槭]]（麻幹花）、{{link-ja|日本鐵杉|コメツガ}}（米栂）、{{link-ja|衛氏冷杉|シラビソ}}（白檜曾）、[[岳樺|岳樺]]、{{link-ja|唐檜|トウヒ}}、{{link-ja|七竈|ナナカマド}}、{{link-ja|針蕗|ハリブキ}}、[[黑櫻桃|黑櫻桃]]（深山櫻）等樹木與[[蕨葉草|蕨葉草]]、{{link-ja|蟹蝙蝠|カニコウモリ}}、木曾薊（キソアザミ）、木曾千鳥（キソチドリ）、[[加拿大草茱萸|加拿大草茱萸]]（御前橘）、{{link-ja|小梅蕙草|コバイケイソウ}}、{{link-ja|山荷葉|サンカヨウ}}、{{link-ja|芹葉鹽釜|セリバシオガマ}}、{{link-ja|竹縞蘭|タケシマラン}}、[[七筋姑|七筋姑]]（燕萬年青）、[[五葉黃連|五葉黃連]]（梅花黄蓮）、{{link-ja|舞鶴草|マイヅルソウ}}、[[野捕蟲堇|野捕蟲堇]]、{{link-ja|雪笹|ユキザサ}}等花草<ref name="日本山名辞典 (1992)、116頁" /><ref name="新日本山岳誌 (2005)、972-973頁" /><ref name="山と高原地図1993年版" /><ref name="花かおる御嶽山" />。

御嶽山下部的山林地帶則有{{link-ja|日本水青岡|イヌブナ}}、[[楓樹|楓樹]]、[[連香樹|連香樹]]、[[日本栗|日本栗]]、[[白樺|白樺]]、[[華東椴|華東椴]]、{{link-ja|水楢|ミズナラ}}等[[落葉闊葉林|落葉闊葉林]]與[[日本七葉樹|日本七葉樹]]、{{link-ja|黑檜|クロベ}}、[[日本花柏|日本花柏]]、[[日本扁柏|日本扁柏]]等[[自然林|自然林]]，以及[[日本落葉松|日本落葉松]]、[[日本柳杉|日本柳杉]]、日本扁柏等組成的人工林，還有{{link-ja|岩鏡|イワカガミ}}、{{link-ja|九蓋草|クガイソウ}}、[[日本百合|日本百合]]、[[中國芒|中國芒]]、{{link-ja|蔓紫陽花|ツルアジサイ}}、鳥足弁麻（トリアショウマ）、[[紫斑風鈴草|紫斑風鈴草]]（螢袋）、[[日本藍盆花|日本藍盆花]]、[[柳蘭|柳蘭]]等花草<ref name="山と高原地図1993年版" /><ref name="花かおる御嶽山" />。其中，「木曾扁柏」與「秋田杉」、「津輕檜」合稱『[[日本三大列表|日本三大美林]]』<ref name="日本山名辞典 (1992)、116頁" />。

現在，林野廳已在此地規劃「御岳蕨葉草」與「胡桃島偃松等」兩座植物群落保護林<ref>{{Cite web |url=http://www.rinya.maff.go.jp/chubu/policy/business/conservation/hogorin/syokubutu.html |title=植物群落保護林 |publisher=林野庁中部森林管理局 |accessdate=2013-02-01 |archive-url=https://web.archive.org/web/20130128143834/http://www.rinya.maff.go.jp/chubu/policy/business/conservation/hogorin/syokubutu.html |archive-date=2013-01-28 |dead-url=no }}</ref>。
1910年，植物學者[[小泉源一|小泉源一]]在此山發現[[附子屬|附子屬]]的御嶽附子（''Aconitum metajaponicum'' Nakai）<ref>{{Cite journal|author=門田裕一 |year=1984 |title=東北地方におけるトリカブト属トリカブト亜属植物(キソポウゲ科)の形態的変異と分布 |publisher=国立科学博物館 |journal=国立科学博物館専報 |volume=17 |pages=89}}</ref>。現在是環境省的極危物種<ref>{{Cite web |url=http://www.jpnrdb.com/search.php?mode=map&q=06030301395 |title=日本のレッドデータ検索システム「オンタケブシ」 |publisher=（エンビジョン環境保全事務局） |accessdate=2013-02-03 |archive-url=https://web.archive.org/web/20160813142539/http://jpnrdb.com/search.php?mode=map&q=06030301395 |archive-date=2016-08-13 |dead-url=yes }}</ref>。

=== 生藥「百草丸」 ===
[[File:Phellodendron_amurense_JPG1b.jpg|thumb]]。是山麓製藥公司製造的{{link-ja|百草丸|百草丸}}（御岳百草丸等）主要成分]]
江戶時代[[草藥醫學|本草學]]研究興盛，此地是[[尾張藩|尾張藩]]採集藥草的地點<ref name="hino-seiyaku" />。1716年至1735年（享保年間），本草學者丹羽正伯在山域進行生藥採集與調査。1804年至1818年（[[文化_(元號)|文化]]年間），水谷豐文發行《木曾採藥記》<ref name="hino-seiyaku" />，記載山頂附近有駒草生長<ref name="御岳の信仰と登山の歴史 (1988)、205頁">[[#御岳の信仰と登山の歴史|御岳の信仰と登山の歴史 (1988)、205頁]]</ref>。1844年（天保15年），以周邊木曾谷山域採取的植物製作的[[壓花|壓葉書]]《木曾產草花根皮類》出版，並以圖鑑形式介紹了青栂櫻、{{link-ja|御山龍膽|オヤマリンドウ}}、黑百合、[[東北堇菜|東北堇菜]]、珍車、{{link-ja|水芭蕉|ミズバショウ}}等50種植物<ref name="御岳の信仰と登山の歴史 (1988)、201-202頁">[[#御岳の信仰と登山の歴史|御岳の信仰と登山の歴史 (1988)、201-202頁]]</ref>。隨著本草學的發展，尾張藩的山村代官負責在此進行藥草的調査、栽培、管理<ref name="御岳の信仰と登山の歴史 (1988)、201-202頁" />。1849年（嘉永2年），傳聞開闢王瀧口的普寛行者弟子壽光行者，登上御嶽山採集百種靈草製作生藥<ref name="naganoken-seiyaku" />，之後王瀧村設立「百草元之碑」記載依普寛行者遺法所製造的百草<ref name="御岳山霊なる山の素顔(2010)、77-79頁">[[#御岳山霊なる山の素顔|御岳山霊なる山の素顔(2010)、77-79頁]]</ref>。隨著御嶽信仰的擴展，木曾御嶽的「御神藥」百草傳至各地<ref name="hino-seiyaku" />，成為御嶽參拜的土產<ref>[[#よみがえる木曽人の物語|よみがえる木曽人の物語 (1996)、126頁]]</ref>。現在，南山麓的{{link-ja|長野縣製藥|長野県製薬}}（御岳百草丸）與{{link-ja|日野製薬|日野製藥}}（御嶽山日野百草丸）利用[[關黃柏|黄檗]]製造「{{link-ja|百草丸|百草丸}}」（胃腸藥）<ref name="hino-seiyaku" /><ref name="naganoken-seiyaku" /><ref name="御岳山霊なる山の素顔(2010)、77-79頁" />。藥罐上印有作為御嶽信仰象徴的紋章「山三丸」（可見於御嶽神社與登山道的靈神碑上）。明治時期以後，由於成為士兵的常備藥，為了滿足需求過度採集，使得現在的黄檗多半由[[中國|中國]]等地進口<ref name="御岳山霊なる山の素顔(2010)、77-79頁" />。

=== 國有林 ===
[[File:Mount_Ontake_from_Tanohara_1997-06-01.jpg|thumb]]
御嶽山域的森林在江戶時代是由尾張藩管理，明治時代成為御料林，現在山域則多為國有林<ref name="八ヶ岳・御岳と中央アルプス(1983)、164-165頁" />。其大部分已被指定為{{link-ja|水源涵養保安林|水源かん養保安林}}<ref name="飛騨川森林計画区の計画概要">{{Cite web |url=http://www.rinya.maff.go.jp/chubu/policy/business/sinrinkeikaku/pdf/gaiyouhidagawa.pdf |title=飛騨川森林計画区の計画概要 |publisher=林野庁中部森林管理局 |format=PDF |accessdate=2013-01-31 |archive-url=https://web.archive.org/web/20130128150716/http://www.rinya.maff.go.jp/chubu/policy/business/sinrinkeikaku/pdf/gaiyouhidagawa.pdf |archive-date=2013-01-28 |dead-url=no }}</ref>。另外，御岳特定地理等保護林（1,540公頃）等是為了維持自然環境與保護動植物所成立<ref name="飛騨川森林計画区の計画概要" />。而千間樽國有林與胡桃島國有林有部分區域被指定為「御岳自然休養林」<ref>{{Cite web |url=http://www.rinya.maff.go.jp/chubu/policy/business/invitation/rekumori/ontake/pdf/ontakeitizu2.pdf |title=御岳自然休養林 |publisher=林野庁中部森林管理局 |format=PDF |accessdate=2013-02-01 |archive-url=https://web.archive.org/web/20130201211115/http://www.rinya.maff.go.jp/chubu/policy/business/invitation/rekumori/ontake/pdf/ontakeitizu2.pdf |archive-date=2013-02-01 |dead-url=no }}</ref>。

== 登山 ==
[[File:Hida_Mountains_from_Mount_Ontake.jpg|thumb]]
[[File:Mount_Ontake_from_Tanohara_2005-03-27.jpg|thumb]]
[[File:Fujimi-ishi_in_Mount_Ontake.jpg|thumb]]
古時，御嶽山雖是民間信仰對象，但僅有少數修驗者登上山頂。江戶時代，覺明行者與普寛行者開闢黑澤口與王瀧口後，開啟了全國各地御嶽講信徒集體登拜的熱潮，直至現在仍可看見身著{{link-ja|白裝束|白装束}}的登拜者。而黑澤口等地也可看見御嶽講的先達協助背負信眾的行李與山小屋物資上山<ref name="tisana tabi" /><ref name="御岳山霊なる山の素顔 24頁">[[#御岳山霊なる山の素顔|御岳山霊なる山の素顔(2010)、24頁]]</ref>。

1868年（明治元年），黑澤口8合目的「女人堂」開業，是御嶽山第一間[[山小屋|山小屋]]<ref name="Nyonindo">{{Cite web |url=http://www.kiso.ne.jp/~ontake.85.kt/ |title=木曽御嶽山登山 黒沢口 山小屋 女人堂 2470m |publisher=女人堂 |accessdate=2013-02-14 |archive-url=https://web.archive.org/web/20130213195344/http://www.kiso.ne.jp/~ontake.85.kt/ |archive-date=2013-02-13 |dead-url=no }}</ref>。1872年（明治5年）解除[[女人禁制|女人禁制]]後，女性開始能往女人堂以上前進<ref name="Nyonindo" /><ref>[[#日本アルプスの登山と探検|日本アルプスの登山と探検 (1997)、291頁]]</ref>。明治初期，外國人的登頂展開了近代登山史<ref name="御岳の信仰と登山の歴史 (1988)、207頁">[[#御岳の信仰と登山の歴史|御岳の信仰と登山の歴史 (1988)、207頁]]</ref>。1894年，傳教士{{link-en|沃爾特·韋斯頓|Walter Weston}}登頂<ref name="日本三百名山 (1997)、148頁" />。而與韋斯頓同時期的[[美國|美國]][[天文學家|天文學家]][[帕西瓦爾·羅威爾|帕西瓦爾·羅威爾]]也在訪日時登頂，並在著作《Occult Japan》（第1章「御嶽」）中記述當時的情形<ref>[[#パーシヴァル・ローウェル|パーシヴァル・ローウェル (1895)、第1章「御嶽」]]</ref>。

由於登山口海拔高，因此御嶽山成為登山客能一日登頂的3,000公尺以上山岳<ref name="長野県中信・南信日帰りの山 (2006)、87-91頁">[[#長野県中信・南信日帰りの山|長野県中信・南信日帰りの山 (2006)、74-78頁]]</ref>。另外，山麓的開田中學校與王瀧中學校、下呂市的小學校等也會舉辦學校登山活動<ref name="御岳山霊なる山の素顔 15頁">[[#御岳山霊なる山の素顔|御岳山霊なる山の素顔(2010)、15頁]]</ref>。不過，若在積雪期登山，由於為獨立峰，山頂風勢強勁，地面結冰，登山客有滑落的可能，也有可能視線不佳難以確認路況<ref>[[#日本雪山登山ルート集|日本雪山登山ルート集 (1996)、158-159頁]]</ref><ref>{{Cite web |url=http://www.pref.gifu.lg.jp/kurashi/bosai/sangaku/11115/index.data/H28guidebook.pdf |title=岐阜県北アルプス登山ガイドブック（平成28年度版） |publisher=岐阜県防災課 |format=PDF |pages=9 |accessdate=2016-11-17 |archive-url=https://web.archive.org/web/20161117212350/http://www.pref.gifu.lg.jp/kurashi/bosai/sangaku/11115/index.data/H28guidebook.pdf |archive-date=2016-11-17 |dead-url=no }}</ref>。在近年掀起百名山熱潮後，[[旅行社|旅行社]]推出了許多登山旅遊行程，讓一般民眾更能登上御嶽山。

1979年10月，由於火山噴發，山頂至4合目限制入山。1981年，入山限制縮小至山頂部火山口附近<ref name="日本の名山 246頁" />。之後，當地依據氣象廳發表的噴發警戒等級進行入山限制的調整<ref>{{Cite web |url=http://www.pref.nagano.lg.jp/police/mobile/sangaku/index.html |title=長野県警山岳情報 |publisher=モバイル長野県警察 |accessdate=2016-11-17 |archive-url=https://web.archive.org/web/20161117212526/http://www.pref.nagano.lg.jp/police/mobile/sangaku/index.html |archive-date=2016-11-17 |dead-url=no }}</ref>。2014年9月，噴發警戒等級上調至3（入山限制），火山口周邊約四公里的範圍禁止進入<ref name="jma_20140927_lv3"></ref>。2015年6月，噴發警戒等級下調為2（火山口周邊限制），禁止進入區域縮少至火山口周邊約一公里的範圍<ref name="jma_20170821_lv1"></ref>。2017年8月，噴發警戒等級下調為1（留意活火山），現在禁止進入區域仍維持在火山口周邊約一公里的範圍。依據「岐阜縣北阿爾卑斯地區及活火山地區山岳遭難防止相關條例<ref name="Gifu Pref Regulations for Mountaineer"></ref>」，要進入御嶽山火山口周邊約四公里的範圍時，須提出登山申請書<ref name="jma_Mt.Ontake">{{cite web|url=http://www.data.jma.go.jp/svd/vois/data/tokyo/STOCK/activity_info/312.html|title=御嶽山の活動状況|accessdate=2017/10/10|author=気象庁|archive-url=https://web.archive.org/web/20171012150237/http://www.data.jma.go.jp/svd/vois/data/tokyo/STOCK/activity_info/312.html|archive-date=2017-10-12|dead-url=no}}</ref>。

=== 登山道 ===
御嶽山各方向都有登山道<ref name="tizu-kokudo" /><ref name="日本登山図集 (1986)、54-57頁" /><ref name="山と高原地図(2011)" /><ref>[[#東海・北陸の200秀山|東海・北陸の200秀山 (2009)、56-57頁]]</ref>。使用小坂口、日和田口的登山客，除了登上劍峰，也有部分是前往飛驒頂上、繼子岳、或摩利支天山。一之池西側稜線的登山道是劍峰往二之池小屋的路線之一。以前，還有經王瀧川支流濁川旁的濁川溫泉（已不存在）前往劍峰與繼母岳鞍部的松原新道，但在1984年長野縣西部地震中崩塌，現已廢道<ref name="日本登山図集 (1986)、54-57頁" />。
[[File:Brocken_spectres_in_Mount_Ontake.jpg|thumb]]]]

==== 王瀧口 ====
王瀧口是一般車道能抵達的最高登山口（田之原，標高2,160公尺），是前往山頂劍峰的最短路線。途中風景優美。登山口田之原至王瀧頂上之間的8合目與9合目有避難小屋。每年7月1日會舉行相當於[[開山|開山]]的開山祭。積雪期會封閉往登山口田之原的道路，但在初春時期可以改利用御嶽2240滑雪場纜車進行雪地登山。1合目有祭拜國常立尊、大己貴命、少彥名命的御嶽神社里宮<ref name="木曽のおんたけさん―その歴史と信仰(2009)、108-110頁" />。3合目有清瀧與新瀧，清瀧祭祀清瀧不動明王，是登拜者進行{{link-ja|水行|垢離}}的行場<ref name="木曽のおんたけさん―その歴史と信仰(2009)、108-110頁" />。5合目有求子神靈十二權現，6合目有眼睛神明八海山大神<ref name="木曽のおんたけさん―その歴史と信仰(2009)、108-110頁" />。標高約2,190公尺的田之原有田之原山莊與木曾御嶽自然休養林田之原自然公園，登山季時汽車可開至此地<ref name="山と高原地図 (1993)、地図表面" />。
* 田之原 - 7合目大江權現 - 金剛童子（8合目下） - 9合目石室 - 王瀧頂上 - 劍峰<ref>{{Cite web |url=http://www.vill.otaki.nagano.jp/ontake_tozan/tozan04.html |title=木曽御嶽山（王滝口）登山道の紹介 |publisher=王滝村 |accessdate=2011-03-24 |archive-url=https://web.archive.org/web/20110517073323/http://www.vill.otaki.nagano.jp/ontake_tozan/tozan04.html |archive-date=2011-05-17 |dead-url=yes }}</ref>

==== 黑澤口 ====
[[File:Mount_Ontake_from_Nyonindo.jpg|thumb]]
覺明行者開闢的最古老登山道，也是與{{link-ja|御嶽教|御嶽教}}、木曾御嶽本教關係最深的登山道。夏天可見詠唱「懺悔反省、六根清淨」的白衣登拜者<ref name="日本の山1000 (1992)、350-351頁" /><ref>[[#木曽のおんたけさん―その歴史と信仰|木曽のおんたけさん―その歴史と信仰(2009)、7頁]]</ref>。各合目有有人山小屋。現在7合目附近有[[索道|空中纜車]]飯森站（標高2,150公尺），與王瀧口登山口的田之原海拔相近，因此大多登山客會利用空中纜車至飯森站。與王瀧口一樣，每年7月1日會舉行相當於開山的開山祭。1合目有祭祀大己貴命的御嶽神社若宮與祭祀少彥名命的御嶽神社里宮<ref name="木曽のおんたけさん―その歴史と信仰(2009)、107-108頁" />。登拜道場可看見許多靈神碑，其中4合目的屋敷場是御嶽山最大規模的靈神場<ref name="木曽のおんたけさん―その歴史と信仰(2009)、107-108頁" />。松尾瀑布經油木尾根、百間瀑布至8合目女人堂的路線稱為百間瀑布道<ref name="山と高原地図 (1993)、地図表面" />。百間瀑布有百間瀑布小屋<ref name="木曽のおんたけさん―その歴史と信仰(2009)、107-108頁" />。
* 中之湯（6合目）或空中纜車飯森站 - 行場山莊（7合目） - 女人堂（8合目） - 石室山莊（9合目） - 覺明堂 - 劍峰<ref>{{Cite web |url=http://www.kankou-kiso.com/miru/index_3.html |title=御嶽山に登ろう・登山コース |publisher=木曽町観光協会 |accessdate=2011-03-24 |archiveurl=https://web.archive.org/web/20160420163313/http://www.kankou-kiso.com/miru/index_3.html |archivedate=2016-04-20 }}</ref>

==== 開田口 ====
木曾側三個登山口中唯一與信仰無關的登山道。原為營林署（現森林管理署）的作業道。距離長，可充分享受大自然。4合目附近有湧水，之後至三之池為止皆無水源。登山口海拔較低（1,500公尺），適合健壯人士。
* 開田口4合目 - 7合目避難小屋跡 - 三之池避難小屋 - 賽之河原 - 二之池 - 劍峰
[[File:Hida_peak_of_Mount_Ontake.jpg|thumb]]

==== 小坂口 ====
往劍峰的路途較長，較少信仰登山者，多為一般登山客。開山在每年6月15日於御嶽神社飛驒口里宮舉行<ref>{{Cite web |url=http://www.city.gero.lg.jp/kankou/view.rbz?of=2&ik=0&pnp=14&cd=444 |title=御嶽山に夏を告げる!!御嶽山山開き |publisher=下呂市 |date=2009-06-12 |accessdate=2013-02-12 }}</ref>。本路線以[[溫泉|溫泉]]為主打特色。另外，還有一條從登山口經胡桃島，至窺視岩（のぞき岩）避難小屋下與濁河溫泉登山道會合的路線，又稱為胡桃島路線。飛驒頂上的五之池小屋在2010年增建新館，容納人增至100人<ref>{{Cite web |url=http://gonoike.jp |title=御嶽山 飛騨頂上 五の池小屋 |publisher=五の池小屋 |accessdate=2016-11-17 |archive-url=https://web.archive.org/web/20161111140527/http://gonoike.jp/ |archive-date=2016-11-11 |dead-url=no }}</ref>。
* {{link-ja|濁河溫泉|濁河温泉}} - 湯之花峠 - 窺視岩避難小屋 - 飛驒頂上（{{link-ja|五之池小屋|五の池小屋}}） - 賽之河原 - 二之池 - 劍峰

==== 日和田口 ====
前往繼子岳頂的登山路線，也是與宗教無關的路線。1998年{{link-ja|CIAO御岳滑雪度假村|チャオ御岳スノーリゾート|CIAO御岳滑雪場}}開發後，登山客多使用全年運行的[[索道|纜車]]上山，使得原登山口使用人數下降。利用纜車上山可至與王瀧口高度相似的地點（2,190公尺），但經繼子岳前往最高峰劍峰的距離依然比其他路線長。此外，本路線的整備程度也較其他路線低。2010年，索道僅在滑雪季（12月初旬 - 5月初旬）運行，春至秋季無法搭乘。
* 日和田口或空中纜車山頂站 - 繼子岳 - 飛驒頂上（五之池小屋） - 賽之河原 - 二之池 - 劍峰

=== 山小屋 ===
{{Commonscat|Mountain huts on Mount Ontake|御嶽山的山小屋}}
[[File:Ninoike_huts_Mount_Ontake.jpg|thumb]]
御嶽山的宗教登山興盛，因此山上有許多[[山小屋|山小屋]]或宗教小屋。其中五之池小屋是近代的阿爾卑斯風格山小屋<ref name="岐阜県の山 (2009)、79-81頁" />。各登山道與山頂都設有山小屋與避難小屋<ref name="日本百名山地図帳(2005)、158-161頁" /><ref>{{Cite book|date=2010-12-15 |title=山と溪谷2011年1月号付録（山の便利手帳2011） |publisher=山と溪谷社 |pages=166-178 |asin=B004DPEH6G}}</ref>。這些山小屋的客廳與客房內多掛著御嶽神社的掛軸。另外，有許多小屋是在王瀧口、黑澤口開山祭的7月1日以後開始營業，至8月底至9月底為止。御嶽神社、御嶽教、木曾御嶽本教的開山是在7月10日至9月10日，這段期間在小屋可見許多穿著白裝束的宗教登山者。五之池小屋的營業時間是6月1日至10月15日，時間較其他小屋長。8合目與7合目的山小屋僅在紅葉季營業。

許多山小屋是以二之池池水做為水源來源，其中距離二之池較遠的「五之池小屋」與7合目以下山小屋以外的各山小屋合組「二之池水組合」，共同建設與維護水道設施。擁有豐富水源的「二之池本館」、「二之池新館」、「石室山莊」、「覺明堂」可讓住宿者洗澡，也有某些山小屋在人數少的時候開放讓住宿者使用工作人員的浴室。然而，人類活動也帶來環境問題，政府正推動山小屋改善廁所與排泄物處理設施<ref>{{Cite web |url=http://www.pref.nagano.lg.jp/kikaku/kensei/soshiki/yosan/kuniyobo/documents/07_26.pdf |title=山小屋トイレ等の改善推進について |publisher=長野県環境部 |format=PDF |date=2010-10 |accessdate=2016-11-17 |archive-url=https://web.archive.org/web/20161117212545/http://www.pref.nagano.lg.jp/kikaku/kensei/soshiki/yosan/kuniyobo/documents/07_26.pdf |archive-date=2016-11-17 |dead-url=no }}</ref>。

露營可利用胡桃島口登山口的「胡桃島露營場」，其他登山道上沒有露營地，<ref name="日本百名山地図帳(2005)、158-161頁" />。

==== 燒印 ====
有些有人山小屋與山頂神社社務所可為宗教登山使用的金剛杖進行[[烙印|燒印]]。1891年（明治24年），登山道的神社會立起提供「登山杖紀念燒印」的看板<ref>[[#日本アルプスの登山と探検|日本アルプスの登山と探検 (1997)、284頁、287頁]]</ref>。雖然燒印在[[富士山|富士山]]非常熱門，但在御嶽山，提供燒印的山小屋正在不斷減少。此外，不同於富士山有大量需求，由於御嶽山的需求者少，登山客需要多等待燒印加熱的時間，費用也較富士山高。

== 地理 ==
御嶽山是[[木曾川|木曾川]][[水系|水系]]的源頭，西側有{{link-ja|飛驒川|飛騨川}}、東與南側有王瀧川。劍峰山頂有[[三角點|一等三角點]]（點名「御岳山」、標高3,063.<small>41</small> m）<ref name="kijyun" />，為{{link-ja|一等三角點百名山|一等三角点百名山}}之一<ref>[[#一等三角点百名山|一等三角点百名山 (1988)]]</ref>。最高點在御嶽神社西側的岩石標高點<ref name="kokudo" />。在日本所有山岳中排名第14高<ref>{{Cite web |url=http://www.gsi.go.jp/KOKUJYOHO/MOUNTAIN/mountain.html |title=日本の主な山岳標高（標高別表示） |publisher=国土地理院 |accessdate=2013-02-02 |archive-url=https://web.archive.org/web/20120502035009/http://www.gsi.go.jp/KOKUJYOHO/MOUNTAIN/mountain.html |archive-date=2012-05-02 |dead-url=no }}</ref>。

東南中腰的王瀧村一帶稱為「御岳高原」。

=== 周邊各山 ===
[[File:20101130木曽山脈.jpg|thumb]]，左後是御嶽山，右後是[[飛驒山脈|飛驒山脈]]。之間沒有稜線相連]]
[[File:Mt.Ontake_and_Mt.Norikuradake_from_Mt.Yatsugatake_01.jpg|thumb]]眺望御嶽山（左）與[[乘鞍岳|乘鞍岳]]（右）]]
由於是獨立峰，周圍許多山岳都可見到御嶽山的山容。山頂寬廣，形成複數尖峰<ref name="山と高原地図(2011)">[[#山と高原地図|山と高原地図(2011)の付属図]]</ref>。在岐阜縣側有前衛山，分別為岐阜百山之一的御前山（1,646公尺）與同為岐阜百山的下呂御前山（1,412公尺）。

=== 河川 ===
[[File:Miure_Dam_lake_survey.jpg|thumb]]水系王瀧川上游的{{link-ja|三浦水壩|三浦ダム}}]]
御嶽山是木曾川水系的源頭，最後流入[[伊勢灣|伊勢灣]]<ref name="tizu-kokudo" />。山頂東南方13.4公里處有王瀧川的{{link-ja|牧尾水壩|牧尾ダム}}與水庫'''御岳湖'''，負責供給[[愛知縣|愛知縣]]等地的飲用水、工業用水及農業用水，其上游有[[關西電力|關西電力]]發電用的的{{link-ja|三浦水壩|三浦ダム}}（山頂西南方11.0公里）。王瀧川[[支流|支流]]濁川的上游赤川有激烈侵蝕的地形，被稱作「地獄谷」<ref name="山と高原地図(2011)" /><ref>{{Cite web |url=http://watchizu.gsi.go.jp/watchizu.html?longitude=137.4740333&latitude=35.88713611 |title=地図閲覧サービス・地獄谷（長野県木曽郡王滝村） |publisher=国土地理院 |accessdate=2011-10-05 |archive-url=https://web.archive.org/web/20120505014038/http://watchizu.gsi.go.jp/watchizu.html?longitude=137.4740333&latitude=35.88713611 |archive-date=2012-05-05 |dead-url=no }}</ref>。

* 西野川、白川、王瀧川（木曾川支流）
* 飛驒川
* 秋神川、小坂川（飛驒川支流）

=== 主要山口 ===

* [[野麥峠|野麥峠]] - 御嶽山與乘鞍岳之間。標高1,672公尺。主要道路{{link-ja|長野縣道、岐阜縣道39號奈川野麥高根線|長野県道・岐阜県道39号奈川野麦高根線}}。
* 長峰峠 - {{link-ja|國道361號|国道361号}}岐阜縣與長野縣交界處，是暮岩川（飛驒川支流）與西野川（木曾川支流）的[[分水嶺|分水嶺]]。
* 九藏峠 - 國道361號至開田高原的山口，九藏峠展望台可往西南西方遠望御嶽山。
* 濁河峠 - 秋神川與濁河川（小坂川支流）的分水嶺。
* 白巣峠 - 白川（王瀧川支流）與{{link-ja|付知川|付知川}}的分水嶺，有林道通過。
* 鞍掛峠 - 水無川（王瀧川支流）與竹原川（飛驒川支流）的分水嶺，有林道通過。

=== 交通 ===

南山麓的王瀧川在1923年（大正12年）完成{{link-ja|木曾森林鐵道|木曽森林鉄道}}王瀧森林鐵道的鋪設，1975年（昭和50年）5月30日以前負責國有林的材木運輸<ref>{{Cite web |url=http://www.vill.otaki.nagano.jp/sinnrinntetudou/rintetu.html |title=第3回王滝森林鉄道フェスティバル開催 |publisher=王滝村 |accessdate=2013-01-31 |archiveurl=https://web.archive.org/web/20160822114949/http://www.vill.otaki.nagano.jp/sinnrinntetudou/rintetu.html |archivedate=2016-08-22 }}</ref>。1925年（大正14年），御嶽自動車商會（現在的御嶽交通）開業，營運山麓往來御嶽山登山口的路線巴士。1963年（昭和38年），西山腹的名古屋營林局小坂營林署營運的小坂森林鐵道濁河溫泉線開通，負責運送採伐的，但在1971年（昭和46年）改由卡車運輸而廢止。1966年（昭和41年），從山麓的王瀧村{{link-ja|長野縣道256號御岳王瀧黑澤線|長野県道256号御岳王滝黒沢線}}至田之原的「收費道路林道黑石線」全線開通，1997年（平成9年）4月1日改為村道41號線（御岳Skyline）免費開放<ref name="日本登山図集 (1986)、54-57頁" />。{{link-ja|長野縣道473號上松御岳線|長野県道473号上松御岳線}}在東南山麓經白崩林道可是山岳中段的中之湯<ref name="日本登山図集 (1986)、54-57頁" />。東山麓有{{link-ja|長野縣道20號開田三岳福島線|長野県道20号開田三岳福島線}}通過，「御岳Blue Line」可從木曾町三岳的木曾溫泉附近至御岳空中纜車滑雪場。其中鹿之瀨溫泉往上的路段在冬季關閉<ref>{{Cite web |url=http://www.town-kiso.com/kurashi/douro/100070/100086/ |title=町道 冬期通行止め情報 |publisher=木曽町 |accessdate=2016-11-17 |archive-url=https://web.archive.org/web/20161022034129/http://www.town-kiso.com/kurashi/douro/100070/100086/ |archive-date=2016-10-22 |dead-url=yes }}</ref>。東南山麓的縣道20號開田三岳福島線旁有{{link-ja|道之驛三岳|道の駅三岳}}。東北山麓有{{link-ja|國道361號|国道361号}}、北山腹有{{link-ja|岐阜縣道463號朝日高根線|岐阜県道463号朝日高根線}}、西北山腹{{link-ja|岐阜縣道435號御岳山朝日線|岐阜県道435号御岳山朝日線}}通過。西山麓的下呂市小坂町可由{{link-ja|岐阜縣道441號濁河溫泉線|岐阜県道441号濁河温泉線}}至濁河溫泉，又稱「御嶽Panolama Line」。小三笠山（標高2,029公尺）南面有御岳林道與王瀧林道。

* 田之原天然公園 - 御嶽山登山道王瀧口附近的天然公園，有穿越濕原與針葉林的步道與展望台<ref name="tanohara">{{Cite web |url=http://www.rinya.maff.go.jp/chubu/policy/business/invitation/rekumori/kisoontake/index.html |title=木曽御岳 |publisher=中部森林管理局 |accessdate=2011-03-24 |archive-url=https://web.archive.org/web/20110323204707/http://www.rinya.maff.go.jp/chubu/policy/business/invitation/rekumori/kisoontake/index.html |archive-date=2011-03-23 |dead-url=no }}</ref>。
* 御岳空中纜車 - 御岳空中纜車滑雪場在2013年結束滑雪場業務，冬季停開<ref>{{Cite web |url=http://www.ontakerope.co.jp/ |title=御岳ロープウェイ |publisher=御岳ロープウェイ |accessdate=2013-02-14 |archive-url=https://web.archive.org/web/20130216000414/http://www.ontakerope.co.jp/ |archive-date=2013-02-16 |dead-url=no }}</ref>。
* 開田高原 - 知名的[[蕎麥|蕎麥]]產地與{{link-ja|木曾馬|木曽馬}}之里，有騎馬體驗設施。
* 自然湖 - {{link-ja|長野縣西部地震|長野県西部地震}}在[[木曾川|木曾川]][[支流|支流]]王瀧川上游造成的堰塞湖，湖中豎立許多枯木，夏季[[獨木舟|獨木舟]]盛行<ref name="御岳山霊なる山の素顔 49頁">[[#御岳山霊なる山の素顔|御岳山霊なる山の素顔(2010)、49頁]]</ref>。
* {{link-ja|牧尾水壩|牧尾ダム|御岳湖}} - 王瀧川牧尾水壩所形成的[[湖|湖]]。
* 御嶽休暇村 - 御岳高原的[[名古屋市|名古屋市]]保險休養地，周邊有木曾御岳鄉村俱樂部的[[高爾夫球場|高爾夫球場]]。
* {{link-ja|濁河溫泉|濁河温泉}} - 御嶽山濁河口登山口的[[溫泉|溫泉]]
* [[東京大學|東京大學]][[天文台|天文台]]木曾觀測所 - 東京大學大學院理學系研究科附屬天文學教育研究中心的木曾觀測所。1977年（昭和55年）2月，{{link-ja|香西洋樹|香西洋樹}}與{{link-ja|古川麒一郎|古川麒一郎}}在此發現[[小行星|小行星]]，命名為『[[小行星2330|御嶽]]』。
* {{link-ja|飛驒御嶽高原高地訓練區|飛騨御嶽高原高地トレーニングエリア}} - 西北山腹的高地訓練設施。2007年（平成19年）在濁河溫泉附近的下呂市開設「御嶽環景運動場」。濁河溫泉與CIAO御岳渡假村之間有「飛驒御嶽[[高橋尚子|尚子]]Boulder Road」<ref>{{Cite web |url=http://www.hida-athlete.jp/training_c/nigorigo.html |title=トレーニングゾーン（濁河温泉） |publisher=飛騨御嶽高原高地トレーニングエリア |accessdate=2013-02-12 |archive-url=https://web.archive.org/web/20121209065941/http://www.hida-athlete.jp/training_c/nigorigo.html |archive-date=2012-12-09 |dead-url=no }}</ref>。
* 小坂瀑布周遊路線 - 西山腹下呂市小坂町的飛驒川支流小坂川有200處落差達五公尺以上的瀑布<ref name="tizu-kokudo" /><ref name="ontake_kazan04" />，該網站介紹了許多多瀑布周遊路線<ref name="osaka-taki" />。

== 風景 ==
御嶽山是個有壯大山容的獨立峰，從[[濃尾平野|濃尾平野]]（尾張與美濃）與遠方山岳都可望見<ref name="よみがえる木曽人の物語 (1996)、119頁" />。東海地方有許多學校將山名加入校歌中<ref group="注釋">例如：愛知縣立一宮北高等學校「みはるかす尾張の平野独り立つ御岳」、愛知縣立小牧南高等學校「朝あけの山なみ越えて空かぎる御岳はるか」、北名古屋市立師勝中學校「御嶽の峰に雲たなびきて」、稻澤市立大里東小學校「雪の御岳[[伊吹山|伊吹]]や{{link-ja|多度山|多度山|多度}}の山なみ遠く望む時」</ref><ref>[[#木曽のおんたけさん―その歴史と信仰|木曽のおんたけさん―その歴史と信仰(2009)、126-127頁]]</ref>。5月中旬，東面的9合目附近的殘雪是知名的{{link-ja|雪形|雪形}}（殘留的白雪背景與黑色地面形成的圖案）<ref>{{Cite web |url=https://www.city.azumino.nagano.jp/soshiki/32/10310.html |title=雪形・長野県中部 |publisher=[[安曇野市|安曇野市]] |date=2015-12-28 |accessdate=2016-11-17 |archive-url=https://web.archive.org/web/20160814182708/http://www.city.azumino.nagano.jp/soshiki/32/10310.html |archive-date=2016-08-14 |dead-url=no }}</ref>。

==噴發事故==
{{主|2014年御嶽山噴發}}
[[File:Mount_Ontake_from_Kurakake_Pass.JPG|thumb]]
2014年9月27日，當地時間11點52分，御嶽山再次噴發，事前並無預警，因而造成多人死傷。此次噴發，濃煙往南向山麓蔓延至3公里以外地方，根據途徑附近的飛機報告，高度可能達到11000米以上。火山灰覆蓋近山頂許多小屋，地上累積達15公分。同日12時36分，日本氣象廳發佈火山口周邊警報，並將火山噴發警戒級別提升到級別3的 入山限制（最高警戒級別為級別5），並指出於火山口附近4公里範圍內，有必要警戒隨著火山爆發而飛出的岩石與火山礫。噴發時，正逢賞楓季及週末休假日，山頂附近估計有超過250名遊客，火山在强烈喷发几分钟后，一度有所“停息”，趁这一间隙大部分登山者得以安全撤离，但仍有近40人被困在山上的度假屋（部分受困者於28日成功下山）。

9月27日下午，[[氣象廳_(日本)|日本氣象廳]]地震火山部火山課課長北川贞之在新闻发布会上称，自進入2014年9月以後御嶽山地震频度开始增加，仅9月11日一天便发生了超过80次地震；但日本氣象廳没有观测到山表面因地下熔岩上升而导致的膨胀现象，“仅基于地震的次数判断是否为喷发前兆，这难度很大。”<ref>{{Cite web |url=http://news.sina.com.cn/w/p/2014-09-28/025930925504.shtml |title=“日本火山喷发天降巨石砸向登山客”，《新京报》2014年9月28日 |access-date=2014-09-28 |archive-url=https://web.archive.org/web/20141001042724/http://news.sina.com.cn/w/p/2014-09-28/025930925504.shtml |archive-date=2014-10-01 |dead-url=no }}</ref>

日本氣象廳從8月29日起開始觀測到火山性地震，9月10日地震52次，11日地震85次。12日以後呈現減少態勢，26日僅為6次。其他測量儀器（除了地震儀，設置了測量岩漿上升導致的山體變化的全球導航衛星系統、傾斜測量儀、長焦相機等）的地殼變動數據並未明顯變化，因此未將警戒級別上調。東京大學火山學名譽教授荒牧重雄分析稱「噴發前未發現山體膨脹，噴煙呈白色」很有可能是「蒸汽爆炸」即岩漿的熱量將其上方的地下水膨脹成水蒸氣，將聚集在火山口的岩石爆炸或吹飛。較難以發現前兆且可能短時間內多次反覆噴發。<ref>{{cite news|title=日本為何沒能發出御嶽山噴發警告？|url=http://zh.cn.nikkei.com/politicsaeconomy/politicsasociety/11231-20140928.html|accessdate=2014年9月29日|newspaper=日經中文網|date=2014年9月28日|archive-url=https://web.archive.org/web/20140928150514/http://zh.cn.nikkei.com/politicsaeconomy/politicsasociety/11231-20140928.html|archive-date=2014-09-28|dead-url=no}}</ref>

[[长野县|长野县]]警方10月1日发布消息称，此次喷发共造成48人死亡，死者人数超过了1991年长崎县云仙普贤岳喷发事故的43人，成为战后日本最严重的火山灾难<ref>{{Cite web |url=http://news.sina.com.cn/w/2014-10-01/173230942780.shtml |title=中国新闻网：日本御岳山喷发死者升至48人 |access-date=2014-10-01 |archive-url=https://web.archive.org/web/20141006170709/http://news.sina.com.cn/w/2014-10-01/173230942780.shtml |archive-date=2014-10-06 |dead-url=no }}</ref><ref name="sankei201409290003">{{Cite web|url=http://sankei.jp.msn.com/affairs/news/140929/dst14092900030001-n1.htm|title=海外メディアも人的被害の大きさ速報|publisher=産経新聞社|date=2014-09-29|accessdate=2014-09-30 |archiveurl=https://web.archive.org/web/20150122010400/http://www.sankei.com/affairs/news/140929/afr1409290001-n1.html |archivedate=2015-01-22 }}</ref><ref>[http://www.nikkei.com/article/DGXLASDG01H1L_R01C14A0MM8000/ 御嶽山噴火、47人の死亡確認　戦後最悪の火山災害に ] {{Wayback|url=http://www.nikkei.com/article/DGXLASDG01H1L_R01C14A0MM8000/ |date=20180902220554 }}2014年10月2日閲覧</ref>。
日本列島位於多個地球板塊交界處，有110座活火山，20世紀日本最大火山噴發是1914年鹿兒島縣[[櫻島|櫻島]]，流出的熔岩湧向大海，將櫻島和大隅半島連為一體，櫻島變成半島，58人死亡。最大傷亡是1926年5月北海道[[十勝岳|十勝岳]]噴發發生「融雪型火山土石流」。導致144人死亡和失蹤。<ref>{{Cite web |url=http://zh.cn.nikkei.com/politicsaeconomy/politicsasociety/11257-20140929.html |title=日本的火山噴發史 |access-date=2014-10-03 |archive-url=https://web.archive.org/web/20141006072107/http://zh.cn.nikkei.com/politicsaeconomy/politicsasociety/11257-20140929.html |archive-date=2014-10-06 |dead-url=no }}</ref>

10月23日長野縣資訊:御嶽山噴發造成57人死亡，6人失蹤。下山136人(重傷27人，輕傷32人)。由於擔心降雪等原因，搜救要等到明年春季之後才能重啟。
