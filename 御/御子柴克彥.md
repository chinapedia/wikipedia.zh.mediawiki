'''御子柴克彥'''（{{jpn|j=御子柴 克彦|hg=みこしば かつひこ|rm=Mikoshiba Katsuhiko|f=h}}，{{bd|1945年|3月3日}}），日本[[腦神經|腦神經]]科學家、[[分子生物學|分子生物學]]家、[[遺傳學|遺傳學]]家，現任[[理化學研究所|理化學研究所]]腦科學綜合研究中心研究員、[[東京大學|東京大學]]名譽教授<ref>{{Cite web|url=http://www.jst.go.jp/icorp/jpn/evaluation/report150/mikoshiba.htm|title=御子柴克彦氏の略歴|publisher=科学技術振興機構|accessdate=2010-01-31}}</ref><ref>{{Cite web|date=2009-02-24|url=http://www.riken.jp/r-world/topics/090224/index.html|title=御子柴克彦グループディレクターが第40回内藤記念科学振興賞を受賞|publisher=理化学研究所|accessdate=2010-01-31}}</ref>、[[慶應義塾|慶應義塾]]評議員。[[紫綬褒章|紫綬褒章]]表彰。

== 生平 ==
1945年3月3日，御子柴克彥誕生於日本[[長野縣|長野縣]]，成長於[[東京都|東京都]]。[[慶應義塾大學|慶應義塾大學]]醫學部畢業。醫學博士（慶應義塾大學）。

*1974年 慶應義塾大學醫學部生理學教室 専任講師
*1976年 法國[[巴斯德研究院|巴斯德研究院]] 研究員
*1982年 慶應義塾大學醫學部生理學教室 助教授
*1985年 {{link-ja|大阪大學蛋白質研究所|大阪大学蛋白質研究所}} 教授（迄1992年）
*1986年 {{link-ja|岡崎國立共同研究機構|岡崎国立共同研究機構}} {{link-ja|基礎生物學研究所|基礎生物学研究所}}、教授（兼任、迄1991年）
*1992年 {{link-ja|東京大學醫科學研究所|東京大学医科学研究所}} 教授（迄2007年）、[[理化學研究所|理化學研究所]] 主任研究員（兼任、迄1997年）
*1995年 新技術事業團（現獨立行政法人{{link-ja|科學技術振興機構|科学技術振興機構}}）ERATO, ICORP-SORST專案代表研究者
*1997年 理化學研究所 腦科學綜合研究中心 發育・演化研究組（迄2009年）
*2003年 瑞典[[卡罗琳学院|卡罗琳学院]] 外國人客座教授
*2004年 [[東京慈惠會醫科大學|東京慈惠會醫科大學]]客座教授、[[山形大學|山形大學]]客座教授
*2005年 [[日本學術會議|日本學術會議]]会員
*2007年 東京大學 名譽教授
*2008年 韓國[[首爾國立大學|首爾國立大學]]外國人教授
*2009年 理化學研究所 腦科學綜合研究中心 發育神經生物學小組組長
*2010年 慶應義塾大學醫學部客座教授

== 貢獻 ==
御子柴克彥是最初闡明[[IP3受體|IP3受體]]（IP3 receptor）神經細胞通道的科學家，近年主要成就還有：
*2000年培育了一種[[內質網|內質網]]不向[[海馬體|海馬體]]輸送鈣質的[[基因改造|基因改造]]鼠，發現增強記憶力的奧秘。
*2005年培育了一種不帶有IP3受體Ⅱ型與Ⅲ型[[蛋白質|蛋白質]]的基因改造鼠，開啟[[乾眼症|乾眼症]]的新治療方法。
*2013年發現引發[[阿茲海默症|阿茲海默症]]的新病因。<ref>{{Cite web |url=http://www.yokohama-cu.ac.jp/univ/pr/press/130906.html |title=医学群 五嶋教授ら研究グループが、アルツハイマー型認知症を発症する新たなメカニズムを発見！ | 横浜市立大学 |access-date=2014-06-11 |archive-url=https://web.archive.org/web/20140714122548/http://www.yokohama-cu.ac.jp/univ/pr/press/130906.html |archive-date=2014-07-14 |dead-url=yes }}</ref>
