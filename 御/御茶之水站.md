'''御茶之水站'''（{{jpn|j=御茶ノ水駅|kyu=御茶ノ水驛|hg=おちゃのみずえき|hgt=おちやのみづえき|rm=Ochanomizu eki|f=y}}）位於[[日本|日本]][[東京都|東京都]][[千代田區|千代田區]]與[[文京區|文京區]]，是[[東日本旅客鐵道|東日本旅客鐵道]]（JR東日本）、[[東京地下鐵|東京地下鐵]]的[[鐵路車站|鐵路車站]]。

[[神田川_(東京都)|神田川]]（外堀）南側（千代田區側）是JR東日本站體、北側（文京區側）是東京地下鐵站體。JR東日本站體的所在地是千代田區[[神田駿河台|神田駿河台]]二丁目、東京地下鐵站體是文京區[[湯島|湯島]]一丁目。

==停靠路線==
此站有JR東日本的各線（後述）與東京地下鐵的[[丸之內線|丸之內線]]停靠，是轉乘站。中央線快速的[[車站編號|車站編號]]是「'''JC 03'''」。中央總武線各站停車的車站編號是「'''JB 18'''」。丸之內線的車站編號是「'''M 20'''」。

JR東日本車站是所屬線[[中央本線|中央本線]]與以本站為終點的[[總武本線|總武本線]]（支線）的分岐站。中央本線本站以西（[[新宿站|新宿站]]方向）為急行線（快速線）與緩行線的[[複複線|複複線]]區間。急行線行駛停靠[[東京站|東京站]]的[[中央線快速|中央線快速電車]]，緩行線行駛與總武本線（[[錦糸町站|錦糸町站]]方向）直通運轉的[[中央·總武緩行線|中央·總武線各站停車]]。但早晨、深夜時段也行駛停靠東京站的中央緩行線各站停車。

另外，依據[[特定都區市內|特定都區市內]]制度，本站屬於「[[特定都區市內#設定區域一覧|東京都區內]]」、「[[東京山手線內|東京山手線內]]」。

==歷史==
站名由來請參考地名[[御茶之水|御茶之水]]。

=== 甲武鐵道 ===
現在中央本線的[[新宿站|新宿]]～[[八王子站|八王子]]區間是由[[私鐵|私鐵]]{{link-ja|甲武鐵道|甲武鉄道}}建設。之後甲武鐵道計畫將路線延伸至東京市中心，1895年（明治28年）4月3日通車至{{link-ja|飯田町站|飯田町駅}}。此飯田町站位於現在的[[飯田橋站|飯田橋站]]東側，之後作為貨物車站於1999年（平成11年）廢止。接著甲武鐵道為增加列車運行班次，以及減輕蒸汽機車的噪音與空氣汙染，進行路線[[電氣化鐵路|電氣化]]。1904年（明治37年）12月31日，飯田町至御茶之水的[[複線|複線]]電氣化路線通車，御茶之水站開業。御茶之水～[[中野站_(東京都)|中野]]之間1日有28班往返、往新宿10分一班<ref name = "飯田町駅ものがたり">[[#飯田町駅ものがたり|「飯田町駅ものがたり」]]</ref>。當時的御茶之水站位於現在站址的新宿側、{{link-ja|御茶之水橋|御茶ノ水橋}}對側。站房位於現在[[神田警察署|神田警察署]]御茶之水交番的位置<ref name = "市街線点描">[[#市街線点描|「甲武鉄道の市街線点描」]]</ref>，為西式木造平房建築<ref name = "駅のはなし_140">[[#駅のはなし|『駅のはなし』p.140]]</ref>，月台為側式月台2面2線<ref name = "今昔50年_55">[[#今昔50年|『中央線 オレンジ色の電車今昔50年』p.55]]</ref>。

這一區間的建設，由於在東京市區改正委員會要求避免影響道路，將路線調整在{{link-ja|外濠 (東京都)|外濠 (東京都)|外濠}}內側。另又為了保存堤岸與崖地景觀，加上湧水問題，施工十分困難，使得御茶之水站被迫設於神田川與崖地之間的狹窄地點<ref name = "バリアフリー検証_75-76">[[#バリアフリー検証|「JR御茶ノ水駅のバリアフリー化検証」pp.75 - 76]]</ref>。

甲武鐵道接著持續進行御茶之水站東側區間的建設。但1906年（明治39年）10月1日，甲武鐵道依據{{link-ja|鐵道國有法|鉄道国有法}}國有化，御茶之水站成為國有鐵道車站，御茶之水站東延工程也由國鐵接手。1908年（明治41年）4月19日通車至{{link-ja|昌平橋站|昌平橋駅}}。同年10月12日制定國有鐵道線路名稱，御茶之水站所屬路線命名為中央本線<ref name = "飯田町駅ものがたり" />。

=== 總武本線 ===
1923年（大正12年）9月1日發生[[關東大地震|關東大地震]]，遭受祝融部分燒毀的御茶之水站在緊急修復後重新啟用<ref name = "バリアフリー検証_76">[[#バリアフリー検証|「JR御茶ノ水駅のバリアフリー化検証」p.76]]</ref>。神田川旁的崖地也因地震造成大規模崩落<ref name = "今昔50年_55" />。

在此之前輸送量不大的中央本線在關東大地震後，因運送重建資材與砂石，輸送量急增。另外作為失業對策事業的一環，大正末期進行御茶之水～中野間的[[複複線|複複線]]化工程<ref name = "百年史9_184-185">[[#百年史9|『日本国有鉄道百年史』9 pp.184 - 185]]</ref>。當時的總武本線是以[[兩國站|兩國站]]為起點，旅客須轉搭巴士前往其他路線車站，因此總武本線已有延長至市區連結其他路線的規劃，並以關東大地震市區重建為契機，收購土地開始建設兩國至御茶之水的高架路線<ref name = "高架線工事_846">[[#高架線工事|「御茶ノ水・両国間高架線工事に就て」p.846]]</ref>。

隨著工程動工，御茶之水站也從原地往東側的{{link-ja|御茶之水橋|御茶ノ水橋}}與{{link-ja|聖橋|聖橋}}之間移動。出入口設於御茶之水橋與聖橋兩側。月台改為兩座島式月台，連結兩國的總武本線在內側，中央本線在外側。總武本線在出御茶之水後，爬上[[千分比|千分比]]33的上坡跨越中央本線上行線<ref name = "高架線工事_847">[[#高架線工事|「御茶ノ水・両国間高架線工事に就て」p.847]]</ref>。該線連通中央本線緩行線、急行線兩線，配線上可進行折返。月台完成時全長152.2公尺，川側的中央本線上行、總武本線下行月台寬度為6.5公尺，山側的總武本線上行、中央本線下行月台寬度為5.8公尺<ref name = "画報_39">[[#画報|「御茶の水両国橋間高架線建設工事に就て」p.39]]</ref>。

由於工程需在來來往往的中央本線電車旁進行，並且必須緊鄰{{link-ja|駿河台|駿河台}}住家建造高12公尺的[[擋土牆|擋土牆]]等等，施工十分困難。施工由大倉土木（現在的[[大成建設|大成建設]]）擔任<ref name = "高架線工事_848">[[#高架線工事|「御茶ノ水・両国間高架線工事に就て」p.848]]</ref><ref name = "画報_38-39">[[#画報|「御茶の水両国橋間高架線建設工事に就て」pp.38 - 39]]</ref>。總武本線側工程於1932年（昭和7年）7月1日完成，御茶之水站開始停靠總武本線電車，接著1933年（昭和8年）9月15日御茶之水～飯田町間複複線化工程完工，中央線急行電車（現在的[[中央線快速|中央線快速]]）開始運行<ref name = "百年史9_184-185" />。

隨著總武本線工程，御茶之水也新建第二代站房<ref name = "駅のはなし_140-142">[[#駅のはなし|『駅のはなし』pp.140 - 142]]</ref>。當時的建築界吹起[[維也納分離派|維也納分離派]]風格，試圖以新型建材尋找最適合的架構，而不被過去風格所束縛<ref name = "百年史9_259">[[#百年史9|『日本国有鉄道百年史』9 p.259]]</ref>。因此，使用[[鋼|鐵]]、[[玻璃|玻璃]]、[[混凝土|混凝土]]等材料、採無裝飾實用主義的國際樣式建築誕生，也是日本現代建築運動的作品之一<ref name = "世界の駅_41-44">[[#世界の駅|『世界の駅・日本の駅』pp.41 - 44]]</ref>。新站房由[[東京大學|東京帝國大學]]建築學科畢業的[[鐵道省|鐵道省]]官員、建築家{{link-ja|伊藤滋 (建築家)|伊藤滋 (建築家)|伊藤滋}}所設計。設計當時，由於鄰近[[湯島聖堂|湯島聖堂]]，雖有他方提出尊重東洋愛好的請求，但伊藤不予採納，而是在設計上重視與聖橋（1927年完成）、御茶之水橋（1931年完成）之間的調和。伊藤也將車站視為道路的一部分，重視旅客的流動<ref name = "世界の駅_45">[[#世界の駅|『世界の駅・日本の駅』p.45]]</ref>。此概念成為以後通勤電車車站的設計基礎。

=== 地下鐵 ===
[[第二次世界大戰|第二次世界大戰]]後，國鐵進行月台延長與屋頂新設工程。1954年（昭和29年）1月20日，[[帝都高速度交通營團|帝都高速度交通營團]]（營團）丸之內線御茶之水站開業，1969年（昭和44年）12月20日[[東京地下鐵千代田線|營團千代田線]][[新御茶之水站|新御茶之水站]]也在附近開業<ref name = "今昔50年_55-57">[[#今昔50年|『中央線 オレンジ色の電車今昔50年』pp.55 - 57]]</ref>。丸之內線御茶之水站一開始研擬國鐵站駿河台側與國鐵站下方等方案，在節省建設經費的原因之下炫澤現在的神田川左岸設站。御茶之水站由於位於面對神田川的急斜坡，站房為半地下式構造，在時任營團總裁鈴木清秀「不追求不必要的宏偉華麗」方針下，採取與國鐵御茶之水站一樣的國際樣式建築。車站設計由輔佐國鐵御茶之水站設計師伊藤滋的土橋長俊所主導的土橋大野建築事務所擔任。第二次世界大戰中因鋼筋混凝土等資材不足而中斷的日本鐵道現代建築再次復活，從此開始國鐵站房多採用國際樣式建築<ref name = "遺産_54-58">[[#遺産|『東京鉄道遺産』pp.54 - 58]]</ref>。丸之內線在本站附近建設十，曾發現[[繩文時代|繩文時代]]文物及[[貝塚|貝塚]]，並命名為{{link-ja|御茶之水貝塚|お茶の水貝塚}}。

1968年（昭和43年）7月16日晚上10點30分左右，本站1號月台往[[豐田站|豐田]]的10輛編組電車的車門夾住乘客緊急停駛，後方往[[高尾站_(東京都)|高尾]]10輛編組電車的駕駛因認為先行列車已出發，切斷[[自動列車停止裝置|自動列車停止裝置]] (ATS) 電源又忽略停車信號而發生追撞事故，造成150人以上輕重傷<ref name = "朝日19680717">{{Cite news | title = 国電、御茶ノ水駅で追突 百五十人以上重軽傷 御茶ノ水駅の国電追突事故 | newspaper = 朝日新聞東京版朝刊 | date = 1968-07-17}}</ref>。

=== 車站改良工程 ===
[[國鐵分割民營化|國鐵分割民營化]]後，JR東日本在1988年（昭和63年）提出御茶之水站改建計畫，1989年以「都市型未來志向示範站」為目標，提出事業費65億日圓，賞金2000萬日圓的公開募集活動。最優秀賞的設計是座3層檢築，樓板面積約11,000[[平方公尺|平方公尺]]，1樓是列柱與拱門裝飾的大廳層，2樓是[[水族館|水族館]]與會廳，3樓是[[美術館|美術館]]。但之後由於難以在列車不中斷運行的情況下增建強壯支柱支撐上方建築，JR因此解散團隊，放棄改建計畫<ref>{{Cite news | title = 駅改築計画ストップ JRはチーム解散「新御茶ノ水駅」 | newspaper = 朝日新聞東京版朝刊 | date = 1993-10-14}}</ref>。

由於放棄計畫，1932年完成的車站構造持續使用至21世紀。車站由於位於神田川與駿河台之間的狹窄空間，從構造、位置上不容易增設無障礙設施，雖設置了輪椅用升降機，但無法增設電梯與電扶梯。由於周邊有許多大型醫院，在許多高齡乘客的請求下，2002年周邊8醫院聯合向JR東日本提出增設無障礙設施，2006年12月下旬發起電梯、電扶梯設置連署運動，2008年千代田區長也向JR東日本提出要求<ref name = "バリアフリー検証_77">[[#バリアフリー検証|「JR御茶ノ水駅のバリアフリー化検証」p.77]]</ref>。

對此，JR東日本在2010年3月26日發表本站將在2010年度年底進行無障礙設施整備。同日千代田也發表與JR東日本合作進行站前廣場整備。整備內容包括線路上方架設人工地盤，付費區內新設連絡通道，御茶之水橋口站房與聖橋口站前廣場整備。另外，聖橋口站房移至人工地盤上，並增設電梯與電扶梯等無障礙設施<ref name = "バリアフリー検証_78">[[#バリアフリー検証|「JR御茶ノ水駅のバリアフリー化検証」p.78]]</ref><ref name = "PR20100326">{{Cite web | url = http://www.jreast.co.jp/press/2009/20100322.pdf | title = JR中央線御茶ノ水駅バリアフリー整備について | publisher = 東日本旅客鉄道 | date = 2010-03-26 | format = PDF | accessdate = 2013-09-08}}</ref>。

這次的計畫，受限於本站位置，是高難度的大型工程，列車運行也有可能因此變更2013年秋季起，站內與周邊開始耐震補強工程，無障礙設施等相關整備目標在2018年度完工，站前廣場機能目標在2020年度完成<ref name = "PR20130903">{{Cite web | url = http://www.jreast.co.jp/press/2013/20130902.pdf | title = JR中央線御茶ノ水駅バリアフリー整備等の本体工事着手について | publisher = 東日本旅客鉄道 | date = 2013-09-03 | format = PDF | accessdate = 2013-09-08}}</ref>。

而JR中央線則計畫在2020年度在快速電車導入兩輛雙層綠色車廂成為12輛編組運行。停靠快速電車的1、4號月台將實施月台延長工程因應<ref>[http://www.jreast.co.jp/press/2014/20150203.pdf 中央快速線等へのグリーン車サービスの導入について] 2015年2月4日 JR東日本</ref>。

本站是[[島式月台|島式月台]]2面4線[[地面車站|地面車站]]，站房為[[跨站式站房|跨站式站房]]。月台位於御茶之水橋與{{link-ja|聖橋|聖橋}}之間，兩橋南側皆有出口，分別稱為御茶之水橋口與聖橋口。主站房在御茶之水橋側（水道橋側）。聖橋口東側（秋葉原側）有個僅在平日早晨開放的出口專用臨時剪票口。軌道位於高地擋土牆與[[神田川_(東京都)|神田川]]之間，因此月台十分狹窄。[[廁所|廁所]]在御茶之水橋口與聖橋口兩剪票口內，但無多功能廁所。

御茶之水橋口站房在1932年（昭和7年）7月總武本線通車時啟用，是由伊藤滋設計的現代建築，重視旅客流動的設計成為之後通勤電車車站的樣板<ref name = "駅のはなし_140-142" />。西側的聖橋口站房經歷大規模改建後已不見過去樣貌，御茶之水橋口站房自建設以來尚未有大幅變動<ref>[[#遺産|『東京鉄道遺産』pp.44 - 45]]</ref>，但自2013年度起開始行無障礙設施工程。

月台外側2線為中央線快速，內側2線為中央、總武線各站停車使用，同方向列車可在同月台轉乘，屬於[[複複線|方向別複複線]]。為此本站前後軌道呈現立體交會<ref name = "東京圏複々線">[[#東京圏複々線|「東京圏複々線区間の配線と運転の興味」]]</ref>。在[[三鷹站|三鷹]]～御茶之水間，只有本站可不上下樓梯轉乘同方向列車。

中央線與總武線各站停車列車在本站[[直通運轉|相互直通運轉]]，但早晨與深夜兩線分離運行。在此情況下，總武線在本站折返，中央線在本站水道橋方向實施急行線與緩行線的轉線駛往東京站，列車採用快速班次使用的[[JR東日本E233系電力動車組|E233系電車]]。總武線上行列車停靠2號月台後，接著轉入中央緩行線下行本線進入3號月台成為總武線下行列車<ref group = "注">橫渡線間隔充足，不影響1號月台停靠。</ref>。在該時段，[[千葉站|千葉]]方向往[[新宿站|新宿]]、[[中野站_(東京都)|中野]]、三鷹方向或三鷹方向往[[錦糸町站|錦糸町]]、[[津田沼站|津田沼]]、千葉方向時，都須在本站轉乘。

本站是[[對向式月台|對向式月台]]2面2線地下車站。

1號月台[[銀座站_(日本)|銀座]]方向有一條連結[[池袋站|池袋]]方向線路的單向橫渡線。從池袋～御茶之水間開業至延伸[[東京站|東京]]為止，該線作為拖上線使用，通車至[[淡路町站|淡路町]]時（當時為單線運轉）作為往池袋方向電車的橫渡線。2007年6月增設[[月台門|月台門]]，由於進行車輛與月台間隙調整工程，2008年3月22日暫時停用，23日恢復使用。

過去定期券售票處在JR口剪票口前，1993年11月移往東京站後改設繼續定期券售票機。

與JR車站不同，兩剪票口設有電梯。另外，通往1號月台的剪票口電梯可直通[[東京醫科齒科大學|東京醫科齒科大學]]。

車站由於位於面對神田川的陡坡，採用半地下構造站房設計。車站設計由輔佐國鐵御茶之水車站設計師伊藤滋的土橋長俊所主導的土橋大野建築事務所擔任，並採取與國鐵御茶之水車站一樣的國際樣式建築<ref name = "遺産_54-58" />。

過去可轉乘[[東京地下鐵千代田線|千代田線]][[新御茶之水站|新御茶之水站]]，在[[都營地下鐵新宿線|都營地下鐵新宿線]][[小川町站_(東京都)|小川町站]]開業後，新御茶之水站可直通淡路町站，因此兩站解除轉乘機制。

JR線北側緊鄰神田川。丸之內線車站位於神田川北側、御茶之水橋對岸。聖橋在東、御茶之水橋在西。車站周邊有[[明治大學|明治]]、[[日本大學|日本]]、[[順天堂大學|順天堂]]、[[東京醫科齒科大學|東京醫科齒科]]等大學，有『日本的[[拉丁區|拉丁區]]』之稱。另外，此地也聚集許多樂器店與體育用品店、醫院等。

另外，[[御茶之水女子大學|御茶之水女子大學]]的最近站是丸之內線[[茗荷谷站|茗荷谷站]]與[[東京地下鐵有樂町線|有樂町線]][[護國寺站|護國寺站]]。
