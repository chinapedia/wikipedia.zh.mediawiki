'''御徒町站'''（{{jpn|j=御徒町駅|hg=おかちまちえき|rm=Okachimachi eki|f=y}}）位於[[日本|日本]][[東京都|東京都]][[台東區|台東區]][[上野|上野]]五丁目，是[[東日本旅客鐵道|東日本旅客鐵道]]（JR東日本）[[京濱東北線|京濱東北線]]、[[山手線|山手線]]的[[鐵路車站|鐵路車站]]。京濱東北線的[[車站編號|車站編號]]是'''JK 29'''，山手線則是'''JY 04'''。

== 停靠路線 ==
停靠的路線在正式軌道名稱上只有[[東北本線|東北本線]]1條路線（詳詳參見路線條目與{{link-ja|鐵道路線的名稱|鉄道路線の名称}}），列車運轉系統則有[[京濱東北線|京濱東北線]]列車與[[山手線|山手線]]2個列車系統，旅客資訊中也不顯示「東北（本）線」。另外依據[[特定都區市內|特定都區市內]]，本站屬於「[[特定都區市內#設定區域一覧|東京都區內]]」與「[[東京山手線內|東京山手線內]]」。

== 歷史 ==
* 1925年（[[大正|大正]]14年）11月1日：[[鐵道省|鐵道省]][[東北本線|東北本線]]（[[京濱東北線|電車線]]）車站開業。
* 1949年（[[昭和|昭和]]24年）6月1日：[[日本國有鐵道|日本國有鐵道]]成立。
* 1987年（昭和62年）4月1日：[[國鐵分割民營化|國鐵分割民營化]]，成為[[東日本旅客鐵道|東日本旅客鐵道]]（JR東日本）車站。
* 1988年（昭和63年）3月13日：京濱東北線開始運行快速班次，午間時段不停靠本站。
* 1990年（[[平成|平成]]2年）1月22日：車站北側高架下的春日通發生塌陷事故，造成17人受傷。
* 2001年（平成13年）11月18日：[[IC卡|IC卡]]「[[Suica|Suica]]」啟用。
* 2015年（平成27年）3月14日：改點。周六假日的京濱東北線快速班次停靠本站。

=== 站名由來 ===
取自開業當時的町名（御徒町、仲御徒町）。御徒町的由來請參照[[御徒町|御徒町]]。

== 車站構造 ==
[[島式月台|島式月台]]2面4線的[[高架車站|高架車站]]。出口有北口與南口兩處。

1號線東側有三條[[上野東京線|上野東京線]]與回送列車使用的線路，由於設有轉轍器<ref>北側從1號線側起分別可通往上野站5號線、6號線、7至9號線。南側分別可通往東京站方向的下行線、上行線、秋葉原站附近的留置線。</ref>，上野東京線須減速通過本站。
