'''御影車站'''（{{jpn|j=御影駅|hg=みかげえき|rm=Mikage eki|f=y}}）是位於[[北海道|北海道]][[上川郡|上川郡]][[清水町_(北海道)|清水町]]御影本通1丁目的[[北海道旅客鐵道|北海道旅客鐵道]]（JR北海道）[[根室本線|根室本線]]的[[鐵路車站|鐵路車站]]。車站編號為'''K26'''。電報略號為'''ミケ'''。

車站名稱是來自該地區的產物[[花崗岩|花崗岩]]（御影石）。

== 歷史 ==
* 1907年（[[明治|明治]]40年）9月8日 - 以[[日本國有鐵道|國有鐵道]]的'''佐念頃車站'''開始營業。一般車站。
* 1922年（[[大正|大正]]11年）10月15日 - 改稱為'''御影車站'''。
* 1971年（[[昭和|昭和]]46年）10月2日 - 廢除貨物處理。
* 1984年（昭和59年）2月1日 - 廢除行李處理。
* 1987年（昭和62年）4月1日 - [[國鐵分割民營化|國鐵分割民營化]]後，由JR北海道繼承。

== 車站構造 ==
* 御影車站為擁有2個[[側式月台|側式月台]]及2條鐵路路線的地面車站。位於車站大樓旁的1號線是本線，而交換列車等則會使用2號線。
* 為無人車站，由[[新得車站|新得車站]]管理。

== 車站周邊 ==
