{{三民主義}}

《'''國父思想'''》，即為孫中山思想。是以[[孫中山|孫中山]]的1924年[[三民主義|三民主義]]演講本為基本骨架，並加上《[[孫文學說|孫文學說]]》、《[[國父全集|國父全集]]》等相關言論著作，而整理出的[[教科書|教科書]]性質書籍。1953年，臺灣的高級中學的三年級增列「三民主義」一科。而1970年代之後，為中華民國專科學校以上之必修課程的課程之一，現已刪除相關考試項目。

== 大陸時期 ==
1929年，[[國民革命軍北伐|國民革命軍北伐]]名義上統一中國後，實行公民教育。其1929年「高中課程暫行標準」公民教育課程則稱為「黨意」，以三民主義為基礎。但在1932年的「中學課程標準」，又改回稱為「公民」<ref>[http://www.ccunix.ccu.edu.tw/~edume/topic/lee.pdf 李琪明：海峽兩岸德育教育沿革與現況之比較] {{webarchive|url=https://web.archive.org/web/20150923201233/http://www.ccunix.ccu.edu.tw/~edume/topic/lee.pdf |date=2015-09-23 }}</ref>而當時在小學方面，則有[[商務印書館|商務印書館]]的「新時代三民主義教科書」為之流通。<ref>[http://www.cuhk.edu.hk/ics/21c/issue/articles/055_990157.pdf 謝泳：從小學教科書看兩黨教育理念]</ref>

== 遷台以後 ==
1953年，因應當時[[反共抗俄|反共抗俄]]國策，[[中華民國教育部|教育部]]聘請[[盧紹稷|盧紹稷]]起草「高級中學三民主義課程標準」草案。審查通過後，於隔年一月實施。但在[[台師大|台師大]]公民訓育系與[[中國文化大學|中國文化大學]]三民主義研究所設置之前，[[臺灣地區|臺灣地區]]均未有合格三民主義教師。據1975年教育部請[[台師大|台師大]]教育系行政協助調查顯示，非本科教受三民主義課程教師達46%。而三民主義也為當時[[大學聯合招生考試|大學聯考]]必考科目之一，隨著[[臺灣省戒嚴令|解嚴]]，1999年的八十九年度大學聯考廢除考三民主義一科。

而在大專院校方面，1950年起大專則恢復三民主義課程，並在1964年更名為「國父思想」，教育部本意試圖藉由大學學術研究方法，使得「三民主義學術化」。1990年在全國大學校長會議中，教育部宣布再更名為「中華民國憲法與立國精神」，並於八十二學年度實施。但在釋字380號被[[司法院|司法院]]大法官宣布違憲，則該課程轉為「通識課程」。<ref>[http://nccur.lib.nccu.edu.tw/bitstream/140.119/33208/6/45101006.pdf 吳儒佳：一個消失中的意識型態--以三民主義在台灣的傳播為例，第二章 三民主義教育之實施、變革與檢討]</ref>

== 現狀 ==
在[[中華民國教育部|中華民國教育部]]的「九五暫綱」將高中原有之「三民主義」、「公民」、「現代社會」等三科目，整微為「公民與社會」一科。三民主義作為獨立課程在[[中華民國教育|中華民國教育]]上走入歷史。
