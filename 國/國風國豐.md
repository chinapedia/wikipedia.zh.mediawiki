
'''國風國豐'''是[[香港電台第二台|香港電台第二台]]音樂節目，介紹[[大中華|大中華]]音樂的節目，以專業角度推介及分析大中華的流行音樂及歌手，擴寬聽眾的音樂選擇及認識。<ref>[http://hk.apple.nextmedia.com/entertainment/art/20060414/5831275 容祖兒昨日到香港電台接受周國豐主持的《國風國豐》訪問]，[[蘋果日報|蘋果日報]]，2006-4-14</ref>，擴寬聽眾的音樂選擇及認識<ref>[http://ent.sina.com.cn/y/p/2006-04-18/11221053708.html 刘德华计划留长发 受邀唱08奥运会主题曲]，新浪娛樂，2006-4-18</ref>。

香港電台第二台在2005年9月進行節目改革，將中午時段娛樂節目《娛樂滿天星》取消，分拆為《國風國豐》（音樂為主）及《948娛樂網》（娛樂為主）的兩個節目。《國風國豐》由資深電台DJ周國豐主持，以推介中、港、台甚至是東南亞各地華人歌曲，與及訪問樂壇華語歌手為主題。節目播出不足一年，香港電台第二台在2006年5月再次進行節目改革，《國風國豐》播出時間由中午時段改為黃昏時段。後期因周國豐兼任[[香港電台普通話台|香港電台普通話台]]行政工作，較多將節目預先錄音轉播。該電台節目最後播出日期為2011年11月18日<ref>{{Cite web |url = http://www.discuss.com.hk/viewthread.php?tid=16160745 |title = 香港討論區貼文——點解無國豐國豐 |author = bm10 |date = 2011-11-18  |language = zh-hant |accessdate = 2016-10-20 }}</ref>。播放一小時「四大天王專訪——張學友」<ref>{{Cite web |url = https://www.youtube.com/watch?v=owOHEOgYsBI |title = 2011.11.18 香港電台 國風國豐 專訪 張學友 (1/4) |author = 黃哇拉 |date = 2012-02-27  |language = zh-hant |accessdate = 2016-10-20 |quote = ……最後一晚的國風國豐　…… }}</ref>。

作為一個推動大中華音樂的起點，《國風國豐》為後來香港電台在製作「全球華語歌曲排行榜」<ref>[http://programme.rthk.org.hk/channel/radio/programme.php?p=5972 rthk.hk 香港電台網站: 全球華語歌曲排行榜 節目介紹]</ref>提供了基礎。
