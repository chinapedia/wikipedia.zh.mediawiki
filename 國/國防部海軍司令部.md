'''國防部海軍司令部'''，為[[中華民國國防部|中華民國國防部]]依《[[s:國防部組織法|國防部組織法]]》<ref group="註" name="國防部組織法">{{wikisource-inline|國防部組織法}}</ref>特設的軍事機關，負責辦理[[中華民國海軍|中華民國海軍]]軍事事務的規劃、督導及執行。現址位於[[臺灣|臺灣]][[臺北市|臺北市]][[中山區_(台北市)|中山區]]大直營區，下轄政治作戰室、督察室、人事軍務處、軍事情報處、戰備訓練處、後勤處、計劃處、戰鬥系統處、主計處等幕僚部門。<ref name="100年國防報告書">{{Cite web|url=http://2011mndreport.mnd.gov.tw/|title=100年國防報告書|author=|date=|publisher=中華民國國防部|language=zh-tw|accessdate=2012-06-25|quote=|deadurl=yes|archiveurl=https://web.archive.org/web/20120109005042/http://2011mndreport.mnd.gov.tw/|archivedate=2012-01-09}}</ref>

== 沿革 ==
*1912年4月，[[袁世凱|袁世凱]]繼任[[中華民國臨時大總統|中華民國臨時大總統]]，[[北洋政府|北洋政府]]設[[海軍部|海軍部]]。 <ref name="海軍組織沿革">{{Cite web |url= http://navy.mnd.gov.tw/Publish.aspx?cnid=3297&Level=2 |title= 海軍組織沿革 |author= |date= |publisher= 中華民國海軍司令部 |language= zh-tw |accessdate= 2012-06-25 |quote= }}</ref>
*1928年12月，[[國民政府|國民政府]]成立'''海軍署'''。
*1929年6月，國民政府裁併海軍署，成立'''海軍部'''，直隸於[[國民政府行政院|國民政府行政院]]。
*1938年1月31日，裁撤海軍部，改編為'''海軍總司令部'''。
*1945年9月，國民政府在[[軍政部|軍政部]]之下增設'''海軍處'''。
*1945年12月，國民政府撤銷海軍總司令部。
*1946年4月，海軍處擴編為'''海軍署'''，仍隸軍政部。
*1946年6月1日，海軍署擴編為'''海軍總司令部'''；同年10月16日正式成立，隸屬國防部。
*1949年5月，海軍總司令部遷[[高雄市|高雄市]][[左營區|左營區]]。
*1954年4月，海軍總司令部遷[[臺北市|臺北市]][[中山區_(臺北市)|中山區]][[大直|大直]]。
*2002年3月，海軍總司令部改稱為「'''國防部海軍總司令部'''」。
*2006年2月16日，海軍總司令部再改稱為「'''國防部海軍司令部'''」<ref name="海軍組織沿革"></ref>。
*2013年1月28日，海軍「合永專案」，自行研發建造的首艘艇「合永艇」、第二艘艇「合茂艇」及第三艘艇「合壽艇」在[[左營軍港|左營軍港]]東碼頭編成，未來正式加入兩棲登陸作戰序列，為海軍增添重要戰力<ref>[http://www.nownews.com/2013/01/28/11937-2897600.htm 海軍「合壽艇」今成軍　增添兩棲登陸戰力]</ref>。
