'''國際專案管理師'''（{{lang|en|'''Project Management Professional'''}}）<ref>http://www.pmi.org.tw/Dynamic/detail?Code=news&page=656</ref>，通常簡稱'''PMP'''，是由美国[[國際專案管理學會|國際專案管理學會]]（{{lang|en|Project Management Institute}}，PMI）組織進行的一項[[專案管理|專案管理]]領域的資格認證，PMP採取資格審查與考試相结合的方式進行，其理論體系主要依據為美國[[國際專案管理學會|國際專案管理學會]]<ref>{{Cite web|url=http://www.pmi.org.tw/|title=社團法人國際專案管理學會台灣分會 - 首頁|accessdate=2018-05-05|work=www.pmi.org.tw}}</ref>編寫的《[[專案管理知識體系指南|專案管理知識體系指南]]》（A Guide to the Project Management Body of Knowledge，PMBOK® Guide）。
截至 2018年2月底截止，全球PMP人數為 839,682人 (美國[[國際專案管理學會|國際專案管理學會]] [http://www.pmitoday-digital.com/pmitoday/april_2018?pg=4#pg4 官網數據])

== 考試摘要 ==
PMP考試規範說明五方面的考核領域<ref>https://www.pmi.org/-/media/pmi/documents/public/pdf/certifications/project-management-professional-exam-outline.pdf</ref>。

# 專案啟始(13%)
# 專案規劃(24%)
# 專案執行(31%)
# 專案監視與管制(25%)
# 專案結束(7%)

考試包括基於規範的200個單選題。上面括号中的数字是每个领域所占比重。

== 目的 ==
政府、商业和其他组织为提高專案成功率，应用PMI的《[[專案管理知識體系指南|專案管理知識體系指南]]》中标准化的和不断进化的项目管理理论，聘請[http://pmi.org PMP]认证的專案经理。

== 考试说明 ==
PMP考试通过Prometric（普尔文）公司提供的全球考试中心以电脑答题的形式提供。此外也在没有Prometric考试中心的地区提供纸质答题方式，在中国大陆，Prometric考试中心也提供纸质答题方式。考试包括200道单选题，25道是预览题目，不计入考试成绩，就是25道题无论答的对错，都不计入成绩。考试成绩由其他175道题算得。

通过电脑答题的应试者在提交答案时立即可得到成绩，通过笔试的应试者通常在4周内取得成绩报告。

帮助PMP应试者通过考试的资料有很多。教辅书籍、在线课程和教室课堂都可以在网路找到。

考点由PMI授权，培训机构也会被授权组织考点。

== 考試資格 ==
PMP考試資格，大學以上學歷需要專案經驗4500小時以上、35 小時課程訓練時數；高中以上學歷需要專案經驗7500小時以上、35 小時課程訓練時數<ref>{{Cite web|url=https://www.pmi.org/certifications/types/project-management-pmp|title=Project Management Professional Certification {{!}} PMP|accessdate=2018-05-16|work=www.pmi.org|language=en}}</ref>。
