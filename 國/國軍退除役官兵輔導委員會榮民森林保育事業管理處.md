'''國軍退除役官兵輔導委員會榮民森林保育事業管理處'''（簡稱'''退輔會森林保育處'''、'''退輔會森保處'''），為[[國軍退除役官兵輔導委員會|國軍退除役官兵輔導委員會]]所屬的農林機構，處址設於[[宜蘭市|宜蘭市]]。成立背景為1959年時，為安置[[中華民國國軍|中華民國國軍]]退除役官兵開發[[東西橫貫公路|東西橫貫公路]]沿線森林資源而設立的單位，一方面可以安置國軍退除役官兵，另一方面藉其之力開發橫貫公路沿線資源，形成雙贏的局面。

== 沿革 ==
1959年1月31日，因應[[中橫公路|中橫公路]]即將通車，如何在中橫沿線安置退除役官兵是一大問題；為此，退輔會訂定「橫貫公路沿線森林資源開發方案」，並成立「台灣橫貫公路森林開發籌備處」。同年10月10日，「'''台灣橫貫公路森林開發處'''」（後簡稱森林開發處）正式成立。<ref>行政院退輔會榮民森林保育事業管理處編，《棲蘭山森林經營計畫》（台北：退輔會，2003），頁11。</ref><br />

森林開發處之成立，與中橫開發有絕對關係。理論上，台灣的國有林地在當時隸屬[[林產管理局|林產管理局]]（[[林務局|林務局]]前身，後簡稱林管局）管轄，但退輔會因身負安置[[榮民|榮民]]的任務，中橫沿線是政府認為其人煙稀少、資源豐富，做為安置用可避免許多問題，因此特別將中橫沿線（含宜蘭至[[梨山|梨山]]、[[霧社|霧社]]至[[大禹嶺|大禹嶺]]兩條支線）兩側各十公里之天然資源，依法予以限制，作為安置榮民之用。森林開發處開始輔導榮民從事林業工作，退輔會的榮民安置計畫正式從平地進入到台灣山林。

1963年4月1日更名為「'''行政院國軍退除役官兵就業輔導委員會橫貫公路森林開發處'''」，1972年併編退輔會榮民林業工作隊。1975年更名為「'''行政院國軍退除役官兵輔導委員會森林開發處'''」，1998年2月再更名為「'''行政院國軍退除役官兵輔導委員會榮民森林保育事業管理處'''」，2013年11月再更名為「'''國軍退除役官兵輔導委員會榮民森林保育事業管理處'''」。

== 管轄範圍 ==
森林保育處從林務局接收[[棲蘭山|棲蘭山]]與[[大甲溪|大甲溪]]林區，總面積87,889[[公頃|公頃]]，其範圍跨越[[宜蘭縣|宜蘭縣]]、[[新北市|新北市]]、[[桃園市|桃園市]]、[[新竹縣|新竹縣]]、[[台中市|台中市]]等縣市。雖然所轄面積廣大，但保安林占80％左右，可開發之經濟林地只有20％、約有一萬七千公頃。

== 林保處與林務局 ==
森林保育處獲得林務局撥發之林地，但其林地管理權與林產處分權還是在林務局身上，森林保育處不管造林或者伐木都需經過林務局同意，相關林政措施的主導權還是在林務局。至於伐木雖然與安置榮民有關，但林木處分亦是林務局之權利，只是政府充許森林保育處以專案的方式，發給其伐木許可，准許其販賣林木。<ref>姚鶴年，《台灣森林史料圖文彙編》（台北：農委會，2001），頁223。</ref>因此，森林開發處（退輔會）、[[大雪山林業公司|大雪山林業公司]]（省營）與[[台灣中興紙業|台灣中興紙業]]（開發[[林田山|林田山]]，省營）為戰後伐木業三個專案伐木之機構；前後兩者不同的是，森林保育處對持有山林地之權利至今仍保有，大雪山與林田山林區最後都回歸林務局手中，因此森林保育處是台灣林業中相當特別的案例。
