'''利比亞國民解放軍'''，前身為[[利比亞|利比亞]][[人民軍|人民軍]]，為[[利比亞全國委員會|利比亞全國委員會]]所屬的[[武裝力量|武裝力量]]。利比亞人民軍是於2011年、以推翻利比亞強人[[穆阿邁爾·卡扎菲|穆阿邁爾·卡扎菲]]上校為目標而組成，並於同年五月改名為為國民解放軍。<ref>{{cite news|url=http://military.people.com.cn/BIG5/14793895.html|title=卡扎菲軍隊掀起“叛逃潮” 反對派部隊更名利比亞國民解放軍|author=王爔旋|publisher=新華網|date=2011年6月1日}}</ref>國民解放軍成員多為缺乏作戰經驗的平民，以及部分由利比亞政府軍投誠而來的官兵；平民們在開赴戰場前曾受過短期軍事訓練。<ref>{{cite news|url= http://www.zaobao.com/special/hotspot/pages/hotspot110403.shtml|title =利比亚革命军调整策略 派倒戈军人与志愿者上阵|author = 联合早报|publisher = 联合早报网 |date = 2011年4月3日}}</ref>

== 参加战争 ==
参加了[[2011年利比亚内战|2011年利比亚内战]]，成功地推翻了[[卡扎菲|卡扎菲]]政权。

參加了2012年[[叙利亚內戰|叙利亚內戰]]。

== 武器裝備 ==
[[File:People_on_a_tank_in_Benghazi1.jpg|thumb]]
利比亞國民解放軍所擁有之武器多為自政府軍所放棄的裝備，包括相當數量的[[AK-47突擊步槍|AK-47突擊步槍]]與[[火箭推進榴彈|火箭推進榴彈]]（[[RPG|RPG]]），與少數民間私下擁有的舊式武器。國民解放軍亦擁有不少擄獲而來的[[戰車|戰車]]與[[裝甲車|裝甲車]]，以及加裝防空砲與多管火箭的[[武裝改裝車|武裝改裝車]]，甚至是[[米格-21|米格-21]]、[[米格-23|米格-23]]等[[战机|战机]]。[[法国|法国]]、[[卡塔尔|卡塔尔]]、[[英國|英國]]、與[[美國|美國]]也曾提供迷彩服、突擊步槍、反裝甲火箭、通訊及醫療裝備供國民解放軍使用，如法国便曾在内战中空投超过40[[吨|吨]]的武器予國民解放軍。
