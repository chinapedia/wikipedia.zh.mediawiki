'''國家高速網路與計算中心'''（{{lang-en|National Center for High-Performance Computing}}），全名'''國家實驗研究院高速網路與計算中心'''，簡稱'''國研院國網中心'''、'''國網中心'''，是[[中華民國|中華民國]]的研究機構，1991年經[[行政院|行政院]]核准成立，1992年進駐[[新竹科學園區|新竹科學園區]]，1993年正式對外運轉提供高速計算服務，時稱'''國家高速電腦中心'''。2003年正式轉制為[[財團法人|財團法人]]並改為現名，隸屬於[[國家實驗研究院|國家實驗研究院]]，簡稱「國研院國網中心」，致力於高效能計算、儲存、網路、平台整合等技術，擁有全台灣最強大的高速計算能量和學術網路資源。國網中心目前有三大分部，分別為新竹本部、台中分部、台南分部。國網中心承接「挑戰2008：國家發展重點計畫」，引領建置台灣高品質學術研究網路（TaiWan Advanced Research and Education Network, TWAREN），並推動台灣知識格網（Knowledge Innovation National Grid, KING）。<ref>{{Cite web |url=http://www.nchc.org.tw/tw/about/ |title=有關國網中心 |accessdate=2011-06-22 |work= |publisher=國網中心 |date=2011-06-22 |deadurl=yes |archiveurl=https://web.archive.org/web/20100329193807/http://www.nchc.org.tw/tw/about/ |archivedate=2010-03-29 }}</ref>

國網中心的[[超級電腦|超級電腦]]「[[御風者|御風者]]」，在2011年6月發佈的[[TOP500|TOP500]]全球最快超級電腦中，御風者排名第42名。<ref>{{Cite web |url=http://top500.org/list/2011/06/100 |title=TOP500 List - June 2011 (1-100) |accessdate=2011-06-22 |work= |publisher=top500.org |date=2011-06 }}</ref>
<ref>{{Cite web |url=http://www.nchc.org.tw/tw/news/index.php?NEWS_ID=169 |title=「御風者」－國網中心新建超級電腦進全球前百大 開創台灣大規模高速計算新紀元 |accessdate=2011-06-22 |work= |publisher=國網中心 |date=2011-06-22 }}</ref>而同是國網中心的[[超級電腦|超級電腦]][[台灣杉_(超級電腦)|台灣杉]]二號，目前為全臺灣最大運算能量的高速[[計算機|計算機]]，在2018發佈的[[TOP500|TOP500]]全球最快超級電腦中，台灣杉二號排名第20名。<ref>{{Cite web |url=https://www.top500.org/lists/2018/11/ |title=TOP500 List - NOVEMBER 2018 |accessdate=2018-11 |work= |publisher=top500.org |date=2018-11 }}</ref>

==國網中心超級電腦==
國網中心所建置的超級電腦，自1993年6月起即以第485名進入到TOP500排行榜裡<ref>{{Cite web |url=http://www.top500.org/site/history/1248 |title=National Center for High Performance Computing - Site history |accessdate=2011-06-22 |work= |publisher=top500.org |date= }}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，期間數次取得較佳的名次，分別為2002年6月以IBM系統（IBM Cluster 1350/1350A）得到TOP500第60名，[[2007年6月|2007年6月]]，以當時新完成的「[[IRIS|IRIS（超級電腦）]]」拿到第35名，<ref>{{Cite web |url=http://www.nchc.org.tw/tw/news/index.php?NEWS_ID=61 |title=台灣高速計算效能 首度登上全球前35強 |accessdate=2011-06-22 |work= |publisher=國網中心 |date=2007-06-28 }}</ref>
之後因其他國家積極投資超級電腦，2009年11月被擠出500大行列，一直到2011年6月「[[御風者|御風者]]」才重返TOP500超級電腦排行榜名列第42名。<ref>{{Cite web|url=http://www.ithome.com.tw/itadm/article.php?c=68297|title=國網中心正式發表「[[御風者|御風者]]」超級電腦|accessdate=2011-06-22|author=蘇文彬|work=|publisher=電週文化|date=2011-06-22|deadurl=yes|archiveurl=https://web.archive.org/web/20110625144310/http://www.ithome.com.tw/itadm/article.php?c=68297|archivedate=2011-06-25}}</ref>
2012年6月18日，國網中心自主研發的超級電腦「[[Formosa_5|Formosa 5]]」，於[[歐洲|歐洲]]2012{{Link-en|國際高速計算會議|ACM/IEEE Supercomputing Conference}}（ACM/IEEE Supercomputing Conference）中公佈之第39屆TOP500中，一舉拿下第232名，蟬聯挺進全球前五百大的佳績。<ref>[http://www.nchc.org.tw/tw/news/index.php?NEWS_ID=211 【打造台灣超級電腦】國網中心Formosa 5 挺進全球前五百大排名] 即時訊息/新聞/國家實驗研究院 2012-6-20</ref>2018年，「[[台灣杉二號|台灣杉二號]]」在TOP500中取得第20名，是目前最好的成績，能源效率Green500則排名第10。<ref>{{Cite web |url=https://www.nchc.org.tw/posts/2pi7ihZyrD/%E5%9C%8B%E5%85%A7%E8%87%AA%E7%A0%94%E8%87%AA%E8%A3%BDAI%E8%B6%85%E7%B4%9A%E9%9B%BB%E8%85%A6%E3%80%8A%E5%8F%B0%E7%81%A3%E6%9D%89%E4%BA%8C%E8%99%9F%E3%80%8B%E7%B7%A0%E9%80%A0%E6%96%B0%E7%B4%80%E9%8C%84 |title=國內自研自製AI超級電腦《台灣杉二號》締造新紀錄計算能量世界排名20名 能源效率世界排名10名 |accessdate=2018-11-13 |work= |publisher=國網中心 |date=2018-11-13 }}</ref>
