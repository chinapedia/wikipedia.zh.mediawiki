'''國道站'''（{{jpn|j=国道駅|hg=こくどうえき|rm=Kokudō eki|f=y}}）是一個位於[[神奈川縣|神奈川縣]][[橫濱市|橫濱市]][[鶴見區_(橫濱市)|鶴見區]]{{link-ja|生麥|生麦}}五丁目，屬於[[東日本旅客鐵道|東日本旅客鐵道]]（JR東日本）[[鶴見線|鶴見線]]的[[鐵路車站|鐵路車站]]。[[車站編號|車站編號]]是'''JI 02'''。

== 車站構造 ==
相對式月台2面2線的[[高架站|高架站]]、鐵路高架橋下瀰漫著昭和時代的氣氛。設有簡易[[Suica|Suica]]剪票機。由於月台是彎曲的，列車與月台間會有縫隙，另外，當列車進站時，會響起警報聲。

根據JR的[[特定都區市內|特定都區市內]]制度，本站屬「橫濱市內」車站分類。

== 歷史 ==
* 1930年（[[昭和|昭和]]5年）10月28日 - 鶴見臨港鐵道的站開業（只有旅客營業）。
* 1943年（昭和18年）7月1日 - 鶴見臨港鐵道線的國有化、[[日本國有鐵道|國鐵]]鶴見線的站。
* 1971年（昭和46年）3月1日 - 無人站化。
* 1987年（昭和62年）4月1日 - [[國鐵分割民營化|國鐵分割民營化]]成為東日本旅客鐵道站。

==站名的由来==
鶴見線和第一京濱國道（[[國道15號|國道15號]]、開業當時的[[國道1號|國道1號]]）在交叉點上，因此命名。

== 其他 ==
*1949年（[[昭和|昭和]]24年）的[[黑澤明|黑澤明]][[電影|電影]]作品《[[野良犬_(1949年電影)|野良犬]]》為首，現在屢次作為電影、電視劇的外景拍攝地，也有高架橋下瀰漫異樣氣氛的車站被介紹的事。

*從照片也能看出來每棟建築物正面和右邊的中間（正好是行人信號右側）外壁的凹凸，是由於戰爭中美軍機槍掃射的彈痕。

*[[light|light]]公司發售的電腦遊戲《潮風の消える海に》主要以鶴見線海芝浦支線為舞台，國道站也成為重要的舞台之一。
