'''國際觀光振興機構'''（{{jpn|j=国際観光振興機構|hg=こくさいかんこうしんこうきこう|rm=Kokusai kankō shinkō kikō|f=h}}；[[英語|英語]]譯名：{{lang|en|Japan National Tourism Organization}}，簡寫為'''{{lang|en|JNTO}}'''）是[[日本|日本]][[國土交通省|國土交通省]]轄下的[[獨立行政法人|獨立行政法人]]，成立目的為在國際上推展日本[[旅遊|旅遊]]事業，並幫助官方（{{link-ja|觀光廳|観光庁}}）執行商業性質的旅遊事務。2003年10月1日成立，總部位於[[東京都|東京都]][[新宿區|新宿區]][[四谷|四谷]]。從2008年7月開始，在日本國內開始使用「日本政府觀光局」的通稱<ref>[https://www.jnto.go.jp/jpn/news/press_releases/080627_seifukankoukyoku.html JNTOが通称「日本政府観光局」を使用]</ref>（對[[台灣|台灣]]也使用此名），在[[華語|華語]]地區則主要使用「日本國家旅遊局」通稱。

== 沿革 ==
* 1893年（[[明治|明治]]26年） - 日本成立首個吸引外國遊客來日的斡旋機構「'''喜賓會'''」（{{lang|ja|喜賓会}}）
* 1912年（[[大正|大正]]元年） - 成立「'''日本旅客局'''」（{{lang|ja|ジャパン･ツーリスト･ビューロー}}）
* 1930年（[[昭和|昭和]]5年）4月 - 於[[鐵道省|鐵道省]]轄下設立「'''國際觀光局'''」（{{lang|ja|国際観光局}}）
* 1931年（昭和6年）12月 - 成立「'''（財）國際觀光協會'''」（{{lang|ja|（財）国際観光協会}}）
* 1942年（昭和17年）11月 - 廢原鐵道省的「國際觀光局」
* 1942年（昭和17年）12月 - 原「日本旅客局」改組為「'''東亞旅行社'''」（{{lang|ja|東亜旅行社}}）
* 1943年（昭和18年）12月 - 解散「（財）國際觀光協會」，並將其職務與「東亞旅行社」統合為「'''（財）東亞交通公社'''」（{{lang|ja|（財）東亜交通公社}}）延續
* 1945年（昭和20年）9月 - 原「（財）東亞交通公社」更改為「（財）日本交通公社」（{{lang|ja|（財）日本交通公社}}）
* 1946年（昭和21年）6月 - 設置「[[運輸省|運輸省]]鐵道總局觀光課」（{{lang|ja|運輸省鉄道総局観光課}}）
* 1949年（昭和24年）6月 - 昇格為「運輸省大臣官房觀光部」（{{lang|ja|運輸省大臣官房観光部}}）
* 1955年（昭和30年）8月 - 昇格為「運輸省觀光局」（{{lang|ja|運輸省観光局}}）
* 1955年（昭和30年）6月 - 從「（財）日本交通公社」分離並成立「（財）國際觀光協會」
* 1959年（昭和34年）4月 - 「（財）國際觀光協會」與「（社）全日本觀光聯盟」統合為「'''（特）日本觀光協會'''」（{{lang|ja|（特）日本観光協会}}）
* 1964年（昭和39年）4月 - 成立「'''特殊法人國際觀光振興會'''」（{{lang|ja|特殊法人国際観光振興会}}）。同時，「'''（特）日本觀光協會'''」（{{lang|ja|（特）日本観光協会}}）則從「特殊法人國際觀光振興會」及「（社）日本觀光協會」分離
* 2003年（平成15年）10月 - 改組為「'''獨立行政法人國際觀光振興機構'''」

== 宗旨 ==
在海外地區作宣傳、對國外旅客進行觀光介紹，及對促進國外旅客來訪有效地進行必要業務，振興國際觀光事業。

== 視野及目標 ==
視野： 通過振興入境旅遊業，以旨實現「觀光立國」目標。

目標： 對{{link-ja|日本旅遊活動|ビジット・ジャパン・キャンペーン}}（VJC）作出貢獻，於2010年之前實現訪日國外旅客達1,000萬人次目標。

== 主要事業活動 ==
#促進國外遊客到訪
#策劃歡迎國外遊客
#代理舉辦通譯案內士試驗事宜
#對國際觀光進行調查及研究
#出版與國際觀光相關書籍
#促進國際會議的舉辦並使其順利進行
