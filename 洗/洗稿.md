“'''洗稿'''”一词流行于新闻传媒行业内部，是脱胎于“[[洗钱|洗钱]]”的新词，指新闻传媒（特别是'''新闻网站'''）及'''网络文学行业'''通过一系列手段将稿子由“黑”洗“白”，掩盖其真实来源，争取审查时间差或躲避著作权。<ref name=":0">{{Cite web|url = http://data.chinaxwcb.com/epaper2014/epaper/d5705/d7b/201401/41514.html|title = “洗稿”转载能否免责？——追踪财新传媒为记者作品维权事件|accessdate = 2015-10-30|author = 魏芳|date = 2014-01-23|publisher = 中国新闻出版报}}</ref>

中国大陆的洗稿现象分为三种，即'''规避新闻发布政策的洗稿'''、'''以非法转载为目的的洗稿'''，以及'''网络文学方面的洗稿'''。

== 新闻发布类 ==

=== 规避政策 ===
中国大陆的互联网新闻政策{{noteTag|name=policy|1=早在2000年，国务院新闻办公室和信息产业部发布《互联网站从事登载新闻业务管理暂行规定》，里面提到“非新闻单位依法建立的综合性互联网站，……经批准可以从事登载中央新闻单位、中央国家机关各部门新闻单位以及省、自治区、直辖市直属新闻单位发布的新闻的业务，但不得登载自行采写的新闻和其他来源的新闻。”2005年中华人民共和国国务院新闻办公室公布的《互联网新闻信息服务管理规定》第三章第十六条规定：“……互联网新闻信息服务单位，转载新闻信息或者向公众发送时政类通讯信息，应当转载、发送中央新闻单位或者省、自治区、直辖市直属新闻单位发布的新闻信息，并应当注明新闻信息来源，不得歪曲原新闻信息的内容。……互联网新闻信息服务单位，不得登载自行采编的新闻信息。”}}<ref>[http://www.gov.cn/flfg/2005-09/29/content_73270.htm 《互联网新闻信息服务管理规定》].中华人民共和国中央人民政府.[2005-09-29].</ref>限制了各大媒体报道新闻的权利，新闻网站若不在国家[[网信办|网信办]]官网公布的[http://www.cac.gov.cn/2015-05/05/c_1115179188.htm 《可供网站转载新闻的新闻单位名单》]中，将不具备新闻采编权。

根据互联网新闻信息服务单位分类{{noteTag|name=type|1=（一）新闻单位设立的登载超出本单位已刊登播发的新闻信息、提供时政类电子公告服务、向公众发送时政类通讯信息的互联网新闻信息服务单位；
（二）非新闻单位设立的转载新闻信息、提供时政类电子公告服务、向公众发送时政类通讯信息的互联网新闻信息服务单位；
（三）新闻单位设立的登载本单位已刊登播发的新闻信息的互联网新闻信息服务单位。}}，二类新闻网站（搜狐网、新浪网、凤凰网等商业门户网站）发布的时政类新闻，必须来自《可供网站转载新闻的新闻单位名单》。<ref>{{Cite web|url = http://www.mediaob.com/opinion/2015/0506/1717.html|title = 规范转载“白名单”解读：洗稿是违规行为|accessdate = 2015-10-30|author = 编辑网(公众号)|date = 2015-05-06|publisher = 传媒大观察}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

所以，当网络媒体想要发表非正规来源的稿件时，就想办法让白名单<!-- 《可供网站转载新闻的新闻单位名单》 -->中的网站先转载这个稿子，然后再从白名单网站中把这个稿子转出来。

=== 非法转载 ===
对于原发布单位禁止转载的稿件，通过数次转载来隐匿作品最初诸多信息而将其发表。

据业内人士介绍，一种可能的“洗稿”方式是：先以网友的身份将稿件转贴至论坛，再从论坛转载至新闻频道，出处注为“某某论坛”；或者直接转载过来，但出处注为“某某网”，这样原媒体追踪侵权也比较困难了。<ref name=":0" />

这种行为类似于将他人稿子改头换面而隐去原作者信息后重新发表，本质上是一种比较隐晦的剽窃和抄袭行为。在洗稿过程中，一些转载媒体在转载后隐去文章的原作者，甚至对一些内容进行了增删、修改，还可能涉嫌侵犯相关权利人的署名权、修改权和保护作品完整权。<ref>{{Cite web|url = http://opinion.caixin.com/2014-01-20/100631025.html|title = 汪涌：“洗稿”侵权性质更恶劣|accessdate = 2015-10-30|author = 财新实习记者 田园|date = 2014年1月20日|publisher = 财新网}}</ref>

== 网络小说类 ==
除新闻媒体的洗稿外，网络小说行业的洗稿、抄袭现象层出不穷。有些小说在总体内容与他人作品的节奏和大纲相似，仅仅更改了背景和一些细节后就重新发布。或是将他人作品的故事桥段、模式和情节化用在自己的作品里，这称之为“融梗”。这就给抄袭的认定带来了挑战。《[[花千骨|花千骨]]》、《[[三生三世十里桃花_(小说)|三生三世十里桃花]]》、《[[锦绣未央|锦绣未央]]》等已经被改编成电影、电视剧的网络小说都被曝光涉嫌抄袭他人作品。<ref>{{Cite web|url=http://tech.sina.com.cn/i/2017-09-28/doc-ifymfcih7255433.shtml|title=从摘抄到洗稿、融梗 网络文学为何频现“抄袭门”|accessdate=2018-11-12|work=tech.sina.com.cn}}</ref>
