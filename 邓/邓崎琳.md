'''邓崎琳'''（{{bd|1951年|11月|}}），[[湖南省|湖南]][[津市市|津市]]人，1978年4月加入中国共产党。1975年10月参加工作。研究生学历，硕士学位。教授级高级工程师。

== 生平 ==
1975年10月毕业于[[武汉科技大学|武汉科技大学]]冶炼专业。1978年4月加入[[中国共产党|中国共产党]]。1999年晋升为教授级高级工程师。1992年7月任[[武汉钢铁|武钢集团]]总经理助理。1995年4月任武钢集团副总经理。1999年4月任武钢集团党委常委、副总经理。2001年8月任武钢集团党委常委、副总经理、钢铁有限公司总经理。2004年12月任武钢集团总经理、党委副书记。2013年7月任武钢集团董事长、党委书记。是第十一届、十二届全国人大代表<ref>{{cite web|author=新华网|title=第十一届全国人大代表|url=http://news.xinhuanet.com/ziliao/2008-03/03/content_7708230.htm?node=36628|publisher=新华网|accessdate=2014-01-07}}</ref><ref>{{cite news|title=全国人大代表信息-邓崎琳|url=http://www.npc.gov.cn/delegate/viewDelegate.action?dbid=111563|accessdate=2013-04-05|newspaper=中国人大网}}</ref><ref>{{cite news|title=全国人大代表信息-邓崎琳|url=http://www.npc.gov.cn/delegate/viewDelegate.action?dbid=121546|accessdate=2013-04-05|newspaper=中国人大网|deadurl=yes|archiveurl=https://web.archive.org/web/20140505055309/http://www.npc.gov.cn/delegate/viewDelegate.action?dbid=121546|archivedate=2014-05-05}}</ref>。2015年6月，不再担任武钢集团董事长、党委书记。2015年8月29日，[[中国共产党中央纪律检查委员会|中纪委]]网站宣布，邓崎琳涉嫌严重违纪违法，正接受组织调查<ref>{{cite web|url=http://www.ccdi.gov.cn/jlsc/201508/t20150829_61216.html|title=武汉钢铁(集团)公司原董事长、党委书记邓崎琳被调查|publisher=中央纪委监察部网站|accessdate=2015-08-29|deadurl=yes|archiveurl=https://web.archive.org/web/20150917202346/http://www.ccdi.gov.cn/jlsc/201508/t20150829_61216.html|archivedate=2015-09-17}}</ref>。2015年12月，全国人大常委会接受其辞去代表职位<ref name=2015-12a>{{cite news|author1=全国人民代表大会常务委员会|title=全国人民代表大会常务委员会公告 〔十二届〕第十六号|url=http://www.npc.gov.cn/npc/xinwen/2015-12/28/content_1957461.htm|accessdate=2016-01-09|agency=全国人民代表大会常务委员会|date=2015年12月27日}}</ref>。2016年1月8日被开除党籍，移送司法机关<ref>{{cite web |url = http://www.ccdi.gov.cn/yw/201601/t20160108_72264.html |title = 武钢集团原党委书记、董事长邓崎琳被开除党籍 |accessdate = 2016年1月8日 |publisher = 中央纪委监察部网站}}</ref>。2017年5月31日，广东省佛山市中级人民法院一审公开宣判，邓崎琳犯受贿罪，涉案金额达5539万元，判处有期徒刑15年，并处罚金人民币500万元<ref>{{Cite web|url=http://news.sina.com.cn/c/nd/2017-05-31/doc-ifyfqvmh9722257.shtml|title=武钢原董事长邓崎琳受贿5539万元被判15年|accessdate=2017-05-31|author=|date=2017-05-31|work=|publisher=新浪新聞}}</ref>。
