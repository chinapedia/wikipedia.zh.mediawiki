'''鄧森山'''（1993年－2009年8月2日）男，[[中华人民共和国|中國]][[广西壮族自治区|广西壮族自治区]][[桂林市|桂林市]][[資源縣|資源縣]]人。因接受戒除[[网络成瘾症|网瘾]]治療而死亡，死亡事件引起巨大爭議。父亲邓飞，母亲周娟<ref name='Trial'>{{cite news| first=| last=| coauthors=| authorlink=| title=打死网瘾少年 四名教官受审| date=2010-02-02| publisher=| url=http://epaper.nddaily.com/A/html/2010-02/02/content_1006785.htm| work=南方都市报| pages=| accessdate=2010-02-02| language=zh-hans}}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

== 生前健康狀況 ==
身高155厘米，體重65公斤。鄧森山的父亲称，孩子身体強壯，只有网瘾，無[[傳染病|傳染病]]，无[[犯罪记录|犯罪记录]]。

7月31日曾到[[北海市|北海市]]游泳，可游出[[海岸線|海岸線]]100多米。並在距離海岸線五六十米的距離救回一名中年婦女。<ref name = ZGW/>

== 網癮治療訓練 ==
2009年8月1日下午，鄧被父母帶至“[[南寧起航拯救訓練營|南寧起航拯救訓練營]]”參加訓練<ref name='NFZM'>{{cite news | url=http://www.infzm.com/content/32355 | title= 网瘾少年进"训练营"13小时后死亡 多处伤痕 | work =南方週末 | accessdate = 2009-08-10 | language = zh-hans }}</ref>，計畫為期一個月，費用7000元[[人民幣|人民幣]]<ref name='ZGW'>{{cite news | author=刘发丁 | coauthors= |authorlink= | title=广西一名未满16岁“网瘾”少年被辅导教师暴打致死 | date=2009-08-02 | publisher= | url =http://www.cnr.cn/gundong/200908/t20090802_505418349.html | work =中国广播网 | pages = | accessdate = 2009-08-10 | language = zh-hans }}</ref>。但是南宁起航拯救训练营並無合法招生資質<ref name = Xinhua/>。事發后，當地警方刑事拘留了四名訓練營的[[教官|教官]]。

== 死亡 ==
* 8月1日，鄧森山當晚參加了訓練<ref name='Xinhua'>{{cite news| author=蒋雪林| coauthors=刘万强| authorlink=| title=广西网瘾少年死亡事件13嫌犯被刑拘 学校被取缔| date=2009-08-07| publisher=| url=http://news.xinhuanet.com/edu/2009-08/08/content_11845088.htm| work=新華網| pages=| accessdate=2009-08-10| language=zh-hans| deadurl=yes| archiveurl=https://web.archive.org/web/20090812054137/http://news.xinhuanet.com/edu/2009-08/08/content_11845088.htm| archivedate=2009-08-12}}</ref>，因未完成教官指令被關禁閉<ref name = ZGW/><ref name = NFZM/>，在禁閉室遭到教官毆打<ref name = chinanews/>。
* 8月2日凌晨兩點多，鄧發生[[嘔吐|嘔吐]]現象<ref name = Xinhua/>。
* 凌晨3：15鄧被送往吳圩衛生院搶救<ref name = Xinhua/>。
* 3:40，搶救無效死亡<ref name = Xinhua/>。死者身上有多處外傷<ref name = Xinhua/>。

== 屍檢結果 ==
广西[[南宁市|南宁]]警方8月18日晚上通报了邓森山的死亡原因。法医鉴定结论为<ref name='chinanews'>{{cite news | first= | last= | author= 刘万强 |authorlink= | title=广西警方通报"网瘾少年"死因 身体多处挫伤致死 | date=2009-08-18 | publisher= | url =http://www.chinanews.com.cn/gn/news/2009/08-18/1824138.shtml | work =中国新闻网 | pages = | accessdate = 2009-08-19 | language = zh-hans }}</ref>：
{{quote|
邓森山系由于全身多处软组织挫伤致创伤性呼吸窘迫综合征致呼吸、循环衰竭而死亡。

== 事件結果 ==
=== 政府调解 ===
事件經[[南宁市|南宁市]][[江南区_(南宁市)|江南區]]政府調解，而并非經法院判決，結果如下<ref>{{cite news |author=刘发丁 |coauthors= 张垒 |authorlink= |title = 广西“网瘾少年”家属获赔百万 |date=2009-08-27 |publisher= |url = http://www.cnr.cn/gxfw/xwzx/gxyw/200908/t20090827_505447532.html |work = 中广网 |pages = |accessdate = 2009-09-02 |language = zh-hans }}</ref>——
* “相關責任方”向死者家屬賠償人民幣104.5萬元，并達成其他協議，其中包括：
** 广东番禺励志体育策划服务部（南宁起航训练营）向死者家属赔偿人民币50万元；
** 與上者合作办营的广西电子技工学校支付40万“补助款”。
* 另有“死者家属在邕期间的吃住费用共约18万元”，由江南区人民政府支付。

=== 法院审判 ===
2010年2月1日，邓森山在训练营被打死一案在广西南宁开庭。4名教官因涉嫌[[故意伤害罪|故意伤害罪]]被[[检察院|检察院]]提起公诉<ref name="Trial" />。

殴打邓森山的4名教官，年龄从19岁到23岁不等。其中两人是广西某[[师范学院|师范学院]]的在校学生，自称利用暑假出来打工。据教官称，校方对学员采取严厉的惩罚措施，但在对外宣传上谎称“不虐待，不打骂”<ref name="Trial" />。

== 事件影響 ==
繼前不久揭露的[[楊永信|楊永信]]電擊事件，鄧森山死亡事件發生后，在中國大陸，關於[[網癮|網癮]]的爭議劇烈升級，民眾和專家都熱議網癮話題，引發媒體進行了更多的相關專題報導。
