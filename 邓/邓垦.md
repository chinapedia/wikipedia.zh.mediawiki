'''邓垦'''（{{bd|1911年|11月|2017年|10月15日}}）原名'''邓先修'''，[[四川|四川]][[广安|广安]]人，[[中国共产党|中国共产党]]及[[中华人民共和国|中华人民共和国]]官员，[[湖北省人民政府|湖北省人民政府]]副省长，[[邓小平|邓小平]]胞弟<ref name=shishi/>。

==生平==
1911年11月邓垦生于[[四川|四川]][[广安|广安]]<ref name=shishi/>。1931年，进入[[上海|上海]][[国立暨南大学|国立暨南大学]]历史社会学系学习，与[[江上青|江上青]]为同窗。当时暨南大学文学院院长[[陈钟凡|陈钟凡]]聘请了[[邓初民|邓初民]]、[[李达_(哲学家)|李达]]、[[钱亦石|钱亦石]]等左派教授任教。1933年春因国民党当局干涉，暨南大学开除100多名左派学生，大多数是中共党员和共青团员。1933年冬，国民政府教育部下令将陈钟凡院长及左派教授一起赶走。1933年下学期，暨南大学在教育部的压力下，将历史社会学系改组为历史地理学系，删除了敏感的“社会学”字眼。在这种形势下，邓垦、江上青均未毕业便提前离校，1934年上学期该系仅有三名毕业生<ref>{{Cite web|url=http://cpc.people.com.cn/GB/64162/64172/64915/5030729.html|title=专家披露珍贵史料——邓小平胞弟与江泽民叔父曾是暨大同窗|date=2006-11-13|publisher=中国共产党新闻网}}</ref><ref>{{Cite web |url = http://news.jnu.edu.cn/sdjn/wh/2015/03/04/17241534983.html  |title =邓小平之弟邓垦与江泽民之养父江上青在暨南同窗  |author =  |date = 2015-03-04 |publisher =暨南大学新闻网  |language =  |accessdate =  |quote =  }}</ref>。

1935年9月邓垦加入[[中国共产主义青年团|中国共产主义青年团]]，1937年5月加入[[中国共产党|中国共产党]]。1939年9月到1940年3月，在[[延安|延安]][[抗日军政大学|抗日军政大学]]学习。1940年3月任延安抗日军政大学图书馆干事。1941年9月任延安解放日报社编辑。1945年6月起，历任东北吉林地委宣传部长，吉林分省委宣传部副部长、吉林地委宣传部部长。1947年春，任东北[[勃利县|勃利县]]县长。1948年春，任东北[[佳木斯市|佳木斯市]]市长、市委书记。1949年12月，任四川[[泸州|泸州]]专署专员。1951年6月，任[[重庆市|重庆市]]教育局党委书记兼副局长。1952年，任重庆市教育局局长。1956年12月，任重庆市副市长。1966年6月，任[[武汉市|武汉市]]副市长，1966年8月下放[[五七干校|“五七”干校]]劳动。因与邓小平的关系而被长期关押。1973年1月任武汉市革命委员会文教办公室主任、武汉市革命委员会副主任、中共武汉市委委员、常委。1979年3月起，历任中共武汉市委副书记、副市长，武汉市委书记、武汉市政协主席。1981年12月任[[湖北省人民政府|湖北省人民政府]]副省长<ref name=huanqiu>{{Cite web|url=http://paper.people.com.cn/hqrw/html/2008-08/16/content_104726.htm|title=“哥哥为革命不回家”|date=|publisher=环球人物2008年8月（下）|language=}}</ref><ref name=shishi/>。

1982年从一线领导岗位退下。1982年11月任湖北省人民政府顾问。1983年11月任中共湖北省顾问委员会委员、湖北省人民政府顾问。1984年9月任中共湖北省顾问委员会委员、湖北省人民政府顾问、湖北省对外友好协会会长。1986年3月任中共湖北省顾问委员会委员、湖北省对外友好协会会长。1986年5月任湖北省对外友好协会会长，1986年12月离休<ref>{{Cite web |url = http://news.ifeng.com/a/20171030/52861996_0.shtml |title = 邓小平弟弟邓垦逝世享年106岁 曾任湖北省副省长 |author =  |date =2017-10-30  |publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref>。

2017年10月15日，邓垦在[[深圳|深圳]]病逝，享年106岁<ref name=shishi>{{Cite web|url=http://news.163.com/17/1030/22/D21GB6FC0001875N.html|title=邓小平弟弟邓垦逝世享年106岁 曾任湖北省副省长_网易新闻|date=2017-10-30|last=网易|work=}}</ref>。

==家庭==
*父：邓绍昌，先后娶四个妻子，共生育九个子女，活到成年的有七个（四男三女）。
*母：淡邓氏，邓绍昌第二任妻子，二人育有五个子女，邓垦是次子。
**长姐：邓先烈
**长兄：[[邓小平|邓小平]]，原名邓先圣
**二姐：邓先珍，夭折
**三弟：邓先治，又名邓蜀平
*继母：萧氏，邓绍昌第三任妻子，二人育有一子。
**四弟：邓先清
*继母：夏伯根，邓绍昌第四任妻子，与前夫唐惠民育有一女，与邓绍昌育有二女。
**妹：邓先芙，夏伯根与前夫唐惠民所生
**妹：邓先蓉，夭折，夏伯根与邓绍昌所生
**妹：[[邓先群|邓先群]]，夏伯根与邓绍昌所生
*妻：丁华
