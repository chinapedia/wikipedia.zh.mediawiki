'''邓卫平'''（{{bd|1955年|11月||}}），[[江苏|江苏]][[南通|南通]]人，[[中纪委|中纪委]]驻公安部纪检组组长。

== 生平 ==
1969年12月参加工作，1973年7月加入[[中国共产党|中国共产党]]，福建广播电视大学在职大专班中文专业毕业，在职大专学历。

早年参加[[中国人民解放军|中国人民解放军]]，为[[福建省军区|福建省军区]]独立2师4团2营4连战士、副班长、班长。退伍后分配到[[福建省|福建省]][[闽清县|闽清县]]第二瓷厂，任[[政工|政工]]组工作人员。1977-78年，任闽清县委[[整党|整党]]办、团县委工作人员。1978-83年，任福建省[[莆田|莆田]]地委组织部干部科干部。1983年夏，进入中共[[福州市|福州市]]委组织部工作，历任干部二科干部、副科长、科长。1990年11月，转任中共福州市鼓楼区委副书记。1993年8月，任中共福州市郊区区委副书记、政法委书记。1995年11月，任中共福州市晋安区委书记。1998年9月，任福州市国家安全局局长、党委书记。2006年5月，任[[福建省国家安全厅|福建省国家安全厅]]副厅长、党委委员。2009年4月，任福建省国家安全厅副厅长、党委委员。2011年9月，任中共福建省纪委副书记。2014年1月，调任[[中共广西壮族自治区党委|中共广西壮族自治区党委]]常委、自治区纪委书记。2015年3月，转任[[公安部|公安部]]纪委书记、党委委员<ref>{{cite web |url = http://news.qq.com/a/20150328/035506.htm |title = 邓卫平任公安部纪委书记：曾在福建工作近45年 |publisher = 腾讯网 |accessdate = 2015-03-28}}</ref>；次月，兼任公安部督察长<ref>{{cite web |url = http://news.qq.com/a/20150328/035506.htm |title = 公安部纪委书记邓卫平兼任公安部督察长 |publisher = 中国经济网|accessdate=2015-05-05}}</ref>。2016年1月，改任[[中纪委|中纪委]]驻公安部纪检组组长<ref>{{cite web |url = http://news.163.com/16/0116/10/BDEOL91G00014AED.html |title = 公安部原纪委书记邓卫平出任中纪委驻公安部纪检组组长 |publisher = 网易 |accessdate = 2016-01-16}}</ref>。
