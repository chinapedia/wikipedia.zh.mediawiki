'''邓处讷'''（{{bd|？||894年|catIdx=D}}），[[表字|字]]'''冲韫'''，[[中国|中国]][[唐朝|唐朝]]末年军阀，[[景福_(唐朝)|景福]]二年（893年）起作为[[节度使|节度使]]掌控[[武安军|武安军]]直至[[乾宁|乾宁]]元年（894年）被[[刘建锋|刘建锋]]败杀，后者夺取武安军。

== 家世和效力闵勖 ==
邓处讷生年不详，[[邵州|邵州]]龙潭人。年轻时是本州士兵，积功为军校，[[乾符|乾符]]年间跟着[[江西|江西]]将[[闵勖|闵勖]]去[[越南|安南]]征讨蛮族、防秋。[[黄巢之乱|黄巢之乱]]期间，盗贼杀[[湖南观察使|湖南观察使]][[李巢|李巢]]。[[中和_(唐朝)|中和]]元年（881年）十二月，闵勖率部返回，路过[[潭州|潭州]]，逐湖南观察使[[李裕|李裕]]，召诸州戍校说：“天下未定，今与君等保护州邑，待天子之命，如何？”众人称善，推举闵勖为[[留后|留后]]接管[[湖南|湖南]]。<ref>[[:s:資治通鑑/卷254|《资治通鉴》卷二百五十四]]</ref>邓处讷想回邵州省亲，闵勖以邓处讷为邵州兵马留后。<ref name=NBT186>[[:s:新唐書/卷186|《新唐书》卷一百八十六]]</ref><ref name=h9k>《九国志·邓处讷传》</ref>

== 接管武安军 ==
[[光启|光启]]二年（886年），[[衡州|衡州]][[刺史|刺史]][[周岳|周岳]]攻潭州，闵勖迎已在[[蔡州|蔡州]]一带称帝的[[秦宗权|秦宗权]]的部将[[黄皓_(唐)|黄皓]]助其守城御敌。但黄皓入城后即杀了闵勖。周岳随后败杀黄皓，接管当时已被重命名为[[钦化军|钦化军]]的湖南，<ref name=h9k/>最终被任为节度使，其时钦化军也被改名武安军。<ref>[[:s:資治通鑑/卷256|《资治通鉴》卷二百五十六]]</ref>

邓处讷得知闵勖死讯，认为罪在周岳，着缟素为闵勖之死哀哭，对入吊诸将们说：“闵仆射驻军长沙，亲自抵抗盗贼，布衣粝食养士卒，救民于涂炭，不幸为鼠辈所图，我与公等都受仆射大恩，现在周岳无故杀之，我欲与公等竭一州之力为仆射报仇，可以吗？”诸将都同意：“留后为善人雪冤，敢不从命！”于是邓处讷练兵七年，景福二年（893年）十二月以二千精兵联合[[朗州|朗州]]刺史[[雷满|雷满]]攻潭州，克之，斩杀周岳，枭首，自称留后。<ref name=NBT10>[[:s:新唐书/卷10|《新唐书》卷十]]</ref>抚慰将吏，内外都很高兴。潭州人报知朝廷，邓处讷被任为湖南节度兵马留后。<ref name=h9k/>乾宁元年（894年）二月，[[唐昭宗|唐昭宗]]任邓处讷为全权节度使。<ref name=ZZTJ259>[[:s:資治通鑑/卷259|《资治通鉴》卷二百五十九]]</ref>

== 败亡 ==
五月，刘建锋、[[马殷|马殷]]率领的军队接近武安军。邓处讷遣本乡豪族首领邵州指挥使[[蒋勋|蒋勋]]、[[邓继崇|邓继崇]]率步骑三千守[[龙回关|龙回关]]以御之。马殷说服蒋勋、邓继崇，刘建锋的成功是被预言的、不可抵挡的，于是蒋勋、邓继崇解散军队。马殷取其衣甲，直趋潭州。潭州守军以为他们是邵州兵，不做防备。马殷径入城中军府，邓处讷正在内府设宴，毫无防备，被裨将宋全节所擒，斩于长史桥侧，<ref name=h9k/>刘建锋接手军镇。<ref name=NBT10/><ref name=ZZTJ259/><ref>[[:s:新唐書/卷190|《新唐书》卷一百九十]]</ref>
