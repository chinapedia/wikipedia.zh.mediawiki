'''瑶里'''，古名'''窑里'''，位于[[江西省|江西省]][[景德镇市|景德镇市]][[浮梁县|浮梁县]]东北部，距景德镇市区约55公里，北部与安徽省的[[休宁县|休宁县]]毗邻；东部与江西省的[[婺源县|婺源县]]接壤。瑶里始建于西汉末年，因陶瓷发祥地而得名，现在是[[中国历史文化名镇|中国历史文化名镇]]之一。瑶里还以茶叶闻名，素有“瓷之源、茶之乡、林之海”之美誉<ref>{{cite web |url=http://v.youku.com/v_show/id_XNzUwNTExMg==.html |title=中央电视台：瑶里风光 |accessdate=2009-08-28 |author= |publisher= |language=zh-hans }}</ref>。2005年该镇被列为[[国家重点风景名胜区|国家重点风景名胜区]]、[[国家森林公园|国家森林公园]]、矿山公园、4A级旅游景区、中国首批生态村。<ref name="tea1"/>

==历史==
[[明代|明代]]以前，瑶里一直是景德镇地区的制瓷中心，之后（特别是到了[[清代|清代]]），由于长期的大规模开采，境内的[[高岭土|高岭土]]储量大量减少，制瓷的主要原料应接不济，导致瓷器作坊和瓷工南迁至景德镇。至今瑶里仍留存有古矿坑、窑址和作坊；水运码头、古驿道、商业码头、集镇、村落等遗迹<ref>{{cite web |url=http://www.jxcn.cn/1669/2009-3-3/30099@486396.htm |title=人间失落的天堂 |accessdate=2009-09-26|author= |publisher=中国江西网 |language=zh-hans }}</ref>。

1937年，[[中国共产党|中国共产党]]军事将领[[陈毅|陈毅]]在此将共产党领导的[[游击队|游击队]]进行军事整编，中国共产党官方称其为'''瑶里整编'''，整编的主要内容是：统一番号、思想整顿、加强军事训练和部队扩编等。整编后的游击队被编入[[新四军|新四军]]第一支队第二团第三营，陈毅任第一支队司令员。浮梁县政府于1988年在当年事发地建有“瑶里改编纪念碑”，之后又对陈毅居住过的民宅进行了修缮，并辟为“陈毅旧居”纪念馆<ref>{{cite web |url=http://www.ylfjq.com/news/News_View.asp?NewsID=124 |title=新四军瑶里改编 |accessdate=2009-09-26 |author= |publisher=瑶里风景区 |language=zh-hans |deadurl=yes |archiveurl=https://web.archive.org/web/20090814182904/http://www.ylfjq.com/news/News_View.asp?NewsID=124 |archivedate=2009-08-14 }}</ref>。

==地理==
瑶里地处[[皖|皖]][[赣|赣]]交界的山区，镇中心坐落在山谷地带，海拔约200米；北部及东北地势较高，海拔多在400米以上，与安徽省接壤的五股尖海拔1618米<ref name="gk"/>。境内生态环境良好，植被茂密且种类丰富，森林覆盖率达94%以上，其中不乏[[原始森林|原始森林]]。瑶河是瑶里境内唯一的大河，由北向南从镇中心穿过，河水清澈见底，放养了较多的观赏鱼。瑶里的气候特征与景德镇市区差异较大，冬季极端最低气温可在摄氏零下8度以下，非常寒冷，下雪也比景德镇城区大而厚；由于植被茂密，夏季白天纵然是高温，晚上也十分凉爽。

== 经济与物产 ==
[[中华民国|中华民国]]及其以前的[[明清|明清]]时期，瑶里系当年[[浮梁县|浮梁县]]重要的工商城镇，商业经济非常发达，尤其以陶瓷生产出名。[[中华人民共和国|中华人民共和国]]建立后，瑶里的陶瓷产业逐步向景德镇城区迁移，现今几乎没有陶瓷工业和陶瓷商贸活动。据[[江西省|江西省]]的官方统计数据显示，2007年瑶里镇国内生产总值为人民币20651万元，财政收入214.2万元，农民人均纯收入4429元<ref name="tea1">{{cite web |url=http://www.jxwmw.cn/zhuantipind/system/2008/09/28/010068530.shtml |title=江西省浮梁县瑶里镇介绍 |accessdate=2009-08-28 |author= |publisher=江西文明网 |language=zh-hans }}</ref>。
瑶里镇已被列入中华人民共和国之[[历史文化名镇|历史文化名镇]]，从而使其在周边地区的知名度日益显现。而今，外地人赴瑶里的旅游消费已经构成镇政府重要的财政收入。<ref name="gk"/>

瑶里盛产[[茶叶|茶叶]]，尤以[[绿茶|绿茶]]出名。每年新茶上市时，瑶里镇的商业区会云集大量商贾前来采购。苦竹豆腐是瑶里的特产，在景德镇地区具有较大名气。

== 民俗文化 ==
历史上，瑶里与安徽南部的[[休宁|休宁]]、[[祁门|祁门]]、[[歙县|歙县]]、[[黟县|黟县]]和[[屯溪|屯溪]]属于同类文化整体，即'''徽州文化圈'''，故瑶里民俗习惯接近[[皖南|皖南]]的徽派文化。徽派建筑是瑶里镇的一大特色，错落有致地分布在瑶河两岸，特别明清时期古老的建筑至今保存完好，与江西省婺源县的古建筑十分类似，尽显徽派建筑风格。每年的中国农历新年，镇上都要举办舞龙活动，场面热烈。<ref name="gk"/>据村民介绍，如果舞龙队伍进到家里，暗地扯几条“龙须”会给全家带来好运，特别是新婚夫妇将“龙须”放在枕头下则会有早得贵子的兆头。瑶里的语言属[[徽州方言|徽州方言]]，而景德镇大部分地区属[[赣方言|赣方言]]，故其语言与景德镇话迥异。

==旅游==
[[File:Watermill_remains.JPG|right]]

瑶里是景德镇市的重点旅游区，镇内建有与旅游配套的卧龙潭宾馆，主要景点有：
*瓷茶古镇游览区（明清[[徽派建筑|徽派建筑]]、[[新四军|新四军]]瑶里改编旧址）
*梅岭风景区
*汪湖原始森林
*南山瀑布（宽逾百米，落差60余米）
*徽州古道
*高岭国家矿山公园（高岭村、东埠古街）
*绕南陶瓷主题园区
