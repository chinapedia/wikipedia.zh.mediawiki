《'''忍者哈特利'''》（[[日語|日語]]：{{lang|ja|忍者ハットリくん}}，直譯即'''-{忍者服部君}-'''）為一部[[日本|日本]][[動漫|動漫]]，原作者是[[藤子不二雄Ⓐ|藤子不二雄Ⓐ]]。[[台灣|台灣]]普遍舊譯為「忍者小叮噹」，[[大然文化|大然文化]]首次代理授權版漫畫時曾翻為《忍者娃娃》。[[香港|香港]]早在1983年在[[無綫電視|無綫電視]][[翡翠台|翡翠台]]首播，而[[台灣|台灣]]一直至2001年（民國90年）才於[[台視|台視]]首播。

== 簡介 ==
主角貫藏是一名修行的少年[[忍者|忍者]]，從[[伊賀國|伊賀]]到三重市區，並寄居在三葉家，其中描述哈特利與家中獨子健一的友情…等，並介紹（或創作）許多不同的[[忍術|忍術]]，也由於忍者出現在現代社會，而引來各式各樣的騷動。內容輕鬆風趣，符合[[藤子不二雄|藤子不二雄]]一直以來的「少年遇上不可思議的人物」之公式故事。

== 登場角色 ==
=== 主要人物 ===
; 哈特利甘藏（{{lang|ja|服部貫蔵（ハットリカンゾウ）}}）
::配音：[[周婉侶|周婉侶]](1983)	/[[馮詠恩|馮詠恩]](2013-2015)/[[雷碧娜|雷碧娜]](2017~)（香港）；[[林美秀_(配音員)|林美秀]]（台灣）
: 第一男主角，通稱服部（{{lang|ja|ハットリくん}}），[[伊賀|伊賀]]流的少年忍者，[[服部半藏|服部半藏]]的子孫，外表特徵是微向上翹的嘴巴和臉頰上的螺旋形酒窩。5月5日生，金牛座，推定年齡為10歲，身高140公分，體重40公斤。(真人版的哈特利則是成年人，身高約170左右。)
: 為了修行，寄居在[[東京|東京]]的三葉家中，早期背貼在健一房間天花板睡覺，後來為了怕打擾健一，便在房間天花板上自建閣樓，作其與新藏和獅子丸的房間，並於天花板及閣樓外牆分設出入口。
: 平常穿著藍色忍者服，洗澡時會將衣服摺起來紮在頭上，但頭巾即使是洗澡也不摘下來，沒戴頭巾的樣子連弟弟新藏也曾未看過。
: 口頭禪是「你逆」（日本語羅馬拼音：Nin Nin）。
: 日本配音版對男性多以「先生」稱呼，女性則稱為「小姐」，只有與死敵煙卷不會這樣稱呼。
: 遇到胸部大的婦女易想到自己的母親而臉紅（曾被小燕子誤認為移情別戀）。
: 最糗的事情是尿床（十字[[手裏劍|手裏劍]]圖形）（95集），最害怕的事物是打針和青蛙，曾因為被-{煙卷}-抓到弱點然後企圖算計、知道自己抹了蛤蟆的油後因此差點被嚇到昏厥、後來導致皮膚過敏。
: 曾在健一的要求下挑戰過用單腳站立的金氏世界記錄，不過仍在將要破紀錄的最後一刻被-{煙卷}-算計而沒有達成。
: 另外，在第111集時，以為天上掉下來的恐怖東西是飛碟，但其實是雪。
: 煙卷和獅子丸在第107集和第112集都叫他死魚眼。

; 三葉健一（{{lang|ja|三葉ケン一（みつば けんいち）}}）
::配音：[[雷碧文|雷碧文]]（台灣）
: 副主角，戴[[近視|近視]][[眼鏡|眼鏡]]的小學生，通稱健一（{{lang|ja|ケン一（けんいち）}}）。2月28日生。
: 學業與體育都非常差勁，暗戀夢子， 與哈特利兄弟二人是好朋友，健一也常求助於貫藏的忍法。
: 也是煙卷常常捉弄的對象。

; 哈特利新藏（{{lang|ja|服部心蔵（ハットリシンゾウ）}}）
:配音：[[傅其慧|傅其慧]]→[[謝佼娟|謝佼娟]]（台灣）
: 5歲，甘藏的弟弟，通稱小新（{{lang|ja|シンちゃん}}），是見習忍者，臉頰上有紅色的圓形。
: 由於年齡幼小稚嫩，並沒有哥哥穩重。平常愛跟健一和獅子丸玩耍。
: 身穿紅色忍者服（早期設定為水色），內面穿著樹脂造的輕型護甲，使用竹製手裡劍。
: 必殺技是大哭，分貝值破表，能讓人昏迷，在第62集煙卷用忍術讓新藏忘了大哭的方法。
: 因為聲線、個性與服裝的顏色，常被誤以為是女性。

; 煙卷煙藏（{{lang|ja|ケムマキ・ケムゾウ}}）
::配音：[[魏伯勤|魏伯勤]]（台灣）
: 伊賀眼中的反派主角。[[甲賀|甲賀]]流的精銳少年忍者，甘藏的死對頭，文武雙全智計多端，雖然忍術遜於甘藏，但詭計與智謀方面為眾人之最。
: 獨居在[[東京|東京]]，與甲賀忍者貓影千代為伴。頭髮上紮有條小辮子。相較於沒有上學的哈特利等人，選擇與健一等人念同個小學班級。暗戀夢子。
: 由於甲賀的規定，必須對外隱藏忍者身份（身邊除了伊賀忍者外，只有健一知曉真實身份）。在眾人面前扮得乖巧，實際上有點壞心眼並愛惡作劇，而常捉弄健一為樂，以取悅夢子，但偶而會展露善良的一面，也偶爾有與甘藏合作的時候。
: 以忍者身分現身時穿淺綠色忍者服，不戴頭巾。
: 曾經用口香糖發明新忍術來擊敗哈特利（第80集），他也用忍術千手觀音發射火藥砲彈擊敗哈特利（第112集），但是最後以失敗收場。

; 河合夢子（{{lang|ja|河合夢子（かわい ゆめこ）}}）
:配音：[[傅其慧|傅其慧]]→[[謝佼娟|謝佼娟]]（台灣）
: 新原作及動畫版登場（舊版漫畫有類似形象角色但未列為主要人物） 
: 成績優異且異性緣佳，但愛利用男性幫她處理事情，常常使喚喜歡她的健一和煙卷，也是二人爭執的來源。興趣是製作和菓子及料理，尤其是辛辣咖哩。

; 獅子丸（{{lang|ja|獅子丸（ししまる）}}) 
::配音：[[符爽|符爽]]（台灣）
: 於新版原作漫畫及動畫版登場。
: [[伊賀|伊賀]]流的忍者狗，像綿花球一樣，頭上有十字標誌。1979年7月24日生。
: 會聽懂和模仿說人類的話，很有禮貌的傻憨而饞嘴，可把冰箱裡的食物全食光。
: 最愛吃[[竹輪|竹輪]]<ref>[[台灣|台灣]]以「忍者哈特利」為譯名的動畫版稱之為[[點心|點心]]，[[香港|香港]]則為[[獅子狗魚蛋|獅子狗魚蛋]]。</ref>。
: 武器是骨頭手裡劍，也能用四肢把身上的皮肉拉成四方形以施展忍術飛天法，拿手絕招是「憤怒火球」（此乃向讀者公開招募創作出來的獨門絕技），在第116集獅子丸使出忍術憤怒火球。流星火球攻擊煙卷和影千代。
: 日本配音版常以「哈特利主人」稱呼哈特利。
: 口頭禪是「{{lang|ja|ニョニョ}}（Nyo Nyo）」。
: 與[[甲賀|甲賀]]忍者貓影千代對立。（且常稱影千代為三腳貓）
: 在第107集時煙卷用詭計讓它和甘藏變成敵人，但最後失敗了。
: 曾經和影千代合作過，在第111、117集。

; 影千代（{{lang|ja|影千代（かげちよ）}}）
::配音：[[林美秀_(配音員)|林美秀]]（台灣）
: 於新原作及動畫版登場。
: 煙卷飼養的忍者貓，皮毛為黑色，脖子圍著紅色的絲巾，會與人類談話。
: 雖身為雌性,但有著巾幗不讓鬚眉的情操。
: 愛捉弄獅子丸（且常稱獅子丸為大笨狗），忍術也比獅子丸好一點。
: 絕招是貓咪電電攻擊，是用塑料板摩擦身上的毛造出靜電攻擊，在第107集煙卷曾經讓影千代和獅子丸合作，雙方使用貓咪電電攻擊和忍術憤怒火球擊敗貫藏。
: 在第117集假扮獅子丸的分身，和獅子丸聯手欺騙貫藏，讓貫藏以為獅子丸成功使出分身術。

; 小池老師（{{lang|ja|小池先生（こいけ）}}）
::配音：[[符爽|符爽]]（台灣）
: 健一等人班上的導師，類似角色經常出現在藤子F的作品中。
: 性格糊塗，對學生不錯，但對工作外的事情較為幼稚。目前正仰慕著愛子老師。

=== 伊賀流 ===
; 小燕子（{{lang|ja|ツバメ}}）
::配音：（台灣）[[雷碧文|雷碧文]]
: 動畫原創角色，後於新版原作逆輸入。
: 推定年齡是8歲，伊賀流女忍者。善於吹奏橫笛操控他人，但有一集她被自己的忍術打敗了。
: 暗戀甘藏，常到健一家探望他並引起騷動。

; 忍者怪獸吉波（{{lang|ja|忍者怪獣ジッポウ}}）
::配音：（台灣）[[符爽|符爽]]
: 第162集登場。
: 習得忍術的巨龜模樣的忍者怪獸，甘藏4歲時在伊賀山中修行所認識的，幼時常與甘藏一起練功、玩耍，之後因為甘藏離開[[伊賀|伊賀]]到[[東京|東京]]而感到寂寞，所以到[[東京|東京]]尋找甘藏。
: 性格溫和，食慾非常大，但一惹怒牠後果則不堪設想。
: 拿手忍術有「忍術六十四片三明治拍擊」、「忍術岩石大迴旋」、「忍術尾巴手裡劍」、「忍術煤氣噴射」等，另外與獅子丸有合體忍術「忍術憤怒岩石大火球」。
: 喜歡的食物是大福麻糬。

; 哈特利陣藏（{{lang|ja|服部陣蔵（ハットリジンゾウ）}}）
: 伊賀忍者學校的老師。

; 哈特利兄弟之母（{{lang|ja|ハットリ兄弟の母}}）

; 雲隠才藏（{{lang|ja|雲隠才蔵（クモガクレサイゾウ）}}）
: 為故事後期登場之三位忍者當中身型肥胖的。使用外型如香菸一般的忍具中產生出雲煙，然後可以製造出如筋斗雲並且隱藏在其中使出「捲雲」忍術，同時該筋斗雲也可以是雷雲。
:人物設定來自於[[真田十勇士|真田十勇士]]之一的[[霧隱才藏|霧隱才藏]]。

; 石川五助（{{lang|ja|石川五助（イシカワゴスケ）}}）
: 為故事後期登場之三位忍者當中身型瘦小的。擅長偷盜方面的忍術。外表為頭巾的結綁在鼻子上、如同過去典型的小偷一般。
: 人物設定來自於實際曾活躍在安土桃山時代之劫富濟貧的義賊[[石川五右衛門|石川五右衛門]]。

; 猿飛猿助（{{lang|ja|猿飛猿助（サルトビサルスケ）}}）
: 甘藏小時候的鄰居之一，長相跟猴子相似的少年，同時是有名的忍者猿飛佐助的後代子孫。在故事後期登場之三位忍者當中擔任首領，擅長操作猴子方面的忍術。
: 於家鄉的學校放假時來到東京探望，目前正暫時居住在祖母家。
:人物設定來自於同樣為[[真田十勇士|真田十勇士]]之一、且與[[霧隱才藏|霧隱才藏]]齊名的[[猿飛佐助|猿飛佐助]]。

=== 甲賀流 ===
; 忍金魚（{{lang|ja|忍の金魚}}）
: 外表與一般的金魚差不多，但雙眼之間有甲賀流的印記，且放入水中會游向甲賀方向。
: 能在條狀糞便上弄出類似摩斯電碼的暗號，用於傳遞情報。

; 黑貓（{{lang|ja|ハゲベエ}}）
: 新作品登場角色（動畫未登場）。由藤子Ⓐ筆名著作的作品《小黑貓》作為客串。羨慕煙卷而從甲賀來的高額頭忍者。
: 平常的時候都是在標本儲存室內負責處理標本的工作過生活。

; 白貓齋（{{lang|ja|白猫斎（はくびょうさい）}}）
: 新作品登場角色（動畫未登場）。甲賀所有忍者貓們的總長，伊賀也以「連老虎也逃走的白貓齋」來稱呼他，外觀為長毛的貓妖。
: 只有影千代特別恐懼，曾經爬在煙卷的頭上。擁有超能力，最擅長瞬間移動和心電感應。

; 煙之介（{{lang|ja|ケムノスケ}}）

=== 電視動畫 ===
由[[朝日電視台|朝日電視台]]系列首播，期間於1981年至1987年，總共694集。台灣方面由[[台灣電視公司|台灣電視公司]]首播，譯為《忍者小叮噹》，角色名稱參考《[[哆啦A夢|哆啦A夢]]》小叮噹版本、配音則沿襲《[[哆啦A夢|哆啦A夢]]》大山時期，未能對應者除外；後由[[中國電視公司|中國電視公司]]重播，亦重新翻譯及配音，改名為《-{忍者哈特利}-》，但官方僅願售前200集供播，[[八大綜合台|八大綜合台]]更連環重播幾年時間（後來則由[[MOMO親子台|MOMO親子台]]接手播映）。香港在1983年在[[無綫電視|無綫電視]]首播，名為《-{忍者小靈精}-》，並在播映時使用[[粵語|粵語]]版[[主題曲|主題曲]]〈忍者〉。<ref>{{cite web | url=http://mypaper.pchome.com.tw/mynamepier/post/1322988684| title=關於《忍者小靈精》的一些印象| accessdate=|author=}} </ref>
