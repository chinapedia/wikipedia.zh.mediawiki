『'''忍者部隊月光'''』（にんじゃぶたい げっこう）是1964年（昭和39年）1月3日至1966年（昭和41年）10月2日<ref>1966年4月 - 6月は中断。</ref>[[フジテレビジョン|フジテレビ]]系全130話放送由[[国際放映|国際放映]]製作的[[特撮|特撮]]节目。由[[吉田竜夫|吉田竜夫]]的漫画『少年忍者部隊月光』原作，時代在[[第二次世界大戦|第二次世界大戦]]中移到現代<ref name = "全怪獣">{{Cite book|和書 |date = 1990-03-24 |title = [[全怪獣怪人|全怪獣怪人]] |publisher = [[勁文社|勁文社]] |volume = 上巻 |pages=68|id=C0676 |isbn = 4-7669-0962-3 }}</ref><ref name = "画報">{{Cite book |和書 |editor=竹書房/イオン編 |date=1995-11-30 |title=超人画報 国産架空ヒーロー40年の歩み |publisher=[[竹書房|竹書房]] |pages=54 |id=C0076 |isbn=4-88475-874-9}}</ref>。
118話改名叫『'''新忍者部隊月光'''』（しんにんじゃぶたいげっこう）<ref name = "全怪獣" /><ref name = "画報" />。
==内容==
日本警方特殊部门下由[[伊賀流|伊賀流]]・[[甲賀流|甲賀流]]忍者末裔组成的忍者部队，利用忍术与科技，与国际犯罪组织和阴谋团体战斗，维护和平和正义。
