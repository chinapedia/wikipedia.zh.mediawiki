== 剧情概述 ==
五位“天才少年”被神秘导师慧眼选中，组成“世界数学大赛”攻关小组，从此早于同龄人开始苦乐交织的大学生涯。但纵使智力通天，小天才们也一样压抑不住青春的躁动，承受着荷尔蒙迸发的煎熬，在甜蜜和疼痛中成长，也终于明白生而为人的无奈……

== 制作 ==
导演肖洋少年时期就读于[[西安交通大学|西安交通大学]]少年班。2008年为著名导演[[冯小刚|冯小刚]]的电影《[[非诚勿扰_(电影)|非诚勿扰]]》[[剪辑|剪辑]]<ref>{{cite news|url=http://culture.gmw.cn/newspaper/2015-01/03/content_103438569.htm|title=冯小刚御用剪辑师肖洋 谈电影梦|date=2015-01-03|publisher=深圳特区报|accessdate=2015-01-11|deadurl=yes|archiveurl=https://web.archive.org/web/20150610175320/http://culture.gmw.cn/newspaper/2015-01/03/content_103438569.htm|archivedate=2015-06-10}}</ref>，从此知名，其电影剪辑作品三度入围金马奖。本片为他的导演处女作，创意亦来自于他在少年班的经历，知名监制陈国富不仅投资本片，还担任监制<ref>{{cite news|url=http://news.mtime.com/2015/04/08/1541312.html|title=《少年班》缘起《风声》？|date=2015-04-09 |publisher=时光网|accessdate=2015-06-10}}</ref>。

2014年9月24日剧组在[[河海大学|河海大学]]清凉门校区学生八舍前取景，6名男生只用丝巾遮面，脚踩白球鞋，手捂私处，在校园小道上一路裸奔，引发大量学生围观<ref>{{cite news|url=http://news.ifeng.com/a/20140925/42075212_0.shtml|title=南京：河海大学校园6名男生裸奔 回应称系拍电影|date=2014-09-24|publisher=凤凰网|accessdate=2015-02-18}}</ref>。

饰演方厚政的李佳奇和饰演王大法的柳希龙是导演从在东北、河南、重庆等地的武校、艺校里找了一万多名孩子中选出的。

== 评价 ==
但凡是处女作大多会成为半自传体，这条定律就连出身于天才少年班的肖洋也躲不过，他把自己中学同班的奇才同学们的故事连同自己的经历都卖了一个遍，于是拍出了这部电影。有人在里面看到了青春故事，夸赞里面终于没有撕逼和堕胎；有人在里面看到了奇人怪事，挑剔其中鼓吹“偏才教育”；但更多的人看见了曾经天才的自己，角色代入了董子健所扮演的“吴未”<ref>{{cite news|url=http://finance.sina.com.cn/roll/20150620/053922482213.shtml|title=《少年班》看天才故事，挖掘天才的自己|date=2015-06-20|publisher=新浪|accessdate=2015-06-21}}</ref>。

在这部电影中，非常细致的呈现出少年班天才儿童的真实状况，未经世事的少年在大学中的表现，天才儿童与成人世界之间的隔阂，天才儿童的世界总是那样的不可理喻。但这部电影给人很纠结的感觉，其实没说到点子上，或者说本片的导演，肖洋作为1994届少年班的成员之一，也根本没打算说到点子，只是打算为观众展示真正的少年班是什么样子<ref>{{cite news|url=http://movie.mtime.com/219379/reviews/7904114.html|title=《少年班》：天才补完计划功亏一篑|date=2015-06-21|publisher=时光网|accessdate=2015-06-21}}</ref>。

M1905电影网认为，影片在故事的叙说上模棱两可的同时也重心不稳，反倒是添加了不少趣味情节、搞笑段落；导演既忘了将故事讲圆满，也忽略了对少年们形象的复合书写，而只是描绘了几个有情欲的怪咖，除了吴未这条鲶鱼还稍微饱满一些，其他角色都扁平的就像是[[iOS_8|iOS 8]]里的图标一样，让人觉得寡淡无味。<ref >{{cite web|url=http://www.1905.com/news/20150623/904117.shtml#p1 | language= | title=《少年班》叙事重心不稳 没跳脱青春片情爱俗套 |publisher=1905电影网 |date=2015-06-23 |accessdate= }}</ref>
