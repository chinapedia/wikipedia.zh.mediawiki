《'''少狼'''》（{{lang-en|'''''Teen Wolf'''''}}）又译《'''少年狼人'''》，是一部由[[MTV|MTV电视台]]播出的[[美国|美国]][[电视剧|电视剧]]，由{{link-en|傑夫·戴維斯|Jeff Davis (writer)}}創作，故事鬆散地改编自1985年的英文同名电影《[[少狼_(電影)|-{zh-hk:狼胎小子;zh-tw:十八歲之狼;zh-cn:十八岁之狼;}-]]》，本劇讲述[[泰勒·珀西|泰勒·珀西]]所飾演的一个17岁青年Scott McCall被狼[[咬伤|咬伤]]后变成[[狼人|狼人]]的故事，而[[狄倫·歐布萊恩|狄倫·歐布萊恩]]飾演Scott的好朋友"Stiles" Stilinski。首集在2011年MTV电影奖后播出后次日晚播出第二集；之后每周一晚播出一集。该剧在[[美国|美国]]拍摄。

《少狼》於2011年6月5日首播。2016年7月21日，在[[聖地亞哥國際漫畫展|漫畫展]]上宣布第六季會是最後一季<ref>{{Cite web |url = https://variety.com/2016/tv/news/teen-wolf-final-season-6-trailer-ending-1201819854/ |title = ‘Teen Wolf’ Ending After Season 6; MTV Unveils New Trailer at Comic-Con |author = Laura Prudom |date = 2016-07-21 |publisher = Variety |language = en}}</ref>。本劇最後一集在2017年9月24日播出<ref>{{Cite web |url = https://tvline.com/2017/08/17/teen-wolf-series-finale-date-september-2017-airdate/ |title = Teen Wolf Series Finale Date Set |author = Andy Swift |date = 2017-08-17 |publisher = TVLine |language = en}}</ref>。

==简介==

===第一季===
该剧围绕着Scott McCall（[[泰勒·珀西|泰勒·珀西]] 饰演）在Beacon山高中的生活。一天晚上他和他的好朋友 Stiles (綽號，真名為Mieczysław Stilinski)（[[狄倫·歐布萊恩|狄倫·歐布萊恩]] 饰演）拉他去找尸体壮胆时他被狼人咬伤，从此他的生活急剧变化。Stiles发现后帮助他克服。

Scott数日回到当晚被咬的地方寻找自己丢失的助吸器时碰到了Derek Hale（[[泰勒·霍奇林|泰勒·霍奇林]] 饰演），获得狼人力量的Scott也进入曲棍球队并和新来的转学生Allison Argent（{{link-en|克莉絲朵·里德|Crystal Reed}} 饰演）展开恋情。在和Allison的首次约会正好是满月，控制不住自己的Scott找借口离开并发现在屋檐上的Derek也是狼人。Derek知道后威胁Scott退出曲棍球队并和Allison分手。

曲棍球队队长Jackson Whittemore（[[卡爾登·海尼斯|卡爾登·海尼斯]] 饰演）怀疑Scott使用了违禁药品，于是威胁Scott发现真相后要告诉Allison，Scott在队伍中的表现也让Jackson的女友Lydia Martin（{{link-en|荷蓮·蘿丹|Holland Roden}} 饰演）对Scott产生好感。而就在某天晚上Scott和Derek在森林中被猎人攻击，而这个猎人竟然是Allison的父亲，更让Scott想不到的是Argent家族是一个猎人家族。

===第二季===
Derek開始組織自己的狼群Isaac、Erica和Boyd，Jackson被咬後的效應也慢慢浮現。鎮上開始出現新的離奇謀殺案，新的超自然生物出現肯比山。Allison的爺爺也回到了肯比山進行復仇計劃。

===第三季===
新的阿爾法狼群出現肯比山，幕後主使想要成為新的領導。Lydia開始發現自己擁有的超能力。新的超自然生物野狐侵入內心黑暗的人想要造成混亂衝突。

===第四季===
鎮上出現不斷出現殺手們獵殺肯比山的超自然生物們，同時Peter的不計名債卷也被偷走，這一切源自於一份死亡名單。而Allison的姑姑也復活回到肯比山帶走Derek。

===第五季===
超自然生物卡美拉及死亡醫生開始出現於肯比山。Scotty也將面對團隊四分五裂的危機。

===第六季===
幽靈騎士的出現讓肯比山的人們消失。Allison的爺爺也準備了最後的復仇計畫。
