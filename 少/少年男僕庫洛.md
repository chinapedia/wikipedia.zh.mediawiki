{{日本作品|link=no|'''少年男僕庫洛'''|少年メイドクーロ君}}為[[柊柾葵|柊柾葵]]的女性向[[正太|正太]][[漫画|漫画]]。早期於{{tsl|ja|松文館}}發行的合集漫畫《{{lang|ja|ショタみみLOVE}}》 vol.6上連載，標題為「{{nihongo|庫洛君華麗的日常|クーロ君の華麗なる日常}}」。期後由同公司的Diamond Comices加以單行本化。2009年10月22日，由[[Soft_on_Demand|Soft on Demand]]發行漫畫電影版DVD。

== 故事 ==
為了還掉失蹤的爸爸所欠的錢，而在白龍院家工作的風宮庫洛。因為主人米斯的興趣、讓他雖然是個男生卻穿了女僕的衣服。
因此也讓他過著對於主人的性騷擾會加以還擊，但有時卻又會加以接受的生活。

時而過激、時而恩恩愛愛。這是一篇有著[[虐待狂|虐待狂]]性格和王子氣息的主人和有著傲嬌性格的少年僕人所生的[[閃光|閃光]]情侶愛情故事。

== 登場角色 ==
; {{nihongo|風宮庫洛|風宮 クーロ|Kazamiya Kūro|Couraud}}
: 生日：4月1日／血型：A型
: 為了父親的借款而以男生的身分成為女僕的少年。第一人稱「俺」。
: 平常反抗時，不但不會叫米斯主人、還會加以還擊、但在傲嬌模式時會變得相當柔順。
: 母親——{{nihongo|露拉|{{zy|ルーラ|Rūra}}}}過世，父親則下落不明。

; {{nihongo|白龍院米斯|白龍院 ミース|Hakuryūin Mīsu|Mies}}
: 日本首屈一指的大財團——白龍院家的少爺，也是庫洛的主人。第一人稱為「私」。
: 雖然是個渾身散發著王子氣息的「[[抖S|抖S]]」，卻對庫洛的眼淚攻勢毫無招架之力。
: 與同父異母的弟弟——弗雷分開生活。

; {{nihongo|白龍院弗雷|はくりゅういん フレン|Hakuryūin Frien}}
:米斯的同父異母弟弟，跟兄長互相競爭白龍院接班人的位置。
:外表看似清純可憐，但實際上卻相當[[腹黑|腹黑]]，剛回來日本時還曾對庫洛下手過。期後似乎和哥哥同樣愛上庫洛。

; {{nihongo|-{zh-hk:賀維善;zh-tw:哈比夏姆;zh-cn:哈比夏姆}-·上田|ハビシャム・上田|{{lang|ja|ハビシャム・ウエダ}}|Havisham Ueda}}
: 米斯的管家。是[[威爾斯|威爾斯]]大魔法学部主席、在做管家前也是特種部隊的一員，總之是個謎一般的人。
