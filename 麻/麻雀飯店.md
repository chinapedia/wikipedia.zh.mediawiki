《'''麻雀酒店'''》，日本漫畫家[[山東ユカ|山東ユカ]]的[[四格漫畫|四格漫畫]]作品，分別於2005年、2006年及2008年7月在[[竹書房|竹書房]]所發行的四格漫畫雜誌單篇刊登，2008年12月起在《Manga Life》（{{lang|ja|まんがライフ}}）連載，2010年集結成單行本出版，並於2013年4月起推出3分鐘的電視動畫。

==概要==
本作以在鬧區裡面設立的[[商務旅館|商務旅館]]「麻雀飯店」擔任櫃台工作的武鬥派旅館職員佐藤小百合為主角的故事。作者有位在[[飯店|飯店]]任職的朋友常分享工作上的逸聞，他以這些逸聞為題材畫成四格漫畫。

== 登場人物 ==
; {{nihongo|佐藤小百合|佐藤 小百合||[[聲優|聲]]：[[茅原實里|茅原實里]]}}
: 麻雀飯店的新進員工，擔任櫃台工作。外表漂亮、身材有如[[模特兒|模特兒]]並擁有[[巨乳|巨乳]]，是容易讓男性為之傾心的女性。但她卻有一身強大的[[功夫|功夫]]及巨大的力氣，有著可將人[[秒殺|秒殺]]的能力，有从事暗杀类的经验，因此常幫旅館排除酒醉客鬧事等狀況。因為是新人，客人多時常會手忙腳亂。
; {{nihongo|塩川環|塩川 環（たまき）||聲：[[长嶋阳香|长嶋阳香]]}}
: 麻雀飯店的主任，擔任櫃台工作，她的身材嬌小但工作十分精明幹練。
; {{nihongo|御園|御園（みその）||聲：[[岸尾大輔|岸尾大輔]]}}
: 大學生，在麻雀飯店[[打工|打工]]，擔任[[大夜班|大夜班]]櫃台工作。
; {{nihongo|塩川（兄）|塩川（兄）||聲：[[小野大輔|小野大輔]]}}
: 麻雀飯店的營業部長，是塩川環的哥哥。他是重度的[[妹控|妹控]]。
; {{nihongo|古城優|古城 ゆう||聲：[[結城飛鳥|結城飛鳥]]}}
: 任職於麻雀飯店的服務部，常喬裝成顧客住進飯店進行評鑑工作。
; {{nihongo|酒井|酒井||聲：[[森嶋秀太|森嶋秀太]]}}
:十分思念小百合的酒店常客。

==动画==
2013年4月开始在[[AT-X|AT-X]]、[[KBS京都|KBS京都]]、[[埼玉电视台|埼玉电视台]]播放。AT-X在每集動畫後播放『スパロウズホテル ご利用の前に』。由[[茅原実里|茅原実里]]和[[長嶋阳香|長嶋阳香]]演出，[[池上真彦|池上真彦]]擔任旁白。而KBS京都和埼玉电视台則在每集動畫後和之前的『[[漫研社|漫研社]]』一樣，播放『{{link-ja|柴犬子小姐|しばいぬ子さん}}情報corner』。

2013年4月16日官方网站宣布，由于[[作画崩坏|作画崩坏]]表示歉意，启用{{tsl|ja|いまざきいつき}}作为新的监督等新的工作人员负责剩下的话数和DVD版。原有话数将全部免费播放，但不久撤销了免费播放的消息。<ref>[http://www.cinematoday.jp/page/N0052172 アニメ「スパロウズホテル」が“作画崩壊”を認める　異例の対応にネット騒]シネマトゥデイ 2013年4月16日</ref><ref>第7話からはスタッフが一新され、『ちょぼらうにょぽみ劇場 あいまいみー』と同様に[[いまざきいつき|いまざきいつき]]が多くの作業を担当している。[http://www.takeshobo.co.jp/sp/tv_sparrows/news.html 【ご報告】TVアニメ「スパロウズホテル」についてお知らせ]公式サイト 2013年4月16日閲覧</ref>
