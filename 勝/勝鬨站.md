'''勝鬨站'''（{{jpn|j=勝どき駅|hg=かちどきえき|rm=Kachidogi eki|f=y}}）是位於[[日本|日本]][[東京都|東京都]][[中央區_(東京都)|中央區]][[勝鬨|勝鬨]]二丁目，屬於[[東京都交通局|東京都交通局]]（[[都營地下鐵|都營地下鐵]]）[[大江戶線|大江戶線]]的[[鐵路車站|鐵路車站]]。[[車站編號|車站編號]]是'''E 17'''。

由於「鬨」的漢字不是日本的[[常用漢字|常用漢字]]，故以[[平假名|平假名]]表記。

大門站務管理所門前仲町站務區管轄，交托{{link-ja|東京都營交通協力會|東京都営交通協力会}}。

== 歷史 ==
* 2000年（[[平成|平成]]12年）12月12日：本站開業。
* 2003年（平成15年）7月11日：A2b出入口啟用。A2出入口更名為A2a出入口。
* 2010年（平成22年）12月1日：A4a出入口啟用。A4出入口更名為A4b出入口。
* 2011年（平成23年）8月20日：[[月台門|月台門]]啟用。
* 2014年（平成26年）4月1日：業務委託化。
* 2019年（平成31年）2月11日：往[[大門站_(東京都)|大門]]方向月台移動至新月台<ref name="mainichi_20181207">{{Cite news|url=https://mainichi.jp/articles/20181207/k00/00m/040/190000c|title=都営大江戸線・勝どき駅に新ホーム 2月11日から利用可能|newspaper=毎日新聞|date=2018-12-07|accessdate=2018-12-07|archiveurl=https://archive.today/20181207100424/https://mainichi.jp/articles/20181207/k00/00m/040/190000c|archivedate=2018-12-07|dead-url=no}}</ref>。

== 車站構造 ==
本站為[[地下車站|地下車站]]，月台採[[島式月台|島式月台]]1面2線設計。

月台與剪票口、剪票口與A4b出入口之間設有電梯。

== 大規模改建工程 ==
2001年4月，{{link-ja|晴海島海神廣場|晴海アイランドトリトンスクエア}}開業後，早上[[繁忙時間|繁忙時間]]在本站下車前往週邊辦公大樓的通勤乘客大量增加，使車站非常擁擠。接到乘客投訴後，東京都交通局於2003年7月11日在A2出口北側增設A2b出口，而原A2出口則改稱A2a出口。並在早上繁忙時間（7時至10時）實施人潮管制措施，透過調整手扶電梯的運行方向，限制A2a出口只供進站，A2b出口只供離站；利用欄柵分隔進站和離站的人流；及安排保安員在場控制秩序。

除了辦公大樓外，車站附近地區亦建有不少高層[[公寓|公寓]]大樓。而現時公寓大樓的數目亦正在增加，預計使用本站的人數將會繼續增加。面對此趨勢，車站需要大規模改建，以容納更多的乘客。

而東京都交通局的改建計畫的重點，是在現有月台南側增建內環線（兩國、飯田橋方向）專用月台，而現有月台則成為外環線（大門、六本木方向）專用月台。同時現有的車站大堂將會擴闊，連接兩個月台。改建工程在2010年動工，2019年2月11日啟用。本站成為都營地下鐵罕見的雙[[側式月台|側式月台]]<ref>{{Cite news |url = http://www.tokyo-np.co.jp/article/national/news/CK2010010702000233.html |script-title = ja:勝どき駅ホーム増設 『ラッシュ』に切り札 |newspaper = [[東京新聞|東京新聞]] |date = 2010-01-07 |accessdate = 2010-01-22 |deadurl = yes |archiveurl = https://web.archive.org/web/20100108105935/http://www.tokyo-np.co.jp/article/national/news/CK2010010702000233.html |archivedate = 2010-01-08 }}</ref>。地下1樓與地面的出入口都新設1個，若包括扶手電梯增設工程則全部工程在2020年6月完工，總工程費用為160億日圓<ref>[http://www.tokyo-np.co.jp/article/tokyo/list/201812/CK2018122002000156.html 「ホーム増設 混雑緩和／都営大江戸線 勝どき駅／2月11日に使用開始」]『東京新聞』朝刊2018年12月22日（都心面）2018年12月28日閲覧。</ref>。

== 使用情況 ==
2017年度1日平均上下車人次為'''102,315人'''（上車人次：51,197人、下車人次：51,118人）<ref>[http://www.kotsu.metro.tokyo.jp/subway/kanren/passengers.html 各駅乗降人員一覧] - 東京都交通局</ref>。大江戶線內僅次於新宿站、大門站、六本木站名列第4位，同時是都營地下鐵全線運量最高的單獨站。

開業當時的預估1日平均上下車人次約28,000人。但在開業後，周邊地區的[[晴海_(東京都中央區)|晴海]]與[[勝鬨|勝鬨]]等陸續興建起高層公寓與辦公大樓。特別在2001年4月{{link-ja|晴海島海神廣場|晴海アイランドトリトンスクエア}}開業後乘客大幅上升。開業2年後的2002年度運量突破6萬人。為了因應平日早晨尖峰時刻的大量通勤人潮，站內配有警備員指揮。
