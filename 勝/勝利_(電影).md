《'''勝利'''》（''The Uncle Victory''）是一部2014年的中國大陸劇情電影，由北京百世佳聯影視傳媒、旗幟數字傳媒、中國電影集團、完美世界影視以及張猛導演工作室聯合出品<ref>{{cite web|url=http://ent.163.com/13/0412/09/8S8IJRS9000300B1.html | language= | title=张猛新片《胜利》首曝海报 2014年上映|publisher=网易娱乐 |date=2013-04-12 |accessdate=2014-06-22 }}</ref>，[[張猛_(導演)|張猛]]執導並由[[黃海波|黃海波]]和[[張歆藝|張歆藝]]主演。

電影講述刑滿釋放的陳勝利通過開辦一個幼兒園而化解十年的恩怨，並在中途與小美發生了一場既熱烈又遺憾的愛情。本電影獲得了第17屆[[上海國際電影節|上海國際電影節]]評委會大獎<ref name="上海國際電影節"/>。

2014年9月29日，[[廣電總局|廣電總局]]正式下發“封殺劣跡藝人”的通知，由吸毒、嫖娼等違法犯罪行為者作為主創人員參與製作的電影都被要求暫停播出和上映<ref name="封殺">{{cite web|url=http://ent.sina.com.cn/s/m/2014-10-08/15384220780.shtml | language= | title=总局封杀劣迹艺人：吸毒嫖娼禁出轨待定|publisher=新浪娱乐|date=2014-10-08   }}</ref>。本電影因此公映時間未知。

== 劇情簡介 ==
1990年代末期，黑道大哥陳勝利因意外傷害罪入獄十年。刑滿釋放後陳勝利回歸東北後曾試圖重新確立江湖地位，然而卻遇到了護士孫小美及擔任了幼兒園長。陳勝利試圖洗去背後的關公紋身和看護孩子成長來迎接新生，但是……

== 製作 ==
=== 拍攝 ===
電影《勝利》在開拍前曾以電影項目《我的身上有刺青》於2012年第12屆上海國際電影節電影項目市場單元獲得“特別關注獎”，故事題材依然取自東北，講述了一位東北黑道大哥自我救贖，以德報怨的故事<ref>{{cite web|url=http://ent.sina.com.cn/m/c/2012-06-27/23293670105.shtml| language= | title=《刺青》上海电影节引关注 张猛筹备新片|publisher=新浪娱乐 |date=2012-06-27 |accessdate=2014-06-22 }}</ref>。於2013年2月在[[遼寧省|遼寧省]][[阜新市|阜新市]]開機拍攝<ref>{{cite web |url=http://www.sd.xinhuanet.com/news/2013-05/15/c_115775535.htm |language= |title=黄海波张歆艺主演《胜利》:文艺片没那么文艺 |publisher=新华網 |date=2013-05-15 |accessdate=2014-06-22 |deadurl=yes |archiveurl=https://web.archive.org/web/20160304192251/http://www.sd.xinhuanet.com/news/2013-05/15/c_115775535.htm |archivedate=2016-03-04 }}</ref>，並全程採用膠片拍攝<ref name="网易news"/>。張猛表示選擇在阜新拍攝的原因是“希望藉該城市留存的工業質感和時代興衰的背景，來印證每一個普通人身上都留下了“時代的痕跡””<ref>{{cite web|url=http://ent.qq.com/a/20130418/000236.htm | language= | title=张猛《胜利》张歆艺造型美艳 黄海波“刑满释放”|publisher=腾讯娱乐 |date=2013-04-18 |accessdate=2014-06-22 }}</ref>。

=== 演員 ===
導演[[張猛_(導演)|張猛]]接受訪問時曾表示選擇[[黃海波|黃海波]]、[[張歆藝|張歆藝]]參演的原因是因為他們兩人的氣質和東北比較接近<ref name="网易news">{{cite web|url=http://news.163.com/14/0427/13/9QRELOQ700014AED.html | language= | title=张猛拍“黑帮片”|publisher=网易娱乐 |date=2014-04-27 |accessdate=2014-06-22 }}</ref>。本劇主要角色「陳勝利」一角就是導演張猛的舅舅，為此了 [[黃海波|黃海波]]曾特意去幼兒園體驗生活，還和張猛的舅舅見面<ref name="网易news"/>。張猛的女兒也在片中出演一個角色<ref name="网易news"/>。

== 上映及宣傳 ==
本電影雖然入圍了第17屆[[上海國際電影節|上海國際電影節]]，但是在該電影節上發表的媒體手冊卻沒有和其他金爵競賽片一樣提供電影的公映排片以及採訪安排，而電影節方也沒有就此回應<ref>{{cite web|url=http://ent.qq.com/a/20140618/046771.htm | language= | title=上海电影节：《胜利》成“禁忌”引媒体不满|publisher=腾讯娱乐 |date=2014-06-18 |accessdate=2014-06-22 }}</ref>。

2014年9月29日，[[廣電總局|廣電總局]]正式下發“封殺劣跡藝人”的通知，由吸毒、嫖娼等違法犯罪行為者作為主創人員參與製作的電影都被要求暫停播出和上映<ref name="封殺"/>，曾犯嫖娼行為的[[黃海波|黃海波]]令電影《勝利》品因此受到影響<ref>{{cite web|url=http://hlj.ifeng.com/ent/detail_2014_09/26/2960050_0.shtml | language= | title=广电总局正式下劣迹令通知 文章姚笛暂时安全|publisher= 凤凰网 |date=2014-09-26 }}</ref>。

《勝利》入圍了2014年香港亞洲電影節的「亞洲單元」 並在11月1日和11月5日分別在[[香港|香港]]兩間戲院限量上映<ref>{{cite web|url=http://timable.com/event/695903 | language= | title=《勝利》亞洲國度 香港亞洲電影節 2014 |publisher= 香港亞洲電影節 |accessdate=2014-11-06 }}</ref><ref>{{cite web|url=http://ent.sina.com.cn/m/c/2014-11-06/10594236420.shtml | language= | title=黄海波新片涉坐牢台词:改造好才能放出来 |publisher= 新浪娱乐 |date=2014-11-06 }}</ref>。

== 獎項及提名 ==
本電影入圍了第17屆[[上海國際電影節|上海國際電影節]]，並獲得評委會大獎<ref name="上海國際電影節">{{cite web|url=http://ent.qq.com/a/20140622/023708.htm | language= | title=17届上海电影节闭幕 张猛《胜利》获评委会大奖|publisher=腾讯娱乐 |date=2014-06-22 |accessdate=2014-06-22 }}</ref>，評委會對《勝利》評語稱，“這是一部大無畏的電影，讓我們看到一位年輕導演的探索精神和他的野心”<ref>{{cite web|url=http://ent.163.com/14/0622/21/9VCHOS5T00034TPU.html | language= | title=第17届上海电影节闭幕 黄海波新片获大奖|publisher= 网易娱乐 |date=2014-06-22 |accessdate=2014-06-23 }}</ref>。
