'''勝沼精藏'''（[[日文|日文]][[漢字|漢字]]：'''勝沼 精蔵'''，{{bd|1886年|8月28日|1963年|11月9日|Katsunuma Seizō}}）是生於[[兵庫縣|兵庫縣]][[神戶市|神戶市]]的[[醫學家|醫學家]]、[[醫學博士|醫學博士]]，其専長是[[血液學|血液學]]、[[神經學|神經學]]，曾任[[名古屋大學|名古屋帝國大學]]醫學院教授、第三任[[名古屋大學|名古屋大學]][[校長|校長]]、第三任[[:ja::国立病院機構名古屋医療センター|国立名古屋病院]]院長。

勝沼精藏曾被提名1953年[[諾貝爾生理學或醫學獎|諾貝爾生理學或醫學獎]]<ref>[http://www.nobelprize.org/nomination/archive/show_people.php?id=4732 Nomination Database - nobelprize.org]</ref>。

== 簡歷 ==
1886年([[明治|明治]]19年)出生於[[神戶|神戶]]。

1911年(明治44年)自[[東京大學|東京帝國大学]][[醫學院|医学部]]畢業。

1912年(明治45年)進入東京帝國大学[[三浦謹之助|三浦謹之助]]的三浦内科實習。

1913年([[大正|大正]]2年)在[[病理學|病理学]]教室擔任勤務。

1918年(大正7年)至[[法國|法國]]留學。

1919年(大正8年)於愛知県立医学専門学校教授一職就任，[[巴黎和會_(1919年)|巴黎和會]]時被安排於[[产科|産科]]隨[[西園寺公望|西園寺公望]]同行。

1923年(大正12年)於[[愛知醫科大學|愛知医科大学]]教授一職就任。

1931年(昭和６年)於[[名古屋醫科大學|名古屋医科大学]]教授一職就任。

1932年(昭和７年)任名古屋医科大学付属[[醫院|病院]]院長。

1949年(昭和24年)任[[名古屋大學|名古屋大学]]総長。

1954年(昭和29年)獲頒[[文化勳章|文化勳章]]。

== 學經歷・相關人物 ==

* 畢業自[[:ja::静岡県立静岡高等学校|県立静岡中学]]
* 畢業自[[舊制第一高等學校|第一高等学校]]
* 擔任[[西園寺公望|西園寺公望]]的[[主治醫師|主治醫師]]直至其逝世
* 倡議成立[[全身捐贈|献体]]團體[[:ja::不老会|不老会]]
* 1926年，「[[氧化酶|氧化酶]]的組織学的研究」獲頒[[日本學士院獎|学士院賞]]
* 1943年，確認[[腦波|腦波]]研究的重要性，在日本組織了一個研究小組（小組後來由[[本川弘一|本川弘一]]接掌 <ref>時実利彦・藤森聞一・島薗安雄・佐野圭司　『新脳波入門』　南山堂，東京，1959年，第2頁。</ref>，1947年併入腦波研究委員會，1952年改為日本脳波学会。1971年與日本筋電図学会合併為日本脳波・筋電図学会，進一步於2000年更名為日本臨床神経生理学会）
* 1954年11月3日獲頒[[日本文化勳章|文化勲章]]。
* 逝世後於1963年11月10日追贈[[勳一等旭日大綬章|勳一等旭日大綬章]]，完成追敘[[從二位|従二位]]品秩
* 其子[[勝沼晴雄|勝沼晴雄]]曾擔任東京大学医学部公衆衛生学教授兼医学部長、[[杏林大学|杏林大学]]副学長
* 其姪子[[勝沼信彦|勝沼信彦]]為[[醫學家|醫學家]]，曾任[[德島文理大學|徳島文理大学]]校長
