'''勝浦市'''（{{jpn|j=勝浦市|hg=かつうらし|rm=Katsuura shi|f=y}}）是位于[[千葉縣|千葉縣]][[房總半島|房總半島]]的一[[市|市]]。人口未達兩萬，是千葉縣內人口最少的一市。以日本三大早市之一而有名。

== 地理 ==
本市位於[[房總半島|房總半島]]東南部，面太平洋（外房），坐落在[[千葉市|千葉市]]南方約60公里處。經[[東京灣跨海公路|東京灣跨海公路]]至東京約75公里，屬都心75公里圈<ref name="syokai">{{Cite web |url=http://www.city.katsuura.chiba.jp/syokai/index.html |title=勝浦市紹介 |publisher=勝浦市 |accessdate=2012-04-10}}</ref>。

全市可分東北部的總野地區、西北部的上野地區、東南部的勝浦地區、西南部的興津地區<ref name="syokai" />。主要幹道有太平洋沿岸的{{link-ja|國道128號|国道128号}}，與縱貫市域的{{link-ja|國道297號|国道297号}}。

中心地帶勝浦地區的{{link-ja|勝浦漁港 (千葉縣)|勝浦漁港 (千葉県)|勝浦漁港}}漁獲量在縣內僅次於{{link-ja|銚子漁港|銚子漁港}}<ref name="chiba-pref">{{Cite web |url=http://www.pref.chiba.lg.jp/kyouiku/bunkazai/sonohoka/kyoudo/fuukoumeibi/asaichi.html |title=21.勝浦藩と勝浦の朝市・カツオまつり |publisher=[[千葉縣|千葉縣]] |accessdate=2012-04-10 }}</ref>，是歷史悠久的[[漁港|漁師町]]。西北部的上野地區屬{{link-ja|房總丘陵|房総丘陵}}，海拔約150～250公尺。西南部興津地區擁有{{link-ja|勝浦海中公園|勝浦海中公園}}，海岸線多屬[[溺灣|溺灣海岸]]，已被指定為[[南房總國定公園|南房總國定公園]]。全市約有三分之二為山地<ref name="gaiyo">{{Cite web |date= |url=http://www.city.katsuura.chiba.jp/seikatsukankyo/pdf/hakusyo/01.pdf |title=勝浦市の概要 |format=PDF |publisher= |accessdate=2012-04-10}}</ref>，受限於地形，人口增加不易。

== 沿革 ==
=== 地名由來 ===
地名來自與{{link-ja|天富命|遠見岬神社}}一同從[[阿波國|阿波國]]遷往[[房總半島|房總半島]]定居的'''勝占'''[[忌部氏|忌部氏]]<ref>参考：「日本各地における阿波忌部の足跡～安房国編～」（林博章著）</ref>。另外，[[那智勝浦町|紀伊半島的勝浦]]也是同樣由來。

=== 歷史 ===
*[[繩文時代|繩文時代]]，{{link-ja|夷隅川|夷隅川}}旁已有小規模聚落，並發現自興津運送鮑魚至[[奈良|奈良]]的紀錄<ref name="gaiyo"/>。
*[[戰國時代_(日本)|戰國時代]]，[[安房國|安房國]][[里見氏|里見氏]]武將{{link-ja|正木時忠|正木時忠}}入主{{link-ja|勝浦城|勝浦城}}，管轄此地<ref name="gaiyo" />。
*1590年（[[天正|天正]]18年）正木氏在[[小田原征伐|小田原征伐]]後返回安房國，[[德川家康|德川家康]]幕臣{{link-ja|植村泰忠|植村泰忠|植村土佐守泰忠}}入主勝浦城、經營町屋。為現在勝浦市區的基礎。
*興津地區在江戶時代是幕府的[[天領|直轄領]]，妙覺寺設有[[仙台藩|仙台藩]]取締所，是江戶與東北之間的重要港口<ref name="gaiyo" />。
* 1913年（[[大正|大正]]2年）6月20日 - {{link-ja|勝浦站|勝浦駅}}開業。
* 1937年（[[昭和|昭和]]12年）4月1日 - 勝浦町與豐濱村合併成立'''勝浦町'''。
* 1953年（昭和28年）5月18日 - 制定{{link-ja|國道128號|国道128号}}。
* 1955年（昭和30年）2月11日 - 勝浦町、興津町、上野村、總野村合併成立'''勝浦町'''。
* 1958年（昭和33年）10月1日 - 勝浦町施行[[市制|市制]]，為'''勝浦市'''。

==== 漁業 ====
勝浦漁港是日本重要的[[鰹魚|鰹魚]]港。該港鰹魚捕獲量曾在1990年達到日本第一。此外，由於海岸線多屬溺灣海岸，多岩場，其他主要漁產有[[鮪魚|鮪魚]]、[[红金眼鲷|红金眼鲷]]、[[旗魚|旗魚]]、[[烏賊|烏賊]]等<ref name="chiba-pref" />。
