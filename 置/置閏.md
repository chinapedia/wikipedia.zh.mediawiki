'''置閏'''是一些[[曆法|曆法]]中插入閏日、閏週或閏月，以使曆法能跟隨[[季節|季節]]或[[月相|月相]]。[[陰陽曆|陰陽曆]]可能需要插入閏日和閏月。

== 太陽曆 ==
[[回歸年|回歸年]]（太陽年）都不是剛好的整數天（大約是365.24天），但是[[曆年|曆年]]都必須是整數的天數。最常見的方式，是在不同的曆年中插入天數來調整。

在太陽曆，這造成平年有365天，大約每四年要添加1日（閏日），造成有366天的[[閏年|閏年]]（[[儒略曆|儒略曆]]、[[格里曆|格里曆]]和[[印度國定曆|印度國定曆]]）。

[[克諾珀斯法典|克諾珀斯法典]]（Decree of Canopus）是西元前239年[[古埃及|古埃及]][[法老|法老]][[托勒密三世|托勒密三世]]頒布的太陽年置閏系統，直到西元前25年[[羅馬皇帝|羅馬皇帝]][[奧古斯都|奧古斯都]]成功改革[[科普特曆|亞歷山大曆]]之後才被廢除。

在[[儒略曆|儒略曆]]及[[格里曆|格里曆]]，改進了置閏的方法，將額外添加一天做為二月最後一天的年份稱為閏年。在儒略曆，是每四年有一個閏年；在格里曆，年份能被100整除，但不能被400除盡的年份不再是閏年，而變得更為精確。因此，西元2000年是閏年，而1700年、1800年和1900年都不是閏年。

'''歲末/歲餘日'''（{{lang-en|epagomenal}}<ref>From ἐπαγόμενος, ''epagomenos'' (present participle passive of ἐπάγειν, ''epagein'' "to bring in") + -al</ref>）是太陽曆中不屬於任何月份的日子，每年通常包括5天（[[古埃及历法|古埃及历法]]、[[科普特曆|科普特曆]]、[[埃塞俄比亞曆|埃塞俄比亞曆]]、[[瑪雅曆|瑪雅曆]]、和[[法國共和曆|法國共和曆]]），但是在某些曆（[[科普特曆|科普特曆]]、[[埃塞俄比亞曆|埃塞俄比亞曆]]、和[[法國共和曆|法國共和曆]]）中每4年會有一年有6天這種日子。在2015年之前，[[巴哈伊曆法|巴哈伊曆法]]的平年有4天這種日子，閏年則有5天，以便能與格里曆吻合。從2015年開始（172 B.E.），這種日子的設置將用以確定每年開始於春分日。

== 陰陽曆 ==
{{see|章 (曆法)}}
太陽年不是[[月#朔望月|陰曆月]]的整數倍（它大約是12.37[[朔望月|陰曆月]]），所以陰陽曆的一年必須具有數量可變的[[月|月]]。平常的一年是12個月，閏年添加一個月成為13個月。通常每2-3年便會插入一個閏月（參見[[藍月|藍月]]）。如何插入一個閏月，通常使用固定的週期循環，例如19年的[[默冬章|默冬章]]（[[希伯來曆|希伯來曆]]和[[復活節計算表冊|推斷復活節]]）或使用[[月相|月相]]（[[印度曆|印度陰陽曆]]和中國的[[農曆|農曆]]）。由於當前[[近日點|近日點]]在冬季 (北半球)，冬季節氣之間較短，較容易分配到[[中氣|中氣]]；因此若採用[[無中置閏法|無中置閏法]]，閏月較常發生在夏季。

[[佛曆|佛曆]]通常會添加閏日和閏月。

== 伊斯蘭曆 ==
[[伊斯蘭曆|伊斯蘭曆]]通常有12個月，它們每年都是30天和29天交替排列的，但在30年內會有11年在最後一個月添加閏日。一些歷史學家將前伊斯蘭曆的{{Link-en|Nasi'|Nasi'}}也解釋為閏日。
 
[[伊朗曆|伊朗曆]]以太陽曆的計算為基礎，與格里曆有相似的結構，因此也會置閏，是伊斯蘭教曆法中的例外<ref>{{cite web | url=http://www.alislam.org/library/history/ahmadiyya/73.html | title=Hijri-Shamsi Calendar | publisher=Al Islam | accessdate=June 14, 2014}}</ref>。

== 閏秒 ==
[[國際地球自轉服務|國際地球自轉服務]]可以在任何一個月（首選是6月和12月）的結束時插入或刪除1秒鐘，稱為[[閏秒|閏秒]] <ref>[http://www.merriam-webster.com/dictionary/leap+second leap second by Merriam-Webster OnLine]</ref>。

== 其它的用法 ==
[[ISO_8601|ISO 8601]]包括[[ISO週日曆|規範]]平常一年52週，但任何一年若有53個星期四，當年就增加一星期而有53週，稱為閏週。

決定一年是否置閏，可以用計算（[[儒略曆|儒略曆]]、[[格里曆|格里曆]]、[[希伯來曆|希伯來曆]]），或由觀察確定（[[伊朗曆|伊朗曆]]）。
