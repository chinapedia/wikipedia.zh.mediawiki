'''中委關係'''，是指歷史上的[[中華人民共和國|中國]]和[[委內瑞拉|委內瑞拉]]的關係，兩國於1974年6月28日建交。兩者關係因支持社会主义的委內瑞拉前總統查韋斯和中國前國家主席[[胡錦濤|胡錦濤]]而變得越來越重要。

== 經貿關係 ==
兩國貿易於1999年以前少於5億美元，2009年已達75億美元。2014更升至138.9亿美元，其中中國进口94.46亿美元，委內瑞拉進口44.44亿美元，中方主要出口电器、计算机、电子产品、机械设备等，主要进口原油、成品油、铁矿砂及其精矿等<ref>{{cite web|url=http://www.fmprc.gov.cn/mfa_chn/gjhdq_603914/gj_603916/nmz_608635/1206_608880/sbgx_608884/|title=中国同委内瑞拉的关系|publisher=中國外交部}}</ref>。中國因此成為委內瑞拉第二大貿易伙伴，委內瑞拉亦成為中國最主要拉丁美洲投資地。委內瑞拉亦向中國出口石油及其他資源。

[[委內瑞拉危機|委內瑞拉危機]]爆發以來，委內瑞拉多次向中國寻求资助。2018年9月12日马杜罗访华抵达北京时，甚至对媒体称，中国就像委内瑞拉的“老大哥”，对进一步深化委内瑞拉与中国的关系“期望甚高”<ref name="die">[http://news.dwnews.com/global/news/2019-02-11/60117264.html 美媒：中国援助委内瑞拉的500亿美元不划算]</ref>。

截至2018年9月，委內瑞拉有約200億美元未償還中國貸款<ref>{{Cite news |url=https://www.bbc.com/zhongwen/trad/chinese-news-45524575 |title=中國向委內瑞拉再伸援手 習近平馬杜羅各有所圖 |work=BBC中文網 |date=2018-09-15}}</ref>。[[2019年委内瑞拉总统地位危机|2019年委内瑞拉总统地位危机]]期間有媒體統計委內瑞拉欠中國貸款高達500億美元<ref name="die"></ref>。

== 文化關係 ==
1981年两国签订政府间文化合作协定。2003年6月，委內瑞拉舉行“中国文化周”。中国自1961年开始接受委内瑞拉留学生，截至2013年，共接受委奖学金生146名，中方派出留学生5名。2006年8月，中委签署《中国旅游团队赴委内瑞拉旅游实施方案的谅解备忘录》。2007年1月，中國正式启动中国旅游团队赴委旅游组团业务。

2008年10月底，中国成功为委发射“委内瑞拉一号”通信卫星，2009年1月正式交付委方使用。此系中国在拉美地区第一颗整星出口、在轨交付的商业卫星。2012年9月底，中国成功为委发射“委内瑞拉遥感卫星一号”。

== 軍事關係 ==
1981年，中委达成互设武官处协议。20世纪90年代以来，两军高层交往与合作增多。近年来，中委兩國多位軍官均有進行互訪。
