'''中央军委装备发展部试验鉴定局'''，位于[[北京市|北京市]]，是[[中央军委装备发展部|中央军委装备发展部]]下属局，负责试验鉴定工作。

==沿革==
在[[深化国防和军队改革|深化国防和军队改革]]中，2016年1月撤销[[中国人民解放军总装备部|中国人民解放军总装备部]]，组建[[中央军委装备发展部|中央军委装备发展部]]。中央军事委员会装备发展部设'''中央军委装备发展部试验鉴定局'''。原[[中国人民解放军总装备部综合计划部|中国人民解放军总装备部综合计划部]]副部长夏清月任第一任局长<ref name=pp>{{Cite web |url = http://www.thepaper.cn/newsDetail_forward_1621815 |title = 夏清月任军委装备发展部试验鉴定局局长 |author =  |date =2017-02-18   |publisher =澎湃新闻  |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://www.donglingtech.com/cn/gy03-01.asp?id=556 |title = 军委装备发展部机关领导莅临东菱考察调研 |author =  |date =2016-10-12  |publisher = 东菱科技 |language =  |accessdate =  |quote =  }}</ref>。
