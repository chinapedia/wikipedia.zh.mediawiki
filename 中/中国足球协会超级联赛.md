'''中国足球协会超级联赛'''（{{lang-en|Chinese Football Association Super League}}，缩写{{lang|en|CSL}}）是由[[中国足球协会|中国足球协会]]所主辦的最高等級職業足球聯賽。该联赛开始于2004年，脱胎自原[[中国足球甲级A组联赛|中国足球甲级A组联赛]]。第一届有12支球队参加，首兩届暫停降级制度以便扩充球队数量，於2006年恢復“升二降二”的升降級制度（但由於2006年[[四川冠城|四川冠城]]解散、2007年[[上海申花|上海申花]]與[[上海联城|上海联城]]合併、2008年[[湖北武汉职业足球俱乐部|武汉光谷]]退出及2012年[[大連實德|大連實德]]解散，故这四年都只实行“升二降一”，即从[[中国足球甲级联赛|中甲]]中升兩隊，降一隊到[[中国足球甲级联赛|中甲]]）。

2008年赛季起正式恢复到原先计划的“升二降二”制度。在中超成立之初，由于硬件设施不能达到[[中国足协|中国足协]]所制定的中超准入标准，[[辽宁队|辽宁队]]的参赛资格曾引起过很大争议。2006年4月，为进一步完善中国足球产业的市场化进程，中国足球协会与所有中超联赛参赛俱乐部共同出资成立了中超联赛有限责任公司。2007年由于[[申花联城合并|上海联城和上海申花的合并]]，总共有15支俱乐部队参加2007赛季中超联赛。2008年，第一次有16支队伍参加中超联赛，但是於10月初，[[湖北武汉职业足球俱乐部|武汉光谷]]因称中国足协违反程序对其进行不公处罚而退出中超联赛。

因為國家隊長期成績不佳，加上中國足球各级聯賽均存在管理腐敗，賭球成風問題，2009年后已經引起了中華人民共和國中央領導的關注，包括[[習近平|習近平]]，[[劉延東|劉延東]]等[[党和国家领导人|党和国家领导人]]分別在不同場合提到中國足球問題以及親身參與中國足球的會議。2009年11月[[中華人民共和國公安部|中華人民共和國公安部]]展開[[中國足壇反賭風暴|中國足壇反賭風暴]]行動，大範圍打擊抑制足球發展的賭球行為。

[[2012年中國足球超級聯賽|2012年中國足球超級聯賽]]十六支球隊單賽季投入超過30億[[人民幣|人民幣]]（約5億[[美元|美元]]），為歷屆最高；一些財力雄厚的球隊亦網羅瞭如[[德羅巴|德羅巴]]、[[阿內爾卡|阿內爾卡]]、[[凱塔|凱塔]]、[[羅申巴克|羅申巴克]]、[[卡努特|卡努特]]、[[雅庫布|雅庫布]]、[[巴里奧斯|巴里奧斯]]等球星，中超自賭球風波後再次開始被中國國內社會甚至世界球壇所關注；而2012年全年的票房亦為2004年中超成立以來最高。

[[2013年中國足球超級聯賽|2013年中國足球超級聯賽]]初期，雖然阿內爾卡，德羅巴離開分別加盟[[尤文圖斯|尤文圖斯]]和[[加拉塔薩雷|加拉塔薩雷]]，但中國足球超級聯賽依舊吸引了[[埃爾克森|埃爾克森]]、[[米西莫維奇|米西莫維奇]]、[[紀堯姆·瓦羅|瓦羅]]、[[羅蘭多·斯基亞維|斯基亞維]]、[[帕特裏西奧·托蘭佐|托蘭佐]]、[[瓦格納·勒夫|勒夫]]等球星的加盟。

[[2014年中國足球超級聯賽|2014年中國足球超級聯賽]]亦吸引了[[蒙蒂略|蒙蒂略]]、[[迪亞曼蒂|迪亞曼蒂]]、[[吉拉迪諾|吉拉迪諾]]、{{link-en|卢卡斯·比亚特里|Lucas Viatri|比亚特里}}、[[邁克·漢克|漢克]]、[[德揚|德揚]]等大牌球星加盟。而且2014赛季中超各俱乐部的总收入突破20亿元，创下中超历史新高。另外中超上座率高居亚洲第一。但在同时，各俱乐部收支失衡，经营模式仍不具备可持续发展性。直至2014赛季中超联赛结束时，各俱乐部总体仍处于亏损状态，只有5家俱乐部实现盈利，分别是[[广州恒大|广州恒大]]、[[贵州人和|贵州人和]]、[[上海上港|上海上港]]、[[上海申鑫|上海申鑫]]以及[[广州富力|广州富力]]<ref>{{cite web|title=中超盈利俱乐部仅有5家 高工资是亏损原因之一|url=http://www.chinanews.com/ty/2014/11-13/6768874.shtml|accessdate=2014-11-13|publisher=中国新闻网}}</ref>。为長遠發展，16支中超球队也會投資興建自己球會運動場及足球場。
 
[[2015年中國足球超級聯賽|2015年中國足球超級聯賽]]又有[[卡希爾|卡希爾]]、[[塔爾德利|塔爾德利]]、[[阿蘭|阿蘭]]、[[高拉特|高拉特]]、[[羅比尼奧|羅比尼奧]]、[[保利尼奧|保利尼奧]]、[[吉安|吉安]]、[[登巴·巴|登巴·巴]]、[[古德約翰森|古德約翰森]]、[[穆罕默德·西索科|穆罕默德·西索科]]等大牌球員加盟。據權威機构統計，2015年中國足球超級聯賽在冬季和夏季的引援投入均歷史性的排行世界第二，僅次於英格蘭足球超級聯賽。 

[[2016年中國足球超級聯賽|2016年中國足球超級聯賽]]，[[拉米雷斯|拉米雷斯]]、[[傑克遜·馬丁內斯|傑克遜·馬丁內斯]]、[[布拉克·耶勒马兹|伊爾馬茲]]、[[雷纳托·奥古斯托|奧古斯托]]、[[瓜林|瓜林]]、[[馬丁斯|馬丁斯]]、[[特謝拉|特謝拉]]、[[若|若]]、[[拉維奇|拉維奇]]、[[熱爾維尼奧|熱爾維尼奧]]等大牌球員在賽季初加盟，夏季轉會更是引進了[[胡爾克|胡爾克]]、[[佩萊|佩萊]]、[[帕皮斯·西塞|帕皮斯·西塞]]、[[埃兰·扎哈维|紮哈維]]、[[烏賈|烏賈]]等大牌球員。

== 賽制 ==
聯賽採取主客雙循環賽制比賽，每支隊伍與各球隊對賽兩次，主客各一次。
* '''中超联赛决定名次办法'''
（一）每队胜一场得3分，平一场得1分，负一场得0分；

（二）中超联赛结束，积分多的队名次列前；

（三）如果两队或两队以上积分相等，依下列顺序排列名次：
# 积分相等队之间相互比赛积分多者，名次列前
# 积分相等队之间相互比赛净胜球多者，名次列前
# 积分相等队之间相互比赛进球数多者，名次列前
# 积分相等队之间，将按权重比例计算预备队成绩、U19联赛成绩、U17联赛成绩、U15联赛成绩、U14联赛成绩和U13联赛成绩, 排名靠前者, 联赛名次列前。
# 积分相等队在当年中超联赛全部比赛中净胜球多者，名次列前
# 积分相等队在当年中超联赛全部比赛中进球数多者，名次列前
# 全年比赛红黄牌扣分少者(不包括纪律处罚)，名次列前；
# 以抽签的办法决定名次

== 中超經濟效益 ==
隨着中國足球的系统化、制度化以及球迷人數與日俱增，平均每場比賽的入場人數人數已高達3萬五千人次，一些國內[[德比|德比]]戰，如上海德比戰、廣州德比戰更高達六萬人，帶動球場內外飲食店生意興旺。中超在亞洲的排名已進佔前十名，世界排名前一百，[[廣州恒大|廣州恒大]]更成為世界百強球會之一。目前中超在中國全國的整季轉播費用每年高達50億人民幣，每間球會所販賣的周邊商品、球衣多不勝數，也造就不少娛樂公司發展機會，由於每場比賽都需要啦啦隊、歌唱藝人、模特兒擔任球會寶貝，每場賽事光是在表演環節，開銷就高達百萬人民幣，加上近年不少當打球星加盟中超聯赛，令中超吸引力與日俱增。
與此同時,因為每場比賽也有過萬人次入場,當地政府每場也需要出動超過五百名[[中國|中國]][[公安|公安]]、五百名[[中國|中國]][[武警|武警]].每場比場球會會自行聘請專業[[警衛|警衛]]維持秩序,視乎戲碼而定,少則數百人,大戰可能高達五千人以上.因為每場比賽也超過五萬人次人流,亦帶動附近商場及食肆生意.中超足球已成為一盤不斷賺錢的大生意,長遠目標成為亞洲第一聯賽,每場入場人次超過十萬人次.

== 外援政策 ==
2004-2006年，每个俱乐部可以同时注册3名外援，比赛中可以同时上场2人<ref>[http://www.ewen.cc/qikan/bkview.asp?bkid=47076&cid=90146 中超出台新规定 限制外援 工资封顶] {{webarchive|url=https://web.archive.org/web/20140714192330/http://www.ewen.cc/qikan/bkview.asp?bkid=47076&cid=90146 |date=2014-07-14 }}</ref>。

2007-2008年，每个俱乐部可以同时注册4名外援，比赛中可以同时上场3人。

2009-2016年，每个俱乐部可以同时注册5名外援（至少1名需为亚足联会员国的球员），比赛中可以同时上场4人（至少1名需为亚足联会员国的球员）<ref name="竞赛规程">{{Cite web|title=2009赛季中国足球协会超级联赛规程|url=http://sports.sina.com.cn/j/2009-03-20/17564280280.shtml|publisher=中国足协}}</ref>。

2017年，每个俱乐部可以同时注册5名外援（至少1名需为亚足联会员国的球员），比赛中可以同时上场3人。

每个赛季每个中超俱乐部总共可以注册7名外援，如果在赛季前注册了5名外援，则在赛季中可以更换2名外援。[[2010年中国足球超级联赛|2010赛季]]参加亚冠联赛的球队不受这个规定限制，之后的赛季则取消了这一特权。

最初[[香港|香港]]球员被认定为亚洲外援<ref>{{cite web|title=9名试训外援抢2只饭碗 进绿城队比考公务员还难|url=http://sports.sohu.com/20090724/n265445003.shtml|accessdate=2009-07-25|publisher=搜狐体育}}</ref>，在2009年5月底曾修改为香港、澳門、台灣球员在中超、中甲联赛中被视为国内球员，不再占用亚洲外援名额<ref>{{cite web|title=足协宣布港脚摆脱亚洲标签 规则改变或产生重大影响|url=http://news.xinmin.cn/rollnews/2009/06/02/2036869.html|accessdate=2009-09-22|publisher=新民网}}</ref>。2016年起，新签约的港澳臺球员重新在中超、中甲联赛中占用亚洲外援名额。

2018赛季起，中超的外援注册方案为全年累计报名最多6人，同时报名人数是4人，累计上场人次不能超过3人。<ref>{{cite web|url=http://sports.sina.com.cn/china/j/2017-12-22/doc-ifypvuqf0231426.shtml|title=足协新政:U23出场数不少于外援数 无时间规定|publisher=新浪体育}}</ref>另外，2018赛季中超每个俱乐部可以引进一名首次注册地为港澳台足协的球员作为内援，且2016年1月1日前签订合同的港澳台球员仍视作内援。<ref>{{cite web|url=https://sports.sina.cn/china/2018-02-01/detail-ifyrcsrw4265932.d.html|title=足协港澳台引援新规作用不大 华裔归化球员成焦点|publisher=新浪体育}}</ref>

中超球队不能使用外籍守门员（包括在香港、澳門、臺灣足球协会注册的球员）。

==俱乐部支出限制==
從[[2018年|2018年]]赛季開始，为防止一些资金已達世界級球會實力的[[中超|中超]]球會再以[[天價|天價]]購買球員，球会需再付多一倍價錢的金額给[[中國足協|中國足協]]。

[[2019年|2019年]]開始，设置俱乐部总支出的最高限额。其中，中超联赛2019、2020、2021年的限额分别为12亿、11亿和9亿，俱乐部全年奖金总额（全年赢球奖金和平球奖金的总额），纳入俱乐部总支出限额内。其中中超赢球奖金不得超过300万元一场，但此支出限额不包括青训等支出。[[中國|中國]]球員限定二千萬[[人民幣|人民幣]]轉會費，部分国脚可按上浮20%执行。外援限定四千五百萬[[人民幣|人民幣]]轉會費<ref>[http://shanghai.xinmin.cn/xmsq/2018/12/20/31467667.html 中国足协：中超俱乐部2019年总支出不得超过12亿]</ref>。

== 裁判 ==
除部分亚洲赛事外，基本无缘执法FIFA世界杯和[[奥林匹克运动会足球比赛|奥运会足球比赛]]。
