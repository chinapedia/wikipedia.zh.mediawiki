'''中国人民解放军总医院第八医学中心'''，位于[[北京市|北京市]][[海淀区|海淀区]]黑山扈路甲17号，为[[中央军委联勤保障部队|中央军委联勤保障部队]]直属[[中国人民解放军总医院|中国人民解放军总医院]]的八个医学中心之一，[[正师级|正师级]]<ref>{{Cite web|url=http://www.hdcdc.org/PicInfo/2010-11-23/20101123170112.html|title=共同学习、相互借鉴|accessdate=2017-11-11|author=|date=2010-11-23|work=|publisher=北京市海淀区疾病预防控制中心}}</ref><ref name=搜狐>{{Cite web |url = http://www.sohu.com/a/273558387_267160 |title = 军队医院更名继续，解放军总医院八个医学中心亮相 |author =  |date =2018-11-06  |publisher =搜狐  |language =  |accessdate =  |quote =  }}</ref>，是一所综合性[[三级甲等医院|三级甲等医院]]。

== 沿革 ==
1958年，[[重庆|重庆]]的中国人民解放军第四十一医院、[[武汉|武汉]]的中国人民解放军第三一五医院、[[南京军事学院|南京军事学院]]附属医院合并成立'''中国人民解放军第三〇九医院'''<ref name=bjw>[http://www.beijing.cn/rcpage/yiyuan/page/100435.shtml 309医院（中国人民解放军第309医院），北京网，于2014-12-12查阅] {{webarchive|url=https://web.archive.org/web/20150923183348/http://www.beijing.cn/rcpage/yiyuan/page/100435.shtml |date=2015-09-23 }}</ref>，隶属[[中国人民解放军总后勤部|中国人民解放军总后勤部]]。2004年5月，根据[[中央军委|中央军委]]下达的部队精简命令，解放军第三〇九医院被并入中国人民解放军总医院，对内称'''中国人民解放军总医院第三〇九临床部'''，对外称'''中国人民解放军总医院第二附属医院'''。2009年8月，解放军总医院第二附属医院转隶[[中国人民解放军总参谋部|中国人民解放军总参谋部]]，称'''中国人民解放军总参谋部总医院'''<ref name="ifeng">{{Cite web|url=http://news.ifeng.com/a/20141212/42704332_0.shtml|title=落马女少将沾丈夫光升迁 分房时发动“群众斗群众”|accessdate=2017-11-11|author=|date=2014-12-12|work=|publisher=凤凰网}}</ref>。

2008年9月21日，北京市第一家国际氩氦冷冻肿瘤治疗中心在该医院挂牌<ref name=bjw/>。

2009年，该医院被确定为工伤医疗康复双定点医院；2011年，中国人民解放军第三〇九医院康复医学中心在[[小西天|小西天]]中心门诊部揭牌成立。

2012年秋，该医院由发热门诊扩建成的新门诊大楼举行落成仪式<ref name=ifeng/>。

2014年，[[海淀区|海淀区]]西北部医联体启动。这个区域医疗联合体以中国人民解放军第三〇九医院为核心医院，这是首个驻京军队[[三级甲等医院|三级甲等医院]]加入医联体。

2005年底至2012年，[[高小燕|高小燕]]大校担任该医院[[政治委员|政治委员]]。2014年11月27日，军方发布消息称，已晋升为少将任职于[[解放军信息工程大学|解放军信息工程大学]]的高小燕因在担任309医院政委期间涉嫌受贿而被[[中国人民解放军总直属军事检察院|军队检察部门]]带走。高小燕成为[[中国共产党第十八次全国代表大会|中共十八大]]以来第一位涉嫌腐败而落马的女将军。<ref name=ifeng/>

2016年1月，在[[深化国防和军队改革|深化国防和军队改革]]中，该医院原上级单位[[中国人民解放军总参谋部|中国人民解放军总参谋部]]撤销，该医院转隶[[中央军委机关事务管理总局|中央军委机关事务管理总局]]，仍称中国人民解放军第三〇九医院<ref>{{Cite web|url=http://www.309yy.com/News/Articles/Index/304|title=我院医疗队赴四川甘孜参加大型公益巡诊义诊活动|accessdate=2017-11-11|author=|date=2016-08-19|work=|publisher=解放军第309医院}}</ref>。

2018年，[[中国人民解放军陆军总医院|原陆军总医院]]、[[中国人民解放军海军总医院|原海军总医院]]、[[中国人民解放军第三〇二医院|原第三〇二医院]]、原第三〇九医院、[[中国人民解放军第三〇七医院|军事科学院军事医学研究院原附属医院]]、[[中国人民武装警察部队总医院|原武警总医院]]均已由[[中国人民解放军总医院|中国人民解放军总医院]]暂管<ref>{{Cite web |url = https://www.thepaper.cn/newsDetail_forward_2334813 |title = 官方披露：原陆军总医院等单位已由解放军总医院暂管 |author =  |date =2018-08-10  |publisher =澎湃新闻  |language =  |accessdate =  |quote =  }}</ref>。2018年11月有关报道该院已更名为'''中国人民解放军总医院第八医学中心'''， 正式归属中国人民解放军总医院<ref name=搜狐/>。
