'''中国人民解放军空军总医院'''，位于[[北京市|北京市]][[海淀区|海淀区]][[阜成路|阜成路]]30号（西钓鱼台），隶属[[中国人民解放军空军后勤部|中国人民解放军空军后勤部]]，是中国人民解放军空军最高医疗机构，[[三级甲等医院|三级甲等医院]]。

==简介==
1950年5月，[[中国人民解放军空军|中国人民解放军空军]]决定在北京建设空军医院。1954年7月，番号定为“中国人民解放军空军总医院”。<ref name=jj/>1956年10月25日，以陆军第十一军医院为基础，正式成立空军总医院。<ref>{{Cite web |url =http://news.xinhuanet.com/mil/2016-12/08/c_129396375.htm  |title = 空军总医院历经六十年发展壮大、迎接改革强军考验记事 |author =  |date =2016-12-08  |publisher = 新华网 |language =  |accessdate =  |quote =  }}</ref><ref>空军总医院建院五十周年纪念活动回眸，空军医学杂志2006(4)</ref><ref name=liushi>{{Cite web |url =http://www.kj-hospital.com/in_info.asp?infoId=2227  |title =忆传统 颂改革 谱新篇 医院召开建院60周年纪念大会  |author =  |date =2016-10-28  |publisher =空军总医院  |language =  |accessdate =  |quote =  }}</ref>空军总医院是中华人民共和国军事航空医学的发源地，自建院起主要承担空军飞行人员航空病、常见病诊疗防治研究，医学鉴定任务，负责空军部队和联勤部队官兵的健康保障及疾病救治，并对地方医疗卫生机构开展卫生帮扶工作。<ref name=jj>{{Cite web |url = http://www.kj-hospital.com/dh.asp?id=295&sid=298 |title = 空军总医院简介 |author =  |date =  |publisher =空军总医院  |language =  |accessdate = 2017-01-11 |quote =  }}</ref>

1978年[[改革开放|改革开放]]后，空军总医院迎来新发展。到20世纪末，空军总医院已发展出一批国家和军队重点学科。作为中华人民共和国首批对外开放的部队医院，空军总医院面向地方患者提供医疗服务。1994年，成为国家和军队首批三级甲等医院和国际爱婴医院。2012年，以高分通过三级甲等医院复评。<ref name=jj/>

1996年11月，空军总医院接受“921工程”的任务。随后为[[神舟五号|神舟五号]]、[[神舟六号|神舟六号]]载人飞船升空提供了卫勤保障。2009年8月，空军总医院完成了中国第二批预备[[航天员|航天员]]和第一批女预备航天员的初选任务。2010年1月，空军总医院党委作出《关于大力加强临床航空医学建设的决定》，成立临床航空医学研究所。<ref name=jj/>

1960年4月，空军总医院成功抢救162名[[苯|苯]]中毒中学生。1976年[[唐山大地震|唐山大地震]]，空军总医院派出医疗队。空军总医院还参加过“[[九八抗洪|九八抗洪]]”、抗击[[非典|非典]]、[[2008年北京奥运会|2008年北京奥运会]]安保。2008年，四川[[汶川大地震|汶川大地震]]，空军总医院派出100多人的医疗队参加抗震救灾，获授“全国抗震救灾英雄集体”称号。2009年医院启动“心蕾工程”后，累计救治贫困地区及少数民族地区[[先天性心脏病|先天性心脏病]]患儿1042名。医院获[[中华人民共和国民政部|中华人民共和国民政部]]、[[中国人民解放军总政治部|中国人民解放军总政治部]]评为全国拥政爱民模范单位。2013年后，医院完成了野战医疗所和空运医疗队各项演习演练卫勤保障任务。<ref name=jj/>
