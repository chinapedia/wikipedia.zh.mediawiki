'''中国人民解放军空军工程大学'''，简称'''空军工程大学'''，本部位于[[陕西省|陕西省]][[西安市|西安市]][[灞桥区|灞桥区]]长乐东路甲字1号，隶属[[中国人民解放军空军|中国人民解放军空军]]，是空军综合性军事高等院校，是中国人民解放军五所综合大学之一、中国人民解放军对外开放院校<ref>空军工程大学，思想教育研究2005(10)</ref>。

==沿革==
;中国人民解放军空军工程学院
*1959年12月31日，中央军委决定，在空军新组建一所工程学院。1960年1月，空军党委确定自空军部队和院校选调干部，成立工程学院筹建处。1961年1月，步兵第一九〇师师部机关和步兵第五六八团作为工程学院组建基础。1961年2月，国防部发布命令，授予学院“'''中国人民解放军空军工程学院'''”番号<ref name=gchxy>{{Cite web |url =http://www.gk114.com/yjsyx/3095/3190/2011071246383.htm  |title = 空军工程大学工程学院 |author =  |date = 2011-07-12 |publisher = 中国招生考试网 |language =  |accessdate =  |quote =  }}</ref>。1961年9月1日，中国人民解放军空军工程学院正式成立。
*1961年开始招收学员。学院设办公室、训练部、政治部、院务部。学院下辖飞机发动机系、军械系、特设系、无线电系、机场建筑系。1963年2月增设工程系<ref name=sanqinyou/>。
*1969年11月9日，更名'''中国人民解放军空军第一专科学校'''<ref name=gchxy/><ref name=sanqinyou/>。
*1975年5月7日，升格'''中国人民解放军空军工程学院'''<ref name=gchxy/><ref name=sanqinyou/>。
*1976年5月14日，学院成为空军直属单位<ref name=sanqinyou/>。
*1978年2月，学院设训练部、政治部、研究部、院务部，下设一系、二系、三系、四系、五系、六系和外训大队<ref name=sanqinyou/>。
*1979年4月，空军工程学院列入全国重点院校<ref name=sanqinyou/>。
*1981年11月7日，撤销研究部，增设函授研究生教育处<ref name=sanqinyou/>。
*1982年3月18日，中央军委批准空军工程学院为学士学位授予单位，专业是机场建筑和洞库、营房建筑<ref name=sanqinyou/>。
*1986年8月11日，国务院学位委员会批准空军工程学院为硕士学位授予单位，学科专业是通信与电子、岩土工程、公路工程、航空发动机、飞行器导航控制系统、航空宇航系统工程、管理工程<ref name=sanqinyou/>。

;中国人民解放军空军导弹学院
*中国人民解放军空军第八航空预备学校：1953年6月，在[[河北|河北]][[保定|保定]]建立'''中国人民解放军第八航空预科总队'''。1954年，改建'''中国人民解放军第八航空预备学校'''<ref>{{Cite web |url = http://news.ifeng.com/a/20170824/51742448_0.shtml |title = 空军二预校——我的母校!_ |author =  |date = 2017-08-24 |publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref>。
*中国人民解放军空军技术学校：1955年，华北军区第四文化师范学校、[[防空军高射炮兵学校|防空军高射炮兵学校]]（第三营修理系）合并，建立'''中国人民解放军防空军技术学校'''。1957年，更名'''中国人民解放军空军技术学校'''。1958年，防空军技术学校（高炮技术干部培训）分出，并入[[空军高炮学校|空军高炮学校]]。
*1958年6月4日，空军下达了组建空军导弹学校的命令，以驻[[河北省|河北省]][[保定市|保定市]]的第八航空预备学校、驻[[吉林省|吉林省]][[长春市|长春市]]的空军技术学校为基础，在保定成立空军导弹学校。1958年9月29日，国防部发布命令，命名空军导弹学校为'''中国人民解放军空军第十五航空学校'''。1958年10月正式成立<ref name=sanqinyou/>。
*1959年2月，空军第十五航空学校由河北省保定市迁至[[陕西省|陕西省]][[三原县|三原县]]<ref name=sanqinyou/>。
**1959年，部分分出，参与合组[[海军特种技术学校|海军特种技术学校]]，后为[[海军航空工程学院|海军航空工程学院]]。
**1959年，部分分出，并入[[西安炮兵学校|西安炮兵学校]]，后为[[第二炮兵工程学院|第二炮兵工程学院]]。
*1960年1月，更名'''中国人民解放军空军高级专科学校'''<ref name=sanqinyou/>。
*1963年3月，更名'''中国人民解放军空军第四高级专科学校'''<ref name=sanqinyou/>。
*1965年7月，升格'''中国人民解放军空军技术学院'''<ref name=sanqinyou/>。
*1969年11月，改建'''中国人民解放军空军第二专科学校'''<ref name=sanqinyou/>。
*1975年3月，改建'''中国人民解放军空军第二高射炮兵学校'''<ref name=sanqinyou/>。
*1978年10月，升格'''中国人民解放军空军第二高射炮兵学院'''<ref name=sanqinyou/>。
*1983年7月，更名'''中国人民解放军空军地空导弹学院'''<ref name=sanqinyou/>。
*1986年6月，更名'''中国人民解放军空军导弹学院'''。学院（及1958年后的其前身各校）设训练部、政治部、院（校）务部、技术部（1986年6月以后撤销技术部），自1958年至1960年辖三个学员大队；1960年至1978年10月辖5个系，1978年10月以后辖4个系；1986年6月空军导弹学院调整了系的设置，设雷达工程、计算工程、导弹工程、指挥工程4个系。学院有硕士学位授予权。学制2至4年不等,培训分为研究生、大学本科、大学专科、中专、士官5个层次<ref name=sanqinyou/>。

;中国人民解放军空军电讯工程学院
*1950年，建立'''中国人民解放军西北航空处西安通信总队'''。设在西安西门外飞机场跑道旁，此处在[[中华民国|中华民国]]时期称为“劳动营”（即用来关押被扣留的“投奔[[延安|延安]]人员”的集中营）。设4个学员大队。首先开展“劳动建校”。
*1951年，更名'''中国人民解放军第三航空预科总队'''。团级编制，[[杜宇波|杜宇波]]任校长，[[陈昂|陈昂]]任副校长，[[史道|史道]]任政委。
*1954年，更名'''中国人民解放军第三航空预备学校'''。
*1957年，组建'''中国人民解放军空军通信学校'''<ref name=dianxun>{{Cite web |url = http://news.xinhuanet.com/mil/2008-01/14/content_7420030.htm |title = 空军电讯工程学院 |author =  |date = 2008-01-14 |publisher =新华网  |language =  |accessdate =  |quote =  }}</ref>。
**1958年，空军通信学校原第三预备航空学校部分分出，参与合组[[中国人民解放军第十六航空学校|中国人民解放军第十六航空学校]]。
*1969年，撤销空军通信学校。
**空军通信学校（无线电部分）分出，参与合组[[空军第四专科学校|空军第四专科学校]]，后为[[空军雷达学院|空军雷达学院]]。
*1974年，复建'''中国人民解放军空军通信学校'''。
*1986年，升格'''中国人民解放军空军电讯工程学院'''。1986年开始招收培养硕士生，1990年被批准为硕士学位授予单位<ref name=dianxun/>。
*1992年10月，空军电讯工程学院并入[[西安通信学院|西安通信学院]]<ref name=dianxun/>。
*1993年7月，西安通信学院原空军电讯工程学院部分再次分出，恢复'''中国人民解放军空军电讯工程学院'''<ref name=dianxun/>。

;中国人民解放军空军工程大学
*1999年7月，空军工程学院、空军导弹学院、空军电讯工程学院合并，组建'''中国人民解放军空军工程大学'''<ref name=gaijian/><ref name=xiaoxun/>。1999年7月22日，中国人民解放军空军工程大学成立大会在西安举行，空军司令员[[刘顺尧|刘顺尧]]宣读了中央军委主席[[江泽民|江泽民]]签署的命令，并向大学授军旗，空军政委[[乔清晨|乔清晨]]为校牌揭幕<ref>{{Cite web |url =http://www.people.com.cn/rmrb/199907/23/newfiles/wzb_19990723001027_3.html  |title = 空军工程大学组建 |author =  |date =  |publisher = 人民日报1999年7月23日，第3版 |language =  |accessdate =  |quote =  }}</ref>。三个学院分别改为空军工程大学工程学院、空军工程大学导弹学院、空军工程大学电讯工程学院<ref name=yuanxi/>。到2010年代，该校主要为空军航空兵、地面防空兵、通信兵等兵种培养高层次工程技术军官和指挥军官，承担着空军70%地面军官和20多个国家留学生的培训任务<ref name=xunli/>。
*2004年，空军工程大学新组建理学院。2012年，空军工程大学的3所专业学院分别更名为航空航天工程学院、防空反导学院、信息与导航学院，并新组建空管领航学院（由空军工程大学航空管制系、[[空军第二飞行学院|空军第二飞行学院]]预警指挥引导系合并组建）、装备管理与安全工程学院（在空军工程大学工程学院航空装备管理工程系的基础上创建）<ref name=gaijian/><ref name=xunli/><ref name=yuanxi/>。
*2008年1月，空军工程大学与其中心校区所在的[[西安市|西安市]][[灞桥区|灞桥区]][[浐灞生态区|浐灞生态区]]签订军地共建协议<ref>{{Cite web |url = http://news.sina.com.cn/c/2008-01-18/142713285219s.shtml |title = 空军工程大学与浐灞生态区签订军地共建协议 |author =  |date = 2008-01-18 |publisher = 新浪 |language =  |accessdate =  |quote =  }}</ref>。
*2014年，新一届学校党委班子研究确立了“面向空天 脚踏实地 传承创新”校训<ref name=xiaoxun>{{Cite web |url = http://www.sxdaily.com.cn/n/2015/0427/c266-5671409.html |title = 空军工程大学新校训镌刻历史与时代的烙印 |author =  |date = 2015-04-27 |publisher = 陕西传媒网 |language =  |accessdate =  |quote =  }}</ref>。
*2017年，在[[深化国防和军队改革|深化国防和军队改革]]中，中国人民解放军空军第一航空学院并入空军工程大学，改为[[中国人民解放军空军工程大学航空机务士官学校|中国人民解放军空军工程大学航空机务士官学校]]<ref name=gaijian>{{Cite web |url = http://www.thepaper.cn/newsDetail_forward_1703862 |title =原空军第一航空学院已改建为空军工程大学航空机务士官学校  |author =  |date = 2017-06-08  |publisher =  澎湃新闻|language =  |accessdate =  |quote =  }}</ref><ref name=xunli>{{Cite web |url =  http://www.mod.gov.cn/services/2017-06/08/content_4782380.htm|title = 2017军校巡礼第九站：空军工程大学（附报考指南） |author =  |date =2017-06-08  |publisher =中华人民共和国国防部  |language =  |accessdate =  |quote =  }}</ref>。
*2017年7月21日，中国人民解放军空军工程大学成立大会在西安举行，空军政委[[于忠福|于忠福]]将[[中国人民解放军军旗|中国人民解放军军旗]]授予该校校长[[张凤鸣|张凤鸣]]、政委[[朱瑞_(空军)|朱瑞]]<ref name=liangxiang>{{Cite web |url =  http://www.thepaper.cn/newsDetail_forward_1746377|title =空军工程大学军政主官亮相：张凤鸣、朱瑞分任校长、政委  |author =  |date = 2017-07-29  |publisher =澎湃新闻  |language =  |accessdate =  |quote =  }}</ref>。

==专业设置==
2017年，学校有28个本科专业（方向）、63个硕士学位授权点、36个博士学位授权点、8个博士后科研流动站。拥有5个军队重点学科，2个国家重点学科，16个军队（省）级重点实验室和实验教学示范中心，1个国家重点实验室，3个国家级实验教学示范中心。在2012年全国一级学科整体水平评估中，2个学科（航空宇航科学与技术、兵器科学与技术）进入前十名。2017年招生的本科专业有<ref name=xunli/>：
*机械工程
*机械电子工程
*电气工程及其自动化
*电子信息工程
*通信工程
*导航工程
*飞行器动力工程
*飞行器适航技术
*武器系统与工程
*武器发射工程
*运筹与任务规划
*火力指挥与控制工程
*空天防御指挥与控制工程
*航空管制与领航工程
*军事设施工程
*场站管理工程
*指挥信息系统工程
*航空装备工程
*雷达工程
*导弹工程
*无人装备工程<ref name=xunli/>

*中心校区：位于陕西省西安市[[灞桥区|灞桥区]]长乐东路甲字1号。设防空反导学院（前身是空军导弹学院）、空管领航学院、装备管理与安全工程学院、理学院<ref name=yuanxi/>。
*鹿塬校区：位于陕西省西安市灞桥区。设航空航天工程学院（前身是空军工程学院）<ref name=yuanxi/>。
*桃园校区：位于陕西省西安市[[莲湖区|莲湖区]]。设信息与导航学院（前身是空军电讯工程学院）<ref name=yuanxi/>。
