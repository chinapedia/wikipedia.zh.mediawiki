'''中国人民解放军陆军防化学院'''，简称'''陆军防化学院'''，位于[[北京市|北京市]][[昌平区|昌平区]]，隶属[[中国人民解放军陆军|中国人民解放军陆军]]，担负为全军培养中、初级防化指挥军官和中、高级防化工程技术军官的任务<ref>{{Cite web |url = http://www.81.cn/rdzt/2014/0421bkjx/2015-05/05/content_6474395.htm |title =解放军防化学院简介  |author =  |date = 2015-05-05 |publisher = 中国军网 |language =  |accessdate =  |quote =  }}</ref><ref name=xunli>{{Cite web |url = http://www.mod.gov.cn/services/2017-06/14/content_4782858.htm |title =2017军校巡礼第二十四站：陆军防化学院（附报考指南）  |author =  |date =2017-06-14  |publisher = 中华人民共和国国防部 |language =  |accessdate =  |quote =  }}</ref>。

== 沿革 ==
*1950年11月，中央军委军训部部长[[萧克|萧克]]向中央建议成立防化学兵学校，并建议野战军和兵团各编防化科，军编防化股，师、团、营设防化参谋。萧克的建议获[[周恩来|周恩来]]赞成<ref name=dumo>{{Cite web|url = http://military.china.com/zh_cn/history4/62/20090519/15489009.html|title = 新中国的防化学兵：一支与“毒魔”打交道的部队|author = |date = 2009-05-19|publisher = 中华网|language = |accessdate = |quote = |archive-url = https://web.archive.org/web/20090605200953/http://military.china.com/zh_cn/history4/62/20090519/15489009.html|archive-date = 2009-06-05|dead-url = yes}}</ref>。
*1950年12月11日，[[毛泽东|毛泽东]]亲自批准成立'''中国人民解放军化学兵学校'''<ref name=huiyi>{{Cite web |url = http://mt.sohu.com/20170324/n484628206.shtml |title = 装满青春回忆的军校——解放军防化学院 |author =  |date = 2017-03-24 |publisher =搜狐  |language =  |accessdate =  |quote =  }}</ref><ref name=xiangmo>{{Cite web |url =  http://china.cnr.cn/news/200708/t20070801_504529520.shtml|title = 特别报道：降魔神兵建奇功 |author =  |date = 2007-07-29 |publisher = 中国广播网 |language =  |accessdate =  |quote =  }}</ref>。校址在[[四川省|四川省]][[江津县|江津县]]。1951年9月迁至[[河北省|河北省]][[昌平县|昌平县]]（今[[北京市|北京市]][[昌平区|昌平区]]）<ref name=cidian> {{Cite book | author = 中国人民解放军总参谋部兵种部防化编研室 | title = 核化生防护大辞典 | location =  | publisher = 上海辞书出版社 | date =2000年  | pages =  | ISBN =  | accessdate =  | url =  | language =  | quote = 化学兵学校 }} </ref>。
*1958年，更名'''中国人民解放军防化学兵学校'''<ref name=huiyi/>。
*1961年1月25日，更名'''中国人民解放军第一防化学兵学校'''（校址北京市昌平县）；原防化学兵学校部分人员与陆军某团机构组成'''中国人民解放军第二防化学兵学校'''（校址[[浙江省|浙江省]][[江山县|江山县]]）<ref name=cidian/>。
*1962年10月，第一防化学兵学校与第二防化学兵学校合并为'''中国人民解放军防化学兵学校'''<ref name=cidian/>。
<br/>
*1960年到1962年，[[中国人民解放军军事工程学院|中国人民解放军军事工程学院]]分建与扩建，将新建的防化兵工程系调出，移交给有关兵种，拟扩建成单独的兵种工程技术学院——'''中国人民解放军防化学兵工程学院'''<ref name=huiyi/>。1960年11月28日，[[中国人民解放军总参谋部|中国人民解放军总参谋部]]批复同意防化学兵工程学院编制序列。1961年7月正式成立。
*1962年1月5日，由防化学兵工程学院筹建处、[[中国人民解放军军事工程学院|中国人民解放军军事工程学院]]第六系、中国人民解放军第二政治学校、守备第二十三师机关等单位在[[吉林省|吉林省]][[长春市|长春市]]组建成立'''中国人民解放军防化学兵学院'''，隶属防化学兵部领导。1969年2月撤销<ref name=jlszjsz> {{Cite book | author = 吉林省地方志编纂委员会 | title = 吉林省志·卷十四·军事志 | location =  | publisher = 吉林人民出版社 | date = 1996年 | pages =146、148  | ISBN =  | accessdate =  | url =  | language =  | quote =  }} </ref>。
**中国人民解放军第二政治学校：1954年，以中国人民解放军第五政治学校为基础，在长春市成立'''中国人民解放军第二政治学校'''。1961年撤销<ref name=jlszjsz/>。
<br/>
*1969年，防化学兵学院、防化学兵学校合并，部分教职工到北京密云组建'''中国人民解放军防化技术学校'''<ref name=huiyi/>。
*1975年，更名'''中国人民解放军防化学校'''<ref name=huiyi/>。
*1978年，在北京昌平县重建升格成为'''中国人民解放军防化学院'''<ref name=huiyi/>。
*1986年，更名'''中国人民解放军防化指挥工程学院'''<ref name=huiyi/>。副军级，隶属[[中国人民解放军总参谋部|中国人民解放军总参谋部]]<ref>{{Cite web |url =http://edu.people.com.cn/n/2015/0326/c354322-26754201.html  |title =第二章  普通高校专业解析（十五）  |author =  |date = 2015-03-26 |publisher =  人民网|language =  |accessdate =  |quote =  }}</ref>。
*2011年，更名'''中国人民解放军防化学院'''<ref name=huiyi/>。
*2015年12月，在[[深化国防和军队改革|深化国防和军队改革]]中，该校由隶属[[中国人民解放军总参谋部|中国人民解放军总参谋部]]转隶[[中国人民解放军陆军|中国人民解放军陆军]]<ref name=xfd>[http://army.81.cn/content/2016-07/13/content_7152472.htm 防化学院助力打造国家核应急先锋队，中国陆军网，2016-07-13]</ref>。
*2016年7月12日，防化学院举行“国家核应急行动技术支持中心”和“中国核应急救援队理论教学基地”成立揭牌仪式<ref name=xfd/>。
*2017年，在[[深化国防和军队改革|深化国防和军队改革]]中，调整组建'''中国人民解放军陆军防化学院'''<ref>{{Cite web |url = http://news.ifeng.com/a/20170530/51183107_0.shtml |title =  2017年军队招生单位名录发布 军改后院校新名称曝光|author =  |date = 2017-05-30 |publisher =凤凰网  |language =  |accessdate =  |quote =  }}</ref>。

==专业设置==
2017年时，学院以工学、军事学门类为主，承担学历教育、任职教育双重任务。开设防化兵指挥、生化防护工程、装备保障指挥、核技术与安全等4个本科专业，拥有9个硕士学位授予点、1个博士学位授予点。2017年招生的本科专业有<ref name=xunli/>：
*作战指挥
*辐射防护与核安全<ref name=xunli/>
