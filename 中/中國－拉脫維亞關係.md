'''中國－拉脫維亞關係'''，是指歷史上的[[中國歷史|中國]]和[[拉脫維亞歷史|拉脫維亞]]（包括[[中華人民共和國|中華人民共和國]]和[[拉脫維亞共和國|拉脫維亞共和國]]）之間的雙邊關係。中國和拉脫維亞在現代的兩國建交前已有貿易和文化上的接觸，中華人民共和國和拉脫維亞共和國則在1991年9月12日建立外交關係。

== 歷史 ==
在[[第二次世界大戰|第二次世界大戰]]時，有拉脱维亚人到中國参與[[抗日戰爭|抗日戰爭]]<ref>{{cite web |title=中国驻拉脱维亚使馆举办庆祝抗战胜利65周年招待会 |url=http://lv.chineseembassy.org/chn/xwdt/t736884.htm |website=中华人民共和国驻拉脱维亚共和国大使馆 |accessdate=2015-08-18 |archiveurl=https://archive.is/20150818164409/http://lv.chineseembassy.org/chn/xwdt/t736884.htm |archivedate=2015-08-18 |deadurl=no |date=2010-09-02 }}</ref>。

拉脱维亚曾是[[蘇聯|蘇聯]]的領土；1991年8月22日，{{link-ru|拉脫維亞蘇維埃社會主義共和國最高蘇維埃|Верховный Совет Латвийской ССР}}宣佈拉脱维亚恢复独立<ref>{{cite web |title=拉脱维亚国家概况 |url=http://www.fmprc.gov.cn/mfa_chn/gjhdq_603914/gj_603916/oz_606480/1206_607040/ |website=中华人民共和国外交部 |accessdate=2015-08-18 |archiveurl=https://archive.is/20150818155354/http://www.fmprc.gov.cn/mfa_chn/gjhdq_603914/gj_603916/oz_606480/1206_607040/ |archivedate=2015-08-18 |deadurl=no }}</ref>。在{{link-en|苏联国务委员会|State Council of the Soviet Union}}承认包括拉脱维亚在內的[[波羅的海三國|波羅的海三國]]独立後，中華人民共和國也在同年9月7日承认這些国家独立，並派遣時任[[中華人民共和國外交部副部长|外交部副部长]]的[[田曾佩|田曾佩]]訪問這三國<ref>{{cite book|author1=中华人民共和国外交部外交史编辑室|title=中国外交概览|publisher=世界知识出版社|year=1992}}</ref>{{Rp|231}}。

翌日，中華人民共和國派代表到波罗的海三国，與這些國家谈判建交<ref name=Esta>{{cite web |title=苏联解体：中国包机出访与独联体国家进行建交谈判 |url=http://military.china.com/history4/62/20150512/19673496_all.html |website=中华网 |accessdate=2015-08-19 |archiveurl=https://archive.is/20150719083736/http://military.china.com/history4/62/20150512/19673496_all.html |archivedate=2015-07-19 |deadurl=yes |date=2015-05-12 }}</ref>。代表人員先到中国駐[[列宁格勒|列宁格勒]]（現稱[[圣彼得堡|圣彼得堡]]）[[总領事馆|总領事馆]]，从总領事馆借來两辆汽车，先到[[爱沙尼亚|爱沙尼亚]]，然後到拉脱维亚和[[立陶宛|立陶宛]]；當時，波罗的海三国獨立了不久，所以很愿意得到外國的承认，與中華人民共和國建交的過程大致順利<ref name=Esta/>。波罗的海三国都不同意用[[俄語|俄語]]写建交公报，但這些國家當時沒有人懂讀寫[[中文|中文]]，中華人民共和國也沒人懂得它們的語言；中華人民共和國早已料到這一點，所以特地从驻英国大使馆调来一名秘书，把以中文草擬的建交公报翻譯為[[英語|英語]]<ref name=Esta/>。

1991年9月12日，中華人民共和國與拉脱维亚建立外交关系<ref>{{cite web |title=中国与各国建交简况：欧洲-拉脱维亚 |url=http://nation.chinaso.com/detail/20150420/1000200032788801429497783451526110_1.html |website=中国搜索 |accessdate=2015-08-19 |archiveurl=https://archive.is/20150503051228/http://nation.chinaso.com/detail/20150420/1000200032788801429497783451526110_1.html |archivedate=2015-05-03 |deadurl=no |date=2015-04-20 }}</ref>。在中華人民共和國與波羅的海三國的建交公報中，關於[[台灣問題|台灣問題]]的篇幅較多；拉脫維亞在建交公報中承認世界上只有[[一個中國|一個中國]]、[[台灣|台灣]]是中國一部分，又承認中華人民共和國是唯一合法的中國政府<ref>{{cite journal|journal=新新聞周刊|issue=241–250|publisher=新新聞周刊雜誌社|year=1991}}</ref>。

1992年1月，[[中華民國|中華民國]]外交人員[[蔣孝嚴|蔣孝嚴]]來到波罗的海三国，期間與拉脱维亚建立总领事级外交关係；拉脱维亚與中華民國方面簽訂文件，表示承認台湾是一個[[主权独立国家|主权独立国家]]，認定其管辖權及於[[臺澎金馬地區|臺澎金馬地區]]<ref name=Conflict>{{cite book|author1=张春英|title=“台独”逆流/看台湾丛书|publisher=福建人民出版社|year=2002}}</ref>{{Rp|204}}。及後，中华民国在拉脱维亚首都[[里加|里加]]設置总领事馆<ref name=Conflict/>{{Rp|204}}。中華民國在拉脱维亚设立总领事馆后，
中華人民共和國发表声明，宣布撤走駐拉脱维亚的大使馆，但沒有與該國断絕邦交；同時，中華人民共和國決定增進拉脱维亚周边国家的對華關係，防止類似事件重演<ref>{{cite book|author1=曲星|title=中囯外交五十年|publisher=江苏人民出版社|year=2000|isbn=9787214025579}}</ref>{{Rp|535}}。

1994年7月，拉脱维亚政府派員到中華人民共和國，承诺與中華民國断绝外交领事关系，並與中華人民共和國簽訂關於两国关系正常化的联合公报<ref>{{cite book|title=世界知识年鉴|year=2006|url=https://books.google.com.hk/books?id=Qx3lAAAAMAAJ}}</ref>{{Rp|655}}；中華民國駐拉脱维亚的外交機構亦易名為「[[台北代表處|台北代表團]]」<ref>{{cite journal|title=海外學人|issue=275–287}}</ref>。

2018年9月18日，中国国家主席[[习近平|习近平]]在人民大会堂会见来华出席[[夏季达沃斯论坛|夏季达沃斯论坛]]的拉脱维亚总统[[萊伊蒙茨·維尤尼斯|韦约尼斯]]，韦约尼斯表示，拉中双方保持着良好的政治互信和密切交流，合作潜力巨大。拉脱维亚高度评价“[[一带一路|一带一路]]”重要倡议，认为中东欧和中国合作机制对拉脱维亚非常重要，拉方也愿致力于推动[[中國－歐盟關係|欧中关系]]发展。会见后，两国元首共同见证了双边合作文件的签署<ref>{{Cite web |url = http://www.xinhuanet.com/politics/2018-09/18/c_1123449843.htm |author = 李忠发 |date = 2018年9月18日 |title =  习近平会见拉脱维亚总统韦约尼斯  |publisher = [[新华网|新华网]]}}</ref>。

中国和拉脱维亚在現代的兩國建交前已有貿易來往；1960年，中国大陸從[[拉脫維亞蘇維埃社會主義共和國|拉脫維亞蘇維埃社會主義共和國]]進口了一批{{link-en|拉脱维亚褐牛|Latvian Brown}}，其中有30头牛被分配到[[湖南省|湖南省]]<ref>{{cite journal|author1=施启顺|title=引入拉脱维亚褐牛的初步观察|journal=中国兽医杂志|issue=2|year=1963}}</ref>。

根據{{link-en|經濟複雜性研究中心|The Observatory of Economic Complexity}}網站上的數據，[[中国大陸|中国大陸]]到拉脱维亚的出口貿易曾在2007年至2008年上升到超過4億[[美元|美元]]的高位，雖然其後稍有回落，但在2010年至2012年繼續呈上升趨勢<ref name=tradea/>；2012年，中国大陸出口了5.54億美元到拉脱维亚，最多貨物是[[機械|機械]]設備<ref>{{cite web|author=Alexander Simoes|title=产品中国出口到拉脱维亚 (2012)|url=https://atlas.media.mit.edu/zh/explore/tree_map/hs/export/chn/lva/show/2012/|publisher=Observatory of Economic Complexity|accessdate=2015-08-20}}</ref>。拉脱维亚到中国大陸的出口貿易在2011年由去年的超過4,000萬美元上升至超過7,000萬美元，但其後有回落；2012年，拉脱维亚出口了不足7,000萬美元到中国大陸<ref name=tradeb/>。

== 文化關係 ==
拉脫維亞與中國的文化接觸可追溯至現代的兩國成立前。在1869年生於[[拉脱维亚|拉脱维亚]]的{{link-ru|彼得·彼得羅維奇·施密特|Шмидт, Пётр Петрович (востоковед)}}是一名[[漢學家|漢學家]]，他曾学习[[汉語|汉語]]、[[满語|满語]]和[[蒙古語|蒙古語]]，大學毕业後來到中国进修<ref name=Con/>{{Rp|118}}；他在中國待了三年，期間结识了一些來自[[欧洲|欧洲]]的汉学家，又把自己的所见所闻详细记录下來，常把自己撰寫的文章寄到[[俄羅斯|俄羅斯]]发表<ref>{{cite book|author1=阎国栋|title=俄国汉学史: 迄于1917年|publisher=人民出版社|year=2006|isbn=9787010059464}}</ref>{{Rp|376}}。此後，施密特在俄羅斯的學府教授汉語和满語，並撰寫了《[[官话|官话]]语法试编》一书<ref name=Con>{{cite book|author1=阎国栋|title=俄罗斯汉学三百年/列国汉学史书系|publisher=学苑出版社|year=2007|isbn=9787507729085}}</ref>{{Rp|118}}。

拉脫維亞被認為是波羅的海國家的[[漢學|漢學]]研究中心，當地不少學府提供汉語課程，中華人民共和國也有為拉脫維亞的汉語教學單位提供[[教師|教師]]、[[獎學金|獎學金]]和[[教材|教材]]；在中華人民共和國派遣教師到拉脫維亞前，當地有來自台灣的教師教授汉語<ref>{{cite journal|author1=彭飞|author2=张红|title=拉脱维亚汉语教学的近况|journal=云南师范大学学报(对外汉语教学与研究版)|volume=9|issue=5|year=2011}}</ref>。[[拉脱维亚大学|拉脱维亚大学]]設有一所[[孔子学院|孔子学院]]，学院向拉脱维亚大学学生提供免费汉語教學，非拉脱维亚大学学生則要繳交學費；這所孔子学院主要教授汉语和[[中華文化|中華文化]]，优秀学生有机会得到奖学金，從而到中国[[留学|留学]]<ref>{{cite web |title=拉脱维亚大学孔子学院简介 |url=http://www.ci.lu.lv/?page_id=2 |website=拉脱维亚大学孔子学院 |accessdate=2015-08-20 |archiveurl=https://archive.is/20150820070510/http://www.ci.lu.lv/?page_id=2 |archivedate=2015-08-20 |deadurl=no }}</ref>。

[[香港|香港]]和拉脫維亞的學校曾有對方的交換生，而香港的[[大學|大學]]亦有[[教授|教授]]來自拉脫維亞<ref>{{cite web |title=行政長官款待拉脫維亞總統致辭全文 |url=http://www.info.gov.hk/gia/general/200404/17/0417160.htm |website=政府資訊中心 |accessdate=2015-08-20 |archiveurl=https://archive.is/20150820071536/http://www.info.gov.hk/gia/general/200404/17/0417160.htm |archivedate=2015-08-20 |deadurl=no }}</ref>。
