'''中日韓統一表意文字擴展區B'''（{{lang-en|CJK Unified Ideographs Extension B}}）是一個[[Unicode|Unicode]]標準，在Unicode版本3.1被引入。

擴展B區包含有42,711個新的漢字，位置在 U+20000–U+2A6D6。根據[http://www.cse.cuhk.edu.hk/~irg/irg/N777_CJK_B_CoverNote.pdf IRG N777號文件]，這四萬多個漢字分別從以下字典或字集中取得：
*《[[康熙字典|康熙字典]]》中出現的18,486個未收錄漢字（包括一個在[[補遺|補遺]]篇出現的漢字）；
*《[[汉语大字典|汉语大字典]]》中出現的28,914個未收錄漢字；
*《[[辭源|辭源]]》中出現的66個未收錄漢字；
*《[[辭海|辭海]]》中出現的247個未收錄漢字；
*《[[汉语大词典|汉语大词典]]》中出現的553個未收錄漢字；
*《[[中国大百科全书|中国大百科全书]]》中出現的86個未收錄漢字；
*[[北京大学|北大]][[方正排版系统|方正排版系统]]中出現的65個未收錄漢字；
*《[[四庫全書|四庫全書]]》中出現的522個未收錄漢字；
*[[香港增補字符集|香港增補字符集]]中出現的1,081個未收錄漢字；
*[[日本工業標準|日本工業標準]]的[[JIS_X_0213|JIS X 0213]]第3水準及第4水準的302個未收錄漢字；
*南韓 [[PKS_5700-3|PKS 5700-3:1998]] 中出現的166個未收錄漢字；
*北韓 [[KPS_9566|KPS 9566-97]] 和 [[KPS_10721|KPS 10721-2000]] 國家標準所收錄的5,642個漢字；
*台灣 [[國家標準中文交換碼|CNS 11643]] 的第4至7和15平面所收錄的30,177個漢字；
*越南 [[TCVN|TCVN]]、[[VHN|VHN]] 01:1998 和 VHN 02:1998 所收錄的4,232個[[喃字|喃字]]；
這些漢字中重複的漢字有不少，所以經過整理之後，實際總數只有42,711個漢字。

另外，在 U+2F800–U+2FA1D 的位置，放了542個來自台灣的[[兼容漢字|兼容漢字]]。

因擴展B區在整理上有缺陷，收錄了以下8個本來應該與其他漢字統一的字：<ref>http://std.dkuug.dk/JTC1/SC2/wg2/docs/n2644.pdf</ref><ref>{{Cite web | last=Cook | first=Richard | title=Defect Report on Duplicate Encoded CJK Forms | url=http://std.dkuug.dk/JTC1/SC2/wg2/docs/n2644.pdf | publisher=ISO/IEC JTC1/SC2/WG2 | date=6 October 2003 | accessdate=2012-03-28 }}</ref>
*U+20457 {{Unihan|20457}} = U+34A8 {{Unihan|34A8}}
*U+2420E {{Unihan|2420E}} = U+3DB7 {{Unihan|3DB7}}
*U+27144 {{Unihan|27144}} = U+8641 {{Unihan|8641}}
*U+27EAF {{Unihan|27EAF}} = U+FA23 {{Unihan|FA23}}
*U+23515 {{Unihan|23515}} = U+204F2 {{Unihan|204F2}}
*U+249E9 {{Unihan|249E9}} = U+249BC {{Unihan|249BC}}
*U+2A415 {{Unihan|2A415}} = U+24BD2 {{Unihan|24BD2}}
*U+26866 {{Unihan|26866}} = U+26842 {{Unihan|26842}}
而在 WG2 N1155文件中，亦列出了152對可考慮統一的漢字。
