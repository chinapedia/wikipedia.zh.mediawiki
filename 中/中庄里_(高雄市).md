'''中-{庄}-里'''，位於[[高雄市|高雄市]][[大寮區|大寮區]]北部的一個里，北與後庄里、東和江山里、南和前-{庄}-里、翁園里為鄰<ref name="行政區域圖">高雄縣大寮鄉行政區域圖，內政部，2005年11月</ref>，東至光明路三段、西至鳳捷路，南至翁園圳，北至鳳屏一、二路。人口為高雄市第15多、大寮區最多的一個里，有12,387人，面積2489049.96平方公尺。

== 地名 ==
此地從前大部分為方家家族聚集地，而有方厝之名。村內中最早開發的中心點處，古稱中站仔。昔日在方厝南邊有從事打鐵業的人家聚居而設下店面，所以有打鐵店之名。中站仔的東邊稱頂頭，西邊稱下頭，後統稱中-{庄}-。

== 歷史 ==
從清治到日治初期，今中-{庄}-里地區隸屬[[小竹下里|小竹下里]]大藔庄，庄內有頂大藔、下大藔、後壁藔、芎蕉腳、考潭藔、洪厝埕等聚落<ref>《臺灣堡圖》，臺灣總督府，1904年</ref>。

1920年（[[大正|大正]]9年），全臺行政區劃大改制，原[[小竹上里|小竹上里]]所轄之舊制街庄磚仔磘-{庄}-、翁公園-{庄}-、山仔頂-{庄}-，及小竹下里所轄之拷潭-{庄}-、赤崁-{庄}-、大藔-{庄}-合併成立新制街-{庄}-大寮-{庄}-，隸屬[[高雄州|高雄州]][[鳳山郡|鳳山郡]]。原舊制大藔庄改制並雅化「藔」字而成為「大寮」大字<ref>《新舊對照管轄便覽》</ref>。

1946年（民國35年），大寮-{庄}-改制為大寮鄉，隸屬於高雄縣。2010年12月25日隨[[五都改制|五都改制]]，-{中庄}-村改為高雄市大寮區中-{庄}-里。

== 主要街道 ==
[[鳳屏路|鳳屏一、二路]]（[[台1線|台1線]]）為本里和後-{庄}-里的界線，是本里精華繁榮之路段，連結[[高雄市|高雄市]][[鳳山區|鳳山區]]和[[屏東縣|屏東縣]][[屏東市|屏東市]]之路段，西有光明路三段和松寮路橋，為通往[[鳥松區|鳥松]]的中要路段。住宅區南北向（由東往西排）的街道有：萬隆街、工商路、自立路、信義路、自強街、八德路、民安街、文德街、金華街、仁愛路、保德街、華德街、崇文街、崇民街、華德街、崇仁街、德昌街、忠孝路、勝利街、博愛街、瑞祥街、建國街、武昌街、國華街、新民街、文華街、光明路三段。東西向（由北往南排）的街道有：[[鳳屏路|鳳屏一、二路]]、民和街、四維路、民富街、民安街、文昌路、同德街、永仁街、永安街、永吉街、中-{庄}-路、立德路、永興街、承德街、保德街、打鐵店街。
