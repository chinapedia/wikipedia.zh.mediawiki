'''中华人民共和国宁波海事法院'''，位于[[浙江省|浙江省]][[宁波市|宁波市]][[鄞州区|鄞州区]]中兴路801号，是[[中华人民共和国|中华人民共和国]]的一所[[海事法院|海事法院]]。

== 沿革 ==
1992年12月4日，最高人民法院作出《关于设立宁波海事法院的决定》（法发[1992]40号），根据《全国人大常委会关于在沿海开放城市设立海事法院的决定》，批准在[[浙江省|浙江省]]设宁波海事法院，管辖[[浙江省|浙江省]]所属港口和水域（包括所辖岛屿、所属港口和通海的内河水域）内发生的一审海事、海商案件。1993年1月1日，宁波海事法院开始受理案件。<ref name=jianjie>[http://www.nbhsfy.cn/latest_col.jsp?cid=002001 法院介绍，中华人民共和国宁波海事法院，于2015-04-04查阅]</ref><ref>[http://www.chinalawedu.com/news/1200/23079/23084/23143/2006/3/zh8783112456101360022184-0.htm 最高人民法院关于设立宁波海事法院的决定，1992年12月4日] {{webarchive|url=https://web.archive.org/web/20150408024057/http://www.chinalawedu.com/news/1200/23079/23084/23143/2006/3/zh8783112456101360022184-0.htm |date=2015年04月8日 }}</ref>

==管辖==
宁波海事法院管辖[[浙江省|浙江省]]所属港口和水域（包括所辖岛屿、所属港口和通海的内河水域）内发生的一审海事、海商案件。上诉案件由[[浙江省高级人民法院|浙江省高级人民法院]]管辖。<ref name=jianjie/>

2016年，最高人民法院将相关海事行政案件纳入海事法院管辖范围。2017年2月，宁波海事法院被指定试点管辖海事刑事案件。2017年8月21日，宁波海事法院以交通肇事罪，判处[[马耳他|马耳他]]籍货轮“卡塔利娜”二副艾伦有期徒刑三年六个月，这是中华人民共和国海事法院首次审理海事刑事案件<ref>{{Cite web |url =http://www.thepaper.cn/newsDetail_forward_1770321  |title = 我国海事法院首次审理刑案宣判：致14死5失踪船员判3年半 |author =  |date = 2017-08-22  |publisher =澎湃新闻  |language =  |accessdate =  |quote =  }}</ref>。
