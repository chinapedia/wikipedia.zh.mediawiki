'''中村家住宅'''是[[琉球國|琉球國]][[士族|士族]][[賀氏中村家|賀氏中村家]]的住宅，位於現時[[日本|日本]][[沖繩縣|沖繩縣]][[中頭郡|中頭郡]][[北中城村|北中城村]]，是現存琉球國時代的士族住宅之一。被列為日本指定[[重要文化財|重要文化財]]<ref name="沖縄">{{Cite web |title=指定重要文化財 中村家住宅|url=http://www.nakamura-ke.net/}}</ref><ref name="文化財">{{Cite web |title=沖縄県北中城村 「中村家住宅」|url=http://sumai.panasonic.jp/sumu2/bunka/2011_0420/index.html}}</ref><ref name="観光">{{Cite web|title=国指定重要文化財 中村家住宅|url=http://www.vill.kitanakagusuku.lg.jp/site/view/contview.jsp?cateid=6&id=114&page=1}}{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>。

中村家住宅是少數沒有在[[沖繩島戰役|沖繩島戰役]]中毀壞的珍貴傳統住宅，[[美國治理琉球時期|美國治理時期]]的1956年被當時的[[琉球政府|琉球政府]]列為指定重要文化財。1972年5月15日[[沖繩返還|美國將沖繩移交日本]]的當天，{{link-ja|主屋|母屋}}（{{ryu|r=大家|kn=ウフヤ}}）、[[別屋|別屋]]（[[琉球語|琉球語]]：{{lang|ryu|アシャギ}}）、[[倉庫|倉庫]]（{{ryu|r=高倉|kn=タカグラ}}）、前屋（{{ryu|r=前ぬ家|kn=メーヌヤー}}）及[[豬廁|豬廁]]（[[琉球語|琉球語]]：{{lang|ryu|フール}}）同時列為[[沖繩本島|沖繩本島]]民家中的指定國家[[重要文化財|重要文化財]]。至1981年又把宅地列入指定重要文化財，石牆和[[影壁|影壁]]（{{ryu|r=屏風|kn=ヒンプン}}）則列為重要文化財的附屬部份<ref name="地方">{{Cite web |title=中村家住宅|url=http://www.joyphoto.com/japanese/travel/090131/nakamura.html}}</ref><ref name="施設">{{Cite web |title=中村家住宅施設|url=http://www.nakamura-ke.net/house/}}</ref><ref name="沖縄" />。
== 由來 ==
賀氏中村家的祖先本為{{link-ja|豪農|豪農}}，在[[第一尚氏王朝|第一尚氏王朝]]末年成為當時忠臣[[護佐丸|護佐丸]]的隨扈，1440年（明[[正統|正統]]五年）跟隨護佐丸奉[[尚忠王|尚忠王]]之命從[[讀谷山間切|讀谷山間切]]（今[[沖繩縣|沖繩縣]][[讀谷村|讀谷村]]）移居到[[中城村|中城村]]，後來[[勝連間切|勝連]][[按司|按司]][[阿麻和利|阿麻和利]]出兵[[護佐丸阿麻和利之亂|討伐護佐丸]]後，家族也隨之而離散。至[[第二尚氏王朝|第二尚氏王朝]]後期的1720年（琉球[[尚敬王|尚敬王]]八年，清[[康熙|康熙]]五十九年），賀氏元祖[[賀國鼎|賀國鼎]]（伊集[[親方|親方]]治亮）任[[中城間切|中城間切]]內之地頭職，是賀氏中村家一門之始<ref name="大宗">{{Cite web |title=大宗賀国鼎伊集親方治亮|url=http://www.rekishi-archive.city.naha.okinawa.jp/archives/item6/4897}}</ref><ref name="文化財" />。至1750年（[[尚敬王|尚敬王]]三十八年，清[[乾隆|乾隆]]十五年），領[[仲村渠村|仲村渠]]地頭職、位階為[[親雲上|親雲上]]的賀氏第三代在[[首里|首里]]購入一間士族宅第，然後整座拆除移至[[中城村|中城村]]的[[丘陵|丘陵]]地築成此宅<ref name="沖縄" /><ref name="系図">{{Cite web |title=中村家家系図|url=http://www.nakamura-ke.net/history/}}</ref><ref name="琉球" />。

== 建築概要 ==
[[File:Nakamura_House_Kitanakagusuku24n3104.jpg|thumb]]位於住宅入口和影壁之間]]
[[File:Nakamura's_Residence_(17361650492).jpg|thumb]]

中村家住宅佔地1,560.67平方米，在18世紀中第二尚氏王朝[[尚敬王|尚敬王]]在位期間建成，是糅合士族宅第及上層農家特色的琉球傳統住宅<ref name="地方" />。身為中級士族的賀氏中村家由於同時務農，住宅也糅合了士族家和農民家的特點，在士族居所的形式上加上倉庫、貯物室、畜舍等農家設施。現時整個住宅是由主屋（內有[[廚房|廚房]]（{{ryu|r=台所|kn=トゥングワ}}））、別屋、倉庫、豬廁、前舍、影壁、[[井|井]]（{{ryu|r=井|kn=カー}}）構成<ref name="地方" /><ref name="施設" />。屋頂是以{{link-ja|琉球赤瓦|沖縄赤瓦}}鋪砌，瓦片用[[灰泥|灰泥]]黏合，屋頂上有[[琉球獅子|琉球獅子]]坐鎮，相連的屋頂有如[[山脈|山脈]]延綿起伏。其東、南、西面皆以由{{link-ja|琉球石灰岩|琉球石灰岩}}築成的[[圍欄|圍牆]]環繞，[[石牆|石牆]]內種有[[福木|福木]]作為[[防風林|防風林]]，種植防風林和以石灰黏瓦都是為了預防[[颱風|颱風]]侵襲<ref name="沖縄" /><ref name="建築家">{{Cite web|title=建築家・北野彰作さんのブログ「私の好きな空間016」|url=http://www.o-uccino.jp/kenchikuka/designer_0000000150/blog_0000000538/|deadurl=yes|archiveurl=https://web.archive.org/web/20151119163436/http://www.o-uccino.jp/kenchikuka/designer_0000000150/blog_0000000538/|archivedate=2015-11-19}}</ref>。

琉球傳統的房子重視[[風水|風水]]，中村家住宅也不例外。住宅坐北向南，地形前低後高。入口位於南方，後有一[[影壁|影壁]]可遮擋視線，在風水上是為了防止[[邪靈|邪靈]]和[[煞氣|煞氣]]入侵<ref>{{Cite web |title=ヒンプン（顔隠し掘）|url=http://www.nakamura-ke.net/house/hinpun/}}</ref>。風水學上以東南為最適合人居住的方位，反之西北則為最差的方位。中村家住宅的佈局就把接待王府役人的別屋建在東南方，豬廁則建在西北方。屋頂[[屋檐|出檐]]尺度較深，故有[[檐柱|檐柱]]支撐，下有[[檐廊|檐廊]]（{{ryu|r=雨端|kn=アマハジ}}），這種設計可遮擋強烈的[[陽光|陽光]]和避免[[雨水|雨水]]打進屋內<ref name="文化財" /><ref name="施設">{{Cite web |title=中村家住宅施設|url=http://www.nakamura-ke.net/house/}}</ref><ref name="沖縄" /><ref name="琉球">{{Cite web |title=琉球王国の建築術が遺した一輪の花「中村家住宅」|url=http://www.asahi.com/and_M/interest/SDI2014041043311.html}}</ref><ref name="建築家" />。
[[File:Nakamura_House_Kitanakagusuku18n3104.jpg|thumb]]
[[File:Nakamura_House_Kitanakagusuku20n3104.jpg|thumb]]
現有的瓦葺屋頂是琉球國滅亡、[[廢藩置縣|廢藩置縣]]後的樣式。在琉球國時代，只有高級士族的住宅可以用[[瓦|瓦]]葺屋頂，因此身為中級士族的賀氏中村家，住宅的屋頂只可鋪上[[竹|竹]]片<ref name="建築家" />。整個住宅使用[[羅漢松|羅漢松]]、[[厚皮香|木斛]]等當時為士族宅第專用、[[庶民|庶民]]禁止使用的高級木材。主屋、副屋皆為[[廡殿式|廡殿式]]瓦葺屋頂，倉庫則為[[攢尖頂|攢尖頂]]，以琉球獨特的建築手法融入[[中國傳統建築|中國傳統建築]]和日本[[鎌倉時代|鎌倉]]、[[室町時代|室町時代]]建築樣式建造而成<ref name="文化財" />，為不使用[[釘子|釘子]]的[[榫接屋|榫接建築]]（{{ryu|r=貫木屋|kn=ヌチジヤー}}）<ref name="琉球" />。現存的建築當中除了主屋的廳堂部份外都在琉球國被日本吞併後的[[明治時代|明治時代]]再有重修<ref name="沖縄" /><ref name="観光" />。
[[File:Nakamura_House_Kitanakagusuku05n3104.jpg|thumb]]
主屋面積為174.5平方米，主屋是[[第二尚氏王朝|第二尚氏王朝]]後期流傳至今的樣式，深10.7米、寬9.6米，主屋南部為廳堂，分成三個間隔，由東方順序為[[客廳|客廳]]、[[佛廳|佛廳]]、[[起居廳|起居廳]]。佛廳對正住宅入口，裡面的[[佛壇|佛壇]]本來還有擺放祖先[[牌位|牌位]]，這是由於琉球人重視祖先崇拜。北面的內房作為[[寢室|寢室]]、[[產房|產房]]之用<ref>{{Cite web |title=母屋|url=http://www.nakamura-ke.net/house/mainhouse/}}</ref>。連接正房西側的是廚房，曾在[[明治時代|明治時代]]重修，深8.5米，寬8.7米。內有[[灶神|灶神]]的神位，琉球人毎月初一、十五拜祭灶神。屋頂比較低，這是因為屋頂內的空間是用作貯物，放置柴薪和食材。屋頂開有一個稱為[[息道|息道]]（{{ryu|r=息道|kn=イーチミー}}）的透氣孔。旁邊有面積12[[疊蓆|疊蓆]]的房間<ref>{{Cite web |title=トゥングワ（台所）|url=http://www.nakamura-ke.net/house/kitchen/}}</ref>，用作整理農作物<ref name="沖縄" /><ref name="建築家" /><ref name="観光" />。

別屋的西北角與主屋東南角相連，面積62平方米，深8.6米，寬7.5米，供[[首里|首里]]王府役人到訪巡視時留宿<ref>{{Cite web |title=アシャギ（離れ屋敷）|url=http://www.nakamura-ke.net/house/annex/}}</ref>。倉庫深4.8米，寬3.9m兩層高。下用作貯物，上層則用作[[穀倉|穀倉]]。此倉庫的柱是方柱，與一般琉球倉庫多使用圓柱不同，{{link-ja|屋頂空間|屋根裏}}下方傾斜防止[[老鼠|老鼠]]入侵<ref>{{Cite web |title=高倉（籾蔵）|url=http://www.nakamura-ke.net/house/higestorage}}</ref>。前舍位於西側，面積64.8平方米，深10.4米，寬5.9米，共有兩層，下層為馬屋（{{ryu|r=馬ぬ家|kn=ウマヌヤー}}），用作飼養[[馬|馬]]、[[牛|牛]]、[[山羊|山羊]]等[[家畜|家畜]]<ref name="地方" />，上層則用於存放製[[黑糖|黑糖]]用的[[甘蔗|甘蔗]]<ref>{{Cite web |title=メーヌヤー（家畜小屋兼納屋）|url=http://www.nakamura-ke.net/house/barn/}}</ref>。前舍的後方為[[豬廁|豬廁]]，寬6.4米，深4.7米，用石建成，分成三格，養豬的同時亦為人的[[廁所|廁所]]，以人的排洩物作為豬的[[飼料|飼料]]<ref>{{Cite web |title=フール（豚小屋）|url=http://www.nakamura-ke.net/house/pigpen/}}</ref>。

== 觀光資料 ==
現時中村家住宅屬[[北中城村|北中城村]]的收費觀光設施，每天上午9時至下午5時30分開放供遊客參觀。由[[那霸市|那霸市]]前往該處，經國道58號需時約35分鐘，經[[沖繩自動車道|沖繩自動車道]]的{{link-ja|那霸交匯處|那覇インターチェンジ}}需時約15分鐘<ref name="観光" />。
