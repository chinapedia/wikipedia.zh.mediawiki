'''中共中央办公厅毛主席纪念堂管理局'''（简称'''毛主席纪念堂管理局'''或'''毛管局'''），是[[中共中央办公厅|中共中央办公厅]]下属的正司局级[[事业单位|事业单位]]，[[参照公务员法管理|参照公务员法管理]]，负责[[毛主席纪念堂|毛主席纪念堂]]及[[毛泽东|毛泽东]]遗体的管理工作<ref name="daxiu" /><ref name="shydw">{{cite web |url = http://www.gjsy.gov.cn/cxzl/ |title = 中共中央办公厅毛主席纪念堂管理局 |website = 事业单位在线 |publisher = [[国家事业单位登记管理局|国家事业单位登记管理局]] |language = zh-cn |date = |accessdate = 2018-02-20|quote =  }}</ref>。

==沿革==
中共中央办公厅毛主席纪念堂管理局成立于1977年<ref name=daxiu/>。设在[[北京市|北京市]][[东城区_(北京市)|东城区]][[前门东大街|前门东大街]]11号（即[[荷兰使馆旧址|荷兰使馆旧址]]）<ref name=shydw/>。同在此院办公的还有[[国务院参事室|国务院参事室]]与[[中央文史研究馆|中央文史研究馆]]<ref>{{Cite web |url = http://news.china.com/domestic/945/20160604/22808283.html |title =国务院参事室揭秘:轿车时刻准备将建言送给中央领导  |author =  |date =2016-06-04  |publisher = 中华网 |language =  |accessdate =  |quote =  }}</ref>。

== 职责 ==
毛主席纪念堂管理局负责“管理毛主席纪念堂，纪念[[毛泽东生平|毛主席光辉业绩]]。”具体履行的职能有：<ref name="shydw" /><ref name="daxiu"/>
# 毛主席遗体保护
# [[毛主席纪念堂|毛主席纪念堂]]场所管理与设施维护
# 瞻仰和纪念活动接待
# 老一代革命家业绩宣传
# 全国[[爱国主义教育基地|爱国主义教育基地]]职能履行<ref name="shydw" />
