'''中島壽發動機'''（{{lang-ja|寿〔ことぶき〕エンジン}}）乃[[日本|日本]][[中島飛機|中島飛機]]公司研發製造的氣冷式[[星型引擎|星型]]飛機用[[往复式发动机|活塞發動機]]，採單列9汽缸、底置[[凸輪軸|凸輪軸]]（over head valve，縮寫成OHV）的設置。[[大日本帝國海軍|大日本帝國海軍]]命名為'''壽'''，[[大日本帝國陸軍|大日本帝國陸軍]]則命名成'''Ha-1'''（{{lang-ja|ハ1}}），性能提升的版本則為'''Ha-24'''。關於此系列發動機的生產數量，總數約7千具左右。

== 概要 ==
1923年（[[大正|大正]]12年）起[[英國|英國]][[布里斯托飛機公司|布里斯托飛機公司]]授權中島飛機生產{{link-en|布里斯托Jupiter發動機|Bristol Jupiter}}，於是後者以此型發動機為基礎，另外也受到了[[美國|美國]][[普惠公司|普惠公司]][[普惠R-1340發動機|普惠R-1340發動機]]（中島飛機亦獲得授權生產）的影響。1929年（[[昭和|昭和]]4年）開始研發，公司內部代號為「NAH<ref>「NA」即為「Nakajima Air-cooled（中島氣冷式）」之縮寫，參看{{ja}}[http://www.cit.nihon-u.ac.jp/laboratorydata/kenkyu/publication/journal_a/a40.1.7.pdf 中島91式戦闘機の調査（その４）]，第2頁。</ref>」，雖以布里斯托Jupiter發動機為基礎，但行程設計成160mm，可看得出受到{{link-en|萊特R-1820發動機|Wright R-1820}}的影響。

1931年12月「壽一型」、「壽二型」陸續受到日本軍方採用，接著1934年10月「壽二型改一」（460匹馬力，軍方稱「Ha-24」）；1937年9月「壽二型改二」、「壽二型改三」（630匹馬力）、「壽三型」（690匹馬力，軍方稱「Ha-1甲」）；1938年11月「壽四一型」（650匹馬力，軍方稱「Ha-1乙」）等也獲得採用。其中「壽二型改二」、「壽二型改三」還分成舊、新二種類型，原因是1934年間此二型發動機各試作一具，卻未獲得軍方採用；中島飛機遂另行改做，但仍採用原名。此外像[[中島光發動機|光發動機]]舊稱壽三型，與軍方命名之壽三型也容易產生混淆。至於1931年10月開始著手研發的「壽五型」發動機，公司內部代號是「HS」，輸出為450匹馬力。

== 規格（壽2型） ==
* 型式：氣冷式9汽缸[[星型引擎|星型引擎]]
* [[排氣量|排氣量]]：24.1[[公升|公升]]
* 全長：1,021mm
* [[直徑|直徑]]：1,280mm
* 缸徑×行程：146mm×160mm
* 乾燥重量：350[[公斤|公斤]]
* [[機械增壓器|機械增壓器]]：機械式1段1速
* 供油方式：[[化油器|化油器]]
* 額定[[馬力|馬力]]：500hp / 2,100rpm（[[高度|高度]]2,500[[公尺|公尺]]）
* 起飛馬力：600hp / 2,300rpm
