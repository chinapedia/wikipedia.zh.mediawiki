'''中国人民解放军东部战区海军训练基地'''，位于[[浙江省|浙江省]][[江山市|江山市]]，是[[中国人民解放军东部战区海军|中国人民解放军东部战区海军]]的训练基地。<ref>{{Cite web |url = http://www.mod.gov.cn/photo/2014-09/07/content_4535607.htm |title = 东海舰队某训练基地迎来今年海军首批30名新兵 |author =  |date = 2014-09-07 |publisher = 中华人民共和国国防部 |language =  |accessdate =  |quote =  }}</ref><ref name=hb/>

== 沿革 ==
1983年7月，'''中国人民解放军海军东海舰队训练基地'''成立。该基地由原海军外训队、东海舰队训练中心、干部教导大队、外生培训办公室、东海舰队第一、二、三训练团、东海舰队后勤部医训队等单位合并组建。下设7个大队，分别承担海军舰队部分军官、士官、志愿兵的轮训、培训任务，专业士兵的培训任务及外训任务。专业训练内容主要有：航海、机电、声纳、雷达、测距、信号、报务、枪炮、水雷、导弹、防化、柴油机、卫生、汽车修理等等。全基地设21个教研室，1个翻译室，共有干部、教员、译员370余人。该基地成立后，截至1988年10月底，共培训新兵31733人，专业兵24700人，轮训及培训干部、士官、志愿兵2451人，外军学员557人（来自4个国家）。<ref>{{Cite web |url =http://www.shtong.gov.cn/newsite/node2/node2245/node75342/node75350/node75482/node75484/userobject1ai92179.html  |title =   上海军事志 第四编军事训练、教育 第八章军事院校的教育与训练 第七节中国人民解放军海军东海舰队训练基地|author =  |date =  |publisher =上海通  |language =  |accessdate = 2017-01-07 |quote =  }}</ref>

2016年，在[[深化国防和军队改革|深化国防和军队改革]]中，中国人民解放军海军东海舰队改为[[中国人民解放军东部战区海军|中国人民解放军东部战区海军]]。该基地随之改称'''中国人民解放军东部战区海军训练基地'''（同时仍可称“中国人民解放军海军东海舰队训练基地”）。<ref>{{Cite web |url = http://mil.news.sina.com.cn/china/2016-03-07/doc-ifxqafha0421546.shtml |title = 成立东部战区海军 是不是东海舰队就没有了？ |author =  |date = 2016-03-07 |publisher =新浪  |language =  |accessdate =  |quote =  }}</ref><ref name=hb>{{Cite web |url =  http://www.hbctc.edu.cn/info/1101/2963.htm|title =校领导率队赴海军部队江山训练基地展开调研  |author =  |date = 2016-04-15 |publisher = 湖北交通职业技术学院 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url =  http://military.people.com.cn/n1/2016/0216/c1011-28127360.html|title =  东海舰队训练基地瞄准一线培育一专多能人才|author =  |date = 2016-02-16 |publisher = 人民网 |language =  |accessdate =  |quote =  }}</ref>
