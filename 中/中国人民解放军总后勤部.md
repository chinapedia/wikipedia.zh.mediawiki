'''中国人民解放军总后勤部'''，是[[中央军事委员会|中央军委]]实行总部制时期的[[後勤學|后勤]]工作领导机关，在中央军委领导下，管理中国人民解放军各项后方勤务事宜，组织实施全军后勤保障工作。2016年1月在[[深化国防和军队改革|深化国防和军队改革]]中撤销，改组为'''[[中央军事委员会后勤保障部|中央军事委员会后勤保障部]]'''<ref name=gfs/>。

==沿革==
===红军时期===
1927年8月1日[[南昌起义|南昌起义]]、1927年9月[[秋收起义|秋收起义]]，诞生了[[中国共产党|中国共产党]]领导的军队，部队中专门设有从事后勤保障的人员和机构，后勤保障机构大都沿袭[[北伐战争|北伐战争]]时期[[国民革命军|国民革命军]]的后勤体制设立辎重队、副官处、经理部、军需处等<ref name=gfs>{{Cite web |url = http://gfs.gdut.edu.cn/info/1053/3309.htm |title = 中国人民解放军总后勤部的沿革 |author =  |date = 2011-04-02 |publisher = 广东工业大学国防生网 |language =  |accessdate =  |quote =  }}</ref>。[[中国工农红军|中国工农红军]]成立初期，后勤工作领导机构继续大致沿用北伐战争时期国民革命军的编制，在部队中设军需处、军医处、留守处等分管有关后勤业务<ref name=lishi>曾凡奇、汤家玉，人民解放军总部体制的历史沿革，军事史林2016年7期</ref>。例如1928年4月由[[朱德|朱德]]、[[毛泽东|毛泽东]]领导的[[中国工农红军第四军_(红一方面军)|中国工农红军第四军]]在[[江西省_(中華民國)|江西]][[井冈山|井冈山]]成立时，便设立了专门的各级后勤机构，军部设经理处、军医处、军械处、留守处，团设经理科或辎重队和卫生队，连设司务长和卫生员，并在其中建立中共党组织，设政治委员和政治工作人员<ref>宋淳桐，毛泽东后勤保障思想的分析与启示，传承2016(7):32-34</ref>。

1930年5月中旬，[[中共中央|中共中央]]在[[上海|上海]]秘密召开[[全国苏维埃区域代表大会|全国苏维埃区域代表大会]]的同时，单独举行红军代表会议，共有14个红军军团派代表参加，史称“[[全国红军代表会议|全国红军代表会议]]”，会议颁发了《中国工农红军编制草案》，作为统一的军制<ref>{{Cite web |url = http://dangshi.people.com.cn/GB/151935/176588/176591/10555425.html |title = 全国红军代表会议(1930年5月中旬) |author =  |date =  |publisher =中国共产党新闻网 |language =  |accessdate = 2017-09-27 |quote =  }}</ref>。《中国工农红军编制草案》规定设立红军的经理卫生部。当时，经理卫生部实际上仅负责[[红一方面军|红一方面军]]部分后勤保障和救治红军伤病员的工作<ref name=gfs/>。1930年8月，中共中央决定在[[中共中央军事委员会|中共中央军事委员会]]下设秘书处、总政治部、参谋部、武装工农部、军务部、士兵运动委员会、经理卫生部等七个部委，但始终未能建立<ref name=bianqian>{{Cite web |url = http://news.ifeng.com/a/20170724/51490320_0.shtml |title = 解放军统帅部变迁：建军九十年总部领导体制沿革 |author =  |date =2017-07-24  |publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref>。1931年1月，红军的经理卫生部改称经理部，范树德任部长；同时红军还设立了财务部、军医处，[[杨立三|杨立三]]任财务部部长，[[贺诚_(中将)|贺诚]]任军医处处长。1931年11月[[中华苏维埃共和国|中华苏维埃共和国]]成立后，下设的[[中华苏维埃共和国中央革命军事委员会|中华苏维埃共和国中央革命军事委员会]]（简称“中革军委”）随之在[[江西|江西]][[瑞金|瑞金]]成立。中革军委首次尝试实行总部领导体制，下设总参谋部、总政治部、总经理部、总军医处等机构，其中的'''中央革命军事委员会总经理部'''是中共军队历史上最早设立的总后勤部，范树德任部长。当时总经理部下设总兵站处、抚恤委员会、后方办事处（1932年4月成立）这三大机构，杨立三任总兵站处处长，贺诚兼任抚恤委员会主任，[[高自立|高自立]]后来任后方办事处主任；同时设'''中央革命军事委员会总军医处'''，贺诚兼任处长，总军医处同参谋部（后改称总参谋部）、总政治部、总经理部合称中国工农红军的“四总部”<ref name=gfs/><ref name=bianqian/><ref name=lishi/>。

1932年9月，中央革命军事委员会总经理部改称总供给部，下设军实处（处长[[张元培|张元培]]）、财务处、兵工厂、粮秣处等机构；中央革命军事委员会总军医处改称总卫生部，贺诚任部长兼政委，总供给部和总卫生部也同时归中国工农红军总部领导<ref name=gfs/>。1933年5月，中革军委从前方移至后方瑞金，在前方另行组织中国工农红军总部兼红一方面军总部，原中革军委所属总部转隶红军总部，由红军总部统一指挥<ref name=bianqian/>。1934年2月[[中华苏维埃共和国|中华苏维埃共和国]]第二次全国代表大会后产生了新一届中央革命军事委员会，中国工农红军的后勤保障部门除总供给部、总卫生部外，又设立了'''中国工农红军总兵站部'''，杨立三调任总兵站部部长；[[杨至成|杨至成]]接任总供给部部长，叶季壮仍任政委。1934年4月，叶季壮任总供给部部长兼政委<ref name=gfs/>。

在[[长征|长征]]途中，1935年1月[[遵义会议|遵义会议]]后，根据中革军委指示，总供给部与中革军委第四局合并，叶季壮仍任部长。1935年6月[[红一方面军|红一方面军]]、[[红四方面军|红四方面军]]会师后，红军后勤机构仅设总供给部和总卫生部，[[林伯渠|林伯渠]]任总供给部部长，叶季壮调任军实处处长，贺诚接任总卫生部部长<ref name=gfs/>。

1935年10月中央红军长征抵达陕北。1935年11月，[[中国工农红军西北革命军事委员会|中国工农红军西北革命军事委员会]]和'''中国工农红军西北革命军事委员会后方办事处'''成立，[[周恩来|周恩来]]任办事处主任（后叶季壮接任）。当时红军的主要后勤部门有总供给部（部长叶季壮）、总兵站部（部长杨立三，后杨至成接任）、总卫生部（部长[[黄克诚|黄克诚]]，后[[姬鹏飞|姬鹏飞]]接任）。1936年12月[[西安事变|西安事变]]后不久，中国工农红军西北革命军事委员会后方办事处改称'''中国工农红军总后方勤务部'''，叶季壮兼任部长，[[李文楷|李文楷]]任政委，下辖总供给部、总兵站部、总卫生部，各部主要领导不变<ref name=gfs/>。

===从抗战到内战===
1937年8月，中共中央在[[陕北|陕北]][[洛川|洛川]]召开[[中共中央政治局|中共中央政治局]]扩大会议（[[洛川会议|洛川会议]]），会后成立了[[毛泽东|毛泽东]]任主席的[[中共中央革命军事委员会|中共中央革命军事委员会]]（简称中央军委）。1937年8月25日，根据中央军委命令，陕北主力红军改编为[[国民革命军第八路军|国民革命军第八路军]]，1937年9月11日又改为[[国民革命军第十八集团军|国民革命军第十八集团军]]。[[抗日战争|抗日战争]]期间，中央军委总部设在[[延安|延安]]，对外名义上称“八路军总部”。八路军成立后，设八路军总指挥部（简称“八路军总部”），又称第十八集团军总司令部（简称“集总”），是八路军的最高领导机构。八路军刚成立时，八路军总部机关设司令部（参谋处）、政治部、军需处、军医处、兵站部。八路军出师[[华北|华北]]后，八路军总部随部队开赴[[山西|山西]]抗日前线。此后中共军队的后勤保障建设分成前方和后方两个部分。在[[延安|延安]]的原中国工农红军总后方勤务部精编为供给部（部长[[张元寿|张元寿]]）、卫生部（部长姬鹏飞（吉洛））、副官处（处长[[徐曰文|徐曰文]]），这三个部门直属中央军委总参谋部。八路军前方后勤部门主要有供给部（部长[[赵尔陆|赵尔陆]]、副部长[[谢古今|谢古今]]（黄达））、兵站部（部长杨立三、副部长[[周玉成|周玉成]]）、卫生部（部长[[姜齐贤|姜齐贤]]）。另外八路军总部也设有副官处（处长[[唐延杰|唐延杰]]、副处长[[余忠良|余忠良]]）<ref name=gfs/><ref name=bianqian/>。

1937年10月12日，南方八省红军游击队改编为[[新四军|新四军]]，原中革军委供给局局长[[宋裕和|宋裕和]]任新四军供给部部长。1938年12月，中央军委决定在晋东南成立八路军前方指挥部（简称“前指”），又称第十八集团军前方总部，实际上大致是“集总”的原班人马，前方指挥部设有前方司令部、野战政治部、前方后勤部，杨立三任前方指挥部后勤部部长，周玉成任前方指挥部供给部部长<ref name=gfs/><ref name=bianqian/>。

1939年6月，中央军委成立'''中共中央革命军事委员会后勤部'''，叶季壮任部长兼政委，杨立三任副部长（1940年之后任部长），后勤部下辖供给部、卫生部、副官处。1940年上半年，中央军委后勤部扩编，下辖总务处、供给部（部长[[张永励|张永励]]）、兵站部（部长[[张令彬|张令彬]]）、卫生部（部长[[饶正锡|饶正锡]]）、工业局、副官处。1940年下半年，中央军委后勤部精简为下辖供给部、兵站部、卫生部。1941年下半年，恢复了总务处、工业局<ref name=gfs/>。

1940年12月22日，八路军后勤工作会议在晋东南八路军总部召开，[[彭德怀|彭德怀]]、[[刘伯承|刘伯承]]、[[邓小平|邓小平]]等参加，会议主要由后勤部部长杨立三主持，这样大规模的后勤工作会议在中共军队当时的历史条件下属首次<ref name=gfs/>。

1940年底，八路军前方后勤部进行了调整，杨立三任部长兼政委，[[谢文翰|谢文翰]]任政治部主任，下设供给部（部长周玉成、政委[[周文龙|周文龙]]）、卫生部（部长[[孙仪之|孙仪之]]，后[[钱信忠|钱信忠]]任部长）、军工部（部长[[刘鼎|刘鼎]]）、兵运部（部长[[刘颂|刘颂]]）、财政经济部（部长[[张慕光|张慕光]]）和冀南银行（行长兼政委[[高捷成|高捷成]]）。1942年2月18日，中央军委公布了中共军队历史上最早的两部后勤工作条例《八路军新四军兵站工作条例》、《八路军新四军供给工作条例》。1943年10月，中央军委后勤部精简为仅设供给部、兵站部、卫生部；同时，八路军前方后勤部也精简，杨立三调任八路军前方指挥部副参谋长，八路军前方后勤部精简为仅设供给部（部长周玉成、政委周文龙）。1944年8月，中央军委后勤部仅保留后勤部这一机构名称，所辖后勤事宜均交由八路军（第十八集团军）后勤部统一管理<ref name=gfs/>。

1945年8月[[抗日战争|抗日战争]]结束后，成立[[晋冀鲁豫军区|晋冀鲁豫军区]]，八路军（第十八集团军）前方后勤部门统一改为晋冀鲁豫军区后勤部。1945年11月，中央军委后勤部改为'''中共中央革命军事委员会总后勤部'''，杨立三任总后勤部部长兼八路军（第十八集团军）后勤部部长<ref name=gfs/>。

1947年6月，在[[延安|延安]]成立[[中国人民解放军|中国人民解放军]]总部，根据中央军委命令，取消八路军（第十八集团军）和新四军的番号，全军部队统一称“[[中国人民解放军|中国人民解放军]]”。不久，中央军委总后勤部改称'''中国人民解放军总后勤部'''（杨立三任部长）<ref name=gfs/>。

1948年5月，中国人民解放军总后勤部设兵站、供给、军工、军械四大机构，负责协调各大军区各大野战军的后勤保障工作。1949年6月，中国人民解放军总部又组建卫生部，苏井观任部长。在[[第二次国共内战|第二次国共内战]]中，实际上中国人民解放军各大军区各大野战军的后勤保障工作是由它们各自的后勤部门解决，各大野战军均有自己专门的后勤机构和后勤领导人：[[第一野战军|第一野战军]]后勤司令员兼政委[[刘景范|刘景范]]；[[第二野战军|第二野战军]]后勤司令员兼政委[[段君毅|段君毅]]；第三野战军后勤司令员兼政委[[刘瑞龙|刘瑞龙]]；[[第四野战军|第四野战军]]后勤部第一部长[[周纯全|周纯全]]、第二部长[[李聚奎|李聚奎]]、政委[[杨至成|杨至成]]；[[华北野战军|华北野战军]]后勤司令员杨立三（兼任）<ref name=gfs/>。

===共和国时期===
1949年[[中华人民共和国|中华人民共和国]]成立后，1949年11月改为'''[[中央人民政府人民革命军事委员会|中央人民政府人民革命军事委员会]]总后方勤务部'''（简称总后勤部），杨立三任部长兼政委。1954年1月，中央人民政府人民革命军事委员会在总后方勤务部财务部的基础上单独设立中央人民政府人民革命军事委员会财务部，杨立三任部长。1954年10月11日，总参谋部发出关于军委总部更改名称的通知，中央人民政府人民革命军事委员会总后方勤务部改称'''中国人民解放军总后方勤务部'''（黄克诚任部长兼政委），中央人民政府人民革命军事委员会财务部改称'''中国人民解放军财务部'''（杨立三任部长兼政委）。当时，总后方勤务部所辖部门在建国初期的军需部、军械部、卫生部、营房管理部、油料部、直属工作部（1952年撤销）的基础上，又增加了干部部、训练部、军需生产部、司令部、政治部、车辆兽力管理部、马政局等机构<ref name=gfs/>。

1955年5月，总后方勤务部增设汽车拖拉机管理部（部长[[韩振纪|韩振纪]]）。1955年8月，中国人民解放军财务部改称[[中国人民解放军总财务部|中国人民解放军总财务部]]，[[余秋里|余秋里]]任部长，[[唐天际|唐天际]]任第一副部长。不久，总后方勤务部裁撤马政局、车辆兽力管理部，设立后方工程建筑部、学校管理部<ref name=gfs/>。

1957年初，根据中央军委命令，撤销中国人民解放军总财务部，原总财务部机关并入总后方勤务部，改称总后方勤务部财务部。1957年8月，撤销总后方勤务部后方工程建筑部。1958年4月，撤销总后方勤务部学校管理部。1958年10月，撤销总后方勤务部干部部<ref name=gfs/>。

1959年10月[[庐山会议_(1959年)|庐山会议]]后，[[洪学智|洪学智]]被打入“反党军事俱乐部”免去总后方勤务部部长职务，[[邱会作|邱会作]]任总后方勤务部部长；此前总后方勤务部已裁撤了兽医局、训练部等机构。1960年3月至10月，总后勤部先后增设物资计划部、运输部、科学计划部、物资计划部<ref name=gfs/>。1960年10月1日，根据总参谋部1960年4月19日《关于修改某些机构、职务名称事的通知》，中国人民解放军总后方勤务部改称'''中国人民解放军总后勤部'''，即日起启用新印章。1965年3月至5月，总后勤部增设工厂管理部、军马部、军需部。后来总后勤部油料部、物资计划部、工厂管理部、军马部等机构陆续撤销<ref name=gfs/>。

[[文化大革命|文化大革命]]爆发后，总后勤部在[[中国共产党第九次全国代表大会|中共九大]]之前几乎被紧跟[[林彪|林彪]]的[[邱会作|邱会作]]等人控制，总后勤部副部长[[傅連暲|傅連暲]]、[[饶正锡|饶正锡]]等都遭残酷迫害。1969年9月，[[王希克|王希克]]、[[陈庞|陈庞]]、[[封永顺|封永顺]]、[[伊文|伊文]]、[[张明远|张明远]]、[[严俊_(少将)|严俊]]任总后勤部副部长，[[戴金川|戴金川]]任副政委。1970年7月[[丁先国|丁先国]]任总后勤部副部长，不久[[邱创成|邱创成]]任总后勤部副部长。1971年[[九一三事件|九一三事件]]发生，中央责令邱会作离职反省，同时陈庞、王希克、伊文、张明远、严俊、丁先国、戴金川被免去总后勤部领导职务，不久余秋里奉中央命令主持总后勤部工作。文革期间，总后勤部下属机构进行了很多调整，例如装备部、企业部被撤销<ref name=gfs/>。

1988年9月14日，恢复军衔制，总后勤部部长[[赵南起|赵南起]]被授予上将军衔，政委[[刘安元|刘安元]]、副部长[[宗顺留|宗顺留]]、[[李伦|李伦]]等人被授予中将军衔<ref name=gfs/>。

2016年，在[[深化国防和军队改革|深化国防和军队改革]]中，撤销中国人民解放军总后勤部，设立[[中央军委后勤保障部|中央军委后勤保障部]]<ref name=gai>{{Cite web |url = http://news.ifeng.com/a/20160111/47027222_0.shtml |title = 军委机关由“四总部”改为15个职能部门(名单) |author =  |date = 2016-01-11 |publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref>。

==人民军医出版社==
人民军医出版社由总后勤部政治宣传部主管，总后勤部卫生部主办<ref>{{Cite web |url =http://www.gapp.gov.cn/zongshu/serviceSearchResultbve.shtml?ID=13919  |title =人民军医出版社  |author =  |date =  |publisher =  国家新闻出版广电总局|language =  |accessdate =2017-02-25  |quote =  }}</ref>。2016年3月底，中央军委印发《[[军队和武警部队全面停止有偿服务|关于军队和武警部队全面停止有偿服务活动的通知]]》。2016年5月7日，军队和武警部队全面停止有偿服务试点任务部署会议举行，7个大单位、17个具体单位被列为试点单位，新闻出版等重点项目被纳入试点范畴。自2015年起，各军队和武警部队出版社的新书及新项目均停止审批<ref>{{Cite web|url=http://www.cbbr.com.cn/article/104077.html|title=军队出版社将试点“全面停止”有偿服务|accessdate=2016-05-10|author=|date=2016-05-10|publisher=中国出版传媒网|language=|quote=}}</ref>。
