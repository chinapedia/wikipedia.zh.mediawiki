'''中国人民革命军事博物馆'''，又称'''中央军委政治工作部军事博物馆'''，简称'''军事博物馆'''、'''军博'''，位于[[北京市|北京市]][[海淀区|海淀区]][[复兴路_(北京)|复兴路]]9号，地处[[长安街|长安街]]西延长线上，隶属[[中央军委政治工作部|中央军委政治工作部]]，为副军级单位。

==简介==
[[File:Jiefangjunlishibowuguan1959.jpg|thumb]]
[[File:Beijing_Military_Museum_Front_-_troops_towards_the_sky.jpg|thumb]]
[[File:Military_Museum_of_the_Chinese_People's_Revolution_201709.jpg|thumb]]
1958年9月21日，中国人民革命军事博物馆正式组建。该馆的展览大楼是北京迎接[[中华人民共和国|中华人民共和国]]建国十周年所建造的[[北京十大建筑|十大建筑物]]之一，于1958年10月动工兴建，1959年7月完工。1959年10月内部预展，1960年8月正式对外开放<ref name="jbjj">{{Cite web|url=http://www.jb.mil.cn/gk/jbjj/201103/t20110330_4469.html|title=军博简介|accessdate=2018-11-02|date=2011-03-30|publisher=中国人民革命军事博物馆|archiveurl=https://web.archive.org/web/20150515044358/http://www.jb.mil.cn/gk/jbjj/201103/t20110330_4469.html|archivedate=2015-05-15}}</ref>。

军事博物馆展览大楼与[[北京电报大楼|北京电报大楼]]和[[北京长途电话大楼|长话大楼]]一样，设计上遵循了当时[[斯大林式建筑|苏式建筑]]的风格。与电报大楼和长话大楼不同的是，军事博物馆展览大楼主楼塔尖上托着的是[[中国人民解放军|中国人民解放军]]军徽，而前两者的塔尖上托着的则是[[五角星|五角星]]。

2008年3月1日起，军事博物馆对外免费开放，成为北京市第一个免费开放的国家级博物馆<ref name="jbjj" />。 2008年5月，获[[国家文物局|国家文物局]]评定为“国家一级博物馆”<ref>{{Cite web |url =  http://www.huaxia.com/wh/whxx/2008/00808947.html?oeld3|title =  中国官方公布首批83家“国家一级博物馆”名单|author =  |date = 2008-05-19 |publisher =华夏经纬  |language =  |accessdate =  |quote =  }}</ref>。2010年7月，经[[中共中央|中共中央]]、[[中央军委|中央军委]]批准，军事博物馆展览大楼加固改造工程正式立项，建筑面积将从6万平方米扩大到15万平方米，陈列展览面积将从3万多平方米扩大到将近7万平方米。2012年5月起，为配合该工程，展览大楼内的基本陈列全部拆除并关闭，250多件大型武器装备转移至展览大楼南广场，继续接待观众参观，博兴大厦的临时展览继续开放。2012年8月，[[中国人民解放军总政治部|中国人民解放军总政治部]]和[[中央军委|中央军委]]领导批准了军事博物馆新的陈列体系建设方案。2012年9月21日，加固改造工程开工<ref name="jbjj" /><ref>{{Cite web|url=http://www.jb.mil.cn/gk/jggzgc/201211/t20121130_6382.html|title=军博展览大楼加固改造工程简介|accessdate=2018-11-02|date=2012-11-30|publisher=中国人民革命军事博物馆|archiveurl=https://web.archive.org/web/20150715024459/http://www.jb.mil.cn/gk/jggzgc/201211/t20121130_6382.html|archivedate=2015-07-15}}</ref><ref name="junshi">{{Cite web|url=http://www.js7tv.cn/news/201705_96231.html|title=全国唯一综合性军事博物馆：中国人民革命军事博物馆|date=2017-05-18|publisher=中国军视网}}</ref>。2017年7月21日至10月31日，军事博物馆大楼实施改扩建以来第一次重大主题展览“铭记光辉历史开创强军伟业——庆祝中国人民解放军建军90周年主题展览”举行<ref>{{Cite web|url=https://www.thepaper.cn/newsDetail_forward_1786356|title=军事博物馆馆长董长军：未来在军博或可“开坦克、登航母”|date=2017-09-06|publisher=澎湃新闻}}</ref>。

军事博物馆与北京市合作，以北京市为主导，共同在北京市[[丰台区|丰台区]]创办[[首都军事文化创意产业示范园|首都军事文化创意产业示范园]]，规划用地3000亩，拟设立兵器展示区、体验娱乐区、军事文化创意制作和文物修复区、军训夏令营区、纪念瞻仰区。2012年5月8日，该项目正式签约<ref name="jbjj" />。

2016年改革前，军事博物馆为副军级单位，隶属[[中国人民解放军总政治部|中国人民解放军总政治部]]，行政主管部门为中国人民解放军总政治部直属工作部，业务主管部门为中国人民解放军总政治部宣传部。军事博物馆下设有政治部、办公室、管理处、展陈管理教育部、文物征管部、展陈设计部、展陈研究部、安全保卫部共8个中层单位以及干休所。军事博物馆还设有学术委员会、艺术委员会、文物鉴定委员会、军博书画院<ref name="jbjj" />。

在[[深化国防和军队改革|深化国防和军队改革]]中，2016年1月中国人民解放军总政治部撤销，改设[[中央军委政治工作部|中央军委政治工作部]]。军事博物馆转隶中央军委政治工作部<ref>{{Cite web|url=http://news.163.com/16/0525/11/BNTKQBCF00014AED.html|title=原总政直工部副部长肖安水少将调任军委政治工作部直工局局长|accessdate=2018-11-02|date=2016-05-25|publisher=网易}}</ref>。

== 展览 ==
[[File:DF2_in_Museum.JPG|right]]]]

军博展览大楼南广场东西两侧分别陈列有海军的[[导弹快艇|导弹快艇]]、[[东风-2型导弹|东风-2型导弹]]、曾参加[[首都各界庆祝中华人民共和国成立50周年大会|2009年国庆阅兵]]的后勤保障车以及10门[[清代|清代]][[铁炮|铁炮]]（东西各五门）。

军博的陈列体系主要为[[中国军事史|中国军事史]]。2012年陈列拆除关闭前，基本陈列共有10个：古代战争馆（西三楼）、近代战争馆（西四楼）、[[土地革命战争|土地革命战争]]馆（东一楼）、[[抗日战争|抗日战争]]馆（东二楼）、[[第二次国共内战|全国解放战争]]馆（东二楼）、新中国国防和军队建设成就馆（东三楼）、[[抗美援朝战争|抗美援朝战争]]馆（东三楼一厅）、兵器馆（中央北大厅）、中国人民解放军对外军事交往友谊馆（四楼中厅）、军事艺术馆（东配楼一至三层）<ref name="jbjj" />。此外，展览大楼内还设有服务设施：服务部（三楼中厅）、礼品馆。展览大楼的西一楼、西二楼、东四楼为临时展厅，用于临时展览。

[[File:The_two_cars.JPG|thumb]]和[[朱德|朱德]]的座车]]
[[File:M1896_pistol_Zhu_De_used_during_the_Nanchang_Uprising.jpg|left]]
古代战争馆和近代战争馆主要陈列由[[春秋|春秋]][[战国|战国]]到[[辛亥革命|辛亥革命]]前这一段历史中的战争资料，其中展出的展品大多为仿制品。在近代战争馆，陈列有少数[[辛亥革命|辛亥革命]]之中使用的[[枪|枪]]支。土地革命战争馆、抗日战争馆和全国解放战争馆陈列[[中国共产党|中国共产党]]成立之后发起的[[第一次国共内战|土地革命战争]]、[[长征|长征]]以及后来[[中国抗日战争|抗日战争]]和[[第二次国共内战|第二次国共内战]]时期的资料和文物。陈列的文物有[[南昌起义|南昌起义]]时起义军使用的旗帜、当时的[[军装|军装]]、番号布、缴获[[国民革命军|国民革命军]]枪支武器以及当时根据地制造的枪支。抗日战争和解放战争馆中除照片文字等展板资料以外，陈列有缴获[[日军|日军]]枪支、[[日本军刀|军刀]]以及很多其他缴获品。

[[File:Beijing_Military_Museum_Main_Hall.jpg|left]]]]
[[File:Captured_T-62_tank.jpg|thumb]]中缴获的[[苏联红军|苏军]][[T-62主战坦克|T-62主战坦克]]]]
兵器馆为军博之中最大的展馆，共占用两层的中厅以及东西两个室外展棚。其中一层以及室外展棚主要用于大型武器展示，二层则为[[枪|枪]][[炮|炮]][[彈藥|弹药]]展示。一层入门后展示了两辆汽车，分别为[[朱德|朱德]]和[[毛泽东|毛泽东]]曾经的座车，两车均参加了1949年[[中华人民共和国|中华人民共和国]][[开国大典|开国大典]]。而在两边陈列的则为[[八路军|八路军]]和[[中国人民解放军|解放军]]从[[日本|日本]]占领军和[[国民革命军|国民革命军]]缴获的重武器，有日本[[92式加农炮|92式加农炮]]、美制[[加农炮|加农炮]]、一辆日本轻型坦克等。一层主厅内中部陈列有[[中国海军|中国海军]]舰艇的模型、[[火箭|火箭]]和[[弹道导弹|弹道导弹]]模型，还有一枚[[东风-1短程弹道导弹|东风-1短程弹道导弹]]实物的壳体。中间展区部左边陈列着[[歼-5|歼-5]]、[[歼-8|歼-8]]等4架飞机以及[[空舰导弹|空舰导弹]]壳体；右边则展出有中型坦克和“猛士”[[越野车|越野车]]等实物展品。最两边的两个展区则展出[[迫击炮|迫击炮]]，[[榴弹发射器|榴弹发射器]]等步兵用重型装备。西侧室外展棚内陈设有参与击沉[[太平号|太平号]]的鱼雷快艇以及[[北洋水师|北洋水师]][[镇远舰|镇远舰]]的铁锚，东侧室外展棚主要为[[第二次国共内战|第二次国共内战]]时期解放军缴获的[[国军|国军]]重型武器如坦克、飞机，还有苏联援助的坦克以及[[珍宝岛事件|珍宝岛事件]]中缴获的苏联[[T-62|T-62]]坦克，建国后击落的美军[[侦察机|侦察机]]残片等。二层则主要展示各种枪支弹药的实物，诸如[[AK-47突击步枪|AK-47]]，[[喀秋莎火箭炮|喀秋莎火箭弹]]（壳体），[[水雷|水雷]]（已去消战斗部）等。

2012年8月，[[中国人民解放军总政治部|中国人民解放军总政治部]]和[[中央军事委员会|中央军委]]领导批准的军事博物馆新的陈列体系建设方案，以军事历史为主，辅以军事科技、军事艺术陈列。陈列重点是[[中国共产党|中国共产党]]领导的革命战争陈列、新中国国防和军队建设陈列、人民军队专题陈列、兵器陈列。此外设有中国历代军事陈列、军事科技陈列、军事艺术陈列、临时展览<ref name=junshi/>。

截至2017年，军事博物馆收藏34万多件文物及藏品。其中国家一级文物1793件，大型武器装备290多件，艺术品1600多件，对外军事交往中受赠礼品2754件<ref name=junshi/>。
