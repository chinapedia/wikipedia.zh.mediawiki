'''中国人民解放军海军舟山水警区'''，机关驻地[[浙江省|浙江省]][[舟山市|舟山市]]，是[[中国人民解放军东部战区海军|中国人民解放军东部战区海军]]的[[水警区|水警区]]。<ref>{{Cite web |url =http://mznews.zjol.com.cn/system/2014/08/01/020175747.shtml  |title = 舟山市: 积极推进军民深度融合发展 |author =  |date = 2014-08-01 |publisher = 浙江在线民政视窗 |language =  |accessdate =  |quote =  }}</ref>

== 沿革 ==
1950年9月，以华东军区海军第四舰队领导机关为基础，在[[上海|上海]]组建成立'''华东军区海军舟山基地'''（1002部队）。1950年10月驻防舟山定海，机关设司令部、政治部、供给处。1951年3月，舟山基地与第七舰队领导机关合并，称'''华东军区海军舟山基地兼第七舰队'''。1952年9月，改称'''中国人民解放军海军舟山基地'''。1955年10月，改属[[中国人民解放军海军东海舰队|中国人民解放军海军东海舰队]]建制，下辖舰艇部队、岸防兵、专业勤务部队。1957年，该部队代号改称1881部队，1962年改称4007部队，1975年改称37502部队。2003年12月编制体制调整，更名为'''中国人民解放军海军舟山保障基地'''。<ref>{{Cite web |url =http://www.guancha.cn/military-affairs/2014_01_20_200963.shtml  |title = 中国海军新年度远海训练编队启航 将首次前往东印度洋进行训练 |author =  |date = 2014-01-20 |publisher = 观察者网 |language =  |accessdate =  |quote =  }}</ref><ref name=jzsjsz>浙江省军事志 第二节 海军，方志出版社，1999年</ref>2012年，海军舟山保障基地撤销。

2012年，'''中国人民解放军海军舟山水警区'''成立，隶属[[中国人民解放军海军东海舰队|中国人民解放军海军东海舰队]]。<ref>{{Cite web |url = http://www.zjwmw.com/07zjwm/system/2012/03/07/018247640.shtml |title = 舟山市举行青少年“学习雷锋精神 争当新区建设先锋”主题活动启动仪式 |author =  |date =2012-03-07  |publisher = 浙江文明网 |language =  |accessdate =  |quote =  }}</ref>海军舟山水警区以原海军部队为框架，与原海军舟山保障基地司政机关整编组建而成。部队驻[[浙江省|浙江省]]等地。除负责辖区作战、机动支援作战等任务外，还负责核生化防护、外事保障等任务。<ref name=ifeng>{{Cite web |url =  http://news.ifeng.com/mil/2/detail_2012_07/12/15976180_0.shtml|title =海军上海水警区成立 肩负扫雷防化支援作战任务  |author =  |date =2012-07-12  |publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref>
