'''中山门'''，是[[西安城墙|西安城墙]]东侧的一座城门，俗称“小东门”，位于[[长乐门|长乐门]]以北，东新街东端。1927年开辟，以纪念国民革命领袖[[孙中山|孙中山]]而得名。

== 历史 ==

[[File:孫中山先生.jpg|thumb]]]]民国16年（1927年），西北国民联军总司令[[冯玉祥|冯玉祥]]督陕驻新城，阔建清代满城内东西大街为东新街，并于街东端城墙开辟城门，于敌台两侧各砌一个券洞。时值[[孙中山|孙中山]]先生逝世两周年，便以“中山”之号命名城门，以纪念国民革命领袖。中山门并列的两个门洞，亦分别取名“东征门”和“凯旋门”，以表示[[国民革命军北伐|国民革命军北伐]]必胜之决心与深意。同年5月1日，[[冯玉祥|冯玉祥]]率部东出[[潼关|潼关]]参加[[国民革命军北伐|北伐]]，即由中山门出城。<ref>{{cite book |author=西安市地方志办公室 |date=2012年11月第1版| edition= |title=民国西安词典 |url=http://xadfz.xa.gov.cn/difangzhinew/muluFrame.jsp?bookname=minguo_xian_cidian |location=西安 |publisher=陕西人民出版社 |page=57 |isbn=978-7-224-10402-8 |author-link= }}</ref><ref>{{cite web |url=https://mp.weixin.qq.com/s?__biz=MzI0Njc3MDg5NQ==&mid=2247484547&idx=1&sn=117d4da9c0469511042c790db24c5bcd |format= |title= 古城西安十八门 |author=碧海风  |date= 2019-01-31|website= |publisher= 碧海风云|access-date= |quote=}}</ref>

因中山门开辟之前，东城墙仅有东门[[长乐门|长乐门]]一座城门，故中山门被民众俗称为“小东门”。

2014年之前，中山门北侧门洞原存中华民国时期的木质城门板，当时也是西安城墙各城门中，唯一一处保存着门板的城门。2014年城门板被拆除。

== 地理位置 ==
中山门外，南北向为环城东路（北段），门内南北向为顺城东路（北段）。中山门向东（城外）连接伍道什字西街，向西（城内）连接东新街。
