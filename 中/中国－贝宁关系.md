'''中國－贝宁关系'''，是指[[中國歷史|歷史上的中國]]和[[贝宁歷史|贝宁]]、以至現在[[中華人民共和國|中華人民共和國]]與[[贝宁共和國|贝宁共和國]]之間的雙邊關係。中華人民共和國和贝宁在1964年11月首次建交，兩国其後斷交，後在1972年12月恢復外交關係；中國曾援建貝寧多個政府部門的辦公大樓，並在2015年第一季成為了貝寧第二大的進口來源地。

== 歷史 ==
贝宁的[[贝宁文化|文化]]和民俗中的一些元素與中國有關連，這被認為是古代的中國與贝宁有來往的證據<ref name=history>{{cite book|author1=李兆良|title=宣德金牌啟示錄：明代開拓美洲|publisher=聯經出版事業公司|year=2013|isbn=9789570842838}}</ref>{{Rp|274}}。贝宁人民懂得用[[失蠟法|失蠟法]]製造[[銅器|銅器]]，據說這是[[鄭和|鄭和]][[鄭和下西洋|南下西洋]]時傳到當地的，不可能是由[[西方國家|西方國家]]傳入<ref name=history/>{{Rp|274}}；贝宁等[[非洲西部|非洲西部]]國家流行的[[伏都教|伏都教]]與[[漢語|漢語]]「巫毒」發音相同，信仰相似，有指是鄭和從中國傳到世界各地的習俗<ref name=history/>{{Rp|275}}。此外，贝宁有婦女穿戴頸圈的民俗，這與[[雲南|雲南]]地區和[[緬甸|緬甸]]的風俗相若；據考證，全球各地只有這三個地區的女性會穿戴這樣的頸圈<ref name=history/>{{Rp|274–275}}。

[[達荷美共和國|達荷美共和國]]（即今天的贝宁）在1960年8月宣佈獨立為國；獨立後，較為接受[[共產主義|共產主義]]的[[蘇魯－米岡·阿皮蒂|蘇魯－米岡·阿皮蒂]]成為達荷美共和國的副總統，但該國的[[總參謀長|總參謀長]]一職卻由傾向[[反共主義|反共主義]]的[[克里斯托夫·索格洛|克里斯托夫·索格洛]]擔任<ref name="otC">{{cite journal|author1=王文隆|title=臺海兩岸政府在達荷美的外交競逐（1964-1966）|journal=國史館館刊|issue=21|date=2009年9月}}</ref>。

達荷美在1962年1月與當時已退到[[台灣|台灣]]的中華民國建立外交關係，又於1964年11月宣佈與中華人民共和國建交<ref name=sides/>；此前，[[美國國務院|美國國務院]]曾派員會晤達荷美總理[[朱斯坦·阿奧馬德貝－托莫坦|朱斯坦·阿奧馬德貝－托莫坦]]，企圖勸阻達荷美與中華人民共和國建交，但未果，阿奧馬德貝更反問為何「[[两个中国|两个中国]]」不可行<ref name="otC"/>。翌年上旬，中華人民共和國派出[[中國駐貝寧大使列表|大使]]到當地，但中華民國的外交人員沒有離開<ref name=sides>{{cite book|title=中國國際法與國際事務年報|issue=3|publisher=臺灣商務印書館|year=1987}}</ref>{{Rp|20}}。於是，中華人民共和國向达荷美提出交涉，达荷美方面决定要求中華民國撤使，否則将驱逐中華民國的人員，中華民國只好撤走使節，並宣佈與達荷美斷交<ref>{{cite book|author1=黄华|title=亲历与见闻: 黄华回忆录|publisher=世界知識出版社|year=2007|isbn=9787501230174}}</ref>{{Rp|130}}；索格洛協助中華民國撤館，又保管大使館上的[[中華民國國旗|中華民國國旗]]<ref name="otC"/>。

後來，克里斯托夫·索格洛在[[政變|政變]]後掌權，成為達荷美的領導人<ref name="otC"/>。1966年，他在沒有任何理由的情況下單方面與中華人民共和國斷交<ref>{{cite book|title=中国二十世紀通鉴: 1901-2000|issue=4|publisher=线装書局|year=2002}}</ref>{{Rp|4348}}，並要求[[新華社|新華社]]的人員離境；同年4月中旬，中華民國與達荷美先後簽署合作協定和建交公報，正式建立外交關係<ref name="otC"/>。1972年10月，贝宁軍官[[馬蒂厄·克雷庫|馬蒂厄·克雷庫]]發動政变，宣佈往後要推行[[社会主义|社会主义]]形式的发展，並在1975年把国號改為「贝宁」<ref>{{cite web|title=贝宁国家概况|url=http://www.fmprc.gov.cn/mfa_chn/gjhdq_603914/gj_603916/fz_605026/1206_605124/|publisher=中华人民共和国外交部|accessdate=2015-06-16|archiveurl=https://web.archive.org/web/20150320051758/http://www.fmprc.gov.cn/mfa_chn/gjhdq_603914/gj_603916/fz_605026/1206_605124/|archivedate=2015-03-20|deadurl=no}}</ref>；他領導下的政府在1972年12月29日与中華人民共和國恢復外交關係，贝宁此後一直與之維持良好的邦交，也接受了中華人民共和國的各項援助<ref>{{cite web|title=中国同贝宁的关系|url=http://www.fmprc.gov.cn/mfa_chn/gjhdq_603914/gj_603916/fz_605026/1206_605124/sbgx_605128/|publisher=中华人民共和国外交部|accessdate=2015-06-16|archiveurl=https://web.archive.org/web/20150430110458/http://www.fmprc.gov.cn/mfa_chn/gjhdq_603914/gj_603916/fz_605026/1206_605124/sbgx_605128/|archivedate=2015-04-30|deadurl=no}}</ref>。

2008年3月，[[贝宁国民议会|贝宁国民议会]]议长[[马蒂兰·纳戈|马蒂兰·纳戈]]在與中國駐當地大使會晤時抨擊[[2008年臺灣入聯公投|臺灣入聯公投]]，認為中華民國舉行此公投旨在推動[[台灣獨立|台灣獨立]]；他表示，贝宁国民议会將代表所有贝宁人，反对台湾舉行此[[公投|公投]]，而贝宁也會堅決反對台灣獨立為國的主張<ref>{{cite web|title=贝宁国民议会议长谴责台湾当局推动“入联公投”|url=http://bj.chineseembassy.org/chn/bzwhzx/t417154.htm|publisher=中华人民共和国驻贝宁共和国大使馆|accessdate=2015-06-17|archiveurl=https://web.archive.org/web/20150617000502/http://bj.chineseembassy.org/chn/bzwhzx/t417154.htm|archivedate=2015-06-17|deadurl=no|date=2008-03-21}}</ref>。

2012年，中華人民共和國出口了24.1億[[美元|美元]]到贝宁，不少貨品是[[紡織品|紡織品]]、[[假髮|假髮]]、[[鞋類|鞋類]]等衣飾<ref name=clothes/>；贝宁在同年出口了2.39億美元來華，貨品主要是[[棉花|原棉]]和[[木材|木材]]<ref name=wood/>。2015年第一季，中國成為了贝宁第二大的[[进口|进口]]來源地<ref>{{cite web|title=2015年第一季度中国为贝宁进口贸易第二大伙伴|url=http://bj.mofcom.gov.cn/article/ddgk/zwminzu/201506/20150600999473.shtml|publisher=中华人民共和国驻贝宁共和国大使馆经济商务参赞处|accessdate=2015-06-16|archiveurl=https://web.archive.org/web/20150616170735/http://bj.mofcom.gov.cn/article/ddgk/zwminzu/201506/20150600999473.shtml|archivedate=2015-06-16|deadurl=no|date=2015-06-02}}</ref>。

不少中國企業在贝宁設有代表機構，當中有[[华为|华为]]等的電信企業，也包括[[中铁五局|中铁五局]]等的建設企業<ref>{{cite web|title=驻贝中资机构|url=http://bj.mofcom.gov.cn/article/ddgk/zwfengsu/201504/20150400955370.shtml|publisher=中华人民共和国驻贝宁共和国大使馆经济商务参赞处|accessdate=2015-06-16|archiveurl=https://web.archive.org/web/20150616231710/http://bj.mofcom.gov.cn/article/ddgk/zwfengsu/201504/20150400955370.shtml|archivedate=2015-06-16|deadurl=yes|date=2015-04-28}}</ref>。

== 文化關係 ==
贝宁中国文化中心於1988年成立，以传揚[[中华文化|中华文化]]、支持非洲国家發展當地文化為目標；該中心曾舉辦[[展覽會|展覽會]]、[[音乐会|音乐会]]等的文化活動，開辦[[汉语|汉语]]、[[武术|武术]]和[[太极拳|太极拳]]課程<ref>{{cite web|title=贝宁中国文化中心|url=http://bj.chineseembassy.org/chn/bzwhzx/t235346.htm|publisher=中华人民共和国驻贝宁共和国大使馆|accessdate=2015-06-16|archiveurl=https://web.archive.org/web/20070715011530/http://bj.chineseembassy.org/chn/bzwhzx/t235346.htm|archivedate=2007-07-15|deadurl=no|date=2006-02-14}}</ref>。[[贝宁国立大学|贝宁国立大学]]內的阿波美－卡拉维大学設有[[孔子学院|孔子学院]]；該学院曾向贝宁的[[SOS兒童村|SOS兒童村]]送贈汉语书籍等物資，並計劃在兒童村推行汉语教學<ref>{{cite web|author1=袁江磊|author2=罗璐|title=阿波美卡拉维大学孔子学院举办中华文化体验活动|url=http://www.hanban.edu.cn/article/2015-06/04/content_601785.htm|accessdate=2015-06-17|archiveurl=https://web.archive.org/web/20150617002103/http://www.hanban.edu.cn/article/2015-06/04/content_601785.htm|archivedate=2015-06-17|deadurl=no|date=2015-06-04}}</ref>。

== 援助 ==
中華人民共和國援助贝宁興修多楝政府建築物，包括該国财政部、外交部和发展部的办公大楼；中國也曾在當地援建一座佔地30公顷的[[体育场|体育场]]，以及一座面积逾70000平方米的会议大厦<ref name=assist>{{cite web|title=中贝合作成果展示|url=http://bj.chineseembassy.org/chn/zbgx/t216949.htm|publisher=中华人民共和国驻贝宁共和国大使馆|accessdate=2015-06-16|archiveurl=https://web.archive.org/web/20070716111335/http://bj.chineseembassy.org/chn/zbgx/t216949.htm|archivedate=2007-07-16|deadurl=no}}</ref>。此外，中国政府在贝宁援建了一所於2014年10月開始營運的职业技术学校，向當地青年提供[[職業教育|職業教育]]，冀能舒解贝宁青年[[失业率|失业率]]高的问题<ref>{{cite web|title=我援贝职业技术学校举行开学仪式|url=http://bj.chineseembassy.org/chn/zbgx/t1202744.htm|publisher=中华人民共和国驻贝宁共和国大使馆|accessdate=2015-06-16|archiveurl=https://web.archive.org/web/20150616162400/http://bj.chineseembassy.org/chn/zbgx/t1202744.htm|archivedate=2015-06-16|deadurl=no|date=2014-10-21}}</ref>。

醫療援助方面，中華人民共和國在[[莫諾省|莫諾省]]援建了一座[[医院|医院]]，設有逾百個床位，提供[[內科學|内科]]、[[外科|外科]]、[[妇产科|妇产科]]等11个[[醫學|醫學]]學科的服務；從1978年至2005年，中國共派遣了13批医疗队到贝宁<ref name=assist/>。
