'''中国人民解放军陆军炮兵防空兵学院'''，简称'''陆军炮兵防空兵学院'''，本部位于[[安徽省|安徽省]][[合肥市|合肥市]]，隶属[[中国人民解放军陆军|中国人民解放军陆军]]，是承担炮兵、防空兵指挥军官培训任务的高等教育院校，同时承担研究生教育、士官教育、外国军事留学生培训、现职干部轮训、优秀士兵保送入学学员培训等任务<ref name=xunli>{{Cite web |url =  http://www.81.cn/jwgz/2017-06/11/content_7634751.htm|title =2017年军校巡礼第十八站：陆军炮兵防空兵学院（附报考指南）  |author =  |date = 2017-06-11 |publisher =中国军网  |language =  |accessdate =  |quote =  }}</ref>。

== 沿革 ==
===中国人民解放军陆军军官学院===
* 1977年，中央军委决定新建'''中国人民解放军炮兵技术学院'''。筹备工作随即展开。隶属[[中国人民解放军总参谋部|中国人民解放军总参谋部]]直接领导，执行军级职权<ref name=ahsjsz/>。 
* 1978年，经中央军委批准，由河南[[郑州|郑州]]迁至安徽合肥<ref name=ahsjsz/>。
* 1986年，更名'''中国人民解放军合肥炮兵学院'''。
* 1999年7月，中央军委主席[[江泽民|江泽民]]签署命令，原中国人民解放军合肥炮兵学院和中国人民解放军南京炮兵学院合并组建'''中国人民解放军炮兵学院'''<ref>{{Cite web |url = http://news.xinhuanet.com/misc/2000-12/31/content_486068.htm |title =中央军委主席江泽民签署命令　组建四所新的军队院校  |author =  |date = 1999-07-02 |publisher =新华网  |language =  |accessdate =  |quote =  }}</ref>。学院本部在[[合肥|合肥]]，在[[南京|南京]]设分院。
* 2011年8月改建为'''中国人民解放军陆军军官学院'''。同时南京分院（原南京炮兵学院部分）独立，恢复为中国人民解放军南京炮兵学院。
* 2016年，在[[深化国防和军队改革|深化国防和军队改革]]中，由[[中国人民解放军总参谋部|中国人民解放军总参谋部]]转隶[[中国人民解放军陆军|中国人民解放军陆军]]<ref>{{cite web|title=解放军陆军军官学院已由原总参转隶陆军|url=http://www.thepaper.cn/newsDetail_forward_1456517|publisher=澎湃新闻|accessdate=2016-05-24}}</ref>。

===中国人民解放军南京炮兵学院===
{{参见|中国人民抗日军政大学#四分校|华东军事政治大学}}
* [[中国人民抗日军政大学|中国人民抗日军政大学]]四分校演变为[[华东军事政治大学|华东军事政治大学]]炮兵大队。
* 1947年3月，[[华东军事政治大学|华东军事政治大学]]炮兵大队分出，改建'''[[华东野战军|华东]]特种兵纵队特科学校'''（简称“特纵特科学校”），在[[山东省|山东省]][[沂水县|沂水县]]满堂坡成立<ref name=rmwcl/><ref name=jsshzhjshzh> {{Cite book | author =江苏省地方志编纂委员会编  | title = 江苏省志·军事志 | location =  | publisher =军事科学出版社  | date =2000年  | pages =  219-220| ISBN =  | accessdate =  | url =  | language =  | quote =  }} </ref>。先后参与[[济南战役|济南战役]]、[[淮海战役|淮海战役]]等<ref name=rmwcl/>。
* 1949年1月，进驻江苏省[[徐州市|徐州市]]九里山。1949年5月，进驻[[南京市|南京市]][[汤山|汤山]]，接收原[[中华民国|中华民国]]陆军炮兵学校校址。1949年11月，学校由团扩大为师的编制<ref name=jsshzhjshzh/>。
* 1950年12月15日，更名'''华东军区炮兵学校'''<ref name=jsshzhjshzh/>。
* 1951年3月10日，中央军委电令，更名'''中国人民解放军第三炮兵学校'''。后改归中央军委建制<ref name=jsshzhjshzh/>。
* 1956年2月，更名'''中国人民解放军南京炮兵学校'''<ref name=jsshzhjshzh/>。
* 1957年，学院开始担负外训和迎外任务<ref name=jsshzhjshzh/>。
* 1969年12月，撤销南京炮兵学校<ref name=jsshzhjshzh/>。
* 1974年，在南京恢复组建'''中国人民解放军炮兵学校'''。1975年3月开学。隶属[[中国人民解放军炮兵|军委炮兵]]（1982年9月改属[[中国人民解放军炮兵|中国人民解放军总参谋部炮兵部]]）<ref name=jsshzhjshzh/>。
* 1977年底，更名'''中国人民解放军第一地面炮兵学校'''<ref name=jsshzhjshzh/>。
* 1986年7月，升格为'''中国人民解放军南京炮兵学院'''<ref name=jsshzhjshzh/>。
* 1999年，与合肥炮兵学院、淮安炮兵学院合并为[[中国人民解放军炮兵学院|中国人民解放军炮兵学院]]，作为南京分院<ref name=rmwcl/>。
<br/>
*1955年5月，中共炮兵党委根据中央军委关于组建军兵种学院的指示，成立'''中国人民解放军炮兵学院筹备委员会'''<ref name=zgqnw>{{Cite web |url = http://agzy.youth.cn/qsnag/agbk/201211/t20121122_2640431_4.htm |title = 延安炮兵学校 |author =  |date =  2012-11-22|publisher =中国青年网  |language =  |accessdate =  |quote =  }}</ref>。
*1956年6月9日，军委5号文件批复，确定以[[南京军事学院|南京军事学院]]炮兵系为基础分建炮兵学院。除军事学院炮兵系外，还抽调原军委炮兵学校、[[中国人民解放军沈阳高级炮兵学校|高级炮兵学校]]的部分教员骨干和炮兵学院筹备组一起，组成'''中国人民解放军炮兵学院'''<ref name=zgqnw/><ref name=rmwcl/>。
*1957年7月9日，[[中华人民共和国国防部|中华人民共和国国防部]]向全军发出电令，中国人民解放军炮兵学院正式成立。隶属军委炮兵领导，院址[[河北省|河北省]][[张家口市|张家口市]][[宣化区|宣化镇]]。1957年8月，授予兵团级军旗一面<ref name=zgqnw/>。
*1969年2月19日，根据[[军委办事组|军委办事组]]《军队院校调整方案》中关于“指挥院校一律撤销”的规定，军委炮兵于1969年12月17日撤销炮兵学院<ref name=zgqnw/>。
*1970年1月，炮兵学院成立留守处<ref name=zgqnw/>。
*1974年4月，中央军委决定在原炮兵学院旧址恢复筹建炮兵军政干部学校。同年6月15日，军委炮兵颁发命令：'''中国人民解放军炮兵军政干部学校'''成立。并授予军级军旗一面<ref name=zgqnw/>。
*1977年11月28日，炮兵军政干部学校改称'''中国人民解放军炮兵学院'''。院址不变，仍在宣化<ref name=zgqnw/>。
*1986年6月9日，根据中央军委（1986）七号文件精神，中国人民解放军炮兵学院改称'''中国人民解放军炮兵指挥学院'''<ref name=zgqnw/><ref name=rmwcl/>。
*1992年，根据中央军委1992年九号文件“全军院校编制体系调整改革方案”，将炮兵指挥学院由中级指挥院校调整为初级院校，更名为'''中国人民解放军宣化炮兵学院'''。授予师级军旗一面<ref name=zgqnw/>。
*1993年6月，总参谋部颁发命令，将宣化炮兵学院恢复为'''中国人民解放军炮兵指挥学院'''。授予军级军旗一面<ref name=zgqnw/>。
*2004年，廊坊陆军导弹学院并入，成为其廊坊校区。
**廊坊陆军导弹学院前身是1978年11月成立的炮兵导弹训练团，1979年4月扩建为炮兵导弹训练基地，1983年6月改建为炮兵导弹学校，1986年6月更名为廊坊炮兵导弹学校，1993年6月更名为中国人民解放军廊坊陆军导弹学院。
<br/>
* 2011年11月，中央军委命令，炮兵指挥学院、炮兵学院南京分院合并组建'''中国人民解放军南京炮兵学院'''，级别定为副军级，校址定于[[南京市|南京市]][[江宁区|江宁区]]汤山镇。2011年11月5日，南京炮兵学院正式组建成立，中国人民解放军总参谋长助理[[陳勇_(將軍)|陈勇]]中将为南京炮兵学院成立授旗<ref name=rmwcl>{{Cite web |url = http://js.people.com.cn/html/2011/11/05/44596.html |title = 南京炮兵学院正式组建成立 定为副军级 |author =  |date = 2011-11-05 |publisher = 人民网|language =  |accessdate =  |quote =  }}</ref><ref name=zgqnw/>。
* 2016年，在[[深化国防和军队改革|深化国防和军队改革]]中，由[[中国人民解放军总参谋部|中国人民解放军总参谋部]]转隶[[中国人民解放军陆军|中国人民解放军陆军]]<ref>{{cite web |title=全军后勤最高学府后勤学院已由原总后转隶中央军委训练管理部 |url = http://www.thepaper.cn/newsDetail_forward_1473027 |publisher=澎湃新闻 |date=2016-05-23}}</ref>。

===中国人民解放军防空兵学院===
*1949年9月28日，创建'''中国人民解放军防空学校'''。校址[[沈阳市|沈阳市]]<ref name=fkbxy>{{Cite web |url =http://www.mod.gov.cn/reports/201403/wzry/2014-09/12/content_4536801.htm  |title =防空兵学院  |author =  |date = 2014-09-12 |publisher = 中华人民共和国国防部 |language =  |accessdate =  |quote =  }}</ref>。
*1951年6月，更名'''中国人民解放军高射炮兵学校'''。1959年4月，迁至[[江苏省|江苏省]][[南京市|南京市]][[孝陵卫街道|孝陵卫]]原高级步兵学校校址。1962年9月缩编，迁至[[湖北省|湖北省]][[武昌|武昌]]，与炮兵工程学院对换校址<ref>{{Cite web|url = http://app.nanjing.gov.cn/njnj/sz8_12/8/html/08Noname374.html|title = 第三节 南京解放后军事院校|author = |date = |publisher = 中国南京|language = |accessdate = 2017-06-04|quote = }}{{Dead link|date=2018年10月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。1962年8月由南京迁至武昌张之洞路13号，隶属中央军委炮兵<ref name=wuhan/>。
*1969年9月，撤销高射炮兵学校，在武汉设留守处<ref name=wuhan>{{Cite web|url =http://www.whfz.org.cn/html/2010/cat_309_0126/19148.html|title =武汉解放和新中国时期|author =|date =2010-01-26|publisher =大武汉方志网|language =|accessdate =|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20170803213015/http://www.whfz.org.cn/html/2010/cat_309_0126/19148.html|archivedate =2017年8月3日|df =}}</ref>。
<br/>
*1951年5月21日，[[中南军区|中南军区]]根据中央军委命令，在[[河南省|河南省]]组建'''中国人民解放军第四炮兵学校'''，执行师级权限<ref name=hnshzhjsz>{{Cite web |url =http://www.hnsqw.com.cn/sqsjk/hnsz/jsz/  |title = 河南省志·军事志 第二章 军事体制  第四节 中国人民解放军  三、军事院校 |author =  |date =  |publisher = 河南省情网 |language =  |accessdate = 2017-06-04 |quote =  }}</ref>。
*1956年2月4日，国防部命令第四炮兵学校改为'''中国人民解放军郑州炮兵学校'''<ref name=hnshzhjsz/>。
*1969年2月19日，撤销郑州炮兵学校<ref name=hnshzhjsz/>。
<br/>
*1970年4月1日，奉中央军委命令，在江苏省南京市汤山成立'''中国人民解放军炮兵技术学校'''，同年11月迁驻河南省[[郑州市|郑州市]]。是隶属军委炮兵部领导的师级学校<ref name=hnshzhjsz/>。
*1974年1月，军委炮兵部根据中央军委和总参谋部指示，将炮兵技术学校调整为'''中国人民解放军高射炮兵学校'''，并授师级军旗一面<ref name=hnshzhjsz/><ref name=wuhan/>。
*1986年7月，根据中央军委命令，将高射炮兵学校改称'''中国人民解放军高炮学院'''，担负培养陆军高炮部队初级指挥军官和外国军事留学生的任务<ref name=hnshzhjsz/>。
*1999年，更名'''中国人民解放军郑州防空兵学院'''。
*2004年5月，根据中央军委命令，以郑州防空兵学院为基础，合并炮兵指挥学院、廊坊陆军导弹学院的防空兵专业，组建'''中国人民解放军防空兵指挥学院'''，定位为中级任职教育院校，由正师级升格为副军级<ref name=fkbxy/>。
*2011年9月，更名'''中国人民解放军防空兵学院'''<ref name=fkbxy/>。
*2016年，在[[深化国防和军队改革|深化国防和军队改革]]中，由[[中国人民解放军总参谋部|中国人民解放军总参谋部]]转隶[[中国人民解放军陆军|中国人民解放军陆军]]<ref>{{Cite web |url =  http://www.81.cn/jmywyl/2016-06/21/content_7111337.htm|title =陆军防空兵跨区基地化训练拉开帷幕  |author =  |date = 2016-06-21 |publisher = 中国军网 |language =  |accessdate =  |quote =  }}</ref>。
