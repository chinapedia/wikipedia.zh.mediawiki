'''中苏拉威西省'''（[[印尼语|印尼语]]：Provinsi Sulawesi Tengah）是[[印度尼西亚|印度尼西亚]]的一个省，位于[[苏拉威西岛|苏拉威西岛]]中部，首府位于[[帕卢|帕卢]]。2010年普查，中苏省共有2,633,420人，2014年1月估计，中苏拉威西省共有人口2,839,290。

中苏拉威西省成立于1964年，总面积{{convert|61841.29|km2|sqmi|0|abbr=on}}<ref>{{cite web |url=http://dds.bps.go.id/eng/download_file/booklet_leaflet/Booklet_August_2011.pdf |title=Statistics Indonesia |last= |first= |date=2011 |website=bps.go.id |publisher= |access-date= |quote=}}</ref>，北与[[哥伦打洛省|哥伦打洛省]]为邻，南边自西往东依次与[[西苏拉威西省|西苏拉威西省]]、[[南苏拉威西省|南苏拉威西省]]和[[东南苏拉威西省|东南苏拉威西省]]接壤，东与[[马鲁古省|马鲁古省]]隔海相望，西滨[[望加锡海峡|望加锡海峡]]。

==历史==
[[File:Tokalalaea_Megalith_2007.jpg|thumb]]
[[罗瑞林都国家公园|罗瑞林都国家公园]]内有超过400块[[巨石文化|花岗岩巨石]]，其中约30块是表现人的石头。它们的大小从几厘米到大约{{convert|4.5|m|ft}}不等。最初制作石头文化的目的不得而知<ref name="National Geographic">{{cite web |url=http://news.nationalgeographic.com/news/2001/12/1212_TVindomegaliths.html |title=Explorer's Notebook: The Riddle of Indonesia's Ancient Statues|author=National Geographic |last= |first= |date=2001-12-12 |website= |publisher= |access-date=2010-10-09 |quote=}}</ref>，考古学研究认为这些巨石文化成型于公元前3000年至公元1300年<ref name="Jakarta Post">{{cite web |url=http://www.thejakartapost.com/news/2005/05/06/c-sulawesi039s-lore-lindu-park-home-biological-wealth.html |title=C. Sulawesi's Lore Lindu park, home to biological wealth |author=Sangadji, Ruslan |last= |first= |date=2005-06-05 |website= |publisher=[[雅加达邮报|The Jakarta Post]] |access-date=2010-10-11 |quote= |deadurl=yes |archiveurl=https://web.archive.org/web/20160303234549/http://www.thejakartapost.com/news/2005/05/06/c-sulawesi039s-lore-lindu-park-home-biological-wealth.html |archivedate=2016-03-03 }}</ref>。

中苏拉威西省有许多洞穴，7个洞穴中有古代岩画，根据2011年印尼和澳大利亚联合考察队的研究，岩画绘制于40,000年前（与[[西班牙|西班牙]]的{{le|蒙特·卡斯蒂略洞穴|Caves of Monte Castillo}}中的岩画成形于同一时期，这个洞穴是欧洲已知的最古老的岩画）<ref>{{cite web |url=http://surabaya.tribunnews.com/2014/10/12/gambar-di-gua-sulteng-berumur-40-ribu-tahun |title=Gambar di Gua Sulteng Berumur 40 Ribu Tahun |author=Satwika Rumeksa |date=2014-10-12}}</ref>。

伊斯兰教在17世纪传播到中苏拉威西地区。荷兰对中苏拉威西的殖民开始于18世纪，并开始了新教传播活动。现在大约四分之一的人是新教徒，是印度尼西亚群岛中最高的比例。

1999到2001年间，该地区一直受到穆斯林和基督徒之间宗教暴力的困扰，超过1,000人死亡<ref name="religious conflict">{{cite web |url=http://news.bbc.co.uk/2/hi/help/3681938.stm |title=Equator – Programme 2 – Asia |last= |first= |date= |website= |publisher=[[BBC_News|BBC News]] |access-date=2006-09-17 |quote=}}</ref>。2001年在[[南苏拉威西省|南苏省]]签署了{{le|第二马利诺协定|Malino II Accord}}，然而，在2006年9月，中苏拉维西基督教统治地区再次爆发骚乱。21世纪初期的暴力事件中，三名领导基督教武装分子的罗马天主教徒在暴动事件中遭到[[枪毙|枪毙]]<ref name="BBC">{{cite web |url=http://news.bbc.co.uk/2/hi/5368922.stm |title=Executions spark Indonesia unrest |last= |first= |date=2006-09-22 |website= |publisher=[[BBC_News|BBC News]] |access-date= |quote=}}</ref>，不过骚乱似乎是针对政府当局而非穆斯林<ref name="BBC"/>。

[[帕卢|帕卢]]是中苏拉威西省的首府以及最大的城市，其他较大城镇还有[[卢武克|卢武克]]、[[波索|波索]]、[[安帕纳|安帕纳]]、[[托利托利|托利托利]]、[[帕里吉|帕里吉]]等。

===建制沿革===

[[荷属东印度|荷属东印度]]时期，中苏拉威西地区属于万鸦老行政区（Residentie Manado），被分为两个县：[[栋加拉县|栋加拉县]]（Afdeeling Donggala）和[[波索县|波索县]]（Afdeeling Poso）。

*1950年并入印度尼西亚，属苏拉威西省；[[栋加拉县|栋加拉县]]的县治从[[栋加拉|栋加拉]]迁到[[帕卢|帕卢]]
*1952年[[栋加拉县|栋加拉县]]和[[波索县|波索县]]改制为Kabupaten<ref name=DGL1>{{cite web|url=http://donggala.go.id/sejarah/|title=Sejarah Kabupaten Donggala|website=[[栋加拉县|Situs Resmi Pemerintah Kabupaten Donggala]]|access-date=2016-12-31}}</ref>
*1959年[[邦盖县|邦盖县]]从[[波索县|波索县]]分出
*1960年[[北苏拉威西省|北-中苏拉威西省]]成立，中苏拉威西地区属之<ref>{{cite web|url=http://hukum.unsrat.ac.id/uu/uu_47_perpu_1960.htm|title=Peraturan Pemerintah Pengganti Undang-Undang Nomor 47 Tahun 1960|last=|first=|date=|website=|publisher=DPR|access-date=|quote=}}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>；[[布奥尔托利托利县|布奥尔托利托利县]]从[[栋加拉县|栋加拉县]]分出<ref name=DGL1></ref>
*1964年中苏拉威西省成立，首府位于帕卢，辖[[栋加拉县|栋加拉]]、[[波索县|波索]]、[[邦盖县|邦盖]]、[[布奥尔托利托利县|布奥尔托利托利]]四县<ref name=DPRD>{{cite web|url=http://dprd.sulutprov.go.id/hal-sejarah.html#ixzz2aHYrW4Nc|title=Sejarah Dewan Perwakilan Rakyat Daerah Provinsi Sulawesi|website=Dewan Perwakilan Rakyat Daerah Provinsi Sulawesi Utara|access-date=2017-01-21|deadurl=yes|archiveurl=https://web.archive.org/web/20140426210946/http://dprd.sulutprov.go.id/hal-sejarah.html#ixzz2aHYrW4Nc|archivedate=2014-04-26}}</ref>
*1994年[[帕卢市|帕卢市]]从[[栋加拉县|栋加拉县]]分出<ref name=":0">Pemerintah Kota Palu. (2009). '''Palu Kota Dua Wajah. '''Palu: CACDS.</ref>

==人口==
中苏省2010年人口普查的总数为2633420人，其中男性1,349,225人，女性1,284,195人，年均增长率1.94％<ref name=bps>{{cite web |url=http://sulteng.bps.go.id/index.php?option=com_content&task=view&id=126&Itemid=31 |title=Badan Pusat Statistik Provinsi Sulawesi Tengah |last= |first= |date= |website=sulteng.bps.go.id |publisher=BPS |access-date= |quote=}}</ref>。中苏省有一定的宗教问题<ref>{{cite web|url=http://awalmula.com/awal-mula-masalah-dan-kronologis-tragedi-poso.html|title=Awal mula masalah dan kronologis tragedi poso|last=|first=|date=|website=awalmula.com|publisher=|access-date=|quote=|deadurl=yes|archiveurl=https://web.archive.org/web/20130730133815/http://awalmula.com/awal-mula-masalah-dan-kronologis-tragedi-poso.html|archivedate=2013年7月30日|df=}}{{cite web |url=http://awalmula.com/awal-mula-masalah-dan-kronologis-tragedi-poso.html |title=存档副本 |accessdate=2017年12月31日 |deadurl=yes |archiveurl=https://web.archive.org/web/20130730133815/http://awalmula.com/awal-mula-masalah-dan-kronologis-tragedi-poso.html |archivedate=2013年7月30日 |df= }}</ref>，[[波索|波索]]和[[帕卢|帕卢]]等城市都曾发生宗教冲突。中苏省居民大多数是穆斯林（占76,6%）。

从2000年到2010年、1990年到2000年的年均人口增长率分别为2.57％和1.96％。
