'''中国人民解放军山东省军区'''，是[[中国人民解放军|中国人民解放军]]的一个省级[[军区|军区]]，[[正军级单位|正军级单位]]，工作範圍為[[山东省|山东省]]，2016年由[[济南军区|济南军区]]改隶[[中央军委国防动员部|中央军委国防动员部]]。

== 沿革 ==
1949年2月，中国人民解放军重组军区，[[山东军区|山东军区]]隶属[[华东军区|华东军区]]。1955年5月1日，从[[南京军区|南京军区]]划出部队，组建济南军区兼山东军区，并取消山东军区番号。1961年7月24日，恢复山东省军区。1961年10月12日，由[[济南军区|济南军区]]动员部改编，11月1日正式办公<ref>{{cite book |title = 中国共产党山东省组织史资料, 1921-1987 ; 山东省组织史资料 : 政权军事政协群团企事业单位, 1949-1987 |url = http://books.google.com/books?id=3xVPAAAAIAAJ |year = 1991 |publisher = 中共党史出版社 |page = 456}}</ref>。
