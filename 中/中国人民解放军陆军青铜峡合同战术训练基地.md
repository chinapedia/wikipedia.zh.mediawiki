'''中国人民解放军陆军青铜峡合同战术训练基地'''，位于[[宁夏回族自治区|宁夏回族自治区]][[吴忠市|吴忠市]][[青铜峡市|青铜峡市]]，是隶属[[中国人民解放军西部战区陆军|中国人民解放军西部战区陆军]]的训练基地。

==沿革==
青铜峡训练基地是20世纪末至21世纪初中国人民解放军组建的合同战术训练基地之一，起初隶属[[济南军区|济南军区]]<ref>{{Cite web |url =http://news.163.com/14/1009/13/A849MSCU00014Q4P.html  |title =  三大军事训练基地鼎立|author =  |date =  2014-10-09|publisher =网易  |language =  |accessdate =  |quote =  }}</ref>。

青铜峡训练基地自组建之后，经常用于举办跨区演习。例如2009年，由[[沈阳军区|沈阳军区]]、[[兰州军区|兰州军区]]、[[济南军区|济南军区]]、[[广州军区|广州军区]]参加的“跨越―2009”演习在沈阳军区[[中国人民解放军陆军洮南合同战术训练基地|洮南训练基地]]、兰州军区青铜峡训练基地、济南军区[[中国人民解放军陆军确山合同战术训练基地|确山训练基地]]、广州军区[[中国人民解放军陆军鹿寨合同战术训练基地|鹿寨训练基地]]展开，这是中国人民解放军自建军以来举行的第一次大规模跨区实兵检验性演习<ref>{{Cite web |url = http://news.xinhuanet.com/mil/2010-01/27/content_12883241.htm |title = 纪实：锋指青铜峡 |author =  |date = 2010-01-27 |publisher =新华网  |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://mil.cnr.cn/zgjd/200908/t20090816_505432387.html |title = 专家：我军远程投送能力需要民用运输力量 |author =  |date = 2009-08-16 |publisher = 中国广播网 |language =  |accessdate =  |quote =  }}</ref>。2015年，青铜峡训练基地举办“火力—2015·青铜峡”陆军炮兵部队跨区基地化系列演习<ref>{{Cite web |url = http://difang.gmw.cn/newspaper/2015-09/18/content_109285315.htm |title = “火力—2015·青铜峡”：1.7万兵力实弹对阵 |author =  |date = 2015-09-18 |publisher =光明网  |language =  |accessdate =  |quote =  }}</ref>。2016年又举办了“火力-2016青铜峡”陆军炮兵旅跨区基地化实兵实弹演习<ref>{{Cite web |url =http://military.china.com.cn/2016-07/27/content_38964999.htm  |title =深入“火力-2016青铜峡”演习战场 探访蓝军风采 |author =  |date = 2016-07-27 |publisher = 中国网 |language =  |accessdate =  |quote =  }}</ref>。

青铜峡训练基地还在对外交流中发挥作用。例如2015年，中巴“友谊—2015”反恐联合训练在青铜峡训练基地举行<ref>{{Cite web |url =  http://news.xinhuanet.com/politics/2015-12/08/c_128510770.htm|title = 中巴“友谊—2015”反恐联合训练在宁夏举行 |author =  |date = 2015-12-08 |publisher = 新华网 |language =  |accessdate =  |quote =  }}</ref>。

青铜峡训练基地自组建后，投资1200多万元人民币，在营区四周植树90多万株，面积2700多亩。此外还参与地方组织的植树活动，植树60多万株，面积7000多亩。基地还主动承担了治理[[黄河|黄河]]的任务。2011年10月13日，第五届“母亲河奖”颁奖仪式在[[北京|北京]]的[[人民大会堂|人民大会堂]]举行，青铜峡训练基地成为唯一获保护母亲河行动特别贡献奖的单位<ref>{{Cite web |url =http://news.cntv.cn/20111017/101033.shtml  |title = 兰州军区某合同战术训练基地 获保护母亲河行动特别贡献奖 |author =  |date = 2011-10-17 |publisher = 央视网 |language =  |accessdate =  |quote =  }}</ref>。

2016年，在[[深化国防和军队改革|深化国防和军队改革]]中，青铜峡训练基地由[[中国人民解放军兰州军区|中国人民解放军兰州军区]]转隶[[中国人民解放军西部战区陆军|中国人民解放军西部战区陆军]]<ref>{{Cite web |url = http://news.qq.com/a/20160813/007018.htm?pgv_ref=aio2015&ptlang=2052 |title = 05A数字化自行火炮在“火力青铜峡”演习中扬威 |author =  |date = 2016-08-13 |publisher =  腾讯|language =  |accessdate =  |quote =  }}</ref>。
