'''中国人民解放军总医院第六医学中心'''，位于[[北京市|北京市]][[海淀区|海淀区]][[阜成路|阜成路]]6号，隶属[[中国人民解放军海军后勤部|中国人民解放军海军后勤部]]，[[三级甲等医院|三级甲等医院]]。

==简介==
中国人民解放军总医院第六医学中心的前身是'''中国人民解放军海军北京医院'''，于1951年经[[中央人民政府政务院|中央人民政府政务院]]总理[[周恩来|周恩来]]批准在北京筹建，1954年2月开始收治伤病员，1954年7月扩建并更名为'''中国人民解放军海军总医院'''，首任院长为萧邦宁。

该医院为1991年至1993年全军“白求恩杯”优胜单位<ref name=wkzx>[http://news.163.com/11/1108/19/7IC4EQR600014AED.html 北京外科整形医院排名情况]，网易，2011-11-08</ref>。1994年，首批获总部评为军队[[三级甲等医院|三级甲等医院]]。该医院是[[中国人民解放军第二军医大学|中国人民解放军第二军医大学]]、[[南方医科大学|南方医科大学]]等多所高校的临床教学医院。该医院7次被[[中国人民解放军总后勤部|中国人民解放军总后勤部]]评为“全军为部队服务先进医院”<ref name=jj>[http://www.hjzyy.com.cn/Hospitals/Main/Description 医院简介]，中国人民解放军海军总医院，于2016-11-20查阅</ref>。

中国人民解放军海军总医院先后完成了[[海河|海河]]、[[淮河|淮河]]、[[黄河|黄河]]抗洪救灾，[[邢台地震|邢台]]、[[唐山大地震|唐山]]、[[汶川大地震|汶川]]抗震救灾，抗击[[非典|非典]]、抵御[[禽流感|禽流感]]，赴[[柬埔寨|柬埔寨]]、[[菲律宾|菲律宾]]开展人道主义救援，及护航行动、联合军演、“和谐使命”系列医疗服务等任务。该医院先后出现了“党的好女儿”[[冯理达|冯理达]]，全国道德模范及[[南丁格尔奖|南丁格尔奖]]获得者王文珍、全国抗震救灾英雄集体等先进模范。<ref name=jj/>

2000年，中国第一个脑外科手术的医用机器人研制成功，在海军总医院成功地为患者施行手术<ref>乔天富、吉尔：《机器人施行脑手术——海军总医院为72位患者手术成功》，《科学新闻》2000(28)</ref>。2001年8月9日：中国人民解放军海军总医院在中国首次成功利用机器人进行远程脑外科手术<ref>[http://www.china.com.cn/aboutchina/txt/2009-08/08/content_18299894.htm 2001年8月9日 机器人完成远程脑外科手术]，中国网，2009-08-08</ref>。2012年11月23日，海军总医院神经外科医院正式挂牌成立<ref name=chengli>[http://jiankang.cntv.cn/20121123/100250.shtml 海军总医院神经外科医院正式挂牌成立]，中国网络电视台，2012-11-23</ref>。

中国人民解放军海军总医院医学美容整形中心成立于1990年，是北京市较早开展整形美容的医疗机构<ref name=wkzx/>。

2012年，海军总医院启动实施为部队服务“百千万”医疗行动。2012年，海军总医院与[[清华大学|清华大学]]签订海洋医学战略合作协议，共同建立“海洋医学与救援联合研究中心”<ref>[http://www.hinews.cn/news/system/2014/09/29/016999446.shtml 海军总医院潜心铸造能打胜仗的“医疗航母”]，南海网，2014-09-29</ref>。

2018年，[[中国人民解放军陆军总医院|原陆军总医院]]、原海军总医院、[[中国人民解放军第三〇二医院|原第三〇二医院]]、[[中国人民解放军第三〇九医院|原第三〇九医院]]、[[中国人民解放军第三〇七医院|军事科学院军事医学研究院原附属医院]]、[[中国人民武装警察部队总医院|原武警总医院]]均已由[[中国人民解放军总医院|中国人民解放军总医院]]暂管<ref>{{Cite web |url = https://www.thepaper.cn/newsDetail_forward_2334813 |title = 官方披露：原陆军总医院等单位已由解放军总医院暂管 |author =  |date =2018-08-10  |publisher =澎湃新闻  |language =  |accessdate =  |quote =  }}</ref>。现已更名为'''中国人民解放军总医院第六医学中心'''<ref>{{Cite web |url = http://www.sohu.com/a/273558387_267160 |title = 军队医院更名继续，解放军总医院八个医学中心亮相 |author =  |date =2018-11-06  |publisher =搜狐  |language =  |accessdate =  |quote =  }}</ref>。
