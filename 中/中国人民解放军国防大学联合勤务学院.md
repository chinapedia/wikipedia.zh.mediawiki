'''中国人民解放军国防大学联合勤务学院'''，位于[[北京市|北京市]]，隶属[[中国人民解放军国防大学|中国人民解放军国防大学]]。

== 沿革 ==
*1951年，创建'''中国人民解放军高级后勤学校'''。
*1951年，[[中国人民解放军第七军|中国人民解放军第七军]]机关改建为'''中国人民解放军第一高级步兵学校'''，隶属[[西北军区|西北军区]]。
*1952年，[[中国人民解放军总后勤部|中央人民政府人民革命军事委员会总后勤部]]部长[[杨立三|杨立三]]提出“创办高层次、正规化后勤院校，培养高层次、专业化后勤指挥人才”的办学思路，该思路很快获得中央领导及军委首长认同<ref name=gmw/>。
*1952年5月16日，[[中央人民政府人民革命军事委员会|中央人民政府人民革命军事委员会]]主席[[毛泽东|毛泽东]]签署命令，正式批准创建“'''中国人民解放军后方勤务学院'''”，直属中央军委领导。该学院由原高级后勤学校与西北军区第一高级步兵学校迁[[北京|北京]]的部分合并组成，与当时的[[中国人民解放军军事学院|中国人民解放军军事学院]]、[[中国人民解放军政治学院|中国人民解放军政治学院]]并称为“三大学院”<ref name=fenjin/><ref name=ma> {{Cite book | author = 马国泉 等主编 | title =新时期新名词大辞典  | location =  | publisher = 中国广播电视出版社 | date =1992年  | pages =  | ISBN =  | accessdate =  | url =  | language =  | quote = 后勤学院 }} </ref><ref name=gmw/>。
*1953年2月1日，后方勤务学院在北京举行成立暨第一期开学典礼<ref name=gmw/><ref name=fenjin/>，典礼上宣读了训词，训词指出了后勤工作的重要地位，并首次提出了后勤是专门的科学。后方勤务学院第一批教官有解放军干部、进步青年、国民党起义军官、[[苏联|苏联]]专家。学院按照“两共一专、突出专业”“勤务为主，技术为辅”“以师为重点”这三项原则，开展本科及专科教育，举办短训班及高级速成班。学院科研工作按照“以教学为中心做好各项工作”的要求，确定将教材建设作为学院创建时期的中心任务，6个月编出91本教材。1954年8月，学院抽调人员协同科研部编写《[[朝鲜战争|朝鲜战争]]后勤工作经验汇编》<ref name=fenjin/>。
*1953年1月31日，毛泽东主席亲自给后方勤务学院颁发《中央人民政府人民革命军事委员会训词》：“对于现代的军队，组织良好的后方勤务工作具有极其重大的意义。任何轻视后勤工作，以为后勤工作不是重要的专门科学，不需要有系统的学习、不需要精通业务的观点是完全错误的。我们必须学习[[苏联|苏联]]军队完整的后勤工作建设，研究[[朝鲜战争|朝鲜战争]]中后勤工作的状况和经验，以达到我军后勤工作现代化和正规化的目的。”<ref name=gmw/>
*1955年，学院第一期学员毕业，[[彭德怀|彭德怀]]、[[贺龙|贺龙]]、[[聂荣臻|聂荣臻]]等领导亲自到学院祝贺，参加毕业典礼。1950年代至1960年代，毛泽东、[[刘少奇|刘少奇]]、[[周恩来|周恩来]]、[[朱德|朱德]]、[[邓小平|邓小平]]等[[党和国家领导人|党和国家领导人]]多次接见学院毕业学员及先进工作者<ref name=gmw/>。
*1956年，中国人民解放军后方勤务学院油料中级班分出，参与组建中国人民解放军后勤学校（后为[[中国人民解放军军事经济学院|中国人民解放军军事经济学院]]）。
*1960年，中国人民解放军后方勤务学院更名为'''中国人民解放军后勤学院'''。
*1961年，中国人民解放军后勤学院营房系、油料系分出，参与组建[[中国人民解放军后勤工程学院|中国人民解放军后勤工程学院]]。
*1962年，[[中国人民解放军炮兵工程学院|中国人民解放军炮兵工程学院]]军械勤务系并入。
*1969年9月，学院撤销<ref name=fenjin/><ref name=gmw/>。参与组建[[中国人民解放军军事政治大学|中国人民解放军军事政治大学]]。
<br>
*1977年，邓小平在军委座谈会上将军队教育训练提高到重要战略地位，亲自批准恢复因“[[文革|文革]]”而停办的军事学院、政治学院、后勤学院三大学院，并要求“把学校办好，让多一点人进学校”<ref name=fenjin/><ref name=gmw/>。
*1977年11月，中央军委决定恢复'''中国人民解放军后勤学院'''，直属中央军委领导，行使大军区级权限<ref name=gmw>{{Cite web |url =http://www.gmw.cn/01gmrb/2009-11/17/content_1009082.htm  |title =站在新的历史起点上  |author =  |date = 2009-11-17 |publisher = 光明网 |language =  |accessdate =  |quote =  }}</ref>。该学院以原中国人民解放军军事政治大学后勤系为基础组建。
*1985年3月，中央军委常务会议研究决定，在军事学院、政治学院、后勤学院（部分人员）合并的基础上组建[[中国人民解放军国防大学|中国人民解放军国防大学]]<ref name=zhanglei/>。1985年10月，中央军委正式批准合并军事学院、政治学院、后勤学院，组建国防大学。同时，确定另组建一所中级后勤指挥学院，仍定名为后勤学院，隶属总后勤部，正兵团级<ref name=fenjin/><ref name=gmw/>。1985年12月撤销后勤学院建制。
<br>
*1986年，建立新的'''中国人民解放军后勤学院'''，隶属[[中国人民解放军总后勤部|中国人民解放军总后勤部]]<ref name=gmw/>。
*1987年12月，后勤学院召开党代表大会，指出办校思想的核心是培养“全面合格”的“指挥勤务人才”<ref name=fenjin/>。
*1991年，[[中央军委主席|中央军委主席]][[江泽民|江泽民]]题词：“办好后勤学院，为建设革命化、现代化、正规化军队努力奋斗！”
*1992年，全军院校编制体制调整，中国人民解放军后勤学院撤销<ref name=gmw/>。
*1993年初，中央军委决定恢复该学院，并更名为'''中国人民解放军后勤指挥学院'''，隶属总后勤部，行使正军级权限<ref name=fenjin/><ref name=gmw/>。
*1993年6月，复建并更名为中国人民解放军后勤指挥学院。
*1993年6月，后勤指挥学院党委进一步明确了“三位一体，合力办学”的方向<ref name=fenjin/>。
*1993年7月20日，中央军委主席江泽民为后勤指挥学院题写院名<ref name=gmw/>。
*1994年，总后勤部确定后勤指挥学院为全军后勤学术研究中心，承担全军后勤学术研究任务，同时明确后勤指挥学院是总后勤部从理论高度指导后勤工作与建设的助手<ref name=fenjin/>。
*1995年，后勤指挥学院被确定为“三重”（即根据中央军委决策，在“[[九五计划|九五]]”期间，总部在全军研究生培养单位重点建设一批院校、学科和实验室，进入21世纪后纳入“2110”工程）的重点建设院校，军事后勤学被定为重点建设学科，战术后勤实验室被定为重点建设实验室<ref name=fenjin/>。
*1997年5月6日，中央军委主席江泽民为后勤指挥学院题词：“办好后勤院校，培养合格人才。”<ref name=gmw/>
*2006年，后勤指挥学院被列入军队“2110工程”整体条件重点建设院校<ref name=fenjin/>。
*2011年6月，更名'''中国人民解放军后勤学院'''<ref name=fenjin/>。
*2016年，在[[深化国防和军队改革|深化国防和军队改革]]中，由[[中国人民解放军总后勤部|中国人民解放军总后勤部]]转隶[[中央军事委员会训练管理部|中央军事委员会训练管理部]]<ref>{{cite web|title=全军后勤最高学府后勤学院已由原总后转隶中央军委训练管理部|url=http://www.thepaper.cn/newsDetail_forward_1473027|publisher=澎湃新闻|accessdate=2016-05-24}}</ref>。
<br>
*2017年，在[[深化国防和军队改革|深化国防和军队改革]]中，后勤学院转隶[[中国人民解放军国防大学|中国人民解放军国防大学]]，组建'''中国人民解放军国防大学联合勤务学院'''<ref name=zhanglei>{{Cite web |url = http://zqb.cyol.com/html/2017-07/20/nw.D110000zgqnb_20170720_1-11.htm |title =国防大学是一所什么样的大学  |author =张磊  |date =  |publisher = 中国青年报2017年07月20日，第11版 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url =  http://military.people.com.cn/n1/2017/0801/c1011-29440220.html|title =永不褪色的军装  |author = 张磊 |date =  2017-08-01|publisher = 人民网 |language =  |accessdate =  |quote =  }}</ref>。

==中国人民解放军军事后勤陈列馆==
