'''中村雀右卫门'''{{BD|1920年||2012年|2月23日|catIdx= Z中}}本命'''青木清治'''，出生于[[大日本帝国|大日本帝国]][[东京都|东京都]]，[[日本|日本]][[歌舞伎|歌舞伎]]艺术大师，被誉为日本歌舞伎“[[人间国宝|人间国宝]]”{{noteTag|1= [[歌舞伎|歌舞伎]]中的“[[女形|女形]]”相当于[[中国戏曲|中国戏曲]][[行當|行當]]中的“[[旦|旦]]”；“[[女形|女形]]”演员被认定为日本“[[人间国宝|人间国宝]]”的共有4位：第七代尾上梅幸、第六代中村歌右卫门、第七代中村芝翫和第四代中村雀右卫门。[http://culture.ifeng.com/whrd/detail_2012_03/06/12992958_0.shtml]}} 。<ref name="ops " >{{cite web |url=http://culture.ifeng.com/whrd/detail_2012_03/06/12992958_0.shtml |title=日本歌舞伎“人间国宝”辞世 养父曾教授梅兰芳 |accessdate= 2012年|work= |publisher=凤凰网 |language =zh-cn}}</ref>

== 生平 ==
中村雀右卫门出生于一个艺术世家，他的父亲[[青木八重太郎|青木八重太郎]]是歌舞伎艺术大师，他的养父[[中岛笑太郎|中岛笑太郎]]也是歌舞伎艺术大师，是第三代中村雀右卫门，其养父还是[[中国|中国]][[京剧|京剧]]大师[[梅兰芳|梅兰芳]]的授技恩师。<ref name="ops " />

27岁时，他才开始出场演出，并常常吃不饱饭，为了生计，他曾改行做[[电影演员|电影演员]]。<ref name="ops " />

30岁时，他出演电影《[[佐佐木小次郎|佐佐木小次郎]]》一举走红，为了自己热爱的歌舞伎艺术，5年后，他再度重操旧业。<ref name="ops " />

35岁，他开始和[[市川寿海|市川寿海]]一起出演，声名日浓。<ref name="ops " />

1964年，他传袭了养父的称号，成为第四代中村雀右卫门。<ref name="ops " />

1999年，他开始担任传统歌舞伎保护协会会长，专门负责保护、继承和发扬日本传统文化。<ref name="ops " /><ref name="txs " />

2012年2月23日下午3点55分，他因[[肺炎|肺炎]]在[[东京都|东京都]][[医院|医院]]去世，享年91岁。<ref name="txs " >{{cite web |url=http://ent.qq.com/a/20120223/001322.htm |title=日本国宝级大师中村雀右卫门因病去世 享年91岁 |accessdate= 2012年|work= |publisher=腾讯网 |language =zh-cn}}</ref><ref name="xlws " >{{cite web |url=http://ent.sina.com.cn/j/2012-02-23/21213563123.shtml |title=日本舞伎大师中村雀右卫门去世 享年91岁 |accessdate= 2012年|work= |publisher=新浪网 |language =zh-cn}}</ref>

== 思想 ==
中村雀右卫门认为艺术的精髓来自于心灵之美，他说：“想要表现出艺术的美仅仅靠演技是远远不够的，虽然生活中我们也会憎恨自己讨厌的人，嫉妒比自己优秀的人，但是请保持一颗对他人的仁爱之心、宽容之心，去努力做到真正地心存美好。舞台上，再卓越的演技也无法掩饰真正的内心。”<ref name="ops " />

== 作品 ==
中村雀右卫门经常出演日本历史上三大名姬：[[时姬|时姬]]、[[八重恒姬|八重恒姬]]、[[雪姬|雪姬]]，年届八旬依然不辍；其经典之作是《[[京鹿子娘道成寺|京鹿子娘道成寺]]》中的“白拍子花子”，被誉为歌舞伎女形集大成之作。<ref name="ops " /><ref name="txs " /><ref name="xlws " />

== 评价 ==
舞台美术家[[朝仓摄|朝仓摄]]：“让我记忆犹新的是他和中村富十郎先生一起表演的舞蹈《二人椀久》。中村先生扮演的旦角不管是从外形上还是从舞蹈上都洋溢着美的气息。这种美是无法学习和复制的，是极其自然的发自内心的美。”<ref name="ops " />
