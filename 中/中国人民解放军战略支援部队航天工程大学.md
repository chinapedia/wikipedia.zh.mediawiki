'''中国人民解放军战略支援部队航天工程大学'''，简称'''航天工程大学'''，位于[[北京市|北京市]]，隶属[[中国人民解放军战略支援部队|中国人民解放军战略支援部队]][[中国人民解放军战略支援部队航天系统部|航天系统部]]领导管理，是培养航天指挥管理与工程技术人才的综合性大学<ref name=xunli>{{Cite web |url = http://www.mod.gov.cn/services/2017-06/15/content_4783023.htm |title = 2017军校巡礼第二十五站：航天工程大学（附报考指南） |author =  |date = 2017-06-15 |publisher = 中华人民共和国国防部 |language =  |accessdate =  |quote =  }}</ref>。

== 沿革 ==
*1978年6月，创建'''国防科委干部学校'''，隶属[[中国人民解放军国防科学技术委员会|国防科委]]<ref name=nfw>{{Cite web|url =http://gz.southcn.com/content/2016-01/11/content_140527336.htm?from=singlemessage&isappinstalled=0|title =中央军委装备发展部成立 总装备部成为历史|author =|date =2016-01-11|publisher =南方网|language =|accessdate =|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20170901074144/http://gz.southcn.com/content/2016-01/11/content_140527336.htm?from=singlemessage&isappinstalled=0|archivedate =2017年9月1日|df =}}</ref>。
*1982年，更名'''国防科工委干部学校'''，隶属[[中华人民共和国国防科学技术工业委员会|国防科工委]]。
*1986年，升格'''国防科工委指挥技术学院'''<ref name=nfw/>。
*1999年，更名'''中国人民解放军装备指挥技术学院'''，隶属[[中国人民解放军总装备部|中国人民解放军总装备部]]<ref name=nfw/>。
*2011年，更名'''中国人民解放军装备学院'''。是隶属中国人民解放军总装备部的正军级任职教育院校<ref>{{Cite web |url = http://www.thepaper.cn/newsDetail_forward_1709933 |title = 军媒披露：航天工程大学以原装备学院为基础重建 |author =  |date = 2017-06-15 |publisher = 澎湃新闻 |language =  |accessdate =  |quote =  }}</ref>。
*2016年，在[[深化国防和军队改革|深化国防和军队改革]]中，由[[中国人民解放军总装备部|中国人民解放军总装备部]]转隶[[中央军事委员会训练管理部|中央军事委员会训练管理部]]<ref>{{cite web|title=原总参、总后、总装所辖装备学院等多所军校转隶军委训管部|url=http://www.thepaper.cn/newsDetail_forward_1483043|publisher=澎湃新闻|date=2016-06-13}}</ref>。
*2017年，在[[深化国防和军队改革|深化国防和军队改革]]中，经中央军委批准，以原装备学院为基础重建为'''中国人民解放军战略支援部队航天工程大学'''，隶属[[中国人民解放军战略支援部队航天系统部|中国人民解放军战略支援部队航天系统部]]领导管理<ref name=xunli/><ref>{{Cite web |url =http://www.thepaper.cn/newsDetail_forward_1721167  |title =上海第二军医大学被命名为“中国人民解放军海军军医大学”  |author =  |date = 2017-06-29 |publisher = 澎湃新闻 |language =  |accessdate =  |quote =  }}</ref>。

==专业设置==
2017年组建后，中国人民解放军战略支援部队航天工程大学设有20多个本科专业方向，拥有12个一级学科硕士点，5个一级学科博士点，3个博士后科研流动站。2017年招生的本科专业有<ref name=xunli/>：
*指挥信息系统工程
*武器发射工程
*测控工程
*航天装备工程
*光电信息科学与工程
*侦察情报
*遥感科学与技术
*通信工程
*信息对抗技术<ref name=xunli/>

==校区==
2017年组建后，中国人民解放军战略支援部队航天工程大学设有怀柔、昌平、沙河三个校区，均在[[北京市|北京市]]。学校设有研究生院、航天指挥学院、宇航技术学院、航天信息学院、航天勤务学院、航天士官学校、基础教学部<ref name=xunli/>。
