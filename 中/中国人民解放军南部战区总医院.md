'''中国人民解放军南部战区总医院'''，位於[[广东省|广东省]][[广州市|广州市]][[越秀区|越秀区]][[流花路|流花路]]111号，隶属[[桂林联勤保障中心|桂林联勤保障中心]]，为[[三级甲等医院|三级甲等医院]]<ref name="醫院簡介">{{Cite web|url=http://www.gzzyy.com/Index/index.php/Show/index/no/1251697047|title=医院简介|accessdate=2012-07-09|work=广州军区广州总医院|publisher=|date=|deadurl=yes|archiveurl=https://web.archive.org/web/20120719192847/http://www.gzzyy.com/Index/index.php/Show/index/no/1251697047|archivedate=2012-07-19}}</ref>。

== 简介 ==
1932年，[[陈济棠|陈济棠]]提出了建设“模范新广东”的计划。1933年，兴建'''广东总医院'''，即现今华侨楼。1936年，更名为'''广东陆军总医院'''。1937年，医院增建国防军医学校第一分校。1938年，[[广州沦陷|广州沦陷]]，医院及学校迁走。医校迁往[[广西|广西]][[桂林|桂林]]，医院则先后遷址[[廣西|廣西]]、[[廣東|廣東]]、[[湖南|湖南]]。1944年，医院被迫迁往[[江西|江西]]，並与国民政府军医署失去联系。1945年，医院就地解散<ref name = "悠久歷史"/>。 

1945年，[[抗日战争|抗日战争]]后，在医院原址上重组国民政府军政部'''广州陆军总医院'''。1947年，医院改称国军联勤总部'''广州总医院'''。1949年，更名为国民政府国防部军医署'''陆海空第二总医院'''<ref name = "悠久歷史"/>。

1949年10月14日广州解放，[[中国人民解放军|中国人民解放军]]接管医院。此時醫院設備人員匱乏。1978年8月，医院升为[[正师级|正师级]]。1985年，撤销[[武汉军区|武汉军区]]，医院改为'''中国人民解放军广州军区第二总医院'''。1986年，更名'''中国人民解放军广州军区广州总医院'''。医院是军队首批[[三级甲等医院|三级甲等医院]]之一<ref name="悠久歷史">{{Cite web|url=http://www.gzzyy.com/Index/index.php/Show/index/no/1251697381|title=悠久历史|accessdate=2012-07-09|work=广州军区广州总医院|publisher=|date=|deadurl=yes|archiveurl=https://web.archive.org/web/20120715131500/http://www.gzzyy.com/Index/index.php/Show/index/no/1251697381|archivedate=2012-07-15}}</ref>。

2016年，在[[深化国防和军队改革|深化国防和军队改革]]中，由[[中国人民解放军广州军区|中国人民解放军广州军区]]转隶[[桂林联勤保障中心|桂林联勤保障中心]]，更名'''中国人民解放军广州总医院'''<ref>{{Cite web |url =http://www.81.cn/jfjbmap/content/2017-02/22/content_170081.htm  |title =  一身“兵味” 站好“军姿”|author =  |date =2017-02-22  |publisher = 中国军网 |language =  |accessdate =  |quote =  }}</ref>。

2018年，更名为'''中国人民解放军南部战区总医院'''<ref>{{Cite web |url = https://news.sina.cn/2018-11-23/detail-ihmutuec3034133.d.html?vt=4 |title = 解放军五大战区总医院全部成立 |author =  |date =2018-11-23  |publisher =澎湃新闻  |language =  |accessdate =  |quote =  }}</ref>。

2016年改革前，医院为9所高等医学院校的教学医院，并且设有国家重点专病中心1个、国家药物临床试验机构科室9个、全军医学专科（病）中心9个、军区医学研究所和专科中心12个、[[博士后|博士后]]科研工作站2个<ref name = "醫院簡介"/><ref name = "悠久歷史"/><ref name="人才濟濟">{{Cite web|url=http://www.gzzyy.com/Index/index.php/Show/index/no/1251697176|title=人才济济|accessdate=2012-07-09|work=广州军区广州总医院|publisher=|date=|deadurl=yes|archiveurl=https://web.archive.org/web/20120619114244/http://www.gzzyy.com/Index/index.php/Show/index/no/1251697176|archivedate=2012-06-19}}</ref><ref>{{Cite web|url=http://www.gzzyy.com/Index/index.php/Show/index/no/1251697292|title=成就斐然|accessdate=2012-07-09|work=广州军区广州总医院|publisher=|date=|deadurl=yes|archiveurl=https://web.archive.org/web/20120619114304/http://www.gzzyy.com/Index/index.php/Show/index/no/1251697292|archivedate=2012-06-19}}</ref>。

==广州总医院一五七分院==
[[File:157JiJyun.jpg|缩略图]]
1937年8月，成立[[红四方面军|红四方面军]]第十一军卫生所。1942年改编为[[晋察鲁豫军区|晋察鲁豫军区]]白求恩医院一所。1948年5月改编为[[中原军区|中原军区]]白求恩总医院第二分院<ref name=gdywq/>。后几经改编，先后为第二野战军第二野战医院、第二野战军卫生部总医院二院五所。1950年1月至1951年9月，作为军医院驻[[四川省|四川省]][[重庆市|重庆市]]新桥，曾编制为第二野战军军政大学附属医院、西南军区军政大学附属医院一分院。1951年5月，合组为西南军区第二后方医院。1951年9月，驻[[湖北省|湖北省]][[孝感|孝感]]，更名中南军区临时第二后方医院。1952年6月，更名陆军第六十六医院。1953年4月，驻[[河南省|河南省]][[漯河|漯河]]<ref name=yiyuanjianjie>{{Cite web|url=http://www.gzzyy157.com/about.asp?tid=61|title=医院简介|accessdate=2012-07-09|work=广州军区广州总医院附属一五七医院|publisher=|date=|deadurl=yes|archiveurl=https://web.archive.org/web/20120808010243/http://www.gzzyy157.com/about.asp?tid=61|archivedate=2012-08-08}}</ref><ref name=youjiulsh>{{Cite web|url=http://www.gzzyy157.com/about.asp?tid=64|title=悠久历史|accessdate=2012-07-09|work=广州军区广州总医院附属一五七医院|publisher=|date=|deadurl=yes|archiveurl=https://web.archive.org/web/20151223190341/http://www.gzzyy157.com/about.asp?tid=64|archivedate=2015-12-23}}</ref>。1954年6月，改编为陆军第一五七医院<ref name=gdywq/>。1955年7月，迁至[[湖南省|湖南省]][[耒阳县|耒阳县]]<ref name = yiyuanjianjie/><ref name=youjiulsh/>。1958年1月，由[[湖南省|湖南省]][[耒阳县|耒阳县]]移驻广东省广州市郊白灰场。1987年，全院有副主任军医以上人员48人，设22个临床和医技科室。自1960年代初起，医院开展中西医结合治疗研究，1971年被[[周恩来|周恩来]]誉为“中西医结合的一面红旗”。医院设全军中西医结合内科专科中心、广州军区肛肠外科专科中心<ref name=gdywq> {{Cite book | author =广东省地方史志编纂委员会 编  | title = 广东省志·军事志 | location =  | publisher = 广东人民出版社 | date = 1999年 | pages =413  | ISBN =  | accessdate =  | url =  | language =  | quote =  }} </ref>。1996年，被评为[[三级甲等医院|三级甲等医院]]。2009年8月，移交广州军区广州总医院，更名为'''中国人民解放军广州军区广州总医院附属一五七医院'''<ref name = yiyuanjianjie/><ref name=youjiulsh/>。后更名为'''中国人民解放军广州军区广州总医院一五七分院'''<ref>{{Cite web|url = http://www.gdifi.org.cn/d9446802a4425/208.jhtml|title = 一五七医院党政干部赴省食检所参加 食品安全风险交流活动|author = |date = 2017-01-03|publisher = 广东省食品检验所（广东省酒类检测中心）|language = |accessdate = |quote = }}{{Dead link|date=2019年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。2017年更名'''中国人民解放军广州总医院一五七分院'''。
