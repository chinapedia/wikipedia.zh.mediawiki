'''中国人民解放军空军勤务学院'''，简称'''空军勤务学院'''，原'''徐州空军学院'''，是一所培养[[中国人民解放军空军|中国人民解放军空军]][[后勤|后勤]]指挥管理人才的军事高等院校，[[副军级|副军级]]<ref>[http://www.xzqwxy.com/Article/ShowArticle.asp?ArticleID=3 中国人民解放军徐州空军学院简介]</ref>。

== 沿革==
* 1954年10月，于[[太原市|太原市]]建立'''中国人民解放军空军后方勤务学校'''。
* 1958年5月，迁至[[徐州市|徐州市]]，更名'''中国人民解放军空军后勤学校'''。
* 1969年，撤销空军后勤学校。
* 1977年，复建空军后勤学校。
* 1986年6月，更名'''中国人民解放军空军勤务学院'''。
* 1992年，更名'''中国人民解放军后勤管理技术学院'''。
* 1993年6月，回归空军建制，更名'''中国人民解放军空军后勤学院'''。
* 2004年，更名'''中国人民解放军徐州空军学院'''。
* 2012年4月，更名'''中国人民解放军空军勤务学院'''。
*2016年至2017年，在[[深化国防和军队改革|深化国防和军队改革]]中，空军勤务学院保留<ref>{{Cite web |url = http://news.ifeng.com/a/20170530/51183107_0.shtml |title =  2017年军队招生单位名录发布 军改后院校新名称曝光|author =  |date = 2017-05-30 |publisher =凤凰网  |language =  |accessdate =  |quote =  }}</ref>。
