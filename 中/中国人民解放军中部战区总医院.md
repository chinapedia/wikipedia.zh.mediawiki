'''中国人民解放军中部战区总医院'''，位于[[湖北省|湖北省]][[武汉市|武汉市]][[武珞路|武珞路]]627号，隶属[[武汉联勤保障基地|武汉联勤保障基地]]，是一家集医疗、教学、科研于一体的综合医疗机构，[[湖北省|湖北省]]首批三级甲等医院之一。

==沿革==
*1946年6月，在[[黑龙江省|黑龙江省]][[勃利县|勃利县]]成立“东北民主联军第八后方医院”，隶属[[东北民主联军|东北民主联军]]。该医院先后参加了“[[三下江南|三下江南]]、[[四保临江|四保临江]]”和[[辽沈战役|辽沈战役]]、[[平津战役|平津战役]]<ref name=yaoxue>中国人民解放军广州军区武汉总医院，药学服务与研究2008年第5期</ref><ref name=whjsz/>。
*1948年，更名中国人民解放军第二十五后方医院。
*1949年6月下旬，医院随[[中国人民解放军第四野战军|中国人民解放军第四野战军]]南下至[[武汉|武汉]]<ref name=whjsz/><ref name=yaoxue/>。进驻汉口中滑坡原国军武汉陆军总医院旧址。
*1950年3月，以该院为基础组建中国人民解放军中南军区陆军医院<ref name=yaoxue/>。后称中国人民解放军第一五一医院。
*1955年，[[武汉军区|武汉军区]]成立，医院划归武汉军区管辖。
*1956年2月，更名中国人民解放军武汉军区总医院<ref name=whjsz>{{Cite web |url =  http://www.whfz.gov.cn:81/was40/detail?record=1&primarykeyvalue=ID%3D21847&channelid=47777|title =第二节　驻汉部队医院  |author =  |date =  |publisher = 武汉市情网 |language =  |accessdate =2017-06-15 |quote =  }}</ref><ref name=xuanjie/>。
*1962年，医院由汉口中滑坡迁至[[武昌|武昌]]武珞路现址。
*1985年9月，随着武汉军区撤销，[[湖北省军区|湖北省军区]]和驻湖北省的部队划归[[广州军区|广州军区]]，医院更名为中国人民解放军广州军区第一总医院<ref name=whjsz/><ref name=yaoxue/>。
*1987年1月，正式定名为'''中国人民解放军广州军区武汉总医院'''，师级建制<ref name=whjsz/>。
*1995年5月，被[[中国人民解放军总后勤部|中国人民解放军总后勤部]]评为[[三级甲等医院|三级甲等医院]]。1996年8月，被[[中华人民共和国卫生部|中华人民共和国卫生部]]、[[世界卫生组织|世界卫生组织]]、[[联合国儿童基金会|联合国儿童基金会]]评为爱婴医院。2000年，被[[中国人民解放军总政治部|中国人民解放军总政治部]]评为“全军为部队服务先进医院”。2001年，成为湖北省（武汉市）城镇职工基本医疗保险首批定点医院。2004年，被武汉市评选成为文明品牌示范医院和诚信医院。2005年，中国人民解放军总政治部、中国人民解放军总后勤部先后转发武汉总医院抓医德医风建设的做法和经验<ref name=whjsz/>。
*2016年，在[[深化国防和军队改革|深化国防和军队改革]]中，由[[中国人民解放军广州军区|中国人民解放军广州军区]]转隶[[武汉联勤保障基地|武汉联勤保障基地]]下属的[[郑州联勤保障中心|郑州联勤保障中心]]，更名为中国人民解放军武汉总医院<ref>{{Cite web |url =http://news.ifeng.com/a/20170513/51087420_0.shtml |title =福利打包！军人军属最新医疗待遇全在这里  |author =  |date =2017-05-13  |publisher =凤凰网  |language =  |accessdate =  |quote =  }}</ref>。
*2018年，更名为'''中国人民解放军中部战区总医院'''<ref>{{Cite web |url = https://news.sina.cn/2018-11-23/detail-ihmutuec3034133.d.html?vt=4 |title = 解放军五大战区总医院全部成立 |author =  |date =2018-11-23  |publisher =澎湃新闻  |language =  |accessdate =  |quote =  }}</ref>。
