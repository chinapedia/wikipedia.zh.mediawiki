'''中國交通建設股份有限公司'''（{{hkex|1800}}、{{sse|601800}}），簡稱'''中交'''、'''中交建'''或'''中交建設'''，在2005年成立，由[[中港集團|中港集團]]及[[路橋集團|路橋集團]]合併而成。集團主要從事交通基建建設、基建設計、[[疏浚|疏浚]]及[[港口|港口]][[機械|機械]]製造業務，是目前中國最大的港口建設及設計企業；中國領先的公路、橋樑建設及設計企業；中國最大、世界第三大的疏浚企業；上市子公司分別是[[上海振華港口機械|上海振華港口機械]]（ZPMC）和[[路橋建設|路橋建設]]，其中上海振華是全球最大的[[集裝箱|集裝箱]][[起重機|起重機]][[製造商|製造商]]。董事長是[[刘起涛|刘起涛]]，總裁是[[陈奋健|陈奋健]]。<ref>[http://www.aastocks.com/chi/fundamental/profile.asp?symbol=1800 中國交通建設公司資料] {{webarchive|url=https://web.archive.org/web/20070607142031/http://www.aastocks.com/chi/fundamental/profile.asp?symbol=1800 |date=2007-06-07 }}</ref>2010年8月5日[[中国房地产开发集团公司|中国房地产开发集团公司]]整体并入中国交通建设集团有限公司(即中交建母公司)，成为其全资子企业<ref>[http://www.sasac.gov.cn/n1180/n1566/n258252/n258629/7455965.html]</ref>，但現時未有注入上市公司的計劃。

==業務併購==
2014年12月11日中國交通建設全資附屬公司中交國際（香港）控股有限公司與澳洲[[禮頓控股|禮頓控股]]公司簽署了股權買賣協議，斥資9.531億澳元（約61.13億港元）收購禮頓控股旗下澳洲工程承包服務商John Holland[[喬恩荷蘭集團|喬恩荷蘭集團]]全部權益

2015年3月30日中國交通建設重組旗下的疏浚業務，並設立中交疏浚，以推進中交疏浚設立後將中交疏浚分拆上市。：中交疏浚設立後新發行117.1億股普通股用於收購公司持有的中交天津航道局有限公司100%股權、中交上海航道局有限公司100%股權和中交廣州航道局有限公司100%股權。該等股權以上述三家航道局最近一期經審計的淨資產金額為基準確定對價為234.3億元。後續中交疏浚將以現金收購公司下屬的中交國際航運有限公司和香港海事建設有限公司，收購價格以資產評估結果為基準確定。收購完成後，中交國際航運有限公司和香港海事建設有限公司將成為中交疏浚全資或控股子公司。

2017年10月26日，中國交通建設宣佈，中交國際、SPV公司及Aecon就建議收購事項訂立協議，據此，SPV公司（中國交通建設間接全資附屬公司）在加拿大商業公司法下根據安排計劃以約14.5億加元收購Aecon股份。於建議收購事項完成後，Aecon將成為中國交通建設的間接全資附屬公司。但是，在2018年5月23日，加拿大联邦政府认为此收购不符合加拿大的国家安全利益，因此否决此协议。

== [[IPO|IPO]]投資者 ==
中國交通建設是[[中国大陆|中国大陆]]最大的[[基础设施|基建]][[中国中央企业|中国中央企业]]之一，具有獨特龍頭概念。招股時獲得香港不少富翁，包括[[李嘉誠|李嘉誠]]、[[李兆基|李兆基]]、[[劉鑾雄|劉鑾雄]]、[[郭鶴年|郭鶴年]]，以及[[沙特阿拉伯|沙特阿拉伯]][[皇子|王子]][[艾勒瓦利德·本·塔拉勒·阿勒沙特|阿勒瓦利德·本·塔拉勒]]（Alwaleed bin Talal）等，透過私人名義或策略性股東認購股份，也得到[[中国人寿保险|中國人壽]]集團、[[周大福|周大福]]及[[新加坡|新加坡]]政府投資機構（GIC）作為策略性股東。

== 名稱誤解 ==
很多股民在中交建上市[[招股|招股]]時，以為它是[[中國銀行|中國銀行]]、[[交通銀行|交通銀行]]及[[建設銀行|建設銀行]]的[[合資公司|合資公司]]，但其實它是從事[[基建|基建]]業，而不是[[銀行|銀行]]業。到現時為止，依然有人誤會中交建為內地銀行公司。不過，中交建本身的確持有部份中資銀行如[[交通銀行|交通銀行]]和[[招商銀行|招商銀行]]的股份。
