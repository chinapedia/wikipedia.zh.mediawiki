'''中央军委装备发展部国防知识产权局'''，位于[[北京市|北京市]]，是[[中央军委装备发展部|中央军委装备发展部]]下属局，负责国防知识产权工作。

==沿革==
1990年7月30日，国务院、中央军事委员会批准发布《国防专利条例》，其中第三条规定：“[[中华人民共和国国防科学技术工业委员会|中华人民共和国国防科学技术工业委员会]]（以下简称国防科工委）设立国防专利局。国防专利申请，统一由国防科工委国防专利局（以下简称国防专利局）受理和审查；经国防专利局审查认为符合本条例规定的，由中国专利局授予国防专利权。”<ref>{{Cite web|url =http://www.sipo.gov.cn/zcfg/flfg/zl/xzfg/201509/t20150908_1172426.html|title =国防专利条例(1990年)|author =|date =2015-09-08|publisher =中华人民共和国国家知识产权局|language =|accessdate =|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20170603201308/http://www.sipo.gov.cn/zcfg/flfg/zl/xzfg/201509/t20150908_1172426.html|archivedate =2017年6月3日|df =}}</ref>。依照该条例的规定，'''中华人民共和国国防科学技术工业委员会国防专利局'''成为受理和审查国防专利的唯一机构。

1998年，中华人民共和国国防科学技术工业委员会改组为[[中国人民解放军总装备部|中国人民解放军总装备部]]，下设有'''中国人民解放军总装备部国防专利局'''，负责受理和审查国防专利<ref>{{Cite web|url =http://www.sipo.gov.cn/zldlgl/jlycj/201310/t20131023_828632.html|title =国防专利局|author =|date =2009-06-25|publisher =中华人民共和国国家知识产权局|language =|accessdate =|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20171105134353/http://www.sipo.gov.cn/zldlgl/jlycj/201310/t20131023_828632.html|archivedate =2017年11月5日|df =}}</ref>。2003年，[[中华人民共和国国家知识产权局|中华人民共和国国家知识产权局]]同意经中国人民解放军总装备部国防专利局指定的34家机构作为办理涉及国防专利申请事务的专利代理机构<ref>{{Cite web|url =http://www.sipo.gov.cn/zwgg/gg/201310/t20131023_837703.html|title =涉及国防专利申请事务的专利代理机构公告（第84号）|author =|date =2003-01-07|publisher =中华人民共和国国家知识产权局|language =|accessdate =|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20170625144244/http://www.sipo.gov.cn/zwgg/gg/201310/t20131023_837703.html|archivedate =2017年6月25日|df =}}</ref>。2004年11月1日起，新制定的《国防专利条例》施行，其中第三条规定：“国家国防专利机构（以下简称国防专利机构）负责受理和审查国防专利申请。经国防专利机构审查认为符合本条例规定的，由国务院专利行政部门授予国防专利权。”“国务院国防科学技术工业主管部门和中国人民解放军总装备部（以下简称总装备部）分别负责地方系统和军队系统的国防专利管理工作。”<ref>{{Cite web |url = http://news.sina.com.cn/c/2004-10-09/19183864661s.shtml |title = 温家宝江泽民签署命令公布国防专利条例(全文) |author =  |date = 2004-10-09 |publisher =  新浪|language =  |accessdate =  |quote =  }}</ref>

2012年8月2日，中国国防知识产权战略实施工作会议在北京召开，会上揭牌成立'''中国人民解放军总装备部国防知识产权局'''<ref>{{Cite web |url = http://www.chinanews.com/tp/2012/08-02/4078878.shtml |title = 解放军总装备部国防知识产权局正式揭牌成立 |author =  |date = 2012-08-02 |publisher =中新网  |language =  |accessdate =  |quote =  }}</ref>。

在[[深化国防和军队改革|深化国防和军队改革]]中，2016年1月撤销[[中国人民解放军总装备部|中国人民解放军总装备部]]，组建[[中央军委装备发展部|中央军委装备发展部]]。中央军委装备发展部设'''中央军委装备发展部国防知识产权局'''。蔡镭出任中央军委装备发展部国防知识产权局第一任局长<ref>{{Cite web|url = http://www.etiri.com.cn/article_001002_1654.html|title = 装备发展部国防知识产权局局长蔡镭一行来所调研|author = |date = 2016-07-12|publisher = 工业和信息化部电子科学技术情报研究所|language = |accessdate = |quote = |archive-url = https://web.archive.org/web/20161009151130/http://www.etiri.com.cn/article_001002_1654.html|archive-date = 2016-10-09|dead-url = yes}}</ref>。

2017年3月，为深入贯彻落实军民融合国家战略，促进国防专利技术向民用领域转化应用，军委装备发展部国防知识产权局集中解密国防专利3000多件，通过全军武器装备采购信息网陆续发布，首次发布解密国防专利信息2346件。这是国防专利制度实施三十余年来，第一次集中发布国防专利解密信息<ref>{{Cite web |url = http://news.ifeng.com/a/20170305/50754870_0.shtml |title =  中央军委首次集中发布国防专利解密信息|author =  |date =2017-03-05  |publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref>。
