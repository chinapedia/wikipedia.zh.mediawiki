'''中央军委办公厅警卫局'''，位于[[北京市|北京市]]，是[[中央军委办公厅|中央军委办公厅]]下属局，负责中央军委首长等的安全警卫工作。

==沿革==
该局的前身是中央军委办公厅警卫处。其主要职责之一是负责中央军委领导的安全警卫工作。1966年8月[[中共八届十一中全会|中共八届十一中全会]]在北京召开，[[林彪|林彪]]在中国共产党内领导人的排列中上升为第二位，由于其地位上升，所以根据中央的既有规定，林彪的安全警卫工作由中央军委办公厅警卫处转交[[中央警卫局|中央警卫局]]负责<ref>{{Cite web |url = http://news.ifeng.com/history/zhongguoxiandaishi/detail_2013_03/28/23632398_0.shtml|title =林彪叛逃后我奉命做的几件事  |author =武健华  |date =  |publisher =炎黄春秋2012(5):74-80  |language =  |accessdate =  |quote =  }}</ref>。

1973年5月，[[吕恒礼|吕恒礼]]被选调至[[中央军委办公厅|中央军委办公厅]]任职，起初在中央军委办公厅警卫处工作，当时警卫处的主要职责是负责中央军委副主席[[叶剑英|叶剑英]]、[[徐向前|徐向前]]、[[聂荣臻|聂荣臻]]等在军队工作的[[中共中央政治局委员|中共中央政治局委员]]以及外地临时到北京开会的[[许世友|许世友]]、[[李德生|李德生]]、[[陈永贵|陈永贵]]、[[吴桂贤|吴桂贤]]等中共中央政治局委员的安全警卫工作，同时负责中共中央和中央军委在[[京西宾馆|京西宾馆]]举行的各类重要会议的安全警卫工作。吕恒礼多次参加首长随身、现场及多项重要会议的安全警卫工作。1976年，在同“[[四人帮|四人帮]]”斗争最激烈的时期，中央军委几位老元帅的安全警卫工作显得更为重要，当时警卫处总在紧张状态中。1976年[[粉碎四人帮|粉碎“四人帮”]]当晚，吕恒礼在警卫处值班室值班，随时防止突发事件发生<ref name=lkw>{{Cite web|url =http://www.lkwww.net/news/2007-12/19328.shtml|title =龙口将军之吕恒礼|author =|date =2007-12-18|publisher =龙口网|language =|accessdate =|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20170905002219/http://www.lkwww.net/news/2007-12/19328.shtml|archivedate =2017年9月5日|df =}}</ref>。

后来，该警卫处改为中央军委办公厅警卫局。

在[[深化国防和军队改革|深化国防和军队改革]]中，2016年1月组建新的[[中央军委办公厅|中央军委办公厅]]。中央军委办公厅继续设'''中央军委办公厅警卫局'''<ref name=zhuren/><ref>{{Cite web |url =http://www.qhdzjllc.org.cn/content.asp?ArticleID=4461  |title = 市委书记孟祥伟、市长张瑞书到中央军委办公厅汇报工作 |author =  |date =2017-02-13  |publisher =秦皇岛市政府驻北京联络处  |language =  |accessdate =  |quote =  }}</ref>。
