'''中場'''（{{lang-en|'''Midfielder'''}}）是[[足球|足球]]中一個重要位置，主要在球場中間活動，負責連繫[[前鋒_(足球)|前鋒]]與[[後衛_(足球)|後衛]]，就如右方圖中藍色位置。他們的主要功能是保持足球控制權在己方、向對手作出攔截爭奪皮球、製造攻勢，以至入球。一些中場球員是擔任防守性較強的位置，一些中場球員是擔任較前位置為前鋒創造機會。一場球賽中，中場球員的數量並不固定，通常視乎球隊需要及球員特性而安排。

出色的中場球員需要很高的技術及體能，以應付任何球賽中的攔截、帶球、射門、調動及傳球。

== 防守中場 ==
[[File:396px-Boisko_PositionsDMidfield.PNG|thumb]]
'''-{防守}-中場'''或'''後-{}-腰'''（'''Defensive midfielder'''）是指專門負責防守的中場球員，於中場負責阻截對手攻勢，然後將球交予隊友發動攻勢。

現今的防守中場球員需要很高的攔截技術和體能以應付在全場奔跑整個球場。

當代出色的防守中場包括[[加度素|-{zh-hans:加图索;zh-hk:加度素;zh-tw:加圖索;}-]]、[[羅伊·基恩|-{zh-hans:罗伊·基恩;zh-hk:堅尼;zh-tw:羅伊·基恩;}-]]、[[甘比亞素|-{zh-hans:坎比亚索;zh-hk:甘比亞素;zh-tw:坎比亞索;}-]]、[[馬基里尼|-{zh-hans:马克莱莱;zh-hk:馬基利尼;zh-tw:馬克萊萊;}-]]、[[馬斯查蘭奴|-{zh-hans:马斯切拉诺;zh-hk:馬斯查蘭奴;zh-tw:馬斯切拉諾;}-]]、[[法明尼|-{zh-hans:弗拉米尼;zh-hk:法明尼;zh-tw:弗拉米尼;}-]]、[[艾辛|艾辛]]、[[盧卡斯·雷瓦|盧卡斯]]、[[拉萨纳·迪亞拉|-{zh-hans:拉萨纳·迪亚拉;zh-hk:拉辛拿·迪亞拉;zh-tw:迪亞拉;}-]]、[[耶耶·托尼|亞亞圖雷]]、[[塞尔希奥·布斯克茨|-{zh-hans:塞尔希奥·布斯克茨;zh-hant:塞爾希奧·布斯克斯;zh-hk:沙治奧·布斯基斯;zh-tw:塞爾吉奧·布斯克茲;zh-mo:沙治奧·布斯基斯;}-]]、[[沙比·阿朗素|-{zh-hans:沙比·阿朗素;zh-hk:沙比·阿朗素;zh-tw:沙比·阿隆索;zh-mo:沙比·阿朗素;}-]]、[[亚历山大·宋|-{zh-hans:亚历山大·宋;zh-hk:阿歷山大·桑治;zh-tw:亞歷山大·宋;}-]]，[[尼热尔·德容|-{zh-hans:尼热尔·德容;zh-hk:尼祖·迪莊;zh-tw:尼熱爾·德容;}-]]、[[尼高路·簡迪|坎特]]等。

== -{zh-hans:中前卫;zh-hk:正中場;zh-tw:中前衛;}- ==
'''正中-{}-場'''或'''中前-{}-卫'''（'''Central midfielder'''），此類球員包括[[謝拉特|-{zh-hans:杰拉德;zh-hk:謝拉特;zh-tw:杰拉德;}-]]、[[保羅·史高斯|-{zh-hans:斯科尔斯;zh-hk:史高斯;zh-tw:斯科爾斯;}-]]、[[沙維|-{zh-hans:哈维;zh-hk:沙維;zh-tw:哈維;}-]]、[[迪羅斯|-{zh-hans:德罗西;zh-hk:迪羅斯;zh-tw:德羅西;}-]]及 [[托尼·克罗斯|克羅斯]]。他們是球隊主力的大腦，經常負責派球給前鋒和邊中場、翼鋒或進攻中場等較有攻擊性的球員，亦有時會進入後防，進行防守工作，是個輔助性極高的位置。

== 全能中場 ==
一些正中場會被稱為「'''全能中場'''」（'''"Box-to-box" midfielder'''），他們能夠按照戰術作出多項工作，連結後防與進攻，參與防守乃至插上助攻。大多數這類多才多藝的球員，體力充沛是最大特色，擅長攔截、傳球、射門及保持控球權。<ref>{{cite web|url=http://news.bbc.co.uk/sport1/hi/football/1957308.stm|title=Box to box Bowyer|date=2002-04-29|publisher=BBC Sport|accessdate=2012-11-08 | location=London}}</ref>

著名的例子有[[保羅·史高斯|-{zh-hans:斯科尔斯;zh-hk:史高斯;zh-tw:斯科爾斯;}-]]、[[羅伊·基恩|-{zh-hans:罗伊·基恩;zh-hk:堅尼;zh-tw:羅伊·基恩;}-]]、[[卡里克|-{zh-hans:迈克尔·卡里克;zh-hk:卡域克; zh-tw:卡里克;}-]]、[[謝拉特|-{zh-hans:杰拉德;zh-hk:謝拉特;zh-tw:杰拉德;}-]]、[[林伯特|-{zh-hans:兰帕德;zh-hk:林柏特;zh-tw:蘭帕德;}-]]、[[沙維|-{zh-hans:哈维;zh-hk:沙維;zh-tw:哈維;}-]]、[[恩尼斯達|-{zh-hans:伊涅斯塔;zh-hk:恩尼斯達;zh-tw:伊涅斯塔;}-]]、[[法比加斯|-{zh-hans:法布雷加斯;zh-hk:法比加斯;zh-tw:法布雷加斯;}-]]、[[波歷克|-{zh-hans:巴拉克;zh-hk:波歷克;zh-tw:巴拉克;}-]]、[[巴斯坦·舒韋恩史迪加|-{zh-hans:施魏因斯泰格;zh-hk:舒韋恩史迪加;zh-tw:史汪斯泰格;}-]]、[[艾辛|艾辛]]、[[韋拉|韋拉]]、[[尼維特|-{zh-hans:帕维尔·内德维德;zh-hk:帕維爾·尼維特;zh-tw:帕維爾·內德維德;}-]]、[[埃斯特万·坎比亚索|-{zh-hans:坎比亚索;zh-hk:甘比亞素;zh-tw:坎比亞索;}-]]、[[迪羅斯|-{zh-hans:德罗西;zh-hk:迪羅斯;zh-tw:德羅西;}-]]、[[克劳迪奥·马尔基西奥|-{zh-hans:马尔基西奥;zh-hk:馬治斯奧;zh-tw:馬爾基西奧;}-]]、[[阿图罗·比达尔|-{zh-hans:比达尔;zh-hk:維達爾;zh-tw:維達爾;}-]]及[[耶耶·托尼|亞亞圖雷]]由於他們的覆蓋範圍極廣，包括己方至敵方禁區邊，成功的球隊少不得全能中場。

== 墮後組織核心 ==
'''墮後組織核心'''（'''Deep-lying playmaker'''）

是指有些球員喜歡撤回後場的位置，並經常創造機會的進攻組織者，主要是因為他們傳球的能力十分出色和從墮後的位置主導比賽。儘管他們在場上的位置較為墮後，但他們不是歸類為單純的防守中場；搶斷和防守並非他們擔當的主要職責，他們同時可能需要控球中場的支援。墮後組織核心通常適量的防守職責，被認為是有很高自由度的球場支配者。
一個墮後組織核心的基本特性包括：
* 良好的長傳球調度能力，對前鋒和邊鋒傳球。因此這類球員主要的特色是的試圖以精確的長傳球給前場的隊友，在打防守反擊時，長傳將深入到對方半場。當對方的[[後衛|後衛]]被拉向中間或向中前場移動時，這類球員會積極試圖縱深長傳，例如向無人盯防的前鋒長傳，向側翼跑動中的邊線球員傳球等。
* 體能，球員需要覆蓋足球場的大部分範圍以隨時接應隊友的傳球。在頂級足球聯賽，組織核心的每場比賽跑動距離平均超過10公里。
* 良好的短傳球能力。雖然許多[[組織核心|墮後組織核心]]的有很高天賦和創造力，他們仍需要有一定的短傳球當他們墮後於半場，以避免不必要的防守反擊。他們亦要善於一腳傳球（{{lang-en|'''One-touch football'''}}）以在接球時對方逼搶下保持控球權。
* 創造力/視野。墮後組織核心，具備大範圍視野、閱讀球賽的能力，發現空位、創造機會是他們的天職。
當代出色的墮後組織核心包括[[派路|-{zh-hans:皮尔洛;zh-hk:派路;zh-tw:皮爾洛;}-]]、[[沙比·阿朗素|-{zh-hans:哈维·阿隆索;zh-hk:沙比·阿朗素;zh-tw:哈維·阿隆索;}-]]、[[沙維|-{zh-hans:哈维;zh-hk:沙維;zh-tw:哈維;}-]]、[[塞尔希奥·布斯克茨|塞爾吉歐·布斯克斯]]。而歷來最為人所熟知的，就要數[[约翰·克鲁伊夫|克魯伊夫]]了。

== 組織核心 ==
'''組織核心'''（'''Playmaker'''）

[[組織核心|組織核心]]主要站在進攻中場的位置，除經常創造機會的進攻組織者，比一般進攻中場不同的是，他們額外要控制比賽節奏，給認為是主導球賽的進攻。
一個組織核心的基本特性包括：
* 良好的短傳球能力。許多[[組織核心|組織核心]]的有很高天賦和創造力，亦要善於一腳傳球（{{lang-en|'''One-touch football'''}}）。
* 創造力/視野。組織核心，具備前場視野、閱讀球賽的能力，發現空位、創造機會是他們的天職。
當代出色的組織核心包括[[保羅·史高斯|保羅·史科爾斯]]、[[施丹|-{zh-hans:齐达内;zh-hk:施丹;zh-tw:席丹;}-]]、[[列基美|-{zh-hans:里克尔梅;zh-hk:列基美;zh-tw:里克爾梅;}-]]、[[托迪|-{zh-hans:托蒂;zh-hk:托迪;zh-tw:托蒂;}-]]、[[法比加斯|-{zh-hans:法布雷加斯;zh-hk:法比加斯;zh-tw:法布雷加斯;}-]]等。而組織核心中具備特高入球能力者，便是我們常說的典型10號球員了。最著名而又把這角色演繹得最為淋漓盡致的在足球史上共有六位：[[米歇尔·普拉蒂尼|普拉蒂尼]]、[[比利|比利]]、[[迭戈·马拉多纳|馬拉度納]]、[[约翰·克鲁伊夫|克魯伊夫]]、[[朗拿甸奴|-{zh-hans:罗纳尔迪尼奥;zh-hk:朗拿甸奴;zh-tw:羅納汀荷;}-]]和[[恩尼斯達|-{zh-hans:恩尼斯達;zh-hk:恩尼斯達;zh-tw:伊涅斯達;}-]]。

== 進攻中場 ==
[[File:396px-Boisko_PositionsAMidfield.PNG|thumb]]
'''進攻-{}-中場'''，或稱為'''前-{}-腰'''、'''攻擊中場'''（'''Attacking midfielder'''），專門負責組織進攻，製造機會予前鋒，或親自攻門。勝任這位置的球員都擁有良好的傳球和閱讀球賽能力，在適當時候傳球予前鋒攻門；也需要全面的個人技術以作個人突破，協助進攻。不少進攻中場也具備優異的遠射能力，經常作出令對手難以防備的遠距離射門。<ref>{{cite web|url=http://www.talkfootball.co.uk/guides/positions_in_football.html|title=Positions in football|publisher=talkfootball.co.uk|accessdate=2012-11-08}}</ref>

[[保羅·史高斯|保羅·史科爾斯]]、[[施丹|-{zh-hans:齐达内;zh-hk:施丹;zh-tw:席丹;}-]]、[[李華度|-{zh-hans:里瓦尔多;zh-hk:李華度;zh-tw:里瓦爾多;}-]]、[[列基美|-{zh-hans:里克尔梅;zh-hk:列基美;zh-tw:里克爾梅;}-]]、[[瑞恩·吉格斯|-{zh-hans:吉格斯;zh-hk:傑斯;zh-tw:吉格斯;}-]]、[[托迪|-{zh-hans:托蒂;zh-hk:托迪;zh-tw:托蒂;}-]]、[[朗拿甸奴|-{zh-hans:罗纳尔迪尼奥;zh-hk:朗拿甸奴;zh-tw:羅納汀荷;}-]]、[[恩尼斯達|-{zh-hans:伊涅斯塔;zh-hk:恩尼斯達;zh-tw:伊涅斯塔;}-]]及[[利昂内尔·梅西|梅西]]等被視為當代最出色的進攻中場，其他優秀的進攻中場包括[[迪高|-{zh-hans:德科;zh-hk:迪高;zh-tw:德科;}-]]、[[史奈達|-{zh-hans:斯内德;zh-hk:史尼達;zh-tw:斯內德;}-]]、[[桑·馬達|-{zh-hans:马塔;zh-hk:馬達;}-]]、[[香川真司|香川真司]]、[[奧斯爾|-{zh-hans:厄齐尔;zh-hk:奧斯爾;zh-tw:厄齊爾;}-]]、[[托尼·克罗斯|克羅斯]]、[[辛迪·卡苏拿|-{zh-hans:桑蒂·卡索拉;zh-hk:辛迪·卡苏拿;zh-tw:桑蒂·卡索拉;}-]]、[[托马什·罗西基|羅西奇]]、[[法比加斯|-{zh-hans:法布雷加斯;zh-hk:法比加斯;zh-tw:法布雷加斯;}-]]、[[菲利普·库迪尼奥|庫迪尼奧]]、[[范德法特|范德華特]]、[[卡卡|卡卡]]等。

== 中場指揮官 ==
將球隊的攻與守兩相串連，指揮全體上落節奏，彷如團隊大腦的，就是中場指揮官了。他們未必擁有高速度，也不一定具備絕頂體能，然而長短俱精，閱讀球賽能力過人，能於適當時間轉換攻守，交出致命傳送，並具備良好射術。當中的典範有：[[保羅·史高斯|保羅·史科爾斯]]、[[恩尼斯達|-{zh-hans:伊涅斯塔;zh-hk:恩尼斯達;zh-tw:伊涅斯塔;}-]]、[[安德烈亚·皮尔洛|皮爾洛]]、[[米歇尔·劳德鲁普|麥可‧勞德魯普]]、[[托马什·罗西基|托馬什·羅西基]]、[[格奥尔基·哈吉|哈吉]]、[[史杜高域|斯托伊科維奇]]、[[兹沃尼米尔·博班|博本]]、[[華達拉馬|華德拉馬]]、[[罗伯特·普罗辛内茨基|普羅辛內斯奇]]等等。

==-{zh-hans:边锋;zh-hk:翼鋒;zh-tw:翼鋒;}- ==<!-- This section is linked from [[David_Beckham|David Beckham]] -->

'''翼-{}-鋒'''或'''邊-{}-锋'''（'''Winger'''）主要在邊線進攻。由於經常需要在邊線擺脫對手[[後衛_(足球)|邊衛]]的糾纏，然後向球場中央的攻擊球員傳球以製造攻勢，所以-{zh-hans:边锋;zh-hk:翼鋒;zh-tw:翼鋒;}-一般都擁有過人的速度、優異的-{zh-hant:盤扭;zh-hans:控球}-技術和良好的傳球（尤其是長傳）能力。但現在很多-{zh-hans:边锋;zh-hk:翼鋒;zh-tw:翼鋒;}-也會直接切入對方禁區射門。


當代出色的-{zh-hans:边锋;zh-hk:翼鋒;zh-tw:翼鋒;}-包括：[[內馬爾|內馬爾]]、[[迪馬利亞|-{zh-hans:迪马里亚;zh-hk:迪馬利亞;zh-tw:迪馬里亞;}-]]、[[加里夫·巴利|-{zh-hans:格瑞斯·贝尔;zh-hk:加里夫·巴利;zh-tw:格瑞斯·貝爾;}-]]、[[曼奇尼|-{zh-hans:亚历山德罗·法奥赫·阿曼汀·曼奇尼;zh-hk:文仙尼;zh-tw:曼奇尼;}-]]、[[維辛迪|-{zh-hans:维森特;zh-hk:維辛迪;zh-tw:維森特;}-]]、[[基利斯馬|-{zh-hans:夸雷斯马;zh-hk:哥利斯馬;zh-tw:夸雷斯馬;}-]]、[[羅賓奴|-{zh-hans:罗比尼奥;zh-hk:羅賓奴;zh-tw:羅比尼奧;}-]]、[[洛賓|-{zh-hans:阿扬·罗本;zh-hk:洛賓;zh-tw:阿揚·羅本;}-]]、[[費高|-{zh:菲戈;zh-hans:菲戈;zh-hk:費高;zh-tw:菲戈;}-]]、[[列貝利|-{zh-hans:里贝里;zh-hk:列貝利;zh-tw:里貝里;}-]]、[[瑞恩·吉格斯|-{zh-hans:吉格斯;zh-hk:傑斯;zh-tw:吉格斯;}-]]、[[祖亞昆|-{zh-hans:华金;zh-hk:祖亞昆;zh-tw:華金;}-]]、[[西奥·沃尔科特|沃爾科特]]、[[拿斯利|拿斯利]]、[[大衛·席爾瓦|大衛．席爾瓦]]、[[馬爾科·羅伊斯|-{zh-hans:罗伊斯;zh-hk:萊斯;zh-tw:羅伊斯;}-]]、[[纳尼|-{zh:纳尼;zh-hans:纳尼;zh-hk:蘭尼;zh-tw:納尼;}-]]、[[佩德罗·罗德里格斯·雷德斯马|-{zh-hans:佩德罗;zh-hk:柏度;zh-tw:佩德羅;}-]]、[[阿莱克西斯·桑切斯|-{zh-hans:阿莱克西斯·桑切斯;zh-hk:阿歷斯·山齊士;zh-tw:阿萊克西斯·桑切斯;}-]]等。當然，當今兩大球星[[基斯坦奴·朗拿度|-{zh-hans:克里斯蒂亚诺·罗纳尔多;zh-hk:基斯坦奴·朗拿度;zh-tw:克里斯蒂亞諾·羅納度;}-]]、[[美斯|-{zh-hans:梅西;zh-hk:美斯;zh-tw:梅西;}-]]也屬此型的強者，但二人俱具備比翼鋒更多更強大的能力，並不止於此。另外，雖然[[碧咸|-{zh-hans:贝克汉姆;zh-hk:碧咸;zh-tw:貝克漢;}-]]並非傳統的边锋，但他在右翼位置卻具備了無人能及的傳中能力，乃是邊中場的異數。
