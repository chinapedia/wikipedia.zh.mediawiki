'''中国妇女儿童博物馆'''（{{lang-en|The Chinese Museum of Women and Children}}）位于中国[[北京市|北京市]][[东城区_(北京市)|东城区]][[建国门内大街|建国门内大街]]23号，是中国首家以妇女儿童为主题的国家级博物馆，隶属于[[全国妇联|全国妇联]]。

==历史==
该馆于2006年3月25日奠基。2009年9月26日揭牌，[[全国人大常委会副委员长|全国人大常委会副委员长]]、[[全国妇联|全国妇联]]主席[[陈至立|陈至立]]出席揭牌仪式并讲话。2010年1月10日开馆，[[中共中央政治局委员|中共中央政治局委员]]、[[国务委员|国务委员]][[刘延东|刘延东]]出席开馆仪式，[[全国人大常委会副委员长|全国人大常委会副委员长]]、[[全国妇联|全国妇联]]主席[[陈至立|陈至立]]出席并讲话<ref>{{Cite web |url = http://edu.163.com/10/0111/08/5SO1T8N800293I4V.html |title = 刘延东陈至立出席中国妇女儿童博物馆开馆仪式 |author =  |date = 2010-01-11 |publisher =网易  |language =  |accessdate =  |quote =  }}</ref><ref name=qq>{{Cite web |url = http://news.qq.com/a/20090926/001098.htm |title =  中国妇女儿童博物馆揭牌|author =  |date =2009-09-26  |publisher = 腾讯 |language =  |accessdate =  |quote =  }}</ref>。

==展览==
中国妇女儿童博物馆与全国妇联机关大楼、[[中国妇女活动中心|中国妇女活动中心]]构成了一组建筑群。该博物馆建筑由[[中国航空工业规划设计研究院|中国航空工业规划设计研究院]]、[[中国建筑设计研究院|中国建筑设计研究院]]、巴马丹拿建筑及工程有限公司设计。博物馆建筑面积约3.56万平方米，展陈面积约8000平方米。建筑共10层，其中地上6层，地下4层。地上1层为多功能厅、临时展览厅，地上2层、3层为儿童馆，4至6层为妇女馆<ref name=qq/><ref name=shunli>{{Cite web |url = http://news.xinhuanet.com/newscenter/2007-09/12/content_6709044.htm |title =  中国妇女儿童博物馆各项建设工作进展顺利|author =  |date = 2007-09-12 |publisher = 新华网 |language =  |accessdate =  |quote =  }}</ref>。 

中国妇女儿童博物馆开馆初期，馆藏文物3万多件，分为石器、陶器、瓷器、青铜器、民族服饰、生产工具、生活用品、纪念品、工艺品等实物，光盘、照片等影像资料，题词、书籍等文献资料，从不同角度多方位介绍了中国妇女儿童的生活状态、地位变化、文化习俗、社会贡献<ref name=qq/><ref name=shunli/>。
