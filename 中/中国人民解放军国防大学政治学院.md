'''中国人民解放军国防大学政治学院'''，本部位于[[上海市|上海市]]，隶属[[中国人民解放军国防大学|中国人民解放军国防大学]]。

==沿革==
===中国人民解放军南京政治学院===
[[File:Nanjing_Political_College_2.JPG|250px]]
[[File:Nanjing_Political_College_1.JPG|250px]]
[[File:Nanjing_Political_College_3.JPG|250px]]

1977年8月23日，在北京召开中央军委座谈会，[[邓小平|邓小平]]在会上指出，训练政治干部的学校要增加，可专门办政治学校。两个月零十四天后，1977年11月中央军委颁发15号文件，决定组建'''中国人民解放军南京政治学校'''，同期组建的还有[[中国人民解放军西安政治学院|中国人民解放军西安政治学校]]<ref name=sishi/>。

1977年12月，在[[江苏省|江苏省]][[南京市|南京市]]成立中国人民解放军南京政治学校<ref name=zouchu/><ref name=cnr/><ref name=zhw/>。1978年3月20日开学<ref name=cnr>{{Cite web |url =http://www.cnr.cn/military/tebie/njzzxy/  |title = 走向大众——南京政治学院马克思主义理论教员群体传播党的创新理论纪实 |author =  |date = 2008-12-29 |publisher = 中国广播网 |language =  |accessdate =  |quote =  }}</ref><ref name=sishi>{{Cite web |url =http://www.thepaper.cn/newsDetail_forward_1718290  |title =南京政治学院建校40周年，曾赢得“军中北大”盛誉  |author =  |date = 2017-06-26 |publisher = 澎湃新闻 |language =  |accessdate =  |quote =  }}</ref>。

1986年2月，中央军委印发《关于军队院校教育改革的决定》和《全军院校体制改革精简整编方案》，明确“南京政治学校改称南京政治学院”，培训层次以本科为主<ref name=sishi/>。1986年6月，正式升格为'''中国人民解放军南京政治学院'''<ref name=zhw>{{Cite web|url =http://news.china.com/domestic/945/20141217/19116458_all.html|title =解放军南京政治学院政治部主任马向东被查|author =|date =2014-12-17|publisher =中华网|language =|accessdate =|quote =}}{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>。1986年秋，举行南京政治学院命名大会暨八六级学员开学典礼<ref name=sishi/>。

1992年，'''中国人民解放军长沙政治军官进修学院'''并入，成为'''中国人民解放军南京政治学院长沙分院'''。1993年，南京政治学院长沙分院改建为'''中国人民解放军长沙政治学院'''。1999年4月并入[[中国人民解放军国防科学技术大学|中国人民解放军国防科学技术大学]]<ref>{{Cite web |url = http://news.xinhuanet.com/mil/2001-02/26/content_7076.htm |title =国防科技大学  |author =  |date = 2001-02-26 |publisher = 新华网 |language =  |accessdate =  |quote =  }}</ref>。

1999年5月，中国人民解放军空军政治学院转隶并入，成为中国人民解放军南京政治学院上海分院<ref name=cnr/>。空军政治学院的前身是1951年1月在[[河南省|河南省]][[洛阳市|洛阳市]]组建的'''中国人民解放军空军军政干部学校'''，1954年9月迁至[[南京市|南京市]]中和桥，改建成'''中国人民解放军空军政治学校'''。1962年7月迁至[[上海市|上海市]]，1986年6月更名为'''中国人民解放军空军政治学院'''。1999年5月转隶并入南京政治学院，改称中国人民解放军南京政治学院上海分院，是隶属[[中国人民解放军总政治部|中国人民解放军总政治部]]、面向全军培养政治军官的中级任职教育院校及全军“2110工程”重点建设院校。2011年6月23日，中央军委决定撤销南京政治学院上海分院建制，改建成'''中国人民解放军南京政治学院上海校区'''。同年内，南京政治学院上海校区荣誉馆被列为上海市国防教育基地<ref>{{Cite web |url = http://www.shanghai.gov.cn/nw2/nw2314/nw24651/nw39559/nw39597/u21aw1141533.html |title = 上海年鉴2015  三十二、军事（三）军事院校 |author =  |date =2016-06-24  |publisher = 中国上海 |language =  |accessdate =  |quote =  }}</ref>。

2001年，南京政治学院在被批准为博士学位授权点后，首次招收博士生<ref name=wangrz/>。2004年1月，经中央军委批准，南京政治学院为中级任职教育院校，编制等级是正军级，承担为全军培养政治指挥干部、政治机关干部、政治工作专门人才的任务<ref name=cnr/>。截至2008年，该学院是军队“2110工程”重点建设院校，建有博士后科研流动站2个，博士学位授权点12个，硕士学位授权点24个，还建有全军舆论战、心理战专业研究中心，兵情实证研究所等研究机构<ref name=cnr/>。

2016年，在[[深化国防和军队改革|深化国防和军队改革]]中，其原来的上级单位[[中国人民解放军总政治部|中国人民解放军总政治部]]撤销，该院转隶[[中央军事委员会训练管理部|中央军事委员会训练管理部]]<ref>{{Cite web |url =http://military.people.com.cn/n1/2016/0726/c1011-28585938.html  |title =中央军委训练管理部注重发挥资源优势精准实施定点帮扶工作  |author =  |date =2016-07-26  |publisher = 人民网 |language =  |accessdate =  |quote =  }}</ref>。

2017年，在[[深化国防和军队改革|深化国防和军队改革]]中，原南京政治学院与其他多所军队院校合并，调整组建为新的院校<ref>{{Cite web |url = http://www.thepaper.cn/newsDetail_forward_1728329 |title = 原南京政治学院已与其他院校进行合并，调整组建为新的院校 |author =  |date =2017-07-09   |publisher =澎湃新闻  |language =  |accessdate =  |quote =  }}</ref>。

中国人民解放军南京政治学院本部位于[[江苏省|江苏省]][[南京市|南京市]][[鼓楼区|鼓楼区]][[中山北路_(南京)|中山北路]]两侧（252、254号和303、305号），上海校区位于[[上海市|上海市]][[江湾五角场|五角场]]地区（[[杨浦区|杨浦区]]黄兴路2040号）。南京院本部主体建筑为中国古典式建筑，原为[[国民政府行政院旧址|国民政府行政院]]和[[国民政府交通部旧址|交通部大楼]]。

*2012年8月，[[武警总部|武警总部]]和[[上海市人民政府|上海市人民政府]]举行会商，研究决定了武警政治学院迁建一系列问题，启动学院迁建工程。2015年12月29日，位于[[上海市|上海市]][[松江区|松江区]]业煌路658号的武警政治学院新校区正式建成启用<ref>{{cite news |title=颜晓东少将任武警部队政治工作部主任 |url = http://china.caixin.com/2015-12-30/100894183.html |accessdate=2015-12-31 |website=财新网}}</ref><ref>{{Cite web|url = http://sj.shmzj.gov.cn/gb/mzsjq/mzxw/qjxw/u1ai5558.html|title = 武警政治学院正式搬迁入驻我区|author = |date = 2015-12-31|publisher = 松江民政|language = |accessdate = |quote = }}{{Dead link|date=2019年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref name=jbjz/>。新校区教学区占地面积350亩，毗邻[[佘山|佘山]]，三面环水<ref name=jbjz/>。
