'''中国人民解放军陆军确山合同战术训练基地'''，位于[[河南省|河南省]][[驻马店市|驻马店市]][[确山县|确山县]]，是隶属[[中国人民解放军中部战区陆军|中国人民解放军中部战区陆军]]的训练基地。

==沿革==
确山训练基地的前身是1960年组建的[[武汉军区|武汉军区]]炮兵靶场。1999年10月，该靶场更名为确山合同战术训练基地（2000年正式改建升格），隶属[[中国人民解放军济南军区|中国人民解放军济南军区]]，是[[中国人民解放军|中国人民解放军]]第二批建设的大型山地作战训练基地<ref name=xhw/><ref name=wy/><ref name=queshan/>。经过数次扩建改造，建成了现代化训练基地。确山训练基地的地形复杂，多处高地海拔超过800米。确山训练基地可保障步兵、防空兵、装甲兵等兵种在联合战役背景下山地攻防、远程机动、机降、伞降、特种作战、反恐作战等的演练<ref name=xhw/><ref name=wy>{{Cite web |url =http://news.163.com/07/1031/09/3S4GPQVN00011232.html  |title = 确山合同战术训练基地探秘：精锐部队在此诞生 |author =  |date = 2007-10-31 |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url =  http://www.henan.gov.cn/zwgk/system/2013/08/02/010414510.shtml  |title = 驻马店市旅游局开展迎“八一”拥军走访慰问活动 |author =  |date =2013-08-02  |publisher =河南省政府门户网站  |language =  |accessdate =  |quote =  }}</ref>。

确山训练基地自组建之后，每年有数万官兵来此训练，确山训练基地承担了“确山”、“前卫”等大型军事演习的任务。2011年承担了“前卫-211”济南军区信息化大练兵，其中构建了全部模拟实战的电子对抗环境。确山训练基地还是中国人民解放军第一个大型专用特种作战训练场<ref name=xhw/>。

确山训练基地还在对外交流中发挥作用。2004年9月25日，16个国家的军队领导及军事观察员、驻华武官共60多人，在确山训练基地观摩了济南军区某部“铁拳-2004”演习，这是[[中华人民共和国|中华人民共和国]]成立后截至2014年中国人民解放军邀请国家最多、外军观察员层级最高、对外展示规模最大的演习。2013年11月，各国驻华武官、军事观察员、军事留学生等大约350位外军人员在确山训练基地观摩为期两日的“确山·前锋-2013b”演习，这是截至2013年参加观摩的外军人员最多的一次中国人民解放军演习<ref name=xhw>{{Cite web |url = http://news.xinhuanet.com/mil/2014-10/10/c_127080398_5.htm |title = 揭秘解放军三大军事训练基地：朱日和科技含量最高 |author =  |date = 2014-10-10 |publisher =新华网  |language =  |accessdate =  |quote =  }}</ref>。

2016年，在[[深化国防和军队改革|深化国防和军队改革]]中，确山训练基地由[[中国人民解放军济南军区|中国人民解放军济南军区]]转隶[[中国人民解放军中部战区陆军|中国人民解放军中部战区陆军]]<ref name=budui>{{Cite web |url = http://henan.people.com.cn/n/2014/0801/c351638-21849942.html |title =  驻马店市领导深入军营走访慰问驻军官兵|author =  |date =2014-08-01  |publisher = 人民网 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://www.huaxia.com/thjq/jsxw/dl/2016/07/4912869.html |title = 中部战区陆军某训练基地冒着高温酷暑改造演习场 |author =  |date = 2016-07-05 |publisher =华夏经纬网  |language =  |accessdate =  |quote =  }}</ref>。2016年11月中旬，中部战区陆军确山训练基地伞降训练风洞试飞成功，这座风洞是中国人民解放军首座专门用于伞降训练的风洞<ref>{{Cite web |url =  http://www.js7tv.cn/news/201612_70447.html|title =中部战区陆军某训练基地伞降训练风洞试飞成功  |author =  |date = 2016-12-01 |publisher = 中国军视网 |language =  |accessdate =  |quote =  }}</ref>。
