'''中国人民解放军空军航空大学'''，简称'''空军航空大学'''，位于[[吉林省|吉林省]][[长春市|长春市]]，是以培养飞行人才为主的综合性高等军事院校，隶属[[中国人民解放军空军|中国人民解放军空军]]<ref name=zgkj>{{Cite web |url = http://kj.81.cn/content/2012-12/14/content_5143016.htm |title = 空军航空大学 |author =  |date =2012-12-14  |publisher =中国空军网  |language =  |accessdate =  |quote =  }}</ref><ref name=xunli>{{Cite web |url = http://www.81.cn/jwgz/2017-06/13/content_7637639.htm |title = 2017军校巡礼第二十二站：空军航空大学（附报考指南） |author =  |date = 2017-06-13 |publisher =中国军网  |language =  |accessdate =  |quote =  }}</ref>。

==沿革==
中国人民解放军空军航空大学的前身可以追溯至1946年创办的[[中国人民解放军|中国人民解放军]]历史上第一所航空学校——[[东北民主联军航空学校|东北民主联军航空学校]]<ref name=zgkj/>。
*1946年3月1日，'''东北民主联军航空学校'''在[[吉林|吉林]][[通化|通化]]成立，隶属[[东北民主联军|东北民主联军]]，这是[[中国人民解放军|中国人民解放军]]历史上第一所航空学校。1946年4月，由通化迁至[[牡丹江|牡丹江]]。1946年11月迁至东安（今[[密山|密山]]）。1948年1月1日更名'''东北人民解放军航空学校'''，隶属[[东北人民解放军|东北人民解放军]]。1948年3月迁回牡丹江。[[辽沈战役|辽沈战役]]结束后，1949年3月迁至吉林[[长春|长春]]。1949年5月更名'''中国人民解放军航空学校'''。因该校驻地一直在[[中国东北|中国东北]]，故人称“东北老航校”<ref name=ccrb>{{Cite web |url =http://ccrb.1news.cc/html/2017-05/19/content_520883.htm  |title = 拥有18个“空军之最” |author =  |date =2017-05-19  |publisher = 长春日报 |language =  |accessdate =  |quote =  }}</ref><ref name=baijiahao/><ref name=jishi>{{Cite web |url = http://news.ifeng.com/a/20170807/51581299_0.shtml |title =中国人民空军及航空事业纪事(1949年前)  |author =  |date =2017-08-07  |publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref>。

;中国人民解放军空军长春飞行学院
*1948年4月24日，成立'''东北人民解放军航空学校第三大队'''。主要负责对新学员进行入伍教育，机务工作教育<ref name=jishi/>。
*1949年9月14日，中国人民解放军航空学校发布命令，决定取消第四大队建制，以原第三大队为基础成立'''中国人民解放军航空学校混合大队'''<ref name=jishi/>。1950年，东北军区航空学校混合大队在[[长春|长春]]创建<ref name=ccrb/>。
*1950年1月14日，航空学校混合大队改编为'''中国人民解放军空军入伍生大队'''。
*1950年5月23日，更名'''中国人民解放军航空学员预科总队'''<ref name=baijiahao>{{Cite web |url = http://baijiahao.baidu.com/s?id=1561342112264040&wfr=spider&for=pc |title = 那些年我们一起上过的军校——空军航空大学|author =  |date =2017-03-09  |publisher =百家号  |language =  |accessdate =  |quote =  }}</ref>。后来空军长春飞行学院以1950年5月23日为建校纪念日。
*1951年7月13日，更名'''中国人民解放军第一航空预科总队'''<ref name=jlszjsz/>。
*1954年5月，更名'''中国人民解放军第一航空预备学校'''<ref name=ccrb/><ref name=jlszjsz> {{Cite book | author = 吉林省地方志编纂委员会 | title = 吉林省志·卷十四·军事志 | location =  | publisher = 吉林人民出版社 | date = 1996年 | pages =145-149  | ISBN =  | accessdate =  | url =  | language =  | quote =  }} </ref>。
*1955年，成为唯一被列入空军司令部序列的航空预备学校。
*1958年3月，[[中国共产党中国人民解放军空军委员会|空军党委]]决定，撤销第二、三、四、五、六、七、八航空预备学校，保留第一航空预备学校。
*1959年2月20日，更名'''中国人民解放军空军第一预备学校'''。
*1964年9月15日，更名'''中国人民解放军空军第一航空预备学校'''。
*1965年4月，更名'''中国人民解放军空军第一预备学校'''。
*1969年10月25日，撤销空军第一预备学校<ref name=jlszjsz/>。
*1973年11月5日，经中央军委批准复校，更名'''中国人民解放军空军第一航空预备学校'''<ref name=jlszjsz/>。
*1986年，更名'''中国人民解放军空军第一飞行基础学校'''<ref name=ccrb/>。
*1993年，升格'''中国人民解放军空军长春飞行学院'''<ref name=ccrb/>。
*1993年，空军第二飞行基础学校并入，改为空军长春飞行学院保定分院。1999年，撤销空军长春飞行学院保定分院。
**中国人民解放军空军长春飞行学院保定分院：1958年3月，[[中国共产党中国人民解放军空军委员会|空军党委]]决定，撤销第二、三、四、五、六、七、八航空预备学校，保留第一航空预备学校，同时在[[江西|江西]][[南昌|南昌]]以原第三、四航空预备学校为基础创建新的'''中国人民解放军第二航空预备学校'''。1959年，更名'''中国人民解放军空军第二预备学校'''。1959年4月，空军第二预备学校从南昌迁址保定，与原[[中国人民解放军空军工程大学|空军第八航空预备学校]]部分人员合并。1964年，更名'''中国人民解放军空军第二航空预备学校'''。1965年，更名'''中国人民解放军空军第二预备学校'''。1969年，撤销空军第二预备学校。1973年，复建'''中国人民解放军空军第二航空预备学校'''。1986年，更名'''中国人民解放军空军第二飞行基础学校'''。1993年，撤销空军第二飞行基础学校，并入空军长春飞行学院成为'''中国人民解放军空军长春飞行学院保定分院'''。1999年，空军长春飞行学院保定分院撤销<ref name=eryuxiao>{{Cite web |url = http://news.ifeng.com/a/20170824/51742448_0.shtml |title = 空军二预校——我的母校!_ |author =  |date = 2017-08-24 |publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref>。
*2004年6月9日，转隶空军航空大学。同年改称'''中国人民解放军空军航空大学飞行基础训练基地'''<ref>{{Cite web |url = http://mil.news.sina.com.cn/2005-06-13/2335296773.html |title = 空军航空大学用科学发展观指导办学纪实 |author =  |date = 2005-06-13 |publisher = 新浪 |language =  |accessdate =  |quote =  }}</ref>。

;中国人民解放军空军第二航空学院
*1952年8月1日，在吉林长春建立'''中国人民解放军空军第九航空学校'''<ref name=jlszjsz/><ref name=dier>{{Cite web |url = http://www.mod.gov.cn/service/2007-07/09/content_4085955.htm |title = 空军第二航空学院 |author =  |date = 2007-07-09 |publisher = 中华人民共和国国防部 |language =  |accessdate =  |quote =  }}</ref>。
*1956年，第九航空学校气象预报系分出，并入空军气象专科学校（后为[[中国人民解放军空军气象学院|中国人民解放军空军气象学院]]）。
*1967年5月，第九航空学校更名'''中国人民解放军空军第二航空机务学校'''<ref name=jlszjsz/>。（同时，中国人民解放军空军第十五航空学校（筹）更名[[中国人民解放军空军第九航空学校|中国人民解放军空军第九航空学校]]。）
*1969年，撤销第二航空机务学校。
*1978年，复建第二航空机务学校。
*1986年6月，升格'''中国人民解放军空军第二航空技术专科学校'''。
*1992年10月，升格'''中国人民解放军空军第二航空学院'''<ref name=dier/>。
*2004年6月9日，转隶空军航空大学。主体改为空军航空大学本部。

;中国人民解放军空军第七飞行学院
*1949年11月18日，奉中央军委命令，利用东北老航校整编后留在[[牡丹江|牡丹江]]的剩余人员，改建牡丹江航校。校部设在牡丹江东北老航校校址内，飞行教官及其他专业技术人员主要是东北老航校原[[日本|日本]]籍留用人员<ref name=diqi>{{Cite web |url = http://www.81.cn/2016kjlhx/2016-05/31/content_7079944_2.htm |title = 东北老航校：人民军队的第一所航空学校 |author =  |date = 2016-05-31 |publisher = 中国军网 |language =  |accessdate =  |quote =  }}</ref>。
*1949年12月20日，中央军委颁布命令，统一命名各航校，牡丹江航校定名'''中国人民解放军第七航空学校'''<ref name=diqi/><ref>{{Cite web |url =  http://www.81.cn/syjdt/2015-03/12/content_6392069.htm|title =揭秘：最早的中国航校  |author =  |date =2015-03-12  |publisher = 中国军网 |language =  |accessdate =  |quote =  }}</ref>。1950年1月5日正式开学<ref name=diqijj>{{Cite web |url =http://edu.sina.com.cn/gaokao/2011-09-21/1456313500.shtml  |title =空军第七飞行学院简介  |author =  |date = 2011-09-21 |publisher = 新浪 |language =  |accessdate =  |quote =  }}</ref>。
*1952年1月，更名'''中国人民解放军空军第七航空学校'''<ref name=diqi/>。该校先后培训了[[中华人民共和国|中华人民共和国]]第一批女航空员和第五批女飞行员，组建了中国人民解放军空军第一个直升机大队<ref name=jlszjsz/>。
*1967年2月，学校扩编到7个训练团，驻地分布在黑龙江省、吉林省、辽宁省<ref name=hljszjsz/>。
*1968年7月，由牡丹江市迁驻长春市<ref name=jlszjsz/>。1968年7月7日，校部从牡丹江市迁驻长春市的原[[中国人民解放军空军第二航空学校|中国人民解放军空军第二航空学校]]驻地。这时该校留驻黑龙江省的还有两个团：驻双城的二团、驻王岗的五团<ref name=hljszjsz/>。
*1986年6月，更名'''中国人民解放军空军第七飞行学院'''<ref name=diqi/>。
*2004年6月9日，转隶空军航空大学。2004年9月30日改称'''中国人民解放军空军航空大学飞行训练基地'''<ref name=diqi/>。

;中国人民解放军空军第十三飞行学院
*1953年1月，陆军第七十五师师部调空军，在[[北京南苑机场|北京南苑机场]]组建'''中国人民解放军空军中级指挥员训练班'''<ref name=xiexch> {{Cite book | author = 谢雪畴 | title = 穿破硝烟的身影 | location =  | publisher = 山西人民出版社 | date =  2012年| pages =  | ISBN =  | accessdate =  | url =  | language =  | quote = 作者简介 }} </ref>。

*1963年，更名'''中国人民解放军空军第一高级专科学校'''<ref name=xiexch/><ref name=shisan>{{Cite web |url = http://www.mod.gov.cn/service/2009-09/09/content_4086529.htm |title =飞行学院介绍：空军第十三飞行学院  |author =  |date = 2009-09-09 |publisher = 中华人民共和国国防部 |language =  |accessdate =  |quote =  }}</ref>。

*1976年6月，更名'''中国人民解放军空军第十三航空学校'''<ref name=shisan/>。
*1979年，迁址[[安徽|安徽]][[蚌埠|蚌埠]]<ref name=ahszjsz> {{Cite book | author = 安徽省地方志编纂委员会 | title = 安徽省志·军事志 | location =  | publisher = 安徽人民出版社 | date = 1995年 | pages = 517-518 | ISBN =  | accessdate =  | url =  | language =  | quote =  }} </ref>。

;中国人民解放军空军航空大学
*2004年6月9日，空军长春飞行学院、空军第七飞行学院、空军第二航空学院合并，组建成立'''中国人民解放军空军航空大学'''。后成为中央军委批准的军队“2110工程”重点建设院校<ref name=xunli/>。空军第二航空学院主体改为空军航空大学本部；空军长春飞行学院改为空军航空大学飞行基础训练基地；空军第七飞行学院改为空军航空大学飞行训练基地。
*2012年4月，空军第十三飞行学院和空军第三飞行学院第一训练团并入<ref>空军航空大学，高等教育研究学报2012年04期</ref>。2012年4月28日，空军第十三飞行学院编为空军航空大学蚌埠校区，组建空军航空大学飞行教官训练基地<ref>{{Cite web|url =http://www.bbnews.cn/xw/bb0yw/2012/05/383611.shtml|title =空军航空大学飞行教官训练基地在蚌揭牌|author =|date =2012-05-01|publisher =蚌埠新闻网|language =|accessdate =|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20160924002646/http://www.bbnews.cn/xw/bb0yw/2012/05/383611.shtml|archivedate =2016-09-24}}</ref>。
*2017年，在[[深化国防和军队改革|深化国防和军队改革]]中，调整组建新的'''中国人民解放军空军航空大学'''<ref>{{Cite web |url = http://news.ifeng.com/a/20170530/51183107_0.shtml |title =  2017年军队招生单位名录发布 军改后院校新名称曝光|author =  |date = 2017-05-30 |publisher =凤凰网  |language =  |accessdate =  |quote =  }}</ref>。

*飞行技术
*侦察情报（1）
*侦察情报（2）
*目标工程
