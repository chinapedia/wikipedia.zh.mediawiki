'''中華人民共和國人大代表獨立候選人'''是指1990年代後期開始出現的沒有[[中華人民共和國|中華人民共和國]]官方背景的[[人大代表|人大代表]][[候選人|候選人]]。不過，這些獨立候選人中成功當選的人卻被認爲是「屈指可數」。另外，[[親中媒體|立場親中共的媒體]]也對他們的動機提出了質疑。

==概況==
長久以來，中華人民共和國的人大代表選舉被指責爲「走過場」，「當選人始終是當局內定的人」。<ref name="B2" />

而在1998年，[[姚立法|姚立法]]以「自薦參選人」的身份成功當選當屆湖北省潛江市的人大代表。<ref name="DG" />另外，在2003年也有類似的成功案例。<ref>{{Cite web |url = http://morning.scol.com.cn/2003/05/22/200305223590530314313.htm |title = 天府早報：独立候选人首当人大代表}}</ref>

在2006年中華人民共和國人大代表選舉時，[[中國泛藍聯盟|中國泛藍聯盟]]曾組織一百多名成員以[[中國國民黨精神黨員|中國國民黨精神黨員]]个人身份獨立參選人大代表選舉。<ref>[http://www.boxun.com/news/gb/china/2006/08/200608181157.shtml 中央電台專訪文炎：中國泛藍聯盟將如何操作人大代表的選舉2006年8月18日]</ref>

在2011年中華人民共和國啓動縣鄉級人大代表選舉時，出現了一批獨立候選人。這些獨立候選人的身份包括下崗[[工人|工人]]、普通[[市民|市民]]和[[學生|學生]]，也有[[作家|作家]]、[[教授|教授]]、[[記者|記者]]、[[律師|律師]]和[[IT業|IT業]]人士。<ref name=B2>{{Cite web|url= http://www.bbc.co.uk/zhongwen/trad/chinese_analysis/2011/11/111117_china_election.shtml|publisher=BBC|title=特寫：從獨立參選人遭遇看中國民主進程}}</ref>其中比較知名的有[[李承鵬|李承鵬]]、[[夏商|夏商]]、[[梁樹新|梁樹新]]。李承鵬甚至組織了陣容強大的助選團。<ref name=DG>{{Cite web|url=http://news.takungpao.com.hk/mainland/yangguang/q/2013/0325/1510665.html|title=「請投我一票！」——「獨立候選人」的參選之路|publisher=大公網}}</ref>

但是，最後當選的獨立候選人卻寥寥無幾。大部分獨立候選人在初選中就被淘汰。<ref name=B2/><ref name=DG/>

2012年，北京地區的部分獨立候選人舉行了新年聚會。<ref>{{Cite web|url=http://www.voachinese.com/content/article-20120110-independent-candidates-new-year-137005173/792727.html|title=獨立人大代表參選人新年聚會|publisher=VOA}}</ref>

==獨立人士受到政治打壓==
雖然中華人民共和國的法律規定公民有選舉權和被選舉權，但是獨立候選人在選舉中卻受到了當局不同程度的打壓。有獨立候選者受到了當局的威脅，甚至有人受到拘禁和毆打。<ref name="DW">{{Cite web |url = http://18.dwnews.com/news/2011-11-07/58285411.html |title = 北京基层人大8日选举 独立候选人全遭出局 |publisher=多維新聞}}</ref><ref name=VOA1>{{Cite web |url = http://www.voacantonese.com/content/article-9-s-hsz-dc-talk-to-independent-candidates-133290603/935114.html |title = 多名中國人大獨立參選人被迫退出選舉 |publisher=[[VOA|VOA]]}}</ref>

在2006年中華人民共和國人大代表選舉，[[中國泛藍聯盟|中國泛藍聯盟]]成員作为[[中國國民黨精神黨員|中國國民黨精神黨員]]獨立參選並且製作了大量「從[[中國共產黨|揭露執政黨腐敗弊端]]到以[[兩德統一|東西德模式統一]]」的選舉綱領，並選舉名片和文化衫挨家挨戶發放，後均遭[[公安部国内安全保卫局|有关机构]]約談勸退，其中，[[文炎|文炎]]、[[蔡愛民|蔡愛民]]、倪江峰、左曉環等拒不退出的泛藍參選人分別被不明身份人士暴力毆打，軟禁在家或被開除公職，[[中國國民黨精神黨員|中國國民黨精神黨員]]參選被迫中止。<ref>[http://www.rfa.org/mandarin/yataibaodao/sunbuer-20060912.html 中國泛藍聯盟參選人文炎遭毆打入院]</ref>

[[美國之音|美國之音]]報道說，獨立候選人之一，[[新余|新余]]鋼鐵公司職工魏忠平在2012年2月28日被強制帶到鐵坑鐵礦，並受到多人24小時監守。他在被拘期間遭到多人毆打，導致了左耳[[中耳炎|急性中耳炎]]、以及三根[[肋骨|肋骨]]和[[橫突骨|橫突骨]]的骨折。他在被打後到3月17號獲釋的13天內多次要求治療卻無果。當地支持獨立候選人的律師也遭到了毆打。<ref name=VOA1/><ref>{{Cite web |url = http://www.voachinese.com/content/article-20111001-independent-candidates-130905368/788501.html |title=人大代表独立候选人继续受到打压|publisher=[[VOA|VOA]]}}</ref>另外，貴州省獨立參選人幸清賢在2011年11月1日得知﹐他已經在初選中被“淘汰”。幸清賢認為初選過程不公正。例如﹐有當地民眾反映﹐參加初選提名的人都是領導指定的﹐到會的每個“推薦人”都得到了20元補助費。他還說，他散發競選傳單的行爲被當地派出所指責爲「擾亂秩序」。<ref name=VOA1/>

在北京，獨立參選人北京交通大學教授張映偉也宣佈退出2011年的人大代表競選。獨立候選人最終全部被淘汰。<ref name="B2" />

==評論==
[[File:GreatHall_auditorium.jpg|缩略图]]議場]]

===正面===
中華人民共和國憲法學教授[[童之偉|童之偉]]對獨立候選人的出現發表評論，稱「（人大代表獨立候選人）是在計劃政治原野上點燃了市場政治的火炬。」<ref name="B2" />

===負面===
有親中共的觀點認爲，部分獨立候選人「別有圖謀」，試圖顛覆「現有政權」。<ref>{{cite web |url = http://www.chinaelections.com/article/1588/224969.html|title=浦兴祖：“独立候选人”现象辨析}}</ref>

===其它評論===
[[全国人大常委会法制工作委员会|全国人大常委会法制工作委员会]]負責人在接受[[中國網絡電視臺|中國網絡電視臺]]的採訪時表示「中華人民共和國只有經由協商後得以提名的『正式候選人』...没有所谓的『独立候选人』。『独立候选人』没有法律依据。」<ref>{{Cite web |url = http://news.ifeng.com/mainland/detail_2011_06/08/6889506_0.shtml |title=全国人大法工委：“独立候选人”无法律依据 |publisher=中國網路電視臺}}</ref>

有人認爲，在中華人民共和國現有制度下，獨立候選人這一途徑是「走不通的」。<ref name="DG" />
