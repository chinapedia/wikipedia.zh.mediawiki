'''中華民國衛哨兵'''指的是[[中華民國國軍|中華民國國軍]]執行衛哨勤務的[[軍人|軍人]]，職責是對出入營門的人員實施身分查驗，同時在有必要及合法的時候查驗出入人員攜帶的物品或行李。<ref>{{cite|title=國防部軍法教育 問題解答|url=http://www.mnd.gov.tw/Publish.aspx?cnid=444&p=378|publisher=中華民國國防部|quote=營門衛哨兵對出入人員實施身分查驗，及隨車（身）攜行物品（行李）查驗，有其必要性及合法性，但執行時要採取合法手段……}}</ref>

根據《[[:s:陸軍禮節條例|陸軍禮節條例]]》（1928年），「'''衛兵'''」是指檢查風紀及[[屯駐地|衛戍]]的士兵，「'''衛哨'''」是指站崗的衛兵。<ref>《[http://lis.ly.gov.tw/lghtml/lawstat/version2/90017/9001717072600.htm 陸軍禮節條例] 》{{webarchive|url=https://web.archive.org/web/20141227213651/http://lis.ly.gov.tw/lghtml/lawstat/version2/90017/9001717072600.htm |date=2014-12-27 }}（1928年）第三條：「巳、凡稱為'''衛兵'''，係合風紀衛兵及衛戍衛兵而言，'''衛哨'''者，站崗衛兵之謂。」</ref> 經過修訂以後，現時的「'''衛兵'''」是指檢查風紀及衛戍的士兵，「'''哨兵'''」是指在陣中勤務令以外執行步哨的士兵。<ref>《[http://lis.ly.gov.tw/lghtml/lawstat/version2/90017/9001731043000.htm 陸軍禮節條例] 》{{webarchive|url=https://web.archive.org/web/20141228001118/http://lis.ly.gov.tw/lghtml/lawstat/version2/90017/9001731043000.htm |date=2014-12-28 }}（1942年）第二條：「稱'''衛兵'''者，指風紀衛兵及衛戍衛兵。稱'''哨兵'''者，指陣中勤務令以外之步哨。」</ref>

==衛哨勤務==
在營區派遣衛兵[[巡邏|巡查]]及站哨的目的，是爲了確保營區內外時常保持警戒，以維護營區設施及人員的安全，並維持軍隊內部的風紀。<ref name=edu /> 執勤衛兵按照[[指揮官|營區指揮官]]、值星官及值日官（或勤務單位主官、值星官及值日官）的指令執行勤務，分爲一般衛兵勤務和特別衛兵勤務兩種。<ref name=edu />

衛哨勤務表由[[長官|長官]]排定，一般勤務通常是以2小時爲一個班次，每人每天要輪1－2班（假日時有可能增加至3－4班），站哨地點在槍房和營門內外，[[志願役|志願役]]常常輪站大夜班，但可填寫換哨單錯開放假時間。勤務的編組包括衛兵司令、衛舍長、衛兵長和衛兵，而衛舍設置通常有衛兵司令辦公室、衛兵宿舍和崗亭。<ref name=edu />

==衛哨勤務守則==
[[File:Kai-shou_Review_Stand_before_Event_Opening_20131012.jpg|thumb]]營區開放活動，陸軍士兵在介壽臺周圍站哨。]]
衛哨兵需要熟悉的一般守則包括《用槍時機》、《用槍要領》、《衛兵守則》和《安官守則》，特別守則根據個別單位和哨站而有不同。
{{quotebox|title=用槍時機|align=left|fontsize=95%|quote=
一、當生命身體受暴行脅迫，非使用武器不能抵抗或自衛時；<br />
二、群眾暴動，非使用武器不能鎮壓時；<br />
三、所警衛之人員、物資車船航空器受危害脅迫時，非使用武器不能保護時；<br />
四、因防衛駐守之土地、場所建築物受襲擾或擅闖經警告仍不聽從非使用武器不能制止時；<br />
五、要犯逃脫非使用武器不能制止時。
|source=陸總部，《衛哨勤務守則》}}

{{quotebox|title=用槍要領|align=left|fontsize=95%|quote=
一、如非情勢異常急迫，應先口頭警告並對空鳴槍警告之；<br />
二、經警告後已有畏服之情況時應立即停止使用；<br />
三、使用武器應注意，勿傷及他人；<br />
四、如非情勢確有必要應儘量避免傷及其致命部位；<br />
五、武器使用後，應將經過情形即時報告該營長官。
|source=陸總部，《衛哨勤務守則》}}

==衛兵和哨兵的敬禮==
[[File:Executive_Officer,_1st_Lieutenant_Huan_Directing_Cars_on_Crossroad_20140329.jpg|thumb]][[湖口鄉|湖口]]營區開放活動，在二號門外指導交通的[[中尉|中尉]]副連長。]]
衛哨兵執行勤務時，按《陸軍禮節條例》須向[[國旗|國旗]]、長官、軍官率領的部隊、軍官佐及軍士[[敬禮|敬禮]]，敬禮方式包括扶槍禮、舉刀禮和舉手禮，如有特別任務時亦可不敬禮；兵員出入營門時應先向衛兵敬禮，衛哨兵則立正原地以[[敬礼#注目礼|注目]]答禮。<ref>《[http://lis.ly.gov.tw/lghtml/lawstat/version2/90017/9001733111800.htm 陸軍禮節條例] 》{{webarchive|url=https://archive.is/20141227224336/http://lis.ly.gov.tw/lghtml/lawstat/version2/90017/9001733111800.htm |date=2014-12-27 }}（1944年）第五十四、五十五、五十六及五十七條</ref>

現行《中華民國軍人禮節》對衛兵和哨兵的敬禮規範如下：<ref name=edu>{{cite|title=軍事知能 軍隊生活規範|url=http://web.nutn.edu.tw/gac370/teaching/know/know-2.htm|publisher=國立臺南大學軍訓室}}</ref>
* 對出入營門的各級軍官應行提槍敬禮；
* 對出入營門的各級士官長及士官應行立正敬禮；
* 入伍生及士兵出入營門時應先行向衛兵敬禮，衛兵回以立正答禮；
* 衛兵和哨兵在部隊經過時應行立正，並向帶隊長官敬禮；
* 哨兵應就原位置對各級軍官及士官行持槍禮，如因執行勤務而有不便時，可免行禮。

==衛哨交接==
衛哨交接是[[哨兵|哨兵]]交接勤務時的動作，上下哨在安全士官（帶哨士官）的指導下完成換哨。<ref>{{cite|title=指揮官囑完善衛哨勤務|url=http://afpc.mnd.gov.tw/Publish.aspx?cnid=1636&p=63626&Level=1|date=2014-12-09|publisher=[[中華民國憲兵|憲兵指揮部]]轉載|newspaper=[[青年日報|國防部青年日報社]]}}</ref><ref>{{cite|author=韓永聲|title=北部地區後備指揮部舉辦全軍「衛哨勤務示範觀摩」|url=http://afrc.mnd.gov.tw/thenewindex/News.aspx?NewsID=102|publisher=[[國防部後備指揮部|國防部後備指揮部]]|deadurl=yes|archiveurl=https://web.archive.org/web/20141229163354/http://afrc.mnd.gov.tw/thenewindex/News.aspx?NewsID=102|archivedate=2014-12-29}}</ref> 一般衛哨交接的動作不似[[儀隊|儀隊]]交接那般複雜，但同樣有一套經過編排的流程、[[正步|正步]]和{{tsl|en|Drill commands|口令 (步操)|口令}}，大致流程如下：
# 上哨兵到哨所向安全士官報到及領取[[槍支|槍支]]，聽從安全士官的指令填充[[彈藥|彈藥]]並確認無誤；
# 安全士官帶領上哨兵到達崗位，[[監督|監督]]上下哨兵交接勤務、位置和裝備，確認完成；
# 下哨兵隨同安全士官返回哨所，在安全士官的指令下清查彈藥、槍支和狀況，確認無誤並離開。

==相關法律==
衛哨兵失職時，不管是否造成軍事上的利益損失，可按《[[:s:陸海空軍刑法_(民國102年立法103年公布)|陸海空軍刑法]]》定罪，與衛哨站崗直接相關的罪名有：第34條「衛兵、哨兵或其他擔任警戒職務之人，因睡眠、酒醉或其他相類之情形，而廢弛職務……」（衛兵哨兵廢弛職務罪）或第35條「衛兵、哨兵或其他擔任警戒、傳令職務之人，不到或擅離勤務所在地者……」（衛兵哨兵擅離勤務所在地罪）處刑。<ref>{{cite|title=國防部軍法教育 案例教材第二冊 衛哨巡查失職案例|url=http://www.mnd.gov.tw/Publish.aspx?cnid=444&p=407|publisher=[[中華民國國防部|中華民國國防部]]}}</ref><ref name=laws /> 他人故意造成衛哨兵無法執行巡查或交接勤務時，可按《陸海空軍刑法》第36條「無故不依規定使衛兵、哨兵或其他擔任警戒、傳令職務之人交接者……」（違規使衛兵哨兵交接罪）處刑。<ref name=laws>《[http://law.mnd.gov.tw/scp/Query4B.asp?FullDoc=%A9%D2%A6%B3%B1%F8%A4%E5&Lcode=A012000001 陸海空軍刑法]》（2014年1月15日修正）</ref>

==流行文化==
{{unreferenced|章節|time=2014-12-26T01:40:26+00:00}}
衛哨兵執行勤務時需要長時間站哨，所以在[[BBS|BBS]]站內稱爲「'''狂站士'''」，勤務排在輕鬆時段時稱爲「'''爽哨'''」。

; 一般狂站士
: 指正常執行衛哨勤務之衛兵。

; 站兩歇四
: 站哨2小時，休息4小時，一個人每天要站4次共8小時哨。

; 站兩歇六
: 站哨2小時，休息6小時，一個人每天要站3次共6小時哨。

; 站四歇八
: 站哨4小時，休息8小時，一個人每天要站2次共8小時哨。

; 爆氣狂站士
: 指執行站哨兩小時，休息四小時。
: 未依規定休息，執行衛哨勤務超過2小時之衛兵。

; 拖哨
: 指接班衛兵因故遲到五分鐘以上。

; 空軍爽站士
: 站哨2小時，站哨次數依單位各有不同。

; 槍房
: 站哨2小時，有椅子可坐，爽站士中的爽站士。

; 營區內普通哨
: 站哨2小時。如路途較遠需騎腳踏車到達，天高皇帝遠，蚊子吸飽血。

; 營區外圍哨
: 站哨2小時。負責營區外圍門禁管制，突發狀況最多。
 
;拖哨仔
:對時常無故拖哨者的稱呼。
