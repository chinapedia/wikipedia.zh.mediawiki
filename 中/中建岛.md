'''中建岛'''（{{lang-en|Triton Island}}；{{vie|v=Đảo Tri Tôn|hn=島知尊}}）位于[[西沙群島|西沙群島]]西南处，是西沙群岛最南端的[[岛屿|岛屿]]，距离西沙群岛[[永兴岛|永兴岛]]178公里。为纪念1946年[[中华民国政府|中华民国政府]]派往接收西沙群岛的“中建号”军舰而得名。因从海南岛去往[[南沙群岛|南沙群岛]]的中途上，故被渔民称为“半路峙”，又盛产各种海螺，故又被渔民称为“螺岛”。该岛是在[[台礁|台礁]]顶部发育成的灰沙岛，四周有沙堤，内有残留浅水[[潟湖|潟湖]]。台风大潮时常会淹没岛的大部分。現時由[[中華人民共和國|中華人民共和國]]實控。

== 歷史 ==
* 1935年，中華民國水陸地圖審查委員會公佈名稱為「土莱塘岛」。<ref>{{Cite journal|title=文化生态学视角下的南海诸岛地名文化|author=赵静|url=http://www.rddl.com.cn/CN/abstract/abstract2806.shtml|journal=熱帶地理|issue=36|doi=|others=|year=2016|volume=6|page=1047|pmid=}}</ref>
*1946年，[[中華民國政府|中華民國政府]]派海军“中建号”军舰接收本岛，因此得名为'''中建岛'''。
[[File:中建艦.jpg|thumb]]
* 1975年5月，解放軍一艘炮艇運送了一支7人小分隊登上中建島，開啟駐防歷史，中華民國國軍並未回應。
* 1978年1月，中建島守備隊正式成立。為了改善居住條件，守島官兵也展開綠化，自帶泥土樹苗和菜種，在珊瑚沙上填土造地。自此中建島確立納入中國解訪軍實際管理版圖。
* 1982年8月11日，[[中華人民共和國|中華人民共和國]][[中央军委|中央军委]]授予中建岛守备队“爱国爱岛天涯哨兵”荣誉称号。<ref>[http://news.mod.gov.cn/forces/2011-06/02/content_4244778.htm]</ref>
* 1996年，中国政府发布关于[[领海|领海]]范围的声明，中建岛有七点是[[中国领海基点|中国领海基点]]。<ref>[http://www.mfa.gov.cn/chn/gxh/zlb/tyfg/t556673.htm 中华人民共和国政府关于中华人民共和国领海基线的声明 中华人民共和国外交部]</ref>

== 地理位置 ==
全岛海拔平均2米，四周礁盘略呈圆形，岛大致呈三角形，长1850米，宽约800米，面积1.5平方公里<ref>[http://www.hnszw.org.cn/data/news/2009/06/43638/ 海南史志网：第一节 西沙群岛的主要岛礁]</ref>。

== 天氣概況 ==
平均气温在31℃以上，且四季海风超4级。

== 生態資源 ==
=== 植物 ===
本来中建岛遍地珊瑚沙砾，基本没有植被，自20世纪七十年代守岛官兵进驻中建岛以来，进行生态改造，运来土壤、植物，进行了大量植树绿化<ref>[http://news.xinhuanet.com/mil/2010-01/14/content_12807004.htm 新華網：走近祖國西沙中建島守島兵]</ref>，使中建岛由“沙洲”变“绿洲”。

=== 動物 ===
每到春末夏初季节，就会有数万只[[大凤头燕鸥|大凤头燕鸥]]到岛上栖息。[http://www.people.com.cn/GB/paper39/3009/406408.html]

== 島上狀況 ==
有[[中国人民解放军海军|中国人民解放军海军]]部队驻守。岛上有中国主权碑。经过疏浚航道，在岛西部建有码头。由[[中国移动|中国移动]]通讯架设移动通讯[[基站|基站]]。
