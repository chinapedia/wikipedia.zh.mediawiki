'''中国法院网'''，正式名称'''中国法院国际互联网站'''，办公地址为[[北京市|北京市]][[东城区_(北京市)|东城区]]东花市北里西区22号，是经[[中华人民共和国最高人民法院|中华人民共和国最高人民法院]]批准成立，经[[国务院新闻办公室|国务院新闻办公室]]批准从事登载新闻业务的综合性新闻网站，并经[[国家广电总局|国家广电总局]]批准自办视频播放业务。中国法院网是世界上最大的法律网站、法律新闻网站<ref name=lianxi>{{Cite web |url = http://www.chinacourt.org/article/detail/2009/04/id/353385.shtml |title = 联系我们 |author =  |date = 2012-02-11 |publisher = 中国法院网 |language =  |accessdate =  |quote =  }}</ref>。

==简介==
中国法院网由[[中华人民共和国最高人民法院|中华人民共和国最高人民法院]]主管，人民法院报社（后改为[[人民法院新闻传媒总社|人民法院新闻传媒总社]]）主办<ref>{{Cite web |url =http://www.chinacourt.org/zhuanti1/fzjc/ |title =中国法院网简介 |author =  |date = |publisher = 中国法院网 |language =  |accessdate = 2017-04-05 |quote =  }}</ref>。 中国法院网在中共最高人民法院[[党组|党组]]领导下，依托人民法院新闻传媒总社开展工作<ref name=lianxi/>。

2002年2月，中国法院网开始试运行。2002年3月14日，推出测试版（一）。2002年3月，山东省青岛市市北区人民法院，[[中华人民共和国青岛海事法院|中华人民共和国青岛海事法院]]，[[西藏自治区高级人民法院|西藏自治区高级人民法院]]成为中国法院网第一批地方频道。2002年4月18日，推出测试版（二）。2002年6月26日，最高人民法院院长办公会议确定由[[人民法院报|人民法院报]]社负责最高人民法院政府网站的技术管理和维护，并主办中国法院网。2002年7月12日，王运声召集会议，听取了中国法院网工作汇报，确定中国法院网正式版运行，并确定了开通日期。2002年8月16日，在北京[[人民大会堂|人民大会堂]]举行中国法院网开通仪式，最高人民法院院长[[肖扬|肖扬]]致电祝贺，最高人民法院党组副书记、常务副院长[[祝铭山|祝铭山]]，副院长[[刘家琛|刘家琛]]、[[曹建明|曹建明]]，政治部主任[[苏泽林|苏泽林]]参加仪式。2002年11月11日，中国法院网接管最高人民法院政府网站技术管理和维护工作。2002年12月，决定中国法院网与人民法院报电子版域名合并。2002年12月19日，中国法院网法治论坛开通<ref>{{Cite web |url =http://www.chinacourt.org/article/detail/2012/8/id/540271.shtml  |title = 2002年中国法院网大事记 |author =  |date = 2012-08-10 |publisher =中国法院网  |language =  |accessdate =  |quote =  }}</ref>。

中国法院网首开网上征集司法解释意见的先河，承担了司法解释和规范性文件的征求意见工作。2003年起，中国法院网开始直播，其中有嘉宾访谈直播，还有庭审直播，包括最高人民法院首次进行的网上庭审直播。2008年8月，中国法院网的全国人大立法及新法规速递专题获中国人大新闻奖<ref name=bfw/>。

2007年1月1日，由中国法院网自主研发的全国法院执行案件信息管理系统在全国正式运行。运行仅一个月，该系统便已收集全国各级法院新收执行案件30万件<ref name=bfw/>。

2009年4月14日，按照最高人民法院“阳光审判”的要求，中国法院网为各地法院开通了专用电子信箱。2009年5月4日，最高人民法院通过中国法院网开通了民意沟通电子邮箱<ref name=bfw/>。

2012年3月1日，全新改版的中国法院网正式上线<ref name=bfw>{{Cite web |url =http://news.enorth.com.cn/system/2012/03/02/008772577.shtml  |title =中国法院网全新改版回眸与展望  |author =  |date = 2012-03-02 |publisher =  北方网|language =  |accessdate =  |quote =  }}</ref>。2014年8月19日，“中国法院网APP”安卓版正式上线<ref>{{Cite web |url =http://www.chinacourt.org/article/detail/2014/08/id/1367465.shtml  |title =  指尖上的法院新闻平台 中国法院网APP正式上线|author =  |date = 2014-08-19 |publisher =中国法院网  |language =  |accessdate =  |quote =  }}</ref>。2014年9月9日，中国法院网新版正式上线。
