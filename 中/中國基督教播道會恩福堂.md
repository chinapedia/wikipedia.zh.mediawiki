'''中國基督教播道會恩福堂'''，（簡稱：'''恩福堂'''或'''恩福'''，英文：'''Evangelical Free Church of China - Yan Fook Church，EFCC Yan Fook Church'''），於1987年9月成立，是[[香港|香港]]著名[[基督教|基督教]][[教會|教會]]，隸屬[[中國基督教播道會|中國基督教播道會]]。 

堂址位於[[香港|香港]][[九龍|九龍]][[長沙灣|長沙灣]][[長沙灣道|長沙灣道]]789號恩福中心，是香港規模最大的基督教會堂之一。1984年，參加的[[信徒|信徒]]大約100人；2004年，崇拜人數已超過5,000人<ref name="history">[http://www.yanfook.org.hk/tc/aboutYF.jsp#2 中國基督教播道會恩福堂（官方網站 ）- 歷史]</ref><ref name="ghs">[http://www.gospelherald.com.hk/news/gen_1023.htm 2007年09月22日基督日報《蘇穎智牧師訪談：恩福成績表》]</ref>；2008年尾，崇拜人數增長至10,000人以上。

== 歷史 ==
恩福堂原是中國基督教播道會窩打老道山福音堂的一個分堂。早於1984年10月，窩打老道山福音堂為了解決當時擁擠的崇拜情況，其中為數約一百名的會眾遷往位於[[香港|香港]][[九龍塘|九龍塘]][[方方小學|方方小學]]的校舍舉行崇拜聚會。1987年9月，恩福堂正式成立。從1987年到2004年4月，恩福堂先後用過十處不同地方聚會。直到2004年5月，搬入位於[[長沙灣|長沙灣]]的自建物業「恩福中心」聚會。

[[麗新集團|麗新集團]]原本計劃於恩福中心現址興建[[長沙灣廣場|長沙灣廣場]]三期，但於經濟低迷下告吹，結果於2000年將地皮出售予恩福堂。

== 基督教信仰 ==
恩福堂根據聖經持守信仰立場包括<ref name="abt">[http://www.yanfook.org.hk/tc/aboutYF.jsp 恩福堂信仰立場]</ref> : 
*聖父、聖子、聖靈三位一體的獨一真神。祂既是萬物的創造者，也是掌管宇宙的主宰。
*神的兒子耶穌基督由聖靈藉童貞女[[聖母瑪利亞|馬利亞]]感孕而生，擁有完全的神人二性。
*因為神的兒子耶穌基督被釘在[[十架|十架]]上流血犧牲，為世人贖罪，所以使人脫離罪的權勢與刑罰。
*死後三天，耶穌基督從死裏復活並且升天，坐在全能的父神的右邊，代信徒祈求。
*現今作為中保及大祭司的[[耶穌|耶穌]]基督是教會的元首。將來祂必定從天再度降臨並建立國度。 祂也是審判活人和死人的審判主宰。
*聖靈在五旬節降臨，是[[三位一體|三位一體]]的第三位，為要榮耀聖父和聖子。祂使罪人知罪悔改，居住在信徒心中並使信徒藉以得勝成聖。
*新舊約66卷[[聖經|聖經]]皆為神所默示的話語，絕對真確可信，是信徒們在信仰上和行為上的最高準則。
*全人類死後都必復活，信主的人有永生，可在[[天堂|天堂]]享永福；不信的人被定[[罪|罪]]，必落火湖永[[死|死]]，直到永遠的。
*[[魔鬼|魔鬼]]乃罪惡的根源，運行在悖逆之人心內，最終必受永刑。

== 組織 ==
除了設立執事會，恩福堂組織架構可分為多個部門 <ref name="dept">[http://www.yanfook.org.hk/tc/aboutYF.jsp#4 恩福堂部門]</ref>：
*敬拜/禱告事工： 敬拜部、禱告部
*傳道/差傳事工： 傳道部、差傳部 、中國事工部、 普通話部
*成長/造就事工： 成人基督教教育部、 組長訓練部、圖書館
*關顧/團契事工： 慈惠部、輔導部、健關部 、社關部、家工部、青少部 、兒童部 、男人事工部 、康體事工部
*部門支援/事工協調： 大眾傳播部 、資訊傳播小組(IT)、禮賓部 、會友部

== 牧養模式 ==
恩福堂同時採用團契和小組的牧養模式<ref name="history"/><ref name="ghg">[http://www.gospelherald.com.hk/news/gen_1024.htm 2007年09月24日基督日報《「恩福」蘇穎智牧師訪談：恩福增長之秘訣》]</ref> 。 
*團契是以年齡和婚姻狀況劃分，目前有長青、成人、伉儷、職青、大專、中學、單親的團契、媽媽團契和已婚團契等 9個組別。
*每個團契由許多小組所組成；而每一小組的成員是固定的。1

== 恩福聖經學院 ==
{{main|恩福聖經學院}}
為了培育信徒領袖、教徒個人素質及屬靈生命，恩福堂於2001年成立[[恩福聖經學院|恩福聖經學院]]<ref name="site">[http://www.yfbi.org 恩福聖經學院 （官方網站）]</ref>。 該學院提供由證書至碩士等不同程度的神學課程，供恩福堂會友和其他教會信徒修讀。

== 爭議 ==
=== 家暴條例 ===
播道會恩福堂牧師[[蘇穎智|蘇穎智]]在《[[家暴條例|家暴條例]]》中發表過一些爭議言論，包括曾公開呼籲信徒集會抗議家暴條例的修訂<ref>[http://www.youtube.com/watch?v=gNkoY6xuNO8 youtube《家庭暴力條例》恩福堂蘇穎智牧師]</ref>，同性戀不應該得到暴力條例的保護，並指出同性同居者會引起很多社會問題。但是為什麼只是[[大學生|大學生]]與[[男性|男性]]會因為家暴條例的修訂成為[[性工作者|性工作者]]和[[性奴|性奴]]，蘇穎智卻沒有交代。

=== 蘇穎智牧師被指公開推介3個建制派議員 ===
有教徒稱蘇穎智牧師在2008年教堂崇拜聚會的講道中要求教徒支持該教會的會友、[[建制派|建制派]]的九龍西選區候選人[[梁美芬|梁美芬]]、會計界候選人[[陳茂波|陳茂波]]和醫學界候選人[[何栢良|何栢良]]，違反公平公開的[[選舉|選舉]]及[[政教分離|政教分離]]精神，宗教勢力干預政治。如果屬實，有關人士即可能觸犯[[法例|法例]]。但是當日何栢良和陳茂波都有出席教堂講道並稱蘇牧師沒要求教徒投票給他們而只為3人代為[[祈禱|祈禱]]，而梁美芬則表示不知情<ref>[http://news.sina.com.hk/cgi-bin/nw/show.cgi/274/3/1/852466/1.html 2008-08-28 明報《恩福堂牧師被指推介3建制派》]</ref>。
