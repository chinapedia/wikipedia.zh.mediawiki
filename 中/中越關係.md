'''中越關係'''（{{vie|v=Quan hệ Trung Quốc – Việt Nam|hn=關係中國－越南|f=y}}）是[[中國|中國]]與[[越南|越南]]現在及過去存在的政權及人民之間的關係。

雖然中國和越南同屬於[[漢字文化圈|漢字文化圈]]，但是中國曾四度征服越南，這段歷史令越南人對中國抱有疑心<ref name="LOC" /><ref>Forbes, Andrew. [http://www.atimes.com/atimes/Southeast_Asia/ID26Ae01.html "Why Vietnam Loves and Hates China"]. Asia Times. 26 April 2007.</ref><ref>'China: The Country to the North'. Forbes, Andrew, and Henley, David (2012). ''Vietnam Past and Present: The North''. Chiang Mai: Cognoscenti Books. ASIN: B006DCCM9Q.</ref>，對中國崛起看法的負面程度僅次於[[日本|日本]]。而當今中國大陸和越南的政權——[[中華人民共和國|中華人民共和國]]與[[越南社會主義共和國|越南社會主義共和國]]縱然都是[[社會主義|社會主義]]國家，不過兩國的雙邊關係比較動盪。中華人民共和國雖然曾經在越戰中由於種種原因而大量支援由[[越南共產黨|越南共產黨]]建立的[[越南民主共和國|越南民主共和國]]（北越），但是兩國關係在1976年[[柬越戰爭|柬越戰爭]]後惡化，使越南倒向蘇聯一方，中國更曾於1979年在中越邊境與越南[[中越戰爭|開戰]]。冷戰結束後，由於經濟改革因素，雙方都開始改善兩國之間的外交關係和促進合作，然而在[[南海|南海]]主權問題方面，兩國仍然時有爭執。

越南民间对中国又爱又恨<ref>[http://news.dwnews.com/global/news/2017-11-22/60025080_all.html 记者手记：越南对中国浓烈的爱恨情仇]</ref><ref>[http://www.infzm.com/content/62315 越南：天堂太远，中国很近]</ref><ref>[http://oversea.huanqiu.com/political/2012-08/3003908.html 《亚洲时报》：越南对华既爱又恨]</ref><ref>[http://www.sohu.com/a/213436808_99925460 越南人爱中国吗? 越南网友: 对中国又爱又恨]</ref>。普遍在政治军事方面反中<ref>[http://www.guancha.cn/PanJinE/2015_11_09_340579.shtml 潘金娥：真实的越南——青年、媒体和腐败]</ref>，文化親中<ref>[http://wemedia.ifeng.com/7530282/wemedia.shtml 一份来自越南迷妹的报告：中国文化在越南的那些真与假]</ref><ref>[http://world.huanqiu.com/exclusive/2016-09/9478723.html “国剧”火到越南 展现中国文化软实力]</ref>。越南的国家政策大量模仿中国，處處有中國的影子<ref>[http://www.yzzk.com/cfm/content_archive.cfm?id=1367894175697&docissue=2005-37 越南處處有中國的影子]</ref>。

==歷史==
===二戰前===
根據越南史籍[[大越史記全書|大越史記全書]]的記載，中國和越南在中國的[[戰國時期|戰國時期]]（越南的[[安陽王|古蜀]]時期）就已經開始來往。越南歷史上曾[[越南北屬時期|被中國統治過四次]]，从[[西汉|西汉]]直到[[五代十国|五代十国]]时期长达一千多年，[[明朝|明朝]]15世纪初期也被中国直接统治二十年。但是越南人先後以武力推翻中國人對他們的統治，並且以向[[中国皇帝|中国皇帝]]称臣纳贡的方式維持自己作為一個[[藩屬國|藩屬國]]的獨立地位。

在1884年，越南正值[[阮朝|阮朝]]時期。當時統治中國的[[清朝|清朝]]和[[法兰西第三共和国|法國]]爆發[[中法戰爭|中法戰爭]]，最終清朝的[[李鴻章|李鴻章]]與法國簽訂新約，承認法國對越南的宗主權。自此，中國便失去了對越南的影響力。

===冷戰===
{{See also|中国－南越关系}}
在歷史上，中越關係的形態是越南人擔心中國會再次支配他們<ref name="LOC">{{cite web|url=http://lcweb2.loc.gov/cgi-bin/query/r?frd/cstdy:@field(DOCID+vn0111)|title=Vietnam - China|accessdate=2008年6月3日|publisher=[[美國|美國]][[國會圖書館|國會圖書館]]}}</ref>。和[[蘇聯|蘇聯]]一樣，中國是奉行共產主義的[[北越|北越]]的一個重要的戰略性盟友。中華人民共和國在1949年成立以後，在1950年1月18日與北越建交<ref name="Sina">{{cite web|url=http://english.sina.com/1/2005/1028/51407.html|title=China-Vietnam Bilateral Relations |accessdate=2008年6月3日|publisher=新浪網|date=2005年10月28日}}</ref>，並且在1950年至1975年期間向北越提供武器、軍訓和補給品去抵抗[[南越|南越]]和與之結盟的[[美國|美國]]。<ref name="人民網">{{cite web|url=http://politics.people.com.cn/GB/8198/137122/137123/8239498.html|title=中越双边关系|accessdate=2011年6月25日|publisher=人民網|date=2007年6月29日}}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>不過越南共產黨仍然懷疑中國有支配越南的希望<ref name="LOC" />。

中越關係是[[中蘇交惡|中蘇交惡]]的原因之一。[[北部湾事件|北部湾事件]]以後，[[鄧小平|鄧小平]]曾秘密地向北越承諾如果他們放棄所有蘇聯的援助，中華人民共和國將援助北越10億[[人民幣|人民幣]]。

===中越戰爭===
[[File:Quân_đội_duyệt_binh_ở_Trường_Sa.JPG|thumb]]某一島嶼駐紮的越南士兵]]
{{See also|中越戰爭}}

越戰期間，越南民主共和国和中華人民共和國同意延遲解決雙邊的領土問題（包括北部湾的邊界問題和[[南海|南海]]上[[西沙群島|西沙群島]]和[[南沙群島|南沙群島]]主權爭議），直至越南实现统一為止。<ref name="LOC"/>[[北部湾|北部湾]]蘊藏的石油，引起了中越兩國在海上的衝突。在1973年，北越宣布容許外資在南海開採石油，於是中華人民共和國在1974年派軍隊[[西沙之戰|收复當時由南越軍方控制的西沙群島]]<ref name="LOC" />。南越在1975年滅亡以後，越南軍方控制了双方存有主權争议的[[南沙群島|南沙群島]]的多数岛礁<ref name="LOC" />。

1975年4月北越推翻南越政权，至1976年{{vi-link|越南共产党第四次全国代表大会|Đại hội Đảng Cộng sản Việt Nam IV}}后，越南确立了以[[黎笋|黎笋]]为总书记的领导体制和亲[[苏联|苏联]]的政治路线。1978年，越南和苏联签署《越苏友好合作互助条约》，并与中華人民共和國就在越[[华侨|华侨]]问题发生纠纷。6月，中国政府发表「关于越南政府驱逐华侨的声明」<ref name="寒山碧">寒山碧原著，伊藤潔縮譯，唐建宇、李明翻譯：《鄧小平傳》，香港：東西文化事業公司，1993年1月</ref>{{rp|195}}。同年底，越南[[柬越战争|侵入柬埔寨]]，推翻了亲中的[[红色高棉|红色高棉]]政权。这一时期，中華人民共和國一方面斷絕對越南的經濟援助<ref name="LOC" /><ref name="KF1" />，一方面谴责越南当局追随苏联的“[[霸权主义|霸权主义]]”而推行“地区霸权主义”，号召组织世界范围内的“反霸国际[[统一战线|统一战线]]”与之斗争<ref>广西社会科学院印支研究所：越南地区霸权主义问题(1984)</ref>{{rp|105－113}}；越南方面则批判中国从事[[扩张主义|扩张主义]]，指中國有意侵佔越南的領土，並有着稱霸[[亞洲|亞洲]]的思想<ref>{{cite book|author=林濤|title=世界華文報告文學獎|publisher=明報|location=香港|date=2001|chapter=南海潮—難民潮}}</ref>。越南出兵柬埔寨是中越关系在1970年代末期至1980年代末期恶化的主要原因<ref>{{cite web |url = http://military.china.com/zh_cn/history4/62/20100920/16155077.html |title = 1990年代中越關係正常化始末：越南人主動服軟 |date = 2010年9月20日 |publisher = 中國網 |pages = 1 |accessdate = 2011年8月17日 |archive-url = https://web.archive.org/web/20100923094652/http://military.china.com/zh_cn/history4/62/20100920/16155077.html |archive-date = 2010年9月23日 |dead-url = yes }}</ref>。

在1979年2月17日，中國人民解放軍在中越邊境上與越南爆發[[中越戰爭|中越戰爭]]，在兩個星期的戰鬥過後，解放軍一度逼近[[河內|河內]]，不過解放軍於同年3月5日撤出中越邊境。<ref name="LOC"/><ref name="GS">{{cite web |url = http://www.globalsecurity.org/military/world/war/prc-vietnam.htm |title = Chinese invasion of Vietnam |accessdate = 2008年6月3日 |publisher = Global Security.org}}</ref>雙方在這次戰役中傷亡慘重，解放軍的傷亡人員達4萬名，越軍的傷亡人員則超過10萬名。<ref name="GS"/> 中越之間持續的和談在1979年12月破裂，而中越雙方都開始鞏固邊防。越南在中越邊境要點建立大小不一的“公安屯”<ref>{{cite web|url=http://military.china.com/zh_cn/critical3/27/20050411/12232165_1.html|title=再论关于1979年对越自卫反击战的几个焦点问题|date=2005年4月11日|publisher=中华網|pages=1|accessdate=2011年8月17日|deadurl=yes|archiveurl=https://web.archive.org/web/20090208025509/http://military.china.com/zh_cn/critical3/27/20050411/12232165_1.html|archivedate=2009年2月8日}}</ref>，並派出60萬名軍人和民兵駐守邊境；中華人民共和國則派出40萬名軍人駐守中方邊境。<ref name="GS" />在1980年代，中越雙方在邊境上有著零星的衝突，中華人民共和國要求越南无条件撤出柬埔寨作为两国重启谈判的先决条件。<ref name="LOC" /><ref name="GS" />1988年3月，中越在南沙群岛赤瓜礁发生[[赤瓜礁海战|海上冲突]]，中方击沉2艘，击伤1艘越南海军舰艇，并在南沙群岛的6个礁盘上建立了据点<ref>{{cite web|url=http://www.sznews.com/culture/content/2011-07/25/content_5870837.htm|title=28分钟结束战斗 揭秘1988年中越赤瓜礁海战|accessdate=2011年8月16日|publisher=深圳新闻网|deadurl=yes|archiveurl=https://web.archive.org/web/20150513103235/http://www.sznews.com/culture/content/2011-07/25/content_5870837.htm|archivedate=2015年5月13日|df=}}</ref>。整个1980年代，中華人民共和國一直請求蘇聯出面使越南撤出柬埔寨，并指出苏联帮助越南侵占柬埔寨是改善[[中俄關係|中苏关系]]的其中一個障碍。苏联方面起初拒绝，至1988年底，苏共中央总书记[[戈尔巴乔夫|戈尔巴乔夫]]接见[[钱其琛|钱其琛]]时表示愿意早日解决柬埔寨问题。1989年1月，越南宣布将在柬埔寨政治解决协议制定后，不迟于1989年9月底前从柬埔寨撤军完毕<ref>钱其琛：《外交十记》 第一章,我亲历的中苏关系正常化 ISBN 7501221316</ref>。

===雙邊關係改善===
隨著1990年代初[[蘇聯解體|蘇聯解體]]和越南撤出柬埔寨，中越雙方關係開始有改善。雙方都計劃在1990年9月在[[成都|成都]]舉行的一次秘密峰會上達致關係正常化，最終兩國於1991年達成外交關係正常化<ref name="KF1" />。自1991年起，雙方的領導人及高級官員都進行了互訪。中越兩國都承認和支持柬埔寨在1991年以後成立的政府和支持兩國進入[[世界貿易組織|世界貿易組織]]的申請<ref name="KF1" />。在1999年，[[越南共產黨|越共]][[越南共產黨中央委員會總書記|中央總書記]][[黎可漂|黎可漂]]訪華，在北京會見了[[中國共產黨|中共]][[中國共產黨中央委員會總書記|中央總書記]]兼[[中華人民共和國|中華人民共和國]][[中華人民共和國主席|主席]][[江澤民|江澤民]]。他宣布了一個改善並強化中越關係的“16字方針”；一個關於中越雙方共同合作的聯合聲明在2000年簽訂<ref name="KF1" />。中越兩國也成功在1999年至2000年期間解決關於[[北部湾|北部湾]]地區的邊界和領海主權問題<ref name="Sina" /><ref name="KF1" />。一個在2002年由中國和東盟簽訂的聯合協議標示了一個維持地區和平和承諾反對武力衝突的進程<ref name="KF1"/>。在2002年，江澤民出訪越南，並與越南簽訂了很多協議去深化雙邊的經貿合作和化解雙方比較突出的矛盾<ref name="Sina" />。

==商貿往來==
中越兩國自從在1991年恢復了雙邊的貿易往來之後，雙方貿易總額由1991年的3200萬[[美元|美元]]增長至2004年的約72億美元<ref name="AT">{{cite web |url=http://www.atimes.com/atimes/Southeast_Asia/GG21Ae01.html |title=China, Vietnam find love |accessdate=2008年6月3日|publisher=亞洲時報|date=2005年7月21日}}</ref>。兩國政府已制定了目標，務求雙方貿易總額於2010年增長至100億美元<ref name="AT"/>。越南向中華人民共和國出口[[原油|原油]]、煤炭、咖啡和食物；而中華人民共和國則向越南出口藥品、機器、[[石油|石油]]、肥料和汽車零件。中華人民共和國已成為越南第二大貿易夥伴和最大的入口產品的來源地<ref name="Sina" /><ref name="AT" />。兩國正成立一個起於中國[[雲南省|雲南省]]，終於越南北部的省份和城市的“經濟走廊”以及在北部湾地區成立類似的經濟特區，把[[廣西壯族自治區|廣西壯族自治區]]的[[南寧|南寧]]與越南的[[諒山省|諒山省]]、[[河內|河內]]、[[海防市|海防市]]和[[廣寧省|廣寧省]]連接起來。<ref name="AT"/>連接兩國的鐵路、航空和海路交通現已開通，另外兩國在邊境省份一共設有7對陸路口岸。<ref name="Sina" />雙方也成立了合資企業，例如位於越南[[太原省|太原省]]的綜合煉鋼廠，該廠生產了很多鋼鐵製品<ref name="AT" />。

==边界领土问题==
1999年12月30日，中越双方在[[河内|河内]]签署了《中华人民共和国和越南社会主义共和国陆地边界条约》<ref>{{cite web |url = http://www.npc.gov.cn/wxzl/gongbao/2000-12/17/content_5008962.htm |title = 中华人民共和国和越南社会主义共和国陆地边界条约 |accessdate = 2010年8月15日 |publisher = 中国人大网 |date = 2000年12月17日}}</ref>，确认了以1887年[[中法界约|中法界约]]等历史边界条约为基础，双方宣称本着平等协商，互谅互让的友好精神解决陆地边界问题。2000年，中越两国各自国内完成了批准边界条约的法律程序。2001年，由两国派出代表成立了“中越陆地边界联合勘界委员会”，负责实地勘界立碑的工作，然而该项工作长期进展缓慢，进度落后于预期，直至2009年，最终完成中越陆地边界全线勘界立碑工作，双方随后签署了《中越陆地边界勘界议定书》<ref>{{cite web |url = http://news.sohu.com/20091118/n268310920.shtml|title = 中越签署陆地边界勘界文件 愿共同维护南海和平 |accessdate = 2011年8月15日 |publisher = 搜狐新闻 |date = 2009年11月28日}}</ref>以及边界与口岸管理制度等文件。上述文件于2010年7月14日生效<ref>{{cite web |url = http://news.xinhuanet.com/world/2010-07/14/c_111952218.htm |title = 中越举行陆地边界勘界议定书等法律文件生效仪式 |accessdate = 2011年8月15日 |publisher = 新华网 |date = 2010年7月14日}}</ref>，至此，中越两国已经解决了陆地边界争议问题。

2000年12月25日，中越双方在[[北京|北京]]签署了《中华人民共和国和越南社会主义共和国关于两国在北部湾领海、专属经济区和大陆架的划界协定》，在北部湾地区确立了21个海上分界点，双方宣称本着公平原则解决了在北部湾地区的领海、专属经济区和大陆架划分<ref>{{cite web |url = http://www.npc.gov.cn/wxzl/gongbao/2004-08/04/content_5332197.htm |title = 中华人民共和国和越南社会主义共和国关于两国在北部湾领海、专属经济区和大陆架的划界协定 |accessdate = 2011年8月15日 |publisher = 中国人大网 |date = 2004年8月4日}}</ref>，大致上各自获得了50%的北部湾海域。此外，还签署了《中华人民共和国政府和越南社会主义共和国政府北部湾渔业合作协定》，跨双方管辖海域划定了一块“渔业共同作业区”，建立了联合巡逻等制度。<ref>{{cite web |url = http://www.fmprc.gov.cn/chn/gxh/zlb/tyfg/t145558.htm |accessdate=2011年8月15日|title=中越北部湾划界协定情况介绍|publisher=中华人民共和国外交部网站|date=2000年12月29日}}</ref>至2004年，双方完成了上述协定的各自国内法定批准程序，2004年6月30起，上述协定生效<ref>{{cite web|url=|title=中越北部湾划界协定和渔业合作协定今日生效|accessdate=2011年8月15日|publisher=搜狐新闻|date=2004年6月30日}}</ref>，两国关于北部湾海域的争议基本宣告解决。

在西沙群岛和南沙群岛的主权争议方面，双方和平解决争端的外交进展缓慢。2002年，中華人民共和國和包括越南在内的[[东盟|东盟]]十国签署了“[[南海各方行为宣言|南海各方行为宣言]]”，宣言承诺争议各方将根据《[[联合国海洋法公约|联合国海洋法公约]]》等公认的国际法准则，寻求和平解决争议之道，并在争端解决前保持克制，避免问题复杂化。<ref>{{cite web |url = http://www.fmprc.gov.cn/chn/pds/wjb/zzjg/yzs/dqzz/nanhai/t848051.htm |title = 南海各方行为宣言 |accessdate = 2011年8月15日 |publisher = 中华人民共和国外交部网站 |date = 2004年11月6日}}</ref>但上述宣言并非国际法意义上的条约，不具有法律约束效力<ref>{{cite web |url = http://www.fmprc.gov.cn/chn/gxh/tyb/fyrbt/jzhsl/t830587.htm |title = 2011年6月14日外交部发言人洪磊举行例行记者会 |accessdate = 2011年8月15日 |publisher = 中华人民共和国外交部网站 |date = 2011年6月14日}}</ref>。
