'''中国园林博物馆'''，位于[[北京市|北京市]][[丰台区|丰台区]]射击场路15号，是中国首座以[[中国园林|园林]]为主题的国家级博物馆。

==简介==
[[File:The_Grand_Hall_in_Beijing_Garden_Museum.jpg|thumb]]》局部为原型，利用墙壁栽种绿植营造的垂直绿化景观，长28.4米，高9.8米，使用了[[黄金葛|黄金葛]]、[[黑金刚|黑金刚]]、[[玉龙草|玉龙草]]等十余种近两万株植物<ref>{{Cite web |url =  http://news.yuanlin.com/detail/2013916/159024.htm|title =李国强：中国园林博览馆前程似锦植物墙设计与建造  |author =  |date =  2013-09-16|publisher = 中国园林网 |language =  |accessdate =  |quote =  }}</ref>。]]
中国园林博物馆位于[[丰台区|丰台区]]鹰山脚下，地处[[永定河|永定河]]畔。[[北京市公园管理中心|北京市公园管理中心]]负责该博物馆的建设、运营、管理。2010年该馆开始筹建，2013年5月开馆。开馆时逢[[中国国际园林博览会|中国国际园林博览会]]在丰台区举办。该馆占地面积达到6.5万平方米，建筑面积达到49950平方米，由主体建筑、室内展园、室外展区组成。主体建筑内有28200平方米面积可供展览使用。该馆为公益性文化机构。<ref>[http://www.gardensmuseum.cn/cn/about.aspx 园博馆简介，中国园林博物馆，于2014-12-14查阅]</ref><ref>[http://bjwb.bjd.com.cn/html/2012-09/12/content_136822.htm 北京将打造博物馆中心区，北京晚报，2012-09-12] {{webarchive|url=https://archive.is/20141214153026/http://bjwb.bjd.com.cn/html/2012-09/12/content_136822.htm |date=2014-12-14 }}</ref>

2014年3月，[[北京市文物局|北京市文物局]]依法按程序准予中国园林博物馆注册登记，中国园林博物馆正式成为在北京市文物局注册登记的博物馆。<ref>[http://www.bjww.gov.cn/2014/12-19/1418953279578.html 北京市文物局2014年一季度文博事业大事记，北京文博，2014-12-09]</ref>
