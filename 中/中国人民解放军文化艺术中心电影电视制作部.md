'''中国人民解放军文化艺术中心电影电视制作部'''，厂标为'''中国人民解放军八一电影制片厂'''，简称'''八一电影制片厂'''，位于[[北京市|北京市]][[丰台区|丰台区]][[广安门|广安门]]外[[六里桥_(立交桥)|六里桥]]北里甲1号，隶属[[中国人民解放军文化艺术中心|中国人民解放军文化艺术中心]]。

==沿革==
1951年3月，[[中央人民政府人民革命军事委员会|中央人民政府人民革命军事委员会]][[中国人民解放军总政治部|总政治部]]决定在军队序列内成立电影制片厂，经[[中央人民政府文化教育委员会|中央人民政府文化教育委员会]]审议批准，暂名为'''中央人民政府人民革命军事委员会总政治部军事教育电影制片厂'''。建厂筹备委员会的主任委员为总政治部文化部部长[[陈沂_(少将)|陈沂]]，副主任委员是[[北京电影制片厂|北京电影制片厂]]副厂长[[汪洋_(北影)|汪洋]]、训练总监部训练局局长[[华雷|华雷]]、总政治部文化部电影处副处长[[唐瑜为|唐瑜为]]。筹委会下设筹备处，[[胡介民|胡介民]]为秘书长，[[夏国瑛|夏国瑛]]、[[王苹|王苹]]为副秘书长。<ref name=honghuo/>

1952年8月1日，经总政治部批准，'''中央人民政府革命军事委员会总政治部解放军电影制片厂'''在北京正式建厂。厂长是从[[西南军区|西南军区]]政治部文化部副部长任上调来的[[陈播|陈播]]，第二厂长是[[司徒慧敏|司徒慧敏]]。<ref name=honghuo/>成立之初，仅摄制军事教育片和纪录片。1955年起，开始摄制故事片。<ref name=zgw>[http://www.china.com.cn/chinese/2005/dybn/947344.htm 中国电影大事记，中国网，2005-08-22]</ref>

1956年1月，经总政治部批准，更名为'''中国人民解放军总政治部八一电影制片厂'''。1965年12月22日，[[中央军委|中央军委]]决定，将该厂改名为“'''中国人民解放军总政治部电影制片局'''”，该局对外影片字幕仍用“八一电影制片厂”。<ref>[http://www.cnfm.org.cn/2009-04/15/cms6432article.shtml 中国电影大事记1965，中国电影博物馆，2009-04-15]</ref>1968年9月16日，更名为'''中国人民解放军八一电影制片厂'''。<ref name=honghuo>莎羽，红祸——彭勃结怨江青始末，人民日报出版社，2006年</ref>

1968年4月24日，该厂成立革命委员会。1973年秋，[[叶剑英|叶剑英]]主持[[中央军委|中央军委]]的工作，与[[李德生|李德生]]商量撤消八一厂的革命委员会，恢复为文革前的厂长制。但是三团两厂（[[钱浩亮|钱浩亮]]掌管的京剧团、[[刘庆棠|刘庆棠]]掌管的歌舞团、[[中央乐团|中央乐团]]及[[北京电影制片厂|北京电影制片厂]]、八一电影制片厂）是[[江青|江青]]的“试验田”，军队管不了八一厂。所以，叶剑英、李德生交代革命委员会主任[[彭勃|彭勃]]给[[周恩来|周恩来]]总理写报告，建议将八一厂革命委员会改为厂长制。周恩来在彭勃的报告上批示“照办”。根据事前准备，收到批示后，彭勃当即召开全厂大会，任命[[王心刚|王心刚]]为厂长，彭勃为党委书记兼政委。<ref name=honghuo/>

在[[深化国防和军队改革|深化国防和军队改革]]中，2016年1月中国人民解放军总政治部改为[[中央军委政治工作部|中央军委政治工作部]]，八一电影制片厂转隶中央军委政治工作部，业务由[[中央军委政治工作部宣传局|中央军委政治工作部宣传局]]文化处指导。<ref>[http://www.81.cn/2016czsl80s/2016-06/22/content_7113961.htm 长征题材影片《第四道封锁线》在江西紧张拍摄，中国军网，2016-06-22]</ref>

2018年，在[[深化国防和军队改革|深化国防和军队改革]]中，原八一电影制片厂、原总政歌舞团、原总政歌剧团、原总政话剧团、原总政军乐团合并成立[[中国人民解放军文化艺术中心|中国人民解放军文化艺术中心]]，其中原八一电影制片厂更名为“'''中国人民解放军文化艺术中心电影电视制作部'''”，并由副军级编制降为正师级编制，但仍保留八一电影制片厂厂标。合并调整后，原八一电影制片厂的七部一处（政治部、生产部、技术部、文学部、故事片部、军事片部、电视部、行政管理处）全部撤销；原故事片部的导演室、摄影室、美术室不变。原八一电影制片厂八大车间（录音车间、剪辑车间（数字工作站）、洗印车间、特技烟火车间、照明车间、置景车间、化服道车间、动力车间）有留、有撤、有合，其中化服道车间与置景车间合并<ref>{{Cite web |url = http://news.sina.com.cn/c/nd/2018-02-02/doc-ifyrcsrw7757091.shtml |title =八一电影制片厂被曝遭裁撤 曾拍摄芳华士兵突击等  |author =  |date =2018-02-02  |publisher =新浪  |language =  |accessdate =  |quote =  }}</ref><ref name=changbiao>{{Cite web |url =http://3g.163.com/dy/article/DKQRNFE7051780GA.html  |title =“解放军文化艺术中心电影电视制作部”成立 八一厂厂标不会摘  |author =  |date =2018-06-21  |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = https://www.thepaper.cn/newsDetail_forward_2341395 |title = 原八一电影制片厂开始以新名称公开亮相 |author =  |date = 2018-08-12  |publisher =澎湃新闻  |language =  |accessdate =  |quote =  }}</ref>。
