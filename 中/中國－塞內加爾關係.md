'''中國－塞內加爾关系'''，是指歷史上的[[中國歷史|中國]]和[[塞內加爾歷史|塞內加爾]]（包括現在[[中華人民共和國|中華人民共和國]]與[[塞內加爾共和國|塞內加爾共和國]]）之間的雙邊關係。中華人民共和國與塞內加爾在1971年12月7日建立外交關係，雖然兩國的邦交曾在1996年中斷，但已在2005年恢復；目前，中國和塞內加爾之間保持經貿和文化上的往來。中国大陸在2013年是塞内加尔最大的進口來源地。

== 歷史 ==
=== 古代 ===
[[清朝|清朝]]時已有[[華工|華工]]到塞内加尔工作：1858年，有173名华工來到塞内加尔，他們在當地成為了各式各樣的[[建筑工人|建筑工人]]；此外，法国在1880年至1882年於中國[[广州|广州]]、[[汕头|汕头]]、[[上海|上海]]等地招募了一批华工，他們部分人被运到塞内加尔，負責修筑[[铁路|铁路]]<ref name=workers>{{cite web |title=塞内加尔 |url=http://www.hqbwy.org.cn/heritage_show.asp?ArticleID=120 |website=华侨博物院 |accessdate=2015-10-10 |archiveurl=https://archive.is/20151010013331/http://www.hqbwy.org.cn/heritage_show.asp?ArticleID=120 |archivedate=2015-10-10 |deadurl=no |date=2008-04-12 }}</ref>。這些塞内加尔华工工作繁重，還受當地[[衛生|衛生]]和[[戰爭|戰爭]]的問題困擾<ref name=workers/>。

=== 現代 ===
塞內加爾在1959年與[[馬利|馬利]]（當時名為「蘇丹」）組成[[馬里聯邦|馬里聯邦]]，其後在一年後退出馬里聯邦，獨立為國<ref>{{cite web |title=塞內加爾國家概況 |url=http://big5.chinanews.com/gj/zlk/2014/01-16/301.shtml |website=中新網 |accessdate=2015-10-10 |archiveurl=https://archive.is/20151010014849/http://big5.chinanews.com/gj/zlk/2014/01-16/301.shtml |archivedate=2015-10-10 |deadurl=yes }}</ref>。

塞內加爾在獨立之初已承認中華人民共和國，該國把中華人民共和國統治[[中國大陸|中國大陸]]、[[中華民國|中華民國]]退到[[台灣|台灣]]的情況視為[[国家分裂|国家分裂]]，被認為是「[[兩個中國|兩個中國]]」主張的支持者<ref name=phu>{{cite conference|author=王文隆|title=中華民國退出聯合國後駐外機構的轉型（1971- 1979）|conference=外交檔案的利用與典藏研討會|date=2008-02-23|url=http://archives.sinica.edu.tw/old/conferences/f080223/pdf/2-3.pdf|accessdate=2015-10-10|archiveurl=https://web.archive.org/web/20160602082643/http://archives.sinica.edu.tw/old/conferences/f080223/pdf/2-3.pdf|archivedate=2016-06-02|deadurl=yes}}</ref>{{Rp|28}}。當時，塞內加爾與中華人民共和國洽商建立外交關係，但又不願意驅逐中華民國在當地的代表，又希望中華人民共和國不要在該國首都[[達喀爾|達喀爾]]設使館，由中華人民共和國駐外大使兼任駐塞內加爾代表；中華人民共和國認為塞內加爾需先驅逐當地的中華民國代表，兩國建交過程出現阻礙<ref name=phu/>{{Rp|28}}。其後，美國總統[[尼克遜|尼克遜]]即將[[1972年尼克遜訪華|訪問中華人民共和國]]的消息傳出，[[塞內加爾總統|塞內加爾總統]][[利奧波德·塞達爾·桑戈爾|利奧波德·塞達爾·桑戈爾]]便表示[[聯合國|聯合國]]需要中華人民共和國參與，但反對把中華民國排除在外；中華人民共和國派員與桑戈爾接洽，呈上[[中華人民共和國總理|中華人民共和國總理]][[周恩來|周恩來]]的親筆信，而中華民國方面亦動員[[科特迪瓦|科特迪瓦]]、[[馬達加斯加|馬達加斯加]]和[[扎伊爾|扎伊爾]]的總統去信塞內加爾，遊說對方不要承認中華人民共和國<ref name=phu/>{{Rp|28}}。

但是，塞內加爾最終支持[[聯合國大會2758號決議|聯合國大會2758號決議]]，贊成由中華人民共和國取代中華民國的聯合國席位，又宣布承認中華人民共和國是[[中國|中國]]唯一的合法政府；同時，桑戈爾表示承認中華民國是獨立[[主權國家|主權國家]]、並希望繼續與之保持友好關係，他請求中華民國不要撤回派到塞內加爾的農耕隊，又建議中華民國把[[國號|國號]]更改為「台灣」，後者遭對方拒絕<ref name=phu/>{{Rp|28–29}}。後來，塞內加爾在1971年12月7日與中華人民共和國建交<ref>{{cite book|author1=黃剛|year=2009|title=國際事務論述集: 漢賊不兩立、開羅宣言及聯合國中國代表權等之再議}}</ref>{{Rp|50}}。

1995年，塞内加尔面臨一場经济危机，政局出现不稳，當地人的生活水平亦下降；中華人民共和國便對塞内加尔施以援助，協助當地盖楼房，派出医疗队到當地，又與該國进行农业合作，但都未能有效緩解塞内加尔的经济問題<ref name=broke>{{cite book|author1=王嵎生|author2=杨健|title=资深外交官看世界|publisher=上海三联书店|year=2005|isbn=9787542622440}}</ref>{{Rp|148}}。这时，中華民國政府派員喬裝為商人，游说包括總統{{link-en|阿卜杜·迪烏夫|Abdou Diouf}}在內的塞内加尔政府高層，並與迪烏夫的夫人接觸，邀请她來台灣吃喝玩樂，又給她餽贈礼品；她便向迪烏夫傳遞中華民國的信息，聲稱只要塞内加尔與中華民國復交，便可獲得3,000万至6,000万[[美元|美元]]的酬金<ref name=broke/>{{Rp|148}}。結果，塞内加尔在1996年和中華民國建交，並與中華人民共和國斷交<ref>{{cite book|author1=姜殿铭 (編)|author2=萧敬|author3=曹治洲|author4=修春萍|title=台湾一九九六|publisher=九洲图书出版社|year=1997|isbn=9787801141972}}</ref>{{Rp|457}}。

2005年10月，塞內加爾與中華人民共和國簽署聯合公報，恢復邦交，並互相派駐大使；塞內加爾總統[[阿卜杜拉耶·瓦德|阿卜杜拉耶·瓦德]]為此致函[[中華民國總統|中華民國總統]][[陳水扁|陳水扁]]，表示以後只會與台灣維持經商和文化上的聯繫，又在信中明確表示國家之間沒有[[朋友|朋友]]，只會有[[利益|利益]]<ref name=letter>{{cite news |author1=楊孟瑜 |title=兩岸再掀外交戰　台灣失掉塞內加爾 |url=http://news.bbc.co.uk/chinese/trad/hi/newsid_4370000/newsid_4376800/4376896.stm |accessdate=2015-10-10 |agency=BBC中文網 |date=2005-10-25 |archiveurl=https://archive.is/20151010102921/http://news.bbc.co.uk/chinese/trad/hi/newsid_4370000/newsid_4376800/4376896.stm |archivedate=2015-10-10 |deadurl=no }}</ref>。中華民國方面認為，塞內加爾此舉是因為中華人民共和國的利誘和施壓，又批評中華人民共和國「蠱惑利誘」當地總統及野心政客「外交轉向」<ref name=letter/>。

== 文化關係 ==
中國的[[辽宁大学|辽宁大学]]與塞内加尔的[[达喀尔大学|达喀尔大学]]於2011年3月下旬签訂协议，在达喀尔大学合作設立一所[[孔子学院|孔子学院]]<ref>{{cite news |author1=方亮 |title=辽宁大学与塞内加尔达喀尔大学共建孔子学院 |url=http://www.jyb.cn/world/zwyj/201103/t20110329_422451.html |accessdate=2015-10-10 |agency=辽宁日报 |date=2011-03-29 |archiveurl=https://archive.is/20151010123034/http://www.jyb.cn/world/zwyj/201103/t20110329_422451.html |archivedate=2015-10-10 |deadurl=no }}</ref>；除了孔子学院外，塞内加尔部分大学亦有提供[[漢語|漢語]]教學，課程受到當地學生歡迎，有畢業生畢業後為[[中國中央電視台|中國中央電視台]]工作<ref>{{cite news |title=汉语教学在塞内加尔大学受欢迎 |url=http://www.hanban.edu.cn/article/2014-04/30/content_534480.htm |accessdate=2015-10-10 |agency=国际在线 |date=2014-04-30 |archiveurl=https://archive.is/20151010124635/http://www.hanban.edu.cn/article/2014-04/30/content_534480.htm |archivedate=2015-10-10 |deadurl=no }}</ref>。中華人民共和國亦為會塞内加尔的學生提供[[獎學金|獎學金]]：在2006至2009[[学年|学年]]，共有70名塞内加尔學生接受中國政府的獎學金，並赴華[[留學|留學]]<ref>{{cite web |title=教育交流 |url=http://sn.chineseembassy.org/chn/zsgx/jyjl/ |website=中华人民共和国驻塞内加尔共和国大使馆 |accessdate=2015-10-10 |archiveurl=https://archive.is/20151010125745/http://sn.chineseembassy.org/chn/zsgx/jyjl/ |archivedate=2015-10-10 |deadurl=no |date=2009-07-29 }}</ref>。
