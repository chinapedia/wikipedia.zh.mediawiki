'''中国人民解放军第一一八医院'''，位于[[浙江省|浙江省]][[温州市|温州市]][[鹿城区|鹿城区]]康乐坊嘉福寺巷15号，隶属[[中国人民解放军东部战区|中国人民解放军东部战区]]，是综合性三级乙等医院。

==简介==
1944年1月，在[[江苏省|江苏省]][[涟水县|涟水县]]创建'''华东军区国际和平医院'''<ref name=zhuoy/><ref name=jj>{{Cite web |url =http://111.1.253.70:81/index.php?m=content&c=index&a=lists&catid=2  |title = 医院简介 |author =  |date =  |publisher =中国人民解放军第一一八医院  |language =  |accessdate = 2017-06-25 |quote =  }}</ref>。医院自成立后，先后经历[[涟水战役|涟水战役]]、[[鲁南战役|鲁南战役]]、[[孟良崮战役|孟良崮战役]]、[[济南战役|济南战役]]、[[淮海战役|淮海战役]]、[[渡江战役|渡江战役]]、[[上海战役|上海战役]]、[[一江山岛战役|一江山岛战役]]等战役<ref name=zhuoy/><ref name=chuantong/>。1950年10月19日[[中国人民志愿军|中国人民志愿军]]跨过[[鸭绿江|鸭绿江]]参加[[抗美援朝战争|抗美援朝战争]]时，医院正在浙江[[嘉兴|嘉兴]]整休，1950年11月20日夜从[[集安|集安]]跨过鸭绿江参战<ref name=chuantong>{{Cite web |url =http://db.81.cn/content/2016-08/02/content_7187168.htm  |title =解放军第118医院用优良传统打造强院之路  |author =  |date = 2016-08-02 |publisher =军报记者  |language =  |accessdate =  |quote =  }}</ref>。1951年7月扩编建成'''[[中国人民志愿军|中国人民志愿军]]第三十三兵站医院'''，1953年12月归中国人民志愿军第四分部领导<ref name=zjsjsz/>。1955年1月从[[抗美援朝战争|朝鲜战场]]回国<ref name=jj/>。1954年12月，医院改称'''中国人民解放军第十七预备医院'''，先进驻浙江[[宁波|宁波]]，后来移驻浙江温州，1955年初划归[[中国人民解放军浙江省军区|浙江军区]]领导。1960年6月，改称'''中国人民解放军第十七野战医院'''。1963年12月改为驻军医院，番号为'''中国人民解放军第一一八医院'''。1986年11月，划归[[中国人民解放军南京军区|中国人民解放军南京军区]]后勤部建制领导<ref name=zjsjsz> {{Cite book | author =浙江省军事志编纂委员会编  | title = 浙江省军事志 | location =  | publisher =方志出版社  | date = 1999年 | pages =  | ISBN =  | accessdate =  | url =  | language =  | quote = 第三篇 驻军 第三章 中华人民共和国成立后驻军 第五节 医院(疗养院) }} </ref><ref name=zhuoy/>。2000年代，该医院曾为[[中国人民解放军第一一三医院|中国人民解放军第一一三医院]]温州分院<ref name=wkj>{{Cite web |url =http://news.sina.com.cn/c/2004-07-21/07433148954s.shtml  |title =南京军区113医院温州分院院长王开鉴勇唱“ 黑脸”  |author =  |date =  2004-07-21|publisher = 新浪 |language =  |accessdate =  |quote =  }}</ref>。后又独立。

2016年，在[[深化国防和军队改革|深化国防和军队改革]]中，转隶[[中国人民解放军东部战区|中国人民解放军东部战区]]<ref name=chuantong/><ref>{{Cite web |url =http://military.cnr.cn/zgjq/20170509/t20170509_523746044_2.html |title = 东部战区某通信站加强内部关系提高部队凝聚力 |author =  |date = 2017-05-09 |publisher = 央广军事 |language =  |accessdate =  |quote =  }}</ref>。<!--具体隶属军种及隶属关系待查-->
