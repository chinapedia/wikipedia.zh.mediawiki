'''中国人民解放军陆军三界合同战术训练基地'''，位于[[安徽省|安徽省]][[明光市|明光市]][[三界镇_(明光市)|三界镇]]，是隶属[[中国人民解放军东部战区陆军|中国人民解放军东部战区陆军]]的训练基地。

三界训练基地是[[淮河|淮河]]以南最大的合同战术训练基地，能供多[[軍種|軍種]][[师_(军队)|師]]級對抗演練。在[[中国人民解放军陆军朱日和合同战术训练基地|朱日和合同战术训练基地]]建成前也是解放军最大和唯一的合同战术训练基地<ref name=gfb/>，有“东方[[欧文堡国家训练中心|欧文堡]]”之称<ref>{{cite news|title=90年90个第一丨三界：第一个合同战术训练基地|url=http://news.xinhuanet.com/mil/2017-07/10/c_129651669.htm|publisher=新华网|date=2017-07-10}}</ref>。

==沿革==
三界训练基地是解放军第一个合同战术训练基地，于1986年4月成立，是当时[[亚洲|亚洲]]占地面积最大的合同战术训练基地。在1985年[[百万大裁军|百万大裁军]]的背景下，该基地的建成标志着解放军训练现代化的开始，成为当时军事改革的试验田。在建设该训练基地之前，中国军队领导知道美国建立了[[欧文堡国家训练中心|欧文堡国家训练中心]]，便先后去考察三期，认为美国这种做法可仿效，乃决定成立三界训练基地<ref name=gfb>{{Cite web |url = http://news.mod.gov.cn/action/2014-10/09/content_4542398_3.htm |title =  媒体盘点我军3大训练基地：朱日和科技含量高|author =  |date = 2014-10-09 |publisher =中华人民共和国国防部  |language =  |accessdate =  |quote =  }}</ref><ref name=budui>{{Cite web |url = http://news.xinhuanet.com/mil/2006-07/28/content_4886732.htm |title = 我军训练基地拥有专业化程度最高的模拟敌军部队 |author =  |date = 2006-07-28 |publisher =新华网  |language =  |accessdate =  |quote =  }}</ref>。

1987年起，部队开始进驻三界训练基地训练<ref name=budui/>。1987年10月，三界训练基地开展了“871”红蓝对抗演习，这次师规模的实兵对抗演习是解放军首次红军和蓝军的对抗演习。当时三界训练基地已建有模仿战场的工事、战壕、掩体，并已完成计算机作战模拟系统、激光交战辅助系统、初级战场监视系统等等。1991年，三界训练基地建成一支团级规模的蓝军模拟部队。随后三界训练基地的建设模式被中国人民解放军全军参照，自1990年代中期起，中国人民解放军的合同战术训练基地先后组建<ref name=gfb/>。1998年3月，该基地首次对外军开放，接待了[[美国陆军|美国陆军]]训练代表团随机观看了一场实兵对抗性演练<ref name=budui/>。

随着[[中国人民解放军陆军朱日和合同战术训练基地|朱日和合同战术训练基地]]的建成，三界训练基地已不再是解放军最大的合同战术训练基地，但三界训练基地依旧发挥着作用。21世纪初，先后举办“必胜-2009”实兵自主对抗演习、“2012·南京对抗”实兵演习、“火力-2014三界”实兵演习<ref name=gfb/><ref name=xhw>{{Cite web |url = http://news.xinhuanet.com/mil/2014-10/10/c_127080398_4.htm |title = 揭秘解放军三大军事训练基地：朱日和科技含量最高 |author =  |date =2014-10-10  |publisher = 新华网 |language =  |accessdate =  |quote =  }}</ref>。

2016年，在[[深化国防和军队改革|深化国防和军队改革]]中，三界训练基地由[[中国人民解放军南京军区|中国人民解放军南京军区]]转隶[[中国人民解放军东部战区陆军|中国人民解放军东部战区陆军]]<ref>{{Cite web |url = http://mt.sohu.com/20160727/n461282712.shtml |title = 安徽省党政军领导走访慰问驻皖部队 |author =  |date =2016-07-27  |publisher = 搜狐 |language =  |accessdate =  |quote =  }}</ref>。
