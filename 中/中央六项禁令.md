'''“六项禁令”'''是2012年12月4日，[[中国共产党|中国共产党]]内就[[中共十八大以来的反腐败工作|全面从严治党]]、改进党风建设提出的六项基本指标{{NoteTag|习总书记在2012年12月4日的中央政治局会议上提出了中央“八项规定”，浙江省委也针对习总书记“八项规定”中的这一规定制定了更加详细的条款，称之为“六项禁令”。}}，原为[[中共浙江省委|中共浙江省委]]依据总书记[[习近平|习近平]]提出的[[中央八项规定|中央八项规定]]而再创造的详细条款，因每条以“严禁”开头<ref>{{Cite web |author = [[中国共产党浙江省委员会|中国共产党浙江省委员会]] |language = zh-cn |url = http://zjrb.zjol.com.cn/html/2012-12/27/content_1930977.htm?div=-1 |title = 六项禁令 |newspaper = [[浙江日报|浙江日报]] |date = 2012-12-27 |accessdate = 2016-10-10 |quote = 严禁用公款搞相互走访、送礼、宴请等拜年活动。各地各部门要大力精简各种茶话会、联欢会，严格控制年终评比达标表彰活动，单位之间不搞节日慰问活动，未经批准不得举办各类节日庆典活动。}}</ref>，被称为“六项禁令”。

== 历史沿革 ==

=== 背景 ===
{{See also|中國的貪污|中华人民共和国腐败案件列表|四风}}

[[六项禁令|六项禁令]]、[[八项规定|八项规定]]未提出之前，由于受历史遗留问题，在[[中国共产党|中国共产党]]领导下的公务员、官职人员身上日益出现[[形式主义|形式主义]]、[[官僚主义|官僚主义]]、[[享乐主义|享乐主义]]和[[奢靡之风|奢靡之风]]等，[[四风|四风]]不正的严重情况，以及一些公职人员[[中國的貪污|腐败贪污]]，滥用职权之便，持续高消费和出手阔气，行贿走后门之风弥漫，更有一批执政官员成为中国[[黑社会|黑社会]]的保护伞等情况，例如：
* 1951年底，[[刘青山张子善案|刘青山张子善案]]发。1952年2月10日，[[中华人民共和国政府|中华人民共和国政府]]以[[贪污|贪污罪]]名公开处决[[张子善|张子善]]与[[刘青山_(河北省)|刘青山]]。1950年至1951年，二人在任[[天津专区|天津专区]]领导期间，盗窃地方粮款28,9151万元（[[人民币|旧币]]）、防汛水利专款30亿元、救灾粮款4亿元、干部家属救济粮款1亿4千万元，克扣修理机场民工供应补助粮款5亿4千3百3十万元，赚取治河民工供应粮款3亿7473万元，倒卖治河民工食粮从中渔利22亿元；此外还以修建为名骗取银行贷款60亿元，从事非法经营。

* 2005年9月，[[經濟合作與發展組織|經濟合作與發展組織]]（OECD）在[[北京|北京]]舉行的「亞太反貪腐倡議（Anti-corruption Initiative for Asia-Pacific）」研討會上發表報告指出，隨經濟改革，中國的貪污問題日益猖獗。2004年中國涉及貪污的金額高達4090億元人民幣至6830億元人民幣。
* 2007年9月26日，[[柏林|柏林]][[透明國際|透明國際]]指出，在一百八十個國家的評比中，[[中國|中國]][[貪污感知指數|清廉指數]]（CPI）第七十二名。

因此，中共当局和领导人开始了一系列的自身整顿运动，虽然并非[[文革|文革]]期间的自上而下的[[阶级斗争|阶级斗争]]，而是受[[中国大陆|中国大陆]]广大百姓的要求和监督下，其约束力比较显著，例如[[邓小平|邓小平]]时期的“[[清理三种人|清理三种人]]”、[[江泽民|江泽民]]时期的“[[三个代表|三个代表]]”；[[胡锦涛|胡锦涛]]时期的“[[唱红打黑|唱红打黑]]”：

* 中国共产党第十七届[[中央纪律检查委员会|中央纪律检查委员会]]第二次全体会议上,前[[中共中央總書記|中共中央總書記]][[胡锦涛|胡锦涛]]发表讲话，表示今后会把反腐败工作的成效看作是取信于民的重要指标，这是中国进入转型发展近30年来第一次明确提出来的关于反腐败目标的看法<ref>{{cite news |url = http://cpc.people.com.cn/GB/66888/77791/6783889.html|title=任建明：总书记强调把反腐工作成效看作取信于民的重要指标|publisher=中国共产党新闻网|accessdate=2008-08-30}}</ref>。
* 2012年11月15日，[[習近平|習近平]]在[[中共十八屆一中全會|中共十八屆一中全會]]上當選[[中國共產黨中央委員會總書記|中國共產黨中央委員會總書記]]，他上任後與[[中共中央紀委|中共中央紀委]]書記[[王岐山|王岐山]]實行嚴厲的[[中共十八大以来的反腐败工作|反腐行動]]，導致大量在任或已經卸任的官員落馬，包括原中央政治局常委[[周永康|周永康]]和原中央軍委副主席[[郭伯雄|郭伯雄]]、[[徐才厚|徐才厚]]等退休高官。

=== 过程 ===
2012年12月4日，[[中共中央政治局|中共中央政治局]]召开会议，由时任中共中央总书记[[习近平|习近平]]提出，并出台了改进工作作风、密切联系群众的[[八项规定|八项规定]]。

2012年12月26日，浙江省官员依据[[八项规定|八项规定]]的内容，自行再创出'''中央六项禁令'''，后经中共当局以及中共所控制下的新闻媒体广泛转发，并正式定名为'''[[六项禁令|六项禁令]]'''。

=== 结果 ===
2012年12月26日，[[六项禁令|六项禁令]]的提出给[[中共|中共]]在自身管理上提供了条文依据，而这[[六项禁令|六项禁令]]至2016年后还在实行。

==內容==
{{Cquote|
一、严禁用公款搞相互走访、送礼、宴请等拜年活动。各地各部门要大力精简各种茶话会、联欢会，严格控制年终评比达标表彰活动，单位之间不搞节日慰问活动，未经批准不得举办各类节日庆典活动。上下级之间、部门之间、单位之间、单位内部一律不准用公款送礼、宴请。各地都不准到省、市机关所在地举办乡情恳谈会、茶话会、团拜会等活动，已有安排的，必须取消。各级党政干部一律不准接受下属单位安排的宴请，未经批准不准参与下属单位的节日庆典活动。

二、严禁向上级部门赠送土特产。各地各部门各单位一律不准以任何理由和形式向上级部门赠送土特产，包括各种提货券。各级党政干部不得以任何理由，包括下基层调研等收受下属单位赠送的土特产和提货券。各级党政机关要严格纪律要求，加强管理，杜绝在机关收受和分发土特产的情况发生。

三、严禁违反规定收送礼品、礼金、有价证券、支付凭证和商业预付卡。各级领导干部一定要严格把关，严于律己，要坚决拒收可能影响公正执行公务的礼品、礼金、有价证券、支付凭证和商业预付卡，严禁利用婚丧嫁娶等事宜借机敛财。

四、严禁滥发钱物，讲排场、比阔气，搞铺张浪费。各地各部门不准以各种名义年终突击花钱和滥发津贴、补贴、奖金和实物；不准违反规定印制、发售、购买和使用各种代币购物券（卡）；不准借用各种名义组织和参与用公款支付的高消费娱乐、健身活动；不准用公款组织游山玩水、安排私人度假旅游、出国（境）旅游等活动；不准违反规定使用公车、在节日期间公车私用。

五、严禁超标准接待。领导干部下基层调研、参加会议、检查工作等，要严格按照中央和省委的有关要求执行。

六、严禁组织和参与赌博活动。各级党员干部一定要充分认识赌博的严重危害性，决不组织和参与任何形式的赌博活动。<ref>[http://zjrb.zjol.com.cn/html/2012-12/27/content_1930977.htm?div=-1 《六项禁令》，浙江日報]</ref>
}}

== 影响 ==
八项规定和六项禁令影响部分高端商品和服务的消费。据北京市商务委称，受中央八项规定的影响，北京餐饮企业纷纷转型，使得大众餐饮逆势而上，同比增长一成多。<ref>{{cite web |author1=中国经济新闻联播 |title=“八项规定”和“六项禁令”让高端餐饮尝试转型 |url=http://hb.ifeng.com/dfzx/detail_2013_08/06/1076339_0.shtml |accessdate=2019-01-07}}</ref> 在2014年八项规定实施一年多以后，有调查显示高档酒品、茶叶等销售额有明显下降。奢侈品的市场也疲软。<ref>{{cite web |author1=广州日报 |title=调查称八项规定一年半高档酒品茶叶等降幅最大 |url=http://www.chinanews.com/sh/2014/04-25/6106029.shtml |accessdate=2019-01-07}}</ref>

== 评价 ==
有社会学者分析称：[[八项规定|八项规定]]、[[六项禁令|六项禁令]]、反对“[[四风|四风]]”使[[中国大陆|中国大陆]]地区全面进入反腐倡廉的新政<ref>{{Cite web |url = http://news.xinhuanet.com/legal/2016-02/01/c_1117959296.htm|title =“八项规定”令中国灰色消费遭遇严冬 |author =新华网 |date = 2012-12-27 |publisher =浙江日报|language =zh-cn |accessdate =2016-10-10|quote =新华社北京２月１日电（杨思琪　李思思）寒潮来袭，26岁的张晶感受到的不仅是天气的寒冷，还有生意的冷清。分析人士指出，以八项规定、六项禁令、反对“四风”为代表的中国反腐倡廉新政，一定程度上影响了中国奢侈品消费市场。与此同时，另一个暗藏着灰色消费的购物卡行业也举步维艰。}}</ref>，从一定程度上影响了中国奢侈品灰色消费市场，从根源上剔除'''剥削阶级思想'''<ref>{{Cite web |url = http://www.qstheory.cn/dukan/hqwg/2016-05/25/c_1118927249.htm|title = 新理念引领反腐败斗争新实践 |author = 求是网-《红旗文稿》 |date =2012-12-27|publisher =朱新现|language =zh-cn |accessdate = 2016-10-10 |quote =党的十八大以来，以习近平同志为总书记的党中央着眼于新的形势任务，把全面从严治党纳入“四个全面”战略布局，把党风廉政建设和反腐败斗争作为全面从严治党的重要内容，正风肃纪，反腐惩恶，着力构建不敢腐、不能腐、不想腐的体制机制。在深入揭示腐败本质、探索中国共产党反腐败工作规律的基础上，党中央提出了一系列关于惩治与预防腐败的新理念，反腐败斗争压倒性态势正在形成。长期以来，我们把腐败现象归根于受剥削阶级思想的影响,认为“共产党天然与腐败现象绝缘”，广大党员对腐败有着内在的免疫力和抵抗力，这种认识掩盖、回避、漠视腐败产生的可能性和腐败问题的严重性，从而在治理腐败问题上存在不足和出现漏洞。全面从严治党新时期，党中央认识到腐败问题的客观性，指出：“任何权力都面临被腐蚀的危险，执政党永远会面对与腐败的斗争。”这是开展惩治和预防腐败工作的根本出发点。党中央从中国现实出发，正视和揭露腐现象和问题，提出“全党必须警醒起来”“打铁还需自身硬”的要求，以猛药去疴、重典治乱的决心，高调反腐、强力肃贪，筑牢反腐败斗争的定力。}}</ref>，大大警醒了中国执政官员莫腐败的自觉性。
