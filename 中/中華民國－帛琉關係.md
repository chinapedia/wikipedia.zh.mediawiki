'''中華民國－帛琉關係'''是指[[中華民國|中華民國]]與[[帛琉共和國|帛琉共和國]]之間的外交關係。

==歷史==
*1999年12月29日，[[中華民國外交部|中華民國外交部]]長[[程建人|程建人]]與帛琉國務部長-{zh-cn:萨比诺·阿纳斯塔西奥;zh-tw:安薩賓}-（Sabino Anastacio）在科羅爾分別代表兩國政府簽署建交公報<ref>{{cite web|url=http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2000/mofa_5.htm|title=中華民國外交部89年外交年鑑|publisher=[[中華民國外交部|中華民國外交部]]|date = 2001-04 |accessdate=2018-03-01}}</ref>，中華民國與帛琉建立大使級外交關係<ref>{{cite news|url=http://www.mofa.gov.tw/News_Content_M_2.aspx?n=5028B03CED127255&sms=5ED24855AD8E6C58&s=0634720A3326473E|title=中華民國政府與帛琉共和國政府依照國際法原則，尤其是關於國家平等、相互尊重主權及互惠之原則，為加強兩國之友好及合作關係，爰經決定自1999年12月29日起建立全面外交關係。|publisher=[[中華民國外交部|中華民國外交部]]|date=1999-12-29|accessdate=2015-11-02}}</ref>。

*2000年2月17日，中華民國在前首府[[科羅爾|科羅爾]]設立大使館。<ref name=h2>{{cite web|url=http://multilingual.mofa.gov.tw/web/web_UTF-8/YearBook/105/PDF/%E5%A4%96%E4%BA%A4%E5%B9%B4%E9%91%91%E5%AE%8C%E6%95%B4%E6%8E%92%E7%89%88.pdf|title=中華民國外交部105年外交年鑑|publisher=[[中華民國外交部|中華民國外交部]]|date = 2017-05 |accessdate=2018-03-01}}</ref>

*2001年10月4日，帛琉在[[臺北市|台北]]設立大使館。<ref name=h2/>

==兩國貿易==
2018年帛琉是中華民國第126名貿易伙伴，兩國貿易額26,448,881美元，其中中華民國對帛琉出口25,704,602美元，是第105大出口國，進口744,279美元，是第158大進口國，與2017年貿易額相比貿易總額約增加13.86%、出口約增加13.80%、進口約增加16.11%。<ref>{{cite book |url = http://cus93.trade.gov.tw/FSCI/ |title = 經濟部國際貿易局貿易統計系統 |publisher = [[經濟部國際貿易局|經濟部國際貿易局]] |date = 2019-02-23 |accessdate = 2019-03-05}}</ref>
*中華民國－帛琉主要出口項目：生鮮或冷藏魚、石油瀝青、鋼鐵製品、塑膠製品、飲水、自動資料處理機等。<ref name=bft>{{Cite web |url = https://www.trade.gov.tw/World/Detail.aspx?nodeID=45&pid=581418|title = 各國經貿資訊 |publisher =[[經濟部國際貿易局|經濟部國際貿易局]]|date = 2017-01-09|accessdate = 2018-03-04}}</ref>
*中華民國－帛琉主要進口項目：鋁、鐵、銅廢料及碎屑、原木等。<ref name=bft/>

===航空客運===
兩國之間有直航航線，來往帛琉之客運航線如下：

1.{{flagicon|ROC}}[[臺灣桃園國際機場|臺北-桃園]]→{{flagicon|PLW}}[[科羅爾|柯羅]]-[[羅曼·莫圖國際機場|羅曼·莫圖國際機場]]（由[[中華航空|中華航空]]經營）。

＊上述目的地國際機場為鄰近該國首都或該國最主要樞紐機場，若要前往該國國內線機場詳見{{link-en|帛琉機場列表|List of airports in Palau|帛琉機場}}。

＊航線會因航空公司營運狀況更改或取消，出發前應與負責該航線的航空公司確認航線及航班，以免影響權益。

==邦交挑戰==
中華人民共和國是帛琉最大的旅客來源國，2017年11月中華人民共和國政府藉由頒布旅遊禁令，想對帛琉政府施加壓力促使帛琉政府轉換外交承認，雖然帛琉現任雷蒙傑索政府保證對中華民國關係不變<ref>{{cite web|url=https://www.ettoday.net/news/20180102/1084636.htm|title=大陸旅遊禁令威脅與台灣斷交　帛琉：與台灣關係不變|publisher=[[東森新聞雲|東森新聞雲]]|date=2018-01-02|accessdate=2018-05-11}}</ref>，因頒布旅遊禁令，中華人民共和國旅客人數銳減，加上2018年7月帛琉太平洋航空飛往香港航線宣布停飛<ref>{{cite web|url=https://tw.news.yahoo.com/%E5%A0%85%E4%B8%8D%E8%88%87%E5%8F%B0%E7%81%A3%E6%96%B7%E4%BA%A4%E5%90%9E%E5%8C%97%E4%BA%AC%E6%81%90%E5%9A%87-%E5%B8%9B%E7%90%89%E8%88%AA%E7%A9%BA%E5%81%9C%E9%A3%9B%E4%B8%AD%E5%9C%8B-011042904.html|title=堅不與台灣斷交吞北京恐嚇 帛琉航空停飛中國|publisher=[[新頭殼|新頭殼]]|date=2018-07-18|accessdate=2018-10-07}}</ref>，造成國家經濟收入減少，帛琉國會現任參眾兩院議長<ref>{{cite web|url=https://www.radionz.co.nz/international/pacific-news/363194/call-for-palau-to-increase-cooperation-with-china|title=Call for Palau to increase cooperation with China|publisher=[[紐西蘭國際廣播電台|紐西蘭國際廣播電台]]|date=2018-08-02|accessdate=2019-03-17}}</ref>及兩位曾經擔任駐中華民國大使都轉向支持中華人民共和國<ref>{{cite web|url=https://udn.com/news/story/7331/3319685|title=帛琉總統挺台灣 大選後恐向左轉|publisher=[[聯合新聞網|聯合新聞網]]|date=2018-08-20|accessdate=2018-10-07}}</ref>，曾代表帛琉政府與中華民國政府簽訂建交公報現任眾議院議長-{zh-cn:萨比诺·阿纳斯塔西奥;zh-tw:安薩賓}-（Sabino Anastacio）也發布聲明認為台帛1999年建交公報，明顯已過時，甚至預期在2020年前帛琉現任雷蒙傑索政府就會轉向中華人民共和國<ref>{{cite web|url=https://times.hinet.net/times/news/22006669|title=帛琉議長預期 兩年內帛琉就會與大陸建交|publisher=[[聯合新聞網|聯合新聞網]]|date=2018-10-07|accessdate=2018-10-07}}</ref>，面對原本支持中華民國的帛琉政要紛紛轉向，如何維持兩國邦誼也是中華民國政府現今急需面對的課題。
