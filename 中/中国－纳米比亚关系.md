'''中國－納米比亞關係'''，是指歷史上的[[中國歷史|中國]]和[[納米比亞歷史|納米比亞]]（包括[[中華人民共和國|中華人民共和國]]和[[納米比亞共和國|納米比亞共和國]]）之間的雙邊關係。中華人民共和國曾支持納米比亞獨立，兩國在1990年3月21日建立外交關係；目前，中国和纳米比亚在經貿和文化上均有來往。

== 歷史 ==
《[[嶺外代答|嶺外代答]]》、《[[諸蕃志|諸蕃志]]》和《[[明史外國傳|明史外國傳]]》均提及了一個名為「木蘭皮國」的古國<ref name=Study>{{cite journal |author1=林貽典 |title=關於美洲由鄭和船隊發現的見解 |journal=鄭和研究與活動簡訊 |date=2002-08-30 |issue=6 |url=http://proj.ncku.edu.tw/chengho/newsletter/no6.html |archiveurl=https://archive.is/20150701164835/http://proj.ncku.edu.tw/chengho/newsletter/no6.html |archivedate=2015-07-01 |deadurl=no |accessdate=2015-07-29 }}</ref>。其中，《明史外國傳》記載「木蘭皮國」內有一頭體型巨大的[[羊|羊]]，它身高數尺，尾巴好像一把[[扇子|扇子]]，而[[喀拉哈里沙漠|喀拉哈里沙漠]]的確有符合上述特徵的[[大羚羊|大羚羊]]；書中又記載說，假如「木蘭皮國」內忽然吹起秋風，當地的人和牲畜便要趕緊喝水，否則便會渴死，這個記載也符合納米比亞的氣候<ref name=Study/>。基於以上與納米比亞風土人情吻合的記載、加上《諸蕃志》和《明史外國傳》對「木蘭皮國」地理位置的描述，有學者認為納米比亞就是「木蘭皮國」<ref name=Study/>；但是，學者對「木蘭皮國」所指何地亦有其他看法<ref>{{cite journal|author1=中国造船工程学会|title=舰船知识|issue=1–15}}</ref>{{Rp|63}}。亦有學者認為，[[鄭和|鄭和]]的船隊曾在1419年的夏季到訪納米比亞<ref name=Study/>。

[[南非|南非]]在1920年開始統治當時被稱為「[[西南非|西南非]]」的納米比亞<ref name=intro>{{cite web |title=纳米比亚国家概况 |url=http://www.fmprc.gov.cn/mfa_chn/gjhdq_603914/gj_603916/fz_605026/1206_605970/ |website=中华人民共和国外交部 |accessdate=2015-07-30 |archiveurl=https://archive.is/20150730093859/http://www.fmprc.gov.cn/mfa_chn/gjhdq_603914/gj_603916/fz_605026/1206_605970/ |archivedate=2015-07-30 |deadurl=no }}</ref>；中華人民共和國支持納米比亞從南非獨立，從1973年起每年向[[联合国|联合国]]設立的纳米比亚基金捐款，又多次要求南非政府履行[[联合国安理会|联合国安理会]]{{link-en|联合国安全理事会435号决议|United Nations Security Council Resolution 435|关于纳米比亚独立的决议}}<ref>{{cite book|author1=钱其琛|author2=王泰平|author3=张光佑|author4=马可铮|title=新中国外交50年|volume=2|publisher=北京出版社|year=1999}}</ref>{{Rp|736}}，但沒有參與該决议的投票<ref>{{cite web|title=Resolution 435|url=http://unscr.com/en/resolutions/doc/435|website=UNSCR|accessdate=2015-07-31}}</ref>。中華人民共和國亦有為爭取纳米比亚獨立的組織提供援助，而[[西南非洲人民组织|西南非洲人民组织]]主席[[萨姆·努乔马|萨姆·努乔马]]亦在1983年、1985年和1986年三度率团訪問中華人民共和國<ref>{{cite book|author1=郑启荣 (編)|title=改革开放以来的中国外交 (1978-2008)|work=强国之路. 纪念改革开放30周年重点书系|publisher=世界知识出版社|year=2008}}</ref>{{Rp|197}}。當時，南非政府把纳米比亚之爭與[[古巴|古巴]]軍事介入[[安哥拉内战|安哥拉内战]]一事相提並論；中華人民共和國反對南非的觀點，認為兩個問題之間沒有關係，把二事挂钩只會使纳米比亚問題更複雜<ref>{{cite book|author1=谢益显|title=中国当代外交史（1949-2001）——外交学丛书|publisher=中国青年出版社|year=1997|edition=2}}</ref>{{Rp|414}}。

納米比亞于1990年3月21日正式宣布独立<ref name=intro/>，同日與中華人民共和國建立邦交<ref>{{cite book|title=新中国对外文化交流史略|publisher=中国友谊出版公司|year=1999|isbn=9787505714779}}</ref>{{Rp|324}}。2018年3月，两国建立全面战略合作伙伴关系<ref>{{cite web |title=中国同纳米比亚的关系 |url=https://www.fmprc.gov.cn/chn//gxh/cgb/zcgmzysx/fz/1206_38/1206x1/t6628.htm |accessdate=2018-09-02}}</ref>。

== 經貿關係 ==
中華人民共和國在2009年出口了2.65亿[[美元|美元]]貨物到纳米比亚，纳米比亚同年出口了3.12亿美元貨物來華，兩國的雙邊贸易总额為5.77亿美元；2014年，中華人民共和國出口了5.45美元到纳米比亚，纳米比亚則出口3.18亿美元來華，雙邊贸易总额為8.63億美元<ref>{{cite web |title=2009-2014年中纳双边贸易额 |url=http://na.mofcom.gov.cn/article/zxhz/201507/20150701053817.shtml |website=中华人民共和国驻纳米比亚共和国大使馆经济商务参赞处 |accessdate=2015-08-01 |archiveurl=https://archive.is/20150801094122/http://na.mofcom.gov.cn/article/zxhz/201507/20150701053817.shtml |archivedate=2015-08-01 |deadurl=yes |date=2015-07-21 }}</ref>。中国與纳米比亚的雙邊贸易數量不大，但增长较快；中華人民共和國出口到纳米比亚的貨物包括{{link-en|机电学|Electromechanics|机电产品}}、[[化學工業|化工]]产品等，纳米比亚則主要出口[[海鮮|海鮮]]和[[矿产|矿产品]]到中國<ref name=Econ>{{cite web |title=合作简介 |url=http://na.mofcom.gov.cn/article/zxhz/201409/20140900744096.shtml |website=中华人民共和国驻纳米比亚共和国大使馆经济商务参赞处 |accessdate=2015-08-01 |archiveurl=https://archive.is/20150801094603/http://na.mofcom.gov.cn/article/zxhz/201409/20140900744096.shtml |archivedate=2015-08-01 |deadurl=no |date=2014-09-25 }}</ref>。

中國企业在纳米比亚承包的工程涵蓋[[采矿业|采矿业]]、[[建筑|建筑]]、[[通信|通信]]等範疇，共创造了超過4000个就业机会；除了投資和工程承包外，中華人民共和國也有向纳米比亚提供援助和[[贷款|贷款]]，資助後者興修[[學校|學校]]、[[医院|医院]]、[[公路|公路]]等[[公共設施|公共設施]]<ref name=Econ/>。

== 文化關係 ==
[[中国地质大学|中国地质大学]]與{{link-en|纳米比亚大学|University of Namibia}}合办了一所[[孔子学院|孔子学院]]，該学院在2013年8月23日成立<ref>{{cite web |title=纳米比亚大学孔子学院正式成立 |url=http://na.chineseembassy.org/chn/zngx/kjwh/t1074233.htm |website=中华人民共和国驻纳米比亚共和国大使馆 |accessdate=2015-08-01 |archiveurl=https://archive.is/20150801104306/http://na.chineseembassy.org/chn/zngx/kjwh/t1074233.htm |archivedate=2015-08-01 |deadurl=no |date=2013-09-08 }}</ref>；亦有纳米比亚華商在當地創辦[[基金会|基金会]]，資助纳米比亚学生在中國大陸接受[[医学|医学]]教育<ref>{{cite web |title=驻纳米比亚大使忻顺康出席华商捐赠奖学金仪式 |url=http://na.chineseembassy.org/chn/zngx/kjwh/t1087979.htm |website=中华人民共和国驻纳米比亚共和国大使馆 |accessdate=2015-08-01 |archiveurl=https://archive.is/20150801103610/http://na.chineseembassy.org/chn/zngx/kjwh/t1087979.htm |archivedate=2015-08-01 |deadurl=no |date=2013-10-11 }}</ref>。

== 軍事關係 ==
[[中国人民解放军海军|中国人民解放军海军]]的第十六批护航编队曾在2014年到訪纳米比亚[[鲸湾港|鲸湾港]]，民眾可以登上盐城舰和洛阳舰，兩國[[海军|海军]]亦會舉行[[足球比赛|足球比赛]]等交流活动；此外，解放军海军和{{link-en|纳米比亚海军|Namibian Navy}}亦曾举行联合演练<ref>{{cite web |title=中国海军第十六批护航编队访问纳米比亚 |url=http://na.chineseembassy.org/chn/zngx/kjwh/t1165182.htm |website=中华人民共和国驻纳米比亚共和国大使馆 |accessdate=2015-08-01 |archiveurl=https://archive.is/20150801100823/http://na.chineseembassy.org/chn/zngx/kjwh/t1165182.htm |archivedate=2015-08-01 |deadurl=no |date=2014-06-12 }}</ref>。

2015年，《{{link-en|納米比亞人報|The Namibian}}》引述一封納米比亞駐華大使寄往{{link-en|納米比亞外交部|Ministry of Foreign Affairs (Namibia)|該國外交部}}的密函，聲稱中華人民共和國考慮在鯨灣港建立一座[[海軍|海軍]][[基地|基地]]；{{link-en|納米比亞國防部|Ministry of Defence (Namibia)}}發言人不予承認或否認，[[中华人民共和国国防部|中华人民共和国国防部]]則認為有關引述是「編造故事」<ref>{{cite news |title=中國否認將在納米比亞建海軍基地 |url=http://www.bbc.com/zhongwen/trad/world/2015/01/150121_china_namibia_military_base |accessdate=2015-08-01 |agency=BBC 中文网 |date=2015-01-21 |archiveurl=https://archive.is/20150801032817/http://www.bbc.com/zhongwen/trad/world/2015/01/150121_china_namibia_military_base |archivedate=2015-08-01 |deadurl=no }}</ref>。
