'''中瓦關係'''（{{lang-en|China–Vanuatu relations}}），是指歷史上的[[中國歷史|中國]]和[[瓦努阿圖歷史|瓦努阿圖]]（包括[[中華人民共和國|中華人民共和國]]和[[瓦努阿圖共和國|瓦努阿圖共和國]]）之間的雙邊關係。中華人民共和國和瓦努阿圖在1982年3月26日建立外交關係，兩國有經貿和文化上的來往，中華人民共和國亦有向瓦努阿圖提供援助。

== 歷史 ==
最早的一批[[瓦努阿圖華人|瓦努阿圖華人]]在[[清朝|清朝]]末年來到當地，他們起初為瓦努阿圖的[[莊園|莊園]]主人工作，後來開始自己經商；這些[[華人|華人]]在中國的親戚很多會到瓦努阿圖投奔他們，令瓦努阿圖華人的數量有所增加<ref>{{cite journal|author1=郭又新|title=南太平洋岛国华侨华人的历史与现状初探|journal=东南亚研究|issue=6|year=2014|url=http://max.book118.com/html/2015/0704/20395463.shtm}}</ref>{{Rp|88}}。

1906年10月，[[英國|英國]]和[[法國|法國]]签訂公约，決定共同管轄當時被稱為「[[新赫布里底|新赫布里底]]」的瓦努阿圖；瓦努阿圖在1980年7月30日独立為國，其後在1982年3月26日与中華人民共和國建立邦交<ref name="Intro">{{cite web |title=瓦努阿图概况 |url=http://vu.chineseembassy.org/chn/ljwlat/gk/t461403.htm |website=中华人民共和国驻瓦努阿图共和国大使馆 |accessdate=2015-08-10 |archiveurl=https://archive.is/20150810135057/http://vu.chineseembassy.org/chn/ljwlat/gk/t461403.htm |archivedate=2015-08-10 |deadurl=no |date=2008-06-03 }}</ref>。

[[六四事件|六四事件]]後，中華民國駐[[斐济|斐济]]代表机构的負責人向一些瓦努阿图政府高層提供赠金，又批評中華人民共和國在當地援建的议会大厦出現問題，提出由中華民國為當地援建议会大厦，换取瓦努阿图承認退到[[台灣|台灣]]的中華民國；該名負責人又指出，不少台湾[[渔船|渔船]]在瓦努阿圖[[埃斯皮里图桑托岛|埃斯皮里图桑托岛]]海域作业，這些渔船需要向瓦努阿圖當局缴交捕鱼费，故中華民國有需要在當地设立机构<ref name=Taiwan>{{cite web |title=我同瓦努阿图总理利尼的一些交往 |url=http://www.sdfao.gov.cn/art/2010/5/14/art_476_3751.html |website=山东省人民政府外事办公室 |accessdate=2015-08-10 |archiveurl=https://archive.is/20150810154737/http://www.sdfao.gov.cn/art/2010/5/14/art_476_3751.html |archivedate=2015年8月10日 |deadurl=yes |date=2010-05-14 |df= }}</ref>。及後，中華人民共和國為此派员到瓦努阿圖，但[[瓦努阿圖總理|瓦努阿圖總理]]{{link-en|沃爾特·利尼|Walter Lini}}在中華人民共和國人員到達時正於外岛休假，不會與外國使节會晤，瓦努阿圖外交部部長當時又在外訪；中華人民共和國的人員便與瓦努阿圖外交部常务秘书會面，他不支持中華民國在瓦努阿圖首都[[维拉港|维拉港]]設立机构，但對中華民國為瓦努阿圖援建议会大厦的建議不置可否<ref name=Taiwan/>。中華人民共和國其後再派員到瓦努阿圖，會見總理利尼，利尼表示「相信两国关系定会向前发展」<ref name=Taiwan/>。在會面後，瓦努阿图内阁讨论对华关系的問題，有些官員主张接受中華民國的援助条件，有些官員則認為要與中华人民共和国保持友好关系；總理利尼最後決定拒绝接受中華民國為當地援建议会大厦，亦不會與中華民國磋商建立外交關係<ref name=Taiwan/>。1989年11月3日，瓦努阿图政府同意中華人民共和國在當地设置[[大使馆|大使馆]]<ref>{{cite book|author1=中华人民共和国外交部外交史编辑室|title=中国外交概览|publisher=世界知识出版社|year=1990}}</ref>{{Rp|334}}。

1992年，{{link-en|瑟奇·沃霍尔|Serge Vohor}}出任瓦努阿圖外交部部長<ref>{{cite web |author1=宋宏伟 |title=南太平洋上的一场反“台独”阻击战 |url=http://www.zhongguotongcuhui.org.cn/tylt/2005ndwq/201210/t20121019_3203382.html |website=中国和平统一促进会 |accessdate=2015-08-11 |archiveurl=https://archive.is/20150811134947/http://www.zhongguotongcuhui.org.cn/tylt/2005ndwq/201210/t20121019_3203382.html |archivedate=2015-08-11 |deadurl=no |date=2008-06-10 }}</ref>。同年9月，中華民國與瓦努阿圖簽訂「政府间相互承认」的联合公报；事後，中華民國稱這是當局第一次以「相互承认」的方式与中華人民共和國邦交国建立「官方关系」<ref>{{cite book|author1=高民政|title=台湾政治纵览|publisher=华文出版社|year=2000|isbn=9787507509960}}</ref>{{Rp|311}}。時任[[中華民國外交部|中華民國外交部]]部長[[錢復_(中華民國)|錢復]]後來接受傳媒訪問，認為當局與瓦努阿圖相互承认是「對國際、對內、對中共」的「試探氣球」<ref>{{cite book|author1=邵宗海|title=兩岸關係: 兩岸共識與兩岸岐見|publisher=五南圖書出版股份有限公司|year=1998|isbn=9789571115306}}</ref>{{Rp|426}}。

2004年11月，時任瓦努阿圖總理的沃霍尔突然出访台湾，期間與中華民國外交部部長[[陈唐山|陈唐山]]签署建交公报，建立瓦努阿圖與中華民國的外交關係，此前沒有與其他党派和内阁成员商議<ref name=job>{{cite news |author1=高楠 (编) |author2=郭拥军 |title=“建交”闹剧笑死人 瓦努阿图不欢迎台湾 |url=http://www.people.com.cn/GB/guoji/14549/3056845.html |accessdate=2015-08-11 |agency=环球时报 |date=2004-12-13 |archiveurl=https://archive.is/20150811085431/http://www.people.com.cn/GB/guoji/14549/3056845.html |archivedate=2015-08-11 |deadurl=no }}</ref>；[[中華人民共和國外交部|中華人民共和國外交部]]發言人指當局與瓦努阿圖的關係「出現了一些問題」，要求瓦努阿圖實行兩國建交公報的相關原則，但沒有與該國斷絕外交關係<ref>{{cite news |title=萬那杜對兩岸「雙重承認」　大陸未斷交 |url=http://news.tvbs.com.tw/old-news.html?nid=467829 |accessdate=2015-08-11 |agency=TVBS新聞 |date=2004-11-18 |archiveurl=https://archive.is/20150811091051/http://news.tvbs.com.tw/old-news.html?nid=467829 |archivedate=2015-08-11 |deadurl=no }}</ref>。[[美聯社|美聯社]]引述瓦努阿圖政府發言人指，瓦努阿圖絕對不會與中華人民共和國斷絕邦交，該國承認中華民國只是因為當局承諾提供60億[[美元|美元]]的援助，但中華民國外交部否認此指控<ref>{{cite news |title=萬那杜雙重承認　外交部：建交有效 |url=http://news.tvbs.com.tw/old-news.html?nid=469314 |accessdate=2015-08-11 |agency=TVBS新聞 |date=2004-11-08 |archiveurl=https://archive.is/20150811092114/http://news.tvbs.com.tw/old-news.html?nid=469314 |archivedate=2015-08-11 |deadurl=no }}</ref>；沃霍尔又接受傳媒訪問，表示希望瓦努阿圖能同時得到中華人民共和國和中華民國的援助，認為與中華民國建交能夠補充中華人民共和國援助的不足之處，他又在訪問中表示「[[神|神]]並沒有說不能和台灣建交」<ref name="the">{{Cite thesis|type=碩士論文|publisher=東吳大學|author=張瑋|title=南太平洋島國在台海兩岸的外交選擇─以諾魯、吉里巴斯與萬那杜為例|url=http://163.14.136.66:8080/ir/retrieve/621/100SCU05227012-001.pdf|archiveurl=https://web.archive.org/web/20150811152658/http://163.14.136.66:8080/ir/retrieve/621/100SCU05227012-001.pdf|archivedate=2015-08-11|deadurl=no|accessdate=2015-08-11}}</ref>{{Rp|147, 149}}。有瓦努阿圖官員認為沃霍尔訪台是因為當地經濟面臨困境，也有學者形容沃霍尔的做法是把國家利益和個人意向結合<ref name="the"/>{{Rp|149}}。

事後，瓦努阿圖内阁两度作出决议，撤销該國與中華民國的建交公报；但是，沃霍尔其後发表声明，認為與外國建交毋須内阁会议批准，内阁的决议亦没有法律效力，此令沃霍尔與其他内阁成员的關係進一步惡化<ref name=job/>。同年12月2日，[[瓦努阿图议会|瓦努阿图议会]]对沃霍尔提出[[不信任動議|不信任動議]]，但該國最近實施了限制[[反对党|反对党]]提出不信任動議的宪法修正案；瓦努阿图最高法院首席大法官其後決定议会有权进行不信任投票，雖然沃霍尔提出上诉，但上诉法庭亦決定維持原判<ref name=job/>。最後，瓦努阿图议会決定免去沃霍尔的总理职务<ref name=job/>。

2005年8月，瓦努阿圖设立驻北京大使馆<ref>{{cite web |title=China-Vanuatu Relations |url=http://vu.chineseembassy.org/eng/zwgx/t467290.htm |website=中华人民共和国驻瓦努阿图共和国大使馆 |accessdate=2015-08-10 |archiveurl=https://archive.is/20120715193017/http://vu.chineseembassy.org/eng/zwgx/t467290.htm |archivedate=2012-07-15 |deadurl=no }}</ref>。

根據{{link-en|經濟複雜性研究中心|The Observatory of Economic Complexity}}網站上的數據，中国大陸到瓦努阿图的出口貿易在2011年由去年的不足3,000萬美元大幅提升至超過8,000萬美元<ref name=tradea/>，但在2012年大幅下降至3,140萬美元；2012年，中国大陸出口到瓦努阿图的貨物最多是[[礦產|礦產品]]，其次是[[金屬|金屬]]（及其製品）<ref>{{cite web|author=Alexander Simoes|title=产品中国出口到瓦努阿图 (2012)|url=https://atlas.media.mit.edu/zh/explore/tree_map/hs/export/chn/vut/show/2012/|publisher=Observatory of Economic Complexity|accessdate=2015-08-12}}</ref>。瓦努阿图到中国大陸的出口貿易曾在2011年由去年的不足80萬美元升至超過240萬美元，在2012年稍微跌至220萬美元；瓦努阿图在2006年至2012年主要出口畜產來中国大陸，但該國出口到中国大陸的畜產曾在2010年下降<ref name=tradeb/>至2,503美元，只佔當年出口總額的0.34%<ref>{{cite web|author=Alexander Simoes|title=产品瓦努阿图出口到中国 (2012)|url=https://atlas.media.mit.edu/zh/explore/tree_map/hs/export/vut/chn/show/2012/|publisher=Observatory of Economic Complexity|accessdate=2015-08-12}}</ref>。

== 文化關係 ==
[[中國國際廣播電台|中國國際廣播電台]]設於瓦努阿圖的[[中國國際廣播電台馬勒庫拉分台|分台]]提供[[中文|中文]]、[[英语|英语]]和[[法语|法语]]的节目；其中，中文节目的內容包括[[中國音樂|中國音樂]]，法语节目則會介绍中国與世界的热点问题<ref>{{cite news |title=瓦努阿图总理亲自出席我台瓦努阿图调频台开播仪式 |url=http://gb.cri.cn/14404/2007/09/13/2685@1761637.htm |accessdate=2015-08-12 |agency=国际在线 |date=2007-09-13 |archiveurl=https://archive.is/20150812032228/http://gb.cri.cn/14404/2007/09/13/2685@1761637.htm |archivedate=2015-08-12 |deadurl=no }}</ref>。[[中国中央电视台|中国中央电视台]]亦有在瓦努阿圖廣播<ref name="Intro"/>。瓦努阿图华侨創辦了提供中文和英文內容的《维拉时报》，這是當地首份中英文雙語報章；《维拉时报》在2015年创刊，計劃每周發行一期，主要报導中国和瓦努阿圖兩地的時事新闻<ref>{{cite web |title=瓦努阿图首份中英文报纸《维拉时报》创刊发行 |url=http://vu.chineseembassy.org/chn/zwgx/rw/t1230836.htm |website=中华人民共和国驻瓦努阿图共和国大使馆 |accessdate=2015-08-12 |archiveurl=https://archive.is/20150812044306/http://vu.chineseembassy.org/chn/zwgx/rw/t1230836.htm |archivedate=2015-08-12 |deadurl=no |date=2015-01-23 }}</ref>。

截至2015年8月，瓦努阿图共有上百名[[学生|学生]]、政府官员、[[军官|军官]]等人员到中國大陸参與学习或培训，其中有些人只在中国逗留數星期至數月，有些則在中国待了一年以至多年<ref>{{cite web |title=驻瓦努阿图大使谢波华在瓦努阿图-中国同学会成立仪式上的讲话 |url=http://vu.chineseembassy.org/chn/xwdt/t1287389.htm |website=中华人民共和国驻瓦努阿图共和国大使馆 |accessdate=2015-08-12 |archiveurl=https://archive.is/20150812045958/http://vu.chineseembassy.org/chn/xwdt/t1287389.htm |archivedate=2015-08-12 |deadurl=no |date=2015-08-08 }}</ref>。

== 援助 ==
中華人民共和國向瓦努阿图提供了经济和技术上的援助，為當地援建了[[校舍|校舍]]、[[牙科|牙科]]诊所等项目<ref>{{cite web |title=合作简介 |url=http://vu.mofcom.gov.cn/article/zxhz/hzjj/200203/20020300009825.shtml |website=中华人民共和国驻瓦努阿图大使馆经济商务参赞处 |accessdate=2015-08-12 |archiveurl=https://archive.is/20150812051625/http://vu.mofcom.gov.cn/article/zxhz/hzjj/200203/20020300009825.shtml |archivedate=2015-08-12 |deadurl=no |date=2007-04-26 }}</ref>。此外，中國亦有派遣医疗队到瓦努阿图的兩所[[医院|医院]]工作，並向當地提供医疗设备和[[藥物|藥物]]；根據中華人民共和國驻瓦努阿图大使館经商参贊处在2013年7月發表的文章，中華人民共和國累计派出了逾220個[[医生|医生]]到瓦努阿图，他們在當地累计施行了數千次临床[[手术|手术]]，並医治了数万個當地病人<ref>{{cite web |title=瓦努阿图《独立报》刊文介绍中国援瓦医疗队情况 |url=http://www.mofcom.gov.cn/article/i/jyjl/l/201307/20130700192013.shtml |website=中华人民共和国商务部 |accessdate=2015-08-12 |archiveurl=https://archive.is/20150812054929/http://www.mofcom.gov.cn/article/i/jyjl/l/201307/20130700192013.shtml |archivedate=2015-08-12 |deadurl=no |date=2013-07-09 }}</ref>。有中國医院队的队长獲[[瓦努阿图总统|瓦努阿图总统]]頒授独立勋章<ref>{{cite web |title=驻瓦努阿图大使谢波华出席瓦总统为我援瓦人员颁发“独立勋章”仪式 |url=http://vu.chineseembassy.org/chn/xwdt/t1285369.htm |website=中华人民共和国驻瓦努阿图共和国大使馆 |accessdate=2015-08-12 |archiveurl=https://archive.is/20150812060008/http://vu.chineseembassy.org/chn/xwdt/t1285369.htm |archivedate=2015-08-12 |deadurl=no |date=2015-07-31 }}</ref>。

2015年，瓦努阿图遭受[[氣旋帕姆|氣旋帕姆]]吹袭，災情嚴重<ref name="Aid">{{cite web |title=驻瓦努阿图大使谢波华在瓦努阿图灾后重建圆桌会议上的讲话 |url=http://vu.chineseembassy.org/chn/xwdt/t1287387.htm |website=中华人民共和国驻瓦努阿图共和国大使馆 |accessdate=2015-08-12 |archiveurl=https://archive.is/20150812064529/http://vu.chineseembassy.org/chn/xwdt/t1287387.htm |archivedate=2015-08-12 |deadurl=no |date=2015-08-08 }}</ref>。中華人民共和國向瓦努阿图提供一批總值3,000万[[人民币|人民币]]的物资，这些物资包括[[糧食|糧食]]、[[帐篷|帐篷]]等；中国駐瓦努阿图医疗队到一些受灾地区運送药物，當地的中资机构和华侨亦有提供援助<ref name="Aid"/>。
