'''中華小姐環球大賽'''（[[英语|英语]]：'''Miss Chinese Cosmos Pageant'''），简称'''中华小姐'''、'''华姐'''，是[[香港|香港]][[凤凰卫视|凤凰卫视]]举办的全球性选美盛事，于2003年举办首届。是“拉近全球华人距离、建立华人新生代美丽标准”的[[选美大赛|选美大赛]]，它推崇“美丽与智慧同行，内在与外在兼修”，宣导“美丽与活力、勇敢与智慧、个性与爱心”的华人新女性形象的树立，是唯一由电视媒体独家主办的时间跨度最长、空间跨度最大，覆盖全球的历时半年的选美比赛。大赛的冠亚季殿军均有在[[凤凰卫视|凤凰卫视]]工作的机会（例如主持人）。

2008年因[[汶川大地震|汶川大地震]]，当年没有举行赛事，改以“中华小姐环球关爱行动”取代。<ref>[http://phtv.ifeng.com/zhonghuaxiaojie/news/200808/0828_4028_752270.shtml “中华小姐关爱行动”完成灾后援建志愿服务]</ref>

==冠名==
每年赛事的冠名商均不同，2013年起，该赛事由[[广东长隆集团|广东长隆集团]]冠名。

==赛程与赛制==
===参赛者资格===
所有参赛者须符合以下所有条件：<ref>[http://baoming.miss.ifeng.com/signup.php?_c=info&_a=agreement 参赛须知]</ref><ref>[http://miss.ifengus.com/?q=content/regulation-chinese 2014 中華小姐環球大賽北美賽區參賽規則]</ref>
# 參賽者必須為從未有過任何形式的[[婚姻|婚姻]]、從未有過生育的女性自然人； 
# 參賽者必須為14－28周歲（以参赛年10月1日為準，即出生日期在1986年10月1日—1996年10月1日之間）；
# 在[[中国大陆|中国大陆]]、[[香港|香港]]、[[澳门|澳门]]、[[台湾|台湾]]以外的出生的參賽者必須擁有華人血統；
# 參賽者必須持有有效[[身份證件|身份證件]]及[[旅遊證件|旅遊證件]]；
# 參賽者必須有良好的外型基礎，身體健康，身高在1.65m以上，身材勻稱，五官端正；
# 參賽者必須受過良好的正規教育，[[大專|大專]]以上學歷或在讀，具備良好的交流與溝通能力；
# 參賽者必須無任何不良或[[犯罪|犯罪]]記錄，且無裸體照片刊登於任何[[成人雜誌|成人雜誌]]及互聯網上。
# 參賽者必须保证报名信息真实有效，如发现有不实情况，一律取消参赛资格。

另外，凤凰卫视员工及家属都不得参加本次比赛，以示公允。

===赛区===
大赛共分为以下赛区：
*{{CHN}}：[[北京|北京]]、[[深圳|深圳]]（2016年改为[[珠海|珠海]]）
*网络赛区
*中國內地賽區 
*加拿大賽區
*海外赛区：[[欧洲|欧洲]]、[[美洲|美洲]]、[[澳大利亚|澳大利亚]]、[[新西兰|新西兰]]、[[东南亚|东南亚]]、[[非洲|非洲]]（2015年起停办）（部分海外赛区设有子赛区）
其中欧洲赛区和美洲赛区由凤凰卫视当地分台直接主办，其他赛区则由当地机构经凤凰卫视授权主办。

===选拔===
赛事的选拔的程序如下：
# 组委会将对报名资料进行初选，如获通过，将正式获得参赛资格；
# 组委会将会通过官方网站以及电子邮件等形式通知参赛者面试的具体时间、地点及面试细节；
# 组委会将在面试后另行公告及通知入围下一阶段赛事的参赛选手；
# 通过初试后入围集中阶段赛事的参赛者须另行签订由主办方拟订之参赛合约，方可获得后续各阶段的完整参赛资格。

===赛程===
每年比赛的赛制如下：
# [[报名|报名]]：5-6月（有时会延期至7月初）
# [[面试|面试]]：7月
# [[夏令营|夏令营]]：8月
# 海外之旅：9月中旬
# 公益之旅：9月下旬（2015年已取消）
# [[总决赛|总决赛]]晚会：10月

在面试期间，选手需要通过各级赛区的面试和决赛，才能进入接下来的赛程。
