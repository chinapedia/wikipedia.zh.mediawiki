'''中油嘉義溶劑廠支線'''是自[[北回歸線車站|北回車站]]分歧，由[[台灣中油股份有限公司|中油]]專用鐵路貨運支線。

== 路線資料 ==
*經營管轄：中國石油公司（今[[台灣中油公司|台灣中油公司]]）
*路線距離：北回車站＝嘉義中油煉製研究所
*[[軌距|軌距]]：1,067毫米
*車站數：2（含起訖車站）
*開業時間：[[台灣日治時期|日治時代]]
*廢止時間：2003年
*雙線區間：無，全線單線 
*電化區間：無

== 運行形態 ==
*已停駛。（目前部分路段已改建為自行車道，名稱為嘉油鐵馬道）

== 使用車輛 ==
*1.日本新潟重工製造之D-1，1961年製，(今日已消失)
*2.日本日立重工製造之D-2，1980年製
*3.美國plymouth製造之D-3，1989年製 <ref>《台灣鐵路環島風情-特殊路線篇》，2004年，蘇昭旭著，人人出版股份有股份有限公司 </ref>

== 簡介 ==
本線於日治時代由[[台灣拓殖化學會社|台灣拓殖化學會社]]建造。從北回歸線車站向北分歧而出，平行於縱貫線鐵路之東側而北行，後向東進入市區，穿越[[嘉義市|嘉義市]]世賢路，終止於嘉義中油煉製研究所（地方上常稱之為溶劑廠）。亦有稱為「中油煉研所線」者。2002年8月14日停駛<ref>鄧志忠/古庭維《臺灣舊鐵道散步地圖》，2010，頁112</ref>。
