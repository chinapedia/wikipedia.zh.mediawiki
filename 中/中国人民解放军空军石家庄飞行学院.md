'''中国人民解放军空军石家庄飞行学院'''，简称'''空军石家庄飞行学院'''，是空军初级指挥院校，隶属[[中国人民解放军中部战区空军|中国人民解放军中部战区空军]]。

==沿革==
;空军第四飞行学院
*1949年10月6日，中央军委决定，在[[哈尔滨|哈尔滨]]、[[长春|长春]]、[[锦州|锦州]]、[[沈阳|沈阳]]、[[济南|济南]]、[[北京|北京]][[南苑|南苑]]各组建一所航空学校<ref name=lnszjsz> {{Cite book | author = 辽宁省地方志编纂委员会办公室主编 | title = 辽宁省志·军事志 | location =  | publisher =  辽宁科学技术出版社| date = 1999年 | pages = 289 | ISBN =  | accessdate =  | url =  | language =  | quote =  }} </ref>。
*1949年12月1日，在[[沈阳|沈阳]]北陵机场组建'''中国人民解放军第二驱逐机学校'''<ref name=disi>{{Cite web |url =  http://www.mod.gov.cn/service/2009-09/09/content_4086523.htm|title = 飞行学院介绍：空军第四飞行学院 |author =  |date = 2009-09-09 |publisher = 中华人民共和国国防部 |language =  |accessdate =  |quote =  }}</ref>。
*1949年12月20日，中央军委批复空军，同意将沈阳的第二驱逐机学校定名'''中国人民解放军第四航空学校'''，执行师级权限<ref name=lnszjsz/>。1949年12月24日正式更名<ref name=disi/>。
*1950年12月，迁址[[山西省|山西省]][[太原市|太原市]]<ref name=lnszjsz/>。
*1952年12月，迁址[[河北省|河北省]][[石家庄市|石家庄市]]。
*1976年，更名'''中国人民解放军空军第四航空学校'''。
*1986年10月，更名'''中国人民解放军空军第四飞行学院'''<ref name=disi/>。

;空军第六飞行学院
*1949年12月1日，在[[北京|北京]]组建'''中国人民解放军第四驱逐机学校'''<ref name=diliu>{{Cite web |url =  http://www.mod.gov.cn/service/2009-09/09/content_4086526.htm|title = 飞行学院介绍：空军第六飞行学院 |author =  |date =  2009-09-09 |publisher = 中华人民共和国国防部 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://pic.people.com.cn/GB/73694/6243603.html |title = 组图：北京空军第六航空学校历史照片 |author =  |date =2007-09-10  |publisher = 人民网 |language =  |accessdate =  |quote =  }}</ref>。1949年12月11日开学。
*1949年12月24日，更名'''中国人民解放军第六航空学校'''。后迁至[[河北省|河北省]][[涿县|涿县]]（今[[涿州市|涿州市]]）<ref name=diliu/>。
*1976年6月，更名'''中国人民解放军空军第六航空学校'''<ref name=diliu/>。
*1986年，更名'''中国人民解放军空军第六飞行学院'''<ref name=diliu/>。
*1992年8月，空军第六飞行学院与空军第十二飞行学院合并为新的'''中国人民解放军空军第六飞行学院'''<ref name=diliu/>。

;空军第十二飞行学院
*1952年9月30日，经空军党委批准，以[[川南军区|川南军区]]机关为基础，筹建临汾航空学校。
*1952年12月12日，军委空军转[[中央人民政府人民革命军事委员会|中央军委]]12月7日电令，正式命名临汾航校为'''中国人民解放军第十二航空学校'''，代号空字〇二七部队，正师级建制。受军委空军直接领导。校部设在[[山西省|山西省]][[临汾市|临汾市]]。
*1976年，更名'''中国人民解放军空军第十二航空学校'''。
*1986年，更名'''中国人民解放军空军第十二飞行学院'''。
*1992年8月，空军第十二飞行学院与空军第六飞行学院合并为新的空军第六飞行学院，空军第十二飞行学院的番号撤销<ref name=diliu/>。

;空军石家庄飞行学院
*2011年，解放军四总部联合召开第十六次全军院校会议，决定将空军6所飞行学院整合组建为空军哈尔滨、石家庄、西安3所飞行学院<ref>{{Cite web |url =http://theory.people.com.cn/GB/16812836.html  |title =中国军校改革背后的战略指向  |author =  邱蜀林|date = 2012-01-06 |publisher =中国共产党新闻网  |language =  |accessdate =  |quote =  }}</ref>。
*2012年4月，由空军第四飞行学院、空军第六飞行学院等单位合并组建'''中国人民解放军空军石家庄飞行学院'''<ref name=baijiasi>{{Cite web |url = http://baijiahao.baidu.com/s?id=1563845550966765&wfr=spider&for=pc |title = 那些年我们一起上过的军校——石家庄飞行学院 |author =  |date = 2017-04-05 |publisher = 百家号 |language =  |accessdate =  |quote =  }}</ref>。是初级指挥飞行军官任职教育院校<ref>{{Cite web |url =http://www.mod.gov.cn/reports/201403/wzry/2014-09/12/content_4536769.htm  |title =空军石家庄飞行学院  |author =  |date =2014-09-12   |publisher =中华人民共和国国防部  |language =  |accessdate =  |quote =  }}</ref>。副军级<ref>{{Cite web |url =http://news.sina.com.cn/o/2012-01-21/112623835432.shtml  |title = 港媒:空军院校体系趋成熟折射空军发展新走向 |author =  |date = 2012-01-21 |publisher = 新浪 |language =  |accessdate =  |quote =  }}</ref>。隶属[[中国人民解放军北京军区空军|中国人民解放军北京军区空军]]。
*2016年，在[[深化国防和军队改革|深化国防和军队改革]]中，由[[中国人民解放军北京军区空军|中国人民解放军北京军区空军]]转隶[[中国人民解放军中部战区空军|中国人民解放军中部战区空军]]<ref>{{Cite web|url=http://news.ifeng.com/a/20160629/49261251_0.shtml|title=韩合机场空域使用协调会召开|accessdate=2017-08-31|author=|date=2016-06-29|work=|publisher=凤凰网}}</ref>。
*2017年，在[[深化国防和军队改革|深化国防和军队改革]]中，调整组建新的'''中国人民解放军空军石家庄飞行学院'''<ref>{{Cite web |url =http://www.mod.gov.cn/shouye/2017-06/29/content_4783975.htm  |title = 国防部公布调整改革后军队院校名称 |author =  |date =2017-06-29  |publisher = 中华人民共和国国防部 |language =  |accessdate =  |quote =  }}</ref>。
