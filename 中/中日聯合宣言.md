《'''中日关于建立致力于和平与发展的友好合作伙伴关系的联合宣言'''》（{{lang-ja|平和と発展のための友好協力パートナーシップの構築に関する日中共同宣言}}），简称《'''中日联合宣言'''》、《'''日中联合宣言'''》，為1998年11月26日，時任中共中央總書記、中國國家主席[[江澤民|江澤民]]訪問[[日本|日本]]，與時任日本內閣總理大臣[[小淵惠三|小淵惠三]]舉行峰會後共同發表的宣言。这份文件确立了双方的“友好合作伙伴关系”，也是中日四个重要政治文件中唯一一份没有双方签名的文件<ref name="xuebao">{{cite journal |author1=张耀武 |title=中日关系中四个政治文件的特色及其意义 |journal=大连大学学报 |date=2012-10 |volume=33 |issue=5 |pages=58-62 |accessdate=2018-11-17 |issn=1008-2395}}</ref>。

== 背景 ==
1998年11月25日至30日，正值《[[中日和平友好条约|中日和平友好条约]]》缔结20周年之际，时任[[中华人民共和国主席|中华人民共和国主席]][[江泽民|江泽民]]受邀访问[[日本国|日本国]]，系江泽民出任[[中共中央总书记|中共中央总书记]]后第二次访日。在访问期间，江泽民受到[[明仁天皇|明仁天皇]]和[[美智子皇后|美智子皇后]]会见<ref>{{cite web |title=1998年11月25日：江泽民主席对日本进行正式访问 |url=http://www.cnr.cn/2008zt/hjtfr/xwrl/lshg/200804/t20080429_504775741.html |website=中广网 |publisher=新华社 |accessdate=2018-11-17 |archive-url=https://web.archive.org/web/20140718062416/http://www.cnr.cn/2008zt/hjtfr/xwrl/lshg/200804/t20080429_504775741.html# |archive-date=2014-07-18 |dead-url=no }}</ref>，并受邀在[[早稻田大学|早稻田大学]]做了题为《以史为鉴　开创未来》的演讲<ref>{{cite web |author1=江泽民 |title=以史为鉴　开创未来——在日本早稻田大学的演讲 |url=http://www.people.com.cn/item/ldhd/Jiangzm/1998/jianghua/jh0104.html |website=人民网 |accessdate=2018-11-17 |date=1998-11-28 |archive-url=https://web.archive.org/web/20041209060825/http://www.people.com.cn/item/ldhd/Jiangzm/1998/jianghua/jh0104.html# |archive-date=2004-12-09 |dead-url=no }}</ref>。

江泽民等人也与[[日本内阁总理大臣|日本内阁总理大臣]][[小渊惠三|小渊惠三]]等人举行会谈，深入交换了双方的观点。会上，中日双方都同意中日关系对两国均为最重要的双边关系之一，并深刻认识到双方在和平与发展方面的作用与责任，宣布面向21世纪，建立致力于和平与发展的友好合作伙伴关系。会后，双方发布了这份宣言。

就双方没有均签署这份文件的问题，[[共同社|共同社]]记者在11月28日的记者会上询问江泽民，江泽民表示，双方都未考虑过要签署这份宣言，但他认为，公布的宣言是双方的庄严承诺，应当遵守<ref name="zdh">{{cite news |title=资料：江泽民主席在日本举行记者招待会 |url=http://news.cri.cn/gb/18824/2008/04/30/2185@2039624.htm |accessdate=2018-11-17 |agency=新华社 |publisher=国际在线 |date=1998-11-28}}</ref>。

== 具体内容 ==
《中日联合宣言》分为四段。序言首先介绍了江泽民访日及其背景。第一章主要介绍双方的安全观，重申双方交往基于[[和平共处五项原则|和平共处五项原则]]和《[[联合国宪章|联合国宪章]]》的基本准则，表达了支持[[联合国安理会|联合国安理会]]改革、支持[[核裁军|销毁核武器]]、反对[[核扩散|核扩散]]与[[核试验|核试验]]、加强双方交流合作的四项主张。第二章主要表述了双方对于亚太地区形势的观点，双方均认为亚洲将会在世界舞台上起到更加重要的作用，双方支持维护地区和平并主张以和平方式解决争端，不行使武力或以武力相威胁，同时双方也对[[亚洲金融危机|亚洲金融危机]]表示关切，并表示将积极参与[[东盟地区论坛|东盟地区论坛]]等地区协作机制，支持一切增进双方互信的措施。在第三章中，双方宣布面向21世纪，建立致力于和平与发展的友好合作伙伴关系，双方重申恪守《[[中日联合声明|中日联合声明]]》和《[[中日和平友好条约|中日和平友好条约]]》，以正视过去、正确认识历史为发展关系的重要基础，双方确信增强人员往来对增进互信十分重要；此外，拟双方设置专线电话，并将加强双方的对话机制，促进双方的青少年合作；双方决定在平等互利基础上，建立长期稳定的经贸合作关系，进一步拓展在高新科技、信息、环保、农业、基础设施等领域的合作。此外，日方重申了[[一个中国原则|一个中国原则]]，表示支持中国加入[[世界贸易组织|世界贸易组织]]，并将遵守[[村山谈话|村山谈话]]精神，对过去的历史表示反省。

== 影响 ==
以这一联合宣言的精神为基，双方签署了《[[:s:中日关于进一步发展青少年交流的框架合作计划|中日关于进一步发展青少年交流的框架合作计划]]》、《[[:s:中日面向21世纪的环境合作的联合公报|中日面向21世纪的环境合作的联合公报]]》以及《[[:s:中日关于在科学与产业技术领域开展交流与合作的协定|中日关于在科学与产业技术领域开展交流与合作的协定]]》，促进青少年交流、环境合作和科学与产业技术合作；并发布了《[[:s:中日两国关于面向21世纪加强合作的联合新闻公报|中日两国关于面向21世纪加强合作的联合新闻公报]]》。其中，《科技领域交流合作协定》中涉及的新干线等技术的出口问题，为后来[[中国高铁|中国高铁]]的发展创造了条件<ref>{{cite book|author=毛利和子|title={{lang|ja|日中関係　戦後から新時代へ}}|year=2006|publisher=岩波新書|page=153}}</ref>。在这份文件中，双方就致力于和平与发展的友好合作伙伴关系这一点，重新定位双方关系为“友好合作伙伴关系”，确定了中日关系于二十一世纪发展的方向；日方也首次同意以“侵略”二字来描述近代的[[中日战争史|中日战争史]]，重申遵守[[村山谈话|村山谈话]]精神<ref name="xuebao" />。
