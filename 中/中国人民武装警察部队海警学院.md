'''中国人民武装警察部队海警学院'''，简称'''海警学院'''，位于[[浙江省|浙江省]][[宁波市|宁波市]][[北仑区|北仑区]]小港红联，是[[中国人民武装警察部队海警总队|中国人民武装警察部队海警总队]]最高学府，被誉为“共和国边防警官的摇篮”，编入[[中国人民武装警察部队|中国人民武装警察部队]]战斗序列，为[[副军级|副军级]]<ref>{{Cite web|url =http://www.gahjxy.com/|title =公安海警学院简介|author =|date =|publisher =公安海警学院|language =|accessdate =2017-04-06|quote =}}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

海警学院是经[[中华人民共和国教育部|教育部]]备案的独立招生[[普通高等学校|普通高等学校]]。

== 沿革 ==
*1982年，创建'''中国人民武装警察部队浙江省总队海巡训练大队'''（'''武警浙江总队海巡训练大队'''），隶属[[中华人民共和国公安部|公安部]]<ref>{{Cite web |url = http://mini.eastday.com/a/170606181121229.html |title = 这所警校是经教育部批准成立的公安部直属现役制普通高校！ |author =  |date =2017-06-06   |publisher = 东方头条 |language =  |accessdate =  |quote =  }}</ref>，正团级<ref name=xuzhu/>。
*1983年9月，[[中华人民共和国国务院|国务院]]下达国发〔1983〕149号文件，批准在原武警浙江总队海巡训练大队的基础上筹建'''中国人民武装警察部队水面船艇学校'''（'''武警水面船艇学校'''），直属[[武警总部|武警总部]]，正师级。当时武警浙江总队海巡训练大队实际上还未正式建立起来。此后在[[镇海县|镇海县]]（现学校驻地宁波市北仑区，时属镇海县管辖）开始筹建，筹备组23人，下设营建、教学、政工、后勤四个小组<ref name=xuzhu>许文军、朱健，海警学院27年变迁，人民公安2010(15)</ref>。
*1985年9月1日正式开学<ref name=xuzhu/>，暂时由[[中国人民武装警察部队浙江省总队|中国人民武装警察部队浙江省总队]]代管，执行[[副师级|副师级]]权限。
*1993年8月，转隶[[中华人民共和国公安部|公安部]]，编入[[公安边防部队|公安边防部队]]，更名'''公安边防水面船艇学校'''<ref name=xuzhu/>。
*1997年9月，中止中专招生，全面实施大专教学<ref name=xuzhu/>。
*1999年3月，[[中华人民共和国教育部|教育部]]正式批准建立'''公安海警高等专科学校'''<ref name=xuzhu/>。
*2009年12月，接受全国高等学校设置评议委员会专家组的考察。2010年1月21日，全国高评委委员在[[广西|广西]][[南宁|南宁]]举行五届四次会议，学校以65票（总票数67票）的全国第一高票（共45所学校）获得通过，会议同意“在公安海警高等专科学校基础上建立公安海警学院”<ref name=xuzhu/><ref>{{cite news |author1 = 袁金龙 |author2 = 朱健 |title = “中国海警摇篮”27年培养人才2万多名——公安海警学院成为公安边防部队教育科研重要基地 |newspaper =[[人民公安报|人民公安报]] |date = 2010-06-25 }}</ref>。
*2010年6月26日，升格成立本科院校'''公安海警学院'''，成为中华人民共和国唯一专门培养海上执法人才的现役院校，为[[公安边防部队|公安边防部队]]最高学府，国务委员、公安部部长[[孟建柱|孟建柱]]出席成立大会并为学院揭牌<ref name=xuzhu/>。公安海警学院直属于[[中华人民共和国公安部|中华人民共和国公安部]]，由[[公安部边防管理局|公安部边防管理局]]管理。
*2018年7月1日，根据《[[深化党和国家机构改革方案|深化党和国家机构改革方案]]》和《[[武警部队改革实施方案|武警部队改革实施方案]]》，海警队伍轉為由[[武警|武警]]部队领导指挥，學校更名為'''中国人民武装警察部队海警学院'''。<ref>{{Cite web |url =http://quan.cnnb.com/wap/thread/view-thread/tid/7248512?random=542881|title = 公安海警学院正式更名为中国人民武装警察部队海警学院|author =  阿拉播报|date =2018-08-08  |publisher = 阿拉宁波网 |language =  |accessdate = 2018-11-19 |quote =  }}</ref>

==专业设置==
2017年招生的本科专业有：
*航海技术专业（舰艇指挥方向）
*武器发射工程专业（海警舰载武器指挥方向）
*治安学专业（治安管理方向）
*法学专业（公安现役法制方向）
*轮机工程专业（机电技术指挥方向）
*信息安全专业（公安机要方向）
*电子信息工程专业（信息指挥方向）
*通信工程专业（通信指挥方向）
*管理科学专业（后勤管理方向）<ref>{{Cite web |url = http://www.sohu.com/a/192695165_567494 |title = 2018高考军校报考：公安海警学院专业解读 |author =  |date = 2017-09-18 |publisher =搜狐  |language =  |accessdate =  |quote =  }}</ref>
