'''中野'''（{{jpn|j=中野|hg=なかの|rm=Nakano|f=y}}）是[[東京都|東京都]][[中野區|中野區]]的町名，下分中野一丁目至中野六丁目，2013年10月1日為止的人口有25,386人<ref>[http://www.city.tokyo-nakano.lg.jp/dept/101500/d015810_d/fil/chouchou201310.xls 中野区町丁別人口　平成25年（2013年）10月1日現在]</ref>。[[郵遞區號|郵遞區號]]164-0001。

== 地理 ==
位於中野區中央。町內有{{link-ja|中野區役所|中野区役所}}、[[中野SUNPLAZA|SUNPLAZA]]、{{link-ja|中野SUNMALL商店街|中野サンモール商店街|SUNMALL}}、[[中野百老匯|中野百老匯]]、[[中野站_(東京都)|中野站]]前商店街等。

東鄰中野區[[東中野_(中野區)|東中野]]，南與中野區[[中央_(中野區)|中央]]之間以[[大久保通|大久保通]]為界，北與中野區[[新井_(中野區)|新井]]、[[野方|野方]]、[[上高田|上高田]]之間以{{link-ja|早稻田通|早稲田通り}}為界，西接[[杉並區|杉並區]][[高圓寺南|高圓寺南]]、[[高圓寺北|高圓寺北]]。

中野站可搭乘[[中央線快速|中央線快速]]、[[中央、總武緩行線|中央、總武緩行線]]、[[東京地下鐵東西線|東京地下鐵東西線]]，人潮眾多。此外中野站前是{{link-ja|中野通|東京都道420号鮫洲大山線}}。

另一方面，車站周邊以外以住宅區為主，在此可見獨戶住宅、公寓等。此地屬木賃帶的一部分，也可見到單身年輕人住在[[土木結構建築|木造]]房屋裡。

此外，站北口[[警視廳|警視廳]]中野警察學校舊址周邊進行大規模再開發事業，2012年{{link-ja|中野四季的都市|中野四季の都市}}開幕。

== 歷史 ==
=== 地名由來 ===
意為武藏野'''中'''的原'''野'''。

=== 町名的變遷 ===
中野在1966年（[[昭和|昭和]]41年）10月1日實施住居表示。
* 中野一丁目 = 宮園通一丁目與三丁目的部分區域、城山町
* 中野二丁目 = 宮園通三丁目與四丁目的部分區域、千光前町、中野站前的部分區域
* 中野三丁目 = 宮園通四丁目與五丁目的部分區域、中野站前的部分區域、桃園町
* 中野四丁目 = 圍町、新井町的部分區域、野方町二丁目的部分區域
* 中野五丁目 = 打越町、天神町、昭和通三丁目的部分區域、新井町的部分區域
* 中野六丁目 = 文園町、昭和通二丁目的部分區域
