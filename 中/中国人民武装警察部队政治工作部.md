'''中国人民武装警察部队政治工作部'''，位于[[北京市|北京市]]，是[[中国人民武装警察部队|中国人民武装警察部队]]总部机关职能部门，负责中国人民武装警察部队政治工作。

==沿革==
1983年1月，中国人民武装警察部队总部成立，总部机关设有司令部、政治部、后勤部。中国人民武装警察部队总部机构原定是正兵团级，后来改为[[副大军区级|副大军区级]]，其司令部、政治部、后勤部为正军级。1995年12月，中央军委决定：中国人民武装警察部队总部从副大军区级调整为[[正大军区级|正大军区级]]，中国人民武装警察部队总部司令部、政治部从正军级调整为副大军区级<ref name=wjzl> {{Cite book | author =  中共中央组织部、中共中央党史研究室、中央档案馆 编  | title = 中国共产党组织史资料 附卷二 中国人民解放军组织（1949.10—1997.9） | location = 北京 | publisher = 中共党史出版社 | date =2000年  | pages =425-432  | ISBN =  | accessdate =  | url =  http://www.hprc.org.cn/gsw/detail_zzslk.jsp?channelid=75034&record=3238| language =  | quote = 下部　社会主义事业发展新时期（1976.10—1997.9） 第六章　中国人民武装警察部队及武警部队党委 }} </ref><ref>{{Cite web|url = http://news.21cn.com/junshitiandi/2003/07/30/1202960.shtml|title = 中国人民武装警察部队|author = |date = 2003-07-30|publisher = 21CN新闻|language = |accessdate = |quote = |archive-url = https://web.archive.org/web/20071012184336/http://news.21cn.com/junshitiandi/2003/07/30/1202960.shtml|archive-date = 2007-10-12|dead-url = yes}}</ref>。

2016年改革前，中国人民武装警察部队政治部下设：组织部、干部部、宣传部、联络部、群工部、人民武警出版社<ref>{{Cite web|url = http://www.takungpao.com/paper/content/2012-10/28/content_1302483.htm|title = 武警总员额120万|author = |date = 2012-10-28|publisher = 大公报|language = |accessdate = |quote = }}{{Dead link|date=2018年8月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

2016年1月1日，[[新华社|新华社]]发布《中央军委关于[[深化国防和军队改革|深化国防和军队改革]]的意见》，其中将推进武警部队改革列为2016年改革任务之一，2016年年底前将“基本完成阶段性改革任务”。2016年3月初，武警部队开始按新体制编制运行，武警部队机关由之前的司令部、政治部、后勤部改为四部委（参谋部、政治工作部、后勤部、纪律检查委员会）<ref>{{Cite web |url = http://www.guancha.cn/military-affairs/2016_06_05_362975.shtml |title =2016军改最新消息：武警部队总部机关已由原三大部改为四部委  |author =  |date = 2016-06-05 |publisher =观察者网  |language =  |accessdate =  |quote =  }}</ref>。
