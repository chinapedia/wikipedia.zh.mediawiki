'''中島飛機'''（{{lang-ja|中島飛行機}}）為1917年至1945年間曾存在的[[日本|日本]][[飛機|飛機]]製造商，由[[大日本帝國海軍|大日本帝國海軍]]退役的[[中島知久平|中島知久平]]所創立，在[[第二次世界大戰|第二次世界大戰]]中乃日本主要的[[發動機|發動機]]、[[戰鬥機|戰鬥機]]研發製造商，從設計到生產可一貫化作業，在二戰結束前超越[[三菱財閥|三菱財閥]]飛機生產部門，成為日本最大、世界上數一數二的飛機生產商。

== 沿革 ==
* 1917年（[[大正|大正]]6年）12月21日：中島知久平自海軍退役後，在[[群馬縣|群馬縣]]{{link-ja|尾島町|尾島町}}（今[[太田市|太田市]]）和親兄弟中島喜代一、中島乙未平等人設立'''飛機研究所'''（{{lang-ja|飛行機研究所}}）<ref>後來成為吞龍工廠。</ref>，是為第二家民營飛機製造商。 
* 1918年（大正7年）：[[川西清兵衛|川西清兵衛]]加入經營，並變更公司名稱成'''日本飛機製作所'''（{{lang-ja|日本飛行機製作所}}）。 
* 1919年（大正8年）：4月收到[[大日本帝國陸軍|陸軍]]委託製造20架飛機。川西清兵衛因故與中島發生爭執，憤而在10月底求去。11月27日中島也提出辭呈，擬退出經營階層而轉作技術職，但兩天後新田銀行通過10萬[[日圓|日圓]]融資案，仍保留所長現職；11月30日通過{{link-ja|井上幾太郎|井上幾太郎}}的介紹，往來銀行改成[[三井物產|三井物產]]，並在12月26日易名為'''中島飛機製作所'''（{{lang-ja|中島飛行機製作所}}）。
* 1920年（大正9年）：4月陸軍委製70架、海軍30架。
* 1925年（大正14年）：11月東京工廠落成啟用。
* 1931年（[[昭和|昭和]]6年）：再度改名為'''中島飛機株式會社'''（{{lang|ja|中島飛行機株式会社}}），[[授權資本|資本額]]增為600萬日圓。
* 1937年（昭和12年）：[[九七式戰鬥機|九七式戰鬥機]]成為陸軍制式飛機。
* 1938年（昭和13年）：武藏工廠落成，專門組裝[[星型引擎|星型引擎]]。
* 1940年（昭和15年）：小泉工廠成立，專為海軍組裝戰鬥機，並利用[[小泉線|東武小泉線]]{{link-ja|西小泉車站|西小泉駅}}運輸。
* 1941年（昭和16年）：設計生產之[[一式戰鬥機|一式戰鬥機]]經陸軍正式採用。 
* 1942年（昭和17年）：開始與興亞工業大學（[[千葉工業大學|千葉工業大學]]前身）[[建教合作|建教合作]]。
* 1944年（昭和19年）：開設宇都宮工廠，專為陸軍組裝飛機，並開始生產制式戰機[[四式戰鬥機|四式戰鬥機]]；同年4月開始著手研發大型[[戰略轟炸機|戰略轟炸機]][[富嶽轟炸機|富嶽]]。因[[第二次世界大戰|二戰]]進入最後階段，12月該公司旗下所有工廠創下7,940架的最高生產數量。
* 1945年（昭和20年）：4月1日改編成為第一軍需工廠，雖沿用公司名稱，但事實上已收歸國營。由於日本戰敗，8月16日起將原先收歸國營的工廠全數歸還，並易名為'''富士產業株式會社'''，同時中島知久平辭去取締役社長之職，8月22日由中島乙未平就任社長。12月收到[[駐日盟軍總司令|駐日盟軍總司令]]（GHQ）的財閥解體指令，正式於1950年5月解散，共拆分成12家公司。 
 
日本戰敗之後，中島飛機總計共生產了29,925架各式飛機，其中由[[三菱重工業|三菱重工業]]設計的[[零式艦上戰鬥機|零戰]]，總數10,449架中約有2/3為該公司所製造。由於二戰時該公司各地工廠為美軍的主要轟炸目標，戰後多數工廠已殘破不堪，加上GHQ嚴禁該公司再研發飛機，許多技術工程師轉進汽車相關產業，對日本車壇貢獻良多。

== 解體後的發展 ==
GHQ下令中島飛機解體後，多數公司目前仍繼續經營：
* [[富士重工業|富士重工業]]：由東京富士產業（本社）、富士工業（太田吞龍工廠和武藏野三鷹工廠）、富士汽車工業（伊勢崎工廠）、大宮富士工業（大宮工廠）、宇都宮車輛（宇都宮工廠）等共同成立，「[[速霸陸汽車|速霸陸]]」為該公司的汽車[[品牌|品牌]]名稱。
* THK韻律：濱松工廠戰後改組成富士精密工業，在[[石橋正二郎|石橋正二郎]]（[[普利司通|普利司通]]輪胎創辦人）牽線下與[[王子汽車|王子汽車]]合併，1961年自王子汽車分離獨立成力知茂朋友製造（{{lang|ja|リズムフレンド製造}}），1967年易名力知茂汽車部品製造，1991年改名成力知茂公司，2007年機械廠[[THK_(有限公司)|THK]]買下並更名為{{link-ja|THK力知茂|THKリズム}}。
* 富士機械：由前橋工廠改組，曾更名為富士機器，再易名成現名，現為富士重工業之子公司。
* [[輸送機工業|輸送機工業]]：由半田製作所改組，1950年易名愛知富士産業，1953年再改成現名，現為富士重工業之子公司。
* 牧田沼津：三島工廠在戰後改組成富士機械工業，1966年更名成富士羅賓，2013年被[[母公司|母公司]][[牧田|牧田]]吸收合併。
* GKN：栃木工廠改組成栃木富士產業，後易名成GKN齒盤扭矩科技（{{lang|ja|GKNドライブライントルクテクノロジー}}），再更改成現名。
* 岩富士工業：戰後黑澤尻工廠改組成岩手富士產業，再變更成岩富士工業（{{lang|ja|イワフジ工業}}），現為[[新明和工業|新明和工業]]之子公司。
