'''中目黑'''（{{jpn|j=中目黒|hg=なかめぐろ|rm=Naka-meguro|f=y}}）是[[日本|日本]][[東京都|東京都]][[目黑區|目黑區]]一個地方名稱。
#目黑區東部的町名，下設中目黑1丁目至中目黑5丁目，人口14,222人<ref>[http://www.city.meguro.tokyo.jp/gyosei/tokei/tokei/jinko/setaisu/h25/files/mjinko-h2507.xls 町丁別世帯数および人口表　平成25年度（2013）]{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。
#廣義上的中目黑，是指以[[東急東橫線|東急東橫線]]、[[東京地下鐵日比谷線|東京地下鐵日比谷線]][[中目黑站|中目黑站]]（行政上之地名為「上目黑」）為中心，包括住居表示上的中目黑、上目黑、青葉台、東山一帶地區。

== 地理 ==
=== 住居表示的中目黑 ===
住居表示的中目黑，是指[[東急東橫線|東急東橫線]]東側[[東京都道416號古川橋二子玉川線|駒澤通]]的東部一帶，下設一丁目至五丁目。地區中央有[[東京都道317號環狀六號線|山手通]]、[[目黑川|目黑川]]。

=== 廣域地名的中目黑 ===
廣義上的中目黑是指以[[東急東橫線|東急東橫線]]、[[東京地下鐵日比谷線|東京地下鐵日比谷線]][[中目黑站|中目黑站]]（行政上之地名為「上目黑」）為中心的區域。

與澀谷、代官山一帶連接，這裡是最佳居住住宅區調查中排名甚高的地區。目黑區役所位於中目黑（行政上之地名為「上目黑」）、屬於目黑區的行政中心地域。

車站前再開發計劃正進行中，在上目黑二丁目地區的中目黑GT（[[中目黑GateTown|中目黑GateTown]]）於2002年4月開幕。中目黑GT由高層大廈（120m・地上25層）、Height棟（地上15階建）、Terrace棟（地上6階建）共3棟建築物構成。
2009年，上目黑一丁目地區的再開發計劃「Nkamearukas（[[中目黑Atlas_Tower|中目黑Atlas Tower]]）」完工。

== 地名由來 ==
來自舊[[荏原郡|荏原郡]][[中目黑村|中目黑村]]。

== 沿革 ==
*1889年（[[明治|明治]]）：22年「市制町村制」施行，為目黑村[[大字|大字]]中目黑。
*1922年（[[大正|大正]]11年）：為目黑町大字中目黑。
*1932年（[[昭和|昭和]]7年）：成立目黑町一～三丁目。
*1967年（昭和42年）：實施新住居表示，中目黑二丁目全部與一、三丁目、三田、上目黑一丁目各一部分合併為現行的'''中目黑'''。

== 交通 ==
經過中目黑的鐵道有[[東急東橫線|東急東橫線]]及地下鐵日比谷線。

東橫線由[[澀谷站|澀谷]]經中目黑至[[橫濱站|橫濱]]方向、日比谷線往[[六本木站|六本木]]、[[霞關站_(東京都)|霞關]]、[[銀座站_(日本)|銀座]]方向，只需短時間轉車，十分便利。

道路方面，此地是[[山手通|山手通]]（環狀六號線）與[[駒澤通|駒澤通]]交差的地點，交通便利。預定將來在山手通地下將開通[[首都高速道路中央環狀線|首都高速道路中央環狀線]]，但沒有在中目黑設出入口的計劃。
