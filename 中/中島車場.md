'''中島車場'''位於[[高雄市|高雄市]][[前鎮區|前鎮區]][[中島半島|中島半島]]旁，車場兩側為[[擴建路|擴建路]]與建基街，辦理出入[[前鎮加工出口區|前鎮加工出口區]]之貨物列車車頭及車輛調度等業務。原為[[臺灣鐵路管理局|臺灣鐵路管理局]][[高雄港站|高雄港站]]所轄之[[調車場|調車場]]，現已廢止。

== 車站構造 ==
{{章節擴充}}

== 利用狀況 ==
* 原為高雄臨港線的四處車場——[[前鎮車場|前鎮車場]]、'''中島車場'''、[[苓雅寮車場|苓雅寮車場]]與[[草衙車場|草衙車場]]之一。
* 現已廢止。

== 歷史 ==
* 1970年6月：中島車場設立。
* 2008年11月19日：第一臨港線西段（包括中島車場全站及中島線、苓雅寮車場全站、高雄港站全站及濱線）准予停用。
* 2008年12月16日：高雄港站裁撤，12月31日辦理交接<ref>[http://www.railway.gov.tw/Kaohsiung-Harbor/CP.aspx?SN=11550 高雄港車站-車站簡介]</ref>，原為高雄港站所屬之業務移至[[高雄車站|高雄車站]]管理。
* 2011年7月12日：中島車場正式裁撤。
