{{Infobox Bilateral relations|中華民國與貝南|ROC|BEN|filetype=png}}

'''中華民國與貝南關係'''是指[[中華民國|中華民國]]與[[貝南|貝南共和國]]（1975年之前稱'''[[達荷美共和國|達荷美]]'''）之間的關係。兩國於1962－1973年有官方[[外交#外交上的承認|外交關係]]但曾經中斷，二次斷交後，目前沒有在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。對貝南的相關事務由'''{{Link-en|駐奈及利亞聯邦共和國臺北貿易辦事處|Taipei Trade Office in the Federal Republic of Nigeria}}'''、'''[[駐法國台北代表處|駐法國台北代表處]]'''{{Link-en|多重委派|Dual accreditation|共同管轄}}。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-235-01fd3-03-1.html |title = 駐館位置及聯絡方式 |publisher = [[中華民國外交部|中華民國外交部]] [[外交部領事事務局|領事事務局]]}}</ref>

==政治==
===外交===
[[File:Voting_res_2758.png|thumb]]，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》，其中包括當時的[[達荷美共和國|達荷美共和國]]。]]
{{see also|中華民國外交|{{link-en|貝南外交|Foreign relations of Benin}}}}
1962年1月18日，建立大使級[[外交#外交上的承認|外交關係]]。於首都[[波多諾伏|新港]]設立'''中華民國駐[[達荷美共和國|達荷美共和國]][[外交代表機構#大使馆|大使館]]'''，並派駐[[特命全權大使|大使]]。

1964年11月12日，與中華民國首次斷交。

1966年4月21日，恢復大使級外交關係。

1973年1月19日，與中華民國第二次斷交、關閉大使館，至今邦交未曾恢復。

===簽證===
[[File:Visa_requirements_for_Taiwanese_citizens_(6).png|thumb]]的[[中華民國國民|中華民國公民]]必須申請[[簽證|簽證]]方可入境貝南。]]
[[File:Visa_requirements_for_Beninese_citizens.png|thumb]]。]]
{{see also|中華民國簽證政策|中華民國公民簽證要求}}
{{see also|{{link-en|貝南簽證政策|Visa policy of Benin}}|{{link-en|貝南公民簽證要求|Visa requirements for Beninese citizens}}}}
兩國公民皆須申請[[簽證|簽證]]方可入境對方國家。持[[中華民國護照|中華民國護照]]的[[中華民國國民|中華民國公民]]可前往貝南駐[[巴黎|巴黎]][[領事館|總領事館]]申請簽證。入境須出示[[黃熱病|黃熱病]]與[[腦膜炎|腦脊髓膜炎]]的[[疫苗接種|疫苗接種]]證明（黃皮書）。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-235-01fd3-02-1.html |title = 簽證及入境須知 |publisher = 外交部領事事務局}}</ref>

持{{link-en|貝南護照|Beninese passport}}的{{link-en|貝南人口|Demographics of Benin|貝南公民}}抵台參加國際會議、運動賽事、商展活動，則可以[[中華民國簽證政策#电子签证|電子簽證入境中華民國]]，停留最多30天並不得延期。<ref>{{Cite web |url = https://www.boca.gov.tw/cp-228-4488-8208a-1.html |title = 誰具備申請電子簽證的資格? |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-215-643-5e740-1.html |title = 我的電子簽證效期多久？|publisher = 外交部領事事務局}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將貝南歸類為'''橙色警戒'''：避免非必要旅行（2016年10月27日）；<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-123-faa9e-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>[[中華民國衛生福利部|衛生福利部]]的國際間旅遊疫情建議等級表將貝南歸類為'''第一級注意'''：提醒遵守當地的一般預防措施（2018年2月13日）。<ref>{{Cite web |url = http://www.cdc.gov.tw/InternationalTravel/CountryInfo.aspx?treeid=aa2d4b06c27690e6&nowtreeid=fb6a405b1d67aa69&tid=971D6DDB49AAD20F |title = 國際重要疫情資訊 |publisher = [[中華民國衛生福利部|中華民國衛生福利部]]}}</ref>

==經濟==
{{see also|臺灣經濟|{{link-en|貝南經濟|Economy of Benin}}}}
===貿易===
2017年，貝南是中華民國第157大貿易夥伴、第135大[[進口|進口]]夥伴、第148大[[出口贸易|出口]]夥伴。出口至貝南的金額為362萬1,676[[美元|美元]]，年增17.236%；自貝南進口的金額為353萬2,417美元，年增20.138%。貿易呈現[[淨出口|出超]]（[[盈餘|盈餘]]）8萬9,259美元，年減40.062%。※2016年，貝南是中華民國第162大貿易夥伴、第133大進口夥伴、第156大出口夥伴。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>
