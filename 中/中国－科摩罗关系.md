'''中國－科摩羅關係'''，是指[[中國|歷史上的中國]]和[[科摩羅歷史|科摩羅]]、以至[[中华人民共和国|中华人民共和国]]和[[科摩羅聯盟|科摩羅聯盟]]之間的雙邊關係。根據[[考古|考古]]發現和一些需要查證的說法，中國與科摩羅在古代已有聯繫；現代的科摩羅在1975年11月13日與中華人民共和國建立外交關係，中華人民共和國由此成為首個與科摩羅建交的國家。兩國現有經貿和文化上的交流，中國亦對科摩羅施以援助。

== 歷史 ==
=== 古代 ===
中國與科摩罗之間的聯繫可追溯至古代，科摩罗曾發現[[唐朝|唐朝]]時的[[瓷|瓷片]]<ref>{{cite book|author1=林晓东|author2=巫秋玉|title=郑和下西洋与华侨华人文集|publisher=中国华侨出版社|year=2005|isbn=9787802220430}}</ref>{{Rp|518}}，當地島嶼[[昂儒昂島|昂儒昂島]]也出土了兩片[[南宋|南宋]]時的中國[[青瓷|青瓷]]<ref>{{cite book|author1=李丽 (編)|author2=肖小勇|title=考古与文物研究|publisher=中央民族大学出版社|year=2007|isbn=9787811082593}}</ref>{{Rp|87}}。有指，[[明朝|明朝]]時[[七下西洋|七下西洋]]的[[鄭和|鄭和]]船隊在航行過程中曾途經科摩羅群島，但這說法需要進一步考證<ref name=ship>{{cite web |author1=田興春 (編) |author2=李新烽 |title=第三節：活的生物實驗室 |url=http://chinese.people.com.cn/BIG5/42476/74202/74203/5153519.html |website=人民網 |accessdate=2016-02-08 |archiveurl=https://archive.is/20160208033618/http://chinese.people.com.cn/BIG5/42476/74202/74203/5153519.html |archivedate=2016-02-08 |deadurl=no |date=2006-12-11 }}</ref>。

科摩罗最早的[[華人|華人]]來自[[馬達加斯加|馬達加斯加]]，他們只到了科摩羅群島中的[[馬約特島|馬約特島]]（現為[[法國|法國]][[海外省|海外省]]，主權有爭議），沒有到達今科摩罗的其他领土<ref name=ship/>。1894年上半年，共54名中國人來到[[非洲|非洲]]的法國属地，其中包括科摩羅<ref>{{cite book|author1=陈翰笙|title=华工出囯史料汇编|volume=8|publisher=中华书局}}</ref>{{Rp|158}}。

=== 現代 ===
1973年12月，[[联合国大会|联合国大会]]通过决议，确认科摩羅群島的领土完整，并要求統治當地的法国维护科摩羅的统一與完整；中華人民共和國在[[联合国|联合国]]的代表赞成這項决议，並認為马约特岛必須交給科摩羅人民<ref name=hist/>{{Rp|179–180}}。科摩羅在1975年7月6日宣布独立建國<ref name=hist>{{cite book|author1=裴坚章|title=中华人民共和国外交史: 1970-1978|publisher=世界知识出版社|year=1994}}</ref>{{Rp|179}}，[[中華人民共和國總理|中華人民共和國總理]][[周恩来|周恩来]]於同月12日致电科摩羅領導人[[艾哈邁德·阿卜杜拉|艾哈邁德·阿卜杜拉]]，承认科摩罗建國並祝贺該國独立<ref>{{cite book|author1=黎家松|title=中华人民共和国外交大事记|volume=4|publisher=世界知识出版社|year=2003|isbn=9787501221523}}</ref>{{Rp|201}}<ref name=esta>{{cite web |title=王毅会见科摩罗外长哈桑 |url=http://www.mfa.gov.cn/web/wjbzhd/t1163131.shtml |website=中华人民共和国外交部 |accessdate=2016-02-08 |archiveurl=https://archive.is/20160208073515/http://www.mfa.gov.cn/web/wjbzhd/t1163131.shtml |archivedate=2016-02-08 |deadurl=no |date=2014-06-07 }}</ref>；[[中華民國國民大會|中華民國國民大會]]出版的《世界各國憲法大全》認為，科摩羅的獨立、{{link-en|科摩羅政黨列表|List of political parties in the Comoros|政黨組織}}和{{link-fr|科摩羅憲法|Constitution des Comores|憲法}}深受[[共產主義|共產主義]]思想影響，並直接和間接地受[[中國共產黨|中國共產黨]]的鼓勵<ref>{{cite book|title=世界各國憲法大全|volume=6|publisher=國民大會憲政硏討委員會|year=1966}}</ref>{{Rp|32}}。

1975年11月13日，中華人民共和國與科摩羅建交<ref>{{cite book|title=世界知识年鉴|year=2006|url=https://books.google.com.hk/books?id=Qx3lAAAAMAAJ}}</ref>{{Rp|387}}，是首個與科摩罗建立外交關係的国家<ref name=esta/>。

根據{{link-en|經濟複雜性研究中心|The Observatory of Economic Complexity}}網站上的數據，[[中国大陸|中国大陸]]到科摩羅的出口貿易在1995年至2008年一直呈上升趨勢，並在2008年超過4,000萬[[美元|美元]]以上，雖曾下降至2011年的接近2,000萬美元，但其後回升至2013年的超過一億美元；貨品類別方面，[[家具|家具]]的出口額亦在2010年至2013年呈上升趨勢<ref name=tradea/>。科摩羅到中国大陸的出口額曾在2008年至2009年達到40萬美元左右的高峰，[[丁香|丁香]]在出口額中佔了相當大的比重；科摩羅到中国大陸的出口額其後在2011年跌至不足五萬美元，雖然在2012年回升至超過10萬美元（其中大部分是機械產品），但於2013年再次下降<ref name=tradeb/>。

== 文化關係 ==
{{link-fr|科摩罗大学|Université des Comores}}設立了[[孔子課堂|孔子課堂]]，由該校和[[大连大学|大连大学]]共同承建，於2014年7月揭牌成立<ref>{{cite web |title=大连大学第二所孔子学院项目科摩罗大学孔子课堂举行揭牌仪式 |url=http://comores.cmr.chinesecio.com/zh-hans/node/66 |website=网络孔子学院 |accessdate=2016-02-08 |archiveurl=https://archive.is/20160208110513/http://comores.cmr.chinesecio.com/zh-hans/node/66 |archivedate=2016-02-08 |deadurl=no |date=2014-08-06 }}</ref>。中華人民共和國在1982年開始向科摩罗的學生提供[[奖学金|奖学金]]名额，並在2011至2012年度向該國提供49个名额<ref name="culture">{{cite web |title=中国同科摩罗的关系 |url=http://www.chinanews.com/gj/zlk/2014/01-16/251_2.shtml |website=中国新闻网 |accessdate=2016-02-08 |archiveurl=https://archive.is/20160208114246/http://www.chinanews.com/gj/zlk/2014/01-16/251_2.shtml |archivedate=2016-02-08 |deadurl=no }}</ref>。

中華人民共和國與科摩罗簽訂了文化合作协定，中國的藝術表演團體曾多次訪問科摩罗，而中國也在當地舉辦了多個文化活动，例如[[电影|电影]]放映、[[手工艺|手工艺]]作品展覽等<ref name="culture"/>。2015年9月，大连大学的巡演表演团到訪科摩罗，進行[[舞蹈|舞蹈]]、[[中樂|中樂]]等表演，獲該國教育部部长宴请<ref>{{cite web |title=大连大学民乐巡演团在科摩罗掀起了中国风 |url=http://comores.cmr.chinesecio.com/zh-hans/node/76 |website=网络孔子学院 |accessdate=2016-02-08 |archiveurl=https://archive.is/20160208120729/http://comores.cmr.chinesecio.com/zh-hans/node/76 |archivedate=2016-02-08 |deadurl=no |date=2015-10-08 }}</ref>。

== 援助 ==
中華人民共和國對科摩羅提供醫療援助，例如在1994年起共派遣了10批醫療隊援助科摩羅（數據截至2015年9月），這些醫療隊員曾為科摩羅民眾提供免費[[诊断|诊断]]服務<ref>{{cite web |title=驻科摩罗使馆与援科医疗队联合举办义诊活动 |url=http://www.fmprc.gov.cn/web/zwbd_673032/gzhd_673042/t1294307.shtml |website=中华人民共和国外交部 |accessdate=2016-02-08 |archiveurl=https://archive.is/20160208131243/http://www.fmprc.gov.cn/web/zwbd_673032/gzhd_673042/t1294307.shtml |archivedate=2016-02-08 |deadurl=no |date=2015-09-07 }}</ref>；此外，中國亦向科摩罗赠送[[外科手術|外科手術]]设备和[[药品|药品]]，在昂儒昂島援建[[医院|医院]]，並計劃派专家為当地的[[白内障|白内障]]患者進行恢復視力手术<ref>{{cite web |title=我援科摩罗医院项目举行隆重开工暨奠基仪式 |url=http://km.mofcom.gov.cn/article/zxhz/201206/20120608182205.shtml |website=中华人民共和国驻科摩罗大使馆经济商务参赞处 |accessdate=2016-02-08 |archiveurl=https://archive.is/20160208144233/http://km.mofcom.gov.cn/article/zxhz/201206/20120608182205.shtml |archivedate=2016-02-08 |deadurl=no |date=2012-05-30 }}</ref><ref>{{cite web |title=援科摩罗“光明行”项目考察组来科考察 |url=http://km.mofcom.gov.cn/article/zxhz/201512/20151201217063.shtml |website=中华人民共和国驻科摩罗大使馆经济商务参赞处 |accessdate=2016-02-08 |archiveurl=https://archive.is/20160208135636/http://km.mofcom.gov.cn/article/zxhz/201512/20151201217063.shtml |archivedate=2016-02-08 |deadurl=no |date=2015-12-23 }}</ref>。科摩罗曾經盛行[[疟疾|疟疾]]，中華人民共和國的政府機關和企業合共提供了8,000万[[人民币|人民币]]的经费，让科摩罗採用[[青蒿素|青蒿素]]防治疟疾；[[广州中医药大学|广州中医药大学]]的团队到科摩罗開展清除疟疾项目，超过223万人次參與計劃，服用抗疟疾的药品，該國最終在2014年達到了疟疾感染率下降和零死亡<ref>{{cite news |title=青蒿素抗疟在非洲获巨大成功 帮科摩罗实现零死亡 |url=http://www.chinanews.com/gn/2015/10-07/7556723.shtml |accessdate=2016-02-08 |agency=中国新闻网 |date=2015-10-07 |archiveurl=https://archive.is/20160208143929/http://www.chinanews.com/gn/2015/10-07/7556723.shtml |archivedate=2016-02-08 |deadurl=no }}</ref>。

除了醫療援助外，中華人民共和國亦曾為科摩羅援建設施，例如[[火力發電廠|火力發電廠]]、[[体育场|体育场]]等<ref>{{cite web |title=中科合作 |url=http://km.mofcom.gov.cn/article/zxhz/ |website=中华人民共和国驻科摩罗大使馆经济商务参赞处 |accessdate=2016-02-08 |archiveurl=https://archive.is/20160208145319/http://km.mofcom.gov.cn/article/zxhz/ |archivedate=2016-02-08 |deadurl=yes }}</ref>。
