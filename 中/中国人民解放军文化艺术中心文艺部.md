'''中国人民解放军文化艺术中心文艺部'''，是[[中国人民解放军|中国人民解放军]]文化艺术团体，隶属[[中国人民解放军文化艺术中心|中国人民解放军文化艺术中心]]。

== 沿革 ==
中国革命军队中的文艺团体历史悠久。[[第一次国共合作|第一次国共合作]]时期，1925年1月18日，[[黄埔军校|黄埔军校]]“血花”剧社成立，宗旨是“将革命的艺术来改造社会”，用革命艺术实现[[孙中山|孙中山]]提出的“唤起民众”，提倡“艺术可以改造社会，艺术可以美化人生”。“血花”剧社隶属黄埔军校政治部，实际领导人是政治部主任[[周恩来|周恩来]]，日常事务工作由剧社总务主任、中共党员[[李之龙|李之龙]]负责，[[陈赓|陈赓]]也在该剧社十分活跃<ref name=dafeng>{{Cite web |url =http://wemedia.ifeng.com/42667525/wemedia.shtml  |title = 看过《芳华》之后再来聊聊部队文工团的前世今生 |author =  |date =2017-12-27  |publisher =凤凰网  |language =  |accessdate =  |quote =  }}</ref>。

[[中国共产党|中国共产党]]领导下的部队文工团诞生于[[第一次国共内战|第一次国共内战]]期间。1929年12月，《[[古田会议|古田会议]]决议》要求在红军大队（相当于营）[[士兵委员会|士兵委员会]]内设俱乐部，军队政治宣传科设艺术股，出版石印或者油印画报，并且从各部队中挑选文艺积极分子组成宣传队<ref name=dafeng/>。1929年[[古田会议|古田会议]]后，[[中国工农红军|中国工农红军]]各部队从士兵中挑选人员组成宣传队。1931年中国工农红军中央军事政治学校成立“八一剧团”，成为中国共产党军队中首个以艺术活动为主的专业性艺术团体。此后，红军各军团、方面军陆续成立文艺团体<ref name=junshi/>。1933年[[春节|春节]]期间，由红军将士同台演出的四幕话剧《庐山之雪》是中共军队历史上首部有据可查的原创红色戏剧，表现[[中央红军|中央红军]]取得[[中国共产党第三次反围剿战争|第三次反围剿]]胜利，打进[[南昌|南昌]]城，即将杀上[[庐山|庐山]]活捉[[蒋介石|蒋介石]]的故事，[[红一军团|红一军团]]保卫局长[[罗瑞卿|罗瑞卿]]任总导演并饰蒋介石，红一军团军团长[[林彪|林彪]]饰红军司令员，红一军团政委[[聂荣臻|聂荣臻]]饰红军政委，红一军团政治部主任[[罗荣桓|罗荣桓]]饰红军政治部主任。第一次国共内战时期，中共军队文艺团体通常称剧社、剧团、宣传队等<ref name=dafeng/>。[[抗日战争|抗日战争]]时期和[[第二次国共内战|第二次国共内战]]时期，中共军队文艺团体又有很大发展，通常称剧社、文艺工作团（简称文工团）、宣传队等<ref name=junshi>{{Cite web |url = http://www.js7tv.cn/news/201712_125473.html |title =从《芳华》谈20世纪70年代部队文工团  |author =  |date =  2017-12-25|publisher =中国军视网  |language =  |accessdate =  |quote =  }}</ref>。

抗日战争时期，1938年，[[八路军|八路军]]总部炮兵团在[[山西|山西]]成立，并同时组建炮兵团宣传队，又称“怒吼剧社”。1944年11月，遵照[[中共中央军委|中共中央军委]]命令，以八路军炮兵团为基础，在[[延安|延安]][[南泥湾|南泥湾]]成立中共军队历史上第一所兵种院校——[[延安炮兵军校|延安炮校]]，同时成立炮校宣传队，对外仍称“怒吼剧社”。1945年抗日战争胜利后，延安炮校奉命开赴[[中国东北|中国东北]]。1946年初，在怒吼剧社的基础上，在[[吉林|吉林]][[通化|通化]]正式组建炮校文工团。一年后，[[东北民主联军|东北民主联军]]成立炮兵司令部，与炮校机关为[[一套人马，两块牌子|一套人马两块牌子]]，炮校文工团同时也是炮兵文工团。1948年1月改称[[中国人民解放军东北野战军|中国人民解放军东北野战军]]炮兵纵队文工团。[[辽沈战役|辽沈战役]]结束后，改称[[中国人民解放军第四野战军|中国人民解放军第四野战军]]特种兵文工团（简称四野特种兵文工团）<ref name=sohu>{{Cite web |url =http://www.sohu.com/a/53167210_635866  |title = 总政歌舞团曾赴前线演出 团长交代后事 |author =  |date =2016-01-08  |publisher =搜狐  |language =  |accessdate =  |quote =  }}</ref>。

[[中华人民共和国|中华人民共和国]]成立后，1950年底，[[中国人民解放军|中国人民解放军]]各野战军（大军区）、兵团、军（包括部分省军区）和各特种兵及部分院校都设有文工团，不少团级部队也有宣传队。据统计，当时全军共有专业文艺单位300余个，文工团（队）员6.4万多人。1951年起，全军文工团、队，除[[中国人民志愿军|中国人民志愿军]]外的调整、精简、整编工作开始，原则是逐步减少数量，提高质量。1953年，[[中国人民解放军总政治部|总政治部]]提出全军各大单位要“建立以歌舞为主有专业分工的综合性的文工团”；军、师文工团、队要实现“一专多能”。遵照该要求，各大军区、军兵种普遍建立了专业歌舞团或者以歌舞为主的综合性文工团<ref name=junshi/><ref name=dafeng/>。

1951年5月，'''[[中央人民政府人民革命军事委员会|中央人民政府人民革命军事委员会]]总政治部文艺工作团'''（简称军委总政文工团，1954年改称'''[[中国人民解放军总政治部|中国人民解放军总政治部]]文艺工作团'''，简称总政文工团）成立<ref name=junshi/><ref name=dafeng/><ref name=youlai>{{Cite web |url =http://ent.sina.com.cn/x/2005-05-13/1707723214.html  |title = 总政话剧团的由来 |author =  |date = 2005-05-13 |publisher =新浪  |language =  |accessdate =  |quote =  }}</ref><ref name=hjtjj/>。总政文工团由四野特种兵文工团与[[华北军政大学|华北军政大学]]文工团、装甲兵政治部文工团合并组建<ref name=sohu/>。人员主要来自四野特种兵文工团、华北军政大学文工团、装甲兵政治部文工团（原为[[中国人民解放军第一野战军|第一野战军]]第二兵团文工团）、军委工程兵学校文工团、第四高级步兵学校文工团（原[[抗大|抗大]]总校文工团）、西南军区政治部战斗文工团（原[[八路军|八路军]]一二零师战斗剧社）、总高级步兵学校文工团（原[[新四军|新四军]]雪枫剧社）、总后勤部文工团等单位<ref name=hjtjj>{{Cite web |url =http://ent.sina.com.cn/x/2005-05-13/1701723206.html  |title = 总政话剧团简介 |author =  |date = 2005-05-13 |publisher =新浪  |language =  |accessdate =  |quote =  }}</ref>。建团方针是“要建成一个全军专业的、示范的、正规的、代表全军的文工团”，该方针由总政治部文化部部长[[陈沂|陈沂]]提出，第一届全军宣教文化工作会议上通过，经总政治部主任[[罗荣桓|罗荣桓]]同意，报[[周恩来|周恩来]]批准<ref name=youlai/><ref name=liushi>{{Cite web |url =http://www.81.cn/jsdj/2013-06/30/content_5471184.htm  |title = 能下基层边防，能上艺术殿堂——写在总政歌舞团建团60周年之际 |author =  |date =2013-06-30 |publisher = 中国军网 |language =  |accessdate =  |quote =  }}</ref>。团部下辖研究室、戏剧队、乐队、舞蹈队（以后又合并称歌舞剧队）、演出股<ref name=youlai/>。同期，各军区、军兵种陆续成立或调整了文艺工作团（简称文工团）<ref name=junshi/><ref name=dafeng/>。1952年，总政治部计划仿效[[苏联|苏联]]体制，将全国有影响的剧种集中成立“中国大剧院”，在总政文工团原有的歌剧队和舞剧队的基础上，再增加一个京剧队、一个评剧队和一个越剧队。1952年6月，总政文工团副团长[[史行|史行]]、剧作家[[黄宗江|黄宗江]]和总政歌舞团的[[兰茜|兰茜]]南下[[上海|上海]]，确定以玉兰剧团作为组建越剧队的基础，随后[[胡野檎|胡野檎]]任越剧队队长<ref name=yulan>{{Cite web |url =http://xmwb.xinmin.cn/html/2012-09/28/content_59_1.htm  |title =我的越剧人生  |author = 王文娟 |date = 2012-09-28 |publisher = 新民晚报 |language =  |accessdate =  |quote =  }}</ref>。总政文工团成立初期，下设戏剧队（1953年5月改称[[话剧|话剧]]团<ref name=youlai/>）、歌舞队（1953年5月改称歌舞团<ref name="tuanzhang" />）、[[评剧|评剧]]团、[[越剧|越剧]]团、[[京剧|京剧]]团、[[曲艺|曲艺]][[杂技|杂技]]团<ref name=junshi/><ref name=dafeng/>。1955年，总政治部文化部决定将总政文工团京剧团和西南军区京剧院移交[[中华人民共和国文化部|中华人民共和国文化部]]，1955年11月7日合并成立[[中国京剧院|中国京剧院]]第四团<ref>黎青，中国京剧院第四团成立，戏剧报1955(12):51</ref>。1954年将[[广州军区|广州军区]]歌剧团调入总政文工团，为歌舞团的歌剧团，经过充实扩建于1956年9月22日正式成立为总政文工团歌剧团，建团后合并了总政歌舞团的歌舞剧研究室<ref name=gjtcl>{{Cite web |url = http://www.dzwww.com/yule/yy/201601/t20160118_13708067.htm |title = 总政歌剧团易名中央军委政治工作部歌剧团 |author =  |date = 2016-01-18 |publisher = 大众网 |language =  |accessdate =  |quote =  }}</ref>。后经调整，1960年代初总政文工团仅保留歌舞团、歌剧团、话剧团。同期，海军、空军政治部文工团分别一分为二成歌舞团和话剧团；各大军区政治部文工团也拆分为歌舞团和话剧团，有的还有杂技团等<ref name=junshi/><ref name=dafeng/>。

[[文化大革命|文革]]爆发后，1966年12月总政文工团解散歌舞团等部门<ref name="tuanzhang" />。1969年9月4日，根据[[军委办事组|军委办事组]]公布军队文工团精简方案，除各大军区、海军、空军、铁道兵、工程兵、第二炮兵、各军区空军、海军舰队编制文工团外，其余文工团、队一律撤销。精简后各文工团一律改称宣传队，如总政文工团改称'''中国人民解放军总政治部宣传队'''（简称总政宣传队）。原各文工团内部的团、队也按部队编制改名<ref name=junshi/>。例如1970年4月，恢复成立总政宣传队二中队（后改歌舞队）<ref name="tuanzhang" />。 1975年5月，根据[[中共中央军委|中共中央军委]]通知，各专业宣传队陆续恢复文工团及歌舞团、话剧团的名称等<ref name=junshi/>。1978年7月，总政文工团恢复歌舞团。1980年12月，总政文工团建制撤销，下属各团分别独立为'''中国人民解放军总政治部歌舞团'''（简称总政歌舞团）、'''中国人民解放军总政治部歌剧团'''（简称总政歌剧团）、'''中国人民解放军总政治部话剧团'''（简称总政话剧团）<ref name=dafeng/><ref name="tuanzhang" /><ref>[http://yule.sohu.com/20100621/n272957679.shtml 中国人民解放军总政治部歌舞团简介，搜狐，2010-06-21]</ref>。[[中国剧院|中国剧院]]是总政歌舞团下辖的演出场地。

在[[深化国防和军队改革|深化国防和军队改革]]中，2016年3月，中国人民解放军总政歌舞团正式摘牌，更名为'''中国人民解放军歌舞团'''<ref>[http://www.nmg.xinhuanet.com/xwzx/jrjj/2016-03/24/c_1118423440.htm 总政歌舞团正式摘牌更名为解放军歌舞团]，新华网，2016-03-24</ref>。又称'''中央军委政治工作部歌舞团'''。歌舞团隶属新组建的[[中央军委政治工作部|中央军委政治工作部]]<ref>[http://www.whnews.cn/news/node/2016-03/21/content_6712009.htm 军队文工团改革:三大隶属歌舞剧团均已重新亮相] {{Wayback|url=http://www.whnews.cn/news/node/2016-03/21/content_6712009.htm |date=20161009152842 }}，威海网，2016-03-21</ref><ref>[http://military.people.com.cn/n1/2016/0321/c1011-28212560.html 军委政治工作部歌舞团亮相 前身系总政歌舞团]，人民网，2016-03-21</ref>。同时，总政歌剧团、总政话剧团分别更名'''中央军委政治工作部歌剧团'''、'''中央军委政治工作部话剧团'''<ref name=dafeng/>。

2018年，在[[深化国防和军队改革|深化国防和军队改革]]中，原八一电影制片厂、原总政歌舞团、原总政歌剧团、原总政话剧团、原总政军乐团合并成立[[中国人民解放军文化艺术中心|中国人民解放军文化艺术中心]]<ref>{{Cite web |url = http://news.sina.com.cn/c/nd/2018-02-02/doc-ifyrcsrw7757091.shtml |title =八一电影制片厂被曝遭裁撤 曾拍摄芳华士兵突击等  |author =  |date =2018-02-02  |publisher =新浪  |language =  |accessdate =  |quote =  }}</ref><ref name=changbiao>{{Cite web |url =http://3g.163.com/dy/article/DKQRNFE7051780GA.html  |title =“解放军文化艺术中心电影电视制作部”成立 八一厂厂标不会摘  |author =  |date =2018-06-21  |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref>。其中，原总政歌舞团、原总政歌剧团、原总政话剧团、原总政军乐团合并为'''中国人民解放军文化艺术中心文艺部'''<ref>{{Cite web |url = http://www.sohu.com/a/232720200_126166 |title =孙丽英：为人民歌唱是我一直的追求  |author =  |date = 2018-08-13 |publisher = 搜狐 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url =  http://www.zunyi.gov.cn/sy/qxdt/201806/t20180615_755782.html|title = 播州区平正乡红心（团结）村乡村振兴动员大会召开 |author =  |date = 2018-06-15 |publisher = 中国遵义 |language =  |accessdate =  |quote =  }}</ref>。
