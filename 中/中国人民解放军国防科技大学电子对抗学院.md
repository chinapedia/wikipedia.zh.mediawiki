'''中国人民解放军国防科技大学电子对抗学院'''，位于[[安徽省|安徽省]][[合肥市|合肥市]]，隶属[[中国人民解放军国防科技大学|中国人民解放军国防科技大学]]。

== 沿革 ==
*1975年12月18日，[[叶剑英|叶剑英]]向[[中共中央|中共中央]]和[[毛泽东|毛泽东]]呈交《关于加强电子对抗工作的报告》，建议组成国务院、中央军委电子对抗和雷达管理领导小组，成立[[总参四部|总参四部]]。1975年12月20日，毛泽东在该报告上批示两字“很好”。1975年12月22日，[[邓小平|邓小平]]阅毛泽东该批示，并批送[[中共中央政治局|中共中央政治局]]同志传阅后退叶剑英<ref>{{Cite web |url =http://www.china.com.cn/zhuanti2005/txt/2004-08/01/content_5624058.htm  |title = 1975年12月 |author =  |date = 2004-08-01 |publisher =  中国网|language =  |accessdate =  |quote =  }}</ref><ref name=huiyi>{{Cite web |url = http://www.sohu.com/a/127706863_371069 |title =  装满青春回忆的军校——解放军电子工程学院 |author =  |date = 2017-03-02  |publisher = 搜狐 |language =  |accessdate =  |quote =  }}</ref>。
*1976年，经国务院、中央军委批准，'''中国人民解放军电子工程学院'''筹建。1978年11月定点在[[合肥市|合肥市]]<ref name=ahszjsz> {{Cite book | author =安徽省地方志编纂委员会 编  | title =安徽省志·军事志  | location =  | publisher = 安徽人民出版社 | date = 1995年 | pages = 515-516 | ISBN =  | accessdate =  | url =  | language =  | quote = 电子工程学院 }} </ref>。
*1979年9月16日，正式成立中国人民解放军电子工程学院<ref name=sanshi/><ref name=huiyi/>。属[[中国人民解放军总参谋部|中国人民解放军总参谋部]]直接领导<ref name=ahszjsz/>。
*1991年11月，[[中央军委主席|中央军委主席]][[江泽民|江泽民]]视察安徽，详细询问该学院建设情况，并在1991年11月23日为该学院题词：“努力培养合格人才，大力加强电子对抗部队建设。”<ref name=huiyi/>
*2000年4月5日，[[中央军委副主席|中央军委副主席]][[胡锦涛|胡锦涛]]对学院建设做出专门批示：“万同志：此事的确事关重大，请批示有关部门抓紧论证决策。”学院此后实现了由电子对抗院校向信息作战院校的转型<ref name=huiyi/>。
*2016年，在[[深化国防和军队改革|深化国防和军队改革]]中，中国人民解放军电子工程学院原来的上级部门[[中国人民解放军总参谋部|中国人民解放军总参谋部]]撤销，该学院转隶[[中央军事委员会训练管理部|中央军事委员会训练管理部]]<ref>{{Cite web |url = http://military.people.com.cn/n1/2016/0726/c1011-28585938.html |title =中央军委训练管理部注重发挥资源优势精准实施定点帮扶工作  |author =  |date = 2016-07-26 |publisher =人民网  |language =  |accessdate =  |quote =  }}</ref>。
*2016年4月，[[中共中央总书记|中共中央总书记]]、[[中华人民共和国主席|国家主席]]、中央军委主席[[习近平|习近平]]视察安徽，接见了该学院师以上领导干部<ref name=huiyi/>。
*2017年，在[[深化国防和军队改革|深化国防和军队改革]]中，[[中国人民解放军国防科学技术大学|中国人民解放军国防科学技术大学]]、[[中国人民解放军国际关系学院|中国人民解放军国际关系学院]]、[[中国人民解放军国防信息学院|中国人民解放军国防信息学院]]、[[中国人民解放军西安通信学院|中国人民解放军西安通信学院]]、中国人民解放军电子工程学院、[[中国人民解放军理工大学|中国人民解放军理工大学]]气象海洋学院为基础组建[[中国人民解放军国防科技大学|中国人民解放军国防科技大学]]。校本部设在[[长沙|长沙]]，内设学院位于长沙、[[南京|南京]]、[[武汉|武汉]]、[[合肥|合肥]]等地<ref name=xin>{{Cite web |url =http://www.thepaper.cn/newsDetail_forward_1699945  |title = 新国防科大今年以国防科大、国关学院等6所院校为基础重建 |author =  |date =  2017-06-03 |publisher =  澎湃新闻|language =  |accessdate =  |quote =  }}</ref>。其中，中国人民解放军电子工程学院改为'''中国人民解放军国防科技大学电子对抗学院'''。2017年7月，中国人民解放军国防科技大学电子对抗学院挂牌<ref name=liangxiang>{{Cite web |url = http://www.thepaper.cn/newsDetail_forward_1748674 |title = 陆军炮兵防空兵学院、国防科技大学电子对抗学院亮相合肥 |author =  |date = 2017-08-01 |publisher = 澎湃新闻 |language =  |accessdate =  |quote =  }}</ref>。
