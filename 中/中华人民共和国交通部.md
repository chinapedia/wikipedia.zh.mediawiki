'''中华人民共和国交通部'''，简称'''交通部'''，是[[中华人民共和国国务院|中华人民共和国国务院]]已撤销的主管全国公路交通、水运交通的[[国务院组成部门|组成部门]]。

2008年3月15日，[[第十一届全国人民代表大会第一次会议|十一届全国人大一次会议]]决定撤销交通部。将交通部、[[中国民用航空总局|民航总局]]的职责以及[[中华人民共和国建设部|建设部]]的指导城市客运职责整合，组建[[中华人民共和国交通运输部|中华人民共和国交通运输部]]。

1954年，根据[[一届全国人大一次会议|一届全国人大一次会议]]决定，撤销[[中央人民政府政务院|中央人民政府政务院]]，组建[[中华人民共和国国务院|中华人民共和国国务院]]，[[中央人民政府交通部|中央人民政府交通部]]相应调整为'''中华人民共和国交通部'''。交通部设部长1人、副部长2至4人，实行部长负责制。部长领导本部门的工作，召集和主持部务会议，签署上报国务院的重要请示报告和下达的指示命令；副部长协助部长工作<ref>{{Cite book|title=中华法学大辞典 宪法学卷|last=许崇德主编|first=|publisher=北京：[[中国检察出版社|中国检察出版社]]|date=1995-03|isbn=7-80086-280-1|location=|pages=824}}</ref>；具体工作由[[国务院第六办公室|国务院第六办公室]]（主管交通、邮电<ref>{{Cite book|title=毛泽东纪事 1893-1976 下|last=李新芝、郑明俊主编|first=|publisher=北京：[[中央文献出版社|中央文献出版社]]|date=2013-01|isbn=978-7-5073-3609-2|location=|pages=658}}</ref>）协助总理[[周恩来|周恩来]]负责掌管<ref name=u5 />。成立初期，交通部的内设机构有：办公厅、计划统计局、财务会计局、材料供应局、干部局、劳动工资局、教育局、监察局、技术局、机务局、公路总局、海河运输局、航务工程局、航道管理局、商务局、船舶登记局、港航监督局、电讯局、船厂管理局、基本建设局、专家工作室、法律室、参事室和政治部<ref name=u5 />。

1958年2月27日，国务院决定[[中国民用航空局|中国民用航空局]]改为交通部部属局；3月19日，根据周恩来提议，[[第一届全国人民代表大会常务委员会|第一届全国人大常委会]]第九十五次会议批准了国务院的这一决定<ref>{{Cite book|title=中国民用航空志 东北地区卷|last=徐奇长主编|first=|publisher=北京：[[中国民航出版社|中国民航出版社]]|date=2007-11|isbn=7-80110-798-5|location=|pages=47}}</ref>。随后，民航局又于1960年11月17日改名为交通部中国民航总局，由[[中国人民解放军空军|中国人民解放军空军]]和交通部分工领导<ref>{{Cite book|title=民航简史|last=李永编著|first=|publisher=北京：[[中国民航出版社|中国民航出版社]]|date=2010-08|isbn=978-7-80110-996-5|location=|pages=63}}</ref>。1962年4月13日，[[第二届全国人大常委会|第二届全国人大常委会]]第五十三次会议批准周恩来提议，将交通部所属中国民用航空总局改为[[国务院直属机构|国务院直属机构]]。1965年，交通部机关设：办公厅、政策研究室、计划统计司、财务会计司、劳动工资司、技术司、教育局、材料供应局、民间运输局、水运局、公路运输局、公路工程管理局、保卫局、参事室和政治部、中监委派驻监察组。1973年5月，交通部机关设：办公室、政治部、铁路运输局、水运局、公路局、机车车辆局、工务电务局、铁路基建局、水运基建局、水运工业局、铁路工业局、计划统计局、财务局、物资局、人事局、外事局、公安局、安全监察委员会、科学技术委员会。

1969年11日，国务院决定，将[[中华人民共和国铁道部|铁道部]]、交通部和[[中华人民共和国邮电部|邮电部]]的[[中国邮政|邮政]]部分机构整合，组建新的交通部<ref name=hs/><ref name=u5>{{Cite book|title=中华人民共和国中央政府机构 1949-1990|author=苏尚尧|publisher=北京：[[经济科学出版社|经济科学出版社]]|date=1993-03|isbn=7-5058-0504-5|location=|pages=269-274}}</ref>。

1975年1月，[[邓小平|邓小平]]主持中央党政工作，并开始对各个领域进行整顿工作，其中就包括交通领域<ref>{{Cite book|title=新中国行政体制沿革|author=许超|publisher=北京：[[世界知识出版社|世界知识出版社]]|date=2012-04|isbn=978-7-5012-4223-8|location=|pages=28}}</ref>。1月17日，介于铁路管理复杂性，[[第四届全国人民代表大会|第四届全国人大]]第一次会议决定，将铁道部分离出交通部<ref>{{Cite book|title=中国行政改革概论|author=许放|publisher=北京：[[冶金工业出版社|冶金工业出版社]]|date=2012-03|isbn=978-7-5024-5868-3|location=|pages=85}}</ref><ref>{{Cite book|title=政府机构改革及运作全书|author=[[刘国光|刘国光]]|author2=[[邢贲思|邢贲思]]|author3=[[杨启先|杨启先]]|publisher=北京：[[中共中央党校出版社|中共中央党校出版社]]|date=1998-04|isbn=7-5035-1781-6|location=|pages=108-109}}</ref>。[[国务院机构改革#赵紫阳时期——1982年|1982年中国政府机构改革]]后，交通部统一领导和归口管理全国水路、[[中国公路交通|公路交通]]的行政、运输生产和建设工作。1998年3月，[[第九届全国人民代表大会|第九届全国人大]]第一次会议批准[[国务院机构改革#朱镕基时期——1998年|国务院改革方案]]，将原由交通部代管的[[中华人民共和国大连海事法院|大连]]、[[中华人民共和国青岛海事法院|青岛]]、[[中华人民共和国天津海事法院|天津]]、[[中华人民共和国上海海事法院|上海]]、[[中华人民共和国武汉海事法院|武汉]]、[[中华人民共和国广州海事法院|广州海事法院]]划归人民法院系统统一管理，纳入国家司法体系<ref>{{Cite book|title=干部人才工作常用文件选编 上|last=中共辽宁省委组织部编|first=|publisher=沈阳：[[辽宁人民出版社|辽宁人民出版社]]|year=2007.12|isbn=978-7-205-06250-7|location=|pages=359-365}}</ref>；出租车管理职能下放到地方政府，而汽车出入境运输职责划入交通部管理<ref name="hs">{{Cite book|title=中华人民共和国政府机构五十年|last=[[国家行政学院|国家行政学院]]编著|first=|publisher=北京：[[党建读物出版社|党建读物出版社]]|year=2000.06|isbn=7-80098-406-0|location=|pages=116-117}}</ref>。

1980年2月4日，交通部机关设：办公厅、计划统计局、财务局、基本建设局、外事局、人事局、教育局、科学技术局、公安局、水运局、公路局、安全局、港务监督局、通信导航局、行政管理局。保留工业局、物资局。1981年4月15日，国务院批准交通部增设进出口办公室。1982年7月，国务院批准交通部机关设：办公厅、海洋运输管理局、内河运输管理局、生产调度局（1987年8月更名为企业管理局）、水上安全监督局、公路局、基本建设局、计划统计局、财务会计局、劳动工资局、教育局、科学技术局、公安局、物资局、机关事务局、老干部管理局。1984年4月29日，[[审计署|审计署]]驻交通部审计局成立<ref>{{Cite book|title=中国审计年鉴 1989-1993|last=《中国审计年鉴》编辑委员会编|first=|publisher=北京：中国审计出版社|year=1994.12|isbn=7-80064-344-1|location=|pages=297}}</ref><ref>{{Cite book|title=审计法与审计实务全书|last=审计法与审计实务全书编委会编写|first=|publisher=北京：[[北京燕山出版社|北京燕山出版社]]|year=1995.06|isbn=7-5402-0142-8|location=|pages=667}}</ref>。

1994年2月25日，国务院办公厅发出《国务院办公厅关于印发交通部职能配置、内设机构和人员编制方案的通知》（国办发〔1994〕29号），批准交通部设13个职能司局和机关党委<ref name="报道0417A">{{cite web|author1=国务院办公厅 |title=国务院办公厅关于印发交通部职能配置、内设机构和人员编制方案的通知 |url= http://www.gov.cn/zhengce/content/2010-11/12/content_7892.htm |website=中国政府网 |accessdate=2019-04-17 |location=北京 |language=zh-cn}}</ref>。

2008年3月15日，[[第十一届全国人民代表大会第一次会议|第十一届全国人民代表大会第一次会议]]通过《第十一届全国人民代表大会第一次会议关于国务院机构改革方案的决定》，批准《国务院机构改革方案》。方案规定：“组建[[中华人民共和国交通运输部|交通运输部]]。将'''中华人民共和国交通部'''、[[中国民用航空总局|中国民用航空总局]]的职责，[[中华人民共和国建设部|建设部]]的指导城市客运职责，整合划入[[中华人民共和国交通运输部|'''交通运输部''']]。组建[[中国民用航空局|国家民用航空局]]，由交通运输部管理。[[国家邮政局|国家邮政局]]改由交通运输部管理。保留[[中华人民共和国铁道部|铁道部]]，继续推进改革。不再保留交通部、中国民用航空总局。”<ref>{{Cite web  |url =http://www.npc.gov.cn/npc/xinwen/syxw/2008-03/12/content_1413911.htm  |title =关于国务院机构改革方案的说明——2008年3月11日在第十一届全国人民代表大会第一次会议上  |author =华建敏  |date =2008-03-11  |publisher =中国人大网  |language =  |accessdate =  |quote =  |archive-url =https://web.archive.org/web/20180320170547/http://www.npc.gov.cn/npc/xinwen/syxw/2008-03/12/content_1413911.htm  |archive-date =2018年3月20日  |dead-url =no  }}</ref><ref>{{Cite web |url = http://www.gov.cn/2008lh/content_921411.htm |title = 两会授权发布：国务院机构改革方案 |author =  |date = 2008-03-15 |publisher = 中国政府网 |language =  |accessdate =  |quote =  |archive-url = https://web.archive.org/web/20180126012934/http://www.gov.cn/2008lh/content_921411.htm |archive-date = 2018-01-26 |dead-url = no }}</ref><ref>{{Cite web|url = http://www.gov.cn/2008lh/content_921283.htm|title = 十一届人大一次会议关于国务院机构改革方案决定|author = |date = 2008-03-15|publisher = 中国政府网|language = |accessdate = |quote = |archive-url = https://web.archive.org/web/20180320172551/http://www.gov.cn/2008lh/content_921283.htm|archive-date = 2018-03-20|dead-url = no}}</ref>

== 职责 ==
根据《{{s|交通部职责配置、内设机构和人员编制方案}}》，2008年建制撤销前，交通部承担下列职能<ref name="报道0417A"/>：
{{columns-list|2|
# 根据国民经济和社会发展需要，制订全国公路和水路交通行业发展战略、方针、政策和法规，并监督执行。
# 根据国家的总体布局，组织编制全国公路和水路交通行业发展规划，制订固定资产投资、运输生产、工业、科技、教育中长期计划和年度计划，并监督实施。
# 负责公路、水路交通的行业管理和运输组织管理，对关系国计民生的重点物资运输、紧急物资运输进行必要的调控，组织重点交通工程建设的实施。
# 会同有关部门培育和管理交通运输市场和交通基础设施建设市场，建立完善信息、服务体系，引导交通运输业优化结构、协调发展。
# 负责全国公路及其设施的建设、养护、管理和规费稽征，负责汽车维修市场、汽车驾驶学校和驾驶员培训工作的行业管理，指导城乡客、货运输的衔接协调工作。
# 负责全国港航设施的建设、养护、管理和规费稽征，负责水上港航监督、船舶及海上设施检验、通信导航、救助打捞、船舶代理、外轮理货、港口、航道和港航设施建设使用岸线的行业管理。
# 指导交通行业的体制改革和企业管理工作，负责部属单位国有资产的管理和保值增值的监督。
# 指导交通行业的精神文明建设和职工队伍建设，组织、指导交通行业人才预测、教育、培训、交流和劳动工资工作，管理部属单位的主要领导干部，管理和指导交通系统的公安工作。
# 负责多边和双边政府间有关交通方面的涉外工作，负责利用外资、开展国际交通合作与交流工作，归口管理公路、水路方面与国际组织有关的事宜，维护我国公路和水路运输权益。
# 制订交通科技政策、技术标准和规范，组织重大科技开发，推动行业科技进步。负责归口的交通工业产品认证和质量监督。
# 协同或会同有关部门制定投资、价格、税收、信贷、劳动工资、外汇和其他有关交通行业的经济政策。
# 承办国务院交办的其他事项。
}}
