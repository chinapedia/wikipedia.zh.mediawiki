'''中国人民解放军空军指挥学院'''，简称'''空军指挥学院'''，位于[[北京市|北京市]][[海淀区|海淀区]]北四环西路88号，是培养空军中高级指挥军官的军事高等院校，隶属[[中国人民解放军空军|中国人民解放军空军]]<ref name=mading/>。

==沿革==
*1958年9月12日，按照[[中华人民共和国国防部|国防部]]命令，在[[中国人民解放军军事学院|中国人民解放军军事学院]]空军系的基础上，组建'''中国人民解放军空军学院'''。[[中国人民解放军空军|中国人民解放军空军]]副司令员[[刘震|刘震]]上将兼任院长、政委<ref>{{Cite web |url = http://www.mod.gov.cn/reports/201403/wzry/2014-09/12/content_4536788.htm |title =空军指挥学院  |author =  |date = 2014-09-12 |publisher = 中华人民共和国国防部 |language =  |accessdate =  |quote =  }}</ref><ref name=mading>马德祖、丁晓峰，空军的“国防大学”——中国人民解放军空军指挥学院，国防2001年08期</ref>。
*1969年2月19日，撤销空军学院<ref name=cc> {{Cite book | author =军事科学院编  | title = 军事大辞海·下 | location =  | publisher = 长城出版社 | date = 2000年 | pages =  | ISBN =  | accessdate =  | url =  | language =  | quote =空军指挥学院  }} </ref><ref name=mading/>。
*1974年10月9日，遵照中央军委转发全军院校调整领导小组《关于全军恢复和增建41所院校的报告》，在空军学院原校址建立'''中国人民解放军空军军政干部学校'''，执行军级权限，直属空军建制领导<ref name=mading/>。
*[[粉碎四人帮|粉碎“四人帮”]]后，[[中共中央军委|中共中央军委]]重申要把教育训练摆在战略地位，并于1978年1月决定“各军兵种的军政干校改为各军兵种的学院”<ref name=mading/>。
*1977年11月7日，空军军政干部学校改建为'''中国人民解放军空军学院'''<ref name=cc/>。1978年5月18日正式更名<ref name=mading/>。
*1986年6月23日，更名'''中国人民解放军空军指挥学院'''。主要任务是培养空军中、高级指挥军官和参谋军官，以及研究生层次的空军战役战术研究人员和教员<ref name=cc/><ref name=mading/>。学院执行正军级权限。
*1999年底，根据中央军委要求，空军指挥学院重建外训系。2000年3月，首批4个国家的10名学员来到空军指挥学院学习。2009年起，空军指挥学院在全军院校中率先采取“中外合训、混编合训”的方式，此后这种外训模式在全军承担外训任务的机构中推广<ref>{{Cite web |url =  http://news.sina.com.cn/c/sd/2012-07-23/102924826638.shtml|title = 外军精英走进中国空军课堂:同吃同住同学习 |author =  |date =2012-07-23  |publisher = 新浪 |language =  |accessdate =  |quote =  }}</ref>。
*2017年，在[[深化国防和军队改革|深化国防和军队改革]]中，调整组建新的'''中国人民解放军空军指挥学院'''<ref>{{Cite web |url =http://www.mod.gov.cn/topnews/2017-06/29/content_4783973.htm  |title = 国防部公布调整改革后军队院校名称 |author =  |date = 2017-06-29 |publisher = 中华人民共和国国防部 |language =  |accessdate =  |quote =  }}</ref>。
