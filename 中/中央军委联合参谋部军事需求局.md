'''中央军委联合参谋部军事需求局'''，位于[[北京市|北京市]]，是[[中央军委联合参谋部|中央军委联合参谋部]]下属局，负责军事需求工作。

== 沿革 ==
在[[深化国防和军队改革|深化国防和军队改革]]中，2016年1月撤销[[中国人民解放军总参谋部|中国人民解放军总参谋部]]，组建[[中央军委联合参谋部|中央军委联合参谋部]]。中央军委联合参谋部下设'''中央军委联合参谋部军事需求局'''<ref>{{Cite web|url =http://www.zunyi.gov.cn/zwgk/jcxxgk/zfgb/2016/dlq_06/201608/t20160809_423057.html|title =2016年5月政务大事记|author =|date =2016-08-09|publisher =中国遵义|language =|accessdate =|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20170202014417/http://www.zunyi.gov.cn/zwgk/jcxxgk/zfgb/2016/dlq_06/201608/t20160809_423057.html|archivedate =2017-02-02}}</ref><ref name=kaihui>{{Cite web |url = http://news.xinhuanet.com/local/2016-09/29/c_129306184.htm |title = 长沙县开慧通用机场项目开工 |author =  |date =  2016-09-29|publisher = 新华网 |language =  |accessdate =  |quote =  }}</ref>。各战区联合参谋部也分别设立军事需求局。各级军事需求局的主要任务是研究部队实战化的作战需求，从而以作战需求为指引，提高部队实战化水平<ref>{{Cite web |url =http://mil.sohu.com/20160617/n454872964.shtml?_t_t_t=0.9692052798345685  |title =实战化：不辱使命磨砺胜战之刃  |author =  |date = 2016-06-17 |publisher =搜狐  |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url =http://world.huanqiu.com/hot/2016-02/8604878.html  |title = 解放军新战区在磨合中走向联合 |author =  |date = 2016-02-25 |publisher =环球网  |language =  |accessdate =  |quote =  }}</ref>。

中央军委联合参谋部军事需求局成立后，承担了原由[[中国人民解放军总参谋部|中国人民解放军总参谋部]]承担的对[[通用机场|通用机场]]建设的审批工作<ref>{{Cite web |url = http://www.chinaairports.org.cn/Generalairport/41030.htm |title = 通用机场建设审批流程 |author =  |date = 2015-08-21 |publisher = 中国民用机场协会 |language =  |accessdate =  |quote =  }}</ref><ref name=kaihui/>。
