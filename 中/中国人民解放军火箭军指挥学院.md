'''中国人民解放军火箭军指挥学院'''，简称'''火箭军指挥学院'''，是[[中国人民解放军火箭军|中国人民解放军火箭军]]中等指挥院校。

== 沿革 ==
* 1977年11月，在[[湖北省|湖北省]][[武汉市|武汉市]][[江岸区|江岸区]]创建'''中国人民解放军第二炮兵学校'''<ref name=cnr>{{Cite web |url =http://mil.cnr.cn/zgjx/jxxl/200902/t20090202_505220592.html |title = 第二炮兵指挥学院 |author =  |date =2009-02-02  |publisher =中国广播网  |language =  |accessdate =  |quote =  }}</ref>。
* 1979年5月，升格'''中国人民解放军第二炮兵学院'''<ref name=cnr/>。正军级。
* 1986年6月，更名'''中国人民解放军第二炮兵指挥学院'''<ref name=cnr/>。正军级。
* 2016年，在[[深化国防和军队改革|深化国防和军队改革]]中，更名'''中国人民解放军火箭军指挥学院'''<ref>{{cite news|title=中国人民解放军火箭军指挥学院金寨教学基地揭牌|url=http://www.ahjinzhai.gov.cn/DocHtml/1/99/12/00085620.html|accessdate=2016-09-05|agency=金寨县人民政府|date=2016-04-25|archive-url=https://web.archive.org/web/20160924071202/http://www.ahjinzhai.gov.cn/DocHtml/1/99/12/00085620.html|archive-date=2016-09-24|dead-url=yes}}</ref>。
* 2017年，在[[深化国防和军队改革|深化国防和军队改革]]中，调整组建新的'''中国人民解放军火箭军指挥学院'''<ref>{{Cite web |url =http://www.mod.gov.cn/shouye/2017-06/29/content_4783975.htm  |title =国防部公布调整改革后军队院校名称  |author =  |date = 2017-06-29 |publisher =中华人民共和国国防部  |language =  |accessdate =  |quote =  }}</ref>。
