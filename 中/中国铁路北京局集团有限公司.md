'''中国铁路北京局集团有限公司'''，原名'''北京铁路局'''，是[[中国铁路总公司|中国铁路总公司]]下属的公司，管辖范围包括[[北京|北京]]、[[天津|天津]]兩个[[直辖市|直辖市]]和[[河北省|河北省]]的铁路，以及[[山东|山东]]、[[河南|河南]]、[[山西省|山西省]]的少部分地区<ref>[http://www.china-mor.gov.cn/dzdt/tlgxfw_small.html 铁路局管辖范围] {{webarchive|url=https://web.archive.org/web/20100329123908/http://www.china-mor.gov.cn/dzdt/tlgxfw_small.html |date=2010-03-29 }} 中华人民共和国铁道部 </ref>。至2007年末，北京鐵路局管理的線路總延展長度為7721.4公里，營業里程4688.1公里，辖有15条铁路干线和车站484个。

==沿革==
[[File:Logo_of_BJRA.jpg|thumb]]
1945年[[中国抗日战争|中国抗日战争]]胜利后，为[[中华民国交通部|中华民国交通部]]平津区铁路管理局，下设天津分局、张家口分局。1949年1月[[平津战役|平津战役]]后，接收组建“[[中国人民革命军事委员会|中国人民革命军事委员会]]铁道部平津铁路管理局”。

1993年，北京铁路局进行了工商登记<ref name=guapai/>。

2005年分拆出[[太原铁路局|太原铁路局]]，原北京铁路局管辖的山西省内铁路网转交太原铁路局管辖。

在“2007中国企业500强排行榜”中，北京铁路局排名第87位，比上一年下跌了9名<ref>[http://news.xinhuanet.com/newscenter/2007-09/01/content_6645853.htm 2007中国企业500强名单]</ref><ref>[http://www.tieliu.com.cn/Article/ShowArticle.asp?ArticleID=7807 铁路17家企业进入2006年中国企业500强] {{webarchive|url=https://web.archive.org/web/20070927223210/http://www.tieliu.com.cn/Article/ShowArticle.asp?ArticleID=7807 |date=2007-09-27 }}</ref>。北京鐵路局官網[[域名|域名]]「036」，來源於1970年代一位優秀服務員的胸牌號。<ref>{{cite news|url=http://www.gmw.cn/01gmrb/2000-05/01/GB/04%5E18407%5E0%5EGMA2-012.htm|title=北京铁路的“036”牌|publisher=[[光明日报|光明日报]]|date=2000-05-01|accessdate=2015-02-11}}</ref>

2017年4月，[[中國鐵路總公司|中國鐵路總公司]]與[[中國中鐵|中國中鐵]]轄下中鐵二院、中鐵國際及北京鐵路局合組財團，獲准進入競投[[加利福尼亞高速鐵路|加利福尼亞高速鐵路]][[民間興建營運後轉移模式|民間興建營運後轉移模式]]營運權第二階段招標，但最後由[[德國鐵路|德國鐵路]]為首財團中標。

2017年11月4日，经[[国家工商行政管理总局|国家工商行政管理总局]]核准，北京铁路局同其他10个铁路局名称变更为'''中国铁路北京局集团有限公司'''（(国)名称变核内字[2017]第5280号）<ref>{{Cite web |url =http://www.thepaper.cn/newsDetail_forward_1851596  |title = 15家铁路局将集体变身集团公司：名称变更已通过核准 |author =  |date =2017-11-06   |publisher =澎湃新闻  |language =  |accessdate =  |quote =  }}</ref>。

2017年11月15日，在国家企业信用信息公示系统中，中国铁路北京局集团有限公司已完成工商登记变更。工商登记变更后，中国铁路北京局集团有限公司仍为中国铁路总公司100%控股的公司，公司由原先的全民所有制企业变成有限责任公司（非自然人投资或控股的法人独资）企业，注册资本由原先的2144.07539亿元变更为2489.5969亿元人民币，经营范围不变<ref name=wu>{{Cite web |url = http://www.thepaper.cn/newsDetail_forward_1865510|title =北京济南等14家铁路局正式变身路局集团：完成工商登记变更|author =  |date = 2017-11-15 |publisher =澎湃新闻  |language =  |accessdate =  |quote =  }}</ref>。

2017年11月19日，中国铁路北京局集团有限公司正式挂牌<ref>{{Cite web |url =  http://www.thepaper.cn/newsDetail_forward_1870782|title = 18个铁路局已改制为铁路局集团有限公司，今日正式挂牌 |author =  |date = 2017-11-19 |publisher = 澎湃新闻 |language =  |accessdate =  |quote =  }}</ref><ref name=guapai>{{Cite web |url = http://finance.ifeng.com/a/20171119/15803099_0.shtml |title = 18家铁路局集团改制后集中挂牌！附主要领导人简历 |author =  |date = 2017-11-19 |publisher =凤凰网  |language =  |accessdate =  |quote =  }}</ref>。

*天津供电段
*石家庄供电段
*衡水供电段
*唐山供电段
