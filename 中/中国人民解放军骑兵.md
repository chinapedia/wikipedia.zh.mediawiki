'''中国人民解放军骑兵'''，是[[中国人民解放军陆军|中国人民解放军陆军]]已撤销的战斗[[兵种|兵种]]。

== 沿革 ==
=== 红军时期 ===
1928年4月，西北工农革命军骑兵队成立，这是[[中国共产党|中国共产党]]领导的军队中第一支骑兵部队<ref>{{Cite web|url =http://www.mzb.com.cn/html/report/1602247959-1.htm|title =驰骋在新时代的古老兵种|author =|date =2015-07-31|publisher =中国民族宗教网|language =|accessdate =|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20170326230732/http://www.mzb.com.cn/html/report/1602247959-1.htm|archivedate =2017-03-26}}</ref><ref name=diyi>{{Cite web |url = http://www.horsechinaone.com/view/w/2014/30/2316.html |title = 中国人民解放军骑兵史|author =  |date = 2014-08-01 |publisher = 第1赛马网 |language =  |accessdate =  |quote =  }}</ref>。

1933年9月21日，在[[陕西省|陕西省]][[耀县|耀县]]薛家寨，红二十六军第四团改编为骑兵团<ref>{{Cite web|url =http://www.htqly.org/Detail.aspx?Id=15645|title =忆红二十六军骑兵团|author =高锦纯|date =|publisher =西北革命历史网|language =|accessdate =2017-03-26|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20170327075916/http://www.htqly.org/Detail.aspx?Id=15645|archivedate =2017-03-27}}</ref><ref>{{Cite web |url = http://www.crt.com.cn/news2007/News/jryw/1311610524976875H72053EIC0D73DA.html |title = 特稿：红二十六军大事记 |author =  |date = 2013-11-06 |publisher = 中红网 |language =  |accessdate =  |quote =  }}</ref>。团长[[黄于祥|黄于祥]]，政委[[杨森_(红军)|杨森]]（后为[[张秀山|张秀山]]）<ref name=shui>水森，红军时期的骑兵部队有哪些?，社会科学1987年第05期</ref>。1935年9月18日，中央驻北方代表团在[[陕西省|陕西省]][[延川县|延川县]][[永坪镇|永坪镇]]召开中共西北工委和中共鄂豫陕省委联席会议，决定将[[红二十五军|红二十五军]]、红二十六军、[[红二十七军|红二十七军]]合编为[[中国工农红军第十五军团|中国工农红军第十五军团]]，红二十五军、红二十六军、红二十七军分别改编为七十五师、七十八师、八十一师。红二十六军第四十二师骑兵团编为红十五军团骑兵团，团长[[康健民|康健民]]，政委[[高锦纯|高锦纯]]。1935年11月上旬，红十五军团骑兵团调归中央军委直属，番号为中国工农红军骑兵第一团，康健民留任团长，[[张爱萍|张爱萍]]接任政委<ref name=zhw>{{Cite web|url = http://military.china.com/history4/62/20141215/19106195_all.html|title = 解放军骑兵第一师：驰骋疆场38个春秋英雄部队|author = |date = 2014-12-15|publisher = 中华网|language = |accessdate = |quote = |archive-url = https://web.archive.org/web/20170327075840/http://military.china.com/history4/62/20141215/19106195_all.html|archive-date = 2017-03-27|dead-url = yes}}</ref>。1936年2月，骑兵第一团划归[[刘伯承|刘伯承]]领导的援西军指挥<ref name=shui/>。

1935年11月，红十五军团骑兵团调归中央军委后，留下的原骑兵团第2连编为红十五军团直属骑兵连。1936年6月27日，75师会同73师攻占了[[宁夏|宁夏]][[同心县|豫旺县]]城，歼灭[[马鸿逵|马鸿逵]]骑兵旅一个营以及地方民团一部，缴获战马600多匹。骑兵连与红十五军团手枪团在豫旺县王家团庄合编为新的红十五军团骑兵团。1936年7月1日，中央军委授予该团番号为中国工农红军骑兵第三团<ref name=zhw/>。团长[[田维扬|田维扬]]（1936年9月[[韦杰|韦杰]]接任团长），政委夏云廷（后改名[[夏云飞|夏云飞]]），副团长[[曹子学|曹子学]]，政治处主任[[汪家道|汪家道]]。

1936年4月，[[红四方面军|红四方面军]]骑兵师组建，[[红四军|红四军]]军长[[许世友|许世友]]任红四方面军骑兵司令员兼骑兵师师长，[[秦贤道|秦贤道]]任政委。1936年5月，红四方面军骑兵师成立庆祝大会在[[甘孜|甘孜]]举行。走出草地后，由[[马良骏|马良骏]]任师长，[[张荣|张荣]]任政委；不久，由[[董俊彦|董俊彦]]任师长，秦贤道任政委，[[李庆雍|李庆雍]]任政治部主任。1936年10月下旬，红四方面军骑兵师随红四方面军总部西渡[[黄河|黄河]]，改称[[西路军|西路军]]骑兵师。1937年1月，在[[高台战斗|高台战斗]]中，西路军骑兵师大部受损，师长董俊彦、政委秦贤道、政治部主任李庆雍、第二团团长张子英等200多名指战员牺牲。1937年2月上旬，西路军总部决定，重新组建西路军骑兵师，[[杜义德|杜义德]]任师长，[[刘庆南|刘庆南]]任政委，[[李彩云|李彩云]]任参谋长。3月12日，在[[梨园口战斗|梨园口战斗]]中，西路军骑兵师被歼灭<ref name=chen>{{Cite web |url =  http://book.people.com.cn/n/2012/1113/c69360-19567364-1.html|title = 横刀立马:55位骑兵出身的开国将军 |author = 陈辉 |date =  |publisher =党史博览2009(12):28-31  |language =  |accessdate =  |quote =  }}</ref><ref>武卫国，西路军骑兵师的组建与发展，文艺生活旬刊2012(3):172-172</ref>。

1936年11月，[[红五军|红五军]]骑兵团组建，团长[[吕仁礼|吕仁礼]]，政委[[温玉成|温玉成]]。1937年1月，在[[高台战斗|高台战斗]]中，红五军骑兵团被歼灭<ref>{{Cite web |url = http://www.crt.com.cn/news2007/News/shmw/166816922251FHI7J8CG03EJD26G4.html |title = 特稿：劫后余生——吕仁礼 |author =  |date = 2016-06-08 |publisher =中红网  |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://wxrb.luaninfo.com/detail/?time=2016-10-27&id=9574&t=1 |title = 意志坚定善决断——长征中的吕仁礼 |author =  |date = 2016-10-27 |publisher = 皖西日报 |language =  |accessdate =  |quote =  }}</ref>。

1936年5月，[[中国工农红军第一方面军|中国工农红军第一方面军]]第一军团骑兵团成立于陕北保安县<ref name=han>韩增寿，铁骑纵横晋察冀——记晋察冀军区骑兵团(营)，党史博采：纪实版1997年第2期</ref>。

=== 抗日战争至第二次国共内战 ===
1937年[[抗日战争|抗日战争]]全面爆发后，根据国共两党协议，1937年8月20日，中国工农红军第一、二、四方面军和陕北红军等部改编为国民革命军第八路军（[[八路军|八路军]]），下辖[[115师|115师]]、[[120师|120师]]、[[129师|129师]]。中国工农红军骑兵第一团（中央军委骑兵团）改编为八路军第120师骑兵营。中国工农红军骑兵第三团（红十五军团骑兵团）改编为八路军第129师骑兵营<ref name=zhw/>。中国工农红军第一方面军第一军团骑兵团改编为八路军第115师骑兵营<ref name=han/>。

*八路军第115师骑兵营（后改为晋察冀军区骑兵团）：
:115师骑兵营成立后，营长[[刘云彪|刘云彪]]，政委[[萧锋|萧锋]]（1937年底[[蔡顺礼|蔡顺礼]]接任政委）。1937年8月22日，115师骑兵营在织田镇举行抗日誓师大会。1937年9月23日，115师骑兵营在[[倒马关|倒马关]]击毙日军二三十人，倒马关之战是八路军出师对日作战取得的首次胜利，拉开了[[平型关战斗|平型关战斗]]的序幕。1937年10月18日至11月11日，骑兵营配合其他部队接连收复[[曲阳县|曲阳县]]、[[唐县|唐县]]、完县（今[[顺平县|顺平县]]）、[[满城县|满城县]]4座县城，为开辟[[晋察冀边区|晋察冀边区]]北岳抗日根据地奠定了基础。1937年11月，[[晋察冀军区|晋察冀军区]]成立，[[聂荣臻|聂荣臻]]任司令员兼政委，骑兵营归属晋察冀军区指挥。此后骑兵营向[[保定|保定]]活动，深入[[平汉铁路|平汉铁路]]以东游击，破坏铁路，制止敌南进。骑兵营先后袭击保定西关车站、[[于家庄站|于家庄站]]、[[方顺桥站|方顺桥站]]、[[望都站|望都站]]。1940年初，骑兵营扩建为晋察冀军区骑兵团，刘云彪任团长，蔡顺礼任政委，[[范昌标|范昌标]]任副团长兼一营营长，[[包镇|包镇]]任参谋长，下辖4个营，共计5000余人。1940年8月[[百团大战|百团大战]]爆发后，骑兵团发动群众开展了数十次破路战斗。1941年初，因日军推进据点及封锁沟，骑兵活动受限，马匹补充困难，晋察冀军区骑兵团改编为一营步兵、一营骑兵及团直等3个单位。1941年5月、7月、11月，日军3次实施“强化治安”运动，修建三道防线，晋察冀军区骑兵团相应建立三道防线。1942年4月12日，团长刘云彪病逝。1942年1月，政委蔡顺礼调离，[[曾海亭|曾海亭]]接任政委。范昌标调抗大学习，包镇接任副团长，[[钟天发|钟天发]]接任参谋长。1942年7月，晋察冀军区三分区政委[[王平_(上将)|王平]]到骑兵团团部驻地北店头村传达[[聂荣臻|聂荣臻]]司令员的指示，将骑兵团骑兵改为步兵，番号不变。团长[[李钟奇|李钟奇]]，政委曾海亭。全团编为两个营，每营3个连，加上团部特务连、侦察连，合计11个连约两千余人。1943年1月，为粉碎日军“铁壁合围”，晋察冀军区决定把大团改为小团，部队分散活动。骑兵团取消营，改编成4个步兵连，一个侦察连，一个特务连，加上团部，合计1225人，李钟奇调任抗大二分校总队长，[[唐子安|唐子安]]接任团长。1943年初，骑兵团转至完县山前活动。1943年5月初，日军包围晋察冀军区三分区驻地，骑兵团完成保护三分区机关突围任务后，于5月7日在完县马耳山同日军3个团遭遇，政委曾海亭等400多人牺牲。1943年10月，范昌标自抗大调回骑兵团任团长，[[熊光焰|熊光焰]]任政委，[[苏洪道|苏洪道]]任参谋长。1944年3月，骑兵团在唐县杨家会整编，与四十一团合编成1个团，并与5个兄弟团组成野战旅，4月调到陕北参加保卫延安战斗<ref name=han/><ref>{{Cite web |url = http://dangshi.people.com.cn/n/2015/0826/c85037-27519653.html |title =  逢敌必亮剑——记抗日烽火中的晋察冀军区骑兵团|author =  |date = 2015-08-26 |publisher = 中国共产党新闻网 |language =  |accessdate =  |quote =  }}</ref>。

*八路军第129师骑兵营（后改为八路军第129师骑兵团、晋冀鲁豫野战军第七纵队直属骑兵团）：
:1937年11月，[[日军|日军]]占领[[山西省|山西省]][[阳泉市|阳泉]]，随后以部分兵力进犯八路军129师师部驻地[[和顺县|和顺县]]城，师长[[刘伯承|刘伯承]]命129师骑兵营阻击。和顺阻击战是骑兵营第一次单独同日军交战，歼敌100多人，成功完成了掩护129师师部转移的任务。1937年11月，根据[[中共中央|中共中央]]指示，八路军129师进入依托[[太行山|太行山]]区的晋冀豫地区开创抗日根据地。129师骑兵营进入冀西后，先后在[[赞皇县|赞皇县]]、[[元氏县|元氏县]]、[[临城县|临城县]]歼俘土匪1000多人，歼灭伪军200多人。1937年11月30日，日军为保障[[平汉铁路|平汉铁路]]安全，调集日军200多人，分乘18辆汽车，计划围歼129师骑兵营。129师骑兵营主力在北马村设伏，夜战歼灭日军140多人，烧毁敌汽车18辆。129师师长刘伯承称赞“北马村战斗是一次诱敌深入的歼灭战”<ref name=zhw/>。

:1938年2月初，129师骑兵营自冯村、田家口之间地域跨越铁路，占领[[宁晋县|宁晋县]]城，歼灭伪军100多人，俘虏汉奸[[王怀庆|王怀庆]]。2月8日，129师骑兵营扩编为129师骑兵团<ref name=zhw/>。团长夏云飞（1938年2月中旬由[[王振祥|王振祥]]继任团长），政委[[邓永耀|邓永耀]]（1938年4月邓永耀升任东进纵队政治部主任，由[[赵青山|赵青山]]代理政委），副团长[[梁俊亭|梁俊亭]]。（邓永耀升任东进纵队政治部主任后，团长梁俊亭，政委王振祥，副团长[[黄家景|黄家景]]。）<ref> {{Cite book | author =黄晓峰  | title =骑兵团：亲历者口述一二九师铁血骑兵抗战纪实  | location =  | publisher = 西苑出版社 | date = 2013年 | pages =  | ISBN =  9787515103037 | accessdate =  | url =  | language =  | quote =  }} </ref>1938年，129师骑兵团参谋长为[[何正文|何正文]]，政治处主任为[[况玉纯|况玉纯]]。3月25日，骑兵团在司令员[[陈再道|陈再道]]指挥下，在郑家庄等地全歼日军任命的保安司令刘魔头，毙俘敌900余人，毙日军50多人。5月到6月，骑兵团陆续攻克[[永年县|永年县]]、[[成安县|成安县]]、[[魏县|魏县]]等十座县城，毙伤日军130多人，击毁敌汽车27辆，毙俘伪军2500多人。7月，129师政委[[邓小平|邓小平]]抵达永年县，到骑兵团看望了指战员。1939年1月，日军[[第10师团|第10师团]]等部3万多人，分成11路大规模“扫荡”冀南根据地，计划消灭八路军129师师部及主力部队。129师骑兵团掩护129师机关转移后，在刘伯承、邓小平指挥下，配合冀南根据地军民开展反“扫荡”斗争，进行战斗100多次，毙伤日军、伪军3000多人，粉碎了日军控制冀南平原的计划<ref name=zhw/>。

:1943年，该骑兵团在[[河南省|河南省]][[新乡县|新乡县]]城东台头村全歼[[贺兰亭|贺兰亭]]部1个团，创造出骑兵团单独全歼一个步兵团的战例。1945年10月，该骑兵团配合[[冀鲁豫军区|冀鲁豫军区]]发动[[邯郸战役|邯郸战役]]，毙俘敌3000多人。1945年11月，冀鲁豫军区组成[[晋冀鲁豫野战军|晋冀鲁豫野战军]]第七纵队，司令员[[杨勇|杨勇]]，八路军第129师骑兵团改编为晋冀鲁豫野战军第七纵队直属骑兵团<ref name=zhw/>。

:[[第二次国共内战|全面内战]]爆发后，根据中央军委指示，刘伯承、邓小平指挥晋冀鲁豫野战军挺进豫东。晋冀鲁豫野战军第七纵队直属骑兵团随主力部队参加了[[陇海路战役|陇海路战役]]及[[定陶战役|定陶战役]]。1947年1月15日，骑兵团在定陶截击敌增援[[巨金鱼战役|巨金鱼战役]]的整编68师、55师各一部，歼俘敌2000多人。1947年3月，晋冀鲁豫野战军第一、第七纵队合编为第一纵队，骑兵团改隶第一纵队，成为晋冀鲁豫野战军第一纵队直属骑兵团。此后，骑兵团参加[[豫北战役|豫北战役]]、[[鲁西南战役|鲁西南战役]]。1947年7月14日，在鲁西南战役围歼敌32师、70师共计3个半旅的六营集战斗中，仅仅骑兵团2营便全歼敌70师1个团，俘敌团长李金成以下550多人。1947年8月11日，根据中央军委指示，刘伯承、邓小平开始[[挺进大别山|挺进大别山]]，骑兵团被编为右路军后卫。1947年8月17日，骑兵团通过[[黄泛区|黄泛区]]，多次打退敌骑兵一旅的进攻，掩护主力部队成功进入[[大别山|大别山]]区<ref name=zhw/>。

:1948年底，晋冀鲁豫野战军第一纵队骑兵团参加了[[淮海战役|淮海战役]]，在新蔡、埠阳等地牵制侧击敌12兵团[[黄维|黄维]]部，以迟滞敌行动。1948年12月5日，敌黄维兵团被歼灭后，骑兵团调赴[[淮阴县|淮阴县]]承担晋冀鲁豫野战军机关警卫任务，直到淮海战役结束<ref name=zhw/>。

*八路军第120师骑兵营（后改为五县河防警备司令部骑兵团、八路军留守兵团骑兵团、陕甘宁晋绥联防军骑兵旅。后与塞北军分区骑兵第2、3团合编为绥蒙军区骑兵旅）：
:1937年8月20日到24日，[[中共中央政治局|中共中央政治局]]在陕北[[洛川县|洛川县]]冯家村召开扩大会议（即[[洛川会议|洛川会议]]），中央军委骑兵团奉命进驻冯家村周边承担会议警戒任务。此后，中央军委骑兵团改编为八路军第120师骑兵营。9月3日，骑兵营脱离120师建制，暂编为[[绥德|绥德]]、[[米脂|米脂]]等五县河防警备司令部骑兵团。12月，中央军委将八路军[[延安|延安]]总部留守处改编成[[陕甘宁边区|陕甘宁边区]]留守兵团，骑兵团调归陕甘宁边区留守兵团建制，番号为留守兵团骑兵团。1943年3月，为贯彻中共中央的“精兵简政”指示，留守兵团骑兵团、保安骑兵团、359旅骑兵大队的1、2中队，合编为[[陕甘宁晋绥联防军|陕甘宁晋绥联防军]]骑兵旅。5月，骑兵旅赴[[富县|富县]]待命，粉碎了[[胡宗南|胡宗南]]兵分九路进攻[[陕甘宁边区|陕甘宁边区]]的计划。1945年4月上旬，陕甘宁晋绥联防军骑兵旅自[[南泥湾|南泥湾]]启程东渡黄河，开赴大青山抗日游击根据地<ref name=zhw/>。

*大青山骑兵支队（撤编后部队改为塞北军分区骑兵第1、2、3团。后骑兵第2、3团与陕甘宁晋绥联防军骑兵旅合编为绥蒙军区骑兵旅）：
:1938年6月，根据中共中央开辟[[大青山抗日根据地|大青山抗日游击根据地]]的决策及八路军总部的决定，120师以358旅715团和师直骑兵连，组成大青山抗日游击支队。[[李井泉|李井泉]]任支队长，[[姚喆|姚喆]]任支队参谋长，[[彭德大|彭德大]]任支队政治部主任。1939年6月，大青山抗日游击支队改编为大青山骑兵支队，下辖三个骑兵营。李井泉任司令员（1940年5月姚喆接任司令员），姚喆任副司令员。1940年4月到12月，日军连续11次大规模“扫荡”大青山抗日游击根据地，大青山骑兵支队同日军、伪军战斗95次，毙敌794人，俘敌193人。1940年底，根据[[毛泽东|毛泽东]]、[[朱德|朱德]]给八路军120师师长[[贺龙|贺龙]]《关于将大青山骑兵支队营建制改为团的建制的请示》的批复，大青山骑兵支队下辖的三个骑兵营分别改成骑兵团。1942年春夏，日军连续“扫荡”大青山抗日游击根据地，大青山骑兵支队对日军、伪军昼夜轮番游击作战。7月，日军调集2万多兵力，对大青山抗日游击根据地发动秋季大“扫荡”，大青山骑兵支队分批分期转战至绥西、绥中。1942年底，[[中共中央晋绥分局|中共中央晋绥分局]]及晋绥军区为统一对绥远、雁北工作的领导，取消大青山骑兵支队番号，原骑兵1、2、3团番号不变，转隶晋绥军区塞北军分区。1944年到1945年，日军被迫从[[绥远省|绥远省]]不少据点撤出，[[中国国民党|中国国民党]]军队趁机抢占战略要地，并且向大青山地区增派军队以控制大青山地区。塞北军分区骑兵第2、3团进行反击。1945年8月，毛泽东发表《对日寇的最后一战》的声明后，姚喆奉命集中原大青山骑兵支队兵力，配合晋绥军区主力及晋察冀军区一部，截断[[平绥铁路|平绥铁路]]，收复沿线的[[丰镇市|丰镇]]、[[集宁区|集宁]]等等城镇。1945年9月，绥蒙军区组建，司令员姚喆、政委[[张达志|张达志]]。陕甘宁晋绥联防军骑兵旅、塞北军分区骑兵2、3团在商都地区合编为绥蒙军区骑兵旅<ref name=zhw/><ref name=xiaohe>晓何，情系大青山的骑兵司令姚喆，党史纵横2013年09期</ref>。

*平北骑兵大队：
:1942年1月，[[晋察冀军区|晋察冀军区]]平北军分区创建骑兵大队，晋察冀军区领导[[聂荣臻|聂荣臻]]很关心这支骑兵大队。根据平北军分区的报告，聂荣臻指示晋察冀骑兵团抽调干部到平北骑兵大队<ref name=zhw/>。

*察北骑兵：
:1945年9月，晋察冀军区成立察北军分区，下辖三个骑兵团。1946年到1948年，察北三个骑兵团在察北坚持斗争的同时，先后参加[[热西战役|热西战役]]及[[察南绥东战役|察南绥东战役]]，大量歼敌，保证了[[隆化战斗|隆化战斗]]的胜利，受到[[冀热察军区|冀热察军区]]记大功一次<ref name=zhw/>。

*绥蒙军区骑兵旅：
:1945年9月，绥蒙军区组建，司令员姚喆、政委[[张达志|张达志]]。陕甘宁晋绥联防军骑兵旅、塞北军分区骑兵2、3团在商都地区合编为绥蒙军区骑兵旅。1945年11月30日，绥蒙军区骑兵旅在[[武川县|武川县]]蒙克寺大东滩经13个小时战斗，毙敌657人，受到中央军委通令嘉奖，全旅荣记大功一次。1948年1月，绥蒙军区趁[[傅作义|傅作义]]部主力东调的机会，西渡黄河。1948年1月14日，绥蒙军区骑兵旅西渡黄河，击溃[[河曲县|河曲县]]的[[朱玉美|朱玉美]]部。1948年3月20日到4月上旬，绥蒙军区骑兵旅配合[[晋察冀野战军|晋察冀野战军]]发动[[察南绥东战役|察南绥东战役]]，歼敌18000多人<ref name=zhw/>。

:1948年7月，绥蒙军区骑兵旅编为[[西北野战军|西北野战军]]第八纵队骑兵旅。在配合[[辽沈战役|辽沈战役]]发动的[[察绥战役|察绥战役]]中，西北野战军第八纵队骑兵旅、察北骑兵团先后作战16次，歼敌1484人。1948年11月底，西北野战军第八纵队骑兵旅配合主力部队参加了[[平津战役|平津战役]]。12月8日，骑兵旅越过平绥铁路，在[[怀来县|怀来县]]以南十八家一带，抗击了敌2个旅攻击。12月18日，骑兵旅参与围追堵截[[张家口|张家口]]之敌，此战除敌第11兵团司令[[孙兰峰|孙兰峰]]率少数骑兵逃脱外，敌54000多人被全歼<ref name=zhw/>。

:1949年2月12日，西北野战军第八纵队骑兵旅、察北军区3个骑兵团以及蒙骑一部，奉命开入绥北发动围剿绥北敌骑战役。1949年2月15日至20日，歼灭了自[[张北县|张北县]]开到绥北乌兰花、武川一带的敌骑第12旅及敌第11兵团司令孙兰峰残部，歼敌900多人。1949年3月，刚改编而成的中国人民解放军骑兵第一师南下雁北，协同[[察哈尔军区|察哈尔军区]]围歼[[大同市|大同]]守敌。4月上旬，配合[[华北军区|华北军区]]副司令员[[徐向前|徐向前]]指挥的[[太原战役|太原战役]]。4月24日太原战役结束，大同守敌万余人于4月29日接受中国人民解放军整编，大同和平解放<ref name=zhw/>。

*[[新四军|新四军]]第四师骑兵团：
:1940年6月1日，新四军第六支队（后来改成新四军第四师）集中八个团以及[[中国人民抗日军政大学|中国人民抗日军政大学]]分校等在司令部驻地[[安徽省|安徽省]][[涡阳县|涡阳县]]新兴集召开大会时，[[宿县|宿县]]西部各据点的日军、伪军乘坐汽车奔袭至会场附近。新四军第六支队司令员兼政委[[彭雪枫|彭雪枫]]经与参谋长[[张震_(上将)|张震]]等商议后，决定新四军第六支队成立骑兵连，各旅、团先后成立骑兵排、班。1941年4月，新四军第四师与[[汤恩伯|汤恩伯]]节制的青海骑兵第八师在[[津浦铁路|津浦铁路]]以西地区战斗，新四军第四师伤亡惨重。战后，新四军第四师师长彭雪枫决定将师部骑兵连扩充为新四军第四师骑兵团，号称“红色哥萨克”<ref name=chen/>。[[黎同新|黎同新]]为第一任团长兼政委。不久，[[周纯麟|周纯麟]]接任团长，[[康步云|康步云]]、[[姚运良|姚运良]]先后任政委<ref> {{Cite book | author = 曹鴻茂、孟秀玲、周善平 | title = 新四軍騎兵團征戰發展紀實 | location =  | publisher = 軍事科學出版社 | date = 2009年 | pages =  | ISBN = 9787802372238 | accessdate =  | url =  | language =  | quote =  }} </ref><ref>新四军第四师骑兵团照片，档案与建设2015年第1期</ref>。1941年冬，彭雪枫决定骑兵团开展短期突击训练。1942年，为保卫[[洪泽湖|洪泽湖]]地区夏收，骑兵团奉命消灭在沙山集抢粮的日军，仅用9分钟便将300多名日军砍倒大半，俘虏80多人<ref name=chen/>。1942年11月13日，骑兵团参加了为期33天的反“扫荡”作战，参加战斗10余次，毙伤俘日伪军200余人。1943年，日伪军对新四军根据地开展“蚕食”和“伪化”，新四军进袭日伪军薄弱据点，扩大[[津浦铁路|津浦铁路]]以东抗日根据地，这时骑兵团已扩为4个大队，1000多人马<ref name=chch>{{Cite web |url = http://roll.sohu.com/20130821/n384665073.shtml |title =叱咤抗日战场的 “红色哥萨克”（下）  |author =  |date = 2013-08-21 |publisher = 搜狐 |language =  |accessdate =  |quote =  }}</ref>。

*中国人民解放军骑兵第四师：1950年2月，绥远起义的原国民党整编骑兵第12旅[[鄂友三|鄂友三]]部改编而成，师长鄂友三，政治委员[[白正刚|白正刚]]。  
*中国人民解放军骑兵第五师：原为[[中国人民解放军第四野战军|中国人民解放军第四野战军]]骑兵第一师，师长[[何能彬|何能彬]]/[[田维扬|田维扬]]/[[王明贵|王明贵]]，政委[[林茂源|林茂源]]，副师长[[钟明彪|钟明彪]]，政治部主任[[刘克|刘克]]。前身是[[冀热辽军区|冀热辽军区]][[朱德|朱德]]骑兵师。
*中国人民解放军骑兵第六师：[[平津战役|平津战役]]中在北平接受和平改编的国民党新编骑兵4师，1949年2月先称独立骑兵第五师，师长[[刘春方|刘春方]]，政委[[李赤然|李赤然]]/[[刘光裕|刘光裕]]/刘兴畴/王功学。1949年7月改称中国人民解放军骑兵第六师，隶属第十九兵团进军西北，驻宁夏中宁、吴忠等地。
*中国人民解放军骑兵第七师：1949年12月20日新疆起义部队原国民党整编骑兵第1师改编为骑兵第7师，驻奇台，师长[[韩有文|韩有文]]，政治委员[[于春山|于春山]]，第一副师长韩荣福，第二副师长郭全梁，第三副师长马全吉，参谋长李纲，政治部主任杨贯之。
*中国人民解放军骑兵第八师：1949年12月20日新疆起义部队原国民党整编第42师骑兵部队改编为骑兵第8师，驻莎车，师长[[马平林|马平林]]，政治委员[[张献奎|张献奎]]，第一副师长李朝弻，第二副师长刘抡元，参谋长祝源开，政治部主任杨烈光。 
*[[中国人民解放军新疆军区|中国人民解放军新疆军区]]独立骑兵师：1950年4月，新疆军区抽调骑兵8师23团、第2军骑兵大队和2军5师独立团，组建独立骑兵师，进驻于田，准备进军西藏[[阿里地区|阿里]]。师长兼政委[[何家产|何家产]]。 
*[[中国人民解放军内蒙古军区|中国人民解放军内蒙古军区]]骑兵第一师：由[[兴安盟|兴安盟]][[蒙古族|蒙古族]]武装组建，参加了西满解放战争的剿匪、围困长春、辽沈战役。
*中国人民解放军内蒙古军区骑兵第二师：[[哲里木盟|哲里木盟]]蒙古族武装组建，参加了西满解放战争的剿匪、围困长春、辽沈战役。
*中国人民解放军内蒙古军区骑兵第三师：[[冀察热辽军区|冀察热辽军区]]（二级军区）下属的昭乌达盟骑兵第4师与卓索图盟纵队于48年9月合编为内蒙古人民解放军骑兵第10师，1949年5月改称内蒙古骑兵第3师。
*中国人民解放军内蒙古军区骑兵第四师：锡察解放区，1949年5月由骑兵11师改称。原日伪蒙疆政府的骑兵部队在1945年8月起义部队发展。
*中国人民解放军内蒙古军区骑兵第五师：锡察解放区，1949年5月由骑兵16师改称。由中共在内蒙古的党组织在1946年初创建的部队。  

此外，一些部队还下辖骑兵团：
*中国人民解放军第一军骑兵团：解放青海省之后由于剿匪需要，于1950年由第1军军直侦察连、第1、2、3师侦察连为基础组建。
*中国人民解放军第二军骑兵团：和平解放新疆进驻南疆需要。
*中国人民解放军第三军骑兵大队：解放[[河西走廊|河西走廊]]后，由军直侦察连和各师侦察连为基础组建。
*中国人民解放军第三军骆驼骑兵团：收编起义部队国民党[[西北行营|西北行营]]独立骆驼兵团。团长贺新民。
*中国人民解放军第四军骑兵团：解放陇南后剿匪需要。
*中国人民解放军第十八军骑兵支队：[[第一野战军|第一野战军]]组建的部队，从青海进军西藏，参加了[[昌都战役|昌都战役]]的大迂回。转隶第18军领导。
*[[中国人民解放军第三野战军|中国人民解放军第三野战军]]教导师骑兵第二十五团（原新四军第四师骑兵团）：团长[[程朝先|程朝先]]、政委[[郭萍|郭萍]]、副团长[[朱传贤|朱传贤]]、副政委[[席华亭|席华亭]]，参谋长[[孟昭贤|孟昭贤]]、政治处主任[[王开一|王开一]]。留在皖北剿匪。

1952年5月，根据中央军委制定的《军事整编计划》，中国人民解放军骑兵第一师、骑兵第三师、河南省军区骑兵团（原八路军冀鲁豫军区骑兵团）3支骑兵部队于察哈尔省[[万全县|万全县]]合编为新的中国人民解放军骑兵第一师。1952年11月，中共中央、中央军委命令骑兵第一师转隶[[西北军区|西北军区]]建制，开赴甘、青、川边执行剿匪任务。1953年，骑兵第一师等部队全歼[[马良|马良]]、[[马元祥|马元祥]]部，击毙马元祥，活捉马良。5月28日，中共中央、中央军委表彰了甘、青、川边剿匪部队。1959年3月26日至10月底，骑兵第一师等部队参加[[1959年藏区骚乱|西藏平叛]]。1962年，中央军委命令骑兵第一师进驻[[新疆|新疆]]担负战备防御任务。1964年8月11日，根据[[中国人民解放军总参谋部|中国人民解放军总参谋部]]批复编制调整，骑兵第一师除留骑兵第3团外，其他部队按照陆军步兵南方甲种师编制，使骑兵第一师从单一的骑兵部队扩编成步、骑混编的骑兵师。1969年[[珍宝岛事件|珍宝岛事件]]后，1969年9月16日，根据中央军委指示，骑兵第一师改番号为陆军第八师<ref name=zhw/>。

1951年2月，中国人民解放军骑兵第五师改编为步兵第166师，军马调入[[中国人民志愿军|中国人民志愿军]]炮兵部队。 

1950年11月，撤销了内蒙古骑兵第一师、第二师、第三师建制，成立了东蒙的6个军分区。  

1952年5月，撤销了内蒙古骑兵第四师和第五师，并从6个军分区所辖独立骑兵团中各抽调一部分，重新组建中国人民解放军（国防）骑兵第五师，归[[华北军区|华北军区]]建制。

===兵种撤销===
1985年[[百万大裁军|百万大裁军]]后，摩托化和机械化取代了骡马化，骑兵不再作为中国人民解放军兵种，仅象征性地在中国西部的[[内蒙古|内蒙古]]、[[新疆|新疆]]、[[甘肃|甘肃]]保留了两个骑兵营和几个骑兵连。仅有的两个骑兵营是[[内蒙古军区|内蒙古军区]]骑兵第一营（驻[[巴彦淖尔盟|巴彦淖尔盟]][[五原县|五原县]][[阴山|阴山]]脚下）、骑兵第二营（驻[[锡林郭勒盟|锡林郭勒盟]][[锡林浩特市|锡林浩特市]]）<ref name=diyi/><ref>{{Cite web |url = http://news.sohu.com/88/47/news146434788.shtml |title =特写:我军最后骑兵部队-内蒙古军区骑兵第一营  |author =  |date = 2001-08-31 |publisher = 搜狐 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://news.xinhuanet.com/mil/2009-11/17/content_12476541.htm |title = 探访骑兵第二营:铁骑已远去 英雄不孤独 |author =  |date =2009-11-17  |publisher = 新华网|language =  |accessdate =  |quote =  }}</ref>。

在内蒙古军区骑兵第二营的驻地[[锡林郭勒盟|锡林郭勒盟]]建立了“[[中国人民解放军骑兵博物馆|中国人民解放军骑兵博物馆]]”<ref name=diyi/>。

== 军马生产管理 ==
中央军委马政局最初由[[中国人民解放军炮兵|军委炮兵]]代管，为中国人民解放军炮兵司令部马政局。局长[[杨宗胜|杨宗胜]]，副局长[[卜云龙|卜云龙]]。1950年6月，载涛作为特邀人士出席[[中国人民政治协商会议第一届全国委员会|中国人民政治协商会议第一届全国委员会]]第二次会议，提出“改良马种以利军用”的提案。1950年8月10日，[[中央人民政府人民革命军事委员会|中央人民政府人民革命军事委员会]]主席[[毛泽东|毛泽东]]签署委任令，任命[[载涛|载涛]]为中国人民解放军炮兵司令部马政局顾问。1951年10月，马政局转隶[[中国人民解放军总后勤部|总后勤部]]<ref>{{Cite web |url =  http://www.thepaper.cn/newsDetail_forward_1648205|title = 从清朝贝勒到新中国马政顾问：军报撰文回忆载涛与骑兵发展 |author =  |date = 2017-03-26 |publisher = 澎湃新闻 |language =  |accessdate =  |quote =  }}</ref>。

在各地设立军马场，发展军马生产。1947年3月1日创建的牡丹江军马场，是解放军首次创办的军马场，场长[[郑新潮|郑新潮]]，隶属于[[东北民主联军|东北民主联军]]炮兵后勤部。
