'''中国铁道博物馆'''（{{lang-en|China Railway Museum }}）是[[中国|中国]]铁路唯一的国家级专业博物馆，位于[[北京市|北京市]]。作为隶属于[[中国铁路总公司|中国铁路总公司]]的[[事业单位|事业单位]]，'''中国铁道博物馆'''主要负责铁路文物、科研成果等展品的收藏、保管、陈列、展示及编辑研究工作，促进铁路科技现代化，作为爱国主义宣传教育和科学普及教育的基地。中国铁道博物馆有三个馆：正阳门馆、东郊馆、[[詹天佑纪念馆|詹天佑纪念馆]]。<ref>{{cite web|author1=中国铁道博物馆|title=关于铁博——历史沿革|url=http://www.china-rail.org/zjtb/gzzc/201707/t20170703_534.html}}</ref>

==历史==
[[File:ChinaRailwayMuseum-The_locomotive_exhibition_hall.JPG|thumb]]
中国铁道博物馆的前身是成立于1978年的铁道部科学技术馆。其定位为[[中华人民共和国铁道部|铁道部]]宣传[[中华人民共和国铁路运输|中国铁路]]科学技术现代化的官方单位。主要任务包括配合铁道部科教司、[[中国铁道学会|中国铁道学会]]参与中国铁路科学技术交流和普及工作；参加组织和筹办国内外展览会；组织和筹办以铁路科学技术为主题的展览会；对中国铁路建设成就、科技成果编写出版画册及技术说明，制作展览[[模型|模型]]等；对铁路历史文物进行搜藏、展示以及保护工作。但铁道部科学技术馆成立之初并无正式的馆址。1991年6月，铁道部拟建中国铁道博物馆或博览中心，馆址选在铁道部科学研究院实验基地附近。1993年3月，铁道部科学技术馆由铁道科学研究院代管。<ref>关于调整部科学技术馆领导体制的决定（铁编〔1993〕1号文件）</ref>

2000年开始，铁道部科学技术馆开始筹划建立国家级的铁路博物馆，并进行大量前期工作，征集了大量的铁路文物史料。2002年初，位于北京市朝阳区的“铁道部科研试验展示交流中心机车展示库”开建，场地邻近北京大山子[[中国国家铁道试验中心（东郊分院）|环形铁道]]。2002年11月2日，中国铁道部与[[中国大陆|中国大陆]]著名[[火车迷|铁路迷]]社区海子铁路网合作，新建成的铁道部科学技术馆特别对全国铁路爱好者试验性开放展览一天<ref>{{Cite web |url=http://club.travel.sina.com.cn/viewthread.php?tid=37828 |title=关注北京的铁道博物馆 |accessdate=2010-09-04 |publisher=新浪 |date=2002-10-30 }}</ref><ref>{{Cite web |url=http://www.hasea.com/gettogether/2002-11-2.htm |title=车迷有约 2002.11.2 北京 |accessdate=2010-09-04 |publisher=海子铁路网 |date= |deadurl=yes |archiveurl=https://web.archive.org/web/20061128132448/http://www.hasea.com/gettogether/2002-11-2.htm |archivedate=2006-11-28 }}</ref>，机车车辆陈列厅同日正式对外展出，时任铁道部部长[[傅志寰|傅志寰]]亦出席了开馆典礼。

2003年9月，根据中央编制委员会办公室中央编办复字〔2003〕113号文件《关于铁道部科学技术馆更名为中国铁道博物馆的批复》，铁道部决定铁道部科学技术馆更名为中国铁道博物馆。<ref>{{Cite web|url=http://www.china-rail.org/lsyg.html|title=关于铁博-历史沿革|accessdate=2014-08-14|publisher=中国铁道博物馆|date=|deadurl=yes|archiveurl=https://web.archive.org/web/20140814071850/http://www.china-rail.org/lsyg.html|archivedate=2014-08-14}}</ref> 9月22日，铁道部发布铁劳卫函〔2003〕384号文件《关于铁道部科学技术馆更名为中国铁道博物馆等事项的通知》，铁道部科学技术馆正式更名为中国铁道博物馆<ref>{{Cite web |url=http://www.china-mor.gov.cn/tdgk/tdgk_2004.html |title=2003年铁道概况【铁道部机构及所属单位调整】 |accessdate=2010-09-04 |publisher=中华人民共和国铁道部 |date= |deadurl=yes |archiveurl=https://web.archive.org/web/20100919101255/http://china-mor.gov.cn/tdgk/tdgk_2004.html |archivedate=2010-09-19 }}</ref>。

中国铁道博物馆东郊馆主体包括机车车辆陈列厅、综合展厅和铁路专题展厅。占地157334平方米，现建筑面积20500平方米。馆内设综合展厅建筑面积4000平方米，展示有关中国铁路历史展览。还设有機車車輛陳列廳，陳列廳建築面積16500平方米，廳內有8條展示線路，可以同時展示80-90輛铁路機車車輛，包括历代铁路机车、客车等。此外，展馆东侧还有一个车站外形的影视拍摄基地，电视剧《[[历史转折中的邓小平|历史转折中的邓小平]]》中的部分场景即在此拍摄。<ref>{{cite web|url=http://www.china-rail.org/aqgl/tbupload/tbInfo476.htm|title=电影《邓小平在黄山》在东郊馆取景拍摄|publisher=中国铁道博物馆|date=2014-10-30|accessdate=2015-07-10|deadurl=yes|archiveurl=https://web.archive.org/web/20150711074051/http://www.china-rail.org/aqgl/tbupload/tbInfo476.htm|archivedate=2015-07-11}}</ref>
