'''中川昭一'''（{{bd|1953年|7月19日|2009年|10月4日|catIdx=Nakagawa Shouichi}}），為[[日本|日本]][[政治家|政治家]]。

曾擔任[[日本眾議院|日本眾議院]]議員（8次當選）、[[農林水産大臣|農林水産大臣]]（第27、41代）、[[經濟產業大臣|經濟產業大臣]]（第3、4、5代）、[[財務大臣|財務大臣]]（第10代）、兼[[特命擔當大臣|特命擔當大臣]]（金融擔當）。

本籍[[北海道|北海道]][[廣尾郡|廣尾郡]]，生於[[東京|東京]][[澀谷區|澀谷區]]，1978年3月自[[東京大學|東京大學]]法學部政治學科畢業。中川昭一是第二代[[世襲|世襲]]議員。其父親[[中川一郎|中川一郎]]曾為中川派 (自由革新同友会)的領袖，為[[日本眾議院|日本眾議院]]議員（7次當選），曾擔當[[科學技術廳長官|科學技術廳長官]]及[[農林水産大臣|農林水産大臣]]；其叔父[[中川義雄|中川義雄]]為現任[[日本參議院|日本參議院]]議員（2次當選）。

== 仕途 ==
1983年12月初次當選眾議院議員，曾先後擔任[[經濟產業大臣|經濟產業大臣]]、[[農林水產大臣|農林水產大臣]]、[[自由民主黨政務調查會|自由民主黨政調會長]]等職。中川昭一為「[[中國威脅論|中國威脅論]]」提倡者。2007年2月27日在[[名古屋|名古屋]]演講中提出：「今後十五年[[台灣|台灣]]若發生問題，二十年之內日本可能成為中國的一省。」言论，強調[[中華人民共和國|中華人民共和國]]軍事擴張的嚴重性<ref>[http://www.libertytimes.com.tw/2007/new/feb/28/today-p7.htm 中國軍事擴張威脅 中川擔憂：日恐成中國一省] {{webarchive|url=https://web.archive.org/web/20070929132908/http://www.libertytimes.com.tw/2007/new/feb/28/today-p7.htm |date=2007-09-29 }}。</ref>
，引起不少中國人震驚。他在日本政論月刊《[[諸君|諸君]]》專訪時也曾提出要抵制「在一年內中國如武力犯台或實施大規模的鎮壓人權行動，日本有必要考慮杯葛[[2008年夏季奥林匹克运动会|北京奧運]]」<ref>[http://www.libertytimes.com.tw/2007/new/mar/2/today-p10.htm 日自民黨政調會長：中若犯台 日應杯葛北京奧運] {{webarchive|url=https://web.archive.org/web/20070929133735/http://www.libertytimes.com.tw/2007/new/mar/2/today-p10.htm |date=2007-09-29 }}。</ref>
。同年4月4日，中川在[[札幌|札幌]]演講中，曾以比喻對於中日[[东海油气田问题|東海油氣田開採爭議]]暗指中國為「小偷」。<ref>[http://www.libertytimes.com.tw/2007/new/apr/6/today-int5.htm 東海油田爭議 日暗批中是小偷] {{webarchive|url=https://web.archive.org/web/20070815021444/http://www.libertytimes.com.tw/2007/new/apr/6/today-int5.htm |date=2007-08-15 }}，2007年4月6日[[自由時報|自由時報]]，</ref>引起國際爭議。

在[[第45屆日本眾議院議員總選舉|第45屆日本眾議院議員總選舉]]，中川在[[北海道|北海道]]11區爭取第9度連任失敗。由於惜敗率不及[[町村信孝|町村信孝]]及[[武部勤|武部勤]]，未能在北海道比例代表中復活當選。

== G7記者會失態請辭和落選議員 ==
2009年2月14日，一向愛好杯中之物的中川在[[羅馬|羅馬]][[八大工業國組織|八大工業國組織]]財長會議記者會時，因神情呆滯，精神恍惚而致把記者問題弄錯；記者會被日本廣泛報道，及後並傳至亞洲與美國的新聞報道，剛巧遇上現任首相[[麻生太郎|麻生太郎]]民望少於一成的時間；引起日本朝野指責其表現令國家蒙羞，如前首相[[森喜朗|森喜朗]]、民主黨[[小澤一郎|小澤一郎]]等，外間猜測中川因喝酒所致<ref>{{cite news|url=http://hk.news.yahoo.com/article/090216/4/apx2.html|title=醉醺醺開內閣會議|publisher=明報|date=2009-02-17|language=|deadurl=yes|archiveurl=https://web.archive.org/web/20090221010759/http://hk.news.yahoo.com/article/090216/4/apx2.html|archivedate=2009-02-21}}</ref>，但他兩天後在[[東京|東京]]辯稱他只因大量服食[[感冒|感冒]]藥所致。首相仍然挽留，但因民主黨之強大壓力下，中川於在同月17日晚上向首相麻生太郎請辭，首相並已接受之，並在日本時間19:15在財務省向外宣佈。

其後，媒體再爆出當天記者會之後，中川昭一與當地日本大使館職員和財務省長官到梵蒂岡的博物館參觀時，漠視守則，跨越欄杆進入遊客禁止地，更伸手觸摸館內石像，使警鐘鳴響，造成巨大騷擾。

及後在同年8月30日晚眾議院選舉，中川昭一作為自民黨候選人在自家「地盤」北海道11區敗給了民主黨候選人。

== 暴斃 ==
日本[[共同社|共同社]]引述東京警方表示，中川昭一在2009年10月4日早上8點多被妻子發現倒斃在[[東京|東京]][[世田谷區|世田谷區]]住所的床上。報道指中川身上無表面傷痕，房間內沒有混亂跡象，也沒有發現遺書等；但屍體解剖調查的結果，認為其循環器官可能有慢性疾病，其葬禮及告別式於9日上午11時在東京都麻布山善福寺舉行。<ref>{{cite news|language=zh-hant|url=http://china.kyodo.co.jp/modules/fsStory/index.php?sel_lang=schinese&storyid=74420|title=日本「醉酒門」前財務相中川昭一突然死亡|publisher=[[共同社|共同社]]|date=2009-10-04 15:00(UTC+8)|accessdate=2009-10-04|deadurl=yes|archiveurl=https://web.archive.org/web/20100418203619/http://china.kyodo.co.jp/modules/fsStory/index.php?sel_lang=schinese|archivedate=2010-04-18}}</ref>

== 妻子「代夫出征」 ==
中川病逝後，自民黨北海道11區的支部長期士氣低迷，找不到合適的候選人。2011年，中川的妻子[[中川郁子|中川郁子]]終於決定要奪回丈夫失去的東西，參選[[第46屆日本眾議院議員總選舉|第46屆大選]]，2012年舉行的大選，郁子成功擊敗三年多前曾戰勝昭一的[[石川知裕|石川知裕]]，首次當選議員，為昭一洗雪了恥辱。
