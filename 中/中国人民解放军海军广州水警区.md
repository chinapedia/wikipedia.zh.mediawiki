'''中国人民解放军海军广州水警区'''，机关驻地[[广东省|广东省]][[广州市|广州市]]，是[[中国人民解放军南部战区海军|中国人民解放军南部战区海军]]的[[水警区|水警区]]。<ref name=syw>{{Cite web |url =  http://sy.mca.gov.cn/article/sxxx/201607/20160700884283.shtml|title =海军广州水警区与驻地携手打造暖心惠兵工程  |author =  |date = 2016-07-11 |publisher = 中国双拥网 |language =  |accessdate =  |quote =  }}</ref>

== 沿革 ==
2014年1月21日，海军快艇第二十六支队改编海军广州水警区命名大会召开，海军快艇第二十六支队正式改编为'''中国人民解放军海军广州水警区'''。这次部队改编不是简单的名称改变，而是部队军事改革中的大事，广州水警区首任司令员为石志坤大校。<ref name=hzq>{{Cite web |url =http://hzmz.haizhu.gov.cn/XWDT_545/MZYW_557/201401/t20140130_246204.htm  |title = 马红安同志出席海军广州水警区命名大会 |author =  |date = 2014-01-23 |publisher =  海珠区民政局|language =  |accessdate =  |quote =  }}</ref>

海军广州水警区所属部队需要常年外出执行任务，有的官兵常驻国家一类海岛，条件艰苦。2016年，海军广州水警区党委投入近50万元人民币，与驻地相关部门携手开展“三建三联”活动。“三建三联”即建设法律援助站,建立家属服务热线，建设军港WIFI热点，联合双拥办组织联谊活动，联系高校开展心理服务，联络公司开展幸福家庭授课。<ref name=syw/>

在[[深化国防和军队改革|深化国防和军队改革]]中，2016年春节前编制体制调整后，海军广州水警区海上防卫辖区扩大。<ref>{{Cite web |url = http://www.81.cn/jfjbmap/content/2016-02/22/content_2045.htm |title = 投身改革，正确对待走留 |author =  |date =2016-02-22  |publisher = 中国军网 |language =  |accessdate =  |quote =  }}</ref>

==附：原海军广州基地==
1969年12月组建海军广州基地（38001部队），隶属南海舰队建制领导。辖38012部队（汕头水警区）、38092部队（海军快艇第二十六支队），各舰、艇、船大队，及医院、仓库、修理厂等。<ref name=gd>广东省志·军事志（1949·10·～），广东人民出版社，1999年</ref>2003年12月编制体制调整，海军广州基地更名为'''中国人民解放军海军广州保障基地'''。2012年撤销编制。

==附：原海军湛江基地==
1952年12月8日，海军海南岛基地和陆军第129师师部合并，组成海军西营基地，归属中南军区海军建制。1957年3月5日，改称海军湛江基地。<ref name=da/><ref name=gd/>1964年5月，和海军川岛水警区合并，海军湛江基地番号撤销。<ref name=gd/>1978年3月9日晚8时许，驻湛海军码头发生[[051型导弹驱逐舰|重大军舰爆炸事件]]，驱逐舰第二支队的海军160舰爆炸沉没。<ref name=da>{{Cite web|url =http://zjxs.zhanjiang.gov.cn/MLXS/Content.aspx?id=15|title =大事记|author =|date =2011-10-10|publisher =霞山人民政府|language =|accessdate =|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20170113141234/http://zjxs.zhanjiang.gov.cn/MLXS/Content.aspx?id=15|archivedate =2017-01-13}}</ref>1985年10月，以南海舰队后勤部机关为基础，重新组建海军湛江基地（38009部队），归属南海舰队建制，辖各舰、船大队，第1、2、3工程处，及医院、修理厂、仓库等。<ref name=gd/>2003年12月编制体制调整，海军湛江基地更名为'''中国人民解放军海军湛江保障基地'''。2012年撤销编制。
