'''中国人民解放军空军汉口场站'''，位于[[湖北省|湖北省]][[武汉市|武汉市]][[新洲区|新洲区]][[阳逻街道|阳逻街道]]毛集村，隶属中国人民解放军空军航空兵第十三师。

==沿革==
1950年，[[中国人民解放军空军|中国人民解放军空军]]接管[[武汉王家墩机场|王家墩机场]]<ref name=whdq/>。此后在王家墩机场设'''中国人民解放军空军王家墩场站'''。1960年代初，升格组建'''中国人民解放军空军王家墩基地'''。1970年代，空军王家墩基地撤销，恢复为'''中国人民解放军空军王家墩场站'''<ref>{{Cite web |url =http://www.hbjc.gov.cn/jcyry/jcyld/201503/t20150319_629828.shtml  |title = 何畏 |author =  |date = 2015-03-19 |publisher =  湖北省人民检察院|language =  |accessdate =  |quote =  }}</ref>。1993年，改称'''中国人民解放军空军汉口场站'''<ref name=whdq/>。王家墩机场位于东经114度14分30秒，北纬30度35分30秒，距[[武汉长江大桥|武汉长江大桥]]6.4公里，可用起飞跑道距离2400米，机场总面积502.8万平方米。1987年前，王家墩机场主要供空军歼击机使用，1987年后改为主要供空军运输机使用，1991年对跑道局部翻修加固，以便适应部队运输机技术要求。1995年后，因武汉市城市发展，机场所在地已成为武汉市中心城区，国家及中央军委将机场搬迁列入计划。1976年以及1983年10月，[[武汉南湖机场|武汉南湖机场]]两次翻修跑道时，民航湖北省管理局均曾临时使用王家墩机场。1979年6月到1984年5月，经停武汉的民航大型飞机均在王家墩机场起降，民航湖北省管理局设有航站。1985年12月9日到2000年12月31日，武汉航空公司以汉口机场为营运基地<ref name=whdq>{{Cite web |url = http://www.whfz.gov.cn:8080/pub/dqwx/delsz/csjsz/DQP/DSZ3/200801/t20080101_19466.shtml |title =  武汉市志(1980-2000) 第一卷：总类 城市建设志 第七篇 交通设施 第四章 民航设施 第二节　机场 |author =  |date =  |publisher = 武汉市地情文献 |language =  |accessdate = 2017-07-13 |quote =  }}</ref>。

1997年，开始筹备建设空军阳逻新机场<ref name=touru/>。武汉市决定通过市场经济手段，筹资为空军还建空军阳逻新机场，再以空军移交的占地面积4000亩的王家墩机场为中心，重新规划周边11.36平方公里范围内的建设，建成[[武汉中央商务区|武汉中央商务区]]。2001年7月，国务院和中央军委批复同意王家墩机场迁建<ref>{{Cite web |url =  http://news.carnoc.com/list/10/10558.html|title = 武汉王家墩机场迁建顺利　新机场年内动工 |author =  |date = 2001-08-28 |publisher = 民航资源网 |language =  |accessdate =  |quote =  }}</ref>。汉口王家墩机场迁建后，武汉市100余平方公里的城区实现无净空限制。2007年6月6日，空军阳逻新机场竣工典礼举行，空军副司令员[[杨东明|杨东明]]等领导剪彩。空军汉口场站迁至空军阳逻新机场。空军阳逻新机场位于[[武汉市|武汉市]][[新洲区|新洲区]][[阳逻街道|阳逻街道]]毛集村<ref name=touru>{{Cite web |url = http://www.yangluo.cn/article-12144.html |title =  空军阳逻新机场正式投入使用|author =  |date =  2007-06-07 |publisher = 阳逻在线 |language =  |accessdate =  |quote =  }}</ref>。

空军汉口场站原隶属[[中国人民解放军广州军区空军|中国人民解放军广州军区空军]]空军航空兵第十三师，为该师第三十八团驻地<ref name=xzhqu>{{Cite web |url = http://www.yangluo.cn/article-20146.html |title = 新洲区军民联欢会在空军汉口机场军用礼堂举行 |author =  |date = 2013-08-02  |publisher = 阳逻在线 |language =  |accessdate =  |quote =  }}</ref>。2016年，在[[深化国防和军队改革|深化国防和军队改革]]中，仍隶[[中国人民解放军中部战区空军|中国人民解放军中部战区空军]]空军航空兵第十三师<ref name=hannan>{{Cite web |url =  http://news.carnoc.com/list/410/410555.html|title =汉南通用机场飞行程序竣工设计通过军民航审查  |author =  |date = 2017-07-10 |publisher = 民航资源网 |language =  |accessdate =  |quote =  }}</ref>。
