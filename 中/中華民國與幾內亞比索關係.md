'''中華民國與幾內亞比索關係'''是指[[中華民國|中華民國]]與[[幾內亞比索共和國|幾內亞比索共和國]]之間的關係。兩國於1990－1998年有官方[[外交#外交上的承認|外交關係]]，斷交後，目前沒有在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。對幾內亞比索的相關事務由'''{{link-en|駐葡萄牙台北經濟文化中心|Taipei Economic and Cultural Centre, Lisbon, Portugal}}'''{{Link-en|多重委派|Dual accreditation|兼轄}}。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-250-5289e-03-1.html |title = 駐館位置及聯絡方式 |publisher = [[中華民國外交部|中華民國外交部]] [[外交部領事事務局|領事事務局]]}}</ref>

==政治==
===外交===
[[File:Voting_res_2758.png|thumb]]，[[幾內亞比索|幾內亞比索]]當時屬於[[葡屬幾內亞|葡萄牙殖民地]]，因此不能對[[中國與聯合國#代表權問題|中華民國續留聯合國]]一案參與投票，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》。]]
{{see also|中華民國外交|{{link-en|幾內亞比索外交|Foreign relations of Guinea-Bissau}}}}
{{see also|中華民國駐幾內亞比索大使列表{{!}}中華民國駐幾內亞比索外交官列表|幾內亞比索駐中華民國大使列表{{!}}幾內亞比索駐中華民國外交官列表}}
1990年5月26日，兩國建立大使級[[外交#外交上的承認|外交關係]]。於首都[[比索_(城市)|比索]]設立'''中華民國駐幾內亞比索共和國[[外交代表機構#大使馆|大使館]]'''；於首都[[台北市|台北]]設立'''幾內亞比索共和國駐中華民國大使館'''，並派駐[[特命全權大使|大使]]。<ref name="87年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac1998/section_3/page3-1-3.htm |title = 《中華民國87年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref>

1990年10月8日，幾內亞比索國務委員會主席{{noteTag|國家元首}}[[若奥·贝尔纳多·维埃拉|維拉]]應中華民國總統[[李登輝|李登輝]]邀請，率領外交部長薩曼多（Julio Semedo）等官員抵台訪問，並參加[[中華民國國慶|中華民國國慶]]活動。<ref>{{Cite web |url = https://www.mofa.gov.tw/News_Content_M_2.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=9721261D80606AB7 |author = 公眾外交協調會 |date = 1990年10月4日 |title = 幾內亞比索共和國（REPUBLIC OF GUINEA-BISSAU）|publisher = 中華民國外交部}}</ref>1992年10月，外交部長薩曼多、計畫暨國際合作部長卡多索（Bernardino Cardoso）抵台參加中華民國國慶活動。<ref>{{Cite web |url = https://www.mofa.gov.tw/News_Content_M_2.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=1EA956DAA598B877 |author = 公眾外交協調會 |date = 1992年10月10日 |title = 外交部於十月十日下午五時至七時在臺北賓館舉行盛大國慶酒會 |publisher = 中華民國外交部}}</ref>

1994年9月，中華民國外交部長[[錢復|錢復]]以總統特使身分，率團前往幾內亞比索，參加總統維拉的連任{{link-en|就職典禮|Inauguration}}。<ref>{{Cite web |url = https://www.mofa.gov.tw/News_Content_M_2.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=29B52240CEE1766E |author = 公眾外交協調會 |date = 1994年9月23日 |title = 外交部長錢復將以中華民國總統特使身分 |publisher = 中華民國外交部}}</ref>

1996年，幾內亞比索外交暨僑務部長施樂華（Fernando Delfim da Silva）、退除役官兵事務部長馬內（A. MANE）、常駐聯合國副代表達荷薩（Mário Lopes da Rosa）抵台訪問。<ref name="86年外交年鑑"/>

1997年，中華民國外交部長[[章孝嚴|章孝嚴]]、台北市副市長[[陳師孟|陳師孟]]、行政院農業委員會副主任委員林享能訪問幾內亞比索；幾內亞比索國會議長[[马兰·巴卡伊·萨尼亚|薩尼亞]]、漁業部長施爾華（Arthur Da Silva）、比索市長梅迪納（Paulo Carlos Medina）抵台訪問。<ref name="86年外交年鑑"/>

1998年，中華民國外交部長[[胡志強|胡志強]]訪問幾內亞比索；幾內亞比索總統夫人[[若奥·贝尔纳多·维埃拉|維拉]]、總理[[卡洛斯·科雷亚|柯雷亞]]、農業部長施農華（Avito Jose da Silva）抵台訪問。<ref name="87年外交年鑑"/>

1998年4月23日，幾內亞比索與[[中華人民共和國|中華人民共和國]]簽署《復交公報》。4月24日，中華民國中止與幾內亞比索的外交關係。<ref name="87年外交年鑑"/>

===簽證===
[[File:Visa_requirements_for_Taiwanese_citizens_(6).png|thumb]]的[[中華民國國民|中華民國公民]]必須申請[[簽證|簽證]]方可入境幾內亞比索。]]
[[File:Visa_requirements_for_Guinea-Bissauan_citizens.png|thumb]]。]]
{{see also|中華民國簽證政策|中華民國公民簽證要求}}
{{see also|{{link-en|幾內亞比索簽證政策|Visa policy of Guinea-Bissau}}|{{link-en|幾內亞比索公民簽證要求|Visa requirements for Guinea-Bissauan citizens}}}}
兩國公民皆須申請[[簽證|簽證]]方可入境對方國家。持[[中華民國護照|中華民國護照]]的[[中華民國國民|中華民國公民]]可前往幾內亞比索駐[[葡萄牙|葡萄牙]][[外交代表機構#大使馆|大使館]]申請簽證。也可至該國[[簽證#电子签证|電子簽證]]系統申請，或以[[簽證#落地签证|落地簽證]]的方式入境（須事前獲得移民局的核准函）。經幾內亞比索轉機時間超過1天，亦須至電子簽證系統申請[[過境簽證|過境簽證]]。入境須出示[[黃熱病|黃熱病]]的[[疫苗接種|疫苗接種]]證明（黃皮書）。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-250-5289e-02-1.html |title = 簽證及入境須知 |publisher = 外交部領事事務局}}</ref>

持{{link-en|幾內亞比索護照|Guinea-Bissau passport}}的{{link-en|幾內亞比索人口|Demographics of Guinea-Bissau|幾內亞比索公民}}抵台參加國際會議、運動賽事、商展活動，則可以[[中華民國簽證政策#电子签证|電子簽證入境中華民國]]，停留最多30天並不得延期。<ref>{{Cite web |url = https://www.boca.gov.tw/cp-228-4488-8208a-1.html |title = 誰具備申請電子簽證的資格? |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-215-643-5e740-1.html |title = 我的電子簽證效期多久？|publisher = 外交部領事事務局}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將幾內亞比索歸類為'''橙色警戒'''：避免非必要旅行（2015年1月8日）；<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-279-931c6-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>[[中華民國衛生福利部|衛生福利部]]的國際間旅遊疫情建議等級表將該國歸類為'''第二級警示'''：對當地採取加強防護（2018年10月30日）。<ref>{{Cite web |url = http://www.cdc.gov.tw/InternationalTravel/CountryInfo.aspx?treeid=aa2d4b06c27690e6&nowtreeid=fb6a405b1d67aa69&tid=DF7EE2CD2088C699 |title = 國際重要疫情資訊 |publisher = [[中華民國衛生福利部|中華民國衛生福利部]]}}</ref>

==經濟==
{{see also|臺灣經濟|{{link-en|幾內亞比索經濟|Economy of Guinea-Bissau}}}}
===貿易===
2017年，幾內亞比索是中華民國第130大貿易夥伴、第193大[[進口|進口]]夥伴、第110大[[出口贸易|出口]]夥伴。出口至幾內亞比索的金額為1,837萬5,124[[美元|美元]]；自幾內亞比索進口的金額為7萬5,473美元，年減88.998%。貿易呈現[[淨出口|出超]]（[[盈餘|盈餘]]）1,829萬9,651美元，年減2,767.716%。※2016年，幾內亞比索是中華民國第197大貿易夥伴、第160大進口夥伴、第229大出口夥伴（無出口額，與其他9個[[經濟體|經濟體]]並列最低）。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>

==合作==
===援助===
1990年6月，成立中華民國駐幾內亞比索醫療團，負責支援首都[[比索_(城市)|比索]]郊區的[[坎順古|卡松果]]醫院門診與行政管理工作。11月，成立[[農學|農業技術]]團，示範推廣[[稻米|稻米]]、[[雜糧|雜糧]]、[[蔬菜|蔬菜]]等作物，以及[[農業機械|農業機械]]修護及[[竹|竹]]工藝等。<ref name="86年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac1997/3-1-3.html |title = 《中華民國86年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref>

1995年4月，兩國規劃1995－2000年的[[農業|農]]、[[醫療|醫]]、[[漁業|漁]]、[[技職教育|技職訓練]]及[[新聞|新聞]][[文化|文化]]交流等方面合作。<ref name="86年外交年鑑"/>

此外，幾內亞比索並派員赴台參加[[中小企業|中小企業]]、[[賦稅|賦稅]]、[[產業|產業]]、[[人力資源|人力資源]]、[[衛生|衛生]]與[[醫療保健|醫療保健]]、[[台灣奇蹟|台灣經濟發展]]等多項經貿訓練班。<ref name="86年外交年鑑"/>

1998年4月24日，兩國斷交後，即停止一切合作關係，並撤離駐團人員。<ref name="87年外交年鑑"/>

===事件===
兩國在邦交期間，[[中國輸出入銀行|中國輸出入銀行]]（簡稱輸銀）提供[[貸款|貸款]]資助幾內亞比索。斷交後，輸銀於2016年6月向[[美國紐約南區聯邦地區法院|美國紐約南區聯邦地區法院]]提告{{noteTag|中國輸出入銀行在[[紐約|紐約]]設有辦公室，過去貸款給[[中華民國外交#邦交國|中華民國的邦交國]]時由輸銀出面與對方國家銀行簽約，合約可以適用{{link-en|紐約州法律|Law of New York}}，對中華民國比較有保障。}}，求償4,920萬美元，金額相當該國前1年度[[預算|預算]]將近1/5。幾內亞比索主張此案已過追訴期，但根據《[[路透社|路透社]]》所見文件，向[[纽约州法院系统|紐約州地方法院]]提出的所有求償案都可以追溯到1990年代初期，幾內亞比索官員受訪時表示，希望能在2017年底解決此案。<ref>{{Cite web |url = https://udn.com/news/story/5/2375838 |author = 田思怡 |date = 2017年3月31日 |title = 與台灣斷交的非洲國家小心了 銀行打官司討回64億貸款 |publisher = [[聯合新聞網|聯合新聞網]]}}</ref><ref>{{Cite web |url = http://www.appledaily.com.tw/appledaily/article/headline/20170401/37603375/ |author = 綜合報導 |date = 2017年4月1日 |title = 非洲2斷交國 判還我64億 |publisher = [[蘋果日報_(台灣)|蘋果日報]]}}</ref>
