'''中国－塞舌尔关系'''，是指歷史上的[[中國歷史|中國]]和[[塞舌尔歷史|塞舌尔]]（包括現在[[中華人民共和國|中華人民共和國]]與[[塞舌爾共和國|塞舌爾共和國]]）之間的雙邊關係。中華人民共和國在1976年6月30日與塞舌尔建交，兩國有經貿和文化上的交流。

== 歷史 ==
塞舌爾群岛首批[[塞舌爾華人|華人移民]]在1886年遷居至當地<ref name=Trade>{{cite book|author1=巫乐华|title=华侨史概要|publisher=中国华侨出版社|year=1994}}</ref>{{Rp|79}}，他們是23名原本居於[[毛里求斯|毛里求斯]]的华侨，此後仍有華人在塞舌爾和毛里求斯兩地之間來往<ref name=Gen>{{cite book|author1=周南京|title=华侨华人百科全书: 历史卷|publisher=中国华侨出版社|year=2002}}</ref>{{Rp|419}}；塞舌爾的華人在1893年開始从事[[香莢蘭|香草]]加工等[[种植业|种植业]]工作<ref name=Trade/>{{Rp|79}}，不少華人與當地婦女通婚，他們的[[混血儿|混血儿]]很难被人認出是华人後代，故被政府视为{{link-en|塞舌尔人口|Demographics of Seychelles|塞舌尔人}}的一員<ref name=Gen/>{{Rp|419}}。第一任[[塞舌爾總統|塞舌爾總統]][[詹姆斯·曼卡姆|詹姆斯·曼卡姆]]也是华人後裔，他祖籍[[广东|广东]][[順德|順德]]<ref>{{cite book|author1=曾祥委|author2=曾漢祥|title=南雄珠璣巷移民的歷史與文化|publisher=暨南大學出版社|year=1995|url=http://www.chinareviewnews.com/crn-webapp/cbspub/secDetail.jsp?bookid=17701&secid=17774|archiveurl=https://archive.is/20130425093124/http://www.chinareviewnews.com/crn-webapp/cbspub/secDetail.jsp?bookid=17701&secid=17774|archivedate=2013-04-25|deadurl=no|chapter=珠璣巷移民後裔的海外拓殖}}</ref>。

塞舌爾曾是[[英國|英國]]的[[殖民地|殖民地]]，在1976年6月29日宣佈獨立為國<ref>{{cite web |title=塞舌尔国家概况 |url=http://www.fmprc.gov.cn/mfa_chn/gjhdq_603914/gj_603916/fz_605026/1206_606138/ |website=中华人民共和国外交部 |accessdate=2015-07-24 |archiveurl=https://archive.is/20150724140348/http://www.fmprc.gov.cn/mfa_chn/gjhdq_603914/gj_603916/fz_605026/1206_606138/ |archivedate=2015-07-24 |deadurl=no }}</ref>。在此之前，中华人民共和国和塞舌爾已有來往：中华人民共和国支持争取塞舌爾獨立的[[人民党_(塞舌尔)|塞舌尔人民联合党]]，曾三度向該組織提供財政援助；1975年1月，時任塞舌爾自治政府总理的曼卡姆向[[中国驻坦桑尼亚大使列表|中国驻坦桑尼亚大使]]表示，塞舌尔將在独立後與中华人民共和国建立邦交<ref name=Common>{{cite book|author1=裴坚章|title=中华人民共和国外交史: 1970-1978|publisher=世界知识出版社|year=1994}}</ref>{{Rp|180}}。塞舌爾在獨立翌日與中华人民共和国建立[[大使|大使]]级外交关系<ref name=Common/>{{Rp|180}}。

根據{{link-en|經濟複雜性研究中心|The Observatory of Economic Complexity}}網站上的數據，中国大陸在2012年出口了3000萬[[美元|美元]]到塞舌尔，貨物涵蓋[[機械|機械]]設備、[[金屬|金屬]]製品、[[紡織品|紡織品]]等領域<ref name=tradea/>。塞舌尔到中国大陸的出口貿易在2007年曾一度超過400萬，但出口額在2009年大幅下降<ref name=tradeb/>至28,300美元<ref>{{cite web|author=Alexander Simoes|title=产品塞舌尔出口到中国 (2009)|url=https://atlas.media.mit.edu/zh/explore/tree_map/hs/export/syc/chn/show/2009/|publisher=Observatory of Economic Complexity|accessdate=2015-07-25}}</ref>，其後有少量回升；2012年，塞舌尔出口了23.8萬美元來中国大陸，93.95%的貨物為廢[[塑料|塑料]]，其餘貨物為畜產<ref name=tradeb/>。塞舌尔同年出口了282萬美元到[[香港|香港]]，大部分貨物是畜產<ref>{{cite web|author=Alexander Simoes|title=产品塞舌尔出口到香港 (2012)|url=https://atlas.media.mit.edu/zh/explore/tree_map/hs/export/syc/hkg/show/2012/|publisher=Observatory of Economic Complexity|accessdate=2015-07-25}}</ref>。

== 文化關係 ==
塞舌尔曾舉辦「中國日」活動，期間有摄影展覽、[[農曆新年|農曆新年]][[庙会|庙会]]、[[中国大陆电影|中国电影]]放映等文化節目；在「中國日」期間，塞舌爾首都[[維多利亞_(塞舌爾)|维多利亚]]將會有街道掛上[[花燈|花燈]]和[[中華人民共和國國旗|中華人民共和國國旗]]，被[[塞舌爾旅遊和文化部|塞舌爾旅遊和文化部]]部長{{link-fr|阿兰·圣安戈|Alain Saint-Ange}}形容為「迷你[[唐人街|唐人街]]」<ref>{{cite web |title=塞舌尔首届“中国日”进入倒计时 |url=http://sc.china-embassy.org/chn/whjl/t1121549.htm |website=中华人民共和国驻塞舌尔共和国大使馆 |accessdate=2015-07-25 |archiveurl=https://archive.is/20150725063703/http://sc.china-embassy.org/chn/whjl/t1121549.htm |archivedate=2015-07-25 |deadurl=no |date=2014-01-22 }}</ref>。中華人民共和國駐塞舌尔大使館曾與當地一所[[酒店|酒店]]合辦「中国周」，每天的活動均有一個主題，節目包括图片展覽、学习烹飪[[中國菜|中國菜]]、中国电影放映等；中華人民共和國駐塞舌尔大使館向該酒店贈送有關中國的[[藝術品|藝術品]]、[[图片|图片]]、[[書籍|書籍]]和[[DVD|光碟]]，而當地一所[[旅行社|旅行社]]和[[航空公司|航空公司]]亦參與協辦「中国周」活動<ref>{{cite web |title=塞舌尔“中国周”活动拉开帷幕 |url=http://sc.china-embassy.org/chn/whjl/t695418.htm |website=中华人民共和国驻塞舌尔共和国大使馆 |accessdate=2015-07-25 |archiveurl=https://archive.is/20150725070916/http://sc.china-embassy.org/chn/whjl/t695418.htm |archivedate=2015-07-25 |deadurl=no }}</ref>。

中華人民共和國駐塞舌尔大使館曾應當地文化部門的邀请，組織两名中国[[教练|教练]]到塞舌尔教授[[太极拳|太极拳]]，课程以[[長者|長者]]、[[学生|学生]]和其他社区人士為目標<ref>{{cite web |title=塞舌尔民众踊跃学习中国太极 |url=http://sc.china-embassy.org/chn/whjl/t1057037.htm |website=中华人民共和国驻塞舌尔共和国大使馆 |accessdate=2015-07-25 |archiveurl=https://archive.is/20150725072620/http://sc.china-embassy.org/chn/whjl/t1057037.htm |archivedate=2015-07-25 |deadurl=no |date=2013-07-08 }}</ref>。塞舌尔文化部門也曾邀请被譽為「中国国家级[[剪纸|剪纸]]传承人」的段建珺到當地教授剪纸，他在第三期課程中教授了約50名學員，又把一件作品送給旅遊和文化部部長圣安戈；圣安戈決定向塞舌尔市民收集剪纸作品，並舉行评奖活動<ref>{{cite web |title=第三期剪纸培训班在塞开展 |url=http://sc.china-embassy.org/chn/whjl/t1064942.htm |website=中华人民共和国驻塞舌尔共和国大使馆 |accessdate=2015-07-25 |archiveurl=https://archive.is/20150725074626/http://sc.china-embassy.org/chn/whjl/t1064942.htm |archivedate=2015-07-25 |deadurl=no |date=2013-08-07 }}</ref>。

{{link-en|塞舌尔大学|University of Seychelles}}的[[孔子课堂|孔子课堂]]由該校和[[大连大学|大连大学]]在2014年联合开办，将會在當地發展[[汉语|汉语]]教学、推广[[中華文化|中華文化]]<ref>{{cite web |title=塞舌尔首座孔子课堂揭牌成立 |url=http://sc.china-embassy.org/chn/zxxx/t1166301.htm |website=中华人民共和国驻塞舌尔共和国大使馆 |accessdate=2015-07-25 |archiveurl=https://archive.is/20150725081718/http://sc.china-embassy.org/chn/zxxx/t1166301.htm |archivedate=2015-07-25 |deadurl=no |date=2014-06-17 }}</ref>。

== 援助 ==
中華人民共和國在2007年開始派出經[[中國共產主義青年團|中國共產主義青年團]]選拔的青年[[義工|義工]]到塞舌尔，義工將會在當地的服务机构進行[[医疗卫生|医疗卫生]]、[[資訊科技|資訊科技]]、[[建筑工程|建筑工程]]、[[音乐教育|音乐教育]]等範疇的社會服务<ref>{{cite web |title=中国青年志愿者第六批援塞舌尔服务队赴塞 |url=http://www.ccyl.org.cn/bulletin/zgb_qkjl/201411/t20141126_721500.htm |website=中国共青团网 |accessdate=2015-07-25 |archiveurl=https://archive.is/20150725091946/http://www.ccyl.org.cn/bulletin/zgb_qkjl/201411/t20141126_721500.htm |archivedate=2015-07-25 |deadurl=no |date=2014-11-26 }}</ref>。

在医疗援助方面，中華人民共和國曾派遣醫疗队和醫疗義工到塞舌爾，[[中国人民解放军海军|中国人民解放军海军]]的医疗船也曾訪問當地，為當地市民提供义务诊症；中國亦曾向塞舌爾贈送[[藥物|藥物]]和医疗器材，並在當地援建[[医院|医院]]<ref>{{cite web |title=史忠俊大使参加援塞医疗队议定书签字仪式 |url=http://sc.china-embassy.org/chn/dsxx/t919984.htm |website=中华人民共和国驻塞舌尔共和国大使馆 |accessdate=2015-07-25 |archiveurl=https://archive.is/20150725093531/http://sc.china-embassy.org/chn/dsxx/t919984.htm |archivedate=2015-07-25 |deadurl=no |date=2012-04-03 }}</ref>。

2015年的塞舌尔獨立日是該國首個官方確立的[[国庆日|国庆日]]，當地將會有庆祝活動，中華人民共和國捐贈了一批相关物资<ref>{{cite web |title=殷立贤大使出席援塞庆典活动物资捐赠仪式 |url=http://sc.china-embassy.org/chn/zxxx/t1273566.htm |website=中华人民共和国驻塞舌尔共和国大使馆 |accessdate=2015-07-25 |archiveurl=https://archive.is/20150725094427/http://sc.china-embassy.org/chn/zxxx/t1273566.htm |archivedate=2015-07-25 |deadurl=no |date=2015-06-16 }}</ref>。
