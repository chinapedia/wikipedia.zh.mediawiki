'''中国人民解放军海军潜艇学院'''，简称'''海军潜艇学院'''、'''青岛海军潜艇学院'''，本部位于[[山东省|山东省]][[青岛市|青岛市]]，隶属[[中国人民解放军海军|中国人民解放军海军]]，为军事高等教育院校。

== 沿革 ==
;中国人民解放军海军潜水艇学校
* 1953年3月20日，中央军委命令以原[[川北军区|川北军区]]领导机关、旅顺潜艇学习队为基础，组建'''中国人民解放军第四海军学校'''。校长傅继泽，政治委员王昕。驻青岛市[[毛奇兵营|登州路77号]]<ref name=sdshqw>{{Cite web |url = http://lib.sdsqw.cn/bin/mse.exe?seachword=%u6F5C%u8247%u5B66%u9662&K=b2&A=12&rec=19&list=&page=&run=13 |title = 青岛市志 军事志 第一篇 驻军与军事机构  第四章 人民军队 第六节 军事院校 |author =  |date =  |publisher = 山东省情网 |language =  |accessdate =2017-06-10  |quote =  }}</ref>。1953年8月13日正式组建。1954年9月20日迁至[[浮山所|浮山所]]附近的湛流干路23号（今山东路10号）<ref>{{cite web|url=http://qingdao.iqilu.com/qdminsheng/2017/0722/3630678.shtml|title=上世纪50年代，潜水艇学校是这样在青岛从无到有的|date=2017-07-22|work=|publisher=青岛日报 齐鲁网}}</ref>。1956年开办本科教育<ref name=xunli/>。
* 1957年9月23日，经国防部批复，第四海军学校改称'''中国人民解放军海军潜水艇学校'''。执行师级权限。
* 1969年2月29日，根据中央军委关于军队院校调整方案，撤销海军潜水艇学校<ref name=sdshqw/>。

;中国人民解放军潜水艇士兵学校
*1960年7月13日，[[中国人民解放军总参谋部|中国人民解放军总参谋部]]命令：“以潜水艇学校水兵训练团为基础组建潜水艇士兵学校”。1960年8月20日，'''中国人民解放军海军潜水艇士兵学校'''成立。校长李怀章，政治委员王统军<ref name=sdshqw/>。驻[[毛奇兵营|登州路77号]]。1964年迁驻大沙支路4号。
*1969年2月29日，根据中央军委关于军队院校调整方案，撤销海军潜水艇士兵学校<ref name=sdshqw/>。

;中国人民解放军海军潜艇学院
*1972年6月28日，海军党委决定以原海军潜水艇学校、原海军潜水艇士兵学校恢复重建'''中国人民解放军海军潜艇学校'''（师级）<ref name=sdshqw/>。
*1974年1月3日，根据中央军委命令，潜艇学校升为军级，隶属军委海军。校长张震春（代），政治委员陈辉<ref name=sdshqw/>。*1983年8月24日，奉中央军委命令，改为'''中国人民解放军海军潜艇学院'''。院长苏军，政治委员陈辉。驻山东路10号<ref name=sdshqw/><ref>{{Cite web |url = http://world.huanqiu.com/exclusive/2014-04/4977108.html |title =近百印尼官兵走进中国潜艇学院 感叹中国海军透明  |author =  |date = 2014-04-22 |publisher = 环球网 |language =  |accessdate =  |quote =  }}</ref><ref name=mod>{{Cite web |url =http://www.mod.gov.cn/reports/201403/wzry/2014-09/12/content_4536753.htm  |title =  海军潜艇学院|author =  |date =2014-09-12  |publisher = 中华人民共和国国防部 |language =  |accessdate =  |quote =  }}</ref>。副军级。
*2009年，拍卖了位于青岛市市南区山东路10号的老校区，学校本部迁至青岛市[[崂山|崂山]]风景区附近的李沧区金水路<ref>{{Cite web |url =http://news.ifeng.com/mil/2/200906/0610_340_1196766.shtml  |title =  青岛海军潜艇学院土地要拍卖 41.7亿元起价|author =  |date =2009-06-10  |publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref>。
*2017年，在[[深化国防和军队改革|深化国防和军队改革]]中，海军潜艇学院的性质调整为军事高等教育院校<ref>{{Cite web |url = http://news.ifeng.com/a/20170530/51183107_0.shtml |title =  2017年军队招生单位名录发布 军改后院校新名称曝光|author =  |date = 2017-05-30 |publisher =凤凰网  |language =  |accessdate =  |quote =  }}</ref><ref name=xunli>{{Cite web |url = http://www.81.cn/jwgz/2017-06/08/content_7632282.htm |title =军校巡礼第十站：海军潜艇学院（附报考指南）  |author =  |date = 2017-06-08 |publisher = 中国军网 |language =  |accessdate =  |quote =  }}</ref>。

== 专业设置 ==
1987年开始招收硕士研究生，1990年获硕士学位授予权。2002年被批准为全军第一批军事硕士专业学位教育试点院校。2003年获博士学位授予权。2009年建成“战术学”、“船舶与海洋工程”博士后科研流动站<ref>{{Cite web |url = http://yz.chsi.com.cn/sch/schoolInfo--schId-368272,categoryId-475627,mindex-1.dhtml |title =海军潜艇学院 学校简介 |author =  |date =  |publisher = 中国研究生招生信息网 |language =  |accessdate =2017-06-10  |quote =  }}</ref><ref name=xunli/>。截至2017年，学院有19个技术兵培训专业，17个士官职业技术教育大专专业，3个现职军官培训专业，7个本科教育专业，12个硕士学位授权点，2个博士学位授权点，2个博士后科研流动站。7个本科教育专业是<ref name=xunli/>：
*电子信息工程
*水声工程
*航海技术
*救助与打捞工程
*船舶与海洋工程
*武器系统与工程
*武器发射工程<ref name=xunli/>
