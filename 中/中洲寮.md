'''中洲寮'''（亦作「中州寮」{{notetag|如安南區公所、Google map。又過去寫作「中洲藔」。}}）為[[臺南市|臺南市]][[安南區|安南區]]的聚落，是「十六寮」之一，屬於廣義的[[和順_(安南區)|和順]]地區{{NoteTag|廣義的和順指的是日治時期的「[[安順庄|安順庄]]和順寮」，泛指[[臺19線|臺19線]]安和路段與長和路一帶的聚落{{r|臺19}}，狹義則指舊和順。}}{{r|府地}}{{r|臺19}}。該聚落現在劃分成[[州北里|州北里]]與[[州南里|州南里]]，北面為與[[安定區_(臺南市)|安定區]]之區界、東北有塭南里、東邊有東和里、南面有新順里與安順里、西邊有布袋-{里}-<ref>[http://www.annan.gov.tw/main.php?menu=neigh_2&notice_id=6 臺南市安南區公所 各里簡介 州北里]</ref><ref>[http://www.annan.gov.tw/main.php?menu=neigh_2&notice_id=5 臺南市安南區公所 各里簡介 州南里]</ref>{{r|市}}。

該聚落名稱的由來是因為居民祖先大多來自[[學甲區|學甲]]中洲，雖然聚落位於安南區的東北角，但因為臺19線的經過，所以有良好的發展{{r|府地}}{{r|臺19}}{{r|市}}。

== 沿革 ==
中洲藔在清末[[臺江內海|臺江內海]]淤積成陸之後，漸有移民來此開墾{{NoteTag|1=具體開墾時間有咸豐年間（1860年左右）{{r|府地}}與光緒年間（1875年左右）{{r|市}}等說法。}}。早期居民多來自學甲中洲，以姓林、姓邱為大宗{{NoteTag|1=一說邱姓來自於[[篤加|篤加]]{{r|市}}。}}{{r|府地}}{{r|臺19}}{{r|市}}。清末被劃分在[[臺灣縣|臺灣縣]][[武定里|武定里]]，後來改制為[[安平縣|安平縣]][[外武定里|外武定里]]{{r|臺19}}。清朝時由於墾地大戶曾對中洲藔與其他村莊嚴苛抽糧，於是居民團結起來，推舉「林有」為代表向官府爭地，最後成功{{r|市}}。而此事傳說有學甲慈濟宮保生大帝顯靈幫助，是以居民感念而從慈濟宮分香供奉，後來建立庄廟保安宮供奉保生大帝{{r|臺19}}{{r|市}}。

日治初期，中洲藔屬於外武定里和順藔庄，後來改為外武定區和順藔庄{{r|臺19}}。大正九年（1920年）改制後，中洲寮成為「臺南州新豐郡安順庄和順寮」底下的一個[[小字|小字]]{{r|市}}。而日治時期因附近建有三崁店製糖所（[[永康糖廠|永康糖廠]]），居民多種甘蔗，附近也設有糖鐵運送原料至糖廠{{r|臺19}}。

二次大戰後，中洲寮因為聚落過大，所以被劃分成兩個里{{r|市}}。

== 設施 ==
中洲寮境內設有[[臺南市立和順國民中學|和順國中]]、[[臺南市安南區和順國民小學|和順國小]]、國立啟智學校等學校，此外設有安南醫院與仁愛之家{{r|臺19}}。臺南市肉品市場也設在中洲寮，過去該市場{{NoteTag|1=即日治時期的臺南州臺南市家畜市場{{r|臺19}}。}}位於中西區的西門路一帶，1973年向台糖買地後遷到現址{{r|臺19}}。

聚落信仰中心是保安宮<ref>[http://www.taconet.com.tw/sedan 臺南市安南區中州寮保安宮網站 本宮沿革]</ref>，供奉分香自[[學甲慈濟宮|學甲慈濟宮]]的保生大帝{{r|市}}{{r|臺19}}。此外當地尚有佛教的福國寺與基督教長老會和順教會{{r|臺19}}。

中洲寮夜市會在每週二、六營業，所在的空地過去是[[永康糖廠|永康糖廠]]的集蔗場{{r|臺19}}。而過去中洲寮裡曾有「集成戲院」，由王姓家族出地，集合地方仕紳資金而成，故名「集成」，但隨著時代演變而沒落結束營業{{NoteTag|該戲院除電影外，也有上演布袋戲與歌仔戲{{r|臺19}}。位置大約在四信安南分社附近{{r|臺19}}。}}{{r|臺19}}。

== 其他 ==
當地俗諺「外塭仔出網，六塊寮撿孔，五塊寮出泥鰍籠，中洲（州）寮出人通」，其中的「中洲（州）寮出人通」的是因為中洲寮發展較為快速繁榮，相對地有許多遊手好閒之人{{r|府地}}{{r|臺19}}。
