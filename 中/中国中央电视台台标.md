'''中国中央电视台台标'''介绍[[中国中央电视台|中国中央电视台]]的台标在电视广播中的应用和变迁。

中国中央电视台1979年至2001年使用的旧台标呈[[蝴蝶|蝴蝶]]形，并包含“TV”两个字母，启用于1979年，由原央视职工张德生设计<ref name="rmwjtb">{{cite news |url = http://www.people.com.cn/GB/shehui/44/20011128/614132.html |title = CCTV台标设计者告央视侵权 法院驳回原告诉讼请求 |website = 人民网 |date = 2001-11-28 }}</ref>，通稱“蝴蝶标”。蝴蝶标形似[[人造卫星|人造卫星]]运转轨迹、[[原子核|原子核]]、[[电视发射塔|电视发射塔]]上的蝴蝶[[天线|天线]]，是“CCTV”四个字母的变形体，颜色采用[[彩色电视|彩色电视]]的[[三原色|三原色]]（即[[红|红]]、[[绿色|绿]]、[[蓝|蓝]]）。1998年，央视不再使用蝴蝶标；不过在之后相当长的一段时间内，在央视[[记者|记者]][[麦克风|麦克风]]、以及[[中國國際電視總公司|中國國際電視總公司]]（及其[[子公司|子公司]]）發行或代理進口之影片[[版權|版權]]宣告畫面（採用最早期金色蝴蝶標）上仍可以看到蝴蝶标。

目前中国中央电视台使用的双线CCTV台标，启用于1998年，由[[日本|日本]]设计师[[铃木智己|铃木智己]]设计<ref>{{Cite web|url=http://sun-moon.ne.jp/works/graphicdesign/366|title=日本Sun&Moon设计官网|accessdate=|author=|date=|publisher=}}</ref>。其中第二个“C”为红色，寓意“中国”、“中央”的含义。最初的边缘为方角，2001年7月9日将边缘美化成圆角并适当拉宽。实际使用时为灰色半透明台标。当前，中国中央电视台使用的台标仍以带频道名称的双线台标为主。

== 台标变迁 ==
=== 字幕台标时期 ===
1958年9月2日，中国中央电视台前身“北京电视台”没有频道标识，仅在节目片尾出现[[中共中央主席|中共中央主席]][[毛泽东|毛泽东]]手书的“'''北京电视台'''”字样<ref>{{cite web |url = http://www.tudou.com/programs/view/XsPJAkpCpcQ |title = 毛主席会见蓬皮杜总统 |website = 土豆 |authors = 北京电视台制作、[[法国电视一台|法国电视一台]]转录 |year = 1973 }}</ref>。1978年5月1日，更名后的“中央电视台”字样由中共中央主席、[[国务院总理|国务院总理]][[华国锋|华国锋]]题写，并且在电视片尾上出现。1981年，“中央电视台”字样改为[[宋体|宋体]]美术字。

1978年5月1日，更名后的央视采用“蝴蝶标”作为台标，但并未悬挂在屏幕中。1983年，北京地区的[[CCTV-1|CCTV-1]]开始使用白色“蝴蝶标”+“中央电视台”台标，置于屏幕左下角；1984年改为“蝴蝶标”+“中央电视台-1”，仅在整半点报时期间及节目开始时出现。通过微波及卫星向全国转播的CCTV-1无台标。

1985年，CCTV-1全国版与北京版统一使用文字式的“中央电视台”，出现在电视画面的左下角，仅在整半点报时及节目开始时出现；此阶段台标有黑色描边<ref>[http://www.tudou.com/programs/view/ZqetsgfrhSQ/ 齐天乐晚会]，土豆</ref>。[[CCTV-2|CCTV-2]]、[[CCTV-3|CCTV-3]]也改为类似的“中央电视台-X”（X表示频道序号，如“中央电视台-2”）台标。

1987年2月1日，由于CCTV-2上星播出，CCTV-1改用文字式的“中央电视台-1”，出现在电视画面的左下角，仅在整半点报时及节目开始时出现<ref>{{cite web |url = http://www.tudou.com/listplay/uMKHzq5M0FQ/HM1_wZAWlz0/dkbi |title=新闻联播 1991年6月1日 开始 主持人:李修平/罗京 |website = tudou.com }}</ref>。起初台标字体仍为黑色描边，在央视迁入[[中央电视台彩色电视中心|彩电中心]]后改为无描边白色字体。后又进行一定微调，字体横向收窄。

=== 蝴蝶台标时期 ===
[[File:CCTV_tower_from_entrance_-_panoramio.jpg|thumb]]正门的蝴蝶标。]]
1992年2月1日，央视一套蝴蝶标悬挂在电视屏幕左下角，不过仍然是只在整点和半点或广告和节目开始时出现<ref>{{cite web |url = http://www.tudou.com/programs/view/Oyw9jn48LhQ/ |title = 1991 cctv1 日立D8A  广告 |website = tudou.com }}</ref>；其他频道以“中央电视台－X”的字幕台标仍然使用。1992年7月的[[巴塞罗那奥运会|巴塞罗那奥运会]]，央视正式全天候悬挂蝴蝶标；到了1992年8月，台标位置改到屏幕左上角，后来变小<ref>{{cite web |url = http://www.tudou.com/programs/view/IT2j0Gqbm9k/ |title = [经典广告]松下J27录像机 1992 |website = tudou.com }}</ref>。

1995年11月30日，[[CCTV-3|戏曲·音乐频道]]（现为综艺频道）上星播放，与当年陆续开播的[[CCTV-5|体育频道]]、[[CCTV-6|电影频道]]、[[CCTV-8|文艺频道]]（现为电视剧频道）一同作为首批专业加密卫星频道。同年开播了中国现今唯一的卫星电视公共服务平台：[[CCTV-7|少儿·军事·农业·科技频道]]（现为军事·农业频道）。特别的是，该五条频道的报时器字体全部采用Helvetica，而且台标的质感也不一样。

1998年5月27日，[[CCTV大富|CCTV大富]]在[[日本|日本]]落地，台标“单线CCTV+蝴蝶大富”至2011年1月1日凌晨停用，宣传标志为“蝴蝶大富”和文字“大富电视台”字样。

=== 旧单线台标时期 ===
1998年6月1日早晨，央视开始使用透明“'''CCTV'''”字形，但仅限于电视频道画面上。蝴蝶标在央视记者话筒上及央视各种宣传品上使用，同时央视电视宣传片开始启用'''双线CCTV台标'''<ref>{{cite web |url = http://www.tudou.com/programs/view/XIeHAgkqsxM/ |title = 1999 01 19 cctv1 新闻联播前面5分钟 |website = tudou.com }}</ref>。[[2000年悉尼奥运会|2000年悉尼奥运会]]期间，[[CCTV-5|体育频道]]改为奥运频道，台标使用透明“'''CCTV+奥运五环'''”。

=== 旧双线台标时期 ===
[[File:CCTV-1.svg|thumb]]双线台标，这一版本台标不带频道名称。]]
2001年7月8日晚，因应当年7月9日早晨6点整[[中国中央电视台科学教育频道|科学·教育频道]]（现为[[中国中央电视台科教频道|科教频道]]）与[[中国中央电视台戏曲频道|戏曲频道]]的开播、以及[[2008年夏季奥林匹克运动会申办过程|北京申奥]]的开始，央视开始正式使用现在的'''双线CCTV台标'''。勺标刚启用时，和频道宣传片一致，勺内的数字是空心的<ref>{{YouTube|NVD_7NQvqaM|北京申奥成功}}</ref><ref>{{cite web |url = http://www.tudou.com/programs/view/TnxP4nUgWfA/ |title = 2001 cctv2 广告 郎永淳 胥午梅 |website = tudou.com }}</ref>，不过因为很难辨识出各频道序号，此版台标用了一个多月就停用，其中1套于2001年7月15日凌晨做出微调，其他频道于2001年8月17日上午做出微调。（注：序号的字体是[[KabelC_(字体)|KabelC]]）

2001年8月17日上午，勺标做出微调，将CCTV双线字体高亮化，勺标则透明化，中间的序号也改为了高亮，不过序号的字体仍然使用KabelC，至此空心勺标仅在宣传片中出现。[[CCTV-12|CCTV-12]]之后开播的频道，均未使用本版标识，而是用勺标放大版，中间直接冠以黑体字体的频道名称，如“'''CCTV-新闻'''”、“'''CCTV-少儿'''”、“'''CCTV-音乐'''”、“'''CCTV-E&F'''”,以及“'''CCTV-E'''”、“'''CCTV-F'''”，“'''CCTV-{{rtl-lang|ar|العربية}}'''”、“'''CCTV-{{lang|ru|Pусский}}'''”、“'''CCTV-高清'''”、“'''CCTV-NEWS'''”、“'''CCTV-娱乐'''”、“'''CCTV-戏曲'''”。

2004年，央视开播付费频道，付费频道采用“'''CCTV+圆弧+频道名称'''”的方式。

[[2004年雅典奥运会|2004年雅典奥运会]]期间，[[CCTV-5|体育频道]]改为奥运频道，在其下方加上了奥运五环标志，是央视各频道中第一个采用“'''英文+数字+图案'''”形式的频道。但是与后来的台标不同的是，2011年后勺标“CCTV-5”下方为综艺体的“体育”字样。2008年[[北京奥运会|北京奥运会]]期间，[[CCTV-5|体育频道]]改为奥运频道，使用现行的双线“CCTV+奥运五环”，之后每逢夏季奥运会的暑期亦是如此（从2018年逢冬季奥运会期间亦是如此，期前的冬奥会仅宣传片使用）。

從2008年[[北京奧運會|北京奧運會]]起，为避免和奥运会公共信号默认左上角记分牌遮挡，當标清频道轉播奧運會賽事時，勺標會移動到屏幕右上角。之后CCTV-5、5+转播的赛事亦是如此。实际上，得益于2008年北京奥运会赛事第二天观众的电话投诉得以解决<ref>[http://blog.sina.com.cn/s/blog_4f1a777b0100aidy.html 听取邯郸市民建议  央视台标挪位置]</ref>。

2009年7月25日，[[中央电视台阿拉伯语国际频道|中央电视台阿拉伯语国际频道]]正式开播。台标起初为CCTV-A标志，后改为CCTV加勺型加阿拉伯语的阿拉伯语单词“'''CCTV-{{rtl-lang|ar|العربية}}'''”，为该台最早以外语单词为LOGO文字的两个外语频道之一，另一个为[[中国中央电视台俄语国际频道|中国中央电视台俄语国际频道]]“'''CCTV-{{lang|ru|Pусский}}'''”。

2009年9月21日凌晨，[[CCTV-2|财经频道]]台标微调时在其下方加上了[[微软雅黑|微软雅黑]]字体的“'''财经频道'''”四个简体汉字，是央视各频道中第一个采用“'''英文+数字+汉字'''”形式的频道。但是与后来的台标不同的是，2011年后勺标“CCTV-2”下方为综艺体的“财经”字样。

2018年9月26日，[[中国中央电视台4K超高清试验频道|CCTV-超高清]]频道开播，首日使用了旧版“双线CCTV”+勺型标的台标。

=== 新双线台标时期 ===
[[File:CCTV-1综合.png|thumb]]双线台标，这一版本台标带有频道名称。]]
2011年1月1日凌晨，因应当天早晨8点整[[中国中央电视台纪录频道|纪录频道]]的开播，央视所有频道的台标做出了微调<ref>{{cite book |url = http://news.sina.com.cn/m/2011-01-03/100321751246.shtml |title = 中央电视台2011年全面更换台标 |website = 新浪 |author=京华时报 |year=2011 }}</ref>。[[CCTV-13|新闻频道]]、[[CCTV-14|少儿频道]]、[[CCTV-15|音乐频道]]、[[中国中央电视台高清综合频道|高清综合频道]]（官网）、[[CCTV-F|法语国际频道]]、[[CCTV-E|西班牙语国际频道]]分别改为“'''CCTV-13'''”、“'''CCTV-14'''”、“'''CCTV-15'''”、“'''CCTV-22'''”、“'''{{lang|fr|CCTV-Français}}'''”、“'''{{lang|es|CCTV-Español}}'''”。1套至15套在台标下方加频道名称，如“'''CCTV-1 综合'''”，其中的序号字体改为[[Futura-Bold|Futura-Bold]]，频道名称字体为汉仪综艺。娱乐频道、海外戏曲频道、[[中国中央电视台高清综合频道|高清综合频道]]（电视画面）的台标也进行了微调，将勺内频道名称字体改为汉仪综艺。[[中国中央电视台中视购物频道|中视购物频道]]（不再使用付费频道的圆弧标识）采用双线勺标，付费频道的台标也做出修改，将频道名称的字体统一改为汉仪综艺字体。日本的[[CCTV大富|CCTV大富]]频道台标也由“单线CCTV+蝴蝶大富”标志改为双线勺子台标，并在下面标注“{{lang|ja|中国语放送}}”（主要是遮盖央视台标，目前一些央视节目在日本播出是录制央视的画面，有时台标容易被认出如“CCTV-1综合”，但如央视春晚等节目是使用央视的公共信号，无央视台标）。

2011年11月23日，东京时间18点起，于东京时间18:00-24:00（北京时间每天17:00-23:00），CCTV大富频道尝试该频道直播节目的主声道和副声道分别使用日语同声传译和汉语原音广播播放，其他节目则以汉语原音广播并提供中文和日文字幕，在此期间，台标下方注记由“{{lang|ja|中国語放送}}”变为“{{lang|ja|二カ国語放送試播中}}（即中日双语试播）”，其他时段则变回“{{lang|ja|中国語放送}}”。

2012年，央视开办3D试验频道，使用独立的大字“3D”+小字“试验频道”标志，字体仍是汉仪综艺。

2012年1月23日起，CCTV大富频道正式全天双语化播出，台标下方注记由“{{lang|ja|中国語放送}}”变为“{{lang|ja|二カ国語放送}}（即中日双语播出）”。

2013年8月，原高清综合频道置换为体育赛事频道，台标的频道名称字体与1套至15套的台标一致，但序号“5+”为Arial Black字体。

2015年4月1日，央视台标进行微调，中文国际频道、体育频道和外语频道台标横向压缩，使其适应16:9格式。电影频道右下角更换全新金黄色台标（对外播出的版本仍用旧版标志）。2015年5月1日，财经频道台标横向压缩为16:9形式。2015年5月8日，电影频道台标横向压缩为16:9形式。2015年8月17日，除音乐频道外，全部国内公共频道、五大外语频道标清版全部横向压缩为16:9形式。2015年8月31日，音乐频道横向压缩为16:9形式。

2017年2月，央视设立面向全国各地广电网络有线电视互动平台的“CCTV央视专区”，其点播的央视节目带则是成片节目左上角加上CCTV双线台标，质感同CCTV奥运频道。从2017年7月起面向全国上星综合频道和地面频道播出的中宣部系政论纪录专题片（如《[[将改革进行到底|将改革进行到底]]》、《[[法治中国|法治中国]]》、《[[大国外交_(纪录片)|大国外交]]》、《[[强军|强军]]》、《[[辉煌中国|辉煌中国]]》、《[[我们一起走过_(纪录片)|我们一起走过]]》等），大多地方电视台播出的片源是央视提供的成片节目，左上角加上“CCTV-1 综合”台标，与真正电视荧屏上的CCTV-1高、标清对比，质感还是有些差异（实际上看右上角有无报时器就可以辨别出来）。

2018年9月，CCTV4K超高清频道筹备开播，在上星测试期间的9月26日当天使用使用双线“CCTV”+勺型标标志，勺标内为综艺体“超高清”字样，悬挂于屏幕左上角；而在9月27日之后则改用使用双线“CCTV”+勺型标标志，勺标内为“4K”字样，并在此标志下方加挂综艺体的“超高清”。

=== 新单线台标时期 ===
[[File:CCTV.svg|thumb]]
[[File:CGTN.svg|thumb]]台标，这一版本台标在电视画面中以半透明显示。]]
2015年1月6日起，新闻频道在凌晨时段出现新版台标宣传片，同时在新闻频道凌晨时段节目的滚动条、片尾和宣传片出现单线CCTV标识，该台标只是央视的内部宣传标志，并非正式启用台标；2016年1月1日，[[中央电视台中文国际频道|中央电视台中文国际频道]]正式启用新版台标宣传片以及角标；2017年8月27日，[[中华人民共和国第十三届运动会|天津全运会]]开幕当日，[[中央电视台体育频道|中央电视台体育频道]]、[[中央电视台体育赛事频道|体育赛事频道]]在全运会赛事期间的赛事片头（包括《体育新闻》等栏目）出现单线CCTV标识。目前除新闻频道外，央视其它所有频道的宣传片（如节目预告以及公益广告等）已经全部采用单线CCTV标志。近年，单线CCTV标志亦开始在央视记者话筒上出现。

2016年12月31日，因应[[中国环球电视网|中国环球电视网]]正式上线，“'''CCTV-NEWS'''”、“'''{{lang|fr|CCTV-Français}}'''”、“'''{{lang|es|CCTV-Español}}'''”、“'''CCTV-{{rtl-lang|ar|العربية}}'''”、“'''CCTV-{{lang|ru|Pусский}}'''”以及“'''CCTV-9 Documentary'''”的台标分别改为新单线台标的“'''CGTN'''”、“'''{{lang|fr|CGTN Français}}'''”、“'''{{lang|es|CGTN Español}}'''”、“'''CGTN {{rtl-lang|ar|العربية}}'''”、“'''CGTN {{lang|ru|Pусский}}'''”以及“'''CGTN Documentary'''”，其中CGTN标识字体同CCTV新单线台标一致。CGTN所有频道的高清信号右上角亦没有任何标记，只是左上角的CGTN台标偏小且完全适应16:9格式；反而标清信号的左上角台标偏大拉伸，和高清信号的台标位置尺度有点偏差。
