'''中国印钞造币总公司'''，是[[中国人民银行|中国人民银行]]直属的国家[[法定货币|法定货币]]印制企业。主要从事[[人民币|人民币]]印制与铸造、钞票纸、银行[[信用卡|信用卡]]的研制生产、印钞造币专用机械和银行机具制造等业务，此外还负责印制[[增值税|增值税]]专用发票、[[有价证券|有价证券]]、银行专用票据、高级防伪证书等。

==沿革==
[[甲午战争|甲午战争]]后，外国[[银元|银元]]大量流入[[清朝|清朝]][[奉天省|奉天省]]，导致[[纹银|纹银]]外流、制钱短缺且贬值、[[通货膨胀|通货膨胀]]加剧。为了解决奉天制钱短缺的问题，[[盛京将军|盛京将军]][[依克唐阿|依克唐阿]]在1896年2月呈《奉天拟购机设局铸造银元》[[奏折|奏折]]；获批复后，依克唐阿派人赴[[湖北|湖北]]、[[广东|广东]]，考察设计机器局章程和制造银元的成色、范文等等，还从[[天津|天津]][[礼和洋行|礼和洋行]]订购了铸币机器及[[锅炉|锅炉]]。1896年8月23日下午，依克唐阿等官员抵达奉天东关外边门里的[[奉军|奉军]]旧营房，在空地上埋下奠基石，宣布[[奉天机器局|奉天机器局]]成立。奉天机器局后演变为今[[沈阳造币有限公司|沈阳造币有限公司]]。两年后，奉天機器局正式生产一角、二角、五角、一元银币，1900年开始铸铜元<ref name=lishi>{{Cite web |url =http://www.cbpm.cn/home/cn/aboutUs/c651689/list.shtml  |title =发展历程  |author =  |date =  |publisher = 中国印钞造币总公司 |language =  |accessdate = 2018-08-10 |quote =  }}</ref>。
 
1908年，清朝官员[[陈锦涛|陈锦涛]]等人在[[美国|美国]]签订[[合同|合同]]，仿“美京国立印刷局”之制，在中国建[[度支部|度支部]]印刷局；此为今[[北京印钞有限公司|北京印钞有限公司]]的前身。1909年冬，自美国聘来画家、钢版雕刻师[[海趣|海趣]]（Lorenzo J. Hatch）及格兰特等其他四位技术人员，传授钢版雕刻、[[凹版印刷|凹版印刷]]技术。经过两年，度支部印刷局基建初成、设备到位。朝廷批准：“印刷局由度支部奏设，办理全国纸币及各项有价证券”，“[[邮票|邮票]]、[[印花税|印花]]票、车票、[[公债|公债]]票、官照、[[文凭|文凭]]、契尾、粮串、盐茶引、牙贴、各项官用证券，一律归印刷局印刷”。清廷聘海趣设计雕刻了“[[大清银行兑换券|大清银行兑换券]]”原版，这是中国钢版雕刻、凹版印刷钞票的开端，但该兑换券因1912年清朝覆灭而未面世。1914年由中国人自行设计、雕刻、印刷的“[[殖边银行兑换券|殖边银行兑换券]]”，成为最先面世的中国钢版雕刻、凹版印刷钞票<ref name=lishi/>。

1928年起，[[中国共产党|中国共产党]]领导的各革命根据地印制发行过200余种货币。当时货币印制工艺简单，票面设计富有革命色彩。1928年，[[毛泽东|毛泽东]]接受四县经济委员兼防务主任[[王佐|王佐]]的建议，在[[井冈山革命根据地|井冈山革命根据地]]创建了中国共产党领导下的第一个造币厂——上井造币厂。1937年[[抗日战争|抗日战争]]全面爆发后，中国共产党领导的[[陕甘宁边区|陕甘宁边区]]、[[晋察冀边区|晋察冀边区]]、[[晋冀鲁豫边区|晋冀鲁豫边区]]、华中等各抗日根据地分别设立银行，印制发行货币。有的抗日根据地还生产了凹版印刷的钞票。1945年8月，抗日战争胜利时，中国共产党领导的各[[解放区|解放区]]和抗日根据地已逐渐连成一体。该时期货币印制成熟，设备随着抗日战争胜利而获得更新，使用胶印机印制的货币更加精美<ref name=lishi/><ref name=jj>{{Cite web |url = http://www.cbpm.cn/home/cn/aboutUs/development/list.shtml |title = 简介 |author =  |date =  |publisher =中国印钞造币总公司  |language =  |accessdate =2018-08-10   |quote =  }}</ref>。

[[第二次国共内战|第二次国共内战]]时期，1946年11月和1947年初，在中国共产党领导下，分别筹建了太行造纸厂、长城造纸厂。1948年底，在[[中国人民银行|中国人民银行]]成立前夕，中国北方多个解放区的印钞造币厂合并成立第一、二、三印刷局及直属厂。1948年底，中国人民银行在[[石家庄|石家庄]]成立。1948年至1953年，共21个印刷厂参与印制了第一套[[人民币|人民币]]。解放区的印钞厂担负了大部分票券的设计、原版制作、印制任务。1948年12月1日，中国人民银行在[[河北省|河北省]][[平山县|平山县]]支行发行第一批人民币<ref name=lishi/>。

同时从1945年到1949年，中国共产党方面还陆续接收了[[日本|日本]]及[[国民政府|国民政府]]方面的许多印钞造币机构。1945年8月底，接收日本人在[[张家口市|张家口市]]办的长青路印刷厂。1948年11月，接收[[中央造币厂|中央造币厂]]沈阳保管处。1949年1月，接收[[中央印制厂|中央印制厂]]北平厂，更名为中国人民印刷厂。1949年5月，接收中央印制厂上海厂。<ref name=lishi/>。

1949年12月，成立'''中国人民银行印制管理局'''。1949年，三个印刷局分别同[[北京|北京]]、[[天津|天津]]、[[上海|上海]]等印刷厂合并。到1949年末，已有14个印制企业，职工17000多人。1950年，精简为10个企业，职工11240人。1958年，精简为4个企业（北京印钞厂、上海印钞厂、沈阳造币厂、上海造币厂），职工6917人。1980年，[[国务院|国务院]]批准成立'''中国人民银行印制总公司'''，对内使用中国人民银行印制管理局名称。1991年，更名为'''中国印钞造币总公司'''<ref name=lishi/>。

在[[三线建设|三线建设]]中，从1960年代初到1980年代末，陆续在[[四川|四川]]、[[山西|山西]]、[[江西|江西]]建成三个三线企业：
*国营东河印制公司：1963年，中国人民银行总行党组决定在[[四川省|四川省]]西南新建后备印钞和造纸厂。1965年9月，国务院和[[国家计委|国家计委]]以《（65）计财字756号》批准了中国人民银行的报告。
*国营一四五厂：1970年8月，[[中华人民共和国财政部|财政部]]通知在[[山西省|山西省]][[高平县|高平县]]建设一四五厂，战时是五四一厂的后备厂，平时是总、分厂关系。
*国营七一二厂：1970年7月，财政部上报国务院《关于五四二厂小三线建设请示》获得“同意”，厂址定在[[江西省|江西省]][[莲花县|莲花县]]<ref name=lishi/>。

1980年代以来，随着[[改革开放|改革开放]]的深入，中国印钞造币事业经历了体制改革、调整扩建，引进了国外先进技术。1983年，中国成为“[[太平洋|太平洋]]沿岸国家印钞者会议正式成员国”<ref name=lishi/>。1993年，成都印钞有限公司迁到[[成都|成都]][[温江区|温江]]。2005年以来，中国印钞造币总公司在中国人民银行总行的统一部署下，逐步建立起现代企业制度<ref name=jj/>。
