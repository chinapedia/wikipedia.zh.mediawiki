'''中国人民武装警察部队士官学校'''，简称'''武警士官学校'''，本部位于[[浙江省|浙江省]][[杭州市|杭州市]]，是浙江省唯一一所部队院校，是[[中国人民武装警察部队|中国人民武装警察部队]]总部下属的培养武警指挥干部的军事高等院校<ref>{{Cite web|url = http://news.zust.edu.cn/html/yaowenjujiao/ttxw/18512.html|title = 学校与中国人民武装警察部队士官学校签订全面共建合作协议|author = |date = 2017-08-09|publisher = 浙江科技学院|language = |accessdate = |quote = |archive-url = https://web.archive.org/web/20170719100930/http://news.zust.edu.cn/html/yaowenjujiao/ttxw/18512.html|archive-date = 2017-07-19|dead-url = yes}}</ref>。

== 沿革 ==
;中国人民武装警察部队杭州士官学校
*1984年4月，组建成立'''中国人民武装警察部队杭州指挥学校'''（'''武警杭州指挥学校'''），隶属[[武警浙江总队|武警浙江总队]]。学校驻杭州市花坞<ref name=zuochl/><ref name=zjzx/><ref> {{Cite book | author = 黄正淼主编 | title = 浙江省军事志 | location =  | publisher = 方志出版社 | date =1999年  | pages =  | ISBN =  | accessdate =  | url =  | language =  | quote =第四节 军事学校   }} </ref>。
*2004年，建立'''[[中国人民武装警察部队指挥学院|中国人民武装警察部队指挥学院]]杭州分院'''，与武警杭州指挥学校[[合署办公|合署办公]]<ref>{{Cite web |url = http://www.huaue.com/gkao/200475150159.htm |title =江苏：部分军校可补填志愿  |author =  |date = 2004-07-05 |publisher =华禹教育网  |language =  |accessdate =  |quote =  }}</ref>。
*2004年起，根据军队院校有关招生规定，军事指挥本科专业开始由内部定向招生扩展为地方、部队混合招生<ref name=zuochl/>。
*2006年，经中央军委、武警总部批准，武警杭州指挥学校更名'''中国人民武装警察部队杭州指挥学院'''。2006年9月5日正式更名，武警浙江总队[[王平安|王平安]]总队长为学院揭牌<ref name=zjzx>{{Cite web |url = http://zjnews.zjol.com.cn/05zjnews/system/2011/09/01/017814685.shtml |title = 武警杭州士官学校挂牌成立 |author =  |date = 2011-09-01 |publisher =浙江在线  |language =  |accessdate =  |quote =  }}</ref><ref name=zuochl>{{Cite web |url = http://news.sina.com.cn/c/2006-09-06/04259945030s.shtml |title = 武警杭州指挥学院昨成立 |author =  |date =2006-09-06  |publisher = 新浪 |language =  |accessdate =  |quote =  }}</ref>。
*2011年7月，在武警部队院校调整撤并中，经中央军委、武警总部批准，更名'''中国人民武装警察部队杭州士官学校'''，由副师级单位升格为正师级单位，转型为任职教育院校，主要担负武警后勤技术、管理、维修等专业的士官培训工作。2011年9月1日，武警杭州士官学校举行挂牌暨开学典礼仪式，武警部队副司令员[[戴肃军|戴肃军]]，[[中共浙江省委|中共浙江省委]]常委、秘书长、政法委书记[[李强_(江苏省委书记)|李强]]为学校揭牌<ref name=zjzx/><ref>{{Cite web |url = http://zjnews.zjol.com.cn/05zjnews/system/2011/09/02/017815395.shtml |title = 武警杭州士官学校隆重举行挂牌暨开学典礼仪式 |author =  |date = 2011-09-02 |publisher =浙江在线  |language =  |accessdate =  |quote =  }}</ref>。学校是[[武警总部|武警总部]]直属单位。
*2012年，经[[中国人民武装警察部队后勤部|中国人民武装警察部队后勤部]]后营〔2012〕602号文件核准，同意武警杭州士官学校新校区建设项目，资金来源为自筹，建设地址位于[[富阳市|富阳市]]（今[[杭州市|杭州市]][[富阳区|富阳区]]）[[富春街道|富春街道]]三联村<ref>{{Cite web |url = http://www.zjdpc.gov.cn/art/2014/4/28/art_659_647135.html  |title =武警杭州士官学校新校区建设项目  |author =  |date = 2014-04-28  |publisher = 浙江省发改委 |language =  |accessdate =  |quote =  }}</ref>。

;中国人民武装警察部队石家庄士官学校
*1983年，经[[中华人民共和国国务院|国务院]]批准，[[武警河北总队|武警河北总队]]教导大队改建为'''中国人民武装警察部队石家庄指挥学校'''（'''武警石家庄指挥学校'''）<ref name=gfb>{{Cite web |url =http://www.mod.gov.cn/service/2008-06/02/content_4085906.htm  |title =  武警石家庄指挥学院|author =  |date =2008-06-02  |publisher =中华人民共和国国防部  |language =  |accessdate =  |quote =  }}</ref>。1984年成立，为中专学校<ref name=jdrcw/>。
*2000年，学历层次调整为大专<ref name=jdrcw/>。
*2004年，建立'''[[中国人民武装警察部队指挥学院|中国人民武装警察部队指挥学院]]石家庄分院'''，与武警石家庄指挥学校[[合署办公|合署办公]]。
*2006年9月，武警石家庄指挥学校升格'''中国人民武装警察部队石家庄指挥学院'''，主要承担武警河北总队生长干部培养、营以下干部培训和武警部队信息化士官培训任务<ref name=jdrcw/><ref name=gfb/>。
*2011年，武警部队院校编制体制调整改革，经[[中央军事委员会|中央军委]]批准，学院改建为'''中国人民武装警察部队石家庄士官学校'''，主要承担武警部队信息化士官培养和学兵培训任务，同时承担武警部队通信与计算机、保密档案、文化影视专业等12个工种的职业技能鉴定工作。培训任务分为四类：士官职业技术教育、士官资格培训、士官升级培训、士官任职培训<ref name=jdrcw>{{Cite web |url = http://www.81rc.mil.cn/Civilianpost/2014-09/12/content_6136080.htm |title = 武警部队石家庄士官学校 |author =  |date =2014-09-12  |publisher = 军队人才网 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url =  http://news.ifeng.com/a/20170515/51097002_0.shtml|title = 那些年我们一起上过的军校 ——武警石家庄士官学校 |author =  |date = 2017-05-15 |publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref><ref name=guapai/>。2011年9月7日，武警石家庄士官学校举行了挂牌仪式暨开学典礼。武警石家庄士官学校成为武警部队仅有的两所士官学校之一（另一所是[[中国人民武装警察部队士官学校|武警杭州士官学校]]）<ref name=guapai>{{Cite web|url = http://121.28.35.251/content.jsp?code=000224522/2011-10010|title = 武警石家庄士官学校挂牌|author = |date = 2011-09-08|publisher = 石家庄市政府信息公开平台|language = |accessdate = |quote = |archive-url = https://web.archive.org/web/20170904113157/http://121.28.35.251/content.jsp?code=000224522%2F2011-10010|archive-date = 2017-09-04|dead-url = yes}}</ref>。
*2012年，武警石家庄士官学校迁建项目招标，迁建目的地是河北省石家庄市[[正定县|正定县]]南岗村<ref>{{Cite web |url = http://www.86gc.net/article/html/137481.html |title = 中国人民武装警察部队石家庄士官学校迁建项目 |author =  |date = 2012-10-23 |publisher = 中国工程网 |language =  |accessdate =  |quote =  }}</ref>。

;中国人民武装警察部队士官学校
*2017年，在[[深化国防和军队改革|深化国防和军队改革]]中，中国人民武装警察部队杭州士官学校、中国人民武装警察部队石家庄士官学校等单位调整组建'''中国人民武装警察部队士官学校'''，本部位于杭州<ref>{{Cite web |url =  http://www.mod.gov.cn/topnews/2017-06/29/content_4783973.htm|title = 国防部公布调整改革后军队院校名称 |author =  |date = 2017-06-29 |publisher =中华人民共和国国防部  |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url =  http://www.zjabank.com/Index/info?article_id=52586|title =警银联建，共育英才 ——省中行郭心刚行长一行慰问武警士官学校  |author =  |date =2017-07-28  |publisher =浙江省银行业协会  |language =  |accessdate =  |quote =  }}</ref>。学校是[[武警总部|武警总部]]直属单位。
