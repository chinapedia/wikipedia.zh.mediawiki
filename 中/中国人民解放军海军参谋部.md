'''中国人民解放军海军参谋部'''，位于[[北京市|北京市]]，是[[中国人民解放军海军|中国人民解放军海军]]机关职能部门。

==沿革==
1950年4月14日，[[中央人民政府人民革命军事委员会|中央军委]]海军领导机构在北京成立<ref>{{Cite web |url = http://cpc.people.com.cn/GB/64162/64165/78561/79761/5555446.html |title = 1950年4月14日 中央军委海军领导机构成立 |author =  |date =  |publisher = 中国共产党新闻网 |language =  |accessdate = 2017-01-23 |quote =  }}</ref><ref>{{Cite web|url =http://history.huanqiu.com/mil1/2013-04/3801559.html|title =人民解放军70个军如何变成18个集团军？|author =|date =2013-04-08|publisher =环球网|language =|accessdate =|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20130818111722/http://history.huanqiu.com/mil1/2013-04/3801559.html|archivedate =2013年8月18日|df =}}</ref>。下设'''中国人民解放军海军司令部'''。

2016年，在[[深化国防和军队改革|深化国防和军队改革]]中，中国人民解放军海军司令部撤销，组建'''中国人民解放军海军参谋部'''<ref name=wy>{{Cite web |url =  http://news.163.com/17/0111/21/CAHFSGFG000187VE.html|title = 海军原司令部训练部部长郁忠转任海军参谋部训练局局长 |author =  |date = 2017-01-11 |publisher =  网易|language =  |accessdate =  |quote =  }}</ref>。
