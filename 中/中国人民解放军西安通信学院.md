'''中国人民解放军西安通信学院'''（简称'''西安通信学院'''，或'''解放军西安通信学院'''），是一所培养初级通信指挥军官、工程技术军官的军事高等院校，隶属[[中央军事委员会训练管理部|中央军事委员会训练管理部]]，[[正师级|正师级]]。<ref>[http://www.rvedu.com/daxue/3905.html 教育点评网：西安通信学院简介] {{webarchive|url=https://web.archive.org/web/20111020174532/http://www.rvedu.com/daxue/3905.html |date=2011-10-20 }}</ref><ref>[http://learning.sohu.com/2004/03/06/88/article219318825.shtml 搜狐教育：西安通信学院]</ref>

学院分为三个营区：
*生活区：[[西安市|西安市]][[长安中路|长安中路]]、[[西安市|西安市]][[丈八东路|丈八东路]]
*教学区：[[西安市|西安市]][[王曲镇|王曲镇]]

== 沿革 ==
{{参见|解放军通信技术学校|解放军重庆通信学院}}
*1978年，[[解放军通信技术学校|解放军通信技术学校]]（有线通信专业）升格为'''中国人民解放军有线电通信技术学校'''。
*1986年，升格'''中国人民解放军西安通信学院'''。
{{参见|空军电讯工程学院}}
*1992年，[[空军电讯工程学院|空军电讯工程学院]]并入。
*1993年，西安通信学院（原空军电讯工程学院部分）分出，改建[[空军电讯工程学院|空军电讯工程学院]]，后参与组建[[空军工程大学|空军工程大学]]。
*2016年，在[[深化国防和军队改革|深化国防和军队改革]]中，其原来的上级单位[[中国人民解放军总参谋部|中国人民解放军总参谋部]]撤销，该院改隶[[中央军事委员会训练管理部|中央军事委员会训练管理部]]。<ref>[http://kaoyan.eol.cn/ge_di_kao_yan/shaanxi/shaan_xi/201609/t20160926_1452831.shtml 西安通信学院2017年硕士研究生招生简章，中国研究生招生信息网，2016-09-26]</ref>
*2017年，在[[深化国防和军队改革|深化国防和军队改革]]中，[[中国人民解放军国防科学技术大学|中国人民解放军国防科学技术大学]]、[[中国人民解放军国际关系学院|中国人民解放军国际关系学院]]、[[中国人民解放军国防信息学院|中国人民解放军国防信息学院]]、中国人民解放军西安通信学院、[[中国人民解放军电子工程学院|中国人民解放军电子工程学院]]、[[中国人民解放军理工大学|中国人民解放军理工大学]]气象海洋学院为基础重建[[中国人民解放军国防科学技术大学|中国人民解放军国防科学技术大学]]。校本部设在[[长沙|长沙]]，内设学院位于长沙、[[南京|南京]]、[[武汉|武汉]]、[[合肥|合肥]]等地<ref name=xin>{{Cite web |url =http://www.thepaper.cn/newsDetail_forward_1699945  |title = 新国防科大今年以国防科大、国关学院等6所院校为基础重建 |author =  |date =  2017-06-03 |publisher =  澎湃新闻|language =  |accessdate =  |quote =  }}</ref>。
