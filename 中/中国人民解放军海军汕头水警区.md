'''中国人民解放军海军汕头水警区'''，机关驻地[[广东省|广东省]][[汕头市|汕头市]]，是[[中国人民解放军海军福建基地|中国人民解放军海军福建基地]]的[[水警区|水警区]]。

== 沿革 ==
1950年12月21日，以陆军41军部分人员为基础，组建汕头独立巡防区，隶属中南军区海军建制。卢仕盛任主任，高永久任政委。1954年11月10日，海军奉军委总参谋部1954年11月3日电令，将汕头巡防区扩建为汕头水警区，仍归中南军区海军建制，执行师级权限。1969年12月25日，海军电复[[南海舰队|南海舰队]]，根据[[中央军委|中央军委]]1969年12月22日批复，汕头水警区机关从汕头迁至[[礐石街道|礐石]]。

汕头水警区曾参加过[[八六海战|八六海战]]、[[万山海战|万山海战]]等战斗，涌现出“[[海上先锋艇|海上先锋艇]]”、“钢铁战士”[[麦贤得|麦贤得]]等英雄集体和人物，并产生了海上“先锋精神”。<ref name=tongren/>

2016年初，在[[深化国防和军队改革|深化国防和军队改革]]中，汕头水警区由[[南海舰队|南海舰队]]转隶[[东海舰队|东海舰队]][[中国人民解放军海军福建基地|福建基地]]。<ref name=tongren>{{Cite web |url = http://st.southcn.com/content/2016-02/21/content_142721621.htm |title =  海军“铜仁舰”在汕正式入列命名|author =  |date =2016-02-21  |publisher =  南方网|language =  |accessdate =  |quote =  }}</ref>
