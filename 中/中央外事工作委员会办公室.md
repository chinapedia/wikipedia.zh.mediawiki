'''中央外事工作委员会办公室'''，标准简称'''中央外办'''，亦简称'''中央外事办'''，为[[中央外事工作委员会|中央外事工作委员会]]的办事机构，是[[中共中央直属机构|中共中央直属机构]]。

==沿革==
1958年3月6日，[[中共中央|中共中央]]、[[中华人民共和国国务院|国务院]]发出《关于中共中央设立外事小组和国务院设立外事办公室》的联合通知；其中，[[中华人民共和国外交部|外交部部长]][[陈毅|陈毅]]兼任“中央外事小组”组长，全面负责外事工作；在国务院系统中，对应设立“国务院外事办公室”，陈毅兼主任，[[廖承志|廖承志]]任副主任<ref>{{Cite web|url=http://www.sxfao.gov.cn/wsbc-jgzn/erji-cwb-jj.jsp?urltype=tree.TreeTempUrl&wbtreeid=54|title=历史沿革|accessdate=2018-03-22|publisher=陕西省人民政府外事办公室|archiveurl=https://web.archive.org/web/20150315083013/http://www.sxfao.gov.cn/wsbc-jgzn/erji-cwb-jj.jsp?urltype=tree.TreeTempUrl&wbtreeid=54|archivedate=2015-03-15}}</ref>。

[[文化大革命|文化大革命]]期间，中央外事小组撤销。1970年6月国务院精简机构，国务院外事办公室撤销，业务并入外交部<ref>{{Cite web|url=http://dangshi.people.com.cn/GB/138900/15471928.html|title=吴凡吾：“外交家”陈毅重视民间外交|accessdate=2018-03-22|date=2011-08-22|publisher=人民网}}</ref><ref>{{Cite web|url=http://news.163.com/05/0525/18/1KK9NCRS00011247.html|title="文革"大办"五七"干校　源自毛泽东"五七指示"|accessdate=2018-03-22|date=2005-05-25|publisher=网易}}</ref>。1981年，中共中央决定，恢复“[[中央外事工作委员会|中央外事工作领导小组]]”；[[李先念|李先念]]任组长，[[万里|万里]]任副组长<ref name="shy2">{{Cite web|url=http://news.takungpao.com/mainland/focus/2013-09/1887233.html|title=外事安全海权三办合一  确保中国海洋权益|accessdate=2018-03-22|date=2013-09-06|publisher=大公网}}</ref>。

1988年，根据[[第七届全国人民代表大会#第一次会议|第七届全国人大第一次会议]]通过的《[[国务院机构改革#李_鹏时期——1988年|国务院机构改革]]方案》，设“'''国务院外事办公室'''”，承担[[中央外事工作委员会|中央外事工作领导小组]]的具体事务<ref>{{Cite web|url=http://www.chinalawedu.com/news/1200/21752/21754/21766/21792/2006/3/so64232521736002153-0.htm|title=关于国务院机构设置的通知|accessdate=2018-03-22|date=1988-08-13|publisher=法律教育网|archiveurl=https://web.archive.org/web/20150402142224/http://www.chinalawedu.com/news/1200/21752/21754/21766/21792/2006/3/so64232521736002153-0.htm|archivedate=2015-04-02}}</ref>。

1998年8月，中共中央、国务院决定，撤销“国务院外事办公室”，组建“'''中央外事办公室'''”。中央外事办公室作为中央外事工作领导小组办事机构，列入[[中共中央直属机构|中共中央直属机构]]序列<ref>{{Cite web|url=http://d.youth.cn/tpxw_35291/201309/t20130902_3807321.htm|title=中国外事系统高官密集调整 学科背景多元化|accessdate=2018-03-22|date=2013-09-02|publisher=中国青年网}}</ref><ref name="gong2">{{Cite web|url=http://www.usc.cuhk.edu.hk/PaperCollection/Details.aspx?id=7458|title=中国外交决策机制变迁研究(1949～2009年)|accessdate=2018-03-22|author=宫力、门洪华、孙东方|date=|publisher=中国研究服务中心}}</ref>。

2000年9月，中共中央决定，组建“中央国家安全工作领导小组”，与“中央外事工作领导小组”[[合署办公|合署办公]]，[[一个机构两块牌子|一个机构两块牌子]]。[[江泽民|江泽民]]任组长，[[钱其琛|钱其琛]]任副组长，[[刘华秋|刘华秋]]任'''中央外事工作领导小组办公室'''和'''中央国家安全工作领导小组办公室'''主任<ref name="shy2" /><ref name="gong2" />。

2012年下半年，中央设立“[[中央海洋权益工作领导小组|中央海洋权益工作领导小组]]”，负责海洋权益事务的议事和协调。中央海洋权益工作领导小组设立了办事机构“'''中央海洋权益工作领导小组办公室'''”，与中央外事工作领导小组办公室[[合署办公|合署办公]]<ref>{{Cite web|url=http://epaper.nhaidu.com/2013-03/03/content_93738.htm|title=海洋权益维护高层次协调机构 中央海权办已成立|accessdate=2018-03-22|date=2013-03-03|publisher=海都网}}</ref>。

2014年[[中央国家安全委员会|中央国家安全委员会]]成立后，中央外事工作领导小组不再保留“中央国家安全工作领导小组”的名义。中央外事工作领导小组办公室随之不再保留“中央国家安全工作领导小组办公室”的名义<ref>{{Cite web|url=http://i.ifeng.com/news/dalu/news?cid=0&aid=79423469|title=媒体：两大超级机构相继运作 栗战书掌国安办|accessdate=|author=|date=2014-03-07|publisher=凤凰网|language=|quote=}}</ref>。

2018年3月中共中央印发的《[[深化党和国家机构改革|深化党和国家机构改革]]方案》称，“[[中央全面深化改革领导小组|中央全面深化改革领导小组]]、[[中央网络安全和信息化领导小组|中央网络安全和信息化领导小组]]、[[中央财经领导小组|中央财经领导小组]]、[[中央外事工作领导小组|中央外事工作领导小组]]改为委员会。为加强党中央对涉及党和国家事业全局的重大工作的集中统一领导，强化决策和统筹协调职责，将中央全面深化改革领导小组、中央网络安全和信息化领导小组、中央财经领导小组、中央外事工作领导小组分别改为[[中央全面深化改革委员会|中央全面深化改革委员会]]、[[中央网络安全和信息化委员会|中央网络安全和信息化委员会]]、[[中央财经委员会|中央财经委员会]]、[[中央外事工作委员会|中央外事工作委员会]]，负责相关领域重大工作的顶层设计、总体布局、统筹协调、整体推进、督促落实。”“4个委员会的办事机构分别为[[中央全面深化改革委员会办公室|中央全面深化改革委员会办公室]]、[[中央网络安全和信息化委员会办公室|中央网络安全和信息化委员会办公室]]、[[中央财经委员会办公室|中央财经委员会办公室]]、[[中央外事工作委员会办公室|中央外事工作委员会办公室]]。”<ref name="fangan2">{{Cite web|url=http://www.xinhuanet.com/2018-03/21/c_1122570517.htm|title=中共中央印发《深化党和国家机构改革方案》|accessdate=2018-03-21|date=2018-03-21|publisher=新华网}}</ref>

同时该方案还规定：“不再设立[[中央维护海洋权益工作领导小组|中央维护海洋权益工作领导小组]]。为坚决维护国家主权和海洋权益，更好统筹外交外事与涉海部门的资源和力量，将维护海洋权益工作纳入中央外事工作全局中统一谋划、统一部署，不再设立中央维护海洋权益工作领导小组，有关职责交由中央外事工作委员会及其办公室承担，在中央外事工作委员会办公室内设维护海洋权益工作办公室。”“调整后，中央外事工作委员会及其办公室在维护海洋权益方面的主要职责是，组织协调和指导督促各有关方面落实党中央关于维护海洋权益的决策部署，收集汇总和分析研判涉及国家海洋权益的情报信息，协调应对紧急突发事态，组织研究维护海洋权益重大问题并提出对策建议等。”<ref name="fangan2" />

==职责==
根据《{{s|中央外事工作委员会办公室职能配置、内设机构和人员编制规定}}》，中央外办承担下列职责<ref>{{Cite web|url=http://www.china.com.cn/cpc/2011-04/15/content_22369387.htm|title=中央外事办公室主要职能|accessdate=2018-03-22|date=2011-04-15|publisher=中国网}}</ref>：
{{columns-list|2|
# 对国际形势和执行外交政策中的重大问题、外事管理工作进行调查研究，提出建议。
# 承办中央外事工作领导小组全体会议和办公会议的会务工作，催办会议决定事项，承办外事协调工作。
# 代党中央拟定和修订外事工作的某些全国性规定，审核中央、国家机关各部门和各省、自治区、直辖市制定的重要外事规定。
# 办理中央、国家机关各部门和各省、自治区、直辖市报送中央外事工作领导小组和国务院的有关重要外事问题的请示、报告。
# 承办中央外事工作领导小组和国务院交办的其他事项。
}}
