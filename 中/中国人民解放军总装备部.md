'''中国人民解放军总装备部'''曾是[[中国人民解放军|中国人民解放军]]的裝備工作機關。成立於1998年，由[[中央軍事委員會|中央軍事委員會]]領導。總裝備部負責全军的装备工作。並負責對裝備活動經費的審查、監督、審計，並負責裝備理論等工作。總裝備部下設有司令部、政治部、军兵种装备、后勤部及综合计划部等部門。2016年1月在[[深化国防和军队改革|深化国防和军队改革]]中撤销，改组为'''[[中央军委装备发展部|中央军委装备发展部]]'''。

==国防工业出版社==
国防工业出版社（第二社名是“新时代出版社”）于1954年11月经[[中共中央宣传部|中共中央宣传部]]批准成立，是中央级综合性科技出版社。先后归原[[中华人民共和国第二机械工业部|二机部]]、[[中华人民共和国第三机械工业部|三机部]]、国务院国防工办、[[国防科工委|国防科工委]]领导。1998年起，由'''中国人民解放军总装备部'''直接领导<ref name=gmw>{{Cite web |url =  http://www.gmw.cn/01ds/2004-11/17/content_134234.htm|title = 国防工业出版社喜迎50华诞 |author =  |date = 2004-11-17 |publisher =  光明网|language =  |accessdate =  |quote =  }}</ref>。由中国人民解放军总装备部政治部电子信息基础部主管，中国人民解放军总装备部政治部主办<ref>{{Cite web |url =  http://www.gapp.gov.cn/zongshu/serviceSearchResultbve.shtml?ID=13932|title =国防工业出版社  |author =  |date = 2017-02-25 |publisher =国家新闻出版广电总局  |language =  |accessdate =  |quote =  }}</ref>。1990年代起至2004年，国防工业出版社进行了四次改革，两次获评为“全国良好出版社”，1998年被评为“全国优秀出版社”<ref name=gmw/>。该社以[[江泽民|江泽民]]主席题词“传播科技知识，为国防现代化服务”作为办社宗旨<ref>{{Cite web |url = http://news.xinhuanet.com/book/2013-11/12/c_125687435.htm |title =许西安：版权贸易助推童书出版丰富多彩  |author =  |date = 2013-11-12 |publisher = 新华网 |language =  |accessdate =  |quote =  }}</ref>。

2016年3月底，中央军委印发《[[军队和武警部队全面停止有偿服务|关于军队和武警部队全面停止有偿服务活动的通知]]》。2016年5月7日，军队和武警部队全面停止有偿服务试点任务部署会议举行，7个大单位、17个具体单位被列为试点单位，新闻出版等重点项目被纳入试点范畴。自2015年起，各军队和武警部队出版社的新书及新项目均停止审批<ref>{{Cite web |url = http://www.cbbr.com.cn/article/104077.html |title = 军队出版社将试点“全面停止”有偿服务 |author =  |date = 2016-05-10 |publisher = 中国出版传媒网 |language =  |accessdate =  |quote =  }}</ref>。
