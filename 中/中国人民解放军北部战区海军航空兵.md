'''中国人民解放军北部战区海军航空兵'''，机关驻[[青岛市|青岛市]]，是[[中国人民解放军北部战区海军|中国人民解放军北部战区海军]]的[[中国人民解放军海军航空兵|海军航空兵]]。

==沿革==
1951年7月，'''中国人民解放军海军青岛基地防空兵'''正式成立。1955年，该部队组建了中国人民解放军第一支也是唯一一支水上飞机部队<ref name=xhw/>。

1954年3月18日，6架[[中华民国空军|中华民国空军]]F-47战斗轰炸机从[[台湾|台湾]]侵入大陆，企图轰炸[[中华人民共和国|中华人民共和国]]舰艇。海军航空兵第一个装备喷气式飞机的歼击机团海航2师6团副大队长崔巍、中队长姜凯起飞迎敌，在[[浙江省|浙江省]]南田地区与敌方4架F-47遭遇，两人击落敌机2架，迫使另2架逃走。南田空战揭开了中国人民解放军夺取浙东沿海制空权斗争的序幕，这也是[[中国人民解放军海军航空兵|中国人民解放军海军航空兵]]组建以来的第一次战斗。在随后两个月内，海军航空兵部队先后击落敌机8架，击伤2架<ref name=xhw/><ref>{{Cite web |url =http://news.163.com/10/0824/14/6ES14BQS000146BC.html  |title = 揭秘中国海军航空兵 美航母老对手？ |author =  |date = 2010-08-24 |publisher =网易  |language =  |accessdate =  |quote =  }}</ref>。

1955年1月，在配合其他部队展开[[一江山岛战役|一江山岛战役]]中，日后属于海军北海舰队航空兵的部队的战机击落击伤敌机10架，一江山岛战役是中国人民解放军历史上首次陆海空三军协同作战<ref name=xhw/>。

1959年6月，在海军青岛基地防空兵的基础上成立了'''中国人民解放军海军青岛基地航空兵部'''。1960年7月1日，'''中国人民解放军海军北海舰队航空兵'''在原青岛基地航空兵部的基础上正式组建<ref name=xhw>{{Cite web |url =http://news.xinhuanet.com/mil/2010-07/03/content_13797950.htm  |title = 北海舰队飞行员曾空中拼刺刀俘获美飞行员 |author =  |date = 2010-07-03 |publisher = 新华网 |language =  |accessdate =  |quote =  }}</ref><ref name=rmw>{{Cite web|url =http://qd.people.com.cn/GB/190560/12043626.html|title =北海舰队航空兵庆祝组建五十周年|author =|date =2010-07-03|publisher =人民网|language =|accessdate =|quote =}}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

1965年9月20日，北海舰队航空兵部队的[[高翔_(海航)|高翔]]和僚机黄风生一起迎击侵入中国[[海南岛|海南岛]]领空的一架美国[[F-104星式战斗机|F-104C战斗机]]，高翔驾驶[[歼-6|歼-6]]在距敌机仅291米时追赶射击，直到距敌机仅39米处才拉起机头脱离，成功击落敌机<ref name=xhw/>。

1975年5月，中国人民解放军第一支舰载机部队——北海舰队航空兵某舰载直升机部队成立。1980年1月3日，中国舰载直升机在[[上海|上海]][[长江口|长江口]]外第一次着船成功。舰载机部队成立后，随即担负多项国家任务。1980年5月18日，中国发射运载火箭，4架舰载直升机分别承担遥测、航测、打捞火箭数据舱任务<ref name=xhw/>。

1984年9月27日，中国政府宣布：“中国将派出考察队，前往南大洋和[[南极洲|南极洲]]进行科学考察，并在南极建立一个科学考察站。”北海舰队航空兵舰载机部队随即组成机组，随考察船编队到[[南极洲|南极洲]]参加建站工作，在南极洲上空创造出中国的多项第一<ref name=xhw/>。

21世纪初，北海舰队航空兵舰载机多次随海军舰艇编队出访。2008年参加了[[2008年北京奥运会|2008年北京奥运会]]安保任务，2009年参加了庆祝人民海军成立60周年“多国海军活动”任务。舰载机随舰多次到[[亚丁湾|亚丁湾]]执行护航任务。2008年，北海舰队航空兵驻[[青岛|青岛]]部队奉命到一线清理[[浒苔属|浒苔]]<ref name=xhw/>。

为支援地方经济建设，北海舰队航空兵向地方转让了[[青岛流亭机场|青岛流亭机场]]，开放了部分沿海机场，并陆续调整了24个训练空域，截至2010年每年安全保障民航飞行将近30万架次<ref name=rmw/>。

2016年，在[[深化国防和军队改革|深化国防和军队改革]]中，北海舰队改称北部战区海军，北海舰队航空兵随之改称'''中国人民解放军北部战区海军航空兵'''<ref>{{Cite web |url =http://military.people.com.cn/n1/2016/0406/c1011-28253750.html  |title =北部战区海军预警机和警戒机进行海上超低空飞行  |author =  |date = 2016-04-06 |publisher = 人民网 |language =  |accessdate =  |quote =  }}</ref>。
