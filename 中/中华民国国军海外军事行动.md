'''中華民國國軍海外軍事行動'''，[[中華民國國軍|中華民國國軍]]的[[軍事行動|海外軍事行動]]與國軍海外活動。
==海軍==
;中華民國海軍敦睦遠航訓練支隊曾有以下海外行動：
{{Main|敦睦艦隊}}
* 1967年，[[郭勳景|郭勳景]]少將前往韓國、琉球、菲律賓及美國關島訪問，同時建立起中華民國海軍每年正式派遣支隊敦睦訪問友邦的任務<ref name="歷年敦睦支隊率隊長官及活動簡介">{{Cite web |url = http://museum.mnd.gov.tw/Publish.aspx?cnid=1458&p=37005 |title = 歷年敦睦支隊率隊長官及活動簡介 |date = 2012-10-11 |publisher = 國軍歷史文物館 |language = zh-tw |accessdate = 2015-06-06 |archiveurl = https://web.archive.org/web/20150221183323/http://museum.mnd.gov.tw/Publish.aspx?cnid=1458&p=37005 |archivedate= 2015-02-21}}</ref>。
* 1968年，海軍首次訪問美國夏威夷，由[[劉作柄|劉作柄]]少將率領<ref name="歷年敦睦支隊率隊長官及活動簡介"/>。
* 1969年，海軍遠航支隊在[[劉定邦|劉定邦]]少將的率領下，創下中華民國史上空前創舉，訪問澳洲及紐西蘭，這是唯一的一次，也是最後一次<ref name="歷年敦睦支隊率隊長官及活動簡介"/>。
* 1970年，海軍遠航支隊在青島海校第5期畢業的[[楊松泉|楊松泉]]少將的率領下，第一次訪問菲律賓美海軍基地蘇比克灣<ref name="歷年敦睦支隊率隊長官及活動簡介"/>。
* 1971年，海軍敦睦支隊在[[劉和謙|劉和謙]]少將的率領下，訪問美國、琉球群島及韓國，其中美國包含關島、中途島、珍珠港及檀香山4個港口，為海軍敦睦支隊訪問美國最多港口之支隊<ref name="歷年敦睦支隊率隊長官及活動簡介"/>。
* 1972年，海軍敦睦支隊在[[袁昌炎|袁昌炎]]少將的率領下，訪問美國關島及日本[[本州島|本州島]][[吳港|吳港]]<ref name="歷年敦睦支隊率隊長官及活動簡介"/>。是海軍敦睦支隊第一次踏上日本本州島<ref name="歷年敦睦支隊率隊長官及活動簡介"/>。
* 1973年，海軍艦隊第1次前往越南、新加坡訪問及巡弋南疆，同時也是首次在航程中遭遇兩個颱風<ref name="歷年敦睦支隊率隊長官及活動簡介"/>。此次由海軍[[沈鐸|沈鐸]]少將率領<ref name="歷年敦睦支隊率隊長官及活動簡介"/>。
* 1977年，海軍艦隊訪問美國關島及珍珠港，為敦睦支隊最後一次訪問美國，後續中美於68年斷交，即未再前往敦訪<ref name="歷年敦睦支隊率隊長官及活動簡介"/>。
* 1978年，海軍艦隊第3次訪問南非，並與南非海軍實施海上聯合操演<ref name="歷年敦睦支隊率隊長官及活動簡介"/>。此次由[[吳子房|吳子房]]少將率領<ref name="歷年敦睦支隊率隊長官及活動簡介"/>。
* 1982年，海軍的遠航史上首次訪問沙烏地阿拉伯，這是第一次，之後從未再訪問過<ref name="歷年敦睦支隊率隊長官及活動簡介"/>。此次由[[蕭楚喬|蕭楚喬]]少將率領<ref name="歷年敦睦支隊率隊長官及活動簡介"/>。
* 1990年，海軍艦隊訪問斐濟、印尼、新加坡等3國<ref name="歷年敦睦支隊率隊長官及活動簡介"/>。自此次開始，官校專科班及陸戰隊學生每年都會被挑選參加敦睦遠航<ref name="歷年敦睦支隊率隊長官及活動簡介"/>。此次由[[李崑材|李崑材]]少將率領<ref name="歷年敦睦支隊率隊長官及活動簡介"/>。
* 1991年，海軍艦隊訪問印尼、新加坡、韓國等3國<ref name="歷年敦睦支隊率隊長官及活動簡介"/>。中華民國第1艘國建國造快速油彈補給艦－[[武夷號油彈補給艦|武夷號油彈補給艦]]，首次接替玉台艦，參與敦睦遠航，是第1次有女性軍、士官隨敦睦支隊參與國內航訓<ref name="歷年敦睦支隊率隊長官及活動簡介"/>。此次由[[沈方祥|沈方祥]]少將率領<ref name="歷年敦睦支隊率隊長官及活動簡介"/>。
* 1995年，海軍艦隊5艘，包含第一次參加敦睦遠航任務的成功級艦與濟陽級艦，訪問印尼及新加坡2國，此次由[[譚明|譚明]]少將率領<ref name="歷年敦睦支隊率隊長官及活動簡介"/>。首次特別遴選8位優秀女性軍士官隨艦出國<ref name="歷年敦睦支隊率隊長官及活動簡介"/>。
* 1999年，海軍艦隊訪問諾魯、索羅門、新加坡及菲律賓等4國<ref name="歷年敦睦支隊率隊長官及活動簡介"/>。當時軍艦在菲律賓靠泊馬尼拉期間，中華人民共和國駐菲大使要求菲國不准中華民國海軍遠航支隊懸掛國旗，支隊長下令各艦全程懸掛中華民國國旗3日<ref name="歷年敦睦支隊率隊長官及活動簡介"/>。本次由[[陳永康|陳永康]]少將	率領<ref name="歷年敦睦支隊率隊長官及活動簡介"/>。
* 2000年，海軍艦隊由[[姜龍安|姜龍安]]少將率領，訪問馬紹爾、索羅門、新加坡及印尼等4國<ref name="歷年敦睦支隊率隊長官及活動簡介"/>。適逢索羅門發生內戰，奉命擔任撤僑工作，搭載中華民國同胞7人返國<ref name="歷年敦睦支隊率隊長官及活動簡介"/>。
* 2001年，海軍艦隊由[[高廣圻|高廣圻]]少將（後昇任[[國防部長|國防部長]]）率領，第1次訪問中美洲與東加勒比海地區等友邦國家<ref name="歷年敦睦支隊率隊長官及活動簡介"/>。這次是首次以海軍艦船懸掛中華民國國旗航行通過巴拿馬運河進入加勒比海，同時也打破中華民國海軍遠航訓練史上（35年以來）訪問國家最多（8個）、航行最遠（24,320浬）、時間最久（約3個月）及不靠岸航行最多天數（26天）的紀錄<ref name="歷年敦睦支隊率隊長官及活動簡介"/>。
* 2002年，海軍艦隊由[[劉俊英|劉俊英]]少將率領，訪問諾魯、吐瓦魯、索羅門、帛琉、印度尼西亞及新加坡等6國<ref name="歷年敦睦支隊率隊長官及活動簡介"/>。

==空軍==
{{See also|中華民國駐越軍事顧問團}}

美軍曾向[[中華民國空軍|中華民國空軍]]抽調48架[[F-5|F-5A戰機]]移交予[[南越空軍|南越空軍]]，並於[[台中|台中]][[清泉崗機場|清泉崗機場]]（當時號稱東南亞最大的機場，係為了支援[[B-52|B-52]]進行轟炸而構築）派駐[[F-4|F-4]]中隊填補<ref name="diic">{{cite web |url=http://www.diic.com.tw/mag/mag265/265-82.htm |title=被遺忘的越戰創傷──1967年台灣西貢使館被炸始末 |publisher=全球防衛誌 265 期 |date=2006 |author=高智陽 |deadurl=yes |archiveurl=https://web.archive.org/web/20121010022122/http://www.diic.com.tw/mag/mag265/265-82.htm |archivedate=2012-10-10 }}</ref>。

{{See also|大漠計畫}}

1974年，中華民國空軍派出兩名上校軍官前往約旦擔任軍事顧問和高級戰術教官。1975年前空軍總司令[[陳衣凡|陳衣凡]]任駐約旦大使，中華民國派遣一個軍事教練小組赴約旦，進行戰鬥機基本戰術訓練教學，同時地勤人員也跟進。
