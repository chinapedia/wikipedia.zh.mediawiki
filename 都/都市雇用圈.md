'''都市雇用圈'''（{{Jpn|j=都市雇用圏|hg=としこようけん}}，{{Lang-en|Urban Employment Area}}，UEA）是{{Link-ja|金本良嗣|金本良嗣}}以日本為研究對象，以[[僱用|僱用]]為基準的[[都市圈|都市圈]]的定義<ref>{{Cite journal |author1 = 金本良嗣 |author2 = 徳岡一幸 |title = 日本の都市圏設定基準 |url = http://www.csis.u-tokyo.ac.jp/UEA/UEADef.pdf |date = 2002 |publisher = 応用地域学会 |journal = 応用地域学研究 |number = 7 |pages = 1-15}}</ref>，在[[城市經濟學|城市經濟學]]中用作標準的[[都市|都市]]的定義<ref>{{Cite web|title=都市と地域経済：地域創成の鍵は何か|url=http://www.rieti.go.jp/jp/special/special_report/079.html|publisher=[[経済産業研究所|経済産業研究所]]|author=森知也|accessdate=2016-11-18 |language=ja}}</ref>。

== 中心都市與郊外的定義 ==
=== 中心都市的定義 ===
滿足以下任意條件的[[市町村|市町村]]為中心都市。存在多個的情況下，以其集合為中心。
# [[人口集中地区|DID]]（人口集中地区）人口在1万以上的市町村，且不是其他都市的[[郊外|郊外]]。
# 滿足郊外市町村的条件，但
#: a. 從業常住人口比在1以上，且
#: b. DID人口在中心市町村的3分之1以上，或10万以上。
# 部分区滿足上述条件的[[政令指定都市|政令指定都市]]。

=== 郊外的定義 ===
往中心都市的通勤率
# 10%以上者為（1次）郊外市町村，
# 往郊外市町村的通勤率超過10%，且不存在超過其通勤率的其他市町村的情況下，該市町村為2次以下的郊外市町村。但是，
#:a. 相互之間通勤率在10%以上的市町村對，通勤率高的一方為低的一方的郊外。
#:b. 中心都市由多個市町村構成的情況下，往這些市町村全体的通勤率在10%以上的市町村為其郊外。
#:c. 存在2個以上通勤率超過10%的中心都市時，通勤率最高者為中心都市的郊外。
#:d. 往中心都市及郊外市町村的通勤率分別超過10%的情況下，通勤率最大者為郊外。

* 2010年相比2005年減少的都市圈人口以{{Color|red|紅字}}顯示。

* 中心都市的DID人口合計在5万人以上的情況下稱為「大都市雇用圈」（Metropolitan Employment Area, MEA）（白），5万人以下的情況下稱為「小都市雇用圈」（Micropolitan Employment Area, McEA）（黄）。
