'''红线'''（{{Lang-ga|Líne Dhearg}}<ref>{{Cite web |url=http://www.luas.ie/ga/# |title=存档副本 |access-date=2017-02-06 |archive-url=https://web.archive.org/web/20170324224215/https://www.luas.ie/ga/# |archive-date=2017-03-24 |dead-url=yes }}</ref>）是[[都柏林轻轨|都柏林轻轨]]两条线路中的一条。红线轻轨在[[都柏林|都柏林]]的市中心以东西向行驶，然后穿过[[利菲河|利菲河]]，最后向西南行驶至[[塔拉_(爱尔兰)|塔拉]]。有分支线连接[[城西_(都柏林)|城西社区]]、[[萨格特|萨格特]]。红线开通于2004年9月26日。

红线于2001年3月开工建设，由[[意大利|意大利]]安萨尔多工程公司（Ansaldo）和[[澳大利亚|澳大利亚]]MVM工程公司承建<ref>[http://www.rte.ie/news/2001/0308/luas-business.html] RTÉ News:O'''Rourke awards £196m LUAS contract</ref>，由[[爱尔兰|爱尔兰]]Transdev公司（前称Connex）运营<ref name="RPA_Luas_Homepage">{{cite web |url=http://www.rpa.ie/luas |title=Luas home page |publisher=Railway Procurement Agency |accessdate=2008-04-04 |archiveurl=https://web.archive.org/web/20080312173217/http://www.rpa.ie/luas |archivedate=2008-03-12 |deadurl=yes }}</ref>。
