'''都紀女加'''（{{jpn|j=都紀女加|hg=つきめか|rm=Tsukimeka}}，{{bd|？||？||catIdx=Tsukimeka}}），又稱'''都紀女加王'''，是《[[先代舊事本紀|先代舊事本紀]]》記載的[[日本|日本]][[古墳時代|古墳時代]]的{{Link-ja|王 (日本)|王 (皇族)|王}}，第15代[[應神天皇|應神天皇]]的曾孫，筑志米多[[國造|國造]]之祖。《[[日本書紀|日本書紀]]》和《[[古事記|古事記]]》均沒有其相關記載。

== 系譜 ==
按《[[先代舊事本紀|先代舊事本紀]]》中的國造本紀記載，都紀女加是第15代[[應神天皇|應神天皇]]之子稚沼毛二俁命（《[[日本書紀|日本書紀]]》稱為稚野毛二派皇子，《[[古事記|古事記]]》稱為若沼毛二俁王）之孫，換言之是應神天皇的曾孫{{Sfn|都紀女加王墓（平凡社）|1980年}}。

== 墓 ==
[[File:Uenobyuuzuka-kofun_Tsukimekaou-bo_baichou.JPG|thumb]]模型]]
都紀女加的墓地是由[[宮內廳|宮內廳]]指定，位於[[佐賀縣|佐賀縣]][[三養基郡|三養基郡]][[上峰町|上峰町]]坊所的'''都紀女加王墓'''{{Sfn|都紀女加王墓（平凡社）|1980年}}，[[考古學|考古學]]上名為「上之謬塚古墳」（{{lang|ja|上のびゅう塚古墳}}）<ref>「上のびゅう」是地名。</ref>。

古墳的結構是屬於{{Link-ja|前方後圓墳|前方後円墳}}，前方部份朝向西面{{Sfn|都紀女加王墓（平凡社）|1980年}}。[[墳丘墓|墳丘]]長達49米，估計是建於5世紀，但是尚未發現埋葬部份{{Sfn|都紀女加王墓（平凡社）|1980年}}。墳丘表面上有[[埴輪|埴輪]]和{{Link-ja|葺石|葺石}}，墳丘四周則由[[壕溝|壕溝]]包圍{{Sfn|都紀女加王墓（平凡社）|1980年}}。1943年，墓地獲[[宮內省|宮內省]]（現[[宮內廳|宮內廳]]）指定為都紀女加王的墓地{{Sfn|都紀女加王墓（平凡社）|1980年}}。

墓地一帶的[[丘陵|丘陵]]曾經被稱為目達原古墳群，其中包括大塚古墳、古稻荷塚古墳、稻荷塚古墳、塚山古墳、瓢簞塚古墳、無名塚古墳和橫田南古墳等{{Sfn|目達原古墳群（平凡社）|1980年}}，但是在1942年至1943年期間由於興建目達原飛行場（{{Link-ja|目達原駐屯地|目達原駐屯地}}）而被破壞，改為葬於上之謬塚古墳北面的墳丘模型{{Sfn|目達原古墳群（平凡社）|1980年}}{{Sfn|目達原古墳群（国史）}}，而古墳群的近處則是[[吉野里遺跡|吉野里遺跡]]。

== 後裔國造 ==
按《[[先代舊事本紀|先代舊事本紀]]》中的國造本紀記載，都紀女加有以下的[[國造|國造]]後裔。
* 竺志米多國造
*: 志賀高穴穗朝（第13代[[成務天皇|成務天皇]]）在位期間，{{Link-ja|息長氏|息長氏|息長}}{{Link-ja|真人 (日本)|真人|公}}同祖的稚沼毛二俁命之孫都紀女加被任命為國造。位處於後來的[[肥前國|肥前國]][[三根郡|三根郡]]米多鄉（現[[佐賀縣|佐賀縣]][[上峰町|上峰町]]一帶）或[[筑前國|筑前國]][[夜須郡|夜須郡]]馬田鄉以及[[下座郡|下坐郡]]馬田鄉附近（現[[福岡縣|福岡縣]][[朝倉市|朝倉市]]一帶）<ref name="国造">『国造制の研究 -史料編・論考編-』（八木書店、2013年）pp. 276-277。</ref>。由於都紀女加是[[安康天皇|安康天皇]]和[[雄略天皇|雄略天皇]]的表兄弟，因此有些意見認為「志賀高穴穗朝」可能是指和風諡號為穴穗天皇的安康天皇{{Sfn|都紀女加王墓（平凡社）|1980年}}。另外，按《古事記》若野毛二俁王（稚野毛二派皇子）之子[[大大迹王|意富富杼王]]為「筑紫之末多君」等的祖先<ref name="国造"/>。
