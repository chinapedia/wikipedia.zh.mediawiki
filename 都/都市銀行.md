'''都市銀行'''（{{jpn|j=都市銀行|hg=としぎんこう}}）是指[[日本|日本]]的[[普通銀行|普通銀行]]中，總行設於[[東京都區部|東京]]、[[大阪市|大阪]]等大都市、廣域進行業務的[[銀行|銀行]]。簡稱'''都銀'''（{{jpn|j=都銀|hg=とぎん}}）。此定義雖無法律依據，也無明確定義，但現在一般來說共有5間銀行（定義上為4間銀行）。

== 歷史==
都市銀行是1968年10月開始的[[財務大臣|大藏大臣]]諮詢機關「金融制度調査會」第1分科會於「普通銀行諸問題」的審議中提出「普通銀行中'''以6大都市或符合的其他都市為總部'''，在全國或數地擁有'''廣域營業基盤'''」的定義<ref>マーケティング・エクセレンス著『図解30分で「銀行のしくみ」がすっきりわかる本』38頁、ISBN 978-4-7980-2565-0</ref>。

當時，都市銀行共有{{link-ja|第一銀行 (日本)|第一銀行|第一銀行}}、[[三井銀行|三井銀行]]、{{link-ja|富士銀行|富士銀行}}、[[三菱銀行|三菱銀行]]、{{link-ja|協和銀行|協和銀行}}、{{link-ja|日本勸業銀行|日本勧業銀行}}、[[三和銀行|三和銀行]]、[[住友銀行|住友銀行]]、{{link-ja|大和銀行|大和銀行}}、[[東海銀行|東海銀行]]、[[北海道拓殖銀行|北海道拓殖銀行]]、{{link-ja|神戶銀行|神戸銀行}}、{{link-ja|東京銀行|東京銀行}}等13行（依{{link-ja|統一金融機關代碼|統一金融機関コード}}排序）。1968年，日本相互銀行從{{link-ja|相互銀行|相互銀行}}轉換為普通銀行，並更名為{{link-ja|太陽銀行|太陽銀行}}成為都市銀行一員。1969年，{{link-ja|埼玉銀行|埼玉銀行}}從[[地方銀行|地方銀行]]轉換為都市銀行。1971年，第一銀行與日本勸業銀行合併為{{link-ja|第一勸業銀行|第一勧業銀行}}，1973年神戶銀行與太陽銀行合併成立{{link-ja|太陽神戶銀行|太陽神戸銀行}}，之後長時間維持13行體制，但在北海道拓殖銀行破產與都市銀行相互合併重整後，2013年起形成現在的5行體制。

現在的都市銀行皆是擁有大量存款資金的[[巨型銀行|巨型銀行]]與其系列銀行、或能與其匹敵的銀行所組成。[[郵貯銀行|郵貯銀行]]由於其國營的郵政存款特殊性，不屬於都市銀行。另外，{{link-ja|商工組合中央金庫|商工組合中央金庫}}也不屬於都市銀行。

== 曖昧的基準 ==
[[埼玉里索那銀行|埼玉里索那銀行]]由於其前身{{link-ja|埼玉銀行|埼玉銀行}}而列為「都市銀行」之一，但卻不符「全國展開業務」的基準<ref>舊埼玉銀行沒有大阪、札幌等分行。</ref>。相對於僅大企業做為顧客的[[瑞穗實業銀行|瑞穗實業銀行]]（當時）<ref>登記上是舊{{link-ja|富士銀行|富士銀行}}更名的銀行，但總行所在地與業務內容則是繼承身為長期信用銀行的{{link-ja|日本興業銀行|日本興業銀行}}。</ref>被列為都市銀行，由[[長期信用銀行|長期信用銀行]]轉換而來的[[新生銀行|新生銀行]]與{{link-ja|青空銀行|あおぞら銀行}}雖符合「總行位於大都市，請在全國展開業務」的條件但卻未在都市銀行之列。

從全國銀行協會等的發行物、[[日本銀行|日本銀行]]的定義<ref>[http://www.boj.or.jp/statistics/outline/note/financial.htm/ 用語の定義]、日本銀行</ref>、[[財務省_(日本)|財務省]]的分類<ref>[http://www.mof.go.jp/jgbs/topics/bond/bidders/index.htm 国債に係る入札参加者一覧]、財務省</ref>來看，埼玉里索那銀行與瑞穗實業銀行（當時）都被列入「都市銀行」。而在[[金融廳|金融廳]]網站公開的「執照、許可、登錄等業者一覧<ref>[http://www.fsa.go.jp/menkyo/menkyoj/ginkou.pdf 銀行免許一覧（都市銀行・信託銀行・その他）]、金融庁</ref>」中，瑞穗實業銀行（當時）列為金融廳所管之「都市銀行」，[[新生銀行|新生銀行]]及{{link-ja|青空銀行|あおぞら銀行}}列為金融廳所管之「其他」類別，埼玉里索那銀行則列為[[關東財務局|關東財務局]]所管之「其他」類別。

== 行政上的都市銀行 ==
在金融行政的統計資料等，都市銀行同等於普通銀行內的本廳直轄銀行。

{{link-ja|銀行法|銀行法}}第59條第2項，金融廳長官之部分權限委任給財務局長（含財務支局長）。這是由於金融廳本身沒有地方支分部局，而成立當時的母體財務省則是委任給地方支分部局的情況而訂立。依據此項規定，銀行法施行令第17條之2第1項列舉各別權限（銀行業執照、銀行合併、經營基本事項以外），同條第4項再補充由金融廳長官指定的業務不適用此項規定。換言之，金融廳長官不委任給財務局長的金融機關由金融廳長官行使其權限。

依此規定為基礎再制定「不適用銀行法施行令第17條之2第1項至第3項規定之金融廳長官權限等事項」（2002年（平成14年）金融廳告示第35號）。其中，[[瑞穗銀行|瑞穗銀行]]、[[瑞穗實業銀行|瑞穗實業銀行]]（當時）、[[三井住友銀行|三井住友銀行]]、[[三菱東京UFJ銀行|三菱東京UFJ銀行]]、[[里索那銀行|里索那銀行]]5行為「本廳直轄銀行」，同於行政上的「都市銀行」。而從長期信用銀行轉換而來的2行（新生銀行、青空銀行）、{{link-ja|新形態銀行|新たな形態の銀行}}與信託兼營銀行雖然也是本廳直轄銀行，但不屬於都市銀行。另外，埼玉里索那銀行因不是本廳直轄銀行，在金融廳資料中不屬於都市銀行，又由於未加入{{link-ja|全國地方銀行協會|全国地方銀行協会}}與{{link-ja|第二地方銀行協會|第二地方銀行協会}}，因此不屬於地方銀行、[[第二地方銀行|第二地方銀行]]，而是列入關東財務局所管之「其他」類別。

# 埼玉銀行成立當時為地方銀行。1969年轉換為都市銀行。
# 背景灰色的是{{link-ja|特殊銀行 (日本金融史)|特殊銀行 (日本金融史)|特殊銀行}}或[[長期信用銀行|長期信用銀行]]、[[外匯|外匯]]專業銀行、第二地方銀行等。
# 住友銀行在1986年與{{link-ja|平和相互銀行|平和相互銀行}}合併，三井住友銀行在2003年與{{link-ja|若潮銀行|わかしお銀行}}合併。
# 舊大和、朝日的埼玉縣內分行由埼玉里索那銀行繼承，其他分行由里索那銀行繼承。里索那銀行於2005年合併{{link-ja|奈良銀行|奈良銀行}}。
# 新生銀行是都銀部門轉讓設立的長信銀後繼銀行，在圖表中不屬於都市銀行。
