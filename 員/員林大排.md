'''員林大排'''，又稱'''員林大排水溝'''、'''員林大排水道'''，在歷史上有不同稱呼如'''[[臺灣溝|臺灣溝]]'''、'''[[鹿港溪|鹿港溪]]'''<ref>{{cite web|title=濁水溪平原的地勢分析與地形變遷|author=[[張瑞津|張瑞津]]|year=1985|url=http://www1.geo.ntnu.edu.tw/~shensm/Course/CourseWork/TaiGeom_Stu90/19%E6%BF%81%E6%B0%B4%E6%BA%AA%28%E6%9B%B4%E6%96%B0%E7%89%88%29/chp.htm|page=213}}</ref>、'''[[大武郡溪|大武郡溪]]'''（清代）<ref>{{cite book|title=〈鹿港古風貌之硏究〉|author= 辜偉甫, 漢寶德|publisher= 彰化縣鹿港文物維護及地方發展促進委員會|year=1978|quote= 鹿港溪:原爲大武郡溪,源於大武郡山,西出馬芝遴,至鹿仔港入於海,今爲員林大排水溝。}}</ref>，為彰化縣八堡圳灌溉水利系統之一。<ref>{{cite web|title=員林大排水工程記|url=http://memory.ncl.edu.tw/tm_cgi/hypage.cgi?HYPAGE=document_ink_detail.hpg&subject_name=%E8%87%BA%E7%81%A3%E7%A2%91%E7%A2%A3%E6%8B%93%E7%89%87&subject_url=&project_id=twrb&dtd_id=12&xml_id=0000007457}}</ref>從[[臺灣日治時期|日治時代]]到[[臺灣戰後時期|中華民國接管臺灣]]後，各段經過加深、加寬和挖新水道取直整治，全長25.4公里，上游面寬53公尺，下游面寬102公尺主幹流經[[社頭鄉|社頭鄉]]、[[員林市|員林市]]、[[埔心鄉|埔心鄉]]、[[埔鹽鄉|埔鹽鄉]]、[[福興鄉_(臺灣)|福興鄉]]、[[鹿港鎮|鹿港鎮]]等1市14鄉鎮，東界為[[八卦山|八卦山]]，北邊是[[八堡圳|八堡水利委員會第一圳]]{{efn|註：原在員林大排南邊，在員林萬年附近穿越而過}}，南邊有八堡二圳與西溝，面積約1萬9229公頃，排水與灌溉面積達2萬2304公頃{{r|陳}}。員林大排在鹿港下游稱為'''福鹿溪'''，原河道鹿港人俗稱舊港溝(2018年彰化縣政府開始鹿港溪重現動工)，埔鹽段又稱'''台灣溝'''，埔心段又稱'''柳溝'''（'''柳仔溝'''）{{r|埔}}。

== 沿革 ==
員林大排的建設規劃約始於日大正十一年（1922年），用來排除員林柴頭井上游的山洪與社頭、田中、員林一帶的淹水，並降低沿線地下水位，以改善農田且增加耕種面積{{r|陳}}。當時的計畫中員林大排長度為25.45公里，排水量為每秒336.7立方公尺，渠寬80.3公尺，深3.3公尺，受益面積7萬5千公頃{{r|陳}}。開闢計畫於昭和七年（1932年）再次設計，預計十年後完成，之後工程於昭和十六年（1941年）12月8日動工，但因為戰爭等因素導致物資人力缺乏，工程斷斷續續，到日本時代結束時僅完成13多公里，另有5公里只完成一部分{{r|陳}}，而此一時期完成的水段大致是鹿港海岸到埔心鄉油車村二抱巷口的部分{{r|埔}}。

[[二次大戰|二次大戰]]之後，[[臺灣省水利局|臺灣省水利局]]在員林設置工程處，打算依照日本的計畫繼續施工，但[[中華民國政府|中華民國政府]]在治臺初期預算不足，且有部分水利地未順利收購，工程依然屢次中斷，後來在民國四十年（1951年）[[臺灣省政府|臺灣省政府]]編列預算且獲得[[美援|美援]]與[[農復會|農復會]]的貸款才使得工程能順利進行，於民國四十四年（1955年）宣告完工{{r|陳}}，修築到社頭鄉鴨母湳一帶{{r|埔}}。完工之後，立有「員林大排水告成紀念碑」，由當時的[[臺灣省政府建設廳水利局|臺灣省政府建設廳水利局]]局長章錫授敬撰，[[臺灣省八堡水利委員會|臺灣省八堡水利委員會]]主任委員李功垂敬立{{r|陳}}。

[[台76線|台76線]]自[[福爾摩沙高速公路|國道三號]]中興系統交流道分出，經[[八卦山隧道|八卦山隧道]]後沿員林大排兩側，以高架形式與員林大排的平面道路共構，在埔鹽鄉與國道一號交匯，續往西北通車至埔鹽鄉與福興鄉交界處之台19線彰水路埔鹽交流道為止。往西終點至芳苑鄉[[漢寶|漢寶]]路段，因此路線引起地方爭議而停擺10年未動工至今，目前暫時通行員林大排平面道路，可通往鹿港鎮，並連接[[台61線|台61線]]西濱快速道路。

== 柳橋晚眺 ==
柳橋晚眺是1961年選出的彰化縣八景之一，柳橋位於埔心鄉的員鹿路上，柳橋之名是舊有橋梁在1952年4月改建後所取的名字，附近一帶在1954年左右計劃成風景區，遂由[[埔心鄉公所|埔心鄉公所]]在柳溝西側蓋八卦亭，並建碼頭、設置遊艇{{r|埔}}。然而到了1971年時因為員林大排上游受工農業廢水汙染，並有[[布袋蓮|布袋蓮]]蔓延，使得柳溝不再適合泛舟，鄉公所便結束此一公共造產事業，而柳橋晚眺一景也因而不存{{r|埔}}。

== 龍舟競賽 ==
=== 鹿港慶端陽活動 ===
1978年起每年於端午節舉辦「鹿港慶端陽」<ref>[http://tourism.chcg.gov.tw/tc/festivals.aspx?classifyID=21 2015鹿港慶端陽]</ref>活動，其中重頭戲既是在福鹿溪舉辦龍舟競賽及跨河拔河競賽（全台唯一舉辦過），而於[[老三台|老三台]]時期每年皆有[[實況轉播|實況轉播]]；但於道路擴建時將賽事移至吉安水道舉辦。於2015年「鹿港慶端陽」時端午龍舟賽事移回福鹿溪舉辦。
