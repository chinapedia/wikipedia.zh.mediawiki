-{zh-hans:'''隔空播放'''（{{lang-en|AirPlay}}）;zh-hant:'''AirPlay'''}-是[[苹果公司|苹果公司]]的在[[iOS|iOS]]4.2及OS X Mountain Lion中加入的一种播放技术，可以将[[iOS|iOS]]和[[OS_X_Mountain_Lion|OS X Mountain Lion]]或更新版上的檔案（包括视频、相片和镜像）传送到支持AirPlay的设备（如：[[HomePod|HomePod]]和[[Apple_TV|Apple TV]]）。

使用Airplay需要[[iOS|iOS]]4.2版本的设备或Mac电脑上的[[iTunes|iTunes]]10.1以上版本。视频镜像则需要iOS 5及OS X Mountain Lion。<ref>[http://www.weiphone.com/apple/news/2010-11-23/AirPlay_FAQs_226922.shtml AirPlay常见问题解答]</ref>

在2017年11月初的[[iOS_11|iOS 11]]更新，這功能（僅[[簡體中文|簡體中文]]）漢化為-{「隔空播放」}-http://www.sohu.com/a/201753489_114760</ref><ref>[https://www.ithome.com/html/iphone/332305.htm 功能变中文名！苹果iOS11.2测试版更加本土化 - 苹果,iOS11.2 - IT之家]</ref>，[[繁體中文|繁體中文]]地區依然不變。<ref>https://www.hksilicon.com/articles/1464410</ref>{{參見|iOS 11#中文化}}

== 简介 ==
AirPlay技术最初是在2010年11月12日发布的iTunes 10.1中引入，可以将iTunes中的所有音乐都无线同步到揚聲器，並將[[iOS|iOS]]裝置的畫面同步至[[Apple_TV|Apple TV]]。

== 使用方法 ==
当用户的设备（[[Mac|Mac]], [[PC|PC]], [[iPod_touch|iPod touch]], [[iPhone|iPhone]], [[iPad|iPad]]）与支持AirPlay技术的音箱在同一无线局域网（[[WIFI|WIFI]]）内的时候，用户在[[Mac|Mac]]或[[PC|PC]]上打开[[iTunes|iTunes]] 10或更高版本后，就会在iTunes音量条旁边（iTunes 10在窗口的右下角）看到一个AirPlay按钮。点击该AirPlay按钮，然后选择需要同步到的设备，就可以选择将音乐或视频无线同步到设备进行播放。

而[[iOS_11|iOS 11]]用户则可以在控制中心及支持AirPlay功能的第三方软件界面找到AirPlay按钮，按下去即可。
