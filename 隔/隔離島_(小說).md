《'''隔離島'''》（{{lang-en|'''''Shutter Island'''''}}）是[[丹尼斯·勒翰|丹尼斯·勒翰]]所寫的暢銷小說，2003年4月由[[哈珀科林斯出版集團|哈珀科林斯出版集團]]出版。[[不赦島|改編電影]]於2010年2月上映。勒翰表示他寫小說是向[[哥德小說|哥德]]設定、[[B級片|B級片]]和{{link-en|低俗雜誌|Pulp magazine|低俗}}致敬。他形容該小說是[[勃朗特三姊妹|勃朗特三姊妹]]與1956年電影《[[天外魔花|天外魔花]]》的混合作品。他組織該書比他上一本《{{link-en|神祕河流 (小說)|Mystic River (novel)|神祕河流}}》較為緊湊。<ref>{{Cite web |url = http://www.powells.com/post/interviews/dennis-lehane-meets-the-bronte-sisters |title = Dennis Lehane Meets the Bronte Sisters |author = Dave Weich |date = 2006-10-10 |publisher = Powell's Books |language = en}}</ref>

[[波士頓灣|波士頓灣]]{{link-en|長島 (麻省)|Long Island (Massachusetts)|長島}}上的醫院和場地啟發了勒翰創作小說內的醫院和島嶼。勒翰曾在小孩時期於{{link-en|1978年美國東北部暴風雪|Northeastern United States blizzard of 1978|1978年暴風雪}}期間與他叔叔和家人參觀該處。<ref>{{Cite web |url = http://www.patriotledger.com/x1650245399/Real-local-flavor-on-display-in-Shutter-Island |title = Real local flavor on display in 'Shutter Island' |author = Ed Symkus |date = 2010-02-19 |publisher = The Patriot Ledger |language = en}}</ref>

== 改編作品 ==
=== 電影 ===
{{main|不赦島}}
小說被改編成電影，由[[馬田·史高西斯|馬田·史高西斯]]執導，[[里安納度·狄卡比奧|里安納度·狄卡比奧]]飾演Teddy Daniels，[[麥克·雷法路|麥克·雷法路]]飾演Chuck Aule，[[賓·京士利|賓·京士利]]飾演Dr. Cawley，以及[[麥士·馮·西度|麥士·馮·西度]]Dr. Naehring。

=== 有聲書 ===
小說的[[哈珀科林斯出版集團|哈珀科林斯]][[有聲書|有聲書]]版由[[大衛·史崔森|大衛·史崔森]]朗讀。

小說的{{link-en|Audible Inc.|Audible Inc.|Audible}}有聲書版由{{link-en|Tom Stechschulte|Tom Stechschulte}}朗讀。
