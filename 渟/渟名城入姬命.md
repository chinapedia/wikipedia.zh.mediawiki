'''渟名城入姬命'''（{{jpn|j=渟名城入姫命|hg=ぬなきいりびめのみこと|rm=Nunakiiribimenomikoto}}{{Sfn|渟名城入姫命（古代氏族）|2010年}}，{{bd|？||？||catIdx=Nunakiiribimenomikoto}}）是[[日本|日本]]古代[[記紀|記紀]]記載的[[皇女|皇女]]，《[[日本書紀|日本書紀]]》稱其為'''渟名城入姬命'''、'''淳名城入姬命'''或'''渟名城稚姬命'''，《[[古事記|古事記]]》稱為'''沼名木之入日売命'''，《古事記》並沒有記載其相關事蹟。第10代[[崇神天皇|崇神天皇]]的皇女，聞名於與[[奈良縣|奈良縣]][[天理市|天理市]]的[[大和神社|大和神社]]祭神[[倭大國魂神|倭大國魂神]]相關的事蹟。

== 系譜 ==
{{smaller|名稱以《日本書紀》的稱呼為先，括號內的是《古事記》等的稱呼}}

按《[[日本書紀|日本書紀]]》和《[[古事記|古事記]]》記載，渟名城入姬命是第10代[[崇神天皇|崇神天皇]]與[[妃|妃]]尾張大海媛（意富阿麻比賣）之女。

其中，《日本書紀》指出渟名城入姬命的同母兄是八坂入彥命（八坂之入日子命），同母妹是[[十市瓊入媛命|十市瓊入媛命]]（十市之入日賣命）；《古事記》則指其兄長為《日本書紀》未有記載的大入杵命。

== 紀錄 ==
按《日本書紀》崇神天皇6年條記載，由於當時百姓流離失所，加上發生叛亂等，國內情勢不穩，對此崇神天皇希望能夠設置地方供奉[[天照大神|天照大神]]和[[倭大國魂神|倭大國魂神]]，於是便讓[[豐鍬入姬命|豐鍬入姬命]]在倭{{Link-ja|笠縫邑|笠縫邑}}供奉天照大神，渟名城入姬命則負責供奉倭大國魂神。然而，由於渟名城入姬命脫髮，而且太過瘦弱，因此無法拜祭倭大國魂神{{Sfn|渟名城入姫命（古代氏族）|2010年}}。

按《日本書紀》[[垂仁天皇|垂仁天皇]]25年3月條，大倭大神以「一雲」來發出供奉自己的[[神諭|神諭]]，中臣探湯主以占卜決定由「渟名城稚姬命」負責。天皇隨即命渟名城稚姬前往以神地穴磯邑（現奈良縣[[櫻井市|櫻井市]]穴師）之名，在大市（奈良縣櫻井市芝附近）的長岡岬供奉，但是渟名城稚姬太過瘦弱而無法拜祭，因此便改由大倭[[姓_(日本政治結構)|直]]之祖{{Link-ja|市磯長尾市|市磯長尾市|長尾市宿禰}}代為供奉。這裡提及的「渟名城稚姬命」可能與渟名城入姬命是同一人{{Sfn|渟名城入姫命（古代氏族）|2010年}}。

== 考証 ==
有關渟名城入姬命在《日本書紀》上的事蹟，均是[[奈良縣|奈良縣]][[天理市|天理市]][[大和神社|大和神社]]的起源。其中，有些意見認為崇神天皇在血統上不能供奉[[倭國|倭國]]的[[國魂|國魂]]，因此便讓{{Link-ja|倭國造|倭国造}}一族大倭直負責{{Sfn|渟名城入姫命（古代氏族）|2010年}}。
