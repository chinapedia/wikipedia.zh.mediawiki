'''屏東賽嘉滑翔飛行場'''，位於[[臺灣|臺灣]][[屏東縣|屏東縣]][[三地門鄉|三地門鄉]]賽嘉村賽嘉航空公園（佔地2600公頃）內。

== 起源 ==
於西元1980年(民國70年)，由屏東縣高樹鄉泰山村潘氏兄弟-潘豐慶、潘豐義及幾名飛行愛好者-余繼高等人開發此場地，也因潘氏兄弟地緣的關係，跟賽嘉村林姓地主租借此山坡地，來開發無動力飛行場，並開時規畫周邊道路，申請無動力飛行空域，電線地下化等周邊設施..。初期都是以滑翔翼為主，起飛場上山的道路也難行，也無固定的降落場地。

== 地理位置 ==
* 海拔高度400公尺，飛行落差卻高達330公尺的飛行起飛場，東部山勢陡峻，上升氣流極佳為天然飛行地區，西部平坦河川地，無障礙物，因受中央山脈的屏障，更不易受季節氣候的氣流影響，氣流穩定，因此一年四季均適合各項飛行活動，包含飛行傘(滑翔傘)、滑翔翼、等等..。
* 因地形優勢，尤其冬天氣流為台灣最佳飛行場地，故每年吸引全世界各地飛行傘好手朝聖飛行。
== 歷史紀錄 ==
1975年                            -陳進川開發滑翔翼，與飛行同好一同開始進行飛行活動。

1985年2月                      -民生報1985年2月5日報導，於1985年2月2日，打破飛行傘越野紀錄，從賽嘉飛行場起飛，王容南飛行36公里、郝鉅32公里。

1987年                            -陳進川、簡萬全，向空降部隊借了一頂高空傘，在屏東縣三地鄉的賽嘉航空公園，開始研究飛行。

2002年1月                       -民生報報導，徐列東以無動力飛行傘，於2001年12月31日，從賽嘉起飛，花費2個小時左右時間，飛越中央山脈。

2010年12月18日至12月19日        -賽嘉飛行傘全國錦標賽

2012年2月24日至28日     -「2012年PWCA世界杯飛行傘-台灣站」會前賽，於賽嘉飛行傘場舉辦賽事。

2017年3月                       -由蘇沛倫取得台灣第一張，合法無動力飛行傘場場證。

2017年12月30日至1月2日-舉辦2017 飛行傘全國排名賽暨國際邀請賽。

2017年12月29日至1月1日-於此場地舉辦，2018 飛行傘國際邀請賽暨全國排名賽、賽嘉飛行嘉年華。

2018年11月17日至11月18日-TAIWAN PG XC OPEN - SAI JIA 飛行傘國際邀請賽暨全國排名賽

2018年12月28日至1月1日  -TAIWAN PG XC OPEN台灣飛行傘國際邀請賽暨全國排名賽

== 氣候 ==
一年四季都可以飛行，夏天上午天氣適合飛行，下午會有雷陣雨，冬天10月-3月左右，氣流最穩定，不下雨，是最佳的飛行季節。

== 飛行器的種類 ==
飛行傘(滑翔傘/paragliding)--單人飛行傘(滑翔傘)、雙人飛行傘(無經驗，需要教練操作帶飛行)

滑翔翼(hang gliding)--全台灣唯一看得的到滑翔翼飛行的剩下屏東賽嘉飛行傘



== 飛行者等級 ==
初階到高階都適合飛行

適合初學者-(場地地理環境安全，起飛場和降落場很近，起飛場180度，前方45度斜坡，降落場及起飛場場地都很大。)

適合中高級以上-(適合挑戰越野飛行，飛行路線可沿185線公路飛行，往六龜方向，公路兩側都可以找空地降落，回收人員時方便。) 

==賽嘉飛行嘉年華==
賽嘉飛行嘉年華是賽嘉滑翔飛行場一年一度的例行性的活動，也有農業和原住民特展，原本是年年都有的活動，2012年賽嘉飛行嘉年華之後，因賽嘉滑翔飛行場未經合法程序及執照使用該場地，屬於非法場地，此活動暫停5年，屏東縣政府也在飛行場起飛點及降落點貼上公告禁止飛行傘飛行，但飛行場仍有少數人在飛行，於2017年由屏東縣政府發布無動力飛行照才合法化之後，終於在2017年12月30日～2018年1月1日復辦本活動，引起大量遊客前來觀賽！

== 交通資訊 ==

=== 大眾運輸 ===
台鐵

高雄搭乘台鐵至屏東火車站下車→轉搭屏東客運(往高樹廣興)→下廣興站可步行即可達賽嘉航空運動公園

=== 自行開車 ===
# 路線 一 :  國道3號長治交流道下→接24號省道(往三地門鄉方向)→經三地門大橋→接185縣道(經廣興)→達賽嘉村→(沿指標行駛)直行即可達賽嘉飛行場起飛區
# 路線二 : 國道3號九如交流道下→接屏24線省道(往往屏北高中方向)→經塩埔鄉往泰山→接185縣道(經廣興)→達賽嘉村→(沿指標行駛)直行即可達賽嘉飛行場起飛區
