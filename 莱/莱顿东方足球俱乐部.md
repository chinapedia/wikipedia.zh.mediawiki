'''立頓奧連特足球會'''（{{lang-en|'''Leyton Orient F.C.'''}}，{{pron-en|ˌleɪtən ˈɔːriənt}}）是位於[[英格蘭|英格蘭]][[倫敦|倫敦]]東部[[沃爾瑟姆福里斯特區|沃爾瑟姆福里斯特區]][[萊頓_(英國)|萊頓]]的足球俱乐部，於2006/07年球季升上[[英格蘭足球甲級聯賽|英格蘭甲級聯賽]]作賽，球隊常被稱為「'''-{zh-hans:东方; zh-hant:奧連特;}-'''」（'''{{lang|en|Orient}}'''）或「'''{{lang|en|The O's}}'''」。

球隊主場「競賽室球場」是以前主席[[Barry_Hearn|-{巴里·赫恩}-]]（{{lang|en|Barry Hearn}}）的體育推廣公司「競賽室運動」（{{lang|en|Matchroom Sport}}）為名。赫恩於1995年僅以5[[英鎊|英鎊]]的代價從已故的前主席托尼·伍德（{{lang|en|Tony Wood, OBE}}）手上取得球隊控制權，伍德因[[卢旺达|卢旺达]]內戰而失去其種植[[咖啡|咖啡]]的生意。[[英國第四台|英國第四台]]曾拍攝一輯名為「奧連特：五鎊足球會」（''{{lang|en|Orient: Club for a Fiver}}''）的[[紀錄片|紀錄片]]報導當時接近倒閉的球隊情況。

== 主場球場 ==
[[File:Brisbane_Road.jpg|thumb]]
'''布-{里}-斯班路球場'''（'''Brisbane Road'''）獲冠名贊助稱為「競賽室運動場」（''Matchroom Stadium''），本名「奧斯邦尼路球場」（''Osborne Road''），是位於[[英格蘭|英格蘭]][[倫敦|倫敦]]東部[[萊頓_(英國)|萊頓]]的[[足球|足球]][[運動場|運動場]]，自1937年以來已是立頓奧連特足球會的主場球場。它亦是[[托特纳姆热刺足球俱乐部|熱剌]]預備隊的駐紮地，為全坐席可容9,271名觀眾。

球場由四座獨立看台組成，分別為「西看台」（West Stand），現時的主看台，設有球會辦公室、精品店、票房、餐飲設施和球員更衣室；「東看台」（East Stand）是從前的主看台，亦是最老舊的看台；「家庭北看台」（North Family Stand）於2007年落成，是最新的一座看台；「湯美·莊士東看台」（Tommy Johnston Stand）建於1999年，前稱「南看台」（South Stand），於2008年宣佈以球會歷來的神射手的名字命名。

== 為國效勞 ==
1914/15年球季，當時仍稱為-{zh-hans:克莱普顿东方; zh-hant:克萊普頓奧連特;}-（Clapton Orient）的職球員共41人，全數應徵為國上戰場，是[[英格蘭足球聯賽|聯賽]]中首支響應保衛國土的球隊。

全部隊員被派往[[法國|法國]]北部的戰場，參與一場由1916年7月1日激戰到11月18日名為[[索姆河战役|索姆河战役]]的慘烈戰鬥，單是首天英軍就有約2萬人陣亡，接近4萬人受傷，整場戰役傷亡人數多達150萬人。

奧連特共有13名隊員在戰時受傷及3人陣亡，陣亡者包括士兵威廉·强纳斯（William Jonas）、乔治·史考特（George Scott）及士官長理察·麦费顿（Richard McFadden），其中强纳斯的屍體更未能尋回。
