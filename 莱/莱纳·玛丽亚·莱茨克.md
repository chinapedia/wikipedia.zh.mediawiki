'''李海纳'''（'''Rainer Maria Latzke'''，{{Bd|1950年|12月28日||}}）是一名[[德国人|德国]] [[艺术家|艺术家]]，他致力于 [[错视画|错视画]] 和 [[壁画|壁画]] 领域的创作和研究。 他现任教于 [[犹他州州立大学|犹他州州立大学]]， 并且创办了[[壁画学院|壁画学院]] <ref name="Institute of Frescography">{{cite web | url = http://www.frescography.com/| title = Institute of Frescography}}</ref>。 他还被聘为上海 [[复旦大学|复旦大学]][[视觉艺术学院|视觉艺术学院]]<ref name="Honorary Professor">{{cite web| url =  http://www.siva.edu.cn/renda/node6813/english/Photo_News/u1a1698908.html| title =  Honorary Professor of the Shanghai Institute of Visual Art| deadurl =  yes| archiveurl =  https://web.archive.org/web/20110707025349/http://www.siva.edu.cn/renda/node6813/english/Photo_News/u1a1698908.html| archivedate =  2011-07-07}}</ref>的荣誉教授以及北京德稻教育机构的艺术大师。<ref name="Beijing De Tao Masters Academy">{{cite web | url =  http://www.detaoma.com/pro.asp?Cls=657 | title =  北京德稻教育机构 | deadurl =  yes | archiveurl =  https://web.archive.org/web/20101122173148/http://detaoma.com/pro.asp?Cls=657 | archivedate =  2010-11-22 }} </ref>。李海纳还被俄罗斯艺术家工会列为在过去400年中(十八--二十一世纪)，世界上最优秀的艺术家之一。<ref name="10.000 world best artists">{{cite web| url =  http://painters.artunion.ru/best_engl.htm| title =  List of 10.000 world best artists of the Russian Federation Artists Trade Union| deadurl =  yes| archiveurl =  https://web.archive.org/web/20101031173414/http://painters.artunion.ru/best_engl.htm| archivedate =  2010-10-31}}</ref>

李海纳出生于1950年，在德国[[科隆市|科隆市]]附近长大，家里还有8个兄弟姊妹，父亲阿尔方斯是一名艺术教师，母亲丽莎也是一名艺术家。李海纳在[[德国|德国]][[美因茨大学|约翰内斯古滕贝格大学]]（简称美因茨大学）学习艺术、哲学和教育学；1972-1976年，李海纳在[[德国杜塞尔多夫艺术学院|德国杜塞尔多夫艺术学院]]继续深造艺术，师从[[约瑟夫`博伊斯|约瑟夫`博伊斯]]和[[格哈德·克里特|格哈德·克里特]]<ref name=" “Keimpis Poetry Magazine ” ">{{cite web
 |url         = http://kempis.nl/mag/kunstakademie-dusseldorf-rundgang-2009-1
 |title       = Article in Keimpis Poetry Magazine about the Academy of Fine Arts Duesseldorf ”
 |deadurl     = yes
 |archiveurl  = https://web.archive.org/web/20101230031328/http://kempis.nl/mag/kunstakademie-dusseldorf-rundgang-2009-1
 |archivedate = 2010-12-30
}}</ref>。
1974年，他同时获得了 [[教育学|教育学]]和[[哲学|哲学]]的最高学位；1976年，他获得了美术最高学位。之后，李海纳在家乡的艺术大学里授课。1980年，他开始游学 [[意大利|意大利]]，先后在 [[弗洛伦萨|弗洛伦萨]]和[[罗马|罗马]]学习[[文艺复兴时期|文艺复兴时期]]的绘画技巧和 [[壁画|壁画]]艺术。1981年，他和多丽丝·搏艾克结婚，并育有三个孩子，分别取名为雷内·马库斯（生于1981年），他娶了与英国艺术家保拉·哈蒙德为妻， 凯瑟琳·玛利亚（生于1983年） 莫里斯·阿玛迪厄斯（生于1985年）。凯瑟琳·玛利亚嫁给了美国人保罗·史密斯，并在2008年为莱茨克诞下一名外孙，取名崔斯坦·弗朗西斯。 
1982年起，李海纳的壁画事业开始突飞猛进。他与伦敦 [[哈罗德百货公司|哈罗德百货公司]]的合作开始受到 [[中东|中东]]皇室家庭等上流人士的青睐。一些大型公司也开始委托他进行壁画创制，比如，他就曾为 [[梅赛德斯奔驰|梅赛德斯奔驰]]汽车公司题为“世界车辆”的100周年庆祝展览设计绘制了壁画。他还为许多摇滚乐队进行创作，并为 [[蝎子乐队|蝎子乐队]]制作了电脑控制的电子壁画，题为“曼哈顿之夜”。1992年，李海纳入选福布斯杂志“明日之星——近十年具有杰出贡献的文化引领者”名单，被认为是壁画界的杰出艺术家。

1984年，李海纳购得位于 [[比利时|比利时]]凯特尼斯（Kettenis）的“泰尔城堡”（Chateau Thal），城堡建造于1760年，李海纳使用壁画修复了这座有38个房间的城堡。他在城堡内的工作室里向众多学生和学徒传授壁画创作技艺，引领了当代室内壁画设计的文艺复兴。

1995年，李海纳移居 [[蒙特卡洛|蒙特卡洛]]，并在1998年购得古老的“帕拉杜”别墅。别墅原先的主人是因发明[[立体声宽银幕电影|立体声宽银幕电影]] 而获得[[奥斯卡|奥斯卡]]奖的发明家[[亨利·克里蒂安|亨利·克里蒂安]]。他对别墅进行了翻修，在已有的湿壁画基础上加入了现代新型壁画。别墅被一座大型公园环绕，有由[[居斯塔夫1埃菲尔|居斯塔夫1埃菲尔]]建造的眺望台，可眺望地中海，在这座宁静的别墅中，李海纳出版了德语版和英语版的《梦想世界——用幻境画装饰房间》。

居住在别墅期间，他还继续开发研究壁画制作和壁画复制的新技术。早在1988年他就开始致力于这项技术的研究，并于2000年获得该项技术的专利。此外，他还涉足音乐，为歌曲作词作曲，他同法国南部工作室的音乐师以及 [[菲尔·帕默|菲尔·帕默]]合作，后者还和 [[罗杰·达尔特里|罗杰·达尔特里]],以及[[恐怖海峡|恐怖海峡]]乐队有过合作。
李海纳在<ref name="USU">{{cite web | url = http://www.art.usu.edu/department/faculty.php | title = Utah State University, Faculty & Staff | deadurl = yes | archiveurl = https://web.archive.org/web/20090402132317/http://www.art.usu.edu/department/faculty.php | archivedate = 2009-04-02 }}</ref>[[犹他州大学|犹他州大学]]担任副教授，教授壁画艺术，同时他也是[[犹他州|犹他州]][[洛根|洛根]][[壁画学院|壁画学院]]的主席.

== 壁画复制技术 ==
[[Image:Frescography.jpg|thumb]]的程序设计截图]]
[[Image:Royal_Clipper_Lobby.jpg|thumb]]的大厅的[[frescography|壁画]]]]
80年代晚期 李海纳开始致力于研究壁画制作的新技术 从而发明了[[壁画复制技术|壁画复制技术]]以及[[电脑技术辅助壁画|电脑技术辅助壁画]]软件.这个专利技术将电脑程序与梦想世界设计工作室相结合，<ref name="Download.com">{{cite web | url = http://download.cnet.com/Dreamworlds-Design-Studio/3000-18496_4-10907042.html?tag=mncol | title = Download.com/Cnet - Dreamworlds Design Studio}}</ref> 用户可以将独立的图像组合成个人壁画设计。设计的成品将由艺术家工作室根据墙体的实际大小转移到帆布上，最后用类似粘贴墙纸的工艺将壁画粘贴在指定的地方。 只需要短短几个小时，壁画复制技术就能在计算机上完成壁画设计，然后再需要二到四周的时间来制作完成整个艺术作品并运送至指定地点。在广告中，代理商赞美其作品几乎和真的壁画一样完美。迄今为止，在欧洲，该项专利技术已售出高达300多次。
在世界各地著名的建筑中，人们都可以看到运用李海纳壁画复制技术制作的壁画作品。比如在[[维也纳市政厅|维也纳市政厅]]，在世界上最大的帆船[[皇家帆船|皇家帆船]]船舱内等.
在欧洲现在有大约300多个经纪商代理李海纳的壁画产品， 其品牌名为壁画大师<ref name="Search Results">{{cite web | url = http://www.google.be/search?client=firefox-a&rls=org.mozilla%3Ade%3Aofficial&channel=s&hl=de&q=frescomaster&meta=&btnG=Google-Suche|title = Google Search 'Frescomaster'}}</ref>
[[Image:Bergl_1_chroma.jpg|thumb]] [[维也纳|维也纳]], [[奥地利|奥地利]], 1770, 欧洲的墙壁和天顶壁画成品]]

===壁画学院===
2009年，李海纳成立了 [[壁画学院|壁画学院]] (IOF).<ref name="Institute of Frescography"/> 。IOF 是一个被官方认可 [[无盈利性质|(NFP)]] 的学院; 旨在提升公众对壁画艺术和壁画绘画的认识和兴趣。IOF也致力于研究 [[美术史|美术史]], [[数码再造技术|数码再造技术]], [[印刷|印刷]] 方法和材料, 以及[[壁画|壁画]] 修复技术. 

通过与德国中央艺术历史学院合作，IOF收集存档了40,000余幅从[[哥特式艺术|哥特]]时期到19世纪末期欧洲壁画 [[作品|作品]]的图像。另外，IOF还整理存档了“装饰世界”，其内容涵盖5000与个艺术装饰主题。“装饰世界”是基于自19世纪以来多彩石板印刷史上两本装饰百科全书：Auguste Racinet's L'Ornement polychrome Volumes I and II（1875-1888）及Dupont-Auberville's L'Ornement des tissus(1877)，而整理的。
同时，IOF也涉足创办壁画[[博物馆|博物馆]] 与其他壁画艺术展览项目。
