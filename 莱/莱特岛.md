'''雷伊泰島'''（Leyte）是[[菲律賓|菲律賓]][[群島|群島]]中的[[米沙鄢群島|米沙鄢群島]]的[[島嶼|島嶼]]之一。[[面積|面積]]7213[[平方公里|平方公里]]。北臨[[薩馬海|薩馬海]]；東北隔[[聖胡安尼克海峽|聖胡安尼克海峽]]與[[薩馬島|薩馬島]]相望，兩島有跨海大橋相連；南面為[[棉蘭老海|棉蘭老海]]，與[[棉蘭老島|棉蘭老島]]相望；西南為[[保和島|保和島]]；西為[[宿務島|宿務島]]；西北為[[馬斯巴特島|馬斯巴特島]]。

[[第二次世界大戰|第二次世界大戰]]時[[道格拉斯·麥克阿瑟|麥克阿瑟]]在1944年10月20日從此地涉水登陸，並說「我回來了」。之後與島上日軍激戰，以此島收復菲律賓。

最著名的戰役就是[[雷伊泰灣海戰|雷伊泰灣海戰]]和[[雷伊泰島戰役|雷伊泰島戰役]]，其中只耗時四天的雷伊泰灣海戰，成為歷史上最大的海上戰役。

== 地理 ==
島上有崎嶇的[[山|山]]區，最高峰為[[洛比峰|洛比峰]]，[[海拔|海拔]]1349[[公尺|公尺]]。

[[河流|河流]]多向北流入[[卡里加拉灣|卡里加拉灣]]或向西留入[[雷伊泰灣|雷伊泰灣]]。

[[平原|平原]]在沿海一帶，主要在雷伊泰島北部靠近[[獨魯萬市|獨魯萬市]]和[[奧爾莫克|奧爾莫克]]Ormoc City一帶。

Ormoc旁邊有重要的地熱發電廠。

== 經濟 ==
居民主要從事[[農業|農業]]，也有[[漁業|漁業]]。

=== 農業 ===
主要農作物為[[稻米|稻米]]、[[玉蜀黍|玉蜀黍]]。[[經濟作物|經濟作物]]為[[椰子|椰子]]、[[馬尼拉麻|馬尼拉麻]]、[[香蕉|香蕉]]、[[甘蔗|甘蔗]]等。

=== 礦業 ===
島上有[[錳|錳]]礦，西北部有[[沙岩|沙岩]]及[[石灰石|石灰石]]岩。

== 政治 ==
全島屬於菲律賓所有，行政區分[[雷伊泰省|雷伊泰省]]及[[南雷伊泰省|南雷伊泰省]]。奧爾莫克及塔克洛班港二[[直轄市|直轄市]]。

== 居民 ==
全島[[人口|人口]]1,598,942[[人|人]]（1980年[[統計|統計]]數字），主要方言為[[宿霧語|宿霧語]]及[[薩馬-雷伊泰語|薩馬-雷伊泰語]]。人口主要集中在雷伊泰省，有1,302,648人（1980年統計數字），雷伊泰省是菲律賓人口甚稠密的省份之一。
