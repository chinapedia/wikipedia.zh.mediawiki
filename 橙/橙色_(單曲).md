《'''橙色'''》（{{lang-ja|オレンジ}}）是[[GReeeeN|GReeeeN]]的第16張單曲，於2012年4月25日由[[環球音樂_(日本)|NAYUTAWAVE RECORDS]]發行。

== 概要 ==
《橙色》距離前作《[[強忍淚水，總有一天|強忍淚水，總有一天]]》發行約兩個月，為[[GReeeeN|GReeeeN]]於2012年發行的第2張單曲。單曲的「初回限定盤」附送收錄了《橙色》[[音樂影片|音樂影片]]的DVD。<ref>{{ja icon}}{{cite web|url=http://natalie.mu/music/news/68120 |script-title=ja:GReeeeNニューシングルは資生堂＆花王のCMソング収録 |publisher=[[Natalie_(網站)|Natalie]] |date=2012-04-22 |accessdate=2012-07-10}}</ref>

== 歌曲 ==
歌名「[[橙色|橙色]]」（{{lang|ja|オレンジ}}）是指[[日落|夕陽]]的顏色，歌詞中多次出現「晚霞（{{lang|ja|夕焼け}}）」一詞，亦有「橙色的背影（{{lang|ja|オレンジの背中}}）」、「染成橙色（{{lang|ja|オレンジ色に染めて}}）」等短句。<ref>{{ja icon}}{{cite web |url=http://listen.jp/store/diskreview_37113.htm |script-title=ja:ＧＲｅｅｅｅＮ／オレンジ |publisher=Listen Japan |date=2012 |accessdate=2012-07-14 }}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref>{{ja icon}}{{cite web |url=http://listen.jp/store/artword_1174351_146483.htm |script-title=ja:オレンジの歌詞｜GReeeeN |publisher=Listen Japan |date=2012 |accessdate=2012-07-10 |deadurl=yes |archiveurl=https://web.archive.org/web/20120520142630/http://listen.jp/store/artword_1174351_146483.htm |archivedate=2012-05-20 }}</ref>

音樂網站''hotexpress''指這首歌中可以感受到學生時代的心情，是一首青春歌曲，並就這首歌讚賞GReeeeN有關青春苦樂的歌曲。該評論又指這首歌可以讓人回想起學生時代心跳緊張的情景（指戀愛方面），搖滾的音樂亦表現出奔馳的感覺，從而描繪出對愛人的愛慕。評論特別提出4分20秒唱到的一句「我喜歡你 我喜歡你（{{lang|ja|君が好き 君が好き}}）」充滿力量。<ref>{{ja icon}}{{cite web|url=http://www.hotexpress.co.jp/review/3515/ |script-title=ja:学生気分を味わえる!? GReeeeNの新青春ソング登場 |publisher=hotexpress |date=2012 |accessdate=2012-07-10}}</ref>

《橙色》是[[資生堂|資生堂]]「SEA BREEZE」的廣告歌曲，《我要興高采烈～你也是嗎？》（{{lang|ja|ワイはワイワイでいいワイ〜おまえワィ?〜}}）是[[花王株式會社|花王]]「Clear Clean（{{lang|ja|クリアクリーン}}）」的廣告歌曲。<ref>{{ja icon}}{{cite web|url=http://artist.cdjournal.com/d/orange/4112022141 |script-title=ja:GReeeeN ／ オレンジ |publisher=CD Journal |date=2012 |accessdate=2012-07-10}}</ref>

== 反應 ==
單曲首周估計銷量約為1.4萬張，於[[Oricon公信榜|Oricon公信榜]]單曲周榜取得第8位。<ref name="oricon weekly" />

在[[RIAJ付費音樂下載榜|RIAJ付費音樂下載榜]]上，《橙色》以第1位首次上榜。<ref name="riaj">{{ja icon}}{{cite web|url=http://satsuki.musicdb.gr.jp/all_info/ranking_weekly/WeeklyRankingAction.do?term=2012f&weeklyCd=20120424&rankKbn=50 |script-title=ja:レコード協会調べ 2012年04月18日～2012年04月24日 ＜略称：レコ協チャート（「着うたフル(R)」）＞ |publisher=[[日本唱片協會|日本唱片協會]] |date=2012 |accessdate=2012-07-10 |deadurl=yes |archiveurl=https://web.archive.org/web/20120612091859/http://satsuki.musicdb.gr.jp/all_info/ranking_weekly/WeeklyRankingAction.do?term=2012f |archivedate=2012年6月12日 }}</ref>歌曲的排名在其後連續5周都位於20位內。<ref>{{ja icon}}{{cite web|url=http://satsuki.musicdb.gr.jp/all_info/ranking_weekly/WeeklyRankingAction.do?term=2012f&weeklyCd=20120605&rankKbn=50 |script-title=ja:レコード協会調べ 2012年05月30日～2012年06月05日 ＜略称：レコ協チャート（「着うたフル(R)」）＞ |publisher=[[日本唱片協會|日本唱片協會]] |date=2012 |accessdate=2012-07-10 |deadurl=yes |archiveurl=https://web.archive.org/web/20141001055103/http://satsuki.musicdb.gr.jp/all_info/ranking_weekly/WeeklyRankingAction.do?term=2012f&weeklyCd=20120605&rankKbn=50 |archivedate=2014年10月1日 }}</ref>
