《'''鍊金三級魔法少女'''》（{{lang-ja|錬金3級 まじかる?ぽか～ん}} / 簡稱：{{lang|ja|まじぽか}}）是{{lang|ja|がらくたはうす}}原作的電視動畫，於2006年4月4日至6月20日期間播放共有十二話二十四集，另有[[DVD|DVD]]特典中包括的一話，每集之間並無連續性。後來改編成[[漫畫|漫畫]]作品，由[[MediaWorks|MediaWorks]]發售單行本全一冊。

== 故事大綱 ==
四位從魔界到來人間界的公主，一起在一棵巨樹上的樹屋居住，因為不習慣人間界的生活而鬧出很多搞笑趣事。

== 人物 ==
（中文名稱為暫譯版本。）

;悠瑪（{{lang|ja|ゆうま}}）
:（CV：[[齋藤桃子|齋藤桃子]]）
: 本作品的女主角。對魔法咒語不太靈光，而且有時候會蠻不講理和發脾氣。外表和內心跟孩子一樣，總是抱著[[胡蘿蔔|胡蘿蔔]]的[[枕頭|枕頭]]睡覺。偶然有[[沒穿內褲|沒穿內褲]]的時候，原因是每回洗了之後沒曬乾。一直想要大魔法師的魔杖，曾經在[[七夕|七夕]]和[[聖誕節|聖誕節]]中許願。對一般人類的世界充滿好奇，但對事物較後知後覺。當其餘三位少女對悠瑪「剛剛發現的新奇事物」不感驚訝時，總令她大失所望。

;帕克拉（{{lang|ja|パキラ}}）
:（CV：[[平野綾|平野綾]]）
:[[吸血鬼|吸血鬼]]少女，有飛行的特徵。有著強橫的性格，不過在喜歡的人（帥哥）面前性格便會變得很老實。血是她的最愛，不過可以用[[番茄汁|番茄汁]]代替血來充饑。也有吸血鬼的弱點──[[紫外線|紫外線]]，不過也有防紫外線的對策：外出時頭上會戴著寫了「rariru!萵苣」的紙盒。雖然是吸血鬼，胸前卻戴上[[十字架|十字架]][[項鍊|項鍊]]，也不在乎吃[[蒜|蒜]]，不過會討厭有蒜味的人。身體無法從攝影器材中拍出來，不過可以用熱能攝錄機（溫度記錄器）映出。死穴是[[貧乳|貧乳]]，對其他貧乳的女孩有著某種情結（？），一直想要豐胸機「BU-600」。

;莉露（{{lang|ja|りる}}）
:（CV：[[生天目仁美|生天目仁美]]）
: 好食慾旺盛的[[狼女|狼女]]。在4人裡面胸部最大（？），有著褐色的皮膚和狗耳朵。常識是在4人裡面最高，交友關係也比較寬廣。愛吃肉。看到圓形的東西就會變成小狗。非常喜歡雨和水窪，在[[梅雨|梅雨]]期會得意洋洋地自己一個人玩。雖然有着狼人怕[[銀|銀]]器的弱點，不過第1回以後這個設定好像已被忘記了。曾經遇過擔任當地博物館警衛的男性同種族並且開始談戀愛，但因為最後在旅館房間內太過驚愕、誤咬了對方[[陰莖|閃閃發光的東西]]的關係而整個告吹。

;鐵子（{{lang|ja|鉄子╱あいこ}}）
:（CV：[[明坂聰美|明坂聰美]]）
: 體重300[[公斤|公斤]]的[[人造人|人造人]]。手腳和頸有金屬露出。氣力奇大，不過性格怕事、內向，有時候會被大家指指點點。在樹屋中擔任著家務的工作，擅長做飯但自己不能吃飯。電是能量的來源。會在睡眠的時候充電，不過[[內存|內存]]很低。一直憧憬肉體，曾經特意去仙女星座打算轉換為肉體。機器想是她全部可愛的朋友，怎麼也不被扔掉。

;ケイミィ
:（CV：[[のみこ|のみこ]]）
:透明人。幫助解說4個公主旁邊事物的人。在14話中作為看不見的第5位神秘成員。其他也只是偶然發言,不過，在一邊21話中進行英語會話遊戲的裁判和幫不能吃飯的鐵子喝下「草莓[[納豆|納豆]]果汁」。感覺上好像她是4個公主的監督。經常發送報告書到魔界的立場（？）。

;タン（{{lang|ja|タン}}）
:（CV：[[羽多野涉|羽多野涉]]）
:悠瑪的黑色兔形精靈，能變身成她的帽子。

;ジュン（{{lang|ja|ジュン}}）
:（CV：[[のみこ|のみこ]]）
:悠瑪的桃色兔形精靈。
