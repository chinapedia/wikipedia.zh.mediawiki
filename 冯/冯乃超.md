'''冯乃超'''（1901年10月12日—1983年9月），笔名'''冯子韬'''。祖籍[[广东|广东]][[南海|南海]]，生于[[日本|日本]][[横滨|横滨]][[山下町|山下町]]56番地<ref>{{Cite web |url=http://www.yhcqw.com/html/hyp/2008/47/084795512D3IG3370KABK72AD05EDIG9.html |title=存档副本 |access-date=2009-10-16 |archive-url=https://web.archive.org/web/20100115215542/http://yhcqw.com/html/hyp/2008/47/084795512D3IG3370KABK72AD05EDIG9.html |archive-date=2010-01-15 |dead-url=yes }}</ref>。中国现代诗人、文艺活动家。[[创造社|创造社]]和[[左联|左联]]等文艺社团的领导人。曾任中共中央人事部副部长、[[中山大学|中山大学]]副校长、[[广东省政协|广东省政协]]副主席等职。[[李声韵|李声韵]]丈夫。

==生平==
出身日本侨领之家。父亲冯德明。

1923年，在日本第八高等学校理科毕业后，先后就读于[[京都帝国大学|京都帝国大学]]哲学系、[[东京帝国大学|东京帝国大学]]哲学系社会学科，后改学美学与美术史。1926年加入[[日本共产青年同盟|日本共产青年同盟]]外围组织。1926年3月开始，在[[创造社|创造社]]《创造月刊》上发表组诗《幻想的窗》等具有象征色彩的诗歌，成为创造社出版部日本东京分部的联络人，并参加了日本革命学生组织的马克思主义读书会和艺术研究会，参与日本无产阶级文艺运动，接受苏联和日本的“左”翼文艺理论。

1927年秋[[大革命|大革命]]失败后，在[[成仿吾|成仿吾]]邀请下毅然弃学回国参加革命工作。1928年1月发表在《文化批判》创刊号上的两首诗《上海》和《与街头上人》，标志其诗风从低沉到革命的转变。1928年任《[[创造月刊|创造月刊]]》、《[[文化批判|文化批判]]》编辑，成为著名文学团体创造社后期的中坚和主将。1928年9月经[[潘汉年|潘汉年]]介绍加入[[中国共产党|中国共产党]]。1929年秋与[[郑伯奇|郑伯奇]]等组织[[中国艺术剧社|中国艺术剧社]]，任文学部长。与[[鲁迅|鲁迅]]等筹组[[中国左翼作家联盟|中国左翼作家联盟]]（“左联”），被推举为《理论纲领》的起草人。1930年3月2日左联成立，当选常务委员，并任左联首任党团书记兼宣传部长。1931年2月任[[中共中央宣传部|中共中央宣传部]]文化工作委员会（“文委”）书记、[[中国左翼文化界总同盟|中国左翼文化界总同盟]]（“文总”）党团书记。1931年3月任中共中央机关报《[[红旗周报|红旗周报]]》编辑。

1932年中共上海地下党组织遭到严重破坏，以[[中共中央特科|中共中央特科]]成员身份到[[武汉|武汉]]投靠岳父（[[同盟会|同盟会]]元老、[[湖北省民政厅|湖北省民政厅]]长[[李书城|李书城]]），在湖北省民政厅统计股任职，并作情报工作。<ref>http://qkzz.net/magazine/1002-4980/2001/21/8953.htm</ref>

1938年在[[长江局|长江局]]领导下，参加筹备[[中华全国文艺界抗敌协会|中华全国文艺界抗敌协会]]（“文协”），先后任筹备会书记、文协理事、常务理事，编辑《抗战文艺》。4月在[[国民政府军事委员会政治部第三厅|国民政府军事委员会政治部第三厅]]工作，任第七处第三科（对敌宣传科）科长，是[[郭沫若|郭沫若]]的助手。<ref>{{Cite web |url=http://www.yhcqw.com/html/hyp/2008/47/084795512D3IG3370KABK72AD05EDIG9.html |title=存档副本 |access-date=2009-10-16 |archive-url=https://web.archive.org/web/20100115215542/http://yhcqw.com/html/hyp/2008/47/084795512D3IG3370KABK72AD05EDIG9.html |archive-date=2010-01-15 |dead-url=yes }}</ref>是第三厅中共领导小组成员，还是中共特支书记，并领导“[[孩子剧团|孩子剧团]]”。1940年10月，第三厅撤销，成立[[国民政府军事委员会政治部文化工作委员会|国民政府军事委员会政治部文化工作委员会]]，出任文工会支部书记。1942年任[[中共中央南方局|中共中央南方局]]文化工作委员会（“文委”）委员。

[[重庆谈判|重庆国共谈判]]期间，任中共代表团顾问。1946年1月任[[中共中央南方局|中共中央南方局]]统战委员会文化组副组长，5月任[[上海工委|上海工委]]委员、“文委”书记。1946年10月，赴香港，任[[中共中央华南分局|中共中央华南分局]]香港工委委员、“文委”书记，主管香港文化和统战工作，编辑《大众文艺丛刊》。<ref>{{cite web |url=http://www.sysu.edu.cn/zdcq/xz4.htm |title=存档副本 |accessdate=2009-10-16 |deadurl=yes |archiveurl=https://web.archive.org/web/20101203084449/http://www.sysu.edu.cn/zdcq/xz4.htm |archivedate=2010-12-03 }}</ref>1947年7月编选了[[瞿秋白|瞿秋白]]著《[[论中国文学革命|论中国文学革命]]》和诗集《[[毛泽东颂|毛泽东颂]]》，由中共在香港的出版社[[海洋书屋|海洋书屋]]出版。1949年春，[[中华全国文学艺术工作者代表大会|中华全国文学艺术工作者代表大会]]筹备委员、党组代表成员、代表资格审查委员会主任；[[华北人民政府|华北人民政府]]教育委员会委员，第一届全国政协代表，[[中国人民政治协商会议第一届全体会议|中国人民政治协商会议第一届全体会议]]。<ref>{{cite web |url=http://www.sysu.edu.cn/zdcq/xz4.htm |title=存档副本 |accessdate=2009-10-16 |deadurl=yes |archiveurl=https://web.archive.org/web/20101203084449/http://www.sysu.edu.cn/zdcq/xz4.htm |archivedate=2010-12-03 }}</ref> 

中华人民共和国成立后，任[[中央人民政府|中央人民政府]][[政务院文教委员会|政务院文教委员会]]副秘书长兼人事处处长、[[中共中央宣传部|中共中央宣传部]]干部处处长、[[中央纪律检查委员会|中央纪律检查委员会]]委员、中央直属机关党委文教分党委书记。1950年9月任[[中央人事部|中央人事部]]副部长兼第四局局长。

应[[叶剑英|叶剑英]]、[[方方|方方]]要求，1951年2月降级出任[[中山大学|中山大学]]副校长兼党委书记，从管高教部的变为被高教部管，但仍然服从安排。<ref>http://qkzz.net/magazine/1002-4980/2001/21/8953.htm</ref>后曾任中山大学党委第一书记。与校内非中共的校长[[许崇清|许崇清]]、副校长[[陈序经|陈序经]]、名教授[[陈寅恪|陈寅恪]]等均关系融洽。曾当选[[广东省政协|广东省政协]]副主席。相继兼任中共广东省委高等学校委员会第一书记、中共广东省委委员、中共广东省委文教部副部长等职。当选为第一届全国人民代表大会代表、中共八大代表、广东省政协副主席、第四第五届全国政协委员、第一第四届[[中国文联|中国文联]]全国委员会委员。[[潘汉年|潘汉年]]被定为汉奸特务后，他依旧坚持潘是自己的入党介绍人。

[[文革|文革]]期间受冲击。文革期间到北京，曾要求[[周扬|周扬]]去向[[冯雪峰|冯雪峰]]道歉。1971年7月被解放。1975年4月调离中山大学转任[[北京图书馆|北京图书馆]]顾问。
==家庭==
独生女[[冯真|冯真]]，画家，1950年代苏联留学。[[中央美术学院|中央美术学院]]教授。

==评价==
[[陈寅恪|陈寅恪]]夫人[[唐筲|唐筲]]曾说冯乃超：“虽是个老党员，但倒是个念书的。”

*小说散文集《傀儡美人》，1929年，[[上海|上海]][[长风书店|长风书店]]
