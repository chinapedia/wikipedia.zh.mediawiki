'''冯玉祥墓'''，位于中国[[山东省|山东省]][[泰安市|泰安市]][[泰山|泰山]]西麓、西溪口东侧，是著名[[民国军阀|民国军阀]][[冯玉祥|冯玉祥]]的墓葬。1988年被列为[[全国重点文物保护单位|全国重点文物保护单位]]。

冯玉祥曾于1932年3月至10月、1933年8月至1935年10月两次在泰山隐居<ref>{{cite web |url = http://news.sina.com.cn/c/2009-08-24/064416173274s.shtml |title=《隐居泰山不忘造福当地百姓 冯玉祥：百姓眼中的布衣将军》|accessdate=2010-06-23 |authors = 侯海燕、王颜 |publisher=大众网-齐鲁晚报 }}</ref>，1948年在[[黑海|黑海]]因轮船失火遇难后，遵照其生前意愿，于1953年10月迁葬至此。

该墓坐东面西，正对跨西溪石峡两岸的大众桥（为其隐居泰山期间所倡建），沿4段共66级台阶而上。墓以泰山所产花岗石砌筑，墓壁正中嵌冯玉祥铜铸浮雕头像，其上横镌[[郭沫若|郭沫若]]所题写的“冯玉祥先生之墓”，其下为嵌于黑色磨光花岗石方碣之上、冯玉祥作于1940年的言志诗《我》<ref>{{cite web |url = http://www.wenwu.net.cn/infoview.asp?infoid=3823 |title=《国宝专项经费规划-冯玉祥墓》|accessdate=2010-06-23 |website = 国家文物局经费阳光工程网 }}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。
