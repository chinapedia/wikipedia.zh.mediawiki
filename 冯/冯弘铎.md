'''冯弘铎'''（{{bd|9世纪||907年|catIdx=F}}<ref name=h9k>《[[九国志|九国志]]》</ref>），[[中国|中国]][[唐朝|唐朝]]末年[[军阀|军阀]]，893年—902年间控制[[昇州|-{昇}-州]]地区（在今[[江苏|江苏]][[南京|南京]]）。

== 家世 ==
冯弘铎是[[涟水|涟水]]人。初为涟水所属的[[武宁军节度使|感化军]]偏将，曾被小吏所辱，同来自涟水且从小与他友善的偏将[[张雄_(唐朝)|张雄]]为他辩护，使得[[武宁军节度使|感化军节度使]][[时溥|时溥]]怀疑他俩。<ref name=NBT190>[[:s:新唐書/卷190|《新唐书》卷一百九十]]</ref>[[光启|光启]]二年（886年）十月，得知受到时溥怀疑，张雄和冯弘铎聚众三百出走，渡过[[长江|长江]]，攻占[[镇海军节度使|镇海军]]治下的[[苏州_(隋朝)|苏州]]，张雄自称[[刺史|刺史]]。最终他聚集起一支五万人、一千多艘船的军队，称其为天成军。<ref name=ZZTJ256>[[:s:資治通鑑/卷256|《资治通鉴》卷二百五十六]]</ref>

== 效力张雄 ==
三年（887年）四月，张雄最终被[[淮南|淮南]]六合镇遏使[[徐约|徐约]]逐出苏州，沿长江流浪到[[东海|东海]]，又返回沿江而上；冯弘铎追随他。<ref name=ZZTJ256/>当月，左厢都知兵马使[[毕师铎|毕师铎]]和他授以淮南节度权的盟友[[宣歙观察使|宣歙观察使]][[秦彦|秦彦]]控制淮南军部扬州，与围困扬州的[[庐州|庐州]]刺史[[杨行密|杨行密]]之间爆发战斗。八月，秦彦、毕师铎因食物短缺，希望得到张雄强兵的援助，授张雄仆射告身，以三通尚书告身授裨将冯弘铎等，出城中金玉珠缯换取张雄富余的粮食。但张雄军得到城中的财物而变富后，拒绝加入秦、毕一方作战，顾自离去。<ref name=ZZTJ257>[[:s:資治通鑑/卷257|《资治通鉴》卷二百五十七]]</ref>张雄最终落脚于[[上元|上元]]，[[唐昭宗|唐昭宗]]以上元为中心新设-{昇}-州，以张雄为刺史。<ref name=ZZTJ258>[[:s:資治通鑑/卷258|《资治通鉴》卷二百五十八]]</ref>[[景福|景福]]二年（893年）八月，张雄卒，冯弘铎代为刺史。<ref name=NBT10>[[:s:新唐书/卷10|《新唐书》卷十]]</ref><ref name=ZZTJ259>[[:s:資治通鑑/卷259|《资治通鉴》卷二百五十九]]</ref>

== 统治昇州 ==
冯弘铎善骑射，侃侃而谈如同儒者。<ref name=NBT190/>他拥有当地最强大的舰队，由远方好而耐用数十年的木材制成，被称为楼船，<ref name=ZZTJ263>[[:s:資治通鑑/卷263|《资治通鉴》卷二百六十三]]</ref>舰队之盛，闻名天下。为刺史后，扩建城防以备战守。[[乾宁|乾宁]]二年（895年）（《十国春秋·吴太祖世家》作三年（896年）四月），他和已占有淮南的[[杨行密|杨行密]]结好，请求归附。<ref name=h9k/><ref name=NHFD61>[[:s:新五代史/卷61|《新五代史》卷六十一]]</ref>但他又派宾客牙将[[尚公乃|尚公乃]]劝杨行密把[[润州|润州]]移交给他。杨行密拒绝，尚公乃说：“公不听我们的请求，但能胜过楼船吗？”但此事后，杨行密和冯弘铎之间并无大规模冲突。<ref name=NBT190/>

[[天复|天复]]二年（902年）三月，被[[宦官|宦官]][[韩全诲|韩全诲]]和军阀[[凤翔节度使|凤翔节度使]][[李茂贞|李茂贞]]控制的唐昭宗被另一大军阀[[宣武军节度使|宣武军节度使]][[朱全忠|朱全忠]]围攻，以左金吾将军[[李俨_(吴)|李俨]]为江、淮宣谕使，鼓励东南诸镇加入杨行密麾下攻打朱全忠。在李俨宣谕的诏书中，冯弘铎被任为[[武宁军节度使|武宁军]]（即感化军）节度使，出涟水攻打徐州、宣州。<ref name=ZZTJ263/><ref>[[:s:新唐書/卷188|《新唐书》卷一百八十八]]</ref>

此时冯弘铎感到夹在杨行密和杨行密附庸[[宁国军节度使|宁国军节度使]][[田頵|田頵]]之间，不安，但自恃有楼船，没加入任何一方。田頵图谋冯弘铎的领地，招募曾为冯弘铎造船的工人为自己造船。工人反对，称田頵的领地内缺乏冯弘铎使用的木料，造出的船不会那样耐久。田頵则回答：“造就是了，我只需要用一次。”田頵很快建成了战舰队，准备迎战冯弘铎。<ref name=ZZTJ263/>

-{昇}-州数有怪象，人们背着抱着婴儿奔走，劝止不住。冯弘铎意识到田頵的企图，大将冯晖、颜建建议他先攻田頵。六月，他因而调舰队在长江上向西南行进，称准备攻打[[镇南军节度使|镇南军节度使]][[钟传|钟传]]，其实意在攻打田頵。发兵当日，淮口有大风，卷起屋子，巨木在空中飞舞，<ref name=h9k/>-{昇}-州人大惊，说州要易主了。<ref name=NBT190/>杨行密派使者阻止他，他不听。但他在[[葛山|葛山]]迎战田頵舰队时，却被田頵大破。<ref name=ZZTJ263/><ref name=NHFD61/>

冯弘铎遭败后聚集舰队余部，计划沿江而下去东海。杨行密怕他重建舰队为后患，派使者去犒军，对他说，如果降顺，本人和军队都会得到善待，甚至还说如果冯弘铎要得[[扬州|扬州]]、当[[淮南节度使|淮南节度使]]，他也愿意相让。<ref name=NBT190/>冯弘铎左右都哭着愿意听命于杨行密。冯弘铎到[[东塘|东塘]]，九月，杨行密乘飞舻，带十余人，穿常服，不持兵器前去相见，到冯弘铎的船上，执冯弘铎手慰勉，冯弘铎全军感动欢呼，因而加入了杨行密的淮南军。杨行密任冯弘铎为淮南节度副使，以[[李神福|李神福]]代为昇州刺史，<ref name=NHFD61/>给冯弘铎的馆舍、馈赠都很丰厚，对冯弘铎所部将吏予以安置。<ref name=h9k/>此后冯弘铎事杨行密，不复叛。<ref name=NBT10/><ref name=SGCQ8>[http://www.archive.org/stream/06080946.cn#page/n2/mode/2up 《十国春秋》卷八]</ref>这时杨行密对尚公乃说：“颇记求润州时否？”尚公乃答以各为其主。杨行密笑道：“你事我杨叟如事冯公，无忧了！”<ref name=ZZTJ263/>田頵击破冯弘铎后，去广陵谢杨行密，趁机求领[[池州|池州]]、[[歙州|歙州]]，杨行密不许，最终导致田頵叛乱。<ref>[[:s:新唐書/卷189|《新唐书》卷一百八十九]]</ref><ref>[[:s:資治通鑑/卷264|《资治通鉴》卷二百六十四]]</ref>

冯弘铎一次陪杨行密祭祀[[汉高祖|汉高祖]]庙，树上有两只鸟栖息，杨行密命冯弘铎射之。冯弘铎一箭就将两鸟都射落。天祐四年（907年）卒。<ref name=h9k/>
