'''委内瑞拉华人'''（{{lang-es|''Chino-venezolano''或''Sino-venezolano''}}）是出生在[[委内瑞拉|委内瑞拉]]或者移民到委内瑞拉的[[華人|華人]]，約400,000名。<ref>{{cite web |url=http://convergencia.uaemex.mx/rev38/38pdf/LIZCANO.pdf |title=存档副本 |accessdate=2011-07-19 |0=2013-06-26 |deadurl=yes |archiveurl=https://web.archive.org/web/20130626000000/http://convergencia.uaemex.mx/rev38/38pdf/LIZCANO.pdf |archivedate=2013-06-26 }} 201页 {{es icon}}</ref>委内瑞拉华人幾乎都是從事烹饪领域。在委内瑞拉，中餐馆非常多，主要原因为{{link-es|委内瑞拉人|Venezolanos}}喜好[[粤菜|粤菜]]。<ref>{{cite web |url=http://asiapacifico.bcn.cl/en/articles/chinese-community-latin-america/ |title=存档副本 |accessdate=2012-07-24 |0=2011-02-06 |deadurl=yes |archiveurl=https://web.archive.org/web/20110206012823/http://asiapacifico.bcn.cl/en/articles/chinese-community-latin-america |archivedate=2011-02-06 }} ''拉丁美洲的华人群体'' {{en icon}}</ref>此外，委内瑞拉也是[[拉丁美洲|拉丁美洲]]最大的华人聚集地之一。[[巴伦西亚_(委内瑞拉)|巴伦西亚]]是大型华人群体所在地之一，并且有着形形色色的[[中华文化|中华文化]]的市场。在那些地方，从熏鸭到正宗[[陶器|陶器]]俱有，而且还有一家中文报社。在[[加拉加斯|加拉加斯]]埃尔博斯克区埃尔博斯克大街就有一条[[唐人街|唐人街]]。虽然委内瑞拉各大城市都存有唐人街，惟它们在各个城镇均很平凡，並非观光性質的部分。

来委内瑞拉的中国移民主要有两波，第一波主要為[[广东人|广东人]]，始于1847年；<ref>http://gadantre-1.blogia.com/2009/022101-los-chinos-en-la-venezuela-de-hoy.php “……第一批华人来到委内瑞拉始于1847年……” {{es icon}}</ref>第二波始于1940年代至1950年代，由于采油热潮而在1970年代中期达到顶峰。

委内瑞拉华人间广泛使用[[粵語|粵語]]，尤其以[[恩平话|恩平话]]，近来又有[[臺湾人|臺湾人]]移民至此，增添了他们在语言上和文化上的多样性。委内瑞拉最大的[[華人|華人]]群体来自[[恩平|廣東恩平]]，这些人都使用[[恩平话|恩平话]]。来自世界上其他地方的华人们也定居在了委内瑞拉，其中尤以来自[[菲律宾|菲律宾]]和[[古巴|古巴]]的居多。前者在1970年代曾受到了[[费迪南德·马科斯|费迪南德·马科斯]]的迫害，而后者则是因为[[菲德尔·卡斯特罗|菲德尔·卡斯特罗]]的[[共产主义|共产主义]]革命没收了他们的企业。
