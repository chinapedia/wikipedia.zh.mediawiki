《'''东方红'''》是一部紀錄和歌頌20世紀初[[五四运动|五四运动]]至[[中華人民共和國|中華人民共和國]]成立期间，中国人民在[[中国共产党|中国共产党]]特別是[[毛澤東|毛澤東]]的领导下所进行的反帝、反封建、反官僚主义的艰苦卓绝的革命斗争的大型[[音樂劇|音樂劇]]。是1964年，为[[中华人民共和国国庆节|十五周年国庆]]所筹备的剧目。由[[周恩来|周恩来]]担任总[[导演|导演]]。10月2日晚，在[[人民大会堂|北京人民大会堂]]首次上演。参与者达3700多人<ref name="a1"/>。參加演出的人员达3500多人。有称为1960年代红色艺术巅峰之作<ref name="a1">{{Cite web |url=http://news.163.com/10/1001/03/6HSNLJDE00014AED.html |title=《东方红》：中国革命的“创世纪”史诗|accessdate=2018-11-26 |author= |date=2010-10-01 |work= |publisher=网易，来源：华龙网-时代信报(重庆)|language=Zh-hans }}</ref>。

大型音樂舞蹈史詩《東方紅》继承了中国歌舞艺术的传统，发扬了中国人民所喜爱的载歌载舞的形式，采用了歌舞、[[大合唱|大合唱]]、齐唱、独唱、群舞、独舞等多种艺术表现方法，并且有[[中樂|民族樂隊]]和[[管弦樂|西洋管弦樂隊]]联合演奏。全劇共分八場，全部史诗由30多首曾在历史上产生广泛影响的革命歌曲和20多个舞蹈组成，其中包括5個大合唱、7个表演唱，穿插18段诗朗诵，变换场景37处。

1960年10月至11月，[[中国人民解放军空军司令员|中国人民解放军空军司令员]][[刘亚楼|刘亚楼]]随团出访朝鲜。在朝期间，代表团观看[[朝鲜人民军|朝鲜人民军]]演出的大型歌舞《三千里江山》。此次演出，给刘亚楼留下深刻印象<ref name="b1"/>。回国途中，刘亚楼观看了[[沈阳军区|沈阳军区]]举办的晚会<ref name="a5">{{Cite web |url=http://dangshi.people.com.cn/n/2015/1003/c85037-27658812.html |title=《东方红》是怎么诞生的|accessdate=2018-11-26 |author=王建柱 |date=2015-10-03 |work=责编：杨丽娜、常雪梅 |publisher=中国共产党新闻网，来源：解放日报|language=Zh-hans }}</ref>。年底，刘亚楼在杭州与[[中国人民解放军空军政治工作部文工团|空政文工团]]副团长张双虎的谈话中，表示“现在的年轻人没有经历过战争，不懂得中国革命的历史，应该运用文艺形式，对部队、对青年进行革命传统教育。沈阳军区唱了4首歌曲，空军也要唱，你们文工团也要唱……”。为落实刘亚楼指示，空政文工团开始研究演出节目。由空政文工团总团副团长牛畅负责创作。创作之初，牛畅等创作者曾参考了《三千里江山》的节目单。最终，决定创作一出以革命历史歌曲为主要内容的大歌舞表演<ref name="b1"/>，亦派出张士燮、朱正本、姚学诚、陈杰等词曲作家至江西搜集整理革命历史歌曲<ref name="a5"/>。期间，张士燮、朱正本创作了歌曲《[[十送红军_(歌曲)|十送红军]]》。

最初节目定名为《狂飙曲》，又改为《光辉的历程》。排练时，演员为三百人。后由于[[罗瑞卿|罗瑞卿]]反对歌颂空军的意见，引发风波。刘亚楼要求“现成的历史歌曲可以唱，不要另写<ref name="b1"/>。”公演前夕，《光辉的历程》一名被否定。依牛畅建议<ref name="a5"/>再度改名为《革命历史歌曲表演唱》。

1961年8月1日，《革命历史歌曲表演唱》在[[北京中山公园|北京中山公园]]音乐堂首次公演，后在[[民族文化宫|民族文化宫]]礼堂演出。全剧共有九场16景，演唱46首歌曲<ref name="b1"/>，歌曲包括《[[秋收暴动歌|秋收暴动歌]]》、《[[八月桂花遍地开|八月桂花遍地开]]》、《[[三大纪律八项注意#歌曲|三大纪律、八项注意]]》<ref name="a5"/><ref name="b2"/>、《[[会师歌|三大主力会师歌]]》、《[[义勇军进行曲|义勇军进行曲]]》、《[[松花江上|松花江上]]》、《[[抗日军政大学校歌|抗日军政大学校歌]]》、《[[到敌人后方去|到敌人后方去]]》、《[[大刀进行曲|大刀进行曲]]》、《[[游击队歌|游击队歌]]》、《[[南泥灣_(歌曲)|南泥湾]]》<ref name="b2"/>等。有人物、情节、舞蹈、演唱，各场之间以朗诵词串场。为避免创作痕迹，创作者皆称编辑。周恩来、[[叶剑英|叶剑英]]、罗瑞卿等人皆观看过演出<ref name="b1">{{cite journal en|quotes =

1964年年底，空政文工团应[[中国共产党上海市委员会|中共上海市委]]邀请，组织200余人赴沪演出<ref name="b2"/>。在[[上海市人民委员会|上海市人民委员会]]大礼堂演出了《革命历史歌曲表演唱》。此次演出历时一个月，观众达七万余人<ref name="a5"/>{{efn|陈爽文章<ref name="b2"/>称空政文工团在上海演出40场，历时1个多月，观众7200余人。}}。广州、兰州、济南、南京、杭州等地艺术团体派人专程至上海观摩<ref name="b2"/>。1962年初，总政文化部副部长李伟对《革命历史歌曲表演唱》的名称提出修改意见，提出定名为《东方红》<ref name="a5"/>。《革命历史歌曲表演唱》亦有拍摄成电影<ref name="b2"/>。

1964年，上海文艺界为庆祝[[上海解放|上海解放]]十五周年，并迎接第五届“[[上海之春|上海之春]]”创作大歌舞《在毛泽东旗帜下高歌猛进》。上海文化局副局长[[许平|许平]]担任艺术指导。5月23日，在[[上海文化广场|文化广场]]<ref name="a3">{{Cite web |url=http://history.people.com.cn/GB/199250/241996/17609671.html |title=辉煌“绝唱”(1)|accessdate=2018-11-26 |author= |date=2012-04-09  |work=责任编辑：肖静 |publisher=人民网|language=Zh-hans }}</ref>举行的“上海之春”开幕式上首演。此剧为八场十六景<ref name="a1"/>，有序曲一节。参演者有二千余人。期间演出两场，闭幕后又演出七场<ref name="a3"/>。首轮演出观众达7万余人。研究者认为《在毛泽东旗帜下高歌猛进》的演出已相当成熟，将它视为《东方红》的底本。有回忆称，《在毛泽东旗帜下高歌猛进》在创作过程中，曾直接借阅空政文工团《革命历史歌曲表演唱》的演出台本，加以借鉴改编<ref name="b2"/>。

6月18日，副总理[[陈毅|陈毅]]观看《在毛泽东旗帜下高歌猛进》演出后，指示中共上海市委，“班子不要散，请总理来看一次<ref name="a3"/>。”7月13日，总理周恩来观看后，决定以此为基础改编，国庆在北京演出<ref name="a1"/>。

=== 1964年演出 ===

周恩来从上海回到北京后，7月18日，在[[国务院常务会议|国务会议]]上提议，创作以中国共产党历史为题材、展现[[毛泽东思想|毛泽东思想]]的大型歌舞作品，作为国庆十五周年[[献礼|献礼]]。7月20日，在[[中南海|中南海]][[西花厅|西花厅]]，周恩来与主管文艺工作的中宣部副部长[[周扬_(政治人物)|周扬]]、文化部副部长[[齐燕铭|齐燕铭]]、对外文委副主任[[张致祥|张致祥]]等人商讨创作事宜<ref name="a5"/>。7月24日，周扬向[[陆定一|陆定一]]、[[康生|康生]]、周恩来、[[彭真|彭真]]正式提出请示。7月30日，拟定13人组成的领导小组。组长周扬，副组长[[梁必业|梁必业]]、[[林默涵|林默涵]]等，组员齐燕铭、张致祥、[[陈亚丁|陈亚丁]]、[[周巍峙|周巍峙]]、[[许平|许平]]、[[吕骥|吕骥]]等。组织指挥工作小组主任陈亚丁，副主任周巍峙、许平等。这个领导小组后又被称为大歌舞指挥部。由于《在毛泽东旗帜下高歌猛进》的名称过长<ref name="a3"/>，采用总政文化部副部长李伟在1962年的意见<ref name="a5"/>，定名为《东方红》。

8月1日，周恩来批准周扬请示。《东方红》的创作、排演正式开始。12位参加《在毛泽东旗帜下高歌猛进》的人员奉命进京，参加《东方红》排演<ref name="a3"/>。选择演出人员时，周恩来考虑当时[[毛泽东|毛泽东]]提倡的“[[全国学人民解放军|大学解放军]]”，决定以部队人员为主。提出演出人员以[[总政文工团|总政文工团]]为主体，[[中央歌剧团|中央歌剧团]]、[[中央樂團|中央乐团]]、[[东方歌舞团|东方歌舞团]]等政府文艺单位参与。因此解放军各军区文工团、地方系统专业、业余文艺团体都有参加演出<ref>{{Cite web |url=http://history.people.com.cn/GB/199250/241996/17609672.html |title=辉煌“绝唱”(2)|accessdate=2018-11-26 |author= |date=2012-04-09  |work=责任编辑：肖静 |publisher=人民网|language=Zh-hans }}</ref>。演员除以驻京文艺团体为主外，抽调了[[南京军区|南京军区]]、[[济南军区|济南军区]]文工团等单位人员<ref name="a5"/>，亦有来自上海的演员<ref name="b2"/>。共计三千余人，其中合唱队有一千余人<ref name="a5"/>。

10月2日，在人民大会堂正式公演。由于取得巨大成功，在人民大会堂连演十余场。至16日结束时，共演出14场。期间，毛泽东、周恩来、彭真、[[贺龙|贺龙]]等人在10月6日，[[中国人民解放军总政治部|中国人民解放军总政治部]]举办的国庆晚会上观看了《东方红》<ref name="a2"/>。《東方紅》的後兩場「祖國在前進」及「世界在前進」遵照毛澤東的意見删除<ref name="a5"/>，在電影版中亦如此，只選取序曲和前六場。

=== 1965年电影版 ===

1964年《东方红》公演成功后。周恩来提议拍摄成电影。[[北京电影制片厂|北京電影製片廠]]、[[八一电影制片厂|八一电影制片厂]]和[[中央新闻纪录电影制片厂|中央新聞紀錄電影製片廠]]聯合将此剧攝制成彩色寬銀幕舞台藝術片。

大歌舞指挥部扩大领导小组成员。[[中华人民共和国文化部|文化部]]电影司[[司徒慧敏|司徒慧敏]]及[[八一电影制片厂|八一]]、[[北京电影制片厂|北影]]、[[中央新闻纪录电影制片厂|新影]]的负责人成为新的成员。电影导演为[[王苹|王苹]]、[[李恩杰|李恩杰]]，摄影为[[薛伯青|薛伯青]]、[[钱江|钱江]]等，并成立了《东方红》电影导演团。用了近一年时间对《東方紅》进行修改，电影共花费三个月时间录制完成。在1965年9月18日，拍摄完最后一个电影镜头<ref name="a4">{{Cite web |url=http://history.people.com.cn/GB/199250/241996/17609674.html |title=辉煌“绝唱”(4)|accessdate=2018-11-26 |author= |date=2012-04-09  |work=责任编辑：肖静 |publisher=人民网|language=Zh-hans }}</ref>。10月1日<ref name="a1"/>，作为国庆献礼片<ref name="a4"/>公映。在文革期間，因為政治原因，電影版東方紅處於被禁狀態，1977年重新上映。
