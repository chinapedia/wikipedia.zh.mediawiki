'''东绒线胡同'''是位于中国[[北京市|北京市]][[西城区|西城区]]中部的一条[[胡同|胡同]]，现已拓宽为道路。

==简介==
东绒线胡同位于[[西长安街|西长安街]]以南，为东西走向。在[[国家大剧院|国家大剧院]]兴建之前，东绒线胡同东到[[人大会堂西路|人大会堂西路]]，西到[[北新华街|北新华街]]，全长643米，平均宽度9米。2007年竣工的国家大剧院占据了东绒线胡同东段，东绒线胡同缩短为东到[[石碑胡同|石碑胡同]]、[[兵部洼胡同|兵部洼胡同]]交口处，西到北新华街<ref name=dhly/>。

[[明朝|明朝]]因该胡同西口有木板桥，故得名“板桥”。东段[[清朝|清朝]]称“张相公庙”，因同名庙宇而得名<ref name=dhly/>。[[朱一新|朱一新]]《京师坊巷志稿》卷上记载：<ref name=zhu>朱一新，京师坊巷志稿，北京古籍出版社，1982年</ref>

后来，该胡同东西两段合并，统称为“板桥”。1911年后并入绒线胡同。后来又将双沟沿并入，使绒线胡同向东延伸到司法部。1965年，以北新华街为界，将绒线胡同分为东、西两段，分别命名为[[西绒线胡同|西绒线胡同]]、东绒线胡同<ref name=dhly/><ref name=lvyou>旦其、于志强，石碑胡同 阳春白雪与市井生活的分界线，旅游世界2014(9):104-109</ref>。

东绒线胡同北侧有北京市西城区长安小学（[[北京市第二实验小学|北京市第二实验小学]]长安校区），原名东绒线胡同小学，原是[[清朝|清朝]]右翼宗学旧址。右翼宗学原先在[[西单|西单]]以北的[[小石虎胡同|小石虎胡同]]，清朝[[乾隆|乾隆]]十九年（1754年）迁移至此处<ref name=dhly>{{Cite web |url = http://www.bjdclib.com/subdb/laneculture/lane/200907/t20090703_4876.html |title =  东绒线胡同|author =  |date =  2009-07-03|publisher = 东华流韵 |language =  |accessdate =  |quote =  }}</ref>。
