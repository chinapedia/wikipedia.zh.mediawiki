'''东风-41型洲际弹道导弹'''（'''DF-41'''，北約代號'''CSS-X-10'''）是[[中华人民共和国|中华人民共和国]]研制的一种三级[[固体燃料|固体燃料]]机动型[[洲际弹道导弹|洲际弹道导弹]]<ref>[http://mil.eastday.com/m/20090115/u1a4111183.html 俄将军:解放军2010年将接收东风-41洲际导弹] {{Wayback|url=http://mil.eastday.com/m/20090115/u1a4111183.html |date=20120824204302 }} 2009年1月15日 东方军事网</ref>，可携带大約10枚[[多目标重返大气层载具|分导式多弹头]]，最大射程约14000公里<ref name=missilethreat-2014>{{cite web |url=https://missilethreat.csis.org/missile/df-41/ |title=DF-41 (CSS-X-10) |publisher=Missilethreat.com |date=12 February 2014 |accessdate=8 March 2014 |archive-url=https://web.archive.org/web/20170202231805/https://missilethreat.csis.org/missile/df-41/ |archive-date=2017-02-02 |dead-url=no }}</ref>（最新改进型约15000公里<ref name="sina_df4112"/>），超过美国[[LGM-30民兵导弹|LGM-30民兵导弹]]的13000公里。据信其最高速度可达25[[马赫|马赫]]<ref name="armedforces-int.com"/>，圓周偏差率100~200米，从中国大陆发射几乎可以打擊地球上任何一個目標。该导弹可機動發射，有公路版東風41洲際導彈和鐵路版東風41洲際導彈。

== 描述 ==
东风-41型洲际弹道导弹是一种[[多目标重返大气层载具|多目标重返大气层载具]]导弹，属于第三代洲际导弹改良而出的，综合能力已達致美国[[LGM-30義勇兵洲际弹道导弹|LGM-30義勇兵洲际弹道导弹]]和[[三叉戟导弹|三叉戟-II导弹]]的水平。

2014年8月，中国[[陕西|陕西]]省环境监测中心网站上做出有关DF-41洲际弹道导弹设置环境监测点的事件的新闻报道<ref>[http://mil.sohu.com/20140801/n403014684.shtml 中国东风41导弹被证实 地方政府正前期保障] {{Wayback|url=http://mil.sohu.com/20140801/n403014684.shtml |date=20161119064147 }}, 搜狐军事</ref>。这是东风-41第一个正式的公开证明，这也证明了东风-41的发展已接近尾声。得到公众的关注后不久这条新闻报告（以及整个网站）被撤下<ref name=afp-20140801>{{cite news |url=http://www.telegraph.co.uk/news/worldnews/asia/china/11005061/China-confirms-new-generation-long-range-missiles.html |title=China 'confirms new generation long range missiles' |work=AFP |publisher=Daily Telegraph |date=1 August 2014 |accessdate=2 August 2014 |archive-url=https://web.archive.org/web/20140826052934/http://www.telegraph.co.uk/news/worldnews/asia/china/11005061/China-confirms-new-generation-long-range-missiles.html |archive-date=2014-08-26 |dead-url=no }}</ref>。美媒报道称东风-41有望在2015年开始服役<ref>美报告：中国DF-41明年服役 已部署2个旅DF-21D http://military.people.com.cn/n/2014/1014/c172467-25833622.html {{Wayback|url=http://military.people.com.cn/n/2014/1014/c172467-25833622.html |date=20150330174132 }}</ref>。

到目前为止，关于东风-41型洲际弹道导弹的任何信息，都处于高度保密状态，所有媒体的信息来源都基本出于估测和军事发烧友在网上晒的照片<ref>[http://club.mil.news.sina.com.cn/thread-730406-1-1.html?sudaref=mil.news.sina.com.cn&retcode=0  这又是什么家伙，好粗壮！]</ref><ref>{{Cite web |url=http://mil.news.sina.com.cn/2013-12-19/1705755793.html |title=深度：DF-41与俄白杨-M导弹比还是有那么点差距  |access-date=2015-08-31 |archive-url=https://web.archive.org/web/20150906042802/http://mil.news.sina.com.cn/2013-12-19/1705755793.html |archive-date=2015-09-06 |dead-url=no }}</ref>。2015年9月3日的“纪念抗战胜利70周年阅兵式”中，东风-41型洲际导弹没有出现，取而代之的是[[东风-31|东风-31A型]]和[[东风-5型洲际弹道导弹|东风-5B型]]“分导式”洲际导弹<ref>{{Cite web |url=http://mil.news.sina.com.cn/2015-09-03/1130838377.html |title=专家:阅兵战略导弹部队是在亮家底 |access-date=2015-09-03 |archive-url=https://web.archive.org/web/20150906161855/http://mil.news.sina.com.cn/2015-09-03/1130838377.html |archive-date=2015-09-06 |dead-url=no }}</ref>。

对于东风-41型洲际导弹的最大射程，有12000公里、14000公里和15000公里三种不同的说法，其中12000公里为中国境内腹地（河南）发射经过北极覆盖美国本土全部领土的射程要求，14000公里为中国境内腹地发射经过太平洋覆盖美国本土全部领土的射程要求，15000公里为中国境内任何一点发射覆盖欧洲、北美任何目标的射程要求。

2015年底美國媒體大幅報導[[喬治城大學|喬治城大學]]亞洲軍控項目已經確認有东风-41型的鐵路版發射成功，隨後央視間接證實，<ref>{{Cite web |url=http://news.cntv.cn/2015/12/22/VIDE1450795325808777.shtml |title=美惊呼中国“陆上核航母”现身 |access-date=2015-12-24 |archive-url=https://web.archive.org/web/20151224201504/http://news.cntv.cn/2015/12/22/VIDE1450795325808777.shtml |archive-date=2015-12-24 |dead-url=no }}</ref>這種重型洲際飛彈使用公路車輛發射雖然可行但有不少問題，首先是巨大的重量對卡車輪胎和底盤的維護保障有壓力，不少路面和橋梁承重有問題，更多涵洞和橋底也無法讓這種巨型車輛通過或轉彎，所以配合發展大規模的[[鐵路|鐵路]]和隧道系統，利用[[火車|火車]]來運輸和發射，利用大量隧道來隱蔽，是更合理經濟的方式構建一個大數量且機動的洲際飛彈發射網，在敵方核打擊下生存率較高，有充分二次打擊能力，並且多彈頭和誘餌標靶散佈都已經試射成功表示只要一枚飛彈穿過敵國反導防護網，就能造成重大死傷。

== 研发历史 ==

1984年，东风41立项，工程编号204工程，目的是为了加强陆基战略威慑力量，用来替换性能逐渐落后的[[東風-5型洲際彈道飛彈|東風-5型洲際彈道飛彈]]。

1991年，解决固体燃料问题。

1994年，首次高弹道试射成功。

1999年，计算机模拟试射美国本土成功。

2010年，交付部队。

2016年，正式列装<ref>{{Cite web |url=http://mil.news.sina.com.cn/jssd/2016-12-02/doc-ifxyiayr8791560.shtml |title=军媒报道有玄机：东风41洲际导弹或已低调列装部队 |access-date=2016-12-02 |archive-url=https://web.archive.org/web/20161202165349/http://mil.news.sina.com.cn/jssd/2016-12-02/doc-ifxyiayr8791560.shtml |archive-date=2016-12-02 |dead-url=no }}</ref>。
