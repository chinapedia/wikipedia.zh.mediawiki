'''東鳳鎮'''是[[中國|中國]][[廣東省|廣東省]][[中山市|中山市]]下辖的一个[[镇|镇]]，位于中山市北部，面積54.8平方千米，人口約14萬。

== 歷史 ==
“東鳳”一名取自於近代的三個轄區：東海，鳳儀，鳳鳴。但現在的含義已轉變為一向東的鳳凰，官方則是以一向東（右）的金色鳳凰作為標誌。

東鳳直到[[明朝|明]]中至[[清朝|清]]初才開始有人移居於此，皆因[[元代|元代]]以前東鳳大片地區仍然為汪洋大海，而此後才續漸由於西江支流的沖漬而淤漬成數個沙洲，如大拗沙，罟步沙，伯公沙等，但現在現在己多去掉“沙”字，而成為當地地名。初時移居當地的人多以垦耕為業，原住地以隣近縣鎮為居多，如本市的[[小欖|小欖]]，[[黃圃|黃圃]]和[[順德|順德]]的[[大良|大良]]，[[勒流|勒流]]。後築成多個以”圍”字為後輟的地域。

東鳳歷史上曾多次與隣近鎮區合並和分開自立。15到16世紀初在最早於漬成的[[大拗|大拗]]，[[罟步|罟步]]及[[伯公|伯公]]沙洲上始有人垦耕，當時分屬大欖都和黃旗都。宣統二年（1910年）改屬三，九區。1946年三區轄域改稱[[東海鄉|東海鄉]]，九區轄域稱鳳儀鄉。1950年[[鳳儀鄉|鳳儀鄉]]又分立為鳳儀和鳳鳴兩鄉。1955年並入[[南頭區|南頭區]]。1957年與南頭分立時稱東鳳鄉。1958年10月成立東鳳公社，於當年年底並入[[小欖公社|小欖公社]]，次年又分出自立。後又分立為東鳳和和平兩公社，屬[[南頭區|南頭區]]。1974年又復析出稱東鳳公社。1983年稱區。1986年12月始成立鄉級鎮至今。

== 地域 ==
東鳳鎮大致為一西北──東南走向的地域。分別位於四極的村級行政區為：最東和最南的[[東罟步村|東罟步村]]，最西的[[安樂村|安樂村]]，以及最北的[[同安村|同安村]]。東鳳地處[[珠三角|珠三角]]西岸，[[西江|西江]]下游支流[[鷄鴉水道|鷄鴉水道]]和[[小欖水道|小欖水道]]相夾的淺陸上。為[[中山市|中山市]]一僅次於[[黃圃鎮|黃圃鎮]]的最北部鄉鎮。[[中心排河|中心排河]]為一西北──東南橫貫東鳳中心的河涌。與本市[[南頭鎮|南頭鎮]]，[[黃圃鎮|黃圃鎮]]，[[阜沙鎮|阜沙鎮]]，[[東昇鎮|東昇鎮]]，[[小欖鎮|小欖鎮]]以及佛山市[[順德區|順德區]]的[[容桂街辦|容桂街辦]]和[[均安鎮|均安鎮]]阰隣。

== 行政区划 ==
东凤镇現下轄1個社区和12個村。

*社区：[[東興社區|東興社區]]
*村：[[同安村|同安村]]，[[伯公村|伯公村]]，[[和泰村|和泰村]]，[[東和平村|東和平村]]，[[東罟步村|東罟步村]]，[[永益村|永益村]]，[[安樂村|安樂村]]，[[民樂村|民樂村]]，[[小瀝村|小瀝村]]，[[穗成村|穗成村]]，[[吉昌村|吉昌村]]，[[西罟步村|西罟步村]]。

== 經濟 ==
[[File:東鳳電子電器城.JPG|right]]
東鳳從改革開放始續漸走向以小家電為主導的電子生產業。但近來各種工業也稍有發展。位於轄區內的[[美日潔寶有限公司|美日潔寶有限公司]]，[[鐵將軍防盜設備有限公司|鐵將軍防盜設備有限公司]]，[[高雅線圈制品有限公司|高雅線圈制品有限公司]]，以及於東鳳設立分廠區的[[美的集團股份有限公司|美的集團股份有限公司]]為較知名的企業。

== 市政和民生 ==
東鳳人口約為13萬多，其中戶籍人口7.16萬，非戶籍人口4.12萬（2005年資料）。而當中還不包括没有登記在案的外來人員。

[[File:東鳳公園.JPG|right]]
東鳳近年來在河道污染，道路建設以及市政公園綠地上花了一定力度。從新規劃建成了[[東鳳公園|東鳳公園]]，拆御[[舊東鳳市場|舊東鳳市場]]改建為[[鳳凰公園|鳳凰公園]]。以及在多個地點增建了市民健身和休閒場所。隨著經濟的發展，多個地產也逐步發展起來，如位於[[小瀝|小瀝]]的[[逸湖半島地產|逸湖半島地產]]。以及位於[[伯公|伯公]]的[[東御世家地產|東御世家地產]]。但隨著近年來大量外來勞工的擁入，逐漸出現治安惡化的跡象。公安部門己於最近在各個主要公衆地點安裝了治安監察儀器。

東鳳目前還没有大學，但有中學3間，其中初級中學2間，高級職業學校1間，小學12間，其中公辦小學11間，民辦小學1間，以及幼稚園17間。

*高級職業中學：[[東鳳理工學校|東鳳理工學校]]
*普通初級中學：[[東鳳中學|東鳳中學]]，[[東鳳二中|東鳳二中]]

== 交通 ==
[[105國道|105國道]]南北貫穿西北部，為一車流輛極度煩忙的干道，并曾發生過多起重大交通事故。[[364省道|364省道]]穿越中心全境與[[105國道|105國道]]相交會，以及較多的鄉級道路和街道，其中[[興華路|興華路]]為東鳳最為熱閙的街道之一。東鳳現時没有當地的巴士公司。但由中山市公共汽車公司營運的巴士經往主要路段：

*20路車：由[[東鳳車站|東鳳車站]]駛往[[黃圃車站|黃圃車站]]
*31路車：由東鳳[[同安|同安]]駛往[[南區_(中山市)|南區]][[城南客運站|城南客運站]]
*60路車：由[[東鳳醫院|東鳳醫院]]駛往[[東罟村|東罟村]]，是惟一完全在境內行駛之巴士路線。
*73路車：由[[城南客運站|城南客運站]]到[[黃圃車站|黃圃車站]]，其中經往東鳳鎮[[東阜公路|東阜公路]]。
*210路車：由[[黃圃車站|黃圃車站]]至[[中山汽車總站|中山汽車總站]]，其中東鳳段經往整條[[105國道|105國道]]。



除此之外當地市民日常也使用渡淪過往[[小欖水道|小欖水道]]到逹[[小欖鎮|小欖鎮]]和[[東昇鎮|東昇鎮]]，或[[鷄鴉水道|鷄鴉水道]]到逹[[南頭鎮|南頭鎮]]。而未來建成之[[廣珠城際軌道交通|廣珠城際軌道交通]]也將過境東鳳，但没有設立上落站。

== 旅遊點 ==
[[File:發展中之逸湖半島地產.JPG|left]]
東鳳并没有正式發展旅遊業，但某些地點仍然可以稍為提點：

*[[興華路|興華路]]：為東鳳開發最為早和商業最為集中的街道之一。
*巴士沿線：60路車可欣賞到東鳳最純樸之鄉土風貌。
