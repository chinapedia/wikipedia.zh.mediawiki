'''东仓桥'''为位于[[中国|中国]][[江苏省|江苏省]][[无锡|无锡]][[宜兴市|宜兴市]][[宜城街道|宜城街道]][[月城街|月城街]]北端的一座单孔石拱桥，东西向跨东仓河（南偏西28°），始建于南宋[[宝庆|宝庆]]年间，明代重建，清[[咸丰|咸丰]]时和抗战后曾重修，现状桥面、护栏等为1997年3月整修，但主体仍存明代风格，为宜兴城内唯一幸存的古桥<ref>{{cite web |url=http://www.yxwbw.com/old/jishi.asp|title=《文博记事》|accessdate=2013-12-13|work=|publisher=宜兴文博网}}</ref>。该桥由花岗石、青石混合构筑，长30米，顶宽3.60米，堍宽4.70米，净跨8.30米，矢高5.50米，桥栏高0.75米，东西两侧各沿34级和28级台阶而上（其中东侧有12级台阶1997年被拆除，今改为水泥台阶），台阶坡度16°<ref>{{cite web |url=http://www.yxwbw.com/show.asp?id=79|title=《宜兴古桥梁》|accessdate=2013-12-13|work=|publisher=宜兴文物}}</ref>。
