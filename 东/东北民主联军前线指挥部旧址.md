'''东北民主联军前线指挥部旧址'''又名'''[[中国人民解放军第四野战军|中国人民解放军第四野战军]][[辽沈战役|辽沈战役]]前线指挥部旧址'''。位于[[黑龙江省|黑龙江省]][[双城市|双城市]][[昌盛街|昌盛街]]优干胡同6号。始建于1917年。原为[[吉林省警察厅|吉林省警察厅]]警备处处长赵冀的私人住宅<ref>{{Cite web |url = http://heilongjiang.dbw.cn/system/2006/11/27/050618217.shtml |title=“四野”辽沈战役前线指挥部旧址修缮一新 |accessdate = 2013-07-08 |author = 王凤超、王殿臣、魏立斌 |editor = 吕博 |date=2006-11-27 |work = |publisher = 东北网（来源：哈尔滨日报） |language = zh-hans }}</ref>。[[第二次国共内战|第二次国共内战]]时期的1946年至1948年9月，[[东北民主联军|东北民主联军]]前线指挥部驻扎于此地。

1990年12月29日，[[黑龙江省人民政府|黑龙江省人民政府]]公布为[[第三批黑龙江省文物保护单位|第三批黑龙江省文物保护单位]]。2006年，维修后建成“中国人民解放军第四野战军辽沈战役前线指挥部旧址纪念馆”。2013年5月，[[中华人民共和国国务院|中国国务院]]公布为[[第七批全国重点文物保护单位|第七批全国重点文物保护单位]]（[[第七批全国重点文物保护单位中的近现代重要史迹及代表性建筑|近现代重要史迹及代表性建筑]]）
