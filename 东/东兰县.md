'''东兰县'''是[[中国|中国]][[广西壮族自治区|广西壮族自治区]][[河池市|河池市]]所辖的一个[[县|县]]。总面积为2435平方公里，2002年总人口27.8万人，其中壮族占85%。

== 行政区划 ==
东兰县辖5个镇、8个乡、1个民族乡。

== 经济 ==
2004年，全县实现地区生产总值68749万元，其中第一产业增加值25718万元，第二产业增加值 16690万元，第三产业增加值26341万元；财政收入7886万元；全社会固定资产投资52050万元；社会消费品零售总额26886万元；农民人均纯收入1458元；城镇居民可支配收入5629元。

== 名人 ==
* [[韦拔群|韦拔群]]，壮族，中国现代农民运动领袖之一。
*[[韦国清|韦国清]]，壮族，中国人民解放军将领，1955年被授予上将军衔，历任华东野战军纵队司令、中共广西自治区委员会书记、广州军区政治委员、解放军总政治部主任等职
*[[韦杰|韦杰]]，壮族，中国人民解放军中将。1929年参加百色起义和中国红军第七军。参加第二次至第五次中央苏区反“围剿”作战。长征期间，任军团侦察连连长，二渡赤水，先敌抢占娄山关，为夺取桐遵战役的胜利作出重要贡献。随后率部巧渡金沙江，抢渡大渡河，为大部队顺利渡江开辟通道。抗日战争和第二次国共内战期间，率部参加百团大战、安阳和上党、平汉、陇海战役，挺进大别山，进攻定陶、太原等战役。率军参加攻占大西北和大西南等。中共建政后，任中国人民志愿军第六十军军长，参加抗美援朝第五次战役。回国后，任成都军区副司令员等职。
*[[覃健|覃健]]，中国人民解放军中将。1929年参加百色起义和中国红军第七军。参加第二次至第五次中央苏区反“围剿”作战和二万五千里长征。抗日战争和第二次国共内战期间，率部参加漳南和盐城、济南、淮海、渡江、上海等战役。中共建政后，1950年，任中国人民志愿军第九兵团参谋长，参与指挥长津湖战役等。回国后任南京军区副参谋长。
*[[韦祖珍|韦祖珍]]，壮族，中国人民解放军将领，少将
*[[覃士冕|覃士冕]]，中国人民解放军少将。1929年参加百色起义和中国红军第七军。参加第三次中央苏区反“围剿”和二万五千里长征。抗日战争期间，参加平型关战斗和开辟鲁南敌后抗日根据地。第二次国共内战期间，为保卫临沂，指挥一个旅阻击国民党一个师和快速纵队的进攻，连战20余日，以少胜多。中共建政后，任广西军区第二政委、海南军区副司令员、广西壮族自治区政协副主席等职。
