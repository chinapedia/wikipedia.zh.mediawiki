'''東鄉語'''是[[中國|中國]][[東鄉族|東鄉族]]的語言，屬於[[阿爾泰語系|阿爾泰語系]][[蒙古語族|蒙古語族]]。该语言使用地區主要为[[甘肅省|甘肅省]][[臨夏回族自治州|臨夏回族自治州]]的[[東鄉族自治縣|東鄉族自治縣]]和[[新疆維吾爾自治區|新疆]][[伊犁哈薩克自治州|伊犁哈薩克自治州]]的[[伊寧縣|伊寧縣]]和[[霍城縣|霍城縣]]。

东乡语有大量汉语、[[突厥语族|突厥语]]、[[波斯语|波斯语]]和[[阿拉伯语|阿拉伯语]]借词<ref>{{cite news |title=大陆东乡语濒危 民间抢救中 |url=https://www.chinatimes.com/cn/realtimenews/20141211004541-260409?chdtv |accessdate=2019-04-06 |work=中时电子报 |date=2014/12/11}}</ref>。以往认为东乡族有语言而无文字<ref>{{Cite journal|title=东乡语保护与研究思考|author=马永峰；马兆熙|url=http://www.cnki.com.cn/Article/CJFDTotal-MZLT201503018.htm|journal=民族论坛|issue=3|doi=|others=|year=2015|volume=|page=|pmid=|quote=|access-date=2019-04-15|archive-url=https://web.archive.org/web/20190415085415/http://www.cnki.com.cn/Article/CJFDTotal-MZLT201503018.htm|archive-date=2019-04-15|dead-url=no}}</ref><ref name=":1" />。2015年，[[甘肃省|甘肃省]]通过对东乡族语言和文字进行首次大规模系统调查，确认了[[东乡族|东乡族]]拥有本民族文字<ref name=":1">{{Cite web|title=调查发现：中国东乡族有文字|url=http://news.cntv.cn/2015/11/23/ARTI1448246064210812.shtml|work=|accessdate=2019-04-13|archive-url=https://web.archive.org/web/20190413134707/http://news.cntv.cn/2015/11/23/ARTI1448246064210812.shtml|archive-date=2019-04-13|dead-url=no|author=|date=|publisher=央视新闻|quote=}}</ref>{{r|中国作家网}}。

== 族群、历史及现状 ==
{{主条目|东乡族}}
东乡语经过了[[古代蒙古語|原始蒙古语]]—[[中古蒙古語|中古西支蒙古语]]—蒙古尔语—东乡语的演变过程。该语言与[[保安语|保安语]]、[[土族语|土族语]]、[[托茂语|托茂语]]关系较近，而与[[蒙古语|蒙古语]]关系稍远<ref>{{Harvnb|新疆维吾尔自治区民族语言文字工作委员会|2002|p=290}}</ref>。

东乡人是一个多起源的民族，目前多认为东乡人起源于中亚的撒尔塔人以及信伊斯兰教的蒙古人<ref>{{Harvnb|谢小冬; 王勋陵; 安黎哲|2002}}</ref><ref>{{Harvnb|马志勇|1983|p=31-48}}</ref>。撒尔塔人是12至13世纪中亚撒尔塔地区信仰伊斯兰教的各种族群的泛称，他们操[[突厥语族|突厥语]]、[[伊朗语支|伊朗语]]等语言，并不说蒙古语族的语言。1219年，蒙古帝国征服撒尔塔地区，将大批撒尔塔人编入军队，撒尔塔人与蒙古军队一同作战，此后蒙古人给予撒尔塔人仅次于蒙古人的优厚待遇。13世纪中后期，撒尔塔人在甘肃河州一带屯垦，转为民户。以这些就地屯垦的撒尔塔人和当地伊斯兰化的蒙古人为主体，融合[[汉族|汉人]]和[[藏族|藏人]]<ref>{{Harvnb|包萨仁|2017|p=26}}</ref>，逐渐于元末明初形成了东乡人。由于撒尔塔人本身语言并不统一，加之周边说不同语言的民族的融入，东乡族逐渐放弃突厥语、伊朗语等，改以蒙古语族语言为共同语<ref>{{Harvnb|新疆维吾尔自治区民族语言文字工作委员会|2002|p=285-286}}</ref>。

甘肃东乡族自治县境内的东乡族多数处于单一语言环境中，其教育状况特别是汉语教育状况较中国其他民族差，[[文盲率|文盲率]]很高，多数学龄前儿童不会说汉语，而大多能掌握东乡语。根据一些学者在2017年于东乡族自治县[[春台乡|春台乡]]的调查，当地东乡人几乎全部会说东乡语，他们把东乡语视为最重要的语言，对本民族语言具有很高的认同感<ref name="调查之三">{{Harvnb|张丽;杨婷|2017|page=91-95}}</ref>。新疆、青海和宁夏也有东乡语使用者分布，其中新疆特别是伊犁一带仍有较大的东乡语社区，这些东乡人主要是清朝平定[[准噶尔汗国|准噶尔汗国]]之后被清政府迁徙而来的。在新疆，聚居较集中的东乡人能掌握一定的东乡语，亦能掌握汉语、维吾尔语、[[哈萨克语|哈萨克语]]和[[锡伯语|锡伯语]]等其他民族语言，而东乡人分布分散的地区则多完全转用汉语。<ref name="新疆语言">{{Harvnb|新疆维吾尔自治区民族语言文字工作委员会|2002}}</ref>

== 音系 ==
===辅音===
东乡语有29个辅音音位，在音节中仅有单辅音而无[[复辅音|复辅音]]。阿·伊布拉黑麦、布和等学者认为东乡语中的塞音塞擦音有清浊对立，而马国良、刘照雄等学者则认为东乡语塞音塞擦音无清浊对立，有送气对立<ref name="声学分析1">{{Harvnb|张瑞珊|2009|page=921}}</ref>。经过声学分析，东乡语中塞音并无浊音，东乡语中的塞音和塞擦音与[[普通话|普通话]]一样都是清音且仅存在送气对立。<ref name="声学分析2">{{Harvnb|张瑞珊|2009|page=922}}</ref>

=== 元音 ===
东乡语有7个单元音{{sfnp|Field|1997|p=37}}和许多复元音，其中{{IPAlink|ə}}与{{IPAlink|ɯ}}可能是同一个音位。东乡语也无长短元音的对立。 <ref name="那德木德元音">{{Harvnb|那德木德|1982}}</ref>

在元音和谐律方面，东乡语拥有阳性、中性和阴性三种元音，阳性元音为{{IPAlink|ɑ}}、中性元音有{{IPAlink|i}}、{{IPAlink|u}}、阴性元音有{{IPAlink|ə}}、{{IPAlink|o}}<ref name="元音和谐现状探析25">{{Harvnb|布和|1983|p=25}}</ref>。与多数蒙古语族语言不同，东乡语的元音和谐并不严整，仅用于少数词汇，因而其[[元音和谐律|元音和谐律]]已经趋向瓦解<ref name="蒙古语族语言概论">{{Harvnb|德力格尔玛，波·索德|2006|p=10-11}}</ref>，有些学者甚至认为东乡语的元音和谐已经消失，仅在构词表现出了一些遗存。<ref name="元音和谐现状探析24">{{Harvnb|布和|1983|p=24}}</ref>

== 词汇来源 ==
东乡语中词汇的来源极为丰富。其中蒙古语族词汇和汉语借词最多，除此之外还有突厥语、波斯语、阿拉伯语和[[藏语|藏语]]借词<ref name="新疆292-293">{{Harvnb|新疆维吾尔自治区民族语言文字工作委员会|2002|p=292-293}}</ref>。由于东乡人的主要来源中亚撒尔塔人，这些撒尔塔人后来转用蒙古语族语言，因而有些学者认为东乡语中的突厥语、波斯语、阿拉伯语词汇并不应该被视为借词，而是东乡语保留下来的底层词汇<ref name="东乡语和东乡人60">{{Harvnb|徐丹; 文少卿; 谢小冬|2012|p=60}}</ref>。由于新疆东乡人长期与维吾尔族等民族杂居，新疆东乡语中有着一定量的[[维吾尔语|维吾尔语]]借词，这也是新疆东乡语与甘肃东乡语的主要区别<ref name="新疆294"/>。

== 方言 ==
东乡语方言口音差异不大，根据语音和词语差异，可将东乡语分为锁南坝土语、汪家集土语和四甲集土语。这三种土语可以互通。其中锁南坝土语分布于东乡族自治县大部分地区以及临夏县、和政县等地，使用人数占东乡语总使用人口的约一半；汪家集、四甲集土语的使用者则分别占三成和两成<ref name="简志土语">{{Harvnb|刘照雄|1981|p=4-5}}</ref>。新疆和甘肃均有三种土语分布<ref name="新疆294">{{Harvnb|新疆维吾尔自治区民族语言文字工作委员会|2002|p=294}}</ref>。
三种土语的差别很细微，相互之前可以完全互通。在语法上三种土语完全一致，在语音上，汪家集土语的部分词汇元音卷舌色彩重。词汇方面，三种土语之间的差异主要是词汇来源不同导致的，如锁南坝土语的的“粘合”一词使用汉语借词、“碎”则使用东乡语固有词，汪家集土语则相反。{{r|简志土语}}

== 文字 ==
东乡语目前主要有东乡语记音符号和“小经”两种书写系统。前一种以拉丁字母书写，是暂定的东乡语官方书写系统，这套记音符号以汉语拼音方案为基础制定并于1994年通过甘肃省少数民族语言文字工作办的认定<ref name=":0">{{Cite web|title=甘肃“抢救”保护东乡族语言|url=http://www.huaxia.com/zhwh/msdl/dl/2014/12/4189924.html|work=www.huaxia.com|accessdate=2019-04-05|author=|date=2014-12-12|publisher=华夏经纬网|archive-url=https://web.archive.org/web/20150330095600/http://www.huaxia.com/zhwh/msdl/dl/2014/12/4189924.html|archive-date=2015-03-30|dead-url=no}}</ref>。但这套文字仅作为临时的记音方案，主要用于编写词典和学校教学<ref>{{Cite web|title=甘肃东乡:境外慈善资金帮助推进汉语东乡语双语教学_网易新闻|url=http://news.163.com/10/0707/09/6AVRCGQB000146BC_all_mobile.html|work=news.163.com|accessdate=2019-04-05}}</ref>，2002年至2012年间。东乡族自治县曾以东乡语记音符号为文字进行东乡语和汉语双语教学实验，但最终因为经费不足等原因而终止<ref>{{Harvnb|马永峰|2018|p=49-51}}</ref>。

在民间，少数东乡人以[[阿拉伯字母|阿拉伯字母]]拼写东乡语，这套文字被称为“土话小经”或“小经”<ref name=":0" />，用在注释经文、书信及记录民间故事、诗歌等各个方面<ref>{{Harvnb|马自祥|2005|p=98-107}}</ref>。在2015年之前，尽管学者们已经意识到了东乡语小经的存在，但多数学者并不认为这是一种成熟文字，自2015年开始，土话小经才引起的学者的注意<ref>{{Cite web|title=调查发现：中国东乡族有文字-新华网|url=http://www.xinhuanet.com/politics/2015-11/23/c_1117226943.htm|work=www.xinhuanet.com|accessdate=2019-04-13|archive-url=https://web.archive.org/web/20190413134704/http://www.xinhuanet.com/politics/2015-11/23/c_1117226943.htm|archive-date=2019-04-13|dead-url=no}}</ref>。“土话小经”有35个字母、9个附加符号和3个标点符号<ref>{{Harvnb|阿·伊布拉黑麦·陈元龙|2015|p=61}}</ref>。土话小经没有经过规范化，没有字母表，书写有很大随意性，存在一字多音或者一音多字的现象。对于阿拉伯语和波斯语的借词，土话小经以原词的书写方法书写，因而存在许多仅仅书写借词的字母。和回族小经类似，东乡语土话小经以音节为单位连写。<ref>{{Harvnb|阿·伊布拉黑麦·陈元龙|2015|p=66}}</ref>

学者推断东乡语土话小经于14、15世纪在[[回族|回族]][[小儿经|小儿经]]的启发下产生<ref name="中国作家网">{{cite web |author1=钱丽花 |title=“小经”文字能否力证东乡族有文字？ |url=http://www.chinawriter.com.cn/mzwy/2015/2015-12-18/260979.html |website=中国作家网 |publisher=中国民族报 |accessdate=2019-04-06 |date=2015-12-18}}</ref>。目前已知最早的以东乡语小经书写的作品为清中前期东乡族宗教学者[[马哈三|马哈三]]的札记<ref>{{Harvnb|阿·伊布拉黑麦·陈元龙|2015|p=63}}</ref>。根据学者于2014年至2015年的调查，有约14.44%的甘肃东乡人能拼读“小经”，其中三分之一能熟练使用<ref>{{Harvnb|阿·伊布拉黑麦·陈元龙|2015|p=62}}</ref>。一些学者正在收集整理土话小经，以期经过规范化后形成一种以阿拉伯字母书写的正式的东乡语文字<ref>{{cite web |title=小经文：东乡族的民族文字 |url=http://www.gxmzb.net/content/2015-12/14/content_11980.htm |website=广西民族报网 |publisher=兰州日报 |accessdate=2019-04-06 |date=2015-12-14}}</ref><ref>{{Cite web|url=http://news.haiwainet.cn/n/2016/0516/c3541083-29925837.html|title=中国多个少数民族正创立文字 东乡族用阿拉伯文|accessdate=2019-4-5|author=|date=2016-05-16|publisher=中新网}}</ref>。但至今东乡语仍然主要作为口语使用，土话小经和东乡语记音符号均不普及，会汉语的东乡人多以汉语为书面语言。有时东乡人也会用同音汉字表示东乡语词汇、甚至用汉字写东乡语对联<ref>{{cite web |title=东乡语对联 |url=http://news.ifeng.com/a/20140809/41506337_0.shtml |website=凤凰网资讯 |publisher=兰州晨报 |accessdate=2019-04-06 |date=2014-08-09}}</ref>。
