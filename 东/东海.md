'''東海'''，亦稱'''東中國海'''、'''中國東海'''，是指[[長江|長江]]出海口以南、[[中國大陸|中國大陸]]以東的大片[[海|海域]]，為[[太平洋|太平洋]]西部的[[陸缘海|陸缘海]]之一。南接[[台灣海峽|台灣海峽]]，北臨[[黃海|黃海]]以长江口北侧与[[韩国|韩国]][[济州岛|济州岛]]的连线为界，東臨[[太平洋|太平洋]]，以[[日本|日本]][[九州_(日本)|九州]]和[[琉球群島|琉球群島]]為界。東海的面積大約是70餘萬[[平方公里|平方公里]]，平均水深在1千餘米，最深處接近[[日本|日本]][[冲繩本島|冲繩本島]]西側的[[中琉界沟|中琉界沟]]（{{lang-ja|'''沖縄トラフ'''}}），約為2,700米。鹽度爲31-32‰，東部爲34‰。海水溫度平均9.2°C。冬季南部水溫在20°C以上。整個海區介于北緯23°00′～33°10′，東經117°11′～ 131°00′之間。

==歷史名稱沿革==
*中國自古以來至[[元朝|元朝]]年間，稱此海域為東海。
*[[明朝|明朝]]時，此海稱為大明海（請參照《[[坤輿萬國全圖|坤輿萬國全圖]]》）
*元明之後，華夏大陸漸有中土之稱，此海域漸稱為[[東中國海|東中國海]]直到現在。
*現下周邊國家各有表述，中國稱東海，[[韓國|韓國]]稱南海，昔[[琉球國|琉球尚氏]]稱琉球海。[[日本|日本]]在二次大戰前普遍以{{lang|ja|東支那海}}表記，現今則使用{{lang|ja|東シナ海}}，將支那一詞改以片假名表示。

== 地理資訊 ==
[[黑潮暖流|黑潮暖流]]（日本暖流）在[[台灣|台灣]]東側迴流北上至[[日本列島|日本列島]]，根據調查研究，該暖流有一分支即[[台灣暖流|台灣暖流]]在[[巴士海峽|巴士海峽]]進入台灣海峽而後北上進入東海，另有一分支（被中國学者命名為[[黄海暖流|黄海暖流]]）進入東海北部，影響到黄海。

與[[渤海|渤海]]和黃海相比，東海有較高的水溫和較大的[[鹽度|鹽度]]，潮差6米—8米，水呈藍色。又因東海屬於[[亚热带|亞熱帶]]和[[温带|溫帶]]氣候，利於[[浮游生物|浮遊生物]]的繁殖和生長，是各種魚蝦繁殖和棲息的良好場所，也是中國周邊海洋生産力最高的海域。

東海的優良港灣很多，如上海港位於長江下游[[黄浦江|黃浦江]]口，[[航道|航道]]深闊，水量充沛，江內風平浪靜，適合巨輪停泊。

中國大陸[[廣東省|廣東省]][[南澳島|南澳島]]與台灣南端鵝鑾鼻的連線是東海與[[南海|南海]]的分界線。

東海的海灣以[[杭州灣|杭州灣]]最大，流入東海的河流有長江、[[钱塘江|錢塘江]]、[[闽江|閩江]]及[[濁水溪|濁水溪]]等。中國流入東海的江河，長度超過百公里的河流有40多條，其中長江、錢塘江、歐江、閩江等四大水系是注入東海的主要江河。因而，東海形成一支巨大的低鹽水系，成爲中國近海營養鹽比較豐富的水域，其鹽度在34‰以上。因東海位隅亞熱帶，年平均水溫20℃—24℃，年溫差7℃—9℃。

東海66℅的面積爲[[大陸棚|大陸棚]]，寬度超過600公里。東海的東部爲弧狀的大陸坡與[[沖繩海槽|沖繩海槽]]，最深處在台灣島東北部，深達2322米。大陸坡長1100公里，平均寬35公里。遠離大陸的大陸坡上發育有衆多深切大陸坡的海底峽谷，其規模往往超過陸地上河流形成的大峽谷。在中國東海的中、南部大陸坡上，已發現有超過14條規模不等的海底峽谷，長10—50公里，寬1—15公里，下切深度50—500米，平面形狀呈鵝掌形、樹枝形或蛇曲形，有的海底峽谷一直延伸到沖繩海槽底。

== 自然資源 ==
廣闊的東海大陸棚海底平坦，過往水質優良，又有多種水團交彙，爲各種魚類提供良好的繁殖、索餌和越冬條件。東海是中國最主要的良好[[漁場|漁場]]，曾盛産[[大黄鱼|大黃魚]]、[[小黄鱼|小黃魚]]、[[带鱼科|帶魚]]和[[墨斗魚|墨魚]]，但該四種魚類現在已因[[過度捕魚|過度捕魚]]而在東海瀕臨滅絕<ref name=people2013>{{Cite web |url=http://finance.people.com.cn/GB/n/2013/0417/c1004-21163613.html |title=东海无鱼可捕虾也快被电完 专家采标本11天无收 |website=人民網 |date=2013-04-17}}</ref>。[[舟山群島|舟山群島]]附近的漁場曾被稱爲中國海洋魚類的寶庫。但是到2010年代環境污染和過度捕撈已引致東海漁場出現水域荒漠化<ref>{{Cite web |url=http://env.people.com.cn/GB/n1/2016/0330/c1010-28236964.html |title=环境污染和过度捕捞导致“东海无鱼” |website=人民網 |date=2016-03-30}}</ref>。

在東海大陸架上蘊藏著極爲豐富的[[石油|石油]]資源。中國從1974年開始就在東海進行石油、[[天然氣|天然氣]]勘測，並發現了多個油田，在東海陸架盆地西湖凹陷區域有[[春晓油气田|春曉]]、平湖、殘雪、斷橋、天外天等油氣田，占地面積達2.2萬平方公里。探明的天然氣儲量達700億立方米以上。

== 主權爭端 ==
在东中国海[[大陆架|大陆架]]上蕴藏着极为丰富的石油资源，而且是[[中华人民共和国|中華人民共和国]]（中國大陸）、[[日本國|日本國]]、[[大韓民國|大韓民國]]、[[中華民國|中華民國]]（[[臺灣|臺灣]]）等各方的[[海洋|海洋]][[战略|战略]]要地。该海域存在着台湾、日本、中國大陸三方的[[釣魚臺列嶼|釣魚臺列嶼]]（日本稱之為'''尖阁诸岛'''）及附近海域归属问题；以及中日[[东海油气田问题|东海油气田问题]]；中韩[[专属经济区|专属经济区]]划界问题和[[苏岩礁|苏岩礁]]（'''离於岛'''）问题。
