{{Vgname|東京幻都|ja=東亰ザナドゥ|en=Tokyo Xanadu|aka=东京Xanadu|cn=东京迷城|tw=東京幻都}}是[[日本Falcom|日本Falcom]]公司制作的[[动作角色扮演游戏|动作角色扮演游戏]]，开发于[[PlayStation_Vita|PlayStation Vita]]平台，於2015年9月30日发售。

在尚未有官方中文版推出之前，曾經在網路上被玩家譯為「東京迷城」或「東京Xanadu」，玩家命名由來是2006年被譯為{{Vgname|迷城國度|en=Xanadu Next|}}這款由[[英特衛|英特衛]]代理中文版的遊戲譯名。

2016年8月5日宣布推出中文版，分別是PSV版本的《東京幻都》，與增加新劇情與要素的[[PS4|PS4]]版本的《東京幻都 eX+》，将於2017年3月30日推出。<ref>PS Vita / PS4《東京幻都》確定將在 2016 年冬季推出繁體中文版 http://gnn.gamer.com.tw/3/135593.html</ref>

2017年12月8日推出[[Windows|Windows]]版的《東京幻都 eX+》。

== 概要 ==
本作是以Falcom公司初期名作《{{link-ja|Xanadu|ザナドゥ (ゲーム)}}》為命名靈感的新系列遊戲，也是创社以来首个以现实世界为背景的游戏。
标题中“京”字故意写作古字“亰”，是为了突出不协调感。
本作是Falcom第一次使用的正式動畫作為片頭，由3hz製作<ref>http://dengekionline.com/elem/000/001/050/1050463/ 【電撃PS】『東亰ザナドゥ』OPアニメは“STUDIO 3Hz”が担当！ 5人目の適格者“北都美月”の情報も公開に 電撃オンライン 2015年5月13日</ref>。

== 世界觀 ==

本作以东京郊外的架空城市“杜宫市”为舞台，描写了在异界迷宫“Xanadu”中战斗的数名[[高中生|高中生]]的故事。“杜宫市”的原型为Falcom公司所在地[[立川市|立川市]]及[[多摩地域|多摩地域]]。

== 故事 ==

故事的男主角，「時坂 洸」，在某個夜晚打工後回家時看見自己的剛剛四月份從美國轉來的同學在洸班上擔任同班兼班長，也就是故事女主角，「柊 明日香」被幾名不法分子纏上了，主角洸想英雄救美，跟著明日香到了一座天橋下，看見幾名不法分子真集體欺負著明日香，這時，天橋底下突然冒出一到紅色之門，也就是通往異界的門，幾名不法分子，明日香與洸都被捲入這道門內，在這道門的另一個世界，洸發現明日香並不是一個普通的學生，明日香擁有著洸無法想象的特殊力量。從此開始，洸也因此而改變了自己的日常生活，在明日香的帶領下，解決一道道的異界之門……
