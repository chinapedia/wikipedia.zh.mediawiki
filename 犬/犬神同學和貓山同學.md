《'''犬神同學和貓山同學'''》（{{lang|ja|犬神さんと猫山さん}}），是{{lang|ja|[[葛城一|くずしろ]]}}作畫的[[日本|日本]][[百合_(文化)|百合系]][[四格漫畫|四格漫畫]]作品，於[[一迅社|一迅社]][[漫畫雜誌|漫畫雜誌]]《[[Comic百合姬|Comic百合姬]]》2012年1月號（2011年11月18日發售）連載至2017年4月號（2017年2月18日發售）。

== 概要 ==
犬神八千代是貓控，貓山鈴是犬控。這兩個名字與興趣完全不符的女孩，在春意盎然的一天邂逅了。而由於對方長相氣質與自己的興趣十分吻合，她們兩人也很快進入了一種卿卿我我的境界……

== 登場人物 ==
; {{nihongo|犬神八千代|犬神 八千代（いぬがみ やちよ）|Inugami Yachiyo}}
: 配音員：[[上坂堇|上坂堇]]
: 主人角。16歲。1年A班。犬似的貓派。抖M。成績很好，擅長數學。

; {{nihongo|貓山鈴|猫山 鈴（ねこやま すず）|Nekoyama Suzu}}
: 配音員：[[東山奈央|東山奈央]]
: 16歲。1年B組。猫似的犬派，容易害羞的傲嬌少女。
: 與大學二年級的姐姐一起生活。

; {{nihongo|柊木秋|柊木 秋（ひいらぎ あき）|Hiiragi Aki}}
: 配音員：[[大坪由佳|大坪由佳]]
: 1年B組。16歲。屬於劍道部。犬神和貓山的朋友，國中1年級時就已經認識犬神，而高中時與貓山同班。擅長國語。
: 小時候有遭狗咬傷及貓抓傷的慘痛經歷，因此現在不擅長應付動物。

; {{nihongo|杜松美希音|杜松 美希音（ねず みきね）|Nezu Mikine}}
: 配音員：[[堀野紗也加|堀野紗也加]]
: 1年A組。小動物系。屬於生物部，髮型是雙馬尾，身高跟貓山鈴一樣嬌小，因此同樣受到八千代的喜愛。

; {{nihongo|牛若雪路|牛若 雪路（うしわか ゆきじ）|Ushiwaka Yukiji}}
: 配音員：藤本葵
: 2年級生。生物部部長，個性溫柔的療癒系女孩。小時候就認識杜松。不擅長對付紅色的東西。

; {{nihongo|猿飛空|猿飛 空（さるとび そら）|Sarutobi Sora}}
: 配音員：{{tsl|ja|長弘翔子}}
: 1年A組班長。戴著眼鏡，髮型是茶色波浪長髮。視犬神為對手。

; {{nihongo|鳥飼雲雀|鳥飼 ひばり（とりかい ひばり）|Torikai Hibari}}
: 配音員：[[山崎惠理|山崎惠理]]
: 1年B組。由於病弱所以長期住院，與猿飛是青梅竹馬。

; {{nihongo|貓山珠喜|猫山 珠喜（ねこやま たまき）|Nekoyama Tamaki}}
: 配音員：[[原田瞳|原田瞳]]
: 大學2年生。貓山鈴的姐姐。

; {{nihongo|虎生真梨|虎生 真梨（とらお まり）|Torao Mari}}
: 學生會會長。外表漂亮，不過毒舌，性格刻薄，但是粉絲很多。喜欢棒球。

; {{nihongo|龙崎乙姬|龍崎 乙姫（りゅうさき おとめ）|Ryuusaki Otome}}
: 學生會副會長。俊脸美人，在女校中如同王子一样的存在，和虎生一样粉丝很多，喜欢棒球。

== 電視動畫 ==
在2014年1月發售的「漫畫百合姬」2014年3月號中提及，發表本作品短動畫化。於同年4月10日起在日本[[埼玉電視台|埼玉電視台]]、[[神奈川電視台|神奈川電視台]]、[[京都放送|京都放送]]、[[SUN電視台|SUN電視台]]等放送，包含[[NICONICO|NICONICO]]網路直播放送。

動畫正式放送（全12話）的1週前（2014年4月3日），播放聲優出演的宣傳節目。
