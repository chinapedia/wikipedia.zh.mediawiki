'''骚乱'''是[[社会|社会]]不同[[阶层|阶层]]之间或阶层内部的冲突与对立，包括社会底层为争取自身权益与统治阶层的对抗，或因统治阶层不当行为激发民怨，引发的与统治阶层冲突与对立。长时间的持续对立，可能会使冲突范围扩大化，冲突的升级可能诱发暴动，導致政局动荡与战争，最终导致[[政权更迭|政权更迭]]。

骚乱通常是[[隨機|隨機]]發生並混亂無[[秩序|秩序]]的，對象可以是政府官員、工廠老闆、政党，導火線通常是不特定的事件，例如[[阿拉伯之春|阿拉伯之春]]中[[穆罕默德·布瓦吉吉|穆罕默德·布瓦吉吉]]的自焚。骚乱背後皆有其動機：對整個大環境的不滿。當骚乱次數越趨頻繁，規模漸大，群眾漸增。骚乱開始時是有預謀的行動時就導致叛亂，但骚乱亦可以是由有秩序的示威遊行演變而成。

== 其他稱呼 ==
* 暴亂、暴動：帶犯罪意味，用在暴徒失控而使用暴力的情況。如[[非法集会|非法集会]]人士[[破壞社會安寧|破壞社會安寧]]，即屬“暴動”<ref>{{Cite web|url=https://news.now.com/home/local/player?newsId=168303|title=【新聞智庫】暴亂或騷亂|accessdate=2019-01-01|work=news.now.com|language=zh-hk}}</ref>。
* 群体事件：中國政府有時使用的委婉詞，淡化事件的嚴重性。
* 打砸抢烧：中國政府有時使用這個詞，將參與者指涉為犯罪份子。
* 起義：帶正面意味，指參與者是正義的一方，要對抗不正義的國家、黨政、企業或既有利益者。

== 特点 ==
* 数量多，规模较大。
* 涉及的阶层行业多，主体成分多元化。
* 群体性事件的指向对象不同，内容不同。
* 表现方式激烈，内部矛盾逐渐分化或对抗化。
* 组织程度逐渐提高，[[矛盾|矛盾]]逐渐趋向政治化。
* 各种[[矛盾|矛盾]]相互交织，处置难度加大。

==可能原因==
===深层原因===
* 社会[[贫富差距|贫富差距]]拉大。
* 职工经济利益和[[民主|民主]]权利受到侵犯。
* [[官僚主义|官僚主义]]作风和[[腐败|腐败]]现象造成干群关系紧张。
* 信息公开不透明，群众[[知情权|知情权]]受限。
* 人们的思想意识和价值观念日趋多元化、复杂化。

===直接原因===
一般由某个小型事件使群众性情绪煽起导致多层矛盾急剧激化，如[[瓮安事件|瓮安事件]]、[[石首事件|石首事件]]。即使是个人[[自杀|自杀]]事件，仍可能因家属的不满而诱发骚乱，如[[京温事件|京温事件]]和[[2015年12月永昌县少女跳楼事件|永昌县少女跳楼事件]]。

== 后果 ==
* 民众集体温和的抗议（如[[游行|游行]][[示威|示威]]、群体[[请愿|请愿]]行为）。
* 局部、小范围与统治当局一般性对立，表现为一般性冲突。
* 大规模的冲突。可以为局部大规模冲突事件，或因此引发全局性冲突、骚乱或直至与执行当局的武装对立。武装对立可以引发政局动荡，持续长时间骚乱事可以导致战争，直至政权更迭。例如中国历史上绝大多数朝代更迭都是从群体性事件引发民变导致。

== 各地群体性事件 ==
=== 中国大陆 ===
[[中国历史|中国历史]]上绝大多数朝代更迭都是从小规模事件开始导致[[民变|民变]]最终引发改朝换代。

仅2009年，中国全国共发生近9万起各类群体性事件，而这些事件均与[[官民冲突|官民冲突]]有关。[[中华人民共和国政府|中华人民共和国政府]]及官员造成的官民矛盾被认为是事件产生的直接诱因<ref>{{Cite journal |url = http://www.china.com.cn/book/zhuanti/qkjc/txt/2010-01/13/content_19228539.htm |title = 群体性事件症结在于官民矛盾 |accessdate = 2013-07-21 |author = [[于建嵘|于建嵘]] |journal = 《[[中国报道_(杂志)|中国报道]]》 |year = 2010年 |issue = 第1期 |language = zh-Hans }}</ref><ref>{{Cite journal |url = http://news.qq.com/a/20081227/000820.htm |title = 2008年恶性事件密集发生 官民矛盾步入显化阶段 |accessdate = 2013-07-21 |author = 杨军（《[[南风窗|南风窗]]》记者 ） |date=2008年12月27日 |journal = 《[[南风窗|南风窗]]》 |language = zh-Hans }}</ref>。
