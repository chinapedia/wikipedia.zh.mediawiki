〈'''心跳☆早晨'''〉（[[英語|英]]：Doki Doki☆Morning、[[日語|日]]：ド・キ・ド・キ☆モーニング）是[[BABYMETAL|BABYMETAL]]的第一個作品，形式為DVD單曲。2011年10月24日由重音部RECORDS發行。

== 發行 ==
DVD單曲分為重音部版本與BABYMETAL版本，販售內容皆為DVD加上印有樂團標誌的[[毛巾|毛巾]]。2011年10月22日與23日於{{link-ja|Mt.RAINIER HALL SHIBUYA PLEASURE PLEASURE|Mt.RAINIER HALL SHIBUYA PLEASURE PLEASURE}}舉行的「櫻花學院祭☆2011」會場發售，10月24日[[亞馬遜日本|亞馬遜日本]]在[[淘兒唱片|淘兒唱片]]新宿店限量發售<ref>{{cite web|url=http://www.sakuragakuin.jp/news/|title=重音部 BABYMETAL リリース情報|publisher=さくら学院 公式サイト 連絡板|accessdate=2015-08-05|language=ja}}</ref>。2011年4月27日《{{link-ja|櫻花學院2010年度 ～message～|さくら学院 2010年度 〜message〜}}》收錄此曲。10月24日以單曲形式發行，11月2日在全球[[iTunes|iTunes]]發行數位音樂下載<ref>{{cite  news|url=http://tower.jp/article/feature_item/2011/10/07/0701|title=BABYMETAL、テビュー曲のミュージック・クリップとタオルがセットに|newspaper=TOWER RECORDS ONLINE|publisher=[[淘兒唱片|淘兒唱片]]|date=2011-10-12|accessdate=2013-11-26|language=ja}}</ref>。2012年4月6日發行CD形式單曲，2014年8月5日在[[亞馬遜公司|亞馬遜數位商店]]上架數位版單曲。

== 音樂類型 ==
融合{{link-pt|口香糖音樂|Bubblegum pop}}和重金屬的實驗曲，旋律琅琅上口，除樂器以外是典型的偶像音樂。主題是關於少女早上賴床來不及出門的日常生活<ref>{{Cite web|url=http://www.squabblebox.co.uk/2014/07/review-babymetal-babymetal-2014/|title=REVIEW: Babymetal – Babymetal (2014)|work=SquabbleBox|date=2014-07-16|accessdate=2016-08-20|language=en}}</ref><ref name ="jcast20111216">{{cite news|url=http://www.j-cast.com/2011/12/16116731.html|title=12〜13歳女の子の「ヘビメタアイドル」「さくら学院 BABYMETAL」に世界が大注目|newspaper=J-CASTニュース|publisher=株式会社ジェイ・キャスト|date=2011-12-16|accessdate=2016-01-11|language=ja}}</ref>。

== 音樂錄影帶 ==
影片由田中紫紋執導<ref>{{cite web|url=http://qreators.jp/qreator/tanakashimon|title=田中紫紋|work=QREATORS|publisher=株式会社QREATOR AGENT|accessdate=2015-08-05|language=ja}}</ref>，2011年10月12日於[[YouTube|YouTube]]上的[[TOY'S_FACTORY|TOY'S FACTORY]]官方頻道首先上傳<ref>{{cite web|url=http://www.youtube.com/watch?v=4QbAXXXOJF8|title=BABYMETAL - ド・キ・ド・キ☆モーニング Doki Doki☆Morning (Edit ver.)|publisher=[[YouTube|YouTube]]|date=2011-10-12|accessdate=2016-01-11|language=ja}}</ref>。2012年11月8日BABYMETAL官方頻道也上傳了該音樂錄影帶<ref>{{cite web|url=http://www.youtube.com/watch?v=cirhQ8iLdbw|title=BABYMETAL - ド・キ・ド・キ☆モーニング - Doki Doki☆Morning (OFFICIAL) |publisher=BABYMETAL（[[YouTube|YouTube]]）|date=2012-11-08|accessdate=2016-01-11|language=ja}}</ref>。美國重金屬媒體{{link-en|Metal Injection|Metal Injection}}將它列為「2011年15大熱門話題」中的第9位<ref name ="jcast20111216"/>。

== 現場表演 ==
2010年11月28日在[[橫濱紅磚倉庫|橫濱紅磚倉庫]]1館3樓大廳舉行的「櫻花學院祭☆2010」首次表演，同時這也是BABYMETAL三名成員第一次公開演出，當時年紀最大的[[中元鈴香|SU-METAL]]還沒過13歲生日<ref>{{cite news|url=http://www.cdjournal.com/main/news/sakura-gakuin/35386|title=さくら学院、初単独イベント＜さくら学院祭☆2010＞が大成功！|newspaper=CDJournal|publisher=株式会社音楽出版社|date=2010-11-30|accessdate=2016-01-11|language=ja}}</ref><ref>{{cite web|url=http://www.oricon.co.jp/music/special/2010/sakuragakuin1208/|title=さくら学院『かけがえのない青春が炸裂した!! 1日限りの学院祭』|work=ORICON STYLE|publisher=ORICON|author=斉藤貴志|date=2010-12-08|accessdate=2016-01-11|language=ja}}</ref>。

2016年時SU-METAL回憶說：「我清楚的記得當時在表演中段...我們都要假裝趴在地上賴床......我很怕觀眾會笑我們，這樣很令人尷尬。但他們的反應讓我感到高興，那個瞬間我才意識到這就是SU-METAL、這就是BABYMETAL。」<ref>{{cite web|url=http://www.fuse.tv/videos/2016/05/babymetal-first-performance-fox-god-interview|title=Babymetal Talk Their First Performance Ever, Meeting the Fox God: Watch - Fuse |work={{link-en|Fuse (TV channel)|Fuse}}|date=2016-05-29|accessdate=2016-05-30|language=en}}</ref>
