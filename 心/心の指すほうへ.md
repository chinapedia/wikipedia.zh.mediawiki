'''心の指すほうへ'''是日本乐队[[Grram|Grram]]的[[迷你專輯|迷你專輯]]。

== 概述 ==
“悲しいほど 今日の夕陽 きれいだね”被選入[[讀賣電視臺|讀賣電視臺]]·[[日本電視臺|日本電視臺]]的動畫「[[名偵探柯南|名偵探柯南]]」的主題曲。<ref>[http://www.barks.jp/news/?id=1000076506 久川実津紀を擁する新人バンドgrramに熱い視線] ''BARKS''</ref><ref>[http://www.oricon.co.jp/news/music/2006248/full/ デビュー前バンド・grram、『コナン』ED曲抜てき] ''ORICON STYLE''</ref><ref>[http://www.hotexpress.co.jp/news/120126_grram/ 「名探偵コナン」EDテーマに、デビュー前の新星バンドを抜擢] {{webarchive|url=https://web.archive.org/web/20131020170105/http://www.hotexpress.co.jp/news/120126_grram/ |date=2013-10-20 }} ''hotexpress''</ref>。[[Chicago_Poodle|Chicago Poodle]]的[[花沢耕太|花沢耕太]]首次參與包括主打歌的歌曲錄製<ref>[http://chicagopoodle.com/information/#news 花沢耕太、楽曲提供決定!!] {{webarchive|url=https://web.archive.org/web/20130721012304/http://chicagopoodle.com/information/ |date=2013-07-21 }} ''シカゴプードルオフィシャルホームページ''</ref>。
