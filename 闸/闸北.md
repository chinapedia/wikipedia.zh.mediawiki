'''闸北'''，亦曾名为'''沪北'''，上海地区名，位于[[吴淞江|吴淞江]]下游中段北岸地区。原指新闸至老闸一线以北地区，含今[[闸北区|闸北区]]及[[普陀区|普陀区]]等部分地区，后随闸北区的成立及其辖境的扩张，泛指该区。

==名源==
1672年，为了阻挡黄浦江的潮水倒灌入吴淞江上游，故在吴淞江上今福建路位置设置了一座挡潮石闸桥。1735年，原闸桥坍毁后，在其上游方向今[[大统路_(上海)|大统路]]附近新建了一座闸桥，而原闸桥被称为老闸，新建闸桥则称为新闸。对于老闸至新闸一线的吴淞江以北地区，因此得名为闸北<ref>{{cite web |url = http://www.shtong.gov.cn/newsite/node2/node2245/node70862/node70869/node70914/node70918/userobject1ai73711.html |language=zh |title=上海地名志 >> 第三篇居民地地名 >> 第二章地片、区片 >> 第一节地片  |publisher = 上海地方志办公室 |accessdate = 2012-11-28 }}</ref>。

== 地区发展 ==
[[上海开埠|上海开埠]]前，闸北地区以[[虬江|虬江]]为界，虬江以南因水路交织，临近吴淞江，并有闸桥通往[[上海城墙|上海县城]]，所以逐渐形成市集，而[[虬江路|虬江]]以北则多为农田和自然村。1850年代起，随着租界的辟立和[[太平天国运动|太平天国运动]]，导致大批外地移民涌入上海。其中一些贫民因负担不了租界内的居住成本，所以选择临近租界的闸北地区落户，最初有梅园头村和唐家弄等数个居民点。此外，由于当时沪上客货多仰赖上游[[苏州|苏州]]等地依吴淞江运抵，闸北沿河地区逐渐成为农产品的集散地，市面逐渐繁荣。同时境内有乡村道路一直通往[[宝山县|宝山县]]内的[[江湾镇|江湾]]、[[真如镇|真如]]等市镇<ref>{{cite web|url=http://www.shtong.gov.cn/newsite/node2/node4/node2249/zabei/node38956/index.html |language=zh |title=区志>>闸北区志>>总述 |publisher = 上海市地方志办公室 |accessdate = 2012-11-28 }}</ref>。

1864年起，租界当局开始在苏州河上兴建桥梁，随后又兴办铁路等，扩张形势明显。为了抵御租界的大规模扩张，当时粤籍商人[[陈绍昌|陈绍昌]]、浙商[[祝承桂|祝承桂]]联合地方士绅，呈请[[两江总督|两江总督]]允许闸北地区开为商埠。在得到核准后，士绅组成[[闸北工程总局|闸北工程总局]]，负责境内市政建设。加之[[沪宁铁路|沪宁铁路]]和[[淞沪铁路|淞沪铁路]]的建设、长途公路的建造，闸北迅速形成沪上重要的交通枢纽，吸引大批官商前来投资。至1920年代，闸北地区已经形成上海的重要工业基地和文化重镇<ref>{{cite web |url = http://www.shtong.gov.cn/newsite/node2/node2245/node69543/node69547/index.html |language=zh |title=上海建筑施工志>>大事记 |publisher=上海市地方志办公室 |accessdate=2012-11-28 }}</ref>。

但至[[上海1930年代|1930年代]]，由于[[一二八|一二八]]和[[淞沪会战|淞沪会战]]两次战争，闸北成为战场几乎沦为一片废墟，逐渐成为难民聚集区，并成为上海著名的贫民窟。自1980年起，随着大规模城市改造的开始，闸北地区拆除了大批危房，并以[[上海站|上海站]]为中心重建了商业区，并在科教方面着重投入，基本实现了该地区的商贸文教等的复兴<ref>{{cite journal |author=张笑川 |url = http://www.cnki.com.cn/Article/CJFDTotal-SZTD200902018.htm |language=zh |title = 战争与近代上海闸北城区的变迁 |journal=苏州科技学院学报（社会科学版） |date=2009 |accessdate=2012-11-28 }}</ref>。
