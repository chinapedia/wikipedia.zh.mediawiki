'''疣豬'''（[[學名|學名]]：'''''{{lang|la|Phacochoerus africanus}}'''''），屬[[偶蹄目|偶蹄目]][[豬科|豬科]][[疣豬屬|疣豬屬]]，主要分布於[[非洲|非洲]]，[[熱帶雨林|熱帶雨林]]和[[北非|北非]][[沙漠|沙漠]]以外之地。

== 居住環境 ==
[[File:_Phacochère9.jpg|thumb]]
為世界上少數能夠在只有些許水的環境下，依舊能夠存活的[[動物|動物]]。能在超過[[常溫|常溫]]許多的[[高溫|高溫]]下存活。

== 特徵 ==
[[File:Two_Warthogs_Phacochoerus_africanus_grazing_in_Tanzania.ogv|thumb]]
成年疣豬體色會有些許不同。除了背部長著一道灰色鬃毛及兩頰外，身上的毛都很稀疏。頭约占身體的三分之一，且有兩對[[疣|疣]]及[[獠牙|獠牙]]，約15～25吋長。體重約75－100公斤。[[壽命|壽命]]約15年。[[懷孕|懷孕]]期約175[[天|天]]。

=== 雌雄辨別 ===
公的比母的體型略大，獠牙比母的長，最大的區別是公的有兩對疣（在挖土取食時，有助於保護眼睛），而母的只有一對疣，沒有嘴附近的那一對。

== 習性 ==
日間覓食，洗[[泥巴|泥巴]]澡，會像[[犀牛|犀牛]]那樣渾身塗滿泥巴，用以消暑、降溫和消滅身上[[寄生蟲|寄生蟲]]。在[[懷孕|懷孕]]期間，母疣豬會趕走身邊的小疣豬，將自己隔離起來。

群居動物，牠們經常會組成一個小型家庭團隊，主要是由母疣豬和小疣豬組成。公疣豬自己生活，只有在交配期間才會加入群體。在爭奪母疣豬的儀式中，公疣豬之間的搏鬥非常兇殘和血腥，用獠牙直接進攻，頭與頭撞擊。

== 食物 ==
主要為[[青草|青草]]、[[苔草|苔草]]及[[塊莖植物|塊莖植物]]，偶而也會吃一些[[腐肉|腐肉]]。
