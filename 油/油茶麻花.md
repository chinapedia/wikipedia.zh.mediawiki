'''油茶麻花'''，也称'''油茶泡麻花'''或'''麻花油茶'''，是一种[[西安|西安]]坊间的传统早餐，将[[麻花|麻花]]煮于[[炒面_(面粉)|油茶]]中食用。油茶麻花是唯一将油茶与麻花组合在一起的食物，也是西安人的传统早餐。<ref>{{Cite web |url= http://www.sxdaily.com.cn/n/2013/0507/c388-5126878.html |title= “北式早茶”甲天下：西安人最爱的十大经典早餐 |author=  |date= 2013年5月7日 |publisher= 陕西传媒网/腾讯网 |language= zh-hans |accessdate= 2013年8月23日 |quote=  }}</ref>麻花油茶是西安的油茶品种之一，其他品种还有疙瘩油茶、杏仁油茶、壶壶油茶等。<ref>{{Cite web |url= http://xuewen.cnki.net/read-R2009110010004966.html |title= 西安油茶 |author=  《陕西百科全书》编委会 |date=  |publisher= 知网/陕西百科全书 |language= zh-hans |accessdate= 2013年8月23日 |quote=  }}</ref>

==特点==
油茶麻花的做法是将面粉用清油炒熟，配上杏仁、花生、黄豆，用开水兑开，取炸好的新鲜麻花泡入<ref>{{Cite web |url= http://www.sn.xinhuanet.com/2009-02/25/content_15794050.htm |title= 西安小吃之油茶 |author=  |date= 　2009-02-25 |publisher= 新华陕西旅游网 |language= zh-hans |accessdate= 2013年8月23日 |quote=  |deadurl= yes |archiveurl= https://web.archive.org/web/20120508044747/http://www.sn.xinhuanet.com/2009-02/25/content_15794050.htm |archivedate= 2012年5月8日 }}</ref>，口感绵软芳香，既不油腻还有营养。<ref>{{Cite web |url= http://news.sina.com.cn/c/2012-10-02/082425293857.shtml |title= 十一游西安 西部网每天推荐一套地道美食（二） |author=  |date= 2012年10月2日 |publisher= 新浪网/西部网 |language= zh-hans |accessdate= 2013年8月23日 |quote=  }}</ref>

==渊源==
