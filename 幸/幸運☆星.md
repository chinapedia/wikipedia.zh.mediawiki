《'''幸運☆星'''》是[[日本|日本]][[漫畫家|漫畫家]][[美水鏡|美水鏡]]所繪的一系列[[四格漫畫|四格漫畫]]及改編而成的[[電子遊戲|電子遊戲]]及[[電視動畫|電視動畫]]。原作漫畫從2004年1月在[[角川書店|角川書店]]《[[Comptiq|Comptiq]]》連載，另外也在《[[少年ACE|少年ACE]]》等雜誌上連載（參見[[#連載雜誌|連載雜誌]]）。

== 概要 ==
* 原本不是預定連載的作品，而是填滿雜誌空頁的應急作；但受到廣大的歡迎，因而正式開始連載。

* 2005年8月24日發售[[廣播劇|廣播劇]][[CD|CD]]。接著於2005年12月1日發售《幸運☆星 可愛少女學園》[[任天堂DS|任天堂DS]]遊戲，續作為2007年的《真・幸運☆星 可愛少女學園~啟程~》[[任天堂DS|任天堂DS]]遊戲，並在2008年推出第一款在[[PlayStation_2|PlayStation 2]]平台上的遊戲《幸運☆星 ~陵櫻学園 櫻藤祭~》。在2006年10月也公佈由[[京都動畫|京都動畫]]製作電視動畫版，OVA則是在2008年以DVD型式發售。

* 《[[天漫|天漫]]》2012年1月號宣佈[[天聞角川|天聞角川]]將在[[中国大陆|中国大陆]]推出單行本並於同年3月號起與日本同步連載<ref>[http://www.gztwkadokawa.com/?article-264.html 《天漫》3月号激增至400页！]</ref><ref>[http://bbs.gztwkadokawa.com/thread-427663-1-1.html 3月《天漫》超特厚号新鲜送达！有真相] {{webarchive|url=https://web.archive.org/web/20160304133306/http://bbs.gztwkadokawa.com/thread-427663-1-1.html |date=2016-03-04 }}</ref>。

== 故事簡介 ==
描寫日本[[埼玉縣|埼玉縣]]的陵櫻[[高中|高中]]裡面的幾位普通的高中女生，她們平常的悠閒生活（不包括戀愛要素）。連載開始時，主要的四位登場人物都是高中一年級學生。目前她們已讀大學。

== 舞台 ==
; 陵櫻學園高中部
: 此方等人就讀的學校，以作者的母校（{{link-ja|春日部共榮高等學校|春日部共栄中学高等学校}}）為原型。一共有十三個班級<ref>官方導覽本，第103頁。</ref>。
: 女生的夏季制服是淺藍色，冬天則是紅色系的[[日本學生服|水手服]]（与《[[To_Heart|To Heart]]》中的女生校服只有领口装饰上的差异）。男生則是[[立領學生服|立領學生服]]。
: 雖然有游泳池，但沒有游泳課。
: 入學考試的結果不公佈在學校內，而是直接寄送到考生的家中。
; 柊家
: 鷹宮[[神社|神社]]（動畫版是以[[埼玉縣|埼玉縣]][[北葛飾郡|北葛飾郡]][[鷲宮町|鷲宮町]]（現・[[久喜市|久喜市]]鷲宮）的[[鷲宮神社|鷲宮神社]]為雛形）。
: 使用舊型的黑色電話機。與[[無線電話|無線電話]]和[[來電顯示|來電顯示]]無緣。（注：服務開始當初有推出轉接器讓黑色電話機也可以使用[[來電顯示|來電顯示]]服務，但是現在已經停止販售）
; [[秋葉原|秋葉原]]
: 日本最常聚集[[御宅族|御宅族]]之地，此方常常來這裡的GAMERS和[[Animate|Animate]]購物。另外，此方也在這裡的某家[[Cosplay|Cosplay]][[咖啡廳|咖啡廳]]打工。
; [[攝影棚|攝影棚]]
: Lucky Channel的錄製現場。主持就是小神晶。有兩代助手，分別為[[白石稔|白石稔]]和[[小野大輔|小野大輔]]。
: 在動畫第21話中，白石稔終於不能忍受小神晶的黑暗一面和小野大輔搶去他助手之位，終於失控，結果錄製現場混亂，桌子、椅子被打翻，佈景板被破壞。

本作播出後，片頭中小鏡登場的[[鷲宮神社|鷲宮神社]]在劇中設定成是柊家的神社，爾後於一瞬之間，參訪鷲宮神社的人突然多了起來，令鷲宮神社附近的居民一時之間變得緊張兮兮。但最後鷲宮神社也利用[[聖地巡禮|聖地巡禮]]之名，在鷲宮神社旁的店家賣起了一些《幸運☆星》的紀念品招攬顧客。

但由於柊家採用了鷲宮神社為背景，也意外造成了當地觀光收入大增，2008年元旦初詣開始的一個月內造成了2000萬日元以上的觀光收入，所以鷲宮町町長決定於2008年4月7日，柊家一家人將成為鷲宮町的一員，並發給「[http://www.flickr.com/photos/23767630@N03/2307562910/ 特別住民票]」證書（頒發給虛擬人物的一種表揚狀，類似給虛擬人物的戶口名簿或身分證之類的特別證書），由此可知柊家對當地帶來的影響力。2012年1月1日，作者美水鏡更是親自參拜了鷲宮神社，并獻上了親自繪製的繪馬，當日的參拜者就達到47萬之多<ref>《天漫》雜誌2012年3月上，397頁。</ref>。

另外，因為作者的筆名（名字和劇中的女角之一小鏡（{{lang|ja|かがみ}}）同名）、自畫像、作畫風格以及對角色描繪的細膩程度，使不少讀者把作者聯想成女性。其男性身分曾在日本和臺灣網路上引發一陣討論<ref>{{cite web |url=http://www.syu-ta.com/blog/2007/05/09/034726.shtml |title=存档副本 |accessdate=2007-05-30 |deadurl=yes |archiveurl=https://web.archive.org/web/20070613034214/http://www.syu-ta.com/blog/2007/05/09/034726.shtml |archivedate=2007-06-13 }}</ref>。

而其片頭曲CD《{{lang|ja|もってけ！セーラーふく}}》預定於2007年5月23日發售，但完整版片頭曲早在5月15日開始在網絡上流傳。
