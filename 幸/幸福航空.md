'''幸福航空'''，成立于2008年，是中国大陆的一家支线[[航空公司|航空公司]]，由[[中国航空工业集团|中国航空工业集团]]和[[中国东方航空|东方航空]]合资成立，创立时總部位於[[西安|西安]]。2015年，幸福航空与奥凯航空支线业务合并。2016年6月，合并后的幸福航空将基地机场由[[西安咸阳国际机场|西安咸阳国际机场]]变更为[[天津滨海国际机场|天津滨海国际机场]]<ref>{{cite web|title=关于变更幸福航空有限责任公司经营许可证所载基地机场的公示|url=http://www.caac.gov.cn/XXGK/XXGK/TZTG/201606/t20160616_38499.html|website=中国民用航空局|accessdate=2016-06-16}}</ref>。
[[File:Joy_air.jpg|缩略图]]]]

== 簡介 ==
幸福航空有限责任公司由[[中国航空工业第一集团公司|中国航空工业第一集团公司]]和[[中国东方航空股份有限公司|中国东方航空股份有限公司]]共同出资，注册资本金为10亿元人民币，其中中国一航以现金出资6亿元，占比60%；东航股份以现金出资4亿元，占比40%。董事长由中国航空工业第一集团公司党组成员、副总经理胡问鸣担任<ref>{{cite web |url=http://wiki.carnoc.com/wiki/%E5%B9%B8%E7%A6%8F%E8%88%AA%E7%A9%BA/ |title=存档副本 |accessdate=2009-10-10 |deadurl=yes |archiveurl=https://web.archive.org/web/20090825123408/http://wiki.carnoc.com/wiki/%E5%B9%B8%E7%A6%8F%E8%88%AA%E7%A9%BA/ |archivedate=2009-08-25 }}</ref>。主要飞机都为[[支线飞机|支线飞机]]，目前运营机种为[[新舟60|新舟60]]並於2009年8月15日實現首航，未来会引进[[ARJ21|ARJ21]]。

== 标志 ==
幸福航空的标志由5个莲花的花瓣、“幸福航空”和“Joyair”中英文字样组成。整个标志也可以看作是一只头向西部飞翔的鸟。

== 飞行事故 ==
*2014年12月11日下午，幸福航空公司一架航班号为JR1518、从银川起飞经停榆林到太原的MA60(新舟60)客机，在飞抵榆林机场上空时，因起落架故障在空中盘旋90多分钟后，最终成功安全着陆，包括机组人员在内的45名人员无一受伤。<ref>{{cite news|title=幸福航空客机出故障盘旋90分钟 多名女乘客吓哭|url=http://news.sina.com.cn/c/2014-12-12/024731275721.shtml|publisher=sina|date=2014-12-12}}</ref>

*2014年2月4日，由太原經鄭州飛往合肥的幸福航空1533號班機（[[新舟60|新舟60]]，編號B-3455）在[[鄭州新鄭國際機場|鄭州新鄭國際機場]]落地滑跑過程中，前起落架突然收起，導致機頭觸地，機上無人傷亡。<ref>{{cite news|title=JR1533航班机头触地　河南监管局介入调查|url=http://news.carnoc.com/list/272/272597.html|publisher=民航资源网|date=2014-02-04}}</ref>
*2015年5月10日幸福航空一架[[新舟60|新舟60]]飛機（編號B-3476）於[[福州長樂國際機場|福州長樂國際機場]]降落時失控，衝出跑道，機翼折斷，事故造成7人受傷。
