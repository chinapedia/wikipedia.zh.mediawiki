《'''幸腹塗鴉'''》（{{lang-ja|幸腹グラフィティ}}）是[[川井真琴|川井真琴]]的一部[[四格漫畫|四格漫畫]]作品，于《[[Manga_Time_Kirara_Miracle!|Manga Time Kirara Miracle!]]》2012年3月号<ref>{{cite news |script-title=ja:まんがタイムきららミラク3月号発売、独立創刊で月刊化 |url=http://natalie.mu/comic/news/62770 |publisher=[[Natalie_(网站)|Comic Natalie]] |date=2012-01-16 |accessdate=2016-07-16 |language=ja }}</ref>連載至2016年11月号<ref>{{cite web |script-title=ja:まんがタイムきららミラク 2016年11月号 |url=http://www.dokidokivisual.com/magazine/miracle/book/index.php?mid=524 |publisher=Manga Time Kirara Web |date=2016-09-16 |accessdate=2016-09-17 |language=ja }}</ref><ref>{{cite news |script-title=ja:「幸腹グラフィティ」完、アニメ化された女子中学生のぽかぽかお食事4コマ |url=http://natalie.mu/comic/news/202143 |publisher=Comic Natalie |date=2016-09-16 |accessdate=2016-09-17 |language=ja }}</ref>。於2015年1月至3月播出改編自原作的[[電視動畫|電視動畫]]。標題中的「幸腹」是取自「幸福」的[[諧音_(語言學)|諧音]]。本作还有一个特色为四位女主角在吃东西时的画风十分细腻而有别于平时的画风。

== 故事簡介 ==
町子涼是一名習慣獨自生活的女孩子。有一天，在阿姨拜託下，她開始於週末款待森野麒麟到家中暫住，以便來往彼此都有入讀的美術學校。之後他們遇上志同道合的同學椎名，一同開展烹調美食帶來幸福的[[治愈系|治愈系]]故事<ref>漫畫第二卷介紹。</ref>。

== 電視動畫 ==
2014年在6月16日發行的《Manga Time Kirara Miracle!》8月號宣布製作電視動畫。翌年2015年1月至3月在[[TBS電視台|TBS電視台]]、[[每日放送|MBS]]、[[中部日本放送|CBC電視台]]及[[BS-TBS|BS-TBS]]首播<ref>{{Cite news|url= http://natalie.mu/comic/news/121315|title=幸腹塗鴉決定於TBS電視台首播，同時官方網站開張。|work=Comic Daily|date=2014-07-16|accessdate=2014-08-10|language=ja}}</ref><ref>{{cite web|url=http://www.tvdrama-db.com/drama_info/p/id-58543|title=幸腹塗鴉|publisher=Furusaki Yasunari|accessdate=2015-03-27|language=ja}}</ref>，並可設定字幕顯示，同時主要角色的台詞有用顏色作區別（除了椎名和其他角色是用通常的[[白色|白色]]表示之外，町子涼的台詞是用[[黃色|黃色]]、森野麒麟的台詞是用[[水色|水藍色]]）。官方亦特別隨著播放進度，進行為故事中出現之食物提供食譜的計劃。另外，本作品是自2014年1月首播的《[[櫻Trick|櫻Trick]]》以來，第2部被改編成動畫的《Manga Time Kirara Miracle！》連載作品。作品標語為「'''{{lang|ja|さぁ、幸せが動き出す}}♪'''」。

集數方面，不是使用一般常見的「第一話」「第二話」，而是「第一道菜（{{lang|ja|ひとしなめ}}）」「第二道菜（{{lang|ja|ふたしなめ}}」，並有加入「鬆軟的（{{lang|ja|ほかほか}}）」或「香甜的（{{lang|ja|じゅわっ}}」等許多種[[擬聲詞|擬聲詞]]作為各話的副標題。
