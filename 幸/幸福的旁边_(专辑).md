《'''幸福的旁边'''》是[[中国大陆|中国大陆]][[流行摇滚|流行摇滚]][[乐队|乐队]][[花儿乐队|花儿乐队]]的首张[[专辑|专辑]]，由[[京文唱片|京文唱片]]于1999年1月18日发行，因為专辑发行时乐队3位成员的平均年龄未满18岁，被称为“中国第一支未成年乐队”，所以专辑一经发行就引起关注，是1999年中國摇滚类专辑销量最高的专辑之一，正版销量为15万张。<ref name=sina040422/>台湾版《'''放学啦'''》由[[滚石唱片|滚石唱片]]旗下的[[魔岩唱片|魔岩唱片]]于1999年12月16日发行。<ref name=bjqn091014/>

== 制作 ==
1998年2月，当时14岁的[[大张伟|大张伟]]、15岁的[[王文博|王文博]]以及19岁的[[郭阳|郭阳]]组成的花儿乐队在北京的一家酒吧里演出，被[[新蜂音乐|新蜂音乐]]的老板[[红枫|红枫]]发掘。1998年9月新蜂音乐正式成立，花儿乐队与新蜂音乐正式签约，并开始了《幸福的旁边》的录音制作。<ref name=qqmusic091021/>由于当时三位成员都正在上学，所以都选择在[[双休日|双休日]]进行录音工作，前后共用了17个双休日完成录音，而后期混音则用了七十个小时完成。<ref name=nbm001/> 专辑名的由来是大张伟的创意，他的同桌交了新的女朋友，整天说自己特别“幸福”，而大张伟就坐在他的“旁边”，所以取名《幸福的旁边》。<ref name=book/>

== 发行 ==
1999年1月18日，《幸福的旁边》正式发行，正版销量15万张，前后也出现了多种盗版CD和磁带，音乐内行人士估计这些盗版CD和磁带的销量已经超过百万。<ref name=sina040422/>

同年12月16日，《幸福的旁边》的台湾版《放学啦》由魔岩唱片发行，花儿乐队因此引起海外媒体关注，美国《[[华尔街日报|华尔街日报]]》以及[[有线电视新闻网|有线电视新闻网]]对乐队进行了专题报道。<ref name=sina040422/>

== 现场演唱 ==
2009年11月14日，花儿乐队于[[北京工人体育馆|北京工人体育馆]]举行了“花样十年”告别演唱会，在一段表演中，乐队重新演唱了收录于合辑《[[中國火#中國火叄|中国火 叁]]》中的单曲《放学啦》，这首歌也是花儿乐队的第一首歌，之后也收录在《幸福的旁边》中，但与收录于《中国火 叁》中的版本编曲不同。之后还演唱了专辑《幸福的旁边》中的歌曲《结果》、《静止》、《花儿》，在演唱《花儿》时，主唱大张伟突然留下眼泪。<ref name=qq091115/>

2011年5月1日，花儿乐队在滚石唱片的“滚石三十周年”[[北京鸟巢|北京鸟巢]]演唱会上“临时合体”，这是自2009年6月乐队宣布解散后，四位成员时隔两年再聚首共同演出。在这场演唱会上，花儿乐队演唱了收录于这张专辑中的歌曲《放学啦》、《静止》、《破灭》。<ref name=sina110504/>

2016年11月19日，大张伟在“人间精品起来嗨”[[北京工人体育馆|北京工人体育馆]]个人演唱会上演唱了收录于这张专辑中的歌曲《花》、《破灭》、《向我开炮》。<ref>http://www.weibo.com/dazhangwei831</ref>

2016年12月25日，大张伟在“人间精品起来嗨”[[北京工人体育馆|北京工人体育馆]]个人演唱会上演唱了收录于这张专辑中的歌曲remix版《静止》、原版《花》、《破灭》、《向我开炮》。

== 翻唱 ==
;莫文蔚
1999年4月，滚石唱片旗下的魔岩唱片获得花儿乐队的海外发行权之后，为了替《放学啦》在台湾发行造势，滚石唱片旗下的女歌手[[莫文蔚|莫文蔚]]翻唱了《幸福的旁边》收录的歌曲《破灭》，收录于她的专辑《[[你可以|你可以]]》，歌曲名改名为《消灭》，是这张专辑的主打歌。<ref name=sina040422/>

;杨乃文
1999年9月，同样也是为了替《放学啦》在台湾发行造势，魔岩唱片旗下的女歌手[[杨乃文|杨乃文]]翻唱了《静止》，收录于她的专辑《[[silence|silence]]》，也是作为这张专辑的主打歌。<ref name=sina040422/>
