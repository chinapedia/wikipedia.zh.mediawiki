'''邢丹遇害案'''发生于2011年4月13日晚。当[[邢丹|邢丹]]所乘的车行驶到[[惠深沿海高速公路|惠深沿海高速公路]]惠东稔山路段时，遭到三人（两位未成年人）石块袭击，邢丹在车上遭袭身亡。由于邢丹为著名慈善歌手[[丛飞|丛飞]]的遗孀，此事在社会引起了很大反响。

==背景==
{{see also|丛飞|邢丹}}
<!-- 注释出：[[File:Xing_Dan.jpg|thumb]] -->
邢丹1981年出生，2003年起从事空乘职业<ref>{{cite web |url= http://news.163.com/11/0417/12/71RFJPJ500014AED.html |title=还原邢丹_网易新闻中心 |first= |last=|work=163.com |year=2011 [last update] |accessdate=17 April 2011}}</ref>，2004年与[[丛飞|丛飞]]（自发为失学儿童义演，曾获2005年感动中国年度人物）结婚<ref>{{cite web|url=http://news.daynews.com.cn/shxw/1152114.html|title=感动中国人物丛飞遗孀邢丹罹难 愿他们天堂里继续洒爱心（图）|first=|last=|work=news.daynews.com.cn|year=2011 [last update]|accessdate=17 April 2011|deadurl=yes|archiveurl=https://web.archive.org/web/20110511135436/http://news.daynews.com.cn/shxw/1152114.html|archivedate=2011年5月11日}}</ref>，一直照顾丛飞直到2006年丛飞患胃癌逝世。

邢丹与丛飞生有一女，名叫邢小丛飞。

==案发过程==

2011年4月13日23时左右，林良健（作案时15岁）、黄金泉（作案时19岁）、蔡志成（作案时16岁）<ref>{{cite web |url=http://gd.nfdaily.cn/content/2011-04/17/content_22826183.htm |title=邢丹案破案过程公布 3疑犯未到法定年龄或定罪量刑-南方报网 广东 |first= |last= |work=nfdaily.cn |year=2011 [last update] |accessdate=17 April 2011 |deadurl=yes |archiveurl=https://web.archive.org/web/20110420155224/http://gd.nfdaily.cn/content/2011-04/17/content_22826183.htm |archivedate=2011年4月20日 }}</ref>（三人名字未完全公布）从自己所居住的村庄游荡到[[惠深沿海高速公路|惠深沿海高速公路]]惠东稔山段附近，在翻越了高速公路隔离栏后，为了取乐，三人捡起高速路旁的石块和混凝土块向高速路上的汽车投掷，在先后砸中三辆车后，23时45分左右，林良健所掷石块砸中了邢丹所乘的小轿车<ref>{{cite web |url=http://news.xinhuanet.com/legal/2011-04/17/c_121314953.htm |title=邢丹所乘小车被石块掷中照片公布_法治频道_新华网 |first= |last= |work=news.xinhuanet.com |year=2011 [last update] |accessdate=17 April 2011 |deadurl=yes |archiveurl=https://web.archive.org/web/20110509193012/http://news.xinhuanet.com/legal/2011-04/17/c_121314953.htm |archivedate=2011年5月9日 }}</ref>。邢丹被石块迎面砸中，其立刻被送到大亚湾区霞涌医院接受抢救，但是由于伤势过重，于4月14日0时20分因“左面部受钝物撞击致左下颌粉碎性骨折闭合性颅脑损伤”导致的失血过多而死亡<ref>{{cite web |url= http://news.163.com/11/0417/02/71QCB45500014AED.html |title=丛飞妻亡因被投石取乐(组图)_网易新闻中心 |first= |last=|work=163.com |year=2011 [last update] |accessdate=17 April 2011}}</ref>。

==事后==

在邢丹被砸中身亡后，家人没有告诉邢丹的女儿母亲逝世的噩耗，而是告诉她“妈妈出差，外婆也感冒生病了（处理后事）”<ref>{{cite web |url= http://news.163.com/11/0416/02/71NS3ITR00014AED.html |title=警方称丛飞遗孀身亡系人为 同行男子否认遭抢_网易新闻中心 |first= |last= |work=163.com |year=2011 [last update] |accessdate=17 April 2011}}</ref>。

4月16日4时，林，黄，蔡三人被抓获，三人时候交代是因为取乐才用石块砸车，不存在抢劫的说法。而由于砸中邢丹的混凝土块由15岁的未成年者砸中，其无法受到刑事处罚<ref>《中华人民共和国刑法》第十七条规定：“已满十四周岁不满十六周岁的人，犯故意杀人、故意伤害致人重伤或者死亡、强奸、抢劫、贩卖毒品、放火、爆炸、投毒罪的，应当负刑事责任。”对除此以外的其他犯罪不负刑事责任。</ref>。

4月18日上午，邢丹告别会在深圳市殡仪馆大礼堂举行， 邢丹的父母及亲属、丛飞的父母及亲属和深圳市各界干部群众上千人参加了邢丹同志告别会。邢丹的遗体火化后，择日会与在吉田墓园的丛飞遗骨合葬<ref>{{cite web|author=记者 徐华 何泳|title=深圳人挥泪痛别邢丹-邢丹同志告别会昨天举行|url=http://sztqb.sznews.com/html/2011-04/19/content_1531048.htm|publisher=[[深圳特区报|深圳特区报]]|accessdate=2011-04-19|location=[[深圳|深圳]]|pages=A6|language=zh-hans|date=2011-04-19|deadurl=yes|archiveurl=https://web.archive.org/web/20110512181023/http://sztqb.sznews.com/html/2011-04/19/content_1531048.htm|archivedate=2011-05-12}}</ref>。

2012年7月12日惠州市中级人民法院通报该案的一审结果：被告人林良健犯故意伤害罪，判处有期徒刑十二年，另两名被告人黄金泉、蔡志成犯以危险方法危害公共安全罪，分别被判处有期徒刑十年、有期徒刑六年，三位被告需赔偿原告人968086.45元<ref>{{cite web |url= http://news.163.com/12/0712/16/867PGO4E00011229.html |title=丛飞遗孀邢丹遭石袭身亡案3名被告获刑_网易新闻中心 |first= |last=|work=163.com |year=2012 [last update] |quote=968086.45 |accessdate=12 July 2012}}</ref>。宣判结束后，三位被告人均表示不会上诉<ref>{{cite web |url= http://news.163.com/12/0712/16/867PGO4E00011229.html |title=丛飞遗孀邢丹遭石袭身亡案3名被告获刑_网易新闻中心 |first= |last=|work=163.com |year=2012 [last update] |accessdate=12 July 2012}}</ref>。
