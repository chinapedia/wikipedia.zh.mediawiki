'''邢福增'''（{{lang-en|Dr. Ying Fuk-tsang}}）<!--（{{bd|1964年||||catIdx=Y}}）-->是宗教歷史的學者，從事中國基督教及當代中國政教關係研究多年，現任[[香港中文大學崇基學院神學院|香港中文大學崇基學院神學院]]院長。

==簡歷==
邢福增是土生土長的香港人，早年就讀[[天主教新民書院|天主教新民書院]]（1982年畢業）和[[新法書院|新法書院]]（1983年預科畢業）。1983年入讀[[香港中文大學|香港中文大學]]，主修歷史，1987年獲得文學士。畢業後繼續研究中國教會史，1989年及1995年分別獲得香港中文大學的哲學碩士及哲學博士學位。邢氏專研當代中國政教關係、中港基督教史及當代中國基督教發展。<ref name="ccnews">{{ cite web |url=http://www.cuhk.edu.hk/ccc/publications/newsletter/pdf/newsletter20141101.pdf |title=崇基校園通訊第50卷第18期 |date=2014-11-01 |publisher=崇基學院 |author=黃晊晄 }}</ref>邢氏信奉基督教，於1987年在循道衛理安素堂受洗，並於1999年成為為循道衛理聯合教會的義務教士。<ref name="faith100">{{ cite web |url=http://faith100.org/%E5%9B%9E%E9%A6%96%E6%81%A9%E5%85%B8%E8%B7%AF%EF%BC%8C%E6%86%91%E4%BF%A1%E8%B8%8F%E4%B8%80%E6%AD%A5/6046
 |title=回首恩典路，憑信踏一步 |date=2014-08-01 |publisher=信仰百川 |author=邢福增 }}</ref>

1993年至2004年，邢氏任職香港[[建道神學院|建道神學院]]副教授，兼任基督教與中國文化研究中心主任。2004年，邢氏轉職[[香港中文大學|香港中文大學]]，任[[香港中文大學|香港中文大學]]文化與宗教研究系教授，宗教與中國社會研究中心副主任、[[香港中文大學崇基學院神學院|崇基神學院]]教授（龐萬倫基督教與中國文化教席）、[[香港中文大學崇基學院神學院|崇基神學院]]當代基督宗教教學資源中心主任，2014年起出任[[香港中文大學崇基學院神學院|崇基神學院]]院長。他並兼任香港基督教循道衛理聯合教會義務教士。<ref name="faith100" /><ref name="theo">{{ cite web |url=https://www.theology.cuhk.edu.hk/tc/academics/teaching-staff/ying-fuk-tsang |title=教授團隊：邢福增 |date= |publisher=崇基學院神學院 |author= }}</ref>

邢福增關注中國內地的地上和地下教會的處境，以及宗教政策的改變。在1990年代曾寫過幾本有關內地政教關係的書，其中以1999年出版的《當代中國政教關係》最有名，內容提及中共的宗教政策，是基於「消滅、打擊、控制」等原則，批判中國政治對神學思想的約制。<ref name="next160714">{{ cite book |url=http://hk.next.nextmedia.com/article/1375/17754297 |title=壹週人物：背起十字架，邢福增 |date=2016-07-14 |publisher=壹週刊第1375期 |author=關卓凌 |location= | pages=  |ISBN= }}</ref>2015年起，特別關注中國內地拆十字架事件對政教關係的影響。<ref name="ctimes161217">{{ cite book |url=http://christiantimes.org.hk/Common/Reader/News/ShowNews.jsp?Nid=97398&Pid=5&Version=0&Cid=220&Charset=big5_hkscs |title=邢福增二○一六的香港故事 |date=2016-12-17|publisher=時代論壇 |author=羅民威  }}</ref>
