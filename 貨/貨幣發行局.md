'''貨幣發行局'''（'''Currency Board'''）是[[固定匯率制|固定匯率制]]的一種，即固定本國[[貨幣|貨幣]]與某特定[[外幣|外幣]]之[[匯率|匯率]]，並嚴格按照既定兌換比例，使[[貨幣發行量|貨幣發行量]]隨[[外匯儲備|外匯存儲量]]聯動的[[貨幣制度|貨幣制度]]。貨幣發行局的掛鈎貨幣通常是[[美元|美元]]或-{zh-cn:[[欧元|欧元]]; zh-hk:[[歐元|歐元]];}-。

[[香港聯繫匯率制度|香港聯繫匯率制度]]（Linked Exchange Rate）也是貨幣發行局，但允許匯率在固定範圍內浮動。因為[[三元悖論|三元悖論]]，「聯繫」不僅指匯率與美元[[掛鈎|掛鈎]]，而是[[貨幣政策|貨幣政策]]也跟隨[[美國|美國]]，港元利率長期走勢必須跟隨美元利率走勢。

== 历史 ==
1849年，毛里求斯首創了貨幣局制度。十九世纪末，英国为[[殖民地|殖民地]]提出并设立了联系汇率制。[[香港|香港]]作为英国殖民地而采用此汇率制度，与[[英镑|英镑]]掛鈎，1972年取消。1983年再度启用，改与[[美元|美元]]掛鈎，金管局與發鈔銀行的交易以7.80港元兑1美元的匯率結算，並允許市場匯率在7.75至7.85之間自由浮動。

比较典型的例子有[[阿根廷比索|阿根廷比索]]对[[美元|美元]]的联系汇率制和[[保加利亚|保加利亚]][[列弗|列弗]]对[[西德马克|西德马克]]（如今为[[欧元|欧元]]）的联系汇率制。

== 優点 ==
* 对本币[[汇率|汇率]]稳定的更强承诺和更小的逆转可能
* 稳定了币值，降低了市场的[[交易费用|交易费用]]

== 缺點 ==
* 聯繫匯率制必須以充裕的外匯儲備作為支持，否則在[[金融危機|金融危機]]到來時非常脆弱，例如：[[阿根廷|阿根廷]]在經濟危機的持續動蕩中放棄了聯繫匯率制，大幅下調了官方[[匯率|匯率]]，並引進了雙重匯率制。
* [[中央銀行|中央銀行]]採用聯繫匯率制，無法發揮[[最終貸款人|最終貸款人]]的作用，不能通過放寬[[貨幣政策|貨幣政策]]提供[[流動性|流動性]]，也不能直接融資來支援陷入問題的[[商業銀行|商業銀行]]，實際上把[[貨幣政策|貨幣政策]]的決策權讓給了所聯繫貨幣的管理當局。
* 聯繫[[美元|美元]]的情況下，經濟美元化，[[中央銀行|中央銀行]]損失[[貨幣發行|本幣發行]]、本幣需求增長和本幣存量[[利息|利息]]這三項[[鑄幣稅|鑄幣稅]]收入。
