'''夢雪'''為中國福建省[[東南衛視|東南衛視]]的[[製片人|製片人]]、[[策劃人|策劃人]]、[[編導|編導]]、[[主持人|主持人]]<ref name="dnws" >{{cite web |url=http://www.xydn.net/a/Moderator.asp|title=相約東南－主持人介紹|accessdate=2015-06-24 |work= |publisher=東南衛視/海峽衛視 |language =zh-cn}}</ref>。本名'''馬鑫'''<ref name="hsinchu"/>。她製作並主持的《相約東南》欄目以及「相約東南」系列晚會與[[台灣|台灣]][[東森電視台|東森電視台]]、[[香港|香港]][[無線電視台|無線電視台]]、香港[[鳳凰衛視|鳳凰衛視]]及[[美國|美國]][[百人會|百人會]]建立有長期合作關係，為福建對外宣傳的一個重要文化品牌<ref name="xydn"/>。

== 簡介 ==
夢雪出生及成長於遼寧瀋陽{{r|相約名人坊}}。1989年在[[瀋陽人民廣播電台|瀋陽人民廣播電台]]开始了她的主持生涯。當時她主持的《熱線點播》為收聽率极高的一档節目。

1991年，她隻身一人來到福建，在[[中國華藝廣播公司|中國華藝廣播公司]]主持《華廣服務台》、《少男少女》等節目。1993年創作並策劃拍攝電視散文《在路上》並於同年起主持《音樂潮》、《電視商場》等欄目，在觀眾中確立了娛樂節目主持人的地位。

1997年，以其本人命名的名人訪談節目《夢雪時間》開播，她本人也從一名娛樂節目主持人轉換為集採編播於一身的全能型主持人<ref name="dnws"/>。該節目於2002年改名為《相約名人坊》，並進入[[東南衛視|東南衛視]]的播出平台{{r|相約名人坊}}。2005年，該節目再次更名為《相約東南》並轉給[[海峽衛視|海峽衛視]]製作，夢雪仍然擔任製片人與主持人。該節目與2007年4月起在東南衛視與海峽衛視兩個平台聯合播出<ref name="xydn" >{{cite web |url=http://www.xydn.net/a/Company.asp|title=相約東南－欄目介紹|accessdate=2015-06-24 |work= |publisher=東南衛視/海峽衛視 |language =zh-cn}}</ref>。

2009年起，夢雪開始擔任全球[[閩南語|閩南語]][[台語流行音樂|歌曲]]創作演唱大賽、「[[媽祖|媽祖]]之光」文藝晚會、「[[客家|客家]]之歌‧義薄雲天」文藝晚會等多個大型活動的總[[導演|導演]]<ref>{{cite web |url=http://www.fjsen.com/content/2009-12/20/content_2536622.htm|title=第三屆全球閩南語歌曲創作演唱大賽50強晉級賽啟動|accessdate=2015-06-24 |work= |publisher=東南網|language =zh-cn}}</ref><ref>{{cite web |url=http://fj.sina.com.cn/news/z/2010-12-19/150285567.html|title=2010全球閩歌賽晉級賽儀式今日啟動 巨星大腕齊聚|accessdate=2015-06-24 |work= |publisher=新浪網|language =zh-cn}}</ref><ref>{{cite web |url=http://news.163.com/10/0416/00/64BQBRFR00014AEE.html|title=「媽祖之光」晚會為青海受災民眾祈福|accessdate=2015-06-25 |work= |publisher=網易|language =zh-cn}}</ref><ref name="hsinchu">{{cite web |url=http://report.hsinchu.gov.tw/OpenFront/report/show_file.jsp?sysId=C103AE014&fileNo=1|title=2013年「客家之歌‧義薄雲天」兩岸文化交流活動合作案出國報告|accessdate=2015-06-25 |work= |publisher=新竹縣政府|language =zh-tw}}</ref>。她於2010年為「媽祖之光」文藝晚會在[[彰化縣|彰化縣]]體育館演出隨[[中共福建省委|中共福建省委]]宣傳部來到[[台灣|台灣]]，並拜訪[[鹿港天后宮|鹿港天后宮]]<ref>{{cite web |url=https://www.youtube.com/watch?v=wK5-eRUCtvM|title=990928媽祖之光在彰化 福建省宣傳部蒞臨鹿港|accessdate=2015-06-25 |work= |publisher=youtube|language =zh-tw}}</ref>。

== 獲獎 ==
1991年獲得「全國對[[台灣|台]]宣傳廣播一等獎」。1992年憑藉《少男少女》節目獲得「全國對[[台灣|台]]宣傳廣播一等獎」與「全國新聞大賽二等獎」。1993年憑藉《在路上》獲得全國首屆電視散文大賽一等獎。1997年獲得「全國主持人金話筒大賽銅獎」<ref name="dnws"/>。2003年獲選為第六屆「百優電視節目主持人」之一<ref>{{cite web |url=http://www.cctv.com/anchor/special/C10554/20030820/100956.shtml|title=第六屆百優電視節目主持人名單|accessdate=2016-01-17 |work= |publisher=[[中國中央電視台|CCTV]].com|language =zh-cn}}</ref>。

2012年，其主創的「[[媽祖|媽祖]]之光」文藝晚會被評為「2012年度福建電視文藝及少兒節目獎二等獎」<ref>{{cite web|url=http://www.fjgd.gov.cn/gbdsxh/cypx/201306/t20130606_594839.htm |title=2012年度福建電視文藝及少兒節目獎獲獎目錄 |accessdate=2015-06-25 |work= |publisher=福建省廣播電影電視局 |language=zh-cn |archiveurl=https://archive.is/20150623161251/http://www.fjgd.gov.cn/gbdsxh/cypx/201306/t20130606_594839.htm |archivedate=2015-06-23 |deadurl=yes }}</ref>。
