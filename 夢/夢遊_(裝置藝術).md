《'''夢遊'''》（Daydream），是位於臺灣[[台北車站|台北車站]]與[[中山地下街|中山地下街]]之間的[[裝置藝術|裝置藝術]]，由[[台北當代藝術館|台北當代藝術館]]設置。
[[File:Daydream_plaque_by_Museum_of_Contemporary_Art,_Taipei_20170429.jpg|thumb]]

== 造型 ==
此藝術品又稱「鳥人愛維斯（Aves）」，位於中山地下街與台北車站M1出口之間。由藝術家[[何采柔|何采柔]]和郭文泰在2010年共同創作，材質為[[玻璃纖維|玻璃纖維]]、[[不鏽鋼|不鏽鋼]]、氟碳烤漆。造型為巨大鳥頭、長鳥喙、藍色大眼，搭配纖細的人體模型，右手拿一枝鉛筆。有水流會自動從鳥頭流下。<ref name="沈伯丞"/><ref name="張可臻"/><ref name="藝術館"/><ref name="王偉帆"/><ref name="米雪"/><ref name="聯合晚報"/>

2015年，因缺水曾暫停清洗，導致累積水漬、污垢，讓外觀像是沾了尿等黃色穢物<ref name="王偉帆"/>。
== 理念==
創作者解釋其創作理念是：「站立在一片草原之中，戴著鳥頭的女孩手上握著鉛筆，臉上流下的水覆蓋到她整個身體，永遠長不大的女孩，手上握著鉛筆也永遠寫不出字…」<ref name="聯合晚報"/><ref name="三立新聞"/>

== 反應 ==
有評論者將它比喻為[[古埃及|古埃及]]神[[托特|托特]]，認為是創作者何采柔的化身，畫筆則是記錄飛行於異想與真實的工具<ref name="沈伯丞"/>。有觀眾把它握的筆當成刀<ref name="張可臻"/>；或認為它的流水像是在流汗<ref name="聯合晚報"/>。

2015年，[[PTT|PTT]]有討論台灣最醜的地方特色[[吉祥物|吉祥物]]，有網友回應中此裝置藝術<ref name="米雪"/>，稱為「鳥頭人」；參賽者還有「虱目魚小子」{{efn|虱目魚小子，FRP材質雕塑像，由2015年[[行政院農業委員會水土保持局|水保局]]台南分局補助新台幣八十多萬元，以凸顯該地的[[虱目魚|虱目魚]]產業。該年7月22日下午吊裝時，雕像眼睛並無眼白，讓民眾、陳姓女設計者都反應詭異、可怕，聯想到[[鬼娃恰吉|鬼娃恰吉]]。數日後，水保局台南分局局長黃振全請來社區人員、原創設計者會勘，將雕像畫出眼白。<ref name="楊金城"/><ref name="楊金城2"/>}}，以及最後獲勝、被戲稱「屌面人」<ref name="黃益源"/>的「金蕉王」{{efn|金蕉王，為藝術家周耀東在2015年[[高雄燈會|高雄燈會]]以特色花燈展出，表現旗山農產香蕉與臺灣香蕉王國的歷史。燈會結束後就運至旗山。對於獲勝最醜，作者表示不會生氣。<ref name="民視新聞"/>}}。後來夢遊Daydreams又在新聞上被宣傳為「最醜吉祥物」之一<ref name="張可臻"/>。[[台北當代藝術館|台北當代藝術館]]則視它為「歪著頭的可愛女孩」<ref name="藝術館2"/>。[[廖信忠|廖信忠]]則開玩笑稱它因為腳擺內八字、有「憂鬱症文青般柔弱的身軀」，所以是[[文青|文青]]；更表示自己經常走不出臺北車站地下街，就是因為被它所詛咒<ref name="廖信忠"/>。

日本人稱為「鳥人間」<ref name="台北駅の鳥人間"/>。日本遊客評語有「奇怪生物」、「謎樣生物」、「可怕的東西」、「台灣最奧妙的生物」等詞，成為他們心中另類奇特的「台北印象」、「台灣印象」<ref name="三立新聞"/><ref name="自由時報"/>。由於造型十分引人注目，日本遊客也拍照並且分享至[[twitter|twitter]]，還有人至台北當代藝術館買吊飾當紀念品<ref name="米雪"/>。甚至有一名在臺日本人Yosimichi Iwhata將此像與[[自由女神像|自由女神像]]、[[西鄉隆盛|西鄉隆盛]]像齊名，稱為旅行中印象最深刻的雕像之一<ref name="聯合晚報"/>。

[[台灣海峽兩岸觀光旅遊協會|台灣海峽兩岸觀光旅遊協會]]上海辦事分處主任李嘉斌表示，此裝置藝術是中國大陸年輕旅客一定會造訪與拍照的景點<ref name="人間福報"/>。
