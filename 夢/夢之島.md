'''夢之島'''（{{jpn|j=夢の島|hg=ゆめのしま|rm=Yumenoshima|f=y}}）是[[東京都|東京都]][[江東區_(東京都)|江東區]]的[[地名|地名]]，位於城東地域內。

[[郵遞區號|郵遞區號]]為136-0081<ref name="postal" />。

== 概要 ==
「夢之島」是[[東京港#埋立地|東京灣]]埋立14號地中，{{link-ja|東京灣岸道路|[東京湾岸道路|灣岸道路}}以北的部分。[[夢之島公園|夢之島公園]]占據區域大部分地區。

14號地灣岸道路以南是[[新木場|新木場]]1丁目〜4丁目。北邊的夢之島大橋連接[[新砂|新砂]]。西邊與[[辰巳_(江東區)|辰巳]]之間以曙運河為界，東邊與[[江戶川區|江戶川區]]{{link-ja|臨海町|臨海町}}之間以荒川為界。

== 地勢 ==
夢之島地區大部分為公園、運動設施；公共設施，湾岸道路以南的新木場是則以辦公室為主。夢之島與西邊的[[辰巳_(江東區)|辰巳]]擁有大面積綠地，[[夢之島公園|夢之島公園]]擁有東京都內數一數二的綠地面積，宛如浮於海上的綠色之島。

近[[東京港|東京港]]，[[首都高速灣岸線|首都高速灣岸線]]與JR[[京葉線|京葉線]]可直接到達[[千葉縣|千葉縣]][[浦安市|浦安市]]、[[船橋市|船橋市]]、[[千葉市|千葉市]]。藉由[[東京臨海高速鐵道臨海線|臨海線]]新木場站，可通往{{link-ja|台場公園|台場公園|品川、台場}}。此外，夢之島大橋可前往[[深川_(江東區)|深川]]下町地區。

地區西側有南北向的[[明治通_(東京都)|明治通]]通過，明治通西側是夢之島競技場、夢之島運動廣場。明治通東側大部分為夢之島公園，園內有東京運動文化館、{{link-ja|東京都立第五福龍丸展示館|東京都立第五福竜丸展示館}}、{{link-ja|夢之島熱帶植物館|夢の島熱帯植物館}}等，地區東端有新江東清掃工場、夢之島碼頭，少年棒球場等。周邊的填海地多為倉庫與工廠，[[東雲_(江東區)|東雲]]、[[豐洲|豐洲]]、[[潮見站|潮見]]等地高層公寓大廈、辦公大樓林立。

== 歷史 ==
戰前的1939年（[[昭和|昭和]]14年），為了在[[東京灣|東京灣]]建設飛機場科始進行填海工程。1941年（昭和16年）因資材不足而中止工程，改為海水浴場<ref>{{Cite book|author=東京都港湾局|title=東京港史 第一巻 各論|date=|year=1994|accessdate=|publisher=東京都|pages=304-305|author2=|author3=|author4=|author5=|author6=|author7=|author8=|author9=}}</ref><ref name=":0">{{Cite news|title=東京・夢の島、名前の由来は海水浴場　空港計画も｜エンタメ！｜NIKKEI STYLE|last=日本経済新聞社・日経BP社|url=https://style.nikkei.com/article/DGXNASFK13038_U3A111C1000000|accessdate=2018-06-22|language=ja|work=NIKKEI STYLE}}</ref>。

1950年代，東京都内垃圾急增，東京都決定此地為[[堆填|垃圾填埋場]]，1957年（昭和32年）12月開始填埋，持續到1967年（昭和42年）。其中，1961年（昭和36年）7月23日北部發生大火，延燒面積達4萬平方公里<ref name="nenpyo">昭和災害史事典、P.13</ref>。相當於當時總面積的40%<ref name="nenpyo"/>。
1965年（昭和40年）7月16日，夢之島的大量蒼蠅隨著強勁南風擴散到江東區西南部一帶<ref>『アサヒグラフ』1965年7月16日号</ref>。由於政府的消毒作業無法完全解決，在警察、消防、自衛隊的協助之下，進行「夢之島焦土作戦」焚燒夢之島垃圾<ref name=":0" /><ref>[http://www.union.tokyo23-seisou.lg.jp/shiro/nakattara/02.html 「ハエの天国」と呼ばれた夢の島] 東京二十三区清掃一部事務組合</ref>。

填埋結束後11年的1978年（昭和53年），東京都立[[夢之島公園|夢之島公園]]開園。隨著整備的進行，一改過去的垃圾島氛圍，成為綠色之島，[[京葉線|京葉線]]開通後成為人氣景點。
1975年（昭和50年）3月1日成為江東區正式町名。當時未設「丁目」，2009年（[[平成|平成]]21年）11月1日實施{{link-ja|住居表示|住居表示}}，設夢之島一、二、三丁目。

== 名稱由來 ==
最早預定作為飛機場，後來計畫改為遊樂園。或許因為這個原因，當時一些媒體將此地稱為「夢之島」，之後逐漸成為此地代稱。1969年（昭和44年）成為正式行政地名。
