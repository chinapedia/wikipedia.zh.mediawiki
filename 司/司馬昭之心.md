《'''司馬昭之心'''》（{{lang-ja|さかしき人にみるこころ}}）是[[light_(遊戲品牌)|light]]於2008年5月30日發售的[[戀愛冒險|戀愛冒險]]類型[[日本成人遊戲|成人遊戲]]，簡稱「{{lang|ja|さかここ}}」。2009年5月29日發售以純紀子為女主角的[[衍生作品|衍生作品]]《{{lang|ja|どんちゃんがきゅ～}}》，2010年8月27日發售以純真路乃為女主角的衍生作品《{{lang|ja|まじのコンプレックス}}》與合集《{{lang|ja|神様のりんご}}》。

官方將遊戲類型定為「{{lang|ja|クールビューティーペダンチック恋愛ADV}}」，其中的「{{lang|ja|ペダンチック}}（Pedantic）」意思是「賣弄學問的」，故事中一重和亞利美會多番較量知識。

== 故事 ==
勾坂一重在[[學校圖書館|學校圖書館]]內，突然不知何故被素昧平生的女學生說「討厭」自己。對害怕遭人厭惡、裝好學生的一重而言，無法忍受這等事態，開始藉由幫忙圖書委員的工作等，試圖接觸亞利美，後演變成兩人的知識競爭。

== 人物 ==
; {{nihongo|勾坂一重|勾坂 一重（こうさか かずしげ）}}
:本作主角，3年級學生。自小害怕遭人討厭而逢迎他人，在小學時期被取了外號『EKC（ええかっこしい）』。一重對此[[心理創傷|一直介懷]]，中學時搬家至東京後改變形象避免被識破真面目，力圖成為真正的型男。

; {{nihongo|真柄亞利美|真柄 亜利美（まがら ありみ）||[[聲優|聲]]：[[青山由香里|青山由香里]]}}
:本作女主角，3年級的圖書委員。[[越前市|越前市]]出身，因為通學和家裡分住。配戴鈦框[[眼鏡|眼鏡]]，醞釀出知性但不易親近的氛圍。

; {{nihongo|純紀子|純 紀子（すみ のりこ）||聲：[[牧泉美|牧泉美]]}}
:真路乃的姊姊，3年級，亞利美的朋友。綽號叫『'''どんちゃん'''』（小鈍）。個性溫吞而[[天然呆|天然呆]]，但為人很踏實。在續作《[[どんちゃんがきゅ～|どんちゃんがきゅ～]]》擔當女主角。

; {{nihongo|純真路乃|純 真路乃（すみ まじの）||聲：[[木村あやか|木村あやか]]}}
:紀子的妹妹，1年級學生。和姊姊相反，有著堅毅性格及暴力傾向，自詡為紀子的保護者，但其實精神上依賴著紀子。

== 評價 ==
本作為light的低價格系列作品，在[[美少女ゲームアワード|美少女ゲームアワード]]2008獲得低價格獎的金賞<ref>{{cite web|url=http://www.bishojyogameaward.org/2008/result.html|title=美少女ゲームアワード2008　結果発表特設ページ|archiveurl=https://web.archive.org/web/20100209044835/http://www.bishojyogameaward.org/2008/result.html|archivedate=2010-02-09 }}</ref>。在追加生產時，為感謝玩家支持，將本作和另外一款低價格系列作品《[[しょぱん!|しょぱん!]]》推出2合1包裝販售，但價格不變。
