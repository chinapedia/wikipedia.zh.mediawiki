《'''真恋～寄语枫秋～'''》是一款[[美少女游戏|美少女游戏]]兼[[视觉小说|视觉小说]]，是[[中国大陆|中国大陆]]的美少女游戏制作组SP-time继《雪之本境~解境篇~》所创作的第五部作品，也是SP-time第一部有配音的作品<ref name="配音花絮" />以及「真恋系列」的第一部作品<ref>{{cite web|author1=二次面|title=专访国产Galgame制作团队SP-time：玩家的支持是我们的动力|url=http://www.anicoga.com/archives/31|website=ANICOGA|accessdate=2018-05-30|archiveurl=https://web.archive.org/web/20180530043059/http://www.anicoga.com/archives/31|archivedate=2018-05-30|deadurl=no|date=2015-07-14}}</ref>，于2014年通过[[光盘|光盘]]发售实体版，后于2016年10月5日登上[[Steam|Steam]]游戏平台。故事以虚构的地点枫音乡为舞台，讲述男主角郭恒逸与一同就读于枫华高中的姚枫怡、姚枫茜和唐海音之间的恋爱故事。

== 故事简介 ==
男主角郭恒逸是在枫音乡枫华高中就读的一名高二理科生，小时候便认识流浪的唐海音以及在枫音山上迷路的[[孪生|孪生]]姐妹姚枫怡和姚枫茜，但后来因为心脏病发作住院而忘记了和唐海音有关的记忆，出院后心脏病也未能痊愈因此不能进行剧烈运动。高二文理分班后，在姚枫茜的提议下，他们准备在学校成立[[话剧|话剧]]社。在一天晚上，郭恒逸为了救海中的唐海音而心脏病恶化垂死，同时姚枫怡也因为遭遇车祸死亡，姚枫怡在死前要求将[[心脏移植|心脏移植]]给郭恒逸。手术成功后，为了避免让郭恒逸受到打击，姚枫茜打扮成姚枫怡的样子以隐瞒姚枫怡以死亡的事实，后来郭恒逸得知真相后陷入自责。后来为了重新振作并在高中留下美好的回忆，他们决定重建话剧社。

在此过程中，郭恒逸的感情也开始游荡在姚枫茜和唐海音之间。根据玩家的选择，两位女主角的好感度会发生变化，最终会进入不同的结局<ref>{{cite web |author1=kuroNekk |title=《真恋~寄语枫秋~》无剧透攻略 |url=https://www.cngal.org/guide:%E3%80%8A%E7%9C%9F%E6%81%8B~%E5%AF%84%E8%AF%AD%E6%9E%AB%E7%A7%8B~%E3%80%8B%E6%97%A0%E5%89%A7%E9%80%8F%E6%94%BB%E7%95%A5 |website=中文Galgame资料站 |accessdate=2018-06-04 |archiveurl=https://web.archive.org/web/20180604122430/https://www.cngal.org/guide:%E3%80%8A%E7%9C%9F%E6%81%8B~%E5%AF%84%E8%AF%AD%E6%9E%AB%E7%A7%8B~%E3%80%8B%E6%97%A0%E5%89%A7%E9%80%8F%E6%94%BB%E7%95%A5 |archivedate=2018-06-04|deadurl=no |date=2015-07-26}}</ref>。游戏共有五个结局，其中两个是好结局，分别跟姚枫茜和唐海音确定了恋爱关系，另外三个是坏结局。

== 制作与发行 ==
此游戏由SP-time制作组所制作，在《真恋～寄语枫秋～》之前，SP-time已制作有四部游戏作品，分别是《翡翠月》、《雪之本境》、《雾之本境》和《雪之本境~解境篇~》，《真恋～寄语枫秋～》是第五部作品，同时也是SP-time第一部有配音的作品，以及第一部登上Steam游戏平台的作品。《雪之本境》、《雾之本境》和《雪之本境~解境篇~》是SP-time所制作的「本境系列」作品，而《真恋～寄语枫秋～》则是「真恋系列」作品，前者的剧情以悬疑推理为主，后者则是以纯爱为主。游戏的主要负责人兼编剧小仆表示，自己在大学期间便有创作《真恋～寄语枫秋～》这类纯爱型游戏的想法，在创作《雪之本境~解境篇~》期间，小仆也有把下一部作品创作成纯爱型而不是悬疑型的打算<ref name="公式设定集" />{{rp|38}}。2013年，小仆和SP-time开始了《真恋～寄语枫秋～》的创作，原定于2013年9月28日发售，但后来因为游戏制作规模过大以及小仆的家庭原因而导致游戏的发布日期推迟了一年，直到2014年10月，游戏才通过[[漫展|漫展]]和[[淘宝网|淘宝网]]发售<ref>{{cite web|author1=梦崽子|title=【资讯】国产GAL大坑《真恋～寄语枫秋～》今秋发售|url=http://news.missevan.com/news/article?newsid=23306|website=MissEvan|publisher=顺风动漫|accessdate=2018-05-30|archiveurl=https://archive.today/20180530044052/http://news.missevan.com/news/article?newsid=23306|archivedate=2018年5月30日|deadurl=no|date=2014-09-28|df=}}</ref>。

游戏的[[限定版|初回限定版]]分为「枫茜之忆」和「月海之音」两个版本，除了游戏本体之外，还包含有女主角姚枫茜和唐海音的特典，如音乐[[CD|CD]]、[[海报|海报]]、[[书签|书签]]等。<ref>{{cite web|title=国产GAL“本境系列”团队SP-TIME新作《真恋～寄语枫秋～》|url=http://www.cosplay8.com/news/game/2014/0313/57011.html|website=cosplay8|accessdate=2018-05-27|archiveurl=https://web.archive.org/web/20180527071416/http://www.cosplay8.com/news/game/2014/0313/57011.html|archivedate=2018-05-27|deadurl=no|date=2014-03-13}}</ref>

2016年1月30日，游戏登上Steam青睐之光，2月份该作品在Steam青睐之光被投诉下架，原因是主角唐海音与日本动画《[[我的青春恋爱物语果然有问题|我的青春恋爱物语果然有问题]]》中的雪之下雪乃非常相像且头饰涉嫌侵犯版权。主画师恶魔风铃（Devil_Campanula）对于侵权指控表示无奈，并称其创作角色的头饰，是自己结合蝴蝶结和穗子所得<ref>{{cite web|title=SP-time游戏角色被质疑抄袭 Steam绿光游戏下架画师背锅|url=http://acg.178.com/201602/248820064606.html?pc=1|website=178动漫频道|accessdate=2018-05-27|archiveurl=https://web.archive.org/web/20180527070303/http://acg.178.com/201602/248820064606.html?pc=1|archivedate=2018-05-27|deadurl=no|date=2016-02-07}}</ref>，后来经过申诉游戏重新上架。2016年10月5日，游戏正式登上Steam游戏平台。2016年10月28日，游戏的音乐[[原声集|原声集]]在Steam上作为[[可下载内容|可下载内容]]发售<ref>{{cite web|title=True Love ~Confide to the maple~ Original Soundtrack|url=https://store.steampowered.com/app/501750/True_Love_Confide_to_the_maple_Original_Soundtrack/|website=Steam|accessdate=2018-05-27|archiveurl=https://web.archive.org/web/20180527074255/https://store.steampowered.com/app/501750/True_Love_Confide_to_the_maple_Original_Soundtrack/|archivedate=2018-05-27|deadurl=no|date=2016-10-28}}</ref>。2017年6月15日，原声集中秋石作曲的音乐作为另一份可下载内容供玩家免费下载<ref>{{cite web |title=True Love ~Confide to the maple~Featured Composer 秋石(Qiushi) |url=https://store.steampowered.com/app/580210/True_Love_Confide_to_the_mapleFeatured_Composer_Qiushi |website=Steam |accessdate=2018-06-04 |archiveurl=https://web.archive.org/web/20180604102547/https://store.steampowered.com/app/580210/True_Love_Confide_to_the_mapleFeatured_Composer_Qiushi |archivedate=2018-06-04|deadurl=no|date=2017-06-15}}</ref>。

在游戏中，小仆引入了SP-time其他作品中的角色作为客串角色以整合游戏的世界观，如归海枫和庭雪。此外，游戏中也出现了其他制作组作品的客串，例如游戏场景出现了《[[高考恋爱100天|高考恋爱100天]]》、《[[且听琴语_Grobda_Remix|且听琴语 Grobda Remix]]》和《[[第九日|第九日]]》等中国大陆制作的美少女游戏的海报，在实体版游戏中，也出现了《高考恋爱100天》中女主角木馨的客串。<ref name="叽咪叽咪">{{cite web |author1=转基因指甲油 |title=一叶知秋，真恋永存 |url=https://kol.gimmgimm.com/article/info?id=12363 |website=叽咪叽咪 |accessdate=2018-06-03 |archiveurl=https://web.archive.org/web/20180603160222/https://kol.gimmgimm.com/article/info?id=12363 |archivedate=2018-06-03|deadurl=no |date=2018-03-24}}</ref>

== 音乐 ==
游戏的音乐由秋石、恒曌和REi所创作，其中秋石在此之前也有跟SP-time合作，为《雾之本境》和《雪之本境~解境篇~》创作了音乐，负责剧本的小仆也参与了音乐的创作。游戏中有部分[[背景音乐|背景音乐]]沿用了《雪之本境》、《雾之本境》、《雪之本境~解境篇~》的背景音乐，如{{单双书号转换|本境之约}}。

游戏有两首[[片头曲|片头曲]]，分别是{{单双书号转换|枫海}}和{{单双书号转换|永恒的等待}}。{{单双书号转换|枫海}}由秋石作词和作曲，由Jada多多演唱，于2014年3月10日发布[[音乐录像|音乐录像]]。{{单双书号转换|永恒的等待}}由恒曌作曲，最初仅有伴奏，于2014年9月24日发布了伴奏版的音乐录像<ref name="Official">{{cite web |title=真恋寄语枫秋 |url=http://sp05.sp-time.com/index.html |website=SP-time |accessdate=2018-06-04 |archiveurl=https://web.archive.org/web/20180604090314/http://sp05.sp-time.com/index.html |archivedate=2018-06-04|deadurl=no}}</ref>。随后，SP-time进行了歌词征集活动并对收到的投稿进行评选，这也是SP-time所举行的《真恋～寄语枫秋～》系列活动的第一场，最后于2014年10月30日公布了征集结果，并选用司徒音_taoistly所作的词作为正式版本<ref>{{cite web |title=真恋寄语枫秋系列活动第一弹！OP2歌词征集活动 活动结果发表 |url=http://sp05.sp-time.com/op2 |website=SP-time |accessdate=2018-06-03 |archiveurl=https://web.archive.org/web/20180603160236/http://sp05.sp-time.com/op2 |archivedate=2018-06-03|deadurl=no}}</ref>。2014年11月1日，SP-time和[[被窝声次元|被窝声次元]]合作举行了系列活动第二场，向玩家征集{{单双书号转换|永恒的等待}}的演唱版。一个月后，浅木ゆき的演唱版本被评选为最终大奖，原先仅有伴奏的{{单双书号转换|永恒的等待}}音乐录像也更换为浅木ゆき的演唱版本<ref>{{cite web |title=真恋寄语枫秋 SP-time x 被窝 系列活动第二弹！真恋声募计划 活动结果公布 |url=http://sp05.sp-time.com/sm47.html |website=SP-time |accessdate=2018-06-03 |archiveurl=https://web.archive.org/web/20180603160238/http://sp05.sp-time.com/sm47.html |archivedate=2018-06-03|deadurl=no}}</ref>。

== 评价 ==
游戏中给主角配音的配音演员是中国大陆比较出名的声优，SteamCN上的玩家有太阳就有光对主角的配音表示肯定，但也指出了路人角色的配音不专业且有些有配音有些没有配音，显得不统一。此玩家还表示欣赏游戏中的中国特色对白，同时也建议游戏中增加更多的诗句以模仿[[仙剑奇侠传系列|仙剑奇侠传系列]]的[[过场|过场]]效果。<ref>{{cite web |author1=有太阳就有光 |title=Steam国产GAL《-真恋 ~寄语枫秋~》评测 |url=http://www.a9vg.com/news/201706/0886013112.html |website=A9VG电玩部落 |publisher=SteamCN |accessdate=2018-06-04 |archiveurl=https://web.archive.org/web/20180604104752/http://www.a9vg.com/news/201706/0886013112.html |archivedate=2018-06-04|deadurl=no |date=2017-06-15}}</ref>

叽咪叽咪上的用户转基因指甲油同样是肯定了声优的配音质量，并肯定了音乐的质量，但指出了剧情由悲到喜的转变过于唐突。<ref name="叽咪叽咪" />
