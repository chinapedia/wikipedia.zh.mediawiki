'''真平'''是為臺灣漫畫家[[葉宏甲|葉宏甲]]所發表的《[[諸葛四郎|諸葛四郎]]》系列漫畫作品裡主要角色之一。

== 名字來源 ==
此角色名字由來為引用自「真誠平實」詞語。<ref>{{cite book |title=台灣漫畫風城50年 |year=1999 |publisher=新竹市立文化中心 |isbn=957-024-745-2 |pages= 79頁 }}</ref>另有被引述為作者葉宏甲所從事的漫畫家職業界裡、多數人平實的個性。<ref>{{cite book |title=漫畫百寶箱 : 自編．自導．畫漫畫 |date=2000年5月 |publisher=雄獅圖書 |isbn=957-474-007-2 |pages=75頁 |accessdate=2010年9月18日}}</ref>

== 作品的描述 ==
真平是漫畫《諸葛四郎》系列裡主角[[諸葛四郎_(漫畫角色)|諸葛四郎]]的義弟，也是最好的朋友與夥伴，倆人在故事中多有一同出生入死、突破困境的場面。在作品裡有如第一男配角的定位。<ref name="Page63">{{Cite web|url=http://ir.lib.ntnu.edu.tw/retrieve/47529/metadata_02_07_s_05_0014.pdf%E2%80%8E|title=製造英雄─以葉宏甲漫畫「諸葛四郎」系列為中心|author=林佳叡|format=pdf|work=國立臺灣師範大學臺灣史研究所|pages=63頁|accessdate=2014年3月18日|date=2000年5月}}{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>外貌上為細眼睛、頭後方綁單一髮髻的模樣。服裝在初期為長袖布衣，之後在「大戰雙騎士」中和四郎一同換成短袖布衣，在手臂前臂上纏繞白繃帶的模樣。

性格上與四郎同樣忠勇、堅決果敢，但較少像四郎一樣有急中生智、以計擒敵的描寫。在劇情中也較多擔任因疏忽而落入敵方圈套裡、而等待四郎解救的角色。和四郎相比之下，真平在故事中也較常出現一時下手過重而意外將對手殺死的較不謹慎場面。<ref name="Page63"/>

真平個人擁有不亞於四郎之下的劍技，作者葉宏甲除了讓真平隨著故事發展與四郎一同成長外；並在《[[大戰龍虎十劍士|大戰龍虎十劍士]]》故事中讓他習得了武林絕技「真空斬」，<ref>{{cite book |title=台灣漫畫風城50年 |year=1999年 |publisher=新竹市立文化中心 |isbn=957-024-745-2 |pages=80頁 |accessdate=2010年9月18日}}</ref>雖然之後葉宏甲對於讓真平與四郎學會絕世武功、影響到後續登場敵方人物強度一事感到後悔。<ref>{{Cite web|url=http://ir.lib.ntnu.edu.tw/retrieve/47529/metadata_02_07_s_05_0014.pdf%E2%80%8E|title=製造英雄─以葉宏甲漫畫「諸葛四郎」系列為中心|author=林佳叡|format=pdf|work=國立臺灣師範大學臺灣史研究所|pages=37頁|accessdate=2014年3月18日|date=2000年5月}}{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

== 真人演出 ==
在真人演出作品方面，最早飾演真平的演員為[[傳清華|傳清華]]，是在1962年電影《雙雄大鬥雙假面》中登場，<ref>{{Cite book |title=聯合報印縮本25(中華民國五十一年一至四月) |location=台灣 |publisher=聯經出版事業 |date=1980年9月 |pages=283頁 |id=行政院新聞局出版事業登紀證局版台業字第0130號 |accessdate=2014年3月18日}} </ref>後續1978年電影《諸葛四郎大鬥雙假面》裡，則是[[衛子雲|衛子雲]]擔任。<ref>{{Cite web |url=http://folkartist2.e-lib.nctu.edu.tw/collection/yeh/03_report/03_1977report08.htm |title=諸葛四郎大鬥雙假面 暢銷漫畫名著上銀幕 |work=浩然藝文數位博物館 |publisher=[[國立交通大學|國立交通大學]] |accessdate=2014年3月18日 }}</ref>另外1985年電視劇《四郎與真平》由[[龍傳人|龍傳人]]演出，其中先前在《諸葛四郎大鬥雙假面》裡扮演真平的衛子雲則在此電視劇系列飾演諸葛四郎。<ref>{{Cite web|url=http://ir.lib.ntnu.edu.tw/retrieve/47529/metadata_02_07_s_05_0014.pdf%E2%80%8E|title=製造英雄─以葉宏甲漫畫「諸葛四郎」系列為中心|author=林佳叡|format=pdf|work=國立臺灣師範大學臺灣史研究所|pages=40頁|accessdate=2014年3月18日|date=2000年5月}}{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

== 大眾文化引用 ==
因真平與諸葛四郎為好友關係，網路小說家[[蔡智恆|蔡智恆]]的作品《檞寄生》一書中，有以「諸葛四郎的好朋友」作為一個人物署名的代號。<ref>{{cite book |title=檞寄生 |year=2003年 |publisher=紅色文化 |author=[[蔡智恆|蔡智恆]] |isbn=986-78-8822-7 |pages=50～51頁 |accessdate=2014年3月18日}}</ref>
