《'''真實之淚 true tears'''》是由[[P.A.WORKS|P.A.WORKS]]所製作的[[日本|日本]]的[[電視動畫|電視動畫]]作品<ref>{{cite web|url=http://acg.gamer.com.tw/acgDetail.php?s=598|title=【動畫】真實之淚|publisher=[[巴哈姆特電玩資訊站|巴哈姆特電玩資訊站]]|accessdate=2014-01-21|language=zh-tw}}</ref>。

== 概要 ==
本作自2008年1月开始播放，以[[富山县|富山县]]为舞台（担当动画制作的[[P.A.WORKS|P.A.WORKS]]社址就在富山县[[南砺市|南砺市]]），故事觀與[[真實之淚_true_tears|同名遊戲]]相同，但角色和剧情均为原创<ref>{{cite web|url=http://game.watch.impress.co.jp/docs/20060208/true.htm | script-title=ja:ブロッコリー、「true tears～トゥルーティアーズ～」発表会開催木谷氏らが開発裏話などを披露。アニメ化も決定！|publisher=GAME Watch|accessdate=2014-01-21 |language=ja}}</ref>。

[[DVD|DVD]]于2008年发售，[[Blu-ray|Blu-ray]] BOX则是在2010年3月26日发售。

== 故事 ==
以插畫家為夢想的高中生——仲上真一郎，與真一郎同居的同學——湯淺比呂美（被仲上家收留），平時與真一郎的父母過著四人的家庭生活。某日，真一郎在學校裡遇見了爬到樹上卻下不來的女孩——石動乃繪，還對真一郎說了些不可思議的話。自從這次的相遇，使得真一郎與比呂美、乃繪、好友野伏三代吉、童年玩伴安藤愛子之間的關係開始產生微妙的變化。故事描寫著他們在各自的苦惱中下定決心而成長，是一部青澀的校園戀愛劇。

== 登場人物 ==
===主要角色===
; {{nihongo|仲上 真一郎|仲上 眞一郎（なかがみ しんいちろう）||[[聲優|聲優]]：[[石井真|石井真]]}}
:本作主角，就讀麥端高校一年B班。當地釀酒廠「仲上酒廠」廠長的獨子，夢想成為繪本畫家，每晚都進行著繪畫的創作。也屢次向出版社投稿作品過，然而身為應當繼承家業的獨子，使得母親對他的作為抱持反對的態度。雖被稱作「當家花旦」，卻不太情願地學習著麥端祭上表演的麥端舞，與比呂美為同居關係的青梅竹馬。雖然沒有表明，但真一郎內心一直偷偷喜歡着比呂美。一直想創作插畫卻沒有勇氣起步的真一郎，在乃绘的引導下，開始作畫，也認知到自己的責任，產生勇氣開始面對自己心中的比呂美，為了讓比呂美不再流淚而努力著。而在身邊的乃绘為了真一郎努力想獻上自己的眼淚，讓真一郎內心開始動搖。曾因为与石动乃绘哥哥的约定在第七集对乃绘请求交往。为乃绘画过一个系列的漫画，后来逐渐明确自己的内心，最终选择了比吕美。在比吕美搬走时曾骑自行车去追，并向比吕美作出承诺，最后与乃绘分手，向比吕美告白。
; {{nihongo|湯淺 比呂美|湯浅 比呂美（ゆあさ ひろみ）||聲優：[[名塚佳織|名塚佳織]]}}
:真一郎小學時就相識的青梅竹馬，也是同班同學。麥端高校女子籃球部6號球員，成績優秀，運動神經也超群，被三代吉稱讚「有著優等生的氣息，卻一點都不驕傲的完美女孩」。由於一年前父母雙亡，與湯淺夫婦交情甚深的真一郎的父親便收留她，自此開始與真一郎同居、幫忙仲上家事務的生活。因真一郎母親關係，把自己對真一郎情感封閉在心中，直到乃绘的出現讓呂比美開始重新面對真一郎情感。最终和石动纯结束了“恋人的游戏”，真一郎知道了自己感情的所属。真一郎找到乃繪，坦白了自己心底喜欢的是比吕美，见到乃繪会让他有所动摇。也就是这样，真一郎和乃繪分手了。在发生了各种事情后，真一郎找到了比吕美，真一郎能够面对比吕美，能够为少女擦拭眼泪。两人最终相爱交往。
; {{nihongo|石動 乃绘|石動 乃絵（いするぎ のえ）||聲優：[[高垣彩陽|高垣彩陽]]}}
:一年前轉至麥端高校的轉學生，就讀一年A班。因為奇特的想法與不理會別人目光的個性，被同學間視為是怪人。在祖母過世之後，眼淚就再也哭不出來了。實際上卻出奇的健談，長相與言行舉止都和年齡不符，比較像個小孩子，卻擁有能看透別人內心的敏銳感性。 常到學校的雞舍對著雞說話，是個不可思議的女孩。在與真一郎相處中，漸漸真心喜歡上真一郎。
; {{nihongo|安藤 愛子|安藤 愛子（あんどう あいこ）||声优：[[井口裕香|井口裕香]]}}
:比真一郎大一歲的小時玩伴，三代吉的女朋友，擁有茶色短髮的開朗活潑少女。時常幫忙家中經營[[今川燒|今川燒]]店，因此料理的技術不容置疑。經常為真一郎排解心中煩惱，然而身高卻老是被真一郎開玩笑，雖然跟三代吉交往，事實上喜歡的是真一郎。曾强吻真一郎，并向其告白，但被拒绝。
; {{nihongo|野伏 三代吉|野伏 三代吉（のぶせ みよきち）||聲優：[[吉野裕行|吉野裕行]]}}
: 真一郎的好友與同班同學。之前真一郎介紹愛子給他，兩人便以此為契機開始交往，對愛子的感情也相當認真。常在放學後與真一郎一起到訪今川燒店。

===次要角色===
; {{nihongo|石動 純|石動 純（いするぎ じゅん）||聲優：[[增田裕生|增田裕生]]}}
:石動乃绘的哥哥。螢川高校3年級生，籃球部4號球員，俊秀的面容與挺拔的運動身材讓他在女學生間相當有人氣。而家庭狀況使得他非常照顧且疼愛妹妹乃繪，被比呂美稱作[[妹控|妹控]]。但其實他對妹妹的感情深厚到甚至超越了親情，愛上了乃繪，卻沒發現，還以為這樣是保護。無論什麼事決定，眼中只有妹妹，除了乃繪以外，其他人事物都不重要。看出乃絵喜歡上真一郎，逼迫真一郎與乃絵交往，以此為條件，自己與比呂美交往。但最终被比吕美看出是妹控而被拒絕。
:最后祭典時在打工的店裏表达出了自己對乃繪的爱，並告诉了乃繪自己的感情。
; {{nihongo|仲上 宗弘|仲上 宗弘（なかがみ　むねひろ）||聲優：[[藤原啟治|藤原啟治]]}}
:真一郎的父親，家傳酒業「仲上酒廠」的廠長，沈默寡言的個性給人嚴厲父親的形象，實際上卻是溫和而善解人意，對於真一郎不繼承家業而以插畫家為目標的事情表示理解與支持，把比呂美當作自己的孩子一樣對待。與比呂美的父母親是舊識。
; {{nihongo|仲上 栞|仲上 しをり（なかがみ　しをり）||聲優：[[高橋理惠子|高橋理惠子]]}}
:真一郎的母親，較為感情用事的女性。非常溺愛真一郎，相反地則將比呂美視為眼中釘，施予外人般冷漠地對待。對真一郎不能克紹箕裘的行為感到反對與不解。是造成比呂美個性轉變的原因，在比呂美發生事故之後，知道自己的過錯而接受比呂美，並請求比呂美的原諒，之後開始支持著比呂美對真一郎的情感。
; {{nihongo|酒廠的少年|酒蔵の少年||聲優：[[土倉有貴|土倉有貴]]}}
:在仲上酒廠當學徒，和善而有朝氣的平頭少年，天真老實的個性有時會讓人誤以為是[[天然呆|天然呆]]。
; {{nihongo|黑部 朋與|黒部 朋与（くろべ ともよ）||聲優：[[渡邊智美|渡邊智美]]}}
:比呂美的朋友，同樣從屬於女子籃球部，為9號球員。稍稍支持著比呂美對真一郎的情感。
; {{nihongo|乃繪的奶奶|乃絵のおばあちゃん（のえのおばあちゃん）||聲優：[[瀨能禮子|瀨能禮子]]}}
:在乃繪小時候就過世了。告訴乃繪要拿走乃繪的眼淚到天堂去。還有說能拿走眼淚只有自己重要的人。
