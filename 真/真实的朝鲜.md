'''《真实的朝鲜》'''是[[中国|中国]]作家[[叶永烈|叶永烈]]在2008年出版的长篇[[纪实|纪实]]作品，属于“叶永烈世界观”丛书之一，由天津教育出版社出版发行。

== 内容 ==
书中作者以游客身份在[[丹东|丹东]]参加当地[[朝鲜|朝鲜]]旅行团赴朝[[旅游|旅游]]，并写下此游记。作者也曾在朝鲜驻中国大使馆申办[[签证|签证]]但不成功<ref>见《真实的朝鲜》第12页“申请朝鲜簽證始末”</ref><ref name=poenword/>。书中以一个普通中国游客的角度去观察朝鲜的现状，介绍了在朝鲜看到的奇特[[社会|社会]]和[[政治|政治]]现象，并刊载了大量在朝鲜拍摄到的图片，其中不乏在行进中的[[火车|火车]]和[[客车|客车]]上随机拍摄的作品，因这种行为不被朝鲜当局认可，为此多次与[[导游|导游]]和随团的朝鲜[[国安|国安]]人员争执。

== 中國出版被禁 ==
本书的出版并不顺利。作者在离开朝鲜回国时受到严格的半小时重点检查，但仍完整带回在朝鲜的记录，并回到[[上海|上海]]写出此书<ref name=poenword>[http://www.open.com.hk/0809p69.html 《真实的朝鲜》怎样成为禁书？] {{webarchive|url=https://web.archive.org/web/20081005163536/http://www.open.com.hk/0809p69.html |date=2008-10-05 }}，香港《开放》杂志，2008年九月号刊载</ref>。但由于中国与朝鲜本身特殊的历史关系，虽然多次被出版社邀稿，但都因顾忌政治因素而没敢出版。后[[广东|广东]][[杂志|杂志]]《同舟共进》分两期摘引此书的部分章节发表题为“'''告诉你一个真实的朝鲜'''”的文章，反应热烈，但仍然没有出版商敢于出版<ref name=poenword/>。后由北京新华立品图书有限公司在删除三节内容后最终在中国出版<ref name=poenword/>。

作者虽对此书保持低调，但由于[[新浪|新浪]]、[[搜狐|搜狐]]等[[门户网站|门户网站]]刊载，[[北京人民广播电台|北京人民广播电台]]全文连播而导致官方的注意，并且在朝鲜大使馆致函[[中華人民共和國外交部|中華人民共和國外交部]]要求停止发行此书后，[[中华人民共和国新闻出版总署|中国国家新闻出版总署]]对此书发出禁令，但只是在大型书店下架禁售，其他地方仍有销售，在朝鲜方面再次致函外交部后，出版总署才对全国所有书店下架禁售<ref name=poenword/>。作者认为这是为了顾及中国与朝鲜的兄弟同盟关系而必要牺牲的作为<ref name=poenword/>。

此书被禁后除了书店禁售，一些网络图书销售网站也撤下此书，网站的连载也被迫停止。但在2008年末[[当当网|当当网]]店庆的时候曾以原价处理此书6折的价格销售，目前已再无销售，但部分小型网络书店仍有销售。

==其他地方出版==
据作者撰文说，[[韩国|韩国]]有出版社有意出版此书的无删节版<ref name=poenword/>。

目前，在[[香港|香港]]，[[新东方出版（香港）有限公司|新东方出版（香港）有限公司]]已出版该书的无删节版，名为《解密朝鲜（大陆禁书《真实的朝鲜》完整版）》，价格为[[港币|港币]]98元。

2008年8月作者为此写下“《真实的朝鲜》怎样成为禁书”一文，刊载在2008年九月号的香港[[开放杂志|开放杂志]]。
