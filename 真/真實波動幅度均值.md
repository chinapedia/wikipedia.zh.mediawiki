'''真實波動幅度均值'''（Average True Range，'''ATR'''）是由[[美國|美國]]威爾德（[[:en:J._Welles_Wilder|Welles Wilder JR.]]）所發展出來的[[技術分析|技術分析]]指標，以N天的[[移動平均數#指數移動平均|指數移動平均數]]平均後的交易波動幅度。

== 計算方法 ==
某日的簡單交易幅度是當日之「最高價<math>H_t</math> −最低價<math>L_t</math>」。而'''真實波動幅度'''（True Range，'''TR'''）則包含前一日的收盤價<math>C_{t-1}</math>，是以<math>H_t</math>、<math>L_t</math>、<math>C_{t-1}</math>三個價格做比較，求出當日股價波動的最大幅度。

===〖TR公式〗===
:<math>TR = max(H_t, C_{t-1})- min(L_t, C_{t-1})</math>

:〔註〕
:*<math>H_t</math>表<math>t</math>日的最高價；
:*<math>L_t</math>表<math>t</math>日的最低價；
:*<math>C_{t-1}</math>表<math>t-1</math>日的收盤價；
:*<math>H_t</math>恆大於<math>L_t</math>。

===〖ATR公式〗===
'''真實波動幅度均值'''（'''ATR'''）便是「真實波動幅度」之<math>n</math>日[[移動平均數#指數移動平均|指數移動平均數]]。

:<math>ATR = EMA_{(TR, n)}</math>

== 意義 ==
波動幅度的概念表示可以顯示出交易者的期望和熱情。大幅的或增加中的波動幅度表示交易者在當天可能準備持續買進或賣出股票。波動幅度的減少則表示交易者對股市沒有太大的興趣。
