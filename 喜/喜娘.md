==資格==
依照各地習俗以及家庭、新人本身，對喜娘的資格不盡相同。例如有些同時要求喜娘是「[[好命婆|好命婆]]」或「[[全福人|全福人]]」，有些要求全福人要丈夫、儿女、[[兄弟姐妹|兄弟姐妹]]、公婆、子孙俱全的<ref name="a1"/>，亦有些要求同鄉中與丈夫感情和睦、本身為[[嫡妻|嫡妻]]並有子的年長婦人<ref name="唐榮">{{Cite web |url=http://manwe.lib.u-ryukyu.ac.jp/library/digia/tenji/tenji2012/#17 |title=唐栄林氏婚姻家禮 |accessdate=2016-04-03 |author=林文海|date= |work= |publisher= |language=Zh-hant|quote=選擇同鄉年長夫婦諧和而生嫡子者 }}</ref>。朝鮮傳統則是由新娘的[[姨母|姨母]]、[[姑母|姑母]]或其他年長女親戚擔任<ref name="韓國">{{Cite book |title=韓國的民俗與文化 |author=扈貞煥|date=2006年 |work= |publisher=臺灣商務印書館|isbn=9789570521160 |language=Zh-hant|page=120 }}</ref>。日本古代則要求是四十歲以上、有性經驗的年長婦女。

無論是否職業喜娘，一般都要求較年長的婦女，但也有例外。例如[[清代|清代]][[蘇州|蘇州]]的喜娘就多由年輕婦女擔任<ref name="清稗1">{{Cite web |url=http://open-lit.com/listbook.php?cid=94&gbid=322&bid=14909&start=0 |title=清稗類鈔 奴婢類 |accessdate=2016-04-03 |author=徐珂 |date= |work= |publisher=開放文學 |language=Zh-hant |archive-url=https://web.archive.org/web/20160908172122/http://www.open-lit.com/listbook.php?cid=94&gbid=322&bid=14909&start=0 |archive-date=2016-09-08 |dead-url=yes }}</ref>。而清代[[衡州|衡州]]由於有特殊的[[鬧洞房|鬧洞房]]風俗，喜娘是被鬧的對象，因此要求容貌清麗、歌曲工雅的職業喜娘<ref name="清稗2">{{Cite web|url=http://open-lit.com/listbook.php?cid=41&gbid=322&bid=14856&start=0|title=清稗類鈔 婚姻類|accessdate=2016-04-03|author=徐珂|date=|work=|publisher=|language=Zh-hant}}{{Dead link|date=2018年7月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>。

==職務==
傳統上在正式婚礼前一天，喜娘需在新娘家陪伴新娘，有些會負責為新娘[[上頭|上頭]]。正式婚礼时，需要她履行仪式、講祝福語等。依照各地習俗不同，有些只需待在女家一直陪伴和照顧新娘至新郎上門[[親迎|親迎]]，有些則會前往男家請新郎至女家迎親。浙江古代的送家老嫚和日本古代的介添女還要在正婚禮完結、新人入[[洞房|洞房]]後指導新人[[行房|行房]]。

現代的喜娘除了傳統照顧新娘、協助儀式進行外，有些會在新人策劃婚禮時已經給予意見和指導，甚至帶動氣氛、設計婚禮遊戲等<ref name="大妗" />。除了正式婚禮中外，也會參與婚前[[納徵|納徵]]以及婚禮後{{link-en|見舅姑禮|Pyebaek}}等儀式，甚至婚礼结束后，有些會代替伴娘陪伴新娘[[回門|回門]]<ref name="a2">{{Cite web |url=http://www.qztv.cn/wenzinews.aspx?id=56079 |title=闽都风情：“金牌喜娘”传承福州传统民俗文化 |accessdate=2016-04-03 |author=编辑：龙敏 |date=2012-08-19 |work= |publisher=泉州广播电视台网站，来源：福建新闻网 |language=Zh-hans |archive-url=https://web.archive.org/web/20160426173337/http://www.qztv.cn/wenzinews.aspx?id=56079 |archive-date=2016-04-26 |dead-url=yes }}</ref>。
==職業喜娘的社會地位==
古代中國的職業喜娘被視為賤業，常由[[丐戶|丐戶]]、[[墮民|墮民]]、[[蜑民|蜑民]]等[[賤民|賤民]]或下層平民婦女擔任，例如[[清代|清代]][[紹興|紹興]]的喜婆和[[寧波|寧波]]的送娘子皆由墮民婦女擔任<ref name="嵊泗"/>，被稱為'''墮民嫂'''，[[蘇州|蘇州]]喜娘則是平民婦女<ref name="清稗1" />。由於出身低微，文化水平不高，社會地位也不高。少數地區如清代[[衡州|衡州]]的喜娘由於會在鬧洞房時被男賓調戲，甚至與男賓[[性交|性交]]，以致聲名狼藉<ref name="清稗2" />。直至二十世紀後期，不少職業喜娘學歷也很低，被視為草根文化<ref name="福州">{{Cite web |url=http://pic.fznews.com.cn/jujiao/20150104/54a899641be7b.shtml?p=1 |title=福州喜娘与时俱进创新婚庆习俗　影响力走向全国 |accessdate=2016-04-03 |author=记者：顾伟 |date=2015-01-04 |work= |publisher=福州新聞網，来源：福州晚报 |language=Zh-hans }}</ref>，形象迷信、老舊，被視為只是斟茶遞水的女工<ref name="阿嬸">{{Cite web |url=http://ubeat.com.cuhk.edu.hk/%E4%B8%AD%E5%B9%B4%E5%A9%A6%E5%A5%B3%E5%81%9A%E5%A4%A7%E5%A6%97%EF%BC%9A%E6%88%91%E5%93%8B%E5%94%94%E5%86%8D%E4%BF%82%E3%80%8C%E9%98%BF%E5%AC%B8%E3%80%8D/ |title=中年婦女做大妗：我哋唔再係「阿嬸」 |accessdate=2016-04-03 |author=編輯：植敏欣、記者：劉潔伶 |date=2014-05-12 |work= |publisher= 114期大學線|language=Zh-hant }}</ref>。

自廿一世紀起，職業喜娘被視為一門傳承傳統婚禮文化的婚禮[[顧問|顧問]]及{{link-en|婚禮策劃|Wedding planner}}<ref name="大妗" /><ref name="阿嬸" />，是學貫古今、時尚、思想入流、大方得體的事業女性<ref name="摩登">{{Cite web |url=http://wedding.esdlife.com/articles/wedding-beforemarry/20131104_chinese_customs_consultants/ |title=摩登大妗姐 盡訴心中情 |accessdate=2016-04-03 |author=Connie |date=2013-11-03 |work= |publisher= 新婚生活易|language=Zh-hant }}</ref>。
==與伴娘的差異==
喜娘和[[伴娘|伴娘]]雖然同樣是在婚禮期間照顧新娘和協助儀式，但傳統上兩者的身份、職責以及負責的範圍皆不同。相當於伴娘的在典籍中稱為[[媵|媵]]、[[婦從者|婦從者]]、侍女、侍者等。伴娘主要是陪同新娘由[[娘家|娘家]]前往男家時照料[[新娘|新娘]]，並作為女家代表參與部份儀式，到男家後即與稱為[[御|御]]或[[婿從者|婿從者]]的[[伴郎|伴郎]]交換侍候對象，為[[新郎|新郎]]鋪蓆、洗手、斟酒、奉饌等。而喜娘除了照顧新娘外，還作為禮儀指導者，指導新娘行禮，在婚禮中擔任引導、教導新娘的角色，並且會說吉祥語帶動氣氛，與伴娘在旁協助新娘的角色不同。

無論是古代還是現代，伴娘都是由未婚的新娘同輩或晚輩女性擔任，只有少數地區如[[北京|北京]]例外，當地把未婚伴娘稱為[[伴姑|伴姑]]，已婚伴娘稱為[[伴嫂|伴嫂]]<ref name="老北京">{{Cite web |url=http://www.atoomu.com/life/she/su/NTI0Mzc3.shtml |title=老北京結婚的13項禮儀流程|accessdate=2016-07-24 |author= |date= |work= |publisher=2016-06-25 |language=Zh-hant }}</ref><ref name="舊京" />。富貴家新娘一般由其[[侍婢|侍婢]]陪嫁，常作為[[陪嫁丫头|陪嫁丫头]]，在新娘婚後在新娘夫家繼續侍候主人。一般新娘則以妹妹、未婚同輩或晚輩女性或外聘年輕未婚女性擔任婚禮中的婦從者，婚禮結束後即完成職務回家<ref name="電白">{{Cite web |url=http://www.dbtv.gd.cn/index.php/culture |title=电白民间习俗|accessdate=2016-04-03 |author=徐珂 |date= |work= |publisher=電白廣播電視台 |language=Zh-hant }}</ref><ref name="台灣">{{Cite book |title=台灣舊俗圖集 |author=作者： 劉興欽 |date=2007-11-01 |work= |publisher=聯經出版公司 |isbn=9789570832112 |language=Zh-hant|page=120 }}</ref>。而喜娘則是由較為年長的婦女擔任，雖然實際上也是勞動者，但在婚禮上有指導新娘的[[長輩|長輩]]身份。

在二十世紀後期，不少新人在舉行婚禮時不再聘用喜娘，以致一些本來由喜娘負責的工作和儀式改由伴娘擔任，使兩者角色有所混淆。而部份地區的婚禮傳統上雖然同時有喜娘和伴娘，但[[日常語言|日常語言]]中並不作區分，以致混淆，例如[[漢語|漢語]]中的「送嫁娘」、[[日語|日語]]中的「新婦介添人」、[[韓語|韓語]]的「手母」等，只是籠統地指婚禮上協助新娘的人，並沒有仔細區分身份、職務。而中國古代稱為「伴娘」者多指喜娘，與現代漢語中的「伴娘」所指不同<ref name="清稗2" />。

此外，喜娘在[[華人|華人]]地區和日本皆職業化，伴娘一般並非專門的職業。雖然古今皆有年輕女性受僱擔任伴娘，但尚未形成一種專門[[行業|行業]]。

==與媒人的差異==
由於在中國傳統社會中，不少職業喜娘同時也是職業[[媒人|媒人]]。古人結婚必需由媒人撮合，一些媒婆在撮合新人後又在婚禮上擔任喜娘，以致民間有把喜娘等同媒人的情況。其實喜娘和媒人不但職責有異，嚴格上也不是同一人擔任。在《[[周禮|周禮]]》、《[[朱子家禮|朱子家禮]]》、《[[大明會典|大明會典]]》、《[[明集禮|明集禮]]》等所記載的婚禮中，姆與媒無論身份、職責皆完全不同。媒人的性別在不同文化中有不同規定，例如《周禮》中的媒為男性，後世典籍中的媒人則沒有性別規定，而日本的媒人則是一對[[夫婦|夫婦]]擔任。而現代一些地區的新人雖然是[[自由戀愛|自由戀愛]]結合，但仍依照傳統在婚禮上設有相當於媒人的介紹人，並邀請親友、[[上司|上司]]、[[民意代表|民意代表]]等擔任儀式上的介紹人，而同時又會聘用喜娘負責指導新娘禮儀，兩者各司其職。

媒人在傳統上除了撮合新人，還會在[[納采|納采]]、[[納徵|納徵]]、[[納吉|納吉]]等[[訂婚|訂婚]]儀式中負責來往男女兩家。而喜娘一般只在正婚禮上陪同新娘出門到男家至新人[[合巹|合巹]]、[[同牢|同牢]]，有時會包括見舅姑和歸寧儀式，一般不參與訂婚禮儀。但現代一些地區的婚禮不再設有媒人，於是本來由媒人負責的納徵、[[搬嫁妝|搬嫁妝]]等婚前儀式就由喜娘擔任。現代新式婚禮中的介紹人除了履行傳統儀式外，還會在婚禮上[[致詞|致詞]]，喜娘則只負責指導禮儀和帶動氣氛。

==與贊禮的差異==
[[贊禮|贊禮]]即婚禮[[司儀|司儀]]，與喜娘同樣有指導禮儀的職責。但贊禮是於婚禮場地中負責主持、主導婚禮儀式程序，並於場中大聲宣讀程序、儀式，要求聲線洪亮，本身不需要貼身照顧、指導新娘。而喜娘則是在新娘身邊指導新娘依照贊禮的指示行禮。

==與新娘秘書的差異==
[[新娘秘書|新娘秘書]]是21世紀新興的行業，只需負責新娘的髮飾、彩妝和服裝造型，不必深入瞭解婚嫁的民俗和禮儀，本身亦非婚禮儀式的參與者。
