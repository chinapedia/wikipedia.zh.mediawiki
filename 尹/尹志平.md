'''尹志平'''（{{bd|1169年||1251年|catIdx=尹}}），字'''大和'''，道號'''清和子'''，山東[[莱州_(古代)|萊州]]人，[[全真教|全真教]]第六任掌教，[[劉處玄|劉處玄]]與[[丘處機|丘處機]]弟子，自少好學仙道，在山東一帶道觀修道，勤於打坐祈拜，隨丘處機到[[撒馬爾罕|撒馬爾罕]]覲見[[成吉思汗|成吉思汗]]。丘處機歸真後，尹志平嗣位掌教，創建[[燕京|燕京]][[白雲觀|白雲觀]]，為朝廷舉行祈福[[齋醮|齋醮]]，到多處宮觀講道，德高望重。期間全真教趨於極盛，信徒眾多，吸納和重建各地宮觀，歸入全真教名下主持。尹志平掌教10年，傳位師弟[[李志常|李志常]]，退位歸隱，[[諡號|諡號]]「'''清和妙道廣化真人'''」<ref name="鄭">{{cite journal|author=鄭素春|title=〈全真道士尹志平（1169-1251）的宗教實踐〉|url=http://www.rsd.fju.edu.tw/Upload/22-07%E5%B0%B9%E5%BF%97%E5%B9%B3%E7%9A%84%E5%AE%97%E6%95%99%E5%AF%A6%E8%B8%90(1).pdf|journal=《輔仁宗教研究》|year=2011|volume=22|pages=109-138|accessdate=2017-12-30|language=zh-hant}}</ref>。

== 生平 ==
=== 入道 ===
尹志平祖先在北宋時是官宦之家，父、祖樂善好施<ref name="鄭"/>{{rp|114}}，1182年13歲時，遇到[[全真教|全真教]]第二任掌教[[馬丹陽|馬丹陽]]，出家入道，成為全真教道士，1187年，為父親所逼而還俗，被鎖於家中<ref name="蘇">{{cite book |author=蘇德樸（Eskildsen Stephen）|others=宋學立譯|editor=張廣保|title=《多重視野下的西方全真教研究》| pages=129-156|chapter=〈尋找証驗：全真教早期的內境和其他入定現象〉|publisher=齊魯書社|location=濟南|year=2013|isbn=7533325761|language=zh-hans}}</ref>{{rp|133}}，尹志平再三逃走，家人方准許他入道。尹志平到[[洛陽|洛陽]]拜訪[[王重陽|王重陽]]弟子[[劉處玄|劉處玄]]<ref name="鄭"/>{{rp|114-115}}，拜劉處玄為師<ref name="蘇"/>{{rp|133}}，其父母又追來阻止，把他鎖住，尹志平再次逃去，到山東[[昌邑市|昌邑]]縣結庵修道，大有感悟，移居[[福山區|福山]]草庵，在當地照顧貧病弱小，到[[濰州|濰州]]雲遊時，金朝官員完顏龍虎讓尹志平居於玉清觀。1191年，尹志平到棲霞觀拜[[丘處機|丘處機]]為師，受丘處機器重。此後尹志平再向王重陽弟子[[郝大通|郝大通]]請教《[[易經|易經]]》，自[[王處一|王處一]]傳受籙法，聲譽日漸提高，前來就學的人愈來愈多。1216年，蒙古入侵，尹志平到海島避禍，戰亂後回到[[濰州|濰州]]玉清觀<ref name="鄭"/>{{rp|115}}。

1219年，[[成吉思汗|成吉思汗]]派[[劉仲祿|劉仲祿]]為使者，請丘處機前去覲見，求傳授長生之術<ref name="姚">{{cite book |author=姚道中|others=宋學立譯|editor=張廣保|title=《多重視野下的西方全真教研究》|pages=157-177|chapter=〈丘處機與成吉思汗〉|publisher=齊魯書社|location=濟南|year=2013|isbn=7533325761|language=zh-hans}}</ref>{{rp|163}}，劉仲祿先到濰州會見尹志平，尹志平認為可趁此機會，以全真教義勸化蒙古人，遂陪同劉仲祿到[[萊州_(古代)|萊州]]拜見丘處機，遊說師父應邀。丘處機挑選弟子18人隨行，於1220年出發<ref name="姚"/>{{rp|165}}，1221年到達[[撒馬爾罕|撒馬爾罕]]，1222年南下到達成吉思汗汗庭，並與可汗同回撒馬爾罕<ref name="姚"/>{{rp|168-169}}。1223年，丘處機率弟子東還，到[[雲中|雲中]]時，聽聞山東戰亂，派遣尹志平南下招撫勸降。次年丘處機居於[[燕京|燕京]][[長春宮_(道觀)|長春宮]]，前來禮敬的信眾甚多，尹志平不敢受此供奉，先後到河北[[縉雲山_(延慶山峰)|縉雲山]]秋陽觀、[[德興府|德興]]龍陽觀隱居<ref name="鄭"/>{{rp|116}}。1227年，丘處機仙遊時，遺令宋道安繼位，尹志平為第一副手。宋道安因年紀已老，請尹志平代替自己出任掌教<ref name="景">{{cite book|author=景安寧|title=《道教全真派宮觀、造像與祖師》|publisher=中華書局|location=北京|year=2012|isbn=7101086888|language=zh-hans}}</ref>{{rp|84}}。

===掌教===
尹志平嗣教後，把燕京[[長春宮_(道觀)|長春宮]]東部改建為[[白雲觀|白雲觀]]，在觀內修建處順堂，供奉[[丘處機|丘處機]]仙骨，並請人在堂內畫上丘處機西遊的壁畫。1230年，有人誣告處順堂壁畫不敬，竟在靈堂繪畫會見蒙古大汗一事，蒙古人乘機打擊全真教，尹志平即日下獄，後來為師弟[[李志常|李志常]]代替入獄<ref name="景"/>{{rp|187}}。1231年，尹志平到河北[[灤平县|灤平]]主持[[齋醮|齋醮]]，為國祈福。1232年，[[窩闊台|窩闊台]]可汗到[[順天府|順天府]]，尹志平前往迎見<ref name="鄭"/>{{rp|117}}。1233年，尹志平到燕京舉行[[黃籙齋|黃籙齋]]醮，提出讓位予李志常，眾人反對，他又到燕京華陽觀講道，到東北[[義州|義州]]舉行下元醮，在通仙觀過冬。次年遊[[醫巫閭山|閭山]]太玄觀，然後回到燕京<ref name="鄭"/>{{rp|118}}。1235年，尹志平奉詔在[[和林|和林]]修築道院<ref name="鄭"/>{{rp|126}}。金末蒙古入侵關中，陝西[[鄠縣|鄠縣]]全真教祖庭靈虛觀被毀，道士遇害，[[京兆府|京兆府]][[總管|總管]]田雄信奉全真教，去信請尹志平前來。1236年，尹志平來到祖庭，籌劃重建道觀<ref name="景"/>{{rp|180}}，田雄並把關中許多道觀託付尹志平<ref name="鄭"/>{{rp|119}}。尹志平到[[終南山|終南山]]已殘破的宗聖觀，委託李志柔加以重修<ref name="景"/>{{rp|199}}，回程時，與[[李志常|李志常]]到[[雲中|雲中]]接受聖旨，剃度道士千人，次年回到燕京。尹志平素常有退隱之志，教門事務多交予李志常負責，與蒙古朝廷的往來，往往也透過李志常進行。1238年正月，尹志平傳位予李志常，共計掌教10年<ref name="鄭"/>{{rp|126、119}}。

=== 退隱 ===
1239年，尹志平修建[[燕京|燕京]]附近的五華山燒丹院，次年移居[[大房山|大房山]]真陽觀<ref name="鄭"/>{{rp|119}}。1241年初，尹志平應邀再到祖庭[[重阳宫|重陽宮]]（前靈虛觀），主持重葬[[王重陽|王重陽]]的儀式，參與者達數千人<ref name="景"/>{{rp|181}}。1242年，尹志平到[[沁州|沁州]]神霄宮，翌年移居[[太原|太原]]天寶觀，再回五華山建五華宮。1249年，元廷下旨賜號「清和演道玄德真人」，改五華觀為五華宮。1251年，在大房山清和宮（前真陽觀）歸真，年82<ref name="鄭"/>{{rp|120}}。
== 修行 ==
[[File:尹志平《葆光集》.png|thumb]]
尹志平天資甚高，孩童時，不經意地已有出神忘我的入定經驗，「冥然漠然，不覺心形俱喪」<ref name="蘇"/>{{rp|131}}。入道後，習慣每晚在[[北斗七星|北斗]]下祈拜，並打坐瞑想；拜[[劉處玄|劉處玄]]為師後，夜間在樹下[[靜坐_(冥想)|靜坐]]存思時，在異象中見劉處玄前來，揮刀斷其頭，意思是換去尹志平的頭面；10日後又見劉處玄來，剖出其心，意思是去除其俗心；再10日後，劉處玄來剖開其腹，取去一切內臟。自此尹志平認識到自己已超越塵俗，由俗人變成道人<ref name="蘇"/>{{rp|133-134}}。後來尹志平仍有不少[[神秘主義|神祕體驗]]，如曾在打坐中，感到體內之氣向上沖湧，頭頂應聲裂開，「甘液」從上而降<ref name="蘇"/>{{rp|151}}。全真教著重「功行」，即勤奮刻苦的修煉，與行善之舉<ref name="蘇"/>{{rp|152}}。尹志平認為，人的悟道成道，與所積的福德厚薄有關。道士積累「功行」，可以影響來世，來生與師父或聖賢再聚，一起修行，直至達致真道<ref name="鄭"/>{{rp|123-124}}。人要立志，節制情慾；佛道二家的心性修為，並無分別，信眾不必刻意追求通靈顯聖之事，只要功行積累深厚，心靈自得<ref name="鄭"/>{{rp|128-129}}。
== 貢獻 ==
[[File:尹志平《清和真人北遊語錄》.png|thumb]]
尹志平曾拯救百姓，蒙古人入侵時向百姓招撫諭降，救活不少性命，皈依入於[[全真道|全真道]]者，可以免差役和賦稅<ref name="鄭"/>{{rp|124}}。掌教時，他修復許多戰後荒廢的的[[宮觀|宮觀]]，並把許多舊有宮觀吸納入全真教，任命全真教高道為住持，並開始為全真教制定「全真[[清規|清規]]」，掌教期間師弟[[宋德方|宋德方]]開始編纂[[道藏|道藏]]《[[玄都寶藏|玄都寶藏]]》<ref name="鄭"/>{{rp|131-132}}。尹志平嗣教後，信徒眾多，供奉甚豐，廣受愛戴，出遊時百姓沿途膜拜，可謂道門精神領袖，因此重葬[[王重陽|王重陽]]時，尹志平雖已退位，亦由他主持大典<ref name="鄭"/>{{rp|117、127}}。[[丘處機|丘處機]]晚年時，全真教已是北方最大教派<ref name="景"/>{{rp|112}}。尹志平掌教時，全真教趨於極盛，元代賈戫稱「教風之盛，自三代而下，未有若此時也」<ref name="鄭"/>{{rp|130}}；弋彀說尹志平「徒侶遍天下，聞望重朝野。……自古教法之盛，功德之隆，惟清和師為最<ref name="鄭"/>{{rp|134}}。」尹志平詩文書信編為《葆光集》3卷，在[[燕京|燕京]]及出遊東北時的講道，記錄為《清和真人北遊語錄》4卷<ref name="鄭"/>{{rp|128}}。

== 小說改編 ==
[[金庸|金庸]][[武俠小說|武俠小說]]《[[射雕英雄傳|射雕英雄傳]]》、《[[神雕俠侣|神雕俠侣]]》中均有出現。唯在《神雕俠侣》原版中，尹志平被設定迷戀女主角[[小龍女|小龍女]]，晚上夢囈唸著小龍女，又一遍一遍在紙上寫下其名字<ref name="王">{{cite book|author=王怡仁|title=《金庸妙手改神雕》|url=http://books.google.com.hk/books?id=4lyHAgAAQBAJ&printsec=frontcover&hl=zh-TW#v=onepage&q&f=false|publisher=心一堂有限公司|location=香港|year=2013|isbn=9888058991|language=zh-hant}}</ref>{{rp|46}}。一次他乘[[小龍女|小龍女]]無法動彈，加以迷姦，事後他對全真教和[[丘處機|丘處機]]有愧，心裏卻對迷姦小龍女沾沾自喜；後來尹志平為護救小龍女，胸口撞上小龍女劍尖而逝去<ref name="王"/>{{rp|191、193、202}}。根據一項在[[河南|河南]]所做的調查，受《神雕俠侶》影響，人們普遍認為尹志平人品不好，很少人了解到尹志平是個「道行高深的道士」<ref name="孫">{{Cite news|url=http://newpaper.dahe.cn/hnsb/images/2014-06/19/A12/12.pdf|title=〈輕薄小龍女？尹志平沒幹這事兒〉|author=孫科|pages=A12|language=zh-hans|publisher=《河南商報》|date=2014-06-19|accessdate=2017-12-29}}</ref>。2003年，金庸到[[華山|華山]]出席「華山論劍」節目，陝西省道教協會的道士加以攔阻，抗議《神雕俠侶》虛構尹志平污辱小龍女一事，侮辱全真教<ref name="孫"/>。在小說修訂三版中，金庸更動角色設定，把污辱小龍女的人改寫成虛構人物「[[甄志丙|甄志丙]]」，性格亦不再那麼污濁<ref name="王"/>{{rp|191、193-194}}，而尹志平則改為在《神雕俠侣》時期醉心內丹煉氣之道，極少涉足江湖而沒什麼戲分。全真教亦寫得更光明磊落，胸襟廣闊<ref name="王"/>{{rp|202-206}}。
