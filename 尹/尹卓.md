'''尹卓'''（{{bd|1945年|9月||}}），[[汉族|汉族]]，[[中华人民共和国|中华人民共和国]]政治人物與將領，开国少将尹明亮之子，[[第十一届全国政协委员|第十一届全国政协委员]]。中国人民解放军海军少将。央视《[[军情连连看|军情连连看]]》等节目嘉宾主持<ref>[http://military.people.com.cn/GB/52578/52579/182515/11028030.html 尹卓，人民网]</ref>。

早年加入[[中国共产党|中国共产党]]，担任海军信息化专家咨询委员会主任。2008年，当选第十一届全国政协委员<ref>{{cite news|title=第十一届全国政协委员名单|url=http://www.indaa.com.cn/zt/2009qglh/zxzs/200903/t20090303_127573.html|accessdate=2013-04-09|work=新华社北京1月25日电 中国人民政治协商会议第十一届全国委员会委员名单(共2237人，2008年1月25日政协第十届全国委员会常务委员会第二十次会议通过)|date=2008年1月25日|deadurl=yes|archiveurl=https://web.archive.org/web/20130928014403/http://www.indaa.com.cn/zt/2009qglh/zxzs/200903/t20090303_127573.html|archivedate=2013年9月28日|df=}}</ref>，代表科学技术界，分入第二十九组。<ref>{{cite web |title=全国政协信息-尹卓 |url = http://www.cppcc.gov.cn/CMS/wylibary/showJcwyxtInfoWylibary.action?tabJcwyxt.guid=11W0009910&wytype=wjwy |accessdate=2013-04-11 |publisher = 全国政协}}</ref>

== 家庭 ==
父亲[[尹明亮|尹明亮]]（1915-1999），是中华人民共和国开国少将之一，曾任[[福州军区|福州军区]]副政委。
