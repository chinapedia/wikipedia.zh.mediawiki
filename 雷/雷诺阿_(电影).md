《'''雷诺阿'''》（''Renoir''），片名又稱《印象雷諾瓦》，是一部2012年[[法国|法国]]剧情片，根据法国[[印象派|印象派]]画家[[皮埃尔·奥古斯特·雷诺阿|皮埃尔·奥古斯特·雷诺阿]]晚年的经历改编而成。<ref name="theupcoming">{{cite web |url=http://www.theupcoming.co.uk/2013/03/04/film-review-renoir/|title=Film review: Renoir |accessdate=2013-03-26 |work=The Upcoming}}</ref>导演为[[吉爾布都|吉爾布都]]，入选[[第65届戛纳电影节|第65届戛纳电影节]]一种关注单元<ref name="Official Selection">{{cite web |url=http://www.festival-cannes.fr/en/article/58878.html |title=2012 Official Selection |accessdate=2012-05-20 |work=Cannes}}</ref><ref name="guardian">{{cite web |url=http://www.guardian.co.uk/film/2012/apr/30/cannes-2012-seven-films-lineup |title=Cannes 2012: seven films join the lineup |accessdate=2012-05-20 |work=The Guardian}}</ref>。本片代表法国角逐[[第86届奥斯卡金像奖|第86届奥斯卡金像奖]]最佳外语片<ref name="France">{{cite web |url=http://www.lexpress.fr/culture/cinema/renoir-representera-la-france-aux-oscars-2014_1282079.html |title= Renoir représentera la France aux Oscars 2014|accessdate=2013-09-16 |work=L'Express}}</ref><ref name="HWR">{{cite web |url=http://www.hollywoodreporter.com/news/oscars-france-nominates-renoir-foreign-629841 |title= Oscars: France Nominates 'Renoir' in Foreign Language Category |accessdate=2013-09-16 |work=Hollywood Reporter}}</ref><ref name="Guardian">{{cite web |url=http://www.theguardian.com/film/2013/sep/16/renoir-biopic-oscars-submission |title=Renoir biopic to be France's official Oscars submission |accessdate=2013-09-17 |work=The Guardian}}</ref>。

==剧情==
讲述法国[[印象派|印象派]]画家[[皮埃尔·奥古斯特·雷诺阿|皮埃尔·奥古斯特·雷诺阿]]晚年请来了一位女[[模特儿|模特儿]]安德蕾，她不仅提升了雷诺阿的创作热情，也改变了雷诺阿二儿子[[让·雷诺阿|让·雷诺阿]]的人生。让·雷诺阿从[[一战|一战]]战场腿部负伤归来，在与安德蕾相处过程中两人恋爱。然而待让·雷诺阿身体恢复健康后，他又毅然回到军队继续参战。皮埃尔·雷诺阿于1919年逝世。

最终让·雷诺阿成为一名[[电影导演|电影导演]]，安德蕾成为一名女[[演员|演员]]，1920年至1930年间是一对夫妻，两人于1979年逝世。
