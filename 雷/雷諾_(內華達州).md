'''里诺'''（{{lang-en|'''Reno'''}}）是[[美國|美國]][[內華達州|內華達州]]北部[[瓦肖县|瓦肖县]] (Washoe County) 裡的一個城市，亦是郡政府的所在地。根據2010年人口普查，雷諾市區內人口為22萬5221人，處於[[拉斯維加斯|拉斯維加斯]]（Las Vegas）及[[亨德森|亨德森]]（Henderson）之後，是內華達州第三大城市，據2015年的估計雷諾的人口則已達24萬1445人。雷諾位於內華達州首府[[卡森城_(內華達州)|卡森城]]（Carson City）北面約{{convert|26|mile|km}}、[[太浩湖|太浩湖]]（Lake Tahoe）東北面約{{convert|22|mile|km}}的地方，城市的東面與[[斯帕克斯|斯帕克斯]]（Sparks）接壤。雷諾被稱為“世界上最大的小城市”，以[[博彩業|博彩業]]聞名，亦是著名博彩集團凱撒娛樂集企業（Caesars Entertainment Corporation，舊稱Harrah's Entertainment）的發源地。另外，全世界最大的[[角子機|角子機]]製造商[[國際博弈科技公司|國際博弈科技公司]]（International Game Technology，簡稱IGT）的總部所在地亦設於此城。

== 歷史 ==
早於1850年代，就已有少量[[拓荒者|拓荒者]]居於雷諾的土地上。他們大多數聚居於[[特拉基河|特拉基河]]（Truckee River）沿岸流域的[[特拉基草原|特拉基草原]]（Truckee Meadows），特拉基河連接著[[太浩湖|太浩湖]]和[[金字塔湖|金字塔湖]]（Pyramid Lake）的水道，流經的特拉基草原的土地相當肥沃，適合[[農業|農業]]耕種。故此，這些拓荒者就開始在當地種植農作物，繼而售予途經的人獲取利潤。當時的人如要西行往[[加利福尼亞州|加利福尼亞州]]的話，都要經過[[唐納湖|唐納湖]]（Donner Lake），該附近一帶的地勢險要，旅者通常都會先在特拉基草原購置糧食以作補給。

淘金熱時期，在鄰近的[[維吉尼亞城_(內華達州)|維吉尼亞城]]（Virginia City）發現了[[金|金]]礦（1850年），及後又於1859年發現[[銀|銀]]礦，大批淘金者湧至當地的哥斯托克礦脈（Comstock Lode），特拉基草原頓成了哥斯托克礦脈與外間的連繫要點。

為了將維吉尼亞城與通往加利福尼亞州的小道相連接，查理斯·富勒（Charles Fuller）於1859年在特拉基河上建起了一條收費木橋作連結之用。不久，小橋附近慢慢形成了一個社區，開始有人開設旅店以招待路過的客人。兩年後，富勒將此橋售予馬龍·歷克（Myron Lake），他繼續將這個社區發展，增設了不少旅館和餐店。而為了紀念歷克對這個小社區的貢獻，社區的名稱被起為“[[歷克渡口|歷克渡口]]”（Lake's Crossing）。

[[File:Reno_with_mountains.png|250px]]
[[File:Reno_BLC.jpg|250px]]
1868年，因有感各州的交通未能完善接通，[[中太平洋鐵路公司|中太平洋鐵路公司]]（Central Pacific Railroad）打算加建一條西行鐵路，連接[[聯合太平洋鐵路公司|聯合太平洋鐵路公司]]（Union Pacific）的鐵路，以形成國內第一條跨州份鐵路。馬龍認為鐵路的興建可以帶來商機，遂將土地無償給予中太平洋鐵路公司，以換取他們在歷克渡口設立火車站。1868年5月13日火車站建成，同日雷諾亦正式成立。新市鎮的名字取自[[美國內戰|美國內戰]]時陣亡的聯邦軍軍官積西·里诺[[陸軍少將|陸軍少將]]（Major General Jesse L. Reno）的姓氏。

1872年，[[維吉尼亞與特拉基鐵路|維吉尼亞與特拉基鐵路]]（Virginia and Truckee Railroad）伸延至雷諾，為新城市的經濟帶大的重大的發展。在此之後的幾十年來，雷諾慢慢發展成為商業和農業的中心，更成為[[薩克拉門托|薩克拉門托]]和[[鹽湖城|鹽湖城]]之間的州際鐵路間的中途站，吸引了不少外來旅客。但是，雖然雷諾的商業經濟發展蓬勃，但內華達州政府的發展重點卻仍然放在礦業身上。內華達州最先重點開發維吉尼亞城、然後再接而發展[[杜魯巴|杜魯巴]]（Tonopah）和[[高菲特|高菲特]]（Goldfield）兩個礦產豐富的城市，但依然未把重點放在雷諾身上。

一直到20世紀初期，由於淘金熱開始退卻，礦業開始式微。內華達州政府開始將資料轉移到一些非礦業為主的城市，其中最重點開發的就是雷諾和[[拉斯維加斯|拉斯維加斯]]。但與此同時，有一些礦業為主要產業的小城市就開始荒廢。雖然內華達州的礦業已不及上兩個世紀般發達，但內華達州現時的[[黃金|黃金]]產量仍佔全世界的11%。

拉斯維加斯素以方便的[[結婚|結婚]]程序而成為結婚的熱門城市，但身處同州的雷諾則是20世紀初期美國極其著名的“離婚城”。在當時，[[離婚|離婚]]一直被認為是一件不光榮的事情，甚少人會談及或想到離婚這事兒。不過，雷諾於1908年則通過了一項著名的“免責離婚”法案，說明男女雙方只需要同住滿一段時期、及繳交一筆定額費用就可到雷諾申請離婚，程式簡單快捷。這項法案吸引了當時美國各州成千上萬的夫婦到該城辦理離婚手續。及後，內華達州政府就逐步將要求同住的期限縮短，由最初的半年，慢慢縮短至1927年通過的3個月、以及1931年通過的6個星期，辦理離婚頓成為了雷諾的支柱產業。加上1931年內華達州政府亦通過將賭博合法化，頓時為雷諾帶來了重大的經濟效益。

2000年代初期，雷諾面對著高速的經濟增長。由於生活成本比起相鄰的[[加利福尼亞州|加利福尼亞州]]的大部分地區要低，故此雷諾逐掀起一股房產熱潮，大型的建築工程於雷諾隨處可見。但是，房價亦同時地以幾何級數遞升。[[雷諾|雷諾]]於2006年成為全美國最高樓價的城區之一，但2007年底的美國[[經濟大衰退|經濟大衰退]]連帶導致雷諾地區房市場崩潰，對該地區經濟造成巨大衝擊。

== 氣候 ==
雷諾地處[[海拔|海拔]]4,400呎之上的[[沙漠|沙漠]]山谷地。[[氣候|氣候]]方面則屬於四季分明的[[温带大陆性气候|温带大陆性气候]]，不過比起美國的其他地方來說，雷諾是相對溫暖的城市。雷諾的[[冬季|冬季]]會降雪，不過雪量一般都不會很多。另外，降雨多於冬、[[春|春]]二季，[[夏季|夏季]]和[[秋季|秋季]]則非常乾燥，與加利福尼亞州頗為相似。雖然在1990年代開始，雷諾的平均氣溫開始逐漸提高，但都甚少超過[[攝氏|攝氏]]37度或以上。每年七月的氣溫約為攝氏10度至32度，而一月的時候則會降至攝氏零下5度至攝氏7度左右。

== 教育 ==
[[File:Reno2002.jpg|250px]]
雷諾的學校主要分為公立及私立，其中以公立學校為主。雷諾的公立學校屬於瓦肖郡校區（Washoe County School District）轄下，設有十一間高中、以及初中及小學等教育設施。

[[內華達大學雷諾分校|內華達大學雷諾分校]]（University of Neveda, Reno，簡稱UNR）是內華達州最古老的大學，大學另設有校區於拉斯維加斯（University of Neveda, Las Vegas，簡稱UNLV）。學校始創於1886年，原先只是預科學院，後期已發展成一所規模完善的大學，提供有礦業工程、農業、旅遊業等課程，更是全美提供[[巴斯克|巴斯克]]研究課程的兩所學院其中之一。另外，雷諾另有一座社區大學：特拉基草原社區大學（Truckee Meadows Community College，簡稱TMCC）。

== 博彩業 ==
[[File:Reno_–_Circus_Circus_and_Silver_Legacy_casinos.jpg|thumb]]
[[File:IGT_headquarters_offices.jpg|thumb]]
在1960年代之前，雷諾曾經是全世界博彩業最著名的城市，但由於[[拉斯維加斯|拉斯維加斯]]的急速冒升、[[美國航空|美國航空]]收購了雷諾航空令航班減少、以及加利福尼亞州的[[印第安博彩|印第安博彩]]（Indian Gaming）的興起，令雷諾的博彩收益急劇下降。一些小賭場相繼結業，而大賭場亦只能以極微的收益勉強維持，特別在冬季的時候，客源就顯得比平常更少。只有在週末、假日以及一些特別日子，雷諾的賭場收益才能有增長。

面臨著旅遊博彩業的重大危機，雷諾舉辦了很多特別活動來刺激產業，包括有車展、小型[[賽車|賽車]]、[[保齡球|保齡球]]賽事以及[[熱氣球|熱氣球]]競賽等，這一系列的活動成功令雷諾的旅客數字逐漸回升。

雷諾的賭場还经常举办大型的文艺演出。包括[[梅艳芳|梅艳芳]]、[[梁咏琪|梁咏琪]]和[[鍾嘉欣|鍾嘉欣]]等人在内的国际知名艺人都曾在雷諾举办过演唱会，主要观众来自附近的[[加州|加州]]。

== 運動展業 ==
雷諾的運動產業尚稱發達。[[內華達大學雷諾分校|內華達大學雷諾分校]]的所屬體育部門以內華達狼群隊為名，數次參加了[[籃球|籃球]]及[[美式足球|美式足球]]等美國[[全國大學體育協會|NCAA]]所舉辦的賽事，並取得不錯的成績。其他職業運動組織包括籃球[[國家籃球發展聯盟|NBDL]]下的[[雷諾大角羊隊|雷諾大角羊隊]]（Reno Bighorns）、以及棒球3A[[小聯盟|小聯盟]][[太平洋岸聯盟|太平洋海岸聯盟]]（Pacific Coast Conference）下的[[雷諾王牌隊|雷諾王牌隊]]（Reno Aces）。

== 舊區復興 ==
由於舊城區有很多小型[[賭場|賭場]]結業，這些賭場的遺址就成了空置的樓房。這亦吸引了一些其他州份的地產發展商將這些空置的建築物購入，再將之重建為公寓。

另外，由於舊城區的維吉尼亞街（Virginia St.）的噪音和交通問題日益嚴重，當局遂成立一個名為[http://www.renoretrac.com/ 雷諾溝道計劃]（'''Reno Trench Project'''，簡稱'''ReTRAC'''），將一些現存火車軌道重建，並安排火車軌道建於街道之下，此舉令居民的生活得到改善，亦改善了舊城區於旅客心目中的形象。整個計劃已於2005年11月22日完成。

== 人口 ==
根據2000年的人口普查，雷諾常住人口約為180,480人，分佈73,904戶及41,681個家庭內。[[人口密度|人口密度]]約為每[[平方公里|平方公里]]1,008.3人，另每平方公里則有443.9個居住單位。膚色人種分佈則主要為[[白人|白人]]，佔77.46%，[[非洲|非]]裔美國人佔2.58%，印地安人口為1.25%，來自[[太平洋|太平洋]]島國的人種佔0.56%，[[亞洲|亞]]裔人佔5.29%，混血人種佔3.6%，19.18%為[[拉丁|拉丁]]裔人，而其他人種則共佔9.26%。

73,904戶當中有27.6%的是與18歲以下小童同住的，40.5%為夫婦同住，10.6%為未婚女性當家，43.6%獨居或為非家庭成員同住。每戶平均組成人口為2.38人，而每個家庭的組成人口則為3.06人。

年齡分佈方面，18歲以下人口約佔23.2%，11.4%則為65歲以上人士。年齡中位數為34歲，人口分佈上男性比較多，平均每104個成年男性才有100個女性。

每戶年度入息中位數為40,530[[美元|美元]]、家庭年度入息中位數則為49,582美元。男性的入息中位數比女性的高，分別是33,204美元對26,763美元一年。全個城市的個人平均收入為22,520美元，當中12.6%的人口生活於[[貧窮線|貧窮線]]之下。
