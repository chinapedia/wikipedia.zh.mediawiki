雷内.勒维（René Lew），巴黎精神病学家、拉康派著名精神分析家。
==介紹==
雷内·勒维是法国结构主义思想家、[[精神分析学|精神分析学]]家[[雅克·拉康|雅克·拉康]]<ref>{{Cite web|url=https://fr.wikipedia.org/wiki/Jacques_Lacan/|title=Jacques_Lacan|deadurl=yes|archiveurl=https://web.archive.org/web/20170501185617/https://fr.wikipedia.org/wiki/Jacques_Lacan|archivedate=2017-05-01}}</ref>的弟子，曾是由拉康亲手创办的[[巴黎弗洛伊德事业学院（École_de_la_cause_freudienne）|巴黎弗洛伊德事业学院（École de la cause freudienne）]]<ref>{{Cite web |url=http://www.causefreudienne.net/|title=causefreudienne}}</ref>的成员。1981年拉康去世后不久，他退出该协会，并且于1984年创立拉康派协会精神分析之维（dimension de la psychanalyse），是该协会的督导分析家<ref>{{Cite web |url=http://dimpsy.online.fr/ |title=dimpsy.online}}</ref>。他现在还担任国际拉康组织Convergencia（聚合）<ref>{{Cite web |url=http://www.convergenciafreudlacan.org/|title=convergenciafreudlacan}}</ref>的法国秘书长，曾在埃斯基罗尔精神病院（Hôpital Esquirol）<ref>{{Cite web |url=http://www.hopital-esquirol.fr/ |title=esquiro |deadurl=yes |archiveurl=https://web.archive.org/web/20160716184857/http://www.hopital-esquirol.fr/ |archivedate=2016-07-16 }}</ref>长年开展工作，实践精神分析的临床，并且他还曾担任塞纳河畔伊夫里（Ivry-sur-Seine）医学与精神教育中心的主任医师。1987年，他在巴黎11区创立了精神分析门诊。<ref>{{Cite web |url=http://www.lapartdeloeil.be/fr/auteurs_details.php?vid=132 |title=La Part de l’Œil }}</ref>他还是1982年创建的利西马科斯协会（Lysimaque）的主席。勒维出版了众多文章，会议论文，以及书籍<ref>{{Cite web |url=http://lysimaque.fr |title=Un trajet}}</ref>。

勒维先生目前积极地在医院以及C. M. P. P.（医学心理学教育中心）开展多个研讨班，他发展出了一套关于自闭症<ref>{{Cite web |url=https://zh.wikipedia.org/wiki/自閉症|title=自閉症}}</ref>和精神病<ref>{{Cite web |url=https://zh.wikipedia.org/wiki/精神病|title=精神病}}</ref>的精神分析理论，该观念已经开始成为对青少年工作接待的典型模式之一。 他与精神病开展个体精神分析治疗工作长达三十余年。他曾多次前往欧洲各地、南美，甚至中国成都精神分析中心，参加精神分析大会并开展精神分析的培训工作。

由他编辑的精神分析期刊有：眼眸（La part de l`œil），弗洛伊德阅读笔记（Cahiers de lectures freudiennes）共22期<ref>{{Cite web |url=http://www.lutecium.org/pro.wanadoo.fr/espace.freud/topos/psycha/lysima/lysimaq.htm |title=La Lysimaque Cahiers de lectures freudiennes }}</ref>。
