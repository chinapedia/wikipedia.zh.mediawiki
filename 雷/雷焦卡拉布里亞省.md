'''雷焦卡拉布里亞省'''（{{lang-it|Provincia di Reggio Calabria}}）是[[意大利|意大利]][[卡拉布里亚大区|卡拉布里亚大区]]的一个[[意大利省份列表|省]]。它是意大利大陆最南端的省份，与西西里岛之间隔着[[墨西拿海峡|墨西拿海峡]]。[[阿斯普羅蒙特山|阿斯普羅蒙特山]]位于西部地区，由于雷焦卡拉布里亞省拥有很长的海岸线，因此在夏季是一个受欢迎的旅游目的地。首府为[[雷焦卡拉布里亚|雷焦卡拉布里亚]]。
 
自2018年起被[[雷焦卡拉布里亞廣域市|雷焦卡拉布里亞廣域市]]所取代。

== 经济 ==

该地区因盛产[[香柠檬|香柠檬]]而闻名，产区主要在[[意大利|意大利]][[卡拉布里亚大区|卡拉布里亚大区]]的[[爱奥尼亚海|爱奥尼亚海]]地区，在某种程度上，它是整个地区的象征。<ref>{{cite web |url=http://www.tradewindsfruit.com/content/bergamot.htm |title=Bergamot: ''Citrus aurantium ssp bergamia'' |publisher=Trade Wind Fruits |accessdate=2015-09-26}}</ref>大部分香柠檬来自温度适宜的地方。它还在[[法国|法国]]，[[阿根廷|阿根廷]]，[[巴西|巴西]]，[[土耳其|土耳其]]和美国[[喬治亞州|-{zh-cn:佐治亚州; zh-tw:喬治亞州; zh-hk:佐治亞州;}-]]种植。Piana di Gioia Tauro肥沃的地区种植[[克里曼丁红橘|克里曼丁红橘]]。<ref name=Discover>{{cite web |url=http://www.italia.it/en/discover-italy/calabria/reggio-calabria.html |title=Reggio Calabria |publisher=Discover Italy |accessdate=2015-09-26}}</ref>该地区还种植其他柑橘类水果和橄榄树，当地的许多经济体都参与了橄榄油的提取和柑橘类产品的加工。
