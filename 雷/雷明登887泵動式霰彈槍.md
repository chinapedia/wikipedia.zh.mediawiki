'''雷明登887“硝基馬格南”'''（{{Lang-en|'''Remington Model 887 Nitro Mag'''}}）是一款由[[美国|美国]]槍械製造商[[雷明登武器公司|雷明登武器公司]]所研製及生產的[[泵动式|泵动式]][[霰彈槍|霰彈槍]]，可以發射2{{frac|3|4}}英吋、3英吋和3{{frac|1|2}}英吋[[霰彈|12鉛徑霰彈]]。

據稱，該霰彈槍使用了稱為ArmorLokt的[[聚合物|聚合物]]表面處理，<ref name="Remington Overview"/>其設計目的是在經歷任何類型的天氣狀況以後仍然有效而且不會出現任何外部表面的[[鏽蝕|鏽蝕]]。<ref name="Heggenstaller American Rifleman">{{cite web |url=http://www.americanrifleman.org/Webcontent/pdf/2009-6/2009617111722-rem887nitro.pdf|author=Adam Heggenstaller|title= Up-Armored Pump: Remington's 887 Nitro Mag|accessdate=January 22, 2010 |work= |publisher=American Rifleman|date=}}</ref>這樣使得雷明登887具有“[[太空時代|太空時代]]”的外觀，而這也是該槍的一個更為重要的定義特徵。<ref name="Heggenstaller Incoming">{{cite web|url=http://www.shootingillustrated.com/forum/blog/heggenstaller/index.php?showentry=86|title=New Pump From Remington|author=Adam Heggenstaller|date=December 19, 2008|work=|publisher=Incoming|accessdate=January 10, 2010}}{{dead link|date=十一月 2017 |bot=InternetArchiveBot }}</ref>

== 設計和功能 ==
正如它的名字所暗示的，雷明登887“硝基馬格南”的[[膛室|膛室]]能夠裝填3{{frac|1|2}}英吋[[霰彈|12鉛徑超級馬格南霰彈]]藥筒。正是這種方面，它的競爭對手是同樣地專為發射3{{frac|1|2}}英吋超級馬格南霰彈藥筒而研製的[[莫斯伯格835終極馬格南泵動式霰彈槍|莫斯伯格835“終極馬格南”]]。<ref name="Firearm Blog">{{cite web |url=http://www.thefirearmblog.com/blog/2008/12/20/new-pump-shotgun-from-remington-model-887-nitromag/|title= New pump shotgun from Remington: Model 887 NitroMag|accessdate=January 22, 2010|work= |publisher=The Firearm Blog|date=December 20, 2008}}</ref>另一方面，雷明登887的外觀也經常地與[[伯奈利新星泵動式霰彈槍|伯奈利新星]]相比較。<ref name="Wakeman">{{cite web|url=http://www.randywakeman.com/Preview_Remington_2009_887NitroMagPump.htm|author=Randy Wakeman|title=Preview: Remington's 2009 887 Nitro Mag Pump Action|accessdate=January 23, 2010|work=|publisher=|date=|deadurl=yes|archiveurl=https://web.archive.org/web/20100131084054/http://www.randywakeman.com/Preview_Remington_2009_887NitroMagPump.htm|archivedate=一月 31, 2010}}</ref>而雷明登887的小冊子中出現莫斯伯格835和伯奈利新星的通過對比亦使這一點得以證實。<ref name="887 Brochure"/>

=== ArmorLokt表面處理 ===
雷明登887的最為顯著的特徵就是ArmorLokt表面處理。整枝887的[[機匣|機匣]]和槍管都塗覆了具有玻璃填充[[尼龙|尼龙]]的材料以保護該槍的內部[[钢|钢]]材。藉由這種方式，鋼材提供了該槍的強度，而高分子聚合物則保護其內部運作原理避免受到各種元素，包括惡劣天氣以下所產生的[[腐蚀|腐蚀]]所影響。許多的廠商都拿出了多種方法來協助保護槍械的金屬表面，但採用了與聚合物重疊注塑包覆成型的該槍卻是一個獨特的概念。<ref name="Heggenstaller American Rifleman">{{cite web |url=http://www.americanrifleman.org/Webcontent/pdf/2009-6/2009617111722-rem887nitro.pdf|author=Adam Heggenstaller|title= Up-Armored Pump: Remington's 887 Nitro Mag|accessdate=January 22, 2010 |work= |publisher=American Rifleman|date=}}</ref>

雷明登宣稱，ArmorLokt表面處理堅不可摧，甚至用以宣稱其為最耐用的霰彈槍，而且有若干測試可以協助支持這種說法。該公司的技術工程師將雷明登887進行含鹽腐蝕及浸沒的測試，並且檢查是否出現洩漏和在聚合物中分離，而且沒有被發現。<ref name ="887 Brochure"/>而在進行的第二項測試中，超過10,000發霰彈從一根雷明登887的槍管中發射，槍管的塗料沒有任何顯示出現分離的跡象。更小規模的測試亦已經被評論家們進行過了，這在一定程度上驗證了這些的結果。<ref name="Heggenstaller American Rifleman"/>

但該武器中亦有幾個關鍵零部件並未受到ArmorLokt表面處理。在多個網站中指出，其[[彈匣#管式彈倉|管式彈倉]]前端組件出現了直接來自原廠的[[生銹|生銹]]問題。

槍管、[[機匣|機匣]]以及大型聚合物製造的合成前護木表面的輪胎胎面型花紋圖案，具有防滑表面效果。這也給了槍械獨特的外觀，但也是經常被批評的原因。<ref name="Heggenstaller Incoming"/>

=== 雷明登的召回 ===
為了安全起見，雷明登正在對在2013年12月1日至2014年11月24日期間生產的所有M887霰彈槍進行自願性召回。雷明登指出這批被標記的槍械有著可能會將撞針鎖定到前端位置的缺陷。如果一發霰彈藥筒己經上膛而撞針在前端位置的情況下，有可能會導致霰彈槍在無意間因撞針撞擊底火而導致擊發。

雷明登將繼續生產M887霰彈槍，因為這個問題不是M887設計本身的固有問題，只是一些有缺陷的部件投入生產中的槍械裡。每枝經過檢查的霰彈槍都會在[[枪栓|枪栓]]上打上標記，表明該槍已經返廠、檢查、必要時進行修理並進行試射以確保其性能已達到100％。<ref>{{Cite web|url=http://www.guns.com/2014/12/01/remington-issues-recall-on-select-model-887-shotguns/|title=Remington issues recall on select Model 887 shotguns|last=Slowik|first=Max|date=2014-12-01|website=Guns.com|access-date=2016-07-06}}</ref>

== 與雷明登870比較 ==
設計的核心，具體而言，[[槍機|槍機]]、[[機匣|機匣]]以及[[槍管|槍管]]，都是以著名的[[雷明登870泵動式霰彈槍|雷明頓870]]為藍本。這些都構成了雷明登887的[[钢|钢]]製“核心”。<ref name="Remington Waterfowl Overview">{{cite web |url=http://www.remington.com/products/firearms/shotguns/model-887/model-887-nitro-mag-waterfowl.aspx |title=Model 887 Nitro Mag Waterfowl |accessdate=January 22, 2010 |work= |publisher=Remington Arms Company, Inc. |date=December 20, 2008 |deadurl=yes |archiveurl=https://web.archive.org/web/20100115071126/http://www.remington.com/products/firearms/shotguns/model-887/model-887-nitro-mag-waterfowl.aspx |archivedate=一月 15, 2010 }}</ref>然而，除了這些，887實際上與870相比還算是顯著不同的，而且並不是專門為了取代雷明登870而設計。<ref name="Heggenstaller American Rifleman"/>

雷明登887提供了多種從雷明登870改進而成、而通常都是在使用者友好性的名義以下改進的設計。例如滑動式槍機的釋放裝置，是一個位於[[扳機護環|扳機護圈]]上半部分表面的大型[[三角形|三角形]]按鈕，這種設計即使在帶著[[手套|手套]]或是雙手發麻以下都很容易使用。相比以下，雷明登870的滑動式槍機的釋放裝置位於扳機護圈的左側，而且只是一個小型的金屬按鈕。<ref name="Heggenstaller American Rifleman"/>

雷明登887也比雷明登870更容易分解下來和清理。與雷明登870不同的是，分解雷明登887並不需要任何工具（戰術型除外，它需要使用一柄[[內六角板手|內六角板手]]和[[螺丝起子|螺丝起子]]將槍管／[[彈匣#管式彈倉|管式彈倉]]夾拆卸下來），<ref name="Heggenstaller American Rifleman"/>而且分解雷明登887的速度比雷明登870更快，只需要兩分鐘就能夠將雷明登887[[不完全分解|不完全分解]]和重新組裝。

== 衍生型 ==
雷明登最初銷售的兩款887版本為'''887“硝基馬格南”'''（{{Lang-en|887 Nitro Mag}}）<ref name="Remington Overview"/>和'''887“硝基馬格南”水禽型'''（{{Lang-en|887 Nitro Mag Waterfowl}}）<ref name="Remington Waterfowl Overview"/>。其後雷明登亦推出了多款全新型號，其中包括'''887“硝基馬格南”戰術型'''（{{Lang-en|887 Nitro Mag Tactical}}）、'''887“硝基馬格南”骨頭收集者型'''（{{Lang-en|887 Nitro Mag Bone Collector}}），以及'''887​​“硝基馬格南”迷彩組合型'''（{{Lang-en|887 Nitro Mag Camo Combo}}）。

887“硝基馬格南”戰術型類似於基本型，但槍管改為469.9毫米（18.5英吋），一根[[彈匣#管式彈倉|管式彈倉]]延長彈筒，[[機匣|機匣]]頂部裝上一條[[皮卡汀尼導軌|皮卡汀尼配件安裝導軌]]，以及管式彈倉與槍管連接處兩側各裝上皮卡汀尼配件安裝導軌。

887“硝基馬格南”水禽型與基本版本非常相似，而最主要的區別是表面處理。水禽版本覆蓋了包括莫西奧克的分解無限型（{{Lang-en|Mossy Oak's Break-Up Infinity}}）以及真樹優勢MAX-4 HD迷彩型（{{Lang-en|Realtree Advantage Max-4 HD camo}}）兩種表面處理，這使得它正如其名的非常適合狩獵。<ref>[http://www.outdoorwriter.net/?p=357 OutdoorWriter.net－Remington Releases 887 Mossy Oak Infinity Camo Shotgun Combo]</ref><ref name="Heggenstaller American Rifleman"/>887水禽型也比887基本型略為沉重。<ref name="887 Brochure"/>
