'''雷蛇屬'''（[[學名|學名]]：''Bolyeria''）是[[蛇亞目|蛇亞目]][[島蚺科|島蚺科]]下的一個相信已經[[絕滅|絕滅]]的[[單型|單型]][[屬|屬]]，屬下只有曾聚居於[[毛里裘斯|毛里裘斯]]的'''雷蛇'''（又稱「'''圓島雷蛇'''」，''B. multocarinata''）一種蛇類，目前未有任何亞種被確認。<ref name="ITIS-G">{{ITIS|ID=202191|taxon=''Bolyeria''|year=2007|date=18 August}}</ref>紀錄中最後一次人類發現雷蛇蹤影的時候是1975年，地點是毛里裘斯的[[圓島|圓島]]，此後再無人發現過雷蛇出沒，因此相信牠們已經絕種。<ref name="ITIS-S">{{ITIS|ID=202192|taxon=''Bolyeria multocarinata''|year=2007|date=18 August}}</ref><ref name="ICUN-BM">[[世界自然保護聯盟瀕危物種紅色名錄|世界自然保護聯盟瀕危物種紅色名錄]]：[http://www.iucnredlist.org/search/details.php/2864/all 雷蛇]</ref>

== 特徵 ==
雷蛇在相信已滅絕之前，只分布於區區1.5[[平方公里|平方公里]]的細小範圍裡，牠們棲息於喬木樹林或熱帶草原之中。雷蛇體長約有一[[米_(單位)|米]]，身體以[[棕色|棕色]]為主。牠們的鼻端較為尖銳，體型仿似圓筒。在1949年的時候，雷蛇的數量已經因為過少而備受關注，自1975年後更加從此銷聲匿跡。相信牠們滅絕的原因，是因為其棲息地的[[植被|植被]]被大量[[山羊|山羊]]及[[兔子|兔子]]嚼食，引致水土流失，令雷蛇生態大受影響，並相繼死亡。<ref name="Day81">Day, D：《The Doomsday Book of Animals》，[[倫敦|倫敦]]：Ebury Press，1981年。ISBN 0-670-27987-0</ref>

== 地理分布 ==
從前，雷蛇主要分布於毛里裘斯，尤多出沒於圓島一帶。<ref name="Day81"/><ref name="McD99"/>記載中其標本產地是「積克遜港」，應為手民之誤。<ref name="McD99"/>

== 保育狀況 ==
目前雷蛇已被[[世界自然保護聯盟|世界自然保護聯盟]]列入「絕滅」等級（EX）<ref name="ICUN-BM"/>，這代表著雷蛇已經被確認為完全絕滅，是無一倖存的物種。<ref name="ICUN-23">[http://www.iucnredlist.org/info/categories_criteria1994#categories 世界自然保護聯盟物種分類定義]</ref>
