'''雷炎均'''，{{BD|1914年||1999年|catIdx=Lei雷}}[[美國華人|美國華僑]]，[[中華民國空軍|中華民國空軍]][[二級上將|二級上將]]，曾任[[國防部空軍司令部|空軍]]副[[中華民國空軍司令|總司令]]、副[[中華民國參謀總長|參謀總長]]、[[中華航空公司|中華航空公司]]總經理與董事長。

== 出身 ==
出生於美國[[華盛頓州|華盛頓州]][[西雅圖|西雅圖]]。祖籍[[廣東|廣東]][[台山|台山]]。1931年間，在美國[[奧勒岡州|奧勒岡州]][[波特蘭|波特蘭]][[美洲華僑航空學校|美洲華僑航空學校]]隨第一次世界大戰王牌飛行員加拿大籍布朗（Roy Brown) 學習飛行，志在報效祖國。與美籍華裔子弟[[蘇英祥|蘇英祥]]、[[李月英|李月英]]、[[陳瑞鈿|陳瑞鈿]]、[[黃泮揚|黃泮揚]]、[[劉龍光|劉龍光]]、[[林覺天|林覺天]]、[[雷國來|雷國來]]、[[楊仲安|楊仲安]]、[[鄺廷|鄺廷]]、[[黃桂燕|黃桂燕]]等皆同期畢業自該校。返國後就讀中央航空學校第3期畢業.

== 戰績 ==
後返國參戰，加入第五大隊，第二十八分隊， 戰功彪炳，成為[[抗日戰爭|抗日]]空戰英雄。

== 軍事生涯 ==
1942年前往[[美國|美國]][[指揮參謀學院|指揮參謀學院]]深造。1945年任[[喀拉蚩|喀拉蚩]]盟軍教官，抗戰勝利後任中華民國駐日軍事代表團員，在台歷任[[美軍協防台灣司令部|美軍協防司令部]]主任連絡官，中華民國空軍情報署長，作戰署長，作戰司令，空軍副總司令。1967年任副參謀總長，1970年晉升為空軍二級上將。

== 民航生涯 ==
1974年退役，轉任[[中華航空公司|中華航空公司]]總經理，董事長。1978年退休。

== 婚姻 ==
元配[[李月英|李月英]]為華僑航空學校同學，成為美國[[女子航空勤務飛行隊|女子航空勤務飛行隊]]的一員，但意外喪生。
