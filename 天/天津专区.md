'''天津专区'''是1949年8月至1958年12月、1961年6月至1970年1月间断存在于[[河北省|河北省]]的[[专区|专区]]，因辖区环绕[[直辖市|直辖市]][[天津市|天津市]]且属于天津市的经济腹地而得名，1970年改'''天津地区'''，1973年8月部分辖区并入[[天津市|天津市]]，余下辖区更名[[廊坊|廊坊地区]]。天津专区存在期间，行政机构“天津专区[[行政专员公署|行政专员公署]]”曾先后驻[[杨柳青镇|杨柳青镇]]、[[沧州|沧州]]、[[天津市|天津市]]等地。<ref name="autogenerated0">{{cite book |title = 《河北省地名志·廊坊地区分册》 |publisher=河北省地名办公室 |author=河北省地名办公室 编著}}</ref>

== 辖境和机构沿革 ==
[[File:石家大院.jpg|thumb]]]]

1949年8月，<u>河北省天津区行政督察专员公署</u>在[[永清县|永清县]]成立，因环绕直辖市天津市周边且属于天津的经济腹地而得名。成立之初，天津专区管辖原冀中八分区的[[静海县|静海县]]、[[文安县|文安县]]、[[大城县|大城县]]、[[天津县|天津县]]、[[杨柳青镇|杨柳青镇]]和十分区的[[霸县|霸县]]、[[永清县|永清县]]、[[安次县|安次县]]、[[武清县|武清县]]、[[胜芳镇|胜芳镇]]及十五分区的[[宝坻县|宝坻县]]、[[宁河县|宁河县]]、汉沽镇。专区成立之后，区机关由永清县搬迁并常驻[[杨柳青镇|杨柳青镇]][[石家大院|石家大院]]。1950年11月，河北省天津区行政督察专员公署改名为<u>河北省人民政府天津区专员公署</u>。<ref name="autogenerated1">{{cite book |title = 《廊坊市土地志》 |publisher=中国大地出版社 |author=廊坊市土地志编纂委员会 施泰 |year=2001年 |ISBN = 7-80097-411-1}}</ref>1952年4月，由于天津专区管辖的天津县隔断了同属于天津市管辖的[[塘沽|塘沽]]与[[天津市区|天津市区]]，故将天津县划归天津市领导，使得天津市区与塘沽区连为一体。同年，将原[[沧县专区|沧县专区]]下辖的[[青县|青县]]、[[任丘县|任丘县]]划入天津专区。至此，天津专区共辖12个县市、3个镇和223个乡。<ref name="autogenerated0" />

1958年12月，天津专区撤销，武清县、[[静海县|静海县]]、[[黄骅县|黄骅县]]、[[沧县|沧县]]、[[盐山县|盐山县]]、[[宁河县|宁河县]]、[[吴桥县|吴桥县]]、[[交河县_(金朝)|交河县]]、[[献县|献县]]、[[河间县|河间县]]、[[任丘县|任丘县]]、霸县交由天津市领导。1961年6月21日，经[[国务院|国务院]]批准恢复天津专区建制。天津专区隶属[[河北省|河北省]]和[[天津市|天津市]]双重领导，以天津市领导为主。<ref name="autogenerated1" />专署驻[[天津市|天津市]]，将静海县、[[霸县|霸县]]、武清县、宝坻县、蓟县划归天津专区，同年，恢复文安县（辖大城县）、永清县、安次县。1962年8月，原唐山专区的[[宁河县|宁河县]]划归天津专区，恢复大城县、固安县、香河县、大厂回族自治县，并将将划归青县部分收回区域重新划归大城县。此时，天津专区辖固安县、蓟县、宝县、永清县、文安县、大城县、静海县、安次县、三河县、香河县、[[大厂回族自治县|大厂回族自治县]]、宁河县、胜芳镇、霸县14个县、镇，296个人民公社、6269个生产大队。1963年，废胜芳镇，改建胜芳镇人民公社。1964年三河县建立城关镇。1965年，安次县建立[[廊坊|廊坊镇]]。1967年11月，天津专区改为'''天津地区'''，隶属河北省[[革命委员会|革命委员会]]，地区机关驻天津市。1967年，<u>天津专区革命委员会</u>成立，辖区无变动。1969年3月，天津专区机关迁入廊坊镇。<ref name="autogenerated0" />

1973年8月，国务院决定，将宁河县、静海县、武清县、蓟县、宝坻县5个县划归天津市领导。1974年1月，经国务院批准，“天津地区”更名为“'''[[廊坊地区|廊坊地区]]'''”。1978年8月，撤销天津地区革命委员会，建立<u>廊坊地区行政公署</u>。<ref name="autogenerated0" />河北省围绕在直辖市天津市周边的“天津专区”或“天津地区”的区划概念随之成为历史。此后，这一地区沿用廊坊的名称至今。时至今日，天津专区这一区划和称谓已经鲜为人知，甚至在其基础上成立的河北省[[廊坊市|廊坊市]]的政府官方网站的历史沿革介绍中亦没有介绍，其存在仅在少数地方志中有记载。<ref>{{cite web|url=http://www1.lf.gov.cn/WebSite/Item/17914.aspx|title=历史沿革|publisher=廊坊市政府门户网站|accessdate=May 21, 2011}}{{Dead link|date=2018年7月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>

== 著名事件 ==

1951年11月，[[刘青山张子善案|刘青山张子善案]]案发。1952年2月10日，[[中华人民共和国政府|中华人民共和国政府]]以[[贪污|贪污罪]]公开处决中共石家庄市委副书记、前天津地委书记[[刘青山|刘青山]]和地委副书记、天津专区专员[[张子善|张子善]]，这是[[中华人民共和国|中华人民共和国]]成立以来的第一起腐败大案，在当时轰动全国。<ref>{{cite web | url=http://cpc.people.com.cn/GB/64156/64157/4512322.html | title=刘青山、张子善案件 | publisher=中国共产党新闻网 | accessdate=May 21, 2011}}</ref>
