{{日本作品|天使的3P！|天使の3P!}}是[[蒼山探|蒼山探]]著作，[[Tinkle|Tinkle]]繪製插畫的日本[[輕小說|輕小說]]作品。

== 概要 ==
《天使的3P！》為[[蒼山探|蒼山探]]的第2部作品，前作為獲得第15回[[電擊小說大賞|電擊小說大賞]]銀賞的《[[蘿球社！|蘿球社！]]》，也是繼該作後蒼山探與[[Tinkle|Tinkle]]的再次合作。此作有3首由作者以[[Vocaloid|Vocaloid]]所作的作中曲<ref>{{cite web|url=http://www.nicovideo.jp/mylist/46234945|script-title=ja:天使の3P!作中曲|publisher=[[Niconico|Niconico]]|accessdate=2016-10-02}}</ref>。並在「電擊文庫 秋之祭典2015」中決定進行團體活動並改編成[[廣播劇CD|廣播劇CD]]<ref>{{Cite web |url=http://dengekionline.com/elem/000/001/133/1133021/|script-title=ja:『天使の3P！』ドラマCD化も発表！ 「電撃文庫 秋の祭典2015」|work=電擊萌王.com|date=2015-10-04|accessdate=2016-10-03|language=ja}}</ref>。標題的3P意指「Three Pieces」<ref name="ACGer">{{cite web|url=http://hkacger.com/archives/30282|title=【"糟糕"新番】輕小說「天使的 3P」TV動畫化：媽媽你聽我解釋，事情不是你所想的那樣啊！|publisher=ACGer|date=2016-10-02|accessdate=2016-10-03}}</ref>。

2016年10月2日在「電擊文庫 秋之祭典2016」中決定改編成電視動畫<ref name="anime">{{Cite web |url=http://gs.dengeki.com/news/84688/|script-title=ja:『天使の３P!』が2017年TVアニメ化決定！　さらにキャスト陣による「3P!ユニット」プロジェクトが進行中！|work=電擊G's magazine.com|date=2016-10-02|accessdate=2016-10-03|language=ja}}</ref>，於2017年7月10日播出。

== 故事簡介 ==
拒絕上學的高中生貫井響喜歡用歌唱軟體製作歌曲並上傳到網路上。某一天響收到了一封想和他見面的郵件，而當響前去赴約時，沒想到和他見面的卻是五島潤、紅葉谷希美、金城空三名小學五年級的女生。

== 登場角色 ==
=== 主角 ===
; {{nihongo|貫井響|貫井 響（ぬくい きょう）|聲：[[井上雄貴|井上雄貴]]<ref name="CHARACTER">{{cite web|url=http://www.tenshi-no-3p.com/chara/|title=CHARACTER|work=TV動畫「天使的3P！」官方網站|language=ja}}</ref>}}
: 就讀新井田西高校普通科1年A班的男學生。由於小學的國語課發表的詩被大家嘲笑，因此從中學一年級開始便不去上學，於閒暇的時候便開始學吉他、並以「響P」這個名字在視頻網站上傳自己創作的曲子。

=== 小翅膀 ===
==== Lien de famille ====
; {{nihongo|五島潤|五島 潤（ごとう じゅん）|聲：[[大野柚布子|大野柚布子]]<ref name="CHARACTER"/>}}
: 就讀城見台小學校5年2班的女孩子。負責吉他，也是樂團的主唱。個性羞怯且有點愛哭。聽力很好，能馬上判別響創作的樂曲。
; {{nihongo|紅葉谷希美|紅葉谷 希美（もみじだに のぞみ）|聲：[[遠藤祐里香|遠藤祐里香]]<ref name="CHARACTER"/>}}
: 就讀城見台小學校5年2班的女孩子。負責貝斯。個性不服輸，同時有女孩子纖細的一面。由於是英國混血兒，跟同年紀的女孩子相比身材很好。
; {{nihongo|金城空|金城 そら（かねしろ そら）|聲：[[古賀葵|古賀葵]]<ref name="CHARACTER"/>}}
: 就讀城見台小學校5年2班的女孩子。負責打鼓。常常一副想睡覺的表情，但在三人組中卻很有行動力。

==== 其他住人 ====
; {{nihongo|鳥海櫻花|鳥海 桜花（とりうみ さくら）|聲：[[伊藤加奈惠|伊藤加奈惠]]<ref name="CHARACTER"/>}}
: 就讀新井田西高校普通科1年A班的女學生，響的同學，與響在中學時期就有見過面。喜歡響創作的詩，經常幫不上學的響送講義。在麵包店「Sanois」打工，對響有好感，興趣是釣魚。
; {{nihongo|佐渡正義|佐渡 正義（さわたり まさよし）|聲：[[大川透|大川透]]<ref name="CHARACTER"/>}}
: 兒童養護設施「小翅膀」的負責人，個性不拘小節，被設施的孩子們稱為神父。喜歡搖滾樂並收集各種樂器，也是讓潤她們組成樂團的契機，興趣是釣魚。

=== Dragon≒Nuts ===
; {{nihongo|貫井胡桃|貫井 くるみ（ぬくい くるみ）|聲：[[日高里菜|日高里菜]]<ref name="CHARACTER"/>}}
: 就讀城見台小學校5年2班，響的妹妹，在學校擔任班長，並幫忙家裡作大部分的家事。平時雖對響態度差，但其實很關心哥哥。被小梅挑撥後加入Dragon≒Nuts。
; {{nihongo|尾城小梅|尾城 小梅（おぎ こうめ）|聲：[[花澤香菜|花澤香菜]]<ref name="CHARACTER"/><ref name="dengeki20170605"/>}}
: 就讀城見台小學校5年2班的女孩子。以「霧夢」一名進行活動，很有人氣的網路繪師，因為喜歡響的歌曲而認識響並且幫他作[[插畫|畫]]。
: 本身是雙龍島上貴龍神社的「龍之女巫」，從出生開始就要被當作「御靈代」侍奉。後來在響等人的幫忙下解放了御靈代的使命，和柚葉一起搬到響居住的地方生活。
; {{nihongo|相江柚葉|相ヶ江 柚葉（あいがえ ゆずは）|聲：[[井口裕香|井口裕香]]<ref name="CHARACTER"/><ref name="dengeki20170605"/>}}
: 就讀城見台小學校5年2班的女孩子。有著如同國中生的身體外觀，是代代侍奉雙龍島「龍之女巫」的家族。在動畫中擔任主演唱。
