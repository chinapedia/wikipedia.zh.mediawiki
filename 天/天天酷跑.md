《'''天天酷跑'''》是[[腾讯|腾讯]]手机游戏平台于2013年9月16日推出的一款支持以[[微信|微信]]和[[QQ|QQ]]账号登录并可以与好友分享游戏积分和排名的[[跑酷|跑酷]]类[[电子角色扮演游戏|电子角色扮演]]游戏。由腾讯旗下[[天美艺游|天美艺游]]研发。该游戏是[[中华人民共和国|中国]]第一款月收入超过1亿人民币的手机游戏<ref name="首亿">[http://mall.cnki.net/magazine/Article/DNMI201311083.htm 《《天天酷跑》》已成为国内首款月收入过亿的手游]《[[电脑迷|电脑迷]]》2013年第11期 [[中国知网|中国知网]]</ref>。

== 规则 ==

《天天酷跑》每次游戏需要颗“爱心”，爱心使用后需要一段长时间恢复，好友之间不可以赠送爱心。钻石可以通过美元或[[Q币|Q币]]支付获得，也可以在每日任务或游戏中获得少量金山。

此外，
《天天酷跑》
可以向[[微信|微信]]、[[QQ|QQ]]好友发出求助、
挑战、
炫耀
等消息，
并将成绩分享到[[朋友圈|朋友圈]]或[[QQ空间|QQ空间]]。

== 版本 ==

《天天酷跑》于2013年9月13日开始内测<ref>[http://pao.qq.com/webplat/info/news_version3/6019/6021/6024/m4397/201309/225669.shtml 《天天酷跑》不删档限量测试]2013-09-12 《天天酷跑》官网新闻公告</ref>，9月16日公测<ref name="首亿" />。2013年11月12日新版本新增“极速世界”并陆续新增人物、坐骑、宠物、地图等游戏元素若干。当前版本是于2014年1月17日发布的PVP版本（v1.0.8.0），新增“PVP模式”、积分商店、穿越奖励等游戏元素，并增加了宠物、坐骑等元素的种类；其特色是能够与全球玩家或好友进行[[PK|PK]]<ref>[http://pao.qq.com/webplat/info/news_version3/6019/6021/6025/m4397/201401/243976.shtml 《天天酷跑》PVP新版更新] 天天酷跑官网新闻公告 2014-01-17</ref>。

== 收入 ==
《天天酷跑》在公测前半个月（2013年9月16日至9月30日）收入超过7000万人民币，日峰值收入约700万。于10月中旬收入过亿，成为[[中华人民共和国|中国]]第一款月收入超过1亿[[人民币|人民币]]的手机游戏<ref name="首亿" />。
