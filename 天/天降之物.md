《'''天降之物'''》（{{lang-ja|そらのおとしもの}}，直譯為「從天而降之物」，或者「從天而降之人」）是[[水無月すう|水無月すう]]的漫畫作品，並改編成輕小說、電視動畫、劇場版和遊戲。

== 概要 ==
*2007年開始在[[角川書店|角川書店]]旗下的《[[月刊少年Ace|月刊少年Ace]]》連載。<ref>{{cite news|title=Sora no Otoshimono/Heaven's Lost Property Manga to End in January|url=http://www.animenewsnetwork.com/news/2013-12-25/sora-no-otoshimono/heaven-lost-property-manga-to-end-in-january|publisher=動畫新聞網|date=2013-12-25|accessdate=2016-01-26}}</ref>在[[单行本|單行本]]第4集發售時發佈動畫化消息，於2009年10月4日播放<ref>{{cite news|title=10月4日放送開始｢そらのおとしもの｣ 早くもDVD化決定|url=http://animeanime.jp/article/2009/09/20/5352.html|publisher=アニメ！アニメ！|accessdate=2016-01-26}}</ref>。

*2010年3月20日，在{{lang|ja|『「そらのおとしもの」大感謝祭〜「そらのおとしもの」青春ヒットパラダイス〜』}}活動中發表製作第二季動畫的消息；並於6月25日正式公布標題為「{{lang|ja|そらのおとしものf（フォルテ）}}」，目前已播映完畢。

*2010年11月官方釋出有關推出劇場版動畫作品的消息<ref>[http://gnn.gamer.com.tw/0/48340.html 空之女王降臨大銀幕《天降之物》將推出劇場版動畫作品報導]</ref>并在2011年6月25日於日本上映<ref>{{cite news|title=『劇場版そらのおとしもの 時計じかけの哀女神（エンジェロイド）』角川シネマ新宿・シネマサンシャイン池袋他にて6月25日より全国ロードショー|url=http://news.livedoor.com/article/detail/5424310/|publisher=[[ライブドアニュース|ライブドアニュース]]|date=2011-03-18|accessdate=2016-01-26|deadurl=yes|archiveurl=https://web.archive.org/web/20160202223128/http://news.livedoor.com/article/detail/5424310/|archivedate=2016-02-02}}</ref>。2012年1月6日在台灣上映<ref>{{cite news|title=人氣聲優日笠陽子 《天降之物》獻聲演唱 |url=http://ilooker.com.tw/news_info.php?ctid=1&id=5071|publisher=iLOOKER電影網|date=2011-12-30|accessdate=2016-01-26}}</ref>。

*2012年3月1日官網宣布决定制作第三季。

*2013年10月25日官網宣布更改製作第三季的決定為劇場版<ref>{{cite news|title=人氣動畫續集《天降之物 第三期》確定為『2014年完全新作』劇場版、製作人親自下海編劇！|url=http://www.animen.com.tw/NewsArea/NewsItemDetail?NewsId=2246&categoryId=600&tagName=%E5%A4%A9%E9%99%8D%E4%B9%8B%E7%89%A9&realCategoryId=1&subCategoryId=3|publisher=Animen|date=2013-10-25|accessdate=2016-01-26}}</ref>，作為完全新作的該片將於2014年4月26號公開上映<ref name="ANN-2014-film">{{cite web|url=http://www.animenewsnetwork.com/news/2013-10-24/sora-no-otoshimono/heaven-lost-property-new-anime-to-play-in-theaters-in-2014|title=Sora no Otoshimono/Heaven's Lost Property's New Anime to Play in Theaters in 2014|publisher=[[動畫新聞網|動畫新聞網]]|date=2013年10月24日報導|accessdate=2013年11月4日|language=en}}</ref>，而此次劇場版的製作陣容將延續動畫第1、2期時的原班人馬。

*2014年1月25日在《月刊少年Ace》中刊載作品完結篇，為7年的連載畫上句點。<ref>{{cite news|title=＜そらのおとしもの＞アニメ化された人気マンガが完結へ |url=http://news.nicovideo.jp/watch/nw892664 |publisher=[[niconico|niconico]] |date=2013-12-26 |accessdate=2016-01-26 |deadurl=bot: unknown |archiveurl=https://web.archive.org/web/20131229103423/http://news.nicovideo.jp/watch/nw892664 |archivedate=2013-12-29 }}</ref>

== 劇情簡介 ==
{{more plot}}
居住在[[日本|日本]][[九州_(日本)|九州]][[福岡縣|福岡縣]]空美町裡的男主角——櫻井智樹，以和平至上為[[座右銘|座右銘]]，自小睡覺時偶爾夢見同一個夢：一個長了翅膀的女性向自己求救。然後有一天晚上遇到從天而降及自稱是「娱乐型人造天使」的不明生物——伊卡洛斯，伊卡洛斯把降落後見到的第一位人類，也就是智樹稱呼為「主人」，自此兩人同居在一起，開始智樹的脫軌生活……。

== 登場人物 ==
=== 主要人物 ===
;{{nihongo|櫻井智樹|桜井 智樹［さくらい ともき］||[[聲優|聲]]：[[保志總一朗|保志總一朗]]}}
:本作主人公，座右銘為「和平第一」的普通國中生（2年C班的學生，和伊卡洛斯、寧芙及楚原同班），身高150cm，體重49kg，直到某日天空上掉下一隻不明生物後日子出現轉變。他對天使的那種有意無意的關心，卻感動原本無感情的天使。現在正與伊卡洛斯、寧芙和阿斯特蕾亞生活在一起。外貌普通，偶爾有點帥氣，但實際上大多時間是個白痴，喜歡到學校的女更衣室，澡堂等女性出沒的地方偷窺，對學校女同學的事情（如胸圍、內衣花色等）非常瞭解。幼時因看了楚原的小狗圖案底褲、導致其可怕手刀覺醒而遭痛扁的緣故，所以討厭狗。被學生會會長五月田根美香子和守形英四郎設計陷害而進入新大陸發現社。
:遇見伊卡洛斯前經常作夢，夢中出現長著翅膀面容不清的女子（代達羅斯）向其求救，夢醒後會流淚，但是在伊卡洛斯出現後夢境開始出現轉變。其夢境後來被發現與西那普斯相連，守形多次利用Dive Game進入調查。
:不論在動畫及漫畫中，智樹經常裸體出鏡，即使被女孩子看見也不會介意。但奇怪的是至今沒有對伊卡洛斯做出色情行為<ref>動畫第一季第一話除外</ref>。推測是因為喜歡伊卡洛斯的緣故。
:據推測是遭美香子以去電擊胯下股體<ref>漫畫第50話</ref>，令胯下有發射能量波的能力，威力強大得甚至可擊穿伊卡洛斯的「Aegis」防禦圈<ref>漫畫第53話</ref>。但是似乎必須要本人累積一定的損傷或者一定程度的「覺悟」後，才能夠使用這個能力。
:在伊卡洛斯提出去沒有人的地方時<ref>漫畫第54話</ref>，想到伊卡洛斯的付出有感而發，隨後注意到身旁有兩個伊卡洛斯（一個受傷倒地，一個身邊駕著）時，還沒搞清楚狀況就被駕著自己的伊卡洛斯·梅蘭打傷。
:漫畫68話與楚原穿越時空回到童年時代，重新體驗自己幼時的時光，還和幼時的自己交流一番，之後才想起那天有奇怪的哥哥出現過。意外的發現一件和自己記憶中不符合的事情，儘管當下頭痛，卻沒有馬上想起來。
:雖然嘴巴常說追求和平第一的平凡生活，經常把麻煩禍因牽拖身邊的人造天使們。但事實上，除人造天使外，還有行事瘋狂的守形學長和腹黑且倫理崩壞的五月田根學姐必會替他製造更多問題，還有幼年時親自打造的奪命手刀女楚原，再加上自己也常有不正常的行為，這些條件就足以讓他過著非凡的生活。
:漫畫70話時遇見向他索吻的卡俄斯，智樹說「長大點再說吧」後，卡俄斯啟動潘朵拉進化為大人，讓智樹明白卡俄斯人造天使的身分。
:漫畫71話中得知卡俄斯「吃掉」日和的原因是因為他於某天說出：「人造天使什麼的別給我回來！」而想自己頭腦好一點所以「吃掉」日和後自己刺向卡俄斯的黑色鋼翼，經伊卡洛斯診症後得知傷勢極為嚴重，但最終於72話受到奧蕾迦娜的治療下甦醒。
:漫畫77話中因伊卡洛斯的犧牲而來到西納普斯，並使用伊卡洛斯的可變雙翼系統核心（智樹稱為人造天使的心靈）打倒米諾斯（但沒有殺死他），其後到達石板所在處發現義經的屍體，後利用守形的滑翔傘去到西納普斯人們冬眠的巨蛋建築物時，發現代達羅斯並得知楚原就是代達羅斯，之後和代達羅斯一起再次去到石板所在處，打算停止守形的願望，然而石板僅剩的最後一行被守形的願望用掉了，但守形和美香子留下已經完成的破解程式，使得石板上守形的願望被消除，最終智樹許了「把大家…回到原來」的願望令地上的所有人（包括人造天使）復活，並回到原來的世界。
:擁有技能：
:*胯下能量炮<ref>無正確譯名</ref>：從胯下發射超弩級的能量波，威力大到可驅散雲霧或擊穿伊卡洛斯的Aegis（請見漫畫第53話）。
:;{{nihongo|櫻井智子|桜井 智子（原作）/桜井 トモ子（動畫）［さくらい ともこ］|[[聲優|聲]]：[[藤田咲|藤田咲]]}}

: 智樹的爸爸，長的跟智樹很像只差有戴眼鏡，但個性與兒子相差甚大。本身是入贅婚所以婚後改姓櫻井。會用手刀制裁不正經的智代，雖然有時會被智樹的媽媽抓去埋起來，但不知為何每次最後一定出現並制裁智代。
; 不知名的母女
: 漫畫50和51話的最後出現，在與智樹第一次見面的時候<ref>漫畫50話</ref>，裸體的智樹以詭異的光束砲將天上的雨雲驅散後，獲得了母女倆極大的好感。51話的最後，母女倆透過天文用望遠鏡發現被伊卡洛斯轟到月球上搗糕的智樹。另外，也許是因為第一次見面的時候，智樹表現太過於帥氣，所以那位年輕的媽媽內心似乎對智樹有著好感以上的遐想。
; 原射靶攤的阿伯
: 只在漫畫中出現，智樹每年都會光顧的射擊屋，但獎品太寒酸被智樹嫌。後面改開章魚燒屋，也被智樹嫌太難吃。之後似乎在經營壽司店<ref>漫畫35話</ref>。
: 後面改開炒麵攤，再炒麵上放上蛋糕，被智樹說很難吃，又被哈比姐批評難吃。
: 美香子舉辦的活動似乎也都會參加。

== 用語 ==
; 西納普斯（Synapse）
: 在天空漂浮經常移動的領域，以守形的說法為「新大陸」，通常都會在世界各地飄浮，目前因不明原因停留在智樹所住的空美鎮，裡面居住的角色都長有[[翅膀|翅膀]]，但都不能做夢。高度文明及科技使他們一切欲望都能得到滿足，人類及人造天使都是他們創造出來以自娛。但是長期物質豐盛生活反而使西納普斯人失去生存希望，自殺自殘者不計其數，使西納普斯人口由數億銳減至只有約三千人，多番試圖以「規則」扭轉狀況但沒有作用，大部分的居民眼見自己創造出來玩弄的人類，反而得到一直渴求的幸福，因此選擇陷入沉睡，在夢境中化作人類生活。主要以卡片作為傳送裝置，可以傳送各種物件；一張新式的卡能使用多種能力，一張舊式的卡則對應一種能力。
; 人造天使（Angeloid）
: 西納普斯製品，頸部裝有可無限延長鎖鍊的不明生物，具有各種功能。伊卡洛斯、妮姆芙、阿斯特蕾亞為人造天使的第一世代機種，卡俄斯和賽蓮則為第二世代。她們使命是要服從主人命令，所以她們都不能理解「自由」的意義。
: 人造天使的翅膀碰到水後會變重，所以都不會游泳（賽蓮是注重於水戰，克服無法游泳的例外）。
: 另因人造天使都不會睡覺，故不會做夢、也不能進入Dive Game（但卡俄斯有入侵夢境的能力）。代達羅斯在劇場版初段中解釋，這設定是希望人造天使們能藉沉睡以外的方式找到幸福。
: 不同的人造天使的位於耳朵位置部件形狀也不一樣，如日和是由左右兩側三片部件組成。
; 規則（Rule）
: 西納普斯中樞所建造的大型石塔，能夠實現任何人的願望<ref>但人造天使不能使用</ref>，曾多次被西納普斯居民寫下重塑世界的願望，現時已被寫滿而機能停止<ref>原本最後一個願望是守形寫的，但之後他以破解程式消除該段文字，使智樹能夠寫下恢復世界的願望</ref>。
: 漫畫72話前被智樹他們稱為石板，後於72話得知其名為「規則」。
; 蠐螬
: 金龜子的幼蟲，米諾斯和其人造天使形容地上的人的名稱。
; [[宙斯|宙斯]]（Zeus）
: 天上的主人[[米諾斯|米諾斯]]所製造，西納普斯最強的防空系統。名字取自[[希臘神話|希臘神話]]中的眾神之王。
; [[潘朵拉|潘朵拉]]（Pandora）
: 代達羅斯為了不讓人造天使受過多的拘束，在第一代人造天使（但第二代的卡俄斯也有）身上安裝讓其（在感情激盪時）能使用的程式。開啟此程式後，人造天使的能力會得到大幅的提升，翅膀形狀也會改變。名字取自[[希臘神話|希臘神話]]中的第一個女性。

=== 海外播出 ===
《天降之物》、《天降之物f》由日方授權美國[[Crunchyroll|Crunchyroll]]網站進行[[海外聯播動畫|海外聯播]]。{{r|en}}

== 剧场版 ==
=== 第1作 ===
{{main|天降之物 計時的悲傷女神}}
《[[天降之物_計時的悲伤女神|天降之物 計時的悲伤女神]]》，于2010年11月10日在《[[Newtype_(雜誌)|月刊Newtype]]》（{{lang|ja|月刊ニュータイプ}}）2010年12月号宣布制作剧场版，2011年6月25日公映<ref>[http://www.kadokawa-pictures.co.jp/official/sorano_otoshimono_movie/theater.shtml 上映劇場一覽]</ref>（於2012年1月6日（五）在台上映），故事内容为风音日和的故事，部分剧情和漫画中的风音日和的故事不同。

=== 第2作 ===
{{main|天降之物Final 永遠的我的鳥籠}}
《[[天降之物Final_永遠的我的鳥籠|天降之物Final 我永遠的鳥籠]]》（又譯作：天降之物 Final 永遠的我的鳥籠），是「天降之物」系列第二次改編為劇場版電影，同時也設定為所有動畫系列的完結篇，故事將圍繞在伊卡洛斯的「戰略用人造天使」身份。於2014年4月26日上映。制作公司更改為[[Production_IMS|Production IMS]]。
