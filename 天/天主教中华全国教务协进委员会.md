'''天主教中华全国教务协进委员会'''（簡稱'''天主教教務協進會'''，又稱為'''天主教中央局'''；英文譯名：{{lang|en|'''Catholic Central Bureau'''}}）是20世纪中叶[[中国天主教|中国天主教]]的全国教务领导和协调机构，设于[[中華民國|中華民國]][[上海市_(中華民國)|上海市]][[岳阳路|岳阳路]]197号。

== 歷史 ==
1947年12月3日，[[聖座|聖座]]第一任驻华公使[[黎培理|黎培理]]總主教指示创办天主教中华全国教务协进委员会。下设文化事业组、法律组、教友传教组（领导各地的[[圣母军|圣母军]]活动）、劳工组、修院教育组、学校教育组、宗座善会组、联络组、留学生服务组、农业组（设于南京）。在上海、北京、天津设立3个分会。委员十余人，来自于南北各省的不同传教区，以及不同的修会背景，国籍方面大致中外各占一半：
*[[華理柱|華理柱]]（[[天主教江门教区|天主教江门教区]]主教、美国[[玛利诺外方传教会|玛利诺外方传教会]]会士），任秘书长；
*[[趙玉明|趙玉明]]（[[天主教林东监牧区|天主教林东监牧区]]主教、生于法国的[[加拿大|加拿大]][[魁北克外方傳教會|魁北克外方傳教會]]会士），任副秘书长、修院教育组主任；
*[[陳哲敏|陳哲敏]]（教廷驻华公使馆的中文秘書）；
*[[高乐康|高乐康]]（François-Xavier Legrand，比利时籍），任文化事业组主任，主编该会机关报英文《中国传教士》（China Missionary）杂志；
*梅雨丝（Meeus Charles，比利时籍），任财务组组长；
*[[莫克勤|莫克勤]]（爱尔兰圣高隆庞会传教士），来自[[天主教汉阳教区|天主教汉阳教区]]，任圣母军组组长；
*[[沈士贤|沈士贤]]（中国籍），来自汉阳教区，宗座善会组主任，负责发展圣母军；
*田望霖（Allain De Terwangne，比利时籍），任法律组主任；
*[[董世祉|董世祉]]（中国籍），任协进会教理讲座组组长<ref>阮仁泽、高振农等主编：《上海宗教史》，第750页，上海人民出版社，1989年</ref>；
*张介眉（中国籍）；
*高思谦（[[天主教汾阳教区|天主教汾阳教区]]副主教），学校教育组、劳工组主任；
*毛振翔（中国籍），留学生服务组主任；
*明德（F. A. Mc Vaire又译麦槐尔，美国籍），中国天主教福利会主任

[[陳哲敏|陳哲敏]]、[[沈士賢|沈士賢]]、[[董世祉|董世祉]]为1950年代中国天主教“四大才子”之列（连同[[张伯达|张伯达]]），陳哲敏为羅馬傳信大學教授、哲學博士，沈士賢拥有欧洲大学的神学、哲学及政治三个博士头衔<ref>胡美玉：《乐在苦中》第三章</ref>。陈哲敏、沈士贤均在[[震旦女子文理學院|震旦女子文理學院]]和震旦女中兼课，借机接触青年学生，鼓励他们忠于天主教信仰，激发宗教热情<ref>述良：四川[[萬縣教區|萬縣教區]]的兩位現代偉人，教友生活周刊2001年</ref>。在震旦女子文理院成立了上海圣母军第一个支会——“善导圣母军支团”。1949年初，又在震旦女中先后组建2个支团，分别名“可奇之母支团”和“仁慈之母支团”，共22人<ref>[http://www.shtong.gov.cn/newsite/node2/node4/node2249/luwan/node37494/node37507/node62749/userobject1ai22515.html 卢湾区志 >> 第三十二编民族·宗教 >> 第二章宗教 >>第一节　天主教] {{webarchive|url=https://web.archive.org/web/20090109013041/http://www.shtong.gov.cn/newsite/node2/node4/node2249/luwan/node37494/node37507/node62749/userobject1ai22515.html |date=2009-01-09 }}</ref>。1949年3月27日，沈士贤率上海圣母军人员在伯多禄堂举行誓师大会<ref>[http://www.shtong.gov.cn/newsite/node2/node2245/node75195/node75203/node75283/node75297/userobject1ai91966.html 上海宗教志 >> 第四编天主教 >> 第二章团体 >>第六节　其他团体] {{webarchive|url=https://web.archive.org/web/20131219141831/http://www.shtong.gov.cn/newsite/node2/node2245/node75195/node75203/node75283/node75297/userobject1ai91966.html |date=2013-12-19 }}</ref>。此后，圣母军在上海市迅速发展。

1951年6月6日，[[中国人民解放军上海市军事管制委员会|中国人民解放军上海市军事管制委员会]]宣布查封天主教中华全国教务协进委员会。1951年9月6日，陳哲敏、[[沈士賢|沈士賢]]、侯之正，高樂康，[[赵玉明|赵玉明]]5人在上海天主教協進會，[[莫克勤|莫克勤]]神父在高隆邦會同日被捕，關在上海市南車站路看守所（关押特务的地方）。1951年10月8日，上海市军管会宣布取缔[[圣母军|圣母军]]。

沈士賢因风湿性心脏病在上海市监狱医院病逝。侯之正神父也在关押期间病逝。1954年赵玉明主教、莫克勤神父被驱逐出境<ref>述良：[http://www.catholic.org.tw/cathlife/2002/2475/22.htm 陳哲敏神父的迴響] {{webarchive|url=https://web.archive.org/web/20060115134901/http://www.catholic.org.tw/cathlife/2002/2475/22.htm |date=2006-01-15 }}</ref>。

1958年10月18日，[[華理柱|華理柱]]主教在上海被判处20年徒刑。

1960年3月17日，[[陳哲敏|陳哲敏]]作为[[龔品梅反革命集团案|龔品梅反革命集团]]成员之一，同1955年9月8日被捕的[[龔品梅|龔品梅]]主教等人一起受审，被判处20年徒刑。判刑后，被送到安徽[[黃山|黃山]]，修築登山公路。1961年8月26日，陈哲敏在安徽省[[巢湖|巢湖]]地区[[庐江县|庐江县]][[白湖農場|白湖農場]]因營養极度缺乏、勞役過重死亡。

1970年7月10日，因中美关系将要有重大变化，華理柱主教被提前释放，成为最后一个离开中国大陸的西方传教士。

== 後續 ==
1952年，黎培理來到台灣，在[[台北|台北]]恢復了教廷駐華公使館（後於1959年升格為[[大使館|大使館]]）以及天主教教務協進會<ref name="主教團月誌1">[http://tc.wangchao.net.cn/baike/detail_203551.html 主教團之設立與結構 - 天主教中國主教團月誌 民國86年11月]</ref>。1967年4月21日，[[中國主教團|中國主教團]]（今[[天主教台灣地區主教團|台灣地區主教團]]）成立，天主教教務協進會改組為該團之秘書處，對外仍保留「天主教教務協進會」之名稱，但現已少用。
