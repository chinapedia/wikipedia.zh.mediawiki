'''天妃宫碑'''全名“'''御制弘仁普济天妃宫之碑'''”，位于[[中国|中国]][[江苏省|江苏省]][[南京市|南京市]][[鼓楼区|鼓楼区]][[静海寺|静海寺]]内，为明代[[南京天妃宫|南京天妃宫]]遗物，由明成祖[[朱棣|朱棣]]于[[永乐_(明朝)|永乐]]十四年（1416年）[[郑和|郑和]]第四次下西洋归来时所立，以纪念船队在航行中所受到的[[妈祖|妈祖]]庇佑，为直接反映[[郑和下西洋|郑和下西洋]]事迹的明代碑刻之一<ref>{{cite web |url=http://js.xhby.net/system/2012/07/10/013732599.shtml|title=《天妃宫碑住上了“阳光房”》|accessdate=2012-07-15|work=柳扬|publisher=新华报业网}}</ref>。清[[咸丰_(年号)|咸丰]]年间天妃宫毁于战乱，唯存此碑。碑以青石刻制，高5.48米，宽1.5米、厚0.5米，原龟趺头部已断裂，为今人修复，碑额为阴刻篆书，两面均刻双龙缠球纹，碑文为阴刻楷书，四周刻缠枝莲纹，全文共699字，现已风化严重、日渐模糊<ref>{{cite web |url=http://js.xhby.net/system/2012/03/15/012935411.shtml|title=《596岁天妃宫碑将穿上“保护衣” 今后游客只能远观》|accessdate=2012-07-15|work=王慧|publisher=新华报业网}}</ref>。另于重建的天妃宫碑亭内存有该碑的复制品<ref>{{cite web |url=http://www.fjsen.com/b/2009-01/23/content_1148601.htm|title=《南京天妃宫》|accessdate=2012-07-15|work=|publisher=东南网}}</ref>。
