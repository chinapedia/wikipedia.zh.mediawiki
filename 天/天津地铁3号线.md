'''天津地铁3号线'''，于2007年正式开工建设，2012年10月1日全线开通试运营，正线全长34公里，是天津市快速轨道交通网中的南北骨干线，途径华苑产业园区、水上公园、周邓纪念馆、天塔、津湾广场、天津站、[[天津北站|天津北站]]及[[北宁公园|北宁公园]]等地标。天津地铁3号线设[[南站站|南站]]、[[小淀站|小淀]]等共26座车站，地下站18座、高架站7座、地面站1座。其中，[[红旗南路站|红旗南路站]]可与[[天津地铁六号线|6号线]]换乘，[[营口道站|营口道站]]可与[[天津地铁一号线|1号线]]换乘，[[天津站站|天津站站]]可与[[天津地铁二号线|2号线]]、[[天津地铁九号线|9号线]]换乘，[[北站站|北站站]]可与[[天津地铁六号线|6号线]]换乘，[[张兴庄站|张兴庄站]]可与[[天津地铁五号线|5号线]]换乘。天津地铁3号线与天津地铁2号线合设1座主控制中心。天津地铁3号线南延长至[[西青区|西青区]][[天津南站|天津南站]]工程于2012年内开工，于2013年12月28日投入运营。

== 地理概述 ==
[[File:Line3WDD.JPG|thumb]]
26座车站的站台除高新区站、大学城站、华北集团和丰产河站为高架站侧式站台、西康路站为地下站侧式站台以及小淀站为地面站侧式站台之外全为地下站岛式站台。天津地铁三号线与天津地铁二号线合设1座主控制中心。天津地铁三号线自[[西青区|西青区]]的天津南站引出，过[[外环线|外环线]]进入天津市区，沿迎水道、过[[水上公园|水上公园]]，沿水上北路、气象台路、[[营口道|营口道]]、[[赤峰道|赤峰道]]，经[[天津站|天津站]]及天津站后广场附近的居住小区，沿昆纬路、三马路、下穿[[天津北站|天津北站]]及[[北宁公园|北宁公园]]，沿铁东路、穿过北环铁路，沿宜兴埠镇规划道路，过外环线后由地下改为高架沿津围公路前行，过丰产河后，由高架转为地面到达终点小淀。

== 工程建设 ==
[[File:Tianjin_metro_line_3_experience_ticket_back.png|thumb]]
天津地铁3号线略晚于2号线开始施工，工程进行的过程曲折，由于资金、施工和媒体不负责任报道等原因，竣工日期曾几度推迟。<ref>{{cite web |url=http://www.022net.com/channel/ts002/ |title=轨道上的天津 |publisher=人民网天津频道 |date=2012-10-11 |accessdate=2012年11月3日 |deadurl=yes |archiveurl=https://web.archive.org/web/20121114074840/http://www.022net.com/channel/ts002/ |archivedate=2012年11月14日 }}</ref>

2012年3月，3号线列车在华苑车辆段成功进行了首次热滑试验。其于9月15日开始对持有试乘票的乘客开放，並且已經於2012年10月1日，正式开通试运营。

== 运营时间 ==
天津地铁三號線於2012年10月1日開通试運營，雙向對開首班車的运营时间为6点，末班車的运营时间为22点。全線運行需時59分鐘。

== 列車 ==
{{main|天津地铁3号线列车}}
天津地鐵三號線列車由[[南车青岛四方机车车辆|中國南車青島四方機車車輛]]製造，在本線路運營。

== 車站 ==
[[File:Tianjin_Metro_Map_Line3_201806.png|center]]
3号线共设26座车站，其中地下站18座、地面站1座、高架站7座。
=== 换乘站 ===
==== 已投入使用 ====
* [[营口道站|营口道站]]：换乘1号线。1号线站台与站厅位于同一层，改造时已经预留换乘，乘客可通过站厅下至3号线岛式站台。
* [[天津站站|天津站]]：换乘2号线与9号线。3号线站台位于地下四层，2号线与9号线站台位于地下三层，3号线站台可通过楼梯直接换乘2号线曹庄方向，前往2号线滨海国际机场方向与9号线需经过地下二层站厅换乘。
* [[张兴庄站|张兴庄站]]：换乘5号线。3号线建设时已做预留，为站台垂直换乘。
* [[北站站|北站]]：换乘6号线。3号线建设时已做预留，为站台垂直换乘。
* [[红旗南路站|红旗南路站]]：换乘6号线。3号线建设时已做预留，为站台垂直换乘。
==== 将会成为换乘站的车站 ====
* [[和平路站_(天津地铁)|和平路站]]：换乘4号线。3号线建设时已做预留，为换乘层短通道换乘。
* [[中山路站|中山路站]]：换乘7号线。3号线建设时未做预留，且两站台相距较远，预计为通道换乘。
* [[天塔站|天塔站]]：换乘7号线。3号线建设时未做预留，换乘方式未知。
* [[西康路站|西康路站]]：换乘8号线。3号线建设时未做预留，换乘方式未知。
* [[吴家窑站|吴家窑站]]：换乘11号线。3号线建设时未做预留，换乘方式未知。

== 行车安排 ==
3号线采取单一线路运营，所有列车皆往返南站及小淀站之间，全程运行时间51分钟。
