[[天主教香港教區|天主教香港教區]]最初本屬[[天主教澳門教區|天主教澳門教區]]的管轄範圍<ref name="livro">{{pt}}{{zh-hant}}{{cite book |author=謝德恩（林家駿為原作者） |coauthors=劉鈺馨、何劉一星|title=天主教澳門教區主教簡介 |year= 2006 |publisher=澳門教區社會傳播中心 |location=澳門 |isbn=9993787701}}</ref>；由於[[教廷|教廷]]傳信部為了擺脫[[葡萄牙|葡萄牙]]保教權干預教會事務<ref>{{zh-hant}}{{cite book|author=田英傑 |title=香港天主教掌故 |year=1983 |publisher=聖神研究中心、[[聖神修院|聖神修院]]校外課程部|location=香港 }}</ref>，因此教廷於1841年4月22日宣布把[[香港島|香港島]]及其周圍六里地方脫離天主教澳門教區管轄範圍而成立宗座監牧區——香港監牧區，並委任教廷傳信部駐[[澳門|澳門]]代表若瑟神父為首任宗座監牧。1874年11月7日，天主教香港監牧區被升格為[[天主教香港教區|香港代牧區]]，其管轄範圍也擴至[[廣東省|廣東省]]的[[寶安縣|新安縣]]、[[归善县|归善县]]及[[海豐縣|海豐縣]]；同時委任高神父為首任宗座代牧。及至1946年，教廷把香港代牧區升格為一個聖統制主教區，成為現時的天主教香港教區；同日，香港代牧區第四任宗座代牧[[恩理覺|恩理覺]]被委任為首任天主教香港教區正權主教<ref name="DHK150">{{zh-hant}}{{cite book|author=天主教香港教區成立150週年慶典籌委會 |title=香港天主教會一百五十週年紀念特刊 |year=1991 |publisher=天主教香港教區成立150週年慶典籌委會 |location=香港 }}</ref>。

在天主教教義上，宗座監牧是由一名神父擔任管理宗座監牧區的最高負責人，而宗座監牧區是直屬於[[教廷|教廷]]傳信部的一個最雛型的[[傳教區|傳教區]]；宗座代牧是由一名領銜主教擔任宗座代牧區的首牧，而宗座代牧區和宗座監牧區一樣都是由直屬於教廷傳信部{{NoteTag|按天主教教律規定，宗座代牧只是歷史上古老教區的[[領銜主教|領銜主教]]而並不屬於該區的正權主教<ref name="DHK150" />}}<ref name="DHK150" />。正權主教是天主教教區的首牧外，也被視為[[宗徒|宗徒]]繼承的傳遞者<ref name="CCC">{{cite book |title=Catechism of the Catholic Church (CCC) |year=1993 |publisher=[[梵蒂岡出版社|梵蒂岡出版社]] |location=[[梵蒂岡|梵蒂岡]]}}</ref>，由[[聖神|聖神]]（Holy Spirit）挑選並透過人類即[[教宗|教宗]]在適當時委任<ref>{{cite web |title = How a Bishop is Chosen |website = frpat.com |accessdate=2010年5月28日 |url = http://frpat.com/bishopchoice.htm }}</ref>，因而[[天主教香港教區|天主教香港教區]]的[[主教|主教]]也不例外。被委任的天主教香港教區首牧都是作為[[基督|基督]]及教宗委派於天主教香港教區的代表<ref name="CCC" />；他除了在[[教宗|教宗]]職權的領導下負責分擔天主教香港教區的職責外，還需負責牧靈及慈善等工作<ref name="livro" />。

現時，天主教香港教區的管轄範圍為[[香港特別行政區|香港特別行政區]]全境，合共52個堂區<ref>{{cite web |title = 香港教區教務統計 |publisher = 天主教香港教區 |accessdate = 2010年5月20日 |url = http://www.catholic.org.hk/v2/b5/cdhk/a08statistics.html |archive-url = https://web.archive.org/web/20151117063717/http://www.catholic.org.hk/v2/b5/cdhk/a08statistics.html |archive-date = 2015年11月17日 |dead-url = yes }}</ref>。至目前為止，[[胡振中|胡振中]]、[[陳日君|陳日君]]以及湯漢均在出任天主教香港教區主教時被[[教宗|教宗]]擢升為[[樞機|樞機]]。下列為[[天主教香港教區|天主教香港教區]]歷任首牧列表{{NoteTag|表格中有斜杠符號（/）表示該格不適用}}<ref name="DHK150" />。
