'''天啟'''（{{lang-en|Apocalypse}}），本名'''恩·沙巴·奴爾'''（En Sabah Nur），是[[漫威漫畫|漫威漫畫]]中的虛構超級反派角色，為世上首個[[變種人|變種人]]，由作家{{link-en|路易絲·西蒙森|Louise Simonson}}和藝術家{{link-en|傑克森·吉伊斯|Jackson Guice}}所創作。天啟首次於《{{le|X-因子 (漫畫)|X-Factor (comics)|X-因子}}》#5中登場<ref>{{cite web|url=http://www.comicbookresources.com/?page=article&id=50581 |title=Walt Simonson Remasters "Star Slammers," Talks "X-Men: Apocalypse" |publisher=Comic Book Resources |date=2014-01-29 |accessdate=2014-05-27}}</ref>。

於2016年電影《[[X戰警：天啟|X戰警：天啟]]》中，由[[奧斯卡·伊薩克|奧斯卡·伊薩克]]飾演<ref name="OscarIsaac" /><ref name="x2016" /><ref name="variety" />。[[IGN|IGN]]將天啟列為漫畫反派百強排名第24名<ref>{{cite web |url=http://comics.ign.com/top-100-villains/24.html |title=Apocalypse is Number 24 |publisher=Comics.ign.com |date= |accessdate=2010-06-17 |deadurl=yes |archiveurl=https://web.archive.org/web/20100625081140/http://comics.ign.com/top-100-villains/24.html |archivedate=2010-06-25 }}</ref>。

== 人物 ==
天啟出生於[[埃及第一王朝|埃及第一王朝]]，由於皮膚灰黑，臉面上有淺藍色線條被認為是個怪物而被棄掉，被[[沙漠|沙漠]]中的一批遊蕩者收養，飽受「[[適者生存|適者生存]]」理論教育，在殘酷的戰鬥中長大。收養天啟的頭目似乎聽說過某種預言，認定這個小子將成為統治者。當時埃及的法老其實是來自30世紀的時空旅人[[征服者康|征服者康]]，他也知道天啟的潛力，打算控制他，但沒有成功就被迫離開了。

後來的幾千年中，天啟遊蕩人間，發動戰爭以測試國家、民族什麼才是「適者」。這個過程中，他發現了一些外星科技，進一步強化自己。到了20世紀，天啟認為超能力者與普通人類不可共存，因此他堅決反對「X戰警」這種支持和平共處的理想主義者。天啟是漫威世界的超級反派之一，和很多超級英雄發生過關聯。他不僅自己擁有很強的超能力，還有一批為他效力的使徒。

==能力==
他是首位變種人，具有強大的改造能力，他擁有過人的力氣，並且可以瞬間移動、飛行、及操控能量。最初只有附身的能力，但因為每當他快死的時候就會附上另一位變種人的身體，令他得到被他附身的變種人的異能。他每次附身就多一種能力，所以他幾乎有所有的變種人的能力。

==和其他角色的關係==
'''[[驚惡先生|驚惡先生]]'''（Mr. Sinister）
: 維多利亞時代的不死科學家，後期因接觸天啟而成為變種人，繼天啟又一X戰警系列中的大反派，長期以來是X戰警的死敵。漫畫後期被鳳凰寄身的獨眼龍等人消滅其創造的「驚惡城市」，但他並未死去，反而活了下來。

'''天啟四骑士'''
: 天啟的手下，四名重要使徒，名字分別為戰爭、飢荒、瘟疫、死亡。

'''[[機堡(漫威漫畫)|機堡]]
: 年幼時受天啟感染。
