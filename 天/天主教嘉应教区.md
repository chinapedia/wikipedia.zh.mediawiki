'''嘉应教区'''是[[天主教|天主教]]在中国[[广东省|广东省]]东北部设立的一个[[教区|教区]]。

== 历史 ==
1850年，[[法国|法国]][[巴黎外方传教会|巴黎外方传教会]]的神父将天主教传入广东嘉应（梅州）。到19世纪末，信徒发展到3000多人。

1925年，嘉应6县（梅县、兴宁、五华、蕉岭、平远、大埔）以及龙川、和平等县交给[[美国|美国]][[玛利诺外方传教会|玛利诺外方传教会]]负责。1929年2月20日，从汕头代牧区分设'''嘉应监牧区'''，1935年6月18日升格为设'''嘉应代牧区'''。

1946年4月11日，[[罗马教廷|罗马教廷]]在中国建立[[圣统制|圣统制]]，嘉应代牧区升格为'''嘉应教区'''。

1951年4月，外国[[传教士|传教士]]被[[中共政府|中共政府]]勒令全部回国，教务由中国籍神父张一之主持；1964年全部[[教堂|教堂]]被中共政府強制关闭，並沒收教会財產，直至1979年才准許恢复宗教活动。

== 教堂 ==
1949年有29间，2005年增加到45座。

== 教徒 ==
1950年有教徒22819人，2005年增加到4万多人。

== 历任主教 ==
*[[福尔德|福尔德]]（Francis Xavier Ford），美国玛利诺会士，1892年1月11日生于美国， 1917年12月5日晋升神父，1929年4月28日起，任嘉应监牧。1935年6月18日升为嘉应代牧，9月21日祝圣主教。1946年4月11日升为嘉应主教，1952年2月21日死于狱中（60岁）。
*[[鍾銓璋|鍾銓璋]]：获前教宗[[若望·保禄二世|若望·保禄二世]]认可主教，2000年病逝。
*[[廖宏清|廖宏清]]：1966年8月9日出生于广东省五华县长布镇天主教家庭。1985年－1991年修读于武昌[[天主教中南神哲学院|天主教中南神哲学院]]，1991年7月25日祝圣为教区第一位新神父；2003年9月26日，前教宗[[若望·保禄二世|若望·保禄二世]]任命他为教区第三任正权主教。
