'''天津市高级人民法院'''是[[中華人民共和國|中華人民共和國]][[天津市|天津市]]的[[高级人民法院|高级人民法院]]。

==沿革==
1949年1月17日，'''天津市人民法院'''成立，隶属[[天津市人民政府|天津市人民政府]]。1955年1月，天津市人民法院改为'''天津市高级人民法院'''。1956年，又设立了天津市中级人民法院。1958年2月11日，天津市改为[[河北省|河北省]]省辖市，撤销天津市高级人民法院，将天津市中级人民法院改为河北省天津市中级人民法院。1968年3月，天津市各级人民法院实行军事管制。1973年，撤销军事管制，恢复'''天津市高级人民法院'''、天津市中级人民法院建制<ref>{{Cite web |url = http://www.chinacourt.org/article/detail/2012/10/id/605401.shtml |title = 津城“老法院”浓缩百年审判史 |author =  |date = 2012-10-09 |publisher =中国法院网  |language =  |accessdate =  |quote =  }}</ref>。
