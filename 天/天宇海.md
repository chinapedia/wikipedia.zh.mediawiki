'''天宇海'''（{{lang-en|'''Oceanaire'''}}）是[[香港|香港]]的一個私人屋苑，位於[[沙田區|沙田區]][[馬鞍山_(香港市鎮)|馬鞍山]]保泰街18號，鄰近[[港鐵|港鐵]][[恆安站|恆安站]]。於2011年10月入伙。屋苑共有6座（每座再分為AB座），單位總數1143個，已包括14間獨立屋。單位於2010年10月預售樓花，並於四日內全數沽清。

== 分層單位 ==
所有單位為三至四房設計之大單位。共30層（不設4、13、14、24層）；兩梯四戶。
* 48間「天宇大宅」設於30樓連私人空中天台
* 25間「海畔花園大宅」設於平台層向海的連花園大宅。
* 6間「池畔花園大宅」設於平台層向會所的連花園大宅。
* 14間「御花園獨立屋」三層連花園及天台的獨立屋
== 交通 ==
住客車位409個；51個位於平台層。30個訪客車位。44個電單車位及358個單車車位<ref>[http://app.midland.com.hk/residential_ebook/%A4%D1%A6t%AE%FC-oceanaire  天宇海Oceanaire - 網上樓書- 美聯物業Midland Realty]</ref>，住客可由每座住宅大樓電梯直達所屬地庫停車場。

== 住客會所 ==
「''Club Oceanaire''」為天宇海之私人住客會所，由獲獎設計公司[[世界设计伙伴|世界设计伙伴]]（Design Worldwide Partnership）負責設計。會所共五層連園林佔地達9萬平方呎，造價達3.2億。 設施包括：多用途體育館、保齡球館、宴會廳、排舞室、私人迷你影院、戶外園林餐廳、小童遊戲間、室內外泳池以及長達45米的室外園林度假泳池。

=== 屋苑管理 ===
屋苑管理首兩年由[[高衛物業管理有限公司|高衛物業管理有限公司]]（為[[長江實業|長江實業]]集團之附屬公司）負責，分層單位每平方尺每月2.31元，獨立屋為2.27元。根據售樓書管理費包括由[[和記環球電訊|和記環球電訊]]所提供之寬頻上網及電話通訊費用；收樓時發展商透過律師樓通知業主取消此安排及調低管理費。

== 樓層被指誤導 ==
有買家向[[無線電視新聞|無線電視新聞]]投訴，指自己買入屋苑5樓下面的平台「海畔花園大宅」，但收樓時卻發覺「平台層」實為地面，比一輛雙層巴士的高度還要矮，「海畔」單位卻與海隔着行人路、馬路迴旋處及[[馬鞍山海濱長廊|海濱長廊]]，無法看到海景，有立法會議員及學者指發展商有意誤導買家。但發展商長實回應指，樓層分佈是因應地勢及建築設計而定，已獲政府部門審批，亦有在樓書內清楚列明<ref>[http://news.sina.com.hk/news/2/1/1/2575496/1.html 平台大宅貼地面 天宇海被轟誤導] {{webarchive|url=https://web.archive.org/web/20120214063333/http://news.sina.com.hk/news/2/1/1/2575496/1.html |date=2012-02-14 }}</ref>。

另外，因「平台層」乃於行人路旁，行人可以隨時透過落地玻璃飽覽業主生活及屋內情況，亦能夠伸手敲打屋內玻璃。再加上「平台層」附近有馬路，音量不時都維持在65至75[[分貝|分貝]]<ref>[http://www.youtube.com/watch?v=EWEmBW7NqmA 長實新樓 - 天宇海: 平台層即是地下，五樓即是二樓，政府話 OK (2012-02-12)]</ref>。
