'''新华中学'''是一所位于[[中华人民共和国|中华人民共和国]][[天津市|天津市]]的公立中学，座落在[[天津市|天津市]][[河西区_(天津市)|河西区]][[马场道|马场道]]99号，是[[天津市|天津市]]所直属市级重点中学之一，是传统“天津市五所”之一 。始建于1914年，前身为聖功學堂。新华中学现任校长为曹学良。<ref>[http://www.xinhuaedu.cn/site/default/13.html 新华简介] {{webarchive|url=https://web.archive.org/web/20120305083944/http://www.xinhuaedu.cn/site/default/13.html |date=2012-03-05 }}</ref>

== 历史 ==
民國三年甲申（1914年6月），当时[[天津|天津]]率先開始大興創辦學校之風，但所辦公立私立學校大多設在中國地界，而私人出資在租界裏辦學的還很少。夏景如先生為普及教育提出在租界設立學校。此主張得到天主教會神甫李魯宜、楊仁址與教友陳盡仁、英實夫等大力支持，並各自拿出錢財作為辦學的費用，初推英（實夫之妻）夫人懷清擔任校長。在[[天津法租界|天津法租界]]義慶裏租校舍招生開學。定名為“聖功學堂”，又名“聖功女學校”。“聖功”兩字出自<易經>，“蒙卦第四：蒙以養正，聖功也。”這是一座天主教會女子學校，創始人中以天主教愛國人士為主。校訓為：“溫，良，恭，儉”。1915年，夏景如成为圣功女学校首任校长。1941年，学校在[[马场道|马场道]]陶园建设圣功楼，中学部随即迁入，而小学部迁回[[天津法租界|天津法租界]]，改名为“劝业场小学”。<ref>[http://blog.sina.com.cn/s/blog_4882283d0100dul5.html 臺灣驚現天津聖功女中的克隆校]</ref>1978年，改名为“新华中学”并被确定为天津市首批重点校。1994年，被列入《中国名校大全》。现为天津市教育委员会直属重点中学。

== 校名沿革 ==
新华中学、初创时，名为“圣功学堂”，后易名为“圣功女中”。1952年12月，圣功女中改为“市立天津师范学院女附中”。1956年，改名为“天津师范学院附中”。1958年后，先后改名为“天津师范大学第一附属中学”、“天津师范大学附属中学”和“河北大学附属中学”。1973年易名为“天津市新华中学”。

== 在台复校 ==
1963年，圣功女中[[在台复校|在台复校]]，地址位于[[臺南市|臺南市]][[北區_(臺南市)|北區]]，現稱為“臺南市私立聖功女子高級中學”。新华中学校长曾两次访问该校，海峡两岸两“圣功”交流频繁。<ref>[http://www.jwb.com.cn/jwb/html/2009-05/01/content_306663.htm 从“圣功”到“新华”]{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>在台復校後,聖功女中校務蒸蒸日上，聖功女中在台灣持續稟持天主教大愛的精神，每兩年均會舉行愛心園遊會幫助弱勢團體。

== 概况 ==
天津市新华中学位于[[天津市|天津市]][[河西区_(天津市)|河西区]][[马场道|马场道]]99号，占地4万平方米，以新华中学为依托的民办校华宁中学位于[[河西区_(天津市)|河西区]]琼州道111号，占地9亩。新华中学与华宁中学共有90个教学班，在校学生5100人。目前，新华中学全校共有教职员工334人。<ref>[http://www.xinhuaedu.cn/site/default/16.html 师资队伍] {{webarchive|url=https://web.archive.org/web/20111006180918/http://www.xinhuaedu.cn/site/default/16.html |date=2011-10-06 }}</ref>

== 知名校友 ==
* 鄭汝銓——現任天津基督教女青年會名譽總幹事，市政協委員，市婦聯名譽執委，市基督教三自愛國會名譽主席。首屆聖功學生。著名愛國宗教人士、社會活動家、教育家、慈善家、與中國話劇同齡的話劇演員。曾長期擔任天津、南京基督教女青年會總幹事等職務。
* 英木蘭修女——現爲天主教會北京教區教務委員會副主席、北京天主教愛國會副主席、北京市政協委員。出身于天主教世家的英氏家族。
* 胡蝶——民國時期著名影星
* 伍小平——中国科学院院士，、中國科技大學教授、著名數力專家，55屆畢業生。
* 寇紀淞——1965屆高中畢業生，在校期間曾獲天津市數學競賽一等獎。教授,博士生導師,全國政協委員,天津大學副校長, 天大天財公司董事長。
* 楊希——中國女排勇奪五連冠的主攻手。
* 吳奇——67屆高中畢業生，曾任天津中醫附院內科醫生，1988年三藩市中醫針灸學院教授，三藩市天津海外聯誼會會長。
* 高懷忠——66屆高中畢業生。2001年任國務院國有資產監督管理委員會監事會主席。
* 王素卿——68屆高中畢業生。現任國家住房和城鄉建設部建築市場管理司司長，全國註冊建築師管理委員會主任，全國註冊結構工程師管理委員會主任。
