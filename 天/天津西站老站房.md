'''天津西站老站房'''是原[[天津西站|天津西站]]的候车室。位于原[[天津老城|天津老城]]城西第三區[[子牙河|子牙河]]與[[南運河|南運河]]之間的河北赵家场（今[[天津市|天津市]][[红桥区|红桥区]]西站前街1号）。始建于1909年8月，1910年12月14日投入运营，建築風格為德式[[新古典主義|新古典主義]]，目前，该建筑被[[中华人民共和国国务院|中华人民共和国国务院]]批准为[[天津全国重点文物保护单位列表|全国重点文物保护单位]]<ref>[[中华人民共和国国务院|中华人民共和国国务院]]，二零一三年五月三日公布，[[天津市人民政府|天津市人民政府]]立</ref>，[[天津市文物保护单位|天津市文物保护单位]]<ref>[[天津市人民政府|天津市人民政府]]，一九九七年六月二日公布</ref>和[[天津历史风貌建筑|特殊保护等级历史风貌建筑]]<ref>[[天津市人民政府|天津市人民政府]]，二零零五年八月三十一日公布</ref>。天津西站老站房作為[[紅橋區|紅橋區]]的百年標誌性建築现因[[天津西站|天津西站]]的改扩建工程而搬离原址并将作为天津铁路博物馆被永久保护，成为西站城市副中心的主要标志之一。<ref>[http://www.022net.com/2010/5-7/424776172676933.html 天津西站小洋楼进入整体抬升阶段 将作为博物馆永久保护] {{webarchive|url=https://web.archive.org/web/20130731071930/http://www.022net.com/2010/5-7/424776172676933.html |date=2013-07-31 }}</ref>

1898年9月，[[英国|英国]]和[[德国|德国]]的资本集团在[[伦敦|伦敦]]举行会议并决定承办津镇铁路（[[天津|天津]]至[[镇江市|镇江]]）。清政府屈服于压力，于1899年5月签订了借款草约，1908年签订了借款合同，并将津镇铁路改为津浦（[[南京市|南京]][[浦口区|浦口]]）铁路。但是与众不同的是，[[津浦铁路|津浦铁路]]北段当时从半截开始施工，从[[静海县|静海县]]的良王庄向天津市内修筑，这样做是因为天津起点的地址始终未定。督办铁路大臣吕海寰派出的调查员报送了一份极有价值的材料：“河北赵家场后空地有二顷有余，地势平坦，既无庐舍，又无坟墓，堪为建设总站之用。”当时所称的“河北赵家场”，实际就是如今的[[红桥区|红桥区]]北营门至南运河一带，由于当时南运河并未裁弯，故在河北岸，被称为“河北赵家场”，距离今天的天津西站不过500米。后在此修建了赵家场站，即现在的天津西站候车室。<ref>[http://news.enorth.com.cn/system/2006/12/15/001491846.shtml 纪录天津近代史沧桑 天津西站]</ref>

天津西客站主站房为砖混结构二层楼房，建筑整体坐南朝北，立面强调对称式构图，造型丰富，建筑平面呈凸字形。建筑顶部为红瓦坡顶，开有[[老虎窗|老虎窗]]和烟囱，<ref name="roof">[http://news.enorth.com.cn/system/2008/04/08/003099822.shtml 整修日耳曼风格候车楼 百岁西站再现德式风情]</ref> 外立面为清水砖墙，窗套、立柱、花饰及入口台阶均为石材，上雕有[[南极仙翁|南极仙翁]]仙鹤还有龙嘴的图案与西洋风味的洋楼相配可谓中西合璧，窗式和窗套变化多样，是一座具有典型折衷主义风格的德式[[新古典主義|新古典主義]]建筑。<ref>《天津历史风貌建筑简介》</ref>

== 平移和保护 ==
2009年9月24日，天津西站候车楼采用滑动摩擦平移方法平移至新址，这是天津市首例木结构建筑的平移工程，第一阶段向南平移135米。<ref>{{cite web |url=http://www.022net.com/2009/9-25/422726353023874.html |language=中文 |title=天津西站候车楼开始平移 将成为铁路博物馆 |publisher=人民网天津视窗 |date=2009年9月25日 |accessdate=2011年2月25日 |deadurl=yes |archiveurl=https://web.archive.org/web/20090928111633/http://www.022net.com/2009/9-25/422726353023874.html |archivedate=2009年9月28日 }}</ref>10月23日，第二阶段向东平移40米。11月9日上午11时天津西站候车楼完成平移，到达新址地基所在地终点。<ref>{{cite web |url=http://www.022net.com/2009/10-23/483723333141446.html |language=中文 |title=天津西站候车楼立定左转 平移175米刷国内纪录 |publisher=人民网天津视窗 |date=2009年10月23日 |accessdate=2011年2月25日 |deadurl=yes |archiveurl=https://web.archive.org/web/20130731090613/http://www.022net.com/2009/10-23/483723333141446.html |archivedate=2013年7月31日 }}</ref>2010年5月4日西站候车楼改造进入整体抬升阶段，工程设计由98个千斤顶将候车楼抬升3.6米。平移完成后的西站小洋楼作为铁路博物馆永久保留，并与西沽公园共同构成天津西站地区城市副中心的两大景观。<ref>{{cite web |url=http://www.022net.com/2009/11-10/423921203250291.html |language=中文 |title=天津西站小洋楼移到位 成西站副中心永久历史地标 |publisher=人民网天津视窗 |date=2009年11月10日 |accessdate=2011年2月25日 |deadurl=yes |archiveurl=https://web.archive.org/web/20130731090644/http://www.022net.com/2009/11-10/423921203250291.html |archivedate=2013年7月31日 }}</ref><ref>{{cite web |url=http://www.022net.com/2010/5-5/484129152648477.html |language=中文 |title=图文:天津西站小洋楼开始整体抬升 预计抬升到3.6米 |publisher=人民网天津视窗 |date=2010年5月5日 |accessdate=2011年2月25日 |deadurl=yes |archiveurl=https://web.archive.org/web/20130731075355/http://www.022net.com/2010/5-5/484129152648477.html |archivedate=2013年7月31日 }}</ref>

此次整修的原则是“修旧如故，有机更新”。依照候车楼的老照片和历史档案，恢复原有的[[老虎窗|老虎窗]]和烟囱，墙体外立面上安装的排水管、空调等室外设施将被遮蔽住。

== 轶事 ==
[[二十世纪|二十世纪]][[八十年代|八十年代]]，在刘文亨和班德贵合说的[[相声|相声]]作品《新风尚》中，讲述了[[春节|春节]]期间一位来自[[广东|广东]]的老人在[[天津西站|天津西站]]下车，老人想找自己的儿子但由于不识字又只会说[[粤语|粤语]]，在刚下过雪的冬天却急得满头大汗，[[逗哏|逗哏]]演员刘文亨化身天津西站周边的交通民警将老人安顿在西站交通队后便只身来到位于西站前街的天津电工器材制造西厂，希望得到该厂南方职工的帮助。当时正值工厂午休时间，民警便来到工厂食堂，焦急之际刚好进来一位广东职工。二人立刻回到交通队立即帮老人“翻译”出了老人的儿子叫于浩海，底包袱为捧哏演员班德贵拿于浩海的粤语发音“咦呼嗨”抓哏。<ref>{{cite web |url=http://www.022net.com/2009/2-5/485426152321579.html |language=中文 |title=天津西站：百年老车站的生态变迁(图) |publisher=人民网天津视窗 |date=2009年2月5日 |accessdate=2011年2月25日 |deadurl=yes |archiveurl=https://web.archive.org/web/20090216084451/http://www.022net.com/2009/2-5/485426152321579.html |archivedate=2009年2月16日 }}</ref>
