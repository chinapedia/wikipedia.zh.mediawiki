《'''天天開心'''》、《'''開心舞台'''》、《'''金舞台'''》、是[[臺灣電視公司|臺灣電視公司]]於1986年至2000年播出的三個午間[[綜藝節目|綜藝節目]]，皆由[[曹景德|曹景德]]製作。雖號稱三個節目，實則[[三位一體_(消歧義)|三位一體]]，節目內容大同小異，主要的差別是在播出時間和主持人。《天天開心》有[[黃西田|黃西田]]作詞作曲兼主唱的台語同名主題曲，原本在節目片頭中使用該曲；但在播出800多集後（大約在1991年初），製作單位將片頭開放給[[台語流行音樂|台語流行音樂]][[歌手|歌手]]打歌，改為由各[[唱片公司|唱片公司]]提供旗下台語歌手的當時最新主打歌（只播第一段）搭配片頭播出。《天天開心》結束多年後，台視於2011年5月找回原班底推出續作《'''天天又開心'''》，希望再創《開心》熱潮，但[[收視率|收視率]]不佳，僅播出一季便停播。

== 節目內容 ==
*每集大致分為三段，每段之間以[[廣告|廣告]]區隔。以1990年代初期的《天天開心》結構為例：第一段是無固定單元名稱的時裝短劇單元，第二段是古裝短劇單元〈古代人劇場〉，第三段是皆無固定單元名稱的台語歌手打歌時間與[[俚語|俚語]]時間。
*《金舞台》的節目尾聲，固定用台語講的台詞是：「（司馬玉嬌：）'''《金舞台》三十分鐘『咻』一個就過去了，'''（青蓉：）'''又要跟你講『再見』囉！'''（石松：）'''下禮拜日中午十二點至十二點半，請繼續收看台視——'''（石松、司馬玉嬌、青蓉：）'''《金舞台》！'''」

==單元介紹==
; 《天天開心》
*時裝短劇（本單元無官方名稱）：時裝短劇單元，短劇名稱各不相同。
*〈古代人劇場〉：古裝短劇單元，短劇名稱各不相同。
*打歌時間（本單元無官方名稱）：開放給台語歌手現場演唱新歌，搭配[[台視大樂團|台視大樂團]]現場伴奏。
*〈俚語時間〉：是以介紹台語[[俚語|俚語]]來說明做人處事的道理。每集4位主持人向觀眾一鞠躬以後，以一人一句方式唸每句7字的台語[[四句聯|四句聯]]作為[[定場詩|定場詩]]，然後解說當天介紹的俚語涵義。最後主持人們會以台語說：「咦～要注意！」（不一定是「要注意」三字，要看當天介紹的俚語涵義而定。例如，如果當天介紹的俚語涵義是鼓勵[[終身學習|終身學習]]，該句台詞就會改為：「咦～充實自己！」）來加強效果並做為節目的結尾，此一橋段後來成為不少人在日常生活中經常模仿的用語。此單元推出時本無固定的單元名稱，直到1990年代中、後期才定名為〈俚語時間〉。
*〈天天翻NEWS〉：「說[[新聞|新聞]]」單元，「翻」字官方讀音「něw」。高群飾演「海口群」，司馬玉嬌飾演「膨風美」，石松飾演「興啼松」，卓勝利飾演「報馬卓」，[[楊凱琪|楊凱琪]]飾演「雜念琪」。由其中二或三人擔任[[台視新聞|台視新聞]][[主播|主播]]，對談[[報紙|報紙]]上的一則[[社會新聞|社會新聞]]。結尾與〈俚語時間〉相同，或以固定台詞「天天翻NEWS，天天大家富」作結。
*〈鄉親逗陣來唱歌〉：開放觀眾來信報名的歌唱單元，無競賽成分，同樣搭配台視大樂團現場伴奏。報名的觀眾在[[明信片|明信片]]背面填寫個人基本資料（[[姓名|姓名]]、聯絡地址、聯絡[[電話|電話]]），寄到台視（收件地址即[[台視大樓|台視大樓]]地址），即完成報名。

; 《開心舞台》
*〈民謠小調〉：歌仔戲單元，所唱的都是觀眾耳熟能詳的歌仔戲調，能欣賞中國傳統說唱藝術的奧妙、進而了解流傳在台語的民間故事。
*〈卓嬸婆與嬌小妹〉：短劇單元，卓勝利反串扮演卓嬸婆，司馬玉嬌扮演嬌小妹。卓嬸婆是個粗魯卻自命嬌滴滴的中年婦女，她梅開五度，剋死四個「先夫」。卓嬸婆的最後一位先夫怕死於非命，拔腿就跑；卓嬸婆不堪寂寞，又起了嫁人的念頭。嬌小妹是卓嬸婆的女兒，反對卓嬸婆再嫁。全劇便在卓嬸婆找工作、找男人之間，牽扯出不少笑料。

; 《金舞台》<ref>王淑燕，〈台視推出台語綜藝節目「金舞台」〉，《[[電視周刊|電視周刊]]》第1254期（[[台視文化公司|台視文化公司]]1986年出版）。</ref>
*〈舊情綿綿〉：介紹台語老歌。
*〈千奇百怪〉：介紹無奇不有的新鮮事物。
*〈古早民俗傳奇短劇〉：新編台灣民間故事。
*〈土地公土地婆〉：古裝短劇單元，石松飾演土地公，青蓉飾演土地婆。
*〈大爆笑〉：以靈活創新的時裝短劇表演時事趣聞。
*一般歌曲（本單元無官方名稱）：邀請著名台語歌星現場演唱代表歌曲。

;《天天又開心》
*〈現代人劇場〉：時裝短劇單元，以「天天民宿」與「開心民宿」為主要場景。由長青飾演天天民宿老闆，唐儷飾演天天民宿老闆的外甥女，潘麗麗飾演開心民宿老闆娘，高群飾演開心民宿老闆娘的弟弟，每天都有不同的劇情發生在這兩間民宿。
*〈俚語時間〉：完全重製《天天開心》的同名單元。

==大事紀==
*1990年9月20日，石-{松}-透露，[[中華民國總統|中華民國總統]][[李登輝|李登輝]]曾於該年7月召見他並嘉勉他「要為保留[[台灣文化|台灣文化]]繼續奮鬥」，李登輝並希望將《天天開心》每日三句的鄉土俚語出書。該書已由[[台視文化公司|台視文化公司]]整理出版。<ref>中華民國電視學會電視年鑑編纂委員 編纂，《中華民國電視年鑑第七輯：民國七十九年至八十年》，中華民國電視學會1992年6月30日出版。</ref>
*1992年12月5日，[[中國電視公司|中國電視公司]]籌備中的綜藝節目《[[娛樂星聞_小燕有約|娛樂星聞 小燕有約]]》邀請《天天開心》製作人曹景德及主持人石-{松}-、卓勝利、司馬玉嬌在本月26日20:00《娛樂星聞 小燕有約》開播時擔任第一集來賓，曹景德已徵得台視節目部經理[[熊廷武|熊廷武]]同意；《娛樂星聞 小燕有約》第一集內容重點為回顧該年娛樂新聞，「表揚」該年收視率表現突出的節目。<ref>王祖壽 報導，〈小燕到中視 26日“有約”〉，《[[民生報|民生報]]》1992年12月6日第12版。</ref>
*2018年2月15日，方駿、長青、高群於中視[[除夕|除夕]]特別節目《新春旺旺[[冰冰show|冰冰show]]》共同演出表演節目〈天天開心經典脫口秀〉，開場合唱《天天開心》主題曲。
