'''天泰宾馆'''（'''中国人民解放军火箭军招待所'''），位于[[北京市|北京市]][[西城区|西城区]][[南礼士路|南礼士路]]头条1号，隶属[[中国人民解放军火箭军后勤部|中国人民解放军火箭军后勤部]]。

==简介==
2012年8月，中国人民解放军第二炮兵后勤部招待所（天泰宾馆）开业，隶属[[中国人民解放军第二炮兵|中国人民解放军第二炮兵]]后勤部<ref name=xjb>{{Cite web|url =http://news.21cn.com/caiji/roll/a/2013/0304/04/14754429.shtml|title =媒体盘点人大代表团接待宾馆：5年26家参与接待|author =|date =2013-03-04|publisher =21CN|language =|accessdate =|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20170329051259/http://news.21cn.com/caiji/roll/a/2013/0304/04/14754429.shtml|archivedate =2017年3月29日|df =}}</ref>。2012年6月28日，由建峰建设集团股份有限公司负责施工的天泰宾馆工程获得第二炮兵优质营房工程奖<ref>{{Cite web |url = http://www.jianfengjt.cn/cn/2012-10-16/ |title = 集团负责施工的北京天泰宾馆荣获第二炮兵优质营房工程一等奖 |author =  |date = 2012-10-16 |publisher = 建峰集团 |language =  |accessdate =  |quote =  }}</ref>。2015年12月，中国人民解放军第二炮兵升格为[[中国人民解放军火箭军|中国人民解放军火箭军]]，该宾馆隶属中国人民解放军火箭军后勤部，成为中国人民解放军火箭军招待所（天泰宾馆）<ref>{{Cite web |url = http://www.lawxp.com/case/c25558190.html |title = 原告韩◑◑与被告中国人民解放◑◑◑◑◑后勤部招待所、被告中国平安财产保◑◑◑◑◑公司北京分公司生命权、健康权、身体权纠纷一案 |author =  |date =  |publisher =汇法网  |language =  |accessdate =2017-03-28  |quote =  }}</ref>。

天泰宾馆开业后，自2013年起承担每年“全国[[两会|两会]]”中的[[全国人大|全国人大]]会议代表团的接待任务<ref>{{Cite web |url = http://www.npc.gov.cn/pc/12_5/node_31919.htm |title =十二届全国人大五次会议驻地  |author =  |date =  |publisher = 中国人大网 |language =  |accessdate = 2017-03-28 |quote =  }}</ref><ref name=xjb/>。
