'''天門戰鬥'''發生於1932年5月2日－30日，地點則是在[[中國|中國]][[湖北省_(中華民國)|湖北省]][[天门县|天门县]]张家场（在今天门市[[九真镇|九真镇]]），中共方面又称为张家场战斗或九真张家场战斗<ref name="a3">{{Cite web |url=http://www.tmswb.gov.cn/dstd/jdxd/201206/t20120612_211102_8.html |title=《中国共产党天门历史》（第一卷）》第八章“左”倾错误的危害与天门苏区的丧失|accessdate=2016-07-12 |author= |date=2012-06-12 |work= |publisher=中共天门市委办公室网站，来源：中共天门市委党史研究室|language=Zh-hans }}</ref>，交戰一方為[[中華民國|中華民國]][[國民革命軍|國民革命軍]]，另一方則為[[中國共產黨|中國共產黨]]所領導之[[中國工農紅軍|中國工農紅軍]][[红三军|红三军]]。此次战斗，国军取得完全胜利，亦被认为是在[[夏曦|夏曦]]主导下，红三军“得不偿失的消耗战”<ref name="a2">{{Cite web |url=http://military.china.com/history4/62/20140805/18685551_1.html |title=红三军保卫洪湖苏区：瓦庙集战斗后红军只剩2000人(2) |accessdate=2016-07-12 |author= |date=2014-08-05 |work= |publisher=中华网 |language=Zh-hans |deadurl=yes |archiveurl=https://web.archive.org/web/20160809101713/http://military.china.com/history4/62/20140805/18685551_1.html |archivedate=2016-08-09 }}</ref>。

这是自5月初，国军对红军发起的[[刁汊湖戰鬥|刁汊湖戰鬥]]的另一戰場，戰鬥起初，「鄂豫皖剿匪軍左路」第44師師長蕭之楚急行佔領[[應城|應城]][[皂市|皂市]]，並以此為據點對天門猛攻。<ref>[http://lib.jmu.edu.cn/departments2/magazine/philosophyol/1932.htm 中華民國史事日誌] {{webarchive|url=https://web.archive.org/web/20160827183737/http://lib.jmu.edu.cn/departments2/magazine/philosophyol/1932.htm |date=2016-08-27 }}</ref>5月23日，[[贺龙|贺龙]][[段德昌|段德昌]]和代政委[[宋盘铭|宋盘铭]]率红三军主力，对皂市附近九真镇张家场<ref name="a3"/>一带的国军第44师132旅及补充团(分由44師長蕭之楚、[[徐源泉|徐源泉]]([[湖北全省清郷督辦|湖北全省清郷督辦]])率)发起袭击。29日，国军第44师131旅和补充一团从皂市来援，红军退至文家墩。30日，国家第10军特务团来援。红三军撤出战斗。国军131旅旅长于兆龙以下八百人伤亡<ref>{{Cite web |url=http://cpc.people.com.cn/GB/69112/78233/78687/78977/5442457.html |title=《贺龙年谱》1932年　三十六岁 |accessdate=2016-07-12 |author= |date= |work= |publisher=中国共产党新闻网，来源：《贺龙年谱》|language=Zh-hans }}</ref>。31日，国军出击至柳河，红三军主力已撤退<ref name="a2"/>。

张家场战斗遗址现已列入天门市[[爱国主义教育基地|爱国主义教育基地]]<ref name="a1">{{Cite web |url=http://news.cnhubei.com/xw/2013zt/2013qhxj/qhdl/tianmen/zjdh/cs_zjdh/201502/t3185422.shtml |title=张家场战斗遗址|accessdate=2016-07-12 |author=编辑：刘晓洁  |date=2015-02-15 |work= |publisher=荆楚网|language=Zh-hans }}</ref>。
