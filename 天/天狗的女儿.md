《'''天狗的女儿'''》（{{lang-ja|町でうわさの天狗の子}}）是[[日本漫畫家|日本漫畫家]][[岩本奈绪|岩本奈绪]]的[[日本漫畫|日本漫畫]]作品，在[[小學館|小學館]]的《[[月刊flowers|月刊flowers]]》2007年6月號開始連載到2013年12月號結束。<ref>{{Cite web |url=http://natalie.mu/comic/news/102319 |title=「天狗の子」6年半の連載に幕！最終巻は原画集付き限定版も |work=[[natalie|natalie]] |date=2013-10-28}}</ref>單行本全12冊。

2009年獲得TV Bros.漫畫賞。<ref>{{Cite web |url=http://natalie.mu/comic/news/23670 |title=岩本ナオ、TVブロスマンガ大賞受賞で喜びのコメント |publisher=natalie |date=2009-11-11 |accessdate=2013-11-17}}</ref>2010年獲得第55回[[小學館漫畫賞|小學館漫畫賞]]少女漫畫組得獎。<ref>{{Cite web |url=http://natalie.mu/comic/news/102319 |title=第55回小学館漫画賞決まる。天狗、SKET、深夜食堂 |work=natalie |date=2010-03-05}}</ref>
