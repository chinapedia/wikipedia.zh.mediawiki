'''天主教永嘉教區（拉丁語：Dioecesis Iomchiavensis）'''是[[天主教|天主教]]在[[中國|中國]][[浙江省|浙江省]]南部所設立的一個[[教區|教區]]，俗称温州教区。在聖座規定中屬[[天主教杭州總教區|杭州總教區]]。

== 历史 ==
===成立初期===
1949年6月29日（圣保禄宗徒瞻礼），罗马教廷宣布，从[[宁波教区|宁波教区]]分设永嘉教区，教务由宁波教区主教[[戴安德|戴安德]]（André-Jean-François Defebvre）代理，直到新主教就职为止。戴安德主教任命[[苏希达|苏希达]]神父为永嘉教区副主教。

===強拆十字架===
{{Main|浙江強拆十字架運動}}
[[浙江省政府|浙江省政府]]自2014年初開始，強行拆除浙江省內[[基督新教|基督新教]]教堂頂部的十字架。甚至，夷平整座教堂的運動。於2016年2月25日凌晨五時，天主教地下團體永強堂區狀元天主堂的十字架，被當地政府拆除。<ref>{{cite news|url=http://china.ucanews.com/2016/02/26/%E6%BA%AB%E5%B7%9E%E5%86%8D%E6%9C%89%E5%A4%A9%E4%B8%BB%E6%95%99%E5%A0%82%E5%8D%81%E5%AD%97%E6%9E%B6%E8%A2%AB%E6%8B%86%EF%BC%8C%E7%82%BA%E4%BB%8A%E5%B9%B4%E9%A6%96%E5%AE%97/ |publisher=[[天主教亞洲通訊社|天主教亞洲通訊社]] |title=溫州再有天主教堂十字架被拆，為今年首宗 |language=zh-hans|date=2016年2月25日|accessdate=2016年2月26日}}</ref>

===強行安裝監視器===
[[浙江省政府|浙江省政府]]規定溫州市內的天主教和基督教教會必須在2017年3月底前，在各教堂內外安裝監視器。但由於安裝突然，教區未能及時回應和阻止。

== 教堂 ==
永嘉教區的主教座堂位於[[溫州市|溫州市]]的[[温州周宅祠巷圣保禄主教座堂|温州周宅祠巷圣保禄主教座堂]]， 教区內官方應可及地下團體共有188所圣堂<ref name=singapore>新加坡中华公教联络社：《中国天主教指南》、河北信德室：《中国天主教手册》</ref>

== 教徒 ==
温州地区目前集中了浙江省的大多数的天主教徒，约有15万人，有「中國耶路撒冷」之稱<ref name=singapore />。
