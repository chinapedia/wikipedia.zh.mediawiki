'''天南星'''（[[学名|学名]]：{{lang|la|''Arisaema heterophyllum''}}）是[[天南星科|天南星科]][[天南星属|天南星属]]的植物。

== 形态 ==
多年生有毒[[草本|草本]]；块茎略呈球形；掌状复叶，披针形小叶；夏季开花，[[肉穗花序|肉穗花序]]外包紫色或绿色佛焰苞；[[浆果|浆果]]多数，成熟时为鲜红色。

== 分布 ==
分布于[[朝鲜|朝鲜]]、[[日本|日本]]以及[[中国大陆|中国大陆]]的除西北和[[西藏|西藏]]外的大部分省区等地，生长于海拔300米至2,700米的地区，见于灌丛或草地。

== 药用 ==
中医上以[[球茎|球茎]]入药，性温、味苦辛，有毒，功能祛风化痰，主治[[中风|中风]]、[[破伤风|破伤风]]、小儿惊风、风痰癫痫等症；外敷痈肿。

本品研末用牛胆汗拌制而成，称“胆南星”，性凉、味苦，功能清火化痰、镇惊定痫，主治惊痫、痰厥、风痰、眩晕等症。
