'''天忍日命'''（{{lang|ja|あまのおしひ の みこと}}）為《[[記紀|記紀]]》二書裡的記述，祂是[[日本神話|日本神話]]裡隨著[[天津彥彥火瓊瓊杵尊|天津彥彥火瓊瓊杵尊]]降臨[[葦原中國|葦原中國]]的[[神祇|神祇]]，亦是{{link-ja|大伴氏}}祖神。

== 概要 ==
=== 日本書紀之記載 ===
《日本書紀》卷第二神代裡的第二種說法提到[[高皇產靈神|高皇產靈尊]]以眞床覆衾裹著[[瓊瓊杵尊|天津彦國光彦火瓊瓊杵尊]]，並引開天磐戸和排開天八重雲，使之降臨[[葦原中國|葦原中國]]。此時大伴連遠祖'''天忍日命'''、帥來目部遠祖[[天津久米命|天槵津大來目]]背負天磐靫、臂著稜威高鞆、手捉天梔弓及[[天羽羽矢|天羽羽矢]]及八目鳴鏑、帶頭槌劍，立於天孫之前。三神抵[[日向國|日向]][[高千穗峰|高千穗之山峰]]，輾轉到了吾田國（今[[南薩摩市|南薩摩市]]）長屋笠狹之碕時[[鹽土老翁|事勝國勝長狹]]出現，此人請天孫在該地任意遊之。

=== 古事記之記載 ===
據《古事記》上卷之記錄，[[瓊瓊杵尊|天津日子番能邇邇藝命]]離開天之石位，撥天之八重雲，挾其神威前往[[葦原中國|葦原中國]]。於天浮橋下至浮島，又於浮島下降立於[[筑紫國|筑紫]][[日向國|日向]]之[[高千穗峰|高千穗]]之久士布流岳。後來命'''天忍日命'''、[[天津久米命|天津久米命]]二人取天石箭筒，佩帶頭椎之大刀、天之波士弓及天之真鹿兒矢，立於邇邇藝命之面前而仕奉。

=== 其他 ===
雖然《古事記》本身並無記載，但《伴氏系圖》提到天忍日命乃[[高皇產靈神|高皇產靈神]]之子<ref>參看{{ja}}[http://www.asahi-net.or.jp/~sg2h-ymst/history1.html 大伴氏の歴史] {{webarchive|url=https://web.archive.org/web/20120513112423/http://www.asahi-net.or.jp/~sg2h-ymst/history1.html |date=2012-05-13 }}。</ref>。而[[神武東征|神武東征]]時隨侍在[[神武天皇|神武天皇]]之側的道臣命（{{ja}}ミチオミノミコト），為天忍日命的[[曾孫|曾孫]]。

== 解說 ==
值得注意的是，《古事記》記載天忍日命和天津久米命是同等地位，但是《[[日本書紀|日本書紀]]》卷第三記錄[[神武東征|神武東征]]時，提及天忍日命子孫道臣命之屬下為天津久米命之子孫大來目部，二者地位並不對等。

== 信仰 ==
日本祭祀天忍日命的[[神社|神社]]並不常見，通常在大伴氏後代居住之地，主要有下列：
* 降幡神社（[[大阪府|大阪府]][[南河内郡|南河内郡]]河南町）
* 油日神社（[[滋賀縣|滋賀縣]][[甲賀市|甲賀市]]）
* 矢保佐神社（[[長崎縣|長崎縣]][[壹岐市|壹岐市]]）
