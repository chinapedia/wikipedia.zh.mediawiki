'''天鵝座質量模擬器'''（[[英語|英語]]：Cygnus Mass Simulator）為[[軌道科學公司|軌道科學公司]]在2013年4月21日對[[安塔瑞斯火箭|安塔瑞斯火箭]]首次進行「A-ONE」試驗時所使用的[[樣板_(航太)|樣板]]（Boilerplate），它於[[美國|美國]][[維吉尼亞州|維吉尼亞州]][[瓦勒普斯島|瓦勒普斯島]][[中大西洋地區太空中心|中大西洋地區太空中心]][[中大西洋地區太空中心0號發射臺|0號發射臺]]升空<ref name = "軌道科學公司"/>。天鵝座質量模擬器其功能為模擬[[天鵝座宇宙飛船|天鵝座宇宙飛船]]作為載荷時的質量大小，最後這個虛擬的有效載荷被送至240公里至260公里高（150英哩至160英哩）的軌道上<ref name = "軌道科學公司"/>，並且以傾角51.6度環繞軌道<ref>{{Cite web |url= http://www.orbital.com/NewsInfo/release.asp?prid=852 |title= Orbital Successfully Launches First Antares Rocket |author= Barron Beneski |date= 2013年4月21日 |publisher= [[軌道科學公司|軌道科學公司]] |language= en |accessdate= 2013年4月23日 |quote=  }}</ref>。這次升空發射的物件還包括有4個[[微型衛星|微型衛星]]（Miniaturized satellite）[[立方衛星|立方衛星]]（CubeSat）<ref>{{Cite web |url= http://www.newspacewatch.com/articles/spaceflight-services-installs-four-nanosats-on-antares-rocket.html |title= Spaceflight Services installs four nanosats on Antares rocket |author= Clark Lindsey |date= 2013年3月21日 |publisher= NewSpace Watch |language= en |accessdate= 2013年4月23日 |quote=  }}</ref>，而隨著這次升空使得[[美國國家航空暨太空總署|美國國家航空暨太空總署]]根據[[商業軌道運輸服務|商業軌道運輸服務]]（Commercial Orbital Transportation Services）的合同必須支付研發金費<ref>{{Cite web |url= http://www.nasa.gov/centers/johnson/pdf/543091main_NNJ08TA32S-Amendment%207.pdf |title= Space Act Agreement Amendment Seven between NASA and Orbital Sciences Corporation for COTS |author= Chris Bergin |date= 2012年3月 |publisher= [[美國國家航空暨太空總署|美國國家航空暨太空總署]] |language= en |accessdate= 2013年4月23日 |quote=  }}</ref>。

== 載荷 ==
這次發射主要有效載荷為[[天鵝座宇宙飛船|天鵝座宇宙飛船]]的模擬器（CMS），其高度5,061毫米（199.25英吋）、直徑2,900毫米（114英吋）以及質量為3,800公斤（8,400磅）<ref name = "A-ONE Mission">{{Cite web |url= http://www.orbital.com/Antares-Cygnus/files/Pre-Launch-Guest-Briefing.pdf |title= Antares Test Launch “A-ONE Mission” Overview Briefing |author=  |date= 2013年4月17日 |publisher= [[軌道科學公司|軌道科學公司]] |language= en |accessdate= 2013年4月23日 |quote=  }}</ref>，其內配置有22個[[加速規|加速規]]、2個麥克風、12個數位溫度計、24個[[熱電偶|熱電偶]]和12個[[應變片|應變片]]<ref name = "A-ONE Mission"/>。

而次級有效載荷則為4個從天鵝座質量模擬器釋放的[[立方衛星|立方衛星]]（CubeSat）<ref name = "A-ONE Mission"/>，其中3個為美國國家航空暨太空總署[[艾姆斯研究中心|艾姆斯研究中心]]（Ames Research Center）推出[[手機星|手機星]]（PhoneSat）計畫之一<ref name = "A-ONE Mission"/>，並且分別以電話的發明者[[亞歷山大·格拉漢姆·貝爾|亞歷山大·格拉漢姆·貝爾]]命名為[[亞歷山大_(衛星)|亞歷山大]]（Alexander）、[[格拉漢姆_(衛星)|格拉漢姆]]（Graham）和[[貝爾_(衛星)|貝爾]]（Bell）<ref name = "A-ONE Mission"/>。這3枚衛星的發射目的為證明[[智慧型手機|智慧型手機]]得以在立方衛星上作為[[航空電子|航空電子]]使用<ref name = "A-ONE Mission"/>，而三者都有種1.124千克（2.48磅）的鋰電池以提供電力<ref name = "A-ONE Mission"/>。第四個衛星則是由私人公司所提供，其目的為利用地球磁場來對衛星狀態進行控制並開發地球成像之實驗<ref name = "A-ONE Mission"/><ref>{{Cite web |url= https://apps.fcc.gov/els/GetAtt.html?id=122025&x=. |title= Cosmogia Dove-1 Orbital Debris Assessment Report (ODAR) |author=  |date= 2012年6月1日 |publisher= Cosmogia |language= en |accessdate= 2013年4月23日 |quote=  }}</ref>。
