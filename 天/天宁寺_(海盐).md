'''海盐天宁寺'''为位于[[中国|中国]][[浙江省|浙江省]][[嘉兴市|嘉兴市]][[海盐县|海盐县]][[武原街道|武原街道]]海滨西路143号（原[[海盐城墙|海盐县城]]西门外三里）的一座佛教禅寺，原名禅悦院，北宋崇宁四年（1105）改今额<ref>清·王彬等修，徐用仪等纂，《光绪海盐县志·卷七·舆地考·寺观》：天宁永祚禅寺（图经）在县治西三里，基九十余亩，前临乌坵塘，水四面绕之。初名禅悦院，宋崇宁四年赐今额。淳熙己酉僧长溪创建佛殿、山门、宝塔，嘉定二年永模建圆通殿，元后至元三年僧梵琦起建镇海塔，至正二十四年梵琦重修之，三十五年正观建方丈。宣德间惟轼建僧堂。正统七年宗(王车王)建西方殿、山门，道彝、文胜继建钟楼、轮藏殿、廊庑。成化九年德杰重建方丈，二十年德珤重修殿塔。嘉靖三十年浩大修正殿，易木柱以石，三十九年守朴修宝塔，易塔中板台以石，置守塔僧舍。万历三十一年重修宝塔，三十七年思达修正殿、转藏殿、钟楼、山门、两廊及建准提阁，里人吕鸣治请大藏。（伊府志）国朝康熙八年重建钟楼，乾隆间僧源翰重建佛阁、宝塔。（新纂）道光十八年僧无碍重修宝塔，道可继之，重修大殿。粤匪之乱，山门及僧寮悉燬，仅存大殿、佛阁遭损。同治间僧玉峰重建禅堂、方丈及佛阁前左右廊庑。</ref>。寺院四面环水，今中轴线上依次为山门、大雄宝殿、千佛阁和镇海塔，其中千佛阁为明崇祯元年（1628）重建（其须弥座台基西侧束腰处存有明崇祯“本山重筑石台记”）、清代重修，镇海塔塔基为元代遗存（2006年在此基础上重建镇海塔<ref>{{cite web |url=http://zjrb.zjol.com.cn/html/2006-09/29/content_171840.htm|title=《海盐：元代镇海塔重现雄姿》|accessdate=2015-04-06|work=黄擎、汤燕萍|publisher=浙江日报}}</ref>），其它均为近年新建，山门面阔三间，单檐歇山顶，大雄宝殿、千佛阁面阔均为五间两层，重檐歇山顶，镇海塔为七层八面的楼阁式塔。

千佛阁通面宽27.5米，通进深17米，高22米，建筑面积777.5平方米，斗栱为五铺作双下昂，柱子设有[[侧脚|侧脚]]，其[[须弥座|须弥座]]与室内梁、枋、斗栱、雀替等处存有花卉、动物、建筑等图案的木雕、石雕300余幅<ref>{{cite web |url=http://culture.zjol.com.cn/05culture/system/2006/08/11/007802739.shtml|title=《千佛阁》|accessdate=2015-04-06|work=|publisher=浙江在线}}</ref>，为浙江省单层面积最大的木构古建筑之一，被建筑学家[[陈从周|陈从周]]誉为“浙江第一阁”<ref>{{cite web |url=http://www.cnjxol.com/xwzx/jxxw/qxxw/hy/content/2011-04/27/content_1673677.htm|title=《“浙江第一阁”迎来大整修》|accessdate=2015-04-06|work=迟伟涛|publisher=嘉兴在线}}</ref>。
