'''天爐座α'''（α Fornacis, α For）是[[天爐座|天爐座]]最亮的恆星，但只比4等星亮一點。中國傳統名'''天苑增三'''<ref>[[伊世同|伊世同]] 《中西对照恒星图表》</ref>，它在西洋的固有名稱是'''Dalim''' （出自[[朱塞普·皮亞齊|皮亞齊]]的星表''{{link-en|Palermo Catalogue|Palermo Catalogue}}''）<ref>{{cite book
  |editor= Piazzi, G
  |date = 1814
  |title = Praecipuarum Stellarum Inerrantium Positiones Mediae Ineunte Saeculo XIX: ex Observationibus Habitis in Specula Panormitana ab anno 1792 ad annum 1813
  |location = [[Palermo|Palermo]]
  |page = 21}}</ref><ref>{{cite book
  |author= Kunitzsch, P.
  |date = 1959
  |title = Arabische Sternnamen in Europa
  |publisher = [[Otto_Harrassowitz|Otto Harrassowitz]]
  |location = [[Wiesbaden|Wiesbaden]]
  |page = 155}}</ref><ref>{{cite book
  |author= Laffitte, R.
  |date = 2005
  |title = Héritages arabes: Des noms arabes pour les étoiles
  |edition = 2<sup>éme</sup> revue et corrigée
  |publisher = Librairie Orientaliste Paul Geunthner / Les Cahiers de l'Orient
  |location = [[Paris|Paris]]
  |page = 229}}</ref>和'''Fornacis'''（在{{link-en|Burritt|Elijah Hinsdale Burritt}}的''[[星表|星圖]]''）<ref>{{cite book
  |author= Burritt, E. H.
  |date = 1835
  |title = Atlas, Designed to Illustrate the Geography of the Heavens 
  |edition = new
  |publisher = F. J. Huntington
  |location = New York
  }} plate III.</ref>。這顆恆星最初的名稱是''' [[波江座|波江座]]12'''。

天爐座α的[[光譜類型|光譜類型]]是F8IV，其中的IV表示它的亮度是在演化上已經離開[[主序星|主序星]]的[[次巨星|次巨星]]<ref name=apj627_2/>。它的質量比太陽重約33%，年齡是29億歲<ref name=aaa373/><ref name=aaa418/>。天爐座α是一對[[聯星|聯星]] <ref name=aaa341/>，有著高[[自行|自行]]運動<ref name=SIMBAD/>。這個系統顯示出[[紅外過量|紅外過量]]，暗示可能有類似[[岩屑盤|岩屑盤]]的拱星盤<ref name=aaass96_3/>

第二顆恆星已經被確認是顆[[藍掉隊星|藍掉隊星]]，它可能是合併了之前存在的第三顆恆星，或是累積了它的物質。它是個強X射線源，質量約為太陽的78% <ref name=Fuhrmann2015/>。

這顆恆星的[[空間速度|空間速度]]元件是（U、 V、 W） = （–35，+ 20，+ 30）km/s<ref name=gliese/>。大約35萬年前，天爐座α星曾於[[A型主序星|A型主序星]]的[[時鐘座ν|時鐘座ν]]有過一次親密的接觸。估計曾互相接近至{{Convert|0.265|ly|pc|abbr=on}}的距離，使這兩顆恆星都有岩屑盤<ref name=deltorn_kalas01/>。
