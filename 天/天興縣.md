'''天興縣'''，[[臺灣明鄭時期|臺灣明鄭時期]]時的兩個縣級行政區之一，鄭經統治時將縣改制為[[散州|散州]]，臺灣為清朝統治後，原天興縣疆域被分成了[[臺灣府|臺灣府]]底下的[[諸羅縣|諸羅縣]]與[[臺灣縣_(1684年-1887年)|臺灣縣]]{{r|戴}}。另外臺南縣政府於1983年9月在其底下的佳里鎮佳里興（今[[佳里區|佳里區]][[佳里興|佳里興]]）立有「古天興縣治紀念碑」，但天興縣治是否在佳里興仍有爭論{{r|戴}}。

== 沿革 ==
1661年（[[明|明]][[永曆_(南明)|永曆]]15年）4月，[[南明|南明]][[延平郡王|延平郡王]][[鄭成功|鄭成功]]，力取[[荷蘭東印度公司|荷蘭東印度公司]]，攻打[[臺灣|臺灣]]，在攻下「[[普羅民遮城|普羅民遮城]]」（今日[[臺南市|臺南市]][[赤崁樓|赤崁樓]]）之後，將其改名為[[東都明京|東都明京]]，並設立[[承天府|承天府]]，下轄天興縣、[[萬年縣_(臺灣)|萬年縣]]二縣。天興縣縣治位於[[佳里興|佳里興]]（今臺南市佳里區），另一說位於大目降（今臺南市[[新化區|新化區]]）與新港（今[[新市區_(台灣)|新市區]]）之間，由[[鄭成功征臺部將列表|莊文烈]]任首任知縣<ref>丁萬復、林木榮，《臺西風情》，90年4月增版</ref>，此外尚有其他說法。

1664年，[[明鄭|明鄭]]繼承者[[鄭經|鄭經]]放棄[[福建|福建]]部分領土，將[[東都|東都]]改為[[東寧|東寧]]，承天府所屬兩縣升格為[[州|州]]，並增添[[澎湖|澎湖]]、南路與北路三個安撫司。天興縣因此而改稱'''天興州'''，並設有北路安撫司。

1683年，[[清朝|清朝]]將領[[施琅|施琅]]攻下臺灣，廢除明鄭時期的行政區編制。1684年，清廷設[[臺灣府|臺灣府]]，隸屬[[福建省|福建省]]，下轄[[諸羅縣|諸羅縣]]、[[臺灣縣_(1684年-1887年)|臺灣縣]]、[[鳳山縣_(臺灣)|鳳山縣]]三縣，天興縣疆域被分為[[諸羅縣|諸羅縣]]與[[臺灣縣_(1684年-1887年)|臺灣縣]]{{r|戴}}。

== 次級行政區劃 ==
天興縣底下設有十三個里，分布在今臺南市中南部地區，分別是[[武定里|武定里]]、[[永康里_(堡里)|永康里]]、[[長興里_(堡里)|長興里]]、[[保大里|保大里]]、[[歸仁里_(堡里)|歸仁里]]、[[永豐里_(堡里)|永豐里]]、[[新豐里_(堡里)|新豐里]]、[[廣儲里|廣儲里]]、[[感化里|感化里]]、[[新化里_(堡里)|新化里]]、[[永定里_(堡里)|永定里]]、[[善化里|善化里]]及[[開化里_(堡里)|開化里]]<ref>[http://thcts.ascc.net/themes/rb02.php 《明鄭時期臺灣行政疆域圖》，中央研究院台灣歷史文化地圖網站]</ref>{{r|戴}}。

其中善化里、新化里、永定里、開化里四里後來被納入清朝的諸羅縣轄區，餘九里納入臺灣縣轄區{{r|戴}}<ref>{{cite book|author=[[蔣毓英|蔣毓英]]、[[季麒光|季麒光]]、[[楊芳聲|楊芳聲]] |title=《臺灣府志·卷一 坊里》|year=1685年}}</ref>。

== 縣治 ==
關於天興縣的縣治所在地，雖然臺南縣政府已在佳里興立碑紀念，但學界尚未有定論，大致有五種說法{{r|戴}}：
*府治[[鎮北坊|鎮北坊]]：如[[連橫|連橫]]《[[臺灣通史|臺灣通史]]》（1921年）與陳世慶編纂的《臺灣省通志·卷三 政事志·建置篇》（1972年）即認為天興縣治在「鎮北坊米市」。
*原在府治東安坊，後遷新港社（今新市）：[[石萬壽|石萬壽]]〈臺灣府署建置始末及遺址碑自註〉（1983年）、〈二層行溪流域行政區劃的變遷〉（1988年）與《永康鄉志》（1988年）採此說，其來源應是綜合[[陳文達|陳文達]]《[[臺灣縣志|臺灣縣志]]》（1720年）寫說「臺灣府公署」為「偽天興州舊署」與〈永曆十八年臺灣軍備圖〉將天興縣註記在「新港半番民」聚落附近的記載{{r|戴}}。此外清朝的《臺灣府圖志》（1795年前）、《臺灣府方輿考證》（1836年）、《清一統志臺灣府（嘉慶重修一統志）》（1842年）均載明「天興廢州」在「新港」{{r|戴}}。
*大目降（今新化）與新港之間：如王世慶〈臺灣之名稱與行政區域之建置〉（1976年），此一說是根據〈永曆十八年臺灣軍備圖〉而來{{r|戴}}。
*大目降：如[[黃典權|黃典權]]〈鄭延平臺灣世業〉（1980年）。
*不確定是否為佳里興：盛清沂〈明鄭內政考略〉（1976年）寫「相傳天興設治於佳里興」，而葉振輝《臺灣開發史》（1995年，2003年再版）則寫說「天興縣治在開化里（今佳里），一說在大目降（今新化）」，亦未斷定佳里興為天興縣治{{r|戴}}。
