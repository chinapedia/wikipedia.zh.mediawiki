'''天馬航空'''（{{lang-ja|'''スカイマーク'''}}，{{lang-en|'''Skymark Airlines'''}}）是日本的一間[[低成本航空公司|低成本航空公司]]，總部設於[[東京|東京]]<ref>[http://www.skymark.co.jp/ja/company/ {{lang|ja|会社概要}}]{{Ja}}</ref>。其經營多條定期日本國內航線，亦有往[[韓國|韓國]][[首爾|首爾]]的國際[[包機|包機]]。天馬航空的[[樞紐機場|樞紐]]是[[東京國際機場|東京國際機場]]及[[神戶機場|神戶機場]]。

有別於一般認知的[[廉價航空|廉價航空]](或低成本航空公司)，除了可提上飛機的手提行李(1件，不超過10KG)，天馬航空在票價上已包含了3件合計不超過20KG的免費托運行李額度。

天馬航空計劃於2014年前將[[神戶機場|神戶機場]]發展為其第二個國內線樞紐，並打算開辦由神戶往[[札幌|札幌]]、[[仙台|仙台]]、[[松山市|松山]]、[[福岡|福岡]]、[[熊本|熊本]]及[[沖繩|沖繩]][[那霸|那霸]]的航線。<ref>[http://www.kobe-np.co.jp/news/shakai/0001679468.shtml {{lang|ja|スカイマーク、神戸発着路線を大幅拡充へ}}]{{Ja}}，[http://www.kobe-np.co.jp/ 神戶新聞]，2009年1月30日</ref>

2010年，天馬航空宣布向[[空中巴士|空中巴士]]（Airbus S.A.S.）订购四架空中巴士[[A380|A380]]型客机（后又追加2架），成為日本首家訂購A380型客機的航空公司，2014年4月8日，天马航空（Skymark）A380在法国[[图卢兹|图卢兹]]首飞，并在[[德国|德国]][[汉堡|汉堡]]進行机舱工程和装饰，并首次做出了開拓更遠航線（[[伦敦|伦敦]]、[[法兰克福|法兰克福]]、[[巴黎|巴黎]]和[[纽约|纽约]]）的計劃<ref name="plan">{{cite web |url=http://mshandong.com/air-press/2014/04/20/low-cost-carrier-lcc-in-japan/6121/ |title= 探索日本低成本航空公司 |work=[http://mshandong.com/ 山東週刊，SCHANTUNG] |date=2014年4月20日}}</ref>。2014年7月29日，空中巴士公司懷疑天馬航空無力支付A380訂單的價錢，宣佈取消訂單<ref name="A380cancel">{{cite web |url=http://www.airbus.com/newsevents/news-events-single/detail/airbus-terminates-purchase-order-with-skymark-airlines-for-a380s/ |title= Airbus terminates purchase order with Skymark Airlines for A380s |work=[http://www.airbus.com/ Airbus] |date=2014年7月29日}}</ref>。

2015年1月28日，天馬航空宣布申請破產重整，旗下A330客機從2015年2月1日起全部暫時停飛，並同時停飛航線與班機。而破產重整手續已經於2016年3月經由日本政府核可而結束<ref name="exbankrupt"/>，目前已經恢復以往的獲利狀態，將逐步恢復破產前停飛的航線與班機，擴大營運版圖。

== 歷史 ==
天馬航空於1996年成立，並於1998年9月19日開始運作。這時日本正撤銷航空管制，開放國內線的航權，天馬航空正是在這個背景下成立的第一家航空公司，其成立時便主打低廉票價，是日本最早以[[廉價航空公司|廉價航空]]概念經營的航空公司。公司主要的股東包括社長西久保慎一（35.47%）及{{link-ja|H.I.S.旅行社|エイチ・アイ・エス|H.I.S.旅行社}}（27.62%）。公司現時共有1269名員工。

天马航空社长西久保慎一曾表示，引进[[A330-300|A330-300]]将标志着天马航空进入一个新的阶段，这些飞机将主要用于日本的国内干线（東京至[[福岡|福岡]]、札幌[[新千歲機場|新千歲機場]]<ref name="plan"/>）和未来的国际航线。随着新飞机的引入，天马航空的女性空乘人員也全面换装，以宝蓝色的贴身制服搭配迷你裙、黄色丝巾，展现窈窕的身材曲线，为了强调年轻化，天马航空的空服人员平均年龄为20岁。

2015年1月28日，因已經出現現金短缺，該公司決定放棄自主重建，將根據日本《[[民事再生法|民事再生法]]》向[[東京地方法院|東京地方法院]]申請[[破產保護|破產保護]]<ref>{{cite news|url=http://www.appledaily.com.tw/realtimenews/article/new/20150128/550637/|title=日本第三大　天馬航空擬申請破產保護|publisher=蘋果日報|date=2015-01-29}}</ref>。

2016年3月，[[東京地方法院|東京地方法院]]允許天馬航空脫離[[破產保護|破產保護]]管制<ref name="exbankrupt">{{cite news|url=https://www.flightglobal.com/news/articles/skymark-airlines-emerges-from-bankruptcy-with-final-423582/ |title=Skymark Airlines emerges from bankruptcy with final repayment |date=2016年3月29日}}</ref>。
