'''天智天皇'''（{{jpn|j=天智天皇|hg=てんじてんのう|rm=Tenji Tennō}}；{{bd|626年||672年|1月7日|catIdx=T}}），[[日本|日本]]第38代[[天皇|天皇]]（668年[[正月初三|正月初三]]至671年[[十二月初三|十二月初三]]在位）。父[[舒明天皇|舒明天皇]]，母[[皇极天皇|皇极天皇]]（重祚为[[齐明天皇|齐明天皇]]）。諱'''葛城皇子'''（{{jpn|j=葛城皇子|hg=かづらきのおうじ|rm=Kadzuraki no ōji}}），又稱'''中大兄皇子'''（{{jpn|j=中大兄皇子|hg=なかのおおえのおうじ|rm=Nakanoooe no ōji}}）。

== 生平 ==
645年（[[大化|大化]]元年），与[[中臣鎌足|中臣鎌足]]计划讨平[[苏我氏|苏我氏]]。继而以孝德、齐明两天皇之皇太子身份（孝德為其堂兄、齐明為其生母），从事对[[大化革新|大化革新]]各项政策的拟定和实行。

其间，依[[百济|百济]]之请，派援军支援[[朝鲜|朝鲜]]，但在[[白江口之战|白江口之战]]中大败于[[唐朝|唐朝]]与[[新罗|新罗]]的联军。660年，[[齐明天皇|齐明天皇]]死后，从事内政的整顿，迁都于[[近江|近江]]的[[大津京|大津京]]。662年奉皇令與[[中臣鎌足|中臣鎌足]]制订《[[近江令|近江令]]》。668年即位。669年10月15日，派東宮皇太弟[[大海人皇子|大海人皇子]]至內大臣[[中臣鎌足|中臣鎌足]]府，赐姓藤原，次日[[藤原鎌足|藤原鎌足]]卒。670年2月制成《[[庚午年籍|庚午年籍]]》，是為日本最早的全國性[[戶籍|戶籍]]。陵墓在[[京都|京都]]名山科陵。

中大兄取得政權的過程得到其弟[[大海人皇子|大海人皇子]]的幫助甚大，也因此封[[大海人皇子|大海人皇子]]為皇太弟，但即位天皇之後仍難以避免政權鬥爭與猜忌，與[[大海人皇子|大海人皇子]]逐漸疏遠。

== 日本國誕生 ==
660年唐朝軍隊渡海與[[新羅|新羅]]軍隊聯合滅了[[百濟|百濟]]王國後，天智天皇為了倭國國家安全，派出倭國救援軍隊與被滅國的百濟在663年白村江之戰敗給唐帝國軍隊後，此時倭國與唐朝帝國為敵。天智天皇解散原本倭國與日本島內諸國聯盟重新建立覆蓋現今日本列島的統一國家，將倭國首都遷移到「大津」，制定成文法典「{{tsl|ja|近江令|近江律令}}」其中規定倭王對外稱號為「明神御宇日本天皇」，此為國號「日本」與「天皇」的明文起源，668年天智天皇於首都大津既位天皇，在唐帝國滅了[[高句麗|高句麗]]王國後，天智天皇為了求和派[[遣唐使|遣唐使]]到中國祝賀[[唐高宗|唐高宗]]皇帝，670年末，天智天皇派遣到新羅國的使者告知倭國國號改為[[日本國|日本國]]，此為日本國由來。<ref name = "日本史的誕生：東亞視野下的日本建國史，原文名稱：日本史の誕生: 千三百年前の外圧が日本を作った_2016"> {{ cite web | title = 日本史的誕生：東亞視野下的日本建國史，原文名稱：日本史の誕生: 千三百年前の外圧が日本を作った | url = https://www.books.com.tw/products/0010730076 | publisher = 八旗文化 | author = 岡田英弘，譯者：陳心慧 | isbn = 9789869356220 | language = zh-tw | date = 2016-10-05 }} </ref>
