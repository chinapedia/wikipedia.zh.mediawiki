'''天安門毛澤東像污損案'''是指在1989年5月23日下午2時，3位來自[[湖南省|湖南省]][[瀏陽市|瀏陽市]]的青年[[魯德成|魯德成]]、[[喻東嶽|喻東嶽]]和[[余志堅|余志堅]]針對[[北京市|北京市]][[天安門|天安門]]所懸掛巨幅[[毛澤東|毛澤東]]畫像所進行的毀損行動。三人最初為了聲援主張反對貪腐和要求中國共產黨改革的[[六四事件|六四事件]]抗議學生而在5月19日前往北京市，然而他們則主張完全推翻中國共產黨並且建立民主政府<ref name = "懷念喻東岳">{{Cite web |url= http://64memo.com/b5/15441.htm |title= 懷念喻東岳 |author= [[余志堅|余志堅]] |date= 2005-06-03 |publisher= 六四檔案 |language= zh-hant |accessdate= 2014-01-15 |quote=  }}</ref>。儘管余志堅和喻東嶽曾經向學生領導人提及建議和宣言但並沒有獲得答覆，之後認為無法和學生有進一步對話的三人於5月22日開始策劃針對毛澤東畫像的破壞行動<ref>{{Cite book | author = {{Link-en|鄭霭玲|Denise Chong}} | title = Egg on Mao: A Story of Love, Hope and Defiance | location = 加拿大多倫多 | publisher = {{Link-en|加拿大藍燈書屋|Random House of Canada}} | date = 2011-08-30 | pages = 第216頁 | ISBN = 978-0307355805 | accessdate = 2014-05-21 | url = http://books.google.com.tw/books?id=BpzFZXGNfR4C&printsec=frontcover&hl=zh-TW#v=onepage&q&f=false | language = en | quote =  }} </ref>。

第二天他們先是於附近購買了20顆雞蛋並且填滿油漆，之後就抵達畫像附近準備展開行動。三人先是於天安門城樓懸掛「五千年專制到此可以告一段落」和「個人崇拜從今可以休矣」標語，不久便將盛滿[[顏料|顏料]]的蛋殼扔向毛澤東畫像上。之後3人遭到由[[北京高校學生自治聯合會|北京高校學生自治聯合會]]大學學生組成的糾察隊抓住，並且送至帳篷中問話。最初三人認為可以藉此與學生領導人會面並且告訴他們其行動的象徵意義，但是參與學生則是擔心三人為中國政府為了故意抹黑抗議行動而派遣的間諜。最後決定在當天下午5時召開中外記者新聞發表會，會議上要求三人公開承認是個人作為而「與民主運動無關」。

隨後在當天晚上7時，北京高校學生自治聯合會將3人移交給北京市[[公安機關|公安機關]]處理；而民主運動人士也在播出的電視節目上譴責這一行為，並且多次強調示威學生與3名青年污損毛澤東像的作為並沒有關聯。到了晚上10時遭到污損的毛澤東畫像被撤下來，並且隨即換上了備用畫像。之後魯德成、喻東嶽和-{余志}-堅分別被[[人民法院_(中國)|人民法院]]判處16年[[有期徒刑|有期徒刑]]、20年有期徒刑和無期徒刑，其中喻東嶽更因為在監獄中遭遇虐待而罹患[[精神疾病|精神疾病]]<ref>{{Cite web |url= http://www.hkhkhk.com/main/messages/14329.html |title= 向毛像泼漆 六四民运人士获释 |author=  |date=  |publisher= [[中国人权民运信息中心|中国人权民运信息中心]] |language= zh-hans |accessdate= 2014-01-15 |quote=  }}</ref>。這次事件之後，包括[[周勇軍|周勇軍]]、[[封從德|封從德]]、[[王丹|王丹]]和[[吾爾開希|吾爾開希]]等學生領導人對於示威學生造成3人遭到公安機關逮捕並判刑感到抱歉，同時亦有以此次事件為主題的文學著作以及話劇表演出現。

== 事件 ==
=== 前往北京 ===
在出發前余志堅托人向學校表示因為母親生病而要請假回家照料，而魯德成也向公司謊稱要送妻子回娘家而請假離去；喻東嶽則是在向《瀏陽日報》報社社長留下便箋表示已經完成份內工作，因此希望能夠休假並且北上<ref name = "黃花崗雜誌"/>。5月18日上午，3人乘車經由湘潭來到了[[長沙市|長沙市]]準備搭乘當天晚上9時點的特快列車直接前往北京市，3人在空餘時間參加了長沙市聲援北京示威學生的遊行隊伍<ref>{{Cite web |url= http://www.rfa.org/english/china/china_maoprotester-20060222.html |title= China Releases Mao Portrait Protester After 17 Years |author=  |date= 2006-02-22 |publisher= [[自由亞洲電台|自由亞洲電台]] |language= en |accessdate= 2014-01-15 |quote=  }}</ref>。但由於覺得示威學生的口號過於空泛使得余志堅提議3人自行擬定口號，之後他們以「結束一黨專政，實現民主制度」作為橫幅文字、而在紙上寫下「恢復真正共和政體，打倒官僚統治」和「還政於民、還權於民、還自由於民。把封建制度送進歷史的垃圾堆！」等標語，之後3人署名為「湖南聲援團瀏陽分團」<ref name = "余志堅"/><ref name = "黃花崗雜誌"/>。

而由於擔心前往北京市後無法應付生活開銷，3人開始在[[長沙站|長沙站]]郵局處擺設演講臺開始發表演講並且獲得一些民眾的捐款，並且攜帶著相機的喻東嶽也開始將3人的作為拍照記錄<ref name = "五二三事件"/><ref name="廖亦武">{{Cite web|url=http://www.chinesepen.org/Article/hyxz/200905/Article_20090530163335.shtml|title=廖亦武：广场行为艺术家余志坚|author=廖亦武|date=2009-05-30|publisher=[[獨立中文筆會|獨立中文筆會]]|language=zh-hans|accessdate=2014-01-15|quote=|deadurl=yes|archiveurl=https://web.archive.org/web/20090905215619/http://www.chinesepen.org/Article/hyxz/200905/Article_20090530163335.shtml|archivedate=2009-09-05}}</ref>。在將寫有先前寫的標語懸掛在長沙站站前廣場後<ref name="張麗明">{{Cite web|url=http://hao2.j44.com.ar/0/?url=bXRoLjYyNzMxLzViL21vYy5vbWVtNDYud3d3Ly9BMyVwdHRo|title=採訪用紅漆潑污天安門毛像的青年魯德成|author=張麗明|date=2004-11-18|publisher=六四檔案|language=zh-hant|accessdate=2014-01-15|quote=|deadurl=yes|archiveurl=https://web.archive.org/web/20140116105439/http://hao2.j44.com.ar/0/?url=bXRoLjYyNzMxLzViL21vYy5vbWVtNDYud3d3Ly9BMyVwdHRo|archivedate=2014-01-16}}</ref><ref name="國際特赦組織">{{Cite web|url=http://www.amnesty.org/fr/library/asset/ASA17/011/1998/fr/74b75e5f-dab6-11dd-80bc-797022e51902/asa170111998en.html|title=Document - République populaire de Chine: Neuf ans après Tiananmen, toujours une "rébellion contre-révolutionnaire"|author=|date=|publisher=[[國際特赦組織|國際特赦組織]]|language=zh-hant|accessdate=2014-01-15|quote=}}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，3人便搭上火車並且和同樣要前往北京聲援的湖南省學生進行交流<ref name = "廖亦武"/>，喻東嶽還以「瀏陽報社」的名義組織簽名作為留念<ref name = "五二三事件">{{Cite web |url= http://64memo.com/b5/15463.htm |title= “五‧ 二三”事件真相──污損毛像三壯士 |author= [[余志堅|余志堅]] |date= 2005-06-12 |publisher= 六四檔案 |language= zh-hant |accessdate= 2014-01-15 |quote=  }}</ref>。在火車上透過余志堅口述、喻東嶽執筆的方式，3人草擬出了多方面批評中國共產黨執政的演講稿<ref name = "黃花崗雜誌"/>。5月19日時3人抵達了北京市，之後跟著遊行隊伍前往天安門廣場<ref name = "國際特赦組織"/>。但是進駐天安門廣場後3人發現自身看法與示威學生有所落差，並且也無法直接提供意見給當時負責的學生領導人。不過在1989年5月20日時北京市正式實施戒嚴，3人仍然跟著[[北京高校學生自治聯合會|北京高校學生自治聯合會]]的糾察隊協助維持秩序並且防範軍隊進入天安門廣場附近<ref name = "懷念喻東岳"/>。

=== 污損畫像 ===

5月22日10時因為不滿北京高校學生自治聯合會發表了《勝利大撤退》一文，支持繼續佔領天安門廣場的3人開始與一旁的學生辯駁；甚至喻東嶽執筆寫下了《何謂勝利？》和《民眾的民主之路在何方？》的廣播稿交給廣播站，但是廣播站卻並沒有回應<ref name = "Xiu Lu"/><ref name = "黃花崗雜誌"/><ref name = "真相與反思 (上)"/>。當天晚上余志堅表示希望能夠設法在天安門廣場上做出一些舉動，並且獲得喻東嶽和魯德成的支持。儘管一開始他們想到藉由天安門廣場上自焚的方式來呼籲學生不應該對於中國共產黨有所期待<ref name = "廖亦武"/><ref name = "懷念喻東岳"/>，但是擔心沒有事先的準備反而會引起他人的誤會，之後改決定將天安門城樓上的毛澤東畫像拆除<ref name = "真相與反思 (上)"/>。到了隔天早上，3人前往現場勘察發現將天安門城樓上懸掛的毛澤東主席巨幅畫像拆除並不可能<ref name = "廖亦武"/>，而以油漆將畫像刷除這個想法也因不實際而遭到否決<ref name = "余志堅專訪">{{Cite web |url= http://64memo.com/b5/12700_3.htm |title= 余志堅專訪(1) |author= [[自由亞洲電台|自由亞洲電台]] |date= 2004-06-01 |publisher= 六四檔案 |language= zh-hant |accessdate= 2014-01-15 |quote=  }}</ref>，最後3人決定以雞蛋蛋殼塗抹油畫顏料來達到同樣效果<ref name = "真相與反思 (上)"/><ref>{{Cite web |url= http://www.theglobeandmail.com/news/national/canadians-win-freedom-for-chinese-dissident/article706445/ |title= Canadians win freedom for Chinese dissident |author=  |date= 2009-03-17 |publisher= 《[[環球郵報|環球郵報]]》 |language= en |accessdate= 2014-01-15 |quote=  }}</ref>。

而在5月23日當天，數百萬名參與示威活動的學生則計畫發起抗議活動以表達對於北京市實施戒嚴持續3天的不滿<ref name="Mao’s Portrait Defaced">{{Cite web |url= http://www.alliance.org.hk/english/Tiananmen_files/890523.html |title= Mao’s Portrait Defaced |author=  |date=  |publisher= [[香港市民支援愛國民主運動聯合會|香港市民支援愛國民主運動聯合會]] |language= en |accessdate= 2014-01-15 |quote=  |deadurl= yes |archiveurl= https://web.archive.org/web/20140116132030/http://www.alliance.org.hk/english/Tiananmen_files/890523.html |archivedate= 2014-01-16 }}</ref>。到了中午12時3人買了雞蛋、油畫顏料、宣紙和墨汁等工具<ref name="盛雪">{{Cite web|url=http://www.chinesepen.org/Article/hyxz/200603/Article_20060319163743.shtml|title=盛雪:一步之遥——平安或苦难|author=盛雪|date=2006-03-19|publisher=[[獨立中文筆會|獨立中文筆會]]|language=zh-hans|accessdate=2014-01-15|quote=|deadurl=yes|archiveurl=https://web.archive.org/web/20140116101310/http://www.chinesepen.org/Article/hyxz/200603/Article_20060319163743.shtml|archivedate=2014-01-16}}</ref>，並且在天安門廣場西側的[[北京中山公園|北京中山公園]]內調配顏料並且裝入20顆蛋殼中<ref name = "蘋果日報"/>。3人接著給自己的家人寫下了遺書，之後由余志堅起草文字、喻東嶽在宣紙上寫下「五千年專制到此可以告一段落」和「個人崇拜從今可以休矣」的標語<ref name = "喻東嶽"/>。下午2時10分，3人先把兩張標語分別張貼在天安門城樓主城門門洞的兩側<ref name = "盛雪"/>，並且余志堅也開始將附近的民眾加以驅離<ref name = "五二三事件"/>。在2時20分左右，由22歲的喻東嶽先行把裝滿顏料的雞蛋蛋殼往天安門廣場上的巨幅[[毛澤東|毛澤東]]畫像丟擲後，之後26歲的魯德成隨後也跟進開始丟擲蛋殼<ref name = "五二三事件"/>，另外25歲的余志堅則和其他學生一同維持附近的秩序<ref name = "真相與反思 (上)"/><ref>{{Cite web |url= http://www.64tianwang.com/bencandy.php?fid=13&id=4505 |title= Calgary man is Tiananmen Square `hero' |author= {{Link-en|加通社|Postmedia News}} |date= 2009-06-07 |publisher= [[六四天網|六四天網]] |language= en |accessdate= 2014-01-15 |quote=  |archive-url= https://web.archive.org/web/20140116122548/http://www.64tianwang.com/bencandy.php?fid=13&id=4505 |archive-date= 2014-01-16 |dead-url= yes }}</ref>。之後3人也高聲呼喊「打倒官僚政府！倡導自由民主！」的口號，並且獲得附近一些群眾的支持。

然而當時在天安門廣場上的示威學生也懷疑[[公安機關|公安機關]]的便衣成員會企圖藉由事端抹黑學生抗議活動，並且為之後可能發生的鎮壓行動作為藉口<ref name = "Mao’s Portrait Defaced"/>。半小時候後隸屬於[[北京高校學生自治聯合會|北京高校學生自治聯合會]]的糾察隊成員前往現場盤查3人身分，之後喻東嶽和魯德成馬上被抓住並且先送到了[[人民英雄紀念碑|人民英雄紀念碑]]附近的糾察隊指揮部內。當時擔任學生糾察隊隊長的[[張健_(六四事件)|張健]]便詢問了兩人身分，之後張健告訴兩人之所以詢問行為的理由並且要求兩人應該負起責任<ref name = "真相與反思 (上)"/>。而另外一方面由於在過程中旁邊示威群眾的反對，使得余志堅反而是由[[北京工人自治聯合會|北京工人自治聯合會]]的糾察隊帶走並且與[[周勇軍|周勇軍]]會面，但是之後北京高校學生自治聯合會則要求將3人集合以澄清事件<ref name="周勇軍">{{Cite web |url= http://www.89-64.org/b5/13394.htm#top |title= 我與「湖南三壯士」事件 |author= [[周勇軍|周勇軍]] |date= 2004-08-04 |publisher= 六四檔案 |language= zh-hant |accessdate= 2014-01-15 |quote=  |archive-url= https://web.archive.org/web/20160305054452/http://www.89-64.org/b5/13394.htm#top |archive-date= 2016-03-05 |dead-url= yes }}</ref>。同時參與抗議活動的學生也很快在城樓下面掛起「這不是學生、人民幹的」的白布橫額，同時一些示威群眾的認為這一行為有辱國家和黨的象徵<ref name = "SFChoi8964">{{Cite web |url= http://sfchoi8964.wordpress.com/2009/05/05/523-%E9%A2%A8%E8%B5%B7%E9%9B%B2%E6%B9%A7-%E7%8B%BC%E4%BD%95%E6%99%82%E4%BE%86-%E6%AF%9B%E5%83%8F%E8%A2%AB%E6%BD%91/ |title= 風起雲湧 狼何時來 毛像被潑 |author= SFChoi8964 |date= 2009-05-05 |publisher= 一位香港記者的八九．六四 |language= zh-hant |accessdate= 2014-01-15 |quote=  }}</ref><ref name = "伍潔芳">{{Cite web |url= http://www.nytimes.com/1989/05/24/world/upheaval-in-china-chinese-take-umbrage-at-attack-on-mao-portrait.html |title= Upheaval in China; Chinese take Umbrage At Attack on Mao Portrait |author= [[伍潔芳|伍潔芳]] |date= 2009-05-21 |publisher= 《[[紐約時報|紐約時報]]》 |language= en |accessdate= 2014-01-15 |quote=  }}</ref>；而由於北京市天氣在之後隨即轉陰也使得許多民眾以為是因為汙損毛澤東畫像所造成的<ref name = "SFChoi8964"/>，對此魯德成之後表示：「這些都是中共專製文化下折射的東西。當時正值春夏之交，北京下雨的天氣，卻被民間神話了。<ref name = "蘋果日報"/>」

=== 移交公安 ===
[[File:Defacement_of_Mao_Zedong's_Portrait_on_the_Tiananmen,_1989.jpg|thumb]]

學生之後將3人帶到停靠在[[金水橋|金水橋]]地麵包車隔離看管，期間則有《中國農民報》的記者對他們進行採訪<ref name = "周勇軍"/>。而參與抗議活動的學生們後續仍針對3人的行為和背景質詢了2個小時，但儘管3人在在拿出證件表明身分後仍然遭到許多學生的質疑<ref name="封從德">{{Cite web |url= http://www.89-64.org/b5/14585.htm#Head1 |title= 天安門三壯士的歷史教訓──我見證的砸毛像事件 |author= [[封從德|封從德]] |date= 2005-03-23 |publisher= 六四檔案 |language= zh-hant |accessdate= 2014-01-15 |quote=  |archive-url= https://web.archive.org/web/20160305064604/http://www.89-64.org/b5/14585.htm#Head1 |archive-date= 2016-03-05 |dead-url= yes }}</ref>。而無法信任3人的學生要求喻東嶽、余志堅和魯德成提出字條並且按壓手印，以聲明3人污損畫像的舉動與學生發起的示威活動並無關連。當天下午5時，學生進一步要求喻東嶽、余志堅和魯德成在[[中國國家博物館|中國國家博物館]]前出席中外記者新聞發布會，公開承認是其個人所為而且整個污損畫像的行動與學生發起的「民主運動無關」<ref name = "廖亦武"/>，不過3人也藉由記者會發表其政治主張<ref name = "周勇軍"/><ref name = "盛雪"/>。而在記者會結束後3人則接受[[中國中央電視台|中國中央電視台]]記者[[陳鐸|陳鐸]]的專訪，並且這段畫面則在當天晚上的《[[新聞聯播|新聞聯播]]》上播出<ref name = "余志堅專訪"/>。同時在[[中華人民共和國國家安全部|中華人民共和國國家安全部]]人員的要求下，當時留守天安門廣場的部分學生糾察隊成員被迫討論喻東嶽、余志堅和魯德成的去留<ref name = "余志堅專訪"/><ref name = "張麗明"/>。

其中一部分學生認為3人是受到中國政府的指使所派遣而來<ref name = "封從德"/>，另外一部分學生則認為這是自發行動，最後學生代表經投票表決後決定將3人先行交到[[公安機關|公安機關]]處理<ref name = "張麗明"/>。到了當天晚上6時，北京高校學生自治聯合會的糾察隊告知3人將把其移交北京市東城分局的天安門管理處。之後由代表北京高校學生自治聯合會的[[郭海峰|郭海峰]]負責這次移交過程，其中郭海峰向3人索取了名片並且要求簽名，3人則把在長沙站募款而來的剩餘資金遞交給他<ref name = "五二三事件"/>。而到了管理處時郭海峰要求負責人提供一份有著親筆簽名的收條，並且表示：「你有義務向我們通報這三人的下落，這件事我們以後一定還要過問的！<ref name = "盛雪"/>」而在當天晚間晚播出的電視節目中，民主運動人士則聲明示威活動與3人毀損畫像的行為無關，並且亦對喻東嶽、余志堅和魯德成公開指責。而遭到汙損的毛澤東畫像在3時50分時，由天安門廣場上管理處便利用雲梯先以綠色軍毯覆蓋畫像<ref name = "伍潔芳"/><ref>{{Cite web |url= http://renminbao.com/rmb/articles/2007/5/12/44277.html |title= 中共大凶兆！天安门老毛画像被烧毁 |author= 辛馨 |date= 2007-05-12 |publisher= 《[[人民报_(美国)|人民報]]》 |language= zh-hant |accessdate= 2014-01-15 |quote=  }}</ref>。到了當天晚上10時25分，管理處在五建公司以及運輸公司的協助下把被污損的毛澤東畫像取下來，並且隨即換上了備用的毛澤東畫像<ref>{{Cite web |url= https://chinadigitaltimes.net/chinese/2013/06/%E9%99%88%E5%B0%8F%E9%9B%85%EF%BC%9A%E6%88%91%E6%89%80%E7%9B%AE%E7%9D%B9%E7%9A%84%E8%9B%8B%E5%87%BB%E6%AF%9B%E5%83%8F%E6%83%85%E6%99%AF/ |title= 陈小雅：我所目睹的“蛋击毛像”情景 |author= 陈小雅 |date= 2013-06-03 |publisher= [[中國數字時代|中國數字時代]] |language= zh-hans |accessdate= 2014-01-15 |quote=  }}</ref>。

== 後續 ==
=== 秘密審判 ===
喻東嶽、余志堅和魯德成在東城派出出所關一天晚上後先被移往東城區看守所，之後移送前往北京市第一看守所<ref name = "周勇軍"/>。到了6月15日時，中國公安機關針對3人的作為進行了正式的逮捕行動。而遭到逮捕的3人隨後在1989年7月11日時，於[[北京市人民法院列表|北京市中級人民法院]]的地下室開始就相關行動進行秘密審判。其中先前喻東嶽所拍攝有關在長沙市演講乃至於製作海報的照片便被檢察機關視為犯罪證物<ref name = "五二三事件"/>，而為了表示有法治基礎中國政府還以100元人民幣幫助3人聘請律師<ref name = "Xiu Lu"/>。在審判過程中喻東嶽還堅持3人毀損毛澤東畫像的行為實際上是一種行動藝術<ref name = "廖亦武"/>，不過整個審判過程大約經過2個小時便宣告結束<ref>{{Cite web |url= http://www2.macleans.ca/2009/10/01/the-men-who-threw-egg-on-mao/ |title= The men who threw egg on Mao |author= {{Link-en|鄭霭玲|Denise Chong}} |date= 2009-10-01 |publisher= 《[[麥克林|麥克林]]》 |language= en |accessdate= 2014-01-15 |quote=  }}</ref>。之後北京市中級人民法院宣布延期宣判，一直到同年8月11日時才正式提交判決書<ref name = "喻東嶽"/><ref name = "魯德成"/>，其中魯德成、喻東嶽和余志堅分別被人民法院以「反革命破壞」和「反革命煽動」等罪名判處16年有期徒刑、20年有期徒刑和無期徒刑<ref name = "美國之音 1">{{Cite web |url= http://www.voachinese.com/content/a-21-w2006-02-22-voa16-58206517/1069435.html |title= 天安門運動一主要人物被釋放出獄 |author=  |date= 2006-02-21 |publisher= [[美國之音|美國之音]] |language= zh-hant |accessdate= 2014-01-15 |quote=  }}</ref><ref>{{Cite web |url= http://www.asianews.it/news-en/Man-who-defaced-Mao's-portrait-freed-after-17-years-5472.html |title= Man who defaced Mao's portrait freed after 17 years |author=  |date= 2006-02-23 |publisher= {{Link-en|亞洲新聞|AsiaNews}} |language= en |accessdate= 2014-01-15 |quote=  }}</ref>。

1"/><ref>{{Cite web |url= http://www.ifex.org/china/2006/02/23/journalist_who_was_driven_mad_in/ |title= Journalist who was driven mad in prison freed after 16 years; a second journalist re-arrested after taking part in hunger strike |author= [[無國界記者|無國界記者]] |date= 2006-02-23 |publisher= {{Link-en|IFEX|IFEX (organization)}} |language= en |accessdate= 2014-01-15 |quote=  }}</ref>；在2009年時他經由泰國抵達美國，並且獲得[[庇護權|政治庇護]]<ref name = "海濤">{{Cite web |url= http://www.voachinese.com/content/portait-green-paint-splatters/1723160.html |title= 同是巨像遭潑墨，潑者命運大不同 |author= 海濤 |date= 2013-08-04 |publisher= [[美國之音|美國之音]] |language= zh-hant |accessdate= 2014-01-15 |quote=  }}</ref>。

不過被判處20年有期徒刑的喻東嶽則在1992年時，因為長期的單獨禁閉和虐待而於監獄中開始罹患[[精神分裂症|精神分裂症]]<ref>{{Cite web |url= http://news.bbc.co.uk/chinese/simp/hi/newsid_4900000/newsid_4902000/4902046.stm |title= 中国民运人士鲁德成抵达加拿大 |author=  |date= 2006-04-12 |publisher= [[BBC中文網|BBC中文網]] |language= zh-hans |accessdate= 2014-01-15 |quote=  }}</ref><ref>{{Cite web |url= http://www.open.com.hk/old_version/2004_11p44.html |title= 中國關押作家世界之冠 |author= 趙達功和張裕 |date=  |publisher= 《[[開放雜誌|開放雜誌]]》 |language= zh-hant |accessdate= 2014-01-15 |quote=  }}</ref>。並且他一直到2006年2月時才得以減刑出獄，之後在2009年時抵達美國並且獲得政治庇護<ref name = "海濤"/>。期間包括[[國際特赦組織|國際特赦組織]]和[[無國界記者|無國界記者]]等團體都針對喻東嶽在監獄服16年徒刑後罹患精神分裂症一事發表前譴責<ref>{{Cite web |url= http://www.voachinese.com/content/a-21-2006-02-24-voa50-58871207/1098953.html |title= 人權組織就喻東嶽案提出譴責 |author=  |date= 2006-02-23 |publisher= [[美國之音|美國之音]] |language= zh-hant |accessdate= 2014-01-15 |quote=  }}</ref>，而[[對話基金會|對話基金會]]主席{{Link-en|肯約翰|John Kamm}}則認為喻東嶽「是最後一位獲釋的天安門事件重要人物」。同樣在六四事件期間擔任重要角色的[[劉曉波|劉曉波]]對此則表示：「對於一個青年人，因為往一個畫像上丟兩個臭雞蛋什麼的，居然就能夠被判20年，而且聽他家人説他現在已經精神失常了，連他的母親，接他的弟弟都不認識了。我想這在某種程度上代表了整個六四事件，這個喻東嶽釋放在某種程度上也是一個具有某種象徵意義性的事件。<ref>{{Cite web |url= http://www.voachinese.com/content/a-21-w2006-02-23-voa42-63123077/1053408.html |title= 六四蛋擲毛像者獲釋但精神失常 |author=  |date= 2006-02-22 |publisher= [[美國之音|美國之音]] |language= zh-hant |accessdate= 2014-01-15 |quote=  }}</ref>」

=== 影響和評價 ===
早在1992年時，同樣出身於湖南省的學生領導人[[唐柏橋|唐柏橋]]便在為[[人權觀察|人權觀察]]所撰寫的《湖南人權報告》中提到喻東嶽、余志堅和魯德成的情況<ref>{{Cite web |url= http://www.nytimes.com/1992/06/01/world/china-is-accused-of-torturing-3-who-defaced-mao-portrait.html |title= China Is Accused of Torturing 3 Who Defaced Mao Portrait |author= [[紀思道|紀思道]] |date= 1992-06-01 |publisher= 《[[紐約時報|紐約時報]]》 |language= en |accessdate= 2014-01-15 |quote=  }}</ref>。但是儘管唐柏橋數度發起連署活動譴責中國政府對於3人的作為，但是長期以來包括學生領導人以及社會輿論並沒有給這次事件有高度關注<ref name = "周勇軍"/><ref name="曾慧燕">{{Cite web|url=http://www.chinesepen.org/Article/sxsy/200606/Article_20060606164915.shtml|title=天安门三壮士悲惨命运始末----一个六四事件至今还未结束的悲剧|author=曾慧燕|date=2006-06-06|publisher=[[獨立中文筆會|獨立中文筆會]]|language=zh-hans|accessdate=2014-01-15|quote=|deadurl=yes|archiveurl=https://web.archive.org/web/20090925045823/http://www.chinesepen.org/Article/sxsy/200606/Article_20060606164915.shtml|archivedate=2009-09-25}}</ref>。到了2002年6月時，湖南省的自由撰稿人[[陳少文|陳少文]]在紐約的歷史文化刊物《[[黃花崗雜誌|黃花崗雜誌]]》上發表了《八九真英雄 湖南三壯士》一文，內容介紹了整起事件的經過並且對於其作為給予了高度評價<ref name = "曾慧燕"/>。[[周勇軍|周勇軍]]在看完這篇文章後，在2004年8月4日也發表了長篇調查報導《我與「湖南三壯士」事件》作為紀念，並且開始呼籲社會大眾以及其他曾經參與學生運動的人士關注此事<ref name = "周勇軍"/>。另外華裔加拿大作家{{Link-en|鄭霭玲|Denise Chong}}則是在2009年時，與{{Link-en|加拿大藍燈書屋|Random House of Canada}}合作出版了以這次事件為主題的著作《{{Link-en|毛像上的雞蛋：一位污損神像、揭露專制的普通人的故事|Egg on Mao: The Story of an Ordinary Man Who Defaced an Icon and Unmasked a Dictatorship}}》<ref>{{Cite web |url= http://news.singtao.ca/toronto/content_column_tor.php?IndexID=1815496 |title= 華裔作家據魯德成經歷撰寫.《毛像上的雞蛋》9月面世 |author=  |date= 2009-06-02 |publisher= 《[[星島日報|星島日報]]》 |language= zh-hant |accessdate= 2014-01-15 |quote=  }}</ref><ref>{{Cite web |url= http://www.magazine.utoronto.ca/feature/denise-chong-book-chinese-political-prisoner-lu-decheng/ |title= Act of Defiance |author= John Allemang |date= 2009年 |publisher= [[多倫多大學|多倫多大學]] |language= en |accessdate= 2014-01-15 |quote=  }}</ref>。而魁北克劇作家{{Link-en|卡罗拉·弗雷谢特|Carole Fréchette}}在報紙上得知有關喻東嶽出獄消息後也開始調查相關資料，並且在2012年時以此為題材創作了話劇《想到喻》（Thinking of Yu），內容講述了喻東嶽的故事以及不同人對於此次事件的看法<ref>{{Cite web |url= http://www.rcinet.ca/patrimoineasiatique/zh/%E4%B8%AD%E6%96%87-%E4%B8%BA%E4%BB%80%E4%B9%88%E5%86%99%E5%96%BB%E4%B8%9C%E5%B2%B3%EF%BC%9A%E9%87%87%E8%AE%BF%E8%AF%9D%E5%89%A7%E3%80%8A%E6%83%B3%E5%88%B0%E5%96%BB%E3%80%8B%E4%BD%9C%E8%80%85/ |title= 为什么写喻东岳：采访话剧《想到喻》作者弗雷谢特 |author=  |date=  |publisher= [[加拿大國際廣播電台|加拿大國際廣播電台]] |language= zh-hans |accessdate= 2014-01-15 |quote=  }}</ref>。

到了今日，許多參與抗議行動的學生領導人都對於造成喻東嶽、余志堅和魯德成遭到公安機關逮捕並且判刑一事感到抱歉<ref name = "盛雪"/><ref>{{Cite web |url= http://64memo.com/b5/13725.htm |title= 天安門廣場上的「極少數人」──讀余志堅、魯德成《關於對學生領袖態度的聲明》 |author= [[任不寐|任不寐]] |date= 2004-11-18 |publisher= 六四檔案 |language= zh-hant |accessdate= 2014-01-15 |quote=  }}</ref><ref>{{Cite web |url= http://www.rfa.org/english/news/china/Mao-05192009113730.html |title= Mao Portrait Protesters Get Asylum |author=  |date= 2006-06-06 |publisher= [[自由亞洲電台|自由亞洲電台]] |language= en |accessdate= 2014-01-15 |quote=  }}</ref>，同時一些評論也將這次汙損畫像事件和[[民主女神|民主女神]]以及[[王維林|王維林]]阻擋坦克列為六四事件期間的象徵性作為<ref name = "余志堅專訪"/>。其中獨立撰稿人[[余杰|余杰]]便表示：「湖南三君子史無前例地挑戰神壇上的毛澤東，堪稱反獨裁、反暴君的偉大壯舉。他們的所作所為決非一時興起，而是經過深思熟慮……他們為此付出了慘痛的代價……<ref>{{Cite web |url= http://www.weijingsheng.org/report/report2005/report2005-12/WJSF2ndPrize051205dedicationA171-W81.htm |title= 生存与牺牲 -- 第二届“魏京生中国民主斗士奖”对喻东岳、吴义龙、陈西的颁奖词 |author=  |date= 2005-12-05 |publisher= 魏京生基金會 |language= zh-hans |accessdate= 2014-01-15 |quote=  }}</ref>」在2005年3月時，[[封從德|封從德]]發表了《天安門三壯士的歷史教訓──我見證的砸毛像事件》一文並且認為當時由於天安門廣場上並沒有明確領導組織而造成這次事件發生<ref name = "封從德"/>。在2006年4月25日時，[[周鋒鎖|周鋒鎖]]等曾經參與示威活動的人士便趁著魯德成前往美國時於招待會上公開表示歉意<ref name = "曾慧燕"/>；而同年5月4日時，包括[[王丹|王丹]]、[[吾爾開希|吾爾開希]]、[[王超華|王超華]]和[[蔣品超|蔣品超]]等人在慶祝魯德成獲得釋放的連署聲明中發表道歉聲明，表示對於當時學生決定將3人交給公安機關處理而導致3人遭到監禁感到遺憾<ref name = "曾慧燕"/><ref>{{Cite web |url= http://www.rfa.org/english/news/in_depth/china_maoprotester-20060518.html |title= Tiananmen Veteran Voices Remorse Towards Mao Portrait Protesters |author=  |date= 2006-05-18 |publisher= [[自由亞洲電台|自由亞洲電台]] |language= en |accessdate= 2014-01-15 |quote=  }}</ref>。不過外地高校學生聯合會總指揮[[連勝德|連勝德]]則表示雖然對於3人之後的結果感到難過，但是並不認為將3人移交給公安機關是錯誤的做法<ref name = "曾慧燕"/>。而[[鄭義_(作家)|鄭義]]則在文章中提到他並無法認定將3人移交給公安機關是否是正確決定，但是他仍然對於3人在監獄中的待遇感到十分難過<ref>{{Cite web |url= http://www.peacehall.com/news/gb/pubvp/2003/06/200306040601.shtml |title= 郑义：未完成的忏悔 |author= [[鄭義_(作家)|鄭義]] |date= 2003-06-04 |publisher= [[博訊|博訊]] |language= zh-hans |accessdate= 2014-01-15 |quote=  }}</ref>。

=== 類似行動 ===
自從1989年喻東嶽、余志堅和魯德成三人向毛澤東畫像丟擲雞蛋以後，中國還至少發生過三次與天安門懸掛的毛澤東畫像有關的破壞案件。其中第一次發生在2007年5月，根據官方媒體[[新華社|新華社]]報導指出一名來自[[烏魯木齊市|烏魯木齊市]]的無業男子向畫像投擲燃燒物而造成局部損壞；隨後天安門廣場便遭到封鎖並且因為這次事件而提高戒備層級，而根據報導指出該人遭到警方拘留但是當前下落不明<ref>{{Cite web |url= http://www.reuters.com/article/2007/05/12/us-china-mao-vandal-idUSPEK27297620070512 |title= Vandal damages China's iconic Mao portrait |author=  |date= 2007-05-12 |publisher= [[路透社|路透社]] |language= en |accessdate= 2014-01-15 |quote=  }}</ref>。第二次破壞事件則是發生在2010年4月5日下午，一名從黑龍江省前往北京市的抗議人士向畫像潑灑墨水，以希望各界能夠重視人權問題，後來他則被北京市警方拘留<ref>{{Cite web |url= http://www.telegraph.co.uk/news/worldnews/asia/china/7565768/Chinese-protestor-throws-ink-at-portrait-of-Chairman-Mao.html |title= Chinese protestor throws ink at portrait of Chairman Mao |author= {{Link-en|彼得·福斯特 (記者)|Peter Foster (journalist)|彼得·福斯特}} |date= 2010-04-08 |publisher= 《[[每日電訊報|每日電訊報]]》 |language= en |accessdate= 2014-01-15 |quote=  }}</ref>。

第三次事件則是發生在2014年3月5日時，一名年約30歲的抗議人士[[孫兵|孫兵]]前往天安門廣場，希望籍由[[全國人民代表大會|全國人民代表大會]]召開期間向政府表達不滿。之後他向毛澤東畫像左下角潑灑油墨，該名抗議人士很快被警方拘留，当局亦隨即建立封鎖線包圍現場。由於孫兵過去便曾因為故意犯罪而被判處有期徒刑，其本次犯罪属于刑满释放5年內的累犯，依中国法律从重处罚。最終北京市東城區人民法院作出一審判決，以尋釁滋事罪判處孫兵有期徒刑1年2個月。<ref>{{Cite web |url= http://en.rsf.org/china-website-editor-three-citizens-15-03-2014,46008.html?utm_source=ReportersWithoutBorders&utm_medium=twitter |title= WEBSITE EDITOR, THREE CITIZEN JOURNALISTS ARRESTED FOR COVERING PROTESTS |author=  |date= 2014-03-15 |publisher= [[無國界記者|無國界記者]] |language= en |accessdate= 2014-05-21 |quote=  }}</ref><ref>{{Cite web |url= http://cpj.org/2014/03/three-journalists-detained-after-reporting-on-tian.php |title= Three journalists detained after reporting on Tiananmen |author=  |date= 2014-03-18 |publisher= [[保護記者委員會|保護記者委員會]] |language= en |accessdate= 2014-05-21 |quote=  }}</ref>
