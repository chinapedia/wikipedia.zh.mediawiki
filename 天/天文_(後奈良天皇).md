'''天文'''是[[日本|日本]]的[[年號|年號]]之一。在[[享祿|享祿]]之後，[[弘治_(日本)|弘治]]之前。指1532年到1554年的期間。這個時代的[[天皇|天皇]]是[[後奈良天皇|後奈良天皇]]。[[室町幕府|室町幕府]]的將軍是[[足利義晴|足利義晴]]、[[足利義輝|足利義輝]]。

== 改元 ==
* [[享祿|享祿]]5年7月29日（[[陽曆|陽曆]]1532年8月29日） 因室町幕府12代將軍[[足利義晴|足利義晴]]之申請、戰亂等災異而改元
* 天文24年10月23日（陽曆1555年11月7日） 改元為弘治

== 出典 ==
出自『[[易經|易經]]』的「仰以觀於'''天文'''、俯以察於地理」。

== 天文年間要事 ==
* 元年[[八月廿四|8月24日]]（1532年） - [[山科本願寺|山科本願寺]]遭火攻。京都町眾的法華一揆暴亂，宗徒六角氏進行襲撃。
* 5年[[七月廿七|7月27日]]（1536年） - [[天文法華之亂|天文法華之亂]]。與法華宗對立的延曆寺眾徒擊破京都的法華一揆暴亂，焚燬21所寺院。
* 10年[[六月十四|6月14日]]（1541年） - 武田晴信（後來的[[武田信玄|信玄]]）將父親[[武田信虎|信虎]]放逐出[[甲斐|甲斐]]。
* 12年[[八月廿五|8月25日]]（1543年） - 葡萄牙船漂流到[[種子島|種子島]]，[[鐵砲|鐵砲]]（火繩槍）傳入日本（有一說認為是1542年）
* 15年[[十二月二十|12月20日]]（1546年） - [[足利義輝|足利義輝]]就任為[[室町幕府|室町幕府]]13代將軍。
* 17年[[十二月三十|12月30日]]（1548年） - 長尾景虎（後來的[[上杉謙信|上杉謙信]]）代替兄長[[長尾晴景|長尾晴景]]繼承家督，進入[[越後|越後]]・[[春日山城|春日山城]]。
* 18年[[二月廿四|2月24日]]（1549年） - [[齋藤歸蝶|濃姬]]嫁給[[織田信長|織田信長]]。
* 18年[[七月初三|7月3日]]（1549年） - [[方濟各·沙勿略|方濟各·沙勿略]]所乘的船到達鹿兒島。[[7月22日_(陰曆)|7月22日]]登陸。
* 18年[[十一月廿七|11月27日]]（1549年） - 三河的[[松平氏|松平氏]]領地被納入[[今川義元|今川義元]]支配下，松平竹千代（後來的[[德川家康|德川家康]]）成為今川之人質前往[[駿府|駿府]]居住。
