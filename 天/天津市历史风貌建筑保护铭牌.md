[[File:天津中法工商银行大楼局部2.jpg|thumb]]的天津市历史风貌建筑保护铭牌]]
'''天津市历史风貌建筑保护铭牌'''是依据地方性法规《天津市历史风貌建筑保护条例》，经[[天津市国土资源和房屋管理局|天津市国土资源和房屋管理局]]提名，天津市历史风貌建筑保护专家咨询委员会评审，最终由[[天津市人民政府|天津市人民政府]]批准，对被正式确认的[[天津历史风貌建筑|天津历史风貌建筑]]由[[天津市人民政府|天津市人民政府]]统一设立的保护标志。目前，[[天津市|天津市]]已有的746幢<ref>[http://news.163.com/10/0831/11/6FDO1PK500014J7L.html 天津市召开《历史风貌建筑保护条例》实施五周年总结表彰会]</ref>历史风貌建筑被授予天津市历史风貌建筑保护铭牌。<ref>[http://www.tjzz.com.cn/fccs.asp?id=239 天津市历史风貌建筑实施挂牌保护]{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

== 颁发 ==
[[File:耀华中学礼堂铭牌.jpg|thumb]]的天津市首块历史风貌建筑保护铭牌]]
2006年8月8日，[[天津市|天津市]]举行了“历史风貌建筑保护铭牌揭幕仪式”，揭幕仪式由[[天津市国土资源和房屋管理局|天津市国土资源和房屋管理局]]副局长路红主持，作家[[航鹰|航鹰]]与[[耀华中学|耀华中学]]的学生代表为[[耀华中学|耀华中学]]揭幕天津市首块历史风貌建筑保护铭牌。作为首个被挂牌的历史风貌建筑，耀华中学的历史风貌建筑保护铭牌名称为[[耀华学校礼堂|原天津公学礼堂]]，保护等级为特殊保护，编号为0110038。<ref>[http://news.enorth.com.cn/system/2006/08/08/001378902.shtml 天津8日举行历史风貌建筑保护铭牌揭幕仪式]</ref>

== 样式 ==
天津市历史风貌建筑保护铭牌由黑色[[大理石|大理石]]材料制成，铭牌上的内容样式共十一行：首行和第二行为“天津市历史风貌建筑”的中英文；第三行和第四行为建筑名称的中英文；第五行和第六行为保护等级的中英文、第七行和第八行为编号的中英文、第九行和第十行为颁发机关天津市人民政府的中英文。末行为公布时间。在第五行和第八行之间的右侧区域绘有该历史风貌建筑所在的地图。另外，一些重点建筑还附有历史风貌建筑简介铭牌，该铭牌也由黑色[[大理石|大理石]]材料制成，在内容上对该建筑进行中英文介绍。<ref>{{Cite web |url=http://www.tjwh.gov.cn/wwbl/wbjw/0608/06-08-17-wbls.html |title=五百历史风貌建筑挂牌 |access-date=2010-12-24 |archive-url=https://web.archive.org/web/20140202180203/http://www.tjwh.gov.cn/wwbl/wbjw/0608/06-08-17-wbls.html |archive-date=2014-02-02 |dead-url=yes }}</ref>2013年9月，天津市历史风貌建筑首批全新保护铭牌陆续亮相，新铭牌在原有的基础上增加[[二维码|二维码]]信息，天津市民或者游客可以通过手机扫描二维码就能了解每幢天津历史风貌建筑相关的历史资料，获取更多关于该建筑的信息。<ref>[http://www.fmjz.cn/Lists/List/DispForm.aspx?ID=209 天津历史风貌建筑首批全新保护名牌亮相] {{webarchive|url=https://web.archive.org/web/20140203220718/http://www.fmjz.cn/Lists/List/DispForm.aspx?ID=209 |date=2014-02-03 }}</ref>
同时，天津市历史风貌建筑办公室将进一步完善后台网络资源平台，按重点区域和保护等级的先后顺序，逐批进行更换铭牌。每幢天津历史风貌建筑将配有文字信息、历史图片、楼层结构、平面图、楼内场景，甚至每个屋子历史上的用途都能得到体现。对于特殊保护建筑和重点保护建筑，还将链接“内部三维展示平台”。<ref>[http://news.enorth.com.cn/system/2013/08/14/011224756.shtml 天津市历史风貌建筑名牌增二维码 首批下月亮相]</ref>

== 相关规定 ==
{{wikisource|天津市历史风貌建筑保护条例}}
根据《天津市历史风貌建筑保护条例》规定：“天津市历史风貌建筑保护铭牌由[[天津市人民政府|天津市人民政府]]统一设立，任何单位和个人不得擅自设置、移动或者涂抹、改动，损坏历史风貌建筑的保护标志。”<ref>[http://www.022net.com/2006/8-9/434325192990682.html 天津举办“历史风貌建筑保护铭牌揭幕仪式”] {{webarchive|url=https://web.archive.org/web/20060828171446/http://www.022net.com/2006/8-9/434325192990682.html |date=2006-08-28 }}</ref>
