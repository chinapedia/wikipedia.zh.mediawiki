'''天球'''（英語：'''Celestial sphere'''），是在[[天文學|天文學]]和[[導航|導航]]上想出的一個與[[地球|地球]]同圓心，並有相同的[[自轉軸|自轉軸]]，[[半徑|半徑]][[無窮|無限大]]的[[球|球]]。[[天空|天空]]中所有的物體都可以當成投影在天球上的物件。地球的[[赤道|赤道]]和[[兩極|地理極點]]投射到天球上，就是[[天球赤道|天球赤道]]和[[天極|天極]]。天球是[[位置天文學|位置天文學]]上很實用的工具。

在[[亞里斯多德|亞里斯多德]]和[[托勒密|托勒密]]的模型，天球想像成實際的物體，而不僅僅是一個幾何的投影（參見[[天球模型|天球模型]]）。

== 視差效應 ==
天球可以使用[[地心|地心]]為中心，也可以使用[[地表|地表]]為中心的體系。前者的作法是假設觀察者位於地球的中心，因此不需要考慮[[視差|視差]]的影響。後者的中心是觀測者位於的地球表面，因此[[地平面|地平]]視差的影響是不能忽略的，特別是月球的位置。

== 天半球 ==
天球被[[赤道|赤道]]的投影分為兩個空間，分別為南天球和北天球。同樣的，我們也可以在天球上找到北回歸線、南回歸線、天北極和天南極，可以量化天球上各種物體的方向，建構出各種的[[天球坐標系統|天球坐標系統]]。

因为地球是自[[西|西]]向[[東|東]]旋轉，所以我们就把天球看成是自東向西旋轉。

[[太陽|太陽]]在天球上每天移动约1度，一年则移动一周（360度），这称之为太陽周年视运动，太阳中心在天球上视运动的轨迹则是[[黄道|黄道]]。

== 恆星時 ==
隨著地球繞著軸心以23小時56分的周期由[[西|西]]向[[東|東]]自轉，天球和所有的天體都以[[天極|天球極點]]為中心，以相同的時間由東向西移轉動，這就是[[周日運動|周日運動]]。因此恆星會由東方升起，通過南北向的子午線過[[中天|中天]]，然後在西方沒入（除非這顆恆星是[[拱極星|拱極星]]）。第二天晚上，天體在特定的時間會再東升，但是因為我們的鐘表時間是24小時整，因此會提早4分鐘升起；再過一晚早8分鐘升起，也就是說每天都會提早4分鐘。

這種與鐘表時間明顯的不一致是因為太陽不是固定在天球上的天體，每天會在天球上的[[黃道|黃道]]，一個[[大圓|大圓]]，向東移動約1°（這是太陽的[[周年運動|周年運動]]，一年在天球上360°，或是完整的圓周）。如果以4分鐘的時間對應於角度的1°（360° = 24小時），則需要額外的4分鐘太陽才會回到上回的相對位置，例如中天的子午線，完成完整的一圈運動要24小時（這是忽略季節性變化的平均值，參見[[均時差|均時差]]。

因此，一般的時鐘顯示的是[[太陽日|太陽時]]，[[天文學家|天文學家]]研究恆星運動使用的是[[恆星時|恆星時]]，等同於太陽時的23小時56分。

== 恆星球 ==
天球也可以引用到天球的物理模型，就是所謂的恆星球，這種天球可以顯示出在給定的時間和地點可以看見的星座。
