'''天福山起义'''，[[中共胶东特委|中共胶东特委]]于1937年12月24日在[[山东省|山东省]][[文登县|文登县]][[天福山|天福山]]发动的一次[[抗日|抗日]]武装起义，起义成立了[[山东人民抗日救国军第三军|山东人民抗日救国军第三军]]。

== 背景 ==
1937年7月7日，[[卢沟桥事变|卢沟桥事变]]爆发，[[日本|日本]]发动全面[[侵华战争|侵华战争]]。9月22日，[[中国国民党|中国国民党]][[中央通讯社|中央通讯社]]以《中国共产党为公布国共合作宣言》为题发表[[中国共产党中央委员会|中国共产党中央委员会]]交付中国国民党的《[[:s:中共中央為公佈國共合作宣言|中共中央为公布国共合作宣言]]》，以[[国共合作|国共合作]]为基础的[[抗日民族统一战线|抗日民族统一战线]]正式形成。<ref name=sdyq>{{Citeweb |title=抗战初期山东党组织领导的抗日武装起义 |url=http://www.sdyq.org/2015/kangri_0424/269.html |publisher=山东涌泉网 |date=2015-04-24 |access-date=2016-10-12 |archive-url=https://web.archive.org/web/20170305003657/http://www.sdyq.org/2015/kangri_0424/269.html |archive-date=2017-03-05 |dead-url=no }}</ref>

10月，[[侵华日军|侵华日军]][[矶谷廉介|矶谷廉介]][[第十师团|第十师团]]由[[津浦铁路|津浦铁路]]南下占领[[德州市|德州]]后，国民政府山东省主席[[韩复榘|韩复榘]]率军南逃。鉴于日军入侵山东、国军南逃，而山东人民的抗日情绪高涨，[[中共山东省委|中共山东省委]]制定了分区域发动抗日[[武装起义|武装起义]]的计划，将山东划分了冀鲁边、[[胶东半岛|胶东]]、鲁中、鲁东等等十个区域，每个区域在发动武装起义后以“抗日游击队”或“山东人民抗日救国军”的名义各成立一军。<ref name=sdyq/><ref name=jd705>{{Citeweb |title=胶东革命历史事件:天福山起义 |url=http://www.jiaodong.net/news/system/2015/07/05/012779102.shtml |publisher=胶东在线 |date=2015-07-05 |access-date=2016-10-12 |archive-url=https://web.archive.org/web/20161012234746/http://www.jiaodong.net/news/system/2015/07/05/012779102.shtml |archive-date=2016-10-12 |dead-url=no }}</ref>

== 起义过程 ==
1937年12月上旬，1936年被国民政府逮捕的原中共胶东临时工委书记[[理琪|理琪]]被保释出狱，获释后受山东省委指派返到胶东重建[[中共胶东特委|中共胶东特委]]。12月15日，理琪在特委驻地文登县[[沟于家村|沟于家村]]召开特委扩大会议，会议传达了山东省委关于分区发动武装起义，组建“山东人民抗日救国军第三军”的指示。会议经过讨论决定以[[昆嵛山红军游击队|昆嵛山红军游击队]]为基础，于12月24日在文登、荣成、威海边界的天福山举行起义。<ref name=jd705/>

24日拂晓，理琪、[[吕志恒_(哈尔滨市长)|吕志恒]]、[[林一山|林一山]]、[[柳运光|柳运光]]、[[于得水|于得水]]等中共胶东特委及起义领导人与八十余名起义群众陆续到达天福山。理琪宣布[[山东人民抗日救国军第三军|山东人民抗日救国军第三军]]成立，参加起义人员组成第三军第一大队，于得水任大队长，[[宋澄|宋澄]]任政治委员。<ref name=jd705/>

== 后续发展 ==
起义后，理琪、吕志恒、林一山等几位主要负责人，分头到各地继续发动群众，扩大武装；于得水和宋澄等则率一大队西进。12月31日，一大队在文登县岭上村被文登县长李毓英率部包围，政委宋澄等29人遭逮捕，于得水突围，史称“[[岭上事件|岭上事件]]”。
