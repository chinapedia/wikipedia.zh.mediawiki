'''天主教石鐘山紀念小學'''是一所位於[[香港|香港]][[荃灣|荃灣]][[海濱花園|海濱花園]]旁的全日制[[天主教|天主教]]津貼小學，為紀念石鐘山先生對[[天主教香港教區|天主教香港教區]]的貢獻而於1997年創辦。

==SCS的定義==
*'S'=study smart 學習有方
*'C'=courage to create 勇於創新
*'S'=serve god and other 侍主侍人

== 辦學宗旨 ==
本校秉承校訓「愛主愛人，求道更新」，本著基督精神，讓學生在優良及愉快的環境內，善用資訊科技促進學習，從而發揮個人的多元智能，尊重他人，尊重生命；關心社會，關心國家，接受全人教育。

== 學校簡況 ==
學校於1997年創辦，並在1999年2月11日舉行開幕典禮，由香港主教胡振中樞機祝聖及揭幕。學校以SCS核心價值（SCS Core Values）為本，為學生提供全人教育。包括學習有方（Study smart）、勇於創新（Courage to create）及侍主侍人（Serve God and others）。學校校舍寬敞，環境優美，學生享有較大之活動空間。

== 輔導教學計劃 ==
學校本著有教無類的精神，接納不同能力的學生。因材施教，開設不同拔尖保底班別及課程。學校提供專業支援人員，包括駐校教育心理學家、
資源老師、學生輔導老師、加強輔導教學老師合作提供支援。

學校推廣資優教育普及化，發展學生潛能。資優課程包括有Super 領袖教室、中文創意寫作、全腦教室、創意科藝工程及奧數小精英等。

學校最近推出STEM教育,而部分資優學生更接受了長沙灣天主教英文中學的STEM課程。

== 考試安排 ==
全年有3次考試。另外各科設進展性評估、專題研習、閱讀報告及口頭匯報等。

== 學校特色 == 
•學校為學生設計多元化的校本課程。校本課程分別有古文扑扑齋自學課程、英文拼音及圖書閱讀計劃、中英文演說技巧訓練、無紙藝術21、我是未來領袖課程、拉闊生命德育課程及校本電腦課程等。

•	學校於各級推行多元化語境學習活動，提升學生對兩文三語的學習興趣及能力。各課活動分別有古詩背誦、集誦表演、個人演講、攤位遊戲 Show and Tell, Story Telling, Book Presentation, English Passbook, LRLE  handbook, 普通話小冊子及中文自學小冊子等。

•	學校提供不同資訊科技輔助教與學。各個資訊科技教學分別有WELNET (中英數常學習平台)、電子互動白板、21世紀技能學習中心 (CSL)、校園電視台及KINECT體能動感學習等。

•	學校推廣多元化課外活動，設制服團隊八隊，包括小童軍、幼童軍、公益少年團、少年警訊、基督小先鋒、紅十字會少年團、女童軍及升旗隊，參與學生約400人。另外更設有特別組及循環小組，包括有English Buddies, SCS Ambassadors, 創藝坊、中文小秀才、普通話精英大本營、多元智能Board Game、足球小子、繽紛手鐘樂、宗教及德育劇團、無紙美術及校園電視台等。各項校際比賽及活動，學生表現出色，獲得獎項超過1000個。

•	學校鼓勵家校溝通，定期舉行分級家長座談會、家長茶座，促進家長及教師之間的溝通。另外，一年一度的親子運動會、明愛賣物會及家教會旅行等，讓家長及師生能提升彼此的互動。

•	透過「專題研習」培養學生的共通能力

•	推行「體適能獎勵計劃」及「跳繩強心計劃」，發展學生的運動潛能

•	透過週會活動推行全方位的「拉闊生命教育」
