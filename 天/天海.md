'''天海'''（[[天文_(年號)|天文]]5年？ - [[寬永|寬永]]20年[[10月2日_(舊暦)|10月2日]]，即{{bd|1536年||1643年|11月13日|catIdx=T}}）從[[安土桃山時代|安土桃山時代]]到[[江戶時代|江戶時代]]初期的[[天台宗|天台宗]]僧侶，也稱'''南光坊天海'''。[[諡號|諡號]]為'''慈眼大師'''。以作為[[徳川家康|徳川家康]]的智囊，使家康能夠創設江戶幕府而因此活躍的政治僧侶。

== 生涯 ==
關於天海的前半生無法詳細考究。

但成為家康的家臣後，他的事蹟才開始出現。天正16年（1588年）他來到[[武藏國|武藏國]]的無量壽寺北院（位於現在[[埼玉縣|埼玉縣]][[川越市|川越市]]。現在的[[喜多院|喜多院]]）。這時，他兼任江戶崎不動院。根據[[淺草寺|淺草寺]]的史料、在進攻北條家之時、天海和浅草寺的住持忠豪就已經在家康的幕僚之中。之後天海會前往關東的原因，也是因為受到家康的賞識而過去。

在豪海之後，天海擔任北院的住持是慶長4年（1599年）的事。[[關原之戰|關原之戰]]時，據說是以參謀身分服侍家康。之後作為家康的參謀，由他負責和朝廷談判等工作。在1607年就任[[比叡山|比叡山]]的探題並且再興了[[延暦寺|延暦寺]]。然後又在慶長17年（1612年）着手重建無量壽寺北院、並把寺號改為喜多院，以作為關東天台宗的總本山。慶長18年（1613年）時，又從家康那邊受命擔任日光山貫主，並且再興了本坊・光明院。他本人自己也深度涉及到作為[[大阪之戰|大阪之戰]]開端的「[[方廣寺鐘銘事件|方廣寺鐘銘事件]]」。

元和2年（1616年）、病危的家康，以遺言託付天海等人處理有關葬禮的儀式和廟號。家康死後，[[崇傳|以心崇伝]]和[[本多正純|本多正純]]等為了以要採用哪一種廟號而爭辯著。天海主張作為「[[權現|權現]]」並以「[[山王一実神道|山王一実神道]]」來祭拜家康，以心崇伝則主張家康應作為「[[明神|明神]]」並以「[[吉田神道|吉田神道]]」來祭拜。由於無法確定兩方意見，2代將軍[[德川秀忠|德川秀忠]]於是詢問天海，天海便以「明神」是作為豐国大明神而被[[豊臣秀吉|豊臣秀吉]]所採用的廟號，但那之後的豊臣氏卻因此滅亡，所以相當不祥而进一步建議秀忠考慮，於是家康的廟号即因此決定以「東照大権現」命名。

之後天海也繼續侍奉著2代將軍[[德川秀忠|德川秀忠]]和3代將軍[[德川家光|德川家光]]，寛永元年（1624年）在忍岡創建[[寛永寺|寛永寺]]。為了江戶的都市計画，所以天海構想以[[陰陽道|陰陽道]]和[[風水|風水]]為基礎來設計以保護江戶。寛永20年（1643）天海以108歳年齡過世。在那5年後，從朝廷那邊受贈慈眼大師的諡號。

[[慶安|慶安]]元年（1648年）時、天海所着的『寛永寺版（天海版）大藏經』，則在幕府的支援下完成。

其墓所位於現在的[[栃木縣|栃木縣]][[日光市|日光市]]。

== 杜撰的天海 ==
天海為蘆名氏出身説的問題在於它的證據過於牽強。舉例來說、芦名氏的家徽為三浦家的「丸に三引き両」。但是天海所用的家徽（今日在喜多院或上野的兩大師堂都能看見）為「丸に二引き両」和「三宅輪宝」，與蘆名氏有明顯的差別。

而「丸に二引き両」不但為足利氏的家徽，同時在[[美濃國|美濃國]][[惠那市|惠那市]]的[[明知遠山氏|遠山氏]]也使用這個家徽。另一個「三宅輪宝」家徽，正如字面所示為[[三河國|三河國]][[豐田市|豐田市]]的三宅氏的家徽。家徽和姓氏一樣，一般是不會拿和自己沒關係的來使用。況且使用至少兩種和自己出身無關的家徽，就算不是如天海一樣的高僧也會有人如此使用。不過據說天海曾經參觀過1561年在信濃國所發生的第四次[[川中島之戰|川中島之戰]]。

有出自[[小說|小說]]等等的見解，認為天海是足利家第12代將軍[[足利義晴|足利義晴]]之子。

另外也有說法說天海和在[[本能寺之變|本能寺之變]]討伐織田信長，之後在[[山崎之戰|山崎之戰]]中被認為戰死的[[明智光秀|明智光秀]]是同一個人物（有常常被提出說天海在[[日光市|日光]]的墳墓有叫做「[[明智平|明智平]]」的這個地方）。如果天海和光秀為同一人的話則享年116歲，用天海作為光秀的年紀來看的話則顯得稍微不夠合理，因此也有兩者是相似人物的可能性。鑑定天海和光秀的書信筆跡（2000年8月6日、TBS系列「日立 世界・ふしぎ発見!」放映）。如果依據那個鑑定結果，則天海和光秀是不同人，但兩人的字跡有幾個是類似的，所以據說能推定兩人有著類似父子般的近親關係。從這裡被認為天海可能是光秀的表弟，所以有傳出天海可能為在[[本能寺之變|本能寺之變]]中擔任先鋒、而在[[山崎之戰|山崎之戰]]戰敗後用馬跨越[[琵琶湖|琵琶湖]]湖上然後在[[坂本城|坂本城]]自殺的[[明智秀滿|明智秀滿]]（舊名：三宅弥平次）。

此外也有天海死去的年紀為135歲，並且為光秀和秀滿兩人分為兩代扮演天海的見解。

== 天海登場的作品 ==
* [[戰國BASARA|戰國BASARA系列]]：伴隨在小早川身旁的的神秘高僧，總是低聲細語的傳授智慧給予決斷，雖然對外宣稱本名與身世不明，但真實身分是明智光秀。
* [[甲賀忍法帖|甲賀忍法帖]]：向為了竹千代(德川家光)・国千代(德川忠長)間白熱化的繼承之爭而苦惱的[[徳川家康|徳川家康]]、提出以用[[忍者|忍者]]間的決鬥來決定繼承人的建議。
* [[柳生忍法帖|柳生忍法帖]]：作為在故事中和黑手來往的重要人物登場
* [[櫻花大戰|櫻花大戰]]：率領「黒之巣會」、目標是在太正時代讓江戸幕府復活
* 新 [[鬼武者|鬼武者]] 夢之曙光：異形者，為了驅逐「幻魔」、而與[[結城秀康|結城秀康]]一起奮鬥，但他的真正身分是一、三代的主角明智左馬介。
* [[少女杀手阿墨|少女杀手阿墨]]：培養阿墨並把她當作刺客一樣的命令她、之後保護了阿墨
* [[人中之龍_登場!|人中之龍 登場!]]：為了擁護秀忠、而暗殺秀康。企圖政變。
* [[仁王|仁王]]：陰陽師, 阿福的師傅，守護靈為玄武。後期揭露其身份為明智光秀。
