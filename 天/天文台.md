'''天文臺'''又称'''观象台'''，是指研究和觀測[[天文現象|天文現象]]的機構。天文臺觀測天文現象時，為了能更加精確地作出觀測結果，天文臺的觀測站都會建於山上，因為地面上的城市燈光過亮，會影響[[天文望遠鏡|天文望遠鏡]]觀測的準確性。

== 历史 ==
公元前2600年，[[古埃及|古埃及]]为了观测[[天狼星|天狼星]]，建立了迄今为止已知世界上最早的天文台；前2000年，[[巴比伦|巴比伦]]也建立了天文台。相傳[[中国|中国]]在大约2500年前，也开始有天文台，当时称为'''清台'''、'''灵台'''、'''观象台'''。我們的

古代许多国家的天文台常常不但是天文观测的场所，也是运用[[占星学|占星学]]的场所，也因此天文台一般都为统治者所控制。

15-16世纪，欧洲的一些天文学家开始建立自己的天文台，其中很著名的就是[[丹麦|丹麦]]天文学家[[第谷|第谷]]1576年在[[哥本哈根|哥本哈根]]建立的天文台，它配备了当时最先进的天文仪器。

[[天文望远镜|天文望远镜]]发明后，天文台得到了发展。1667年法国建立了[[巴黎天文台|巴黎天文台]]；1675年[[英国|英国]]建立了[[格林尼治天文台|格林尼治天文台]]。

20世纪，[[天体物理学|天体物理学]]的发展进一步促进了天文台的发展，许多天文台装备了大口径的[[反射望远镜|反射望远镜]]。目前世界上大约有400个大型的天文台。

== 分类 ==
[[File:天体仪.jpg|thumb]]
现代天文台分为：
* [[光学天文台|光学天文台]]：主要装备各光学天文仪器，如[[光学天文望远镜|光学天文望远镜]]、[[太阳镜|太阳镜]]等，从事[[方位天文学|方位天文学]]或[[天体物理学|天体物理学]]方面的研究
* [[射电天文台|射电天文台]]：一般主要由巨型甚至超巨型的无线接受设备和基站等构成，装备[[射电望远镜|射电望远镜]]，观察的范围更大，受干扰小，从事[[射电天文学|射电天文学]]的研究
* [[空间天文台|空间天文台]]，主要有一些用于[[空间观测|空间观测]]的[[人造卫星|人造卫星]]组成，配备非常先进的光学观测系统。

另外还有一些[[教学天文台|教学天文台]]以及[[大众天文台|大众天文台]]，教学天文台一般设置在一些大学学院，用于教学研究，而大众型天文台主要起到普及天文的作用，对大众开放。

== 特殊例子 ==
* [[香港天文台|香港天文台]]實際上是[[香港|香港]]的[[氣象台|氣象台]]。雖然天文台會向市民提供天文資料，但最主要卻是提供氣象服務。
