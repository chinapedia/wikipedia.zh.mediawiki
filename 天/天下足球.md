《'''天下足球'''》（英文：'''Total Soccer'''），[[中国中央电视台体育频道|中国中央电视台体育频道]]（CCTV5）[[足球|足球]]資訊节目，2000年11月27日启播。逢周一19：30－21：25于CCTV5播出，每期115分钟（有时会适当延长）。口号为“最纯粹的足球，最高级的享受”，现时男主持人为[[邵圣懿|邵圣懿]]，女主持人为[[杨茗茗|杨茗茗]]或[[王曦梁|王曦梁]]。

== 启播原因 ==
2000年之前，[[央视|央视]]唯一的足球資訊节目是《[[足球之夜|足球之夜]]》。但因为[[欧洲|欧洲]]足球联赛比赛主要在[[周末|周末]]播出，为适应关注欧战赛事观众的需要，故将《[[足球之夜|足球之夜]]》国际部分独立出来，改至周一播出，从而启播《天下足球》。

近年因中国足球事业陷入低谷，加上收看欧洲足球联赛观众人数日渐增多，《天下足球》已成为[[中国大陆|中国大陆]]地区影响力最大的足球资讯节目。

== 内容 ==
=== 概况 ===
天下足球节目主要播出世界[[足坛|足坛]]重要比赛录像（因为版权问题，某些赛事录像无法播出，如[[德甲联赛|德甲联赛]]等）以及重大新闻，同时设有一些内容与足球相关的小栏目，以增加娱乐性。有时则会整期推出特别专题。

=== 解说 ===
节目启播初期，节目内容解说由主持人兼任，2002年后则由节目编辑担任。目前主要解说者为[[朱晓雨|朱晓雨]]、[[刘嘉|刘嘉]]二人。其中，西甲板块编辑王涛已于2014年离开央视。

=== 常设栏目 ===
==== 绿茵重量级 ====
除推出特别专题外，每期均播出的栏目，为节目主要部分。播放国际上重要赛事录像，配以解说。

==== 足球制造 ====
播放国际足坛重要新闻，以及球员、球队动态。

==== 绝对巨星 ====
不定期播出，介绍国际足坛现役及退役著名球星，展现球星成长历程和经历。曾介绍球星包括：[[保罗·马尔蒂尼|保罗·马尔蒂尼]]、[[范巴斯滕|范巴斯滕]]、[[菲戈|菲戈]]、[[罗纳尔迪尼奥|罗纳尔迪尼奥]]、[[加斯科因|加斯科因]]、[[舍甫琴科|舍甫琴科]]、[[罗伯特·巴乔|罗伯特·巴乔]]、[[维埃里|维埃里]]、[[大卫·西曼|大卫·西曼]]、[[里瓦尔多|里瓦尔多]]、[[齐达内|齐达内]]、[[亨利|亨利]]、[[索尔斯克亚|索尔斯克亚]]、[[吉格斯|吉格斯]]、[[卢卡·托尼|卢卡·托尼]]、[[德罗巴|德罗巴]]、[[卡卡|卡卡]]等。

==== Top10 ====
固定栏目。以“十大”形式，总结回顾足坛重大事件及重要人物。

因该栏目较受欢迎，节目组曾联合[[武汉出版社|武汉出版社]]，推出《[[天下足球百大Top10|天下足球百大Top10]]》一书，内容为文字版Top10系列精华。同时附送两张[[VCD|VCD]]光碟，为Top10系列精彩回顾。

==== 看球听歌 ====
不定期播出，以[[MV|MV]]形式展示足球精彩镜头。

==== 大画足球 ====
以[[Flash动画|Flash动画]]形式，讲述足坛的人与事。

==== 戏说足球 ====
2008年起启播，搜罗球场内外的趣闻趣事。

==== 十大扑救、进球 ====
评选出上一周赛场上10个最精彩的进球及[[门将|门将]]扑救动作。

=== 著名专题 ===
*《星·球》：以50大形式，回顾[[绿茵场|绿茵场]]上球星的经典技术动作。如“五十大脚后跟进球”、“五十大长途奔袭”。
*《贝影》：制作于[[贝克汉姆|贝克汉姆]]宣布将转会[[美国大联盟|美国大联盟]]后，回顾他在[[曼联|曼联]]、[[皇家马德里|皇家马德里]]的经历。主题音乐《[[Because_Of_You|Because Of You]]》。2013年，贝克汉姆宣布退役，天下足球栏目组又因此制作了名为《贝影-大结局》的专题节目。用来回顾和总结贝克汉姆的职业生涯点滴。
*《十二码》：总结历史上的著名[[点球|点球]]、点球高手，以及关于点球的趣闻。
*《世界足坛一百单八将》：将足坛明星与《[[水浒传|水浒传]]》内容相结合。选出108位与[[梁山好汉|梁山好汉]]特征相似的球星。由[[朱晓雨|朱晓雨]]主持。
*《疯狂的足球》：共分四季，其中第一季分三集，第二、三、四季为单集一季，搜索球场内外爆笑镜头，配上[[搞笑|搞笑]]旁白，由[[王涛|王涛]]主持。
*《为足球而歌唱》：播放欧洲联赛豪门的[[队歌|队歌]]或[[球迷歌曲|球迷歌曲]]，以及国际赛事著名歌曲。
*《亨利：谁与争锋》制作于亨利转会美国大联盟纽约红牛队后，回顾了亨利的国家队和俱乐部经历。2014年12月，亨利宣布退役，天下足球栏目组又因此制作了名为《亨利：谁与争锋-大结局》的专题节目。用来回顾和总结亨利的职业生涯点滴。
*《黄金十年》后顾了2000-2010年这十年之间国际足坛的发展和大起大落，评选了“十年最佳阵容”“十年最佳球员”以及十年中的一场国家队经典比赛和一场俱乐部经典比赛。
*《演员的自我修养》以足球场上球员“表演”为主题的爆笑专题片，分别于2011年10月10日和2011年11月14日的《天下足球》节目中播出。
*《天下足球奥斯卡》2011年12月26日和2012年1月2日播出，专题对2011年世界足坛进行了年终盘点。12月26日的第一期在球迷中引起了巨大的反响，尤以“2011十大感动瞬间”最受好评。专题宣传片主题曲《When We Stand Together》。

== 争议 ==
=== 邀请蒂埃里·亨利玩盗版游戏 ===
中央电视台体育频道2007年6月4日晚播出的《天下足球》栏目，被指出公开使用走私游戏机和盗版软件。

节目中有一部分是主持人段瑄和嘉宾法国前锋蒂埃里·亨利玩Xbox 360游戏[[Winning_Eleven_X|Winning Eleven X]]。该Winning Eleven X是不完全的民间汉化版本，执行此游戏的[[XBOX_360|XBOX 360]]主机必须经过特殊改装。另外XBOX 360和Winning Eleven X皆未在中国大陆正式发行。事后《天下足球》节目组表示对是否使用盗版不知情。
===直播失误===
2007年8月13日，主持人段暄在《天下足球》节目的现场直播中，出现了上半身身着西装，却在播音桌下伸出一条没穿西裤的腿的画面<ref>{{cite web |url=http://www.hinews.cn/news/system/2010/10/29/011379878.shtml|title=央视转播乌龙盘点：段暄裤衩出境 火箭直播被掐|date=2010年10月29日 |accessdate=2018年1月9日|publisher=南海网|work= |language=zh-cn}}</ref>。

事情过去十年，直播失误再次发生。2017年6月12日，第一个片段播放结束，演播室中主持人杨茗茗在直播中肉色[[奶罩|奶罩]]左边带子不慎滑落，直到镜头第三次切回演播室，带子才被收了回去<ref>{{cite web|title=[天下足球]20170612 C罗与梅西:绝代双骄（一）|url=http://tv.cctv.com/2017/06/12/VIDEYTluidtozvSP0WeFj6ZK170612.shtml|website=CCTV.com|date=2017年6月12日|accessdate=2018年1月9日|language=zh-cn}}</ref>。
