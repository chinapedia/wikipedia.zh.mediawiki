{{天津}}
[[天津市|天津市]]是[[中华人民共和国|中华人民共和国]]的[[直辖市|直辖市]]、[[国家中心城市|国家中心城市]]和中国北方最大[[沿海开放城市|沿海开放城市]]。天津位于[[华北平原|华北平原]]东北部的[[海河|海河]]各支流交汇处，东临[[渤海|渤海]]，北依[[燕山|燕山]]，介于东经116°43′－118°04′，北纬38°34′－40°15′之间，是以城市为主体的[[中央直辖市|中央直辖市]]行政区划单位。天津是中国北方少有的山、河、湖、海、湿地齐全的城市，有[[海河|海河]]在城中蜿蜒而过。<ref>{{cite web| url=http://www.022net.com/2013/8-4/48684914296715.html| title=孙春兰在天津市委十届三次全会上的讲话| publisher=人民网| accessdate=2013-08-18| deadurl=yes| archiveurl=https://web.archive.org/web/20130809221004/http://www.022net.com/2013/8-4/48684914296715.html| archivedate=2013-08-09}}</ref>天津市全境属于温带季风气候，主要受季风环流影响，四季分明。天津市陆地面积为11916.85平方公里<ref name="天津2012统计年鉴">{{cite book | title=天津2012统计年鉴 | publisher=天津人民出版社 | year=2012}}</ref>，海洋管辖面积为3000余平方公里。

天津自古因漕运而兴起，明永乐二年十一月二十一日（1404年12月23日）正式筑城，是中国古代唯一有确切建城时间记录的城市。<ref name="建城记录">{{cite news | title=天津：沉淀着厚重的历史 | date=2007年11月18日 | agency=《工人日报》 | accessdate=2011-12-23}}</ref>截止到2012年，天津市的常住人口达到了1413.15万人，市域内的城镇化率达到81.55%。<ref>{{cite web | url=http://www.022net.com/2013/4-4/4165571425177.html | title=天津人口发展状况报告发布 常住人口达1413万 | publisher=人民网 | accessdate=2013-08-23 | deadurl=yes | archiveurl=https://web.archive.org/web/20160304123753/http://www.022net.com/2013/4-4/4165571425177.html | archivedate=2016-03-04 }}</ref>

== 地理区位 ==
[[File:China_Tianjin.svg|thumb]]
天津市的地理坐标位于东经116°43′－118°04′，北纬38°34′－40°15′之间，位于中华人民共和国的[[华北地区|华北地区]]，处于国际时区的东八区。天津市市域范围东起汉沽洒金坨以东的陡河西干渠，西至静海县子牙河王进庄以西的滩德干渠，东西宽117公里；北起蓟县[[黄崖关|黄崖关]]，南至大港翟庄子沧浪渠。天津市的东、西、南方分别与[[河北省|河北省]]的[[唐山|唐山]]、[[承德|承德]]、[[廊坊|廊坊]]、[[沧州|沧州]]地区接壤，并与[[山东省|山东省]]、[[辽宁省|辽宁省]]隔海相望。

截至2011年，天津市市域总面积为11860.63平方公里<ref name="天津2012统计年鉴" />，陆界长1137.48公里<ref>[http://cn.chinagate.cn/ECONOMIC/2007-11/30/content_9328447.htm 天津地理位置、行政区划、人口民族概况]，中国发展门户网，2010-8-5查阅</ref>。而随着[[天津港|天津港]][[东疆保税港区|东疆港]]等新港区和第二港口[[天津南港|天津南港]]的建设等[[天津填海造陆工程|填海造陆工程]]的进展，天津的[[海岸线|海岸线]]持续增长，从2000年的146.53公里，增长至2012年的302.62公里。<ref name="2000-2012年中国北方海岸线时空变化分析">{{cite journal | title=2000-2012年中国北方海岸线时空变化分析 | author=徐进勇 | journal=地理学报 | volume=63|date=2013年5月}}</ref><ref>{{cite web |url = http://news.enorth.com.cn/system/2008/02/29/002897899.shtml |title = 天津将投资600亿建设我国最大填海造陆工程 |publisher = 北方网 |accessdate = 2013-08-16 |deadurl = yes |archiveurl = https://web.archive.org/web/20140221104720/http://news.enorth.com.cn/system/2008/02/29/002897899.shtml |archivedate = 2014-02-21 }}</ref>

== 地理环境演变 ==
=== 第四纪以来 ===
天津地区位于[[渤海湾|渤海湾]]西岸，自新生代的新近纪以来，地表环境以陆地为主，当时地表的沉积物基本上是河流湖泊环境中的沉积物，并且普遍埋藏有陆相生物化石群。根据研究结果表明，天津地区在第四纪成为陆地环境阶段，多为气候寒冷的冰期。尤其在距今1.8万年前的[[第四纪|第四纪]]末次冰期气候盛行时期，北半球中纬度的气温比现在要低10-15摄氏度。据此推算，当时的海平面比现在低132米左右，当时的[[黄海|黄海]]、[[渤海|渤海]]等浅海海地露出海面，与华北平原连城广袤的大平原，为陆地生物迁徙提供了条件。在天津附近的渤海海底，有考古工作者曾发现过活跃于第四纪冰期的[[披毛犀|披毛犀]]和[[猛犸象|猛犸象]]化石。<ref name="天津地理">{{cite book |title = 天津地理 |publisher=北京师范大学出版社 |author=仲小敏、李兆江 |year=2011 |isbn=9787303122356}}</ref>

=== 原始聚落形成时期 ===
天津平原地区的开发大约开始于距今6000年前的[[新石器时代|新石器时代]]，而且经过了一个由北向南、自西向东逐步推进的过程。1974年，考古工作者首先在天津北郊刘家码头发掘出石斧和石磨棒，以后又陆续在[[武清区|武清区]]、[[宁河县|宁河县]]和[[宝坻区|宝坻区]]发现了同类的石斧和石磨棒<ref name="津门考古">{{cite book |title = 津门考古 |publisher=天津人民出版社 |author=天津市文物管理处 |year=1982 |location=天津}}</ref>。经鉴定，在天津出土的石器均属于新石器时代的遗物。因而可以证明，当时天津北部和西部地区地势较高、河流纵横、土质肥沃，新时期时代已经有群居的先民在这里进行农耕和渔猎活动，因此出现了许多原始的乡村聚落。<ref name="津门考古" />与此同时，天津东部地区仍然爱处于海侵之中。根据史料记载，自商代开始，[[黄河|黄河]]曾先后三次改道在天津附近入海<ref name="黄河水利史述要" />，经过数千年的淤积作用，海水才逐渐退到现在的海岸位置。<ref name="天津地理" />

=== 早期城市聚落形成时期 ===
经过战国时期和秦代的开发，到西汉初年天津地区逐步富饶，人类居住和经济活动的有利条件增多。因天津平原濒临渤海，当时的滹池河（今[[滹沱河|滹沱河]]）、沽水（今[[北运河|北运河]]）、治水（今[[永定河|永定河]]）、桃水（上游为今[[拒马河|拒马河]]）以及泒水（今[[海河|海河]]）、庚水（今[[蓟运河|蓟运河]]）等均在天津分流入海。独立的海河水系尚未形成，黄河则在东平舒以南入海。<ref name="黄河水利史述要">{{cite book | title=黄河水利史述要 | publisher=水利出版社 | author=黄河水利委员会}}</ref>

天津平原上有著名的洼淀[[雍奴薮|雍奴薮]]，因而此地的捕鱼业和煮盐业兴盛。当时的朝廷为了税收和管理，在此设置了[[无终县|无终县]]。县城内有朝廷派驻的管理机构，周围建有城垣，形成了天津平原上早期的城市聚落。因天津平原水系众多，航运便利，这些城市聚落成为当地的政治经济中心以及水运交通枢纽。<ref name="天津地理" />

西汉末期，由于局部海侵，海平面在天津附近上升了大约1米，当时海岸在今天的[[军粮城街道办事处|军粮城]]一带。滨海平原4米等高线以下的地区均受渍水淹没，形成一片沼泽，天津滨海平原上的许多原始聚落因此消失，西汉初年在天津所设立的四座县城均被废弃。直到东汉时期，天津地区开始海退，沼泽面积有所萎缩。<ref>{{cite journal | title=《水经注》和天津地理 | author=韩嘉谷 | journal=历史地理 | year=2006 | volume=特刊}}</ref>中国古代地理名著《[[水经注|水经·鲍丘水注]]》曾有记载：“自是水之南，南极滹沱，西至泉洲雍权，东及于海，谓之雍权薮。其泽野，有九十九店，支流条分，往往迳通”。<ref>《水经·鲍丘水注》</ref>

=== 地质特征 ===
天津地区地质史历时30亿年以上，由[[太古代|太古代]]发展到[[第四纪|第四纪]]，才基本形成现今的地貌，当中经过了三个主要阶段，即地台基底形成阶段、地台盖层发育阶段和大陆边缘活动带阶段。<ref>{{cite web |url = http://www.tjsqgt.gov.cn/Lists/List122/DispForm.aspx?ID=7 |title = 天津地质发展简史 |publisher=天津市市区国土资源分局网站 |accessdate=2013-08-16}}</ref>从区域地质角度，按照地质力学的观点分析，天津地区在大地构造上可以划分成燕山褶皱带和属于新华夏系的华北沉降带两大构造单元。天津地处两大构造体系交界之处，尤其以第四纪以来的地壳活动最为明显。<ref>{{cite journal |title = 天津市地面沉降的特征及其危害 |author=董克刚、周俊、于强 |journal=地质灾害与环境保护 |year=2007}}</ref>天津地区的地层主要有太上古届和中上元古界、古生界及新生界，其形成年代漫长，较为连续。蓟县北部山区出露的中上元古界地层形成于距今18亿-8.5亿年前，属于海相、滨海相沉积，垂直厚度仅万米。由于该地层完整连续、界限清晰、构造简单、变质程度低，原始的沉积结构、构造和化石保存完好，为研究元古宙中晚期的岩石圈、水圈和生物圈发展演化提供了的研究证据，因此被确定为“中上元古界标准地层剖面”。<ref>{{cite web |url = http://news.sina.com.cn/c/2003-11-08/09441081300s.shtml |title = 中上元古界史书 |publisher=今晚报 |accessdate=2013-08-16}}</ref>

=== 地貌环境 ===
天津地势以平原和洼地为主，堆积平原区面积达11192.7平方公里，约占全市总面积的93%，其中近80%是河网密布的[[湿地|湿地]]和盐沼。<ref>{{cite web|url=http://www.tjbcgt.gov.cn/Lists/List75/DispForm.aspx?ID=3 |title=天津市地形地貌 |publisher=天津市国土房管局北辰区国土资源分局网站 |accessdate=2013-08-16 |deadurl=yes |archiveurl=https://web.archive.org/web/20101222094143/http://www.tjbcgt.gov.cn/Lists/List75/DispForm.aspx?ID=3 |archivedate=2010-12-22 }}</ref>因北部有低山[[丘陵|丘陵]]，海拔由北向南逐渐下降，属于由燕山[[山脉|山脉]]向滨海平原的过渡地带。北部平均海拔1052米；东南部频临[[渤海湾|渤海湾]]，平均海拔仅3.5米，是华北平原的最低点，亦是中国海拔最低的大城市。天津市的最高点为[[蓟县|蓟县]]的九顶山，海拔高度为1078.5米。此外，天津地表植被有针叶林、针阔叶混交林、落叶阔叶林、灌草丛、草甸、盐生植被、沼泽植被、水生植被、沙生植被、人工林、农田种植和城市绿化景观植物等。由于，天津市区的绿化树木曾以白毛杨为主，故春末夏初会时常有“飞絮”的现象发生，迫使天津市园林部门正在逐步以白蜡树等代替。<ref>{{cite web|url=http://www.022net.com/2013/5-23/481970332626939.html|title=从今年起天津将用白蜡树等逐渐替代散播飞絮的杨树|publisher=人民网|accessdate=2013-08-22|deadurl=yes|archiveurl=https://web.archive.org/web/20160305074952/http://www.022net.com/2013/5-23/481970332626939.html|archivedate=2016-03-05}}</ref>

=== 气候 ===
[[File:天津冬季的海河冰上垂钓.jpg|thumb]]
天津市位于中纬度[[欧亚大陆|欧亚大陆]]东岸，全境属[[温带季风气候|温带季风气候]]，主要受季风环流影响，四季较为分明。天津地区夏季受西太平洋[[副热带高气压|副热带高气压]]影响，以吹偏南风为主；冬季则受蒙古冷高气压控制，盛行偏北风。天津地区的一月平均气温为-5.4℃到-3.0℃，七月平均气温为25.9℃到26.7℃，年平均气温为12.3℃，年降雨量为500～700毫米，平均无霜期为196～246天。<ref>{{cite web |url = http://travel.sina.com.cn/china/2008-11-27/153039364.shtml |title = 天津地理环境概述 |accessdate=2012-12-09}}</ref>

根据天津气象站1910年以来的观测资料显示，近100年天津地区的平均气温呈显著上升趋势，其中，冬、春两季最低气温增加趋势最明显；夏季和全年高温阈值显著降低，但最长暖日天数自1990年代开始趋于增多；低温阈值显著上升，冬季升幅最大，冬、春两季霜冻日数和最长冷夜日数均显著减少。特别是，1980年代后天津地区的最低气温和冬季气温的快速升高。<ref name="近100年天津平均气温与极端气温变化">{{cite journal |title = 近100年天津平均气温与极端气温变化 |author=郭军，任国玉，任雨 |journal=高原气象 |date=2011年5月}}</ref>天津地区气温上升的趋势与全球变暖北半球的趋势一致。<ref name="气候变暖与天津粮食生产的关系">{{cite journal |title = 气候变暖与天津粮食生产的关系 |author=梁平德，吴凤珍 |journal=应用气象学报 |date=1993年1月}}</ref>同时，天津地区的气温年际间差异和极端性正在减弱趋于稳定。<ref name="近100年天津平均气温与极端气温变化" />

=== 水系 ===
[[File:炫彩津门99海河夜景.jpg|thumb]]干流|200px]]
{{Seealso|天津水系}}
天津市市域内河流密布、洼淀湿地亦广泛分布，河流和湿地的分布对[[天津城市规划|天津城市布局]]有着较大的影响。天津地处[[海河|海河]]流域下游，是海河五大支流[[南运河|南运河]]、[[北运河|北运河]]、[[子牙河|子牙河]]、[[大清河|大清河]]、[[永定河|永定河]]的汇合处和[[入海口|入海口]]，素有“九河下梢”之称。<ref>{{cite web |url = http://news.enorth.com.cn/system/2011/03/18/006187087.shtml |title = “九河下梢天津卫”九河都是嘛 说着简单解释难 |publisher=北方网 |accessdate=2013-08-17}}</ref>海河的干流流经天津市域，市内主要河流包括：月牙河、卫津河、津河、新开河等。

=== 海洋 ===
天津[[海岸线|海岸线]]地处[[渤海|渤海]]西部海域，长达302.62公里。<ref name="2000-2012年中国北方海岸线时空变化分析" />天津的海洋面积370多平方公里，正在开发利用的滩涂资源、富含大量[[浮游生物|浮游生物]]、游泳生物、[[底栖生物|底栖生物]]和潮间带生物的海洋生物资源。此外，在海水资源利用方面，天津自古以来就是著名的盐产地，拥有中国最大的盐场[[长芦盐场|长芦盐场]]。而且自2010年起推行的[[海水淡化|海水淡化]]工程将解决城市的淡水不足问题。<ref>{{cite web |url = http://finance.chinanews.com/ny/2013/04-15/4731323.shtml |title = 天津滨海新区数十万市民喝上海水淡化水 |publisher=中国新闻网 |accessdate=2013-08-17}}</ref>

=== 自然资源 ===
天津市的矿产资源、油气资源以及地热资源丰富。特别是在海洋油气资源方面，天津地区现已发现45个含油构造，储量可观。<ref>{{cite web|url=http://www.tj.gov.cn/zjtj/zrdl/zrdl/|title=天津地理位置|publisher=天津政务网|accessdate=2013-08-16|deadurl=yes|archiveurl=https://web.archive.org/web/20130721071700/http://www.tj.gov.cn/zjtj/zrdl/zrdl/|archivedate=2013-07-21}}</ref>大规模的油田有[[大港油田|大港油田]]和[[渤海油田|渤海油田]]等。<ref name="天津通鉴">{{cite book | title=天津通鉴 | publisher=中国青年出版社 | author=天津市地方志编修委员会}}</ref>在太阳能利用方面，天津地区属于太阳能资源较丰富的二类地区，全年可利用天数约200天，可大规模利用程度不高。<ref>{{cite web | url=http://news.enorth.com.cn/system/2013/08/23/011250546.shtml | title=解放南路打造“生态城” 首个零碳建筑即将竣工 | publisher=北方网 | accessdate=2013-08-23}}</ref>

== 经济地理特征 ==
早在6000年前的[[新石器时代|新石器时代]]，天津就有了农业垦殖活动。在古代，天津因漕运兴起，是中国北方的军事重镇（天津卫）及重要的商贸集散中心。近代，随着商业和金融业的繁荣及工业的兴起，天津发展成为北方重要的经济中心和重要的工业城市。清末民初，天津工商业规模一度超过北京。近代，天津在河海运通航枢纽的基础上，加之接近首都的特殊区位优势，成为中国最早对外开埠的开放性城市之一，成为华北的经济中心城市和北方金融中心。中华人民共和国成立前，天津的工业产值是北京市的7倍，直至1970年代初，天津在城市人口和经济规模方面被北京超越，均居于中国北方第二的位置。<ref name="天津地理" />

目前，天津市的三次产业中，第二产业具主导地位，且产业结构的地区差异明显。天津市的第一产业主要分布在远郊区县，第二产业以[[滨海新区|滨海新区]]和环城四区为主，第三产业则主要分布在[[天津中心城区|中心城区]]。<ref>{{cite journal |title = 区域协调与天津产业结构重组 |author=尹海林、曾坚、李哲 |journal=城市环境设计}}</ref>

== 人文地理特征 ==

=== 人口地理特征 ===
[[File:Anonymous-Ming_Chengzu.jpg|thumb]]
天津从一个军事防御性的城池和因漕运而兴起的商埠逐渐发展成为现代化特大型城市，人口的数量、分布和构成都随着历史的发展在不断地变化。古代天津的河漕、海运和芦盐之利不仅推动了整个天津商业的发展，也是天津人口聚集的重要因素。北宋时期，天津海河作为当时宋、辽两国的界河，南北贸易繁荣，并吸引了人口的聚集。金、元时期，天津凭借占据“地当九河要津，路通七省舟车”的地缘优势<ref>{{cite web | url=http://epaper.tianjinwe.com/tjrb/tjrb/2007-04/10/content_246639.htm | title=写在《七十二沽花共水》出版之时 | publisher=天津网 | accessdate=2013-08-23 | author=罗文华 }}{{Dead link|date=2018年8月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，在大直沽和[[三岔河口|三岔河口]]形成了天津的商业和人口的聚集地。明代，卫城的建立，燕王[[朱棣|朱棣]]被明太祖[[朱元璋|朱元璋]]派往天津一带戍边，同时带去了大量皖北、苏北人，随着当时天津内城集市的形成和发展，人口逐渐增多。<ref>{{cite journal |title = 明代天津人口与城市性质的变化 |author=高艳林 |journal=南开学报 |year=2011}}</ref>清代，天津盐业发展和盐商的崛起加速了人口的聚集。

近代，天津开放为[[通商口岸|通商口岸]]后，工商业依托[[天津租界|天津租界]]的外资和河海航运枢纽的区位优势工商业迅速繁荣，人口快速增长，在1840年天津城市人口为20万，至1880年已达到50万。清末民初，天津市由漕运枢纽迅速发展成为中国北方人口规模最大的城市、港口和工商业中心，至1935年，天津市总人口达到359.6万，其中城市人口151.5万。依据2013年天津的行政区划范围，天津市总人口达到了400万。根据[[第六次人口普查|第六次人口普查]]的数据显示，截止到2010年11月1日，天津市的常住人口总数为12938224人。<ref>{{cite web |url = http://news.enorth.com.cn/system/2011/04/29/006466166.shtml |title = 天津市公报2010年第六次全国人口普查主要数据 |publisher = 北方网 |accessdate = 2013-08-16 |deadurl = yes |archiveurl = https://web.archive.org/web/20131226001753/http://news.enorth.com.cn/system/2011/04/29/006466166.shtml |archivedate = 2013-12-26 }}</ref>

自进入21世纪以来，天津的人口结构存在严重的老龄化问题，是中国继[[上海市|上海市]]和[[北京市|北京市]]之后第三个进入老龄化社会的城市<ref>{{cite journal |title = 天津市人口老龄化发展趋势及对策研究 |author = 苗青芳 |journal=知识经济 |year=2012}}</ref><ref>{{cite web |url = http://finance.ifeng.com/news/hgjj/20090611/999907.shtml |title = 天津成为继北京和上海后全国第3个老龄化城市 |publisher=凤凰网 |accessdate=2013-08-17}}</ref>。

=== 文化地理特征 ===
天津是[[国家历史文化名城|国家历史文化名城]]。<ref>{{cite web|url=http://www.022net.com/2011/8-12/504669222935954.html|title=天津市历史文化名城名镇名村保护工作 获国家检查组充分肯定|publisher=人民网|accessdate=2013-08-17|deadurl=yes|archiveurl=https://archive.is/20130817141721/http://www.022net.com/2011/8-12/504669222935954.html|archivedate=2013-08-17}}</ref>

在历史上天津的军事地位显要，因此[[尚武文化|尚武文化]]是天津历史早期的文化标志。早在西汉时期，天津蓟县境内的[[渔阳镇|渔阳]]是西汉政府北部边防要塞。唐五代之后，天津地区又称为宋、辽对峙的前沿阵地，常年有军队驻守于此。<ref name="天津通志大事记">{{cite book | title=《天津地方志》《天津通志：大事记》 | publisher=天津市地方志编修委员会办公室 | year=2007年 | location=天津}}</ref>金、元时期，天津境内设置了直沽寨和海津镇，驻有众多军队。<ref name="天津通志大事记" />明永乐二年，天津设卫筑城后，大量屯兵，军事职能更加强化，军队的士兵以及士兵家眷成为早期天津城市居民的主体。<ref name="天津通志大事记" />

古代天津军事上的建制影响到了当时的[[天津市行政区划|行政区划]]、[[天津城市规划|城市建筑布局]]、乡村聚落的分布和军事管理体制。军队的屯垦制度形成了天津独特的地名文化，并促进了天津早期的农耕文化的发展。<ref name="天津地理" />天津的方言和婚俗等习俗均与周边河北省等地区不同，但与皖北地区一致<ref>{{cite web | url=http://news.enorth.com.cn/system/2011/08/29/007224883.shtml | title=安徽宿州婚俗与天津一样 婚礼上看“妈妈例儿” | publisher=北方网 | accessdate=2013-08-24}}</ref>，在北方形成了天津独特的[[天津话|方言孤岛]]和婚俗孤岛，都与早期军事移民影响有关。<ref>{{cite web | url=http://www.chinanews.com/cul/2011/07-13/3177313.shtml | title=天津方言赴皖寻根 固镇方言是天津“母语” | publisher=中国新闻网 | accessdate=2013-08-24}}</ref>

== 城市地理特征 ==
[[File:天津市总体战略示意图.jpg|天津市城镇体系]]
=== 城镇化和城镇体系 ===
天津的城市聚落自古因漕运而兴起，从古代的筑城到近代开埠专管租界的独立规划建设，再到1949年后的规划发展，天津已经成在中国的[[城镇体系|城镇体系]]中兼具中心城市和门户城市的职能。2010年2月，[[中华人民共和国住房和城乡建设部|中华人民共和国住房和城乡建设部]]编制的《全国城镇体系规划（草案）》中，天津市被确定为中国的[[国家中心城市|国家中心城市]]。<ref name="规划纲要">{{cite book | title=《全国城镇体系规划纲要（2005-2020年）》 | author=中国国家建设部 | location=北京 | year=2005年}}</ref>天津内部的城镇体系离散度较高，存在均衡性较低的问题。<ref>{{cite journal | title=天津市城市体系空间分布特征分析与研究 | author=丁鼎，杨滨盛，张修龙 | journal=安徽农业科技 |date=2009年3月}}</ref>天津内部城镇体系的核心已经逐渐从[[天津中心城区|天津中心城区]]的单一核心，逐步发展成为[[天津中心城区|天津中心城区]]-[[滨海新区|滨海新区核心区]]的双核心城镇体系。同时，天津市的门户港口由[[天津港|天津港]]的单一门户港口逐步发展成为，天津港-天津南港的双门户港口形态。这一城镇体系被成为“双城双港”的结构，“双城”是指[[天津市区|天津市中心城区]]和[[滨海新区|滨海新区]]核心区；“双港”是指[[天津港|天津港]]和[[天津南港工业区|天津南港]]。截止到2012年，天津的城镇化率达到81.55%，年均增长略高于1个百分点。<ref name="天津城镇化率达到81.55%">{{cite web | url=http://www.chinadaily.com.cn/hqpl/zggc/2013-04-04/content_8676610.html | title=天津城镇化率达到81.55% | publisher=中国日报 | accessdate=2013-08-23 }}{{Dead link|date=2019年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

=== 土地利用和规划 ===
天津由于近代[[天津租界|天津租界]]的开发，成为中国较早引入西方城市规划理念并进行发展和实践的城市。<ref name="天津市城市规划志">{{cite book | title=天津市城市规划志 | publisher=天津科学技术出版社 | author=天津城市规划志编纂委员会 | year=1994年 | location=天津 }}</ref>1984年，天津市已建成的约200平方公里，根据1986版天津市总体规划，天津市到2000年总用地规模为330平方公里，平均每人城市用地86.9平方米，人口密度降至每平方公里1.15万人。<ref name="天津市城市规划志" />

== 地理区划 ==
=== 综合自然区划 ===
根据地形地貌、土壤水文等条件，天津的综合自然地理区划共分为山地丘陵区、平原区、海岸带和浅海区四个区划。<ref name="天津地理" />山地丘陵区位于蓟县县城[[渔阳镇|渔阳镇]]以北，属于[[燕山|燕山]]山地南侧的一部分，面积651平方公里。这一区域海拔高度在50-1000米，地势北高南低。该区域地处夏季东南季风的迎风坡，[[地形雨|地形雨]]较多，降水量在700-900毫米，是天津市域内最多雨的地区。低山、丘陵、谷地是该区域的主要地貌类型。区域内的河流大部分是季节性河流，汛期时水流湍急，枯水期河床内分布着大小不一的乱世暴露于河床，河床的砂砾层仍有潜流。蓟县的山地丘陵区是天津的木材、矿产和建筑材料主产区，又是干鲜果品主产地和休闲度假风景旅游区。<ref name="天津通鉴" />天津市的平原区分布在山地丘陵以南的广大区域，属于华北平原的一部分，是天津市地形的主要特征，总面积为10654平方公里。天津市东部沿海地区有狭长的海岸带。天津沿海的平均高潮位为海拔高度1.14米，平均低潮位为海拔高度-1.14米，天津潮间带的宽度为3-7.3公里，最宽可达10公里，总面积为336平方公里。<ref name="天津地理" />

=== 农业区划 ===
[[File:Tian_jian_preserved_vegetable_天津冬菜.JPG|thumb]]]]

根据天津市地域内的地貌、气候、土壤、植被等的自然条件的差异，以满足城市需求为目标天津市的农业可分为都市型、滨海型和城郊型，据此天津市域的农业区划可以划分为都市型环城农业区、都市型滨海农业区、城郊型平原综合农业区和城郊型山区生态农业区四个农业区划。<ref name="天津“三型”农业区域布局与结构调整探讨">{{cite journal | title=天津“三型”农业区域布局与结构调整探讨 | author=李瑾 | journal=农业技术经济 |date=2000年4月}}</ref>亦有学者认为，滩涂和渤海大陆架也应属于天津市的独立的农业区划之一。<ref>{{cite book | title=天津农业地理 | publisher=天津科学技术出版社 | author=天津师范大学地理学 | year=1981 | location=天津}}</ref>

都市型环城农业区位于[[天津中心城区|天津市中心城区]]周边，实行多种经营，农林牧副渔工商逐渐一体化，形成了畜牧、蔬菜、果品等主导产业和水产、花卉、种苗等产业群，同时建设有一批以鲜活为特点的副食品生产基地。其中[[东丽区|东丽区]]以花卉和畜禽种业为特色，[[西青区|西青区]]以循环生态农业为特色，[[津南区|津南区]]以优质水稻为特色，[[北辰区|北辰区]]以无公害副食品和果品为特色。

都市型滨海农业区位于天津东部[[滨海新区|滨海地区]]，滩涂渔业区以水产养殖为主，滨海综合农业区以葡萄和冬枣为特色。<ref>{{cite web |url = http://news.163.com/12/1231/09/8K1TR91O00014JB6.html |title = 天津滨海新区实施农业与文化共同发展战略 |publisher=网易 |accessdate=2013-08-17}}</ref>

城郊型平原综合农业区位于[[宁河县|宁河县]]、[[静海县|静海县]]、[[武清区|武清区]]、[[宝坻区|宝坻区]]和[[蓟县|蓟县]]南部的平原地带。[[宁河县|宁河县]]以优质水稻、棉花、商品粮以及银鱼、紫蟹等水产品为特色，武清区的主要农副产品有小麦、玉米、面糊啊、瓜菜和鲜蛋等，牛奶和蔬菜是两大农业主导产业。[[静海县|静海县]]土地肥力偏低、缺磷，除了种植小麦、玉米和杂粮等，还有金丝小枣、棉花和向日葵等作物。[[蓟县|蓟县]]南部是天津主要的粮果畜产品主产地，[[宝坻区|宝坻区]]则是中国北方重要的粮棉生产基地，耕种面积达115.8平方公里，经济作物中尤其以“三辣”，即五叶齐大葱、红皮大蒜和天鹰椒为特色，可远销至海外。<ref>{{cite web|url=http://www.tj.gov.cn/zjtj/qurk/xzqh/200708/t20070822_1648.htm|title=宝坻区概况|publisher=天津政务网|accessdate=2013-08-17|deadurl=yes|archiveurl=https://web.archive.org/web/20130630004516/http://www.tj.gov.cn/zjtj/qurk/xzqh/200708/t20070822_1648.htm|archivedate=2013-06-30}}</ref>

=== 行政区划 ===
{{Main|天津市行政区划}}

天津市共有16个在民政部注册的县级行政区划单位，13个[[市辖区|市辖区]]（其中滨海新区为[[副省级区|副省级区]]），3个[[县|市辖县]]。市辖区分为中心城区、环城区和滨海新区。中心城区指[[和平区_(天津市)|和平区]]、[[河东区_(天津市)|河东区]]、[[河西区_(天津市)|河西区]]、[[南开区|南开区]]、[[红桥区|红桥区]]、[[河北区|河北区]]。按照服务业功能，中心城区按照金融和平、商务河西、科技南开、金贸河东、创意河北和商贸红桥的功能定位。<ref>{{cite web|url=http://www.022net.com/2010/10-24/506625343113478.html|title=天津发展新布局焕发巨大活力 推动三个层面联动协调发展|publisher=人民网|accessdate=2013-08-17|deadurl=yes|archiveurl=https://web.archive.org/web/20110928025335/http://www.022net.com/2010/10-24/506625343113478.html|archivedate=2011-09-28}}</ref>环绕市区的环城区为[[东丽区|东丽区]]、[[西青区|西青区]]、[[津南区|津南区]]、[[北辰区|北辰区]]、[[武清区|武清区]]和[[宝坻区|宝坻区]]。副省级区是[[滨海新区|滨海新区]]，由2009年11月时由原塘沽区、汉沽区、大港区以及天津经济技术开发区等区域整合而成。市辖县包括[[宁河县|宁河县]]、[[静海县|静海县]]和[[蓟县|蓟县]]。<ref>{{cite web|url=http://www.tj.gov.cn/zjtj/qurk/xzqh/|title=天津政务网之行政区划|publisher=天津政务网|accessdate=2013-08-17|deadurl=yes|archiveurl=https://web.archive.org/web/20130823031530/http://www.tj.gov.cn/zjtj/qurk/xzqh/|archivedate=2013-08-23}}</ref>

除此之外，天津市河东区还管辖位于河北省涉县的一块飞地，即[[天津铁厂街道|天津铁厂街道]]。其前身是1969年经国务院、中央军委批准，天津市以6985工程的名义在河北省涉县秘密建立一座钢铁厂。<ref>{{cite web| url=http://www.tiantie.com/jianjie_moban.jsp?BigClassID=1&SmallClassID=12&NewsID=354| title=天铁大事记| publisher=天津天铁冶金集团| accessdate=2013-08-24}}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

== 环境问题 ==

天津市存在淡水资源匮乏的资源型环境问题，人均占有量只有全国的1/15，特别是随着工业的快速发展，水资源供给更加紧张。<ref>{{cite journal | title=天津水资源承载力与经济协调发展研究 | author=王媛，徐利淼 | journal=天津师范大学学报(自然科学版) |date=2003年1月}}</ref>此外，天津市境内存在地表水质污染、大气污染、土壤污染等环境问题。

在地表水质污染方面，天津受到来自上游的输入型污染较为严重。特别是引滦入津水源地的污染严重<ref>{{cite web |url = http://news.hexun.com/2012-02-02/137668851.html |title = 引滦入津水源地遭鱼污染 天津恳请中央出面解决 |publisher = 和讯网 |accessdate = 2013-08-18}}</ref>。诸如，[[承德市|承德市]]的污水会直接排入滦河流域支流[[武烈河|武烈河]]，污染[[滦河|滦河]]水质。目前，承德市正在建设污水处理项目，以确保承德市排放的污水达标<ref>{{cite web |url = http://news.enorth.com.cn/system/2013/08/20/011242869.shtml |title = 承德市强力治污：让天津人民喝上安全可靠的水 |publisher = 北方网 |accessdate = 2013-08-21}}</ref>。

天津的空气污染问题同样较为严重，但根据研究显示近10年来总体处于稳定水平。其中，最主要的污染负荷为可吸入颗粒物PM10，其次二氧化硫与二氧化氮的污染由于冬季供暖，季节性规律较强，在冬季较为突出<ref>{{cite journal |title = 天津滨海新区秋冬季大气污染特征分析 |author = 王莉莉、王跃思等 |journal = 环境污染及其防治 |volume = 7|date=2011年7月}}</ref>。而可吸入颗粒物PM10中的PM2.5污染亦较为严重，平均质量浓度为223毫克每立方米，其中有机物、硫酸根等酸性阴离子和元素碳是含量较高的3种组分，污染主要来自机动车尾气和燃煤，海盐粒子则较少<ref>{{cite journal |title = 天津冬季大气中PM2.5及其主要组分的污染特征 |author = 李伟芳，白志鹏，魏静东，刘爱霞，赵玉洁，金陶胜 |journal = 中国环境科学 |date=2008年6月}}</ref>。特别是冬季集中供暖期间内当大气稳定度增加时，区域大气扩散能力随之减弱，天津大气污染概率增大、易发生雾霾天气<ref>{{cite journal |title = 采暖期间区域气象条件与天津大气污染概率关系 |author = 吴振玲、 周梁丹、 解以扬、孟冬梅、孙玫玲 |journal = 气象科技 |year = 2008}}</ref>。
