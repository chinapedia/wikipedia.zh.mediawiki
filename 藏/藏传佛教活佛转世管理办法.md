《'''藏传佛教活佛转世管理办法'''》，由'''国家宗教事务局令第5号'''公布，是[[中华人民共和国|中华人民共和国]][[国务院直属机构|国务院直属机构]][[国家宗教事务局|国家宗教事务局]]颁布的[[部门规章|部门规章]]<ref name="gov.cn">{{cite web|title=国家宗教事务局令（第5号）藏传佛教活佛转世管理办法|publisher=[[中华人民共和国中央政府|中华人民共和国中央政府]] |url = http://www.gov.cn/gongbao/content/2008/content_923053.htm |date=无日期 |accessdate=2014-05-04|language=zh-cn}}</ref>。该规章规定，所有[[活佛|活佛]]（藏语称“朱古”，意为“化身”<ref name=z10>《藏传佛教活佛转世管理办法》颁布10周年 http://www.chinanews.com/gn/2017/12-08/8395407.shtml</ref>）须由寺庙提交转世申请才能得到官方认可。该规章在2007年得到中国政府通过，并于同年9月1日生效<ref name="gov.cn" />。

== 背景 ==

[[藏传佛教|藏传佛教]]的信徒认为，[[喇嘛|喇嘛]]与其他宗教人物能够有意识地影响自己[[转世|转世]]的过程，并时常通过多次转世来利益眾生。

活佛转世制度乃藏传佛教之特色，始于13世纪。由于其影响及作用，中国历代王朝均对活佛相当重视。<ref name="tku"/>中国各时期政权对活佛转世制定了不同程度的政策与方案。[[乾隆皇帝|乾隆皇帝]]所著《[[喇嘛說|喇嘛說]]》中就有如下语句：

{{quote|蓋中外黃教<ref group="註">即藏傳佛教[[格鲁派|格鲁派]]，以黃帽聞名。</ref>，總司以此二人<ref group="註">指[[達賴|達賴]]与[[班禪|班禪]]。</ref>。各部蒙古，一心歸之。興黃教，即所以安眾蒙古，所系非小。<ref name="tku"/>}}

基于此，[[清廷|清廷]]采取了活佛管理制度，并在乾隆帝治下趋于完善。<ref name="tku"/>

[[乾隆_(年号)|乾隆]]五十八年（1793年），清廷颁布了管理西藏之行政法规《[[欽定藏內善後章程二十九條|欽定藏內善後章程二十九條]]》，其内容亦被写入《[[欽定理藩院則例|欽定理藩院則例]]》《[[大清會典|大清會典]]》与《[[大清會典事例|大清會典事例]]》。<ref name="tku"/>

民國二十二年（1933年），国民政府制定《十四世达赖喇嘛转世掣签征认办法》<ref name=z10/>。

民國二十四年（1935年）至二十七年（1938年）間，[[中華民國|中華民國]][[國民政府|國民政府]]相繼頒佈了《[[s:管理喇嘛寺廟條例|管理喇嘛寺廟條例]]》《[[s:喇嘛登記辦法|喇嘛登記辦法]]》《[[s:喇嘛任用辦法|喇嘛任用辦法]]》《[[s:喇嘛獎懲辦法|喇嘛獎懲辦法]]》與《[[s:喇嘛轉世辦法|喇嘛轉世辦法]]》等針對藏傳佛教僧人的法規，以規範寺廟管理和活佛轉世、僧人登記、任用與獎懲。<ref name="tku">{{cite thesis |author=廉湘民 |title=清以來中央政府管理藏傳佛教活佛轉世主要法規比較研究 |url=http://tibet.tku.edu.tw/upload/article/20090627161005-%E6%B8%85%E4%BB%A5%E4%BE%86%E4%B8%AD%E5%A4%AE%E6%94%BF%E5%BA%9C%E7%AE%A1%E7%90%86%E8%97%8F%E5%82%B3%E4%BD%9B%E6%95%99%E6%B4%BB%E4%BD%9B%E8%BD%89%E4%B8%96%E4%B8%BB%E8%A6%81%E6%B3%95%E8%A6%8F%E6%AF%94%E8%BC%83%E7%A0%94%E7%A9%B6.pdf |date=2009-06-27 |publisher=[[淡江大學|淡江大學]]西藏研究中心 |accessdate=2014-05-09|language=zh-tw}}</ref>

1937年，国民政府制定《十世班禅转世征认办法》<ref name=z10/>。

1989年8月19日，国务院批复西藏人民政府呈报的《关于寻访认定第十世班禅额尔德尼·确吉坚赞转世灵童的请示》<ref name = c>http://www.livingbuddha.cn/view-2ddfd29a3f0840adb101db62581247f9.html</ref>。

1991年2月，中共中央、国务院发布《[[s:中共中央、国务院关于进一步做好宗教工作若干问题的通知|中共中央、国务院关于进一步做好宗教工作若干问题的通知]]》（中发〔1991〕6号）<ref>{{cite web |title=【理论学习】《中共中央、国务院关于进一步做好宗教工作若干问题的通知》 |url=http://www.guit.edu.cn/dqsw/info/1013/1073.htm |accessdate=2019-03-21}}</ref>，中国政府开始批准活佛转世，至《藏传佛教活佛转世管理办法》颁布前，有超过1,000名[[西藏|西藏]]及中国其它地区的活佛的转世被批准<ref name=h>http://www.sara.gov.cn/xwfb/xwjj20170905093618359691/517007.htm</ref>。

1995年11月29日，国务院代表罗干宣读《国务院关于特准经金瓶掣签认定的坚赞诺布继任为第十一世班禅额尔德尼的批复》<ref name = c/>，完成了[[班禅额尔德尼·确吉杰布|第十世班禅转世灵童]]的寻访、认定及坐床<ref name=h/>。

2004年2月20日，根据蒙藏委员会会参法字第0930000312号令，《[[s:喇嘛轉世辦法|喇嘛转世办法]]》被废止。

2007年8月3日，中国[[国家宗教事务局|国家宗教事务局]]发布了《[[s:藏传佛教活佛转世管理办法|藏传佛教活佛转世管理办法]]》。

2016年，“藏传佛教活佛查询系统”正式上线<ref name=z10/>。

2017年，在西藏已经有60多位新活佛按照《藏传佛教活佛转世管理办法》认定转世<ref name=z10/>。

== 藏传佛教活佛转世管理办法 ==

在《藏传佛教活佛转世管理办法》中，[[藏传佛教|藏传佛教]][[活佛|活佛]]的所有转世行为都需要得到政府批准，否则会被视作“非法”或“无效”。该规章声称其旨在“规范活佛转世事务管理”，并规定“活佛转世应当遵循维护国家统一、维护民族团结”，“不受境外任何组织、个人的干涉和支配”。规章同样要求申请活佛转世的寺庙是“依法登记的藏传佛教活动场所，且具备培养和供养转世活佛的能力”。<ref name="gov.cn"/><ref name="Daily">{{cite news|url=http://www.chinadaily.com.cn/china/2007-08/04/content_5448242.htm|date=2007-08-04|title=Reincarnation of living Buddha needs gov't approval|trans_title=活佛转世须政府批准|publisher=[[中国日报|中国日报]]|accessdate=2007-08-09|language=en}}</ref>

转世申请须提交至县级政府宗教事务部门，而后逐级上报，由省级政府宗教事务部门审批。依据重要度与转世影响，申请可能会被要求提交至更高级的四个政府部门审批，由低到高为：省级政府宗教事务部门、省级政府、国家宗教事务局与[[中华人民共和国国务院|国务院]]。<ref name="Daily"/><ref name="Red"/><ref name="gov.cn"/>

规章的具体管理与落实分为十四条，包括理论、条件、许可过程、转世宗教团体的义务和责任与对违反规章行为的处罚。条款声称依法保障藏传佛教常规宗教活动、保护藏传佛教信徒的宗教信仰。<ref name="Daily"/>其中第十一条称，违反《办法》擅自办理活佛转世事宜将会遭到行政处罚；如构成犯罪，则会被追究刑事责任。<ref name="gov.cn"/>

国家宗教事务局称：“政府仅管理与国家和公众利益相关的宗教事务，并不干涉纯粹的内部宗教事务”。<ref name="Daily"/>

== 影响与评价 ==
祖古在中文里常被称作“[[活佛|活佛]]”（即“在世的佛陀”）。[[祖古|祖古]]是藏传佛教的重要部分，形成了具有影响力的宗教人物团体。他们的持续转世被认为是重新执掌其身份地位的过程。通常会有一名以上的候选者竞争最终得到实质上转世的名额，同时由当局来决定谁是真正有能力的申请人。<ref name="Red">{{cite news|url=http://www.theaustralian.com.au/archive/news/reincarnation-red-tape/story-e6frg6t6-1111114106139|publisher=《[[澳洲人报|澳洲人报]]》|title=Reincarnation red tape|trans_title=转世的繁文缛节|accessdate=2014-05-08|date=2007-08-04|language=en}}</ref>

2012年7月13日，藏人行政中央驻[[德里|德里]]办事处成立的活动小组组织200多名藏人手持[[雪山狮子旗|雪山狮子旗]]和标语在中国驻新德里大使馆门前游行示威，抗议[[中国共产党|中国共产党]]“破坏藏传佛教仪轨”并传达了流亡藏人对《办法》的“强烈反对”。随后抗议者还向大使馆递交了一份写给时任[[中国共产党|中国共产党]][[中国共产党中央委员会总书记|中央委员会总书记]][[胡锦涛|胡锦涛]]的请愿信，要求尊重藏人信仰，撤销《藏传佛教活佛转世管理办法》。<ref>{{cite web|author=|title=新德里藏人游行示威抗议中共干涉宗教|url=http://www.vot.org/?p=13623|publisher=西藏之声|date=2012-07-13|accessdate=2014-05-09|language=zh-hans}}</ref>
