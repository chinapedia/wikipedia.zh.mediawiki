《'''乐翻天'''》（{{lang-en|'''Happy Hotel'''}}）导演和编剧都是[[王岳伦|王岳伦]]，主演[[姜武|姜武]]、[[宁静|宁静]]。<ref>{{cite web |url=http://ent.sina.com.cn/m/2012-06-29/23223672255.shtml |title=资料：电影《乐翻天》故事梗概 |accessdate=2012 |publisher=新浪网 |work= |language=zh-hans}}</ref>

== 剧情 ==
该片讲述一对夫妻和一个来中国大陆讨账的韩国小混混之间的搞笑故事。

==花絮==
该片由王岳伦担任导演和编剧，王岳伦妻子[[李湘_(主持人)|李湘]]担任制片人，李湘好友[[杜海涛|杜海涛]]友情客串。<ref>{{cite web |url=http://ent.ifeng.com/movie/news/mainland/detail_2012_07/09/15878608_0.shtml |title=《乐翻天》8月2日上映 李湘：票房不超5千万卖火锅 |accessdate=2012 |publisher=凤凰网 |work= |language=zh-hans}}</ref>
