《'''乐府诗集'''》是一部专门录辑[[汉代|汉代]]以迄[[唐五代|唐五代]]乐府诗的诗歌总集，也编入了部分汉以前流传的古歌辞，<ref>{{cite web |url=https://books.google.co.kr/books?id=SDKpYgfP5xUC&printsec=frontcover&dq=%E4%B9%90%E5%BA%9C%E8%AF%97%E9%9B%86&hl=zh-CN&sa=X&ved=0ahUKEwix0JrT45LfAhUHhrwKHQAcBAkQ6AEIJjAA#v=onepage&q=%E4%B9%90%E5%BA%9C%E8%AF%97%E9%9B%86&f=false|title=乐府诗集|accessdate=|work=|publisher=Google图书|language=}}</ref>为[[宋代|宋代]][[郭茂倩|郭茂倩]]编纂。<ref>{{cite web |url=http://history.people.com.cn/n1/2018/0426/c372329-29952367.html|title=李白《静夜思》在流传过程中有哪些讹误？|accessdate=2018年4月26日|work=|publisher=人民网|language=}}</ref>《樂府詩集》是一部较為完備的樂府詩總集，收集漢至唐五代時期的樂府民歌、文人作品和先秦歌謠，按其曲調分為十二類，成為早期研究樂府的重要文獻。《樂府詩集》曾被清初藏书大家[[徐乾学|徐乾学]]、[[季振宜|季振宜]]等收藏，1918年被近代学者[[傅增湘|傅增湘]]所得。20世纪三四十年代，傅增湘将书重新装裱。<ref name="人民网">{{cite web |url=http://paper.people.com.cn/rmrbhwb/html/2007-12/03/content_33656150.htm|title=宋刻本《乐府诗集》（善本掌故）|accessdate=2007-12-03 |work=|publisher=人民网|language=}}</ref>

==诗集简介==
《乐府诗集》集录了[[先秦|先秦]]歌谣，[[两汉|汉朝]]至[[唐朝|唐]][[五代|五代]]的[[乐府诗|乐府诗]]，全书共一百卷。把乐府诗分为郊庙歌辞（12卷）、燕射歌辞（3卷）、鼓吹曲辞（5卷）、横吹曲辞（5卷）、相和歌辞（18卷）、清商曲辞（8卷）、舞曲歌辞（5卷）、琴曲歌辞（4卷，5曲、9引、12操）、杂曲歌辞（18卷）、近代曲辞（4卷）、杂歌谣辞（7卷）以及新乐府辞（12卷）等12大类。<ref>{{cite web |url=https://books.google.com/books?id=leZyDwAAQBAJ&pg=PA293&lpg=PA293&dq=%E3%80%8A%E4%B9%90%E5%BA%9C%E8%AF%97%E9%9B%86%E3%80%8B%E6%8A%8A%E4%B9%90%E5%BA%9C%E8%AF%97%E5%88%86%E4%B8%BA%E9%83%8A%E5%BA%99%E6%AD%8C%E8%BE%9E%EF%BC%8C%E7%87%95%E5%B0%84%E6%AD%8C%E8%BE%9E%EF%BC%8C%E9%BC%93%E5%90%B9%E6%9B%B2%E8%BE%9E%EF%BC%8C%E6%A8%AA%E5%90%B9%E6%9B%B2%E8%BE%9E%EF%BC%8C%E7%9B%B8%E5%92%8C%E6%AD%8C%E8%BE%9E%EF%BC%8C%E6%B8%85%E5%95%86%E6%9B%B2%E8%BE%9E%EF%BC%8C%E8%88%9E%E6%9B%B2%E6%AD%8C%E8%BE%9E%EF%BC%8C%E7%90%B4%E6%9B%B2%E6%AD%8C%E8%BE%9E%EF%BC%8C%E6%9D%82%E6%9B%B2%E6%AD%8C%E8%BE%9E%EF%BC%8C%E8%BF%91%E4%BB%A3%E6%9B%B2%E8%BE%9E%EF%BC%8C%E6%9D%82%E6%AD%8C%E8%B0%A3%E8%BE%9E%E5%92%8C%E6%96%B0%E4%B9%90%E5%BA%9C%E8%BE%9E%E7%AD%8912%E5%A4%A7%E7%B1%BB&source=bl&ots=Refqw2tsmM&sig=ljGPiMq5YfpBWo_CiBbQqMsuYEk&hl=zh-CN&sa=X&ved=2ahUKEwji5aCm7JLfAhUEvLwKHaJwBKsQ6AEwBXoECAIQAQ#v=onepage&q=%E3%80%8A%E4%B9%90%E5%BA%9C%E8%AF%97%E9%9B%86%E3%80%8B%E6%8A%8A%E4%B9%90%E5%BA%9C%E8%AF%97%E5%88%86%E4%B8%BA%E9%83%8A%E5%BA%99%E6%AD%8C%E8%BE%9E%EF%BC%8C%E7%87%95%E5%B0%84%E6%AD%8C%E8%BE%9E%EF%BC%8C%E9%BC%93%E5%90%B9%E6%9B%B2%E8%BE%9E%EF%BC%8C%E6%A8%AA%E5%90%B9%E6%9B%B2%E8%BE%9E%EF%BC%8C%E7%9B%B8%E5%92%8C%E6%AD%8C%E8%BE%9E%EF%BC%8C%E6%B8%85%E5%95%86%E6%9B%B2%E8%BE%9E%EF%BC%8C%E8%88%9E%E6%9B%B2%E6%AD%8C%E8%BE%9E%EF%BC%8C%E7%90%B4%E6%9B%B2%E6%AD%8C%E8%BE%9E%EF%BC%8C%E6%9D%82%E6%9B%B2%E6%AD%8C%E8%BE%9E%EF%BC%8C%E8%BF%91%E4%BB%A3%E6%9B%B2%E8%BE%9E%EF%BC%8C%E6%9D%82%E6%AD%8C%E8%B0%A3%E8%BE%9E%E5%92%8C%E6%96%B0%E4%B9%90%E5%BA%9C%E8%BE%9E%E7%AD%8912%E5%A4%A7%E7%B1%BB&f=false|title=人文百科大全集|accessdate=|work=|publisher=Google图书|language=}}</ref>
