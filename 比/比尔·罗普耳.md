{{Vgname2|比尔·罗普耳|en=Bill Roper|{{bd|1965年|3月27日}}}}，出生于[[加利福尼亚州|加利福尼亚州]][[康科德|康科德]]，是一位[[游戏设计师|游戏设计师]]。自1994年加入[[暴雪娱乐|暴雪娱乐]]后在世界范围内广为人知<ref>{{cite web |url=http://www.pcgames.com.cn/news/cyda/guoji/0709/944088.html |title=Bill Roper简介 |publisher=[http://www.pcgames.com.cn 太平洋游戏网] |accessdate=2009-08-11}}</ref>，他曾在多个游戏工作室和创意开发部门工作（包括[[暴雪娱乐|暴雪娱乐]]、[[旗舰工作室|旗舰工作室]]）。从2011年6月至今为迪斯尼互动工作室的副总裁/总经理<ref>[http://www.giantbomb.com/bill-roper/3040-3444/ Roper biography]</ref>。此外他也是一位音乐家，他是爱尔兰民谣乐队The Poxy Boggards的创始成员之一。

== 暴雪娱乐 ==

在1994年至2003年间，罗普耳曾担任[[暴雪北方|暴雪北方]]的副总裁和[[暴雪娱乐|暴雪娱乐]]的董事。罗普耳负责监督管理了所有的游戏计划，协调内部的开发团队，领导[[暴雪娱乐|暴雪娱乐]]的计划监督团队，对暴雪游戏的指导起决定作用。

比尔·罗普耳在暴雪娱乐各大主要游戏的开发中皆担任了监制的位置，并对《[[暗黑破坏神_(游戏)|暗黑破坏神]]》、《[[星際爭霸系列|星际争霸]]》和《[[魔兽争霸系列|魔兽争霸]]》几部游戏的成功起到了关键作用，并被誉为“暗黑破坏神之父”<ref>{{cite web |url=http://www.gamespot.com.cn/news/2009/0317/96035.shtml |title=暗黑之父Bill Roper |publisher=[[GameSpot|GameSpot]] |accessdate=2009-08-13 }}{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。他同样参与指导了暴雪游戏的配音工作，包括以个人名义向全球的游戏社区介绍新游戏。

== 旗舰工作室 ==

罗普耳离开[[暴雪娱乐|暴雪娱乐]]后，他创立了[[旗舰工作室|旗舰工作室]]并担任[[CEO|CEO]]的工作。这一工作室开发了游戏《[[暗黑之门：伦敦|暗黑之门：伦敦]]》（但在发行之初评价很低<ref>{{en}}{{cite web|url=http://www.metacritic.com/games/platforms/pc/hellgatelondon?q=Hellgate |title=暗黑之门：伦敦 (pc: 2007): 评测 |publisher=Metacritic.com |date=2007-10-30 |accessdate=2009-03-09}}</ref><ref>{{en}}{{cite web|url=http://www.1up.com/do/feature?cId=3166996 |title=5部最差游戏 |publisher=1up.com |date= |accessdate=2009-03-09}}</ref><ref>{{en}}{{cite web|url=http://www.g4tv.com/thefeed/blog/post/682534/The_MMO_Report_Friday_February_1st.html?detectflash=false& |title=MMO报道：2月1日星期五 |publisher=G4tv.com |date= |accessdate=2009-03-09}}</ref>）和《[[神话_(游戏)|神话]]》（根本没能发行）。

尽管公司内部出现解雇的传闻<ref>{{en}}{{cite web|url=http://www.flagshipped.com/2008/06/05/fss-senior-programmer-speaks-out |title=FSS高级开发师发言 |publisher=Flagshipped.com |accessdate=2009-03-09 |deadurl=yes |archiveurl=https://web.archive.org/web/20081228013418/http://www.flagshipped.com/2008/06/05/fss-senior-programmer-speaks-out |archivedate=2008-12-28 }}</ref>，但在随后公司声明此事子虚乌有<ref>{{cite web |url=http://forums.hellgatelondon.com/showthread.php?t=101425 |title=存档副本 |accessdate=2009-08-14 |deadurl=yes |archiveurl=https://web.archive.org/web/20090125191611/http://forums.hellgatelondon.com/showthread.php?t=101425 |archivedate=2009-01-25 }}</ref>。公司先后将《神话》和《暗黑之门》的权利转交给了[[HanbitSoft|HanbitSoft]]和[[Comerica|Comerica]]<ref>{{en}}{{cite web |url=http://www.1up.com/do/feature?pager.offset=0&cId=3169356 |title=对旗舰工作室比尔·罗普耳的采访 |publisher=1up.com |accessdate=2009-03-09 |deadurl=yes |archiveurl=https://archive.is/20120717152546/http://www.1up.com/do/feature?pager.offset=0&cId=3169356 |archivedate=2012-07-17 }}</ref>；而亚洲市场的合作伙伴和银行则将其知识产权作为用于寻找新投资商的贷款的附属担保品<ref>{{en}}{{cite web|last=Linde |first=Aaron |url=http://www.shacknews.com/laryn.x?story=51935 |title=评论—《暗黑之门：伦敦》知识产权被旗舰工作室作为贷款附属担保品 |publisher=Shacknews |date=2008-03-26 |accessdate=2009-03-09 |deadurl=yes |archiveurl=https://web.archive.org/web/20080731080949/http://www.shacknews.com/laryn.x?story=51935 |archivedate=2008-07-31 }}</ref>。目前，《神话》和《暗黑之门》的知识产权已分别转交至HanbitSoft和Comerica。

== 晦涩工作室 ==

2008年11月罗普耳担任了[[晦涩工作室|晦涩工作室]]最新游戏《{{link-en|勇士OL|Champions Online}}》的设计师和执行监制<ref>{{en}}{{cite web|url=http://www.champions-online.com/node/154 |title=晦涩工作室雇佣业界老手比尔·罗普耳 |publisher=Champions-online.com |date=2008-11-03 |accessdate=2009-03-09}}</ref>。在2009年的纽约动漫展（New York Comicon 2009）上，罗普耳告诉读者他“像个小孩一样玩《勇士》”，而他为《勇士OL》测试版所作的贡献使他在公司担任了一个正式职位<ref>{{en}}[http://www.tentonhammer.com/node/63995 2009年纽约动漫展：晦涩工作室]</ref>。

== 音乐表演 ==

自1994年，罗普耳在[[文艺复兴游行|加利福尼亚文艺复兴节]]中表演“黑船长”的角色。他是[[:en:Poxy_Boggards|Poxy Boggards]]，一个民歌乐队的作曲家和联合创立者之一。他作为乐队的男中音和男高音。在最新的专辑《[http://www.poxyboggards.com/?page_id=4 Anchor Management]{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}》中，他还演奏了[[萨克斯|萨克斯]]、[[大号|大号]]和[[六孔小笛|六孔小笛]]<ref>{{en}}[http://www.poxyboggards.com/?page_id=2 Poxy Boggards 乐队成员]</ref>。

== 荣誉 ==

在[[IGN|IGN]]的“史上百大游戏设计师”中，罗普耳名列第41位<ref>{{en}}[http://games.ign.com/top-100-game-creators/41.html Bill Roper: 最佳游戏设计师] {{webarchive|url=https://web.archive.org/web/20090306125006/http://games.ign.com/top-100-game-creators/41.html |date=2009-03-06 }}</ref>。

== 游戏作品 ==

* 《[[暗黑破坏神II|暗黑破坏神II]]》 (2000) - 总监制、配音、突击队
