'''比基尼环礁'''（{{lang-en|'''Bikini Atoll'''}}，亦作{{lang|en|'''Pikinni Atoll'''}}），{{IPAc-en|ˈ|b|ɪ|k|ᵻ|ˌ|n|iː}}或 {{IPAc-en|b|ᵻ|ˈ|k|iː|n|i}}; [[马绍尔语|马绍尔语]]: '{{lang|mh|Pikinni}}', 是屬于[[马绍尔群岛|马绍尔群岛]]国的一个[[堡礁|堡礁]]，由23個小島環繞着一個面積達229.4[[平方英里|平方英里]]（594.1 km<sup>2</sup>）的[[潟湖|潟湖]]組成。[[美国|美国]]从1946年到1958年在[[马绍尔群岛|马绍尔群岛]]共进行了20多次[[原子弹|原子弹]]和[[氢弹|氢弹]]的爆炸。2010年7月31日被列入[[世界文化遺產|世界文化遺產]]，是該國第一個世界遺產。<ref name="">{{cite news|publisher=[[聯合國教科文組織|聯合國教科文組織]]世界遺產中心

==命名==

比基尼環礁的英文名({{lang|en|Bikini}})是由[[德屬新幾內亞|德屬新幾內亞]]時期德國的命名({{lang|de|Bikini}})而來的。而德國命名是由當地人對其的稱呼，「{{lang|mh|Pikinni}}」所命名的。馬紹爾語中，''Pik''的意思是陸地，''Ni''則是椰子，因此其島名在馬紹爾語中的意思是「椰子島」<ref name=turssel2>{{cite web|url=http://www.trussel2.com/MOD/LocP.htm#Pikinni|title=Marshallese-English Dictionary-Place Name Index|accessdate=8 August 2013|deadurl=no|archiveurl=https://web.archive.org/web/20121027150952/http://www.trussel2.com/MOD/LocP.htm#Pikinni|archivedate=27 October 2012}}</ref>。

比基尼环礁被德國航海家及探險家[[奥托·冯·科策布|Otto von Kotzebue]]發現之時，本來被命名為'''Eschscholtz环礁'''，以紀念[[波罗的海德国人|波罗的海德国裔]]科學家{{link-en|Johann Friedrich von Eschscholtz|Johann Friedrich von Eschscholtz}}。

== 文化影響 ==
{{参见|十字路口行动}}
[[File:Crossroads_Baker_(wide-zoom).jpg|thumb]]]]
[[美国|美国]]从1946年到1958年在[[马绍尔群岛|马绍尔群岛]]共进行了20多次[[原子弹|原子弹]]和[[氢弹|氢弹]]的爆炸。與此同時，時裝界在當時亦同時發明了一種和[[女用內衣|女用內衣]]相差無幾的[[泳衣|泳衣]]，因为这种泳衣相当暴露，完全突破当时人们的传统思想底线，发明者认为其影响力无异于一次核爆，故取名於這次在比基尼环礁的核試，叫作“[[比基尼泳衣|比基尼泳衣]]”。
