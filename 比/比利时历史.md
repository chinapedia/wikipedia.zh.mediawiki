{{比利时历史}}
无论是[[地理|地理]]上还是[[文化|文化]]上，[[比利时|比利时]]都处于[[欧洲|欧洲]]的十字路口，在过去的2000年内，其见证了各种种族与文化的兴盛与衰败。也正因为这样，比利时是欧洲真正的种族熔炉，[[凯尔特人|凯尔特人]]、[[罗马人|罗马人]]、[[德意志人|德意志人]]、[[法国人|法国人]]、[[荷兰人|荷兰人]]、[[西班牙人|西班牙人]]和[[奥地利人|奥地利人]]在此都留下了文化的痕迹。

== 史前時代 ==
[[Image:Belgian_flint_knives.jpg|thumb]]
在比利時地區曾經發現公元前1750石器時代的活動遺址，例如最初使用的錢幣；而此人種也是廣義日耳曼民族的一支，羅馬共和國的軍事家凱薩在其著作高盧戰記中，就曾敘述該地的原住民的生活狀況。

== 早期历史（1830年前） ==
=== 羅馬帝國時期 ===
{{main|比利时高卢}}
前54年，本地区被[[罗马共和国|罗马共和国]]将军[[凯撒|凯撒]]征服，当时主要由凯尔特人居住。公元一世纪罗马人建立[[比利时高卢|高卢比利时行省]]，其领域涵盖了当今比利时、荷兰、卢森堡以及德国西部、法国东北部一部分。行政中心在[[兰斯|兰斯]]。
=== 黑暗時期 ===
{{main|佛蘭德伯國}}
公元5世纪[[罗马帝国|罗马帝国]]崩溃后，[[日耳曼人|日耳曼人]]大举入侵，其中的一支[[法兰克人|法兰克人]]随后建立了[[墨洛温王朝|墨洛温王朝]]，其领土包括了现在的比利时。墨洛温王朝之后由[[卡洛林王朝|卡洛林王朝]]取代。在今比利时地区，形成了'''佛兰德伯国'''。

=== 中世紀 ===
公元843年，[[卡洛林王朝|卡洛林王朝]][[查理曼|查理曼]]的孫輩们簽署了《[[凡爾登條約|凡爾登條約]]》，將統一帝國瓜分為東、中和西三個部分。[[佛兰德伯国|佛兰德伯国]]的[[斯海尔德河|斯海尔德河]]以西部分归属[[西法兰克|西法兰克]]，以东部分归属[[東法蘭克|東法蘭克]]。这也是当今[[比利时法语区|比利时法语区]]与[[比利时荷语区|比利时荷语区]]分别的由来。

1405年，佛兰德领地女继承人[[玛格丽特三世_(佛兰德)|玛格丽特三世]]与[[勃艮第|勃艮第]]公爵[[菲利普二世_(勃艮第)|菲利普二世]]结婚，佛兰德遂并入[[勃艮第公国|勃艮第公国]]。到1433年的时候，比利时以及卢森堡、大部分荷兰地区是[[勃艮第公国|勃艮第公国]]的一部分，后来因勃艮第女公爵玛丽嫁给[[马克西米利安一世_(神圣罗马帝国)|马克西米利安一世]]，这一地区从而变成[[奧地利哈布斯堡王朝|奧地利哈布斯堡王朝]]的领土。以后又因[[哈布斯堡王朝|哈布斯堡王朝]]的联姻与分家，这一地区又被[[西班牙哈布斯堡王朝|西班牙哈布斯堡王朝]]统治。但在[[西班牙王位繼承戰爭|西班牙王位繼承戰爭]]後，[[奧地利|奧地利]]取得了比利時。

=== 法據時期 ===
1789-1790年间的比利时革命与1789年的[[法國大革命|法國大革命]]在时间上交叠，该运动寻求从奥地利获得独立。在Jean-André van der Mersch的指挥下，Brabant起义军击败了奧地利军队，尼德蘭地區宣布獨立，并成立比利时合众国。但旋即在該年又為奧地利人所重新領有。1794至1795年間南尼德蘭遭到[[法蘭西第一共和國|法蘭西第一共和國]]進攻與占領，與此同時結束了哈布斯堡的統治，他們被切割成九個行政區成為法國的一部分，在1797年奧地利正式簽署條約把該地割讓給法國。

1797年一件具有重要意義的事件爆發：[[魯汶大學|魯汶大學]]遭到關閉，在[[法國|法國]]最初的幾年統治下來，[[比利時|比利時]]地區經濟完全受到剝削，連安特衛普這個港口都被禁止對外貿易，人民只被允許購買法國高昂的商品，居高不下的稅率，使市場只信賴[[黃金|黃金]]和[[白銀|白銀]]，能用的[[貨幣|貨幣]]只有金、[[銀幣|銀幣]]，民不聊生。這段期間約有八十萬比利時人出走。<ref>{{en}} {{cite web |author=Alexander Ganse |publisher=[[Korean_Minjok_Leadership_Academy|Korean Minjok Leadership Academy]] |url=http://www.zum.de/whkmla/region/lowcountries/bel17951799.html |title=Belgium under French Administration, 1795-1799 |accessdate=2008-04-03}}</ref>在法據時期[[荷蘭語|荷蘭語]]不許公開使用，種下今日荷語區和法語區不合的前因，<ref name="ulaval">{{fr}} {{cite web|url=http://www.tlfq.ulaval.ca/axl/europe/belgiqueetat_histoire.htm |title=Petite histoire de la Belgique et ses conséquences linguistiques |author=Jacques Leclerc (associated member of the [[Trésor_de_la_langue_française_au_Québec|Trésor de la langue française au Québec]]) |work=L'aménagement linguistique dans le monde |publisher=[[Université_Laval|Université Laval]] |date=2008-11-09 |accessdate=2009-01-16 |deadurl=yes |archiveurl=https://web.archive.org/web/20090118013123/http://www.tlfq.ulaval.ca/AXL/Europe/belgiqueetat_histoire.htm |archivedate=2009-01-18 }}</ref><ref name="digitaal">{{nl}} {{cite web |url=http://www.digitaalbrussel.be/thema/toerisme/geschiedenis/franseov.asp |title=De Franse overheersing (1792 - 1814年[[拿破崙一世|拿破崙一世]]迫於[[反法聯盟|反法聯盟]]的力量而退位，法國對比利時的統治也旋即結束，但在隔年拿破崙試圖返回法國重掌政權，拿破崙知道他僅有一次反敗為勝的機會，他必須在比利時地區和聯盟一決高下，同年六月十六日他率軍穿越比利時地區。

[[Image:Waterloo_Lion.jpg|thumb]]
在同年拿破崙的最後一戰於比利時開打，他遭遇到[[普魯士|普魯士]]與[[英國|英國]]的軍隊圍攻，在六月十八日拿破崙終於在比利時的[[滑鐵盧戰役|滑鐵盧]]被擊敗，拿破崙可以說是敗在自身的策略以及部屬未能來的即前來救援，次日早晨最後的戰鬥也結束，拿破崙受到再次流放的命運。當時[[荷蘭國王|荷蘭國王]][[威廉一世_(尼德蘭)|威廉一世]]的奧倫治王子在戰場有功，這位王子也就是後來的[[威廉二世_(尼德蘭)|威廉二世]]，在1826年年輕的威廉率領的荷蘭軍再次於[[滑鐵盧|滑鐵盧]]打了勝戰，而擊敗了比利時的軍隊。

===荷據時期===
[[File:欧洲_-_1815.svg|300px]]
1815年拿破崙敗於滑鐵盧時，主要的參與國有英國、普魯士以及[[俄羅斯帝國|俄國]]，這幾個國家在戰後於維也納召開和會，也就是著名的「[[維也納會議|維也納會議]]」，會中確立奧屬尼德蘭和尼德蘭共和國共組為荷蘭聯合王國，此舉主要是懼於奧屬尼德蘭留在法國會增加法國的國力，他們決定拥戴一位信奉新教的國王，也就是時任奧倫治親王的[[威廉一世_(尼德蘭)|威廉一世]]。

== 独立 ==
{{see also|布魯塞爾法語化}}
1831年，比利时爆发[[比利時獨立|比利時獨立]]運動，从荷兰统治下独立，成为[[世袭|世袭]][[君主立宪制|君主立宪制]]王国，并选择了一位[[德国|德国]][[貴族|貴族]]，[[萨克森-科堡-哥达王朝|萨克森-科堡-哥达公国]]的王子[[利奥波德一世_(比利时)|利奥波德]]作為比利时的第一任国王。

比利时立国后奉行中立政策，但在两次世界大战中都被德国占领。二战以后，比利时放弃中立原则，参加了[[北约|北约]]。后来又参加了[[欧洲经济共同体|欧洲经济共同体]]。

== 殖民主义时代 ==
比利时历史上有一个主要的[[殖民地|殖民地]]：[[比属刚果|比属刚果]]，即现在的[[刚果民主共和国|刚果民主共和国]]。比属刚果于1885年由[[柏林会议|柏林会议]]贈给当时的比利时国王[[利奥波德二世_(比利时)|利奥波德二世]]。利奥波德将此地命名为[[刚果自由邦|刚果自由邦]]，并收为自己的[[剛果自由邦|私人领地]]，进行残酷的统治，同时大量种植[[橡胶|橡胶]]，以满足当时世界对[[輪胎|橡胶轮胎]]的需求。

在国际舆论的压力下，利奥波德于1908年放弃了对刚果的私人统治，把刚果交予比利时政府。后来比属刚果于1960年独立。

1902年比利时在中国[[天津|天津]]获得一块[[天津比租界|租界]]<ref>《中外旧约章汇编》第2册，第34页</ref>，1929年声明退还中国，1931年正式举行仪式退还中国。

== 第一次世界大战 ==
1914年[[第一次世界大战|第一次世界大战]]，8月3日德国对法宣战，同时占领宣布中立的比利时，由此来进攻法国。8月4日英国因德国对比利时宣战，随即向德国宣战。

== 第二次世界大战 ==
在[[第二次世界大战|第二次世界大战]]战前，比利时试图寻找保持不结盟的中立政策。但是在1940年5月10日，德军入侵比利时，由此进攻法国。

在1940年比利时被打败后，众多比利时战士和平民设法躲避至英国，并且加入了流亡比利时政府。

== 战后比利时 ==
在过去的半个世纪中比利时作为一个现代化、高科技的欧洲国家而兴盛，它同时也是[[北约|北约]]和[[欧盟|欧盟]]的成员国。但是北部说荷兰语的弗拉芒人和南部讲法语的[[瓦隆|瓦隆]]人之间长期的紧张关系，最终导致了近些年来的[[宪法|宪法]]修正案，赋予比利时南北两半正式的自治权，因此比利时现在是一个[[联邦|联邦]]国家。

2007年6月10日，举行联邦议会选举。<ref>吴黎明，[http://news.xinhuanet.com/world/2007-06/10/content_6224201.htm 比利时举行大选]，新华网</ref>

== 現代 ==
2010年[[比利时议会|比利时议会]]选举结果显示，分裂派政党“新联盟党”（N-VA）在[[弗拉芒|弗拉芒]]处于领先地位。而该政党终极目的是与南部的法语区彻底决裂。而荷兰语党派与法语党派对于组建联合政府无法调节，而导致2010年到2011年比利时陷入了[[2010-2011年比利時政府難產|长期的无政府状态]]。
