'''比斯開橋'''（{{lang-es|puente de Vizcaya}}<ref>{{Cite book |title=Santuarios marianos de Álava, Guipúzcoa y Vizcaya |author=José Agustín Elustondo, Antonio Villarejo Garaizar |url=https://books.google.com/books?isbn=8474905559 |year=2000 |isbn=8474905559 |pages=p. 346|language=es}}</ref>，{{lang-eu|Bizkaiko Zubia}}<ref>{{Cite web |url=https://puente-colgante.com/?lang=eu |title=Bizkaia Zubia webgunea |accessdate=2018-11-03|language=eu}}</ref>）位于[[西班牙|西班牙]][[比斯開省|比斯開省]]，橋長164米，橫跨[[內維翁河|內維翁河]]的入海口，連接[[波尔图加莱特|波尔图加莱特]]和[[格乔|格乔]]的[[阿雷拿斯|阿雷拿斯]]区（{{lang|es|Las Arenas}}）。此橋建成於1893年，是世界上最古老的[[運渡橋|運渡橋]]，由工程師帕拉西歐(Alberto Palacio)設計建造，造橋工程於1890年4月10開始，1893年7月28日<ref>{{Cite web |url=http://www.puente-colgante.com/index.php/es/puente-vizcaya/historia.html |title=Puente Bizkaia - Historia |accessdate=2018-11-03 |language=es |archive-url=https://web.archive.org/web/20161122155234/http://www.puente-colgante.com/index.php/es/puente-vizcaya/historia.html |archive-date=2016-11-22 |dead-url=yes }}</ref>完工，只用了3年時間。2006年，它被[[联合国教科文组织|联合国教科文组织]]列入[[世界遺產|世界遺產]]。

==概要==
比斯开桥是世界上第一座供行人和车辆通过的高空[[運渡橋|運渡橋]]，[[欧洲|欧洲]]、[[非洲|非洲]]和南、北美洲的很多大桥都是仿照该桥建造的，不过保存至今的为数不多。由于别出心裁地使用了螺纹钢筋轻质技术，比斯开桥被誉为[[工业革命|工业革命]]时代最杰出的[[钢铁|钢铁]]桥梁建筑之一。<ref>{{Cite web |url=http://whc.unesco.org/zh/list/1217 |title=比斯開橋 |accessdate=2018-11-03|language=zh}}</ref>
