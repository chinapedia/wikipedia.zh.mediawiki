'''比尔森啤酒'''，常常被简称为'''比尔森'''（德语：'''pils'''，英语：'''pilsener'''），也叫'''比尔森式啤酒'''，一般指的是使用[[比尔森式酿造法|比尔森式酿造法]]制出的[[啤酒|啤酒]]。它源自[[捷克|捷克]][[比尔森|比尔森]]市，是一种使用[[下层发酵法|下层发酵法]]，[[浅色麦芽|浅色麦芽]]和苦味较重的[[啤酒花|啤酒花]]酿造的[[窖藏啤酒|窖藏啤酒]]（lager [[拉格|拉格]]啤酒），[[麦汁浓度|麦汁浓度]]最高不超过12.5度，酒精度多在5%以下。

== 历史 ==
一直到十九世纪四十年代，绝大多数的捷克啤酒使用的还是较原始的[[上层发酵法|上层发酵法]]，酿制出的啤酒一般既暗淡又混浊，且产品的口感往往很不稳定，使顾客们经常对酒的质量抱怨不已。到了1839年，[[比尔森|比尔森]]市的市民们终于对这种状况忍无可忍，他们决定集资建立一座他们自己的酿酒厂：“市民酿酒厂”。新酒厂计划采用的是源自[[德国|德国]][[巴伐利亚|巴伐利亚]]的先进的下层发酵法，它将使啤酒的清澈度，香味和保鲜期获得很大的提升。为此酒厂专门聘请了巴伐利亚著名酿酒师[[约瑟夫·格罗尔|约瑟夫·格罗尔]]（1813-1887）来设计、研究新啤酒的制作工艺，后者最后大胆采用了当时新诞生不久的[[浅色麦芽|浅色麦芽]]，使得啤酒呈现出日后成为比尔森啤酒标志的金黄色。1842年10月5日，第一桶比尔森啤酒终于面世。新式麦芽带来的迷人金黄色泽，比尔森市引以为傲的甘甜水源，左近[[萨切克市|萨切克市]]的上等[[啤酒花|啤酒花]]和当时最先进的巴伐利亚[[窖藏啤酒发酵法|窖藏啤酒发酵法]]，这些因素通通添加到一起使得皮尔森啤酒一经面世就立即引起了轰动。恰恰那正是[[铁路|铁路]]刚刚问世不久并迅猛发展的年代，随着交通方式的进步，很快，皮尔森啤酒和[[皮尔森酿造法|皮尔森酿造法]]便在整个[[中欧|中欧]]普及开来。

== 近现代的比尔森啤酒 ==
随着19世纪末[[卡尔·冯·林德|卡尔·冯·林德]]将现代冷冻技术引进到[[德国|德国]]，过去对[[下层发酵法|下层发酵法]]来说必不可少的用来维持发酵时低温环境的大地窖变得不再必要，这使许多过去因条件不足无法酿造比尔森式啤酒的地方也得以开始酿造比尔森式啤酒。然而，直到不久之前，正宗的经典式比尔森啤酒依然在使用酒厂下的大地窖和传统的敞开式酒桶。这一传统技术到了1993年才开始被大型窖用酒罐取代。但传统方法依然用于生产少量的样酒以测试酒的质量好坏。

== 今日的比尔森啤酒 ==
比尔森啤酒以世界上首种“金色啤酒”著称。现代比尔森啤酒的色泽从淡黄色到金黄色不等，所使用的[[香料|香料]]和味道彼此也都有着很大的不同：比如在原产地捷克，比尔森啤酒的口味一般比较清淡，而德国产的比尔森啤酒的口感往往要苦一些（尤其在比尔森啤酒比较流行的北德），有时甚至有类似泥土的味道。
