《'''舌尖上的義大利'''》（{{lang-ja|ピアシェーヴォレ！ 〜piacevole〜}}）是由Nab_At（{{tsl|ja|渡辺敦子 (アニメーター)|渡邊敦子}}）創作的[[日本|日本]][[日本漫画|漫畫]]作品，自2013年10月11日至2017年10月27日在[[comico|comico]]（{{tsl|ja|NHN comico}}）上連載，作品名稱中的「piacevole」在[[義大利語|義大利語]]為「愉快」之意。[[單行本|單行本]]由EARTH STAR Comics（[[EARTH_STAR_Entertainment|EARTH STAR Entertainment]]）出版，標題為{{Nihongo|《'''piace～我的義大利料理～'''》|ピアシェ〜私のイタリアン〜}}。

使用單行本標題的電視動畫自2017年1月開始播放<ref name="animeka">{{Cite web|url=http://natalie.mu/comic/news/206368|script-title=ja:渡辺敦子原作アニメ「ピアシェ」に千本木彩花ら出演、ビジュアルや映像も|date=2016-10-21|publisher=コミックナタリー|accessdate=2016-10-25|language=ja}}</ref>。

== 故事簡介 ==
故事發生在供應各式各樣美味料理的義大利料理店——{{tsl|en|Trattoria|義大利餐廳}}「菲斯塔」。女高中生七瀨萌里菜在那裡與充滿個性的人們共事後，開始對料理的世界產生興趣。

== 登場人物 ==
; {{Nihongo|七瀨萌里菜|七瀬 萌里菜（ななせ もりな）|聲：[[千本木彩花|千本木彩花]]<ref>{{cite web|url=http://piace-anime.com/character/morina/|script-title=ja:七瀬 萌里菜|accessdate=2016年12月30日|language=ja}}</ref>}}
: 本作主角，女高中生，在義大利餐廳「菲斯塔」擔任服務生。個性單純，脾氣很好，但是有人侮辱料理時會生氣，其力道可以徒手破壞一道牆。以一位義大利廚師為目標，向麻呂等人學習料理。目前和麻呂交往。

; {{Nihongo|北原麻呂|北原 マロ（きたはら マロ）|聲：[[米內佑希|米內佑希]]<ref>{{cite web|url=http://piace-anime.com/character/maro/|script-title=ja:北原 マロ|accessdate=2016年12月30日|language=ja}}</ref>}}
: 「菲斯塔」的代理店長，還是小學生但擁有天才般的料理技術，但是個性仍然像小孩子。接替兩年前出外旅行後便沒有回來的父親經營餐廳，放學後會回到店裡幫忙工作。喜歡萌里菜。目前和萌里菜交往。

; {{Nihongo|近野桐秀|近野 桐秀（こんの きりひで）|聲：[[阿座上洋平|阿座上洋平]]<ref>{{cite web|url=http://piace-anime.com/character/kirihide/|script-title=ja:近野 桐秀|accessdate=2016年12月30日|language=ja}}</ref>}}
: 「菲斯塔」的主菜廚師，對食材有很強烈的執著，甚至有戀物的傾向。

; {{Nihongo|藤木瑠梨|藤木 瑠梨（ふじき るり）|聲：[[日笠陽子|日笠陽子]]<ref>{{cite web|url=http://piace-anime.com/character/ruri/|script-title=ja:藤木 瑠梨|accessdate=2016年12月30日|language=ja}}</ref>}}
: 「菲斯塔」的副店長兼會計，平時很嚴厲，但偶爾也有溫柔的一面。

; {{Nihongo|折木衛|折木 衛（おれき えい）|聲：[[八代拓|八代拓]]<ref>{{cite web|url=http://piace-anime.com/character/ei/|script-title=ja:折木 衛|accessdate=2016年12月30日|language=ja}}</ref>}}
: 「菲斯塔」的Dolce（甜點）師傅，雖然技術高超，但是雙手得了腱鞘炎因此無法做料理，因為長相帥氣，所以受女生歡迎。極度愛好女色，個性看似輕浮，但是很關心麻呂，期望麻呂不要鑽牛角尖，步上他的後塵，並一直期待可以出現麻呂喜歡的人來改變他。不允許任何人侮辱菲斯塔。

; {{Nihongo|帝莉沙羅|帝莉 沙羅（ていり さら）|聲：[[五十嵐裕美|五十嵐裕美]]<ref>{{cite web|url=http://piace-anime.com/character/sara/|script-title=ja:帝莉 沙羅|accessdate=2016年12月30日|language=ja}}</ref>}}
: 高級日本料理店「祭禮」老闆的孫女，具有敏銳的嗅覺與味覺，喜歡萌里香的料理。
