'''曹陽'''（{{bd|1962年|12月}}<ref name=cfadata>{{cite web|title=1987年第6届全运会足球比赛|url=http://cfa002.cfadata.com/china/1987qyh.html|website=中国足球资料|accessdate=2016-10-22}}</ref>），中国足球教练，退役足球运动员。曹阳祖籍[[黑龙江省|黑龙江省]]，生于[[广东省|广东省]][[兴宁县|兴宁县]]<ref name=jqjl>{{cite book|editor=《国学家古直》编委会|title=激情九龙 再展雄风 纪念册|date=2011-09|pages=62}}</ref>（今广东省[[梅州市|梅州市]]代管的[[兴宁市|兴宁市]]）。球员时代，曹阳司职后卫，随广东省队获得[[四运会|四运会]]男足少年组冠军，之后入选[[中国国青队|中国国青队]]，与广东省足球队参加了[[五运会|五运会]]、[[六运会|六运会]]足球比赛，两次出战[[省港杯|省港杯]]。退役之后，曹阳执教[[广东日之泉|广东日之泉]]五年，带队从[[中国足球乙级联赛|中国足球乙级联赛]]打入[[中国足球甲级联赛|中国足球甲级联赛]]，卸任主教练之后出任俱乐部总经理，一年内两次担任代理主教练，带领球队保级成功。他还在2009年[[中华人民共和国第十一届运动会|十一运会]]带领的广东队杀入决赛、摘得银牌。2013年，曹阳加盟新成立的[[梅州客家|梅州客家]]，出任主教练，冲击中甲失败后改任总经理，并在2016年出任代理主帅。

== 球员生涯 ==
曹阳早年就读兴宁城镇三小、兴宁一中。1973年，他加入兴宁县业余体育学校足球班。1978年升入广东省足球三队。1979年，曹阳随广东省队获得[[四运会|四运会]]男足少年组冠军<ref name=jqjl/>。四运会后，他入选[[中国国青队|中国国青队]]<ref name=jqjl/>。他曾两次为国出征，在[[尼泊尔|尼泊尔]]、[[新加坡|新加坡]]比赛。1980年，他被上调至广东青年队。1981年，曹阳进入广东省足球队。<ref name=tyz>{{cite book|editor=《兴宁县体育志》编写小组|title=兴宁县体育志|date=1986|pages=87}}</ref>随后代表省队参加了[[五运会|五运会]]、[[六运会|六运会]]足球比赛。<ref name=jqjl/><ref name="jiemi">{{cite news|title=揭秘广东主帅：拼命三郎成功转型 曾4-2痛击上海|url=http://sports.sina.com.cn/c/2009-08-01/14224512082.shtml|accessdate=2016-10-22|work=东方体育日报|date=2009-08-01}}</ref>他还两次参加[[省港杯|省港杯]]足球赛<ref name=tyz/>。

曹阳原本司职边前卫，后来改打边后卫和中后卫。他擅长助攻，抢断凶狠，有“拼命三郎”之称。<ref name=jqjl/><ref name=tyz/>

== 退役之后 ==
=== 广东日之泉 ===
退役之后，曹阳继续从事足球工作。2000年，他加盟[[广东足协|广东足协]]竹料基地。<ref name=jiunian>{{cite news|title=九年苦行僧一朝偿所愿 全运会获亚军曹阳终得回报|url=http://sports.sina.com.cn/c/2009-08-03/01254514587.shtml|accessdate=2016-10-22| work =新快报|date=2009-08-03}}</ref>2004年，上级指派曹阳组建[[中华人民共和国第十一届运动会|十一运]]广东男足代表队。2007年，以广东全运队为骨干的[[广东日之泉足球俱乐部|广东日之泉足球俱乐部]]成立，曹阳担任主教练。<ref name="jiemi"/>[[2008年中国足球乙级联赛|2008年]]，曹阳带领日之泉从[[中乙联赛|中乙联赛]]冲上[[中甲|中甲]]<ref>{{cite news|title=广州足球冲甲背后的故事 日之泉好男儿有真性情|url=http://news.66wz.com/system/2008/12/22/101035748.shtml|accessdate=2016-10-22| work=广州日报|date=2008-12-22}}</ref>。2009年十一运会上，曹阳带领的广东队杀入决赛，不敌上海队，摘得银牌<ref name=jiunian/>。执教日之泉五年，曹阳一直无法将球队带上[[中超|中超]]。[[2012年中国足球甲级联赛|2012年]]，俱乐部定下升入中超联赛的目标，[[塞尔维亚|塞尔维亚]]人[[德拉甘·可可托维奇|德拉甘·可可托维奇]]接替曹阳执掌广东日之泉，曹阳改任总经理。<ref>{{cite news|title=解读日之泉百日维新的背后 洋帅成中甲下课第1人|url=http://www.hinews.cn/news/system/2012/03/27/014212251.shtml|accessdate=2016-10-22| work =新浪体育|date=2012-03-27}}</ref>然而新赛季两轮之后，俱乐部就将可可托维奇解雇，曹阳代理主教练一职。<ref>{{cite news|title=日之泉解聘可可托维奇　总经理曹阳出任代理主教练|url=http://news.ifeng.com/gundong/detail_2012_03/28/13485526_0.shtml|accessdate=2016-10-22| work =新快报|date=2012-03-28}}</ref>7月28日，曹阳结束代理任期，重回总经理岗位，[[巴西|巴西]]教练[[荷西·列卡度·蘭保|里卡多]]接手球队。这时广东日之泉刚刚结束五连败，险些坠入降级区。<ref>{{cite news|title=里卡多接替曹阳出任日之泉新主帅 教练球员日趋港化|url=http://sports.sina.com.cn/b/2012-07-30/11006156070.shtml|accessdate=2016-10-22| work =足球|date=2012-07-30}}</ref>里卡多上任后，日之泉的成绩没有起色，仍然无法摆脱降级危险。8月底，俱乐部决定将里卡多降为助理教练，曹阳重新当上了代理主教练。这时中甲联赛只剩下最后七轮。<ref>{{cite news|title=日之泉总经理临危受命洋帅下课 赢球奖金增20万|url=http://cnsoccer.titan24.com/2013-01-04/285910.html|accessdate=2016-10-22| work =南方都市报|date=2012-08-30}}</ref>赛季末，广东日之泉保级成功<ref>{{cite news|title=日之泉换主场胜保级成功 主帅盼明年继续在奥体|url=http://cnsoccer.titan24.com/2013-01-02/239701.html|accessdate=2016-10-22| work =搜狐体育|date=2012-10-20}}</ref>。赛季结束后，曹阳离开了广东日之泉，加盟新成立的[[梅州客家足球俱乐部|梅州客家足球俱乐部]]<ref>{{cite news|title=区楚良将出任广州日之泉总经理 最终目标带队冲超|url=http://sports.sina.com.cn/b/2012-12-08/13596329038.shtml|accessdate=2016-10-22| work =羊城晚报|date=2012-12-08}}</ref>。

=== 梅州客家 ===
[[2013年中国足球乙级联赛|2013年]]，曹阳带领新成立的梅州客家队征战中国足球乙级联赛。预赛中球队排在南区第一，但是全国总决赛第一轮就被[[山东滕鼎|山东滕鼎]]淘汰。<ref>{{cite news|title=梅州现状：华侨哀叹家乡足球 比赛场地被垃圾占据|url=http://cnsoccer.titan24.com/2013-10-16/307336.html|accessdate=2016-10-22| work =体坛周报|date=2013-10-16}}</ref>2014年，曹阳转任总经理<ref>{{cite news|title=梅州五华足球队更换主教练  张军替代谭恩德执掌教鞭|url=http://mzrb.meizhou.cn/html/2015-05/28/content_63194.htm|accessdate=2016-10-22|work=梅州日报|date=2015-05-28|archive-url=https://web.archive.org/web/20161022161007/http://mzrb.meizhou.cn/html/2015-05/28/content_63194.htm|archive-date=2016-10-22|dead-url=yes}}</ref>。2016年7月19日，梅州客家主教练{{link-en|卢克·尼赫特|Luc Nijholt|-{zh-hans:尼赫特;zh-hk:尼積荷特}-}}辞职，曹阳出任代理主教练<ref>{{cite news|title=梅州客家主教练辞职 总经理曹阳暂掌帅印|url=http://news.dayoo.com/sports/201607/20/140001_47481417.htm|accessdate=2016-10-22|work=广州日报|date=2016-07-20|deadurl=yes|archiveurl=https://web.archive.org/web/20161022161950/http://news.dayoo.com/sports/201607/20/140001_47481417.htm|archivedate=2016年10月22日|df=}}</ref>。
