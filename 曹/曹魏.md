'''魏'''（220年12月10日－266年2月8日<ref name="smy" />，史称'''曹魏'''、'''魏朝'''）。是中國歷史上[[漢朝|漢朝]]末期[[三國|三國]]之中据有北方及[[中原|中原]]的政權。始於220年[[曹丕|曹丕]]逼迫[[漢獻帝|漢獻帝]]劉協禪讓帝位，篡漢為'''魏'''，因承繼漢朝，故具[[法統|法統]]地位。至266年魏又被[[司馬炎|司馬炎]]篡奪，改號為'''[[西晉|晉]]'''。

[[曹操|曹操]]受封魏公時，治所在[[東漢|東漢]]時期[[魏郡|魏郡]]所在地的[[鄴|鄴]]，因此漢獻帝封他為「魏公」建立诸侯国——魏国，且如同汉朝初期诸侯王制度可以设置丞相以下百官<ref>《三国志·魏志·武帝纪》：今以冀州之河东、河内、魏郡、赵国、中山、常山、钜鹿、安平、甘陵、平原凡十郡，封君为魏公。……魏国置丞相已下群卿百寮，皆如汉初诸侯王之制。</ref><ref>《三国志·魏志·荀攸传》：魏国初建，为尚书令。</ref>，之后又进封「魏王」并以卞氏为魏国王妃<ref>《三国志·魏志·后妃传》：太祖建国，始命王后。</ref><ref>《三国志·魏志·后妃传·武宣卞皇后》：（建安）二十四年，拜为王后，策曰："夫人卞氏，抚养诸子，有母仪之德。今进位王后，太子诸侯陪位，群卿上寿，减国内死罪一等。"</ref>，以曹操之女为魏国公主<ref>《三国志·魏志·武帝纪》：夏五月，天子进公爵为魏王。……天子命王女为公主，食汤沐邑。</ref>，后来曹操之子曹丕篡漢时便以「魏」為國號。又因为是曹氏政权，故史稱「曹魏」，以區別於其他名「[[魏|魏]]」的政權<ref>{{cite book zh |author = 徐俊 |title = 中國古代王朝和政權名號探源 |publisher = 華中師範大學出版社 |date = 2000年11月 |location = 湖北武昌 |pages = 78-80 |url = |doi = |id = |isbn = 7-5622-2277-0 }}</ref>。

魏是三國時期最為強大，領土最遼闊的政權，灭[[蜀汉|蜀汉]]前疆域達到近300萬平方千米{{請求來源}}。263年，魏軍攻滅蜀汉，同年佔領[[广州_(古代)|廣州]]，至此曹魏疆域達到全盛，约400萬平方千米。由於曹魏盤踞[[中原|中原]]，所以這區人口也是最多。期間最重要的政治改革有[[陳群_(三國)|陳群]]的[[九品中正制|九品中正制]]，對魏晉時代之政治產生深遠影響。

== 歷史 ==
[[File:Shinju-kyo_Bronze_Mirror.JPG|thumb]]。]]
曹魏未立國前，[[東漢|東漢]]已進入群雄割據的時代。建安元年（196年）[[曹操|曹操]]迎[[漢獻帝|漢獻帝]]到許昌，[[挾天子以令諸侯|挾天子以令諸侯]]，迫令各地割據勢力必須遵奉曹操，而軍事方面則選編精銳，組成一批強大的騎兵勁旅——[[虎豹騎|虎豹騎]]，在平定中原的重要戰役中屢次建功，在掌握軍政的發展下開啟魏晉南北朝[[霸府政治|霸府政治]]的先河，為後來的篡漢立魏打下了基礎。200年的[[官渡之戰|官渡之戰]]後控制了[[中原|中原]]北方地區大部分，並準備南下一舉攻佔全國。

但由於曹軍未適應到南方的地理環境，在水土不服，天氣陡轉等因素下，於在208年的[[赤壁之戰|赤壁之戰]]中敗於[[孫權|孫權]]與[[劉備|劉備]]的南方聯軍，退守北方。但由於曹軍所傷多為原[[劉表|劉表]]麾下水軍與降軍，因此孫劉勢力亦無法撼動曹操政權。由於曹操年事已高，於是終其一生只控制了中原一帶。其後漢獻帝封曹操為魏王，打破了漢高祖所訂的[[白馬之盟|白馬之盟]]，220年，曹操去世後由次子曹丕襲位。

曹丕逼迫漢獻帝禪讓，篡漢為帝，定國號魏，並定都[[洛陽|洛陽]]（或称[[曹魏五都|曹魏五都]]），曹魏正式建立。曹操雖未稱帝，但曹丕稱帝后追尊他為魏武帝（廟號太祖）。漢中王[[劉備|劉備]]於221年在[[成都|成都]]自立為帝，國號漢，史稱蜀漢。最後[[孫權|孫權]]也於229年在[[建業|建業]]稱帝國號吳，至此三國正式形成。

魏文帝曹丕稱帝七年就去世。曹丕死後，長子曹叡即位，是為魏明帝。曹魏朝廷此時分為兩大派，一是以[[曹真|曹真]]和[[曹休|曹休]]為主的曹氏一族、二是以[[司馬懿|司馬懿]]世家和[[贾逵_(三国)|賈逵]]世家為主的新勢力，日漸形成嚴重的對立，埋下日後的[[高平陵之變|高平陵之變]]的種子。而[[夏侯惇|夏侯惇]]和[[夏侯渊|夏侯渊]]死後，夏侯世家人才能力沒落，漸漸遠離權勢。導致司馬世家勢力龐大，夏侯世家無法抗衡，日後[[高平陵之變|高平陵之變]]的發生後，夏侯世家遭到流放邊境、投降蜀漢與孫吳。

[[曹操|曹操]]出身寒族，且與閹宦有關，不以儒學為務，與當時的豪族、士大夫不同。曹操曾下「求才三令」，強調重才不重德，並以[[法家|法家]]之術為治，要摧破豪族的儒學。曹操為一代梟雄，不僅得到眾多寒族人才支持，也得到部份豪族士大夫支持，如[[荀彧|荀彧]]、[[荀攸|荀攸]]。荀彧更為曹操引進不少[[士大夫|士大夫]]階層的人才。官渡之戰，曹勝袁敗，士大夫豪族不得不暫時忍耐屈服，卻嗣機恢復。終於他們支持出身士族的[[司馬懿|司馬懿]]，向曹氏奪回政權。<ref>萬繩楠：《陳寅恪魏晉南北朝史演講錄》（合肥：黃山書社，1987），頁9-13。</ref>

曹魏都是在與[[蜀漢|蜀漢]]、[[孫吳|孫吳]]的戰事中度過，如蜀漢的[[諸葛亮|諸葛亮]]發動多次[[諸葛亮北伐|北伐]]攻打魏國，[[曹叡|曹叡]]多次力拒來犯守護國土。太傅[[司馬懿|司馬懿]]在[[諸葛亮北伐|諸葛亮北伐]]戰事中立下不少戰功，在曹魏的政治地位漸漸提升，直至[[高平陵之變|高平陵之變]]，司馬懿利用[[兵變|兵變]]，剷除了曹家[[宗室|宗室]]的[[曹爽|曹爽]]，后其子司马师在政治斗争中又铲除了夏侯家的[[夏侯玄|夏侯玄]]，之後[[夏侯霸|夏侯霸]]投靠[[蜀漢|蜀漢]]，司馬氏家族權傾全朝成為新的霸府，[[司马师|司马师]]、[[司馬昭|司馬昭]]兄弟成為朝廷中最有權勢的朝臣，能擅自廢立皇帝。魏帝[[曹髦|曹髦]]不甘司馬氏威脅自己帝位，亲自攻打司馬昭，司马昭命令親信[[賈充|賈充]]派[[成濟|成濟]]弒害曹髦，事後僅成濟被處死，而[[司馬氏|司馬氏]]家族則沒受牽連，因此曹魏於此時名存實亡。

隨著[[蜀漢|蜀漢]]國力日下，263年魏國司馬氏展開攻漢計劃，派遣[[鍾會|鍾會]]、[[鄧艾|鄧艾]]、[[諸葛緒|諸葛緒]]等等攻漢，結果漢後主[[劉禪|劉禪]]出降，蜀漢亡國，隨後司馬昭便平定由[[鍾會|鍾會]]、[[姜維|姜維]]、[[劉璿|劉璿]]等等蜀漢殘黨與部份鍾會勢力所發起的[[鍾會之亂|鍾會之亂]]。後司馬昭死，其子[[司馬炎|司馬炎]]襲晉王、相國位，於265年篡魏自立，國號晉，曹魏滅亡。

[[曹奂|曹奂]]及后人被封为陈留王，在[[晋朝|晋朝]]受到很高的待遇，[[陈留王国|陈留王国]]历经4朝，经[[西晋|西晋]]、[[东晋|东晋]]、[[刘宋|刘宋]]，一直传至[[南北朝|南朝]][[南齐|南齐]]，国祚之长在历史上实为罕见。

==军事==
曹魏於曹操死前，兵力約30-45萬，曹操死後，曹丕濫兵用戰，興建修房，死了不少的精兵，據當時的史書記載，兵力約13-16萬，為曹魏的低落時期，到了[[曹叡|曹叡]]統治，由於推動多子政策，兵力大增到40-47萬，到司馬家族主政後，約到60-65萬，為最高峰，司馬家族統治後的第23年（滅蜀一年），兵力破百萬。

== 文化 ==
曹魏雖然是以軍事起家，但曹魏一族在文學上具有相當成就，如曹操和其子[[曹丕|曹丕]]和[[曹植|曹植]]都擅於寫詩，時稱三曹，後世又稱[[建安文學|建安文學]]。後期君主也頗有藝術造詣，如曹操其孫[[曹叡|曹叡]]擅長詩賦；[[曹髦|曹髦]]擅長詩文、繪畫，被譽為才子。

== 經濟 ==
魏、漢、吳三國中以曹操最重視[[農業|農業]]（用[[毛玠|毛玠]]兵農合一之策），其中以曹魏人口最多，[[屯田制|屯田]][[墾荒|墾荒]]的面積最廣。

曹魏重視農業的另一實證，是其大力興修[[水利|水利]]，其工程的規模和數量在三國中首屈一指。如233年，[[關中|關中]]一帶闢建渠道，興修水庫，一舉改造三千多頃[[鹽鹼地|鹽鹼地]]，所獲使國庫大為充實。再如曹魏在[[河南|河南]]的水利工程，其成果使糧食產量倍增。

[[諸葛亮北伐|諸葛亮北伐]]時期對曹魏的經濟影響甚是巨大，<ref>《晉書·宗室傳》：每諸葛亮入寇關中，邊兵不能制敵，中軍奔赴，輒不及事機。</ref><ref>《三國志·魏書·郭淮傳》：蜀出鹵城。是時，隴右無谷</ref>從[[辛毗|辛毗]]、[[楊阜|楊阜]]的奏章中，不止一次提到諸葛亮北伐造成的經濟困難。<ref>《三國志·魏書·楊阜傳》：是時大司馬曹真伐蜀，遇雨不進。阜上疏曰：「……今年兇民饑，宜發明詔損膳減服，技巧珍玩之物，皆可罷之。……今者軍用不足，益宜節度。」</ref><ref>《三國志·魏書·楊阜傳》：阜上疏曰：「……方今二虜合從，謀危宗廟，十萬之軍，東西奔赴，邊境無一日之娛；農夫廢業，民有饑色。……」</ref><ref>《三國志·魏書·辛毗傳》：毗上疏曰：「竊聞諸葛亮講武治兵，而孫權市馬遼東，量其意指，似欲相左右。備豫不虞，古之善政，而今者宮室大興，加連年谷麥不收。……」</ref>
