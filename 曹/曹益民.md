'''曹益民'''（{{bd|1962年|6月||}}），[[四川|四川]][[蓬安|蓬安]]人。[[中国人民解放军少将|中国人民解放军少将]]。

== 生平 ==
曹益民是[[中国共产党|中国共产党]]党员。[[中国人民大学|中国人民大学]]行政管理专业毕业。曾任[[兰州军区|兰州军区]]司令部军务部参谋、总参谋部办公厅（[[傅全有|傅全有]]）秘书、总参谋部军务部第一编制局局长。2005年12月，任[[新疆军区|新疆军区]]独立第四摩步师师长。2007年6月，任兰州军区[[中国人民解放军陆军第二十一集团军|陆军第二十一集团军]]参谋长。2009年11月，任陆军第二十一集团军副军长。2011年8月，任[[中国人民解放军陆军第四十七集团军|陆军第四十七集团军]]军长。2013年3月，任陆军第二十一集团军军长<ref>{{cite news|title=曹益民：向制度化经常化要效益  陆军第二十一集团军军长　曹益民|url=http://theory.people.com.cn/n/2013/0916/c40531-22933481.html|accessdate=2015-08-08|agency=解放军报}}</ref>。2017年1月，任[[中国人民解放军西部战区陆军|中国人民解放军西部战区陆军]]参谋长<ref>{{cite news|title=第21集团军原军长曹益民少将已任西部战区陆军领导|url=http://www.thepaper.cn/newsDetail_forward_1607177|accessdate=2017-01-25|newspaper=澎湃新闻}}</ref><ref>{{Cite web |url = http://www.thepaper.cn/newsDetail_forward_1639342 |title = 第21集团军原军长曹益民调任西部战区陆军参谋长 |author =  |date = 2017-03-15 |publisher = 澎湃新闻 |language =  |accessdate =  |quote =  }}</ref>。

2009年7月晋升少将军衔。

2013年，当选[[第十二届全国人大代表|第十二届全国人大代表]]<ref>{{cite news|title=全国人大代表信息-曹益民|url=http://www.npc.gov.cn/delegate/viewDelegate.action?dbid=122949|accessdate=2013-04-05|newspaper=全国人大网}}</ref>。
