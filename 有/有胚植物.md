'''有胚植物'''，又称为'''高等植物'''，是那些最熟悉的[[植物|植物]]，包括[[苔藓植物门|苔藓植物门]]、[[地钱纲|地钱纲]]、[[角苔纲|角苔纲]]、[[蕨类植物|蕨类]]、[[石松|石松]]、[[裸子植物|裸子植物]]、[[开花植物|开花植物]]等，但不包括[[绿藻|绿藻]]。有胚植物都是具有专门的生殖器官的复杂[[多细胞|多细胞]][[真核生物|真核生物]]，除极少例外，都通过[[光合作用|光合作用]]获取能量，如吸收光，从[[二氧化碳|二氧化碳]]中生成养料。有胚植物具有的特化的[[生殖器官|生殖器官]]中有非生殖性的组织，这一点可以与多细胞[[藻类|藻类]]相区分。有胚植物主要是适应陆地的生活环境，虽然有些再一次进入水中生活，这样，有胚植物也被称为'''陆生植物'''。

== 描述 ==
有胚植物是在5.4亿年前的[[古生代|古生代]]从复杂的[[绿藻|绿藻]]发展而来的。有胚植物现存的最紧密的亲戚是[[轮藻|轮藻]]。 这些藻类经历[[单倍体|单倍体]]世代（配子体）和[[双倍体|双倍体]]世代（孢子体）的[[世代交替|世代交替]]。

显微镜下，有胚植物的细胞结构类似于绿藻，区别在于细胞分裂时有{{tsl|en|phragmoplast|成膜体}}形成细胞壁。有胚植物的另一个不同点是有一个较大的中央[[液泡|液泡]]，其[[膨压|膨压]]提供植物细胞的支持力，使它能维持形状。

多细胞藻类虽然也有[[世代交替|世代交替]]，但是有胚植物的名称来自于在母本的[[配子体|配子体]]组织中养育受精卵——[[孢子体|孢子体]][[胚胎|胚胎]]这一[[共有衍征|共有衍征]]。

最早的有胚植物的[[孢子体|孢子体]]在结构和功能上差别很大，仍旧很小并在短暂的生命周期中依赖于母体。这种植物被称为[[苔藓植物|苔藓植物]]，现存的有：

* [[苔藓植物门|苔藓植物门]]
* [[角苔門|角苔門]]
* [[地钱門|地钱門]]

所有苔藓植物都相对小并局限于潮湿环境中，依赖水来散播它们的[[孢子|孢子]]。有一些植物在[[志留纪|志留纪]]更好地适应了新出现的陆地环境，并在[[泥盆纪|泥盆纪]]变得多样化，同时传播到不同的陆地环境。这些植物被称为[[维管植物|维管植物]]或导管植物。维管植物具有[[维管组织|维管组织]]或[[管胞|管胞]]，水分通过维管在[[植物|植物]]体内输送，外面还有防止干燥的表皮。大部分维管植物的孢子体已经发展成独立的植株，已经进化出真正的[[叶|叶]]，[[茎|茎]]和[[根|根]]，但配子体仍然很小。

但有的维管植物依然依靠[[孢子|孢子]]繁殖，現存有：

* [[石松門|石松門]]  
* [[蕨類植物門|蕨類植物門]]  

其他种类的维管植物是在[[古生代|古生代]]后期出现的，已经不用孢子繁殖，进化出可以抵御干燥的[[种子|种子]]，所以叫做[[种子植物|种子植物]]。种子植物的有受精卵发育的整个幼年孢子体都包含在种子之中，有的种子即使它们的母体是生长在多[[水|水]]的环境之下，也可以抵御极度干旱的环境。種子植物現存有：

* [[松柏門|松柏門]]
* [[蘇鐵門|蘇鐵門]]
* [[銀杏門|銀杏門]]
* [[買麻藤門|買麻藤門]]
* [[被子植物門|被子植物門]]

前四类被稱之為[[裸子植物|裸子植物]]的[[多系群|多系群]]，因为它们的[[胚珠|胚珠]]是裸露的，不被[[心皮|心皮]]包裹。相对地，[[被子植物门|被子植物]]的胚珠是被心皮包裹，形成果肉。被子植物是在[[侏罗纪|侏罗纪]]才最后出现的一类[[植物|植物]]，在[[白垩纪|白垩纪]]时迅速繁殖，成为现代[[地球|地球]]陆地上最大的[[生物|生物]]量组成部分。

目前[[植物分类学|植物分类学]]上意见并不统一，有的分类学家认为[[植物界|植物界]]只包括有胚植物，但也有些分类学家认为有胚植物单独应该算做一[[门|门]]，他们将陆生植物和[[轮藻|轮藻]]一起划为一个[[亚门|亚门]]。

== 系統發生學和分類 ==
植物高層的分類變動得非常地大。一些學者認為[[植物界|植物界]]只包括有胚植物，而其他學者則有不同的命名和規範。此處列出的類群通常被視為門，但有時也會被視為綱，且偶爾會被縮至只剩兩個門。一些分類學則將有胚植物放在總門上，並將其和部份輪藻一起歸為一個亞界，稱之為[[鏈形植物|鏈形植物亞界]]。

从[[分子生物学|分子生物学]]角度看，有胚植物和[[绿藻|绿藻]]形似，它们都具有[[真核生物|真核]]，有由两层原生质分泌形成的，有叶绿体可以进行[[光合作用|光合作用]]生成[[淀粉|淀粉]]，主要叶绿体都是由叶绿体a和叶绿体b组成的，两者都有大液泡，没有中心质，除了配子以外也都没有鞭毛。

传统上高等植物包括四个门：苔藓植物门、蕨类植物门、裸子植物门和被子植物门。苔藓和蕨类又合称[[颈卵器植物|颈卵器植物]]；蕨类植物、裸子植物和被子植物又合称[[维管植物|维管植物]]，裸子植物和被子植物合称[[种子植物|种子植物]]，[[显花植物|显花植物]]。
