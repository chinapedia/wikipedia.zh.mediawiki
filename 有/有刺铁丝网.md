[[File:Barbed_wire,_Melbourne_Museum.jpg|thumb]] 
'''有刺鐵絲網'''（barbed wire），是一種防禦型的金屬絲，身上佈滿尖銳的刃口，刃與刃之間亦有相同的距離。這類絲網價格低廉，用來保護[[牆|牆]]內的財產，並是[[第一次世界大戰|第一次世界大戰]]裡[[塹壕戰|塹壕戰]]的重要防守設備。

若一個人或一隻[[動物|動物]]經過有刺鐵絲網時，便會遭受到[[皮膚|皮膚]]上的痛楚，嚴重者更會受傷。有刺鐵絲網的製造十分簡單，只需要幾條固定鐵絲網的柵柱，然後再利用例如[[訂書機|訂書機]]的工具，即可完成構造。就算一個不熟練的製造者，也能夠很快完成構造。

在1865年，Louis Jannin發明了有刺鐵絲網。1874年，[[迪卡尔布_(伊利诺伊州)|伊利诺伊州迪卡尔布]]的Joseph F. Glidden等人改良為更成功的鐵絲網。在最早時期使用鐵絲網的目標，是想限制[[牛|牛]]和其他牲畜的活動範圍。由於鐵絲網較其他類型的柵欄更容易建造。因此它在19世紀初被美國通行，平民百姓更能大規劃地利用鐵絲網來限制畜牧。

== 防人柵欄 ==
有刺鐵絲網大都能夠有效地防止生畜逃跑，但对人类来说，只要在攀爬时巧妙地避开铁刺，越過常规有刺鐵絲網并非难事。為了防範人類的前進，一些高度设防的地方例如[[监狱|监狱]]，就會使用一種特制的鐵絲網——刀片刺網（razor wire）。這種鐵絲網身上每隔數寸就附有尖銳的金屬刺條，在未受保護的情況下足以令一個人受傷。所以，刀片刺網比起其他普通的有刺鐵絲網更難攀越。有时刀片刺網上的刺條更连接電路，攀爬者接触到刺条便会遭到电击，常見於監獄、[[戰俘營|戰俘營]]、[[古拉格|古拉格]]等收容設施。在[[第一次世界大戰|第一次世界大戰]]期間，這類有刺鐵絲網被用來阻擋敵兵前進，另外更使用它來防範[[戰俘|戰俘]]逃亡。在某些學校的圍牆上也有鐵絲網的設置，除了防盜外還能具體遏止學生[[逃學|逃學]]。部分的[[國界|國界]]上也有鐵絲網的身影，如[[美墨邊境|美墨邊界]]、[[柏林圍牆|柏林圍牆]]、[[三十八度線|三十八度線]]、[[巴勒斯坦|巴勒斯坦]]的猶太人[[基布茲|屯墾區]]、[[馬其諾防線|馬其諾防線]]、南北[[塞浦路斯|塞浦路斯]]邊界、南北[[蘇丹|蘇丹]]邊界等等。

鐵絲網除了軍事和收容用途外，也用在一些國家的[[大使館|大使館]]和[[領事館|領事館]]中防止外來的敵意人士擅闖進入，尤其在兩國外交處於險惡的地區，如美國和以色列的使領館為防恐怖份子和[[暴民|暴民]]攻擊而在圍牆頂端加裝了鐵絲網。

現今，於世界各地的[[群體事件|群體事件]]、[[示威|示威]]、[[抗議|抗議]]活動中，維持秩序的[[鎮暴警察|鎮暴警察]]會使用一種稱為「蛇籠」的彈簧狀鐵絲網，可從一輛專門的運輸拖車上將蛇籠拉出來以阻絕抗議隊伍的前進。

== 有刺鐵絲網帶來的傷害 ==
[[File:Drut_kolczasty_cm01.jpg|thumb]]
過度地穿越鐵絲網會對皮膚做成一連串的傷害，但最主要都是看接觸的位置來取決。若想穿越有刺鐵絲網而又不被刺傷的話，穿越者必需全神貫注，尤其是注意自己的衣服有否被刺入，方能減少受傷的機會。常常被鐵絲網刺傷的大部份都是那些[[馬|馬]]、[[蝙蝠|蝙蝠]]或[[鳥|鳥]]類。但馬類最容易驚慌，因為當牠們接觸到鐵絲網時，通常會被刃口撕掉整塊肉，嚴重者更刺入骨內。由於這種功效，有刺鐵絲網經常會被用來對付美國的[[騎兵|騎兵]]。有些馬的傷口能夠痊癒，但有些則導致馬匹的殘廢或死亡。至於鳥類或蝙蝠等飛行動物，因為沒有足夠的能力去觀察出細微的絲網，而被撕傷或刺傷皮膚。因此，在澳大利亞某一年間，就有超過60種野生動物被有刺鐵絲網刺傷至死。野外友好欄杆計劃組織（wildlife friendly fencing）因而被分配工作來解決此問題。

== 拆除方法 ==
在戰場上，工兵會負責拆除鐵絲網的工作，起初工兵使用大剪刀去把鐵絲網剪開來開路，但要把網絲逐條剪開很浪費時間。在第一次世界大戰前，英國軍隊中的印度人兵團就發明了炸藥筒（Bangalore Torpedo），將裝滿高性能炸藥的鐵筒插進鐵網底下然後將炸藥筒引爆，以爆破出來的破片和衝擊波即時把鐵絲網斬斷和撐開。炸藥筒成為了各國軍隊的工兵清除鐵絲網的兵器並使用至今。
