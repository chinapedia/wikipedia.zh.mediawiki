'''危險駕駛引致他人死亡'''（{{lang-en|'''Dangerous Driving Causing Death'''}}），是一項[[香港|香港]]的[[刑事|刑事]][[罪名|罪名]]，是[[交通意外|交通意外]]的[[司機|司機]]有機會被指控的刑事罪名。

== 危險駕駛 ==
危險駕駛有別於[[不小心駕駛|不小心駕駛]]或[[魯莽駕駛|魯莽駕駛]]<ref>[http://www.legco.gov.hk/yr99-00/chinese/bc/bc61/papers/c1540-01.pdf]</ref><ref>{{cite web |url=http://www.td.gov.hk/road_safety/dangerous_driving/index_tc.htm |title=存档副本 |accessdate=2008-05-03 |deadurl=yes |archiveurl=https://web.archive.org/web/20080419031346/http://www.td.gov.hk/road_safety/dangerous_driving/index_tc.htm |archivedate=2008-04-19 }}</ref>。

危險駕駛包括[[無牌駕駛|無牌駕駛]]、不理會[[交通燈|交通訊號]]、不理會路面交通情況、不顧[[警方|警方]][[路障|路障]]或交通指揮、[[超速駕駛|超速駕駛]]、小路出大路不停車又不減速、[[扒頭|扒頭]]（超車）、玩[[飄移|飄移]]、[[酒後駕駛|酒後駕駛]]、[[衝紅燈|衝紅燈]]等。

危險駕駛本身已是刑事罪行。最高刑罰為罰款[[港幣|港幣]]25,000元及監禁3年，如屬首次定罪，將被取消駕駛資格半年或以上；如其後再次被定罪，則將被取消駕駛資格1年半或以上。並須強制修習[[駕駛改進課程|駕駛改進課程]]及[[交通違例記分制|交通違例記分制]]之中記10分。

但如因為危險駕駛，引致他人（包括[[乘客|乘客]]、[[行人|行人]]和其他車輛的司機）死亡或受到嚴重傷害，一般將會控以「危險駕駛引致他人死亡」和「危險駕駛引致他人身體受嚴重傷害」的罪行。

== 法例 ==
依《[[香港法例|香港法例]]》第374章《道路交通條例》的《危險駕駛引致他人死亡》，最高刑罰為[[罰款|罰款]]HK$50,000及監禁10年；如屬首次定罪者，可被取消駕駛資格5年或以上；如其後再次被定罪，則可被取消駕駛資格10年或以上<ref>[http://www.legislation.gov.hk/blis_ind.nsf/b0bf723264bb0d11482564820006a88f/6282c7a74d1dc6878825648c0007d3d6?OpenDocument]</ref>。

而《危險駕駛引致他人身體受嚴重傷害》，最高刑罰為[[罰款|罰款]]HK$50,000及監禁7年；如屬首次定罪者，可被取消駕駛資格2年或以上；如其後再次被定罪，則可被取消駕駛資格5年或以上。

如受害者為[[警務人員|警務人員]]，如警務人員傷勢較輕，亦可以被加控「襲擊警務人員」罪，該罪之最高刑罰為罰款$5,000及監禁6個月。

上述三項罪名均須強制修習[[駕駛改進課程|駕駛改進課程]]，及[[交通違例記分制|交通違例記分制]]之中記10分；另外加入『犯罪情節特別嚴重』加刑因素，如司機干犯任何危險駕駛罪行時，其體內
#酒精濃度達第3級（即每100毫升呼氣/血液/尿液分別超過66微克/150毫克/201毫克的酒精），或拒絕接受酒精呼氣/血液/尿液測試；或
#含有任何份量的指明違禁藥物(即海洛英、氯胺酮、「冰」、大麻、可卡因或「搖頭丸」
會視作『犯罪情節特別嚴重』，有關罪行的最高罰款額、最高監禁期和最短停牌期各增加50%。在2010年的19修訂案增加的2AB款及2011年道路交通條例的修正案年第24號第6條中，法庭有權對再犯者或犯罪情節特別嚴重者判處終身停牌等更嚴格的刑罰。

此外，如犯罪情節特別嚴重，根據案例有機會以[[嚴重傷人|嚴重傷人]]或[[誤殺|誤殺]]作為交替控罪。例如為[[1983年|1983年]][[邱德根|邱德根]]兒子邱達成在[[何文田|何文田]][[公主道|公主道]]撞死交通警員，最後因誤殺罪入獄4年。<REF>[http://orientaldaily.on.cc/cnt/news/20091202/00176_005.html 83年邱達成撞死警誤殺囚四年]</ref>

== 引致他人死亡 ==
*他人，所指的是肇事[[司機|司機]]以外的人，包括[[乘客|乘客]]、[[行人|行人]]和其他車輛的司機等。
*[[死亡|死亡]]有別於受傷、[[重傷|重傷]]，包括[[併發症|併發症]]。例如一位[[的士司機|的士司機]]在[[單程路|單程路]]反向行車，後座乘客受驚引致[[心臟病|心臟病]]發死亡。
*導致，是[[因果關係|因果關係]]。他人死亡未必因為當時的危險駕駛引發。
