'''潘廸生'''爵士，[[英帝國司令勳章|CBE]]，[[銀紫荊星章|SBS]]（Sir '''Dickson Poon'''，{{bd|1956年|6月19日|||Poon, Sir Dickson}}），[[香港|香港]]商人，旗下業務包括[[鐘錶|鐘錶]]、[[百貨公司|百貨公司]]與著名[[品牌|品牌]][[時裝|時裝]]零售。2015年受英女皇封為[[下級勳位爵士|下級勳位爵士]]。

潘廸生是鐘錶商人[[潘錦溪|潘錦溪]]之子，他在香港就讀[[聖若瑟小學|聖若瑟小學]]，後前往[[英國|英國]]就讀中學，大學則在[[美國|美國]]就讀，主修[[哲學|哲學]]及[[經濟|經濟]]。1978年大學畢業後返回香港，在其父的鐘錶行擔任售貨員，並曾到[[瑞士|瑞士]][[日内瓦|日内瓦]]鐘錶商 [[Chopard|Chopard]] 擔任[[學徒|學徒]]18個月。

== 品牌零售 ==
{{main|廸生創建}}
* 1980年，潘廸生以其父提供的100萬美元[[資本|資本]]，於[[中環|中環]][[置地廣場|置地廣場]]創辦[[廸生鐘錶珠寶店|廸生鐘錶珠寶店]]。1981年取得品牌 [[Charles_Jourdan|Charles Jourdan]] 的香港[[代理權|代理權]]，1983年則取得 [[Polo_Ralph_Lauren|Polo Ralph Lauren]] 品牌香港代理權。1985年潘廸生收購上市公司[[順福企業|順福企業]]，將其易名為[[廸生創建|廸生創建]]，並把其零售業務注入。1987年11月，以5,200萬[[美元|美元]]收購以生產[[打火機|打火機]]著名的法國品牌[[:en:ST_Dupont|都彭]]。
* 1991年10月，潘廸生以5,370萬[[英鎊|英鎊]]收購當時陷入財政困難的英國[[百貨公司|百貨公司]][[夏菲尼高|夏菲尼高]]([[Harvey_Nichols|Harvey Nichols]])，並將其業務轉蝕為盈。1992年11月則把其零售業務擴展至[[中國|中國]]及[[台灣|台灣]]多個大城市。近年他購入[[日本|日本]][[西武百貨|西武百貨]]在香港及中國的業務，並在中國多個城市開設分店。2000年[[科網熱潮|科網熱潮]]時期曾創辦[[廸生數碼|廸生數碼]]，進軍科技業務，並曾在[[地鐵|地鐵]][[九龍站|九龍站]]開設[[廸生數碼世界|廸生數碼世界]]，但在2005年結束。2005年在[[旺角|旺角]][[朗豪坊|朗豪坊]]開設西武百貨分店，2005年9月在中環[[置地廣場|置地廣場]]開設 [[Harvey_Nichols|Harvey Nichols]] 百貨店。2006年底於[[尖沙咀|尖沙咀]][[九龍酒店|九龍酒店]]再開設西武百貨分店。

集團現時之零售網絡合共248間店鋪，分佈於香港、中國大陸、台灣、新加坡、馬來西亞和菲律賓等地。香港佔銷售額佔65%、台灣佔18%、中國佔8%，而其他東南亞地區則佔9%。

== 娛樂事業 ==
1984年，潘廸生進軍[[電影|電影]]業，創辦[[德寶電影公司|德寶電影公司]]。開拍過多部電影，並請來多位演員演出，包括[[周潤發|周潤發]]、[[許冠文|許冠文]]、[[楊紫瓊|楊紫瓊]]、[[林子祥|林子祥]]等，代表作有《[[秋天的童話|秋天的童話]]》。他亦曾開辦[[DMI唱片|DMI唱片]]公司，並為[[陳百強|陳百強]]等歌手推出唱片，唯公司於1992年結業。

== 家庭 ==
潘廸生有3段婚姻，首任妻子[[楊敏德|楊敏德]]是[[上海|上海]][[紡織|紡織]]商人[[楊元龍|楊元龍]]的長女，為他誕下一名女兒[[潘楚穎|潘楚穎]]。第二任妻子為[[動作|動作]]女演員[[丹斯里|丹斯里]][[楊紫瓊|楊紫瓊]]。現任妻子則為[[余桂珠|余桂珠]]，亦出身富裕，二人在1992年結婚，余桂珠並為他誕下四名小孩。

潘廸生排第四，有三位姊姊其中的一位姊姊[[潘金翠|潘金翠]]，是[[東亞銀行|東亞銀行]]主席[[李國寶|李國寶]]妻子。

== 名下馬匹 ==
潘廸生是[[香港賽馬會|香港賽馬會]]會員，1991年開始養[[馬|馬]]，擁有馬匹有[[輕輕鬆鬆|輕輕鬆鬆]]、[[跑聖|跑聖]]等，1997年，潘廸生以60萬美元購入[[法國|法國]]馬[[健步如飛|健步如飛]]，在1998年競賽時發生意外而需[[人道毀滅|人道毀滅]]，其後潘廸生入禀香港法院，向馬會及已故騎師[[簡慧榆|簡慧榆]]追討賠償，成為香港首宗因馬匹遭人道毀滅而向馬會索償的案件。1999年，潘廸生再購入一匹日本馬[[閃閃星|閃閃星]]，因右後膝蓋有碎骨而需提早退役。
