'''潘德利斯·弗格里斯''' ({{lang-el|Παντελής Βούλγαρης}}; 1940年10月2日-) 是一位[[希腊|希腊]]电影导演和编剧。

== 事业 ==
1972年的首部电影 ''To proxenio tis Annas'' 赢得Thessaloniki Greek Film Festival 首奖。1989年电影 ''The Striker with Number 9'' 入选第39届[[柏林电影节|柏林电影节]]<ref name="Berlinale 1989">{{cite web |url=http://www.berlinale.de/en/archiv/jahresarchive/1989/02_programm_1989/02_Programm_1989.html |title=Berlinale: 1989 Programme |accessdate=2011-03-11 |work=berlinale.de}}</ref>。两年后又以 ''Quiet Days in August'' 竞争第41届[[柏林影展|柏林影展]]<ref name="Berlinale 1991">{{cite web |url=http://www.berlinale.de/en/archiv/jahresarchive/1991/02_programm_1991/02_Programm_1991.html |title=Berlinale: 1989 Programme |accessdate=2011-03-26 |work=berlinale.de}}</ref>。2005年以 ''Brides'' 入选第27届[[莫斯科国际电影节|莫斯科国际电影节]]<ref name="Moscow2005">{{cite web|url=http://www.moscowfilmfestival.ru/miff34/eng/archives/?year=2005 |title=27th Moscow International Film Festival (2005) |accessdate=2013-04-14 |work=MIFF |deadurl=yes |archiveurl=https://web.archive.org/web/20130403124619/http://moscowfilmfestival.ru/miff34/eng/archives/?year=2005 |archivedate=2013-04-03 }}</ref>。2014年以《[[小英格兰|小英格兰]]》获得第17届[[上海国际电影节|上海国际电影节]]最佳影片、最佳导演和最佳女演员三个大奖<ref>{{cite web| url=http://www.sd.xinhuanet.com/news/2014-06/23/c_1111275710.htm| language=| title=上影节希腊《小英格兰》大胜 黄海波新片获评大奖| publisher=新浪娱乐| date=2014-06-23| accessdate=| deadurl=yes| archiveurl=https://web.archive.org/web/20160304111229/http://www.sd.xinhuanet.com/news/2014-06/23/c_1111275710.htm| archivedate=2016-03-04}}</ref>。

== 个人生活 ==
其妻子Ioanna Karystiani是一位女作家、编剧。
