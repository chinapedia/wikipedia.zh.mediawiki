'''鞑靼斯坦共和国'''（{{lang-rus|Респу́блика Татарста́н|Respublika Tatarstan}}，{{lang-tt|Татарстан Республикасы}}）是一个在[[俄羅斯|俄羅斯]]境內的[[俄罗斯自治共和国|共和国]]，屬[[伏爾加聯邦管區|伏爾加聯邦管區]]。面积6.8万[[平方公里|平方公里]]，人口3,779,800（2002年數據），首府[[喀山|喀山]]。

== 历史 ==
[[鞑靼斯坦|鞑靼斯坦]]一带最早的国家是[[伏尔加保加利亚|伏尔加保加利亚]]，他们有贸易商来往[[波羅的海|波羅的海]]、[[中東|中東]]、[[中亞|中亞]]，后被[[蒙古帝国|蒙古]]入侵，成为[[金帳汗國|金帳汗國]]附庸。后分裂出[[喀山汗国|喀山汗国]]。

喀山汗国原为金帐汗国属地，1438年为鞑靼贵族[[兀鲁·穆罕默德|兀鲁·穆罕默德]]所建，首府喀山城。初期国势强盛，曾经于1445年击败[[莫斯科大公国|莫斯科大公国]]军队，俘获大公[[瓦西里二世|瓦西里二世]]。1469年[[沙皇|沙皇]][[伊凡三世|伊凡三世]]利用汗国的内讧，曾夺取喀山城。1487年莫斯科大公国再度占领喀山，并扶植了以{{link-en|穆罕默德·阿明|Möxämmädämin of Kazan}}为首的傀儡政府，汗国成为俄国的属地。1504年阿明摆脱了俄国的控制，1521年汗国同[[克里米亚汗国|克里米亚汗国]]和[[诺盖人|诺盖人]]结盟，并起兵包围了莫斯科。不久，汗国依附[[鄂圖曼帝国|鄂圖曼帝国]]。1552年[[伊凡四世|伊凡四世]]亲率15万大军，携大炮150门，进攻喀山，汗国灭亡。在喀山汗国亡国后，其它几个[[鞑靼人|鞑靼人]]汗国相继被俄国征服。

在[[俄罗斯帝国|沙俄]]和[[苏联|苏联]]时期，以[[俄罗斯人|俄罗斯人]]为主的政府对鞑靼人采取了分化和同化政策。现在的[[巴什基尔人|巴什基尔人]]和[[楚瓦什人|楚瓦什人]]可以被看成是鞑靼人的分支，[[巴什基尔语|巴什基尔语]]是鞑靼语的方言，楚瓦什人是改信[[东正教|东正教]]的鞑靼人。今日鞑靼斯坦的鞑靼人民族文化和宗教在俄罗斯突厥人保存得最为完整，民族意识也相当强烈。鞑靼人也是[[泛突厥主义|泛突厥主义]]思想发起者，源自19世纪鞑靼人知识分子，后被[[土耳其|土耳其]]继承和传播。

[[鞑靼苏维埃社会主义自治共和国|鞑靼苏维埃社会主义自治共和国]]成立于1920年5月27日，当时隶属于[[俄罗斯苏维埃联邦社会主义共和国|俄罗斯苏维埃联邦社会主义共和国]]，也是俄罗斯唯一的韃靼人[[自治共和国|自治共和国]]。1990年8月30日，鞑靼斯坦发表主权宣言，改国名为鞑靼斯坦共和国。

== 政治 ==
进入20世纪80年代末90年代初，[[苏联|苏联]]政局剧烈动荡，联盟面临[[蘇聯解體|解体]]，1990年8月30日，[[鞑靼苏维埃社会主义自治共和国|鞑靼苏维埃社会主义自治共和国]]最高苏维埃发表主权宣言，改国名为'''鞑靼斯坦共和国'''，希望脱离苏联和俄罗斯独立，宣布除国防外，其他一切权利归共和国所有。1991年苏联解体后，鞑靼境内[[分离主义|分离主义]]势力膨胀，希望脱离俄罗斯联邦的势力于1992年3月21日发动了一场[[全民公投|全民公投]]，结果希望独立的人占全共和国选民的61.4%。

为了制止[[独立运动|民族分离运动]]，時任[[俄罗斯总统|俄罗斯总统]][[叶利钦|叶利钦]]发动[[第一次車臣戰爭|车臣战争]]，试图以武力制止分离运动，[[俄羅斯聯邦政府|俄羅斯聯邦政府]]与鞑靼斯坦共和國政府签署第一个双边分权条约，以条约形式和平划分利益和权限范围，制约地方分离主义倾向的过度蔓延。1994年2月15日，俄罗斯联邦与鞑靼斯坦共和国签署了《[[俄罗斯联邦国家权力机关与鞑靼斯坦国家权力机关之间关于划分管辖范围和相互授权的条约|俄罗斯联邦国家权力机关与鞑靼斯坦国家权力机关之间关于划分管辖范围和相互授权的条约]]》，条约规定，鞑靼斯坦共和国作为[[主权国家|主权国家]]与俄罗斯联邦[[自由联合|联合]]，可以参加国际关系和对外经济关系，在税收、财政等方面享有特权。这个条约给予鞑靼斯坦超出俄罗斯宪法规定的权力，实际上鞑靼斯坦成为留在俄罗斯联邦内的具有特殊地位和特权的国家。2017年7月24日，签订于1994年的双边条约到期。这使得俄罗斯境内最后的一个[[共和国|共和国]]失去其特殊地位。

== 地理 ==
鞑靼斯坦共和国位于东欧平原东部、伏尔加河中游地段、俄罗斯中央地区和伏尔加河流域地区的结合部，面积6.8万平方公里。疆域南北相距290公里，东西相距460公里。该共和国西接[[楚瓦什共和国|楚瓦什共和国]]，东接[[巴什科尔托斯坦共和国|巴什科尔托斯坦共和国]]，西北接[[马里埃尔共和国|马里埃尔共和国]]，北接[[乌德穆尔特共和国|乌德穆尔特共和国]]和[[基洛夫州|基洛夫州]]，南接[[奥伦堡州|奥伦堡州]]、[[萨马拉州|萨马拉州]]、[[乌里扬诺夫斯克州|乌里扬诺夫斯克州]]。境内90%的领土属低地平原，森林面积占共和国总面积的六分之一。境内主要河流有伏尔加河（流经鞑靼斯坦共和国境内的长度是177公里）和[[卡马河|卡马河]]（流经鞑靼斯坦共和国境内的长度是380公里），水域面积占领土的6.4%。

== 气候 ==
鞑靼斯坦共和国处于[[温带|温带]]，气候属[[温带大陆性气候|温带大陆性气候]]，冬季气温在零下15℃～零下25℃之间，夏季气温在17℃～25℃之间，植物生长期是170天。

== 行政区划 ==
[[File:Qazan_agglomeration.png|250px]]
鞑靼斯坦行政单位的数量有43个区、10个共和国附属市、8个区属市、9个市属区、24个市镇、833个行政农庄。喀山市是鞑靼斯坦共和国的行政中心，也是最大城市，建于1438年，现有人口约110.5万（2002年）。从莫斯科市到喀山市的距离为797公里。其它主要城市有：[[阿尔梅季耶夫斯克|阿尔梅季耶夫斯克]]市，1953年建市；[[切尔内|切尔内]]市，1930年建市；[[布古利马|布古利马]]市，1781年建市；[[叶拉布加|叶拉布加]]市，1780年建市；[[扎因斯克|扎因斯克]]市，1978年建市；[[泽廖诺多利斯克|泽廖诺多利斯克]]市，1932年建市；[[列宁诺戈尔斯克_(俄罗斯)|列宁诺戈尔斯克]]市，1955年建市；[[下卡姆斯克|下卡姆斯克]]市，1966年建市；[[奇斯托波尔|奇斯托波尔]]市，1781年建市。

== 人口民族 ==
鞑靼斯坦的人口多年来保持稳定数字，2002年统计为377.98万。其中城市人口占73.8%，农村人口占26.2%；男性占46.3%，女性占53.7%。民族方面，鞑靼人占52.9%，俄罗斯人占35.9%，楚瓦什人占3.3%，其它民族为4.2%。而在1989年，鞑靼人占48.1%，俄罗斯人占45.5%，楚瓦什人占3%，其它占3.4%。
