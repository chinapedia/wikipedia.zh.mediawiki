{{日本作品|link=no|銀魂|ぎんたま|Gintama}}是[[日本漫畫家|日本漫畫家]][[空知英秋|空知英秋]]的[[少年漫畫|少年漫畫]]作品，於2003年12月8日開始連載。

本作是以科幻[[時代劇|時代劇]]為題材的搞笑漫畫。以風格獨特的毒舌、吐槽、無厘頭、異想天開的惡搞，為作品的特色與賣點，但也不時穿插感人的故事，以及人生道理的描寫。

2003年12月8日起[[漫畫|漫畫]]版的《銀魂》於[[集英社|集英社]]旗下雜誌《[[週刊少年Jump|週刊少年Jump]]》連載<ref>{{cite web|url=http://jump.shueisha.co.jp/henshu/backnumber/2004/02.html|title=2004年新年2号|publisher=[[集英社|集英社]]|language=ja|archiveurl=https://web.archive.org/web/20080205050754/http://jump.shueisha.co.jp/henshu/backnumber/2004/02.html|archivedate=2008年2月5日|accessdate=2014年2月3日|deadurl=yes}}</ref>，71冊[[單行本|單行本]]於2004年4月2日至2017年12月4日發售<ref>{{cite web|url=http://books.shueisha.co.jp/CGI/search/syousai_put.cgi?isbn_cd=4-08-873623-0&mode=1|title=銀魂―ぎんたま― 1|publisher=[[集英社|集英社]]|accessdate=2004年4月2日|language=Ja}}</ref><ref>{{cite web|url=http://books.shueisha.co.jp/CGI/search/syousai_put.cgi?isbn_cd=978-4-08-881226-7&mode=1|title=銀魂―ぎんたま― 71|publisher=[[集英社|集英社]]|accessdate=2017年12月4日|language=Ja}}</ref>。截至2015年7月，漫畫至第60卷發行為止，累積總計賣出達到6000萬本以上<ref>{{cite web|archiveurl=https://web.archive.org/web/20130827193957/http://shop.beams.co.jp/news/20130624_210996.html|url=http://shop.beams.co.jp/news/20130624_210996.html|archivedate=2013年8月27日|title=銀魂 x MANGART BEAMS TコラボTシャツ|date=2013年6月24日|publisher=Beams|accessdate=2014年11月23日|language=ja|deadurl=yes}}</ref>，其原因與電影版的熱潮有莫大的關係。

2013年1月17日，[[腾讯动漫|腾讯动漫]]与集英社达成协议，获得本作电子版发行权<ref>{{cite web |url=http://games.qq.com/a/20130117/000132.htm|title=腾讯游戏签约集英社11部漫画 推出火影忍者OL |accessdate=2013年2月6日|publisher=腾讯动漫|date=2013年1月17日|language=zh-cn}}</ref>。

== 劇情大綱 ==
{{main|銀魂角色列表}}

{{more plot}}
[[江戶時代|江戶時代]]末期（本作的20多年前），被稱為「天人」（{{lang|ja|あまんと}}）的謎之[[外星人|外星人]]來襲，人類與天人之間的戰爭瞬即爆發，為數眾多的武士和攘夷派志士均參與戰鬥。

[[幕府_(日本)|幕府]]見識到天人強大的實力後，立刻低頭，放棄武士不管，並與天人簽訂[[不平等條約|不平等條約]]，准許他們入國。之後「[[廢刀令|廢刀令]]」頒發，使武士們已經忘記怎樣去貫徹自己的武士道。自此，幕府為天人所控制，形成「[[傀儡政權|傀儡政權]]」，造成權利的腐敗，幫派和惡徒也因而肆意猖狂。

在這樣的時代，一個仍舊保留著武士之魂的男人[[坂田銀時|坂田銀時]]與同伴[[志村新八|志村新八]]、[[神樂_(銀魂)|神樂]]经营着萬事屋（只要有錢什麼工作都接），愉快地過著異想天開的生活。

== 世界觀 ==
故事背景設定在日本的[[江戶幕府|江戶幕府]]後期的[[江戶|江戶]]城（[[東京|東京]]），在故事中被稱作「大江戶」，是一個由於外星人「天人」（あまんと）突然降臨，要求日本幕府開國，於是讓江戶從而迅速[[現代化|現代化]]並星際化的空想世界。「大江戶」可說是想象中，一個從江戶幕府一下子直接躍入星際時代的東京，因此在19世紀[[幕末|幕末]]的日本風土的基礎上，又融合了20世紀的現代科技，以及[[未來|未來]][[科幻|科幻]]色彩等元素，形成十分新鮮而奇妙的獨特氣氛。其中天人的進入實則揉合了當時歷史上德川幕府備受外國勢力侵犯的寓意。

在《銀魂》的大江戶世界中，除了可以見到[[江戶時代|江戶時代]]的和服、留著[[月代|月代]][[髷|髮髻]]的男性、梳著[[島田髷|島田髷]]的女性、木造的長屋，以及[[武士|武士]]、[[劍客|劍客]]、[[忍者|忍者]]等[[時代劇|時代劇]]所特有的風景外，還同時也出現[[摩天大樓|摩天大樓]]、[[電視|電視]]、[[電腦|電腦]]、[[汽車|汽車]]、[[手機|手機]]等[[現代|現代]]科技，以及[[外星人|外星人]]、[[太空船|太空船]]與[[仿生人|人造人]]等未來世界才存在的科幻要素。故事中並未出現[[明治維新|明治維新]]的新政權，仍是[[德川幕府|德川幕府]]的天下，但[[幕府將軍|幕府將軍]]實際上不過是被「天人」（外星人）幕後操控的一個傀儡政權罷了；當時的武士們則一如歷史中的幕末時代一樣，分為守護幕府的勢力，以及想推翻幕府的[[攘夷|攘夷]]志士。

'''攘夷活動相關'''
:; 攘夷戰爭（じょういせんそう）
:: 天人侵略地球時爆發持續二十年的戰爭，坂田銀時、桂小太郎、坂本辰馬、高杉晉助、黑子野太助都曾參與過后半期的活动。
:; 攘夷志士（じょういしし）、攘夷浪士（じょういろうし）
:: 為了拯救國家而打算排除天人的人們，並對天人支配的幕府採取[[恐怖活動|恐怖活動]]，甚至還有很多人藉此作惡。
:; 鬼兵隊（きへいたい）
:: 高杉晉助在攘夷戰爭中率領的義勇軍。戰爭結束後差點被殲滅，現在又由河上萬齋、來島又子、武市變平太和岡田似藏等人的加入而復活。
:: 鬼兵隊之名名字是參照[[高杉晉作|高杉晉作]]在[[長州藩|長州藩]]率領的「[[奇兵隊|奇兵隊]]」。
:; 姉古原之戰（あねこはらのたたかい）
:: 攘夷戰爭之後爆發的幕府權力之爭，鞏固一橋派的勢力，銀時此時受到池田夜右衛門的幫助，秘密逃獄。名字參照[[姉川之戰|姉川之戰]]與[[關原之戰|關原之戰]]。

'''外星相關'''
:; 天人（あまんと）
:: 對來訪地球的外星人的稱呼，沒有特定的種族。天人的種族單位皆由星系國家所形成，故事中已確認有天人的國家大使和大使館。
:; 異形（えいりあん）
:: 天人以外的外星生物之通稱。
:; 春雨（はるさめ）
:: 銀河系裡勢力龐大的[[宇宙海盜|宇宙海盜]]，組織系統由元老為首，其下則有提督和十二師團。和鬼兵隊之間是結盟關係，而神樂的哥哥神威則是第七師團團長。
:; 天導眾（てんどうしゅう）
:: 控制幕府的十二人集團，背後還從事犯罪活動。
:; 龍宮城
:: 由名為「乙姬」的天人所統治的[[海底|海底]][[渡假村|渡假村]]，屬於「移動都市國家」。
:: 名字是參考[[浦島太郎|浦島太郎]]登場的龍宮（龍宮城）。
:; 毘夷夢星（びいむせい）
:: 開發星際波動激光砲的星球。
:: 名稱參考物理名詞[[射束|射束]]（beam）。
:; 凹凸神教
:: 信奉凹凸神的一個來自宇宙的教團，核心成員皆為雙性人，他們所謂的神其實是一個衛星兵器，能散布一種能逆轉性荷爾蒙的病毒，這種病毒在空氣中無法存活太久，因此不會傳染，凹凸神教利用這種病毒在許多星球進行非法的傳教活動。
:; 玄武族
:: 有鋼鐵般的肉體，即使是子彈也難傷絲毫（詳細可參考動畫29集）。
:; 千鳥（ちどり）
:: 銀河系裡勢力龐大、規模僅次於春雨的[[宇宙海盜|宇宙海盜]]，在地球上以販賣人口來獲取利益，已解散。

:; 艾特納（{{lang|ja|アルタナ}}）
:: 天人發現的一種新能源，號稱是「循環大地的能量洪流」「星球的生命力」，地球人則稱之為「龍脈」。
:: 人們掌握運用艾特納換成實用資源，宇宙才得以蓬勃發展，星系間長距離移動也變得快捷，同時也發展出高毀滅性武器，因此展開多次爭奪「艾特納」的戰爭。後來人們終於了解其危險性，於是互相簽訂互不侵犯他國艾特納的條約。艾特納的大門「洞口」（穴）由各國聯合成立的組織—「艾特納保全協會」所控管，也就是後來「天導眾」的前身，也因此他們控制了許多星球，榨乾他們的能源。
:: 而天導眾執著於地球上的艾特納的原因，可能是因為地球上的艾特納擁有比其他星球更獨特的力量。位於洞口上方擁有超脫常識的巨大植被，洞口上面的泉水據說能夠治百病，這份力量帶給生態定律未知的巨大影響力，而這份力量至正體現在了天照院奈落的始祖「虛」（＝吉田松陽）的身上。
:: 目前已知的和艾特納有关的有两人：'''虚'''；'''江华'''。

'''傭兵部族'''
:; 夜兔族（やとぞく）
:: 最強最兇狠的傭兵一族，曾摧毀無數星球。有「弒親」的習俗，因而走上滅絕的道路。外表和人類無異，打扮和穿著都有相當濃厚的中華風，其特色為白到近乎透明的皮膚，且具有誇張的怪力以及可怕的恢復力，無論傷再重都可以一夜間復原（骨折例外，也無法生出斷肢）。
:: 弱點是陽光，被稱為「被太陽拋棄的一族」。主力武器是蠻力，[[油紙傘|油紙傘]]更是不離手，因為紙傘末端能發射[[子彈|子彈]]，而且傘面能防彈。有些會在身上纏著[[繃帶|繃帶]]。強健的體魄會消耗許多能量，所以有相當大的食量。
:: 夜兔的母星名叫「徨安」，遭到行星联盟总攻击后被毁灭成为死星。
:: 在本作登場的夜兔族有神樂、星海坊主、神威、阿伏兔、雲業、鳳仙、陸奧、江華。
:; 荼吉尼族（だきにぞく）
:: [[鬼_(日本)|惡鬼]]容貌的種族。原本是一群既好戰又兇殘的種族，但其實上也有像屁怒絽那樣溫和的人存在。頭上的角被視為「荼吉尼族的驕傲」，分別象徵尊嚴與靈魂，所以絕對不能折斷。由於體質敏感而不能洗熱水澡，一接觸到[[潤滑液|潤滑液]]就會燃燒。
:: 在本作登場的荼吉尼族有屁怒絽及其家人、鬼獅子、王蓋(荼吉尼傭兵首領)。

:; 辰羅族（しんらぞく）
:: 外表猶如[[精靈|精靈]]的種族。其特徵為尖耳和額頭上的白毫，執行暗殺任務時都會將臉用布包住。和夜兔、荼吉尼不同，辰羅擅長團體戰，會以犧牲個人來達到團體的勝利。
:: 在本作登場的辰羅族是歌舞伎町四天王之一的孔雀姬華陀、蒼達(辰羅傭兵首領)。
:; 蓮蓬族（れんほうぞく）
:: 伊莉莎白所屬的種族。有「白色幻影」之稱，跟夜兔族一樣有著人類的外表，穿著白色布偶裝，內部只穿內褲。蓮蓬族服裝非常相似，除了伊莉莎白未作裝飾外，大多會在服裝上加上有個人特色的裝飾。可放置戰鬥用品，也可作為太空衣使用。
:: 食物為[[紅白機|紅白機]][[卡帶|卡帶]]（[[勇者鬥惡龍系列|DQ]]和[[太空戰士系列|FF]]都是隊長才能吃的高級品，DQ5和FF4則是將軍吃的）。蓮蓬次世代型白衣計畫的產物對抗武士用的，白色惡魔-頑侍（長的像鋼彈，日文發音也諧同鋼彈），所以被新八形容蓮蓬軍為聯邦軍。
:: 從桂的口中得知人類3個重要的袋子：薪水袋、老媽（日語'''お袋'''，媽媽的俗稱）、陰X。
:: 蓮蓬的首領為母艦的中樞系統「SAGI」（其實是泡在水裡的腦袋，名稱參考自[[Evangelion|新世紀福音戰士]]中的超級電腦「MAGI」）能讓母艦變成巨大的蓮蓬。
:; 毘夷夢族（びいむぞく）
:: 原形源自《[[星球大戰|星球大戰]]》中的[[尤達|尤達]]。

== 外傳作品 ==
; 3年Z班銀八老師（さんねんズィーぐみ ぎんぱちせんせい）
: 惡搞曾在『赤丸JUMP』刊登兩次的『3年Z班金八老師』。銀時所扮演的「銀八老師」戴著眼鏡、穿白衣、在講桌前叼着菸、總說著無關要緊的事。動畫版里银八叼的不是菸而是糖果（棒棒糖）。銀八老師嘴裡的棒棒糖會冒菸是因為他高速舔棒棒糖。班上學生是《銀魂》里的主要登場人物（新八、神樂、阿妙、近藤、土方、沖田、桂、長谷川等人），教员由另一部分主要人物担任（月咏、全藏、八郎之母等）。
: 之後的《銀八老師》是由大崎知仁將它小說化，單行本發售了7本（參考小說項目）。接著小說從『JUMP SQ』的創刊號開始連載到2008年的5月號止。動畫DVD該季第1卷限定版都會附贈收錄銀八老師日常對話的DRAMA CD。
: 動畫版會不定期播放「請告訴我!銀八老師」由銀八來回答觀眾對於銀魂本篇的相關問題。在第4期的ED以雲和照片的形式表示出銀八老師劇中景象，第16期的ED以畢業典禮的模樣登場。

; 金魂（きんたま）
: 金魂篇，主角由坂田銀時變成了一個有著一頭金色離子燙、衣服顏色和銀時完全相反、性格親切大方、深受歌舞伎町群眾喜愛的自稱為“坂田金時”的男人。不僅如此，銀時身邊所有的朋友（登勢、桂、小猿、月詠、日輪、阿妙、九兵衛……甚至志村新八和神樂）都失去了對他的記憶，取而代之的是坂田金時。然而沒有忘記銀時的仍有兩人，不，一台和一隻：小玉和定春。通過小玉的敘述才了解到，由於銀時總是遊蕩在外喝酒，新八和神樂就拜託源外老頭創造了一個集合了銀時所有優點、同時摒棄了他的所有缺點的完美的機器人——坂田金時（全名：超合金製完全體，坂田銀時二號機）。但是這個機器主動抹去了所有人對銀時的記憶，包括創造他的源外老頭，目的據其本人說是為了讓銀魂完結，從而使自己成一部全新漫畫的主人公（雖然成功了但是只有1話，隨後劇情又回到銀魂），而金時無法影響到同為機器人的小玉和身為動物的定春。於是，為了奪回主角的寶座，銀時與金時開始了一場激烈的的鬥爭。最後受銀時等人感化，決定讓自己回爐重造。

== 刊載軼事 ==
《銀魂》開始連載時，總編輯、責任編輯及作者自己，都不看好這部作品未來的前途，並擔心如此有「個性」的作品名（「銀魂」音似「金玉」（きんたま kintama），乃是日文睪丸的俗稱；據說這是作者經家庭會議後決定的）、充滿吐槽題材的風格，能否於《週刊少年Jump》弱肉強食的殘酷世界中生存。

在連載十回左右時，《銀魂》在刊載人氣排行始終一直徘徊在低位，面臨隨時被腰斬的危機。但連載至第11回左右，由於穿插富有人情味的情節，人氣開始逐漸回升；之後獨特的角色設定、吐槽及冷笑話，漸漸為讀者接受，並日益受到高度評價，如今本作已成為Jump的人氣作品之一，至現在仍繼續連載中。
