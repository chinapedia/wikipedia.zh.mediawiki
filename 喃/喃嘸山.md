'''喃嘸山'''，又名'''嘉頓山'''（Garden Hill）和'''水庫山'''、'''煙墩山'''，高90.6米，是[[香港|香港]][[深水埗區|深水埗區]]的一座小山丘<ref name=garden>{{cite news|url=http://hk.apple.nextmedia.com/supplement/travel/art/20140705/18787137|author=邵超 |title=獅子山下 鬧市小丘|work=《[[蘋果日報_(香港)|蘋果日報 (香港)]]》 |date=2014-07-05 |accessdate=2015-01-10}}</ref>。西面山腰靠近[[大埔道|大埔道]]，北面為[[澤安邨|澤安邨]]。

== 歷史 ==
1998年啟德機場關閉前有一座飛機訊號燈塔，有行車徑方便維修員上山。嘉頓山之名來自山下附近的[[嘉頓|嘉頓]][[麵包|麵包]]廠房<ref name="garden"/>，現為附近居民的晨運徑。市面上的地圖只顯示了晨運徑的路徑，並沒有顯示此山的名字，其原因可能是「'''嘉頓山'''」只是居民約定俗成的稱呼，而非政府正式確認的地名。

2017年3月，九龍西立法會議員[[梁美芬|梁美芬]]在立法會財務委員會提出興建觀景台及升降機，希望讓居民及遊客欣賞深水埗日夜美景。結果被[[本土研究社|本土研究社]]成員陳劍青批評有關建議畫蛇添足，並無考慮到實際需要，認為山野景點化，只會帶來破壞。<ref>{{cite news|author1=周詠雯|title=【郊野景點化】梁美芬建議嘉頓山建升降機　陳劍青批呃老人家票|url=https://www.hk01.com/%E7%86%B1%E8%A9%B1/81383/-%E9%83%8A%E9%87%8E%E6%99%AF%E9%BB%9E%E5%8C%96-%E6%A2%81%E7%BE%8E%E8%8A%AC%E5%BB%BA%E8%AD%B0%E5%98%89%E9%A0%93%E5%B1%B1%E5%BB%BA%E5%8D%87%E9%99%8D%E6%A9%9F-%E9%99%B3%E5%8A%8D%E9%9D%92%E6%89%B9%E5%91%83%E8%80%81%E4%BA%BA%E5%AE%B6%E7%A5%A8|accessdate=2017-03-31|publisher=香港01|date=2017-03-31}}</ref>

== 特色 ==
遊人步行15分鐘登上山頂，便能夠俯瞰[[深水埗|深水埗]]和[[石硤尾|石硤尾]]的[[公共屋邨|公共屋邨]]和舊式[[唐樓|唐樓]]，並眺望[[昂船洲大橋|昂船洲大橋]]，是欣賞及拍攝[[夕陽|夕陽]]和[[西九龍|西九龍]]夜景的熱門地點，

在[[啟德機場|啟德機場]]關閉前，為保障機場航道安全，政府為九龍區的建築物訂立安全高度限制，機場關閉前遊人能登上山頂俯瞰[[西九龍|西九龍]]，眺望[[香港島|香港島]]、[[東九龍|東九龍]]和[[大嶼山|大嶼山]]景致。

== 登山途徑 ==
[[File:HK_Garden_Hill_Outdoor_stairs_Lands_Department_No_Climbing.JPG|缩略图]]
[[美荷樓|美荷樓]]青年旅舍（[[石硤尾邨|石硤尾邨]]41座舊址，1954年建成，[[香港一級歷史建築|一級歷史建築]]）及[[萨凡纳艺术设计学院#香港|萨凡纳艺术设计学院]]（SCAD，[[北九龍裁判署|北九龍裁判署]]舊址，[[香港二級歷史建築|二級歷史建築]]）後均有晨運徑可以到達嘉頓山，短短十數分鐘就可登頂。<ref>{{cite book|url=https://books.google.com.hk/books?id=bUCyAAAAQBAJ&pg=PT191&dq=%22%E5%98%89%E9%A0%93%E5%B1%B1%22&hl=zh-CN&sa=X&ei=9jjUVNPwDsTmsATbj4LIBQ#v=onepage&q=%22%E5%98%89%E9%A0%93%E5%B1%B1%22&f=false|title=街知巷聞──行街|author=明報編輯部|year=2013|isbn=9789888206681}}</ref>

== 流行文化 ==
1980至90年代有多部電影及電視在嘉頓山拍攝，例如電影《[[喋血街頭|喋血街頭]]》、《[[一本漫畫闖天涯|一本漫畫闖天涯]]》、[[TVB|TVB]]劇集《[[男儿本色_(2009年电视剧)|男兒本色]]》、《[[怒火街頭|怒火街頭]]》、《[[怒火街頭2|怒火街頭2]]》、[[香港電台|香港電台]]電視劇《性本善》等。在電影《[[喋血街頭|喋血街頭]]》、《[[一本漫畫闖天涯|一本漫畫闖天涯]]》中可見已拆卸的行車徑和飛機訊號燈塔舊貌。
