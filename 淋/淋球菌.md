'''淋球菌'''（學名：{{lang|la|''Neisseria gonorrhoeae''}}，'''奈瑟氏球菌'''）又稱'''淋病雙球菌'''（{{標音|字=淋|拼音=lìn}}）、'''淋病奈瑟菌'''，是導致[[淋病|淋病]]的病原菌，和[[腦膜炎奈瑟菌|腦膜炎奈瑟菌]]同屬於[[奈瑟菌屬|奈瑟菌屬]]，是[[革蘭氏陰性菌|革蘭氏陰性菌]]一個屬種。球菌是需要複雜營養的，在實驗室培養要加入營養補充劑。在巧克力細菌培養基（或熱的血培養基）加入二氧化碳可加速其生長。這些球菌在細胞內的典型隨意排列是成一對的（ diplococci ） 。

== 淋病的症狀 ==
遭淋病雙球菌感染所造成的症狀非常不同，且有10%的男性和80%的女性被感染後是沒有症狀的。感染[[淋病|淋病]]的男性有很高的機會罹患[[前列腺癌|前列腺癌]]。
== 其他症狀 ==
* 引發嬰兒[[結膜炎|結膜炎]]，結膜炎常見於新生嬰兒，[[硝酸銀|硝酸銀]]滴進於他們眼中，作為一項對付淋病雙球菌的預防措施。
* 常見於男性，引發[[尿道炎|尿道炎]]，排尿期間有燒灼感。

== 常用的抗生素 ==
[[淋病|淋病]]是對[[青黴素|青黴素]]家族抗生素有抵抗能力的，現在常用的抗生素是[[頭孢曲松鈉|頭孢曲松鈉]]（ceftriaxone）（第三代[[頭孢菌素|頭孢菌素]]cephalosporin ）。

病人懷疑受到感染，同時也應測試[[衣原體|衣原體]]（''Chlamydia）''感染，因為很多時他們都是協同感染的。
