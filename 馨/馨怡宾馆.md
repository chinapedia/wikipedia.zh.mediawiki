'''馨怡宾馆'''（'''中央军委机关事务管理总局招待局太平路招待所'''），位于[[北京市|北京市]][[海淀区|海淀区]][[永定路|永定路]]169号，隶属[[中央军事委员会机关事务管理总局招待局|中央军委机关事务管理总局招待局]]。

==简介==
馨怡宾馆的前身是[[中国人民解放军工程兵|军委工程兵]]招待所，1954年5月18日成立，1982年前隶属于军委工程兵司令部。1982年至1992年，隶属于中国人民解放军总参谋部工程兵部。1992年，转隶[[中国人民解放军总参谋部军训部|中国人民解放军总参谋部军训部]]<ref name=jj/>。后成为中国人民解放军总参谋部军训部太平路招待所（馨怡宾馆），此后又改为中国人民解放军总参谋部军训部招待所太平路分所（馨怡宾馆）<ref name=jj/><ref name=zhbiao>{{Cite web |url = http://www.ccgp.gov.cn/cggg/dfgg/zbgg/201701/t20170117_7867097.htm |title = 北京地区（城六区）2016-2018年度会议定点单位协议采购项目中标公告：006中标公告附件2：第二包中标供应商名单 |author =  |date = 2017-01-17 |publisher =中国政府采购网  |language =  |accessdate =  |quote =  }}</ref>。2016年，[[中国人民解放军总参谋部|中国人民解放军总参谋部]]撤销，该宾馆转隶中央军委机关事务管理总局招待局，成为中央军委机关事务管理总局招待局太平路招待所（馨怡宾馆）<ref name=jj/>。

馨怡宾馆设有办公室、客房部、馨怡酒楼、总务部、财务。馨怡宾馆经常接待国家机关、军队的各类会议，2016年获评为中央和北京市的北京地区（城六区）2016-2018年度会议定点单位<ref name=jj>{{Cite web |url =http://www.ydl169.com/wenxin/article.asp?articleid=6  |title = 馨怡宾馆简介 |author =  |date =  |publisher = 馨怡网 |language =  |accessdate = 2017-03-28 |quote =  }}</ref><ref name=zhbiao/>。
