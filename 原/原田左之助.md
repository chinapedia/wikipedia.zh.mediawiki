'''原田 左之助'''（[[天保|天保]]11年(1840年) - [[明治|明治]]元年[[5月17日_(旧暦)|5月17日]](1868年7月6日)），[[新撰組|新撰組]]第十隊組長。[[伊予松山藩|伊予松山藩]]出身。[[諱|諱]]為'''忠一'''。從[[谷三十郎|谷三十郎]]那邊學[[種田流|種田流槍術]]並得到免許皆傳的身份。

== 人物 ==
曾為[[伊予松山藩|伊予松山藩]]內的中間(官名)，之後[[脱藩|脱藩]]。原田本人有一點性急，有一次在爭吵時被某位武士認為是「連切腹的禮節都不知道的小吏」，於是原田立刻切腹給他看。幸好傷勢輕微而沒有生命危險，從那次受傷以後就以「不死的左之助」之名在隊上出名。不過在他的腹部依然保持著一字型的傷口，在天氣好的時候會把傷口示眾，讓他們知道這種傷口和他們在故事裡所聽到的是不一樣的。會說我的腹部已經知道了金屬(暗喻刀)的味道了。並把他原來的家徽改為他切腹的一字型傷口的造型。

原田除了有被說是一位愛妻家外，還是一位美男子，和[[永倉新八|永倉新八]]的關係相當好。但原田也是一位個性粗暴的人，在喝酒時會誇耀自己的傷口，大聲叫著「切吧、切吧」。砍殺長州間諜楠小十郎時不但從背後攻擊，砍殺完還邊笑邊說：「啊 真高興」，因此遭到近藤勇的嚴厲斥責。

== 新撰組 ==
從[[近藤勇|近藤勇]]的江戶道場、[[試衛館|試衛館]]以來就在的元老隊士，到後來成為第十隊組長。以作為種田流槍術([[宝藏院流槍術|宝藏院流]])的長槍名手而聞名。身為副長的[[土方歲三|土方歲三]]對於第一隊的[[沖田總司|沖田總司]]和第十隊的原田左之助最為信賴，並經常借用他們的力量。在作戰時作為[[殿軍|殿軍]]的組長而指揮著第十隊參戰，所以在新撰組的重要戰役中一定會有原田的名字活躍著。包括（[[芹澤鴨|芹澤鴨]]一派的粛清、斬殺[[長州|長州]]的間諜・楠小十郎、暗殺大阪西町奉行與力・内山彦次郎、[[池田屋事件|池田屋事件]]、[[禁門之變|禁門之變]]、[[三条制札事件|三条制札事件]]、[[油小路事件|油小路事件]]等）

有一陣子，原田被懷疑是暗殺[[坂本龍馬|坂本龍馬]]的兇手（暗殺現場中有一個掉落的刀鞘，[[伊東甲子太郎|伊東甲子太郎]]作證說是原田和其他二人所有。又當時事件發生時有聽到兇手講出四國的方言（「這渾球！」），實際上暗殺龍馬以[[京都見廻組|京都見廻組]]之説最為有力(新撰組參與的可能性不大)。2004年的NHK大河劇[[新撰組！|新撰組！]]有好好利用當時的傳聞讓暗殺坂本龍馬的犯人為原田左之助的描寫。

原田作為新撰組一員參加[[鳥羽・伏見之戰|鳥羽・伏見之戰]]、[[甲陽鎮撫隊|甲陽鎮撫隊]]等戰爭之後、和永倉新八一起組成[[靖兵隊|靖兵隊]]。但不知什麼原因而離開並前往江戶加入[[彰義隊|彰義隊]]，由於在[[上野戰爭|上野戰爭]]中所受的傷未癒，在明治元年5月17日死亡。得年29，因為太晚入隊，因此隊上的名簿中沒有記載原田的名字。

原田在[[靖兵隊|靖兵隊]]徑過山崎村（福島縣境內）時，退出了隊伍，被推測可能是想要和留在江戶的家人團聚，但是來到江戶附近，卻發現自己無法脫身，於是又加入[[彰義隊|彰義隊]]，當時他和大部份人馬駐紮在寬永寺內。

== 馬賊傳説 ==
據明治四十年所報的新聞，[[甲午戰爭|甲午戰爭]]時，松山傳出有位老軍人說出自己的過往並且自報姓名說「我是原田左之助」。老軍人自報是原田的新聞真假不明。傳說原田是經由上野、[[新潟市|新潟]]、[[下關|下關]]、[[釜山|釜山]]到達中國並成為[[馬賊|馬賊]]的頭目。

== 其他 ==
長子的名字叫茂、這個原田茂的「茂」字是取自於[[江戶幕府|江戶幕府]]第14代[[征夷大將軍|征夷大將軍]][[德川家茂|德川家茂]]。妻子原田雅一直活到昭和6年，然後在許多孫子的看護下死去。

在新選組證言等記錄文件和《新選組三部曲》（[[子母澤寬|子母澤寬]]著）裡原田雅}都有談到自己的丈夫左之助的事情。
