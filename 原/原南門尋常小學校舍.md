'''原南門尋常小學校舍'''位於[[臺南市|臺南市]][[中西區_(臺灣)|中西區]]、日治時期的[[南門町|南門町]]，於民國九十年（2001年）7月16日公告為臺南市歷史建築{{r|文}}。該建築在[[臺灣日治時期|臺灣日治時期]]時是「南門尋常小學校」（台南第二尋常小學校、南門國民學校），二次大戰後曾一度作為臺南市政府，現在則是[[台南市立建興國民中學|台南市立建興國民中學]]的校舍{{r|王}}。

== 沿革 ==
[[File:初代臺南市政府廳舍.jpg|thumb]]
南門尋常小學校在日治時期的大正四年（1915年）4月申請設立，當時是叫作「台南第二尋常高等小學校」{{NoteTag|1=相對於該校之名的「台南第一尋常高等小學校」後來改稱「花園尋常小學校」，位於今臺南市立公園國民小學{{r|王}}。}}，為專供日人子女就讀的學校{{r|傅}}。由於當時還沒建好校舍，便暫借第一尋常小學的校舍（今[[臺南一中|臺南一中]]校舍{{NoteTag|1=當時的「台南第一尋常高等小學校」位於竹園町，於大正十二年（1923年）時才遷往花園町，原校舍讓渡予臺南第二中學校使用。}}）上課，直到大正八年（1919年）校舍完工後才遷入新址{{r|王}}。

後來因教育制度更改、以及當地於臺南市町名改正，該校於大正十年（1921年）改稱為「南門尋常小學校」{{r|王}}。而因昭和九年（1934年）增設了高等科，所以又改稱「南門尋常高等小學校」{{r|傅}}，後來到了昭和十六年（1941年）又改為「南門國民學校」{{r|王}}。
[[File:原南門尋常小學校舍左邊.JPG|220px]]
後來二次大戰後，學校本身於民國三十四年（1945年）12月1日遷移到今臺南市永福路二段86號，改名為「臺南市中區第二國民學校」；民國36年（1947年）2月1日，又改為「臺南市中區成功國民學校」，但該年9月又改為「臺南市中區永福國民學校」，即今臺南市立永福國民小學{{r|永}}。至於校舍本身成為了[[臺南市政府|臺南市政府]]廳舍使用；一直到民國五十八年（1969年）向[[中華民國空軍|中華民國空軍]]收回[[原臺南州廳|原臺南州廳]]，方遷至該處辦公（即今[[國立臺灣文學館|國立臺灣文學館]]）{{r|王}}。而民國六十二年（1973年）又曾進行全面整修，對結構進行補強並抽換了樓板與屋頂{{r|傅}}。

臺南市道路「府前路」即因道路在戰後重新命名時，市府座落在此而訂{{r|王}}。唯市府遷走後，現址由建興國中使用至今但道路名未易。

== 建築特色 ==
該建築坐南朝北，主要使用磚材，為對稱的一字型平面建築，面寬超過百公尺，而有簡單門廊的主入口便位在中間{{r|傅}}。門廳內有四根方壁柱和四根圓柱，柱頭模仿得像[[愛奧尼克柱式|愛奧尼克柱式]]，但為方形{{r|傅}}。南面有平拱走廊，於東西兩端有圓拱形門洞{{r|王}}。
[[File:臺南福安坑溪今貌.JPG|250px]]
== 其他 ==
民國九十九年（2010年）初，建興國中拆除了前門右側圍牆，令[[臺灣府城|臺灣府城]]昔日兩大溪流之一的[[福安坑溪|福安坑溪]]部分重現，但該溪已不復溪流原貌，較像帶有古味的溝渠{{r|王}}。
