'''W．原璞'''是聾人胡慧賢創立的[[曲奇|曲奇]]品牌，主要使用[[有機|有機]][[麵粉|麵粉]]、[[腰果|腰果]]和[[香油|香油]]等材料製成<ref>{{cite web |language=zh-hk |url=http://www.sharpdaily.hk/article/index/4104/20111028/18826 | title=爽專題：失聰女 曲奇代說話 |publisher=[[爽報_(香港)|爽報 (香港)]] |date=2011-10-28 |accessdate=2012-03-09}}</ref>。

==成立背景==
胡慧賢在[[真鐸學校|真鐸學校]]畢業後，她求職屢受挫折，試過被騙，失業數載，賦閒在家。她自學烘焙有機曲奇，建立自家品牌，推廣[[公平貿易|公平貿易]]<ref>{{cite web |language=zh-hk |url=http://www1.hk.apple.nextmedia.com/template/apple/art_main.php?iss_id=20111028&sec_id=4104&art_id=15748676 | title=聽障女受盡歧視 創品牌推廣公平貿易 製曲奇拾自信 |publisher=[[蘋果日報_(香港)|蘋果日報 (香港)]] |date=2011-10-28 |accessdate=2012-03-09}}</ref>。

==原璞的由來==
原︰原本；原始之意。

璞︰就是反璞，現今無論人心及物質都距離自然純璞太遠，現在就要是開始回向歸真的時候，一切就像一個圓，由起點開始，一路行到某時間亦會回歸起點。

==特色==
* 曲奇的材料是來自[[果仁|果仁]]、[[花生|花生]]及[[燕麥|燕麥]]，也用了公平貿易腰果醬製成，亦沒有添加[[牛油|牛油]]。
* 全部使用[[素食|素食]][[材料|材料]]，用[[豆奶|豆奶]]及亞麻籽代替[[雞蛋|雞蛋]]、[[牛奶|牛奶]]及牛油等成份<ref>{{cite web |language=zh-hk |url=http://www.ellaws.com/images/news/WeekendWeekly.pdf |title=失聰少女的手造曲奇 |publisher=[[新假期|新假期]] |date=2011-03 |accessdate=2012-03-09 }}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。
* 「啖忘憂」曲奇，用[[肉桂|肉桂]]製成，中醫角度具除冷、止痛作用。

==協助機構==
胡慧賢是聾人機構「[[龍耳|龍耳]]」的會員，該機構積極協助她融入社會，盼喚起社會關注聾人<ref>{{cite web |language=zh-hk |url=http://www1.hk.apple.nextmedia.com/template/apple/art_main.php?iss_id=20111028&sec_id=4104&subsec_id=11867&art_id=15748678 | title=聽障人士支援少 |publisher=[[蘋果日報_(香港)|蘋果日報 (香港)]] |date=2011-10-28 |accessdate=2012-03-09}}</ref><ref>{{cite web |language=zh-hk |url=http://www.mingpaohealth.com/cfm/health3.cfm?File=20090618/emotion/zka1.txt |title=用心聆聽 用手溝通 |publisher=[[明報|明報]] |date=2009-06-18 |accessdate=2012-03-09 }}{{Dead link|date=2018年7月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>。
