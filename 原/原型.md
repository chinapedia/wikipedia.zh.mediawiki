'''原型'''是首創的模型，代表同一類型的人物、物件、或觀念。原型在[[文學|文學]]與[[心理學|心理學]]中很重要。原型往往被以後的[[作者|作者]]一再的重複模仿與重塑。

== 心理學中的原型 ==

[[榮格|榮格]][[分析心理學|分析心理學]]中有四大原型：
* '''[[自性|自性]]'''（Self）：[[自性|自性]]是人心靈的中心。
* '''[[阿尼瑪|阿尼瑪]]'''（Anima）：[[阿尼瑪|阿尼瑪]]是男人[[無意識|無意識]]中的女人性格與形象。[[阿尼瑪|阿尼瑪]]可以是負面的也可以是正面的。
* '''[[阿尼瑪斯|阿尼瑪斯]]'''（Animus）：[[阿尼瑪斯|阿尼瑪斯]]是女人[[無意識|無意識]]中的男人性格與形象。[[阿尼瑪斯|阿尼瑪斯]]可以是負面的也可以是正面的。
* '''[[陰影_(心理學)|陰影]]'''（Shadow）：'''陰影'''是人[[無意識|無意識]]中的與自我（ego）相反的性格與形象。在《[[蝙蝠俠:開戰時刻|蝙蝠俠:開戰時刻]]》中，想要成為正義的悍衛者的布魯斯心中存在[[殺人復仇|殺人復仇]]與[[恐懼|恐懼]]的[[陰影|陰影]]。這兩種陰影分別由[[忍者大師|忍者大師]]（Ra's al Ghul）與[[稻草人|稻草人]]（Sacrecrow）代表。經由[[阿尼瑪|阿尼瑪]]瑞琪兒的導引，[[老智者|老智者]]们的幫助，布魯斯戰勝自己心中殺人復仇與恐懼的陰影，終於成為正義的悍衛者。

其他分析心理學原型：

* '''[[死亡|死亡]]'''：[[閻羅王|閻羅王]]，[[死神|死神]]。

=== 原型心理學 ===
[[榮格|榮格]][[分析心理學|分析心理學]]衍生出[[原型心理學|原型心理學]]（Archetypal Psychology）。
荣格心理学对于大多心理问题的形成一般归结为某一“原型”没有得到良好发展而受到阻碍，由此精神系统作出自我调整而表现为神经症或别的问题。荣格心理学治疗目标因此是使受挫折的原型获得应有的发展。

== 文藝著作中的原型 ==
{{seealso|定型角色}}
以下為一些常見的角色原型：

=== 文學作品 ===
[[莎士比亞|莎士比亞]]是文學原型大師，但他並非文學原型最初的的原創者，而是由[[寓言|寓言]]和[[神話|神話]]中得到原型，再於其作品中將原型定義而發揚光大。常用作原型，由[[莎士比亞|莎士比亞]]創作的角色有：

* '''[[羅密歐與茱麗葉|羅密歐與茱麗葉]]'''是遭遇家人反對為愛殉情男女的原型，相似於中國故事中的[[梁山伯與祝英台|梁山伯與祝英台]]，都在不少後來的創作中再用。
* '''[[哈姆雷特|哈姆雷特]]'''是為報復殺父之仇經歷痛苦掙扎悲劇英雄的原型，其故事情節則開創分離、轉型、與歸來三部曲的原型。[[趙氏孤兒|趙氏孤兒]]是類似的中國故事。
* '''[[法斯塔夫|法斯塔夫]]'''是體型臃腫的牛皮大王和老饕的原型。

=== 電影 ===
* '''[[英雄|英雄]]'''的原型：戲劇中往往有英雄人物，例如[[星際大戰|星際大戰]]中的[[路克·天行者|路克·天行者]]，[[駭客任務|駭客任務]]中的[[尼歐|尼歐]]等(Neo from the Matrix is actually not an archetype for hero because the matrix is an anti-hero movie. )，西方將最早的英雄原型分為兩種，皆出自荷馬的作品，分別是《奧德賽》中的尤里西斯與《伊里亞德》中的阿基里斯，前者是富有人性、有吸引力、甚至愛惡作劇的，例如印第安那瓊斯、詹姆士龐德等。後者則有《魔戒》中的亞拉岡、美國隊長等等。
* '''無法阻擋的邪惡'''的原型：與英雄相反的角色，如[[柯恩兄弟|柯恩兄弟]]的電影[[險路勿近|險路勿近]]中的「安東·奇哥」。
* '''老智者'''的原型：[[星際大戰|星際大戰]]中的[[歐比王·肯諾比|歐比王·肯諾比]]（Obi-Wan（Ben）Kenobi）及[[尤達|尤達]]、[[魔戒|魔戒]]中的[[甘道夫|甘道夫]]、[[亞瑟王|亞瑟王]]的巫師[[梅林|梅林]]等。
* '''[[彼得潘症候群|長不大的男孩]]'''的原型：與老智者相反的角色，如[[小飛俠|小飛俠]]。
