'''原日軍步兵第二聯隊營舍'''，為'''臺灣步兵第二聯隊'''的營舍，位於[[臺南市|臺南市]][[東區|東區]]（[[台灣日治時期|日治時期]]為[[旭町_(臺南市)|旭町]]）[[國立成功大學|成功大學]]光復校區。臺灣步兵第二聯隊是日治時期[[臺灣軍|台灣軍]]下的一支部隊，其營區內尚存五棟營舍，分別是第二聯隊本部（藝術研究所），第七、八中隊營舍（工業設計系館），第五、六中隊營舍（歷史系館），軍訓室和歷史文物館 ，是極少數留存的日治時期軍事建築<ref name=":0">{{Cite book|title=台南市古蹟與歷史建築總覽|last=傅|first=朝卿|publisher=台灣建築與文化資產出版社|year=2001年11月|isbn=957-30880-4-5|location=台南市|pages=188-191頁}}</ref>。此營區在1927年以前（1927年在屏東新設飛行第八聯隊），日本帝國國土最南方的陸軍基地，所以成為日本皇室成員行啟、御成臺灣經常視察之地<ref name=":1">{{Cite web|url=http://treesinncku.blogspot.com/2015/08/blog-post.html|title=成大榕園的大榕樹|accessdate=|author=蔡侑樺|date=2015-08-31|publisher=}}</ref>。

營區內的三棟建築，第二聯隊本部，第七、八中隊營舍，第五、六中隊營舍，於民國82年（1993年）11月10日公告為國定古蹟<ref>{{Cite web|url=https://nchdb.boch.gov.tw/assets/advanceSearch/monument/20031110000003|title=文化部文化資產局|accessdate=|author=|date=|publisher=原日軍臺灣步兵第二聯隊營舍}}</ref>。

== 沿革 ==
[[File:成功大學大成館-南面.jpg|250px]]
[[File:成功大學禮賢樓.JPG|250px]]

日本自明治28年（[[1895年臺灣|1895年]]）統治臺灣，軍制經數次改變，在明治40年（1907年）後，陸軍步兵第一聯隊進駐臺北，步兵第二聯隊則進駐臺南<ref name=":0" />。步兵第二聯隊營舍建築由日本陸軍經理部營繕組設計，為明治末期大正初期的作品<ref name=":0" />。

二次大戰後，步兵第二聯隊營舍由[[中華民國國軍|中華民國國軍]]接收，改為第二、八、十軍軍部，成為光復營區<ref name=":2">{{Cite book|title=台南歷史深度旅遊|last=遠流台灣館|first=遠流台灣館|publisher=遠流出版社|year=2003|isbn=957-30880-4-5|location=台北市|pages=188-191頁}}</ref>，直到民國55年（1966年）10月5日時才轉給[[國立成功大學|成功大學]]使用，改為成大「光復校區」<ref name=":0" />。其中大成館曾作為行政大樓、造船學系以及工業設計學系與夜間部使用，而禮賢樓在成大接收之前為指揮大樓，納入大學後，改為客座教授或歸國學人的招待所，故名「禮賢樓」，現在是藝術研究所<ref name=":0" />。

== 建築特色 ==
工業設計系館與歷史系館在構造上，為日本陸軍最早實驗採用鋼筋混擬土的建築之一禮賢樓則是磚造為主；營舍的空間平面皆是一字型，在四周有迴廊<ref name=":0" />。兵舍均置於一有通風孔的基座上，主入口位於北面正中央有六柱古典羅馬式門廊，正後方也有入口，但門廊只有四柱，而左右兩側三分之一處亦設有次入口，東西兩端也有出入口，但並沒有門廊<ref name=":0" />。本部外觀與兵舍相似，但在北面有兩個入口門廊，門廊之間有五個連續拱圈，而兩側有兩個連續拱圈；東西兩面各有三個獨立拱圈和裝飾性的小扶壁<ref name=":2" />，而南面則為十四個連續拱圈，此外主入口門廊的式樣也與兵舍大不相同<ref name=":0" />。

工業設計系館的東西側邊入口陽台尚有五芒星之裝飾，其為日軍的象徵之一<ref>{{Cite web|url=http://ir.lib.ncku.edu.tw/bitstream/987654321/155043/2/%E5%8D%9A%E5%BE%8C%E5%B7%A5%E4%BD%9C%E5%A0%B1%E5%91%8A.doc|title=國立成功大學「邁向頂尖大學計畫」延攬優秀人才工作報告表|accessdate=|author=蔡侑樺|date=2015|publisher=}}</ref>。

== 修復 ==
2013年，工業設計系館修復動工，補強結構，修復屋頂與外牆去漆復原，於2015年完工。2016年，歷史系館修復動工，但囿於工程經費，2018完工時只修復了北側外牆<ref>{{Cite web|url=https://udn.com/news/story/11322/3193216|title=成大歷史系館 整修做半套|accessdate=|author=|date=2018-06-11|publisher=聯合報}}</ref>。

== 榕園 ==

面對工業設計系館的榕園是昔日步兵的演練場，其中最大的一棵榕樹是1923年來臺的日本皇太子裕仁（[[昭和天皇|昭和天皇]]）所植，是成功大學的標幟之一<ref name=":0" />。1925年來臺的[[秩父宮雍仁親王|雍仁親王]]、1926年來臺的[[高松宮宣仁親王|宣仁親王]]，以及1935年來臺的[[李王|昌德宮李王]]亦曾在此處種植榕樹，但現已不存。<ref name=":1" />
