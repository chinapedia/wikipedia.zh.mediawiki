'''原田氏'''，為[[日本|日本]]之[[氏族|氏族]]，以大藏氏流原田氏為主要代表。原田氏在日本一般姓氏排名第五十一位。

==起源==
# 源於[[漢獻帝|漢獻帝]]玄孫[[劉阿知|劉阿知]]：[[漢獻帝|漢獻帝]]玄孫[[劉阿知|劉阿知]]（[[阿知使主|阿知使主]]）於289年率子[[劉都賀|劉都賀]]、舅父[[趙輿德|趙輿德]]集族人[[劉國鼎|劉國鼎]]、[[劉濤子|劉濤子]]、[[劉鶴明|劉鶴明]]、[[劉信子|劉信子]]等男女共兩千餘人東渡<ref>[http://books.sina.com/artbook/allembracing/2007-11-16/ba9238.shtml 「唐人」：中國古代歷史上四次移民日本的浪潮] {{webarchive|url=https://web.archive.org/web/20090901061459/http://books.sina.com/artbook/allembracing/2007-11-16/ba9238.shtml |date=2009-09-01 }}</ref>。時值[[應神天皇|應神天皇]]在位，任命他擔任朝臣，賜號為「東漢使主」，定居於[[大和國|大和國]][[高市郡|高市郡]][[檜前村|檜前村]]，即今日[[奈良縣|奈良縣]]檜前村，當地與[[岡山縣|岡山縣]][[倉敷市|倉敷市]]現都建有「阿知宮」，以紀念劉阿知<ref>[http://www.liu-home.com/tour/view.asp?id=717 日本的劉姓] {{webarchive|url=https://web.archive.org/web/20090713213835/http://www.liu-home.com/tour/view.asp?id=717 |date=2009-07-13 }}</ref>。劉阿知與[[劉都賀|劉都賀]]曾代表日本至[[孫吳|孫吳]]找尋擅長養蠶、織布、裁縫的工匠<ref>[http://www001.upp.so-net.ne.jp/Strumgeschutz-3/harada.htm 大藏一門原田氏]</ref>，[[雄略天皇|雄略天皇]]賜劉都賀姓「直」，子孫遂以直為姓。劉都賀生[[山木直|山木直]]、[[志努直|志努直]]、[[爾波伎直|爾波伎直]]三子，分別衍生出數十個日本姓氏<ref>[http://books.sina.com/artbook/allembracing/2007-11-16/ba9238.shtml 「唐人」：中國古代歷史上四次移民日本的浪潮] {{webarchive|url=https://web.archive.org/web/20090901061459/http://books.sina.com/artbook/allembracing/2007-11-16/ba9238.shtml |date=2009-09-01 }}</ref>。其子孫曾以天皇的賜姓東漢氏為姓。
# 源於[[坂上氏|坂上氏]]，出於劉阿知後裔，372年，[[仁德天皇|仁德天皇]]賜姓[[坂上|坂上]]。
# 源於[[大藏氏|大藏氏]]，出於劉阿知後裔，476年，[[雄略天皇|雄略天皇]]又改賜姓[[大藏|大藏]]。
# 源於[[大藏氏|大藏氏]]始祖為[[大藏廣隅|大藏廣隅]](おおくらのひろすみ)。於而大藏廣隅為[[飛鳥時代|飛鳥時代]]人，於[[壬申之亂|壬申之亂]]時，與[[赤染德足|赤染德足]]、[[坂上囯麻呂|坂上囯麻呂]]、[[古市黑麻呂|古市黑麻呂]]、[[竹田大德|竹田大德]]等人，隨[[大海人皇子|大海人皇子]]之子[[高市皇子|高市皇子]]逃離[[大津市|大津京]]，而後於[[鹿深|鹿深]][[積殖山|積殖山]]口一帶，與大海人皇子會合，其後的發展在史料上則缺乏紀錄。
# 源於大藏廣隅的八世孫[[大藏春實|大藏春實]]，於[[天慶之亂|天慶之亂]]討伐[[藤原純友|藤原純友]]而聞名，被任為[[大宰府|大宰府]]府官，其後世子孫便世襲大宰府官職，而成為[[九州|九州]]氏族。
# 源於大藏春實九世孫[[大藏種直|大藏種直]]生有五子，長子[[大藏賴種|大藏賴種]]，居[[高橋城|高橋城]]，後裔為[[高橋氏|高橋氏]]；次子[[大藏幸種|大藏幸種]]，於居[[江上城|江上城]]，後裔即[[江上氏|江上氏]]；三子[[大藏種雄|大藏種雄]]，為[[秋月氏|秋月氏]]之祖；四子大藏種成，居[[原田城|原田城]]，後帶便以原田為氏；五子[[大藏種景|大藏種景]]，為[[波多江氏|波多江氏]]的始祖。
# 大藏氏流原田氏：[[大藏|大藏]]氏流原田氏起源於[[大藏氏|大藏氏]]，大藏氏為日本[[平安朝|平安朝]]時期的[[貴族|貴族]]。
# 源於1131年([[天承|天承]]元年)，居住於原田的大藏氏後裔正式以原田爲姓。原田氏後裔並在九州[[福岡市|福岡市]]立有「漢太公廟」，以表明自己為[[漢高祖|漢高祖]][[劉邦|劉邦]]後裔。<ref>[http://www.liu-home.com/tour/view.asp?id=717 日本的劉姓] {{webarchive|url=https://web.archive.org/web/20090713213835/http://www.liu-home.com/tour/view.asp?id=717 |date=2009-07-13 }}</ref>

==關聯姓氏==
# [[高橋氏|高橋氏]]：源於大藏春實九世孫[[大藏種直|大藏種直]]長子系[[大藏賴種|大藏賴種]]的後裔，因居[[高橋城|高橋城]]，後裔改姓[[高橋氏|高橋氏]]。
# [[江上氏|江上氏]]：源於大藏春實九世孫[[大藏種直|大藏種直]]次子系[[大藏幸種|大藏幸種]]的後裔，因居[[江上城|江上城]]，後裔改姓[[江上氏|江上氏]]。
# [[秋月氏|秋月氏]]：源於大藏春實九世孫[[大藏種直|大藏種直]]三子系[[大藏種雄|大藏種雄]]的後裔，因居[[秋月城|秋月城]]，後裔改姓[[秋月氏|秋月氏]]。
# ''''''原田氏'''：源於大藏春實九世孫[[大藏種直|大藏種直]]四子系[[大藏種成|大藏種成]]的後裔，因居[[原田城|原田城]]，後裔改姓'''原田氏''''''。
# [[波多江氏|波多江氏]]：源於大藏春實九世孫[[大藏種直|大藏種直]]五子系[[大藏種景|大藏種景]]的後裔，後裔改姓[[波多江氏|波多江氏]]。

大藏氏後裔分衍繁多，除前述五姓外，尚有[[三原氏|三原氏]]、[[田尻氏|田尻氏]]、[[原氏|原氏]]、[[小金丸氏|小金丸氏]]、[[鞍手氏|鞍手氏]]、[[岩戶氏|岩戶氏]]、[[美氣氏|美氣氏]]、[[山崎氏|山崎氏]]、[[古賀氏|古賀氏]]、[[枝吉氏|枝吉氏]]、[[米倉氏|米倉氏]]、[[海頭氏|海頭氏]]、[[別府氏|別府氏]]、[[安永氏|安永氏]]、[[天草氏|天草氏]]、[[山鹿氏|山鹿氏]]、[[坂井氏|坂井氏]]、[[粥田氏|粥田氏]]、[[砥上氏|砥上氏]]<ref>[http://www001.upp.so-net.ne.jp/Strumgeschutz-3/harada.htm 大藏一門原田氏]</ref>等幾支。
