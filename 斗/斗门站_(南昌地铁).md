'''斗门站'''位于[[江西省|江西省]][[南昌市|南昌市]][[南昌县|南昌县]]迎宾大道与富山五路交叉口，是[[南昌地铁|南昌地铁]]{{南昌地铁线路名|3}}的一个地下车站；车站预计于2020年和3号线一同启用。

== 车站结构 ==
车站位于迎宾大道与汽车大道交叉口，为地下二层岛式车站<ref name=":0" />{{rp|37}}{{rp|39}}。

== 历史 ==
* 2015年5月19日南昌市对《南昌市轨道交通3号线工程可行性研究报告》进行评估，为22站，全长28.5km，全地下；车站总数相比上一次的多一站<ref>{{Cite web|url=http://jx.ifeng.com/nc/detail_2015_05/19/3912868_0.shtml|title=南昌评估地铁3号线工可报告 计划2020年建成|accessdate=|author=|date=2015年5月19日|publisher=凤凰江西}}</ref>。
* 2015年7月20日南昌市轨道交通3号线工程环境影响评价文件拟受理情况的公示，<ref>{{Cite web |url = http://www.jxepb.gov.cn/ZWGK/HPC/JSXMHP/HPJB/2015/7cae73ca82d348c38549a442bd064c1b.htm |title = 南昌市轨道交通3号线工程环境影响评价文件拟受理情况的公示 |author =  |date = 2015-07-20 |publisher = 江西省环境保护厅 |language =  |accessdate =  |archiveurl = https://web.archive.org/web/20160304234742/http://www.jxepb.gov.cn/ZWGK/HPC/JSXMHP/HPJB/2015/7cae73ca82d348c38549a442bd064c1b.htm |archivedate = 2016-03-04 |dead-url = yes }}</ref>公布了《南昌市轨道交通3号线工程环境影响报告书（公示稿）》中确认莲塘站从城南路（富山五路北侧）南移到银河城附近，并在汽车大道（目前名为富山五路）新增'''汽车大道站'''<ref name=":0">{{cite book|author=中铁上海设计院集团有限公司|authorlink=|coauthors=|title=《南昌市轨道交通3号线工程环境影响报告书（公示稿）》南昌市轨道3号线环评1.pdf|url=http://www.jxepb.gov.cn/resource/uploadfile/file/20150727/20150727110925363.pdf|year=|publisher=2015年7月20日|location=|isbn=|pages=|access-date=2016年8月7日|archive-url=https://web.archive.org/web/20151118050639/http://www.jxepb.gov.cn/resource/uploadfile/file/20150727/20150727110925363.pdf|archive-date=2015年11月18日|dead-url=yes}}</ref>{{rp|7}}。
* 2015年8月4日《关于南昌市轨道交通3号线工程规划选址的批前公示》，“汽车大道站”位于迎宾大道与汽车大道交叉口<ref>{{Cite web|url=http://www.jxjst.gov.cn/chengxiangguihua/gongshigonggao/2015-08-04/45805.html|title=关于南昌市轨道交通3号线工程规划选址的批前公示|accessdate=|author=|date=2015-08-04|publisher=江西省住房和城乡建设厅|deadurl=yes|archiveurl=https://web.archive.org/web/20170219025640/http://www.jxjst.gov.cn/chengxiangguihua/gongshigonggao/2015-08-04/45805.html|archivedate=2017-02-19}}</ref><ref>{{Cite web|url=http://jiangxi.jxnews.com.cn/system/2015/08/05/014117273.shtml|title=南昌地铁3号线规划选址批前公示 全线设车站22座（图）|accessdate=|author=|date=2015-08-05|publisher=中国江西网}}</ref>。
* 2016年7月26日报道位于南昌县迎宾中大道与富山五路交叉口处的斗门站完成了交通导改<ref>{{Cite web|url=http://www.ncmtr.com/index.php?s=/Home/News/detail/id/1080.html|title=地铁3号线斗门站完成交通导改|accessdate=2016-08-10|author=|date=2016-07-25|work=南昌轨道交通集团有限公司|publisher=}}</ref><ref>{{Cite web|url=http://www.ncnews.com.cn/xwzx/ncxw/bwyc/2016-07/26/content_1622414.htm|title=南昌地铁3号线斗门站完成交通导改|accessdate=2016-08-10|author=|date=2016-07-26|work=南昌新闻网|publisher=}}</ref>。
* 2016年8月6日南昌晚报报道了地铁2、3号线沿线车站改名情况；本站改名为'''斗门站'''，位于位于迎宾中大道与富山五路路口<ref>{{Cite web|url=http://www.ncwbw.cn/html/2016-08/06/content_314602.htm|title=地铁2、3号线30个站点有了新名字|accessdate=2016-08-10|author=|date=2016年8月6日|work=南昌晚报|publisher=}}</ref>。
