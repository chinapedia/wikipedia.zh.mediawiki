'''楚国'''，又称'''荆'''、'''荆楚'''、'''楚荆'''，[[中国历史|中国历史]][[商朝|商朝]]后期至[[春秋战国时期|春秋战国时期]]的一个[[诸侯国|诸侯国]]。楚國是強大的封國，[[楚莊王|楚莊王]]是[[春秋五霸|春秋五霸]]之一，楚國後來亦是[[戰國七雄|戰國七雄]]之一。楚国國君[[芈姓|芈]]<ref>“芈”，[[现代标准汉语|普通话/国语]]中与“米”字同音，[[汉语拼音|汉语拼音]]为mǐ。</ref>姓[[熊姓|熊]]氏，最早兴起于[[丹江|丹江流域]]的[[丹水|丹水]]和[[淅水|淅水]]交汇的[[老城镇_(淅川县)|淅川]]一带<ref>{{cite web |url  = http://epaper.hf365.com/hfwb/html/2011-01/25/content_376376.htm |title=河南库区发掘工作圆满结束，出土文物已通过验收 |publisher=合肥晚报|date=2011年1月25日}}</ref><ref>{{cite web |url = http://news.ifeng.com/gundong/detail_2011_01/25/4440732_0.shtml |title = 科大考古队觅宝千余件 |publisher=凤凰网 |date=2011年1月25日}}</ref>。其全盛时的最大辖地大致为现在的[[湖北|湖北]]、[[安徽|安徽]]、[[河南|河南]]、[[湖南|湖南]]、[[浙江|浙江]]、[[上海|上海]]、[[江苏|江苏]]、[[江西|江西]]、[[重庆|重庆]]、[[貴州|貴州]]、[[山东|山东]]部分地方。公元前223年，[[秦滅楚之戰|秦滅楚之戰]]，爲秦所滅。公元前209年，楚地起義軍將領[[項梁|項梁]]起兵反秦，在[[薛城|薛城]]立[[楚懷王|楚懷王]]的孫子熊心為[[楚後懷王|楚後懷王]]，重新建立楚國，後稱[[西楚|西楚]]，不久[[項羽|項羽]]滅秦朝並派人殺死熊心，自立為西楚霸王。後來項羽在[[楚漢戰爭|楚漢戰爭]]中兵敗自殺，楚國亦再度滅亡。

== 祖源 ==
大致可分为以下几说<ref>[http://www.hb.xinhuanet.com/cwh/2005-03/07/content_3829991.htm 《楚国的族源有谁知？》] {{webarchive|url=https://web.archive.org/web/20051128214429/http://www.hb.xinhuanet.com/cwh/2005-03/07/content_3829991.htm |date=2005-11-28 }}，楚文化，[[新华网|新华网]]湖北频道。</ref><ref>{{cite web |url = http://www.zh5000.com/ZHJD/gxjd/2006/qitabu/ls/1/gxjd-0280.htm |title = 楚族的渊源}}</ref>：
* [[颛顼|颛顼]]（高阳）說<ref>《史记·楚世家》：“楚之先祖出自帝颛顼高阳。高阳者，黄帝之孙，昌意之子也。”</ref>：楚国诗人[[屈原|屈原]]在其《[[离骚|离骚]]》中亦称：“帝[[高阳|高阳]]之苗裔兮，朕皇考曰伯庸。”<ref>李零《楚国族源、世系的文字学证明》</ref>
* [[祝融|祝融]]說：[[胡厚宣|胡厚宣]]在《楚民族源于东方考》（[[北京大学|北京大学]]《史学论丛》第一册，1934年）认为楚之始祖为[[祝融|祝融]]。
* [[华夏|华夏]]说：王族出自华夏，夏商周三代分封诸侯国一直奉行“夏君夷民”的做法。
* [[东夷|东夷]]说：[[郭沫若|郭沫若]]《中国古代社会研究》（[[中国华侨出版社|中国华侨出版社]]，2008年，ISBN 9787802224735）和胡厚宣《楚族源于东方考》认为楚为东夷部落。
* [[土著|土著]]说：楚人来自[[中原|中原]]，但臣民大多为當地土著[[部落|部落]]{{noteTag|1=即屈家嶺文化，一般稱為「三苗」或「楚蠻」，《早期中国文明:江汉文化与荆楚文明》認為「三苗」是虛無縹緲的存在，應改稱為「炎帝祝融文化」。}}。  
* 西來说：认为楚之先祖高阳氏颛顼来自在[[中国西北|中国西北]]境，即[[昆仑山|昆仑山]]一带，还有学者认为楚人是来自[[西亚|西亚]][[拜火教|拜火教]]的[[米地亚人|米地亚人]]。
* 部落融合说：楚先民是由[[华夏|华夏]]高阳氏的一支，南下与当地[[土著|土著]]融合的结果。

== 历史 ==
=== 起源 ===
楚之先祖出自[[黄帝|黄帝]]之孙，[[昌意|昌意]]之子，帝[[颛顼|颛顼]]高阳氏。颛顼后第五代曾孫[[吳回_(神話)|吴回]]，是帝[[高辛氏|高辛氏]]的火正（火官），主管天火与地火，能光融天下，[[帝喾|帝喾]]命之曰[[祝融|祝融]]<ref>《国语·郑语》：夫黎为高辛氏火正，以淳耀敦大，天明地德，光照四海，故命之曰祝融。</ref>。其部落分布在商都的陪都[[朝歌|朝歌]]的南方（今[[河南|河南]][[新郑|新郑]]一带）。吴回之子[[陆终|陆终]]，生有六子，幼子曰[[季连|季连]]，芈姓，是楚之先祖。[[周文王|周文王]]曾拜季连后裔[[鬻熊|鬻熊]]为师，由于这层师徒关系，[[鬻熊|鬻熊]]曾孙[[楚熊绎|熊绎]]被周朝二代君主[[周成王|周成王]]封为子楚（楚国国君，爵位为子爵）<ref>《史记·卷四十·楚世家第十》：熊绎当周成王之时，举文、武勤劳之後嗣，而封熊绎於楚蛮，封以子男之田，姓琇氏，居丹阳。</ref>。

[[西汉|西汉]]史家[[司马迁|司马迁]]在《[[史记|史记]]·楚世家》中记载[[西周|西周]]初期楚人领袖熊绎：“当成王之时，举文、武勤劳之后嗣，而封熊绎于楚蛮，封以子男之田，姓芈氏，居丹阳。”此一论述认为商末周初，楚先人祝融的后裔从北方迁徙至“[[丹阳_(楚国)|丹阳]]”。“楚鬻熊居[[丹阳_(楚国)|丹阳]]，武王徙郢”，楚国国都在[[楚武王|楚武王]]时期从丹阳迁至郢，据考古学家[[刘玉堂|刘玉堂]]、[[王红星|王红星]]、[[高崇文|高崇文]]研究，古“丹阳”位于[[丹水|丹水]]和[[淅水|淅水]]交汇之处（今[[河南省|河南省]][[淅川县|淅川县]][[丹江口水库|丹江口水库]]淹没区），因处丹水之北，故称为[[丹阳_(楚国)|丹阳]]。国学大师[[钱穆|钱穆]]在《[[国史大纲|国史大纲]]》中说：“楚之先亦颛顼后，始起在汉水流域丹、淅水入汉水处，曰丹阳”,而楚国[[令尹|令尹]]（相当于宰相）之墓[[子庚|子庚墓]]<ref>[http://www.hnta.cn/Info/lyzx/gb/106578.shtml 楚国令尹“王子午”墓中 王子午鼎]</ref>和大量[[淅川楚墓|楚国贵族墓]]<ref>[http://lw.xinhuanet.com/htm/content_2108.htm 淅川楚墓“意外”发掘]</ref>在淅川县[[仓房镇|仓房镇]]的[[香严寺_(淅川县)|香严寺]]下寺附近的发现，印证了古丹阳位于河南淅川境内。楚国历史有1000多年，其中有300多年在[[丹江|丹江]]流域建都。

据《[[清华简|清华简]]·楚居》记载，商王[[盘庚|盘庚]]的女儿妣隹征婚，楚部族的首领[[季连|季连]]趁机追上了她，此后便居住在盘地，与妣隹生了[[𦀚伯|𦀚伯]]、远仲两个儿子<ref>《清华简·楚居》</ref>，他被尊为楚国的始祖。可见楚人与[[殷人|殷人]]存在亲缘关系，这也为后来楚国人“不服周”埋下了伏笔。湖北省武汉市有商代的[[盘龙城|盘龙城]]遗址，这里出土有青铜器和玉器，说明商代时荆楚地区已有初步发达的文明。

=== 建国及南征 ===
[[File:春秋诸侯大国简图.png|thumb]]

[[殷商|殷商]]后期时，楚国人逐渐从[[河南|河南]][[新郑|新郑]]一带南迁到[[河南|河南]][[淅川|淅川]]一带（古丹阳）。[[武王克殷|武王克殷]]时，楚国并未参与殷商或者周朝任何一方，[[周武王|周武王]][[即位|即位]]，楚國也没有马上归顺周天子。

[[周康王|周康王]]时，[[楚熊绎|熊绎]]去朝见周天子，接受了周天子的封号。此后，楚国经常向周王朝进献桃木弓、枣木弓。

[[周康王|周康王]]时的一次诸侯大会，由于西周王朝视楚国为“南蛮”，故[[楚熊绎|熊绎]]没有受到一个诸侯该有的礼遇，反而与当时的狄夷[[鲜卑|鲜卑]]一起守燎，并负责茅蕝缩酒，根本是僕役的工作。在这之后，[[熊绎|熊绎]]知耻近勇带领国人开始了“筚路蓝缕”的艰苦创业。

[[周昭王|周昭王]]时，由于害怕楚国会崛起，周王南征荆楚。[[周昭王|周昭王]]在回师的途中渡过[[汉水|汉水]]时被伏击身亡，结果西周“丧六师于汉”。这是楚国与西周的第一次大规模冲突。

[[周夷王|周夷王]]时，诸侯开始轻慢周天子，中央政府的威权开始稍弱，楚国开始了分庭抗礼的初步尝试，[[熊渠|熊渠]]在[[江汉|江汉]]一带的民心和威望很高，遂兴兵伐庸，杨粤，至于鄂，还说出了“我蛮夷也，不与中国之号谥”的话，并立长子康为句亶王，中子红为鄂王，少子执疵为越章王。

[[周厉王|周厉王]]时，由于楚国未真正崛起，加上周天子為人兇暴，是以[[熊渠|熊渠]]畏周伐楚，亦去其王号。

[[周宣王|周宣王]]时，由于耿耿楚国于怀，[[周宣王|周宣王]]在江汉间封了许多周室成员或者依附于周朝的势力，史称“江汉诸姬”，西周的战略就是试图以这些亲周势力封锁江汉通往中原的门户用以遏制楚国的北下对西周造成威胁。但最終未能達到目的，江汉诸姬尽被吞并、臣服融合于楚。

不满自己国大爵小的楚君[[楚熊通|熊通]]從傳說中找到例証，他說其先祖[[鬻熊|鬻熊]]是[[周文王|周文王]]的[[帝師|老师]]，有辅佐推翻[[殷商|殷商]]，建立周朝的功劳，只因早死而仅得到[[子爵|子爵]]，遂攻打姬姓诸侯随国，目的是要周天子晋升其[[爵位|爵位]]，遭拒，熊通率兵再次征讨[[随国|随国]]（今湖北[[随州|随州]]），迫使随侯投降。熊通因此在前704年自称为王（后諡武）<ref>《史记·卷四十·楚世家第十》：三十五年，楚伐随。随曰：“我无罪。”楚曰：“我蛮夷也。今诸侯皆为叛相侵，或相杀。我有敝甲，欲以观中国之政，请王室尊吾号。”随人为之周，请尊楚，王室不听，还报楚。三十七年，楚熊通怒曰：“吾先鬻熊，文王之师也，蚤终。成王举我先公，乃以子男田令居楚，蛮夷皆率服，而王不加位，我自尊耳。”乃自立为武王，与随人盟而去。</ref>。被楚国消灭的小国还有[[古麋国|古麋国]]（湖北[[郧阳|郧阳]]<ref>{{cite web |url = http://www.my285.com/shishu/msjs/38.htm |title = 明史纪事本末·第三十八卷·平郧阳盗 |access-date = 2011-02-08 |archive-url = https://web.archive.org/web/20110216035248/http://www.my285.com/shishu/msjs/38.htm |archive-date = 2011-02-16 |dead-url = yes }}</ref>、[[房州|房州]]<ref>{{cite web |url = http://www.uus8.com/book/html/old/%E4%B8%AD%E5%9B%BD%E5%8F%A4%E5%85%B8%E6%96%87%E5%AD%A6%E5%A4%A7%E5%90%88%E9%9B%86/zw8-4/%E4%B8%AD%E5%9B%BD%E5%8F%A4%E5%85%B8%E5%90%8D%E5%AE%B6%E6%96%87%E9%9B%86%E5%90%88%E9%9B%86/%E3%80%8A%E5%A4%AA%E5%B9%B3%E5%BE%A1%E8%A7%88%E3%80%8B%E3%80%90%E5%88%9D%E6%A0%A1%E7%89%88%E3%80%91/167.htm |title = 《十道志》曰：房州，房陵郡。土地所属与金州同。古麋国也。 }}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>一带）、卢国、罗国（湖北[[襄阳|襄阳]]、[[荆门|荆门]]一带），一路向南征伐百苗等大小诸侯国，成为江漢流域的霸主。

=== 争霸中原 ===
熊通子[[楚文王|楚文王]]定都于[[郢|郢]]<ref>《史记·卷四十·楚世家第十》：子文王熊赀立，始都郢。</ref>（今湖北省[[宜城市|宜城市]]西南）<ref>《楚文化史》,张正明</ref>，因取名于该地已被填埋的湖泊“疆浧”而得名<ref>《楚居》：众不容於免，乃溃疆浧之波而宇人焉，抵今曰郢。</ref>，後人通稱為“鄢郢”，以区别于日后为防备吴国的再次入侵而迁徙的新国都[[纪南城|纪郢]]及一系列皆称之为“郢”的同名陪都。此后为扩充领域，楚国与[[中原|中原]]的中原诸侯国经常发生战事。

[[楚成王|楚成王]]时，楚在令尹[[子文|子文]]的治理下更显强盛。前632年，晋楚[[城濮之战|城濮之战]]，楚人大败。晋师“三日馆谷”，向周天子献楚捷“驷介百乘、徒兵千，”，楚国几十年不敢北向。《子犯和钟》即记载此事，中有“诸楚荆不听命于王所，子犯及晋公率西之六师博伐楚荆，孔休大功，楚荆丧厥师，灭厥渠”云。后来的[[楚莊王|楚莊王]]武功彪炳，又励精图治，选拔[[孙叔敖|孙叔敖]]施行文冶，使楚国经济繁荣、文化鼎盛。前606年莊王征伐[[陆浑戎|陆浑之戎]]（在雒邑西南），派人向[[周定王|周天子]]问九鼎之轻重<ref>《史记·卷四十·楚世家第十》：八年，伐陆浑戎，遂至洛，观兵於周郊。周定王使王孙满劳楚王。楚王问鼎小大轻重，对曰：“在德不在鼎。”庄王曰：“子无阻九鼎！楚国折钩之喙，足以为九鼎。”王孙满曰：“呜呼！君王其忘之乎？昔虞夏之盛，远方皆至，贡金九牧，铸鼎象物，百物而为之备，使民知神奸。桀有乱德，鼎迁於殷，载祀六百。殷纣暴虐，鼎迁於周。德之休明，虽小必重；其奸回昏乱，虽大必轻。昔成王定鼎于郏鄏，卜世三十，卜年七百，天所命也。周德虽衰，天命未改。鼎之轻重，未可问也。”楚王乃归。</ref>。

前597年，楚围[[郑国|郑]]，迫郑降<ref>《史记·卷四十·楚世家第十》：十七年春，楚庄王围郑，三月克之。</ref>，[[晋国|晋]]派兵救郑，两军于[[邲之战|邲]]会战，晋大败<ref>《史记·卷四十·楚世家第十》：夏六月，晋救郑，与楚战，大败晋师河上，遂至衡雍而归。</ref>。此后，[[陈国|陈]]、[[蔡国|蔡]]、[[许国|许]]、郑俱从楚。公元前594年，楚围[[宋国|宋]]，宋告急于晋，晋不能救，宋遂与楚言和，尊楚<ref>《史记·卷四十·楚世家第十》：二十年，围宋，以杀楚使也。宋五月，城中食尽，易子而食，析骨而炊。宋华元出告以情。庄王曰：“君子哉！”遂罢兵去。</ref>。这时中原各国除晋、[[齐国|齐]]、[[鲁国|鲁]]之外，尽尊[[楚庄王|楚庄王]]为霸主，确立了楚庄王在[[春秋五霸|春秋五霸]]的历史地位。史载，透过战事，楚国先后吞并了四十五个较弱小的诸侯方国。

[[邲之战|邲之战]]后，楚庄王称霸一时，但是晋国国势犹不可忽视，晋、楚双方形成南北对峙、各霸一方。前546年，宋国执政[[向戌|向戌]]倡议弭兵，奔走于晋国执政[[赵武|赵武]]和楚国[[令尹|令尹]][[子木|子木]]之间，使晋、楚、齐、[[秦国|秦]]、宋、鲁、郑、[[卫国|卫]]、陈、蔡、[[曹国|曹]]、许、[[邾国|邾]]、[[滕国|滕]]等十四国于宋召开弭兵大会<ref>《左传·襄公二十七年》：宋向戌善于赵文子，又善于令尹子木，欲弭诸侯之兵以为名。如晋，告赵孟……晋人许之。如楚，楚亦许之……齐人许之。告于秦，秦亦许之。皆告于小国，为会于宋。</ref>。结果由前述十国签订盟约，规定除齐、秦等大国外，小国须既朝晋又朝楚，霸权乃由晋、楚二强平分，尊两国为共同霸主。史称‘[[春秋弭兵会盟|向戌弭兵]]’。

=== 衰而复兴 ===
[[File:战国形势图（前350年）（简）.png|thumb]]

前506年，[[楚昭王|楚昭王]]时，[[吴国|吴国]]君主[[阖闾|阖闾]]派[[孙武|孙武]]、[[伍子胥|伍子胥]]率军攻打楚国，于[[柏举之战|柏举之战]]中击败楚国，攻占其首都郢，楚国几乎覆亡<ref>《史记·卷四十·楚世家第十》：十年冬，吴王阖闾、伍子胥、伯嚭与唐、蔡俱伐楚，楚大败，吴兵遂入郢，辱平王之墓，以伍子胥故也。</ref>。这时[[越国|越国]][[允常|允常]]趁机攻打吴国，[[秦国|秦国]]也出兵帮助楚国，楚国得以保全。

及后[[楚惠王|楚惠王]]即位，继续执行安邦定国、伺机发展的方针。对外，楚惠王四十二年（前447年），楚兼并蔡<ref>《史记·卷四十·楚世家第十》：四十二年，楚灭蔡。</ref>。两年后，兼并[[杞国|杞]]，楚东拓疆土至泗水之上，尽有江淮以北之地<ref>《史记·卷四十·楚世家第十》：四十四年，楚灭杞。与秦平。是时越已灭吴而不能正江、淮北；楚东侵，广地至泗上。</ref>。楚简王元年（前431年），楚又兼并掉[[莒国|莒]]<ref>《史记·卷四十·楚世家第十》：简王元年，北伐灭莒。</ref>。

=== 吴起变法 ===

进入[[战国_(中国)|战国]]时代，主要的六大强国皆曾进行片面或全面性的政经改革，唯有楚国固步自封未曾稍加变法。前400年，自晋分家的[[韩国_(战国)|韩]]、[[赵国|赵]]、[[魏国_(战国)|魏]]三国屡次联军攻楚<ref>《史记·卷四十·楚世家第十》：悼王二年，三晋来伐楚，至乘丘而还。</ref>。当时在位的[[楚悼王|楚悼王]]为求富国强兵，任用[[吴起|吴起]]开始变法兴革，使楚国国力迅速强大起来<ref>《史记·卷六十五·孙子吴起列传第五》：楚悼王素闻起贤，至则相楚。明法审令，捐不急之官，废公族疏远者，以抚养战斗之士。要在彊兵，破驰说之言从横者。於是南平百越；北并陈蔡，卻三晋；西伐秦。诸侯患楚之彊。</ref>。由于楚悼王不幸早逝，吴起失去了坚强的靠山，被旧贵族残酷杀害<ref>《史记·卷六十五·孙子吴起列传第五》：及悼王死，宗室大臣作乱而攻吴起，吴起走之王尸而伏之。击起之徒因射刺吴起，并中悼王。</ref>。虽然变法时间不长，但变法的成果还是有所保留，比如之后的楚国封君的继承不超过三代没有自己独立的武装，一切财政大权听命于中央等等。

至[[楚威王|楚威王]]时代，楚国的西部领土已扩张至[[巴国|巴]][[古蜀|蜀]]之间，东灭[[越国|越国]]，兴盛一时。前306年，在经过5年的[[昭滑|昭滑]]谍报与卧底工作后，[[楚怀王|楚怀王]]派后者率军消灭越国残余势力，占领吴、越故地<ref>《史记·卷四十一·越王句践世家第十一》：楚威王兴兵而伐之，大败越，杀王无彊，尽取故吴地至浙江，北破齐於徐州。</ref>，并设立江东郡。 《史记。甘茂传》“王前尝用台滑于越，而内行章义之难，越国乱，故楚南塞厉门而郡江东。计王之功，所以能如此者，越国乱而楚治也。”《楚策一．楚王问于范环》亦有记载，范环曰：“且王尝用滑于越而纳句章，昧之难，越乱，故楚南察濑胡而野江东，计王之功所以能如此者，越乱而楚治也。”

=== 衰落 ===
[[File:ZH-楚国地图260BCE.jpg|thumb]]
[[楚怀王|楚怀王]]即位后，开始重用[[屈原|屈原]]等大臣，针对主要弊政进行一系列的革新，却引来楚国[[贵族|贵族]]一致反对。日久，难以抵挡反对声浪的楚怀王遂意兴阑珊，变法终告失败。前299年，楚怀王晚年被[[秦昭襄王|秦昭襄王]]骗去秦国<ref>《史记·卷四十·楚世家第十》：三十年，秦复伐楚，取八城。秦昭王遗楚王书曰：“……寡人原与君王会武关，面相约，结盟而去，寡人之原也。敢以闻下执事。”楚怀王见秦王书……於是往会秦昭王。昭王诈令一将军伏兵武关，号为秦王。楚王至，则闭武关，遂与西至咸阳，朝章台，如蕃臣，不与亢礼。楚怀王大怒，悔不用昭子言。秦因留楚王，要以割巫、黔中之郡。楚王欲盟，秦欲先得地。楚王怒曰：“秦诈我而又彊要我以地！”不复许秦。秦因留之。</ref>，客死咸阳，欲兴兵夺回楚王的楚国军队又为秦将[[白起|白起]]击败<ref>《史记·卷四十·楚世家第十》：顷襄王横元年，秦要怀王不可得地，楚立王以应秦，秦昭王怒，发兵出武关攻楚，大败楚军，斩首五万，取析十五城而去。</ref>。楚国国势迅速衰弱。

[[楚顷襄王|楚顷襄王]]即位后，周赧王二十二年（前293年），秦国派大将白起攻打韩国，在伊阙获大胜。伊阙之战后，秦昭王写给楚顷襄王一封国书说：“楚国背叛了秦国，秦国将率领诸侯军攻打楚国，决一雌雄。希望您重整军队，以便痛快地打一场。”楚顷襄王很担心，便打算跟秦国讲和。周赧王二十三年（前292年），楚国到秦国迎接新妇，秦楚讲和。

周赧王三十年（前285年），楚顷襄王与秦昭王在宛（今河南省南阳市宛城区一带）友好相会，议和结亲。

周赧王三十二年（前283年），楚顷襄王与秦昭王在楚国的别都鄢（今湖北省宜城市东南）友好相会，同年秋季，又和秦昭王在秦国的穰（今河南省邓州市）相会。<ref>《[[史记|史记]]卷四十·楚世家》</ref> 

前281年[[楚顷襄王|楚顷襄王]]意图联合齐国和韩国联合讨伐秦国为楚怀王复仇，前280年，秦先发制人，派大将司马错率领军队从陇西郡出兵，攻取了楚国的黔中郡，楚顷襄王被迫割让上庸（今湖北省竹山县东南）和汉江以北的土地给秦国。次年（前279年）秦白起率军大举攻楚，占领邓，并水淹鄢城，城中百姓死亡数十万，之后势如破竹，一举攻陷楚国国都郢，烧毁其先王陵墓夷陵（今湖北省宜昌市夷陵区），向东进兵至竟陵（今湖北省潜江市东北）楚军败，楚顷襄王在兵败后，向东北方溃逃至陈（今河南省淮阳县），被迫迁都于此以自保。<ref>《[[史记|史记]]卷七十六·平原君虞卿列传》</ref>

=== 危局 ===
前263年，[[楚考烈王|楚考烈王]]繼位，任用[[春申君|春申君]]為[[令尹|令尹]]，春申君派兵助[[趙國|趙國]]解[[邯鄲郡|邯鄲]]之圍，又領兵滅[[魯國|魯國]]，楚國一度復興。但前241年，春申君組織東方國家最後一次[[合縱|合縱]]，但被秦軍所敗，[[楚考烈王|楚考烈王]]怕秦國報復，再次迁都至更東面的[[寿春城遗址|寿春]]<ref>《史记·卷四十·楚世家第十》：二十二年，与诸侯共伐秦，不利而去。楚东徙都寿春，命曰郢。</ref>。

前238年楚考烈王死后<ref>《史记·卷四十·楚世家第十》：二十五年，考烈王卒，子幽王悍立。</ref>，春申君門客[[李園|李園]]發動政變殺害春申君，楚國国力更加一蹶不振。[[秦始皇|秦王政]]亲政后，知楚将[[项燕|项燕]]擅战，先遣[[李信|李信]]为将，领二十万兵马，欲灭楚，惨败<ref>《史记·卷七十三·白起王翦列传第十三》：遂使李信及蒙恬将二十万南伐荆……荆人因随之，三日三夜不顿舍，大破李信军，入两壁，杀七都尉，秦军走。</ref>。

===郢都淪陷===
前229年（秦王政十八年），楚軍敗走，秦軍把已經完全控制郢都的疆域，並改郢為[[南郡|南郡]]，並委任來自韓國的降將[[內史騰|騰]]駐守於此。

===覆亡===
前223年，由秦國老将[[王翦|王翦]]所統率的秦师六十万大败楚军，俘虏了楚国国君<ref>《史记·卷七十三·白起王翦列传第十三》：王翦果代李信击荆……翦因举兵追之，令壮士击，大破荆军。至蕲南，杀其将军项燕，荆兵遂败走。秦因乘胜略定荆地城邑。岁馀，虏荆王负刍，竟平荆地为郡县。</ref>。到了公元前222年，王翦平定了楚国长江以南的广大土地，楚国彻底灭亡<ref>《史记·秦始皇本纪》</ref>。

楚人作为六国最强的国家，對於秦國的亡國之辱極為痛恨，在秦末天下大亂時楚地開始興起「楚虽三户，亡秦必楚」的順口溜以彰興兵雪恨之志。后来项燕之孙[[项羽|项羽]]果然灭秦。

=== 复国余绪 ===
{{main|西楚}}

[[秦二世|秦二世]]元年（前209年）七月，赴[[燕京|燕京]]的戍卒領袖[[陈胜|陈胜]]、[[吴广|吴广]]，自稱[[秦始皇|秦始皇]]長子[[扶蘇|扶蘇]]與[[楚國|楚國]][[大將軍|大將軍]][[項燕|項燕]]，發動[[大澤起義|大澤起義]]，建立[[张楚|张楚]]，恢复楚国。旋即败亡。陈胜败亡后，[[陳勝|陳勝]]的[[大司馬|大司馬]][[秦嘉|秦嘉]]，立楚宗室[[景驹|景驹]]为[[楚假王|楚假王]]（即代理楚王），後又稱楚王。[[项燕|项燕]]之子[[项梁|项梁]]起兵，將[[景驹|景驹]][[斩杀|斩杀]]<ref>《史记·卷七·项羽本纪第七》：当是时，秦嘉已立景驹为楚王……项梁……乃进兵击秦嘉。秦嘉军败走，追之至胡陵。嘉还战一日，嘉死，军降。景驹走死梁地。</ref>。

前208年六月，项梁立[[楚怀王|楚怀王]]孙[[熊心|熊心]]，仍为[[楚义帝|楚怀王]]。[[陈婴|陈婴]]为上柱国，项梁自号为[[武信君|武信君]]，楚国复国<ref>《史记·卷七·项羽本纪第七》：於是项梁然其言，乃求楚怀王孙心民间，为人牧羊，立以为楚怀王，从民所望也。陈婴为楚上柱国，封五县，与怀王都盱台。项梁自号为武信君。</ref>。

楚国复国后，[[齐国|齐国]]受到秦将[[章邯|章邯]]的进攻，项梁引兵北攻[[亢父|亢父]]、救[[东阿|东阿]]，破秦军。后不久项梁战死。而楚将[[项羽|项羽]]、[[刘邦|刘邦]]率楚军西略地至雍丘，与秦军战，大破之，斩其将[[李由|李由]]（丞相[[李斯|李斯]]之子），还攻外黄，未下。

秦二世三年（前207年），楚怀王徙都[[盱眙|盱眙]]，并[[吕臣|吕臣]]、[[项羽|项羽]]军，自将之。以[[沛公|沛公]][[刘邦|刘邦]]为砀郡长，封武安侯，将砀郡兵；项羽为[[长安侯|长安侯]]，号[[鲁公|鲁公]]；[[吕臣|吕臣]]为[[司徒|司徒]]，其父[[吕青|吕青]]为[[令尹|令尹]]<ref>《史记·卷七·项羽本纪第七》：楚兵已破於定陶，怀王恐，从盱台之彭城，并项羽、吕臣军自将之。以吕臣为司徒，以其父吕青为令尹。以沛公为砀郡长，封为武安侯，将砀郡兵。</ref>。

秦将章邯攻赵，赵向楚求救兵。楚怀王分兵，令[[宋义|宋义]]为上将军，[[项羽|项羽]]为次将、[[范增|范增]]为末将，北上救赵<ref>《史记·卷七·项羽本纪第七》：王召宋义与计事而大说之，因置以为上将军，项羽为鲁公，为次将，范增为末将，救赵。诸别将皆属宋义，号为卿子冠军。</ref>。同时令[[沛公|沛公]][[刘邦|刘邦]]为将，率楚军西略咸阳。项羽斩杀宋义，夺其军权，北上于[[巨鹿之战|巨鹿之战]]大破秦军主力，虏秦将[[王离|王离]]，秦将[[涉间|涉间]]不降楚，自烧杀。与此同时，楚将刘邦经过一系列战斗，成功攻入关中。（前207年）十月，沛公刘邦军驻灞上，秦王[[子婴|子婴]]素车白，系颈以组，封皇帝印玺，向刘邦投降。[[秦朝|秦朝]]灭亡。

前207年十一月，项羽引兵西入关。十二月中，至[[戏|戏]]（今陕西[[西安|西安]]），与劉邦会霸上[[鸿门宴|鸿门宴]]。汉王元年（前206年）正月，项羽自封为[[西楚|西楚]][[霸王|霸王]]，分封诸侯，比如刘邦被分封为[[汉王|汉王]]。尊楚怀王为[[义帝|义帝]]<ref>《史记·卷七·项羽本纪第七》：项王使人致命怀王。怀王曰：“如约。”乃尊怀王为义帝。</ref>。四月，项羽徙义帝都[[郴州|郴州]]，途中，使六和王[[省布|黜布]]、九江王[[英布|黥布]]与衡山王[[吴芮|吴芮]]弑义帝于江南。

西元前202年9月，項羽所率楚軍兵敗[[垓下|垓下]]，楚霸王在長江北岸烏江渡自刎，[[楚漢戰爭|楚漢戰爭]]結束，西楚國滅亡。

=== 语言 ===
{{main|楚語}}
在春秋时代，[[楚语|楚语]]和中原语言之间有较明显的区别。楚国的官名和其他诸侯国不同：楚国[[宰相|宰相]]叫[[令尹|令尹]]，[[司马|司马]]叫莫敖。除此之外，《[[左传|左传]]·宣公五年》还记载春秋时代楚语的两个特殊词语：
* 鬬穀为“哺乳”
* 於菟（wūtú，[[上古汉语|上古汉语]]发音为*qa-laa）为「虎」（hǔ，*qhlaaʔ）（有人怀疑可能與[[缅甸语|缅甸语]] ကျား ''kla'' 同源）

我们之所以知道这两个词的意思，是因为令尹[[子文|子文]]名字叫做鬬穀於菟，《左传》对他名字的原意做了解释。

《左传·庄公二十八年》有一句话：“楚言而出”（今译：“他们讲楚语就撤退了”），证明当时的楚语和中原的语言有差異。

在[[战国_(中国)|战国时代]]，[[屈原|屈原]]《[[楚辞|楚辞]]》所用词汇也显示出楚国语言的特点。

=== 音乐 ===

[[File:Bianzhong.jpg|right]]出土的[[曾侯乙編鐘|曾侯乙編鐘]]]]

据《左传 襄公十八年》载“南风不竞多死声”，楚音绵软无力。楚国巫风盛行<ref>[[王逸|王逸]]在《楚辞章句》记“昔楚国南郢之邑，沅湘之间，其俗信鬼而好祠”。</ref>，喜欢举行祭祀活动，由巫女主持祭祀。祭祀时要奏乐、歌唱、跳舞以娱神。[[王国维|王国维]]在《宋元戏曲史》中曾谈到楚国：“至于浴兰沐芳，华衣若芙，缓节安歌，歌舞之盛也。乘风载云之词，生别新知之语，荒淫之意也。是则灵之为职，或偃蹇以像神，或婆娑以乐神……”

=== 風俗 ===
楚俗尊[[鳳|鳳]]尚赤、崇火拜日、喜[[巫|巫]]近[[鬼|鬼]]。

楚人認為自己是[[日神|日神]]遠裔，[[火神|火神]]嫡嗣，日火皆為赤色，故而尚赤，楚地廣闊。楚王族是祝融（中國神話中的火神）的後裔于史有载，尚赤之俗源遠流長。《墨子》中有記載“楚莊王鮮冠組纓，絳衣博袍……”。“絳衣”就是大紅色的衣服。王族的信仰和喜好具有很大的影響力，如“楚王好細腰，宮中多餓死”。王族尚赤，久而久之，楚地百姓也都以赤為貴了。從現今出土的楚文物也可以佐證楚人尚赤。
