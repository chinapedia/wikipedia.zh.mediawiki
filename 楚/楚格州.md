'''楚格州'''（[[德語|德語]]、[[羅曼語|羅曼語]]：Zug, [[法語|法語]]：Zoug, [[意大利語|意大利語]]：Zugo）位於[[瑞士|瑞士]]中部，是該國面積的第三小的州份，僅比[[巴塞爾城市州|巴塞爾城市州]]及[[內阿彭策爾州|內阿彭策爾州]]兩個州大。

== 地理 ==
楚格州西鄰[[阿爾高州|阿爾高州]]及[[琉森州|琉森州]]，北接[[蘇黎世州|蘇黎世州]]，南面與東面與[[施維茨州|施維茨州]]接壤。楚格州位於丘陵起伏的[[高原|高原]]上，州內最高點為韋爾特舒匹茲峰（Wildspitz）（海拔1580米）。境內主要的[[湖泊|湖泊]]計有[[楚格湖|楚格湖]]（Zugersee）及[[阿格利湖|阿格利湖]]（Ägerisee）。楚格州的主要[[河流|河流]]是羅澤河（Lorze），它把[[楚格湖|楚格湖]]和[[阿格利湖|阿格利湖]]連接起來，然後流向西北方與[[羅伊斯河|羅伊斯河]]（Reuss）滙合。

== 人口 ==
該州2002年[[人口|人口]]大約100,900，主要信奉[[羅馬天主教|羅馬天主教]]，主要語言是瑞士德語的瑞士中部[[方言|方言]]。在1814年以前楚格州屬於康斯坦斯教區，後來被編入[[巴澤爾|巴澤爾]]教區。

== 經濟 ==
主要工業有金屬品製造業、紡織業、機械製造業、電子業及製造Kirsch（一種含酒精飲品）。
楚格州的稅率為全國最低，因此吸引了不少企業到其首府[[楚格|楚格]]市成立總部。

== 交通 ==
楚格州有鐵路連接[[琉森|琉森]]及[[蘇黎世|蘇黎世]]，兩段鐵路於Arth-Goldau滙合，Arth-Goldau是貫通德瑞意的聖哥達鐵路的其中一站。

== 市鎮 ==
由於楚格州面積太小，所以沒有分區，而[[瑞士的市镇|市鎮]]級行政區共有11個，包括[[楚格市|楚格市]]（Zug），[[上阿格利|上阿格利]]（Oberägeri），[[下阿格利|下阿格利]]（Unterägeri），[[孟辛根|孟辛根]]（Menzingen），[[巴阿|巴阿]]（Baar），[[卡姆|卡姆]]（Cham），[[休倫堡|休倫堡]]（Hünenberg），[[施坦豪森|施坦豪森]]（Steinhausen），[[里希|里希]]（Risch），[[瓦爾希韋|瓦爾希韋]]（Walchwil），[[新海姆|新海姆]]（Neuheim）
