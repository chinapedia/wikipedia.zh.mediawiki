'''楚瓦什人'''（{{lang-cv|чӑваш}}），[[俄罗斯|俄罗斯]][[少数民族|少数民族]]。自称恰瓦什人。主要分布在[[俄罗斯|俄罗斯]][[楚瓦什共和国|楚瓦什共和国]]，部分分布在[[鞑靼斯坦共和国|鞑靼斯坦共和国]]、[[巴什科尔托斯坦共和国|巴什科尔托斯坦共和国]]以及乌里扬诺夫、古比雪夫、萨拉托夫等地，近代有少数人移居到[[西伯利亚|西伯利亚]]和[[中亚|中亚]]地区。属[[蒙古人种|蒙古人种]]与[[欧罗巴人种|欧罗巴人种]]的混合类型。使用[[楚瓦什语|楚瓦什语]]，分上、下楚瓦什两大方言，属[[阿尔泰语系|阿尔泰语系]][[突厥语族|突厥语族]]。有以[[斯拉夫字母|斯拉夫字母]]为基础的[[拼音文字|拼音文字]]。楚瓦什人历史上虽长期遭受异族统治，但仍保持着自己的民族语言、独特的物质文化生活和丰富的民间创作。

== 歷史 ==
楚瓦什人是來自中亞說突厥語的民族（特別是[[保加爾人|保加爾人]]），那些[[鐵勒|鐵勒]]人隨[[匈人|匈人]]西遷至[[東歐|東歐]]與當地[[烏拉爾語族|烏拉爾語族]]及[[伊朗語族|伊朗語族]]混合而成。当时他们是[[伏尔加保加利亚|伏尔加保加利亚]]，是[[可萨人|可萨人]]臣下，現在多數當是[[韃靼人|韃靼人]]的一種。

== 語言 ==
楚瓦什人的語言[[楚瓦什語|楚瓦什語]]屬[[突厥語族|突厥語族]]，含有大量突厥語辭彙，但是和一般突厥語分別很大。阿拉伯地理學家著重指出，從南俄到中國，突厥語是相通的。只有[[伏爾加河|伏爾加河]]流域的保加爾人與[[可薩人|可薩人]]例外。一般同属突厥语族的[[土耳其|土耳其]]人聽不懂，[[芬蘭|芬蘭]]人也聽不懂，是一種十分特殊的突厥語（[[烏戈爾語|烏戈爾語]]）。楚瓦什語有大量早期突厥語發展的痕迹。这些人可能在突厥汗國沒立國時已到伏爾加河。即是說楚瓦什語在蒙古語在已從突厥語分離出來，但現代書面與口語突厥語尚未形成時形成的。巴托爾德更進一步說，楚瓦什語就是[[北匈奴|北匈奴]]語。他們說的是一種未分化的突厥語，他們本身也是突厥化的[[烏拉爾語系|烏拉爾語系]]民族與斯拉夫人後人。

== 宗教 ==
楚瓦什人原有传统信仰，18世纪中叶起改信[[东正教|东正教]]。

== 經濟 ==
楚瓦什人过去主要从事[[农业|农业]]，种植[[黑麦|黑麦]]、[[燕麦|燕麦]]、[[大麦|大麦]]、[[大麻|大麻]]等，兼事[[畜牧业|畜牧业]]。制作木器、编织麻代等手工业也很发达。擅长刺绣和木刻，图案复杂，色彩鲜艳，被誉为刺绣之乡。[[十月革命|十月革命]]后，经济有很大发展，是俄罗斯的重要工业基地和农业先进地区。
