'''楚留香'''是[[古龍|古龍]][[武俠小說|武俠小說]]系列的經典男主角，是劫富濟貧的[[俠盜|俠盜]]，[[江湖|江湖]]中人尊稱他“盜帥”、“香帥”。

==介紹==
楚留香为人风流倜傥、足智多谋、观察入微、善良多情。在书中，他多次揭破错综复杂的[[阴谋|阴谋]]，也多次以他的机智、经验和镇定赢得了本来武功高他數倍的絕頂高手。而他的武功本来也属上乘，尤其轻功高绝，世上无人可及（在某些版本中並得李尋歡真傳[[小李飛刀|小李飛刀]]），一刀見血。不殺敵手、不願沾染血腥是他的特色之一，是以不論是敵是友，對他都是褒多於貶。

他的师承极为神秘，江湖上知晓的人极少。连无所不知的[[水母阴姬|水母阴姬]]门下也只推测出他是[[夜帝|夜帝]]（见古龙另一小说《[[铁血大旗门|铁血大旗门]]》）的关门弟子。而唯一可能知道他师承的[[胡铁花_(小说)|胡铁花]]认为，这个推测虽然不完全正确，但也与事实十分接近了。另一說法為[[鐵中棠|鐵中棠]]，因為鐵中棠在地洞被困期間，夜帝曾傳授武功。

在楚留香在第一部小说《[[血海飄香|血海飄香]]》裡出场时，他已经“成名近十年”。少年时代，他与胡铁花、[[姬冰雁|姬冰雁]]一起，被称为“雁蝶为双翼，花香满人间”。楚留香当时左有飞雁，右有彩蝶，笑傲江湖，纵横天下。一次胡铁花在醉酒后答应与华山派的清风女剑客[[高亞男|高亞男]]成亲，酒醒后反悔，不告而别；之后不久，姬冰雁也离开了。楚留香独自闯荡江湖，直到七年之后（高亚男追了胡铁花三年，之后胡铁花又在[[沙漠|沙漠]]边的小镇上徘徊四年），在《[[大沙漠|大沙漠]]》裡，楚留香才再次见到胡铁花和姬冰雁。

楚留香的年纪虽然从未在书中明确说明，但在《[[蝙蝠傳奇|蝙蝠傳奇]]》裡，楚留香说出胡铁花是三十三岁。而楚留香与胡铁花是从小一起长大的，七八岁时就曾经一起去偷酒喝，也一起在雪地裡玩，因此楚留香的年纪也应该与此相差不远。

楚留香与没有家的胡铁花不同，他有一艘自己的[[船|船]]。而且，船上有三个美丽的女孩子，是身世极苦、从她们十一、二岁时就跟着他的红颜知己：温柔体贴、精于易容的[[苏蓉蓉|苏蓉蓉]]，博闻强记、头脑清楚的[[李红袖|李红袖]]，还有妙手烹饪、调皮可爱的[[宋甜儿|宋甜儿]]。

他的其他朋友也很多：先友后敌的[[无花和尚|无花和尚]]、看似冰冷却热情如火的姬冰雁、外号「杀人不见血，剑下一点红」的[[中原一点红|中原一点红]]、好客的掷杯山庄庄主[[左輕侯|左輕侯]]，以及快网[[張三|張三]]。

楚留香的生活中也有许多可爱的女子。在《[[楚留香傳奇|楚留香傳奇]]》裡，他有[[蘇蓉蓉|蘇蓉蓉]]、李红袖和宋甜兒长伴身边，而且他似乎对蘇蓉蓉情有独钟，她姑母也称他是苏蓉蓉的心上人。蘇蓉蓉精於[[易容術|易容術]]，也擅長使毒。個性善解人意。大漠之王的女儿、女扮男装的[[黑珍珠|黑珍珠]]显然对他十分倾心。在《大沙漠》有冰雪聰明的[[龜兹國|龜兹國]][[琵琶公主|琵琶公主]]；《[[畫眉鳥|畫眉鳥]]》中有水母陰姬的弟子[[宮南燕|宮南燕]]；《[[鬼戀俠情|借屍還魂]]》裡有坚强刚毅的[[石繡雲|石繡雲]]等；《蝙蝠傳奇》裡有责任重大的[[華真真|華真真]]，没有眼睛的可怜女子[[東三娘|東三娘]]；《[[桃花傳奇|桃花傳奇]]》有活潑鬼靈精的[[張潔潔|張潔潔]]及[[艾青_(小说人物)|艾青]]、[[艾虹|艾虹]]、[[卜阿鵑|卜阿鵑]]等[[麻衣教|麻衣教]]弟子；《[[新月傳奇|新月傳奇]]》裡有冷漠高贵的[[新月_(小说人物)|新月]]。其中跟張潔潔甚至還有一子。

古龍為楚留香寫了八個故事：《血海飄香》（又名《天楓十四郎》）、《大沙漠》、《[[畫眉鳥_(小說)|畫眉鳥]]》（這三部合稱《鐵血傳奇》或《楚留香傳奇》）、《[[借屍還魂|借屍還魂]]》（又名《鬼戀俠情》）、《蝙蝠傳奇》、《桃花傳奇》、《新月傳奇》、《[[午夜蘭花|午夜蘭花]]》（這五部合稱《[[楚留香新傳|楚留香新傳]]》）。其中，《借屍還魂》、《蝙蝠傳奇》、《桃花傳奇》、《新月傳奇》特別描寫楚留香跟女主角的感情：對楚留香而言，除了三姝外，她們是最重要的女子。

==電視劇==
===1979年===
《[[楚留香_(1979年電視劇)|楚留香]]》，[[香港|香港]][[無線電視|無線電視]]，[[王天林|王天林]]監製，鄭少秋飾楚留香，趙雅芝飾蘇蓉蓉，[[高妙思|高妙思]]飾李紅袖，[[廖安麗|廖安麗]]飾宋甜兒，吳孟達飾[[胡鐵花_(小說)|胡鐵花]]，[[夏雨_(香港)|夏雨]]飾姬冰雁，關聰飾無花，[[汪明荃|汪明荃]]飾沈慧珊，[[黃樹棠|黃樹棠]]飾中原一點紅，[[程可為|程可為]]飾高亞男。

《[[俠盜風流|俠盜風流]]》，同一時間，當時的[[麗的電視|麗的電視]]（亞洲電視前身）亦推出同樣的版本，與無綫電視展開收視之戰，一時成為「雙胞胎」事件。但由于開播前夕与無綫電視劇集《楚留香》產生版權糾紛，播出前被迫更名。而且，劇中人物名稱、稱謂等全部進行配音更改。由[[潘志文|潘志文]]飾楚留香，[[魏秋樺|魏秋樺]]飾蘇蓉蓉，[[曾江|曾江]]飾姬冰雁、[[秦沛|秦沛]]飾無花、[[文雪兒|文雪兒]]飾黑珍珠、[[羅樂林|羅樂林]]飾中原一點紅。

《楚留香》在台灣原本以[[金鐘獎|金鐘獎]]「外片觀摩」方式，在[[中視|中視]]播出首兩集（[[粵語|粵語]]原音）；大受好評後，中視正式購入台灣播映權，並以[[國語|國語]][[配音|配音]]播出。1982年4月18日20:00開播之後，創下百分之七十的高[[收視率|收視率]]；每當週末晚上一到，[[計程車|計程車]]司機不載客，[[夜市|夜市]]小販不擺攤，全待在家裡看戲，連犯罪率都大為降低；當時僅能以「萬人空巷」形容，《楚留香》主題曲更是傳誦台灣大街小巷。

該劇基本上是《鐵血傳奇》的故事，不過和原著有多處不同。其一是汪明荃飾演的角色沈珊姑，名字改為「沈慧珊」，戲份也大增；另外，香帥多了一項絕技「彈指神功」。後期劇情改動幅度極大，蘇蓉蓉被改編為殺手組織派來臥底的殺手，同時也是楚留香最心愛的女人，最後蘇蓉蓉又因香帥而死，楚留香痛不欲生，決定退出江湖，雲遊四海。

由於《楚留香》大受歡迎，其電視劇續作不少。

該劇[[主題曲|主題曲]]由[[顧嘉煇|顧嘉煇]]作曲、編曲，[[鄧偉雄|鄧偉雄]]與[[黃霑|黃霑]]填詞，鄭少秋主唱。在台灣播出時的國語主題曲，則由[[原野三重唱|原野三重唱]]的男高音[[王強|王強]]（已故）主唱。

《[[楚留香傳奇_(電視劇)|楚留香传奇]]》，製片人[[游健鳴|游健鳴]]，[[朱孝天|朱孝天]]飾楚留香，[[陳浩民|陳浩民]]飾胡鐵花，[[胡靜|胡靜]]飾蘇蓉蓉，[[孫菲菲|孫菲菲]]飾李紅袖，[[秋瓷炫|秋瓷炫]]飾[[石觀音|石觀音]]，[[劉佳|劉佳]]飾宋甜兒，[[崔鵬|崔鵬]]飾[[無花|無花]]，[[王傳一|王傳一]]飾[[中原一點紅|中原一點紅]]，[[蕭薔|蕭薔]]飾[[水母陰姬|水母陰姬]]和[[魚尺素|魚尺素]]。本劇2007年12月10日在[[中國中央電視台電視劇頻道|中國中央電視台電視劇頻道]]（CCTV-8）首播，每晚三集，全劇由血海飄香、大沙漠、畫眉鳥三個故事組成。本剧的编剧是執筆过《[[大清後宮|大清後宮]]》、《[[最後的格格|最後的格格]]》、《[[煙花三月|煙花三月]]》、《[[胭脂雪|胭脂雪]]》等熱門剧集的[[于正|-{于}-正]]，其在受訪時強調本次改編將讓香帥風流不在，故而加重了主人公對[[宋甜兒|宋甜兒]]的感情描寫，讓香帥獲得真正的愛情。

==遊戲==
2001年，[[宇峻科技|宇峻科技]]（今[[宇峻奧汀科技|宇峻奧汀科技]]）推出[[電腦遊戲|電腦遊戲]]《[[楚留香新傳_(電腦遊戲)|楚留香新傳]]》，主要劇情包括原作的《借屍還魂》、《蝙蝠傳奇》、《桃花傳奇》、《新月傳奇》，以及原創香帥小時候的故事。插畫家是[[東方月|東方月]]。

===作品差異===
《楚留香新傳》與原作主要差異如下：
# 加入沒有在該四部作品登場的三姝（蘇蓉蓉、李紅袖、宋甜兒）、中原一點紅。
# 原作小說中主要敵人都成為《午夜蘭花》中提到的神秘組織幹部（如[[薛笑人|薛笑人]]、枯梅大師、卜阿鵑）；《新月傳奇》登場的[[史天王|史天王]]更是神秘組織的頭目，也是遊戲中最後的敵人。
# 《借屍還魂》[[小禿子|小禿子]]、[[小麻子|小麻子]]與[[小火神|小火神]]戲份合併成小火神一人
# 《借屍還魂》[[花金弓|花金弓]]與[[薛紅紅|薛紅紅]]戲份合併成花金弓一人。
# 《借屍還魂》兩位名醫[[張簡齋|張簡齋]]及[[王雨軒|王雨軒]]的稱號及地理位置應為「南張北王」，遊戲為配合之後新月傳奇之故，變為「南王北張」。
# 《蝙蝠傳奇》原隨雲並非與金靈芝一同墜海死，而是被楚留香一行人擊敗。
# 《蝙蝠傳奇》楚留香與東三娘在蝙蝠島上的劇情全都改為與華真真，並添加華真真戲份。
# 《蝙蝠傳奇》登場的[[英萬里|英萬里]]未死，白獵（白蠟燭）此角刪去。
# 《桃花傳奇》增加楚留香與張潔潔上華山見到華真真與高亞男的劇情。
# 《桃花傳奇》卜阿鵑劇情改為賣身葬父。
# 《新月傳奇》登場的新月與史天王均未死。
# 《新月傳奇》登場的[[白雲生|白雲生]]後遭[[豹姬|豹姬]]與卜阿鵑所殺。

《楚留香新傳》有幾個主要結局：
# 楚留香與三姝乘船遊世界（本體結局）
# 楚留香與石繡雲情侶結局（[[資料片|資料片]]新增結局）
# 楚留香與華真真情侶結局（資料片新增結局）
# 楚留香與張潔潔情侶結局（資料片新增結局）
# 楚留香與玉劍公主（即新月）情侶結局（資料片新增結局）
# 楚留香與三姝情侶結局（資料片新增隱藏結局）
# 胡鐵花與金靈芝情侶結局（資料片新增結局）
# 胡鐵花與高亞男情侶結局（資料片新增結局
