'''楚青'''（{{bd|1923年||2016年|2月21日|catIdx=Ch楚}}），原名'''詹永珠'''，为[[中国人民解放军大将|中国人民解放军大将]][[粟裕|粟裕]]的妻子。

== 生平 ==

1923年出生于[[江苏|江苏]][[扬州|扬州]]，早年参加[[新四军|新四军]]，后为避免家里受牵连，而改名楚青。1939年3月加入中国共产党。1941年2月26日，与粟裕于[[如东县|如东县]]石庄（新四军一师司令部所在地）结婚。<ref>{{Cite web|url = http://cpc.people.com.cn/GB/68742/106364/106365/6779417.html|title = 无悔的追求——记粟裕夫人新四军老战士楚青|accessdate = |author = |date = |publisher = 中国共产党新闻网}}</ref><ref>{{Cite web|url = http://www.thepaper.cn/newsDetail_forward_1434232|title = 开国大将粟裕将军夫人楚青逝世，享年93岁|accessdate = |author = |date = 2016-02-21|publisher = 澎湃新闻}}</ref>

[[中华人民共和国|中华人民共和国]]成立后，楚青和粟裕共同经历了1958年军委扩大会议粟裕受到批判，之后帮助粟裕整理[[粟裕回忆录|回忆录]]，于1988年完成。楚青也积极为为粟裕平反而奔走，并在1994年最终实现。

== 家庭 ==

粟裕与楚青育有二子一女。
* 长子[[粟戎生|粟戎生]]，1942年生于[[江苏|江苏]][[扬州|扬州]]，1961年参加[[中国人民解放军|解放军]]，毕业于[[哈尔滨军事工程学院|哈尔滨军事工程学院]]导弹控制专业，1999年晋升[[中国人民解放军中将|中将]]军衔。曾任[[第67集团军|第67集团军]]参谋长，率部参加[[两山战役|两山战役]]；后调任[[第24集团军|第24集团军]]军长、[[北京军区|北京军区]]副司令员等职，2006年1月退休。<ref>{{cite journal |author=余玮 |title=粟裕长子粟戎生中将的军礼 |journal=党史纵览 |year=2008 |month= |volume= |issue=08 |pages= |id= |url= |accessdate= |quote= }}</ref><ref>{{cite news |author=粟戎生 |coauthors= |title=粟裕之子粟戎生讲述父亲戎马人生 |url=http://news.sina.com.cn/c/p/2011-02-07/062021921450.shtml |work=南方都市报 |publisher= |date=2011-02-07 |accessdate=2011-02-22 }}</ref>
* 次子[[粟寒生|粟寒生]]，1947年出生，1968年初参加解放军，从事海上舰艇作业，后转业至[[交通部|交通部]]、[[中国远洋公司|中国远洋公司]]，曾任远洋公司副经理。<ref>{{cite journal |author=杨必军 |title=粟裕教子“十字诀” |journal=文史天地 |year=2004 |month= |volume= |issue=12 |pages= |id= |url= |accessdate= |quote= }}</ref>
* 女儿[[粟惠宁|粟惠宁]]，1949年出生，参加解放军，[[大校|大校]]军衔，1975年与[[陈毅|陈毅]]之子[[陈小鲁|陈小鲁]]结婚。
