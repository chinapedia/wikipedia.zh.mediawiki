'''毛野國'''（{{jpn|j = 毛野国|hg = けの の くに/け の くに|rm = Keno no kuni/Ke no kuni|kyu = 毛野國|hgt = }}），是古代[[日本|日本]]在設置[[令制國|令制國]]之前存在的一個國。其領域為現在日本[[關東|關東]]地區北部（[[北關東|北關東]]）[[群馬縣|群馬縣]]和[[栃木縣|栃木縣]]一帶。有'''毛野'''、'''毛州'''的別稱。

毛野國名稱的由來，目前共有兩種說法。一種說法認為來源於[[倭王權|倭王權]]對當地原住民[[蝦夷|蝦夷]]的稱呼「毛人」；另一種說法則認為當地於一年之間在同一塊地中可以培育兩種穀類作物（[[二毛作|二毛作]]），故稱為「毛野」。

在[[古墳時代|古墳時代]]，毛野地區的人以毛野川（今[[鬼怒川|鬼怒川]]）為中心，形成了自己獨特的文化圈——毛野國。

依據《[[先代舊事本紀|先代舊事本紀]]》卷第十〈國造本紀〉之內容，[[仁德天皇|仁德天皇]]（第16代[[天皇|天皇]]）時，毛野國被分爲'''{{link-ja|上毛野國|上毛野国|上毛野國}}'''和'''{{link-ja|下毛野國|下毛野国|下毛野國}}'''<ref>{{Cite book|author = [[蘇我馬子|蘇我馬子]]等|title = 先代舊事本紀|location = |publisher = [[洛陽書林|洛陽書林]]　[[前川茂右衞門|前川茂右衞門]]|date = 1644年|accessdate = 2019年3月23日|url = http://dl.ndl.go.jp/info:ndljp/pid/2563305/36}}</ref>。其首領各為{{link-ja|上毛野國造|上毛野国造}}（{{link-ja|上毛野氏}}）和{{link-ja|下毛野國造|下毛野国造}}（{{link-ja|下毛野氏}}），是{{link-ja|豐城入彥命|豊城入彦命}}（[[崇神天皇|崇神天皇]]長子）的後代。此二氏在當時與{{link-ja|吉備氏}}、[[筑紫氏|筑紫氏]]並列為大[[豪族|豪族]]。

[[和銅|和銅]]六年（713年），上毛野國改稱'''[[上野國|上野國]]'''<ref>{{Cite book|author = [[群馬縣|群馬縣]]敎育會|title = 群馬縣史. 第1卷|location = |publisher = 群馬縣敎育會|date = 1927年|accessdate = 2019年3月23日|url = http://dl.ndl.go.jp/info:ndljp/pid/1258714/21}}</ref>。又8世紀前半，下毛野國改稱'''[[下野國|下野國]]'''<ref>{{Cite web|url = https://www.inishie.tochigi.jp/history.html|title = とちぎの歴史|accessdate = 2019年3月23日|website = とちぎいにしえの回廊|publisher = [[栃木縣|栃木縣]]|language = ja}}</ref>。
