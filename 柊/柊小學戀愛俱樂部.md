== 故事簡介 ==
櫻井雛子是個就讀小學五年級的女孩子。受到就讀高中生的姐姐的影響而開始對戀愛有興趣的雛子，為了「當遇到自己喜歡的人時，談戀愛才不會失敗」的理由與同班同學的'''霧生螢'''與'''日下部澪'''一起成立「戀愛部」，但是她們能夠成功嗎？<ref name="Gamer" />

== 登場人物 ==
; {{nihongo|櫻井 雛子|桜井 雛子（さくらい ひなこ）||}}
:本作品的女主角，就讀小學五年級。個性開朗與好奇心旺盛的性格。對戀愛有所興趣，由於小學到高中皆為男女校舍分開的緣故，擔心無法好好的談戀愛，但是為了讓男生跟女生能夠在一起而設立戀愛部，並且成為社團的社長{{refTag|name=A1|1=單行本第一冊第19頁}}。而在森林學校的時候與織倉海相遇，在這之後雛子非常在意他{{refTag|name=C|1=單行本第二冊第20頁}}。剛開始因為年齡太小不允許帶手機{{refTag|name=A2|1=單行本第一冊第68頁}}，後來因為由奈有帶手機的緣故，使得她變得想要有手機，在暑假的時候在一個禮拜之內寫完暑假作業以及幫忙做家事{{refTag|1=單行本第四冊第69頁}}，完成母親的要求後才允許她買手機{{refTag|1=單行本第四冊第84頁}}。
:是單行本各冊唯一[[封面人物|封面人物]]。
; {{nihongo|霧生 螢|霧生 蛍（きりゅう ほたる）||}}
:雛子的同班同學也是好朋友，戀愛部的成員之一。留著長頭髮，戴著一副眼鏡，個性冷靜像成年人般的思維，與性格相似的赤瀨優相同。她擅長讀書，但是她不擅長運動{{refTag|1=單行本第二冊第14頁}}。因為自己的名字被雛子寫進戀愛部的申請書，因此加入戀愛部{{refTag|name=D|1=單行本第二冊第21頁}}。父母親經常不在家的緣故，所以經常帶著手機{{refTag|name=A1}}。不允許赤瀨優教導雛子不該學的東西{{refTag|1=單行本第三冊第88頁}}。
; {{nihongo|日下部 澪|日下部 澪（くさかべ みお）||}}
:雛子與螢的同班同學同時也是好朋友，戀愛部的成員之一。個性活潑好動，說話時的樣子像男孩子。進入戀愛部的理由與螢相同{{refTag|name=D}}。稱呼雛子為「小雛」（{{lang|ja|ひな}}）。擅長運動，並且受到女學生的歡迎{{refTag|1=單行本第三冊第94頁}}，喜歡青梅竹馬的大樹。
; {{nihongo|藤矢|藤矢（ふじや）||}}
:雛子等人班上的班導師{{refTag|1=單行本第一冊第22頁}}。從以前開始對戀愛沒有好印象，認為談戀愛對小學生來說實在是太早了，對有男生加入戀愛部則是不予置評。去年是赤瀨班上的副班導師{{refTag|1=單行本第三冊第12－14頁}}。
; {{nihongo|來-{栖}- 月子|来栖 月子（くるす つきこ）<ref>{{cite web|url=http://azumayuki.jugem.jp/?eid=495 | script-title=ja:フルコンプｗ |publisher={{lang|ja|AZUMAYUKI DIARY}} |date=2010-06-10|accessdate=2011-04-01 |language=ja}}</ref>||}}
:柊小學的保健老師，同時也是戀愛部的指導老師，將保健室當作戀愛部的社團教室，是個個性和善的女性。經常戲弄藤矢老師。
; {{nihongo|櫻井 小鳥|桜井 小鳥（さくらい ことり）||}}
:雛子的姐姐，就讀高中，與雛子同住一個房間{{refTag|1=單行本第一冊第12頁}}。跟同校的學長告白之後一起交往著{{refTag|1=單行本第一冊第62－68頁}}。
; {{nihongo|織倉 海|織倉 海（おりくら かい）||}}
:就讀柊小學的學生，與赤瀨優跟菊池大樹同班。在森林學校期間遇到膝蓋受傷的雛子，給她[[创可贴|创可贴]]，之後雛子非常在意他{{refTag|name=C}}。後來再度相遇時卻不記得雛子的事情。他喜歡魚類的緣故，相關的知識非常豐富。因為雛子的要求之下，被優與大樹兩人引誘之下到視聽教室玩[[野球拳|野球拳]]，他卻不曉得輸了要進入戀愛部；雖然他贏了野球拳，但是負起連帶責任而來到社團{{refTag|name=B|1=單行本第二冊第149頁}}。
; {{nihongo|赤瀨 優|赤瀬 優（あかせ ゆう）||}}
:與織倉海跟菊池大樹是同班同學，成績非常優秀，具有領袖的氣質，不喜歡向其他人示弱。因為父母親離婚的緣故，與姐姐兩人一起生活{{refTag|1=單行本第三冊第14、15頁}}。個性與螢一樣如同成年人般的思維，覺得他的姐姐和與霧生螢散發著相同的氣息{{refTag|1=單行本第三冊第19頁}}。最初進社團時提議玩野球拳，後來輸了遊戲，因此負起責任加入戀愛部{{refTag|name=B}}。
; {{nihongo|菊池 大樹|菊池 大樹（きくち だいき）||}}
:與織倉海跟赤瀨優是同班同學，與澪是青梅竹馬{{refTag|1=單行本第二冊第68頁}}。在進社團之前對戀愛部沒有太大的興趣{{refTag|1=單行本第二冊第116頁}}，在優提議之下玩野球拳，但是輸得非常徹底{{refTag|1=單行本第二冊第134頁}}。最後優輸了野球拳，因此負起連帶責任加入戀愛部{{refTag|name=B}}。
; {{nihongo|上條 由奈|上条 ゆな（かみじょう ゆな）||}}
:就讀柊小學的學生，同時也是雛子的朋友，留著縱卷髮的少女。在暑假的時候，偶然在藥局遇到雛子，之後在學校跟她變得親近，因此成為她的朋友。雙親總是不在家的緣故，經常看家而不習慣自己一個人洗澡{{refTag|1=單行本第四冊第153頁}}。

== 舞台設定 ==
; {{nihongo|柊小學|柊小学校}}
:故事舞台的私立小學，從小學到高中的一貫教育。由於採取[[男女分班|男女分班]]，因此校舍分為男子校舍以及女子校舍，而上課的時候也是男女分開上課{{refTag|1=單行本第一冊第18頁}}。因為是私立小學的緣故所以有制服，男女皆為[[日本學生服|水手服]]，男生為短褲，女身為連身洋裝。
; {{nihongo|戀愛部|恋愛部}}
:練習戀愛的社團，主要以[[約會|約會]]的練習活動為主。這是雛子為了將來談戀愛不會失敗而成立的社團，由來-{栖}-老師擔任指導老師。因為戀愛部的名稱容易引發誤會，因此申請成立時以「'''戀'''舊'''愛'''新研究俱樂'''部'''」{{noteTag|1=日語版原文為「{{lang|ja|'''連'''携'''相'''性研究'''部'''}}」，而此三字的日語與「戀愛部」同音，而中文版翻譯時採用相同的方式取「'''戀'''舊'''愛'''新研究俱樂'''部'''」。}}簡稱「戀愛部」這個牽強的名義，因此校長允許成立社團{{refTag|1=單行本第一冊第41頁}}。而戀愛部的標誌是由[[新手駕駛標誌|初學者]]標誌與[[心形符號|愛心標誌]]作結合{{refTag|1=單行本第一冊第38頁}}。最初只有雛子、澪以及螢等三個女生團員，而後來海、大樹以及優等人加入之後才有男生團員。
