'''盎司'''（{{lang-en|'''ounce'''}}；符號：oz），或作'''盎斯'''，[[香港用語|港譯]]「安-{}-士」或「盎士」，為[[英制|英制]][[質量|質量]]、[[重量|重量]]或[[體積|體積]]的单位，旧称'''英两'''或'''唡'''。

==作為質量單位==
* [[常衡|常衡]]盎司：質量单位，整体缩写为oz.av。
: 1盎司 = 28.349523125[[克|克]]
: 1盎司 = 16[[打兰|打兰]]（dram）
: 1盎司 = 437.5[[格林_(單位)|格林]]
: 16盎司 = 1[[磅|磅]]（pound）

* [[金衡|金衡]]盎司：質量单位，整体缩写为oz.tr（英）、oz.t（美），常见于金银等[[贵金属|贵金属]]的计量。
: 1盎司 = 480格林
: 1盎司 = 31.1034768 克
: 12盎司 = 1磅

* [[药衡|药衡]]盎司：質量单位，同金衡盎司，整体缩写为ap oz。
: 1盎司 = 31.1034768克

==作為重量的單位==
1[[盎司力|盎司力]]（Ounce-force）等於{{frac|16}}[[磅力|磅力]]或{{convert|1/16|lbf|N|7|abbr=off|lk=out|disp=output only}}。<ref>[https://en.oxforddictionaries.com/definition/ounce-force ounce-force]</ref>

==作為體積單位==
* 液盎司（Fluid ounce）是容量计量单位，符号为fl. oz。
: 160英-{制}-液盎司 = 1英-{制}-[[加侖|加侖]]
: 1英-{制}-液盎司 = 28.4130625[[毫升|毫升]]
: 128美-{制}-液盎司 = 1美-{制}-加侖
: 1美-{制}-液盎司 = 29.5735295625毫升

== 用盎司表示厚度 ==
在[[印刷電路板|印刷電路板]]及[[柔性印刷电路板|柔性印刷電路板]]的製作中，會用盎司表示其[[銅箔|銅箔]]的厚度，1盎司銅箔表示在一平方[[英尺|英尺]]的面積上舖重量一盎司的銅，相當於0.034[[公厘|公厘]]或1.3[[密耳|mil]]<ref>[https://www.cadence.com/tw/Resources/thickness.pdf cadence 銅箔厚度的計算] {{Wayback|url=https://www.cadence.com/tw/Resources/thickness.pdf |date=20160409074546 }}</ref>。
