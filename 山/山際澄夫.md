'''山際澄夫'''（{{bd|1950年||||Yamagiwa Sumio}}）是日本的[[新聞工作者|新聞工作者]]、前[[產經新聞|產經新聞]]記者。{{来源请求|曾於富士電視台節目抨擊日本政府未對台灣援助東日本大地震表達感謝是外交腦死。}}

== 簡歷 ==
[[山口縣|山口縣]][[下關市|下關市]]出身。[[明治大學|明治大學]]畢業後，1975年進入[[產經新聞社|產經新聞社]]。在{{link-ja|政治部記者|政治部記者|政治部}}歴任首相官邸、外務省主任等。[[波斯灣戰爭|波斯灣戰爭]]期間，遠赴[[伊拉克|伊拉克]]進行採訪。曾任[[紐約|紐約]]支局長，2002年離開報社<ref>{{cite web |url=http://bookweb.kinokuniya.co.jp/htm/4898311172.html |title=これでも朝日新聞を読みますか？ - 著者紹介 |publisher=紀伊國屋書店 |accessdate=2011-05-14}}</ref><ref name="特集">{{cite web |url=http://www.ch-sakura.jp/hodo/election.html?id=630 |title=特集 - 評論人 |publisher=[[樱花频道|チャンネル桜]] |accessdate=2011-05-14}}</ref>。

他主要在外交、政治、媒體等領域的言論性雜誌《[[諸君！|諸君！]]》（{{lang|ja|諸君!}}）、《[[WiLL|WiLL]]》、《[[Voice|Voice]]》上投稿<ref name="特集" />，也時常以評論人的身份出席電視節目（如《[[討論到天亮|討論到天亮！]]》、《[[北野武的TV擒抱|北野武的TV擒抱]]》（{{lang|ja|TVタックル}}）等）。
