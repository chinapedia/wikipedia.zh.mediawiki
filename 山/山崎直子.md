'''山崎直子'''（{{bd|1970年|12月27日|||Yamazaki Naoko}}）是[[日本|日本]]继[[向井千秋|向井千秋]]之后的第二位女性航天員。出生在[[千葉縣|千葉縣]][[松戶市|松戶市]]。舊姓角野。
==生平==
出生于[[松戶市|松戶市]]，童年时代在[[札幌市|札幌市]]度过了2年时光<ref>{{Cite web |url=http://www.nasa.gov/mission_pages/shuttle/shuttlemissions/sts131/interview_yamazaki_prt.htm |title=Preflight Interview: Naoko Yamazaki, Mission Specialist |author=NASA |date=2010-03-08|language=en |accessdate=2010-03-11}}</ref>。受[[天文館|天文館]]及[[旅行者1號|旅行者太空探測器]]傳來的照片、以及動畫作品[[宇宙戰艦大和號|宇宙戰艦大和號]]的影響而立下了前往宇宙的志向<ref>{{cite web |author=JAXA |script-title= ja:山崎直子 初めての宇宙飛行に向けて～子どもに宇宙の素晴らしさを伝えたい～ |url=http://www.jaxa.jp/article/interview/vol52/ |publisher=JAXA |accessdate=2019-04-09|language=ja}}</ref>。1989年3月毕业于[[御茶水女子大学|御茶水女子大学]]附属高中，1993年3月毕业于[[東京大学|東京大学]]工学院航空专业，1996年获得东京大学航空航天工程[[硕士|硕士]]学位<ref name=bio1/>。

1999年2月被JAXA确定为宇航员候选人，1999年4月开始参加国际空间站宇航员基础训练，2001年9月成为正式宇航员<ref name=bio1>{{cite web |url=http://iss.jaxa.jp/astro/profile_e.html#yamazaki |title=Naoko Yamazaki |last=JAXA |date=2008-05-19 |publisher=Japanese Aerospace Exploration Agency |accessdate=2008-11-18 |deadurl=yes |archiveurl=https://web.archive.org/web/20101124225826/http://iss.jaxa.jp/astro/profile_e.html#yamazaki |archivedate=2010-11-24|language=en}}</ref>，11月获得[[松戶市|松戶市]]民荣誉奖<ref>{{cite web|url=http://www.city.matsudo.chiba.jp/index/profile/shimineiyosyou.html|script-title=ja:松戸市民栄誉賞|publisher=松戸市公式ホームページ|date=2011-03-10|language=ja|access-date=2019-04-09|archive-url=https://web.archive.org/web/20110305054554/http://www.city.matsudo.chiba.jp/index/profile/shimineiyosyou.html|archive-date=2011-03-05|dead-url=yes}}</ref>。

2000年12月与自称为「世界第一位[[民間商業宇航員|民間商業宇航員]]」的[[山崎大地|山崎大地]]结婚。2006年2月取得了[[搭乘運用技術者|搭乘運用技術者]]的資格。2007年受邀为日本电视动画节目《[[火箭女孩|火箭女孩]]》中的宇航员配音。

2010年4月5日至4月20日搭乘[[發現號太空梭|發現號太空梭]]前往[[國際空間站|國際空間站]]，參加設備組裝的任務<ref>{{cite web|author=Harwood, William|url=https://www.nytimes.com/2010/04/06/science/space/06shuttle.html?hp |title=Shuttle Lifts Off for Space Station|publisher=[[The_New_York_Times|The New York Times]]|location=[[New_York_City|New York City]]|date=2010-04-05|language=en}}</ref>。

2011年宣布退役，开始从事宇宙科普教育活动<ref>{{cite book |author=塩谷京子 |script-title= ja:10分で読める 宇宙や世界を冒険した人の伝記 |date=2015年 |publisher=学研プラス |isbn=9784059153436 |page=29 |url=https://books.google.com/books?id=php9DAAAQBAJ&pg=PT29& |language=ja}}</ref>，2018年5月19日参加了[[上海科技馆|上海科技馆]]科技节特别活动<ref>{{cite web |author1=郑莹莹|author2=李姝徵 |title=日本“航天妈妈”山崎直子现身沪上 与观众分享航天梦 |url=http://www.chinanews.com/sh/2018/05-19/8517789.shtml |publisher=中国新闻网 |language=zh-cn |date=2018年5月19日}}</ref>。
