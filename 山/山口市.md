'''山口市'''（{{jpn|j=山口市|hg=やまぐちし|rm=Yamaguchi shi|f=y}}）是位於[[日本|日本]][[山口縣|山口縣]]中部的城市，亦是山口縣的[[縣廳所在地|縣廳所在地]]；面積為山口縣內最大，南臨[[瀨戶內海|瀨戶內海]]，北接[[島根縣|島根縣]]，主要市區位於南部，全市人口數為山口縣內第二多，次於[[下關市|下關市]]。

雖然山口市是山口縣縣廳所在，為該縣的行政中心，但山口縣內的經濟活動多集中於[[周南市|周南市]]、[[防府市|防府市]]和[[下關市|下關市]]，也成為日本各縣廳所在地中，少見的純粹以行政機能為主的城市，但由於主要市區已發展有超過五百年以上的歷史，觀光業也成為市中心的主要產業；位於南部的小郡地區，由於具有交通便利的優勢，反而成為各公司設立負責山口縣營運的主要據點。

山口市市也是日本各都道府縣的縣廳所在地中人口較少的，曾多次都道府縣的縣廳所在地中人口最少的一個，現在為人口第三少的城市，僅略多於[[島根縣|島根縣]]縣廳[[松江市|松江市]]和[[山梨縣|山梨縣]]縣廳[[甲府市|甲府市]]。

以山口市為主場的職業球隊有[[日本職業足球聯賽|日本職業足球聯賽]]的[[山口雷法足球俱樂部|山口雷法足球俱樂部]]。

== 歷史 ==

現在的山口市轄區南部主要市區在令制國時代屬於[[周防國|周防國]][[吉敷郡|吉敷郡]]，東部屬於[[佐波郡_(山口縣)|佐波郡]]，北部山區則屬於[[長門國|長門國]][[阿武郡|阿武郡]]。

在[[南北朝時代|南北朝時代]]此地由[[大內氏|大內氏]]統治並成為其主要據點，在15世紀[[應仁之亂|應仁之亂]]後，在大內氏統治下相對於其他地方安定許多，不少[[京都|京都]]的文化人來到這裡避難，一時間亦被稱為西京都；加上與[[中國|中國]]、[[朝鮮|朝鮮]]的貿易往來頻繁，發展出了獨特的{{Tsl|ja|大内文化|大內文化}}。

1552年來自[[瓦倫西亞|瓦倫西亞]]的[[傳教士|傳教士]]{{Tsl|en|Cosme de Torres}}曾在此帶領日本信徒[[圣诞节|圣诞节]]的祝禱活動，成為全日本最早進行聖誕節活動的地方。

16世紀大內氏被[[毛利氏|毛利氏]]取代後，由於毛利氏的主要據點是位於[[安藝國|安藝國]]的[[廣島城|廣島城]]，失去了政治中心的地位；在1600年[[關原之戰|關原之戰]]後，毛利氏失去了原有的居城廣島城，被減封剩下[[長門國|長門國]]及[[周防國|周防國]]，在選擇新的居城地點時曾一度將本地的{{Tsl|ja|鴻ノ峰|鴻之峰}}納入考慮，但最終選擇了最偏遠的[[萩|萩]]興建[[萩城|萩城]]，直到1863年長州藩因決定要對抗外國勢力，為避免臨海的萩城遭受外國軍艦自海上的攻擊，才將藩廳遷至山口的[[山口城|山口政事堂]]，山口才再次成為政治的中心，長州藩也因此被改稱為「山口藩」，同時也成為19世紀末[[倒幕運動|倒幕運動]]的重要據點。<ref name="hagicastle-history01">{{Cite web| url=http://hagi.jp/~castle/history_01.html| title=萩城の歴史 ＞ 1. 毛利氏城地の決定| publisher=萩城復原研究會| accessdate=2019-03-16| language=ja}}</ref>

在[[廢藩置縣|廢藩置縣]]後，仍持續成為山口縣的政治中心；1889年日本實施[[町村制|町村制]]，現在的山口市轄區在當時分屬{{Tsl|ja|仁保村 (山口県)|仁保村 (山口縣)|仁保村}}、{{Tsl|ja|小鯖村}}、{{Tsl|ja|大内村 (山口県)|大內村 (山口縣)|大內村}}、{{Tsl|ja|宮野村 (山口県)|宮野村 (山口縣)|宮野村}}、'''{{Tsl|ja|山口町}}'''、{{Tsl|ja|上宇野令村}}、{{Tsl|ja|下宇野令村}}、{{Tsl|ja|吉敷村}}、{{Tsl|ja|大歳村|大歲村}}、{{Tsl|ja|平川村_(山口県)|平川村_(山口縣)|平川村}}、{{Tsl|ja|秋穂町|秋穗町|秋穗村}}、{{Tsl|ja|鋳銭司村|鑄錢司村}}、{{Tsl|ja|秋穂二島村|秋穗二島村}}、{{Tsl|ja|名田島村}}、{{Tsl|ja|陶村_(山口県)|陶村_(山口縣)|陶村}}、{{Tsl|ja|小郡町|小郡町|小郡村}}、{{Tsl|ja|嘉川村}}、{{Tsl|ja|阿知須町|阿知須町|井關村}}、{{Tsl|ja|出雲村}}、{{Tsl|ja|島地村}}、{{Tsl|ja|串村_(山口県)|串村_(山口縣)|串村}}、{{Tsl|ja|八坂村 (山口県)|八坂村 (山口縣)|八坂村}}、{{Tsl|ja|柚野村 (山口県)|柚野村 (山口縣)|柚野村}}、{{Tsl|ja|篠生村}}、{{Tsl|ja|生雲村}}、{{Tsl|ja|地福村}}、{{Tsl|ja|徳佐村|德佐村}}、{{Tsl|ja|嘉年村}}共28個行政區劃。

1905年至1929年期間，山口町先是陸續併入上宇野令村、下宇野令村、吉敷村後成為新設立的第一代'''山口市'''。

1944年山口市再次與周邊的行政區劃合併為新的第二代'''山口市'''，但僅三年後，原本一同併入的{{Tsl|ja|阿知須町}}和{{Tsl|ja|小郡町}}又先後於1947年及1949年從山口市中分出，成為獨立的行政區劃。

1950年代至1960年代的昭和大合併期間，山口市又陸續併入鑄錢司村、大內町、仁保村、小鯖村，同時其他行政區劃也先後整併至剩下[[阿東町|阿東町]]、{{Tsl|ja|徳地町|德地町}}和{{tsl|ja|秋穂町|秋穗町}}。

2005年山口市與小郡町、秋穗町、阿知須町、德地町再次進行[[市町村合併|合併]]，成為新的第三代'''山口市'''<ref>{{cite web| language =ja| publisher =山口市政府| title =山口市の概要| url =http://www.city.yamaguchi.lg.jp/daboutyamaguchi/gaiyou/index.htm| author =| date =2010-01-16| accessdate =2010-08-05| deadurl =yes| archiveurl =https://web.archive.org/web/20091226114115/http://www.city.yamaguchi.lg.jp/daboutyamaguchi/gaiyou/index.htm| archivedate =2009-12-26}}</ref>，2010年，位於北部的[[阿東町|阿東町]]也被併入山口市。

== 交通 ==

[[File:Shin-Yamaguchi_Station_2.jpg|thumb]]

轄內並無機場，距離最近的機場是位於南側[[宇部市|宇部市]]境內的[[山口宇部機場|山口宇部機場]]。

[[山口車站|山口車站]]雖然位於主要市區內，但位於市區南部小郡地區的[[新山口車站|新山口車站]]才是轄內最主要的轉乘車站，通過山口市轄內的所有鐵路路線包括[[山陽新幹線|山陽新幹線]]、[[山陽本線|山陽本線]]、[[山口線|山口線]]、[[宇部線|宇部線]]都交會於此。

高速公路則有[[中國自動車道|中國自動車道]]、[[山陽自動車道|山陽自動車道]]、{{Tsl|ja|山口宇部道路}}通過轄內，並在轄區南部主要市區周邊相互形成路網。

客運則仰賴{{Tsl|ja|防長交通}}和[[中國JR巴士|中國JR巴士]]所經營的路線。

山口市主要市區為自[[室町時代|室町時代]]開始發展的城市，因此有多個歷史悠久的廟宇，包括{{Tsl|ja|月輪寺_(山口市)|月輪寺_(山口市)|月輪寺}}、[[瑠璃光寺|瑠璃光寺]]、{{Tsl|ja|常栄寺_(山口市)|常榮寺_(山口市)|常榮寺}}、{{Tsl|ja|龍福寺 (山口市)|龍福寺 (山口市)|龍福寺}}、{{Tsl|ja|古熊神社}}、{{Tsl|ja|山口大神宮}}、{{Tsl|ja|豊榮神社・野田神社|豐榮神社、野田神社}}、{{Tsl|ja|仁壁神社}}、{{Tsl|ja|木戸神社|木戶神社}}。

16世紀中，第一位到達日本傳教的[[天主教|天主教]]傳教士[[方濟·沙勿略|方濟·沙勿略]]曾在1551年來到山口，因此在1952年興建了{{Tsl|ja|山口サビエル記念聖堂|沙勿略紀念聖堂}}紀念他到日本四百年，其獨特的外型也成為山口市內的一個景點。

[[湯田溫泉|湯田溫泉]]位於主要市區南側，[[長門湯本温泉|長門湯本温泉]]、{{Tsl|ja|俵山温泉|俵山溫泉}}、{{Tsl|ja|川棚温泉|川棚溫泉}}被合稱為「{{Tsl|ja|防長四湯}}」，為山口縣內較具有歷史的溫泉區。<ref>{{Cite web| url=http://www.kazan.or.jp/J/QA/sr/qa-4126.html| title=火山についてのQ＆A - Question #4126| publisher=日本火山學會| date=2003-08-22| language=ja| accessdate=2019-03-04}}</ref>

== 教育 ==

[[File:Yamaguchiuniv.jpg|thumb]]

位於山口市的高等教育學校包括[[山口大學|山口大學]]、[[山口縣立大學|山口縣立大學]]、{{Tsl|ja|山口学芸大学|山口學藝大學}}、[[山口藝術短期大學|山口藝術短期大學]]，其中屬於國立的山口大學的歷史可以追溯到1815年創立的山口講堂，在1949年整併了山口周邊地區許都的高等學校後成為了現在的山口大學。

== 媒體 ==

由於是山口縣的縣廳所在地，大部分山口縣的媒體包括[[NHK山口放送局|NHK山口放送局]]、[[山口電視台|山口電視台]]、[[山口朝日放送|山口朝日放送]]、{{Tsl|ja|エフエム山口|FM山口}}都是將主要據點設在此，例外的只有據點設於[[周南市|周南市]]的[[山口放送|山口放送]]和設在[[下關市|下關市]]的[[山口新聞|山口新聞]]。
