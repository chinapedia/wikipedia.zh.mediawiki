'''山形市'''（{{jpn|j=山形市|hg=やまがたし|rm=Yamagata shi|f=y}}）是位於[[日本|日本]][[山形縣|山形縣]]中部偏東的一個[[都市|都市]]，同時是山形縣的縣廳所在地及人口最多都市，也是管轄[[村山地方|村山地方]]的[[村山綜合支廳|村山綜合支廳]]辦公室所在地。山形市是山形都市圈的中心城市，現為[[施行時特例市|施行時特例市]]。山形市自[[最上氏|最上氏]]在此修築[[山形城|山形城]]之後發展為一大城下町。江戶時代時，由於藩主頻繁變更，山形城下町規模縮小，但町人町卻得到發展，使得城市整體仍得以保持一定規模。廢藩置縣後，山形市成為山形縣廳所在地，和山形縣內其他城市的規模差距也有拉大。現在由於山形市前往[[東北地方|東北地方]]最大城市[[仙台市|仙台市]]的交通十分便利，因此近年亦逐漸開始有仙台市衛星城市的色彩。

== 歷史 ==
[[File:Yamagata_Castle_hon-maru.JPG|thumb]]
山形市歷史悠久，市內發現了[[古墳時代|古墳時代]]的遺跡（嶋遺跡）<ref>{{cite web |url=https://www.city.yamagata-yamagata.lg.jp/kankokyaku/sub3/bunkazai/6327pd0608162049.html |script-title=ja:史跡　嶋遺跡について |accessdate=2017-07-05|publisher={{lang|ja|山形市}} |language=ja}}</ref>。日本實施[[日本律令制|律令制]]之後，山形市地區屬於最上郡。在[[平安時代|平安時代]]初期的文獻《和名類聚抄》中，記載有在今山形市南側有「山方鄉」（「山方」和「山形」同音），此為「山形」地名的由來<ref name="history">{{cite web |url=http://www.pref.yamagata.jp/bunkyo/bunka/about/803profile.html |script-title=ja:山形県について |accessdate=2017-07-05 |publisher={{lang|ja|山形県}} |language=ja |deadurl=yes |archiveurl=https://web.archive.org/web/20170312185958/http://www.pref.yamagata.jp/bunkyo/bunka/about/803profile.html |archivedate=2017年3月12日 |df= }}</ref>。平安時代時，山形市轄區大部分地區屬於皇室領莊園大山莊。在平安時代末期，山形名為最上，是出羽路上的主要宿場之一<ref name="intro">{{cite web |url=https://www.city.yamagata-yamagata.lg.jp/kakuka/shoko/brandsuishin/sogo/chukatukeikaku/gazoufile/keikakuzenbun-2.pdf |script-title=ja:山形市中心市街地活性化基本計画 |accessdate=2017-07-05 |publisher={{lang|ja|山形市}} |language=ja}}</ref>。進入[[鎌倉時代|鎌倉時代]]後，關東武士團開始成為山形縣各地的統治者。這些武士團在之後以地名為勢力名稱，開始本地化。

[[室町時代|室町時代]]中期，{{link-ja|斯波兼頼|斯波兼頼}}攻入村山地方，並之後改以「最上」為姓，包括山形市在內的村山地方自這一時期至江戶時代初期是[[最上氏|最上氏]]的勢力範圍<ref name="intro"></ref>。最上氏第11代當主[[最上義光|最上義光]]以山形為據點統一了最上、村山地區<ref name="history"></ref>，並且修築了[[山形城|山形城]]及城下町，確立了今日山形的城市格局。[[關原之戰|關原之戰]]後，最上氏的[[石高|石高]]一度達57萬石，山形也因此得到發展<ref name="Chishi">{{cite book |author={{lang|ja|田村俊和·石井英也・日野正輝}}|script-title=ja:《日本の地誌4　東北》 |year=2008 |publisher={{lang|ja|朝倉書店}} |location=東京都 |isbn=4-254-16764-4|language=ja}}</ref>{{rp|382}}。然而1622年（[[元和_(日本)|元和]]8年）之後，最上家因內訌頻繁而被轉封，此後山形城領主頻繁變更，石高也被大幅削減，城市陷入蕭條。18世紀時，山形城的武家屋敷已被拆除，改為農地。幕末時期，山形城本丸內甚至都沒有建築<ref name="Chishi"></ref>{{rp|383}}。與城下町相對的是，七日町、十日町等町人町則因紅花貿易而日益繁榮，成為東北地方前列的商業城市。這一城下町荒廢的同時町人町卻日益發展的情況在[[江戶時代|江戶時代]]十分罕見<ref name="Chishi"></ref>{{rp|383}}。

[[File:Bunsyokan2006_Front_Gate.jpg|thumb]]
[[廢藩置縣|廢藩置縣]]之後，山形成為山形縣的所在地。和許多舊城下町利用城郭土地興建政府設施相對，山形縣將縣廳等政府機構和公共設施修建在新市區，舊城下町的格局因此並未發生大幅改變<ref name="Chishi"></ref>{{rp|383}}。1889年，山形正式設[[市制_(日本制度)|市]]，成為日本首批「市」之一<ref name="Hyakunen">{{cite book |author={{lang|ja|平岡昭利}}|script-title=ja:《東北 地図で読む百年》 |year=2000 |publisher={{lang|ja|古今書院}} |location=東京都 |isbn=4-7722-5052-2|language=ja}}</ref>{{rp|107}}。1894年，山形市發生大火，造成超過1,000戶住宅被燒毀<ref name="Chishi"></ref>{{rp|383}}。1901年，隨著[[奧羽本線|奧羽本線]]通車，山形市進入鐵路時代，山形站設在山形城附近。加上這一時期日本陸軍陸軍步兵三十二連隊入駐山形城，使得山形城附近地區得到開發，發展為山形市的交通中心<ref name="Chishi"></ref>{{rp|384}}。1911年，山形市再次發生大火，市役所和縣廳等政府機構在這次大火中被燒毀<ref name="Chishi"></ref>{{rp|383}}。1915年，重建的山形縣廳舍竣工，即現在的[[文翔館|文翔館]]。1929年，山形市開始實施土地區劃整理事業，是較早實施土地區劃整理事業的城市，建成區加速擴大。山形市轄區範圍亦在1931年和1943年兩次擴大。在[[第二次世界大戰|第二次世界大戰]]中，山形市是日本少數未遭到空襲的縣廳所在地城市<ref>{{cite web |url=http://crd.ndl.go.jp/reference/modules/d3ndlcrdentry/index.php?page=ref_view&id=1000188488 |script-title=ja:第2次世界大戦時の山形県の空襲被害について知りたい。 |accessdate=2017-07-05|publisher={{lang|ja|国立国会図書館}} |language=ja}}</ref>。

二戰之後，山形市在1954年進行了大規模合併，市轄區面積擴大了近20倍。1960年代土地區劃整理事業加速實施之後，更加快了山形市市區擴張的速度<ref name="Chishi"></ref>{{rp|384}}。1968年，山形市制定了山形市都市開發基本計劃<ref name="Chishi"></ref>{{rp|385}}。1975年，山形縣廳自市中心搬遷至市區東部之後更加快了山形市近郊地區的開發<ref name="Chishi"></ref>{{rp|386}}。1992年，[[山形新幹線|山形新幹線]]開業，使得山形市和[[東京|東京]]之間的陸路交通有所改善。山形市原本以南北方向的街道為主。但在1990年代之後，山形市在山形站附近地區進行大規模開發（代表工程是{{link-ja|霞城Cenral|霞城セントラル}}）<ref>{{cite web |url=http://kajocentral.com/outline/outline.html |script-title=ja:霞城セントラル概要 |accessdate=2017-07-05|publisher={{lang|ja|霞城セントラル}} |language=ja}}</ref>，並興建東西方向的道路，以促進市區活性化<ref name="Hyakunen"></ref>{{rp|114}}。2001年，山形市成為[[施行時特例市|特例市]]，並將在2019年成為[[中核市|中核市]]<ref>{{cite web |url=https://www.city.yamagata-yamagata.lg.jp/kakuka/somu/chukakushi/sogo/b0n9j4.pdf |script-title=ja:中核市 |accessdate=2017-07-05|publisher={{lang|ja|山形市}} |language=ja}}</ref>。

== 地理 ==
[[File:Yamagata-zao_ski_1_(200712).jpg|thumb]]
山形市位於{{link-ja|山形盆地|山形盆地}}南部，市中心地區則位於{{link-ja|馬見崎川|馬見ヶ崎川}}沖積形成的[[沖積扇|沖積扇]]上。馬見崎川在歷史上多次發生洪水，給當地民眾造成很大危害。自江戶時代開始，馬見崎川的治水就成為當地的一大議題。在江戶時代，雖歷代藩主均曾治理馬見崎川，但因藩主頻繁變更，導致洪水仍然頻發<ref name="Hyakunen"></ref>{{rp|112}}。1890年，山形市政府修建大堤，其費用是當時山形市年財政收入的約4倍<ref name="Hyakunen"></ref>{{rp|112}}。1913年洪水之後，山形市對馬見崎川河床進行疏浚，並興建二重堤壩，此後馬見崎川不再氾濫<ref name="Hyakunen"></ref>{{rp|112}}。山形市東部的[[藏王連峰|藏王連峰]]屬於[[奥羽山脈|奥羽山脈]]，也是山形縣和[[宮城縣|宮城縣]]的縣界。藏王連峰也是日本少數可以觀測到[[霧凇|霧凇]]景觀的地區<ref>{{cite web |url=http://zaoropeway.co.jp/zao/winter/ |script-title=ja:樹氷 |accessdate=2017-07-05|publisher={{lang|ja|蔵王ロープウェイ}} |language=ja}}</ref>。山形市轄區西南部的白鷹丘陵是[[村山地方|村山地方]]和[[置賜地方|置賜地方]]的分界線<ref>{{cite web |url=http://www.thr.mlit.go.jp/yamagata/river/enc/genre/01-kiso/kiso0102_003.html |script-title=ja:白鷹丘陵と五百川峡谷 |accessdate=2017-07-05|publisher={{lang|ja|東北地方整備局}} |language=ja}}</ref>。山形市除了馬見崎川之外還有{{link-ja|須川|須川}}等河流，均是[[最上川|最上川]]的支流。

=== 氣候 ===
由於山形市位於盆地之中，因此呈現內陸性氣候的特徵，冬夏溫差較為明顯。山形市在夏季是東北地方最為炎熱的地區之一，常有最高氣溫超過35℃的情況出現。但由於一日之內氣溫變化明顯，因此夜間和早晨氣溫較為涼爽。而冬季的山形市氣候寒冷，亦不乏降雪天氣出現，但由於{{link-ja|出羽山地|出羽山地}}遮擋住了在[[日本海|日本海]]發生的雪雲，使得山形市是山形縣內降雪量較少的地區。山形市的氣候觀測點被[[日本氣象廳|日本氣象廳]]選為日本平均氣溫和降水量的觀測地點之一<ref>{{cite web |url=http://www.data.jma.go.jp/cpdinfo/temp/qa_temp.html |script-title=ja:世界と日本の気温、降水量の経年変化に関して、よくある質問 |accessdate=2017-07-05|publisher={{lang|ja|気象庁}} |language=ja}}</ref>。1933年7月25日，山形測候所記錄下40.8℃的最高氣溫紀錄。直到2007年[[埼玉縣|埼玉縣]][[熊谷市|熊谷市]]和[[岐阜縣|岐阜縣]][[多治見市|多治見市]]記錄下40.9℃的氣溫為止，該紀錄是日本最高氣溫紀錄<ref>{{cite web |url=http://kscalar.kj.yamagata-u.ac.jp/museum/DisplayRoom/yamagata/yamagata3.html |script-title=ja:日本最高気温の記録40.8度 |accessdate=2017-07-05|publisher={{lang|ja|山形大学}} |language=ja}}</ref>。

== 人口 ==
1889年設市時，山形市有人口2.48萬人<ref name="Hyakunen"></ref>{{rp|107}}。1897年時，山形市人口超過[[米澤市|米澤市]]，成為山形縣內人口最多的城市<ref name="Chishi"></ref>{{rp|383}}。1920年日本首次人口普查時，山形市有人口48,399人<ref>{{cite web |url=http://www.stat.go.jp/data/chouki/zuhyou/02-09.xls |script-title=ja:都市別人口|accessdate=2017-07-05|publisher={{lang|ja|総務省統計局}} |language=ja}}</ref>。此後隨著都市發展和轄區範圍，山形市人口持續增加，在1950年人口普查時超過10萬人大關。1970年人口普查時，山形市人口突破20萬大關<ref name="population">{{cite web |url=https://www.city.yamagata-yamagata.lg.jp/kakuka/kikaku/kikaku/sogo/gazoufile/hattenkeikaku/jinkovision.pdf |script-title=ja:山形市人口ビジョン |accessdate=2017-07-05|publisher={{lang|ja|気象庁}} |language=ja}}</ref>。1980年代之後，由於人口高齡化、少子化，山形市人口增長趨緩。2005年時，山形市人口達到峰值256,012人，此後開始減少<ref name="population"></ref>。2017年6月1日，山形市有人口252,317人<ref>{{cite web |url=https://www.city.yamagata-yamagata.lg.jp/kakuka/kikaku/kikaku/sogo/gazoufile/toukei/suikei/pf0104142435/lc788h.pdf |script-title=ja:推計人口 |accessdate=2017-07-05|publisher={{lang|ja|山形市}} |language=ja}}</ref>。2012年，山形市的[[總和生育率|總和生育率]]有1.38，低於日本平均值（1.41）和山形縣平均值（1.44）<ref name="population"></ref>。

== 政治 ==
山形市和[[上山市|上山市]]、[[天童市|天童市]]、[[山邊町|山邊町]]、[[中山町|中山町]]在日本[[眾議院_(日本)|眾議院]]選舉中屬於{{link-ja|山形縣第1區|山形県第1区}}<ref>{{cite web |url=http://www.soumu.go.jp/main_content/000240044.pdf |script-title=ja:山形県 |accessdate=2017-07-05 |publisher={{lang|ja|総務省}} |language=ja}}</ref>。山形1區是[[自由民主黨_(日本)|自民黨]]較強的選區，自2003年開始，除了2009年[[第45屆日本眾議院議員總選舉|第45屆日本眾議院議員總選舉]]之外，歷次選舉都是自民黨的[[遠藤利明|遠藤利明]]獲勝<ref>{{cite web |url=https://www.city.yamagata-yamagata.lg.jp/shiseijoho/sub16/senkyo/9c8dcsenkyokekka.html |script-title=ja:選挙の結果（オープンデータ）|accessdate=2017-07-05 |publisher={{lang|ja|山形市}} |language=ja}}</ref>。山形市現任市長是{{link-ja|佐藤孝弘|佐藤孝弘}}，他在2011年就曾參加市長選舉但未當選，在2015年首次當選<ref>{{cite web |url=https://www.city.yamagata-yamagata.lg.jp/shicho/shicho.html |script-title=ja:市長プロフィール |accessdate=2017-07-05 |publisher={{lang|ja|山形市}} |language=ja}}</ref>。山形市議會現在共有33名議員，其中翔政會是最大會派，有12名議員<ref>{{cite web |url=http://www.city.yamagata-yamagata.lg.jp/gikai/giinshokai/kaiha.html |script-title=ja:会派 |accessdate=2017-07-05 |publisher={{lang|ja|山形市}} |language=ja}}</ref>。{{link-ja|山形縣議會|山形県議会}}中，山形市共有9名議員<ref>{{cite web |url=http://www.pref.yamagata.jp/assembly/introduction/7600006senkyokubetu.html |script-title=ja:議員紹介選挙区別、五十音順 |accessdate=2017-07-05 |publisher={{lang|ja|山形県}} |language=ja}}</ref>。

== 經濟 ==
[[File:Kajo_Central_View_of_top_Original.jpg|thumb]]
2014年度，山形市市內生產總值有9,166億日元，佔山形縣縣內生產總值的約四分之一。人均市民所得有300.5萬日元<ref>{{cite web |url=http://www.pref.yamagata.jp/ou/kikakushinko/020052/data/shik/02_gaiyoban_h26.pdf |script-title=ja:市町村民経済計算 |accessdate=2017-07-05 |publisher={{lang|ja|山形県}} |language=ja}}</ref>。山形市耕地面積有5,200公頃，農業就業人口有3,634人<ref name="agriculture">{{cite web |url=http://www.machimura.maff.go.jp/machi/contents/06/201/index.html |script-title=ja:山形県山形市 |accessdate=2017-07-05 |publisher={{lang|ja|農林水産省}} |language=ja}}</ref>。2015年，山形市的農業生產額有113.7億日元，其中以果實類的比例最多，其次是稻米類和蔬菜類<ref name="agriculture"></ref>。山形縣有「果樹王國」之稱，山形市的主要水果產品有[[櫻桃|櫻桃]]、[[法蘭西梨|法蘭西梨]]、[[葡萄|葡萄]]等<ref>{{cite web |url=http://www.y-noushinkyo.jp/modules/noushinkyo5/index.php?id=1 |script-title=ja:山形市の特産農産物 |accessdate=2017-07-05 |publisher={{lang|ja|山形市}} |language=ja}}</ref>。山形市的主要蔬菜產品有食用[[菊花|菊花]]、[[菠菜|菠菜]]、[[南瓜|南瓜]]等<ref>{{cite web |url=http://www.y-noushinkyo.jp/modules/noushinkyo7/index.php?id=1 |script-title=ja:山形市の伝統野菜 |accessdate=2017-07-05 |publisher={{lang|ja|山形市}} |language=ja}}</ref>。

山形市自平安時代後期開始就生產{{link-ja|山形鋳物|山形鋳物}}等工藝品<ref>{{cite web |url=http://www.pref.yamagata.jp/ou/shokokanko/110010/kogeihin/sp01-1.html |script-title=ja:特集記事 |accessdate=2017-07-05 |publisher={{lang|ja|山形県}} |language=ja}}</ref>，製造業歷史悠久。2014年度，山形市製造品生產額達2043.43億日元，佔山形縣總量的7.8%<ref>{{cite web |url=http://www.pref.yamagata.jp/ou/kikakushinko/020052/data/kogy/h26kakuho_all.pdf |script-title=ja:平成26年　山形県の工業 |accessdate=2017-07-05 |publisher={{lang|ja|山形県}} |language=ja}}</ref>。山形市的製造業企業數和從業者數雖然位居縣內前列，但製造品生產僅位居第五位（僅次於[[米澤市|米澤市]]、[[東根市|東根市]]、[[酒田市|酒田市]]、[[鶴岡市|鶴岡市]]）<ref>{{cite web |url=http://www.meti.go.jp/statistics/toppage/topics/kids/doc/06_yamagata.pdf |script-title=ja:情報通信機械器具製造業、電子部品・デバイス・電子回路製造業等が盛んな山形県 |accessdate=2017-07-05 |publisher={{lang|ja|経済産業省}} |language=ja}}</ref>。總部位於山形市的銀行有{{link-ja|山形銀行|山形銀行}}<ref>{{cite web |url=http://www.yamagatabank.co.jp/company/gaiyou/enkaku/ |script-title=ja:会社沿革 |accessdate=2017-07-05 |publisher={{lang|ja|山形銀行}} |language=ja}}</ref>和{{link-ja|輝煌銀行|きらやか銀行}}<ref>{{cite web |url=https://www.kirayaka.co.jp/aboutus/outline/company.php |script-title=ja:会社概要 |accessdate=2017-07-05 |publisher={{lang|ja|きらやか銀行}} |language=ja}}</ref>。山形市在城下町蕭條之後憑藉紅花貿易而得以維持城市規模，是山形縣的商業中心<ref>{{cite web |url=http://sisyouren.main.jp/%E5%B1%B1%E5%BD%A2%E5%B8%82%E3%81%AE%E5%95%86%E6%A5%AD |script-title=ja:山形市の商業 |accessdate=2017-07-05 |publisher={{lang|ja|山形市商店街連合会}} |language=ja}}</ref>。山形市集中了山形縣約半數的批發業銷售額和約四分之一的零售業銷售額<ref>{{cite web |url=http://www.meti.go.jp/statistics/tyo/syougyo/dms/2009/pdf/ken06.pdf |script-title=ja:山形県の卸売年間商品販売額の５割が山形市に集中 |accessdate=2017-07-05 |publisher={{lang|ja|経済産業省}} |language=ja}}</ref>。但另一方面，山形市中心地區的商業面臨市民外流郊外大型購物中心和仙台市購物的局面，出現嚴重的空洞化問題<ref name="intro"></ref>。

== 文化教育 ==
[[File:Hanagasa_Festa_2002.jpg|210px]]
山形市使用的方言是{{link-ja|村山方言|村山弁}}，屬於{{link-ja|東北方言 (日語)|東北方言|東北方言}}的{{link-ja|南奧羽方言|南奥羽方言}}，是典型的「{{lang|ja|ズーズー弁}}」（「{{lang|ja|じ}}」、「{{lang|ja|ず}}」、「{{lang|ja|ぢ}}」、「{{lang|ja|づ}}」不分）<ref>{{cite book |author={{lang|ja|真田信治}}|script-title=ja:《方言学》 |year=2011 |publisher={{lang|ja|朝倉書店}} |location=東京都 |isbn=978-4-254-51524-4|language=ja}}</ref>{{rp|15-21}}。每年8月5日舉辦的{{link-ja|花笠祭|花笠まつり|山形花笠祭}}是山形市最大的節慶活動，於1963年開始舉行<ref>{{cite web |url=http://www.mountain-j.com/hanagasa/html/history.html |script-title=ja:山形花笠まつりの由来 |accessdate=2017-07-05 |publisher={{lang|ja|山形花笠まつり}} |language=ja}}</ref>。現在山形花笠祭已成為和[[仙台七夕|仙台七夕]]、[[青森睡魔祭|青森睡魔祭]]、[[竿燈|秋田竿燈]]並列的東北四大祭之一<ref>{{cite web |url=http://www.mountain-j.com/hanagasa/html/link.html |script-title=ja:東北四大まつり |accessdate=2017-07-05 |publisher={{lang|ja|山形花笠まつり}} |language=ja}}</ref>。[[芋煮會|芋煮]]是山形市最著名的鄉土料理之一。山形市每年均舉辦日本規模最大的芋煮會，使用的大鍋直徑達6米<ref>{{cite web |url=http://www.y-yeg.jp/imoni/about.php |script-title=ja:日本一の芋煮とは |accessdate=2017-07-05 |publisher={{lang|ja|日本一の芋煮会フェスティバル協議会事務局}} |language=ja}}</ref>。

{{link-ja|山形新聞|山形新聞}}總部位於山形市，是山形縣的地方報紙，日發行量約20萬份<ref>{{cite web |url=http://www.yamashinhanbai.jp/orikomi/busuu_top.html |script-title=ja:山形県折込広告枚数表 |accessdate=2017-07-05 |publisher={{lang|ja|山形新聞}} |language=ja}}</ref>。山形市內的電視台有[[NHK山形放送局|NHK山形放送局]]、[[NNN|NNN]]系列的[[山形放送|山形放送]]<ref>{{cite web |url=http://ybc.co.jp/copo/outline/ |script-title=ja:会社概要 |accessdate=2017-07-05 |publisher={{lang|ja|山形放送}} |language=ja}}</ref>、[[全日本新聞網|ANN]]的[[山形電視台|山形電視台]]<ref>{{cite web |url=http://www.yts.co.jp/contents/company/cat22/ |script-title=ja:企業情報 |accessdate=2017-07-05 |publisher={{lang|ja|山形テレビ}} |language=ja}}</ref>、[[JNN|JNN]]的[[TVU山形|TVU山形]]<ref>{{cite web |url=http://www.tuy.co.jp/contents/company/ |script-title=ja:会社案内 |accessdate=2017-07-05 |publisher={{lang|ja|テレビユー山形}} |language=ja}}</ref>、[[FNN|FNN]]的[[櫻桃電視台|櫻桃電視台]]<ref>{{cite web |url=http://www.sakuranbo.co.jp/info/company/ |script-title=ja:会社案内 |accessdate=2017-07-05 |publisher={{lang|ja|さくらんぼテレビジョン}} |language=ja}}</ref>。以山形市為主場的職業體育俱樂部有參加[[日本職業足球乙級聯賽|日本職業足球乙級聯賽]]的[[山形山神|山形山神]]<ref>{{cite web |url=http://www.montedioyamagata.jp/club/footstep.html |script-title=ja:モンテディオ山形の歩み |accessdate=2017-07-05 |publisher={{lang|ja|モンテディオ山形とは}} |language=ja}}</ref>。

[[山形大學|山形大學]]設於1949年，是山形縣唯一的國立大學，其中人文社會科學院、地域教育文化學院、理學院所在的小白川校區及醫學院所在的飯田校區位於山形市<ref>{{cite web |url=https://www.yamagata-u.ac.jp/jp/university/introduction/outline/ |script-title=ja:大学概要 |accessdate=2017-07-05 |publisher={{lang|ja|山形大学}} |language=ja}}</ref>。山形市還有[[山形縣立保健醫療大學|山形縣立保健醫療大學]]一所公立大學<ref>{{cite web |url=http://www.yachts.ac.jp/off/daigakugaiyou/enkaku.html |script-title=ja:沿革 |accessdate=2017-07-05 |publisher={{lang|ja|山形県立保健医療大学}} |language=ja}}</ref>，{{link-ja|東北藝術工科大學|東北芸術工科大学}}、{{link-ja|東北文教大學|東北文教大学}}兩所私立大學。

== 交通 ==
[[File:E3-2000_Yamagata_20161009.jpg|thumb]]
山形市內的鐵路由[[東日本旅客鐵道|東日本旅客鐵道]]運營。市內有[[奧羽本線|奧羽本線]]中[[福島車站_(福島縣)|福島站]]至[[新莊站|新莊站]]部分的[[山形線|山形線]]、連接山形站和左澤站的[[左澤線|左澤線]]、連接山形市羽前千歲站和仙台市[[仙台站|仙台站]]的[[仙山線|仙山線]]，合共三條[[在來線|在來線]]；以及[[山形新幹線|山形新幹線]]一條[[迷你新幹線|迷你新幹線]]<ref name="traffic">{{cite web |url=https://www.city.yamagata-yamagata.lg.jp/kakuka/kikaku/kikaku/sogo/gazoufile/pf0314133116/moukeiseikeikaku-3.pdf |script-title=ja:交通の現状整理 |accessdate=2017-07-05 |publisher={{lang|ja|山形市}} |language=ja}}</ref>。2010年度，山形市內鐵路交通乘客數達508萬人次<ref>{{cite web |url=https://www.city.yamagata-yamagata.lg.jp/kakuka/kikaku/kikaku/sogo/gazoufile/pf1124105222/pf1124105314/165sfq.pdf |script-title=ja:本市の公共交通を取り巻く環境 |accessdate=2017-07-05 |publisher={{lang|ja|山形市}} |language=ja}}</ref>。[[山形站|山形站]]是山形市鐵路交通的中心，2015年度日均客流量有1.07萬人次<ref>{{cite web |url=https://www.jreast.co.jp/passenger/2015_02.html |script-title=ja:各駅の乗車人員 |accessdate=2017-07-05 |publisher={{lang|ja|JR東日本}} |language=ja}}</ref>，佔山形市鐵路總客流量約八成<ref name="traffic"></ref>。

途徑山形市轄區的高速公路有[[山形自動車道|山形自動車道]]和[[東北中央自動車道|東北中央自動車道]]<ref>{{cite web |url=http://www.jaf.or.jp/member/dguide/exway/pdf/tohoku/Tohoku_ex_or.pdf |script-title=ja:高速道路案内図 |accessdate=2017-07-05 |publisher={{lang|ja|日本自動車連盟}} |language=ja |deadurl=yes |archiveurl=https://web.archive.org/web/20150501051529/http://www.jaf.or.jp/member/dguide/exway/pdf/tohoku/Tohoku_ex_or.pdf |archivedate=2015-05-01 }}</ref>。山形市內的巴士線路<ref>{{cite web |url=http://www.yamakobus.co.jp/rosenbus/rosenzu/yamagatasi.htm |script-title=ja:山形市中心部 |accessdate=2017-07-05 |publisher={{lang|ja|山交バス}} |language=ja}}</ref>及前往[[本州|本州]]各主要城市之間的高速巴士<ref>{{cite web |url=http://www.yamakobus.co.jp/kousoku/jikokuhyo.html |script-title=ja:高速・都市間バス時刻表 |accessdate=2017-07-05 |publisher={{lang|ja|山交バス}} |language=ja}}</ref>大多由{{link-ja|山交巴士 (山形縣)|山交バス (山形県)|山交巴士}}運營。{{link-ja|仙台-山形線|仙台 - 山形線|仙台和山形之間的高速巴士}}行駛時間只需約1小時且日車次數約80次，使得仙台和山形之間的交通十分便利，但這也是導致山形市民外流仙台購物的重要原因之一<ref name="intro"></ref>。山形商工會議所在2011年開通了100日元環形巴士，以促進山形市中心地區活性化<ref name="traffic"></ref>。山形市轄區內並沒有機場，距山形市最近的機場是位於東根市的[[山形機場|山形機場]]。

== 觀光 ==

山形市中心地區內主要觀光景點有[[日本100名城|日本100名城]]之一的[[山形城|山形城]]<ref>{{cite web |url=http://www.oshirobu.com/japan100/010.html |script-title=ja:山形城 |accessdate=2017-07-05 |publisher={{lang|ja|お城部.com}} |language=ja}}</ref>、舊山形縣廳[[文翔館|文翔館]]<ref>{{cite web |url=http://www.gakushubunka.jp/bunsyokan/facilities/floor02.html |script-title=ja:館内の紹介 |accessdate=2017-07-05 |publisher={{lang|ja|文翔館}} |language=ja}}</ref>等地。位於山形市郊外的[[立石寺|立石寺]]又名山寺，已有超過1,150年的歷史。寺院的大多數建在山上，風景獨特，[[松尾芭蕉|松尾芭蕉]]亦曾到訪並留下俳句<ref>{{cite web |url=http://www.rissyakuji.jp/about.html |script-title=ja:立石寺について |accessdate=2017-07-05 |publisher={{lang|ja|宝珠山 立石寺}} |language=ja}}</ref>。立石寺的根本中堂是日本最古老的現存橅木建築，也是[[重要文化財|重要文化財]]，具高度文物價值<ref>{{cite web |url=http://www.yamaderakankou.com/origin/ |script-title=ja:山寺の由来 |accessdate=2017-07-05 |publisher={{lang|ja|山寺観光協会}} |language=ja}}</ref>。

位於山形、宮城兩縣交界處的藏王連峰屬於[[藏王國定公園|藏王國定公園]]，以火山地形著稱<ref>{{cite web |url=http://www.env.go.jp/council/12nature/y121-20/mat06_1.pdf |script-title=ja:蔵王国定公園の公園区域及び公園計画の変更案の概要 |accessdate=2017-07-05 |publisher={{lang|ja|環境省}} |language=ja}}</ref>。{{link-ja|山形藏王溫泉滑雪場|山形蔵王温泉スキー場}}是日本面積最大的滑雪場<ref>{{cite web |url=http://www.ski-ichiba.jp/navi/article/270/ |script-title=ja:人気スキー場5選！人気な理由を調べてみました。 |accessdate=2017-07-05 |publisher={{lang|ja|スキー市場情報局}} |language=ja}}</ref>，擁有多條面向不同遊客的滑道<ref>{{cite web |url=http://www.zao-ski.or.jp/gerende/ |script-title=ja:ゲレンデ案内 |accessdate=2017-07-05 |publisher={{lang|ja|蔵王索道協会}} |language=ja}}</ref>。[[藏王溫泉|藏王溫泉]]的泉水呈乳白色，是強酸性的硫磺泉，具有促進血液循環和美膚的效果，是東北地方最大的溫泉度假地之一<ref>{{cite web |url=http://www.zao-spa.or.jp/about/ |script-title=ja:蔵王温泉について |accessdate=2017-07-05 |publisher={{lang|ja|蔵王温泉観光協会}} |language=ja}}</ref>。
