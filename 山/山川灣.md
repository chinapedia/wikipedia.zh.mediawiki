'''山川灣'''（{{Jpn|j=山川湾|hg=やまがわわん|rm=Yamagawa-wan}}）是[[日本|日本]][[鹿兒島縣|鹿兒島縣]][[指宿市|指宿市]]的一個[[海灣|海灣]]。位於[[薩摩半島|薩摩半島]]南端，[[鹿兒島灣|鹿兒島灣]]沿岸，因[[火山活動|火山活動]]而形成。縱深約2公里，寬約700-{zh-hans:米; zh-hant:公尺;}-。最深處在海灣彎曲處，深約50-{zh-hans:米; zh-hant:公尺;}-。

灣內由[[沙咀|沙咀]]劃分為內灣和外灣。內灣海況平穩，建有[[山川港|山川港]]，是停泊大型船隻的天然良港<ref>{{Cite web |url = http://youkoso-ibusuki.com/marugoto/spot/%E5%B1%B1%E5%B7%9D%E6%B8%AF/ |title = 山川港 |author = 指宿市考古博物館 時遊館 Cocco はしむれ |language = ja |accessdate = 2014-04-29 }}</ref>。

== 概況 ==

山川灣是經[[薩摩半島|薩摩半島]]東南部[[阿多火山臼|阿多火山臼]]系列[[火山|火山]]活動而形成的一個[[火山口|火山口]]遺跡。約5500年前，也就是在[[池田湖|池田湖]]、[[鰻池|鰻池]]形成的同一時期，[[鹿兒島灣|鹿兒島灣]]一側[[海岸|海岸]]附近發生了[[岩漿|岩漿]][[水蒸汽爆炸|水蒸汽爆炸]]，形成了一個直徑約2公里的[[窪地|窪地]]。窪地東側火山壁[[山崩|山崩]]後，[[海水|海水]]湧入形成了半[[圓|圓]]形的海灣。後來南側附近的[[沙咀|沙咀]]再向北伸，便形成了目前的[[地形|地形]]。爆炸產生的大量噴發物遺留在周邊[[地層|地層]]，有的厚達30-{zh-hans:米; zh-hant:公尺;}-<ref>{{Cite web |url = http://yamagawa-minato.com/midokoro.html/ |title = 山川港って？ |author = いっど、いっが、山川港の会 |language = ja |accessdate = 2014-04-29 |deadurl = yes |archiveurl = https://web.archive.org/web/20140429230223/http://yamagawa-minato.com/midokoro.html |archivedate = 2014-04-29 }}</ref>。

山川灣地質活動頻繁，與成川盆地、[[池田湖|池田湖]]、[[鰻池|鰻池]]、池底和松窪等活火山群一道被日本[[氣象廳|氣象廳]]指定為C級[[活火山|活火山]]地帶，受到嚴密監視<ref>{{Cite web |url = http://www.data.jma.go.jp/svd/vois/data/fukuoka/518_Ikeda_Yamagawa/518_index.html |title = 池田山川 |author = 日本気象庁 |language = ja |accessdate = 2014-04-29 }}</ref>。

海灣周圍是高約80-{zh-hans:米; zh-hant:公尺;}-的火山壁型山崖。海岸地帶開發有川尻溫泉。
