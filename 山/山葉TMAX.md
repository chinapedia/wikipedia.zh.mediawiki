'''Yamaha TMAX'''為[[日本|日本]][[山葉發動機|山葉發動機]]公司製造販賣的一款[[摩托車|摩托車]]，屬於[[速克達|速克達]]的一種，定位為道路運動型摩托車。在[[臺灣|臺灣]]牌照領為[[大型重型機車|大型重型機車]]。
因其名稱發音，因此被暱稱為'''T（踢）媽'''、'''T（踢）媽媽'''。

== 概要 ==
2000年秋天，向歐洲國家輸出2001年款的日本國外式樣。而日本國內式樣於2001年8月開始銷售。

== TMAX 500（2001～2011） ==
===第1代(2001-2003)===
搭載引擎為排氣量499[[立方厘米|cc]]的[[水冷引擎|水冷]][[四冲程|4行程]][[DOHC|DOHC]]4[[氣門|氣門]]直列雙[[汽缸|汽缸]]，採用[[鍛造|鍛造]][[活塞|活塞]]及缸徑66mm、活塞行程73mm長行程式的[[汽缸本體|陶瓷汽缸]]。

傳動箱位於引擎右方，使用[[無段自動變速器|CVT]][[自动变速器|自排變速系統]]，採用兩段[[鏈條|鏈條]]減速比傳遞至後輪。<ref>[http://www.moto7.net/2012/07/t-max.html 新、舊T-Max 比一比：新車到底好在哪？]MOTO7 專業摩托車資訊。</ref>

為了達到低重心化，前輪特別加重，以及採用水平並列汽缸、[[機油|機油]]潤滑方式的[[乾式油底殼|乾式油底殼]]。
===第2代(2004-2007)===

===第3代(2008-2011)===
經過了6年的淬鍊，第二代TMAX採用了後掠式頭燈，讓視覺感更有侵略性，後方的扶手也從前代的一體式改為分離式設計，排氣管伴隨著車尾的角度跟著上揚，從渾圓流線的風格轉為銳利紋理分明的風格。在車殼底下看不到的地方則是已經脫胎換骨，採用全新開發的CF壓鑄鋁合金車架，比之前的鑽石型鋼管車架減少5公斤；前端換上43mm正叉、15吋前輪框，讓軸距微幅增加了5mm，煞車系統升級對向四活塞卡鉗，加上重心非常低的引擎擺放位置，給人既穩定又輕快的感受，引擎部分改變了集氣箱的位置，壓縮比再次提高到11.0：1，提升各轉速的出力表現。

== TMAX 530（2012～） ==
===第4代(2012-2014)===
2012年版，將引擎排氣量小幅度提升為530cc，型號為TMAX 530<ref>[http://www.yamaha-motor.com.tw/motor/TMAX.aspx TMAX 530產品介紹] {{webarchive|url=https://web.archive.org/web/20130302223634/http://www.yamaha-motor.com.tw/motor/TMAX.aspx |date=2013-03-02 }}台灣山葉。</ref>。仍使用[[水冷引擎|水冷]][[四冲程|4行程]][[DOHC|DOHC]]4[[氣門|氣門]]直列雙[[陶瓷|陶瓷]][[汽缸|汽缸]]。

TMAX 530傳動裝置，使用[[無段自動變速器|CVT]][[自动变速器|自排變速系統]]及一段式的皮帶輪減速，為高效能V型皮帶傳動。採用無傳動箱設計，改採鋁合金後搖臂的配置，可提升操控性能。

並且改變配重、減輕重量。
===第5代(2015-2016)===
由43mm正叉改為41mm倒叉，輻射對四卡鉗。ABS防煞車鎖死系統（台灣版本可選要ABS或不要），大燈改為LED及日行燈，取消左側置物欄並改成DC12V車充。

鑰匙變更為Keyless感應式，免匙發車

===第6代(2017-)===
主要變革為外型以及加入了電控系統，外觀上更改為類似第一代的圓潤造型，尾燈改為左右分離，獨立式左右方向燈，油門由傳統的鋼索式改為YCC-T電子油門，並加入TCS循跡系統，油門可選擇Town模式和Sport模式。
車身部分，後搖臂改為多連桿懸吊並增加40mm
