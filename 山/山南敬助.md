'''山南敬助'''（{{bd|1833年||1865年||}}），全名'''山南敬助知信'''，出身於[[陸奧|陸奧]][[仙台藩|仙台藩]]，文武方面都相當優秀，個性也非常溫厚，常被人親切地稱做「山南先生」。在[[江戶|江戶]]的千葉道場學習[[北辰一刀流|北辰一刀流]]之後出師。並在敗給[[近藤勇|近藤勇]]後而加入其門下。1863年，以試衛館一門的身分，與近藤勇、[[土方歲三|土方歲三]]一同加入壬生浪士組。歷任[[新選組|新選組]]副長、總長，一度受傷後就經常臥病在床，之後脫離新選組遭到[[沖田總司|沖田總司]]逮捕，並以[[切腹|切腹]]方式而死。

== 出身 ==
從[[仙台|仙台]]藩脱藩後，前往[[江戶|江戶]]，山南在仙台也用三南這個姓，但詳情不明。曾獲得[[小野派一刀流|小野派一刀流]]免許皆傳的身分，之後成為[[北辰一刀流|北辰一刀流]]的[[千葉周作|千葉周作]]門人。

== 試衛館 ==
前往[[近藤勇|近藤勇]]的[[天然理心流|天然理心流]]劍術道場[[試衛館|試衛館]]挑戰，但敗給近藤。這時對於近藤的劍術和人品感到欽佩，進而與試衛館的門人一起行動。並且與這些聚集在試衛館的土方歳三、沖田總司、[[永倉新八|永倉新八]]等都成為之後新選組的幹部。

文久元年（1860年）8月，在[[府中六所宮|府中六所宮]]進行近藤就任天然理心流第四代傳人的野外比賽，山南也以紅軍身分參與比賽。文久2年（1861年）正月時、曾與[[沖田總司|沖田總司]]一起出差到小野路教授劍術。

== 新選組 ==
文久3年（1862年）2月，山南與近藤加入了以保護[[幕府將軍|將軍]]和[[尊王攘夷|尊王攘夷]]為目的，由[[清河八郎|清河八郎]]提案所組成的[[浪士組|浪士組]]，並且成為前往[[京都|京都]]的壬生浪士組中的一員。3月26日，為了爭奪隊内的權力，他與近藤、[[芹澤鴨|芹澤鴨]]一起在四條大橋暗殺[[殿内義雄|殿内義雄]]。壬生浪士組也從此成為近藤派、芹澤派兩派所組成之組織，近藤和芹澤擔任局長、山南作為近藤派與[[土方歲三|土方歲三]]一起就任局長助（副長）（至於芹澤派則是由[[新見錦|新見錦]]就任局長助）。

以[[尊王攘夷|尊王攘夷]]為目的所組成的浪士組，被當時負責維持京都治安的[[會津藩|會津藩]]收編，成為專門取締不法[[浪人|浪人]]的組織。文久3年（1862年）7月，山南和土方一起在和服商「岩城升屋」的激烈戰鬥中，擊退數名闖入的不法浪人。之後8月18日發生「八一八[[禁門之變|禁門之變]]」，山南在事發2天後與土方一起斬殺掉潛伏於京都的[[长州藩|長州]]系浪人。

文久3年（1863年）的9月，山南參與老早就和近藤不合的[[芹澤鴨|芹澤鴨]]暗殺行動。此後，浪士組被近藤派（試衛館派）所統一，組織重新改造，並由山南擔任總長，他的地位則處於局長近藤、副長[[土方歲三|土方歲三]]之間，但實際上並無實權。從[[暗殺|暗殺]]事件到後來的[[切腹|切腹]]事件，山南的名字完全地從新選組的活動紀錄中消失。[[元治|元治]]元年（1864年）6月發生著名的[[池田屋事件|池田屋事件]]，但山南也沒有參加該次行動。（有一說是山南因為生病，而在基地留守。）

元治元年（1864年）11月，[[伊東甲子太郎|伊東甲子太郎]]入隊。伊東和山南都屬於同門的北辰一刀流，而且伊東是熱烈的[[尊王攘夷|尊王攘夷]]論者、学問也相當高。所以他因此受到特別的待遇，升上比山南地位還高的[[參謀|參謀]]位置。

== 脱逃 ==
加入新選組後的第4年，元治2年（1865年）2月時，山南突然留下「前往江戶」的紙條，並且就這樣脫離新選組。由於新選組的隊規（[[局內法度|局內法度]]）嚴格規定只要擅自脫隊，一律是[[切腹|切腹]]死罪。所以近藤、土方立刻派遣[[沖田總司|沖田總司]]擔任追捕的工作，並且在[[近江国|近江国]][[大津市|大津]]將山南逮捕，並將他帶回基地。據說山南看到前來追捕他的人是與他交情深厚的沖田，主動露面向沖田打招呼。沖田試探性問他：「你認為自己能逃得走嗎？」山南則回答：「若追兵是你，我就無法（刀劍相向）。」於是毫無抵抗地隨沖田返回。

關於山南逃走的原因，有以下各個說法。

*根據[[西本願寺|西本願寺]]侍臣[[西村兼文|西村兼文]]的說法，逼迫山南逃走的原因是[[基地|基地]]轉移的問題。由於[[新選組|新選組]]把基地移轉到[[西本願寺|西本願寺]]，但因為隊員增加使得基地更為狹小，而且西本願寺的[[勤王|勤王]]色彩相當濃厚。和[[長州藩|長州藩]]的關係也相當親密，使得土方想要一舉根絕這個關係。所以土方堅決反對寺方遷移基地的要求並且中止這個行動。但勤王意志相當強的山南則強力反對土方的作法，並且近藤和土方就這件事完全沒有交集。使得山南對於對方的作為相當不滿，造成山南終於下定決心脫離新選組。

*山南對於當初以[[尊王攘夷|尊王攘夷]]為目的所組成，但卻漸漸為了維護[[幕府|幕府]]統治，而實際上不斷壓迫勤王志士的[[新選組|新選組]]存在意義，日益感到疑惑。為了不違自己初衷，因此決定脫逃。

*和之後脫離新選組的伊東有密約，所以預先離開新選組。

*根據[[平成10年|平成10年]]所找到的《浪士文久報国記事》（[[永倉新八|永倉新八]]的手記），元治元年（1864年）6月26日因為山南生病，被從發生[[禁門之變|禁門之變]]的根據地，調到新選組基地擔任守備工作，因而無法被寫入禁門之變的事蹟裡，因此對於自己的病感到苦惱。

== 切腹 ==
[[伊東甲子太郎|伊東甲子太郎]]和從試衛館以來就結交的好友[[永倉新八|永倉新八]]，在山南被判決切腹後，私下都勸山南再次逃走，但山南卻決意赴死，以維持身為[[武士|武士]]的尊嚴。在一些作品中，記載在永倉新八的好意幫忙下，讓山南的紅粉知己[[島原|島原]]（地名）[[藝妓|藝妓]]天神明-{里}-在山南死前，與他見面，並且還流傳著明-{里}-與山南訣別之悲劇故事（但在永倉的手記《新撰組顛末記》和《浪士文久報国記事》中都沒有關於明-{里}-的記述、所以這段是[[子母澤寛|子母澤寛]]創作的可能性相當高；山南死後，據說[[沖田總司|沖田總司]]一直幫他照顧著明-{里}-。當時明-{里}-已染上當時是[[絕症|絕症]]的[[肺結核|肺結核]]，沖田也因此被她傳染，最後也因[[肺結核|肺結核]]而死）。

元治2年（1865年）2月23日，山南切腹，享年33歲。[[介錯|介錯]]人則由山南主動請求，由他一向疼愛、有如親弟弟一般的[[沖田總司|沖田總司]]來執行。他切腹前告訴沖田：「在我沒出聲要你動手之前，你都不許揮刀。」他冷靜地忍受劇烈痛苦，凜冽地完成[[切腹|一文字切腹]]的完整過程，讓旁觀者都為之動容。這次的切腹，[[近藤勇|近藤勇]]以「即使是[[淺野内匠頭|淺野内匠頭]]{{noteTag|name=a|1=《[[忠臣藏|忠臣藏]]》故事中[[赤穗浪士|赤穗浪士]]的主人，因為砍傷吉良義央，被迫切腹}}這樣的人，也沒辦法那麼漂亮的結束自己生命」，深深讚嘆。而以「魔鬼副長」聞名的[[土方歲三|土方歲三]]也在山南切腹的時候，忍不住掉下了眼淚。山南是死於嚴厲「局內法度」的新選組最高階幹部；由於他平日的敦厚與人望，以及最後選擇以捍衛武士的尊嚴方式而死，因此即使背負擅自逃隊的罪名，山南仍被新選組的隊友所敬佩，

山南的墓所位於京都的[[光緣寺|光緣寺]]。

[[伊東甲子太郎|伊東甲子太郎]]也在山南死後幫他寫了四首和歌。

== 軼聞 ==
*據說山南因為其溫厚的性格，而被壬生村附近的孩子和女孩所仰慕著。
*據說山南身材並不高大，而且有著皮膚白皙充滿魅力的臉孔，氣質與當時大多粗魯、兇暴的[[武士|武士]]、[[浪士|浪士]]極不同。
*在8月18日政變出動時、近藤和土方自己穿著盔甲，但卻沒有把盔甲交給身為總長的山南，因而激怒了他；後來是靠著[[松原忠司|松原忠司]]的居間勸誡才解決。
*在2004年的[[NHK|NHK]]歷史[[大河劇|大河劇]]《[[新選組!|新選組!]]》，描寫山南切腹的第33集〈友の死〉，是在49集中唯一破例重新播出的一集。並且在播出時得到極高的反響。其中還有部份狂熱的戲迷，在播出前寫信給NHK，要求「請不要殺死山南」。
*山南的名字在暗殺事件後，自新撰組活動名單消失。有一說是山南因暗殺事件後，對[[新選組|新選組]]的暴力、殺戮的作風感到厭惡，因此之後幾乎都沒有參加新選組的活動。
