'''山茶科'''（Theaceae）大部分[[属|属]]原产于[[亚洲|亚洲]]东部，多分布[[热带|热带]]及[[亚热带|亚热带]]地区，南至[[马来西亚|马来西亚]]，北至[[日本|日本]]，也有几个属是生长在[[美洲|美洲]]的。

== 形态 ==
绝大部分山茶科植物是常绿的[[灌木|灌木]]或[[乔木|乔木]]，但[[紫茎属|紫茎属]]和[[北美大头茶属|北美大头茶属]]是落叶植物。山茶科植物[[叶姓|叶]]为单叶，互生，一般为革质，没有托叶，花常两性，辐射对称，单生，有时簇生，萼片和[[花瓣|花瓣]]常各5枚，雄蕊常甚多，分离或基部结合，或集成5束，[[子房|子房]]上位，大多数3-5室，每室有二至数[[胚珠|胚珠]]；果实为裂开的[[蒴果|蒴果]]，近木质，有时肉质不裂。

== 用途 ==
山茶科最著名的品种是[[茶树|茶树]]，叶含咖啡碱是世界著名[[饮料|饮料]]之一；还有许多品种可以作为[[花卉|花卉]]观赏植物，如[[金花茶|金花茶]]和各种[[山茶花|山茶花]]；木荷属尚有些品种为[[木材|木材]]来源，[[油茶|油茶]]及山茶的种子可榨油，供食用及工业用途。

== 分类 ==
约有30[[属|属]]700余[[种|种]]，[[中华人民共和国|中国]]约有15属400余种，主要产于[[长江|长江]]以南各地。有的分类学家将山茶科分入[[厚皮香科|厚皮香科]]，也有的将厚皮香科[[植物|植物]]分入山茶科的。

[[克朗奎斯特分类法|克朗奎斯特分类法]]单独列出一个[[山茶目|山茶目]]，2003年的[[APG_II_分类法|APG II 分类法]]将山茶科和厚皮香科共同列入[[杜鹃花目|杜鹃花目]]，厚皮香科和[[五列木科|五列木科]]亲缘更近。
