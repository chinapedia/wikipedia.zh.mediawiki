'''山田氏'''，是日本的一支[[氏族|氏族]]。目前是日本第十三大姓氏。

== 起源 ==
古代的山田氏也是有多個來源，大多屬於[[渡来人|渡来系]]的氏族：
# 根據日本『[[新撰姓氏録|新撰姓氏録]]』內表明：[[右京|右京]]地區的'''山田宿祢氏'''，是屬於東周[[周霊王|周霊王]]的太子晋之子孫，而相同於山田造（山田宿祢）。
# [[河内国|河内国]]的'''山田宿祢氏'''，則為三國時[[曹魏|曹魏]][[司空|司空]][[王昶_(三國)|王昶]]的子孫。
# 山田連氏（與山田宿祢氏是共同祖先）
# 山田造氏（與山田宿祢氏是共同祖先）
# [[和泉国|和泉国]]的山田造氏（被記錄是[[新羅|新羅]]的'''天佐疑利命'''之子孫）
# 平安時代初期，眾所周知的能吏[[山田古嗣|山田古嗣]]
# '''清和源氏満政流'''的山田氏：[[平安時代|平安時代]]末期に[[清和源氏|清和源氏]][[源満政|満政]]流[[八島氏|八島氏]]の一族であった[[浦野重直|浦野重直]]（山田先生）が[[尾張国|尾張国]][[山田郡_(尾張国)|山田郡]][[山田荘|山田荘]]を領したことに始まる。[[尾張源氏|尾張源氏]]の嫡系にあたる氏族とも考えられ、[[京都|京]]と[[鎌倉|鎌倉]]を結ぶ交通の要衝に代々勢力を持った。[[鎌倉時代|鎌倉時代]]初期には、[[承久の乱|承久の乱]]で京方の勇将として活躍した[[山田重忠|山田重忠]]を出している。[[戦国時代_(日本)|戦国時代]]には[[今川氏|今川氏]]の家臣である[[山田景隆|山田景隆]]がおり、[[岡崎城|岡崎城]]代を勤めた

==著名人物==
*[[山田詩子|山田詩子]]（1963年－）、絵本作家、絵本翻訳家。(參見日文維基百科)
*[[山田孝之|山田孝之]]（1983年10月20日－）是出生於日本鹿兒島縣薩摩川內市的男演員及歌手
*[[山田優|山田優]]（1984年7月5日－），（舊姓：山田），婚後名小栗優，是一名模特兒、歌手、女演員及藝人。
*[[山田涼介|山田涼介]]（1993年5月9日－），日本演員、歌手，為尊尼事務所旗下藝人
