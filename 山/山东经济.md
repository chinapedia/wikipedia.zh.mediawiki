'''[[山东|山东]]'''是中国经济较为发达的省份，经济实力处在中国前列，是发展速度较快的[[中国一级行政区|省份]]。2007年以来经济总量居[[中华人民共和国省级行政区地区生产总值列表|第3位]]。2015年山东省实现生产总值63002.3亿元，比上年增长8.0%。山东经济发展三大支柱为：固定资产投资（总额4.74万亿元，增长13.9%）、社会消费品零售（总额2.78万亿元，增长10.6%）和出口（总额1440.6亿美元，下降0.4%）。不过目前，山东面临经济下行的压力仍然较大。<ref>{{cite web|title=2015年各省GDP：山东GDP跻身6万亿俱乐部 仅次广东江苏|url=http://www.guancha.cn/economy/2016_01_24_349062.shtml|website=观察者|publisher=澎湃新闻|accessdate=2016-11-28|date=2016-01-24}}</ref>

==经济史==
[[清圣祖|清圣祖]][[康熙|康熙]]六十一年（1722年），山东耕地达到9000余万亩。

1979-2008年30年，按绝对数比，2008年[[山东省地区生产总值|GDP总量]]为1978年的137倍（不变价，基于同期全国平均物价水平为基准），GDP年均增速11.6％，仅次于[[广东省地区生产总值|广东]]、[[浙江省地区生产总值|浙江]]、[[福建省地区生产总值|福建]]和[[内蒙古自治区地区生产总值|内蒙古]]居[[中华人民共和国省级行政区地区生产总值列表|全国第5位]]。1978-2008年，除1979、1981、1986、1989和1990年5个年份，其余年份GDP年增长率高于10％，GDP增幅最高的年份为1993年，年增长率达到20.4％。[[山东省地区生产总值|山东省GDP总量]]在全国的位次从1978年的第4位，至2007年稳居第3位。山东1988年GDP总量首次突破1千亿元，2002年突破1万亿元，2006年突破2万亿元。基于美元折算，1995年突破500亿美元，2000年破1千亿美元，2010年破5000千亿美元达到5,786亿美元。山东人均经济占有量一直处在前[[中国省级行政区人均地区生产总值列表|10位]]，2008年人均GDP为32,936元（4,021美元），2010年41,106元（6,072美元），高于同期全国人均水平37个百分点居第9位。2011年GDP总量（最终核实数）达到45,361.85亿元（7,023.26亿美元），与上年相比增幅达到10.9％，占全国的比重达到9.62％，人均47,190元，约合7,306美元<ref>参见：《山东统计年鉴2012》</ref>。2014年山东GDP近1万亿美元，人均GDP9911美元。

== 各类产业 ==
===农业 ===
山东是中国的农业大省，农业增加值长期稳居中国各省第一位。山东不仅栽培植物、饲养畜禽品种资源丰富，而且可资利用的野生动、植物资源也很丰富。在植物资源中，有[[小麦|小麦]]、[[玉米|玉米]]、[[地瓜|地瓜]]等粮食作物和[[棉花|棉花]]、[[花生|花生]]等经济作物40多种，有[[蔬菜|蔬菜]]、[[瓜类|瓜类]]60多种，有[[林木|林木]]、[[果树|果树]]、[[茶树|茶树]]、[[桑树|桑树]]、[[柞岚|柞岚]]等木本植物660多种，有淀粉糖类、脂肪油类、纤维类、芳香油类、鞣酸栲胶类、药用类、土农药类等野生经济植物1350多种。在动物资源中，有[[家畜|家畜]]、[[家禽|家禽]]等饲养动物10多种，有中小型哺乳兽类55种，有[[留鸟|留鸟]]、[[夏候鸟|夏候鸟]]、[[冬候鸟|冬候鸟]]、[[旅鸟|旅鸟]]270多种，有[[害虫|害虫]]天敌563种，有农业害虫763种。此外还有内陆水生[[维管束植物|维管束植物]]30多种，内陆[[淡水鱼类|淡水鱼类]]70多种。山东省的粮食产量较高，粮食作物种植分夏、秋两季。夏粮主要是[[冬小麦|冬小麦]]，秋粮主要是[[玉米|玉米]]、[[地瓜|地瓜]]、[[大豆|大豆]]、[[水稻|水稻]]、[[谷子|谷子]]、[[高粱|高粱]]和[[小杂粮|小杂粮]]。其中小麦、玉米、地瓜是山东的三大主要粮食作物。

=== 工業 ===
山东的工业发达，工业总产值及工业增加值据中国各省前三位，特别是一些大型企业较多，号称“群象经济”。现在山东正在准备投巨资打造半岛先进制造业基地，以后山东的制造业会更上一层楼。

此外由于山东是全国重要的粮棉油肉蛋奶的产地，因此在[[轻工业|轻工业]]特别是[[纺织|纺织]]和[[食品工业|食品工业]]相当发达。

[[重工业|重工业]]企业发展迅速，重点工矿业企业有[[齐鲁石化|齐鲁石化]]、[[山东电力|山东电力]]、[[山东钢铁|山东钢铁]]（[[济钢|济钢]]、[[莱钢集团|莱钢]]、[[金岭铁矿|金岭铁矿]]）、[[日照钢铁|日照钢铁]]、[[山东海化|山东海化]]、[[胜利油田|胜利油田]]、[[兖矿集团|兖矿集团]]、[[中国铝业|中国铝业]][[山东铝厂|山东铝厂]]、[[南山集团|南山集团]]、[[晨鸣纸业|晨鸣纸业]]等。

=== 交通运输业 ===
==== 公路 ====
{{see also|山東高速公路}}

山東有密集的[[高速公路|高速公路]]網，通車里程至2008年底为4285公里，居全國前列。1993年建成第一條[[濟青高速公路|濟青高速公路]]。現有的「三縱兩橫」網絡主要包括南北向的[[京福高速公路|京福]]、[[京滬高速公路|京滬]]、[[沈海高速公路|沈海]]三線與東西向的[[青兰高速公路|青兰]]、[[青銀高速公路|青銀]]兩線。全省多數城市之間可在半日互達。新建成的與在建的高速公路多基於「五縱四橫一環」的藍圖。山東省計畫在2012年使高速公路的通車里程達到5000公里，實現全省80%的縣、市、區由高速公路全程連接。

依托一流的密集的优质公路网，山东的公路客运极其发达。[[济南长途客运总站|济南长途客运总站]]是全国较大的客运站之一，日均发送旅客7万人次，[[春运|春运]]时期可达到15万人次，这里有发往全省各地的班车，也发往[[北京|北京]]、[[天津|天津]]、[[河南|河南]]、[[河北|河北]]、[[中国东北地区|东北]]、[[上海|上海]]、[[内蒙|内蒙]]、[[湖北|湖北]]、[[陕西|陕西]]、[[山西|山西]]、[[新疆|新疆]]等地的车次。

==== 鐵路 ====
南北走向的主要有[[京沪铁路|京沪铁路]]、[[京九铁路|京九铁路]]、[[京滬高速鐵路|京滬高速鐵路]]、[[藍煙铁路|藍煙铁路]]等鐵路。京滬鐵路最為繁忙，在山東境內有三條複線，其中一條是貨運專用。蓝烟铁路连接青岛、烟台两大港口。

東西走向的主要有[[膠濟鐵路|膠濟鐵路]]、[[胶济客运专线|胶济客运专线]]等鐵路。2006年膠濟鐵路電氣化改造完成，成為山東省的第一條[[電氣化鐵路|電氣化鐵路]]，2008年建成[[胶济客运专线|胶济客运专线]]，实现了客货分运，最高时速可以达到200公里。还有[[石德铁路|石德铁路]]、[[兖石铁路|兖石铁路]]、[[邯济铁路|邯济铁路]]、[[新兖铁路|新兖铁路]]及一些通往矿区的支线。

省境內樞紐性的鐵路枢纽及其匯通的鐵路線列舉如下：濟南枢纽（京滬-膠濟-邯济）、[[青岛铁路枢纽|青岛枢纽]]（胶济-胶新-胶黄-蓝烟）、德州枢纽（京滬-石德，[[北京鐵路局|北京鐵路局]]管）、兖州枢纽（京滬-新兖-兖石）、菏澤枢纽（京九-新兖）。

按照铁道部的划分，山东省内现有特等火车站4个，分别为[[济南站|济南站]]（客货运车站）、[[济西站|济西站]]（货运编组站、全国十大路网级货运编组站之一）、[[济南西站|济南西站]]（京沪高速铁路客运车站）、[[青岛站|青岛站]]（客运车站）

铁道部[[济南铁路局|济南铁路局]]在2008年调整后管辖山东全省（除德州）和（[[烟大铁路轮渡|渤海铁路轮渡]]及旅顺西站）地区干支线铁路网。济南铁路局，包括济南西、济南两大机务段，济南、青岛、济西、济南西4座客运火车站，济南、青岛两大客运段。负责济南西-山海关，石家庄-济南-青岛等多条重要交路。保障青岛、日照等沿海港口枢港运输，晋煤与内陆其他物资出海和北京、东北地区与江浙地带铁路运输。在2007年全国企业500强中，居于铁路企业的中上等。

==== 航空 ====
山東的主要機場有[[濟南遙牆國際機場|濟南遙牆國際機場]]、[[青島流亭國際機場|青島流亭國際機場]]、青岛胶东国际机场（在建）、[[煙台蓬莱國際機場|煙台蓬莱國際機場]]、[[济宁曲阜机场|济宁曲阜机场]]、[[临沂机场|临沂机场]]、[[威海大水泊机场|威海大水泊机场]]、[[东营永安机场|东营永安机场]]、[[潍坊南苑机场|潍坊南苑机场]]、[[蓬莱沙河口机场|蓬莱沙河口机场]]。另外，还有一处海上救援机场，位于烟台蓬莱市。

==== 海运 ====
山东主要有[[青岛|青岛]]、[[烟台|烟台]]、[[潍坊|潍坊]]、[[日照市|日照]]、[[威海|威海]]、[[龙口|龙口]]、[[东营|东营]]、[[羊角沟|羊角沟]]等港口。[[青島港|青島港]]始建於1892年，港口總吞吐量3億噸（2008年度），居於中國內地海港的第五位，同期[[日照港|日照港]]、[[烟台港|烟台港]]分列国内第十和第十三位。青岛港集裝箱吞吐量1002萬TEU標准箱（2008年度），居中國內地海港的第五位，是世界15大集裝箱港口之一，同时烟台港列国内第十二位。青岛港有前往日本[[下关市|下关]]港的渡轮航行。烟台港则由[[烟台中韩轮渡有限公司|烟台中韩轮渡有限公司]]运营[[烟台|烟台]]-[[仁川|仁川]]国际轮渡航线。威海有[[威海|威海]]-[[大連|大連]]、[[威海|威海]]-[[仁川|仁川]]、[[龍眼港|龍眼港]]-[[平澤|平澤]]、[[石島|石島]]-[[仁川|仁川]]客貨滾裝航綫。

== 地方财政 ==

[[青岛市|青岛市]]自1986年成为[[计划单列市|计划单列市]]，[[财政|财政]]不归[[山东省人民政府|山东省人民政府]]管辖，而是与[[中华人民共和国国务院|国务院]]对接。山东的经济规模虽然长年排全国第三，但自2016年出现财政亏损，不再向[[中华人民共和国中央政府|中央政府]]提供财政盈余，需要中央政府[[财政转移支付|财政转移支付]]以维持财政平衡<ref name="a1">{{Cite web |url=http://cj.sina.com.cn/article/detail/1010236564/349470?column=realty&ch=9 |title=重磅：神奇的财政平衡术，31省财政转移支付全景图 |accessdate=2018-05-01 |author= |date=2017-08-07 |work= |publisher=新浪网|language=Zh-hans }}</ref>。
