== 历史 ==
[[File:Sandakan_Sabah_SandakanMemorialPark-04.jpg|右]]
由于日本人在二战初期成功,  许多盟军士兵被关押在太平洋的一个营地里。1942年7月,
已经有1500名澳大利亚战俘从新加坡转移到了山打根，作为被迫劳力来建造一个军用机场。
人数在1943年增加，约有2500名囚犯已被安置在营地。

战争结束时，在现场调查期间发现的所有战俘的遗骸都被转移到纳闽岛的一个军事公墓。那些可以确认的人被埋在坟墓和标上一个名字，而那些无法确认的人被列在一致的板块，在纳闽岛也在新加坡。

1995年、沙巴州政府与澳洲政府达成协议，澳洲退伍军人协会(RSL)和三打根市政委员会建立了这个纪念地址。澳大利亚政府用一个方尖碑建造了这个展览馆，恢复了设施余留的技术设备并在这个纪念地址上安装了一台挖掘机、蒸汽发电机和其它发电机。纪念地址于1999年3月18日举行开幕典礼。

== 位置 ==
纪念公园位于三打根机场西南1.5公里处，这里是澳大利亚B部队前营地的一部分。

== 站 ==
纪念碑被分为六个站，这些站是通过环形路线连接起来的。

=== 挖掘机 ===
[[File:Sandakan_Sabah_SandakanMemorialPark-05.jpg|右]]
这个挖掘机是用来为日本人修建机场的。在维修期间，它被一名澳大利亚战俘破坏，直到现在才重新开始使用。

=== 蒸汽发电机和发电机 ===
蒸汽发电机与其它发电机成为营地的主要电源。用木头烧的蒸汽发电机，用来照明营地和围栏。

=== 大树 ===
“大树”是一个巨大的样本，它原本矗立在今天的纪念方尖碑上。它是战俘集中营的主要结构。战争结束后不久，这棵树被一场大火烧毁了。2008年4月25日，一棵新的蒙斯里鲍姆被种植在公园入口附近。

=== 粮食仓库、储存区和厨房 ===
日本人还为日本人经营了一个粮食仓库和厨房，水罐被保存在混凝土中。

=== 主要入口和出入道路 ===
主要入口位于纪念公园的东面，在储藏区之前。通往机场的路向左边然后往右就是从三打根到拉瑙的8号路大街。从这里开始，所有的三个行军都到拉瑙。

== 纪念亭 ==
三打根纪念亭于1999年3月18日由澳大利亚退伍军人事务部部长布鲁斯·斯科特开幕。它包括一个关于战俘集中营和死亡行军的永久展览。

== 方尖碑的说明 ==
[[File:Sandakan_Sabah_SandakanMemorialPark-02.jpg|右]]
“三打根纪念”的方尖碑是一个黑色的石碑，它的直径约为10米。这题词是:
<center>

山打根纪念
纪念所有那些 
在这里遭受痛苦和死亡的人，
在死亡行军
和在兰瑙

</center>

== 战俘的路线 ==
纪念公园是三次死亡行军中“战俘路线”的第一站。这条路线始于三打根，结束于拉瑙“最后一个战俘集中营”。路线上的每个站都有一个标志。
