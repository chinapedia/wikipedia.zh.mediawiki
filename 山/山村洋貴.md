'''山村洋貴'''{{bd}}，[[日本|日本]][[男性|男性]][[動畫師|動畫師]]、[[人物設計|人物設計師]]，隸屬於[[Studio_Pastoral|Studio Pastoral]]。

== 概要 ==
於[[代代木動畫學院|代代木動畫學院]]畢業後，在成人遊戲公司{{lang|ja|スタジオルクス}}擔當過原畫職位。其後加入了[[Studio_Pastoral|Studio Pastoral]]，不過在長期合作的另一間公司[[SHAFT|SHAFT]]都有專用工作台。

以前曾使用筆名'''{{lang|ja|むらやまたかひろ}}'''，離開{{lang|ja|スタジオルクス}}後亦有使用'''{{lang|ja|むらやま某}}'''、'''{{lang|ja|田園タロヲ}}'''等名字參與其他遊戲作畫。後來轉職成動畫師，展開同人活動，組織起名為{{lang|ja|すたじお田園}}，發佈過《[[不可思議的教室|不可思議的教室]]》、《[[魔法老師_(動畫)|魔法老師]]》、《[[絕望先生|絕望先生]]》等的同人誌。<ref>{{cite web |url=http://page6.auctions.yahoo.co.jp/jp/auction/f67179186 |title=存档副本 |accessdate=2011-07-03 |deadurl=yes |archiveurl=https://archive.is/20120712092157/http://page6.auctions.yahoo.co.jp/jp/auction/f67179186 |archivedate=2012-07-12 }}、{{cite web |url=http://page10.auctions.yahoo.co.jp/jp/auction/m80743055 |title=存档副本 |accessdate=2011-07-03 |deadurl=yes |archiveurl=https://archive.is/20120720032448/http://page10.auctions.yahoo.co.jp/jp/auction/m80743055 |archivedate=2012-07-20 }} Yahoo!Japan相關同人誌拍賣</ref>
