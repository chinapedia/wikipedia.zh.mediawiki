《'''山風短'''》，為「'''山'''田'''風'''太郎'''短'''篇傑作集」的簡稱，原著是[[山田風太郎|山田風太郎]]，由[[瀨川雅樹|瀨川雅樹]]所作畫改編的[[時代劇|時代劇]]漫畫。此為瀨川雅樹繼《[[甲賀忍法帖|甲賀忍法帖]]》、《[[柳生忍法帖|柳生忍法帖]]》之後所改編的山田風太郎之作品，每本單行本都是完全獨立的單集短篇故事。

==登場人物==
===第一幕 女忍紅騎兵===
====主人公====
*[[直江兼續|直江兼續]]
: 通稱'''山城守'''，[[上杉氏|上杉家]]的家老，[[眾道|眾道]]的高手。
*陽炎 / 大島山十郎
: 有著女人外表的美少年，擁有一身好武術，他真正的身分是...

====上杉家====
*[[上杉景勝|上杉景勝]]
: [[豐臣政權|豐臣政權]]的[[五大老|五大老]]之一，非常討厭女人。
*[[千坂景親|千坂民部]]
: 上杉家的家老，擅長外交工作。
*[[前田慶次|前田咄然齋]]
: [[傾奇者|傾奇者]]，[[前田利家|前田利家]]的外甥。
*[[上泉泰綱|上泉主水]]
: 劍術的達人，是劍聖[[上泉信綱|上泉信綱]]的孫子。
*[[車斯忠|車丹波]]
: 從[[佐竹氏|佐竹家]]轉至上杉家仕官的武將，剛力無雙，也是位鐵砲名人。
*[[岡野左內|岡野左內]]
: 從[[蒲生氏|蒲生家]]轉至上杉家仕官的豪傑，是名[[吉利支丹|吉利支丹]]的教徒。

====德川家====
*[[德川家康|德川家康]]
: [[豐臣政權|豐臣政權]]的[[五大老|五大老]]之一，覬覦天下。
*[[結城秀康|結城秀康]]
: 德川家康的次子，勇猛善戰。

====真田家====
*[[真田昌幸|真田昌幸]]
: 被喻為「表裡比興之人」，為了讓上杉家加入西軍而使用了某種手段...
*[[真田幸村|真田幸村]]
: 真田昌幸的次子。

===第二幕 劍鬼喇嘛佛===
====主人公====
*[[細川興秋|長岡興秋]]
: 通稱'''與五郎'''，細川忠興的次男，有著母親[[明智玉|細川加西亞]]樣貌血統的美男子。滿腦子都是劍術，跟宮本武藏決鬥是他最大的目標。
*登世
: 細川家忍組頭目袋兵齋的孫女，擁有一身好武藝，被袋兵齋與細川忠興下令對長岡興秋進行某種密術...

====細川家====
*[[細川忠興|細川忠興]]
: 細川家當主，對於興秋的行為舉止相當受不了。
*袋兵衛
: 細川家忍組'''青龍寺組'''的頭領。

====豐臣家====
*[[千姬|千姬]]
: 於[[大坂夏之陣|大坂夏之陣]]時救了登世，大坂之陣結束後返回德川家。
*[[塙直之|塙團右衛門]]
: 大坂之陣時加入豐臣方的武將，看不起沒有率領兵力的宮本武藏。

====浪人====
*[[宮本武藏|宮本武藏]]
: 號稱生涯無敗績的大劍豪。比起劍術對決，他反倒是持續教導興秋做人的道理。
