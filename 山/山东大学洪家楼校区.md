'''山东大学洪家楼校区'''，也称'''山东大学东区老校'''，位于[[山东省|山东省]][[济南市|济南市]][[历城区|历城区]]洪家楼5号，[[洪家楼天主教堂|洪家楼天主教堂]]东北侧，目前包括政治学与公共管理学院、法学院、外国语学院和艺术学院四个学院<ref>{{cite web|url=http://www.sdu.edu.cn/2010/xyzl.html|title=《山东大学学院总览》|accessdate=2010-06-19|work=|publisher=山东大学|archive-url=https://web.archive.org/web/20100623093556/http://www.sdu.edu.cn/2010/xyzl.html|archive-date=2010-06-23|dead-url=yes}}</ref>，由于该校园内花木繁多，被学生形象地称为“植物园”。

== 历史 ==
该校区最早为始建于1936年的教会学校'''济南私立懿范女子中学'''，由美国的天主教玛利亚修女会所创办，其主要建筑为教学楼、学生宿舍和修女院，均采用灰砖，坡屋顶上覆以红瓦，另包括百余间平房<ref>{{cite web|url=http://jnsb1.e23.cn/html/jnsb/20080819/jnsb144322.html|title=《懿范女子中学旧址》|accessdate=2010-06-19|work=|publisher=济南时报}}{{Dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>，该校园与周边的洪家楼天主教堂、方济各会华北总修（道）院、方济圣母传教修女会院、方济各会仁慈堂、方济各会神甫修士宿舍等构成了济南的一处较大的天主教教会建筑群<ref>{{cite book |author=张润武、薛立 |title=《图说济南老建筑》|year=2007 |publisher=济南出版社 |location=济南 |isbn=9787807102934}}</ref>。

1948年后，懿范女子中学并入黎明中学（后成为公立济南中学），原校址改作[[山东农业大学|山东省农学院]]校舍。1958年9月，按照山东省省委的决定，农学院迁往泰安，原在青岛办学的[[山东大学|山东大学]]迁至济南，并以此处作为一处校址至今，其后在济南东郊利农庄原拟作山东农展馆的征地划拨山东大学，用以建设新校（即今日的[[山东大学中心校区|山东大学中心校区]]）<ref>{{cite web |url=http://ouceducnxiaobao.cuepa.cn/show_more.php?tkey=&bkey=&doc_id=289954|title=《山东大学从青岛迁济南始末》|accessdate=2010-06-19 |work=田广渠|publisher=中国海洋大学校报}}</ref>。

== 主要建筑 ==

以下为早期建筑：

*1号楼：建于1936年，原为懿范女子中学教学楼，地上两层、地下一层，与2号楼均为济南天主教方济各会第三会所建，坐北面南，采用“H”型平面，楼内教室均沿中间的走廊两侧布置<ref>{{cite web |url=http://city.finance.sina.com.cn/city/2007-02-09/82190.html|title=《文物被重新认识 济南市级“文保”家族待添新丁》|accessdate=2010-06-19 |work=王海丞|publisher=舜网-济南日报}}</ref>，今为[[山东大学政治学与公共管理学院|政治学与公共管理学院]]所在地。
*2号楼：建于1936年，原为懿范女子中学的学生宿舍和修女院，地上两层、地下一层<ref>{{cite book |author=张润武、张复合等|title=《中国近代建筑总览-济南篇》|year=1996|publisher=中国建筑工业出版社|location=北京|isbn=}}</ref>，今为公安处。

以下为后期陆续新建：

*3号楼：原艺术学院所在地，今在外国语学院北侧建有艺术学院新楼并计划建设音乐厅。
*4号楼：法学院所在地，其东侧建有法学院新楼（[[逸夫法学楼|逸夫法学楼]]）。
*5号楼：[[山东大学图书馆|山东大学图书馆]]政法分馆所在地。
*6号楼：原哲学与社会发展学院所在地。
*7号楼：外国语学院所在地。
*8号楼：公共教学楼。
*9号楼：原物理学院所在地。
