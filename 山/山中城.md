'''山中城'''是位於日本古[[伊豆國|伊豆國]][[田方郡|田方郡]]山中（今[[静岡縣|静岡縣]][[三島市|三島市]]山中新田）的[[山城|山城]]，現在被指定為日本的國家史跡。
== 概要 ==
[[File:YamanakaJo_ShojiBori.JPG|250px]]
[[後北條氏|北條氏]]所築之城，該城也是[[小田原城|小田原城]]的支城。箱根十城之一。

三島市對此城進行了反映當時情況的整修，為避免堀與土壘等遺跡風化，而在其上覆土種植草皮來保護，且在清楚地把握畝堀與障子堀的構造下整修而使該城成為了能瞭解北條氏的築城方法的城跡。

雖然此城隨著北條氏的滅亡而廢棄，但也因此大量保留了北條氏獨特的城郭構造而受到注目。

每年在5月的第三個星期天時會在此城舉行山中城祭。

== 歷史 ==
[[永祿|永祿]]年間（1558年 - 1570年）時在[[北條氏康|北條氏康]]指示下建築此城。該城位於防衛北條氏根據地小田原西邊最重要的據點，以侵吞[[東海道|東海道]]的形式建造。

在[[北條氏政|北條氏政]]時因與[[豐臣秀吉|豐臣秀吉]]的關係惡化，山中城便進行整修以強化防禦，結果在還沒來得及完工時便得迎擊豐臣軍。
{{和暦|1590}}，[[小田原之戰|小田原之戰]]中[[豐臣秀次|豐臣秀次]]率領7萬的大軍攻擊山中城，而在城將[[松田康長|松田康長]]等4千人的奮戰下，豐臣一方也出現了部將[[一柳直末|一柳直末]]等眾多陣亡者，但在猛烈攻擊之下，該城還是只花了半天便被攻下，而自松田以下到城兵幾乎全數陣亡。此戰之後，該城便被廢棄。

該城在{{和暦|1930}}時被指定為國家史跡。從{{和暦|1973}}時三島市開始將該城整建成公園，同時也進行了學術上的調查。

{{和暦|2006}}4月6日時該城被選為[[日本100名城|日本100名城]]（40號）之一，而從{{和暦|2007}}6月開始舉行了全國性的日本100名城紀念章收集大會。
