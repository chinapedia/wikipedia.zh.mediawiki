'''山田美樹'''（{{bd|1974年|3月15日|||Yamada Miki}}），[[日本|日本]]女性[[政治家|政治家]]、[[經濟產業省|經產]][[官僚|官僚]]，[[自由民主黨_(日本)|自由民主黨]]黨員。出身於[[東京都|東京都]][[品川區|品川區]]。至今連續共當選2屆[[眾議院_(日本)|眾議院議員]]。在自民黨內屬於[[清和政策研究會|清和政策研究會]]（細田派）。

== 早年經歷 ==
父親是[[工程師|工程師]]，母親是[[鋼琴|鋼琴]]教師。高中畢業於[[櫻蔭高等學校|櫻蔭高等學校]]。大學畢業於[[東京大學|東京大學]]法學部。1996年進入[[通商產業省|通商產業省]]（現在的[[經濟產業省|經濟產業省]]）工作，負責[[世界貿易組織|世界貿易組織]]架構下的貿易談判，以及[[特許廳|特許廳]]的組織改革。26歲到美國[[哥倫比亞大學|哥倫比亞大學]]留學，取得經營學[[碩士|碩士]]學位。歸國後被省內安排到[[內閣官房|內閣官房]]工作。30歲從經產省退官，先後加入[[波士頓諮詢公司|波士頓諮詢公司]]和[[愛馬仕|愛馬仕]]。

== 政治生涯 ==
2011年11月，參加[[自由民主黨_(日本)|自由民主黨]]東京都連實施的選舉候選人公募合格，被公認為[[第46屆日本眾議院議員總選舉|第46屆日本眾議院議員總選舉]]自民黨在[[東京都第1區|東京都第1區]]的候選人。山田還受到出身於東京1區，由於年齡問題而引退的前[[內閣官房長官|內閣官房長官]][[與謝野馨|與謝野馨]]的大力支援，確認她為與謝野的接班人。2012年舉行的第46屆總選舉，作為政治新人的山田以1,134票的微弱優勢，在東京1區擊敗強敵——[[民主黨_(日本)|民主黨]]的前[[經濟產業大臣|經濟產業大臣]][[海江田萬里|海江田萬里]]。在全國300個選舉區中，東京1區是最後一個決出勝負的選區。<ref>[http://sankei.jp.msn.com/politics/news/121217/elc12121703360074-n1.htm {{lang|ja|山田美樹氏、最後に滑り込み　選挙区で海江田氏破る　東京1区}}] {{webarchive|url=https://web.archive.org/web/20131110163232/http://sankei.jp.msn.com/politics/news/121217/elc12121703360074-n1.htm |date=2013-11-10 }} 產經新聞 2012年12月17日</ref>山田的勝利轟動全國。儘管海江田隨後即在比例區復活當選，而且接任民主黨代表，但比例復活的陰影使海江田一直被指責缺乏做黨首的資格。

兩年後的2014年12月[[第47屆日本眾議院議員總選舉|第47屆日本眾議院議員總選舉]]，山田再次對決海江田。經過兩年的時間，選前各項民調都顯示山田對海江田優勢明顯。加上自民黨決心舉全黨之力支援山田，誓令最大在野黨黨首的海江田在小選舉區落敗。選舉結果山田順利在小選舉區連續兩次擊敗海江田。<ref>[http://mainichi.jp/select/news/20141215k0000m010198000c.html {{lang|ja|衆院選:山田美樹氏「金星」　東京１区で海江田氏破る}}] {{webarchive|url=https://web.archive.org/web/20141217182023/http://mainichi.jp/select/news/20141215k0000m010198000c.html |date=2014-12-17 }} 每日新聞</ref>而且海江田的惜敗率遠比上屆低，意外地未能比例復活，徹底落選。僅當選議員兩年的山田完成了令時隔65年再一次有最大在野黨黨首落選的壯舉。<ref>{{Cite web|url=http://www.hochi.co.jp/topics/20141215-OHT1T50128.html|script-title=ja:【衆院選】民主・海江田氏落選!野党第1党党首では49年片山元首相以来の屈辱|publisher=[[體育報知|體育報知]]|deadurl=yes|archiveurl=https://web.archive.org/web/20150924114556/http://www.hochi.co.jp/topics/20141215-OHT1T50128.html|archivedate=2015-09-24}}</ref>

== 政策主張 ==
* 曾經在服飾相關企業[[愛馬仕|愛馬仕日本]]有經營企劃的經驗，因此打出「通過時尚與文化向世界宣傳」的政治主張。<ref>[http://miki-yamada.com/profile/ 官網自我介紹]</ref>。
* 贊成修改[[日本國憲法|日本國憲法]]。<ref name="mainichi2012">{{Cite news|url = http://senkyo.mainichi.jp/46shu/kaihyo_area_meikan.html?mid=A13001001001|script-title = ja:2012衆院選 東京1区 山田美樹 |newspaper = [[毎日新聞|毎日新聞]] |publisher = [[毎日新聞社|毎日新聞社]] }}</ref>
* 贊成通過政府解釋憲法行使[[集體自衛權|集體自衛權]]。<ref name="mainichi2012"/>
* 不支持日本進行[[日本核武装論|核武装]]。<ref name="mainichi2012"/>
* 贊成日本加入[[跨太平洋戰略經濟夥伴關係協議|跨太平洋戰略經濟夥伴關係協議]]（TPP）。<ref name="mainichi2012"/>
