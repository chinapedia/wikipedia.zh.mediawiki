'''山中樵'''（やまなか　きこり，{{bd|1882年|10月3日|1947年|11月11日}}），日本[[福井縣|福井縣]][[福井市|福井市]]人，是位圖書館學者，曾任[[臺灣總督府圖書館|臺灣總督府圖書館]]第五任館長。1941年，獲敘勳四等授瑞寶章。[[二次大戰|二次大戰]][[日本投降|日本投降]]後仍留在臺灣協助[[范壽康|范壽康]]與[[吳克剛|吳克剛]]成立臺灣省立圖書館，直到1947年發生[[二二八事件|二二八事件]]才在該年5月返回日本，並於同年11月11日病逝，葬於[[輪王寺|輪王寺]]。{{r|張}}

== 早年生活 ==
1888年4月，就讀外記丁尋常小學校（今仙台市立上杉山通小學校）。但由於其父調任[[角田市|角田]]警察署長，也隨著轉入角田尋常高等小學校（今角田市立角田小學校），1892年3月，畢業於角田尋常高等小學校。1900年，高等師範學校（今[[筑波大學|筑波大學]]前身之一）考試合格，但由於年齡不足取消入學許可。同年3月，畢業於[[仙台市|仙台市]]之宮城縣尋常中學校（今宮城縣仙台第一高等學校）。

== 任教 ==
1901年，任奈良縣立工業學校（今奈良縣立御所工業高等學校）教員。1906年3月，中等教員歷史科（西洋史）檢定試驗合格。同年11月，任奈良縣立工業學校教諭兼舍監。同年同月轉任宮城縣立高等女學校（今宮城縣宮城第一高等學校）教授歷史，同時擔任同窗會幹事長。在職期間，由於得到考古學研究家[[高橋健自|高橋健自]]的指導與薰陶，對考古學研究產生極大興趣。

== 其他 ==
在[[大日本帝國|大日本帝國]]進入台灣後，日本決定拆除[[台北城|台北城]]，而與此同時，山中樵也發出不要拆除城門的聲音。
也因此使得[[台北府城北門|北門]]、[[台北府城東門|東門]]、[[台北府城南門|南門]]和[[台北府城小南門|小南門]]得以保存至今，而[[台北府城西門|西門]]則在山中樵提出之前拆除，因此西門並沒有保存下來。
