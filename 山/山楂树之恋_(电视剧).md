{{Engname|《山楂树之恋》|Under the Hawthorn Tree}}是一部2012年首播的中国大陆当代爱情悲剧，改编自美籍华裔女作家艾米所著同名长篇小说《[[山楂树之恋|山楂树之恋]]》，亦是2010年上映的华语电影《[[山楂树之恋_(电影)|山楂树之恋]]》的电视剧版本。该剧由[[幸福蓝海影视|幸福蓝海影视文化集团股份有限公司]]、[[中视传媒股份有限公司|中视传媒股份有限公司]]、北京凤凰联动文化传媒有限公司等联合出品，[[李路_(导演)|李路]]执导，并由[[王珞丹|王珞丹]]、[[李光洁|李光洁]]、[[黄梅莹|黄梅莹]]、[[林继东|林继东]]、[[孙璐|孙璐]]、[[翟万臣|翟万臣]]等领衔主演。全剧以[[文化大革命|文化大革命]]即将结束的1975年前后为背景，讲述了一位家庭成分不好的女孩与一位高干子弟之间一段凄美动人的爱情故事<ref name=info>{{cite news | title=电视剧《山楂树之恋》江苏卫视3月12日呈现 | url=http://ent.sina.com.cn/v/m/2012-03-01/09353569223.shtml | newspaper=[[新浪|新浪娱乐]] | date=2012-03-01}}</ref>。

本剧于2010年8月28日在故事发生地湖北[[宜昌|宜昌]]正式开机拍摄<ref>{{cite news | title=电视版《山楂树》开机 李光洁王珞丹上香祈福 | url=http://ent.qq.com/a/20100828/000167.htm | newspaper=[[腾讯娱乐|腾讯娱乐]] | date=2010-08-28}}</ref>，同年12月20日杀青关机<ref>{{cite news| title=电视剧版《山楂树之恋》在夷陵区完美封镜| url=http://www.sxylcm.cn/art/2010/12/21/art_4129_302456.html| newspaper=三峡夷陵传媒网| date=2010-12-21| deadurl=yes| archiveurl=https://web.archive.org/web/20140714113018/http://www.sxylcm.cn/art/2010/12/21/art_4129_302456.html| archivedate=2014-07-14}}</ref>。2012年3月12日，本剧在[[江苏卫视|江苏卫视]][[黄金时段|黄金档]]全国首播<ref name=info/>。

==剧情==
1974年的春天，[[毛泽东|毛泽东]]发起的[[文化大革命|文化大革命]]已经是强弩之末，“静秋”这个“家庭成分不好”的[[高中|高中]]女生被[[下放|下放]]到西村坪编写小学教材，当了一名[[知青|知青]]，她在那里遇到了勘探队员“老三”，老三家庭背景好，根正苗红，父亲是[[中国共产党|中国共产党]]党员和[[中国人民解放军|中国人民解放军]]高级军官，两人在这个青涩、朦胧的花季雨季里相爱，由于当时中国大陆的封闭和特殊政治气候，两人的爱情被拆散，最终成了一幕悲剧。

== 花絮 ==
[[张艺谋|张艺谋]]执导的电影版《[[山楂树之恋_(电影)|山楂树之恋]]》曾取得巨大成功。<ref>{{cite web |url=http://ent.sina.com.cn/v/m/2012-03-30/06583593947.shtml |title=《山楂树之恋》首播告捷 李路对王珞丹表演满意 |accessdate=2012 |publisher=新浪网 |work= |language=zh-hans}}</ref>

该剧情节细腻、动人，讲述特殊政治大环境时代的被摧残的爱情，引发关注，观众表示“可以接受”，也间接启发人们，任何人都无法摆脱政治环境的约束和控制。<ref>{{cite web |url=http://ent.163.com/photoview/00AJ0003/62497.html |title= 王珞丹《山楂树之恋》催泪剧照|accessdate=2012 |publisher=网易 |work= |language=zh-hans}}</ref><ref>{{cite web |url=http://ent.sina.com.cn/v/m/2012-03-22/14423587546.shtml |title=《山楂树之恋》催泪袭来 王珞丹变身哭戏王(图) |accessdate=2012 |publisher=新浪网 |work= |language=zh-hans}}</ref><ref>{{cite web |url=http://ent.163.com/12/0318/07/7SS3VU2700032DGD.html |title=《山楂树之恋》热播 王珞丹:母亲劝我别演了 |accessdate=2012 |publisher=网易 |work= |language=zh-hans}}</ref>

王珞丹表示出于“静秋”是“无怨无悔”。<ref>{{cite web |url= http://ent.sina.com.cn/v/m/2012-03-15/09273580939.shtml|title=王珞丹谈《山楂树之恋》：演静秋无怨无悔(图) |accessdate=2012 |publisher=新浪网 |work= |language=zh-hans}}</ref>
