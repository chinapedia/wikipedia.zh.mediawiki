'''山廈村張氏宗祠'''，又稱'''華封堂'''<ref>{{cite web |language=zh-hk |url=http://www.lcsd.gov.hk/ce/Museum/Monument/form/shan_ha_tsuen_doc_3.pdf |title=元朗山廈村：生計、教育 |publisher=''lcsd.gov.hk'' |accessdate=2011-02-23 |archive-url=https://web.archive.org/web/20110920015502/http://www.lcsd.gov.hk/CE/Museum/Monument/form/shan_ha_tsuen_doc_3.pdf |archive-date=2011-09-20 |dead-url=yes }}</ref>，位於[[香港|香港]][[新界|新界]][[元朗區|元朗區]][[屏山_(香港)|屏山]]山廈村（今稱山下村），於[[嘉慶|嘉慶]]20年（1815年）興建，是村民聚會及祭祖之所，1999年被列為[[香港法定古蹟列表|法定古蹟]]。

==歷史==
山廈村（今稱山下村）的開村始祖乃張直臣，早於[[明朝|明朝]]因經營鹽業，由[[東莞|東莞]]簧村遷至[[寶安縣|新安]][[橫洲_(元朗)|橫洲]]，於[[清朝|清朝]][[順治|順治]]年間遷居山廈村現址。[[康熙|康熙]]元年清廷厲行[[遷界令|遷界令]]，張氏族人被迫遷徙。康熙8年復界後，部分族人未有隨回，致使分居各地。時至今日，山廈村仍為張氏的單姓村落<ref>{{cite web |language=zh-hk |url=http://www.amo.gov.hk/form/shan_ha_tsuen_doc_1.pdf |title=元朗山廈村：立村 |publisher=''amo.gov.hk'' |accessdate=2011-02-23}}</ref>。

張氏宗祠由該族第22代的張炳南、張志廣、張耀晃及張瑞一於[[嘉慶|嘉慶]]20年興建。30至50年代宗祠曾用作鄉村小學，供村中子弟入讀，而女孩子僅限於晚間時分亦可在祠內就學。1958年當族人在欖口村附近建成華封小學後，祠內前廳的左邊廂房便由校長室改成村公所，歷時長達10年，現作貯物之用；左邊側廊曾用作廚房。

==建築==
張氏宗祠是傳統2進一院3開間式青磚建築，門額金字「張氏宗祠」，大門外掛有[[南海区|南海]]陳景舒<ref>陳景舒，1931年生，[[廣東|廣東]][[佛山|佛山]]人。現為中國書法家協會會員、廣東省書法家協會主席、廣東省文史研究館館員及廣東省書法藝術基金會會長。擅[[隸書|隸書]]，其適遍兩漢簡牘碑刻，旁及歷代書跡，博取眾長，形成出入於[[東漢|東漢]]隸書與[[西漢|西漢]]竹簡之間的個人隸書面目。</ref>署「'''華'''民愛國體 '''封'''爵振家聲」[[對聯|對聯]]；廂房位於天井兩側，後進明間放置歷代[[祖先|祖先]][[神主牌|木主]]的神龕，頂層供奉[[唐朝|唐朝]]丞相[[張九齡|張九齡]]的靈位，山廈村及廣東東莞張氏族人多供奉張九齡為祖先<ref>{{cite web |language=zh-hk |url=http://www.amo.gov.hk/form/shan_ha_tsuen_doc_5.pdf |title=元朗山廈村：張氏宗祠 |publisher=''amo.gov.hk'' |accessdate=2011-02-23}}</ref>；兩側偏廳分別掛有[[咸丰_(年号)|咸豐]]11年「旨賞換花翎」及[[辛酉|辛酉]]科「武魁」牌匾。[[香港政府|香港政府]]於1998年中至1999年末由[[建築署|建築署]]及[[古物古蹟辦事處|古物古蹟辦事處]]負責監督為張氏宗祠進行復修工程，慶祝重修竣工的開光典禮於1999年12月19日舉行。
