'''侯友宜'''{{BD|1957年|6月7日|||CatIdx=H侯}}，[[中華民國|中華民國]][[政治人物|政治人物]]，現任[[新北市市長|新北市市長]]。[[嘉義縣|嘉義縣]][[朴子市|朴子市]]人，曾任[[內政部警政署|內政部警政署]]署長、[[中央警察大學|中央警察大學]]校長、[[新北市|新北市]]副市長，也是[[中華民國史|中華民國史]]上任職[[內政部警政署|內政部警政署]]署長時最年輕者。於[[2018年中華民國直轄市長及縣市長選舉|2018年中華民國直轄市長及縣市長選舉]]中，以30萬票大勝對手[[蘇貞昌|蘇貞昌]]，當選第3任新北市長。<ref>{{Cite web |url=https://www.cna.com.tw/news/firstnews/201811240238.aspx|title=侯友宜當選如履薄冰 蘇貞昌籲綠營檢討反省 |date=2018-11-25  |website=中央社 }}</ref>

== 家庭背景 ==
侯友宜父親是[[軍人|軍人]]，被[[國民政府|國民政府]]徵召，參加[[國共內戰|國共內戰]]，退伍後為豬肉販<ref>{{Cite web |url=http://news.ltn.com.tw/news/politics/breakingnews/2510194 |title=參加後憲模範父親表揚 侯友宜稱讚爸爸只為國家做事 |access-date=2018-08-05 |archive-url=https://web.archive.org/web/20180806024557/http://news.ltn.com.tw/news/politics/breakingnews/2510194 |archive-date=2018-08-06 |dead-url=no }}</ref>。侯友宜1957年6月7日出生於[[嘉義縣|嘉義]][[朴子|朴子]]。家中四兄弟姊妹中，排行第三，次兄侯明鋒為知名醫師，有「南臺灣[[乳癌|乳癌]]第一刀」之稱<ref>{{Cite web |url=https://tw.appledaily.com/headline/daily/20110430/33354799 |title=「乳癌第一刀」侯明鋒1天篩檢400婦 |date=2011-04-30  |website=蘋果日報 }}</ref><ref>{{Cite web |url=http://news.ltn.com.tw/news/life/breakingnews/2505823 |title=高醫4院長交接 「南台灣乳癌第一刀」侯明鋒接任附院代理院長 |date=2018-08-01  |website=自由時報 }}</ref>，曾任[[小港醫院|小港醫院]]院長。

侯友宜與妻子[[任美鈴|任美鈴]]，育有三女一子<ref>{{cite news |author=祁容玉 |last= |first= |date=2018-11-21 |title=兩個女兒被曝光 侯友宜：我也嚇一跳 |url=https://udn.com/vote2018/story/10958/3493287 |newspaper=[[聯合報|聯合報]] |location= |access-date=2018-11-28}}</ref>。1992年5月15日，兒子侯乃維在[[健康幼稚園火燒車事件|健康幼稚園火燒車事件]]中喪生<ref>{{Cite web |url=http://news.ltn.com.tw/news/politics/breakingnews/2407334 |title=侯友宜參選新北市 二哥侯明鋒：全家人都曾反對 |access-date=2018-08-03 |archive-url=https://web.archive.org/web/20180803163543/http://news.ltn.com.tw/news/politics/breakingnews/2407334 |archive-date=2018-08-03 |dead-url=no }}</ref>。

侯友宜是藝人[[納豆_(藝人)|林郁智]]（納豆）表舅<ref>{{cite web|url=https://news.tvbs.com.tw/politics/951937 | title=跟著納豆喊！　侯變身「國民阿舅」走親民風 |date=2018-07-08 |accessdate=2018-07-08}}</ref>。

== 學歷 ==
侯友宜幼年時為[[棒球|棒球]]好手，曾為朴子少棒隊[[捕手|捕手]]<ref>{{cite web|url=https://theme.udn.com/theme/story/6314/219769 | title=小時當鐵捕 長大變神捕 |date=2001-04-08 |website=聯合報 }}</ref>。侯友宜[[嘉義中學|嘉義中學]]畢業後，同時考取[[國防醫學院|國防醫學院]]、[[高雄師範學院|高雄師範學院]]與[[中央警官學校|中央警官學校]]，家人希望他步入[[医学|杏林]]或[[教育界|杏壇]]，但侯友宜看了《十大神探》一書之後，排名第一的主角就是會「壁虎功」的刑警<ref>{{cite web |url=https://www.myplus.com.tw/mag-article.aspx?nid=579 |title=從波麗士到管新北市 突破警察生涯-侯友宜 |date=2012-09-05 |accessdate= |archive-url=https://web.archive.org/web/20180801155114/https://www.myplus.com.tw/mag-article.aspx?nid=579 |archive-date=2018-08-01 |dead-url=no }}</ref>，加上上大學後想放鬆，才會選擇自認為課業壓力較輕的警官學校，刑事警察系畢業。在擔任刑事局副局長任內，侯友宜於中央警察大學犯罪防治研究所進修<ref>{{cite web |url= http://old.ltn.com.tw/2004/new/mar/29/today-p11.htm |title= 侯友宜 用生命追求真相 |date= 2004-03-29 |accessdate=  |archive-url= https://web.archive.org/web/20180801125002/http://old.ltn.com.tw/2004/new/mar/29/today-p11.htm |archive-date= 2018-08-01 |dead-url= no }}</ref>，並於2005年拿到博士學位。<ref>{{cite web|url=https://theme.udn.com/theme/story/6314/219766| title=鐵漢拿博士 白天工作 晚上苦讀 |date=2005-07-13 |website=聯合新聞網 }}</ref>
