'''侯宗宾'''（{{bd|1929年|1月|2017年|11月14日|catIdx=H}}），[[河北|河北]][[南和|南和]]人，[[中国共产党|中国共产党]]及[[中华人民共和国|中华人民共和国]]官员，中共第十三、十四届中央委员会委员，第十四届[[中央纪委|中央纪委]]副书记<ref name=fhw>{{Cite web |url =http://news.ifeng.com/a/20171119/53399730_0.shtml  |title = 中纪委原副书记侯宗宾逝世 |author =  |date = 2017-11-19 |publisher =凤凰网  |language =  |accessdate =  |quote =  }}</ref><ref name=xtw/>。

== 生平 ==
侯宗宾是高中学历。1945年10月参加中国共产党领导的革命，1946年6月加入中国共产党。1945年到1949年，历任河北省南和县村民兵队队长、中队长、一区武装干事、区委委员。1949年随军南下，到[[湖南省|湖南省]]工作。1949年到1952年，历任中共[[慈利县|慈利县]]区委委员、副区长、区长、区委书记。1952年到1953年，任中共慈利县委副书记。1953年到1965年，任湖南[[湘潭|湘潭]]电机厂工会副主席、主席、党委副书记。其间，1956年到1957年在中共中央第五中级党校学习，1963年到1964年在[[中国人民解放军政治学院|中国人民解放军政治学院]]学习<ref name=fhw/><ref name=xtw/>。

1965年，侯宗宾调到[[甘肃省|甘肃省]]任职。1965年到1966年，任甘肃[[兰州|兰州]]综合电机厂党委书记兼厂长。1966年到1969年，在“[[文化大革命|文化大革命]]”中受到冲击，下放劳动。1969年到1975年，任兰州综合电机厂革委会副主任、党委副书记、书记。1975年到1978年，任甘肃省机械工业局副局长、甘肃省燃料化工局副局长、兰州石油化工机器厂党委代书记。1978年到1983年，任[[甘肃省经济委员会|甘肃省经济委员会]]副主任。1983年到1986年，任[[中共甘肃省委|中共甘肃省委]]常委、[[甘肃省人民政府|甘肃省人民政府]]副省长（1983年5月任，1986年5月辞）。1986年到1987年8月，任中共甘肃省委副书记<ref name=fhw/><ref name=xtw/><ref>{{Cite web |url =http://www.xtnews.gov.cn/system/2010/10/19/006393456.shtml  |title =侯宗宾  |author =  |date =2010-10-19  |publisher = 邢台新闻网 |language =  |accessdate =  |quote =  }}</ref>。

1987年8月到1990年3月，侯宗宾任[[中共陕西省委|中共陕西省委]]副书记、[[陕西省人民政府|陕西省人民政府]]代省长、省长。1990年3月到1992年11月，任[[中共河南省委|中共河南省委]]书记。1992年10月到1997年9月，任[[中共中央纪律检查委员会|中共中央纪律检查委员会]]常委、副书记（协助主持日常工作）。1998年3月当选为[[第九届全国人民代表大会|第九届全国人民代表大会]][[全国人民代表大会内务司法委员会|内务司法委员会]]主任委员。他是中共第十三届、十四届中央委员，[[中共十四大|中共十四大]]当选为中央纪律检查委员会委员、常委、副书记。2003年12月[[离休|离休]]，享受正部长级待遇<ref name=fhw/><ref name=xtw>{{Cite web |url = http://www.xtrb.cn/zt/2009-11/23/content_208496.htm |title =本网网友贴出：河北邢台市籍政要简历 |author =  |date =2009-11-23  |publisher = 邢台网 |language =  |accessdate =  |quote =  }}</ref>。

2003年离休后，侯宗宾学习[[书法|书法]]，曾多次向[[欧阳中石|欧阳中石]]等书法家请教，常参加书法展览等活动。2007年1月、5月分别在[[深圳|深圳]][[关山月美术馆|关山月美术馆]]、中央纪委活动中心举行个人书法摄影展。2017年8月3日，中央纪委官网刊发《中央纪委原副书记侯宗宾：“共产党没有任何私利，就是全心全意为人民服务”》一文，在采访中侯宗宾说：“反腐败斗争一定要进一步深入下去，这样才能最终赢得压倒性胜利”，“相信党和国家会越来越好”<ref name=fhw/>。

2017年11月14日，侯宗宾在北京病逝，享年89岁<ref>{{Cite web|url=http://cpc.people.com.cn/n1/2018/0108/c87393-29750157.html|title=侯宗宾同志逝世--新闻报道-人民网|accessdate=2018-01-16|last=1010|work=cpc.people.com.cn}}</ref><ref name=fhw/>。
