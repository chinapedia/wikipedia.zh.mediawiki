'''美國聯邦調查局警察'''（[[英語|英語]]：'''Federal Bureau of Investigation Police''', '''FPU'''）是[[聯邦調查局|聯邦調查局]]的下級機關，主要任務為保護[[聯邦調查局|聯邦調查局]]的設施、財產、人員、雇員、訪客等的安全，並可執行獲授權的法律與行政規則。

== 管轄範圍 ==
聯邦調查局警察在下列機構或地點內執法時，擁有全權防治、逮捕、執行與調查的權限：
* [[約翰·埃德加·胡佛大樓|約翰·埃德加·胡佛大樓]]（聯邦調查局總部）
* 位於[[維吉尼亞州|維吉尼亞州]][[匡提科|匡提科]]的聯邦調查局學院
* 位於[[維吉尼亞州|維吉尼亞州]][[匡提科|匡提科]]的聯邦調查局實驗室
* 位於[[紐約市|紐約市]][[曼哈顿下城|曼哈頓下城]]的行動辦公室
* 位於[[西維吉尼亞州|西維吉尼亞州]][[克拉克斯堡_(西弗吉尼亚州)|克拉克斯堡]]的聯邦調查局犯罪司法資料勤務處

== 專門業務 ==
約有240名的聯邦調查局警察被歸類為專業人員，其中依專長不同分為情報分析師、語言專業人員、科學家、資訊科技專業人員與其他專業技術人員。<ref>[http://www.ojp.usdoj.gov/bjs/pub/ascii/fleo04.txt FBI police statistics]  {{webarchive|url=https://web.archive.org/web/20071214215730/http://www.ojp.usdoj.gov/bjs/pub/ascii/fleo04.txt |date=2007-12-14 }}Wayback Machine<span>.</span></ref><ref>{{Cite web|url=http://www.fbijobs.gov/126.asp|title=FBI Police joblink at FBI Jobse|accessdate=2012-12-30|date=2011-11-19|publisher=Fbijobs.gov|deadurl=yes|archiveurl=https://web.archive.org/web/20121018234610/http://www.fbijobs.gov/126.asp|archivedate=2012-10-18}}</ref>

== 訴訟 ==
2007年8月2日，約100名聯邦調查局警官向美國聯邦索賠法院提起訴訟，求為判決命[[聯邦調查局|聯邦調查局]]支付約100萬元的積欠薪資。原告於訴訟聲明中表示聯邦調查局並未遵守聯邦調查局改革法案中的2002年法規；該法規要求聯邦調查局警察的薪資俸給需比照[[美国特勤局|美國特勤局]]的同官等人員。全案仍在審理中。<ref>{{Cite web |url=http://www.passmanandkaplan.com/FBI-Wage-Claim.shtml |title={title} |access-date=2016-06-30 |archive-url=https://web.archive.org/web/20140809082300/http://www.passmanandkaplan.com/FBI-Wage-Claim.shtml |archive-date=2014-08-09 |dead-url=yes }}</ref>
