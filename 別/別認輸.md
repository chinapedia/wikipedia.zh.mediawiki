《'''別認輸'''》（[[日語|日語]]：{{lang|ja|負けないで}}），是[[日本|日本]][[摇滚樂團|摇滚樂團]][[ZARD|ZARD]]的第6張單曲。1993年1月27日發行。是ZARD的代表作之一和最暢銷的單曲。還是日本經典的應援歌曲。

==簡介==
ZARD的代表歌曲之一，最具知名度和最受歡迎的歌曲。<ref>[http://rn.oricon.co.jp/news/ranking/45838/ {{lang|ja|心に残るZARDの名曲、1位は「負けないで」}}]</ref>

原本這首歌是打算作為專輯的收錄曲目之一，後來才單曲化發行。

[[坂井泉水|坂井泉水]]表示她當初為歌曲填詞時是打算寫關於戀愛內容的，但她聽完[[織田哲郎|織田哲郎]]為她做好的樣本曲子後，突然覺得「像首應援歌」，於是就改寫應援歌歌詞了，為這首歌填詞比平時填詞享受很多。單曲發行的時候，正值團塊Junior世代的學生面對升學考試的時期，因此引起很大的共鳴。<ref>《[[Golden_Best_～15th_Anniversary～|Golden Best ～15th Anniversary～]]》附屬「Golden History 1991〜2006」（坂井泉水談）</ref>

作曲人織田哲郎曾認為坂井唱得很「奇怪」，但這首歌就因為坂井獨特的唱腔而大熱並且成為經典。<ref>[http://www.oricon.co.jp/news/music/45869/ {{lang|ja|『ZARD/坂井泉水さんを偲ぶ会』に織田哲郎、大黒摩希ら「残念」}}]</ref>

被用作同社的女演員[[松雪泰子|松雪泰子]]主演的[[富士電視台|富士電視台]][[日本電視劇|電視劇]]《[[白鳥麗子|白鳥麗子]]》的[[主題曲|主題曲]]。

1993年2月5日，第5次上[[朝日電視台|朝日電視台]]音樂節目《[[MUSIC_STATION|MUSIC STATION]]》演出的ZARD就是演唱這首歌，當時[[町田文人|町田文人]]、[[道倉康介|道倉康介]]和[[池澤公隆|池澤公隆]]已經事實上退出樂團，那次也是ZARD最後一次上電視音樂節目。

單曲發行首週在Oricon公信榜排行第2位，已經創下ZARD當時初動的最高紀錄了。<ref>{{Cite web |url=http://www.musictvprogram.com/corner-ranking-1993-2-8.html |title={{lang{{!}}ja{{!}}1993年2月8日付 シングルチャート TOP10}} |access-date=2010年10月25日 |archive-url=https://web.archive.org/web/20170924143649/http://www.musictvprogram.com/corner-ranking-1993-2-8.html |archive-date=2017年9月24日 |dead-url=yes }}</ref>隨後幾週顯示出強大的後勁，連續十個星期停留在Oricon公信榜前10位。在發行後第四週，終於取得冠軍。<ref>{{Cite web |url=http://www.musictvprogram.com/corner-ranking-1993-3-1.html |title={{lang{{!}}ja{{!}}1993年3月1日付 シングルチャート TOP10}} |access-date=2010年10月25日 |archive-url=https://web.archive.org/web/20180628102423/http://www.musictvprogram.com/corner-ranking-1993-3-1.html |archive-date=2018年6月28日 |dead-url=yes }}</ref>發行後第八週，銷量突破百萬。是1993年度日本單曲銷量第6位。<ref>{{Cite web |url=http://www.musictvprogram.com/corner-ranking-1993.html |title={{lang|ja|1993年間シングルランキング TOP100}} |access-date=2010-10-25 |archive-url=https://web.archive.org/web/20100805065235/http://www.musictvprogram.com/corner-ranking-1993.html |archive-date=2010-08-05 |dead-url=yes }}</ref>總銷量高達164.5萬張，是ZARD銷量最高的單曲。<ref>{{Cite web |url=http://www.musictvprogram.com/corner-ranking-single.html |title={{lang|ja|歴代シングルランキング}} |access-date=2010-10-25 |archive-url=https://web.archive.org/web/20100827183309/http://www.musictvprogram.com/corner-ranking-single.html |archive-date=2010-08-27 |dead-url=yes }}</ref>

這首歌作為應援歌曲廣受歡迎，並且隨著時間的推移越來越經典。1993年的日本，正值[[日本泡沫經濟|泡沫景氣]]爆破不久，並且進入了經濟長期停滯的「[[失去的十年|失去的十年]]」，民眾承受的生活和心理壓力比以前大很多，因此這首歌被譽為是「失去的十年」的應援主題歌。

在日本許多體育運動，尤其是高校[[棒球|棒球]]，都很喜歡將這首歌作為入場式歌曲和應援歌。<ref>[http://www.oricon.co.jp/music/special/070625_01.html {{lang|ja|ZARD特集『ZARDの好きなシングルを一挙紹介！』}}]</ref>1997年，[[文部省|文部省]]將這首歌列入高校音樂教科書。

2007年ZARD的主唱[[坂井泉水|坂井泉水]]意外逝世之後，在當年的《[[24小時電視_「愛心救地球」|24小時電視 「愛心救地球」]]》和[[第58回NHK紅白歌合戰|第58回NHK紅白歌合戰]]中都設有坂井泉水的追悼環節，《別認輸》作為ZARD的經典歌曲都出現了。
