{{作品名稱|name=''Before I Go to Sleep''|tw=別相信任何人|cn=在我入睡前|equiv-hk=tw|equiv-sg=tw}}是一部根據2011年{{link-en|S.J.華森|S. J. == 劇情 ==
每天早上醒來，克莉絲汀（[[妮可·基嫚|妮可·基嫚]] 飾）都身處在陌生的房間，身旁躺著一個不認識的男人。男人耐心說明：我是妳丈夫班恩，妳今年40歲，20年前遭遇嚴重車禍，從此記憶受損。每天她也會接到陌生的醫師來電，要她到衣櫥裡的鞋盒找出被藏起來的祕密。 

原來，克莉絲汀(妮可基曼飾)在每天睡前會用相機拍下今天所發生的的事作為自己的「影像備忘錄」，提醒明天失憶的自己。克莉絲汀藉著影像，一天天重建自己的歷史，但她所發現的細節和班恩、醫師的說法卻經常產生矛盾。這一天她醒來，打開相機，沒想到卻看到影像中的自己說: 別相信班恩……

== 製作 ==
2011年5月1日，知名美國導演[[雷利·史考特|雷利·史考特]]買下了小說《-{[[別相信任何人|別相信任何人]]}-》的電影改編版權，並邀請{{link-en|羅文·約菲|Rowan Joffé}}編劇與執導。2012年2月，[[妮可·基嫚|妮可·基嫚]]加盟此片<ref>{{Cite web |url=http://blogs.indiewire.com/theplaylist/nicole-kidman-in-talks-to-star-in-rowan-joffes-psychological-thriller-before-i-go-to-sleep |title=Nicole Kidman In Talks To Star In Rowan Joffe's Psychological Thriller 'Before I Go To Sleep' |access-date=2014-04-26 |archive-url=https://web.archive.org/web/20131113101757/http://blogs.indiewire.com/theplaylist/nicole-kidman-in-talks-to-star-in-rowan-joffes-psychological-thriller-before-i-go-to-sleep |archive-date=2013-11-13 |dead-url=yes }}</ref>，同年10月31日，[[馬克·史壯|馬克·史壯]]亦加入演員陣容<ref>[http://variety.com/2012/film/news/mark-strong-set-for-sleep-1118061486/ Mark Strong set for 'Sleep']</ref>。2013年2月6日，[[柯林·佛斯|柯林·佛斯]]確定加入劇組<ref>[http://www.hollywoodreporter.com/news/nicole-kidman-wants-colin-firth-391082 Nicole Kidman Wants Colin Firth for S.J. Watson Thriller]</ref>。

== 反响 ==
美国首周末三天仅获得203万美元，分别创下了妮可·基德曼和科林·费斯两人主演影片职业生涯的最低记录。

此片的口碑比较差。根據[[爛番茄|爛番茄]]網站，新鮮度仅有37％<ref>http://www.rottentomatoes.com/m/before_i_go_to_sleep/</ref>；在[[Metacritic|Metacritic]]網站則有42分（滿分100）<ref>http://www.metacritic.com/movie/before-i-go-to-sleep</ref>；[[IMDb|IMDb]]平均為6.5分。英國《{{link-en|每日星報|Daily Star (United Kingdom)}}》的影評人Andy Lea則對[[妮可·基嫚|妮可·基嫚]]的表現給予正面評價<ref>[http://www.dailystar.co.uk/movies/397176/Review-trailer-Before-I-Go-To-Sleep-starring-Nicole-Kidman-and-Colin-Firth Review and trailer: Before I Go To Sleep (15) is a nail-biting tour de force]</ref>。
