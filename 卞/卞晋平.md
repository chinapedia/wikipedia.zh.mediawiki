'''卞晋平'''（{{bd|1953年|10月||}}），[[河北|河北]][[定兴|定兴]]人，[[中华人民共和国|中华人民共和国]]政治人物，[[全国政协委员|全国政协委员]]、副秘书长。

==生平==
卞晋平是[[中国共产党|中国共产党]]党员。1970年5月参加工作。1970年5月到1976年10月，在[[山西省|山西省]][[运城地区|运城地区]]报社工作。1976年11月到1982年2月，为[[山西大学|山西大学]]政治系学生、教师。1982年2月到1984年12月，为[[中共中央党校|中共中央党校]]理论部硕士研究生，获法学硕士学位。1984年12月到1990年5月，任全国政协研究室理论处副处长、秘书局办公室主任。1990年5月到1994年4月，任全国政协研究室副主任兼[[全国政协干部培训中心|全国政协干部培训中心]]副主任。1994年4月到1995年4月，任[[人民政协报|人民政协报]]社总编辑。1995年4月到2002年8月，任全国政协研究室主任兼新闻办公室主任。2002年8月到2006年2月，任[[全国政协办公厅研究室|全国政协办公厅研究室]]主任（副部长级）、[[全国政协机关党组|全国政协机关党组]]成员。2006年3月到2008年10月，任全国政协副秘书长、全国政协机关党组成员。2008年10月起，任[[全国政协文史和学习委员会|全国政协文史和学习委员会]]副主任（驻会）。他是第十届、十一届、十二届全国政协委员，第十届、十一届全国政协副秘书长<ref>{{cite news|title=第十一届全国政协委员名单|url=http://www.indaa.com.cn/zt/2009qglh/zxzs/200903/t20090303_127573.html|accessdate=2013-04-09|newspaper=新华社北京1月25日电 中国人民政治协商会议第十一届全国委员会委员名单(共2237人，2008年1月25日政协第十届全国委员会常务委员会第二十次会议通过)|date=2008年1月25日|deadurl=yes|archiveurl=https://web.archive.org/web/20130928014403/http://www.indaa.com.cn/zt/2009qglh/zxzs/200903/t20090303_127573.html|archivedate=2013年9月28日}}</ref><ref>{{cite news|title=全国政协信息-卞晋平|url=http://www.cppcc.gov.cn/CMS/wylibary/showJcwyxtInfoWylibary.action?tabJcwyxt.guid=11W0014960&wytype=wjwy|accessdate=2013-04-11|newspaper=全国政协委员会}}</ref><ref>{{Cite web |url = http://news.cnr.cn/special/2014lh/zb/mzjs/news/201403/t20140306_515009641.shtml |title =全国政协文史和学习委员会驻会副主任卞晋平简介  |author =  |date =2014-03-06  |publisher =央广网  |language =  |accessdate =  |quote =  }}</ref>。
