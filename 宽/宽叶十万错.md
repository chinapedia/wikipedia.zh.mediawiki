'''宽叶十万错'''（[[学名|学名]]：{{lang|la|''Asystasia gangetica''}}），又稱赤道櫻草，[[爵床科|爵床科]][[十万错属|十万错属]]多年生草本植物

<p>高約0.5米。莖4棱、向上伸延。葉對生，葉柄長1至4厘米，葉片呈卵形至橢圓形，長5至11厘米，寬2至5厘米，全緣或具微小圓齒。在長3至4厘米的總狀花序上，排列出花蕾綻放至果實形成的不同步驟：基部的花早開及先結果，頂部的花最為晚開，花期為九月至翌年二月，花果並存的狀態最常見於12月至翌年2月。蒴果長圓形，長1.8至2厘米。種子2至4顆，邊緣無毛。

=== 原產地 ===
*[[非洲|非洲]]、[[南亞|南亞]] ( [[印度|印度]] ) 、[[馬來西亞|馬來西亞]]。

=== 分佈 ===
*[[非洲|非洲]]、[[印度|印度]]、[[亞洲|亞洲]]、[[台灣|台灣]]、[[泰國|泰國]]、[[中南半島|中南半島]]、[[馬來半島|馬來半島]]、[[雲南|雲南]] ( [[中國|中國]] ) 、及[[中國|中國]][[廣東|廣東]]。

== 亞种 ==
*[[寬葉十萬錯|寬葉十萬錯]] （{{lang|la|''Asystasia gangetica''}} ssp. gangetica）：原變種，原產[[亞洲|亞洲]]及[[印度|印度]]。花朵比較大，可達30–40 mm，且整朵[[花朵|花朵]]有[[白色|白色]]、[[黃色|黃色]]、[[粉紅色|粉紅色]]、[[淡紫色|淡紫色]]、[[淡紫紅色|淡紫紅色]]、[[桃紅色|桃紅色]]等品種。 
*[[小花十萬錯|小花十萬錯]]（{{lang|la|''Asystasia gangetica''}}）：原產[[非洲|非洲]]，于熱帶及亞熱帶地區逸為野生，生长于海拔1,000米以下地区。花冠長1.2至1.5厘米，花較原變種細，最長只有25 mm，白色、5裂、最下瓣中央有一片紫斑。

=== 用途 ===
[[小花十萬錯|小花十萬錯]]於熱帶非洲的[[肯尼亞|肯尼亞]]和[[烏干達|烏干達]]缺乏[[葉菜類|葉菜類]]時，充作[[蔬菜|蔬菜]]食用。傳至東南亞地區後，充作[[牧草|牧草]]。
在非洲作婦女[[分娩|分娩]]期間減緩[[陣痛|陣痛]]的草藥，某些國家用來治療[[風濕|風濕]]和[[氣喘|氣喘]]。

[[寬葉十萬錯|寬葉十萬錯]]：原變種於[[台灣|台灣]]稱'''赤道櫻草'''、[[赤道櫻花|赤道櫻花]]、[[〖|〖]] [[活綠菜|活綠菜]]、[[枸杞菜|枸杞菜]]、[[日本龍葵|日本龍葵]] ( 台灣[[南部|南部]]通稱 ) [[〗|〗]] 、[[日本黑子仔菜|日本黑子仔菜]] ( 台灣[[中北部|中北部]]通稱 ) 、[[貴妃菜|貴妃菜]] ( [[野菜火鍋|野菜火鍋]]通稱 ) 、[[日本枸杞|日本枸杞]]、[[日本枸杞葉|日本枸杞葉]]、[[寬葉馬偕花|寬葉馬偕花]]、[[中國紫蘿蘭|中國紫蘿蘭]]，別名:[[十万錯|十万錯]]、[[盜偷草|盜偷草]]、[[跌打草|跌打草]]、[[烏甜菜|烏甜菜]]、[[活力菜|活力菜]]。充作蔬菜食用，也用它的[[莖|莖]]、[[葉|葉]]來做[[跌打損傷藥|跌打損傷藥]]，治[[骨折|骨折]]、[[跌打腫痛|跌打腫痛]]、[[外傷出血|外傷出血]]和[[散瘀消腫|散瘀消腫]]。有[[清熱解毒|清熱解毒]]、[[涼血止血|涼血止血]]、[[利濕通淋|利濕通淋]]、[[消腫散瘀|消腫散瘀]]、[[接骨止血|接骨止血]]、[[通血脈|通血脈]]、[[治血瘀|治血瘀]]、[[腫瘤|腫瘤]]、[[風寒感冒|風寒感冒]]、[[傷風頭痛|傷風頭痛]]、[[咳嗽氣喘|咳嗽氣喘]]、[[胸悶嘔吐|胸悶嘔吐]]、[[胎動不安|胎動不安]]、[[心血管疾病|心血管疾病]]的功效。主治[[感冒發燒|感冒發燒]]、[[肺炎|肺炎]]、[[支氣管炎|支氣管炎]]、[[咽喉疼痛|咽喉疼痛]]、[[肝炎|肝炎]]、[[黃疸型肝炎|黃疸型肝炎]]、[[膽囊炎|膽囊炎]]，[[濕熱瀉痢|濕熱瀉痢]]、[[泌尿系感染|泌尿系感染]]、[[小便淋痛|小便淋痛]]、[[口腔潰瘍|口腔潰瘍]]、[[白帶|白帶]]、[[腸風下血|腸風下血]]、[[痔瘡腫痛|痔瘡腫痛]]、[[皮膚紅腫|皮膚紅腫]]、[[蜂窩組織炎|蜂窩組織炎]]、[[燒燙傷|燒燙傷]]。


此花亦有[[斑葉栽培種|斑葉栽培種]]，一般稱為：[[斑葉活力菜|斑葉活力菜]]。是一種常見的園藝植物。

=== 栽培 ===
繁殖以[[播種|播種]]、[[扦插法|扦插法]]均可，容易成活。定植後約7～8週即可採收。一般經濟栽培採用[[扦插法|扦插法]]繁殖，截取單節或雙節扦插於穴盤，保持濕潤即可，成活率極高，也可取長枝條直接扦插於畦面上，類似[[空心菜|空心菜]]或[[葉用甘藷|葉用甘藷]]之種植方式，[[扦插|扦插]]後保持畦面濕潤，很容易成活。赤道櫻草生長快速，需肥量甚高，每採收2到3次可施一次有機質肥料，土壤經常保持濕度有助赤道櫻草之生長，使葉片大且嫩，品質較佳，赤道櫻草種植後，根系旺盛，可存活多年，每年冬季開花期只要去除地上部，再施用有機質肥料，就有如更新一般，春天將長出翠綠之新芽。赤道櫻草每公頃每次採收量約2500～3000公斤。[[冬季|冬季]]遇低溫，生長減緩，產量下降快速，且葉片容易轉成[[黃綠色|黃綠色]]，為[[赤道櫻草|赤道櫻草]][[生長|生長]]較[[差|差]]之季節。[[植株|植株]]高約可到[[16|16]]~[[50|50]]公分，[[花期|花期]]：[[10月|10月]]~隔年[[2月|2月]]。

喜陽，也稍耐陰，在疏鬆肥沃排水性好的土壤中生長，適合在[[戶外|戶外]]或[[室內|室內]]種植，容易栽培。

=== 營養價值 ===
營養成份豐富，有[[蛋白質|蛋白質]]（3.96公克/100公克）、[[類胡蘿蔔素|類胡蘿蔔素]]（3.36毫克/100公克）、[[維生素E|維生素E]]（4.4毫克/100公克）、[[鐵|鐵]]（5.7毫克/100公克）等，[[抗氧化|抗氧化]]活性強。嫩梢、嫩葉味道清新、獨具一格，又富含營養價值，別稱「[[活力菜|活力菜]]」。可煮[[蛋花湯|蛋花湯]]、[[煎蛋|煎蛋]]，或做小吃，如[[碗粿|碗粿]]、[[水餃|水餃]]。也可製成[[紫羅蘭捲|紫羅蘭捲]]、[[櫻草粥|櫻草粥]]、[[薑絲櫻草|薑絲櫻草]]、[[香吻櫻草湯|香吻櫻草湯]]等，烹飪方式如同一般蔬菜，可熱炒或慢火燜煨，營養成分豐富，並不會隨季節變化而不同。
