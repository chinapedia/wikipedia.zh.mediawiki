'''答剌麻八剌'''（{{mong|m=ᠳᠠᠷᠮᠠᠪᠠᠯᠠ|Ppp=Darmabala}}；{{bd|1264年||1292年|catIdx=Y}}），[[元朝|元朝]][[元世祖|世祖]]忽必烈太子[[真金|真金]]之次子，[[元武宗|元武宗]]、[[元仁宗|元仁宗]]之父，娶蒙古[[弘吉剌|弘吉剌]]部出身的[[答己|答己]]为正妃，娶郭氏为偏妃。

1286年真金太子去世后[[元世祖|元世祖]]本打算立答剌麻八剌为皇太子，但他因病1292年去世，1307年答剌麻八剌的儿子[[海山|海山]]即位，是为[[元武宗|元武宗]]，元武宗追尊答剌麻八剌为皇帝，为'''答剌麻八剌'''上[[廟號|廟號]]'''順宗'''，汉文[[諡號|諡號]]'''昭聖衍孝皇帝'''。

== 生平 ==
[[至元|至元]]二十二年十二月十日（1286年1月5日），[[真金|真金]]太子去世，[[元世祖|元世祖]]属意孙子答剌麻八剌为继承人。至元二十八年（1291年），元世祖令答剌麻八剌出镇[[怀州|怀州]]，1292年春，元世祖前往[[上都|上都]]，答剌麻八剌留在[[大都|大都]]治病，两个月后病逝。

[[大德|大德]]十一年五月二十一日（1307年6月21日），答剌麻八剌的儿子[[海山|海山]]在[[上都|上都]][[大安阁|大安阁]]即位，是为[[元武宗|元武宗]]，元武宗在即位当天追尊父亲'''答剌麻八剌'''为[[皇帝|皇帝]]，尊母亲[[弘吉剌·答己|弘吉剌·答己]]为[[皇太后|皇太后]]。

== 尊谥庙号 ==
大德十一年九月十一日（1307年10月7日），元武宗为'''答剌麻八剌'''上[[廟號|廟號]]'''順宗'''，汉文[[諡號|諡號]]'''昭聖衍孝皇帝'''。<ref>[http://www.guoxue.com/shibu/24shi/yuanshi/yuas_022.htm 《元史·武宗本纪一》]</ref>

《顺宗皇帝谥册文》<ref>《顺宗皇帝谥册文》作者为程钜夫，册文内容节选自：苏天爵《国朝文类》卷十《册文》。</ref>，内容如下：

{{quote|<br /><blockquote>孝子嗣皇帝臣某，谨再拜稽首言曰：

臣闻显亲所以为子，追远所以厚民。矧必百世祀而位弗隆，为天子父而养弗逮。是宜称秩，以达纯诚。钦惟皇考皇帝渊穆有容，神明莫测。文孙文子，钟至爱于两宫；宜君宜王，膺具瞻于四海。当璧之祥未卜，弃踪之迹已遥。兴言欲报之恩，昊天罔极；对越有成之命，夙夜惟寅。昭哉玄德之升，圣矣生知之异。衍莫衍于昌后嗣而有天下，孝莫于孝配前烈而茂本支。念兹继体之初，益切中心之慕。谨遣摄太尉某官某，奉玉册玉宝，上尊谥曰昭圣衍孝皇帝，庙号顺宗。

伏惟明明降鉴，序于祖宗；攸跻攸宁，永锡祚胤。</blockquote>
}}

== 顺宗实录 ==
[[至大|至大]]元年三月二十日（1308年4月11日），[[元武宗|元武宗]]命翰林国史院纂修《顺宗实录》、《成宗实录》<ref>参见《元史》卷二十二《武宗本纪一》</ref>。

[[皇庆|皇庆]]元年十月二十六日（1312年11月25日），翰林学士承旨[[玉连赤不花|玉连赤不花]]等向[[元仁宗|元仁宗]]进呈《顺宗实录》、《成宗实录》、《武宗实录》<ref>参见《元史》卷二十四《仁宗本纪一》</ref>。其中，《顺宗实录》共有1卷。明朝初年史官修《元史》，参照实录修成《顺宗传》（《睿宗传》、《裕宗传》、《显宗传》、《顺宗传》合为一卷），《顺宗实录》今已失传，其主干内容保存在《顺宗传》中。

玉连赤不花等《进三朝实录表》<ref>《进三朝实录表》作者为程钜夫，内容节选自苏天爵《国朝文类》卷十六《表》。</ref>，内容如下：

{{quote|<br /><blockquote>一人御极，聿严金匮之藏；三后在天，实监玉堂之纪。粤若稽古，克底成书。钦惟皇帝陛下孝友慈仁，温文睿哲。统之垂，业之创，念昔继承；功以著，德以彰，在兹纂录。首崇笔削之任，式宏龟鉴之图。臣等职忝禁林，才非良史。系年系月，岂足尽于先朝；作典作谟，庶有征于今日。臣等以所编成《顺宗皇帝实录》一卷，《成宗皇帝实录》五十六卷，《事目》十卷，《制诏录》七卷，《武宗皇帝实录》五十卷，《事目》七卷，《制诏录》三卷，总计一百三十四卷，缮写已毕，谨具进呈。</blockquote>
}}

== 轶闻 ==
有野史传闻认为，答剌麻八剌的正妃[[答己|答己]]不能生育，他的三个儿子长子[[魏王|魏王]][[阿木哥|阿木哥]]，次子[[元武宗|元武宗]]海山，三子[[元仁宗|元仁宗]]爱育黎拔力八达，以及一个女儿鲁国公主[[祥哥剌吉|祥哥剌吉]]都是偏妃[[郭氏|郭氏]]所生。

根据史书记载，[[郭氏|郭氏]]先嫁给答剌麻八剌，在魏王阿木哥出生之后，[[答己|答己]]才嫁给答剌麻八剌为正妃。

传闻认为，[[答己|答己]]不能生育，于是就抱养[[郭氏|郭氏]]后生的两个儿子海山和爱育黎拔力八达为嫡子，抱养女儿[[祥哥剌吉|祥哥剌吉]]为嫡女，郭氏为了儿女的前途也同意了。

此传闻并非毫无依据，根据《元史》记载，1307年元武宗即位，封阿木哥为[[魏王|魏王]]，赐兽纽金印，1311年三月元仁宗即位，六月，魏王阿木哥前往大都朝觐，仁宗对大臣说：“朕与阿木哥同父而异母，朕不抚育，彼将谁赖？其赐钞二万锭，他勿援例。”<ref>[http://www.guoxue.com/shibu/24shi/yuanshi/yuas_024.htm 《元史·仁宗本纪一》]</ref> 1312年又赐庆元路定海县六万五千户为食邑，对他特别的好，其实天下人都知道阿木哥与元武宗元仁宗同父异母，又何必元仁宗特别点出，这一句话反而让人有些怀疑了。

*  母亲：[[伯藍也怯赤|阔阔真]]王妃，1294年[[元成宗|元成宗]]登基后尊为[[皇太后|皇太后]]，1300年去世后元成宗上[[谥号|谥号]]'''徽仁裕聖皇后'''

'''兄弟'''
*  大哥：[[甘麻剌|甘麻剌]]，1290年封为[[梁王|梁王]]，出镇云南，1292年改封[[晋王|晋王]]，移镇漠北，1302年去世，1324年元泰定帝追尊为皇帝，上庙号[[元顯宗|元顯宗]]，母亲[[伯藍也怯赤|阔阔真]]王妃
*  三弟：[[元成宗|元成宗]][[鐵穆耳|鐵穆耳]]，1293年受皇太子宝，1294年元世祖去世后登基称帝，1307年去世，母亲[[伯藍也怯赤|阔阔真]]王妃

'''妻妾'''
* 正妻 [[答己|答己]]王妃，1307年元武宗登基后尊为[[皇太后|皇太后]]，1311年元仁宗登基后继续尊为皇太后，1320年元仁宗登基后尊为[[太皇太后|太皇太后]]，1322年病逝，1323年[[元英宗|元英宗]]上[[谥号|谥号]]'''昭献元圣皇后'''
* [[郭氏|郭氏]]妃子，生[[魏王|魏王]][[阿木哥|阿木哥]]

'''儿子'''
* 長子 [[魏王|魏王]][[阿木哥|阿木哥]]，郭妃所生，1307年元武宗即位，封为[[魏王|魏王]]，赐兽纽金印。1311年元仁宗赐钞二万锭，1312年，赐庆元路定海县六万五千户为食邑。不久因为获罪贬谪到[[高丽|高丽]]耽罗岛，后来贬谪到[[高丽|高丽]]大青岛，后来因涉嫌谋反被贬谪到山西[[大同市|大同]]。1324年，[[元泰定帝|元泰定帝]]将阿木哥召回京师，1324年六月去世。长子[[阿鲁|阿鲁]]，1330年被[[元文宗|元文宗]]封为[[西靖王|西靖王]]，出镇陕西；次子[[孛罗帖木儿|孛罗帖木儿]]，1324年袭封[[魏王|魏王]]，1353年在河南因为疏于防范被[[红巾军|红巾军]]所杀。
* 次子 [[元武宗|元武宗]][[海山|海山]]，生母[[答己|答己]]王妃，1307年即位称帝，1311年去世
* 三子 [[元仁宗|元仁宗]][[爱育黎拔力八达|爱育黎拔力八达]]，生母[[答己|答己]]王妃，1311年即位称帝，1320年去世

'''女儿'''
* 鲁国公主 [[祥哥剌吉|祥哥剌吉]]（约1283-1331年），下嫁[[碉阿不拉|碉阿不拉]]，1307年元武宗即位，封鲁国大长公主，1311年元仁宗即位，进号皇姐大长公主。热衷书画收藏，有“皇姊图书”印。
