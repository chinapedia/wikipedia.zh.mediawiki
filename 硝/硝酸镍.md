'''硝酸镍'''是镍的[[硝酸盐|硝酸盐]]，化学式为Ni(NO<sub>3</sub>)<sub>2</sub>，以六水合物Ni(NO<sub>3</sub>)<sub>2</sub>·6H<sub>2</sub>O最为常见。

== 性质 ==

硝酸镍是一种绿色[[单斜|单斜]]结晶，有吸潮性，在潮湿空气中迅速[[潮解|潮解]]，在干燥空气中稍微[[风化|风化]]。易溶于水、[[氨水|氨水]]、[[液氨|液氨]]、[[乙醇|乙醇]]，微溶于[[丙酮|丙酮]]，水溶液呈酸性。受热时失去4个结晶水，温度高于110°C时分解为[[碱式盐|碱式盐]]，继续加热则分解为棕黑色[[三氧化二镍|三氧化二镍]]和绿色[[氧化亚镍|氧化亚镍]]的混合物。有毒，有致癌性。与有机物接触可引起燃烧和爆炸。<ref name="zghg">中国化工产品大全 上卷，Bc220 硝酸镍，页208</ref>

== 制备 ==

1、由[[镍|镍]]板与[[浓硝酸|浓硝酸]]发生反应，再经稀释、调节酸度、静置、过滤、滤液酸化、减压蒸发浓缩、冷却结晶、离心分离得到成品。<ref name="zghg" />

: <math>\rm \ 3Ni+8HNO_3\rightarrow 3Ni(NO_3)_2+4H_2O+2NO\uparrow</math>

2、由含镍工业废料酸溶、精制，沉淀出[[氢氧化镍|氢氧化镍]]，再用稀硝酸溶解得到。<ref name="zghg" />

== 用途 ==

用于电镀镍铬合金制件，使制件镀层细致，也用于制造蓄电池和彩釉着色，以及用于制造其他镍盐和镍催化剂。<ref name="zghg" />
