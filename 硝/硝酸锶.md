'''硝酸锶'''是一种无机化合物，化学式为Sr(NO<sub>3</sub>)<sub>2</sub>。

== 性质 ==
硝酸锶是一种无色或白色[[等轴晶系|等轴晶系]]结晶，易溶于水、[[液氨|液氨]]，微溶于无水[[乙醇|乙醇]]和[[丙酮|丙酮]]。加热时先放出氧生成[[亚硝酸锶|亚硝酸锶]]，继续加热则分解为[[氧化锶|氧化锶]]，并放出[[一氧化氮|一氧化氮]]和[[二氧化氮|二氧化氮]]气体：
: 2 Sr(NO<sub>3</sub>)<sub>2</sub> —<sup><u>高温</u></sup>→ 2 SrO + 4 NO<sub>2</sub>↑ + O<sub>2</sub>↑<ref name=wj_bsd>《无机化学（下册）》.陈亚光 胡满成 魏朔 主编.北京师范大学出版集团 北京师范大学出版社. ISBN 978-7-303-11944-8. 第13章 碱金属和碱土金属</ref>
硝酸锶为[[氧化剂|氧化剂]]，与有机物和还原剂接触、摩擦、碰撞及遇火可能引起剧烈燃烧和爆炸。有毒！<ref name="zghg">中国化工产品大全 上卷，Bc225 硝酸锶，页213</ref>

== 制备 ==
1、[[天青石|天青石]]用[[盐酸|盐酸]]溶解后除钙，加入饱和[[碳酸钠|碳酸钠]]溶液沉淀出[[碳酸锶|碳酸锶]]，过滤，用[[硝酸|硝酸]]溶解，再经压滤、滤液蒸发浓缩、结晶、离心分离，得到硝酸锶成品。<ref name="zghg" /><ref name=daijiten>《化学大辞典》.共立出版.1993年</ref><ref name=jikkenkagakukoza>日本化学会編《新実験化学講座 無機化合物の合成II》.丸善.1977年</ref>

:<math>\rm \ SrCO_3+2HNO_3\rightarrow Sr(NO_3)_2+H_2O+CO_2\uparrow</math>

== 用途 ==
硝酸锶用于制造红色[[烟火|烟火]]，用于制造铁路、航道和机场等的信号灯。国防工业中用于制造信号弹、火焰筒、曳光弹。也用于制造电视显像管中的红色发光剂、光学玻璃。还用于医药，制造[[萤光体|萤光体]]。<ref name="zghg" />
