'''硝酸钡'''是[[钡|钡]]的[[硝酸盐|硝酸盐]]，分子式为Ba([[氮|N]][[氧|O]]<sub>3</sub>)<sub>2</sub>。硝酸钡室温下为白色固体，可溶于水。和其他可溶的[[:Category:钡化合物|钡化合物]]一样，硝酸钡有毒，使用时必须注意。

== 应用 ==
*[[巴拉托|巴拉托]]是一种[[密度|密度]]较大的[[炸药|炸药]]，由硝酸钡、[[三硝基甲苯|TNT]]和[[粘合剂|粘合剂]]组成。
*由[[铝粉|铝粉]]和硝酸钡混合得到的[[闪光粉|闪光粉]]具有爆炸性。
*硝酸钡与[[铝热剂|铝热剂]]混合可得到TH3型铝热剂，用于[[手榴弹|手榴弹]]中（[[鋁熱手榴彈|鋁熱手榴彈]]）。
*硝酸钡也被用于[[氧化钡|氧化钡]]的制取、[[真空管|真空管]]工业及制造绿色[[焰火|焰火]]。

== 生产 ==
硝酸钡可通过两种方法生产：
#将大块的[[碳酸钡|碳酸钡]]溶于[[硝酸|硝酸]]，此时含铁杂质沉淀下来。然后[[过滤|过滤]]，蒸發结晶即可得硝酸钡。
#加热[[氯化钡|氯化钡]]与[[硝酸钠|硝酸钠]]的混合溶液，分离结晶得硝酸钡。

== 危险 ==
与所有可溶钡化合物类似，摄取或吸入硝酸钡是有毒的。中毒症状有肌肉收缩（尤其是脸部和颈部）、[[呕吐|呕吐]]、[[腹泻|腹泻]]、腹痛、肌肉颤动、焦虑、虚弱、呼吸吃力、[[心律不齐|心律不齐]]和抽搐。发生在接触硝酸钡几小时或几天后的[[心力衰竭|心力衰竭]]或[[呼吸衰竭|呼吸衰竭]]有可能导致死亡。硝酸钡也可能引起对肾的损伤。<ref>[http://www.jtbaker.com/msds/englishhtml/B0432.htm Barium Nitrate<!-- Bot generated title -->]</ref>.

[[硫酸盐|硫酸盐]]溶液，如[[泻盐|泻盐]]（[[硫酸镁|硫酸镁]]）或[[硫酸钠|硫酸钠]]可用于钡化合物中毒时的[[解毒剂|解毒剂]]，其机理是生成了难溶于水且无毒的[[硫酸钡|硫酸钡]]沉淀。

吸入硝酸钡也可能对[[呼吸系统|呼吸系统]]有刺激性。虽然皮肤或眼部接触不如摄取或吸入毒害大，但以上两者亦有可能导致[[发炎|发炎]]、痛痒、红肿或疼痛。

食入硝酸钡后应当立即求医。如果咨询医生后现象没有改善，食入者应当立即去医院。
