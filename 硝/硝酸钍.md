'''硝酸钍'''是一种[[无机化合物|无机化合物]]，[[化学式|化学式]]为Th(NO<sub>3</sub>)<sub>4</sub>，为极易潮解的放射性氧化剂。
==制备==
[[独居石|独居石]]精矿用50％[[氢氧化钠|氢氧化钠]]溶液在140°C下溶解，后经热水稀释和70°C下保温陈化，得到独居石碱溶饼。用[[硝酸|硝酸]]溶解独居石碱溶饼，经两次萃取和反萃，得硝酸钍溶液：
: LnPO<sub>4</sub> + 3NaOH → Ln(OH)<sub>3</sub> + Na<sub>3</sub>PO<sub>4</sub>
: Th<sub>3</sub>(PO<sub>4</sub>)<sub>4</sub> + 12NaOH → 3Th(OH)<sub>4</sub> + 4Na<sub>3</sub>PO<sub>4</sub>
: Th(OH)<sub>4</sub> + 4HNO<sub>3</sub> → Th(NO<sub>3</sub>)<sub>4</sub> + 4H<sub>2</sub>O
从溶液中根据条件的不同可以结晶出四、五或六水合物。
==物理性质==
无色或白色六角板状易[[潮解|潮解]]结晶，有[[放射性|放射性]]。易溶于水，水溶液呈酸性。溶于[[乙醇|乙醇]]。

硝酸钍水合物的比放射性强度为3.93 kBq/g。<ref>[http://www.sigmaaldrich.com/catalog/product/aldrich/89150?lang=zh&region=CN 硝酸钍 水合物 99%(T) Sigma-Aldrich产品说明书]</ref>

==化学性质==
在空气中加热至500°C时分解为[[二氧化钍|二氧化钍]]。在较低温度可以生成碱式盐。

硝酸钍还能形成络合物，如MgTh(NO<sub>3</sub>)<sub>6</sub>·8H<sub>2</sub>O、CoTh(NO<sub>3</sub>)<sub>6</sub>·8H<sub>2</sub>O、Th(NO<sub>3</sub>)<sub>4</sub>·6DMSO等。<ref name=wj />

==用途==
用于汽灯纱罩发光体和制取其他[[钍|钍]]化合物。
