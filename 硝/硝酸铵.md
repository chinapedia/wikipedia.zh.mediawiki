'''硝酸銨'''是一種化合物，銨陽離子的硝酸鹽(由氨離子和弱酸根組成)。它的化學式是NH<sub>4</sub>NO<sub>3</sub>（簡化為N<sub>2</sub>H<sub>4</sub>O<sub>3</sub>），是一種白色結晶固體，在水中溶解度大。它主要用於農業作為高氮肥料。<sub>[4]</sub> 其他用途是作為採礦、採石和土木建築中使用的爆炸混合物的成分之一。它是{{le|肥料炸彈|ANFO}}主要成分，這是一種普遍的工業[[炸藥|炸藥]]，佔北美炸藥的80％; 類似的配方也用於簡易爆炸裝置(IED俗稱土製炸彈)。由於擔心有被誤用的可能性，許多國家正把它從消費性應用中階段性淘汰不使用。 製作方法: 硝酸銨94%， 柴油6% 木粉少許一些金屬鋁粉，優點，低成本，容易使用。

== 性质 ==

=== 物理性质 ===
硝酸铵是無色斜方或單斜晶體。溶於水、[[甲醇|甲醇]]和[[乙醇|乙醇]]。溶於水時能吸收大量熱能而降低溫度。

=== 化学性质 ===
硝酸铵遇热能够分解。
* 169℃：<ce>NH4NO3={2H2O} +N2O</ce>
* 230℃，反应迅速进行：<ce>4NH4NO3=2N2\text{↑}+2{N2O}+8{H2O}+O2\text{↑}</ce>
* 300℃：<ce>5NH4NO3->[\text{△}]4{N2}\text{↑}+2{HNO3}+9H2O</ce>
* 400℃以上，反应极为猛烈：<ce>2NH4NO3->[\text{△}]4{H2O}+2{N2}\text{↑}+O2\text{↑}</ce>

== 反應生成 ==
硝酸銨在智利最乾旱的地區阿塔卡馬沙漠，被發現為天然礦物（硝石的類銨物和其他硝石礦物例如硝酸鈉），通常是在地表層和/或與其他硝酸鹽，氯酸鹽，碘酸鹽和鹵化物礦物質結合一起。過去硝酸銨在那裡開採，但實際上現在使用的化學物質幾乎100％是合成的。

== 生產 ==
硝酸銨的工業化生產涉及氨與硝酸的酸鹼反應：<sub>[5]</sub>

:HNO<sub>3</sub> + NH<sub>3</sub>→NH<sub>4</sub>NO<sub>3</sub>

'''合成方法：'''

1. 由[[氨水|氨水]]與[[硝酸|硝酸]]作用製成。  NH<sub>3</sub>·H<sub>2</sub>O<sub>(aq)</sub>+HNO<sub>3(aq)</sub>→NH<sub>4</sub>NO<sub>3(aq)</sub>+H<sub>2</sub>O<sub>(l)</sub>

2. 把[[氨氣|氨氣]]吸收在[[硝酸|硝酸]]中製成。 NH<sub>3(g)</sub>+HNO<sub>3(aq)</sub>→NH<sub>4</sub>NO<sub>3(s)</sub>

氨以無水形式（即氣體形式）使用，硝酸則是高濃度的。這種反應是劇烈的，由於其高放熱性的物質本性。通常溶液形成後的濃度約為83％，剩餘的水會依據等級不同，被蒸發成濃度95％至99.9％的硝酸銨（AN熔體）。然後將95％至99.9％的AN熔體在噴霧塔中製成“顆粒”或小珠，或者通過在旋轉的滾筒中噴霧和翻滾而製成稍大顆粒。小顆粒或大顆粒都可以進一步乾燥，冷卻，然後包膜以防止結塊。這些小或大顆粒是典型的AN商業產品。

該方法所需的氨可通過哈伯法(Haber process)從氮和氫獲得。由哈伯法生產的氨再通過奧士華法(Ostwald  process)可氧化成硝酸。另一種生產方法是變形的硝磷銨法(Odda process)：

:Ca(NO<sub>3</sub>)<sub>2</sub> + 2NH<sub>3</sub> + CO<sub>2</sub> + H<sub>2</sub>O → 2NH<sub>4</sub>NO<sub>3</sub> + CaCO<sub>3</sub>

產物碳酸鈣和硝酸銨可以純化分離或者以鈣化硝酸銨的形式出售。硝酸銨也可以通過置換反應製造：

:(NH<sub>4</sub>)<sub>2</sub>SO<sub>4</sub> + Ba(NO<sub>3</sub>)<sub>2</sub>→2NH<sub>4</sub>NO<sub>3</sub> + BaSO<sub>4</sub>↓(s)

:NH<sub>4</sub>Cl + AgNO<sub>3</sub>→NH<sub>4</sub>NO<sub>3</sub> + AgCl(s)

=== 反應 ===
硝酸銨可與金屬氫氧化物反應，釋放氨並形成鹼金屬硝酸鹽：

:NH<sub>4</sub>NO<sub>3</sub> + MOH→NH<sub>3</sub> + H<sub>2</sub>O + MNO<sub>3</sub>（M = Na，K）

硝酸銨加熱時不會殘留殘留物：

:NH<sub>4</sub>NO<sub>3</sub>→N<sub>2</sub>O(g) + 2H<sub>2</sub>O

硝酸銨也是可以由排放入大氣中的NO，SO<sub>2</sub>和NH<sub>3</sub>化合形成，也是PM10的次要成分<sub>[6]</sub>

== 應用 ==

=== 肥料 ===
硝酸銨是NPK(氮、磷、鉀)比例為34-0-0（氮含量34％）的重要肥料[8]它比尿素（46-0-0）濃度低，這使得硝酸銨在運輸上稍微處於劣勢。但硝酸銨的優勢在於它比尿素更穩定，並且不會迅速將氮氣逸散到大氣中。當氣候溫暖的時候，尿素最好在雨水來到之前儘快使用，或用土壤覆蓋，以盡量減少氮氣的流失。<sub>[9] [10]</sub>

=== 爆破 ===
硝酸銨本身不是一種爆炸物[11]，但當與主要爆炸物如疊氮化物或燃料如鋁粉或燃料油組合時，它很容易形成具有不同性能的爆炸性混合物。

硝酸銨在耐撞擊方面而言是極其鈍感的[[炸藥|炸藥]]，比安全炸藥[[C4炸药|C4炸药]]更为鈍感。一支工業8#[[雷管|雷管]](起爆[[C4炸药|C4炸药]]只需要6#)都不足以起爆混合了敏化劑的硝酸銨。硝酸銨是最難引爆的[[硝酸|硝酸]]炸藥，[[撞擊感度|撞擊感度]]是：50kg鎚50cm落高，0%爆炸。相比起著名炸藥[[硝化甘油|硝化甘油]]的200g鎚，20cm落高，100%爆炸的感度，可见硝酸铵的钝感。而且硝酸銨一旦溶於水，起爆感度更是大大下降，根本是人不可能撞擊引爆的。

然而硝酸铵对高温的耐受力就很差了。前文提到，一百六七十多摄氏度就可以让硝酸铵分解，这几乎相当于遇到火星就爆炸。相比之下C4炸药即使是扔进火里也只会缓慢燃烧。

==== 與燃油混合 ====
肥料炸彈(ANFO)是94％硝酸銨（“AN”）和6％燃油（“FO”）的混合物，被廣泛用作散裝工業炸藥。<sub>[12]：1</sub>ANFO低成本和易於使用的優點用於採煤，採石，金屬開採等初階應用，不論在耐水性，氧氣平衡，高爆震速度和小直徑性能等優點都大於傳統工業炸藥。<sub>[12]：2</sub>

==== 恐怖活動 ====
硝酸銨爆炸物被用於1970年威斯康辛大學麥迪遜分校的史特林廳爆炸事件上，1995年的奧克拉荷馬城轟炸機，2011年德里爆炸事件，2013年海得拉巴爆炸事件和2011年在奧斯陸進行的轟炸。

2009年11月，在巴基斯坦前邊界省份馬拉克蘭省（西北邊境省）上部地區，下轄區，斯瓦特，奇拉爾和馬拉坎德地區，禁止使用硫酸銨，硝酸銨和硝酸銨鈣肥料 – 根據西北邊境省政府的報導，這些化學品被武裝分子用來製造炸藥。由於這些禁令，“使火柴點火的原料-氯酸鉀-已經超過化肥成為叛亂分子的首選爆炸物。”<sub>[13]</sub>

=== 利基應用 ===
因為硝酸銨在水中的溶解是高度吸熱的，所以被使用在某些即溶冰敷袋上。它也被用於獨立爆炸的“燃料”，如胍硝酸鹽<sub>[14] [15]</sub>，日本的[[安全氣囊|安全氣囊]]大廠高田公司製造的安全氣囊的氣體中使用硝酸銨作為5-氨基四唑的低成本（但不太穩定）替代品，因沒有加入[[乾燥劑|乾燥劑]]，使得硝酸銨久了之後受潮及高溫而變質，氣囊作動時會因力道過大而使零件碎片飛散，反而造成車上駕駛及乘客因此受傷甚至死亡。在造成14人死亡後因不安全被召回<sub>[16]</sub> 受影響的汽車品牌甚廣，全球召回的車輛達上千萬輛。<ref>[http://www.appledaily.com.tw/realtimenews/article/new/20160216/797099/ 高田氣囊風暴 國內大規模召修28萬輛次]</ref>

硝酸铵除了工業用和軍用[[炸藥|炸藥]]，及其主要的用途做[[化肥|氮肥]]之外，也可用於[[殺蟲劑|殺蟲劑]]、[[冷凍劑|冷凍劑]]、[[氧化氮|氧化氮]]吸收劑，製造[[笑氣|笑氣]]、煙火等。

== 安全、管理和儲存 ==
健康和安全數據記載在供應商提供的安全資料表上，在網際網路上可以找到。<sub>[17]</sub>為了因應數起導致許多人死亡的爆炸，美國環境保護署（EPA），職業健康與安全（OSHA）以及酒精，煙草和槍枝管理局聯合頒布了安全準則。<sub>[18]</sub>

加熱或任何點火源可能會引起劇烈的燃燒或爆炸。<sub>[19]</sub>硝酸銨可與可燃和還原材料反應，因為它是一種強氧化劑。雖然主要是用於肥料，但亦可用於爆炸物。有時候用來開鑿土地做農場池塘<sub>[20]</sub> <sub>[21]</sub>，硝酸銨也用於改變其他爆炸物的爆炸速度，例如把三硝基甲苯(TNT)轉變成阿馬托炸藥(一種無煙炸藥)。硝酸銨的儲存和處理有許多可行的安全準則<sub>[22]</sub>。硝酸銨不應該存放在易燃物質附近。硝酸銨與某些物質如氯酸鹽，無機酸和金屬硫化物不相容，接觸會導致劇烈或甚至猛烈的分解。<sub>[23]</sub>

硝酸銨的臨界相對濕度為59.4％，高於此值將吸收大氣中的水分。因此，將硝酸銨儲存在密封容器中是很重要的。否則它會吸濕結成塊。硝酸銨可以吸收足夠的水分液化。將硝酸銨與某些其他肥料混合可以降低臨界相對濕度。<sub>[24]</sub>這種材料有被作為爆炸物的潛在性，促使了監管措施。例如，在澳大利亞，“危險品條例”於2005年8月生效，以執行處理此類物質的許可。<sub>[25]</sub>許可證僅授予申請者（工業），並採取適當的安全措施，以防止任何濫用。其他用途，如教育和研究目的可以考慮使用，但個人使用則不允許。有執照處理該物質的人員仍須經授權人員監督，並在執照獲得許可前應該要通過安全和國家警察檢查。

== 健康危害 ==
健康和安全數據顯示在物質安全資料表中，可從供應商處獲得，並可在網際網路上找到。<sub>[27]</sub>

硝酸銨對健康並不十分有害，通常用於化肥產品。<sub>[27] [28] [29]</sub>

硝酸銨的半數致死劑量(LD50)為2217毫克/公斤(有毒物質的質量和試驗生物體重比)，<sub>[30]</sub>其比較約為食用鹽的三分之二。

== 災害 ==
硝酸銨在加熱時分解成一氧化二氮和水蒸氣（不是爆炸性反應）;然而，它可以被爆炸引起爆炸分解。由於硝酸銨本身可提供燃燒所需氧氣，所以大量儲存的硝酸銨會有很高的火災風險，而且也可能被引爆：如1947年德克薩斯州城市發生的大災難，那次災難導致硝酸銨的儲存和處理規定產生重大變更。<sub>[31]</sub>

導致爆炸的兩大主要類型是：

*由於引爆機制的轉變而產生爆炸。先是從一堆爆炸物中的炸藥包起爆，接著由爆炸外殼射進易爆物引爆，或由爆炸混合物與一堆易爆物接觸引爆。例子是Kriewald，Morgan（今日Sayreville，新澤西州），Oppau和Tessenderlo。

*爆炸是由火災蔓延到硝酸銨本身（德克薩斯城，布雷斯特，奧克代爾PA），或由火災中的硝酸銨與可燃物的混合物（Repauno，切諾基，Nadadores）造成。火災至少需要在一定的溫度上，完成從燃燒到爆炸的轉換（一種稱為“爆燃到爆轟轉變”的現象）。純的、緊湊的硝酸銨(AN)是穩定的，很難點燃，並且許多情況下，甚至不穩定的AN都沒有在火中爆炸。
