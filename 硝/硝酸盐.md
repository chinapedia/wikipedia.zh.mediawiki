硝酸鹽是一個多原子離子其分子式NO<sub>3</sub><sup>−</sup>和分子量62.0049克/mol。硝酸鹽同樣描述為有機官能團RONO<sub>2</sub>。這些硝酸酯是一專業炸藥。

'''CP#3'''是硝酸根离子NO<sub>3</sub><sup>−</sup>形成的盐。许多[[金属|金属]]都能形成硝酸盐，包括无水盐或[[水合物|水合物]]。

== 结构 ==
与[[碳酸根|碳酸根]]离子类似，硝酸根离子的[[分子构型|分子构型]]是[[平面三角形分子构型|平面三角形]]，并且具有以下[[共振式|共振式]]：

== 制备 ==
[[硝酸|硝酸]]与金属、金属[[氧化物|氧化物]]或[[碳酸盐|碳酸盐]]反应是最简单的制备硝酸盐的方法。某些含水的硝酸盐如[[硝酸铍|Be(NO<sub>3</sub>)<sub>2</sub>]]，[[硝酸镁|Mg(NO<sub>3</sub>)<sub>2</sub>]]和[[硝酸铜|Cu(NO<sub>3</sub>)<sub>2</sub>]]加热後會分解 ，因此得不到相应的无水硝酸盐。无水硝酸盐可通过下列途径制得：<ref>张青莲等。《无机化学丛书》第四卷 氮磷砷分族。北京：科学出版社。</ref>
*在液态[[四氧化二氮|N<sub>2</sub>O<sub>4</sub>]]中反应：
::Ni(CO)<sub>4</sub> + N<sub>2</sub>O<sub>4</sub> → Ni(NO<sub>3</sub>)<sub>2</sub> + 2NO + 4CO
*在纯HNO<sub>3</sub>-N<sub>2</sub>O<sub>5</sub>或液态[[五氧化二氮|N<sub>2</sub>O<sub>5</sub>]]中反应：
::TiCl<sub>4</sub> + 4N<sub>2</sub>O<sub>5</sub> → Ti(NO<sub>3</sub>)<sub>4</sub> + 2N<sub>2</sub>O<sub>4</sub> + 2Cl<sub>2</sub>
*与[[卤素|卤素]]的硝酸盐在低温反应。如[[硝酸氯|硝酸氯]]ClNO<sub>3</sub>：
::TiCl<sub>4</sub> + 4ClNO<sub>3</sub> −(-80℃)→ Ti(NO<sub>3</sub>)<sub>4</sub> + 2Cl<sub>2</sub>

某些金属还可形成通式为MO<sub>x</sub>(NO<sub>3</sub>)<sub>y</sub>的碱式硝酸盐，如[[硝酸氧铋|BiO(NO<sub>3</sub>)<sub>2</sub>]]。

大多数硝酸盐为离子型晶体，易溶于水。某些无水盐具有挥发性。

== 性质 ==
=== 热不稳定性 ===
硝酸盐分解的产物可以分为以下几类：
*[[碱金属|碱金属]]和[[碱土金属|碱土金属]]的硝酸盐分解产生[[亚硝酸盐|亚硝酸盐]]和氧气：
: 2 KNO<sub>3</sub> → 2 KNO<sub>2</sub> + O<sub>2</sub>↑
*[[金属活动性|金属活动性顺序表]]在[[Cu|铜]]之前的'''其他'''硝酸盐分解产生金属氧化物和[[氮氧化物|氮氧化物]]：
: 2 Pb(NO<sub>3</sub>)<sub>2</sub> → 2 [[氧化铅|PbO]] + 4 [[二氧化氮|NO<sub>2</sub>]]↑ + O<sub>2</sub>↑
: 4 Bi(NO<sub>3</sub>)<sub>3</sub> → 2 Bi<sub>2</sub>O<sub>3</sub> + 12 NO<sub>2</sub>↑ + 3 O<sub>2</sub>↑
: Th(NO<sub>3</sub>)<sub>4</sub> → ThO<sub>2</sub> + 4 NO<sub>2</sub> + O<sub>2</sub>↑
* 金属活动性位于Cu之后的产生金属[[单质|单质]]：
: 2 AgNO<sub>3</sub> → 2 Ag + 2 NO<sub>2</sub>↑+ O<sub>2</sub>↑
* 含有还原性阳离子的硝酸盐热分解时，会和硝酸根发生[[氧化还原反应|氧化还原反应]]：
: NH<sub>4</sub>NO<sub>3</sub> → N<sub>2</sub>O、N<sub>2</sub> + ……
: Mn(NO<sub>3</sub>)<sub>2</sub> → MnO<sub>2</sub> + ……
: Ce(NO<sub>3</sub>)<sub>3</sub> → CeO<sub>2</sub> + ……
* 需要注意的是，一些物质在分解至最终产物之前，会产生中间产物，如[[硝酸镧|La(NO<sub>3</sub>)<sub>3</sub>]] → LaONO<sub>3</sub> → [[氧化镧|La<sub>2</sub>O<sub>3</sub>]]等。
* 铜(II)、钛(IV)、钴(III)、锆(IV)的无水硝酸盐受热可以以气态的形式存在。
=== 氧化性 ===
硝酸盐都是氧化剂，其固态或液态的氧化性远大于溶液中的氧化性。如[[硝酸钾|硝酸钾]]溶液和[[碳|碳]]不会反应，但是硝酸钾固体和碳加热，会剧烈反应（反应热使硝酸钾熔化）。

== 检验 ==
硝酸根离子可在[[酸性|酸性]]介质中，通过和铁（II）反应产生棕色环加以定性检出。参见[[棕色环实验|棕色环实验]]。

总反应为：
::3Fe<sup>2+</sup> + NO<sub>3</sub><sup>−</sup> + 4H<sup>+</sup> → 3Fe<sup>3+</sup> + NO + 2H<sub>2</sub>O

虽然该反应已有很久的历史，但其机理却是不久前经[[分光光度法|分光光度法]]及[[电位滴定法|电位滴定法]]的系统研究后才弄清楚的：<ref>I. R. Epstein, K. Kustin and L. J. Warshaw, [[美国化学会志|''J. Am. Chem. Soc.'']] '''102''':3751 (1980).</ref>
::Fe<sup>2+</sup> + NO<sub>3</sub><sup>−</sup> + 2H<sup>+</sup> → Fe<sup>3+</sup> + NO<sub>2</sub> + H<sub>2</sub>O
::Fe<sup>2+</sup> + NO<sub>2</sub> + H<sup>+</sup> → Fe<sup>3+</sup> + HNO<sub>2</sub>
::Fe<sup>2+</sup> + HNO<sub>2</sub> + H<sup>+</sup> → Fe<sup>3+</sup> + NO + H<sub>2</sub>O
::Fe<sup>2+</sup> + NO → FeNO<sup>2+</sup>
::2NO<sub>2</sub> + H<sub>2</sub>O → HNO<sub>2</sub> + NO<sub>3</sub><sup>−</sup> + H<sup>+</sup>
::2HNO<sub>2</sub> → NO + NO<sub>2</sub> + H<sub>2</sub>O
::NO + NO<sub>3</sub><sup>−</sup> + H<sup>+</sup> → NO<sub>2</sub> + HNO<sub>2</sub>

其中棕色环是由FeNO<sup>2+</sup>（第四步）引起的，[[速控步|速控步]]则是最后一步。

硫酸-二苯胺法也能用于硝酸根的检验。

== 配位化合物 ==
硝酸根和金属离子可以按多种方式配位，包括单齿、双齿、叁齿或端梢、桥式等。
