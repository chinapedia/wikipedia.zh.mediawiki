'''妮姬塔·丹妮絲'''（'''{{lang-en|Nikita Denise}}'''，{{bd|1976年|7月25日|}}），是一名[[捷克|捷克]][[色情演員|色情演員]]、[[成人模特兒|成人模特兒]]及[[色情片|成人影片]][[導演|導演]]。<ref name="iafd"/>

== 生平 ==
丹妮絲在1998年自家鄉[[捷克斯洛伐克|捷克斯洛伐克]]搬至[[多倫多|多倫多]]，開始在[[脫衣舞|脫衣舞]]俱樂部工作。她的第一次演出是在2000年《North Pole 13》中與[[狄倫·戴|狄倫·戴]]（Dillon Day）的合作。<ref>{{cite web|url=http://avn.com/index.cfm?objectid=D7147661-0FCD-5672-11963F71C2B51D3D |title=Czech Mate: Nikita Denise Returns |accessdate=2008-02-03 |author=Nelson X |date=2008-02-01 |publisher=[[成人影帶新聞|AVN]] |deadurl=yes |archiveurl=https://web.archive.org/web/20080209010604/http://www.avn.com/index.cfm?objectID=D7147661-0FCD-5672-11963F71C2B51D3D |archivedate=2008-02-09 }}</ref><ref>{{cite web|url= http://xfanz.com/news/87932|title= Nikita Denise Visits Playboy Radio|accessdate= 2008-02-03|author= Tod Hunter|date= 2007-12-18|publisher= xfanz.com}}</ref>

2006年，妮姬塔獻出導演的處女作《Nikita's Extreme Idols》。這部片也是她休息超過一年後的首度演出。<ref>{{cite web|url=http://www.ainews.com/story/10150/|title=Nikita Denise Directorial Debut: Nikita's Extreme Idols|accessdate=2006-06-13}}</ref>

2006年7月24日，妮姬塔參加[[理查·克里斯帝|理查·克里斯帝]]（Richard Christy）主持的節目《Inside the Porn Actor's Studio》。<ref name="imdb">{{cite web | title = Nikita Denise| publisher = IAFD.com|url =http://www.imdb.com/name/nm0219196/| accessdate = 2008-11-10 }}</ref> 

丹妮絲已演出超過350部影片<ref name="iafd">{{cite web | title = Personal Bio Nikita Denise| publisher = IAFD.com|url =http://www.iafd.com/person.rme/perfid=NikitaDenice/gender=f/Nikita-Denise.htm| accessdate = 2008-01-10 }}</ref>，並贏得多項大獎，包括2002年[[成人影帶新聞獎|成人影帶新聞獎]]年度女表演者。
