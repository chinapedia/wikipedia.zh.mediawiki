'''寢屋川市'''（{{jpn|j=寝屋川市|hg=ねやがわし|rm=Neyagawa shi|f=y}}）是位於[[日本|日本]][[大阪府|大阪府]]東北部[[淀川|淀川]]左岸的城市，屬于大阪府北河内地區，距離大阪市市中心約15公里。現為[[施行時特例市|施行時特例市]]，預定於2019年升格為[[中核市|中核市]]。

轄區西部靠近淀川一側為平原，海拔高度僅2至3公尺，東半部則為延伸自{{Tsl|ja|生駒山地}}的丘陵地，

城市名來自于流經市內的一級河川{{Tsl|ja|寝屋川|寢屋川}}，但寢屋川的名稱則是來自轄內的地名「寢屋」，該地過去由於設有{{Tsl|ja|高野街道}}上供往返旅客住宿的「寢屋」，因而得名。

== 历史 ==

在令制國時代屬於[[河內國|河內國]]，現在的轄區西北部屬於[[茨田郡|茨田郡]]，南部屬於[[讚良郡|讚良郡]]，東部屬於[[交野郡|交野郡]]。

在[[江戶時代|江戶時代]]大部分區域為幕府直屬或旗本的領地，也有部分區域為{{Tsl|ja|加納藩}}和[[小田原藩|小田原藩]]領地

19世紀明治維新後，這些區域曾先後隸屬大坂裁判所（後來的[[大阪府|大阪府]]）、[[河內縣_(日本)|河內縣]]、[[堺縣|堺縣]]，1881年又隨著堺縣被併入大阪府後成為大阪府的轄區。

1889年實施[[町村制|町村制]]，現在的寢屋川市轄區在當時分屬{{Tsl|ja|友呂岐村}}、{{Tsl|ja|九個荘町|九個莊町|九個莊村}}、{{Tsl|ja|水本村}}、{{Tsl|ja|豊野村_(大阪府)|豐野村_(大阪府)|豐野村}}、'''{{Tsl|ja|寝屋川村|寢屋川村}}'''共五個行政區劃；1943年寢屋川村、豐野村、友呂岐村、九個莊町[[市町村合併|合併]]為新設立的'''寢屋川町'''，1951年寢屋川町升格改制為'''寢屋川市'''，1961年，位於東側的水本村也併入寢屋川市，形成現在寢屋川市的轄區範圍。

2000年日本開始實施[[特例市|特例市]]制度，寢屋川市也在2001年即成為特例市；2015年日本廢除特例市制度，但也將[[中核市|中核市]]的人口門檻降低至20萬，寢屋川市也因此規劃將在2019年升格為中核市。

在21世紀初[[平成大合併|平成大合併]]期間，也曾有人提出將寢屋川市、[[枚方市|枚方市]]、[[交野市|交野市]]合併為人口超過70萬人的大型城市，甚至可以進一步成為[[政令指定都市|政令指定都市]]，但此計畫並未有任何實際的進展。

== 交通 ==

轄內的鐵路交通有[[京阪電氣鐵道|京阪電氣鐵道]][[京阪本線|京阪本線]]及[[西日本旅客鐵道|西日本旅客鐵道]][[片町線|片町線]]，以南北向分別穿過轄區的中部及東部區域。京阪本線的{{Tsl|ja|寝屋川市駅|寢屋川市車站}}為轄內最主要的車站，此車站周邊也是市內最主要的商業區。

快速道路則有{{Tsl|ja|第二京阪道路}}自東部穿過，往北可至[[京都市|京都市]]，往南可接[[近畿自動車道|近畿自動車道]]。

== 教育 ==

在1950年代以前，轄內最初只有{{Tsl|ja|寝屋川市立東小学校|寢屋川市立東小學校|東}}、{{Tsl|ja|寝屋川市立西小学校|寢屋川市立西小學校|西}}、{{Tsl|ja|寝屋川市立南小学校|寢屋川市立南小學校|南}}、{{Tsl|ja|寝屋川市立北小学校|寢屋川市立北小學校|北}}、{{Tsl|ja|寝屋川市立第五小学校|寢屋川市立第五小學校|第五}}，共五間小學；1960年代後隨著經濟發展開始有大量住宅區出現，轄內也才開始大量設置小學，現在轄內共有24間小學。

而中學部分，全市共有12間公立中學，其中前面十間中學則是依據成立時間直接依序命名為{{Tsl|ja|寝屋川市立第一中学校|寢屋川市立第一中學校|第一}}、{{Tsl|ja|寝屋川市立第二中学校|寢屋川市立第二中學校|第二}}、{{Tsl|ja|寝屋川市立第三中学校|寢屋川市立第三中學校|第三}}、{{Tsl|ja|寝屋川市立第四中学校|寢屋川市立第四中學校|第四}}、{{Tsl|ja|寝屋川市立第五中学校|寢屋川市立第五中學校|第五}}、{{Tsl|ja|寝屋川市立第六中学校|寢屋川市立第六中學校|第六}}、{{Tsl|ja|寝屋川市立第七中学校|寢屋川市立第七中學校|第七}}、{{Tsl|ja|寝屋川市立第八中学校|寢屋川市立第八中學校|第八}}、{{Tsl|ja|寝屋川市立第九中学校|寢屋川市立第九中學校|第九}}、{{Tsl|ja|寝屋川市立第十中学校|寢屋川市立第十中學校|第十}}，直到1980年代後新設立的中學才不再以數字命名。

高等教育則有{{Tsl|ja|大阪電気通信大学|大阪電氣通信大學}}和[[攝南大學|攝南大學]]兩所大學。
