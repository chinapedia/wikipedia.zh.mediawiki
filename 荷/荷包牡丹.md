'''荷包牡丹'''（[[学名|学名]]：''{{lang|la|Lamprocapnos spectabilis}}''），又名'''兔儿牡丹'''、'''鱼儿牡丹'''、'''铃儿草'''、'''铃心草'''、'''瓔珞牡丹'''、'''荷包花'''、'''蒲包花'''、'''土当归'''<ref name="GMSY">{{zh}}[[赵学敏|赵学敏]]（1765年）《本草纲目拾遗》</ref>、'''活血草'''<ref name="GMSY"/>、'''華鬘草（日本）'''，是[[紫堇科|紫堇科]][[荷包牡丹屬|荷包牡丹屬]]的[[多年生|多年生]][[草本|草本]]植物，原产於[[中国|中国]]北部，花多为桃红色与白色的复色花。廣泛分布在[[亞洲|亞洲]]東部，自[[西伯利亞|西伯利亞]]南部到[[日本|日本]]的地區。19世纪时荷包牡丹才被引进[[欧洲|欧洲]]，而在[[北美|北美]]一般只能靠人工栽培。<ref>{{en}}{{cite web |url=http://www.efloras.org/florataxon.aspx?flora_id=1&taxon_id=109863 |title=Dicentra in Flora of North America |accessdate=2009-02-01 |author=Kingsley R. Stern |publisher=eFloras.org}}</ref>

荷包牡丹花形玲珑可爱，花色艳丽多彩，叶丛错落美观，是[[温带|温带]]地区很受欢迎的[[观赏植物|观赏植物]]，可用来布置[[花境|花境]]和[[花坛|花坛]]，同时也是很好的[[切花|切花]]材料。

荷包牡丹的近缘种包括[[兜状荷包牡丹|兜状荷包牡丹]]（''{{lang|la|D. cucullaria}}''）、[[加拿大荷包牡丹|加拿大荷包牡丹]]（''{{lang|la|D. canadensis}}''），而同属主要的观赏花卉有[[大花荷包牡丹|大花荷包牡丹]]（''{{lang|la|D. macrantha}}''）和[[美麗荷包牡丹|美麗荷包牡丹]]（''{{lang|la|D. formosa}}''）等，主要的栽培种包括白花荷包牡丹（''{{lang|la|Dicentra spectabilis}}'' {{lang|en|'Alba'}}）。

== 命名 ==
[[File:Dicentra_Spectabilis.jpg|thumb]]
荷包牡丹的[[种加词|种加词]]''{{lang|la|spectabilis}}''意为“艳丽的”，属名''{{lang|la|Dicentra}}''出自[[希腊语|希腊语]]，意为“双[[花距|花距]]的”，而荷包牡丹的每一个中文名都有其独特的来历。荷包牡丹的花语是答应追求、答应求婚<ref>{{zh}}{{cite web |url=http://www.yanlinghm.com/news_8042 |title=荷包牡丹花语 |publisher=鄢陵花木网 |date=2008-11-20 |accessdate=2009-02-01 }}{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，这源於民间的流传有关荷包牡丹得名的美丽传说“玉女思君”<ref>{{zh}}{{cite web|url=http://www.esgweb.net/Article/Class58/Class124/Class125/200704/15085.htm | title=“荷包牡丹”的传说 |publisher=e景苑 | date=2007-04-25 |accessdate=2009-02-01 }}</ref><ref>{{zh}}{{cite journal zh |url=http://www.storyparty.com/Story_Detail.aspx?ID=12510 |author=胡正山 |title=《荷包牡丹》 |journal=中国花卉园艺 |date=2002 |issue=4 |accessdate=2009-02-01 |deadurl=yes |archiveurl=https://web.archive.org/web/20070528015727/http://www.storyparty.com/Story_Detail.aspx?ID=12510 |archivedate=2007-05-28 }}</ref><ref>{{zh}}{{cite web |url=http://dqy.wms.edu/onews.asp?/345.html |author=丁琦娅 |title=荷包牡丹·中国玫瑰 |publisher=丁琦娅随笔 |date=2008-06-29 |accessdate=2009-02-01 |archiveurl=https://web.archive.org/web/20160304213351/http://dqy.wms.edu/onews.asp?%2F345.html |archivedate=2016-03-04 |deadurl=yes }}</ref>。瓔珞牡丹则是取其[[瓔珞|瓔珞]]般的外形。鱼儿牡丹是古人所赋，出自[[宋朝|宋朝]]诗人[[周必大|周必大]]的《詠魚兒牡丹並序》：“鱼儿牡丹，得之湘中，花红而蕊白，状似双鱼，累累相比，枝不胜压，而下垂若俯首然，鼻目可辨，叶与牡丹无异，亦多二月开，因是得名。”“花嫔”出自周必大《詠魚兒牡丹》：“天教姚黄主芳菲，合有宫嫔次列妃。玉颈圆瑳宜粉面，霞裙深染学翚衣。枝头窈窕鱼双贯，风里翩跹凤对飞。莫把根苗方芍药，留春不似送将归。”铃儿草则是源自[[明成祖|明成祖朱棣]]与[[马铎|马铎]]的对联“风吹不响铃儿草，雨打无声鼓子花。”<ref>{{zh}}[[梁章鉅|-{zh-hans:梁章钜;zh-hant:梁章鉅}-]]（1842年）《巧对录》卷四</ref>

== 特征 ==
荷包牡丹株高30－90厘米，根状地下茎肉质，根茎粗壮，茎紫红色，光滑无毛，茎基有多枚黄棕色大鳞片；叶柄长，叶对生，长约20厘米，为二回三出羽状复叶（“三出”指一叶柄上集生有三小叶），[[:wikt:缺刻|缺刻]]，有基部[[楔形|楔形]]的倒卵形深裂片，背面有白色粉末，叶形与[[牡丹|牡丹]]叶相似；花茎长1.5厘米，基部有2枚细小尖头苞片，花同向下垂，形如心形[[荷包|荷包]]，3－15枚花顶生为拱形总状花序，每枚2.5－5厘米长，披针形桃红色萼片2枚，长4－4.5毫米，花瓣共4枚，长约2.5厘米，外层2枚花瓣大多为桃红色，变种有白色，花瓣基部包囊状，顶部变狭外翻，内层2枚花瓣狭长突出，顶部聚缩，外部白色，内部紫红色；6枚雄蕊分为2列，[[子房上位|子房上位]]，1室，花柱细长，盾状柱头2裂。花期为早春至盛夏。蒴果为细长圆形，种子细小，先端有细毛。

中文名“荷包牡丹”就源於其花如荷包，叶似牡丹叶，而英文俗名之一“滴血的心”（{{lang|en|bleeding heart}}）也是因其形似[[心脏|心脏]]。

== 栽培 ==
[[File:Dicentra.spectabilis.closeup.jpg|thumb]]
[[File:Lamprocapnos_spectabilis_pods.jpg|thumb]]
荷包牡丹生长於温带较凉爽地区，喜阳光散射的半阴环境，较耐寒，怕高温暴晒，因此盛夏休眠期要置於通风良好的荫蔽处。喜湿润，但怕积水，因此要保持土壤半墒，过湿则烂根，过干则叶黄，而盛夏和冬季[[休眠期|休眠期]]要保持土壤微润；喜多[[腐殖质|腐殖质]]的疏松肥沃中性或弱碱性土壤，而沙土和粘土也可生长，不过会生长不良；喜肥，宜施用少量有机肥或三元[[复合肥|复合肥]]，花蕾显色後停肥，休眠期不施肥。<ref name="ShiYang">{{zh}}{{cite journal zh|author=佚名|date=2005|title=《荷包牡丹的莳养》|journal=科学大观园|volume=|issue=7|url=http://www.bast.net.cn/bjkpzc/tszr/zwdg/yhxd/35598.htm|accessdate=2009-02-01|deadurl=yes|archiveurl=https://web.archive.org/web/20090109154915/http://www.bast.net.cn/bjkpzc/tszr/zwdg/yhxd/35598.htm|archivedate=2009-01-09}}</ref>可[[播种|播种]]、[[扦插|扦插]]或[[營養繁殖|分株]]繁殖，可在早春或深秋栽培。嫁接荷包牡丹可选用多年生[[芍药|芍药]]作[[砧木|砧木]]。<ref>{{zh}}{{cite web|url=http://www.foodmate.net/topic/324/105074.html | title=根接法救活荷包牡丹 |publisher=食品伙伴网 |date=2008-03-08 |accessdate=2009-02-01 }}</ref>虫害有[[蚜虫|蚜虫]]、[[蛞蝓|蛞蝓]]、[[蜗牛|蜗牛]]和[[介壳虫|介壳虫]]，病害有[[叶斑病|叶斑病]]。<ref>{{zh}}{{cite web |date=2006-03-05 |title=荷包牡丹的栽培技术 |publisher=农博花木 |url=http://flower.aweb.com.cn/2006/3/5/1120060305034600.shtml |accessdate= 2009-02-02 }}</ref>

荷包牡丹有数个栽培种，其中包括白花荷包牡丹以及[[英国|英国]]栽培种“金心”（''{{lang|la|Dicentra spectabilis}}'' {{lang|en|'Goldheart'}}）。“金心”於1997年培育，特点是其嫩黄色的叶子在盛夏时会变为黄绿色。

== 药用 ==
[[File:Lamprocapnos_spectabilis_seeds.jpg|thumb]]
荷包牡丹根可作为解毒活血镇痛的[[中药材|中药材]]使用，在《岭南采药录》中记载荷包牡丹根能散血、消疮毒、除风、和血，药性为辛、苦、温，归肝经，内服可用於治疗[[金疮|金疮]]、[[疮毒|疮毒]]、[[胃痛|胃痛]]，外敷可用於治疗跌打肿痛<ref name="ShiYang"/>，正如汪连仕《采药书》中所载“用其根捣汁，酒冲服之，令人沉醉，金疮之圣药也。”<ref>{{zh}}{{cite web |url=http://www.tcmlib.com/zy/html59/showdetail-353832342ce88db7e58c85e789a1e4b8b9e6a0b92c7a79.html |title=荷包牡丹根的功效 |publisher=意古中医 |accessdate=2009-02-01 }}{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

不过，荷包牡丹全草有潜在毒性<ref>{{en}}{{cite web|url=http://www.pfaf.org/database/plants.php?Dicentra+spectabilis | title=''Dicentra spectabilis'' |publisher=Plants For A Future |accessdate=2009-02-01 }}</ref>，部分人接触此植物会有皮肤刺痛感，源自{{link-en|荷包牡丹碱|Bicuculline}}和{{link-en|原阿片碱|Protopine}}等[[异喹啉|异喹啉]]类生物碱的作用<ref>{{en}}{{cite web |url=http://www.uvm.edu/~pass/perry/oh63harm.html |title=Potentially Harmful Perennials OH 63 |accessdate=2009-02-01 |author=Leonard P. Perry |date=1997-02-28 |publisher=Perry's Perennial Pages}}</ref><ref name=Leikin2007>{{en}}{{Cite book en | editor =Jerrold B. Leikin; Frank P. Paloucek | date = 2007 | title = Poisoning and toxicology handbook | pages = 886 | isbn = 1420044796 | publisher = CRC | location = Boca Raton, Fla }}</ref>，因此要注意用量，否则可能出现[[呕吐|呕吐]]、[[腹泻|腹泻]]、[[呼吸衰竭|呼吸衰竭]]、[[心脏麻痹|心脏麻痹]]等症状。
