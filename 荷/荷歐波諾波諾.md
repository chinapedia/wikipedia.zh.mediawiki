'''荷歐波諾波諾'''（Ho'oponopono，ho-o-pono-pono）是[[夏威夷歷史|古夏威夷]]一種基于[[和解|和解]]和[[寬恕|寬恕]]的[[替代疗法|替代疗法]]。相似的傳統在[[南太平洋|南太平洋]]諸島亦可見到，包含[[薩摩亞|薩摩亞]]、[[大溪地|大溪地]]、以及[[紐西蘭|紐西蘭]]等地。傳統上，荷歐波諾波諾由治療祭司{{efn|又稱為[[卡胡納|卡胡納]]。}}對患者執行，但現在多由患者家中長者或患者本人自行執行。

== 玻里尼西亞文化 ==
[[File:Oceania_(World-Factbook).jpg|thumb]]
在許多[[玻里尼西亞|玻里尼西亞]]文化中，人們認為一個人的過失{{efn|稱作''hara''或''hala''。}}會導致疾病。有些人認為是過失觸怒了眾神，有些人認為是過失吸引了邪神，另也有些人認為是過失導致的罪引發疾病。然而，他們認為進行某些「解開錯誤」的儀式可用以贖罪<ref name="Oliver"/>{{rp|157}}。

在[[南太平洋|南太平洋]]的[[萬那杜|萬那杜]]群島，人們認為疾病通常考歸因於不當性行為或憤怒。一位當地人說：「如果你的憤怒持續了二至三日，疾病將降臨在你身上」<ref name="Parsons"/>{{rp|55}}。而治療此種疾病的方法則是透過懺悔，可由病患或其家屬進行之。若無人懺悔，則該病患可能會死亡。萬那杜人相信，隱瞞是疾病動力的來源。當過失被懺悔後，它就無法再控制該人<ref name="Parsons"/>{{rp|61}}。

[[夏威夷人|夏威夷人]]、[[所羅門群島|所羅門群島]]的[[蒂蔻皮亞島|蒂蔻皮亞島]]居民、以及[[庫克群島|庫克群島]]的[[拉罗汤加岛|拉罗汤加岛]]居民相信父親的罪業會落在其子女身上。如果某人的子女生病了，則其父母涉嫌爭吵或不當行為。除了疾病，社會失序亦可能導致土地不育或其他災難<ref name="Parsons"/>{{rp|70}}。而和諧只能通過懺悔和道歉得到恢復。

在[[库克群岛|库克群岛]]的[[普卡普卡岛_(库克群岛)|普卡普卡岛]]，傳統上需要對患者進行此類懺悔儀式，以決定合適的行動方針來進行治療<ref name="Parsons"/>{{rp|151}}。

相似的傳統在亦可在[[薩摩亞|薩摩亞]]<ref name="Parsons"/>{{rp|12}}、[[大溪地|大溪地]]<ref name="Parsons"/>{{rp|159}}、以及[[紐西蘭|紐西蘭]][[毛利人|毛利人]]社群等地見到<ref name="Parsons"/>{{rp|217}}<ref name="Buck"/>{{rp|405-406}}<ref name="Handy"/>{{rp|242}}。

== 傳統習俗 ==
[[File:Hala_lei.JPG|thumb]]的果實製成的花環。這與[[摩洛凱島|摩洛凱島]]的荷歐波諾波諾儀式傳統有關]]

荷歐波諾波諾在夏威夷語字典中被定義為「心理清洗：通過祈禱、討論、認罪、悔罪、並相互賠償和[[寬恕|寬恕]]的家庭會議矯正人際關係<ref name="Pukui, Elbert"/>。」從字面上看，「荷歐」（ho{{okina}}o）是個助詞，用以將名詞轉為動詞，就像英文中的「to」放在名詞前面的用法一樣。在這裡，它將名詞「{{le|波諾 (夏威夷語)|pono|波諾}}」（pono）轉變為動詞，其定義為：
{{cquote|
善良、正直、道德、道德品質、正確或適當的程序、卓越、幸福、繁榮、福祉、利益、真實情況或性質、職責；正確的、正義的、公正的、良性的、公平的、有益的、成功的、井井有條、準確、緩解、減輕；應該，必須，必要的。}}

至於「波諾波諾」（Ponopono）的定義是則是：「恢復正常；塑造、修正、修改、調整、改正、調節、安排、糾正、整理、使有序。」

夏威夷學者{{le|瑪麗·卡薇娜·卜奎|Mary Kawena Pukui}}寫道，荷歐波諾波諾是古代夏威夷的習俗<ref name="Pukui, Haertig, Lee"/>{{rp|61-62, 67}}，並被當代夏威夷長輩口述歷史所證實<ref name="Chai"/>{{rp|47-50}}。卜奎在她出版於1958年的書首次記錄她童年時{{efn|卜奎生於1895年。}}的相關經驗與觀察<ref name="Pukui, Handy"/>{{rp|184-185}}。作家{{le|馬克斯·弗里丹·隆恩|Max Freedom Long}}曾在1917年至1926年之間居於夏威夷，他並在1936年出版的書中敘述了有關夏威夷家庭執行傳統的荷歐波諾波諾的紀錄<ref name="Long1936"/>{{rp|246-248}}<ref name="Long1948"/>{{rp|250-252,279,303}}{{efn|雖然該書中之敘述並非所有都是夏威夷傳統，但這些特定章節是對於荷歐波諾波諾真實的描述。}}

雖然「荷歐波諾波諾」這個詞尚未被使用，但早期的夏威夷史學家紀錄了相關的信仰，亦即疾病是因為違反了{{le|卡普 (夏威夷)|kapu|卡普}}戒律所造成的，而且該疾病無法被治癒除非患者得以贖罪。贖罪儀式通常有祈禱祭司（kahuna pule）或治療祭司（kahuna lapa{{okina}}au）的協助，尋求神祇<ref name="Kamakau"/>{{rp|95}}<ref name="Malo"/>{{rp|75}}或與患者有糾紛者的寬恕<ref name="Titcomb"/>。

卜奎將此習俗描述為用以導正破碎的家庭關係的延伸家庭會議。有些家庭每天或每週集合一次，以防患於未然<ref name="Chai"/>{{rp|52-54}}。其餘家庭則當有人生病時才會集合，他們並相信疾病是憤怒、內疚、指責和缺乏寬恕的壓力所造成的<ref name="Pukui, Haertig, Lee"/>{{rp|60}}。庫普納·娜娜·威利（Kupuna Nana Veary）寫道，當她家中任何一位兒童生病時，她的祖母就會詢問其父母「你們做了什麼事？」他們相信唯有獲得全體家庭成員完整的寬恕，才能消除該疾病<ref name="Veary"/>{{rp|34}}。

== 儀式 ==
荷歐波諾波諾藉由獲取煩惱的原因和源頭來糾正、恢復和維持家庭成員之間以及與他們的神的良好的關係。在通常情況下，此儀式由家族中最資深的成員進行，並聚集全家人。如果此家庭無法自行解決此問題，則他們將尋求一個受人尊敬的局外人協助。

儀式從祈禱開始。接著陳述問題，並討論過失。家庭成員被期待一起合作解決問題，而不是“緊抓著過失不放”。他們可採取一個或多個的靜默期做為情緒與傷病糾結的反思，每個人的情感都被認可。然後進行自白、懺悔和寬恕。參與成員互相釋放（稱作「kala」）並彼此放手。他們切斷了過去（{{okina}}oki），並以儀式性的大餐，稱作「pani」，作為其終結。該大餐中的食物通常包含{{le|卡拉海帶|kala seaweed}}以作為釋放的象徵<ref name="Pukui, Haertig, Lee"/>{{rp|60-80}}。

在[[摩洛凱島|摩洛凱島]]上的卡胡納馬拉威利威利家庭，他們給人用[[林投|哈拉樹]]果實做成的{{le|花圈 (夏威夷)|lei (Hawaii)|花圈}}來象徵荷歐波諾波諾儀式的完成<ref name="Lee"/>{{rp|49}}。
