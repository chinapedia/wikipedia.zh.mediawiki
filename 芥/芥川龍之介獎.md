'''芥川獎'''，正式名稱為'''芥川龍之介獎'''，乃是紀念日本[[大正時代|大正時代]]的文豪[[芥川龍之介|芥川龍之介]]（1892-1927）所設立的[[文學獎|文學獎]]，並由主辦單位[[文藝春秋|文藝春秋]]頒發給「[[雅文學|雅文學]]（純文學）」新人作家的一個獎項<ref>Fukue, Nastuko, "[http://www.japantimes.co.jp/text/nn20120214i1.html?utm_source=feedburner&utm_medium=feed&utm_campaign=Feed%3A+japantimes+(The+Japan+Times%3A+All+Stories) Literary awards run spectrum]", ''[[Japan_Times|Japan Times]]'', 14 February 2012, p. 3.</ref>；現今的主辦單位已改為[[日本文學振興會|日本文學振興會]]。

1935年（昭和10年）由[[菊池寬|菊池寬]]在《[[文藝春秋|文藝春秋]]》4月號（追悼直木三十五號）的連載專欄「話之紙屑桶」中提出以當年2月去世的[[直木三十五|直木三十五]]、1927年去世的芥川龍之介的名稱來創立新人獎的構想。從此以後，每年舉行兩次的選拔活動。上半年度（12月1日～5月31日之前公開發表的作品）是在7月中旬進行選拔、8月中旬頒獎，刊載於《[[文藝春秋(雜誌)|文藝春秋]]》9月號。下半年度（6月1日～11月30日之前公開發表的作品）是在翌年1月中旬進行選拔、同年2月中旬頒獎，在《文藝春秋》3月號刊載。該獎項在二次大戰時1945年一度中斷過，後來在1949年重新開始進行選拔活動。

芥川獎的遴選並非採取公開招募的方式，而是根據選拔委員的協議來決定候補及得獎人選。這些選拔委員從報章[[雜誌|雜誌]]上，新人作家或無名作家所發表的純文學短篇作品中，挑選出最優秀的作品予以獎勵，得獎者頒贈一百萬日圓的獎金和[[懷錶|懷錶]]一只。芥川龍之介獎的評選會在東京[[築地|築地]]的老舖[[料亭|料亭]]「[[新喜樂|新喜樂]]」1樓舉行，頒獎儀式及得獎者的記者會於1個月在[[東京會館|東京會館]]舉行。現今的評審委員為[[島田雅彥|島田雅彥]]、[[奧泉光|奧泉光]]、[[小川洋子|小川洋子]]、[[川上弘美|川上弘美]]、[[堀江敏幸|堀江敏幸]]、[[高樹信子|高樹信子]]、[[宮本輝|宮本輝]]、[[村上龍|村上龍]]、[[山田詠美|山田詠美]]九位。

芥川獎是日本文學的最高榮譽之一<ref>NATSUKO FUKUE. [http://www.japantimes.co.jp/text/nn20120214i1.html "Literary awards run spectrum"], japantimes.com, Feb. 14, 2012</ref><ref>[http://www.britannica.com/EBchecked/topic/11844/Akutagawa-Prize "Akutagawa Prize."] ''[[Encyclopædia_Britannica|Encyclopædia Britannica]]''. ''[[Encyclopædia_Britannica_Online|Encyclopædia Britannica Online]]''. Encyclopædia Britannica Inc., 2012. Web. 14 Feb. 2012. </ref>，其中芥川獎是[[純文學|純文學]]獎的代表獎項，而[[直木獎|直木獎]]則是[[大眾文學|大眾文學]]的代表獎項；芥川獎以鼓勵新人作家為宗旨，直木獎則是給予已出書的大眾文學作家一項榮譽的肯定。
