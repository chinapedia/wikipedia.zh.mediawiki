'''晋阳古城遗址'''，位于[[山西省|山西省]][[太原市|太原市]][[晋源区|晋源区]][[晋源镇|晋源镇]]附近<ref name="a1">{{Cite web |url=http://www.sxkaogu.net/webshow/aritcleDetail.shtml?articleId=1986 |title=山西晋阳古城遗址|accessdate=2016-07-14 |author= |date=2015-02-09 |work= |publisher=山西省考古研究所网站|language=Zh-hans }}</ref>，春秋至五代时期的城池遗址。2001年，列入[[第五批全国重点文物保护单位|第五批全国重点文物保护单位]]。

== 历史 ==

晋阳古城为[[春秋时期|春秋]]末期[[赵国|赵国]]立国之前的发祥地和建国初期的第一座[[都城|都城]]，由晋国正卿[[赵鞅|赵鞅]]命家臣[[董安于|董安于]]创建，至前386年[[赵敬候|赵敬候]]迁都于[[赵邯郸故城|邯郸城]]。[[智伯|智伯]][[晉陽之戰|水灌晋阳]]、豫让刺赵、[[三家分晋|三家分晋]]等重大历史事件发生于此。

秦汉三国时期为军事重镇，曾为秦[[太原郡|太原郡]]、两汉魏晋并州治所以及西汉韩国、西汉代国、东汉太原国、西晋太原国都，西晋末年由刘琨扩建；前秦末年，苻丕于此称帝；北魏末期，晋阳城成为尔朱氏和高氏遥控朝政的“霸府”所在，后成为东魏、[[北齐|北齐]]的别都和实际政治军事中心，此时晋阳城得到高氏政权大力营建；隋初为并州治所和晋王杨广封地，杨广对晋阳城进行进一步营建，后为唐国公封地。

隋末继承唐国公的[[李渊|李渊]]于[[晋阳起兵|晋阳起兵]]即是此处。[[唐朝|唐朝]][[貞觀_(唐朝)|贞观]]十一年（637年），[[并州_(北朝)|并州]][[长史|长史]][[李勣|李勣]]筑东城<ref name="a2">《[[新唐书|新唐书]]·志第二十九·地理三》北都，天授元年置......晋阳宫在都之西北，宫城周二千五百二十步，崇四丈八尺。都城左汾右晋，潜丘在中，长四千三百二十一步，广三千一百二十二步，周万五千一百五十三步，其崇四丈。汾东曰东城，贞观十一年长史李勣筑。两城之间有中城，武后时筑，以合东城。宫南有大明城，故宫城也......</ref>，为[[太原县_(隋朝)|太原县]]县治，西城则为[[晋阳县_(秦国)|晋阳县]]县治。[[武则天|武则天]]即位后升为北都，后并州长史[[崔神庆|崔神庆]]建中城，将东西两城连为一体<ref name="a1"/>。西北有[[晋阳宫|晋阳宫]]<ref name="a2"/>。

唐中宗废北都，唐玄宗开元十二年复为北都，[[天宝_(唐朝)|天宝]]元年改“北京”，唐肃宗初年罢京号，又复为北都，终唐不改；唐末，晋王李克用以晋阳为都与朱温建立的后梁对抗，李克用之子李存勖灭后梁建立后唐，设晋阳为北京，立宗庙于晋阳；后唐末期，坐镇晋阳的河东节度使石敬瑭灭后唐建后晋，继续以晋阳为北京；后晋末期，河东节度使刘知远于晋阳称帝灭后晋，建立后汉，再以晋阳为北京；郭威灭后汉后，刘知远之弟刘崇于晋阳建立北汉，北宋建立后，赵匡胤、[[赵光义|赵光义]]相继讨伐[[北汉|北汉]]，并于太平兴国四年因北汉皇帝投降而占领晋阳，赵光义于五月十八下令火烧晋阳并引[[汾河|汾河]]、晋水冲灌城池废墟，古城被彻底毁灭。

明朝初期，晋王[[朱棡|朱棡]]曾计划在晋阳古城废墟上重建太原，因大风吹坏工程设施而放弃，后在古城遗址南部建[[明太原县城|太原县城]]（今晋源东街、北街、西街、南街四村所在晋源老城），古城遗址上形成古城营、棘针、城北、罗城、壑里、东关、南城角、晋阳堡、东城角、梁家寨、赵家山、箱子、北河下、北瓦窑等村落。<ref>[http://www.tydao.com/2007/taiyuan/70830-yizhi.htm 晋阳古城遗址的现状与保护]</ref>。20世纪中叶，考古发现石刻、古城墙遗址<ref>[http://www.wenwuchina.com/episteme/list9/detail139/30148.html 太原市晋阳古城遗址保护管理条例]</ref>。

== 遗址概况 ==

古城面积大约20[[平方公里|平方公里]]，遗址以自然耕地为主，分为古城遗址和寺观墓葬遗址。古城遗址为中心，周围依山势分布众多的寺观建筑，同时在西山脚下缓坡地带，广泛分布着墓葬遗址。

其古城西部的西山上有[[天龙山石窟|天龙山石窟]]和[[圣寿寺|圣寿寺]]、开化寺遗址和[[蒙山大佛|蒙山大佛]]及摩崖石刻遗址、[[童子寺|童子寺]]遗址和龙山大佛及摩崖造像遗址、昊天观遗址和龙山石窟、太山龙泉寺、硫磺沟石窟、明仙寺、悬瓮寺遗址、天龙山高欢避暑宫遗迹、店头古村、蒙山北汉避暑行宫遗址、豫让桥遗址、窦大夫祠、净因寺、多福寺、烈石寒泉等古迹，山脚下及古城遗址周围有晋祠古建筑群、华严经石刻、崇福寺遗址、奉圣寺遗址、王郭村台骀庙、青阳河台骀庙、明秀寺、九龙庙、旧惠明寺阿育王塔、华塔寺遗址及石像埋藏点、雨花寺遗址、上生寺遗址、下生寺遗址等古迹。<ref>[http://www.sxcr.gov.cn/wmsx/show.php?itemid=734 晋阳古城遗址 ]</ref>。
