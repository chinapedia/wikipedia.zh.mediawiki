《'''電影版 聲之形'''》（{{lang-ja|映画 聲の形 The Shape of Voice}}，{{lang-en|''A Silent Voice''}}）是一部於[[2016年電影|2016年]]上映的日本[[動畫電影|動畫]]電影，由[[山田尚子|山田尚子]]執導，[[吉田玲子|吉田玲子]]編劇，劇情改編自[[大今良時|大今良時]]所著的漫畫《[[聲之形|聲之形]]》。片中角色由[[西屋太志|西屋太志]]設計，而配樂則由{{link-ja|牛尾憲輔|agraph}}負責<ref name="Stingray"/>。[[入野自由|入野自由]]、[[早見沙織|早見沙織]]、[[悠木碧|悠木碧]]、[[小野賢章|小野賢章]]及[[金子有希|金子有希]]等人為片中的主要角色配音。

劇情講述一名少年石田將也在小學時帶頭欺凌一名聽障的少女西宮硝子，不料導致老師連坐懲罰而使得友情破裂。多年後，因有霸凌他人的前科，導致名聲狼藉而遭受同學的排擠、且已就讀高中的將也決定去向硝子表達歉意。

電影於2016年9月17日在[[日本|日本]]首映，並於2017年2月至6月間在全球各地上映。《電影版 聲之形》收穫了影評人的普遍好評，而票房總額則達23億日元（1,956萬美元）<ref name="gross"/>。該片於[[第40回日本電影金像獎|第40回]][[日本電影金像獎|日本電影金像獎]]頒獎典禮上榮獲優秀動畫作品獎。

== 劇情 ==
石田將也就讀的小學轉來了一名聽障的少女西宮硝子。將也在班上帶頭欺凌硝子，並多次毀損硝子的[[助聽器|助聽器]]。一次，硝子的家長因助聽器多次毀壞而找來學校，詢問班上是否有人霸凌硝子。其他同樣參與霸凌行為的同學將責任全推到將也身上，不甘的將也試圖將其他人招出但不果，還弄哭了一名女同學。該事件後，將也遭到了其他同學的排擠，且原先的兩個朋友島田一旗與廣瀨啓祐也反過來帶頭欺凌將也。

高中時，將也因有霸凌他人的前科而遭受同學的排擠，並選擇孤立自己，當做對自己所作所為的懲罰。這時，將也萌生了自殺的念頭，並決定在輕生前歸還硝子小學時的筆記本，請求她的諒解。兩人見面時，將也用手語對硝子道歉，使硝子頗為吃驚。將也表示，他想與她重新當朋友。

在學校，將也目睹了同學永束友宏被另一名同學欺負。將也不情願地走出來面對兩人，他借出自己的腳踏車為友宏解圍，但後來才意識到腳踏車有去無回。將也試圖去手語教室見硝子，打算與她一同去橋上用麵包餵鯉魚，但被結絃擋在教室門口外。結絃告訴將也硝子不在。回家的路上，友宏牽著將也的自行車朝他走來，並表示他費盡全力才從稻田中尋回。兩人最終成了朋友。一次，將也再度前往手語教室拜訪硝子，但仍遭到結絃的阻擋。結絃自稱是硝子的男友。

隔天，將也前去公園接他的姪女瑪麗亞，並發現結絃睡在遊樂器材中，看起來無家可歸。那天晚上，將也不情願的請結絃回自己家吃晚餐並借宿一晚。結絃在半夜時偷偷離開，將也發覺並跟了上來。途中，將也告訴結絃他曾如何對待硝子，並正努力彌補她。結絃也告訴將也，他是硝子的妹妹而非男友。將也與硝子一同搭乘火車前去拜訪小學同學佐原美世子。佐原看到將也與硝子走在一起時頗為訝異。為了避開佐原與硝子的談話，將也謊稱自己要去廁所，並碰上了正在發傳單的小學同學植野直花。

將也發現硝子正在橋上餵鯉魚，便走上前。當將也去買更多麵包時，硝子送給將也一份禮物，並鼓起勇氣向他告白。但由於她因聽障而影響到的發音問題，使將也聽不太懂她所說的內容。硝子沮喪的回家，告訴結絃她告白失敗了。在學校，將也的同學真柴智得知硝子被霸凌的過去，十分痛恨惡霸的他感到義憤填膺。然而，將也小學時的同學川井美樹告訴他，將也正是那個兇手。眾人聚集在橋上爭論了起來，最後不歡而散，只剩下將也、結絃與硝子三人。

夏天，將也與硝子一同前往煙火大會。硝子自言自語，說她必須早點離開。硝子離開不久後，結絃請將也去她家拿相機。當將也走進房裡時，發現硝子站在陽台並準備往下跳，將也為了救她而跌到樓下的河裡昏迷。將也昏迷後的一個晚上，硝子夢到將也說他正在死亡。硝子驚醒，跑到河邊哭泣。奇蹟醒來的將也跑到河邊找硝子。他宣稱一切都是他的錯，硝子不必自責。將也為過去的錯誤道歉，並希望她能幫他好好活下去。

將也到家後，直花透露，在將也墜樓並掉到河裡後，是島田與廣瀨聯手將他救起的，儘管他們不希望將也知道。將也發現，在他為了救硝子而墜樓昏迷後，眾人對他的看法逐漸改變。最終，將也與朋友們一同參加學校的文化祭，終於覺得自己已經得到救贖了。

: 高中生，在小學時帶頭霸凌硝子，但在東窗事發後遭其他同樣參與霸凌的同學推卸責任，不甘的將也試圖將其他人招出但不果，從此遭其他同學孤立。到了高中，他因有霸凌前科而受排擠，並選擇孤立自己。片中，將也不敢面對的人臉上都會打個叉<ref name="雍小狼"/>。原著漫畫作者[[大今良時|大今良時]]稱讚了入野的配音表現，表示「他的聲音讓人沒有壓力，並帶有高超的演技，甚至讓人覺得很有可能是漫畫讀者在腦內再生的聲音」<ref name="自由時報 (2017-03-23)"/>。
* [[早見沙織|早見沙織]] 聲演 西宮硝子<ref name="VAPart1"/>
: 聽障少女，在小學時遭到將也等人的霸凌。導演山田尚子表示，當她首次聽見早見的聲音時，就確定硝子的配音員非早見莫屬，稱「那個專注、倔強而又直白的西宮硝子就是這樣！」<ref name="痞客邦 (2017-03-22)"/>早見表示，「硝子很有人情味，這點非常吸引我；她和家人或朋友談話時擺出一副自滿的臉，讓人覺得十分可愛。要為她配音在技術層面上很困難，因為配音現場安靜的彷彿像湖水旁安靜澄澈的空氣，一根針掉在地上都能聽見，但這些應該不是西宮能留意到的。因此，在為硝子配音時，我發出了跟想像中完全不同的聲音」<ref name="痞客邦 (2017-03-22)"/>。

== 製作 ==
漫畫《[[聲之形|聲之形]]》即將要改編成動畫的消息於2014年11月19日發布的《聲之形》最終話裡公布<ref name="ANN (2014-11-17)"/>，隨後在12月17日補充說明，《聲之形》將被改編成一部動畫電影<ref name="ANN (2014-12-15)"/>。根據在2015年10月14日發行的《[[週刊少年Magazine|週刊少年Magazine]]》2015年第46期中的內容，[[京都動畫|京都動畫]]與[[山田尚子|山田尚子]]將分別擔任負責進行改編作業的工作室及該片的導演<ref name="ANN (2015-10-11)"/>。2016年4月8日，工作室公開了改編電影的官方網站，同時宣布[[吉田玲子|吉田玲子]]將為該片撰寫劇本，[[西屋太志|西屋太志]]設計角色，並定於2016年9月17日在日本上映<ref name="JPRelease"/>。

== 上映 ==
《電影版 聲之形》於2016年9月17日在日本的120家劇院首映<ref name="JPRelease"/><ref name="JPTheater"/>。此外，該片還於分別於10月22日和2017年2月5日在2016年{{link-en|蘇格蘭愛情動畫節|Scotland Loves Animation}}及[[倫敦|倫敦]]的[[當代藝術學院|當代藝術學院]]進行放映<ref name="Scotland Loves Animation"/><ref name="ICA"/>。紫色計劃於3月9日在新加坡和馬來西亞發行了該片<ref name="Purple Plan"/>，而英國及愛爾蘭地區的發行商則是{{link-en|Anime Limited|Anime Limited}}，於3月15日上映<ref name="Anime Limited"/>。澳大利亞和新西蘭分別於4月9日及4月16日由{{link-en|麥德曼娛樂|Madman Entertainment}}發行<ref name="Madman Entertainment"/>。在菲律賓，電影於5月10日上映，並由先驅電影擔任發行商<ref name="Anime Pilipinas (2017-05-02)"/>。《聲之形》將於2017年9月8日在中國上映，但片长比其他地区上映的版本缩减了将近20分钟<ref name="动漫星空 (2017-05-12)"/><ref>[http://www.g-cores.com/articles/26809 《声之形》正式定档，9月8日上映].机核网.2017-08-24.[2017-08-28].</ref>。電影的[[藍光光碟|藍光]]及[[數碼多功能影音光碟|DVD]]版本於5月17日發布<ref name="Blu-ray/DVD"/>。

=== 票房 ===
日本方面，首映達120家劇院。兩天後，該片共賣出20萬張電影票，收入2.83億日元，次於[[新海誠|新海誠]]的《[[你的名字|你的名字]]》位居第二<ref name="JPTheater"/>。截至2016年11月30日，電影的觀影人次達170萬人，票房累積超過22億日元<ref name="GrossTotal"/>。

台灣方面，首週三天票房為新台幣2000萬元<ref name="自由時報 (2017-03-28)"/>；上映3週全台票房破新台幣6000萬元，名列台灣影史最賣座日本片歷史排名中的第5位<ref name="自由時報 (2017-04-16)"/>；最終全台票房為新台幣6866萬元<ref>{{cite web |title=全國電影票房截至2017年五月前資訊 |url=https://www.tfi.org.tw/Content/TFI/PublicInfo/全國電影票房截至2017年五月前資訊_v2_0604.pdf |website=[[國家電影中心|國家電影中心]] |date=2017-06-05}}</ref>。

中国大陆方面，电影上映的首周3天票房达到约3,300万人民币，位列当周票房榜第四名<ref>[https://weibo.com/2304129841/FluRMhWBc? @中国电影报].新浪微博.2017-09-12.[2017-09-19].</ref><ref>[http://www.cbooo.cn/m/656273 声之形 电影详情].艺恩电影智库.[2017-09-19].</ref>。

== 反應 ==
=== 評價 ===
該片所獲評價多為正面。[[爛蕃茄|爛蕃茄]]上收集的17篇專業影評文章中，有16篇給出了「新鮮」的正面評價，「新鮮度」為94%，平均得分7.6分（滿分10分）<ref name="rt"/>，而基於另一影評網站[[Metacritic|Metacritic]]上的4篇評論文章，其中4篇予以好評，無差評，平均分為80（滿分100），綜合結果為「普遍良好的評價」<ref name="Metacritic"/>。

《[[每日電訊報|每日電訊報]]》的{{link-en|羅比·柯林|Robbie Collin}}給予電影4顆星（滿分5顆）的評價，並特別指出導演[[山田尚子|山田尚子]]營造出的複雜高中生活，與乾淨單純的時刻所形成的鮮明對比<ref name="Robbie Collin"/>。《[[衛報|衛報]]》的{{link-en|彼得·布萊德肖|Peter Bradshaw}}表示，《電影版 聲之形》使他想起《{{link-en|我的青春期|My So-Called Life}}》，並稱該片是一部「引人入勝的電影：微妙、感性且精緻」<ref name="Peter Bradshaw"/>。《[[廣播時報|廣播時報]]》的特雷弗·約翰斯頓打出4顆星（滿分5顆）的分數，表示，「電影具質問意味的主題是一回事，但真正使該片脫穎而出的是女導演山田尚子的導演功力，她使片中充滿著意想不到的轉折和畫面，並提出了觀察世界的新方法」<ref name="Trevor Johnston"/>。《好萊塢新聞》的路克·雷恩·鮑多克稱讚《電影版 聲之形》為「有力、動人且驚人的完美之作」，並指出導演山田尚子的風格「內斂又飽含詩意」<ref name="Luke Ryan Baldock"/>。

擔任[[文化廳媒體藝術祭|文化廳媒體藝術祭]]動畫部門評審的高橋良輔表示，「電影本身題材就很困難，當初打算把漫畫改編成電影時，就包含了青春期會面對的掙扎、跟他人溝通的困難等複雜內容，這都得透過磨練多年的表現技巧與其魅力才能完美呈現」<ref name="中時電子報 (2017-03-26)"/>。《[[你的名字|你的名字]]》導演[[新海誠|新海誠]]稱《電影版 聲之形》為「夢幻般的作品」，同時也稱讚[[入野自由|入野自由]]的配音表現，並表示電影「高雅端正的演出讓人想模仿都模仿不了」<ref name="Makoto Shinkai Twitter"/>。

然而，《[[世紀報|世紀報]]》的傑克·威爾遜給予電影2.5顆星（滿分5顆）的評價，並表示，「對於一部不只有一名角色考慮自殺的電影，《電影版 聲之形》出奇的平靜且缺乏情感方面的力量，使它不夠真實」<ref name="Jake Wilson"/>。
