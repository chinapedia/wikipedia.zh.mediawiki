《'''電腦冒險記'''》是2001年4月6日至2002年3月29日在[[東京電視網|東京電視網]]上播放的[[電視動畫|電視動畫]]，全52話，臺灣[[三立都會台|三立都會台]]及香港[[亞洲電視|亞洲電視]]以同名《電腦冒險記》公開播映。

== 解說 ==
其特色是登場機器人以及戰鬥場景皆是以3DCG所製作，但CG模組和同時期的其他CG動畫相較下是極為儉樸、動作也是最低限度的狀態，評價並不怎麼好（據說CG部分是由3名工作人員所製作）。而故事、平常的動畫場景、音樂等CG以外的部分評價是比較高的。

玩具由TAKARA（現在的[[TOMY|TAKARA TOMY]]）所發售，以具備透過纜線和電視相接就能夠遊玩[[電視遊戲|電視遊戲]]功能的機器人玩具為中心，但是價格偏高（遊戲重心的グラディオン價格是7,980日圓），銷售並不理想。

== 故事 ==
背景發生在2100年，故事開始時，一個叫魔法之門的伺服器遭到來自外太空的電腦病毒入侵以致癱瘓，並企圖征服全地球，守衛魔法之門的十三個網絡騎士當中十二個受病毒操控，主角的弟弟被困，於是主角健與最後一名網絡騎士並肩作戰，喚醒了所有網絡騎士。
經過連場激戰後，電腦病毒實體化，然而由於健的努力最後被消滅。

:: 能操作風之力，個性見義勇為，是個值得信賴的夥伴，其速度在網路騎士中是數一數二的快。[[豹|捷豹]]型網路騎士。[[跑車|跑車]]所變形而成。
:: 必殺技為可發射無限能源彈的「'''格林破壞砲（ガトリングバスター）'''」。

* '''鯊魚王（シャークオン）：'''<small>[[吉野裕行|吉野裕行]]聲</small>
:: 擅長水中戰，橫衝直撞的攻擊先鋒，麻煩製造者。[[鯊魚|鯊魚]]型網路騎士。[[氣墊船|氣墊艇]]所變形而成。
:: 必殺技為發射高壓水柱的「'''海洋龍捲風（アクアトルネード）'''」。

* '''獅鷲王（グリフィオン）：'''<small>[[中村悠一|中村悠一]]聲</small>
:: 個性冷靜且足智多謀，眾人的作戰軍師，堅持信念絕不妥協。[[獅鷲|獅鷲]]型網路騎士。[[戰鬥機|戰鬥機]]所變形而成。
:: 必殺技為獅鷲王雙翼砲台所發射的導彈攻擊「'''導向射擊（ホーミングショット）'''」。

* '''不死鳥王（フェニクオン）：'''<small>[[鈴村健一|鈴村健一]]聲</small>
:: 渴望和平，講話喜歡參雜英文，自稱火焰王子。[[鳳凰|鳳凰]]型網路騎士。[[新幹線|SF新幹線]]所變形而成。
:: 必殺技為可溶解鋼鐵的火焰攻擊「'''熱炎波動（ヒートウェーブ）'''」。

* '''三頭犬王（ケルベリオン）：'''<small>[[宮下道央|宮下道央]]聲</small>
:: 雙胞胎騎士的哥哥，十分信賴對方。好戰暴力，經常口出惡言，擅突擊。[[三頭犬|三頭犬]]型網路騎士。[[戰車|鋸齒戰車]]所變形而成。
:: 必殺技為尾部的鋸齒圓盤斬切敵人的「'''螺旋鋸刃（スパイラルカッター）'''」。

* '''雙頭犬王（オルトリオン）：'''<small>[[陶山章央|陶山章央]]聲</small>
:: 雙胞胎騎士的弟弟，兩人默契一流。個性與哥哥完全相反，冷靜沉穩。專長為後勤輔助。[[雙頭犬|雙頭犬]]型網路騎士。[[戰車|尖鑽戰車]]所變形而成。
:: 必殺技為發射能粉碎岩石的尖鑽能源彈的「'''巨型尖鑽（ジャイアントドリル）'''」。

* '''魔像王（ゴレムオン）：'''<small>宮下道央、陶山章央聲</small>
:: 雙胞胎騎士合體後的巨大網路騎士。當由三頭犬王主導人格時，攻擊力較強；反之，由雙頭犬王主導時，防禦力為其優勢。[[魔像|魔像]]型網路騎士。
:: 雙手上的鋸盤和尖鑽為其武器。

* '''飛龍王（ワイバリオン）：'''<small>小西克幸聲</small>
:: 結城凱的夥伴。擁有騎士中最優秀的飛行能力，對小孩的心思特別敏感。[[飛龍|飛龍]]型網路騎士。[[螺旋槳飛機|水上戰機]]所變形而成。
:: 可變形為「'''勝利飛翼（ビクトリーウィング）'''」與格蘭迪昂合體為飛行模式。必殺技為嘴部發出的光束砲的「'''咆哮凱撒（ハウリングゲイザー）'''」以及肩部發射的火箭彈。

* '''獨角獸王（ペガシオン）：'''<small>鈴村健一聲</small>
:: 擅長以幻覺作為攻擊手段，冷靜沉著。其獨特的審美觀有助於戰鬥中的決斷。[[天馬|天馬]]型網路騎士，頭部為[[獨角獸|獨角獸]]，下半身為[[半人馬|半人馬]]。[[裝甲車|六輪裝甲車]]所變形而成。
:: 獨角獸王有兩種模式可以和巨神王合體：「手臂型態」裝備於巨神王的右手；「頭盔型態」則變為巨大頭盔與巨神王合體為巨神王獨角獸（ダイタリオンペガス）。
:: 武器為雙劍「幻象劍（ミラージュソード）」，及雙肩上發射的「天馬光束（ミラージュソード）」。必殺技為「'''超級幻象劍（スーパーミラージュソード）'''」。

* '''獅虎王（ライガオン）：'''<small>中村悠一聲</small>
:: 個性冷酷，盯上的獵物決不放過。[[奇美拉|獅子]]型網路騎士。[[直升機|螺旋直昇機]]所變形而成。
:: 獅虎王有兩種模式可以和巨神王合體：「手臂型態」裝備於巨神王的左手；「頭盔型態」則變為巨大頭盔與巨神王合體為巨神王獅虎（ダイタリオンライガ）。
:: 武器為「戰戟護盾（ランサープロテクト）」。必殺技為能源光束「'''電子戰戟（エレクトリックランサー）'''」。
