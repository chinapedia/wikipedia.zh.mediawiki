'''電子墨水'''（{{lang-en|E Ink}}）是指[[E_Ink公司|電子墨水公司]]（E Ink Corporation）<ref>{{cite web |url = http://www.eink.com/ |title = E Ink 電子墨水公司 |accessdate = 2009-08-04}}</ref>製造的[[電子紙|電子紙]]。该公司成立于1997年，建基於美國[[麻省理工學院|麻省理工學院]][[媒體實驗室|媒體實驗室]]（MIT Media Lab）的研究。

電子墨水通常会制成薄膜，用于電子[[顯示器|顯示器]]，尤其用於[[電子書|電子書]]，如[[Sony_Reader|Sony Reader]]、[[Onyx_Boox|Onyx Boox]]、iLiad、Cybook Gen3、[[Amazon_Kindle|Amazon Kindle]]、Logic的 eReader和Polymer Vision的Readius裝置。

電子墨水具有超低耗電量的特點，在顯示的影像有變化時才耗用電力，在保持顯示的影像時可以不耗電，但缺點是反應速度慢，不適合用於播放如動畫及電影等動態畫面，較適合用於靜態顯示，另一個缺點未能彩色顯示，只能用於灰階顯示。[[摩托羅拉|摩托羅拉]]出品的F3是首部配備電子墨水技術屏幕的[[流動電話|流動電話]]。另外，[[三星電子|三星電子]] Alias 2上顯示按鍵的改變也運用了該技術。北美洲版本的《[[时尚先生_(刊物)|时尚先生]]》是首份使用電子墨水封面的雜誌。

根據製造商的描述，「電子墨水主要由大量細小[[微膠囊|微膠囊]]（microcapsules）组成，這些微膠囊約為人類頭髮直徑大小。每個微膠囊中包含悬浮于澄清液體之中的帶正[[電荷|電荷]]的白粒子和帶負電荷的黑粒子。设置[[電場|電場]]为正時，白粒子向微膠囊頂部移動，因而呈现白色；同時，黑粒子被拉到微膠囊底部，从而隱藏。如施加相反的电场，則黑粒子會在膠囊頂部出現，因而呈现黑色。」<ref>{{cite web |url = http://www.eink.com/technology/howitworks.html |title = E Ink Corporation - Our Technology |author = 電子墨水公司 |accessdate = 2009-08-04 |deadurl = yes |archiveurl = https://web.archive.org/web/20091014045544/http://www.eink.com/technology/howitworks.html |archivedate = 2009-10-14 }}</ref>

2009年6月1日，電子墨水公司宣佈，同意接受主要商務伙伴台灣廠商[[元太科技|元太科技]]（Prime View International）收購，數值達2.15億美元。<ref>{{cite news |title=元太2.15億美元 併購E-Ink |url=http://www.libertytimes.com.tw/2009/new/jun/2/today-e19.htm |publisher=自由時報電子報 |date=2009-06-02 |accessdate=2009-08-04 |deadurl=yes |archiveurl=https://web.archive.org/web/20090605062325/http://www.libertytimes.com.tw/2009/new/jun/2/today-e19.htm |archivedate=2009-06-05 }}</ref><ref>{{cite news|title=元太科技電子紙相關技術|url=http://cdnet.stpi.org.tw/techroom/pclass/pclass_A003.htm|publisher=國家實驗研究院科技政策研究與資訊中心|date=2005-08-19|accessdate=2009-08-04|deadurl=yes|archiveurl=https://web.archive.org/web/20090414042114/http://cdnet.stpi.org.tw/techroom/pclass/pclass_A003.htm|archivedate=2009-04-14}}</ref>自此，這項主流電子紙技術成為元太科技，同時也是台灣造紙大廠[[永豐餘|永豐餘]]百分之百持有的子公司。<ref>E Ink 為台灣永豐餘集團的子公司  {{cite web |url=http://tw.eink.com/about_us.html |title=存档副本 |accessdate=2015-12-28 |deadurl=yes |archiveurl=https://web.archive.org/web/20160310152214/http://tw.eink.com/about_us.html |archivedate=2016-03-10 }}</ref>

併購之後，電子墨水公司繼續電子墨水技術的研究發展，并取得一定的突破<ref>{{cite web |author1=BOOX.HK |title=電子墨水屏介紹 |url=http://www.boox.hk/article/short_essay_about_e_ink_screen.html |website=www.boox.hk |publisher=BOOX.HK}}</ref>。例如，Carta的高對比度，Regal提升了刷新速度，還有軟性屏幕的開發等等。
