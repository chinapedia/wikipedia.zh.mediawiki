'''電光大橋'''為[[臺灣|臺灣]]一座位於[[臺東縣|臺東縣]][[關山鎮_(臺灣)|關山鎮]]境內的[[橋樑|橋樑]]，該橋梁橫跨於[[卑南溪|卑南溪]]之上，是聯繫關山鎮市區以及關山鎮[[電光里|電光里]]的聯絡橋樑，由[[臺東縣政府|臺東縣政府]][[關山鎮公所|關山鎮公所]]進行管理與維護。<ref>{{cite web |url=https://www.youtube.com/watch?v=qFc-TJyo0Hc |title=關山鎮導覽 一窺電光橋興建故事 }}</ref>

== 歷史 ==
=== 興建緣起 ===
關山鎮電光里位於卑南溪的東側，而關山市區則位於對岸的西側，雖僅一水的阻隔，但當時並無聯繫橋梁的建設，因此當地電光里居民僅能在每年十月雨季過後，往返關山以竹、木搭成的便橋通行，有時一年要築橋三、四次，因為雨季來臨時，便橋就被沖走，只得繞道[[池上鄉|池上]]或[[瑞源村|瑞源]]轉往關山。<ref name="小城故事"/>

也由於無永久性橋梁的建置，居民往往只能冒險溯溪渡河，因此時常發生居民遭到洪水沖走的意外事件。距今四十餘年前，[[臺北工專|臺北工專]]學生吳昇財（前[[關山工商|關山工商]]校長吳昇齊之兄）回鄉渡假因涉水而溺斃，1957年（民國46年）6月2日，[[電光國小|電光國小]]校長黃定龍帶著他均未滿十歲的兩個孩子，黃子輝、黃子晃回關山過[[端午節|端午節]]，走在便橋上卻被洪水連木橋一起沖走，結果三人均溺斃。還有里民陳盛德和王便，也在民國59年10月間，亦遭到卑南溪水沖走。<ref name="小城故事"/>

多年來電光里民在[[里民大會|里民大會]]及關山鎮[[鎮民代表|鎮民代表]]會上均曾提案興建永久性聯絡橋樑，當時的[[關山鎮公所|關山鎮公所]]亦多次爭取補助興建大橋，但因金費龐大均未能如願。<ref name="小城故事"/>

=== 興建歷程 ===
直至1985年（民國74年）9月3日，有一位電光里的國中畢業生，寫了9張信紙向[[臺灣省|臺灣省]]前[[省主席|省主席]][[邱創煥|邱創煥]]傾訴：3年來涉水過橋就學的往事，懇求興建大橋。又經當時擔任[[省議員|省議員]]的[[林忠信|林忠信]]、[[洪文泰|洪文泰]]及各級長官的奔走，以及關山鎮前鎮長[[彭盛煥|彭盛煥]]的爭取，終獲省主席的同意，於1985年（民國74年）10月間將原函發交臺東縣政府轉關山鎮公所設計與建造。<ref name="小城故事">{{cite web |url=http://kslove.myweb.hinet.net/KS_Park/Page/KnowMation/sTown_story.htm |title=小城故事 台東電光大橋}}</ref>

電光大橋工程自1987年（民國76年）動工，1988年（民國77年）11月大橋主體完工，其餘週邊工程一直到1992年（民國81年）04月才全數完成。<ref name="小城故事"/>

=== 養護現況 ===
臺東縣[[海端鄉|海端鄉]][[崁頂國小|崁頂國小]]師生自2013年3月起，利用週三下午到附近橋梁做健檢，意外發現，電光大橋保護橋墩鋼板有許多已變形、與橋墩脫離，甚至鋼筋裸露，鏽蝕嚴重，汛期或地震來襲，橋梁恐會垮。[[臺東縣議會|臺東縣議會]]副議長陳宏宗前往現場會勘，他強調，會與相關單位聯繫盡速改善。<ref name="中時電子報01"/>台東縣政府[[建設處|建設處]]長許瑞貴表示，將會同[[第八河川局|第八河川局]]、公所等單位現場會勘，待鑑定結果出爐再研擬補救方法。<ref name="中時電子報01">{{cite web |url=http://www.chinatimes.com/newspapers/20130427000607-260107 |title=中時電子報 戶外教學 驚見電光大橋變危橋 }}</ref>

== 諸元 ==
電光大橋工程包括民族路地下道及引道工程、關山堤防，總工程費共計新台幣一億五千三百五十三萬元整，全長1700公尺，橋長720公尺，橋面寬度7.5公尺，每孔45公尺，引道780公尺。<ref name="小城故事"/>
