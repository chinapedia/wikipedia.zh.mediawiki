[[物理學|物理學]]中，'''電荷共軛宇稱'''（{{lang-en|charge conjugate parity}}、{{lang|en|charge parity}}或{{lang|en|C  parity}}，可標為{{lang|en|'''CP'''}}）是粒子的相乘[[量子數|量子數]]，用以描述一些粒子在[[電荷共軛|電荷共軛]]的對稱運算下的行為。

電荷共軛改變所有量子荷（{{lang|en|quantum charge}}）的正負號，這些量子荷為相加量子數，包括有[[電荷|電荷]]、[[重子數|重子數]]與[[輕子數|輕子數]]，以及[[味荷|味荷]]如[[奇異數|奇異數]]、[[魅數|魅數]]、[[底數|底數]]、[[頂數|頂數]]與[[同位旋|同位旋]]（''I''<sub>3</sub>）。相對地，電荷共軛不改變粒子的[[質量|質量]]、[[線動量|線動量]]或[[自旋|自旋]]。

== 數學形式 ==
考慮一運算<math>\mathcal{C}</math>可將一粒子轉變成其[[反粒子|反粒子]]：
:<math>\mathcal C \, |\psi\rangle = | \bar{\psi} \rangle</math>。
兩個狀態都必須是可歸一化，因此
:<math> 1 = \langle \psi | \psi \rangle = \langle \bar{\psi} | \bar{\psi} \rangle = \langle \psi |\mathcal{C}^\dagger \mathcal C| \psi \rangle</math>，
意味著<math>\mathcal C</math>是[[幺正|幺正]]的：
:<math>\mathcal C \mathcal{C}^\dagger =\mathbf{1}</math>。

對此粒子進行重複兩次的<math>\mathcal{C}</math>運算，
:<math> \mathcal{C}^2 |\psi\rangle = \mathcal{C} |\bar{\psi}\rangle = |\psi \rangle</math>，
可以看出<math>\mathcal{C}</math>有如下性質：
: <math>\mathcal{C}^2=\mathbf{1}</math>
: <math>\mathcal{C}=\mathcal{C}^{-1}</math>。

將所有性質統整可得：
:<math>\mathcal{C}=\mathcal{C}^{\dagger},</math>
意即電荷共軛[[算符|算符]]是[[自伴算符|自伴算符]]，也因此是一個可觀測物理量。

===本徵值與本徵態===
電荷共軛 <math>\mathcal C</math> 運算的本徵態<math>|\psi\rangle</math>與本徵值<math>\eta_C</math>關係如下：
:<math>\mathcal C \, |\psi\rangle = \eta_C \, | \psi \rangle</math>。

如同[[宇稱|宇稱]]，重複<math>\mathcal{C}</math>運算兩次，則粒子狀態不變：
:<math>\mathcal{C}^2|\psi\rangle = \eta_C \mathcal{C} |{\psi} \rangle = \eta_{C}^{2} |\psi\rangle = | \psi \rangle</math>，
使得本徵值<math>\eta_C</math>只能為<math>\pm 1</math>。此稱為粒子的電荷共軛宇稱。

上面的條件意味著<math>\mathcal C|\psi\rangle</math>與<math>|\psi\rangle</math>有相同的量子荷，因此只有真正中性的系統（所有量子荷與[[磁矩|磁矩]]皆為零）是電荷共軛宇稱的本徵態。符合此條件的有：
* [[光子|光子]]
* 正反粒子約束態，如中性π介子（π<sup>0</sup>）、η介子、[[正電子素|正電子素]]。

== 電荷共軛宇稱守恆的實驗驗證 ==
* <math>\pi^0\rightarrow 3\gamma</math>：觀測到中性[[π介子|π介子]]<math>\pi^0</math>會衰變為雙光子γ+γ，因此我們可認定π介子有<math>\eta_C=(-1)^2=1</math>的性質。然而，每增加一個γ會在π介子的電荷共軛宇稱中引入一個-1的因子；衰變成3γ則會違反電荷共軛宇稱守恆。過去曾進行了此種衰變的實驗驗證<ref>{{cite journal|last=MacDonough|first=J.|journal=Phys. Review|year=1988|volume=D38|page=2121|display-authors=etal}}</ref>，其中應用到產生π介子的反應過程：<math>\pi^{-} + p \rightarrow \pi^0 + n</math>。

*<math>\eta \rightarrow \pi^{+} \pi^{-} \pi^{0}</math><ref>{{cite journal|last=Gormley|first=M.|journal=Phys. Rev. Lett.|year=1968|volume=21|page=402|bibcode = 1968PhRvL..21..402G |doi = 10.1103/PhysRevLett.21.402 |display-authors=etal}}</ref>：{{le|η介子|Eta meson}}的衰變。

*<math>p \bar{p}</math>[[湮滅|湮滅]]。<ref>{{cite journal|last=Baltay|first=C|journal=Phys. Rev. Lett.|year=1965|volume=14|page=591|bibcode = 1965PhRvL..14..591R |doi = 10.1103/PhysRevLett.14.591 |display-authors=etal}}</ref>
