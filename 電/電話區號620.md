'''電話區號620'''為在2001年2月3日時從{{le|電話區號316|Area code 316}}拆分出來的電話區號。此區號大多涵蓋了[[堪薩斯州|堪薩斯州]]南部。此電話區號的範圍從[[科羅拉多州|科羅拉多州]]東界延伸至[[密蘇里州|密蘇里州]]邊界，而{{le|威奇托都會區|Wichita Metropolitan Area}}（316）並不包含在內。

電話區號620的撥入寬鬆（permissive dialing）期直至2001年11月3日結束。涵蓋此電話區號的主要城市有{{le|道奇城 (堪薩斯州)|Dodge City, Kansas|道奇城}}、{{le|加登城 (堪薩斯州)|Garden City, Kansas|加登城}}以及{{le|哈欽森 (堪薩斯州)|Hutchinson, Kansas|哈欽森}}等。

==電話區號620之分界==
電話區號620及{{le|電話區號785|Area code 785}}之間的分界大致上以[[4號堪薩斯州州道|4號堪薩斯州州道]]和{{le|96號堪薩斯州州道|K-96 (Kansas highway)}}的路徑由西向東行經。<!--行經[[麥克弗森縣_(堪薩斯州)|麥克弗森縣]]後則傾向於{{le|135號美國國道 (堪薩斯州)|Interstate 135 (Kansas)|135號美國國道}}，之後沿著{{le|堪薩斯州收費公路|Kansas Turnpike}}繼續向東行經至[[萊昂縣_(堪薩斯州)|萊昂縣]]恩波里亞北部，電話區號620、電話區號785以及{{le|電話區號913|Area code 913}}的交會點。而電話區號620的北界與電話區號913的南界共同止於{{le|堪薩斯城都會區|Kansas City metropolitan area}}南部。-->

另外[[塞奇威克县_(堪萨斯州)|塞奇威克县]]全境，[[巴特勒縣_(堪薩斯州)|巴特勒縣]]和[[哈維縣_(堪薩斯州)|哈維縣]]部分區域仍舊使用電話區號316。
