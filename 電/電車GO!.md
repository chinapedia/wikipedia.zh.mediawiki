{{vgname|電車GO!|ja=電車でGO!}}是一款由[[太东|太东]]制作和发行的[[模拟经营游戏|模拟经营游戏]]。<ref name="RJ384">{{Cite journal|journal = [[鉄道ジャーナル|鉄道ジャーナル]] |date = 1998-10 |volume = 32 |issue = 10 |page = 131 |publisher = [[鉄道ジャーナル社|鉄道ジャーナル社]] }}</ref>本游戏最初于1997年3月[[街机|街机]]发行，游戏后移植至[[PlayStation_(游戏机)|PlayStation]]。<ref name="gamefaqs">{{cite web | url = http://www.gamefaqs.com/ps/572801-densha-de-go/data| title = Release information | publisher = [[GameFAQs|GameFAQs]]| date = | accessdate = 2016-05-14}}</ref> 1997年9月，本游戏的加强版《電車GO!EX》发行。本游戏后移植至[[Windows|Windows]]。

== 游戏内容 ==

本游戏为电车模拟游戏，玩家控制着电车，在一定时间内，在规定的车站范围停车。

在原始的街机版《電車GO!》共收录了山阴本线、京滨东北线、东海道本线和山手线。

== 发行 ==

游戏在发行之初，提出了“你就是司机”的口号。在街机，街机台模仿了电车头的形状。游戏收到欢迎。

加强版《电车GO！EX》于1997年9月在街机发行。游戏后移植世嘉土星。游戏在原有的线路中追加了雨雪模式。
