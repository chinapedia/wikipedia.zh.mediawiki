'''股票代號'''（{{lang-en|'''stock symbol''' 或 '''ticker symbol'''}}）是一組[[縮寫|縮寫]]、[[代碼|代碼]]或[[代號|代號]]，用來識別某單一[[股票市場|股票市場]]裡某支進行[[證券|證券]]交易的[[股票|股票]]。股票代號可以由[[英文字母|英文字母]]、[[數字|數字]]、或兩者混和組成，有時也會加入符號，如：. 或 :。

英文裡「'''ticker symbol'''」原是指早期股票[[電報|電報]]收發機列印在收發用紙帶上的[[符號|符號]]。

今日，在公開市場交易的證券、[[期貨|期貨]]或[[選擇權|選擇權]]商品都會分配一組「交易代號」，以利商品識別和交易的進行。有時也會加入市場名稱縮寫來分別市場，利於辨識，例如：2038.TW 或 2038:TW（海光）、2038.HK 或 2038:HK（[[富士康國際|富智康]]）；1802.TW 或 1802:TW（[[台玻|台玻]]）、1802.JP 或 1802:JP（[[大林組|大林組]]）。
