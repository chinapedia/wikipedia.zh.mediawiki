'''盂城驿'''{{NoteTag|应写作“盂”城驿而非“孟”城驿。}}位于中国[[江苏省|江苏省]][[高邮市|高邮市]]南门外馆驿巷，[[京杭大运河|京杭大运河]]东侧，是明清时[[大运河|大运河]]沿线规模最大的[[驿站|驿站]]<ref name=":0">{{Cite web |url = http://www.dili360.com/article/p53b24a5f9bdce03.htm |title = 高邮盂城驿：明清时大运河沿线规模最大驿站 |accessdate=2016-07-01 |author=|date=2014-07-01 |publisher=中国国家地理}}</ref>。驿站名中的盂城二字是高邮的别称<ref>{{Cite book |title=高邮县志 |author=王鹤|author2=杨杰 |author3=高邮县编史修志领导小组 |publisher=江苏人民出版社|year=1990}}</ref>{{Rp|3}}<ref>{{Cite book |title=江苏省志：地理志 |publisher=江苏古籍出版社 |year=1999 |volume=2}}</ref>{{Rp|88}}，取自[[秦观|秦观]]的《送孙诚之尉北海》“吾乡如覆盂，地处扬楚脊。”一句<ref>{{Cite journal|title=水陆两栖揽下一路仆仆风尘——盂城驿|author=娄华东 |url = http://d.wanfangdata.com.cn/Periodical/zhmj201208004 |journal=中华民居|issue=8|doi=10.3969/j.issn.1674-3954.2012.08.004 |others=|year=2012|volume=|pmid=}}</ref><ref>{{Cite book |title=万象更新：揚州建国后史料专辑 |publisher=江苏文史资料编辑部 |year=1997 |volume=97}}</ref>{{Rp|41}}。

驿站建于[[明朝|明]][[洪武|洪武]]八年（1375年）。[[永乐|永乐]]年间重修，[[嘉靖|嘉靖]]三十六年（1557年）毁于火，[[隆庆_(明朝)|隆庆]]二年（1568年）重修。[[清朝|清]][[康熙|康熙]]、[[嘉庆|嘉庆]]、[[道光|道光]]先后修葺。据记载，驿站鼎盛时厅房多达100多间，驿马多达65匹，驿船多达18条，马夫和水夫多达200多人<ref name=":5">{{Cite news |url = http://www.china.com.cn/chinese/zhuanti/yhsy/1193549.htm |title=国内保存最完好的古驿站——盂城驿 |author=黄晔华|website =中国网 |accessdate=2006-04-25}}</ref><ref name=":3">{{Cite web |url = http://www.chinagrandcanal.com/view.php?id=1632 |title=中国第一驿站孟城驿 |accessdate=2017-01-13 |publisher=大运河遗产保护管理办公室}}</ref>。盂城驿现已被列为[[全国重点文物保护单位|全国重点文物保护单位]]和[[国家4A级旅游景区|国家4A级旅游景区]]，并作为大运河的一部分被列为世界遗产。现在的盂城驿占地面积约为1.6万平方米<ref name=":6">{{Cite book |title=中国文物地图集：江苏分册 |publisher=广东省地图出版社 |year=2008 |isbn=9787503144486}}</ref>{{Rp|381}}，现存的建筑有门厅、皇华厅（正厅）、驻节堂（后厅）等。

== 历史沿革 ==
盂城驿于明洪武八年（1375年）建立，<ref name=":4">{{Cite book|title=江苏省志|publisher=江苏人民出版社|year=|volume=84}}</ref>{{Rp|270}}<ref>{{Cite book|title=江苏省志|publisher=江苏人民出版社|year=|volume=34}}</ref>{{Rp|152}}。明嘉靖年间（1557）被日军烧毁，后于明隆庆年间（1568年）由[[知州|知州]]赵来亨重建，<ref name=":6" />{{Rp|381}}<ref>{{Cite book|title=江苏名鎮志|last=汪文超|first=|last2=江苏省地方志编纂委员会|publisher=江苏古籍出版社|year=1993|isbn=|location=|pages=}}</ref>{{Rp|660}}清嘉庆、道光年重修。<ref>{{Cite journal|title=关注大运河，关注文化线路|author=|url=|journal=中国文化遗产|issue=3|doi=|year=2009|volume=|issn=1672-7819|pmid=}}</ref>{{Rp|16}}现存的绝大部分建筑都是清代重建的。<ref>{{Cite web|url=http://www.jssdfz.com/index.php?m=content&c=index&a=show&catid=20&id=5908|title=盂城驿|accessdate=2017-01-07|author=|date=2015-07-14|publisher=江苏省地方志编纂委员会办公室}}</ref>1913年（民国2年）1月，[[北洋军阀|北洋军阀]]宣布裁撤掉中国所有的驿站，盂城驿也不例外，它被埋没在历史的长河之中。<ref name=":0" /><ref>{{Cite book|title=清代邮戳志|author=孙君毅|publisher=中国集邮出版社|year=1984}}</ref>{{Rp|1}}

1985年，中国在开展文物普查工作时发现了这座驿站的文化价值。<ref>{{Cite news|url=http://news.xinhuanet.com/newscenter/2002-12/16/content_661081.htm|title=江苏高邮盂城古驿：见证中国邮驿事业2千年|date=|work=新华网|accessdate=2002-12-16}}</ref><ref>{{Cite journal|title=大运河畔的“稀世遗珍──盂城驿|author=高鸣|url=http://www.cnki.com.cn/Article/CJFDTotal-SLTD199501012.htm|journal=水利天地|issue=1|doi=|others=|year=1995|volume=|issn=1002-3305|pmid=}}</ref>1996年，盂城驿被列为[[第四批全国重点文物保护单位|第四批全国重点文物保护单位]]。<ref>{{Cite web|url=http://www.gov.cn/guoqing/2014-07/21/content_2721166.htm|title=国务院关于公布第四批全国重点文物保护单位的通知|accessdate=2017-01-07|author=|date=1996-11-20|publisher=中国政府网}}</ref>此后的5年，内盂城驿进行了两次修复。<ref name=":0" />2013年，盂城驿的修复全面结束并对外开放。<ref>{{Cite web|url=http://www.ccdy.cn/lvyou/zixun/201310/t20131021_782231.htm|title=明代邮政驿站江苏盂城驿扩容后开放|accessdate=2017-01-13|author=蒋芳|date=2013-10-21|publisher=中国文化传媒网}}{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>

2014年，盂城驿作为[[大运河|大运河]]的一部分被列为[[世界遗产|世界遗产]]。<ref>{{Cite book|url=http://www.pishu.com.cn/skwx_ps/multimedia/download?ID=28C49B5EC6427325094625BD0DF3A7135F81FEC1224DB9E6&siteid=14&Type=|title=大运河（扬州段）文化遗产的保护利用|author=姜师立|author2=张益|author3=吴益群|publisher=|year=}}</ref>{{Rp|3}}同年，盂城驿被列为国家3A级旅游景区。<ref>{{Cite web|url=http://xxgk.gaoyou.gov.cn/gyxxgk/U0000/201611/cf22eac2de2a4ff6a0ad92c1d0ddcac6.shtml|title=盂城东市招商新闻发布会|accessdate=2017-01-13|author=|date=2016-10-28|publisher=高邮市人民政府办公室}}{{Dead link|date=2018年7月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>2015年，盂城驿被列为[[国家4A级旅游景区|国家4A级旅游景区]]。<ref>{{Cite web|url=http://www.yangzhou.gov.cn/qxxw/201512/3e672790273c4b4eaf21dd393ec16a82.shtml|title=盂城驿获批国家4A级旅游景区|accessdate=2017-01-07|author=|date=2015-12-25|publisher=扬州市人民政府办公室}}{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref><ref>{{Cite web|url=http://www.jiangsu.gov.cn/zwfw/lstd/lyz/xxfu/201604/t20160411_428080.html|title=高邮市旅游业发展态势良好|accessdate=2017-01-07|author=|date=2016-04-11|publisher=江苏省人民政府}}</ref><ref>{{Cite web|url=http://www.gaoyou.gov.cn/gaoyou/zwyw/201512/be1190da53b8421a96e8671beccabff7.shtml|title=盂城驿获批国家4A级旅游景区|accessdate=2016-01-13|author=殷朝刚|date=2015-12-25|publisher=高邮市人民政府办公室}}{{Dead link|date=2018年8月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>2016年第一季度，盂城驿共接待了13.84万名，创造了历史新高。<ref>{{Cite web|url=http://www.jiangsu.gov.cn/zwfw/lstd/lyz/xxfu/201604/t20160415_428759.html|title=一季度高邮市旅游业总收入超十亿元|accessdate=2017-01-07|author=|date=2016-04-15|publisher=江苏省人民政府}}</ref>

== 总体布局 ==
[[File:Yucheng_Postal_Stop_ZhuJie_Hall_interior.jpg|right]]
盂城驿的中心建筑是一座三层楼高的[[鼓楼|鼓楼]]，坐北朝南，<ref name=":1">{{Cite web|url=http://www.yangzhou.gov.cn/ycdjj/201406/3b374fb17ae446df9f1667c117ad034c.shtml|title=盂城驿|accessdate=2017-01-13|author=|date=2014-06-03|publisher=扬州市人民政府办公室}}{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>是盂城驿最高的建筑。鼓楼底层的墙面嵌有《捐款纪念碑》和《重修盂城驿记》。第二层外部设有一块写有“古驿重光”的匾额；此外，鼓楼还设置了[[中国明史学会|中国明史学会]]执行会长[[刘重日|刘重日]]题写的“置邮传命”和中国国家外交部副部长[[姜恩柱|姜恩柱]]题写的“飞”等匾额。第三层外部的侧边则悬挂有匾额，上题“鼓楼”二字，由[[东南大学|东南大学]]教授[[潘谷西|潘谷西]]所写。<ref>{{Cite news|url=http://js.ifeng.com/travel/lx/urban/detail_2014_01/22/1770890_1.shtml|title=高邮盂城驿 明代稀世遗珍|date=|work=凤凰江苏|publisher=凤凰新媒体|accessdate=2014-01-22}}</ref>建筑还设有匾额“国脉所系”。<ref name=":0" /><ref>{{Cite news|url=http://paper.people.com.cn/rmrbhwb/html/2016-10/12/content_1717898.htm|title=盂城驿怀古|date=|work=人民日报海外版|accessdate=2016-10-12}}</ref><ref name=":2">{{Cite web|url=http://www.yangzhou.gov.cn/you/201111/195dd845cb5d47139dc817e9d5cbfa70.shtml|title=盂城驿|accessdate=2017-01-13|author=|date=2006-11-02|publisher=扬州市人民政府办公室}}{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>在此可俯瞰周边景色。<ref>{{Cite web|url=http://www.gaoyou.gov.cn/gaoyou/gyly/201604/fd4aea7d4d1e452d9d599580ae2bebff.shtml|title=盂城驿|accessdate=2017-01-13|author=|date=2016-04-15|publisher=高邮市人民政府办公室}}{{Dead link|date=2018年8月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

古驿的管理中心皇华厅又名正厅，当地人俗称接官厅，<ref name=":4" />{{Rp|152}}是一座明代的五开间建筑，旧时主要用来传达政令、迎接官员宾客，堂中置有一副楹联。驻节堂又称后厅，是驿丞和州官接待来客的地方。堂内梁柱的历史可追溯至明代，桌椅均由红木制作，屏门上方悬有匾额，匾额上题写有“驻节堂”三字，东西房均为寝房。后院则为驿卒舍和库房厨房等地。<ref name=":3" /><ref name=":2" />此外，盂城驿内还设有息厅、敞厅、秦邮公馆、驿马饮水地等建筑。<ref name=":5" />

== 保护修复与管理工作 ==
[[File:盂城驿国保.jpg|thumb]]
本节旨在列出除被列为世界遗产和文物保护单位等措施之外的保护修复工作。[[同济大学|同济大学]]、[[东南大学|东南大学]]城市规划设计研究院、[[浙江古建筑设计研究院|浙江古建筑设计研究院]]、[[河海大学|河海大学]]、[[扬州建筑设计研究院|扬州建筑设计研究院]]、[[扬州园林设计院|扬州园林设计院]]等单位均参与到了盂城驿的修复工作中。<ref>{{Cite web|url=http://www.jsgyjs.gov.cn/show_news.asp?News_Class=3&Nid=2187|title=盂城驿修缮整治工程|accessdate=2017-01-07|author=|date=2015-04-23|publisher=高邮市城乡建设局}}{{Dead link|date=2018年7月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>根据政府网站的说法，盂城驿的保护区划可分为遗产区和缓冲区两部分，管理要点包括禁止拆迁重建、坚持“最小干预原则”、注意防火和日常维护工作等。<ref name=":1" />

2016年，盂城驿景区管理委员会成立。<ref>{{Cite web|url=http://www.gaoyou.gov.cn/gaoyou/zwyw/201605/8e6b1eca65f145cb83ead5c4a9cca8f6.shtml|title=盂城驿景区管委会昨成立|accessdate=2017-01-13|author=管玮玮|date=2016-05-06|publisher=高邮市人民政府办公室}}{{Dead link|date=2018年8月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>高邮市文广新局局长魏道智担任委员会主任，高邮市文广新局副局长兼文物局局长赵小山担任副主任和盂城驿景区管理委员会办公室主任二职。<ref>{{Cite web|url=http://xxgk.gaoyou.gov.cn/gyxxgk/E0000/201605/042b9fd00b344366aa882bdb1cd08634.shtml|title=市政府关于成立盂城驿景区管理委员会的通知|accessdate=2017-01-13|author=|date=2016-04-14|publisher=高邮市人民政府}}{{Dead link|date=2018年7月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>
