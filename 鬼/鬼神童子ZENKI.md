{{日本作品|鬼神童子ZENKI|鬼神童子ZENKI／きしんどうじゼンキ}}是[[谷菊秀|-{谷}-菊秀]]原作、[[黑岩善宏|黑岩善宏]]負責繪製的[[日本漫畫|日本漫畫]]，起初於[[集英社|集英社]]《[[月刊少年Jump|月刊少年Jump]]》1992年8月號以短篇的方式發表，之後從同雜誌的12月號開始連載，至1996年9月增刊號完結，[[單行本|單行本]]全12冊。2005年由[[竹書房|竹書房]]發行全7冊的新書版。中文代理方面，集英社版的單行本曾經由[[台灣|台灣]][[長鴻出版社|長鴻出版社]]發行、[[香港|香港]]由[[天下出版|天下出版]]發行。竹書房版的單行本目前無中文版。

媒介方面，1995年由[[東京電視網|東京電視網]]播出同名[[#電視動畫|電視動畫]]，1997年發行[[#OVA|OVA]]的外傳動畫。

== 劇情概要 ==
; 原作
: [[役小角|役小角]]是日本最偉大的降魔師，他役使「前鬼」（又名「朱鬼神」）對抗世上的邪惡。小角逝世之前，把前鬼封印在一個石碑中，並留下一個護法輪。此石碑被封印在{{nihongo|「鬼神鎮」|式神町}}的「役咒法堂」正殿。當災難出現、護法輪發出光芒時，就對著天空畫[[五芒星|五芒星]]，唸小角秘傳的咒語，使前鬼甦醒。
: 在千年以後的鬼神鎮，貓丸與犬丸為了尋找傳說中「役小角的祕寶」而來到鬼神鎮，想以此「祕寶」成為[[億萬富翁|億萬富翁]]。他們趁夜侵入役咒法堂正殿，破壞了結界，撕除了「祕寶」盒上的符咒，打開盒蓋一看，裡面是一顆{{nihongo|「寄生果」|憑依の実}}。寄生果看起來像是一顆眼球，會附身在被強烈慾望薰心的身上，然後依不同的欲望變成不同的{{nihongo|「寄生獸」|憑依獣}}。小角的後代役小明，擁有能力讓前鬼解除封印，以前鬼的力量消滅寄生獸、解救被寄生果附身的人。

; 電視動畫版
: 役小角是千年前的降魔師，擁有極高強的咒力，他役使前鬼打敗暗黑神{{nihongo|「羯磨」|カルマ}}。小角打敗羯磨之後，把前鬼封印起來，直到世間再度瀰漫邪氣。
: 在千年以後的鬼神鎮，被強烈欲望薰心的一個男人解開了原本被封印的果實「寄生果」上的封印。小角的後代役小明，擁有能力讓前鬼解除封印，以前鬼的力量消滅寄生獸、解救被寄生果附身的人。

; 外傳OVA：{{nihongo|《鬼神童子 ～黯鬼奇譚～》|鬼神童子ZENKI外伝 ～黯鬼奇譚～}}
: 故事發生在櫻花盛開的季節，某個深山村落陸續傳出了4位少女遇害的消息。接獲委託的小明與前鬼來到了這個村落進行調查，此時她們發現了一個秘密：遇害的少女們幾乎都是17歲的少女。而事件的背後隱藏了一個更驚人的事實：原來這些少女們都是為了使村裡傳說中的'''鬼'''復活的活祭品。

: 在故事開頭遭到寄生獸的攻擊，千鈞一髮之際解開了前鬼的第一段封印，並得到能役使前鬼的「護法輪」，其後與前鬼為祛除邪惡而戰。故事中，逐漸學會各種咒術，如「火炎咒」、「烈風咒」。直呼前鬼與後鬼，不尊稱「前鬼殿下」、「後鬼殿下」。
: 在原作及電視動畫版中提供最多[[殺必死|殺必死]]，包括[[全裸|全裸]]入浴及出浴<ref group="注">例如：原作〈大法師·鴉紋之章1：鬼神解縛秘咒〉、原作〈寄生獸果實之章1：木乃伊旅館〉。</ref>、被寄生獸割掉[[比基尼泳衣|比基尼泳衣]]上半部<ref group="注">例如：原作〈依賴之章：夏天、海水、吞食美麗容貌的男人！〉。</ref>等。

; {{nihongo|'''鬼神前鬼'''|鬼神前鬼（ZENKI）<ref  group="注">電視動畫版標記為「ZENKI」。</ref>||[[配音員|CV]]：（日本）[[小杉十郎太|小杉十郎太]]、（臺灣共通）[[魏伯勤|魏伯勤]]、（香港）[[林保全|林保全]]、（香港VCD版）[[雷霆|雷霆]]}}
: 前鬼被解開第二段封印時最初的形體，擁有驚人的力量。初期以受役小角賦予、從自己右手肘冒出的{{nihongo|「金剛角」|金剛角（ヴァジュラ）}}作為武器，搭配小明的咒術可使出各種招式。首次被小明解開第二段封印時想要殺了小明以重獲自由，卻被小明的護法輪變回小孩的型態而作罷，隨即放棄殺了小明，但經常與小明吵架，常用「臭丫頭<!--（長鴻代理譯作「臭三八」。）-->」對小明稱呼。
: 在原作中，由於役小角秘傳的咒語{{nihongo|「鬼神解縛秘咒」|鬼神解縛秘呪（きしんげばくひじゅ）}}咒文不完全，無法發揮全部力量。
: 在電視動畫版中期，得到黃金龍化身的「黃金斧」作為武器，卻因為無法駕馭而陷入暴走狀態。
: 絕招為「極霸光」（{{lang|ja|極覇光（ルドラ）}}，中視版譯為「極道光」），引用電光之力向周遭釋放。若全力而為，則能將方圓數-{里}-夷為平地；但同時也會反傷其身，輕則虛脫乏力，重則消滅殆盡。

:: 在原作中，小明唸完「鬼神解縛秘咒」全部咒文以後，前鬼的完全體。可使用自己與生俱來的「雷之力」。金剛角斷裂以後，藉由小明的力量，長出「金剛光角」作為武器。因使用極霸光而全身消滅殆盡。

:: 在原作中，這是前鬼因使用極霸光而全身消滅殆盡以後，藉由小明的「生命之光」與後鬼的「轉生之陣」，所生的形體。在電視動畫版後期，這是前鬼以自己的力量，加以小明和後鬼的力量，進化而成的形體。擁有所向披靡的力量。

:: 在原作中，最後決戰時，前鬼的形體。可輕易解決超鬼神前鬼難以對付的敵人。

: [[平安時代|平安時代]]人物，紀州高野的高僧，為告知役小綾黃金斧的秘密而來，後協助小明一行人見到黃金龍取得黃金斧。
; {{nihongo|'''寄生果'''|憑依の実〔ひょういのみ〕}}
: 外型酷似眼球，會附身到有著強烈慾望的生物上，並擴大其慾望。當吸收寄主足夠慾望或受到邪力催化時，寄生果會轉化成寄生獸，吞噬寄主意識；然而只要在寄主未被完全支配前打倒寄生獸，寄生果就會與寄主分離，寄主就不會有生命危險。一個寄生果只能附身在一個寄主身上。同一個寄生果，無論更換多少個寄主，都只能轉化成同一隻寄生獸，寄生獸的力量隨寄主而異。在原作中，寄生果附身時及轉化成寄生獸時，會以[[心電感應|心電感應]]向寄主傳達簡短的訊息（例如「我會實現你的願望」之類）；在電視動畫版中則無。
: 在電視動畫版故事中期，羯磨死後殘存的怨念，使寄生果進化成{{nihongo|「羯磨果」|カルマの実}}，寄生獸轉化成更強大的{{nihongo|「羯磨獸」|羯磨獣}}。

== 電視動畫 ==
日本國內，1995年1月9日起至同年12月25日為止，毎週一的晚間18時00分至18時30分（[[日本標準時間|日本標準時間]]）在[[東京電視網|東京電視網]]（TXN）首播。全51話。此外該動畫也是日本至今唯一一部仍在{{link-ja|瀨戶內電視台製作動畫時段|テレビせとうち制作アニメ枠}}播出的少年漫畫改編動畫。並在首播期間曾經錄製成[[VHS|VHS]]和[[LD|LD]]，均由[[波麗佳音|Pony Canyon]]發行。

台灣方面，台灣首播權則由[[中國電視公司|中國電視公司]]取得（以下簡稱「中視版」），並以雙語播出，主聲道為國語配音，副聲道為日語配音。除了[[八大電視|八大電視]]重播時另外製作國語配音（以下簡稱「八大版」）以外，台灣其他電視頻道播的國語配音版本都是中視版<ref group="注">此外台灣首播日期約在1995年至1996年間，某一回[[陳美鳳_(台灣)|陳美鳳]]與[[周麟_(藝人)|周麟]]2人一起主持中視週六午間[[綜藝節目|綜藝節目]]《[[歡樂傳真|歡樂傳真]]》時，在節目尾聲由擔任來賓的[[鄭進一|鄭進一]]介紹了接下來將要播放的《鬼神童子ZENKI》。1997年8月，[[中華電視公司|中華電視公司]]週六晚間綜藝節目《[[紅白勝利|紅白勝利]]》跳槽中視以前，中視曾在週六晚間連續播出《鬼神童子ZENKI》與《[[柔道英雌|YAWARA！]]》各一集。</ref>。

動畫版的基本設定大都忠實於漫畫原作，與漫畫版不一樣的是各回登場的角色、憑依獸有些微的不同。而主要角色後鬼的設定也和漫畫版大不同。由於漫畫版有很多[[殺必死|殺必死]]（[[裙底風光|露內褲]]、露出[[乳頭|乳頭]]）、[[性感|性]]描寫及血腥殘虐的成人畫面，因此動畫播出時則是將這些成人畫面予以控制注<ref group="注">不過只有第1集一直到故事終盤性描寫的表現大多集中在主角役小明身上。</ref>。還有，動畫版第3話的內容改編自短篇版、第31話的內容改編自漫畫連載期間發表的外傳。

2007年4月17日，由[[Kids_Station|Kids Station]]以每週一集方式進行實質的再放送。同時TPO錄製成總共2卷的DVD-BOX組在網路以限定版的方式持續進行販售至今，並附贈主角役小明的[[黏土人|黏土公仔]]。
