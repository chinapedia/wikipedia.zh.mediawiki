'''鬼星團'''，也稱為'''蜂巢星團'''（拉丁文是"Praesepe"）、'''M44'''、'''NGC 2632'''或'''Cr 189'''，是位於[[巨蟹座|巨蟹座]]的一個[[疏散星團|疏散星團]]。它是最靠近[[太陽系|太陽系]]的疏散星團之一，並且有著比其它鄰近疏散星團更多的恆星。在黑暗的夜空下，裸眼看見的鬼宿星團像是一個模糊的斑塊，因此在遠古時代就有紀載，中國稱他為積尸氣。古代的天文學家[[托勒密|托勒密]]描述他是"巨蟹胸部的集團"，並且是[[伽利略|伽利略]]用望遠鏡研究的第一批天體之一
<ref name="Messier44">

}}</ref>。

== 歷史 ==
[[伽利略|伽利略]]是第一位使用望遠鏡觀察鬼宿星團的人，並且解析出40顆恆星，時為1609年。[[查爾斯·梅西耶|梅西爾]]在1769年精確的測量它在天空中的位置，並將它加入[[梅西爾天體|他的著名目錄]]中。大多數的梅西爾天體都是黯淡而且容易與彗星混淆的天體，但是[[獵戶座大星雲|獵戶座大星雲]]（M42，M43）、[[昴宿星團|昴宿星團]]（M45）和鬼宿星團（M44）也在目錄中，實在令人好奇。一種可能性是梅西爾想超越它的競爭對手，[[尼可拉·路易·拉卡伊|拉卡伊]]，在1755年的目錄中所登載的42個天體，因此它添加了這些明亮、知名的天體，以提升它目錄中天體的數量<ref>

 |author      = Frommert, Hartmut
 |date        = 1998
 |url         = http://messier.seds.org/m-q&a.html#why_M42-45

 |accessdate  = 2005-03-01

}}</ref>。

古代希臘人與羅馬人將鬼宿星團視為2隻驢子的食槽，[[鬼宿三|鬼宿三]]（巨蟹座γ星）及[[鬼宿四|鬼宿四]]（巨蟹座δ星）則分別代表這2隻驢子，牠們也是[[戴奧尼索斯|戴奧尼索斯]]與[[西勒諾斯|西勒諾斯]]（Silenus）踏上討伐[[泰坦|泰坦]]旅途上的坐騎<ref>
{{cite web |title=M44 |url=http://messier.seds.org/m/m044.html |publisher=[[SEDS|SEDS]]|accessdate=2005-02-06}}</ref>。

[[喜帕恰斯|喜帕恰斯]]（' c '.130 BC）在他的星表中稱這個星團是"小小的雲"（'Nephelion'）<ref name=Hinckley>{{cite book|url=http://www.constellationsofwords.com/stars/Praesaepe.html|title=Star Names |first=Richard  Hinckley|last=Allen|date=1889|page=112}}</ref>；  [[托勒密|托勒密]]的''[[天文學大成|天文學大成]]''中包括7個星雲（其中4個是真實的星雲），鬼宿星團即為其中之一<ref>http://messier.seds.org/xtra/history/deepskyd.html#ptolemy</ref>，他描述其為"巨蟹胸部的集團"  <ref>{{cite web|url=http://messier.seds.org/m/m044.html|title=Messier Object 44|publisher=[[SEDS|SEDS]]|accessdate=2013-09-28}}</ref>。[[亞拉|亞拉]]（''c''.260-270 BC）在他的詩歌' Phainomaina '描述這個星團像"小霧"（或''Achlus''）<ref name=Hinckley/>。

在中國，這個模糊的斑塊位於28宿的第23宿，[[鬼宿|鬼宿]]。28宿是國古天文學使用的星空劃分，類似於黃道星座。中國古代的天文學家認為這是魂魄或魔鬼乘坐的馬車，而其外觀如同柳絮與其種子般飄逸，柔弱而沒有活力，因此被稱為帶有些許禪意的"積尸氣" <ref name="Hinckley"/>；它也會簡略的被稱為"積尸"。

== 組織和成分 ==
像所有的[[星團|星團]]一樣，鬼宿星團也有[[質量層化|質量層化]]<ref name="Kraus2007"/><ref name="Adams2002"/><ref name="Portegies2001">

 |author=Portegies Zwart SF
 |author2=McMillan SL
 |author3=Hut P
 |author4=Makino J
 |date=2001
 |title=Star cluster ecology IV. Dissection of an open star cluster: Photometry
 |journal=[[Monthly_Notices_of_the_Royal_Astronomical_Society|Monthly Notices of the Royal Astronomical Society]]
 |volume=321 |pages=199–226
 |bibcode= 2001MNRAS.321..199P
 |doi=10.1046/j.1365-8711.2001.03976.x
|arxiv = astro-ph/0005248 }}</ref>，這意味著明亮的大質量恆星都集中在星團的核心，而黯淡、質量較低的恆星則填充在暈部（有時稱為冕）。估計這個星團的核心半徑大約是3.9秒差距（12.7光年），它的[[洛希極限|潮汐半徑]]約12秒差距（39光年）<ref name="Kraus2007"/><ref name="Adams2002"/>。但是，潮汐半徑還包括"路過"的恆星和"不懷好意"的星團成員。

這個星團至少包含1,000顆受到引力束縛的恆星，總質量約為500-600太陽質量<ref name="Kraus2007"/><ref name="Adams2002"/>。最近的一項調查1010顆高度可能是成員的恆星，其中68%是[[紅矮星|M矮星]]，30%是類太陽的F、G、和K型恆星，大約2%是明亮的A型恆星<ref name="Kraus2007"/>。其中有5顆巨星，其中4科的光譜是K0 III，第5顆是G0 III<ref name="Klein-Wassink1927"/><ref name="Kraus2007"/><ref name="Abt1999">

== 行星 ==
在2012年9月，在鬼宿星團內發現了兩顆行星，各別環繞著母恆星。這一發現是首度在星團中發現像[[太陽|太陽]]的恆星有像地球的行星環繞著。在星團中曾經發現過行星，但是沒有繞著像太陽的恆星公轉。

這兩顆恆星已被命名為Pr0201b和Pr0211b，結尾的b表示是行星。被發現的這兩顆行星都是熱木星，它們是大質量的[[氣體巨星|氣體巨星]]，但與[[木星|木星]]不同的是軌道非常靠近其母恆星。
 
描述發現行星的公告發表在[[天文物理期刊|天文物理期刊]]，以薩姆·奎因為第一作者的函件形式刊出。奎因的團隊與[[哈佛-史密松天體物理中心|哈佛-史密松天體物理中心]]的大衛萊瑟姆使用[[史密松天體物理台|史密松天體物理台]]的[[弗雷德·勞倫斯·惠普爾天文台|弗雷德·勞倫斯·惠普爾天文台]]<ref>{{Cite web|url=http://www.jpl.nasa.gov/news/news.php?release=2012-289|title=First Planets Found Around Sun-Like Stars in a Cluster|accessdate=September 14, 2012}}</ref>。
