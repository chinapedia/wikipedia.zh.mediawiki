'''鬼筆科'''（英文：Phallaceae）是[[鬼筆目|鬼筆目]]下兩個[[科_(生物)|科]]的其中一個，其[[菌蓋|菌蓋]]不成傘狀。多具有臭味以吸引蒼蠅助其繁殖。鬼筆科多半出現於[[熱帶|熱帶地區]]，而其底下共有21[[屬_(生物)|屬]]和77[[種_(生物)|種]]<ref name="urlMycoBank: Lysuraceae">{{cite web |url=http://www.mycobank.org/MycoTaxo.aspx?Link=T&Rec=80970 |title=Lysuraceae Corda 1842 |publisher=International Mycological Association |work=MycoBank |accessdate=2010-08-06}}</ref>。

== 描述 ==
鬼筆科的[[孢托|孢托]]（成熟[[子實體|子實體]]）一般為筆狀，且為[[海綿質|海綿質]]，頂部有鐘形[[子層托|子層托]]，上有青綠色的孢子黏液物覆蓋，還不時發出惡臭，因而吸引蠅類、蝸牛及螞蟻咬噬，故其菌蓋多半坑坑洞洞的。<ref name="AKAKAKAK">[http://edresource.nmns.edu.tw/ShowObject.aspx?id=0b819d5e7e0b81d9d3360b81da4a44]</ref>

== 繁殖 ==
鬼筆科屬於[[腹菌類|腹菌類]]，成熟的[[孢子|孢子]]是包埋在包被內，等[[子實體|子實體]]破裂後才會裸露散發出孢子，不同於[[傘菌類|傘菌類]]孢子成熟後裸露在空氣中飄散的形式。<ref name="AKAKAKAK"></ref>
== 食用 ==
雖然鬼筆科的[[真菌|菌類]]多半會發出惡臭，不過仍有一些被當作高級食材來食用。例如：在中國，[[竹蓀屬|竹蓀屬]]中的菌類，的[[菌裙|菌裙]]和[[菌柄|菌柄]]被當作高級的食材。<ref>[http://www.2to1agri.com/apagri/16.nsf/ByUNID/317E7C1DD7783B0948256B8C0016E724?opendocument]</ref>例如：[[竹蓀屬|長裙竹蓀]]多與肉共煮，成肉湯，味美。1990年代人工培植成功後，常見於[[齋菜|齋菜]]中。早期出產的仍有很濃的臭味和滑潺潺的，後期出產的已有改善。至今野生竹蓀仍然是名貴食材。
