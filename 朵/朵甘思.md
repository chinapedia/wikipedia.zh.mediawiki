{{西藏歷史}}
'''朵甘思'''（{{bo|z=མདོ་ཁམས|w=mdo khams}}），又名'''朵甘'''、'''朵康'''<ref name=cgjhsc>{{cite journal |journal=長庚人文社會學報 |url=http://cgjhsc.cgu.edu.tw/data_files/CGJ3-2-01.pdf |title=漢藏歷史關係的新思考：一個反思性歷史研究 |author=[[王明珂|王明珂]] |volume=3 |issue=2 |year=2010 |pages=221-252 |language=zh-hant |deadurl=no |archiveurl=https://web.archive.org/web/20160304185930/http://memo.cgu.edu.tw/cgjhsc/CGJ3-2-01.pdf |archivedate=2016-03-04 }}</ref>、'''多康'''等，为[[中国|中国]]古代一地区名。范围大致相当于今[[西藏自治区|西藏自治区]][[昌都地区|昌都地区]]、[[青海省|青海省]][[玉树藏族自治州|玉树藏族自治州]]东部和[[四川省|四川省]][[甘孜藏族自治州|甘孜藏族自治州]]。

朵甘思的意思是汇合的区域。近代一般简称[[康区|康]]（{{bo|z=ཁམས|w=khams}}）。

== 吐蕃王朝 ==
[[吐蕃王朝|吐蕃王朝]]7世纪开始对外扩张，其向东征服的地区被称为朵甘思，与[[卫藏四茹|卫藏四茹]]相区别。后来朵甘思北部被分出[[朵思麻|朵思麻]]，南部仍被称为朵甘思。

== 元朝 ==
{{Main|元朝治藏歷史}}

[[忽必烈|忽必烈]]曾经过此处南征[[大理|大理]]。[[元朝|元]][[中統_(年號)|中统]]和[[至元_(忽必烈)|至元]]年间，该地区各部落相继投向元朝政府。至元十三年（1276年），设宁远府（府城在今四川[[乾寧縣|乾宁]]北），至元十五年（1278年），又设李唐州（今四川[[理塘|理塘]]）。

元灭[[南宋|南宋]]之后，将原[[吐蕃|吐蕃]]之地划归[[宣政院|宣政院]]管理，朵甘思属[[吐蕃等路宣慰使司都元帅府|吐蕃等路宣慰使司都元帅府]]，又被称为朵甘思宣慰司。

== 明朝 ==
{{Main|明朝治藏歷史}}

[[明朝|明朝]]在当地置[[朵甘衛|朵甘衛]]，後來升為[[朵甘行都指挥使司|朵甘行都指挥使司]]、[[朵甘行都指挥使司|朵甘都指挥使司]]。
