'''孙破虏吴夫人'''（{{bd|？||202年<ref name="卒年">卒年有争议，裴松之根据《会稽典录》记载的“建安十二年到十三年阙，无举者，云府君遭忧”，认为吴夫人理应卒于207年而并非202年。</ref>}})，名字不详，[[吴郡|吴郡]][[吴县|吴县]]（今[[江苏省|江苏省]][[苏州市|苏州市]]）人，是江东孙氏政权的开创者[[孙坚_(东吴)|孙坚]]的[[嫡妻|元配妻子]]，[[孙策|孙策]]、[[孙权|孙权]]、[[孫翊|孫翊]]、[[孫匡|孫匡]]的亲生母亲，又稱為'''吴夫人'''。次子孙权自立为吴国[[皇帝|皇帝]]后，她受追尊为'''武烈皇后'''<ref>中国古代皇帝常常追尊生前未成为皇帝的生父为皇帝，相应的，生前未成为皇后的生母则被追尊为皇后，因为孙坚被孙权追尊为'''武烈皇帝'''，而吴夫人則受追尊为'''武烈皇后'''。</ref>。

== 生平 ==
===才貌雙全===
父親是丹陽太守吴煇，字光修<ref>《太平寰宇记》：吴煇字光修,丹阳太守,吴景父也</ref>，吳氏一族本來是[[吴郡|吴郡]][[吴县|吴县]]人（今[[江苏省|江苏省]][[苏州市|苏州市]]），後來遷徙至浙江[[钱唐县|钱唐]]，因為雙親早亡，所以吳氏與弟弟[[吴景|吴景]]同住<ref>《三國志·吳書·孫破虜吳夫人傳》：本吳人，徙錢唐，早失父母，與弟景居。</ref>。

[[孙坚|孙坚]]聽聞吴氏的才藝與美貌兼全，想要娶她。吴夫人的亲戚们嫌弃孙坚为人輕佻狡詐，所以將孫堅拒絕。孙坚對此感到非常羞愧和遗憾。吴夫人知道孙坚不好惹，就对亲戚们说：“为什么要因为吝惜我这个小女子而招惹祸事呢？如果孫堅待我不好，也是我命该如此。”<ref>《三国志·吴书·妃嫔传》载吴夫人原话：“何爱一女以取祸乎？如有不遇，命也。”</ref>于是这样，吴夫人便嫁给了孙坚。吴夫人共为孙坚生下了四子一女：[[熹平|熹平]]四年（公元175年）生长子[[孙策|孙策]]；[[光和|光和]]五年（182年）生次子[[孙权|孙权]]；光和七年或[[中平|中平]]元年（184年）生三子[[孙翊|孙翊]]；之后又生幼子[[孙匡|孙匡]]和女儿。

===異奇貴象===
吳氏當初懷孕的時候，夢見月亮進去懷裡，之後生下了孫策。及後在懷孫權的時候，又夢見太陽進去懷裡。之後告訴孫堅說：“妾昔日懷著孫策的時候，夢見月亮入懷裡；如今又夢見太陽入懷裡，為什麼會這樣呢？”孫堅回答：「太陽和月亮，是陰陽的能量精氣，是極其貴象的征兆。我們的子孫大概會興家赤旺吧！」<ref>《搜神記》：孫堅夫人吳氏，孕而夢月入懷。已而生策。及權在孕，又夢日入懷。以告堅曰：「妾昔懷策，夢月入懷；今又夢日，何也」堅曰：「日月者，陰陽之精，極貴之象，吾子孫其興乎。」</ref>。漢[[光和|光和]]五年（182年），孫堅擔任下邳县丞的時候嫡次子孫權出生，生母为吳郡豪族[[吴夫人|吴夫人]]，其面相方頰大口，銳目有神，孫堅覺得驚奇，認為有貴氣的象相。

184年，孙坚募诸商旅及淮、泗精兵，合千许人，与朱儁征讨黄巾军。在發軍之前，吴夫人等人都被留在寿春。时任[[长沙郡|长沙郡]]（[[治所|治所]]在今[[湖南省|湖南省]][[长沙市|长沙市]]）[[太守|太守]]的孙坚起兵响应[[董卓讨伐战|讨伐董卓]]的[[关东|关东]]联军，舉家遷至[[庐江郡|庐江郡]][[舒县|舒县]]（今[[安徽省|安徽省]][[庐江县|庐江县]]西南），吴夫人母子们均随行。

===撫育嚴施===
191年，袁術派孙坚攻打[[襄陽|襄陽]][[刘表|刘表]]，戰鬥途中孫堅戰死，吳氏當時家住在舒縣，她撫養教育孫堅留下的幼兒，作為單身母親嚴施教育<ref>《建康實錄》：後（孫）堅薨，夫人家於舒，撫育孤幼，嚴於母訓。</ref>。當時孙策在袁術麾下，吴氏的弟弟[[吴景|吴景]]，当时任[[丹杨郡|丹杨郡]]（治所在[[宛陵县|宛陵县]]，今[[安徽省|安徽省]][[宣城市|宣城市]]）[[太守|太守]]，但未到任，留在[[吴郡|吴郡]][[曲阿县|曲阿县]]（今[[江苏省|江苏省]][[丹阳市|丹阳市]]）<ref>吴景是[[袁术|袁术]]向东汉朝廷所推荐的丹杨太守，当时还有在任的丹杨太守[[周昕|周昕]]，但是吳景与袁术交恶，所以暂时留在曲阿。</ref>，于是孙策用船载着母亲前往曲阿与舅舅会合，也将父亲的大體暂葬曲阿。

194年，孙策为[[袁术|袁术]]攻打庐江，扬州牧[[刘繇|刘繇]]因此不满和猜疑。当时吴夫人和幼子们都处于刘繇的势力之下，处境岌岌可危。孙坚旧将[[朱治|朱治]]派人赴曲阿接到吴夫人一家，提供照料。可是袁术后来听说孙坚获得了[[傳國璽|傳國璽]]，拘禁吴夫人强行抢夺玉璽<ref>《後漢書·劉焉袁術呂布列傳》、《三國志·孫破虜討逆傳》注引《山陽公載記》</ref>。在孙策征讨江东各地的过程中，吴太夫人从曲阿迁往[[历阳县|历阳县]]（今安徽省[[和县|和县]]），又迁往[[阜陵县|阜陵县]]（今安徽省[[全椒县|全椒县]]附近）。孙策征服江东吴郡（治所在[[吴县|吴县]]，今江苏省[[苏州市|苏州市]]）、[[会稽郡|会稽郡]]（治所在[[山阴县_(古代)|山阴县]]，今[[浙江省|浙江省]][[绍兴市|绍兴市]]）等地后，驻扎在吴郡吴县（今江苏省苏州市），吴太夫人也因此回到了故乡。

据《[[江表传|江表传]]》记载，孙策在治理江东和吴郡的末期，逮捕了传教行医的[[道士|道士]][[于吉|于吉]]，想要杀掉他。很多人同情于吉，他们让妇女们拜见孙策的母亲吴太夫人，请她救于吉。吴太夫人便对孙策说，于吉为将士们治病疗伤也是为了帮助孙策的军队，不可以杀<ref>《三国志·吴书·孙破虏讨逆传》裴松之注引《江表传》载吴太夫人原话：“于先生亦助军作福，医护将士，不可杀之。”</ref>。但孙策不听，最终将于吉杀害。该事件后来被写入《搜神记》，更被[[罗贯中|罗贯中]]写入《[[三国演义|三国演义]]》，并加以神化，成为孙策的重要死因之一。

200年，孙策遇刺身亡，吴太夫人又辅佐年轻的次子[[孙权|孙权]]，她对孙氏政权充满关切，引见张昭以及董袭等人，问江东是否可以保全，以观察众人的心意。董袭回答说:"江东地势，有山川之固，昔日讨逆将军对民众有恩，现在讨虏将军继承基业，大小事务由张昭主持，我等为爪牙。如今是地利人和的时机，没有什么好担忧的。“受到了众人的赞同。<ref>《吴书·董袭传》：策薨，权年少，初统事，太妃忧之，引见张昭及袭等，问江东可保安否，袭对曰：‘江东地势，有山川之固，而讨逆明府，恩德在民。讨虏承基，大小用命，张昭秉众事，袭等为爪牙，此地利人和之时也，万无所忧。’众皆壮其言。”</ref>吴夫人在军事和政治方面都对孙权特别有帮助<ref>《吴书 张严程阚薛传第八》：孙权初承统，春秋方富，太夫人以方外多难，深怀忧劳，数有优令辞谢，付属以辅助之义。每有异事密计及章表书记、与四方交结、常令紘与张昭草创撰作</ref>。吴太夫人去世时，临终召见[[张昭|张昭]]，嘱托后事。

== 身后事与遗产 ==

[[File:北寺塔.jpg|right]][[报恩寺塔_(苏州)|报恩寺塔]]]]

吴太夫人去世后，与夫君[[孙坚|孙坚]][[合葬|合葬]]，两人的合葬墓位于今[[苏州市|苏州]][[盘门|盘门]]外青旸地<ref>根据《苏州历史》，孙坚本葬曲阿（今丹阳市），后迁葬吴县（今苏州市）。</ref>，俗称“[[孙王坟|孙王坟]]”，[[孙权|孙权]]称帝后改“墓”为“陵”<ref>皇帝的坟墓称“陵”，因为孙坚已经被追尊为“武烈皇帝”。</ref>，号“[[高陵_(东吴)|高陵]]”。而吴太夫人的长子[[孙策|孙策]]的墓也在附近，俗称“[[孙将军坟|孙将军坟]]”。

[[黄武|黄武]]八年[[夏|夏]][[四月十三|四月十三]][[丙申|丙申]]日（公元229年5月23日），孙权自立为[[皇帝|皇帝]]，建立[[东吴|吴国]]，追尊父亲孙坚为'''武烈皇帝'''，同时追尊母亲吴太夫人为'''武烈皇后'''。

孙权称帝后，为了报答母恩，于[[赤乌|赤乌]]年间（238年－251年）在吴太夫人的故居（今[[苏州市|苏州市]]人民路北北塔公园内）建造了一座追念母親的寺塔，名为“报恩寺”（[[南朝梁|南朝梁]]和[[南宋|南宋]]时两次重建，又经历代兴废，现寺已不在，仅存宝塔，塔名[[报恩寺塔_(苏州)|报恩寺塔]]），今俗称[[北寺塔|北寺塔]]，是苏州古城的标志性建筑之一。

[[明代|明代]][[罗贯中|罗贯中]]所著的小说《[[三国演义|三国演义]]》中也出现了吴太夫人。事蹟大致與历史相合，但描写吴太夫人有一个妹妹也嫁给了孙坚，即[[吴国太|吴国太]]。[[正史|正史]]中没有提到吴太夫人有妹妹，只说孙坚有一个[[庶出|庶出]]的儿子即[[孙朗|孙朗]]，后人就以这一点推断孙坚除了[[嫡妻|嫡妻]]吴太夫人外，至少还有一个[[妾室|妾室]]，但没有证据表明这个妾室就是吴太夫人的妹妹。不过，小说中说吴太夫人未有生女，是妹妹吴国太生了女儿即[[刘备|刘备]]的[[孫夫人|孙夫人]]（传说名孙尚香，又名孙仁）<ref>《三国演义》第七回《袁绍磐河战公孙，孙坚跨江击刘表》载：“却说孙坚有四子，皆吴夫人所生：长子名策，字伯符；次子名权，字仲谋；三子名翊，字叔弼；四子名匡，字季佐。吴夫人之妹，即为孙坚次妻，亦生一子一女：子名朗，字早安；女名仁。”</ref>，这一点和史实不符；史料记载了吴太夫人确实生了女儿<ref>《三国志·吴书·妃嫔传·孙破虏吴夫人》：“生四男一女。”</ref>，但这个女儿是不是后来嫁给刘备的孙夫人則未明说。正史中，除了孙夫人外，孙坚有女儿嫁给[[弘咨|弘咨]]，也有女儿嫁入陈家生女陈氏，故無以确定哪个女儿为吴太夫人所生。

== 影响与评价 ==

吴太夫人在孙策治理江东的过程中起过积极作用。据《[[会稽典录|会稽典录]]》记载，孙策的[[功曹|功曹]][[魏腾|魏腾]]，因为违背孙策的主张而遭责备，孙策甚至想要杀了他。[[士大夫|士大夫]]们忧虑惊恐，但想不到办法救魏腾。吴太夫人知道后，靠在一口大井边，说孙策刚征服[[江南|江南]]<ref>三国时代的江东和后来的江南大体上范围一致。</ref>不久，还没有完全成功，应当对贤士以礼相优待，忘却他们的过失而凭他们的功劳加以录用；又说魏腾于公并没有错，如果孙策杀了他，日后众人都会背叛；最后她说，如果孙策不听她的话，她就先投井自尽，以免看见日后親眼見到孫策众叛亲离的下场。<ref>吴太夫人原话：“汝新造江南，其事未集，方当优贤礼士，舍过录功。魏功曹在公尽规，汝今日杀之，则明日人皆叛汝。吾不忍见祸之及，当先投此井中耳。”</ref>孙策大惊，只好放了魏腾。这也说明了吴太夫人的智慧与权谋<ref>《三国志》评价：夫人智略权谲，类皆如此。</ref>。

年少的孙权继承父兄基业后，吴太夫人又辅佐孙权治理军国大事，继续发挥积极作用，对巩固孙氏父子在江东的统治帮助很大<ref>《三国志·吴书·妃嫔传·孙破虏吴夫人》：“及权少年统业，夫人助治军国，甚有补益。”</ref>。据《江表传》记载，在吴太夫人去世的那一年（[[建安|建安]]七年，即公元202年），击破[[袁绍|袁绍]]不久的[[曹操|曹操]]兵威日盛，写信给孙权，要求他把自己的儿子送到[[东汉|东汉]]首都[[许县|许县]]（今[[河南省|河南省]][[许昌市|许昌市]]）作[[人质|人质]]。孙权召集群臣商议对策，群臣都犹豫不决。孙权不想送儿子当人质，便与周瑜两个人拜见吴太夫人，以求定夺。周瑜坚持认为不能送人质，并对太夫人和孙权陈述利害。吴太夫人也认为周瑜说得对，又因为周瑜与她死去的长子[[孙策|孙策]]同年，让孙权把周瑜当成兄长来对待<ref>《三国志·吴书·周瑜传》裴松之注引《江表传》载吴太夫人原话：“公瑾议是也。公瑾与伯符同年，小一月耳，我视之如子也，汝其兄事之。”</ref>，最终决定不送人质。
