《'''孫子兵法'''》，即《'''孫子'''》，又稱作《'''武經'''》、《'''兵經'''》、《'''孫武兵法'''》、《'''吳孫子兵法'''》，是中國古代的[[兵書|兵書]]，作者為[[春秋時期|春秋]]末期的[[齊國|齊國]]人[[孫武|孫武]]（字長卿）。一般認為，《孫子兵法》成書於[[專諸|專諸]]刺[[吳王僚|吳王僚]]之後至[[闔閭|闔閭]]三年孫武見吳王之間，也即前515至前512年，全書為十三篇，是孫武初次見面贈送給吳王的見面禮；事見[[司馬遷|司馬遷]]《[[史記|史記]]》：「孫子武者，齊人也，以兵法見吳王闔閭。闔閭曰：子之十三篇吾盡觀之矣」<ref>[[司馬遷|司馬遷]]《[[史記|史記]]》卷六十五列傳第五孫子吳起列傳</ref>。

== 特色 ==
* [[太極|太極]]的思想
「形兵之極，至於無形」

* 慎戰的思想
「兵者，國之大事，死生之地，存亡之道，不可不察也」

* 全爭的思想
「必以全爭於天下，故兵不頓而利可全」

* 先勝的思想
「勝兵先勝而後戰，敗兵先戰而後求勝。」，「昔之善戰者，先為不可勝，以待敵之可勝」

== 兵法概述 ==
* 孫子認為戰爭是國家大事，關係國家生死存亡，不可不研究明察。（兵者，國之大事，死生之地，存亡之道，不可不察也。）
* 嚴格說來，孫子是一個“反戰者”，这里的战特指正面战场上的军事行为以及战斗。原因在於'''[[戰爭|戰爭]]是[[資源|資源]]消耗戰'''（日費千金），而且正面战场的战斗，即使能够击溃敌军，己方的损失也是不可避免的。最佳的勝利就是不戰而勝，也就是不通过正面战斗，而通过其他的方法来对敌军进行消耗从而击溃敌人（善戰者，不戰而屈人之兵；或全國為上，破國次之）。
* 孫子最早在兵法上提出軍事[[情報|情報]]的重要性，他主張知彼知己，百戰不殆；不知彼而知己，一勝一負；不知彼不知己，每戰必殆。一方面要用各種方法（包括派[[間諜|間諜]]）獲取、探取敵方軍情、軍力、部署、動向、補給，同時將本身的軍情嚴格保密，或發放假軍情，虛虛實實，詐騙敵方，令敵方信假為真。
* 孫子強調備戰。（故用兵之法，勿恃敵之不來，恃吾有以待之；勿恃敵之不攻，恃吾有所不可攻也。）
* 孫子說最上等的作戰是運用謀略，其次是運用外交手段，再其次是動用軍隊，最下等的作戰是強攻城池。（上兵伐謀；其次伐交；其次伐兵；其下伐城。）
* 孫子時常有[[譬喻|譬喻]]式的描述，諸如他注重勢與節（[[時機|時機]]），形容前者如湍急[[河|河]]水上的漂石，後者如飛鳥攻擊獵物；其中最有名者，乃後來被[[武田信玄|武田信玄]]所使用的[[風林火山|風林火山]]之旗：「其疾如風，其徐如林，侵掠如火，不動如山，難知如陰，動如雷霆。」
* 孫子主張，戰爭的勝利在於敵方露出破綻，有可趁之機。
* 孫子主張，用兵在於正奇相用（凡戰者，以正合，用奇勝。）：正可視之為按牌理出牌，奇可視為不按牌理出牌，這兩者沒有一定規則，不需拘泥。後，在《[[唐太宗李衛公問對|李衛公問對]]》裡面，[[唐太宗|唐太宗]]特別問[[李靖|李靖]]正奇之間的比例，李靖回答沒有一定。
* 孫子主張，戰場上的情況只有帶兵的[[將領|將領]]了解，明知道進攻會失敗，就算[[君主|君主]]下令進攻也要違抗「將在外，君命有所不受。」
* 談到對敗兵的處置，特別主張「留其退路」，以避免絕地反撲。所以孫子主張：窮寇莫追，圍師必闕。
* 孫子指善用兵者行軍時重視軍隊各位置的相互照應，要像[[率然|率然]]（一種[[蛇類|蛇類]]）一樣可以首尾相救應。
* 戰場局勢，類似出奇兵原則，孫子也主張「投之亡地然後存，陷之死地然後生」，意味要考量部隊特色，適時激勵拼死一搏的氣勢。這一點在後來[[漢|漢]][[韓信|韓信]][[井陘之戰|井陘之戰]]背水一戰大破[[趙國|趙國]][[陳餘|陳餘]]時使用。
* 孫子認為兵無常勢，就好像水一樣，形勢不同，所以可以因敵人的變化，而取得勝利，方能用兵如神。（故兵無常勢，水無常形；能因敵變化而取勝者，謂之神。）

== 編撰過程 ==
孫武原是[[齊國|齊國]]人，後來輾轉到[[吳國|吳國]]。當時適逢公子光政變夺取王位，是為[[吳王闔閭|吳王闔閭]]，[[伍員|伍員]]聽說其才能，向吳王推薦，孫武就帶著這13篇晉見吳王，獲得重用。後，吳王進攻[[楚國|楚國]]，將領就是孫武與伍員<ref>{{Cite web|url=http://www.epochtimes.com/b5/1/7/6/c2722.htm|title=春秋末期吳國軍事思想家－－孫武|accessdate=2013-08-20|publisher=大紀元文化網|deadurl=yes|archiveurl=https://web.archive.org/web/20130528204004/http://www.epochtimes.com/b5/1/7/6/c2722.htm|archivedate=2013-05-28}}</ref>。

== 翻譯本 ==
自西元600多年左右，《孫子兵法》被翻譯成許多不同語言，開始流傳到世界各國。最初書版被帶到[[日本|日本]]，及後至[[東南亞|東南亞]]，再在18世紀尾傳入西方各國。此書在[[歐洲|歐洲]]最先出現的是法文版，由[[法國|法國]][[耶穌會士|耶穌會士]][[錢德明|錢德明]]（Jean Joseph Marie Amiot）在[[巴黎|巴黎]]於1772年出版，書名是''Art Militaire des Chinois''。

1905年，首個[[英文|英文]]版在[[東京|東京]]面世，翻譯者是[[英國|英國]]軍官卡爾斯羅普（Captain Everard Ferguson Calthrop），但可惜由於卡爾斯羅普不認識漢字，此版本是從日文譯過去的。在這以後，不同的英譯本陸續出版，到了現在約有17種英譯本，最受歡迎的是由[[美國|美國]]將軍（{{tsl|en|Samuel B. Griffith|塞繆爾．B．格里菲思}}（Samuel B. Griffith）翻譯的版本，但譯得最好的可能是由[[漢學家|漢學家]][[翟林奈|翟林奈]]（Lionel Giles）翻譯的版本<ref>[http://www.china.com.cn/zhuanti/zyw/2007-04/07/content_8081008.htm 《孫子兵法》西行路]</ref>。

== 影響 ==
=== 軍事方面 ===
吳王[[闔閭|闔閭]]稱：「子之十三篇，吾盡觀之矣。」表示他對孫武作品的欣賞。吳王知孫子能用兵，拜為大將。孫子以三萬兵，西破強楚二十萬兵，北威齊晉，顯名諸侯。因《孫子兵法》的關係，孫武被人譽作「兵聖」。

此書對中國古代的軍事學產生了巨大的影響，被奉為[[兵家|兵家]]經典。它是中國古代軍人必須研讀的一本軍事著作，許多著名的[[軍事家列表|軍事家]]都對此書作過注解。

《[[韓非子|韓非子]]》裡面也有提到孫武的故事（但[[韓非|韓非]]的故事僅支持其論點，未必真實發生）：某天[[吳|吳]]王[[闔閭|闔閭]]問[[孫武|孫武]]：「甚麼人都可以用兵法訓練嗎？」孫武答：「可以。」於是吳王把他的後宮佳麗全部交給孫武，要他練出一支娘子軍。這批娘子軍起初認為吳王只是開玩笑，仍在校場上嬉鬧。孫武為了建立軍紀，把帶頭的隊長，也是吳王最寵愛的兩個妃子當場[[斬首|斬首]]，其他佳麗於是震懾於孫武的威信，嚴格遵守規定，任憑孫武指揮。

[[明朝|明朝]]軍事家[[劉伯溫|劉伯溫]]的《[[百戰奇略|百戰奇略]]》包含孫子兵法。孫子：「知彼知己，百戰不殆」，「故知戰之地，知戰之日，則可千里而會戰」。劉伯溫：「凡興兵伐敵，所戰之地，必預知之。師至之日，能使敵人如期而來，與敵則勝。……知戰之地，知戰之日，則可千里而會戰。」[[杜牧|杜牧]]注孫子《用間篇》：「不知敵情，軍不可動；知敵之情，非間不可」：劉伯溫：「凡欲征戰，先用間諜，覘敵之眾寡、虛實、動靜，然後興師，則大功可立，戰無不勝。」

《孫子兵法》流傳出中國後，對[[東亞|東亞]]及[[東南亞|東南亞]]均有或輕或重不同程度的影響，而譯成西方語言後，在歐洲亦大受歡迎；相傳[[法國皇帝|法國皇帝]][[拿破崙|拿破崙]]更對此書愛不釋手。[[英國|英國]]軍事理論家[[李德·哈特|李德·哈特]]感稱：「在《孫子兵法》中，把我20多部著作所涉及的戰略和戰術原則幾乎包羅無遺。」<ref>利德爾．哈特，《戰略論：間接路線》鈕先鍾譯，474頁，麥田出版</ref>

《孫子兵法》中的幾個章節是[[中華民國政府|中華民國政府]][[軍訓|軍訓]]教育的教材，而目前的[[預官考試|義務役預備軍官考試]]亦將《孫子兵法》列為[[國文|國文]]科必考項目<ref>{{Cite web

=== 其他方面 ===
《孫子兵法》的思想影響超越軍事應用，實際上亦是[[博弈論|博弈策略]]的經典著作，如在[[棋藝|棋藝]]對壘或[[運動競技|運動競技]]方面。《孫子兵法》不僅在世界軍事領域發揮著重要的影響，對[[政治|政治]]、[[經濟|經濟]]、[[商業|商業]]、[[人事管理|人事管理]]和[[市場策略|市場策略]]等與博弈有關的領域亦有指導意義。法國談論日常生活戰略的{{tsl|fr|Michel de Certeau|Michel de Certeau}}，在《{{tsl|en|The Practice of Everyday Life|日常生活的戰略}}》（''The Practice of Everyday Life''）一書中，也提到應讀中國的經典《孫子兵法》與《[[易經|易經]]》。

2001年，[[何炳棣|何炳棣]]提出「老子辯證思維源於孫子兵法」的說法。
