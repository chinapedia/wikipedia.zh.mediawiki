'''孙红雷'''（{{bd|1970年|8月16日|catIdx=S孙}}），[[中国|中国]]内地影视[[男演员|男演员]]，毕业于[[中央戏剧学院|中央戏剧学院]]表演系音乐剧班。1999年因出演[[张艺谋|张艺谋]]电影《[[我的父亲母亲|我的父亲母亲]]》中的儿子一角，正式开始了其影视演艺生涯。其后的十余年间，孙红雷相继与[[徐克|徐克]]、[[冯小刚|冯小刚]]、[[陈凯歌|陈凯歌]]等知名华人[[导演|导演]]合作。

== 生平 ==
*孙红雷出生于[[哈尔滨|哈尔滨]]，小名三狼。父亲孙振山为一位哲学教师，母亲朱凤琴已经逝世，有两个哥哥。
*1987年，获[[黑龙江|黑龙江]]省[[霹雳舞|霹雳舞]]大赛一等奖，1988年获全国霹雳舞大赛第二名。
*1995年至1997年，在[[中央戏剧学院|中央戏剧学院]]学生表演系音乐剧班(补招)大专学习。1997年成为[[中国国家话剧院|中国国家话剧院]]演员。2001年[[北京鑫宝源影视投资有限公司|北京鑫宝源影视投资有限公司]]签约演员。

=== 感情生活 ===
*孙红雷的首任女友是[[丁嘉丽|丁嘉丽]]，两人在出演话剧《居里夫人》时认识和相恋，直到孙红雷认识巩俐之后，两人的2年的恋情结束。<ref name=" cat" >{{cite web |url=http://ent.ifeng.com/idolnews/mainland/detail_2013_01/04/20761630_0.shtml |title=揭孙红雷5段情史 女友选美照首曝光 |accessdate=2013年1月4日 |publisher=凤凰网 |archiveurl= http://ent.ifeng.com/idolnews/mainland/detail_2013_01/04/20761630_0.shtml|archivedate=2013年1月4日 |work= |language=zh-cn}}</ref>
*2004年，孙红雷在北京参加东北同乡会，遇到了[[李馨雨|李馨雨]]，在开车送李馨雨回家之后，两人交往日密，李馨雨爱上了孙红雷，但是孙红雷仍然爱着巩俐，当李馨雨盘问出这个结果之后，两人的感情结束。<ref name=" cat" />
*2013年1月，孙红雷女友曝光，女方名叫“王骏迪”，王骏迪是东北人，曾留学[[奥地利|奥地利]]，是女高音演員、模特、钢琴师及爵士鼓鼓手，曾得过多项艺术类奖项，两人已经爱情长跑6年<ref name=" cat" /><ref>{{cite web |url=http://ent.ifeng.com/idolnews/mainland/detail_2010_07/21/1810003_0.shtml?_from_ralated |title= 孙红雷好事临近明年“造人” 带女友巴黎买豪宅|accessdate=2010年7月21日 |publisher=凤凰网 |archiveurl=http://ent.ifeng.com/idolnews/mainland/detail_2010_07/21/1810003_0.shtml?_from_ralated |archivedate=2013年1月4日 |work= |language=zh-cn}}</ref>。
*2014年10月1日，孫紅雷在個人[[微博|微博]]公布婚訊，表示已和交往邁入7年的王駿迪在[[法國|法國]]結婚<ref>[http://star.chinatimes.com/news/20141001004266-261101 孫紅雷法國大婚 娶29歲小女友] {{webarchive|url=https://web.archive.org/web/20141001134422/http://star.chinatimes.com/news/20141001004266-261101 |date=2014-10-01 }}</ref>。2017年12月16日，孙红雷在微博宣布，其妻正式生下一名女儿<ref>{{cite web |url=http://ent.qq.com/a/20171216/006229.htm?t=1513408924543|title= 47岁孙红雷正式升级当爸爸 发文报喜：宝贝女儿诞生|accessdate=2017-12-16 |publisher=腾讯娱乐 }}</ref>。
