'''孙志远'''（{{bd|1911年||1966年|10月11日|catIdx=S孫}}）原名'''秉哲'''，曾用名'''国柱'''、'''建安'''，[[河北省|河北省]][[定县|定县]]大王梅村人，[[中国共产党|中国共产党]]官员。<ref name=hb>毛泽东与河北，石家庄：河北人民出版社，2006年，第207页</ref>

==生平==
孙志远1928年考入[[北平师范大学|北平师范大学]]理科预科<ref name=hb/>，1929年参加[[中国共产主义青年团|中国共产主义青年团]]，1930年转入[[中国共产党|中国共产党]]。1937年以前，一直在[[白区|白区]]（即国民党统治区）工作。此后历任[[冀中军区|冀中军区]]政治部主任，[[晋绥军区|晋绥军区]]政治部副主任，[[晋绥野战军|晋绥野战军]]政治部主任，纵队政治委员，兵团政治部主任。<ref name=sun>孙志远同志逝世，人民日报1966年10月12日</ref>

[[中华人民共和国|中华人民共和国]]成立后，历任[[西南军政委员会|西南军政委员会]]秘书长，[[中央人民政府政务院|中央人民政府政务院]]副秘书长，[[国务院第三办公室|国务院第三办公室]]副主任，[[国家经济委员会|国家经济委员会]]副主任，[[第三机械工业部|第三机械工业部]]部长等职。在担任[[西南军政委员会|西南军政委员会]]秘书长期间，曾参加西藏和平解放谈判，并参与签订《[[十七条协议|十七条协议]]》。1958年，在[[中国共产党第八次全国代表大会|中国共产党第八次全国代表大会]]第二次会议上，当选为[[中国共产党中央委员会候补委员|中国共产党中央委员会候补委员]]<ref>{{Cite book|title=中华人民共和国政府机构五十年|last=国家行政学院编著|first=|publisher=北京：党建读物出版社|year=2000年6月|isbn=7-80098-406-0|location=|pages=224}}</ref>。他还担任[[中华人民共和国国防委员会|中华人民共和国国防委员会]]委员、[[中国人民政治协商会议全国委员会|中国人民政治协商会议全国委员会]]委员等职务。<ref name=sun/>

1966年10月11日，孙志远因病在北京逝世，享年55岁。<ref name=sun/>

==家庭==
*妻：赵磊。曾任[[北京邮电学院|北京邮电学院]]党委副书记。<ref name=jiaren>{{Cite web |url = http://www.zgdsw.org.cn/n/2014/1218/c244523-26232377.html |title = 李井泉和他的家人 |author =王晓建、双石  |date =  |publisher = 党史博览2014(11) |language =  |accessdate =  |quote =  }}</ref>
*养子：孙巨，原名李巨一。1945年生。是[[李井泉|李井泉]]、[[肖里|肖里]]夫妇第三个儿子。名字“巨一”出自[[毛泽东|毛泽东]]在[[中共七大|中共七大]]总结报告中提到的“巨大统一”。孙志远、赵磊夫妇是李井泉、肖里的婚姻介绍人。赵磊在战争年代接受节育手术，失去生育能力。李井泉将李巨一送给孙志远夫妇，改姓名为孙巨。孙巨自[[北京四中|北京四中]]毕业后考进[[清华大学|清华大学]]无线电系，1980年留学[[美国|美国]]，获[[南加利福尼亚大学|南加利福尼亚大学]]电子工程博士学位。回中国后进入[[康华公司|康华公司]]及[[中国科协|中国科协]]，担任中国国际经济技术开发中心总经理。1993年进入[[招商局集团|招商局集团]]直到2006年退休，历任招商局集团北京公司副总经理，招商局集团金融中心、企业部、技术部总经理。孙巨之妻是[[中国人民解放军空军总医院|中国人民解放军空军总医院]]眼科医生赖迎利（开国上将[[赖传珠|赖传珠]]之女，孙志远的外甥女）。<ref name=jiaren/>
