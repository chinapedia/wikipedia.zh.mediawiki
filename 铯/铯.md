'''铯'''（'''Caesium'''或'''Cesium'''，舊譯作'''鏭'''）是一种[[化学元素|化学元素]]，[[化学符号|化学符号]]为'''Cs'''，[[原子序数|原子序]]为55。铯属于[[碱金属|碱金属]]，带银金色。

铯色白质软，[[熔點|熔點]]低，28.44 ℃时即会熔化。它是在室温或者接近室温的条件下为液体的五种[[金属|金属]]元素之一{{NoteTag|其它四种金属为[[铷|铷]]（熔点为39 ℃）、[[钫|钫]]（估计熔点为27 ℃）、[[镓|镓]]（熔点为30 ℃）以及[[汞|汞]]（熔点为-39 ℃）。[[溴|溴]]在室温下也是液体，其熔点为-7.2 ℃，但是它是[[卤素|卤素]]而不是金属。}}。铯的[[物理性质|物理性质]]和[[化学性质|化学性质]]与同为碱金属的[[铷|铷]]和[[钾|钾]]相似。该金属极度活泼，并且能够[[自燃|自燃]]。它是具有[[稳定同位素|稳定同位素]]的元素中[[电负性|电负性]]最低的，其稳定同位素为铯-133。铯通常是从[[铯榴石|铯榴石]]中提取出来的，而其[[放射性同位素|放射性同位素]]，尤其是铯-137，是更重元素的[[衰变产物|衰变产物]]，可从[[核反应堆|核反应堆]]产生的[[放射性废料|废料]]中提取。

1860年，两位德国化学家[[罗伯特·威廉·本生|罗伯特·威廉·本生]]和[[古斯塔夫·基尔霍夫|古斯塔夫·基尔霍夫]]通过刚刚研究出来的[[焰色反应|焰色反应]]发现铯，並以拉丁文「{{lang|la|caesius}}」（意為天藍色）作为新元素的名称。铯最早的小规模应用是作为[[真空管|真空管]]以及[[光电池|光电池]]的吸收剂。1967年，[[国际单位制|国际单位制]]中的[[秒|秒]]开始以[[铯-133|铯-133]]的[[发射光谱|发射光谱]]中一个特殊的频率作为定义。自此之后，铯广泛地用于[[原子钟|原子钟]]。二十世纪九十年代以来，用于[[钻井液|钻井液]]的[[甲酸盐|甲酸铯]]成为铯元素的最大应用。该元素在[[化学工业|化学工业]]以及[[电子产业|电子产业]]等有重要用途。其放射性同位素铯-137的[[半衰期|半衰期]]大约为30年，可以用于[[医学|医学]]、工业[[测量仪器|测量仪器]]以及[[水文学|水文学]]。虽然铯仅有轻微的[[毒性|毒性]]，但其金属却是一种有害的材料；若其放射性同位素释放到了环境中，将对健康造成较大的威胁。

== 性质 ==

=== 物理性质 ===
[[File:CsCrystals.JPG|left]]中的高纯度铯-133|alt= 玻璃安瓿中的看起来像是松树枝的Y形黄色晶体]]
铯是一种非常柔软、[[延展性|延展性]]很强的的白色金属，其[[莫氏硬度|莫氏硬度]]在所有的元素中最低，仅0.2。在存在[[痕量|痕量]]的[[氧气|氧气]]的情况下会变暗<ref name=USGS>{{cite web|url = http://pubs.usgs.gov/of/2004/1432/2004-1432.pdf|format = PDF|publisher = United States Geological Survey|accessdate = 2009-12-27|title = Mineral Commodity Profile: Cesium|first1 = William C.|last1 = Butterman|first2 = William E.|last2 = Brooks|first3 = Robert G.|last3 = Reese, Jr.|year = 2004|deadurl = yes|archiveurl = https://web.archive.org/web/20070207015229/http://pubs.usgs.gov/of/2004/1432/2004-1432.pdf|archivedate = 2007-02-07}}</ref><ref>{{cite book|title = Exploring Chemical Elements and their Compounds|author = Heiserman, David L.|publisher = McGraw-Hill|year = 1992|isbn = 0-8306-3015-5|pages = 201–203}}</ref><ref>{{cite book |title=The Chemistry of the Liquid Alkali Metals |last=Addison |first=C. C. |year=1984 |publisher=Wiley|isbn= 0471905089|accessdate=2012-09-28 |url=http://www.cs.rochester.edu/users/faculty/nelson/cesium/cesium_color.html}}</ref>。铯的[[熔点|熔点]]为28.4℃，为少数几种在接近[[室温|室温]]的条件下为液态的金属元素。[[汞|汞]]是唯一的熔点低于铯的金属元素{{NoteTag|放射性元素[[钫|钫]]可能具有更低的熔点，然而，由于其放射性，难以分离的足够的钫以进行测试<ref>{{cite web|url=http://periodic.lanl.gov/87.shtml |title=Francium |publisher=Periodic.lanl.gov |accessdate=2010-02-23}}</ref>}}<ref name="autogenerated1">{{cite web|url=http://pubs.acs.org/cen/80th/print/cesium.html |title=C&EN: It's Elemental: The Periodic Table – Cesium |publisher=American Chemical Society|accessdate=2010-02-25|author=Kaner, Richard|year = 2003}}</ref>。此外，该金属具有较低的沸点，仅有641℃，为除了汞以外沸点最低的金属元素。铯的化合物燃烧时具有蓝色<ref name=CRC74>{{cite book|url=http://books.google.com/?id=QdU-lRMjOsgC&pg=PA13|page=13|first=Charles T.|last=Lynch|publisher=CRC Press|year=1974|title=CRC Handbook of Materials Science|isbn=978-0-8493-2321-8}}</ref><ref name="flametests"/>或者紫色<ref name="flametests">{{cite web |url=http://www.chemguide.co.uk/inorganic/group1/flametests.html |title=Flame Tests |author=Clark, Jim |year=2005 |work=chemguide |accessdate=2012-01-29}}</ref>。

[[File:Cesium_ampoule_in_acrylic_cube.jpg|thumb]]

铯可以和其它[[碱金属|碱金属]]以及[[金|金]]形成合金，并且与汞形成[[汞齐|汞齐]]。在温度低于650℃的条件下，铯无法和[[钴|钴]]、[[铁|铁]]、[[钼|钼]]、[[镍|镍]]、[[铂|铂]]、[[钽|钽]]或者[[钨|钨]]形成合金。铯可以和[[锑|锑]]、[[镓|镓]]、[[铟|铟]]以及[[钍|钍]]形成具有[[感光性|感光性]]的[[金属互化物|金属互化物]]<ref name="USGS" />。铯可以和除锂之外的碱金属混合形成合金，并且摩尔比例为41%铯，47%钾以及12%钠的合金的熔点为-78℃，在所有已知的金属合金中熔点最低<ref name="autogenerated1" /><ref>{{cite web|url=http://symp15.nist.gov/pdf/p564.pdf|format=PDF|title=Density of melts of alkali metals and their Na-K-Cs and Na-K-Rb ternary systems|author=Taova, T. M. ''et al.''|work=Fifteenth symposium on thermophysical properties, Boulder, Colorado, USA|date=2003-06-22|accessdate=2010-09-26|deadurl=yes|archiveurl=https://web.archive.org/web/20061009133313/http://symp15.nist.gov/pdf/p564.pdf|archivedate=2006-10-09}}</ref>。

已经研究过若干种汞齐：{{chem|CsHg|2}}为黑色并具有紫色金属[[光泽|光泽]]，而CsHg具有金色，同样具有金属光泽<ref>{{cite journal|doi=10.1016/S0079-6786(97)81004-7|journal = Progress in Solid State Chemistry|volume = 25|year = 1997|pages = 73–123|title = Alkali metal amalgams, a group of unusual alloys|first = H. J.|last = Deiseroth|issue = 1–2}}</ref>。

=== 化学性质 ===
[[File:Cesium_water.theora.ogv|left]]
金属铯具有高度的活性，并且非常容易[[自燃|自燃]]。除了在空气中能够自发燃烧以外，铯在很低温度下就能够与水发生爆炸性地反应，比碱金属中的其他元素更剧烈<ref name="USGS" />。铯可以在温度低达-116℃的条件下与冰发生反应<ref name="autogenerated1" />。由于其高度活性，该金属被分类为[[危险性物质|危险性物质]]。铯通常在诸如[[矿物油|矿物油]]等的干燥的饱和[[烃|烃]]中储存和运输。类似的，必须在[[無活性氣體|惰性气体]]的保护下处理铯，例如[[氩|氩]]。然而，铯-水的爆炸威力通常比同样量的钠-水的威力小，这是由于铯在接触到水的时候立即爆炸，使得聚集起[[氢气|氢气]]的时间很少<ref>Gray, Theodore (2012) ''The Elements'', Black Dog & Leventhal Publishers, p. 131, ISBN 978-1-57912-895-1.</ref>。铯可以储存于真空密封的[[硅硼酸盐玻璃|硅硼酸盐玻璃]]制造的[[安瓿|安瓿]]中。若铯的量超过100[[克|克]]，需要将铯放置在密封的不锈钢容器中运输。

铯的化学性质与其他碱金属类似，但是更接近于元素中期表中其上面的元素[[铷|铷]]的化学性质<ref name="greenwood" />。由于铯是一种碱金属，可以预期其通常的化合价为+1{{NoteTag|在包含有Cs<sup>−</sup>阴离子的[[碱化物|铯化物]]中情况不同，此时铯具有-1的化合价<ref name=caeside>{{cite journal|journal = [[Angewandte_Chemie|Angewandte Chemie International Edition]]|year = 1979|first = J. L.|last = Dye|title = Compounds of Alkali Metal Anions|volume = 18|issue = 8|pages = 587–598|doi = 10.1002/anie.197905871}}</ref>。此外，苗茂生2013年的计算结果表明，在压强极大（超过20 GPa）的条件下，内部的5p电子可以形成化学键，这时铯能够表现出p区元素的性质。该发现表明在该条件下可能存在铯化合价为+2至+6的铯氟化物<ref>{{cite web|last=Moskowitz|first=Clara|title=A Basic Rule of Chemistry Can Be Broken, Calculations Show|url=http://www.scientificamerican.com/article.cfm?id=chemical-bonds-inner-shell-electrons|publisher=Scientific American|accessdate=2013-11-22}}</ref>}}。由于铯具有较高的原子量，并且比其他非放射性碱金属电正性更强，有一些较小的区别<ref name="HollemanAF">{{cite book|publisher = Walter de Gruyter|year = 1985|edition = 91–100|pages = 953–955|isbn = 3-11-007511-3|title = Lehrbuch der Anorganischen Chemie|first1 = Arnold F.|last1 = Holleman|last2 = Wiberg|first2 = Egon |last3 =Wiberg|first3 = Nils|chapter = Vergleichende Übersicht über die Gruppe der Alkalimetalle| language = de}}</ref>。铯是电正性最强的稳定化学元素{{NoteTag|[[钫|钫]]可能具有更强的电正性，但是由于其高放射性，从未进行过实验测量。钫的第一[[电离能|电离能]]指出其[[相对论效应|相对论效应]]可能使其活性降低，并且使其电负性高于[[周期表趋势|周期表趋势]]所预期的值<ref>{{cite journal|last1 = Andreev|first1 = S. V.|last2 = Letokhov|first2 =V. S. |last3 =Mishin|first3 =V. I.|title = Laser resonance photoionization spectroscopy of Rydberg levels in Fr|journal = [[Physical_Review_Letters|Physical Review Letters]]|year = 1987|volume = 59|pages = 1274–76|doi = 10.1103/PhysRevLett.59.1274|pmid=10035190|bibcode=1987PhRvL..59.1274A|issue = 12}}</ref>}}。铯离子也比较大，并且与其他更轻的[[碱金属|碱金属]]相比[[软硬酸碱理论|较软]]。

=== 化合物 ===
[[File:CsCl_polyhedra.png|thumb]]

铯化合物多数都含有{{chem|Cs||+}}[[离子|离子]]，它能与很多种离子形成离子键。一个值得注意的例外是[[碱化物|铯化物]]中含有{{chem|Cs||−}}。<ref>{{cite journal|journal = [[Angewandte_Chemie|Angewandte Chemie International Edition]]|year = 1979|first = J. L.|last = Dye|title = Compounds of Alkali Metal Anions|volume = 18|issue = 8|pages = 587–598|doi = 10.1002/anie.197905871}}</ref> 其他例外还包括一些[[低氧化物|低氧化物]]（参见下面的氧化物章节）。

回到普通的铯化合物，Cs<sup>+</sup>的盐通常是无色的，除非阴离子有颜色。 许多简单的盐具有[[潮解|潮解]]性，但比更轻的其他碱金属弱。铯的[[乙酸盐|乙酸盐]]、[[碳酸盐|碳酸盐]]、[[卤化物|卤化物]]、[[氧化物|氧化物]]、[[硝酸盐|硝酸盐]]和[[硫酸盐|硫酸盐]]可溶于水。 [[复盐|复盐]]通常溶解度较小，硫酸铝铯溶解度较小的性质常用来从矿石中提纯铯。与锑（例如{{chem|CsSbCl|4}}）、[[铋|铋]]、[[镉|镉]]、[[铜|铜]]、[[铁|铁]]和[[铅|铅]]形成的复盐通常溶解度很小<ref name=USGS/>。

[[氢氧化铯|氢氧化铯]]（CsOH）是一种具有强烈吸水性的强[[碱|碱]]<ref name="greenwood">{{cite book|last1 = Greenwood|first1 = N.N.|last2 = Earnshaw|first2 = A.|title = Chemistry of the Elements|publisher = Pergamon Press|place = Oxford, UK|year = 1984|isbn = 0-08-022057-6}}</ref>。它能迅速腐蚀[[半导体材料|半导体材料]]（例如[[硅|硅]]）的表面。<ref>{{cite book|url = http://books.google.com/?id=F-8SltAKSF8C&pg=PA90&lpg=PA90&dq=caesium+hydroxide+etch+quartz&q=cesium|title = Etching in microsystem technology|author = Köhler, Michael J.|page = 90|publisher = Wiley-VCH|isbn = 3-527-29561-5|year = 1999|format=}}</ref> 过去化学家曾认为CsOH是“最强的碱”，因为大阳离子Cs<sup>+</sup>与OH<sup>-</sup>的相互作用很微弱<ref name=CRC74/>。氢氧化铯确实是碱性最强的[[酸碱电离理论|阿伦尼乌斯碱]]，但是许多无法存在于水溶液中的化合物的碱性远比CsOH强，例如[[正丁基锂|正丁基锂]]和[[氨基钠|氨基钠]]<ref name="greenwood" />。

铯与金的化学计量1:1的混合物加热后可以反应形成黄色的[[金化铯|金化铯]]。这里的金阴离子表现为[[拟卤素|拟卤素]]。该化合物能够与水发生剧烈反应，生成[[氢氧化铯|氢氧化铯]]、金属金以及氢气。在液氨中，金化铯可以与铯特定的离子交换树脂反应以产生四甲基金。类似的铂化物，红色的铂化铯{{chem|Cs|2|Pt}}包含表现得像拟硫族的铂阴离子<ref>{{cite journal|title=Effects of relativistic motion of electrons on the chemistry of gold and platinum|journal=Solid State Sciences|date=2005-11-30|volume=7|issue=12|pages=1464–1474|doi=10.1016/j.solidstatesciences.2005.06.015|last=Jansen|first=Martin}}</ref>。

==== 复合物 ====
与所有的金属阳离子类似，Cs<sup>+</sup>与[[路易斯碱|路易斯碱]]在溶液中形成复合物。由于该粒子较大，Cs<sup>+</sup>通常的[[配位数|配位数]]大于6，而较轻的碱金属阳离子的典型配位数为6。这个趋势在CsCl中很明显，其配位数为8。铯离子较高的配位数以及[[软硬酸碱理论|软度]]（形成[[共价键|共价键]]的趋势）是将其与其它阳离子分离的基础，在实践中用于对核废料的整治，也就是将<sup>137</sup>Cs<sup>+</sup>与大量的非放射性的K<sup>+</sup>分离<ref>{{cite journal |last1=Moyer |first1=Bruce A. |last2=Birdwell |first2=Joseph F. |last3=Bonnesen |first3=Peter V. |last4=Delmau |first4=Laetitia H. |title=Macrocyclic Chemistry |pages=383–405 |year=2005 |doi=10.1007/1-4020-3687-6_24 |chapter=Use of Macrocycles in Nuclear-Waste Cleanup: A Realworld Application of a Calixcrown in Cesium Separation Technology |isbn=1-4020-3364-8}}</ref>。

====卤化物====
[[氟化铯|氟化铯]]（CsF）是一种易[[潮解|潮解]]的白色固体，经常用于[[有机氟化学|有机氟化学]]中作为[[氟|氟]]阴离子的来源<ref>{{cite journal|author=F. W. Evans, M. H. Litt, A. M. Weidler-Kubanek, F. P. Avonda|title=Reactions Catalyzed by Potassium Fluoride. 111. The Knoevenagel Reaction|year=1968|journal=Journal of Organic Chemistry|volume=33|pages=1837–1839|doi=10.1021/jo01269a028|issue=5}}</ref>。氟化铯具有[[石盐|石盐]]结构，也就是Cs<sup>+</sup>和F<sup>−</sup>以和[[氯化钠|氯化钠]]中的Na<sup>+</sup>和Cl<sup>−</sup>相同的方式堆积成[[立方紧密堆积|立方紧密堆积]]阵列<ref name=greenwood/>。值得注意，铯和氟在所有已知的元素中分别具有最高和最低的[[电负性|电负性]]。

[[氯化铯|氯化铯]]以简单[[立方晶系|立方晶系]]结晶。该结构包含以双原子为基础的立方晶格[[原胞|原胞]]，每个[[配位数|配位数]]都是8，也被称为""氯化铯结构""<ref name="HollemanAF" />。氯原子位于立方的顶点上，而铯原子位于立方中央。该结构与[[溴化铯|溴化铯]]、[[碘化铯|碘化铯]]以及许多其它不包含铯离子的化合物相同。相反，大多数其它的碱金属卤化物采用[[氯化钠|氯化钠]]结构<ref name="HollemanAF" />。采用氯化铯结构的原因是Cs<sup>+</sup>的[[离子半径|离子半径]]为174 pm，而{{chem|Cl||−}}的半径为181 pm<ref>{{cite book|last=Wells|first = A.F.| year=1984| title=Structural Inorganic Chemistry| edition=5|publisher=Oxford Science Publications|isbn=0-19-855370-6}}</ref>。

===同位素===
{{Main|铯的同位素}}
铯一共有39种已知的[[同位素|同位素]]，其[[质量数|质量数]]（原子核中的[[核子|核子]]数）在112至151之间。其中一些同位素在年老的恒星中由较轻的元素通过捕获慢中子（[[S-过程|S-过程]]）合成<ref>{{cite journal|doi=10.1146/annurev.astro.37.1.239|author=Busso, M.; Gallino, R.; Wasserburg, G. J.|title=Nucleosynthesis in Asymptotic Giant Branch Stars: Relevance for Galactic Enrichment and Solar System Formation|journal=Annula Review of Astronomy and Astrophysics|volume=37|year=1999|pages=239–309|url=http://authors.library.caltech.edu/1194/1/BUSaraa99.pdf| format=PDF|accessdate=2010-02-20|bibcode=1999ARA&A..37..239B}}</ref>，也可以在[[超新星|超新星]]爆发的过程[[R-过程|R-过程]]中合成<ref>{{cite book|first=David|last=Arnett|year=1996|title=Supernovae and Nucleosynthesis: An Investigation of the History of Matter, from the Big Bang to the Present|publisher=Princeton University Press|page=527|isbn=0-691-01147-8}}</ref>。然而，铯仅有唯一的[[稳定同位素|稳定同位素]]，也就是具有78个[[中子|中子]]的铯-133（<sup>133</sup>Cs）。尽管其[[原子核自旋|原子核自旋]]量子数较大（7/2+），可以在其共振频率11.7 [[MHz|MHz]]处对该同位素进行[[核磁共振|核磁共振]]的研究<ref name=NMR>{{cite journal|doi=10.1016/0277-5387(96)00018-6|title=Complexation of caesium and rubidium cations with crown ethers in N,N-dimethylformamide|year=1996|last1=Goff|first1=C|journal=Polyhedron|volume=15|page=3897|last2=Matchette|first2=Michael A.|last3=Shabestary|first3=Nahid|last4=Khazaeli|first4=Sadegh|issue=21}}</ref>。
[[File:Cs-137-decay.svg|thumb]]铯-137以94.6%的概率通过β衰变发射出512keV的电子衰变为钡-137m（I=11/2-，t=2.55分），随后以85.1%的概率通过662keV的伽玛衰变转变为钡-137。此外，铯-137可以以0.4%的概率通过beta衰变直接转变为钡-137。
放射性同位素[[铯-135|<sup>135</sup>Cs]]具有非常长的半衰期，大约230万年，而[[铯-137|<sup>137</sup>Cs]] 和[[铯-134|<sup>134</sup>Cs]]的半衰期分别为30年和2年。<sup>137</sup>Cs通过[[β衰变|β衰变]]转变为分解为寿命很短的[[钡-137m|<sup>137m</sup>Ba]]，随后转变为非放射性的钡，而<sup>134</sup>Cs直接转变为<sup>134</sup>Ba。具有质量数为129、131、132以及136的同位素的半衰期在一天和两周之间，而大多数其它的同位素的半衰期仅数秒或者少于一秒。至少有21种亚稳[[核同质异能素|核同质异能素]]，除了<sup>134m</sup>Cs（半衰期少于3小时），都很不稳定，并且半衰期仅数分钟甚至更短<ref>{{cite journal|doi = 10.1016/0022-1902(55)80027-9|title = The half-life of Cs137|year = 1955|last1 = Brown|first1 = F.|last2 = Hall|first2 = G.R.|last3 = Walter|first3 = A.J.|journal = Journal of Inorganic and Nuclear Chemistry|volume = 1|pages = 241–247|issue = 4–5}}</ref><ref name=nuclidetable>{{cite web|url=http://www.nndc.bnl.gov/chart/|title=Interactive Chart of Nuclides|publisher=Brookhaven National Laboratory|author=Sonzogni, Alejandro|location=National Nuclear Data Center|accessdate=2008-06-06}}</ref>。

同位素<sup>135</sup>Cs是[[铀|铀]]的[[长寿命裂变产物|长寿命裂变产物]]之一，可以在[[核反应堆|核反应堆]]中产生<ref>{{cite conference| conference = Seventh Information Exchange Meeting on Actinide and Fission Product Partitioning and Transmutation|date = 14–16 October 2002|place = Jeju, Korea|first1 = Shigeo|last1 = Ohki|first2 = Naoyuki|last2= Takaki|title =Transmutation of Cesium-135 with Fast Reactors|url = http://www.oecd-nea.org/pt/docs/iem/jeju02/session6/SessionVI-08.pdf|format = PDF|accessdate=2010-09-26}}</ref>。然而，其裂变产物产量在大多数反应堆中都会比较少，这是由于其前体，[[氙-135|<sup>135</sup>Xe]]是极强的[[中子毒物|中子毒物]]，经常在衰变为<sup>135</sup>Cs之前嬗变为稳定的[[氙-136|<sup>136</sup>Xe]]<ref>{{cite report|url=http://canteach.candu.org/library/20040720.pdf|title=CANDU Fundamentals|format=PDF|publisher=CANDU Owners Group Inc.|chapter=20 Xenon: A Fission Product Poison|accessdate=2010-09-15|deadurl=yes|archiveurl=https://web.archive.org/web/20110723231319/http://canteach.candu.org/library/20040720.pdf|archivedate=2011-07-23}}</ref><ref>{{cite journal|journal = Journal of Environmental Radioactivity|title = Preliminary evaluation of <sup>135</sup>Cs/<sup>137</sup>Cs as a forensic tool for identifying source of radioactive contamination|first1 = V. F.|last1= Taylor|first2 = R. D.|last2 = Evans|first3 = R. J.|last3= Cornett|doi = 10.1016/j.jenvrad.2007.07.006 |volume = 99 |issue = 1|year = 2008|pages = 109–118|pmid = 17869392}}</ref>。

由于铯137能够β衰变为<sup>137m</sup>Ba，它是一种很强的[[伽玛射线|伽玛射线]]发射体<ref>{{cite web|url=http://www.epa.gov/rpdweb00/radionuclides/cesium.html|title=Cesium | Radiation Protection|publisher=U.S. Environmental Protection Agency|date=2006-06-28|accessdate=2010-02-15|deadurl=yes|archiveurl=https://web.archive.org/web/20110315034747/http://www.epa.gov/rpdweb00/radionuclides/cesium.html|archivedate=2011-03-15}}</ref>。其半衰期使得它和[[锶-90|<sup>90</sup>Sr]]一同成为[[中等寿命裂变产物|中等寿命裂变产物]]。由于这两种同位素，[[乏核燃料|乏核燃料]]在经过数年的冷却后仍在数百年内具有放射性<ref>{{cite report|url=http://www.ieer.org/reports/transm/hisham.html |title=IEER Report: Transmutation – Nuclear Alchemy Gamble |publisher=Institute for Energy and Environmental Research |date=2000-05-24 |accessdate=2010-02-15|first=Hisham|last=Zerriffi}}</ref>。例如，<sup>137</sup>Cs和<sup>90</sup>Sr一起成为了[[切尔诺贝利核事故|切尔诺贝利核事故]]之后其周围区域内最大的放射源<ref>{{cite report |url=http://www.iaea.org/Publications/Booklets/Chernobyl/chernobyl.pdf |title=Chernobyl's Legacy: Health, Environmental and Socia-Economic Impacts and Recommendations to the Governments of Belarus, Russian Federation and Ukraine |publisher=International Atomic Energy Agency |format=PDF |accessdate=2010-02-18 |deadurl=yes |archiveurl=https://web.archive.org/web/20100215212227/http://www.iaea.org/Publications/Booklets/Chernobyl/chernobyl.pdf |archivedate=2010-02-15 }}</ref> 。由于其较低的中子捕获率，无法通过[[中子捕获|中子捕获]]处理铯-137，所以必须让它进行衰变<ref>{{cite journal|doi=10.3327/jnst.30.911| title = Transmutation of Cesium-137 Using Proton Accelerator|first1 = Takeshi|last1= Kase|first2 = Kenji|last2= Konashi|first3 = Hiroshi|last3 = Takahashi|first4 = Yasuo|last4= Hirao|volume = 30|issue = 9|year = 1993|pages = 911–918|journal = Journal of Nuclear Science and Technology}}</ref>。

几乎所有的核裂变产生的铯都由开始富含中子的裂变产物的[[β衰变|β衰变]]产生，例如各种[[碘的同位素|碘的同位素]]以及[[氙的同位素|氙的同位素]]<ref>{{cite book|isbn = 978-1-56032-088-3|publisher = Taylor & Francis|year = 1992|first = Ronald Allen|last = Knief|url = http://books.google.com/?id=EpuaUEQaeoUC&pg=PA43|page= 42|chapter = Fission Fragments|title = Nuclear engineering: theory and technology of commercial nuclear power}}</ref> 。由于碘和氙容易挥发，并且可以通过核燃料或者空气扩散，辐射性的铯经常在远离裂变地点的地方产生<ref>{{cite journal|title = Release of xenon-137 and iodine-137 from UO2 pellet by pulse neutron irradiation at NSRR|last = Ishiwatari|first = N.|last2 = Nagai|first2 = H.|pages = 843–850|volume = 23|issue = 11|journal = Nippon Genshiryoku Gakkaishi|osti = 5714707}}</ref>。自从1945年开始进行[[核试验|核试验]]，铯-137被释放入大气层并作为[[放射性尘埃|放射性尘埃]]落回地球表面<ref name="USGS" />。

===存象===
[[File:Pollucite-RoyalOntarioMuseum-Jan18-09.jpg|thumb]]

铯是一种相对比较稀有的元素，估计在[[地壳|地壳]]中的[[化学元素丰度|化学元素丰度]]约为3[[百萬分率|ppm]]（百万分之三）<ref>{{cite journal|last1=Turekian|first1=K.K.|last2=Wedepohl|first2=K. H.|year=1961|title=Distribution of the elements in some major units of the Earth's crust|journal=Geological Society of America Bulletin|volume=72|issue=2|pages=175–192|doi=10.1130/0016-7606(1961)72[175:DOTEIS]2.0.CO;2|issn=0016-7606|bibcode = 1961GSAB...72..175T }}</ref>。铯在所有元素中丰度排第45名，而在所有金属中丰度排第36名。尽管如此，铯的丰度要高于锑、镉、[[锡|锡]]以及[[钨|钨]]，并且比[[银|银]]和[[汞|汞]]的丰度高了两个数量级，然而其丰度仅为[[铷|铷]]的3.3%<ref name="USGS" />

由于其离子半径很大，铯是“[[不相容成分|不相容成分]]”之一<ref>{{cite web|url=http://www.asi.org/adb/02/13/02/cesium-occurrence-uses.html|title=Cesium as a Raw Material: Occurrence and Uses|first=Simon|last=Rowland|publisher=Artemis Society International|date=1998-07-04|accessdate=2010-02-15}}</ref>。在[[分级结晶|分级结晶]]的过程中，铯在液相中浓缩，并且最后结晶。因此，铯的大矿床为在富集过程中形成的伟晶岩矿石。由于铯不像[[铷|铷]]那样可以将[[钾|钾]]置换出来，碱蒸发盐矿物天然氯化钾（KCl）和[[光卤石|光卤石]]（{{chem|KMgCl|3|·6H|2|O}}）中仅含0.002%的铯。所以铯仅能在若干种矿物质中找到。铯含量较丰富的矿物质包括[[绿柱石|绿柱石]]（{{chem|Be|3|Al|2|(SiO|3|)|6}}）和[[氟硼钾石|氟硼钾石]]（{{chem|(K,Cs)BF|4}}）。在密切相关的矿物质[[草莓红绿柱石|草莓红绿柱石]]（Cs(Be<sub>2</sub>Li)Al<sub>2</sub>Si<sub>6</sub>O<sub>18</sub>）中{{chem|Cs|2|O}}的质量百分比可达15%，而在稀有矿物[[硼铯铝铍石|硼铯铝铍石]]（{{chem|(Cs,K)Al|4|Be|4|(B,Be)|12|O|28}}）中可达8.4%，在分布更广的[[硼锂铍矿|硼锂铍矿]]中含量较少<ref name="USGS" />。[[铯榴石|铯榴石]]{{chem|Cs(AlSi|2|O|6|)}}是唯一的有经济意义的铯矿石,在世界上某些地方的伟晶岩矿石中可以找到，并且它和有更重要的商业应用的[[锂|锂]]矿石
[[锂云母|锂云母]]和[[透锂长石|透锂长石]]共生。在伟晶岩矿石中，晶粒比较大并且容易与矿石其它部分分离，是采矿的高品质矿石<ref name="Cerny">{{cite journal|title=The Tanco Pegmatite at Bernic Lake, Manitoba: X. Pollucite|first1=Petr|last1=Černý|authorlink1=Petr Černý|first2=F. M.|last2=Simpson|journal=Canadian Mineralogist|volume=16|pages=325–333|year=1978|url=http://rruff.geo.arizona.edu/doclib/cm/vol38/CM38_877.pdf|format=PDF|accessdate=2010-09-26}}</ref>。

世界上最重要的也是储量最丰富的铯产地是位于[[加拿大|加拿大]][[马尼托巴|马尼托巴]]的[[伯尼克湖|伯尼克湖]]的[[堂科矿|堂科矿]]。估计其矿床中有35万吨铯榴石矿石，占世界储量的2/3<ref name="Cerny" /><ref name="USGS-Cs2" /> 尽管铯榴石中的铯的化学计量含量为42.6%，来自该矿床的纯净铯榴石样品中铯的含量仅约34%，平均含量24%<ref name="USGS-Cs2">{{cite web|title=Cesium|last=Polyak|first=Désirée E.|url=http://minerals.usgs.gov/minerals/pubs/commodity/cesium/mcs-2009-cesiu.pdf|format=PDF|publisher=U.S. Geological Survey|accessdate = 2009-10-17}}</ref>。商业用铯榴石包含19%的铯<ref>{{cite book|last=Norton|first=J. J.|year=1973|chapter=Lithium, cesium, and rubidium—The rare alkali metals|editor=Brobst, D. A., and Pratt, W. P.|title=United States mineral resources|publisher=U.S. Geological Survey Professional|volume=Paper 820|pages=365–378|url=http://pubs.er.usgs.gov/usgspubs/pp/pp820|accessdate=2010-09-26}}</ref>。[[津巴布韦|津巴布韦]]的[[比基塔|比基塔]]伟晶岩矿床生产[[透锂长石|透锂长石]]，但是也有大量的铯榴石。[[纳米比亚|纳米比亚]]的[[埃龙戈区|卡里比布沙漠]]中也在开采大量的铯榴石<ref name="USGS-Cs2" />。目前世界的年开采量为5-10吨，目前的储量可供开采数千年<ref name="USGS" />。

== 生产 ==

开采铯榴石是一种选别工艺，和其他大多数金属采矿相比仅以小规模进行生产。首先将矿石击碎，并且进行手工筛选，但是经常不做富集，然后将矿石粉碎。随后主要通过三种方式将铯从铯榴石中提取出来：酸消解、碱分解、以及直接还原<ref name="USGS" /><ref name=Burt>{{cite book|last = Burt|first = R. O.|year = 1993|chapter = Caesium and cesium compounds|title = Kirk-Othmer encyclopedia of chemical technology|edition = 4th|place = New York|publisher = John Wiley & Sons, Inc.|volume = 5|pages = 749–764|isbn = 978-0-471-48494-3}}</ref>。

在酸消解中，硅酸盐铯榴石的岩石溶解在诸如[[盐酸|盐酸]]（HCl）、[[硫酸|硫酸]]（{{chem|H|2|SO|4}}）、[[氢溴酸|氢溴酸]]（HBr）或者[[氢氟酸|氢氟酸]]（HF）中。采用盐酸可以产生可溶解氯化物的混合物，而无法溶解的铯的氯化物复盐以氯化铯锑（{{chem|Cs|4|SbCl|7}}）、氯碘化铯（{{chem|Cs|2ICl}}）或者氯化铯铈（{{chem|Cs|2|(CeCl|6|)}}）的形式沉淀出来。分离之后，将提纯的沉淀复盐分解，在水分蒸发后得到纯的氯化铯。采用硫酸的消解的方法将得到无法溶解的复盐铯[[矾|矾]]（{{chem|CsAl(SO|4|)|2|·12H|2|O}}）。铯矾与[[碳|碳]]一起烘烤后将转化为不溶于水的氧化铝，得到的产物通过沥滤得到[[硫酸铯|硫酸铯]]（{{chem|Cs|2|SO|4}}）溶液<ref name="USGS" />。

将铯榴石与[[碳酸钙|碳酸钙]]和[[氯化钙|氯化钙]]一同烘烤可以得到不易溶解的硅酸盐和可溶解的[[氯化铯|氯化铯]]。通过用水沥滤或者用[[氨水|氨水]]稀释，可以得到稀释的氯化铯溶液。该溶液可以通过蒸发以得到氯化铯或者转变为铯矾或者碳酸铯。还可以在真空罐中使用钾、钠或者钙对矿石直接还原以直接获得金属铯，但是该方法在经济上不可行。

大多数以盐的形式开采的铯直接转化为用于诸如[[油井|油井]]等应用的[[甲酸|甲酸铯]]（HCOO<sup>−</sup>Cs<sup>+</sup>）。为了向发展中的市场供货，[[卡博特公司|卡博特公司]]在1997年在位于[[加拿大|加拿大]]马尼托巴的伯尼克湖附近的堂科矿建立了生产厂，年产甲酸铯溶液1900m<sup>3</sup><ref>{{cite journal|last1 = Benton|first1 = William|last2 = Turner| first2 = Jim|year = 2000|title = Cesium formate fluid succeeds in North Sea HPHT field trials|journal = Drilling Contractor|issue = May/June|pages = 38–41|url = http://www.iadc.org/dcpi/dc-mayjun00/m-cabot.pdf|format=PDF|accessdate=2010-09-26}}</ref>。铯的主要小规模商用化合物为[[氯化铯|氯化铯]]以及[[硝酸铯|硝酸铯]]<ref name="CEC" />。

此外，金属铯可以通过从矿石中得到的提纯化合物获得。氯化铯以及其它卤化铯可以在700℃至800℃的高温下与钙或者钡一同加热以将铯还原，然后通过蒸馏得到金属铯。采用相同的方法，碳酸铯以及氢氧化铯等等可以通过使用[[镁|镁]]来还原<ref name="USGS" />。铯还可以通过电解熔融的[[氰化物|氰化铯]]来得到。极纯净并且无气体的铯可以通过以390℃加热[[叠氮化物|叠氮化铯]]使其分解来得到，而叠氮化铯可以通过硫酸铯溶液和叠氮化钡反映得到<ref name="Burt" />。在真空中，[[铬酸盐|重铬酸铯]]可以和[[锆|锆]]反应得到纯净的铯，而不产生其它气体产物<ref name=CEC>{{cite book|isbn = 978-3-11-011451-5|url = http://books.google.com/?id=Owuv-c9L_IMC&pg=PA198|page = 198|author = transl. and rev. by Eagleson, Mary |year = 1994|publisher = de Gruyter|location = Berlin|title = Concise encyclopedia chemistry}}</ref>：
:{{chem|Cs|2|Cr|2|O|7}} + 2 {{chem|Zr}} → 2 {{chem|Cs}} + 2 {{chem|ZrO|2}}+ {{chem|Cr|2|O|3}}。

2009年纯度为99.8%的铯（金属含量）大约为每克10美元，但是其化合物便宜很多<ref name="USGS-Cs2" >{{cite web|title=Cesium|last=Polyak|first=Désirée E.|url=http://minerals.usgs.gov/minerals/pubs/commodity/cesium/mcs-2009-cesiu.pdf|format=PDF|publisher=U.S. Geological Survey|accessdate = 2009-10-17}}</ref>。

==历史==
[[File:Bunsen-Kirchhoff.jpg|thumb]]

1860年，[[罗伯特·威廉·本生|罗伯特·威廉·本生]]和[[古斯塔夫·基尔霍夫|古斯塔夫·基尔霍夫]]在来自[[德国|德国]][[巴德迪尔克海姆|巴德迪尔克海姆]]的[[矿泉水|矿泉水]]中发现了铯。由于该元素在其发射光谱中明亮的蓝色线，他们选择了拉丁文中意为天蓝色的单词''caesius''为其命名{{NoteTag|本生引用了由[[Nigidius_Figulus|Nigidius Figulus]]创作的[[Aulus_Gellius|Aulus Gellius]] [[Aulus_Gellius|Noctes Atticae]] II：“Nostris autem veteribus caesia dicts est quae Graecis, ut Nigidus ait, de colore coeli quasi coelia.''}}<ref name="BuKi1861" /><ref name="Weeks">{{cite journal |title = The discovery of the elements. XIII. Some spectroscopic discoveries |pages = 1413–1434|last = Weeks|first = Mary Elvira |authorlink=Mary Elvira Weeks|doi=10.1021/ed009p1413|journal = [[Journal_of_Chemical_Education|Journal of Chemical Education]] |volume =9 |issue =8 |year = 1932|bibcode = 1932JChEd...9.1413W}}</ref> 。铯是第一种采用[[光谱学|光谱学]]的方法发现的元素，与本生和基尔霍夫发明[[光谱仪|光谱仪]]相隔仅仅一年<ref name="autogenerated1" />。

为了获得纯净的铯样品，需要蒸发44000升水以得到240千克浓缩的盐溶液。[[碱土金属|碱土金属]]或者以[[硫酸盐|硫酸盐]]的形式或者[[草酸盐|草酸盐]]的形式沉淀，而溶液中仅仅剩下[[碱金属|碱金属]]。在转化为硝酸盐并采用[[乙醇|乙醇]]提取后，可以获得不含[[钠|钠]]的混合物。该混合物中的[[锂|锂]]可以用[[碳酸铵|碳酸铵]]沉淀出来。[[钾|钾]]、[[铷|铷]]和铯可以与[[氯铂酸|氯铂酸]]形成不溶盐，然而这些盐在热水中的溶解度有差别。这样更不易溶于水的的铷和铯的六氯铂酸盐可以通过[[分级结晶|分级结晶]]的办法获得。在使用氢气还原六氯铂酸盐之后，可以利用铷和铯的碳酸盐在酒精中的溶解度差异分离开。该工艺可以从最初的44000升水中生产出9.2克[[氯化铷|氯化铷]]和7.3克[[氯化铯|氯化铯]]<ref name="BuKi1861">{{cite journal |title = Chemische Analyse durch Spectralbeobachtungen |pages = 337–381 |first1 = G.|last1 = Kirchhoff |first2 = R.|last2 = Bunsen|authorlink1 = Gustav Kirchhoff|authorlink2 = Robert Bunsen|doi = 10.1002/andp.18611890702 |journal = [[Annalen_der_Physik|Annalen der Physik und Chemie]] |volume = 189 |issue = 7|year = 1861|bibcode = 1861AnP...189..337K}}</ref>。

本生和基尔霍夫使用所获得的氯化铯估计这种新元素的原子量为123.35（目前接受的值为132.9）<ref name="BuKi1861" />。他们试图使用电解熔融氯化铯的方法来生成铯单质，但是没有获得金属物质，只获得了一种蓝色的各项同性的物质，"不论用肉眼还是在显微镜下"都无法观察到最细微的金属物质的特征。因此，他们认为它是低氯化物{{chem|Cs|2|Cl}}。实际上他们获得的可能是金属铯和氯化铯的[[胶体|胶体]]混合物<ref>{{cite book|last=Zsigmondy|first=Richard |title=Colloids and the Ultra Microscope|publisher=Read books|year=2007|isbn=978-1-4067-5938-9|page=69|url=http://books.google.com/books?id=Ac2mGhqjgUkC&pg=PAPA69}}</ref>。使用水银电极电解氯化铯的水溶液可以得到铯汞齐，在有水的条件下可以分解<ref name="BuKi1861" />。纯净的铯单质最终由德国化学家[[卡尔·赛特贝格|卡尔·赛特贝格]]在跟随[[弗里德里希·奥古斯特·凯库勒·冯·斯特拉多尼茨|凯库勒]]和本生读取博士学位的时候获得。1882年，他通过电解[[氰化铯|氰化铯]]获得了金属铯，而没有出现氯化铯的问题<ref name="Sett">{{cite journal |title =Ueber die Darstellung von Rubidium- und Cäsiumverbindungen und über die Gewinnung der Metalle selbst|doi =10.1002/jlac.18822110105 |year =1882 |last1 =Setterberg |first1 =Carl |journal =Justus Liebig's Annalen der Chemie |volume =211 |pages =100–116}}</ref>。

历史上，铯的的最重要的用途是用于研究和开发，主要是用在化学和电子领域。20世纪20年代用于无线电[[真空管|真空管]]之前，铯几乎没有任何应用。铯在真空管中起两个作用，吸收制造完毕后剩余的氧气的[[吸气剂|吸气剂]]以及可以起到增加加热的[[阴极|阴极]]的[[导电性|导电性]]作用的涂层。一直到20世纪50年代，铯才开始被认为是高性能的工业金属<ref>{{cite journal|last = Strod|first = A.J.|year = 1957|title = Cesium—A new industrial metal|journal = American Ceramic Bulletin|volume = 36|issue = 6| pages = 212–213}}</ref>。无放射性的铯开始被用于[[光电池|光电池]]、[[光电倍增管|光电倍增管]]、[[红外光谱学|红外光谱仪]]、多种有机反应的催化剂，[[闪烁计数器|闪烁计数器]]以及[[磁流体发电|磁流体发电]]<ref name="USGS" />。

自从1967年，[[国际单位制|国际单位制]]基于铯的性质定义了其时间单位，也就是秒。[[国际单位制|国际单位制]]将一秒定义为不受外场干扰的铯-133的原子基态的两个[[超精细结构|超精细结构]][[能级|能级]]间跃迁所对应的辐射的9,192,631,770个周期的持续时间<ref name=USNO>{{cite web|title = Cesium Atoms at Work|publisher=Time Service Department—U.S. Naval Observatory—Department of the Navy|url = http://tycho.usno.navy.mil/cesium.html|accessdate = 2009-12-20}}</ref>。

== 应用 ==
===石油勘探===
目前非放射性铯的最大的用途是[[石油提取工业|石油提取工业]]中使用的基于[[甲酸铯|甲酸铯]]的[[钻井液|钻井液]]<ref name="USGS" />。通过使氢氧化铯与甲酸反应制成的甲酸铯的水溶液（HCOO<sup>–</sup>Cs<sup>+</sup>）在20世纪90年代中期开始用于油井钻井以及[[完井液|完井液]]。甲酸铯作为钻井液的功能是润滑钻头，将切下来的岩石带至地表，并且在钻井期间维持对地层的压力。完井液的作用是在钻井完成后生产前辅助控制硬件进驻，而甲酸铯在其中的作用是维持压力<ref name="USGS" />。

甲酸铯盐溶液的密度很高，达2.3 g/cm<sup>3</sup><ref name="Down">{{cite conference|conference = IADC/SPE Drilling Conference|location = Miami, Florida, USASociety of Petroleum Engineers|first1 = J. D.|last1 = Downs|first2 = M.|last2 = Blaszczynski|first3 = J.|last3 = Turner|first4 = M.|last4 = Harris|doi = 10.2118/99068-MS|url = http://www.spe.org/elibinfo/eLibrary_Papers/spe/2006/06DC/SPE-99068-MS/SPE-99068-MS.htm|archiveurl = https://web.archive.org/web/20071012122901/http://spe.org/elibinfo/eLibrary_Papers/spe/2006/06DC/SPE-99068-MS/SPE-99068-MS.htm|archivedate = 2007-10-12|title = Drilling and Completing Difficult HP/HT Wells With the Aid of Cesium Formate Brines-A Performance Review|date = February 2006|deadurl = yes}}</ref>，再加上大多数铯化合物所具有的相对良性的性质，减少了钻井液中有毒的高密度悬浮固体的需求。与其它许多高密度液体的成分不同，相对来说甲酸铯不会破坏环境<ref name="Down" />。因此该技术具有很大的技术优势、工程优势以及环境优势。甲酸铯盐溶液可以与甲酸钾、甲酸钠混合以将液体的密度降低至水的密度（1.0 g/cm<sup>3</sup>）。此外，甲酸铯还可以生物降解以及回收，并且可以循环使用，考虑到其较高的价格（每[[桶|桶]]4000美元），这一点非常重要。<ref>{{cite journal|last = Flatern|first = Rick|year = 2001|title = Keeping cool in the HPHT environment|journal = Offshore Engineer|issue = February|pages = 33–37}}</ref>。碱金属的甲酸盐可以安全的处理，而且不像他们所替代的高密度盐溶液（如[[溴化锌|溴化锌]]溶液）那样具有腐蚀性，因此不会对生产底层或者井下的金属造成损害。使用这些甲酸盐也减少了清洁和处理的费用<ref name="USGS" />。

===原子钟===
[[File:Usno-mc.jpg|thumb]]
[[File:FOCS-1.jpg|thumb]]
铯[[原子钟|原子钟]]观察铯-133原子在[[超精细结构|超精细结构]]之间跃迁产生的[[电磁辐射|电磁辐射]]，并以此作为参考点。1955年，第一个精确的铯原子钟由[[路易斯·艾森|路易斯·艾森]]在[[英国国家物理实验室|英国国家物理实验室]]建成<ref>{{Cite journal |first1=L.|last=Essen|first2=J.V.L.|last2=Parry |year=1955 |title=An Atomic Standard of Frequency and Time Interval: A Caesium Resonator |journal=[[Nature_(journal)|Nature]] |volume=176 |pages=280 |doi=10.1038/176280a0|bibcode = 1955Natur.176..280E |issue=4476}}</ref>。在过去的半个多世纪中，人们不停的改进铯原子钟，并且使用它作为标准时间和频率测量的基准。这些-{zh-hans:钟; zh-hant:鐘;}-测量频率的精度为2-3×10<sup>-14</sup>，相当于时间测量的精度为每天2纳秒，或者140万年1秒。目前最先进的铯原子钟的精度超过了10<sup>-15</sup>，这意味着从6600万年前[[白垩纪－第三纪灭绝事件|恐龙灭绝]]的时代起其误差仅为2秒钟<ref name="USGS" />，被认为是“人类目前所达到的最精确的单位实现”<ref name="USNO" />。

铯原子钟也用于监控手机时间传输以及互联网信息流的网络中<ref>{{cite news|first = Monte|last = Reel|date = 2003-07-22|title = Where timing truly is everything|newspaper = Washington Post|page = B1|url = http://www.highbeam.com/doc/1P2-284155.html|accessdate = 2010-01-26|archive-url = https://web.archive.org/web/20130429044454/http://www.highbeam.com/doc/1P2-284155.html|archive-date = 2013-04-29|dead-url = yes}}</ref>。

===电力电子===
铯蒸气[[热离子发电机|热离子发电机]]是将热能转化为电能的低功率装置。在双电机[[真空管|真空管]]转换器中，铯将在阴极聚集的空间电荷清除，并通过这种方式增强电流<ref>{{cite journal| last1 = Rasor| first1 = Ned S.| first2 = Charles|last2 =Warner| title = Correlation of Emission Processes for Adsorbed Alkali Films on Metal Surfaces| journal = Journal of Applied Physics| volume = 35| issue = 9| pages = 2589–2600| date = September 1964|doi = 10.1063/1.1713806|bibcode = 1964JAP....35.2589R}}</ref>。

铯也因其具有[[光电效应|光电发射]]的性质而重要，它可以将光能转化为电流。由于诸如金属化合物[{{chem|K|2|CsSb}}等的铯基阴极发射电子的电压阈值较低，通常将铯用于[[光电池|光电池]]中<ref>{{cite web|url=http://www.americanelements.com/cs.html |title=Cesium Supplier & Technical Information |publisher=American Elements |accessdate=2010-01-25}}</ref>。使用铯的光电发射装置包括[[光学字符识别|光学字符识别]]装置、[[光电倍增管|光电倍增管]]以及[[摄像管|摄像管]]<ref>{{cite journal |doi=10.1063/1.3215593|title = K <sub>2</sub>CsSb Cathode Development| journal = American Institute of Physics Conference Proceedings |year = 2009|volume = 1149| pages = 1062–1066|first1 =John|last1 = Smedley| first2 =Triveni|last2 = Rao| first3 =Erdong|last3 = Wang}}</ref><ref>{{cite journal|first = P.|last = Görlich|title = Über zusammengesetzte, durchsichtige Photokathoden|journal = Zeitschrift für Physik|volume = 101|pages = 335–342|year =1936|doi =10.1007/BF01342330|bibcode = 1936ZPhy..101..335G|issue = 5–6}}</ref>。尽管如此，[[锗|锗]]、[[铷|铷]]、[[硒|硒]]、[[硅|硅]]、[[碲|碲]]以及其他几种元素可以替代铯作为光电发射材料<ref name="USGS" />。

[[碘化铯|碘化铯]]（CsI）、[[溴化铯|溴化铯]]（CsBr）以及[[氟化铯|氟化铯]]（CsF）晶体可以用于[[闪烁计数器|闪烁计数器]]中的[[闪烁体|闪烁体]]。由于闪烁计数器适用于检测[[伽玛射线|伽玛射线]]和[[X射线|X射线]]，它们广泛的用于矿物勘探以及粒子物理学研究中。由于铯的原子量较大，可以提供较好的停止能力，因此可以提高检测能力。铯化合物也提供了更快的响应速度（CsF）并且不易吸水（CsI）。

铯蒸气用于许多普通[[磁强计|磁强计]]中<ref>{{cite journal|doi = 10.1007/s00340-005-1773-x|title = Comparison of discharge lamp and laser pumped cesium magnetometers|year = 2005|last1 = Groeger|first1 = S.|first2 = A. S.|first3 = A.|journal = Applied Physics B|volume = 80|pages = 645–654|last2 = Pazgalev|last3 = Weis|arxiv = physics/0412011 |bibcode = 2005ApPhB..80..645G|issue = 6}}</ref>。铯也在[[分光光度法|分光光度法]]中作为[[内部标准|内部标准]]使用<ref>{{cite book|url = http://books.google.com/?id=z9SzvsSCHv4C&pg=PA108|page = 108|isbn = 978-0-471-28572-4|chapter = Internal Standards|year = 1994|first1=Mary C.|last1=Haven|first2=Gregory A.|last2=Tetrault|first3=Jerald R.|last3=Schenken|publisher = John Wiley and Sons|location = New York|title = Laboratory instrumentation}}</ref>。和其它碱金属类似，铯和[[氧|氧]]的联系紧密，因此在真空管中铯被用作[[吸气剂|吸气剂]]<ref>{{cite book|url=http://books.google.com/?id=1o1WECNJkscC&pg=PA391&lpg=391 |title=Photo-electronic image devices: proceedings of the fourth symposium held at Imperial College, London, September 16–20, 1968|volume = 1|publisher = Academic Press|year = 1969|first = James D.|last = McGee|page = 391|isbn=978-0-12-014528-7}}</ref>。铯还用于[[激光|高能激光]]、[[日光灯|蒸气辉光灯]]以及蒸气[[整流器|整流器]]中<ref name="USGS" />。

===离心液===
由于其密度较高，氯化铯、硫化铯、以及[[三氟乙酸|三氟乙酸铯]]通常在分子生物学中用于密度梯度离心操作<ref>Manfred Bick, Horst Prinz, "Cesium and Cesium Compounds" in Ullmann's Encyclopedia of Industrial Chemistry 2005, Wiley-VCH, Weinheim. {{DOI|10.1002/14356007.a06 153}}.</ref>。该技术主要用于分离生物样本中的[[病毒|病毒颗粒]]、[[细胞器|细胞器]]以及[[核酸|核酸]]<ref>{{cite book|url = http://books.google.com/?id=1kn89nI2gUsC&pg=PA61|pages = 61–62|isbn = 978-0-89603-564-5|chapter = Gradient Materials|editor =Desai, Mohamed A.|year = 2000|publisher = Humana Press|location = Totowa, N.J.|title = Downstream processing methods}}</ref>。

===化学与医学应用===

===核应用与同位素应用===

[[铯-137|铯-137]]在工业应用中是一种非常常见的作为[[伽玛射线|伽玛射线]]发射源的同位素。其优势在于它的半衰期大约30年，可以通过[[核燃料循环|核燃料循环]]获得，并且其最终产物[[钡-137|钡-137]]是一种稳定的同位素。其较高的水溶性是其缺点，使得它无法用在用于食品和医疗用品的大型池式辐射器中<ref name="Takeshi">{{cite web|url = http://earth1.epa.gov/radiation/docs/source-management/csfinallongtakeshi.pdf|title = The material flow of radioactive cesium-137 in the U.S. 2000|first = Takeshi|last = Okumura|date = 2003-10-21|format = PDF|accessdate = 2009-12-20|publisher = United States Environmental Protection Agency|deadurl = yes|archiveurl = https://web.archive.org/web/20110720163223/http://earth1.epa.gov/radiation/docs/source-management/csfinallongtakeshi.pdf|archivedate = 2011-07-20}}</ref>。铯-137已经被用在农业、癌症治疗、食品消毒、污水污泥处理以及外科手术设备中。<ref name="USGS" /><ref>{{cite book|last = Jensen|first = N. L.|year = 1985|chapter=Cesium|title = Mineral facts and problems|publisher = U.S. Bureau of Mines|volume = Bulletin 675|pages = 133–138}}</ref>。铯的放射性同位素可以用在[[放射线疗法|放射线疗法]]中针对某些癌症治疗<ref>{{cite web|url=http://www.medicalnewstoday.com/releases/91994.php|title=IsoRay's Cesium-131 Medical Isotope Used In Milestone Procedure Treating Eye Cancers At Tufts-New England Medical Center|date= 2007-12-17 |publisher=Medicalnewstoday.com |accessdate=2010-02-15}}</ref>，然而由于目前已经有了更好的替代品，且放射源中易溶于水的[[氯化铯|氯化铯]]可能造成大范围污染，放疗中逐渐不再采用铯放射源<ref>{{Cite book|url = http://books.google.com/books?id=bk0go_-FO5QC&pg=PA22|isbn = 978-0-07-005115-7|chapter = Caesium-137 Machines|title = Radiation therapy planning|first = Gunilla Carleson|last = Bentel|publisher = McGraw-Hill Professional|year = 1996|accessdate=2010-09-26|pages = 22–23}}</ref><ref>{{Cite book|isbn = 978-0-309-11014-3| url =http://books.google.com/books?id=3cT2REdXJ98C|title = Radiation source use and replacement: abbreviated version|author = National Research Council (U.S.). Committee on Radiation Source Use and Replacement|publisher = National Academies Press|year = 2008}}</ref>。在许多工业测量计中都采用了铯-137，包括湿度计、密度计、水平仪以及厚度计<ref name="gauges">{{cite book|chapter = Level and density measurement using non-contact nuclear gauges|isbn = 978-0-412-53400-3|url =http://books.google.com/?id=RwsoQbHYjvwC&pg=PA82|pages = 82–85|editor= Loxton, R., Pope, P.|year = 1995|publisher = Chapman & Hall|location = London|title = Instrumentation : A Reader}}</ref>。[[测井|测井]]设备中也会使用铯-137来测量与岩层中的电子密度<ref>{{cite journal|doi = 10.1146/annurev.ea.13.050185.001531|title = Downhole Geophysical Logging|year = 1985|last1 = Timur|first1 = A.|last2 = Toksoz|first2 = M. N.|journal = Annual Review of Earth and Planetary Sciences|volume = 13|page = 315|bibcode = 1985AREPS..13..315T}}</ref>。

铯-137也用于[[水文学|水文学]]研究中。铯是核裂变反应的产物。自从大约1945年[[核试验|核试验]]开始，一直到20世纪80年代中期，铯137被释放进入大气层，然后立即被吸收入水溶液中。那个时期的年度变化与土壤和沉积层有相关性。铯-134以及含量更少的铯-135也用于水文学研究作为核电工业中产生的铯的度量。这两种同位素不像铯-133或者铯-137那样常见，而且仅能通过人为过程产生<ref>{{Cite journal|first=Carol|last=Kendall |url=http://wwwrcamnl.wr.usgs.gov/isoig/period/cs_iig.html |title=Isotope Tracers Project – Resources on Isotopes – Cesium|publisher=National Research Program – U.S. Geological Survey |accessdate=2010-01-25}}</ref>。<!--http://books.google.com/books?id=pWDQnxd-r1UC&pg=PT360 &pg=PT12 -->

===其它应用===
[[File:Electrostatic_ion_thruster-en.svg|thumb]]

在早期的为长距离星际航行的[[航天器推进|航天器推进]]器设计的[[离子推力器|离子推力器]]中使用铯和汞作为推进剂。在离子化方法中，需要通过与加载电压的[[钨|钨]]电极接触剥去推进剂的外层电子。考虑到铯对飞船部件的腐蚀作用，对离子推进器的研发转向使用惰性气体推进剂，例如[[氙|氙]]。在地面测试中处理惰性气体推进剂，并且其对飞船造成影响的可能性也更小<ref name="USGS" />。最终在1998年发射的试验飞船[[深空1号|深空1号]]中使用氙作为推进剂<ref>{{cite journal|doi =10.1063/1.1150468|title =NSTAR Xenon Ion Thruster on Deep Space 1: Ground and flight tests (invited)|year =2000|last1 =Marcucci|first1 =M. G.|last2 =Polk|first2 =J. E.|journal =Review of Scientific Instruments|volume =71|pages =1389–1400|bibcode = 2000RScI...71.1389M|issue =3}}</ref><ref>{{cite web|url = http://gltrs.grc.nasa.gov/reports/1999/TM-1999-209439.pdf|format = PDF|title = A Synopsis of Ion Propulsion Development Projects in the United States: SERT I to Deep Space I|first1 = James S.|last1 = Sovey|first2 = Vincent K.|last2 = Rawlin|first3 = Michael J|last3 = Patterson|publisher = NASA|accessdate = 2009-12-12|deadurl = yes|archiveurl = https://web.archive.org/web/20090629225625/http://gltrs.grc.nasa.gov/reports/1999/TM-1999-209439.pdf|archivedate = 2009-06-29}}</ref>。尽管如此，已经建造的[[场发射电子推进|场发射电子推进]]推力器仍然使用加速诸如铯等的金属离子的简单系统来形成推力。<ref>{{cite conference|url=http://trs-new.jpl.nasa.gov/dspace/handle/2014/11649|title=In-FEEP Thruster Ion Beam Neutralization with Thermionic and Field Emission Cathodes|format=PDF|conference=27th International Electric Propulsion Conference|place=Pasadena, California|pages=1–15|accessdate=2010-09-26|author=Marrese, C.; Polk, J.; Mueller, J.; Owens, A.; Tajmar, M.; Fink, R. and Spindt, C.|date=October 2001|deadurl=yes|archiveurl=https://web.archive.org/web/20100527071653/http://trs-new.jpl.nasa.gov/dspace/handle/2014/11649|archivedate=2010-05-27}}</ref>

在[[红外|红外]][[照明弹|照明弹]]中使用[[硝酸铯|硝酸铯]]作为[[氧化剂|氧化剂]]和[[烟火着色剂|烟火着色剂]]来燃烧[[硅|硅]]<ref>{{cite web|url=http://www.freepatentsonline.com/6230628.html |work=United States Patent 6230628|title=Infrared illumination compositions and articles containing the same |publisher=Freepatentsonline.com |accessdate=2010-01-25}}</ref>，例如LUU-19照明弹<ref>{{cite web|url = http://www.fas.org/man/dod-101/sys/dumb/luu19.htm|title = LUU-19 Flare|publisher = Federation of American Scientists|date = 2000-04-23|accessdate = 2009-12-12|deadurl = yes|archiveurl = https://web.archive.org/web/20110609030345/http://www.fas.org/man/dod-101/sys/dumb/luu19.htm|archivedate = 2011-06-09}}</ref>。这是因为硝酸铯的大部分辐射光都位于[[近红外光谱|近红外光谱]]中<ref>{{cite journal|doi = 10.1016/j.tca.2006.04.002|title = Determination of the temperature and enthalpy of the solid–solid phase transition of caesium nitrate by differential scanning calorimetry|year = 2006|last1 = Charrier|first1 = E.|first2 = E.L.|first3 = P.G.|first4 = H.M.|first5 = B.|first6 = T.T.|journal = Thermochimica Acta|volume = 445|pages = 36–39 |last2 = Charsley|last3 = Laye|last4 = Markham|last5 = Berger|last6 = Griffiths}}</ref>。铯也被用来减少[[SR-71黑鸟式侦察机|SR-71黑鸟式侦察机]]的[[尾气|尾焰]]的[[雷达散射截面|雷达特征]]<ref>{{cite book|isbn = 978-1-84176-098-8|page = 47|title=Lockheed SR-71: the secret missions exposed|last=Crickmore|first = Paul F.|publisher=Osprey|year=2000}}</ref>。铯也经常和铷一起以碳酸盐的形式加入玻璃以减少玻璃的导电性并且增加[[光纤|光纤]]与[[夜视镜|夜视设备]]的稳定性和持久性。氟化铯或者氟化铝铯也用在以含[[镁|镁]][[铝|铝]]合金钎焊的焊剂配方中<ref name="USGS" />。

==健康以及安全危害==

[[File:AirDoseChernobylVector.svg|thumb]]

大多数人极少能碰到铯的化合物，但是由于铯的化学性质接近钾，大多数化合物只有轻微的毒性。暴露在大量的铯化合物中会引起[[应激性|应急过度]]以及[[痉挛|痉挛]]，但是由于在自然情况下不会遇到这样大的剂量，铯并不是重要的化学环境污染物<ref>{{cite journal|doi = 10.1080/10934528109375003|title = Cesium in mammals: Acute toxicity, organ changes and tissue accumulation|year = 1981|last1 = Pinsky|first1 = Carl|first2 = Ranjan|first3 = J. 金属铯是最活泼的元素之一，并且接触水后会发生爆炸。铯接触水后产生大量的热，加热了同时产生的氢气并将其点燃，引起猛烈地爆炸。其它的碱金属也能发生相似的情况，但是铯和水的反应如此剧烈以至于可以和冷水反应<ref name="USGS" />。铯的[[自燃温度|自燃温度]]为-116°C，因此铯极度容易自燃，并且在空气中会发生爆炸性的燃烧，形成[[超氧化铯|超氧化铯]]以及各种氧化物。超氧化铯是一种非常强的[[碱|碱]]，可以迅速的腐蚀玻璃<ref name=RSC>{{cite web|url=http://www.rsc.org/periodic-table/element/55/caesium|accessdate=2010-09-27|publisher=Royal Society of Chemistry|title=Chemical Data – Caesium – Cs}}</ref>。

由于人类的活动，[[生物圈|生物圈]]中存在少量的铯的同位素[[铯-134|铯-134]]和[[铯-137|铯-137]]，并且在不同地点放射强度也有所不同。
