'''宣弘社'''（日語假名：{{lang|ja|せんこうしゃ}}），是1950年代到1970年代日本制作[[电视电影|电视电影]]的单位，在2002年转让到「圣宣弘社（現[[电通冒险齿轮|电通冒险齿轮]]）」。

== 歴代社長 ==
=== 旧宣弘社（1941年－2002年） ===
# 初代 姓名不詳 （小林利雄之父、在任：1941年－1948年）
# 二代目 [[小林利雄|小林利雄]] （在任：1948年－2001年，会長）
# 三代目 小林隆吉（在任：2001年－2002年），小林利雄的長男<ref name = "60年代" />。少年時代曾參與《月光假面》的演出<ref name = "60年代">石橋春海 「宣弘社の現在」『60年代 蘇る昭和特撮ヒーロー』 コスミック出版〈COSMIC MOOK〉、2013年12月5日、124-125頁。ISBN 978-4-7747-5853-4</ref>。

=== 宣弘社（2008年－） ===
* 小林隆吉
