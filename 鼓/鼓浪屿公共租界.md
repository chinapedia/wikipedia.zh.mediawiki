'''鼓浪屿公共租界'''是中国在清末到民国时期的2个公共租界之一（另外一个公共租界是[[上海公共租界|上海公共租界]]）。

在[[鸦片战争|鸦片战争]]期间，英军占领了[[鼓浪屿|鼓浪屿]]岛，直到1845年才撤军。1843年后，[[厦门|厦门]]根据《[[中英南京条约|中英南京条约]]》开辟为[[通商口岸|通商口岸]]，英国另在厦门本岛内港海岸获得了一块滩地兴建英租界，鼓浪屿则保持原有风貌。

==歷史==
*1895年中日[[甲午战争|甲午战争]]后，[[日本|日本]]統治[[台湾|台湾]]，为避免日本进一步觊觎厦门，清朝政府决定寻求“国际保护”，请列强“兼护厦门”<ref>张震世：《公共租界时期的鼓浪屿》，《厦门文史资料》第3辑，第20页</ref>。

*1902年1月10日（光绪二十七年十二月初一日），[[英国|英国]]、[[美国|美国]]、[[德国|德国]]、[[法国|法国]]、[[西班牙|西班牙]]、[[丹麦|丹麦]]、[[荷兰|荷兰]]、[[瑞挪联盟|瑞挪联盟]]、日本等9国驻厦门领事与[[清朝|清朝]][[福建省|福建省]]兴泉永道台延年在鼓浪屿日本领事馆签订《厦门鼓浪屿公共地界章程》<ref>中国第一历史档案馆外务部档案，综合类，第4462号</ref>，鼓浪屿成为公共租界<ref>{{Cite web |url=http://www.laozihao.cn/shownews.asp?newsid=469# |title=清三 |access-date=2008-02-15 |archive-url=https://web.archive.org/web/20071008190435/http://www.laozihao.cn/shownews.asp?newsid=469# |archive-date=2007-10-08 |dead-url=yes }}</ref>，次年1月，鼓浪屿公共租界工部局成立。在此前后，陆续有英、美、法、德、日等13个国家先后在岛上设立[[领事馆|领事馆]]。

这样，厦门地区就形成2个租界隔海对峙的局面。厦门英租界面积狭小，完全是商贸区；而鼓浪屿面积较大，形成大规模的西式住宅区。

*这段时期，有很多[[传教士|传教士]]来到鼓浪屿，他们建立的学校对[[中国教育|中国现代教育]]有重大影响<ref>[http://www.amoymagic.com/discovergulangyu5eduzh.htm]</ref>，例如：1898年，英国牧师韦玉振与夫人韦爱莉到鼓浪屿传教时创办的“怀德幼稚园”是中国第一所[[幼儿园|幼儿园]]（现已更名为日光幼儿园）<ref>{{cite web |url=http://www.fjedu.gov.cn/newsInfo.aspx?pkId=14032 |title=存档副本 |accessdate=2008-02-15 |deadurl=yes |archiveurl=https://web.archive.org/web/20070928024716/http://www.fjedu.gov.cn/newsInfo.aspx?pkId=14032 |archivedate=2007-09-28 }}</ref>。

*1930年厦门英租界被[[国民政府|国民政府]]收回后，鼓浪屿公共租界就成为厦门地区唯一的租界。
*1938年日军进攻并占領厦门，鼓浪嶼成為一座“孤島”（[[廈門|廈門]]，[[金門|金門]]皆為日軍控制），大批[[华人|华人]]為避日禍移居鼓浪屿或中轉返回自由區。
*1941年12月[[太平洋战争|太平洋战争]]爆发后，日军进驻鼓浪屿公共租界。
*1943年1月11日，美国和英国决定放弃在[[中国|中国]]的特殊权利。
*1943年5月28日，[[汪精卫政权|汪精卫政权]]收回鼓浪屿公共租界。
*1945年，鼓浪屿公共租界被[[国民政府|国民政府]]正式收回。为厦门市[[鼓浪屿区|鼓浪屿区]]，2003年降为[[鼓浪屿街道|鼓浪屿街道]]，下辖龙头、内厝2社区。
