'''滚铁圈'''，俗称“'''滚轱辘圈'''”或者“'''滚铁环'''”，是20世紀中後期一度流行於中國的一种游戏。主要分佈於鄉村地區中，在许多鄉土文学作品中也不乏對“滚铁圈”的描寫。

滚铁圈和“[[丢沙包|丢沙包]]”、“[[抓骨头|抓骨头]]”、“[[拍画片|拍画片]]”、“[[橡皮筋|橡皮筋]]”等许多游戏一起，构成了许多中国人的儿时记忆。铁圈既是娱乐工具也是健身器材，它由一个铁圈和一个铁钩组成。遊戲時先將铁圈套入铁钩，然後手持鐵鉤控制鐵圈，使其在地上按照規定的路線移動。這種遊戲方式需要较好的平衡能力和力量。

玩得好的人，不仅可以用铁钩控制铁圈的方向，还能玩出很多不同的花樣。例如讓鐵圈飛起來再接住等等。限于人们当时的物质生活水平，铁圈的制作材料往往是工地或垃圾堆中的废旧铁丝。所以，制作一个质量较好的铁圈，需要经过长时间的寻找和运气，但也可使用手中的一些多余材料或玩具与别人交换获得。

随着生活的进步，可供选择的娱乐的健身项目越来越多，滚铁圈也渐渐变成一种回忆，但仍有一批对此项运动钟爱的人，其中不乏成年人和老年人，除了做为一种运动项目锻炼身体以外，可能更多的是一种情怀。

另外台灣的民俗遊戲也有滾鐵圈这一项目,至今仍有部分地區保留。

滚铁圈在古代亚洲、欧洲和非洲的历史中都有记录。在亚洲，古代中国人最早玩这个游戏，在欧洲则是古希腊最早玩。

== 简介 ==
滾鐵圈，俗稱“滾軲轆圈”，是一種七八十年代農村裏的遊戲，既是娛樂工具也是健身器材。由一個鐵圈和一個鐵鉤組成，操作時需要極大的平衡能力和技術。

滾鐵圈是以前的小孩常玩的一種遊戲，最酷的遊戲就是滾鐵圈。

== 操作 ==
它需要極大的平衡能力和技術，假如你不具備這兩點的話,就很難玩好它。

它由一個鐵圈和一個鐵鉤組成。你要用鐵鉤勾住鐵圈讓鐵圈滾動起來，自己也跟著跑起來，不過這鐵圈可是很容易倒的哦！

== 主要信息 ==
鐵圈並不難找，然後另用鐵絲彎個鐵鉤作為推手，便算大功告成。於是，小夥伴們招呼在一起，相擁結伴地沿街推著鐵圈往前走，一時間笑聲、鐵圈撞擊聲此起彼伏，蔚為大觀。玩得好的人，不僅可以用鐵鉤控制鐵圈的方向，直走、轉彎，還能用鐵鉤在鐵圈底部猛一發力，讓鐵圈歸去來兮，或使鐵圈跳起而不倒。

滾鐵圈，俗稱“滾軲轆圈”，其來源與發展歷程已無從考證，七八十年代農村裏的會有小孩用於玩耍，是一種娛樂工具也是健身器材。
