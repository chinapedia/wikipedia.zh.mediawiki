'''敦化路街道'''是[[中华人民共和国|中国]][[山东省|山东省]][[青岛市|青岛市]][[市北区|市北区]]下辖的一个街道。

== 行政区划 ==

敦化路街道下辖西吴社区、东吴社区、山东路社区、金坛路社区、伊春路社区、连云港路社区。<ref>{{Cite web|url=http://www.stats.gov.cn/tjsj/tjbz/tjyqhdmhcxhfdm/2015/37/02/03/370203014.html|title=统计用区划和城乡划分代码/山东省/青岛市/市北区/敦化路街道办事处|accessdate=2017-04-23|author=国家统计局|date=2015-09-30|publisher=}}</ref>
== 人口 ==

2010年[[中国第六次人口普查|中国第六次人口普查]]时，敦化路街道共有人口43726人。共有家庭16409户，平均每户2.54人。14岁以下的少年儿童共4659人，占总人口10.65%；15-64岁人口共34423人，占总人口78.72%；65岁及以上的老年人共4644人，占总人口的10.62%。男性共计21454人，占总人口49.06%；女性共计22272，占总人口50.93%。本地居住的人口中，拥有本地户籍的人口为23098人，占52.82%。<ref name="人口普查2010">{{cite web| url =http://www.yearbookchina.com/navipage-n2013030152000017.html| title =《中国2010年人口普查分乡、镇、街道资料》| author =中国国务院人口普查办公室、中国国家统计局人口就业统计司等| year =2010| website =统计年鉴分享平台| page =表15 山东省乡、镇、街道人口| format =收费| access-date =2018-07-19| archive-url =https://web.archive.org/web/20180719084356/http://www.yearbookchina.com/navipage-n2013030152000017.html| archive-date =2018-07-19| dead-url =no}}</ref>
