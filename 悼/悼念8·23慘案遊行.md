'''悼念菲律賓遇難港人聯合大遊行'''<ref name="星島20100829" />，又稱'''悼念8·23慘案遊行'''，是指2010年8月29日，香港多個政黨就[[馬尼拉人質事件|馬尼拉人質事件]]發起的靜默遊行，亦是繼1989年[[八九學運|八九學運]]以及2003年[[嚴重急性呼吸系統綜合症|沙士]]時期後，香港[[泛民主派|泛民主派]]及[[親建制派|親建制派]]再一次合作。值得一提的，是遊行的日期剛好是死難者的[[頭七|頭七]]。另外，當天[[香港天文台|香港天文台]]發出了一號熱帶氣旋警告，該風暴名稱正是香港命名的[[強烈熱帶風暴獅子山_(2010年)|獅子山]]。

==背景==
{{main|馬尼拉人質事件}}
於2010年8月23日當地時間上午10時半左右<ref>{{cite web|url=http://edition.cnn.com/2010/WORLD/asiapcf/08/23/philippines.bus.hostages/index.html|title=Gunman in the Philippines ends standoff by killing 8, wounding 7 - CNN.com|first=By the CNN Wire|last=Staff|date=|website=cnn.com}}</ref>，已遭革職的前菲律賓高級警務[[督察|督察]][[羅蘭多·門多薩|羅蘭多·門多薩]]以[[搭便車|搭便車]]為名，手持[[M16突擊步槍|M16突擊步槍]]登上香港[[康泰旅行社|康泰旅行社]]在[[菲律賓|菲律賓]]的[[旅遊巴|旅遊巴]]，挾持車上23名[[人質|人質]]。事件持續11小時後以流血告終，造成旅行團8死7傷<ref>[http://edition.cnn.com/2010/WORLD/asiapcf/08/23/philippines.bus.hostages/index.html?hpt=T1#fbid=Swz4WbZ0xz0&wom=false Bloody end to Philippines hostage drama]，[[CNN|CNN]]，23-8-2010 17:07(UTC)</ref><ref>[http://www.bbc.co.uk/news/world-asia-pacific-11055015 Philippine police end Manila bus hijack]，[[BBC|BBC]]，23-8-2010 16:56(UTC)</ref>。

==遊行經過==
是次活動主要由[[民主黨_(香港)|民主黨]]、[[公民黨_(香港)|公民黨]]、[[工聯會|工聯會]]及[[民建聯|民建聯]]聯合籌辦<ref>{{cite web |url=http://news.hkheadline.com/dailynews/content_hk/2010/08/28/120924.asp |title=存档副本 |accessdate=2010年8月29日 |deadurl=yes |archiveurl=https://web.archive.org/web/20100901133617/http://news.hkheadline.com/dailynews/content_hk/2010/08/28/120924.asp |archivedate=2010年9月1日 |df= }} [[頭條日報|頭條日報]] 8月28日報導 </ref>，[[自由黨_(香港)|自由黨]]、[[民協|民協]]及[[社民連|社民連]]均有派人出席。民主黨負責舞台音響、立法會致送花圈、牌照申請等事宜；民建聯準備五萬條黃色絲帶派給參加者；工聯會安排糾察人員。<ref name="news.sina.com.hk">{{cite web |url=http://news.sina.com.hk/cgi-bin/nw/show.cgi/2/1/1/1835528/1.html |title=存档副本 |accessdate=2010年8月29日 |deadurl=yes |archiveurl=https://web.archive.org/web/20100831045954/http://news.sina.com.hk/cgi-bin/nw/show.cgi/2/1/1/1835528/1.html |archivedate=2010年8月31日 |df= }} [[星島日報|星島日報]]報導 </ref>維園的六個足球場劃為隊伍集合的地點，其中最近天后的足球場已搭建舞台，台上拉起黑底白字標語，寫有「沉痛哀悼 遇難同胞，要求徹查 事件真相」字句，現場就播出哀樂。<ref name="星島20100829" />

遊行由[[維園|維園]]出發，以遮打花園為目的地。是次遊行當中，大部分人士均響應主辦單位，以靜默表示哀悼及以穿著黑色或白色素衣。另一方面，大會將派發黃色絲帶供市民繫上，遊行人士也可攜帶鮮花向死難者致哀。由於遮打花園既是慣常遊行集會抗議的場地，也是菲傭在星期日休假聚集的地點，所以大會也有呼籲遊行人士保持冷靜，避免出現種族仇視的情緒<ref name="存档副本">{{cite web |url=http://hk.news.yahoo.com/article/100827/4/jxa0.html |title=存档副本 |accessdate=2010-08-28 |deadurl=yes |archiveurl=https://web.archive.org/web/20100904085112/http://hk.news.yahoo.com/article/100827/4/jxa0.html |archivedate=2010-09-04 }}</ref>。另一方面，各政黨只准帶備一中一英的示威橫額，讓市民辨認走入隊伍。<ref name="news.sina.com.hk"/>

遊行先由立法會主席[[曾鈺成|曾鈺成]]宣讀立場，批評[[菲律賓政府|菲律賓政府]]處理8月23日劫持事件的手法犯有重大錯失，促請香港特區政府嚴正要求馬尼拉當局嚴查真相，並公開道歉。聲明也同時要求港府爭取直接參與調查工作，並為他日可能發生的同類型事件制定明確的應變機制。然後，全場默哀三分鐘，然後沿電車路遊行至[[遮打花園|遮打花園]]，拉起「哀悼遇害同胞」及「徹查事件真相」中英文標語，但不會喊口號。遊行隊伍龍頭1小時後抵達遮打花園，立法會議員帶領默哀1分鐘，向遇難者獻花<ref name="存档副本"/>。遊行人數達8萬，這是一次立法會全體議員歷史性的跨黨派靜默遊行，各黨派罕有的走在一起在這悲傷的氛圍中展示團結<ref>{{cite news|url=http://www.bbc.co.uk/zhongwen/trad/china/2010/08/100829_hongkong_manila_mourning.shtml|title=八萬港人遊行悼念馬尼拉慘案遇難者|publisher=[[BBC|BBC]]|date=2010-08-29}}</ref>。遊行人士將黃絲帶綁在欄杆，以示哀悼後散去。<ref name="星島20100829">{{cite news | language =  | author =  | coauthors =  | url = http://hk.news.yahoo.com/article/100829/3/jy0q.html | title = 8萬人遊行悼菲遇難港人 | work =  | publisher = 星島 | pages =  | date = 2010-08-29 | accessdate = 2010-08-31 }}</ref>

遊行隊伍行經軒尼詩道時，警方封閉三條行車線，沿途有很多市民加入<ref>{{cite web|url=http://www.881903.com/Page/ZH-TW/newsdetail.aspx?ItemId=274344&csid=261_341|title=881903.com 商業電台 - 遊行悼念在菲被殺人質沿途很多市民加入|author=|date=|website=881903.com}}</ref>，有不少市民均自製橫額。至下午四時半，龍頭已經到達遮打花園。<ref>{{cite web |url=http://hk.apple.nextmedia.com/realtime/art_main.php?iss_id=20100829&sec_id=6996647&art_id=14395492 |title=存档副本 |accessdate=2010年8月29日 |deadurl=yes |archiveurl=https://web.archive.org/web/20100831165945/http://hk.apple.nextmedia.com/realtime/art_main.php?iss_id=20100829&sec_id=6996647&art_id=14395492 |archivedate=2010年8月31日 |df= }} </ref>不過，因為參加人數眾多，市民需要分批出發，等候期間10多名市民抵不住烈日暴曬，懷疑中暑暈眩嘔吐送院。<ref>{{cite web |url=http://hk.apple.nextmedia.com/template/apple/art_main.php?iss_id=20100830&sec_id=4104&subsec_id=11866&art_id=14396869 |title=存档副本 |accessdate=2010年8月30日 |deadurl=yes |archiveurl=https://web.archive.org/web/20100831170432/http://hk.apple.nextmedia.com/template/apple/art_main.php?iss_id=20100830&sec_id=4104&subsec_id=11866&art_id=14396869 |archivedate=2010年8月31日 |df= }} </ref>
