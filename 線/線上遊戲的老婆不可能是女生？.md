《'''線上遊戲的老婆不可能是女生？'''》（{{lang-ja|ネトゲの嫁は女の子じゃないと思った？}}）是{{tsl|ja|聴猫芝居|聽貓芝居}}所作的[[日本|日本]][[輕小說|輕小說]]作品，[[插圖|插圖]]由[[Hisasi|Hisasi]]繪畫，由[[ASCII_Media_Works|ASCII Media Works]]出版，中文版則由[[台灣角川|台灣角川]]發行。2015年8月7日宣佈改編為電視動畫<ref>{{cite news|title=熱門輕小說《線上遊戲的老婆不可能是女生？》宣布動畫化，「電擊文庫 秋之祭典2015」時還會有舞台活動！|url=http://www.animen.com.tw/NewsArea/NewsItemDetail?NewsId=11481&categoryId=1&tagName=none&realCategoryId=1&subCategoryId=9|publisher=Animen新聞部|accessdate=2015年8月7日|date=2015年8月7日}}</ref>。

== 故事簡介 ==
就讀縣立前崎高中的主角'''西村英騎'''是網絡遊戲《传奇时代》（Legendary Age ，簡稱LA）的玩家，由於曾在遊戲中告白，但被對方告知是假冒女生的人妖，而對此有陰影，並堅持把遊戲和現實分得清清楚楚。有一天竟被網絡遊戲中的女玩家'''玉置亞子'''告白了，雖然在遊戲中接受了亞子的表白而結婚，但仍然不敢確信亞子在現實中是女生。直到有一天，盧西安所屬四人公會「後巷貓（Alley Cats）」{{註|1=第1-3卷中工会名称翻译为「陣列貓」（Array Cats），第4卷起翻译變為「後巷貓」（Alley Cats），因文中御圣院杏提到“后巷猫就是野猫的意思”。日文中Array Cats和Alley Cats读音均为アレイキャッツ。因此「陣列貓」與「後巷貓」在文中含義一致，均指主角們所處的工會。}}（後巷貓公會）的會長決定舉行線下聚會，結果英騎發現公會內四人竟然全是同高中的學生，且自己以外的三人、包括亞子、公會會長'''御聖院杏'''（LA內名稱：亞普力克）、會員'''瀬川茜'''（LA內名稱：修凡）都是可愛的女孩子。

但是，亞子是一個無法區別現實與遊戲、將兩者重度混淆的患者，她會在現實世界做出遊戲內的行為：包括以遊戲暱稱稱呼別人，認定英騎是自己現實中的丈夫，以及「使用技能」等等。為了讓亞子能夠恢復正常，眾人成立了「'''現代通信電子遊戲社'''」（即是網遊社），他們的遊戲生活就此延續至學校。

== 登場角色 ==
=== 主要人物 ===
; {{nihongo|西村英騎|西村 英騎（にしむら ひでき）|聲：[[豐永利行|豐永利行]]}}
: 本作男主角，小說中的第一人稱敘事者。前崎高中1年級生。在傳奇時代（簡稱LA）中是防具至上主義的鎧甲騎士'''盧西安'''（{{lang|ja|ルシアン}}），公會中的[[坦克_(電子遊戲)|坦克]]。由於在LA中告白遭拒、並得知對方是[[電子遊戲術語列表#玩家|網路人妖]]，因此將網路世界與現實世界區分得很清楚。但在公會舉辦聚會並遇見真實世界的亞子後，漸漸地喜歡上她。在LA以外是個[[御宅族|御宅族]]。

; {{nihongo|玉置亞子|玉置 亜子（たまき あこ）|聲：[[日高里菜|日高里菜]]}}
: 本作女主角，西村的同學。前崎高中1年級生。具有嚴重的[[家裡蹲|家裡蹲]]傾向和[[遊戲成癮|遊戲成癮]]，完全無法分辨網路遊戲與現實世界的區別。在LA中以本名'''亞子'''（{{lang|ja|アコ}}）做為角色暱稱，是外觀至上主義的[[電子遊戲術語列表#遊戲角色|牧師]]，經常穿著外觀可愛但實用性低的裝備。與遊戲中的盧西安是[[網婚|夫妻]]。在現實中、會以遊戲角色暱稱稱呼他人，有溝通和社交方面的障礙，曾經拒絕與西村交往，因為不想從夫妻降級到情侶。

=== 縣立前崎高中 ===
; {{nihongo|瀨川茜|瀬川 茜（せがわ あかね）|聲：[[水瀨祈|水瀨祈]]}}
: 前崎高中1年級生。西村的同學。個子嬌小的[[金髮|金髮]][[雙馬尾|雙馬尾]]女孩，在現實中極度厭惡御宅族，但事實上也是網路遊戲玩家。在LA中是持劍的男性角色'''修凡'''（{{lang|ja|シュヴァイン}} 配音-[[松風雅也|松風雅也]]），攻擊至上主義的劍舞者。她在現實中人際關係優秀、相當受歡迎，但拒絕了所有的告白，只因為戀愛會讓玩遊戲的時間變少。她嚮往男人之間的友情，但並不是腐女。

; {{nihongo|御聖院杏|御聖院 杏（ごしょういん きょう）|聲：[[市道真央|M·A·O]]}}
: 前崎高中2年級生。前崎高中學生會會長，同時也是公會會長。外表亮麗、成績優秀、收入優渥，但由於不善交際、沒有除了公會成員以外的朋友。在LA中是付費至上主義的男性魔法使'''亞普力可'''（{{lang|ja|アプリコット}} 配音-[[羽多野涉|羽多野涉]]），在遊戲或現實中面對問題時、大部分都會以錢來解決。但遊戲技巧非常差，若不使用付費道具、則實力遠遜於亞子。

; {{nihongo|秋山奈奈子|秋山 奈々子（あきやま ななこ）|聲：[[大和田仁美|大和田仁美]]}}
: 前崎高中1年級生。PTA自治聯盟會長的孫女，人際關係相當好。在LA中是輕鬆玩至上主義的女性召喚師'''瑟蒂'''（{{lang|ja|セッテ}}）。她是茜的好友，在發現茜是線上遊戲玩家的事後、幫她保守秘密，並在之後也一起成為玩家並加入社團。
: 小說第十三卷參加學生會長競選，敗給高石涼香，但受御聖院杏之委託接任現代通信電子遊戲部社長。

; {{nihongo|齊藤結衣|斉藤 結衣（さいとう ゆい）|聲：[[南條愛乃|南條愛乃]]}}
: 前崎高中的老師，西村的班導師，指導科目為國文，戴眼鏡的女性，是電子遊戲社的顧問。在LA中是等級相當高的紅衣主教'''貓姬'''（{{lang|ja|猫姫}}），擁有大量狂熱擁護者。是西村第一次在LA中告白的對象，被杏抓到此把柄後接下顧問一職。

; {{nihongo|雙葉美柑|双葉 みかん（ふたば みかん）}}
: 小說第十卷登場，前崎高中的新入學生，就讀一年五班，瑞姬的朋友與同學，與西村英騎一起值班的圖書委員。原要加入現代通信電子遊戲部，但自己認為目前贏不過亞子，要等到自己可以贏過亞子才要加入社團。對傳奇時代的遊戲理解相當快。
: 小說第16卷加入現代通信電子遊戲部。

; 高石涼香
: 小說第十卷登場，前崎高中的新入學生，就讀一年五班，瑞姬的朋友與同學。因憧憬學生會及會長而要加入學生會，卻誤入現代通信電子遊戲部，發現課金的程度後隨即跑了。
: 小說第十三卷參加學生會長競選，擊敗秋山奈奈子而成為新任學生會長。

=== 主要人物的家族 ===
; {{nihongo|西村瑞姬|西村 瑞姫（にしむら みずき）|聲：[[飯塚麻結|飯塚麻結]]}}
: 英騎的妹妹，重度兄控。在LA中是一名武僧'''巧巧'''（{{lang|ja|シュシュ}}），屬於單打獨鬥型玩家。
: 小說第16卷加入現代通信電子遊戲部。

; {{nihongo|亞子的母親|亜子の母|聲：[[高本惠|高本惠]]}}
: 亞子的母親，有著跟亞子如出一轍的容貌。經常從亞子口中聽到英騎的事，因此對於英騎有好感，在英騎初次到家裡拜訪時放心的把亞子房間的鑰匙交給英騎。
: 嫉妒心強，當丈夫收到其他女性（包含亞子）送的情人節巧克力時會變得非常恐怖。

; {{nihongo|亞子的父親|亜子の父}}
: 上班族，長相英俊。經常從亞子口中得知英騎的事。在英騎眼中，是玉置家中最正常的人。

; {{nihongo|御聖院時宗|御聖院 時宗（ごしょういん ときむね）}}
: 杏的伯公。前崎高中的理事長。
: 已把實務交由自己侄子（杏的父親）打理。自稱是個掛名的理事長。
: 年輕時，曾經闖入自己妻子與相親對象的相親會場，並且將她帶走。本人至今對此事感到相當自豪。因此對突然闖入的英騎不加責備，反而十分欣賞。

; {{nihongo|英騎的母親|英騎の母|聲：}}
: 英騎與瑞姬的母親，是一名身形嬌小的女性。
