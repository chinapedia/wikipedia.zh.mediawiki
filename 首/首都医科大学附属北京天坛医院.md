'''首都医科大学附属北京天坛医院'''（简称'''北京天坛医院'''），是[[首都医科大学|首都医科大学]]附属的一所以神经科为重点的[[三级甲等医院|三级甲等]]综合医院<ref>{{cite web |url=http://www.bjtth.org/News/AboutUs/Default/View.aspx?id=1194 |title=天坛医院介绍 |publisher=北京天坛医院 |accessdate=2013-08-12 }}{{Dead link|date=2018年7月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref>{{cite web|url=http://www.bjtth.org/News/AboutUs/Default/View.aspx?id=446 | title=天坛医院依法执业注册信息 |publisher=北京天坛医院 |accessdate=2013-08-12}}</ref>。

==简介==
北京天坛医院初建于1930年代，前身是[[中国国民党|国民党]]'''陆军第三十一后方医院'''。1949年2月，由[[中国共产党中央军事委员会|中央军委]]卫生部接管，改为'''中国人民解放军华北军区第二后方医院'''，同年4月改为'''[[华北行政委员会|华北行政委员会]]人民医院'''，同年10月划归[[中央人民政府卫生部|中央人民政府卫生部]]领导，更名'''华北人民医院'''。1953年1月改为'''卫生部中央直属机关第一医院'''。1956年8月由[[中华人民共和国卫生部|中华人民共和国卫生部]]交给北京市公共卫生局管辖，更名为'''北京天坛医院'''。“[[文革|文革]]”期间，1970年2月天坛医院奉命全部迁往[[甘肃省|甘肃省]]。1970年7月，原天坛医院留守人员和[[崇文区|崇文区]]前门医院部分人员，在天坛医院原址成立了'''北京市崇文区崇文医院'''。1975年9月，北京市卫生局将该院划归该局直接领导，改为'''北京崇文医院'''。1978年9月，北京市卫生局将该院交给北京第二医学院，成为'''北京第二医学院附属崇文医院'''。1982年1月7日，经[[北京市人民政府|北京市人民政府]]批准，北京第二医学院附属崇文医院更名为'''北京天坛医院'''。1982年4月1日起，恢复原北京天坛医院名称。1982年4月，北京市神经外科研究所和神经外科医护人员75名自[[首都医科大学宣武医院|宣武医院]]迁至北京天坛医院<ref>{{Cite web |url =  http://www.bjtth.org/Html/News/Articles/100237.html|title =  北京天坛医院历史沿革|author =  |date =  |publisher = 首都医科大学附属北京天坛医院 |language =  |accessdate = 2018-01-24 |quote =  }}</ref>。

1956年8月23日，在北京市党政领导关怀下，[[王忠诚|王忠诚]]等神经外科专家在[[北京同仁医院|北京同仁医院]]建立了神经外科专科。1982年，由宣武医院迁至北京天坛医院。北京天坛医院由此成为中国神经外科医疗、教学、研究、预防的顶尖机构<ref>首都医科大学附属北京天坛医院神经外科简介，首都医科大学学报2005年04期</ref>。北京天坛医院此后也以1956年8月23日作为建院日<ref name=jj/>。早在1975年，为解决颅脑患者住院难、手术难的问题，[[国家计委|国家计委]]和北京市共同决定，将宣武医院院长兼神经外科主任王忠诚及其领导的神经外科从宣武医院分出，利用北京崇文医院的基础，建一所有300张病床的颅脑外科专科医院，国家计委为此批下120万元人民币的专项经费。但到1980年仍未开工建设，国家计委要把专项经费收回。为此，[[中共北京市委|中共北京市委]]卫生体育部部长兼北京市卫生局党组书记和局长的[[张青季|张青季]]派石金祥去当该医院党委书记，抓紧筹建该医院。医院首任院长为王忠诚。经过调研，石金祥等院领导决定不建专科医院，而要建综合性医院，石金祥征得[[北京市人民政府|北京市人民政府]]负责卫生工作的副市长[[白介夫|白介夫]]同意后，又找国家计委副主任[[房维中|房维中]]，房维中表示因国家缺线，三年前批给该院的120万元还要减去20万，只能给该院100万元。1983年房维中鉴于国家经济形势有所好转，表示国家计委可以下拨更多的施工经费。1983年该院列为北京市重点建设工程。1985年卫生部部长[[崔月犁|崔月犁]]来院视察。该院的筹建是边基建施工，边开院医疗，边报批设计，边申请经费。从1980年开始施工到1988年竣工，号称有800张病床，实际上共建成840张病床，其中40张用于急救。1982年该院开始接收颅脑外科患者，1989年试运行一年，1990年正式运营。1988年落成时，国家计委副主任房维中、[[北京市政协|北京市政协]]主席白介夫一起来院查看工程质量。筹建初期，该院一直使用“北京市崇文区崇文医院”名称，1981年3月2日在白介夫主持的市长办公会上，石金祥提出院名问题，白介夫提议定名天坛医院。院名后来由院长王忠诚题写<ref name=shi/>。

1985年之后，该医院更名'''首都医学院附属北京天坛医院'''<ref>宋金福、唐明忠，天坛医院手术室等在工作及消毒后空气中细菌种及量的变化，中国医刊1993(10):17-19</ref>。1994年更名'''首都医科大学附属北京天坛医院'''。该院还是'''首都医科大学第五临床医学院'''。1997年，该院由[[中国医学科学院|中国医学科学院]]与北京市卫生局共建成为'''中国医学科学院北京天坛医院'''和'''中国医学科学院神经科学研究所'''。北京天坛医院和北京市神经外科研究所是世界三大神经外科研究中心之一，是[[亚洲|亚洲]]最大的神经外科临床、科研、教学基地和[[世界卫生组织|世界卫生组织]]在中国的神经科学培训合作中心<ref name=jj>{{cite web|url=http://www.bjtth.org/Html/News/Articles/102888.html | title=北京天坛医院介绍 |publisher=首都医科大学附属北京天坛医院 |date=2015-10-16}}</ref><ref>{{Cite web |url = http://www.huaxia.com/hxjk/yyzn/bj/1080179.html?id=xh=ih |title =首都医科大学附属北京天坛医院  |author =  |date = 2008-08-07 |publisher =华夏经纬网  |language =  |accessdate =  |quote =  }}</ref>。1996年11月被卫生部命名为全国十所文明服务示范医院之一<ref>刘海宁，加强精神文明建设 建好示范医院，中国卫生1999(5)</ref>。

截至2016年，该医院有学科52个，床位1167张，职工2752人，年门急诊量140多万人次；神经学科有2个实验研究中心、19个基础研究室、5个临床研究室、2个防治研究室等科研基础设施；拥有突出贡献专家11名，享受[[政府特殊津贴|政府特殊津贴]]专家30名，正高级专家211名，医生827名，护士1050名。1978年至2015年，该医院共获得省部级以上科研项目808项，省部级科技奖208项，国家科技奖22项，发表[[SCI|SCI]]论文1543篇、中文论文7355篇，著书218部。[[王忠诚|王忠诚]]院士四次获国家科技进步二等奖；1997年9月获[[香港|香港]][[何梁何利基金|何梁何利基金]]科学与技术成就奖；2008年12月获国家科学技术最高奖。[[赵继宗|赵继宗]]院士三次获国家科技进步二等奖<ref name=liushi>{{Cite web |url = http://health.china.com.cn/2016-08/20/content_8976088.htm |title = 北京天坛医院“继往开来，承传人文精神”学术文化论坛暨六十周年院庆活动召开 |author =  |date = 2016-08-20 |publisher =中国网  |language =  |accessdate =  |quote =  }}</ref>。

北京天坛医院原址坐落于[[天坛公园|天坛]]外坛西南部，天坛祈谷坛门内。2018年搬迁前，门牌号是[[北京市|北京市]][[东城区_(北京市)|东城区]]天坛西里6号。2010年，北京天坛医院迁建工程由北京市人民政府批准立项<ref name=wangchen>冯蕾，王晨:出走天坛，中国医院院长2011(6):70-74</ref>。2013年12月26日，北京天坛医院迁建工程在新址[[丰台区|丰台区]]花乡桥东北区域开工建设。这是继2008年地坛医院整体搬迁后北京市又一家市属三级甲等医院整体搬迁工程，也是北京市第一家从中心城区整体搬迁的三级甲等综合医院。新建的北京天坛医院位于丰台区花乡桥东北区域，[[北京四环路|西南四环]]内，总占地面积272.6亩，总建筑面积35.2万平方米，是天坛医院原有建筑面积的3.8倍。整体布局按功能分为A、B、C三个区，其中A区、B区经空中连廊和地下通道连接。A区是主医疗区，包括综合门诊楼、专科门诊楼、住院病房楼、医技楼、急诊抢救楼、感染疾病科楼、康复医学楼等；B区是医疗保健和科研教学区，包括北京市神经外科研究所及行政科研和医疗保健用房；C区是教学宿舍区。新院区一期总床位将达1650张，比天坛医院原有床位增加500张。根据[[北京市人民政府|北京市人民政府]]《2010-2012年促进南城加快发展行动计划》报告，北京市城市南部地区医疗卫生机构数量不到北部地区的三分之一，其中50家三级甲等医院中位于城南的仅11家。北京天坛医院新址建设将加快优质医疗资源从城区向郊区转移，缓解城南医疗资源紧缺局面<ref>{{Cite }}</ref>。新建的北京天坛医院带有多项节能环保措施。该工程作为疏解非首都功能项目被列入北京市重点工程<ref name=wangyi>{{Cite web |url = http://news.163.com/16/0409/13/BK7CDEF600014Q4P.html |title =  天坛医院新院 明年底竣工|author =  |date = 2016-04-09 ==北京市神经外科研究所==
