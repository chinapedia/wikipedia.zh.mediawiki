'''首都門'''（{{lang-ar|'''برج بوابة العاصمة'''}}）位於[[阿拉伯聯合大公國|阿聯酋]]首都[[阿布扎比|阿布扎比]]的摩天大廈；是該市的地標<ref>{{zh-hant}}{{cite web | title=獲世界第一斜塔　"首都門"打敗比薩斜塔  |publisher=[[中天新聞|中天新聞]] |date=2010年6月8日 |accessdate=2010年7月2日 |url=http://video.chinatimes.com/video-cate-cnt.aspx?cid=6&nid=32676 }}</ref>。首都門於2007年開始動工，於2011年年底落成。建築物樓高35層、高160米<ref name="Emporis" />，其傾斜角度達18度，較[[意大利|意大利]][[比薩斜塔|比薩斜塔]]更斜4倍而成為現今[[健力士世界紀錄大全|健力士世界紀錄大全]]最斜的人造塔樓。

==興建過程==
首都門於2007年9月由阿布扎比國家展覽公司開始動工<ref name="apple">{{zh-hant}}{{cite news|publisher=[[蘋果日報|蘋果日報]] |date=2010年6月8日 |title=阿布扎比「首都門」膺全球第一斜塔  }}</ref>；同年11月，開始為首都門進行打[[樁|樁]]工程。2008年4月開始建造核心牆體。2009年2月開始進行建造外牆；同年5月，首都門的建築高度達到{{convert|100|m|ft|link=on}}；首都門的傾斜角度在同年6月已具雛形；首都門並於同年10月15日興建至建築物最終高度的{{convert|160|m|ft|link=on}}<ref>{{en}}{{cite web | title= CAPITAL GATE MAKES IT TO T

</ref>；外部核心結構在同年12月完工，而外牆首階段噴漆也於翌月完成。首都門於2010年2月開始進行內部裝修；翌月開始興建[[橋樑|橋樑]]把首都門和阿布扎比國家展覽中心連接。首都門於同年4月平頂；<ref>{{en}}{{cite web |title=CONSTRUCTION |publisher=首都門官方網站 |accessdate=2010年6月30日 |url=http://www.capitalgate.ae/construction.php |deadurl=yes |archiveurl=https://web.archive.org/web/20100511003819/http://www.capitalgate.ae/construction.php |archivedate=2010年5月11日 }}</ref>，至2011年落成。

==建築特色==
首都門由阿布扎比國家展覽中心設計；11樓及以下樓層均是[[垂直|垂直]]設計；而12樓至35樓向西逐漸傾斜18度，其傾斜角度是[[意大利|意大利]][[比薩斜塔|比薩斜塔]]的四倍<ref name="apple" /><ref name="GWR">{{en}}{{cite web |title=<nowiki>ADNEC’s</nowiki> Capital Gate achieves Guinness World Records™ recognition for <nowiki>‘World’s Furthest Leaning Manmade Tower’</nowiki> |publisher=健力士世界紀錄大全 |date=2010年6月7日 |accessdate=2010年7月2日 |url=http://community.guinnessworldrecords.com/_ADNECs-Capital-Gate-achieves-Guinness-World-Records-recognition-for-Worlds-Furthest-Leaning-Manmade-Tower/blog/2381779/7691.html |deadurl=yes |archiveurl=https://web.archive.org/web/20100613002954/http://community.guinnessworldrecords.com/_ADNECs-Capital-Gate-achieves-Guinness-World-Records-recognition-for-Worlds-Furthest-Leaning-Manmade-Tower/BLOG/2381779/7691.html |archivedate=2010年6月13日 }}</ref>；惟首都門的傾斜角度是刻意設計而成，和比薩斜塔因地基流失而呈現傾斜有所不同<ref>{{zh-hant}}{{cite news|publisher=[[東方日報|東方日報]] |date=2010年6月7日 |title=阿布扎比首都門成全球最斜人工塔 }}</ref>。另一方面，除了首都門外牆由不同弧度的728塊[[菱形|菱形]]玻璃而成之外，首都門展現斜肋構架的不對稱對角線結構以用作支撐首都門傾斜角度而導致的外部負荷<ref name="apple" />

==用途==
首都門主要用作[[酒店|酒店]]及商用辦公室用途；寫字樓部分佔地{{convert|20900|sqm|sqft|link=on}}；而酒店則佔地{{convert|25050|sqm|sqft|link=on}}<ref>{{en}}{{cite web |title=FACT FILE |publisher=首都門 |accessdate=2010年7月2日 |url=http://www.capitalgate.ae/factfile.php |deadurl=yes |archiveurl=https://web.archive.org/web/20100602005409/http://www.capitalgate.ae/factfile.php |archivedate=2010年6月2日 }}</ref>。酒店部份位於首都門18樓至35樓，是為[[凱悅酒店集團|凱悅酒店集團]]旗下首間位於[[阿拉伯聯合大公國|阿聯酋]]首都[[阿布達比|阿布達比]]的五星級酒店，名為'''首都門凱悅酒店'''（{{lang-en|Hyatt Capital Gate}}）<ref name="GWR" />。酒店將於2011年年底開幕，並提供189間酒店客房；其酒店大堂設於首都門18樓，而水療和健身中心則位於大樓的19樓<ref>{{en}}{{cite web |title=5-star hotel, Hyatt at Capital Centre |publisher=首都門 |accessdate=2010年7月2日 |url=http://www.capitalgate.ae/hotel.php |deadurl=yes |archiveurl=https://web.archive.org/web/20100601144004/http://www.capitalgate.ae/hotel.php |archivedate=2010年6月1日 }}</ref>。

==世界紀錄==
2010年1月，首都門外部建築完成之時，[[健力士世界紀錄大全|健力士世界紀錄大全]]派出委員到現場為建築進行評估。同年6月，[[健力士世界紀錄大全|健力士世界紀錄大全]]因首都門傾斜角度為向[[西|西]]18度，較在[[意大利|意大利]][[比薩斜塔|比薩斜塔]]傾斜角度多出四倍<ref>{{en}}{{cite web | title=http://www.adnec.ae/group/capital-gate.php |publisher=阿布扎比國家展覽公司 |accessdate=2010年7月2日 |url=http://www.adnec.ae/group/capital-gate.php }}</ref>；因而把首都門認證為「世界最遠傾向的人造塔」<ref>{{zh-hant}}{{cite news|publisher=MSN 新聞頻道  |date=2010年6月7日 |accessdate=2010年7月2日  |title=比薩斜塔輸了！　阿布達比「首都門」全球最斜 |url=http://news.msn.com.tw/news1733533.aspx }}</ref><ref>{{en}}{{cite web |title=ADNEC's Capital Gate achieves Guinness record for <nowiki>‘World’s</nowiki> Furthest Leaning Manmade Tower |publisher=ADNEC |date=2010年6月5日 |accessdate=2010年7月2日 |url=http://www.capitalgate.ae/images/pdf/guinness_record_english.pdf |deadurl=yes |archiveurl=https://web.archive.org/web/20100704222530/http://www.capitalgate.ae/images/pdf/guinness_record_english.pdf |archivedate=2010年7月4日 }}</ref><ref>{{en}}{{cite web|title=Capital Gate Tower in Abu Dhabi certified world's furthest leaning manmade tower |url=http://www.worldinteriordesignnetwork.com/news/capital_gate_tower_in_abu_dhabi_certified_worlds_furthest_leaning_manmade_tower_100607/ |publisher=World Interior Design Network |date=2010年6月7日 |accessdate=2010年7月2日 |deadurl=yes |archiveurl=https://web.archive.org/web/20100611072956/http://www.worldinteriordesignnetwork.com/news/capital_gate_tower_in_abu_dhabi_certified_worlds_furthest_leaning_manmade_tower_100607/ |archivedate=2010年6月11日 }}</ref><ref>{{en}}{{cite web|author=Mail Foreign Service |url=http://www.dailymail.co.uk/news/worldnews/article-1285590/Abu-Dhabis-Capital-Gate-leans-nearly-times-Tower-Pisa-claim-world-record.html |title=Abu Dhabi's Capital Gate 'leans nearly five times more' than Tower of Pisa to claim world record |publisher=Mail Online |date=2010年6月10日 |accessdate=2010年7月2日 }}</ref>。

首都門是採用「[[預先修弧核心|預先修弧核心]]」（pre-cambered core）設計以承受建築物的18度傾斜角度而所產生的中心壓力問題；這項技術利用了{{convert|15000|sqm|sqft|link=on}}的[[混凝土|混凝土]]及和一萬噸[[鋼材|鋼材]]，使建築物的核心稍微偏離建築中心；混凝土會在首都門越建越高時被壓緊以增加強度；建築物也會因樓層重量增加時移向垂直定位。首都門動用了490支[[樁|樁]]柱，每支樁柱進深地面超過{{convert|30|m|ft|link=on}}<ref name="GWR" />。
