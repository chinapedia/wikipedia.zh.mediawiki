'''首都圈电铁中央線'''（{{kor|諺=수도권 전철 중앙선|漢=首都圈 電鐵 中央線|羅=Sudogwon jeoncheol jungang seon|f=y}}）是[[大韓民國|大韓民國]][[首都圈|首都圈]]的一條電氣化鐵路，行走於[[首爾|首爾特別市]][[龍山區_(首爾)|龍山區]][[龍山站_(首爾)|龍山站]]所屬之[[京元線|京元本線]]及[[中央線_(韓國)|中央本線]]，至[[京畿道|京畿道]][[楊平郡|楊平郡]][[龍門站_(京畿道)|龍門站]]。由1978年起分階段提升為電氣化[[複線鐵路|複線鐵路]]，至2009年全線提升完畢。現為韓國首都圈電鐵的一部分，由[[韓國鐵道公社|韓國鐵道公社]]營運，路線顏色以翡翠綠標示。

== 概要 ==
現今首都圈電鐵中央線由[[龍山站__(首爾)|龍山站]]到[[回基站|回基站]]的一段，原是[[京元線|京元本線]]的一部分，是由龍山往[[光雲大站|城北]]<ref>{{cite web | title=韓國旅遊官方網站：城北站更名為光雲大學站 | date=2013-02-25 | url=http://www.visitkorea.or.kr/cht/MA/MA_CH_9_4.jsp?cid=1798337 | language=zh | accessdate=2013-09-13 | archive-url=https://web.archive.org/web/20130928173802/http://www.visitkorea.or.kr/cht/MA/MA_CH_9_4.jsp?cid=1798337 | archive-date=2013-09-28 | dead-url=yes }}</ref>所經過的路段。電氣化[[複線鐵路|複線鐵路]]提升工程展開後，1978年完成龍山至[[清涼里站|清涼里]]路段，到2005年12月16日完成清涼里至[[德沼站|德沼]]的路段後，由龍山經清涼里往德沼的電氣化鐵道排除了往城北一段，成為一條完整區域電鐵路線，正式名為「中央線」，全名為「首都圈電鐵中央線」，<ref>{{cite web | title=首都圏電鉄中央線 | url=http://ko.wikipedia.org/wiki/수도권_전철_중앙선 |language=ko |accessdate=2013-09-25}}</ref>納入[[韓國首都圈電鐵|韓國首都圈電鐵]]之內，有別於行走首爾清涼里至慶州的廣域鐵道「中央本線」。<ref name="jungang_line">韓國事務部 / 管轄部指引，韓國事務部2010-439號公告《韓國鐵道距離表》有關[[:ko:대한민국_국토해양부|國土海洋部]]與  [[:ko:한국철도공사|韓國鐵道公社]]運作的「中央本線」和「中央線」的正式名稱：中央本線和支線的各種說法。（原：국토해양부 고시 제2010-439호 《한국철도거리표》, 국토해양부. / 관할 부처인 [[:ko:대한민국_국토해양부|대한민국 국토해양부]]와 운영 주체인 [[:ko:한국철도공사|한국철도공사]]의 정식 명칭은 '''중앙본선'''이고 ‘중앙선’은 중앙본선과 그 지선을 통틀어 가리키는 말이다.）</ref> 

電氣化複線鐵路工程其後陸續完成：2007年12月完成了德沼至[[八堂站|八堂]]的路段、2008年12月完成八堂至[[菊秀站|菊秀]]一段，至2009年12月完成菊秀至[[龍門站_(京畿道)|龍門]]路段後，由龍山至[[京畿道|京畿道]][[楊平郡|楊平郡]]龍門站的電氣複線鐵道已完全完成。2010年12月[[上鳳站|上鳳站]]及[[梧濱站|梧濱站]]亦啟用，中央線由龍山至龍門，全線共有28站。

在鐵路連接方面，除了龍山至清涼里的一段京元本線，可接駁多條區域電鐵線<ref>包括1至5號線、盆唐線、新盆唐線及京義線，詳見本文「車站列表」</ref>及廣域鐵道<ref>包括國鐵京釜線、京元本線、中央本線，及湖南高速線及ITX-青春高速列車，詳見本文「車站列表」。</ref>線外，經[[京春線|京春線]]往[[江原道|江原道]][[春川市|春川市]]各站的可於上鳳站或[[忘憂站|忘憂站]]轉乘。另外由龍山站有[[ITX-青春|ITX-青春]]城際高速列，停靠中央線[[玉水站|玉水]]、[[往十里站|往十里]]、清涼里及上鳳站，經京春線往春川沿線各站。另外上鳳站共亦可轉乘[[首爾地鐵7號線|7號線]]，九里站預計2017年可轉乘[[首爾地鐵8號線|8號線]]。

當2014年龍山站與[[京義線|京義線]]連接後，即成一條全長145.3公里，連接京春線、中央線，及京義線，由東面春川市往西北[[汶山站|汶山]]的直通路線。<ref name="Seoulmetro">{{cite web |title=路程計算自Seoul Metro Traffic Center |url=http://dmzap1.seoulmetro.co.kr/station/eng/linemap.action |publisher=Seoul Metro |language=en |accessdate=2013-09-14 }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

== 历史 ==
*1978年12月9日 - [[龍山站_(首爾)|龍山站]]至[[清凉里站|清凉里站]]12.7公里路段以電氣化[[複線鐵路|複線]]運作。
*1980年4月1日 - [[西冰庫站|西冰庫站]]、[[漢南站|漢南站]]及[[回基站|回基站]]啟用。
*1985年10月18日 - 配合[[首爾地鐵3號線|3號線]]轉乘，[[玉水站_(首爾)|玉水站]]開始啟用。
*2001年7月1日 - 玉水站遷移工程完成。
*2005年12月16日 - [[清凉里站|清凉里站]]至[[德沼站|德沼站]]17.2公里路段以電氣化複線運作。
*2007年12月27日 - 徳沼站至[[八堂站|八堂站]]5.9公里路段以電氣化複線運作。
*2008年4月1日 - 列車開始以單人操作。
*2008年12月1日 - 特快列車開始運行。
*2008年12月29日 - 八堂站至[[菊秀站|菊秀站]]15.5公里路段以電氣化複線運作。
*2009年12月23日 - 菊秀站至[[龍門站_(京畿道)|龍門站]]15.5公里路段以電氣化複線運作。
*2010年12月21日 - [[上鳳站|上鳳站]]及[[梧濱站|梧濱站]]啟用。上鳳站至[[忘憂站|忘憂站]]0.6公里路段以電氣化複線運作，兩站亦增加[[京春線|京春線]]轉乘。
*2014年12月 - 連接[[京義線|京義線]]，與中央線直通運輸。

== 運行情況 ==
中央線行走[[龍山站_(首爾)|龍山站]](K110)至[[龍門站_(京畿道)|龍門站]](K137)，全長71.2公里。<ref name="Seoulmetro" />按實際路線畫分，中央線是由龍山站至[[清凉里站|清凉里站]]的部分「[[京元線|京元本線]]」<ref>{{cite web |title=京元本線 |url=http://ko.wikipedia.org/wiki/경원선 |publisher=wikipedia |language=ko |accessdate=2013-09-28}}</ref>，加上由清凉里站至龍門站的部分「[[中央線_(韓國)|中央本線]]」<ref name="jungang_line"/>所組成。（現運作於[[大韓民國|南韓]]的「[[京元線|京元本線]]」，全線由龍山經清凉里站至[[白馬高地站|白馬高地]]，而「[[中央線_(韓國)|中央本線]]」全線由清凉里經龍門站後繼續前往慶州<ref>{{ cite web |title=慶州站 |url=http://ko.wikipedia.org/wiki/경주역 |publisher=wikipedia |language=ko |accessdate=2013-09-28}}</ref>）。

由龍山往返龍門之間的列車，每天由05:00~20:00之間，平均每小時14班。<ref name="Seoulmetro" />行走中央線的部分列車經[[清凉里站|清凉里]]後，繼續行走[[光雲大站|光雲大學]]、[[議政府站|議政府]]、[[逍遥山站|逍遥山]]及其餘「京元本線」各站，而清凉里至逍遥山亦屬於[[首爾地鐵1號線|1號線]]的運行路線。部分列車在到達龍門站後，會繼續行走「中央本線」往慶州方向。

同樣行走部分中央線的，還有經龍山、玉水、往十里及清涼里，然後接[[京春線|京春線]]往[[春川市|春川市]]的[[ITX-青春|ITX-青春]]城際高速列車 ，每天各有10班從龍山及[[春川站|春川]]開出。<ref>{{cite web |url=http://www.korail.com |language=ko |title=韓國鐵道公社 |accessdate=2013-09-28}}</ref>

=== 乘載量變動 ===
以下是2005年至2006年中央線通車後，中央線運作路段所屬的京元本線與中央本線，總乘客數量與各站平均乘客的比較，包含2000年至2004年，中央線未通車前該段京元本線人數。<ref>統計數字取自「首都圈電鐵中央線」韓語維基條目（[[:ko:수도권_전철_중앙선|수도권 전철 중앙선]]），重新製表。</ref>可看出在首年通車後，京元本線與中央本線兩段區間的平均乘客量祇相差約7%，這表示較少乘客經清涼里後，換乘往光雲大學等方向；或是大多數乘客由德沼到清涼里後，轉乘往光雲大學等方向的列車。<ref>根據2006年京元本線及中央本線各站平均人數比較</ref>而清涼里至德沼的高乘客量，亦為2006年後中央線電氣化複線鐵路延伸至八德及龍門提供了支持數據。
