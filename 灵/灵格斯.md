'''靈格斯翻譯家'''（Lingoes）是一套[[免費軟體|免費]]的桌面[[字典|字典]]/[[詞典|詞典]]軟體，该軟體的首個版本於2006年9月1日發佈。软件同时发布有不带词典文件及预带基础词典文件的版本，使用者还可以自行下载挂载其他附加词典。

2009年3月16日，灵格斯软件曾被[[美国国际开发署|美国国际开发署]]向[[埃及|埃及]]的学校推荐使用。<ref>[http://www.cnbeta.com/articles/81694.htm Cnbeta:美国国际开发署向埃及275所学校推荐灵格斯]</ref>

== 特色 ==
靈格斯翻譯家最初版本的主程序大小只有1.5MB，其2.7.6.2版主程序也仅有5MB，還內置了一部實用的英漢詞典。在推出2.4.2Beta版同時更發布了官方綠色便攜版2.4.2 Portable，并同步更新至今。軟體包含了多語種詞典支援，多語言多向翻譯，詞語索引列表，[[在线词典|联机词典]]，[[鼠标|鼠标]]螢幕取詞，[[劃詞翻譯|劃詞翻譯]]，[[剪貼簿|剪貼簿]]取詞，支援[[真人語音|真人語音]]庫，[[語音合成|語音合成]]引擎支援，[[全文翻譯|全文翻譯]]（智能識別語種）和[[熱鍵|熱鍵]]控制自定義等功能。通過詞典檔的配合，可以查詢翻譯全世界包括[[英語|英語]]、[[法語|法語]]、[[德語|德語]]、[[俄語|俄語]]、[[西班牙語|西班牙語]]、[[葡萄牙語|葡萄牙語]]、[[意大利语|意大利语]]、[[简体中文|简体中文]]、[[繁体中文|繁体中文]]、[[日語|日語]]、[[韓語|韓語]]、[[越南語|越南語]]、[[瑞典語|瑞典語]]、[[阿拉伯语|阿拉伯语]]、[[土耳其語|土耳其語]]、[[波兰语|波兰语]]、及[[世界語|世界語]]等80多種語言文字。

靈格斯用户可以自行制作词典源文件（[[文本文件|文本文件]]格式），上传到灵格斯官网，经管理员编译为ld2格式后讓其他用户下载使用。靈格斯用户已经上传了许多優秀的詞典，如《[[牛津英語詞典|牛津高階英語詞典]]》、《牛津高階英漢雙解詞典》、《朗文當代英語詞典》、《柯林斯高階英語學習者詞典》、《現代英漢詞典》、《英文同義詞反義詞辭典》、《高級漢語詞典》、《中文同義詞反義詞辭典》。但作者所使用的词典编译器目前久久没有公开发布。<ref>{{Cite web |url=http://www.lingoes.cn/zh/dictionary/dict_format.php |title= Lingoes 词典源文件格式说明 |accessdate=2013-03-13 }}</ref>

灵格斯与[[互动百科|互动百科]]合作创建了“互动百科联机词典”，内置在中文版灵格斯词霸中。此外。靈格斯翻譯家在發布2.5版同時也發布了[[英文維基百科|英文維基百科]]的離線詞典。

== 評價 ==
由於主程序體積小，佔用系統資源少，多語言詞典，多向翻譯，界面美觀，開放式詞典管理，免費使用等特色，靈格斯翻譯家一問世就受到了廣泛關注，對[[中國大陸|中國大陸]]詞典翻譯軟體市場份額佔領最大的[[金山詞霸|金山詞霸]]起了不小的衝擊。

除了語言翻譯，靈格斯還具有類似網絡搜索的功能。通過內置的各種詞典、百科全書和網絡釋義等工具，靈格斯可以提供比網絡搜索更為精確的解釋功能。

然而，主要批評在於靈格斯詞霸無法提供版權詞典的「合法」引用證明，受到了部分较为重視[[版權|版權]]問題的用戶的批評。

同[[星際譯王|星際譯王]]一样，官方網站不少字典檔并没有得到官方授权，其中包括牛津高阶英语词典（第6版）、牛津学生英语搭配词典、牛津高阶英汉双解词典、美国传统词典（第4版）、美国传统词典（英汉双解）等在内的大量字典均存在版权问题。
