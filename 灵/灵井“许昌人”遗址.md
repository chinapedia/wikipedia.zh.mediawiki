'''灵井“许昌人”遗址'''是1965年发现的位于[[河南|河南]][[许昌县|许昌县]][[灵井镇|灵井镇]]的一个“[[现代人|现代人]]”考古遗址<ref name="遗址">{{cite web |url=http://www.hawh.cn/zyfg/2010-06/13/content_88151.htm |language= |title=灵井“许昌人”遗址 |publisher=河南文化网 |date=2010-06-13 |accessdate= |archiveurl=https://web.archive.org/web/20150610090149/http://www.hawh.cn/zyfg/2010-06/13/content_88151.htm |archivedate=2015-06-10 |deadurl=yes }}</ref>。2008年成为河南省级文物保护单位。2013年入选第七批[[全国重点文物保护单位|全国重点文物保护单位]]。<ref name="证据">{{cite web|url=http://henan.qq.com/a/20140509/010885.htm | language= | title=许昌人”2号头骨化石出土 或是古人食脑证据 |publisher=大河报 |date=2014-05-09  |accessdate= }}</ref>

== 发掘 ==
2005年河南省文物考古研究所第一研究室首次进行考古发掘，之后进行数度发掘。文化层分上、下文化层，深7.2–5.6米。上文化层出土细石器、牙制工具等。下文化层出土18种[[动物|动物]][[化石|化石]]，以及石器、石片、断块等石器类型。原料主要有[[脉石英|脉石英]]和[[燧石|燧石]]，还发现有大量[[骨器|骨器]]。<ref name="遗址"/>

2014年4月，[[河南省文物考古研究院|河南省文物考古研究院]]在灵井许昌人遗址9号探方西部约3平方米的范围内，新出土了包括有完整的枕骨、部分顶骨、眉脊、面骨和颅底骨等共计27块古人类头骨化石断块。至今对该遗址已发掘面积近450平方米，完成发掘面积仅120平方米，而整个遗址面积可达三万平米以上。<ref>{{cite web |url=http://zy.takungpao.com/2014/0507/123465.html |language= |title=灵井许昌人遗址新发现人类头骨化石 |publisher=民主与法制网 |date=2014-05-07 |accessdate= |deadurl=yes |archiveurl=https://web.archive.org/web/20160304074028/http://zy.takungpao.com/2014/0507/123465.html |archivedate=2016-03-04 }}</ref>

== 意义 ==
该遗址具有很大的研究价值，其中2007年12月在T9深4.99米处发现的距今8-10万年的人类[[头盖骨|头盖骨]]化石，被学术界认为是研究中国现代人类起源的重大发现，其考古发现被评为“2007年全国十大考古新发现”。<ref>{{cite web |url=http://weijinwenhua.xcu.edu.cn/news_view.asp?newsid=159 |language= |title=灵井许昌人遗址的发现 |publisher=魏晋文化研究所 |date=2010-04-24 |accessdate= |deadurl=yes |archiveurl=https://web.archive.org/web/20130522204623/http://weijinwenhua.xcu.edu.cn/news_view.asp?newsid=159 |archivedate=2013-05-22 }}</ref>
[[细石器|细石器]]的发现为研究[[东北亚|东北亚]]细石器的传播与发展十分重要。<ref name="遗址"/>
