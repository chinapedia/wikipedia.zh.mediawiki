梅州'''灵光寺'''位于广东省[[梅州市|梅州市]][[梅县|梅县]]雁洋镇阴那村[[阴那山|阴那山]]五指峰，原名圣寿寺，为省级文物保护单位、[[国家4A级景区|国家4A级景区]]，是[[广东四大名寺|广东四大名寺]]之一。寺大门前挺立两棵古柏，一生一死，名生死树。[[文化大革命|文化大革命]]期间，灵光寺遭到严重破坏，[[改革开放|改革开放]]后，1978年7月18日公布为[[广东省文物保护单位|广东省文物保护单位]]。<ref name=":1" /><ref name=":0">{{Cite web|url = http://www.mzta.gov.cn/info/you/jq/2011/1114/2049.html|title = 灵光寺旅游区|accessdate = |author = |date = 2015-06-25|publisher = 梅州旅游网}}</ref>

== 历史 ==
寺始建于唐朝[[咸通|咸通]]二年（公元861年），为高僧[[潘了拳|潘了拳]]{{注释|1=自号惭愧，福建省延平府沙县人}}创建，原名圣寿寺，明朝洪武十八年（1385年），广东[[监察御史|监察御史]]梅鼎捐钱扩建，易名“灵光寺”，并亲书“灵光寺”三字嵌刻在山门殿的门额上<ref>{{Cite web|url=http://www.meizhou.cn/news/1005/21/10052100005.html|title=千年古刹灵光寺|accessdate=|author=|date=2010-05-21|publisher=梅州网}}</ref>。

明朝某年曾发生多起[[崇祯太子案|崇祯太子案]]，灵光寺曾隐藏[[崇祯帝|崇祯帝]]的真正太子[[朱慈烺|朱慈烺]]，慈烺太子在灵光寺出家，法号奯{{注释|1=奯：音货（huo），自创字。}}山和尚，死后寺中曾长期供有他的神位，称为“稗子菩萨”音“太子菩萨”<ref>{{Cite web|url=http://www.meizhou.cn/news/0507/01/050701088.html|title=崇祯太子慈火良与灵光寺|accessdate=|author=|date=2005-07-01|publisher=梅州网}}</ref><ref>{{Cite web|url=http://www.meizhou.cn/news/1103/25/11032500063.html|title=明太子隐身灵光寺|accessdate=|author=|date=2011-03-25|publisher=梅州网}}</ref>。

== 布局 ==
灵光寺坐东向西，占地面积4290平方米，建筑面积4629平方米。寺庙由佛殿、天王殿（山门殿）、罗汉殿、诸天殿、钟楼、鼓楼、经堂、斋堂、大悲阁、太史馆、半日闲等组成。寺门前有唐高僧[[潘了拳|潘了拳]]手植的两株千年古柏，一枯一荣，俗称“生死树”，枯者已300余年。<ref name=":1">{{Cite web|url=http://www.meizhou.cn/news/0812/19/081219088.html|title=千年古刹灵光寺|accessdate=|author=|date=2008-12-19|publisher=梅州网}}</ref>
