'''灵山岛'''，古称'''灵山'''、'''水灵山'''、'''水灵山岛'''，是[[中华人民共和国|中国]][[黄海|黄海]]西北部近岸的一座[[海岛|海岛]]，由[[青岛市|青岛市]][[黄岛区|黄岛区]][[灵山卫街道|灵山卫街道]]管辖。

== 名称由来 ==
据《灵山卫志》载：“其南一望无际者海也，有山涌出其间，伊若翠屏天开者灵山也。”又据《名胜志》载：“未雨而云，先日而曙，若有灵焉”，故名灵山岛。{{参|{{Cite web|url = http://club.qingdaonews.com/showAnnounce.php?topic_id=3874091&board_id=131|title = 沧桑古城灵山卫|accessdate = 2016-01-30|author = |date = |publisher = 青岛新闻网}}}}

== 历史 ==
灵山岛自古以来便是渔船避风之所。在[[战国|战国]]时期就已成为[[齐国|齐国]]的海上运输咽喉。在1980年代初期，灵山岛西部的沙嘴子村就出土了[[西周|西周]]时期的[[鼎|铜鼎]]，铜锛和陶垒等文物。《胶州志》记载，灵山岛“多竹木，居民数十家，鸡犬桑麻，以樵苏为生。”[[明朝|明朝]]初年设立[[灵山卫_(古代)|灵山卫]]（现[[灵山卫街道|灵山卫街道]]所在地），灵山岛则在其管辖范围之内，一些来自[[四川省|四川省]]的[[王姓|王姓]]移民也在同一时期迁居于此。

== 地理 ==
灵山岛最高峰歪头顶[[海拔|海拔]]513.6米，是[[中国北方|中国北方]]第一高岛，仅次于[[台湾岛|台湾岛]]与[[海南岛|海南岛]]。岛上山峰四面陡峭，不易攀登，只有岛西一面地势较为倾斜。在城子口以东，群峰高耸，丛山林立，有前阳山、象鼻山、望海楼山等山峰。曾被誉为[[胶州直隶州|胶州]]八景之一——“灵岛浮翠”。当地有“灵山岛戴帽，大雨三两天到。”故对岛民及对岸百姓来说，只要看到灵山岛上云雾罩顶，即知此为降雨先兆。

== 交通 ==
岛上有环岛公路，对外唯一交通方式为水路运输。渡船一般从积米崖码头到达灵山岛城子口码头，平日每天一班，往返票价50元[[人民币|人民币]]。
