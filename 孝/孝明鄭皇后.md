'''孝明皇后'''（{{bd|8世纪|?|865年|12月26日|catIdx=Z}}<ref>另說卒於860年12月26日</ref>），[[郑姓|姓鄭氏]]，一说为[[尔朱氏|尔朱氏]]，名不详 , [[中國|中國]][[唐朝|唐朝]]時期[[皇族|皇族]]女性，[[唐宣宗|唐宣宗]]、[[安平公主_(唐憲宗)|安平公主]]生母。

== 早年生涯 ==

鄭氏生年未有记载，[[籍贯|籍贯]][[润州_(隋朝)|潤州]][[丹楊縣|丹楊縣]]。在[[元和_(唐朝)|元和]]初年[[李錡|李錡]]謀反時，郑氏及笄的时候，有善面相之人對李錡說，鄭氏將來必當生下天子，李錡大喜而將鄭氏納為妾室。後來李錡被殺，鄭氏以叛臣眷屬的身分被没入[[掖庭宫|掖廷]]，充為寵冠六宮[[懿安皇后|郭貴妃]]的[[宮女|宮女]]，宪宗在贵妃宫里出入时看见郑氏姿容绝豔，便召幸了她，在[[元和_(唐朝)|元和]]五年（810年）六月二十二日，於[[大明宮|大明宮]]生下憲宗第十三子[[唐宣宗|李忱]]，又生下憲宗第十四女[[安平公主_(唐憲宗)|安平公主]]。此后，郑氏是否获得正式的[[妃嫔|妃嫔]]封号，史书无载。憲宗駕崩後，李忱被封為光[[王|王]]，鄭氏亦成為光王[[太妃|太妃]]。

唐憲宗駕崩後，由鄭氏當年侍奉的郭貴妃所生的遂王李恆登基，是為[[唐穆宗|唐穆宗]]，往後又歷經[[唐敬宗|敬宗]]、[[唐文宗|文宗]]、[[唐武宗|武宗]]等三帝，自憲宗駕崩的元和十五年（820年）算起，直到[[会昌_(年号)|會昌]]六年（846年）為止，在這二十六年間鄭氏一直都只是個沒沒無聞、數朝前的王太妃。

== 一躍皇太后 ==

[[唐武宗|唐武宗]]在位六年，為求長生不老而服用方士所貢[[金丹|鉛丹]]，最終導致中毒身亡、英年暴卒。武宗崩時雖遺有五子，皆可嗣其位；惟宦官干預朝政，擁立武宗庶叔[[唐宣宗|光王李怡]]即位，是為[[唐宣宗|唐宣宗]]。光王太妃鄭氏也因此一躍成為[[皇太后|皇太后]]。據[[舊唐書|舊唐書]]記載，在[[会昌_(年号)|會昌]]六年，鄭氏之弟[[鄭光|鄭光]]曾夢見一輛車載著日月，光耀天下，被認為是吉兆，不久，宣宗登基，鄭氏成為皇太后，鄭光也獲得升遷，任官檢校戶部尚書、諸衛將軍、平盧節度使等等。

宣宗對於生母鄭太后孝養至誠，此時宮中除鄭太后以外，還有兩位太后：一是文宗生母[[貞獻皇后|蕭太后]]；另一位是鄭氏曾經服侍過的[[懿安皇后|郭太皇太后]]。郭氏在其子穆宗登基後成為皇太后，孫敬宗登基後又成為[[太皇太后|太皇太后]]。

郭氏自被穆宗尊為皇太后起，一直以來都是[[唐朝|唐朝]]皇室的母儀代表，不論郭氏與每位皇帝之間親疏與否，郭氏一直都受到每位皇帝的尊敬與孝養，即使敬宗、文宗各自尊生母為皇太后，對於郭氏仍孝養依然。但在宣宗登基後，因鄭太后曾為郭太皇太后當年的宮女，因此兩人之間素有舊怨，連帶使宣宗對郭太皇太后有所疏遠，這使郭太皇太后大為不滿，甚至曾欲跳樓自盡，雖後來郭太皇太后被左右攔阻並未墜樓，但她當晚即去世。

郭太皇太后去世後，宣宗因為鄭太后之故，不願讓她祔憲宗[[太廟|太廟]]，即使曾有禮官力爭，宣宗仍不讓郭太皇太后祔廟。

== 太皇太后時期 ==
宣宗在位十三年後因服食丹藥中毒駕崩，其子李漼登基，是為[[唐懿宗|唐懿宗]]，而鄭太后也以懿宗祖母身份成為[[太皇太后|太皇太后]]，這也是唐朝第二與最後一位太皇太后。

對於鄭太皇太后的薨辰，[[新唐書|新唐書]]與[[舊唐書|舊唐書]]出現分歧：新唐書記載，在[[咸通|咸通]]三年（862年）懿宗曾在三殿侍宴鄭太后，三年後鄭太后逝世；至於舊唐書則是記載鄭太皇太后於[[大中_(唐朝)|大中]]末年逝世。不論如何，鄭氏在當時算來是相當長壽。鄭氏死後葬於唐憲宗的[[唐景陵|景陵]]旁園，[[諡號|諡號]]'''孝明皇后'''。

== 親屬 ==
弟弟[[鄭光|鄭光]]，其子[[鄭漢卿|鄭漢卿]]為終義昌軍節度使，[[郑汉璋|郑汉璋]]官至兖州节度使；侄女[[郑德柔|郑德柔]]，嫁[[齐州|齐州]]司马[[冯翊|冯翊]]人鱼某。

== 電視影劇 ==
在2009年[[香港電視廣播有限公司|香港電視廣播有限公司]]所拍攝的[[古裝|古裝]][[電視劇|電視劇]]'''[[宮心計|宮心計]]'''中，由[[韓馬利|韓馬利]]飾演鄭氏一角。劇中鄭氏與[[懿安皇后|郭氏]]為敵對關係，郭氏屢次提及鄭氏當年的宮女身分時，是以「洗腳婢」一詞來奚落鄭氏，但在史書中並沒有記載鄭氏的宮女職責；劇中所演繹的鄭氏，和史實中很大的出入是，鄭氏在唐宣宗年間就被毒殺，但實際上，鄭氏一直活到唐懿宗年間，並成為太皇太后。
