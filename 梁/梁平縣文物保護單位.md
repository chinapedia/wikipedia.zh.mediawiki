== 梁平縣第三批縣級文物保護單位（21處）（2011年9月7日）<ref>[http://lp.cq.gov.cn/zfxx/zfwj/11/09/13/18724.html 梁平縣人民政府關於公布第三批縣級文物保護單位的通知（梁平府發〔2011〕61號）]{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }} 梁平縣人民政府關於公布第三批縣級文物保護單位的通知（梁平府發〔2011〕61號） 各鄉鎮人民政府（街道辦事處）、縣級各部門： 我縣歷史悠久，文化底蘊深厚，梁平人民在漫長的歷史長河中，用自己的雙手創造了光輝燦爛的歷史文化。根據《中華人民共和國文物保護法》等法律法規規定，結合我縣文物保護現狀，確定了21處文物價值較高的文物點作為第三批縣級文物保護單位，現予以公布。 各級有關部門、單位要認真貫徹落實「保護為主、搶救第一、合理利用、加強管理」和「保護文化遺產，發展文化事業」的方針，正確處理文物保護與利用的關係，切實加強文物保護單位的保護管理，充分利用文物資源進行愛國主義和革命傳統教育，促進文物保護事業健康發展。 二○一一年九月七日</ref> ==
