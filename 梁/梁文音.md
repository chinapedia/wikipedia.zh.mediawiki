'''梁文音'''（[[魯凱語|魯凱語]]：{{lang|dru|'''Leheane Palray'''}}，{{bd|1987年|4月8日|||catIdx=Liang梁}}），[[臺灣|臺灣]][[華語流行音樂|華語流行音樂]][[女歌手|女歌手]]，出生於[[高雄市|高雄市]]，[[和春技術學院|和春技術學院]]五專部應用外語科英文組畢業、[[國立雲林科技大學|國立雲林科技大學]]二技部應用外語系肄業（因演藝事業而辦理正式休學）。<ref>{{cite news|language=zh-hant|author=記者顏妍|url=http://www.epochtimes.com/b5/8/12/12/n2359953.htm|title=傑出休學生梁文音 回校賣專輯氣氛high|publisher=NOWnews【WEnews】|date=2009-03-13|accessdate =2010-02-22}}</ref> [[臺灣原住民|臺灣原住民]][[魯凱族|魯凱族]]和[[泰雅族|泰雅族]][[混血儿|混血儿]]，參與[[超級星光大道_(第二屆)|第二屆《超級星光大道》]]歌唱選秀節目，於2008年1月18日總決賽中以第二名「亞軍」佳績出道。曾在電影《[[海角七號|海角七號]]》裏飾演60年前的[[小島友子|小島友子]]。2009年入圍[[第20屆金曲獎|第20屆金曲獎]]最佳新人獎，[[第十五屆新加坡金曲獎|第十五屆新加坡金曲獎]]榮獲最受歡迎新人獎和最受歡迎女歌手獎。<ref>{{cite news|title=梁文音專訪：天籟之音、天使心腸|publisher=基督日報|url=http://www.gospelherald.com.hk/news/cul-2793|date=2013-12-11|author=|accessdate=2015-08-30|language=zh-hant}}</ref>

==生平==
梁文音在她10多歲的時候，父母親因故相繼離世。13歲的她與弟弟两人住進了[[六龜山地育幼院|六龜山地育幼院]]，而當時年級尚小的姊姊也辛苦在外邊工作邊讀書。在住進育幼院6到7年的這段時間，是梁文音人生的分水嶺。梁文音學習敞開心胸去接納別人給她的愛與關懷，也在信仰上有了堅強的依靠。梁文音說自己擁有一個「暢快音符的童年」，從小在音符下長大的她，對歌唱非常地熱心，非常認真看待每次演出的機會，用歌聲表達出她要帶給聽眾們的。因為小時候得到身邊許多熱心人士的幫助及信仰帶給她的力量，讓她更懂得感恩和珍惜現在所擁有的，並立志要當個有影響力的人。

梁文音因為曾經參選「尋找[[巴冷公主|巴冷公主]]」選拔，其總決賽就在綜藝節目《[[我愛黑澀會|我愛黑澀會]]》節目中舉行。梁文音打敗眾多原住民歌唱好手贏得冠軍，獲得機會與[[王宏恩|王宏恩]]在萬人演唱會合唱《巴冷公主》一曲，因此後來經常被稱作「[[巴冷公主|巴冷公主]]」<ref>{{cite news|language=zh-hant|author=記者陳品汎|url=http://wenews.nownews.com/news/3/news_3007.htm|title=「巴冷公主」　成長坎坷展韌性|publisher=大紀元時報|date=2008-12-12|accessdate=2010-02-22}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>或「巴冷」。之後梁文音獲邀短暫地在《我愛黑澀會》中當過黑澀會美眉，後來離開我愛黑澀會轉戰《[[超級星光大道|超級星光大道]]》比賽。

梁文音參加[[超級星光大道_(第一屆)|第一季《超級星光大道》]]的踢館賽時已表現出色，評審[[黃韻玲|黃韻玲]]亦讚賞其具穿透力的歌聲及音域之廣闊，令梁文音決定報名參加[[超級星光大道_(第二屆)|第二季《超級星光大道》]]。比賽最終雖然以些微分數落敗給[[賴銘偉|賴銘偉]]屈居第二，但於賽後2008年12月發行個人首張專輯《[[愛的詩篇|愛的詩篇]]》，並獲得了[[第20屆金曲獎|第20屆金曲獎]]最佳新人獎入圍。媒體與樂評美譽梁文音為[[張清芳|張清芳]]的接班人。<ref>{{cite web|url=http://www.hitoradio.com/news/6_1_1.php?news_id=70946 | title=HIT周末!專訪梁文音，李明依盛讚歌喉好似張清芳！|publisher=Hitoradio.com |date=2010-02-05 |accessdate=2010-02-22 }}</ref>

2013年搭配[[台視|台視]]、[[三立電視台|三立]]、[[東森|東森]]偶像劇《[[回到愛以前|回到愛以前]]》強打效應，使得片尾曲《分手後不要做朋友》廣被熱門傳唱，也順勢攻佔各大數位音樂排行榜冠軍，例如登上[[iTunes|iTunes]]單曲榜、KKBOX綜合新歌榜、KKBOX單曲榜、KKBOX新歌榜、myMusic單曲榜、Omusic綜合單曲榜以及Omusic國語單曲榜皆奪得第一名，該曲更成為梁文音近年音樂代表作之一。2015年為[[台視|台視]]歌唱選秀節目《[[Super_Star_我要當歌手2|Super Star 我要當歌手2]]》紅隊隊長。

== 對多元成家爭議的看法 ==
在2013年梁文音接受[[美國|美國]]《[[基督日報|基督日報]]》專訪，提到自己沒有參加反對[[多元成家立法草案|多元成家立法草案]]的遊行，但表示：「沒有錯，這種東西（多元成家）會使人跌倒，造成很多負面影響。可是我覺得，[[同性戀|同性戀]]朋友是需要被神愛、被神恢復的…」、「我相信生命都是可以改變的，沒有一件事情在神裏面是不能成就的。」<ref>{{Cite web |url = http://www.ettoday.net/news/20131221/308993.htm |title = 反「多元成家法案」　梁文音：同性戀朋友需要被神恢復 |author = 蕭采薇 |date = 2013-12-21 |publisher = 中時雜誌 |language = zh-hant |accessdate = 2015-04-03}}</ref>她並呼籲基督教友愛同志，提到：「他們可能會有很大的反應、很不舒服」，且表明要帶領同志認識神的愛，才能明白是非對錯與改變生命。<ref>{{Cite web |url = http://www.nownews.com/n/2013/12/22/1063473 |title = 「同性戀需被神恢復」　梁文音遭轟歧視 |date = 2013-12-22 |publisher = 今日新聞 |language = zh-hant |accessdate = 2015-04-03}}</ref>而[[牧師|牧師]][[歐陽文風|歐陽文風]]也認為梁文音過去所說：「如果法律通過了婚姻平權，我完全接受……」，並接著講「你還不明白我們所經歷的歧視，如果法律不通過，你根本不會為我們仗義執言，因為你根本不認為歧視是歧視。」<ref>{{Cite web |url = http://www.nownews.com/n/2014/10/11/1451090 |title = 牧師再批梁文音無知　「你根本不認為歧視是歧視」 |date = 2014-10-11|publisher = 今日新聞 |language = zh-hant |accessdate = 2015-04-03}}</ref>

梁文音在發表「同性戀朋友需要被神恢復」言論後，引發同志族群的不滿；隔年，臺灣女歌手[[萬芳|萬芳]]在[[中國廣播公司|中國廣播公司]]主持的《i-散步》節目中，播放梁文音的[[漫情歌|漫情歌]]專輯中《愛其實很殘忍》一曲，接著說：「愛有時候真的很殘忍，尤其是以愛之名，行恨之實的時候。有時候覺得很殘忍而且很可惡…最後就來聽這首表達支持同志婚姻的作品《[[平等之愛|Same Love]]》。」<ref>{{Cite web|url=http://www.appledaily.com.tw/realtimenews/article/new/20141008/483950/|title=梁文音淚崩　消毒「被神恢復」沒惡意|author=李志展|date=2014-10-08|publisher=蘋果日報|language=zh-hant|accessdate=2015-04-03}}</ref> <ref name=apple20141003>{{cite news|title=【動即時】「草莓歌手」梁文音道歉　萬芳《不要傷心了》|publisher=蘋果日報| url=http://www.appledaily.com.tw/realtimenews/article/new/20141003/480691/|date=2014-10-03|author=蕭孜玹、朱健弘|accessdate=2015-08-27|language=zh-hant}}</ref>梁文音為此承受極大的壓力， [[環球唱片_(台灣)|環球唱片]]也希望她休息幾天，恢復最佳狀態後，再出來面對歌迷和媒體。<ref>{{Cite web|title=梁文音被萬芳打臉後　音樂會10通告全喊卡|url=http://www.appledaily.com.tw/realtimenews/article/new/20141002/480691/|author=蕭孜玹|date=2014-10-02|publisher=蘋果日報|language=zh-hant|accessdate=2015-04-03}}</ref> 之後在臉書粉絲團貼出取消演出聲明稿。<ref name=apple20141003/>

後來梁文音在記者會談到「恢復說」，她說：「恢復指的是祝福跟撫慰，我的本意是好的，被誤會覺得蠻委屈的。」梁文音也再度強調自己沒有覺得誰生病，所以需要恢復，「恢復等同安慰，按照你需要的加給你，對我來說是祝福的話語。」

她表示：「因為身邊很多同志朋友在情路上都比較坎坷，所以才會這樣說。我完全尊重任何一個人求他們要的。」

梁文音表示自己從小到大，都一直在說「求神恢復我」，唱詩歌的時候也一定會唱到，「我來台北工作後，才真的接觸到同志朋友，我身邊也有很多同志，他們會跟我分享他們的愛情路，有的是家人都認同，有的是家人不認同，所以六日只能跑來台北約會，我唯一想到的是他們受傷了，所以才會講『求神恢復他們』。」

梁文音表示，半數以上的人都說因為她是基督徒，基督教反對同志，所以講出來的話就是在反對同志，「但我的信仰不是這樣教我的，聖經上沒有寫過一定要去支持或反對什麼，最重要的三大原則是『信』、『望』、『愛』，我連愛都還學不會，要怎麼去批評別人，更沒有資格完全反對或支持，如果婚姻平權在台灣通過了，那我相信神的安排是有道理的，如果沒有通過，我們還是一樣要過日子，但我樂見婚姻平權的通過。」<ref>{{Cite news|url=https://star.ettoday.net/news/410863|title=梁文音淚崩談同志「恢復」：我尊重每個人追求他們要的 {{!}} ETtoday星光雲|last=ETtoday新聞雲|newspaper=ETtoday星光雲|accessdate=2018-07-28|language=zh-TW}}</ref><ref>{{Cite news|url=https://star.ettoday.net/news/411008|title=基督徒梁文音「恢復」同志　本意遭曲解1年淚崩訴委屈 {{!}} ETtoday星光雲|last=ETtoday新聞雲|newspaper=ETtoday星光雲|accessdate=2018-07-28|language=zh-TW}}</ref>

== 評價 ==
* [[袁惟仁|袁惟仁]]：她在[[真情部落格|真情部落格]]裏這樣評論過梁文音：“貧窮的孩子早當家，這句話有部分是印證在梁文音身上。”出道至今從不懂得怎麽去面對演藝圈的大環境和媒體的小女孩，如今更懂得在自己熱愛的事業上找到屬於自己更自在的位子。在鏡頭前笑得更開懷的她，面對媒體也顯得對答如流，更加成熟及自在。從走出育幼院，到殘酷的舞臺上，再踏上演藝歌手的路程；這一路辛苦的經歷，讓梁文音更加的堅強勇敢。因為這樣，梁文音更懂得珍惜身邊人給她的關懷也願意去分享這份愛。而其在[[好消息電視台|好消息電視台]]裏所唱，來自[[六龜山地育幼院|六龜山地育幼院]]詩班裏常唱的《那雙看不見的手<ref>[http://www.youtube.com/watch?v=gW6l6o_u7Ik 梁文音-那雙看不見的手 (good tv) ]</ref>》也廣為人知，感動了無數人。

* [[黃韻玲|黃韻玲]]：“像這種黃鶯出谷的聲音其實是最悅耳動聽的。”（真情部落格）
* [[袁惟仁|袁惟仁]]：“她的聲音很輕柔，可是在輕柔裏面又帶著一些感傷，帶著一些她自己的故事。我覺得這樣子的聲音模式在目前華語樂壇裏面是沒有的。”（真情部落格）
* [[黃國倫|黃國倫]]：“梁文音擁有獨特的鼻腔共鳴，再綜合張清芳、許茹芸及阿妹的音色，但卻有自己的味道。”（2009年樂評）
