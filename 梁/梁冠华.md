'''梁冠华'''（{{bd|1964年|9月30日}}），男，[[汉族|汉族]]，[[籍贯|籍贯]][[北京|北京]]。现为[[北京人民艺术剧院|北京人民艺术剧院]][[一级演员|一级演员]]<ref name=bjry>{{cite web |url=http://www.bjry.com/main.asp?Url=Stars/content.asp?starsId=7 |title=北京人民艺术剧院——梁冠华 |deadurl=yes |archiveurl=https://web.archive.org/web/20160312041141/http://bjry.com/main.asp?url=stars%2Fcontent.asp%3Fstarsid%3D7 |archivedate=2016-03-12 }}</ref><ref>{{cite web |url=http://www.liangguanhua.com/viewthread.php?tid=7&extra=page%3D1 |title=梁冠华个人简历 }}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。同时为中国戏剧家协会会员、北京戏剧家协会会员、中国话剧艺术研究会会员、中国电视艺术家协会会员，并任北京市青联委员、北京市[[政协委员|政协委员]]。因《[[神探狄仁杰|神探狄仁杰]]》 饰 [[狄仁杰|狄仁杰]]一角引起广泛关注。

== 生平 ==
1981年6月进入[[北京人民艺术剧院|北京人民艺术剧院]]演员训练班。1984年9月毕业并留任[[北京人民艺术剧院|北京人民艺术剧院]]演员至今。

== 曾获奖项 ==
*1984年在《红白喜事》中扮演热闹获全国戏曲、话剧、歌剧观摩演出主演二等奖
*1987年获第四届中国戏剧梅花奖
*1995年被评为北京市优秀青年知识分子
*1995年话剧《鸟人》获北京市政府颁发“优秀新剧目展演优秀表演奖”
*1996年获第三届中国话剧研究会金狮表演奖
*1998年获国务院颁发政府津贴
*1999年因出演《古玩》中韩红木获1996－1999年度北京市新剧（节）目评奖演出优秀表演奖
*2000年荣获第十届春燕奖电视剧优秀男主角奖
*2001年获第十八届中国戏剧梅花奖
*2002年第二十一届全国电视剧“飞天奖”最佳男演员奖
