'''梁小冰'''（[[英語|英語]]：{{lang|en|'''Sabrina Leung Siu-bing'''}}，{{bd|1969年|12月29日}}），[[香港|香港]]著名[[女演員|女演員]]、[[女歌手|女歌手]]、女[[模特兒|模特兒]]。梁小冰於1990年參加[[無線電視|無線電視]][[香港小姐|香港小姐]]競選，並獲得季軍，隨後簽約成為無線合約藝員。因多為演出古裝劇集，被傳媒之為「'''古裝皇后'''」，同時為一名實力派演員，是九十年代無線電視當家花旦。2000年與男星[[陳嘉輝|陳嘉輝]]結婚 。為前[[香港電視|香港電視]]旗下藝員。

== 生平 ==
=== 廣告 ===
1988年，仍於[[天主教母佑會蕭明中學|天主教母佑會蕭明中學]]就讀的梁小冰，開始為夏士蓮潤膚露及[[港式快餐|港式快餐]][[大家樂|大家樂]]拍攝廣告，直至畢業。<ref>[http://www.youtube.com/watch?v=poC6mG1DfSg 香港中古廣告: hazeline夏士蓮潤膚露(梁小冰)1988]</ref><ref>[http://www.youtube.com/watch?v=ph1o1BLZyMA 香港中古廣告: 大家樂永遠100分(梁小冰尾鏡)1989]</ref>

1990年，於香港小姐競選獲得季軍後，與冠軍[[袁詠儀|袁詠儀]]及亞軍[[翁杏蘭|翁杏蘭]]一同為[[富士菲林|富士菲林]]拍攝廣告，三人知名度逐漸提升。<ref>[http://www.youtube.com/watch?v=rdngqxG_B1w 香港中古廣告: fuji富士 港姐(袁詠儀,翁杏蘭,梁小冰)1990]</ref>

1998年，約滿[[無線電視|無線電視]]的梁小冰重新為廣告商拍攝不同類型的廣告，大部分廣告均於[[中國|中國]]拍攝及播放。

2009年8月9日，與丈夫[[陳嘉輝|陳嘉輝]]於[[西貢|西貢]]拍攝林淦生淨斑靈10分鐘的廣告，隨後廣告於無線電視播映。<ref>[http://hk.apple.nextmedia.com/realtime/art_main.php?iss_id=20090811&sec_id=7018882&art_id=13084476 梁小冰 拍拖順便搵銀] 蘋果日報 2009年08月09日 (12:03 am)</ref>

=== 電視 ===
1990年8月，當時21歲的梁小冰參加[[無線電視|無線電視]]舉辦的[[香港小姐|1990年度香港小姐競選]]。於總決賽「型格大挑戰」環節中，其表演轉帽動作時失手把帽子掉在地上，但仍臨危不亂，順勢坐在地上擺出一個「手托香腮」的姿勢以完成表演。其十分專業的處理方法，被當年為評審之一的影星[[周潤發|周潤發]]評為最有演戲天份的香港小姐，最後獲得該年度的季軍。隨後梁小冰代表香港參與1990世界環球小姐。<ref>[http://www.misshongkongpageant.com/article3.htm 張文瑛 美的約會 梁小冰參選港姐只得一個理由] 香港電視 第1348期 1993年9月</ref> 另外，同年簽約成為無線合約藝員，並參演首部處境喜劇《[[同居三人組|同居三人組]]》，但只為客串性質。

1991年9月，首次擔任資訊節目《[[K-100|K-100]]》半年主持。同年11月，首次主演[[電視電影|電視電影]]《[[共飲長江水|共飲長江水]]》，劇中飾演反叛而喜愛跳舞的小如。

1992年，首次於《[[血璽金刀|血璽金刀]]》擔任劇集的女主角，飾演楊憑虎之女楊千千，以及同年9月主演電視電影《[[疾風柔情|疾風柔情]]》，飾演衣著及行為大膽的酒家女Vivian，逐漸於劇集中嶄露頭角。其後主演劇集《[[風之刀|風之刀]]》、《[[孤星劍|孤星劍]]》及《[[武尊少林|武尊少林]]》。

1994年，於劇集《[[清宮氣數錄|清宮氣數錄]]》飾演心狠手辣、性情大變的[[慈禧太后|慈禧太后]]、《[[天子屠龍|天子屠龍]]》分飾淘氣的[[冰月公主|冰月公主]]和溫柔的水月兩角，以及《[[尋龍劍俠賴布衣|尋龍劍俠賴布衣]]》土裡土氣的丫頭方青梅，令觀眾留下深刻印象。<ref>[http://blog.udn.com/linn/255952#ixzz1Ubkcpoj9 戲精小冰--美麗與戲劇的感動 - 小冰親衛隊 - udn部落格]</ref>

1997年，與[[陳小春|陳小春]]及[[馬浚偉|馬浚偉]]等演員拍攝劇集《[[鹿鼎記_(1998年電視劇)|鹿鼎記]]》，並於1998年首播。其劇中飾演任格剛烈、驕傲任性的[[阿珂|阿珂]]，以及高貴艷麗、烈性哀婉的[[陳圓圓|陳圓圓]]，獲得觀眾一致好評。此為其於無線電視的最後一部劇集，其後因約滿離開無線，轉往[[台灣|台灣]]及[[內地|內地]]發展。梁小冰亦言於無線拍戲期間，需節省往來[[電視城|電視城]]的交通費、經常於棚內休息室過夜，以及經常於古裝劇集佩戴頭套關係，頭髮因而脫落，身體亦變得廋弱，離開無線能替自己減少壓力和休息。

1999年，於[[中國|中國]][[深圳|深圳]]拍攝劇集《[[食神_(1999年電視劇)|食神]]》，因此劇於[[亞洲電視|亞洲電視]]播放，成為於亞洲電視首次演出的劇集。

2000年3月，與[[陳嘉輝|陳嘉輝]]、[[羅志祥|羅志祥]]等演員主演的劇集《[[新梁山伯祝英台|新梁山伯祝英台]]》於台灣[[中國電視|中國電視]]首播，其飾演女扮男裝而反斗精靈的[[祝英台|祝英台]]，而於大結局一幕則於[[梁山伯|梁山伯]]之墳墓前以淚洗面，淒慘不已，令觀眾扣緊心弦<ref>[http://www.1chinastar.com/news/ll/fanti/64037.shtml 梁小冰哭墳以淚洗面]{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }} 夜光新聞 2000-03-29</ref>，同年簽約成為亞視一年的合約藝員。<ref>[http://www.1chinastar.com/news/ll/fanti/1093680.shtml 梁小冰加盟亞視獲力捧]{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }} 夜光新聞 2001-09-09</ref>

2003年2月，於亞洲電視史詩劇集《[[萬家燈火|萬家燈火]]》中，飾演四海為家的獨立女性攝影師吳淑芳。

2003年5月，因香港爆發[[沙士|沙士]]疫潮，與[[黃日華|黃日華]]、[[周海媚|周海媚]]等演員參演電視電影《[[亞洲英雄|亞洲英雄]]》，以帶出時代訊息，其後該電視電影於[[有線電視|有線電視]]播映。<ref>[http://paper.wenweipo.com/2003/05/18/EN0305180056.htm 周海媚演殉職女醫生] 文匯報 2003-05-18</ref>

2005年7月2日，內地劇集《[[幻影神針|幻影神針]]》於亞洲電視首播，其於劇中飾演天賦異稟的佟馥兒。<ref>[http://app.hkatv.com/atvnews/view.php?id=1536 《幻影神針 》 周六隆重獻映 溫碧霞、梁小冰示範蜂針絕技] {{Wayback|url=http://app.hkatv.com/atvnews/view.php?id=1536 |date=20160304123904 }} 亞洲電視公佈 2005-06-28 Tue</ref>

2005年7月16日，為亞洲電視舉辦的《[[亞洲先生競選|2005亞洲先生競選]]》擔任評判及其「亞洲猛男大獎」的頒獎嘉賓。<ref>[http://app.hkatv.com/atvnews/view.php?id=1578 《2005亞洲先生競選 》星期六晚 8:35 數碼港 隆重直播] {{Wayback|url=http://app.hkatv.com/atvnews/view.php?id=1578 |date=20160304193637 }} 亞洲電視特別節目推介  2005-07-15 Fri</ref><ref>[http://hk.cnmdb.com/newsent/20050716/77018 圖文：12號趙湧獲亞洲猛男大獎 影星梁小冰頒獎] {{webarchive|url=https://web.archive.org/web/20090805013438/http://hk.cnmdb.com/newsent/20050716/77018 |date=2009-08-05 }} 新浪娛樂 2005-7-16</ref>

2010年8月21日，與丈夫[[陳嘉輝|陳嘉輝]]、[[戚黛黛|戚黛黛]]及[[袁文傑|袁文傑]]於亞洲電視《[[亞洲小天使才藝大賽|亞洲小天使才藝大賽]]》擔任司儀。<ref>[http://paper.wenweipo.com/2010/08/21/EN1008210003.htm 亞視以「小天使」迎戰「港男」] 文匯報 2010-08-21</ref>

2010年10月，與[[葉翠翠|葉翠翠]]、[[何傲兒|何傲兒]]、[[張嘉兒|張嘉兒]]及[[高海寧|高海寧]]出席由[[曾志偉|曾志偉]]主持的《[[名人廚房|名人廚房]]》錄影。<ref>[http://orientaldaily.on.cc/cnt/entertainment/20101008/00282_012.html?pubdate=20101008 志偉率港姐鬥煮] 東方日報 2010年10月8日(五)</ref>

2012年，加盟城市電訊旗下香港媒體製作，代表作為《[[導火新聞線|導火新聞線]]》、《[[驚異世紀|驚異世紀]]》，並於2015年播映。其中《[[導火新聞線|導火新聞線]]》迴響甚大，準備於2015年11月拍攝電影版，可惜梁小冰未能抽空參演。

=== 電影 ===
1997年11月，首次參演電影《[[熱血最強_(電影)|熱血最強]]》，由[[曾志偉|曾志偉]]、[[李子雄|李子雄]]及[[楊采妮|楊采妮]]等演員主演，其後主要於[[中國|中國]]地區拍攝電影。

2005年，首次主演福音電影《[[向生命致敬|向生命致敬]]劇場：天國戀曲》，並於[[越南|越南]][[柬埔寨|柬埔寨]]進行拍攝。<ref>[http://hk.cnmdb.com/newsent/20050718/78752 梁小冰柬埔寨拍戲被男主角求婚 送糖果給兒童] {{webarchive|url=https://web.archive.org/web/20090805013446/http://hk.cnmdb.com/newsent/20050718/78752 |date=2009-08-05 }} 新浪娛樂 2005-7-18</ref> 該電影於2006年1月24日舉行首映禮。

2006年，於電影《大丈夫2》擔任客串角色。

2010年，於電影《[[綫人_(電影)|綫人]]》擔任客串角色。

2011年，於[[中國|中國]]地區拍攝電影《[[水滸傳_(電影)|水滸傳]]》。

2015年，參與《[[踏血尋梅|踏血尋梅]]》，飾演男主角臧Sir ( [[郭富城|郭富城]]飾 ) 的前妻阿嫦，此片是梁小冰首部三級電影。

=== 音樂 ===
1994年，梁小冰於各歌手的音樂錄影帶中參與演出，包括[[李克勤|李克勤]]、[[陳慧嫻|陳慧嫻]]及[[許志安|許志安]]。

1998年1月，因約滿[[無線電視|無線電視]]，簽約[[誠信娛樂發展有限公司|誠信娛樂發展有限公司]]，並推出合輯《[[愛在今天|愛在今天]]EP》。

2000年3月，為中視劇集《[[新梁山伯祝英台|新梁山伯祝英台]]》演唱主題曲《快樂人兒》<ref>[http://www.youtube.com/watch?v=PufSpY_W8xw 少年梁祝片頭!!!]</ref>，並收錄於《[[少年梁祝千禧大全集|少年梁祝千禧大全集]]》合輯內。

2002年，為亞視劇集《[[女俠丁叮噹|女俠丁叮噹]]》演唱主題曲《百變神通》，隨後收錄於《[[勢在亞洲．超人氣電視劇主題曲巡禮|勢在亞洲．超人氣電視劇主題曲巡禮]]》合輯內。其後主唱[[基督教|基督教]]之歌曲，沒有收錄於任何專輯內。

=== 舞台劇 ===
2010年12月，梁小冰首次參演[[舞台劇|舞台劇]]《[[女活祭|女活祭]]》，並於[[葵青劇院|葵青劇院]]上映5場。

2011年5月8日，憑《[[女活祭|女活祭]]》於《[[第三屆香港小劇場獎|第三屆香港小劇場獎]]》獲頒「優秀女演員獎」，梁小冰亦言首次演出舞台劇作品便獲獎感到非常意外，頒獎嘉賓[[鮑起靜|鮑起靜]]亦覺得梁小冰有運氣，因鮑起靜自己曾演出兩部舞台劇，但一次提名都未得到過。<ref>[http://www.takungpao.com/history/top/2011-05-9/800235.html 第三屆小劇場獎頒八獎項]{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }} 大公網 2011年5月9日 04:39:57</ref>

=== 結婚生子及婚後生活 ===
1992年，梁小冰拍攝[[無線電視|無線電視]]劇集《[[兄兄我我|兄兄我我]]》時，與同劇演員[[陳嘉輝|陳嘉輝]]互生好感，繼而開始發展戀情。

2000年，兩人正式結婚，結束長達八年的愛情長跑。適逢劇集《[[新梁山伯祝英台|新梁山伯祝英台]]》於[[台灣|台灣]][[中視|中視]]首播，兩人首先於2月29日在台灣舉辦一場古式婚禮，以便替該劇造勢及讓台灣影迷先行替其祝福。3月4日，兩人亦於香港教堂舉行婚禮。<ref>[http://www.1chinastar.com/news/ll/fanti/60849.shtml?cc=10189 梁小冰下嫁陳嘉輝] {{webarchive|url=https://web.archive.org/web/20160825211846/http://www.1chinastar.com/news/ll/fanti/60849.shtml?cc=10189 |date=2016-08-25 }} 夜光新聞 2000-02-29</ref> 翌年4月，兩夫婦到[[美國|美國]][[夏威夷|夏威夷]]補行度蜜月。<ref>[http://www.1chinastar.com/news/ll/fanti/1066693.shtml?cc=10189 梁小冰夏威夷補度蜜月] {{webarchive|url=https://web.archive.org/web/20160825190239/http://www.1chinastar.com/news/ll/fanti/1066693.shtml?cc=10189 |date=2016-08-25 }} 夜光新聞 2001-04-22</ref>

2006年12月，梁小冰開始懷孕，預產期為2007年4月，當時其叫其醫生暫不把胎兒性別告知，待出世一刻才揭曉。<ref>[http://hk.cnmdb.com/newsent/20061229/886133 梁小冰成功懷孕五個半月 預產期明年四月底] {{webarchive|url=https://web.archive.org/web/20090805012935/http://hk.cnmdb.com/newsent/20061229/886133 |date=2009-08-05 }} 新浪娛樂 2006-12-29</ref> 另外，兩夫婦於同年[[聖誕節|聖誕節]]平安夜出席圈中好友[[鄺文珣|鄺文珣]]之婚禮。<ref>[http://bbs.ent.163.com/bbs/tvb/243764.html#6 老公唱情歌 鄺文珣感動又喊又笑] 東方日報 2006-12-25</ref>

2007年4月9日，丈夫陳嘉輝為梁小冰拍攝懷孕照片時，誕下男嬰，母子平安，貴為父親和母親的陳嘉輝及梁小冰亦言希望兒子能健康、快樂地成長。<ref>[http://hk.cnmdb.com/newsent/20070512/1025443 梁小冰拍大肚寫真導致早產 上月產下男嬰(圖)] {{webarchive|url=https://web.archive.org/web/20090805013635/http://hk.cnmdb.com/newsent/20070512/1025443 |date=2009-08-05 }} 新浪娛樂 2007-5-12</ref>

== 其他 ==
=== 個人評價 ===
梁小冰從首次擔任主角的劇集《[[血璽金刀|血璽金刀]]》至今，其所出演的角色均大受觀眾喜歡及讚賞，其中以[[古裝劇|古裝劇]]佔多。由於古裝扮相十分嬌俏迷人，數有「'''古典美人'''」之稱。梁小冰近年積極發展國內市場，2005年被國內傳媒評為擁有「優雅古典浪漫氣質」的香港小姐，外號「'''蝴蝶美人'''」。早前[[台灣|台灣]]地區接連播映幾部由梁小冰主演的香港劇集，同時於網路上掀起了一陣「小冰旋風」，討論其演技與美貌，並有影迷為其於網路上設立粉絲網頁（見外部連結）。其個人的獨特魅力令喜歡其的男女影迷繼續為她痴迷。
