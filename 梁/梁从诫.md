'''梁从诫'''（{{bd|1932年|8月4日|2010年|10月28日|catIdx=Liang梁从诫}}），出生于[[北平|北平]]，知名[[环保|环保]]人士，曾任环保组织[[自然之友|自然之友]]会长、[[中国文化书院|中国文化书院]]副院长、[[全国政协常务委员|全国政协常务委员]]，被誉为“中国民间环保第一人”<ref name="ifeng">{{cite web |url = http://finance.ifeng.com/huanbao/hbrw/20090826/1146531.shtml |title = 自然之子：梁从诫的绿色人生 |publisher = 凤凰网 |accessdate = 2010-02-27}}</ref>。他是著名建筑学家[[梁思成|梁思成]]与[[林徽因|林徽因]]之子，著名思想家及[[戊戌變法|戊戌變法]]領袖[[梁啟超|梁啟超]]之孫。

他的名字“'''从诫'''”源自中国古代建筑家[[李诫_(北宋)|李诫]]之名<ref name="bqn">{{cite web|url=http://www.fon.org.cn/content.php?aid=6701|title=宁可少一个历史学家　也要多一个自然之子|publisher=北京青年报|accessdate=2010-02-27|deadurl=yes|archiveurl=https://web.archive.org/web/20110308214234/http://www.fon.org.cn/content.php?aid=6701|archivedate=2011-03-08}}</ref><ref name="sina">{{cite web |url = http://finance.sina.com.cn/leadership/crz/20080224/11134539741.shtml |title = 梁从诫：向极端发展主义宣战 |publisher = 新浪财经 |accessdate = 2010-02-27}}</ref><ref>{{cite book |author =［美］费慰梅 |authorlink=费慰梅 |others=成寒 |title=《林徽因与梁思成》 |date = 2010-11 |publisher=法律出版社 |location=北京 |isbn = 978-7-5118-1376-3 |pages = 74 language = zh-hans |chapter = 10.太太的客厅}}</ref>。

== 生平 ==
1949年梁从诫17岁时，他曾背着当时是国旗国徽评选委员会顾问之一的父亲[[梁思成|梁思成]]提交了一个[[中华人民共和国国旗|新中国国旗]]设计方案，并在2992份方案中脱颖而出，入选了38份候选方案之一<ref name="bqn" />。

1950年，梁从诫曾想与父母一样成为一名建筑师，但因两分之差未考取父亲当系主任的[[清华大学|清华大学]]建筑系，随后转入历史系<ref name="bqn" />。1952年进入[[北京大学|北京大学]]历史系学习，并于1958年毕业。后前往[[云南大学|云南大学]]历史系任教，直至1962年。[[文化大革命|文化大革命]]期间，下放江西[[五七干校|五七干校]]劳动。1980年代时，作为[[中国大百科全书出版社|中国大百科全书出版社]]编辑，参与创办了《[[百科知识|百科知识]]》月刊。在此期间，他开始关注环境问题<ref name="bqn" />。从1989年开始，他担任[[全国政协|全国政协]]委员及常委，并任政协人口、资源、环境委员会委员。1993年，中国历史上第一次民间绿色讨论会[[玲珑园会议|玲珑园会议]]召开。随后，他与[[杨东平|杨东平]]、[[梁晓燕|梁晓燕]]、[[王力雄|王力雄]]共同创办了环保[[NGO|NGO]]——[[自然之友|自然之友]]并任会长。该组织也是中国最早在民政部门注册的民间环保组织之一<ref>{{cite web|url=http://www.fon.org.cn/channal.php?cid=2|title=自然之友简介|publisher=自然之友|accessdate=2010-02-27|deadurl=yes|archiveurl=https://web.archive.org/web/20100322232034/http://www.fon.org.cn/channal.php?cid=2|archivedate=2010-03-22}}</ref>。1998年，兼任[[中国文物学会|中国文物学会]]副会长。2002年，担任“[[北京奥组委|北京奥组委]]环境顾问”，2005年获“绿色中国年度人物”奖。2010年10月28日下午，因肺部疾病在北京去世<ref>{{cite web |url = http://news.china.com.cn/rollnews/2010-10/28/content_4938702.htm|title=自然之友创会会长梁从诫先生今日逝世 |publisher=中国网 |date=2010-10-28 |accessdate=2010-10-28}}</ref>。

== 自然之友 ==
在梁从诫的领导下，[[自然之友|自然之友]]开始关注各种环保问题，包括对[[滇金丝猴|滇金丝猴]]、[[可可西里|可可西里]][[藏羚羊|藏羚羊]]等濒危动物的保护，以及[[怒江|怒江]]水坝建设、[[圆明园|圆明园]]湖底铺衬等社会问题<ref name="china">{{cite web |url = http://www.china.com.cn/chinese/huanjing/1043283.htm |title = 2005“绿色中国年度人物”之三：梁从诫 |publisher=中国网 |accessdate=2010-02-27}}</ref>。1998年，时任美国总统[[克林顿|克林顿]]访华时，梁从诫与其他几位民间环保人士一同受邀與克林顿见面<ref name="gsean" />。同年，时任英国首相[[布莱尔|布莱尔]]访华之际，他曾给布莱尔写信请求他制止英国的藏羚羊非法交易，布莱尔当天回信表示支持，并于第二天与他会面<ref>{{cite web |url = http://finance.sina.com.cn/economist/jingjixueren/20081211/10385621668.shtml |title = 梁从诫：绿色中国推动者 |publisher=南方周末 |accessdate=2010-02-27}}</ref>。

由于健康原因，梁从诫晚年不再直接指导自然之友的工作，他希望自然之友也能够因此減少對他的依賴，走上真正的自力更生道路<ref name="ifeng" />。

== 评价 ==
梁从诫因其对[[中国环保运动|中国环保运动]]的贡献受到了很多好评。当他放弃史学研究创立自然之友时，学界泰斗[[季羡林|季羡林]]曾说他“宁愿丢一个[[历史学家|历史学家]]，也要多一个‘[[自然之友|自然之友]]’”<ref name="sina" />。2008年，《[[南方周末|南方周末]]》在选择其作为[[改革开放|改革开放]]30年间[[1994年中国|1994年]]度代表人物时，称其为“中国[[非政府组织|非政府组织]]的精神源头”<ref name="gsean" />。

不过同时，梁从诫的某些言论也遭到了一些争议。如他反对在[[怒江|怒江]]上建坝，就曾引起过争论<ref>{{cite web |url =  http://scitech.people.com.cn/GB/39327/3558333.html |title = 方舟子快评：水坝寿命到底有多长？|publisher=人民网 |accessdate = 2010-02-27}}</ref>。他的“敬畏自然”观念也遇到了诸如[[中国科学院院士|中国科学院院士]][[何祚庥|何祚庥]]等的反对<ref>{{cite web|url =http://www.xschina.org/show.php?id=3106|title =上帝·人·自然 ——简评梁从诫与何祚庥的争论|publisher =学术中华|accessdate =2010-02-27|deadurl =yes|archiveurl =https://archive.is/20070611001418/http://www.xschina.org/show.php?id=3106|archivedate =2007-06-11}}</ref>。1999年，电视集《[[人间四月天|人间四月天]]》放映后，梁从诫对于[[林徽因|林徽因]]与[[徐志摩|徐志摩]]的恋爱故事提出了异议，认为是对“历史事实和文化精神的双重歪曲”，并说《人间四月天》一诗是母亲写给当时还未出生的自己的。不过，此说引发了文化界的一些置疑<ref>{{cite web|url =http://chinese001.com/wxsj/zgwx/xdwx/zjda/xuzhimo/00069.jsp|title =林徽因没有爱过徐志摩吗？|publisher =中国语文网|accessdate =2010-02-27|deadurl =yes|archiveurl =https://web.archive.org/web/20100104180603/http://chinese001.com/wxsj/zgwx/xdwx/zjda/xuzhimo/00069.jsp|archivedate =2010-01-04}}</ref>。

== 家庭 ==
梁从诫出身于中国近代史上知名的[[梁启超家族|梁启超家族]]。他的祖父是中国近代重要思想家、[[戊戌变法|戊戌变法]]领袖[[梁启超|梁启超]]。他的父母[[梁思成|梁思成]]和[[林徽因|林徽因]]则是都是[[中国建筑史|中国建筑史]]上的一代名家。不过他自己则认为他没感受到这种光环，反而因祖父梁启超的缘故在“文革”中被批斗为“中国最大的[[保皇派|保皇派]]的孙子”而感到受辱<ref>{{cite web |url = http://www.people.com.cn/GB/shizheng/8198/31983/32177/2383979.html |title=梁从诫：名门之后 |publisher=人民网 |accessdate=2010-02-27}}</ref>。不过，他也不讳言要通过自己的家世背景来推动中国环保事业的发展<ref>{{cite web |url = http://www.sinovision.net/index.php?module=pregram&act=details&col_id=170&news_id=17024 |title = 楊瀾訪談錄－－梁從誡 |publisher=美国中文网 |accessdate=2010-02-27}}</ref>。

中国现代哲学泰斗[[金岳霖|金岳霖]]由于与梁思成、林徽因夫妇间有着特殊缘分，他晚年一直都和梁从诫一家住在一起，梁从诫则称其为“金爸”，情同父子<ref>{{cite web |url = http://www.81890.gov.cn:88/LaoNian/sanwen/ShowArticle.asp?ArticleID=3470 |title = 最是人间四月天（下） |publisher = 中国81890月湖老年网 |accessdate = 2010-02-27 }}{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

梁从诫与前妻周如枚（原北大校长[[周培源|周培源]]之女）育有一子，名周志兵（又名梁鉴，[[中国文物学会|中国文物学会]]常务理事）<ref>{{cite web|url=http://www.trends.com.cn/people/view/2010-05/265981.shtml|title=名媛梁周洋 只要对生命有热诚|publisher=《时尚·COSMOPOLITAN》|date=2010-05-24|deadurl=yes|archiveurl=https://web.archive.org/web/20100528112759/http://www.trends.com.cn/people/view/2010-05/265981.shtml|archivedate=2010-05-28}}</ref>。第二任妻子方晶则是一名学校教师，他们育有一女，名梁帆。她在[[纽约市立大学|纽约市立大学]]攻读哲学博士<ref>{{cite web|url=http://emotion.pclady.com.cn/star/1001/507068_3.html|title=家族：梁启超家的女人们|publisher=太平洋女性网|accessdate=2010-02-27|deadurl=yes|archiveurl=https://web.archive.org/web/20101129054509/http://emotion.pclady.com.cn/star/1001/507068_3.html|archivedate=2010年11月29日|df=}}</ref>。
