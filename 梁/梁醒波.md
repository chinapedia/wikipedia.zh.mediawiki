'''梁醒波'''{{MBE}}（{{bd|1908年||1981年|2月12日|catIdx=Leung}}），原名'''梁侍海'''，[[廣東|廣東]][[南海區|南海]]人，於[[新加坡|新加坡]]出生，是華人界粵劇（[[文武生|文武生]]和「-{丑生王}-」）電影及電視的一代始祖。

==簡歷==
父親[[梁悅|梁悅]]是[[新加坡|星洲]]紅伶「聲架悅」，姐姐則是紅伶花旗金。梁醒波早年醉心[[粵劇|粵劇]]，傾慕[[馬師曾|馬師曾]]，學「馬腔」。後來，他說服了父親讓他到[[馬來西亞|馬來西亞]]拜「[[大膽根|大膽根]]」為師。大膽根傳授了各種古老排場。梁醒波十七歲那年開始落班，隨即演出《[[高平關取級|高平關取級]]》。他在[[新加坡|新加坡]]和[[馬來西亞|馬來西亞]]各地演小武，聲譽日隆，成為「南洋四大天王」之一。

1939年，[[馬師曾|馬師曾]]在[[香港|香港]]的「[[太平劇團|太平劇團]]」缺人，特邀梁醒波加入任文武生。梁醒波從此便和太太與女兒留港發展，1941年改投「寶鼎劇團」。翌年，日軍大舉進攻[[香港|香港]]，梁醒波攜妻女逃往[[廣州|廣州]]，然後又至[[澳門|澳門]]、[[湛江|湛江]]，最後到了[[廣西|廣西]]。他與[[譚蘭卿|譚蘭卿]]、[[劉克宣|劉克宣]]等組織了一個戲班「新太平劇團」，在廣西各城鎮巡迴演出。

[[二次大戰|二次大戰]]後，梁回到香港，參加新世界劇團。後來身體日漸發胖，不適合再演[[小武|小武]]戲，由班政家鄧十四推薦，改演-{[[丑生|丑生]]}-。他天生有逗樂的本領，插科打諢已臻化境。先後在「[[仙鳳鳴劇團|仙鳳鳴]]」、「[[雛鳳鳴劇團|雛鳳鳴]]」、「[[大龍鳳劇團|大龍鳳]]」等著名粵劇團演出，從此奠定他在香港粵劇界和電影界鼎鼎大名的「-{丑生王}-」的地位。

1950年，梁醒波得著名粵劇編劇家[[任護花|任護花]]引薦，開始拍電影，演出第一個作品《[[臨老入花叢|臨老入花叢]]》（1950）。<ref>[http://www.takungpao.com/news/07/08/07/XK-776530.htm 話說任護花/葉輝] {{webarchive|url=https://web.archive.org/web/20070813000735/http://www.takungpao.com/news/07/08/07/XK-776530.htm |date=2007-08-13 }},大公報</ref>多年來，參演影片接近四百部粵語戲曲片和歌舞喜劇，並且曾經為電影《[[桃李爭春|桃李爭春]]》（1962年）作詞。<ref>[http://www.dianying.com/ft/title/tlz1962 桃李爭春(1962年)],中文電影資料庫</ref>

1967年，[[香港無綫電視|香港無綫電視]]（TVB）正式啟播，梁醒波成為開台功臣之一，並主持大型綜藝節目《[[歡樂今宵|歡樂今宵]]》達十數年之久，直至其病逝前三日仍然演出歡樂今宵。

另一方面，梁醒波曾連任五屆[[八和會館|八和會館]]主席，對[[粵劇|粵劇]]貢獻良多。1977年4月13日，港督[[麥理浩|麥理浩]]爵士代表[[伊利沙伯二世|英女皇]]向梁醒波頒受了[[MBE|MBE]]勳銜，以表其藝術成就，是首位香港藝人獲得此榮銜。<ref>{{cite web|url=http://www.hkfilm100.asia/hkfilm-history-recognition-late.html|title=香港電影一百年|publisher=香港電影一百年|date=2009年|accessdate=2009-10-04|archive-url=https://web.archive.org/web/20090305084336/http://hkfilm100.asia/hkfilm-history-recognition-late.html|archive-date=2009-03-05|dead-url=yes}}</ref>

1981年2月12日，梁醒波因[[腸癌|腸癌]]病逝[[香港|香港]]，享年73歲。

2009年，香港大學教育學院中文教育研究中心，聯同經濟日報出版社合作出版梁醒波的傳記：《梁醒波傳—亦慈亦俠亦詼諧》<ref>{{ cite book |url= |title=梁醒波傳—亦慈亦俠亦詼諧 |date=2009 |publisher=經濟日報出版社 |author=吳鳳平、鍾嶺崇 |location=香港 | pages=  |ISBN=9789626785959 }} </ref>，由[[吳鳳平|吳鳳平]]博士及[[鍾嶺崇|鍾嶺崇]]博士編著。<ref>{{cite web |url=http://www.etpress.com.hk/contents/etpress/tod/focus/f08/ |title=梁醒波101 歲冥壽 |publisher=香港經濟日報 |date=2009-10-12 }} </ref>

== 家庭 ==
梁氏與夫人[[顧文娟|顧文娟]]（1916年－2008年9月15日）<ref>[http://hk.apple.nextmedia.com/entertainment/art/20080930/11664530 梁醒波夫人舉殯] 蘋果日報 2008-09-30</ref>育有三子四女，長子梁乃業赴英留學後回港從醫，其中三名女兒均為知名藝人。長女[[梁葆貞|梁葆貞]]以扮演處境喜劇[[香港八一至香港八六系列|《香港81》至《香港86》系列]]的「順嫂」聞名。次女梁葆文（[[文蘭_(演員)|文蘭]]）是電影演員，活躍於五六十年代，曾演出電影《嬌嬌女》<ref>{{cite web |url=https://www.youtube.com/watch?v=kbmH3eMxpXY |title=嬌嬌女 - 1956年出品 - 由文蘭,梁醒波,淑子,江一帆,黎灼灼,黎雯,李鵬飛,曾楚霖主演 |date= |publisher= |author=YouTube }}</ref>，她也是李小龍兒時好友和舞伴<ref>{{cite news |url=https://hk.entertainment.appledaily.com/entertainment/daily/article/20101114/14659494 |title=李小龍70冥壽專題：李小龍好舞功 |date=2010-11-14 |publisher=蘋果日報 |author= }}</ref>。么女[[梁寶珠|梁寶珠]]是粵劇演員，曾與[[陳寶珠|陳寶珠]]曾經合組「孖寶劇團」。<ref>http://www.dianying.com/ft/topics/hkstars/LiangXingbo.php 梁醒波] [[中文電影資料庫|中文電影資料庫]]</ref>

2018年，梁醒波孫兒梁智宏於過去數年間突然愛上了爺爺的作品，適逢梁醒波於2018年110歲冥壽，梁智宏出力協助重播電影及辦展覽作回顧，希望年輕的一代，能認識這位名伶<ref>[https://bkb.mpweekly.com/%E6%96%87%E5%8C%96/20180714-78366 【星期日人物】惡補爺爺作品 梁醒波孫變粵劇迷 帶波叔到2018]</ref>。

== 軼事 ==
香港電影資料館一級助理館長龍智傑表示蕭芳芳曾向梁醒波請教授搞笑的心得，但梁醒波表示她學不來，因為觀眾看到自己的脂肪震動就自然會笑<ref>[https://bkb.mpweekly.com/cu0002/20180715-78387 文武生功架的丑生 梁醒波亦莊亦諧 文蘭眼中的慈父]</ref>。
