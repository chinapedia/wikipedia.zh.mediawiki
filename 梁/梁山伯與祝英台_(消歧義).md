'''梁山伯與祝英台'''（簡稱'''梁祝'''）可以指：
== 民間傳說 ==
* [[梁山伯與祝英台|梁山伯與祝英台]]：中國四大[[民間傳說|民間傳說]]之一，內容敘述梁山伯與祝英台悲劇性的愛情故事。
== 越劇 ==
* [[梁山伯與祝英台_(越劇)|梁山伯與祝英台 (越劇)]]：[[越劇|越劇]]的一齣經典劇目。
== 電影 ==
* [[梁山伯與祝英台_(1953年電影)|梁山伯與祝英台 (1953年電影)]]：1953年由[[上海电影制片厂|上海电影制片厂]]根據[[民間傳說|民間傳說]]《梁山伯與祝英台》改編而拍成的[[越剧|越剧]]電影。
* [[梁山伯與祝英台_(1955年電影)|梁山伯與祝英台 (1955年電影)]]：1955年根據[[民間傳說|民間傳說]]《梁山伯與祝英台》改編而拍成的[[香港|香港]][[粵語長片|粵語長片]]。
* [[梁山伯與祝英台_(1963年電影)|梁山伯與祝英台 (1963年電影)]]：1963年的[[黃梅調|黃梅調]]電影，由[[邵氏電影公司|邵氏電影公司]]出品，所有版本中最為出名的一部。
* [[梁山伯與祝英台_(潮語電影)|梁山伯與祝英台 (潮語電影)]]：根據[[民間傳說|民間傳說]]《梁山伯與祝英台》改編而拍成的香港[[潮州話|潮州話]]電影。
* [[梁山伯與祝英台_(1964年電影)|梁山伯與祝英台 (1964年電影)]]：1964年根據[[民間傳說|民間傳說]]《梁山伯與祝英台》改編而拍成的香港[[國語|國語]]電影，由[[電懋|電懋]]公司出品。
* [[梁祝_(1994年電影)|梁祝 (1994年電影)]]：1994年的香港电影，由[[徐克|徐克]]导演，[[吴奇隆|吴奇隆]]、[[杨采妮|杨采妮]]等主演。
* [[武俠梁祝|武俠梁祝]]：2008年華語電影。

== 動畫 ==
* [[蝴蝶夢-梁山伯與祝英台|蝴蝶夢-梁山伯與祝英台]]：2003年12月31日上映的[[動畫電影|動畫電影]]，由台湾[[中央電影事業股份有限公司|中央電影事業股份有限公司]]製作，[[蕭亞軒|蕭亞軒]]、[[劉若英|劉若英]]、[[吳宗憲|吳宗憲]]配音。
== 音樂 ==
* [[梁祝小提琴协奏曲|梁祝小提琴协奏曲]]：由[[上海音乐学院|上海音乐学院]]学生[[何占豪|何占豪]]、[[陈钢|陈钢]]以越剧《梁山伯与祝英台》旋律为基础所作,于1959年首演的[[小提琴协奏曲|小提琴协奏曲]]。
== 電視劇 ==
* [[七世姻緣－第二世－梁山伯與祝英台|七世姻緣－第二世－梁山伯與祝英台]]，1975年播出[[香港|香港]][[麗的映聲|麗的映聲]]製作的[[連續劇|連續劇]]，由[[文千歲|文千歲]]、[[歐陽佩珊|歐陽佩珊]]、[[陳儀馨|陳儀馨]]等主演。
* [[民間傳奇之梁山伯與祝英台|民間傳奇之梁山伯與祝英台]]，1977年播出[[香港|香港]][[無線電視|電視廣播有限公司]]製作的[[連續劇|連續劇]]，由[[劉松仁|劉松仁]]、[[李琳琳|李琳琳]]、[[羅浩楷|羅浩楷]]、[[陳復生|陳復生]]、[[楊志恆|楊志恆]]等主演。
* [[七世夫妻之梁山伯與祝英台|七世夫妻之梁山伯與祝英台]]：1999年的[[民視無線台|民視無線台]][[八點檔|八點檔]][[連續劇|連續劇]]，由[[賈靜雯|賈靜雯]]、[[趙擎|趙擎]]、[[張晨光|張晨光]]、[[張鳳書|張鳳書]]、[[李國超|李國超]]等主演。
* [[少年梁祝|梁山伯與祝英台 (2000年電視劇)]]：2000年3月2日播出的[[臺灣|臺灣]][[中國電視公司|中國電視公司]][[連續劇|連續劇]]，由[[羅志祥|羅志祥]]、[[梁小冰|梁小冰]]、[[陳嘉輝|陳嘉輝]]、[[歐漢聲|歐漢聲]]等主演。
* [[梁山伯與祝英台_(2007年電視劇)|梁山伯與祝英台 (2007年電視劇)]]：2007年的[[中國大陸電視劇|中國大陸電視劇]]，由[[何潤東|何潤東]]、[[董潔|董潔]]等主演。
* [[梁山伯與祝英台新传|梁山伯與祝英台新传]]：2017年的[[中國大陸電視劇|中國大陸電視劇]]，由[[晏紫东|晏紫东]]、[[陈梦希|陈梦希]]、[[林枫松|林枫松]]等主演。

== 音樂劇 ==
* [[梁祝_(音樂劇)|梁祝 (音樂劇)]]：2003年台灣[[大風音樂劇場|大風音樂劇場]]製作的音樂劇，由[[鍾耀光|鍾耀光]]作曲，[[李小平|李小平]]導演，[[辛曉琪|辛曉琪]]、[[王柏森|王柏森]]、[[黃士偉|黃士偉]]、[[洪瑞襄|洪瑞襄]]等主演。

== 芭蕾舞劇 ==
* [[梁祝_(芭蕾舞剧)|梁祝 (芭蕾舞剧)]]：2001年[[上海芭蕾舞剧团|上海芭蕾舞剧团]]在[[2001年上海国际艺术节|2001年上海国际艺术节]]闭幕式首演的原创民族芭蕾舞剧，由[[罗怀臻|罗怀臻]]编剧，[[徐坚强|徐坚强]]作曲，[[辛丽丽|辛丽丽]]导演，[[上海芭蕾舞剧团|上海芭蕾舞剧团]]演出。

== 舞台劇 ==
* [[梁祝_(舞台劇)|梁祝 (舞台劇)]]：1999年由[[謝君豪|謝君豪]]、[[蓋鳴輝|蓋鳴輝]]等主演。
* [[梁祝下世传奇_(专辑)|梁祝下世传奇 (专辑)]]：2005年的[[舞台劇|舞台劇]]，由[[何韻詩|何韻詩]]、[[周國賢|周國賢]]、[[梁祖堯|梁祖堯]]、[[楊淇|楊淇]]等主演。
