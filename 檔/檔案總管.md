'''Windows 檔案總管'''（Windows Explorer），是[[Microsoft_Windows|Microsoft Windows]]及[[React_OS|React OS]]操作系统中浏览电脑中檔案与文件夹结构的基本工具。
<!-- 通常[[Windows|Windows]]使用者不會特別注意這個功能，大多僅使用[[開始功能表|開始功能表]]開啟「文件」、「圖片」、「音樂」等資料夾，在檢視資料夾的同時，其實就是在使用Windows 檔案總管。 存疑，应该很多人都注意吧 -->

== 功能 ==
開啟Windows 檔案總管的視窗稱為「瀏覽」窗格，可以為任何檔案進行排序，把地址轉成資料夾形式供使用者檢視，並且列出檔案詳細資料，或是檢視縮圖等；並且使用者可以在Windows 檔案總管進行檔案移動，將檔案從一處移動到另一處，也可以直接更改檔案名稱。在[[工作管理員|工作管理員]]內名稱皆為explorer.exe。

==歷史==
=== Windows 95及Windows NT 4.0===
[[File:Windows_Explorer_95.png|thumb]]]]
在[[Windows_95|Windows 95]]及[[Windows_NT_4.0|Windows NT 4.0]]中，该工具只有非常基本的瀏覽窗口，僅僅只顯示檔案圖示，不显示任何附加訊息，工具列只有「檔案」、「編輯」、「檢視」、「說明」，這個瀏覽模式從[[Windows_3.X|Windows 3.X]]的[[Program_Manager|Program Manager]]（progman.exe，程序管理器）傳承自[[Windows_95|Windows 95]]，Windows Explorer此名稱從Windows 95開始使用，所以Windows 95是第一款包含Windows Explorer的[[Windows|Windows]]版本。右上角的三顆按鈕最小化、最大化／往下還原、關閉按鈕也成了經典樣式，至[[Windows_10|Windows 10]]都不曾改變。

基本的功能已經可以達到不用使用指令讓檔案複製、移動。
{{-}}
=== Windows 98 ===
[[File:My_Computer_in_Win98_zh_cn.png|thumb]]]]
[[Windows_98|Windows 98]]比[[Windows_95|Windows 95]]中增加了標準工具列，上一頁及下一頁按鈕，「我的最愛」觀念也從Windows 98開始，支援[[Active_Desktop|Active Desktop]]。左方也多了類似[[Windows_XP|Windows XP]]概念的資訊列，顯示基本檔案訊息及所在資料夾圖示，背景為藍天。
{{-}}
=== Windows 2000和Windows ME ===
[[File:My_Computer_in_Win2000_zh_cn.png|thumb]]]] 
[[Windows_2000|Windows 2000]]和[[Windows_ME|Windows ME]]與[[Windows_98|Windows 98]]類似，但工具列比以往增加了搜尋檔案的選項。
{{-}}
=== Windows XP和Windows Server 2003 ===
[[File:Windows_Explorer_on_Windows_XP.png|thumb]]]]
[[Windows_XP|Windows XP]]和[[Windows_Server_2003|Windows Server 2003]]中比以往添加了Windows XP樣式的資訊列，位於檔案總管左方，但若開啟資料夾樹狀圖則左方資訊列消失。資訊列上也會有檔案關聯性的執行方式，例如，在檢視圖片資料夾時，左方就會有「播放幻燈片」選項；以及連結功能，到某些特定的資料夾，無法自定。並且也有詳細資料之顯示，皆以區塊表示，可以縮小。
{{-}}
=== Windows Vista與Windows 7 ===
[[File:Windows_Vista_Explorer.png|thumb]]]]
[[File:Windows_Explorer_on_Windows_7.png|thumb]]]]
在[[Windows_Vista|Windows Vista]]與[[Windows_7|Windows 7]]中，傳統工具列預設關閉。左方有最愛的連結以及資料夾以層遞展開的樹狀圖，最愛的連結可由使用者自訂資料夾連結放入內；上方則有目前檢視檔案類型的關聯性最常使用的功能，例如，在檢視音樂資料夾時，上方就會有「播放所有曲目」選項；下方則有該檔案的詳細資料及縮圖。網址列不以「X:\XXX\XXX\」方式呈現，而是以「資料夾▶資料夾」方式呈現，「▶」可點選檢視該資料夾內所包括的資料夾；並且也添加了搜尋列以便搜尋該資料夾內的檔案。

右圖是在「電腦」的Windows檔案總管，在Windows Vista可見到所有磁碟下方都會有使用率的條狀物。
{{-}}
=== Windows 8 ===
[[File:Windows_8_explorer.png|thumb]]、[[Windows_8.1|Windows 8.1]]]]
[[Windows_8|Windows 8]]中的文件資源管理器以類似[[Microsoft_Office|Microsoft Office]]的[[Ribbon|Ribbon]]設計取代傳統功能列，並配合[[觸控|觸控]]操作。<ref>http://chinese.engadget.com/2011/08/30/microsoft-details-windows-8-explorer-tweaks-adds-ribbon-ui/</ref>
{{-}}
=== Windows 10 ===
[[Windows_10|Windows 10]]中的文件資源管理器亦延續[[Windows_8|Windows 8]]、[[Windows_8.1|Windows 8.1]]的[[Microsoft_Office|Microsoft Office]]的[[Ribbon|Ribbon]]設計取代傳統功能列，並配合[[觸控|觸控]]操作，唯一不同的是[[Windows_10|Windows 10]]版本的檔案總管大幅縮減視窗邊框。

== 檢視方式類型 ==
* 圖示：使檔案由左至右並排，檔案或資料夾名稱寫在圖示下方，自[[Windows_Vista|Windows Vista]]开始分成四種大小，在[[Windows_XP|Windows XP]]則仅有預設为32x32。
:* 小圖示：16x16
:* -{zh-hans:中等图标;zh-hant:中圖示}-：48x48
:* 大圖示：128x128
:* -{zh-hans:超大图标;zh-hant:特大圖示}-：256x256
* -{zh-hans:平铺;zh-hant:並排}-：使檔案由左至右並排，檔案或資料夾名稱寫在圖示右方，右上黑色文字為檔案名稱，右下灰色文字為檔案類型。
* -{zh-hans:列表;zh-hant:清單}-：類似詳細資料，僅是以最小圖示由上至下、由左至右排列，沒有檔案詳細內容显示。
* -{zh-hans:详细信息;zh-hant:詳細資料}-：以表格式顯示，最左方顯示檔案最小縮圖及檔案名稱，右方諸多表格顯示檔案詳細資料，包括分級、作者、檔案類型等，要显示的信息列可自定义。
