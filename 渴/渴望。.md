《'''渴望。'''》（{{jp|j=渇き。|hg=かわき。|rm=Kawaki.|extra2={{lang-en|The World of Kanako}}|lead=yes}}）是一部由[[中島哲也|中島哲也]]執導、於2014年上映的日本[[驚悚|驚悚]][[懸疑|懸疑]]電影，由[[役所廣司|役所廣司]]主演。

== 背景 ==
《渴望。》改編自{{link-ja|深町秋生|深町秋生}}的小說作品《無盡的渴望》（{{lang|ja|果てしなき渇き}}），該書曾經於2005年獲得「[[「這本推理小說了不起！」大獎|『這本推理小說了不起！』大獎]]」<ref name="konomisu">{{Cite web| title = 第3回『このミス』大賞| publisher = 宝島社| url = http://konomys.jp/archives/vol_03| accessdate = 2013-08-08}}</ref><ref name="allabout">{{Cite web| author = 梅村千恵| title = この新人がすごい!?2『果てしなき渇き』| publisher = All About| date = 2005-03-17| url = http://allabout.co.jp/gm/gc/208179/| accessdate = 2013-08-08}}</ref>。

本片由曾經執導2010年電影《[[告白_(電影)|告白]]》之[[中島哲也|中島哲也]]擔任導演<ref name="movie">{{Cite web| title = 中島哲也監督『告白』以来3年ぶり新作『渇き。』製作決定 役所広司、橋本愛、中谷美紀ら出演| publisher = [[オリコン|ORICON STYLE]]| date = 2013-08-07| url = http://www.oricon.co.jp/news/movie/2027334/full/| accessdate = 2013-08-08}}</ref>。中島哲也稱他對小說故事非常感興趣，尤其是當中的男主角：「近年的日本電影中沒見過這麼爛的人。他遍體鱗傷之後，反而才找到對女兒的親情。這個過程令人氣憤難耐又緊張刺激，讓我想拍他的電影」。中島因而捨棄了拍攝《[[進擊的巨人|進擊的巨人]]》[[進擊的巨人#真人電影|真人版電影]]的機會而接拍本片，又表示：「這是如惡夢一般的作品，絕不可能被拍成電影，但我的目標就是要拍出日本影史前所未見的暴力奇想電影。」<ref name=yahoo/>

男女主角則分別由資深男演員[[役所廣司|役所廣司]]和新晉女演員[[小松菜奈|小松菜奈]]飾演，並有[[妻夫木聰|妻夫木聰]]和[[中谷美紀|中谷美紀]]等著名演員助演，陣容被稱為「日本影史最華麗」<ref>{{cite web|title=書介：渴望|url=http://paper.wenweipo.com/2014/07/28/OT1407280010.htm|publisher=[[文匯報|文匯報]]|accessdate=2014-08-02|date=2014-07-28}}</ref>。導演中島哲也曾表示只有役所廣司能同時展現瘋狂、滑稽、恐怖和輕率於一身，但他最初認為役所不會想接演這麼殘忍的故事，所以「已經死了半條心」，不過跟中島哲也合作過的役所廣司則表示「能夠演出這個角色是我當演員的幸福」，即時接受邀請。新人小松菜奈則獲得中島「兼具無邪的妖豔與駭人的純真，以壓倒性的氣勢降臨新世代蛇蠍美人」的評價<ref name=yahoo/>。

電影於2013年末完成拍攝<ref>{{Cite web| title = 公開日も発表に。鬼才・中島哲也監督の新作『渇き。』がついに完成!| publisher = ぴあ映画生活| date = 2014-04| url = http://cinema.pia.co.jp/news/164572/56660/| accessdate = 2014-04-06}}</ref>，原本預定於2014年7月4日在日本上演<ref>{{Cite web| author = 中山雄一朗| title = 『告白』中島哲也監督の最新作がついに完成!公開日は7月4日に決定| publisher = シネマトゥデイ| date = 2014-04-04| url = http://www.cinematoday.jp/page/N0061938| accessdate = 2014-04-06}}</ref>，不過最後片商決定提早一週在6月27日於戲院上映，並向高中生以及大學生提供1000[[日圓|日圓]]入場優惠直至7月4日（即原定的上映日期）<ref>[http://www.cinematoday.jp/page/N0063020 中島哲也監督『渇き。』が異例の公開繰り上げ！6月27日公開に変更 ]シネマトゥデイ 2014年5月16日</ref>。優惠期後來延長至7月18日<ref>{{cite web|title=延長決定!!学生1000円 学割キャンペーン! !|url=http://kawaki.gaga.ne.jp/news.html|publisher=GAGA|accessdate=2014-08-02}}</ref>。

電影宣傳標語是「深愛的女兒，原來是一隻魔鬼。」（{{lang|ja|愛する娘は、バケモノでした。}}）和「將你理性吹飛的劇藥娛樂」（{{lang|ja|あなたの理性をぶっ飛ばす劇薬エンタテインメント!!}}）<ref name="com20140415">{{Cite web| title = でんぱ組.inc、中島哲也新作「渇き。」劇中歌を提供 予告も公開| publisher = [[映画.com|映画.com]]| date = 2014-04-15| url = http://eiga.com/news/20140415/11/| accessdate = 2014-04-18}}</ref>。[[映畫倫理委員會|映畫倫理委員會]]將本片評為限制級（R15+）<ref>{{cite web|title=渴望|url=http://movie.douban.com/subject/24973419/|publisher=豆瓣电影}}</ref>。

== 劇情概要 ==
前刑警藤島昭和（役所廣司 飾）在漂亮又成績優異的女兒藤島加奈子（小松菜奈 飾）失蹤後，四處展開搜尋。然而沒想到在尋找女兒的過程中，卻被無端捲入一樁又一樁不可思議的事件，也漸漸了解到了女兒不為外人所知的瘋狂面貌。藤島在噩夢清醒之後，等著他的是究竟地獄還是天堂？<ref name=yahoo>{{cite web|title=渴望(The World of Kanako)|url=https://tw.movies.yahoo.com/movieinfo_main.html/id=5232|publisher=Yahoo!奇摩電影|accessdate=2014-08-02}}</ref>
