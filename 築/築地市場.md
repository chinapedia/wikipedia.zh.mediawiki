'''築地市場'''（{{jpn|j=築地市場|hg=つきじしじょう|rm=Tsukiji shijō|g=y}}）是位於[[日本|日本]][[東京都|東京都]][[中央區_(東京都)|中央區]][[築地|築地]]的公設[[批發|批發]][[市場|市場]]，為[[東京都|東京都]]11座[[東京都中央批發市場|中央批發市場]]之一，亦是全世界營運規模最大的{{link-en|魚市場|Fish market}}，從1935年開設至2018年10月6日為止<ref>{{cite news|url=https://www.asahi.com/articles/ASL9Z7D1RL9ZUTIL02D.html|script-title=ja:仲卸「寂しいけど、ありがとうだな」築地市場が営業終了|trans-title=築地市場結束營業　中盤商：雖然感覺寂寞，還是滿懷感謝|language=ja|author=|work=朝日新聞電子版|publisher=[[朝日新聞社|朝日新聞社]]|date=2018-10-06|accessdate=2018-10-10|dead-url=no|archiveurl=https://web.archive.org/web/20181010113136/https://www.asahi.com/articles/ASL9Z7D1RL9ZUTIL02D.html|archivedate=2018-10-10}}</ref>。除了批發市場區塊之外，旁側尚有規模龐大的[[傳統市場|傳統市場]]及[[商店街|商店街]]，稱為「場外市場」，與別稱為「場內市場」的批發市場本體相對。

築地場內市場停用後將會拆除，改建为[[停車場|停车场]]予[[2020年夏季奥林匹克运动会|2020年東京奧運]]使用<ref>[https://www.channel8news.sg/news8/world/20181006-wld-japan-tsukji/4147616.html 日本东京筑地市场 最后一天营业 业者访客满心不舍]</ref>，其功能由同步啟用之[[豐洲市場|豐洲市場]]繼承。場外市場則原地保留。

== 簡介 ==
築地市場整體上有兩個不同的部分：狹義上的築地市場僅指'''批發市場'''區域，又稱為「'''場內市場'''」（{{jp|j=場内市場|hg=じょうないしじょう|rm=Jōnai shijō}}），有7家註冊批發業者、約1000家註冊{{link-ja|中盤商|仲卸|仲卸}}（中盤商）業者在此進行魚貨與蔬果的[[拍賣|拍賣]]，面積約為23萬[[平方公尺|平方公尺]]；廣義則包括批發市場東北側的[[傳統市場|傳統市場]]，稱為'''築地場外市場商店街'''（{{jp|j=築地場外市場商店街|hg=つきじじょうがいしじょうしょうてんがい|rm=Tsukiji jōgai shijō shōtengai}}），通稱「'''場外市場'''」（{{jp|j=場外市場|hg=じょうがいしじょう|rm=Jōgai shijō}}）、或逕稱為「場外」，由銷售廚房器具、餐廳用品、雜貨、海鮮的小型批發與零售店、以及許多餐館所組成，特別是壽司店。

除了在[[築地|築地]]六丁目的停車場部份地方外，築地市場（場內市場）的地址編號為「[[東京都|東京都]][[中央區_(東京都)|中央區]]築地五丁目2番1號」。距離市場最近的車站為[[都營地下鐵|都營地下鐵]][[大江戶線|大江戶線]]的[[築地市場站|築地市場站]]，另一較近的車站則是約300公尺外的[[東京地下鐵|東京地下鐵]][[日比谷線|日比谷線]][[築地站|築地站]]。

現時於築地市場處理的貨品種類除了[[海鮮|海鮮]]（處理量日本最大）外，還有[[蔬果|蔬果]]（包括[[蔬菜|蔬菜]]和[[水果|水果]]；雖然少為人知，在東京是僅次{{link-ja|大田市場|大田市場}}的第二位）、[[雞肉|雞肉]]、[[雞蛋|雞蛋]]、[[醃|醃菜]]和各種加工食品（[[豆腐|豆腐]]、[[豆芽|豆芽]]、[[急凍食品|急凍食品]]等）。2005年的數據顯示，處理所有種類貨品合計約91萬6866公噸（每日處理海鮮2167公噸，蔬果1170公噸），金額約為5657億日圓（每日處理海鮮17.68億日圓，蔬果3.2億日圓）<ref>[http://www.shijou.metro.tokyo.jp/tonai/02/index.html 東京都中央批發市場：各市場的所在地與規模（2005年4月1日查閱）] {{webarchive|url=https://web.archive.org/web/20051124171218/http://www.shijou.metro.tokyo.jp/tonai/02/index.html |date=2005年11月24日 }}</ref>。

拍賣場和中間商所在的主要建築物設計成一弧形，最主要是由於市場最初附設有[[日本國有鐵道|國鐵]]{{link-jp|東京市場站|東京市場駅}}，站內外的軌道路線與市場建築平行，用此讓鮮魚貨物列車等入線。但[[载货汽车|公路貨車]]崛起後，從一般貨物到新鮮食品都逐漸由貨車取代貨物列車運送，受此影響冷凍車、活魚車等該站停靠的貨物列車逐漸被淘汰，而車站亦在1984年廢止，市場內外的路線均被撇除。市場的青果門附近仍在{{link-ja|汐留站 (國鐵)|汐留駅 (国鉄)|舊汐留站}}遺址的延伸處留下了零散的柏油路作為路線的痕跡，那條行人路亦留有了原[[平交道|閘道]]的警報機。

[[銀座|銀座]]商業區就在附近，有如此地點條件的批發市場在其他地區並不多。

== 歷史 ==
[[File:Tsukiji_Fish_Market_under_construction.JPG|225px]]時代]]
[[File:Daigo_Fukuryu_Maru_Monument.jpg|225px]]

從[[江戶時代|江戶時代]]起負責東京食品流通業務的是[[日本橋_(東京)|日本橋]][[河岸|魚河岸]]為首的市場群，但被1923年（[[大正|大正]]12年）9月發生的[[關東大地震|關東大地震]]摧毀。同年12月，借用原本為外國人居留地（築地居留地）的[[海軍省|海軍省]]屬地作為臨時的東京市設魚市場的地點，同時在此開始建設新的鮮魚市場 - 築地市場。

* 1927年（[[昭和|昭和]]2年）取得築地主要建築用地12239.92坪。
* 1928年（[[昭和|昭和]]3年）築地主要用地填海工程開始。
* 1933年（[[昭和|昭和]]8年）築地主建築的工程項目幾乎完成，並舉行峻工儀式。
* 1935年（[[昭和|昭和]]10年），於[[東京市|東京市]][[京橋區|京橋區]]（今東京都中央區）的[[築地|築地]]開設東京市中央批發市場。
* 1936年（[[昭和|昭和]]11年），東京魚市場株式會社開始舉行正式拍賣競標。
* 1954年（昭和29年），美國在[[比基尼環礁|比基尼環礁]]進行的[[氫彈|氫彈]]試爆實驗（[[城堡行動|城堡行動]]中的Bravo實驗）時受污染的[[第五福龍丸|第五福龍丸]]漁獲中的鮪魚、大青鯊等等於同年3月15日進貨到築地市場，因存在這些含有高[[放射性|輻射]]的魚獲而導致市場陷入混亂。拍賣被中斷，剩下的海產亦沒有商家願意開價競投。行政指示將受污染的海產埋在場內，在埋葬地點興建「原爆鮪魚」之墳，向現世人流傳那場災難（截至2006年，因應市場重修等的需要而暫時移設到夢之島第五福龍丸展示館）。現在市場的外牆掛有紀念牌，據說是為了不阻礙往來而改成鐵板的。
* 1995年（[[平成|平成]]7年），隨著[[都營大江戶線|都營地鐵大江戶線]]（當時的都營12號線）[[築地市場站|築地市場站]]的工程而需要調查市場的地底，受到[[傳媒|傳媒]]報導「會不會發掘這些屍體」而成為新聞焦點，但結果並無發掘到任何的屍體。
* 1995年3月20日，[[奧姆真理教|奧姆真理教]]發動[[東京地鐵沙林毒氣事件|地鐵沙林毒氣事件]]，在聚集人數眾多（當時來說）的[[築地站|築地站]]亦最多受害者死傷，附近的[[聖路加國際醫院|聖路加國際醫院]]由於接收多數需要急救的傷者而整天陷入混亂。
* 2000年（平成12年）12月12日，隨著[[都營地下鐵|都營地下鐵]][[大江戶線|大江戶線]]全線通車的同時，[[築地市場站|築地市場站]]亦開始投入運作（場內設有出入口）。

== 構造設施 ==
[[File:Sushi_restaurant_by_raisin_bun_in_the_Tsukiji_fish_market,_Tokyo.jpg|225px]]
[[File:Osakana_hukyucenter.jpg|225px]]

; 正門
: 向著[[新大橋通|新大橋通]]的築地市場門口。位於[[警視廳|警視廳]][[築地警察署|築地警察署]]中央市場和[[築地市場站|築地市場站]]的A1出口。隔街為{{link-ja|國立癌症研究中心|国立がん研究センター}}中央醫院與[[朝日新聞|朝日新聞]]東京本社。

; 青果門
: 由於附近有處理蔬果的拍賣場而得名的門口。很多運送蔬果的貨車通行。

; 勝鬨門
: 毗鄰[[勝鬨橋|勝鬨橋]]的築地市場門口。附近是市場範圍內的「魚類普及中心資料館」、「築地市場福利會館」，還有市場範圍外的[[勝閧橋|勝閧橋]]和收集該區資料並對外公開的「勝鬨橋資料館」、隔著[[晴海通|晴海通]]的[[Nichirei|Nichirei]]總公司。

; 海幸橋門
: 鄰接波除神社的築地市場門口。名字來源於架在[[築地川|築地川]]支流上的拱橋「海幸橋」，現在支流被填造地，作為停車場由中央區管理經營，橋的結構性物件橋桁與欄干正被撤除中。

; 市場橋門
: 在向著新大橋通的門口中最東面（或許是北）的門口，因附近是市場橋而得名。由於通道狹窄而且常作為後門，只作為離場方向通行。

; 相關事業者經營所
: 跟鮮魚和蔬果沒直接的關係的一些店舖正遷入此，一列排至9號楝。

; 鐘樓
: 雖然曾在東京都的建築頂上，但現在已經撤除。目前通道名稱仍保留為[[時計塔通|時計塔通]]。

; 進貨保管所
: 在場內有數個供中間商等寄存已購入商品的地方。在這裏聚集貨物，往停靠著的汽車上載貨物並送到商店或飲食店。常被稱為「潮待茶屋（等待潮漲的茶莊）」或簡稱「茶屋」。名稱源於魚河岸還在日本橋時，需使用圍繞在[[東京市|東京市]]四周的河川、運河等的水路並使用船上載魚獲，等到滿潮時才開始運送工作而來。使用茶屋是必需要有合同的。

; 東京都市場衛生檢查所
: 屬檢驗部門的[[食品衛生監視員|食品衛生監視員]]負責在市場內流通的食品的監視和檢驗，以及進行對業者的衛生指導。歸東京都福利保健局管轄，足立市場和大田市場都有分所。

; 場外市場商店街
: 築地市場的周圍亦有很多以採購者為對象的店鋪，形成被叫做「場外」的商店街。相對於這邊，築地市場則被叫做「場內」。場外比起場內較多普通客和旅遊客。

; 波除神社
: 與築地市場的海幸橋門鄰接的[[神社|神社]]。正如字面上說亦有「填海地」別稱的「築地」地名一樣，這附近一帶都是填海地區。雖然早在德川家康的時代，萬治元年（1658年）時就從[[築地本願寺|築地本願寺]]開始填海造地，但接連不斷因大浪而使工程進展困難。但是自從把在海中發現的五穀神像當神來供奉之後，風浪就平靜下來工程亦順利完成。這就是「波除」的來源，而且亦一直信仰著消災和航海安全的神。又，從築地市場鄰接著的地方可看到，有「海老塚（蝦的墳）」、「鮨塚（生魚墳）」、「玉子塚（雞蛋墳）」等供奉與海鮮或食品有關連的亡魂的墳墓。

== 作為旅遊點的市場 ==
[[File:Storage_and_selling_aisle_at_the_Tsukiji_fish_market,_Tokyo,_Japan.jpg|225px]]
[[File:Tsukiji_fish_market_-_Flickr_-_GregTheBusker_(1).jpg|225px]]

由於築地市場有很多新鮮食品和食店，食家頻頻透過[[大眾傳播|大眾傳播]]做宣傳拍攝，越來越多的普通的參觀客和購買客到訪。現在不僅是東京的旅遊指南，連以外國人為對象的指南亦將築地納入在內，成為日本的旅遊點之一。

並且，築地市場別稱為場內市場，而毗鄰築地市場的商店街則稱為場外市場；市場範圍內主要是以東京都管轄的批發商、中間商和相關業者這些向業界銷售為前提的店，而場外市場與普通的商店街一樣較多以普通客和旅遊客為對象的店，遊客通常會在[[新大橋通|新大橋通]]和[[晴海通|晴海通]]的十字路口起至市場為止的地區遊覽。
* 在休市日，場內大部份的店舖都不營業（場外亦大致上跟隨）。
* 上午時場內大部份的店都會打烊。
* 星期日、[[假日|假日]]基本上是休市。
* 除星期日外包括假日在內每月大約擬定兩次不定期休市，平日會在星期三休市（休市日魚店和飲食店的商品會有在前一天進貨的時候）。
* 除年尾年初和盂蘭盆會以外原則上不會3連休（2連休或以下之後開市）。
* 駛進場內的車輛需預先進行登記，非登錄車輛禁止入場。
* 上午時計程車等亦禁止入場。
* 附近很少停車場，最好儘可能使用公共交通工具。
* 最近的車站是[[都營地下鐵大江戶線|都營地下鐵大江戶線]]的[[築地市場站|築地市場站]]，比較近的還有[[東京地下鐵日比谷線|東京地下鐵日比谷線]]的[[築地站|築地站]]。
* 有從JR[[新橋站_(日本)|新橋站]]往築地市場場內的[[都營巴士|都營巴士]]（順帶一提乘都營巴士是最短的交通線路）。

但是，隨著普通客的猛增，違反市場規則和禮貌的參觀者和普通客亦見增加，引起阻障市場本來的業務的問題，便禁止參觀部份的拍賣。
作為普通客到訪市場的時候需注意以下事項：
* 由於汽車、[[堆高機|堆高機]]等車輛不時往來，一定要走在通道的旁邊。
* 集體行動或在食店排隊等情況時，不要妨礙通道和干擾旁邊的店鋪等等。
* 路面上會有難行的地方和濕滑的地方，不要穿高跟鞋到訪。
* 由於會處理海鮮，穿著被水弄濕了也沒打緊的服裝較好。
* 注意周圍環境，讓路給拿著沉重的貨物的人和正在趕忙的人。
* 由於連休前會非常擠擁，最好暫時不要到訪。
* 拍照時不可使用[[閃光燈|閃光燈]]，以免妨礙拍賣作業。

關於參觀場內和休市日相關的事，可參看都的官方網頁 [https://web.archive.org/web/20080524032517/http://www.shijou.metro.tokyo.jp/kengaku/02_01.html] 來查詢。

以[[東京都中央批發市場|中央批發市場]]來說有很強的如聚集的貨品分、決定價格的公共市場性質，多以原本是對業者（小規模店、飲食店，日式餐館）的批發價格來銷售，與上野的[[阿美橫丁|阿美橫丁]]等不同基本上沒有以過度減價為由的交涉而作出設定價格。亦有店舖以旅客或普通客身份，採購商品本身是小批量買入的業者亦都很多，雖說作為一般消費者身份並不會被拒絕，但與阿美橫丁等混在一起時亦很多時會被謝絕。

== 變遷 ==
{{main|豐洲市場}}
築地市場建築之初是為[[鐵路列車|鐵路列車]]運輸而設計的，只有狹小的貨車停泊空地。由於築地市場處理數量的擴大（2005年中處理2140公噸為日本最多），相應設施變得窄小且從1935年啟用至今的設施亦破損，以銀座等靠近築地那樣環境條件佳等作為借鑑，因而檢討於2012年遷移到[[東京都|東京都]][[江東區_(東京都)|江東區]]的[[東京瓦斯|東京瓦斯]]工場遺址的[[豐洲市場|豐洲新市場]]。作為東京方面與築地市場業界的協議機關，而建立新市場建設協議會，並於2004年7月訂定「豐洲新市場基本計劃」。

時任[[東京都知事|東京都知事]][[石原慎太郎|石原慎太郎]]認為「築地又舊又不清潔，若為都民與消費者的利益著想，市場不能再維持下去。也沒其他適合地點」而沒有更改遷移計劃的打算，並表明有將遷移後的築地遺址作為正在進行招聘活動的[[2020年夏季奧林匹克運動會|2020年東京奧運]]的媒體中心的構思<ref>每日新聞　2007年3月10日</ref>。

但是，已弄清楚將要遷移到的地方由於曾設置[[東京瓦斯|東京瓦斯]]的施設而受到[[土壤污染|土壤污染]]。雖然[[東京都|東京都]]等方面，因為會在將要遷移到的用地的地面上摞上4.5米未受污染的土壤，並舖上[[瀝青|瀝青]]，認為沒有問題，但仍有以“豐洲正被污染”的反對遷移運動。證實此事的是，都於2007年10月6日發表的調查結果中表明地下水中[[苯|苯]]、[[氰化物|氰化物]]、[[鉛|鉛]]和[[砷|砷]]的污染物，以及土壤中的苯、氰化物、砷的污染物含量超標。

又，石原都知事於2006年9月8日的定期記者會中會見記者時說到[[日本放送協會|NHK]]總部遷移到築地市場現址的發言雖然引起了轟動，但NHK以「沒有這樣的計劃」徹底否定此事<ref>{{cite web |url=http://www.zakzak.co.jp/gei/2006_09/g2006091118.html |title=存档副本 |accessdate=2008年5月30日 |deadurl=yes |archiveurl=https://web.archive.org/web/20080514162937/http://www.zakzak.co.jp/gei/2006_09/g2006091118.html |archivedate=2008年5月14日 |df= }}</ref>。

另外，本來東京都就反對築地市場的遷移地點定為豐洲，而是想重新修建築地市場。

除此之外石原都知事在2007年的都知事選舉中出席當選的會面時表示，由於築地市場存在大量的[[石棉|石棉]]，因此作出遷移是必要的。築地市場「場內市場」搬到豐洲，但「場外市場」則留守築地大本營，並增建新的商業設施，命名為「築地魚河岸」，繼續傳承築地孕育出的食文化<ref>{{cite web|url=http://travel.ulifestyle.com.hk/DetailNews.php?id=ADsRYxEsA3cMLA&utm_source=utravel&utm_campaign=home&utm_content=slider3&utm_medium=title|language=zh|title=東京築地仲有海鮮魚生食「築地魚河岸」明年開業|publisher=[[U_Travel|U Travel]]|date=2015年10月12日}}</ref>。

=== 搬遷 ===
2015年7月17日，東京都政府與水產品、蔬果批發、中盤商和零售業等市場業者代表協調後，決定築地市場自2016年11月3日休市四天搬遷、11月7日啟用豐洲市場<ref>{{cite news|url=http://asahichinese.com/article/news/AJ201507200010|title=丰洲新市场接棒筑地 明年11月7日开业|publisher=朝日新聞|date=2015年7月20日|deadurl=yes|archiveurl=https://web.archive.org/web/20150721230743/http://asahichinese.com/article/news/AJ201507200010|archivedate=2015年7月21日}}</ref>；但[[小池百合子|小池百合子]]在2016年接任東京都知事後，在同年8月31日宣布因豐洲市場存在諸多問題，延後築地市場移轉豐洲的時間<ref>{{cite news|url=http://mainichi.jp/articles/20160831/k00/00e/010/249000c|title=小池都知事：豊洲市場移転延期を正式発表「三つ疑問ある」|publisher=每日新聞|date=2016年8月31日}}</ref>。經過一年擱置後，東京都政府與市場業者代表最終在2017年12月20日，決定築地市場在2018年10月搬遷、10月11日啟用豐洲市場<ref>{{Cite news|url=https://www.nikkei.com/article/DGXMZO24852030Q7A221C1MM0000/|script-title=ja:豊洲移転、18年10月11日に決定　東京都|language=ja|date=2016-12-20|accessdate=2018-06-28|work=[[日本經濟新聞|日本經濟新聞]]}}</ref><ref>{{Cite news|url=http://www.shijou.metro.tokyo.jp/press/2017/1220.html|script-title=ja:豊洲市場の開場日の決定及びこれに伴う平成３０年東京都中央卸売市場における臨時休開場日の変更について|language=ja|date=2016-12-20|accessdate=2018-06-28|work=[[東京都中央批發市場|東京都中央批發市場]]}}</ref>。

=== 未來用途 ===
日本東京都廳計劃將築地魚市場舊址，先改建為[[2020年東京奧運會|2020東京奧運會]]的臨時停車場，供運動員及教練使用。當局於2019年1月23日公佈進一步計劃草案，預想在奧運後把土地劃分成多區，除興建國際會議中心和展覽館，亦希望在該處興建酒店和吸引遊客的設施。整個工程預計在2030年至2040年完成。地塊會變成國際會議中心、展覽館、高級酒店及餐廳等設施的集中地，以「建立及推展東京這個品牌」。有關公眾諮詢會截至3月份<ref>[https://hk.news.appledaily.com/international/daily/article/20190124/20597880 築地市場舊址　擬改建會議中心] 蘋果日報 2019-01-24</ref><ref>[http://www.tokyo-np.co.jp/article/national/list/201901/CK2019012402000136.html 築地跡地に国際会議場　「食」構想消え「稼ぐ」場に] 2019年1月24日 朝刊</ref>。
