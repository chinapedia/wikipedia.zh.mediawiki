'''築地市場站'''（{{jpn|j=築地市場駅|hg=つきじしじょうえき|rm=Tsukiji-shijō eki|f=y}}）是位於[[日本|日本]][[東京都|東京都]][[中央區_(東京都)|中央區]][[築地|築地]]五丁目，屬於[[東京都交通局|東京都交通局]]（[[都營地下鐵|都營地下鐵]]）[[大江戶線|大江戶線]]的[[鐵路車站|鐵路車站]]。站名併稱是「[[朝日新聞社|朝日新聞社]]前」。[[車站編號|車站編號]]是'''E 18'''。

== 歷史 ==
* 2002年（[[平成|平成]]12年）12月12日：開業。
* 2011年（平成23年）9月24日：[[月台門|月台門]]啟用。

== 車站構造 ==
本站是一[[地下車站|地下車站]]，月台為[[島式月台|島式月台]]1面2線設計。

== 車站周邊 ==

[[File:Tsukiji_as_seen_from_Shiodome.jpg|thumb]]
如站名所示，本站鄰[[築地市場|築地市場]]，站內有時也能感受到[[海鮮|海鮮]]特有的氣味<ref>『鉄道ピクトリアル』2001年7月臨時増刊号 特集:東京都営地下鉄 pp93-94</ref>。
* [[築地站|築地站]]（[[東京地下鐵日比谷線|東京地下鐵日比谷線]]）－最接近本站的車站，但並非官方設定的換乘站。此外，規劃大江戶線時本站的暫定名稱亦是「築地」。
* [[銀座|銀座]]－本站月台鄰近銀座地區（近[[昭和通|昭和通]]），故本站某程度上是大江戶線的銀座玄關口。

=== 巴士路線 ===
最接近本站的巴士站是位於{{link-ja|東京都道、千葉縣道50號東京市川線|東京都道・千葉県道50号東京市川線|新大橋通}}上的「國立癌症研究中心前」停留所及「築地市場正門前」停留所。以下路線停兩個停留所，並由東京都交通局（[[都營巴士|都營巴士]]）營運
* 市01：往[[新橋站_(日本)|新橋站]]前（循環線，平日途經築地中央市場（不停築地市場正門前））

=== 水上巴士 ===
濱離宮內設有兩座碼頭，分別由{{link-ja|東京都觀光汽船|東京都観光汽船}}和[[東京都公園協會|東京都公園協會]]營運以下觀光船路線：
* 隅田川線：往[[東京港|日之出棧橋]]／往[[淺草|淺草]]
* 東京水邊線：往[[台場海濱公園|台場海濱公園]]／往{{link-ja|櫻橋 (東京都)|桜橋 (東京都)|櫻橋}}／往[[葛西臨海公園|葛西臨海公園]]／往[[兩國|兩國]]

== 更名的可能性 ==
隨著築地市場計畫搬遷到[[江東區_(東京都)|江東區]][[豐洲|豐洲]]地區，本站亦有需要更名。如果成事，將會是自1989年3月19日江戶橋站更名為[[日本橋站_(東京都)|日本橋站]]以來，再有都營地下鐵車站更名。但由於豐洲地區的[[臨海線_(東京臨海新交通)|百合鷗號]]原本已有「市場前站」，而且計畫搬遷的只有中央市場部分，築地場外市場將會留在原址，因此也有不更名的可能。
