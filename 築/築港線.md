'''築港線'''（{{jpn|j=築港線|hg=ちっこうせん|rm=Chikkō sen|f=y}}）是一條位於[[日本|日本]][[愛知縣|愛知縣]][[名古屋市|名古屋市]]的鐵路路線，隸屬[[名古屋鐵道|名古屋鐵道]]。該線在營運初期名為'''築港支線'''（{{jpn|j=築港支線|hg=ちっこうしせん|rm=Chikkō shisen|f=y}}），是[[常滑線|常滑線]]的支線<ref name="社史">{{Cite book|editor=[[名古屋鐵道|名古屋鐵道]]|year=1961|title=名古屋鉄道社史|asin=B000JAMKU4}}</ref>{{rp|697}}。
== 路線資料 ==
* 路線距離：1.5公里
* [[軌距|軌距]]：1,067毫米
* 站數：2個（包括起終點站）
* 複線路段：沒有（全線[[單線鐵路|單線]]）
* 電氣化鐵路：全線電氣化（直流1,500V）
* [[閉塞_(鐵路)|閉塞方式]]：票券閉塞式
* 最高速度：60公里/小時
== 歷史 ==
該線由名鐵其中一所前身公司{{tsl|ja|愛知電気鉄道|愛知電氣鐵道}}（愛電）興建，於1924年1月15日通車<ref>{{cite web|url={{NDLDC|2955598/7}}|title=地方鉄道運輸開始並営業哩程変更|work=官報|publisher=[[大藏省|大藏省]]印刷局|date=1924-02-26|accessdate=2017-03-19}}</ref>。其主要功用是貨運輸送，在通車當年（1924年）的下半年即錄得43,000[[公噸|公噸]]的貨運量，佔同期愛電全公司貨運量的23%<ref name="社史" />{{rp|164}}。在之後的數十年，築港線為愛電及合併後的[[名古屋鐵道|名古屋鐵道]]帶來可觀的貨運收入，直至[[日本國有鐵道|日本國鐵]]於1960─1970年代夥同[[名古屋港|名古屋港]]建造{{tsl|ja|名古屋臨海鉄道|名古屋臨海鐵道}}。此後，築港線因其貨運業務逐漸轉移至新鐵路，而轉型為[[名古屋港|名古屋港]]東部的通勤路線。

1990年代，該線旁邊設立了{{tsl|ja|HSST}}[[磁浮列車|磁浮列車]]試驗線。該試驗線在2004年廢止。[[東部丘陵線|東部丘陵線]]通車前，其使用車輛即在此試驗線測試<ref name="rail fan">{{Cite journal|author=太田健一|date=2003-02-01|title=モハユニ ■HSST実験線に愛知万博線の量産先行車|journal = RAIL FAN|issue=2|volume=50|publisher=鉄道友の会|pages=19}}</ref>。

== 服務概況 ==
現時該線於每天繁忙時間提供普通列車服務，並在部份日子的清晨或晚間加開臨時列車。所有班次（包括臨時列車）皆穿梭往返大江站及東名古屋港站，前往其他地區的乘客需在大江站轉乘常滑線<ref name="TT東名港平">{{cite web|url=http://www.meitetsu.co.jp/ekibetsu/timetable/timetable/tt221211.html|title=東名古屋港站平日時刻表|publisher=名古屋鐵道|accessdate=2017-03-19}}</ref><ref name="TT東名港休">{{cite web|url=http://www.meitetsu.co.jp/ekibetsu/timetable/timetable/tt221212.html|title=東名古屋港站假日時刻表|publisher=名古屋鐵道|accessdate=2017-03-19}}</ref>。
