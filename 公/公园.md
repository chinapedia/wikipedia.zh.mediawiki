'''公園'''是供公眾消遣遊憩的場所。公園可以指下列場所：
# 市街地的一區，栽植草木或設置[[噴水池|噴水池]]的遊憩場所。
# 市街地的一角，設置兒童遊樂器材或體育設施的兒童公園及體育公園，即[[遊樂場|遊樂場]]。
# 親近動植物和自然生態為目的的[[動物園|動物園]]、[[植物園|植物園]]、[[風景區|風景區]]或[[森林|森林]]遊樂區。
# 保護自然生態或史蹟的[[國家公園|國家公園]]、[[森林公園|森林公園]]、[[郊野公園|郊野公園]]、[[海岸公園|海岸公園]]。
# 環繞某個主題而建立的[[主題公園|主題公園]]。

== 起源 ==
早期的公園出現於[[波斯國|波斯國]]的苑囿，原目的為供騎射的馳道和遮蔽風雨的處所，美化後成為公園。公園在[[古希臘|古希臘]]時期，是露天集會場地，希臘人在露天場所從事運動、社交活動，其後更結合了[[藝術|藝術]]與[[宗教|宗教]]的功能。到[[文藝復興|文藝復興]]後期，樹林、[[浮雕|浮雕]]長廊、鳥舍和[[野生動物|野生動物]]成為此時期公園的特點。

而[[中國|中國]]的公園，則最早記載於[[南北朝|南北朝]]時期《[[北史|北史]]》景穆十二王傳：「任城王澄表減公園之地，以給無業。」而从宋朝起，就有[[私家园林|私家园林]]对公众开放的习俗，并沿袭至明清<ref>{{Cite web |url=http://news.163.com/17/0122/05/CBC4CGH900018AOP.html |title=私家园林的开放传统|accessdate=2017-12-20 |author=吴钩 |date=2017-01-22 |work=责任编辑：王晓易_NE0011 |publisher=网易，来源：南方都市报(深圳) |language=Zh-hans }}</ref>。不过[[中國|中國]]真正具备现代意义上的是属分享共用的公园，直到1865年在[[广州|广州]][[沙面|沙面]]租界才初见，是由分属英方的女王公园与法方的法国公园共同组成的一片岛内的公共绿地。

== 功能 ==
公园主要可以承担自然元素的供给，为民众消遣提供场所，拥有成为地域性中心的潜质。
