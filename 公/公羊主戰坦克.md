'''C1公羊坦克'''（C1 Ariete）是[[義大利陸軍|義大利陸軍]]的第三世代[[主力戰車|主力戰車]]。由義大利國內自行研製與生產，並於1995年開始服役至今。

== 研發 ==
義大利在1982年提出研制新型主戰坦克的計畫，以替換300輛舊式的[[M60巴頓|M60巴頓]]。由多家公司共同研製。1984年完成整體規劃及系统設計。該計畫共包括了兩種裝甲戰鬥車輛：'''[[半人馬裝甲車|半人馬裝甲車]]'''和'''C1主戰坦克'''。而公羊首輛原型車於1986年推出，1988年時已經完成六輛原型車。雖然是戰後第一次開發的國產坦克，但是大量採用120mm滑膛炮和複合裝甲等戰後世界先進技術，還算是比得上第三世代坦克的性能。
== 開發公司 ==
C1公羊坦克是由多間公司分別研製的計畫成品。

* [[奧托梅萊拉|奧托梅萊拉]]（Oto Melara）：開發主砲塔、主砲管
* [[菲亞特|菲亞特]]（FIAT）：車身
* [[依維柯|依維柯]]（Iveco）：動力裝置
* {{tsl|it|Officine Galileo|伽利略公司}}（Officine Galileo）：火控系統
* [[塞克爾|塞克爾]]（Aero Sekur SpA）：[[NBC|NBC]]核生化防護裝置

一輛公羊主戰坦克的制造成本為700萬美元。

== 武裝 ==

主砲為[[萊茵金屬|萊茵金屬]]設計的120mm44倍徑[[Rh-120滑膛砲|Rh-120滑膛砲]]，授權[[奧托梅萊拉|奧托梅萊拉]]公司生產，配用的彈種有尾翼穩定脫殼穿甲彈和[[破甲弹|破甲弹]]，可攜帶42發砲彈。指揮塔蓋上的7.62mm機槍，可由車長在車内操作射擊。砲塔兩側共8座多用途發射器。
裝甲採用全焊接車體與砲塔，砲塔正面與車體又配有複合裝甲，此外能掛載更多裝甲套組。
火控系統[[TURMS|TURMS]]（Tank Universal Reconfigurable Modular System）FCS具有[[獵殲|獵殲]]能力，附有雷射測距儀與紅外線熱影像儀。
另外具有核生化防護系統SP-180和[[英國宇航|英國宇航]]的雷射警示裝置。

== 動力 ==

公羊坦克的動力裝置為水冷式渦輪增壓柴油機，最大功率956千瓦，坦克的最大行駛速度為65KM/H。
傳動裝置為德國[[采埃孚|ZF公司]]製的LSG-3000自動變速箱。懸掛裝置為扭力桿式，並有附液壓避震器。

== 目前情況 ==

原本在1994年之前預定購買300輛的公羊坦克，由於義大利的國家財政危機和[[冷戰|冷戰]]終結，又在軍費預算中被優先削減。結果，1995年～2002年共計200輛生產，和原先計畫減少了許多。

目前正在進行'''公羊MK-2'''的研發，預計配置500輛。
