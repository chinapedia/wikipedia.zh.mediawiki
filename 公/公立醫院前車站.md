'''公立醫院前站'''（{{jpn|j=公立病院前駅|hg=こうりつびょういんまええき|rm=Kōritsu-byōin-mae eki|f=y}}）位於[[熊本縣|熊本縣]][[球磨郡|球磨郡]][[多良木町|多良木町]]大字多良木，是[[球磨川鐵道|球磨川鐵道]][[湯前線|湯前線]]的車站。

車站名稱的由來是位於站址東南約400公尺處的球磨郡公立多良木醫院。

== 車站構造 ==
岸式1面1線，可供4節車廂停靠的無人車站。有簡易站房和候車室，月台上有公共電話。沒有設置廁所。

== 車站周邊 ==
週邊是田園地帶，附近除了有古多良木的村落以外，也散落著商店。
同時，沿著國道219號，有[[九州產業交通|產交巴士]]的古多良木公車站。在球磨郡公立多良木醫院的附近也有公立多良木醫院前公車站。

*球磨郡公立多良木醫院 - 東南側。國道219號沿伸。
*[[國道219號|國道219號]] - 東南側。約400公尺。
*[[柳橋川|柳橋川]]
*柳橋川橋 - 國道219號的橋。
以下需經過柳橋川橋
*上球磨消防署
*横馬場簡易郵局
*多良木警察署

== 歷史 == 
*1989年10月1日 - 湯前線交由[[球磨川鐵道|球磨川鐵道]]營運時設立。
