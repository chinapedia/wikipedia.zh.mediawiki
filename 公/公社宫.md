'''公社宫'''（Palazzo Comunale）又名'''哥特宫'''（Palazzo Gotico）是[[意大利|意大利]]北部[[皮亚琴察|皮亚琴察]]的一座宫殿，现在为市政厅所在地。

== 历史 ==
该宫殿位于意大利北部[[波河|波河]]南岸城市皮亚琴察。1281年，[[归尔甫派和吉伯林派|皇帝党]]人Alberto Scoto在此建造宫殿，聘请四位皮亚琴察建筑师：卡尼亚诺瓦、内格里、贝尔曼和博尔盖托<ref>{{cite web |url=http://web2.comune.piacenza.it/benvenuti/piacenza-si-presenta/citta-storica/palazzo-gotico|title=Palazzo Gotico|website=web2.comune.piacenza.it|publisher=|accessdate=2015年11月5日}}</ref>。

宫殿原本应该是四边形的，但工程由于流行瘟疫停止。只有宫殿的北侧完成了。结果成了伦巴第哥特式风格的民用[[卵形|卵形]]建筑一个很好的例子。<ref>{{cite web |url=http://turismo.provincia.pc.it/it/scopri-il-territorio/arte-e-cultura/ville-dimore-teatri-storici/item/palazzo-gotico.html|title=Palazzo Gotico di Piacenza|website=turismo.provincia.pc.it|publisher=|accessdate=2015年11月5日}}</ref><ref>{{cite web |url=http://www.piacenzamusei.it/index/it/pagine/visita-al-palazzo-279/palazzo-gotico/.html|title=Palazzo Gotico - Visita al Palazzo|website=piacenzamusei.it|publisher=|accessdate=2015年11月5日}}</ref>。
 
内部有一个大的休息室，在1644年，成为一个剧场，由克里斯托弗Rangon设计。
