'''公园街道'''，是{{PRC admin/fullname|36/01/02/001/000|3}}下辖的一个[[乡镇|乡镇]]级行政单位。<ref>{{PRC admin/ref sgc|{{PRC admin/parent|36/01/02/001/000}}}}</ref>


== 简介 ==
1954年设立公园街道，因一旁的[[八一公园|八一公园]]而得名，是[[东湖区|东湖区]]政府驻地所在街道。

== 历史沿革 ==
1949年为上营坊街行政事务所，1954年成立公园街道，1960年改称公园公社，1978年分出部分辖区组建人民大道公社（今[[大院街道|大院街道]]）<ref>[http://www.xzqh.org/html/show/jx/9736.html 东湖区大院街道_行政区划网(区划地名网)]</ref>，1997年复称街道。

1997年时街道辖区为1.2平方千米，人口3.8万，省外办侧、市体委南、上营坊、皇殿侧北、广场北路西、江西宾馆侧、八一大道西、二附院侧、戴家巷南、戴家巷北、经堂巷第一、第二、第三、第四、区府西、环湖路南、北、佑民寺、花园角西、花园角东、南昌二中、江西省妇产医院、南昌军分区、省政府基建队、南昌有色治金设计院25个居委会；办事处驻地为民德路101号。<ref>[http://www.xzqh.org/html/show/jx/9741.html 东湖区公园街道_行政区划网(区划地名网)]</ref>

== 人口 ==
2000年[[中华人民共和国第五次全国人口普查|第五次全国人口普查]]公园街道总人口30157人，户籍人口27598人，居住在本地的户籍人口23480人<ref>[http://www.stats.gov.cn/tjsj/pcsj/rkpc/dwcrkpc/ 国家统计局：《2000年第五次人口普查数据》]</ref>。
