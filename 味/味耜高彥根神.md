'''味耜高彥根神'''（{{lang|ja|あぢすきたかひこね の かみ、あぢしきたかひこね の かみ}}）為[[日本神話|日本神話]]之[[神祇|神祇]]，《[[日本書紀|日本書紀]]》作味耜高彥根神；《[[古事記|古事記]]》記錄成阿遲鉏高日子根神、阿遲志貴高日子根神、阿治志貴高日子根神；《[[出雲國風土記|出雲國風土記]]》寫作阿遲須枳高日子命；此外還有阿遲鋤高日子根神、迦毛大御神（{{lang|ja|かも の おおみかみ}}）等別名。

阿遲鉏高日子根神乃是[[大國主神|大國主神]]及[[田心姬|田心姬]]（[[宗像三女神|宗像三女神]]之一）所生，同母胞妹為[[高姬|高姬]]。

== 概要 ==
=== 日本書紀之記載 ===
依據《日本書紀》卷第二神代之內容，在[[葦原中國|葦原中國]]平定後，'''味耜高彥根神'''昇天弔[[天稚彥|天稚彥]]之喪，因二者容貌相近，天稚彥的[[妻子|妻子]][[親屬|親屬]]靠過來攀牽其衣帶，又哭又笑地說天稚彥回來了。味耜高彥根神被誤認為死者，覺得觸霉頭，忿然大怒拔出[[十握劍#大葉刈（神戶劍）|大葉刈]]（又名神戶劍）劈開喪屋，落而爲山，成為[[美濃國|美濃國]]藍見川之上喪山。

=== 古事記之記載 ===
《古事記》上卷記載的故事和前段類似，'''阿遲鉏高日子根神'''前往弔唁[[天稚彥|天若日子]]之喪，但因二人容貌神似，天若日子之父、妻誤認阿遲鉏高日子根神為天若日子。阿遲鉏高日子根神覺得自己被誤認為死者而勃然大怒，拔出名為大量（亦名神度劍）的[[十握劍#大量劍（神度劍）|十拳劍]]切開靈堂，並將之踹往一旁。

=== 出雲國風土記之記載 ===
《出雲國風土記》神門郡高岸鄉條記錄，'''阿遲須枳高日子命'''年幼時日夜啼哭不止，於是讓祂居住在高大房屋內，並設梯子可以自由上下養育之。此外，阿遲須枳高日子命與天御梶日女（{{lang|ja|あまのみかぢひめ}}）育有[[雨神|雨神]]多伎都比古（{{lang|ja|たきつひこ}}）。

== 解說 ==
神名中的「耜（{{lang|ja|すき}}）」就是[[農具|農具]][[鋤頭|鋤頭]]；[[江戶時代|江戶時代]][[日本國學|國學]]專家[[本居宣長|本居宣長]]所撰之《{{link-ja|古事記傳|古事記伝}}》則認為「志貴（{{lang|ja|しき}}）」是磯城裡以石板鋪設路面的「石疊」。

別名「迦毛大御神（{{lang|ja|かも の おおみかみ}}）<ref>迦毛（{{lang|ja|かも}}）的漢字另可寫成「賀茂」。</ref>」意為賀茂社之神，亦即[[大和國|倭國]]葛城賀茂社的鴨氏祭祀之神。不過，有人認為鴨氏乃從[[出雲國|出雲國]]遷移至大和國。而《古事記》稱之為「迦毛大御神」，這種「大御神」的稱呼除此之外僅見於[[天照大神|天照大御神]]。

== 信仰 ==
祭祀味耜高彥根神的[[神社|神社]]主要有：
* 都都古別神社（[[福島縣|福島縣]][[東白川郡|東白川郡]][[棚倉町|棚倉町]]）
* 土佐神社（[[高知縣|高知縣]][[高知市|高知市]]）
* 鳴無神社（高知縣[[須崎市|須崎市]]）：主祭神為[[一事主神|一言主神]]。
* 高鴨神社（[[奈良縣|奈良縣]][[御所市|御所市]]）
* 鴨神社（[[岡山縣|岡山縣]][[玉野市|玉野市]]長尾）
* 阿須伎神社（[[島根縣|島根縣]][[出雲市|出雲市]]大社町）
* [[日光二荒山神社|日光二荒山神社]]（[[栃木縣|栃木縣]][[日光市|日光市]]）
