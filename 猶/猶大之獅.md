'''猶大之獅'''({{lang|en|'''Lion of Judah'''}})，亦作'''猶太之獅'''<ref>見《[[新約全書|新約全書]]·[[啟示錄|啟示錄]]》第五章第5節。有[[天主教|天主教]]中文譯本「猶大支派中的獅子」譯作「猶太之獅」，見[http://www.21music.org/NEW/BLESSING/NTestment/reveal/reveal01.htm#c5 天主教音乐网：默示錄]{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，是[[以色列十二支派|以色列十二支派]]當中[[猶大支派|猶大支派]]的象徵。作為昔日[[猶大國|猶大國]]的首都的[[耶路撒冷|耶路撒冷]]，它的市徽亦飾有猶大之獅。猶大之獅亦曾經是[[埃塞俄比亞|埃塞俄比亚]]的國徽，出現在[[埃塞俄比亞國旗|埃塞俄比亚国旗]]之上。此外，所羅門王騎「猶大之獅」的雕刻在埃塞俄比亚亦到處可見。

== 來源 ==
「猶大之獅」的說法源於《[[創世記|創世記]]》第四十九章第9節。當時，[[猶大|猶大]]的父親[[雅各|雅各]]臨終，他把他的12個兒子叫到身邊，並為他們一一祝福。當他為猶大祝福時，說了以下的話：

: <u>猶大</u>是個小獅子；
: 　我兒啊，你抓了食便上去。
: 你屈下身去，臥如公獅，
: 　蹲如母獅，誰敢惹你？
: <p align=right>-- [[和合本|和合本]]《舊約全書》‧《創世記》第四十九章第9節</p>

自此以後，猶大支派就以獅子作為其象徵。

== 猶大之獅與猶太教 ==

== 猶大之獅與基督教 ==

在[[基督教|基督教]]的傳統中，獅子通常都用來代表[[耶穌|耶穌]]。有不少基督教團體及差傳機構都使用「猶大之獅」作徽號或名稱。

在《[[新約全書|新約全書]]·[[啟示錄|啟示錄]]》第五章第5節：
{{quote|長老中有一位對我說：「不要哭！看哪，<u>猶大</u>支派中的獅子，[[大衞|大衞]]的根，他已得勝，能以展開那書卷，揭開那七印。」}}

另外，[[魯益師|魯益師]]在他的《[[那裡亞故事集|那裡亞故事集]]》裡，亦採用了獅子阿司能來代表那一個世界的[[救世主|救世主]]。這很可能亦是受到《啟示錄》的影響。

== 猶大之獅與拉斯特法里教派 == 

[[拉斯特法里教派|拉斯特法里教派]]是一個奉前埃塞俄比亞廢王[[海爾·塞拉西一世|海爾·塞拉西一世]]為[[上帝|上帝]]轉生的宗教。在這個教派裡，他們用「猶大之獅」來作為海爾·塞拉西一世的象徵。他們認為，海爾·塞拉西一世是猶大支派的後裔，有着[[大衛王|大衛王]]及[[所羅門王|所羅門王]]的直系血脈，亦是《啟示錄》裡所提及的「猶大支派中的獅子」。
