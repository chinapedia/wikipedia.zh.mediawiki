'''星系際恆星'''是不屬於任何一個[[星系|星系]]的恆星。科學界近年來對這些恆星有著許多的討論，普遍認為這些恆星的來源是星系碰撞的結果。

== 發現 ==
直到最近，恆星只存在於星系內仍是共同的信念。但是當1997年發現星系際恆星後，這個信念被推翻了<ref>{{cite web|url=http://hubblesite.org/newscenter/archive/releases/1997/02 |title=NewsCenter - Hubble Finds Intergalactic Stars (01/14/1997) - Introduction |publisher=HubbleSite |date=1997-01-14 |accessdate=2010-12-09}}</ref>。首度在[[室女座星系團|室女座星系團]]發現可能有上兆顆的這種恆星存在<ref>{{cite web|url=http://hubblesite.org/newscenter/archive/releases/1997/02/text/ |title=NewsCenter - Hubble Finds Intergalactic Stars (01/14/1997) - Release Text |publisher=HubbleSite |date=1997-01-14 |accessdate=2010-12-09}}</ref>。

== 形成 ==
雖然，這種恆星的行程帶著點神祕，但最常見的理論是這些恆星是由兩個或更多個星系碰撞，然後有些恆星被拋入更為廣闊的太空之內。

== 觀測的歷史 ==
在1997年，[[哈伯太空望遠鏡|哈伯太空望遠鏡]]發現在[[室女座星系團|室女座星系團]]的星系之間有著大量的星系際恆星。稍後，在1990年代末期，科學家發現[[天爐座星系團|天爐座星系團]]也有另一個星系際恆星的集團。

== 質量 ==
雖然不知道這些恆星的精確質量，但估計這些恆星的質量是[[室女座星系團|室女座星系團]]質量的10%，這意味著這些恆星的質量大於室女座星系團中2,500個星系中的任何一個單獨的星系。

== 位置 ==
第一顆被發現的星系際恆星位於銀河系所在的[[室女座星系團|室女座星系團]]。這些恆星形成的大質量群與最靠近的星系距離大約在30萬光年。
