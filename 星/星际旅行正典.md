'''星际旅行正典'''（{{lang-en|Star Trek canon}}）一般来说，包括电视剧《[[星际旅行：初代|星际旅行：初代]]》、《[[星际旅行：下一代|星际旅行：下一代]]》、《[[星际旅行：深空九号|星际旅行：深空九号]]》、《[[星际旅行：航海家号|星际旅行：航海家号]]》、《[[星际旅行：进取号|星际旅行：进取号]]》、《[[星际迷航：发现号|星际迷航：发现号]]》以及[[星际旅行电影列表|系列电影]]。<ref name="StarTrek.com">[http://www.startrek.com/startrek/view/help/faqs/faq/676.html What is considered Star Trek "canon"?] {{webarchive|url=https://web.archive.org/web/20100628174255/http://www.startrek.com/startrek/view/help/faqs/faq/676.html |date=2010-06-28 }} FAQ entry at the official [http://www.startrek.com/ Star Trek website]. URL retrieved 25th November 2006.</ref>2003年当时星际旅行的官方站点认为这定义并非一成不变，到底什么才算[[星际旅行|星际旅行]]{{link-en|正典 (虚构作品)|Canon (fiction)|正典}}其概念亦不是固定，对此采取开放态度<ref name="StarTrek.com" />；而[[派拉蒙|派拉蒙]]在2009年重啟電影設定歷史走向不同，居主導地位的[[CBS_(電視網)|CBS]]於2016年6月公開劃分'''主時間線'''（Prime Timeline）和'''凱文時間線'''（Kelvin Timeline）以作區別。

== 主時間線 ==
沿襲以往設定者歸於「主時間線」，主要作品有：
;電視劇
*《[[星际旅行：初代|星际旅行：初代]]》
*《[[星际旅行：下一代|星际旅行：下一代]]》
*《[[星际旅行：深空九号|星际旅行：深空九号]]》
*《[[星际旅行：航海家号|星际旅行：航海家号]]》
*《[[星际旅行：进取号|星际旅行：进取号]]》
*《[[星际迷航：发现号|星际迷航：发现号]]》
;電影
*《[[星际旅行电影列表#主時間線|星艦迷航記]]》系列（1979年－2002年）

== 凱文時間線 ==
重啟電影設定因未來人干涉過去歷史而造成時間線變動，[[CBS_(電視網)|CBS]]方面認為需有可參考標的，由負責{{link-en|星际旅行百科全书|The Star Trek Encyclopedia}}的奧田夫婦（Michael & Denise Okuda）構思命名，以聯邦星艦凱文號（U.S.S. Kelvin）被毀為轉折時間點<ref>{{cite web |url=https://twitter.com/CaptainGeko/status/745096154933714948 |title=FYI, "Kelvin Timeline" is the official name for that universe timeline according to CBS. |publisher= |via=[[Twitter|Twitter]] |author=Al Rivera |language= |date= 2016-06-20 |accessdate=|archiveurl= |archivedate=}}</ref><ref>{{cite web |url=http://trekcore.com/blog/2016/06/star-trek-alternate-universe-finally-gets-official-name/ |title=STAR TREK Alternate Universe Finally Gets Official Name |publisher= TrekCore |via= |author= |language= |date= 2016-06-26 |accessdate=|archiveurl= |archivedate=}}</ref><ref>{{cite book |title=The Star Trek Book |author=Paul Ruditis | publisher=[[多林金德斯利|DK]] |page= 14 |type= |url=https://www.amazon.com/gp/reader/146545098X/ |date=2016-06-07 |isbn = 978-1-4654-5098-2 |language=|via=[[Amazon.com|Amazon.com]]}}</ref>，目前「凱文時間線」作品僅[[派拉蒙|派拉蒙]]自2009年重啟的電影系列：

*《[[星际旅行电影列表#凱文時間線|星际迷航]]》系列（2009年－）
