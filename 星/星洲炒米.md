'''星洲炒米'''是一種炒[[米粉|米粉]]，常見於[[香港|香港]]和[[馬來西亞|馬來西亞]]，在[[美國|美國]]、[[加拿大|加拿大]]和[[澳洲|澳洲]]的[[粵菜|粵菜]]餐館也有供應<ref>[http://home.meishichina.com/recipe-97665.html 星洲炒米粉]，美食天下，2012-11-2</ref>。星洲炒米的「星洲」是指[[新加坡|新加坡]]，然而在新加坡卻沒有這種名稱的炒米粉<ref>[http://www.shw365.com/mstd/mskt/200812/55167.html 星洲炒米的做法]，生活网</ref>。星洲炒米是香港常見的米粉食品，但有部分人士認為源自於馬來西亞的[[吉隆坡|吉隆坡]]，但香港的星洲炒米和馬來西亞的炒米是有明顯分別，香港的星洲炒米一定會加入咖哩，馬來西亞的炒米卻使用番茄醬和辣椒醬代替。

== 材料 ==
星洲炒米的材料通常包括[[蝦|蝦]]、[[叉燒|叉燒]]和[[雞蛋|雞蛋]]，並按照不同地區的烹調習慣，加入[[芽菜|芽菜]]、包菜丝、萝卜丝、[[辣椒|辣椒]]絲、[[蕃茄醬|蕃茄醬]]或[[咖哩|咖哩醬]]等食材。

== 起源 ==
星洲炒米的起源已不可考，一般認為加入咖哩的星洲炒米是出現在香港的炒米粉食品<ref>[http://the-sun.on.cc/cnt/news/20090920/00405_003.html 星洲無星洲炒米 香港無香港炒麵]，太陽報，2009-2-20</ref>，至於使用番茄醬調味的炒米粉，則有以下一些說法。

使用番茄醬代替咖哩的星洲炒米是起源自[[馬來西亞|馬來西亞]][[吉隆坡|吉隆坡]]的一道料理，如同在[[廣州|廣州]]的“[[揚州炒飯|揚州炒飯]]”，在[[揚州|揚州]]反而吃不到這種味道，星洲炒米在[[新加坡|新加坡]]也是並不常見。在[[吉隆坡|吉隆坡]]日治末期的其中一個午夜，一名大老闆將車子停在相熟的[[大排檔|大排檔]]前，要求檔主幫忙做一道為他充飢。當時該名檔主已準備收攤，但不敢得罪該名大老闆或希望能多一些生意，就把一些賣剩的碎[[米粉|米粉]]、[[叉燒|叉燒]]、[[芽菜|芽菜]]、[[蝦米|蝦米]]、[[雞蛋|雞蛋]]和[[辣椒|辣椒]]一起炒出一道雜碎炒米粉，結果得到該名老闆的大讚。在被問起該道料理的名稱時，檔主不好意思是一些碎料炒碎米粉，就隨口說了'''星洲炒米'''這名稱。接下來經過該老闆向友人推薦，使得這道“星洲炒米”成為該大排檔的招牌料理，名揚開來，流行於馬來半島。现在也流行于不少地区。

== 差別 ==
[[香港|香港]]的星洲炒米是[[茶餐廳|茶餐廳]]、[[港式快餐店|快餐店]]、[[酒樓|酒樓]]和[[大排檔|大排檔]]的常見的[[午餐|午餐]]和[[晚餐|晚餐]]食品，港式星洲炒米也隨著香港移民而傳至海外各地的[[唐人街|唐人街]]，只要在[[香港人|香港人]]開設的[[酒樓|酒樓]]茶市及[[茶餐廳|茶餐廳]]，幾乎都可以吃到這道料理。港式星洲炒米的食材通常包括[[叉燒|叉燒]]、[[蝦仁|蝦仁]]、[[芽菜|芽菜]]、[[雞蛋|雞蛋]]、[[洋蔥|洋蔥]]和[[辣椒|辣椒]]絲<ref>[http://programme.tvb.com/lifestyle/pleasureandleisure/i_info/124236/7/1316 「今晚食乜餸」星洲炒米粉]，TVB - [[都市閒情|都市閒情]]，2012-3-1</ref>，並會加入[[咖哩粉|咖哩粉]]或咖哩醬，令炒米粉變成金黃色，並充滿香辣的味道<ref>[http://lifestyle.etnet.com.hk/column/index.php/leisure/homecooking/7069 星州炒米]，經濟通，2011-11-23</ref>。

[[馬來西亞|馬來西亞]]人的星洲炒米除了[[叉燒|叉燒]]、[[蝦仁|蝦仁]]、[[芽菜|芽菜]]、[[雞蛋|雞蛋]]、[[洋蔥|洋蔥]]、[[辣椒|辣椒]]外，還會加入[[番茄醬|番茄醬]]及[[辣椒醬|辣椒醬]]來調味，製成色澤帶點淡紅色，味道帶點酸、辣及甜的星洲炒米，做法類似廈門炒米。
