'''星期'''又作'''週'''或者'''礼拜'''，是一个[[时间|时间]][[单位|单位]]，每星期有7[[日|日]]。

中國古代[[七曜|七曜]]一詞，起源於先秦時代的天文占星學，又稱七政；漢朝的數學家[[劉洪|劉洪]]著作七曜術講述天文算學，《後漢書志二律历》:「正月甲子朔旦冬至,七曜之起,始於牛初。」；[[東晉|東晉]][[范甯|范甯]]的《春秋穀梁傳序》：“陰陽為之愆度，七曜爲之盈縮。”[[楊士勳|楊士勳]]疏：“謂之七曜者，日月五星皆照天下，故謂之曜。”五星指火、水、木、金、土。在[[唐代|唐代]]以前還未有使用七星指代輪回的7日或一週；[[古巴比伦|古巴比伦]]人首先使用7日為一週的[[时间|时间]]单位，后来[[犹太人|犹太人]]把它传到[[古埃及|古埃及]]，又由古埃及传到[[罗马|罗马]]，公元3世纪以后，就广泛地传播到[[欧洲|欧洲]]各国。[[唐朝|唐朝]]时期在[[西域|西域]]和中国的交流中传入了中土獲得發展，再传入[[朝鮮半島|朝鮮半島]]，[[日本|日本]]。光緒三十一年（1905年）由袁嘉穀所定用「星期」代替「曜日」，到了[[民国|民国]]以後星期被廣泛使用。

== 東亞 ==
{{Main|七政}}
中國的“七曜”和西方以七星命名的七天相對應，大約發生于[[唐朝|唐朝]]。

[[印度|印度]]佛經《宿曜曆經·七曜直日曆品第八》：

{{quote|
“夫七曜者，所谓日月五星，下直人间。一日一易，七日周而复始。……
故今列諸國人呼七曜如後： 　　
: 日曜太陽，胡名蜜，波斯名曜森勿，天竺名阿儞(泥以反)底耶(二合)。 　　
:  月曜太陰，胡名莫，波斯名婁禍森勿，天竺名蘇上摩。
: 火曜熒惑，胡名云漢，波斯名势森勿，天竺名糞盎聲哦囉迦盎。
: 水曜辰星，胡名咥(丁逸反)，波斯名掣森勿，天竺名部(引)陀。　
: 木曜嵗星，胡名鶻勿，波斯名本森勿，天竺名勿哩訶娑跛底(丁以反)。 　　
: 金曜太白，胡名那歇，波斯名數森勿，天竺名戌羯羅。 　　
: 土曜鎮星，胡名枳院，波斯名翕森勿，天竺名賒乃以室折囉。 　　
右件七曜，上運行于天，下直于人间。其精靈神驗、内外典籍，具備自南西北三方諸國。”}}

其中“具備自南西北三方諸國”一語，可見當時佛家宣揚，七星命名七天為一周的方法當時已通行於南、西、北，希望東方的大唐也採納這種紀日法。

唐朝时，七曜和二十八宿结合生成曜日。《宿曜经下》曰：“太陽直日（日曜日也），月與尾合。太陰直日（月曜日也），月與心合。火曜直日，月與壁合。水曜直日，月與昴合。木曜直日，月與井合。金曜直日，月與張合。土曜直日，月與亢合。已上名金剛峰日。”[[二十八宿|二十八宿]]及其对应的动物与七曜相配，组成[[值日星君|值日星君]]：“角木蛟、亢金龙、氐土蝠、房日兔、心月狐、尾火虎、箕水豹、斗木獬、牛金牛、女土貉、虚日鼠、危月燕、室火猪、壁水㺄、奎木狼、娄金狗、胃土彘、昴日鸡、毕月乌、觜火猴、参水猿、井木犴、鬼金羊、柳土獐、星日马、张月鹿、翼火蛇、轸水蚓”。老黄历的值日星君就是曜日。'''日曜日'''是[[星期日|星期日]]，'''月曜日'''是[[星期一|星期一]]，'''火曜日'''是[[星期二|星期二]]，'''水曜日'''是[[星期三|星期三]]，'''木曜日'''是[[星期四|星期四]]，'''金曜日'''是[[星期五|星期五]]，'''土曜日'''是[[星期六|星期六]]。中國清末前使用七曜日為一周，在[[中華民國|民國]]成立後改稱星期，其中的「星」字便是指這七曜，但在[[日本|日本]]、[[大韓民國|-{zh-hans:韩国; zh-hant:南韓;}-]]和[[朝鲜|-{zh-hans:朝鲜; zh-hant:北韓;}-]]仍沿用曜日名稱。[[台灣|台灣]][[日治時期|日治時期]]由日本統治，因此亦使用[[七曜|七曜]]，但戰後由中華民國統治後改成星期。

== 西方 ==
[[File:Weekday_heptagram.ant.png|thumb]]
西方语言中星期的表达法大多源于拉丁语。而[[拉丁语|拉丁语]]的情况可分为基督教崛起前和崛起后两种：

基督教崛起前，星期日为“太阳日”，星期一为“月亮日”，星期二为“火星日”（[[瑪爾斯|瑪爾斯]]之日），星期三为“水星日”（[[墨丘利|墨丘利]]之日），星期四为“木星日”（[[朱比特|朱比特]]之日），星期五为“金星日”（[[維納斯|維納斯]]之日），星期六为“土星日”（[[薩圖爾努斯|薩圖爾努斯]]之日）。五颗行星的名称都对应[[古罗马神话|古罗马神话]]中神的名字。

至於天體與特定日的排列順序，則是與[[地心說|地心說]]及占星學有關{{r|數字}}。過去在地心說的觀點下，古人認為天體繞行地球的速度從慢到快依序是土星、木星、火星、太陽、金星、水星、月亮{{r|數字}}。而占星學認為每個天體主掌一個小時，而每天第一個小時的主星便是該日的日主星{{r|數字}}。如某日第1小時是土星支配（土星之日），則該日的第8、15、22小時也是土星支配，第24小時會是火星，而次日第1個小時便會是太陽支配（太陽之日）{{r|數字}}。以此類推則得到土星→太陽→月亮→火星→水星→木星→金星的順序{{r|數字}}。

基督教崛起后，星期一、星期二、星期三、星期四、星期五皆未变，唯星期六改为“安息日（dies Sabbati）”，来自古希伯来语שבת (šabbāṯ)，星期日改为“主的日（dies Dominica）”。

[[法语|法语]]直接采用基督教时代拉丁语的名称（lundi、mardi、mercredi、jeudi、vendredi、samedi、dimanche），参见法语词条[https://fr.wikipedia.org/wiki/Semaine Semaine]。

[[英语|英语]]则完全继承基督教时代前拉丁语的表达，只不过将代表'''前四个'''行星的神换成[[北欧神话|古日尔曼人神话]]中的神：星期二（Tuesday）－战神“[[提尔|提尔]]（Tyr）”的日子，星期三（Wednesday）－（主神）“[[奧丁|奧丁]]（Wodan）”的日子，星期四（Thursday）－雷神“[[索爾|索爾]]（Thor）”的日子，星期五（Friday）－女神“[[弗丽嘉|弗丽嘉]]（Frigg）”的日子，而星期六（Saturday）则沿用罗马神－农神“[[萨图尔努斯|萨图尔努斯]]（Saturnus）”。

德語情況與英語相似，但是星期三「Mittwoch」的意思是「中間之日」（在一週七天的中間），星期六「Samstag」則源自希伯來語的安息日「Sabbat」{{r|數字}}。

=== 宗教 ===
西方的星期最早來自于[[蘇美爾|蘇美爾]]。蘇美爾人建立神塔對神進行禮拜，他們認為有七位星神輪流“值勤”，分別是：太陽神（星期日）、月亮神（星期一）、火星神（星期二）、水星神（星期三）、木星神（星期四）、金星神（星期五）、土星神（星期六），於是建立了七天一星期的制度。蘇美爾以後，星期制被[[古巴比倫|古巴比倫]]人繼承。

古[[猶太人|猶太人]]受到兩河流域文明影響，也接受了星期制。後來，星期的概念隨猶太宗教衍生出來的[[基督教|基督教]]傳播到歐洲。「星期一」、「星期二」……在口語也以「禮拜一」、「禮拜二」……方式稱呼，星期日則稱為「禮拜天」，一個星期又稱為一個「禮拜」。因此，[[闪米特宗教|闪米特宗教]]均包含星期制，但又各有不同：[[犹太教|犹太教]]以星期六为[[安息日|安息日]]，[[基督教|基督教]]以[[星期日|星期日]]为礼拜日，[[伊斯兰教|伊斯兰教]]将星期五定为重大礼拜的[[主麻日|主麻日]]。

在不同地区，一星期的开始时间并不完全一致。在許多以英語作為官方語言國家和[[以色列|以色列]]、[[日本|日本]]、[[香港|香港]]和[[臺灣|臺灣]]<ref>參見中華民國行政院人事行政局公布之政府行政機關辦公日曆表</ref>，一般认为一个星期开始于[[星期日|星期日]]。[[埃及人|埃及人]]的一星期是从[[星期六|星期六]]开始的。[[中國大陸|中國大陸]]以及多數歐洲國家都以星期一為一星期的第一天。

在中国大陆，一般认为[[周末|周末]]包括了星期六和星期天。

[[国际标准化组织|国际标准化组织]]在[[ISO_8601|ISO 8601]]《数据存储和交换形式·信息交换·日期和时间的表示方法》中将星期一定为一星期的第一天。中华人民共和国国家标准GB/T 7408-2005《数据元和交换格式·信息交换·日期和时间表示法》和国际标准相同，也以周一为一周的第一天。
