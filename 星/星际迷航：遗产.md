{{vgname|星际迷航：遗产|en=Star Trek: Legacy}}是由[[美国|美国]][[新英格兰工作室|新英格兰工作室]]开发、[[贝塞斯达软件公司|贝塞斯达软件公司]]发行的一款基于[[星际旅行|星际旅行]]设定的[[电子游戏|电子游戏]]。星际迷航：遗产于2006年年底在北美和欧洲正式发行[[Microsoft_Windows|Windows]]和[[Xbox_360|Xbox 360]]平台版本，这款游戏抢在2006年发行也是为了纪念星际旅行40周年<ref>{{cite web|url=http://startrek.bethsoft.com/news/pressrelease_121506.html |title=Star Trek: Legacy |publisher=Bethesda Softworks |work=Startrek.bethsoft.com |accessdate=2006-12-21 |deadurl=yes |archiveurl=https://web.archive.org/web/20080308143345/http://startrek.bethsoft.com/news/pressrelease_121506.html |archivedate=2008-03-08 }}</ref>。

在游戏中，玩家可以控制一个最多四艘[[星舰|星舰]]的舰队，游戏中的星舰囊括了[[星际旅行：进取号|星际旅行：进取号]]、[[星际旅行：初代|星际旅行：初代]]和[[星际旅行：下一代|星际旅行：下一代]]三个不同时代的[[星际舰队|星际舰队]]星舰。游戏的主要目标是与敌对种族的战争和探索宇宙。

[[Microsoft_Windows|Windows]]平台版本的星际迷航：遗产并没有受到星迷和游戏评论家的好评<ref name="gamespotReview">{{cite web |url=http://www.gamespot.com/pc/sim/startreklegacy/review.html|title=Gamespot Legacy review}}</ref>，在[[GameRankings|GameRankings]]上，星际迷航：遗产的平均得分仅为4.8分<ref>{{cite web|url=http://www.gamerankings.com/htmlpages2/931494.asp?q=star%20trek%20legacy|title= Game Rankings}}</ref>。

==设定==
===单人游戏===
在星际迷航：遗产的单人游戏中，玩家将经历[[星际旅行：进取号|星际旅行：进取号]]、[[星际旅行：初代|星际旅行：初代]]和[[星际旅行：下一代|星际旅行：下一代]]这三个时代，每个时代都有相应的任务和旗舰船，并最多能控制四艘[[星舰|星舰]]组成的舰队。每一个时代都有一个敌对种族，进取号的敌对种族为[[罗慕伦人|罗慕伦人]]，初代为[[克林贡人|克林贡人]]，下一代为[[博格人|博格人]]。在下一代的时代中，玩家还可以体验到[[星际旅行：深空九号|星际旅行：深空九号]]和[[星际旅行：航海家号|星际旅行：航海家号]]的设定。

==评价==
===Xbox 360版本===
星际迷航：遗产Xbox 360版本的评价总体较好<ref>[http://www.metacritic.com/games/platforms/xbox360/startreklegacy Star Trek: Legacy]</ref>，Xbox 360官方杂志称八成玩家认为星际迷航：遗产是最近几年里最好的一款宇宙模拟类游戏；Game Informer网站则给出了7.75分/10分的评价。

Xbox 360版本的星际迷航：遗产在操作上比个人电脑版要简单不少，这也是Xbox 360版的评分要比PC版高的原因之一。Game Chronicles杂志对星际迷航：遗产给出了9.1分/10分的评价，这也是这款游戏的最高评分。Game Chronicles杂志的评论家称：“这款游戏的指挥界面非常直观，让我眼前一亮，我原本以为此类游戏肯定会有极其复杂的指挥系统，但是这款游戏却设计的非常容易上手，一个训练关卡我就全懂了。”<ref>{{cite web |url=http://www.gamechronicles.com/reviews/360/startrek/legacy.htm|title=Official Game chronicle review}}</ref>。

[[游戏基地|游戏基地]]对Xbox 360版本给出了7分/10分的评分，评语称“这款游戏囊括了5部星际旅行系列设定，其星舰之间的战斗十分华丽，标志性舰船也都设计的十分精细。”<ref>{{cite web |url=http://www.gamespot.com/xbox360/sim/startreklegacy/review.html?sid=6163442|title=Official Gamespot 360 review}}</ref>。评语还认为Xbox 360版本和PC版本最大的区别是Xbox 360版的操控更加方便，而PC版则需要及其精确的操作才能控制好手上的星舰。鼠标左右键及滚轮在操控性上远不如手柄出众。游玩Xbox 360版时，仅用几个小关卡的时间就能完全掌握该游戏的操作精髓<ref>{{cite web |url=http://xbox360.ign.com/articles/751/751845p2.html|title=IGN Xbox 360 review}}</ref>。

===PC版本===
虽然PC版本与Xbox 360的唯一区别就是控制方面的差异，但是PC版本的星际迷航：遗产并不受电玩评论家和玩家的欢迎<ref>[http://www.gamestats.com/objects/794/794732//index.html#reviews Star Trek: Legacy]{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref>[http://www.gamestats.com/objects/794/794733/index.html#reviews Star Trek: Legacy Reviews]{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。很多玩家不满PC版的控制和视角系统，因为在PC版中只有一种视角设定，且不能放大或缩小视角<ref name="gamespotReview" /><ref name="actionTripReview">{{cite web|url=http://www.actiontrip.com/reviews/startreklegacy.phtml|title=Action Trip Review|deadurl=yes|archiveurl=https://web.archive.org/web/20100110125346/http://www.actiontrip.com/reviews/startreklegacy.phtml|archivedate=2010-01-10}}</ref>。存档系统也为人所诟病，PC版本中，只有当一个关卡完成后才可以进行存盘工作，在进行关卡时却不能进行存档，这给玩家带来很大的不便。除此之外，PC版本中还有许多未解决的[[BUG|BUG]]，[[多人游戏|多人游戏]]时也经常出现延迟现象，友方星舰的[[AI|AI]]能力也很低<ref name="actionTripReview" />。

尽管画质是星际迷航：遗产的主要卖点之一，但在PC版游戏中经常会出现丢帧、画面变为阴影、细节刻画不完美等问题<ref name=autogenerated1>{{cite web|url=http://startrek.bethsoft.com/games/legacy-overview.html |title=Archived game website |archiveurl=https://web.archive.org/web/20060504020147/http://startrek.bethsoft.com/games/legacy-overview.html |archivedate=2006-05-04 |deadurl=yes }}</ref>。除画质外，游戏性也有很大的进步空间，例如玩家无法占领敌舰，也无法对本方星舰进行升级<ref name=autogenerated1 />。

在PC版星际迷航：遗产中，星舰[[冲量引擎|冲量引擎]]的速度是由鼠标滚轮进行控制，但一些玩家发现在游戏中有时滚轮会失效一段时间<ref>[http://startrek.bethsoft.com/games/legacy-overview.html Star Trek Games - Bethesda Softworks<!-- Bot generated title -->] {{webarchive|url=https://web.archive.org/web/20060504020147/http://startrek.bethsoft.com/games/legacy-overview.html |date=2006-05-04 }}</ref>。[[IGN|IGN]]网站曾发表评论称：“星际迷航：遗产PC版的操控达到了让人恼火的程度。”<ref name="IGNreview">{{cite web |url=http://pc.ign.com/articles/750/750369p4.html|title=IGN Legacy review}}</ref>；ActionTrip网站则称：“星际迷航：遗产是一个好游戏，但PC版却以糟糕的操控系统毁了它。”<ref name="actionTripReview" />；[[GameSpot|GameSpot]]也对PC版的操控颇有意见，评论称：“学习星际迷航：遗产PC版的操作就像做[[噩梦|噩梦]]一般。”<ref name="gamespotReview" />。

作为一款宇宙模拟类游戏，游戏的自由度并不算太高，所有的星舰只能在地图规定的范围内穿梭<ref name="IGNreview" />。其真实性也被人诟病，例如若星舰撞上其他物体，星舰本身不会受到任何伤害，而只是被弹开而已<ref name="gamespotReview" /><ref name="IGNreview" />。

尽管[[贝塞斯达软件公司|贝塞斯达软件公司]]宣传称星际迷航：遗产完全呈现了[[星云|星云]]、[[虫洞|虫洞]]、[[行星|行星]]和[[恒星|恒星]]<ref>{{cite web|url=http://startrek.bethsoft.com/games/legacy-overview.html |title=Legacy website |deadurl=yes |archiveurl=https://web.archive.org/web/20060504020147/http://startrek.bethsoft.com/games/legacy-overview.html |archivedate=2006-05-04 }}</ref>，但大部分恒星都以一个小点表现出来，行星则比玩家的星舰大不了多少<ref name="IGNreview" />。
