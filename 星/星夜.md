《'''星夜'''》（{{lang-nl|'''''De sterrennacht'''''}}）是[[荷蘭|荷蘭]][[后印象派|后印象派]]畫家[[文森特·梵高|文森特·梵高]]於1890年在法国[[聖雷米|聖雷米]]的一家[[精神病院|精神病院]]裏創作的一幅著名[[油畫|油畫]]，是畫家的代表作之一，現藏[[紐約現代藝術博物館|紐約现代藝術博物館]]。

== 创作背景 ==
1889年，在与[[高更|高更]]一次争吵之后，梵高[[思覺失調症|思覺失調症]]复发而割下了自己的左耳，并用手帕包起后送与一名妓女。之后于5月8日自愿前往[[阿尔勒|阿尔勒]]圣雷米的一家精神病院治疗，在那驻留了108天。在入住精神病院期间，梵高创作了大量的绘画作品，共计一百五十多幅油画和一百多幅[[素描|素描]]。在此阶段的绘画，梵高的画风开始趋向于[[表现主义|表现主义]]，作品充满忧郁精神和悲剧性[[幻觉|幻觉]]，其中作品《星夜》正是其中代表之一。<ref name="fyeedu">{{cite web

== 作品含义 ==
此作品為表現性的[[後印象主義|後印象主義]][[畫家|畫家]]梵高的作品，這幅畫有很強的筆觸。油画中的主色调[[藍色|藍色]]代表不開心、陰沉的感覺。很粗的筆觸代表憂愁。畫中景象是一個望出窗外的景象。畫中的樹是[[柏樹|柏樹]]，但畫得像[[火|黑色火舌]]一般，直上云端，令人有不安之感。天空的紋理像[[渦狀星系|渦狀星系]]，并伴随众多星点，而[[月亮|月亮]]则是以昏黄的[[月蚀|月蚀]]形式出现。整幅画中，底部的村落是以平直、粗短的线条绘画，表现出一种宁静；但与上部粗犷弯曲的线条却产生强烈的对比，在这种高度夸张变形和强烈视觉对比中体现出了画家躁动不安的情感和迷幻的意象世界。<ref name="fyeedu"/>梵高生前非常欣賞[[日本|日本]][[浮世繪|浮世繪]]《[[冨嶽三十六景|冨嶽三十六景]]》中的《[[神奈川沖浪裏|神奈川沖浪裏]]》，而《星夜》中天空的涡状星云画风被認為參考並融入了《神奈川沖浪裏》的元素。

[[File:V838_Mon_HST.jpg|thumb]]。]]
在现代的一项科学研究中，科学人员发现梵高的后期作品，包括《星夜》在内，包含有一种物理上称为“[[湍流|湍流]]”的神韵，并推测此神韵来源于梵高由于长期处于癫狂状态中而得到超于常人的感悟能力和绘画表述能力。<ref name="news365"/>而在2004年3月4日，[[美国宇航局|美国宇航局]]和[[欧洲航天局|欧洲航天局]]公布了一张[[哈勃空间望远镜|哈勃]][[太空望远镜|太空望远镜]]拍摄的太空照片，并称“这幅太空摄影作品与梵高的名作《星夜》有‘异常相似’之处。”其中，哈勃太空望远镜所拍照片为一颗名为“[[麒麟座V838|麒麟座V838]]”的[[恒星|恒星]]周围的景象。该恒星位于[[麒麟座|麒麟座]]方向，距离地球2万[[光年|光年]]。<ref name="news365">{{cite web
