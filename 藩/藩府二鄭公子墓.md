'''藩府二鄭公子墓'''位於臺南市南區的桶盤淺墓園內，即過去[[寧南門_(臺南)|大南門]]外的鞍仔庄{{r|石}}，於民國七十四年（1985年）8月19日公告為中華民國三級古蹟，後因改制而改為市定古蹟，原臺南縣市合併為直轄市後改為直轄市定古蹟{{r|文}}。藩府二字指的是「延平郡王府」{{r|南}}，二鄭公子指的是[[鄭成功|鄭成功]]四子[[鄭睿|鄭睿]]（字聖之）與十子[[鄭發|鄭發]]（字省之）二人{{r|石}}。該墓是臺南市列為古蹟的四座古墳之一，另外三座是崇禎十五年（1642年）的[[曾振暘墓|曾振暘墓]]、同為永曆年間鄭家宗族之墓的[[藩府曾蔡二姬墓|藩府曾蔡二姬墓]]與清光緒三年（1877年）的[[施瓊芳墓|施瓊芳墓]]。

== 沿革 ==
鄭睿、鄭發二人為鄭成功之子，隨其父一同來臺，然而沒過多久便因水土不服而去世，死時二人均無嗣，合葬此地{{r|石}}。到了康熙三十八年（1699年），明鄭時期鄭氏家族與文武官員之墓大多遷回中國大陸{{r|石}}，以減少臺灣人民思念故朝之情{{r|傅}}，然而此墓或因地處偏遠{{r|石}}，或因二人無嗣之故{{r|南}}，而未被遷移留了下來，與藩府曾蔡二姬墓為僅存於臺的鄭家之墓。

該墓在清朝的方-{志}-與文獻中並未被記載，[[臺灣日治時期|日治時期]]方始見於[[連橫_(歷史學家)|連橫]]的《雅言》一書{{r|石}}，而在該書中還記載說在此墓不遠處建有庵和塔{{r|文}}。該墓在昭和五年（1930年）時曾整修過{{r|南}}。

二次大戰後，該墓於民國四十一年（1952年）被臺南市文獻委員會發現{{r|傅}}並整修{{r|南}}，民國六十四年（1975年）再修，而成今貌{{r|傅}}。

== 建築特色 ==
該墓格局遵守規制，由墓埕、供臺、墓碑、左右手和墓塚組成，不過除了墓碑之外皆為民國六十四年（1975年）整修時的東西{{r|傅}}。墓碑寬77公分，出土部分高1.07公尺，於上面刻著「皇明聖之、省之二鄭公子墓」{{r|石}}。
