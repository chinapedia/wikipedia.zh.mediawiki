'''閻紅彥'''（{{bd|1909年|9月13日|1967年|1月8日|catIdx=Y閻}}），原名'''阎候雁'''，曾用名'''周济'''、'''陈一川'''，生於[[中国|中国]][[陝西省|陝西省]][[安定县_(元朝)|安定縣]]（今[[子长县|子长县]]），[[中國人民解放軍上將|中國人民解放軍上將]]。

阎红彦1927年参加[[清涧起义|清涧起义]]，后在国军中做兵运工作。历任[[中国工农红军|中国工农红军]][[陕甘游击队|陕甘游击队]]第一支队支队长、陕甘游击队总指挥、红军沿河游击司令、[[紅三十軍|紅三十軍]]軍長。[[抗日战争|抗日战争]]时期，任[[八路軍|八路軍]]後方留守處警備第三團團長、警备第一旅政治委员。[[第二次国共内战|第二次国共内战]]时期，历任[[中原野战军|中原野战军]]第三纵队副政委、[[第二野戰軍|第二野戰軍]]第三兵團副政委兼政治部主任。[[中華人民共和國|中華人民共和國]]成立後，历任[[中共四川省委|中共四川省委]]副書記、副省長、[[成都军区|成都军区]]副政治委员。1955年，作为[[陕北红军|陕北红军]]的代表被授銜上將。1959年8月起，任[[中共雲南省委|中共雲南省委]]第一書記、[[昆明軍區|昆明軍區]]第一政委。「[[文化大革命|文化大革命]]」時受迫害，1967年1月8日在昆明服安眠药自杀。

== 生平 ==
=== 早期经历 ===
1924年入陕北军阀[[井岳秀|井岳秀]]部当兵。先当号兵，后当[[李象九|李象九]]的勤务兵。1925年4月，经李象九介绍加入[[中國共產黨|中國共產黨]]<ref>{{cite book|title=陕西近现代名人录|editor=陕西省中共党史人物研究会|year=1988|publisher=西北大学出版社|pages=285|isbn=7-5604-0081-7|language=zh-hans}}</ref>。1927年11月，阎红彦参加[[清涧起义|清涧起义]]，任第十一旅旅部警卫排长；11月中旬起义失败后，阎红彦逃亡到[[山西|山西]][[绛州|绛州]]，不久又因拦截杨虎城部的烟土被侦缉，乃秘密回陕北，在清涧一带组织游击队，后又到山西。1929年春，阎红彦被派往[[杨赓五|杨赓五]]部做兵运工作，失败后又潜回瓦窑堡。

1931年，阎红彦与[[吴岱峰|吴岱峰]]、[[拓可宽|拓可宽]]、[[杨重远|杨重远]]等创建[[中国工农红军|中国工农红军]][[晋西游击大队|晋西游击大队]]。9月率部渡过黄河进入陕北，被公推为大队长。同年10月率部与[[刘志丹|刘志丹]]领导的南梁游击队会合，任西北抗日反帝同盟军第1支队第1大队大队长<ref name="gf"/>。1932年2月，同盟军改编为中国工农红军陕甘游击队，先后任第1大队大队长、第5支队支队长，率部在渭北地区建立苏维埃政权，被当地群众誉为“神军”。同年6月任陕甘游击队总指挥，参与创建[[陕甘苏区|陕甘苏区]]<ref>{{cite book |language=zh-hans |author=雷秀珍 编著 |title=中国共产党党史人物介绍 |publisher=中国人民大学出版社 |year=1991 |page = 149 |ISBN = 978-7-300-01040-3}}</ref>。8月被[[中共陕西省委|中共陕西省委]]指责为“右倾机会主义”而被撤销总指挥职务。12月，与[[谢子长|谢子长]]、[[刘志丹|刘志丹]]等被指责为“上山主义”、“逃跑主义”等而遭排挤。1933年到上海中央局学习。后赴[[天津|天津]]、[[热河|热河]]、[[北平|北平]]、[[张家口|张家口]]等地，先后在[[察哈尔民众抗日同盟军|察哈尔民众抗日同盟军]]和[[中共河北省委|中共河北省委]]军委工作<ref name="jlz"/>。

1934年5月，阎红彦被选派为出席[[共产国际第七次代表大会|共产国际第七次代表大会]]的代表到[[苏联|苏联]][[莫斯科|莫斯科]]，入[[国际列宁学院|国际列宁学院]]、[[苏联红军陆军大学|苏联红军陆军大学]]学习。后因[[共产国际|共产国际]]与中共中央失去电讯联系，于1935年4月受命送密电码回国，同年底到达陕北。1936年初任中国工农红军沿河游击司令，担负保障[[红一方面军|红一方面军]]东渡黄河的任务。後任[[紅三十軍|紅三十軍]]軍長<ref>{{cite book|language=zh-hans|author=《中国人民解放军军史》编写组|title=中国人民解放军军史：第一卷|year=2010|publisher=军事科学出版社|pages=432|isbn=978-7-80237-381-5}}</ref>，率部参加[[东征战役|东征战役]]。5月部队回师陕北，阎红彦改任红三十军政委，6月重任军长，指挥所部在[[瓦窑堡|瓦窑堡]]地区抗击国军进攻，掩护中共中央机关安全转移，后率部开赴三边。12月到陕西[[杨虎城|杨虎城]]部从事统战工作<ref name="jlz"/>。

=== 抗日战争与第二次国共内战时期 ===
[[抗戰|抗戰]]爆發後，阎红彦任[[八路軍|八路軍]]129师後方留守處主任<ref>{{cite web|url =http://www.sn.xinhuanet.com/2008-07/30/content_13975439.htm|title =红军改编为八路军初期部队作战序列|date =2008-07-30|accessdate =2015-07-06|language =zh-hans|publisher =新华网|deadurl =yes|archiveurl =https://web.archive.org/web/20150924103019/http://www.sn.xinhuanet.com/2008-07/30/content_13975439.htm|archivedate =2015-09-24}}</ref>。1937年11月，陕北八路军部队统一整编，阎红彦任警備第三團團長，保卫[[陕甘宁边区|陕甘宁边区]]<ref>{{cite book |language =zh-hans|author=王辅 |title=日军侵华战争 Volume1 |location = 沈阳 |publisher=辽宁人民出版社 |year=2015 |page = 538 |ISBN = 978-7-205-08145-4}}</ref>。1939年12月，担任警备第一旅兼关中军分区政委<ref>{{cite book|title=中国革命史手册|author=陈荣华|year=1986|publisher=华中师范大学出版社|pages=420|language=zh-hans}}</ref>。1940年春，阎红彦奉命到[[洛川|洛川]]同[[国民政府|国民政府]]代表谈判，然后到[[马列学院|马列学院]]和留守兵团军政研究班学习。1942年，阎红彦在[[中共中央党校|中共中央党校]]学习，参加[[延安整风|延安整风]]。此间[[高岗|高岗]]指控阎红彦“有野心”，中共中央中央对阎红彦的历史进行了全面审查，对他充分肯定<ref name="jlz">{{cite book|author=《中国人民解放军高级将领传编审委员会》，中国中共党史人物研究会编|title=中国人民解放军高级将领传|volume=第12卷|year=2007|location=北京|publisher=解放军出版社|pages=75-122|isbn=7-5065-5288-4}}</ref>。1945年4月，阎红彦参加了[[中共七大|中共七大]]。

1945年9月，阎红彦任晋冀鲁豫野战军第三纵队副司令员，参加[[邯郸战役|邯郸战役]]<ref>{{cite web|language =zh-hans|url=http://www.china.com.cn/military/zhuanti/jianjun80/2007-07/22/content_8560191.htm|date=2007-07-22|accessdate=2015-02-13|title=邯郸战役(1945年)|publisher=中国网}}</ref>。11月，改任第三纵队副政委兼政治部主任，参加[[出击陇海路战役|出击陇海路]]、[[定陶战役|定陶]]、[[鲁西南战役|鲁西南]]诸战役<ref>{{cite book |author=[[张明金|张明金]]、[[赵功德|赵功德]]|title=中国人民解放军历史上的70个军|year=2006 |publisher=解放军文艺出版社|location=北京市西城区地安门西大街40号|pages=192|isbn=9787503319402 |language = zh-hans}}</ref>。1947年4月，参与[[豫北战役|豫北战役]]，歼灭国军第二快速纵队<ref>{{cite book |author=[[王树增|王树增]] |title=解放战争（上）|year=2009 |publisher=人民文学出版社|location=北京|pages=303|isbn=978702007358-0 |language = zh-hans}}</ref>。随后协同友军攻取[[汤阴|汤阴]]，生擒[[孙殿英|孙殿英]]。同年7月，参加进军[[大别山|大别山]]，率部队率先抢渡[[淮河|淮河]]，到达大别山北麓，半个月攻占宛西地区9个县城及大片地区<ref>{{Cite web |url = http://www.hada.gov.cn/w_NewsShow.asp?ID=305:1098 |title = 档案里的故事----宛东战役 |author =  王亚丽 |date = 2011-03-04 |publisher = 南阳市宛城区档案馆 |language = zh-hans |accessdate = 2015-03-20 }}</ref>。1948年参加[[郑州战役|郑州战役]]、[[淮海战役|淮海战役]]等战斗，全歼[[中華民國國軍|国军]][[黄维|黄维]]兵团。1949年2月，出任[[第二野戰軍|第二野戰軍]]第三兵團副政委兼政治部主任<ref>{{cite book|author=《中国人民解放军军史》编写组|title=中国人民解放军军史：第三卷|year=2010|publisher=军事科学出版社|pages=324|isbn=978-7-80237-381-5|language=zh-hans}}</ref>。1949年4月参加[[渡江战役|渡江战役]]；11月攻占[[重庆|重庆]]，阎红彦任重庆军管会主任<ref name="jlz"/>。 

=== 中华人民共和国成立后 ===
1949年12月，阎红彦任中共川东区委第一副书记兼统战部部长、川东行署主任，大力整顿社会秩序，恢复和发展生产。1952年，任[[四川|四川]]省委副書記、副省長。1955年任[[成都军区|成都军区]]副政治委员<ref>{{cite book|title=中华人民共和国军事史要|author=军事科学院军事历史研究所|pages=121|location=北京|publisher=军事科学出版社|year=2005|isbn=7-80137-800-8}}</ref>，被授銜上將，获一级[[八一勋章_(1955年)|八一勋章]]、一级[[独立自由勋章|独立自由勋章]]、一级[[解放勋章|解放勋章]]。1956年9月，当选中共第八屆中央候補委員。1959年8月，阎红彦任[[雲南省|雲南省]]委第一書記、[[昆明軍區|昆明軍區]]第一政委、[[中共中央西南局|中共中央西南局]]書記處書記<ref>{{cite book| language = zh-hans|author=星火燎原编辑部|title=中国人民解放军将帅名录：第一卷|year=2006|publisher=解放军出版社 |location=北京 |isbn=7506550555| pages = 141}}</ref>。1962年，小说《劉志丹》開始連載后，阎红彦寫信予小说作者[[李建彤|李建彤]]，指《劉志丹》中有西北革命的一些問題，閻紅彥為此向李建彤爭論，並向[[康生|康生]]報告，引发“[[反黨小說《劉志丹》案|反黨小說《劉志丹》案]]”<ref>{{cite book |author = 李建彤 |title = 反党小说刘志丹案实录 |publisher = 星克爾出版 |isbn = 978-988-99455-2-7|pages=4|language=zh-hant|date=2007年5月}}</ref>。1963年12月，阎红彦兼任[[中國人民政治協商會議|中國人民政治協商會議]]雲南省委員會主席。此外，阎红彦还是第一至第三届国防委员会委员。

「[[文化大革命|文化大革命]]」時，阎红彦因反对挑动群众冲击政府机关的做法，而受到迫害，1967年1月8日，阎红彦在昆明服安眠药自杀。留有“我是被[[陈伯达|陈伯达]]、[[江青|江青]]逼死的”字条（一说死于心脏病<ref name="jlz"/>）。1978年1月，[[中共中央|中共中央]]对其冤案予以平反，并在[[北京|北京]][[八宝山|八宝山]]革命公墓举行了骨灰安放仪式<ref name="gf">{{Cite web |url = http://www.gov.cn/gjjg/2009-02/02/content_1219361.htm |title = 阎红彦 |date = 2009-02-02 |publisher = 中央人民政府门户网站 |language = zh-hans |accessdate = 2015-07-06 }}</ref>。

== 家庭 ==
第一任夫人[[刘秀英|刘秀英]]。

第二任夫人[[王腾波|王腾波]]，[[四川|四川]][[合江|合江]]人，原名罗学儒。1942年9月，与阎红彦在延安结婚，时任延安女子大学妇女工作团团长。曾任第四届[[四川省政协|四川省政协]]副主席，第五、六届全国人大代表。1998年10月6日病逝。二人有子[[阎泽群|阎泽群]]，有女[[阎小青|阎小青]]<ref>{{cite web|author=蒲德生|title=开国上将与夫人|url=http://club.china.com/data/thread/1011/2710/98/11/9_1.html|publisher=中华网|location=北京|date=2010-03-24|deadurl=yes|archiveurl=https://web.archive.org/web/20150222125258/http://club.china.com/data/thread/1011/2710/98/11/9_1.html|archivedate=2015-02-22}}</ref>。
