'''阎崇年'''（{{bd|1934年|4月24日|catIdx=Y阎}}），[[中華人民共和國|中國]][[歷史學家|歷史學家]]，[[漢族|漢族]]<ref>{{cite news |url = http://news.ifeng.com/history/special/yanchongnian/200810/1010_4788_825456.shtml |title = 此訪談中閻崇年回應}}</ref>，[[山东省|山东省]][[蓬莱市|蓬莱市]]人，[[北京师范大学|北京师范大学]]历史系本科毕业，从事满学研究。

阎崇年创立了[[中国|中国]]第一个满学研究机构——「北京社会科学院满学研究所」、「北京满学会」，现为北京社会科学院满学研究所研究员及北京满学会会长。他先后发表满学、清史论文一百五十余篇。自2004年起在[[中央电视台|中央电视台]]《[[百家讲坛|百家讲坛]]》栏目中主讲「清十二帝疑案」，2006年9月1日起主讲「明亡清兴六十年」。

阎崇年本人的言论被[[汉民族主义|汉民族主义]]者认为偏袒满清而遭到部分[[汉民族主义|汉民族主义]]者的不满和抵制<ref>[http://news.sina.com.cn/pl/2008-10-08/082416415089.shtml 魏英杰：阎崇年挨了民族主义一耳光].新浪网.</ref>。

== 生平 ==
阎崇年生于1934年，在[[山东省|山东省]]完成[[小学|小学]]课程。1949年[[北京|北京]]被中國共產黨佔領後在[[北京六中|北京六中]]读书。

阎崇年在1963年开始研究满学，1980年在北京社会科学院工作至今。近年常出版书籍和举行讲座发表清朝历史研究论文和最新[[研究|研究]]成果。

== 签售被掌掴事件 ==
{{main|閻崇年被打事件}}
2008年10月5日，阎崇年在[[无锡|无锡]][[新华书店|新华书店]]出席签售活動时，被汉网管理员黄海清（网名“大汉之风”）掌掴。黄海清的弟弟稱他的哥哥因无法认同阎崇年关于明清历史的一些观点，所以动手打人<ref>{{cite web |url=http://news.163.com/08/1007/02/4NKBOKHQ00011229.html |title=阎崇年无锡签售时遭掌掴 打人者被治安拘留 |accessdate= 2008年|work=|publisher=网易 |language = zh-hans}}</ref>。“大汉之风”在其被结束拘留后作出辯解，他认为阎崇年-{「}-粉饰屠杀、肯定文字狱、拒绝质疑和沟通-{」}-，并以英国右翼学者[[戴维·欧文|戴维·欧文]]因否认[[纳粹大屠杀|纳粹大屠杀]]被奥地利维也纳一家地方法庭判3年徒刑为自己辩护<ref>{{cite web|url=http://bbs.club.sina.com.cn/thread-332-0/tree-60893-7525--.html|title=大汉之风：我为什么掌掴阎崇年|accessdate=2008年|work=|publisher=新浪网|language=zh-hans}}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。 后来他又接受了天涯社区的访谈邀请<ref>{{cite web|url=http://bbs.tianya.cn/post-free-1445523-1.shtml|title=我为什么掌掴阎崇年|author=[http://www.tianya.cn/19847633 我是大汉之风] |authorlink=  |date=2008-10-20|language=zh-hans|publisher=[[天涯论坛|天涯论坛]]}}</ref>。

一些媒體試圖進一步評論事件背後原因，[[鳳凰網|鳳凰網]]特意對此事件做了一個專題<ref>[http://news.ifeng.com/history/special/yanchongnian/ 阎崇年到底該不該打？]</ref>，對該事件的原因表達各自意見。并在《我們如何面對文化暴力》專題中提及和探討暴力問題<ref>[http://book.ifeng.com/special/qianliexian/ 我們如何面對文化暴力]</ref>。事件發生後引起社會熱議，有認同掌摑閻崇年的支持者表示看了網上的「閻崇年語錄」後認為他該打，亦有學者評論說即便閻崇年個人的學術觀點再為偏頗，也不應該上升到使用武力的地步<ref name="c02">{{cite news|author1=刘义昆|title=刘义昆：从阎崇年挨打看公众是非判断的迷失|url=http://news.sina.com.cn/pl/2008-10-09/100716423301.shtml|accessdate=2017-12-06|agency=金羊网|date=2008-10-09}}</ref>。閻崇年後來在接受採訪時作出澄清，指出網上流傳及一些報紙轉載的「語錄」是仿造和杜撰的，亦有媒體文章對「語錄」和閻崇年的著作和電視節目講話中的言論作出對比，指出「語錄」所言並非事實<ref name="c03">{{cite news|title="阎崇年语录"是否捏造？ 真假完全对照版|url=http://china.huanqiu.com/roll/2008-10/250196.html|accessdate=2017-12-06|agency=新华网|date=2008-10-13}}</ref>。

== 评价 ==
[[國家清史編撰委員會|國家清史編撰委員會]]委員、研究员[[李治亭|李治亭]]評其人學術研究<ref>[http://news.ifeng.com/history/special/paohongyanchongnian2/articles/200810/1020_4827_838207.shtml 《<正说清朝十二帝>质疑》]</ref>：將本来属於[[皇太极|皇太极]]的功绩，强加给[[努尔哈赤|努尔哈赤]]，是為“颠倒史实，任意篡改历史”；“对历史事实缺乏认真研究和考证，将正、误乃至错谬混淆一处，统统作为努尔哈赤的‘功绩’加以赞扬。所列‘十大功绩’，硬是凑数”、“绝非治史者所为”...“作者给努尔哈赤的历史定位，已经远远地背离了努尔哈赤自身的历史真实，大失史学的严肃性与科学性。作为一个学者，亦失应有的学风与史德”

評論家[[張鳴_(學者)|张鸣]]則說：“阎崇年是研究[[清朝|清朝]]的，他研究了[[清史|清史]]一辈子，在《[[百家讲坛|百家讲坛]]》威望很高的，阎崇年很有风度，在[[无锡|无锡]]被掌掴仍旧镇定地签名售书。”<ref>{{cite web |url= http://v.ifeng.com/society/200810/b43c6986-628e-4307-b26c-1bb4077ada5a.shtml?_from_ralated|title= 张鸣：阎崇年被掌掴 很有风度继续签售|accessdate= 2008年|work=|publisher=凤凰网 |language = zh-hans}}</ref>

作家家韓浩月說：「閻崇年的處亂不驚體現了一位學者的涵養，打人者也將會為自己的行為蒙羞……閻崇年算異端人物嗎。他的節目收視率很高，他的書賣得很好，他在各地的粉絲眾多，豈止不算異端，稱之為主流一點也不為過。所以，這樁事件只能說是一位主流人物說了幾句異端的話（那些語錄被證實為偽造）……如果對待一個哪怕說了幾句錯話的人，有更多的人衝上去拳打腳踢，甚至像加爾文那樣以假信仰的名義對異端分子動以私刑，那才是一件真正恐怖的事情。」<ref>[http://www.chinanews.com/cul/news/2008/10-08/1403925.shtml 北京作家力挺阎崇年 评阎崇年遭袭与异端的权利], 新京报, 2008-10-08</ref>
