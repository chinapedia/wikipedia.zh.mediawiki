'''阎宝航'''（{{bd|1895年|4月6日|1968年|5月22日|catIdx=Y}}<ref name="a1"/>），[[表字|字]]玉衡，生于[[奉天省|奉天省]][[海城县_(顺治)|海城县]]（今[[辽宁省|辽宁省]][[海城市|海城市]][[望台乡|望台乡]]小高丽房村<ref>{{cite web|url=http://tv.people.com.cn/GB/39805/42848/10089191.html |title=中国“保尔”准确情报 拯救百万苏联人性命|publisher=人民网，来源：凤凰网|author=|date=2009年9月21日 |accessdate=2012-05-27 |language=Zh-hans}}</ref>）。是潜伏于[[国民政府|国民政府]]内的[[中国共产党|中共]][[间谍|间谍]]。1949年后在[[中华人民共和国外交部|中华人民共和国外交部]]等处任职。[[文化大革命|文化大革命]]时死于[[秦城监狱|秦城监狱]]。1995年，[[俄羅斯聯邦政府|俄罗斯联邦政府]]追授他“[[卫国战争胜利五十周年纪念章|卫国战争胜利五十周年纪念章]]”。

== 生平 ==
1913年，考入[[奉天两级师范学校|奉天两级师范学校]]，期间开始信奉[[基督教|基督教]]<ref name="a1">{{cite web |url = http://www.cass.net.cn/zhuanti/2009wusi/show_News_gongqt.asp?id=6207 |title = 一生无私奉献　风范长留青史　——纪念阎宝航同志逝世二十周年 |publisher = 中国社会科学院网站，原载：人民日报（1988.06.02） |author = [[吕正操|吕正操]] |date =  |accessdate = 2012-05-27 |language = zh-hans }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。1918年毕业后，与同学魏益新、张泊等人在[[沈阳|沈阳]]创办[[奉天贫儿学校|奉天贫儿学校]]，因此成为[[沈阳|沈阳]]的知名人士。1921年，受聘为[[奉天基督教青年会|奉天基督教青年会]]干事。受[[张学良|张学良]]资助，于1927年去英国[[爱丁堡大学|爱丁堡大学]]读社会学。1929年回国，接替[[普赖德|普赖德]]出任奉天基督教青年会总干事。1934年，担任[[新生活运动|新生活运动]]促进总会总干事。[[抗日战争|抗日战争]]爆发后先后担任军事委员会政治部设计委员、重庆市动员委员会设计委员、中央赈济委员会顾问、陪都劝募公债运动主任干事。

1937年經[[周恩来|周恩来]]介绍，阎宝航加入[[中国共产党|中国共产党]]。1941年春，在苏联驻华武官[[罗申|罗申]]的联络下，为莫斯科和延安提供情报<ref>{{cite web |author1=陈祥 |title=《阎明复回忆录》出版揭秘中苏两党特殊间谍 |url=http://www.ifengweekly.com/detil.php?id=2144 |website=凤凰周刊 |accessdate=2019-01-06 |archivedate=2015/08/15}}</ref>。在国民政府内帮助中共搜集了重要情报，包括[[纳粹德国|纳粹德国]]入侵[[苏联|苏联]]的准确日期，[[日本關東軍|日本關東軍]]在[[滿洲|滿洲]]的佈防等<ref>{{Cite news | url=http://gb.cri.cn/3601/2005/10/13/342@736487.htm | title=中共秘密情報員閻寶航身份之謎 | author=王岙 | work=北京科技報 | publisher=[[中國國際廣播電台|中國國際廣播電台]]}}</ref>。1946年，中共令其秘密回到[[中国东北地区|东北]]，任[[辽北省|辽北省]]省长。

1949年后任[[中华人民共和国外交部办公厅|外交部办公厅]]副主任。1952年，[[中共中央|中共中央]]批准其在[[中华人民共和国外交部|外交部]]公开党员身份。1965年起，任[[中国人民政治协商会议第四届全国委员会委员名单|第四届全国政协常务委员]]。阎宝航在“[[文革|文革]]”中被诬陷为“东北帮叛党投敌反革命集团”，并于1967年11月7日被捕。之后惨死于[[秦城监狱|秦城监狱]]，甚至于[[骨灰|骨灰]]都未被保留，因[[江青|江青]]在他的死亡報告批示：「閻寶航是現行[[反革命|反革命]]分子，不通知家屬、不留骨灰、不留遺物」<ref>{{Cite web | url=http://www.yhcqw.com/35/1535.html | title=我父亲阎宝航的情报生涯|author=阎明复|date=2005年12月| website=炎黄春秋}}</ref>。后获得[[平反|平反]]，1978年1月5日，[[中共中央|中共中央]]在[[八宝山革命公墓|八宝山革命公墓]]为阎宝航举行骨灰安放仪式<ref name="a1"/>。

1995年11月1日，俄罗斯驻华大使[[罗高寿|罗高寿]]在俄罗斯驻中国大使馆将总统[[鲍里斯·尼古拉耶维奇·叶利钦|叶利钦]]签署的“卫国战争胜利五十周年纪念章”授予阎宝航及其领导的情报小组。2007年，中国大陆拍摄了以他为主角的传记电视剧《[[英雄无名|英雄无名]]》。<ref>{{Cite web | url=http://dangshi.people.com.cn/GB/144956/10505519.html | title=红色间谍阎宝航的传奇人生：周恩来单线联系的“特殊人物” (2)|author=李恩和|date=2009年12月3日| website=中国共产党新闻网}}</ref>

== 家庭 ==
妻子高素，曾用名高素桐。有三子三女，长女阎明诗、长子阎明新（改名阎大新）、二女阎明英（改名高玲）、二子阎明智、三女[[阎明光|阎明光]]及三子[[阎明复|阎明复]]<ref>{{cite web |url = http://phtv.ifeng.com/program/wdzgx/200909/0921_5447_1357604_1.shtml |title = 中共“潜伏者”阎宝航传奇式“地下”人生|publisher=凤凰网|author=|date=2009年9月21日 |accessdate=2012-05-27 |language=Zh-hans}}</ref>。

女儿阎明光是上海阎宝航社会公益基金会理事长。儿子阎明复曾任[[中华人民共和国民政部|民政部]]副部长。[[阎明智|阎明智]]为俄语翻译，中华人民共和国驻苏使馆一秘，1956年曾负责连夜为中共代表团翻译赫鲁晓夫的[[秘密报告|秘密报告]]。
