'''滕头村'''是[[中国|中国]][[浙江|浙江]][[宁波|宁波]][[奉化|奉化]]的一个村庄，隶属于[[萧王庙街道|萧王庙街道]]。滕头村距离奉化城区中心6公里，距离宁波市中心27公里。由于其经济实力和生态建设而于1993年入选[[联合国环境规划署|联合国环境规划署]]“全球生态500佳”。<ref name="main">[http://house.focus.cn/news/2010-07-12/984561.html 滕头村：世界十佳和谐乡村]</ref>2010年，滕头村与[[溪口镇_(宁波市)|溪口镇]]组成的“宁波溪口-滕头旅游景区”成为[[中国国家5A级旅游景区|中国国家5A级旅游景区]]。<ref>[http://www.yangtse.com/news/ms/201004/t20100419_731098.htm 夫子庙等江苏4景区晋级5A迎世博] {{webarchive|url=https://web.archive.org/web/20100420140751/http://www.yangtse.com/news/ms/201004/t20100419_731098.htm |date=2010-04-20 }}</ref>[[中国2010年上海世界博览会宁波案例馆|滕头案例]]入选[[中国2010年上海世界博览会城市最佳实践区|上海世博会城市最佳实践区]]，成为唯一入选城市最佳实践区的乡村案例。<ref>{{cite web|url=http://www.expo2010.cn/a/20091214/000018.htm|title=展现"新乡土、新生活" 上海世博会宁波案例馆揭开面纱|publisher=2010年上海世博会官网|author=徐笋|date=2009-12-14|accessdate=2011-01-14|deadurl=yes|archiveurl=https://web.archive.org/web/20100219081243/http://www.expo2010.cn/a/20091214/000018.htm|archivedate=2010-02-19}}</ref>

== 村名来历 ==
滕头村的村名来源于当地居民的构成。据说，滕头村的村民大都是来自[[山东|山东]][[滕州|滕州]]傅氏的子孙。傅氏族人南迁到达宁波时，遇海无法前进，因而定居，以“滕州人走到头”之义定村名为滕头村。<ref name="main" />

== 经济开发 ==
滕头村的经济开发始于1965年。在当时中国大陆兴起的“[[农业学大寨|农业学大寨]]”的风潮中，开始平整土地，并不顾当时“走[[资本主义|资本主义]]道路”而受到批判的风险种植[[橘|橘]]树，开展多种经营。[[改革开放|改革开放]]之初，滕头村已经将原先的低产田改造为高产田。此后，滕头村开始改善村民的居住环境，并开始发展规模经营，利用规模化的优势进行集体生产，将剩余劳动力转移到非农产业中。20世纪90年代末，滕头村开始发展现代农业和观光农业。滕头村的工业开发始于1979年，起步于村办服装厂。2001年成立自己的工业园区。在[[第三产业|第三产业]]方面，滕头村成立了滕头建设有限公司和滕头房地产开发有限公司，进行房地产开发，并成立滕头园林绿化工程有限公司，拥有大规模苗木种植基地，为[[2008年北京奥运会|2008年北京奥运会]]和[[2010年上海世博会|2010年上海世博会]]提供绿化服务。<ref name="main" />

== 生态保护 ==
[[File:Ningbo_Tengtou_Case_Pavilion.jpg|thumb]]]]
1993年，滕头村成立世界上最早的乡村环保机构——滕头环境资源保护委员会，对进入滕头的工业企业进行环境保护方面的审核<ref>{{cite web|url=http://paper.wenweipo.com/2010/09/09/CF1009090001.htm|title=百家廊：世博為何選中滕頭村？|publisher=[[文匯報_(香港)|文匯報]]|author=王大慶|date=2010-09-09|accessdate=2011-01-14}}</ref>。同时，整个村庄形成了从规划开始的环保措施，工业区域与生活区域相分离。与此同时，滕头村先后进行了液化气普及、绿化和排放控制。1993年获得联合国“全球生态500佳”称号。<ref name="main" />

== 旅游业 ==
滕头村的旅游开发始于1998年，主要旅游项目为农家乐、盆景园等生态景观。2001年成为[[中国国家4A级旅游景区|中国国家4A级旅游景区]]，2010年与[[溪口镇_(宁波市)|溪口镇]]组成的“宁波溪口-滕头旅游景区”成为[[中国国家5A级旅游景区|中国国家5A级旅游景区]]。<ref name="main" />
