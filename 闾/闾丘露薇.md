'''闾丘露薇'''（{{bd|1969年|9月19日|catIdx=L}}），[[香港|香港]]电视[[记者|记者]]，曾服务于[[凤凰卫视|凤凰卫视]]，担任凤凰卫视全球新闻总监。曾因2003年在[[伊拉克战争|伊拉克战争]]时作为在[[巴格达|巴格达]]地区唯一的华人女记者进行现场报道，而深得[[中国大陆|中国大陆]]观众关注，被誉为“战地玫瑰”<ref>{{cite web | work=[[人民网|人民网]]| title=闾丘露薇，最怕人说我勇敢
| url=http://www.people.com.cn/GB/news/37146/45773/3284424.html | date=2005-03-31| accessdate=2008年7月12日}}</ref>。2015年从[[凤凰卫视|凤凰卫视]]离职<ref name="离职"/>，同年8月至2018年5月在[[宾州州立大学|宾州州立大学]]新闻传播学院攻读博士<ref>{{cite news|title=我的地盘，绝不纵容误解和指责|url=http://cul.qq.com/a/20150526/011669.htm|publisher=腾讯文化}}</ref><ref>{{cite web|url=http://comm.psu.edu/people/individual/rose-luqiu-luwei|title= People Directory - Rose Luqiu Luwei|work=psu.edu}}</ref>。2018年8月到香港浸會大學新聞系任助理教授。<ref>{{cite web|title=DR. LUWEI ROSE LUQIU|url=http://www.jour.hkbu.edu.hk/faculty-member/dr-luwei-rose-luqiu/}}</ref>

==传记==
===早期生活===
1969年出生于[[上海|上海]]，四岁时父母离异，父亲再婚<ref name="文汇报">{{cite web | work=[[文汇报|文汇报]]| title=闾丘露薇战地采访误了功课| url=http://paper.wenweipo.com/2003/06/09/BK0306090004.htm | date=2003-06-09 | accessdate=2008年7月12日}}</ref><ref name="mother">{{cite web | title=闾丘露薇：做母亲改变了我看世界的眼光 | url=http://www.zzmmttm.com/bbs/read.php?tid=242&page=e&fpage=1 | accessdate=2008年7月12日}}</ref>，母亲前往海外定居<ref>{{cite web| title=闾丘露薇的打工岁月| url=http://young.boustead.edu.cn/04/nr/047_01.html| accessdate=2008年7月12日| deadurl=yes| archiveurl=https://web.archive.org/web/20100326032711/http://young.boustead.edu.cn/04/nr/047_01.html| archivedate=2010年3月26日}}</ref>，她与奶奶一起生活，父亲在周末偶尔前来探望<ref name="mother" />。1982年小学毕业后考入上海[[华东师范大学第二附属中学|华东师大第二附中]]。在校期间，学习成绩很优秀而且十分活跃，曾加入上海《[[青年报|青年报]]》上海中学生记者团，并担任了团长<ref name="rose">{{cite web| title=“战地玫瑰”闾丘露薇| url=http://bayaonline.com/news_view.asp?newsid=3830| date=2005-06-21| accessdate=2008年7月12日}}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。1988年考取了上海[[复旦大学|复旦大学]]哲学系。加入大学之后曾任学校音乐之夜节目的主持人。因家庭经济原因不得不外出打工，曾在一家外企任职钟点工，又曾在大学附近的咖啡屋做招待员。上大学半年之后，与母亲重逢，并得到母亲的资助<ref name="jizhe">{{cite web | title=闾丘露薇：打工妹出身的名记者 | url= http://wf66.com/page/20069/233976540E.htm| date=2006-09-24| accessdate=2008年7月12日}}</ref>。

1992年7月，大学毕业前往其母在[[深圳|深圳]]所开办的一家外贸公司工作，但是公司因经营状况不佳，不久之后关闭。随后其母离开深圳去了[[香港|香港]]，而闾丘露薇和其同学则找到了一份为一家[[海南|海南]]汽水厂在深圳市场推销汽水的工作<ref name="career">{{cite web| title=闾丘露薇：人生在职业追求中绽放异彩| url=http://renwu.shangdu.com/html/renwu/shidai/200804/06-216.html| date=2008-04-06| accessdate=2008年7月12日}}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。这段时间由于经济压力，她租住在深圳的[[农民房|农民房]]中，之后也曾经从事过倒卖文化衫及手表等短期销售工作<ref name="jizhe"/>。期间曾面试一份会计师事务所英文翻译一职，由于对专业英文词汇不熟悉，未获聘用<ref>{{cite web| title=《空杯心态》| url=http://news.xinhuanet.com/book/2008-03/13/content_7781294_45.htm| date=2008-03-13| accessdate=2008年7月12日| deadurl=yes| archiveurl=https://web.archive.org/web/20080329113049/http://news.xinhuanet.com/book/2008-03/13/content_7781294_45.htm| archivedate=2008年3月29日}}</ref>，受此打击后在工作之余，闾丘露薇通过参加会计师培训班来自学，终于在1992年11月凭借流利的英语和参加会计师培训班的经历，被深圳[[普华永道|普华永道]]会计师事务所聘用<ref name="jizhe" /><ref name="career" />，经过一年的培训正式上班，初始工作是负责行政<ref>{{cite web | work=[[新浪网|新浪网]]| title=闾丘露薇：闾丘露薇：行走中的玫瑰| url=http://vip.book.sina.com.cn/book/chapter_37832_20232.html | date=2005-01-17| accessdate=2008年7月12日}}</ref>，在参加并通过了[[注册会计师|注册会计师]]专业资格考试，她升为高级[[审计员|审计员]]<ref name="文汇报" /><ref>{{cite web | work=人民网 | title=闾丘露薇：进巴格达没想过出名| url=http://www.people.com.cn/GB/yule/1082/2084256.html | date=2003-09-11| accessdate=2008年7月12日}}</ref>。

===职业生涯===
1995年，与丈夫移民[[香港|香港]]，应聘成为[[传讯电视|传讯电视中天频道]]的翻译。由于传讯电视财务状况不稳定，三个月后，她跳槽到[[TVBS|TVBS]]任电视记者；在TVBS工作了一年之后，又于1997年6月和同事[[曾瀞漪|曾瀞漪]]一起跟随其上司加盟开播不久的凤凰卫视新闻时事节目《[[时事直通车|时事直通车]]》，成为凤凰卫视的第一批记者<ref name="book5">{{cite web | title=《口述凤凰》 作家出版社   | work=闾丘露薇(1)| url=http://book.sina.com.cn/nzt/ent/koushufenghuang/47.shtml | date=2006-04-18| accessdate=2008年7月13日}}</ref>。除了继续其新闻采访工作之外还主持一档财经节目《经贸周刊》并受到好评，随后在全球采访过多次重大事件<ref name="rose" /><ref>{{cite web | work=新浪网 | title=行走中的玫瑰 关于选择(3)| url=http://book.sina.com.cn/longbook/cha/1105959729_xingzouzhongdemeigui/13.shtml | date= | accessdate=2008年7月12日}}</ref><ref name="lucky">{{cite web | work=中国新闻网 | title=闾丘露薇：我是幸运的，因为我在凤凰 | url=http://www.chinanews.com.cn/n/2004-04-01/26/420465.html| date= 2004-04-01| accessdate=2008年7月12日}}</ref><ref name="shanghai girl">{{cite web | work=中国新闻网 | title=同学谈闾丘露薇：她是个很典型的上海女孩 | url=http://news.sina.com.cn/c/2003-03-28/1513970726.shtml| date= 2003-03-28| accessdate=2008年7月13日}}</ref>。2001年11月，[[阿富汗战争|阿富汗战争]]开始，凤凰卫视决定派人前往采访，闾丘露薇率先报名，这是其第一次的战地采访，同时也成为第一位进入[[阿富汗|阿富汗]]的[[华人|华人]]女记者，這使她受到注目，甚至得到了[[朱镕基|朱镕基]]的关注<ref name="rose" /><ref name="shanghai girl" /><ref name="jizhe" />

==争议言论==
2013年6月6日，闾丘露薇在《[[羊城晚报|羊城晚报]]》发表文章《纽约式过马路》。文章称：
{{cquote|在纽约过马路，如果不仔细观察的话，会被简单地评论为“中国式过马路”……但只要仔细看就会发现，行人闯红灯一定是在确保没有车辆，或不影响驾车者车速的时候才过马路，而当行人绿灯亮时，车辆一定是会让走斑马线的行人优先。}}
闾丘露薇对“中国式过马路”和“纽约式过马路”的对比引起了网民几乎一面倒的质疑和吐槽。<ref>{{Cite web |url=http://news.m4.cn/2013-06/1208863.shtml |title=《纽约式过马路》引發网友“闾丘露薇体”群嘲 |access-date=2013-06-10 |archive-url=https://web.archive.org/web/20130610092925/http://news.m4.cn/2013-06/1208863.shtml |archive-date=2013-06-10 |dead-url=yes }}</ref><ref>{{cite web |url=https://www.guancha.cn/society/2013_06_07_149867.shtml |title=闾丘露薇谈“纽约式过马路”：跟中国不同 没车时美国人才闯红灯 |access-date=2018-09-28 |website=[[观察者网|观察者网]]}}</ref>
