'''曜越科技'''（'''Thermaltake'''，或常簡寫為'''Tt'''）是一家成立於1999年的[[台灣|台灣]]的[[電腦硬體|電腦硬體]]生產商，主要生產電腦用[[散熱片|散熱器]]、[[機殼|機殼]]、[[電源供應器|電源供應器]]與及其他電腦周邊裝置。

== 歷史 ==
曜越科技成立於1999年11月。目前曜越主要的生產基地位於[[中國|中國]]的[[廣東省|廣東省]][[東莞市|東莞市]][[橫瀝鎮|橫瀝鎮]]，並在[[美國|美國]]、[[荷蘭|荷蘭]]、[[德國|德國]]、[[中國|中國]]、[[日本|日本]]、[[澳洲|澳洲]]等地設有分公司。<ref>{{cite web |url=http://www.thermaltake.com/milestones.aspx |title=存档副本 |accessdate=2010-06-28 |deadurl=yes |archiveurl=https://web.archive.org/web/20100614012914/http://www.thermaltake.com/milestones.aspx |archivedate=2010-06-14 }} Retrieved 2010-06-24</ref>

曜越科技在初創時主要的產品為[[中央處理器|中央處理器]]與[[圖形處理器|圖形處理器]]散熱片（連或不連風扇）及一般機殼散熱設備，該公司大部分的產品都多少與散熱功能相關，因此才會以「Thermaltake」作為英文品牌名稱。所有產品以[[品質|品質]]、[[效能|效能]]及[[可靠度|可靠度]]為先作為賣點，即公司口號「關鍵三精神」（Key 3 Spirit）。

2000年，曜越科技製造出世界上首款渦輪式散熱風扇“Golden Orb”，供[[Intel|Intel]]的[[奔騰|奔騰]]處理器使用。

基於公司迅速的商業擴張，曜越科技2001年於[[中國大陸|中國大陸]]投資建廠以大幅增加產能。

2002年，獲得[[ISO_9001|ISO 9001]]質量認證。

2004年，發布了世界上首款CPU專用的液冷散熱系統。

2006年6月，完成海外分公司的機構重組，以更好地適應IT市場的迅速發展。

2007年12月12日，在台灣的[[店頭市場|店頭市場]]掛牌發行[[股票|股票]]。

2009年，曜越科技發布高端品牌「LUXA2」，定位是高端消費人群/高端DIY玩家，包括[[HTPC|HTPC]]、[[麥金塔電腦|麥金塔電腦]]之配件等。

2010年，為了配合以副品牌「Tt eSPORTS」進入遊戲用電腦（Gaming PC）市場，曜越科技也成立、贊助包括[[曜越太陽神|曜越太陽神]]等的職業[[電子競技|電子競技]]隊伍。<ref>http://www.thermaltake.com/brand.aspx Retrieved 2010-06-24</ref>

2011年1月，曜越科技總部搬遷至[[台北市|台北市]][[內湖科技園區|內湖科技園區]]。

== 主要產品 ==
[[File:Thermaltake_V9_Case.jpg|thumb]]
一直以來曜越科技主要生產電腦機殼、電源供應器、[[電腦硬體冷卻|電腦硬體冷卻]]設備（包括散熱片、散熱風扇、液冷散熱裝置乃至散熱模組等）、硬碟固定槽等電腦周邊設備。2010年後，除了上述的產品之外，還生產[[電腦鍵盤|鍵盤]]、[[滑鼠|滑鼠]]等電腦配件產品。
[[File:Thermaltake_Installed.jpg|thumb]]
雖然一些曜越科技的電腦[[機殼|機殼]]重量較高，但於電腦狂熱者及[[LAN_party|LAN party]]玩家中仍然受歡迎。其中一些著名的直立式機殼有Xpressar系列、Xaser系列、Armor系列、Kandalf系列、Wing系列、Spedo、Tsunami等，橫躺式機殼有Bach、Mozart系列以及[[LUXA2|LUXA2]]品牌的產品。該公司大部分機殼並不連[[電源供應器|電源供應器]]發售，但曜越科技亦獨立售賣電源供應器產品（ToughPower、PurePower等）。曜越科技也曾推出了一系列安裝於5.25吋[[硬碟|硬碟]]固定槽的電腦周邊，包括HardCano、Media LAB、Xpeaker等。
