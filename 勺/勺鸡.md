'''勺鸡'''（[[学名|学名]]：{{lang|la|''Pucrasia macrolopha''}}）为[[雉科|雉科]][[勺鸡属|勺鸡属]]的[[鸟类|鸟类]]，俗名'''角鸡'''、'''柳叶鸡'''。为[[中国国家二级重点保护野生动物名录|中国国家二级重点保护野生动物]]。

== 特征 ==
[[File:Фазан-коклас.jpg|left]]
雄鸟体长约55厘米，头顶褐棕色。具有冠羽，后有黑色长枕冠。后颈和颈侧棕黄面杂有白纹，体羽大部分呈灰色，各羽具“V”字形黑纹，纹中缀有黑点，中央尾羽和尾上覆羽并杂有栗色黑缘纵纹。雌鸟枕冠和尾较短。

== 分布 ==
分布于[[中国大陆|中国大陆]]的[[西藏|西藏]]、[[云南|云南]]、往北直抵辽宁、东至浙江、[[福建|福建]]、[[广东|广东]]等地，一般栖息于针阔混交林、密生灌丛的 多岩坡地以及山脚灌丛。该物种的模式产地在印度卡毛恩的阿尔摩拉。<ref name='id1744'/>

== 亚种 ==
