'''石元丈晴'''，[[日本|日本]][[电子游戏音乐|电子游戏作曲家]]、[[合成器程序员|合成器程序员]]及[[音乐家|音乐家]]，现就职于[[史克威尔艾尼克斯|史克威尔艾尼克斯]]（原[[史克威尔|史克威尔]]）。他于1999年加入史克威尔并担任《[[圣剑传说_玛娜传说|圣剑传说 玛娜传说]]》的合成器程序员，之后又在几部游戏中担任同职。在2002年他转为担任作曲家，首个作品是《World Fantastista》。此后他又为《[[美妙世界|美妙世界]]》、《[[最终幻想_纷争|最终幻想 纷争]]》和《[[最终幻想_零式|最终幻想 零式]]》等几部大制作游戏谱曲。除了在史克威尔艾尼克斯就职，他还担任SAWA——由他和Sawa Kato在2008年组建的乐队——的作曲人及吉他手，乐团发行了专辑《333》。

== 傳記 ==
據石元丈晴稱，他最早陷入音樂是因為他住在鄉村，沒有其他事情可做<ref name="JWint">{{cite web |title=J-Pop World - SAWA Interview |url=http://www.j-popworld.com/Interviews/SAWA.php |publisher=J-Pop World |accessdate=2011-02-18 |date=2009-01-05}}</ref>。在成為作曲家之前，他首先擔任合成器程式員——负责1999年的《[[圣剑传说_玛娜传奇|圣剑传说 玛娜传奇]]》。在几部游戏后，他开始担任作曲家，并首先为《World Fantasista》配乐。2004年他开始为[[最终幻想系列|最终幻想系列]]作品配乐，而在此之前他曾担任《[[最终幻想X|最终幻想X]]》的合成器程序员。他最后一部担任合成器程序员的作品是2005年游戏《[[王国之心II|王国之心II]]》；之后他就只为史克威尔艾尼克斯担任作曲家。他已经为两部最终幻想作品《[[最终幻想_纷争012|最终幻想 纷争012]]》和《[[最终幻想_零式|最终幻想 零式]]》配乐。

石元还是日本乐团SAWA的成员，在乐团中他以HIZMI署名。他和Sawa Kato在2008年10月建立乐团。Sawa为石元《美妙世界》原生的几首歌演唱并作词。乐队发行了一张专辑《333》<ref name="JWint" />。

== 风格与影响 ==
石元入选[[IGN|IGN]] 2008年十大JRPG谱曲家的第十名<ref>{{cite web |last=Sullivan |first=Meghan |title=Top Ten JRPG Composers |url=http://music.ign.com/articles/937/937683p1.html |publisher=[[IGN|IGN]] |accessdate=2011-02-17 |date=2008-12-18 |deadurl=yes |archiveurl=https://www.webcitation.org/6Ds5PdsZX?url=http://www.ign.com/articles/2008/12/19/top-ten-jrpg-composers |archivedate=2013-01-22 }}</ref>。石元以多种流派作曲，仅在《美妙世界》中就是用了摇滚、嘻哈、电子、流行和实验音乐<ref>{{cite web |last=Napolitano |first=Jayson |title=The World Ends With You: Come on and do the peace! (review) |url=http://www.originalsoundversion.com/the-world-ends-with-you-come-on-and-do-the-peace-review/ |publisher=Original Sound Version |accessdate=2011-02-17 |date=2008-06-24}}</ref>。
