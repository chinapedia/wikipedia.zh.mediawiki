
'''石鲁'''（{{bd|1919年||1982年|catIdx=Feng,冯}}），[[中国画|中国画]]大师，有[[长安画派|长安画派]]旗手之称。石鲁原名'''冯亚衍'''，[[四川|四川]][[仁寿县|仁寿]]人，因仰慕[[石涛|石涛]]、[[鲁迅|鲁迅]]，以石鲁为号。

== 生平 ==

石鲁出身仁寿大族家有庞大的庄园和万卷藏书的藏书楼．曾就读于[[成都|成都]]东方美术专科学校国画系（1934年~1937年）和华西协和大学文学院历史社会学系（1938年初）。1939年来到[[中国共产党|中国共产党]]控制下的[[陕甘宁边区|陕甘宁边区]]，在[[陕西|陕西]][[吴堡|吴堡]]青训班和[[延安公学院|延安公学院]]（1940年）学习。

在陕北期间历任西北文艺工作团美术组长、[[陕甘宁边区|陕甘宁边区]]文化协会美术干部、《群众画报》编辑、[[延安大学|延安大学]]文艺系美术班主任。

1949年后一直在[[西安|西安]]作画，历任西北美术工作者协会副主任、《西北画报》社社长、[[中国美术家协会|中国美术家协会]]陕西分会主席、[[陕西国画院|陕西国画院]]名誉院长、[[中国美院|中国美院]]常务理事、中国画研究院委员、陕西美协主席等。

1964年《[[转战陕北|转战陕北]]》一画被认为犯有严重政治错误，精神受到打击。1965年，石鲁患上[[精神分裂症|精神分裂症]]。[[文化大革命|文化大革命]]中曾遭受政治打击，几经[[牛棚|牛棚]]出逃，被抓回后因其精神分裂才未被枪决。1982年因癌症去世。

== 风格 ==

石鲁性格外向，被称为狂人；画风大胆激进，富于试验性，开一代风气，作品个性鲜明，后期作品画诗书刻融为一体．曾有“野怪乱黑”的评价。石鲁以一首[[打油诗|打油诗]]回应了这种评价：「人骂我野我更野，搜尽平凡创奇迹。人责我怪我何怪，不屑为奴偏自裁。人谓我乱不为乱，无法之法法更严。人笑我黑不太黑，黑到惊心动魂魄。野怪乱黑何足论，你有嘴舌我有心。生活为我出新意，我为生活传精神。」

== 主要作品 ==
《南泥湾途中》、《[[转战陕北|转战陕北]]》、《华岳之雄》、《荷雨图》等，此外还有著作《石鲁学画录》。

2012年9月21日，石鲁家人将家中所藏150件石鲁作品捐献给中国国家博物馆。<ref>{{cite news|title=石鲁150件书画作品捐赠国家博物馆|url=http://news.artxun.com/zhongguomei-1719-8591126.shtml|accessdate=2016-05-24|agency=千龙网|date=2012-09-21}}</ref>2010年，石鲁家属委托拍卖公司将《山区修梯田》拍卖，但拍卖未能成功，拍卖公司迟迟不愿将画作交还，石鲁家属通过诉讼才使得这幅画在2014年8月8日进入中国国家博物馆。<ref>{{cite news|title=历经诉讼石鲁千万元名作《山区修梯田》终回归国博|url=http://news.sina.com.cn/o/2014-08-08/225430655600.shtml|accessdate=2016-05-24|agency=中国新闻网|date=2014-08-08}}</ref>2012年，石鲁家人捐赠了第一批作品。第二批作品原计划在2014年捐出，但此时石鲁次子石强突然离世，中国国家博物馆前去接收作品时发现部分作品和文献不翼而飞，共计158件套。2016年5月20日，中国国家博物馆通告，石鲁家人发表公开声明追索这批作品和文献。<ref>{{cite news|title=著名书画家石鲁158件遗作下落不明，国家博物馆发声明追索|url=http://cul.sohu.com/20160520/n450607734.shtml|accessdate=2016-05-24|agency=澎湃新闻|date=2016-05-20}}</ref>
