'''石碑胡同'''是位于中国[[北京市|北京市]][[西城区|西城区]]中部的一条[[胡同|胡同]]，现已拓宽为道路。

==简介==
石碑胡同位于[[西长安街|西长安街]]南侧，为南北走向，南至[[东绒线胡同|东绒线胡同]]，北至[[西长安街|西长安街]]，全长314米，过去平均宽度为5米。“石碑胡同”这一名称始称自[[明朝|明朝]]，因石碑而得名<ref>{{Cite web |url = http://www.bjdclib.com/subdb/laneculture/lane/200907/t20090702_4230.html |title = 石碑胡同 |author =  |date = 2009-07-02 |publisher =  东华流韵|language =  |accessdate =  |quote =  }}</ref>。关于石碑的由来，相传有两种说法：其一是说石碑胡同南口是[[明朝|明朝]][[锦衣卫|锦衣卫]]旧址，此处曾有锦衣卫题名碑。《[[顺天府|顺天府]]志》记载：“[[刑部|刑部]]在[[北京皇城|皇城]]西，即明锦衣卫故址称建。大堂壁间明锦衣卫旧有锦衣卫题名碑，后毁于火。”另一说是在石碑胡同北口，曾有一座[[汉白玉|汉白玉]]质地的下马碑，刻有“官员人等，至此下马”。明朝的锦衣卫旧址即今国家大剧院所在地。2007年9月国家大剧院竣工<ref name=lvyou/>。 

石碑胡同东侧是[[国家大剧院|国家大剧院]]。西侧自北至南分别和以下胡同相交：[[东安福胡同|东安福胡同]]、[[南文昌胡同|南文昌胡同]]、东绒线胡同<ref name=lvyou>旦其、于志强，石碑胡同 阳春白雪与市井生活的分界线，旅游世界2014(9):104-109</ref>。

2007年，为畅通[[国家大剧院|国家大剧院]]周边道路交通，国家大剧院西侧的石碑胡同扩建为双向各两车道的道路，车辆可从西长安街经石碑胡同及其以南的[[兵部洼胡同|兵部洼胡同]]一直到达[[前门西大街|前门西大街]]<ref>{{Cite web |url =http://news.xinhuanet.com/auto/2007-03/21/content_5874200.htm  |title = “巨蛋”南侧路将连通长安街和前门西大街 |author =  |date =2007-03-21  |publisher = 新华网 |language =  |accessdate =  |quote =  }}</ref>。
