'''石漫滩水库'''位于[[河南省|河南省]][[舞钢市|舞钢市]]，距市区仅2公里，系截流[[洪河|洪河]]支流[[滚河|滚河]]而成。原水库始建于1951年，是中华人民共和国建设的第一座大型水库，坝型为均质土坝。1975年8月8日，因758特大暴雨，最高坝前水位111.40米，超坝顶1.57米，致水库漫坝溃绝，下游[[田岗水库|田岗水库]]随之漫决。

1993年9月水库复建工程开工，1996年12月主体工程完工，1998年7月竣工。工程总投资2.56亿元，控制流域面积230平方公里。按百年一遇洪水设计，千年一遇洪水校核。设计水位110.65米，相应库容1.03亿立方米，校核水位112.05米，相应库容1.2亿立方米。大坝为全断面碾压混凝土重力坝，坝顶高程112.50米，坝长645米，最大坝高40.5米，坝顶宽7米。最大泄量3927立方米每秒。是一座以防洪为主、兼顾城市工业供水及旅游的大型水库。库区全赔高程108.53米，移民高程110.00米。<ref>{{cite web |url=http://www.hnfishtec.cn/reservoir/0925347.html |title=石漫滩水库 |accessdate=2009-10-18 |author=河南省水产技术推广站 |language=zh-hans |archive-url=https://web.archive.org/web/20160306055813/http://www.hnfishtec.cn/reservoir/0925347.html |archive-date=2016-03-06 |dead-url=yes }}</ref><ref>{{cite web |url=http://www.mwr.gov.cn/slfjq/20040917/40271.asp |title=石漫滩水库风景区(图)   |accessdate=2009-10-18|author= |language=zh-hans }}</ref><ref>{{cite web |url=http://www.erlangshanjingqu.cn/plus/view.php?aid=36 |title=石漫滩水库的历史 |accessdate=2009-10-18 |language=zh-hans |deadurl=yes |archiveurl=https://web.archive.org/web/20110406232305/http://www.erlangshanjingqu.cn/plus/view.php?aid=36 |archivedate=2011-04-06 }}</ref>

以水库为中心已建设'''石漫滩水利风景区'''及[[石漫滩国家森林公园|石漫滩国家森林公园]]。

石漫滩水库是中国唯一一座水毁大坝和复建大坝并存的水库。<ref>{{cite web |url=http://www.hnly.com/html/20089/9/22/20080922150251593.shtml |title=石漫滩国家森林公园 |accessdate=2009-10-18 |language=zh-hans }}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>
