'''石井书院'''位於[[中國|中國]][[福建省|福建省]][[晋江市|晋江市]][[安海镇|安海镇]]，前身为鳌头精舍，系修造[[安平桥|安平桥]]的[[黄护|黄护]]响应安海镇官[[朱松|朱松]]所倡，带头捐募所建，曾与[[泉山书院|泉山书院]]、[[小山书院|小山书院]]、[[欧阳书院|欧阳书院]]并称泉州“四大书院”<ref name="黄冬虹">{{Cite news|url=http://www.qzwb.com/rdzt/ctrj/content/2011-04/26/content_3648900.htm|title=泉州古代四大书院|author=黄冬虹|language=zh-cn|publisher=泉州网-泉州晚报|date=2011-04-26|accessdate=2016-11-05}}</ref>。有安海八景“石井书香”<ref name="苏向东">{{Cite news|url=http://cul.china.com.cn/2011-11/04/content_4597470.htm|title=圣域津梁，理学渊源开石井|author=苏向东|language=zh-cn|publisher=文化中国-中国网|date= 2011-11-04|accessdate= 2016-11-05}}</ref>之称。

== 修建历史 ==
石井书院最早原名'''鳌头精舍'''，地址位于安海镇鳌头镜，

唐建中元年（公元780年），[[常衮|常衮]]任福建观察使，建私塾教育自家子弟名为“鳌头社”<ref name="陈增瑞">{{Cite news|url=http://www.fjsen.com/yhzh/2009-10/08/content_3303200.htm|title=安海石井书院源流|author=陈增瑞|language=zh-cn|publisher=炎黄纵横|date=2009-10-08|accessdate=2016-11-05}}</ref>。太平兴国时期泉州府学兴起，安海兴建学社，沿用“鳌头社”之名起名“鳌头精舍”。

宋绍兴年间，是时任安海首任镇官[[朱松|朱松]]，为教导民众，开启民智。发起了兴学的倡议，[[黄护|黄护]]等乡士纷纷响应捐募，扩建鳌头精舍。朱松之子[[朱熹|朱熹]]，任同安县主簿时也曾到此讲学。公元1165年朱熹的学生为纪念朱松、朱熹二人在精舍旁另辟“二朱先生祠”名为'''朱祠'''<ref name="陈增瑞">{{Cite news|url=http://www.fjsen.com/yhzh/2009-10/08/content_3303200.htm|title=安海石井书院源流|author=陈增瑞|language=zh-cn|publisher=炎黄纵横|date=2009-10-08|accessdate=2016-11-05}}</ref>。

宋嘉定四年，镇官[[游绛|游绛]]应安海镇乡贤的请求，向当时的泉州郡守邹应龙申请建造书院。邹应龙应安海乡贤之请求并拨公帑四十万用于修建书院，同时下令泉州[[漕司|漕司]]，泉州[[市舶司|市舶司]]捐助，并派[[朱在|朱在]]（朱熹第三子）主持建院工作，在“鳌头精舍”的基础上进行扩建。隔年（公元1212年）书院落成，与朱祠合二为一。因安海宋朝年间曾设立津卡名为“石井津”故书院名称为“石井书院”。又因书院中祭祀“二朱”故安海镇人称之为'''朱祠'''。

明弘治十年重修，明成化十二年重修，清顺治因迁界被毁，清康熙年间在原址上重修，清乾隆年间扩建，清光绪年间重修。公元2008年晋江市政府重新修缮<ref name="王秀钦">{{Cite news|url=http://www.fjsen.com/d/2010-12/27/content_3964882.htm|title=晋江安海“石井书院”昨日重修开馆|author=王秀钦|language=zh-cn|publisher=东南网|date=2010-12-27|accessdate=2016-11-05}}</ref>。
==建筑格局 ==
[[File:石井书院重修记.jpg|缩略图]]
石井书院最早是按州郡学府的规格修建的<ref name="苏向东">{{Cite news|url=http://cul.china.com.cn/2011-11/04/content_4597470.htm|title=圣域津梁，理学渊源开石井|author=苏向东|language=zh-cn|publisher=文化中国-中国网|date= 2011-11-04|accessdate= 2016-11-05}}</ref>，中间为以大成殿为主轴，后殿为尊德堂，大成殿祭祀孔子以及朱松、朱熹两父子，轴线两侧建有富文、敏行、移忠、立信四个书斋，并有院内有杏坛、乐轩（放置编钟）、庑舍，院前建有碑坊（毁于公元1575年前后）。由于书院经历百年风雨，残破不堪，在加上清末西学东渐，只保留的原本的祠堂部分，学院部分被弃置。公元2008年晋江市政府进行修缮重修并恢复到原来三进院落的规格。

== 影响 ==
被誉为“闽学开宗”书院曾聘请过许多著名理学大儒为山长：如宋代[[顾长卿|顾长卿]]、[[余谦一|余谦一]]；元代[[杨相孙|杨相孙]]、[[陈玄|陈玄]]；明代的[[陈惟白|陈惟白]]、[[刘绍祖|刘绍祖]]；清代[[陈檗仁|陈檗仁]]对书院的教育水平有很大的提升<ref name="胡金兰">{{Cite news|url=http://www.fjedu.com.cn/magazines/mgz4/jcdaodu/deyu/2677CCA667094CA4B276C10B0DFDF1E9.html|title=“闽学开宗圣地”——记安海石井书院|author=胡金兰|language=zh-cn|publisher=福建教育-育德|date=2013-07-02|accessdate=2016-11-05}}</ref>。

书院也教育或影响过许多人，例如[[王慎中|王慎中]]、[[黄汝良|黄汝良]]、[[庄概|庄概]]、[[郑成功|郑成功]]、[[黄虞稷|黄虞稷]],也因为石井书院的设立，安海自宋绍兴年间到清末举人及第达数百人<ref>《安海志》卷二十三《选举》第254-258页</ref>。

书院内还创办过一所新式学堂——晋江安海养正小学，它也是本省最早的新式学堂之一。<ref name="倪少林">{{Cite news|url=http://www.66163.com/Fujian_w/news/fjqb/010119/4_11.html|title=安海“石井书院”|author=倪少林|language=zh-cn|publisher=福建侨报|accessdate=2016-11-05}}</ref>
