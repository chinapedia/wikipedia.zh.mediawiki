'''石马岙社区'''是位于[[浙江省|浙江省]][[岱山县|岱山县]][[高亭镇|高亭镇]]辖下的一个[[渔农村新型社区|渔农村新型社区]]，其区域范围与石马岙村一致，社区管理委员会和[[村民委员会|村民委员会]]合一。人口约5000余。

== 名称来源 ==
石马岙社区之名源自境内“石马庙”。在1992年浙江省撤区扩镇并乡以前，这里也是原岱中乡政府驻地。2005年[[舟山市|舟山市]]进行村庄合并，并设立渔农村新型社区后，原石马岙村与蟹钳岙村、刘家岙村、陈家间村合并，设立新的石马岙村，并单建石马岙社区。

== 地理 ==
石马岙社区位于[[岱山岛|岱山岛]]中部，为岛内重要的交通结点。县城[[高亭|高亭]]通往西部各镇的道路，于本村境内分岔，东北往[[岱东镇|岱东镇]]，西北往[[东沙镇|东沙镇]]，以西往[[岱山机场|岱山机场]]并通往[[岱西镇|岱西镇]]。村境东北部，与岱东镇虎斗村交界的石马岙岭，为[[岱山岛|岱山岛]]内最为著名的长陡坡。

== 人口 ==
石马岙社区总人口约5000余。主要为农业人口。

== 石马岙烈士陵园 ==
