'''石田与朝仓'''（{{lang-ja|石田とあさくら}}）是日本漫畫家[[マサオ|マサオ]]所创作的[[日本|日本]]漫画作品。簡稱'''石あさ'''（石朝）。2013年改编為电视动画。<ref>{{cite web |url= http://www.animenewsnetwork.com/news/2012-11-17/ishida-to-asakura-gag-comedy-manga-gets-tv-anime|title= Ishida to Asakura Gag Comedy Manga Gets TV Anime|date=2012-11-17|accessdate= 2012-11-18|publisher= [[Anime_News_Network|Anime News Network]]}}</ref>

== 简介 ==
作者原本将该作品在自己网站以及[[pixiv|pixiv]]上公开，此后在在漫画杂志《ヤングキング》、《月刊ヤングキング》（[[少年画报社|少年画报社]]刊）上连载。但最終回在漫画杂志刊载时存在不同的结局。

== 人物 ==
;あさくら（中文翻译为朝仓）
:[[声優|声]] - [[中本順久|中本順久]]
:本作主人公。有天然卷的头发和下巴长脸特征的男子高中生。说话时会稍稍停顿最后的一个字。石田和山田的情感对象，除女性胸部以外的东西没有兴趣。将来梦想是成为女子学校的老师。
;石田（いしだ）
:声 - [[森嶋秀太|森嶋秀太]]
:本作主人公。朝仓的同班同学。光头，額头有M字型发痕。朝仓在小学时的儿时玩伴，对朝仓有好感。夢想是将来与朝仓一起经营[[花店|花店]]。
;山田（やまだ）
:声 - [[櫻井トオル|櫻井トオル]]
:男性，朝仓的同班同学。石田的情敌。特征为戴眼鏡、超长门牙、体操服。梦想是将来和朝仓一起经营书店。
;佐藤（さとう）
:声 - [[中野さいま|中野さいま]]
:女性，朝仓的同班同学。巨乳，因为朝仓的一句抱不平而喜欢朝仓。
;木下（きのした）
:声 - [[島本須美|島本須美]]
:朝仓的班主任，童颜巨乳，在学生间人气高。实际上是个[[伪娘|伪娘]]，有两个孩子。
; 奈々子
: 声 - [[鳥部万里子|鳥部万里子]]
: 木下的女儿。
; 山口先生
: 声 - [[郷田ほづみ|郷田ほづみ]]
: 新任教師。
; アイアン山田一号（iron山田一号）
: 声 - 櫻井トオル
: 山田的机器人。
; 女仆爱丽丝
: 声 - [[みくる|みくる]]
: 石田在女仆咖啡店扮演的女仆。
; 店長
: 声 - [[光明寺敬子|光明寺敬子]]
: 石田工作的女仆咖啡店店長。
; 山田之妹
: 声 - みくる
: 与兄同様戴眼鏡穿体操服，并穿[[灯笼裤|灯笼裤]]上学。因为意想不到的变故爱上石田。
; おっぱい星人（[[乳房|乳]]星人）
: 声 - [[四宮豪|四宮豪]]
: 朝仓撞上的UFO上的侵略性宇宙人。
