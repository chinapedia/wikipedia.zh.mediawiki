'''石狩鍋'''（日語:いしかりなべ ‧ Ishikari nabe）是一道[[北海道|北海道]]的鄉土料理，是以[[味噌|味噌]]調味過的[[鮭魚|鮭魚]]為主要食材的一道[[火鍋|火鍋]][[料理|料理]]。

石狩鍋雖然常常會被跟以醃製[[鮭魚|鮭魚]]為主要食材的[[三平汁|三平汁]]搞混，不過石狩鍋所使用的[[鮭魚|鮭魚]]並不是醃製鮭魚，而是沒調味過的生鮭魚。

== 材料 ==
石狩鍋的湯底製作是將鮭魚肉連魚骨，和[[豆腐|豆腐]]、[[洋蔥|洋蔥]]、[[韭菜|韭菜]]、[[白蘿蔔|白蘿蔔]]、[[捲心菜|捲心菜]]、[[香菇|香菇]]、[[胡蘿蔔|胡蘿蔔]]、[[京蔥|京蔥]]等蔬菜，加入[[海苔|昆布]]和[[味噌|味噌]]煮成。有時候還會添加[[酒糟|酒糟]]、[[牛油|牛油]]或[[牛奶|牛奶]]等調味。最後再加上[[山椒|山椒]]粉食用。

== 由来 ==
石狩鍋是以鮭魚產量豐富的[[石狩川|石狩川]]命名的，該處每年都有鮭魚逆流而游上河川上游產卵。
在江戶時代，當地的[[漁夫|漁夫]]在經歷嚴冬下辛勞的捕鮭魚活動後，都會將一些新鮮的鮭魚魚獲，加上洋蔥和白菜等蔬菜熬成味噌湯，作為慶功和慰勞。

1880年（明治13年）在[[石狩市|石狩市]][[石狩川|石狩川]][[河口|河口]]附近開業的食店「金大亭」就把這種料理當成商品出售。直至今天，「金大亭」仍然沿用「元祖石狩鍋」的名號。

== 十勝鍋 ==
十勝鍋（日語:とかちなべ Tokachi Nabe）是在[[帶廣|帶廣]]一帶地方的郷土料理。用和石狩鍋同樣的廚具，加入名産[[豬肉|豬肉]][[味噌|味噌]]做成湯底。

名字是以同樣以鮭魚產量豐富聞名的[[十勝川|十勝川]]命名的。
