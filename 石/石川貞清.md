'''石川貞清'''（生年不詳－1626年5月3日）是[[安土桃山時代|安土桃山時代]]至[[江戶時代|江戶時代]]前期的武將和[[大名|大名]]。[[犬山城|犬山城]]城主。

== 生平 ==
生年不詳，出身於[[美濃國|美濃國]]鏡島。仕於[[豐臣秀吉|豐臣秀吉]]，擔任[[使番|使番]]、金切裂指物使番。[[天正_(日本)|天正]]18年（1590年），在[[小田原征伐|小田原征伐]]中立下功績，受領[[尾張|尾張]]犬山1萬2千石。還擔任[[豐臣家|豐臣家]]在[[信濃|信濃]]木曾的[[太閤藏入地|太閤藏入地]]10萬石的[[代官|代官]]。天正19年（1591年），負責在[[文祿之役|文祿之役]]中作為據點的[[肥前國|肥前國]][[名護屋城|名護屋城]]的[[普請|普請]]工事。[[慶長|慶長]]4年（1599），領地被加增至12萬石。

慶長5年（1600年），在[[關原之戰|關原之戰]]中加入西軍，與[[稻葉貞通|稻葉貞通]]、[[稻葉典通|典通]]父子、[[稻葉方通|稻葉方通]]、[[加藤貞泰|加藤貞泰]]、[[關一政|關一政]]、[[竹中重門|竹中重門]]等人在居城犬山城中籠城，不過他們秘密向東軍的[[井伊直政|井伊直政]]送出書信，約定擔當內應，此時自身出城參加關原的本戰，在宇喜多隊的右翼口北野附近佈陣，在戰敗後，所領被沒收，不過因為在犬山籠城期間，解放支持東軍的木曾鄉士等人質，因此[[池田輝政|池田輝政]]以千枚[[黄金|黄金]]為其求情，所以得以保全性命。之後剃髪並號'''宗林'''，在[[京都|京都]]隱居，以[[茶人|茶人]]、[[商人|商人]]的身份渡過餘生。慶長18年（1613年），被[[幕府|幕府]]以5百石召出（『[[德川除封錄|德川除封錄]]』）。在[[寬永|寬永]]3年（1626年）死去。子孫成為商人。

== 與真田信繁的關係 ==
*妻子是[[真田信繁|真田信繁]]（幸村）的七女於金殿（{{lang|ja|おかね}}）。成為商人並成功後，援助在[[大坂之陣|大坂之陣]]中戰死的信繁的正妻（於金的母親，[[大谷吉繼|大谷吉繼]]的女兒或養女）[[竹林院|竹林院]]，而且在[[京都市|京都市]][[右京區|右京區]][[龍安寺|龍安寺]]的[[塔頭|塔頭]][[大珠院|大珠院]]建造信繁夫妻的墓所[[五輪塔|五輪塔]]。
