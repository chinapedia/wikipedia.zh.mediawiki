'''石晓辉'''（{{bd|1963年|10月}}），男，汉族，[[重庆|重庆]][[九龙坡|九龙坡]]人，工学博士，教授。1994年12月参加工作，1998年5月加入[[中国共产党|中国共产党]]，2011年7月担任[[重庆理工大学|重庆理工大学]]校长、党委副书记<ref>{{cite web |url=http://www.gx211.com/news/201596/n4344292547.html |title=重庆理工大学校长石晓辉一行到浙江吉利汽车研究院走访并看望校友 |date=2015-09-06 |accessdate=2015-12-11  |publisher=中国高校之窗 }}</ref>。

== 生平 ==
1980年9月至1985年9月，石晓辉在[[清华大学|清华大学]]汽车工程系汽车专业学习，获工学学士学位。1985年9月至1994年12月，石晓辉在[[重庆大学|重庆大学]]机械一系工程机械专业学习，先后获工学硕士学位和工学博士学位。

1994年12月至1995年3月，石晓辉在[[重庆工业管理学院|重庆工业管理学院]]担任教师，1995年12月，升为副教授。1995年3月至1997年1月，任重庆工业管理学院汽研所所长。1997年1月至1997年12月，任重庆工业管理学院车辆工程系副主任。

1997年12月至1998年3月，石晓辉任重庆工业管理学院副院长。1998年3月至1999年5月，任重庆工业管理学院副院长兼院机械系主任。1999年5月至1999年12月，任重庆工学院副院长、党委委员兼院机械系主任。1999年11月，升为教授。1999年12月至2009年5月，任重庆工学院副院长、党委委员<ref>{{cite web |url=http://xxgk.cqut.edu.cn/gaikuang/jibenqingkuang/renmian.aspx |title=领导选拨，石晓辉同志的简要情况 |date=2011-07-07 |accessdate=2015-12-09 |publisher=重庆理工大学信息公开网 |deadurl=yes |archiveurl=https://web.archive.org/web/20160304194625/http://xxgk.cqut.edu.cn/gaikuang/jibenqingkuang/renmian.aspx |archivedate=2016-03-04 }}</ref>。

== 近期 ==
2009年5月至2011年7月，石晓辉任重庆理工大学副校长、党委委员。

2011年7月起，石晓辉任中共重庆理工大学委员会委员、常委、副书记，重庆理工大学校长。
