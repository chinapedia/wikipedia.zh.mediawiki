'''石北本線'''（{{jpn|j=石北本線|hg=せきほくほんせん|rm=Sekihoku honsen|f=y}}）是一條由[[北海道|北海道]][[旭川市|旭川市]]的[[新旭川站|新旭川站]]起，途經[[北見市|北見市]]的[[北見站|北見站]]連結[[網走市|網走市]]的[[網走站|網走站]]，屬於[[北海道旅客鐵道|北海道旅客鐵道]]（JR北海道）的鐵路線（[[地方交通線|地方交通線]]）。
== 概要 ==
石北本線，顧名思義，即連結北海道石狩地區與北見地區的路線。以前連結兩地的路線有三條，第一條路線為「'''[[名寄本線|名寄本線]]'''」，由[[宗谷本線|宗谷本線]]的[[名寄站|名寄]]經過[[興部車站|興部]]、[[遠輕站|遠輕]]到達[[北見站|北見]]、[[網走車站|網走]]。第二條是由十勝地區的[[池田車站_(北海道)|池田]]到達北見、網走的「'''網走本線'''」（後來的池北線、[[北海道池北高原鐵道|北海道池北高原鐵道]]）。第三條由[[旭川站|旭川]]，越過北見峠，也就是現在的'''石北本線'''。而在1932年打通北見峠後，石北本線成為石狩與北見兩地區的最短路線。
<!--
当初、札幌方面と北見を結ぶルートは札幌駅 - 旭川駅 - 富良野駅 - 池田駅 - 北見駅のルートであり、その後根室本線の滝川駅 - 富良野駅間の開通でさらに短絡化された。のち名寄本線が開通して少し短絡され、その後石北本線の全通により現在のルートが最短となった。
石北本線が名実ともに現在の形となるのは、1961年に線区の整理統合が行われてからである。現在も遠軽駅の線形がスイッチバックであることが、石北本線の複雑な生い立ちを物語っている。
-->
石北本線的常紋隧道1914年開通，其長度僅507m，但工期卻高達3年，較同時間的大阪電氣鐵道（現在的[[近畿日本鐵道|近畿日本鐵道]]）奈良線的生駒隧道長度3388m，工期33個月相比，工程的難度可想而知。1968年發生[[十勝沖地震|十勝沖地震]]，造成隧道損壞，於1970年修復時，工人發現隧道壁內埋有人骨，雖然經過調查後證實為修築隧道的罹難者（即[[打生樁|打生樁]]，為某種目的而犧牲的人），但是常紋隧道的靈異之說已不脛而走。

== 運轉情形 ==
=== 區域運輸 ===
大體上分為「旭川 - 上川」、「上川、白瀧 - 遠輕」、「遠輕、留邊蘂 - 北見 - 網走」三個主要系統。包含特別快速「北見」（{{lang|ja|きたみ}}），全部列車均實施[[單人運轉|單人運轉]]。
==== 旭川 - 上川間 ====
雖然路線畫分上起點為新旭川車站，但實際上列車均以旭川車站為起車點行駛。由於旭川的近郊區間範圍最遠可達上川，列車的班次上設定為1天8個往復班次，除了中午以外每1 - 2個小時發出一班。亦有部分列車行駛區間為旭川 - 東旭川、當麻、伊香牛等。
==== 上川 - 白瀧 - 遠輕間 ====
上川 - 白瀧間除了特急「鄂霍次克」（{{lang|ja|オホーツク}}）與特別快速「北見」之外，普通列車1天僅1個往復班次。基於此區間內人口數稀少，[[國鐵分割民營化|國鐵分割民營化]]後有3座車站被降為號誌站。雖然白瀧 - 遠輕間的普通列車（包括特別快速「北見」）共有5個往復班次，但是舊白瀧站與下白瀧站停靠的班次上行只有3班，下行更只有1班。
==== 遠輕 - 北見 - 網走間 ====
遠輕 - 北見、網走間的普通列車有4個往復班次，而遠輕 - 生田原間有開行區間列車。另外本區間運轉密度最低為生田原 - 金華間，包含「北見」在內的普通列車1天僅5個往復班次。
金華以東則以北見、網走為中心，由金華、留邊蘂、東相内為起車點，經由北見開往網走的列車班次較多。留邊蘂 - 網走的區間除了中午以外，每1至2小時發出一班。有部分列車直通釧網本線的知床斜里車站、綠車站。值得一提的是，特別快速「北見」在北見車站的停靠時刻恰可接駁來往於釧網本線的旅客。
* 1990年 - 1995年間，網走站 - 北見站 - 遠輕站有開行過每天1個往復班次的快速列車「網走」（{{lang|ja|あばしり}}）。
** 停車站：網走站 - 女滿別站 - 美幌站 - 端野站 - 北見站 - （中間各站停車） - 遠輕站

* [[舊白瀧站|舊白瀧站]]（A46）：2016年3月26日廢除<ref group="報道" name="jrhokkaido-press-20151218-3" />。白瀧站－下白瀧信號場間（88.3公里）

* {{link-ja|鳥之澤臨時乘降場|鳥ノ沢仮乗降場}}：1971年7月1日廢除<ref name="chizucho-42-43" />。緋牛內站－美幌站間（201.7公里）
