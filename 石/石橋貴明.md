'''石橋貴明'''({{bd|1961年|10月22日|||}})、日本的[[搞笑藝人|搞笑藝人]]、主持人、[[歌手|歌手]]、[[演員|演員]]、經紀公司老闆。主持拍檔為同為搞笑藝人[[木梨憲武|木梨憲武]]。有過二次婚姻，其現任妻子也是知名演員[[鈴木保奈美|鈴木保奈美]]。

== 概述 ==
* [[隧道二人組|隧道二人組]]的成員。[[身高|身高]]182cm(曾在音樂節目「[[歌番|歌番]]」表明自己是181cm)。前職是酒店人員。

* [[血型|血型]]是A型。興趣有打棒球、打高爾夫、看電影、飲酒、吃燒肉。

* [[愛稱|愛稱]]有「'''タカさん'''」(貴先生)「'''タカちゃん'''」(小貴)「'''イシバシ'''」(石橋)「'''タカアキ'''」(貴明)等。20代時女性影迷主要以「'''タカちゃん'''」來稱呼他。另外、也有人以「'''バシタカ'''」(橋貴)來稱呼他。拍檔木梨憲武、[[和田秋子|和田秋子]]、[[水谷豊|水谷豊]]、[[古舘伊知郎|古舘伊知郎]]等都以「'''タカアキ'''」來稱呼他(木梨也有以全名稱呼他)。此外，{{link-ja|HIROMI|ヒロミ}}會以「石橋くん」稱呼他、[[勝俣州和|勝俣州和]]則以「貴明さん」稱呼石橋。飲友[[コージー冨田|コージー冨田]]在電視以「石橋貴明」來直接稱呼他具多。[[次長課長|次長課長]][[河本準一|河本準一]]在電視中會叫石橋做「ジジィ!」。深交[[宮崎縣|宮崎縣]][[知事|知事]][[東国原英夫|東国原英夫]]則「タカちゃん」來稱呼。

== 經歴 ==
*生於[[東京都|東京都]][[葛飾區|葛飾區]]的裕福家庭、小學一年級時、父・武夫所經營的「石橋化成」[[破産|破産]]。其後「搬往」(晚上逃走)[[板橋區_(日本)|板橋區]][[成増|成増]]、從此以後被迫要過著極為貧困的生活。
*小學六年級時參與演出『[[アフタヌーンショー|アフタヌーンショー]]』的『ちびっ子ものまね大会』、中學生時則有演出『[[ぎんざNOW!|ぎんざNOW!]]』。
*上高中時受到大他7歲的哥哥影響，從此開始打棒球，之後加入了[[帝京高等學校|帝京高等學校]]棒球部。一年級秋天時守備位置從二壘手兼任投手，人稱「帝京的秘密武器」，不過就這樣被當成秘密藏到畢業。高校時代的後輩有養樂多隊投手[[伊東昭光|伊東昭光]]

*在1984年～1986年與[[木梨憲武|木梨憲武]]、[[可愛かずみ|可愛かずみ]]、[[川上麻衣子|川上麻衣子]]、[[柄沢次郎|柄沢次郎]]、[[前田耕陽|前田耕陽]]合演深夜劇場[[三角關係的憂鬱|三角關係的憂鬱]]

*1998年、與當時的女藝人[[鈴木保奈美|鈴木保奈美]]再婚。當時鈴木已經懷孕3個月、結婚後便從演藝界引退了。另外鈴木也有離婚經驗。與鈴木再婚後、生了3個女兒。

2005年、「MLB主義」で共演した[[牛島和彦|牛島和彦]]監督が率いる[[横浜ベイスターズ|横浜ベイスターズ]]のエグゼクティブ・アドバイザーに就任(以前是[[讀賣巨人|讀賣巨人]]隊球迷)。

*以前會抽[[七星|七星]]牌的香煙，不過己從2005年開始戒煙。
