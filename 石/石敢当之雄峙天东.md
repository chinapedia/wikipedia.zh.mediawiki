{{engname|《石敢当之雄峙天东》|Dare Stone Male Tiandong}}，原名'''《降魔英雄传》'''，根据《[[西游记后传|西游记后传]]》及[[泰山|泰山]][[石敢当|石敢当]]传说改编<ref name="改编"/>，[[钱雁秋|钱雁秋]]执导、[[张子健|张子健]]主演的一部中国古装神话剧。2014年6月開機，10月殺青，2015年2月15日起在[[山东卫视|山东卫视]]、[[贵州卫视|贵州卫视]]晚间黄金档两集连播。

== 剧情概述 ==
大唐年间，魔头奎刚纠结魔界、凡间的各种恶势力，妄图颠倒乾坤，独霸三界。玉帝派哪吒下界寻找三界护法。东岳泰山有一块巨石名为石敢当，受天地灵气孕化。天界异象，三只巨怪逃离天庭作祟人间，石敢当挺身而出，用神力保护百姓，赢得爱戴。五百年后，天神奎刚聚集大批妖邪向三界进攻，玉帝及诸神尽遭毒手，唯有石敢当逃脱。石敢当勇闯阿修罗界，救出玉帝及诸神，最终消灭奎刚，令三界恢复了往日的宁静祥和。至此，石敢当成为仙凡两界名声大噪的英雄人物，除妖降魔，威震四方，人间也有了以泰山石刻以“石敢当”之名来的习俗。

== 制作 ==
本剧是钱雁秋根据自己之前的编剧作品《[[西游记后传|西游记后传]]》，融合[[泰山|泰山]]石敢当的传说改编而成<ref name="改编">{{cite news|url=http://yule.sohu.com/20150210/n408897641.shtml|title=山东卫视《石敢当》将播 众星云集谱神话巨制|date=2015-02-10|publisher=搜狐|accessdate=2015-02-21}}</ref>。制片人魏巍称：“石敢当在民间传说里面不是太完整的神话故事和人物，不像《西游记》、《封神榜》，所以我们在创作的过程中，实际上是把石敢当放到了上古时代的神话体系里面，有大量的情节和人物，基于这个基础来创作的，看这部剧实际上是一个了解中国古代神话很好的平台和机会。”<ref>{{cite news|url=http://www.chinadaily.com.cn/hqcj/xfly/2015-02-15/content_13243581.html|title=《石敢当之雄峙天东》开播 打造最强神话巨制|date=2015-02-15|publisher=中国日报网|accessdate=2015-02-21}}</ref>

2014年9月13日在泰山玉皇顶开机<ref>{{cite news|url=http://sd.china.com/pinpailm/shipin/11164256/20140915/18784610.html?bsh_bid=494588711|title=古装神话剧《石敢当之雄峙天东》泰山开机|date=2014-09-15|publisher=中华泰山网|accessdate=2015-02-21}}</ref>，12月杀青并取得国家广电总局电视剧发行许可证。2015年2月15日在泰山岱庙举行“石敢当 保安康”为全球华人祈福活动暨《石敢当之雄峙天东》开播仪式<ref>{{cite news|url=http://ent.qq.com/a/20150216/045343.htm|title=《石敢当》开播 众“神仙”岱庙雨中为华人祈福|date=2015-02-16|publisher=腾讯|accessdate=2015-02-21}}</ref>。

饰演[[孙悟空|孙悟空]]电视剧形象而知名的演员[[六小龄童|六小龄童]]在本剧中出演[[玉皇大帝|玉皇大帝]]。

钱雁秋在本剧中不仅自编自导，也扮演“[[二郎神|二郎神]]”杨戬。

== 相关 ==
山东卫视为配合本剧播出，特邀请张子健和六小龄童录制了20集宣传片《玉帝妙答石敢当》，每晚电视剧播完两集后播出。主要内容为二人以各种身份介绍拍摄花絮和相关的民间神话传说。

在该剧中塑造了一个名为“[[李衙内|李衙内]]”的角色（在该剧第3－5集中出场），倚仗着其父[[李刚|李刚]]身为[[开封府|开封府]]的[[捕快班头|捕快班头]]在[[开平镇|开平镇]]横行霸道，为非作歹，借此讽刺曾在2010年轰动一时的[[河北大学交通肇事案|河北大学交通肇事案]]（俗称“[[我爸是李刚|我爸是李刚]]”事件）中的肇事者[[李启铭|李启铭]]（又名'''李一帆'''）及其时任[[保定市|保定市]][[北市区|北市区]]公安局副局长的父亲李刚<ref>{{cite web | publisher = 凤凰网 | title = 石敢当之雄峙天东5集6集预告 石敢当玉莲情愫暗生? | url = http://fj.ifeng.com/ent/detail_2015_02/17/3574612_0.shtml | date = 2015年2月17日 | accessdate = 2015年2月17日 | deadurl = yes | archiveurl = https://web.archive.org/web/20150402140902/http://fj.ifeng.com/ent/detail_2015_02/17/3574612_0.shtml | archivedate = 2015年4月2日 }}</ref>。

== 评价 ==
片方自诩本剧亮点是特效制作，国内首部裸眼3D神话力作。但评论仍对此质疑，“斥资千万的特效已然花费在了妖精的造型和变身身上，凡间荣华富贵的帝王殿寒碜得不如一个大户人家”，而台词和逻辑也被指“完全跟不上进度”<ref>{{cite news|url=http://ent.sina.com.cn/r/i/2015-02-27/doc-iavxeafs1353043.shtml|title=《石敢当》成石敢duang 奈何智商跟不上|date=2015-02-27|publisher=新浪|accessdate=2015-02-28}}</ref>。
