'''石谷站'''（{{lang-en|Rockdale station}}）是一個[[城市鐵路|城市鐵路]][[東郊及伊拉瓦拉線|東郊及伊拉瓦拉線]]的車站，位於[[悉尼|悉尼]]南部的[[石谷_(悉尼)|石谷]]，這裏附近亦是[[石谷地方政府區域|石谷地方政府區域]]的總部所在地。

== 歷史==

車站跟隨著路線在1884年10月開幕。

== 月台服務 ==

每小時會有4班列車，繁忙時間會加設班次。由於往邦迪聯運的列車會由2號或4號月台開出，因此乘客需在大堂的電子顯示器上留意下一班列車會在那一個月台上開出；同一樣地，往瀑布/克羅努拉的列車會由3號或5號月台開出，亦需要在電子顯示器上留意下一班列車會在那一個月台上開出。在2006年的時間表，[[南海岸線|<font color=#0072c6>'''南海岸線'''</font>]]列車會在繁忙時間進入此站。<ref>{{cite web en|work=City Rail|title=2006 CityRail timetable － South Coast Line|url=http://www.cityrail.info/news/timetable2006_2.jsp?line=sc|accessdate=16 April 2009|deadurl=yes|archiveurl=https://web.archive.org/web/20080403142053/http://www.cityrail.info/news/timetable2006_2.jsp?line=sc|archivedate=2008年4月3日}}</ref>
