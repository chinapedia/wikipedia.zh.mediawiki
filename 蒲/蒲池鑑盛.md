'''蒲池鑑盛'''（1520年—1578年12月7日）是[[日本戰國時代|日本戰國時代]]至[[安土桃山時代|安土桃山時代]]的武將。[[筑後國|筑後國]]南部的[[戰國大名|戰國大名]]。{{tsl|ja|蒲池氏}}第16代、後蒲池第7代[[當主|當主]]。與父親[[蒲池鑑久|蒲池鑑久]]同樣，受[[大友義鑑|大友義鑑]]下賜[[偏諱|偏諱]]（「鑑」字）。

== 生平 ==
在[[永正|永正]]17年（1520年）出生。筑後[[守護|守護]][[大友氏|大友氏]]配下，持有城池並有大名分的在地領主，擅長兵法，在[[筑前國|筑前國]]、[[肥前國|肥前國]]等地轉戰。

根據『[[西國盛衰記|西國盛衰記]]』記載，父親鑑久因為對[[征夷大將軍|將軍]]供奉有所怠慢而被討伐；[[天文_(後奈良天皇)|天文]]12年（1543年），在鑑久死後，繼承下蒲池蒲池氏的[[家督|家督]]。雖然父親因大友氏而死，不過並無怨恨主君，並繼續向大友義鑑、[[大友義鎮|義鎮（宗麟）]]兩代臣從。

在[[龍造寺家兼|龍造寺家兼]]和其曾孫[[龍造寺隆信|隆信]]因為家臣團的反亂而被流放時，給予保護，並協助他們復歸。天文14年（1545年），在家兼的曾孫隆信被驅離肥前時，令隆信居於家兼曾居住過的蒲池領內[[三瀦郡|三瀦郡]]一木村（現今[[大川市|大川市]]），並賜予3百石扶持，令家臣[[原野恵俊|原野恵俊]]照顧。在隆信返回佐嘉時，令蒲池氏3百精兵護送隆信，結果隆信成功狩回本據地[[佐嘉城|佐嘉城]]。

在[[毛利元就|毛利元就]]取代[[大內氏|大內氏]]，制壓[[中國地方|中國地方]]並開始向[[九州|九州]]侵攻時，與[[大友宗麟|大友宗麟]]在{{tsl|ja|門司城}}戰鬥（[[門司城之戰|門司城之戰]]），於此戰中出陣。[[永祿|永祿]]10年（1567年），在討伐得到元就支援而背叛大友氏的[[高橋鑑種|高橋鑑種]]時，亦有出陣。[[元龜|元龜]]元年（1570年），在與龍造寺隆信戰鬥的[[今山之戰|今山之戰]]中，與[[田尻親種|田尻親種]]一同受宗麟之命，乘上數十隻兵船渡過[[筑後川|筑後川]]，並包圍龍造寺氏的{{tsl|ja|村中城}}。在數度合戰中，得到宗麟的感狀。

[[天正_(日本)|天正]]6年（1578年），在[[薩摩國|薩摩國]]的[[島津氏|島津氏]]與大友氏在[[日向國|日向國]]戰鬥（[[耳川之戰|耳川之戰]]）時，作為大友軍的一翼參戰。雖然生病，亦與嫡子[[蒲池鎮漣|鎮漣]]、三男[[蒲池統安|統安]]一同率領3千士兵出陣（『[[北肥戰史|北肥戰史]]』）。不過已經繼承家督的鎮漣打算背離大友氏，於是以生病為口實，率領2千士兵返回柳川。而自身則留在大友軍中，在開戰後，大友軍崩潰，雖然率領約1千直屬士兵試圖攻入島津氏本營等不斷奮戰，不過亦無法動搖大勢，最終與統安一同戰死（『[[大友記|大友記]]』）。

[[菩提寺|菩提寺]]是{{tsl|ja|崇久寺}}。法名'''松梅院殿長國覺久居士神儀'''。

== 子孫 ==
庶長子[[蒲池鎮久|鎮久]]的兒子[[蒲池貞久|貞久]]居住在[[佐賀藩|佐賀藩]]的諫早，並生下蒲池姓、宇都宮姓的子孫。次子[[蒲池鎮漣|鎮漣]]（嫡子）的嫡子宗虎丸（[[蒲池久鎮|久鎮]]）的名跡由[[蒲池鑑續|蒲池鑑續]]繼承，筑後一族由蒲池姓、首藤姓、宇都宮姓等子孫存續，女兒[[蒲池德子|德子]]與[[大友氏|大友氏]]重臣[[朽網宗曆|朽網宗曆]]的兒子[[朽網鑑房|鑑房]]產下兒子，並再興蒲池名跡，其子孫有成為[[德川幕府|德川幕府]]最後[[西國郡代|西國郡代]]的[[旗本|旗本]][[窪田鎮勝|窪田鎮勝]]（蒲池鎮克），一族有蒲池姓、朽網姓、江口姓、鶴姓、窪田姓的子孫。次男宮童丸（[[蒲池經信|經信]]）以大分[[大庄屋|大庄屋]]的身份，生下子孫。三男[[蒲池統安|統安]]的兒子[[應譽|應譽]]成為瀨高{{tsl|ja|来迎寺|來迎寺}}的[[僧侶|僧侶]]，為了憑弔[[柳川藩|柳川藩]]祖[[立花宗茂|立花宗茂]]的正室[[立花誾千代|誾千代]]，在[[元和_(後水尾天皇)|元和]]7年（1621年）建立[[柳川市|柳川市]]的{{tsl|ja|良清寺}}，其子孫再興蒲池的名跡，於[[江戶時代|江戶時代]]成為柳川藩主立花氏的[[家老|家老]]。另外，歌手[[松田聖子|松田聖子]]（蒲池法子）的生家是良清寺蒲池家的子孫，松田聖子是[[幕末|幕末]]柳川藩的[[蒲池鎮之|蒲池鎮之]]的玄孫。四男[[蒲池統康|統康]]（統春）在龍造寺氏進攻柳川（[[柳川之戰|柳川之戰]]）時戰死。

== 人物、逸話 ==
*如果沒有保護[[龍造寺家兼|龍造寺家兼]]和[[龍造寺隆信|隆信]]兩代，[[龍造寺氏|龍造寺氏]]可能會被消滅，亦不會得到後來的興隆。在『[[肥陽軍記|肥陽軍記]]』中記載「這裡有名為筑後國蒲池鑑盛的人，是於下筑後振威、武勇有名、長於[[和歌|和歌]][[管弦|管弦]]、深情的人」（{{lang|ja|ここに筑後国蒲池鑑盛と云う人は下筑後にて威をふるい武勇のほまれ有り和歌管弦にも長じた情ふかい人なり}}），被評為「義心如鐵」（{{lang|ja|義心は鉄のごとし}}）。

*在[[筑後國|筑後國]]南部領有1萬2千町（約12萬石）的{{tsl|ja|大身}}（『[[大友幕下筑後領主附|大友幕下筑後領主附]]』）。以[[筑後十五城|筑後十五城]]第一大名的身份，統轄筑後。改築{{tsl|ja|柳川城}}，並取代原來的居城{{tsl|ja|蒲池城}}，開掘出現今柳川的觀光名所「水鄉」，柳川城被戲稱為「柳川三年肥後三月肥前筑前朝飯前」，成為[[九州|九州]]第一、難攻不落的堅城，並令其成為{{tsl|ja|蒲池氏}}的本城。另外，亦建築{{tsl|ja|風浪宮}}的本殿，並建立[[城下町|城下町]]，成為現今[[柳川市|柳川市]]的雛型。

*在[[耳川之戰|耳川之戰]]中戰死的事，被『[[筑後國史|筑後國史]]』評為「可比[[湊川之戰|湊川之戰]]中[[楠木正成|楠木正成]]狀烈的戰死」（{{lang|ja|湊川の戦いにおける楠木正成の壮烈な最期にも比せられる}}）。
