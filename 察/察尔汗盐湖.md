'''察尔汗盐湖'''，是[[中国|中国]][[青海省|青海省]]西部的一个[[盐湖|盐湖]]，位于[[柴达木盆地|柴达木盆地]]南部，地跨[[格尔木市|格尔木市]]和[[都兰县|都兰县]]，由[[达布逊湖|达布逊湖]]以及南霍布逊、北霍布逊、涩聂等盐池汇聚而成，总面积5856平方公里，是中国最大的盐湖；[[格尔木河|格尔木河]]、[[柴达木河|柴达木河]]等多条内流河注入该湖。

==地理==
由于水分不断蒸发，盐湖上形成坚硬的盐盖，[[青藏铁路|青藏铁路]]和[[青藏公路|青藏公路]]直接修建于盐盖之上，察尔汗盐湖蕴藏有丰富的[[氯化钠|氯化钠]]、[[氯化钾|氯化钾]]、[[氯化镁|氯化镁]]等[[无机盐|无机盐]]，总储量达20多亿吨，为中国矿业基地之一。察尔汗盐湖是一个[[海成湖|海成湖]]，在[[青藏高原|青藏高原]]形成时留下的古[[特提斯海|特提斯海]]海水积累形成。现在，察尔汗盐湖已经分裂为达布逊湖、南霍布逊盐池、北霍布逊盐池、涩聂盐池等[[盐湖|盐湖]]和[[盐池|盐池]]。<ref>{{Cite web|url =http://www.xiyoutrip.com/spots/show_532.html|title =察尔汗盐湖介绍|publisher =西游旅游网|accessdate =2014-07-13|deadurl =yes|archiveurl =https://web.archive.org/web/20140714153459/http://www.xiyoutrip.com/spots/show_532.html|archivedate =2014-07-14}}</ref>察尔汗盐湖位于[[昆仑山|昆仑山]]和[[祁连山|祁连山]]之间的柴达木盆地中心，并被称作是“青藏高原的聚宝盆”。它主要著名于丰富的盐矿资源和长达32公里的“万丈盐桥”景色。<ref>{{Cite web |url = http://www.guolv.com/qinghai/jingdian/139217.html |title = 察尔汗盐湖 - 青海景点 |date = 2012-12-09 |publisher = 北京神舟国旅 |accessdate = 2014-07-13}}</ref>
