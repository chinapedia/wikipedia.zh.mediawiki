'''察哈尔民众抗日同盟军'''，又称'''察绥抗日同盟军'''。是指1933年由[[冯玉祥|冯玉祥]]、[[吉鸿昌|吉鸿昌]]、[[方振武|方振武]]等[[中国|中国]]将领在[[察哈爾省_(中華民國)|察哈尔省]]组织的一支抗[[日本|日]]部队。
==历史==
1932年8月[[宋哲元|宋哲元]]被南京国民政府任命为[[察哈尔省|察哈尔省]]主席。1932年10月，[[冯玉祥|冯玉祥]]由[[泰山|泰山]]到[[张家口|张家口]]。当时[[察哈尔省|察哈尔省]]主席[[宋哲元|宋哲元]]是冯玉祥的老部下。中共[[北方政治保卫局|北方政治保卫局]]接受冯玉祥的邀请，派[[張慕陶_(陝西)|张慕陶]]、[[吴化之|吴化之]]、[[吴止戈|吴止戈]]等到张家口，组成“张垣特委"，协助冯玉祥共同发动抗日工作。1933年1月17日，[[中华苏维埃共和国|中华苏维埃共和国]]、[[中革军委|中革军委]]、[[中国工农红军|中国工农红军]]总司令部发表宣言，宣告在停止进攻革命根据地、保障人民的自由权利、武装人民的前提下，愿和全国任何系统的军队订立抗日作战协定，共同反对日本侵略者。

1933年1月3日，日军侵占山海关，并占据[[热河省|热河省]]，[[長城戰役|进攻长城各口]]，同时派遣[[伪军|伪军]][[刘桂堂|刘桂堂]]7000多人、[[張海鵬_(軍事將領)|张海鹏]]1800人、[[李守信|李守信]]部6500人共约1万6千多人进占察哈尔北部地区。<ref>《李守信自述》，内蒙古自治区文史资料研究委员会出版，1985年，第126页</ref>[[张垣|张垣]]告急。[[蒋介石|蒋介石]]坚持“[[攘外必先安内|攘外必先安内]]”政策，在华北采取[[不抵抗主义|不抵抗主义]]，派[[何应钦|何应钦]]向侵入华北的日军谋求妥协；1933年3月，军政部长[[何应钦|何应钦]]代理[[国民政府军事委员会|军委会]]北平分会委员长。仅[[宋哲元|宋哲元]]部与日军作战。1933年4月，蒋介石在南昌发表谈话：抗日必先剿共，如共未剿完，谁要抗日，应从严治罪。1933年5月，日军越过占领的冀东、平北长城各口，进至[[三河|三河]]、[[宝坻|宝坻]]、[[顺义|顺义]]线，平津危急。中共广泛发动平津的青年学生、晋冀鲁豫的青年和一些主张抗日的部队到张家口参加察哈尔抗日。冯玉祥的旧部[[吉鸿昌|吉鸿昌]]、[[方振武|方振武]]、[[高树勋|高树勋]]、[[孙良诚|孙良诚]]、[[张凌云|张凌云]]、[[乜玉岭|乜玉岭]]等相继赴张家口参加抗战。从热河退到察哈尔的[[东北义勇军|东北义勇军]]的[[冯占海|冯占海]]、[[李忠义|李忠义]]、[[邓文|邓文]]、[[黄守中|黄守中]]、[[姚景川|姚景川]]、[[富春|富春]]、[[战涛|战涛]]等部也集结在冯玉祥的领导下继续抗战。1933年5月，南京国民政府设立行政院驻北平政务整理委员会，委员长[[黄郛|黄郛]]。

1933年5月26日，冯玉祥在[[张家口|张家口]]正式成立以冯玉祥原部下和[[中国共产党|共产党]]支持者为主组成'''民众抗日同盟军'''，共有10万人，公推冯玉祥为抗日同盟军总司令。在军民代表大会上，明确提出了同盟军的政治主张：
  外抗暴力，内除汉奸卖国贼，武力收复失地，反对妥协投降，取消苛捐杂税，改善工农士兵生活，释放爱国政治犯，保障人民言论、结社、出版、集会和武装自由，以及实现民主政权，联合世界反帝力量共同奋斗。
开设《抗日战线》日报。冯玉祥任命警务处长[[佟麟阁|佟麟阁]]为察哈尔省暂代主席，吉鸿昌代理警务处长兼张垣警备司令，[[张允荣|张允荣]]代理财政厅长。成立了抗日同盟军军事委员会，愿意抗战的部队都给以名义，共有7个军，9个独立师，2个独立旅，军队共12万人，枪支10万支：
* 宋哲元部调往冀东参加长城抗战，留下的少量部队，负责人为察哈尔省警备司令[[佟麟阁|佟麟阁]]，编为第一军军长兼察哈尔省主席，
** [[冯庸|冯庸]]（原热河义勇军）为第一军独立旅长
* [[吉鸿昌|吉鸿昌]]为第二军军长、警备司令兼警察处长，
* 1933年5月冯占海为第四路军总指挥。冯部2万人四路纵队行进整齐，有较完整的炮兵营、骑兵营、工兵营和辎重营。冯部义勇军当时已被改编为国民革命军第六十三军，受何应钦的胁迫，6月5日即奉命南下，脱离了同盟军。
* 邓文为第五路总指挥
* [[阮玄武|阮玄武]]为第五军军长，
* [[张凌云|张凌云]]为第六军军长
** 乜玉岭为第六军游击师师长。
* [[李忠义|李忠义]]（李海青）为十六军军长，
* [[黄守中|黄守中]]热河抗日救国军改编为十八军任军长
* 马冠军为第二十五军军长
* 耿继周为第三十五军军长
* [[方振武|方振武]]为抗日救国军总司令，从山西拉来旧部一个团
* 乜庭宾为察哈尔抗日同盟军独立第四旅旅长
* 山西[[汾阳军校|汾阳军校]]调来的军官和学生编为第二师。汾阳军校的经费是宋哲元、张自忠接济的。从建校开始,宋哲元就将汾阳军校编为他领导的二十九军的第二教导团,按月提供资金。支应遴为师长，政治部主任王沛南。下辖三个团。第一团由尹心田任团长,担任张家口城防警卫;第二团团长贾振中;第三团团长周茂兰。二、三团直接参加了收复康堡、宝康等失地的战斗。
* 以汾阳军校调去一部分人作为骨干，以平津学生和各地来张的青年编成第五师，以中共党员[[宣侠父|宣侠父]]为师长；政治部副主任[[张曙时|张曙时]]。
* 从汾阳军校和各部队调集骨干组成十八师，以中共党员[[许权中|许权中]]为师长。政治部主任[[刘少文|刘少文]]。 
* 同盟军干部学校校长[[张克侠|张克侠]]。
二师、五师、十八师是中共特委直接领导的核心力量，二师政治质量好，武器也比较好，有二千多人。五师、十八师是新编成的，每师有七八百人，骨干强，但武器庞杂陈旧，但士气高昂。这三个师的师长和团长除个别人外都是共产党员，营连干部也大部分是中共党员。为加强抗日同盟军的领导，中共河北省委（顺直省委）撤销“特委”，改为前敌委员会(简称前委)，[[柯庆施|柯庆施]]任前委书记，前委委员[[刘少文|刘少文]]。6月中旬，河北省委就传达中央指示，要前委进行反日、反蒋、反冯玉祥的群众运动。

1933年5月31日，何应钦与日本签订了《[[塘沽协定|塘沽协定]]》。

1933年6月21日起派方振武、吉鸿昌、邓文、李忠义率部向日伪军发动进攻。6月22日，[[康保|康保]]的李守信伪军大部投降，小部溃逃走。6月23日，[[沽源|沽源]]的刘桂堂部被合围，刘桂堂害怕被全歼，率部投降抗日同盟军。7月1日，猛攻[[宝昌|宝昌]]，李守信部和张海鹏部不敌，3000多人被歼灭或者打散，逃到[[多伦|多伦]]。此时多伦伪军大约有5000多人。7月4日，吉鸿昌指挥同盟军攻打多伦，日军顾问不允许李守信弃城逃跑，由于当时暴雨连连，加上多伦周围有一些碉堡，经过三昼夜激战，李守信部下伤亡一千多人，士兵纷纷准备逃散或者投降，日本顾问此时也怕李守信本人突然倒戈，就向李守信表示可以立即突围。李守信即和吉鸿昌的代表联系，希望放出一角让他们走。当时抗日同盟军已经攻城3天，伤亡了1600多人，补给也出现问题，为了防止李守信余部3000多人顽抗，同意李撤走。<ref>《李守信自述》，内蒙古自治区文史资料研究委员会出版，1985年，第131-135页</ref>收复多伦以后，冯玉祥在大会上说：抗日同盟军收复察东四县，有力地戳穿了不抵抗主义所宣称的，日军不可战胜的谎言。实际上并未与日军接战。国民政府行政院院长[[汪精卫|汪精卫]]于1933年7月18日在庐山发表谈话，称抗日同盟军收复多伦，不是取自日本军队之手，而是取自伪军之手，此等傀儡，何足一击！并且在冯玉祥发出的捷报上批注了两个字“无耻”。同时指责冯玉祥和中共勾结，试图赤化察哈尔搞独立。

同时，因为当时全面性抗战尚未爆发，[[蒋介石|蒋介石]]认为该军妨害统一政令、破坏国策，更因为同盟军有中共背景，遂令[[何应钦|何应钦]]指挥16个师沿[[平绥线|平绥线]]进攻张家口，并用金钱收买同盟军的动摇分子。1933年7月31日邓文被刺杀。方振武从山西拉来的抗日救国军的张人杰的三个师又二个旅和热河军的五个师随即也投靠中央。而中共方面一开始就有利用同盟军消耗国民政府军的企图<ref>《关于抗日同盟军纲领等问题的报告》，1933年3月10日，转引自杨奎松《中间地带的革命》，山西人民出版社，2010年，第287页</ref>，另一方面苏联也不信任、不支持冯玉祥，反而认为冯玉祥是故意要挑拨苏日冲突<ref>哈马丹：《中国军阀屈膝投降》，《真理报》1933年6月4日，《1927—1937苏联〈真理报〉有关中国革命的文献资料选编》，第407—410页</ref>。苏联拒绝通过[[外蒙古|外蒙古]]通过察哈尔给冯玉祥提供军火物资援助，要求冯部把部队交给[[国民政府|国民政府]]中央统一抗日。日伪军转为反攻察东。

1933年7月28日，汪精卫、蒋中正联名发表时局通电，向冯玉祥提出四点最后通谍
# 清除匪祸，充实国力。
# 不签订割让或承认条约，逾此限度，不惜牺牲。
# 需求国际同情与协助，但不纵横捭阖。
# 察事盼冯玉祥接受四项原则，勿擅立军政名义，勿妨害中央边防计划，勿滥收散军土匪，勿引用共匪头目。
何应钦增调[[王以哲|王以哲]]、[[黄杰|黄杰]]、[[王敬之|王敬之]]等5个师，加上原有的[[庞炳勋|庞炳勋]]、[[关麟征|关麟征]]、[[冯钦哉|冯钦哉]]的11个师，共16个师15万人进攻张家口。

在日蒋夹击和同盟军内部混乱的情况下，冯玉祥被迫于8月2日发表声明，表示不愿意内战，希望宋哲元回察主持一切，冯本人进退不成问题，向中央政府示好。8月3日，冯玉祥宣布取消他同盟军司令的头衔，把察哈尔省府张家口让给佟麟阁，迎接宋哲元回察哈尔收复权力。8月5日通电全国解散抗日同盟军，政权交给二十九军军长兼察省主席宋哲元接收，8月14日冯玉祥回泰山隐居。

中央随即宣布了抗日同盟军编遣方案：
# 委[[阮玄武|阮玄武]]为商都警备司令。阮各部共编为2个团，富春部编1个团，归其指挥；
# 委[[张凌云|张凌云]]为康保、宝昌警备司令，[[乜玉岭|乜玉岭]]为副司令，所部编为2个团；
# 委[[黄守中|黄守中]]为察省游击支队，所部编为2个大队；
# 委[[刘桂堂|刘桂堂]]为察东游击司令，所部编为3个团；
# [[孙良诚|孙良诚]]部编为1个团，暂归[[张允荣|张允荣]]指挥，孙即日通电宋哲元，自解兵权离察；
# [[刘震东|刘震东]]部编为1个团；
# [[唐聚五|唐聚五]]部编为1个团；
# [[张砺生|张砺生]]由省府聘为顾问，所部解散归还地方；
# 檀自新、张人杰、李忠义各部均照军分会规定编制；
# 姚景川、苏雨生、宋克宾、李子铎各师拟分别编遣；
# 清乡司令韩青芳部已下令解散；
# 冯玉祥部军官队4个大队及佟麟阁部独立旅约500余人均已遣散。

另拟方振武令其出洋，但他不愿离开军队。8月12日，宋哲元进驻张家口。8月14日，冯玉祥离开张家口。8月16日，方振武宣布接任察哈尔抗日同盟军总司令。李守信部3000伪军在7日已经逼近多伦，吉鸿昌部15日连夜不战放弃多伦。剩下的五万抗日同盟军也彻底瓦解，主力全部为宋哲元、商震收编。抗日同盟军只剩下吉鸿昌和方振武的二支武装，大约1万5000人。

8月24日，中共前委领导的二师、五师、十八师在张北县开了反妥协大会，坚持继续抗日反蒋。正式公开共产党在抗日同盟军中的组织身份，并成立革命军事委员会。常委由柯庆施、吉鸿昌、张幕陶、宣侠父、许权中等7人组成，柯庆施任书记，吉鸿昌任总指挥。同时决定部队西开商都，然后出呼和浩特，再南下蔚县，实现中央意图--在河北省中南部地区建立新苏区。当时不妥协的还有方振武的一部(方的主力鲍刚、张人杰都先后投蒋)，其他部队都不参加了。方振武受刘桂堂、汤玉麟等人的影响，直接打到北平去，讨伐蒋介石。

此时主力还有二师、五师、十八师、十六师。前委执行河北省委“开展工农群众的反日运动，配合河北省各地工农群众的革命斗争，创造北方的新苏区和红军”的指示。前委把十六师编给五军，结果被缴械解散。二师师长[[支应遴|支应遴]]是西北军的老人，跟冯抗日可以，跟共产党干红军就不可能了，结果二师被宋哲元改编。1933年8月26日，吉鸿昌和方振武分道扬镳. 吉鸿昌率领3000多人试图去商都同抗日同盟军高树勋会合。但是高树勋已经归附中央，并且连同晋军[[赵承绶|赵承绶]]出兵拦截，吉鸿昌先锋部队300人在战斗中投靠中央。吉鸿昌无奈，只得转进[[独石口镇|独石口]]与[[方振武|方振武]]会合，部队又折向东进。到独石口后，8月27日前委就召开了党的会议。宣布开除[[張慕陶_(陝西)|张慕陶]]的党籍，理由是张右倾，影响了二师未执行前委的指示来独石口会师，被宋哲元改编了。一起被开除党籍的还有二师的党委书记王霖，一团团长尹心田、二团团长贾振中，三团团长周茂兰，共五人，但当时只宣布了张慕陶一人。在独石口住了几天，中共前委领导十八师为一路，吉鸿昌的五师与教导队为一路，方振武部为一路，3路分别东进。部队经过热察边境的[[黑河|黑河]]（[[赤城县|赤城县]]东部的[[潮白河|潮白河]]支流）二、四区东进，“抗日讨贼”，南下攻打由何应钦所率中央军重兵把守的北平，“打到北平过中秋”。沿途村庄稀少，部队吃粮困难，有时只好在地里挖一些白薯和山药蛋充饥。食盐更缺，前边的部队把粮食都吃光了，后面的部队就杀马吃。1933年9月10日，吉鸿昌，方振武，刘桂堂正式把抗日同盟军改变为“抗日讨蒋军”，要求攻下北平，讨伐蒋贼。平北战役从9月下旬(中秋节前夕)到10月15日，共两个多星期。9月23日到怀柔。9月26日日机4架轰炸。9月27日何应钦派万福麟、商震部堵击。五师、十八师加上方振武、吉鸿昌掌握的部队共计六、七千人，处于何应钦的十几万部队的层层包围之中，东北义勇军总指挥[[战涛|战涛]]在作战中被商震第32军机枪扫射中弹牺牲。中共党员[[武止戈|武止戈]]为吉鸿昌部参谋主任，在北平[[顺义县|顺义县]]被日军飞机炸死。吉鸿昌指挥骑兵、炮兵、步兵联合作战夺取了一三〇师[[朱鸿勋|朱鸿勋]]部据守的[[小汤山|小汤山]]。刘桂堂和汤玉麟都宣布不和吉鸿昌合作。10月15日下午三点多钟，获悉、方振武、吉鸿昌都已离开部队了(10月2日柯庆施离开部队)，北平商会会长冷家骥，奉蒋介石、何应钦之命通知剩下一千来人立即开到[[清河街道_(北京市)|清河]]改编成独立旅，由吉鸿昌的参谋长[[崔贡琛|崔贡琛]]当旅长。当天晚上何应钦背信把这支抗日同盟军缴械编散。 

1933年底方振武流亡香港。1934年春，逃亡并隐藏在天津租界的[[中共党员|中共党员]]吉鸿昌，策动参加对中央苏区[[第五次圍剿戰爭|第五次圍剿戰爭]]的旧部两个师反蒋，并策划在老家河南发起反蒋暴动，因而于11月9日在天津被[[租界|租界]]当局逮捕<ref>叶介甫《抗日将领吉鸿昌被枪杀内幕》，《文史春秋》2013年10月</ref>，引渡给北平军分会。后在[[北京|北平]]被國民政府軍事法院“违反国策”罪判处死刑执行枪决。
