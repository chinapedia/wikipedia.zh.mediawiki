'''薛贻矩'''（{{bd|850年||912年|5月25日|catIdx=X}}<ref name="墓志铭">《大梁故[[开府仪同三司|开府仪同三司]]守司空同中书门下平章事弘文馆大学士充诸道盐铁转运等使判建昌宫事河东郡开国公食邑一千五百户食实封一百户赠侍中薛公墓志铭并序》</ref><ref name=ZZTJ268>[[:s:資治通鑑/卷268|《资治通鉴》卷二百六十八]]</ref>），[[表字|字]]'''熙用'''，一字'''式瞻'''，[[河東郡_(中國)|河东]][[闻喜县|闻喜]]（今山西省[[闻喜县|闻喜县]]）人，<ref name=HFD18>[[:s:舊五代史/卷18|《旧五代史》卷一十八]]</ref>[[中国|中国]][[唐朝|唐朝]]及随后的[[后梁|后梁]][[官员|官员]]，在后梁为[[宰相|宰相]]。

== 家世 ==
薛贻矩出自[[河东薛氏|河东薛氏]]西祖第三房，祖父薛存诚官至[[给事中|给事中]]，父薛庭望官至[[虢州|虢州]][[刺史|刺史]]，<ref name=NBT73>《新唐书》卷七十三{{cite web |url=http://www.sidneyluo.net/a/a17/073.htm |title=存档副本 |accessdate=2009-04-08 |0=2009-02-02 |deadurl=yes |archiveurl=https://web.archive.org/web/20090202171923/http://sidneyluo.net/a/a17/073.htm |archivedate=2009-02-02 }}{{cite web |url=http://www.sidneyluo.net/a/a17/table/form75.htm |title=存档副本 |accessdate=2012-02-20 |deadurl=yes |archiveurl=https://web.archive.org/web/20120505165647/http://www.sidneyluo.net/a/a17/table/form75.htm |archivedate=2012-05-05 }}</ref>都有好名声。<ref name=HFD18/>

== 唐朝年间 ==
薛贻矩秀气英俊，和当时聪明优雅的人交游，在文坛有名。[[唐僖宗|唐僖宗]][[乾符|乾符]]年间（874年—879年）中[[进士|进士]]，夺得[[状元|状元]]。<ref>《[[登科记考|登科记考]]》阙载乾符四年（877年）和六年（879年）状元，可见薛贻矩为乾符四年状元。</ref>历任秘书省校书、鄠县尉、<ref name="墓志铭"/>度支巡官、[[集贤院|集贤院]]校理、[[拾遗|拾遗]]、殿中侍御史、起居舍人，召拜[[翰林学士|翰林学士]]，加礼部员外郎、知制诰，转司勋郎中，其职如故。<ref name=HFD18/>又任司勋员外郎，任史职。<ref name="墓志铭"/>

[[乾宁|乾宁]]二年（895年），[[唐昭宗|唐昭宗]]害怕被[[军阀|军阀]][[凤翔军节度使|凤翔军节度使]][[李茂贞|李茂贞]]和[[静难军节度使|静难军节度使]][[王行瑜|王行瑜]]攻打，逃出京城[[长安|长安]]，<ref name=ZZTJ260>[[:s:資治通鑑/卷260|《资治通鉴》卷二百六十]]</ref>薛贻矩没有立即随驾，而是聚集家属，因而没能跟上昭宗，因此被罢官。不久除中书舍人，又为翰林学士，又历任户部侍郎、兵部侍郎、学士承旨。<ref name=HFD18/>[[天复_(唐朝)|天复]]三年（903年），昭宗在[[宰相|宰相]][[崔胤|崔胤]]和权势军阀[[宣武军节度使|宣武军节度使]][[朱全忠|朱全忠]]要求下屠戮[[宦官|宦官]]，<ref name=ZZTJ263>[[:s:資治通鑑/卷263|《资治通鉴》卷二百六十三]]</ref>命薛贻矩将他们的首级送给朱全忠。<ref>[[:s:旧唐书/卷184|《旧唐书》卷一百八十四]]</ref><ref>[[:s:新唐书/卷221|《新唐书》卷二百二十一]]</ref>薛贻矩被发现为[[韩全诲|韩全诲]]等被屠戮的宦官画像，<ref name=NHFD35>[[:s:新五代史/卷35|《新五代史》卷三十五]]</ref>于是被贬[[夔州|夔州]]司户。<ref name=HFD18/><ref>[[:s:旧唐书/卷177|《旧唐书》卷一百七十七]]</ref>

朱全忠迫使朝廷由长安迁都[[洛阳|洛阳]]。<ref name=ZZTJ264>[[:s:資治通鑑/卷264|《资治通鉴》卷二百六十四]]</ref>唐昭宗和[[唐哀帝|唐哀帝]][[天祐_(唐朝)|天祐]]（904年—907年）初年，薛贻矩被除吏部侍郎，但没去洛阳上任。朱全忠素来器重他，在朝堂提起他，于是薛贻矩当天被拜吏部尚书，还没到任，又特拜[[御史大夫|御史大夫]]。<ref name="墓志铭"/><ref name=HFD18/><ref name=NHFD35/>

天祐四年（907年）正月，显然朱全忠最终将夺帝位之际，哀帝派薛贻矩去朱全忠军部[[大梁|大梁]]相劳，薛贻矩请求以臣礼相见。朱全忠对他作揖、升阶。薛贻矩说：“殿下功德在人，天地人心都向着您，[[皇帝|皇帝]]正要行[[舜|舜]]、[[禹|禹]]之事（即[[禅位|禅位]]于朱全忠），臣怎敢违背？”于是以对待皇帝的礼仪，当庭对朱全忠北面拜舞。朱全忠侧身以避，但待之甚厚。薛贻矩回去后对哀帝说朱全忠已经准备[[受禅|受禅]]，于是哀帝准备禅位。三月，薛贻矩为押金宝使，宰相摄[[中书令|中书令]][[张文蔚|张文蔚]]为册礼使，摄[[侍中|侍中]][[杨涉|杨涉]]为押传国宝使，率百官备法驾前去大梁，朱全忠就地受禅，亡唐，建立后梁，朱全忠即梁太祖皇帝。<ref name=NHFD35/><ref>[[:s:旧唐书/卷020下|《旧唐书》卷二十下]]</ref><ref name=NHFD2>[[:s:新五代史/卷002|《新五代史》卷二]]</ref>太祖宴参与禅位典礼的百官于玄德殿，百官多惭惧俯伏不能对，但薛贻矩、礼部尚书册礼副使[[苏循|苏循]]及刑部尚书[[张祎|张祎]]却盛称太祖功德，理应应天顺人。<ref name=ZZTJ266>[[:s:資治通鑑/卷266|《资治通鉴》卷二百六十六]]</ref><ref name=HFD3>[[:s:舊五代史/卷03|《旧五代史》卷三]]</ref><ref name=HFD60>[[:s:舊五代史/卷60|《旧五代史》卷六十]]</ref>

== 后梁年间 ==
改朝换代不久，五月，薛贻矩即为[[中书侍郎|中书侍郎]]、[[同平章事|同平章事]]，为宰相，<ref name=ZZTJ266/><ref name=HFD3/>兼判户部。十月，梁太祖遣文武百官归位，但将薛贻矩等留在洛阳。<ref name=HFD3/>[[开平_(后梁)|开平]]二年（908年），进拜[[门下侍郎|门下侍郎]]、监修国史，判度支，又迁[[弘文馆|弘文馆]]大学士，充盐铁转运使，<ref name=HFD24>[[:s:舊五代史/卷24|《旧五代史》卷二十四]]</ref>累官自仆射至守[[司空|司空]]。三年（909年）九月，判建昌宫事，<ref>《册府元龟》误作[[开运_(后晋)|开运]]三年。</ref>兼延资库使。<ref name=HFD5>[[:s:舊五代史/卷05|《旧五代史》卷五]]</ref><ref name=HFD149>[[:s:舊五代史/卷149|《旧五代史》卷一百四十九]]</ref>六月，太祖以宰相[[韩建|韩建]]和薛贻矩辅政有功，深加奖谕，赐以缯帛。四年（910年）十二月，上《[[:s:上大梁新定格律奏|上大梁新定格律奏]]》：“太常卿[[李燕|李燕]]等重刊定律令三十卷，式二十卷，格一十卷，目录一十三卷，律疏三十卷，共五部一十帙，共一百三卷。敕中书舍人[[李仁俭|李仁俭]]诣阁门奉进，伏请定为《大梁新定格式律令》，颁下施行。”太祖从之。<ref name=HFD147>[[:s:舊五代史/卷147|《旧五代史》卷一百四十七]]</ref>拜相五年，却无显赫事迹可表。<ref name=HFD18/>[[乾化|乾化]]元年（911年），太祖召薛贻矩等宰相于崇勋殿，说：“军旅之间的事，朕自己决断，朝廷庶务，其实倚赖众卿协力辅佐。”<ref>《[[册府元龟|册府元龟]]》卷三一九<宰辅部></ref>二年（912年），薛贻矩在陪同太祖讨伐敌国[[晋_(五代)|晋国]]后，染时疠，奉命回洛阳治病，不久去世。<ref name=ZZTJ268/>太祖震悼良久，命雒苑使曹守珰前去吊祭，又命辍六日、七日、八日朝参，丞相、文武都诣上阁门进名以表奉慰。<ref name=HFD7>[[:s:舊五代史/卷07|《旧五代史》卷七]]</ref><ref>《[[册府元龟|册府元龟]]》</ref>诏赠侍中。<ref name=HFD18/><ref name=NHFD35/>
