'''“郑州”号导弹驱逐舰'''（[[舷号|舷号]]151）简称'''“郑州”舰'''，是[[中国人民解放军海军|中国人民解放军海军]]第四艘[[052C型导弹驱逐舰|052C型导弹驱逐舰]]，装备[[有源相控阵雷达|有源相控阵雷达]]及[[垂直发射系统|垂直发射系统]]，可单独或协同海军其他兵力攻击[[水面舰艇|水面舰艇]]、[[潜艇|潜艇]]，具有较强的远程警戒探测和区域防空作战能力。郑州舰于2008年12月16日由[[江南造船厂|江南造船厂]]开工建造，2010年6月28日上船台，2011年6月25日下水，2013年12月25日交付，2013年12月26日入列，现服役于[[东海舰队|东海舰队]][[驱逐舰第六支队|驱逐舰第六支队]]。郑州舰舰名取自[[河南省|河南省]][[省会|省会]][[郑州市|郑州市]]，其入列命名授旗仪式于2013年12月26日在[[浙江省|浙江省]][[舟山市|舟山市]]举行。<ref>{{cite web|url= http://news.mod.gov.cn/headlines/2013-12/26/content_4480876.htm|title=新型导弹驱逐舰“郑州舰”入列东海舰队|publisher=中华人民共和国国防部| date=2013-12-26 }}</ref>

== 历史 ==
2014年1月中旬，“郑州”号导弹驱逐舰展开年度第一次出海训练。<ref>{{cite web|url= http://www.mod.gov.cn/hdpic/2014-01/21/content_4486272.htm|title=海军郑州舰：检讨式训练助力战斗力向打赢聚焦|publisher=中华人民共和国国防部| date=2014-01-21 }}</ref>

2014年5月18日，作为第一水面舰艇编队指挥舰的“郑州”号导弹驱逐舰与[[哈尔滨号导弹驱逐舰|“哈尔滨”号导弹驱逐舰]]、[[烟台号导弹护卫舰|“烟台”号导弹护卫舰]]及[[柳州号导弹护卫舰|“柳州”号导弹护卫舰]]等中方舰艇与参加[[海上联合-2014中俄海上联合军事演习|“海上联合-2014”中俄海上联合军事演习]]的[[瓦良格号巡洋舰_(1983年)|“瓦良格”号导弹巡洋舰]]、[[潘捷列耶夫海军上将号反潜舰|“潘捷列耶夫海军上将”号反潜舰]]及[[涅韦利斯基海军上将号登陆|“涅韦利斯基海军上将”号登陆舰]]等俄方舰艇在[[上海市|上海市]][[吴淞口|吴淞口]]会合。<ref>{{cite web|url= http://mil.cnr.cn/ztl/zejy/dt/201405/t20140519_515540321.html|title=中俄“海上联合—2014”参演军舰“舰谱” |publisher=中国广播网| date=2014-05-19 }}</ref>双方于5月20日至26日进行了舰艇锚地防御、联合对海突击及联合反潜等课目的演练。<ref>{{cite web|url= http://mil.cnr.cn/zgjd/201405/t20140513_515501444.html|title=中俄两国海军将举行海上联合军事演习|publisher=中国广播网| date=2014-05-13 }}</ref>

2014年7月11日，[[上海|上海]]举行“航海日”主题宣传纪念活动，“郑州”号导弹驱逐舰与[[东海救101轮|“东海救101”轮]]大型救助船舶靠泊[[上海港|上海港]]国际客运中心码头，向市民开放参观。<ref>{{cite web|url= http://mil.cnr.cn/zgjd/201405/t20140513_515501444.html|title=沪举行航海日主题活动 海军郑州舰二度向市民开放|publisher=新浪网| date=2014-07-11 }}</ref>

2014年9月14日，“郑州”号导弹驱逐舰与作战支援舰船、地方执法船、直升机、潜艇等组成的海上编队起程开赴东海某海域展开“海神-2014”海上作战支援实兵演练，共同演练支援掩护海上执法力量维权执法行动、军地联合搜救、开辟海上应急航道、援救失事潜艇等9个实战化科目。<ref>{{cite web|url= http://www.chinanews.com/mil/2014/09-14/6588680.shtml|title=东海举行“海神-2014”海上作战支援实兵演练 |publisher=中新网| date=2014-09-14 }}</ref>

2014年12月12日，“郑州”号导弹驱逐舰、[[泰州号导弹驱逐舰|“泰州”号导弹驱逐舰]]、[[舟山号导弹护卫舰|“舟山”号导弹护卫舰]]、[[益阳号导弹护卫舰|“益阳”号导弹护卫舰]]、[[千岛湖号综合补给舰|“千岛湖”号综合补给舰]]及[[北极星号电子侦察船|“北极星”号电子侦察船]]穿越[[宫古海峡|宫古海峡]]进入[[西太平洋|西太平洋]]。<ref>{{cite web|url=http://military.people.com.cn/n/2014/1214/c172467-26203247.html |title=中国海军舰队再次穿过宫古海峡 战机1周4次往返 |publisher=人民网| date=2014-12-14 }}</ref>

2016年12月25日，据日本[[统合幕僚监部|统合幕僚监部]]报道，正在进行远海训练的[[辽宁舰|辽宁舰]]航母编队由8艘军舰编成。除辽宁舰外，还包括[[052C型导弹驱逐舰|052C型导弹驱逐舰]]“[[鄭州號飛彈驅逐艦|郑州舰]]”（舷号151）、“[[海口號飛彈驅逐艦|海口舰]]”（舷号171），[[052D型导弹驱逐舰|052D型导弹驱逐舰]]“[[長沙號飛彈驅逐艦|长沙舰]]”（舷号173），[[054A型导弹护卫舰|054A型导弹护卫舰]]“[[烟台號飛彈護衛艦|烟台舰]]”（舷号538）、“[[臨沂號飛彈護衛艦|临沂舰]]”（舷号547），[[056A型导弹护卫舰|056A型导弹护卫舰]]“株洲舰”（舷号594），以及[[903型综合补给舰|903A型综合补给舰]]“高邮湖舰”（舷号966）<ref>{{Cite web |url = http://news.ifeng.com/a/20170107/50537597_0.shtml |title = 专家：中国航母发展后劲看好 有望拥有3个航母编队 |author =  |date = 2017-01-07 |publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://henan.people.cn/n2/2017/0207/c378402-29678517.html |title =  中国航母编队：牵动世界目光的航迹 |author = |date = 2017-02-07 |publisher = 人民网  |language =  |accessdate =  |quote =  }}</ref>。

== 事故 ==
2014年12月，在进入西太平洋之后的演习中，该舰舰尾一个系缆舱门被海浪冲开，舱内大量进水。舰上水兵处置过程中一人重伤。<ref>[http://news.163.com/15/0222/19/AJ35OC7N0001124J.html 郑州舰舱门被冲开 士兵关门瞬间被大浪打翻(图)]</ref>

官方稱該級艦没有设计问题。<ref>[http://mil.sohu.com/20150226/n409172929.shtml 郑州舰舱门被海浪冲开之谜揭开:没有设计问题(图)]</ref>
