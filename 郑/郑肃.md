'''郑肃'''（{{bd|8世纪||849年|catIdx=Z}}？），字'''乂敬'''，[[中国|中国]][[唐朝|唐朝]][[官员|官员]]，[[唐武宗|唐武宗]]和[[唐宣宗|唐宣宗]]年间为[[宰相|宰相]]。

== 家世和早期仕途 ==
郑肃生年不详，祖源不详，仅知家族出自[[荥阳|荥阳]]，其世系未被《新唐书·宰相世系表》与其他郑姓宰相一同列入。<ref>《新唐书》卷七十五</ref>祖郑烈，父郑阅，世代[[儒家|儒家]]。<ref name=NBT182>[[:s:新唐書/卷182|《新唐书》卷一百八十二]]</ref>郑肃苦心力学。[[元和_(唐朝)|元和]]三年，擢[[进士|进士]]第，又因书判成绩拔萃，历任节度使府僚佐。<ref name=BT176>[[:s:舊唐書/卷176|《旧唐书》卷一百七十六]]</ref>

== 唐文宗年间 ==
[[唐文宗|唐文宗]][[太和_(唐朝)|太和]]（827年—835年）初，入朝为尚书郎。六年（832年），转太常少卿。他能读古文，擅长经学，博士们对《[[左传|左传]]》、《三礼》（《[[周礼|周礼]]》《[[仪礼|仪礼]]》《[[礼记|礼记]]》）、仪注有疑议时，皆由郑肃决断。<ref name=BT176/>

当时文宗宠爱长子鲁王[[李永_(唐朝)|李永]]，择名儒为其府属官，用户部侍郎[[庾敬休|庾敬休]]兼鲁王傅，户部郎中[[李践方|李践方]]兼鲁王司马，以郑肃守太常卿本官兼鲁王长史。<ref name=BT175>[[:s:舊唐書/卷175|《旧唐书》卷一百七十五]]</ref><ref name=NBT82>[[:s:新唐書/卷082|《新唐书》卷八十二]]</ref>郑肃从此知名。次年李永被立为[[皇太子|皇太子]]，郑肃加给事中。八年（834年），因文宗近臣[[李仲言|李仲言]]有前科，他与同僚[[高铢|高铢]]、[[韩佽|韩佽]]、[[谏议大夫|谏议大夫]][[郭承嘏|郭承嘏]]、[[中书舍人|中书舍人]][[权璩|权璩]]等反对其被提拔为翰林侍讲学士，不果。<ref name=ZZTJ245>[[:s:資治通鑑/卷245|《资治通鉴》卷二百四十五]]</ref><ref name=BT174>[[:s:舊唐書/卷174|《旧唐书》卷一百七十四]]</ref><ref name=NBT165>[[:s:新唐書/卷165|《新唐书》卷一百六十五]]</ref><ref name=NBT179>[[:s:新唐書/卷179|《新唐书》卷一百七十九]]</ref>九年（835年），改刑部侍郎，不久改尚书右丞，权判吏部西铨事。<ref name=BT176/>

[[开成|开成]]元年（836年），出为陕虢都防御观察使、<ref name=BT17-2>[[:s:舊唐書/卷017下|《旧唐书》卷一十七下]]</ref>兼[[御史大夫|御史大夫]]。二年（837年）九月，召拜吏部侍郎。文宗因他曾事太子，言论典正，令兼太子宾客，为太子授经。后因文宗宠妃[[杨贤妃_(唐文宗)|杨贤妃]]弹劾，太子开始失宠，文宗考虑废太子。郑肃趁受召见之机，意图为太子辩护，深切陈说邦国大本、君臣父子之义，文宗感谢他。但在次年（838年）太子近乎被废并于十月死去后，<ref name=ZZTJ246>[[:s:資治通鑑/卷246|《资治通鉴》卷二百四十六]]</ref>四年（839年）闰正月，<ref name=BT17-2/>郑肃被遣出[[长安|长安]]，为检校礼部尚书，兼[[河中|河中]]尹、[[河中节度使|河中节度]]、晋绛观察等使。<ref name=BT176/>这年，[[中书侍郎|中书侍郎]]、集贤大学士[[李德裕|李德裕]]奏称郑肃有好子弟不敢科举。<ref name=BT18-1>[[:s:舊唐書/卷018下|《旧唐书》卷一十八下]]</ref><ref name=NBT44>[[:s:新唐書/卷044|《新唐书》卷四十四]]</ref>郑肃平时与李德裕交好，曾共同处理[[黠戛斯|黠戛斯]]、[[回鹘|回鹘]]等问题。<ref>[[:s:与黠戛斯可汗书|:s:与黠戛斯可汗书]]</ref><ref>[[:s:论回鹘事宜状|:s:论回鹘事宜状]]</ref><ref>[[:s:进所撰黠戛斯书状|:s:进所撰黠戛斯书状]]</ref>

== 唐武宗年间 ==
五年（840年）文宗崩，弟唐武宗继位，<ref name=ZZTJ246/>念故太子李永无罪，将当初陷害李永的人尽行杀害。朝议称郑肃忠正有大臣之节，武宗召回郑肃，拜太常卿。累迁为户部、兵部尚书。又迁[[山南东道节度使|山南东道节度使]]。<ref name=NBT182/>会昌五年（845年）七月，为检校尚书右仆射、[[同平章事|同平章事]]。<ref name=ZZTJ248>[[:s:資治通鑑/卷248|《资治通鉴》卷二百四十八]]</ref><ref name=NBT8>[[:s:新唐書/卷008|《新唐书》卷八]]</ref>加中书、门下二侍郎，监修国史。<ref name=BT176/>

== 唐宣宗年间 ==
六年（846年）三月，武宗崩，叔唐宣宗继位。宣宗不悦时为首相的李德裕专权，四月，罢李德裕宰相职，将其遣出长安。九月，郑肃因与李德裕交好，也被罢相，<ref name=BT176/><ref name=NBT8/>仍同平章事，充[[荆南节度使|荆南节度使]]。<ref name=ZZTJ248/><ref>《旧唐书》本传作复河中节度使。此从《资治通鉴》《新唐书》。</ref>卒，赠[[司空|司空]]，谥'''文简'''。卒年于史无载。<ref name=NBT182/>[[李商隐|李商隐]]文集《[[樊南文集补编|樊南文集补编]]》有《为荥阳公上荆南郑相公状》三篇。[[郁贤皓|郁贤皓]]《[[唐刺史考全编|唐刺史考全编]]》认为郑肃担任荆南节度使直至[[大中_(唐朝)|大中]]三年（849年），其任期或系因去世而告终。
