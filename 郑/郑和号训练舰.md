'''郑和号航海训练舰'''（[[舷号|舷号]]：'''81'''）<ref name="“八一”郑和舰：超级明星舰与军官摇篮">{{cite web |url = http://www.mod.gov.cn/big5/affair/2012-07/02/content_4382283.htm |title=“八一”郑和舰：超级明星舰与军官摇篮 }}</ref>是[[中国|中国]]自行设计建造的第一艘远洋[[航海|航海]][[训练舰|训练舰]]，以中国古代航海家[[郑和|郑和]]命名。于1986年7月12日由上海[[求新造船厂|求新造船厂]]下水，1987年4月27日建造完毕正式服役，隶属[[中国人民解放军海军大连舰艇学院|大连舰艇学院]]管辖，用于海军院校学员远洋航海训练，是[[中国人民解放军海军|中国人民解放军海军]]為海军院校建造的第一艘远洋航海训练舰，被誉为“中国军校第一舰”。

郑和号远洋航海训练舰舰长132.07米、舰宽15.8米，满载时排水量大于6000吨，最大航速18节，可连续航行10000海里，自持力为30昼夜，可承载170名船员以及30名老师和200名军校学员，学员可同时进行多科目的海上实习训练。舰上设有[[卫星导航|卫星导航]]仪和[[全球定位系统|全球定位系统]]等先进设备，以及全自动火炮、火箭深弹发射器等武器装备，教学、通信、机电、生活、医疗设施齐全，并设教学指挥中心和教学彩色[[闭路电视|闭路电视]]系统。

尽管在舰载武器装备上有差距，但是郑和号作为一艘远洋航海训练舰主要是以航海的基本技能训练为主，目前的武器、装备已经可以充分保障学员的海上实操演习。

==出访历史==
*1989年访问[[美国|美国]][[夏威夷|夏威夷]]
*1990年访问[[泰国|泰国]]
*1993年访问[[孟加拉|孟加拉]]、[[巴基斯坦|巴基斯坦]]、[[印度|印度]]、泰国
*2008年访问泰国、[[越南|越南]]和[[柬埔寨|柬埔寨]]
*2009年访问[[日本|日本]]
*2010年访问[[巴布亚新几内亚|巴布亚新几内亚]]、[[汤加|汤加]]、[[新西兰|新西兰]]、[[澳大利亚|澳大利亚]]
*2011年访问俄罗斯[[符拉迪沃斯托克|符拉迪沃斯托克]]和朝鲜[[元山|元山]]
*2012年4月16日，“郑和”号远洋航海训练舰从大连的军港启航，开始环球航行，并前往11个国家进行友好访问。<ref name="和谐使命-郑和舰环球行">{{cite web |url=http://www.chinanews.com/gn/z/zhjhqx/|title=和谐使命-郑和舰环球行}}</ref>
*2013年访问[[韩国|韩国]]、[[马来西亚|马来西亚]]
*2015年访问美国[[珍珠港-希卡姆联合基地|珍珠港-希卡姆联合基地]]<ref>{{cite web|url=https://www.navy.mil/submit/display.asp?story_id=91501|title=Chinese People's Liberation Army Navy Ship Visits Pearl Harbor|language=en|date=2015-10-14|accessdate=2018-11-24}}</ref>
