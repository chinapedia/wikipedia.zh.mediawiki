'''郑道济'''（{{bd|1907年||1988年|}}）原名'''郑钦业'''，今[[山东省|山东省]][[烟台市|烟台市]]西沙旺村人，[[中国共产党|中国共产党]]领导下的第一支海军部队[[八路军|八路军]][[胶东军区海军支队|胶东军区海军支队]]首任支队长。后任[[中华人民共和国|中华人民共和国]][[大连港集团有限公司|大连港务局]]副局长<ref name=zhidui>{{Cite web |url = http://www.sdkrzzjng.com/krgjd/gjdfzlc/201505/t20150529_12468480.htm |title =山东抗日根据地的海军支队  |author =  |date = 2015-05-29 |publisher = 山东抗日战争纪念馆 |language =  |accessdate =  |quote =  }}</ref><ref name=cpc>{{Cite web |url = http://cpc.people.com.cn/GB/85037/8541878.html |title = “海军支队”传奇 |author = 马骏杰 |date =  |publisher =中国共产党新闻网  |language =  |accessdate =  |quote =  }}</ref>。

==生平==
1907年，郑道济生于今山东省烟台市西沙旺村一个工人家庭。1922年至1924年，在烟台市当学徒工。1925年到1926年，考进[[东北海军|东北海军]][[葫芦岛|葫芦岛]]航警学校学兵班学习。1928年到1929年，考进[[青岛|青岛]][[崂山|崂山]]海军士官班学习。1929年到1931年，在“定海”军舰历任下士、中士班长。1931年到1932年，在青岛海军司令部任卫兵班长。1932年到1933年，在青岛[[礼贤中学|礼贤中学]]及北平路小学担任军事教员。1933年到1934年，在隶属海军第三舰队的威海海军教导队担任上士班长。1934年到1937年，在威海海军教导队担任准尉中队副。1938年到1939年，任[[山东省政府|山东省政府]]（主席[[沈鸿烈|沈鸿烈]]）驻[[海阳市|海阳县]]督察员办公室从事联络工作（督察员[[王进修|王进修]]）。1939年到1941年，在山东保安独立第七旅（驻[[莱阳县|莱阳县]]及海阳县，旅长[[姜黎川|姜黎川]]）历任团副、参谋主任、教导大队长。1941年到1942年，在[[蓬莱县|蓬莱县]]政府担任军事教官。1942年，郑道济到[[刘公岛|刘公岛]]加入[[汪精卫政权|汪精卫政权]]海军。1942年到1944年11月，在刘公岛上的[[汪精卫政权|汪精卫政权]]海军威海卫基地练兵营担任卫队队长区队长兼教官<ref name=zhidui/><ref name=cpc/>。

1944年11月5日，郑道济、连城、毕昆山等人在刘公岛发动武装起义。随后率起义部队加入[[八路军|八路军]]。郑道济任八路军[[胶东军区海军支队|胶东军区海军支队]]上校支队长<ref name=zhidui/><ref name=cpc/>。1946年1月到12月，任辽南人民自卫军三纵队、[[安东军区|安东军区]]副司令员。1947年，在[[东北军政大学|东北军政大学]]学习。1948年，出任东北军政大学校务科长。1948年到1950年，历任东北航务总局港务处副处长、机务处副处长。1949年2月在[[沈阳|沈阳]]的东北航务总局，经东北航务总局总局长[[吴志立|吴志立]]、副总局长[[周村_(东北航务总局)|周村]]介绍，加入[[中国共产党|中国共产党]]，1951年2月转为正式党员。1950年到1951年，任安东港湾管局副局长。1951年到1952年，任大连港务局港务处处长。1952年到1963年，任大连港务局港务副局长。1963年到1965年，任大连港务局行政副局长；1965年到1976年，先是待分配工作，后遭到批斗，住进[[牛棚_(文革用语)|牛棚]]、参加劳动改造、下放农村，并曾受到关押审察。[[文革|文革]]结束后获得平反。1979年后，任大连港务局顾问<ref name=wang>王学俭、王班，民族英雄郑道济，2012-10-06</ref><ref>{{Cite web |url = http://www.cssponline.com/plus/view.php?aid=2818 |title =  船舶登记局起航|author = 沈肇圻 |date =  2016-05-20|publisher = 国际船媒 |language =  |accessdate =  |quote =  }}</ref>。

1988年，郑道济病逝，享年81岁<ref name=wang/>。
