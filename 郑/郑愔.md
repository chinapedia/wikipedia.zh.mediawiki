'''郑愔'''（愔的[[讀音|讀音]]同音，{{bd|7世纪||710年|catIdx=Z}}），[[表字|字]]'''文靖'''，<ref name=QTS>《[[全唐诗|全唐诗]]》</ref>[[唐朝|唐朝]]和[[武周|武周]][[官员|官员]]，[[唐中宗|唐中宗]]年间短暂拜为[[宰相|宰相]]。

== 家世 ==
郑愔原本[[鄚姓|姓鄚]]，改姓郑，混入[[荥阳郑氏|荥阳郑氏]]<ref>《朝野佥载·卷四》；唐郑愔曾骂选人为“痴汉”，选人曰：“仆是吴痴，汉即是公。”愔令咏痴，吴人曰：“榆兒复榆妇，造屋兼造车。十七八九夜，还书复借书。”愔本姓鄚，改姓郑，时人号为“鄚郑”。</ref><ref>《太平广记·卷第二百五十五·嘲诮三》：唐郑愔曾骂选人为痴汉。选人曰：“仆是吴痴，汉即是公。”愔令咏痴。吴人曰：“榆儿复榆妇，造屋兼造车。十七八九夜，还书复借书。”愔本姓鄚，改姓郑，时人号为鄚郑。</ref>；父鄚玄昇，[[卫州|卫州]][[刺史|刺史]]。<ref>《新唐书》卷七十五上</ref>郑愔十七岁中[[进士|进士]]。[[武则天|武则天]]年间，武则天侄梁王[[武三思|武三思]]子[[武崇训|武崇训]]迎娶前皇帝[[皇太子|皇太子]][[李显|李显]]嫡女[[李裹儿|安乐郡主]][[李裹儿|李裹儿]]，武三思令宰相和郑愔等词人为赋《花烛行》。<ref>[[:s:旧唐书/卷183|《旧唐书》卷一百八十三]]</ref>郑愔谄事武则天男宠[[张易之|张易之]]、[[张昌宗|张昌宗]]，被推荐为[[殿中侍御史|殿中侍御史]]。<ref name=QTS/><ref name=ZZTJ208>[[:s:资治通鉴/卷208|《资治通鉴》卷二百零八]]</ref>

== 唐中宗复辟年间 ==
[[神龙_(年号)|神龙]]元年（705年），[[张柬之|张柬之]]、[[崔玄暐|崔玄暐]]、[[桓彦范|桓彦范]]、[[敬晖|敬晖]]、[[袁恕己|袁恕己]]等人發動[[神龍革命|神龍革命]]，殺[[张易之|张易之]]、[[张昌宗|张昌宗]]，迫武則天[[禪讓|禪讓]]，李显[[复辟|复辟]]，是为唐中宗，郑愔也因而被贬[[宣州|宣州]]司户参军，又犯[[貪汙罪|貪汙罪]]而潜逃到东都[[洛阳|洛阳]]，密会武三思。当时武三思为中宗宠臣且与有权势的[[韦皇后_(唐中宗)|韦皇后]]私通，正陷入与[[张柬之|张柬之]]、[[崔玄暐|崔玄暐]]等领导者的权力斗争。郑愔初见武三思，先大哭，再大笑。武三思素来贵重，感到迷惑。郑愔答：“我刚见大王时哭，是哀大王将遭戮[[灭族|灭族]]。后大笑，是喜大王得到我郑愔。大王虽然得[[天子|天子]]之宠，但那五人都据将相之权，胆略过人，废[[太后|太后]]（即武则天）如反掌。大王自视自己的势位与太后哪个更重？那五人日夜切齿，欲吃大王的肉，不灭尽大王一族不足以满足他们的心志。大王不去此五人，就危如朝露，却晏然还自以为安如[[泰山|泰山]]，这是我郑愔为大王寒心的地方。”武三思大悦，与郑愔登楼，问以自安之策，引为[[中书舍人|中书舍人]]，于是郑愔与[[崔湜|崔湜]]皆为武三思谋主。<ref name=QTS/>很快，武三思和韦皇后说服中宗表面上给五功臣封王，实则罢其相位。<ref name=ZZTJ208/>

二年（706年）春，上《圣感颂》，中宗为之刻石。<ref>[[:s:新唐书/卷206|《新唐书》卷二百零六]]</ref>郑愔素被敬晖等所废黜，五月任[[许州|许州]]司功参军时，在武三思指使下弹劾当时已都被遣出京城[[长安|长安]]的五功臣与先前三月因计划杀武三思、废韦后而伏诛的[[驸马都尉|驸马都尉]][[王同皎|王同皎]]同谋，六月，五功臣都被贬为边远州司马。七月，武三思又派人故意在洛阳声张自己和韦后的私情，意图激怒中宗，嫁祸五功臣。郑愔时任[[侍御史|侍御史]]，以此弹劾五功臣，请求族诛他们。中宗拒绝了，但将五功臣流放到更远的州，<ref name=BT100>[[:s:旧唐书/卷100|《旧唐书》卷一百]]</ref><ref>[[:s:新唐书/卷129|《新唐书》卷一百二十九]]</ref>他们随后或死去或被武三思下令杀死。<ref>[[:s:旧唐书/卷91|《旧唐书》卷九十一]]</ref><ref>[[:s:新唐书/卷120|《新唐书》卷一百二十]]</ref><ref name=ZZTJ209>[[:s:资治通鉴/卷209|《资治通鉴》卷二百零九]]</ref>

[[景龙|景龙]]元年（707年），中宗庶子皇太子[[李重俊|李重俊]]怒安乐公主李裹儿及武崇训常侮辱自己及试图以李裹儿为[[皇太女|皇太女]]取自己而代之，遂起兵杀武三思、武崇训，但随后被击败，逃亡中被杀。郑愔又亲附韦后，二年（708年）二月在[[太常少卿|太常少卿]]<ref>《资治通鉴》作[[太常卿|太常卿]]。</ref>任上引申歌曲《英王石州》《桑条韦》称中宗、韦后早有为帝为后之预兆，作《桑条歌》十篇献上，<ref>[[:s:旧唐书/卷37|《旧唐书》卷三十七]]</ref><ref >[[:s:新唐书/卷76|《新唐书》卷七十六]]</ref>中宗高兴，给予厚赏。<ref name=ZZTJ209/><ref>[[:s:旧唐书/卷51|《旧唐书》卷五十一]]</ref><ref>[[:s:旧唐书/卷92|《旧唐书》卷九十二]]</ref>郑愔与[[侍中|侍中]][[韦巨源|韦巨源]]、宰相[[宗楚客|宗楚客]]、右补阙[[赵延禧|赵延禧]]等说祥瑞妖异，私下引导韦后行武则天故事。<ref>[[:s:新唐书/卷123|《新唐书》卷一百二十三]]</ref>四月，中宗置脩文馆学士，郑愔在其中。<ref>[[:s:新唐书/卷202|《新唐书》卷二百零二]]</ref>

三年（709年）三月，郑愔在太常少卿兼检校吏部侍郎任上被任为守吏部侍郎、[[同平章事|同平章事]]，为[[实质宰相|实质宰相]]。<ref>[[:s:新唐书/卷4|《新唐书》卷四]]</ref>又任为[[吏部尚书|吏部尚书]]。他与时任[[中书侍郎|中书侍郎]]兼知吏部侍郎同平章事崔湜一起负责选官，两人都腐败，贪赃受贿。<ref>[[:s:旧唐书/卷70|《旧唐书》卷七十]]</ref><ref>[[:s:旧唐书/卷96|《旧唐书》卷九十六]]</ref><ref >[[:s:新唐书/卷50|《新唐书》卷五十]]</ref><ref>[[:s:新唐书/卷102|《新唐书》卷一百零二]]</ref><ref>[[:s:新唐书/卷124|《新唐书》卷一百二十四]]</ref>侍御史[[靳恒|靳恒]]与[[监察御史|监察御史]][[李尚隐|李尚隐]]弹劾之，监察御史[[裴漼|裴漼]]奉命审案。<ref name=BT100/>崔湜和郑愔都被罢官。五月，郑愔流放[[吉州|吉州]]，崔湜贬[[江州_(隋朝)|江州]]司马。<ref>[[:s:新唐书/卷130|《新唐书》卷一百三十]]</ref>中宗[[昭容|昭容]][[上官婉儿|上官婉儿]]、李裹儿、李裹儿新丈夫[[武延秀|武延秀]]从中说情后，崔湜为[[襄州_(西魏)|襄州]]刺史，<ref>[[:s:新唐书/卷99|《新唐书》卷九十九]]</ref>郑愔为江州司马。<ref>[[:s:旧唐书/卷7|《旧唐书》卷七]]</ref><ref>[[:s:旧唐书/卷74|《旧唐书》卷七十四]]</ref>九月，中宗又召曾为修文馆学士的崔湜、郑愔回京参与祭祀天地大礼，又任郑愔为吏部侍郎。<ref name=ZZTJ209/><ref>[[:s:旧唐书/卷185下|《旧唐书》卷一百八十五下]]</ref>

== 中宗死后 ==
四年（710年），中宗暴崩。传统史家认为他是被韦后和李裹儿毒杀，以使韦后如武则天般称帝，李裹儿为皇太女。中宗另一庶子温王[[李重茂|李重茂]]被立为帝，即[[唐殇帝|唐殇帝]]，韦后作为[[皇太后|皇太后]][[摄政|摄政]]握有实权。韦太后贬郑愔为江州司马，郑愔行经[[均州|均州]]，与失宠于中宗并遭贬的中宗子均州刺史谯王[[李重福|李重福]]及洛阳人[[张灵均|张灵均]]密谋起兵讨韦太后。<ref name=ZZTJ209/><ref name=BT86>[[:s:旧唐书/卷86|《旧唐书》卷八十六]]</ref>

中宗崩后不到一个月，中宗妹[[太平公主|太平公主]]和侄子临淄王[[李隆基|李隆基]]政变杀韦太后、李裹儿。李隆基父前皇帝相王[[李旦|李旦]]复辟为[[唐睿宗|唐睿宗]]取代了殇帝。郑愔当时未及举兵，已被召回任秘书少监，这时又贬为[[沅州|沅州]]刺史。他故意留在洛阳，料到李重福在张灵均策划下不久将起事。他在李重福妹[[宜城公主|宜城公主]]驸马都尉[[裴巽|裴巽]]的官邸做准备，起草诈称中宗命李重福继位及李重福称帝诏书各一，<ref name=ZZTJ209/><ref>[[:s:旧唐书/卷191|《旧唐书》卷一百九十一]]</ref>并自署左丞相、<ref name=BT86/>知内外文武事，张灵均以下也依次除署官职。<ref name=NBT81>[[:s:新唐书/卷81|《新唐书》卷八十一]]</ref>

9月9日，李重福到洛阳，住进裴巽家，洛阳官员吃惊逃散。李重福前去左、右屯营意图夺取军权，但留台侍御史[[李邕|李邕]]说服屯营拒之。李重福至营，营中乱箭射之。李重福攻左掖门，也被击退。他逃跑，意图藏匿，次日，搜索军队逼近，他投漕渠溺死。<ref name=BT86/><ref name=NBT81/>郑愔貌丑多胡须，梳头髻穿女人衣服躲在车里想逃，被捕。<ref>[[:s:旧唐书/卷97|《旧唐书》卷九十七]]</ref>与张灵均一同受审时，他吓得大腿发抖不能回答。张灵均则神气自若，看着郑愔说：“我和这种人举事，败了是应该的。”郑愔和张灵均都被带到东都闹市斩杀。<ref name=QTS/><ref >[[:s:新唐书/卷5|《新唐书》卷五]]</ref><ref>[[:s:新唐书/卷125|《新唐书》卷一百二十五]]</ref>郑愔被[[族诛|族诛]]。<ref name=ZZTJ210>[[:s:资治通鉴/卷210|《资治通鉴》卷二百一十]]</ref>

有诗一卷。<ref name=QTS/>
