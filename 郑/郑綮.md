'''郑綮'''（{{bd|9世纪||899年|catIdx=Z}}），[[表字|字]]'''蕴武'''，[[中国|中国]][[唐朝|唐朝]][[官员|官员]]，[[唐昭宗|唐昭宗]][[乾宁|乾宁]]元年（894年）短暂拜为[[宰相|宰相]]。他以作[[歇后语|歇后语]]诗讽刺政局打动唐昭宗闻名，这导致他即使忧虑不安仍被短期拜为宰相。

== 家世 ==
郑綮家世不详，未被列入《[[新唐书|新唐书]]·宰相世系表》。<ref>《新唐书》卷七十五</ref>中[[进士|进士]]后，历任[[监察御史|监察御史]]、[[殿中侍御史|殿中侍御史]]、仓部员外郎，[[唐僖宗|唐僖宗]][[乾符|乾符]]二年（875年）六月改为户部员外郎，<ref>[[:s:旧唐书/卷19下|《旧唐书》卷一十九下]]</ref>又历任金部郎中、刑部郎中、右司郎中。<ref name=BT179>[[:s:舊唐書/卷179|《旧唐书》卷一百七十九]]</ref><ref name=NBT183>[[:s:新唐書/卷183|《新唐书》卷一百八十三]]</ref>

虽然这些朝廷官职都能带来声望，却没有足够的薪水。郑綮因家贫，请求离开京城[[长安|长安]]去做地方官，于是被出为[[庐州|庐州]][[刺史|刺史]]。乾符年间，他捕获为盗的[[杨行愍|杨行愍]]，认为其相貌异于常人，说：“你将要富贵，为何做贼？”松绑放了他，<ref>[[:s:新唐书/卷188|《新唐书》卷一百八十八]]</ref><ref>《十国春秋·吴太祖世家》</ref>后又用为本州步奏官，也曾因其有过失而笞责他。<ref>《[[北梦琐言|北梦琐言]]》</ref>[[广明_(唐朝)|广明]]元年（880年），大规模农民变军[[黄巢|黄巢]]北进，<ref>[[:s:資治通鑑/卷253|《资治通鉴》卷二百五十三]]</ref><ref>[[:s:資治通鑑/卷254|《资治通鉴》卷二百五十四]]</ref>郑綮给黄巢写文牒，请求不犯自己地界。黄巢笑而从之，于是唯独没有冒犯庐州。唐僖宗嘉奖他，赐他绯鱼袋。离任时，他积蓄了钱一千缗，寄放在庐州库内。后来庐州数次失陷，乱军也不劫掠这笔钱。杨行愍任刺史后，将这笔钱送到京师还给了郑綮。<ref name=BT179/><ref name=NBT183/>

郑綮善于写诗，多讽刺时事，不着格调，被时人号为“郑五歇后体”。<ref name=NBT183/>刚从庐州离任时，他和庐州人告别，说：“唯有两行公廨泪，一时洒向渡头风。”他的滑稽大概如此。<ref name=BT179/>

[[王徽_(京兆)|王徽]]任[[御史大夫|御史大夫]]时，<ref>王徽显然仅在[[文德_(唐朝)|文德]]元年（888年）唐僖宗从[[凤翔|凤翔]]重返长安期间短暂担任御史大夫，因《旧唐书·王徽传》称他在凤翔受任但在僖宗回京后辞职。故此事发生在888年。见[[:s:舊唐書/卷178|《旧唐书》卷一百七十八]]和[[:s:資治通鑑/卷257|《资治通鉴》卷二百五十七]]。</ref>奏郑綮为兵部郎中、知御史台杂事，迁[[给事中|给事中]]，赐金紫。僖宗任宰相[[杜让能|杜让能]]弟[[杜弘徽|杜弘徽]]为[[中书舍人|中书舍人]]。郑綮封还诏书，称兄弟互相监督是不妥的。僖宗无视其反对，于是郑綮称病辞职。<ref name=BT179/>

不久，郑綮被召还为右散骑常侍。<ref>《旧唐书》作左散骑常侍。</ref>他每看到朝政有问题，就上表章理论。他的建议虽然多不被受理，但流传京城。执政官员厌恶他，改任他为[[国子祭酒|国子祭酒]]。时人以为他因进谏而被迁散官，执政官员害怕议论，又复任他为右散骑常侍。<ref name=BT179/><ref name=NBT183/>

== 拜相及之后 ==
[[唐昭宗|唐昭宗]][[大顺_(唐朝)|大顺]]后，皇权不振。当时郑綮作诗讽刺朝政，有[[宦官|宦官]]在昭宗面前背诵，昭宗因而注意到郑綮的诗，读到诗中攻击朝政的内容，认为有未尽的内涵。当时杨行愍已改名[[杨行密|杨行密]]，为[[淮南节度使|淮南节度使]]，也盛称郑綮之德。昭宗就于乾宁元年二月趁有司献上朝官名册时，在名册之侧写下任当时为朝议大夫、守右散骑常侍、[[上柱国|上柱国]]、'''荥阳县男'''的郑綮为礼部侍郎，加宰相衔[[同中书门下平章事|同中书门下平章事]]。<ref name=NBT10>[[:s:新唐书/卷10|《新唐书》卷一十]]</ref><ref>《旧唐书·昭宗纪》作乾宁四年（897年）二月事。</ref>[[中书省|中书省]]胥吏去郑綮家参谒，郑綮不以为然，笑着说：“诸君大误，就算天下人都不识字，宰相也轮不到郑五来做。”胥吏说：“出自圣旨特恩，明天诏书就下达了。”郑綮抵住胥吏的手说：“万一如此，会笑死人的。”次日诏书果然下达，亲朋宾客来道贺，郑綮搔着头说：“写歇后语的郑五做宰相，可知时事如何了。”屡次上表辞让，不获准，于是不得不上任宰相。<ref name=BT179/><ref name=NBT183/><ref name=ZZTJ259>[[:s:資治通鑑/卷259|《资治通鉴》卷二百五十九]]</ref>内外都震惊了。<ref name=BK>《十国春秋·备考》</ref>

郑綮拜相后，刚直履职，不再作诙谐语。但他一直觉得自己不是大众所期望的宰相，仍然请求引退。曾有人问他是否有新的诗作，他答：“诗思在灞桥风雪驴子上，此处何以得之？”<ref>《[[北梦琐言|北梦琐言]]》</ref>拜相三月有余，称病请求[[致仕|致仕]]，于是在七月获诏以[[太子少保|太子少保]]致仕。<ref name=NBT183/><ref name=NBT10/><ref name=ZZTJ259/><ref>《旧唐书·昭宗纪》作乾宁四年（897年）四月事。</ref>[[光化|光化]]二年（899年）卒。<ref name=BT179/>

侄孙[[郑珏|郑珏]]。<ref>[[:s:旧五代史/卷58|《旧五代史》卷五十八]]</ref>

== 评价 ==
当时的议论认为昭宗任命的宰相中，对[[张濬|张濬]]、[[朱朴|朱朴]]、郑綮的任命是尤其错误的。

*《旧唐书》史臣曰：逐徐（[[徐彦若|徐彦若]]）、薛（薛王[[李知柔|李知柔]]）于瘴海，置'''綮'''、朴于岩廊。<ref name=BT179/>
*《新唐书》赞曰：观'''綮'''、朴辈不次而用，捭豚臑，拒貙牙（都指拒绝贤能的辅臣），趣亡而已。<ref name=NBT183/>
