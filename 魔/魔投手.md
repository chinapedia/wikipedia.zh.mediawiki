《'''魔投手'''》（侍ジャイアンツ）是一部[[棒球|棒球]][[漫畫|漫畫]]（[[梶原一騎|梶原一騎]]原作、[[井上コオ|井上コオ]]畫），連載於《[[週刊少年Jump|週刊少年Jump]]》。敘述主角王大蠻（番場蛮）在[[日本職棒|日本職棒]][[讀賣巨人|讀賣巨人]]隊中不斷發明魔球，幫助巨人隊[[V9|九連霸]]並得到世界冠軍。[[電視動畫|電視動畫]]由[[讀賣電視台|讀賣電視台]]製作，於1973年10月7日至1974年9月15日在[[日本電視系|日本電視系]]播放，共46話。在[[台灣|台灣]]由[[中國電視公司|中國電視公司]]（中視）首播，在[[韓國|韓國]]由[[文化廣播_(韓國)|文化廣播]]首播於1993年。

== 概要 ==
描述以打倒宿敵[[讀賣巨人|讀賣巨人]]為目標的巨人王牌速球投手『番場蠻』活躍的棒球漫画。

當時的《週刊少年Jump》和讀賣巨人訂立獨占契約，所以能夠讓[[長嶋茂雄|長嶋茂雄]]和[[王貞治|王貞治]]等真實的棒球選手登場<ref>在西村繁男所作的『さらばわが青春の「少年ジャンプ」』（1994年、[[飛鳥新社|飛鳥新社]]）中，[[講談社|講談社]]的『[[週刊少年Magazine|週刊少年Magazine]]』所連載的『巨人之星』雖然違反這個契約，但當時的編輯長[[長野規|長野規]]以為了漫画界j為由默認。不過依據大野茂寫的『サンデーとマガジン』（2009年4月、光文社），1965年『週刊少年Magazine』編輯長[[内田勝|内田勝]]直接找讀賣巨人廣告部的坂本幸夫談論YG標誌及巨人隊選手肖像在『巨人之星』的使用權，巨人隊在了解後與集英社終止契約（P210）。</ref>。雖然有許多像是高跳躍式魔球、大迴旋魔球等真正使用就會違反[[棒球規則|棒球規則]]的魔球，卻以開發魔球並且由對手攻略魔球作為故事主軸。動畫播放時，男子小学生紛紛用[[軟式網球|軟式網球]]「重現」分身魔球，而模仿其投法。

本作是透過讓『[[巨人之星|巨人之星]]』大賣的梶原一騎之力而提出以動畫化為前提的企畫。在連載三年下來，曾因為人氣低迷差點要被[[腰斬_(術語)|腰斬]]，但在動畫決定後人氣再度上昇而免於這個命運。不過當時《週刊少年Jump》[[編集長|編集長]][[西村繁男|西村繁男]]認為，本作進入前5名是靠著安定的人氣，但沒有東西能吸引新讀者<ref>同上的『さらばわが青春の「少年ジャンプ」』、同西村繁男『まんが編集術』（1999年、[[白夜書房|白夜書房]]）</ref>。

對於本作，有著只是模仿『巨人之星』的版本、「二流」等聲音<ref>依據本作動畫版的[[作画監督|作画監督]][[大塚康生|大塚康生]]。『リトル・ニモの野望』（2004年、[[徳間書店|徳間書店]]）、『大塚康生インタビュー アニメーション縦横無尽』（2006年、[[実業之日本社|実業之日本社]]）。漫画評論家[[米澤嘉博|米澤嘉博]]所作『戦後野球マンガ史 手塚治虫のいない風景』（2002年、[[平凡社|平凡社]]）所述</ref>。相對於成為比較對象的『巨人之星』主角星飛雄馬那種宛如求道般的認真態度，本作主角番場蠻當初的設定就是「討厭囂張的傢伙」而要反抗巨人，與雄馬的立場剛好相反。實際上、當初的番場在漫画版中想要加入[[中央聯盟|中央聯盟]]最後一名球隊、在動畫版中則是想要加入[[橫濱海灣之星|大洋鯨]]、但經過一段曲折後加入巨人隊並且成為和『巨人之星』一樣的魔球對決故事。另外在職棒篇後，也被指出有一些點與『新巨人之星』相同<ref>[[豊福きこう|豊福きこう]]著『水原勇気 0勝3敗11S』（1992年、[[宝島社|情報センター出版局]]）第四章</ref>。本作還和『新巨人之星』一樣讓在原作與現實中都未達成的「最終回讓巨人得到優勝」在動畫版中加以實現。另一方面，從番場蠻那種無賴的態度和沒有讓他稱為「師父」的人物登場來看，也有貼近於「[[小拳王|小拳王]]」的意見。

負責作畫的井上コオ本來是[[望月三起也|望月三起也]]的[[助手|助手]]而只是個新人、以短篇漫畫出道，本作卻是他的初連載，事實上他大賣的漫畫只有這個作品。後年本作的番場和烏魯夫、八幡等人也以[[明星系統|明星系統]]而在[[求人情報誌|求人情報誌]]中連載其日常搞笑漫画。井上也有推出以番場蠻外甥為主角的『よみがえれ侍』及『新・巨人之星』「井上コオ版」，這時擔任井上助手的是[[車田正美|車田正美]]。

動畫版播放開始時的1973年10月7日，這個作品的舞台背景是以1973年球季的巨人為準，那時巨人在[[中央聯盟|中央聯盟]]中處於很難獲得優勝的狀態。之後，隨著10月11日巨人×阪神10-10的死鬥（第37話「怒りに燃えた分身魔球」的背景），10月20日巨人先發九人搭乘新幹線通過附近的中日球場時、魔術數字只剩1的阪神輸給中日（第38話「大砲・運命の一打」的背景），10月22日阪神×巨人最終決戰中、巨人勝利而獲得V9（第39話「輝け苦闘のV9」的背景）等事件成為日本職棒史上最戲劇性的發展，都在本作一一表現出來。當然，在企畫當時完全想像不到這種狀況，結果就是讓現實的賽程跟著影響整部故事的發展。

漫画版的結局中，在[[中央聯盟|中央聯盟]]的關鍵戰——巨人對中日戰中，番場蠻因為投太多魔球而被對手大砲萬作打出去，讓他當場站著死在投手丘上。在運動漫畫中從來都沒有選手生涯斷送後、生命也跟著逝去的表現，本作則被梶原一騎在棒球漫画中加以表現出來。這個結果有著被評為「太過簡單地就讓故事變得太激烈」或「武士既然不能獲得優勝，就要以生命來負責」的看法，使得本作以死作結的表現對讀者留下了衝擊，伴隨著動畫後而成為讓人一直談到的人氣作品<ref>[[米澤嘉博|米沢嘉博]]『戦後野球マンガ史 手塚治虫のいない風景』（2002年、[[平凡社|平凡社]]）。『別冊宝島 70年代マンガ大百科』（[[宝島社|宝島社]]、1996年）</ref>。

== 動畫版 ==
1973年10月7日開始播放。由曾作出『巨人之星』的[[東京電影|東京電影]]（{{lang|ja|東京ムービー}}）負責、導演同樣由『巨人之星』的[[長濱忠夫|長濱忠夫]]擔任。

當時的候補導演人選有還在東京電影旗下的[[A_Production|A Production]]（{{lang|ja|Aプロダクション}}，今[[Shin-Ei動畫|Shin-Ei動畫]]）的[[高畑勲|高畑勲]]；但因為正式人選確定、讓高畑在導演完『[[小天使|小天使]]』（{{lang|ja|アルプスの少女ハイジ}}）後離開。[[宮崎駿|宮崎駿]]等人也在負責一話的原畫後一起離職<ref>高畑和宮崎等人因為所追求的風格差異太大、所以後年在雜誌『熱風』2009年2月号的訪問中、宮崎表示「我對於沉迷棒球的人、說真的、難以理解」，不是要對努力負責「『巨人之星』的楠部先生（當時的A Production董事長）說壞話。但是他和我們所想的是完全不同。我們是想要和パク先生（高畑勲）一起作出不同東西的人們」。</ref>。

:本作主角，個性自大卻不認輸。原本是名控球差卻球速過人的投手，但隨著故事的推進而終於變成一名球速和控球兼具的王牌投手。
:當初想要當漁夫養家，但經過一番波折後加入巨人隊。由於父親曾被大鯨魚殺害的關係，討厭巨人隊，甚至一度發誓要打倒巨人隊；不過在川上哲治、王貞治、長島茂雄及八幡太郎平等人的影響下，全心為巨人隊付出。

;番場雪（{{lang|ja|番場ユキ}}）
:中視版人名：小雪／配音：[[吉田理保子|吉田理保子]](日本)、[[許淑嬪|許淑嬪]](台灣)
:番場蠻的妹妹，個性調皮，但很敬愛哥哥。

;番場菊（{{lang|ja|番場キク}}）
:中視版人名：王秋菊／配音：[[金子亞矢子|金子亞矢子]](日本)、[[許淑嬪|許淑嬪]](台灣)

;八幡太郎平（{{lang|ja|八幡太郎平}}）
:中視版人名：陳大仁／配音：[[納谷六朗|納谷六朗]](日本)、[[官志宏|官志宏]](台灣)
:番場蠻在學校和巨人隊的學長。過去在[[甲子園|甲子園]]投過[[完全比賽|完全比賽]]，但在進入日本職棒後，由於個性軟弱而無法發揮實力。
:雖然個性軟弱，卻樂於幫助番場蠻。最後在知道自己的極限後，決定全心幫助番場蠻成為巨人隊的王牌投手，並且因此轉任捕手。

;美波理香（{{lang|ja|美波理香}}）
:中視版人名：林美玲／配音：[[武藤禮子|武藤禮子]](日本)、[[許淑嬪|許淑嬪]](台灣)
:本作女主角，與番場蠻同校，是有錢人家的大小姐，也是番場蠻所愛的人。
:經常周旋於番場蠻和其他人之間，對於番場蠻有一定的影響力，最終在沒有告訴番場蠻的情況下到美國留學。

;眉月光（{{lang|ja|眉月光}}）
:中視版人名：馬海峰／配音：[[井上真樹夫|井上真樹夫]](日本)、[[于正昇|于正昇]](台灣)
:番場蠻的宿敵之一，長相俊秀，擁有優秀打擊能力。高中畢業後，為了與番場蠻對決，直接放棄進入大學而加入[[養樂多燕子|養樂多燕子]]。
:屢次要破解番場蠻的魔球，但最終為了打他的魔球讓自己受傷而被迫暫時離隊養傷。

;大砲萬作（{{lang|ja|大砲万作}}）

:番場蠻的宿敵之一，體型巨大，出身[[飛驒山|飛驒山]]上。在番場蠻到飛驒山上特訓時，表現出他恐怖的怪力及揮棒速度。之後加入[[中日龍隊|中日龍隊]]，而發誓要打敗番場蠻。
:家境困難而且有許多弟妹。一度因為弟妹攻擊番場蠻而引咎離隊；但在番場蠻的訓斥下，決意再次回到球場。

;烏魯夫·吉夫（{{lang|ja|ウルフ・チーフ}}）
:中視版人名：烏魯夫·吉夫／配音：[[桑原毅|桑原毅]](日本)、[[劉傑_(配音員)|劉傑]](台灣)
:番場蠻的宿敵之一，[[美國|美國]]籍[[印第安人|印第安人]]。自稱因為身體內流著狼族的血液，所以打球相當粗暴，在美國棒球界以刻意用[[釘鞋|釘鞋]]傷害守備球員的滑壘法聞名。
:在巨人隊赴美特訓認識了和他一樣不服輸的番場蠻，於是視他為自己的對手，而從美國來到日本並且加入[[阪神虎|阪神虎]]，與番場蠻有多次的激鬥。

;瑞吉·傑克森（{{lang|ja|ロジー・ジャックス}}）
:中視版人名：瑞吉·傑克森／配音：[[細井重之|細井重之]](日本，只限動畫版)、[[于正昇|于正昇]](台灣)
:[[美國職棒|美國職棒]][[奧克蘭運動家|奧克蘭運動家]]隊的王牌打者，擁有相當優異的打擊能力而幫助運動家隊拿到大聯盟冠軍。
:在日美冠軍賽中，一度被番場蠻的魔球壓制，但之後破解他的魔球而對番場蠻產生一定的威脅，最終被番場蠻以奇蹟魔球打敗。

;[[王貞治|王貞治]]（{{lang|ja|王貞治}}）
:中視版人名：王貞治／配音：[[石森達幸|石森達幸]](日本)、[[于正昇|于正昇]](台灣)
:巨人隊的先發第四棒，對於番場蠻而言是個亦師亦友的隊友。

;[[長島茂雄|長島茂雄]]（{{lang|ja|長島茂雄}}）
:中視版人名：吳勝年／配音：[[山田俊司|山田俊司]](日本)
:巨人隊的先發第三棒，特徵是一臉鬍渣。對於番場蠻而言是個亦師亦友的隊友。

;[[川上哲治|川上哲治]]（{{lang|ja|川上哲治}}）
:中視版人名：李川元／配音：[[西田昭市|西田昭市]](日本)、[[孫中台|孫中台]](台灣)

;[[金田正一|金田正一]]（{{lang|ja|金田正一}}）

== 漫画版和動畫版差異 ==
漫画版中登場的一些角色和[[魔球|魔球]]，在動畫版中未登場。

漫画版中沒有主角特訓的描寫，反而是在不曉得何時就輕鬆學會分身魔球；動畫版則相當強調特訓的部分。漫画和動畫中，各魔球的攻略法完全不同，故事內容也不一樣。因此讓故事及[[角色設定|角色設定]]受到漫画原作的拘束，而只能緩緩進行。

另外，漫畫版中實際存在的職業棒球選手（[[野村克也|野村克也]]、[[衣笠祥雄|衣笠祥雄]]、[[張本勳|張本勳]]、[[田淵幸一|田淵幸一]]等人）都有挑戰番場蠻的魔球並且打敗魔球的場景。

在[[中央聯盟|中央聯盟]]9連霸中對[[阪神虎|阪神虎]]戰，漫画版中和現實一樣，隨著[[堀內恒夫|堀內恒夫]]的完封勝(9比0)，讓番場蠻看到他的奮力投球而感動到流淚；動畫版中則是在9局中，堀内上場救援而成為勝利投手。

最大的差異則是在結尾：漫画版中以番場蠻壯烈的死亡作結，動畫版中則是番場蠻打敗世界最強打者[[奧克蘭運動家|奧克蘭運動家]]的瑞吉‧傑克森（被認為是以運動家隊當時的主砲[[Reggie_Jackson|Reggie Jackson]]為原型）贏得美日職棒大賽、獲得[[MVP|MVP]]的快樂結局。

另外日美決賽，實際上是動畫原創事件。會有這樣的差異，是動畫版要播放最終回而在製作時，因為原作還沒完結，所以有需要一個原創結局；由於本作當時是兒童向動畫，於是在電視台的要求下而作成快樂結局。

==魔球種類==
===漫畫版與動畫版共通魔球===
;高跳躍式魔球（ハイジャンプ魔球）
: 不用助跑而直接跳到空中投球。漫画設定中、能跳到15公尺(兩層樓)高。

;大迴旋魔球（大回転魔球）
: 在投手丘上利用單腳來旋轉。製造出許多殘影而因此跑出許多手出來、讓打者完全不曉得從那裡投出。球威比高跳躍式魔球系列更強。漫画設定中、與高跳躍式魔球一樣，是抬高右腳而進入投球模式。因此讓打者無法判斷是投高跳躍式或大迴旋魔球

;分身魔球（分身魔球）
: 動畫版設定中、將球握到變型。讓球的橫向（或是縱向）產生激烈的動搖、使打者看到許多球的分身而難以打擊。漫画的設定中、還提示說「就算被說是犯規，只要裁判沒抓就不算」「反過來利用公認的棒球規則的盲點」、不過沒有說出分身的原理。

===漫畫版獨有魔球===
;高跳躍式大迴旋魔球（ハイジャンプ大回転魔球）
: 在迴轉時跳到空中而從空中投球。是綜合高跳躍式和大迴旋魔球的投法。

;ハラキリシュート

===動畫版獨有魔球===
;蝦式跳躍魔球（エビ投げハイジャンプ魔球）
: 不用助跑而直接跳到空中並且將身體彎曲成蝦子型，利用其反彈的力道來投球。不但球速增加，而且還可以讓打者看不到番場蠻的握球方法。

;奇蹟魔球（ミラクルボール=ハイジャンプ大回転分身魔球=分身魔球+大迴旋魔球+高跳躍魔球）
:動畫版最終回中所使出的魔球，由於番場蠻先前的魔球一一被瑞吉所破，於是在綜合分身魔球+大迴旋魔球+高跳躍魔球的投法後而使出這一招。

== 外傳 ==
動畫版中，番場蠻雖然活著、但卻沒有續集動畫。漫画版的續集則是有以在[[藤田元司|藤田]]的巨人時代，對中日戰中，讓番場蠻的外甥登板的外傳『よみがえれ侍』並且在1991年打工情報誌『[[FromA|FromA]]』7号中出現。番場分別投出「高跳躍式」、「大迴旋」、「分身」3種魔球、將[[落合博滿|落合博滿]]三振掉。八幡則教導番場蠻的外甥（他妹妹的孩子）怎麼投魔球。由於是兩頁的短篇作、之後就沒有下文了。

== 小故事 ==
* 在[[柳田理科雄|柳田理科雄]]的『空想科学漫画讀本』中分別檢視「高跳躍式魔球」和「大迴旋魔球」。柳田寫著「關於分身魔球在『空想非科学大全』已經檢視過了」、實際上『非科学大全』中，是以忍者的「分身術」為例來說明「用迅速的行動來製造分身」的原理、但沒有觸碰到『魔投手』的「分身魔球」。
* 由於擔任補手看慣分身魔球而能夠捕接到球的八幡，是本作中可能可以破解這招的打者之一、但卻沒人想到和他一樣的訓練方法。
* 2002年12月14日播放的超級特別節目'02「[[徳光&所のスポーツえらい人グランプリ|徳光&所のスポーツえらい人グランプリ]]」中、再現了魔投手中的魔球。像是使出分身魔球(讓雙胞胎站在投手丘上並且由其中一人投出來)、高跳躍式魔球(吊纜繩到高處來投球)來對付大聯盟打者邦茲、吉昂比、威廉斯。雖然當初打者會感到困擾、但在習慣後反而能打出全壘打。還有，同節目還再現了[[巨人之星|巨人之星]]大聯盟魔球2号・消失的魔球(在投手丘和本壘板安裝會放出煙霧的機器)、但也是很快就被打出去了。
* 2006年11月4日[[キッズステーション|キッズステーション]]的播放中、在各話結束後，會有『魔投手的問題』的小解說。在有較為科学性的場合中、能知道番場的球速比[[新幹線|新幹線]]還快。
* 動畫版中、番場蠻在打擊時，音樂會換成「[[侍ニッポン|侍ニッポン]]」（作詞：[[西條八十|西條八十]]、作曲：[[松平信博|松平信博]]）、但因為著作權問題而在再次播放及DVD版中都被修改掉。
* [[長嶋茂雄|長嶋茂雄]]和史實及其他作品一様、基本上個性溫和、本作在取代[[川上哲治|川上哲治]]當教練時遇到慘敗、然後還有對驕傲的番場蠻施以鐵拳制裁的熱血面。
