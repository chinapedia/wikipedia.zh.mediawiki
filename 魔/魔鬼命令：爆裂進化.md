《'''魔鬼命令：爆裂進化'''》（{{lang-en|'''''Universal Soldier: Regeneration'''''}}）是一部2009年美國[[科幻片|科幻]][[動作片|動作片]]，為[[約翰·海姆斯|約翰·海姆斯]]執導和剪輯。而攝影師{{link-en|彼得·海姆斯|Peter Hyams}}則是約翰的父親，曾與[[尚-克勞德·范·達美|尚-克勞德·范·達美]]在《{{link-en|時空特警|Timecop}}》（1994年）、《{{link-en|絕命殺陣|Sudden Death (1995 film)}}》（1995年）和《{{link-en|仇敵當前|Enemies Closer}}》（2013年）中合作過。

本片為「[[魔鬼命令系列電影|魔鬼命令系列電影]]」的第三部電影（忽略兩部電視電影），故事上是直接作為1992年電影《[[魔鬼命令|魔鬼命令]]》的續集，並忽略掉1999年的《[[魔鬼命令：重返戰場|魔鬼命令：重返戰場]]》。電影在美國和全球大都以[[錄影帶首映|錄影帶首映]]的方式於2010年2月2日發行，只有少部分的國家有在戲院上映。該片獲得了褒貶不一的評價，但建立了一種受小眾粉絲歡迎的[[邪典追捧|邪典追捧]]。

續集《[[魔鬼命令：轟天決戰|魔鬼命令：轟天決戰]]》於2012年推出。

== 劇情 ==
故事敘述一群由指揮官托波為首的恐怖分子綁架了烏克蘭總理的兒子和女兒作為人質，要求在72小時內將他們被關的同志釋放，不然就要引爆被托波佔據的[[車諾比核電廠|車諾比核電廠]]中的炸彈。恐怖分子和反叛政府的柯林博士合作，靠新一代的再造戰士（NGU）擊退了軍方派出的突襲部隊。情急之下，決定將在瑞士受治療的初代再造戰士路克·狄弗洛召回與其對抗，路克除了救回人質外還得及時解除炸彈。

== 反響 ==
該片發行後獲得了比《魔鬼命令：重返戰場》還好的評價，這使DVD的銷量狀況良好。{{link-en|恐懼基地|Dread Central}}網站在5刀的評價中給了本片3刀，認為電影沒什麼特別的，它就是一部扎實的B級動作片<ref>[http://www.dreadcentral.com/reviews/universal-soldier-regeneration-2009 Dread Central - Universal Soldier: Regeneration Review]</ref>。

== 續集 ==
2010年5月宣布，范·達美和朗格將回歸第四部電影，約翰·海姆斯再次擔任導演。續集《[[魔鬼命令：轟天決戰|魔鬼命令：轟天決戰]]》為系列中首次以[[3D電影|3D]]的格式製作。
