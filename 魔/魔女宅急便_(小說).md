《'''魔女宅急便'''》（{{lang-ja|魔女の宅急便}}）是由日本兒童文學作家[[角野榮子|角野榮子]]所創作的小說。

== 概要==
此小說原先在1982至1983年裡在福音館出版社的月刊{{nihongo|《母之友》|母の友}}上連載，之後於1985年匯集成單冊發行。其後於1993年至2009年期間推出了相關續作，最終以六冊作結，2016-2017年推出兩冊特別編。

== 系列故事 ==

:內容描述著13歲的魔女{{nihongo|琪琪|キキ}}為了面對是否能夠成為獨當一面魔女的考驗，與能和她互相對話的黑貓{{nihongo|吉吉|ジジ}}一同來到{{nihongo|克里克|コリコ}}城市，而琪琪決定利用自身可借助掃帚飛行的能力，為城市裡的居民提供寄送物品的服務。
:之後琪琪在面臨各種寄送物品的狀況下一邊成長，也慢慢對一名喜愛研究飛行、名叫{{nihongo|蜻蜓|とんぼ}}的少年有了接觸。

:系列第二作，是距離前作發表8年後再次推出的小說。
:劇情架構在琪琪來到克里克城市生活一年後的故事，內容為琪琪一如往常地利用飛行的能力為克里克市民遞送物品，之後發生了一些讓琪琪思考魔女在社會上給予他人的感觀、以及能力問題的挫折，而讓琪琪決心向媽媽習得新魔法以充實自己。

== 改編影視作品 ==
{{seealso|魔女宅急便|魔女宅急便 (2014年電影)}}
《魔女宅急便》小說推出後已有改編成各類型的影視作品。其中[[魔女宅急便|動畫版]]是由[[吉卜力工作室|吉卜力工作室]]基於系列第一冊內容所改編，並於1989年上映。<ref>{{Cite book |script-title=ja:スタジオジブリの軌跡 1984-2011 |publisher=[[德間書店|德間書店]] |isbn=978-4-19-720326-0 |pages=53頁 |date=2011年4月1日 |accessdate=2013年5月20日 }}</ref>

音樂劇版則由[[蜷川幸雄|蜷川幸雄]]擔任導演；分別曾在1993、1995、1996年公開演出，而各年份擔任飾演琪琪的演員依序為{{link-ja|工藤夕貴|工藤夕貴}}、[[小高惠美|小高惠美]]及{{link-ja|持田真樹|持田真樹}}。

在2014年上映的[[魔女宅急便_(2014年電影)|真人電影版]]則由[[清水崇|清水崇]]執導，並以曾在電視劇《[[無法呼吸的夏天|無法呼吸的夏天]]》演出的[[小芝風花|小芝風花]]擔任女主角琪琪。<ref>{{Cite web |url=http://hochi.yomiuri.co.jp/entertainment/news/20130424-OHT1T00015.htm |script-title=ja:小芝風花、映画デビューで実写版「魔女の宅急便」主演 |work=[[讀賣新聞|讀賣新聞]] |date=2013年4月26日 |accessdate=2013年5月20日 |deadurl=yes |archiveurl=https://web.archive.org/web/20130501144759/http://hochi.yomiuri.co.jp/entertainment/news/20130424-OHT1T00015.htm |archivedate=2013年5月1日 }}</ref>
