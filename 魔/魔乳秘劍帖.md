{{漫畫}}
{{日本作品|魔乳秘劍帖|魔乳秘剣帖}}是[[山田秀樹|山田秀樹]]所作的[[日本|日本]][[漫畫|漫畫]]作品。 

在[[Enterbrain|Enterbrain]]所發行的[[日本成人遊戲|成人遊戲]]雜誌《[[TECH_GIAN|TECH GIAN]]》上連載中的[[時代劇|時代劇]]漫畫。中文繁體版漫畫由[[台灣角川書店|台灣角川書店]]代理出版。廣播劇CD於2010年10月推出，之後[[電視動畫|電視動畫]]2011年7月開始播放。

== 故事簡介 ==
[[江戶時代|江戶時代]]，{{nihongo|「乳房乃是世上的真理」|乳こそこの世の理}}，[[巨乳|豐乳]]是絕對的財富，[[貧乳|貧乳]]則不被當人看待。在[[幕府|幕府]]中的「魔乳一族」掌控政治、支配刺客、培育[[大奧|大奧]]中的豐乳，其秘傳書記載古今流傳豐胸美乳的技巧。魔乳一族的-{么}-女也是繼承者'''魔乳千房'''帶著秘傳書叛逃，並為女性替天行道……。

== 登場人物 ==
<div class="notice metadata" id="spoiler"><small>'''※[[聲優|聲優]]：廣播劇CD／電視動畫'''</small></div>
; {{nihongo|魔乳千房|魔乳 千房（まにゅう ちふさ）|聲：[[喜多村英梨|喜多村英梨]]／[[壽美菜子|壽美菜子]]}}
:胸則的小女兒，擁有魔乳、胸杜兩族血統，魔乳一族的繼承者，特殊能力是斬乳後將其吸收。小時候第一次與狹山櫻花比試，無意中奪走了櫻花的乳房。
; {{nihongo|楓|楓（かえで）|聲：[[廣橋涼|廣橋涼]]／[[豐崎愛生|豐崎愛生]]}}
:千房的青梅竹馬，天真爛漫笨拙女孩。為了跟隨千房而在村中被行刑乳斬。
; {{nihongo|魔乳影房|魔乳 影房（まにゅう かげふさ）|聲：[[冬馬由美|冬馬由美]]／[[水原薰|水原薰]]}}
:千房的異母姊姊，胸則的次女。残忍的性格。在與千房戰鬥中被奪取乳房。
; {{nihongo|狹山櫻花|狭山 桜花（さやま おうか）|聲：－／[[能登麻美子|能登麻美子]]}}
:魔乳一族的分家。擁有優秀的劍術，與千房比試被奪取乳房，從此成為貧乳，之後受父母疏遠，被人看不起。之後成為魔乳胸幸的手下。
; {{nihongo|魔乳胸則|魔乳 胸則（まにゅう むねのり）|聲：[[柴田秀勝|柴田秀勝]]／[[斧敦志|斧敦志]]}}
:魔乳一族當家，即使年事已高，還是擁有最強大的劍技。
; {{nihongo|魔乳胸幸|魔乳 胸幸（まにゅう むねゆき）|聲：－／[[濱田賢二|濱田賢二]]}}
:胸則的長男，魔乳本家的後繼者。在小時候，已被看好是位優秀劍士，但後來手受傷。乳房鑑定家，但實際卻不喜歡這工作。
; {{nihongo|魔乳胸華|魔乳 胸華（まにゅう きょうか）|聲：－／[[高垣彩陽|高垣彩陽]]}}
:胸則的長女，千房與胸幸的異母姊姊。精通豐胸術。宮殿內的權力鬥爭也有重大影響。
; {{nihongo|胸杜千組|胸杜 千組（むなもり ちくみ）|聲：[[鶴弘美|鶴弘美]]／[[東條加那子|東條加那子]]}}
:千房的母親。胸杜一族因戰敗而寄身於魔乳之里，身為一族之長被胸則納為側室，並逼迫傳授千房「流乳操」，想藉此取得該族相傳秘技，但千組不從選擇自我了斷。
; {{nihongo|小影|小影（こかげ）|聲：－／[[福圓美里|福圓美里]]}}
:影房的女傭。無辜的性格。
; {{nihongo|三重鳩宗|三重 鳩宗（みえ はとむね）|聲：[[長_(日本配音員)|長]]／[[大川透|大川透]]}}
:三重藩領主。喜歡巨乳且不遮掩，除去這一點，做事公正且劍術高超，常常會微服出巡，受人民愛戴。

== 用語 ==
;魔乳流
:在幕府具有影響力的魔乳一族所屬流派，根據地{{nihongo|魔乳之里|魔乳の里}}。魔乳流劍技可不見血地一刀乳斷。

;{{nihongo|流乳操|乳流れ}}
:胸杜一族相傳秘技。其技如移動容器內的水，自在地操縱乳房；唯有憑藉血之傳承，方能獲取力量習得此技。
:雖然母親自殺而未傳授，千房靠著自身天賦學得其技的奪乳，但還不足以自在操縱給予。

;{{nihongo|乳研磨|<ruby><rb>乳研ぎ</rb><rp>（</rp><rt>ちちとぎ</rt><rp>）</rp></ruby>}}
:利用溶入糖跟葛粉的特製白湯，透過祕傳的指壓法、仔細按摩乳房的美乳鍛鍊法。
