{{vgsname|魔兽系列|en=Warcraft}}是由[[暴雪娱乐|暴雪娱乐]]所发布的一整套包括[[長篇小說|長篇小說]]、[[电子游戏|电子游戏]]及基于其他媒介的[[跨媒體製作|跨媒体]]系列作品。该系列由四款核心游戏：[[魔兽争霸：人类与兽人|魔兽争霸：人类与兽人]]、[[魔兽争霸II：黑潮|魔兽争霸II：黑潮]]、[[魔兽争霸III：混乱之治|魔兽争霸III：混乱之治]]、[[魔兽世界|魔兽世界]]及一款卡牌游戏[[炉石传说：魔兽英雄传|炉石传说：魔兽英雄传]]组成。前三款都是[[即时战略游戏|即时战略]]类游戏，对阵双方的玩家需要控制一整支虚拟队伍进行对战，是史上最成功的即時戰略遊戲系列之一。而后的一款是[[MMORPG|MMORPG]]游戏。除魔兽争霸：人类与兽人之外的其他游戏版本都有资料片，资料片中通常包含了新的内容以延长游戏寿命。魔兽争霸II的资料片是[[魔兽争霸II：黑暗之门|魔兽争霸II：黑暗之门]]，魔兽争霸III的资料片是[[魔兽争霸III：冰封王座|魔兽争霸III：冰封王座]]，魔兽世界的资料片有[[魔兽世界：燃烧的远征|燃烧的远征]]、[[魔獸世界：巫妖王之怒|巫妖王之怒]]、[[魔獸世界：浩劫與重生|大地的裂变]]、[[魔兽世界：熊猫人之谜|熊猫人之谜]]、[[魔兽世界：德拉诺之王|德拉諾之霸]]及[[魔兽世界：军团再临|軍臨天下]]。

玩家最初先进入《[[魔兽争霸:_兽人与人类|魔兽争霸: 兽人与人类]]》的[[艾泽拉斯|艾泽拉斯]]。在这个世界中存在着一个名为“[[扭曲虚空|扭曲虚空]]”的地方。在此魔兽系列的历史里，荒芜的世界，魔法门被打开了。众所周知，世界上包含[[艾泽拉斯|艾泽拉斯]]，[[德拉诺|德拉诺]]，[[阿古斯|阿古斯]]、[[卡瑞许|卡瑞许]]以及克索诺斯({{lang|en|Xoroth}})。

魔兽系列的前三款游戏及其资料片都是在[[个人电脑|PC]]和[[Mac_OS|Mac OS]]双平台发布的。这三款都是[[RTS|RTS]]游戏，每一部的剧情都承接上一部，而且会推出新功能以提升游戏可玩性。[[魔兽争霸III：混乱之治|魔兽争霸III]]是系列中第一部开始有[[限定版|典藏版]]的版本，之后的所有续作都有典藏版。[[魔兽争霸II：黑潮|魔兽争霸II]]是系列中第一部支持在[[战网|战网]]上游戏的版本，虽然是游戏发布的几年之后才推出支持战网的版本。魔兽争霸II也是系列中第一部以“Battle合集”（Battle Chest）形式重新发售的版本，合集是包含游戏原版和资料片的捆绑版。魔兽争霸III和[[魔兽世界|魔兽世界]]在初版发布之后也都有相应的“Battle合集”。魔兽争霸II的“战网”版同时也是系列中率先引入[[CD_key|CD key]]验证的版本，用户必须购买游戏拥有自己的CD key才能连接到战网进行游戏。不过[[局域网|局域网]]对战并不需要CD key，虽然当时的猜测是以后的游戏都会如此，[[暴雪娱乐|暴雪娱乐]]的另一款系列游戏[[星海爭霸II：自由之翼|星际争霸II]]中完全取消了局域网对战功能，只能通过战网进行游戏。<ref>{{cite web|url=http://uk.pc.ign.com/articles/999/999171p1.html|title=No LAN for Starcraft II|last=Haynes|first=Jeff|publisher=IGN|accessdate=2009-10-27}}</ref>

1998年，系列中原定于1997年发布的一款[[冒险游戏|冒险游戏]]，{{link-en|魔兽争霸：氏族之王|Warcraft Adventures: Lord of the Clans}}<ref>{{cite web|author=[[暴雪娱乐|暴雪娱乐]]|title=视频: 魔兽冒险游戏氏族之王宣传动画|url=http://v.youku.com/v_show/id_XNTIxMjA1ODA=.html|publisher=[[优酷|优酷]]|accessdate=2013-11-05}}</ref>宣布取消。 <ref>{{cite web|author=于子将|title=外媒曝《魔兽争霸 氏族之王》流产内幕|url=http://games.qq.com/a/20110525/000014.htm|publisher=[[腾讯|腾讯]]游戏|accessdate=2013-11-05|date=2011年5月25日}}</ref>

2004年，暴雪娱乐改变了系列一贯的即时战略游戏类型，发布了一款[[MMORPG|MMORPG]]游戏[[魔兽世界|魔兽世界]]。需要持续付费才能玩，同时会定期地以补丁的形式为系列增加新的内容。魔兽世界很快便在世界范围内普及，在2008年的[[金氏世界紀錄|金氏世界紀錄]]中为最多付费用戶的MMORPG (大型多人在线角色扮演游戏)<ref>https://web.archive.org/web/20080405020338/http://gamers.guinnessworldrecords.com/records/pc_gaming.aspx</ref>。2010年的巔峰期在全球范围内拥有1200万付费用戶。<ref>http://techland.time.com/2013/05/09/the-inexorable-decline-of-world-of-warcraft/</ref>截止到2012年魔兽世界已经有了四部资料片。在[[星海爭霸II：自由之翼|星海爭霸II：自由之翼]]发布期间，[[暴雪娱乐|暴雪]]联合创始人Frank Pearce说：“如果现在有团队有制作另一个魔兽争霸RTS游戏的话，我们也会考虑。目前我们没有从事相关工作，我们有开发[[魔兽世界：大地的裂变|大地的裂变]]、[[星海爭霸II：自由之翼|星海爭霸II：自由之翼]]、[[暗黑破壞神III|暗黑破壞神III]]的团队，当他们的项目结束后他们会立即讨论接下来该做什么。”

2012年，暴雪娱乐宣布了名为《[[炉石传说：魔兽英雄传|炉石传说：魔兽英雄传]]》的免费[[交換卡片遊戲|交換卡片遊戲]]，因为要在年底发布，游戏的封测工作于2013年夏季开始。<ref name="WoW Insider">{{cite web|title=What we know so far about HearthStone: Heroes of Warcraft|url=http://wow.joystiq.com/2013/03/22/hearthstone-what-we-know-so-far/|publisher=WoW Insider|accessdate=22 March 2013}}</ref>

==设定==

===世界===
主条目：[[魔兽系列地名列表|魔兽系列地名列表]]

魔兽系列的大部分故事发生在[[魔兽系列地名列表|艾泽拉斯]]星球。魔兽系列中已知的其他星球有[[魔兽系列地名列表|德拉诺]]（及其破碎遗迹，称为[[魔兽系列地名列表|外域]]）、[[魔兽系列地名列表|阿古斯]]、卡瑞许以及克索诺斯。另外还有提到几个抽象的空间，包括翡翠梦境、元素位面、扭曲虚空和无尽黑暗。系列的第一款游戏魔兽争霸：人类与兽人即发生在艾泽拉斯的人类王国。
=== 主要种族和阵营 ===
{{Main|魔兽世界种族列表}}
魔兽系列的主要种族只有[[人类|人类]]和[[兽人_(魔兽世界)|兽人]]，到了魔兽争霸II时期，增加了[[高等精灵|高等精灵]]、[[矮人|矮人]]、[[巨魔|巨魔]]、[[食人魔|食人魔]]、[[地精|地精]]和[[龙|龙]]。在魔兽争霸III中，包括新增加的[[暗夜精灵_(魔兽世界)|暗夜精灵]]和[[牛头人_(魔兽世界)|牛頭人]]，以及燃烧军团的成员[[亡灵|亡灵]]、[[恶魔|恶魔]]等，4个可以由玩家操纵的种族势力被确立起来。其中包括：
# 人类联盟，由人类，矮人，（高等）精灵组成。在资料片《冰封王座》中，由高等精灵的遗民组成的血精灵，也加入了人类联盟，但是在《冰封王座》的战役中，血精灵的领袖凯尔王子最终叛变，脱离了人类联盟。
# 兽人部落，由兽人，巨魔和牛头人族组成。
# 暗夜精灵，由暗夜精灵哨兵和德鲁伊组成。
# 不死族，这个势力有较多的组成成分。从恶魔（恐惧魔王），被恶魔禁锢和扭曲的傀儡（[[巫妖|巫妖]]，[[女妖|女妖]]），到被恶魔赋予永恒生命的堕落领袖（[[死亡骑士|死亡骑士]]），以及信奉邪恶的各其他种族成员（侍僧，男巫），直到生活于地下的邪恶生物（地穴领主，地穴编织者）和单纯被制造或者召唤出来的尸体（[[食尸鬼|食尸鬼]]，[[憎恶|憎恶]]），和由阿薩斯使用藍龍的骨頭和肌腱變出來的冰霜巨龍，这样一个复杂的组成保证了在《魔兽争霸III》的故事主线中，不死族的无坚不摧。

然而，若包括在魔兽争霸III战役中出现的种族，这个数字可以增加到7个（包括高等精灵，[[纳迦|纳迦]]和血精灵倖存者）.

在[[大型多人在线角色扮演游戏|大型多人在线角色扮演游戏]]－《[[魔兽世界|魔兽世界]]》中，共分为两大敌对阵营，分别为联盟与部落。联盟方由人类、矮人、暗夜精灵、侏儒组成；而部落则包含兽人、[[不死族|不死族]]、牛头人与巨魔四种族。在《[[燃烧的远征|燃烧的远征]]》资料片中，联盟方增加了德莱尼种族，而部落方增加了血精灵种族,《[[魔獸世界：浩劫與重生|浩劫與重生]]》资料片中，联盟方增加了狼人种族，而部落方增加了地精种族。在《[[魔兽世界：熊猫人之谜|魔兽世界：熊猫人之谜]]》版本添加了不单属于联盟或部落的熊猫人，该种族可自选联盟或部落。在该游戏中，其余种族均作为游戏种族。

== 電子競技 ==
魔獸爭霸III是電子競技的主要遊戲項目之一。最初的比賽採用魔獸爭霸III：混亂之治，當資料片發佈後比賽項目改用資料片。為確保競技中的平衡性，暴雪不斷發佈補丁更新遊戲單位的數據，最新版本為魔獸爭霸III：冰封王座1.30。（2018-8-10）
