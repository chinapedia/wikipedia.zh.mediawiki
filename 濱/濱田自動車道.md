'''濱田自動車道'''（{{jpn|j=浜田自動車道|hg=はまだじどうしゃどう|rm=Hamada jidōshadō|f=y}}）是從[[廣島縣|廣島縣]][[北廣島町|北廣島町]]到[[島根縣|島根縣]][[濱田市|濱田市]]的[[日本高速公路|高速公路]]（[[高速自動車國道|高速自動車國道]]）。略稱為'''濱田道'''（{{lang|ja|浜田道／はまだどう}}、HAMADA EXPWY）。本道路同時是[[中國橫斷自動車道|中國橫斷自動車道廣島濱田線]][[千代田系統交流道|千代田系統交流道]]以北的路段與[[山陰自動車道|山陰自動車道]][[濱田系統交流道|濱田系統交流道]] - [[濱田交流道|濱田交流道]]的道路名。高速公路編號與[[廣島自動車道|廣島自動車道]]、[[中國自動車道|中國自動車道]]（廣島北系統交流道 - 千代田系統交流道間與 {{Ja Exp Route Sign|E2A}} 共線）共同編為 {{Ja Exp Route Sign|E74}}。

== 交流道 ==
* 交流道編號欄背景色<span style="color:#BFB">■</span>區間為已通車區間。設施欄背景色<span style="color:#CCC">■</span>區間是未通車區間或未啟用設施，未通車區間名稱全部為暫稱。
* (數字)為其他路線的編號。<數字>為預定編號。
* [[智慧型交流道|智慧型交流道]]背景色為<span style="color:#eda5ff">■</span>。
* 巴士站欄位，○表示為運用中、◆表示為休息中，若無標記則未設置巴士站。

=== 服務區、停車區 ===
本道路並未設置[[公路服務區|服務區]] 。

另外，僅寒曳山停車區上行線有設置商店營業，其他停車區則設有[[自動販賣機|自動販賣機]]（輕食（寒曳山停車區上下線）、[[飲料|飲料]]）銷售商品。全線未設置[[加油站|加油站]]。

== 歷史 ==
* [[1989年|1989年]][[10月18日|10月18日]] : 旭交流道 - 濱田交流道通車。
* [[1991年|1991年]][[12月7日|12月7日]] : 千代田系統交流道 - 旭交流道通車，同時'''全線通車'''。
* [[2003年|2003年]][[9月21日|9月21日]] : 濱田系統交流道通車，與山陰道（江津道路）連接。
* [[2004年|2004年]][[12月18日|12月18日]] : 金城停車區智慧型交流道的社會實驗開始（～[[2005年|2005年]][[3月15日|3月15日]]）。
* [[2007年|2007年]][[4月1日|4月1日]] : 金城停車區智慧型交流道供用開始。
* [[2010年|2010年]][[2月2日|2月2日]] : 全線被指定為免費化社會實驗的對象區間。
