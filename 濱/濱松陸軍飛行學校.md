'''濱松陸軍飛行學校'''（[[日語|日語]]：はままつりくぐんひこうがっこう）為以前位於[[日本|日本]][[静岡縣|静岡縣]][[濱松市|濱松市]]之[[大日本帝國陸軍|大日本帝國陸軍]]飛航訓練[[军校|教育機関]]。

== 概要 ==
1933年（[[昭和|昭和]]8年）在[[陸軍航空本部|陸軍航空本部]]長隷下學校濱松地方開校（濱松陸軍飛行學校令、昭和8年8月1日軍令第10號)。

== 所在地 ==
現：[[静岡縣|静岡縣]][[濱松市|濱松市]][[西區_(濱松市)|西區]]西山町（[[濱松基地|航空自衛隊濱松基地]]）

== 教育要綱 ==
:教育對象
::航空兵科現役下士官・幹部候補生・下士官候補者
:教育區分 
::'''操縱學徒'''
::：航空兵科現役下士官志願人員入學考試合格者（來自1937年10月[[東京陸軍航空學校|東京陸軍航空學校]]畢業生）

::'''戰術學生'''
:::為從事航空兵科幹部候補生及下士官候補者飛行機操縱之訓練

== 學校沿革 ==
*1933年（昭和8年）從立川移駐作為[[飛行第7戰隊_(日本陸軍)|飛行第7戰隊]]練習部之基幹、獨立為[[陸軍航空本部|陸軍航空本部]]長隷下學校
*1935年（昭和10年）設置戰術學生
*1938年（昭和13年）伴隨[[陸軍航空總監部|陸軍航空總監部]]的設置移行為航空總監隷下學校
*1944年（昭和19年）改編為[[濱松教導飛行師團|濱松教導飛行師團]]
