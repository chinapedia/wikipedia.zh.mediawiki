《'''ayumi hamasaki COUNTDOWN LIVE 2014-2015 A Cirque de Minuit ～真夜中のサーカス～'''》（濱崎步2014-2015跨年演唱會A深夜中的馬戲團）是日本歌手[[濱崎步|濱崎步]]於2014年12月29日、30日和31日在[[國立代代木競技場|國立代代木競技場]]第一體育館舉行的跨年演唱會，於2015年4月8日發行DVD及Blu-ray。

== 說明 ==
*本作與濱崎步第16張原創專輯《[[A_ONE_(濱崎步專輯)|A ONE]]》同時發行，發行日當天2015年4月8日即為濱崎步出道17周年的紀念日。<ref>{{Cite web |url = http://ent.163.com/15/0317/10/AKTE3OMP00031H0O.html |title = 滨崎步与林俊杰合作 新专辑《A ONE》4月发售 |date = 2015-03-17 |publisher = 网易娱乐 |language = zh-cn |accessdate = 2015-03-17 }}</ref>

*標題中Cirque de Minuit的法文原意是「午夜的馬戲團」。

*完整收入年末在聖地代代木第一競技場舉行的「午夜馬戲團」風格的跨年演出，包含名曲「[[Duty|Duty]]」和上年12月发行的新曲「[[Zutto.../Last_minute/Walk|Zutto...]]」「[[Zutto.../Last_minute/Walk|Last minute]]」「[[Zutto.../Last_minute/Walk|Walk]]」，還有翻唱自[[宇多田光|宇多田光]]的名曲「[[Movin'_on_without_you|Movin' on without you]]」的首次live披露<ref>{{Cite web |url = http://avex.jp/ayu/news/detail.php?id=1019007 |title = 浜崎あゆみ 16thアルバムとカウントダウンライヴDVD＆Blu-rayをデビュー17周年記念日に同時発売！ |date = 2015-02-13 |publisher = ayumi hamasaki(浜崎あゆみ) offical website |language = ja |accessdate = 2015-02-13 }}</ref>，更收錄演唱會三天的幕後花絮！

*本作另以數量限定生產豪華BOX的形式與《[[A_ONE_(濱崎步專輯)|A ONE]]》搭配販售（详情见[[A_ONE_(濱崎步專輯)#.E7.99.BC.E5.94.AE.E5.9E.8B.E6.85.8B|A ONE 发售型态]]）。
