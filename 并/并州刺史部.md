'''并州刺史部'''，是[[漢朝|漢朝]]十三州刺史部之一，州治在晉陽縣（今[[山西省|山西省]][[太原市|太原市]]西南）。[[西漢|西漢]]州域範圍大致是今日的[[山西省|山西省]]大部，[[東漢|東漢]]時擴大至[[陝西省|陝西省]]北部及[[內蒙古自治區|內蒙古自治區]]中部一帶。

== 名稱由來 ==
「并州」一詞的出現始於春秋以後，後人在《[[尚書|尚書]]·禹貢》九州基礎上，增加幽、-{并}-、營3州，大體指[[常山郡|常山郡]]以北的區域。《[[漢書|漢書]]·地理志》載：「正北曰并州。：其山曰恒山，藪曰昭餘祁，川曰虖池、嘔夷，寖曰淶、易；其利布帛；民二男三女；畜宜五擾，穀宜五種。」

== 西漢并州刺史部 ==
西漢時，領[[太原郡|太原郡]]、[[代郡|代郡]]、[[上黨郡|上黨郡]]、[[雲中郡|雲中郡]]、[[雁門郡|雁門郡]]、[[定襄郡|定襄郡]]6個郡國。

== 東漢并州刺史部 ==
東漢時，領太原郡、上黨郡、[[上郡|上郡]]、[[西河郡|西河郡]]、[[五原郡|五原郡]]、雲中郡、雁門郡、定襄郡、[[朔方郡|朔方郡]]9個郡國，東漢末置[[新興郡|新興郡]]、[[樂平郡|樂平郡]]2郡。
