'''柏原市'''（{{jpn|j=柏原市|hg=かしわらし|rm=Kashiwara shi|f=y}}）是位於日本[[大阪府|大阪府]]東部的城市，主要市區位於轄區西部[[大和川|大和川]]和{{Tsl|ja|石川 (大阪府)|石川 (大阪府)|石川}}匯流處附近的平原區域，東部約三分之二區域為包括{{tsl|ja|生駒山地}}和{{Tsl|ja|金剛山地}}的山區。由於主要市區距離大阪市市區約20公里，因此也發展成大阪市的[[臥城|臥城]]。<ref name="Brief">{{Cite web| url=http://www.city.kashiwara.osaka.jp/docs/2014012900324/| title=柏原市の見どころ| date=2015-04-10| publisher=柏原市政府| accessdate=2018-06-18| language=ja}}</ref>

由於本市和鄰近的[[奈良縣|奈良縣]][[橿原市|橿原市]]的日文發音為非常相近的「Kashiwara」和「Kashihara」，而奈良縣[[御所市|御所市]]也有個名為「柏原」且日文發音相同的地區，這些區域因此經常造成混淆。

主要農產為[[葡萄|葡萄]]，也因此本地有釀[[葡萄酒|葡萄酒]]產業，通常以「河內葡萄酒」或「柏原葡萄酒」的名義在銷售。

== 歷史 ==
[[File:Yamato_River_and_Ishikawa_River_at_Kashiwara01.JPG|thumb]]

在令制國時代分屬於[[河內國|河內國]][[志紀郡|志紀郡]]、[[安宿部郡|安宿部郡]]、[[大縣郡|大縣郡]]三郡，轄區內亦有多所4世紀至7世紀期的[[古墳|古墳]]。過去也由於位於[[大和國|大和國]]利用[[大和川|大和川]]連結[[難波|難波]]的要道而繁榮。

在[[江戶時代|江戶時代]]，此區域多屬[[小田原藩|小田原藩]]和幕府的直屬領地，19世紀末[[明治維新|明治維新]]後，這些區域曾先後隸屬大坂裁判所司農局、大阪府南司農局、[[河內縣_(日本)|河內縣]]、[[堺縣|堺縣]]，直到1881年堺縣被併入[[大阪府|大阪府]]後，此區域也隨之成為大阪府的轄區。<ref>{{Cite web| url=http://www.city.kashiwara.osaka.jp/docs/2018061000028/| title=柏原と明治維新2| date=2018-06-17| publisher=柏原市政府| accessdate=2018-06-18| language=ja}}</ref>

1889年日本實施[[町村制|町村制]]，現在的柏原市轄區在當時分屬'''柏原村'''、{{tsl|ja|国分_(柏原市)|國分_(柏原市)|國分村}}、{{Tsl|ja|玉手|玉手|玉手村}}、{{tsl|ja|堅下|堅下|堅下村}}、{{tsl|ja|堅上|堅上|堅上村}}五個行政區劃。位於北部的柏原村先於1915年升格改制為'''柏原町'''，1939年再與位於西北部山區的堅下村和堅上村合併為新設立的柏原町；而位於東南部的國分村和位於西南部的玉手村則在1931年合併為新設立的國分村，再於1941年升格改制為國分町。柏原町和國分町在1956年進一步合併為新設立的柏原町，1958年再升格為現在的'''柏原市'''。<ref name="Brief"/>

== 觀光資源 ==

1615年[[大阪夏之陣|大阪夏之陣]]曾在本地發生[[道明寺之戰|道明寺之戰]]，此戰中[[後藤又兵衛|後藤又兵衛]]布陣於轄內的{{Tsl|ja|柏原市立玉手山公園|柏原市立玉手山公園|玉手山}}，最終也在此戰死，也因此在此區域有多處與當時戰役有觀之供養碑。<ref>{{Cite web| url=http://www.city.kashiwara.osaka.jp/_files/00058735/natunozin_map.pdf| title=大坂夏の陣を歩く| publisher=柏原市政府| accessdate=2018-06-18| language=ja}}</ref>

== 教育 ==

[[File:Kansai_Women's_College01.JPG|thumb]]
[[File:Oosakakyoiku.jpg|thumb]]

轄內的{{tsl|ja|関西福祉科学大学高等学校|關西福祉科學大學高等學校}}前身為成立於1942年玉手山女學校，玉手山女學校在1948年隨著{{tsl|ja|学制改革|學制改革}}改設為「玉手山高等學校」，1965年其營運單位學校法人玉手山學園新設立[[关西女子短期大学|关西女子短期大学]]，1970年新設立{{tsl|ja|関西医療技術専門学校|關西醫療技術專門學校}}，1997年進一步新設立{{tsl|ja|関西福祉科学大学|關西福祉科學大學}}，高等學校也隨之更名為現在的名稱。

[[大阪教育大學|大阪教育大學]]則於1992年將校本部遷移至此。
