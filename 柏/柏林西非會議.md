'''柏林西非会议'''（[[英语|英语]]：Berlin Conference，意指柏林会议；[[德语|德语]]：Kongokonferenz，意指刚果会议）由[[欧洲|欧洲]]强国在1884年至1885年与[[德国|德国]][[柏林|柏林]]举行，以准备他们在[[非洲|非洲]]建立[[殖民地|殖民地]]与发展贸易。与会国最后达成了'''柏林会议总议定书'''，正式开始[[瓜分非洲|瓜分非洲]]。

==缘由==
1880年代，欧洲国家对非洲有了更大的兴趣。[[亨利·莫頓·史丹利|亨利·莫頓·史丹利]]于1874至1877年发现了[[刚果河|刚果河]]盆地，终于找到非洲最后的“未知的地域”（[[拉丁语|拉丁语]]：terra incognita）。[[比利时|比利时]]国王[[利奥波德二世_(比利时)|利奥波德二世]]于1876年成立了[[国际非洲协会|国际非洲协会]]（[[法语|法语]]：Association Internationale Africaine）。得知史丹利的成功后，利奥波德二世邀请他参与协会的工作。该协会的宗旨是探索非洲与为非洲人民带来“文明”。1878年，另一类似组织[[国际刚果协会|国际刚果协会]]成立，虽与非洲协会关系密切，但比较注重经济发展之工作。比利时国王秘密地买通了刚果协会的投资者，把刚果协会变成追求[[帝国主义|帝国主义]]的组织，而表面上则以看似支持[[慈善|慈善]]事业的非洲协会作掩饰。

由1879年到1884年，史丹利再回到刚果，但不是为求探索领地，而是作为利奥波德二世的探子，谋求在刚果建立一个国家。那边厢，[[法国|法国]]的海军军官[[皮埃尔·萨沃尼昂·德-布拉柴|皮埃尔·萨沃尼昂·德-布拉柴]]到达刚果盆地西岸，并于1881年在[[布拉柴维尔|布拉柴维尔]]（今[[刚果共和国|刚果共和国]]首都）建立了法属殖民地。

而且，[[葡萄牙|葡萄牙]]由于曾经与刚果帝国签署过条约，所以在1884年2月26日与[[英国|英国]]签署条约，企图阻止刚果协会扩张到[[大西洋|大西洋]]的海岸线。

除了以上国家，其他欧陆国家也想在非洲建立殖民地。法国在1881年占领了[[突尼西亚|突尼西亚]]和今刚果共和国的领土，又在1884年得到[[几内亚|几内亚]]。1882年，英国占领名义上属于[[奥斯曼帝国|奥斯曼帝国]]的[[埃及|埃及]]，并因此控制了[[苏丹共和国|苏丹]]与部分的[[索马利亚|索马利亚]]。在1870年与1882年，[[意大利|意大利]]夺得[[厄立特里亚|厄立特里亚]]。刚刚崛起的[[德国|德国]]也在1884年把[[多哥|多哥]]、[[喀麦隆|喀麦隆]]与[[納米比亞|西南非]]（今[[纳米比亚|纳米比亚]]）纳入为他的[[保护国|保护国]]。

== 会议 ==
利奥波德二世成功就有关非洲贸易之事，说服法国与德国。在葡萄牙的倡导下，[[德国总理|德国首相]][[奥托·冯·俾斯麦|奥托·冯·俾斯麦]]于1884年邀请了[[奥匈帝国|奥匈帝国]]、[[比利时|比利时]]、[[丹麦|丹麦]]、[[俄罗斯|俄罗斯]]、[[法国|法国]]、[[荷兰|荷兰]]、[[葡萄牙|葡萄牙]]、[[瑞典|瑞典]]与[[挪威|挪威]]（挪威在1905年前以[[共主邦联|共主邦联]]之关系属于瑞典）、[[西班牙|西班牙]]、[[意大利|意大利]]、[[英国|英国]]、[[美国|美国]]与[[奥斯曼帝国|奥斯曼帝国]] 十五国代表参加柏林会议，以达成共识。

== 总议定书 ==
议定书定下了以下的条款：
* 刚果自由邦正式被定为国际刚果协会的私有财产。于是，二百万平方公里的刚果领土（今[[刚果民主共和国|刚果民主共和国]]之全境）被划为利奥波德二世的私人财产。
* 全体十四个与会国将于刚果盆地全境、[[马拉维湖|马拉维湖]]及其以东邻近地区享有自由贸易之权利；
* [[尼日尔河|尼日尔河]]与[[刚果河|刚果河]]将对船只自由开放；
* 与会国签署禁止贩卖[[奴隶|奴隶]]；
* 根据“有效原则”，与会国只有真正控制某殖民地时，才可以拥有它们。
* 如若任何国家在将来想拥有非洲海岸的任何部分，都必须通知其他签署国或是订立[[保护国|保护国]]。

值得注意的是，柏林协定书的行文中，第一次提及有关与会国之[[势力范围|势力范围]]的国际协定。

== 结果 ==
[[File:FeizhouZhimindi.png|frame]]
会议过后，欧洲列强加速她们瓜分非洲的行动。在几年之内，名义上[[撒哈拉沙漠|撒哈拉沙漠]]以南的非洲领土全被瓜分。1895年时，只有[[利比里亚|利比里亚]]、[[奥兰治自由邦|奥兰治自由邦]]与[[德兰士瓦|德兰士瓦]]仍为[[独立|独立]]国家。[[阿比西尼亚|阿比西尼亚]]（今衣索比亞）于1889年至1896年成功抵抗[[意大利|意大利]]自[[厄立特里亚|厄立特里亚]]而来的侵略，史称[[第一次意阿战争|第一次意阿战争]]，是非洲唯一的独立原居民国家。1902年前，非洲九成以上的领土都被欧陆国家控制。撒哈拉沙漠之土地，大多为法国所有。英国在镇压了[[穆罕默德·艾哈邁德·馬赫迪|穆罕默德·艾哈邁德·馬赫迪]]和与法国解决[[法绍达事件|法绍达事件]]后，与埃及共同统治苏丹。

英国发动了两次[[布尔战争|布尔战争]]，征服了布尔人的国家。1911年，法国与西班牙瓜分了[[摩洛哥|摩洛哥]]。1912年，意大利从[[土耳其|土耳其]]夺得[[利比亚|利比亚]]。1914年，英国正式占领埃及全境，列强瓜分非洲结束。在这个时候，除了利比里亚与埃塞俄比亚之外，其他非洲土地全归欧洲国家所有。

另外列强在会议上罔顾非洲各民族的实际分布，采取直接在[[地图|地图]]上划分势力范围的做法，造成了现今不少非洲国家之间的[[国界|国界]]异常平直，也因此为这些国家遗留了不少影响至今的[[部族|部族]]纷争。
