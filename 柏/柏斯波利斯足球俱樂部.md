== 歷史 ==
=== 成立早年 (1963–1969) ===
柏斯波利斯於1963年由來自[[美國|美國]]的[[拳擊|拳擊]]冠軍選手{{link-en|阿里·艾布杜|Ali Abdo}}成立。當時，球隊只是一隊微不足道的球隊，而且只能於乙組的全國聯賽中比賽。當[[沙恩足球俱樂部|沙恩]]宣佈解散後，大量沙恩的優秀球員轉投柏斯波利斯，球隊於是開始強大起來。1968年，由於全國聯賽並沒有依時舉辦，一個由44隊組成的聯盟式比賽暫時取代聯賽，柏斯波利斯意外地成為聯盟冠軍。並且得到參加1969年於[[泰國|泰國]]舉行的[[亞足聯冠軍聯賽|亞洲冠軍球會盃]]，是球會首次參與亞洲賽，但是球隊於小組賽出局，無緣準決賽<small><ref>{{cite web |language=en |url=http://www.rsssf.com/tablesa/ascup68.html#cc |title=Champions' Cup 1969 |publisher=RSSSF |date=2002-08-15 |accessdate=2009-05-21}}</ref></small>。

=== 塔赫·詹姆希德盃 (1969–1979) ===

=== 在困難之下成功 (1979–1990) ===
在[[伊朗伊斯蘭革命|伊朗伊斯蘭革命]]發生後，球會班主艾布杜離開[[伊朗|伊朗]]，並回到[[美國|美國]]。期間，雖然球隊贏得了[[薩義德·艾士潘迪盃|薩義德·艾士潘迪盃]]，但很多球員都離開了球會。<ref name="Bowling">{{cite web
|title= {{fa icon}} سه سال بحرانی در پرسپولیس:سرگذشت تبعید از بولینگ به داوودیه
|url=http://www.etemaad.com/Released/86-07-05/225.htm
|publisher=Etemaad
|accessdate=
}}</ref>另外，球會的資產暫時交由「壓迫者和退伍軍人基金會」([[波斯語|波斯語]]: '''بنیاد مستضعفان و جانبازان''' ) 打理，並把球會的營運權交予「伊朗體育部」。<ref name="Bowling"/> 

1981年，體育部宣佈將會更改球會的名稱，但球會職員、球員及球迷均強烈反對。 
<ref name="Pirouzi">{{cite web|title={{fa icon}} پرسپولیس چگونه پیروزی شد؟|url=http://www.iran-varzeshi.com/1386/860517/html/footbal5.htm|publisher=Iran Varzeshi|accessdate=|deadurl=yes|archiveurl=https://web.archive.org/web/20080702043849/http://www.iran-varzeshi.com/1386/860517/html/footbal5.htm|archivedate=2008-07-02}}</ref>而且，球隊於一場省聯賽關鍵之役罷賽，以示其反對的決心。最終球隊根據條例被罰3分，無緣冠軍。<ref name="Pirouzi"/>1986年，「壓迫者和退伍軍人基金會」重新接掌球會所有權力後，宣佈改名為'''阿薩迪''' ([[波斯語|波斯語]]: '''آزادی'''，意即自由)。<ref name="Pirouzi"/> 球員於是再次宣佈如果球會更改名字，將會再次罷賽。<ref name="Pirouzi"/>後來，基金會決定放棄球會，並把球會一切重新交回體育部手上。1987年2月16日，得到球員的同意後，體育部正式把球會的名字改為'''皮魯茲''' ([[波斯語|波斯語]]: '''پیروزی'''，意即勝利)。<ref name="Pirouzi"/>話雖如此，但所有球迷均沿用其舊稱，柏斯波利斯。<ref name="Pirouzi"/>

由於1980年代伊朗政治不穩，球會只能於省級的比賽中競賽。在球員兼領隊柏文的帶領下，球會奪得6屆德黑蘭省聯賽冠軍（其中1984-89年獲五連冠）及德黑蘭哈茲菲盃冠軍，成績驕人。這時的柏斯波利斯是德黑蘭的一級強隊，宿敵[[艾斯迪格拿足球俱樂部|艾斯迪格拿]]則相對顯得暗淡無光。

=== 復興 (1990–2001) ===
1990年代是柏斯波利斯的一個輝煌年代，球隊總共奪得了4屆聯賽冠軍及兩屆[[哈茲菲盃|哈茲菲盃]]冠軍。加上有大批優良球員湧現，讓所有柏斯波利斯的球迷看到了1960年代及1970年代的那支夢幻球隊的影子。1991年，球隊更成為首屆[[亞洲盃賽冠軍盃|亞洲盃賽冠軍盃]]冠軍，柏斯波利斯在兩回合制的決賽以1-0擊敗來自[[巴林|巴林]]的球隊[[穆哈拉克足球俱樂部|穆哈拉克]]。這亦是球隊在洲際比賽的第一個冠軍。

球隊贏得1995-96年球季的聯賽冠軍後，國內[[足球|足球]]迷重新把注意力放回柏斯波利斯身上。這季球會原本以10分落後領頭羊，但球隊憑著永不放棄的精神反敗為勝，奪回失落多年的聯賽冠軍。在下一個球季，球隊更成功衛冕[[伊朗甲組足球聯賽|伊朗甲組足球聯賽]]，領先第二名的球隊6分奪冠。在亞洲賽場上，球隊亦打入了[[亞洲俱樂部錦標賽|亞洲俱樂部錦標賽]]準決賽，但被[[韓國|韓國]]球隊[[浦項製鐵|浦項製鐵]]淘汰，最後只能獲得第三名。

在1997-98年球季，柏斯波利斯繼續向國內戰場及亞洲賽場的冠軍進發。但由於[[伊朗足球協會|伊朗足球協會]]與[[亞洲足協|亞洲足協]]未能就賽程達成共識，賽事過度頻密的關係，在聯賽球隊只能奪得季軍。在其後舉行的[[亞洲俱樂部錦標賽|亞洲俱樂部錦標賽]]中，柏斯波利斯亦未能取得好成績，最後被[[中國|中國]]球會[[大連實德|大連萬達]]淘汰，只得第四名。這段時期，柏斯波利斯亦為[[伊朗國家足球隊|伊朗國家足球隊]]提供[[艾哈邁德·雷薩·阿貝扎迪赫|阿貝扎迪赫]]、[[卡里姆·巴盖里|巴希利]]、[[霍達達德·阿齊茲|-{zh-hans:霍達達德·阿齊茲; zh-hk:阿斯斯;}-]]、[[阿里·戴伊|阿里·戴伊]]及[[迈赫迪·马达维基亚|馬達維基亞]]等不少出色球員。

[[1998年世界盃|1998年世界盃]]後，雖然球隊大量優秀球員相繼被[[歐洲|歐洲]]球會挖角，但球隊仍然保持隊型，並邀得[[阿里·卡里米|阿里·卡里米]]加盟。球隊更在1998-99年球季成為[[雙冠王|雙冠王]]，一舉奪得聯賽冠軍及哈茲菲盃冠軍。在其後的一個球季，球隊再次成為冠軍，奪得球會最後一個甲組聯賽冠軍。可是，在亞洲賽場，球隊再次奪得[[亞洲俱樂部錦標賽|亞洲俱樂部錦標賽]]季軍，未能突破球會最佳的成績。

=== 職業聯賽時期 (2001–2008) ===
柏斯波利斯2001年晉身新成立的[[伊朗職業足球聯賽|伊朗職業足球聯賽]]並再一次奪得冠軍，但過程卻相當驚險。最後一輪比賽前，球隊的宿敵[[艾斯迪格拿足球俱樂部|艾斯迪格拿]]還以兩分領先柏斯波利斯，但最後一輪卻落敗了。反之，柏斯波利斯最後一輪擊敗對手，並以一分反壓艾斯迪格拿成為第一屆職業聯賽冠軍。下一屆球隊便沒有那麼幸運，聯賽的過程十分艱巨。[[沙巴罕足球俱樂部|沙巴罕]]奪得了聯賽冠軍，而球會只能奪得第三名。另外，球隊在改革後的[[亞足聯冠軍聯賽|亞足聯冠軍聯賽]]中，亦未能突破分組賽階段，[[2002年至2003年亚洲冠军联赛#D組|小組賽出局]]。

及後，球會新主席阿克巴爾·格咸哈（{{lang|en|Akbar Ghamkhar}}）上任後，他作出了一系列承諾，並開始改革球隊。他公開了球員的薪金，嚴重地激怒了主帥柏文，最終柏文掛冠而去。格咸哈及後聘請了[[克羅地亞|克羅地亞]]籍領隊比高域，開始大灑金錢，購買一些出色的球員。雖然球隊於2003-04年有一個很好的起步，但最終失敗而回。比高域因此被解除職務，由[[德國|德國]]人索貝爾取代其職位。索貝爾上任後，球隊仍然未能於2004-05年奪得聯賽冠軍，索貝爾被逼離去，主席格咸哈亦於此時下台。

賀扎杜拉·哈提卜成為球會新的主席後，再次邀請球會史上最成功的領隊柏文回巢執教。這時，球會的財政問題開始浮現，但哈提卜卻無法化解今次由上任主席格咸哈的揮霍而遺留下來的問題。柏斯波利斯於2005-06年球季竟然了球會史上最差的第9名完成賽事，柏文亦再次辭職而去。

哈提卜於是辭去主席一職，由球會董事局選出安薩里法德成為球會新主席。安薩里法德亦聘請了[[阿里·哈恩|阿里·哈恩]]成為球會新任主帥，<ref>{{cite web|title= {{fa icon}} آری هان وارد تهران شد
|url=http://perspolis-club.net/News.aspx?id=202
|publisher=Persepolis Official Website
|accessdate=2006-01-31}}</ref>哈恩亦成功帶領球會晉身[[哈茲菲盃|哈茲菲盃]]決賽，但在決賽開始前卻因與球會管理層意見不合而被炒了。[[土耳其|土耳其]]人丹尼斯利與柏斯波利斯會方於2006年8月17日簽約，<ref>{{cite web|title={{fa icon}} دنیزلی سرمربی پرسپولیس شد|url=http://www.iran-varzeshi.com/1385/850528/html/footbal1.htm#s203178|publisher=Iran Varzeshi|accessdate=2006-08-18|deadurl=yes|archiveurl=https://web.archive.org/web/20061017092204/http://www.iran-varzeshi.com/1385/850528/html/footbal1.htm#s203178|archivedate=2006-10-17}}</ref><ref>{{cite web|title={{en icon}} Turkish trainer Denizli moves to Persepolis|url=http://www.mehrnews.com/en/NewsDetail.aspx?NewsID=368499|publisher=[[MNA|MNA]]|accessdate=2006-08-18|deadurl=yes|archiveurl=https://web.archive.org/web/20081208204826/http://www.mehrnews.com/en/NewsDetail.aspx?NewsID=368499|archivedate=2008-12-08}}</ref>成為新一任主帥。丹尼斯利的第一個任務就是帶領球會捧盃，可是球會未能於2006年的哈茲菲盃決賽捧盃，失去參加亞足聯冠軍聯賽的機會，失去參加一個可以改善球會財政困難的比賽的資格。丹尼斯利於下年的聯賽只能取得第三名，加上哈茲菲盃於準決賽出局。於是丹尼斯利只能跟隨主席安薩里法德，在球會的歷史上為自己畫上句號。

== 會徽及顏色 ==
柏斯波利斯是根據[[伊朗|伊朗]]著名地標[[波斯波利斯|波斯波利斯]]（[[阿契美尼德帝國|阿契美尼德帝國]]的首都）命名，因此球會會徽包含了[[波斯波利斯|波斯波利斯]]的元素。柏斯波利斯的第一個會徽以古[[波斯|波斯]]及[[瑣羅亞斯德教|瑣羅亞斯德教]]的圖形為基礎。圖形中，一個男人有一對隼的翼並有三層羽毛。

這個會徽只是於早年用了一段短時間，後來柏斯波利斯決定停用該會徽，直至1980年代。在1980年代中，球會創作了第二個會徽，兩隻[[公牛|公牛]]在旁而中間有一隻獎盃，獎盃下方則有[[奧林匹克五環|奧林匹克五環]]。公牛於古波斯文學是代表豐收的標誌，而獎盃則是代表冠軍。至1990年代，球會轉用第三代會徽，會徽變得更為簡潔。除了取消了奧林匹克五環外，獎盃還變得更大更明確。第三代標誌於2004年完成其任務，改為現時的標誌，變得更為簡化。奧林匹克五環重新出現在會徽中，過去版本中的公牛則改為[[獅鷲|獅鷲]]。


柏斯波利斯有一綽號是'''「Sorkhpoushan」'''（[[波斯語|波斯語]]: '''سرخپوشان'''，意思指[[紅色|紅色]]的人），而紅色就是球隊的傳統顏色。自球隊創立起，主場球衣都是以紅色為主要顏色，褲子亦通常是紅色(有些球季是[[白色|白色]]或[[黑色|黑色]])。而作客的球衣則以白色為主色，再加上少許的紅色。

== 主場球場及設施 ==
{{main|阿薩迪體育場}}
[[檔案:Azadistadium_tehran_iran.jpg|right]]]]
'''阿薩迪體育場''' ({{PerB|ورزشگاه آزادی}}) 是[[伊朗|伊朗]]的國家體育場及最大的體育場。官方稱球場可以容納90,000名觀眾入場觀看賽事。<ref>{{en icon}}[http://www.fussballtempel.net/afc/IRN.html Football stadiums of the world]</ref><ref name="{{fa icon}}استاديوم يكصد هزار نفری
"/>球場現時由[[德黑蘭市|德黑蘭]]雙雄柏斯波利斯及[[艾斯迪格拿足球俱樂部|艾斯迪格拿]]共用。

當柏斯波利斯成立時，體育會已經於[[德黑蘭市|德黑蘭]]北部有一系列的體育設施，但已經沒有位置可以再興建一個體育場。於是[[阿里·艾布杜|阿里·艾布杜]]買了一幅地興建了{{link-en|阿柏達納體育場|Rah Ahan Stadium}}<ref name="Khabare Varzeshi Magazine/Norouz 1387/Page 25">Khabare Varzeshi Magazine/Norouz 1387/Page 25</ref>，但由於體育場座位缺乏規劃，加上缺乏其他隊伍支持，最終使用了一場比賽便放棄使用，轉為球隊練習場。<ref name="Khabare Varzeshi Magazine/Norouz 1387/Page 25"/>

在[[阿薩迪體育場|阿薩迪體育場]]興建前，柏斯波利斯是使用{{link-en|沙希德·施雲迪體育場|Shahid Shiroudi Stadium}}。在阿薩迪建成後，球隊所有的主場比賽均在阿薩迪上演。由於阿薩迪體育場於2002-03年球季需要維修，球隊有2場的主場賽事移師{{link-en|塔赫蒂體育場|Takhti Stadium (Tehran)}}，未能在阿薩迪上演。<ref>{{cite web |title={{fa icon}} استاديوم فوتبال - زمين چمن |url=http://www.azadisportcomplex.com/amaken.asp?id=7378 |publisher=Azadi sport complex Website |accessdate= |deadurl=yes |archiveurl=https://web.archive.org/web/20080913070117/http://www.azadisportcomplex.com/amaken.asp?id=7378 |archivedate=2008-09-13 }}</ref>於2006年中，曾經有消息傳出柏斯波利斯會購買一個新的體育場，<ref>{{cite web

== 競爭對手 ==
[[File:PersePolis.jpg|thumb]]觀看柏斯波利斯對沙巴罕的比賽]]
柏斯波利斯是[[德比_(亞洲、非洲及大洋洲)#伊朗|德黑蘭打吡]]的其中一方，另一方為[[艾斯迪格拿足球俱樂部|艾斯迪格拿]]。雙方繼承前身[[沙恩足球俱樂部|沙恩]]及[[艾斯迪格拿足球俱樂部|皇冠]]的宿敵關係，雙方對碰亦成為[[伊朗|伊朗]]的一件盛事。自從[[沙恩足球俱樂部|沙恩]]的球員轉投柏斯波利斯後，球隊成為伊朗的強隊。而皇冠則成為皇室球隊，得到[[伊朗君主列表|伊朗國王]][[穆罕默德·礼萨·巴列维|穆罕默德·礼萨·巴列维]]的支持。由於雙方都是伊朗最強及最成功的球隊，兩隊經常互相爭奪錦標。

另一隊主要競爭對手則是來自[[伊斯法罕|伊斯法罕]]的球隊[[沙巴罕足球俱樂部|沙巴罕]]。兩隊均源自[[沙恩足球俱樂部|沙恩]]，柏斯波利斯在早年吸納了不少沙恩的球員加盟，而沙巴罕則是沙恩的其中一個分支。踏入2000年代，來自伊斯法罕的沙巴罕及[[佐阿漢足球俱樂部|佐阿漢]]實力增強。於2002-03年球季在[[伊朗職業足球聯賽|職業聯賽]]及[[哈茲菲盃|哈茲菲盃]]兩項全國錦標均由伊斯法罕的球隊取得勝利，直接威脅[[德黑蘭市|德黑蘭]]球隊的地位。因此，德黑蘭及伊斯法罕的競爭亦成為兩隊敵對的其中一個因素。
