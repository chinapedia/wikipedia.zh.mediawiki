'''柏𣾷'''（英語：'''The Parkhill'''），位於[[香港|香港]][[新界|新界]][[元朗|元朗]]唐人新村路138號，為[[新世界發展|新世界發展]]旗下的低密度[[香港私人屋苑列表|私人住宅]]項目，由王董建築師事務有限公司（[http://en.wikipedia.org/wiki/Wong_Tung_%26_Partners Wong Tung & Partners]）設計，[[新世界建築有限公司|新世界建築有限公司]]承建。項目設有6幢低座式住宅，合共提供141個單位，外型採用全玻璃幕牆設計，當中近一半為地下花園及頂層天台特色單位，於2016年第一季落成入伙。<ref>[http://www.nwd.com.hk/zh-hant/property/theparkhill 主要銷售物業 > 柏𣾷]</ref><ref>[http://ps.hket.com/content/879431/%E3%80%90%E7%84%A6%E9%BB%9E%E6%96%B0%E7%9B%A4%E3%80%91%E6%9F%8F%F0%A3%BE%B7%E9%A6%96%E6%89%B950%E4%BC%99%20%E5%8D%B0%E8%8A%B1%E7%A8%85%E6%B4%A5%E8%B2%BC%E5%84%AA%E6%83%A0%E5%90%B8%E5%AE%A2/ 柏𣾷首批50伙 印花稅津貼優惠吸客]《經濟日報》2015年10月8日</ref><ref>[http://ps.hket.com/content/1469015/%E3%80%90%E4%BA%A4%E6%8A%95%E6%8C%81%E7%BA%8C%E3%80%91%E5%94%90%E4%BA%BA%E6%96%B0%E6%9D%91%E6%9F%8F%F0%A3%BE%B7%E5%96%AE%E6%97%A5%E6%9A%AB%E6%B2%BD3%E4%BC%99%20%E5%A4%A9%E5%8F%B0%E6%88%B6981.2%E8%90%AC%E6%B2%BD/ 【交投持續】唐人新村柏𣾷單日暫沽3伙 天台戶981.2萬沽]《地產站》2016年7月22日</ref>

== 單位 ==
柏𣾷由6座4層高分層物業組成（第1座至第7座，不設第4座），包括地下、1樓至3樓。間隔方面，實用面積291至930平方呎，由開放式至3房間隔，大致屬長廳及左右廳設計，一律提供傳統獨立廚房。項目當中近五成為特色戶，分佈在地下連花園及3樓頂層連天台單位。<ref>[http://ps.hket.com/content/1429317/新世界元朗南柏𣾷 新世界元朗南「柏𣾷 THE PARKHILL」會所BAMBOO HOUSE及特色單位首度曝光]《地產站》2016年5月19日</ref><ref>[http://www.mpfinance.com/fin/specialpage2.php?sp=1463484206239&node=1465583324074&issue=20160611 元朗柏𣾷 享受田園生活]《明報》2016年6月11日</ref>

== 住客會所 ==
柏𣾷[[住客會所|會所]]名為「'''Bamboo House'''」，面積約11,400平方呎，設施包括[[泳池|戶外泳池]]、[[健身室|健身室]]、閱讀雅座等。會所另設4,302平方呎的[[園林|園林]][[綠化|綠化區]]，提供日光花園、戶外派對區、燒烤庭園及[[兒童遊樂場|兒童遊樂場]]。<ref>[http://ps.hket.com/content/879268/%E3%80%90%E5%8D%B3%E6%99%82%E7%9B%B8%E9%9B%86%E3%80%91%E9%9B%84%E6%93%9A%E6%96%B0%E7%95%8C%E7%AD%96%E7%95%A5%E6%80%A7%E5%9C%B0%E6%AE%B5%20%E6%9F%8F%F0%A3%BE%B7%E5%B1%95%E9%8A%B7%E5%BB%B3%E9%AB%94%E7%8F%BE%E7%8D%A8%E7%89%B9%E8%B3%A3%E9%BB%9E/ 雄據新界策略性地段 柏𣾷展銷廳體現獨特賣點]《經濟日報》2015年10月8日</ref>

== 物業管理 ==
柏𣾷由新世界[[物業管理|物業管理]]有限公司負責管理服務。

== 車位 ==
項目車位總數合共41個，包括31個住宅停車位及10個訪客停車位。

== 交通 ==
