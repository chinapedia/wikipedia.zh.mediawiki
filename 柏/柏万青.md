'''柏万青'''（{{bd|1949年|11月}}），常被称作“「'''柏阿姨'''」，[[上海人|上海人]]，是上海家喻户晓的[[上海电视台|电视台]]调解类谈话节目嘉宾主持，电视公众人物，中共党员。曾任江西省[[资溪县|资溪县]]副县长、上海市静安区[[静安寺街道|静安寺街道]]副处级组织员，现为上海市人大代表、静安区老年协会秘书长、静安区首席人民调解员。

==主要经历==
===早年经历 赣沪两地任职===
1968年，柏万青毕业于[[上海市延安中学|上海市延安中学]]高中，1969年，[[上山下乡|上山下乡]][[插队落户|插队落户]]到江西省[[黎川县|黎川县]]，成为一名女[[知青|知青]]。插队落户期间，柏万青先后担任过当地生产队妇女主任、乡镇干部、资溪县副县长。1996年，通过干部交流回到上海，在上海市静安区先后担任静安寺街道宣传科科长、文明办副主任、副处级组织员、党建办主任等职。<ref>{{cite web|url=http://www.zgfznews.com/fznews/fuzhouxinwen/shehuifazhi/2009/6/30/334407.shtml|title=柏万青：昔日资溪副县长 今朝上海大名人|accessdate=|date=2009-06-30|publisher=抚州日报|language=zh-hans|archive-url=https://web.archive.org/web/20160304085337/http://www.zgfznews.com/fznews/fuzhouxinwen/shehuifazhi/2009/6/30/334407.shtml|archive-date=2016-03-04|dead-url=yes}}</ref>

===担任人民调解员 成为电视名人===
2005年，退休后的柏万青担任上海市静安区老年协会秘书长、静安区首席人民调解员，并成立了「柏万青志愿者工作室」。2007年当选上海市人大代表，2008年，柏万青受邀成为[[上海广播电视台娱乐频道|上海广播电视台娱乐频道]]百姓调解家庭矛盾类节目《[[新老娘舅|新老娘舅]]》、《一呼柏应》的嘉宾主持，她在电视节目中个性突出、爱憎分明，以健谈直率，犀利的点评风格逐渐成为上海家喻户晓的电视名人。<ref>{{cite web|url=http://www.daynews.com.cn/sxrb/cban/C4/1020969.html|title=人民调解制度 社会矛盾“减压阀”|accessdate=|date=2010-09-02|publisher=山西日报|language=zh-hans}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>2010年8月，[[中华人民共和国人民调解法|中国第一部人民调解法]]在[[第十一届全国人大常委会|第十一届全国人大常委会]]中得到通过，人民调解工作拥有了法律依据，<ref name="人民调解法施行">{{cite news |url = http://news.sina.com.cn/c/2010-08-30/094018039411s.shtml|title = 人民调解法明年1月1日起施行|publisher = 新浪网 |date = 2010年8月30日}}</ref>使得电视台更加热衷于此类草根民生节目的制作，柏万青在电视台的出镜率也得到提高，红透了上海电视荧屏的半边天。2011年3月至2012年期间，柏万青还加盟过[[江西卫视|江西卫视]]，担任民生调解类节目《金牌调解》的金牌调解员。<ref>{{cite web|url=http://ent.hunantv.com/z/20110321/878230.html|title=江西卫视全新栏目《金牌调解》开播|accessdate=|date=2011-03-21|publisher=金鹰网|language=zh-hans}}</ref>

== 轶事与争议 ==

===足球轶事===
柏万青是中国足坛名宿[[徐根宝|徐根宝]]及[[上海东亚足球俱乐部|上海东亚足球俱乐部]]的铁杆球迷，<ref>{{cite web|url=http://www.fjsen.com/p/2011-05/05/content_4447921.htm|title=荧屏一妈与足坛一爹 柏万青：我曾为徐根宝哭过|accessdate=|date=2011-05-05|publisher=东方体育日报|language=zh-hans|deadurl=yes|archiveurl=https://web.archive.org/web/20150924015142/http://www.fjsen.com/p/2011-05/05/content_4447921.htm|archivedate=2015年9月24日|df=}}</ref>经常组织球迷去现场支持徐根宝的球队。2013赛季，上海东亚队升入中超，与同城传统强队[[上海申花|上海申花]]形成了德比大战，介于柏万青在上海的影响力，申花球迷自创了「'''踏平八万人,活捉柏万青'''」的球队助威口号，响彻体育场。<ref>{{cite web|url=http://shenhua.online.sh.cn/gb/content/2013-09/06/content_6352062.htm|title=黄飞珏笑解"活捉柏万青" 透柏阿姨要活捉申花球迷|accessdate=|date=2013-09-06|publisher=东方体育日报|language=zh-hans}}</ref><ref>{{cite web|url=http://v.youku.com/v_show/id_XNjAxMDA3ODMy.html|title=蓝魔高喊"踏平八万人，活捉柏万青”|accessdate=|date=2013-08-23|publisher=[[优酷网|优酷网]]视频|language=zh-hans}}</ref>

=== 女性守贞 ===

2011年报道<ref>{{Cite web |url=http://eladies.sina.com.cn/qg/2011/0304/15181056011.shtml |title=上海人大代表柏万青：贞操是女孩最好的嫁妆 |accessdate=2017-11-15 |author= |date=2011-03-04 |work= |publisher=新浪网，来源：汉网-长江日报|language=Zh-hans }}</ref>，她曾在电视节目中要求未婚女性[[守贞|守贞]]，强调：“[[贞操嫁妆论|贞操是女孩给婆家最贵重的陪嫁]]！”

===“被代言”违法P2P理财产品事件===
2016年11月，酷溜网上出现了一则题为《柏万青以及炳恒》的视频，视频中出现了“BH炳恒集团”标志及“上海市人大代表、东方电视台娱乐频道老年明星主持柏万青”字样。之后柏万青表示其从未听说过炳恒集团，从未录制过视频内容，与炳恒财富之间不存在任何代言关系，后炳恒财富因非法吸收公众存款被立案调查。2018年11月19日，上海市[[徐汇区|徐汇区]]人民法院一审判决，炳恒财富被判停止侵权，赔礼道歉，赔偿各项损失共计43000元<ref>{{Cite web |url=http://shanghai.xinmin.cn/xmsq/2018/11/19/31456019.html |title=柏万青“被代言”违法P2P理财产品，法院：P2P公司侵权须赔偿，视频发布者无责 |accessdate=2018-11-21 |author= |date=2018-11-19|work= |publisher=上观新闻|language=Zh-hans }}</ref>。
