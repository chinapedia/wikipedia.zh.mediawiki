'''柏科'''（[[学名|学名]]：{{lang|la|''Cupressaceae''}}）[[植物|植物]]广泛分布在全世界，包括27-30个[[属|属]]，（其中有17个属只有单一[[种|种]]），大约有130-140[[种|种]]。

柏科植物是分布最广的针叶植物，几乎遍布除[[南极洲|南极洲]]以外的世界各地，从[[北纬|北纬]]70° [[挪威|挪威]][[北极|北极]]区域的[[刺柏|刺柏]]到[[南纬|南纬]]55°[[智利|智利]]的[[智利柏|智利柏]]，生活在[[西藏|西藏]][[海拔|海拔]]5200[[米|米]]地区的[[滇藏方枝柏|滇藏方枝柏]]是已知能在海拔最高区域存活的[[木本植物|木本植物]]。

==形态==
绝大部分是雌雄同株的，也有很少种是雌雄异株的，一般都是[[乔木|乔木]]，最高的可达130[[米|米]]，也有少数种是[[灌木|灌木]]。树皮都是发[[红色|红]][[棕色|棕色]]，绳状结构，脱落时都是竖条形，只有少数种例外。

绝大部分种为常绿植物，叶子可成活2－10年，但三个属（落羽杉属、水杉属和水松属）是[[落叶树|落叶树]]。

柏科植物的[[叶|叶]]呈螺旋状排列，芽叶一般是尖针状，成熟后呈鳞片状，但也有的种终生为针状。球花的[[雄蕊|雄蕊]]及具胚珠的鳞片亦交互对生，很少三个轮生；雄球花的雄蕊有2－6药囊，雌球花全部或部分种鳞具有一至多数胚珠、苞鳞和种鳞结合。

卵形或圆球形球果当年或第二年成熟；果实一般为[[坚果|坚果]]，也有[[浆果|浆果]]（如[[刺柏属|刺柏属]]）；鳞片木质扁平或盾形，发育的鳞片具有一至多个[[种子|种子]]，种子具翅或无翅。

== 分类 ==
最新的分类法将以前单独分为“[[杉科|杉科]]”的植物划入柏科，只是其中的[[金松属|金松属]]被单独分为[[金松科|金松科]]。

柏科被划分为7个[[亚科|亚科]]：
* [[杉木属|杉亚科]]（Cunninghamioideae）：[[杉属|杉属]]；
* [[密叶杉属|密叶杉亚科]]（Athrotaxidoideae）：[[密叶杉属|密叶杉属]]；
* [[台灣杉屬|台湾杉亚科]]（Taiwanioideae）：[[台湾杉属|台湾杉属]]；
* [[紅杉亚科|紅杉亚科]]（Sequoioideae）：[[红杉属|红杉属]]，[[巨杉|巨杉属]]，[[水杉属|水杉属]]；
* [[落羽杉亚科|落羽杉亚科]]（Taxodioideae）：[[落羽杉属|落羽杉属]]，[[水松属|水松属]]，[[柳杉属|柳杉属]]；
* [[柏松亚科|柏松亚科]]（Callitroideae）：[[柏松属|柏松属]]，[[澳洲柏松属|澳洲柏松属]]，[[新喀里多尼亚柏松|新喀里多尼亚柏松属]]，[[非洲柏松属|非洲柏松属]]，[[塔斯马尼亚柏松属|塔斯马尼亚柏松属]]，[[智利肖柏属|智利肖柏属]]，[[智利雪松属|智利雪松属]]，[[肖楠属|肖楠属]]，[[智利柏属|智利柏属]]，[[巴布亚柏松属|巴布亚柏松属]]；
* [[柏亚科|柏亚科]]（Cupressoideae）：[[崖柏属|崖柏属]]，[[罗汉柏属|罗汉柏属]]，[[扁柏属|扁柏属]]，[[福建柏属|福建柏属]]，[[翠柏属|翠柏属]]，[[四鳞柏属|四鳞柏属]]，[[小侧柏属|小侧柏属]]，[[侧柏属|侧柏属]]，[[澳洲柏属|澳洲柏属]]，[[柏木属|柏木属]]，[[刺柏属|刺柏属]]。

== 世界之最 ==
[[File:Cupressaceae.PNG|thumb]]
柏科植物包括世界上最大的、最高的、木材最硬的和第二长寿的物[[种|种]]：
* 最大的 – 巨杉属的[[世界爷|世界爷]]；
* 最高的 – 红杉属的[[加州红木|红杉]]；
* 最粗的 - 落羽杉属的[[墨西哥落羽杉|墨西哥落羽杉]]；
* 寿命第二长的 - 智利肖柏属的[[智利柏|智利柏]]（仅次于[[刺果松|刺果松]]）。

== 用途 ==
[[File:PlatycladusOrientalis.jpg|thumb]]
柏科植物很多种类是主要的木材原料，其中重要的是柳杉属、杉属、落羽杉属、翠柏属、红杉属、柏木属、扁柏属和崖柏属的品种。此外在园艺上也是重要的植物种群，尤其是刺柏属植物，被培植出[[蓝色|蓝色]]、[[灰色|灰色]]、[[黄色|黄色]]的各种品系；崖柏属的植物也被培植出多种矮化品种；[[水杉|水杉]]、[[美国扁柏|美国扁柏]]和[[側柏|側柏]]等也是被广泛种植的园艺品种，还有些品种被种植作为[[圣诞树|圣诞树]]。
[[日本柳杉|日本柳杉]] (''Cryptomeria japonica'')是[[日本|日本]]的国树；[[墨西哥落羽杉|墨西哥落羽杉]] (''Taxodium mucronatum'')是[[墨西哥|墨西哥]]的国树；[[巨杉|巨杉]]和[[红杉|红杉]]是[[加利福尼亚州|加利福尼亚州]]的并列的州树；[[落羽杉|落羽杉]]（''Taxodium distichum''）是[[路易斯安那州|路易斯安那州]]的州树；[[北美白松|北美白松]]（''Juniperus virginiana''）木材可以防虫蛀，用来做家具；[[刺柏|刺柏]]浆果（杜松子）是造[[金酒|金酒]]的原料。

[[北美翠柏|北美翠柏]]（''Calocedrus decurrens''）的木材是制造[[铅笔|铅笔]]的主要用材；福建柏的[[根|根]]可以提炼芳香油，用于药和化妆品生产。 

柏科植物的[[花粉|花粉]]容易让部分人产生过敏致病，日本柳杉曾经造成问题。

刺柏是[[胶锈菌属|胶锈菌属]]（''Gymnosporangium''）[[真菌|真菌]]的宿主，这种菌可以造成[[苹果|苹果]]、[[梨|梨]]等果树发生锈病。
