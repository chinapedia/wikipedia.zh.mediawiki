'''柏拉图环形山'''（Plato）是[[月球正面|月球正面]]位于[[雨海|雨海]]东北侧、[[阿尔卑斯山脉_(月球)|阿尔卑斯山脉]]西北端的一座大[[撞击坑|撞击坑]]，约形成于38-32亿年前的[[晚雨海世|晚雨海世]]，略晚于雨海地质龄<ref name="LICD"/>，其名称取自[[古希腊|古希腊]][[哲学家|哲学家]]"[[柏拉图|柏拉图]]"(约公元前428/427年-公元前348/347年)，是由[[意大利|意大利]][[天文学家|天文学家]]暨[[耶稣会|耶稣会]][[天主教|天主教]][[神父|神父]]"[[乔万尼·巴蒂斯塔·里乔利|乔万尼·巴蒂斯塔·里乔利]]"在1651年首次提出，1935年被[[国际天文学联合会|国际天文学联合会]]最终正式确定。

==描述==
该陨坑南面是一系列被称作[[特內里費山脈|特內里費山脈]]的隆起地形、西北伸展着[[冷海|冷海]]，而东面，在[[阿爾卑斯山脈_(月球)|阿尔卑斯山脉]]间分布着数条统称为柏拉图月溪的沟壑，而它的西北则坐落了较小的[[布利斯陨石坑|布利斯陨石坑]]<ref>[http://planetarynames.wr.usgs.gov/images/Lunar/lac_12_wac.pdf Crater Plato on the map LAC-12.]</ref>，该陨坑中心[[月面坐标|月面坐标]]为{{Coord|51.62|N|9.38|W|globe:Moon|nogoogle=1|noosm=1}}，直径100.68公里<ref name="IAU">[http://planetarynames.wr.usgs.gov/Feature/4757 Directory of the International Astronomical Union]</ref>，深度约2.86公里<ref name="LICD"/>，内部容积12100<sup>3</sup><ref name="LICD">Lunar Impact Crater Database (2011) Losiak et al, LPI Lunar Exploration Intern Program (2009). Revised by Ohman, LPI (2011).</ref>。

柏拉图环形山已完全被[[玄武岩|玄武岩]][[熔岩|熔岩]]所淹没，坑内没有中央峰及较醒目的大陨坑，表面只散布有少量细小的撞击坑穴。坑底高出[[雨海|雨海]]近500米，并呈现相对较暗的[[反照率|反照率]]，与周边崎岖的背景地形形成极大的反差，看上去就像一块明显的暗斑。十七世纪[[波兰|波兰]][[天文学家|天文学家]][[约翰·赫维留|约翰·赫维留]]曾将它称之为"大黑湖"。

该陨坑外观轮廓大体圆形，部分边缘略呈锯齿状，沿坑口峰谷峭崿错落，其中有位于西侧壁上高2900米的泽塔峰(ζ)和东侧壁上3200米高的伽玛峰(γ)、2600米高的德尔德峰(δ)以及1900米高的艾普西隆峰(ε)<ref>[http://the-moon.wikispaces.com/Plato Crater site description The Moon.Wikispaces.]</ref>，当太阳光斜照在这些巉刻兀立的峰壁上时，则在坑底投下一些突出的壁影，不过请注意，以上峰高数据尚未得到完全证实，只是引自一些其它的来源。

柏拉图环形山部分内侧壁已产生坍塌，尤其是西侧一处长约15公里的大三角区-泽塔高原，一道地峡将它与坑壁相分离；而西南侧坑壁上则穿切了一道醒目的月谷。柏拉图外观形状较为规则，但从[[地球|地球]]看因[[透视|透视]]作用而呈现为卵状。

==月球瞬变现象==
柏拉图环形山最為人知晓的就是所谓的[[月球瞬变現象|月球瞬变現象]]，包括亮斑、色变、部分区域朦胧雾化以及[[月食|月食]]期间出现闪光，它是已报告月球上出现瞬变現象第二多的地方。这些异常现象目前尚无确切的解释。迄今为止，不管是宇宙探测器或是地球观测站，都没拍摄到任何一张有关该陨坑瞬变現象的照片，因此，瞬变现象极可能只是观察条件，加上太阳光照射角度变化的共同作用所导致。

== 科幻作品 ==
它是1955年[[亞瑟·查理斯·克拉克|亞瑟·查理斯·克拉克]]短篇小說《[[地光_(小說)|地光]]》中的天文观察站；1966年[[罗伯特·海因莱因|罗伯特·安森·海莱恩]]《[[严厉的月亮|严厉的月亮]]》小说中月球上"拥挤"的香港殖民地以及系列电视连续剧《[[太空：1999|太空：1999]]》中的[[月球阿尔法基地|月球阿尔法基地]]。

在美国二十世纪儿童读物作家"杰罗姆·贝蒂"(Jerome Beatty Jr.)所撰写的鲁尼系列儿童丛书中，柏拉图环形山还是主人公"马修·鲁尼"和"玛丽亚·鲁尼"在月球上的穴屋。
