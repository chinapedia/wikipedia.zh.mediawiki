'''艾莉森·麦克'''（{{lang-en|Allison Mack}}，又译：'''爱莉森·麦克'''、'''艾莉森·马克'''，{{bd|1982年|7月29日|catidx=M}}），美国女演员，因在《[[超人前传|超人前传]]》饰演{{link-en|Chloe Sullivan|Chloe Sullivan}}而为人所知。

在2018年，麥克因擔任[[NXIVM|NXIVM]]的其中一名高級成員而得到媒體注意，NXIVM是一個被指控從事勒索、性交易和強迫勞動的組織，為此它被一些專家指責為[[邪教|邪教]]。<ref>{{Cite web |url = https://artvoice.com/2018/03/27/allison-mack-expected-arrested-next/#.Wt6b85criUl |title = Allison Mack expected to be arrested next |author = Chitra Selvaraj |date = 2018-03-27 |publisher = Artvoice |language = en}}</ref><ref>{{Cite web |url = https://www.rollingstone.com/culture/news/nxivm-what-we-know-about-alleged-sex-trafficking-forced-labor-w518483 |title = NXIVM: What We Know About Alleged Sex Trafficking, Forced Labor |author = Laura Barcella |date = 2018-03-28 |publisher = Rolling Stone |language = en}}</ref>2018年4月20日，麥克因為性交易、共謀性交易和共謀強迫勞動的罪名而被[[聯邦調查局|聯邦調查局]]（FBI）逮捕。<ref>{{Cite web |url = https://www.nbcnewyork.com/news/local/Actress-Allison-Mack-Arrested-Nxivm-Sex-Cult-480387783.html |title = 'Smallville' Actress Allison Mack Arrested in Connection to NXIVM Sex Cult Case |author =  Jonathan Dienst |date = 2018-04-23 |publisher = WNBC |language = en}}</ref><ref>{{Cite web |url = http://www.bbc.com/news/world-us-canada-43846243 |title = Allison Mack, Smallville actress, charged over Nxivm sex trafficking |date = 2018-04-21 |publisher = BBC News |language = en}}</ref>麥克在2019年4月承認犯有敲詐勒索罪，目前正在等待2019年9月的判決。

== 法律問題以及與NXIVM的關係 ==
[[File:Nxivm-Indictment-S-1.pdf|thumb]]
2010年，據報導麥克與她的《超人前傳》聯合演員[[克莉斯汀·克魯克|克莉斯汀·克魯克]]一起被招募到[[多層次傳銷|多層次傳銷]]組織[[NXIVM|NXIVM]]的[[溫哥華|溫哥華]]分會<ref>{{Cite web |url = https://www.macleans.ca/economy/business/how-to-lose-100-million/ |title = How to lose $100 million |author = Nicholas Köhler |date = 2010-09-09 |publisher = Maclean's |language = en}}</ref>。在2003年《[[福布斯|福布斯]]》的一篇文章中，NXIVM的擁護者描述組織是一個專注於鼓舞人心的執行管理輔導組織，「就像一個實用的工商管理碩士（MBA）」，可是批評者指責創辦人Keith Raniere是執行「一個心理學上把他的對象崩潰的類似[[邪教|邪教]]計劃」<ref>{{Cite web |url = https://www.forbes.com/forbes/2003/1013/088.html#120732bc1853 |title = Cult of Personality |author = Michael Freedman  |date = 2003-10-13 |publisher = Forbes |language = en}}</ref>。該團體有一個為他們的受害者打上烙印的習慣，這是由麥克首先創作出來的。<ref>{{Cite web |url = https://www.nytimes.com/2018/05/30/magazine/sex-cult-empowerment-nxivm-keith-raniere.html |title = Inside Nxivm, the 'Sex Cult' That Preached Empowerment |author = Vanessa Grigoriadis |date = 2018-05-30 |publisher = The New York Times |language = en}}</ref>

前NXIVM成員{{link-en|莎拉·艾德蒙森|Sarah Edmondson}}在2017年《[[紐約時報|紐約時報]]》和2018年{{link-en|A&E (TV channel)|A&E (TV channel)|A&E}}特別節目中表示麥克招募她到一個領域叫「DOS」，而且她已經在麥克的監督下在麥克家內的入會儀式中被打上烙印。2017年6月，調查記者Frank Parlato報導，麥克的人名首字母被烙印在女性的丹田位置。<ref>{{Cite web |url = https://frankreport.com/2017/06/05/part-1-branded-slaves-and-master-raniere-sources-human-branding-part-of-raniere-inspired-womens-group/ |title = Part 1: Branded Slaves and Master Raniere; Sources: Human branding part of Raniere-inspired women's group |author = Frank Parlato |date = 2017-06-05 |publisher = Frank Report |language = en}}</ref>

2018年組織被破獲，麥克依違反性交易、共謀性交易、共謀強迫勞動罪名，在紐約被逮捕後全盤認罪。<ref>[http://www.chinatimes.com/realtimenews/20180421003060-260404 《超人前傳》女星被捕！驚爆為邪教性奴]</ref>在2019年4月8日，麥克對敲詐勒索指控表示認罪。<ref>{{Cite web |url = https://www.nytimes.com/2019/04/08/nyregion/nxivm-allison-mack.html |title = Allison Mack of ‘Smallville’ Pleads Guilty in Case of Nxivm 'Sex Cult' Where Women Were Branded |author = Michael Gold, Colin Moynihan |date = 2019-04-08 |publisher = The New York Times |language = en}}</ref>
