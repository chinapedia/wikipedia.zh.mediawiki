'''艾拉·纽伯尔'''（{{lang-en|'''Ira Newble'''}}，{{bd|1975年|1月20日|}}），出生于[[密歇根州|密歇根州]][[底特律_(密歇根州)|底特律]]，[[美国|美国]]职业篮球运动员。

== 早年生涯 ==
纽伯尔1993年毕业于[[密歇根州|密歇根州]]的Southfield高中，随后进入迈阿密大学打球。在大学三年级时取得场均6.4分和5.2个篮板，四年级时拿到了场均11.2分，7.5个篮板和1.2次助攻的数据，并且获得了美国All-Mid分区最佳球员的提名。

== NBA时期 ==
他在加盟现球队之前还效力过[[圣安东尼奥马刺队|圣安东尼奥马刺队]]、[[亚特兰大鹰队|亚特兰大鹰队]]和[[克利夫兰骑士队|克利夫兰骑士队]]。

== 争议活动 ==
艾拉·纽伯尔组织了一场NBA球员中的征集签名活动，用于抗议中国政府插手[[达尔富尔|达尔富尔]]问题，并递交抗议信至中国政府以及[[2008年夏季奥林匹克运动会|北京奥运]]组委会。纽伯尔之所以关注这一事件，其中一个原因为他是原籍[[苏丹|苏丹]]的非洲裔美国人。
