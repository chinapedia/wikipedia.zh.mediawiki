'''艾宝俊'''（{{bd|1960年|2月|catIdx=A艾}}），[[中国|中国]][[辽宁省|辽宁省]][[辽阳市|辽阳市]]人。曾任[[中共上海市委|中共上海市委]]常委、[[上海市人民政府|上海市人民政府]]副市长、市政府党组副书记，市[[长兴岛开发建设委员会|长兴岛开发建设委员会]]主任、市临港地区开发建设管委会主任。<ref name="测试">[http://wsxf.sh.gov.cn/swldxxnew/resume_ABJ.aspx 市委常委] {{webarchive|url=https://web.archive.org/web/20130831080249/http://wsxf.sh.gov.cn/swldxxnew/resume_ABJ.aspx |date=2013-08-31 }}，上海市委。</ref>

1983年毕业于[[东北大学|东北工学院]]自动控制系工业自动化专业，毕业后赴[[安徽省|安徽省]][[马鞍山市|马鞍山市]][[华东冶金学院|马鞍山钢铁学院]]进修工业统计和会计。进修结束后于1985年开始任教于东北工学院管理工程系，主讲[[计算机|计算机]]在[[会计|会计]]中的应用。不久艾又考入本校研究生，1988年毕业于企业管理专业，获得硕士学位，其后留校任财经处副总会计师。1989年取得[[会计师|会计师]]资格。1990年至1991年间，艾被派往[[美国|美国]][[北卡罗莱纳州|北卡罗莱纳州]][[阿巴拉契亚州立大学|阿巴拉契亚州立大学]]（Appalachian State University）作为[[访问学者|访问学者]]进修工业会计和税收课程。<ref>[http://www.chinese-edu.org/bbs/ShowPost.asp?ThreadID=1344 中华教育传记中心教育专家人才传记之一]{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

==履历==
1994年9月，应聘进入[[上海宝钢集团公司|上海宝钢集团公司]]，负责成本管理。1998年升任宝钢集团副总经理兼总会计师。2000年2月，升任宝山钢铁股份有限公司总经理。2007年12月27日，被任命为上海市副市长，分管发展改革、计划、编制、人口综合管理、统计、物价、能源建设、口岸、信访、社会稳定等工作。2008年4月，兼任上海市长兴岛开发建设委员会主任。2010年2月，兼任上海市临港产业区管委会主任。2012年5月22日，在中共上海市第十届委员会第一次全体会议当选为市委常委。2013年9月，兼任上海综合保税区管委会主任、上海国际旅游度假区管委会主任、中国（上海）自由贸易试验区管委会主任<ref>{{cite web|accessdate=2014-06-04|url=http://district.ce.cn/newarea/sddy/201309/30/t20130930_1571320.shtml|title=艾宝俊任上海自贸区管委会主任|publisher=中国经济网}}</ref>。

==落马==
2015年11月6日，担任中国（上海）自由贸易试验区管委会主任的艾宝俊意外缺席“新形势下加快推进中国（上海）自由贸易试验区建设”专题培训班开班仪式。仪式由另一位自贸区管委会主任[[沈晓明|沈晓明]]主持<ref>{{cite web|accessdate=2015年11月10日|url=http://news.sina.com.cn/c/2015-11-10/doc-ifxkniur3027553.shtml|title=艾宝俊11月6日缺席自贸区相关会议|publisher=新浪网}}</ref>。2015年11月10日，中纪委监察部网站宣布艾宝俊因涉嫌严重违纪，接受组织调查<ref>{{cite web|accessdate=2015年11月10日|url=http://www.ccdi.gov.cn/xwtt/201511/t20151110_64768.html|title=上海市委常委、副市长艾宝俊涉嫌严重违纪接受组织调查|publisher=中央纪委监察部网站}}</ref>。艾宝俊于当日上午10点乘坐东航航班被押运至北京，而一同被查的还有艾宝俊的儿子[[艾卿|艾卿]]、艾宝俊的弟弟[[艾宝魁|艾宝魁]]，三人分乘三个航班<ref>{{cite web|url=http://finance.sina.com.cn/china/20151114/012523765766.shtml|title=上海首虎艾宝俊儿子也被查 或涉宝钢及自贸区贪腐|publisher=新浪网|accessdate=2015-11-14}}</ref>。据报道，艾宝俊应是涉及宝钢及自贸区双重贪腐而受到追查。他是[[中共十八大|中共十八大]]以来上海首位被调查的省部级官员。据财新网报道，查处艾宝俊的决定在更早时候作出，但考虑到其妻子病危，没有立即宣布。10日正是艾宝俊妻子因肾病去世第七天<ref>{{cite web|accessdate=2015年11月10日|url=http://china.caixin.com/2015-11-10/100872544.html|title=上海副市长艾宝俊落马 曾执掌宝钢多年|publisher=财新网}}</ref>。16日，[[中共中央组织部|中共中央组织部]]称，中央已决定免去艾宝俊的领导职务<ref>{{cite web|accessdate=2015年11月16日|url=http://news.xinhuanet.com/2015-11/16/c_1117156577.htm|title=上海市委常委、副市长艾宝俊被免职 |publisher=新华网}}</ref>。

2016年1月19日中央批准开除党籍、行政开除<ref>{{cite web |url = http://www.ccdi.gov.cn/jlsc/djcf/201601/t20160119_73113.html |title = 上海市原市委常委、副市长艾宝俊严重违纪被开除党籍和公职 |accessdate = 2016年1月19日 |publisher = 中央纪委监察部网站}}</ref>。

2017年2月23日，[[福建|福建]][[漳州市|漳州市]]中院开庭审理艾宝俊受贿、贪污一案，认定他受贿折合人民币4320万元，非法占有折合人民币751万元，法庭将择期宣判。<ref >{{cite web|url=http://news.sina.com.cn/sf/news/ajjj/2017-02-23/doc-ifyavvsk2878851.shtml | language= | title=上海原副市长艾宝俊受贿案一审 将择期宣判 |publisher=正义网 |date= 2017年2月23日|accessdate= }}</ref>

2017年4月28日，[[福建|福建]][[漳州市|漳州市]]中級法院裁定艾寶俊受賄罪判監15年，處沒收個人財產500萬元，貪污罪則判囚10年，沒收個人財產80萬元，兩罪合併執行刑期17年，被沒收個人財產合共580萬人民幣<ref>{{cite web |url = http://www.ccdi.gov.cn/yw/201704/t20170428_98314.html |title = 上海市委原常委、副市长艾宝俊一审获刑十七年 |accessdate = 2017年4月28日 |publisher = 中央纪委监察部网站}}</ref>。

==家族==
*妻子：姓名不详，因罹患肾病2015年11月4日病逝于[[上海交通大学医学院附属瑞金医院|上海市瑞金医院]]。曾是上海资本市场的一名活跃人物，在股市上颇具影响力，尤其在一些国企改革题材股上斩获颇丰。据《财经》杂志报道，她在2015年中国股灾中曾因有操纵市场嫌疑被调查<ref>{{cite web|url=http://finance.sina.com.cn/stock/y/20151111/072423735762.shtml|title=知情人士：艾宝俊妻子在股灾中曾因涉嫌操纵市场被查|publisher=新浪网|accessdate=2015-11-11}}</ref>。
*儿子：艾卿，曾先后供职瑞士银行、德意志银行，现为上海冠英股权投资管理合伙企业合伙人，冠英资本由一家浙江大型民营集团参与出资，旗下共计有冠英一期、冠英科超、冠英清甲等六家资本运作平台。据《华夏时报》报道，艾宝俊和上海覆盖范围最广的无线区域网络“i-Shanghai”之间有着隐秘联系，艾宝俊可能将这一项目交给了其子操作<ref>{{cite web|url=http://finance.sina.com.cn/china/20151114/012523765766.shtml|title=上海首虎艾宝俊儿子也被查 或涉宝钢及自贸区贪腐|publisher=新浪网|accessdate=2015-11-14}}</ref>。
*弟弟：艾宝魁
