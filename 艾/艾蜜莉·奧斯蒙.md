'''艾蜜莉·喬丹·奧斯蒙'''（{{lang|en|Emily Jordan Osment}}，{{bd|1992年|3月10日|catIdx=Osment}}）是一位[[美利堅合眾國|美國]][[女演員|女演員]]和[[唱作歌手|唱作歌手]]。她以[[:en:Spy_Kids_trilogy|《小鬼大間諜》系列]]的葛媞·傑格斯以及[[迪士尼頻道|迪士尼頻道]]所屬的[[電視影集|電視影集]]《[[孟漢娜|孟漢娜]]》中的[[:en:Lilly_Truscott|莉莉·特斯卡]]知名。《[[孟漢娜|孟漢娜]]》曾獲[[艾美獎|艾美獎]]提名<ref>{{cite web|title=The 59th Primetime Emmy Awards and Creative Arts Emmy Awards Nominees|publisher=Academy of Television Arts & Sciences|url=http://www.emmys.org/awards/2007pt/|accessdate=2007-07-31|deadurl=yes|archiveurl=https://web.archive.org/web/20070927001940/http://www.emmys.org/awards/2007pt/|archivedate=2007-09-27}}</ref>，且該影集的首演創造了[[迪士尼頻道|迪士尼頻道]]五百四十萬觀賞人次的紀錄。她也在[[:en:RL_Stine|RL Stine]]的小說改編的[[錄影帶首映|錄影帶首映]]的[[電影|電影]]《[[:en:The_Haunting_Hour_Volume_One:_Don't_Think_About_It|勇闖天關：別去想它]]》裡扮演凱西。2007年十二月她正在拍攝《孟漢娜》的衍生電影《[[:en:The_Hannah_Montana_Movie|孟漢娜電影版]]》，該片已2009年四月十號上映。<ref>{{cite news|title=States fight for ''Hannah Montana'' movie|publisher=CNN|date=2007-11-23|url=http://edition.cnn.com/2007/SHOWBIZ/Movies/11/23/hannahmontana.movie.ap/index.html|accessdate=2007-12-11|deadurl=yes|archiveurl=https://web.archive.org/web/20071126022956/http://edition.cnn.com/2007/SHOWBIZ/Movies/11/23/hannahmontana.movie.ap/index.html|archivedate=2007-11-26}}</ref> 現於[[Freeform|ABC Family／Freeform]]主演影集《[[美味新關係|美味新關係]]》。

== 早年生活 ==
奧斯蒙出生在[[加利福尼亞州|加州]][[洛杉磯|洛杉磯]]<ref>[http://www.imdb.com/name/nm0652089/bio Bio at IMDb]</ref>，那裡是她的家人，包括也是演員的父親[[尤金·奧斯蒙|尤金·奧斯蒙]]和在當老師的母親泰瑞莎·奧斯蒙（冠[[夫姓|夫姓]]，娘家姓是Seifert）仍舊居住的地方。<ref>{{cite web | author= | title=Haley Joel Osment's Biography| url=http://www.nndb.com/people/886/000025811/ | publisher=Notable Names Database| date=2007 | accessdate=2007-08-07}}</ref><ref>[http://www.nndb.com/people/657/000129270/|title=%20 Emily Osment's Biography]{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }} Names Database (2007) Retrieved on 2007-[[08-07|08-07]]</ref>奧斯蒙從未對演藝事業陌生過。她的父親尤金·奧斯蒙演過許多電影，而她的哥哥是[[奧斯卡金像獎|奧斯卡金像獎]]提名的演員[[哈利·喬·奧斯蒙|哈利·喬·奧斯蒙]]。
奧斯蒙出生在[[天主教|天主教]]家庭。<ref name="variety">{{cite news|last=Duke|first=Paul F.|coauthors=|title=Osment, Dafoe honing 'Edges'|pages=|publisher=Variety.com|date=2000-04-25|url=http://www.variety.com/article/VR1117780884.html?categoryid=13&cs=1|accessdate=2007-12-10}}</ref>她曾經是[[加州|加州]][[:en:La_Cañada|拉肯納達市]]的[[:en:Flintridge_Preparatory_School|福臨特里奇預備學校]]的學生。她在《{{link-en|The View|The View}}》裡表示她現在在修線上課程。<!-- YouTube link to The View was a copyright violation -->

== 事業 ==
[[File:Emily_Osment_2010_2.jpg|thumb]]
奧斯蒙進入演藝圈是在1998年，她參與一個送花公司的廣告演出。之後，她繼續拍攝多支廣告。然後在1999年和尤金‧李維及琳達‧漢彌爾頓演出她的第一部電影《女孩們的私密生活》。同年她與葛倫‧克蘿絲在《薩拉、平原、高：冬天的末端》中共同演出，獲得年輕藝術家獎的提名。從此她成為一個演員，演出《由天使接觸》、《六人行》、《太陽的第三岩石》等電視劇。她現在在迪士尼工作，在《孟漢娜》系列中飾演莉莉·塔斯科特。這個角色再度使她獲得年輕藝術家獎的提名。此系列的電視劇創造了540萬觀賞人次的空前記錄，迪士尼公司的總裁讚賞「超乎我們最狂野的想像」。另外，艾蜜莉也協助設計《孟漢娜》系列中的一些服飾。2006年，艾蜜莉受邀參加首次的迪士尼明星賽，成為綠隊的一員，2007年又成為新的黃隊的一員。她也在《勇闖天關：別去想它》中飾演凱西，並在片中和同為孟漢娜系列演員的寇弟‧林利合作。

== 個人生活 ==
艾蜜莉和同為《[[孟漢娜|孟漢娜]]》演員的[[Miley_Cyrus|麥莉‧希拉]]及[[Mitchel_Musso|米契爾‧穆索]]為密友，在忙碌的時候仍互相關照。麥莉教艾蜜莉彈吉他；艾蜜莉教麥莉編織。她也與《[[High_School_Musical|歌舞青春]]》的演員[[Vanessa_Hudgens|凡妮莎‧哈金斯]]、《[[小查與寇弟的頂級生活|小查與寇弟的頂級生活]]》、《歌舞青春》的演員[[Ashley_Tisdale|艾希莉‧提斯黛爾]]和[[宋布蘭|布蘭達‧宋]]是朋友。艾蜜莉喜愛高爾夫，曾與她的哥哥一同參加第十屆年度麥可‧道格拉斯名人高爾夫友誼賽。
