'''阿倫·莊士敦'''或'''艾伦·约翰斯顿'''（'''Alan Graham Johnston'''，{{bd|1962年|5月17日|catIdx=J}}）[[英國廣播公司|英國廣播公司]][[記者|記者]]，曾經駐守[[阿富汗|阿富汗]]、[[加薩|加沙]]和[[烏茲別克|烏茲別克]]。當地時間2007年3月12日，他在加沙被綁架。至3月26日，他成為2年以來被囚禁在加沙時間最長的外國人。他的囚禁，惹來巴勒斯坦許多示威，亦令到英國政府第一次需要跟[[哈馬斯|哈馬斯]]成員會面。7月4日，哈马斯的电视台宣布，约翰斯顿当天凌晨获释。<ref>王志强，洪漫，[http://news.xinhuanet.com/world/2007-07/04/content_6326965.htm 详讯：在加沙被绑架的英国记者获释]{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}，新华网</ref>

== 簡介 ==
阿倫·莊士敦是[[蘇格蘭|蘇格蘭]]人<ref>[http://thescotsman.scotsman.com/uk.cfm?id=429362007 綁架英廣記者在生但下落不明] {{Wayback|url=http://thescotsman.scotsman.com/uk.cfm?id=429362007 |date=20070822221749 }} - 蘇格蘭人報</ref>，1962年5月17日在[[坦桑尼亞|坦桑尼亞]][[林迪|林迪]]出生<ref>[http://news.bbc.co.uk/2/hi/middle_east/6442663.stm 英廣替加沙記者憂心] - 英國廣播公司</ref>。他畢業於蘇格蘭[[鄧迪大學|鄧迪大學]]英文与政治學系[[碩士|碩士]]<ref>[http://news.bbc.co.uk/2/hi/middle_east/6443667.stm 在加沙與風險共存] - 英國廣播公司</ref>，並取得[[卡迪夫大學|卡迪夫大學]]的[[新聞學|傳播系]]文憑<ref>[http://thescotsman.scotsman.com/uk.cfm?id=429362007 綁架英廣記者在生但下落不明] {{Wayback|url=http://thescotsman.scotsman.com/uk.cfm?id=429362007 |date=20070822221749 }} - 蘇格蘭人報</ref> 他曾在蘇格蘭[[杜拉|杜拉]]學院進修。<ref>。[http://www.nzherald.co.nz/section/2/story.cfm?c_id=2&objectid=10433220 加沙忘命爭奪中記者的角色] - [[新西蘭|紐西蘭]]先鋒報</ref>

== 新聞工作 ==
阿倫·莊士敦1991年加入英國廣播公司成為駐外特派員, 曾經駐守阿富汗[[喀布爾|喀布爾]]和烏茲別[[克塔什干|克塔什干]]。<ref>[http://news.bbc.co.uk/2/hi/middle_east/6442663.stm ] - 英國廣播公司</ref> [[塔利班|塔利班]]在阿富汗執政期間, 他正在阿富汗工作。<ref>[http://news.bbc.co.uk/2/hi/middle_east/6443667.stm ] - 英國廣播公司</ref> 他原本是英國廣播公司中東新聞局駐加沙全職記者, 駐守期為期3年直至2007年4月1日為止。被綁架之前, 是西方大型傳媒機構唯一的外國記者。<ref>[http://news.bbc.co.uk/2/hi/middle_east/6442663.stm ] - 英國廣播公司</ref><ref>[http://www.cnn.com/2007/WORLD/meast/03/12/gaza.journalist/index.html 英廣憂心加沙特派員已被綁架] - 美國有線電視新聞網</ref><ref>[http://www.guardian.co.uk/israel/Story/0,,2032385,00.html 蒙面槍手加沙街頭綁架英國記者] - 英國衛報</ref>

由於莊士敦對加沙當地情況熟悉, 成為其他記者信賴和依靠的好幫手, 英國廣播公司稱他為一位富有經驗且受尊敬的記者。<ref>[http://www.guardian.co.uk/israel/Story/0,,2033392,00.html 哈馬斯政府為綁架英廣記者行動] - 英國衛報</ref> 他負責的一輯關於塔利班政權以後的人民生活的特輯獲得[[英國新力電台學院|英國新力電台學院]]銅獎。<ref>[http://www.radioawards.org/winners03/win03_b2.htm 英國新力電台學院得獎者] {{webarchive|url=https://www.webcitation.org/5NzCj4AEx?url=http://www.radioawards.org/winners03/win03_b2.htm |date=2007-04-10 }} - 英國電台學院</ref> [[巴勒斯坦|巴勒斯坦]]資訊部長 Mustafa Barghouti 稱呼莊士敦為「巴勒斯坦人民的朋友」, 並指出他「為我們作出過不少功勞」。<ref>[http://news.bbc.co.uk/2/hi/middle_east/6543773.stm 環球廣播聯盟懇求悉放記者] - 英國廣播公司</ref>

== 綁架 ==
2007年3月12日, 莊士敦駕車往[[耶路撒冷|耶路撒冷]]看完牙醫後<ref>[http://www.guardian.co.uk/israel/Story/0,,2033392,00.html ] - 英國衛報</ref> 進入加沙<ref>[http://www.cnn.com/2007/WORLD/meast/03/12/gaza.journalist/index.html ] - 美國有線電視新聞網</ref>, 工作完畢回家期間, 他的私家車被發現在街上被棄置。 <ref>[http://news.bbc.co.uk/2/hi/middle_east/6442663.stm ] - 英國廣播公司</ref> 他的一張名片在現場被發現, 證明他本人曾在車上。<ref>[http://www.guardian.co.uk/israel/Story/0,,2032741,00.html 失蹤英廣記者了無音訊] - 英國衛報</ref> 當他沒有向英國廣播公司定期用電話滙報後, 英國廣播公司知道他失蹤。<ref>[http://www.guardian.co.uk/israel/Story/0,,2033392,00.html ] - 英國衛報</ref>

根據巴勒斯坦警方說法, 有4名持械男子在莊士敦汽車附近出現<ref>[http://news.bbc.co.uk/2/hi/middle_east/6444473.stm ] - 英國廣播公司</ref>, 後來被持槍者綁架離開。<ref>[http://www.metimes.com/storyview.php?StoryID=20070313-054435-8945r 巴勒斯坦人搜尋綁架英廣記者] {{webarchive|url=https://web.archive.org/web/20070929083733/http://www.metimes.com/storyview.php?StoryID=20070313-054435-8945r |date=2007-09-29 }} - 中東時報</ref> 巴勒斯坦政府隨即宣佈進入緊急狀態, 並設立路障找尋失蹤莊士敦。<ref>[http://news.bbc.co.uk/2/hi/middle_east/6444473.stm 失蹤英廣記者搜索繼續] - 英國廣播公司</ref>

== 各界反應 ==
2007年3月19日，阿倫的父親發表聲明說：“不能這樣對待一個巴勒斯坦人民的朋友，我想對劫持阿倫的人說的是‘請釋放我的孩子，現在，就今天。’”
2007年3月20日，英國外交大臣馬各列特向巴勒斯坦主席阿巴斯提出幫助找尋阿倫的要求。
2007年3月24日，英國主教喬納森稱希望阿倫得到立即釋放，并為之祈禱。
2007年3月26日，英國廣播公司在加沙和倫敦的電視中心之間建立衛星連線，方便員工及時交換關于阿倫的信息。
2007年4月2日，巴勒斯坦記者聯合會在加沙開始了為期3天的示威，要求巴勒斯坦政府加大力度搜尋阿倫下落，并以停止報道政府相關新聞施壓。由300名電視界名人簽名的廣告刊登在英國衛報上，要求阿倫的立即釋放。
2007年4月12日，由英國廣播公司、有線新聞網、天空新聞和半島電視臺英文頻道聯合制作特別節目，要求阿倫的立即釋放，也旨在向大眾宣傳新聞工作的危險性，時長約25分鐘。
<ref>[http://news.bbc.co.uk/2/hi/middle_east/6518873.stm 阿倫·莊士敦失蹤時間表] - 英國廣播公司</ref>
