'''愛德華·-{zh-hant:艾倫;zh-hans:阿伦;}-·“艾德”·哈里斯'''，{{lang-en|Edward Allen "Ed" Harris}}，{{bd|1950年|11月28日|||catIdx=Harris, Edward}}），[[美國|美國]]演員、製片、導演與編劇。曾被提名四次[[奧斯卡金像獎|奧斯卡金像獎]]。

== 早期生活 ==
哈里斯出生於[[紐澤西州|紐澤西州]]的Tenafly地區，為旅遊經紀Margaret與Robert L. Harris之子（其父曾為20年代樂團「Fred Waring」的[[和聲歌手|和聲]]，也在[[芝加哥藝術學院|芝加哥藝術學院]]的書店工作）。<ref>http://movies.yahoo.com/movie/contributor/1800012404/bio</ref> 。他家人還有一個哥哥Robert與一個弟弟Spencer。哈里斯就是在這個[[中產階級|中產階級]]的基督教家庭中被撫養長大的。<ref>{{cite web |url=http://www.tiscali.co.uk/entertainment/film/biographies/ed_harris_biog/2 |title=存档副本 |accessdate=2007-10-17 |deadurl=yes |archiveurl=https://web.archive.org/web/20070820105920/http://www.tiscali.co.uk/entertainment/film/biographies/ed_harris_biog/2 |archivedate=2007-08-20 }}</ref>他在1969年自高中畢業，就讀高中時期他不僅參與足球隊，也在高中的最後一年當上隊長，在學校裡是一位明星運動員並且獲得了[[哥倫比亞大學|哥倫比亞大學]]的體育獎學金。<ref>Rohan, Virginia. == 演藝生涯 ==
哈里斯第一個重要的電影角色是在《Borderline》中演出的[[查理士·布朗遜|查理士·布朗遜]]一角。在電影《Knightriders》中他則扮演了一位摩托車特技騎士。在1983年他成為了一位大明星，在電影《[[太空先鋒|太空先鋒]]》中扮演[[美國國家航空暨太空總署|美國國家航空暨太空總署]]（NASA）的[[太空人|太空人]] [[約翰·葛倫|約翰·葛倫]]；1996年他以扮演《[[阿波羅13號_(電影)|阿波羅13號]]》中的[[NASA|NASA]]太空任務指導員的角色[[金·克蘭茲|金·克蘭茲]]，獲得[[奧斯卡金像獎|奧斯卡金像獎]]的提名。繼而在1999年、2001年與2003年又陸續獲得[[奧斯卡金像獎|奧斯卡金像獎]]的提名推崇，提名影片分別為《[[楚門的世界|楚門的世界]]》、《[[波拉克與他的情人|波拉克與他的情人]]》與《[[時時刻刻|時時刻刻]]》。近來他則出現在[[大衛·柯能堡|大衛·柯能堡]]的電影《[[暴力效應|暴力效應]]》中演出一個復仇心很重的黑幫份子。也在演員[[班·艾佛列克|班·艾佛列克]]所執導的電影《[[失蹤人口|失蹤人口]]》當中與[[凱西·艾佛列克|凱西·艾佛列克]]與[[摩根·弗里曼|摩根·弗里曼]]攜手演出。

哈里斯對於導演工作也十分有興趣，他在2001年執導了第一部電影《波拉克與他的情人》。他也在1996年與2005年分別演出了由電視劇改編的電影《[[荒野情天|荒野情天]]》與《[[失落的小鎮|失落的小鎮]]》。

哈里斯亦活躍於舞台劇的演出，他曾在[[紐約|紐約]]的公共劇院演出[[Neil_LaBute|Neil LaBute]]製作的一人戲劇－"Wrecks"。本劇在[[科克|科克]]的 "Everyman 劇院"進行首演，哈里斯也因這個角色獲得許多重要的獎項提名。

2015年，他與[[連恩·尼遜|連恩·尼遜]]、[[喬爾·金納曼|喬爾·金納曼]]共同主演了動作片《[[一夜狂奔|一夜狂奔]]》。

== 私人生活 ==
他與演員[[Amy_Madigan|Amy Madigan]]在1983年結婚。他們誕下了一名女兒名為Lily。
