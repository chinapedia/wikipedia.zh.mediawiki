'''艾度·迪華特'''（Edo de Waart，{{bd|1941年|6月1日||}}）是[[荷蘭|荷蘭]]近代最卓越的[[指揮家|指揮家]]。他以指揮[[樂團|樂團]]及[[歌劇|歌劇]]節目享負盛名，並將部分二流樂團轉變為一流的樂團，因而被冠以「樂團建造者」之美譽。

== 在學時期 ==
迪華特曾於[[荷蘭|荷蘭]][[阿姆斯特丹|阿姆斯特丹]]市的史韋琳克音樂學院(Sweelinck Conservatory)學習[[雙簧管|雙簧管]]、[[鋼琴|鋼琴]]及[[指揮學|指揮學]]，直至1962年畢業；畢業後之翌年(1963年)，他更被[[阿姆斯特丹|阿姆斯特丹]][[皇家音樂廳樂團|皇家音樂廳樂團]]委任為助理首席雙簧管手。23歲時(1964年)，迪華特更於[[紐約市|紐約市]]的[[德米特裡·米特羅波洛斯|德米特裡·米特羅波洛斯]]指揮大賽中奪獎。奪獎後，他正式加盟[[紐約愛樂樂團|紐約愛樂樂團]]之助理指揮，並在一年時間擔當[[倫納德·伯恩斯坦|倫納德·伯恩斯坦]]之得力助手。

== 指揮時期 ==
迪華特於返回荷蘭後，隨即被[[伯納德·海廷克|伯納德·海廷克]]委任為[[阿姆斯特丹|阿姆斯特丹]][[皇家音樂廳樂團|皇家音樂廳樂團]]助理指揮。1967年，荷蘭管樂合奏組及[[鹿特丹愛樂樂團|鹿特丹愛樂樂團]]委任迪華特為其指揮，他更於1973至1979年間擔任音樂總監。

迪華特於1975年帶領[[三藩市交響樂團|三藩市交響樂團]]作首演，一年後擔任該樂團的總客席指揮，1977至1985年間更擔任該樂團之音樂總監。到1986至1995年間，他更出任[[明尼蘇達樂團|明尼蘇達樂團]]之總指揮。

迪華特歷經多次巡迴演出後，於1989年返回荷蘭，並被[[荷蘭電台愛樂樂團|荷蘭電台愛樂樂團]]委任為音樂總監，直至1994年辭職為止；其後更出任該樂團之[[桂冠指揮|桂冠指揮]]。

迪華特曾於世界多個知名樂團擔任客席指揮，如[[柏林愛樂樂團|柏林愛樂樂團]]、[[萊比錫布業大廳樂團|萊比錫布業大廳樂團]]、[[愛樂管弦樂團|愛樂管弦樂團]]、[[皇家愛樂樂團|皇家愛樂樂團]]、[[瑞士諾曼第樂團|瑞士諾曼第樂團]]、[[波士頓交響樂團|波士頓交響樂團]]、[[克利夫蘭管弦樂團|克利夫蘭管弦樂團]]、[[洛杉磯愛樂樂團|洛杉磯愛樂樂團]]及[[芝加哥交響樂團|芝加哥交響樂團]]。迪華特更熱衷於演繹[[現代音樂|現代音樂]]，如擔任[[约翰·库利奇·亚当斯|约翰·亚当斯]]歌劇《[[尼克松在中国_(歌剧)|尼克遜在中國]]》的首演指揮；此外，他更為[[斯蒂夫·莱奇|斯蒂夫·莱奇]](Steve Reich)的《管樂、弦樂及鍵盤之變奏曲(Variations for Winds, Strings and Keyboards)》及其他作於三藩市的作品作錄音。

作為歌劇節目之經常性指揮，迪華特曾於英、美等地指揮歌劇，如在1971年於[[桑特菲歌劇院|桑特菲歌劇院]]作首演，1975年於[[休士頓大歌劇院|休士頓大歌劇院]]作指揮，1976年於[[皇家歌劇院|皇家歌劇院]]（高文花園）作指揮、及於1976年於[[拜羅伊特節日劇院|拜羅伊特節日劇院]]作指揮。1970年，迪華特更擔任[[荷蘭市立歌劇院|荷蘭市立歌劇院]]之經常性指揮；1980年，他在[[三藩市歌劇院|三藩市歌劇院]]指揮《[[尼伯龍根的指環|尼伯龍根的指環]]》。

近年來，他更於巴黎[[巴士底歌劇院|巴士底歌劇院]]指揮《[[玫瑰騎士|玫瑰騎士]](Der Rosenkavalier)》、及在[[大都會歌劇院|大都會歌劇院]]指揮《[[魔笛|魔笛]]》及《[[費加羅的婚禮|費加羅的婚禮]]》。此外，他更在[[阿姆斯特丹|阿姆斯特丹]]市指揮《卡芭諾娃(Katya Kabanova)》、《[[維特|維特]]》、《[[彼得格林|彼得格林]]》、《[[蝴蝶夫人|蝴蝶夫人]]》及《[[特洛伊人|特洛伊人]]》。其他作品更包括於[[日內瓦歌劇院|日內瓦歌劇院]]上演《[[博理斯·古杜諾夫|博理斯·古杜諾夫]]》及於[[薩爾茨堡音樂節|薩爾茨堡音樂節]]上演《[[費加羅的婚禮|費加羅的婚禮]]》。迪華特亦自1995年起，指揮[[華格納|華格納]]作品，及在2000雪梨奧運藝術節中以演奏《[[諸神的黃昏|諸神的黃昏]]》推向最高潮。2005年，他更領導[[香港管弦樂團|香港管弦樂團]]指揮[[理查·史特勞斯|理查·史特勞斯]]名作《[[莎樂美_(歌劇)|莎樂美]]》及《[[厄勒克特拉_(歌剧)|厄勒克特拉]]》，並贏得好評。

2002年3月，迪華特宣告於2004年將辭去自1999年擔任之[[荷蘭歌劇院|荷蘭歌劇院]]總指揮一職。宣告辭官時，迪華特更提及期望抽更多時間照顧兩名子女，然而，他接受荷蘭報章《忠誠報(Trouw)》專訪時，提及他與荷蘭歌劇院導演[[皮埃爾·奧迪|皮埃爾·奧迪]](Pierre Audi)排演歌劇《[[羅恩格林|羅恩格林]]》，及與[[羅柏特·韋爾松|羅柏特·韋爾松]](Robert Wilson)籌劃《[[蝴蝶夫人|蝴蝶夫人]]》時，曾挑起不少的爭議，並錯過了解人道及方向感。

== 錄音及曾獲之榮譽 ==
迪華特的錄音數量繁多，曾跟不少知名樂團合作灌錄專輯，包括與[[荷蘭電台愛樂樂團|荷蘭電台愛樂樂團]]合作灌錄全套[[古斯塔夫·馬勒|馬勒]]交響曲，及近期與該樂團所灌錄之全套[[拉赫曼尼諾夫|拉赫曼尼諾夫]]管弦樂作品及兩張[[華格納|華格納]]管弦樂作品專輯。

迪華特亦曾獲[[荷蘭政府|荷蘭政府]]頒發[[雄獅|雄獅]][[勳位|勳位]]，及獲[[澳洲|澳洲]]政府頒發非軍事最高榮譽—澳洲[[勳章|勳章]]，表揚他成功將[[雪梨交響樂團|雪梨交響樂團]]提升至世界級地位，提升澳洲及[[雪梨_(城市)|雪梨]]的音樂和藝術生活。近期他又獲[[香港演藝學院|香港演藝學院]]頒發榮譽院士，讚揚他對音樂界的貢獻，特別是他對本地下一代[[音樂家|音樂家]]的培育。
