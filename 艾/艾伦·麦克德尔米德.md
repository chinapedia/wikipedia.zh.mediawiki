'''艾伦·格雷厄姆·麦克德尔米德'''，[[:en:Order_of_New_Zealand|ONZ]]（{{lang-en|'''Alan Graham MacDiarmid'''}}，{{bd|1927年|4月14日|2007年|2月7日|catIdx=MacDiarmid, Alan}}），或译“'''艾伦·麦克迪尔米德'''”，[[化学家|化学家]]，具有[[美国|美国]]和[[新西兰|新西兰]][[双重国籍|双重国籍]]<ref name="zju">{{cite web |url=http://www.zju.edu.cn/zdxw/jd/read.php?recid=5861 |title=科学研究在于人 —— 艾伦·麦克迪尔米德访谈录 |publisher=[[浙大|浙大]]求是新闻网（原载《[[文汇报_(上海)|文汇报]]》） |date=2003年12月8日 |accessdate=2008年11月25日 |language=zh |deadurl=yes |archiveurl=https://web.archive.org/web/20031211000358/http://www.zju.edu.cn/zdxw/jd/read.php?recid=5861 |archivedate=2003年12月11日 }}</ref><ref name="nobel_main">{{cite web |url=http://nobelprize.org/nobel_prizes/chemistry/laureates/2000/index.html |title=The Nobel Prize in Chemistry 2000 |publisher=[[诺贝尔基金会|诺贝尔基金会]] |date= |accessdate=2008年11月25日 |language=en }}</ref><ref name="autobio">{{cite web |url=http://nobelprize.org/nobel_prizes/chemistry/laureates/2000/macdiarmid-autobio.html |title=Alan G. == 童年生活 ==
麦克德尔米德出生于[[新西兰|新西兰]]的[[马斯特顿|马斯特顿]]，家中还有两个哥哥、一个姐姐和一个妹妹。<ref name="autobio"/>他的家境并不好，小学和中学时他先后兼职送过牛奶和报纸。<ref name="autobio"/>当时的[[经济大萧条|经济大萧条]]对新西兰造成了很严重的影响，生活十分困难，于是他的父母决定举家搬迁到离[[惠灵顿|惠灵顿]]不远的下赫特（[[:en:Lower_Hutt|Lower Hutt]]），以求获得更多工作机会。<ref name="autobio"/>大约10岁的时候，父亲的一本旧教科书激发了麦克德尔米德对[[化学|化学]]的兴趣，后来他又在下赫特图书馆找到一本《少年化学家》（''The Boy Chemist''），通过不断续借，阅读了大约一年，做完书中绝大多数实验。<ref name="zju"/><ref name="autobio"/>他在赫特谷中学（Hutt Valley High School）接受了三年中学教育。<ref name="autobio"/>

== 学术生涯 ==
1943年，16岁的麦克德尔米德通过[[新西兰大学|新西兰大学]]的入学考试，进入[[维多利亚大学学院|维多利亚大学学院]]（Victoria University College）{{efn|1=当时[[维多利亚大学学院|维多利亚大学学院]]从属于[[新西兰大学|新西兰大学]]，1961年新西兰大学解散后，维多利亚大学学院独立办学，并更名为“[[惠灵顿维多利亚大学|惠灵顿维多利亚大学]]”（Victoria University of Wellington）。}}学习。<ref name="nzedge">{{cite web |url=http://www.nzedge.com/heroes/macdiarmid.html |title=Alan MacDiarmid |publisher=nzedge.com |date= |accessdate=2008年11月25日 |language=en }}</ref>1944年，他开始在化学系实验室给教授充当助手，从此实现经济上的独立，无需家里的资助。<ref name="autobio"/><ref name="nzedge"/>1947年，完成[[学士学位|学士学位]]后，麦克德尔米德在维多利亚大学学院获得实验员职位，指导本科生进行实验，并继续攻读[[硕士学位|硕士学位]]。<ref name="nzedge"/>期间，他于1949年在《[[自然_(期刊)|自然]]》杂志上发表了自己的第一篇学术文章。<ref name="nzedge"/>1951年，以优异成绩完成硕士学位后，麦克德尔米德未成功申请到[[剑桥大学|剑桥大学]]的奖学金，但获得了[[美国国务院|美国国务院]]颁发的[[富尔布赖特奖学金|富尔布赖特奖学金]]，得以前往[[威斯康星大学麦迪逊分校|威斯康星大学麦迪逊分校]]攻读[[博士学位|博士学位]]。<ref name="autobio"/><ref name="nzedge"/>

在威斯康星大学期间，麦克德尔米德师从诺里斯·霍尔教授，研究[[无机化学|无机化学]]<ref name="nzedge"/>，于1952年和1953年先后取得硕士和博士学位<ref name="utd">{{cite web |url=http://www.utdallas.edu/nsm/chemistry/faculty/macdiarmid.html |title=Chemistry: Faculty: Alan G. MacDiarmid |publisher=得克萨斯大学达拉斯分校 |date= |accessdate=2008年11月25日 |language=en }}</ref>。其后，他又获得奖学金（New Zealand Shell Graduate Scholarship）资助，到英国剑桥大学研究硅氢化物，师从H. J. Emeléus教授<ref name="autobio"/>，于1955年取得剑桥大学博士学位<ref name="utd"/>。

从剑桥大学毕业后，麦克德尔米德到[[苏格兰|苏格兰]][[圣安德鲁斯大学|圣安德鲁斯大学]]短暂任教，随后于1956年转投[[宾夕法尼亚大学|宾夕法尼亚大学]]担任副教授<ref name="nobel_public">{{cite web|url=http://nobelprize.org/nobel_prizes/chemistry/laureates/2000/public.html |title=The Nobel Prize in Chemistry 2000 - Information for the Public |publisher=[[诺贝尔基金会|诺贝尔基金会]] |date=2000年10月10日 |accessdate=2008年11月25日 |language=en |deadurl=yes |archiveurl=https://web.archive.org/web/20111019144626/http://nobelprize.org/nobel_prizes/chemistry/laureates/2000/public.html |archivedate=2011年10月19日 }}</ref>。他在宾夕法尼亚大学工作50年，直到逝世，期间于1964年升任正教授，1988年被聘为“-{zh-hans:布兰查德; zh-hant:布蘭查德}-化学教授”（Blanchard Professor of Chemistry）。<ref name="nobel_public"/>2002年，麦克德尔米德加盟[[德州大学达拉斯分校|德州大学达拉斯分校]]纳米技术研究所，担任指导委员会主席，但仍保留在宾夕法尼亚大学的职务。<ref name="utdann">{{cite web |url=http://www.utdallas.edu/news/archive/2001/macdiarmidnano.htm |title=Nobel Prize-Winning Chemist to Become Distinguished Scholar in Residence at U.T. Dallas: Dr. Alan G. MacDiarmid to Head UTD NanoTech Institute Advisory Board |publisher=得克萨斯大学达拉斯分校 |date= |accessdate=2008年11月25日 |language=en }}</ref>从1994年起，麦克德尔米德与中国[[吉林大学|吉林大学]]长期保持密切的学术合作关系。<ref>{{cite web |url=http://www.jlu.edu.cn/newjlu/xxgk/dsj_2000.htm |title=2000年大事记 |publisher=[[吉林大学|吉林大学]] |date= |accessdate=2008年11月25日 |language=zh |deadurl=yes |archiveurl=https://web.archive.org/web/20081228101414/http://www.jlu.edu.cn/newjlu/xxgk/dsj_2000.htm |archivedate=2008年12月28日 }}</ref><ref>{{cite web|url=http://xiaobao.jlu.edu.cn/~xiaobao/0000/0022/html/xwc3.htm|title=刘中树校长致电祝贺我校名誉教授麦克迪尔米德获诺贝尔化学奖|author=辛文|publisher=[[吉林大学|吉林大学]]校报|date=|accessdate=2008年11月25日|language=zh}}{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>他于1999年被聘为吉林大学名誉教授，2004年起被聘为吉林大学教授。<ref name="jluchem">{{cite web |url=http://chem.jlu.edu.cn/newTeachers/show/yuanshi.htm |title=师资力量 |publisher=[[吉林大学|吉林大学]]化学学院 |date= |accessdate=2008年11月25日 |language=zh |deadurl=yes |archiveurl=https://web.archive.org/web/20080508213658/http://chem.jlu.edu.cn/newTeachers/show/yuanshi.htm |archivedate=2008年5月8日 }}</ref><ref>{{cite web |url=http://www.jlu.edu.cn/newjlu/xxgk/sdxw2004.htm |title=2004年学校十大新闻 |publisher=[[吉林大学|吉林大学]] |date= |accessdate=2008年11月25日 |language=zh |deadurl=yes |archiveurl=https://web.archive.org/web/20081228094851/http://www.jlu.edu.cn/newjlu/xxgk/sdxw2004.htm |archivedate=2008年12月28日 }}</ref>2001年11月12日，中国高校首个以诺贝尔奖获得者名字命名的实验室吉林大学麦克德尔米德实验室揭牌，麦克德尔米德任实验室学术委员会第一任主任。<ref name="jluagm">{{cite web |url=http://www.jlu.edu.cn/newjlu/xxgk/dsj_2001.htm |title=2001年大事记 |publisher=[[吉林大学|吉林大学]] |date= |accessdate=2008年11月25日 |language=zh |deadurl=yes |archiveurl=https://web.archive.org/web/20081228094218/http://www.jlu.edu.cn/newjlu/xxgk/dsj_2001.htm |archivedate=2008年12月28日 }}</ref>他还积极推动新西兰麦克德尔米德先进材料与纳米技术研究所以及中国[[三峡大学|三峡大学]]艾伦·麦克德尔米德再生能源研究所的建立。

== 主要贡献 ==
麦克德尔米德最著名的研究成果是发现与研究[[导电聚合物|导电聚合物]]，即[[导电塑料|导电塑料]]，也正是凭借在这一领域的开创性贡献，他与[[艾伦·黑格|艾伦·黑格]]、[[白川英树|白川英树]]一起获得2000年诺贝尔化学奖。<ref name="nobel_main"/><ref>http://www.ch.ntu.edu.tw/nobel/2000.html</ref>1973年，他开始研究(SN)x，即[[聚氮化硫|聚氮化硫]]，一种具有金属导电性的[[聚合物|聚合物]]。<ref name="bio">{{cite web |url=http://www.sas.upenn.edu/~macdiarm/Bio-Short.html |title=Biographical Sketch: ALAN G. MACDIARMID |publisher=[[宾夕法尼亚大学|宾夕法尼亚大学]] |date= |accessdate=2008年11月27日 |language=en }}</ref>1975年，他在[[日本|日本]][[京都大学|京都大学]]做访问教授期间，有一次到[[东京工业大学|东京工业大学]]进行学术交流，白川英树向他介绍自己合成的银色膜状[[聚乙炔|聚乙炔]]。<ref name="autobio"/>麦克德尔米德随即邀请白川英树到宾夕法尼亚大学进行访问研究。他们尝试通过增大聚乙炔的纯度来提高它的导电性，实验结果却与预期相反。联系到之前在聚氮化硫中加入[[溴|溴]]可将导电性提高10倍，这时他们意识到，聚乙炔里的杂质可能起到[[搀杂剂|搀杂剂]]的作用，从而提高聚乙炔的导电性。因此，他们向聚乙炔中加入了溴，很快在室温下就将导电性提高了数百万倍。<ref name="autobio"/>他们又与当时在宾夕法尼亚大学物理系的艾伦·黑格进行合作，进行物理性质方面的研究。<ref name="autobio"/>他们三人合作的研究成果拥有广泛的应用，例如[[蓄电池|蓄电池]]、电磁屏蔽、防静电扩散、抗腐蚀、弹性塑料[[晶体管|晶体管]]和[[电极|电极]]、[[电致发光|电致发光]]聚合物显示等。<ref name="bio"/>

麦克德尔米德长期致力于导电聚合物的研究，尤其是在聚乙炔和聚苯胺的合成、[[化学性质|化学性质]]、搀杂、[[电化学|电化学]]特性、导电性、[[磁性|磁性]]、[[光学|光学]]性质方面进行了细致的工作。<ref name="jluchem"/><ref name="bio"/>他单独或与人合作发表了大约600篇学术论文，申请到25项[[专利|专利]]。<ref name="zju"/><ref name="drexel">{{cite web |url=http://www.drexel.edu/univrel/commencement/2005/Degree_Recipients.html |title=Commencement 2005: Honorary Degree Recipients |publisher=德雷克塞尔大学 |date= |accessdate=2008年11月25日 |language=en |deadurl=yes |archiveurl=https://web.archive.org/web/20070607203023/http://www.drexel.edu/univrel/commencement/2005/Degree_Recipients.html |archivedate=2007年6月7日 }}</ref>

== 生活 ==
麦克德尔米德在威斯康星大学求学期间，曾担任校内最大的学生组织“国际俱乐部”（International Club）的主席。在国际俱乐部的一次舞会上，他结识了后来的妻子Marian Mathieu。麦克德尔米德在英国学习期间，两人在剑桥大学西德尼·苏塞克斯学院的教堂内举行了婚礼。<ref name="nzedge"/>两人相处了36年后，Marian在1990年去世。这段婚姻给他带来3个女儿、1个儿子和9个孙子、孙女。<ref name="nytimes">{{cite web |url=http://www.nytimes.com/2007/02/08/obituaries/08macdiarmid.html?ex=1328590800&en=58bae3f4a073b70b&ei=5090&partner=rssuserland&emc=rss |title=Alan MacDiarmid, 79, Who Won Nobel for Work With Plastic, Dies |author=KENNETH CHANG |publisher=[[纽约时报|纽约时报]] |date=2007年2月8日 |accessdate=2008年11月25日 |language=en }}</ref>自1991年起，Gayl Gentile成为麦克德尔米德的新伴侣<ref name="nzedge"/>，并于2005年完婚<ref name="nytimes"/>。麦克德尔米德是一位[[天然主义|天然主义]]者，热衷于晒太阳和[[滑水|滑水]]。<ref name="nzherald">{{cite web |url=http://www.nzherald.co.nz/nz/news/article.cfm?c_id=1&objectid=10423031 |title=The Nobel-prize winning naturist - Alan MacDiarmid remembered |author=Kent Atkinson |publisher=新西兰先驱报 |date=2007年2月9日 |accessdate=2008年11月25日 |language=en }}</ref>

2007年2月7日，麦克德尔米德在位于[[费城|费城]]郊区的家中不慎跌倒，经抢救无效，在特拉华县纪念医院逝世。<ref name="utdfyi">{{cite web |url=http://www.utdallas.edu/research/fyi/070216/university.html#d |title=Alan MacDiarmid Nobel-winning Chemist Dies |publisher=得克萨斯大学达拉斯分校 |date=2007年2月16日 |accessdate=2008年11月25日 |language=en }}</ref>据他妻子Gayl Gentile介绍，麦克德尔米德晚年患上骨髓增生异常综合征（myelodysplastic syndrome），一种类似[[白血病|白血病]]的疾病。当时他的生命只剩下几个星期，于是准备最后一次返回新西兰拜访亲友，没料到却在即将出门搭乘去往新西兰的航班之前发生了意外。<ref name="nytimes"/><ref name="utdfyi"/><ref name="penn_gazette">{{cite web |url=http://www.upenn.edu/gazette/0307/gaz02.html |title=Nobel-Winner MacDiarmid Dies |publisher=[[宾夕法尼亚大学|宾夕法尼亚大学]]校报. 2007年第3期 |date= |accessdate=2008年11月25日 |language=en }}</ref>
