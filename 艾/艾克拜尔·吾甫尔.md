'''艾克拜尔·吾甫尔'''（{{bd|1959年|1月||}}），[[维吾尔族|维吾尔族]]，[[新疆|新疆]][[阿克苏|阿克苏]]人，[[中华人民共和国|中华人民共和国]]官员<ref name=jj>{{Cite web |url = http://cctvenchiridion.cctv.com/special/C20768/20080314/101961.shtml |title = 艾克拜尔·吾甫尔简介 |author =  |date =2008-03-14  |publisher =央视网  |language =  |accessdate =  |quote =  }}</ref>。

==生平==
1975年8月至1978年9月，在[[新疆维吾尔自治区|新疆维吾尔自治区]][[阿克苏县|阿克苏县]]依杆其公社接受贫下中农再教育。1978年9月至1981年7月，在[[阿克苏教育学院|阿克苏地区师范学校]][[汉语|汉语]]专业学习。1981年7月至1985年9月，任中共阿克苏县委办公室翻译。1985年9月至1987年7月，在[[中央民族学院|中央民族学院]]民语翻译专业学习。1987年7月至1988年8月，任中共[[阿克苏市|阿克苏市]]委办公室翻译<ref name=jj/>。

1988年8月至1993年3月，任中共阿克苏市委办公室副主任兼市委机关党委书记。1993年3月至1995年9月，任中共阿克苏市委副书记。其间，1993年9月至1994年1月在[[天津大学|天津大学]]经济管理系新疆班学习。1995年9月至1996年1月，任中共[[温宿县|温宿县]]委常委、副县长、代理县长。其间，1995年5月至11月在[[山东省|山东省]]中共[[招远县|招远县]]委挂职任副书记。1996年1月至1997年9月，任中共温宿县委常委、县长。1997年9月至1999年12月，任中共温宿县委副书记、县长。其间，1998年9月至1999年7月在[[中共中央党校|中央党校]]新疆班学习<ref name=jj/>。

1999年12月至2000年11月，任中共阿克苏市委副书记、市长。2000年11月至2003年3月，任中共阿克苏地委委员。其间，1998年9月至2001年7月，在中央党校研究生院经济学专业在职研究生班学习，获得中央党校研究生学历。2003年3月至2005年2月，任中共阿克苏地委副书记。2005年2月至2012年3月，任中共[[喀什地区|喀什]]地委副书记、行署专员<ref name=jj/>。

2012年3月，任[[新疆维吾尔自治区人民政府|新疆维吾尔自治区人民政府]]副秘书长、政府办公厅党组成员，免去其中共喀什地委副书记、行署专员职务<ref>{{Cite web |url = http://district.ce.cn/newarea/sddy/201207/04/t20120704_23460997.shtml |title = 新疆调整沙尔合提·阿汗、塔里哈提·吾逊、麦尔丹·木盖提等职务 |author =  |date =2012-07-04  |publisher =中国经济网  |language =  |accessdate =  |quote =  }}</ref>。2013年3月至2017年1月，任[[新疆维吾尔自治区农业厅|新疆维吾尔自治区农业厅]][[党组|党组]]副书记、厅长<ref>{{Cite web |url =http://news.sina.com.cn/c/2013-03-31/104026693904.shtml  |title =新疆维吾尔自治区人民代表大会常务委员会任免名单  |author =  |date = 2013-03-31 |publisher =新浪  |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url =http://news.hexun.com/2017-01-04/187607086.html  |title = 新疆维吾尔自治区人民代表大会常务委员会任免名单 |author =  |date = 2017-01-04 |publisher = 和讯 |language =  |accessdate =  |quote =  }}</ref>。

2008年起，担任[[第十一届全国人民代表大会|第十一届全国人民代表大会]]新疆地区代表<ref>{{cite web|author=新华网|title=第十一届全国人大代表|url=http://news.xinhuanet.com/ziliao/2008-03/03/content_7708230.htm?node=36628|publisher=新华网|accessdate=2014-01-07}}</ref><ref>{{cite news|title=全国人大代表信息-艾克拜尔·吾甫尔|url=http://www.npc.gov.cn/delegate/viewDelegate.action?dbid=112607|accessdate=2013-04-05|newspaper=全国人大网}}</ref>。
