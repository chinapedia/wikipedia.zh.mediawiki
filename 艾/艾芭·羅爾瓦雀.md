'''艾芭·羅爾瓦雀'''（{{lang-it|Alba Rohrwacher}}，{{bd|1979年|2月17日|catIdx=R}}）是義大利女演員。

== 生平 ==
艾芭·羅爾瓦雀出生於[[佛羅倫斯|佛羅倫斯]]，父親是德國人，母親則是義大利人。2009年，她因演出《{{link-it|喬凡娜的父親|Il papà di Giovanna}}》而獲得[[意大利电影金像奖|大衛獎最佳女主角]]；2014年，她以《[[飢餓的心|飢餓的心]]》於[[第71屆威尼斯影展|第71屆威尼斯影展]]獲得[[最佳演員獎_(威尼斯電影節)|最佳女演員獎]]。<ref name="BBCAwards">{{cite web |url=http://www.bbc.co.uk/news/entertainment-arts-29098103 |title=Roy Andersson film scoops Venice Golden Lion award |accessdate=7 September 2014 |work=BBC News}}</ref>

2016年，她受邀擔任[[第66屆柏林影展|第66屆柏林影展]]評審團成員。<ref name="Jury">{{Cite web |url=http://www.berlinale.de/en/presse/pressemitteilungen/alle/Alle-Detail_27165.html |title=Berlinale 2015: International Jury |accessdate=27 January 2015 |work=Berlinale |deadurl=yes |archiveurl=https://web.archive.org/web/20150215151257/http://www.berlinale.de/en/presse/pressemitteilungen/alle/Alle-Detail_27165.html |archivedate=2015年2月15日 }}</ref><ref name="full-jury">{{cite web |url=http://www.berlinale.de/en/das_festival/preise_und_juries/preise_internationale_jury/index.html |title=International Jury |accessdate=2 Feb 2016 |work=Berlinale}}</ref>
