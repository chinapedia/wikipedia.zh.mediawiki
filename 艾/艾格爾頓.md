'''艾格爾頓'''（{{Lang-en|'''Argleton'''}}）是一個於[[Google地圖|Google地圖]]和[[Google地球|Google地球]]上出現、但真實世界並不存在的[[英國|英國]][[鬼魅城鎮|鬼魅城鎮]]<ref>{{Cite web |url= http://www.abc.es/20091104/ciencia-tecnologia-tecnologia-internet/ciudad-fantasma-google-maps-200911041348.html |title= Argleton, el pueblo que sólo existe en Google Maps |author= Actualizado Miércoles |date= 2009年11月4日 |publisher= [[ABC_News|ABC News]] |language= es |accessdate= 2013年12月6日 |quote=  }}</ref><ref name = "Lenta.ru">{{Cite web |url= http://lenta.ru/news/2009/11/01/argleton/ |title= На Google Maps нашли несуществующий город |author=  |date= 2009年11月1日 |publisher= [[Lenta.ru|Lenta.ru]] |language= ru |accessdate= 2013年12月6日 |quote=  }}</ref>，其標記處位在[[英格蘭|英格蘭]]西北部[[西蘭開夏郡|西蘭開夏郡]][[奧姆斯柯克|奧姆斯柯克]]南方的{{Link-en|A59公路|A59 road}}附近，但實際上該地點其實是隸屬{{Link-en|奧頓 (蘭開夏郡)|Aughton, Lancashire|奧頓}}[[民政教區|民政教區]]的寬廣空地<ref name = "Rebecca Lefort">{{Cite web |url= http://www.telegraph.co.uk/technology/google/6474746/Mystery-of-Argleton-the-Google-town-that-only-exists-online.html |title= Mystery of Argleton, the 'Google' town that only exists online |author= Rebecca Lefort |date= 2009年10月31日 |publisher= 《[[每日電訊報|每日電訊報]]》 |language= en |accessdate= 2013年7月31日 |quote=  }}</ref><ref name = "Leo Hickman">{{Cite web |url= http://www.theguardian.com/technology/2009/nov/03/google |title= Welcome to Argleton, the town that doesn't exist |author= {{Link-en|里歐·希克曼|Leo Hickman}} |date= 2009年11月3日 |publisher= 《[[衛報|衛報]]》 |language= en |accessdate= 2013年7月31日 |quote=  }}</ref>。儘管如此許多網際網路訊息服務公司仍直接採用了[[Google|Google]]所提供的資料內容，這包括有艾格爾頓為實際存在城鎮並且被劃入[[英國郵區編號|L39郵區]]之中等訊息。在這般情況下，許多網站紛紛提供了關於艾格爾頓的房地產資訊、就業服務以及當地天氣等公開內容。但雖然其所提供的相關人士、公司以及服務皆為實際存在的事物，然而資料來源則是源自於其他郵區編號相同的人口密集地區而非「艾格爾頓」本身<ref name = "Rebecca Lefort"/><ref name = "Arjun Ramachandran">{{Cite web |url= http://www.smh.com.au/technology/technology-news/argleton-the-phantom-town-that-google-created-20091104-hwfh.html |title= Argleton: the phantom town that Google created |author= Arjun Ramachandran |date= 2009年11月4日 |publisher= 《[[雪梨晨鋒報|雪梨晨鋒報]]》 |language= en |accessdate= 2013年7月31日 |quote=  }}</ref>。到了2010年1月30日時，Google地圖上已經撤除了關於艾格爾頓的標記<ref name = "Zoe Kleinman">{{Cite web |url= http://www.bbc.co.uk/news/10117573 |title= FutureEverything gathers technology's avant garde |author= Zoe Kleinman |date= 2010年5月18日 |publisher= [[BBC新聞|BBC新聞]] |language= en |accessdate= 2013年12月6日 |quote=  }}</ref>。

== 報導 ==
最早是由於附近的[[蘭開夏郡|蘭開夏郡]][[邊山大學|邊山大學]]擔任網路服務主管的[[迈克·诺兰|迈克·诺兰]]（Mike Nolan）發現這項問題，並且在2008年9月時張貼在自己部落格中<ref name = "Leo Hickman"/><ref>{{Cite web |url= http://blogs.edgehill.ac.uk/webservices/2008/09/09/google-renames-village/ |title= Google Renames Village |author= [[迈克·诺兰|迈克·诺兰]]（Mike Nolan） |date= 2008年9月9日 |publisher= [[邊山大學|邊山大學]] |language= en |accessdate= 2013年12月6日 |quote=  }}</ref><ref>{{Cite web |url= http://blogs.edgehill.ac.uk/webservices/tag/argleton/ |title= Return to Argleton |author= [[迈克·诺兰|迈克·诺兰]]（Mike Nolan） |date= 2010年9月20日 |publisher= [[邊山大學|邊山大學]] |language= en |accessdate= 2013年7月31日 |quote=  }}</ref>。其中艾格爾頓在[[Google地圖|Google地圖]]和[[Google地球|Google地球]]上被標記在位於北緯53.543度、西經2.912度處，一旁鄰近位於[[西蘭開夏郡|西蘭開夏郡]]、約有8,000多人居住的[[奧姆斯柯克|奧姆斯柯克]]以及連絡[[利物浦|利物浦]]與[[約克|約克]]的{{Link-en|A59公路|A59 road}}。而Google亦提供了有關「艾格爾頓當地氣候」、「艾格爾頓房地產」以及「艾格爾頓就業資訊」等網頁資訊服務，這使得艾格爾頓這個虛構地名彷彿是普通的城鎮般擁有其相關資料。到了2009年年初的時候，诺兰的同事[[羅伊·貝菲爾德|羅伊·貝菲爾德]]（Roy Bayfield）決定進一步調查此事，並且親自走到Google地圖所標示的地區以查看該地是否有什麼特別之處。不過貝菲爾德在自己部落格評論時表示該地「看似正常」，並且推斷是由於受到[[魔幻寫實主義|魔幻寫實]]與{{Link-en|精神地理學|Psychogeography|精神地理}}等概念的影響而導致這個不存在地方的出現，而這個故事也一度被當地媒體提起<ref>{{Cite web |url= http://walkinghometo50.wordpress.com/2009/02/22/destination-argleton-visiting-an-imaginary-place/ |title= Destination: Argleton! Visiting an imaginary place |author= [[羅伊·貝菲爾德|羅伊·貝菲爾德]]（Roy Bayfield） |date= 2009年2月22日 |publisher= Walking Home to 50 |language= en |accessdate= 2013年12月6日 |quote=  }}</ref><ref>{{Cite web |url= http://www.osadvertiser.co.uk/news/ormskirk-news/2009/04/02/ormskirk-man-roy-bayfield-visits-the-bermuda-triangle-of-west-lancashire-argleton-in-aughton-80904-23286888/ |title= Ormskirk man Roy Bayfield visits the Bermuda Triangle of West Lancashire – Argleton in Aughton |author= Gemma Jaleel |date= 2009年4月2日 |publisher= Ormskirk & Skelmersdale Advertiser |language= en |accessdate= 2013年12月6日 |quote=  }}</ref>。

但是到了2009年11月時，有關不存在城鎮的消息突然被全球媒體大篇幅報導，而「Argleton」也隨即成為[[Twitter|Twitter]]上最受到歡迎的[[主題標籤|主題標籤]]<ref name = "Leo Hickman"/><ref name = "Gemma Jaleel">{{Cite web |url= http://www.osadvertiser.co.uk/news/ormskirk-news/2009/11/05/advertiser-leads-way-over-mystery-google-map-town-argleton-80904-25090055/ |title= Advertiser leads way over mystery Google Map town Argleton |author= Gemma Jaleel |date= 2009年11月5日 |publisher= Ormskirk & Skelmersdale Advertiser |language= en |accessdate= 2013年12月6日 |quote=  }}</ref>。在2009年11月4日時Google搜尋便能夠找到52,100個「Argleton」相關內容，到了12月23日時便增加至249,000個「Argleton」查詢結果；此外如使用「argleton.com」或者是「argleton village.co.uk」作為[[域名|域名]]的網站也出現，其中前者便開玩笑地表示「他們所講的究竟是什麼？我們這些艾格爾頓好公民確實存在。我們現在就在這！<ref>{{Cite web |url= http://abcnews.go.com/Technology/AheadoftheCurve/google-maps-mystery-phantom-town-exists-online/story?id=8987800 |title= Google Maps Mystery: Phantom Town Only Exists Online |author= Ki Mae Heussner和Ned Potter |date= 2009年11月4日 |publisher= [[ABC_News|ABC News]] |language= en |accessdate= 2013年12月6日 |quote=  }}</ref>」，而後者更是編寫了如艾格爾頓的歷史、著名人物以及當地時事新聞等<ref>{{Cite web |url= http://www.argleton-village.co.uk/ |title= Welcome To Argleton |author=  |date=  |publisher= Argleton, Lancashire village |language= en |accessdate= 2013年12月6日 |quote=  |deadurl= yes |archiveurl= https://web.archive.org/web/20100109182500/http://www.argleton-village.co.uk/ |archivedate= 2010年1月9日 }}</ref>。而一些商業網站也拿艾格爾頓作為商品販售的宣傳標語，例如「我參觀了艾格爾頓並且得到了這件T恤」或者是「紐約、倫敦、巴黎以及艾格爾頓」<ref name = "Arjun Ramachandran"/>。2010年9月18日時，{{Link-en|英國廣播公司第四電台|BBC Radio 4}}的主持人{{Link-en|史帝夫·龐特|Steve Punt}}則於節目《{{Link-en|Punt PI|Punt PI}}》中前往調查「艾格爾頓」<ref>{{Cite web |url= http://www.bbc.co.uk/programmes/b00trzfl |title= Punt PI |author= {{Link-en|史帝夫·龐特|Steve Punt}} |date= 2010年9月18日 |publisher={{Link-en|英國廣播公司第四電台|BBC Radio 4}} |language= en |accessdate= 2013年12月6日 |quote=  }}</ref>。

== 解釋 ==
[[File:Argleton,_view_from_Bold_Lane.JPG|thumb]]

關於「艾格爾頓」的出現，一種說法認為該名稱其實是故意添加的{{Link-en|虛構記項|Fictitious entry|版權陷阱}}，其中《[[每日電訊報|每日電訊報]]》採訪的[[利物浦約翰莫里斯大學|利物浦約翰莫里斯大學]]教授{{Link-en|乔·莫兰|Joe Moran (social historian)}}便認為Google就是藉由虛構城鎮來檢查是否有侵犯版權的行為<ref>{{Cite web |url= http://www.telegraph.co.uk/technology/google/6474746/Mystery-of-Argleton-the-Google-town-that-only-exists-online.html |title= Mystery of Argleton, the 'Google' town that only exists online |author= Rebecca Lefort |date= 2009年10月31日 |publisher= 《[[每日電訊報|每日電訊報]]》 |language= en |accessdate= 2013年12月6日 |quote= Joe Moran, an academic at Liverpool John Moores University and map expert, said: "It could be a deliberate error so people can't copy maps. Sometimes they put in fictional streets as the errors would prove they were stolen. I haven't heard of it before on Google Maps." }}</ref>。類似的例子還有像1930年代時{{Link-en|通用製圖|General Drafting}}公司在其繪製的[[紐約州|紐約州]]地圖上增加了[[阿格羅_(地圖)|阿格羅]]作為版權陷阱，之後沒有仔細檢驗的[[埃索|埃索]]便繼續將該虛構名稱沿用在其地圖上<ref>{{Cite web |url= http://www.newscientist.com/article/mg19225791.500-copyright-traps.html |title= Copyright traps |author= John Lackie |date= 2006年11月25日 |publisher= 《[[新科學人|新科學人]]》 |language= en |accessdate= 2013年12月6日 |quote=  }}</ref><ref>{{Cite web |url= http://www.ianbyrne.free-online.co.uk/special/error2.htm |title= Errors on road maps(2) |author= Ian Byrne |date= 2011年3月27日 |publisher= Petrol (Gasoline) and Oil Road Maps from Europe |language= en |accessdate= 2013年12月6日 |quote=  |deadurl= yes |archiveurl= https://web.archive.org/web/20080928231028/http://www.ianbyrne.free-online.co.uk/special/error2.htm |archivedate= 2008年9月28日 }}</ref>。不過儘管在過去便有製圖員在地圖上設計虛構的街道來防止他人非法盜用地圖資訊，然而這類虛構內容通常並不會像艾格爾頓般如此明顯<ref name = "Lenta.ru"/><ref name = "Rebecca Lefort"/>。

另外一些報導也指出英語中「Argle」這字詞組合的發音與「Google」類似，而且「Argleton」這個名稱的字母也可以組成如「Not Large」或「Not Real G」等片語，其中後者猜想中的「G」更有可能便是代表Google<ref name = "Rebecca Lefort"/><ref>{{Cite web |url= http://news.bbc.co.uk/local/liverpool/hi/people_and_places/newsid_8339000/8339873.stm |title= Mystery of phantom Google village |author=  |date= 2009年11月3日 |publisher= [[BBC新聞|BBC新聞]] |language= en |accessdate= 2013年12月6日 |quote=  }}</ref>。另外也有包括《[[衛報|衛報]]》在內的觀點認為艾格爾頓的英文「Argleton」可能便是將附近的村莊{{Link-en|奧頓 (蘭開夏郡)|Aughton, Lancashire|奧頓}}的英文「Aughton」錯誤拼寫而來，但是這兩個地點其實同時出現在同一張地圖上<ref name = "Leo Hickman"/><ref name = "Gemma Jaleel"/>。其他說法還有像「Argle」其實是指電腦程式設計師經常使用於{{Link-en|偽變量|Metasyntactic variable}}的[[無名氏|無名氏]]化名，或者是作為常用參數術語「[[:en:q:argle-bargle|argle-bargle]]」的簡寫等。

不過擔任{{Link-en|地圖學協會|Society of Cartographers}}會長的{{Link-en|丹尼·多林|Danny Dorling}}認為艾格爾頓更有可能只是「無辜的錯誤」而已<ref name = "Leo Hickman"/>，而Google的發言人也表示「雖然絕大多數的資訊是準確的，但偶爾也會有錯誤發生」，並且鼓勵用戶直接向數據提供者報告任何問題<ref name = "Rebecca Lefort"/>。而負責提供Google地圖資料、總部設於[[荷蘭|荷蘭]]的{{Link-en|Tele Atlas|Tele Atlas}}公司發言人則提到無法解釋為何資料庫中會出現這一異常問題，並且表示將把艾格爾頓從地圖資料庫中刪除<ref name = "Rebecca Lefort"/>。2010年1月30日時，有關艾格爾頓的結果已經從Google地圖上移除，但是用戶仍然可以在[[Google街景|Google街景]]上查詢到地點<ref name = "Leo Hickman"/>。2010年5月時Google地圖的相關服務都刪除了艾格爾頓的資訊，而大約在2010年11月中旬到12初時包括Google地球在內的艾格爾頓相關內容都已經被移除<ref name = "Zoe Kleinman"/><ref>{{Cite web |url= https://maps.google.com/maps?f=q&source=s_q&hl=en&geocode=&q=aughton&sll=53.186288,1.164551&sspn=6.032364,19.753418&ie=UTF8&hq=&hnear=Aughton,+Ormskirk,+Lancashire,+United+Kingdom&ll=53.543164,-2.912278&spn=0.011679,0.038581&t=h&z=15 |title= aughton |author=  |date=  |publisher= [[Google地圖|Google地圖]] |language= en |accessdate= 2013年12月6日 |quote=  }}</ref>。
