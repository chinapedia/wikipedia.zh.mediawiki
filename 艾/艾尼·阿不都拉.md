'''艾尼·阿不都拉'''（{{bd|1902年||1981年|6月29日}}<ref>墓碑作生于1902年，卒于1981年。《中苏关系中的“东突”问题（1944-1945）》作1901年生，1978年卒。</ref>）人称“艾尼·巴图尔”（Gheni Batur，[[西里尔字母|西里尔字母]][[维吾尔文|维吾尔文]]Ғени Батур，巴图尔意为英雄）[[维吾尔族|维吾尔族]]，籍贯不详，[[三区革命|三区革命]]人物。<ref name=wxd>{{cite web|title=王欣登：抗战时期苏联对新疆的扩张渗透与“三区革命”|url=http://www.21ccom.net/articles/lsjd/article_201001204459.html|date=2010-01-20|publisher=[[共识网|共识网]]|journal=伊犁教育学院学报 2003年6月第16卷第2期|archiveurl=https://web.archive.org/web/20160304082935/http://www.21ccom.net/articles/lsjd/article_201001204459.html|archivedate=2016-03-04|deadurl=yes}}</ref><ref name=erkin>Erkin Ekrem，中苏关系中的“东突”问题（1944-1945），两岸发展史研究第六期，第109-213页</ref><ref name=sqgms>新疆三区革命史，民族出版社，1998年，第66页</ref>

==生平==
[[File:Ghéni_Batur.JPG|thumb]][[阿拉木图|阿拉木图]]伊宁路。]]
据[[张治中|张治中]]私人秘书[[陶天白|陶天白]]说，艾尼是位大力士，能够隔墙掷牛。艾尼是个盗贼，曾经遭到[[盛世才|盛世才]]逮捕关押。越狱之后，1944年参加了[[帕提赫·莫斯里莫夫|帕提赫·莫斯里莫夫]]领导的巩哈游击队。1944年8月，苏联侨民帕提赫·莫斯里莫夫组织巩哈游击队，发动“[[巩哈暴动|巩哈暴动]]”。1944年10月，巩哈游击队又攻占巩哈县城。艾尼成为巩哈暴动的发难者之一，在攻打巩哈、伊宁中立下功勋。后来因功获授[[民族军|民族军]]中将军衔<ref name=wxd/>。

1944年11月12日，“伊宁解放组织”宣布成立[[东突厥斯坦第二共和国|东突共和国]]临时政府，艾尼等十六人组成临时政府委员会<ref name=erkin/>。

1945年3月6日，临时政府决定组建[[民族军|民族军]]。艾尼·阿不都拉被任命为军事法院院长。1945年4月8日，民族军在伊宁正式成立<ref name=sqgms/>。

1946年6月1日，临时政府通过决议，授予艾尼、帕提赫·莫斯里莫夫二人“人民英雄”称号<ref name=wxd/>。

1949年12月17日，经[[中央人民政府政务院|中央人民政府政务院]]第11次例会批准，由各族各界人民代表参加的新疆省人民政府委员会正式成立。[[包尔汉|包尔汉]]任主席，[[高锦纯|高锦纯]]、[[赛福鼎·艾则孜|赛福鼎·艾则孜]]任副主席。[[王震|王震]]、[[王恩茂|王恩茂]]、[[阿奇木伯克·霍加|卡尤木柏克·霍加]]（维吾尔族，三区）、[[法铁依·伊凡诺维奇·列斯肯|法铁依·伊凡诺维奇·列斯肯]]（俄罗斯族，民族军）、'''阿不都拉·艾尼'''（维吾尔族，民族军）、[[安尼瓦尔·贾库林|安尼瓦尔·贾库林]]（哈萨克族，三区）、[[安尼瓦尔·汗巴巴|安尼瓦尔·汗巴巴]]（乌孜别克族，“新盟”）、[[艾斯海提·伊斯哈科夫|艾斯海提·伊斯哈科夫]]（塔塔尔族，“新盟”）、[[阿不都克日木汗·买合苏木|阿不都克日木汗·买合苏木]]（维吾尔族，七区）、[[辛兰亭|辛兰亭]]、[[伊不拉音·吐尔地|伊不拉音·吐尔地]]（维吾尔族，“新盟”）、[[屈武|屈武]]（国民党新疆省政府起义人员）、[[帕提汗·苏古尔巴也夫|帕提汗·苏古尔巴也夫]]（哈萨克族，三区）、[[阿不列孜·木合买提|阿不力孜·木合买提]]（维吾尔族，七区）、[[阿不都热合满·穆义提|阿不都热合满·穆义提]]（维吾尔族，“新盟”）、[[禹占林|禹占林]]（回族，七区，战斗社）、[[徐立清|徐立清]]、[[涂治|涂治]]（七区，战斗社）、[[陶峙岳|陶峙岳]]（国民党新疆部队起义人员）、[[舒慕同|舒慕同]]（锡伯族，民族军）、[[买合苏提阿洪|买合苏提阿洪]]（维吾尔族，七区）、[[买买提·艾沙|买买提·艾沙]]（柯尔克孜族，七区）、[[达夏甫|达夏甫·明珠里约夫]]（蒙古族，三区）、[[刘孟纯|刘孟纯]]（国民党新疆省政府起义人员）、[[邓力群|邓力群]]、[[阿不都海依尔·吐烈|阿不都海依尔·吐烈]]（哈萨克族，三区）、[[韩有文|韩有文]]（撒拉族，国民党新疆部队起义人员）共28人为委员<ref>[http://www.gmw.cn/content/2004-12/19/content_148501.htm 风骨浩然包尔汉，光明网，2004-12-19]</ref><ref>{{Cite web |url =http://www.hprc.org.cn/gsyj/rws/rwhy/201512/t20151210_363732.html  |title = 从和平解放到离开新疆 |author =邓力群  |date =  |publisher =  当代中国史研究2015年第5期|language =  |accessdate =  |quote =  }}</ref>。

艾尼家中妻妾30余人，其中包括原宁西县警察局局长颜秉河之妻，他在社会上还有40多个姘妇。[[中华人民共和国|中华人民共和国]]成立后，曾因调戏宾馆女服务员，而被[[王震|王震]]司令员训斥。1951年6月22日，《[[新疆日报|新疆日报]]》刊登了群众控诉艾尼的文章。艾尼无法在新疆立足，不得不迁居到[[苏联|苏联]]<ref name=wxd/>。

1981年，艾尼逝世<ref name=erkin/>。安葬在今[[哈萨克斯坦|哈萨克斯坦]][[阿拉木图|阿拉木图]]。
