艾麗森·艾拉希塔 (Allison Iraheta，1992年4月27日-)，來自[[加州|加州]][[洛杉磯|洛杉磯]]，是一位[[美國|美國]]歌手。她曾因贏得真人秀節目《Quinceañera》以及作為《[[美國偶像|美國偶像]]》[[美國偶像_(第八季)|第八季]]參賽者而知名。她擁有[[薩爾瓦多|薩爾瓦多]]人的血統。

== ''Quinceañera'' ==
2006年，艾麗森贏得電視台真人秀節目《Quinceañera》；獎品為五萬美金以及一張唱片合約。<ref>{{cite web |url=http://www.laprensagrafica.com/index.php/fama/espectaculos/20202.html/ |title=存档副本 |accessdate=2009-05-21 |deadurl=yes |archiveurl=https://web.archive.org/web/20090510033517/http://www.laprensagrafica.com/index.php/fama/espectaculos/20202.html/ |archivedate=2009-05-10 }}</ref><ref>http://www.reuters.com/article/entertainmentNews/idUSTRE51R03O20090228</ref>

== 《[[美國偶像|美國偶像]]》 ==
