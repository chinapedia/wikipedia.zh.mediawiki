'''钛酸钡'''是[[钡|钡]]和[[钛|钛]]的混合[[氧化物|氧化物]]，也称偏钛酸钡，化学式为BaTiO<sub>3</sub>。钛酸钡是一个[[铁电性|鐵電]][[陶瓷|陶瓷]]材料，有[[光折射效应|光折射效应]]及[[压电效應|壓電性质]]。其固态时可有五种[[晶体结构|晶体结构]]，温度从高到低依次为：[[六方晶系|六方]]、[[等軸晶系|等軸]]、[[四方晶系|四方]]、[[斜方晶系|斜方]]及[[三方晶系|三方晶系]]。除等軸外，其余的结构都呈现[[铁电性|铁电性]]。

== 外观 ==
钛酸钡为白色粉末或透明[[晶体|晶体]]。它难溶于水，可溶于浓[[硫酸|硫酸]]，R/S phrase 为{{R20/22}}，{{S28}}A，{{S37}}和{{S45}}。
性状 白色粉末。
 
熔点 1625℃
 
相对密度 6.017
 
溶解性 溶于浓硫酸、盐酸及氢氟酸，不溶于热的稀硝酸、水和碱。
 
熔点:1625℃溶解性:INSOLUBLE

== 生产 ==
制备钛酸钡的方法较多，有[[固相|固相]]法和[[共沉淀|共沉淀]]法等。<ref>刘新锦等.无机元素化学(第二版) 过渡元素(一).北京:科学出版社.2010.01 ISBN 978-7-03-026399-5</ref>

固相法：

钛酸钡可通过[[煅烧|煅烧]][[碳酸钡|碳酸钡]]和[[二氧化钛|二氧化钛]]製得，其中可[[掺杂|掺杂]]任意其他材料。

:TiO<sub>2</sub> + BaCO<sub>3</sub> = BaTiO<sub>3</sub> + CO<sub>2</sub>↑

共沉淀法：

(一) 将[[氯化钡|氯化钡]]和[[四氯化钛|四氯化钛]]按等[[物质的量|物质的量]]混合溶解，加热至70°C，然后滴入[[草酸|草酸]]，得到水合[[草酸钛酰钡|草酸钛酰钡]][BaTiO(C<sub>2</sub>O<sub>4</sub>)<sub>2</sub>·4H<sub>2</sub>O]沉淀，经洗涤、干燥，再进行热解，最后得到产品。

:BaCl<sub>2</sub> + TiCl<sub>4</sub> + 2H<sub>2</sub>C<sub>2</sub>O<sub>4</sub> + 5H<sub>2</sub>O = BaTiO(C<sub>2</sub>O<sub>4</sub>)<sub>2</sub>·4H<sub>2</sub>O↓ + 6HCl

:BaTiO(C<sub>2</sub>O<sub>4</sub>)<sub>2</sub>·4H<sub>2</sub>O = BaTiO<sub>3</sub> + 2CO<sub>2</sub>↑ + 2CO↑ + 4H<sub>2</sub>O

(二) [[偏钛酸|偏钛酸]]经打浆后加入[[氯化钡|氯化钡]]溶液，然后在搅拌下加入[[碳酸铵|碳酸铵]]，生成[[碳酸钡|碳酸钡]]和[[偏钛酸|偏钛酸]]共沉淀，煅烧后得产品。

:BaCl<sub>2</sub> + (NH<sub>4</sub>)<sub>2</sub>CO<sub>3</sub> = BaCO<sub>3</sub> + 2NH<sub>4</sub>Cl

:H<sub>2</sub>TiO<sub>3</sub> + BaCO<sub>3</sub> = BaTiO<sub>3</sub> + CO<sub>2</sub>↑ + H<sub>2</sub>O

有报道称，高纯的[[硝酸钡|硝酸钡]]是[[电动车|电动车]]中 [[EEstor|EEstor]] 钛酸钡电容储能系统的关键组分。<ref>{{cite web |url=http://www.zenncars.com/investor/releases/Certification_EEStor_01_16_2007.pdf |title=存档副本 |accessdate=2008-03-08 |deadurl=yes |archiveurl=https://web.archive.org/web/20070928132904/http://www.zenncars.com/investor/releases/Certification_EEStor_01_16_2007.pdf |archivedate=2007-09-28 }}</ref>

钛酸钡常与[[钛酸锶|钛酸锶]]混合。

== 用途 ==
鈦酸鋇被用作陶瓷[[電容|電容]]器的[[介電質|介電]]材料、麥克風和其他[[傳感器|傳感器]]的壓電材料。{{Fact|作為壓電材料，它在很大程度上取代[[鋯鈦酸鉛|鋯鈦酸鉛]]（[[:en:lead_zirconate_titanate|lead zirconate titanate]]）。|time=2010-12-29T07:01:10+00:00}}

多晶鈦酸鋇顯示正的[[溫度係數|溫度係數]]，使之成為[[熱敏電阻|熱敏電阻]]和自我調節電熱系統的有用材料。

全緻密奈米結晶（nanocrystalline）鈦酸鋇的[[介電係數|介電係數]]高達40%，比其他同樣以傳統方法製造的材料更高。<ref>http://research.ucdavis.edu/NCD.cfm?ncdid=658{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}[http://research.ucdavis.edu/NCD.cfm?ncdid=658]{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

鈦酸鋇結晶被用於[[非線性光學|非線性光學]]。它有高的光耦合增益，並能維持在可見光和近紅外線波段。

它在[[自泵浦相位共軛|自泵浦相位共軛]]（[[:en:self-pumped_phase_conjugation|self-pumped phase conjugation]]，SPPC）的應用所使用的材料中有最高的反射率。它可用於連續波[[四波混頻|四波混頻]]（[[:en:four-wave_mixing|four-wave mixing]]）毫瓦遠程光功率。在光折變應用中，鈦酸鋇可以摻雜其他各種元素，例如[[鈰|鈰]]。<ref>http://www.redoptronics.com/Ce:BaTiO3-crystal.html{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }} [http://www.redoptronics.com/Ce-BaTiO3-crystal.html Ce:BaTiO<sub>3</sub>] {{webarchive|url=https://web.archive.org/web/20070207033312/http://www.redoptronics.com/Ce-Batio3-crystal.html |date=2007-02-07 }}</ref>

在錫中摻入鈦酸鋇的包合物（inclusions）已經顯示創造出一個比鑽石高的黏彈[[勁度|勁度]]（[[:en:stiffness|stiffness]]）的散狀物料（bulk material）。鈦酸鋇在相變時會改變晶體的形狀和體積。這導致複合材料的鋇鈦酸有負的體積彈性模量（[[楊氏模量|楊氏模量]]），即當一個作用力施加在包合物上時，在反方向會產生位移，並進一步勁化複合材料。<ref>http://www.sciencemag.org/cgi/content/abstract/315/5812/620 [http://www.sciencemag.org/cgi/content/abstract/315/5812/620]</ref>

鈦酸鋇[[薄膜|薄膜]][[電光調製器|電光調製器]]（[[:en:Electro-optic_modulator|Electro-optic modulator]]）顯示超過40 GHz的頻率。<ref>http://www.opticsexpress.org/abstract.cfm?URI=OPEX-12-24-5962 [http://www.opticsexpress.org/abstract.cfm?URI=OPEX-12-24-5962]</ref>

鈦酸鋇的熱釋電（pyroelectric）和鐵電（ferroelectric）性質，被應於一些非致冷感應器之類的熱成像攝影機（[[:en:thermal_camera|thermal camera]]）。
