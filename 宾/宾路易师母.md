== 早年 ==

宾路易师母于1861年2月28日出生在英国威尔斯的奈司城，一个虔诚的[[贵格会|贵格会]]信徒家庭。她自幼身体软弱，不能进学校接受教育。19岁那年结婚，嫁给威廉·宾路易。

==重生与奉献==

1882年元旦，经历重生。在23岁生日那天，1884年2月28日，她[[奉献|奉献]]一生在祭坛上。 


== 更深的经历 ==

===转向圣灵===
她从[[慕安得烈|慕安得烈]]所著的《基督的灵》一书得帮助，认识天然人的无能，转向[[圣灵|圣灵]]，得到能力的浇灌。 

===主观的十字架===
宣扬[[十字架|十字架]]是神工作的中心和一切[[属灵|属灵]]事物的根基，[[十字架|十字架]]要对付[[旧人|旧人]]，“钉死”“[[己|己]]”、“[[肉体|肉体]]”和“[[魂生命|魂生命]]”，使人脱离这些辖制。
强调不是"作"，而是"死"。

===属灵争战===
她对[[属灵争战|属灵争战]]以及[[邪灵|邪灵]]的工作多有认识，帮助许多人得释放。 

== 威尔斯大复兴 ==
 
她与一个青年矿工[[罗伯斯|罗伯斯]]伊凡(Roberts Evan，1878-1951)一同带进20世纪初空前的[[威尔斯大复兴|威尔斯大复兴]]（1904年－1905年），那是现代教会历史上最大的一次复兴。。 


== 史百克 ==

1920年代初期，伦敦森林山(Forest Hill)的贵橡浸信会教堂(Honor Oak BaPtist Church)的青年牧师[[史百克|史百克]]（Austin Sparks，1885－1971)参加宾路易师母在伦敦的伊克里斯大厅(Eccleston Hall)每月一次的教牧同工聚会，深受吸引。宾路易师母也对史百克的恩赐和所流露的生命极其赏识，有意指定史百克为她职事的继承人。<br />

但到了1926年，史百克与宾路易师母出现看法上的分歧，开始在贵橡路13号(B,Honor Oak Road），另外成立基督徒交通中心(Christian Fellowship Centre)。这次分裂，对宾路易师母的身心，构成了重大的打击。


== 倪柝声 ==
宾路易师母对于主观[[十字架|十字架]]和[[属灵争战|属灵争战]]的观点，对中国基督徒领袖[[倪柝声|倪柝声]]的影响极其深远。他的名著《属灵人》，即受到她的启发。
