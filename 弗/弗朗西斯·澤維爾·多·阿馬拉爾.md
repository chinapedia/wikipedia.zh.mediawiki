'''弗朗西斯·澤維爾·多·阿馬拉爾'''（{{Lang-en|Francisco Xavier do Amaral}}，1937年－2012年3月6日），[[東帝汶|東帝汶]]民眾有時稱之為「'''阿波澤維爾'''」或「'''澤維爾祖父'''」（Abo Xavier，「阿波」為祖父之意），是東帝汶獨立運動成員暨政治家，也是[[東帝汶獨立革命陣線|東帝汶獨立革命陣線]]創始人之一<ref name = "Peter Alford">{{Cite web |url = http://www.theaustralian.com.au/news/world/resistance-era-president-of-east-timor-francisco-xavier-do-amaral-dies-in-dili-aged-74/story-e6frg6so-1226291130113 |title = Resistance-era president of East Timor, Francisco Xavier do Amaral, dies in Dili, aged 74 |author = Peter Alford |date = 2012年3月7日 |publisher = 《[[澳洲人報|澳洲人報]]》 |language = en |accessdate = 2016年11月5日 |quote =  }}</ref><ref>{{Cite web |url = http://expresso.sapo.pt/feeds/lusa/lusaactualidade/leste-francisco-xavier-do-amaral-o-homem-que-declarou-a-independencia-do-pais-perfil=f709433 |title = Leste: Francisco Xavier do Amaral, o homem que declarou a independência do país (PERFIL) |author = Peter Alford |date = 2012年3月6日 |publisher = 《{{Link-en|快報 (新聞)|Expresso (newspaper)|快報}}》 |language = pt |accessdate = 2016年11月5日 |quote =  }}</ref>。

== 經歷 ==
=== 宣布獨立 ===
澤維爾·阿馬拉爾出生於[[葡屬帝汶|葡屬帝汶]][[馬努法希區|馬努法希區]]，是當地村長長子，從小接受[[葡萄牙|葡萄牙]]教育。他從[[澳門|澳門]]留學返回[[東帝汶|東帝汶]]後，放棄原先的神職人員工作，轉而擔任拉丁語和葡萄牙語教師。葡萄牙各個大城市在1974年4月經歷[[康乃馨革命|康乃馨革命]]後，他在同年5月20日聯合[[若澤·拉莫斯·奧爾塔|若澤·拉莫斯·奧爾塔]]和[[馬里·阿爾卡蒂里|馬里·阿爾卡蒂里]]等人，共同成立{{Link-en|東帝汶社會民主協會|Timorese Social Democratic Association}}。同年9月11日，東帝汶社會民主協會結合其他組織而改組為[[東帝汶獨立革命陣線|東帝汶獨立革命陣線]]。

隨著葡萄牙政府轉而放棄過去的殖民地，當時仍為葡萄牙殖民地的[[葡屬帝汶|葡屬帝汶]]內部局勢更加動亂，其中對立兩方分別東帝汶獨立革命陣線支持群眾，及主張加入[[印度尼西亞|印度尼西亞]]西半部省分者。經過短暫的內部衝突後，東帝汶獨立革命陣線聲稱舉辦公民投票。1975年11月28日，擔任東帝汶獨立革命陣線主席的阿馬拉爾{{Link-en|單方面宣布獨立|Unilateral declaration of independence|單方面}}宣布東帝汶獨立，且宣誓就任第一任[[東帝汶總統|東帝汶總統]]，也是印度尼西亞入侵前首位總統<ref name = "Peter Alford"/>。

=== 獨立運動 ===
[[File:Destroyed_houses_12.jpg|thumb]]。]]

儘管東帝汶獨立革命陣線積極支持馬克思主義，但是其組成的臨時政府非但未獲得政治權力外，獨立宣言也未有任何國家予以承認。在9天過後，印度尼西亞軍隊於12月7日{{Link-en|印度尼西亞入侵東帝汶|Indonesian invasion of East Timor|入侵東帝汶}}，迫使阿馬拉爾逃往山區。隔年7月，印度尼西亞政府宣布東帝汶為其第二十七個省份。而在避難期間，阿馬拉爾積極領導撤往山區的東帝汶獨立革命陣線，反抗[[東帝汶印度尼西亞占領時期|佔領東帝汶]]的印度尼西亞，共計有10萬名農民參與反抗行動。但隨著戰鬥局勢無望、民眾逐漸疲憊與流行病盛行，使得抗爭派與對話派之間發生衝突。

阿馬拉爾曾在1977年9月13日以叛國罪遭到逮捕，不過東帝汶獨立革命陣線趁著囚犯移動到監禁處時襲擊，使其得以解放，但之後仍因對於獨立運動的意見分歧而失勢。1978年8月，阿馬拉爾在一次東帝汶獨立革命陣線的行動中，意外遭到印度尼西亞部隊逮捕。阿馬拉爾先是送至[[峇里島|峇里島]]，並在1978年轉往[[雅加達|雅加達]]監禁。由於考量其知名度，印度尼西亞政府允許其在住家中實行軟禁，這時他結識同樣遭到拘禁的[[沙納納·古斯芒|沙納納·古斯芒]]。

隨著印度尼西亞總統[[蘇哈托|蘇哈托]]在1998年宣布辭職後，東帝汶民眾於隔年舉辦的{{Link-en|1999年東帝汶獨立公民投票|East Timorese independence referendum, 1999|東帝汶獨立公民投票}}獲得壓倒性的漂數通過。儘管發生{{Link-en|親印度尼西亞民兵|Pro-Indonesia militia}}大規模[[1999年東帝汶危機|破壞]]一事，但隨著澳洲派遣部隊進駐，印度尼西亞政府被迫同年撤離東帝汶。[[聯合國|聯合國]]在2000年2月成立{{Link-en|聯合國東帝汶過渡管理政府|United Nations Transitional Administration in East Timor}}後，結束22年監禁生活的阿馬拉爾得以返回東帝汶，但之後得知妻子已經失蹤、且兩名孩子遭到擊斃。

=== 政治活動 ===
[[File:11_Xanana_3.jpg|thumb]]為阿馬拉爾的好友。]]

東帝汶獨立革命陣線協助澤維爾·阿馬拉爾獲得平反，但他並未加入東帝汶獨立革命陣線或是沙納納·古斯芒陣營，而是依照1974年的政黨名稱，重新成立自己的政黨東帝汶社會民主協會。不過在2001年舉行的{{Link-en|2001年東帝汶議會選舉|East Timorese parliamentary election, 2001|東帝汶議會選舉}}上，東帝汶獨立革命陣線和東帝汶社會民主協會獲得7.8%票數而取得少數席次，在88個席次中取得6個席次。因此從2001年開始，阿馬拉爾代表東帝汶社會民主協會參與制憲會議的討論，制憲會議在2002年正式獨立後則改名為第一屆[[國民議會_(東帝汶)|國民議會]]，並持續擔任議員而直到逝世為止<ref>{{Cite web |url = http://www.heraldsun.com.au/archive/news/east-timor-founding-father-francisco-xavier-do-amaral-dies/story-e6frf7lf-1226291215845 |title = East Timor founding father Francisco Xavier do Amaral dies |author = [[澳聯社|澳聯社]] |date = 2012年3月7日 |publisher = 《{{Link-en|先驅太陽報|Herald Sun先驅太陽報}}》 |language = en |accessdate = 2016年11月5日 |quote =  }}</ref>。

阿馬拉爾曾參與2002年4月舉辦的{{Link-en|2002年東帝汶總統選舉|East Timorese presidential election, 2002|東帝汶總統選舉}}，但僅獲得17.3%票數，而遭到獲得82.7%票數的沙納納·古斯芒擊敗。在東帝汶總理[[馬里·阿爾卡蒂里|馬里·阿爾卡蒂里]]推薦下，阿馬拉爾成為東帝汶副總理。2007年時，他再度代表東帝汶社會民主協會當選成為國民議會議員。而隨著東帝汶社會民主協會也加入沙納納·古斯芒政府，阿馬拉爾因而加入內閣。不過在同年4月舉行的{{Link-en|2007年東帝汶總統選舉|East Timorese presidential election, 2007|東帝汶總統選舉}}中，作為總統候選人的阿馬拉爾僅獲得14.39%選票而排行第四名<ref>{{Cite web |url = http://www.smh.com.au/news/World/Timorese-prepare-to-elect-new-president/2007/03/10/1173478717431.html |title = Timorese prepare to elect new president |author = [[澳聯社|澳聯社]] |date = 2007年3月10日 |publisher = 《[[雪梨晨鋒報|雪梨晨鋒報]]》 |language = en |accessdate = 2016年11月5日 |quote =  }}</ref>。

阿馬拉爾也作為總統候選人參與{{Link-en|2012年東帝汶總統選舉|East Timorese presidential election, 2012}}，然而在2012年3月16日第一輪投票前10天，他便在東帝汶首都狄力當地醫院因為癌症逝世，享年75歲。之後東帝汶政府為阿馬拉爾際舉行國葬，而由於阿馬拉爾際出生日期並未確定，東帝汶當地民眾大多以12月3日作為其生日紀念<ref>{{Cite web |url = https://blogs.deakin.edu.au/deakin-speaking/2012/03/06/francisco-xavier-do-amaral-first-president-of-east-timor-politician-and-elder-statesman/ |title = Francisco Xavier do Amaral: First President of East Timor, politician and elder statesman |author = Damien Kingsbury |date = 2012年3月6日 |publisher = [[迪肯大學|迪肯大學]] |language = en |accessdate = 2016年11月5日 |quote =  }}</ref><ref>{{Cite web |url = http://home.patwalsh.net/wp-content/uploads/PW_Winter-of-East-Timor-patriarchs.pdf |title = Winter of East Timor’s Patriarchs |author = Pat Walsh |date =  |publisher = Pat Walsh |language = en |accessdate = 2016年11月5日 |quote =  }}</ref>。
