'''弗里吉亚无边便帽'''（Phrygian cap）又称'''自由之帽'''，本為古代[[小亚细亚|小亚细亚]]的[[弗里吉亚人|弗里吉亚人]]所戴，是一種与头部紧密贴合的[[软帽|软帽]]，其帽尖向前弯曲，典型的颜色是红色。

[[古希臘|古希臘]]、[[古羅馬文化|古羅馬文化]]中，弗里吉亚帽是东方的象征。在18世纪[[美国革命|美国革命]]和[[法国大革命|法国大革命]]中，弗里吉亚帽成为自由和解放的标志而广为传播；例如在名画《[[自由引导人民|自由引导人民]]》（''{{lang|fr|La Liberté guidant le peuple}}''）中，[[自由女神|自由女神]]就佩戴着弗里吉亚帽。[[法兰西共和国|法兰西共和国]]的象征[[玛丽安娜|玛丽安娜]]也戴着弗里吉亚帽。弗里吉亚帽还出现在[[尼加拉瓜|尼加拉瓜]]、[[萨尔瓦多|萨尔瓦多]]、[[哥伦比亚|哥伦比亚]]、[[巴拉圭|巴拉圭]]、[[古巴|古巴]]等国的[[国徽|国徽]]上。卡通画中的《[[蓝精灵|-{zh-tw:藍色小精靈;zh-hans:蓝精灵}-]]》（''Schtroumpf''）也戴着弗里吉亚帽。

== 歷史 ==
=== 古代 ===
[[File:Magi_(1).jpg|thumb]]位在[[君士坦丁堡|君士坦丁堡]]東方，因此在這幅[[拉文納|拉文納]][[聖阿波里奈爾教堂|聖阿波里奈爾教堂]]的[[鑲嵌畫|鑲嵌畫]]中，[[東方三博士|東方三博士]]戴着弗里吉亚帽，表示他們是「東方人」。]]

希臘神話中，[[阿波羅|阿波羅]]將[[弗里吉亞|弗里吉亞]]國王[[彌達斯|彌達斯]]的耳朵變成了驢耳，於是[[彌達斯|彌達斯]]用弗里吉亞帽遮掩耳朵。這個說法最早出自[[阿里斯托芬|阿里斯托芬]]的《[[普魯特斯|普魯特斯]]》，但更早之前便已經出现在希臘畫作中；由於弗里吉亞位處希臘之東，所以古希臘羅馬藝術常用弗里吉亚帽表示某個人物來自東方國度，屬於「[[蠻族|蠻族]]」（本义泛指非希腊人，沒有野蠻的意思）。古希腊人描绘[[特洛伊|特洛伊]]王子[[帕里斯|帕里斯]]的形象时，为了说明他不是希腊人，就在他的头上加一顶弗里吉亚人的帽子。古[[波斯|波斯]]神祇[[密特拉|密特拉]]传入希腊化各国及[[罗马帝国|罗马帝国]]后，他的形象也变成了头戴弗里吉亚帽的男子。

[[馬其頓王國|馬其頓人]]、[[色雷斯|色雷斯人]]和[[十二世紀|十二世紀]]的[[諾曼人|諾曼人]]軍隊有種類似弗里吉亞帽的頭盔，稱為[[弗里吉亞式頭盔|弗里吉亞式頭盔]]（Phrygian type helmets）。

古羅馬時代，獲釋奴隸及其後代佩戴一種名為{{tsl|en|Pileus|pileus}}的圓錐形帽子。[[羅馬共和|羅馬共和]]晚期，{{tsl|en|Pileus|pileus帽}}成為脫離暴政的象徵。一枚於前44年到前42年由[[馬爾庫斯·尤尼烏斯·布魯圖|布魯圖]]在[[小亞細亞|小亞細亞]]發行的硬幣上，有一個{{tsl|en|Pileus|pileus帽}}與兩把短劍（代表[[馬爾庫斯·尤尼烏斯·布魯圖|布魯圖]]兩年前參與刺殺獨裁者[[尤利烏斯·凱撒|尤利烏斯·凱撒]]，解放羅馬人民）。[http://www.uned.es/geo-1-historia-antigua-universal/ROMA/brutus-reverse.jpg]。直到[[羅馬帝國|羅馬帝國]]時，在[[農神節|農神節]]等節慶場合，獲釋的奴隸仍會佩戴pileus帽。近代歐洲人将pileus误认为弗里吉亚帽，因此弗里吉亚帽与[[自由|自由]]和[[解放|解放]]联系起来，得名“自由之帽”（liberty cap）。

=== 革命的標誌 ===
[[File:Louis_le_dernier3.jpg|thumb]]]]
在革命時期的法國，紅色的弗里吉亞帽（[[法語|法語]]：''bonnet rouge''）最早在1790年5月出現於[[特鲁瓦|特鲁瓦]]及[[里昂|里昂]]。[[法國|法國]]的國家象徵[[瑪麗安娜|瑪麗安娜]]也戴着弗里吉亚帽。

1792年[[路易十六|路易十六]]批准憲法時，印刷品中的國王被改為佩戴弗里吉亞帽。1794年，[[斯特拉斯堡主教座堂|斯特拉斯堡主教座堂]]在尖頂上放置了弗里吉亞帽，以防止它被拆除。

在[[巴黎|巴黎]]參與革命的人們紛紛佩戴紅色弗里吉亞帽，彰顯革命熱情與團結。

== 在美洲 ==
[[File:Seal_of_the_United_States_Senate.svg|thumb]]的院徽]]
[[美國參議院|美國參議院]]的院徽上有一頂弗里吉亚帽。

[[中南美洲|中南美洲]]深受[[美國革命|美國革命]]及[[法國大革命|法國大革命]]的影響，因此弗里吉亚帽出现在[[尼加拉瓜|尼加拉瓜]]、[[萨尔瓦多|萨尔瓦多]]、[[哥伦比亚|哥伦比亚]]、[[巴拉圭|巴拉圭]]、[[古巴|古巴]]等国的[[国徽|国徽]]上，包括：
* [[阿根廷國徽|阿根廷國徽]]
* [[玻利維亞國徽|玻利維亞國徽]]
* [[哥倫比亞國徽|哥倫比亞國徽]]
* [[古巴國徽|古巴國徽]]
* [[薩爾瓦多國徽|薩爾瓦多國徽]]
* [[海地國徽|海地國徽]]
* [[尼加拉瓜國徽|尼加拉瓜國徽]]
* [[巴拉圭國旗|巴拉圭國旗]]的背面

== 流行文化 ==
《[[蓝精灵|-{zh-tw:藍色小精靈;zh-hans:蓝精灵}-]]》中的人物大都戴着白色的弗里吉亚帽。其中 ''Papa Smurf''（藍爸爸、精靈爸爸或老爸）戴的是紅色的弗里吉亚帽。
