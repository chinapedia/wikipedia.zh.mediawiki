'''弗萊徹級驅逐艦'''是[[第二次世界大戰|第二次世界大戰]]中[[美國海軍|美國海軍]]的[[驅逐艦|驅逐艦]]級，總計生產了175艘，是單一驅逐艦級生產量最高的，因此也常被當作二战美軍最具代表性的驅逐艦。弗莱彻级驱逐舰在众多方面都被认为性能优秀，胜任[[防空戰|防空]]、[[反潛作戰|反潜]]与水面作战各种任务，航程又足以覆盖美军舰队在太平洋上的行动。此級驅逐艦于二戰後大多仍繼續服役，加上之後移交美國盟友之海軍，總服役時間相當的長。

== 设计 ==
[[File:Fletcher-class_destroyer_technical_drawing_1954.jpg|弗莱彻级驱逐舰设计图]]
弗莱彻级是美军在[[限制海军军备条约|后条约时代]]设计的第一代驱逐舰，其中专门注重在[[太平洋|太平洋]]作战的需要。在[[倫敦海軍條約|倫敦海軍條約]]失效后，美國海軍決定设计比[[格里維斯級驅逐艦|格里維斯級驅逐艦]]更大的驅逐艦以確保足以對抗日本後條約驅逐艦，。弗莱彻級在主武器火力上並沒有明顯的提高，但強化了防空火力，具有更好的擴充性。相比之前的[[班森级驱逐舰|班森级]]和[[格里維斯級驅逐艦|格里维斯级驱逐舰]]，弗莱彻级的排水量增加了约500吨，轴马力也因此从50000匹提高到了60000匹。和前级舰艇相比更大的舰体使得弗莱彻级除了能沿中心线搭载5门{{link-en|Mk 12型5英寸38倍径炮|5"/38 caliber gun}}和2具五联装[[Mk_15型鱼雷|Mk 15型鱼雷]]发射管以对抗它国同级驱逐舰外，还有可观的防空火力。在最初的设计中，弗莱彻级只有一门四联装[[28毫米75倍径高射机炮|28毫米75倍径高射机炮]]和六挺[[白朗寧M2重機槍|勃朗宁M2重机枪]]作为防空火力，在二战爆发后，面对日军航空兵的威胁，防空火力被逐渐加强到了两门双联装[[博福斯40毫米高射炮|博福斯40毫米高射炮]]和七门单装[[厄利孔20毫米机炮|厄利孔20毫米机炮]]。为了应对二战后期日军的[[神风敢死队|神风自杀式攻击]]，自1945年开始共有51艘弗莱彻级驱逐舰接受了进一步的防空火力改装，拆掉了舰艏鱼雷发射管和舰舯双联装40毫米炮，以两门四联装40毫米炮代替，七门单装20毫米炮也被换成六门双联装20毫米炮<ref>Friedman, Norman. ''US Destroyers: An Illustrated Design History'' (revised edition, Naval Institute Press, Annapolis, 2004), pp.111-112.</ref>。在反潜作战方面，原本设计中的弗莱彻级在舰艉部有两架深水炸弹投放轨，在二战期间则在两舷又加装了6具K型深水炸弹发射器（K-gun），可将炸弹抛射至137米之外。

== 歷史 ==
1942年6月4日對美國海軍来说是个值得紀念的日子，在[[中途島之役|中途島]]，美軍擊沈了4艘日本[[航空母艦|航空母艦]]，頓挫了[[日本帝國海軍|日本帝國海軍]]的氣勢。同一天，弗莱彻級的首艦“弗莱彻”號（以纪念著名美国海军将领[[弗萊德·弗萊徹|弗莱德·弗莱彻]]）也在東海岸入役。美國驚人的生產力也在弗莱彻級上得到了展現——從開工到下水平均只需212天，再到正式入役平均只需152天，總計364天，比前級快了10%，龐大的數目為盟軍的最终勝利做出了不小貢獻。<ref>[http://destroyerhistory.org/fletcherclass/ DestroyerHistory.org] {{webarchive|url=https://web.archive.org/web/20081206021258/http://www.destroyerhistory.org/fletcherclass/ |date=2008-12-06 }}</ref>

弗莱彻级驱逐舰在二戰中共制造175艘（另有13艘的订单随着战争结束被取消），損失25艘，其中19艘被擊沈，另6艘損壞過於嚴重放弃修復。44艘得到超過10顆[[戰鬥之星|戰鬥之星]]，19艘得到{{Link-en|海軍部隊嘉獎|Navy Unit Commendation}}，16艘得到{{Link-en|總統部隊嘉許獎|Presidential Unit Citation (United States)}}。二战结束后，所有驱逐舰均被除役封存，但随着[[朝鲜战争|朝鲜战争]]爆发又被大批召回继续服役，其中39艘接受了现代化改装。

在1950年代，大量弗莱彻级驱逐舰在接受现代化改装后被转卖给美国的盟国海军，其中2艘给[[日本海上自衛隊|日本海上自卫队]]、3艘给[[大韓民國海軍|大韩民国海军]]、4艘给[[中華民國海軍|中华民国海军]]。美军剩余的弗莱彻级，除“蘇利文兄弟”号、“纪德”号和“卡森·杨”号外，均在1970年代被解体。最后一艘除役的弗莱彻级驱逐舰是{{Link-en|墨西哥海军|Mexican Navy}}的“{{Link-en|約翰·羅傑斯號驅逐艦 (DD-574)|USS John Rodgers (DD-574)|庫伊特拉華克}}”号（ARM Cuitláhuac），於2001年除役，在爭取保存失敗後於2010年解體，意味着弗莱彻级驱逐舰服役长达近60年，并一直使用到21世纪。<ref name="fl">[http://destroyerhistory.org/fletcherclass/  destroyerhistory.org: ''Fletcher'' class] {{webarchive|url=https://web.archive.org/web/20081206021258/http://www.destroyerhistory.org/fletcherclass/ |date=2008-12-06 }}</ref>

== 同級艦 ==
由於數量眾多，僅簡介比較有名的舰艇：

* “弗莱彻”號（DD-445）：是弗莱彻級命名艦，參與多場海戰，是名列前茅的武勳艦。
* “歐班農”號（DD-450）：二戰美國海軍驅逐艦的第一武勳艦，也是幸運艦，該艦在整個服役生涯中沒有一位船員因受傷而領取[[紫心勳章|紫心勳章]]。在[[維拉拉維拉海戰|維拉拉維拉海戰]]以前完全沒受到夠大的損傷，還創下用馬鈴薯抗擊潛艇的傳說。
*[[慶陽號驅逐艦|「梅蘭利」號]]（DD-528）：曾於1945年[[沖繩島戰役|沖繩島戰役]]中遭日軍神風自殺機攻擊，順利倖存；後於1971年售予中華民國海軍並改名為「慶陽」號，為國軍第一艘自費購買的[[陽字號驅逐艦|陽字號驅逐艦]]；最後於1999年除役、2001年鑿沉為[[人工魚礁|人工魚礁]]。
* “蘇利文兄弟”號（DD-537）：以紀念在[[瓜島海戰|瓜島海戰]]中同時戰死的[[蘇利文兄弟|蘇利文五兄弟]]，現正停泊在紐約州[[布法罗市|布法罗市]]中作为紀念艦。
* [[約翰斯頓號驅逐艦|“約翰斯頓”號]]（DD-557）：[[雷伊泰灣海戰|雷伊泰灣海戰]]中，約翰斯頓號為掩護美軍[[护航航母|護航航母]]，與其他兩艘驅逐艦一道向包括4艘戰艦（包括[[大和號戰艦|“大和”號]]在內），8艘巡洋艦和11艘驅逐艦的日本主力艦隊發起了勇敢的衝鋒。約翰斯頓號用盡所有魚雷，在被戰艦火砲擊中的情況下仍堅持戰鬥向日艦開火，直至戰沈；艦長伊文斯以下186人殉職。
* [[威廉·D·波特号驱逐舰_(DD-579)|“威廉·D·波特”號]]（DD-579）：美國海軍“最倒霉”的艦艇，曾經誤發魚雷险些擊中[[艾奧瓦號戰艦_(BB-61)|艾奧瓦號戰列艦]]<ref>{{cite news|title=二戰美軍最搞笑的驅逐艦 魚雷竟射向羅斯福|url=http://big5.huaxia.com/thjq/jsgoucheng/2010/06/1930072.html|publisher=華夏經緯網|date=2010-06-11|accessdate=2016-04-10}}</ref>；最後於1945年被擊沉。
*{{Link-en|愛爾本號驅逐艦|USS Erben（DD-631)|「愛爾本」號}}（DD-631）：美方將該艦於1963年撥交大韓民國海軍改名為「忠武」號，為南韓海軍建軍第一艘大型水面艦艇。
* “紀德”號（DD-661）：現在[[路易西安納州|路易西安納州]][[巴頓魯治|巴頓魯治]]中作博物館艦。
* “卡森·楊”號（DD-793）：現停泊在[[麻塞諸塞州|麻塞諸塞州]][[波士頓|波士頓]]國家歷史公園中作紀念艦，供遊人參觀。
