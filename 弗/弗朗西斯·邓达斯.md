'''弗朗西斯·邓达斯'''（{{lang-en|Francis Dundas}}，{{bd|1759年||1824年|1月15日}}），英国{{le|贝里克郡|Berwickshire}}人，是一名[[英國陸軍|英國陸軍]]上將，并在1798年至1803年担任[[开普殖民地|开普殖民地]]代理总督。

弗朗西斯·邓达斯是罗伯特·邓达斯和吉英·格兰特的次子。罗伯特·邓达斯是第一任陆军大臣{{le|亨利·邓达斯|Henry Dundas}}的侄子。

== 军事生涯 ==
1775年，他服役于[[第1近卫步兵团|第1近卫步兵团]]。1783年，轉入第45步兵团；并于1787年，调入[[皇家苏格兰团|皇家苏格兰团]]。美国独立战争期间，他跟随[[第一代康沃利斯侯爵查爾斯·康沃利斯|第一代康沃利斯侯爵查爾斯·康沃利斯]]参加克拉普磨房战役<ref>{{cite web
|last = |first = |authorlink = |coauthors = |date = |month = |day = |title = The Battle of Clapp's Mills|work = revolutionarywar101.com|volume = |issue = |pages = |id = |url = http://www.revolutionarywar101.com/battles/810302-clapps-mills/|accessdate = 2008-11-01}}</ref>，[[格林斯普林战役|格林斯普林战役]]，和[[約克鎮圍城戰役|約克鎮圍城戰役]]。

1796年，他出任苏格兰旅（后[[第94步兵团|第94步兵团]]）旅長，并扎营于西库<ref>{{cite book
|last = Miller|first = James|authorlink = |coauthors = |year = 1859|month = |day = |title = The History of Dunbar|publisher = J. Downie|volume = |page = 174|id = |url = http://books.google.com/books?id=MSo2AAAAMAAJ&pg=PA174&lpg=PA174&dq=Francis+Dundas+revolutionary+war&source=web&ots=u_sm9ImQcH&sig=jGZb7RsqUDvo23inF5V6dkyA9Wo&hl=en&sa=X&oi=book_result&resnum=8&ct=result#PPA174,M1|accessdate = 2008-11-01}}</ref>，之后[[罗兰·希尔|罗兰·希尔]]接任。

1796年8月，他受任担任大英帝国占领后的卡普殖民地代理总督。1798年11月21日，他就职并任至1799年12月9日；在1801年4月20日再任，并任至1803年2月20日。期间签署《[[亚眠和约|亚眠和约]]》，使得卡普殖民地归为[[巴达维亚共和国|巴达维亚共和国]]。在他任内，爆发了1798年格拉夫-里内特暴动和[[第三次前线战争|第三次前线战争]]。他的治理风格被认为是独裁的，但仍然适当。

在返回英国后，他担任了数个重要的军職，作為肯特师師長率部参加（[[大卫·邓达斯|大卫·邓达斯]]领导）1804年至1805年的战役。
