'''弗朗西斯堡'''（{{lang-en|Fort Frances}}）是[[加拿大|加拿大]][[安大略省|安大略省]]西北部的一个镇，為[[雷尼河區|雷尼河區]]的行政中心。該市位于[[雷尼河|雷尼河]]畔，与[[美国|美国]][[明尼苏达州|明尼苏达州]][[国际瀑布城|国际瀑布城]]隔河相望。據2011年加拿大人口普查所示，弗朗西斯堡鎮內人口為7952人<ref name="census2011"/>。

== 歷史 ==
法裔探索家皮耶·拉維倫德萊（{{lang|fr|Pierre La Vérendrye}}）於1731年在現弗朗西斯堡一帶設立皮毛交易站，並將之名為聖皮耶堡（{{lang|fr|Fort St. Pierre}}）<ref name="hist">{{cite web |url=http://www.fort-frances.com/visitors-events/heritage-information |title=Heritage Information |publisher=弗朗西斯堡鎮政府 |accessdate=2014-03-20 |deadurl=yes |archiveurl=https://web.archive.org/web/20140320185617/http://www.fort-frances.com/visitors-events/heritage-information |archivedate=2014-03-20 }}</ref>。他於翌年在現[[肯諾拉|肯諾拉]]以南接近[[美加邊境|美加邊境]]一帶設立另一座名為聖夏爾堡（{{lang|fr|Fort St. Charles}}）的交易站，而聖皮耶堡亦逐漸被取代和廢棄。隨着法國在[[七年戰爭|七年戰爭]]中敗於英國，[[新法蘭西|新法蘭西]]於1763年割讓予英國，此帶地域遂落入英國的控制。

18世紀末期，[[哈德遜灣公司|哈德遜灣公司]]和[[西北公司|西北公司]]在此帶展開皮毛交易，兩者之間的競爭激烈，但兩間公司後於1821年合併。哈德遜灣公司於1790年代沿雷尼河設立數座臨時交易站，但待1818年才在此設立一座永久性的據點<ref name="hist"/>。1830年6月，哈德遜灣公司總裁喬治·辛普森（{{lang|en|George Simpson}}）連同夫人弗朗西斯（{{lang|en|Frances}}）到訪此處；此據點遂於同年9月25日正式以總裁夫人命名為弗朗西斯堡<ref name="hist"/>。

歐裔殖民於1870年代相繼抵達此地。為了讓船隻可避開雷尼河上的瀑布來往[[雷尼湖|雷尼湖]]和[[伍兹湖|伍兹湖]]，當局於1876年至1878年間在此修建運河；但隨著聯邦政府落實興建[[加拿大太平洋鐵路|加拿大太平洋鐵路]]，該帶地域的東西向運輸骨幹亦改以鐵路為重點，運河項目遂告取消<ref name="hist"/>。隨著鐵路開通至肯諾拉，經雷尼河和伍兹湖來往弗朗西斯堡和肯諾拉的蒸汽船航線亦成為當地交通的主要命脈<ref name="hist"/>。

19世紀末期，弗朗西斯堡一帶的經濟以林木業為主，在該帶砍伐的樹木可運送到肯諾拉再透過鐵路對外輸出，而數座鋸木廠亦在此設立<ref name="hist"/>。美國企業家愛德華·巴克斯（{{lang|en|Edward Backus}}）於20世紀初斥資在雷尼河上築壩作水力發電之用，又於弗朗西斯堡設立一座紙漿工廠，進一步促進當地發展<ref name="hist"/>。[[雷尼河區|雷尼河區]]於1885年成立後，行政中心設於弗朗西斯堡；而弗朗西斯堡亦於1903年正式建制為鎮<ref name="hist"/>。

== 交通 ==
安大略11號和71號省道橫跨弗朗西斯堡，兩者皆隸屬[[橫加公路|橫加公路]]，而兩者在鎮内的部分路段亦共構運行。11號省道西通雷尼河鎮，東達[[桑德貝|雷灣]]；71號省道北達肯諾拉市外圍，南端在弗朗西斯堡河濱駁上[[弗朗西斯堡-國際瀑布城國際大橋|弗朗西斯堡-國際瀑布城國際大橋]]跨越雷尼河，在對岸駁上美國53號和71號國道。

航空交通方面，弗朗西斯堡機場（{{lang|en|Fort Frances Municipal Airport}}，IATA代碼：YAG，ICAO代碼：CYAG）提供航線來往肯諾拉、雷灣和[[溫尼伯|溫尼伯]]，而對岸美國國際瀑布城的機場則提供航線來往明尼蘇達州其他地點。
