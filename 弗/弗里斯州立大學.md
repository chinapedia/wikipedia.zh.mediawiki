'''弗里斯州立大學'''（'''Ferris State University'''，簡稱：'''FSU'''和'''Ferris'''）是位於[[美國|美國]][[密歇根州|密歇根州]][[大急流市_(密歇根州)|大急流市]]的一所[[公立大學|公立大學]]，成立於1884年。2015年《[[美國新聞與世界報道|美國新聞與世界報道]]》在“中西部地區大學”排名中將其列在第54位。<ref>{{cite web|title=Ferris State University|url=http://colleges.usnews.rankingsandreviews.com/best-colleges/ferris-state-university-2260|publisher=U.S. News and World Report|accessdate=2015-03-08}}</ref> 按學生數量來算，弗里斯州立大學是密歇根州第九大大學<ref>{{cite web |title= Enrollment Report Fall 2010 |publisher= Presidents Council, State Universities of Michigan |url= http://www.pcsum.org/Portals/0/docs/PCSUM%202010%20Enrollment%20Report.pdf |format= PDF |accessdate= 2011-01-21 |deadurl= yes |archiveurl= https://web.archive.org/web/20110727170034/http://www.pcsum.org/Portals/0/docs/PCSUM%202010%20Enrollment%20Report.pdf |archivedate= 2011-07-27 }}</ref>
