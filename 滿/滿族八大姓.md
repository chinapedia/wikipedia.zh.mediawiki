'''滿族八大姓'''，又称'''滿洲八大姓'''、'''满洲八-{著}-姓'''<ref name=浅谈>{{cite journal en|quotes = |author = 李小雪|date =  |title =浅议“满族八大姓”|journal = 《剑南文学·文史研究》|issue = 2012年第08期|pages = |publisher = 《剑南文学》编辑部|location = |issn = |url = |language = Zh-hans}}</ref>、'''满洲八大贵族'''、'''八大家'''等，主要泛指清代社会地位较高的[[满族姓氏|满族姓氏]]，因居住地域等差异，版本上也莫衷一是<ref>{{harvnb|刘庆华|2012|p=80}}</ref>。有学者认为《[[八旗满洲氏族通谱|八旗满洲氏族通谱]]》中的八著姓为其原型<ref name=浅谈/>。
