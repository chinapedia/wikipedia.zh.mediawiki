'''约园'''为位于[[中国|中国]][[江苏省|江苏省]][[常州市|常州市]][[天宁区|天宁区]]兴隆巷常州市第二人民医院内的一座清代私家园林，占地约19亩。该园原为明代官府鹿苑，清乾隆初年为[[谢旻|谢旻]]别业，称谢园，道光时[[赵翼|赵翼]]之孙[[赵起|赵起]]购得此园并修葺改造，时有二十四景、十二峰，取“约乎成园”之意称为约园，又称赵家花园。咸丰十年（1860）毁于兵燹，光绪末年稍经修葺，但不复旧观。民国二十年（1931）在此设立武进县县立医院，1949年后又改为常州工人医院、第二人民医院等，1953年赵氏后人将约园献与政府<ref>{{cite web |url=http://epaper.cz001.com.cn/site1/czrb/html/2011-03/01/content_391824.htm|title=《约园约成林》|accessdate=2015-05-07|work=薛焕炳|publisher=常州日报}}</ref>。

约园原以奇石见长，以曲水闻名，池中叠石成山，筑有方亭，以曲桥与岸边相通，池边罗列不同形态的奇石<ref>{{cite web |url=http://www.chinajsb.cn/gb/content/2006-09/25/content_188886.htm|title=《古典园林中的宅第园林》|accessdate=2015-05-07|work=|publisher=中国建设报}}</ref>。但近年随着医院的逐步扩大，周围高楼四起，古风不再。
