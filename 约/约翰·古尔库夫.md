'''約翰·哥古夫'''（'''Yoann Gourcuff'''，{{bd|1986年|7月11日|catIDx=Gourcuff, Yoann}}）是一名[[法國|法國]][[足球|足球]]運動員，司職[[中場|進攻中場]]，現效力[[法甲|法甲]]球會[[迪安FCO|迪安]]。

出身於法國中下游球會[[洛里昂足球俱樂部|羅連安特]] （當時直到現在一直由古尔屈夫的父親[[基斯頓·哥古夫|基斯頓·哥古夫]]執教），於2003年加盟[[雷恩足球俱樂部|雷恩]]。他代表過法國青年軍出賽，曾於2005年贏得19歲以下[[歐洲國家杯|歐洲國家杯]]冠軍。在2006年他有傳與[[阿積士|阿積士]]、[[華倫西亞足球會|華倫西亞]]和[[阿仙奴足球會|-{zh-hans:阿森納; zh-hk:阿仙奴;}-]]拉上關係，但最後他選擇加盟[[意甲|意甲]]勁旅[[AC米蘭|AC米蘭]]。2008年，他被AC米蘭外借到法甲球隊[[波爾多足球俱樂部|波爾多]]，并在2009年正式轉會。

2010年球季，哥古夫憑藉之前兩季在[[法甲|法甲]]聯賽的出色表現，轉投到另一支法國班霸球會[[里昂足球俱樂部|里昂]]。

由於他主要偏向進攻，故被法國媒體視為球王[[齐内丁·齐达内|齐内丁·齐达内]]的接班人，也有人把他和前[[馬賽足球俱樂部|馬賽]]球星[[佐卡夫|佐卡夫]]比較。但因为没有达到预期，他和纳斯里、本-阿尔法一起，成为了法国“迷失的一代”的代名词。

== 早年生活 ==
哥古夫出生於運動世家，他的母親是籃球運動員，他的哥哥是一個游泳和單車選手，而他的父親是一位前足球運動員，曾經是法國，瑞士和加拿大球會的球員，但他成名的時間是在[[雷恩足球俱乐部|雷恩]]，[[洛里昂足球俱樂部|羅連安特]]和[[勒芒足球會|勒芒]]。他目前擔任[[雷恩足球俱乐部|雷恩]]的領隊。

作為一個青年，哥古夫善於足球和網球，在他這個年齡組成為了頂級球員，但最終選擇專注於足球。

== 球員生涯 ==
;青年軍及雷恩

哥古夫開始他的職業生涯是在[[洛里昂足球俱樂部|羅連安特]]，當時球隊的領隊，他的父親把他送往{{tsl|en|PEF Ploufragan|}}足球學院。後來他的父親執教[[雷恩足球俱乐部|雷恩]]，哥古夫初步認為會去[[南特足球俱樂部|南特]]，但最後也決定跟隨父親，並於2001年與球隊簽署了青年合約。2003年，代表雷恩青年軍的哥古夫在決賽中取得入球，協助球隊以4-1擊敗[[斯特拉斯堡足球俱乐部|斯特拉斯堡]]贏得了2003年青年盃。

正如所料，哥古夫在2003年11月簽署了第一份職業合約，並在2003-04賽季後半部晉升至一隊。他在聯賽首次亮相是在2004年2月7日以0-2擊敗[[歐賽爾足球會|歐賽爾]]的賽事，在第76分鐘後備上陣。他在這賽季共得到8次出場的機會，其中包括他的第一次對陣[[波爾多足球俱樂部|波爾多]]以3-1獲勝。2005-06賽季哥古夫穿著10號球衣，他在聯賽上陣了36場比賽，攻入6球，而首個聯賽入球是以2-0擊敗[[摩纳哥足球俱乐部|摩纳哥]]。

;AC米蘭

哥古夫出色的表現吸引了多間大球會的注意，包括[[荷甲|荷甲]]的[[阿積士|阿積士]]，[[西甲|西甲]]的[[華倫西亞足球會|華倫西亞]]及[[英超|英超]]球隊[[阿仙奴|阿仙奴]]，在2005-06賽季結束後，哥古夫轉投[[意甲|意甲]]的[[AC米蘭|AC米蘭]]，簽訂了為期5年的合約，轉會費是大約350萬歐元。他首次為AC米蘭上陣是8月9日是在[[2006年至2007年歐洲聯賽冠軍盃|歐洲聯賽冠軍盃]]第三圈資格賽對陣[[贝尔格莱德红星足球俱乐部|貝爾格萊德紅星]]，在81分鐘後備入替[[派路|派路]]。他取得首個入球是在[[2006年至2007年歐洲聯賽冠軍盃|歐洲聯賽冠軍盃]]小組賽對陣[[AEK雅典足球會|AEK雅典]]。

接著的兩個賽季，哥古夫的上場時間不足，有效地出現的只有36場聯賽，其中23場是後備上陣。他微薄的上場時間和緩慢的發展，主要是由於受傷及被領隊[[安察洛堤|安察洛堤]]忽視和批評，指出了他的心理題。哥古夫位置上的直接競爭對手是[[巴西國家足球隊|巴西國家足球隊]]主力，[[2007年國際足協世界足球先生|2007年國際足協世界足球先生]]得主[[卡卡|卡卡]]，進一步限制了他的上陣時間。不過，哥古夫也得到[[2006年至2007年歐洲聯賽冠軍盃|2006年至2007年歐洲聯賽冠軍盃]]的獎牌。雖然他在決賽並不是正選的11人及後備人選，但他在[[2006年至2007年歐洲聯賽冠軍盃|歐洲聯賽冠軍盃]]3次的出場使他有資格獲得一枚獎牌。他也協助球隊贏得2007年[[歐洲超霸盃|歐洲超霸盃]]及[[2007年世界冠軍球會盃|2007年世界冠軍球會盃]]。

2006-07賽季後，於2008年5月31日，[[阿德里亚诺·加利亚尼|加利亞尼]]透露，AC米蘭將外借哥古夫到[[法甲|法甲]]球隊[[波爾多足球俱樂部|波爾多]]整個2008-09賽季，讓他得到更多的上場時間。

;波爾多
[[Image:Yoann_Gourcuff.jpg|thumb]]
在[[波爾多足球俱樂部|波爾多]]，哥古夫很快便獲得上陣機會，他首次上陣是在以2-1擊敗[[卡昂足球俱樂部|卡昂]]的賽事，隨後並協助波爾多羸得2008年[[法國超級盃|法國超級盃]]冠軍。哥古夫在2008-09年[[法國聯賽盃|法國聯賽盃]]4-0戰勝[[瓦讷奥林匹克俱乐部|瓦納]]的決賽，他助攻了第三個入球，並攻入了第四個入球。

在接下來的比賽，哥古夫的表現非常出色，並陸續取得入球。最終協助波爾多第六次奪得[[法國甲組足球聯賽|法國甲組足球聯賽]]冠軍，並羸得2008年[[法國超級盃|法國超級盃]]冠軍及2009年[[法國聯賽盃|法國聯賽盃]]冠軍。哥古夫的努力，使他在2008-09賽季被評為法甲年度最佳球員獎。他還入選了法甲年度最佳陣容。
[[File:Yoann_Gourcuff_2009.jpg|thumb]]
由於哥古夫在這賽季表現出色，波爾多希望利用1500萬歐元的優先收購權使哥古夫正式成為球隊的一員。儘管[[AC米蘭|AC米蘭]]總經理[[阿德里亚诺·加利亚尼|加利亞尼]]和新領隊[[莱昂纳多·纳西门托·德·阿劳若|李安納度]]努力說服哥古夫歸隊，但始終未能保證哥古夫的上陣時間。5月28日，波爾多宣佈哥古夫已經從AC米蘭正式轉會，並簽署了4年合約。轉會費未透露，但應該是優先收購權的1500萬歐元。

;2009-10賽季

哥古夫在2009-10賽季正式為[[波爾多足球俱樂部|波爾多]]上陣，他協助球隊以2-0戰勝[[甘岡前進會|甘岡]]羸得2009年[[法國超級盃|法國超級盃]]。他的表現出色，被評為最佳球員。兩個星期後，他在這賽季首場聯賽對陣[[朗斯足球會|朗斯]]梅開二度，結果以4-1獲勝。這場勝利是波爾多12場連勝，創下了球會最長連勝紀錄。哥古夫出色的表現協助球隊在[[2009/10赛季欧洲冠军联赛|2009/10賽季歐洲聯賽冠軍盃]]分組賽以不敗姿態晉身十六強淘汰賽。
12月20日，哥古夫被法國足球雜誌選為2009年[[法國足球|法國足球先生]]，擊敗了[[尼古拉·阿内尔卡|安歷卡]]、[[胡戈·洛里斯|洛里斯]]、[[蒂埃里·亨利|亨利]]及上兩屆得主[[弗兰克·里贝里|列貝利]]。

;里昂
2010年8月23日，[[里昂足球俱樂部|里昂]]在球會官方網頁宣佈與波爾多達成協議成功羅致哥古夫，哥古夫此前亦曾被報導過有意加盟該支球隊。一待通過翌日的體能檢測，哥古夫便會落實以2,200萬[[歐元|歐元]]的身價轉會並簽下一份長期的合約，而里昂會把轉會款項分三期於2012年12月31日之前還清。

== 國家隊 ==
[[File:Gourcuff_coup_de_pied_de_coin.jpg|thumb]]
哥古夫曾經14次代表法國U17上陣並攻入一球。由於他表現出色，只為法國U18上陣7場後便入選法國U19。
2005年19歲以下[[歐洲國家杯|歐洲國家杯]]，他出色的表現協助球隊羸得贏得19歲以下[[歐洲國家杯|歐洲國家杯]]冠軍。羸得他的第一個國際榮譽。

哥古夫隨後便入選法國U21，上陣18次取得4個入球，但卻無法協助球隊取得參加[[歐洲U-21足球錦標賽|歐洲U-21足球錦標賽]]的資格。

哥古夫首次為大國腳上陣是2008年8月20日友誼賽對[[瑞典國家足球隊|瑞典]]的比賽，他在補時至92分鐘時後備上陣，最終法國隊以3-2勝利。他在法國隊取得第一個進球是在2008年10月11日對陣[[羅馬尼亞國家足球隊|羅馬尼亞]]的[[2010年世界盃外圍賽|2010年世界盃外圍賽]]，賽果為2-2。在對[[塞爾維亞國家足球隊|塞爾維亞]]的比賽，法國隊領隊[[雷蒙·杜明尼治|杜明尼治]]把哥古夫放在[[前鋒_(足球)|前鋒]]身後，[[攻擊中場|攻擊中場]]的位置。這位置是[[2006年世界盃足球賽|2006年世界盃足球賽]][[施丹|施丹]]掛靴前所擔任的位置。
