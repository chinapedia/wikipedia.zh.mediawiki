'''约瑟夫·约阿希姆'''（{{lang-de|'''Joseph Joachim'''}}，{{bd|1831年|6月28日|1907年|8月15日|Joachim}}），[[匈牙利|匈牙利]]写法为'''约阿希姆·约瑟夫'''（'''Joachim József'''），[[匈牙利|匈牙利]][[小提琴|小提琴]]演奏家、作曲家、指挥家和音乐教育家，[[德国小提琴学派|德国小提琴学派]]在19世纪后半期的领袖人物。

== 早年 ==
约阿希姆出生于现属于[[奥地利|奥地利]]的Kittsee小城，父母是匈牙利[[犹太人|犹太人]]，1833年随全家移居[[佩斯|佩斯]]，幼年时即跟从Stanislaus Serwaczynski（其后成为[[亨里克·维尼亚夫斯基|亨里克·维尼亚夫斯基]]的老师）学习[[小提琴|小提琴]]。1839年赴[[维也纳|维也纳]]学习音乐。不久，随堂姐芳妮·维特根斯坦（哲学家[[路德维希·维特根斯坦|路德维希·维特根斯坦]]的祖母）移居[[莱比锡|莱比锡]]，在[[门德尔松|门德尔松]]的指导下学习小提琴演奏。1844年，在[[伦敦|伦敦]]非常成功地演出了[[贝多芬|贝多芬]]的[[贝多芬小提琴协奏曲|小提琴协奏曲]]，从此成名。

== 成名后 ==
门德尔松逝世后，约阿希姆留在莱比锡跟随德国小提琴学派大师[[费迪南德·大卫|费迪南德·大卫]]继续其演奏生涯。1848年，[[弗兰兹·李斯特|李斯特]]定居[[魏玛|魏玛]]，发愿要将其建成[[德国|德国]]音乐的“[[雅典|雅典]]”。约阿希姆立即脱离保守气氛浓厚的莱比锡，投奔李斯特。很快，李斯特就召集了一批革新气息浓厚的年轻音乐家。

1852年，约阿希姆又脱离李斯特，移居[[汉诺威|汉诺威]]，同[[罗伯特·舒曼|罗伯特·舒曼]]、[[克拉拉·舒曼|克拉拉·舒曼]]和[[布拉姆斯|布拉姆斯]]等人建立了友谊。1863年成婚。1865年，辞去在[[汉诺威王国|汉诺威王室]]中的供职，以抗议王室对一位犹太乐师的[[种族歧视|种族歧视]]。

1866年，约阿希姆移居[[柏林|柏林]]，创建了当时首屈一指的“姚阿幸弦乐四重奏乐团”。1884年同妻子离婚，原因是他怀疑妻子与布拉姆斯的出版商有染，这导致他与布拉姆斯反目。

1869年，姚阿幸於柏林為當時的普魯士王朝建立柏林皇家音樂院（為柏林藝術大學音樂系的前身），伴隨著孟德爾頌門徒以及布拉姆斯好友的名號，成就他成為最有名、地位也最崇高的演奏家，此後的四十年為他所建立的音樂學院努力。

1907年，约阿希姆在柏林去世。

== 音乐创作 ==
约阿希姆早年曾希望朝[[作曲家|作曲家]]方向发展，《哈姆雷特序曲》等作品接近李斯特的风格，而之后的作品则主要为小提琴而作，擅于发挥高超的演奏技巧，并结合匈牙利民族风格。
