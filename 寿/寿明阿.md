'''寿明阿'''（{{bd|1885年||1947年|}}），[[字|字]]'''玉亭'''，[[蒙古族|蒙古族]]，[[内蒙古|内蒙古]][[哲里木盟|哲里木盟]][[科尔沁右翼后旗|科爾沁右翼後旗]]人，[[中華民國|中華民国]]及[[满洲国|满洲国]]的政治人物。<ref name=xyc/><ref name=lsl/>

==生平==
1912年（民国元年），任[[郭尔罗斯前旗|郭爾羅斯前旗]]協理。1916年（民国5年），获授[[辅国公|輔国公]]。1922年至1924年，第一屆國會復會期間，擔任[[北京政府|北京政府]]的参議院議員。<ref name=xyc>{{Cite book|author = 徐友春主編|title = 民国人物大辞典 増訂版|year = 2007|publisher = 河北人民出版社|isbn = 978-7-202-03014-1}}</ref><ref name=lsl>{{Cite book|author = 劉寿林等編|title = 民国職官年表|year = 1995|publisher = [[中華書局|中華書局]]|isbn = 7-101-01320-1}}</ref>

1922年（民国11年），他任[[東三省|東三省]]保安总司令部顧問。翌年，他任总統府顧問兼[[奉天省|奉天省]]政府諮議。1924年（民国13年），他任蒙古宣撫使署顧問。1931年（民国20年），他任蒙边督办公署蒙務处处長。<ref name=xyc/><ref name=lsl/>

[[满洲国|满洲国]]成立後的1932年（[[大同_(满洲国)|大同]]元年）6月，他任興安总署政務处处長。1934年（[[康德|康德]]元年），他任蒙政部民政司司長。1936年（康德3年）7月，他任[[興安省_(解放區)|興安南省]]省長。1942年（康德9年）4月，他辞任。<ref name=xyc/><ref name=lsl/>

1945年日本投降后，他和[[光复节_(朝鲜半岛)|光复军]]对抗[[中国共产党|中国共产党]]在蘇聯軍隊支持下（蘇聯軍隊進入東北后）成立政权，據說和卢伯航共同抢劫牧民的马群。后来他任[[内蒙古自治政府|内蒙古自治政府]]三等参事。1947年他被共產黨处决。<ref>“内蒙翻身人民法庭”处决大蒙奸寿明阿，人民日报1947年12月7日，第1版</ref>
