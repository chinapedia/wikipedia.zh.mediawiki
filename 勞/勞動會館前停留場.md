'''勞動會館前停留場'''（{{jpn|j=労働会館前停留場|hg=ろうどうかいかんまえていりゅうじょう|rm=Rōdō-Kaikan-mae teiryūjō|f=y}}）是位於[[日本|日本]][[北海道|北海道]][[函館市|函館市]]旭町的[[函館市交通局|函館市交通局]]（函館市電車）東雲線已被廢除的[[鐵路車站|鐵路車站]]。

== 歷史 ==
* 參考[[函館市交通局|函館市交通局]]的歷史。

== 車站構造 ==
* 勞動會館前停留場擁有2個月台（[[側式月台|側式月台]]）、2條電車路線（相反方向）。

== 車站周邊 ==
* 函館市勞動會館
* 函館市政府
* 函館旭町郵政局
