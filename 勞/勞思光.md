'''勞思光'''（{{bd|1927年|9月3日|2012年|10月21日|catIdx=Lao勞思光}}），原名'''勞榮瑋'''，字'''仲瓊'''，號'''韋齋'''，筆名'''思光'''，[[湖南|湖南]][[長沙|長沙]]人，當代重要[[華人|華人]][[哲學家|哲學家]]及教育家。

== 家庭及成長 ==
勞教授是[[湖南省|湖南省]][[長沙府|長沙府]][[善化縣|善化縣]]人，高祖[[勞崇光|勞崇光]]為清[[道光|道光]][[進士|進士]]，曾任[[兩廣總督|兩廣總督]]，1860年將[[九龍|九龍]][[界限街|界限街]]以南的[[九龍半島|九龍半島]]租借給[[英國|英國]]。父親勞競九出身[[保定軍官學校|保定軍官學校]]，曾加入[[中國同盟會|同盟會]]，參與[[辛亥革命|辛亥革命]]，後跟[[白崇禧|白崇禧]]同學，中將退伍。由於出身翰林世家，勞思光打下紮實深厚的國學基礎。其堂兄[[勞幹|勞幹]]也是著名學者，[[中研院院士|中研院院士]]。

高中畢業，勞思光至[[北京大學|北京大學]]哲學系進修，未及畢業，就因1949年[[國民政府|國民政府]]敗退，遂隨父母移居[[台灣|台灣]]，在[[台灣大學|台灣大學]]哲學系完成學業。其後的[[白色恐怖|白色恐怖]]年代，因反對[[國民黨|國民黨]][[獨裁|獨裁]]，主張[[民主|民主]]自由，引起[[法務部調查局|調查局]]與[[警備總部|警備總部]]注意，意圖加害，得其父舊部通知危險，被迫離開台灣，寓居[[香港|香港]]。

勞教授反對[[專制|專制]]，發誓除非[[共產黨|共產黨]]下台，否則不回[[中國大陸|中國大陸]]；又發誓除非台灣[[解嚴|解嚴]]，否則不到台灣，所以勞師下半生從未踏足中國大陸，而在很長一段時間，勞師的著作都被[[台灣政府|台灣政府]]列為禁書，甚至父母過世，也無法返台奔喪。<ref>[http://news.ltn.com.tw/news/politics/paper/629942 拒回共產中國 勞思光長眠宜蘭] 自由時報, 2012-11-12</ref>

== 香港時期 ==
勞教授到港後，專心在學術方面，先後於[[珠海書院|珠海書院]]、[[崇基學院|崇基學院]]及[[香港中文大學|香港中文大學]]教授[[哲學|哲學]]，並在[[美國|美國]][[哈佛大學|哈佛大學]]及[[普林斯頓大學|普林斯頓大學]]從事研究工作。勞教授在中大崇基學院時，除任課之外，尚須編寫教材，譬如勞教授就因此而完成《中國文化要義》一書。後來他更擔任研究所主任一職。勞教授公務雖忙，但著作更勤，特別以《新編中國哲學史》三卷名重於時。但勞教授也非常關心時局世運、重視中國前途。1977年，[[文革|文革]]結束；1980年，[[鄧小平|鄧小平]]徹底掌權，並開始審判[[四人幫|四人幫]]；台灣方面，[[蔣中正之死|蔣介石去世]]，[[蔣經國|蔣經國]]接掌大政，但1971年[[中華民國退出聯合國|中華民國被逼退出聯合國]]，海峽兩岸風起雲湧、變化多端。更直接影響到香港地位的，是中英雙方就關於香港前途的問題展開談判。勞教授關心香港，更關心海峽兩岸的發展，於是在1981年請[[李怡|李怡]]等共同發起組織「前景社」，探討香港及中國的前途。勞教授退休後，任香港中文大學中國文化研究所高級研究員及該校[[逸夫書院|逸夫書院]]高級導師。

勞思光在香港中文大學授課期間，與[[唐君毅|唐君毅]]、[[牟宗三|牟宗三]]齊名，被喻為「香港人文三老」。其所著的《新編中國哲學史》，運用其提倡的「基源問題研究法」進行嚴格的分析，展示中國哲學各時代不同門派的學術見解與內層理境，出版以來，深為學界所重，有學者推重之為同類型著作之中最突出的一本。

== 臺灣歲月 ==
勞教授離台前，曾立誓說除非[[國民黨|國民黨]]終止[[戒嚴|戒嚴]]，否則不回台灣，並時常關切、幫助台灣的[[黨外|黨外]]運動。從香港退休後，因為台灣已經[[解嚴|解嚴]]，1989年年勞教授受台灣[[國立清華大學|國立清華大學]]社會科學院院長[[李亦園|李亦園]]之邀，擔任歷史研究所客座教授；在台期間，勞教授先後任教於清華大學、[[政治大學|政治大學]]、[[東吳大學|東吳大學]]及[[華梵大學|華梵大學]]，擔任[[華梵大學|華梵大學]]教授及[[東吳大學|東吳大學]]端木愷講座教授，並於2001年榮獲台灣行政院文化獎，2002年獲選中央研究院院士。2004年榮獲香港中文大學榮譽文學博士。

勞教授在台，仍然不改關心時事的心懷，除了應邀撰寫報紙短文外，1990年代[[民進黨|民進黨]]人士接觸勞師，勞教授點出民進黨不能只以反對國民黨而自足，而應該對兩岸問題及台灣前景提出正面的方案，此後，民進黨人士草就《[[台灣前途決議文|台灣前途決議文]]》。2005年，勞教授以七十八高齡，仍關心時事，倡議反對六千億元新台幣的[[三項軍購案|軍購案]]，與其他十名中央研究院的院士聯名發表建言，終於力挽狂瀾。勞教授晚年多病，癆病未癒，外加腸胃病，讓身體虛弱，常常感冒，2010年，勞教授因[[大腸癌|大腸癌]]，入院進行切除手術，當時[[馬英九|馬英九]]總統曾親自去醫院探訪。

2012年10月21日，勞教授年邁，因心肌梗塞跌倒，昏厥[[台北|台北]]寓所，經發現後，送醫不治，享年八十五歲<ref>{{cite news|title=85歲哲學大師 勞思光走了|url=http://news.chinatimes.com/politics/50207314/112012102200059.html|publisher=中國時報|date=2012-10-22|language=zh-tw|deadurl=yes|archiveurl=https://web.archive.org/web/20121024232432/http://news.chinatimes.com/politics/50207314/112012102200059.html|archivedate=2012-10-24}}</ref><ref>{{cite news|title=哲學大師勞思光 跌倒不治|url=http://www.libertytimes.com.tw/2012/new/oct/22/today-life10.htm?Slots=BLife|publisher=自由時報|date=2012-10-22|language=zh-tw|deadurl=yes|archiveurl=https://web.archive.org/web/20140311122958/http://www.libertytimes.com.tw/2012/new/oct/22/today-life10.htm?Slots=BLife|archivedate=2014-03-11}}</ref><ref>{{cite news|title=勞思光的中國哲學史 港台學生聖經|url=http://udn.com/NEWS/NATIONAL/NATS2/7445477.shtml|publisher=聯合報|date=2012-10-22|language=zh-tw}}</ref>。勞先生的生日是八月初八，身分證上登記的出生年為1922年，但出生年確為一九二七年，所以終年為八十五歲。

公祭之日，海內外文史哲學界近50個團體參加追思暨公祭，馬英九總統親臨會場頒贈[[褒揚令|褒揚令]]。因其生前表明不願意回到由中國共產黨統治的大陸，學生與家人將其葬於台灣[[宜蘭縣|宜蘭縣]]櫻花陵園。

== 學術思想 ==

勞思光學貫中西，尤其長於[[中國哲學史|中國哲學史]]及現當代[[西方哲學|西方哲學]]。但勞教授不願畫地自限，曾謂「百家出入心無礙，一海東西理可知」，所以平生又特別重視[[比較哲學|比較哲學]]。

勞教授以[[自由主義|自由主義]]者稱著，實是卓然大成的人文學者。勞教授雖然研究成績斐然，但生平不以已有成績自限，而重視不斷反省上進的哲學生命。所以對自己的作品常不自滿，甚至說「每悔多言刪少作」；但對於自己的學問，他又從不輕言放棄，他不但要求個人不斷反省，以免成為封閉的系統，也勸勉弟子，切勿成立門派，而應保持開放性，這都顯示他自由主義的特色。同時，他誨人不倦，喜歡跟青年談學問，雖然自嘆「才難況亂離」，但終生不離教學崗位，晚年常感體力不足，但一登講壇，仍可以連講數小時。

== 著作 ==
勞教授著作遍及文、史、哲學、考據、時論等，尤以一九八零年代所出版之《新編中國哲學史》望重於時，其中獨創「基源問題研究法」認為哲學史不只是「歷史」，還要有思想，也就是哲學問題意識。他認為身為一個知識份子就是有承擔精神，肩負時代使命，因此著作頗多針對社會政治而發的時論。勞思光亦喜歡寫詩與術數自娛，詩中常可見與詩友及同道交往，也流露對國家社會的所思所感，其詩作因延續[[同光體|同光體]]的風格，而好用典，中文學界王隆升、林碧玲及彭雅玲等曾註釋其詩作，並發表系列研究。

勞教授桃李滿門，許多學生及學者都曾選讀或旁聽先生講課，或私下請益。但追隨先生撰寫論文的弟子，在香港博士學位的至少有[[黃慧英|黃慧英]]、[[馮耀明|馮耀明]]、[[溫偉耀|溫偉耀]]等教授，碩士學位則至少有[[關子尹|關子尹]]、[[張燦輝|張燦輝]]、[[劉國英|劉國英]]教授等。而台灣方面，資料未備，但依據[[國家圖書館_(中華民國)|國家圖書館]]全國博碩士論文資料庫，則有[[吳有能|吳有能]]一人。而勞先生過世後，有遺孀黃岱雲女士及女兒勞延韻一人。
