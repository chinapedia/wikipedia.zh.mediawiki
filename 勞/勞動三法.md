'''勞動三法'''，係指[[中華民國法律|中華民國法律]]之'''《工會法》'''、'''《團體協約法》'''及'''《勞資爭議處理法》'''，為勞資關係運作上的重要[[法律|法律]]。1992年11月12日，[[工人立法行動委員會|工人立法行動委員會]]等[[勞工運動|勞工運動]]團體組成「三法一案行動委員會」發起「1112工人鬥陣大遊行」，當時的勞動三法是《勞動基準法》、《工會法》及《勞資爭議處理法》。

==歷史 ==
過去的勞資關係中，[[勞工|勞工]]（勞方）與[[僱主|僱主]]（資方）的地位處於不均等的地位，勞方的[[權力|權力]]遠小於資方所擁有的[[權力|權力]]，加上沒有明確保障[[勞工|勞工]]權益、提升[[勞工|勞工]]地位之法令，使得[[僱主|僱主]]運用勞工對於其之依附性，[[剝削|剝削]][[勞工|勞工]]應有的權益。

[[勞工|勞工]]意識到自身的權利微薄，欲與資方談判卻恐遭責罰，甚至丟失[[工作|工作]]，於是使用團結權組成[[工會|工會]]，往後以[[工會|工會]]為核心，運用勞方[[集體|集體]]的力量，向資方爭取[[利益|利益]]改善、簽署相關保護勞方的[[法律|法律]]文件，目的為提升勞方地位至與資方平等。為了保護勞方上述的[[權利|權利]]，制定《工會法》保障[[勞工|勞工]]組織[[工會|工會]]的「團結權」、制定《團體協約法》保障勞方與資方簽署相關[[法律|法律]]文件、以爭取權益的「交涉權（集體協商權）」。

[[工會|工會]]與資方協商失敗時，雙方可行使爭議權，予對方壓力使之讓步。由於勞方趨於弱勢，資方易有懲戒勞方行使爭議權之情形出現，為了保障雙方的爭議權，於是制定《勞資爭議處理法》。

== 勞動三法推動的目的 ==
[[勞動法|勞動法]]區分為兩大領域，即以《[[勞動基準法_(中華民國)|勞動基準法]]》為核心之個別勞動法及以《工會法》、《團體協約法》及《勞資爭議處理法》等勞動三法為核心之集體勞動法。

勞動三法在立法目的上有別於以《勞動基準法》（簡稱勞基法）為核心的個別勞動法之目的在維持[[勞工|勞工]]之具有[[尊嚴|尊嚴]]的勞動條件，緩和[[勞工|勞工]]在[[契約|契約]]上不對等的地位以及勞動過程之受支配的從屬狀態，並使[[勞工|勞工]]得以積極參與勞動條件的決定，藉此促進勞動條件的維持與改善以及回復勞工之[[人格|人格]]的自主性。

見《勞動基準法》第一條：「為規定勞動條件最低標準，保障[[勞工|勞工]]權益，加強勞雇關係，促進[[社會|社會]]與[[經濟|經濟]]發展，特制定本法；本法未規定者，適用其他[[法律|法律]]之規定。[[雇主|雇主]]與[[勞工|勞工]]所訂勞動條件，不得低於本法所定之最低標準。」

相形之下，集體勞動法所扮演的角色是促使[[勞工|勞工]]脫離弱勢的狀態，透過集體勞動條件的共同決定，例如[[工會|工會]]和[[雇主|雇主]]的團體協商所締結之團體協約來回復[[勞工|勞工]][[人格|人格]]的自主性，為推動勞動三法的目的所在。<ref>{{Cite web|url =http://book.cla.gov.tw/image/no_22/A-4.pdf|title =新勞動三法對台灣工會的意義與未來的課題|author =張鑫隆|date =2010年6月|publisher =台灣勞工季刊|language =zh-hant|accessdate =2014-06-18}}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

== 改革 ==
中華民國《工會法》、《團體協約法》、《勞資爭議處理法》最早分別於民國18年、19年、17年公布，相繼實施後數十年並未有大幅度修改，相關單位以其不合時宜為由，著手進行研修。

《團體協約法》、《勞資爭議處理法》、《工會法》陸續於民國97年、98年、99年完成修正，完成修正的勞動三法一齊於民國100年5月1日[[勞動節|勞動節]]當天施行。<ref>{{Cite web |url = http://www.ccunix.ccu.edu.tw/~shlin/Extendsion/labor%20law%20revision100914..pdf |title = 勞動三法後勞資關係之發展 |author = 李昆鴻 |publisher = 國立中正大學勞工關係學系 |language = zh-hant |accessdate = 2014-06-18 |deadurl = yes |archiveurl = https://web.archive.org/web/20150514115711/http://www.ccunix.ccu.edu.tw/~shlin/Extendsion/labor%20law%20revision100914..pdf |archivedate = 2015-05-14 }}</ref>

=== 《工會法》 ===
{{wikisource|工會法}}
* [[工會|工會]]組織類型的變革

在舊法中，[[工會|工會]]類型分為「產業工會」與「職業工會」二種，經過修法，轉變為「企業工會」、「產業工會」、「職業工會」三種[[工會|工會]]形式。維持舊法中以廠場為範圍之[[工會|工會]]組織，並將其歸為企業工會；不屬於[[非營利|非營利]]性質之組織、團體，亦為企業工會範疇；產業工會係指相關、相同之[[產業|產業]]，例如紡織業、石化業之[[勞工|勞工]]組織[[工會|工會]]；職業工會則指從事相關之[[職業|職業]][[技能|技能]]者共組[[工會|工會]]。《修正條文第六條》

* [[工會|工會]]結社更趨[[民主|民主]]化《工會法第五、六、七章》

明定[[會員|會員]]大會或會員代表大會、[[理事會|理事會]]及[[監事會|監事會]]權責。

* [[工會|工會]]財務監督

為使[[工會|工會]]財務透明化，以確保[[工會|工會]][[會員|會員]]權益，[[工會|工會]]應參照工會財務處理準則，建立[[工會|工會]]自我之財務收支運用及稽核機制。並於每年會員大會或會員代表大會中，提出[[財務管理|財務管理]]之書面[[報告|報告]]。
 
* [[工會|工會]]議決之數額規範

為使[[工會|工會]]落實[[民主|民主]]機制，一般[[會議|會議]]之決議，除應遵守會議規範之議決數額外，[[工會|工會]]就重大事項之議決如章程之訂定，及會員大會或會員代表大會之議決重要事項，應採出席[[會員|會員]]或會員代表三分之二以上同意之[[絕對多數|絕對多數]]議決方式處理。

* 維持強制入會原則，但無明定違反之罰則

過去對於[[勞工|勞工]]加入[[工會|工會]]之[[年齡|年齡]]予以限制須滿十六歲，惟與《[[勞動基準法_(中華民國)|勞動基準法]]》中受僱勞工之[[年齡|年齡]]之規範不符，此種限制有檢討之必要，因此特別明定除代表雇主行使管理權之主管人員，不得加入該企業之[[工會|工會]]外《修正條文第十四條》，廠場或事業單位等之受僱勞工應均有企業工會之[[會員|會員]]資格，但並無明定違反之罰則，產業工會與職業工會也沒有強制入會規定。

* 組織[[工會|工會]]的程序放寬（[[工會|工會]]會務自主化）

刪除須事先向主管機關登記，現採取尊重[[工會|工會]]自主化之立場，採事後報備主義並刪除主管機關派員指導。

* [[工會|工會]]成員單一化

參酌國際勞工公約的精神及國民平等待遇原則，刪除工會理事、監事須具有中華民國[[國籍|國籍]]之限制。另基於工會會員純粹性與工會運作獨立性原則，明定工會會員參加工業團體或商業團體者，不得當選工會理事、監事等工會幹部職務。《修正條文第十九條》

* [[工會|工會]]聯合組織之解禁

舊法第十章(第四十七～五十一條)中，規範[[工會|工會]]的聯合組織，但於新法中完全刪除，僅於第八條第一款中明定「[[工會|工會]]得依需要籌組聯合組織；其[[名稱|名稱]]、層級、區域及屬性，應於聯合組織章程中定之。」

* [[僱主|僱主]]之不當勞動行為的禁止與處罰

為避免[[僱主|僱主]]對求職者、工會幹部、工會發起人、工會會員等，有不當[[解僱|解僱]]、降調之不利待遇，並為加強保護[[勞工|勞工]]加入、籌組[[工會|工會]]之[[權利|權利]]，特於本法明定[[僱主|僱主]]妨害[[工會|工會]]組織、運作及對工會幹部之不利益待遇等[[不當勞動行為|不當勞動行為]]態樣《修正條文第三十五條》，並另訂有處罰規定。《修正條文第四十五條》

* 限縮禁止組織[[工會|工會]]之對象
除現役[[軍人|軍人]]與[[中華民國|國防部]]所屬及依法監督之[[軍火工業|軍火工業]]員工外，[[勞工|勞工]]及[[教師|教師]]均有組織及加入[[工會|工會]]之[[權利|權利]]。但[[公務人員|公務人員]]之結社組織，另依其他[[法律|法律]]規定。《修正條
文第四條》<ref>{{Cite web |url =  http://humanrights.cy.gov.tw/dl.asp?fileName=0113010272671.pdf |title = 我國勞動三權之發展現況與問題分析——歷史、結構與法律的分析途徑 |author = 林良榮 |publisher = 國立高雄大學財經法律學系 |language = zh-hant |accessdate = 2014-06-18 }}</ref> <ref>{{Cite web|url =http://book.mol.gov.tw/image/no_22/A-5.pdf|title =建構工會新發展 《工會法》修正重點|author =王厚誠|date =2010年6月|publisher =台灣勞工季刊|language =zh-hant|accessdate =2014-06-18|deadurl =yes|archiveurl =https://web.archive.org/web/20160304073311/http://book.mol.gov.tw/image/no_22/A-5.pdf|archivedate =2016-03-04}}</ref>

=== 《團體協約法》 ===
{{wikisource|團體協約法}} 
* 確立誠信協商的義務：
過去[[工會|工會]]體制較為薄弱、欠缺自主性，導致勞雇雙方所簽訂之團體協約的覆蓋率極低，即使勞雇雙方簽訂團體協約，觀察其內容，可察覺只徒有形式，[[工會|工會]]未能發揮應有的作用，形同虛設。

因此，於新修正《團體協約法》中導入誠信協商原則，企業無正當理由拒絕協商的義務，佯裝拖延或刻意[[杯葛|杯葛]]協商，以及拒絕提供協商必要資料等行為，將會被處以罰鍰，且違反誠信協商的義務，亦可透過《勞資爭議處理法》處理之，命令違規者在特定期間為或不為一定行為。

=== 《勞資爭議處理法》 ===
{{wikisource|勞資爭議處理法}}
* 爭議行為概念重建：
經修正之《勞資爭議處理法》將爭議行為定義為：「[[勞資爭議|勞資爭議]]當事人為達成其主張，所為之[[罷工|罷工]]或其他阻礙事業正常運作及與之對抗之行為。」並將[[罷工|罷工]]行為重新定義：「[[勞工|勞工]]所為暫時拒絕提供勞務之行為。」使勞資雙方有明確的法源可供行為準則。

此外，新修訂的法規亦承認特定人數以上之[[勞工|勞工]]個人具有勞方之「勞資爭議當事人」的地位，使其有進行爭議行為之[[權利|權利]]，放寬爭議權所保護的範圍。<ref>{{Cite web|url =http://book.cla.gov.tw/image/no_22/A-2.pdf|title =從經濟全球化 談集體勞動三法修正的意涵與影響|author =洪清海|date =2010年6月|publisher =台灣勞工季刊|language =zh-hant|accessdate =2014-06-18}}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

== 修正條文對[[教師|教師]]勞動權益之疑慮 ==
{{wikisource|教師法 (中華民國)}}
於1995年通過之《教師法》，訂定國內[[教師|教師]]可依本法組織[[學校|學校]]教師會、地方縣（市）教師會及中央全國教師會。此一規範僅對[[教師|教師]]之團結權有所突破，[[勞動三權|勞動三權]]中另二權，集體協商權極其有限，爭議權完全無提及。

近十幾年來，教師組織持續呼籲，期望[[教師|教師]]能納入《工會法》，似一般[[勞工|勞工]]，可組織[[工會|工會]]。如今，勞動三法修正的通過與施行，顯示出對於[[教師|教師]][[勞動三權|勞動三權]]的保障有所突破，然而，仍不甚完美，[[教師|教師]]與一般[[勞工|勞工]]間之[[勞動三權|勞動三權]]保障，尚處不平等的地位，以下就新修正之勞動三法，謹列出三大限制對於教師工會的未來發展前景堪慮：

* 教師只可組織職業工會或產業工會，不可於各級[[學校|學校]]內組織學校層級之企業工會。

* 公立學校簽定團體契約前須事先取得上級主管機關的核可。

* [[教師|教師]]不得行使[[罷工|罷工]]。<ref>{{Cite web|url =http://www.emprel.org/index.php?option=com_attachments&task=download&id=53|title =教師工會與勞動三法|author =衛民|date =2010|publisher =勞動與就業關係季刊|language =zh-hant|accessdate =2014-06-18}}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>
