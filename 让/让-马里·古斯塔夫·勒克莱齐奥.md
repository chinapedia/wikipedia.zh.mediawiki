'''勒克莱齐奥'''（[[法语|法语]]：{{lang|fr|Jean-Marie Gustave Le Clézio}}，{{bd|1940年|4月13日|||Le Clézio}}），生于法国尼斯，持有法国和毛里求斯双国籍，著名法语作家，是20世纪后半期法国新寓言派代表作家之一。<br />
<br />
随着其第一篇小说《[[诉讼笔录|诉讼笔录]]》(1963)的出版而一举成名，直到1970前后，其作品仍带有“新小说”的痕迹，基本以探索人性疯狂为主题，之后，受其家族起源，自己游历及对美洲印第安文化的喜爱，作品开始以寓言，幻想为主题(《[[沙漠|沙漠]]》《[[寻金者|寻金者]]》)，题材涉及儿时回忆，自传及家族历史等(《[[非洲人|非洲人]]》)。最近以来，他写作出版了四十多部著作，包括小说，随笔等。<br />
<br />
2008年10月9日，他因为“将多元文化、人性和冒险精神融入创作，是一位善于创新、喜爱诗一般冒险和情感忘我的作家，在其作品里对游离于西方主流文明外和处于社会底层的人性进行了探索”而获得[[诺贝尔文学奖|诺贝尔文学奖]]。<ref>[http://www.apdnews.com/news/50861.html 诺贝尔文学奖获得者] {{webarchive|url=https://web.archive.org/web/20131207003908/http://www.apdnews.com/news/50861.html |date=2013-12-07 }}，亚太日报，2013年12月3日</ref>

== 生平简介 == 


勒克莱齐奥1940年出生于法国[[尼斯|尼斯]]，其父亲Raoul Le Clézio是医生，其母亲Simone Le Clézio与父亲是嫡亲兄妹(同祖父)。18世纪，其家族从法国[[布列塔尼|布列塔尼]]迁到当时法属殖民地[[毛里求斯|毛里求斯]]，之后被英国吞并而获得了英国国籍。因此，勒克莱齐奥自己也认为是受法国文化和毛里求斯文化这两种文化的影响。<br />
<br />
他7岁时和母亲一同前往[[尼日利亚|尼日利亚]]与父亲团聚，就是在前往尼日利亚的这长达一个月的旅程中开始了自己的文学生涯——写了两本书《[[漫长的旅行|漫长的旅行]]》、《[[黑色的奥拉迪|黑色的奥拉迪]]》。他和家人1950年重返尼斯，在完成中学教育后，他1958-1959年在英国[[布里斯托尔大学|布里斯托尔大学]]学英语，1963年在[[尼斯大学|尼斯]]获得了文学院的文学学士学位，1964年在艾克斯[[普罗旺斯大学|普罗旺斯大学]]攻读硕士学位，1985年在[[佩皮尼昂大学|佩皮尼昂大学]]撰写了有关墨西哥早期历史的博士论文。<br />
<br />
后来，他曾在[[曼谷|曼谷]]、[[墨西哥城|墨西哥城]]、[[波士顿|波士顿]]等地的[[大学|大学]]任教。受在不同国家工作和生活的影响，克莱齐奥的作品融入了欧洲、非洲和拉丁美洲等地的不同元素。

== 评价 ==

== 漢譯 ==
第1本被譯介到[[漢語|漢語]]世界的作品是[[金龙格|金龙格]]編譯的《少年心事》（1992年在[[廣西|廣西]][[桂林|桂林]]出版），其他還有《戰爭》、《訴訟筆錄》、《[[流浪的星星|流浪的星星]]》、《金魚》、《乌拉尼亚》等。
=== 在台灣 ===
台灣譯名'''勒·克萊喬'''，2000年，[[邱瑞鑾|邱瑞鑾]]譯出《金魚》，2006年，[[藍漢傑|藍漢傑]]、[[蔡孟貞|蔡孟貞]]譯出《偶遇》（含〈偶遇〉和〈安格利·馬拉〉2篇作品），2009年，[[尉遲秀|尉遲秀]]譯出《飢餓間奏曲》，為得獎後第一部作品，2011年，[[藍漢傑|藍漢傑]]譯出《沙漠》。
