'''若開邦'''（{{lang-my|ရခိုင်ပြည်နယ်}}）是[[緬甸|緬甸]]西南的一個邦，與[[孟加拉|孟加拉]]接壤。面積36,780平方公里，2014年該邦人口為318.8萬人<ref>[https://drive.google.com/file/d/0B067GBtstE5TeUlIVjRjSjVzWlk/view 2014 Myanmar Censu]</ref>。首府[[實兌|實兌]]，下轄5縣，邦境以南北向的[[若開山脈|若開山脈]]為主，島嶼較多。“若開”一詞來自[[巴利語|巴利語]]，即[[食人魔|食人魔]]之地，可能是指其[[矮黑人|矮黑人]]原住民。

== 歷史 ==
古稱阿拉干王国，自古以[[上座佛教|上座佛教]]立国，在文化上与缅甸更为接近，与西边的孟加拉地区也长期和睦相处。在阿拉干王国的古都妙乌（Mrauk U），至今仍保留着一座几百年前[[回教徒|回教徒]]的[[清真寺|清真寺]]。不过，那个时候，定居在阿拉干的回教徒人数很少，主要是从事贸易的商人，今日若開邦總人口數約三百五十萬人，該地分為兩大族群：若開族和羅興亞人。

Dhanyawadi、Waithali、Lemro、[[謬烏王國|謬烏王國]]（Mrauk U）、貢榜王朝佔領的獨立王國（1784年～1826年）、英國統治（1826年～1948年）、緬甸（1948年～）的邦（省）。

== 宗教衝突 ==
若開邦居民[[若開人|若開人]]（Rakhine）多数為信奉[[上座部佛教|上座部佛教]]（南傳佛教）[[佛教徒|佛教徒]]，少数信奉[[伊斯兰教|伊斯兰教]]；由於謬烏王國国王[[明蘇蒙|明蘇蒙]]曾经在临近的[[孟加拉蘇丹國|孟加拉蘇丹國]]避难，因此有了[[穆斯林|穆斯林]]社區，也成为[[緬甸|緬甸]]全国[[穆斯林|穆斯林]]最多的地区。以[[罗兴亚人|罗兴亚人]]（Rohingya）为主的穆斯林人口，居该邦总人口约四分之一。但由于历史和宗教原因，缅甸官方指為是来自孟加拉国的非法定居者，因此不享有[[公民权|公民权]]。
{{main|羅興亞人在緬甸西部的衝突|2012年若開邦騷動}}
2012年6月中，若開邦的穆斯林（[[罗兴亚族|罗兴亚族]]）與佛教徒（若開族、[[緬族|緬族]]等）爆發衝突，緬甸總統[[登盛|登盛]]於6月10日晚間宣布若開邦進入緊急狀態。罗兴亚族穆斯林的大量外逃。與2016年11月初，相似情事再次發生，約3萬人逃離家園。<ref>[http://udn.com/news/story/5/2130027 控緬甸種族淨化洛興雅人 亞洲爆示威]</ref>

== 經濟 ==
若開邦是緬甸最貧窮的州之一。69％以上的人口生活貧困。

水稻是該地區的主要作物，佔農業用地總面積的85％左右。椰子和尼帕棕櫚種植園也很重要。捕魚業是一個主要產業，大部分漁獲運到[[仰光|仰光]]，但也有一些出口。從山上提取木材，竹子和木炭等木製品。少量的劣質原油是從原始的，淺的，手挖井中生產出來的，但石油和天然氣生產尚未發現。

旅遊業正在慢慢發展。古老的皇家城鎮[[妙烏|妙烏]]的廢墟和Ngapali海灘度假勝地是外國遊客的主要景點，但設施仍然是原始的，交通基礎設施仍然不成熟。
