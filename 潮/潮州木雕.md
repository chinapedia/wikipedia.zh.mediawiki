'''潮州木雕'''，又称'''潮州金漆木雕'''，是流行于[[广东省|广东省]]原[[潮州府|潮州府]]属[[潮州市|潮州市]]、[[汕头市|汕头市]]、[[揭阳市|揭阳市]]等地的民间雕刻艺术，尤以多层镂通和金碧辉煌的金漆木雕而著称于世<ref>[http://www.chaofeng.org/article/detail.asp?id=712  潮州木雕]</ref>，与[[东阳木雕|东阳木雕]]、[[黄杨木雕|黄杨木雕]]、[[龙眼木雕|龙眼木雕]]并称为中国四大木雕<ref>{{cite journal|publisher=中新网 |date=2010年10月31日|title=岭南写真：潮州木雕后继乏人 遭遇外来木雕抢滩 |url=http://www.chinanews.com/cul/2010/10-31/2623911.shtml}}</ref>。潮州木雕源远流长，发展于[[唐|唐]][[宋|宋]]，成熟于[[明|明]]末，至[[清|清]]中晚期达到鼎盛。于2006年5月20日列入第一批[[中国国家级非物质文化遗产|中国国家级非物质文化遗产]]名录。<ref>[http://www.ihchina.cn/inc/guojiaminglunry.jsp?gjml_id=339 国家名录339潮州木雕]{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

== 历史 ==
潮州木雕早期风格较为粗犷，至明代已很精美，日用木雕则起源于清代。其历史最早可以追寻到[[唐昭宗|唐昭宗]]年间，在[[刘恂|刘恂]]的《[[岭表异录|岭表异录]]》中<ref>《岭表异录》(卷上)：“广之属郡，潮、循州多野象，矛小而红，最堪作笏。”</ref><ref>《岭表异录》(卷中)：“枹木产江溪中，叶细如桧，身坚类柏，惟根软不胜刀锯。今潮、循人多用其根，刳而为履。当未干时，刻削易如割瓜；既干之后，柔韧不可理也。或油或漆，其轻如通草。暑月著之，隔卑湿地气，有力如杉木。今广州宾从诸郡牧守，初到任，下檐皆有油画枹木履也。”</ref><ref>《岭表异录》(卷中)：“枹椽子，形如瓜，皮似橙而金色，故人重之，爱其香气，京辇豪贵家仃盘筵，怜其远方异果。肉甚厚，白如萝卜。南中女子竞取其肉雕镂花鸟，浸之蜂蜜，点以胭脂，擅其妙巧，亦不让湘中镂木瓜也。”</ref>，就有三则有关潮州雕刻艺术的记载。而潮州金漆木雕的出现，则在北宋[[至和|至和]]元年（1054年）就有文献记录了<ref>《[[永乐大典|永乐大典]]》“潮”字，卷5345林汶可《刻漏记》：“乃择牙校就汀受法，指工绳木，丱金涂漆。历四旬，凡总六十事件，而漏刻成”</ref>。而始建于[[唐玄宗|唐玄宗]]开元二十六年（738年）的[[潮州开元寺|潮州开元寺]]，寺内现保存有一条长约1米[[唐代|唐代]][[木鱼|木鱼]]，及一条[[北宋|北宋]][[政和|政和]]四年的木龙。而[[许驸马府|许驸马府]]亦保存有宋代梁架雕饰，虽然这些木雕饰品刀法简朴，但说明在唐宋时期，潮州已有建筑装饰的木雕工艺了<ref>{{cite book|title=《潮州木雕》 |author=杨坚平 |publisher=广东人民出版社 |date=2008-03-01 |isbn=9787536238039 |pages=第19页|chapter=第二章}}</ref>。

== 艺术特色 ==
[[File:Chaozhou_Woodcarving_in_PN4.jpg|thumb]]
===种类繁多===
潮州木雕作为完全独立的摆设是基本没有的，通常是与建筑、家具及神器结合在一起而作为它们的装饰或构件的。按照用项区分，大致可以分为建筑装饰、神器装饰、家具装饰和案头摆设四类<ref>{{cite book|title=《潮州木雕》 |author=杨坚平 |publisher=广东人民出版社 |date=2008-03-01 |isbn=9787536238039 |pages=第107页|chapter=第七章}}</ref>。
====建筑装饰====
包括牌匾狮、进屏、屐头、檐角、柱头、横梁、门楣、屏风、门饰、窗格、栏杆及梁的雕饰百多种。
====神器装饰====
神器装饰，也称宗教用品装饰品，分有五类：龛椟类(包括大龛、椟仔)，馔盒类(包括馔盒、果碟、糖枋架、饼架等)，宣炉罩类(四方型、八角型、圆筒型、木雕加玻璃型等)，神轿神亭类；游神配套类(包括茶担、鼓架、烛台、香架、香炉等)。
====家具装饰====
包括床、几、桌、椅、茶橱、衣柜、梳妆台、纸煤筒、灯心筒等常见家具。
====案头摆设====
欣赏型的小型木雕

===题材的选择===
木雕所要表达的内容取材广泛，有民间故事小说也有戏剧故事，或是地方熟悉的历史人物，更多是来源于普通民众的生活。
== 雕法 ==
雕刻形式，分[[浮雕|浮雕]]、[[沉雕|沉雕]]、[[圆雕|圆雕]]和[[通雕|通雕]]四类<ref>{{cite book|title=《潮州木雕》 |author=杨坚平 |publisher=广东人民出版社 |date=2008-03-01 |isbn=9787536238039 |pages=第124页|chapter=第七章}}</ref>。

== 制作过程 ==
潮州木雕的制作大致分为三个阶段，第一是“起草图”，第二是“凿粗胚”，第三是“细雕刻”。
=== 木材的选用 ===
潮州木雕常用的木料有以下几种：[[樟木|樟木]]、[[杉木|杉木]]、[[苦楝|苦楝]]、[[黎木|黎木]]、[[杀柴|杀柴]]、[[柯木|柯木]]、[[合欢木|合欢木]]、[[榕木|榕木]]及[[柚木|柚木]]等。这些具有的共同特点，那就是质地结实，又有强大的坚韧力。实际运用中要根据饰物的品类、用途和内容，来分别选用所需木料。其中樟木是主要的木材用料<ref>{{cite book|title=《潮州木雕》 |author=杨坚平 |publisher=广东人民出版社 |date=2008-03-01 |isbn=9787536238039 |pages=第136页|chapter=第八章}}</ref>。

=== 工具的选用 ===
[[File:Tools_of_Chaozhou_Woodcarving.jpg|thumb]]
除了画草图需要的[[弓把|弓把]]与[[夹仔|夹仔]]，常用的工具有以下几种
*[[叩槌|叩槌]]，即敲击凿子用的木槌
*[[方凿刀|方凿刀]]，即平口刀
*[[方湖刀|方湖刀]]，刀刃稍弧形，用来做出稍有弧度的形状
*[[圆凿刀|圆凿刀]]，刀刃弧形，用来做出较圆的形状
*[[挠凿刀|挠凿刀]]，刀口翘起与刀梗成一定角度，用于削平深层其它刀不能触及的地方
*[[雕刀|雕刀]]，刀刃成锐角形，用于刻划线条、花纹。
*[[毛尾刀|毛尾刀]]，镂空雕刻的特制刃刀，形成长锐角形，用于侧面的造型刻画<ref>{{cite book|title=《潮州木雕》 |author=杨坚平 |publisher=广东人民出版社 |date=2008-03-01 |isbn=9787536238039 |pages=第138页|chapter=第八章}}</ref>。

=== 上漆 ===
潮州除大幅度用髹漆贴金之外，还有水漆、退光漆、刀漆、金漆画、五彩漆画等等工种。一般髹漆贴金的需要制漆、滤漆、填料、上漆四道工序<ref>{{cite book|title=《潮州木雕》 |author=杨坚平 |publisher=广东人民出版社 |date=2008-03-01 |isbn=9787536238039 |pages=第147页|chapter=第八章}}</ref>。
