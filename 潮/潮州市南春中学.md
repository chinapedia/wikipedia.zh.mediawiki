'''潮州市南春中学'''，是位于[[中华人民共和国|中华人民共和国]][[广东省|广东省]][[潮州市|潮州市]]的一所市直普通高中，于1976年8月创办，被评为“潮州市一级学校”，是潮州市四所重点高中之一（其余三个为[[潮州金山中学|金山中学]]、[[潮州市高级中学|高级中学]]、[[潮州市华侨中学|华侨中学]]）。

南春中学占地面积17000平方米，现有56个教学班，208名教师，2000多名学生。现任校长陈永序。校址位于[[潮州市|潮州市]]湘桥区南春路和枫春南路的交界处。

== 历史沿革 ==
* 1976年8月，在[[潮安县|潮安县]]人民政府购买了中共汕头地区（现[[汕头市|汕头市]]）委员会党校校舍后，开始创办'''南春初级中学'''，由陈烈泉等人负责筹建工作。
* 1977年3月，南春初级中学正式开学上课，庄建国任该校第一任校长兼党支部书记。
* 1980年9月，南春中学办成完全中学，陈芳清副校长主持全面工作，上台担任第二任校长兼党支部书记。
* 1982年7月，因为[[华国锋|华国锋]]政府在1977年恢复了[[高考|高考]]制度，南春中学的首届高中毕业生参加高考。
* 1992年4月，潮州市升格为[[地级市|地级市]]，同时湘桥区挂牌成立，南春中学在行政上划属湘桥区。
* 2000年，南春中学晋升为“潮州市一级学校”。
* 2005年，南春中学开通官方网站，同时该校取消了初中部。

== 现况 ==
=== 课程设置 ===
潮州市南春中学设有13个课程：政治、语文、数学、英语、物理、化学、历史、地理、生物、体育、音乐、美术、计算机。教师使用普通话、潮州话、英语（仅英语课）授课，每节课程与其它学校一样，长达45分钟。

=== 文化生活 ===
潮州市南春中学每年举办一届艺体节，艺体节包括文艺汇演、校运会、合唱比赛等等。

== 校园设施 ==
教学楼（共2栋）、综合楼、艺体馆、运动场、植物园、地理园。

== 社团活动 ==
潮州市南春中学的学生社团活动非常活跃，丰富多彩，组建了很多学生社团组织。

以下是该校的部分学生社团：

Soul舞蹈社、音乐社、赤魂动漫社、知秋棋牌社、幻手社、鹿鸣汉文化社、希之翼义工社、绿叶文学社、蒲公英心理协会、羽毛社、合唱团、记者站、广播站
