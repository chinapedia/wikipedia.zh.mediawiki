{{日本作品|潮與虎|うしおととら}}是[[日本|日本]][[漫畫家|漫畫家]][[藤田和日郎|藤田和日郎]]在[[小學館|小學館]]《[[週刊少年Sunday|週刊少年Sunday]]》的連載漫畫作品，發表於1990年第6號至1996年第45號。作品主要讲述了一個平凡少年——蒼月潮，和一隻活了數千年的妖怪——阿虎，两者之间发生的冒險故事。簡稱「{{lang|ja|うしとら}}」。

== 剧情介绍 ==
初中生少年蒼月潮與擔任寺廟住持的父親兩人共同生活。
某天，潮在自家倉庫的地下室中遇到了不得了的東西。那是在遙遠的過去，身為凶惡的怪物被人民所恐懼、被“獸矛”封印了五百年的大妖怪。把矛拔出來的潮眼看就要被妖怪吃掉，但與此同時，他由於自沉睡中甦醒的“獸矛”的力量而變身，一瞬間就將聚集了妖氣的其他妖怪擊破。
被矛的威力所震懾而不假思索地要求合作的大妖怪，被潮起了“阿虎”這個名字，兩人開始共同行動。
如此這般，成為“獸矛”傳承者的少年，與一直想找機會吃掉他的大妖怪所組成的“潮虎”組合誕生了。之後，接連將襲來的妖怪們打倒的他們，被捲入了關乎世界存亡的壯大戰鬥當中……

== 用語 ==
; {{nihongo|獸矛|獣の槍}}
: 使用獸矛的人交付靈魂而取得力量。失去靈魂的人換得強大力量及戰鬥後傷勢的快速回復，但卻隨著靈魂逐漸減少而減緩傷勢回復，甚至獸化，獸化最後變成「字伏（阿虎那個樣子）」。獸矛中含有2個人類，也就是變成柄的義龍及做成矛刃的決眉，但是藏在其中的靈魂只有充滿憎恨的義龍。單純的獸矛已經可以殺滅所有妖怪，由此可以說明「人類的憎恨」是最可怕的武器，而使用者的靈魂只是變換自己肉體而得以掌握獸矛的交易。
: 在最終決戰中，失去精神控制的潮以獸矛插向白面額頭，卻慘被捲成碎片。但是卻因禍得福，變成碎片的獸矛反而飛向全日本破壞棲息在人類腦中的婢妖，從而讓人類想起還有希望，阻斷了白面者力量食糧：恐懼。
: 雖然劇情交代最後可以在阿虎體內重新組合回原本的獸矛，但事實上究竟是只能在阿虎體內可以合成，或是任何生物、妖怪體內皆可合成並未說明。但阿虎是一個特殊存在，既是白面者肉體的創造者，又是第一名獸矛持有人，獸矛因白面而產生，而由「特殊存在」的體內而再生（有如白面者的產生一樣），也說得過去。但無論如何，並沒有任何定論。

; {{nihongo|艾雷扎爾鐮刀|エレザールの鎌}}
:法力僧引狹為了除去白面者而創造的鐮刀，最後給予霧雄使用。鐮刀雖然有許多把，但只有霧雄手上這一把才有特殊增幅法力的功效，其他的只是普通一般的法器，或者說是艾雷扎爾鐮刀的「瑕疵品」。

; {{nihongo|字伏盔甲|字伏}}
: 由4隻字伏所組成的。這些字伏原本為了避免失去記憶而石化，等候最終戰的到來再回復身體，卻被白面分身偷襲而粉碎剩下頭部。為了希望在最終戰時與白面者決戰，便化成盔甲要求阿虎帶他們上戰場。在阿虎落海受真由子鼓勵復活後，與阿虎合體前去與潮對決白面者。有主動防禦的能力，但是依然抵擋不了白面者的攻擊，被第七、八尾粉碎。後來變成白面者的模樣，成功破壞第七、八尾，為潮與阿虎帶來反擊的機會。
: 縱觀全劇，全日本沒有任何一隻妖怪能夠單獨抵擋白面者任何一條尾巴的攻擊。有人認為與其說他們戰力不足，不如說是為了襯托「阿虎」的特別，例如阿虎並沒有對白面者特別的憎恨執念、阿虎從未石化、阿虎保有自己意志等。在毀滅之前，對阿虎說「你不需要我們，因為你已經夠強了」，又說「對白面者的憎恨到了極點，就會變成白面者」而變化為白面者的頭攻擊白面者。嚴格說起來，這是動漫鐵律「主角威能」的展現，即使你和主角是一樣的東西，仍然無法像主角一樣強大。

; {{nihongo|錫杖|錫杖}}
: 僧人出外化緣的手杖，上面的圓環震動發出聲響讓人知悉而化緣。在本作中另外作為降妖扶魔的法器，依據個人使用不同，會有不同的設計，如秋葉的手杖可以拆開變成細枝插入阿虎手足以封住行動等。

== OVA ==
1992年至1993年發行，全10話。2002年6月25日DVD化發行，並收錄2頭身SD版的番外篇。另外，第1話至第6話為第1期，第7話至最終話以第2期作稱呼（第2期一部分工作人員和主題曲）。第9、10話『狂風』則在發行之前先行以劇場版的方式，1993年6月5日公開。

第8話有下集預告。之後部分工作人員另外成立動畫工作室[[OLM|OLM]]，其相關人員則參與電視動畫《[[神奇寶貝_(動畫)|神奇寶貝]]》的製作。

== 遊戲 ==
全部由YUTAKA（{{lang|ja|ユタカ}}）發行，對應機種為[[任天堂|任天堂]]的紅白機種，兩款都在1993年發行。
* '''潮與虎'''：1月25日發行，對應機種為[[超級任天堂|超級任天堂]]，類型為動作類遊戲。
* '''潮與虎'''：7月9日發行，對應機種為[[FC遊戲機|FC遊戲機]]，類型為角色扮演遊戲。

== 獎項 ==
该作曾荣获第37回[[小學館漫畫獎|小學館漫畫獎]]少年向部門、1997年[[星雲獎_(日本)|星雲獎]]漫畫部及2006年的{{link-ja|日本媒體藝術100選|日本のメディア芸術100選}}漫畫部門（《[[週刊少年Sunday|週刊少年Sunday]]》唯一得獎連載作品）。

== 舞台劇 ==
2008年起，由作者公認的劇團「Theatre OM」改編成舞台劇，在關西地區以大阪、櫻川為中心公開演出。

== 與GS美神極樂大作戰的相關性 ==
{{Original research|章節|date=2015年9月}}
與一樣在[[週刊少年Sunday|週刊少年Sunday]]連載[[椎名高志|椎名高志]]原作的《[[GS美神極樂大作戰|GS美神極樂大作戰]]》中，該作品女主角美神令子在劇中曾出現拿著獸矛打鬥的場面。而在動畫版《GS美神》方面，美神令子的配音員由鶴弘美擔任，也在OVA版的潮與虎飾演篝。另外，潮與虎的主要人物蒼月潮和中村麻子以路人角色在背景中出場（漫畫版<ref>單行本《GS美神極樂大作戰》第6冊，小學館，1993年3月，第1刷，第86頁 ISBN 978-4-0912-3036-2。</ref>和電視動畫版都有出現）。
