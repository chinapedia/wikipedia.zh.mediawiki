'''涂岭镇'''是[[中华人民共和国|中国]][[福建省|福建省]][[泉州市|泉州市]][[泉港区|泉港区]]下辖的一个[[镇|镇]]。

== 历史沿革 ==
涂岭镇[[宋朝|宋朝]]属城山乡。[[元朝|元朝]]，属忠恕乡。[[清朝|清朝]]，属东坪铺、鳌塘铺、承天铺、樟市铺。

[[中华民国|中华民国]]23年，设涂岭区公所。民国36年，改涂岭乡。

1949年9月，改设第八区。期间经过多次行政区划变动。1956年6月，涂岭区撤销并设立驿坂区。1958年，撤区并乡并设涂岭乡、驿坂乡和东寨乡，后合并为涂岭乡。1958年9月，人民公社化并属超先公社，设涂岭管区。1961年，成立涂岭人民公社。1984年10月，成立涂岭乡。1992年，置[[蕭厝鎮|蕭厝鎮]]涂岭街道。1996年，改设涂岭镇。

== 人口 ==
涂岭镇小坝村为福建省最大的[[蒙古族|蒙古族]]聚居村<ref>{{cite news|author=陈明华|title=泉港深山里藏着蒙古族聚居村|url=http://news.qzwb.com/gb/content/2010-07/12/content_3389324.htm|accessdate=2010年7月12日|newspaper=东南早报|date=2010-07-12}}</ref>。

== 交通 ==
[[国道324线|国道324线]]和涂肖二级公路经过该镇，[[福厦高速公路|福厦高速公路]]在该镇驿坂村设有互通口。

另外，[[漳泉肖铁路|漳泉肖铁路]]经过境内。
