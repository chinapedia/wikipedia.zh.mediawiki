'''囗部'''，是為漢字索引裡為[[部首|部首]]之一，[[康熙字典|康熙字典]]214個部首中的第三十一個（三劃的則為第二個）。就[[繁體|繁體]]和[[簡體|簡體]]中文中，囗部歸於三劃部首。囗部將上下左右包合起來，形成全包圍結構為部字。且無其他部首可用者將部首歸為囗部。要注意的是，囗部與[[口部_(部首)|口部]]不同，口部只出現在字的旁邊，囗部只包圍字的周圍。

漢字〇因形狀與囗相似，也多被歸為此部（部外筆畫為0），如《漢語大字典》、《中華字海》、《新華字典》、《現代漢語詞典》。但亦有將〇歸入[[乙部|乙部]]（因為乙部包括了所有非橫豎撇點的筆畫，〇亦屬此類）。

== 部首單字解釋 ==
===囗===
# wéi：圍的古字；同「囲」。
# guó（ㄍㄨㄛˊ）：「國」的古字。

===〇===
# xīng（ㄒㄧㄥ）：同“'''[[天體|星]]'''”。[[唐|唐]][[武則天|武則天]]所造的字，後世稱「[[則天文字|則天文字]]」。《[[集韻|集韻]] · 青韻》：曐，或省作「[[星|星]]」，唐武后作「[[〇|〇]]」。
# líng（ㄌㄧㄥˊ）：「[[零|零]]」的小寫，但零也被當作小寫用。多用於書寫時占位。
