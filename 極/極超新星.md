'''極超新星'''（{{lang-en|'''superluminous supernova'''}} 或 {{lang|en|'''hypernova'''}}。也称为'''骇新星'''），是[[超新星|超新星]]的一種，是年老的[[特超巨星|極超巨星]]在臨終前的爆發。這種超新星的威力比起一般的超新星要大得多，剩下的核心會直接塌縮為[[黑洞|黑洞]]，在黑洞自轉的兩極會以接近[[光速|光速]]射出高能量[[等離子體|等離子體]]，充著[[伽玛射线|伽玛射线]]，成為科學家認為[[伽玛射线暴|伽玛射线暴]]的可能源頭之一。透過近幾年的多項觀測結果，人們對伽馬射線暴的瞭解增多。

1990年代，极超新星是指爆炸能量相当于100倍[[超新星|超新星]]爆发。现在则将所有极超巨星发生的爆发全部归入极超新星的范畴<ref>{{cite news|author=晨风|title=科学家称极超新星爆发可能摧毁外星生命|url=http://news.sciencenet.cn/htmlnews/2011/3/245436.shtm|accessdate=2011-03-25|newspaper=科学网|date=2011-03-25}}</ref>。

又由於會演變為[[黑洞|黑洞]]的超巨星的數量極少，極超新星爆發的現象也同樣極少，[[天文學家|天文學家]]預測在我們的[[銀河系|銀河系]]內，平均每兩億年會出現一顆極超新星。<ref>{{cite web|title = Superluminous Supernovae |url = https://www.cfa.harvard.edu/news/su201710 |publisher = [[史密松天體物理台|史密松天體物理台]] |date = 2017-03-10 |accessdate = 2017-04-15 |language = en |archiveurl = https://web.archive.org/web/20170416045246/https://www.cfa.harvard.edu/news/su201710 |archivedate = 2017-04-16 |deadurl= no}}</ref>

== 對地球的可能威脅 ==
人們認為，如果一顆極超新星在地球附近的位置爆發，其高能[[輻射|輻射]]定會摧毀地球的生命。根據2005年由[[美國太空總署|美國太空總署]]及[[堪薩斯大學|堪薩斯大學]]的科學家作出之共同研究結果，指出地球於4億5000萬年前的[[奧陶紀-志留紀滅絕事件|奧陶紀-志留紀滅絕事件]]，有可能是一顆極超新星釋出的伽馬射線暴引起的，其過程持續了十秒，並摧毀了地球一半的[[臭氧層|臭氧層]]，使得太陽釋出的[[紫外線|紫外線]]襲擊地球，導致地面及近海面的大量生物[[死亡|死亡]]，從而破壞[[食物鏈|食物鏈]]。

如果極超新星僅發生於比太陽質量大40倍的[[恆星|恆星]]上，那地球將會安全，倖免於難{{可疑}}。而最接近地球的極超新星之可能爆發者，是[[海山二|海山二]]（船底座η）星，它距離地球7,500[[光年|光年]]，質量則為太陽的150倍。
