== 遊戲與劇情 ==
=== 遊戲 ===
本作回歸到線性劇情導向，而劇情的形式與電影《[[炮彈飛車|炮彈飛車]]》相似<ref>[http://nfs.wikia.com/wiki/Need_for_Speed:_The_Run Need for Speed: The Run at The Need for Speed Wiki - Need for Speed series information]</ref>，而故事的男女角特別請來了影視雙棲的[[克莉絲汀娜·韓翠克絲|克莉絲汀娜·韓翠克絲]]和[[西恩·法瑞斯|西恩·法瑞斯]]，不僅擔任配音員，甚至連角色造型也與真人相似<ref>{{Cite web
 |url         = http://xboxlife.tw/show.php?fid=2&tid=2005
 |title       = Need for Speed: The Run《極速快感：亡命天涯》邀來兩大名星擔當演出
 |accessdate  = 2011-10-24
 |author      = GYang
 |work        = Xbox Life
 |deadurl     = yes
 |archiveurl  = https://web.archive.org/web/20111025234535/http://xboxlife.tw/show.php?fid=2&tid=2005
 |archivedate = 2011-10-25
}}</ref>，玩家要從[[舊金山|舊金山]]一路開到[[紐約|紐約]]並打敗211位車手，路途中有許多挑戰，像是機關槍掃射、雪崩等。除此之外遊戲的車輛損傷度也由血條改為回朔點，如果玩家不慎撞毀車輛或出界可以回到上一個檢查點。<br />
與歷代劇情導向遊戲不同的是，這次遊戲的劇情首次採用[[QTE|QTE]]，讓玩家身歷其境的處於冒險之中。

=== 模式 ===
遊戲模式除了競速、時間賽等經典模式之外，還添加對決（Battle）、生存模式（Survial）。
*對決：一對一對決，在時限內超過對手，並且保持第一。
*生存模式：以機關槍掃射、雪崩等任何意外為主，玩家除了要跑完全程，而且還躲過任何意外。

=== 劇情 ===
故事從遊戲主角Jack Rourke從昏迷中甦醒開始，Jack醒來後發現自己雙手膠帶綁在方向盤，而且車輛懸空，當他發現自己處在絞碎機上方時才驚覺大事不妙，費了九牛二虎之力，在車輛完全絞碎前逃了出來，並驾車逃逸，一路上有追兵用機關槍掃射，所幸到達了平交道，在火車通過一瞬間迅速通過，甩掉追兵。

甩掉追兵之後，Jack到了唐人街與一位「The Run」賽事合夥人Sam Harper碰面，「The Run」的賽程是從舊金山到紐約3000英哩(4828公里)，入場費25萬美金，有211人參加，冠軍可獲得2500萬美金，Sam表示Jack可以抽一成。

隔天Jack在倉庫改裝完自己選的車之後，出發前Sam告訴他，到達[[拉斯維加斯|拉斯維加斯]]之前，排名一定要到達150名。而到達[[拉斯維加斯|拉斯維加斯]]之後，又有一個新目標，就是在抵達[[芝加哥|芝加哥]]前達到第50名的排名。經過一番考驗，終於跑完[[拉斯維加斯|拉斯維加斯]]，並抵達[[拉斯維加斯|拉斯維加斯]]大道，但被警察攔下後，趁混亂的時候（因其他對手引開其他警察注意）Jack立刻徒步來逃脫警察追捕並在飯店劫車逃離拉斯維加斯至芝加哥。到芝加哥沒想到就被追兵撞上，Jack下車馬上跑到屋頂上，沒想到就追兵有直升機，而直升機在對Jack掃射，Jack不小心掉到巷子裡，Jack發現了一台警車，趁警察不注意時，把警車搶走，以甩開直升機，直升機掃射油罐車，警車翻車掉在鐵路上，Jack立刻從警車逃出，並在一家車廠裡面再搶一台車已甩開追兵，甩開之後，Jack就到Sam所說的朋友Yuri的車庫裡，選一台車之後就以第一名的名次到達紐約。
