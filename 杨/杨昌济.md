'''杨昌济'''（{{bd|1871年|6月6日|1920年|1月17日|catIdx=Y}}），后改名'''懷中'''，字'''華生'''，[[湖南省|湖南省]][[长沙县|长沙县]]人。[[中国|中国]]近代著名[[教育家|教育家]]、[[伦理学家|伦理学家]]。

== 简介 ==
杨昌济家族世居[[长沙|长沙]][[东乡|东乡]][[开慧镇|板仓]]，故人称“板仓先生”、“板仓杨”。据族谱记载，他于同治十年辛未四月十九亥时出生，自幼喜读[[程朱理学|程朱之学]]，七岁入学进馆发蒙，启蒙老师是他父亲杨书祥。1888年与向振熙结婚。1889年参加长沙县试，一举考上“秀才”。但是两次[[乡试|乡试]]不中，只得在家中设馆授徒。<ref name="20世纪初：大学教授杨昌济 " >{{cite web |url=|title=20世纪初：大学教授杨昌济|accessdate=2009-09-09|work=|publisher=中国青年网|language=zh-hans}}</ref>

1898年，杨昌济就读于[[岳麓书院|岳麓书院]]，并且不顾岳麓书院山长[[王先谦|王先谦]]的阻挠，毅然决然加入了进步团体[[南学会|南学会]]，努力宣传新学、新政，赞同并且在湖南宣传[[谭嗣同|谭嗣同]]的“以民为主”的政治思想。多次聆听谭嗣同等人讲演，有一次发问获得谭嗣同高度肯定。{{notetag|谭嗣同：“于圣贤微言大义晦盲否塞之秋，独能发为如此奇伟精深之问，独岂秦汉以下之学者胸中所能有哉？兹事体大，余以何敢论断。总之，以民为主，如何可以救民，即以如何为是，则头头是道，众论皆通矣。”<ref>{{Cite book|author=杨开智|chapter=回忆父亲杨昌济|editor=王兴国|title=杨昌济文集|last=|first=|publisher=湖南教育出版社|year=2008|isbn=|location=长沙|pages=1287}}</ref>}}[[康梁|康梁]][[戊戌变法|戊戌变法]]失败后，杨昌济避居家乡，研究经世之学。<ref name="20世纪初：大学教授杨昌济 " /><ref name="杨昌济：世居板仓 思想进步 "/><ref name="中国教育家杨昌济诞生 " />

1903年，杨昌济考取官费留日，启程前改名“怀中”。入[[弘文学院|弘文学院]]速成示范科，后转普通科，结业后考入[[东京高等师范学校|东京高等师范学校]]。

1909年，由于[[杨毓麟|杨毓麟]]、[[章士钊|章士钊]]的推荐，<ref name="杨昌济：世居板仓 思想进步 "/> 攻读于[[英国|英国]][[阿伯丁大学|阿伯丁大学]]文科，专修哲学、伦理学。1912年毕业，后赴[[德国|德国]]考察教育。<ref name="20世纪初：大学教授杨昌济 " />

1913年至1918年，杨昌济先后任教于湖南高等师范学院、省立第四师范、[[湖南省立第一師範|湖南省立第一師範]]。[[毛泽东|毛泽东]]、[[蔡和森|蔡和森]]为其最看重的学生。[[鄧中夏|鄧中夏]]亦為其學生。其学生于1918年建立了[[新民学会|新民学会]]。

1918年，应[[蔡元培|蔡元培]]邀请任[[北京大学|北京大学]]教授，教授[[伦理学|伦理学]]，他为赴法勤工俭学学生筹措经费，又推荐[[毛泽东|毛泽东]]到北大图书馆工作，并且促成了女儿[[杨开慧|杨开慧]]与毛泽东的婚恋关系。<ref name="中国教育家杨昌济诞生 " >{{cite web |url=http://www.todayonhistory.com/4/21/china-JiaoYuJiaYangChangJiDanSheng.html|title=中国教育家杨昌济诞生|accessdate=|work=|publisher=历史上的今天|language=zh-hans}}</ref>

1919年[[五四运动|五四运动]]时，发表《告学生》表达对学生的热切期望。

1920年1月17日，杨昌济病逝于[[北京德国医院|北京德国医院]]，归葬长沙县板仓。<ref name="中国教育家杨昌济诞生 " /> 临终前曾致信好友[[章士钊|章士钊]]（时任广州军政府秘书长、[[南北议和|南北议和]]代表），推荐毛泽东和[[蔡和森|蔡和森]]，说：「吾郑重语君，二子海内人才，前程远大，君不言救国则已，救国必先重二子。」<ref>{{cite web |url= http://www.chinanews.com/cul/2011/07-05/3159225.shtml|title= 青年毛泽东在北大韬光养晦 职位低微别人不理睬|accessdate= 2011年7月5日|publisher= 中国新闻网|work= |language =zh-cn}}</ref>

== 思想 ==
=== 学术思想 ===
杨昌济强调无论是研究学问还是做人，都必须具有坚忍不拔的精神，他说：“吾无过人者，惟于坚忍二字颇为着力，常欲以久制胜。”其精神被[[湖湘|湖湘]]学子称为“达化斋法门”而竞相仿效。<ref name="20世纪初：大学教授杨昌济 " /><ref name="中国教育家杨昌济诞生 " />

杨昌济认为一个国家有一个国家的民族精神：“吾人求学海外，欲归国而致之用，不可不就吾国之情形深加研究，何者当因,，何者当革，何者宜取，何者宜舍，了然于心，确有把握而后可以适合本国之国情，而善应宇宙之大势”。 <ref name="20世纪初：大学教授杨昌济 " />

=== 政治思想 ===
杨昌济一生主张改革，早年赞同康梁维新变法，经常和密友[[杨守仁|杨守仁]]讨论国事，提出“非改革不足以图存”。<ref name="20世纪初：大学教授杨昌济 " />

=== 教育思想 ===
杨昌济的教育思想承接[[明朝|明朝]]思想家[[王阳明|王阳明]]的“[[知行合一|知行合一]]”而来，又接收了西方德国[[泡尔生|泡尔生]]等教育家现代教育思想，提出了：知行统一，注重实践，反对空谈；以道德教育为中心，提出“立志、理想、爱国、殉国、勤勉、存诚、立功、勇敢、坚忍、贵我通今”等10方面内容；教育普及应与提高相结合；大力发展高等教育，反对赶时髦的留学之风；注重发现、爱护和培养人才，此乃“悠悠万事，惟此为大”；洋为中用，反对全盘西化。 <ref name="20世纪初：大学教授杨昌济 " />

== 评价 ==
* 1920年1月22日，蔡元培、范源濂、杨度、章士钊、黎锦熙、毛泽东等二十九人，在《北京大学日刊》就杨昌济逝世发表《启事》，赞扬杨昌济“操行纯洁，笃志嗜学”，“雍容讲坛，寒暑相继，勤恳不倦，学生景从”。
* 青年毛泽东写道：“杨先生之涵宏盛大，以为不可及”<ref>{{Cite book|title=毛泽东早期文稿|last=|first=|publisher=湖南人民出版社|year=2008|isbn=|location=长沙|pages=12-13}}</ref>。1920年６月，毛泽东致[[黎锦熙|黎锦熙]]信说：“先生（黎锦熙）及死去了的怀中先生，都是弘通广大，最所佩服。”<ref>{{Cite book|author=毛泽东|chapter=致黎锦熙信（1920年6月7日）|title=毛泽东早期文稿|last=|first=|publisher=：湖南出版社|year=1990|isbn=|location=长沙|pages=478}}</ref>1936年毛泽东在[[延安|延安]]接见[[美国|美国]]记者[[埃德加·斯诺|埃德加·斯诺]]时说：“给我印象最深的教员是杨昌济，他是从[[英国|英国]]回来的留学生，后来我同他的生活有密切的关系。他教授[[伦理学|伦理学]]，是一个[[唯心主义|唯心主义]]者，一个道德高尚的人。他对自己的伦理学有强烈信仰，鼓励学生立志做有益于社会的正大光明的人。……杨在我青年时代对我有过深刻的影响，后来在北京是我真诚的朋友。”<ref>{{cite web |url=http://www.xiexingcun.com/maozedong/dsbj011.htm|title=给我即象最深的老师（读杨昌济《论语类钞》、《劝学篇》等）|accessdate=|work=|publisher=毛泽东读书笔记解析|language= zh-hans}}</ref>
* [[李肖聃|李肖聃]]：“怀中于三十以前既已博究先儒之书，十余年中又益求英日学者之说，固有得于时代之精神，而其心光湛然，力抗流俗，而浩然有以自得于己，则友朋皆莫及也。”<ref>{{Cite book|author=杨昌济|editor=王兴国|title=杨昌济文集|last=|first=|publisher=湖南教育出版社|year=1983|isbn=|location=长沙|pages=379-380}}</ref>
* [[曹典球|曹典球]]在《杨昌济先生传》评价杨昌济说：“先生经戊戌政变之后，潜居影储山麓，少履城市，绝意科举之业，惟常读《通鉴》及王船山《读通鉴论》诸书，以古证今，痛论时弊，储备经世之学，以诏来者。”<ref>{{Cite book|author=杨昌济|editor=王兴国|title=杨昌济文集|last=|first=|publisher=湖南教育出版社|year=1983|isbn=|location=长沙|pages=384}}</ref>“湖南之师范教育，前清末年虽有中、西、南三路师范学堂及优级师范学堂之设，大辂椎轮，不为无功。但至高师成立，始有西洋伦理学、教育学及哲学等课程，其规划皆先生所首创”。“先生自精研中国经史、性理诸学数十年之后，又继续在日、英二国苦学九年{{noteTag|1=实际是九年多，后又赴[[德国|德国]]考察数月，合计在国外为十年 }}之久，对于中西学术源流，政治风俗，了如指掌，加以本身之存养省察，事事物物，无不加以详密之分析，而后出之为语言，发之为文章，经师人师，备诸一身。以故来学之士，一受其薰陶，无不顿改旧时之宇宙观，如饮醇醪，受其影响。是以湖南之师范教育，至先生讲学高师时，乃与普通各科诸教育有所区别。虽先生所采之英、德哲学诸书以作教材，多唯心派家言，然在湖南教育史中实别开一新纪元，不可忽视!”<ref name="杨昌济：世居板仓 思想进步 ">{{cite web |url=http://topics.gmw.cn/2010-10/14/content_1311445.htm|title=《杨昌济：世居板仓 思想进步》|accessdate=2010-10-14|work=|publisher=光明网|language=zh-hans}}</ref>
