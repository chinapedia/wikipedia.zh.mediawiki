'''杨玉良'''（{{bd|1952年|}}），[[浙江省|浙江省]][[嘉兴市|嘉兴市]][[海盐县|海盐县]]人，中国化学家，复旦大学教授，中国科学院院士。曾任[[复旦大学|复旦大学]][[校长|校长]]<ref name="杨玉良 复旦">{{cite web |url = http://www.fudan.edu.cn/new_genview/now_yangyuliang.htm |title = 复旦概况 > 现任领导 杨玉良（校长） |format = htm |publisher = [[复旦大学|复旦大学]] |accessdate = 2010年7月20日 |language = zh-cn |deadurl = yes |archiveurl = https://web.archive.org/web/20101010125804/http://fudan.edu.cn/new_genview/now_yangyuliang.htm |archivedate = 2010年10月10日 }}</ref>。

1988年获得联邦德国Leibniz大奖。美国之音11月13日在一则有关中国学术打假的采访报道中,复旦大学校长榜上有名。复旦大学校长杨玉良被美国之音列名学术作假首位。

== 简历 ==
* 1952年，杨玉良出生于浙江省嘉兴市海盐县。
* 1977年，获得[[复旦大学化学系|复旦大学化学系]][[高分子化学|高分子化学]]与[[物理|物理]]专业[[学士|学士]][[学位|学位]]。毕业后留校担任[[讲师|讲师]]。
* 1984年，获得[[复旦大学材料科学系|复旦大学材料科学系]][[哲学博士|哲学博士]][[学位|学位]]。杨玉良是[[中华人民共和国|中华人民共和国]]首个高分子[[材料学|材料学]]博士。
* 1984年，获得[[中国化学会|中国化学会]]首届青年化学奖。
* 1986年，杨玉良赴[[联邦德国|联邦德国]][[留学|留学]]，在[[马克斯-普朗克学会|马克斯-普朗克学会]]高分子研究所从事研究。
* 1988年，杨玉良回国，在复旦大学任教并从事[[高分子材料|高分子材料]]方面的科研。
* 1993年，晋升为复旦大学[[教授|教授]]。
* 1999年，出任复旦大学副校长。
* 2003年，当选为[[中国科学院院士|中国科学院院士]]。
* 2006年，任[[国务院学位委员会|国务院学位委员会]]办公室主任、[[中华人民共和国教育部|教育部]]学位管理与研究生教育司司长<ref name="杨玉良 复旦" />。
* 2009年1月，出任复旦大学校长。
* 2014年10月24日，因年龄原因，杨玉良不再担任复旦大学校长职务。<ref>{{Cite web|url=http://www.moe.gov.cn/publicfiles/business/htmlfiles/moe/moe_1485/201410/177222.html|title=许宁生任复旦大学校长|accessdate=2014-10-24|work=|publisher=中华人民共和国教育部|date=2014-10-24|deadurl=yes|archiveurl=https://web.archive.org/web/20141024205003/http://www.moe.gov.cn/publicfiles/business/htmlfiles/moe/moe_1485/201410/177222.html|archivedate=2014-10-24}}</ref>
