'''杨美真'''（{{bd|1904年||1991年|4月12日}}）女，[[湖北省|湖北省]][[孝感县|孝感县]]城东南郊杨家村人，[[中华民国|中华民国]]及[[中华人民共和国|中华人民共和国]]政治人物，[[中国民主建国会|中国民主建国会]]创始人之一<ref name=xiaogan>{{Cite web|url = http://www.xiaogan.gov.cn/art/2015/4/8/art_1246_80558.html|title = 杨美真（1904—1991 孝南）|author = |date = 2015-04-08|publisher = 孝感名人库|language = |accessdate = |quote = |archive-url = https://web.archive.org/web/20180213173500/http://www.xiaogan.gov.cn/art/2015/4/8/art_1246_80558.html|archive-date = 2018-02-13|dead-url = yes}}</ref><ref name=hbzx>{{Cite web |url =http://www.hbzx.gov.cn/bencandy.php?fid=49&id=6097  |title =中国民主建国会创始人之一杨美真女士  |author = 鲁鋆 |date = 2014-09-15  |publisher = 中国人民政治协商会议湖北省委员会 |language =  |accessdate =  |quote =  }}</ref>

==生平==
1930年，杨美真自[[上海|上海]][[沪江大学|沪江大学]]毕业。同年到[[中华妇女节制会|中华妇女节制会]]工作，任宣传处处长、副总干事。1934年8月，杨美真作为中国妇女代表，到[[英国|英国]][[伦敦|伦敦]]出席世界妇女节制大会。会后，从英国赴[[美国|美国]]留学。1936年获得美国[[宾夕法尼亚大学|宾夕法尼亚大学]][[社会学|社会学]]硕士学位。1936年9月回中国，历任[[北平|北平]]女青年会总干事、上海新生女子职业中学校长、[[成都|成都]]妇女赈济工业社社长、成都华西金陵男子大学副教授、[[贵阳|贵阳]]女青年会总干事、[[重庆|重庆]]儿童营养会营养研究组主任、上海妇女节制会总干事<ref name=xiaogan/><ref name=hbzx/>。

[[抗日战争|抗日战争]]时期，杨美真参加抗日救亡运动。1945年8月抗日战争胜利后，民主人士[[黄炎培|黄炎培]]、[[胡厥文|胡厥文]]、[[杨卫东|杨卫东]]等人与[[章乃器|章乃器]]发起组织[[中国民主建国会|民主建国会]]。杨美真是核心小组成员之一。1945年12月15日，民主建国会成立大会在重庆举行，杨美真被推选为监事。1946年，民主建国会迁到上海。1947年，民主建国会被[[中国国民党|中国国民党]]当局宣布为非法组织，被迫转入地下活动。不久，杨美真与丈夫章乃器等人到[[香港|香港]]，继续反对[[蒋介石|蒋介石]]的独裁统治，争取和平与民主<ref name=xiaogan/><ref name=hbzx/>。

1948年8月，[[中共中央|中共中央]]发表[[五一口号|“五一”劳动节口号]]。杨美真及民主建国会负责人黄炎培等人很快响应。1949年春，杨美真等人陆续抵达北平。1949年4月15日，[[毛泽东|毛泽东]]在[[香山双清别墅|香山双清别墅]]举办宴会，邀黄炎培、[[施复亮|施复亮]]、[[孙超猛|孙超猛]]、章乃器及夫人杨美真等十一位民主建国会负责人共商国是。这是杨美真首次见到毛泽东。同年5月26日晚，[[周恩来|周恩来]]举行宴会，邀请杨美真等十二人共同商谈民主建国会的前途及任务<ref name=xiaogan/><ref name=hbzx/>。

1949年5月，杨美真参加全国青年代表大会。1951年，杨美真参加[[中央人民政府卫生部|中央人民政府卫生部]]部长[[李德全|李德全]]率领的代表团，到[[朝鲜半岛|朝鲜]]调查[[美军|美军]][[细菌战|细菌战]]罪行。1953年，参加[[贺龙|贺龙]]率领的中国人民第三届赴朝鲜慰问团，1955年，被[[全国妇联|全国妇联]]派到[[民主德国|民主德国]]参加世界妇女节制会会议。1953年、1957年先后参加第二次、第三次全国妇女代表大会<ref name=xiaogan/><ref name=hbzx/>。

1957年，杨美真的丈夫章乃器在[[反右运动|反右运动]]中遭到严厉批判。1957年7月17日《人民日报》发表了章乃器的前妻[[胡子婴|胡子婴]]写的揭发文章《我所了解的章乃器》，这给杨美真极大震撼。中国民主建国会派人来医院，要杨美真离开医院，回中国民主建国会机关揭发章乃器。杨美真没有答应，并说：“我从未听到过他对党有什么不满的话。”杨美真因此遭批判，因“她跟丈夫章乃器站在同一反动立场上”。杨美真又被动员与章乃器离婚，但她仍没有答应。由于受到政治压力很大，1957年10月，杨美真带着孩子搬离章乃器居住的北京[[灯草胡同|灯草胡同]]三十号[[四合院|四合院]]，与章乃器分居。但她仍经常回灯草胡同三十号四合院探望章乃器。当时杨美真带着孩子住在北京[[张自忠路|张自忠路]]附近的[[汪芝麻胡同|汪芝麻胡同]]的一个大院，大院里还住着经济学家[[吴大琨|吴大琨]]（1930年代曾任章乃器秘书）。因没有与“大右派”章乃器划清界限，杨美真被打成“右派分子”，调到全国政协图书馆当管理员。后来，[[聂绀弩|聂绀弩]]的夫人[[周颖|周颖]]时常陪杨美真回灯草胡同三十号，想让杨美真与章乃器夫妇重归于好，杨美真也曾给章乃器写信表达重归于好的意思，但章乃器一直没回心转意。后来章乃器在跳舞时结识了中学音乐教员王者香，并于1964年结婚<ref>叶永烈，“我不会歪曲历史!”——访章乃器夫人杨美真(上)，名人传记月刊2014(6):4-8</ref><ref>叶永烈，“我不会歪曲历史!”——访章乃器夫人杨美真（下），名人传记：上半月2014(7):4-8</ref>。

杨美真曾任[[中国民主建国会|中国民主建国会]]中央秘书处副处长，中国民主建国会会务推进委员会常委、总会委员会常委，第一、二、三、四届中央委员，第一、三、四届中央常委、中央咨议委员会常委。1988年离职休养<ref name=xiaogan/><ref name=hbzx/>。

1991年4月12日，杨美真在北京病逝，享年87岁<ref name=xiaogan/><ref name=hbzx/>。
