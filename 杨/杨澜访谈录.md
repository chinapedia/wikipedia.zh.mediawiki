《'''杨澜访谈录'''》，2001年由[[上海市|上海市]][[东方卫视|东方卫视]]建立的一个节目，由著名主持人[[杨澜|杨澜]]主持，节目以采访谈话的方式表现在观众面前，节目中访谈的对象主要是全球政界、商界、文艺界、体育界知名人士。<ref>{{cite web |url= http://news.qq.com/a/20091226/000913.htm|title= “杨澜访谈”实录2009|accessdate= 2009年|work= |publisher=腾讯网 |language =zh-cn}}</ref><ref name="jfs " >{{cite web |url=http://www.people.com.cn/GB/shenghuo/78/115/20020905/815979.html |title=深度访谈　杨澜：我不是中国的华莱士 |accessdate= 2002年9月5日|work= |publisher=人民网 |language =zh-cn}}</ref>

2013年移入[[北京卫视|北京卫视]]每周日晚23:00播出。

== 概述 ==
《杨澜访谈录》以采访和访谈全球政界、商界、文艺界、体育界知名人士而著名，节目中通过主持人[[杨澜|杨澜]]和受访者的交流，展现受访者人物性格和独到见解，具有一定的深度和广度，同时节目中受访者的精彩对答表现了人类智慧的光芒。<ref>{{cite web|url= http://data.yule.sohu.com/lanmu/tvcolumn/43/|title= 东方卫视《杨澜访谈录》|accessdate= 2001年|work= |publisher= 搜狐网|language= zh-cn}}{{Dead link|date=2018年10月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

== 风格 ==
《[[人民网|人民网]]》点评杨澜在节目中的风格是“舒缓、平和、不带丝毫的张扬之气”。<ref name="jfs " />

杨澜曾用一个成语形容自己在该节目的主持风格是“绵裡藏针”，她还说：“风格是你在具备一定内涵后才体现出来的东西”、“到底是开着轰炸机把这块地都炸平，还是钻井取油？后来我们认为，深刻的问题也可以用一种温和的形式提出来”、“观众要看的是内容，而不是两个人吵架”。<ref name="jfs " />

== 评价 ==
《杨澜访谈录》的制片人[[郝亚兰|郝亚兰]]说：“这个节目，实际上是对杨澜和被采访对象智慧的双重倚重，是杨澜和被采访对象智慧的碰撞交融。”<ref name="jfs " />

《[[粉丝网|粉丝网]]》专文认为：“《杨澜访谈录》是集国际化特色、信息、深度于一体，传递成功人生的进取魅力。”同时指出，“该片在过去十年使国人更好地了解了世界，而未来十年世界将更换地了解中国。”<ref>{{cite web|url=http://bagua.ifensi.com/article-326295.html|title=《杨澜访谈录》十周年 进取着的访谈|accessdate=2010年|work=|publisher=粉丝网|language=zh-cn|deadurl=yes|archiveurl=https://web.archive.org/web/20160319021457/http://bagua.ifensi.com/article-326295.html|archivedate=2016年3月19日|df=}}</ref>
