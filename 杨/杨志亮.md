'''杨志亮'''（{{bd|1962年|11月||}}）[[河南省|河南省]][[武陟县|武陟县]]人，[[中国人民解放军海军|中国人民解放军海军]]少将<ref name=jzrb>{{Cite web|url = http://epaper.jzrb.com/html/2013-07/16/content_213837.htm|title = 碧海丹心写军魂|author = |date = 2013-07-16|publisher = 焦作日报|language = |accessdate = |quote = |archive-url = https://web.archive.org/web/20170107100501/http://epaper.jzrb.com/html/2013-07/16/content_213837.htm|archive-date = 2017-01-07|dead-url = yes}}</ref><ref name=ifeng>{{Cite web |url =http://finance.ifeng.com/a/20170121/15160602_0.shtml  |title = 两名将军缺席海军晋衔仪式，他们为何没到现场？ |author =  |date = 2017-01-21 |publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref>。

==生平==
杨志亮出生在武陟县[[谢旗营镇|谢旗营镇]]小寨村一个普通农民家庭。1981年入伍。1983年考进[[中国人民解放军海军大连舰艇学院|中国人民解放军海军大连舰艇学院]]火炮系。1988年3月，从海军大连舰艇学院火炮系毕业不久，正在[[中国人民解放军海军南海舰队|中国人民解放军海军南海舰队]]502护卫舰上任副枪炮长的杨志亮，参加了[[赤瓜礁海战|赤瓜礁海战]]。1988年3月12日晚，正在[[南沙群岛|南沙群岛]]海域巡逻的502舰接到上级命令，赶往[[赤瓜礁|赤瓜礁]]执行勘察任务。3月13日中午，502舰抵达赤瓜礁海区开展勘察工作。正在勘察作业时，502舰雷达发现[[越南海军|越南海军]]5艘舰船编队驶向赤瓜礁海域。502舰立即起锚拦截越南军舰，并通过高音喇叭向越军喊话，指明这里是中国领海，要求越军立即离开。3月14日黎明，杨志亮带队的第二登礁小组开始登礁。3月14日8时许，僵持继续升级。两军推搡之间，一名越军士兵举枪指向中方一名战士，杨志亮右手将越军士兵从后面拦腰抱住，左手抓住其枪管。子弹射穿了杨志亮的左臂。战斗由此打响。一名战友将杨志亮拖到赤瓜礁礁盘边缘，抓起一根麻绳帮杨志亮包扎止血。随后杨志亮失去知觉。杨志亮醒来后得知，中方获胜，击沉、重创越军舰船3艘，俘虏越军9人，中方仅2人负伤。战后，杨志亮被送回大陆接受治疗。经诊断，杨志亮左侧小臂粉碎性骨折，后经治疗，手臂成功保住并恢复运动功能。杨志亮康复出院后，受到[[中国人民解放军总参谋长|中国人民解放军总参谋长]][[迟浩田|迟浩田]]上将接见。杨志亮因在赤瓜礁海战中的英勇表现而立一等功<ref name=jzrb/><ref name=gairen/>。

此后，杨志亮先后在水面舰艇、海军机关、潜艇、后勤保障部队工作，历任舰艇枪炮长、舰副政委、舰政委、支队副政委等职。2003年，已成为师职干部的杨志亮曾被派到海拔4000米的[[西藏|西藏]][[日喀则|日喀则]]带职锻炼，担任日喀则军分区副政委一年，其间走遍下辖18个县市的边防哨卡，并两次走进海拔5300米的中国人民解放军全军最高边防哨所——查果拉<ref name=jzrb/>。后来，杨志亮出任[[中国人民解放军海军青岛综合保障基地|海军青岛保障基地]]政委<ref name=jzrb/><ref>{{Cite web |url = http://mil.news.sina.com.cn/2010-08-01/0641603633.html |title =海军青岛保障基地党委 抓学习制度促学习深入  |author =  |date = 2010-08-01 |publisher =  新浪|language =  |accessdate =  |quote =  }}</ref>。不久，杨志亮任[[中国人民解放军海军北海舰队|中国人民解放军海军北海舰队]]航空兵副政委，海军大校军衔<ref name=jzrb/>。

2015年，杨志亮调任[[中国人民解放军海军南海舰队|中国人民解放军海军南海舰队]]政治部副主任，成为副军职干部<ref name=gairen>{{Cite web |url = http://news.163.com/16/0325/16/BJ12JOO90001124J.html |title = 杨志亮改任南海舰队政治工作部副主任 |author =  |date = 2016-03-25 |publisher =  网易|language =  |accessdate =  |quote =  }}</ref>。2016年[[深化国防和军队改革|深化国防和军队改革]]中，改任[[中国人民解放军南部战区海军|中国人民解放军南部战区海军]]政治工作部副主任<ref>{{Cite web |url =  http://china.caixin.com/2017-01-20/101047444.html|title = 明星舰长柏耀平、南沙卫士杨志亮晋升少将 |author =  |date = 2017-01-20 |publisher =财新网  |language =  |accessdate =  |quote =  }}</ref>。2016年1月底至2月初，杨志亮随海军慰问组乘军舰到[[南沙群岛|南沙群岛]]慰问，这是杨志亮在赤瓜礁海战后第一次登上南沙岛礁，第一次重登赤瓜礁<ref>{{Cite web |url =http://military.china.com/important/11132797/20160206/21471014_all.html  |title =  中越海战功臣杨志亮28年后重登赤瓜礁|author =  |date = 2016-02-06 |publisher =中华网  |language =  |accessdate =  |quote =  }}</ref>。

2017年1月20日，海军在北京举行晋衔仪式，共7名军官晋升为海军少将，仪式由海军司令员[[沈金龙|沈金龙]]主持。7名获晋升的军官中，杨志亮、[[柏耀平|柏耀平]]缺席。柏耀平是自2016年8月起执行[[亚丁湾|亚丁湾]]、[[索马里|索马里]]海域护航任务的112编队指挥员，此时已完成任务，正在执行对外友好访问任务；杨志亮是2017年1月2日起接替112编队执行护航任务的568编队指挥员，1月2日112编队与568编队在亚丁湾交接任务。晋衔仪式举行时，两人都不在中国，因此缺席<ref name=ifeng/>。

2018年1月1日从权威渠道获悉，原任南海舰队政治工作部副主任的杨志亮海军少将已经履新[[中国人民解放军海军研究院|中国人民解放军海军研究院]]政委。2018年6月，杨志亮离开海军研究院，海军研究院政委由[[赖如鑫|赖如鑫]]接任<ref>[https://www.thepaper.cn/newsDetail_forward_2213940 赖如鑫履新海军研究院政委，南沙海战功臣杨志亮离任]</ref>。
