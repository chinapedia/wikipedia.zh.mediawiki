'''己内酰胺'''（Caprolactam，簡稱CPL），化学式为(CH<sub>2</sub>)<sub>5</sub>C(O)NH的[[有机化合物|有机化合物]]，是6-{{le|氨基己酸|Aminocaproic acid}}（ε-氨基己酸）的[[内酰胺|内酰胺]]，也可看作[[己酸|己酸]]的环状[[酰胺|酰胺]]。纯净的己内酰胺是白色的固体。其年产量大约45万吨。

己内酰胺主要用作制取[[聚合物|聚合物]][[尼龙6|尼龙6]]的[[单体|单体]]。<ref name=Ullmann>{{Ullmann | author1 = Josef Ritz | author2 = Hugo Fuchs | author3 = Heinz Kieczka | author4 = William C. Moran | title = Caprolactam | doi = 10.1002/14356007.a05_031.pub2}}</ref>

== 生产 ==
己内酰胺最早于19世纪后期通过ε-氨基己酸（己内酰胺的水解产物）的成环反应来制备。由于尼龙-6含有较大的商业价值，现在已有许多可用于生产己内酰胺的方法。

现在己内酰胺可由[[环己酮|环己酮]]与[[硫酸羟胺|硫酸羟胺]]发生成[[肟|肟]]反应生成{{le|环己酮肟|Cyclohexanone oxime}}，再经酸处理发生[[贝克曼重排|贝克曼重排]]反应制得：
: [[File:Beckmann_rearrangement.png|420px]]<br style="clear:left;"/>
重排的直接产物是己内酰胺的硫酸氢盐，一般需要用[[氨|氨]]中和以得到游离的己内酰胺，同时生成[[硫酸铵|硫酸铵]]。

另一个主要的工业制备方法是通过[[亚硝酰氯|亚硝酰氯]]使[[环己烷|环己烷]]形成肟。该方法的优点是环己烷比环己酮价格更低廉。在较早的时代，也有通过用氨处理己内酯制备己内酰胺的方法。<ref name=Ullmann/>

环己酮在[[叠氮酸|叠氮酸]]在有[[硫酸|硫酸]]或[[路易斯酸|路易斯酸]]作催化剂的情况下反应的制备方法也曾得到报道。<ref>{{Cite web |url=http://www.chem.uky.edu/courses/che535/AC/Expansion.7-membered.rings.Tetrahedron.2000.4317.pdf |title=Expansion to Seven-Membered Rings |accessdate=2017-07-24 |author=Eric J. == 使用 ==
几乎生产的所有己内酰胺都用于生产[[尼龙6|尼龙6]]。该反应是一个开环[[聚合|聚合]]反应：
:''n'' (CH<sub>2</sub>)<sub>5</sub>C(O)NH  → [(CH<sub>2</sub>)<sub>5</sub>C(O)NH]<sub>''n''</sub>

生产得到的尼龙-6广泛使用于[[纤维|纤维]]和[[塑料|塑料]]制品。

== 安全 ==
己内酰胺具有刺激性，摄取、吸入或皮肤接触有毒，{{LD50}}为1.1 g/kg（大鼠，口服）。1991年，己内酰胺被列入1990年“{{le|美国清洁空气法案|Clean Air Act (United States)}}”的有害空气污染物清单，但于1996年被移除出名单。<ref>[http://www.epa.gov/ttn/atw/pollutants/atwsmod.html EPA - Modifications To The 112(b)1 Hazardous Air Pollutants]</ref>

在[[国际癌症研究机构|国际癌症研究机构]]对[[致癌物质|致癌物质]]的分类中，己内酰胺被归为第4类致癌物“对人体基本无致癌作用”分类的唯一物质。<ref>{{cite web | url=https://monographs.iarc.fr/ENG/Monographs/vol71/mono71-16.pdf | title=Agents Classified by the ''IARC Monographs'', Volume 71, page 395| accessdate=2016-10-21| publisher=[[International_Agency_for_Research_on_Cancer|IARC]]| date=2016-02-22}}</ref>

目前，美国对于处理己内酰胺的工人暂未正式规定允许的接触限值。对于己内酰胺[[粉尘|粉尘]]和[[蒸气|蒸气]]，建议的长时间接触限值为1 mg/m<sup>3</sup>，短期接触限值为3 mg/m<sup>3</sup>。<ref>{{Citation|accessdate=2013-08-11|title = NIOSH Pocket Guide to Chemical Hazards|publisher = CDC | url =http://www.cdc.gov/niosh/npg/npgd0097.html}}</ref>
