'''鼯鼠'''也稱'''飛鼠'''、'''鸓'''，是[[松鼠科|松鼠科]]下的一個[[族_(生物)|族]]，稱為'''鼯鼠族'''（'''Pteromyini'''）。其飞膜可以帮助其在树中间快速的滑行，但由于無法產生足夠的升力，因此鼯鼠只能在树、陆中间滑翔。鼯鼠可以[[滑翔|滑翔]]但不能真正的[[飞行|飞行]]。

目前鼯鼠[[分布|分布]]于[[亚洲|亚洲]]、[[欧洲|欧洲]]和[[美洲|美洲]]的[[热带雨林|热带]]与[[温带雨林|温带雨林]]中，[[人类|人类]]已发现多达43个不同种类的鼯鼠。由于近年来的大量[[猎杀|猎杀]]，鼯鼠很有灭绝的可能。

飛鼠是夜行性動物，獵人要捕獵時直接用探照燈等燈光來照，他就會原地不動。

== 历史 ==
[[File:Flying_squirrel_in_a_tree.jpg|left]]
最早记载鼯鼠并且给予定义的是公元前三世纪中国的[[词典|词典]]《[[尔雅|尔雅]]》。但由于科技等多种原因的不发达，对鼯鼠的定义比较含糊，并且由于鼯鼠可以滑行将其定义为[[鸟类|鸟类]]的一种。后随着时间的推移越来越多人开始研究鼯鼠这一动物，並細分出多個[[屬|屬]]。鼯鼠在古代人眼中有着多种本领，确也有多种不足:《[[荀子|荀子]]‧勸學篇》說：“鼯鼠五技而窮：能飞不能上屋，能缘不能穷木，能游不能渡谷，能穴不能掩身，能走不能先人”。<ref>唐‧[[楊倞|楊倞]]注《荀子‧勸學篇》：「梧鼠當為鼫鼠，蓋本誤為鼯字，傳寫又誤為梧耳。技，才能也。言技能雖多而不能如螣蛇專一，故窮。五技，謂能飛不能上屋。能緣不能窮木。能浮不能渡谷。能穴不能掩身。能走不能先人。」</ref>。明代旅行家[[馬歡|馬歡]]著的《[[瀛涯勝覽|瀛涯勝覽]]》內的啞魯國提到東南亞種鼯鼠：「山林中出一等飛虎，如貓大，遍身毛灰色。有肉翅，如蝙蝠一般，但前足肉翅生連後足，能飛不遠。」

== 型態與生活方式 ==
鼯鼠為松鼠科，后肢之间有皮肤薄膜相连，可以做[[松鼠|松鼠]]般的长距离跳跃造。幼年'''鼯鼠'''一般体体长13厘米以上，成年'''鼯鼠'''则长50厘米以上。鼯鼠为[[夜行动物|夜行动物]]，居住于[[树|树]]中，以[[坚果|坚果]]、[[水果|水果]]、[[植物|植物]]嫩芽、[[昆虫|昆虫]]和小型[[鸟类|鸟类]]为食。鼯鼠一般长有二十二颗[[牙齿|牙齿]]。普通鼯鼠可以生存12年之久。

=== 「飛行」 ===
[[File:Gliding_flying_squirrel.jpg|thumb]]
由于鼯鼠可以[[滑翔|滑翔]]，因此得名飞鼠。飞鼠还可以进行类似松鼠的长距离的跳跃。鼯鼠的滑行得利于其飞膜以及尾巴，其飞膜可以撑开以起到翅膀作為
滑翔，而其尾巴可以控制方向。一只普通鼯鼠可以滑翔47[[米|米]]左右，有人曾经录得一飞膜滑行150米之远。由于鼯鼠的独特飞行生活，经常用于研究之用。

== 保育狀況 ==
几年来，因为多种原因，鼯鼠遭到了严重的捕猎。目前，部分鼯鼠于[[中国大陆|中国大陆]]为[[保护动物|保护动物]]，如果恶意捕捉可以因此得到拘留十天以上。此外，[[猫|猫]]常常以其为食。
