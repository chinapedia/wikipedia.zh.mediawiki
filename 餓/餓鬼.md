鬼概分二類：一、住在地獄世界的，由[[閻魔羅王|閻魔羅王]]治理。二、散在人間的，多數在樹林山野中，故有时亦將樹林唤作「鬼村」<ref>《阿毘達磨藏顯宗論》：「諸鬼本住[[琰魔|琰魔]]王國，從此展轉散趣餘方。此贍部洲南邊直下，深過五百踰繕那量，有琰魔王都，縱廣量亦爾。」</ref><ref name="印順">{{cite book|author1=印順|title=華雨集第四冊－三 鬼與地獄|url=http://yinshun-edu.org.tw/en/Master_yinshun/y28_03_03}}</ref>，其中亦有不少分类，如：停留在前世屍体周围不愿离去的，称爲“守尸鬼”；慳守在前世累積的財富周圍不願離去的，稱爲“錢櫃鬼”、“守財鬼”。阳间野鬼生活在[[人類|人類]]周圍，但通常大部分人無法見到。

[[因果论_(佛教)|因果论]]认为是前世有大贪欲、吝嗇、恶多善少者，会堕落於此道受恶报，'''饿鬼道'''（或稱之為鬼神道、鬼道）是[[六道|六道]][[轮回|轮回]]中的[[惡趣|惡趣]]（恶趋、恶道），代表著[[三不善根|三不善根]]中的[[貪_(佛教)|貪欲]]。此道絕大多數的鬼生活飢饉貧乏，無人施食或無法接受饮食，食物到嘴边会化成火炭，走在河边亦见不到、饮不到河水，故一直处飢渴的状态，貪婪多欲，永遠無法滿足，所以籠統稱之為餓鬼。雖然往往翻譯為「餓鬼」，但其實薜茘多不一定飢餓，其中也有富裕奢華甚至有大[[神通|神通]]者，其力堪比於[[天人|天人]]，如[[夜叉|夜叉]]（勇健鬼）、[[羅剎|羅剎]]（兇惡鬼），被佛教認為是具有大能力的鬼王。有時這些鬼也居住在[[地居天|地居天界]]，如[[四大天王|四大天王]]部眾，故餓鬼道之眾生和民間俗稱的[[鬼|鬼]]又非完全重合。<ref name="顯宗論">《阿毘達磨藏顯宗論》：「鬼有三種：謂無、少、多財。無財，復三：謂炬、針、臭口。少財，亦有三：謂針、臭毛、癭。多財亦有三：謂希祠、棄、大勢，廣釋此九如[[順正理論|順正理]]然諸鬼中，無威德者，唯三洲有，除北俱盧。若有威德，天上亦有。贍部洲西渚有五百，於中有二，唯鬼所居。渚各有城，二百五十。有威德鬼住一渚城，一渚城居無威德鬼。諸鬼多分，行竪而行。於劫初時皆同聖語，後隨處別種種乖訛。」<br/>《[[大毗婆沙論|大毗婆沙論]]》：「四大王眾天及三十三天中，雖有大威德鬼，與諸天眾守門防邏導從給使。……時王見彼有威德鬼。首冠花鬘身著天衣。食甘美食猶如天子。乘象馬車各各遊戲。見無威德鬼。頭髮蓬亂裸形無衣。顏色枯悴以髮自覆。執持瓦器而行乞匃。」</ref>。

==概述==
[[佛經|佛經]]中，關於鬼（薜茘多）的種類，諸經論所說互有不同<ref name=tx>{{cite|url=http://foxue.qq.com/a/20151222/023953.htm|title=不同种类的鬼如何获得最大福报}}</ref>。

如《順正理論》說鬼（薜茘多）可分成三大類：無財、少財、與多財。<ref>阿毘達磨順正理論卷第31</ref><ref name="顯宗論"/><ref>馮達庵《佛法要论》</ref>

#無財鬼又分成：
##炬口（又稱焰口）鬼：飲食入口就化為火焰，这是由极端的吝啬所受到的苦果。
##針口鬼：肚子大，咽喉卻細小如針，無法吞嚥食物。
##臭口鬼：嘴巴腐爛發臭，自熏嘔吐，不能飲食。
#少財鬼分成：
##針毛鬼：體毛坚利似針，無法靠近，且針毛刺穿其身體，十分痛苦。
##臭毛鬼：體毛惡臭，自熏嘔吐，不能飲食。
##癭鬼：咽喉生有癭瘤，致不能飲食，癭瘤還會破膿，以膿水爲食方能暫解飢餓。
#多財鬼則分成：
##希祠鬼：以人類祭祀物（[[食物|食物]]、[[冥鏹|冥鏹紙錢]]等）為食。這類鬼前世爲人時性吝嗇，不惜為非作歹囤聚財富，故感召餓鬼果報，但有飛天神通，能享用祠堂、祭祀的物品。
##希弃鬼：以人類遺洩物、嘔吐物為食。
##大勢鬼：指[[夜叉|夜叉]]、[[羅剎|羅剎]]等大力鬼王，他們的[[神通|神通]]與[[福報|福報]]近於[[天人|天人]]，他們也能降福施災，甚至研習[[佛法|佛法]]、護持修習佛法者。

《大智度論》則說鬼有二種：弊鬼、餓鬼。
#弊鬼者，如天受樂，但與餓鬼同住，為餓鬼之主；
#餓鬼者，腹如山谷，咽如針頭，唯有黑皮、筋、骨三事，無數百歲不聞飲食之名。<ref>《大智度論》卷30</ref>

《正法念處經·餓鬼品》列有36種鬼。

不過，除了這些少數有福鬼眾之外，大多數饿鬼道的众生非常痛苦，其痛苦是我们所无法想像的。有的餓鬼小孩的頭很大，長滿[[膿|膿]][[瘡|瘡]]、腹部鼓漲；经常很饿，连一点吃的也得不到，一滴水也得不到。之所以会投生在饿鬼道是因为[[貪|貪]][[业|业]]的缘故。饿鬼道的众生是[[胎生|胎生]]的，而且每胎便会生下几百个小鬼婴，自己都吃不飽，還要供小孩吃，更加苦楚。

有一種觀點，會把在民間廣受[[香火|香火]]的地方[[神祇|神祇]]，視之為鬼神道中有威德的鬼王，也有視為[[天人_(佛教)|天趣]]者<ref>{{cite web|title=餓鬼|url=https://www.buddhistdoor.org/tc/dictionary/details/%E9%A4%93%E9%AC%BC}}</ref><ref>《[[大毗婆沙論|大毗婆沙論]]》：「問：亦有人所承事以為天者，如筏栗達那神、旃稚迦神、旃荼履迦神、布刺拏跋達羅神（Pūrṇabhadra，富那跋陀）、摩尼跋達羅神（{{tsl|en|Manibhadra|Manibhadra}}）、訶利底神（[[鬼子母|鬼子母]]）、末度塞建陀神（[[塞建陀|塞建陀]]或[[韋馱|韋馱]]）等，為是天趣為鬼趣耶？若是天趣，何故奪人精氣，亦斷人命，受人祠祀？以諸天觀人飲食猶如糞穢，不應貪餮以為香美故。……答：<u>有說，彼是天趣</u>。問：若爾何故奪人精氣，亦斷人命受人祠祀耶？答：彼無是事。但彼所領鬼神有奪人精氣等事。<u>如是說者，彼是鬼趣</u>。……」
</ref>。[[印度神話|印度神話]]中的[[阿修羅_(印度神話)|阿修羅]]，在佛教中，也有被視為鬼趣或[[天人_(佛教)|天趣]]所攝，或別立[[阿修羅_(佛教)|阿修羅趣]]的不定說<ref>《[[大毘婆沙論|大毘婆沙論]]》：「於彼中有阿素洛今當說。謂有餘部立阿素洛為第六趣。彼不應作是說。契經唯說有五趣故。問何故名阿素洛。答素洛是天。彼非天故名阿素洛。……復次素洛名同類。彼先與天相近而住。然類不同故名阿素洛。……問諸阿素洛何趣所攝。答<u>有說是天趣攝</u>。問若爾何故不能入正性離生。答諂曲所覆故。……以為如是諂曲心所覆故。不能入正性離生。復次不可以不能入正性離生故便謂惡趣所攝。如諸達絮及篾戾車人亦不能入正性離生。而非惡趣所攝彼亦應爾。<u>如是說者是鬼趣攝</u>。問若爾何故經說。帝釋語毘摩質怛羅（Vimalacitra）阿素洛王言。汝本是此處天。答帝釋應言汝本是此處鬼。而言天者以尊敬婦公故作此愛語。又令設芝聞生歡喜。問若是鬼者何以與諸天交親耶。答諸天貪美色故不為族姓。如設芝阿素洛女端政無比。是故帝釋納以為妻。亦如大樹緊捺洛王女名為奪意端政無雙。雖是傍生趣攝。而蘇迷王太子蘇達那菩薩求娉為妻此亦如是。問何以復能與天鬪戰。答亦有劣者與勝者共諍。如奴與主鬪狗與人鬪。」<br/>
《[[順正理論|順正理論]]》：「阿素洛者。與諸天眾。違諍交通。言本是天。威德殊勝。由斯等故。天趣所收。諦現觀中。無堪能故。似非人故。多諂曲故。定非天趣。是鬼趣攝。……故阿素洛。唯鬼趣收。」<br/>
《[[大智度論|大智度論]]》：「問曰：阿修羅即為五道所攝。是阿修羅非天非人，地獄苦多畜生形異，如是應鬼道所攝。答曰：不然。阿修羅力與三十三天等。何以故？或為諸天所破，或時能破諸天。如經中說，釋提桓因為阿修羅所破。四種兵眾入藕根孔以自藏翳，受五欲樂與天相似，為佛弟子。如是威力，何得餓鬼所攝？以是故應有六道。復次如阿修羅、[[緊那羅|甄陀羅]]、[[乾沓婆|乾沓婆]]、[[鳩槃荼|鳩槃茶]]、夜叉、羅剎、{{tsl|en|Budha|浮陀}}等大神，是天、阿修羅民眾，受樂小減諸天，威德變化隨意所作。是故人疑言，是修羅非修羅？修羅秦言大也。說者言：是阿修羅非修羅。阿修羅道初得名，餘者皆同一道。問曰：經說有五道，-{云}-何言六道？答曰：佛去久經流遠，法傳五百年後，多有別異，部部不同，或言五道，或言六道。若說五者，於佛經迴文說五。若說六者，於佛經迴文說六。又摩訶衍中，法華經說有六趣眾生。觀諸義旨，應有六道。復次分別善惡，故有六道。善有上中下故，有三善道天人阿修羅。惡有上中下故，地獄畜生餓鬼道。若不爾者惡有三果報，而善有二果，是事相違，若有六道於義無違。問曰：若龍王金翅鳥，力勢雖大猶為畜生道攝。阿修羅亦應餓鬼道攝。何以更作六道？答曰：是龍王金翅鳥雖復受樂，傍行形同畜生故畜生道攝。地獄餓鬼形雖似人，以其大苦故不入人道。阿修羅力勢既大，形似人天故，別立六道是為略說。」</ref>。

===八部鬼神===
{{main|八部众}}
[[四天王|四天王]]所領之八類鬼神，亦稱八部眾<ref>{{cite web|title=八部鬼眾|url=https://www.buddhistdoor.org/tc/dictionary/details/%E5%85%AB%E9%83%A8%E9%AC%BC%E7%9C%BE}}</ref>。即：

# [[乾闥婆|乾闥婆]]（梵 gandharva），譯為香陰，乃不食酒肉，唯以香資陰（陰即身之意），帝釋天樂神即是。
# [[畢舍遮|毘舍闍]]（梵 piśāca），譯為噉精氣、啖精氣鬼，乃食人之精氣及五穀之精氣。
# [[鳩槃荼|鳩槃荼]]（梵 kumbhānda），譯為甕形，以其陰似甕故，即厭魅鬼、冬瓜鬼。
# 薜荔多（梵 preta），譯為餓鬼。
# [[那伽|諸龍]]眾（梵 nāga），龍有四種，即：(1)守天宮殿，持令不落，天龍者一。(2)興雲致雨，利益人間，天龍者二。(3)地龍，決江開瀆。(4)伏藏龍，守轉輪王大福人寶藏。
# [[富單那|富單那]]（梵 pūtana），譯為臭餓鬼或熱病鬼，乃主熱之病鬼。
# [[夜叉|夜叉]]（梵 yakṣa），又稱藥叉，譯為勇健，有地行夜叉、虛空夜叉、天夜叉三種。
# [[羅剎|羅剎]]（梵 rākṣasa），譯為速疾鬼，又稱可畏，以其暴惡可畏故名。

此八類鬼神當中，有些說法認為有些為[[天界|天趣]]所攝，如[[夜叉|夜叉]]等；有些則說為[[畜生道|傍生趣]]所攝，如[[畢舍遮|毘舍闍]]、[[那伽|諸龍]]<ref>《大毗婆沙論》：「如緊捺落、畢舍遮、醯盧索迦、婆路尼折羅、頗勒窶羅等皆傍生趣攝。彼形雖上立而猶有傍生相。謂或有耳尖。或有戴角。或執險杖。或作諸鳥獸頭。或作傍生蹄具。是故皆是傍生趣攝。」<br/>
《順正理論》：「難陀跋難陀等。雖是傍生。然其威德自在。勝諸天眾。」</ref><ref>{{cite web|author1=林崇安|title=夜叉是天趣或鬼趣略探|url=http://enlight.lib.ntu.edu.tw/FULLTEXT/JR-AN/an541180.pdf}}</ref>。

===與「鬼」的不同===
餓鬼和通常所說的「[[鬼|鬼]]」並不相同。餓鬼和人一樣，為六道的一部份，擁有色身（[[物質|物質]]身）；和鬼接近的概念在佛教多是[[中陰身|中陰身]]，是[[有情眾生|有情眾生]]往生前的暫時狀態。

== 佛經故事 ==
===面燃鬼王傳說===
{{main|面燃鬼王}}
[[釋迦佛|釋迦佛]]的侍者[[阿難|阿難]][[羅漢|尊者]]在林間[[禪定|禪定]]時，忽見一位餓鬼鬼王，自稱「面燃」，說阿難三天之後將墮落餓鬼道。想避免就要[[布施|布施]]餓鬼，供養[[三寶|三寶]]。阿難於是請佛陀指示，佛祖授以《[[陀羅尼|陀羅尼]]施食法》，後阿難依法，設「面燃大士」的[[牌位|牌位]]，設齋供僧來祈福，因此獲得解脫。

「面燃」究竟為何邂逅阿難，[[佛教|佛教]]說法有數個：一是祂是[[觀音大士|觀音大士]]的[[化身|化身]]，故稱「[[大士爺|大士爺]]」。二是祂原為諸鬼的[[首領|首領]]，因受觀音大士教化而[[皈依|皈依]]其門下，從此亦被稱作「大士爺」，成為護持[[盂蘭節|盂蘭節]]普渡事項的[[護法神|護法神]]。

===鬼子母神傳說===
{{main|鬼子母神}}
傳說[[釋迦牟尼佛|釋迦牟尼佛]]住世时，饿鬼「[[鬼子母|鬼子母]]」[[神通|神通]]很大，她为了养活許多子女，专杀害人间的婴孩為生。於是，[[釋迦佛|釋迦佛]]施展[[法力|法力]]，把她的鬼子女藏在佛的[[食缽|食缽]]中。鬼子母无法找到孩子，只好向佛求救。佛向她說教，要求她不要殺世間嬰孩為食，並且願意在自己的[[僧|僧]]眾用餐之前，先[[布施|布施]]食品給她們餓鬼。「鬼子母神」悔悟，從此護持佛門，成為佛教的重要[[護法神|護法神]]之一，還專門保護幼童。

由此典故可知，饿鬼道中的飢餓[[众生|众生]]，不单自己难得饮食，而且每胎便有几百个小孩出生，令生活更见困苦。

==佛教施食==
{{main|施餓鬼}}
佛教歷來有施食解救餓鬼的傳統，如[[放焰口|放焰口]]、[[盂蘭盆|盂蘭盆]]供等。

一些佛教徒亦会在每次吃饭时留下一口食物，诵[[咒語|咒語]]及作[[手印|手印]]加持，再把它[[佈施|佈施]]给饿鬼。
