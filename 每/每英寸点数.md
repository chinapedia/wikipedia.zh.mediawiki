'''DPI'''（{{lang-en|'''D'''ots '''P'''er '''I'''nch}}，每英寸点数）是一个量度單位，用於點陣數位影像，意思是指每一[[英吋|英吋]]長度中，取樣或可顯示或輸出點的數目。如：印表機輸出可達600DPI的解析度，表示印表機可以在每一平方英吋的面積中可以輸出600X600＝360000個輸出點。

印表機所設定之[[解析度|解析度]]的DPI值越高，印出的圖像會越精細。印表機通常可以調校解析度。例如[[撞針印表機|撞針印表機]]，解析度通常是60至90 DPI。{{fact|[[噴墨印表機|噴墨印表機]]則可達1200 DPI，甚至9600 DPI。[[激光打印機|激光印表機]]則有600至1200 DPI。}}

[[File:DPI_and_PPI.png|thumb]]

一般顯示器為96 DPI，印刷所需點陣圖的DPI數則視印刷網線數而定。一般150線印刷品質需要350 DPI的點陣圖。而這裡的D（dot）就是像素（pixel）。

因為[[印表機|印表機]]的混色方式是[[CMYK|CMYK]]，所以色彩（也就是色域）表現不如[[螢幕|螢幕]]的[[RGB|RGB]]色光表現來的多與鮮豔，以至於印表機的DPI應該大於顯示器的[[每英寸像素|PPI]]（Pixels Per Inch，像素每英寸），才能與[[RGB|RGB]]在螢幕有差不多的畫質細膩表現。大部分顯示器每像素每種原色都可以有256個層次，合起來則有256<sup>3</sup>種顏色，也就是256 x 256 x 256 = 16777216 色，也就是24bit的[[真彩色|真彩色]]。而印表機則僅有[[16位色|16位色]]左右，因此印表機常常利用[[中間色|中間色]]或[[抖動|抖動]]等方法來製造更多顏色的效果，故此其解析度相當重要。

除了 DPI 外，亦有建議使用'''DPCM'''（'''d'''ots '''p'''er '''c'''enti'''m'''eter）。DPCM和 DPI都納入了[[CSS3|CSS3]]的方案內。<ref>[http://www.w3.org/TR/css3-mediaqueries Media Queries]．w3.org</ref>

[[掃瞄器|掃瞄器]]的取樣分辨率有時會用DPI來量度，但更好的做法是用{{tsl|en|Samples per inch|每英寸取样数}}（SPI）。

[[鼠标|鼠标]]的光电传感器灵敏度常用DPI来量度，但会受显示设备和操作系统[[分辨率|分辨率]]设置的影响，更好的做法是用[[每英寸测量数|每英寸测量数]]（CPI）。
