'''茗荷谷站'''（{{jpn|j=茗荷谷駅|hg=みょうがだにえき|rm=Myōgadani eki|f=y}}）是位於[[日本|日本]][[東京都|東京都]][[文京區|文京區]][[小日向|小日向]]四丁目，屬於[[東京地下鐵|東京地下鐵]][[丸之內線|丸之內線]]的[[鐵路車站|鐵路車站]]。[[車站編號|車站編號]]是'''M 23'''。

此站的後樂園方向設有{{link-ja|小石川車輛基地|小石川車両基地}}。

== 歷史 ==
* 1954年（[[昭和|昭和]]29年）1月20日：開業。
* 1993年（[[平成|平成]]5年）3月：進行月台拓寬、增設樓梯與站房改建（新建車站大樓）的改善工程<ref name="eidan">{{Cite book |title=[[帝都高速度交通營團|帝都高速度交通營團]]史 |publisher=[[東京地下鐵|東京地下鐵]]株式會社 |date=2004-12 |pages=237-238 }}</ref>。
* 1997年（平成9年）12月25日：車站大樓1樓部分（剪票口）啟用<ref>{{Cite journal|date = 1998-4 |journal = 鉄道ピクトリアル |volume = 48 |issue = 4 |page = 82 |publisher = 電気車研究会 }}</ref>。
* 1998年（平成10年）5月1日：車站大樓「茗荷谷站MF大樓」完成，車站改善工程結束。總工費約42億1000萬日圓<ref name="eidan"/><ref>{{Cite book |title=帝都高速度交通營團史 |publisher=東京地下鐵株式會社 |date=2004-12 |page=618 }}</ref>。
* 2004年（平成16年）4月1日：[[帝都高速度交通營團|帝都高速度交通營團]]（營團地下鐵）民營化，本站由東京地下鐵營運。
* 2018年（平成30年）9月、10月：所有出入口設置新型列車資訊顯示器。　

==車站結構==
[[File:Myogadani-Station-2005-6-12_3.jpg|thumb]]

設有[[相對式月台|相對式月台]]2面2線的[[地下車站|地下車站]]。月台在地下鐵隧道出口，本站～後樂園站間為地面線路。月台部分位於地面。剪票口也在地面。車站南方有{{link-ja|小石川車輛基地|小石川車両基地|留置線與車輛維修工廠}}。

1993年至1998年進行車站改良工程，新建車站大樓與拓寬月台（大手町方向從6.0公尺拓寬至6.99公尺，池袋方向月台從6.0公尺拓寬至6.39公尺）<ref name="eidan"/>。

== 站名由來 ==
站名來自於過去的茗荷谷町。車站一帶在實施住居表示後更名為小日向。

== 其他 ==
1995年7月，池袋站內進行[[轉轍器|轉轍器]]交換工程時，[[池袋站|池袋站]]～[[新大塚站|新大塚站]]間停止運行，荻窪方向發車的電車藉由[[橫渡線|橫渡線]]進入2號月台，再從2號月台折返發車。1號月台則是本站～新大塚站間的區間電車使用<ref>{{Cite book |title=[[帝都高速度交通營團|帝都高速度交通營團]]史 |publisher=[[東京地下鐵|東京地下鐵]]株式會社 |date=2004-12 |pages=245-246 }}</ref>。另外，因為當時車站大樓提前完成，2號月台與1號月台之間可透過新設的階梯往來。

丸之內線各站皆使用發車音樂，本站在周邊居民請求下停止使用，之後2009年12月21日起除早晨、深夜（22:00～翌日7:30）外重新使用營團時代的發車蜂鳴器。本站也是丸之內線唯一經常使用營團蜂鳴器的車站<ref>新宿、池袋在尖峰時刻也使用營團蜂鳴器。</ref>。
