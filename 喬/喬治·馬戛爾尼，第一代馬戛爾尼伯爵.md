'''乔治·馬戛爾尼，第一代馬戛爾尼伯爵'''，{{Post-nominals|post-noms=[[巴斯勳章|KB]]}}（{{lang-en|'''George Macartney, 1st Earl Macartney'''}}；{{bd|1737年|5月14日|1806年|5月31日|catIdx=M}}），[[英国|英国]][[政治人物|政治人物]]、[[外交官|外交官]]。

== 早年生平 ==
马戛尔尼出身于一个[[苏格兰|苏格兰]]贵族家庭，在[[爱尔兰|爱尔兰]]出生。1759年，他毕业于[[都柏林|都柏林]][[三一学院|三一学院]]，之后进入[[伦敦|伦敦]][[坦普尔大学|坦普尔大学]]进修，师从荷兰伯爵[[亨利·福克斯|亨利·福克斯]]。

1764年，他被任命为全权特使，赴[[俄罗斯帝国|俄国]]与[[叶卡捷琳娜二世|叶卡捷琳娜二世]]商谈结盟事宜。之后他进入[[英国议会|英国议会]]。1769年，他返回爱尔兰出任[[爱尔兰议会|爱尔兰议会]]议员，并出任[[爱尔兰事务大臣|爱尔兰事务大臣]]。1772年，在他辞职之后，被封为[[骑士|骑士]]。

1775年，他出任[[加勒比群岛|加勒比群岛]]总督，1776年被封为“马戛尔尼男爵”，属爱尔兰贵族序列。1780年，出任[[印度|印度]][[马德拉斯|马德拉斯]]总督，驻今[[昌奈|昌奈]]。1786年，他拒绝出任[[印度总督|印度总督]]，返回英国。1792年，他被加封为“马戛尔尼伯爵”。
==出使中国==
{{Main|馬加爾尼使團}}1793年，英王[[喬治三世|喬治三世]]以祝壽為由，派遣馬加爾尼為正使，率領使團到訪[[清|清朝]]覲見[[乾隆皇帝|乾隆皇帝]]，實則要求中國允許英國派駐公使。<ref>{{Cite wikisource|wslink=清史稿/卷154|title=清史稿　志一百二十九　邦交二英吉利|author=趙爾巽|date=|quote=五十八年，英國王雅治遣使臣馬戛爾尼等來朝貢，表請派人駐京，及通市浙江寧波、珠山、天津、廣東等地，並求減關稅，不許。}}</ref>但乾隆帝以「不合體制」為由，所請事項一律不准。

== 晚年经历 ==
1795年，马戛尔尼作为密使出访[[撒丁王国|撒丁王国]]，商谈[[反法同盟|反法同盟]]事宜，之后他再次被封为“马戛尔尼男爵”，不过这次是作为[[大不列顛|大不列顛]]贵族受封。

1796年底，他被任命为[[好望角|好望角]]总督，统治新获得的[[开普殖民地|开普殖民地]]。1798年11月，他因健康原因辞职。1806年逝世。

== 其他 ==
其后裔[[简·马戛尔尼|简·马戛尔尼]]（Jane Macartney，中文名马珍）於2008年5月開始為《[[泰晤士報|泰晤士報]]》駐[[北京|北京]]特派[[記者|記者]]，[[汶川大地震|汶川大地震]]之際做了許多相關報道。
