'''喬治亞空軍'''為[[喬治亞軍|喬治亞軍]]的空軍部隊，總員1,813人<ref>{{cite web |url=http://www.mod.gov.ge/?l=E&m=5&sm=7 |title=存档副本 |accessdate=2009-05-10 |deadurl=yes |archiveurl=https://web.archive.org/web/20110721030833/http://www.mod.gov.ge/?l=E&m=5&sm=7 |archivedate=2011-07-21 }}</ref>。除了[[Mi-8|Mi-8]]等[[直升機|直升機]]以外，也有各型號的[[Su-25攻擊機|Su-25]]。[[運輸機|運輸機]]使用[[An-2|An-2]]，而[[教練機|教練機]]兼[[攻撃機|攻撃機]]則使用[[L-29|L-29]]。另外，防空飛彈部隊也屬於喬治亞空軍的管轄範圍。

== 沿革 ==
喬治亞於1992年1月1日宣告設立航空部隊<ref>http://www.mod.gov.ge/2007/downloads/Defence_Today-N3(www_mod_gov_ge).pdf{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，是由空軍部隊與防空部隊聯合組成。1995年人員擴充至1,000名，1998年因應部隊重整而設立空軍。因為從舊蘇聯時代開始[[Su-25攻擊機|su-25]]就在[[提比里斯|提比里斯]]生產之故，喬治亞空軍一直以其作為主力機。

== 組織 ==
[[File:Roundel_of_Georgia.svg|thumb]]</center>]]
喬治亞空軍是由空軍司令部控制下的航空與防空兩部隊構成，分別位於[[提比里斯|提比里斯]]附近的[[阿列克謝耶夫卡_(喬治亞)|阿列克謝耶夫卡]]與[[馬紐裡|馬紐裡]]。目前喬治亞空軍正在其[[NATO|NATO]]夥伴的協助下進行現代化工作，尤其是[[美國|美國]]與[[土耳其|土耳其]]。另外，也曾詢問過火鶴二型靶機與中翔UAV的性能與價格<ref>http://open.nat.gov.tw/OpenFront/report/show_file.jsp?sysId=C09701272&fileNo=001{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

== 裝備 ==
由於下述部分單位可能毀於[[2008年南奧塞梯戰爭|2008年南奧塞梯戰爭]]之故，所附數量可能並不準確，總體有攻擊能力的戰機約20架。
