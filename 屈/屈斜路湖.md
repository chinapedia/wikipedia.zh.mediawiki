'''屈斜路湖'''（{{jpn|j=屈斜路湖|hg=くっしゃろこ|rm=Kussharo-ko}}）是位於[[日本|日本]][[北海道|北海道]]東部[[弟子屈町|弟子屈町]]一個天然[[湖泊|湖泊]]，全域被劃入[[阿寒國立公園|阿寒國立公園]]。為日本最大，世界第二大的[[破火山口|破火山口]]湖。過去曾經以水怪[[庫希|庫希]]而聞名。

== 地理 ==
[[File:Lake-Kussharoko-Mihorotoge.jpg|thumb]]
屈斜路湖是日本面積第六大湖泊，平均水深為28.4m米，位於[[破火山口|破火山口]][[屈斜路破火山口|屈斜路破火山口]]內，週圍被[[藻琴山|藻琴山]]、[[サマッカリヌプリ|サマッカリヌプリ]]等山峰環繞，東西寬約29公里，南北長約20公里，是日本最大的[[火山口湖|火山口湖]]<ref name="kussharoko">{{cite web |url=http://www.masyuko.or.jp/pc/sightseeing/kussharoko.html |script-title=ja:屈斜路湖|accessdate=2014-07-29|publisher={{lang|ja|てしかがえこまち推進協議会・弟子屈町}} |language=ja}}</ref>，也是世界第二大的火山口湖。雖然屈斜路湖屬於水深較淺的火山口湖，但東南部的舊噴火口一帶的最深處水深接近120米<ref>{{cite web|url=https://www.town.teshikaga.hokkaido.jp/02sougou/37kankou/05masyuko/vol02.html|script-title=ja:摩周湖ファイル vol.2|accessdate=2014-07-29|publisher={{lang|ja|弟子屈町}}|language=ja|deadurl=yes|archiveurl=https://web.archive.org/web/20101025091621/http://www.town.teshikaga.hokkaido.jp/02sougou/37kankou/05masyuko/vol02.html|archivedate=2010-10-25}}</ref>。

附近有多條小河川流入，南端則有釧路川流出。但自河川流入的水量只佔全湖水量的約20%，其餘的水量都是從地下經湖底流入。湖中央有一座名為中島的島嶼，面積為面積5.7平方公里，周長12公里，是日本最大的湖中[[島|島]]。中島本身是一座自體二重式火山，中央的[[熔岩|熔岩]]圓頂丘是中島的最高點，高355米。

南岸有一個名為[[和琴半島|和琴半島]]的[[半島|半島]]<ref>{{cite web |url=http://www.masyuko.or.jp/pc/sightseeing/wakoto.html |script-title=ja:和琴半島（摩周エリア）|accessdate=2014-07-29|publisher={{lang|ja|てしかがえこまち推進協議会・弟子屈町}} |language=ja}}</ref>。和中島相同，和琴半島在昔日也曾是因火山形成的湖中島嶼，後隨著[[尾札部川的|尾札部川的]]沖積平原的擴大，產生了沙洲並和陸地相連成為[[陸連島|陸連島]]。

== 歷史 ==
屈斜路湖週邊自100萬年以前開始就有火山活動，北側的藻琴山就是當年火山活動產生的山峰。約34萬年前火山開始噴發，約12萬年前出現了最大規模的噴發，當時火山噴發出的[[火山灰|火山灰]]幾乎覆蓋了[[札幌|札幌]]以東的北海道地區。屈斜路湖約在3萬年前形成，此後也沒有再發生足以形成破火山口的大規模噴發<ref>{{cite web |url=http://kam-kankouken.jp/tourism/masyu/special/281.html |script-title=ja:日本最大のカルデラ湖、屈斜路湖|accessdate=2014-07-29|publisher={{lang|ja|釧路湿原・阿寒湖・摩周湖 北海道東部の観光情報ポータルサイト}} |language=ja}}</ref>。

初期的屈斜路湖的面積大約比現在大一倍，湖的形狀幾乎是圓形。然而在3萬年前，由於附近的[[硫磺山_(弟子屈町)|硫磺山]]等熔岩圓頂丘群噴發<ref>{{cite web |url=http://www.data.jma.go.jp/svd/vois/data/sapporo/104_Atosanupri/104_index.html |script-title=ja:アトサヌプリ|accessdate=2014-07-29|publisher={{lang|ja|気象庁}} |language=ja}}</ref>，使得屈斜路湖東南部分成為陸地，形成現在近似豆粒的形狀。

在[[江戶時代|江戶時代]]的地圖及探險資料中，屈斜路湖被記載為「{{lang|ja|クスリ・トー}}」（kusuri-to），在[[愛努語|愛努語]]中是溫泉、藥之湖之意。之後由於釧路川源頭附近有一個名為「{{lang|ja|クッチャロ}}」的村莊（來自阿伊努語kut-car，意為「出口」<ref>{{Cite web|url=http://www.pref.hokkaido.lg.jp/ks/ass/grp/4150.pdf|title=アイヌ語地名リスト|accessdate=2017-08-16|author=北海道庁|date=|publisher=北海道庁}}</ref>），名稱遂改為現在使用的「屈斜路湖」<ref>{{cite web|url=https://www.town.teshikaga.hokkaido.jp/02sougou/37kankou/05masyuko/vol04.html|script-title=ja:摩周湖ファイル vol.4|accessdate=2014-07-29|publisher={{lang|ja|弟子屈町}}|language=ja|deadurl=yes|archiveurl=https://web.archive.org/web/20101025091626/http://www.town.teshikaga.hokkaido.jp/02sougou/37kankou/05masyuko/vol04.html|archivedate=2010-10-25}}</ref>。

== 生態 ==
屈斜路湖由於東北部有來自硫黃山和[[川湯溫泉_(北海道)|川湯溫泉]]帶有強[[酸性|酸性]]（pH值大約是2）的溫泉水流入，因此屈斜路湖湖水的pH值大約是5，是一座酸性湖，並不適宜魚類生存。

過去屈斜路湖內也曾經有過豐富的魚類棲息。在江戶時代末期的1858年（安政5年），[[松浦武四郎|松浦武四郎]]經過此地時，湖內曾有[[遠東哲羅魚|遠東哲羅魚]]、[[珠星三塊魚|珠星三塊魚]]、[[紅鱒魚|紅鱒魚]]等魚類<ref>{{cite web |url=http://www.nies.go.jp/kanko/news/23/23-4/23-4-03.html |script-title=ja:屈斜路湖が自然に中性化した原因を探るには|accessdate=2014-07-29|publisher={{lang|ja|国立環境研究所}} |language=ja}}</ref>。在1917年（大正6年）的調查中，湖中除了有[[遠東哲羅魚|遠東哲羅魚]]、[[珠星三塊魚|珠星三塊魚]]、[[遠東紅點鮭|遠東紅點鮭]]、[[櫻鱒|櫻鱒]]、[[杜父魚|杜父魚]]、[[三刺魚|三刺魚]]生存之外，還有從海中逆流而來的[[大麻哈魚|大麻哈魚]]。除了魚類，水中也有眾多植物浮游生物、[[苦草|苦草]]和[[眼子菜|眼子菜]]等水草生存。難以想像是個屈斜路湖酸性湖。

湖中生態的情況在1929年（昭和4年）時發生變化。在這一年，許多魚類都消失或只出現在小部分範圍，水草也變得十分少見。湖水表面的pH值變為4.9至5.1，呈現酸性。此後雖曾有一段時期酸性度減弱，魚類數量也有恢復。但在1938年（昭和13年）5月屈斜路地震後，因湖底噴出[[硫酸塩|硫酸塩]]，湖水的pH值變至約為4，導致湖中魚類幾乎全部死亡。受此影響，屈斜路湖現在也沒有[[漁業權|漁業權]]。

近年來湖水的酸性度有所降低，放流的[[虹鱒魚|虹鱒魚]]可以生存。

另外，和琴半島是[[晝鳴蟬|晝鳴蟬]]的最北棲息地<ref name="kussharoko"></ref>。1951年，「和琴晝鳴蟬發生地」被指定為日本的國家天然記念物。

也有一些[[大天鵝|大天鵝]]飛至屈斜路湖，在此棲息。

== 傳說 ==
在愛努族的傳說，古代的英雄曾抓到巨大的遠東紅點鮭並將其綁在山上，而暴躁的鮭魚因此擊碎了山峰，形成了湖中的中島<ref name="kussharoko"></ref>。

在昭和50年代（1975年到1985年頃）時，曾有多人在湖面上目擊到巨大物體游泳並在湖面掀起波浪，開始有屈斜路湖內存有巨大的不明生物的流言，並被電視專題報導。這一生物被命名為[[庫希|庫希]]<ref name="kussharoko"></ref>。然而，由於屈斜路湖湖水酸性極強，一般認為這裡很難有大型水生生物存在。

== 溫泉 ==
由於屈斜路湖位於[[火山|火山]]地帶，附近有眾多[[火山|火山]]和[[溫泉|溫泉]]。在東側湖岸直接在湖畔挖掘地面即會湧出溫泉，因此得到「砂湯」的地名，也成為著名的旅遊點。<ref name="kussharoko"></ref>另外受火山活動的影響，自屈斜路湖的湖底也有溫泉噴出。

屈斜路湖湖岸附近的主要溫泉包括[[仁伏溫泉|仁伏溫泉]]、[[和琴溫泉|和琴溫泉]]、[[三香溫泉|三香溫泉]]、コタン温泉[[池之湯溫泉|池之湯溫泉]]和[[砂湯溫泉|砂湯溫泉]]，在湖東約5公里處還有[[川湯溫泉_(北海道)|川湯溫泉]]。
