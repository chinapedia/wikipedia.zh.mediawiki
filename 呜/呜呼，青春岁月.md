《'''呜呼，青春岁月'''》（{{jpn|j=嗚呼、青春の日々}}）是[[日本|日本]]二人组合[[柚子_(組合)|柚子]]（ゆず）的第9张单曲。<ref>{{Cite web|url = http://www.senha-yuzu.jp/discography/single/index9.php|title = ゆず Official Website | ディスコグラフィ | シングル|accessdate = 2016-02-26|author = |date = |publisher = |language = ja|deadurl = yes|archiveurl = https://web.archive.org/web/20150707195711/http://www.senha-yuzu.jp/discography/single/index9.php|archivedate = 2015-07-07}}</ref><ref>{{Cite web|url = http://www.oricon.co.jp/prof/30824/products/225284/1/|title = 嗚呼、青春の日々 | ゆず | ORICON STYLE|accessdate = 2015-07-30|author = |date = |publisher = |language = ja}}</ref>2000年5月31日由其所属公司{{link-ja|Senha&Company|セーニャ・アンド・カンパニー}}发售。本作使柚子第一次登上日本[[Oricon公信榜|Oricon公信榜]]首位。

单曲封面插画由漫画家[[本宫宏志|本宫宏志]]绘制。
