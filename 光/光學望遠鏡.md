'''光學望遠鏡'''是用於收集[[可見光|可見光]]的一種[[望遠鏡|望遠鏡]]，並且經由[[焦點|聚焦]][[光線|光線]]，可以直接[[放大|放大]]影像、進行目視觀測或者[[相片|攝影]]等等，特別是指用於觀察夜空，固定在[[架台|架台]]上的[[單筒|單筒]]望遠鏡，也包括手持的[[雙筒望遠鏡|雙筒鏡]]和其他用途的望遠鏡。

光是由[[光子|光子]]構成，而專業的望遠鏡會由電子探測器來收集光子。光學望遠鏡有三種主要的形式：[[折射望遠鏡|折射望遠鏡]]（使用[[凸透鏡|凸透鏡]]折射聚焦）、[[反射望遠鏡|反射望遠鏡]]（以鏡片反射光線並聚焦）和使用透鏡和反射鏡片組合的[[折反射望遠鏡|折反射望遠鏡]]（複合式望遠鏡），如[[折反射望遠鏡|馬克蘇托夫望遠鏡]]和[[折反射望遠鏡|史密特攝星鏡]]。

== 工作原理 ==
{{see also|反射望遠鏡|折射望遠鏡|折反射望遠鏡}}

設計圖中最基本的元素是收集光線的物鏡（[[透鏡|透鏡]]（1））、在一段距離外的物體（4）在焦平面上形成一個[[實像|實像]]（5）。這個影像可以被記錄或經過作用如同[[放大鏡|放大鏡]]的[[目鏡|目鏡]]（2），讓眼睛（3）可以看見遠處[[放大|被放大]]的[[虛像_(光學)|虛像]]（6）。
[[File:Kepschem2.jpg|thumb]]

使用兩個凸透鏡成像的望遠鏡產生的影像是倒置的，觀賞地面景物的望遠鏡和[[雙筒望遠鏡|雙筒望遠鏡]]使用[[稜鏡|稜鏡]]（一般為[[普羅稜鏡|普羅稜鏡]]）或是在物鏡和目鏡之間再安裝一個或更多的透鏡將影像轉正，這樣就能看見正立像。

許多形式的望遠鏡會使用次鏡（副鏡）甚至第三個鏡片來摺疊光路，這些也許是光學設計的整體部分（[[卡塞格林反射鏡|卡塞格林反射鏡]]和其他類似），但也有望遠鏡以更簡潔的方法和在更方便的位置上安置目鏡或探測器使用。在大型望遠鏡上，這些附加的鏡片通常是為了提供更大的視野或是改善影像的品質。

== 角分辨率 ==
忽略大氣擾動（[[視寧度|視寧度]]或稱視象度）對影像品質的影響和光學望遠鏡的缺點，一架光學望遠鏡的[[角分辨率|角分辨率]]取決於物鏡，也就是望遠鏡口徑大小。[[雷利準則|雷利準則]]提供分辨力的極限值<math>\alpha_R</math>（徑度量）：
:<math>\alpha_R = 1.22 \lambda / D</math>,
此處<math>\lambda</math>是光的[[波長|波長]]，<math>D</math>是望遠鏡的口徑。對[[可見光|可見光]]（<math>\lambda = 550 {rm nm}</math>），公式可寫成：
:<math>\alpha_R = 138 / D</math>.
此處，<math>\alpha_R</math>以[[角秒|角秒]]為單位的極限值，口徑<math>D</math>單位是毫米。
在理想情況下，一對[[聯星|聯星]]的距離即使略小於這個極限值<math>\alpha_R</math>也還能被分辨出來，這就是戴維斯極限：
:<math>\alpha_D = 116 / D</math>.
實際上，口徑越大，角分辨率就越好。

此處要特別強調的是，角解析度不是為望遠鏡的最大[[放大率|放大率]]（或倍率）所提供的，經銷商所提供的最大倍數是望遠鏡倍率的上限值，由於超越了物鏡能力範圍的最大倍率與角分辨率，不能把影像變得更清楚，通常得到的影像品質也是最差的。

對大型的固定地基望遠鏡，角分辨率的極限是由[[視寧度|視象度]]決定，現今發展之望遠鏡安置在大氣層之上，來消除空氣對影像擾動影響角分辨率，也就是[[太空望遠鏡|太空望遠鏡]]、氣球望遠鏡和安裝在飛機上的望遠鏡（[[柯伊伯機載天文台|古柏機載天文台]]、[[同溫層紅外線天文台|同溫層紅外線天文台]]（SOFIA）或將地基望遠鏡加裝[[調適光學|調適光學]]和[[斑點成像|斑點成像]]。）
 
近來，光學望遠鏡的[[綜合口徑|綜合口徑]]陣列變得更實用，經由空間中一組小口徑望遠鏡組合，在小心操控的光學平面連結下，可以獲得更高的解析度。但是這些[[干涉儀|干涉儀]]仍只能用於觀測明亮天體，像是恆星或是[[活躍星系|活躍星系]]核，例如[[參宿四|參宿四]]的星斑影像可以在[https://web.archive.org/web/20070614111315/http://www.mrao.cam.ac.uk/telescopes/coast/betel.html 此]看見。

== 焦長和焦比 ==
[[File:Laser_Towards_Milky_Ways_Centre.jpg|thumb]]利用雷射校準強化畫質的光學望遠鏡]]
焦距決定了望遠鏡在配上[[目鏡|目鏡]]、一定大小的[[CCD|CCD]]或普通[[底片|底片]]後可能觀看的視野大小。望遠鏡的[[焦比|焦比]]（焦距比或f數，即攝影術語之「[[光圈|光圈]]」）是[[焦長|焦長]]和物鏡[[口徑|口徑]]（直徑）比值。因此當口徑（集光力）不變時，焦比低的視野較大。廣角望遠鏡（像是[[天體照相儀|天體照相儀]]）用來追蹤[[衛星|衛星]]和[[小行星|小行星]]，或是從事[[宇宙射線|宇宙射線]]的研究和巡天觀測。低焦比望遠鏡的[[光學系統的像差|像差]]比高焦比的更難以消除。

== 集光力 ==
一架望遠鏡的集光力直接與物鏡（透鏡或鏡片）的[[直徑|直徑]]（即口徑）有關。要注意圓面積與半徑的平方成正比，因此當望遠鏡的鏡片直徑增加三倍時，集光力會增加九倍，口徑越大收集的光線越多；另外靈敏度高的影像設備（如[[CCD|CCD]]）能在較少的光量下獲得比較好的影像品質。

== 研究用望遠鏡 ==
幾乎所有用於研究的大型天文望遠鏡都是反射鏡，其原因是： 
*在採用透鏡之下，必須整塊鏡片材料皆為沒有缺點和均勻而沒有多相性，而反射鏡只需要將一個表面完美的磨光，磨制相對簡易。
*除真空環境下，不同顏色的光在穿透介質時會有不同的播速度，這會造成折射鏡特有的[[色差|色差]]。
*大口徑透鏡在製造和操作上都有技術上的困難。其一是所有的材料都會因為重力而下垂，觀測舉得最高而且也是相對較重的透鏡只能在鏡片周圍加以支撐，另一方面，面鏡除了反射面以外，可以在反射面的背面和其他的側邊進行支撐。

光學望遠鏡大小在20世紀穩定的增加，在1910至1940年增加一倍，在1940至1990年又增加一倍。現在最大的望遠鏡是11公尺的[[南非大望遠鏡|南非大望遠鏡]]和[[霍比-埃伯利望远镜|霍比-埃伯利望远镜]]，以及10.4公尺的[[加那利大型望远镜|加那利大型望远镜]]。

在1980年代，在技術上作出改進的新一代望遠鏡有了長足的進步，這些進步包括多鏡片望遠鏡，可以控制鏡片的個人電腦，另一個主要的進展是旋轉的熔爐，可以用離心力讓望遠鏡的鏡片在融爐中就接近要磨制的形狀（曲率半徑）。

== 其他形式 ==
*[[雙筒望遠鏡|雙筒望遠鏡]]是將兩架單筒望遠鏡肩並肩的組合在一起，而能同時使用的望遠鏡。這種望遠鏡最主要的實用優點不是放大，而是在[[黎明|黎明]]或[[薄暮|薄暮]]時有明亮的視場。與[[指南針|指南針]]結合在一起的單筒或雙筒望遠鏡，在軍隊的炮兵單位和船艦會用於[[三角測量|三角測量]]與[[地形|地形]]（海岸）特徵的導航上。手持的望遠鏡不會受到[[手震|手震]]影響的極限是七倍，因此要有明亮的視場和最佳倍數的雙筒望遠鏡是7×50的規格。

由於雙筒望遠鏡有視場較廣，較明亮且容易操作、較專業望遠鏡便宜等原因，成為[[天文愛好者|天文愛好者]]平時學習觀測的常用器材，而較大口徑的雙筒望遠鏡更成為了一些[[天文愛好者|天文愛好者]]成功尋得新[[彗星|彗星]]的重要器材；另外亦有天文愛好者嘗試把兩具同一口徑的[[反射望遠鏡|反射望遠鏡]]組裝成雙筒望遠鏡。
