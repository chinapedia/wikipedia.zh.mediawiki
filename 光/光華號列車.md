'''光華號'''是[[臺灣鐵路管理局|臺灣鐵路管理局]]所轄[[客運列車|客運列車]]車種之一，是[[自強號列車|自強號]]開行之前的最快速車種，於[[西部幹線|西部幹線]]以[[台鐵DR2700型柴油車|DR2700型柴油客車]]行駛，從臺北到高雄僅需4個多小時，[[鐵道迷|鐵道迷]]戲稱為「白鐵仔」。光華號於1979年7月停駛，原使用車輛改行駛柴油特快車、柴油快車、柴油普通車及普快車。臺鐵偶將各式活動中行駛的[[台鐵DR2700型柴油車|DR2700型柴油客車]]稱呼為光華號，但是由於各站的到開時刻表已不再設定「光華號」字樣，只會顯示為「觀光列車」或「專車」（2016年10月29日臺北站除外）。

[[臺東線|臺東線]]則於1968年11月1日起開行[[舊東線光華號|光華號]]（參見[[舊東線光華號|台東線光華號]]）。

== 簡介 ==
1966年10月31日，為了配合[[先總統蔣公|先總統蔣公]]80歲華誕，[[臺灣鐵路管理局|臺灣鐵路管理局]]於[[縱貫線_(鐵路)|縱貫線]]以[[世界銀行|世界銀行]]貸款新購進口之[[台鐵DR2700型柴油車|DR2700型柴油客車/DR2750專用拖車]]開行光華號，行駛臺北－高雄之間半直達特快車。經由[[海線|海線]]者僅停臺南站，只需4小時45分（高雄7時20分開車、臺南7時57分開車、臺北12時05分到達）。台北－台中間經山線的直達列車只需2小時20分。<ref>許乃懿. 走過台灣三十年. 鐵道情報91/92期 (中華民國鐵道文化協會). 1996年9月/10月: 頁1-35.</ref>
在[[鐵路電氣化|鐵路電氣化]]之前，光華號有十餘年時間為臺灣陸上最快速的交通工具<ref>{{cite web|url=http://activity.pts.org.tw/pts/point/archive2.aspx?story_id=415|title=《紀錄觀點》白鐵光華|publisher=[[公共電視文化事業基金會|公共電視文化事業基金會]]|date=2011-01-25|accessdate=2011-01-25|language=zh-tw}}</ref>。初期某些班次尚有附送[[便當|便當]]（依照1970年的《[[經濟日報_(台灣)|經濟日報]]》，只有2002次有餐點，價錢已包含於票價內，為[[新台幣|新台幣]]18元）。最初票價是臺北－臺中為新台幣71元，臺北－臺南為新台幣138元5角，臺北－高雄為新台幣158元，臺南－高雄為新台幣20元。

由於DR2700型客車維修基地為台中，1970年代部分經由海線南下班次於彰化進行分離，部分繼續南下，部分向北駛往台中；北上列車則於彰化與由台中南下的列車連結後繼續北上。於1970年開始，光華號延駛屏東，於高雄站進行列車分離與連結；1972年延駛基隆，則於台北站進行列車分離與連結。<ref>許乃懿. 走過台灣三十年. 鐵道情報91/92期 (中華民國鐵道文化協會). 1996年9月/10月: 頁1-35.</ref>

由於西部幹線進行[[鐵路電氣化|鐵路電氣化]]工程，光華號的排點漸次減慢，台北 - 高雄間行車時間約達5小時10分，行車時間難以與1978年通車的[[中山高速公路|中山高速公路]]相比；加以鐵路電氣化工程完工後引入速度更快、內裝更高級且有空調的[[台鐵EMU100型電聯車|自強號電聯車]]，光華號於1979年7月14日停駛，原使用DR2700型客車改行駛[[台鐵對號特快|柴油對號快車]]。

== 歷史 ==
*1966年10月31日：開始行駛行駛於台北－高雄之間。
*1968年11月1日：[[臺東線|臺東線]]開行[[舊東線光華號|光華號]]。
*1970年8月1日：光華號開始開行屏東-基隆區間。
*1979年7月14日：西幹線光華號開出最後一班列車。
*1982年2月1日：[[舊東線光華號|臺東線光華號]]因[[臺東線|臺東線]]鐵路拓寬完工而停駛。

===重大事故<ref>{{Cite web|url=http://ecpoem.pixnet.net/blog/post/23500082|title=1973年11月5日嘉義事故|accessdate=2019-03-02|author=poem|date=|publisher=}}</ref>===
*1966年11月4日：光華號第一起行車事故。地點是台北市寶慶路口，一位行人死亡。
*1973年11月5日：2003次與卡車車禍，連同卡車共3死46人輕重傷。嘉義市博愛路「台斗坑平交道」後改建為今之嘉義市博愛陸橋。
*1976年7月21日：2001次與卡車車禍，乘客1人死亡。
*1978年5月13日：2002次與卡車車禍，乘客31傷。
*1979年1月22日：3086次與卡車車禍，機車長與列車長殉職。
在以上事故中，皆無DR2700型客車報廢。
