'''光嚴天皇'''（{{jpn|j=光厳天皇|hg=こうごんてんのう|rm=Kōgon Tennō}}，1313年8月1日—1364年8月5日，舊曆：[[正和|正和]]2年[[七月初九|七月初九]]—[[正平|正平]]19年 / [[貞治|貞治]]3年[[七月初七|七月初七]]）是[[日本|日本]]北朝第一代[[天皇|天皇]]（在位：[[元弘|元弘]]元年9月20日（1331年10月22日） - 元弘3年 / [[正慶|正慶]]2年5月25日（1333年7月7日））。讳'''量仁'''（{{jpn|j=量仁|hg=かずひと|rm=Kazuhito}}）。

== 略歷 ==
量仁是[[持明院統|持明院統]]出身的[[後伏見天皇|後伏見天皇]]的第三皇子。當時，日本的皇室分為[[持明院統|持明院統]]和[[大覺寺統|大覺寺統]]兩派，就皇位繼承問題發生了非常激烈的鬥爭。在[[鐮倉幕府|鐮倉幕府]]的調停下，雙方約定由兩派出身的皇族輪流擔任天皇，即[[兩統迭立|兩統迭立]]。當時所規定的皇位繼承順序是：[[後二條天皇|後二條天皇邦治]]（大覺寺統）→[[花園天皇|花園天皇富仁]]（持明院統）→[[後醍醐天皇|後醍醐天皇尊治]]（大覺寺統）→[[邦良親王|邦良親王]]（大覺寺統、後二條嫡子）→量仁親王（持明院統）。1326年（[[正中|正中]]3年）陰曆3月，邦良親王被後醍醐天皇立為皇太子後，但不久就逝世了。同年（當時已改元[[嘉曆|嘉曆]]）陰曆7月24日，量仁親王在[[鐮倉幕府|鐮倉幕府]]的支持下，被後醍醐天皇立為皇太子。但後醍醐天皇之前試圖將自己的兒子立為太子，因此這引起了天皇的不滿。

1331年（[[元德|元德]]3年）後醍醐天皇的倒幕計劃被鐮倉幕府發覺，陰曆8月，後醍醐天皇逃往[[笠置山|笠置山]]（今[[京都府|京都府]][[相樂郡|相樂郡]]笠置町內），9月20日，鐮倉幕府擁立量仁親王即位，是為光嚴天皇。由[[後伏見天皇|後伏見上皇]]開設[[院政|院政]]，立邦良親王的嫡子[[康仁親王|康仁親王]]為太子。翌年後醍醐天皇被捕，流放[[隱岐|隱岐]]。但後醍醐天皇拒絕退位，與光嚴天皇並立，廣義上的南北朝時代開始。

1333年（[[元弘|元弘]]3年 / [[正慶|正慶]]2年），[[足利尊氏|足利尊氏]]率軍襲擊了鐮倉幕府在[[京都|京都]]設置的[[六波羅探題|六波羅探題]]，光嚴天皇同[[探題|探題]][[北條仲時|北條仲時]]、[[北條時益|北條時益]]以及[[花園天皇|花園上皇]]逃往[[東國|東國]]，但在[[近江|近江]]的番場宿（今[[滋賀縣|滋賀縣]][[米原市|米原市]]）被捕。（[[元弘之亂|元弘之亂]]）5月25日，光嚴天皇被廢黜。然而光嚴天皇拒絕退位。後醍醐天皇以光嚴是自己的廢太子為由，特例尊之為上皇。這是遵循了[[平安時代|平安時代]][[後一條天皇|後一條天皇]]廢太子[[敦明親王|敦明親王]]（小一條院）為上皇的先例。

然而回到京都的[[後醍醐天皇|後醍醐天皇]]在[[建武新政|建武新政]]中失敗，遭致[[武士_(日本)|武士階層]]的不滿。1335年（[[建武|建武]]2年），足利尊氏在[[鐮倉|鐮倉]]反叛後醍醐天皇，後醍醐命令[[新田義貞|新田義貞]]討伐足利尊氏，而光嚴上皇則向尊氏下達了討伐義貞的[[院宣|院宣]]。翌年足利尊氏戰敗，逃亡[[九州島|九州]]；不久足利尊氏恢復了勢力並佔領了[[京都|京都]]，8月15日立光嚴上皇的弟弟豐仁親王即位，是為[[光明天皇|光明天皇]]。光嚴上皇在京都開設[[院政|院政]]，足利尊氏則在京都開設[[室町幕府|室町幕府]]。[[大覺寺統|大覺寺統]]的[[後醍醐天皇|後醍醐天皇]]逃往[[吉野|吉野]]，重新開設朝廷，與光明天皇的[[北朝_(日本)|北朝]]並立，日本歷史正式進入[[南北朝_(日本)|南北朝時代]]。此後在1348年，光明天皇將皇位讓給了光嚴上皇的兒子[[崇光天皇|崇光天皇]]，光嚴上皇依舊以[[治天之君|治天之君]]的身份開設院政。

1350年（[[觀應|觀應]]元年 / 正平5年）[[室町幕府|室町幕府]][[足利將軍家|足利將軍家]]發生[[觀應擾亂|內亂]]，[[足利直義|足利直義]]在與[[執事|執事]][[高師直|高師直]]的爭權中失勢，投奔南朝。同年足利直義率南朝軍在[[播磨國|播磨國]][[光明寺城|光明寺城]]（[[光明寺合戰|光明寺合戰]]）和[[攝津國|攝津國]]打出濱（[[打出濱之戰|打出濱之戰]]）中大破[[足利尊氏|足利尊氏]]所率的北朝軍。足利尊氏不得不向直義妥協，逼迫高師直出家並向直義所支持的南朝投降。次年足利尊氏廢黜了[[崇光天皇|崇光天皇]]和皇太子[[直仁親王|直仁親王]]，停止了光嚴上皇的院政，並將北朝一系的皇族囚禁，將[[三神器|三神器]]送交南朝朝廷，史稱[[正平一統|正平一統]]。

1352年陰曆閏2月，南朝的[[後村上天皇|後村上天皇]]趁[[足利尊氏|足利尊氏]]討伐[[關東|關東]]的[[足利直義|足利直義]]之機，派[[楠木正儀|楠木正儀]]攻陷了京都。光嚴上皇、[[光明天皇|光明上皇]]、[[崇光天皇|崇光天皇]]以及皇太子[[直仁親王|直仁親王]]等人被南朝軍俘虜。3月足利尊氏開始反撲，南朝軍挾光嚴等人撤往[[山城國|山城國]]的男山（今[[京都府|京都府]][[八幡市|八幡市]]），後來經[[河內國|河內國]]東條回到後村上天皇的行宮所在地[[大和國|大和國]]的[[賀名生|賀名生]]（今[[奈良縣|奈良縣]][[五條市|五條市]]）。

光嚴上皇落入與其敵對的大覺寺統南朝手裡，過著囚禁生活，從此徹底斷絕了執政的願望，在[[夢窗疎石|夢窗疎石]]的勸說下歸依佛門，[[法名|法名]]'''勝光智'''（後改'''光智'''）。光嚴上皇被南朝軟禁了5年，於1357年（正平12年 / [[延文|延文]]2年）從河內國[[金剛寺|金剛寺]]回到京都，進入深草金剛壽院，此後在嵯峨小倉一帶隱居。光嚴上皇斷絕了世俗之念，歸依禪門，師事於[[春屋妙葩|春屋妙葩]]。

1362年（正平17年 / [[康安|康安]]2年）陰曆9月，光嚴法皇參拜了[[法隆寺|法隆寺]]。此後法皇雲遊[[大和國|大和]]、[[紀伊國|紀伊]]，而根據《[[太平記|太平記]]》和《[[大乘院日記目錄|大乘院日記目錄]]》的說法，法皇曾前往[[吉野|吉野]]會見[[後村上天皇|後村上天皇]]，與這位昔日的敵人握手言和。但這沒有史實根據。

晚年在[[丹波|丹波]]山國莊的[[常照皇寺|常照皇寺]]（位於今[[京都府|京都府]][[京都市|京都市]][[右京區|右京區]]京北井戶町）潛修佛法。1364年於該地病逝。

光嚴天皇擅於創作[[和歌|和歌]]，是後期[[京極派|京極派]]重要的一員。他曾親自參與了由[[花園天皇|花園上皇]]指導的《[[風雅和歌集|風雅和歌集]]》的編纂。有《[[光嚴院御集|光嚴院御集]]》傳世。
