'''光启公园'''位于[[上海市|上海市]][[徐汇区|徐汇区]][[南丹路|南丹路]]，原名'''南丹公园'''，1983年因纪念[[徐光启|徐光启]]逝世350周年而改为现名，园内主体建筑是[[徐光启墓|徐光启墓]]。2003年，上海现存最古老的民居[[南春华堂|南春华堂]]因抢救性保护搬迁于其中，被辟为[[徐光启纪念馆|徐光启纪念馆]]。光启公园地处上海主要商业区之一的[[徐家汇|徐家汇]]，附近有[[徐家汇圣依纳爵主教座堂|徐家汇圣依纳爵主教座堂]]、[[天主教上海教区|天主教上海教区]][[主教府|主教府]]、[[中国科学院上海天文台|中国科学院上海天文台]]、上海市气象局、[[徐汇区|徐汇区]]政府大楼、[[上海轨道交通|上海轨道交通]][[徐家汇站|徐家汇站]]。

== 徐光启墓 ==
{{Main|徐光启墓}}

'''徐光启墓'''是明末科学家、思想家、政治家、军事家，[[天主教|天主教]][[教友|教友]]领袖和[[护教士|护教士]]，[[礼部尚书|礼部尚书]]兼[[文渊阁大学士|文渊阁大学士]][[徐光启|徐光启]]（1562年4月24日－1633年11月10日）及其部分亲属的墓地。

徐光启家族墓地位于2条河流（[[肇嘉浜|肇嘉浜]]和[[法华泾|法华泾]]）的汇合处，附近后来形成村落，称为[[徐家汇|徐家汇]]。1842年天主教[[耶稣会|耶稣会]]重新来华后，在1847年选中徐家汇建立江南传教区的总部，陆续建成一系列教会机构，形成一大片教会区。1914年，徐家汇以东地区并入[[上海法租界|上海法租界]]，因而徐光启墓所在的教会区也就自然融入了[[上海市区|上海市区]]。1990年代以后，紧邻徐光启墓的[[徐家汇|徐家汇]]地区迅速演变为上海最主要的商圈之一，陆续开设许多商店，矗立起许多高层建筑。

崇祯十四年（1641年），徐光启归葬于上海。墓地共十个墓穴，葬有徐光启、夫人吴氏和他的四个孙辈夫妇。墓地附近形成村落，后称为[[徐家汇|徐家汇]]。光绪二十九年（1903年），天主教江南代牧区在徐光启[[受洗|受洗]]三百周年之际，重修了墓地。1933年徐光启逝世三百周年，墓地又获重修。[[抗日战争|抗日战争]]期间，墓地荒废成为[[菜畦|菜畦]]。1957年，[[上海市文化局|上海市文化局]]重修徐光启墓，复建十字架基台，并将其列为[[上海市文物保护单位|上海市文物保护单位]]。“[[文化大革命|文化大革命]]”期间，墓地的牌坊、华表遭到破坏。1978年墓地重辟为南丹公园，此后陆续修建了一系列纪念物。1983年徐光启逝世350周年，'''南丹公园'''改名为'''光启公园'''。1988年，徐光启墓被[[中华人民共和国国务院|中华人民共和国国务院]]列为[[全国重点文物保护单位|全国重点文物保护单位]]。2003年12月，徐光启墓修复工程竣工，恢复了1903年的墓制。<ref name="a">{{cite web |url = http://www.shtong.gov.cn/newsite/node2/node71994/node82435/node82459/index.html |title = 光启公园 |publisher = 上海市地方志办公室 }}</ref>

徐光启墓占地300平方米，高2.2米，为椭圆形大墓。墓前立有石碑、[[十字架|十字架]]，石人、石马、华表和石牌坊。墓碑为数学家[[苏步青|苏步青]]手书“明徐光启之墓”。石牌坊刻有[[对联|对联]]：“治历明农百世师经天纬地，出将入相一个臣奋武揆文”。墓东侧为碑廊，刻有徐光启手迹、[[查继佐|查继佐]]撰《徐光启传》及[[程十髮|程十髮]]临摹的徐光启画像。<ref name=a></ref>

==南春华堂==
[[File:Xu_Guangqi_Memorial_Hall.jpg|thumb]]

上海现存最古老的民居'''南春华堂'''原位于徐汇区[[梅陇路|梅陇路]]南春华堂5号，2003年经抢救性保护搬迁至光启公园并辟为'''徐光启纪念馆'''。南春华堂建于明[[弘治|弘治]]末年至[[正德|正德]]年间，距今已有500多年的历史。<ref name=a></ref>
