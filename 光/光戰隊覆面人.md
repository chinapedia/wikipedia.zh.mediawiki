《'''光戰隊覆面人'''》<ref name="註釋">關於劇中設定牽涉到[[佛教|佛教]][[大日如來|大日如來]]的法像及[[手印|手印]]等，其實佛家對此方面是頗有忌諱的，除了對佛法的不敬之外，手印也不可以隨便結，此外佛法亦不可用來戲論。</ref>（原題：{{lang|ja|光戦隊マスクマン}}，台譯：光戰隊蒙面人，港譯：超能戰士）是日本[[東映|東映]]公司在1987年製作的「[[超級戰隊系列|超級戰隊系列]]」第11部[[特攝|特攝]]作品。1987年（昭和62年）2月28日至1988年（昭和63年）2月20日期間於[[朝日電視台|朝日電視台]]逢星期五18：00 - 18：25播映，全51集。

本片亦為超級戰隊系列在[[香港|香港]]播放的首部作品（不計[[石森章太郎|石森章太郎]]原作的兩部作品在內），1989年由[[無綫電視翡翠台|無綫電視翡翠台]]以《超能戰士》名稱播出。

本片曾遭奧姆真理教用作宣傳教義之用，例如有些信徒挪用主題曲作為真理教教會歌，特別是劇中姿三十郎長官的「空中漂浮」畫面，更常令人聯想到其教主[[松本智津夫|松本智津夫]]（麻原彰晃）1985年於《ムー》雜誌上發表的所謂「發功浮空照」（但亦有其他說法表示，此照片實為利用相機快門連拍方式偽造）<ref>{{Cite|url=http://npn.co.jp/article/detail/45822915/|title=オウム真理教の影響で発禁になりかけたヒーロー!?}}</ref> 。

== 故事 ==
長達住在地底下已久的地底人，不滿地上人的光明與燦爛和平的生活，決定展開大規模侵略地上世界的邪惡組織～'''地獄帝國丘布'''在齊巴帝王的率領之下，目的就是讓地上的世界變成黑暗世界。於是，知道此情形的光戰隊覆面人長官－姿三十郎，就在日本各地四處尋找五位戰士，還因此成立了為速度與夢想而奔馳的'''姿賽車隊'''，並且在東京的某大樓設置指揮中心，他訓練光戰隊並唯一能戰勝丘布帝國的方法就是提昇他們的力量以超越人體極限，藉此鍛鍊出光能量，讓他們得以發揮，成為真正的光之戰士。

== 姿賽車隊‧覆面人MASKMAN ==

== 地底帝國丘布 ==
統治著人類所未知的地底世界之大帝國。地底下原本是和平的世界，不過，因[[政變|政變]]導致原本統治的地底王室沒落衰敗。而現今支配地下的'''地帝王齊巴'''，以冷血並威權的方式統治著地底王國丘布，而牠那貪得無厭的征服野心，伸向了人類居住的地上世界，遂率領著眾幹部及地帝獸進行了侵略的行動。
