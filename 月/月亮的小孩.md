《'''[https://www.youtube.com/watch?v=ApyJHQc4IlI 月亮的小孩]'''》是台灣導演[[吳乙峰|吳乙峰]]拍攝並於1990年出版上映的紀錄片。這部片以[[白化症|白化症]]者為拍攝對象，呈現他們如何在社會主流社群的歧視當中生活和尋找出路。<ref>author=邱貴芬 |date=2006.01|title=「看見台灣」：台灣新紀錄片研究</ref>曾代表台灣至[[加拿大|加拿大]]、美國[[巴爾的摩|巴爾的摩]]，以及日本[[山形市|山形市]]參加國際紀錄片影展。

由於片長不符合當時電視台所訂的規格，吳乙峰於是至全台巡迴播映，超過400場次的播映紀錄也開啟紀錄片新的映演模式。《月亮的小孩》改變以往觀眾對紀錄片多與社會運動結合的印象(例如1980年代拍攝[[桃園機場事件|桃園機場事件]]一戰成名的的[[綠色小組|綠色小組]])。吳乙峰以弱勢族群為拍攝對象，以感動人心為拍攝動機，創造新的紀錄片風貌。

李泳泉認為《月亮的小孩》在台灣紀錄片中有深遠的影響：
{{cquote|綜觀台灣紀錄片的發展，在形塑現今紀錄片風貌上，影響最深遠者，當推《月亮的小孩》／吳乙峰／全景映像工作室|李泳泉<ref>李泳泉，
2002。〈全景學派的誕生：台灣紀錄片的趨勢觀察（1990-2000）〉，《電影欣賞》第一百一十一期，頁19</ref><!--http://lfbxmljy.blog.sohu.com/178487067.html--><ref>[http://routerjcs.nctu.edu.tw/router/word/31325242016.pdf 再現弱勢：從《月亮的小孩》到《水蜜桃阿嬤》 蔡慶同]</ref>}}。

== 劇情 ==

吳乙峰訪問了許多白化症者的家庭，其中以吳國煌與蕭錦惠的婚姻為主線，從他們結婚到生產，紀錄了夫妻兩人、婆婆和友人朱勝賀等人對於生產的態度和想法。

影片一開始，觀眾聽見吳乙峰訪問一位女性(蕭錦惠)的聲音，畫面中不見人影只有漸漸放大的月亮，吳乙峰訪問的內容衝擊了觀眾平常對白化症者歧視的眼光，激起觀眾對白化症者的同理心。吳乙峰以蕭錦惠誓死要嫁給吳國煌的決心作為開場，不僅引起了觀眾的好奇心，影片特殊的處理方式也展現他對弱勢族群的尊重與關懷。

白化症者經常因為外表異於一般人遭致歧視，吳國煌和朱勝賀也都因此失去在工廠和銀行的工作，後來選擇從事按摩師的工作。湯雅玲和王晨琪則是受到同學的欺負和嘲笑。白化症者在台灣數量並不少，他們會舉辦白化者聯誼活動，彼此交流、訴說自己的故事，讓自己知道自己並不孤單。

劇情進展的最高潮在吳國煌和蕭錦惠的小孩即將出生時，大家都在猜到底生出來會是白頭髮(白化症)還是黑頭髮(不同於一般人是擔心到底是生男還是生女)。

片尾聲音的主角是另一名同樣想嫁給白化症者的女性。吳乙峰問她：「你希不希望跟你父母講一點話？」她立刻拒絕並開始哭泣，她說：「不要，因為他們從來不肯聽我說說我對他的感覺是什麼。」吳乙峰的紀錄片給了這些受訪對象一個發聲的管道，我們可以透過影片聽到他們內心的想法和委屈，而這些心聲平日裡被埋藏的真正原因，正是社會對白化症者的歧視所致。除了提供受訪者發聲的管道，吳乙峰隱藏片頭、片尾兩位女性的影像，也展現他對受訪者的關懷與尊重。

== 貢獻 ==
由於片長不符合當時電視台所訂的規格，無法在電視頻道播放，吳乙峰遂採取全台巡迴放映的模式，在大學、社區尋找放映地點，並透過映後座談和觀眾面對面深度互動，結果影片在全台放映的次數超過400場，場場引發熱烈的討論，「吳乙峰旋風」席捲全台（李道明採訪200a:296-7;李泳泉2006：60-1）。全台巡迴放映也成為紀錄片新型態的映演模式<ref>|author=邱貴芬 |date=2006.01|title=「看見台灣」：台灣新紀錄片研究，頁28</ref>。

《月亮的小孩》打破以往觀眾對紀錄片枯燥且以說明式影片的印象，而是像劇情片般有劇情進展的、賺人熱淚且感動人心的。<ref>[http://lfbxmljy.blog.sohu.com/178487067.html 全景學派的誕生：台灣紀錄片的趨勢觀察（1990-2000） 李泳泉]</ref>。吳乙峰開創紀錄片的新形式，由於導演長期蹲點，觀眾可以在紀錄片中看見導演與被拍攝者像朋友般的互動，雙方的關係是平等的，改變了以往被拍攝者在鏡頭前被「剝削」的角色。被拍攝者可以透過鏡頭發聲，而不是透過影片「替他發聲」，展現高度的紀錄倫理自覺。<ref>|author=邱貴芬 |date=2006.01|title=「看見台灣」：台灣新紀錄片研究，頁26-27</ref>。

井迎瑞認為：《月亮的小孩》不僅是開啟了紀錄片以弱勢族群作為再現對象的趨勢，同時也實踐了一個不同於旁白解說，長期蹲點、參與觀察、生活紀實，被視之為真實電影的再現模式。
{{cquote|《月亮的小孩》在美學形式，態度內容和製作方法等三方面都有著突破性的意義。─井迎瑞1990<ref>http://ccs.nccu.edu.tw/word/HISTORY_PAPER_FILES/1361_1.pdf 再現弱勢的美學與政治：從《月亮的小孩》到《水蜜桃阿嬤》</ref>}}

== 全景傳播基金會 ==
「財團法人全景傳播基金會」成立於1996年，其前身為「全景映像工作室」，由吳乙峰、李中旺、許富進、陳雅芳等人於1988年創立。<ref>[http://www.tidf.org.tw/zh-hant/news/21472 創作回到人民生活的現場　全景傳播基金會獲頒TIDF傑出貢獻獎]</ref>

解嚴前後，台灣社會運動蓬勃，包含政治運動、農民運動、婦女運動、老兵運動與原住民運動等類型多元。因此，紀錄片工作者多扮演挑戰主流媒體真實性、號召群眾反對政
府的公民力量，其中以拍攝[[桃園機場事件|桃園機場事件]]一戰成名的的[[綠色小組|綠色小組]](1986-1990)為代表。1990年起由於前總統[[李登輝|李登輝]]推動[[社區總體營造|社區總體營造]]（在地認同），紀錄片工作者在1990年後不再是站在政府的對立面，而是挖掘相較於官方的「另類」社會景況，所以1990年由吳乙峰主導的全景學派蔚為風潮。

《月亮的小孩》在全台大學院校、社會團體舉辦超過四百場的放映座談，每次映後座談時往往引起觀眾熱烈的討論。全景映像工作室有計畫地留下觀眾的姓名和聯絡資料，成
立「全景聯誼會」，一萬多名的成員成為全景最忠實的觀眾，也使得全景儼然成為一個學派。

全景對於推廣紀錄片不遺餘力，他們於1994年出資舉辦紀錄片人才培育計畫，1995年在行政院文化建設委員會(今[[中華民國文化部|中華民國文化部]])的支持下，於台灣北、中、南各地辦理「地方記錄攝影工作者訓練計畫」課程，培育將近60位來自各行各業的在地工作者投入各種主題文化的紀錄片拍攝，學員包含長期投入淡水歷史研究的[http://tamsui.dils.tku.edu.tw/wiki/index.php/%E5%BC%B5%E5%BB%BA%E9%9A%86 張建隆]、記錄老眷村的外省二代李紀平、蘭嶼護士張淑蘭、歷史老師簡史朗、[http://docworker.blogspot.tw/2009/02/blog-post_26.html 吳耀東]、[[楊力州|楊力州]]以及原住民導演[http://pilin-yapu.blogspot.tw/ 比令‧亞布]等人都是台灣頗具知名度的紀錄片工作者。<ref>[http://www.tidf.org.tw/zh-hant/news/21472 創作回到人民生活的現場　全景傳播基金會獲頒TIDF傑出貢獻獎]</ref>

全景學派作品的特色強調捕捉「感動」人心的人事物，紀錄的對象多為弱勢族群，對被拍攝者相當尊重，因此強調長期蹲點跟拍，甚至在影片完成後會先讓他們觀看是否有需要修改之處，與被拍攝者的關係不再是鏡頭暴力，而是共同作者的概念。旁白也不像以往的紀錄片中說明式的標準[[中文|中文]]，而是吳乙峰本人的台灣國語，觀眾也可以聽見導演與受訪者之間的對話。
