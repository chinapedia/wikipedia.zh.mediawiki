'''月坛体育中心'''，位于[[北京市|北京市]][[西城区|西城区]][[月坛南街|月坛南街]]甲1号，是西城区区属体育场馆。

==简介==
月坛体育中心始建于1970年代末。在此之前，其所在地为房修公司办公楼、招待所、职工宿舍。1980年代修建了篮、排球馆和游泳馆。为承担[[1990年亚运会|1990年北京亚运会]]柔道比赛，1989年建成了可容纳3000人的月坛体育馆（今月坛体育中心综合馆）。<ref name=lvzhou/>

1980年代起，中国各地的体育设施开始“市场化”，1990年代以来又纷纷承接电影放映、音乐会等娱乐活动。1996年，月坛体育中心划出2000平方米面积开设滚轴溜冰俱乐部，可容1000人，成为当时国内最大的滚轴溜冰场。同时，月坛体育中心还成立了天行健身会，成为国内大众健身操的发源地。后来，月坛体育中心又开办了北京银轩老年健身中心、儿童游泳训练班，承办各类运动会、大型会议开幕式，开设文化晚宴剧场，承接娱乐演出。<ref name=lvzhou/>

2005年3月，月坛体育中心被[[北京奥组委|北京奥组委]]选定为[[2008年北京奥运会|2008年北京奥运会]]手球和羽毛球训练馆。为此，月坛体育中心开始全面修缮。修缮完成后，月坛体育中心不再举办滚轴溜冰活动，成为“国家高水平体育后备人才基地”和公众健身场所，为西城区运动学校提供训练场地，向公众开放乒乓球、羽毛球、篮球、田径、游泳馆等设施。<ref name=lvzhou/>

到2008年，月坛体育中心包括“四馆一场”：月坛综合馆、月坛综合训练馆、月坛训练馆、月坛游泳馆、月坛体育场。附属设施包括：[[射击|射击]]、[[乒乓球|乒乓球]]、[[柔道|柔道]]、[[跆拳道|跆拳道]]、[[游泳|游泳]]、[[篮球|篮球]]、[[网球|网球]]、[[排球|排球]]、[[举重|举重]]等体育训练场地。[[羽毛球|羽毛球]]、[[手球|手球]]的训练场地分别设在体育馆和综合训练馆。<ref name=lvzhou>{{Cite web |url = http://news.sohu.com/20080809/n258714198.shtml |title =【月坛体育中心】寸土寸金的“绿洲”  |author =  |date = 2008-08-09 |publisher = 搜狐 |language =  |accessdate =  |quote =  }}</ref>

2012年，为配合[[北京金融街|北京金融街]]西扩工程，体育场南侧建于2004年的月坛体育中心综合训练馆拆除，将在今后几年内原址还建。<ref>{{Cite web |url = http://house.ifeng.com/detail/2012_06_19/25415060_0.shtml |title = 月坛体育中心部分开拆 |author =  |date =2012-06-19  |publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref>

“十三五”期间，西城区规划建设月坛体育中心滑冰场，面积约1800平方米，冬季为冰场，春夏秋季冰场恢复为田径场，计划2018年开工，2021年建成投入使用。<ref>{{Cite web |url =http://news.ifeng.com/a/20161219/50439897_0.shtml  |title = 西城区月坛体育中心将升级改造成滑冰场 |author =  |date = 2016-12-19 |publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://bj.people.com.cn/n2/2016/1220/c82838-29485756.html |title = 西城计划腾退47处文保院落 为北京最大规模文保腾退 |author =  |date = 2016-12-20 |publisher = 人民网 |language =  |accessdate =  |quote =  }}</ref>
