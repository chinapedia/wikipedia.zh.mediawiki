'''月球车'''，又称'''月面车'''，[[中国航天工程|中国航天工程]]全称为'''月面巡视探测器'''，是在月球表面行驶并对[[月球|月球]]考察和收集分析样品的专用车辆。可分为无人驾驶月球车和有人驾驶月球车。无人驾驶月球车由轮式底盘和仪器舱组成，用太阳能电池和蓄电池联合供电。无人驾驶月球车其实并不是一辆车，而是小型化、低功耗、高集成的部分或者全部自主的[[机器人|机器人]]。世界上第一台无人驾驶的月球车是[[苏联|苏联]]的[[月球车1号|月球车1号]]，于1970年11月17日由[[月球17号|月球17号]]探测器送上月球。

有人驾驶月球车主要由月球车的每个轮子的各一台电动机驱动，靠蓄电池提供动力。主要作用于扩大[[宇航员|宇航员]]的活动范围和的减少体力消耗，它可随时存放宇航员采集的岩石和土壤[[标本|标本]]。

目前共有三个国家成功完成月球车在月球表面的降落和行走，分别是[[前苏联|前苏联]]，[[美国|美国]]和[[中华人民共和国|中国]]。

==已经完成的任务==
===月球车1号===
{{main|月球车1号}}
月球车1号是[[苏联|苏联]]发射成功的世界上第一辆成功运行的遥控[[月球车|月球车]]。质量约为756千克，高1[[米|米]]35，长2.2米，宽1.6米。<ref>[http://why.eastday.com/q/20070401/u1a277823.html "上海版"月球车首度亮相]</ref> 依靠4对电驱动，电磁[[继电器|继电器]]制动的轮子实现机动。于1970年11月17日由[[月球17号|月球17号]]送上[[月球|月球]]（降落点位于北纬38°17'西经35°），一直在[[雨海|雨海]]地区工作至1971年10月4日。

===月球车2号===
{{main|月球车2号}}
月球车2号是[[苏联|苏联]]研制的第二个无人月球车，高1.35[[米|米]]，长1.7米，宽1.6米。主要任务同月球车1号相同，也是收集月球表面照片，全车拥有3个摄像头。除摄像头外，月球车2号还拥有[[激光|激光]]测距、[[X射线|X射线]]探测仪、[[磁场|磁场]]探测仪等装置。它以八个相互独立的电动车轮驱动，车体能源来自于[[太阳能电池|太阳能电池]]，车上携带的[[钋|钋]]210放射性元素用来在夜晚为车体供热，保证仪器不因低温而损坏。月球车2号总共工作了4个月，拍摄了86张全景照片和80000张照片。

===阿波罗月球车===
{{main|阿波罗月球车}}
阿波罗月球车是1971至1972年期间，[[阿波罗计划|阿波罗计划]]中第[[阿波罗15号|十五]]、[[阿波罗16号|十六]]和[[阿波罗17号|十七]]次登月任务中，宇航员使用的月球车。可以搭载两名宇航员，以及[[月岩|月岩]]和登月设备。

===玉兔号月球车===
{{main|玉兔号月球车}}
玉兔号月球车是[[中华人民共和国|中华人民共和国]]设计制造的一辆[[月球车|月球车]]，搭载于[[嫦娥三号|嫦娥三号]]月球探测器。[[嫦娥三号|嫦娥三号]]于2013年12月2日凌晨1时30分00秒34毫秒从[[西昌卫星发射中心|西昌卫星发射中心]]由[[长征三号乙运载火箭|长征三号乙增强型运载火箭]]发射<ref name="launch">[http://news.163.com/13/1202/02/9F2BG1N900014JB5.html 嫦娥三号发射全记录]</ref>，于12月14日21時12分成功软着陆于月球表面<ref name="xinhua01">{{cite web |url = http://news.xinhuanet.com/world/2013-12/13/c_118552981.htm |title = 走近“玉兔号”——解密我国首个月球车 |publisher = 新华网 |date = 2013年12月13日}}</ref>，12月15日凌晨4时35分<ref>[http://news.ifeng.com/mainland/special/changesanhao/content-3/detail_2013_12/15/32134182_0.shtml  玉兔号踏上月球表面 六轮着地(图)]</ref>，玉兔号月球车从[[嫦娥三号|嫦娥三号]]中走出，成为自1973年[[苏联|苏联]]的[[月球车2号|月球车2号]]以来再次踏上月球表面的无人驾驶月球车<ref name="BBC">{{cite web |url = http://www.bbc.co.uk/zhongwen/trad/china/2013/12/131214_china_moon.shtml |title = 「嫦娥三號」在月球成功軟著陸 |publisher = BBC |date = 2013年12月14日}}</ref>。

==正在执行的任务==

=== 玉兔二号月球车 ===
{{main|玉兔二号月球车}}
玉兔二号月球车是[[中华人民共和国|中华人民共和国]]设计制造的一辆[[月球车|月球车]]，搭载于[[嫦娥四号|嫦娥四号]]月球探测器，于2019年1月3日定名<ref>{{Cite web|url=http://www.cnsa.gov.cn/n6758823/n6758838/c6805056/content.html|title=揭晓了：嫦娥四号月球车命名“玉兔二号”-国家航天局|accessdate=2019-01-04|work=www.cnsa.gov.cn}}</ref>。[[嫦娥四号|嫦娥四号]]于2018年12月8日凌晨2时23分34秒366毫秒从[[西昌卫星发射中心|西昌卫星发射中心]]由[[长征三号乙运载火箭|长征三号乙改进Ⅲ型运载火箭]]发射，于2019年1月3日10時26分成功软着陆于[[月球背面|月球背面]][[冯·卡门撞击坑|冯·卡门撞击坑]]<ref>{{Cite web|url=http://www.cnsa.gov.cn/n6758823/n6758838/c6805034/content.html|title=嫦娥四号探测器成功着陆月球背面 传回世界第一张近距离拍摄月背影像图-国家航天局|accessdate=2019-01-04|work=www.cnsa.gov.cn}}</ref>。1月3日22时22分，玉兔二号月球车从[[嫦娥四号|嫦娥四号]]着陆器完成分离，成功踏上月面<ref>{{Cite web|url=http://www.cnsa.gov.cn/n6759533/c6805052/content.html|title=嫦娥四号着陆器与巡视器顺利分离　玉兔二号在月背留下人类探测器的第一道印迹-国家航天局|accessdate=2019-01-04|work=www.cnsa.gov.cn}}</ref>。
