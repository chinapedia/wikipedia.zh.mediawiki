'''月城街'''又名'''东风巷'''，位于[[中国|中国]][[江苏省|江苏省]][[无锡|无锡]][[宜兴市|宜兴市]][[宜城街道|宜城街道]]古城东北侧，为宜兴三大历史文化街区之一。此处原为宜兴东门百渎门外的[[月城|月城]]北半部分，南邻解放中路，西靠段家巷，东、北临东仓河（原东护城河的一部分，河上有[[东仓桥|东仓桥]]），自西向东由扁担巷（原主城墙墙基）、月城街和尤斯巷等组成<ref>{{cite web|url=http://www.yxghj.gov.cn/readnews.asp?id=962|title=《宜兴市历史文化名城保护规划》|accessdate=2013-12-01|work=|publisher=宜兴市规划局|deadurl=yes|archiveurl=https://web.archive.org/web/20131203044252/http://www.yxghj.gov.cn/readnews.asp?id=962|archivedate=2013-12-03}}</ref>，现街巷及道路结构尚保持明清格局，内部建筑多建于明清至民国时，其中主街两侧多为商住合一的两层楼房，上为住家，下为店铺，房屋之间由河巷夹弄相连，小夹弄通河埠和大路<ref>{{cite web |url=http://unn.people.com.cn/GB/22220/149429/159151/11712324.html|title=《宜兴三街区确定为历史文化街区》|accessdate=2013-12-01|work=|publisher=人民网}}</ref>。根据宜兴市月城街历史街区保护规划方案，此处将在整修后作为传统居住、商业和文化休闲综合区以及城防系统与传统民居展示区<ref>{{cite web |url=http://news.gmw.cn/newspaper/2013-11/11/content_2405397.htm|title=《宜兴三大历史文化街区保护规划通过省级论证》|accessdate=2013-12-01|work=|publisher=人民网}}</ref>。
