'''眉月湾'''([[拉丁语|拉丁语]]为"Sinus Lunicus")是[[月球正面|月球正面]]位于[[雨海|雨海]]东南边缘的一处[[月海|月海]]。它的西南、东南和东北三面分别坐落着[[阿基米德陨石坑|阿基米德陨石坑]]、[[奥托里库斯陨石坑|奥托里库斯陨石坑]]和[[阿里斯基尔环形山|阿里斯基尔环形山]]、西北方畅开，面向一列小山丘-[[斯匹次卑尔根山脉|斯匹次卑尔根山脉]]。

1959年9月14日，前[[苏联|苏联]][[月球2号|露娜2号]]探测器降落在[[阿基米德坑|阿基米德环形山]]和[[奥托里库斯陨石坑|奥托里库斯陨石坑]]之间的裂口处。为纪念人类太空探测器的首次抵达月球的登陆点<ref name="data_LSI">{{cite web |url=http://www.lunarrepublic.com/atlas/sections/b3.shtml |title=The Full Moon Atlas |publisher=Luna Society International |accessdate=2010-11-03 |archiveurl=https://www.webcitation.org/68rRFNEFz?url=http://www.lunarrepublic.com/atlas/sections/b3.shtml |archivedate=2012-07-02 |deadurl=yes }}</ref><ref name="data_austronaut">{{cite web |url=http://www.astronaut.ru/luna/ussr_a.htm?reload_coolmenus |title=Lunar probes USSR |publisher=Space Encyclopedia ASTROnote |accessdate=2010-11-03 |archiveurl=https://www.webcitation.org/68rVqEQsJ?url=http://www.astronaut.ru/luna/ussr_a.htm?reload_coolmenus |archivedate=2012-07-02 |deadurl=yes }}</ref><ref name="data_NASA">{{cite web |url=http://nssdc.gsfc.nasa.gov/planetary/lunar/moon_landing_map.jpg |title=Moon Landing Map |publisher=NASA |accessdate=2010-11-03 |archiveurl=https://www.webcitation.org/68rVquSjK?url=http://nssdc.gsfc.nasa.gov/planetary/lunar/moon_landing_map.jpg |archivedate=2012-07-02 |deadurl=yes }}</ref>，1970年[[国际天文学联合会|国际天文学联合会]]将该月湾命名为"眉月湾"(Bay of Lunik)。

眉月湾的正式[[月面座标|月面座标]]为北纬31.8°、西经1.4°({{Coord|31.8|N|1.4|W|globe:Moon}})，直径126公里<ref name="data_IAU">{{cite web |url=http://planetarynames.wr.usgs.gov/Feature/5566 |title=Sinus Lunicus |work=Gazetteer of Planetary Nomenclature |publisher=International Astronomical Union (IAU) Working Group for Planetary System Nomenclature (WGPSN) |accessdate=2014-08-06 |archiveurl=https://www.webcitation.org/68rRECxsv?url=http://planetarynames.wr.usgs.gov/Feature/5566?__fsk=-1705927980 |archivedate=2012-07-02 |deadurl=no }}</ref>。[[月湾|月湾]]中最醒目的特征是由阿里斯基尔环形山和奥托里库斯陨石坑的溅射物混合而成的外侧垒壁，以及小卫星坑"[[阿基米德坑|阿基米德 C]]"和"[[阿基米德坑|阿基米德 D]]"。月湾表面[[反照率|反照率]]也因覆盖着这两座陨坑的[[射纹系统|射纹材料]]而变亮。
