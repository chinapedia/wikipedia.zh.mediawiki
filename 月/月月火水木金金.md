{{japanese|japanese=月月火水木金金|kana=げつげつかすいもくきんきん|romaji=GetsuGetsuKaSuiMokuKinKin}}  
《'''月月火水木金金'''》是日本的一首[[海軍|海軍]]軍歌，在日文中一週是以[[七曜|七曜]]來表示，也就是[[日曜日|日]]、[[月曜日|月]]、[[火曜日|火]]、[[水曜日|水]]、[[木曜日|木]]、[[金曜日|金]]、[[土曜日|土]]。這首軍歌歌名的意義代表著一週七天並沒有所謂的[[星期六|星期六]]和[[星期日|星期日]]，慣用表現為「星期六、日也要工作」（{{lang|ja|土日返上で働く}}）的意思。

以下主要說明的是[[第二次世界大戰|第二次世界大戰]]之前的日本軍歌《月月火水木金金》。

== 由來 ==
*最初由[[大日本帝國海軍|大日本帝國海軍]]開始使用。大日本帝國海軍於[[日俄戰爭|日俄戰爭]]勝利後，因「束緊勝利之兜的系帶（{{lang|ja|勝って兜の緒を締めよ}}）」在休日中仍然進行猛烈操練，有部分海軍[[士官|士官]]向同僚說出「這樣不就是星期一一二三四五五嗎（{{lang|ja|これでは、まるで月月火水木金金じゃないか}}）」。這句話很就快在海軍中廣為流傳，該語氣看來就令「月月火水木金金」流露著海軍精神，而一般來說其意味是在猛烈操練下想退縮的海軍官兵自嘲之開端。 
*於[[第二次世界大戰|第二次世界大戰]]中，本曲有讚美勤務的意味，因此在日本民間廣泛地使用。

== 軍歌 ==
*根據上述所記敘成為日本的[[軍歌|軍歌]]。作詞者為每日在海上艦隊勤務工作且像其他日本[[軍艦|軍艦]]上的男兒勇渡[[太平洋|太平洋]]的[[高橋俊策|高橋俊策]]，而[[作曲|作曲]]為海軍[[軍樂隊|軍樂隊]]出身的江口源吾（[[江口夜詩|江口夜詩]]）。
*該曲在1940年（昭和15年）11月，作為[[海軍省|海軍省]]海軍軍事普及部推薦曲由[[內田榮一|內田榮一]]主唱並由[[Polydor_Records|Polydor Records]]發售，但據說當時並不暢銷。但是，由於[[日本放送協會|日本放送協會]]職員偶然錯誤播放成為契機，令本曲瞬間成為[[流行歌|流行歌]]並在日本民間廣為認識而成為逸話。
*[[若山彰|若山彰]]、[[伊藤久男|伊藤久男]]亦有演奏過本曲。別名「艦隊勤務」（在[[JASRAC|JASRAC]]數據庫中，「月月火水木金金」為正式題名，而「艦隊勤務」則是副題）。
*現在為[[海上自衛隊|海上自衛隊]]所演奏的樂曲之一。
*為[[富士電視系|富士電視系]][[綜藝節目|綜藝節目]]「{{lang|ja|[[ドリフ大爆笑|ドリフ大爆笑]]}}」主題「{{lang|ja|[[トントントンカラリンと隣組|トントントンカラリンと隣組]]}}」的原譜換詞歌中有名，但在初期亦是使用該曲的原譜換詞。再者，為[[The_Drifters|The Drifters]]（{{lang|ja|ザ・ドリフターズ}}）擅長的軍歌，並也唱著該曲。

== 其他 ==
*在[[波麗佳音|波麗佳音]]發售的鴨子圓舞曲中（作詞：[[横尾嘉信|横尾嘉信]]／作曲：[[横尾嘉信|横尾嘉信]]／歌：{{lang|ja|[[マユミーヌ|マユミーヌ]]}}），雖然歌詞含有「月月火水木金土」，但與此曲的表演方式有不少差異。
