==分佈及生境==
原產於[[亞洲|亞洲]]南部，現分佈於[[台灣|台灣]]、[[中國大陸|中國大陸]]、[[琉球|琉球]]、[[菲律賓|菲律賓]]、[[印尼|印尼]]、[[馬來西亞|馬來西亞]]、[[斯里蘭卡|斯里蘭卡]]、[[印度|印度]]、[[澳洲|澳洲]]等地，[[中國|中國]]國內分佈於[[海南|海南]]、[[廣東|廣東]]、[[福建|福建]]及[[廣西|廣西]]、[[貴州|貴州]]、[[雲南|雲南]]、[[湖南|湖南]]四省區的南部，多長於[[海拔|海拔]]高的山地疏林或密林中或低地的[[丘陵地|丘陵地]]之中，喜溫暖較濕潤的[[氣候|氣候]]，不耐寒，生長適合溫度介孚於[[攝氏|攝氏]]20－28度，常見於[[石灰岩|石灰岩]]地區，[[花崗岩|花崗岩]]地區亦有分佈。亦有人工栽培作綠籬之用，主要以[[扞插|扞插]]、[[播種|播種]]及[[高空壓條法|高空壓條法]]繁殖，扞插法苗多，但成株慢，播種法則可快速獲大量苗木，故一般大多選用播種法。主要蟲害為[[蚜蟲|蚜蟲]]引致的[[煤煙病|煤煙病]]、[[星天牛|星天牛]]幼蟲侵害基部及[[粉蚧|粉蚧]]<ref name = "臺北花博"/><ref name = "中國植物志"/><ref name = "台灣常用藥用植物圖鑑"/><ref name = "浸會大學"/><ref name = "香花植物"/><ref name = "香港盆栽">{{cite book|title=《香港盆栽》|author= |series=綠川晴|publisher=萬里機構|page=第216-217頁|isbn= 978-962-14-3781-5|date=2008年8月}}</ref>。

==形態特徵==
月橘是一種常綠[[灌木|灌木]]或小[[喬木|喬木]]植物，高約1－10[[米_(单位)|米]]，高可達12米。樹幹及小枝淡黃灰或灰白色，略有光澤，無毛，分枝細長且多，小枝圓柱形，橫切面呈鈍三角形，底邊近圓弧形，木質堅硬。葉為[[奇數|奇數]][[羽狀複葉|羽狀複葉]]，互生，常為3－5片，稀有7片，幼時為[[單葉|單葉]]，[[葉軸|葉軸]]不具翅；小葉3－9片，常為5－7片，卵形、卵狀披針形至近菱形，互生，頂部狹長漸尖、尾狀漸尖，有時短尖或稍凹，基部近圓至寬楔形，變異極大，兩側對稱或一側偏斜，紙質或厚紙質，表面深綠色，有光澤，背面青綠色，密生腺點，腺點乾後呈褐黑色，全緣，呈波浪狀起伏，中脈凸出，沿中脈披微柔毛或近無毛，側脈每邊4－8條，無毛，長約2－9厘米，寬約1－4厘米；小葉近無柄，披微柔毛，長不足1厘米。[[花|花]]為聚傘[[花序|花序]]，頂生或生於上部葉腋內，通常花為10朵以內，有時多達50餘朵，有時為單花，花白色，星狀，極芳香，直徑約3－4厘米；[[花序軸|花序軸]]披微柔毛；[[花梗|花梗]]細長，長約6－10毫米；[[萼片|萼片]]4－5枚，卵狀三角形，宿存，邊緣有疏毛，具油點，長約1－2毫米；[[花瓣|花瓣]]4－5枚，狹長橢圓形或倒披針形，白色或淺黃色，盛花時稍反折，散生淡黃色半透明腺點，披微柔毛，長約2－2.5厘米，寬約2－5毫米；[[雄蕊|雄蕊]]8－10枚，長短相間，長的長約10－12毫米，短的長約6－7毫米；[[花絲|花絲]]線狀，上部漸細而頂端尖，下部扁而較寬，比[[花柱|花柱]]略短，無毛；[[花藥|花藥]]卵形或橢圓形，長不及1毫米；[[雌蕊|雌蕊]][[子房|子房]]上位，長筒形，2室，每室有1－2[[胚珠|胚珠]]，2胚珠時為上下連生；[[花柱|花柱]]棒狀，綠色，連子房長約12毫米；[[柱頭|柱頭]]比子房寬或等寬。[[果|果]]為[[漿果|漿果]]，卵形或橢圓形，先端尖銳，成熟時由綠色轉變為橙黃至朱紅色，有甚多乾後凸起但中央窩點狀下陷的油點，長約1－2厘米，寬5－14毫米，大小變化大；[[種子|種子]]1－2顆；[[種皮|種皮]]具棉質毛<ref name = "中國植物志"/><ref name = "有益花木圖鑒"/><ref name = "台灣行道樹圖鑑"/><ref name = "增訂生草藥性備要"/><ref name = "台灣常用藥用植物圖鑑"/><ref name = "惜樹靈人"/><ref name = "浸會大學"/><ref name = "白雲山植物網"/><ref name = "河北植物志"/><ref name = "香花植物"/><ref name = "海南植物志"/><ref name = "香港盆栽"/><ref>[http://www.cvh.org.cn/difangzhi/guizhou/page/2/276.pdf  貴州植物誌]{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}第276-278頁</ref><ref name = "南方藥用植物圖鑒 ">{{cite book|title=《中國藥用植物圖鑒系列之一 南方藥用植物圖鑒》|author= |series=王玉生、蔡岳文|publisher=汕頭大學出版社|page=第118頁|isbn= 978-7-81036-859-9 |date=2007年7月}}</ref>。

==醫藥用途==
{{medical}}
月橘以葉和帶葉嫩枝入藥，味辛、微苦，性溫，有小毒，歸心、肝、肺經，陰虛火亢者忌用，[[藥材|藥材]]主產於廣西、廣東、福建等地。[[中藥|中藥]]名為九里香，藥用之名始載於《[[嶺南採藥錄|嶺南採藥錄]]》，具祛風除濕、行氣活血、散瘀止痛、解毒等功效，主治胃痛、脘腹氣痛、風濕痹痛、牙痛、疥癬、腫毒、皮膚瘙癢、跌打腫痛、蟲蛇咬傷等。根莖煎劑有抗生育、局部麻醉及解痙等功效，能治平喘、老年慢性支氣管炎等。花果有散瘀止痛、行氣活血、解毒消腫等功效，能治風濕痹痛、胃脘疼痛、跌打損傷、蛇蟲咬傷等。多種[[九里香屬|九里香屬]]植物自古以來已使用九里香之名作入藥之用，而本種與同屬的[[九里香|九里香]]（''Murraya exotica ''）同載錄於《[[中國藥典|中國藥典]]》2005年版，同為[[中藥|中藥]]九里香的法定原植物來源種之一<ref name = "增訂生草藥性備要"/><ref name = "台灣常用藥用植物圖鑑"/><ref name = "浸會大學"/><ref name = "南方藥用植物圖鑒 "/><ref name = "當代藥用植物典4">{{cite book|title=《當代藥用植物典 4》|author= |series=蕭培根|publisher=香港賽馬會中藥研究院|page=第330頁|isbn=978-988-99226-3-4|date=2007年8月}}</ref>。

==同種異名==
*[[九里香|九里香]]（''Murraya exotica''），同為芸香科九里香屬植物，分別在於九里香的自然分佈於離海岸不遠處的砂壤土地，而月橘則分佈於海拔高的山地疏林或密林中或低地的丘陵地之中；九里香喜陽光，耐乾熱，而月橘立地則喜較濕潤及頗耐陰。九里香的花葉均較月橘小，九里香的小葉最寬處為葉片的中部以上，頂端鈍，有時近圓或急尖，長約3厘米，寬約1.5厘米；月橘的小葉最寬處為葉片的中部或中部以下，頂端漸尖或短尖，長約2－8厘米。九里香的成熟果實長與寬約略相等，頂端鈍或急尖；月橘的成熟果實長遠過於寬，多為狹長橢圓形，頂端錐尖<ref name = "中國植物志"/><ref name = "常用中藥材鑑別圖典">{{cite book|title=《常用中藥材鑑別圖典》|author= |series=趙中振、陳虎彪|publisher=萬里機構|page=第250頁|isbn= 978-962-14-4150-8|date=2010年1月}}</ref>。
