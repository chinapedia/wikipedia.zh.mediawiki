'''月讀'''（{{jpn|j = 月読|hg = つきよみ|rm = Tukiyomi|kyu = 月讀|hgt = つきよ<sup>[[上代特殊仮名遣|乙]]</sup>み<sup>[[上代特殊仮名遣|甲]]</sup>}}）是[[日本神話|日本神話]]中[[月神|月神]]。

== 名稱 ==
神名中的「讀（{{lang|ja|よみ}}）」意謂依照[[曆法|曆法]]過生活，由於古人依[[月亮|月亮]]的陰晴圓缺制定曆法，演變成日常生活使用的[[陰陽曆#日本陰陽曆|舊曆]]，故「月讀」具有「數日子過生活」的含意。不過，也有人認為「{{lang|ja|み}}」字意為「神靈」，故「月讀」其實是「月之神」，其他例如[[綿津見|海神]]的「{{lang|ja|わだつみ}}」、[[大山祇神|山祇]]的「{{lang|ja|やまつみ}}」亦可作此解。

'''[[月神|月神]]'''（{{jpn|j = 月神|hg = つき の かみ|rm = Tuki no kami|kyu = |hgt = }}），一般稱作'''月讀尊'''，是[[日本神話|日本神話]]中的[[神_(神道)|神祇]]。[[三貴子|三貴子]]之一。

=== 名稱 ===
* [[名諱|諱]]：'''月弓尊'''（{{jpn|j = 月弓尊|hg = つきゆみ の みこと|rm = Tukiyumi no mikoto|kyu = |hgt = つきゆみ<sup>[[上代特殊仮名遣|甲]]</sup> の みこと}}） - {{link-ja|弦月}}也。此月如[[弓|弓]]，故稱此名。
* [[謚|謚]]：'''月夜見尊'''（{{jpn|j = 月夜見尊|hg = つきよみ の みこと|rm = Tukiyomi no mikoto|kyu = |hgt = つきよ<sup>[[上代特殊仮名遣|甲]]</sup>み<sup>[[上代特殊仮名遣|甲]]</sup> の みこと}}） - [[滿月|望月]]也。
* 謚：'''月讀尊'''（{{jpn|j = 月読尊|hg = つきよみ の みこと|rm = Tukiyomi no mikoto|kyu = 月讀尊|hgt = つきよ<sup>[[上代特殊仮名遣|乙]]</sup>み<sup>[[上代特殊仮名遣|甲]]</sup> の みこと}}，月讀命） - [[晦日|晦月]]・[[新月|朔月]]也。此日月亮無光，讀日數知晦，故稱此名。

=== 事績 ===
==== 日本書紀之記載 ====
依照《[[日本書紀|日本書紀]]》卷第一[[伊奘諾尊|伊奘諾尊]]和[[伊奘冉尊|伊奘冉尊]]產神之故事裡，在[[大山津見|山]]、[[速秋津日子神與速秋津比賣神|川]]、[[綿津見|海]]出生後，伊奘諾尊和伊奘冉尊產下了木之精[[句句廼馳|句句廼馳]]、草之精[[草野姬|草野姬]]（亦名野槌）。於是二神共議：「吾已生[[八島_(消歧義)|大八洲國]]及山川草木。何不生天下之主者歟？」後來共生日神[[大日孁貴|大日孁貴]]（亦名天照大神、天照大日孁尊），此子全身光華明彩、照亮六合之內。二神高興地說：「吾息雖多，未有若此靈異之兒。不宜久留此國。自當早送于天而授以天上之事。」彼時天與地相去不遠，故以[[天柱|天柱]]舉於天上。接著生下'''月神'''（又名'''月弓尊'''、'''月夜見尊'''、'''月讀尊'''），其光彩次於日神，故可配合日神而治。然後又生[[蛭兒|蛭兒]]，雖三歲卻不良於行，故載之於{{link-ja|天磐櫲樟船}}，而順風放棄；最後再生[[素戔嗚尊|素戔嗚尊]]（又名神素戔嗚尊、速素戔嗚尊），此神雖勇猛安忍，卻時常哭泣，令國內人民多以夭折、青山枯黃，故伊奘諾尊和伊奘冉尊向素戔嗚尊說：「汝甚無道。不可以君臨宇宙。固當遠適之於[[根國|根國]]矣！」遂驅逐之。

接著第一種說法記載伊奘諾尊曰：「吾欲生御㝢之珍子。」祂以左手持白銅鏡化出之神謂[[大日孁尊|大日孁尊]]，以右手持白銅鏡化出之神為'''月弓尊'''，又廻首顧眄之間化出之神是謂[[素戔嗚尊|素戔嗚尊]]。大日孁尊及月弓尊二者質性明麗，故使之照臨天地；素戔嗚尊性好殘害，故命令祂下去治理根國。

第六種說法記載從[[黃泉|黃泉]]逃回的伊奘諾尊在[[筑紫國|筑紫]][[日向國|日向]]小戶橘之檍原（今[[宮崎縣|宮崎縣]][[宮崎市|宮崎市]]阿波岐原町）進行禊祓（袪除污穢的淨化儀式），祂清洗左[[眼|眼]]生下[[天照大神|天照大神]]、復洗右眼生下'''月讀尊'''、再洗[[鼻|鼻]]生下[[素戔嗚尊|素戔嗚尊]]。於是勅令三子曰：「天照大神者，可以治[[高天原|高天原]]也。月讀尊者，可以治滄海原潮之八百重也。素戔嗚尊者，可以治天下也。」但是年事已高且生八握[[鬍鬚|鬚髯]]，祂不管天下卻終日哭啼。伊奘諾尊問祂說：「汝何故恆啼如此耶？」答曰：「吾欲從母於根國，只爲泣耳。」前者於是怒曰：「可以任情行矣！」遂逐之。

第十一種說法記錄伊奘諾尊勅令三子曰：「天照大神者，可以御[[高天原|高天之原]]也。'''月夜見尊'''者，可以配日而知天上之事也。素戔嗚尊者，可以御滄海之原也。」[[天照大神|天照大神]]在天上聽聞[[葦原中國|葦原中國]]有[[保食神|保食神]]，於是命令'''月夜見尊'''下降拜訪之。保食神乃回頭向著葦原中國，嘴裡吐出米飯；面向[[海|海]]則嘴裡吐出[[魚|魚]]；面向[[山|山]]則嘴裡吐出毛肉。一切食物具備，擺設宴席以宴請月夜見尊。後者忿然作色曰：「穢哉！鄙矣！寧可以口吐之物，敢養我乎！」於是拔劍殺死保食神，然後回去向天照大神稟告一切。天照大神聽完勃然大怒地說：「汝是惡神！不須相見。」所以天照大神和'''月夜見尊'''一日一夜，白天就和黑夜分開了。

==== 古事記之記載 ====
依照《[[古事記|古事記]]》上卷，自[[黃泉|黃泉國]]逃回的[[伊奘諾尊|伊邪那岐命]]進行禊祓（袪除污穢的淨化儀式），祂脫去的衣物誕生了{{link-ja|衝立船戶神|衝立船戸神}}至[[邊津甲斐弁羅神|邊津甲斐弁羅神]]等共12位[[神祇|神祇]]。接著在中游淨滌身體，產生[[禍津日神|八十禍津日神]]（{{lang|ja|やそまがつひ の かみ}}）、[[禍津日神|大禍津日神]]（{{lang|ja|おほまがつひ の かみ}}）等二神，乃由亡者之汙穢所成之神明。為了消除前二神之禍，又生[[直毘神|神直毘神、大直毘神]]、[[伊豆能賣神|伊豆能賣神]]等三神。接著在水底洗滌時分別生成：{{link-ja|底津綿津見神}}、{{link-ja|底筒之男命}}、{{link-ja|中津綿津見神}}、{{link-ja|中筒之男命}}、{{link-ja|上津綿津見神}}、{{link-ja|上筒之男命}}等六神。此三位[[綿津見神|綿津見神]]乃{{link-ja|阿曇氏|阿曇氏|阿曇連}}等之祖神伊都久神，故阿曇連一族為綿津見神之子[[宇都志日金拆命|宇都志日金拆命]]的子孫。另三位筒之男命乃[[住吉三神|墨江之三前大神]]。接著伊邪那岐命清洗左目時生成[[天照大神|天照大御神]]、洗右目生成'''月讀命'''、洗[[鼻子|鼻子]]生成[[建速須佐之男命|建速須佐之男命]]。

此時伊邪那岐命非常歡喜，說：「吾者生生子而，於生終得[[三貴子|三貴子]]。」所以取下脖子上勾玉所串之項鍊，搖得琮琮作響，賜給天照大御神並曰：「汝命者所知[[高天原|高天原]]矣。」因此這串項鍊名謂御倉板舉神。次向'''月讀命'''說：「汝命者所知[[夜|夜]]之[[食國|食國]]矣。」，復向次詔建速須佐之男命說：「汝命者所知海原矣。」

==== 風土記之記載 ====
《山城國風土記》軼文記載，'''月讀尊'''受天照大神之敕降於[[葦原中國|豐葦原中國]]，拜訪[[保食神|保食神]]時，有一棵湯津桂樹，月讀尊乃倚靠著樹站著，故該地今號「桂里」。按[[月亮|月亮]]與[[桂樹|桂樹]]的關係，可追溯[[中國神話|中國神話]]的[[吳剛_(道教)|吳剛伐桂]]，輾轉傳入日本<ref>參考《日本妖怪大事典》，村上健司編著，[[角川書店|角川書店]]，2005年，ISBN 978-4-04-883926-6，頁95。</ref>。

=== 信仰 ===
日本境內主祭月讀尊的[[神社|神社]]除了各地之{{link-ja|月讀神社}}、{{link-ja|月山神社}}外，主要計有下列：
* '''{{link-ja|月讀宮}}'''（[[三重縣|三重縣]][[伊勢市|伊勢市]]）：[[皇大神宮|皇大神宮]]的別宮。《[[續日本紀|續日本紀]]》卷第卅二[[光仁天皇|光仁天皇]]紀提及，光仁天皇三年八月己酉朔甲寅有暴風雨吹倒樹木、房屋，於是[[占卜|卜卦]]，結果顯示應祭祀[[伊勢神宮|伊勢]]'''月讀神'''。所以光仁天皇下令每年九月獻祭[[馬|馬]]匹給[[荒魂與和魂|荒祭神]]，並入荒御玉命、伊佐奈伎命、伊佐奈美命等於官社，又將度會郡[[神宮寺|神宮寺]]遷徙至飯高郡度瀨山房。除此之外，《[[皇太神宮儀式帳|皇太神宮儀式帳]]》所記載「月讀宮一院」之祭神為「'''月讀命''' 御形馬乘男形，著紫御衣，金作[[大刀|大刀]]佩之。」。因為《[[記紀|記紀]]》二書從未提及月讀神之[[性別|性別]]，此書記錄的男性樣貌相當鮮明。
* {{link-ja|月夜見宮}}（三重縣伊勢市）：[[豐受大神宮|豐受大神宮]]的別宮。

== 月神命 ==
'''月神命'''（{{lang|ja|つきみたま の みこと}}），又稱為'''天月神命'''（{{lang|ja|あめのつきみたま の みこと}}）、'''月讀神'''（{{lang|ja|つきよ<sup>[[上代特殊仮名遣|乙]]</sup>み<sup>[[上代特殊仮名遣|甲]]</sup> の かみ}}）、'''月讀尊'''（{{lang|ja|つきよ<sup>[[上代特殊仮名遣|乙]]</sup>み<sup>[[上代特殊仮名遣|甲]]</sup> の みこと}}），是[[日本神話|日本神話]]中[[月神|月神]]，也是[[壹岐縣主|壹岐縣主]]等祖。

=== 事績 ===
據《[[先代舊事本紀|先代舊事本紀]]》卷第三之記載，[[饒速日尊|饒速日尊]]降臨[[豐葦原之千秋長五百秋長之瑞穗國|豐葦原之千秋長五百秋長之瑞穗國]]時，'''月神命'''等三十二人天降供奉。

《[[日本書紀|日本書紀]]》卷第十五則提到，[[顯宗天皇|顯宗天皇]]3年（[[487年|487年]]）[[二月初一|2月1日]]，[[阿閉事代|阿閉事代]]銜命，出使于[[任那|任那]]。於是，'''月神'''著人謂之曰：「我祖[[高皇產靈|高皇產靈]]，有預鎔造天地之功。宜以民地奉。我月神。若依請獻，我當福慶。」事代由是還京將事情稟奏顯宗天皇，後者批准以[[歌荒樔田|歌荒樔田]]（在[[山背國|山背國]][[葛野郡|葛野郡]]也）奉獻給月神。[[壹伎縣主|壹伎縣主]]先祖[[押見宿禰|押見宿禰]]（{{lang|ja|おしみ の すくね}}）侍祠。
