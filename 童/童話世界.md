《'''童話世界'''》是[[中國電視公司|中國電視公司]]（中視）[[兒童電視節目|兒童節目]]代表作品，播出期間為1987年1月13日－1997年7月2日，每一集演一個故事教導[[兒童|兒童]][[道德|道德]]觀念。製作人是[[汪瑩|汪瑩]]。

== 概要 ==

1980年代，[[老三台|老三台]]多以應付《[[廣播電視法|廣播電視法]]》規定「[[電視台|電視台]]必須有兩檔自製兒童節目」為依據來製播兒童節目，中視在此時期以《童話世界》與《[[兒童天地|兒童天地]]》撐持兒童節目領域<ref>{{cite news|url=http://www.tvcr.org.tw/p-media/media20.htm|title=台灣電視兒童節目探討|newspaper=《媒體識讀教育月刊》第20期|publisher=媒體識讀教育基金會|author=李秀美|date=日期不詳|accessdate=2016-02-06|deadurl=yes|archiveurl=https://archive.is/20160210202054/http://www.tvcr.org.tw/p-media/media20.htm|archivedate=2016-02-10}}</ref>。

== 歷屆主持人 ==

第一代主持人是[[崔麗心|崔麗心]]，第二代（末代）主持人是[[郎雄|郎雄]]。崔麗心在此節目中化身「香香姐姐」，與[[布偶|布偶]]「啄木鳥」搭檔主持；郎雄則是在此節目中化身「郎爺爺」單獨主持，每集都有一群兒童聚集在他面前聽他說故事。[[中視文化公司|中視文化公司]]發行崔麗心時期[[卡式錄音帶|卡式錄音帶]][[有聲書|有聲書]]版。

== 主題曲演唱 ==

[[中視兒童合唱團|中視兒童合唱團]]主唱[[主題曲|主題曲]]，主題曲在郎雄時期不播倒數第二句「多年多年以後，依然有好多的故事」與最末句「說了又說，講了又講，還是說不完」。
*[https://www.youtube.com/watch?app=desktop&persist_app=1&noapp=1&v=s6ebRVhWYHw 中視童話世界主題曲]

== 得獎 ==

《童話世界》在1988年[[第23屆金鐘獎|第23屆金鐘獎]]入圍[[電視金鐘獎|電視金鐘獎]]兒童節目獎，在1990年[[第25屆金鐘獎|第25屆金鐘獎]]入圍電視金鐘獎兒童節目主持人獎，在1991年[[第26屆金鐘獎|第26屆金鐘獎]]獲得電視金鐘獎兒童節目主持人獎（得獎者：郎雄），在1992年[[第27屆金鐘獎|第27屆金鐘獎]]入圍電視金鐘獎兒童節目獎與兒童節目主持人獎。
