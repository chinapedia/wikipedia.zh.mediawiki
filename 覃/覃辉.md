'''覃辉'''（{{bd|1968年10月||catIdx=T覃辉}}）<ref>[http://www.csrc.gov.cn/pub/zjhpublic/G00306212/201809/t20180905_343675.htm 中国证监会行政处罚决定书]</ref>，生于[[北京|北京]]，籍贯[[四川省|四川省]][[达县|达县]]，[[中国|中国]]企业家。作为「卓京系」掌门人，他有「民营传媒富豪第一人」的称号，在北京、[[上海|上海]]等地拥有数十家传媒公司，现任重庆[[长丰通信股份有限公司|长丰通信股份有限公司]]董事长。

历任[[广州|广州]]三菱公司经理、北京卓京投资控股有限公司董事长、[[重庆市|重庆市]][[涪陵区|涪陵区]]国有资产经营公司顾问、重庆三爱海陵股份有限公司董事。公司联营单位─[[星美传媒有限公司|星美传媒有限公司]]。2005年因[[张恩照|张恩照]]案，被有关部门带走协助调查。2006年，[[中华人民共和国最高人民法院|最高人民法院]]对其免于起诉。2009年以最大数额行贿者的身份出现在[[李培英|李培英]]案中。据称，覃辉现居国外。

== 生平 ==

覃辉出身于[[四川|四川]][[达县|达县]]一个普通家庭，其父任职于[[航空航天工业部|航空航天工业部]]五二九厂，母亲曾是达县教师。覃辉出生并成长于北京，早年在重庆求学期间，结识了后来的原配妻子林菁。林菁的父亲是原国家主席[[李先念|李先念]]第二任夫人[[林佳楣|林佳楣]]的弟弟。覃辉1989年大学毕业后，两人一起南下。覃曾在航天部五院的一个下属单位工作了不到一年，之后也曾在广州三菱公司和香港一家公司任职。覃辉的弟弟[[覃宏|覃宏]]是[[星美国际|星美国际]](00198.HK)董事局主席。

1994年，他成立了一家卓京商贸公司。1997年8月，覃辉透过卓京商贸，与其弟覃宏合作创办重庆长丰通信。

覃辉因1999年起运作A股上市公司长丰通信(000892，SH)而在资本市场上扬名。该公司后更名为“星美联合”，不过目前星美联合因连年亏损，已遭特别处理，成为“ST星美”。现在的ST星美已由[[新世界|新世界]](00017，HK)主席[[郑裕彤|郑裕彤]]的女婿[[杜惠恺|杜惠恺]]接盘。

2000年6月，他成立了卓京投资控股有限公司，号称注册资金4800万元，从此开始了大踏步发展。2001年9月，长丰通信与卓京投资合资成立星美传媒。覃辉一度持有香港及中国内地共4家上市公司的权益，其任主席的[[星美出版|星美出版]]，持有香港报纸《[[成报|成报]]》的股权；而星美国际的前身则是由多位艺人创办的东方魅力。

2003年夏天，覃辉通过自己在[[英属维尔京群岛|英属维尔京群岛]]注册的SMI公司(Strategic Media International)，开始了在香港的收购活动。一年多时间里，他渐次入主了三家香港上市公司，分别为[[东方魅力|东方魅力]]([[星美國際|星美國際]]前身0198，HK)、[[流动广告|流动广告]](前代碼為8036.HK，已除牌和結業)，以及[[现代旌旗出版|现代旌旗出版]](8010，HK)，并将另一家上市公司[[阳光文化|阳光文化]]([[優派能源發展|優派能源發展]]前身，0307.HK)所控制的阳光卫视70%的权益拿到手中。

2004年初，覃辉通过购入现代旌旗出版，获得《成报》控制权。

2010年4月26日，星美国际宣布，将以总计约454万美元收购主要从事影视后期制作业务的Photon 80%的股权，双方将在2010年7月签订正式协议，并于2010年第三季度内完成交易。同年7月，香港上市的国内最大的影院运营商之一星美国际公告称，已完成对大股东覃辉名下12家电影院的收购。该项收购总额约12亿港元。覃辉曾表示，星美国际重组之后将主要集中在电影产业，将包括“制作、院线、电影院、后期产品、影视基地”，形成一个上、中、下游产业链，从而实现星美集团的整体上市。

2005年覃辉因涉及原建行行长张恩照案，被有关部门带走协助调查，并逐渐淡出公众视野。2006年4月28日，恰在覃辉办理[[取保候审|取保候审]]一年后，最高院通知，解除其取保候审，并免于起诉。

2009年3月17日上午，首都机场集团公司原董事长李培英案终审在山东省高级法院开庭。在一审及终审的司法材料中，覃辉均以数额最大的行贿者出现在李培英案中，其累计向李行贿1867．68万元，占到李培英受贿总金额的七成。覃辉作为「污点证人」，从没出现于李培英庭审现场，只是向侦查机关提供了书面证言。尚未有司法机关的消息证实覃辉因行贿被起诉。

== 其他事项 ==

覃辉曾经跟香港女艺人[[李嘉欣|李嘉欣]]有交往。他们的关系在2000年的年中完结。
