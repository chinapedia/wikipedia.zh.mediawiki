'''占海特事件'''发生于2012年下半年，当时就读[[上海市|上海市]]新竹园中学的非上海[[户籍|户籍]][[初中生|初中生]]'''占海特'''因其家人違反中國大陸[[計劃生育|計劃生育]]規定和做生意违法[[逃税|逃税]]，使得占海特無法就讀上海的[[高中|普通高中]]<ref name="wangyijiaoyu">{{cite news|url= http://edu.163.com/12/1212/09/8IGVFB3100294JD3_all.html |title= 约辩异地高考少女之母否认炒作：就为读书（全文）|publisher= 网易教育频道综合|date=2012-12-12|accessdate=2017年2月6日}}</ref>，占海特認為此規定妨碍了她的个人私利<ref>{{cite news|url=http://edu.gmw.cn/2012-12/10/content_6098404.htm |archiveurl=https://web.archive.org/web/20170207112556/http://edu.gmw.cn/2012-12/10/content_6098404.htm |archivedate=2017-02-07 |title=占海特之父被刑拘是叠加的非正义 |author=光明网评论员 |publisher=光明网 |date=2012年12月10日 |accessdate=2017年2月5日 |deadurl=yes }}</ref>。在事件中，双方围绕占海特家庭触犯法律法规的情况、占海特家庭是否自我[[炒作|炒作]]、上海本地市民利益保护、中国教育政策等情况发生激烈的争论<ref name="sun2014">{{cite journal|title=“可见性”：社会化媒体与公共领域——以占海特“异地高考”事件为例|journal=西北师大学报杂志|last=孙|first=玮|last2=李|first2=梦颖|volume=51|issue=2|date=2014年3月|accessdate=2017年2月5日}}</ref><ref name="wangyijiaoyu"/><ref>{{cite news|author1=杨江|title=如何解“占海特”这道题？|url=http://www.xinminweekly.com.cn/News/Content/1624|accessdate=2017-12-05|agency=新民周刊|date=2012-12-24}}</ref><ref>{{cite web|author1=汉网-武汉晚报|title=“蝗虫女孩”占海特的中高考约辩(图)|url=https://news.163.com/12/1209/02/8I8GJ4F700014AED.html|website=网易新闻|accessdate=2017-12-05|date=2012-12-09}}</ref><ref>{{cite web|title=今日话题 2012-12-15 第2277期 不给占海特们上学，就请退税|url=http://view.news.qq.com/zt2012/zht/index.htm|website=腾讯评论|accessdate=2017-12-05|date=2012-12-15}}</ref><ref>{{cite news|title=再谈异地高考|url=http://hsb.hsw.cn/2012-12/15/content_8475866.htm|accessdate=2017-12-05|agency=华商报|date=2012-12-15}}</ref>。

占海特出生于[[广东|广东]][[珠海|珠海]]，有[[江西省|江西省]]户籍。她自称“少年公民”，呼吁教育公平，要求在上海参加中考；并主张就地中高考政策不应与考生父母的职业、纳税与社保等信息相关联<ref name="sun2014" />。2012年6月，她在[[新浪微博|新浪微博]]发声伊始便吸引了許多上海外来族群前来响应<ref name="nanfang1">{{cite web|url=http://www.nfyk.com/sy/ShowArticle.asp?ArticleID=4624 |archiveurl=https://web.archive.org/web/20150318131842/http://www.nfyk.com/sy/ShowArticle.asp?ArticleID=4624 |archivedate=2015年3月18日 |title=占海特猜想 |publisher=《南方》杂志 |accessdate=2013-05-29 |deadurl=yes }}</ref>。

以“守沪者联盟”为代表的反对方宣称自己并非“[[仇外|排外]]”，而是维护“契约精神”以及“权利与义务对等”的社会规则，并认为：上海的教育资源来自于上海市民数十年遵守国家计划生育法规所积累，占家超生、涉嫌偷漏税等违规违法行为才是导致其不能在上海中考的原因，占海特只是其背后团体的一颗棋子而已<ref>{{cite news|url=http://epaper.21cbh.com/html/2012-12/14/content_40129.htm|archiveurl=http://news.sohu.com/20121214/n360381371.shtml|archivedate=2012年12月14日|title=“守沪者联盟”成员：反对占海特非排外 是排毒|date=2012年12月14日|publisher=21世纪经济报道|author=贺莉丹|accessdate=2017年2月5日}}</ref>。双方在线上与线下展开了多轮辩论与争吵<ref>{{cite news|url=http://edu.gmw.cn/2012-12/09/content_5952652.htm|title=女孩约辩上海异地高考被骂 家庭遭举报超生偷税|publisher=武汉晚报|date=2012年12月9日|accessdate=2017年2月5日|deadurl=yes|archiveurl=https://web.archive.org/web/20170206185000/http://edu.gmw.cn/2012-12/09/content_5952652.htm|archivedate=2017年2月6日}}</ref>。2012年12月8日，占海特一家在[[上海人民广场|上海人民广场]]进行集会，與前往阻止的[[上海市民|上海市民]]发生冲突，最终，占海特父亲被警方以“非法集会”逮捕<ref>{{cite news|url=http://news.sina.com.cn/c/2012-12-09/202325768283.shtml|title=上海警方证实占海特父亲被拘 称其非法集会|publisher=人民网|date=2012年12月9日|accessdate=2017年2月5日}}</ref>。

==主要人物==
占海特是一名女初中生，1997年8月1日生于[[广东省|广东省]][[珠海市|珠海市]]。其父亲是江西省人。2002年她随父母到上海居住，没有本地[[户口|户口]]的她先後就读上海的公办小学和民办中学。接近初中毕业时，由于占海特父母違反計畫生育規定與沒有按規定繳稅，因此無法進入上海市的高中，只能在上海报读中等职业技术学校或回到户籍所在地江西参加中考升入普通高中学习。由于无法参与上海中考，她与她父亲多次到上海信访办及教育部门，均被拒绝要求<ref name="nanfang1"/><ref name="wangyijiaoyu"/>。

她有一个8岁的弟弟与一个5岁的妹妹，在当时中国的[[中华人民共和国计划生育|计划生育]]政策中属于違法<ref name="haikou1">{{cite web | url=http://edu.sina.com.cn/gaokao/2012-12-08/2111364771.shtml | title=非沪籍女孩约辩异地高考续：遭举报超生偷税 | publisher=海口晚报/新浪网 | accessdate=2013-05-29}}</ref>，按照上海的入户政策，違反计划生育不能擁有上海戶口。<ref name="21cn">{{cite web | url=http://news.21cn.com/today/renwu/2012/12/12/13995581.shtml | title=非沪籍女孩占海特的抗争 | publisher=21CN新闻 | accessdate=2013-05-29}}</ref>

==经过==
2012年6月8日，事件主角在[[新浪微博|新浪微博]]注册并发表自己希望参加上海中考意愿的微博，开始受到关注。她自称“少年公民、自由战士、民主先锋、因非沪籍失学，推动教育公平，矢志不移”<ref name="nanfang1"/><ref name="haikou1"/>。

她的微博内容使用成人化的语句，被不少人认为“过度成熟”，反对者则质疑有人代笔。她父亲称是经过全家商量后才发出微博。10月21日，該微博上发出“约辩”战帖（当时微博为个人实名认证），称邀请京沪户籍人士于10月25日在大沽路100号上海教委辩论，随后她被上海警方约谈<ref name="haikou1"/>。

2012年10月25日上午9点半，她們一家到达上海教委大门口，遇到上海本地居民的反对团体，与其展开激烈争辩。自此，她們声称遭到了生活上的干扰<ref name="haikou1"/>。

2012年12月8日，占海特之父占全喜於[[人民广场_(上海)|上海人民广场]]对上海市民和法制进行挑衅，遭到警方刑事拘留<ref name="依法刑拘">{{Cite web |url = http://politics.people.com.cn/n/2012/1210/c70731-19851851.html |title = 上海警方证实微博"约辩"少女占海特之父被刑拘 |author = 王炬鹏 |date = 2012-12-10|publisher = 中国新闻网 | language = zh-cn | accessdate = 2015-05-12 |format = shtml |quote = 新网上海12月10日电 (记者 陈静)上海警方今日证实，微博“约辩”少女占海特之父占全喜已因涉嫌妨害公务罪被警方刑事拘留。<br/>据透露，占全喜聚众非法集会，拒不配合民警现场劝离，影响社会公共秩序，妨碍警方正常执法。过程中，占全喜将民警抓伤。后经口头传唤无效，占全喜最终被强制带离。|archiveurl = <!--存檔連結--> |archivedate= <!--存檔日期-->}}</ref>。12月8日中午，當事者将她的遭遇和照片发上微博，之后被微博禁言。翌日上午，其微博账号遭到刪除<ref>{{cite web | url=http://news.xinhuanet.com/legal/2012-12/10/c_124069946.htm | title=上海争取异地高考权少女之父因非法集会被拘 | publisher=新华网 | accessdate=2013-05-16}}</ref>。此后，有媒体报道占家違反計畫生育與未按規定繳稅，因而激起上海市民对占家行为的不满<ref name="haikou1"/>。占家表示他们家租住的房屋房东受到居委会的压力，要求他们两个月内搬迁<ref name="21cn"/>。

2013年3月7日，上海[[市委书记|市委书记]][[韩正|韩正]]在参加第十二届全国人大一次会议的上海代表全体会议上称，让每一个孩子接受良好的教育是各级的责任和义务。他又提到，上海目前的[[义务教育|义务教育]]政策符合中央政府方针。韩正称今后会对高考、中职和高职的教育政策继续作出细化，保证学生的求学需求<ref name="hanzhenghuiying">{{cite web | url=http://news.hexun.com/2013-03-07/151823402.html | title=上海市委书记韩正回应“占海特”事件 | publisher=经济观察网 | accessdate=2013-05-29 | author=贾华杰}}</ref>。

==争议==
在中国大陆，上海的教育资源对外来人口实际比较开放，其中上海[[义务教育|义务教育]]阶段随迁子女的比例已占到40%<ref>{{cite web | url=http://news.163.com/12/1224/03/8JF896RV00014AED.html| title=随迁子女占上海义务教育阶段学生数比例超过40%| publisher=人民日报(北京)|date=2012-12-24| accessdate=2013-05-31}}</ref>，占海特不能参加中考的原因是其父母违反了中国大陆[[计划生育|计划生育]]规定<ref>{{cite web | url=http://news.ifeng.com/mainland/special/neidigaokaogaige/content-3/detail_2012_12/14/20157045_0.shtml| title=“守沪者”老崔：占海特不能异地高考应从自身找原因 | publisher=21世纪经济报道|date=2012年12月14日| accessdate=2013-05-31}}</ref>。而很多关于占海特的传闻都是占家自己的一家之言，所谓“占海特受到不公正待遇”的说辞违背事实<ref>{{cite web | url=http://news.sina.com.cn/z/xuanzexingbaodao/| title=“选择性”报道下真相的谜团 | publisher=新观察第198期| accessdate=2013-05-31}}</ref>。

==后续==
占海特后续未在上海继续争取求学。据2016年5至6月媒体报道，在家自学三年的占海特获美国[[普渡大学|普渡大学]]录取，并还在等待国外另一所大学全额[[奖学金|奖学金]]的录取通知书。<ref>{{cite news|url=http://www.rfa.org/cantonese/news/exam-06092016081247.html|title=争取教育平权少女　发愤自学获美大学取录|publisher=自由亚洲电台|date=2016年6月9日|accessdate=2017年2月5日}}</ref><ref>{{cite news|title=上海拒绝美国接纳 反思15岁女孩高考路|url=http://dailynews.sina.com/gb/chn/chnpolitics/chinapress/20160607/23297363367.html|accessdate=2017-12-05|agency=侨报|publisher=新浪新闻|date=2016-06-07}}</ref><ref>{{cite web|author1=凤凰评论|title=美国为何不嫌占海特占用教育资源|url=https://finance.ifeng.com/a/20160608/14473237_0.shtml|website=凤凰财经|accessdate=2017-12-05|date=2016-06-08}}</ref><ref>{{cite book|author1=罗慰年|title=多元化美国教育聚焦: 留学美国的个人见证|date=2017年6月28日|publisher=世界华语出版社|isbn=9781940266008|url=https://books.google.com/books?id=E74pDwAAQBAJ&pg=PA32&lpg=PA32|accessdate=2017-12-05|page=32}}</ref>
