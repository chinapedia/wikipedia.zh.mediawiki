'''片瀨江之島車站'''（{{jpn|j=片瀬江ノ島駅|hg=かたせえのしまえき|rm=Katase-enoshima eki|f=y}}）是一個位於[[神奈川縣|神奈川縣]][[藤澤市|藤澤市]]{{link-ja|片瀨海岸|片瀬海岸}}二丁目，屬於[[小田急電鐵|小田急電鐵]][[江之島線_(小田急電鐵)|江之島線]]的[[鐵路車站|鐵路車站]]。江之島線的[[終點站|終點站]]。[[車站編號|車站編號]]是'''OE 16'''。

此站是是江之島地區3間公司的路線當中唯一設有往[[東京都|東京]]方向（[[新宿站|新宿站]]）的直通列車的車站。站體以其[[龍宮|龍宮]]造型聞名。

片瀨江之島在1999年時入選[[關東車站百選|關東車站百選]]名單之列。

== 年表 ==
* 1929年（[[昭和|昭和]]4年）4月1日 - 開業。為「直通」班次停靠站。
* 1945年（昭和20年）6月 - 廢止「直通」班次，各站停車延伸至全線，本站為其停靠站。
* 1946年（昭和21年）10月1日 - 新增準急班次。
* 1955年（昭和30年）3月25日 - 新增通勤急行班次。
* 1957年（昭和32年） - 夏季運送海灘遊客的臨時列車「快速急行」開始運行。
* 1959年（昭和34年）4月 - 新增急行班次。
* 1999年（[[平成|平成]]11年）10月14日 - 入選[[關東車站百選|關東車站百選]]
* 2002年（平成14年）3月23日 - 新增湘南急行班次。
* 2004年（平成16年）12月11日 - 湘南急行更名為快速急行。
* 2008年（平成20年）2月 - 列車資訊顯示器改為全彩[[發光二極體|LED]]式。

==車站-{zh-tw:構造;zh-hk:結構;zh-cn:构造;}-==
由於是江之島線的終點站，片瀨江之島是一座採用[[港灣式月台|港灣式月台]]設有2座月台、3條線路的[[地面車站|地面車站]]，其中1號線位於最北側。由於站內自入口開始一路至[[剪票口|剪票口]]、[[月台|月台]]之間皆無段差，故無[[階梯|階梯]]、[[電梯|電梯]]、[[電扶梯|電扶梯]]等立體設施的配置。[[廁所|廁所]]位於剪票口內左側。

站內設有可使用[[PASMO|PASMO]]、[[Suica|Suica]]等[[智慧電子票卡|智慧電子票卡]]的檢票設備，供乘客自助進出車站，全站僅有東南側一處檢票口。但在[[海水浴場|海水浴場]]季節與{{link-ja|藤澤江之島煙火大會|ふじさわ江の島花火大会}}舉行期間，會設置有站員駐守的臨時檢票口以因應突增的人潮。

2008年2月，站內的列車資訊顯示器改為全彩[[發光二極體|LED]]式。之後在2014年7月，站名標也改為LED照明式。

== 站房 ==
龍宮城造型的站房在當初僅是臨時站房。開業當時，東海土地電氣株式會社（1922年（[[大正|大正]]11年）12月22日公布執照<ref>[{{NDLDC|974246/435}} 『鉄道省鉄道統計資料. 大正11年度』]（国立国会図書館デジタルコレクション）</ref>）預定建設大船～江之島～茅崎路線，該路線通車後將撤除此站房。

但由於東海土地電氣無法取得路線建設預算，執照即將失效。因此，投入高額費用的站房建築成為實質的主站房。

之後，東海土地電氣執照在1930年（昭和5年）1月失效<ref>[{{NDLDC|2957392/8}} 「鉄道免許取消」『官報』1930年1月31日]（国立国会図書館デジタルコレクション）</ref>，站房也登記為正式站房。東海土地電氣在1926年（大正15年）7月將業務移轉給新公司「江之島電氣鐵道」（後更名為江之島電鐵）後解散<ref>[{{NDLDC|1022003/12}} 『鉄道統計資料. 昭和元年 第3編 監督』]（国立国会図書館デジタルコレクション）</ref>，接著江之島電氣鐵道在1928年（昭和3年）7月收購當時東京電燈經營的軌道線，為現在的[[江之島電鐵線|江之島電鐵線]]。

正面玄關的站名標也使用與龍宮城入口相對應的特殊造型。

== 站名由來 ==
作為觀光地江之島的玄關站，由於江之島電鐵已有「江之島站」，且鄰近片瀨海岸，因而命名為'''「片瀨江之島」'''。另外，江之電是在小田急線開通前1個月將「片瀨」站改名為「江之島」站。
