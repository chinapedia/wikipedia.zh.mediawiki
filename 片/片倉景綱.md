'''片倉景綱'''（{{bd|1557年||1615年|12月4日}}）是[[日本戰國時代|日本戰國時代]]至[[江戶時代|江戶時代]]前期的武將。[[伊達氏|伊達氏]]家臣。[[伊達政宗|伊達政宗]]的近習，後來擔任[[軍師|軍師]]。[[仙台藩|仙台藩]][[片倉氏|片倉氏]]初代，通稱「'''小十郎'''」成為片倉家代代相傳世襲的當家名稱。父親是[[置賜郡|置賜郡]][[永井庄|永井庄]][[八幡神社|八幡神社]]（現今[[成島八幡神社|成島八幡神社]]）的[[神職_(神道)|神職]][[片倉景重|片倉景重]]。

異父同母的姐姐[[片倉喜多|喜多]]是政宗的[[乳母|乳母]]。而[[鬼庭綱元|鬼庭綱元]]（喜多同父異母的弟弟）是義理上的兄長。

== 生涯 ==
[[弘治_(後奈良天皇)|弘治]]3年（1557年）出生，為家中次男。年幼時，雙親相繼死去，因為與姐姐喜多相差近20歲，喜多等同於其母親而養育他，不久後，以養子身份被送到親戚藤田家中，不過因為藤田家中誕下男子，於是回到喜多身邊，兩人再次一起生活。姐姐喜多被認為[[文武兩道|文武兩道]]，又喜好[[兵書|兵書]]，因此景綱自小也受喜多的親自教育。

[[永祿|永祿]]10年（1567年），主君[[伊達輝宗|伊達輝宗]]誕下嫡子[[伊達政宗|政宗]]後，姐姐喜多被任命為政宗的[[乳母|乳母]]。[[天正_(日本)|天正]]初年期間，伊達家城下的[[米澤|米澤]]發生大火，此時景綱的活躍得到承認，於是以徒[[小姓|小姓]]的身份服侍輝宗。此後，因為[[遠藤基信|遠藤基信]]推薦而在天正3年（1575年）成為政宗的近侍，後來成為重臣而得到其重用。

此後，參與天正13年（1585年）的[[人取橋之戰|人取橋之戰]]、天正16年（1588年）的[[郡山合戰|郡山合戰]]、天正17年（1589年）的[[摺上原之戰|摺上原之戰]]、天正18年（1590年）的[[小田原征伐|小田原征伐]]、[[文祿|文祿]]2年（1593年）的[[文祿慶長之役|文祿慶長之役]]、[[慶長|慶長]]5年（1600年）的[[關原之戰|關原之戰]]等大部份政宗參與的主要戰役，數度化解[[伊達氏|伊達氏]]的危機。其中，在小田原征伐之際，建議政宗應加入[[豐臣秀吉|豐臣秀吉]]一方參戰，令政宗因此決定前往小田原參戰。

此外還被任命為[[城代|城代]]，曾擔任[[安達郡|安達郡]][[二本松城|二本松城]]的在番、[[信夫郡|信夫郡]][[大森城|大森城]]城主，在[[奧州仕置|奧州仕置]]後，被任命為[[佐沼城|佐沼城]]城主、[[亙理城|亙理城]]城主等。亦擔當伊達氏對外交渉的[[取次|取次]]（類似代理人的角色），政宗發出的外交文書中，多數還會附上景綱的副狀。

關原之戰後的慶長7年（1602年），主君政宗成為[[仙台藩|仙台藩]]主後，雖然[[德川幕府|德川幕府]]頒佈[[一國一城令|一國一城令]]，但片倉家被允許破例成為[[白石城|白石城]]1万3千石的城主，但因為生病，於是景綱在亙理領內的神宮寺村接受療養，慶長10年（1605年）春天，移至白石城。

慶長19年（1614年）開始的[[大坂之陣|大坂之陣]]時發病，無法跟隨政宗參戰，於是命令兒子[[片倉重長|重長]]代他參戰。[[元和_(後水尾天皇)|元和]]元年（1615年）病逝，享年59歲。[[家督|家督]]由兒子重長（重綱）繼承。

== 人物 ==
*在伊達家中，與「武之[[伊達成實|伊達成實]]」（{{lang|ja|武の伊達成実}}）並稱，被稱為「'''智之片倉景綱'''」（{{lang|ja|智の片倉景綱}}）。一説指擅長[[日本劍術|劍術]]，亦曾擔任[[伊達政宗|政宗]]年幼時的劍術指導等，因此被認為是智勇兼備的武將。另外據說亦是相當厲害的吹笛高手。

*在內政、戰時亦有謀略，以優秀的計策支持政宗，政宗亦多數會直接採納。

*在妻子懷了[[片倉重長|重長]]時，據說因為忌憚當時主君政宗還沒有孩子，所以企圖殺害自己的兒子，不過後來因為政宗說服而無事。

*才能連當時的天下人[[豐臣秀吉|豐臣秀吉]]都有很高評價。在[[奧州仕置|奧州仕置]]時，秀吉希望景綱成為自己的直臣，於是打算立景綱為[[三春藩|三春藩]]5万石[[大名|大名]]，不過景綱因為對政宗的忠義而辭退。

*晩年非常肥胖。根據『[[伊達家世臣家譜|伊達家世臣家譜]]』記載，在[[慶長|慶長]]7年（1602年），政宗因為「年長後肥胖的身體已經不適合重型鎧甲吧」（{{lang|ja|年を取って太ったその体では重い鎧は身に合わないであろう}}）而下賜較輕的鎧甲。從症狀推測，被認為是患了[[糖尿病|糖尿病]]。

*死後，據說有6名家臣因為景仰景綱的人德而[[殉死|殉死]]。

== 逸話　==
*在[[伊達政宗|政宗]]在[[人取橋之戰|人取橋之戰]]中，因為過於深入追擊敵兵，反遭敵兵包圍。這時片倉以「我就是大將伊達政宗，政宗在此，來取我首級。」（{{lang|ja|やあやあ殊勝なり、政宗ここに後見致す}}）以自己是伊達政宗來騙取敵兵的攻擊，以此拯救政宗。

*政宗在小時候因為[[天花|天花]]使得右眼失去視力，病後患病的眼球卻從眼窩突出，這樣的醜貌使得政宗有著大大的卑劣感和沉默的性格。景綱為了導正政宗的性格，於是將政宗拉到侍醫所在的房間，親手抱住政宗的頭，並以短刀一口氣挑出政宗的右眼球。此後，政宗從陰暗的性格，轉變成有活力並且精進於[[文武兩道|文武兩道]]的少年。

*在其他逸話中，擔任[[日本劍術|劍術]]指南時向政宗問道「在戰場中，如果右眼被找到的話，怎麼辦」（{{lang|ja|戦場でその右眼をつかまれたらどうする}}），此時政宗拔出[[脇差|脇差]]並刺入右眼中，命令景綱把其挑出。

*在政宗所寫的信中，有簡略固有名詞的習慣（例如把田村寫為「田」、相馬寫為「相」等），給景綱的信中亦簡寫成「片小」（{{lang|ja|かた小}}）。當景綱把[[家督|家督]]讓給兒子[[片倉重長|重長]]後，重長亦繼承了「小十郎」之名，而政宗給重長的信中，把他的名字簡寫成「片備」（{{lang|ja|かた備}}）（片倉備中守的簡寫）。

*[[慶長|慶長]]5年（1600年），政宗要在相馬領地留宿，在此的前日，迎接政宗的景綱率領7百至8百士兵進入現今[[南相馬市|南相馬市]][[鹿島區|鹿島區]]，並在該地留宿。此時景綱與[[相馬盛胤_(彈正大弼)|相馬盛胤]]的付[[家老|家老]][[加藤左近|加藤左近]]會談，希望可以勸導政宗加入德川一方，左近對此答應。（『[[奧相茶話記|奧相茶話記]]』）

*慶長5年（1600年），當[[越後上杉氏|越後上杉氏]]侵入[[最上氏|最上氏]]領內時，最上家向政宗求援，片倉進言「不用馬上去救援，趁他們兩軍交戰，達到最疲勞的階段時攻入，則一定能消滅上杉軍」（{{lang|ja|すぐに救援には赴かず、両軍入り乱れ、疲労が極みに達した段階で攻め入り、上杉勢を完膚なきまで叩くべし}}）。從作戰方面來看相當合理，但這時政宗因為擔心住在[[山形城|山形城]]的母親[[保春院|保春院]]的安全，於是退回了這套戰略。

*片倉家直到[[明治時代|明治時代]]為止，11代都管治白石之地。現今[[白石市|白石市]]的[[市章|市章]]「[[黑釣鐘|黑釣鐘]]」，是景綱的姐姐[[片倉喜多|喜多]]決定使用的[[旗指物|旗指物]]的[[家紋|家紋]]。在[[仙台藩|仙台藩]]中，片倉氏的家格是御一家，[[慶安|慶安]]4年（1651年）12月1日，在兒子重長時期被列入。

*嫡男重長在[[大坂夏之陣|大坂夏之陣]]中的[[道明寺之戰|道明寺之戰]]中，殺死[[後藤基次|後藤基次]]等人，並多次奮戰，於是有「鬼之小十郎」（{{lang|ja|鬼の小十郎}}）的異名。而重長的兒子[[片倉景長|景長]]亦以「小十郎」為名，在[[伊達騷動|伊達騷動]]中，支持年幼的主君[[伊達綱村|綱村]]。於是代代仕於伊達氏的「片倉小十郎」名號，以後就被稱為伊達家忠臣之鑑。

== 肖像畫 ==
景綱在生前或死後亦沒有立即制作肖像或肖像畫。最舊的肖像畫已經是在[[明治時代|明治時代]]所畫、被收藏於[[仙台市博物館|仙台市博物館]]中的畫。關於景綱所着用的甲冑，因為在畫下肖像時，已經沒有現存的實物，因此是想像出來的。
