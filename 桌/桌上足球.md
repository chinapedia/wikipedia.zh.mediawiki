'''桌上足球'''（'''Table football''' 或 '''foosball'''、'''桌式足球'''或'''手足球'''、'''足球檯'''）是一種在桌面上的足球遊戲，規則是源自於[[足球|足球]]規則。此運動的起源不詳，許多[[歷史學家|歷史學家]]都認同第一台桌上足球出現在1880年代至1890年代的[[西班牙|西班牙]]，[[法國|法國]]或者[[德國|德國]]。

== 歷史 ==
一些人都宣称自己发明了桌上足球这一运动游戏，例如西班牙人[[Alejandro_Finisterre|Alejandro Finisterre]]和法国人[[Lucien_Rosengart|Lucien Rosengart]]。发明专利早在1890年代就已经存在，更多详情参考下面的外部链接。

桌上足球（Table football）在[[美国|美国]]称之为'''foosball'''，是从[[德语|德语]]足球一词'''Fußball'''所演变而来（foosball一般来说在德语中是踢球者的意思）也可称之为'''table soccer'''，在[[西班牙|西班牙]]被称之为'''futbolín'''，在[[智利|智利]]这一运动游戏被称做'''taca-taca'''，而在一些其他[[西班牙语|西班牙语]]国家则被称之为'''futbolito'''，[[意大利|意大利]]则称作'''biliardino''' 或 '''calcio Balilla'''，一些法国风格的参赛者也将其称之为'''footie''' 或 '''baby-foot'''. 在[[台灣|台灣]]將桌上足球稱之為手足球。

2002年，[[ITSF|ITSF]]（国际桌上足球联合会）在法国成立，第一批会员国有比利时、中国、丹麦、法国、德国、意大利、英国和美国（摘自：世界桌式足球联合会官方网站）。作为组织运动体系和框架，调节国际性赛事，联合会成立的目的是为了发扬桌上足球运动。并且将运动建成推广为[[国际奥林匹克委员会|国际奥林匹克委员会]][[IOC|IOC]]所官方承认的体育运动。

== 比賽概述 ==
比赛开始时，球会从一侧的洞眼中发出， 從球台側邊的洞口發球，或也可用手把球放在中場人偶的腳旁。球偶利用碰撞擊球，一般來說在比賽的時候，球的滾動的速度可達40英里／小時（64公里／小時）。此運動（也可稱遊戲）需要快速的反應與一種巧妙的操控，球員需要利用優良調諧運動技能來控制球的滾動。 
 
基本概念包括傳球，當你的在任何一支球桿上掌握有控球權時，然後將球傳遞到更接進球門的球桿，然後更有機會能在你發現有空檔的時候射門。 
 
通常一場比賽是採取5分制，在錦標賽中常常採用5戰3勝或是3戰2勝的方式進行。

== 球台 ==
[[File:foosball_garlando_aerial.jpg|thumb]]式桌上足球比赛]]
[[File:Foosball11Players.jpg|thumb]]

市面上有大量不同版本的桌上足球桌子存在，在世界巡回赛和正式的ITSF锦标赛中使用的是"法式"的Bonzini、"意大利式"的 Roberto Sport和[[Garlando|Garlando]]、"德式"的 Tecball。其他还有一些主要的品牌包括有 Kicker、Rosengart、Eurosoccer/Wood、Löwen-Soccer、Warrior、Lehmacher、Leonhart和Smoby。同时艺术家[[毛里齐奥·卡泰兰|毛里齐奥·卡泰兰]]（Maurizio Cattelan）设计出了一款长7米的巨型桌上足球，并且把其称之为「体育场」，一边各有11个小人。另一款独特的桌上足球作品是由Elevenforty公司所制作的，每张桌子都是纯手工制作，每个小人也都是由现时中球星的模样仿制而来的。

不同的桌子类型对选手的风格有着很大的影响。非美式的桌上足球都采用禁区内设置一个守门员。

== 射門 ==
<!--Add by Jack J Huang jkjuh.tw@gmail.com in Taiwan (TFA)-->
在桌上足球（手足球）運動中有許多基本的射門方法可以很快的上手，就算是新手也可以容易的學習。
* 推桿 (The Push) - 球放置在人偶的側邊，將球桿用推的方式將球連同人偶帶著移動，直到進攻者想要出手的位置之時，進攻者將球桿往後抬高在球的後方後將球擊出。
* 拉桿 (The Pull) - 球放置在人偶的側邊，將球桿用拉（往進攻者的方向）的方式將球連同人偶帶著移動，直到進攻者想要出手的位置之時，進攻者將球桿往後抬高在球的後方後將球擊出。
* 轉桿 (The Rollover) - 將人偶的腳壓在球的上方，此動作英文稱之為Pin，將球桿逆時針旋轉一周後擊到球則完成進攻。如果擊球前旋轉超過360度, 或是擊到球後球杆超過360度, 則被判犯規. '''注意: 擊球前與擊球後的角度分開計算.'''

== 机器人桌上足球 ==
[[机器人|机器人]]桌上足球由[[弗莱堡大学|弗莱堡大学]]的机器人专家所研发，宣称可以击败85%临时拼凑起来的选手。他们使用透明桌台下部的摄像头，以小球的运行轨迹作为基础，使用一个电子控制系统支配高性能的陀螺修正马达用于旋转和移动桌上足球小人。如今一个老练的选手可以在10场比赛中战胜机器人1次[http://www.newscientist.com/news/news.jsp?id=ns99996346]。其他的桌上足球机器人，[[Foosbot|Foosbot]]则宣称研制出人类不可战胜的机器人产品，还有的桌上足球机器人是由[[丹麦科技大学|丹麦科技大学]]的两名学生所研制出的在普通桌子上放架摄像的机器人，但技术仍然不成熟[http://foospmp.myl.dk]。
