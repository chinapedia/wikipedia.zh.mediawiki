'''祝钦明'''（{{bd|7世纪||710年代|catIdx=Z}}），字'''文思'''，[[唐朝|唐朝]]、[[武周|武周]]官员，[[唐中宗|唐中宗]]年间任[[宰相|宰相]]，还是著名儒家学者，但中宗驾崩后因曾奉承[[韦庶人|韦皇后]]而遭到诟病。

== 家世 ==
祝钦明生年不详，是[[雍州|雍州]][[兴平县|始平县]]人。<ref name=BT189-2>[[:s:旧唐书/卷189下|《旧唐书》卷一百八十九下]]</ref>他的父亲[[祝綝|祝綝]]就是名[[儒生|儒生]]，写了很多对[[诸子百家|诸子百家]]学说提出质疑的书；祝綝的学生[[张后胤|张后胤]]官居显要后，在朝推荐祝綝仕官，诏祝綝对策中高第，但祝綝最高只当到尉。<ref name=NBT109>[[:s:新唐书/卷109|《新唐书》卷一百零九]]</ref>

祝钦明少年即熟习《[[五经|五经]]》，又学习历史和百家学说。[[唐高宗|唐高宗]]年间，祝钦明中[[明经|明经]]，拜为東臺典儀。在高宗[[永淳|永淳]]（682年 - 683年）之后、武周[[天授_(武周)|天授]]（690年 - 692年）之间，他又考中英才杰出、业奥《六经》等科，拜[[著作郎|著作郎]]，负责记录皇帝的行为。曾任国子博士，[[久视|久视]]元年（700年）在太子詹事任上时，[[武则天|武则天]]因礼官不甚详明，特诏祝钦明及博士[[韦叔夏|韦叔夏]]、大儒[[郭山恽|郭山恽]]等，每有仪注，皆令参定。<ref name=BT21>[[:s:旧唐书/卷21|《旧唐书》卷二十一]]</ref><ref>[[:s:新唐书/卷122|《新唐书》卷一百二十二]]</ref><ref>[[:s:令韦叔夏等刊定司礼仪注制|:s:令韦叔夏等刊定司礼仪注制]]</ref>[[长安_(年号)|长安]]元年（701年），祝钦明又累迁太子前皇帝[[李显|李显]]的率更令兼[[崇文馆|崇文馆]][[学士|学士]]，与太子右庶子[[崔神庆|崔神庆]]等兼充东宫侍读，<ref>[[:s:旧唐书/卷77|《旧唐书》卷七十七]]</ref>隔日履职，教太子学经。<ref name=NBT109/>他还任[[金紫光禄大夫|金紫光禄大夫]]，曾作林钟宫送神用音乐《永和》：“閟宫实实，清庙微微。降格无象，馨香有依。式昭纂庆，方融嗣徽。明禋是享，神保聿归。”<ref>[[:s:旧唐书/卷31|《旧唐书》卷三十一]]</ref>二年（702年），迁[[太子少保|太子少保]]。<ref name=BT189-2/>

祝钦明与同僚学士郭山恽、[[李宪|李宪]]等都专守先儒章句，不满直弘文馆四门博士[[王元感|王元感]]指摘先儒古义，深刻讥讽，多次诘难，王元感随机应答，不为所屈。<ref name=BT189-2/><ref name=NBT199>[[:s:新唐书/卷199|《新唐书》卷一百九十九]]</ref>

== 唐中宗复位年间 ==
[[神龙_(年号)|神龙]]元年（705年）正月，武则天因政变退位，李显复位为唐中宗。祝钦明因本是中宗的东宫侍读旧臣，二月被擢拜[[国子祭酒|国子祭酒]]，授[[同中书门下三品|同中书门下三品]]，为[[实质宰相|实质宰相]]，加[[银青光禄大夫|银青光禄大夫]]。<ref>《[[全唐文|全唐文]]》</ref>四月，又以太子少詹事兼侍读、国子祭酒任[[刑部尚书|刑部尚书]]，<ref>[[:s:资治通鉴/卷208|《资治通鉴》卷二百零八]]</ref>后又任[[礼部尚书|礼部尚书]]，爵封'''鲁国公'''，食实封三百户，加[[银青光禄大夫|银青光禄大夫]]荣衔，得到朝廷尊崇，发动政变的[[桓彦范|桓彦范]]、[[崔玄暐|崔玄暐]]、[[袁恕己|袁恕己]]和[[敬晖|敬晖]]虽然都成为当权宰相（但不久后即因韦皇后及其情人、中宗的表兄和宠臣[[武三思|武三思]]构陷而失势），也都向祝钦明学习《[[周礼|周礼]]》大义。他还兼修国史。当年曾与礼官等奏[[永徽|永徽]]年间藉田改为先农，先农坛改为帝社坛，<ref>[[:s:明史/卷49|《明史》卷四十九]]</ref>祭先农既改为帝社坛，<ref>[[:s:建太社议|:s:建太社议]]</ref><ref>[[:s:社稷议|:s:社稷议]]</ref>诏准。<ref>[[:s:旧唐书/卷24|《旧唐书》卷二十四]]</ref><ref>[[:s:新唐书/卷14|《新唐书》卷一十四]]</ref>又奏从博士[[张齐贤|张齐贤]]议，以景皇帝为太祖，依博士[[刘承庆|刘承庆]]议，尊崇六室，<ref>[[:s:详定博士等七庙议|:s:详定博士等七庙议]]</ref>获准。<ref name=NBT199/><ref>[[:s:旧唐书/卷25|《旧唐书》卷二十五]]</ref>与[[中书令|中书令]][[韦安石|韦安石]]、尚书右丞[[苏瑰|苏瑰]]、兵部郎中[[狄光嗣|狄光嗣]]等奉诏删定《垂拱格》后至神龙元年已来制敕，为《散颁格》七卷。<ref>[[:s:旧唐书/卷50|《旧唐书》卷五十]]作祝钦明时为[[礼部侍郎|礼部侍郎]]。</ref>又奉诏与[[韦巨源|韦巨源]]、[[唐休璟|唐休璟]]、[[李怀远|李怀远]]、苏瑰等定《垂拱格》及《格后敕》，前后计二十卷，颁下施行。二年（706年），与[[魏元忠|魏元忠]]、[[武三思|武三思]]、[[徐彦伯|徐彦伯]]、[[柳冲|柳冲]]、[[韦承庆|韦承庆]]、[[崔融_(国子司业)|崔融]]、[[岑羲|岑羲]]、[[徐坚|徐坚]]等撰《则天皇后实录》二十卷。<ref name=BT92>[[:s:旧唐书/卷92|《旧唐书》卷九十二]]</ref><ref name=NBT58>[[:s:新唐书/卷58|《新唐书》卷五十八]]</ref>七月，[[中丞|中丞]][[萧至忠|萧至忠]]弹劾祝钦明为逃避服丧而隐瞒亲人死讯，有违孝道，八月，祝钦明因此一度下台，被贬为[[申州|申州]][[刺史|刺史]]。<ref name=BT189-2/><ref name=NBT109/><ref>[[:s:旧唐书/卷7|《旧唐书》卷七]]作九月被贬。</ref><ref>[[:s:新唐书/卷4|《新唐书》卷四]]</ref>百官为此惊悚。<ref>[[:s:新唐书/卷123|《新唐书》卷一百二十三]]</ref>

因担心不能重获权力，祝钦明谄事韦后，不久被召回重任国子祭酒。[[景龙|景龙]]三年（709年）八月，中宗想亲自在长安城南郊祭祀天地，祝钦明和时任[[国子司业|国子司业]]的郭山恽为希旨奉承韦后，引用解释了《[[礼记|礼记]]·祭统》的一段文字，称应由韦后充[[亚献|亚献]]。<ref>[[:s:皇后南郊助祭议|:s:皇后南郊助祭议]]</ref>中宗起初质疑，并咨询礼官们的意见。百官多曲意阿附祝钦明之说，而[[太常博士|太常博士]][[唐绍|唐绍]]、[[蒋钦绪|蒋钦绪]]和国子司业兼[[修文馆|修文馆]]学士[[褚无量|褚无量]]及太常博士[[彭景直|彭景直]]都表示不同意，<ref>[[:s:旧唐书/卷85|《旧唐书》卷八十五]]</ref>认为这仅适用于祭宗庙的场合。<ref>[[:s:新唐书/卷112|《新唐书》卷一百一十二]]</ref><ref>[[:s:新唐书/卷113|《新唐书》卷一百一十三]]</ref><ref>[[:s:祝钦明请南郊皇后充亚献议|蒋钦绪《祝钦明请南郊皇后充亚献议》]]</ref>时任宰相尚书右仆射的苏瑰也认为这很不妥，在中宗面前反驳得祝钦明感到羞愧。中宗让宰相左仆射[[韦巨源|韦巨源]]裁决，韦巨源支持祝钦明的说法，中宗虽为苏瑰所悟，仍然从祝钦明所奏，韦后也得以亚献。<ref>[[:s:旧唐书/卷88|《旧唐书》卷八十八]]</ref><ref>[[:s:旧唐书/卷102|《旧唐书》卷一百零二]]</ref><ref>[[:s:新唐书/卷13|《新唐书》卷一十三]]</ref><ref>[[:s:新唐书/卷125|《新唐书》卷一百二十五]]</ref><ref>[[:s:新唐书/卷200|《新唐书》卷二百]]</ref>但在唐、蒋反对下，韦巨源否决了以祝钦明的以有权势的皇女[[安乐公主|安乐公主]][[李裹儿|李裹儿]]为[[终献|终献]]的提法，<ref>[[:s:旧唐书/卷51|《旧唐书》卷五十一]]</ref>最后中宗以韦巨源本人为终献。<ref name=BT189-2/><ref name=BT21/><ref name=BT92/>四年（710年）五月，韦后设宴庆祝亲戚结婚，祝钦明主动表演[[八风舞|八风舞]]，扭动着肥胖丑陋的身躯，摇头闪目，丑态毕现，虽然逗乐了中宗，其他官员却暗地里觉得恶心，如[[吏部侍郎|吏部侍郎]][[卢藏用|卢藏用]]就私下对学士们感叹：“祝公是举，五经扫地矣。”<ref name=NBT109/><ref>[[:s:资治通鉴/卷209|《资治通鉴》卷二百零九]]</ref>

== 中宗驾崩后 ==
同年六月，中宗暴崩。传统史家认为他是被韦后和安乐公主毒杀的。不久，韦后又被中宗的妹妹[[太平公主|太平公主]]和侄子[[临淄|临淄]]王[[李隆基|李隆基]]政变推翻。中宗的弟弟、之前同样当过皇帝的相王[[李旦|李旦]]继位为[[唐睿宗|唐睿宗]]。[[景云|景云]]元年（710年）十二月，[[侍御史|侍御史]][[倪若水|倪若水]]弹劾祝钦明、郭山恽奉承韦后、希旨误导中宗，指二人为小人，不宜在朝。<ref>[[:s:劾奏祝钦明郭山恽疏|:s:劾奏祝钦明郭山恽疏]]</ref>睿宗贬祝钦明为[[饶州_(隋朝)|饶州]]刺史。<ref>[[:s:资治通鉴/卷210|《资治通鉴》卷二百一十]]</ref><ref>《[[册府元龟|册府元龟]]》误作玄宗登基后事。</ref>儒生都以他为羞。后来，祝钦明又被迁为[[洪州_(隋朝)|洪州]]都督，最后被召回任崇文馆学士，并在[[唐玄宗|唐玄宗]][[先天_(年号)|先天]]初年以金紫光禄大夫、崇文馆学士、[[上柱国|上柱国]]、鲁国公身份参与了《[[妙门由起|妙门由起]]》的编撰，<ref>[[:s:妙门由起序|史崇玄《妙门由起序》]]</ref>不久卒于任上。<ref name=BT189-2/><ref name=NBT109/>

== 评价 ==
*《新唐书》赞曰：'''钦明'''以经授中宗，为朝大儒，乃诡圣僻说，引艳妻郊见上帝，腥德播闻，享胙不终。盖与[[少正卯|少正卯]]顺非而泽，[[庄周|庄周]]以诗书破冢者同科。独保腰领死家箦，宁不幸邪！后之托儒为奸者，可少戒云。<ref name=NBT109/>
*[[明世宗|明世宗]]年间，有白鹊祥瑞，[[太常卿|太常卿]][[许诰|许诰]]献论，被[[给事中|给事中]][[张裕|张裕]]等弹劾，张裕比许诰为祝钦明。<ref>[[:s:明史/卷186|《明史》卷一百八十六]]</ref>
