'''质数币（XPM）'''，又称'''素数币'''，是以寻找质数为[[工作量证明|工作量证明]]<ref name="whitepaper">{{cite web|url=http://primecoin.io/bin/primecoin-paper.pdf|title=Primecoin: Cryptocurrency with Prime Number Proof-of-Work|accessdate=2018-08-30|author1=Sunny King|website=http://primecoin.io/}}</ref><ref name="PoW">{{cite web|url=https://www.coindesk.com/why-digital-currency-players-anonymous/|title=New currency Primecoin searches for prime numbers as proof of work|accessdate=2018-08-30|author1=Bradbury, Danny|website=https://www.coindesk.com/}}</ref>的加密货币。质数币的创始人是Sunny King。

== 质数币的诞生原因 ==
质数币（Primecoin）是Sunny King的一项尝试，旨在避免PoW的能量浪费。因为PoW确实于当下主流共识机制中稳健性最高，为了保持这种稳健性并有意义的使用能量，Sunny King找到了解决方案：寻找质数。质数币仍然使用PoW机制，它挖矿的过程就是寻找质数链。质数在数论领域具有极高价值。质数币是其一的使挖矿过程中消耗的大量能源产生价值的加密货币。

== 质数币的PoW机制 ==
质数币（Primecoin）将寻找坎宁安链（Cunningham chains）和双链（Bi-twin chains）作为PoW（Proof-of-Work）工作量证明机制。

PoW的设计使得质数币网络上的所有节点都可以高效的进行验证工作。<ref name=whitepaper /><ref name="usefulmining">{{cite web|url=https://bitcoinmagazine.com/articles/primecoin-the-cryptocurrency-whose-mining-is-actually-useful-1373298534/|title=Primecoin: The Cryptocurrency Whose Mining is Actually Useful|accessdate=2018-08-30|author1=Buterin, Vitalik|website=https://bitcoinmagazine.com/}}</ref>质数币的PoW（Proof-of-Work）工作量证明机制具有以下特征：

* 质数币的工作是找到具有特定形式的质数链，被质数币共识所接受的三种链：第一类坎宁安链，第二类坎宁安链，以及双链。这些链中的质数将受到一个最大值的限制来保证高效的验证。<ref name=whitepaper /><ref name=usefulmining />
* 随着链长的增长，找到新质数的难度将指数级增长。<ref name=whitepaper />
* 所有网络节点都可以对合理位数的质数进行高效验证。<ref name=whitepaper />
* 梅森数因其位数过多，验证所需时间过长而被排除。<ref name=whitepaper />

== 质数币的经济模型 ==
质数币的稀缺性并不像比特币一样有一个硬上限，而是取决于挖币设备发展中的[[摩尔定律|摩尔定律]]和挖币算法的提升。这样的设计让质数币更贴近类似黄金的自然稀缺性。

随着人类接近[[摩尔定律|摩尔定律]]的极限，质数币的通胀率降低速度会减缓，以一个更缓慢的速度趋向零。这样，质数币仍然有相似于黄金的较好稀缺性，并且安全性可以在维持较低的交易费的同时得到保证。质数币的通胀率相对于点点币而言被设计的较缓，以此来对纯工作量证明的加密货币所需电力消耗作补偿。

== 质数币的优势和长处 ==
* 质数币是目前PoW共识下TPS（transaction per second）每秒传输的交易数最高的币，理论速度高达70，是比特币的十倍。
* 质数币的挖矿过程是有意义的，它通过发掘质数链寻找质数，对质数的研究大有裨益，也对[[黎曼猜想|黎曼猜想]]进行了间接佐证。 
* 技术亮点：
# 每个区块出币量为“999/当前难度平方”，这样的设置可以提高币值的稳定性。平方反比的设定使得系统不会因为算力突然大量涌入而出现大的波动。<ref name=usefulmining />
# 质数无穷多，质数币不仅没有设置币量上限，而且交易费用直接销毁，这样可以有效控制通货膨胀，此类黄金的微通胀模型较其他模型而言具有更好的稀缺性。
