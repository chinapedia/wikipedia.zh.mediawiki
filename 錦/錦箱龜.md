'''錦箱龜'''（[[學名|學名]]：'''''Terrapene ornata'''''）是[[箱龜屬|箱龜屬]]下的一種[[龜|龜]]。主要生活在[[北美洲|北美洲]]，1986年成為[[堪薩斯州|堪薩斯州]]的[[州爬行動物|州爬行動物]]。<ref name="Kansas law">{{cite web|title=2009-73-1901 Kansas Code patriotic emblems, state reptile, designation|url=http://law.justia.com/kansas/codes/2009/chapter73/statutes_31572.html|publisher=Justia|accessdate=February 11, 2011}}</ref><ref name="Shearer315">{{Harvnb|Shearer|1994|p=315}}</ref>

== 亞種 ==

錦箱龜有兩個亞種，即''Terrapene ornata ornata'' (<small>[[Louis_Agassiz|Agassiz]], 1857</small>)和''Terrapene ornata luteola'' (<small>[[Hobart_M._Smith|Smith]] & [[L.W._Ramsey|Ramsey]], 1952</small>)。

== 保護 ==

在[[美國|美國]][[印第安那州|印第安那州]]，錦箱龜是[[瀕危物種|瀕危物種]]。<ref name=312iac>{{cite|author=Indiana Legislative Services Agency|year=2011|title=312 IAC 9-5-4: Endangered species of reptiles and amphibians|work=Indiana Administrative Code|accessdate=28 Apr 2012|url=http://www.in.gov/legislative/iac/}}</ref>
