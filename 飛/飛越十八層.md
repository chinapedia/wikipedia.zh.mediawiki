《'''飛越十八層'''》（{{lang-en|'''You Only Live Twice'''}}）是[[香港|香港]][[電視廣播有限公司|電視廣播有限公司]]拍攝製作的時裝[[劇集|劇集]]，全劇共20集。該劇以1978年上映的電影《[[上錯天堂投錯胎|上錯天堂投錯胎]]》作為創作靈感。<ref>[http://photo.weibo.com/1893802153/wbphotos/large/mid/3746659118683802/pid/70e120a9tw1ejmay6xecej20fs0cdgnc?refer=weibofeedv5 韋家輝的大時代 (1)]</ref>

苗僑偉和戚美珍因為這齣電視劇而相識，拍拖八年後結婚至今。

== 故事大綱 ==
描述一意外橫死的青年，死後誤闖地獄，和魔鬼交易的故事。本劇從虛構的情節裡反映出現代社會的百態。 
意外橫死的黃秋發 ([[苗僑偉|苗僑偉]]飾) ，死後闖到地獄。為了還陽，黃秋發答應替魔鬼 ([[馮淬帆|馮淬帆]]飾) 收買靈魂，魔鬼將黃秋發靈魂寄身於富商商振宇([[苗僑偉|苗僑偉]]分飾)身上。黃秋發初接觸上流社會，盡享榮華富貴，不禁大喜。 化身為商振宇的黃秋發對摯友林天光 ([[廖啟智|廖啟智]]飾)照顧有加，林天光大惑不解，後來兩人終相認。黃秋發生前視女星張樂雯([[戚美珍|戚美珍]]飾)為心中女神，現藉商振宇的身份向她展開熱烈追求。另一方面，黃秋發與商振宇之管家清姐 ([[羅蘭|羅蘭]]飾)、張樂雯之弟弟國之及表妹繆詠心亦建立了深厚感情，發覺生命中最重要的不是金錢，而是人與人之間的珍貴感情。 就在此時，魔鬼苦苦相逼，要脅發盡快收買清姐、國之及詠心的靈魂，否則將他打回原形。發於心不忍，遂跟魔鬼下賭注......

== 軼事 ==
*本劇為苗僑偉與戚美珍的定情劇，二人因拍攝此劇擦出愛情火花，從而發展成情侶及後成為夫妻，偕首至今。
*本劇主題曲《難為正邪定分界》成為經典歌曲，在後來的劇集《[[老表，你好hea！|老表，你好hea]]》之中被再次演繹。早於1998年，[[黃霑|黃霑]]代替麥志誠在《[[顧嘉煇|顧嘉煇]]·黃霑·真友情演唱會》跟葉振棠合唱。
