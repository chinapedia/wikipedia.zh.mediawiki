'''飛鳳宮'''位於[[臺灣|臺灣]][[高雄市|高雄市]][[鳳山區|鳳山區]]，是主祀[[保儀尊王|保儀尊王]]（張府尊王{{NoteTag|廟方過去多以「張府尊王」稱呼主神，現則正名使用「保儀尊王」{{r|謝}}。}}、武安尊王、尊王公）的廟宇，配祀申國夫人、[[王爺神|李府千歲]]、[[註生娘娘|註生娘娘]]、[[福德正神|福德正神]]等神{{r|紅}}{{r|朱}}{{r|故事}}。該廟原本位於高雄[[小港區|小港區]]的[[紅毛港_(高雄市)|紅毛港]]，2007年因聚落遷村而移到今址，並在2012年12月15、16日舉行謝土安座儀式{{r|文}}。該廟因為原在姓楊仔聚落，又稱「姓楊仔廟」，另外又有「塭仔頭廟」的別稱{{r|紅}}{{r|謝}}。遷村之後，因故將正式廟名定為「睢陽飛鳳宮」，強調該廟主神是安史之亂中死守睢陽的[[張巡|張巡]]{{r|謝}}。

昔日飛鳳宮廟前有一菜市場，為當地居民重要活動場所之一{{r|紅}}。又該廟在紅毛港原本有外五營，但其中的東營與中營因為[[大林發電廠|大林發電廠]]建廠而拆除，南營後來也因該電廠興建排水出口而拆除，之後於第二出水口旁重建{{r|紅}}{{r|朱}}。在農曆四月廿三，飛鳳宮會請出保儀尊王副駕前往臺南[[大天后宮|大天后宮]]進香，次日返回{{r|紅}}。

== 沿革 ==
該廟相傳建於清[[光緒|光緒]]年間{{r|朱}}{{r|故事}}，但是光緒廿年（1894年）《鳳山縣采訪冊》與日昭和十三年（1938年）《臺灣宗教與迷信陋習》（曾景來 著）並未記載飛鳳宮的記錄{{r|紅}}。該廟的由來據廟方沿革碑文，是紅毛港孩童在海邊玩耍，用土偶當神像膜拜，結果有一孩童起乩，請示後乃保儀尊王要在此駐駕{{r|謝}}。之後因有靈驗事蹟，信徒乃正式建廟供奉{{r|謝}}。另一說是當地楊姓信徒自臺南迎回主神，最初是位在姓楊仔聚落的一間草寮，後來改建成石建廟宇{{r|故事}}{{r|紅}}{{r|朱}}。而因此緣故，飛鳳宮每年都會前往臺南[[尊王公壇|尊王公壇]]進香{{r|謝}}。不過飛鳳宮工務組長林坤榮否認此說，表示會前往尊王公壇只是因為居民到臺南附近捕魚時發現該廟也供奉尊王公，基於同祀情誼才進香，且飛鳳宮出訪還會前往屏東關山福德宮、臺南祀典大天后宮、高雄[[鳳邑舊城城隍廟|鳳邑舊城城隍廟]]等廟{{r|謝}}。而早期除了姓楊仔的居民之外，原本姓吳仔與姓李仔的居民亦有此廟信徒，但後來因為祭拜意見分歧等因素，姓李仔的信徒從飛鳳宮分香{{NoteTag|濟天宮主委李義三則認為「分香」之說法不妥，表示早年是李姓與楊姓居民到臺南尊王公壇迎請尊王公，後來兩姓意見不合，楊姓分得金身，李姓分得香爐，並另建濟天宮{{r|謝|page=41、42}}。}}，另建濟天宮{{r|紅}}{{r|朱}}{{r|故事}}。姓吳仔的居民後來也另建了[[紅毛港天龍宮|天龍宮]]{{r|謝}}。

民國65年（1976年）重建，歷時1年4個月後完工，是該廟遷村前最後一次重建{{r|紅}}{{r|朱}}{{r|故事}}{{r|謝}}。另外在民國六十年代（1970年代），據說當地有人家裡要娶媳婦，請主神保儀尊王去看戲，結果意外在燒香時燒到神像，於是該廟重塑金身，但將原本的神像放在新神像裡面{{r|紅}}{{r|朱}}{{r|故事}}。

2007年因紅毛港遷村而移到今址，並在2012年12月15、16日舉行謝土安座儀式{{r|文}}{{r|謝}}。而據說因為臺南大天后宮有一匾額過去不知是何廟所贈，後來查證是飛鳳宮所贈，乃請廟方取一正式廟名以利識別，後來廟方討論與擲筊後，才決定在飛鳳宮前加上「睢陽」二字{{r|謝}}。

== 祭祀圈 ==
飛鳳宮的祭祀圈在遷村之前主要是海昌里一帶，並涵蓋海澄、海豐兩里的一部份{{r|謝}}。根據[[紅毛港濟天宮|濟天宮]]主委李義三表示，遷村前飛鳳宮的丁口數為6800丁，為紅毛港各廟中最多{{r|謝}}。

== 傳說 ==
相傳過去在日治時期，紅毛港一帶曾發生瘟疫，飛鳳宮保儀尊王化身成穿白衣的大夫，步行到臺南大天后宮去求來一帖藥方，回紅毛港治病{{r|朱}}。日後保儀尊王託夢，要大家到大天后宮進香謝恩，故該廟有到大天后宮進香參拜的活動{{r|朱}}。

另外傳說日治初期[[林少貓|林少貓]]在南臺灣活動時，某日生了怪病{{NoteTag|一說是得了痔瘡{{r|紅}}{{r|故事}}。}}，在遍尋名醫診治均未見效後，轉而求神問卜{{r|紅}}{{r|朱}}{{r|故事}}。而據說若是無法解決其問題，林少貓便會焚毀神像，鞭打乩童道士{{r|紅}}。後來飛鳳宮被要求前往後壁林幫林少貓「診治」，結果保儀尊王降旨給藥，治好了林少貓的病{{r|紅}}{{r|朱}}{{r|故事}}。林少貓遂對飛鳳宮保儀尊王相當感恩，命人從打狗港出發到大陸唐山{{NoteTag|一說是親自前往香港{{r|朱}}。}}，訂製手工的「刺繡九龍綢緞涼傘」獻給飛鳳宮，且注意不讓手下打擾到紅毛港的居民{{r|紅}}{{r|朱}}{{r|故事}}。該涼傘後來據說用於神明出巡時的前導，但在林少貓勢力遭日方剿滅後，深怕遭受牽連而藏了起來，日後該傘下落不明{{r|紅}}{{r|朱}}{{r|故事}}。

另外傳說過去日治時期日本禁止傳統信仰時，紅毛港居民將廟中神像拿回中藏起來{{r|朱}}。但有一尊神像被日本人發現，用布包住丟到海中{{r|朱}}。但神像後來自己發光，又被漁民撈了回去{{r|朱}}。

又在大林火力發電廠施工拆掉飛鳳宮南營時，保儀尊王神像的鬍子翹了起來，塞到耳際{{r|朱}}。信徒去把發電廠的主任找到廟中祭拜，據說主任答應重建南營之後，神像的鬍子才恢復原狀{{r|朱}}。
