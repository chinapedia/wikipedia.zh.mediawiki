'''胡昌明'''（{{bd|1960年|||}}）[[安徽省|安徽省]][[霍山县|霍山县]]城关人。1978年入伍，[[中国共产党|中国共产党]]党员。[[中国人民解放军国防大学|中国人民解放军国防大学]]研究生学历。[[中国人民解放军少将|中国人民解放军少将]]<ref name=huo>{{Cite web |url = http://www.ahhuoshan.gov.cn/content/detail/53b2353ce1d5bde910cb4d28.html |title =霍山县志 人物 第二节 人物简介 |author =  |date = 2017-02-03 |publisher =霍山县人民政府  |language =  |accessdate =  |quote =  }}</ref>。

==生平==
曾在部队及军事院校任职，后选调到[[中华人民共和国国防部|中华人民共和国国防部]]外事办公室工作，任参谋、秘书。后升任国防部外事办公室综合局副局长。2008年，任首任国防部新闻事务局局长、国防部新闻发言人。2008年5月18日，国防部新闻发言人第一次亮相，即由胡昌明作为国防部新闻发言人在国务院新闻办公室举行的新闻发布会上介绍解放军和武警部队在[[汶川大地震|汶川大地震]]中的抗震救灾情况<ref name=huo/><ref>{{Cite web|url =http://news.xinhuanet.com/mil/2008-05/19/content_8204981.htm|title =胡昌明：生活中人情味很浓的国防部新闻发言人|author =|date =2008-05-19|publisher =新华网|language =|accessdate =|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20080531074411/http://news.xinhuanet.com/mil/2008-05/19/content_8204981.htm|archivedate =2008-05-31}}</ref>。2010年，任国防部外事办公室政策与规划局局长<ref>{{Cite web |url = http://news.163.com/10/0830/13/6FBDBJQJ000146BD.html |title = 东盟武装部队国际救灾法律规程建设研讨会开幕 |author =  |date = 2010-08-30 |publisher =网易  |language =  |accessdate =  |quote =  }}</ref>。后来曾任中国驻[[坦桑尼亚|坦桑尼亚]]大使馆首席军事专家。2013年，任国防部外事办公室副主任。2016年，任[[中央军委国际军事合作办公室|中央军委国际军事合作办公室]]副主任（国防部国际军事合作办公室副主任）<ref>{{Cite web |url =  http://www.thepaper.cn/newsDetail_forward_1453072|title = 国防部首任新闻发言人胡昌明任军委国际军事合作办公室副主任 |author =  |date = 2016-04-06 |publisher = 澎湃新闻 |language =  |accessdate =  |quote =  }}</ref>。2017年，任[[中央军委国际军事合作办公室|中央军委国际军事合作办公室]]主任<ref>{{cite web|title=中央军委国际军事合作办公室副主任胡昌明出任办公室主任|url=http://www.thepaper.cn/newsDetail_forward_1677690|publisher=澎湃新闻|accessdate=2017-05-04}}</ref>。
