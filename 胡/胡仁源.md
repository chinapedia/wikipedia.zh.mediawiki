'''胡仁源'''（{{bd|1883年||1942年|}}）字'''次珊'''，[[浙江|浙江]][[吴兴|吴兴]]人。中华民国教育家。<ref name=pk>[http://www.pku.edu.cn/about/lrxz/lrxz03.jsp 历任校长书记，北京大学，于2012-7-10查阅] {{webarchive|url=https://web.archive.org/web/20080624172940/http://www.pku.edu.cn/about/lrxz/lrxz03.jsp |date=2008-06-24 }}</ref><ref name=sh>[http://museum.sjtu.edu.cn/jdbwginfoplat/platformData/infoplat/pub/wetsite_12/hry_1719/index.html 胡仁源，上海交通大学校史博物馆，于2012-7-10查阅]</ref>

==生平==
1899年到1901年，胡仁源就读[[南洋公学|南洋公学]]师范班。<ref name=sh/>1902年中壬寅科[[举人|举人]]，入[[京师大学堂|京师大学堂]]就读，后来赴[[日本|日本]]留学[[仙台高等学校|仙台高等学校]]、赴[[英国|英国]]留学[[待爾模大学|待爾模大学]]。<ref name=xu>{{Cite book<!--|和書-->|author = 徐友春主編|title = 民国人物大辞典 増訂版|year = 2007|publisher = 河北人民出版社|isbn = 978-7-202-03014-1}}</ref>归国后，曾任[[京师大学堂|京师大学堂]]文科学長<ref name=xu/>、上海[[江南船塢|江南船塢]]副总理<ref name=xu/>、[[北京大学|北京大学]]预科及工科学长。1913年11月，代理北京大学校长。1914年1月到1916年12月，任北京大学校长。1921年到1925年，任[[国立北京交通大学|国立北京交通大学]]机械科教授、铁路管理科科长。<ref name=pk/><ref name=sh/>1926年（民国15年）3月，在[[北京政府|北京政府]][[賈德耀内閣|賈德耀内閣]]、[[胡惟德臨時内閣|胡惟德臨時内閣]]中任教育总長。同年5月辞任，后转任[[国立浙江大学|国立浙江大学]]工学院教授。<ref name=xu/><ref>{{Cite book<!--|和書-->|author = 劉寿林等編|title = 民国職官年表|year = 1995|publisher = 中華書局|isbn = 7-101-01320-1}}</ref>

1942年，胡仁源病逝，享年60岁。<ref name=sh/>
