'''胡陈乡'''是[[浙江省|浙江省]][[宁海县_(浙江省)|宁海县]]下辖的一个[[乡|乡]]，地处县境东部山区，西临[[力洋镇|力洋镇]]，南接[[长街镇|长街镇]]，东北与[[象山县|象山县]]接壤。该乡由东仓、胡陈二乡合并而成。

2010年，土地面积90.40平方千米（含内陆水域2.79平方千米）。

2016年末人口22591人，共8038户，年人均收入约3500元。

乡人民政府驻地为谐和路8号。

== 自然环境 ==
乡域西南靠[[胡陈港|胡陈港]]，东、北、西三面均为沿海丘陵，西部有部分平原。中堡溪自茅芦岗发源，自西南方向注入胡陈港，沿溪亦分布有部分平原。北部[[大丹山|大丹山]]是境内最高峰，也是宁海县境内较高的山峰。

== 农业 ==
特产[[洋芋|洋芋]]、[[杨梅|杨梅]]及[[水蜜桃|水蜜桃]]。

== 交通 ==
省道[[宁象线|宁象线]]自西向东穿越乡境，县道胡陈–长街公路和门棚–张韩公路在乡境内与宁象线交会。 

== 旅游资源 ==
* [[戊己桥|戊己桥]]
戊己桥位于乡内西南部[[胡陈港|胡陈港]]尾的[[大麦塘|大麦塘]]。桥梁俗称“四十八洞桥”，建于清[[道光|道光]]年间，为[[浙江省文物保护单位|浙江省文物保护单位]]。

* [[上宅村|上宅村]]（[[叶梦鼎|叶梦鼎]]故里）
上宅村为唐吏部尚书[[叶裕|叶裕]]和南宋右丞相[[叶梦鼎|叶梦鼎]]故里，村口有为纪念叶梦鼎衣锦还乡的“归锦桥”，亦有鱼池、假山遗址。
