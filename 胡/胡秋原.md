'''胡秋原'''（{{bd|1910年|6月11日|2004年|5月24日|catIdx=H胡}}<ref>[[何慶華|何慶華]]「胡秋原先生生平事略」　[[毛鑄倫|毛鑄倫]] 編《胡秋原先生逝世周年紀念文集》 1~10頁</ref>）原名'''胡曾佑'''，筆名'''未明'''、'''石明'''、'''冰禪'''，[[湖北省|湖北省]][[黃陂|黃陂]]人。<ref name="multiple1">{{cite web |url=http://news.ifeng.com/history/zhongguoxiandaishi/detail_2011_09/11/9107674_4.shtml|title=《揭秘：谁是“海峡两岸破冰第一人”？》|accessdate=2011年9月11日|work=秋石|publisher=凤凰网|language=zh-hans}}</ref>[[中國國民黨|中國國民黨]]黨員，曾任中華民國[[立法委員|立法委員]]，《[[中華雜誌|中華雜誌]]》發行人，[[中國統一聯盟|中國統一聯盟]]名譽主席。

==生平==
15歲時考入[[國立武昌大學|國立武昌大學]]學習[[理工|理工]]，加入[[共青團|共青團]]。1928年，在[[上海|上海]][[復旦大學|復旦大學]]改習[[文學|文學]]。1929年赴[[日本|日本]]，入[[早稻田大學|早稻田大學]]學習[[政治經濟學|政治經濟學]]。1931年後，返上海從事譯著教授，提倡抗日。1933年11月，參與[[福建|福建]]「[[閩變|閩變]]」，他自承有過「與虎謀皮的反動行為」。後去[[香港|香港]]，被[[香港殖民地時期#香港政府|香港政府]]逮捕、驅逐，遂於1934年經[[南洋|南洋]]、[[印度|印度]]、[[埃及|埃及]]至[[歐洲|歐洲]]諸國，後至[[英國|英國]]，讀書於[[圖書館|圖書館]]中。<ref name="《古今文選註》第八十一期">〈理想與勇氣〉[[作者|作者]]胡秋原傳略，載[[梁容若|梁容若]]、[[齊鐵恨|齊鐵恨]]、[[鍾露昇|鍾露昇]]、[[方祖燊|方祖燊]]編《[[古今文選|古今文選註]]》第八十一期，[[台北|台北]]，[[國語日報社|國語日報社]]，1953年4月6日。</ref>1935年赴[[蘇聯|蘇聯]]。<ref name="《古今文選註》第八十一期"/>既失望於蘇聯，1936年再經歐洲遊[[美國|美國]]。<ref name="《古今文選註》第八十一期"/>1937年由美返國，在[[漢口|漢口]]創辦《[[時代日報|時代日報]]》。<ref name="《古今文選註》第八十一期"/>後來成為中國國民黨黨員。1938年至[[重慶|重慶]]，1940年參加[[國防最高委員會|國防最高委員會]]秘書工作，旋被選為[[國民參政會|國民參政會]]參政員。<ref name="《古今文選註》第八十一期"/>戰後還鄉辦學校，1948年被選為[[立法委員|立法委員]]。其間嘗在[[暨南大學|暨南大學]]及[[復旦大學|復旦大學]]教書。<ref name="《古今文選註》第八十一期"/>1949年，前去[[香港|香港]]。1951年，到[[台灣|台灣]]從事文化教育工作，歷任[[台師大|台師大]]、[[世新大學|世新]]、[[政戰學校|政戰學校]]教授，[[中央研究院近代史研究所|中央研究院近代史研究所]]研究員。

1962年，胡秋原與《[[文星雜誌|文星雜誌]]》的[[李敖|李敖]]、[[居浩然|居浩然]]等人打筆仗官司；居浩然以連續三篇（〈科學與民主〉、〈西化與復古〉及〈從[[季米特裡·門捷列夫|門德列夫]]的週期表算起〉）釐清[[科學|科學]]、[[民主|民主]]、[[知識|知識]]等基本觀念，痛批胡秋原不懂[[普朗克常數|普朗克常數]]的定義與[[傅立葉|傅立葉]]的積分公式、還胡扯「[[量子力學|量子力學]]解釋有三派」，最後送給胡對聯：「不知所-{云}-三化論，東拉西扯一團糟」，橫批「信口胡言」。是年10月1日，李敖在《文星雜誌》第六十期發表〈胡秋原的真面目〉，對胡秋原的「閩變」叛國問題進行研究，還說胡秋原是個不誠實、愛說假話的人；胡秋原大怒，一狀告上法院，求償[[新台幣|新台幣]]800萬元。1963年，胡秋原獨資創辦《中華雜誌》以與《文星雜誌》抗衡，一場[[中西文化論戰|中西文化論戰]]遂演變成一場持續近10年的官司。1974年，李敖因叛亂罪被捕，[[台灣高等法院|台灣高等法院]]對胡秋原改判無罪。

1967年，與[[梁容若|梁容若]]發生筆戰，胡秋原批評梁容若在對日抗戰期間曾親附日本，鼓吹皇民思想，是漢奸。

1975年，在《中華雜誌》發表〈漢奸胡蘭成速回日本去〉，批評[[胡蘭成|胡蘭成]]曾在汪精衛政權下擔任官員，指他是漢奸。

在台灣1970年代[[鄉土文學論戰|鄉土文學論戰]]中，胡秋原與[[鄭學稼|鄭學稼]]、[[徐復觀|徐復觀]]等人力保[[陳映真|陳映真]]、[[黃春明|黃春明]]、[[王拓|王拓]]，對抗[[余光中|余光中]]、[[朱西甯|朱西甯]]等人對鄉土文學的攻擊。胡秋原邀請鄉土文學作家進入《中華雜誌》編輯部與發表文章，也親自寫作〈談人性與鄉土之類〉、〈談民族主義與殖民經濟〉、〈中國人立場之復歸〉登載《中華雜誌》，駁斥對鄉土文學是搞[[地域主義|地域主義]]的批評。

1988年9月12日，胡秋原到[[中國大陸|中國大陸]]訪問，與[[鄧穎超|鄧穎超]]在[[人民大會堂|人民大會堂]]見面，討論「和平統一」事宜；後被中國國民黨開除黨籍，曾擔任[[中國統一聯盟|中國統一聯盟]]名譽主席。

1991年8月，[[北京國際漢字研究會|北京國際漢字研究會]]在[[北京|北京]]參與組織召開「海峽兩岸漢字學術交流會」，胡秋原在致海峽兩岸漢字學術交流會的賀信中提出：{{Cquote|一、承認並尊重大陸現行使用中之第一批簡化字，但亦讓傳統正體字自由使用，觀察社會之自由選擇。<br>二、將大陸四十年之學術著作以正體字印刷，與台灣四十年來學術著作進行大規模交換。<br>三、鼓勵兩岸文學作品及電視電影之交流。<br>四、兩岸共同編輯字典，多載同義異字之成語。<br>五、經常舉行會議，討論中國文字之整理、中國語言文字特色之研究、以及中國文法及國文教學之改進等問題。<br>六、共同研究中國文字檢查與索隱方法之改進、機械化與輸入電腦種種問題。<br>秋原深信：經過兩岸共同的努力，必可促進中國人之真誠團結，增進了解，培養感情，加速和平統一日子早日到來；這就是中國文化復興大業的開始，也是民族重建生命偉業的完成。<ref>{{cite news|language=zh-tw|author=李敏生|title=紀念胡秋原先生|url=http://www.haixiainfo.com.tw/174-760.html|publisher=《海峽評論》第174期（2005年6月號）|date=2005-06|accessdate=2014-09-26}}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>}}

2004年5月4日，胡秋原獲頒[[中國文藝協會|中國文藝協會]][[榮譽文藝獎章|榮譽文藝獎章]]，但他因病不克領獎<ref>台北訊，[http://www.rossety.com/fokas/article.php?op=articletext&id=667&db=0&PHPSESSID=8b96b6070224db930d07862551edb4b6 〈政論家胡秋原昨晚病逝〉]，2004年5月25日《[[聯合報|聯合報]]》</ref>。2004年5月24日晚間8時，胡秋原因[[心肺衰竭|心肺衰竭]]病逝[[新北市|臺北縣]][[新店區|新店市]][[耕莘醫院|耕莘醫院]]，享年95歲；當年參與中西文化論戰的李敖表示，胡秋原一生「被國民黨開除三次、被共產黨開除兩次」、立場反反覆覆，胡秋原的[[妻子|妻子]]「卻是極好的人」<ref>2004年5月25日《[[北美世界日報|北美世界日報]]》</ref>。

== 评价 ==
1990年5月27日，在[[國立臺灣師範大學|國立臺灣師範大學]]举行的胡秋原80寿辰演讲会上，91岁的[[陈立夫|陈立夫]]赞扬胡秋原说：“秋原这个人很勇敢！有浩气，他觉得[[海峡两岸|海峡两岸]]今天应该讲和，不能再互相攻击，结果，他就不管三七二十一，一个人忽然跑到[[北京|北京]]去了……”<ref name="multiple1">{{cite web |url=http://news.ifeng.com/history/zhongguoxiandaishi/detail_2011_09/11/9107674_4.shtml|title=《揭秘：谁是“海峡两岸破冰第一人”？》|accessdate=2011年9月11日|work=秋石|publisher=凤凰网|language=zh-hans}}</ref>
