'''胡蓓蔚'''（'''Paisley Hu Pui Wei'''，{{bd|1971年|11月12日|catIdx=H胡}}），[[香港|香港]]女[[歌手|歌手]]、[[演員|演員]]、[[主持人|主持人]]。現為[[無綫電視|無綫電視]]（TVB）基本藝人合約女藝員。

== 簡歷 ==
胡蓓蔚於[[上海|上海]]長大，後來去[[美國|美國]]讀書，畢業後回到香港。小時候在上海學[[歌劇|歌劇]]。到了美國以後也學習了[[R&B|R&B]]和[[Jazz|Jazz]]。<ref>{{web reference|URL=http://www.epochtimes.com/b5/2/12/20/n257948.htm|title=香港新人胡蓓蔚進軍國語歌壇|work=大紀元|year=2002年|date=12月20日}}</ref>

在[[劉以達|劉以達]]的帶領下，參加了一個在上海舉辦的歌唱比賽從而進入樂壇。<ref>{{cite web |title= ::J-SONIC:: bio |url=http://www.j-sonic.net/paisley.html |deadurl=yes |archiveurl=https://web.archive.org/web/20070528163247/http://www.j-sonic.net/paisley.html | archivedate = 2007-05-28 }}</ref>1990年代加入樂壇，初出道時有「翻版王菲」之稱。她在1994年推出過爵士專輯，1996年加盟[[非池中唱片|非池中唱片]]<ref>{{cite web |title=《shall we talk: 別以為我無所謂––劉以達》 |url= http://swtswt.blogspot.com/2006/09/blog-post_10.html   }}</ref>，之後參與過劉以達，[[林海峰|林海峰]]和[[陳冠希|陳冠希]]的專輯。

2007年，胡蓓蔚再次復出推出新個人EP《Don't Think Just Do》，亦擔任TVB音樂節目《[[勁歌金曲|勁歌金曲]]》主持。

與[[胡杏兒|胡杏兒]]、[[胡定欣|胡定欣]]、[[姚子羚|姚子羚]]、[[黃智雯|黃智雯]]、[[李施嬅|李施嬅]]成立「胡說八道會」<ref>[http://hk.on.cc/hk/bkn/cnt/entertainment/20170730/bkn-20170730104321202-0730_00862_001.html 「胡說八道會」澳洲Trip正式出發]</ref>。

== 感情生活 ==
胡蓓蔚與音樂人及演員[[單立文|單立文]]於1996年年底開始交往。2008年3月，二人結婚<ref>{{cite web |title=汪琳下嫁浪漫才子 胡蓓蔚 單立文低調拉埋天窗 - 香港文匯報 |url=http://paper.wenweipo.com/2008/03/29/EN0803290001.htm |date=2008-03-29}}</ref><ref>{{cite web |title=《單立文娶胡蓓蔚娛圈喜事多籮籮》 |work=《明報》|date=2008年3月29日 |url= http://hk.music.yahoo.com/magazine-news-article.html?uid=272626 }}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。
