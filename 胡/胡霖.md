'''胡霖'''（{{bd|1889年||1949年|4月14日|catIdx=Hu胡}}），字政之，笔名冷观<ref name="“报界巨子”胡政之在安庆">{{cite web|url =http://unn.people.com.cn/GB/22220/104368/7453119.html|title = “报界巨子”胡政之在安庆|date =2008年7月1日 |language =zh-hans  |publisher =[[人民网|人民网]]转载[[安庆晚报|安庆晚报]]}}</ref>。[[四川|四川]]成都人。民国著名记者，报人，新记《[[大公报|大公报]]》创办人之一。大公报也正是在胡政之和[[张季鸾|张季鸾]]以及[[吴鼎昌|吴鼎昌]]主持下，成为[[中华民国|中华民国]]大陆统治时期，尤其[[抗日战争|抗日战争]]时期最重要的民间舆论重镇。胡政之在新闻史上既是大公报的代表人物之一，也成为了一个时代报刊的象征性人物。

==生平==
1889年出生官宦家庭，1906年，胡政之父亲因病死于[[五河县|五河县]]知县任上时，17岁的胡政之不得中途辍学，离开[[安庆省立高等学堂|安庆省立高等学堂]]而扶柩回四川。1907年自费赴日留学，入[[東京大學|东京帝国大学]]研习法律。1911年归国，一度曾开办过[[律师事务所|律师事务所]]<ref name="“报界巨子”胡政之在安庆"/>。

1912年应聘出任上海《[[大共和报|大共和报]]》日文翻译，后继任编辑、主笔，编发专电，兼写[[社论|社论]]。

1915年被派驻[[北京|北京]]，以消息快捷受到业内注意。1916年[[安福系|安福系]][[王郅隆|王郅隆]]接办大公报後聘为经理兼总编，开始对《大公报》进行改革，将原有记者开除，同时聘请[[林白水|林白水]]、[[梁鸿志|梁鸿志]]、[[王峨孙|王峨孙]]等为特约访员，每天以电话向[[天津|天津]]发消息，或以快邮寄稿，新闻因此大有改观。同时对版面进行了改革，抛弃旧式书册，改用西式通栏，版面设计都不断提高。在此后一系列事件，如[[张勋复辟|张勋复辟]]、[[马场誓师|马场誓师]]和[[新文化运动|新文化运动]]中都发表过不少有分量的报道和文章。同时也密切注意国际动向，1918年作为大公报记者前住欧洲采访[[巴黎和会|巴黎和会]]，也是与会的唯一中国记者<ref>{{cite web|url =http://v.ifeng.com/his/200812/72955dad-af91-4cd9-a912-1fe07188ebfc.shtml|title = 胡政之采访巴黎和会 亲睹国权沦丧|date =2008年12月 |language =zh |publisher =[[凤凰卫视|凤凰卫视]]}}</ref>。1920年7月回国后不久辞职，后在[[上海|上海]]组织国闻通讯社。

1926年9月[[吴鼎昌|吴鼎昌]]、[[张季鸾|张季鸾]]、胡政之合组[[新记公司|新记公司]]，接办《大公报》。

1945年4月，[[联合国|联合国]]成立大会在[[美国|美国]][[旧金山|旧金山]]举行，他以中国新闻界代表和国民参政会参政员之身份，作为中国代表团成员参会，并在《[[联合国宪章|联合国宪章]]》上签字。

1948年胡政之亲自主持《大公报》[[香港|香港]]版复刊。4月，胡政之突然病发，离开香港飞回[[上海|上海]]就医。辗转病榻一年後，1949年4月14日凌晨三時五十五分在上海寓所病逝去世<ref>{{cite web|url =http://www.takungpao.com.hk/news/09/04/14/TK-1065187.htm|title =胡政之在上海病逝前後──紀念胡政之逝世六十周年|author =王鵬|date =2009年4月14日|language =zh-hant|publisher =[[大公报|大公报]]|accessdate =2009-06-08}}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

==家庭==

*妻子：顾俊琦（胡的续弦，抗战初期成婚，[[顾维钧|顾维钧]]的内侄女。1951年春只身出走香港，后前往[[美国|美国]]）

==评价及影响==
胡政之被认为是一位多才多能的报业工作者。是一位优秀的记者，同时也是一位杰出的时事评论员<ref>{{cite web|url =http://french.2007afdb.org/BIG5/shizheng/252/8397/8399/20020617/754696.html|title =《胡政之先生紀念文集》序|language =zh-hant|date =2002年6月17日|publisher =[[人民网|人民网]]转自香港大公报|accessdate =2009-06-08|author =[[方漢奇|方漢奇]]}}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。通常认为，开创新记《大公报》三巨头中只有张季鸾以文章名世，但如果没有胡政之的经营管理、吴鼎昌的资本，《大公报》的成功也是无法想象的。三个人中，胡政之主持《大公报》的时间最长，从1926年到1949年，如果再加上1916到1919年，先后达27年之久。对胡的贡献常用管理来加以概括<ref>{{cite web |url = http://www.people.com.cn/GB/14677/22114/26470/26471/1751684.html|title = 胡政之：开创百年报业“新路径”|language = zh-hans |publisher =人民网}}</ref>。在胡的经营下，津版大公报从三人复刊时职工不过70人，月支出不满6000元，发行量仅2000多份的惨淡局面，快速成长并扩张，到1936年上海版创刊，职工已达700人，月支出10万元，全国分销机构1300多处，发行量超过10万份，一跃成为具有全国影响的大报。胡政之不仅精于经营，还善于管理。对人才录用，任人唯贤，亲自制定《大公报社职工任用及考核规则》，其中包括6项考核标准、3项奖惩办法。职工福利也有规定，职工享受恤养金、子女教育补助、医药补助、婚丧补助等。此外，还有“年终奖金惯例”等一整套完善的人事制度。这些制度始终得到贯彻执行<ref>{{cite web|url =http://www.022net.com/2008/2-1/442921112368885.html|title =《大公报》的创办人 中国报业经营奇才胡政之|publisher =人民网天津视窗|language =zh-hans|date =2008年2月1日|deadurl =yes|archiveurl =https://web.archive.org/web/20081223185037/http://www.022net.com/2008/2-1/442921112368885.html|archivedate =2008年12月23日}}</ref>。但近些年学界也开始对其早期独自经营大公报时期的作用开始重新审视<ref>{{cite web |url = http://news.sina.com.cn/cul/2005-03-01/4364.html |author =付阳、王瑾|title = 胡政之1916开始对《大公报》的革新|publisher =[[书屋|书屋]]|language =zh-hans |date =2005}}</ref>。胡政之的去世，也被视为中国最好报纸的谢幕，成为大公报命运的一个象征性事件<ref>{{cite web|url =http://news.xinhuanet.com/newmedia/2006-01/04/content_4005294.htm|author =徐百柯|title =胡政之：报人逝，报纸亡|publisher =新华网转载《[[中国青年报|中国青年报]]》|language =zh-hans|date =2006年1月4日|accessdate =2009-06-08|deadurl =yes|archiveurl =https://web.archive.org/web/20090504130808/http://news.xinhuanet.com/newmedia/2006-01/04/content_4005294.htm|archivedate =2009年5月4日}}</ref>。

胡政之一生，有两件事最为后人争议（尤其是中共方面），一是他为了《大公报》的发展，曾经向国民政府申请过20万美元的公价外汇，这被许多人指为是受国民党的津贴；二是在国共和谈将破裂的时候，[[蒋中正|蒋中正]]要召开[[制憲國民大會|制憲國民大會]]，制定[[中華民國憲法|宪法]]，胡政之以「[[制憲國民大會代表#社會賢達|社会贤达代表]]」身份，参加了会议<ref>{{cite web |url = http://www.people.com.cn/GB/14677/22114/26470/26501/1752051.html|title =  胡政之和胡适之|author =谢泳|language = zh-hans |publisher =人民网}}</ref>。然而相關爭議或批評，进入21世纪后，在中國大陸史学界和新闻界又有了新的诠释和评价。
