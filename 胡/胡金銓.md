'''胡金銓'''（{{lang-en|King Hu}}，{{bd|1932年|4月29日|1997年|1月14日}}），香港和台灣的武侠[[電影|電影]][[導演|導演]]，出生於[[北京|北京]]<ref>{{Cite web|url=http://news.mtime.com/2015/03/28/1540932-all.html|title=儒侠胡金铨的红与黑 《侠女》等4部佳作背后的故事|accessdate=2018-04-15|date=2015-04-08|publisher=Mtime时光网|archive-url=https://web.archive.org/web/20180415124932/http://news.mtime.com/2015/03/28/1540932-all.html|archive-date=2018-04-15|dead-url=no}}</ref><ref>{{Cite news|url=https://www.nytimes.com/1997/01/17/arts/king-hu-65-maker-of-kung-fu-films.html|title=King Hu, 65, Maker Of Kung Fu Films|author=|date=1997|work=|newspaper=The New York Times|accessdate=2018-04-15|language=en-US|archive-url=https://web.archive.org/web/20180216070735/http://www.nytimes.com/1997/01/17/arts/king-hu-65-maker-of-kung-fu-films.html|archive-date=2018-02-16|dead-url=no}}</ref>，籍貫[[河北|河北]][[永年|永年]]。他是新派[[武侠电影|武侠电影]]的先驱人物，将中国传统的[[戏曲|戏曲]]元素纳入电影之中，开创了自己独特的电影风格，是中国最早具有世界性声誉的导演之一。

== 生平 ==
胡金銓出生於[[北京|北京]]的書香門第，其祖父为清末[[山西巡抚|山西巡抚]][[胡景桂|胡景桂]]<ref>[http://ny.stgloballink.com/community/201105/t20110502_1557060.html 鄭佩佩籲有心人專研胡金銓導演風格]{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。后前往河北省井陉煤矿坑，在那就读幼稚园，七七事变后返回北京。初中、高中就讀北平[[匯文中學|匯文中學]]。

1950年，胡金铨只身离开北京前往[[香港|香港]]，原本想留学美国，并已经拿到了入学申请书，但由于[[国共内战|国共内战]]无法成行。胡金铨又想报考香港的理工院校，但不被获准，由于未带生活费来港，从而转向就业，入[[嘉華印刷廠|嘉華印刷廠]]當助理[[會計|會計]]兼[[校對|校對]]，與[[宋存壽|宋存壽]]同事，但因脾气倔强遭解雇。之后前往某广告公司任廣告畫師，繪電影廣告。后由[[蒋光超|蒋光超]]介绍，转入[[费穆_(导演)|费穆]]主持之龙马电影公司担任广告、海报、布景等工作，但因不懂电影工作半个月后又被解雇。在绘制电影海报、广告期间，胡金铨同时兼任家庭教师，学生沈家乐的父亲沈天荫看到其绘制的海报后颇为赏识，胡金铨从而进入[[长城制片公司|长城制片公司]]美工科担任陈设，[[万古蟾|万古蟾]]为其主任，胡金铨与[[李翰祥|李翰祥]]两人结识。在此期间，胡金铨为《一板之隔》和《一家春》等电影担任了道具工作。

1953年，胡金铨与[[宋存壽|宋存壽]]、[[蒋光超|蒋光超]]、[[李翰祥|李翰祥]]等七人结为异姓兄弟，号为“七大闲”，其中胡金铨与李翰祥关系最为密切。胡金铨由长城电影公司转到[[永華影業公司|永華影業公司]]后，由李翰祥推荐，胡金铨在[[嚴俊_(演員)|严俊]]导演的《吃耳光的人》《笑声泪痕》中担任主要演员，开始从事演员工作，同时兼职做广播员，并在“美国之音”中结识了[[邹文怀|邹文怀]]，在此期间，胡金铨还将自己的汽车借给邹文怀和李翰祥，帮助他们结婚<ref name="梦">{{Cite web |url=http://data.book.hexun.com/chapter-2132-4-1.shtml |title=我的香港电影梦：胡金铨 |access-date=2019-02-08 |archive-url=https://web.archive.org/web/20181112101238/http://data.book.hexun.com/chapter-2132-4-1.shtml |archive-date=2018-11-12 |dead-url=no }}</ref>。1958年，經[[李翰祥|李翰祥]]介紹入[[邵氏公司|邵氏公司]]任演員，後兼任編劇及助導。1963年在《梁山伯與祝英台》（[[李翰祥|李翰祥]]導演）中名義上任副導演，實際上是分擔李翰祥的導演工作。1964年編導《玉堂春》，這是他在李翰祥策劃下第一次做執行導演之作，是部黃梅調古裝喜劇。1965年正式獨立執導抗戰片《大地兒女》，卻迫於邵氏上層壓力大幅刪改劇情。1966年導演《[[大醉俠|大醉俠]]》，不僅奠定後來的作品風格，亦影響中國武俠片甚深。翌年離開邵氏公司，赴臺籌組[[聯邦公司|聯邦公司]]事業部。

1967年在臺執導《[[龍門客棧|龍門客棧]]》，票房極佳，使他能夠開拍雄心更大的《[[俠女|俠女]]》。《俠女》製作歷時三年，直至1971年方完成，並分成上下兩部放映。1975年《俠女》獲得第二十八屆[[坎城影展|坎城影展]]最高技術委員會大獎，一躍成為國際知名導演。同年攝製《[[忠烈圖|忠烈圖]]》，企圖追求較前此武俠作品更高的意境。1977年胡金銓赴韓拍攝《[[空山靈雨|空山靈雨]]》和《[[山中傳奇|山中傳奇]]》兩片，終以《山中傳奇》獲第十六屆[[金馬獎|金馬獎]][[金馬獎最佳導演|最佳導演]]、[[金馬獎最佳美術設計|最佳美術設計]]獎。1978年入選[[英國|英國]]「電影指南」當年「世界五大導演」<ref>{{Cite news|url=https://www.cw.com.tw/article/article.action?id=5035708|title=俠女．客棧．影人生　胡金銓|author=邱花妹|date=2012-06-25|newspaper=天下雜誌|accessdate=2018-04-15|language=zh-TW|archive-url=https://web.archive.org/web/20180415190346/https://www.cw.com.tw/article/article.action?id=5035708|archive-date=2018-04-15|dead-url=no}}</ref><ref>{{Cite web|url=http://yule.sohu.com/20071224/n254273902.shtml|title=长使英雄泪满襟  十年回望胡金铨|accessdate=2018-04-15|date=2007-12-24|publisher=搜狐娱乐|archive-url=https://web.archive.org/web/20180415190517/http://yule.sohu.com/20071224/n254273902.shtml|archive-date=2018-04-15|dead-url=no}}</ref><ref>{{Cite news|url=http://www.chinatimes.com/realtimenews/20140603003788-260408|title=胡金銓回顧展 15部名作紐約上映|author=吳垠慧|date=2014-06-03|newspaper=中時電子報|accessdate=2018-04-15|language=zh-TW|archive-url=https://web.archive.org/web/20180415190452/http://www.chinatimes.com/realtimenews/20140603003788-260408|archive-date=2018-04-15|dead-url=no}}</ref>。

除富於禪意的古裝片之外，胡金銓也曾於1981年嘗試拍攝時裝喜劇《終身大事》。1992年獲[[香港導演協會|香港導演協會]]頒贈終身榮譽大獎。長年籌拍《華工血淚史》，原訂1997年7月開鏡，卻在該年1月因心導管擴張手術失敗逝世，享年六十五歲<ref>{{Cite web|url=http://nrch.culture.tw/twpedia.aspx?id=21190|title=胡金銓|accessdate=2018-04-15|publisher=臺灣大百科全書|archive-url=https://web.archive.org/web/20180415124958/http://nrch.culture.tw/twpedia.aspx?id=21190|archive-date=2018-04-15|dead-url=no}}</ref>。

== 成就 ==

《大醉侠》是胡金铨早期电影的代表作，也是华语电影中最早使用武术指导的电影<ref name="儒侠">{{Cite web |url=http://news.mtime.com/2015/03/28/1540932-all.html |title=儒侠胡金铨的红与黑 |access-date=2019-02-08 |archive-url=https://web.archive.org/web/20181001104237/http://news.mtime.com/2015/03/28/1540932-all.html |archive-date=2018-10-01 |dead-url=no }}</ref>，成为了20世纪60年代香港新派武侠电影的先驱之作，与[[张彻|张彻]]、[[楚原|楚原]]等人一道改写了当时的武侠片风格。1971年，胡金铨拍摄出了《侠女》，这部电影在戛纳电影节获得了最高技術委員會獎，使其本人赢得了世界性的声誉。

胡金铨注重电影中的艺术性，其电影中借鉴了许多传统戏曲中的元素，尤其是其电影配乐中所用京剧的锣点声尤为人所称道。胡金铨在明史上颇有研究，所拍武侠电影时代也多集中在明朝，其电影中许多服装、房屋等道具的设计细节都经过其仔细的考究。在业界，胡金铨以认真著称，在拍摄《侠女》的过程中花费上千万台币耗时九个月制作古街道，在拍摄影片的过程中也往往为了等合适的天气而耗费大量时间。

胡金铨的电影被许多电影人推崇，[[李安|李安]]、[[徐克|徐克]]、[[许鞍华|许鞍华]]、[[侯孝贤|侯孝贤]]等导演都受其影响，与其多次合作的[[徐枫|徐枫]]、[[上官灵凤|上官灵凤]]、[[郑佩佩|郑佩佩]]、[[白鷹_(演員)|白鹰]]、[[石隽|石隽]]等演员也多受其提携。

== 评价 ==
* {{tsl|en|David Bordwell|大卫·波德维尔}}在其著作《电影诗学》中评价胡金铨为“亚洲最优秀的导演之一”，认为其电影“表现出一种挥洒自如的精湛的专业风格，却又甚少学究气”<ref name="电影诗学">{{Cite book |title=电影诗学 |publisher=广西师范大学出版社 |year=2010年}}</ref>。
* 在纪录片《艺术的追寻：李安谈胡金铨》中，[[李安|李安]]表示胡金铨是中国最早注意到电影的艺术性的人之一，并有志于在电影中表现中国的传统文化，认为自己受到其影响，尤其是受胡金铨电影中的“气韵”影响最深。在其影片《[[卧虎藏龙|卧虎藏龙]]》中，致敬了胡金铨电影《侠女》中著名竹林戏场景。
* 在胡金铨纪念展上，[[徐克|徐克]]接受采访说到《[[新龙门客栈|新龙门客栈]]》中的金镶玉相当于是对胡金铨导演的“某种致敬”，是闯入胡金铨电影世界的某种行为的写照。他认为《侠女》中的场景“到现在没有人可以重做出来”，胡金铨是“一个比我们强很多倍的电影界的巨人”。

== 家庭 ==
胡金铨其祖父为山西巡抚胡景桂。父亲胡源深，字海青，曾留学日本[[京都帝国大学|京都帝国大学]]，学习煤矿专业，在井陉煤矿工作过，后被关入热河劳改营，在20世纪50年代去世<ref name="梦" />。
