'''胡美倫''' ( Wu Mei Lun，1908年 - 1985年2月24日 )，[[香港|香港]][[粵曲|粵曲]]歌手，[[電影|電影]][[演員|演員]]。胡美倫在20世紀40年代自組「明月音樂社」，常到[[香港電臺|香港電臺]]演出，[[粵曲|粵曲]][[歌壇|歌壇]]有「中板王」之美譽，她以中氣十足，發音清晰，抑揚疾除，腔圓調潤，而被稱之為「粵曲金嗓子」<ref>1977年3月5日(星期六)，[[香港|香港]]《[[華僑日報|華僑日報]]》，第七張，第二版。</ref>。

== 家庭 ==
* 自與勞先生結婚後，即息影，專心相夫教子，只偶爾為[[慈善|慈善]]客串演出；
* 兩人育有一子兩女，么女[[勞韻妍|勞韻妍]]為[[香港電臺|香港電臺]]播音員及[[龍翔劇團|龍翔劇團]]花旦<ref>1985年3月5日(星期二)，[[香港|香港]]《[[華僑日報|華僑日報]]》，第四張，第二版。</ref>。

== 心臟病突發 ==
* 1985年2月24日(年初五，[[星期日|星期日]])下午4時30分，在[[醫院|醫院]]因[[心臟病|心臟病]]突發逝世，享年77歲。1985年3月7日(星期四) 中午12時30分在[[香港殯儀館|香港殯儀館]]以[[佛教|佛教]]儀式大殮，下午1時出殯<ref>1985年3月5日(星期二)，[[香港|香港]]《[[華僑日報|華僑日報]]》，第四張，第二版。</ref>。

== 電影 ==
* 胡美倫的銀幕處女作《妻多夫賤》(1936年，飾演第七妾待)；
* 胡美倫的息影作《金屋藏嬌》(1952年，飾演朱太太)；
* 胡美倫一生共參演超過70多齣[[電影|電影]]。

== 粵曲 ==
* [[美國|美國]]勝利唱機公司《月照梨魂》：胡美倫獨唱；
* 《江山美人》：撰曲：[[胡文森|胡文森]]；演唱：胡美倫、[[徐柳仙|徐柳仙]]；
* 《珠聯璧合》：撰曲：[[朱老丁|朱老丁]]；演唱：胡美倫、[[譚伯葉|譚伯葉]]；
* 《芳草送斜陽》：胡美倫、[[張月兒|張月兒]]；
* [[上海|上海]]百代公司唱片《地久天長》：胡美倫、[[桂名揚|桂名揚]]；
* 《大鬧梅知府》：胡美倫、[[半日安|半日安]]；
* 《花開富貴》：胡美倫獨唱；
