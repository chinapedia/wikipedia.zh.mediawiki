'''琵琶湖'''（{{jpn|j=琵琶湖|hg=びわこ|rm=Biwa Ko}}）位於[[日本|日本]][[滋賀縣|滋賀縣]]，為日本最大的[[湖泊|湖泊]]；日本{{link-ja|湖沼水質保全特別措置法|湖沼水質保全特別措置法}}指定湖泊，也列入[[濕地公約|濕地公約]][[國際重要濕地名錄|國際重要濕地名錄]]中。
== 地理 ==
[[File:Mt_ibuki02s3200.jpg|thumb]]]]
琵琶湖佔有滋賀縣六分之一的面積，總面積670.33平方公里，湖岸長241公里，最深103.58米，平均水深41.2米 。

自琵琶湖流出的河流依上下游的不同依序稱為[[瀨田川|瀨田川]]、[[宇治川|宇治川]]、[[淀川|淀川]]、最後流入[[大阪灣|大阪灣]]。此外為了供應[[京都市|京都市]]之[[自來水|自來水]]以及灌溉、發電等種種目的，因而開闢有[[琵琶湖疎水|琵琶湖疎水]]這條人工水道。

在湖頸段的最狹處建有[[琵琶湖大橋|琵琶湖大橋]]，以北的部分稱為'''北湖'''，以南的部分稱為'''南湖'''，北湖面積約為南湖的11倍<ref>[http://www.jstage.jst.go.jp/article/rikusui/67/2/67_123/_article/-char/ja 《陸水学雑誌》〈琵琶湖南湖の面積について〉，芳賀裕樹，2006年]</ref>，最深處位於北湖。

湖水的來源主要為周圍的山地，湖水供應[[京阪神地方|京阪神地方]]的居民用水，也因為經濟的高度成長，導致水質污濁與[[富養化|富養化]]，因此設有琵琶湖條例保護湖水的潔淨。

自古以來為重要的水上交通要道，在鐵路開通以前為日本東部與[[北陸地方|北陸地方]]運輸要道。

== 歷史 ==
琵琶湖大約成形於400萬年前（也有一說為600萬年前），因為[[地殼|地殼]]的變動在現今的[[三重縣|三重縣]]上野地方形成。之後逐漸向北移動至現今的位置。在世界的湖泊中僅次於[[貝加爾湖|貝加爾湖]]、[[坦干依喀湖|坦干依喀湖]]是第三古老的湖泊。

* 1890年供應京都市用水的琵琶湖疎水開通。
* 1950年7月24日成立[[琵琶湖國定公園|琵琶湖國定公園]]。
* 1964年9月琵琶湖大橋開通。
* 1980年制訂琵琶湖條例。
* 1993年列入[[濕地公約|濕地公約]]國際重要濕地名錄中。
* 2003年琵琶湖觀光利用條例施行。

== 生物 ==

[[File:Fish_from_Lake_Biwa_for_sale_at_a_fish_store_in_Otsu,_Shiga,_Japan.jpg|thumb]]
琵琶湖的[[生態體系|生態體系]]相當的豐富化，有超過1000種動植物生長其中。魚類約有46種，貝類約40種，水草約70種，因而被稱日本淡水魚的寶庫。琵琶湖的淡水珍珠養殖也相當有名。

== 琵琶湖八景 ==
1945年6月經由公開徵求選出：
*煙雨－[[比叡山|比叡]]的樹林
*夕陽－[[淀川|瀨田]][[石山寺|石山]]的清流
*涼風－[[雄松崎|雄松崎]]的白汀
*曉霧－[[海津大崎|海津大崎]]的岩礁
*新雪－[[賤岳|賤岳]]的大觀
*明月－[[彥根城|彥根的古城]]
*深綠－[[竹生島|竹生島]]的倒影
*春色－[[安土町|安土]][[近江八幡市|八幡]]的水鄉

==活動==
*自1977年7月2日起，每年的七、八月間舉辦[[日本國際鳥人錦標賽|日本國際鳥人錦標賽]]。
*2006年的日本電影《[[大奧_(2006年電影)|大奧]]》的拍攝場景。
*[[日本競艇|日本競艇]]的24個競艇場之一的所在地。
