{{Vgname|坦克大战|ja=バトルシティー|en=Battle City}}是一款平面[[射击游戏|射击游戏]]。此[[红白机|红白机]]游戏是1985年出版于日本开发商[[南梦宫|南梦宫]]（Namco）。此后，这游戏又在[[Game_Boy|Game Boy]]出版。这游戏是模仿1980年[[街机游戏|街机游戏]]《{{lang|en|Tank Battalion}}》而制作的。《{{lang|en|Tank Battalion}}》也是南梦宫出版。1991年，南梦宫出版了街机续篇《坦克力量》（{{lang|en|Tank Force}}）。

== 剧情 ==
像其它的早期FC游戏，《坦克大战》没有固定的故事情节。当玩家打过了所有的关后，游戏从第一关重新开始。

== 游戏流程 ==
[[File:Battle_City_video_game.jpg|thumb]]
在此游戏里，玩家操作一辆坦克，分別為黃色和綠色坦克，必须消灭所有电脑控制的敌军坦克，而且保护自己的基地。在游戏中，基地的图标是一支老鹰。每一关有20辆敌军坦克。如果他的基地、老鷹被射中、或玩家的所有坦克全部被擊毁，遊戲便告終止。

《坦克大战》红白机原版共有35关。所有的战场面积为13乘13。每一关的地形和障碍都有区别。比方说，砖墙可以被双方的任何坦克销毁，但是钢铁墙只能被玩家升级后的坦克毁灭。树林可以挡住玩家的视线，但是对电脑控制的坦克毫无影响。坦克不能跨过水面但是子弹可以，雪地會使戰車變滑。敌军坦克有四种，区别于装甲。电脑智能也慢慢的按关数提高。闪烁红光’的敌军坦克被消灭后会丢下可以帮助玩家的物品，如加一条命、坦克升级、基地墙壁升级等等。

某些修改的版本可以得到船来过水面，第三级坦克加一个星也可以清除树木，以及敌方可以得到宝物等，增加了游戏的难度。有些版本甚至加入了导弹，和搬运钢铁墙加固基地的功能。现时网络流传的有《烟山坦克》。

《坦克大战》也是第一个可以同时[[双打|双打]]的红白机游戏。两玩家同时配合消灭军。玩家甲操作黃色坦克，玩家乙則操作綠色坦克。在某些版本中，如果玩家击中了战友，他会使战友的坦克暂时停止运行。如果射得太多，更會將戰友的坦克射死。此外，如於雙打遊戲中完成一關後，在該關比戰友擊殺較多敵車的話，能得到一千分獎勵。如果打和，則雙方皆得不到
《坦克大战》Game Boy版本有100关。Game Boy版的难度比游戏机版要高。由于Game Boy的屏幕小，Game Boy版不显示整个战场。在战场的某些角落，玩家失去基地的视野。

虽然《坦克大战》红白机原版只有35关，它在红白机平台上还有众多衍生版。而且，《坦克大战》也是第一个附有编辑器的红白机游戏。编辑器让玩家自意地建造自己的战场。

== 寶物(包括修改版) ==

* 戰車
: 玩家可以多一條命，另外分數超過20000分也會多一條命。反之，敵方吃了會增加防禦力。
* 星星
: 可使玩家的戰車升級，吃1次可使子彈變快，吃2次可以連續發射2發子彈，吃3次可打掉鐵牆，被敵方射中一律重來。反之，敵方吃了1次可以打掉鐵牆。
* 炸彈
: 可使敵方的戰車全部炸毀，但分數只有算炸彈的500分。反之，敵方吃了會把玩家炸掉。
* 時鐘
: 可使敵方的戰車全部停止（停止後的戰車不會發射子彈），過一段時間後又會恢復。反之，敵方吃了會把玩家停止。
* 鋼盔
: 玩家的戰車周圍會有防護牆，可以保護戰車不會被敵方的戰車射到，過一段時間後又會恢復。相当于[[红色警戒系列|红色警戒系列]]的铁幕。反之，敵方吃了會增加防禦力。
* 鏟子
: 可使老鷹周圍的磚牆變成鐵牆，過一段時間又會恢復成磚牆。反之，敵方吃了會把老鷹周圍變成都沒有保護且永久無法變回去。
* 手槍
: 在某些修改版中出現，可令玩家的坦克直接變成終極形態，但被敵方射中會縮減變成吃2個星星的戰車。反之，敵方吃了也會直接變成終極形態且要攻擊4下。
* 船艇
: 在某些修改版中出現，可令玩家的坦克在水上移動。反之，敵方吃了也會在水上移動。

== 敵方戰車 ==
===原版===
* [[輕戰車|輕戰車]]（Light Tank）
:與沒有吃星星的戰車差不多，一台為100分。
* [[裝甲車|裝甲車]]
:所有戰車裡速度最快的戰車（因為底部是輪子），也因為它速度快，大陸人就稱它為「小快車」，一台為200分。
* [[速射砲戰車|速射砲戰車]]
:與「一般戰車」差不多，尾部有多了一個像「J」的符號，子彈速度與吃1顆星星的戰車一樣，一台為300分。
* [[重型戰車|重型戰車]]
:大陸人稱它為「大鐵皮」，因為它擁有強大的裝甲，要攻擊4下，每攻擊一下顏色分別以淺綠色→淺黃色→黃綠色→白色→消失，一台為400分。

===修改版本===
原版的四種戰車隨機出現，而且原先重型戰車的攻擊4下才會消失特性改為所有戰車隨機攻擊1到4下才會消失，但是分數不變。然後有版本敵方戰車也能吃寶物，另外裝甲車和重型戰車子彈速度與速射砲戰車一樣快，如果吃到手槍敵方戰車外型會變成我方第4等級戰車外型，分數有1410分，然而被1P擊殺不算擊殺數，被2P擊殺算1P擊殺數。
