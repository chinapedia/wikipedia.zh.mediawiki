'''坦佩雷-皮尔卡拉机场'''（{{lang-fi|Tampere-Pirkkalan lentoasema}}，{{lang-sv|Tammerfors-Birkala flygplats}}，[[IATA机场代码|IATA]]：'''TMP'''，[[ICAO机场代码|ICAO]]：'''EFTP'''），位于[[芬兰共和国|芬兰共和国]][[西芬兰省|西芬兰省]]的[[皮尔卡拉|皮尔卡拉]]市。该机场主要为[[坦佩雷|坦佩雷]]市服务，距离坦佩雷市区大约17千米。

== 历史 ==
坦佩雷的机场最早建于1936年，位于距市区6千米的Härmälä区。当时[[芬兰航空|芬兰航空]]的前身Aero Oy开通了由坦佩雷往返[[赫尔辛基|赫尔辛基]]、[[瓦萨|瓦萨]]、[[奥卢|奥卢]]和[[凯米|凯米]]等地的航班。直到1979年，新建成的坦佩雷-皮尔卡拉机场替代了Härmälä机场。

== 现状 ==
现在的坦佩雷-皮尔卡拉机场是芬兰第三繁忙的机场和第二繁忙的国际机场。近年来，旅客运输量逐年迅速增长。2007年运送旅客68.8万人次。<ref>{{cite web |url=http://www.finavia.fi/airport_tampere-pirkkala |title=Finavia - Tampere-Pirkkala airport |publisher=finavia.fi |date= |accessdate=2008年11月27日 |language=en |deadurl=yes |archiveurl=https://web.archive.org/web/20081217034515/http://www.finavia.fi/airport_tampere-pirkkala |archivedate=2008年12月17日 }}</ref>这很大程度上归功于[[廉价航空公司|廉价航空公司]][[瑞安航空|瑞安航空]]开通了该机场到[[欧洲|欧洲]]各地的航班。

坦佩雷-皮尔卡拉机场现有两座航站楼。其中2号航站楼是将一处旧航站楼重新启用，成为廉价航空专用航站楼，目前由瑞安航空使用。其他航空公司使用1号航站楼。

坦佩雷-皮尔卡拉机场也是[[芬兰空军|芬兰空军]]萨塔昆塔司令部的驻地，驻扎有[[F/A-18大黄蜂战斗攻击机|F/A-18大黄蜂战斗攻击机]]。
