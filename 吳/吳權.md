'''吳權'''（{{vie|v='''Ngô Quyền'''|hn=吳權}}，{{bd|898年||944年|catIdx=N}}<ref>《東南亞歷史詞典·吳權條》，上海辭書出版社版，208頁。</ref>），[[越南|越南]][[吳朝|吳朝]]的開創者，被稱為'''前吳王'''（{{vie|v=Tiền Ngô Vương|hn=前吳王}}），亦被稱為'''吳先主'''，出身於[[唐朝|唐]]末[[五代|五代]]時期[[交趾|交趾]]地方豪族世家，原為該地自主政權統治者[[楊廷藝|楊廷藝]]牙將。[[大有_(年號)|大有]]十年（937年），南漢的[[交州|交州]]發生兵變，屬將[[矯公羨|矯公羨]]殺死了主管官員，交趾統治者楊廷藝，割據一方，另一屬將'''吳權'''領兵攻打矯公羨，矯公羨便求救於[[劉龑|劉龑]]，劉龑封兒子[[劉洪操|劉洪操]]為交王，然後領兵進攻吳權，結果被吳權打敗，劉洪操陣亡。吳權從此佔有了交州，吳權建立的王朝即[[越南|越南]][[吳朝|吳朝]]。造就[[越南|越南]]脫離[[中國|中國]]統治，被後世視為古代越南獨立建國的重要人物。

== 早年生活 ==
據《[[大越史記全書|大越史記全書]]》所載，吳權是[[交趾|交趾]]唐林人（[[陳仲金|陳仲金]]《[[越南史略|越南史略]]》說唐林位於[[越南|越南]][[山西省_(越南)|山西省]][[福壽縣|福壽縣]]，現今屬[[河内市|河内市]][[山西市社|山西市社]]；《[[安南志略|安南志略]]》則說吳權是[[愛州|愛州]]人），「世為貴族」，父親吳旻曾任本州牧。《大越史記全書》又說吳權出生時便有好兆頭，「'''王'''生有異光滿室，狀類特殊，背上有三黑子，相者奇之，謂可為一方之主，乃命名曰'''權'''」。史藉描述成長的吳權儀表不凡和智勇雙全，「及長魁悟（奇偉也），目光若電，緩步如虎，有智勇，力能扛鼎。」其後，他成為交趾統治者[[楊廷藝|楊廷藝]]手下的一員牙將管理愛州，楊廷藝更「以女妻之」結成姻親。<ref name="吳士連172">吳士連等《大越史記全書·外紀全書·吳紀》，東京大學東洋文化硏究所校合本，172頁。</ref><ref name="黎崱280">黎崱《安南志略·五代時僭竊》，北京中華書局點校本，280頁。</ref><ref name="陳仲金50">陳仲金《越南史略》第二卷第五章，北京商務印書館中譯本，50頁。</ref>

== 擊敗南漢 ==
{{Main|白藤江之戰 (938年)}}
937年，楊廷藝被牙將[[矯公羨|矯公羨]]所殺。吳權乃於次年（938年）自[[愛州|愛州]]舉兵，討伐矯公羨。矯公羨無法應付，向[[南漢|南漢]]求援。南漢國君[[劉龑|劉龑]]早有意奪取[[交趾|交趾]]地區的統轄權，所以「欲因其亂而取之」。劉龑封兒子[[劉洪操|劉洪操]]為[[靜海軍節度使|靜海軍節度使]]及[[交趾|交王]]負責領兵拯救矯公羨，而劉龑自己則屯駐海門（地名）以作聲援。南漢與吳權之間的一戰已勢難避免。

關於南漢方面的戰前準備，據《[[資治通鑑|資治通鑑]]》記載，劉龑的臣僚崇文使蕭益進言：「今霖雨積旬，海道險遠，吳權桀黠，未可輕也，大軍當持重，多用鄉導，然後可進。」意指此戰因有天雨且勞師襲遠，加上吳權的智謀出眾，南漢軍的勝算不高，唯有謹慎進兵及任用[[嚮導|嚮導]]。最終，劉龑沒有聽從。

據《[[大越史記全書|大越史記全書]]》所載，在另一方的吳權先誅殺勾結南漢的矯公羨，及至獲悉劉洪操帶兵來襲，吳權召來將士，分析形勢：「弘操一癡兒耳。將兵遠來，士卒疲弊，又聞公羨死無內應，氣已先奪。吾衆以力待疲，破之必矣。然彼利於艦，不先為之備，則勝負之形未可知也。若使人先於海門，潛植大杙，銳其首，冒之以鉄，彼船隨潮漲入杙內，然後我易制，無有出此者。」意思就是先將巨型木樁包上鐵尖，然後暗藏在[[白藤江|白藤江]]（{{lang|vi|Sông Bạch Đằng}}）心之下，待交戰時有用。

劉吳兩軍最終在白藤江會戰，史稱「[[白藤江之戰_(938年)|白藤江之戰]]」。據《[[新五代史|新五代史]]》，當時南漢戰情慘烈：「（洪操）出兵白藤以攻之。龑以兵駐海門，權已殺公羨，逆戰海口，記植鐵橛海中，權兵乘潮而進，洪操逐之，潮退舟還，轢橛者皆覆，洪操戰死，龑收餘衆而還。」《資治通鑑》、《[[越史略|越史略]]》及《大越史記全書》亦記載，吳權懂得準確把握白藤江潮水漲退的時間，乘潮漲的時候「使人以輕舟挑戰」，先假裝戰敗引南漢軍入江心的木樁陷阱位置，到潮退時，木樁戳破南漢戰艦。戰艦全軍大潰，吳權繼而揮軍進攻，取得勝利。劉龑看到南漢軍隊敗陣，兒子洪操戰死，便唯有「收餘衆而還」。<ref>歐陽修《新五代史·南漢世家·劉隱》，北京中華書局校注本，813頁。</ref><ref>司馬光《資治通鑑》卷二百八十一，後晉高祖天福三年冬十月戊戌條，北京中華書局校注本，9192-9193頁。</ref><ref>《越史略》卷上（收錄於《欽定四庫全書·史部》第466冊），上海古籍出版社書影本，570頁。</ref><ref>吳士連等《大越史記全書·外紀全書·南北分爭紀》，東京大學東洋文化硏究所校合本，171頁。</ref><ref name="陳仲金50"/>

== 建政稱王 ==
[[File:Vietnam_territorial_expansion_(900–1760_AD).gif|200px]]歷代疆域發展圖，其中'''橙色'''部份大約相當於[[吳朝|吳朝]]的統轄範圍。]]
吳權擊敗[[南漢|南漢]]軍隊，除去外來威脅後，便於939年春稱王，立楊廷藝女為-{后}-，定都[[古螺|古螺]]（{{lang|vi|Cổ Loa}}）。另外為了奠立長久之業，他又整頓國內政治，「置百官，制朝儀，定服色。」<ref name="吳士連172"/><ref>陳仲金《越南史略》第三卷第一章，北京商務印書館中譯本，57頁。</ref>

944年，吳權去世，享年四十七歲。在此之前，吳權遺命楊-{后}-的兄弟[[楊三哥|楊三哥]]輔助其子。其後，楊三哥篡位，控制吳權次子[[吳昌文|吳昌文]]等人，長子[[吳昌岌|吳昌岌]]則出逃。<ref name="吳士連172"/>

== 家庭 ==
* 父：[[吳旻|吳旻]]，為本州[[州牧|州牧]]。
* 岳父：[[楊廷藝|楊廷藝]]，吳權亦是他的牙將。
* -{后}-：楊氏，楊廷藝女，吳權稱王時立為-{后}-，《[[大越史記全書|大越史記全書]]》又稱之為「楊國母」。
* 子
** 長子：[[吳昌岌|吳昌岌]]，吳權死後，因[[楊三哥|楊三哥]]篡位而出逃。
** 次子：[[吳昌文|吳昌文]]，吳權死後，楊三哥「以為己子」。
** 子[[吳南興|吳南興]]，吳權死時尚為年幼，依附楊國母。
** 子[[吳乾興|吳乾興]]，吳權死時尚為年幼，依附楊國母。<ref name="吳士連172"/>

== 評價及爭議 ==
* 古代[[越南|越南]]史官[[黎文休|黎文休]]認為，吳權擊敗[[南漢|南漢]]稱王的壯舉，「可謂一怒而安其民，善謀而善戰者也。雖以王自居，未即帝位改元，而我越之正統，庶幾乎復續矣」。[[吳士連|吳士連]]認為吳權之功不獨體現在軍事成就方面，「其置百官，制朝儀，定服定，帝王之規模可見矣。享國不永，未見治效，惜哉。」<ref name="吳士連172"/>[[黎嵩|黎嵩]]評價吳先主是「濟世之奇才」，可是錯誤信任[[楊三哥|楊三哥]]，「顧托非人，禍及後嗣。」<ref>黎嵩《越鑑通考總論》（《大越史記全書》卷首），東京大學東洋文化硏究所校合本，86頁。</ref>
* 越南阮朝的[[嗣德帝|嗣德帝]]則認為吳權是僥倖得勝的：「吳權所遇，僞漢小國，弘操孱兒，致有白藤之捷，幸也，何足稱焉。若遇[[陳霸先|陳霸先]]之手，不爲[[李賁|李]]、[[趙光復|趙]]之續者幾希。」<ref>《[[欽定越史通鑑綱目|欽定越史通鑑綱目]]·前編·卷之五》</ref>
* 近代[[越南|越南]]史家[[陳仲金|陳仲金]]高度評價吳權：「吳權內殺逆臣為主報仇，外則破強敵，保全了國家，真乃是一位流芳千古的忠義之人。賴有吳權這樣的英雄人物，我南國始能擺脫1000年的[[越南歷史#上古傳說與古代中國的統治|北屬]]枷鎖，同時為[[丁朝|丁]]、[[前黎朝|黎]]、[[越南李朝|李]]、[[越南陳朝|陳]]諸朝日後得以在此南境建立自主政權開闢了道路。」<ref name="陳仲金50"/>
* [[越共|越共]]學者與陳仲金的論調相似，說吳權「結束了延續一千多年的亡國時期。我們民族贏得了自主權，揭開了民族長期獨立的序幕。」<ref>越南社會科學委員會《越南歷史》第二部份第四章，北京人民出版社中譯本，第150頁。</ref>另外，關於吳權在[[越南歷史|越南歷史]]上的政治地位，越共學者認為吳權的作為，件件都是越南成為「獨立王國」的起點。他在[[白藤江之戰_(938年)|白藤江之戰]]的勝利，是「為我們民族的歷史開創了一個新時期──越南獨立封建國家發展和越南民族迅速壯大的時期」。尤其是939年，「吳權決定廢除[[中國|北方封建王朝]]的[[節度使|節度使]]職務，自稱為王，建立起一個堂堂正正的獨立王國。[[古螺|古螺]]（[[安陽王|安陽王]]時期甌雒國的古都）又重新被選為十世紀獨立王國的京都。這是一件很有意義的事，它發揚了我們民族長期建國、衛國的光榮傳統，表明了我們民族保衛那經過一千多年不屈不撓地反抗外國封建階級的侵略和統治後所取得的獨立的堅強信念。」<ref>越南社會科學委員會《越南歷史》第二部份第五章，北京人民出版社中譯本，第151-152頁。</ref>
* [[中國|中國]]學者[[郭振鐸|郭振鐸]]、[[張笑梅|張笑梅]]對於吳權是否已帶領越南走向獨立，存有不同見解，認為「非越南獨立國家的開始」，理由有三：
# 吳權稱王，卻從未統一整個越南地區，實權仍操於地方豪族之手，故此[[吳朝|吳氏]]只能說是[[十二使君時期|十二使君]]中的一員，不過屬於其中的強者而已。
# 吳權稱王後，旋即被[[楊三哥|楊三哥]]篡奪，地方豪族據地稱霸，根本不承認吳權兒子[[吳昌岌|吳昌岌]]、[[吳昌文|吳昌文]]的國君地位。吳氏為了自保，最後唯有向[[中國|中國]][[南漢|南漢]]稱臣及請封節度使之職。
# 古代越南史家也認為吳權沒有即皇帝位，如[[黎文休|黎文休]]說他「未即帝位改元」，《[[欽定越史通鑑綱目|欽定越史通鑑綱目]]》亦稱吳氏君主之一的[[吳昌熾|吳昌熾]]與十二使君無異，「舊史皆書為正統，茲削不書，但並列於十二使君，以名統系。」而[[黎嵩|黎嵩]]亦認為[[丁部領|丁部領]]才是越南國家正統的起點，「我越正統之君，實自此始」。<ref>郭振鐸、張笑梅《越南通史》第三編第七章第二節，中國人民大學出版社版，242-244頁。</ref><ref>黎嵩《越鑑通考總論》（《大越史記全書》卷首），東京大學東洋文化硏究所校合本，87頁。</ref>
