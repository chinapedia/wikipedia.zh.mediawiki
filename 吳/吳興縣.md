'''吳興縣'''是[[浙江|浙江]][[湖州|湖州]]地區的舊縣名。

== 沿革 ==
*1949年5月，建立湖州市，[[民國|民國]]時期[[吳興縣_(中華民國)|吳興縣]]變為湖州市與吳興縣兩部分，二者同屬當時的[[嘉興專區|嘉興專區]]。
*1981年撤銷吳興縣併入湖州市。

== 人口 ==
按照當時[[中华人民共和国国务院|國務院]]制定的標准，只有2000人以上的非農業人口居多的聚落，才稱為城鎮。是故下表<ref>資料來源：《嘉興專區1953年中小城市分鎮人口明細表》，湖州市檔案館檔案，48-1-1，第24頁。</ref>中吳興縣的城鎮，只有縣城、[[南潯|南潯]]、[[雙林|雙林]]、[[埭溪|埭溪]]、[[練市|練市]]等5個，許多小的市鎮並未統計入內，這五個城鎮被稱為「五大鎮」。按照該項統計，1953年吳興縣的城鎮人口共計38798人，該年吳興全縣總人口為612623人。

== 行政區劃 ==
*1950年5月，吳興縣烏鎮的9個行政村(其中鎮上4個)劃歸[[桐乡县_(明朝)|桐鄉縣]]，與青鎮合並，統稱[[烏鎮|烏鎮]]。
