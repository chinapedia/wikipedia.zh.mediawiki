'''吴思豫'''（{{bd|1886年||1961年|}}），[[字|字]]'''立凡'''，[[浙江省|浙江省]][[嘉興府|嘉興府]][[嘉興县|嘉興县]]人，[[中華民国|中華民国]]军事将领。<ref name=re>{{Cite book<!--|和書-->|author = 徐友春主編|title = 民国人物大辞典 増訂版|year = 2007|publisher = 河北人民出版社|isbn = 978-7-202-03014-1}}；<br/>{{Cite book<!--|和書-->|author = 劉国銘主編|title = 中国国民党百年人物全書|year = 2005|publisher = 団結出版社|isbn = 7-80214-039-0}}；<br/>{{Cite book<!--|和書-->|author = 東亜問題調査会|title = 最新支那要人伝|year = 1941|publisher = [[朝日新聞社|朝日新聞社]]|isbn = }}</ref>

==生平==
1905年（[[光緒|光緒]]31年），他毕业于[[浙江武備学堂|浙江武備学堂]]步兵科。此后他赴日本留学，入[[東京振武学校|東京振武学校]]，后又入[[陸軍士官学校_(日本)|陸軍士官学校]]步兵科。1910年（[[宣統|宣統]]2年）他毕業归国。[[辛亥革命|辛亥革命]]爆发后，浙江省的革命派成立[[浙江軍政府|浙江軍政府]]，他任参議会参議、总司令部副官处处長。後来他历任陸軍第6師歩兵第98团团長、寧台鎮守使署参謀長。<ref name=re/>

1925年（[[民国紀元|民国]]14年），他到[[广州市|广州]]，任[[蒋介石|蒋介石]]的[[黄埔軍官学校|黄埔軍官学校]]訓練部少将主任兼[[長洲要塞|長洲要塞]]中将司令。此後他参加[[北伐|北伐]]。1927年（民国16年）他任浙江省政府軍事厅中将参謀長。翌年1月，他任[[国民革命軍|国民革命軍]]总司令部办公厅主任。同年，他任参軍处总務局局長，代理参軍处参軍長。<ref name=re/>

北伐结束後的1929年（民国18年）7月，吴思豫代理[[青島市|青島]]特別市市長（为市长[[馬福祥|馬福祥]]的代理）兼青島警備司令。1930年（民国19年）1月至1933年（民国21年）2月，他任首都警察厅厅長。1935年（民国24年）5月，他任[[国民政府軍事委員会|国民政府軍事委員会]]办公厅副主任。翌年1月，他被授予中将衔。<ref name=re/>

[[抗日战争|抗日战争]]期间，吳思豫歷任國民政府軍事委員會辦公廳副主任、軍事委員會銓敘廳中將廳長、典禮局局長、總統府參軍等要職。也曾任[[中華民国考試院|考試院]]法規委員、撫恤委員会副主任委員等文職。戦後的1946年（民国35年）6月，他任[[国民政府|国民政府]]参軍处参軍兼典礼局中将局長。同年11月他当选[[制憲国民大会|制憲国民大会]]代表。1948年（民国37年）5月，他任[[中華民国总統府|总統府]]第4局中将局長兼参軍。1949年（民国38年）5月他去职，迁居[[香港|香港]]。<ref name=re/>

1961年他在[[英属香港|英属香港]]逝世。享年76岁。<ref name=re/>
