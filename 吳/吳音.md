'''吳音'''（{{lang-ja|呉音}}）是[[日本漢字音|日本漢字音]]([[音讀|音讀]])的一種，指日本自[[大和時代|大和時代]]從[[南北朝|南朝]][[建康|建康]]周邊的[[吳語區|吳語區]]傳入在日本使用的漢字音。日本在吳音傳入之後，到[[奈良時代|奈良時代]]則從唐朝[[長安|長安]]傳入新[[漢音|漢音]]。吳音與漢音一樣主要傳承了[[中古漢語|中古漢語]]的特徵。

有观点认为吳音是從[[六朝时代|六朝时代]]經由[[朝鮮半島|朝鮮半島]]輾轉而入日本的，但支持此說法的史料並不充足。一般認為吳音在[[南北朝|南北朝]]時代（西元五至六世紀）從[[南朝|南朝]]直接傳入日本。吳音指的是当时南朝京师建康（今[[南京|南京]]）的[[金陵雅音|金陵雅音]]，[[金陵雅音|金陵雅音]]在东晋衣冠南渡后继承洛阳雅音并融合金陵当地吴音而产生。南北朝時，日本等國尊南朝為華夏之正，受南朝冊封不絕，如當時日本（倭五王）同南朝宋保持朝貢關係。隋朝時融合[[金陵|金陵]]雅音和[[洛陽|洛陽]][[鄴下|鄴下]]音編著《切韻》統一書面聲韻<ref>南北朝時期到隋初時，金陵士人和洛下平民語音最接近之前魏晉漢朝的雅音。[[顏之推|顏之推]]：音韻雅正，「搉而量之，獨金陵與洛下耳。......然冠冕君子，南方為優；閭裡小人，北方為愈。易服與之談，南方士庶，數言可辯；隔垣而聽其語，北方朝野，終日難分。而南染吳越，北雜夷虜，皆有深弊，不可具論。」（《[[顏氏家訓|顏氏家訓]]·音辭》）。故隋朝時結合金陵雅言和洛下音編《切韻》。</ref>，為唐以後所沿襲。

六朝時佛教開始盛行，故吳音多使用於佛教用語與律令紀（古代的法律與政令紀）用語，在漢音導入後也沒有消失，直至現代仍用漢音一併使用。另外「[[古事記|古事記]]」的[[萬葉假名|萬葉假名]]多使用吳音。

== 名稱 ==
根據《日本國語大辭典》，相對於時期較晚的漢音，將舊來發音稱作「吳音」之習俗的最早用例為[[平安時代|平安時代]]中期[[藤原公任|藤原公任]]著北山抄。而於此前「吳音」皆稱作「和音」。而「吳音」之稱乃是步入隋唐，中國北方出身者貶斥切韻音系為吳音後，為日本人所模仿。故此「吳音」之時代及地理源頭不詳，亦可追溯至漢魏時代論。

在吳音的時代日本其實並無「吳音」的名謂，或可推論，這種語音不是吳地一偶地域性的方音，而是當時通行的標準語音，後來的稱謂則是由於隋唐時期新的漢音引入後才出現的。後來，隋朝統一中國，陸法言載錄南北學者討論韻學的成果，編成《切韻》一書，融合南北朝官音，為唐朝長安正音所沿襲，對後世影響甚大。吳音融入日語程度較深(常用於基本詞彙中)，日本古來也將其稱為「和音」。

另外尚有「[[對馬|對馬]]音」和「[[百濟|百濟]]音」的名稱，由來是[[欽明天皇|欽明天皇]]時百濟的僧人[[法明|法明]]來到[[對馬|對馬]]以吳音讀[[維摩經|維摩經]]傳播佛教。

== 字音的推考 ==
推考罕用字的讀音時，相對於漢音可根據中國[[韻書|韻書]]的[[反切|反切]]等資料推定，吳音主要根據古代的佛典與律令等史料推斷，故而有時出現困難，不同的漢和字典也可能有不同的紀錄。

== 特點及與漢音的區別 ==
吳音的來源駁雜，體系不如漢音完善，但也具有下列的特點：

=== 聲母 ===
# 吳音字首的輔音反映出中古[[全濁聲母|全濁聲母]]與清聲母的區別，而[[漢音|漢音]]大多清化。如「{{lang|ja|定}}」吳音{{lang|ja|ヂャウ}}，漢音{{lang|ja|テイ}}；「奉」吳音{{lang|ja|ブ}}，漢音{{lang|ja|ホウ}}。
# 中古的鼻音聲母在漢音中多變為[[塞音|塞音]]或[[塞擦音|塞擦音]]聲母，而吳音保留鼻音聲母。如「{{lang|ja|萬}}」吳音{{lang|ja|モン}}，漢音{{lang|ja|バン}}；「二」吳音{{lang|ja|ニ}}，漢音{{lang|ja|ジ}}。
# 匣母在吳音中有脫落現象（主要見於合口呼字），而漢音中為{{lang|ja|カ}}行音。如：「{{lang|ja|和}}」吳音{{lang|ja|ワ}}，漢音{{lang|ja|クヮ}}；「回」吳音{{lang|ja|ヱ}}，漢音{{lang|ja|クヮイ}}。

#江攝：部分字吳音以{{lang|ja|オウ}}韻表示，漢音以{{lang|ja|アウ}}韻表示。如：「{{lang|ja|江}}」吳音{{lang|ja|コウ}}，漢音{{lang|ja|カウ}}；「邦」吳音{{lang|ja|ホウ}}，漢音{{lang|ja|ハウ}}。

##元韻部分字，吳音以{{lang|ja|オン}}韻表示，漢音以{{lang|ja|エン}}韻表示。如：「{{lang|ja|言}}」吳音{{lang|ja|ゴン}}，漢音{{lang|ja|ゲン}}；「園」吳音{{lang|ja|ヲン}}，漢音{{lang|ja|エン}}。

== 吳音與其他漢字音的關係 ==
六朝東晉衣冠南渡後，中原雅音南傳對南方的方言產生了很大影響，唐朝時有所謂「至今南人能晉語」之說（此處「晉語」指西晉時的洛陽雅音，非指山西晉方言），因此日本的吳音和南方吳方言、閩南方言等有較大的相近性。

=== 吳音與[[閩南語|閩南語]]的相似性 ===
*麻韻部分字吳音讀エ韻這一點，與閩南語白讀音吻合，且不見於其他語言/方言。如 「{{lang|ja|馬}}」吳音{{lang|ja|メ}} 閩南語be；「{{lang|ja|花}}」吳音{{lang|ja|クヱ}} 閩南語hue。
*流攝部分字吳音以{{lang|ja|ウ}}韻表示(相對於漢音用{{lang|ja|ユウ}}韻表示)這一點，也與閩南語吻合，如「{{lang|ja|有}}」吳音{{lang|ja|ウ}} 閩南語u；「{{lang|ja|丘}}」吳音{{lang|ja|ク}} 閩南語k'u。

有說法認為這些類似是吳音與閩南白讀音皆源於南朝語音/五胡亂華前的中原音的證據。

=== 吳音與[[吳語|吳語]]的相似性 ===
聲母
*日母在吳音與吳語白讀中皆為鼻音聲母n。此現象以吳語較為典型，但亦可見於其他方言如贛語，客語，福州話等。
*吳音中匣母的脫落現象與吳語類似，如「{{lang|ja|和}}」吳音{{lang|ja|ワ}} 吳語wu(或前加/ɦ/)，但匣母脫落在吳語中是普遍現象，而在吳音中主要限於合口字，開口字常為ガ行聲母。此現象亦見於其他南方方言如閩南語粵語等。
韻母
*山刪韻吳音多讀{{lang|ja|エン}}韻，吳語多讀e韻，有共同的主要元音。此現象亦見於粵語部分子方言。

=== 吳音與[[朝鮮漢字音|朝鮮漢字音]]的相似性 ===

*效攝、流攝、蟹攝三四等的單元音化。
*牙音、喉音以外的聲母不與合口韻搭配(u介音脫落)。與現代的吳語類似。但漢音亦有此特徵。
*山攝咸攝部分字，吳音以{{lang|ja|オ}}為韻腹，朝鮮漢字音以{{lang|ko|ㅓ}}為韻腹，兩者類似。如 「{{lang|ja|言}}」吳音{{lang|ja|ゴン}} 朝鮮音{{lang|ja|언}}；「{{lang|ja|越}}」吳音{{lang|ja|ヲチ}} 朝鮮音{{lang|ko|월}}；「{{lang|ja|凡}}」吳音{{lang|ja|ボム}} 朝鮮音{{lang|ko|범}}；「{{lang|ja|法}}」吳音{{lang|ja|ホフ}} 朝鮮音{{lang|ko|법}}。
*部分三等字的i介音脫落，此現象不見於漢音。
*梗攝三四等字，吳音的{{lang|ja|ア}}韻腹與朝鮮音的{{lang|ko|ㅓ}}韻腹皆為[[低元音|較低的元音]]，而漢音為[[高元音|較高]]的エ韻腹。

有說法認為這些類似是吳音經由朝鮮半島所傳的殘存證據，一種推測是吳音傳承自百濟音，而現代朝鮮漢字音傳自新羅音，故有相關。
