{{吳線RDT}}
'''吳線'''（{{jpn|j=呉線|hg=くれせん|rm=Kure sen|f=y}}）是一條連結[[廣島縣|廣島縣]][[三原市|三原市]]的[[三原站_(日本)|三原站]]至同縣[[安藝郡_(廣島縣)|安藝郡]][[海田町|海田町]]的{{link-ja|海田市站|海田市駅}}，屬於[[西日本旅客鐵道|西日本旅客鐵道]]（JR西日本）的鐵路線（[[幹線|幹線]]）。

當中三原站－廣站間設有公開招募的䁥稱「瀨戶內細波線」（{{lang|ja|瀬戸内さざなみ線}}），但旅客導覽統一採用「吳線」<ref name="dijest">[http://www.westjr.co.jp/railroad/digest/ 鉄道事業ダイジェスト] {{webarchive|url=https://web.archive.org/web/20140314194736/http://www.westjr.co.jp/railroad/digest/ |date=2014年3月14日 }} - 西日本旅客鉄道</ref>。

== 概要 ==
由於歷史因素，與山陽本線之間設有{{link-ja|經路特定區間|経路特定区間}}。

全線由{{link-ja|西日本旅客鐵道廣島支社|西日本旅客鉄道広島支社|廣島支社}}管轄，三原站－廣站間除廣站外由{{link-ja|三原地域鐵道部|三原地域鉄道部}}管轄。全線位於[[IC卡|IC]][[乘車卡|乘車卡]]「[[ICOCA|ICOCA]]」的岡山、廣島地域，當中只限三原站位於岡山、福山、香川地區，須波站－海田市站間位於廣島地區<ref>[http://www.jr-odekake.net/icoca/area/map/okayama.html ご利用可能エリア 岡山・広島エリア｜ICOCA｜ICOCA：JRおでかけネット] {{Wayback|url=http://www.jr-odekake.net/icoca/area/map/okayama.html |date=20110404093246 }} - 西日本旅客鉄道</ref>。

== 運行形態 ==
雖歸類為幹線，但在1978年10月停駛臥鋪特急列車「[[安藝號列車|安藝號]]」之後，僅[[快速列車|快速]]及[[普通列車|普通列車]]運轉<ref>『JR時刻表』2012年3月号、[[交通新聞社|交通新聞社]]。</ref>。

全線大致以{{link-ja|廣站|広站}}分為東西不同運轉區間，以西往廣島方向屬於JR西日本的{{link-ja|廣島城市網絡|広島シティネットワーク}}範圍內，列車班次較多，一小時平均約有三班列車；以東往三原方向列車班次較少，一至二小時一班列車。部份列車在廣車站更換班次編號後，原車直通運轉而行駛全線。

海田市車站側全列車直通[[廣島車站|廣島車站]]，一部之列車經由廣島車站直通[[山陽本線|山陽本線]]的[[下關車站|下關車站]]、[[可部線|可部線]]的{{link-ja|可部車站|可部駅}}。吳 - 廣島間有快速「{{link-ja|安藝路liner|安芸路ライナー}}」及「{{lang|ja|通勤ライナー}}」。

== 歷史 ==
1903年，連結軍都廣島和軍港吳的吳 - 海田市間的官設（吳線）開業。1904年12月1日，此區間由[[山陽鐵道|山陽鐵道]]租用營運。1906年12月1日，[[鐵道國有法|國有化]]之後回復由[[日本國有鐵道|國有鐵道]]營運。

1930年，三原 - 吳間的'''三吳線'''開業。1935年，三津內海 - 廣間延伸開業，全線通車。全通以後，新開業區間吳線（初代）編入三吳線，改稱'''吳線'''（2代）。

2018年7月6日，[[2018年7月西日本豪雨|西日本豪雨]]後，吳線多處路段受損而全線暫停行駛列車。7月17日起，吳車站至廣島車站間開行巴士、JR西日本緊急調用[[宮島渡輪|宮島渡輪]]在[[吳港|吳港]]至[[廣島港|廣島港]]開行渡船以做為鐵路修復前的替代交通<ref>{{Cite press release |title=鉄道の運転再開及び代行バス等の運行開始の見通しについて（7月17日17時時点） |publisher=国土交通省鉄道局 |date=2018-07-17 |url=http://www.mlit.go.jp/common/001245470.pdf#page=08 |format=PDF |accessdate=2018-07-25|language=ja}}</ref>。8月2日，[[坂車站|坂車站]]至[[海田市車站|海田市車站]]間路段修復通車<ref name="2018-07-31">{{Cite press release |title=広島地区の運転状況について（呉線）|publisher=西日本旅客鉄道株式会社広島支社 |date=2018-07-31|url=http://www.westjr.co.jp/press/article/items/0cc4bcb986d51d76939153c4bdc22b51.pdf|format=PDF |accessdate=2018-07-31|archiveurl=https://web.archive.org/web/20180801071202/http://www.westjr.co.jp/press/article/items/0cc4bcb986d51d76939153c4bdc22b51.pdf|archivedate=2018-08-01|language=ja}}</ref>。8月20日，吳車站至廣車站間路段修復通車<ref name="JRW-PRESS20180816">{{Cite web |date=2018-08-16 |url=https://www.westjr.co.jp/press/article/2018/08/page_12678.html |title=「西日本豪雨（平成30年7月豪雨）」に伴う運転計画・復旧状況について |publisher=JR西日本 |accessdate=2018-08-16|language=ja}}</ref>。9月9日，坂車站至吳車站間路段修復通車<ref name="nhk20180910">{{cite news|url=https://www3.nhk.or.jp/hiroshima-news/20180910/0002300.html|title=ＪＲ呉線再開　バス利用者も喜び|publisher=NHK|date=2018-09-10|language=ja}}</ref>。10月14日：安藝川尻車站至廣車站間路段修復通車<ref name="JRW-PRESS20180914">{{Cite web |date=2018-09-14 |url=https://www.westjr.co.jp/press/article/2018/09/page_13056.html |title=西日本豪雨で被害を受けた山陽線などの運転再開見込み（広島支社） |publisher=JR西日本 |accessdate=2018-09-16}}</ref>。10月28日，安浦車站至安藝川尻車站間路段修復通車<ref>{{Cite web |date=2018-10-16 |url=https://www.westjr.co.jp/press/article/2018/10/page_13235.html |title=西日本豪雨で被害を受けた線区の運転再開見込みについて |publisher=JR西日本 |accessdate=2018-10-29}}</ref>。12月15日，三原車站至安浦車站間路段完成修復，全線恢復通車<ref>{{cite news|url=https://www3.nhk.or.jp/news/html/20181215/k10011748311000.html|title=163日ぶりＪＲ呉線が全線復旧 西日本豪雨で被災 広島|publisher=NHK|date=2018-12-15|language=ja}}</ref>。
