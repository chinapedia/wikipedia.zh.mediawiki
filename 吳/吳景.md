'''吳景'''（{{bd|?||203年|catIdx=W吳}}），[[東吳|東吳]][[孙坚_(东吴)|孫堅]]的妻弟，其姊[[孙破虏吴夫人|吳夫人]]是[[孫策|孫策]]、[[孫權|孫權]]的母親。

== 生平 ==
吳景追隨[[孫堅|孫堅]]征伐有功，受騎都尉。討伐丹楊太守[[周昕|周昕]]，佔領了丹楊後，[[袁術|袁術]]上表吳景爲[[丹楊|丹楊]]太守。

[[初平|初平]]三年(192年)，孫策因父孫堅之死而依附吳景並於曲阿招募了數百兵力，卻遭到涇縣賊帥[[祖郎|祖郎]]襲擊，險些喪命。後孫策依吳景之建言又與[[孫河|孫河]]、[[呂範|呂範]]合眾攻擊祖郎，使其敗走。

之後吳景又被[[劉繇|劉繇]]逼迫而北投[[袁術|袁術]]。袁術用他爲督軍中郎將，與[[孫賁|孫賁]]於橫江討伐[[樊能|樊能]]、[[于糜|于糜]]，又在秣陵擊敗[[笮融|笮融]]、[[薛禮|薛禮]]。當孫策在[[牛渚|牛渚]]受傷時，遇降兵造反，幸被吳景平定及擒下降兵。吳景跟隨孫策討伐劉繇，劉繇出走[[豫章|豫章]]，他奉孫策命到[[袁術|袁術]]處報訊。袁術與[[劉備|劉備]]爭奪[[徐州|徐州]]時，任命吳景為[[廣陵|廣陵]]太守。

袁術僭越稱帝時，孫策曾給袁術書信，勸喻不可，袁術不聽所勸，兩人於是不再往來。孫策接著派人通知吳景，吳景於是東歸，孫策仍任他為丹楊太守。漢廷遣使到來，授銜揚武將軍，仍舊領丹楊。

== 姊姊 ==
*[[孙破虏吴夫人|吳夫人]]，孫堅元配夫人，生有四子，孫策孫權孫翊孫匡，以及一女，但并不确定此女是孙坚的哪个女儿。

== 姊夫 ==
*[[孙坚_(东吴)|孫堅]]，吳夫人夫君，破黃巾，伐董卓，解區星之亂，長沙太守孫堅。

== 外甥 ==
*[[孫策|孫策]]，長男，東漢討逆將軍，吳國奠基者，拜吳侯，孫權稱帝後追諡其長兄為長沙桓王。
*[[孫權|孫權]]，次男，吳大帝，東吳開國君主。
*[[孫翊|孫翊]]，三男，個性驍勇果烈，有孫策之風，偏將軍領單楊太守。
*[[孫匡|孫匡]]，四男，長兄孫策讓其烏程侯爵位給幼弟孫匡繼承，早死。

== 子嗣 ==
吳景卒於建安八年（203年），有子[[吳奮|吳奮]]封新亭候。吴奋子[[吴安|吴安]]袭父爵，坐鲁王[[孙霸|孙霸]]党被杀。吴景另一个儿子即吴奋的弟弟[[吳祺|吳祺]]袭爵，封都亭侯，死后由儿子[[吴纂|吴纂]]袭爵。吴纂娶[[滕胤|滕胤]]女。
