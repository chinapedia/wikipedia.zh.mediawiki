'''吳主光'''，本名'''吳悅能'''（'''{{lang|en|Rev. David NG}}'''，{{bd|1940年||2017年|6月10日|catIdx=Ng吳}}），香港基督教牧師，平安福音堂創辦人。

==生平==
吳主光成長於深水埗，幼時參加黑社會，約於1948至1950年讀崇真小學。於1954至1961年間，在陳喜謙牧師帶領的浸信會少年團聯會夏令會中信主，並受浸成為[[浸信會|浸信會]]會友。<ref>{{ cite web |url=https://christiantimes.org.hk/Common/Reader/News/ShowNews.jsp?Nid=97876&Pid=5&Version=0&Cid=220&Charset=big5_hkscs |title=吳主光：「我不後悔；教會是一代一代向神交賬。」
 |date=2017-01-23 |publisher=時代論壇 |author=羅民威 }}</ref> 後在青山道尾一木屋區內向貧窮人傳福音，至1962年創立一間獨立教會，初時取名[[新圍村福音堂|新圍村福音堂]]，後來發展成為許多分堂之時，即改名為[[平安福音堂|平安福音堂]]；2011年已增長到三十七間堂。

吳主光於1970年畢業於[[香港|香港]][[海外神學院|海外神學院]]，畢業後於[[菲律賓|菲律賓]]牧養教會，兩年後回港於平安福音堂牧會，稍後兼任[[香港|香港]][[海外神學院|海外神學院]]教席，兩年後升任神學院副院長，合共十二年之久。<ref name="background_info">{{cite book zh |author= 吳主光 |title= 雅歌研經亮光 |location= 香港 |publisher= 種籽出版社 |date= 2006 |pages= 封底 |id= ISBN 9622307361}}</ref>

1988年，吳主光接受按牧後移民加拿大，受聘為「列治文華人宣道會」主任牧師。六年半後離開宣道會，在列治文創立列治文平安福音堂。到1999年十一月，吳主光應香港平安福音堂的邀請，從加拿大回流返港，出任九龍城平安福音堂（即現今[[尖沙咀|尖沙咀]][[平安福音堂|平安福音堂]]）長老傳道。同時，吳牧師亦與[[香港|香港]][[海外神學院|海外神學院]]校友創辦[[整全訓練神學院|整全訓練神學院]]，成為該神學院之第一任院長。

2008年底，吳主光因年近七十而退休，辭去[[尖沙咀|尖沙咀]][[平安福音堂|平安福音堂]]傳道人職位，只答應做[[深水埗|深水埗]]和[[尖沙咀|尖沙咀]][[平安福音堂|平安福音堂]]長老並顧問；同時亦辭去[[整全訓練神學院|整全訓練神學院]]之院長職位，只專心任教神學和寫作屬靈書籍。其間，也曾在[[衛道神學研究院|衛道神學研究院]]任客席教授，主要講授「佈道學」與「教會增長」。<ref name="United_Wesleyan_Graduate_Institute">{{cite book zh |author= 衛道神學研究院 |title= 學院概覽2009-2010 |location= 香港 |publisher= 衛道神學研究院 |date= 2009 |pages= 13 }}</ref>

2017年6月10日下午1點10分，吳主光在香港沙田[[威爾斯親王醫院|威爾斯親王醫院]]逝世，享年77載。

==信仰特徵==
吳主光牧師稱自己為有[[美南浸信會|美南浸信會]]背景的「深水埗浸信會」信徒。<ref name="spiritual_route">{{cite web |title=「蒙恩傳統」分享系列|url=http://www.pec.bc.ca/pecgrace/ |publisher=列治文平安福音堂|accessdate=2009年2月10日 |language=zh }}</ref>他的信仰特徵包括以屬靈優先<ref name="PEC_spiritual_route">{{cite web |title=平安福音堂的起源：「屬靈路線」|url=http://www.pec.bc.ca/pecgrace/pecgrace05.doc|publisher=列治文平安福音堂|accessdate=2009年2月27日 |language=zh }}</ref>、認為擘餅（[[聖餐|聖餐]]）聚會是教會蒙福主要原因、視[[羅馬天主教|羅馬天主教]]為[[異端|異端]]<ref name="we_do_not_celebrate_xmas">{{cite web |title=本教會不慶祝聖誕|url=http://www.hkpec.org/tst/pastersharing/041128pastorsharing.htm|publisher=尖沙咀平安福音堂|accessdate=2009年2月27日 |language=zh }}</ref>
<ref name="corner_stone">{{cite web |title=末世衰退潮中的砥柱|url=http://www.hkpec.org/tst/pastersharing/060813pastorsharing.htm|publisher=尖沙咀平安福音堂|accessdate=2009年2月27日 |language=zh }}</ref>
<ref name="bring_catholics_to_christ">{{cite web |title=帶天主教徒歸主|url=http://www.hkpec.org/tst/pastersharing/060820pastorsharing.htm|publisher=尖沙咀平安福音堂|accessdate=2009年2月27日 |language=zh }}</ref>、反對[[靈恩派|靈恩派]]、反對基督教與異端教會大合一<ref name="church_unification">{{cite web |title=教會大合一運動|url=http://www.hkpec.org/tst/pastersharing/pastorsharing17aug03.htm|publisher=尖沙咀平安福音堂|accessdate=2009年2月27日 |language=zh }}</ref>、避免金字塔式的「總會領導」等。<ref name="independent_route">{{cite web |title=平安福音堂的起源：「獨立路線」|url=http://www.pec.bc.ca/pecgrace/pecgrace06.doc|publisher=列治文平安福音堂|accessdate=2009年2月27日 |language=zh }}</ref>

==爭議==
吳主光曾發表許多出位及富爭議性的言論，例如利用例子間接批評爆炸論，在講道中多次宣稱在普世教會內有許多基督徒不得救，好的教會當中也有三份一人不得救<ref name="is little people saved?">{{cite web |title=得救的人少嗎？|url=http://www.woelife.com/html/bottom5/share_lessrescuepeople.html|publisher=woelife.com|accessdate=2009年9月23日 |language=zh }}</ref><ref name="2007 TSTPEC church camp sermon recording">{{cite web|title=二零零七年尖沙咀平安福音堂夏令會講道|url=http://storage.biblemessage.org/tstpec/SummerCamp/2007/CMP2007-N02.mp3|publisher=尖沙咀平安福音堂|accessdate=2009年9月23日|language=zh}}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，差的教會情況更壞。他又指許多基督教界的名人，如著名美國[[葛培理|葛培理]]牧師全家不得救，因為其相信不信耶穌也可以得救。<ref name="Is Billy Graham Saved?">{{cite web |title=葛培理---不得救！|url=http://www.youtube.com/watch?v=OOAZiH5PHSA|publisher=philipberean|accessdate=2009年8月18日 |language=zh }}</ref>復又在2007年[[葛福臨|葛福臨]]佈道團來香港的時候批評佈道團表面上是傳福音，骨子裡其實還是推動「教會大合一運動」<ref name="About Franklin Graham">{{cite web |title=對葛福臨佈道會的立場|url=http://www.hkpec.org/tst/pastersharing/071028pastorsharing.htm|publisher=尖沙咀平安福音堂|accessdate=2009年8月18日 |language=zh }}</ref>。
