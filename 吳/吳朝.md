'''吳朝'''（{{lang-vi|'''Nhà Ngô'''}}，939─965年<ref name="939─965">陳重金（即陳仲金）《越南通史》（即《越南史略》），北京商務印書館，57頁；郭振鐸、張笑梅《越南通史》，中國人民大學出版社，664頁；《東南亞歷史詞典》，上海辭書出版社，509頁。</ref>），亦有稱作「'''吳氏自主政權'''」<ref>郭振鐸、張笑梅《越南通史》，中國人民大學出版社，664頁。</ref>，[[越南歷史|越南歷史]]時期之一。在10世紀以前，越南北部（古稱[[交州|交州]]）長期受[[越南北屬時期|中國統治]]。938年，越南北部（古稱[[交州|交州]]）[[豪族|豪族]][[吳權|吳權]]擊敗中國[[南漢|南漢]]軍隊，並於隔（939）年稱王建政，建立自主政權，是為吳朝。吳朝歷經吳權、[[楊三哥|楊三哥]]、[[吳昌岌|吳昌岌]]、[[吳昌文|吳昌文]]等君主，其間出現內部權力鬥爭及各地豪強割據等問題，後更演成「[[十二使君|十二使君]]」之局，吳朝衰亡。後世肯定吳朝在越南爭取自主的重要性，但對於它是否可稱為越南獨立之始則出現爭議。

== 政局發展 ==
=== 白藤江之戰及建政 ===
{{see also|白藤江之战 (938年)|吳權}}
[[File:Cua_song_Bach_Dang.JPG|thumb]]擊退[[南漢|南漢]]軍隊的古戰場──[[白藤江|白藤江]]]]
吳朝建政的時間，是在[[越南北屬時期|中國統治時代]]的末期。10世紀以前，[[越南|越南]]北部地區（古稱[[交州|交州]]）長期處於中國內地王朝所轄，到10世紀初，[[唐朝|唐朝]]分裂瓦解，形成[[五代十國|五代十國]]局面，交州豪族乘中國內地紛亂，爭取自主權力，歷經[[曲家|曲氏]]家族、[[楊廷藝|楊廷藝]]等統治當地，交州與[[嶺南|嶺南]]地區的割據勢力[[南漢|南漢]]為鄰，雙方時起衝突。<ref>呂士朋《北屬時期的越南》，香港中文大學新亞研究所，140─142頁。</ref>

吳朝的開創者[[吳權|吳權]]，交州[[唐林州|唐林州]]人士，為當地世家大族。楊廷藝統治時，吳權擔任牙將，管治[[愛州|愛州]]一地，並娶廷藝之女為妻。<ref name="大越史記全書172">吳士連等《大越史記全書·外紀全書·吳紀》，東京大學東洋文化硏究所，172頁。</ref>937年，楊廷藝被部將[[矯公羨|矯公羨]]所殺。不久，938年，吳權討伐矯公羨，公羨向南漢求援。南漢皇帝[[劉龑|劉龑]]派皇子[[劉洪操|劉洪操]]（一作劉弘操）率兵救矯公羨。吳權先殺公羨，然後在[[白藤江|白藤江]]海口（在今[[廣安鎮|越南廣安]]東<ref name="越南歷代疆域143">陶維英《越南歷代疆域》，北京商務印書館，143頁。</ref>）部署陷阱，「先於海口多植大杙，銳其首，冒之以鐵」，亦即在木椿末端包上鐵皮，等到潮漲時派出舟師，引南漢海軍船艦到裝有大杙的位置，適值潮退，南漢艦隊「轢橛者皆覆」，於是「漢兵大敗，士卒覆溺者太半」，劉洪操亦死於此役，是為[[白藤江之戰_(938年)|白藤江之戰]]。<ref>歐陽修《新五代史·南漢世家·劉龑》，北京中華書局，813頁；司馬光《資治通鑑·後晉紀》，北京中華書局，9192─9193頁。</ref>吳權擊退南漢軍後，於次年（939年）春自立為王，是為「前吳王」，定都[[螺城|螺城]]（即古螺，在今[[東英縣|東英縣]]）。<ref>吳士連等《大越史記全書·外紀全書·吳紀》，東京大學東洋文化硏究所，172頁；呂士朋《北屬時期的越南》，香港中文大學新亞研究所，142頁；郭振鐸、張笑梅《越南通史》，中國人民大學出版社，240頁。</ref>

=== 楊三哥及吳昌岌、吳昌文時期 ===
{{see also|楊三哥|吳昌岌|吳昌文}}
[[吳權|吳權]]在位六年，於945年去世。<ref name="大越史記全書172"/>他身後的吳朝政局並非長治久安。吳權臨終時，遺命其子由[[楊三哥|楊三哥]]（楊廷藝之子，吳權王后楊氏的兄弟）輔助，但楊三哥卻自奪王位，號稱「平王」。吳權長子[[吳昌岌|吳昌岌]]出逃，得茶鄉范令公協助躲藏，使楊三哥無法捉拿。吳權第二子[[吳昌文|吳昌文]]被三哥收為己子。950年，楊三哥派吳昌文出外討伐叛亂，吳昌文在途中倒戈，推翻平王，奪回權力。<ref>《越史略》卷上（收錄於《四庫全書‧史部》第466冊），上海古籍出版社，571頁；吳士連等《大越史記全書·外紀全書·吳紀·楊三哥》，東京大學東洋文化研究所，172-173頁。</ref>吳昌文及自立為「南晉王」，又於次年派遣人迎接其兄吳昌岌，立為「天策王」，兩人同理國事，曾一起征討[[華閭古都|華閭洞]]割據者[[丁部領|丁部領]]。但後來吳昌岌擅威作福，令吳昌文憤而不再參與政事。<ref>《越史略》卷上（收錄於《四庫全書‧史部》第466冊），上海古籍出版社，571頁；吳士連等《大越史記全書·外紀全書·吳紀·後吳王》，東京大學東洋文化研究所，172-173頁。</ref>洮江郡人周泰不服，吳昌文親征，將之殲滅，但從此「狃勝而驕」。<ref>潘清簡等《欽定越史通鑑綱目》前編卷之五，第二十六頁。</ref>965年，吳昌文在親征太平唐阮二村時陣亡。<ref>《越史略》卷上（收錄於《四庫全書‧史部》第466冊），上海古籍出版社，571頁；吳士連等《大越史記全書·外紀全書·吳紀·後吳王》，東京大學東洋文化研究所，175頁。</ref>

交州雖為自主政權，但其流通的貨幣，仍以[[唐代|唐代]][[開元通寶|開元通寶]]、[[後漢|後漢]][[漢元通寶|漢元通寶]]（948年始鑄）等為主，到日後的[[丁朝|丁朝]]方獨立鑄幣。<ref>《越南歷史貨幣》，中國金融出版社，14及16頁。</ref>

=== 衰亡 ===
{{see also|十二使君時期}}
從楊三哥奪位時起，地方上便出現割據勢力，及至965年吳昌文戰死以後，吳氏朝廷已無力管治全國，吳氏王子[[吳昌熾|吳昌熾]]（吳昌岌之子）只能據守平橋（有說在今[[興安省_(越南)|興安省]]或[[清化省|清化省]]）,與此同時「群雄競起，各據郡邑自守」，後世統稱這些割據者為「[[十二使君時期#十二使君時期的各割據者|十二使君]]」。最後，華閭洞割據者[[丁部領|丁部領]]擊敗各股勢力，奪得全國政權。<ref>《越史略》卷上（收錄於《四庫全書‧史部》第466冊），上海古籍出版社，571─572頁；吳士連等《大越史記全書·外紀全書·吳紀·後吳王、吳使君》，東京大學東洋文化研究所，175─176頁；陳重金（即陳仲金）《越南通史》（即《越南史略》），北京商務印書館，58─59頁。</ref>

== 統治組織 ==
[[File:Tượng_Ngô_Quyền.jpg|thumb]]像]]
吳朝自[[吳權|吳權]]建政後，便草創朝廷架構。據《[[大越史記全書|大越史記全書]]》載，吳權在939年「始稱王」，冊立王后，置百官，制朝儀，定服色，這些措施在後世被視為「帝王之規模可見矣」。<ref name="大越史記全書172"/>在地方行政方面，從[[越南漢喃銘文|銘文]]資料可知，當時有[[縣|縣]]、[[村|村]]等行政單位，另外還有稱為「社」的宗教會社組織，社裡有社主、社副、社判官、社務、都監等位號。<ref name="日早古鐘銘47">《日早古鐘銘》（948年刻成），收錄於潘文閣、蘇爾夢主編《越南漢喃銘文匯編·第一集北屬時期至李朝》，遠東學院、漢喃研究院，47頁。</ref>

== 疆域 ==
有關吳朝領土範圍，越南古代地理著作，如[[阮廌|阮廌]]《[[輿地志|輿地志]]》，認為是「北夾[[兩廣|兩廣]]，南至{{link-vi|地哩|Địa Lý (châu)}}，凡二千八百里；東接[[欽州|欽州]]、海門（在中國[[廣西|廣西]]），西界[[雲南|雲南]]，凡一千七百里」。<ref>阮廌《輿地志》，收錄於《皇黎抑齋相公遺集》，6頁。</ref>現代學者[[陶維英|陶維英]]則指是在「[[雒越|雒越]]人後裔的各州，即北部的中游和平原地區以及[[清化省|清化]]──[[乂安省|乂安]]的中游和平原地區得以行使其權力」。<ref name="越南歷代疆域143"/>

== 宗教 ==
[[佛教|佛教]]傳入[[交州|交州]]後，[[第三次北屬|隋唐時期]]出現了[[滅喜派禪宗|滅喜派禪宗]]及[[無言通禪派|無言通禪派]]等教派，到10世紀的吳朝仍得到流傳，並在日後的[[丁朝|丁]]、[[前黎朝|黎]]、[[李朝_(越南)|李]]各朝受到重視。<ref>杜繼文《佛教史》，江蘇人民出版社，300─301頁。</ref> 至於起源於[[中國|中國]]的宗教，在吳朝亦得以流傳。據[[河內|河內]]慈廉的[[越南漢喃銘文|鐘銘]]資料，吳朝時期有[[道教|玄]]、[[儒|儒]]二門信眾結合，崇信道教的「太始三尊」（即[[元始天尊|元始天尊]]、[[靈寶天尊|靈寶天尊]]、[[道德天尊|道德天尊]]），一同興造道教設施。道教信徒中還設宗教職位，有高級醮祭人員「[[高功|高功]]」一職。<ref>《日早古鐘銘》（948年刻成），收錄於潘文閣、蘇爾夢主編《越南漢喃銘文匯編·第一集北屬時期至李朝》，遠東學院、漢喃研究院，47─48頁。</ref>

== 與中國南漢關係 ==
吳朝與[[中國|中國]][[南漢|南漢]]政權接鄰，南漢在[[白藤江之戰_(938年)|白藤江之戰]]被[[吳權|吳權]]所敗後，「遂棄不復攻」，不再爭奪交州。<ref>歐陽修《新五代史·南漢世家·劉晟》，北京中華書局，816頁。</ref>雖然如此，但南漢對交州仍有影響力。在[[慈廉縣|慈廉縣]]日早村出土的吳朝銅鐘「日早古鐘」（948年刻），銘文上使用南漢年號「[[乾和|乹和]]六年」，現代學者據此認為該古鐘所屬的社，與南漢有一定的關係。<ref>《日早古鐘銘》（948年刻成），收錄於潘文閣、蘇爾夢主編《越南漢喃銘文匯編·第一集北屬時期至李朝》，遠東學院、漢喃研究院，45─48頁。</ref>[[法國|法國]][[漢學家|漢學家]][[蘇爾夢|蘇爾夢]]指出南漢鑄造工藝精湛，「日早古鐘」便可能鑄造於南漢，或是南漢工匠到交州鑄造，反映了雙方有著文化交流。<ref>[http://www.lib.cuhk.edu.hk/conference/occ/Salmon.pdf 香港中文大學第二屆海外華人研究與文獻收藏機構國際合作會議──蘇爾夢著，鄭力人譯《從梵鐘銘文看中國與東南亞的貿易往來》]{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

在[[楊三哥|楊三哥]]奪權事件後，吳氏家族為求自保，乃向南漢稱臣求封<ref>郭振鐸、張笑梅《越南通史》，中國人民大學出版社，243頁。</ref>，於是在954年，吳昌文（一作吳昌濬）遣使到南漢，稱臣入貢及要求冊封。<ref group="注">據中國史籍《[[新五代史|新五代史]]》及元代越人[[黎崱|黎崱]]《[[安南志略|安南志略]]》所述，遣使稱臣的吳朝君主為「[[吳昌文#「吳昌濬」問題|吳昌濬]]」，但越南古代史籍《[[越史略|越史略]]》及《[[大越史記全書|大越史記全書]]》沒有「吳昌濬」的記錄。對此，[[阮朝|阮朝]]史官所編的《[[欽定越史通鑑綱目|欽定越史通鑑綱目]]》猜測「或者[[吳昌文|昌文]]改名昌濬」，中國學者武尚清認為《安南志略》記述正確，因此[[吳昌岌|吳昌岌]]、吳昌濬及吳昌文三人均是真實存在的人物。</ref><ref>《越史略》卷上（收錄於《四庫全書‧史部》第466冊），上海古籍出版社，571頁；吳士連等《大越史記全書·外紀全書·吳紀·後吳王》，東京大學東洋文化研究所，174頁；歐陽修《新五代史·南漢世家·劉晟》，北京中華書局，816頁；黎崱《安南志略》，北京中華書局，280─281頁；潘清簡等《欽定越史通鑑綱目》前編卷之五，第二十五至二十六頁。</ref>這次求封的結果，各種史籍有不同記載，一說南漢答應吳朝要求，並派員出發冊封，但途中吳朝又以「海賊為亂，道路不通」為由，終斷行程；<ref>歐陽修《新五代史·南漢世家·劉晟》，北京中華書局，816頁；黎崱《安南志略》，北京中華書局，280─281頁；潘清簡等《欽定越史通鑑綱目》前編卷之五，第二十五頁。</ref>另一說是南漢封吳昌文為「[[靜海軍節度使|靜海軍節度使]]兼都護」。<ref>司馬光《資治通鑑·後周紀》，北京中華書局，9501頁；吳士連等《大越史記全書·外紀全書·吳紀》，東京大學東洋文化硏究所，174頁。</ref>

== 吳朝的歷史意義 ==
在越南，後世視吳朝為越南脫離中國自立的重要時期。封建時代史官[[黎文休|黎文休]]說[[吳權|吳權]]稱王是「我越之正統，庶幾乎復續矣」<ref name="大越史記全書172"/>，[[法屬印度支那|法屬時代]]越南人士有「我越南復國自主之第一君曰[[吳權|吳王權]]」<ref>黃高啟《越史要：越南史要》，第四十一葉。</ref>、「吳王乘釁起兵，越乃為[[五嶺|五嶺]]以南之獨立國」<ref>吳甲豆《中學越史撮要》夏集，第二至第三葉。</ref>等論調。到現代，越南學術機關[[越南社會科學委員會|越南社會科學委員會]]說它是「堂堂正正的獨立王國」<ref>越南社會科學委員會《越南歷史》，北京人民出版社，第151-152頁。</ref>。

但吳朝的獨立國家地位亦受到質疑。中國學者[[郭振鐸|郭振鐸]]、[[張笑梅|張笑梅]]指出，吳朝在[[吳權|吳權]]時就未曾統一越南，各地均有[[豪族|豪族]]掌權，[[吳昌岌|吳昌岌]]、[[吳昌文|吳昌文]]時仍有地方割據，吳氏朝廷還需向中國[[南漢|南漢]]稱臣求封以自保，而且吳氏君主並未正式稱帝改元，所以難以被視為越南獨立國家之始。<ref>郭振鐸、張笑梅《越南通史》，中國人民大學出版社，242-244頁。</ref>
