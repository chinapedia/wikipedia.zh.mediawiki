'''吳丹鳳'''（Ng Dan Fung，{{bd|1927年}}），籍貫[[廣東省|廣東省]][[順德縣|順德縣]]，1947年第二屆[[香港小姐|香港小姐]]冠軍、已息影的香港電影演員及歌手。

== 「香港小姐」榮銜<ref name="ReferenceA">1947年7月28日，香港《工商日報》，第一張；第三頁。</ref> ==
* 身高5呎1吋（1.55公呎）；體重103磅（46.7公斤），嬌小玲瓏，當時在「順德師範學院」唸書，亦是胞兄的助手，希望能考入藝術專科學校，歌詠尤為愛好。
* 1947年7月27日在「域多利兵房」，約300多觀眾參加籌賬中英水災之游泳選美大會，最初有18位佳麗報名，最後僅有4位臨場出現，包括當時20歲的吳丹鳳（25號），黃安琪（20號）、白麗蓮（21號）及朱麗研（9號），由於參選佳麗人數太少，大會臨時在競選場地呼籲在場女士參與競選，結果有兩位女士響應，包括陳愛梅（42號）及何秀嫻（8號），身穿綠花點泳衣的吳丹鳳與各佳麗在職員領導下成群魚貫繞行游泳池一圈，然後排立中央，被12位評判員評分，其中體格佔40分；體態佔40分；容貌佔20分，在當天下午5時選出：吳丹鳳得359分奪得冠軍，榮膺第二屆1947年『香港小姐』榮銜；陳愛梅得306分奪得亞軍；朱麗妍及白麗蓮得298分奪得雙季軍；黃安琪得288分屬殿軍。
* 「中英影片公司」在當晚於[[英京大酒家|英京大酒家]]的慶功讌上，以港幣1,500元與吳丹鳳簽訂合約，邀請她參演電影《豪門春色》，並定好10天後開鏡。

== 交通新聞 ==
* 1949年7月16日凌晨，吳丹鳳的40歲男朋友倫天樂先生當年住在[[香港島|香港島]][[跑馬地|跑馬地]][[毓秀街|毓秀街]]32號二樓，她搭乘他的轎車，可惜男朋友在香港島[[黃泥涌道|黃泥涌道]]無牌駕駛私家車（車牌：3363）而被起訴<ref>1949年7月17日，香港《工商日報》，第五頁。</ref>。
* 1954年8月7日下午3時，吳丹鳳與第二任丈夫倫天樂搭乘由司機駕駛的私家車（車牌：3857）由住所駛至中環，在[[灣仔|灣仔]][[大佛口|大佛口]]附近的[[軒尼詩道|軒尼詩道]]與一輛向西行的電車相撞，轎車中部被撞凹入，第二任丈夫倫天樂頭部受傷需要送醫院敷藥療治，她與司機僥幸未有受傷<ref>1954年8月8日，香港《工商日報》，第五頁。</ref>。

== 家庭 ==
* 吳丹鳳的父親是茶葉商人，在[[第二次世界大戰|第二次世界大戰]]前已不幸逝世<ref name="ReferenceA"/>。
* 由於香港陷落，吳丹鳳遷居[[廣州|廣州]]，當時31歲經營投機事業的商人區錫元先生鼓盆之戚，遺下一子三女，於1943年在中國大陸娶吳丹鳳作[[繼室|繼室]] ，[[第二次世界大戰|第二次世界大戰]]後，到香港發展，初期開設麵店；後來在灣仔開設「同福煤炭行」。  她的丈夫在[[澳門|澳門]]知道她當選香港小姐的新聞後，在1947年7月30日晚上回到香港島[[洛克道|洛克道]]261號住所，即將吳丹鳳毆打，拳腳相加。  1947年8月1日晚上，吳丹鳳與朋友張威廉夫婦到「凱旋跳舞廳」消遣，回家後，丈夫責罵，以剃刀恐嚇，由女傭尹媽勸阻無效，吳丹鳳被丈夫扯著頭髮拉至閣樓，最後一撮頭髮被扯去，手足被擊至瘀黑。翌早，吳丹鳳偕一對兒女逃回[[西環|西環]]胞兄的住所<ref>1947年8月6日，香港《工商日報》，第一張；第三頁。</ref>。
* 1947年10月14日中午12時30分，吳丹鳳身穿藍薄絨長旗袍與在中國大陸結婚的丈夫區錫元在「冼秉熹律師行」簽訂離婚協約，正式解除夫妻關係。長子區大眼及仍在哺乳中的-{么}-女區惠儀由母親吳丹鳳照顧；惟父親區錫元需要每月支付[[瞻養費|瞻養費]]港幣150元至兒女16歲為止<ref>1947年10月15日，香港《工商日報》，第一張；第三頁。</ref>。
* 吳丹鳳在1952年2月16日再婚，她與香港鐘錶業商會理事長倫天樂先生在香港婚姻註冊署舉行婚禮，當晚在香港島中環[[德輔道中|德輔道中]][[先施公司|先施公司]]六樓的『中國酒家』舉行婚宴。她的丈夫除經營鐘錶業外，亦有涉足電影業及體育圈<ref>1952年2月15日，香港《工商日報》，第五頁。</ref>。

== 息影後的工作 ==
* 吳丹鳳與第二任丈夫倫天樂在1956年8月前往[[瑞士|瑞士]]攷察鐘錶業，回港後寫成《歐遊紀實》，對於歐洲大陸的風光特別瑞士的鐘錶業有極詳細的記載。  1956年11月1日在九龍彌敦道與[[快富街|快富街]]交界的「邵氏大廈」（現在的[[旺角中心|旺角中心]]第一期）開設「瑞士表行」，吳丹鳳任司理<ref>1956年10月31日，香港《工商日報》，第六頁。</ref>。
* 吳丹鳳參加瑞士 製造的Buren（舊譯：漂而亮）名表的公開徵求新中文譯名比賽，她以佛經裡的『寶輪』為名參加，意謂「永行不息」，結果於1961年11月8日晚上8時在香港島德輔道中124號「金城酒家」5樓被評為勝出者。在1961年11月14日(星期二)在九龍彌敦道348號的「大華酒店」(現在的「諾富特酒店 」) 獲得該表廠東主親自頒發港幣9,900元獎金<ref>1961年11月15日，香港《工商日報》，第七頁。</ref>。
