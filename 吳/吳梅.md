'''吳梅'''（{{bd|1884年||1939年|3月17日}}），字'''瞿安'''，号'''霜厓'''，別署'''癯安'''、'''逋飛'''和'''厓叟'''，[[中國近代|中國近代]][[教育家|教育家]]及[[作家|作家]]，更以其[[戲曲|戲曲]]理論而出名。他是最早把[[昆曲|昆曲]]帶入中國大學成為正式科目者。<ref name="國學1"/><ref name="國學2">{{Cite web |url=http://www.guoxue.com/master/wumei/wm04.htm |title= 吴梅|accessdate=2015-01-21|author= |date=|work= |publisher= 國學網}}</ref><ref name="江蘇網">{{Cite web |url=http://sz.jschina.com.cn/qdsz/xqsz/201312/t1361542.shtml |title= 吴梅故居：藏书楼的曲韵年华|accessdate=2015-01-21|date=2013-11-15|author= |date=|work= |publisher= 中國江蘇網}}</ref>

吳梅在[[光緒|光緒]]十年（1884年）出生於江蘇[[苏州府|苏州府]][[长洲县|长洲县]]（今[[苏州市|苏州市]]）的一個破落仕宦之家，<ref name="江蘇網"/> 年幼時父母雙亡，因此過繼給[[吳長祥|吳長祥]]當嗣孫。十二歲時從[[潘少霞|潘少霞]]開始讀書考取功名，十八歲即以第一名補長洲縣學生員，然此後兩度[[鄉試|鄉試]]落第，自此摒棄科舉之念。[[百日維新|百日維新]]失敗後，曾寫《血飛花》悼念[[戊戌六君子|戊戌六君子]]。<ref name="新風">{{Cite web|url=http://jgjs.nanjing.gov.cn/20795/20814/201405/t20140512_2825138.html|title=曲学大师吴梅|accessdate=2015-01-21|author=|date=2014-05-12|work=|publisher=南京新風|deadurl=yes|archiveurl=https://web.archive.org/web/20160304090728/http://jgjs.nanjing.gov.cn/20795/20814/201405/t20140512_2825138.html|archivedate=2016-03-04}}</ref> 光緒二十九年（1903年）前往上海的[[東文學社|東文學社]]學習日語。光緒三十一年秋受聘於[[東吳大學|東吳大學]]，兩年後應[[柳亞子|柳亞子]]之約，加入文學團體神交社，宣統二年（1910年）開始先後在蘇州存古學堂、南京第四師範等中學擔任教師。1912年曾加入[[南社|南社]]。<ref name="國學1"/>

1917年[[蔡元培|蔡元培]]看到了吳梅所寫的在《顾曲麈谈》等文章後，聘請他到[[北京大学|北京大学]]擔任教授、昆曲組導師，[[梅兰芳|梅兰芳]]、[[韩世昌|韩世昌]]等曾是他的學生，一度被當做北大校歌的《北大二十周年纪念歌》和《正宫锦缠道·示北雍诸生》也是由吳梅來譜曲作詞的。<ref name="大公網">{{Cite web|url=http://www.takungpao.com/fk/content/2012-09/02/content_1055418.htm|title=吴梅崑曲写北大|accessdate=2015-01-21|author=|date=2012-09-02|work=|publisher=大公網|archive-url=https://web.archive.org/web/20150122111246/http://www.takungpao.com/fk/content/2012-09/02/content_1055418.htm|archive-date=2015-01-22|dead-url=yes}}</ref> 1922年秋應[[國立東南大学|國立東南大学]]（後更名[[國立中央大學|國立中央大學]]、[[南京大學|南京大學]]）国文系主任[[陈中凡|陈中凡]]之邀，举家搬到[[南京|南京]]大石桥，擔任東大教授。在此期間與學生們成立研究戲曲的“潛社”。<ref name="新風"/> 此外他也曾在广州[[中山大學|中山大學]]、南京[[國立中央大學|中央大學]]、上海[[光華大學|光華大學]]任教，并曾兼课[[金陵大學|金陵大學]]。<ref name="國學1"/>

[[七七事變|七七事變]]發生之後，他在當年9月12日携家人离苏州，後輾轉於[[武漢|武漢]]、<ref>{{Cite web|url=http://www.mingpaomonthly.com/cfm/Archive2.cfm?File=200912/ph/01a.txt|title=張充和與吳梅先生的曲緣|accessdate=2015-01-21|author=孫康宜|date=2009-12|work=|publisher=明鏡月刊|deadurl=yes|archiveurl=https://web.archive.org/web/20150511183142/http://www.mingpaomonthly.com/cfm/Archive2.cfm?File=200912%2Fph%2F01a.txt|archivedate=2015-05-11}}</ref> [[湘潭|湘潭]]、[[桂林|桂林]]、[[昆明|昆明]]等地，1939年1月11日與家人從昆明乘汽車兼步行前往雲南[[大姚|大姚]][[李旗屯|李旗屯]]，同年3月17日在大姚因喉病复发而病逝。<ref name="新風"/><ref name="國學1">{{Cite web |url=http://www.guoxue.com/master/wumei/wumei.htm |title= 吳梅|accessdate=2015-01-21|author= |date=|work= |publisher= 國學網}}</ref>

==成就==
吳梅以其在戲曲創作、研究與教學出名，被譽爲“近代著、度、演、藏各色俱全之曲學大師”。其所作的[[傳奇|傳奇]]有《風動山》、《綠窗怨記》、《東海記》、《血飛花》（未刊行）、《義士記》（未刊行）五部，雜劇有《軒亭秋》、《暖香樓》、《湘真閣》、《落茵記》、《雙淚碑》、《無價寶》、《惆悵爨》七部。此外還有《中國戲曲概論》等大量戲曲論著存世。<ref name="國學2"/>

吳梅在當時的文学界享有盛名，他是[[朱自清|朱自清]]、[[田汉|田汉]]、[[郑振铎|郑振铎]]、[[齐燕铭|齐燕铭]]、[[梅兰芳|梅兰芳]]、[[俞振飞|俞振飞]]等人的老師。1993年，[[中国文联|中国文联]]、[[中国戏剧家协会|中国戏剧家协会]]在其故鄉苏州召舉辦了“吴梅诞生100周年学术讨论会”。<ref name="國學2"/>
