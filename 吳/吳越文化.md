'''吳越文化'''，又稱「'''江浙文化'''」，是指[[江浙|江浙]]地區的地域文化，其範圍大致包括今日的[[蘇南|蘇南]]、江西东北的[[上饶|上饶]]地区[[皖南|皖南]]和[[浙江省|浙江省]]及[[上海市|上海市]]，基本與整個[[吳語方言|吳語方言區]]（包括有系属争议的徽语）相吻合。

[[吴越文化|吴越文化]]形成于[[周代|周代]]，于越部落和勾吴部落本属一族，文化相近，使用同一种语言。由于生产力水平低下，被中原的夏部落视为[[蛮夷|蛮夷]]。之后吴越二国先后引入了[[河洛|河洛]]地区和[[楚国|楚国]]的先进诸夏文化，相继通过北伐成为[[春秋|春秋]]时代的霸主，并在互相征服之中融合，构成了吴越文化的雏形。到了[[晋朝|晋朝]]由于[[五胡乱华|五胡乱华]]，开始受到南渡的门阀贵族影响，成为中国文化中精致典雅的代表。[[唐代|唐代]]吴越经济超过关中和河洛地区，影响力进一步扩大。同时，原先北方文化在北方地区在战争和多次的少数民族统治中淡化，因此，吴越文化又保留了较多了传统[[中国文化|中国文化]]。例如，吴语比较细软优雅，享有“吴侬软语”的美称；吴越饮食以香甜可口为特征，与以香辣为主的西南、以干煮炖为主的北方饮食成鲜明的对比；吴越地区辈出文人墨客和科学家，古代状元和现代中国院士中吴越地区人士要占近四成。

== 歷史 ==
=== 遠古時代 ===
從考古資料來看，吳越文化的淵源可以推溯到[[舊石器時代|舊石器文化時期]]。1985年春吳縣(今[[蘇州市|蘇州市]]吳中区)三山島發現的一處舊石器時代文化遺址，是[[長江|長江]]下遊首次發現的舊石器地點，為研究吳越文化的起源提供了重要資料。

=== [[先秦|先秦]]至[[漢|漢]]時期 ===
早在[[春秋時代|春秋時代]]已有[[吳國|吳國]]和[[越國|越國]]兩個國家，在此期間互相爭霸。但一直到[[漢朝|漢朝]]，江南還處於落後的狀態，一般時人對吳越文化的印象是粗獷，但又帶有一定程度的審美觀。

=== 六朝至隋唐時期 ===
自三國時起因有[[孫權|孫權]]建立的[[吳國|吳國]]，加上中原不斷戰亂，不少原居於黃河流域者紛紛南下，一直至[[南北朝|南北朝]]仍不止，使江南一帶的人口開始有所增長。南渡者把本來是中原的士族文化帶進江南，使吳越文化開始帶有一種書生的氣息。

當時吳越文化主要是以[[玄學|玄學]]為主，是西晉亡國之際士人南遷時，由中原帶入江南，為之江左玄學。

=== [[南宋|南宋]]至明清時期 ===
[[唐亡|唐亡]]以後，中原和[[江南|江南]]一帶各自分裂成多個小國。其中南方因為經濟相對穩定，加上此時又有一批北方人口南遷，使江南一帶的經濟、文化開始超越北方。中原士族對温婉、清秀、恬静的追求，漸漸滲入了吴越文化，形成吳越文化獨有的陰柔特質。

明清時代由於經濟重心在江南，吳越文化在全國中佔了最重要位置。這時的吳越文代具有濃厚的書卷氣息，其中在明代，[[蘇州|蘇州]]就出了[[唐伯虎|唐伯虎]]、[[文徵明|文徵明]]、[[祝枝山|祝枝山]]、[[徐禎卿|徐禎卿]]等四大才子；而流行於[[清代|清代]]的[[考據學|考據學]]正是以江南人為代表。

=== 近現代時期 ===
進入近代，江南經濟逐漸集中於上海。由於全國各地人的聚居，加以外來文化影響，吳越文化逐漸形成以[[上海|上海]]為中心。民國時代不少著名文化人物都是在上海活動，例如[[魯迅|魯迅]]，上海人的文化就是吳越文化的領頭羊。

== 內涵特質 ==

== 語言 ==
[[吳語|吳語]]是吴越地區的通用語言。[[吴语|吴语]]有时也被称为江南话。吴语主要通行于苏南和浙江地区，但位于江南的[[南京|南京]]操由金陵雅音演变成的官话。江南一帶的“[[下江官話|下江官話]]”又稱“[[江南官話|江南官話]]”、“[[淮语|淮语]]”，其中[[南京官話|南京官話]]在近代仍通行南方多个省份，曾一度被倡議作為華語正音。

== 建築 ==
吳越文化在建築方面的特色，可追溯到[[河姆渡|河姆渡]][[木結構|木結構]]房屋。其[[榫卯|榫卯]]件加工得相當規整，且在房屋上不同的部位用不同的榫卯；出土的木構殘件有數千件之多，大致可分為柱頭與柱腳榫卯、平身柱與梁枋交接榫卯、轉角柱榫卯、欄杆榫卯、帶梢釘的榫卯（用於受拉杆件）與企口板（用於地板）之類，據研究，它們是幹欄式房屋的構件。從遺址判斷，房屋長有23米，進深達7米，可能是氏族的“集體宿舍”。這表明當時的木構技術已經經過相當長時間的發展了。吳地建築除了木結構外還有石室，至今仍有很多散佈在太湖丘陵中。與甯鎮地區湖熟文化的土墩墓不同，太湖地區的土墩墓是石室外加封土，也未發現內有葬具或人骨。這些石室大多在山峰或山脊上，大小不等，大的有16米×2米×5米，小的有4米×0.8米×0.5米；多為下寬上窄的狹長形狀。《[[吳越春秋|吳越春秋]]》與《[[越絕書|越絕書]]》中記載有[[勾踐|勾踐]]“拘於石室”，他在越國“走犬若耶，休謀石室”，供人居住的石室外加封土，可能是為了冬暖夏涼。關於石室的用途有多種意見，有人認為是墓，有人認為是藏軍洞，也有人提出多元說。

[[闔閭|闔閭]]為了爭霸的需要，聽從[[伍子胥|伍子胥]]的建議決定修築一座大城，伍子胥經過“相土嘗水，象天法地”，築成周長為47里的闔閭大城，即後世的[[蘇州城|蘇州城]]。還圍以周長為68里的外郭（《越絕書》）。2500年來，蘇州雖幾經破壞與再度繁榮，但城的基本形狀沒變，盤門、閶門等城門的名稱也沿用至今，這份遺產在世界上是罕見的。

== 科技 ==
吳越地區是中華文明的發祥地之一。[[春秋|春秋]]時期[[吳國|吳國]]即以其冶鑄名匠著稱於世；[[南北朝|南北朝]]以後，隨著全國經濟重心南移，吳地成為中國古代科技的重要舞臺，在各個朝代中湧現出像[[沈括|沈括]]、[[祖沖之|祖沖之]]、[[徐光啟|徐光啟]]、[[徐霞客|徐霞客]]、[[葉天士|葉天士]]、[[蒯祥|蒯祥]]這些能在天文、數學、物理、地學、醫藥與建築各門科學技術中代表國家水準的傑出專家。明末清初這裏是西學東漸的主要通道之一，到晚清中西文化再度開展交流時，以[[王韜|王韜]]、[[徐壽|徐壽]]、[[華蘅芳|華蘅芳]]、[[李善蘭|李善蘭]]、[[王季烈|王季烈]]為代表的吳地學者為各門近代科學在中國的傳播作了重大貢獻。

== 教育 ==
江南素以文化教育繁盛著稱。位於[[南京|南京]][[秦淮河|秦淮河]]的[[夫子廟|夫子廟]]是紀念[[孔子|孔子]]的廟宇，夫子廟建築群中的[[江南貢院|江南貢院]]在古代為科舉考試的考場，是江南科舉文化的象徵。江南貢院[[清代|清代]]出[[狀元|狀元]]58人，占全國狀元總數的52%。明清兩代，科舉[[進士|進士]]的比例當中，江南地區特別是江蘇、浙江以及安徽、江西等地一直佔有領先位置。

江南有許多著名的書院。[[四大書院|四大書院]]有三個位於(广义的)江南，即[[嶽麓書院|嶽麓書院]]、[[石鼓書院|石鼓書院]]、[[白鹿洞書院|白鹿洞書院]]。其他還有[[象山書院|象山書院]]、[[麗澤書院|麗澤書院]]、[[東林書院|東林書院]]、[[明道書院|明道書院]]、[[茅山書院|茅山書院]]等。
