'''吳重雨'''（Chung-Yu Wu，{{bd|1950年|5月31日|||吳}}），[[台灣|台灣]][[嘉義縣|嘉義縣]]人，電機工程學家，曾任國立交通大學校長。

== 學歷 ==
吳重雨於1950年生於[[台灣|台灣]][[嘉義縣|嘉義縣]][[東石鄉_(台灣)|東石鄉]]猿樹村，東石國小畢業。取得學位為1972年[[國立交通大學|國立交通大學]]電子物理學士，1976年[[國立交通大學|國立交通大學]]電子工程碩士，1980年[[國立交通大學|國立交通大學]]電子工程博士，2002年[[加州大學|加州大學]]柏克萊分校電機資訊博士後研究。

== 專長 ==    
吳重雨主要研究領域：
* 奈米電子與超大型積體電路。
* 包括混合訊號電路與系統設計。
* 生物晶片。
* 射頻積體電路及電腦輔助分析。

== 經歷 ==
吳重雨於1980年─1983年擔任[[國立交通大學|國立交通大學]]電子工程系[[副教授|副教授]]，1980年─1982年擔任[[國立交通大學|國立交通大學]]半導體中心副主任，1983年至今擔任[[國立交通大學|國立交通大學]]電子工程學系[[教授|教授]]，1984年─1986年擔任[[美國|美國]][[波特蘭大學|波特蘭大學]][[電機工程系|電機工程系]][[副教授|副教授]]，1986年─1989年擔任[[國立交通大學|國立交通大學]]電子工程學系系主任，1989年─1991年擔任[[國立交通大學|國立交通大學]]電子工程系所所長，1991年─1995年擔任[[國科會|國科會]]工程技術發展處處長，1995年─1998年擔任[[國立交通大學|國立交通大學]]研發長，2002年─2006年擔任[[國立交通大學|國立交通大學]][[電機資訊學院|電機資訊學院]][[院長|院長]]，2003年秋季擔任[[美國|美國]][[伊利諾大學|伊利諾大學]]香檳校區電機資訊工程系訪問教授，2004年為[[美國|美國]][[伊利諾大學|伊利諾大學]]香檳校區國際教授，2004年為[[傅爾布萊特計畫|傅爾布萊特計畫]]國際學者，2004年擔任[[國立交通大學|國立交通大學]][[講座教授|講座教授]]，2007年─2011年擔任[[國立交通大學|國立交通大學]][[校長|校長]]，2008年起擔任台灣生醫電子工程協會[[理事長|理事長]]、國際創新創業協會[[理事長|理事長]]，2008年起擔任國科會第二期奈米國家型科技計劃總主持人。<ref>[http://www.ics.ee.nctu.edu.tw/~cywu/aboutme.htm 吳重雨主要經歷]</ref>
