'''庵山'''位于[[福建省|福建省]][[建阳市|建阳市]][[童游街道|童游街道]]童游村，被称为“潭东第一名山”，距[[建阳|建阳]]市区约8公里，最高峰海拔1020米。登临绝顶，建阳、[[建瓯|建瓯]]、[[武夷山|武夷山]]风光尽收眼底。庵山因山中有庵而得名。<ref>据建阳县志记载，庵山地名的来历是因为唐明宗时，有一个叫石湖的处士，结庵其上，始称庵山。</ref>被[[福建省林业厅|福建省林业厅]]评为“[[福建省省级森林公园|福建省省级森林公园]]”。<ref name="建阳市庵山省级森林公园">{{cite web |url=http://www.fjforestry.gov.cn/InfoShow.aspx?InfoID=37993&InfoTypeID=5 |title=建阳市庵山省级森林公园 |accessdate=2013-06-24 |author= |publisher=福建省林业厅 |language=zh-hans |archive-url=https://web.archive.org/web/20130811102247/http://fjforestry.gov.cn/InfoShow.aspx?InfoID=37993&InfoTypeID=5 |archive-date=2013-08-11 |dead-url=yes }}</ref>
== 主要景点 ==
主要景点有灵泉寺、关刀峡、仙姑梳妆、青士迎宾、月景波光、鸳鸯柏树、白鹤仙母、龙坛等十八景。<ref name="建阳庵山介绍">{{cite web |url=http://nanping.cncn.com/jingdian/anshan/profile |title=建阳庵山介绍 |accessdate=2013-06-24  |author= |publisher=欣欣旅游网 |language=zh-hans }}</ref>山分南北双峰，北峰有石峥嵘，称狮子岩，岩下灵泉，号“庵山泉”，又名“龙井”；南峰有石平整，称“龙塘”，塘上垒石为龛，中藏仙骸；山顶有三皇君庙，历数百年香火不断。以庵山主峰为中心，方圆面积约2521公顷，主要森林植被为[[常绿阔叶林|常绿阔叶林]]，中上部天然林皆属生态公益林，其中522.53公顷列为生物多样性自然保护小区。<ref name="建阳市庵山省级森林公园">{{cite web |url=http://www.fjforestry.gov.cn/InfoShow.aspx?InfoID=37993&InfoTypeID=5 |title=建阳市庵山省级森林公园 |accessdate=2013-06-24 |author= |publisher=福建省林业厅 |language=zh-hans |archive-url=https://web.archive.org/web/20130811102247/http://fjforestry.gov.cn/InfoShow.aspx?InfoID=37993&InfoTypeID=5 |archive-date=2013-08-11 |dead-url=yes }}</ref>
=== 灵泉寺 ===
灵泉寺始建于[[唐|唐]][[贞观|贞观]]年间。[[明|明]][[万历|万历]]年间，邑人傅国珍修聋扩建。[[清|清]][[雍正|雍正]]十年(1732年)寺遭火焚。[[清|清]][[乾隆|乾隆]]三十年(1765年)，邑人傅光裔、傅克钦倡捐重建。灵泉寺分三重大门，一重门留有宋代石刻对联一副：“到此山门近天阀，始知蓬莱有仙翁”；二门留有石刻横批“灵泉古迹”；三门留有石刻横批“涌泉胜境”。寺中供祀着三皇元君塑像。灵泉寺前后有千年柳杉六棵。<ref name="建阳庵山介绍">{{cite web |url=http://nanping.cncn.com/jingdian/anshan/profile |title=建阳庵山介绍 |accessdate=2013-06-24  |author= |publisher=欣欣旅游网 |language=zh-hans }}</ref>
