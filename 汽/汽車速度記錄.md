这篇文章列出世界历史上[[车|车]]的最快速度。

1886年造出第一辆[[汽车|汽车]]的是[[Karl_Benz|Karl Benz]]的[[Benz_Patent_Motorwagen|Benz Patent Motorwagen]]，这辆车被公认的唯一最快速度为16公里每小时(10英里每小时)。


==二十世紀末至今量产型汽车最快速度纪录==

[[File:Jaguar_XJ_220_-_Flickr_-_exfordy_(2).jpg|thumb]]
[[File:SSC_Ultimate_Aero_TT_-_1.jpg|thumb]]

1993年，[[積架|積架]] [[:en:_Jaguar_XJ220|XJ220]]创造了量产型汽车最快速度纪录，速度为343公里每小时(213英里每小时)。
次年1994年，[[麥拿輪F1|麥拿輪F1]](McLaren F1)创造了372公里每小时(231英里每小时)的最快速度纪录。
到了2005年2月，被[[Koenigsegg_CCR|Koenigsegg CCR]]创造的403.45公里每小时(250.7英里每小时)所打破。
同年4月，量产型汽车最快速度纪录再被打破，[[Bugatti_Veyron_16.4|Bugatti Veyron 16.4]]创造了408.47公里每小时(253.81英里每小时)速度纪录。
这纪录一直到了2007年10月才被[[SSC_Aero|SSC Aero]]的限量版SSC Ultimate Aero TT所打破，世界上量产型汽车最快速度更改为412.99公里每小时(256.14英里每小时)。2010年8月，Bugatti Veyron Supersports再打破速度紀錄，時速為431.072 km/h(267 mph)。2014年比賽競技，科尼賽克One:1（Koenigsegg One:1）超越了Bugatti Veyron Supersports，以最大極速435km/h擊敗。而首次於2016年3月1日在日內瓦車展公開的Bugatti Chiron理論極速大約是463Km/h。

== 1903年-60英里每小时(96公里每小时) ==
几年之后[[Mercedes_35_hp|Mercedes 35 hp]]的衍生物——[[Mercedes_60_HP|Mercedes 60 HP]]，达到了60英里每小时(96公里每小时)。

== 1948年-160英里每小时(257公里每小时) ==
[[File:Teignmouth_Classic_Car_Show,_21_April_2013_(17).jpg|thumb]]
[[File:1959_Ferrari_410_Superamerica.jpg|thumb]]
设计者花了45年的时间设计[[Mercedes_60_HP|Mercedes 60 HP]]的外形，目的是让它更可靠和一贯性，最后速度达到了160英里每小时(257公里每小时)。[[1948_Jaguar_XK120|1948 Jaguar XK120]]正好在年中创造出这一纪录。

第一辆[[法拉利|法拉利]]汽车在1948年制造完成，使用了他们的[[V12|V12]][[国际汽车大奖赛|国际汽车大奖赛]]（Grand Prix cars）车的发动机。[[410_Superamerica|410 Superamerica]]曾经在1948年末达到257公里每小时(160英里每小时)之上。

== 1961年-150英里每小时(241公里每小时) ==
[[File:Chelsea_Auto_Legends_2012_(7948685856).jpg|thumb]]
[[Jaguar_E-Type|Jaguar E-Type]](或称XK-E)在1961年创下了150英里每小时(241公里每小时)的速度纪录，这几乎是一辆最便宜的[[法拉利|法拉利]]汽车。

== 1962-1992年問題 ==


在1962年已達185 mph (298 km/h)，但在1966年僅有 170 mph (274 km/h)，最快的車反而慢了，而在某些測試中當時的最快名車可達 300-320 km/h，但在另一些記錄僅有270-290 km/h，當時沒法作統一的測試。原因見[[talk:汽車速度記錄|talk:汽車速度記錄]]。
 

== 1994 - 240 mph (386 km/h) ==
1994年，McLaren F1達到時速240英哩(386 公里)並且此記錄ㄧ直保持到2004年才被瑞典的Koenigsegg CCR以241.1英哩(388 公里)打破。

== 附一:在非量產車中的記錄 ==

{{main|陸上速度記錄}}

[[超音速推進號|超音速推進號]](Thrust SSC)在1997年創下了763英里每小時(1228公里每小時)的速度紀錄，速度超越音速(1224公里每小時)，並於高速行駛其間發出[[音爆|音爆]]的巨響。迄今（2005年中）為止Thrust SSC是世界陸上極速紀錄冠軍。
