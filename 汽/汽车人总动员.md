《'''汽车人总动员'''》，是一部2015年中國[[3D动画|3D动画]]喜劇片，由[[卓建荣|卓建荣]]執導、蓝火焰影视制作，於2015年7月4日在中國上映。

== 概述 ==
这部电影由卓建荣执导，厦门蓝火焰影视公司出品，卡丁有限公司投资赞助。在2015年6月1日的新闻发布会上，公布该电影将于2015年7月4日全国公映<ref>{{cite news|title=《汽车人总动员》定档7.4 主海报揭秘呆萌汽车人|url=http://ent.cqnews.net/html/2015-06/02/content_34382986.htm|accessdate=2015-07-15|agency=华龙网}}</ref>，并发布了电影主海报。电影讲述了少年卡卡发明人车对话的K系统，在小助手丁丁的帮助下，设计出K1、K2、K3三辆车，并描述赛车与人的互动关系<ref>{{cite news|title=《汽车人总动员》定档7月4日 开启智能赛车时代|url=http://ent.qq.com/a/20150602/022111.htm#p=1|accessdate=2015-07-15|work=腾讯娱乐网}}</ref>。

== 侵權爭議 ==
制作方称试图打造“绿色正能量”的三无（“无色情”、“无暴力”、“无公害”）电影，但海报上的车辆跟[[皮克斯动画|皮克斯动画]]《[[赛车总动员|赛车总动员]]》中的车辆极其相似而被质疑[[山寨|山寨]]，甚至海报上的车轮涉嫌故意遮挡“人”字，以混淆视听<ref>{{cite news|title=《赛车总动员》遭国产动画山寨 网友：当中国小孩弱智吗|url=http://news.lizhixia.com/hyyx/43754.html|accessdate=2015-07-15|agency=岁岁网|archive-url=https://archive.is/20150603045131/http://news.lizhixia.com/hyyx/43754.html|archive-date=2015-06-03|dead-url=yes}}</ref>。而《汽车人总动员》与蓝火焰影视此前制作的动画《K时代》的故事情节几乎完全一样<ref name=Mtime />。此电影也遭到中国网民的一致性抨击，在[[豆瓣|豆瓣]]电影评分已经被网友评到了2.1（而豆瓣允许打出的最低分是2分），成为豆瓣历史上的最低分电影；同时，在[[IMDB|IMDB]]上也得到了最低的一分，还有在中国的外国网友写影评抨击该片的拙劣抄袭行为<ref>{{cite news|title=好久没见过这么芳草天的电影和导演了|url=http://news.163.com/api/15/0707/09/ATTM2PM100014TUH.html|accessdate=2015-07-15|work=网易|date=2015-07-07}}</ref>。

随着抄袭新闻发酵，海外媒体逐渐注意到事态变化，[[迪士尼|迪士尼]]官方回应称：“我们对出现这样的事件感到担忧。”<ref>"[http://edition.cnn.com/2015/07/07/china/china-movie-disney-cars/index.html Copycat movie? Filmmaker denies knocking off Pixar's 'Cars']" ([http://www.webcitation.org/6Zt6hZOpn Archive]). [[CNN|CNN]]. July 7, 2015. Retrieved on July 9, 2015.</ref>而[[日本电视台|日本电视台]]與[[TBS電視台|TBS電視台]]也曾质疑《汽车人总动员》抄袭，认为片中三辆车的外形酷似《赛车总动员》的主角，电视台工作人員还曾致电给卓建荣求证，但卓建荣自称“是学习不是抄袭”。而当网友在[[新浪微博|新浪微博]]上就此事质疑时，后者则将表达批评的网友称为“新时代的[[汉奸|汉奸]]”<ref name=Mtime>{{cite news|title=《汽车人总动员》被指抄袭 迪士尼官方独家回应：对此事件感到担忧|url=http://news.mtime.com/2015/07/07/1544375.html|accessdate=2015-07-15|agency=Mtime时光网|date=2015-07-07}}</ref>。《[[十万个冷笑话|十万个冷笑话]]》导演[[卢恒宇|卢恒宇]]也质疑《汽车人总动员》抄袭，同样遭到卓建荣的斥责<ref>{{cite news|title=国产动画《汽车人总动员》指抄袭美国 导演不反思破口大骂汉奸|url=http://www.qi-che.com/shehuixinwen/201507071444694.html|accessdate=2015-07-15|agency=汽车中国网}}</ref>。同月，[[金星_(舞者)|金星]]在其脱口秀节目《金星秀》中，直接批评制作方“一个抄袭者手上幹的是小偷的勾当，嘴里讲的是流氓的逻辑。”<ref>{{cite news|title=金星骂陶喆出轨男不为渣男只为渣 以小偷流氓比喻电影抄袭事件|url=http://www.shangc.net/news/n/60082_3.html|accessdate=2015-07-18|work=尚之潮网|agency=尚之潮网}}</ref>

随着批评浪潮继续，媒体开始分析此类劣质国产[[动画片|动画片]]的原因，一方面集中批评于“创新的缺失”<ref>{{cite news|title=“汽车人总动员”的背后折射出创新的匮乏|url=http://www.wenming.cn/wmpl_pd/yczl/201507/t20150713_2728887.shtml|accessdate=2015-07-15|agency=中国文明网|date=2015-07-14}}</ref>。而更多的评论，集中在中国政府对国产动画片的政府补贴、对进口动画限制播出政策，和严格的电影审查等方面。2014年5月30日，卓建荣在接受《[[海峡都市报|海峡都市报]]》采访时表示，进军动漫产业的动机是政府有相关的政策扶持<ref>{{cite news|author1=魏暾、许茵茵|title=卓建荣： “影视+动漫” 两腿错步向前|url=http://szb.mnw.cn/html/2014-05/30/content_4259768.htm|accessdate=2015-07-15|work=新厦门周刊|agency=海峡都市报|date=2014年5月30日}}</ref>。新闻媒体评分析，本片长为85分钟，且目前已经在全国性的院线公映，按照政策，该公司至少可以拿到255000元[[补贴|补贴]]。此外蓝火焰还能从另外两条「人才培训补贴」和「房租补贴」中获益。<ref>{{cite news|title=国产动画：政府补贴的阿斗扶不起|url=http://news.163.com/special/reviews/guochandonghua0715.html|accessdate=2015-07-15|agency=网易新闻|date=2015-07-15}}</ref>但在2016年6月23日，卓建荣称由于“迪士尼和厦门相关主管部门领导一句话”的缘故，该电影没有政府补贴。<ref name="huifu">{{cite news|title=《汽车人总动员》导演回应抄袭:迪士尼抹黑我|url=http://news.17173.com/content/2016-06-24/20160624003621528.shtml}}</ref>

2016年4月，在杭州“第12届中国国际动漫节”上，卓建荣宣布《汽车人总动员2-疯狂逆袭》将于2017年暑期档上映。<ref>[http://news.mtime.com/2016/04/26/1554827.html 《汽车人总动员》居然要拍续集？计划明年上映 预算三千万拟邀TFBoys]</ref>但直至2018年底，該部電影仍未上映。

== 票房 ==
《汽车人总动员》於2015年7月4日上映，一周以来票房总收入300多万人民币，而制作方自称该电影拥有1000万元的制作成本。卓建荣认为，此事是网友的攻击导致票房成绩不佳。尽管面对接踵而来的争议，他又爆料称，公司已经计划拍摄《汽车人总动员2》<ref>{{cite news|title=《汽车人总动员》涉抄袭被批 导演计划拍续集|url=http://www.sznews.com/ent/content/2015-07/13/content_11904373.htm|accessdate=2015-07-15|work=深圳新闻网}}</ref>。7月14日，该片在上海下线。并截止此日，该片票房累计475万元。博纳银兴影城等均声明对此片的不支持<ref>{{cite news|title=《汽车人总动员》偷瓜得豆，好！|url=http://jxcomment.jxnews.com.cn/system/2015/07/14/014044275.shtml|accessdate=2015-07-15|agency=中国江西网-大江时评|date=2015-07-14}}</ref>。

之后，制片方已经将影片卖给[[腾讯视频|腾讯视频]]和[[土豆网|土豆网]]，提供付费观看。

==相关诉讼==
2016年6月23日，迪士尼因《汽车人总动员》涉嫌抄袭《赛车总动员》而起诉厦门蓝火焰影视动漫有限公司、北京基点影视文化传媒有限公司、上海聚力传媒技术有限公司。<ref>[http://news.ifeng.com/a/20160622/49218407_0.shtml 国产电影《汽车人总动员》被控侵权 迪士尼索赔300万]</ref>导演卓建荣指责迪士尼乱用法律，造成司法不公正。<ref name="huifu">{{cite news|title=《汽车人总动员》导演回应抄袭:迪士尼抹黑我|url=http://news.17173.com/content/2016-06-24/20160624003621528.shtml}}</ref>

2016年12月29日，上海市浦東新區人民法院一審判處迪士尼勝訴，蓝火焰需向迪士尼賠償損失100萬元人民幣，而基点亦須對其中的80萬元和承擔連帶賠償負任，兩公司亦同時須賠償迪士尼的合理維權開支35萬元<ref name=appledaily>{{cite news|url=http://hk.apple.nextmedia.com/realtime/china/20161229/56107839|title=山寨動畫抄《反斗車王》 被判賠償130萬人仔|publisher=蘋果日報|date=2016-12-29|accessdate=2016-12-29}}</ref>。
