'''哥本哈根博物館'''（{{Lang-da|Københavns Bymuseum}}）是位於[[丹麥|丹麥]]首都[[哥本哈根|哥本哈根]][[韋斯特布羅|韋斯特布羅]]的一座[[博物館|博物館]]，成立於1901年。<ref>{{Cite web|url = http://denstoredanske.dk/Danmarks_geografi_og_historie/Danmarks_historie/K%C3%B8benhavns_historie/K%C3%B8benhavns_Bymuseum|title = Københavns Museum|accessdate = 2016-03-01|author =|date =|publisher = [[丹麦大百科全书|Den Store Danske Encyklopædi]]|language = da}}</ref>這座博物館是哥本哈根的官方博物館，記錄哥本哈根历史的各个时期。<ref>{{Cite web|url = http://www.copenhagen.dk/en/about|title = About|accessdate = 2016-03-01|author = |date = |publisher = Københavns Museum|deadurl = yes|archiveurl = https://web.archive.org/web/20160229131134/http://www.copenhagen.dk/en/about/|archivedate = 2016年2月29日|df = }}</ref>哥本哈根博物館靠近哥本哈根中央車站，在入口處外曾有一座巨大的中世紀時期哥本哈根的模型，但模型在2013年被拆除。<ref>{{Cite web|url = http://www.copenhagen.dk/dk/om_museet/presse/hvor_er_modelbyen/|title = Hvor er modelbyen?|accessdate = 2016-03-01|author = |date = |publisher = Københavns Museum|language = da|deadurl = yes|archiveurl = https://web.archive.org/web/20160304131817/http://www.copenhagen.dk/dk/om_museet/presse/hvor_er_modelbyen/|archivedate = 2016年3月4日|df = }}</ref>2015年10月9日，博物馆暂停开放，将于2017年迁址至[[哥本哈根市中心|哥本哈根市中心]]。<ref>{{cite web|url=http://www.copenhagen.dk/en/about/press/the_museum_moves_to_stormgade/|title=The Museum Moves to Stormgade|accessdate=2016-03-01|author=|date=|publisher=Københavns Museum}}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>
