'''哥德堡-蘭德維特機場'''（{{lang-sv|Göteborg-Landvetter flygplats}}；{{airport codes|GOT|ESGG}}）是[[瑞典|瑞典]][[哥德堡|哥德堡]]的一個[[國際機場|國際機場]]，由[[瑞典民航管理局|瑞典民航管理局]]營運。它是瑞典全國第二大的機場（次於[[斯德哥爾摩-阿蘭達機場|斯德哥爾摩-阿蘭達機場]]），於2006年共有430萬人次使用<ref> {{cite web |url=http://www.lfv.se/templates/LFV_ListArticle____45955.aspx |title=存档副本 |accessdate=2008-03-04 |deadurl=yes |archiveurl=https://web.archive.org/web/20080612164215/http://www.lfv.se/templates/LFV_ListArticle____45955.aspx |archivedate=2008-06-12 }} </ref>。

蘭德維特機場位於[[哥德堡|哥德堡]]以東20公里、[[布羅斯|布羅斯]]以西40公里，得名自附近的[[海呂達|海呂達]]縣小鎮[[蘭德維特|蘭德維特]]。它擁有國內和國際兩個大堂，但現時都視為同一個大堂。現時所有行李都需經過檢查，故只有一個登記區域。機場的離境和入境大堂都位於同一層，國內大堂位於國際離境大堂旁，而國際離境大堂又位於國際入境大堂旁。

蘭德維特機場於1977年啟用<ref>{{cite web |url=http://www.lfv.se/templates/LFV_InfoSida_Bred____2637.aspx |title=存档副本 |accessdate=2008-03-04 |deadurl=yes |archiveurl=https://web.archive.org/web/20080320050708/http://www.lfv.se/templates/LFV_InfoSida_Bred____2637.aspx |archivedate=2008-03-20 }} </ref>，自此，[[托斯蘭達機場|托斯蘭達機場]]的服務便遷至蘭德維特機場<ref>{{cite web |url=http://www.lfv.se/templates/LFV_InfoSida_Bred____2634.aspx |title=存档副本 |accessdate=2008-07-02 |0=2008-06-12 |deadurl=yes |archiveurl=https://web.archive.org/web/20080612164210/http://www.lfv.se/templates/LFV_InfoSida_Bred____2634.aspx |archivedate=2008-06-12 }}</ref>。

== 歷史 ==
由於舊[[托斯蘭達機場|托斯蘭達機場]]被認為容量太小，蘭德維特機場便於1972年開始建造。為了建造這機場，Kroksjön進行填海工程，成為現時跑道位置。工程十分浩大，動用了1400噸炸藥和300部鑽山機器，炸毀了390萬立方米的山。

建築物於1975年開始建造，兩年後才完成。工程總開支為6億3千萬克朗。1977年10月1日開幕儀式由國王[[卡爾十六世·古斯塔夫|卡爾十六世·古斯塔夫]]出席下舉行。他在開幕禮演講中說道：「我只能說這個擁有瑞典五份之一人口，和大量工廠和公司的地區，現在可以使用一個十分高國際標準的現代機場了。」<ref>"{{lang|sv|...Jag kan bara konstatera att denna region som omfattar nästan en femtedel av Sveriges befolkning och har ett stort antal industrier och företag har nu fått en toppmodern flygplats av mycket hög internationell klass. ...}}"</ref>

首班航機於10月3日上午6時開出。

1980年代間，機場建造了一個停車場，國內航班大堂也有了一個區域客運站，更少航空公司使用閘口在區域內運輸乘客。蘭德維特機場酒店於1987年興建。

機場最繁忙的日子，莫過於1990年[[歐洲優勝者盃|歐洲優勝者盃]]足球決賽在哥德堡舉行的那天了。當日早上，約2萬名球迷聚集於專用樓層，自至傍晚才離開。有更多人則留在普通樓層。另一個繁忙日子是1995年[[世界田徑錦標賽|世界田徑錦標賽]]後，成千上萬來自世界各地的乘客在這機場歸家。

在2007至2008年間，入境大堂進行擴建，以容納每年540萬乘客<ref>[http://www.lfv.se/templates/LFV_InfoSida_Bred____2082.aspx Landvetter Flygplats historia] {{webarchive|url=https://web.archive.org/web/20070825211107/http://www.lfv.se/templates/LFV_InfoSida_Bred____2082.aspx |date=2007-08-25 }}</ref>。現時的行李大堂將會增加三份一運輸帶，也會改為使用旋轉式運輸帶。此外，海關和入境服務亦會遷至其他地方<ref>[http://www.lfv.se/templates/LFV_ListArticle____46245.aspx Ankomsthallen byggs om]{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

== 地面運輸 ==
機場與哥德堡市間之路程是25公里。乘搭[[彩虹巴士|彩虹巴士]]來往哥德堡市中心和機場，只需20分鐘；來往[[哥德堡中央車站|哥德堡中央車站]]和機場則需30分鐘。此外，也有巴士前往[[布羅斯|布羅斯]]。

機場內有7300個泊車位。
