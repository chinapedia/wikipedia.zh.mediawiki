'''博洛尼亚足球俱乐部'''（'''Bologna Football Club 1909'''）是在[[意大利|意大利]]城市[[博洛尼亚|博洛尼亚]]建立的一家足球俱乐部，成立于 1909年10月3日，2009年是博洛尼亞的百周年，现時在[[意大利足球甲級聯賽|意大利足球甲級聯賽]]征战。

博洛尼亞在二次大戰前成績輝煌，曾贏得多屆[[意甲|意甲]]聯賽冠軍。但最近一次卻已是1963–64年的事。八十年代起一直在低組別聯賽作賽，至1996年才回到意甲。但因實力平平，一直只在意甲中下游位置，2004–05年球季更降回落[[意乙|意乙]]。

== 歷史 ==
'''博洛尼亞足球會'''（'''Bologna Football Club'''）的建立是由[[奧地利人|奧地利人]]艾米里奥·阿尔斯坦（Emilio Arnstein）一手促成，他在[[維也納|維也納]]及[[布拉格|布拉格]]讀[[大學|大學]]時開始對[[英式足球|足球]]產生興趣，更在較早時候與兄弟在奧地利成立「黑星足球隊」（Black Star F.C.）。球會於1909年10月3日在[[義大利|義大利]]北部城市[[博洛尼亚|博洛尼亚]]成立，初時名為「AGC博洛尼亚」（''AGC Bologna''），當時採用的紅藍間條球衣源自[[水晶宫足球俱乐部|水晶宮]]。於1910年3月20日進行首場比賽，以9-1大勝同市球隊「維魯斯博洛尼亞」（''Virtus Bologna''）。
 
成立後的首個球季參加地區聯賽，贏得冠軍後獲升班到「溫尼托-艾米里亞諾聯賽」（''Group Veneto-Emiliano''），在參賽四個球季的期間，名次從未低於第五位。博洛尼亞其後參加「北部聯賽」（Northern League），直到[[第一次世界大戰|第一次世界大戰]]爆發才中止所有足球比賽。

;冠軍：1920及1930年代
一戰後博洛尼亞取得更大的成就，於1919/20年球季首次晉級北部聯賽的準決賽；翌年更進一步殺入決賽，但以1-2不敵當時的勁旅[[普罗韦尔切利足球俱乐部|-{zh-hans:普罗韦尔切利;zh-hk:維塞利;}-]]（Pro Vercelli）；1923/24年球季再次晉級決賽，惟敗於最終贏得全國錦標賽冠軍的[[热那亚板球与足球俱乐部|热那亚]]。

1924/25年球季博洛尼亞先取得北部聯賽冠軍，再在全國錦標賽以6-0一面倒擊敗[[羅馬足球俱樂部|羅馬]]，首次登上全國冠軍的寶座；博洛尼亞於1928/29年重溫舊夢，贏得最後一屆的義大利全國錦標賽冠軍，翌年[[意甲|意甲]]正式成立。在[[第二次世界大戰|第二次世界大戰]]前，博洛尼亞分別於1935/36年、1936/37年、1938/39年及1940/41年四次奪得意甲聯賽冠軍。

;後二戰時期
二戰後博洛尼亞成績稍遜，在1950年代到1960年代期間在聯賽經常徘徊在4、5或6位之間，直到1963/64年球季才再度奪冠，這亦是球隊最近一次贏取意甲冠軍，合共獲得7屆義大利全國聯賽錦標。

此後博洛尼亞亦不是一落千丈，在1970年代曾贏得兩次[[意大利盃|意大利盃]]冠軍；1969/70年的意大利盃仍以四隊小組決賽定冠軍，博洛尼亞以6戰4勝1和1負壓倒[[都灵足球俱乐部|-{zh-hans:都灵; zh-hk:拖連奴;}-]]、[[卡利亚里足球俱乐部|卡利亞里]]及[[瓦雷泽足球俱乐部|-{zh-hans:瓦雷泽;zh-hk:華雷斯;zh-tw:瓦雷澤;}-]]（Varese）贏得冠軍；1973/74年意大利盃決賽對[[巴勒莫足球俱樂部|-{zh-hans:帕勒莫; zh-hk:巴勒莫;}-]]，戰至[[加時賽|加時]]後仍然1-1平手，博洛尼亞以[[互射十二碼|互射十二碼]]4-3獲勝再奪冠軍獎盃<small><ref>{{cite web |language=it |url=http://storiedicalcio.altervista.org/palermo_bologna_1974.html |title=Quando al PALERMO scipparono la COPPA ITALIA |publisher=Storie di Calcio  |date= |accessdate=2009-08-27}}</ref></small>。

;升升跌跌
1980年代博洛尼亞開始步向衰落，首先於1981/82年球季與[[卡利亚里足球俱乐部|卡利亞里]]及[[热那亚板球与足球俱乐部|热那亚]]競逐留級，直到最聯賽煞科日主場以1-2負於[[阿斯科利足球俱樂部|阿斯科利]]才宣佈護級失敗，歷來首次降班到[[意乙|意乙]]作賽。但頹勢未止，翌年再降一級到[[意丙|意丙]]，幸而隨即獲得意丙亞軍返回意乙。在意乙打滾四年後，於1987/88年球季贏得意乙冠軍回升[[意甲|意甲]]。但雄風不再，博洛尼亞僅在意甲維持了三季，於1990/91年球季以墊底成績降班意乙。兩年後再次降班意丙。這次花了兩年才贏得意丙冠軍升班意乙，再奪得意乙冠軍，於1996年連續兩年奪冠升班重返意甲。

博洛尼亞在意甲站穩中游位置，更在1998年兩回合累計3-0擊敗[[波蘭|波蘭]]的[[羅切霍茹夫足球俱樂部|羅切霍茹夫]]贏得[[欧洲足联国际托托杯|圖圖盃]]而取得[[歐洲足協盃|歐洲足協盃]]的入場券。好景不常，博洛尼亞在2004/05年球季與[[帕尔玛足球俱乐部|-{zh-hans:帕尔玛; zh-hk:帕爾馬;}-]]進行留級附加賽，兩回合累計1-2落敗而再度降班。

;近期發展
[[File:Roberto_Mancini.jpg|thumb]]是博洛尼亚青訓產品]]
雖然博洛尼亞在降班後失去多員猛將，仍被視為可以立即爭取升班的熱門，但季初的差劣成績導致老於經驗的主教練[[伦佐·乌利维埃里|伦佐·乌利维埃里]]（Renzo Ulivieri）被解雇，由[[安德烈·曼多里尼|安德烈·曼多里尼]]（Andrea Mandorlini）接任掌舵。曼多里尼未能扭轉頹勢，上任僅兩個月，於2006年3月5日被炒，乌利维埃里回朝再度執教，最終博洛尼亞只能取得第8位。翌季乌利维埃里與班主阿尔弗雷多·卡索拉（Alfredo Cazzola）意見不合，於2007年4月14日再被解雇，原任助教的[[卢卡·塞科尼|卢卡·塞科尼]]（Luca Cecconi）暫掌帥印，球隊僅取得第7位的中游成績。2007/08年球季博洛尼亞找來[[丹尼尔·阿里戈尼|丹尼尔·阿里戈尼]]（Daniele Arrigoni）帶領球隊，最終不負眾望，取得聯賽亞席直接升班[[意甲|意甲]]。

在升班意甲前卡索拉與一個以[[美國|美國]]為基地的財團達成賣盤協議，惟交易最終取消。卡索拉只有將球會賣給一個由[[弗兰西斯卡·梅纳里尼|-{zh-hans:弗兰西斯卡·梅纳里尼; zh-hk:法蘭西斯卡·文納蓮妮;}-]]（Francesca Menarini）牽頭的本地集團，文納蓮妮出任球會主席，使她成為意甲第二位女性主席。阿里戈尼獲得新領導層的信任繼續擔任主教練，重返意甲頭炮一鳴驚人，作客[[聖西路球場|聖西路球場]]，憑著[[弗朗西斯科·瓦里亚尼|弗朗西斯科·瓦里亚尼]]（Francesco Valiani）的致勝入球以2-1擊敗[[米兰足球俱乐部|AC米蘭]]；但接著卻是慘不忍睹的9戰8負差劣戰績，在1-5大敗於[[卡利亚里足球俱乐部|卡利亞里]]腳下後，阿里戈尼於2008年11月3日被辭退，隨即由[[施尼沙·米赫洛域|施尼沙·米赫洛域]]取代職務<small><ref name="mihajlovic_bologna">{{cite news|language=it |url=http://www.bolognafc.it/pagen.asp?c=4915&m=11&l=1 |title=Il Bologna a Mihajlovic |publisher=Bologna FC 1909 |date=2008-11-03 |accessdate=2009-08-27 |deadurl=yes |archiveurl=https://archive.is/20090919123831/http://www.bolognafc.it/pagen.asp?c=4915&m=11&l=1 |archivedate=2009-09-19 }}</ref></small>。米赫洛域上場後取得連續5場和局的成績，於12月13日才以5-2擊敗護級對手[[都灵足球俱乐部|-{zh-hans:都灵; zh-hk:拖連奴;}-]]獲得首場勝仗，執教頭9場比賽取得不敗戰績。到了2009年4月14日博洛尼亞以1-4負於[[錫耶納足球俱樂部|錫耶納]]後，10戰僅1勝兼四連敗再次落入降班區域，在球季剩下7輪比賽，米赫洛域亦難逃被炒的命運，由老帥[[朱塞佩·帕帕多普洛|-{zh-hans:朱塞佩·帕帕多普洛; zh-hk:巴柏度普路;}-]]（Giuseppe Papadopulo）接手任教<small><ref>{{cite news |language=en |url=http://www.goal.com/en/news/10/italy/2009/04/14/1208405/breaking-news-bologna-sack-sinisa-mihajlovic-appoint-giuseppe-pa |title=Official: Bologna Sack Sinisa Mihajlovic, Appoint Giuseppe Papadopulo |publisher=goal.com |date=2009-04-14 |accessdate=2009-08-27}}</ref></small>。博洛尼亞在聯賽煞科日以3-1擊敗[[卡塔尼亞足球俱樂部|-{zh-hans:卡塔尼亚; zh-hk:卡坦尼亞;}-]]成功護級。

== 主場球場 ==

'''雷納多·戴亞拿球場'''（'''Stadio Renato Dall'Ara'''）是位於[[義大利|義大利]][[博洛尼亚|博洛尼亚]]的多用途[[運動場|運動場]]，主要用作[[英式足球|足球]]比賽，是博洛尼亚足球俱乐部的主場球場。戴亞拿球場建於1927年，可容38,279名觀眾，又名「''Stadio Littoriale''」。

戴亞拿球場於[[1990年世界盃足球賽|1990年世界盃]]舉行四場比賽，包括三場"D"組分賽，[[沙地阿拉伯國家足球隊|沙地阿拉伯]]0-2負於[[哥倫比亞國家足球隊|哥倫比亞]]、[[南斯拉夫國家足球隊|南斯拉夫]]1-0擊敗哥倫比亞及南斯拉夫4-1擊敗沙地阿拉伯及一場16強淘汰賽，[[英格蘭足球代表隊|英格蘭]]憑著[[戴维·普拉特|-{zh-hans:普拉特;zh-hk:柏列;}-]][[加時賽|加時]]119分鐘的入球以1-0擊敗[[比利時國家足球隊|比利時]]。

1993年[[聖馬力諾國家足球隊|聖馬力諾]]在[[聖馬力諾|國土]]以外的戴亞拿球場進行兩場[[1994年世界盃足球賽|世界盃外圍賽]]主場比賽，分別對[[英格蘭足球代表隊|英格蘭]]及[[荷蘭國家足球隊|荷蘭]]，雖然以1-7負於英格蘭，但聖馬力諾的大卫·瓜尔蒂埃里（Davide Gualtieri）僅用了8.3秒的入球是[[世界盃足球賽|世界盃]]歷來最快的入球<small><ref>{{cite web |language=en |url=http://www.thisislondon.co.uk/sport/article-4050874-details/San+Marino+hero+who+humiliated+England/article.do |title=San Marino hero who humiliated England |publisher=London Evening Standard |date=2003-03-27 |accessdate=2009-08-26 |deadurl=yes |archiveurl=https://web.archive.org/web/20090212105656/http://www.thisislondon.co.uk/sport/article-4050874-details/San+Marino+hero+who+humiliated+England/article.do |archivedate=2009-02-12 }}</ref></small>。

戴亞拿球場於1995年曾辦一場由義大利國家欖球隊對紐西蘭國家欖球隊的國際[[欖球|欖球]]賽，「全黑」以70-6大勝義大利。
