'''博多港塔'''（{{jpn|j=博多ポートタワー|hg=はかたポートタワー}}）是位於[[日本|日本]][[福岡縣|福岡縣]][[福岡市|福岡市]][[博多區|博多區]]築港本町{{link-ja|Bay side place博多碼頭|ベイサイドプレイス博多埠頭}}的塔。

== 概要 ==
博多港塔高100米，竣工於1964年，設計者是[[東京塔|東京塔]]、[[名古屋電視塔|名古屋電視塔]]、第二代[[通天閣|通天閣]]的設計者[[內藤多仲|內藤多仲]]<ref name="nishinippon-np-2014-10-17"></ref>。內藤有「塔博士」之稱，博多港塔也是他設計的「{{link-ja|高塔六兄弟|タワー六兄弟}}」之一。因博多港塔是高塔六兄弟中最後竣工的塔，因此又被稱為「高塔六兄弟中的幼子」<ref>{{cite web |url=http://www.excite.co.jp/News/lifestyle/20160820/Book_stand_019998.html |script-title=ja:東京タワーは"タワー6兄弟"の五男......なんと長男は名古屋テレビ塔 |accessdate=2017-07-17|publisher={{lang|ja|エキサイトニュース}} |language=ja}}</ref>。塔中央有電梯，瞭望台高70米，可在此一覽{{link-ja|博多灣|博多湾}}及福岡市區全景，可免費入場。博多港塔全年入場人數約24萬人<ref name="dayori">{{cite web |url=https://dayori.city.fukuoka.lg.jp/6682 |script-title=ja:博多ポートタワーが50歳を迎えます　 博多ポートタワー開設50周年 |accessdate=2017-07-17|publisher={{lang|ja|福岡市}} |language=ja}}</ref>，開放時間是10點至22點（入場截止時間是21點40分）<ref name="fukuoka-kou">{{cite web |url=http://port-of-hakata.city.fukuoka.lg.jp/healing_leisure/porttower.html |script-title=ja:癒しスポット・レジャー＞博多ポートタワー |accessdate=2017-07-17|publisher={{lang|ja|博多港}} |language=ja}}</ref>。

瞭望台的上層有東洋信號通信社運營的博多[[超短波|VHF]]海岸局「博多港廣播」（{{lang|ja|博多ポートラジオ}}）<ref name="fukuoka-kou"></ref>。該海岸局通過[[雷達|雷達]]及[[無線電|無線電]]等方法掌握進出博多灣的船隻動向，並24小時提供給水手港灣的混雜狀況、氣象等航行必要的訊息<ref name="toyoshingo">{{cite web |url=http://www.toyoshingo.co.jp/site/portal/service/shipinfo/outline/portradio.html |script-title=ja:船舶通信業務 |accessdate=2017-07-17|publisher={{lang|ja|東洋信号通信社}} |language=ja}}</ref><ref name="fukuokakopdf">{{cite web |url=http://port-of-hakata.city.fukuoka.lg.jp/download/pdf/riyokijyun.pdf |script-title=ja:博多港船舶入出港及び岸壁利用基準 |accessdate=2017-07-17|publisher={{lang|ja|福岡市}} |language=ja}}</ref>。

== 沿革 ==
博多港塔開業於1964年，最初是民營娛樂設施「博多天堂」（{{lang|ja|博多パラダイス}}）的主要部分<ref name="fukuoka-kou"></ref>。當時的博多港塔設有10分旋轉一周的旋轉瞭望餐廳<ref name="dayori"></ref>，博多天堂內也設有遊樂園、瞭望溫泉、餐廳、賓館等設施，雖興盛一時但最終因遊客減少而出售。此後博多天堂改名為博多Play Land（{{lang|ja|博多プレイランド}}）並繼續營業，但最終因遊客減少而關閉。博多天堂原址被改用於[[福岡放送|FBS福岡放送]]及福岡市立圖書館<ref name="fukuoka-kou"></ref>。

1975年，博多港塔移交給福岡市<ref name="nishinippon-np-2014-10-17"></ref>。1976年，博多港塔作為「博多港宣傳中心」（{{lang|ja|博多港PRセンター}}）重新開業，展示有關博多港的歷史、福岡市區模型、船舶模型等展品<ref name="ippan02">{{cite web |url=http://www.city.fukuoka.lg.jp/data/open/cnt/3/43920/1/hakata_port_tower_50th_kinen_jigyo.pdf |script-title=ja:博多ポートタワー開設 50 周年記念事業の実施について |accessdate=2017-07-17|publisher={{lang|ja|福岡市}} |language=ja}}</ref>。

博多港塔的外觀塗色最初是紅白兩色，在1991年改為紅色單色<ref name="dayori"></ref>。1999年，為紀念博多港開港100週年，福岡市製作了博多港塔的吉祥物「{{lang|ja|ポートくん}}」，現在仍在使用<ref>{{cite web |url=http://www.yurugp.jp/vote/detail.php?id=00003017 |script-title=ja:ポートくん |accessdate=2017-07-17|publisher={{lang|ja|ゆるキャラグランプリ}} |language=ja}}</ref>。

2005年，為修復{{link-ja|福岡縣西方海域地震|福岡県西方沖地震}}造成的損壞，博多港塔進行了改裝工程，重新粉刷了外觀，並更換了瞭望台的窗戶玻璃和望遠鏡（改為帶顯示器的數位望遠鏡並收費使用），在2006年4月1日改裝開業<ref name="ippan02"></ref>。

2007年9月30日，博多港海濱博物館（{{lang|ja|博多港ベイサイドミュージアム}}）搬遷至博多港塔一層<ref name="ippan02"></ref><ref name="baysidemuseum">{{cite web |url=http://port-of-hakata.city.fukuoka.lg.jp/healing_leisure/bm.html |script-title=ja:癒しスポット・レジャー＞博多港ベイサイドミュージアム |accessdate=2017-07-17|publisher={{lang|ja|福岡市}} |language=ja}}</ref>。
