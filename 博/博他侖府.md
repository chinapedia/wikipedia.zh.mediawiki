'''博他侖府'''（{{thai|t=จังหวัดพัทลุง|r=Changwat Phatthalung}}，{{IPA-th|t͡ɕāŋ.wàt pʰát.tʰā.lūŋ}}）是[[泰國|泰國]]南部其中的一個府。她的邻近府份（北起按顺时针序）为：[[那空是贪玛叻府|那空是贪玛叻府]]、[[宋卡府|宋卡府]]、[[沙敦府|沙敦府]]、[[董里府|董里府]]。華人稱之為'''高頭廊府'''。

== 地理环境 ==
本府位于[[马来半岛|马来半岛]]上，东滂[[宋卡湖|宋卡湖]]北部，该湖的这一部分湖水较浅，全府地形多[[山|山]]，[[洛坤山脉|洛坤山脉]]自然延伸而下贯穿本府西部，[[浦山-崖山国家公园|浦山-崖山国家公园]]（Khao Pu - Khao Ya National Park）座落在与董里府交界的地域。

== 历史渊源 ==

博他侖历史悠久，公元14世纪[[阿育跎耶王国|阿育跎耶王国]]国王[[拉玛提钵一世|拉玛提钵一世]]（King Ramathibodi I）立博他侖为十二个王城之一。公元18世纪末[[扎克里王朝|扎克里王朝]]开国国王[[拉玛一世|拉玛一世]]鉴于当时新国家[[暹罗|暹罗]]与[[帕马|帕马]]（即今日[[缅甸|缅甸]]）依然不断处于交战状态中，他将整个泰国南方置于兵部的直接统辖之下。

五世泰王[[朱拉隆功大帝|朱拉隆功大帝]]曾设立[[那空是貪瑪叻道|那空是貪瑪叻道]]（Monthon Nakhon Si Thammarat），博他侖归道都[[那空是貪瑪叻|那空是貪瑪叻]]城管辖。一直到1931年泰国「化道为府」，博他侖便自立为一府。

== 人口组成 ==
本府百分之11.1人口信奉回教。

== 行政与管理 ==
本府行政上被划分为10个县（ampoe），1个次级县（king ampoe），又进一步被划分为65个区（tambon）及626个村（muban）。
