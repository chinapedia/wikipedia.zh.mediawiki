'''庄市街道'''，是[[中国|中国]][[浙江省|浙江省]][[宁波市|宁波市]][[镇海区_(宁波市)|镇海区]]的一个[[街道_(行政区划)|街道]]，是宁波著名的侨乡。东面与[[蛟川街道|蛟川街道]]相连，南面隔甬江与[[北仑区|北仑区]][[小港街道_(宁波市)|小港街道]]和[[鄞州区|鄞州区]][[梅墟街道|梅墟街道]]相接，西面与[[江北区_(宁波市)|江北区]][[孔浦街道|孔浦街道]]、[[甬江街道|甬江街道]]、[[庄桥街道|庄桥街道]]相连，北面与[[贵驷街道|贵驷街道]]、[[骆驼街道|骆驼街道]]相接。总面积25.13平方公里，人口2.27万，[[街道办事处|街道办事处]]位于兆龙路769号。

== 名称来源 ==
'''庄市'''得名于原庄市镇所属庄市村（今庄市社区）。中华民国19年改现名，2001年改庄市街道。

== 下辖 ==
目前，庄市街道辖有庄市社区、庄一社区共2个社区。共辖有9个行政村。

== 交通 ==
[[329国道|329国道]]在西面经过庄市街道，宁镇公路经庄市街道进入宁波中心城区。

== 著名人物 ==
庄市街道是宁波著名的侨乡。庄市籍宁波帮在[[宁波商帮|宁波商帮]]中占据重要位置。有名的有[[叶澄衷|叶澄衷]]、[[宋炜臣|宋炜臣]]、[[包玉刚家族|包玉刚家族]]、[[邵逸夫家族|邵逸夫家族]]等。

== 旅游 ==
商帮文化是庄市街道的旅游特色。围绕这一特色的旅游点有[[宁波帮博物馆|宁波帮博物馆]]、[[叶氏义庄|叶氏义庄]]、[[包玉刚故居|包玉刚故居]]、[[邵逸夫故居|邵逸夫故居]]等。
