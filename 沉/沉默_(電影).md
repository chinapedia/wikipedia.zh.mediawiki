《'''沈默'''》（{{lang-en|'''''Silence'''''}}）是一部2016年的[[時代劇|歷史時代]]電影，由[[馬丁·史柯西斯|馬丁·史柯西斯]]執導，改編自日本作家[[遠藤周作|遠藤周作]]於1966年出版的[[沈默_(小說)|同名小說]]，由[[安德魯·加菲爾德|安德魯·加菲爾德]]、[[亞當·崔佛|亞當·崔佛]]、[[連恩·尼遜|連恩·尼遜]]、[[窪塚洋介|窪塚洋介]]、[[淺野忠信|淺野忠信]]、[[希朗·漢德|希朗·漢德]]主演。

《沈默》的電影前期製作階段經歷了超過二十年的耽擱和重新斟酌。在2013年1月結束了《[[華爾街狼人|華爾街狼人]]》的拍攝後，史高西斯拒絕接著進行除《沈默》以外的任何電影。在2013年4月19日，史高西斯表明他將在2014年開始在《沈默》的製作。然後{{link-en|艾爾文·溫克勒|Irwin Winkler}}被宣佈擔任監製，{{link-en|蘭德爾·艾米特|Randall Emmett}}和{{link-en|艾米特/富拉/綠洲電影公司|Emmett/Furla/Oasis Films|喬治·富拉}}將透過他們的公司{{link-en|艾米特/富拉/綠洲電影公司|Emmett/Furla/Oasis Films|艾米特/富拉電影公司}}提供資金。此後不久，便計劃在台灣拍攝本片。劇組於2015年在[[臺北市|臺北市]]的[[中影文化城|中影文化城]]、[[新北市|新北市]]（[[貢寮區|貢寮]]、[[瑞芳區|瑞芳]]、[[雙溪區|雙溪]]、[[金山區_(台灣)|金山]]、[[萬里區|萬里]]等區）以及[[花蓮縣|花蓮]]等處拍攝。

本片是導演史柯西斯充滿熱情的長久計畫，超過25年時間醞釀開發。電影由[[派拉蒙影業|派拉蒙影業]]發行，2016年11月29日在羅馬的{{link-en|宗座東方學院|Pontifical Oriental Institute}}首映、並在商業上映前先行在[[梵蒂岡|梵蒂岡]]試映<ref>{{Cite web |url = http://variety.com/2016/film/global/martin-scorsese-meets-pope-francis-talks-jesuit-history-prior-to-silence-screening-1201930025/ |title = Martin Scorsese Meets Pope Francis and Talks Jesuit History Prior to 'Silence' Screening |author = Nick Vivarelli |date = 2016-11-30 |publisher = Variety |language = en}}</ref>，商業上映日期則在2016年12月23日<ref>{{cite web|url=http://variety.com/2016/film/news/silence-martin-scorsese-1201870272/ |script-title=en:Martin Scorsese’s ‘Silence’ Lands Oscar-Season Release Date|work=[[Variety_(magazine)|Variety]]|first=Brent|last=Lang|date=September 26, 2016|accessdate=September 26, 2016}}</ref><ref>{{cite web|url=http://www.hollywoodreporter.com/news/martin-scorseses-silence-gets-december-932451|title=Martin Scorsese's 'Silence' Gets December Release Date|work=[[The_Hollywood_Reporter|The Hollywood Reporter]]|first=Rebecca|last=Ford|date=September 26, 2016|accessdate=September 26, 2016}}</ref>。該片被[[美國電影學會|美國電影學會]]選為2016十大年度電影之一<ref>{{Cite web |url = http://deadline.com/2016/12/afi-awards-film-2016-list-silence-sully-zootopia-1201867080/ |title = AFI Awards: Best Of 2016 Film List Includes 'Silence', 'Hacksaw Ridge' & More |author = Patrick Hipes |date = 2016-12-08 |publisher = Deadline Hollywood |language = en}}</ref>，以及由[[國家評論協會|國家評論協會]]選為2016年度最佳十大電影之一。<ref>{{Cite web |url = http://www.nationalboardofreview.org/award-years/2016/ |title = National Board of Review Announces 2016 Award Winners |publisher = National Board of Review |language = en}}</ref>本片還獲得提名[[第89屆奧斯卡金像獎|第89屆奧斯卡金像獎]]的[[奧斯卡最佳攝影獎|奧斯卡最佳攝影獎]]。《沈默》是史高西斯的史詩信仰宗教三部曲的第三部電影，首部是《[[基督最後的誘惑_(電影)|基督最後的誘惑]]》，其次是《[[活佛傳|活佛傳]]》。

==劇情==
[[File:El_Greco_-_The_Veil_of_St_Veronica_-_WGA10461.jpg|thumb]]所畫的這幅肖像的形式將其形象化。]] 
本片以一名年輕的葡萄牙[[耶穌會|耶穌會]]神父[[克里斯多夫·費雷拉|克里斯多夫·費雷拉]]開始，他目睹了他試圖將天主教信仰帶到日本的皈依者所受的致命酷刑。在日本當局進行酷刑的情況下，神父似乎無能為力地以任何方式幫助他的信徒。

幾年後，在[[澳門|澳門]]的[[聖保祿學院|聖保祿學院]]，一名意大利耶穌會神父[[范禮安|范禮安]]收到有關費雷拉被折磨後在日本放棄信仰（[[叛教|叛教]]）的消息。難以置信之下，費雷拉的葡萄牙學生、耶穌會神父賽巴斯提奧·羅德里奎茲和法蘭西斯柯·卡羅培出發去找他。一名逃離日本自救的酒鬼漁民，吉治郎，同意引導他們。

到了日本的友義村，神父們不安地發現當地的天主教居民被驅趕到地下活動。當一個武士尋找村民們稱為「[[目付|目付]]」的可疑基督徒時，兩個神父都感到震驚，他們把一些村民綁在海邊的木製十字架上，潮水最終淹死了他們。然後將屍體[[火化|火化]]，而神父明白這是為了防止以基督徒方式埋葬。

卡羅培動身去[[平戶市|平戶市]]，相信他們的出現可以迫使[[征夷大將軍|幕府將軍]]恐嚇村莊。羅德里奎茲去了[[五島市|五島市]]，這是最後一個費雷拉居住的地方，發現這地方被毀壞了。他最終與吉治郎重聚，吉治郎背叛了他。曾經伴隨「目付」到友義村的老武士告訴羅德里奎茲，除非他[[叛教|放棄信仰]]，否則其他被俘的基督徒將會受苦。

羅德里奎茲被帶到[[長崎市|長崎市]]，在那裡他與許多日本的皈依者被監禁起來。在法庭上，他被告知日本對[[天主教神學|天主教教義]]深惡痛絕。羅德里奎茲要求見{{link-ja|井上政重|井上政重}}，令他沮喪的是坐在他面前的老人就是負責訴訟的。羅德里奎茲回到監獄，吉治郎很快也被監禁。他向羅德里奎茲解釋，法庭官員威脅他，背叛羅德里奎茲。吉治郎接著說他是一名基督徒，並要求透過懺悔來寬恕他的背叛，羅德里奎茲不情願地答應他。他後來踩踏一個像徵著拋棄信仰行為的「[[踏繪|踏繪]]」（一個粗糙的[[十架苦像|十架苦像]]）後被釋放出來。後來，羅德里奎茲在被警衛看守的情況下被帶到海岸線等待某人。在遠處，他目睹消瘦的卡羅培和另外三名囚犯靠近在海岸線上，分別有警衛看守。仍然在遠處，另外三名囚犯被帶到一艘小船的海上，並逐一把他們從船上掉入水中淹死，同時勸誘卡羅培放棄信仰。羅德里奎茲在陸上被守衛克制，因為他看卡羅培拒絕放棄信仰。然後，他看到絕望的卡羅培在試圖搶救最後一名被淹死的囚犯時在其他三名囚犯的旁邊溺斃了。

過了一段時間，羅德里奎茲最終被帶去會見年老的費雷拉。費雷拉說，他在遭受折磨時叛教，並且說，在國內15年和教堂1年後，他相信天主教在日本是無用的。羅德里奎茲與他斷絕關係，而費雷拉態度強硬。當晚，羅德里奎茲在監獄裡聽到五名基督徒遭受酷刑。費雷拉告訴他，他們已經開始放棄信仰。當羅德里奎茲看著一個「踏繪」時，他聽到基督的聲音允許他踩踏它，而他就踩了。

幾年後，費雷拉去世後，吉治郎要求羅德里奎茲再次赦免他，但羅德里奎茲拒絕，說他不再是一名神父。吉治郎後來被人用袋子抓住。他被帶走，再也沒有聽見他了。多年後，羅德里奎茲去世了。他被放在一個大圓形的木製棺材裡，他的屍體被火化。他手裡拿的是剛來日本的時候給他的那個小小的粗糙十字架。

==製作==
===發展===
史高西斯在2016年12月接受《''America Magazine''》的採訪時表示，他第一次讀到遠藤周作的小說是在1989年，而當時他應[[黑澤明|黑澤明]]的邀請去日本扮演黑澤明的電影《[[夢_(1990年電影)|夢]]》中的角色之一。史高西斯在採訪中描述了當時他正在黑澤明的電影中飾演[[文森·梵谷|文森·梵谷]]時，他在日本的機場和酒店之間旅行的期間，他是如何讀到這本小說。

[[File:Martin_Scorsese_by_David_Shankbone.jpg|thumb]]的電影《夢》中飾演正在繪畫《[[麥田群鴉|麥田群鴉]]》的[[文森·梵谷|文森·梵谷]]時，第一次讀到用來製作本片的遠藤周作小說。]]
本片被認為是史高西斯的一個「受难項目」，從1990年起開發，是史高西斯最有爭議，也有強烈宗教主題的電影《[[基督的最後誘惑_(電影)|基督的最後誘惑]]》上映的兩年後。史高西斯在1989年第一次閱讀這本小說，之後很快獲得了電影版權。<ref name="NoFilmSchool.com">{{Cite web |url = http://nofilmschool.com/2016/12/martin-scorsese-film-silence |title = Watch: Here's the Film Martin Scorsese Has Been Trying to Make for 27 Years |author = Liz Nord |date = 2016-12-09 |publisher = NoFilmSchool.com |language = en}}</ref>

2009年，製作開始，史高西斯和一名製作人員去了日本的[[長崎縣|長崎縣]]，參觀了遠藤周作小說場景的原址。另外的外景則在加拿大尋找。然而，《沉默》在不久後進入了{{link-en|開發地獄|Development hell}}的狀態，而史高西斯決定忙於《[[不赦島|不赦島]]》和《[[雨果的巴黎奇幻歷險|雨果的巴黎奇幻歷險]]》。在2011年12月，史高西斯表示，《沉默》將是他的下一部電影。到了3月，雖然史高西斯原本把它暫時擱置留待以後處理，並因此退出，但是史高西斯簽約《[[華爾街狼人|華爾街狼人]]》，並選擇放在《沉默》前執導。然而，當時史高西斯的宣傳人員表示《沉默》列於首位。5月，維托里奧·切奇-戈里加入本片作為監製<ref name="ref939676644">{{cite web|url=http://blogs.indiewire.com/theplaylist/martin-scorseses-long-developing-silence-finds-backer-in-cecchi-gori-pictures-to-be-produced-first-on-companys-slate-20120501|title=Martin Scorsese's Long Developing 'Silence' Finds Backer In Cecchi Gori Pictures, To Be Produced First On Company's Slate | The Playlist|publisher=blogs.indiewire.com|accessdate=2014-03-02}}</ref>。切奇-戈里參與了《沉默》的前期製作，但多年來不相關的法律糾紛已經中斷了與本片的合作。<ref>{{Cite web |url = http://www.indiewire.com/2012/05/martin-scorseses-long-developing-silence-finds-backer-in-cecchi-gori-pictures-to-be-produced-first-on-companys-slate-110879/ |title = Martin Scorsese's Long Developing 'Silence' Finds Backer In Cecchi Gori Pictures, To Be Produced First On Company's Slate |author = Charlie Schmidlin |date = 2012-05-01 |publisher = IndieWire |language = en}}</ref>

在2013年1月結束了《華爾街狼人》的拍攝後，史高西斯拒絕接著進行除《沉默》以外的任何電影。<ref name="NoFilmSchool.com" />2013年4月19日，史柯西斯表示他將在2014年開始製作《沉默》，這經過了23年的等待。{{link-en|艾爾文·溫克勒|Irwin Winkler}}在同一天宣佈擔任監製，還有{{link-en|蘭德爾·艾米特|Randall Emmett}}和喬治·富拉，並透過他們的公司{{link-en|艾米特/富拉/綠洲電影公司|Emmett/Furla/Oasis Films|艾米特/富拉電影公司}}提供資金給本片製作<ref name="ref-930122061">{{cite web|url=http://www.deadline.com/2013/04/martin-scorsese-to-make-noise-on-silence-at-cannes-emmettfurla-films-funding-the-film/|title=Martin Scorsese To Make Noise On &#039;Silence&#039; At Cannes; Emmett/Furla Funding The Film - Deadline.com|publisher=deadline.com|accessdate=2014-03-02}}</ref>。電影將於[[臺灣|臺灣]]拍攝<ref name="ref-165721091">{{cite web|url=http://m.collider.com/collider/#!/entry/martin-scorseses-silence-gets-financing-poised-to-start-filming-july,5171a8db4ca7a1dba6cca431|title=Onswipe|publisher=m.collider.com|accessdate=2014-03-02|deadurl=yes|archiveurl=https://web.archive.org/web/20140810192811/http://m.collider.com/collider/#!/entry/martin-scorseses-silence-gets-financing-poised-to-start-filming-july,5171a8db4ca7a1dba6cca431|archivedate=2014-08-10}}</ref>。

=== 法律索賠 ===
2012年8月，切奇-戈里電影公司控告史柯西斯違背合約內容。根據公司所說，史高西斯於1990年簽署了一份書面協議來執導《沉默》，聲稱史柯西斯應該於1997年的《[[達賴的一生|達賴的一生]]》後就該完成本片劇本，而切奇-戈里電影公司顯然已經為此投資了超過75萬美元。<ref name="Scorsese Settles Suit">{{Cite web |url = http://deadline.com/2014/01/martin-scorsese-silence-lawsuit-cecchi-gori-pictues-674768/ |title = Martin Scorsese Settles 'Silence' Suit |author = Dominic Patten |date = 2014-01-31 |publisher = Deadline.com |language = en}}</ref>然而，史柯西斯首先選擇拍了《[[午夜速遞|午夜速遞]]》、《[[紐約風雲|紐約風雲]]》和《[[娛樂大亨|娛樂大亨]]》。

2004年，據稱史高西斯為了執導《[[無間道風雲|無間道風雲]]》和《[[不赦島|不赦島]]》而簽署協議推遲本片。2011年，史高西斯表面上同意多一個交易，推遲《沉默》來執導《[[雨果的巴黎奇幻歷險|雨果的巴黎奇幻歷險]]》。在此過程中同意支付「延時費」，切奇-戈里聲稱他並無支付其費用<ref name="ref-427663518">{{cite web|url=http://www.hollywoodreporter.com/thr-esq/martin-scorsese-movies-lawsuit-cecchi-gori-364934|title=Martin Scorsese Sued by Producer Cecchi Gori Over Alleged Deal to Direct &#039;Silence&#039;|publisher=hollywoodreporter.com|accessdate=2014-03-02}}</ref>。但史柯西斯譴責這是「沒價值的訴訟」，並形容這只是一個「媒體噱頭」<ref name="ref865375330">{{cite web|url=http://insidemovies.ew.com/2012/08/24/martin-scorsese-lawsuit/|title=Martin Scorsese responds to lawsuit | Inside Movies | EW.com|publisher=insidemovies.ew.com|accessdate=2014-03-02}}</ref>。這起訴訟於2014年1月17日解決了，而解決的條款則被保密。<ref name="Scorsese Settles Suit" />

=== 編劇 ===
本片是第二次改編遠藤周作的小說，前一次改編是[[篠田正浩|篠田正浩]]執導的[[沉默_(1971年電影)|1971年同名電影]]。史高西斯在1991年與共同編劇和長期合作者{{link-en|傑伊·庫克斯|Jay Cocks}}一起寫了最初的劇本。然而，他們對劇本不滿意，並用了額外15年進行重寫。<ref name="NoFilmSchool.com" />後來，翻譯了遠藤周作的八部小說的官方譯者{{link-en|Van C. Gessel|Van C. Gessel}}，在本片中擔任顧問。最後拍攝出來的劇本在評論界中被評價為對遠藤所寫小說的準確描述。宗教歷史學家{{link-en|哈魯高·娜瓦塔·沃德|Haruko Nawata Ward}}指出，在本片結尾的已故神父手中所拿著的小十字架是史高西斯做出的一個[[作者論|表現自我]]決定，不包括在遠藤的原著中。

=== 選角 ===

===拍攝===
[[File:Thelma_Schoonmaker_KVIFF.jpg|thumb]]
2012年1月，史柯西斯討論了轉為[[3D電影|3D電影]]的可能性<ref name="ref-611762994">{{cite web|url=http://thefilmstage.com/news/martin-scorseses-3d-plans-for-silence-and-sinatra-still-alive/|title=Martin Scorsese’s 3D Plans for ‘Silence’ and ‘Sinatra’ Still Alive|publisher=thefilmstage.com|accessdate=2014-03-02}}</ref>。到2014年2月，史高西斯已經開始在臺灣尋找在夏天拍攝的外景，而最終推遲到2015年初。主要拍攝於2015年1月30日至5月15日在臺灣進行。<ref>{{Cite web |url = http://variety.com/2015/film/news/martin-scorsese-locks-funding-for-silence-1201412246/ |title = Martin Scorsese Locks Funding for 'Silence' |author = Dave McNary |date = 2015-01-22 |publisher = Variety |language = en}}</ref><ref>{{Cite web |url = https://www.screendaily.com/screenasia/scorseses-silence-to-wrap-in-taiwan/5087351.article |title = Martin Scorsese's 'Silence' to wrap in Taiwan |author = Liz Shackleton |date = 2015-05-06 |publisher = Screen Daily |language = en}}</ref>

2015年1月29日，劇組在臺灣[[臺北市|臺北市]][[士林區|士林區]][[中影文化城|中影文化城]]進行搭建拍攝工作所需的古城建築時，發生屋頂與[[鷹架|鷹架]]倒塌事件，造成三名承包商員工一死兩傷<ref>{{cite web|url=http://www.ettoday.net/news/20150129/460330.htm |title=中影文化城舊古城突倒塌　2輕重傷1命危 |publisher =[[ETtoday_東森新聞雲|ETtoday 東森新聞雲]] |date=2015年1月29日}}</ref>；[[臺北市建築管理工程處|臺北市建築管理工程處]]將臨時片場視為新[[違建|違建]]處理，隨後勒令停工。2015年3月，拍攝團隊於[[花蓮縣|花蓮縣]][[壽豐鄉|壽豐鄉]]知名的海岸私人景點「牛山呼庭」拍攝，並有消息指出李奧納多·狄卡皮歐已在花蓮拍攝；此傳聞被工作人員否認。
