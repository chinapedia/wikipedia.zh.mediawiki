'''丁朝'''（{{vie|hn=家丁|qn=Nhà Đinh}}）是[[越南歷史|越南歷史]]的朝代之一，其立國期間為968年─980年。其建立者[[丁部領|丁部領]]先後擊敗了割據越南北部的[[十二使君|十二使君]]勢力，於968年統一全國並稱帝，建國號'''大瞿越'''。越南於10世紀以前，長期受北方鄰國[[中國|中國]]統治。丁朝開國之後訂立制度，逐漸具有國家規模。外交上，丁朝向中國的[[宋朝|宋朝]]通好，遣使朝貢，並首次被中國承認為[[藩屬|藩屬]]而不是領土。<ref name="khamdinh6">[http://lib.nomfoundation.org/collection/1/volume/253/page/6 《欽定越史通鑑綱目·卷之一·丁紀》，第六面]</ref>

丁朝歷丁先皇（丁部領）、[[丁廢帝|丁廢帝]]（丁璿）兩位君主。979年丁部領政變遇害後，繼任的皇帝丁璿年幼，由十道將軍[[黎桓|黎桓]]獨掌朝政。[[宋太宗|宋太宗]]得知後，欲趁亂南下攻滅交趾。此事傳至交趾，黎桓被軍隊推戴為帝，丁朝遂被[[前黎朝|前黎朝]]取代。此後，黎桓率軍在[[白藤江之戰_(981年)|白藤江之戰]]中擊退了宋朝的進犯。

後世歷史學者，視丁朝為越南建國的創始階段，史家[[黎嵩|黎嵩]]稱「我越正統之君，實自此始」<ref name="越鑑通考總論87">《大越史記全書》卷首（黎嵩《越鑑通考總論》），東京大學東洋文化研究所，87頁。</ref>，可見它在歷史發展裡的重要地位。

== 历史 ==
=== 統一與開國 ===
公元10世紀初，越南人開始擺脫[[越南第三次北屬時期|中國統治]]，並於939年建立政權，是為[[吳朝|吳朝]]。但吳朝政局處於不穩狀態，出現[[十二使君|十二使君]]割據的分裂局面。<ref>陳仲金《越南史略》第三卷第一章，戴可來譯，北京商務印書館，57─59頁。</ref>

丁朝開國君主[[丁部領|丁部領]]，[[華閭|華閭洞]]（在今[[寧平省|寧平省]]）人，父親[[丁公著|丁公著]]，曾任[[驩州|驩州]][[刺史|刺史]]。丁部領自年少時起，已獲得當地民眾敬服，立為君長。叔父[[丁預|丁預]]不服，丁部領將之擊敗，樹立聲威，於是「人人畏服」。其後，丁部領與兒子[[丁璉|丁璉]]到[[布海口|布海口]]，投靠當地割據者[[陳覽|陳覽]]（十二使君之一），獲陳覽賞識，取得兵權，勢力更盛。對於各地的割據者，丁部領大張撻伐，「凡征戰所過，易如破竹」，有「萬勝王」的稱號。<ref>吳士連《大越史記王書·本紀全書·丁紀·先皇帝》，東京大學東洋文化研究所，179頁；《欽定越史通鑑綱目》正編卷一，丁先皇帝元年條。</ref>

968年，丁部領掃平群雄，統一全國後，便即皇帝位，尊稱為「大勝明皇帝」，史稱「丁先皇」，改國號「[[大瞿越|大瞿越]]」，以華閭洞為國都。丁部領開國後，著手進行了一系列措施，包括在國都築城鑿池、起宮殿、制定朝儀、置百官、採用年號、定文武僧道階品、行十道軍制等等。據[[越南陳朝|越南陳朝]]的史官[[黎文休|黎文休]]指出，這些措施讓越南從此「制度略備」。<ref>吳士連《大越史記王書·本紀全書·丁紀·先皇帝》，東京大學東洋文化研究所，180─181頁。</ref>

=== 國內紛爭 ===
丁朝曾發生內部亂事，對其王室構成巨大打擊。

979年（丁朝太平十年），春，丁部領的長子[[丁璉|南越王丁璉]]（[[越南漢喃銘文|銘文]]資料作「丁匡璉」），殺皇太子[[丁項郎|丁項郎]]（丁先皇少子，銘文資料作「頂帑僧帑」）。事緣丁先皇雖有長子丁璉曾與他一同創業，但到後來生丁項郎，先皇特別鍾愛，便立為太子。然而卻引起丁璉的不滿，丁璉以丁項郎「卻行惡心，違背若愛寬容」為由，先行下手，暗中派人將之殺害。<ref>吳士連《大越史記王書·本紀全書·丁紀·先皇帝》，東京大學東洋文化研究所，182頁；《佛頂尊勝加句靈驗陀羅尼》（漢喃研究所編號VB2）（979年刻），收錄於潘文閣、蘇爾夢主編《越南漢喃銘文匯編·第一集北屬時期至李朝》，遠東學院、漢喃研究院，63頁。</ref>

此後，丁先皇與丁璉一同被祗候內人[[杜釋|杜釋]]所殺。杜釋因曾夢見流星墜入口中，認為這是吉兆，便心生篡弒之心。979年農曆十月，丁先皇在夜宴後醉臥宮中，杜釋入內殺之，並殺害丁璉。其後，大臣[[阮匐|定國公阮匐]]處斬杜釋，又與十道將軍[[黎桓|黎桓]]共立先皇次子[[丁璿|衞王丁璿]]為帝，是為廢帝。<ref>吳士連《大越史記王書·本紀全書·丁紀·先皇帝》，東京大學東洋文化研究所，183頁。</ref>

丁廢帝繼位時，年方十歲，十道將軍黎桓攝政，自稱「副王」。大臣阮匐、[[丁佃|丁佃]]、[[范盍|范盍]]等認為黎桓將對皇帝不利，便在各地相率起兵，分成水陸兩道，進攻國都[[華閭|華閭]]。丁朝太后[[楊雲娥|楊氏]]獲悉後，即與黎桓共謀平亂，黎桓立刻出擊，攻破丁佃、阮匐的水陸軍隊，殺害二人，繼而擊敗范盍。<ref>吳士連《大越史記王書·本紀全書·丁紀·廢帝》，東京大學東洋文化研究所，184頁。</ref>於是，朝中再無制衡黎桓的力量，大權便落入黎桓手中。

=== 黎桓代丁 ===
980年（丁朝太平十一年、前黎朝黎桓天福元年），中國[[宋朝|宋朝]]得知丁朝內亂，[[宋太宗|宋太宗]]認為可趁此機會兼併越南，便決定南侵。宋廷派出軍隊南下後，楊太后派黎桓自選勇士抗敵。黎桓手下[[范巨倆|范巨倆]]認為「今主上（丁廢帝丁璿）幼弱」「不如先冊十道（十道將軍黎桓）為天子，然後出師可也」，於是黎桓便被軍士擁立為帝，廢丁璿，仍封為衞王。<ref>吳士連《大越史記王書·本紀全書·丁紀·廢帝》，東京大學東洋文化研究所，185頁。</ref>丁朝遂告滅亡，[[前黎朝|前黎朝]]建立。981年（前黎朝黎桓天福二年），黎桓擊敗來襲的宋軍<ref>吳士連《大越史記王書·本紀全書·黎紀·大行皇帝》，東京大學東洋文化研究所，188頁。</ref> ，穩固了前黎朝的基業。

== 政治 ==
=== 刑律 ===
[[丁部領|丁部領]]開國後不久，便採用嚴刑，欲「威制天下」。乃於朝廷上設置大鼎（即油鍋），以及飼養猛虎，在國內下令若有違法，即「受烹嚙之罪」。於是人皆畏服，無有犯者。<ref name="大越史記全書180">吳士連《大越史記王書·本紀全書·丁紀·先皇帝》，東京大學東洋文化研究所，180頁。</ref>[[越南|越南]]史家[[陳仲金|陳仲金]]認為丁朝採行嚴刑，是由於當時國內「還有許多人過慣了亂時生活，不遵守法令」，而這些刑律「如此威嚴，未免太過份，但賴有此刑律，國內之民才漸漸獲得安寧」。<ref name="越南史略60"/>

=== 后妃 ===
970年（丁朝太平元年），設皇后五名，各有名號，分別為「丹嘉」、「貞明」、「矯國」、「瞿國」、「歌翁」。對此，[[越南陳朝|越南陳朝]]時期的史官[[黎文休|黎文休]]批評「自古（皇后）祗立一人，以主內治而已，未聞有五其名者。[[丁部領|先皇]]無稽古學，而當時群臣，又無匡正之者，致使溺私並立五-{后}-」。<ref name="大越史記全書180"/>

=== 職官 ===
{{See also|丁朝官制}}
971年（丁朝太平二年），制定文武官職階品，又委員一批重要官員，如[[阮匐|阮匐]]任「定國公」、劉基任「都護府士師」、[[黎桓|黎桓]]任「十道將軍」。後來於975年（丁朝太平六年），定文武官員的冠服。除了文武職官外，丁朝又對[[佛教|佛]]、[[道教|道]]等宗教人士授給位號，以示尊崇，如[[僧統|僧統]][[吳真流|吳真流]]賜號「匡越大師」、[[張麻尼|張麻尼]]為「僧錄」、[[道士|道士]][[鄧玄光|鄧玄光]]授「崇真威儀」。<ref name="大越史記全書181"/>

== 宗教 ==
丁朝統治者崇信[[佛教|佛教]]。自[[第三次北屬|唐朝統治時期]]興起的[[無言通禪派|無言通禪派]]（又名「禪宗後派」），獲得丁氏朝廷認同，冊封其代表人物吳真流為「匡越大師」<ref>杜繼文《佛教史》，江蘇人民出版社，301頁。</ref>（[[丁朝#職官|詳情見上]]）。此外，丁朝有[[佛頂尊勝陀羅尼|佛頂尊勝陀羅尼]]信仰的流傳，從現存的丁朝石幢[[越南漢喃銘文|銘文]]記載，丁匡璉（即[[丁璉|丁璉]]）殺其弟頂帑僧帑（即[[丁項郎|丁項郎]]）後，為解脫亡弟之靈，便造一百座寶幢，上刻《[[佛頂尊勝陀羅尼|佛頂尊勝陀羅尼]]》經文，並藉此祈求丁先皇政權能夠「永霸天南，恒安寶位」。後世學者指出，公元8、9世紀的[[中國|中國]]及[[朝鮮半島|朝鮮半島]]都有類似的佛頂尊勝陀羅尼石幢，丁朝的石幢銘文可證[[唐朝|唐朝]]的佛頂尊勝陀羅尼信仰不僅東傳朝鮮、[[日本|日本]]，且也南傳越南。<ref>《佛頂尊勝加句靈驗陀羅尼》五篇（10世紀刻），收錄於潘文閣、蘇爾夢主編《越南漢喃銘文匯編·第一集北屬時期至李朝》，遠東學院、漢喃研究院，57─70頁。</ref>[[道教|道教]]亦與佛教一樣，受到朝廷所重視及授任委任職官（[[丁朝#職官|詳情見上]]）。

== 軍事 ==
973年（丁朝太平四年）農曆二月，丁朝定「十道軍」制，以十人為一伍，十伍為一卒，十卒為一旅，十旅為一軍，十軍為一道。全國共有十道軍。軍人都須佩戴四方平頂帽，該帽用皮製成，平頂四邊夾縫，上狹下寬。<ref name="大越史記全書181"/>

按照十道軍制的兵員數目計算，丁朝擁有士兵達一百萬人。對此，越南史家[[陳仲金|陳仲金]]提出質疑：「試想當時我國（指越南）地窄、人稀，從哪裡能招募一百萬軍隊，又從哪裡徵集糧食來養活如此眾多之人呢？或許[[丁部領|先皇]]擁有十萬軍隊，已經算是多的了」。<ref>陳仲金《越南史略》第三卷第二章，戴可來譯，北京商務印書館，61頁。</ref>

== 貨幣與經濟 ==
在丁朝時期，鑄有「大平興寶」（錢幣上的「大」即「太」），它的形制，可以分成兩大類：

一種是「大平興寶」光背錢。錢面上的「大平興寶」四字對讀，書法介乎[[楷書|楷]][[隸書|隸]]之間，錢徑約為23.5[[毫米|毫米]]，厚約1毫米，重約2.2克。這類光背錢每一個都不盡相同，有大字大樣錢、小字大樣錢，有的字跡精細，有的粗壯。

另一種是「大平興寶」背「丁」字錢，此類佔大多數。「丁」字用以記當時君主姓氏，而它的位置則每銖都不同，有「背穿上丁」、「背穿下丁」、「穿背反丁」等等。

「大平興寶」製作粗糙，但風格古樸，與中國[[五代|五代]]時期錢幣有許多相似之處，這說明越南文化深受中國的影響。然而它亦有獨創之處，那就是錢文上的「大平興寶」四字，有別於10世紀以前中國常見的「通寶」、「元寶」和「重寶」，它帶有祝福國家國運興盛、萬世太平的含意。

「大平興寶」的鑄造時期，大約在970年至977年（丁朝太平元年至八年）之間。此後，丁朝內鬥不斷，社會動蕩，鑄錢便逐漸減少，甚或最終停鑄。此外，雖然丁朝始行鑄造「大平興寶」錢，但它的流通範圍，僅局限於現時越南北部地區，即當時丁朝所轄領土，而且中國[[唐朝|唐]]、[[北宋|宋]]所鑄錢幣，仍為丁朝時期的主要流通貨幣，「大平興寶」僅作補充之用。<ref>《越南歷史貨幣》，中國金融出版社，16至17頁；以及114頁。</ref>

== 對外關係 ==
'''丁朝'''時期，與中國的[[宋朝|宋朝]]有著密切關係。972年（丁朝太平三年），丁先皇遣長子[[丁璉|南越王丁璉]]入宋訪問，以示通好。<ref name="大越史記全書181">吳士連《大越史記王書·本紀全書·丁紀·先皇帝》，東京大學東洋文化研究所，181頁。</ref>關於此舉，據越南史家[[陳仲金|陳仲金]]的分析，認為是由於宋朝派兵攻取[[南漢|南漢]]（在今中國[[華南|華南]]地區，亦即越南丁朝的北鄰），丁先皇害怕宋兵來攻，所以先遣使到宋通好，並且為以後的越南歷任政府開創了「恪守向中國朝貢之禮」的先例。<ref name="越南史略60">陳仲金《越南史略》第三卷第二章，戴可來譯，北京商務印書館，60頁。</ref>其後，丁朝與宋朝的使節來往相當頻繁。973年（丁朝太平四年），宋朝遣使冊封丁先皇為「交趾郡王」，封丁璉為「檢校太師靜海軍節度使安南都護」。 975年（丁朝太平六年），秋，宋朝遣使，加授丁璉為「開府儀同三司檢校太師交趾郡王」，在對宋的事務上從此「以[[丁璉|璉]]為主」。<ref name="大越史記全書181"/>976年（丁朝太平七年），丁朝遣使入宋答謝。977年（丁朝太平八年），丁朝遣使入宋，賀[[宋太宗|宋太宗]]即皇帝位。<ref name="大越史記全書182">吳士連《大越史記王書·本紀全書·丁紀·先皇帝》，東京大學東洋文化研究所，182頁。</ref>

丁朝的南鄰[[占城|占城國]]，於979年（丁朝太平十年），由反叛丁朝的[[駙馬|駙馬]][[吳日慶|吳日慶]]所協助下，出兵伐越。吳日慶是[[吳朝|吳朝]]王族，亦是[[十二使君|十二使君]]之一，割據一方，後來被丁部領擊敗。丁部領為免吳日慶生怨，便將公主嫁給他。然而吳日慶一直伺機報復，到丁先皇死時，便引占城兵攻越。占軍出動水軍進攻，但在途中遇上暴風，軍船覆沒，吳日慶與大批軍士溺死，戰事就此結束。<ref>吳士連《大越史記王書·本紀全書·丁紀·廢帝》，東京大學東洋文化研究所，184─185頁。</ref>

此外，丁朝與海外各國存在著商貿關係。據《[[大越史記全書|大越史記全書]]》所載，在976年（丁朝太平七年）的春季，「海外諸國商舶來獻方物」<ref name="大越史記全書182"/>，這反映了當時的外貿情形。

== 丁朝的歷史意義 ==
越南[[後黎朝|後黎朝]]時期史家[[黎嵩|黎嵩]]，高度肯定丁朝在越南脫離[[越南第三次北屬時期#越人自主及建國|中國統治]]及立國歷程中的重要地位：「[[丁部領|丁先皇]]因[[吳朝|吳國]]之喪亂，平[[十二使君|十二使君]]，天與人歸，輿圖混一。……我越正統之君，實自此始。」不過，黎嵩指出丁朝未能為後世從國家制度、社會文化等方面，樹立完善的楷模，「宗廟未立，學校未建，設鼎養虎，以為刑獄之具。遺犀納象，徒為貢獻之勞。禮樂刑政之不修，重門擊柝之不謹」，到丁朝末年時，「權臣攝政，國內離心，丁氏訖籙，是保邦制治之無策也」。<ref name="越鑑通考總論87"/>

中國學者[[郭振鐸|郭振鐸]]、[[張笑梅|張笑梅]]認為，丁朝除了在越南立國歷程上具有創造性，而且還令越南產生三種變化：一是越南經過唐末以來豪強家族的經營，[[封建社會|封建]]大莊園遍布各地，封建土地所有制到丁朝時確立下來。二是丁朝結束十二使君分治的局面，令越南首次出現政治集中及國家統一。第三是丁朝與中國[[宋朝|宋朝]]建立[[朝貢制度|宗藩關係]]，在國際關係上取得了中國的承認及支持。<ref>郭振鐸、張笑梅《越南通史》第一編第一章，中國人民大學出版社，14頁。</ref>
