'''丁項郎'''（{{vie|v='''Đinh Hạng Lang'''|hn=丁項郎}}，{{bd||？|979年||catIdx=D}}），[[越南漢喃銘文|銘文]]資料又作「'''頂帑僧帑'''」，[[越南|越南]][[丁朝|丁朝]]皇帝[[丁部領|丁部領]]的幼子，獲丁部領鍾愛，被立為[[太子|太子]]，但惹起其長兄[[丁璉|丁璉]]不滿。979年，丁項郎為丁璉所殺。

== 生平 ==
{{see also|丁部領|丁璉}}
[[File:VuaDinhTienHoang.jpg|thumb]]像]]
[[File:Dinhlien.jpg|thumb]]像]]
'''丁項郎'''的父親[[丁部領|丁部領]]，為[[越南|越南]][[丁朝|丁朝]]的開創者。<ref>吳士連等《大越史記王書·本紀全書·丁紀·先皇帝》，東京大學東洋文化研究所，179頁。</ref>丁部領長子、丁項郎長兄[[丁璉|丁璉]]（又作丁匡璉）在丁朝創業當中勞苦而功高，獲封為「南越王」，並曾代表丁朝出使[[中國|中國]][[宋朝|宋朝]]，丁部領本來就想傳之以位。<ref>吳士連等《大越史記王書·本紀全書·丁紀·先皇帝》，東京大學東洋文化研究所，182頁；[http://lib.nomfoundation.org/collection/1/volume/253/page/7 Hội Bảo tồn Di sản chữ Nôm─潘清簡等《欽定越史通鑑綱目》正編卷之一，太平十年春「南越王璉殺太子項郎」條]，第七至第八葉。</ref>然而，丁部領後來改變初衷，立丁項郎為太子<ref group="注">丁項郎被立為太子的時間，據《[[越史略|越史略]]》記載是在「太平三年」（972年），另據《大越史記全書》記載是在「太平九年」（978年）。</ref><ref>《越史略》卷上，收錄於《四庫全書·史部》（第466册），上海古籍出版社，573頁；吳士連等《大越史記王書·本紀全書·丁紀·先皇帝》，東京大學東洋文化研究所，182頁。</ref>。丁璉對此極為不滿，遂先下手，派人殺害項郎。事後，丁璉辯稱丁項郎有「不為忠孝，伏事上父（指丁部領）及長兄，卻行惡心，違背若愛寬容」的罪狀，為了「要成家國，永霸門風」，才決意行凶。<ref>吳士連等《大越史記王書·本紀全書·丁紀·先皇帝》，東京大學東洋文化研究所，182頁；《佛頂尊勝加句靈驗陀羅尼》（漢喃研究所編號VB2）（979年刻），收錄於潘文閣、蘇爾夢主編《越南漢喃銘文匯編·第一集北屬時期至李朝》，遠東學院、漢喃研究院，63頁。</ref>

丁項郎遇害後，丁璉下令建造一百座寶幢，上面刻有《[[佛頂尊勝陀羅尼經|佛頂尊勝陀羅尼經]]》，以求解脫丁項郎之靈。這些寶幢仍有一些保留至今，其[[越南漢喃銘文|銘文]]中稱呼丁項郎為「'''頂帑僧帑'''」，並提及他被殺之事。<ref>《佛頂尊勝加句靈驗陀羅尼》五篇（10世紀刻），收錄於潘文閣、蘇爾夢主編《越南漢喃銘文匯編·第一集北屬時期至李朝》，遠東學院、漢喃研究院，57─70頁。</ref>

== 家族人物 ==
*父：[[丁部領|丁部領]]，即丁先皇。
*長兄：[[丁璉|丁璉]]，獲丁部領封為南越王。
*次兄：[[丁璿|丁璿]]，獲丁部領封為衛王，即來的丁廢帝。
※以上各項，散見於吳士連等《大越史記王書·本紀全書·丁紀·先皇帝》，東京大學東洋文化研究所。

== 後世評論 ==
丁項郎以幼子身份，而被其父皇[[丁部領|丁部領]]立為太子，導致長兄[[丁璉|丁璉]]行凶，此事受到[[越南|越南]]後世史家猛烈抨擊。[[後黎朝|後黎朝]]史官[[吳士連|吳士連]]指責丁部領漠視封建制度當中，皇位傳承都是「繼世以嫡，萬世常經，拂是經者，未有不致亂也。間因世亂，立太子則先有功，其或嫡長過惡廢之，然後立次」，而「[[丁璉|南越王璉]]長且有功，未聞過矣。[[丁部領|先皇]]乃愛少子（指丁項郎），而忘其嫡，謂足以伸鍾愛之情，不知適以害之也」，同是又批評丁璉「忍心至殺其弟，天倫滅矣」。<ref>吳士連等《大越史記王書·本紀全書·丁紀·先皇帝》，東京大學東洋文化研究所，182─183頁。</ref>

近現代史家[[陳重金|陳重金]]（即陳仲金）認為，此事件「造成了家庭內部的紛亂」。<ref>陳重金（即陳仲金）《越南通史》（即《越南史略》），北京商務印書館，61頁。</ref>
