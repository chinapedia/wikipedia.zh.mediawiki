== 医学效应 ==
使用胃镜可以直接观察丁基東莨菪鹼的效应。在胃的[[蠕动|蠕动]]过分增多导致不适的情况下口服丁基東莨菪鹼可以在一两分钟内达到胃的蠕动完全停止的效果。

丁基東莨菪鹼是亲水[[季铵阳离子|季铵阳离子]]，因此很难被消化道吸收（吸收率小于10%）。虽然如此通过一系列[[安慰剂|安慰剂]]双盲试验证明它口服和肛服的有效性，尤其对[[肠易激综合征|肠易激综合征]]的痉挛有效。

对于对肠胃系统的医学作用身体的吸收不重要，因为在这些场合下消化道的局部作用是最重要的。因此丁基東莨菪鹼尤其适合于治疗比较轻的痉挛。

由于丁基東莨菪鹼的[[生物利用度|生物利用度]]很小因此在肠胃腔之外它口服几乎没有作用。但是通过[[靜脈注射|靜脈注射]]它对肾和脾绞痛的作用有非常详细的纪录。除静脉注射外丁基東莨菪鹼也可以[[皮下注射|皮下注射]]或者[[肌肉注射|肌肉注射]]。

有文献说丁基東莨菪鹼也可以减轻经痛。

== 用法 ==
* 口服，每天三次，每次10毫克，以減輕[[腸易激綜合徵|腸易激綜合徵]]的症狀。
* 平滑肌鬆弛劑，主治胃腸潰瘍或胃腸炎引起的胃腸痙攣（止痛用），有藥丸或是點滴用注射液……等形態。（後者大多用在醫院的急診室，簡稱「補液」。）

== 副作用 ==
所有治療胃腸痙攣的药物都有导致皮肤红肿、[[腹胀|腹胀]]、[[便秘|便秘]]、视觉受影响、排[[汗|汗]]减少、[[心动过速|心动过速]]、[[人眼调节|人眼调节]]受影响和口腔干燥等副作用。

假如病人已经有过高眼压（[[青光眼|青光眼]]）、泌尿故障、便秘或者心跳不稳等现象的话不宜使用丁基東莨菪鹼。

与[[东莨菪碱|东莨菪碱]]相比丁基東莨菪鹼无法越过[[腦血管障壁|腦血管障壁]]因此不会在中枢神经系统导致副作用。

== 与其它药物结合使用 ==
丁基東莨菪鹼与[[对乙酰氨基酚|对乙酰氨基酚]]配合的药物以{{lang|en|Buscopan Plus<sup>®</sup>}}为商标出售。

== 医学特性 ==
丁基東莨菪鹼被用来减缓[[平滑肌|平滑肌]]的运动，其作用可以持续数小时。它的半衰期为5.1小时。对经痛的作用有争议，原因是其非常小的生物利用度。

== 合成 ==
丁基東莨菪鹼是存在于植物中的[[东莨菪碱|东莨菪碱]]的半合成[[衍生物|衍生物]]。它是使用[[蔓陀缴华属|蔓陀缴华属]]等植物使用制药方法合成的。
