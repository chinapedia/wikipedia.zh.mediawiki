《'''依巴拉度时间'''》（{{jpn|j=イバラード時間|rm=Ibarādo Jikan}}）是[[吉卜力工作室|吉卜力工作室]]于2007年制作的[[动画|动画]][[原创动画录像带|原创动画录像带]]（OVA），以[[DVD|DVD]]和[[蓝光光碟|蓝光光碟]]发行，该片导演为[[井上直久|井上直久]]。<ref>{{Cite web |url=http://www.ghibli.jp/10info/004140.html |script-title=ja:イバラード時間 井上直久新作展が19日より開催されます。 |work=[[吉卜力工作室|吉卜力工作室]] |date=2007年7月18日 |accessdate=2013年5月31日 }}</ref>

片中的故事发生在一个被称为{{nihongo|依巴拉度|イバラード}}的[[架空世界|架空世界]]，其场景也曾成为吉卜力工作室出品的另一部电影《[[侧耳倾听|侧耳倾听]]》的灵感。<ref>{{Cite web |url=http://www.animenewsnetwork.com/news/2007-06-21/studio-ghibli%27s-iblard-jikan-video-trailer-online |title=Studio Ghibli's Iblard Jikan Video Trailer Online |work=Anime News Network |date=2007年6月21日 |accessdate=2013年5月31日 }}</ref>

== 作品內容 ==
此OVA動畫內容是透過數位畫面合成的效果，以動態的方式呈現先前井上直久所繪製的各幅依巴拉度畫作，再配上風格較輕快的配樂來讓觀眾賞析。

而此作品裡頭並未有特定的故事劇情，也未添加任何對白。

== 作品評價 ==
Mania.com評論《依巴拉度時間》是個有著相當精美的畫面、並且能讓人輕鬆賞析的作品，雖然沒有太多會讓人驚豔的橋段，但適合作為展示影視硬體設備能力的影片。<ref>{{Cite web |url=http://www.mania.com/iblard-jikan_article_79597.html |title=Iblard Jikan |work=Mania.com |author=Chris Beveridge |date=2007年7月10日 |accessdate=2013年5月31日 |deadurl=yes |archiveurl=https://web.archive.org/web/20120916212150/http://www.mania.com/iblard-jikan_article_79597.html |archivedate=2012年9月16日 }}</ref>
