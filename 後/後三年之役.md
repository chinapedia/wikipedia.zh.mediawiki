'''後三年之役'''（ごさんねんのえき）是[[平安時代|平安時代]]後期發生在[[陸奧國|陸奧]]・[[出羽國|出羽]]（[[東北地方_(日本)|東北地方]]）的[[戰役|戰役]]。[[前九年之役|前九年之役]]之後，成為東北地方霸者的[[俘囚|俘囚]][[清原氏|清原氏]]滅亡，[[奧州藤原氏|奧州藤原氏]]奠定在東北的基礎，[[河內源氏|源氏]]確立其在[[東國|東國]]的地位。

== 背景 ==
原本東北地方的兩大勢力為安倍氏跟清源氏，安倍氏於[[前九年之役|前九年之役]]滅亡後，清原氏成為東北的霸主。在永保3年（1083年）時，清原氏的當主之位已從{{tsl|ja|清原光頼}}轉移到弟弟{{tsl|ja|清原武則|武則}}一脈，傳至武則的兒子武貞，再傳至{{tsl|ja|清原真衡|真衡}}。武貞在前九年之役後，娶了被斬首的{{tsl|ja|藤原經清}}的妻子，也收養她與經清所生的兒子清衡，之後武貞又與她生一子，名為{{tsl|ja|清原家衡}}。

== 養子成衡及清原氏分裂 ==
清原武貞死後由真衡繼任，由於真衡沒有子嗣，便從平氏過繼一男為養子，即為{{tsl|ja|清原成衡}}。又讓成衡娶[[源賴義|源賴義]]之女為妻。婚禮上，在前九年之役獲有大戰功的長老{{tsl|ja|吉彥秀武}}帶沙金來祝賀，但真衡卻沉迷在與奈良法師下圍棋，等待許久的秀武認為受辱，將沙金灑向庭院而憤然離去，聽聞此事的真衡認為秀武此舉無禮，而欲討伐之，秀武便聯絡一直對真衡不滿的清衡與家衡共同作戰。

== 義家加入與真衡之死 ==
此時[[源賴義|源賴義]]嫡子[[源義家|源義家]]出任陸奧守，他同時也是真衡的妻舅，兩人便結成同一陣線。當真衡出兵北上之時，清衡與家衡聯軍趁此機會攻打真衡的大本營，但在士兵死守及義家及時來援的情況下大敗，兩人都向義家投降，後真衡在進軍出羽的路上急病而死。

== 清衡與家衡之爭 ==
真衡死後，義家將其所領有的奧六郡平分給清衡與家衡，清衡得[[和賀郡|和賀郡]]、[[江刺郡|江刺郡]]及[[膽澤郡|膽澤郡]]；家衡得[[岩手郡|岩手郡]]、[[紫波郡|紫波郡]]及{{tsl|ja|稗貫郡}}。但家衡對此結果不滿，於[[應德|應德]]3年（1086年）出兵偷襲清衡的居所，清衡僅以身免，其他家人全遭到殺害。

義家便幫助清衡對抗家衡，曾一度在沼柵（[[秋田縣|秋田縣]][[横手市|横手市]]雄物川町沼館）吃了一次敗仗。家衡聽從叔父{{tsl|ja|清原武衡}}之言將據點移至難攻不落的金澤柵（横手市金澤中野），而義家聽從吉彥秀武之言改打兵糧戰，包圍金澤柵直到對方兵糧耗盡。家衡及武衡因無糧食而放火燒金澤柵欲逃走，後來都被抓住並斬首。

== 戰後==
朝廷認為義家是以私人名義參戰，拒絕給予軍費，更要求解任陸奧守一職，義家便用家財犒賞手下及有功人士，此舉讓源氏在關東獲得極大名聲，清衡亦改回本姓統治陸奧地方，成為[[奧州藤原氏|奧州藤原氏]]之祖。
