'''後藤萌咲'''（{{jpn|j=後藤 萌咲|hg=ごとう もえ|rm=Gotō Moe|f=y}}，{{bd|2001年|5月20日}}）是[[日本偶像|日本偶像藝人]]，為女子偶像團體[[AKB48|AKB48]] Team A成員，[[名古屋市|名古屋市]]出身<ref name="googleplus_a6CUTcfhh6G" />，所屬經紀公司為[[AKS|AKS]]。

== 經歷 ==
; 2013年
* 11月10日，於「{{link-ja|AKB48集團選秀會議|AKB48のオーディション#AKB48グループ ドラフト会議}}」中，由AKB48 Team K在第一輪指名獲選。
; 2014年
* 2月20日，於『大島Team K「Waiting」公演』中，與同為Team K選秀生的[[下口緋奈奈|下口緋奈奈]]首度站上AKB48的劇場舞台。
* 2月24日，於{{link-ja|Zepp DiverCity Tokyo|Zepp}}舉辦的「[[AKB48集團大組閣祭|AKB48集團大組閣祭]]」中，宣布由選秀生昇格至Team K<ref>{{cite web|url=http://www.oricon.co.jp/special/959/ |script-title=ja:AKB48大組閣祭り『あなたの推しメンはどこの所属に？ 波乱の組閣イベントに密着』|publisher=[[oricon|ORICON NEWS]] |language=ja |date=2014-02-24|accessdate=2016-06-23}}</ref>。
* 5月7日，於『橫山Team K「RESET」公演』初日登場，首次以正式成員身分登台演出<ref>{{cite web|url=http://ameblo.jp/akihabara48/entry-11838606146.html |script-title=ja:5/7(水)の公演メンバー|publisher=AKB48官方Blog |language=ja |date=2014-05-01 |accessdate=2016-06-23}}</ref>。
; 2015年
* 3月26日，於埼玉超級競技場舉辦的「[[AKB48春季單獨演唱會～次期總現正修行中！～|AKB48春季單獨演唱會～次期總現正修行中！～]]」演唱會上宣布新的組閣人事異動上，獲宣布所屬分隊由Team K轉移至Team B<ref>{{cite web|url=http://www.oricon.co.jp/news/2050670/full/ |script-title=ja:AKB春の衝撃人事一覧――川栄卒業、北原NGT移籍、生駒留学解除|publisher=ORICON NEWS |language=ja |date=2015-03-26 |accessdate=2016-06-23}}</ref>。
* 10月25日，宣布新的十二人官方組合「{{link-ja|昆蟲學校|虫かご (AKB48グループの派生ユニット)}}」正式成立，組合的另外十一名成員分別是AKB48[[福岡聖菜|福岡聖菜]]、[[込山榛香|込山榛香]]、[[坂口渚沙|坂口渚沙]]、[[SKE48|SKE48]]的[[小畑優奈|小畑優奈]]與[[後藤樂樂|後藤樂樂]]、[[NMB48|NMB48]]的[[植村梓|植村梓]]與[[藪下柊|藪下柊]]、[[HKT48|HKT48]]的[[荒卷美咲|荒卷美咲]]與[[今村麻莉愛|今村麻莉愛]]、[[NGT48|NGT48]]的[[小熊倫實|小熊倫實]]與[[高倉萌香|高倉萌香]]<ref>{{cite web |url=http://www.tvlife.jp/2015/10/26/48881 |script-title=ja:AKB48の42枚目シングル曲センターは高橋みなみに決定! |publisher={{link-ja|TV LIFE|TV LIFE}} |language=ja |date=2015-10-26 |accessdate=2016-06-23 |deadurl=yes |archiveurl=https://web.archive.org/web/20160424072027/http://www.tvlife.jp/2015/10/26/48881 |archivedate=2016-04-24 }}</ref>。
* 12月26日，獲選為『Team B「正在戀愛中」公演』的初日公演成員之一<ref>{{cite web|url=http://ameblo.jp/akihabara48/entry-12108444734.html |script-title=ja:12/26木崎チームB 初日メンバーのお知らせ|publisher=AKB48官方Blog |language=ja |date=2015-12-20 |accessdate=2016-06-23}}</ref>。
; 2017年
* 6月17日，在[[AKB48第49張單曲選拔總選舉|AKB48第49張單曲選拔總選舉]]拿下第76名，進入Upcoming Girls組，首次在[[AKB48選拔總選舉|AKB48選拔總選舉]]入榜。
* 12月8日，在[[AKB48劇場|AKB48劇場]]12周年特別紀念公演上宣布的第五次組閣（分隊編組調整）上，獲宣布所屬分隊由Team B轉移至Team A<ref>{{cite web|url=https://ameblo.jp/akihabara48/entry-12334870873.html |script-title=ja:組閣発表|publisher=AKB48官方Blog |language=ja |date=2017-12-09 |accessdate=2017-12-12}}</ref>。
; 2018年
* 6月16日，在[[AKB48第53張單曲世界選拔總選舉|AKB48第53張單曲世界選拔總選舉]]拿下第65名，進入Upcoming Girls組，同時首次獲得擔綱歌曲[[AKB48組合構成#中心位置|Center]]（中心成員）的資格。
* 獲選成為韓日合作選秀節目《[[PRODUCE_48|PRODUCE 48]]》的參賽者，最終於第三次綜合評價止步，名次為第24名。

== 人物 ==
* 喜歡的顏色是粉紅色與水色<ref name=G+2014-05-11/>。
* 是家中的獨生女<ref>《AKBINGO!》播出日期：2015年9月22日</ref>。
* 因為受母親的影響而喜歡上[[松田聖子|松田聖子]]<ref>《{{link-ja|UTB|UP to boy}}》2016年5月號</ref>，並在《AKBINGO!》的歌唱比賽單元中演唱了《{{link-ja|天使的WINK|天使のウィンク}}》<ref>《AKBINGO!》播出日期：2014年8月12日</ref>。

=== AKB48集團相關 ===
* 曾參加SKE48第六期研究生試鏡會，於第二次審查時落選<ref>《橫山Team K「RESET」公演MC》出演日期：2014年6月11日</ref>。
* 尊敬的成員是SKE48的前成員[[松井玲奈|松井玲奈]]<ref>{{cite web|url=http://www.oricon.co.jp/news/2030695/full/ |script-title=ja:【AKBドラフト】大島優子、“一回り下”小6コンビにメロメロ「かわいくてしょうがない」|publisher=ORICON NEWS|language=ja |date=2013-11-10|accessdate=2016-06-23}}</ref>。
* AKB48集團各姊妹團分別喜歡的成員是：AKB48的[[島田晴香|島田晴香]]、SKE48的[[松井玲奈|松井玲奈]]、NMB48的[[山本彩|山本彩]]與HKT48的[[田島芽瑠|田島芽瑠]]<ref name=G+2014-05-11>{{cite web|url=https://plus.google.com/117368443117982031627/posts/h9GYN56mzSr |title=本人記事 |publisher=Google+ |language=ja |date=2014-05-11 |accessdate=2016-06-23}}</ref>。
