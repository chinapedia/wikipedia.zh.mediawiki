'''後豐港'''，又作「-{R|后豐港}-」，古稱「洪門港」，為[[金門|金門]]西岸的小漁村。位處[[水頭|水頭]]之北，[[夏墅|夏墅]]（或作「[[下市|下市]]」）之南，[[賢厝|賢厝]]之西。行政劃分上隸屬[[金門縣|金門縣]][[金城鎮_(金門縣)|金城鎮]]-{R|[[金水里|金水里]]}-，全村現有五十多戶居民，總人口約三、五百人。

== 歷史淵源 ==
後豐港為一閩南傳統聚落，主要以[[洪姓|洪姓]]居民組成，[[明|明]]末[[清|清]]初輔佐[[鄭成功|鄭成功]]與[[鄭經|鄭經]]父子的武將[[洪旭|洪旭]]，即出生於此村。<ref>金門在1915年設[[縣|縣]]以前大多數時期皆隷屬於[[福建省|福建省]][[同安縣|同安縣]]，因此金門出生之歷史人物多半被記載為同安人。</ref>當地人稱洪旭為「洪伯爺」（或作「洪伯牙乃閩南語稱謂」），因洪旭曾受[[隆武帝|隆武帝]]封為「忠振伯」之故也。出身此村的人士以洪旭所居之官位最為顯嚇，洪旭遺留給村民的軼事，最為後世子孫津津樂道。相傳洪旭曾將金銀珠寶藏於「十八石窖」，並留下「大水淹不著，小水淹三尺」的謎語，但後世子孫至今無人可破解。

== 地理環境 ==
昔日，金門從事近海捕魚者眾，因後豐港海邊避風效果良好，[[古崗|古崗]]村、[[後湖|後湖]]村漁民經常將船隻停泊於此村海邊，為數可達數十艘。後因經濟型態轉型，務農、捕魚者銳減，現海邊船隻已不多見。後豐港海邊的[[潮間帶|潮間帶]]盛產豐富的[[貝類|貝類]]、蝦蟹、海菜、[[鱟|鱟]]，是此村許多村民及附近村民的經濟來源之一。後因防波堤的興建及水頭商港的擴建，破壞當地潮間帶的生態，使得海產大為縮減，目前僅有海菜、貝類及幼隻的鱟較為常見。

== 宗教信仰 ==
此村的信仰中心為「洪氏宗祠」及「來鳳宮」。每年[[農曆|農曆]]正月十五日，來鳳宮的王爺建醮是此村的傳統。農曆七月中元普渡期間，洪氏宗祠有盛大的祭祀活動。另有「愛國將軍廟」及「萬聖公媽廟」，亦為信奉之對象。目前此村設有「後豐港社區發展協會」，為聯絡村民舉辦活動之主要窗口。

== 海外移民 ==
[[明|明]]朝末年，此村有許多居民移居[[澎湖|澎湖]]，[[澎湖縣|澎湖縣]][[湖西鄉|湖西鄉]][[尖山|尖山]]村的洪姓居民即為後豐港洪氏的後裔。
