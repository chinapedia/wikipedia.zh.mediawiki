'''後硬蕊'''（[[英文|英文]]：'''Post-hardcore'''）是一個發展自[[硬核龐克|硬核龐克]]的音樂類型，也是廣泛的[[龐克搖滾|龐克搖滾]]運動中的一個分支，就如[[後龐克|後龐克]]。許多後硬蕊的樂團都是來自硬核龐克樂壇，或是受到硬核音樂的啟發，但這些樂團更關注於表達內心深處。

隨著一些來自[[美國中西部|美國中西部]]的樂團發行了他們的作品，這個類型逐漸形成於1980年代的中晚期，特別是來自[[華盛頓特區|華盛頓特區]]的樂團[[Fugazi|Fugazi]]，<ref name="Posthardcoreallmusic">{{cite web |url=http://www.allmusic.com/explore/style/post-hardcore-d12962 |title=Explore: Post-Hardcore |work=[[Allmusic|Allmusic]] |accessdate=2011-03-18}}</ref>以及一些聲音些微獨特的樂團，像是[[Big_Black|Big Black]]和[[Jawbox|Jawbox]]，他們更接近起源自[[噪音搖滾|噪音搖滾]]的後硬核。

==特色==
後硬蕊發源自[[硬核龐克|硬核龐克]]，硬核的特色在於節奏快速、高分貝、沉重的貝斯<ref>{{cite book | last =Blush | first =Stephen | title =American Hardcore: A Tribal History | publisher =Feral House | date =2001-11-09 | isbn =0922915717}}</ref>與DIY精神。<ref name="Posthardcoreallmusic"/>在音樂網站 ''[[Allmusic|Allmusic]]'' 中指出：「這些被稱為後硬蕊的新興樂團，通常用複雜和活力的方式來發洩內心的憤怒。在狹義的硬蕊當中，應該採用『快吵原則』。此外，大多數的後硬蕊樂團主唱往往以低聲的哼唱，來取代過去的瘋狂喊叫。」<ref name="Posthardcoreallmusic"/>在Allmusic的資料庫中還指出，樂團採用創造性的方式，去建構、釋放音樂張力，而非「利用簡短、尖銳與瘋狂的爆發力來處理個人所面臨的問題」。<ref name="Posthardcoreallmusic"/>音樂雜誌《Treble》評論人Jeff Terich說：「這些藝術家並不會謹守硬蕊硬性的約束，他們的擴展超越了強力和弦與尖叫，為[[龐克搖滾|龐克搖滾]]精神結合了更多創意性的發洩管道。」<ref name="treblezine">{{cite web |url=http://www.treblezine.com/features/107.html |title=The 90-Minute Guide - Post-Hardcore |author=Terich, Jeff |date=2007-04-24 |work=Treblezine |accessdate=2011-03-20 |deadurl=yes |archiveurl=https://web.archive.org/web/20110404010139/http://www.treblezine.com/features/107.html |archivedate=2011-04-04 }}</ref>[[後龐克|後龐克]]是影響後硬蕊音樂發展的其中之一，後龐克是在1970年代晚期和1980年代早期的英國出現的。<ref name="Posthardcoreallmusic"/>在發展的過程中，許多樂團實驗性的將[[靈魂音樂|靈魂音樂]]、Dub音樂、[[放克|放克音樂]]、[[爵士|爵士]]、放克龐克結合。值得注意的是，許多後硬蕊樂手，都是從硬蕊音樂起家，他們在擁有熟練的技巧後，開始為他們所創作的音樂拓展新的聲音。<ref name="Posthardcoreallmusic"/>

==歷史==
[[File:Albini_atp.jpg|thumb]]的創團成員，在與樂團Shellac的演唱會中。]]
===1980年代===
====源起====
網站''About.com''的Ryan Cooper認為這種類型是「來自[[硬蕊|硬蕊]]樂團本身」<ref name="DefinitionAbout">{{cite web |url=http://punkmusic.about.com/od/punktionary/g/posthardcore.htm |title=Post-Hardcore - A Definition |author=Cooper, Ryan |work=[[About.com|About.com]] |accessdate=2011-04-05}}</ref>，如同樂團[[Black_Flag|Black Flag]]「開始對硬蕊音樂公式化的限制感到無趣，開始在他們的音樂中，加入一些實驗性的聲音」。<ref name="DefinitionAbout"/>一些開始在1980年代早期活動的樂團Saccharine Trust、<ref>{{cite web |url=http://www.trouserpress.com/entry.php?a=saccharine_trust |title=Saccharine Trust |last1=Robbins |first1=Ira |last2=Sprague |first2=David |work=TrousserPress.com |publisher= |accessdate=2011-03-18 |quote=Too early to be post-hardcore but too uncommon for any simple classification, this Southern California quartet doesn't try to create a blizzard of noise — they go at it more artfully, but with equally ear-wrenching results. [...]}}</ref>[[Naked_Raygun|Naked Raygun]]、<ref>{{cite web |url=http://www.allmusic.com/artist/p4989 |title=Naked Raygun |publisher=[[Allmusic|Allmusic]] |accessdate=2011-03-18}}</ref><ref name=TopArtistsAllmusic>{{cite web |url=http://www.allmusic.com/explore/style/post-hardcore-d12962/artists |title=Explore: Post-Hardcore (Top Artists) |work=[[Allmusic|Allmusic]] |accessdate=2011-03-18}}</ref><ref name="Effigies">{{cite web |url=http://www.allmusic.com/artist/effigies-p23913/biography |title=Effigies - Biography |author=Huey, Steve |work=[[Allmusic|Allmusic]] |accessdate=2011-05-12}}</ref> and [[The_Effigies|The Effigies]],<ref name="Effigies"/>，被視為[[後硬蕊|後硬蕊]]的先驅。成立於1981年，[[芝加哥|芝加哥]]的Naked Raygun開始將硬蕊與[[後龐克|後龐克]]（Post-punk）融合，這些影響他們的後龐克樂團有Wire、Gang of Four，<ref>{{cite web |url=http://www.allmusic.com/artist/naked-raygun-p4989/biography |title=Naked Raygun - Biography |author=Prato, Greg |work=[[Allmusic|Allmusic]] |publisher= |accessdate=2011-03-18}}</ref>而書籍''美國硬蕊：一個部落歷史''（American Hardcore: A Tribal History）的作者Steven Blush指出，樂團Naked Raygun使用「婉轉的歌詞與鮮明的後龐克旋律」。<ref name=sb>Blush, Steven. ''American Hardcore: A Tribal History''. Feral House: 2001. p. 222.</ref>同樣另一個從芝加哥場景的樂團The Effigies，發表了受硬蕊龐克樂團[[Minor_Threat|Minor Threat]]與英國後龐克樂團[[The_Stranglers|The Stranglers]]、[[Killing_Joke|Killing Joke]]、[[The_Ruts|The Ruts]]影響的音樂。<ref name="Effigies"/>

1980年到1985年間，許許多多的音樂家開始渴望，將原來硬蕊音樂型態進行實驗性的擴展，他們多半都是出身硬蕊音樂或是和這種音樂有很大的關係。<ref name="Posthardcoreallmusic"/> 這些樂團受到1980年代的[[噪音搖滾|噪音搖滾]]始祖Sonic Youth的影響。<ref name="treblezine" /> 部分與獨立唱片公司''Homestead Records''簽約的樂團都和後硬蕊有關，<ref name="treblezine" /><ref name="Effigies"/>其中包含了Squirrel Bait。<ref>{{cite web |url=http://www.allmusic.com/artist/squirrel-bait-p5509/biography |title=Squirrel Bait - Biography |author=Huey, Steve |work=[[Allmusic|Allmusic]] |accessdate=2011-03-19}}</ref>、Steve Albini所創立的樂團[[Big_Black|Big Black]]與[[Shellac|Shellac]]。<ref name=TopArtistsAllmusic/><ref>{{cite web |url=http://www.theskinny.co.uk/article/43937-shellac-no-free-lunch-still |title=Shellac: No Free Lunch, Still |author=Marticorena, Jorge |date=2008-10-16 |work=[[The_Skinny_(magazine)|The Skinny]] |publisher= |accessdate=2011-03-21}}</ref>Big Black的吉他手Santiago Durango正是來自樂團Naked Raygun，<ref name="BigBlackBio">{{cite web |url=http://www.allmusic.com/artist/big-black-p10143/biography |title=Big Black - Biography |author=Deming, Mark |work=[[Allmusic|Allmusic]] |publisher= |accessdate=2011-03-19}}</ref>這使他們以嚴格的「DIY精神」為名，<ref name="treblezine" />最實際的做法就是他們自掏腰包錄製音樂、安排自己的演出、管理自己並自行進行宣傳。在當時已經有許多獨立樂團擴大自己的廠牌，好撈一筆，但他們頑固地保持獨立。<ref name="BigBlackBio"/>Big Black的音樂受[[工業搖滾|工業搖滾]]的影響，利用鼓機（drum machine）製造出被打斷的效果，<ref name="BigBlackBio"/>而Steven Blush也稱這些有Steve Albini的樂團作品為「一種充斥著對『僵硬的英國後龐樂團Gang of Four』憂心的回應」。<ref name=sb/>在發行單曲《Il Duce》後，Big Black轉到''Touch and Go唱片''，<ref name="BigBlackBio"/>後來的唱片不只重發Big Black先前全部的作品，也發行了來自[[奧斯汀|德州奧斯汀]]的後硬蕊樂團[[Scratch_Acid|Scratch Acid]]全部的作品，<ref name="ScratchAcid">{{cite web |url=http://www.allmusic.com/artist/scratch-acid-p5376/biography |title=Scratch Acid - Biography |author=Erlewine, Stephen Thomas |work=[[Allmusic|Allmusic]] |accessdate=2011-03-19}}</ref>根據[[Allmusic|Allmusic]]資深編輯Stephen Thomas Erlewine所言：「他們為許多採用失真、嘎嘎作響的90年代龐克樂手打下基礎。」<ref name="ScratchAcid"/>

在美國以外的地方，這種類型的音樂成形於[[加拿大|加拿大]]樂團Nomeansno，<ref>{{cite journal |last1=Gold |first1=Jonathan |year= 1996|title=Spins: Platter Du Jour - 7 - NOFX - Heavy Petting Zoo |journal=[[Spin_(雜誌)|Spin]] |publisher= Camouflage Associates |volume=12 |issue=1 |page=113 |url=http://books.google.com.co/books?id=JheoECFjDqMC&lpg=PP1&pg=PA113#v=onepage&q&f=false |accessdate=2011-03-27}}</ref>他們成軍於1979年，和Jello Biafra有點關係，唱片公司為''Alternative Tentacles''。有一位評論家提到，這個樂團在1989年發行專輯《Wrong》，是「有時以來最激進有力的後硬蕊作品」。<ref>{{cite web |url=http://www.dustedmagazine.com/reviews/3038 |title=Nomeansno - All Roads Lead to Ausfahrt |author= Mosurock, Doug |date= 2006-08-03 |work=Dusted Magazine |accessdate=2011-03-27}}</ref>

====華盛頓特區的場景====
1984與1985年間，在華盛頓特區的硬蕊樂壇中（即harDCore<ref>Grubbs, p. 14</ref>），一個新的運動開始出現，並「掃除」了當時的風氣。<ref name="ROSDischord">{{cite web |url=http://www.dischord.com/band/rites-of-spring |title=Rites of Spring |work=[[Dischord_Records|Dischord Records]] |accessdate=2011-03-19}}</ref>這個音樂運動由當地獨立廠牌[[Dischord唱片|Dischord唱片]]（Dischord Records）的樂團領頭，在1980年代一些始祖樂團，如[[Minor_Threat|Minor Threat]]、State of Alert、Void與Government Issue。<ref>{{cite web |url=http://www.dischord.com/history/ |title=Dischord History |author=MacKaye, Ian | page= 1 |date = 1999-11|work=[[Dischord_Records|Dischord Records]] |accessdate=2011-03-19 |authorlink= Ian MacKaye}}</ref><ref name="DischordStylus">{{cite web |url=http://www.stylusmagazine.com/articles/label_profile/dischord-records.htm |title=Dischord Records - Label Profile |last1=McElligatt |first1=Colin |date=2003-09-01 |work=[[Stylus_Magazine|Stylus Magazine]] |accessdate=2011-04-06}}</ref>根據Dischord唱片的網站上所寫的：「這些被大眾媒體認為是龐克搖滾的暴力、無政府主義樂團，開始成為華盛頓特區的主流，一些老牌的龐團忽然發現他們家鄉被這樣的風氣所佔領，開始感到排斥與氣餒，<ref name="ROSDischord"/>」導致了「一段時間的重新定義」。<ref name="ROSDischord"/>這些年間，一些新一波的樂團開始成立，包括[[Rites_of_Spring|Rites of Spring]]、Lunchmeat（就是大家所熟知的[[Soulside|Soulside]]）、Gray Matter、Mission Impossible、Dag Nasty和[[Embrace|Embrace]]，<ref name="DischordHist2">{{cite web |url=http://www.dischord.com/history/page02 |title=Dischord History |author=MacKaye | page= 2 |accessdate=2011-03-19}}</ref>Embrace的成員有樂團[[Minor_Threat|Minor Threat]]的主唱與Dischord唱片的成立人之一Ian MacKaye。這個運動被稱為「革命夏季」（Revolution Summer）。<ref name="ROSDischord"/><ref>Grubbs, p. 22. "After hearing the phrase 'revolution summer' at the Neighborhood Planning Council, Dischord employee Amy Pickering had an idea. [...] With the idea of celebrating a new scene's birth, she sent out anonymous letters stating that Revolution Summer was coming and people should be ready. The summer of '85 was indeed that."</ref>

Rites of Spring更「超越了『帶領這項革命』的殊榮」，挑戰「當時在龐克樂壇中，裝腔作勢並過於盛行的男子氣概」，且「更重要的是」，輕看了「音樂與風格的規則」。<ref name="ROSDischord"/> 媒體記者Steve Huey寫道，當這些樂團「偏離傳統硬蕊關心當時對外界的掛心－也就是，社會與政治異議－他們的音樂上的砭針不再猛烈，事實上比起一般快速到難以區分的三弦來說，這更具有挑戰性與鑑別度，」<ref name="ROSAllmusic">{{cite web |url=http://www.allmusic.com/artist/rites-of-spring-p11541/biography |title= Rites of Spring - Biography|author=Huey, Steve |work=[[Allmusic|Allmusic]] |accessdate=2011-03-19}}</ref> 根據Huey所言，這個新聲音由Hüsker Dü的作品《Zen Arcade》，為「基於革命的硬蕊音樂」標示出「新方向」。<ref name="ROSAllmusic"/> 其他的樂團，被認為是受到[[放克|放克音樂]]的啟發<ref>{{cite web |url=http://www.allmusic.com/album/plays-for-lovers-r320703/review |title= Plays for Lovers - Beefeater - Review |author=Raggett, Ned |work=[[Allmusic|Allmusic]] |accessdate=2011-03-19 |quote=Drawing from funk as much as punk, Beefeater cooks up a groovy combination on their debut album.}}</ref>與1960年代的美國[[流行音樂|流行音樂]]。<ref>{{cite web |url=http://www.allmusic.com/artist/gray-matter-p13081/biography |title=Gray Matter - Biography |author=Foster, Patrick |work=[[Allmusic|Allmusic]] |publisher= |accessdate=2011-03-19 |quote=The recordings, which revealed the influence of early-D.C. punk ('Gray Matter', 'Caffeine Blues'), also warned about the dangers of punk nostalgia ('Retrospect') and featured a surprising cover of [[the_Beatles|the Beatles]]' '[[I_Am_the_Walrus|I Am the Walrus]]' was the first hint of the band's strong pop streak.}}</ref>

根據Eric Grubbs，這新聲音的命名開始發展，有些人認為應該稱他「Post-harDCore」，但在這場景中，另一個更為流傳的稱呼為「[[Emo|Emo-core]]」（情感核、情緒搖滾）。<ref name=Grubbs27>Grubbs, p. 27</ref>後者出現在滑板雜誌''Trasher''裡頭，在華盛頓特區一帶被提出來討論。<ref name=Grubbs27/>有些樂團被視為Emo誕生的貢獻者，<ref name="DefinitionAbout"/><ref>{{cite web |url=http://www.allmusic.com/explore/style/emo-d4525 |title= Explore: Emo |work=[[Allmusic|Allmusic]] |accessdate=2011-03-19}}</ref><ref name="UKHC">{{cite web |url=http://www.viceland.com/int/v16n10/htdocs/this-is-ukhc-not-la-202.php |title=This is UKHC, Not LA |author=Capper, Andy |work=[[Vice_(magazine)|Vice]] |accessdate=2011-03-20 }}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>像是Rites of Spring，有時他被譽為第一個或是早期頭幾個Emo表演者，<ref name="treblezine"/><ref name="ROSAllmusic"/>但樂團的創始人Guy Picciotto和MacKaye自己，都表示反對這個用詞。<ref>{{cite web |url=http://www.markprindle.com/picciotto-i.htm |title=Guy Picciotto interview |author=Prindle, Mark |year=2003 |work=Markprindle.com |accessdate=2011-03-19 |quote=Well, first of all, I don't recognize that attribution. I've never recognized 'emo' as a genre of music. I always thought it was the most retarded term ever. I know there is this generic commonplace that every band that gets labeled with that term hates it. They feel scandalized by it. But honestly, I just thought that all the bands I played in were punk rock bands. The reason I think it's so stupid is that - what, like the [[Bad_Brains|Bad Brains]] weren't emotional? What - they were robots or something? It just doesn't make any sense to me.}}</ref><ref>Grubbs, p. 27–28</ref><ref>{{cite book |last=Cogan |first=Brian |title=The Encyclopedia of Punk |year=2008 |publisher=Sterling |isbn=978-1402759604 |page= 97}}</ref>[[馬里蘭州|馬里蘭州]]附近，被歸類為後硬蕊的類似樂團開始出現，包括了[[Moss_Icon|Moss Icon]]與[[The_Hated|The Hated]]。<ref name="UKHC"/><ref>{{cite web |url=http://allmusic.com/artist/the-convocation-of-p482269/biography |title=The Convocation Of... - Biography |author=Butler, Blake |work=[[Allmusic|Allmusic]] |accessdate=2011-03-20}}</ref>前者個音樂包含Steve Huey所說的「不斷變換的力度，悅耳地彈奏出快速的吉他和絃，與[[吼腔|吼腔]]、[[哭腔|哭腔]]」，<ref name="MI">{{cite web |url= http://www.allmusic.com/artist/moss-icon-p213800/biography |title=Moss Icon - Biography |author=Huey, Steve |work=[[Allmusic|Allmusic]] |accessdate=2011-03-20}}</ref>儘管這些樂團不穩定的存立，這些都影響了後輩的音樂家。<ref name="MI"/>這些樂團都是後來被認為是早期的Emo表演者。<ref name="MI"/>

1980年代第二波於華盛頓特區成立的樂團有[[Shudder_to_Think|Shudder to Think]]、[[Jawbox|Jawbox]]、[[The_Nation_of_Ulysses|The Nation of Ulysses]]和[[Fugazi|Fugazi]]與[[Baltimore,_Maryland|Baltimore]]的[[Lungfish|Lungfish]]。<ref name="DischordHist2"/>MacKaye說那時的Dischord唱片可是前所未有的忙。<ref name="DischordHist2"/>大多的樂團，還有早期的那些，為1989年的合輯《State of the Union》發聲，<ref>{{cite web |url=http://www.dischord.com/release/32 |title=Various Artists - State of the Union |work=[[Dischord_Records|Dischord Records]] |accessdate=2011-03-20}}</ref>這張發行同時也紀錄了1980年代晚期華盛頓特區的龐克樂壇新聲音。<ref>{{cite web |url=http://www.allmusic.com/album/state-of-the-union-dc-benefit-compilation-r23103/review |title=State of the Union: D.C. 這些持續發展的樂團，他們的音樂為Dischord唱片後來的發行鋪路，預示了這些硬蕊音樂將更加的實驗性。<ref name="DischordStylus"/>樂團成員有：MacKaye、Picciotto、Rites of Spring的前鼓手Brendan Canty與前貝斯手Joe Lally，在1989年發行合輯《13 Songs》，包含Fugazi他們早期的同名EP與《Margin Walker》，這張作品，現今被視為一個發展史上，重要的里程碑。<ref name="13s">{{cite web |url=http://www.allmusic.com/album/13-songs-r7861/review |title=13 Songs - Fugazi - Review |author=Kellman, Andy |work=[[Allmusic|Allmusic]] |accessdate=2011-03-20}}</ref> 無獨有偶，樂團1990年發行的專輯《Repeater》，也被視為經典之作。<ref name="FugaziBio"/>他們的[[行動主義|行動主義]]、低價的表演與CD，以及拒絕向主流通路妥協的態度，使樂團獲得普遍的認可。<ref name="FugaziBio"/>另一方面，樂團Jawbox受到「80年代芝加哥傳統崛起風氣」的影響，<ref name="JawboxBio">{{cite web |url=http://www.allmusic.com/artist/jawbox-p13316/biography |title= Jawbox - Biography |author=Kellman, Andy |work=[[Allmusic|Allmusic]] |accessdate=2011-03-20}}</ref>而樂團The Nation of Ulysses則是以「讓人印象深刻地高舉樂團[[MC5|MC5]]馬達嘴的改革性華麗手法」配合著「[[R&B|R&B]]元素（帶有MC5味道的）與[[前衛爵士|前衛爵士]]（{{tsl|en|Avant-garde jazz|Avant Jazz}}）」加上「活潑、輕快的現場表演」，他們是受到「地方與海外新一匹樂團」的激盪成的。<ref>{{cite web |url=http://www.allmusic.com/artist/the-nation-of-ulysses-p13761/biography |title=The Nation of Ulysses |author=Huey, Steve |work=[[Allmusic|Allmusic]] |accessdate=2011-03-20}}</ref>
