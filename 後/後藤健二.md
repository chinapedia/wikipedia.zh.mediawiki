'''後藤健二'''（{{jpn|j=後藤 健二|hg=ごとう けんじ|rm=Gotō Kenji}}；{{bd|1967年|9月22日<ref>『文藝年鑑』2008年</ref>|2015年|1月30日|Gotō Kenji}}），[[日本|日本]]著名的資深自由[[記者|記者]]、[[紀錄片|紀錄片]][[製作人|製作人]]及[[作家|作家]]，長年在[[阿富汗|阿富汗]]、[[車臣|車臣]]與[[敘利亞|敘利亞]]等衝突地區進行[[戰地|戰地]]報導，關注當地平民在戰時的生活狀況，亦出版過關於[[艾滋病|艾滋病]]影響[[非洲|非洲]]的書籍。他的作品常見於[[NHK|NHK]]和[[朝日電視台|朝日電視台]]等多家[[日本媒體|日本媒體]]。

2014年10月，後藤欲前往[[敘利亞|敘利亞]]試圖解救他的朋友，即於2014年8月被[[伊斯兰国|伊斯兰国]]綁架的另一名日本人[[湯川遙菜|湯川遙菜]]。但在10月底後便與外界失去聯絡。幾周後，後藤妻子收到[[伊斯蘭國|伊斯蘭國]]索要[[贖金|贖金]]的電子郵件。<ref name="BBC150120">{{Cite web |url=http://www.bbc.co.uk/zhongwen/trad/world/2015/01/150131_profile_goto |title=新聞人物：誰是後藤健二？ |publisher=BBC |accessdate=2015-01-20| date=2015-01-20}}</ref>2015年1月20日，伊斯兰国在网络上公布一段影片，证实他们已經[[绑架|绑架]]两名日本[[公民|公民]]后藤健二和汤川遥菜，要求[[日本政府|日本政府]]在72小时内交付2亿美元的赎金，否则把人质处决<ref name=mongying/>。同年1月30日後藤遭到[[聖戰士約翰|聖戰士約翰]]斬首。

== 人物經歷 ==
=== 早年 ===
後藤出生於[[日本|日本]][[宮城縣|宮城縣]][[仙台市|仙台市]]<ref>[http://getnews.jp/archives/786815 「「私が代わりに人質に」＝後藤さん母「政府信じる」―涙ながらに訴え・東京時事] {{webarchive|url=https://web.archive.org/web/20150128023223/http://getnews.jp/archives/786815 |date=2015-01-28 }}2015.01.25 14:13 時事通信社 カテゴリー </ref>，是家中的幼子。3歲時父母離異，母親其後改嫁石堂行夫，並改名為石堂順子。他在法政大学第二高等学校完成中學，並於1991年從[[東京|東京]][[法政大學|法政大學]][[社會學|社會學]]系畢業<ref>{{Cite journal|date = 2015-01-29|script-title = ja:人質殺害「イスラム国」禁断レポート「10分300万円」に命を賭けた後藤健二さん47歳 書かれざる数奇な人生|journal = 週刊文春2月5日号|page = 24|publisher = 株式会社文藝春秋|location = 東京都}}</ref>。1996年，他創辦了自己的獨立新聞媒體通訊社「the Independent Press」。

=== 事業 ===
後藤既是一名自由記者，也同時是媒體製作人、紀實文學作家、社會工作者。後藤一直致力於報導世界各地受戰爭蹂躪的國家，關注「戰爭與衝突、難民、貧窮、愛滋病、兒童教育」等人道議題<ref name="independentpress">{{cite web |url= http://ipgoto.com/about/%E5%BE%8C%E8%97%A4-%E5%81%A5%E4%BA%8C|title= Independent Press — about後藤健二}}</ref>，特別是非洲和中東的相關情況。他亦有出版了多部以此為題的作品，對[[塞拉利昂|塞拉利昂]]兒童兵、[[盧旺達|盧旺達]]衝突、[[愛沙尼亞|愛沙尼亞]]的[[愛滋病|愛滋病]]村以及[[阿富汗|阿富汗]]的教育情況進行各種實況紀錄<ref name="jt20150120">{{cite web |url= http://www.japantimes.co.jp/news/2015/01/20/national/islamic-state-threatens-kill-two-japanese-hostages/#.VMC58i4vv-s|title= Islamic State threatens to kill two Japanese hostages |first= Reiji|last= Yoshida|date= 20 January 2015|work= [[The_Japan_Times|The Japan Times]]|publisher= The Japan Times Ltd.|location= Japan|page= |archiveurl= |archivedate= |accessdate= 22 January 2015}}</ref>。2006年，他出版了紀實文學書籍《我們想要和平，不要鑽石》（{{lang|ja|ダイヤモンドより平和がほしい}}），記述了面臨戰爭衝突的國家地區中兒童的生活情況。該書廣受好評，並獲得了第53屆[[日本|日本]][[產業經濟新聞社|產業經濟新聞社]]兒童出版文化獎<ref>{{Cite web |url=http://www.christiantoday.co.jp/articles/13401/20140530/goto-kenji.htm |script-title=ja:【インタビュー】国際ジャーナリスト・後藤健二〜それでも神は私を助けてくださる〜」 |accessdate=2015-01-20 |date=2014-05-30 |publisher=クリスチャントゥデイ}}</ref>。他也是日本文藝家協會的會員<ref name="independentpress"/>。

另外，他亦經常參與[[聯合國|聯合國]]人道支援機構的組織活動，包括[[聯合國兒童基金會|聯合國兒童基金會]]和[[聯合國難民署|聯合國難民署]]<ref name="independentpress"/><ref name="jt20150122_2">{{cite web |url= http://www.japantimes.co.jp/news/2015/01/22/national/how-the-hostages-lives-intertwined/#.VMC4nC4vv-s|title= How the lives of hostages Yukawa and Goto became intertwined|first= Mari|last= Yamaguchi|date= 22 January 2015|work= [[The_Japan_Times|The Japan Times]]|publisher= The Japan Times Ltd.|location= Japan|page= |archiveurl= |archivedate= |accessdate= 22 January 2015}}</ref>。[[2011年東日本大地震|2011年東日本大地震]]期間，他於重災區[[石卷市|石卷市]]及[[氣仙沼市|氣仙沼市]]為[[聯合國兒童基金會|聯合國兒童基金會]]（UNICEF）擔任記錄員<ref>[http://www.asahi.com/articles/ASH1N5V60H1NUTIL03H.html {{lang|ja|「「果敢な記者」人質に　後藤さん、１０月下旬に音信不通」}}]朝日新聞2015年1月20日</ref><ref>[http://www.kahoku.co.jp/tohokunews/201501/20150121_13021.html {{lang|ja|「＜２邦人殺害警告＞仙台出身後藤さん無事祈る」}}] {{webarchive|url=https://web.archive.org/web/20150128115402/http://www.kahoku.co.jp/tohokunews/201501/20150121_13021.html |date=2015-01-28 }}河北新報2015年01月21日水曜日</ref>。社會活動方面，他推動了非營利媒體的公益計劃『The World With You』<ref name="independentpress"/>。近年多在日本全國的中學巡迴演講授課，希望把多年來工作中所遇到的受訪者口中的信息直接傳達給各人<ref name="independentpress"/>。

== 私人生活 ==
* 他曾經說過對於自己的理念：「那些我去採訪報導的地方遭遇著巨大的困難，但即便在那裏，人們也每天生活著……那些人總有些話要說，有信息要傳達。如果我能幫助他們將信息傳遞給世界，那麼就可能促成某種解決方法……若真那樣，我也就可以說，我的工作是成功的了<ref name="BBC150120"/>。」他亦曾經在談到自己的工作抱負時，表示：「我想要擁抱人群。那是我表示親近的最好方式。藉由擁抱，我可以與人們談話。我可以傾聽他們的想法、痛苦和希望。」<ref>{{Cite web |url=http://www.appledaily.com.tw/realtimenews/article/international/20150201/552504/%E3%80%8C%E6%83%B3%E6%93%81%E6%8A%B1%E4%BA%BA%E7%BE%A4%E3%80%8D%E3%80%80%E5%BE%8C%E8%97%A4%E5%82%BE%E8%81%BD%E5%8F%97%E9%9B%A3%E8%80%85%E5%BF%83%E8%81%B2 |title=「想擁抱人群」　後藤傾聽受難者心聲 |publisher=蘋果日報}}</ref>
