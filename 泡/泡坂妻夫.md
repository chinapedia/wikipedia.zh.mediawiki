'''泡坂妻夫'''（{{bd|1933年|5月9日|2009年|2月3日|catIdx=Awasaka, Tsumao}}）<ref>{{Cite web |url=http://homepage1.nifty.com/naokiaward/jugun/jugun103AT.htm |title=直木賞受賞作家 |access-date=2014-02-08 |archive-url=https://web.archive.org/web/20140222043750/http://homepage1.nifty.com/naokiaward/jugun/jugun103AT.htm |archive-date=2014-02-22 |dead-url=yes }}</ref>，本名'''厚川昌男'''（Atsukawa Masao），其筆名是由本名的拼音重新排列組合而成，[[日本|日本]][[推理小說|推理小說]]家兼[[魔術師|魔術師]]，[[東京都|東京都]]人，[[東京都立九段高等学校|東京都立九段高等学校]]毕业。

==生平==
泡坂妻夫的祖上是東京都神田地区有名的[[纹章上绘师|纹章上绘师]]，屋号“松叶屋”，继承家业绘制和服家纹之余，亦从事魔术表演，1968年荣获日本魔术界最高奖项“[[石田天海奖|石田天海奖]]”，后更以本名设立魔术大奖“厚川昌男奖”，留下魔术著作四种，影响十分深远。

1975年，泡坂妻夫以短篇小说《DL2号机事件》获得首届《[[幻影城|幻影城]]》新人佳作奖，从此登龙推理文坛，两年后便以第二部长篇小说《[[失控的玩具|失控的玩具]]》成功夺得[[日本推理作家协会奖|日本推理作家协会奖]]，而且是当年[[直木奖|直木奖]]的候补作品之一。1988年，短篇小说集《折鹤》获[[泉镜花文学奖|泉镜花文学奖]]。1990年终以《荫桔梗》顺利拿下第103届直木奖<ref>{{Cite web |url=http://homepage1.nifty.com/naokiaward/jugun/jugun103AT.htm |title=直木賞受賞作家 |access-date=2014-02-08 |archive-url=https://web.archive.org/web/20140222043750/http://homepage1.nifty.com/naokiaward/jugun/jugun103AT.htm |archive-date=2014-02-22 |dead-url=yes }}</ref>。2009年2月3日，因急性主动脉夹层动脉瘤而病逝，享年75岁。

==風格==
泡坂妻夫对[[魔术|魔术]]、[[玩具|玩具]]、纹章等物事具有精深、独到的研究，这些知识都被融进了他的推理小说里面。日本推理鬼才[[山田风太郎|山田风太郎]]曾说：“如果有日本作家能写出像[[G·K·切斯特顿|G·K·切斯特顿]]那样充满颠覆和创意的推理小说，这个人肯定就是泡坂妻夫。”
