'''已阅门'''，指2009年8月11日[[浙江省|浙江省]][[宁波市|宁波市]][[鄞州区|鄞州区]]政府网站互动平台中，网民咨询[[邱隘镇|邱隘镇]]道路出行难的问题仅仅获得“已阅”两字回复的事件。<ref name="xinhua-zj">{{cite web|url=http://www.zj.xinhuanet.com/newscenter/2009-08/15/content_17406706.htm|title=批复"已阅"就完了？网友狠批"最牛区长信箱回复"|date=2009-08-15|accessdate=2011-01-07|publisher=今日早报|author=翁璟|deadurl=yes|archiveurl=https://web.archive.org/web/20160304130421/http://www.zj.xinhuanet.com/newscenter/2009-08/15/content_17406706.htm|archivedate=2016-03-04}}</ref>这一事件遭到了当事人和网民的讽刺和声讨，也使[[中国大陆|中国大陆]]政府和政府网站中存在的官僚作风得到媒体和社会的关注。<ref>{{cite web|url=http://www.mzyfz.com/news/times/c/20090824/162848.shtml|title=宁波“已阅门”拷问电子政务|date=2009-08-24|accessdate=2011-01-07|publisher=民主与法制时报|author=杨明}}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

== 事件经过 ==
2009年8月5日，网民“天生我才2”在鄞州区“区长信箱”中反映关于邱隘镇断头路导致居民出行难的问题：

{{quote|信件标题：兴宁东路延伸到盛莫路问题 发信时间：2009-08-05 13:21:53

领导好，我想问一下，邱隘居民一直以来都存在出行难的问题。规划上说兴宁东路近期要延伸下来跟邱隘的盛莫路连接起来，不知道具体什么时候实行？？希望政府能够几块解决我们出行难的问题！谢谢}}

2009年8月11日，区长信箱回复：

{{quote|受理编号：00029346 反馈时间：2009-08-11 08:18:00

已阅}}

提问者当天将回复的截图发送到宁波本地的[[天一论坛|天一论坛]]上，称之为“牛X的鄞州区区长信箱回复”，这样的回复方式随即遭到网友抨击和辛辣讽刺。<ref name="xinhua-zj" />同时，有网友也列举了自己的类似经历。

== 媒体报道与官方反应 ==
[[新华社|新华社]]今日早报于2009年8月15日对此事件进行了报道。报道中叙述了鄞州区“区长信箱”受理中心主任陈荣华的采访。陈证实网民的反映属实，并指出办理结果由鄞州区交通局填写，而填写的结果“区长信箱”管理方无法进行管理。对于“已阅”的回复，陈在认为这一回复表示“相关部门会重视起来”的同时，也认为，“已阅”的回复，过于“简单机械”。而鄞州区交通局则向记者表示，对于转给他们的信件，“不需要具体办理，看一下就可以”。但对于这样的解释，网民并不接受，认为“已阅”的回复仅仅是敷衍而已。<ref name="xinhua-zj" />此后，陈向媒体解释，是“软件设计不尽合理”导致将内部信息转变为公开内容。<ref name="chinanews">{{cite web|url=http://www.chinanews.com/sh/news/2009/08-30/1840980.shtml|title=宁波鄞州回应区长信箱回复市民“已阅”事件|date=2009-08-30|accessdate=2011-01-07|publisher=[[中新网|中新网]]|author=何蒋勇}}</ref>

据报道，“已阅门”事件得到了鄞州区区长薛维海的重视，要求对于转发的内容，受理部门不允许只填“已阅”了事。同时，鄞州区政府对网上信访中曾经出现的其他问题也进行了批评，并制定了整改方案。<ref name="qjwb">{{cite web|url=http://politics.people.com.cn/GB/14562/9892105.html|title=宁波“最牛区长信箱”走出“已阅门” 欢迎监督|date=2009-08-19|accessdate=2011-01-08|publisher=[[钱江晚报|钱江晚报]]|author=周皓亮 倪王镇}}</ref>

== 相关评论 ==
由于“已阅门”所体现的问题在中国大陆具有共性，因而这一事件受到了中国大陆媒体的广泛关注。大多数媒体评论认为，“已阅门”是网络时代的官腔，损害了民众对官员的信任。<ref>{{cite web|url=http://www.chinanews.com/gn/news/2009/08-17/1821936.shtml|title=“已阅”之类官腔儿可休矣|author=张丽|publisher=[[北京晚报|北京晚报]]|date=2009-08-17|accessdate=2011-01-08}}</ref><ref>{{cite web|url=http://www.chinanews.com/gn/news/2009/08-17/1821484.shtml|title=“已阅”的霉味|author=刘敏|publisher=[[长江日报|长江日报]]|date=2009-08-17|accessdate=2011-01-08}}</ref><ref>{{cite web|url=http://pinglun.youth.cn/wywy/shsz/200908/t20090818_994172.htm|title=“最牛区长已阅”：一枚官腔网络版“图章”|author=廖洁|publisher=[[国际在线|国际在线]]|date=2009-08-18|accessdate=2011-01-08}}</ref>但也有评论认为，能够对市民的意见作出回应，已经是一种进步。<ref>{{cite web|url=http://www.chinanews.com/gn/news/2009/08-19/1825254.shtml|title=“已阅”开了好头|publisher=[[武汉晚报|武汉晚报]]|date=2009-08-19|accessdate=2011-01-08}}</ref>
