'''时令湖'''([[拉丁语|拉丁语]]:"Lacus Temporis")是[[月球正面|月球正面]]东北侧的一座小[[月海|月海]]区，大小约为190×100公里。其拉丁名称于1976年被[[国际天文学联合会|国际天文学联合会]]批准采纳<ref name=gazetteer>{{cite web

==位置及相关特征==
时令湖的中心[[月面坐标|月面坐标]]为{{Coord|46.8|N|56.2|E|globe:Moon}}<ref name=gazetteer/>，周边环绕着一些无名的小[[月海|月海]]区，其中，东南80公里处坐落着[[希望湖|希望湖]]。时令湖往北70公里是已被[[熔岩|熔岩]]填塞的[[恩底弥翁环形山|恩底弥翁环形山]]，一些高程数据显示，该湖位于直径350公里的撞击坑中心附近，但是否存在这样一座已损毁的陨坑，仍有待商榷<ref name=Wood_2011_lpod>{{cite web

月湖周围分布着很多月坑(不包括无名坑和卫星坑)，东面是[[墨丘利环形山|墨丘利环形山]]、东南是[[卡林顿陨石坑|卡林顿陨石坑]]和[[舒马赫环形山|舒马赫环形山]]、南面坐落着[[穆萨拉环形山|穆萨拉环形山]]、西南是[[胡克陨石坑|胡克陨石坑]]和[[沙克伯勒陨石坑|沙克伯勒陨石坑]]，而[[谢瓦利尔环形山|谢瓦利尔环形山]]和[[阿特拉斯陨石坑|阿特拉斯环形山]]则位于西侧、月湖的西南({{Coord|51.67|N|51.05|E|globe:Moon|name=concentrically crater}})是一座直径7公里保存完好的同心环坑<ref name=Wood_2004_lpod>{{cite web

==描述==
时令湖从西南向西北延伸约190公里，宽度达到100-120公里。它由两个大小接近的圆形区域-可能是已严重损毁并被熔岩淹没的撞击坑组成<ref name=Wood_2004_lpod/>，二者连接地带有几处被熔岩覆盖的破缺陨坑遗迹和直径9公里的年轻卫星坑"[[谢瓦利尔环形山|谢瓦利尔 F]]"，它是该月湖中最大且被熔岩填塞的陨坑，也是截止2015年唯一被命名的撞击坑(不包括位于湖岸上的)<ref>According to [http://planetarynames.wr.usgs.gov IAU nomenclature guide IAU]</ref>。

时令湖及周边无名月海区均较[[希望湖|希望湖]]更亮<ref name=Wood_2011_lpod/>，其表面位于[[月面学|月面]]基准高度下1.7-1.9公里，大约与希望湖处同一水平面。但[[恩底弥翁环形山|恩底弥翁环形山]]却高出基准月面1公里<ref name=JMARS>According to the laser altimeter on the satellite [[月球勘测轨道飞行器|Lunar Reconnaissance Orbiter]], obtained through the program [http://jmars.asu.edu JMARS].</ref>。

依据月湖中陨坑密度估测，该月湖被熔岩覆盖的时间大约在37-38亿年前<ref name=Hiesinger_2011>{{cite book
