'''時裝大樓'''（'''{{lang-ja|ファッションビル}}'''，'''Fashion Building'''）為一[[和製英語|和製英語]]詞，意指以[[時裝|時裝]]和[[快速消費品|生活雜貨]]專門店為主要承租戶的[[商場|商場]]大樓。<ref name="読売新聞">{{cite news|quote=「人気ブランドのブティックやセレクトショップなど衣料・雑貨関係のテナントが集まった商業施設。|title=ファッションビルとは |publisher=[[讀賣新聞|読売新聞]] |date=2009-05-28 |work=読売新聞大阪夕刊}}</ref>

== 定義 ==
在日本，「時裝大樓」並非與[[百貨公司|百貨公司]]或[[超級市場|超級市場]]等為正式的業界團體，而是針對百貨公司等各種業態本身的一種學術研究之對象。雖然《[[大辭泉|大辭泉]]》<ref>《[[大辭泉|大辭泉]]》中的解釋為「fashion＋building、流行に 合ったブティック、レストランなど…」</ref>等許多字典中亦有收錄時裝大樓一詞，但目前在學術上並無確切統一的定義。

包括《日本經濟新聞》<ref>{{cite news|quote=百貨店との取引を主とするアパレル各社が新たに、ファッションビルの販路を強化する。|title=アパレル大手、百貨店以外も開拓　価格抑え新ブランド |work=[[日本經濟新聞|日本経済新聞]] |date=2012-02-03}}</ref>、《朝日新聞》<ref>{{cite news|quote=ファッションビル大手パルコ… |title=パルコ渋谷店「イメージ変えたい」　新ビル対抗、建て替えも |work=[[朝日新聞|朝日新聞]] |date=2011-06-21}}</ref>、《讀賣新聞》<ref name="読売新聞"/>等在內的許多主要媒體經常在報導中使用時裝大樓一詞<ref>{{cite news|quote=…有楽町マリオンにファッションビル「ルミネ有楽町店」が完成… |title=＜ルミネ有楽町店＞5000人が列 銀座の玄関口にオープン |work=[[毎日新聞|毎日新聞]] |date=2011-10-28}}</ref><ref>{{cite news|quote=…百貨店圧倒のファッションビル… |title=【変わる大阪駅】（4）脱・駅ビル |work=[[產經新聞|産経新聞]] |date=2010-11-16}}</ref>，在年輕女性之間<ref name="読売新聞2">{{cite news|quote=ラフォーレが存続しない場合、跡地に新しいファッションビルを希望するか」 との設問には、「強く希望する」「希望する」が、男子高生が48%だったのに対し女子高生が79%と、 特に女性でファッションビルを求める傾向が強かったという。 |title=ラフォーレ 「存続を」女子高生86% 東雲短大生調査 |work=読売新聞 愛媛県版 |publisher=読売新聞 |date=2008-03-07}}</ref>、百貨公司<ref>{{cite news|quote=この地区は百貨店やファッションビル、地下街などさまざまな商業施設が集積し… |title=三越伊勢丹グループ採用情報 |publisher=[[三越伊勢丹控股|三越伊勢丹ホールディングス]] |date=2012-02-03}}</ref>、[[服装|服装]]<ref>{{cite news|quote=婦人のヤング向け新ブランド「スマッキーグラム」新販路となるファッションビルに来春デビュー |title=Press Release |publisher=株式会社三陽商会 |date=2004-11-04}}</ref>和購物中心<ref>{{cite news|quote=主な形態としては…ファッションビルや駅ビル、地下街、アウトレットモール、パワーセンターなど… |title=ショッピングセンターとは？ |publisher=社団法人日本ショッピングセンター協会}}</ref>等業者間也是常用的名詞。日本的[[中小企業廳|中小企業廳]]也使用時裝大樓來描述此類中代表性的大型店鋪<ref>{{cite news|quote=代表的な大型店 百貨店／ファッションビル／駅ビル GMS（総合量販店）スーパーマーケット |title=平成15年度　実践行動マニュアル -なぜ人が街から消えていくのか-消費者にとって魅力あるまちづくり |publisher=中小企業庁}}</ref>。因此時裝大樓已成為日本常見的一般用語。

== 概要 ==
在大都市中，時裝大樓常有著培養年輕[[設計師|設計師]]品牌店面的功能，也能作為新流行的發源地地之一<ref>{{cite news|title=ラフォーレ原宿30周年　若手育て独自文化を発信 |work=[[朝日新聞|朝日新聞]] |date=2008-09-02}}</ref>。此外，由於時裝大樓較重視流行和話題性<ref>{{cite news|quote=「日本初、エリア初、新業態など、常に話題の中心になります」とルミネの出店要請に記述されている。 |title=ニュースの理由 |work=日本経済新聞 夕刊 |publisher=[[日本經濟新聞|日本経済新聞]] |date=2010-11-11}}</ref>，因此大多主要目標客群為對流行敏感度較高的年輕人<ref>{{cite web|quote=ファッションビルは集客力が高く、一般に都市部にあり、ファッション感度が高い。 |title=対日投資のための業種別産業調査（小売流通）調査報告書 |publisher=[[日本貿易振興機構|日本貿易振興機構]]（ジェトロ）対日投資部 |date=2005-03}}</ref><ref>{{cite news|quote=20代女性が等身大の自分を楽しめる場所にしたい。 |title=花崎淑夫ルミネ会長がインタビュー |publisher=日経流通新聞 MJ |date=2010-11-01}}</ref>，實際上時裝大樓的消費者也多為年輕族群<ref name="読売新聞2"/>。

代表性的時裝大樓[[LUMINE|LUMINE]]（ルミネ）中大多數店鋪的商品價格在一萬日圓以下<ref>{{cite news|quote=…多くのテナントの商品が1万円を切る |title=ニュースの理由 |work=[[日本經濟新聞|日本経済新聞]] |date=2010-11-11}}</ref>，與[[百貨公司|百貨公司]]相比價格較低<ref>{{cite news|quote=百貨店に比べると手頃な価格の商品が充実 |title=ファッションビルとは |work=読売新聞 大阪夕刊 |publisher=[[讀賣新聞|読売新聞]] |date=2009-05-28}}</ref>，因此價格也成為時裝大樓與之競爭成長的武器。許多百貨公司系統的[[批發|批發]]業者也使用時裝大樓為品牌開設新的店面<ref>{{cite news|quote=ファッションビルの販路を強化する。…顧客層が百貨店より若いことから、…価格を抑えた新ブランドを投入 |title=アパレル大手、百貨店以外も開拓　価格抑え新ブランド |work=[[日本經濟新聞|日本経済新聞]] |date=2012-02-03}}</ref><ref>{{cite news|quote=婦人のヤング向け新ブランド「スマッキーグラム」新販路となるファッションビルに来春デビュー |title=Press Release |work=株式会社三陽商会 |date=2004-11-04}}</ref>。

時裝大樓的所在地基本上為市中心的商業鬧區<ref>{{cite news|quote=…中心部への立地が多い |title=ファッションビルとは |work=読売新聞 大阪夕刊 |publisher=[[讀賣新聞|読売新聞]] |date=2009-05-28}}</ref>，而[[JR|日本鐵道集團]]（JR）各分公司旗下的車站也有許多轉型為時裝大樓<ref>{{cite news|quote=「駅ビルは商業施設としての館の統一感が取れておらず「ダサイ場所」に近かった…」が「…商品が1万円を切る品揃えの中で顧客吸引力と販売力を蓄積…」してファッションビル化した |title=ニュースの理由 |work=日本経済新聞 夕刊 |publisher=[[日本經濟新聞|日本経済新聞]] |date=2010-11-11}}</ref>，這些車站多位於郊外的地方都市，而店鋪構成與主要客層皆與市中心之時裝大樓相差無幾<ref>{{cite news|quote=「都心型と郊外型に分け、郊外型では得意の高感度ファッションに頼らない売り方にカジを切る。･･･浦和店（さいたま市）に･･･家電量販店のノジマ･･･アウトドア用品「好日山荘」や手芸用品「ユザワヤ」を導入･･･パルコの中心顧客層ではない40代以上が増えた」「仙台店（仙台市）は高感度ファッションをそろえすぎ･･･苦戦が続いた･･･生活雑貨系のテナントを入れて雰囲気を変えたところ、今年7月は売り上げが前年同月の1.2倍に増えた。」 |title=パルコ、異業種連携、郊外、高感度ファッション頼らず |publisher=日経流通新聞 MJ |date=2011-08-24}}</ref>。
