《'''時鐘機關之星'''》（{{lang-ja|クロックワーク・プラネット}}）是一部由[[榎宮祐|榎宮祐]]及[[暇奈椿|暇奈椿]]共同創作的[[輕小說|輕小說]]，插畫由[[茨乃|茨乃]]負責。小說由[[講談社|講談社]]的[[講談社輕小說文庫|講談社輕小說文庫]]出版<ref>{{Cite web|url=http://kc.kodansha.co.jp/magazine/index.php/90008|title=講談社輕小說文庫既刊案内-2013年4月發行|publisher=講談社|accessdate=2013年10月31日|language=ja}}</ref>，漫畫版於講談社《[[月刊少年天狼星|月刊少年天狼星]]》連載。

== 概要 ==
本作由兩位作者「合作」而成，形式非常罕見。本作的創作靈感是來自機械時鐘，起因是榎宮祐老師在海外旅行時，偶然在免稅店看見了精美的機械時鐘，便構思了世界基礎的架構並和暇奈椿老師各寫一組主角發展劇情，完成了這個特殊的作品。<ref>{{Cite web|url=http://www.tongli.com.tw/TNews_View.aspx?TID=20140127173036|title=《Clockwork Planet 時鐘機關之星》第1集 轉動命運！|publisher=東立出版社|accessdate=2015年12月27日}}</ref>

本作是其中一位作者暇奈椿的出道作<ref>記述於小說第一卷的後記（暇奈）。</ref>。

2015年12月宣布動畫化<ref>{{Cite web|url=http://lanove.kodansha.co.jp/official/clopla/|title=官方網站|publisher=講談社|accessdate=2015年12月27日|language=ja}}</ref>，並於2017年4月起播放<ref>{{Cite web |url = http://www.animatetimes.com/news/details.php?id=1482394767 |title = 南條愛乃さん・加隈亜衣さんら『クロックワーク・プラネット』出演声優が判明！　放送時期やスタッフ情報も解禁 |work = animate Times |date = 2016-12-22 |language = ja |accessdate = 2016-12-22}}</ref>。

== 故事簡介 ==
{{more plot}}
——說來突兀，不過世界早就滅亡了。死亡後的地球，變成一切都是用齒輪與發條加以重現、再度建構的世界——「時鐘機關之星」。

'''見浦直人'''是個成績吊車尾的高中生，某一天突然有個黑色箱子墜落在他家裡，箱子裡頭是一具自動人偶少女。少女:「不過是一處故障，就被迫停止運轉兩百年，人類的智能恐怕還沒超越跳蚤的水準吧——？」

在崩毀與殘存之間來回反覆，被改造的世界與不變的人類，當理想與現實產生劇烈衝突之時，直人與機械少女琉紫，以及天才少女瑪莉與保鑣哈爾達的兩場相遇推動了命運的齒輪！<ref>{{Cite web|url=http://home.gamer.com.tw/creationDetail.php?sn=2265256 |title=【心得】一切以齒輪推動的世界———時鐘機關之星 |publisher=巴哈姆特電玩資訊站 |accessdate=2013年12月6日}}</ref><ref>{{Cite web|url=http://www.tongli.com.tw/webpages/book/NO0011001/NO0011001.htm |title=Clockwork Planet 時鐘機關之星》轉動命運的齒輪！宣傳網頁 |publisher=東立出版社 |accessdate=2015年7月30日}}</ref>

== 用語 ==
=== 時鐘機關之星相關 ===
; {{nihongo|時鐘機關之星|時計仕掛けの惑星（クロックワーク・プラネット）}}
: 本作故事的舞台。由「Y」一個人所提出的構想，「Y」帶著龐大到誰都無法理解的資料，向世界宣示由無數齒輪所構成的設計圖。
: 原本[[地球|地球]]仍是一顆活著的星球，但在1000年前地球突然滅亡，科學家們最後得出了「壽終正寢」的結論。其後地球慢慢地停止活動，在大約100年之間成為死亡星球。
: 現時的風、氣溫、重力等等都是由齒輪控制再現的。
: 在時鐘機關之星，城市内部有用發條裝置發動的列車、公車和無人計程車運行。但城市的齒輪之間須要保持旋轉，因此不存在經常連接的道路和鐵路。在城市之間移動的方法有很大的限制。城市之間的主要移動方法有乘搭飛機或使用「圓筒鐵路」。

; {{nihongo|圓筒鐵路|環状鉄道（リング・レイル）}}
: 連接都市之間的鐵路。都市建立在直徑從幾公里到幾十公里不等的巨大齒輪上，巨大齒輪的每一齒上都步滿了無數的小洞，每一齒的直徑約為十公尺，當兩邊的洞對齊的瞬間，會將載有乘客和貨物的巨大圓筒射出。利用圓筒鐵路人們在一天之內會有數次的瞬間大量輸送，每當圓筒鐵路抵達，總站通常都會有大批人潮。
: 發射時產生的噪音，有如數千門大砲同時發射般發出刺耳尖銳的爆炸聲。雖然旅客用圓筒鐵路內部都設有隔音構造。但對於直人來說還是會造成不適。

; {{nihongo|電磁技術|電磁技術}}
: 時鐘機關之星建造之前就有的技術。在故事中有法例監管，嚴禁使用及研究。但實際上不少國家有私底下製造[[電磁武器|電磁武器]]，作為保護國家或攻擊他國的最終手段。
: 電磁技術之所以被禁，是因為電磁技術的電磁波會令時鐘機關之星的零件[[磁化|磁化]]，磁化後的零件會變成磁鐵，失去原來的功用。
: 磁化的物件可以用高溫加熱消磁。Initial-Y系列的自動人偶會在磁化後自動加熱消磁，但受熱能力不高的機體可能有小部分的損傷。　
: 其他的消磁方法有強烈撞擊和強的變化電流。強烈撞擊有機會破壞零件，電也被法例監管，所以兩者在時鐘機關之星皆不是好的消磁方法。

=== 鐘錶技師相關 ===
; {{nihongo|時鐘技師|時計技師}} <ref group="注">台版小說翻譯為鐘錶技師</ref>
: 在本作的世界中僅有2億人。最初所有人都是從「{{nihongo|3級|レアリング}}」開始，在現場的洗禮中升格為「{{nihongo|2級|ゲゼル}}」。「{{nihongo|1級|マイスター}}」是千錘百煉的的技術者，需要將才能和經驗兩方面納入手中才達到。
: 1級時鐘技師因為熟知各種機械身體和自動人偶的結構，可以用手邊的螺絲起子讓全身義體士兵機能癱瘓，及用0.2125秒入侵自動人偶的射控裝置搶奪控制權。1級時鐘技師對上它們時有絕對有優勢。

; {{nihongo|無國境技師團|国境なき技師団}}
: 瑪麗所屬的國際組織，目的是為了保護和維護星球的運行機制。世上僅有的6305名一級時鐘技師中超過半數的人都在其中，在世界各地排除齒輪故障。
: 本身一個非營利性組織，但是有著五大企業作為後援資助。

; {{nihongo|學院|アカデミー}}
: 無國境技師團經營的時鐘技師專業學校，可以說是通往第一級時鐘技師登龍門的專業學院。通常必須要有二級的資質才有資格入學，但若有兩名現役的第一級時鐘技師推薦，就可以作為特待生入學。

; {{nihongo|連接設計師|}}
: 連接設計師能夠把本來不同規格的零件組合，統一規格的自動人偶不能隨意更換內在的零件，需要考慮功率，強度，性能甚至性質的不同，才可把不同規格的零件組合，是一項十分細膩和困難的工作。
: 他們擅長對自動人偶客製化的工作，和對舊自動人偶的維修。一般城市對這方面的需求不高，但對於充滿犯罪者的國家，他們需要各種改造品或者是模仿品。這些國家成為了連接設計師大展拳腳的地方。
: 熟練的連接設計師熟知自動人偶骨骼的結構，只用眼看就能知道自動人偶骨骼結構上那裡出現問題。另外，骨骼的結構不會發出聲音，直人的聽力也無法分辨人偶骨骼結構上的問題。
: 一般的時鐘技師認為，學習組合不相容的零件，不如學習從頭設計，而不會學習有關連接設計師的技術。連1級時鐘技師也只有少數人了解連接設計師的工作，因此頂尖的連接設計師少之有少。
