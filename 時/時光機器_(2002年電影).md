{{作品名稱|name=''The Time Machine''|hk=時光凶間|tw=時光機器|equiv-cn=tw}}是一部[[2002年電影|2002年]]3月8日上映的[[好莱坞|好莱坞]]电影，亦是一部[[美國|美國]][[科幻電影|科幻電影]]，內容改編自[[赫伯特·乔治·威尔斯|H·G·威尔斯]]1895年的同名小說[[時間機器_(小說)|時間機器]]，[[導演|導演]]是[[賽門·威爾斯|賽門·威爾斯]]（[[:en:Simon_Wells|Simon Wells]]），[[製片|製片]]為[[華特·帕克斯|華特·帕克斯]]（[[:en:Walter_Parkes|Walter Parkes]]）和[[大衛·維戴斯|大衛·維戴斯]]，[[編劇|編劇]]為[[約翰·羅根|約翰·羅根]]（[[:en:John_Logan_(writer)|John Logan]]）。

2002年改編後的電影被設定在[[紐約市|紐約市]]，而不是原著小說的[[倫敦|倫敦]]，並包含原著[[小說|小說]]中不存在的元素，包括一個[[浪漫|浪漫]]的故事背景，文明毀滅的新的場景，以及一種新的[[人造語言|人造語言]]。

== 劇情 ==
在1899年，亞歷山大哈迪根博士（[[蓋·皮爾斯|蓋·皮爾斯]]飾）是一個年輕的[[發明家|發明家]]，在[[紐約|紐約]][[哥倫比亞大學|哥倫比亞大學]]教書。不像他的保守的朋友大衛菲比（馬克阿迪），亞歷山大寧願做純理論的研究工作，而不去商業界。他的心上人艾瑪被一個強盜殺害，因此他專心於建設一個[[時間機器|時間機器]]，為了救她。當四年後[[時光機器|機器]]完成，他返回到1899年，但仍然無法阻止她被謀殺，只看到她在一輛無馬的[[馬車|馬車]]中喪生。

為了解開艾瑪無法避免死亡的原因，亞歷山大前進到2030年，找出她的生命是否可以挽回。在紐約公共圖書館，圖書館電腦 VOX-114堅持告訴他，[[時間旅行|時間旅行]]是不可能的，因此亞歷山大放棄，繼續走向[[未來|未來]]，希望找到解答，但到2037年，人類在月球的殖民者更改了月球的軌道，意外導致[[月球|月球]]的解體，重力混亂讓世界呈現出幾乎無法居住的狀態。他趕緊重新啟動時間機器，以躲避被爆炸墜落的碎片，但爆炸震波把他擊昏的同時，他無意識地向前撥動了機器，醒來時機器已經停止在西元802,701年。

在這個時候，[[人類|人類]]已經恢復到原始的生活方式。一些倖存者，被稱為“{{tsl|en|Eloi|埃洛依}}”，在曾經是[[曼哈頓|曼哈頓]]的懸崖兩側居住（該懸崖是在在80萬年間由[[哈德遜河|哈德遜河]]切割為[[峽谷|峽谷]]而形成）。亞歷山大被瑪拉所拯救，她是會講少數英語的幾個{{tsl|en|Eloi|埃洛依}}女人之一。某天晚上，亞歷山大和Mara的年輕兄弟，卡倫，夢到一個可怕的，鋸齒狀的人形怪物，第二天{{tsl|en|Eloi|埃洛依}}就被這些怪物攻擊。瑪拉被稱為“{{tsl|en|Morlock|莫洛克}}”的類人猿的怪物拖到地下，充當食品。為了搶救她，卡倫亞歷山大試圖重新啟動圖書館電腦VOX 114，並詢問出80萬年來地球所發生的事。

亞歷山大獲知{{tsl|en|Morlock|莫洛克}}，通過開放類似於他之前的噩夢進入他們的地下境界，但他被捕獲並扔進瑪拉坐在關在籠子裡的一個地方。在那裡，他遇見一個最聰明的，超級{{tsl|en|Morlock|莫洛克}}，他解釋說，{{tsl|en|Morlock|莫洛克}}在人類社會解體後遁入地下，而{{tsl|en|Eloi|埃洛依}}是那些留在地表上仍然演變的[[人類|人類]]進化的後裔。超級{{tsl|en|Morlock|莫洛克}}是一個階級，是統治{{tsl|en|Eloi|埃洛依}}獵物的怪物。超級{{tsl|en|Morlock|莫洛克}}解釋說，亞歷山大不能改變艾瑪的命運，因為她的死亡，是驅使他把時間機器建設擺在首位的動機：如果艾瑪沒有死，他就不會建造出時光機器，整件事就不會發生，因此救了她，將創建一個[[時間悖論|時間的悖論]]。

亞歷山大在超級莫洛克的同意下獲取重新進入時光機內，他對超級莫洛克說「你忘了一件事....如果...」，然後突然把超級莫洛克拉入機器內，並在機器快速往前轉的同時，在激烈搏鬥中，把超級莫洛克甩出座位。莫洛克手攀座位內的部分肢體雖然沒有老化，但機器外的部分卻迅速死亡。亞歷山大然後把機器停止在西元6,3542,7810年，所有的埃洛伊都被莫洛克統治，地球只剩下一片荒涼，眼前揭示了一個殘酷的，鐵鏽色的天空。

最後，他看到眼前的景象，接受不能拯救艾瑪的結果，亞歷山大轉而再度時光旅行，希望救援瑪拉。釋放她後，他開始破壞時間機器的齒輪，把1899年帶來的懷錶塞進機器。亞歷山大和瑪拉用時間機器累積的能量，作出一個巨大的爆炸，殺死所有{{tsl|en|Morlock|莫洛克}}，並破壞了他們的的洞穴。亞歷山大與瑪拉等埃洛伊人，在西元802,701年，開始了新生活。

影片結束時候螢幕顯示同一位置，不同時間的兩個場景：亞歷山大與瑪拉和卡倫在一個地方快樂生活，那地方曾經是他紐約市的家。菲比和亞歷山大的管家，Watchit夫人，則在1899年的同一個地點，悲傷的討論他的離開。臨行前，亞歷山大的[[實驗室|實驗室]]，Watchit太太回頭看了最後一眼，並自言自語的說，“一路順風，我的好小伙
。”

== 製作 ==
[[夢工廠|夢工廠]]及[[華納兄弟|華納兄弟]]與擁有1960年[[喬治·帕爾|喬治·帕爾]]（George Pal）《時間機器》版權的{{link-en|阿諾德·萊柏維茲|Arnold Leibovit}}娛樂（Arnold Leibovit Entertainment）協商，使得華納兄弟公司和夢工廠能聯手製作這部電影。

=== 特效 ===
莫洛克是利用穿著電子動物面具服飾的演員演出。而他們四肢跑得比力所能及更快的場面是用[[光影魔幻工業|光影魔幻工業]]創造的生物[[CGI|CGI]]來製作的<ref name="Robertson">{{cite web|url=http://www.cgw.com/Publications/CGW/2002/Volume-25-Issue-3-March-2002-/About-Time.aspx |title=About Time | Computer Graphics World |publisher=Cgw.com |date=2002-03-03 |accessdate=2015-10-31}}</ref>。

大部分時光旅行的場景完全是電腦生成的，包括時光機在車庫時的那33秒[[時間旅行|穿越時空]]場景。隨著鏡頭，可以看到從紐約市開始，然後飛到太空，經過國際太空站，然後鏡頭隨著太空梭降落在月球揭示地球未來的月球殖民地。房屋和大樓有如雨後春筍般不斷從地面建起。之後的時間旅行畫面，特效團隊用了地質侵蝕模擬的演算法，來製造出過了好幾個世紀地球的變化<ref name="Robertson"/>。

時光機器運作時周圍的藍色氣泡狀時間扭曲場的效果是用ILM開發的名為rgbe的增程型顏色格式（red、green、blue、exponent channel）來製作的<ref>Paul E. Debevec and [[Jitendra_Malik|Jitendra Malik]], "Recovering High Dynamic Range Radiance Maps from Photographs, Siggraph Proceedings, 1997</ref><ref name="Robertson"/>。

在月球解體時[[防空警報|戒嚴時的警報]]音效也用在《[[班傑明的奇幻旅程|班傑明的奇幻旅程]]》中颶風卡崔娜的警報聲。

=== 原聲帶 ===
完整的{{link-en|時光機器 (原聲帶)|The Time Machine (soundtrack)|原聲帶}}是由[[克劳斯·巴德尔特|克劳斯·巴德尔特]]創作，以《我不屬於這裡》為主題，後來在2008年被[[探索頻道|探索頻道]]用在迷你系列《當我們離開地球》。

== 人物 ==
;亞歷山大哈迪根博士（1869-(1903<ref group="註">劇中圖書館電腦的紀錄，正是他穿越時空時的年份</ref>)-802701-）
:由[[蓋·皮爾斯|蓋·皮爾斯]]演，是哥倫比亞大學應用力學系與工程學系的副教授。照顧他的人是管家瓦琪太太和專利業務員愛因斯坦先生出手相助。
;艾瑪
:由[[西耶娜·盖尔利|西耶娜·盖尔利]]主演，是亞歷山大哈迪根博士的女朋友兼未婚妻，艾瑪死於1899年1月18日，是被一位強盜射死，但在由時光機產生的另外一個平行宇宙中艾瑪是死於車禍。
;瑪拉
:由[[Samantha_Mumba|Samantha Mumba]]主演，是一位年輕的埃洛伊人，會說英文，幫在時光機上昏迷的亞歷山大恢復健康，也是亞歷山大繼艾瑪之後喜歡的人。
;大衛菲比
:由[[馬克阿迪|馬克阿迪]]主演，亞歷山大的好朋友和保守的同事。
;花店老闆
:由Alan Young主演

== 事物 ==
;太陽能風車
:只出現在刪減片段中，亞歷山大哈迪根教授使用這個風車來進行相關課程
;時光機器
:時光機器與1960年電影的時光機器不相同，有一個反應爐、承軸、電池、旋轉葉片、顯示日期的時鐘
;圖書館電腦 VOX 114
;月球殖民計畫
;月球大爆炸
:由於月球殖民計畫出錯將月球炸離軌道，使月球往地球靠近，引潮力增加，月球解體同時也粉碎了地殼，造成地殼變動，被超級莫洛克稱為月球大爆炸
;埃洛伊語
:埃洛伊人溝通用的語言
;莫洛克語
:已經退化成使用叫聲來溝通
;石碑語
:即英語，由於月球爆炸後地表嚴重毀滅，只剩下一些寫著英語的石碑，被埃洛伊人稱為石碑語
