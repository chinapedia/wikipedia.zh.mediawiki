== 遊戲設定 ==
=== 琳帕姆 ===
: 被四個異世界包圍，分別為「機界雷拉爾」、「鬼妖界希爾坦」、「靈界薩普勒斯」、「幻獸界梅特爾巴」，還有幾個「沒有名稱的世界」，我們存在的現實世界也是其中之一。
: 擁有豐富的瑪那，四個異世界屢次以琳帕姆為作侵略目標，因此用來對抗侵略的「送還術」及「召喚術」相當發達，被稱為「艾爾哥之王」的英雄利用結界，半恆久性抵抗來自異世界的侵佔，後來以「艾爾哥之王」為中心成立琳帕姆，王都為塞拉姆。
: 「艾爾哥之王」死後，權力鬥爭使國家分裂為大陸中央的「聖王國」，北方的「舊王國」，西方的「帝國」。
: 召喚師分為「蒼之派閥」和「金之派閥」。

=== 召喚術 ===
: 琳帕姆非常發達的特殊魔法。原本是用來把入侵琳帕姆的侵略者送回原本的世界的送還術，現在把技術逆向轉用，從異世界召喚對象。
: 基本原理是以被稱為「召喚夜之石」的特殊礦石輸入瑪那，開拓連接異世界的道路，并說出對象的真名作為誓約。
: 召喚夜之石有五種顏色，分別為黑色、紅色、紫色、綠色、無色，因應異世界使用相對的顏色。
: 召喚術的使用者被稱為召喚師，從異世界召喚出來的召喚對象也被稱為召喚獸。不送還回原本的世界，留下作為召喚師護衛的被稱為護衛獸。
: 召喚師的家名是根據出身和實力。代表性的有「蒼之派閥」、「金之派閥」和「無色派閥」，也有不屬於特定派閥的召喚師。

=== 舊王國 ===
: 軍人所創立的軍事國家，貫徹極權主義，與外間隔絕，好幾次爆發內亂，現正走向衰亡的國家。
: 以打倒聖王國，讓王國復興作為目標。

=== 帝國 ===
: 帝都為烏魯哥拉，由繼承了「艾爾哥之王」血脈的皇帝所治理的國家，是人們為了反抗舊王國的封建制度而創立的新興國家，。
: 召喚師沒有區分派系，由軍隊嚴格管理。

=== 機界雷拉爾 ===
: 稱為「機械士兵」的[[機器人|機器人]]所生活的世界，機械技術及信息學發達，擁有高機械技術水平。
: 原本有生物居住，因為人類引發機界大戰，讓世界變得不適合生物居住。
: 居住於這個世界的是結合人體與機械的人類，機界大戰中存活的人類躲到地下避難所，重複利用「コールドスリープ」延長壽命。

=== 鬼妖界希爾坦 ===
: [[龍|龍]]、[[鬼|鬼]]和[[妖怪|妖怪]]所生活的世界，是琳帕姆以外唯一有人類居住的異世界，也有很多人由希爾坦移居到琳帕姆。
: 遵守著「野地是人類的領土，山林是妖怪的領土」，兩者很少發生問題，如果產生了磨擦，龍神和鬼神就會委派被稱為「道之人」的宮司和巫女作為調停。
: 有被稱為鬼道、龍道的陰陽術。

=== 靈界薩普勒斯 ===
: [[幽靈|幽靈]]、[[惡魔|惡魔]]和[[天使|天使]]所生活的世界。
: 封印為琳帕姆帶來了最大危機的「大惡魔」馬爾基多斯的「豐收天使」亞爾米尼在這個世界誕生。
: 這個世界的居民沒有實體，是以召喚師的瑪那構成自己的肉體，因消耗非常多的瑪那而無法長時間召喚。
: 擁有被稱為「奇蹟魔法」的法術。

=== 幻獸界梅特爾巴 ===
: 各種各樣的幻獸和[[亞人|亞人]]所生活的綠色世界，通常以同種族一起生活，分成幾個不同部族。
: 他們多數被召喚作為勞動力，有些部族擅長除魔的詛咒。
