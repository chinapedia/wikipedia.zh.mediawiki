《'''叛逃'''》（{{lang-en|'''Ruse of Engagement'''}}，前名《'''ATF反恐'''》，於巡禮時稱為《'''反恐特勤隊'''》），[[香港|香港]][[電視廣播有限公司|電視廣播有限公司]]時裝警匪[[電視劇|電視劇]]，是由[[陳展鵬|陳展鵬]]、[[吳卓羲|吳卓羲]]、[[陳茵媺|陳茵媺]]及[[蒙嘉慧|蒙嘉慧]]領銜主演，並由[[李司棋|李司棋]]、[[關禮傑|關禮傑]]、[[黃德斌|黃德斌]]及[[黎諾懿|黎諾懿]]聯合主演，監製[[王心慰|王心慰]]。本劇為[[2011無綫節目巡禮|2011無綫節目巡禮]]場刊版劇集之一，亦為第17、18屆香港國際影視展中無線電視所推介的劇集之一。

==拍攝期間劇情安排被投訴==
《叛逃》是在2011年5月26日開拍，在試造型時，排名是[[吳卓羲|吳卓羲]]、[[陳展鵬|陳展鵬]]、[[陳茵媺|陳茵媺]]、[[蒙嘉慧|蒙嘉慧]]，但監製[[王心慰|王心慰]]則在感情線上，安排兩大女主角爭奪男二陳展鵬，而不是男一吳卓羲，要他演一名自戀狂幻想者，暗戀陳茵媺，就連徇眾要求的配角[[李亞男|李亞男]]都不安排有感情戲份，使他全劇不得有人喜歡，襯托陳展鵬情聖的威風，因此遭不少投訴。<ref>{{cite web|language =|publisher =成報|title =吳卓羲勾義嫂兄弟相殘 蒙嘉慧返娘家做間諜|url =http://www.singpao.com/NewsArticle.aspx?NewsID=172550&Lang=tc|author =|date =2011年5月27日|accessdate =}}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>報章指吳卓羲憑外購劇《大丫鬟》奪有線視帝和南方電視最佳男演員，足以坐正無線一線男主角之位，但卻被無線冷待屈居做二線，又認為獎項非無線所頒才未受重視。另一方面，三年前，陳展鵬演的劇集，每一套都是配角，如在2011年《[[法證先鋒III|法證先鋒III]]》裡，吳卓羲是男主角，而陳展鵬只是男配角。但在《叛逃》不惜為捧陳展鵬上位，吳卓羲的人氣在他之下，因陳展鵬飾演的角色較為討好，擔演亦忠亦邪帶有神秘及戲劇性的角色，劇中更與陳茵媺和蒙嘉慧三角戀，但無線仍否認此事，更稱吳卓羲為擔正男主角。<ref>{{cite web |language = |publisher = 東方日報|title = 吳卓羲粉絲抱不平 |url = http://orientaldaily.on.cc/cnt/entertainment/20110528/00282_034.html|author = |date = 2011年5月28日 |accessdate = }}</ref>

此外，報章報導在無線官方網討論區有投訴『無線貶有線視帝吳卓羲來捧陳展鵬上位，此造馬不公行為』，但無線為了隱瞞投訴，並將投訴刪除，雖然吳卓羲本人也並不介意，但其支持者卻替他不值。<ref>{{cite web |language = |publisher = 東方日報 |title = 網民投訴貶有線視帝吳卓羲來捧陳展鵬上位|url = http://orientaldaily.on.cc/cnt/entertainment/20110723/00282_022.html 
|author = |date = 2011年7月23日 |accessdate = }}</ref>除此之外，有報章及雜誌指出，陳展鵬突然由配角越數級跳升擔正男主角，是因討好高層及有後台的緣固，所以能在《真相》突然跳升擔正男主角。<ref>{{cite web|language =|publisher =蘋果|title =陳展鵬靠擦掂珍姐火速上位擔|url =http://www.kornhill.com.hk/next/magdetails.php?book=6&issue=215&page=0385056252|author =|date =2011年7月6日|accessdate =}}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref> 也有報章報導指樂易玲為陳展鵬、陳茵媺掃獎鋪路，在《叛逃》加足料狂谷雙陳， 由頭帶到尾，預計他們則是視帝/視后熱門之一，兩人將雙雙於台慶夜掃獎。而結局出來真的如之前報導一樣，兩人的確是頭帶到尾。 <ref>{{cite web|language =|publisher =星島日報|title =樂易玲為陳展鵬及陳茵微掃獎鋪路|url =http://www.ihktv.com/atf-terrorism-chen-zhanpeng-aimee-chan.html|author =|date =2011年9月15日|accessdate =}}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

2014年劇集正式播出時，排名的確改為  [[陳展鵬|陳展鵬]]、[[吳卓羲|吳卓羲]]、[[陳茵媺|陳茵媺]]、[[蒙嘉慧|蒙嘉慧]]，三年時無線否認冷待吳卓羲屈居做二線捧陳展鵬上位，遭投訴一事，並不是無中生有。因此在首集播放時，觀眾已預知陳展鵬戲分是會多過吳卓羲，所以通訊局收到二十宗觀眾投訴，而監製[[王心慰|王心慰]]回應希望觀眾耐心追看，更否認陳展鵬戲分多過吳卓羲。但播畢全劇，陳展鵬戲分確實甚重，多過吳卓羲很多，而且結局更是圍繞陳展鵬及陳茵媺的愛情故事，使到「警匪片」變「愛情片」，講述兩人最終命運，"日嘉走入絕路，有正悲痛"成全套劇的重心點，而吳卓羲在整套劇裡，並沒有什麼戲份可言。但由於開拍時以吳卓羲為擔正男主角宣傳，因此無綫及通訊局在結局週，再次收到觀眾投訴「主角變配角」、「警匪片變愛情片」。監製[[王心慰|王心慰]]有欺騙觀眾收看全套劇集之嫌。

《叛逃》乃是史無前例，不是播出後遭投訴，而是開拍該劇試造型時，就有投訴的劇集。
