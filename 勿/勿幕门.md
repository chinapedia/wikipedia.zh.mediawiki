'''勿幕门'''，原名“井上将门”，是[[西安城墙|西安城墙]]南侧的一座城门，位于[[永宁门|永宁门]]以北，四府街南端。1939年开辟，以纪念国民革命先驱[[井勿幕|井勿幕]]而得名。

== 历史 ==

[[File:Jing_Wumu.jpg|thumb]]]][[卢沟桥事变|卢沟桥事变]]爆发后，民国26年11月（1937）起，日军飞机频繁飞临陕西，轰炸省城西安。

民国28年（1939年），于南四府街（今四府街）南端开辟防空便门，砌单券洞，以方便市民躲避空袭。初曾拟名“东南门”或“震海门”，未果。此后民众长期以“小南门”称之。

民国34年11月（1945年），[[国民政府|国民政府]]追赠国民革命先驱[[井勿幕|井勿幕]]陆军上将衔。同年12月23日，井勿幕遇害27周年之际，[[于右任|于右任]]主持将井勿幕灵柩由蒲城迎至西安，并举行公祭。民国35年2月（1946年），为永久纪念井勿幕，将其故居卢进士巷（今芦荡巷）附近的南北四府街暨琉璃庙街改名“井上将路”，小南门正式命名“井上将门”。民国36年（1947年），改称“勿幕门”，原井上将街亦改名“勿幕街”（今四府街）。<ref>{{cite book |author=西安市地方志办公室 |date=2012年11月第1版| edition= |title=民国西安词典 |url=http://xadfz.xa.gov.cn/difangzhinew/muluFrame.jsp?bookname=minguo_xian_cidian |location=西安 |publisher=陕西人民出版社 |page=59 |isbn=978-7-224-10402-8 |author-link= }}</ref><ref>{{cite web |url=https://mp.weixin.qq.com/s?__biz=MzI0Njc3MDg5NQ==&mid=2247484547&idx=1&sn=117d4da9c0469511042c790db24c5bcd |format= |title= 古城西安十八门 |author=碧海风  |date= 2019-01-31|website= |publisher= 碧海风云|access-date= |quote=}}</ref>

== 地理位置 ==
勿幕门门内为[[四府街|四府街]]，门外为[[红樱路|红樱路]]。
