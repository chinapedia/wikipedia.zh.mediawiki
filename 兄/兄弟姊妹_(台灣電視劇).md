《'''兄弟姊妹'''》是[[台灣電視公司|台灣電視公司]]2004年的[[八點檔|八點檔]][[連續劇|連續劇]]，共136集，製作單位是[[芝山岩工作室有限公司|芝山岩工作室有限公司]]。敘述[[民國60年代|民國60年代]]，一名婦女江美玲與丈夫林志成，與五個小孩子在外表演[[那卡西|那卡西]]的故事。江美玲因故入獄二十年，出獄後，五個小孩子也失蹤。江美玲與義兄曾樂天共同突破萬難，最後也使五個孩子重新團聚。第1-135集（69分鐘），第136集（32分鐘）。

== 故事大綱 ==
民国六十年代，颇具姿色的江美玲与丈夫林志成及兄江乐天组“那卡西”乐团于北投走唱，深受欢迎。<br>

美玲与志成育有三子两女，老大林文吉，学习电子琴，老二林文雄，学习萨克斯风，老三林文山，学习爵士鼓，老四林文美，学习口琴，老五林文丽，学习歌唱。某夜，北投酒场，因将官之子侯永权因贪恋美玲姿色，企图强暴未果，竟失手杀了志成，美玲奋起抵抗，刺杀了永权随扈阿龙，却让另一名随扈阿虎制伏，遭求处无期徒刑，美玲托孤五子予其兄江乐天，然而，一场车祸意外，却让林家五个子女各自分散了二十多年，而林家的次子林文雄，阴错阳差的与阿虎相识，在不知身世的情况下，投身在侯永权把持的黑道组织中，成为永权心腹，绰号「洪声仔」。<br>
出狱后的美玲，巧遇社工王淑珍与其男友郭俊英，在两人的协助下，美玲先遇到了其兄江乐天，这时才知道二十年前的车祸，不仅让儿女们下落不明，也让乐天瞎了双眼，于是，两人决定突破万难，寻回车祸中失踪五个儿女。<br>

侯永权在这二十年间势力扩及黑白两道，其亲信洪声仔，是一个冷酷寡言，思绪深沉、缜密的杀手，专门替永权用黑道的方式，解决一切商场纷扰，此外，洪声仔也与永权之女侯优秀相爱，两人间，有着一份「爱」与「死」的约定，冷面的洪声仔每次出任务前，总会在优秀房中过夜，并在离去前送上一朵玫瑰，患有躁郁症的优秀每隔一个小时，就摘下一朵花瓣，而洪声仔，总是能在花瓣摘完前，回到优秀身边，让优秀替他擦拭嗜血的枪。在这世上，洪声仔与优秀都感觉到，只有彼此才能贴近对方的心，因此，两人的默契，不用言语，也能相通。<br>

美玲在一偶然的机会下，怀疑先进电视台主播唐敏敏为其失散多年的女儿文美，不料，敏敏却执意不肯相认，美玲痛彻心扉，但她仍不放弃希望，终于在淑珍的帮忙下，寻回么女文丽，她原是淑珍爱心协会中所托管的对象，因车祸脑部受创而患有健忘症，但却因此有了预知危险的特殊能力。<br>

敏敏因从事记者的工作，而无意掌握了不利永权的证据，永权欲派洪声仔狙杀之，洪声仔一时心软，下不了手，任务失败，洪声仔返家，准备向永权请罪，永权知敏敏重利轻义、私心可用，遂延揽她到自己的旗下，而敏敏与洪声仔的明争暗斗，也从此开始……

== 轶闻 ==
*劇中還加入了一些時事资讯，例如：侯永權不利的消息傳入曾樂天的耳中，曾樂天感到萬分的高興，就好像得到[[大樂透|大樂透]]六億元一樣的高興。（當時台灣的大樂透一度開出最高獎金[[新台幣|新台幣]]十二億元，造成極轟動的話題之一。）
*而劇中的取景，有時候也在[[台視|台視]]附近的一些工寮或[[人行地下道|人行地下道]]與公車站牌，例如郭俊英與洪聲仔的追逐，就是在[[台視|台視]]附近舊有的人行地下道取景的，而阿不拉悽慘落魄的情景，就是在台視附近的工地取景的。

== 戲外活動 ==
[[中天娛樂台|中天娛樂台]]也曾邀請主要演員參加該台綜藝節目《[[小氣大財神|小氣大財神]]》，並進行例行性的遊戲「激爆骰子樂」與「吹牛大王」等。當天的「營養果汁吧」叫做「兄弟姊妹」，主要是採用[[地瓜葉|地瓜葉]]（「地」與「弟」同音）與大陸妹（台灣俗稱[[A菜|A菜]]為「大陸妹」），與[[韭菜|韭菜]]、[[蒜|蒜]]、[[洋蔥|洋蔥]]打成汁一起喝。
