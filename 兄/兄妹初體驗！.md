《'''兄妹初體驗！'''》（[[日語|日語]]：{{lang|ja|'''兄妹はじめました!'''}}），是[[日本|日本]][[漫畫家|漫畫家]][[愁☆一樹|愁☆一樹]]的一部[[四格漫畫|四格漫畫]]作品。在[[芳文社|芳文社]]的漫畫雜誌《[[Manga_Time_Kirara_MAX|Manga Time Kirara MAX]]》2004年7月號至2009年9月號<ref>{{cite web|url=http://www.dokidokivisual.com/magazine/max/book/index.php?mid=123|script-title=ja:まんがタイムきららMAX 2009年9月号|language=ja|accessdate=2016年4月17日}}</ref>上連載，總共4本[[單行本|單行本]]，已完結。

==劇情簡介==
自小就是[[青梅竹馬|青梅竹馬]]的茜和葵，因為彼此父母的[[再婚|再婚]]，突然成為兄妹。13年過去了，已經成為高中生的茜和葵每天都如兄妹一樣生活在一起，而茜的對葵的心意卻不斷上昇……<ref name="Vol1"/>

==登場角色==
; {{nihongo|高坂茜|高坂 茜（こうさか あかね）}}
: 本作主角之一。高坂家的長子，葵的哥哥。18歲。非常喜歡和疼愛葵，且[[妹控|戀妹情結]]還不斷上昇，雖然葵完全不予理會，但是他為了葵可以做很多事情，是個很有責任感的哥哥。
: 料理社的社員，但是已經完全忘記自己加入了該社，被稱為「幽靈社員」。和綠葉是好友。

; {{nihongo|高坂葵|高坂 葵（こうさか あおい）}}
: 本作主角之一。高坂家的長女，比茜小1歲的妹妹，原姓'''松山'''。17歲。精明能幹可靠，負責打理家中的日常事務，包括財務分配和開支等，對錢財十分精打細算。很關心茜，許多事情都會特意考慮到哥哥，同時也對他有許多要求。
: 頭上左側戴著一個貓咪髮飾，髮飾的表情還會隨著葵的心情而改變。[[貧乳|胸部較小]]。喜歡吃[[螃蟹|螃蟹]]。
: 料理社的社員。和萌蔥是好友。

; {{nihongo|大山萌蔥|大山 萌葱（おおやま もえぎ）}}
: 葵的同班同學，綠葉的妹妹。是個永遠都樂天開朗、天真爛漫的少女，有時還會暴走；是眾人歡樂的源泉。喜歡茜。身材豐滿。
: 料理設的社員。和葵是好友。

; {{nihongo|大山綠葉|大山 緑葉（おおやま りょくよう）}}
: 茜的同班同學，萌蔥的哥哥。與妹妹一樣樂天，但是十分懶散、漫不經心，可以隨時隨地睡著。喜歡葵。
: 和茜一樣是料理社的「幽靈社員」。和茜是好友。

; {{nihongo|若葉弧金|若葉 弧金（わかば こきん）}}
: 葵和萌蔥的同班同學，萌蔥的鄰居和青梅竹馬。性格沉穩，表情一般十分嚴肅，不言茍笑。喜歡萌蔥，對她提出的要求一般都會默然接受，卻總被萌蔥忽略。
: 在嵐山兄妹加入之前是料理社唯一活躍的男社員。

; {{nihongo|朝霞真紅|朝霞 真紅（あさか しんく）}}
: 茜和綠葉的同班同學。身材嬌小，頭上戴著[[獸耳|兔耳]]狀的髮飾。喜歡綠葉。
: 料理社社長，做料理十分了得，對家事也是萬能，而且熱心幫助和照顧他人，是葵和萌蔥等人尊敬的學姊。

; {{nihongo|嵐山真黑|嵐山 真黒（あらしやま しんくろ）}}
: 真白的[[雙胞胎|雙胞胎]]哥哥，在第2卷登場。身材矮小，性格善良好為他人著想。
: 轉學至大家所在的學校就讀，而且加入了料理社。

; {{nihongo|嵐山真白|嵐山 真白（あらしやま ましろ）}}
: 真黑的雙胞胎妹妹，在第2卷登場。身材矮小，平時個性很溫柔，但是對著茜時態度會不友善。
: 轉學至大家所在的學校就讀，而且加入了料理社。
