{{Nihongo|'''兄弟工業株式會社'''|ブラザー工業株式会社|Brother Industries, Ltd.}}（{{Tyo|6448}}）是一家多元化經營的[[日本|日本]]公司，生產各種各樣的產品，包括[[印表機|印表機]]、[[縫紉機|縫紉機]]、大型[[機床|機床]]、[[標籤|標籤]][[打印機|打印機]]和[[打字機|打字機]]、[[傳真機|傳真機]]、[[電腦|電腦]]和其他相關[[電子產品|電子產品]]。<ref>{{cite web |title=Brother Printers — The Multi-Awarded Printer Brand |url=http://www.allprinterreviews.com/brother-printers.html |accessdate=2009-04-24}}</ref>

== 歷史 ==
[[File:Brother_P-Touch_540.jpg|thumb]]]]
=== 成立 ===
[[安井兼吉|安井兼吉]]於1908年建立安井縫紉機公司，專營[[銷售|銷售]]及[[維修|維修]][[縫紉機|縫紉機]]。而安井兼吉正是兄弟工業創辦人的[[父親|父親]]。安井兼吉的大兒子[[安井正義|安井正義]]由九歲起協助父親打理生意，17歲時便到了[[大阪|大阪]]當[[學徒|學徒]]。當時大阪正是日本[[縫紉機|縫紉機]]業發展的中心。安井縫紉機於1928年生產日本自製的縫紉機，這是首台鏈形縫紉機，也是首台以「兄弟牌」行銷的產品。

=== 發展 ===
1941年，兄弟縫紉機銷售有限公司成立， 在[[日本|日本]]開拓銷售網絡。 1947年，在 日本政府要求下，兄弟開始[[出口|出口]]家用[[縫紉機|縫紉機]]到 [[上海|上海]]。1954年建立了兄弟國際企業。1954年於[[紐約|紐約]]建立銷售總部，當時[[美國|美國]]銷售部門希望生產[[手提打字機|手提打字機]]，稱這產品將會越來越受美國人歡迎。後來兄弟研發出來的[[手提打字機|手提打字機]]便出口[[美國|美國]]。直到1971年，兄弟產品20種[[語言|語言]]的[[打字機|打字機]]，並[[行銷|行銷]]全球110個[[國家|國家]]。這使[[打字機|打字機]]成為[[縫紉機|縫紉機]]之外的兄弟[[產品|產品]]。 1958年在[[愛爾頓|愛爾頓]]建立[[歐洲|歐洲]][[生產|生產]][[基地|基地]]。1961年於[[日本|日本]]的總部[[大廈|大廈]]落成，並於1962年由「日本縫紉機株式會社」改名為「兄弟工業株式會社」。1963年，兄弟在[[東京證券交易所|東京證券交易所]]，[[名古屋證券交易所|名古屋證券交易所]]和[[大阪證券交易所|大阪證券交易所]][[股票上市|股票上市]]。1971年，兄弟開發高速[[點陣式打印機|點陣式打印機]]。1987年，[[日本電信電話|日本電信電話]]和兄弟共同開發[[傳真機|傳真機]]。之後，兄弟在[[美國|美國]]的[[傳真機|傳真機]]市場連續七年佔有率排名第一（1994年—2000年）。不過，近年來[[佳能|佳能]]、[[惠普|惠普]]的[[傳真機|傳真機]]漸漸剝奪兄弟的[[傳真機|傳真機]]市場。2003年多功能[[噴墨打印機|噴墨打印機]]My Mio發布。在2010年結合[[傳真機|傳真機]]，且市場佔有率排名第一。

== 軼事 ==
多功能[[噴墨打印機|噴墨打印機]]My Mio的首個系列「Souchikakunin 41」由於[[打印頭|打印頭]]或[[電源|電源]][[電路|電路]]的缺陷而不能[[打印|打印]]，需要返厂維修。<ref>{{cite web |title= 兄弟:常見問題（Q&A）（日文） |url=http://solutions.brother.co.jp/public/faq/faq/000000/000700/000026/faq000726_000.html}}</ref>

兄弟的[[打印機|打印機]]利用[[傳感器|傳感器]]估計[[墨盒|墨盒]]可能是空。「空[[墨盒|墨盒]]」必須更換才可以繼續[[打印|打印]]，即使[[打印|打印]]时不需要用到那个[[顏色|顏色]]的[[墨水|墨水]]。只需用[[絕緣膠帶|絕緣膠帶]]贴在該[[傳感器|傳感器]]中心的位置，[[打印機|打印機]]会以为[[墨盒|墨盒]]已滿。您可以[[打印|打印]]到[[墨水|墨水]]真的用完。
