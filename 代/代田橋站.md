'''代田橋站'''（{{jpn|j=代田橋駅|hg=だいたばしえき|rm=Daitabashi eki|f=y}}）是位於[[日本|日本]][[東京都|東京都]][[世田谷區|世田谷區]][[大原_(世田谷區)|大原]]二丁目，屬於[[京王電鐵|京王電鐵]][[京王線|京王線]]的[[鐵路車站|鐵路車站]]。[[車站編號|車站編號]]是'''KO05'''。

== 年表 ==
* 1913年4月15日，代田橋站作為京王電氣軌道的車站啟用。
* 1944年5月31日，京王與[[東京急行電鐵|東京急行電鐵]]合併後，代田橋站作為京王線的車站運行。
* 1948年6月1日，東急與京王帝都電鐵分拆，代田橋站成為京王的車站。
* 2013年2月22日，以'''KO05'''作為代田橋站的編號。

== 車站構造 ==
側式月台2面2線[[地面車站|地面車站]]，笹塚側為高架。站體在京王八王子側的地下一樓。

月台與閘口、閘口與地上之間設有[[電梯|電梯]]。廁所位於地下一樓付費區內。

== 站名由來 ==
站名來自於[[玉川上水|玉川上水]]暗渠化以前，[[甲州街道|甲州街道]]跨越玉川上水的橋梁名。位置約在車站西北方100公尺左右。

橋名來自於附近地名「代田」（代田村），其由來請參照[[代田_(世田谷區)|代田 (世田谷區)]]。
