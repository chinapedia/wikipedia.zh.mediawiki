'''玛德莲蛋糕'''（{{lang-fr|madeleine}}，{{IPA-fr|mad.lɛn}}，petite madeleine）是一种传统的[[贝壳|贝壳]]形状的小[[蛋糕|蛋糕]]，来自于[[法国|法国]]东北部[[洛林大区|洛林大区]]的两个市镇[[科梅尔西|科梅尔西]]和[[利韋爾丹|利韋爾丹]]。

玛德莲蛋糕是将[[傑諾瓦士蛋糕|热那亚蛋糕]]面糊，放置在[[贝壳|贝壳]]形状的成型模子中烤制而成。其味道较[[海綿蛋糕|海綿蛋糕]]浓郁。传统上是加入细磨的[[坚果|坚果]]，通常是[[杏仁|杏仁]]，或加入[[柠檬|柠檬]]皮丝，使其具有柠檬味道。

== 历史 ==
[[File:Blackberry_madeleines_(13685004513).jpg|thumb]]
玛德莲蛋糕的历史起源是有争议的，新牛津美国词典等资料称玛德莲蛋糕可能得名于一位19世纪的糕点厨师 Madeleine Paulmier，但其他一些文献说 Madeleine Paulmier 是18世纪[[路易十五|路易十五]]的岳父、[[波兰|波兰]]国王、洛林公爵[[斯坦尼斯瓦夫一世|斯坦尼斯瓦夫一世]]的一位女厨师，斯坦尼斯瓦夫一世将此蛋糕由[[洛林|洛林]]推广到[[凡尔赛|凡尔赛]]宫廷以及整个法国<ref>Ava Runge. "Madeleines," in ''La Città Viola,'' Issue 3,2007年2月, 佛罗伦萨国际学校</ref>。《Larousse Gastronomique》提供了关于玛德莲蛋糕的两个互相冲突的版本<ref>{{cite web |url= http://www.thefoodsection.com/foodsection/2004/04/is_my_blog_burn.html |title= Post-Proustian Madeleines |author= Josh Friedland |work= TheFoodSection.com |date= 2004年4月18日 |}}</ref>。

在2006年[[歐洲日|歐洲日]]，[[歐洲聯盟|歐洲聯盟]]輪任主席國[[奥地利|奥地利]]發起的[[歐洲咖啡館|歐洲咖啡館]]活動中，玛德莲蛋糕被选为代表法国的甜食。

== 文学 ==
在法国文学家[[普鲁斯特|普鲁斯特]]著名小说[[追忆似水年华|追忆似水年华]]（如卷一《在斯万家那边》）中，玛德莲蛋糕或称“小玛德莲蛋糕”多次在文中出现。玛德莲蛋糕混合在[[椴树|椴树]]茶中的味道使“叙述者——我”回忆到自己的童年，因而成为青年马塞尔开始努力找回失去的时光并促成写作的[[催化剂|催化剂]]。因此，玛德莲蛋糕也渐渐具有代表[[普鲁斯特|普鲁斯特]]（及其作品）与[[意识流小说|意识流小说]]含义的重要[[符号|符号]]（标示）与象征物。
