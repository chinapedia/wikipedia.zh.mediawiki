'''玛蒂尔达皇后'''（[[拉丁语|拉丁语]]名：{{lang|la|Matilda}}，萨克森语名：{{lang|en|Maud}}或{{lang|en|Maude}}，{{bd|1101年|2月7日|1167年|9月10日|catIdx=M}}，逝世于[[法国|法国]][[鲁昂|鲁昂]]）是[[英格兰|英格兰]]国王[[亨利一世_(英格兰)|亨利一世]]和[[瑪蒂爾達·鄧凱爾德|瑪蒂爾達·鄧凱爾德]]公主的女儿，后来她成为[[神圣罗马帝国|神圣罗马帝国]]皇帝[[亨利五世_(神圣罗马帝国)|亨利五世]]的妻子。亨利五世死后她又嫁给了安茹伯爵[[若弗魯瓦五世_(安茹)|若弗魯瓦五世]]，两人的儿子后来成为英格兰国王[[亨利二世_(英格兰)|亨利二世]]。

== 皇后 ==

玛蒂尔达出生于1101年2月7日，她父亲是英国国王亨利一世，外祖父母是[[苏格兰|苏格兰]]国王[[马尔科姆三世|马尔科姆三世]]和苏格兰的圣玛格丽特。而玛格丽特又是英格兰国王[[刚勇者埃德蒙|刚勇者埃德蒙]]的孙女。

一般认为她是在[[温彻斯特|温彻斯特]]出生的，但最近研究也有人认为她应该是在今天的[[牛津郡|牛津郡]]的一个宫殿裏出生的。玛蒂尔达出生的時候一開始是被命名為Adelaide，之後在她七岁时，被许配给神圣罗马帝国皇帝亨利五世。因此改成和她母親相同的名字——玛蒂尔达。

1111年她被送往神圣罗马帝国接受未來成为亨利五世皇后的婚前训练。1114年1月7日两人在[[沃尔姆斯|沃尔姆斯]]举行盛大婚礼成婚。1116年3月玛蒂尔达与亨利访问[[罗马|罗马]]和[[托斯卡纳|托斯卡纳]]。在亨利外巡的时候玛蒂尔达代替她丈夫统治。

婚後两人之間没有倖存的儿女。亨利死于1125年，玛蒂尔达皇后成为一个没有子女的23岁寡妇。由于玛蒂尔达的弟弟威廉在数年前就已因海难溺亡，她成为英格兰王位唯一合法的继承人。

== 第二次婚姻 ==

玛蒂尔达回到英国，她父亲亨利一世立她为继承人，并安排她的第二次婚姻。1127年她与[[安茹|安茹]]伯爵若弗鲁瓦五世在安茹的[[勒芒|勒芒]]成婚。若弗鲁瓦当时比玛蒂尔达小11岁。若弗鲁瓦使用[[金雀花|金雀花]]作为他的徽章，因此他也被称为“金雀花”，由此他的长子亨利在英格兰建立的王朝被称为[[金雀花王朝|金雀花王朝]]。

两人的婚姻并不美满，因此玛蒂尔达回到她父亲身边。1131年她又返回安茹并与若弗鲁瓦重新修和。两人共有三个儿子，长子亨利出生于1133年3月5日，次子若弗鲁瓦后来成为南特伯爵，出生于1134年，他出生时是难产，玛蒂尔达几乎死于产床上。亨利一世特地跑到安茹来看望他的女儿和外孙，但由于亨利和若弗鲁瓦不和，因此当亨利于1135年12月1日死于[[诺曼底|诺曼底]]时，玛蒂尔达和若弗鲁瓦在安茹。

== 争夺英格兰王位 ==

1135年亨利死后玛蒂尔达本来应该继承王位，但是她的表兄弟[[斯蒂芬_(英格兰)|斯蒂芬]]却打破了他要支持玛蒂尔达的诺言，相反地自己篡取英格兰王位。此后英格兰陷入内战，双方均无法占上风。1139年玛蒂尔达终于获得了足够的军事力量到英格兰向斯蒂芬进行挑战。玛蒂尔达最忠实的支持者是她的同父异母的兄弟[[格洛斯特伯爵罗伯特|格洛斯特伯爵罗伯特]]。

玛蒂尔达最大的胜利是在1141年4月，她的军队击败并俘获了斯蒂芬。斯蒂芬被关押，实际上已经失去了王权。虽然玛蒂尔达此时实际上已经控制了王国，但是她没有将自己设立为女王，而且她的好景不长，11月就因格洛斯特被俘，她不得不释放斯蒂芬作为交换，一年后她的处境就大坏了。她被围困在[[牛津|牛津]]，只有通过在雪地上戴着白色的斗篷她才得以逃到[[华灵富|华灵富]]。1147年她不得不假装为要被埋葬的死尸才得以从[[戴维兹|戴维兹]]逃脱。她的支持者格洛斯特死后她不得不返回[[法国|法国]]。

== 晚年 ==

尽管如此，玛蒂尔达还没有丧失她所有的希望，她的儿子亨利（后来的英王亨利二世）成长为一个有作为的领导人。虽然斯蒂芬在内战中获胜，但是他的统治困难重重。1153年他的儿子死了，而亨利又领军入侵。最后斯蒂芬同意立亨利为他的继承人。

玛蒂尔达晚年退居鲁昂，她在那里有一座自己的宫廷，她试图调停她的长子亨利与次子若弗鲁瓦之间的争吵，但是其结果维持不长。在若弗鲁瓦1158年骤死之前他两次对亨利造反。亨利与他的三弟威廉之间的关系要和谐多了。亨利授予威廉大量英格兰的领地。由于大主教[[托马斯·贝克特|托马斯·贝克特]]反对威廉与他的心上人结婚，威廉逃到鲁昂他母亲的宫廷。玛蒂尔达试图调停威廉与贝克特之间的分歧，但是没有成功。据说威廉于1164年1月因忧郁死于鲁昂。

玛蒂尔达在她的家乡并不受欢迎，英格兰人觉得她太外国化、太傲慢。她会说三种语言：[[法语|法语]]、[[德语|德语]]和[[拉丁语|拉丁语]]。虽然她1141年放弃了加冕为女王的打算，但是她的名字始终出现于她儿子之前。她死于鲁昂并被葬在那里的大教堂中。她的墓碑上的碑文是：“她的生世，婚姻和后代为她带来荣耀：这里沉睡着亨利的女儿、妻子和母亲”。

== 相关作品 ==
=== 历史小说 ===
[[爱丽丝·彼得斯|爱丽丝·彼得斯]]的小说《[[卡德法尔修士|卡德法尔修士]]》的背景就是斯蒂芬与玛蒂尔达的支持者之间的内战。后来拍成电影时[[德里克·雅克比|德里克·雅克比]]演主角。

[[琼·珀莱蒂|琼·珀莱蒂]]也写过一系列历史小说，其中的诺曼三部曲的第三部就是关于斯蒂芬与玛蒂尔达之间的故事。

这场内战也是[[肯·福勒特|肯·福勒特]]的名著《[[圣殿春秋|圣殿春秋]]》中的重要情节。
