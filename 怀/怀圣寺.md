'''怀圣寺'''又称'''光塔寺'''、'''怀圣光塔寺'''、'''龙凤寺'''、'''礼拜寺'''等，位于[[中华人民共和国|中国]][[广州|广州]][[越秀区|越秀区]][[光塔路|光塔路]]56号，始建于[[唐代|唐代]]，是中國现存最早的[[清真寺|清真寺]]，甚至是[[世界|世界]]最早的清真寺之一。'''怀圣寺光塔'''为中华人民共和国[[全国重点文物保护单位|全国重点文物保护单位]]。

== 历史 ==

[[唐高祖|唐高祖]][[武德|武德]]年间（618年－626年），[[伊斯兰教|伊斯兰教]]“封印先知”[[穆罕默德|穆罕默德]]派门徒4人来华传教，其中的[[赛义德·本·阿比·瓦卡斯|赛义德·本·阿比·瓦卡斯]]（古译阿布·宛葛素）于唐[[贞观|贞观]]初年经[[海上丝绸之路|海上丝绸之路]]在广州登陆，开始在中国传教。贞观元年（627年），阿布·宛葛素和侨居广州的阿拉伯穆斯林捐资修建了这座清真寺，为纪念“穆圣”，故取名“怀圣”。怀圣寺于[[元朝|元]][[至正|至正]]三年（1343年）被焚，7年后重建。现存建筑为[[清朝|清]][[康熙|康熙]]三十四年（1695年）重建后的规制。1949年后，怀圣寺进行过三次较大规模的修葺，基本保持了原有建筑风络。

== 建筑特色 ==
怀圣寺坐北向南，占地面积2966[[平方米|平方米]]，整体采用中国传统的对称布局，在主轴线上依次建有三道门、看月楼、礼拜殿和藏经阁。礼拜殿坐西朝东，礼拜时面向圣地麦加（实际朝向约旦境内的[[佩特拉|佩特拉]]，并非麦加），建筑的比例、色彩、装饰均具西亚风格。

=== 光塔 ===
光塔位于寺门西南隅的正式名称“怀圣塔”，因教徒在诵经时常在塔顶用[[阿拉伯语|阿拉伯语]]呼喊“邦卡（呼唤之意）”，也被称为“邦卡楼”。[[粤语|粤语]]里“邦”与“光”谐音，另外塔位于珠江边，唐朝时入夜后塔顶悬灯来为来往的船只导航，故人称“光塔”或“番塔”。

此塔建于唐朝[[贞观|贞观]]年间，具有[[阿拉伯|阿拉伯]]风格，高36米，青砖砌筑，底为圆形，表面涂有灰沙，塔身上开有长方形小孔用来采光。塔内有二螺旋形楼梯绕塔心直通塔顶。塔顶部用砖牙叠砌出线脚，原有金鸡立在上面，可随风旋转以示风向，[[明|明]]初被[[飓风|飓风]]吹落，1934年重修时改砌成尖顶。

== 现状 ==
怀圣寺目前仅对中外[[穆斯林|穆斯林]]开放，并不对社会人士开放。而教徒常在周五和伊斯兰教节日前来聚礼，[[广交会|广交会]]期间许多[[阿拉伯|阿拉伯]]商人前来做礼拜。但其实怀圣寺的门卫和礼品商店的本地穆斯林并不反对非穆斯林入内参观，主要是负责管理此寺的穆斯林家庭不允许随意游览。

怀圣寺同时也是[[广州市伊斯兰教协会|广州市伊斯兰教协会]]所在。
