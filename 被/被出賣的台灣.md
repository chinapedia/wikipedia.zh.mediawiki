《'''被出賣的台灣'''》（{{lang-en|Formosa Betrayed}}），由[[二二八事件|二二八事件]]當時任職於[[美國駐台北領事館|美國駐台北領事館]]副領事的[[柯喬治|柯喬治]]（George Kerr）所著作。敘述年代自1941年日本發動[[太平洋戰爭|太平洋戰爭]]，至1965年出書前夕。以1945年[[台灣戰後時期|台灣戰後時期]]至[[二二八事件|二二八事件]]的發生前後為全書描述的重點。書中直接描述[[國民政府|國民政府]]在台灣的種種貪腐暴政、台灣人民遭受的苦難，以及二二八事件中台灣人民遭國府軍隊[[屠殺|屠殺]]的情況。這本書早年在海外的影響既深且廣，在1970年代許多台灣[[留學生|留學生]]在海外讀過後，內心受到極大衝擊，許多人因此紛紛投入[[台灣獨立運動|台灣獨立運動]]<ref>[http://www.appledaily.com.tw/realtimenews/article/new/20140227/351996/ 《被出賣的台灣》發表，中譯者啜泣表痛心]，吳家翔／台北報導，蘋果日報，2014-02-27日</ref><ref>[http://news.ltn.com.tw/news/focus/paper/816832 小檔案︰被出賣的台灣]，記者曾韋禎，自由時報，2014-09-27</ref>。

== 出版 ==
《被出賣的台灣》英文版於1965年出版<ref>[http://www.appledaily.com.tw/realtimenews/article/new/20140227/351996/ 《被出賣的台灣》發表　中譯者啜泣表痛心]，吳家翔／台北報導，蘋果日報，2014-02-27</ref>。[[前衛出版社|前衛出版社]]社長[[林文欽|林文欽]]表示，此書出版時，[[國民黨|國民黨]]非常緊張，並出動人員到美國將此書全部收走，甚至向美國[[出版社|出版社]]施壓，不准再出版。<ref>[http://www.peoplenews.tw/news/5419fbef-ffe9-4719-a143-ed43bdf6328a 公民，從閱讀中覺醒　《被出賣的台灣》徵文]，記者林冠妙／台北報導，民報，2014-10-10</ref>

== 漢譯本 ==
《被出賣的台灣》於1974年由[[陳榮成|陳榮成]]等譯成中文在海外流傳<ref>[http://news.ltn.com.tw/news/focus/paper/758102 「被出賣的台灣」重譯出版 - 焦點 - 自由時報電子報] 之前的中文版是陳榮成在一九七四年找了幾位留學生幫忙翻譯，是葛超智以其個人經驗及中文資料寫成英文，再翻成中文</ref><ref>[https://www.youtube.com/playlist?list=PLzRVoj6_5W4vQJwlWPX1yJOo-BTayMQny 《Formosa Betrayed被出賣的台灣》重譯校註本 新書發表會 - YouTube]</ref>。
漢譯本最早是日本「玉山書社」出版。而[[臺灣|臺灣]]最早的漢譯本是深耕（伸根）雜誌特刊本（底本應是日本漢譯本），後來才由該雜誌印成書本銷售。深耕本是無授權盜印本，後來才由前衛取得授權印行，刷了好幾刷。

2014年，[[台灣教授協會|台灣教授協會]]認為隨著時代演進，對當時的歷史有更加全面的認識，並認為先前的版本中有部分誤譯，便由詹麗茹和柯翠園重譯校注本，由台灣教授協會發行。

== 內容簡介 ==
作者以二十多年的實際觀察與學術研究的結晶、第一手的詳實資料為基礎，[[自由主義|自由主義者]]的觀點與關懷，描繪1941年－1960年之間被視為戰利品的[[台灣|台灣]]在[[美國|美國]]與[[中國|中國]]的拉扯之下，因自由意識與力量的闕如而造成的悲劇命運。

《被出賣的台灣》是一本非常有影響力的書籍，尤其在[[台灣日治時期|台灣日治時期]]和[[臺灣戰後時期|臺灣戰後時期]]的更替時代。當時柯喬治在[[前美國駐臺北領事館|前美國駐臺北領事館]]任職，因此目睹中國國民黨執政的中華民國政府統治台灣過程、台灣人因此由親中派轉變成反中派的經過。《被出賣的台灣》強烈指責中國國民黨並強調台灣在亞洲的特殊地位。此書只以英文出了一版，直到1992年，台灣的出版公司印出第二版。《被出賣的台灣》目前可合法於網際網路免費下載。
