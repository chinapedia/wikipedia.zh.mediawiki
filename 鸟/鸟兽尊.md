'''鸟兽尊'''，或称'''鸟兽形尊'''，中国古代文物的一个类别，形状似鸟兽的盛酒器的通称。最早定义“鸟兽尊”的是1941年[[容庚|容庚]]所著的《商周彝器通考》，该书单立“鸟兽尊”一类，“于鸟兽形之尊彝统称之为尊”。虽然“鸟兽尊”的最初定名是为了区分青铜器，但形制类似的陶器也被称为“鸟兽尊”。

== 定义 ==
最早定义“鸟兽尊”的是1941年[[容庚|容庚]]所著的《商周彝器通考》，该书单立“鸟兽尊”一类，“于鸟兽形之尊彝统称之为尊”<ref name=tong>容庚:《商周彝器通考》.1941年初版</ref>。鸟兽尊的“尊”与青铜器专名“[[尊|尊]]”的含义或青铜器铭文中“尊”的含义都不相同。专名“尊”是指侈口，腹部粗而鼓张，高圈足，形体较宽的一类青铜器，也就是“似觚觯而巨者”<ref name=tong/>；铭文中的“尊”是酒、食器等礼器的共名，也就是[[王国维|王国维]]在《说彝》中提到的“大共名”<ref name=shuo>王国维:《说彝》.收录于《观堂集林》卷三</ref>；鸟兽尊的“尊”是盛酒器的通称，也就是王国维所称的“小共名”<ref name=shuo/>。<ref name=zong>朱凤瀚:《中国青铜器综论》.上海:上海古籍出版社,2009.ISBN 9787532550555</ref>{{rp|176-191}}

《周礼·春官·司尊彝》中有“掌六尊六彝之位”的记载，“六尊六彝”中冠以鸟兽名字的有鸡彝、鸟彝、虎彝、蜼彝和象尊，此外注释《周礼》的郑众等人还认为六尊中的“献尊”就是“牺尊”。对于这些器名，汉代以来学者的说法不一，主要有尊上的画像、工艺装饰和本身形象三种。商周青铜器定名中将鸟兽形的礼器归为尊类始于宋朝，《博古图录》将外形做牛、象和凫的酒器分别定名为牺尊、象尊和凫尊，这是最早的与后来鸟兽尊定义相符的定名，宋以后金石学者多从《博古图录》。1941年容庚刊行《商周彝器通考》后，学界多将鸟兽形盛酒器统称为鸟兽尊，而少有异议{{efn|郭宝钧认为“尊之作鸟兽形者谓之彝”<ref>郭宝钧:《商周铜器群研究》.147页</ref>}}。<ref name=zong/>{{rp|176-191}}

== 定名 ==
杜廼松在《论青铜鸟兽尊》中指出，每件鸟兽尊定名时应根据其所肖动物称呼其具体名称，如牛尊、羊尊、象尊、鸟尊等，仅具有鸟兽形象而不能确知其为何种动物时，可沿用习惯叫法，称之为“牺尊”。<ref name=du>杜廼松:《论青铜鸟兽尊》,《故宫博物院院刊》, 1995(S1):174-186</ref>

带有铭文的鸟兽尊一般使用青铜器的通称“尊彝”二字来自称，如守宫鸟尊、盠驹尊、邓仲牺尊等。1974年陕西省宝鸡市茹家庄二号墓出土的[[井姬盂鏙|井姬盂鏙]]自称为“盂鏙”{{efn|井姬盂鏙有八字铭文：“{{僻字|𢐗|左“弓”右“鱼”}}白（伯）匄井姬用盂鏙”}}，杜廼松认为它属于罕见称呼或地区别称。<ref name=du/>2000年山西省曲沃县北赵村晋侯墓地113号墓出土的[[晋侯猪尊|晋侯猪尊]]则自名为“飤”{{efn|晋侯猪尊带有五字铭文：“晋侯作旅飤”<ref name=gg/>}}，也属于罕见称呼。<ref name=gg>{{Citeweb|title=西周　青铜猪尊 |url=http://www.gg-art.com/profesional/index.php?detailid=535 |publisher=中华博物 }}</ref>

== 分类 ==
根据鸟兽尊所肖动物的种类，《中国青铜器综论》将常见鸟兽尊主要可分为鸱鸮尊、禽尊、虎尊、象尊、犀尊、牛尊、羊尊、马尊、兔尊和神兽尊等。<ref name=zong/>{{rp|182}}《论青铜鸟兽尊》将之分为鸟尊、鸮尊、鸳鸯尊、鸭尊、牛尊、羊尊、猪尊、马驹尊、象尊、虎尊、牺尊、鸟首兽尊和兔尊等。<ref name=du/>
