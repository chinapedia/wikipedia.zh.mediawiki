'''什洛莫·敏茨'''（Shlomo Mintz；{{bd|1957年|10月30日}}）是一位以色列籍[[小提琴|小提琴]][[大师|大师]], [[中提琴演奏家|中提琴演奏家]]和指挥家。他常常与与世界著名乐团、著名指挥家在全球音乐舞台上合作演出，他的独奏音乐会、室内乐音乐会遍及全世界。

== 个人简介 ==
什洛莫·敏茨生于[[莫斯科|莫斯科]]. 1959年在他两岁的时候，他和父母移居到以色列, 他先随欧洲中部的小提琴学派代表以色列名师伊洛娜·费赫尔学琴。费赫尔把敏茨介绍给了艾萨克·斯特恩，斯特恩成为了敏茨的良师益友。他也是纽约迪蕾教授的学生。

== 音乐生涯 ==
敏茨在他11岁的时候开始他的音乐生涯，和以色列爱乐乐团合作独奏。后来他接到祖宾·梅塔的通知顶替生病的伊扎克·帕尔曼，和以色列爱乐乐团一起演奏帕格尼尼的《第一小提琴协奏曲》。16岁时他在艾萨克·斯特恩指挥下和美国以色列文化基金的赞助下和匹兹堡交响乐团首次于卡内基音乐厅举行首演。随后他到纽约茱莉亚德音乐学校跟随迪蕾教授学习。在他20岁的时候他和诸多著名的指挥家们（例如朱里尼Carlo Maria Giulini, 多拉蒂Antal Dorati, 和 Eugene Ormandi）一起了欧洲之行。在他20多岁的时候和德国留声机唱片公司签署了合约。
1997年敏茨在指挥家约尔·莱维的指挥下在荷兰马斯特里赫特与林堡交响乐团 演奏了帕格尼尼的名琴“加农炮”- 一把由意大利名师Giuseppe Guarneri del Gesù制作的小提琴。此次音乐会是德国电视网络(TROS)发起的，于1997年12月在电视上播出。2012年什洛莫·敏茨庆祝他50年的演艺生涯。

== 指挥和艺术总监生涯 ==
18岁时，敏茨把指挥加入到他的艺术生涯，随后他在国际上指挥过倍受欢迎的交响乐团。例如英国皇家交响乐团、日本广播协会交响乐团和以色列交响乐团。1989年到1993年，他曾是以色列室内交响乐团的音乐顾问，1994年3月他被任命为荷兰马斯特里赫特交响乐团的艺术顾问及首席客座指挥。他在四季中引领交响乐团参加了几周的音乐会，包括一些指挥及小提琴独奏家。在2008年什洛莫·敏茨被任命为萨格勒布爱乐乐团的首席客座指挥。
自2002年起，什洛莫·敏茨被任命为锡安瓦莱斯国际音乐节的艺术总监。

== 教育生涯 ==
什洛莫·敏茨在国际范围内教授大师班。从2012年夏天开始，他将在克莱恩蒙大纳（CH克莱恩蒙大纳古典音乐会）教授大师班。敏茨也是布宜诺斯艾利斯国际小提琴大赛评审团的指导和主席。他也是日本Munetsugu天使小提琴大赛的主席。他曾经也是诸多国际小提琴比赛的评审，例如莫斯科柴可夫斯基小提琴大赛(1993年)，布鲁塞尔伊丽莎白女皇国际音乐大赛（1993年和2001年）。2001年10月敏茨曾是在波兰波兹兰举办的亨里克·维尼亚夫斯基国际小提琴大赛的评审团主席。2002年到2011年被任命为瑞士锡安瓦莱斯国际小提琴大赛的评审团主席。敏茨是以色列里昂 (Keshet Eilon) 国际小提琴课，及在以色里昂为年轻小提琴手举办的夏季高级班的创始人之一，他在过去的18年中是此课程的赞助商 (从1992年到2010年)。

== 音乐作品录==
作为独奏家和指挥家，敏茨保持着积极的录制。与他签约的公司包括德国留声机公司、发行公司Erato、出版者RCA Victor、出版者Avie Records、出版者Challenge Records。敏茨和德国留声机公司合作录制的多数CD被重新发行。例如在2007年4月他的第一张（《布鲁赫和门德尔松小提琴奏鸣曲》）在德国留声机公司的大奖系列中被重新发行（并被称为“世界上最美妙的录制”）。
