'''塔塔尔族'''（{{lang-tt|'''Кытай татарлары'''}}），或称'''塔塔尔人'''、'''[[鞑靼人|鞑靼人]]'''，是[[中国|中国]]官方认定的[[少数民族|少数民族]]之一，主要居住在中国[[新疆|新疆]]地区。“塔塔尔”是“鞑靼”的音译之一。

== 形成 ==
近代从俄国移民至中国境内的[[鞑靼人|鞑靼人]]，在中国境内繁衍至今称塔塔尔族。

== 现状 ==
多数塔塔尔族信仰[[伊斯兰教|伊斯兰教]]，生活、习俗受到伊斯兰文化的影响<ref> [http://www.china.com.cn/ch-shaoshu/index44.htm 中国网]，《民族工作大全》，千里原主编。</ref>，主要分布在[[新疆维吾尔自治区|新疆维吾尔自治区]]的[[伊宁|伊宁]]、[[塔城|塔城]]、[[乌鲁木齐|乌鲁木齐]]。少数散居在[[布尔津|布尔津]]、[[奇台|奇台]]和南疆的主要城市（[[哈爾濱|哈爾濱]]也有他們），3556人（2010人口普查）。奇台县有全国唯一的塔塔尔族乡——[[大泉塔塔尔族乡|大泉塔塔尔族乡]]。

突厥碑文中的塔塔爾人有三十姓與九姓的說法。有六個部落：察罕塔塔爾、阿勒赤塔塔爾、帖烈惕塔塔爾、不魯恢塔塔爾、奎因塔塔爾、和最有名的是禿黑禿兀惕塔塔爾。在現時中國新疆的塔塔爾族也由[[伏爾加韃靼人|伏爾加韃靼人]]、[[克里米亞韃靼人|克里米亞韃靼人]]、[[諾蓋人|諾蓋人]]、葉尼塞塔塔爾人四部分組成。比例分別是5:1:2:2。

此外，中国东北地区还有少量世居土著塔塔尔人，属于欧罗巴人种。

== 语言 ==
中国境内的塔塔尔族有自己的语言，属[[阿尔泰语系|阿尔泰语系]][[突厥语族|突厥语族]]西匈语支，有以[[阿拉伯字母|阿拉伯字母]]为基础的文字。由于长期与[[维吾尔族|维吾尔族]]、[[哈萨克族|哈萨克族]]共处，因而这两个民族的语言、文字也逐渐成为塔塔尔族的日常用语和通用文字。

== 習俗 ==
部落意識在塔塔爾中都有保留，比如[[弘吉剌|弘吉剌]]部、布爾吉部、但由于多數塔塔爾族由游牧生活方式轉為城市生活，部落意識就淡化了。塔塔爾人的姓名就帶上部落的名字。一個塔塔爾人的名字後面，要加上自己父親的名字，然後再加上自己部落的名字，這樣才構成一個完整的人名。

== 文學 ==
塔塔爾人有千年以上的文化傳統，除了有豐富的民間文學外，各個朝代都有大量的作家文學產出。塔塔爾族的民間文學有神話、民間故事、戲劇、民歌、詩等不同種類的作品，而作家文學有小說、長篇敘事詩、短詩、散文等等各類的創作。
