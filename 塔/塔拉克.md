'''塔拉克'''或译'''特拉格'''<ref name="a1"/>（{{lang-ar|طلاق}} ''ṭalāq''）是[[伊斯蘭教|伊斯蘭教]]“[[休妻|休妻]]”的意思，因[[穆斯林|穆斯林]]男子对婚姻具有决定权，男子离婚较女子而言较为轻率<ref>王茜、魏铭清：[http://www.studa.net/wenhuayanjiu/100909/15024680-2.html 维吾尔族婚俗历史演变探索] {{webarchive|url=https://web.archive.org/web/20150620185041/http://www.studa.net/wenhuayanjiu/100909/15024680-2.html |date=2015-06-20 }}</ref>。

当穆斯林男子说出塔拉克后，无论女方同意与否，婚姻视为无效。若双方想要复婚，须请来[[阿訇|阿訇]]念经并重新举行婚礼仪式<ref name="ndcnc">{{cite web |language =Zh-hans |url = http://www.ndcnc.gov.cn/datalib/2004/Nation/DL/DL-165243|title=维吾尔族的婚恋 |publisher=国家数字文化网 |accessdate=2011-08-16 }}</ref>。若女方不愿复婚，则需等待一百天后才可改嫁<ref name="ndcnc" />。若男子连续说出三个塔拉克，则意为男子想永久断绝关系。此时若男子再想复婚，则必须待到女子同他人结婚至第二次离婚后才可以<ref>{{Cite book |author = 程万里 |title = 话说南疆 画说南疆 |location = 乌鲁木齐 |publisher = 新疆青少年出版社 |date = 2006年6月}}</ref>。

== 三遍塔拉克式离婚 ==

[[穆斯林世界|穆斯林世界]]，普遍存在男性一次说三遍塔拉克离婚的习俗。近代以来，各国法律对三遍塔拉克式离婚的合法性有不同规定<ref name="a2">{{Cite web |url=http://world.huanqiu.com/exclusive/2016-10/9537835.html |title=印度要禁穆斯林一夫多妻 称其反妇女且违反宪法|accessdate=2017-06-21 |author=记者：苑基荣 |date=2016-10-12 |work= |publisher=环球网|language=Zh-hans }}</ref><ref name="a3">{{Cite web |url=http://www.cankaoxiaoxi.com/world/20170312/1758661.shtml |title=丈夫说三遍我要离婚！妻子只有无条件接受滚蛋？！印度这三次离婚教义，太奇葩了吧！ |accessdate=2017-06-21 |author= |date=2017-03-12 |work= |publisher=《参考消息》官方网站 |language=Zh-hans |deadurl=yes |archiveurl=https://web.archive.org/web/20170312015657/http://www.cankaoxiaoxi.com/world/20170312/1758661.shtml |archivedate=2017-03-12 }}</ref>。

随着现代通讯的发展，当代穆斯林男子甚至无需面对女方，仅用[[手机短信|手机短信]]<ref>{{cite web|url=http://www.anhuinews.com/zhuyeguanli/system/2010/02/10/002649843.shtml|title=约旦男子发短信就可离婚 司法部门欲插手管制|publisher=中安在线 来源：中国新闻网|author=|date=2010-02-10|accessdate=2017-06-21|language=Zh-hans|archive-url=https://web.archive.org/web/20171024165950/http://www.anhuinews.com/zhuyeguanli/system/2010/02/10/002649843.shtml|archive-date=2017-10-24|dead-url=yes}}</ref>、[[电子邮件|电子邮件]]<ref name="a4">{{Cite web |url=http://www.yesky.com/20000602/73178.shtml |title=伊斯兰法：电子邮件离婚无效？|accessdate=2017-06-21 |author= |date=2000-06-02 |work= |publisher=天极网|language=Zh-hans }}</ref>、[[通讯软件|通讯软件]]<ref name="a1"/><ref name="a3"/>、[[社交软件|社交软件]]<ref name="a1"/><ref name="a3"/>等通讯方式发出三个“塔拉克”单词就可以完成离婚。针对这种情况，各国司法部门又有不同的处理。由于[[伊斯兰国家|伊斯兰国家]]普遍承认[[一夫多妻|一夫多妻]]的合法性，三遍塔拉克式离婚不会影响男性再婚的合法性，但女性再婚的合法性可能得不到认可<ref name="a4"/>。

== 各国法律规定 ==

当代[[穆斯林国家|穆斯林国家]]对三遍塔拉克离婚的合法性有不同规定。有报道称<ref name="a1"/>有二十多个国家禁止，包括[[巴基斯坦|巴基斯坦]]、[[孟加拉国|孟加拉国]]。

非穆斯林国家中，[[印度宪法|印度宪法]]曾允许穆斯林社群依照宗教习惯管理结婚、离婚和继承等事务，具有合法性<ref name="a1">{{Cite web |url=http://henan.163.com/17/0309/17/CF3PQ6IP022709K5.html |title=印度离婚这么容易？丈夫大喊三遍这个词就能离|accessdate=2017-06-21 |author=责任编辑：HN031 |date=2017-03-09 |work= |publisher=观察者网|language=Zh-hans }}</ref><ref name="a2"/>，但该国最高法庭于2017年8月22日裁定相关规定违宪，自此禁止此项习俗<ref>{{cite news|last1=Safi|first1=Michael|title=India court bans Islamic instant divorce in huge win for women's rights|url=https://www.theguardian.com/world/2017/aug/22/india-supreme-court-bans-islamic-instant-divorce-triple-talaq|accessdate=2017-08-22|work=The Guardian|date=2017-08-22}}</ref><ref>{{cite news|title=Muslim men in India could once divorce their wives by saying three words. Until now.|url=https://www.washingtonpost.com/world/muslim-men-in-india-could-once-divorce-their-wives-by-saying-three-words-until-now/2017/08/22/93ccb4f6-871a-11e7-9ce7-9e175d8953fa_story.html?utm_term=.cf5f2edf5971|accessdate=2017-08-22|work=Washington Post|date=2017-08-22}}</ref>。在中华人民共和国成立后，随着《[[中华人民共和国婚姻法|中华人民共和国婚姻法]]》的贯彻，这种习俗在中國穆斯林社群中逐渐消失<ref>{{cite web |language = Zh-hans |url = http://www.tianshannet.com.cn/GB/channel6/57/200112/12/13650.html |title=维吾尔族 |publisher=天山网 |accessdate=2011-08-16 }}</ref>。
