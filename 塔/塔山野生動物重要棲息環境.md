'''塔山野生動物重要棲息環境'''位於[[中華民國|中華民國]][[嘉義縣|嘉義縣]][[阿里山鄉|阿里山鄉]]，是依《野生動物保育法》公告之野生動物重要棲息環境<ref name="林務局">{{cite web |url=http://conservation.forest.gov.tw/ct.asp?xItem=3339&ctNode=176&mp=10 |title=塔山野生動物重要棲息環境 |accessdate=2014-10-26 |publisher=農委會林務局 |deadurl=yes |archiveurl=https://web.archive.org/web/20141026100615/http://conservation.forest.gov.tw/ct.asp?xItem=3339&ctNode=176&mp=10 |archivedate=2014-10-26 }}</ref>。本區原本為阿里山針闊葉樹林自然保護區，為林務局嘉義林區管理處阿里山事業區林班地。占地696.38公頃，海拔高度從1,450公尺至2,600公尺，其內包括造林地及原始林，為針闊葉混合林。區內佔大面積者為[[紅檜|紅檜]]人工林，現存的天然林經齡級判斷，屬於原生林經砍伐後，經天然演替形成的次生林。

[[阿里山森林鐵路|阿里山森林鐵路]]眠月線行經此區<ref>{{cite web|url=http://www.howone.com.tw/railway/news.php?uid=21&id=7693|title=眠月線搶通 斷頭石猴解塵封|work=中國時報|date=2008-06-01|accessdate=2014-10-26|deadurl=yes|archiveurl=https://web.archive.org/web/20141026094230/http://www.howone.com.tw/railway/news.php?uid=21&id=7693|archivedate=2014-10-26}}</ref>。範圍內另有農委會依《[[文化資產保存法|文化資產保存法]]》公告之[[阿里山台灣一葉蘭自然保留區|阿里山台灣一葉蘭自然保留區]]<ref>{{cite web |url=http://country.swcb.gov.tw/docs/chiayi/11%E7%AC%AC%E4%BA%8C%E7%AF%87-3new.pdf |title=嘉義縣縣級鄉村風貌綱要規劃結案成果報告書 |format=PDF |publisher=中華民國景觀學會 |date=2006-10 |accessdate=2014-10-26 |deadurl=yes |archiveurl=https://web.archive.org/web/20141026093000/http://country.swcb.gov.tw/docs/chiayi/11%E7%AC%AC%E4%BA%8C%E7%AF%87-3new.pdf |archivedate=2014-10-26 }}</ref>。
