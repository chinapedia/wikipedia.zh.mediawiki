'''塔尔沙漠'''（Thar Desert），也叫'''印度大沙漠'''，是[[南亚|南亚]]地区西北部的一个[[沙漠|沙漠]]。该沙漠面积200,000平方公里。世界第18大沙漠，亚洲第3大沙漠。 该沙漠大部分位于[[印度|印度]][[拉贾斯坦邦|拉贾斯坦邦]]。而該沙漠伸延至[[巴基斯坦|巴基斯坦]]的部分稱為'''Cholistan'''，因此也經常統稱為'''Thar-Cholistan'''。

== 概况 ==
塔尔沙漠位于[[巴基斯坦|巴基斯坦]]东南部、[[印度|印度]]西北部，西至[[印度河|印度河]]、[[萨特卢杰河|萨特卢杰河]]，东至印度[[马尔瓦高原|马尔瓦高原]]东侧，是印度大沙漠的延伸部分。沙漠地区[[海拔|海拔]]100～200米，[[面积|面积]]约200,000平方公里(77,000平方英里)。沙漠大部分地区为沙质荒漠，东南部[[砾漠|砾漠]]较多，[[沙丘|沙丘]]一般高度约为30至90米，最高达150米。沙漠中[[沙垄|沙垄]]、[[盐滩地|盐滩地]]、[[龟裂地|龟裂地]]广布，有季节性[[盐湖|盐湖]]及[[干河道|干河道]]，[[地下水位|地下水位]]埋藏较深。沙漠中有几个盐湖。除西部的印度河流域与东南部的[[卢尼河|卢尼河]]流域以外，大都为[[内流区|内流区]]。

沙漠属[[亚热带荒漠气候|亚热带荒漠气候]]。受周围[[高原|高原]]山地，特别是西侧[[伊朗高原|伊朗高原]]的影响，降雨量极少。[[年均降雨量|年均降雨量]]小于100毫米。高地雨后有[[季节性河流|季节性河流]]。当地降水量低，西部约为100毫米(4英寸)，东部约为500毫米(20英寸)，约90％降在7∼9月的[[雨季|雨季]]。

[[夏季|夏季]]5到6月最高气温可达48至51℃，最低气温则出现在1月份左右，约5至10℃(41至50℉)，并伴随[[霜|霜]]降。主要灾害是5至6月的强烈[[尘暴|尘暴]]，[[风速|风速]]可达每小时140至150公里(87至93英里)。

大部分地区无[[植物|植物]]生长，少数耐干、热的植物可以生存。在能利用地下水的地区，产有[[小麦|小麦]]和[[棉花|棉花]]。

== 起源 ==
该沙漠的起源存在争议。基本观点有如下几点：
* 对于印度河的过度开发致使植被破坏，水土流失。
* 山脉的阻挡冬季東北季风所带的水汽无法到达塔尔沙漠，
* 由于夏季副熱帶高壓的控制无法形成降雨。

== 生物 ==
塔尔沙漠只有少数耐旱的植物生存。野生动物有[[南亚鸨|南亚鸨]]、[[印度黑羚|印度黑羚]]等。

== 经济 ==
[[File:Young_muslim_woman_in_the_Thar_desert_near_Jaisalmer,_India.jpg|thumb]]
在沙漠里居住的人以农业和畜牧业为生。有水源灌溉的地方出产小麦、棉花、甘蔗、粟、芝麻、豆类和辣椒。农业在当时并非可靠的收入，雨季结束后，33%作物歉收。饮水与生活用水多靠水池贮存雨水。地下水位很低且多为咸水，不便利用。在沙漠中部曾发现优良含水层。1960年印、巴两国签订用水协定，双方都在塔尔沙漠区修建灌溉工程。苏库尔(Sukkur)坝(1932年完成)灌溉巴基斯坦塔尔沙漠的南部，甘格(Gang)灌渠灌溉其北部地区。拉贾斯坦(Rajasthan)灌渠灌溉印度塔尔沙漠广大的土地，赫里盖(Harike)坝灌溉印度塔尔沙漠的北部。

该地区矿藏有褐煤、天然气、石膏、石灰岩、班脱岩和玻璃沙。并有出产湖盐。

畜牧业、林业配合果树和蔬菜在这个干旱地区相对比较可行。该地区的地面面临着频繁的干旱。由于过度放牧，风蚀和水蚀，采矿业和其他行业的因素，导致土地严重退化。
