'''塔林大学'''（{{lang-et|Tallinna Ülikool}}），成立于2005年，是[[爱沙尼亚|爱沙尼亚]]最大的[[高等院校|高等院校]]之一，位于首都[[塔林|塔林]]。这所大学拥有教学及研究人员1200人<ref name="faculty"/>，[[本科生|本科生]]、[[硕士研究生|硕士研究生]]和[[博士研究生|博士研究生]]分别有约近6000人、近3000人和近400人<ref name="students"/>。现任校长为迪德·兰德教授。<ref name="rector"/>

== 历史 ==
2002年5月15日，塔林的几所大学和研究所与爱沙尼亚教育部、塔林市政府之间签署了协议，决定筹办塔林大学。最初参与到塔林大学筹办工作的机构有：爱沙尼亚学术图书馆、爱沙尼亚艺术学院、爱沙尼亚人文研究所、[[爱沙尼亚科学院|爱沙尼亚科学院]]历史研究所、以及塔林师范大学。<ref name="history">{{cite web|url=http://www.tlu.ee/?LangID=2&CatID=1359 |title=History |publisher=塔林大学 |date= |accessdate=2009年5月2日 |language=en }}</ref>

最终，以原塔林师范大学为主体构成的塔林大学在2005年3月18日成立。<ref>{{Et icon}}{{cite web|url=http://www.tlu.ee/?LangID=1&CatID=1740 |title=Ülikoolist |publisher=塔林大学 |date= |accessdate=2009年5月2日 |language= }}</ref>在此之前，欧洲各国首都中仅有塔林没有直接以其城市名来命名的大学。<ref name="history"/>

== 机构设置 ==
[[File:Mare_Building,_Tallinn_University_(2012).jpg|thumb]]

[[File:Ursa_Building,_Tallinn_University_(2006).jpg|thumb]]
塔林大学的教学组织由18个研究所和8个学院展开，分别是：爱沙尼亚[[未来学|未来学]]研究所、爱沙尼亚[[人文|人文]]研究所、[[生态学|生态学]]研究所、[[教育学|教育学]]研究所、爱沙尼亚[[人口统计学|人口统计学]]研究所、爱沙尼亚[[语言|语言]][[文化|文化]]研究所、[[美术|美术]]研究所、[[日尔曼语族|日尔曼]]-[[罗曼语族|罗曼]]语言文化研究所、[[健康科学|健康科学]]与[[体育学|体育]]研究所、[[历史学|历史]]研究所、[[信息学|信息学]]研究所、[[情报学|情报学]]研究所、[[国际学|国际]]与[[社会学|社会学]]研究所、[[数学|数学]]与[[自然科学|自然科学]]研究所、[[政治学|政治学]]与政府管理研究所、[[心理学|心理学]]研究所、[[斯拉夫语族|斯拉夫语言文化]]研究所、[[社会工作|社会工作]]研究所，[[波罗的海|波罗的海]][[电影|电影]][[媒体|媒体]]学院、[[哈普萨卢|哈普萨卢]]学院、[[拉克韦雷|拉克韦雷]]学院、管理、法律与社会学院、人文学院、自然科学与健康学院、教育科学学院、数字技术学院等。塔林大学还设有九个学术中心，五个卓越中心。<ref name="inst">{{cite web|url=http://www.tlu.ee/?LangID=2&CatID=4289 |title=Institutes and Colleges |publisher=塔林大学 |date= |accessdate=2009年5月2日 |language=en }}</ref> 該大學主校區的建築均有拉丁語名稱。

== 国际交流 ==
塔林大学是欧洲大学协会（EUA）、欧洲首都大学网络（UNICA）、波罗的海区域大学网络（BSRUN）等国际大学组织的成员。<ref name="network">{{cite web|url=http://www.tlu.ee/?LangID=2&CatID=3356&ArtID=267&action=article |title=Participation in International Networks |publisher=塔林大学 |date= |accessdate=2009年5月2日 |language=en }}</ref><ref>{{cite web |url=http://www.ulb.ac.be/unica/members.html |title=Members |publisher=欧洲首都大学网络 |date= |accessdate=2009年5月2日 |language=en |deadurl=yes |archiveurl=https://web.archive.org/web/20090413141132/http://www.ulb.ac.be/unica/members.html |archivedate=2009年4月13日 }}</ref><ref>{{cite web |url=http://bsrun.utu.fi/members/locations.html |title=LOCATIONS OF THE MEMBER INSTITUTIONS |publisher=波罗的海区域大学网络 |date= |accessdate=2009年5月2日 |language=en |deadurl=yes |archiveurl=https://web.archive.org/web/20090203070109/http://bsrun.utu.fi/members/locations.html |archivedate=2009年2月3日 }}</ref>

通过[[欧盟|欧盟]]的[[伊拉斯謨計劃|伊拉斯謨計劃]]，塔林大学与[[欧洲|欧洲]]各国的合作大学进行人员交流。<ref>{{cite web |url=http://www.tlu.ee/files/arts/54/lepinc1dc36a64625a48dfbc184613753f738.pdf |title=LLP/Erasmus Agreements |publisher=塔林大学 |date= |accessdate=2009年5月2日 |language=en |format=pdf }}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>此外，塔林大学还与[[比利时|比利时]]、[[丹麦|丹麦]]、[[匈牙利|匈牙利]]、[[格鲁吉亚|格鲁吉亚]]、[[芬兰|芬兰]]、[[法国|法国]]、[[德国|德国]]、[[希腊|希腊]]、[[意大利|意大利]]、[[日本|日本]]、[[韩国|韩国]]、[[拉脱维亚|拉脱维亚]]、[[立陶宛|立陶宛]]、[[挪威|挪威]]、[[俄罗斯|俄罗斯]]、[[西班牙|西班牙]]、[[瑞典|瑞典]]、[[土耳其|土耳其]]、[[乌克兰|乌克兰]]、[[美国|美国]]等国家的38所大学签署了双边合作协议。<ref>{{cite web|url=http://www.tlu.ee/?LangID=2&CatID=3356&ArtID=265&action=article |title=Bilateral Agreements |publisher=塔林大学 |date= |accessdate=2009年5月2日 |language=en }}</ref>

塔林大学与挪威[[奥斯陆大学学院|奥斯陆大学学院]]和意大利[[帕尔马大学|帕尔马大学]]联合开设了一个欧盟Erasmus Mundus硕士课程，即数字图书馆学习硕士课程。<ref>{{cite web |url=http://dill.hio.no/ |title=International Master in Digital Library Learning (DILL) |publisher=奥斯陆大学学院 |date= |accessdate=2009年5月2日 |language=en |deadurl=yes |archiveurl=https://web.archive.org/web/20090421162001/http://dill.hio.no/ |archivedate=2009年4月21日 }}</ref>
