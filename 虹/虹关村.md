'''虹关村'''位于中国[[江西省|江西省]][[上饶市|上饶市]][[婺源县|婺源县]][[浙源乡|浙源乡]]北部，北邻[[安徽省|安徽省]][[休宁县|休宁县]][[板桥乡_(休宁县)|板桥乡]]，曾被誉为“吴楚锁钥无双地，[[徽饶古道|徽饶古道]]第一关”。始建于[[南宋|南宋]][[建炎|建炎]]年间，村民多为[[詹姓|詹姓]]。<ref name="万基耀">{{cite web |title=虹关墨香依稀在 |author=万基耀|publisher=大江网—[[江西日报|江西日报]] |url=http://jiangxi.jxnews.com.cn/system/2010/05/29/011393704.shtml |date=2010-05-29 |accessdate=2011-03-19}}</ref>2009年被评为[[江西省历史文化名村|江西省历史文化名村]]，2010年被评为[[中国历史文化名村|中国历史文化名村]]。<ref>{{cite web |title=我市1村获“中国历史文化名镇名村”称号 |publisher=上饶之窗-[[上饶晚报|上饶晚报]] |url=http://www.srzc.com/news/srxw/shizhengjingji/2010/1217/84036.html |date=2010-12-17 |accessdate=2011-03-19}}</ref>

虹关古时曾出过[[明|明]][[洪武|洪武]]年间[[吏部尚书|吏部尚书]][[詹同|詹同]]、[[詹徽|詹徽]]父子，[[清|清]][[嘉庆|嘉庆]]年间[[直隶州|直隶州]][[知府|知府]][[詹应甲|詹应甲]]等人。<ref name="安徽文化网">{{cite web |title=徽墨古村——虹关村 |publisher=安徽文化网 |url=http://www.ahage.net/lvyou/6319.html |accessdate=2011-03-19 }}{{Dead link|date=2018年7月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref> 虹关古以制[[墨|墨]]闻名，20世纪中期逐渐没落。<ref name="万基耀"/> 

现在村内保存较好的古建筑有虑得堂、顾汝堂、留耕堂、玉映堂、棣芳堂、玉鑑堂、礼和堂、继志堂、务本堂、从是堂、詹大有祖居、光裕堂和六顺堂等。<ref>{{cite journal |title=徽墨名村——婺源虹关古村镇 |author=詹庆德 |journal=今日国土 |year=2008 |volume= |issue=7 |pages= |url= }}</ref>村内著名的'''虹关古樟'''树龄已有千年，胸径3.4米，高26.1米。<ref name="万基耀"/>
