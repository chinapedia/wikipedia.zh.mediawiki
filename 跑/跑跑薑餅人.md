'''跑跑薑餅人'''是一款跑酷遊戲，目前分為[[KakaoTalk|KakaoTalk]]（第九季）以及烤箱大逃亡（第三季）的[[版本|版本]]<ref>{{cite web |author1=ARTRICE |title=[遊戲心得]跑跑薑餅人－淺談遊戲玩法及分析 |url=http://artrice.game.tw/20180726/5462}}</ref>。故事圍繞於具有生命力的薑餅人身上，薑餅人們想逃離被女巫吃掉的命運，因此討論了一連串的計畫，準備從烤箱之中以跳躍或滑壘的方式閃避障礙物脫困。'''Line跑跑薑餅人'''已於2018年6月5日停止營運，目前'''跑跑薑餅人：烤箱大逃亡'''在Google Play與App Store雙平台仍然營運中。

==歷史==

===Kakao 跑跑薑餅人（Cookie run for kakao）===
{{Vgname|跑跑薑餅人(韓版)|en=Cookie Run for kakao}}是一款韓國通訊軟體[[KakaoTalk|KakaoTalk]]的一款跑酷遊戲，遊戲中為全韓文。

===LINE 跑跑薑餅人（LINE cookie run）===
{{Vgname|LINE跑跑薑餅人|en=LINE Cookie Run}}是通訊軟體[[LINE|LINE]]平台上的一款跑酷遊戲，於2014年2月上架。2018年6月5日停止遊戲服務<ref>[https://archive.today/20180322025329/https://notice2.line.me/LGCOOKIE/ios/document/notice?documentId=20080621&lang=zh-Hant 《LINE跑跑薑餅人》停止遊戲服務通知]，跑跑薑餅人通知，2018-03-16[2018-03-22]。 </ref>，已被《跑跑薑餅人－烤箱大逃亡》所取代。

===QQ跑跑薑餅人===
由騰訊代理的跑跑薑餅人遊戲。與LINE版相同。

===大逃亡版本===
{{Vgname|跑跑薑餅人－烤箱大逃亡|en=Cookie Run:Ovenbreak}}於2016年9月27日的預約登錄，預約的玩家登入高達80多萬次，語言支援多國化，並於同年10月28日正式開放<ref name="cob"/>。統合所有國家的遊戲版本，與世界各國的玩家進行限時對戰<ref name="cob">[http://www.mofang.com.tw/NGnews/10000056-10092403-1.html 《跑跑薑餅人：烤箱大逃亡》全球正式上線] {{webarchive|url=https://web.archive.org/web/20180322082243/http://www.mofang.com.tw/NGnews/10000056-10092403-1.html |date=2018-03-22 }}，魔方網，2016-10-28[2018-03-22]。</ref>。第二季於2017年8月30日進行改版<ref name="cob2">[http://www.mofang.com.tw/NGnews/10000056-10097664-1.html 《跑跑薑餅人：烤箱大逃亡》第二季「熱血逃出團」全新大改版] {{webarchive|url=https://web.archive.org/web/20180322082224/http://www.mofang.com.tw/NGnews/10000056-10097664-1.html |date=2018-03-22 }}，魔方網，2017-08-30[2018-03-22]。</ref>。此外還陸續推出聯賽、逃出團<ref name="cob2"/>、餅乾訓練所<ref>[http://www.mofang.com.tw/NGnews/10000056-10098930-1.html 《跑跑薑餅人：烤箱大逃亡》訓練所系統登場！同步推出櫻花餅乾！] {{webarchive|url=https://web.archive.org/web/20180322082249/http://www.mofang.com.tw/NGnews/10000056-10098930-1.html |date=2018-03-22 }}，魔方網，2018-03-11[2018-03-22]。 </ref>。
====薑餅人大逃亡（Oven Break）====
最初的版本叫做 薑餅人大逃亡（Oven Break），畫面比例為4:3，除了有跳躍及滑行外，還多了一個勾盪，但此版本沒有二段跳。

====薑餅人大逃亡 2（Oven Break 2）====
二代的畫風改為現在的畫風，新增寵物，還有BONUS，但要出到3次才能上天堂領獎勵，否則的話只是分數加倍，開始能二段跳，移除勾盪。

==遊戲操作==
在遊戲內，遊戲是以橫向操作，從左至右的移動進行，畫面左下方以Jump跳為主，畫面右下方則以Slide滑壘為主，遊戲內使用跳、滑兩種按鍵來操控薑餅人躲避障礙物及收集果凍累積分數。<ref name="a">Agape，[http://www.chinatimes.com/realtimenews/20140210004203-260412 LINE跑跑薑餅人 跑出第一名的人生！] {{webarchive|url=https://web.archive.org/web/20160425031516/http://www.chinatimes.com/realtimenews/20140210004203-260412 |date=2016-04-25 }} ，[[中時電子報|中時電子報]]，2014年02月10日【引用時間：2015年10月30日】。</ref>韓版為了要平衡一到四區、薑餅人、寵物、寶物、道具以及隨機寶箱的分數與效果，於是在第八季進行相關的調整<ref name="b">[http://blog.naver.com/bravecookie/220673146954 쿠키런 밸런스가 조정되었습니다.] {{webarchive|url=https://web.archive.org/web/20160815072530/http://blog.naver.com/bravecookie/220673146954 |date=2016-08-15 }} naver.com論壇，2016-04-03[2016-05-10]。{{ko}}</ref>。LINE版本於2016年6月28日也隨之正式改版。

===遊戲角色與寵物===
====原版本====
為遊戲玩家使用的角色，從最高階級L>S>A>B>C最低階的薑餅人或寵物，而每種薑餅人或寵物都能向上升級，目前最高提升至8級，而每種薑餅人各有不同的性能及優勢，薑餅人的取得，是由在遊戲中達到任務目的，特定的薑餅人達成解鎖時，即可從鑽石購買，轉為銀幣購買。<ref name="a"/>寵物搭配上特定的薑餅人，也會有不同的額外效果，得宜使用薑餅人與寵物、寶物的搭配，可提高遊戲得分及躲避障礙物的能力。<ref name="a"/>
====烤箱大逃亡====
《跑跑薑餅人－烤箱大逃亡》為遊戲玩家使用的角色，從傳奇、超級稀有、稀有、普通的薑餅人或寵物，而每種薑餅人都能向上升級，升的越高，體力也會上升；每隻寵物因培育等級而提升該寵物能力的效果。但每隻薑餅人都有限跑次數，等級越低，限跑次數越多次，跑完後就要休息，等時間一到，才能繼續跑，而每種薑餅人各有不同的性能及優勢，薑餅人的取得，是等到達該等級時，特定的薑餅人達成解鎖時，即可從寶箱中取得。<ref name="a"/>

===寶物===
原版本的寶物也分成四種階級，其階級分成紫字S>黃字A>藍字B>灰字C，獲得原版本的寶物方法可從寶箱、活動、餅乾滿等、寵物滿級等等......。<ref name="a"/>大逃亡版本的寶物分成優良跟卓越等級，其最高等為12等，後來推出寶物鍍金可增加額外的寶物效果，可從活動與專用寶物寶箱中獲得。每種寶物皆有不同的效果，得宜使用薑餅人與寵物、寶物的搭配，可提高遊戲得分及躲避障礙物的能力。

==活動介紹==
*邀請活動（邀請好友共同玩樂，邀請到一定的人數，可以獲的錢幣/寵物/餅乾/鑽石等...）
*周年活動（周年活動常有集字活動，集完可領取獎勵）
*餅乾主題任務活動（例如：勇敢餅乾的挑戰/天使餅乾的挑戰/奶油泡芙餅乾的挑戰等...）
*限時購買活動（限時購買指定寵物，需用鑽石購買）
*每日登錄活動（期間內登錄遊戲會送獎品）
*寶物抽中機率增大（抽寶箱稀有率增多）
*限時購買寵物活動（期間內可以利用鑽石購買特定寵物）
*集字活動（遊戲中集字獲得獎勵）
*10%經驗與錢幣的加成（LINE版本為周五上午十點至周一16：00點；韓版為每個月的某個時段閣一天加成）
*Bingo活動（自烤箱大逃亡所發出的活動，集滿、連成一線或完成任務時可獲得獎勵）
*節日或內部運動會活動（為配合節日或內部運動會所出的活動，如：《跑跑薑餅人: 烤箱大逃亡》冬季運動會！<ref>[https://www.niusnews.com/=P0v3hn47 《跑跑薑餅人: 烤箱大逃亡》冬季運動會！巨大熊熊果凍重磅登場] {{webarchive|url=https://web.archive.org/web/20180320110656/https://www.niusnews.com/=P0v3hn47 |date=2018-03-20 }}，妞新聞，2018-01-31[2018-03-20]。</ref>）
*火力全開日（星期日、四：餅乾無限奔跑次數；星期一：120%獎盃；星期二、五：交棒免費；星期三：150%錢幣；星期六：坑洞救出免費）
