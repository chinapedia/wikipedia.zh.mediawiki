'''奚晓明'''（{{bd|1954年|6月|}}），[[江苏|江苏]][[常州|常州]]人。1972年1月参加工作，1975年9月加入中国共产党，[[北京大学|北京大学]]法律系经济法专业毕业，研究生学历，法学博士学位。原[[最高人民法院|最高人民法院]]副院长、[[中华人民共和国|中华人民共和国]]二级大法官。2015年8月因涉嫌违纪违法被中央纪委审查，后被开除党籍和公职。2017年2月因受贿罪获判无期徒刑。

== 生平 ==
早年为[[沈阳市|沈阳市]]和平区公安局警察；后毕业于[[吉林大学|吉林大学]]法律系，1991年9月至1992年9月在英国[[伦敦大学|伦敦大学]]学习。历任[[最高人民法院|最高人民法院]]研究室书记员、审判员，经济审判庭副庭长，审判委员会委员、民事审判第二庭庭长。2004年6月起担任中国[[最高人民法院|最高人民法院]]党组成员、副院长、审判委员会委员。二级大法官<ref>{{cite news|title=撒贝宁专访高法副院长奚晓明|url=http://news.sina.com.cn/c/p/2007-03-14/173312517522.shtml|accessdate=2013-07-14|newspaper=新浪}}</ref><ref>{{cite news|title=审判工作要用“国事”眼光 ——最高人民法院副院长奚晓明谈国际金融危机的司法应对|url=http://www.chinacourt.org/article/detail/2009/06/id/361562.shtml|accessdate=2013-07-14|newspaper=中国法院网}}</ref>。

2015年7月12日，[[中共中央纪委|中央纪委]][[中华人民共和国监察部|监察部]]网站公布消息指奚晓明涉嫌严重违纪违法，接受组织调查<ref>{{cite news|title=最高人民法院副院长、党组成员奚晓明涉嫌严重违纪违法接受组织调查|url=http://www.ccdi.gov.cn/xwtt/201507/t20150712_59136.html|accessdate=2015-07-12|newspaper=中央纪委监察部}}</ref>。奚晓明的妻子吴建伟和儿子奚众也被带走调查，办案部门在其儿子家中抄出三亿现金。

奚晓明是自[[黄松有|黄松有]]之后第二位因经济问题落马的[[最高人民法院|最高人民法院]]副院长，他的落马很可能跟山西煤商[[张新明|张新明]]的一起百亿[[矿山|矿山]]争夺案有关<ref>[http://news.163.com/15/0713/08/AUD0OLT40001124J.html 最高法副院长奚晓明被查 疑涉山西前首富案] {{webarchive|url=https://web.archive.org/web/20150713083055/http://news.163.com/15/0713/08/AUD0OLT40001124J.html |date=2015-07-13 }}</ref>。2015年9月29日，中纪委通报奚晓明被双开，移送司法机关<ref>{{cite web |url = http://www.ccdi.gov.cn/xwtt/201509/t20150929_62886.html |title = 最高人民法院原副院长、党组成员奚晓明严重违纪被开除党籍 |accessdate = 2015年9月29日 |publisher = 中央纪委监察部网站}}</ref>。2017年1月10日，[[天津市第二中级人民法院|天津市第二中级人民法院]]一审公开开庭审理，天津市人民检察院第二分院指控：奚晓明利用职位便利，为相关单位和个人在案件处理、公司上市等事项上提供帮助，直接或间接接受财物共计折合人民币1.14596934亿元<ref>{{cite news|title=最高法原副院长奚晓明受贿案开庭 涉案1.14多亿元|url=http://news.163.com/17/0110/16/CAECJ6UV0001899O.html|accessdate=2017-01-10|agency=央视新闻|date=2017-01-10}}</ref>。2017年2月16日，[[天津市第二中级人民法院|天津市第二中级人民法院]]一审宣判，奚晓明犯受贿罪，判处无期徒刑，剥夺政治权利终身，并处没收个人全部财产。奚晓明当庭表示服从判决，不会上诉<ref>{{cite web |url = http://www.ccdi.gov.cn/yw/201702/t20170216_94095.html |title = 最高人民法院原副院长奚晓明受贿案一审宣判 |accessdate = 2017年2月16日|date = 2017年2月16日 |publisher = 中央纪委监察部网站}}</ref>。

==家族==
*配偶：吴建伟，原最高人民法院经济审判庭庭长。
*儿子：奚众，利用奚晓明在最高法院的权势，在深圳开办律师事务所，大量接手特大经济纠纷案件和其它案件，收取巨额代理费用，然后通过奚晓明或明或暗地打招呼、暗示或影响，干预省、地、市三级法院的审判，使得本不可能胜诉的案件得到如愿判决。

==评价==
2015年8月20日，中共中央政法委书记[[孟建柱|孟建柱]]在全国律师工作会议上，就奚晓明被立案调查表示：“作为在最高法院工作33年的老法官，奚晓明却同个别违法律师、司法掮客、不法商人相互勾结，收受巨额贿赂，这是司法界的耻辱<ref>{{cite web |url = http://news.sohu.com/20150821/n419378556.shtml |title = 中央政法委书记谈奚晓明案：这是司法界的耻辱 |accessdate =  2017-02-16|date = 2015-08-21|publisher =新京报 }}</ref>。”

2016年3月13日，最高人民法院院长[[周强|周强]]在宣读最高人民法院工作报告时指出奚晓明案“影响恶劣、教训深刻”<ref>{{cite web |url = http://news.21cn.com/caiji/roll1/a/2016/0313/13/30756341.shtml |title = 周强谈“奚晓明案”：影响恶劣、教训深刻 |accessdate = 2017-02-16 |date = 2016-03-13 |publisher = 中国新闻网 |archive-url = https://web.archive.org/web/20170218063317/http://news.21cn.com/caiji/roll1/a/2016/0313/13/30756341.shtml |archive-date = 2017-02-18 |dead-url = yes }}</ref>。
