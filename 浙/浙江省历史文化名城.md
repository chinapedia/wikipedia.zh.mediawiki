'''浙江省历史文化名城'''是由浙江省文物局、浙江省建设厅进行评审，并由[[浙江省人民政府|浙江省人民政府]]正式对外公布的省级历史文化名城，通常与[[浙江省历史文化名镇|浙江省历史文化名镇]]、[[浙江省历史文化名村|浙江省历史文化名村]]、[[浙江省历史文化街区|浙江省历史文化街区]]一同公布。截至2017年7月，浙江省已经公布了17处省级历史文化名城，其中7处已列入[[中国历史文化名城|中国历史文化名城]]，尚有10处：

*第一批（1991年10月公布，共6个）：<del>[[温州市|温州市]]</del>（已升格为国家级历史文化名城）、[[余姚市|余姚市]]、<del>[[湖州市|湖州市]]</del>（已升格为国家级历史文化名城）、[[舟山市|舟山市]]、<del>[[临海市|临海市]]</del>（已升格为国家级历史文化名城）、<del>[[衢州市|衢州市]]</del>（已升格为国家级历史文化名城）<ref>{{cite web |url=http://www.zhoushan.cn/rdzz/gjjhhzhkf/hywh/201206/t20120621_546674.htm|title=《省级历史文化名城是定海还是舟山？》|accessdate=2016-11-03|work=董瑞兴|publisher=舟山日报}}</ref>。

::增补一：（1995年6月公布，共1个）：<del>[[金华市|金华市]]</del>（已升格为国家级历史文化名城）<ref>{{cite web |url=http://www.jhnews.com.cn/jhwb/2010-03/11/content_946986.htm|title=《名城文物保护工作大事记》|accessdate=2016-11-03|work=|publisher=金华新闻网}}</ref>。

::增补二：（1996年8月公布，共1个）：[[东阳市|东阳市]]<ref>{{cite web |url=http://www.jhnews.com.cn/jhwb/2010-03/11/content_946986.htm|title=《1996年大事记》|accessdate=2016-11-03|work=|publisher=东阳市人民政府}}</ref>。

*第二批（2000年2月公布，共6个）：<del>[[嘉兴市|嘉兴市]]</del>（已升格为国家级历史文化名城）、[[兰溪市|兰溪市]]、[[天台县|天台县]]、[[松阳县|松阳县]]、[[瑞安市|瑞安市]]、<del>[[龙泉市|龙泉市]]</del>（已升格为国家级历史文化名城）<ref>{{cite web |url=http://www.110.com/fagui/law_255110.html|title=《浙江省人民政府批转省文物局、省建设厅关于公布第二批省级历史文化名城、历史文化保护区名单报告的通知》|accessdate=2016-11-03|work=|publisher=110法律法规}}</ref>。

::增补一：（2010年11月公布，共1个）：[[海宁市|海宁市]]<ref>{{cite web |url=http://hnnews.zjol.com.cn/hnnews/system/2010/11/16/012900403.shtml|title=《我市被命名为“省级历史文化名城”》|accessdate=2016-11-03|work=|publisher=海宁新闻网}}</ref>。

::增补二：（2014年7月公布，共1个）：[[丽水市|丽水市]]<ref>{{cite web |url=http://www.gzgp.org.cn/a/difangzhengfuwenjian/20141005/4780.html|title=《浙江省人民政府关于丽水市列为省历史文化名城的批复》|accessdate=2016-11-03|work=|publisher=国资教育培训网}}</ref>。

::增补三：（2015年6月公布，共1个）：[[平阳县|平阳县]]<ref>{{cite web |url=http://www.zj.gov.cn/art/2015/7/1/art_12460_240447.html|title=《浙江省人民政府关于平阳县列为省历史文化名城的批复》|accessdate=2016-11-03|work=|publisher=浙江省人民政府}}</ref>。
