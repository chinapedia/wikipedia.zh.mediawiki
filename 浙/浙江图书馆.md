'''浙江图书馆'''创建于清光绪二十六年（1900），是中国创办最早的省级公共图书馆之一，现为[[国家一级图书馆|国家一级图书馆]]、[[全国古籍重点保护单位|全国古籍重点保护单位]]，建筑面积4.2万平方米，馆藏文献673.44万册（件）<ref>{{cite web|url=http://www.zjlib.cn/sygk/index.htm?li_id=4|title=概况---浙江图书馆|publisher=浙江图书馆|language=zh-cn}}</ref>。

== 历史 ==
清光绪二十六年（1900）11月，受维新思想的影响，杭州士绅[[邵章|邵章]]、[[胡焕|胡焕]]等呈请杭州知府[[朱启凤|朱启凤]]核准，在城东菜市桥畔东城讲舍旧址创办'''杭州藏书楼'''，向当地士子免费阅借，创办时藏书718种、9499册，并在1902年刊行了《杭州藏书楼书目》向社会公示<ref name="zhounian">{{cite web|title=《浙江图书馆建馆110周年主题展》|url=http://www.zjlib.cn/album.html|website=浙江图书馆官网|publisher=浙江图书馆工作人员|accessdate=2016-09-16}}</ref>{{rp|第2－3页}}<ref>{{cite journal zh |author=程小澜|title=《澎湃新知浙江潮 琅嬛清秘文澜阁——浙江图书馆精神百年回顾与展望》|journal=图书馆|date=2004年6月}}</ref>。光绪二十九年（1903）浙江巡抚[[聂缉椝|聂缉椝]]、行省[[翁曾桂|翁曾桂]]与学政[[张亨嘉|张亨嘉]]将杭州藏书楼改为官办，更名为'''浙江藏书楼'''，藏书扩大到3.5万册，隶属浙江提学使司管理，并拨款在城中丰乐桥建大方伯里馆舍<ref group="注">位于今解放路（民国时称中正街）葵巷附近，后改称新民分馆，1963年划归杭州图书馆。</ref>，光绪帝赐匾“敦彝牖慧”。据《浙江藏书楼阅书借书章程》，时读者无论身份，凡“志在通知古今中外者”，均准入楼阅书借书<ref group="注">《浙江藏书楼阅书借书章程》第一条：“书楼之设，原以广开民智，造就人才。无论进士、举人、贡监、生童，但志在通知古今中外者，均准入楼阅书借书”。</ref>。《章程》还涵盖了图书馆开放时间，馆内的服务状况等内容<ref name="zhounian"/>{{rp|第4－5页}}。之后在聂缉椝望扩大馆藏规模的奏请以及光绪帝的批准下，各省局的书片陆续发往了藏书楼<ref name="zhounian"/>{{rp|第7页}}。宣统元年（1909）清廷[[预备立宪|预备立宪]]，在各省设图书馆，经御批准奏，浙江巡抚[[增韫|增韫]]将浙江藏书楼与[[浙江官书局|浙江官书局]]合并，改建为浙江图书馆，并请拨文澜阁旁空地建新馆舍（即孤山馆舍）。宣统三年（1911）7月经[[浙江省咨议局|浙江省咨议局]]议决，将仅存的江浙三馆之一的[[文澜阁|文澜阁]]所藏《[[四库全书|四库全书]]》及其他阁藏图书拨归浙江图书馆<ref name="历史">{{cite journal zh |author=吴育良|title=《浙江图书馆的沿革述略（1900年~1949年）》|journal=河南图书馆学刊|date=2012年10月}}</ref>。

民国元年（1912）夏建成孤山馆舍（白楼），元年2月25日由[[钱恂|钱恂]]出任馆长，期间他为保护《四库全书》将《全书》迁到了红楼，在浙江教育司司长[[沈钧儒|沈钧儒]]在政务会上的提议下红馆成为了浙江图书馆的分馆<ref name="zhounian"/>{{rp|第16 - 17页}}，次年（1913）浙江图书馆改称'''浙江省立图书馆'''，并将白楼作为总馆并正式开馆<ref name="zhounian"/>{{rp|第9页}}，第四年（1915）杭州图书馆与北京两地开展了历时8年对以《四库全书》为底本的286卷书籍的补抄<ref name="zhounian"/>{{rp|第17页}}，图书馆还选送了《浙江通志》等十种木印书籍赴[[巴拿马太平洋博览会|巴拿马太平洋博览会]]展出<ref name="zhounian"/>{{rp|第28页}}，五年（1916）改称'''浙江公立图书馆'''，十六年（1927）复名浙江省立图书馆。自1930年起浙江图书馆在杭州城区内开设5处图书流通点，使用了自主设计的便捷流通书架，并在航船[[江兴轮|江兴轮]]上开展了图书流通业务<ref name="zhounian"/>{{rp|第20页}}。民国二十年（1931）建成大学路馆舍，由[[蔡元培|蔡元培]]题写馆名，浙江省立图书馆与浙江图书馆两个馆名并用，时有藏书24万余册，其中1915年-1923年和1923年-1925年先后组织文澜阁《四库全书》乙卯补抄和癸亥补抄，使其大体恢复旧观，并组织传抄嘉业堂藏书楼所藏《永乐大典》原本44册。至1935年藏书达32万余册，成为除国立[[北平图书馆|北平图书馆]]外藏书最多的省立图书馆。[[抗日战争|抗战期间]]1937年12月1日图书馆迁址到[[建德市|建德]]、次年至[[永康|永康]]、41年至[[丽水|丽水]]，在此期间大学路馆舍的钢铁书架遭日军劫掠，地下室也被其当作马厩<ref name="zhounian"/>{{rp|第43页}}。馆藏的《四库全书》自迁到龙泉后一直向西迁至重庆。浙江图书馆与位于瑞安的玉海楼搬移到[[青田县|青田县]]的南田，善本则密藏在[[庆元县|庆元县]]，玉海楼的藏书也尽归浙江图书馆<ref name="zhounian"/>{{rp|第49页}}，1945年底迁回杭州<ref name="历史" />。1932年1月学者[[陈训慈|陈训慈]]担任馆长，他提出延长开馆时间，在省内开办通信借书等多项便民提议<ref name="zhounian"/>{{rp|第31页}}。陈训慈还多次在馆内举办展览会与学术讲座<ref name="zhounian"/>{{rp|第34页}}。

1951年[[嘉业堂藏书楼|嘉业堂藏书楼]]并入，并获赠[[蒋光焴|蒋光焴]]衍芬草堂特藏共计9656册书籍，1953年再次定名浙江图书馆，1998年建成曙光路新馆，2000年新馆正式开馆并作为总馆使用至今<ref name="简介" /><ref>{{cite web|url=http://www.hangzhou.gov.cn/main/zjhz/hzsz/citymark/417/T83268.shtml|title=《杭州市志 第七卷·文化艺术篇 第七章·图书》|accessdate=2015-06-07|work=|publisher=杭州市政府|deadurl=yes|archiveurl=https://web.archive.org/web/20150610050441/http://www.hangzhou.gov.cn/main/zjhz/hzsz/citymark/417/T83268.shtml|archivedate=2015-06-10}}</ref>。
　

== 馆舍 ==
[[File:Zhejiang_Library_in_Gu_Hill_01_2017-12.jpg|thumb]]
[[File:Zhejiang_Library_in_Daxue_Road_17_2015-01.JPG|thumb]]
该馆现设有馆舍四处，分别为[[杭州市|杭州市]]曙光路总馆（[[西湖区_(杭州市)|西湖区]]曙光路73号、[[宝石山|宝石山]]北麓）、孤山路古籍部（西湖区孤山路28号，西湖[[孤山_(西湖)|孤山]]南麓）、大学路馆舍（[[上城区|上城区]][[大学路_(杭州)|大学路]]102号）和[[湖州市|湖州市]]嘉业堂藏书楼（[[南浔区|南浔区]][[南浔镇|南浔镇]]）。作为总馆的曙光路馆舍建筑面积3.4万平方米，可藏书450万册。建筑坐南朝北，自北向南顺栖霞岭的山势逐步增高，以2-3层为主，局部4层，采用钢筋混凝土框架结构和筏基基础，总体布局呈品字型，东部、东南部为阅览区，西部、西北部为学术交流区，中南部为书库、采编部和业务工作管理区<ref>{{cite journal zh |author=应长兴、林祖藻|title=《浙江省图书馆的建筑特色》|journal=南方建筑|date=2002年4月}}</ref>。

孤山馆舍现用于收藏1949年5月之前的古籍文献和民国图书报刊，建筑由白楼、红楼和青白山居组成，白楼为普通古籍书库和读者阅览室，红楼为古籍编目室和修复室，青白山居为珍本书库。白楼、红楼均以外墙颜色命名，其中白楼为近代中国最早的公共图书馆专用建筑之一，建于1911年5月至1912年夏，为两层西式建筑。红楼位于白楼西侧，原为清政府迎接德国皇太子威廉来杭而建造的国宾馆，建于清光绪三十二年（1906），亦为两层西式建筑，平面为正方形，四周设有回廊，廊柱间以拱形墙饰连接<ref>{{cite web |url=http://hzdaily.hangzhou.com.cn/hzrb/html/2011-05/19/content_1068456.htm|title=《孤山路28号，一座城市的精神轨迹》|accessdate=2015-06-07|work=韩斌|publisher=杭州日报}}</ref>。青白山居又称“杨虎大楼”，建于孤山之巅，原为上海警备司令[[杨虎|杨虎]]的别墅，由[[胡钊|胡钊]]设计，建于1936年5月至次年1月，主楼为三层建筑，墙体、围栏为西式风格，屋顶则采用中式歇山顶，1953年由政府收购并归浙江图书馆使用，1954年3月起用于收藏四库全书以及[[善本|善本]]书<ref>{{cite web |url=http://hzdaily.hangzhou.com.cn/hzrb/html/2011-05/12/content_1064574.htm|title=《青白山居的静好岁月》|accessdate=2015-06-10|work=韩斌|publisher=杭州日报}}</ref>。

大学路馆舍现为阅览室，建筑由原浙江军政府都督[[汤寿潜|汤寿潜]]捐资建造，上海[[庄俊|庄俊]]建筑师事务所设计，建于1929年至1931年3月，为两层西式建筑，采用钢筋混凝土结构，平面为工字形，立面采用[[古典主义建筑|古典主义]]的纵横三段式处理，底部为宽阔高大的台基，中部采用[[多立克柱式|多立克式柱子]][[巨柱式|巨柱式]]构图，其中南立面的16根贴墙柱子将立面分隔为15间，正中3间开红色铁门，其余各间设支摘式方格玻璃钢窗，正中5间檐壁上为蔡元培所题“浙江图书馆”馆名，两侧5间檐壁上各开5个长方形的白框横窗。入内中间为大厅，左右翼和内室为阅览室和书库，屋顶为井字梁<ref>{{cite journal zh |author=傅宏明|year=2000|month=增刊|title=《浙江图书馆大学路馆舍》|journal=浙江建筑}}</ref><ref>{{cite web |url=http://hzdaily.hangzhou.com.cn/hzrb/html/2011-06/09/content_1079736.htm|title=《“浙图”大学路馆舍的时光秘境》|accessdate=2015-06-07|work=周华诚|publisher=杭州日报}}</ref>。

== 藏书和分类 ==
[[File:The_Complete_Library_of_the_Four_Treasuries_Collected_in_Wenlan_Pavilion_2015-03.JPG|thumb]]

浙江图书馆现藏书500多万册，尤以《四库全书》文澜阁本和《[[古今图书集成|古今图书集成]]》著称，此外也藏有敦煌经卷、宋元明刻本、稿抄本、名家批校题跋本及日本、朝鲜、越南等外国刻本<ref name="简介">{{cite web |url=http://www.zjwh.gov.cn/stgk/contnet_30_468.htm|title=《浙江图书馆》|accessdate=2015-06-07|work=|publisher=浙江省文化厅}}</ref>。馆藏古籍集中了数十家私家藏书楼旧藏，如南浔嘉业堂、龙游[[寒柯堂|寒柯堂]]、上虞[[枕湖楼|枕湖楼]]、四明[[墨懈楼|墨懈楼]]、海宁[[衍芬草堂|衍芬草堂]]、余姚梁弄[[五桂楼|五桂楼]]和慈城[[抹云楼|抹云楼]]等<ref>{{cite journal zh |author=丁红、程小澜|title=《浙江图书馆的历史文献收藏及其特色》|journal=图书馆工作与研究|date=2002年1月}}</ref>，此外在嘉业堂藏书楼还藏有古籍版片（[[雕版印刷|雕版印刷]]古籍之后的原版留存）15万片<ref>{{cite web |url=http://zjrb.zjol.com.cn/html/2014-05/06/content_2643740.htm?div=-1|title=《浙图15万张雕版“颐养天年”》|accessdate=2015-06-10|work=刘慧|publisher=浙江日报}}</ref>。

分类方面，杭州藏书楼时期按自行编制的分类法类分图书报刊，光绪二十八年（1902）刊行的《杭州藏书楼书目》将藏书分为经学、史学（掌故與地附）、性理（哲学家言附）、辞章、时务、格致、通学七个主要类目，另附设报章、图表、补遗三个类目。浙江藏书楼时期将馆藏分作古籍和新籍两类，分别按[[四部分類法|四库分类法]]分类和自拟分类体系类分，其中光绪三十三年（1907）刊行的《浙江藏书楼书目》（乙编）将新籍分为法律、政治、宗教、教育、图史、文学、文字、理学、算学、美术、杂志、工业、商业、兵书、生理、农业十六个类目。1912年6月后，馆藏分为保存（古籍中的珍善本）和观览（普通古籍及新籍）两类，古籍分为经、史、子、集、宗教、丛书六类，新籍分为文学、哲学、历史、宗教、天文、地理、博物、理化、数学、法政、农学、工学、商学、兵学、医学、美术及稗官说部十七类。浙江省立图书馆时期，1924年开始采用[[杜定友|杜定友]][[世界图书分类法|世界图书分类法]]类分中文平装书及日文书，以[[杜威十进制图书分类法|杜威十进分类法]]类分西文书，1928年夏又统一改用[[王云五|王云五]]《中外图书统一分类法》<ref>{{cite journal zh |author=吴育良|title=《民国时期的浙江省立图书馆》|journal=图书情报工作网刊|date=2012年12月}}</ref>，现状则采用[[中国图书馆分类法|中国图书馆分类法]]。

== 研究和出版 ==
浙江图书馆曾办有《浙江省立图书馆馆刊》、《文澜学报》和《图书展望》等多种刊物，其中1915年创办《浙江公立图书馆年报》，后改称《浙江图书馆报》，除报道馆内情况外，也介绍世界各国图书馆的历史与现状。1927年创办《浙江省立图书馆月刊》，后改称《浙江省立图书馆馆刊》，包括学术论著、介绍评论、书报摘要、图书馆消息、馆务纪要、统计、新书目录以及国内外图书馆概况撷要等栏目，除反映馆务、汇报进展外，也逐步扩大到对[[图书馆学|图书馆学]]研究成果的报道。此外在1935年还创办《文澜学报》季刊和《图书展望》月刊，前者旨在研究中国传统学术，以中国古代版本、目录和校勘等方面的内容为重点，后者则类似于大众性读书刊物<ref>{{cite journal zh |author=谢灼华|title=《二十世纪上半叶浙江图书馆馆刊的历史地位》|journal=图书馆研究与工作|date=2006年1月}}</ref>。

此外该馆在民国二十五年（1936）11月曾举办[[浙江文献展览会|浙江文献展览会]]，展出浙江图书文物6000余种2万余件（册），观众达7.6万人次，为中国近代史上首次也是最大的文献博览会<ref>{{cite journal zh |author=袁逸|title=《百年沧桑征典籍——浙江省公共图书馆地方文献事业百年概述（1901-2005）》|journal=图书馆研究与工作|date=2007年2月}}</ref>。
