'''秦嶺大熊貓'''（[[学名|学名]]：''{{lang|la|Ailuropoda melanoleuca qinlingensis}}''）是[[大熊猫|大熊猫]]现存的两个[[亞種|亞種]]之一，於1960年代發現，1964年[[北京师范大学|北京师范大学]][[鄭光美|鄭光美]]首次提出，[[浙江大学|浙江大学]]生命科學學院方盛國的研究小組於2005年通過[[形態學|形態學]]以及[[分子生物學|分子生物學]]方法證實<ref name="takungpao 050213">{{cite web|url=http://www.takungpao.com.hk/news/2005-2-13/ZM-364647.htm |title=秦嶺大熊貓被確認為新亞種 |publisher=大公報 |date=2005年2月13日 |accessdate=2013-12-11 |deadurl=bot: unknown |archiveurl=https://web.archive.org/web/20051124170755/http://www.takungpao.com.hk/news/2005-2-13/ZM-364647.htm |archivedate=2005年11月24日 }}</ref>，並得到[[動物分類學|動物分類學]]權威刊物《[[美國哺乳動物學雜誌|美國哺乳動物學雜誌]]》確認<ref name="people">{{cite web |url=http://env.people.com.cn/BIG5/huanbao/259/4481/4482/4019532.html |title=我國發現大熊貓新亞種：秦嶺大熊猫頭圓更像貓 |publisher=人民網 |date=2006年1月12日}}</ref>。

== 分佈 ==
秦嶺大熊猫主要在[[海拔|海拔]]1300至3000米的[[中國|中國]][[陝西省|陝西省]][[秦嶺|秦嶺]]山系南麓腹地繁衍，中心活動區域為[[佛坪國家級自然保護區|佛坪國家級自然保護區]]，由此延伸至[[洋縣|洋縣]]、[[周至|周至]]、[[太白县|太白]]、[[寧陝|寧陝]]、[[留壩|留壩]]等區域。據2004年完成的中國全國第3次大熊猫調查數據顯示，約有273隻野生成年大熊猫於秦嶺存活。<ref name="xawb">{{cite web |url=http://www.xiancn.com/gb/news/2006-02/15/content_790671.htm |title=古老新亚种――秦岭大熊猫 |publisher=西安晚报 |date=2006年2月15日 |accessdate=2013-12-11}}</ref>

== 形態 ==
秦嶺大熊猫與較常見的指名亞種——[[四川大熊猫|四川大熊猫]]（''{{lang|la|A. m. melanoleuca}}''；包括四川、甘肅繁殖地的大熊猫）於12000年前開始隔絕在不同的分佈區，出現各自的[[演化|演化]]史和[[遺傳基因|遺傳基因]]上的差異。在遺傳學上，秦嶺大熊猫的遺傳基因更接近原始熊猫，而四川大熊猫演化的速度較快。在外形上，秦嶺大熊猫有暗棕色胸斑、棕色腹毛，頭骨較小而頭圓、臼齒較大、吻（鼻樑）短、臉闊、外觀好看。這也和四川大熊猫黑色胸斑、白色腹毛，頭骨較大而頭長、臼齒較小有顯著不同。<ref name="takungpao 050213" /><ref name="people" />

== 生存困難 ==
秦嶺大熊猫以[[箭竹|箭竹]]、[[傘竹|傘竹]]為主要食物。牠們面對的生存困難，包括來自自然界的競爭，例如[[野豬|野豬]]群大量繁殖，與大熊猫爭食[[竹筍|竹筍]]<ref>{{cite web|url=http://www.mpinews.com/htm/INews/20060816/ca31636c.htm |title=陝西野豬成災影響大熊猫 |publisher=明報新聞網 |date=2006年8月16日 |accessdate=2013-12-11 |deadurl=bot: unknown |archiveurl=https://web.archive.org/web/20070930211542/http://www.mpinews.com/htm/INews/20060816/ca31636c.htm |archivedate=2007年9月30日 }}</ref>；外來樹種[[落葉松|落葉松]]蠶食竹林生長地<ref>{{cite web|url=http://www.js.xinhuanet.com/fang_tan/2004-12/20/content_3421333.htm|title=秦嶺大熊貓竹林遭外來樹種落葉松“蠶食”|date=2004年12月20日|author=王利梁娟|publisher=新華網-江蘇頻道|accessdate=2013-12-11|deadurl=yes|archiveurl=https://web.archive.org/web/20071006072313/http://www.js.xinhuanet.com/fang_tan/2004-12/20/content_3421333.htm|archivedate=2007年10月6日}}</ref>。另外，人類活動例如林場開發也破壞其棲息地。佛坪國家級自然保護區與[[中國科學院動物研究所|中國科學院動物研究所]]正建設秦嶺大熊猫野外研究基地，以便進行保育工作<ref>{{cite web|url=http://www.sn.xinhuanet.com/2006-12/08/content_8736416.htm|title=陕西佛坪将建成秦岭大熊猫野外研究基地|publisher=新华网陕西频道|date=2006年12月8日|deadurl=yes|archiveurl=https://web.archive.org/web/20150924103010/http://www.sn.xinhuanet.com/2006-12/08/content_8736416.htm|archivedate=2015年9月24日}}</ref>。
