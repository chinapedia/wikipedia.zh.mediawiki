'''秦簡公'''（{{bd|前428年||前400年||Q}}），名'''悼子'''{{NoteTag|王蘧常所著《秦史》认为悼子非秦简公真名。<ref>{{Cite book|author=王蘧常|title=《秦史》|location=上海市瑞金二路272号|publisher=上海古籍出版社|date=2000年12月|pages=第16页|ISBN=7532528154|accessdate=|url=|language=|quote=案《秦本纪》云，灵公季父悼子，是以悼子为简公名；然自桓公以来，史失其名久矣，何因忽出此凶名？疑不可信，姑附于此。}}</ref>}}，[[战国|战国]]时期[[秦国|秦国]]君主。[[秦怀公|秦怀公]]之子，[[秦昭子|秦昭子]]之弟、[[秦灵公|秦灵公]]之叔，{{NoteTag|一说秦简公为秦厉共公之子，秦怀公之弟。<ref>賀次君《史記書錄》引日本高山寺藏本《史記集解·秦本紀》（第23至25頁）：簡公，厲公子而懷公弟也。</ref><ref>司馬貞《史記索隱·秦始皇本紀》：按：本紀簡公名悼子，即剌龔公之子，懷公弟也。且《紀》及《系本》皆以為然，今此文-{云}-「靈公」，謬也。</ref><ref>《世本·诸侯》：简公名悼子，即刺龚公之子，怀公弟也。</ref><ref>《漢書·卷二十·古今人表》：秦簡公，厲公子。</ref>}}在位15年。

秦简公在位期间，被[[魏国_(战国)|魏国]]夺取[[西河郡|河西地区]]（今[[山西省|山西]]、[[陕西省|陕西]]两省间[[黄河|黄河]]南段以西地区），国力进一步衰落。

== 继位 ==
秦国自秦厉共公以后，国内国政长期由旧贵族把持，国君的废立也由他们掌握。<ref>《史记·卷五·秦本纪》：会往者厉、躁、简公、出子之不宁，国家内忧，未遑外事。</ref>前415年，秦灵公去世后，其子[[秦献公|公子師隰]]（即後來的[[秦獻公|秦獻公]]）没有能够继位。秦国国内重臣拥立在[[晋国|晋国]]的秦怀公之子、秦昭子之弟、秦灵公之叔秦简公回国继位，<ref>《史记·卷五·秦本纪》：灵公卒，子献公不得立，立灵公季父悼子，是为简公。</ref><ref name="秦始皇"/>公子连被迫流亡至魏国。<ref>《吕氏春秋·卷二十四·不苟论·当赏》：公子连亡在魏。</ref>

== 河西之战 ==
{{main|河西之战 (战国)}}
魏国自[[魏文侯|魏文侯]]继位后，政治上任用[[翟璜|翟璜]]、[[吴起|吴起]]、[[李悝|李悝]]、[[魏成子|魏成子]]等人进行改革，使魏国国势逐渐强盛。面对内外交困的秦国，魏国采取主动进攻的方式，夺取了秦国的河西地区。<ref name="秦史稿"/>

前413年，魏国大举进攻秦国，魏国上地郡守李悝在[[郑县|郑县]]（今陕西省[[渭南市|渭南市]][[华州区|华州区]]）大败秦军。{{NoteTag|林春溥的《战国纪年》认为李悝在担任上地郡守期间指挥了这场战役，见杨宽所著《战国史料编年辑证》第161页。<ref name="杨宽"/>}}<ref>《史记·卷十五·六国年表》：（秦简公）二年，与晋战，败郑下。</ref>次年，魏文侯派[[魏武侯|太子击]]包围并占领了秦国的[[繁庞|繁庞]]（今陕西省[[韩城市|韩城市]]东南），迁出当地的居民。<ref>《史记·卷十五·六国年表》：（魏文侯）十三年（应为三十四年），公子击围繁庞，出其民。</ref>

前409年{{NoteTag|关于吴起夺取秦国临晋的时间，《史记·卷十五·六国年表》及《史记·卷四十四·魏世家》记载有误，此处采纳林剑鸣所著《秦史稿》的观点，见林所著《秦史稿》第170页。<ref name="秦史稿">{{Cite book|author=林剑鸣|title=《秦史稿》|location=上海市绍兴路54号|publisher=上海人民出版社|date=1981年2月|pages=第165页、第170页|ISBN=|accessdate=|url=|language=|quote=}}</ref>}}，魏文侯任命吴起为主将，攻克秦国河西地区的[[临晋|临晋]]（即王城，今陕西省[[大荔县|大荔县]]东南）、[[元里|元里]]（今陕西省[[澄城县|澄城县]]东南）并筑城。<ref>《史记·卷十五·六国年表》：（魏文侯）十六年（应为三十七年），伐秦，筑临晋、元里。</ref>次年，吴起再次率军攻打秦国，一直打到郑县，攻克[[洛阴|洛阴]]（今陕西省大荔县西南）、[[郃阳|郃阳]]（今陕西省[[合阳县|合阳县]]东南）并筑城。<ref>《史记·卷十五·六国年表》：（魏文侯）十七年（应为三十八年），伐秦至郑，还筑洛阴。</ref><ref>《史记·卷四十四·魏世家》：西攻秦，至郑而还，筑雒阴、合阳。</ref><ref>《史记·卷六十五·孙子吴起列传》：于是魏文候以（吴起）为将，击秦，拔五城。</ref>而秦国只能退守至[[洛河|洛水]]，沿河修建防御工事，并筑[[重泉城|重泉城]]（今陕西省[[蒲城县|蒲城县]]东南的[[重泉村|重泉村]]）加以防守。<ref>《史记·卷十五·六国年表》：（秦简公）七年，堑洛，城重泉。</ref>至此，魏国全部占有原本属于秦国的河西地区，并在此设立[[西河郡|西河郡]]，由吴起担任首任郡守。<ref>《史记·卷六十五·孙子吴起列传》：文侯以吴起善用兵，廉平，尽能得士心，乃以为西河守，以拒秦、韩。</ref>

魏国占领河西地区后，成为秦国东进的心腹大患。前401年，秦简公派兵攻打魏国，到达[[阳狐|阳狐]]，但此时秦军无力穿越西河郡到达河东。{{NoteTag|《史记正义·魏世家》引《括地志》、《水经注》和《九域志》皆记载阳狐位于河东，杨宽认为此时河西地区已经完全被魏国占领，秦军无力越过西河郡到达河东，阳狐的具体位置不详，见杨宽所著《战国史料编年辑证》第202页。<ref name="杨宽"/>}}<ref>《史记·卷十五·六国年表》：（秦简公）十四年，伐魏，至阳狐。</ref>直到前330年，秦国才从魏国手中完全收复河西地区。<ref>《史记·卷五·秦本纪》：（秦惠文王）八年，魏纳河西地。</ref>

== 改革 ==
经过一系列失败后，秦简公被迫在秦国国内进行改革，主要措施有：

佩剑是诸侯国内贵族代表身份的象征，早在改革前秦国国内就有民间佩剑的风气。<ref>{{Cite book|author=白化文|title=《关于青铜剑》|location=北京市|publisher=《文物》1976年第十一期|date=1976年|pages=|ISBN=|accessdate=|url=|language=|quote=}}</ref>秦简公于前409年允许官吏佩剑，<ref>《史记·卷五·秦本纪》：简公六年，令吏初带剑。</ref>次年，又允许百姓佩剑。<ref>《史记·卷六·秦始皇本纪》：简公…其七年。百姓初带剑。</ref>

因河西之战的失败，使秦国失去[[关中平原|关中平原]]重要的产粮区，秦简公被迫于前408年在国内开始按亩征收禾稼作为租税。<ref>《史记·卷十五·六国年表》：（秦简公）七年，初租禾。</ref><ref name="杨宽"/>[[郭沫若|郭沫若]]认为这是秦国国内合法承认土地私有化的标志，也标志着秦国国内地主制度的建立。<ref>{{Cite book|author=郭沫若|title=《奴隶制时代》|location=北京市|publisher=中国人民大学出版社|date=2005年|pages=第35页|ISBN=7300062644|accessdate=|url=|language=|quote=}}</ref>[[祝中熹|祝中熹]]认为秦简公时的“初租禾”与秦献公的“户籍相伍”制度与以及[[秦孝公|秦孝公]]时的“制辕田，开阡陌”前后呼应，是秦国由村社制向[[自耕农|自耕农]]制转化过程中不可缺少的环节。<ref>{{Cite book|author=祝中熹|title=《论秦献公》|location=兰州市甘肃省博物馆|publisher=《陇右文博》2004年第二期|date=2004年|pages=|ISBN=|accessdate=|url=|language=|quote=}}</ref>

== 去世 ==
前400年，秦简公去世，葬于[[丘里_(古代地名)|丘里]]景公西{{NoteTag|《史记·卷六·秦始皇本纪》作僖公西，秦国君主中无僖公。《史记索隐·秦始皇本纪》记载景公一作僖公，依据此说法应为景公西。}}，其子[[秦惠公_(戰國)|秦惠公]]继位。{{NoteTag|《史记索隐·秦始皇本纪》引《竹书纪年》记载秦简公于前406年去世，秦敬公在位13年后传位给秦惠公，与《史记》记载不同。<ref>《古本竹书纪年·魏纪》：简公九年卒，敬公立，十二年卒，乃立惠公。</ref>}}<ref>《史记·卷五·秦本纪》：（秦简公）十六年卒，子惠公立。</ref><ref name="秦始皇">《史记·卷六·秦始皇本纪》：简公从晋来。享国十五年。葬僖公西。生惠公。</ref>
