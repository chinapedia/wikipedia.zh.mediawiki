'''秦大士故居'''又称'''大夫第'''，位于[[中国|中国]][[江苏省|江苏省]][[南京市|南京市]][[秦淮区|秦淮区]]武定桥东、长乐路57、59号，最初为明[[崇祯|崇祯]]时大学士[[何如宠|何如宠]]的宅第，清[[乾隆|乾隆]]时南京籍状元、侍讲学士[[秦大士|秦大士]]致仕返乡后购入居住，后成为大杂院，2001年由南京市政府投资迁出其中住户并修缮，2007年6月作为展览馆对外开放<ref>{{cite web |url=http://js.xhby.net/system/2007/06/28/010074936.shtml|title=《秦状元府修缮完毕今日开放》|accessdate=2014-08-09|work=|publisher=南京晨报}}</ref>。建筑原有三路十进（一说九进），现有东西两路共七进<ref>{{cite web |url=http://news.sina.com.cn/c/cul/2006-06-15/06319210932s.shtml|title=《状元秦大士故居8月底亮相》|accessdate=2014-08-09|work=|publisher=南京日报}}</ref>，各进面阔均为三间。其中东路三进，为原建筑群中轴，依次为门厅（新建）、正厅秋田堂和后楼东山楼，正厅建于清初，柱梁粗大，雕花精美；西路四进，前为门厅，后三进均为住宅，北侧原有花园瞻园<ref>{{cite web|url=http://dfz.nanjing.gov.cn/17629/17632/200907/t20090704_985081.html|title=《秦大士故居》|accessdate=2014-08-09|work=|publisher=南京地方志网|deadurl=yes|archiveurl=https://web.archive.org/web/20140810142335/http://dfz.nanjing.gov.cn/17629/17632/200907/t20090704_985081.html|archivedate=2014-08-10}}</ref>。
