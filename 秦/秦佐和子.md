'''秦 佐和子'''（{{bd|1988年|9月14日||}}），是[[日本|日本]]女性[[聲優|聲優]]，曾是[[日本|日本]]女性偶像團體[[SKE48|SKE48]]Team KII的前成員。[[大阪府|大阪府]]出身。

== 來歷 ==
=== SKE48 ===
2009年11月1日，秦在总人数2,396名的甄选会中获选，成为SKE48 3期生的13人之一<ref>[http://sp.ske48.co.jp/gra_member_JWiWiqXzXY.html SKE48を卒業するメンバー9人の軌跡を紹介!!] - SKE48 Mobile上的同期生小木曽汐莉、上野圭澄的合格日記載、2013年4月13日</ref>。在同月的14日，她与其余的合格者在于[[SUNSHINE_SAKAE|SKE48剧场]]举办的AKB48《[[RIVER|RIVER]]》剧场盘握手会中初次披露。在12月25日舉行的SKE48演唱會[[名古屋一揆|名古屋一揆]]中秦初次登场，正式加入了SKE48。

2010年2月16日，秦於研究生的首次公演中演出，正式在劇場中出道。12月6日，與2期的研究生[[阿比留李帆|阿比留李帆]]以及同期的[[後藤理沙子|後藤理沙子]]、[[矢方美紀|矢方美紀]]一起升格至Team KII，成為了正式成員。隔年的12月8日，秦于AKB48的衍生动漫《[[AKB0048|AKB0048]]》聲優甄选中入選，擔任主要人物神埼鈴子（{{lang|ja|かんざき すずこ}}）的配音<ref name="production_news2">{{Cite web|url=http://www.oricon.co.jp/news/music/2004688/full/|script-title= ja:AKB48、初のTVアニメはSF超大作　「声優選抜」決定で概要明らかに|publisher=ORICON STYLE|date=2011-12-14|language=ja}}</ref><ref>{{ja}}[http://natalie.mu/music/news/61195 アニメ「AKB0048」公開オーディションで声優選抜9名決定]</ref>，是秦第一次为动漫配音。

2011年6月9日，秦出席了[[AKB48第22张单曲选拔总选举|AKB48第22张单曲选拔总选举]]开票活动，在选举中秦得到了第33名，进入了Under Girls，首次进入了总选举圈内<ref>{{cite web|url=http://headlines.yahoo.co.jp/hl?a=20110609-00000527-sanspo-ent|archiveurl=https://web.archive.org/web/20110613013142/http://headlines.yahoo.co.jp/hl?a=20110609-00000527-sanspo-ent|archivedate=2011-06-13|language=ja|script-title=ja:ＳＫＥからは玲奈１０位、珠理奈が１４位に！／第３回ＡＫＢ総選挙|work=サンケイスポーツ|date=2011-06-09|accessdate=2011-06-25|deadurl=yes}}</ref><ref>{{cite web|url=http://mantan-web.jp/2011/06/11/20110611dog00m200002000c.html |language=ja|script-title= ja:AKB48総選挙 ：初ランクインにNot yet・横山、“フレッシュレモン”市川ら9人 |work=每日新聞 |date=2011-06-11 |accessdate=2011-06-25|archiveurl=http://archive.is/jg0E|archivedate=2012-07-11}}</ref>，她发表感言时也激动得痛哭，并感谢了一直支持着自己的粉丝。

2012年6月6日，秦在[[AKB48第27張單曲選拔總選舉|AKB48第27張單曲選拔總選舉~粉絲選擇的64個議席~]]得到第25位,成為Under Girls，为当届选举中三期生中排名最高，秦也对此感到十分吃惊<ref>{{cite web|language=ja|url=http://www.oricon.co.jp/news/music/2012647/full/ |script-title=ja:【第4回AKB総選挙】松井珠理奈、姉妹グループ過去最高9位　SKEが大躍進 |publisher=[[Oricon|Oricon]] |date=2012-06-07 |accessdate=2012-06-13}}</ref><ref>{{cite web|url=http://www.nikkansports.com/entertainment/akb48/news/p-et-tp0-20120607-963509.html |script-title=ja:Ｗ松井だけじゃない！ＳＫＥ大躍進 |publisher=nikkansports.com |date=2012-06-07 |language=ja|accessdate=2012-06-07}}</ref><ref>{{cite web|url=http://www.oricon.co.jp/news/deview/2012778/ |script-title=ja:15議席を獲得！　汗の量で負けないSKE48が総選挙で大躍進|language=ja |publisher=[[Oricon|Oricon]] |date=2012-06-09 |accessdate=2012-06-13}}</ref>。

4月26日，秦等9名《[[AKB0048|AKB0048]]》聲優選拔成員組成了新的衍生團體「[[NO_NAME_(組合)|NO NAME]]」<ref>[http://akb0048.jp/news/ AKB0048 官方网站]</ref><ref>{{Cite news|date=2012-04-26|url=http://www.oricon.co.jp/news/music/2010792/full/|script-title=ja:AKBまゆゆら“声優選抜”が新ユニット「NO NAME」結成！アニメ主題歌デビュー |language=ja|newspaper=ORICON STYLE|publisher=[[Oricon|Oricon]]|accessdate=2012-06-30}}</ref>，並演唱此作品片頭曲《[[關於希望|關於希望]]》和片尾曲《夢想不斷重生》<ref>{{ja}}[http://news.mynavi.jp/news/2012/04/27/003/ 「TVアニメ『AKB0048』、主題歌は声優選抜メンバーのユニット"NO NAME"が担当」] - 2012年04月27日</ref>。

2013年3月7日，秦在SKE48 Team KII剧场公演中宣布将于3月末从SKE48毕业，4月开始进入声优学校学习<ref>{{Cite news|url=http://www.sponichi.co.jp/entertainment/news/2013/03/07/kiji/K20130307005343210.html|language=ja|script-title=ja:SKE48秦佐和子、3月いっぱいで卒業　4月から声優学校へ|newspaper=[[スポーツニッポン|スポーツニッポン]]|publisher=スポーツニッポン新聞社|date=2013-03-07|accessdate=2013-03-07}}</ref>，未来将专注于声优的工作。3月29日，在SKE48劇場举办了秦的畢業公演<ref>[http://www.nikkansports.com/entertainment/akb48/news/f-et-tp0-20130329-1104795.html SKE48秦佐和子ラストステージ] - 日刊スポーツ（2013年3月29日）<br />[http://natalie.mu/music/news/87717 SKE48秦佐和子、卒業公演で約束「必ず声優になります」] - ナタリー （2013年3月30日）</ref>。5月6日，秦参加了在SKE48單曲《巧克力的奴隸》劇場盤的個別握手会，此活动是秦在SKE48的最終活動。
=== 职业声优 ===
2013年7月10日，秦开始在声优杂志《{{link-ja|声优Paradise|声優パラダイスR}}》第18话中连载了新栏目“走吧！Voice Para Production !!”<ref>[https://twitter.com/seiyu_paradise/status/345828947629584384　秦佐和子さんの連載も開始～～! 声優パラダイスvol.18 お楽しみに!]-声優パラダイス公式（2013年06月15日）</ref>。

2014年5月11日，秦正式自声优学校毕业。7月18日，秦在互聯網廣播{{link-ja|ListenRadio|ListenRadio}}中宣布将开始自己的周播广播节目“如果公寓里有多多的声优居住的话...”（{{lang|ja|もしも声優がいっぱい住んでいるマンションがあったら}}）<ref>[https://www.youtube.com/watch?v=fKe6LO7JNGI　「もしも声優がいっぱい住んでいるマンションがあったら」!!　7月21日（月・祝）のイベントでMCを務める 三宅麻理恵さんと秦佐和子さんから動画メッセージ ]-Youtube ZERO-A Official Channel （2014年7月18日）</ref>。

2015年4月3日，秦被宣布将主役在7月开播动漫《{{link-ja|VENUS PROJECT -CLIMAX-|VENUS PROJECT -CLIMAX-}}》，并在动漫中声演“原杰里科”<ref>[http://venusproject.blog.jp/archives/26278867.html　TVアニメ版の主演キャストが決定！秦佐和子・飯田里穂・早瀬莉花！]-VENUS PROJECT公式サイト（2015年04月03日）</ref>。10月1日，秦宣布自己正式加入了由声优为主的经纪公司[[Mausu_Promotion|Mausu Promotion]]。

== 人物 ==
*公演口號為「'''被凝視著就會難為情的，這樣子害羞的我的這份心情，傳達到了嗎？ 我是暱稱しゃわこ的秦佐和子！'''」。
*憧敬的聲優：[[水樹奈奈|水樹奈奈]]、[[堀江由衣|堀江由衣]]，在電視音樂節目中常提及
*非常喜歡聲優，在加入SKE48之前，曾經在聲優訓練所上了一年半左右的課。<ref>SKE48 目標是升格! 研究生BLOG 2010年4月16日的本人記事</ref>
*是個非常害羞的人，有著只要感到難為情的話，便會將臉朝下，並且用瀏海將臉給遮住的習慣。首次參加[[週刊AKB|週刊AKB]]的突擊測驗就連續答對許多題目，並數次成為模範解答，獲得了光榮的第一名，但是本人卻小聲的說「請剪掉」以及「請不要再拍我了」並且將臉遮掩了起來。<ref>『週刊AKB』2011年2月4日・11日放送</ref>
*為[[寶塚歌劇團|寶塚歌劇團]]的超級粉絲，特別是[[宙組|宙組]]的男役[[蘭寿とむ|蘭壽とむ]] ，以前也曾經考慮要加入[[寶塚歌劇團|寶塚歌劇團]]成為男役，但是由於當時（國中三年級）身高不足而放棄了。<ref>SKE48 目標是升格! 研究生BLOG 2010年6月24日的本人記事</ref>
*十分不擅長使用手機打字，於是手機的簡訊回復都相當簡短。<ref>SKE48 目標是升格! 研究生BLOG 2010年9月29日的本人記事</ref>
*擁有7台[[PlayStation_Portable|PlayStation Portable]]，原因是4種型號都想要收集以及不想一直更換UMD，因此目前3000型的兩台，一台為[[魔物獵人|魔物獵人]]用、一台為其他遊戲使用、是個意外地怕麻煩的人。<ref>『週刊AKB』2011年4月22日放送分</ref>
*雖然基本上是個無論在哪都可以睡著的人，除了在門開著的房間以外。<ref>SKE48 目標是升格! 研究生BLOG 2010年12月4日的本人記事</ref>
*為「二次元同好會」的第12號會員兼會長,副會長則是[[松井玲奈|松井玲奈]]，畢業後由[[中西優香|中西優香]]繼承會長一職<ref>出自本人Google+</ref>
*「偶像研究會」的第7號會員
*小時候總是要爬山上下學<ref>ネ申テレビ番外编～SKE48学院 修学旅行～本编</ref>
*非常喜歡咖哩飯,但很少在成員面前吃<ref>アナアナ商會 100915</ref>
*2013年宣布三月底於SKE48畢業，目的是進入聲優學校
