'''秦志暉'''（{{bd|1983年|12月}}），網名'''雁度千山'''、'''秦火火'''，是涉嫌「網路造謠」當事者之網路暱稱，本籍[[湖南省|湖南省]][[衡南縣|衡南縣]]香花村。<ref name="fish " >{{cite web |url=http://ent.163.com/13/0821/07/96PMSU4V00032DGD.html?f=jsearch |title=媒体起底秦火火：曾造谣攻击罗援、张海迪、雷锋等人 |accessdate=2013-08-21  |publisher= 网易|work=京华时报 |quote =|language=zh-hans}}</ref><ref>{{cite web |url=http://finance.huanqiu.com/data/2013-08/4284303.html |title=沙晓乐：“秦火火”们何以“谣翻中国”？ |accessdate= 2013-08-26  |publisher=环球网  |work=中国经济网 |quote =|language=zh-hans}}</ref>

== 名称 ==
根据媒体报道，他曾经先后注册过12个“秦火火”的微博，包括“淮上秦火火”、“炎黄秦火火”、“东土秦火火”和“中国秦火火”。根据办案民警的消息，秦火火里面有“火”字，是为了让他火起来，他通过粉丝数量的增加最终使自己出名，按他的供述，出名之后就会有[[出版社|出版社]]找他出书。<ref>[http://news.163.com/14/0411/06/9PHFEACG00014JB5.html 网络红人“秦火火”诽谤、寻衅滋事案今日开庭审理] [2014-04-11]</ref>

== 生平 ==
1983年，秦志晖出生在湖南省衡南县香花村。高中毕业后，秦志晖在[[广州|广州]]打工。<ref name="fish " />后来，秦志晖北漂，当过文字核对员。<ref name="fish " />
2004年，秦志晖回湖南读书，2005年，秦志晖进入华艺百创公司。<ref>{{cite web |url=http://news.china.com.cn/live/2013-08/25/content_21919738.htm |title=网络推手秦火火：书呆子咋成了“大谣”？  |accessdate=2013-08-25  |publisher= 中国网 |work= |quote =|language=zh-hans}}</ref>2010年，秦志晖进入尔玛中国，2011年离开。<ref name="fish " />

== 秦火火案 ==

=== 主要谣言 ===
根据官方说法，秦火火涉及在[[新浪微博|新浪微博]]造谣传谣3000条左右，引发大量网民转发和负面评论，而他发此信息没有任何根据。其中重点包括<ref name="tang " />：
#[[2011年甬台温铁路列车追尾事故|“7．23”动车事故]]，[[中华人民共和国政府|中华人民共和国政府]]用2亿元[[人民币|人民币]]赔偿死亡的外籍旅客。<ref name=" bull">{{cite web|url=http://ent.163.com/13/0821/08/96PNFQM700032DGD.html?f=jsearch|title=“秦火火”网络造谣被端：曾是郭美美事件的幕后推手|accessdate=2013-08-21|publisher=京华时报|work=|quote=|language=zh-hans|deadurl=yes|archiveurl=https://web.archive.org/web/20140413125900/http://ent.163.com/13/0821/08/96PNFQM700032DGD.html?f=jsearch|archivedate=2014-04-13}}</ref>原微博是“刚得到消息，[[铁道部|铁道部]]已向动车事故中意大利遇难者茜茜协议赔偿三千万欧元（折合人民币接近两亿），据悉，这是铁道部参照欧洲法律中有关人身意外伤害条款后，不得不同意此赔偿协议。若此赔偿协议属实，将开创中国对外个人意外最高赔偿纪录。”
#编造[[雷锋|雷锋]]生活奢侈情节，并称道德楷模雷锋的形象完全是由中国共产党编造的<ref name=" Leifeng" >{{cite book |author=乔安山 |title=《伤痛无声:乔安山忆雷锋》|year=2007年2月1日 |publisher=国家行政学院出版社 |location= |quote = 雷班长早在[[鞍钢|鞍钢]]期间，月薪超过30元，他没有家庭负担，不必寄钱补贴家用，他擦脸用友谊牌雪花膏，还曾在青年商店买了件天津华光皮件厂的皮夹克，花了44元，还买了一条料子裤和一双黑皮鞋。 |pages= |isbn=9787801405777 |language=zh-hans}}</ref>。原微博是“雷锋1959年为自己添置的皮夹克、毛料裤、黑皮鞋等全套高档行头，皮夹克、毛料裤、皮鞋加起来当时在90元左右，而当时雷锋一个月才六块钱。”
