《'''秦时明月壹百步飞剑'''》是电视动画系列《[[秦时明月_(动画)|秦时明月]]》的第一部，2007年2月14日至23日在[[天津电视台|天津电视台]]少儿频道首播，原为10集，但因不符中国大陆动画管理法规而在播出时拆分为20集。2007年5月，《百步飞剑》重制版播出，画质较原版有所提升。动画讲述了剑客盖聂叛出秦国，并带领少年荆天明到达[[墨家|墨家]]机关城的故事。

《百步飞剑》使用了[[动作捕捉|动作捕捉]]等三维动画制作技术，投资达1000万人民币。动画播出后受到好评，先后获得多个奖项。

== 制作 ==
《秦时明月壹百步飞剑》是[[玄机科技|玄机科技]]制作的系列动画《[[秦时明月_(动画)|秦时明月]]》的第一部<ref>{{Cite web|url=http://www.bj.xinhuanet.com/bjpd_sdzx/2007-01/24/content_9142350.htm |title=国内首部3D武侠动画《秦时明月》强袭贺岁荧屏 |accessdate=2015-12-15 |author= |date=2007-01-24 |publisher=新华网北京频道 |archiveurl=https://web.archive.org/web/20151215063636/http://www.bj.xinhuanet.com/bjpd_sdzx/2007-01/24/content_9142350.htm |archivedate=2015-12-15 |deadurl=yes }}</ref>。2005年，还在制作中的《百步飞剑》参加了一些展出，受到中国大陆一些电视台的欢迎。2006年上半年，动画被中国大陆十余家省市电视台预先抢购<ref>{{Cite web|url=http://hzdaily.hangzhou.com.cn/hzrb/html/2007-08/03/content_106355.htm |title=系列片《秦时明月》“动漫之都”春风得意的一种表情 |accessdate=2015-12-15 |author=张藜藜 |date=2007-08-03 |publisher=杭州日报 |archiveurl=https://web.archive.org/web/20151215090902/http://hzdaily.hangzhou.com.cn/hzrb/html/2007-08/03/content_106355.htm |archivedate=2015-12-15 |deadurl=yes }}</ref>。《百步飞剑》采用全[[3D|三维]]CG技术制作<ref name=":0">{{Cite web|url=http://zqb.cyol.com/content/2007-01/30/content_1659636.htm |title=技术唯美打造国产动漫 |accessdate=2015-12-15 |author= |date=2007-01-30 |publisher=中国青年报 |archiveurl=https://web.archive.org/web/20070205170531/http://zqb.cyol.com/content/2007-01/30/content_1659636.htm |archivedate=2007-02-05 |deadurl=yes }}</ref>，投资达1000万人民币<ref>{{Cite web|url=http://cq.qq.com/a/20070124/000134.htm |title=重庆青少频道春节播出首部贺岁动漫 |accessdate=2015-12-15 |author= |date=2007-01-24 |publisher=重庆商报 |archiveurl=https://web.archive.org/web/20151215091847/http://cq.qq.com/a/20070124/000134.htm |archivedate=2015-12-15 |deadurl=yes }}</ref>。动画采用[[动作捕捉|动作捕捉]]技术，通过特聘武术导演将专业的武术动作信息由系统记录下来输入计算机，经处理后由计算机在虚拟镜头中恢复，同时控制材质<ref name=":0" />。

== 播出与发行 ==
《秦时明月壹百步飞剑》于2007年春节期间作为贺岁动画在中国大陆多个地方电视台播出<ref>{{Cite web|url=http://www.bj.xinhuanet.com/bjpd_sdzx/2007-01/24/content_9142350.htm |title=国内首部3D武侠动画《秦时明月》强袭贺岁荧屏 |accessdate=2015-12-15 |author= |date=2007-01-24 |publisher=新华网北京频道 |archiveurl=https://web.archive.org/web/20151215095042/http://www.bj.xinhuanet.com/bjpd_sdzx/2007-01/24/content_9142350.htm |archivedate=2015-12-15 |deadurl=yes }}</ref>。2010年5月，玄机科技宣布《百步飞剑》10集重制版将于5月11日在[[南方电视台|南方电视台]]少儿频道首播。5月11日，玄机科技称因天气原因无法将重制版送至南方少儿频道，届时南方少儿将播放原版《百步飞剑》<ref>{{Cite web|url=http://blog.sina.com.cn/s/blog_5febe35f0100ibir.html |title=南方少儿台原定首播重置版《百步飞剑》因故有所变动 |accessdate=2015-12-11 |author= |date=2010-05-11 |publisher=玄机科技 |archiveurl=https://web.archive.org/web/20151211062016/http://blog.sina.com.cn/s/blog_5febe35f0100ibir.html |archivedate=2015-12-11 |deadurl=yes }}</ref>。次日，官方在[[酷6网|酷6网]]放出重制版第一集的视频<ref>{{Cite web|url=http://v.ku6.com/show/EEa-3SWeU6vMRh8R.html |title=秦时明月第一部百步飞剑重置版第一集 |accessdate=2015-12-11 |author= |date=2010-12-11 |publisher=[[酷6网|酷6网]] |archiveurl=https://web.archive.org/web/20100517140253/http://v.ku6.com/show/EEa-3SWeU6vMRh8R.html |archivedate=2010-05-17 |deadurl=yes }}</ref>，南方少儿频道也在随后播出。

《百步飞剑》原版于2008年2月发行[[DVD|DVD]]<ref>{{Cite web|url=http://comic.qq.com/a/20080218/000043.htm |title=《秦时明月》动漫书与DVD双剑合璧 |accessdate=2015-12-15 |author= |date=2008-02-18 |publisher=腾讯网 |archiveurl=https://web.archive.org/web/20080511190232/http://comic.qq.com/a/20080218/000043.htm |archivedate=2008-05-11 |deadurl=yes }}</ref>，重制版于2011年5月发行DVD。

== 奖项 ==
2007年4月，《百步飞剑》入选中华人民共和国[[国家广播电影电视总局|国家广电总局]]2007年度第一批推荐优秀国产动画片。5月，本片提名第三届中国国际动漫节“美猴奖”最佳系列动画片奖。12月，本片获第20届“星光奖”优秀动画节目奖。2008年1月，本片获第四届“[[金龙奖|金龙奖]]”内地及港澳台年度最佳人气动画奖。10月，本片获[[中国广播电视协会|中国广播电视协会]]2007年度电视节目创优动画类一等奖。11月，本片获第一届厦门国际动漫节“金海豚”动画作品大赛最佳影视动画提名奖。<ref>{{Cite web|url = http://www.qinsmoon.com/qinsmoon/data/gywm/2014-12-20/105.html|title = 《秦时明月》获奖情况|accessdate = 2015-12-16|author = |date = |publisher = 玄机科技}}</ref>
