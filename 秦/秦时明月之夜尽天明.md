《'''秦时明月贰夜尽天明'''》是电视动画系列《[[秦时明月_(动画)|秦时明月]]》的第二部，2008年7月17日至8月3日在[[炫动卡通|上海炫动卡通卫视]]首播，共18集。动画主要围绕着[[墨家|墨家]]机关城保卫战展开。

《夜尽天明》的动画技术较前作有了提升，动画播出后获得了多个奖项。

== 制作 ==
《秦时明月贰夜尽天明》是[[玄机科技|玄机科技]]制作的系列动画《[[秦时明月_(动画)|秦时明月]]》的第二部，延续了第一部《[[秦时明月壹百步飞剑|百步飞剑]]》的故事线。制作组吸收了业内人士及观众的建议与批评，在动画技术方面作了大量的改进与提升；编剧方面，团队力图用做电影的方式来拍摄这部动画；配音方面，在第一部基础上保留了一些受欢迎的配音演员，同时对于一些观众不够满意的演员进行了重新选择，台湾演员[[李立群|李立群]]也参与配音工作<ref>{{Cite web|url=http://comic.qq.com/a/20080305/000006.htm |title=[秦时明月]力推中国配音再攀高峰 |accessdate=2016-04-04 |author= |date=2008-03-05 |publisher=腾讯网 |archiveurl=https://web.archive.org/web/20080517084353/http://comic.qq.com/a/20080305/000006.htm |archivedate=2008-05-17 |deadurl=yes }}</ref>；音乐方面，玄机科技与五家唱片公司签约了九首歌曲作为插曲与片尾曲。<ref>{{Cite web|url=http://comic.qq.com/a/20080429/000057.htm |title=腾讯专访《秦时明月》总导演沈乐平 |accessdate=2016-01-04 |author= |date=2008-04-29 |publisher=腾讯动漫 |archivedate=2016-01-04 |archiveurl=https://web.archive.org/web/20160104050559/http://comic.qq.com/a/20080429/000057.htm |deadurl=yes }}</ref>

《夜尽天明》的制作成本约为第一部的一半<ref>{{Cite web|url=http://game.21cn.com/t/interview/a/2008/0516/14/4729722.shtml |title=21CN专访[秦时明月]导演沈乐平 |accessdate=2016-01-04 |author= |date=2008-05-16 |publisher=21CN游戏频道 |archiveurl=https://web.archive.org/web/20160104051414/http://game.21cn.com/t/interview/a/2008/0516/14/4729722.shtml |archivedate=2016-01-04 |deadurl=yes }}</ref>。

== 播出与发行 ==
2008年2月1日，《秦时明月贰夜尽天明》的样片于[[腾讯|腾讯]]动漫频道播出<ref>{{Cite web|url=http://comic.qq.com/a/20080201/000009.htm |title=《秦时明月之夜尽天明》腾讯独家首发 |accessdate=2016-01-04 |author= |date=2008-02-01 |publisher=腾讯动漫 |archiveurl=https://web.archive.org/web/20160104045029/http://comic.qq.com/a/20080201/000009.htm |archivedate=2016-01-04 |deadurl=yes }}</ref>。《夜尽天明》原定于2008年5月首播，但由于[[汶川大地震|汶川大地震]]导致节目编排延后。2008年7月17日，动画在[[炫动卡通|上海炫动卡通卫视]]首播，同期发行DVD<ref name="qq"/><ref>{{Cite web|url=http://hzdaily.hangzhou.com.cn/hzrb/html/2008-07/22/content_471403.htm |title=《秦时明月》在沪举行新片发布暨DVD新片上市会 |accessdate=2016-04-04 |author= |date=2008-07-22 |publisher=杭州日报 |archiveurl=https://web.archive.org/web/20160404060547/http://hzdaily.hangzhou.com.cn/hzrb/html/2008-07/22/content_471403.htm |archivedate=2016-04-04 |deadurl=yes }}</ref>。

== 奖项 ==
2008年1月，《夜尽天明》获第四届“[[金龙奖|金龙奖]]”年度最佳系列动画奖；5月，本片获第四届中国国际动漫节“美猴奖”动画片最佳形象奖、动画系列连续片奖<ref>{{Cite web|url=http://ent.163.com/08/0712/01/4GK6BEV100032KMI.html |title=中国首部3D武侠动画剧《秦时明月》发布 |accessdate=2016-04-04 |author= |date=2008-07-12 |publisher=网易娱乐 |archiveurl=https://web.archive.org/web/20160404055931/http://ent.163.com/08/0712/01/4GK6BEV100032KMI.html |archivedate=2016-04-04 |deadurl=yes }}</ref>；8月，本片入选入选中华人民共和国[[国家广播电影电视总局|国家广电总局]]2008年度第二批推荐优秀国产动画片<ref>{{Cite web|url=http://hzdaily.hangzhou.com.cn/hzrb/html/2009-10/16/content_761550.htm |title=《秦时明月之夜尽天明》喜获影视动画奖 |accessdate=2016-04-04 |author= |date=2009-10-16 |publisher=杭州日报 |archiveurl=https://web.archive.org/web/20160404055707/http://hzdaily.hangzhou.com.cn/hzrb/html/2009-10/16/content_761550.htm |archivedate=2016-04-04 |deadurl=yes }}</ref>；12月，本片获[[中国动画学会|中国动画学会]]年度推荐作品；2009年6月，本片获2008年度国家广电总局动画精品二等奖<ref>{{Cite web|url = http://www.qinsmoon.com/qinsmoon/data/gywm/2014-12-20/105.html|title = 《秦时明月》获奖情况|accessdate = 2015-12-16|author = |date = |publisher = 玄机科技}}</ref>；同月，本片获第十五届上海电视节“白玉兰奖”国产动画片银奖<ref>{{Cite web|url=http://cartoon.southcn.com/ctnews/chnnews/content/2009-06/15/content_5255207.htm |title=《秦时明月之夜尽天明》喜获第15届上海电视节“白玉兰奖” |accessdate=2016-04-04 |author= |date=2009-06-15 |publisher=南方网 |archiveurl=https://web.archive.org/web/20160404055455/http://cartoon.southcn.com/ctnews/chnnews/content/2009-06/15/content_5255207.htm |archivedate=2016-04-04 |deadurl=yes }}</ref>。
