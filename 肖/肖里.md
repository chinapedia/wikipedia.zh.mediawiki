'''肖里'''<ref>本姓萧，其自用名为“肖里”。</ref>（{{bd|1917年||1969年|4月23日}}）曾用名'''肖国贤'''、'''肖清华'''，女，[[河南省|河南省]][[商水县|商水县]]人，[[中华人民共和国|中华人民共和国]]政府官员，[[李井泉|李井泉]]夫人。

==生平==
肖里生于河南省商水县谭庄镇肖谭村一个开明士绅家庭。祖父是[[清朝|清朝]][[光绪|光绪]]年间的[[武进士|武进士]]，曾获“御前侍卫”的功名。父亲萧润芝，母亲李氏。萧家至[[抗日战争|抗日战争]]初期尚有1200多亩地。<ref name=zglsw/>

肖里早年进入1908年建校的河南省立开封女子师范学校学习，擅长弹[[钢琴|钢琴]]，并且是学校女子[[篮球|篮球]]队主力队员。参加中共领导的革命前后，肖里还用过“肖国贤”、“肖清华”的名字。1937年10月，肖里加入[[中国共产党|中国共产党]]，1938年离开河南，随历史学家[[范文澜|范文澜]]抵达[[延安|延安]]，先入[[中国人民抗日军政大学|中国人民抗日军政大学]]女生队学习，后分到[[中共中央社会部|中共中央社会部]]工作。<ref name=zglsw/>

1941年春，[[李井泉|李井泉]]自前线回到延安的[[中共中央党校|中央党校]]学习，同在中央党校学习的冀中军区政治部主任[[孙志远|孙志远]]及妻子赵磊给李井泉介绍了肖里，促成两人在1941年结婚。<ref name=zglsw/><ref name=qingshan>{{Cite web |url = http://news.ifeng.com/history/zhongguoxiandaishi/detail_2013_07/13/27465420_0.shtml |title = 青山无言——回忆父亲李井泉 |author =申再望  |date =  |publisher = 党史文苑2009年第19期 |language =  |accessdate =  |quote =  }}</ref>

1950年，肖里随进军大西南的[[中国人民解放军|中国人民解放军]]部队抵达[[四川|四川]][[成都|成都]]，任[[西南政法大学|西南人民革命大学]]成都分校教育科科长，后任处长。1951年，出任成都纺织厂筹备处主任。当时她已怀孕，为不影响工作，一个人搬到纺织厂筹备处居住，仅在每周六回家，周日又骑自行车回单位。成都纺织厂建成后，肖里任成都纺织厂厂长兼党委书记，不久升任四川省轻工业厅纺织局局长。1963年，改任四川省轻工业厅副厅长兼政治部主任。<ref name=zglsw/>

1966年[[文化大革命|文化大革命]]爆发后，李井泉被打成“西南地区最大的[[走资派|走资本主义道路的当权派]]”，肖里受到连累。当时四川省轻工业厅院里到处贴有“打倒李井泉！打倒肖里！”的大标语，肖里受到残酷批斗和精神折磨。1967年3月15日，[[江青|江青]]在北京对四川省进京的[[造反派|造反派]]说：“你们还没有把李井泉那个地主老婆关起来？”造反派回到四川后便将肖里关进了秘密监狱。<ref name=zglsw/>

1969年4月23日，肖里在秘密监狱监禁中被迫害致死，年仅52岁。<ref name=zglsw/><ref name=qingshan/>文革结束后，1978年李井泉到四川参加了肖里同志骨灰安放仪式，当天前来参加仪式的有很多老同志，不少人泣不成声，李井泉与他们逐一握手致谢。骨灰安放仪式后，李井泉立刻去四川的[[自贡|自贡]]、[[宜宾|宜宾]]、[[江津|江津]]、[[达县|达县]]、[[万县|万县]]等地，勉励当地同志将对“[[四人帮|四人帮]]”的痛恨化为建设国家的动力。<ref name=qingshan/>
