'''乾生元'''是一家起源于[[苏州市|苏州]][[木渎镇|木渎]]的苏式点心店，创始于[[清朝|清]][[乾隆|乾隆]]四十六年（[[1781年|1781年]]）。以[[松子枣泥麻饼|松子枣泥麻饼]]为招牌，后也开始经营苏式糖果、糕点、炒货、蜜饯等。现由苏州乾生元食品有限公司经营。

== 历史 ==
乾生元创立于清乾隆四十六年（1781年），原名为“费萃泰”，[[光绪|光绪]]七年（1881年）改名为“乾生元”。创立初期以经营生产松子枣泥麻饼而闻名，乾隆皇帝曾将其列为宫廷御膳点心<ref>{{Cite web|url=http://www.sohu.com/a/199840554_531962|title=名扬海内外的老字号美食，苏州乾生元麻饼|accessdate=2018-01-10|date=2017-10-24|work=www.sohu.com}}</ref>。建立以来，一直在[[木渎镇|木渎镇]]经营。

2006年在[[中华人民共和国商务部|中华人民共和国商务部]]的评选中被评定为“中华老字号”<ref>{{Cite web|url=http://news.sina.com.cn/c/2006-11-21/064610556311s.shtml|title=我省35个品牌成“中华老字号”_新闻中心_新浪网|accessdate=2018-01-10|work=news.sina.com.cn}}</ref>。

2009年，乾生元在部分老字号经营压力过大，撤出观前商圈之际，首次进驻观前，在[[观前街|观前街]]开出了两家门店<ref>{{Cite web|url=http://news.sina.com.cn/c/2008-11-11/083714711726s.shtml|title=“老字号”能否与时尚品牌平起平坐_新闻中心_新浪网|accessdate=2018-01-10|work=news.sina.com.cn}}</ref><ref>{{Cite web|url=http://finance.sina.com.cn/roll/20090522/07042855397.shtml|title=中华老字号乾生元开进苏州观前街_滚动新闻_新浪财经_新浪网|accessdate=2018-01-10|work=finance.sina.com.cn}}</ref>。

2010年，乾生元与[[得月楼|得月楼]]共同进入[[上海|上海]]市场，分别在上海开出门店，然而仅两年过去，因生意惨淡，两家都以关张收场<ref>{{Cite web|url=http://sh.eastday.com/m/20120817/u1a6792586.html|title=乾生元折戟申城 专家质疑中华名品街商业氛围不足-上海频道-东方网|accessdate=2018-01-10|last=袁松禄|work=sh.eastday.com}}</ref>。

== 分店 ==
乾生元在木渎、[[山塘街|山塘街]]、观前街、[[苏州工业园区|工业园区]]等地均有门店。目前乾生元除食品外，在山塘街、木渎也经营酒店住宿等。

== 产品 ==
