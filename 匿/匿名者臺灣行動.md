'''匿名者臺灣行動'''（{{Lang-en|'''OpTaiwan'''}}）是2015年時，一個號稱駭客組織[[匿名者|匿名者]]亞洲支部的專頁為了聲援反對[[臺灣高中歷史課綱微調案|臺灣高中歷史課綱微調案]]抗議活動的學生和群眾而發起的網際網路行動。其中在7月31日，自稱「匿名者亞洲」的群體在Facebook專頁上發表支持[[反高中課綱微調運動|反高中課綱微調運動]]學生的影片。隨後「匿名者亞洲」的駭客發起行動並且針對[[中華民國教育部|中華民國教育部]]和[[中華民國總統府|中華民國總統府]]等網站展開攻擊，試圖藉由網際網路攻擊施壓政府單位，迫使中華民國政府們能夠正視抗議學生的訴求<ref name = "趙坤茂">{{Cite web |url = http://udn.com/news/story/7339/1098009 |title = 課綱爭議／「匿名者」的具名 是真名嗎？ |author = [[趙坤茂|趙坤茂]] |date = 2015年8月4日 |publisher = [[聯合新聞網|聯合新聞網]] |language = zh-hant |accessdate = 2015年8月5日 |quote =  }}</ref>。其中「匿名者亞洲」持續運用分散式[[阻斷服務攻擊|阻斷服務攻擊]]方式進行攻擊行動，藉由網站因為連線數量暴增後獲得過多資料使得流量超載，並且導致網頁回應時間較久或暫時無法連線，進而達成使特定網站無法對外提供服務的目的<ref>{{Cite web |url = http://udn.com/news/story/1/1094587 |title = 網站疑遭駭客攻擊 教育部籲譴責 |author = [[中央通訊社|中央通訊社]] |date = 2015年8月2日 |publisher = [[聯合新聞網|聯合新聞網]] |language = zh-hant |accessdate = 2015年8月5日 |quote =  }}</ref>。

之後「匿名者亞洲」針對多個[[中華民國政府|中華民國政府]]單位<ref name = "吳父鄉">{{Cite web |url = http://udn.com/news/story/8466/1097948 |title = 政府資安分級 迎戰駭客  |author = 吳父鄉 |date = 2015年8月4日 |publisher = [[聯合新聞網|聯合新聞網]] |language = zh-hant |accessdate = 2015年8月5日 |quote =  }}</ref>，以及被認為與中華人民共和國關係較好的政黨與電視臺的網站發動攻擊<ref name = "劉子維">{{Cite web |url = http://www.bbc.com/zhongwen/trad/china/2015/08/150803_taiwan_curriculum_protest_anonymous |title = 台灣反課綱抗議：「匿名者黑客攻擊多個政府網站」 |author = 劉子維 |date = 2015年8月3日 |publisher = [[BBC中文網|BBC中文網]] |language = zh-hant |accessdate = 2015年8月5日 |quote =  }}</ref>。後續遭到攻擊者包括了[[中華民國國防部|中華民國國防部]]、[[中華民國經濟部|中華民國經濟部]]、[[中國國民黨|中國國民黨]]、[[新黨|新黨]]以及[[中天電視|中天電視]]等多個官方網站，另外「匿名者亞洲」也曾經表示將針對[[臺灣銀行|臺灣銀行]]網站發動攻擊行動。而先前中華民國政府部門遭遇大規模分散式阻斷服務攻擊，則是發生在2013年與菲律賓之間發生的[[廣大興28號事件|廣大興28號事件]]<ref name = "吳父鄉"/>。

== 背景 ==
=== 組織運作 ===
[[File:Protest_ACTA_2012-02-11_-_Toulouse_-_15_-_Anonymous_guy_with_a_black_and_white_scarf.jpg|thumb]]而參與《[[反仿冒貿易協定|反仿冒貿易協定]]》抗議的匿名者成員。]]

[[匿名者|匿名者]]是由{{Link-en|網路行動主義|Internet activism|網際網路行動}}分子與駭客組成的自發性鬆散組織，但同時也是全世界最大的駭客行動群體。其中匿名者主要是依照理念而聚集而成的群體，整個組織的指揮機構極為鬆散且分散，在這當並沒有固定的領導人、階級或者是隸屬團體。而在匿名者的群體理念中，亦表示任何人都可以宣稱代表或者屬於該組織，彼此之間並無指令關係。匿名者的想法最早在2003年時於美國貼圖論壇[[4chan|4chan]]逐漸成形，最早常以匿名的方式共同針對特定目標發起惡搞或戲謔的行動以取樂。隨後參與的匿名成員在經過協調後在政治觀點上形成一定共識，並且由於該網站用戶都稱作「匿名者」而此作為統稱。匿名者主要核心觀點是反對網際網路監控和審查並且呼籲網際網路自由，以及從之延伸的言論自由、反對歧視、反壟斷和獨裁、民主制度與協助建立屬於公民的世界<ref name = "陳韋廷">{{Cite web |url = http://udn.com/news/story/8466/1097971 |title = 駭客團體「匿名者」 依理念而非指令運作 |author = 陳韋廷 |date = 2015年8月4日 |publisher = [[聯合新聞網|聯合新聞網]] |language = zh-hant |accessdate = 2015年8月5日 |quote =  }}</ref><ref name = "倒數開戰">{{Cite web |url = http://www.appledaily.com.tw/appledaily/article/headline/20150806/36707377/ |title = 倒數開戰 「匿名者」fb突消失 |author =  |date = 2015年8月6日 |publisher = 《[[蘋果日報_(台灣)|蘋果日報]]》 |language = zh-hant |accessdate = 2015年8月5日 |quote =  }}</ref>。

其中匿名者認為理念或想法相似者都可以是匿名者成員，但是也表時參與任何形式的駭客攻擊都是違法的行為，除了表示應當具備相關技能外，同時強調參與成員必須自行承擔被警方查緝以及後續的法律問題。不過其也呼籲沒有技術但想幫助匿名者對抗政府的公民，也可以簡單地藉由分享相關訊息予以協助<ref>{{Cite web |url = http://news.ltn.com.tw/news/politics/breakingnews/1401218 |title = 連番出手嗆政府 「匿名者」的真面目原來是... |author =  |date = 2015年8月5日 |publisher = 《[[自由時報|自由時報]]》 |language = zh-hant |accessdate = 2015年8月5日 |quote =  }}</ref>。匿名者群體間以[[IRC|IRC]]網路聊天程式或社群網站溝通或聯繫，主要在網際網路上攻擊的對象是支持恐怖主義的組織和集權統治的國家；透過分散式[[阻斷服務攻擊|阻斷服務攻擊]]、置換首頁內容、駭入特定網站乃至於直接癱瘓網站等方式攻擊，迫使代表該組織、該宗教或者該國家的網站不得提供服務，進行不流血的抗爭行動<ref name = "倒數開戰"/>。與此同時，匿名者亦幫助各地區公民爭取應有的權益並且伸張正義，促使該地區公民得以發起街頭抗爭行動<ref name = "今日刊文">{{Cite web |url = http://millenniummag-todaynews.blogspot.ca/2015/08/02.html |title = 《匿名者》亞洲支部發動攻擊 臺灣政府多個網站遭癱瘓 |author =  |date = 2015年8月2日 |publisher = 今日刊文 |language = zh-hant |accessdate = 2015年8月5日 |quote =  }}</ref>。

匿名者最早便曾在反對[[山達基|山達基]]以及打擊[[兒童色情|兒童色情]]等事務上有過許多活動，主要活躍在歐美地區和中東地區<ref>{{Cite web |url = http://www.appledaily.com.tw/realtimenews/article/politics/20150805/662456/ |title = 敢下通牒摃政府　「匿名者」究竟是誰？ |author = 劉育良 |date = 2015年8月5日 |publisher = 《[[蘋果日報_(台灣)|蘋果日報]]》 |language = zh-hant |accessdate = 2015年8月5日 |quote =  }}</ref>；而過去也曾對[[美國|美國]]、[[朝鮮民主主義人民共和國|朝鮮民主主義人民共和國]]、[[以色列|以色列]]、[[突尼西亞|突尼西亞]]、[[烏干達|烏干達]]、[[中華人民共和國|中華人民共和國]]、[[菲律賓|菲律賓]]、[[伊斯蘭國|伊斯蘭國]]的政府部門網站發動攻擊，並且高調宣傳其攻擊成果<ref name = "今日刊文"/>。另外許多版權保護機構網站以及[[索尼|索尼]]、[[VISA|VISA]]等多家企業網站也曾遭到駭取，[[阿拉伯之春|阿拉伯之春]]與[[佔領華爾街|佔領華爾街]]運動等政治活動中亦有匿名者參與<ref name = "陳韋廷"/>。在2013年5月，「臺灣匿名者」成員便曾經攻擊菲律賓數百個機構的官方網站，要求菲律賓政府為公務船槍殺[[廣大興28號事件|廣大興28號]]上的臺灣漁民道歉<ref name = "倒數開戰"/>。同樣由駭客組成的「匿名者亞洲」（Anonymous Asia）在2014年[[香港|香港]]進行[[雨傘革命|雨傘革命]]期間，也曾經攻擊過多個[[中華人民共和國政府|中華人民共和國政府]]和[[香港特別行政區政府|香港特別行政區政府]]的網站。其[[Facebook|Facebook]]專頁上發表的貼文大多是以香港和臺灣使用的[[繁體中文|繁體中文]]以及[[英語|英語]]為主，而過去貼文主要和[[香港民主運動|香港民主運動]]有關的內容、以及其他匿名者發表的影片與訊息<ref name = "劉子維"/>，另外也有部分文章內容則是類似來自臺灣地區的言論<ref name = "T客邦">{{Cite web |url = http://technews.tw/2015/08/06/anonymous-asia-attack-taiwan-stock-exchange/ |title = 匿名者攻擊台灣證券交易所，網友反應不一 |author = T客邦 |date = 2015年8月6日 |publisher = 科技新報 |language = zh-hant |accessdate = 2015年8月5日 |quote =  }}</ref>。

=== 反課綱運動 ===
[[File:20150731-02-35-52-P7310052_(20149589845).jpg|thumb]]學生[[中華民國教育部|中華民國教育部]]廣場並且掛起抗議布條。]]

在2014年2月，由於[[中華民國教育部|中華民國教育部]]公布[[臺灣高中歷史課綱微調案|臺灣高中歷史課綱微調案]]而引發一連串相關爭議後，促使部分高級中學學生發起[[反高中課綱微調運動|反高中課綱微調運動]]。其中抗議學生認為執政的[[中國國民黨|中國國民黨]]以微調課程使得內容更為準確的說法，試圖掩飾原先高級中學課綱內有關中國國民一黨專制時期對民眾的打壓，並且進一步加入[[中國中心主義|中國中心主義]]思想。儘管存在上述指控，中華民國教育部仍然表示計畫在2015年8月1日施行課綱內容。2015年7月24日，抗議學生發起[[723佔領教育部行動|723佔領教育部行動]]而一度成功進佔中華民國教育部，其中有9人更成功闖入2樓中華民國教育部部長辦公室。但是隨後有包含24名學生在內、共計33人被束帶和手銬架離中華民國教育部，之後送至[[臺北市政府警察局保安警察大隊|臺北市政府警察局保安警察大隊]]拘捕<ref name="李蔡彥">{{Cite web |url = http://www.wantchinatimes.com/news-subclass-cnt.aspx?id=20150802000143&cid=1104 |title = Anonymous locks down Taiwan government websites |author = 李蔡彥 |date = 2015年8月2日 |publisher = 《[[旺報|旺報]]》 |language = en |accessdate = 2015年8月5日 |quote =  |deadurl = yes |archiveurl = https://web.archive.org/web/20150806004037/http://www.wantchinatimes.com/news-subclass-cnt.aspx?id=20150802000143&cid=1104 |archivedate = 2015年8月6日 }}</ref><ref name="課綱爭議吵不休">{{Cite web|url =http://udn.com/news/story/1/1090872|title =整理包／課綱爭議吵不休 事件發展時間軸|author =|date =2015年8月4日|publisher =[[聯合新聞網|聯合新聞網]]|language =zh-hant|accessdate =2015年8月5日|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20150803011154/http://udn.com/news/story/1/1090872|archivedate =2015年8月3日}}</ref>。

事後中華民國教育部部長[[吳思華|吳思華]]表示將會對進佔者發起訴訟<ref name="法新社">{{Cite web |url = http://www.channelnewsasia.com/news/asiapacific/taiwan-protesters-rip-up/2024130.html |title = Taiwan protesters rip up textbooks, demanding minister's resignation |author = [[法新社|法新社]] |date = 2015年8月2日 |publisher = [[亞洲新聞台|亞洲新聞台]] |language = en |accessdate = 2015年8月5日 |quote =  |deadurl = yes |archiveurl = https://web.archive.org/web/20150808051704/http://www.channelnewsasia.com/news/asiapacific/taiwan-protesters-rip-up/2024130.html |archivedate = 2015年8月8日 }}</ref>。對此[[洪秀柱|洪秀柱]]認為學生應該要遵守法律，而蔡英文則表示對於學生遭到逮捕居留感到痛心。到了7月30日，曾參與先前佔領行動的北區反課綱高校聯盟發言人[[林冠華|林冠華]]燒炭自殺身亡，對此社會各界感到遺憾。而此事亦引發反對臺灣高中歷史課綱微調案的學生情緒高漲而擴大抗爭行動，當天下午抗議學生先是包圍吳思華的座車，晚間更進一步攻進[[立法院|立法院]]<ref name = "課綱爭議吵不休"/>。到了7月31日凌晨，200多名反高中課綱微調運動的學生與群眾接連翻越圍牆再度佔領[[中華民國教育部|中華民國教育部]]前面的廣場<ref name = "李蔡彥"/><ref>{{Cite web |url = http://udn.com/news/story/1/1094597 |title = 網路受攻擊中斷 教育部譴責駭客 |author = 陳智華 |date = 2015年8月2日 |publisher = [[聯合新聞網|聯合新聞網]] |language = zh-hant |accessdate = 2015年8月5日 |quote =  }}</ref>。很快地許多支持學生運動的民眾陸續聚集在中華民國教育部前聲援，要求中華民國教育部部長吳思華下台並且撤回課綱內容<ref name = "劉子維"/>。

對此中國國民黨以及參與課綱微調的學者批評反對黨民主進步黨煽動學生，而民主進步黨則指責中國國民黨不願正面回應並且試圖抹黑學生抗議行動<ref name = "法新社"/>。儘管在7月31日下午，吳思華曾經前往中華民國教育部和學生與群眾談話，但是談判經過30分鐘過後隨即宣告破裂。其中對於抗議學生要求召開臨時會的做法以撤回臺灣高中歷史課綱微調案，吳思華表示中華民國教育部已經決定採行新舊課綱並行作法；同時表示自己是最了解臺灣高中歷史課綱微調案的人，不能在此時如抗議學生的訴求下臺<ref name = "課綱爭議吵不休"/>。另外他還提到由於新課綱教科書已經送印，在完全執行完程序過後便無法宣告暫停施行。對於立法院計畫可能針對臺灣高中歷史課綱微調案召開臨時會處理，吳思華則表示會尊重立法院的決議而透過行政與立法之協商，找出一個可以執行的方案<ref name = "劉子維"/>。

=== 政府防備 ===
[[File:臺灣總統府_Hsiao.jpg|thumb]]過去便曾經多次遭遇網際網路攻擊。]]

自2012年以來，中華民國每年平均發生300多件資訊安全問題之案件，其中需要通報[[行政院院長|行政院院長]]層級的「三級資安事件」在3年來共有19件，其中合理懷疑多數來自中國大陸。其中中華民國政府最常被攻擊單位，包括有[[中華民國總統府|中華民國總統府]]、[[行政院|行政院]]、[[中華民國經濟部|中華民國經濟部]]、[[中華民國外交部|中華民國外交部]]、[[中華民國國家發展委員會|中華民國國家發展委員會]]。中華民國總統府發言人[[陳以信|陳以信]]便表示中華民國總統府每個月都有數千次資訊安全威脅，平均1天內會發生數十次<ref name = "李昭安和楊湘鈞">{{Cite web |url = http://udn.com/news/story/8466/1097965 |title = 聲援反課綱 「匿名者」攻政府網站、嗆駭台銀 |author = 李昭安和楊湘鈞 |date = 2015年8月4日 |publisher = [[聯合新聞網|聯合新聞網]] |language = zh-hant |accessdate = 2015年8月5日 |quote =  }}</ref>。為此在2014年8月，行政院將《國家資通安全科技中心設置條例》草案送交立法院審議。而[[行政院資通安全辦公室|行政院資通安全辦公室]]主任[[蕭秀琴|蕭秀琴]]表示未來希望推動技術服務中心的行政法人化，將人數擴編至200人。當前各個部會可以自行因應常見分散式阻斷服務攻擊和置換網頁攻擊，但是若出現以竊取資料為計畫目標的[[高級長期威脅|高級長期威脅]]攻擊時，行政院國家資通安全會報技術服務中心就會介入協助<ref name = "李昭安">{{Cite web |url = http://udn.com/news/story/8466/1097977 |title = 駭客攻擊一波波 警告意味濃 |author = 李昭安 |date = 2015年8月4日 |publisher = [[聯合新聞網|聯合新聞網]] |language = zh-hant |accessdate = 2015年8月5日 |quote =  }}</ref>。同時為了避免駭客入侵政府網站並竊取機密資料，各個部會都有嚴格的網站管理，同時從外部網路連接至內部網路間有著嚴密防火牆，並且嚴格阻攔來自「黑名單」的連線<ref name = "吳父鄉和李昭安">{{Cite web |url = http://udn.com/news/story/8466/1097967 |title = 聲援反課綱「匿名者」駭網 政府備戰 |author = 吳父鄉和李昭安 |date = 2015年8月4日 |publisher = [[聯合新聞網|聯合新聞網]] |language = zh-hant |accessdate = 2015年8月5日 |quote =  }}</ref>。

針對駭客引起的資訊安全愈來愈險峻，行政院為了妥善因應而在2015年6月4日核定新版的《行政院及所屬各機關資訊安全管理規範》，藉此全面升級資訊安全控管。包括政府機關、國營事業、學術機關等共7,287個單位皆納入分級管轄，而原本分成四級的資訊安全分層管理也直接刪除防護最弱的D級。其中包括中華民國總統府、五院與所屬二級單位、直轄市政府、外交、國防與經濟事務相關單位、以及[[台灣電力公司|台灣電力公司]]與[[台灣中油|台灣中油]]等國營企業和關鍵基礎設施在內，共計有127個單位全都納入A級控管，並且計劃在2016年導入{{Link-en|資訊安全管理制度|Information security management system}}第三方認證。而涉及社會秩序與人民財產的單位則被列為B級，則是計畫在2017年同樣導入資訊安全管理制度認證。其中過去A級和B級僅有300多個單位納入，在新版本中則增加至696個單位。另外剩餘的單位則是被列入C級，人員需要接受資訊安全訓練並且建立防毒、防火牆以及郵件過濾系統<ref name = "吳父鄉"/>。

=== 發起行動 ===

2015年7月31日下午，駭客組織「匿名者亞洲」於Facebook專頁上發表長104秒的《匿名者台灣行動》（Anonymous #OpTaiwan）的[[YouTube|YouTube]]影片，宣布發動「匿名者台灣行動」（OpTaiwan）聲援反高中課綱微調運動的學生。其中影片除了有過去匿名者組織公布消息時相同的片頭和配樂外，並且使用了反課綱學生在[[723佔領教育部行動|723佔領教育部行動]]中遭到警方制伏的片段。而旁白內容中提到已經聽到示威民眾的呼喊，若[[中華民國警察|中華民國警察]]和[[中華民國政府|中華民國政府]]侮辱或傷害抗議群眾，將會攻擊各個政府機構網站；同時還鼓勵抗議群眾繼續打起精神展開抗爭行動<ref>{{Cite web |url = http://news.ltn.com.tw/news/life/breakingnews/1397342 |title = 國際駭客組織「匿名者」 一度癱瘓教育部網站 |author =  |date = 2015年8月1日 |publisher = 《[[自由時報|自由時報]]》 |language = zh-hant |accessdate = 2015年8月5日 |quote =  }}</ref>，並且向中華民國政府表示：「請珍惜我們給你這個機會從前線撤回，這不是威脅，這是一個承諾。我們從不寬恕，也不輕易遺忘，等著瞧吧！<ref>{{Cite web |url = http://www.appledaily.com.tw/realtimenews/article/international/20150731/659891/ |title = 「匿名者」挺學生　威脅攻擊政府網站 |author = 余浚安 |date = 2015年7月31日 |publisher = 《[[蘋果日報_(台灣)|蘋果日報]]》 |language = zh-hant |accessdate = 2015年8月5日 |quote =  }}</ref>」8月1日凌晨1時，「匿名者亞洲」先後向中華民國教育部與中華民國總統府網站發動攻擊。隨後「匿名者亞洲」成功讓中華民國教育部網站一度癱瘓而使得其他用戶無法順利連上，並且在Facebook上張貼網站掛點的照片<ref>{{Cite web |url = http://www.appledaily.com.tw/realtimenews/article/international/20150801/659921/ |title = 【更新】「匿名者」出手　教育部網站已恢復 |author =  |date = 2015年8月1日 |publisher = 《[[蘋果日報_(台灣)|蘋果日報]]》 |language = zh-hant |accessdate = 2015年8月5日 |quote =  }}</ref>，而兩個網站隨後被強制封鎖長達5個小時<ref name = "今日刊文"/>。

其中「匿名者亞洲」在協調菲律賓、加拿大、臺灣、香港、中國大陸和美國的匿名者群體後共同發動駭客攻擊<ref name = "李蔡彥"/>，主要的行動方式則是採取駭客經常使用的分散式[[阻斷服務攻擊|阻斷服務攻擊]]手法<ref name = "劉子維"/>。其中參與行動的駭客同時操作由世界各處以網際網路連結的[[殭屍網路|殭屍網路]]發送[[封包|封包]]請求，針對中華民國教育部等多個中華民國政府網站展開攻擊，而其所使用的殭屍網路則是利用民眾電腦程式的弱點來下達指令<ref name = "謝君臨">{{Cite web |url = http://news.ltn.com.tw/news/society/breakingnews/1404656 |title = 「匿名者」攻擊政府網站 網友當心勿成「跳板」 |author = 謝君臨 |date = 2015年8月7日 |publisher = 《[[自由時報|自由時報]]》 |language = zh-hant |accessdate = 2015年8月5日 |quote =  }}</ref>。透過每個網站都有其服務設計容量的特性，外部大量的分散來源模擬大量使用者瞬間上線的情況，導致特定網站因為連線數量快速增加而封包流量過大。最後由於超過服務設計容量的瞬間查詢與瀏覽流量，使得網站頻寬、網路伺服器或者主機容量超載無法因應，進而導致網頁回應時間較久或暫時無法連線之情形，該特定網站也因此暫時無法對外提供服務<ref name = "徐珮君">{{Cite web |url = http://www.appledaily.com.tw/realtimenews/article/politics/20150803/661641/ |title = 「匿名者」駭政府機關　政院展開鑑識調查 |author = 徐珮君 |date = 2015年8月3日 |publisher = 《[[蘋果日報_(台灣)|蘋果日報]]》 |language = zh-hant |accessdate = 2015年8月5日 |quote =  }}</ref>。

相較於過去駭客阻斷服務攻擊的方式是在同一時間內大量連結政府網站，自單一站點發動攻擊導致網站無法負荷而癱瘓，使得這作法很快可被資訊安全人員發現並且在被阻斷後網站便恢復正常。然而這次駭客透過不同方式發動分散式阻斷服務攻擊，藉由多站點的資訊傳通使得資訊安全人員難以分辨正常連結而難以阻斷，唯一的方法僅有保護主機不受到攻擊，而在攻擊流量結束後網站便馬上恢復正常<ref name = "吳父鄉和李昭安"/>。其中這次分散式阻斷服務攻擊的手法之一是透過部落格平臺[[WordPress|WordPress]]舊版本的[[Pingback|Pingback]]功能，其中駭客向這些使用WordPress平臺的部落格密集地發出指令，在民眾不知情的情況下不斷向特定中華民國政府網站發出大量的Pingback請求，密集地通知更新或者要求回答引用<ref>{{Cite web |url = http://news.ltn.com.tw/news/focus/paper/904917 |title = 「匿名者」駭網站 網友當心別成跳板 |author = 謝君臨 |date = 2015年8月8日 |publisher = 《[[自由時報|自由時報]]》 |language = zh-hant |accessdate = 2015年8月5日 |quote =  }}</ref>。對此資訊及科技教育司在發現有異常流量後，隨即透過資訊安設施引導大量侵襲的目標<ref>{{Cite web |url = http://www.appledaily.com.tw/realtimenews/article/life/20150801/660127/ |title = 反課綱影響？　教部網站又被駭 |author = 蔡永彬 |date = 2015年8月1日 |publisher = 《[[蘋果日報_(台灣)|蘋果日報]]》 |language = zh-hant |accessdate = 2015年8月5日 |quote =  }}</ref>。在分散式阻斷服務攻擊發生後，中華民國教育部與相關單位便設法鑑別攻擊來源，並且在網際網路[[閘道器|閘道器]]便設法阻擋主要攻擊來源的連線，進行流量過濾與清洗工作<ref name = "中央通訊社">{{Cite web |url = http://udn.com/news/story/1/1097915 |title = 「匿名者」攻擊政府網站 政院籲網友理性 |author = [[中央通訊社|中央通訊社]] |date = 2015年8月4日 |publisher = [[聯合新聞網|聯合新聞網]] |language = zh-hant |accessdate = 2015年8月5日 |quote =  }}</ref>。

同時相關單位也隨即調查攻擊來源的所在地未於中華民國境內或者是境外，其中針對境內的攻擊來源便進一步追查來源所屬的單位、電信業者或者是用戶，而對於境外的攻擊來源則是則透過國際{{Link-en|電腦網路危機處理暨協調中心|Computer emergency response team}}通報請求協助查緝攻擊來源<ref name = "中央通訊社"/>。同時中華民國教育部亦依照資訊安全事件通報應變程序，向負責統籌政府資訊安全事務的[[行政院國家資通安全會報技術服務中心|行政院國家資通安全會報技術服務中心]]通報「二級資安事故」<ref>{{Cite web |url = http://newtalk.tw/news/view/2015-08-04/63060 |title = 匿名者再癱法務部等網站 政院：鑑識調查中 |author = 翁嫆琄 |date = 2015年8月4日 |publisher = [[新頭殼|新頭殼]] |language = zh-hant |accessdate = 2015年8月5日 |quote =  }}</ref>。不過到了下午2時，「匿名者亞洲」再度針對中華民國教育部的網站展開攻擊。對於此次攻擊，資訊及科技教育司司長[[李蔡彥|李蔡彥]]表示是這兩波攻擊是中華民國教育部近年來最大一次的攻擊行動。而在同一天，有3名頭戴象徵匿名者之[[蓋伊·福克斯面具|蓋伊·福克斯面具]]的抗議民眾出現在中華民國教育部前廣場表達抗議訴求<ref>{{Cite web |url = http://udn.com/news/story/1/1093539 |title = 匿名者面具現身教育部前廣場 |author = 鄭清元 |date = 2015年8月1日 |publisher = [[聯合新聞網|聯合新聞網]] |language = zh-hant |accessdate = 2015年8月5日 |quote =  }}</ref>。到了晚間，「匿名者亞洲」分別以繁體中文與英語各別發表一篇文章，指出此次行動並非僅有「匿名者亞洲」的成員參與，同時也要感謝海外其他匿名者團體的協助<ref>{{Cite web |url = http://www.appledaily.com.tw/realtimenews/article/politics/20150801/660305/ |title = 「匿名者」駭癱教育部　放話「這只是開始」 |author = 劉育良 |date = 2015年8月1日 |publisher = 《[[蘋果日報_(台灣)|蘋果日報]]》 |language = zh-hant |accessdate = 2015年8月5日 |quote =  }}</ref>。而在文章最後則是寫道：「#Taiwan，我們聽到你的呼喊，這只是開始。<ref>{{Cite web |url = http://news.ltn.com.tw/news/politics/breakingnews/1397982 |title = 癱瘓教育部網站後 「匿名者」預告「這只是開始」 |author =  |date = 2015年8月1日 |publisher = 《[[自由時報|自由時報]]》 |language = zh-hant |accessdate = 2015年8月5日 |quote =  }}</ref>」

== 影響 ==
=== 攻擊作為 ===
新黨主席郁慕明對於此事在Facebook發表新黨聲明稿《國際駭客入侵 全民力抗內亂外患》，指出現在中華民國內政遭到國際駭客組織以行動干預，讓臺灣面臨內亂與外患相互勾結的問題。其中新黨在聲明稿中表示內亂為假借課綱問題影響臺灣青年，並且用革命行動的方式意圖摧毀中華民國政府；並且指稱外患為配合國際勢力將[[釣魚臺|釣魚臺]]劃歸給日本，並且利用國際駭客打壓臺灣內部的反對勢力<ref>{{Cite web |url = http://newtalk.tw/news/view/2015-08-03/63005 |title = 匿名者駭教部 新黨：證明有人勾結國際勢力 |author = 翁嫆琄 |date = 2015年8月3日 |publisher = [[新頭殼|新頭殼]] |language = zh-hant |accessdate = 2015年8月5日 |quote =  }}</ref>。聲明稿中還指出國際駭客組織接連攻擊部分網站，已經證明臺灣前途實際上被國際掌控，並且指稱：「一個在台灣被說已經泡沫化的新黨，還能受到國際勢力的重視，更凸顯新黨存在的必要。<ref>{{Cite web |url = http://news.ltn.com.tw/news/politics/breakingnews/1399575 |title = 遭匿名者攻擊 新黨：證明我們存在的必要 |author =  |date = 2015年8月3日 |publisher = 《[[自由時報|自由時報]]》 |language = zh-hant |accessdate = 2015年8月5日 |quote =  }}</ref>」不過對此[[親民黨|親民黨]]副秘書長[[劉文雄|劉文雄]]則反諷說：「也許再過不久，會有情報單位，蒐集你們（學生）跟外星人溝通的證據。<ref>{{Cite web |url = http://news.ltn.com.tw/news/politics/breakingnews/1400617 |title = 反課綱遭批勾結國際勢力 劉文雄嘆：國家怎能不亂？ |author =  |date = 2015年8月4日 |publisher = 《[[自由時報|自由時報]]》 |language = zh-hant |accessdate = 2015年8月5日 |quote =  }}</ref>」

中國國民黨則表示有關課綱微調議題，臺灣社會有不同看法並且包容多元意見陳述，從不斷論辯中尋求多數共識。然而其嚴厲譴責「匿名者亞洲」憑藉片面之詞，便試圖以違法手段破壞民生系統與金融體系以達成目的。此舉除了超出課綱微調議題的範疇，還可能波及無辜臺灣百姓的生活作息或者是影響到金融秩序，同時社會大眾勢難以接受這般脅迫恐嚇的行徑<ref>{{Cite web |url = http://newtalk.tw/news/view/2015-08-05/63085 |title = 「匿名者」揚言癱瘓民生網站 藍譴責恐嚇 |author = 邱珮文 |date = 2015年8月5日 |publisher = [[新頭殼|新頭殼]] |language = zh-hant |accessdate = 2015年8月5日 |quote =  }}</ref><ref>{{Cite web |url = http://news.ltn.com.tw/news/life/breakingnews/1401569 |title = 「匿名者」點名攻擊證交所 國民黨譴責駭客 |author = 王寓中 |date = 2015年8月5日 |publisher = 《[[自由時報|自由時報]]》 |language = zh-hant |accessdate = 2015年8月5日 |quote =  }}</ref>。專欄作家[[公孫策_(台灣)|公孫策]]則表示這件事不能夠視為反課綱的新聞插曲，而是已經涉及到國家安全層次<ref>{{Cite web |url = http://udn.com/news/story/7339/1100425 |title = 去梯言／把駭客當刺客看待 |author = [[公孫策_(台灣)|公孫策]] |date = 2015年8月5日 |publisher = [[聯合新聞網|聯合新聞網]] |language = zh-hant |accessdate = 2015年8月5日 |quote =  }}</ref>。對此民主進步黨立法委員[[黃偉哲|黃偉哲]]認為發動攻擊癱瘓政府或政黨網站是不應該出現的侵權行為，但是會予以尊重。但是如果駭客成功透過網路防火牆漏洞讓駭入侵或者癱瘓銀行帳務、水電調度等民生系統，就必須以國家安全層次思考而尋求國際合作以找出惡意攻擊者<ref>{{Cite web |url = http://www.appledaily.com.tw/realtimenews/article/politics/20150805/662617/ |title = 匿名者嗆癱民生系統　綠委：應揪出 |author = 黃揚明 |date = 2015年8月5日 |publisher = 《[[蘋果日報_(台灣)|蘋果日報]]》 |language = zh-hant |accessdate = 2015年8月5日 |quote =  }}</ref><ref>{{Cite web |url = http://news.ltn.com.tw/news/politics/breakingnews/1401586 |title = 若匿名者癱瘓民生網站 黃偉哲：尋求國際合作 |author =  |date = 2015年8月5日 |publisher = 《[[自由時報|自由時報]]》 |language = zh-hant |accessdate = 2015年8月5日 |quote =  }}</ref>。

[[國立臺灣大學|國立臺灣大學]]資訊工程學系教授[[趙坤茂|趙坤茂]]呼籲自認為正義使者的駭客克制行為避免損人害己，並且表示：「網路駭客可能是替天行道的羅賓漢，也可能是『順我者生，逆我者亡』的藏鏡人。<ref name = "趙坤茂"/>」醫師[[楊聰財|楊聰財]]則認為駭客會對自己入侵電腦和癱瘓網站的行為解釋是執行正義與表達不滿，對此他表示：「駭客精於電腦，又要保持神祕感，以破壞別人的防火牆、破解密碼、入侵電腦來滿足自尊心，但最終目的還是要表達訴求。」而精神科醫師[[沈政男|沈政男]]則認為駭客雖然破壞網站但是並會造成真正的損害，實際上沒有因為訴求主張而喪失同理心。同時相較於宗教狂熱分子常用激烈手段而和原初主張相互矛盾，駭客的攻擊網站行動仍然以彰顯自己的訴求為主，不會為了造成不便利而使得行動違背初衷<ref>{{Cite web |url = http://udn.com/news/story/1/1101967 |title = 自詡行俠正義 駭客多自戀 |author = 鄭國強 |date = 2015年8月5日 |publisher = [[聯合新聞網|聯合新聞網]] |language = zh-hant |accessdate = 2015年8月5日 |quote =  }}</ref>。[[淡江大學|淡江大學]]資訊工程學系副教授[[黃仁俊|黃仁俊]]則認為分散式阻斷服務攻擊不算真正的入侵，因為其主要是向特定網站提出多筆封包需求而讓網站不斷消耗資源，然而網站癱瘓是否構成犯罪則是要基於法官的見解。同時他認為儘管政府蒐集很多IP位址，但是技術較高的駭客透過[[網絡地址轉換|網絡地址轉換]]或者共用IP位址，而會使得確認犯人的工作極為困難<ref>{{Cite web |url = http://udn.com/news/story/1/1101966 |title = 匿名者駭癱官網 抓到也難定罪 |author = 鄭國強 |date = 2015年8月5日 |publisher = [[聯合新聞網|聯合新聞網]] |language = zh-hant |accessdate = 2015年8月5日 |quote =  }}</ref>。

=== 組織結構 ===
這次針對政府和政黨網站展開攻擊以聲援反高中課綱微調運動的抗議學生活動，也讓「匿名者亞洲」在社會大眾間變得更為知名。不過在網際網路上傳出「匿名者亞洲」並非真正的匿名者組織，只是其他人模仿其發言模式。其中在Facebook專頁便有外國匿名者提供其他匿名者的網站，並且認為「匿名者亞洲」並不能代表全世界的匿名者成員。其中他表示雖然匿名者沒有領導人且任何人都可以隸屬該組織，但是具有關心特定議題與自由權益的一貫信仰<ref name = "郁慕明在找我們嗎"/>。同時也有網友認為「匿名者亞洲」僅使用分散式阻斷服務攻擊手法癱瘓網站，跟過去匿名者真正入侵政府網站的行動相差甚大，因此質疑「匿名者亞洲」是仿冒組織<ref name = "翁嫆琄">{{Cite web |url = http://newtalk.tw/news/view/2015-08-05/63105 |title = 才向台政府下通碟 匿名者臉書挺課綱發文全消失 |author = 翁嫆琄 |date = 2015年8月5日 |publisher = [[新頭殼|新頭殼]] |language = zh-hant |accessdate = 2015年8月5日 |quote =  }}</ref>。

關於許多人詢問該群體是否為亞洲管理者的詢問，但有民眾到匿名者發源的臉書詢問，得到未涉及台灣駭客行動的答案，因匿名者成立的宗旨，主為推動網路自由，反課綱等議題並非關注重點<ref>[https://disp.cc/b/163-8Slc 美國匿名者說台灣的匿名者是山寨貨!]</ref>。此外「匿名者中國」（Anonymous China）發布公告表示「匿名者亞洲」並非管理亞洲國家的匿名者組織，而是由香港民眾在Facebook上所成立的專頁<ref name = "網友建議紛沓而至"/>。不過也有網友透過「匿名者美國」（Anonymous US）官方網站上介紹「匿名者亞洲」結合各地駭客攻擊中華民國教育部等政府網站的新聞，以此代表「匿名者亞洲」實際上是屬於匿名者的行動中<ref name = "翁嫆琄"/>。對此「匿名者亞洲」承認該專頁為香港民眾所建立，原本的用意在於聚合亞洲地區支持匿名者理念的成員交流，所成立的以漢語以及英語溝通的平臺。同時「匿名者亞洲」也表示在「匿名者臺灣行動」中並沒有管理過任何團隊，所有行動都是各個團隊自由參與進行<ref name = "網友建議紛沓而至"/>。

=== 行動方式 ===
對於「匿名者亞洲」攻擊臺灣銀行、臺灣證券交易所等網站的作為，引起部份網友對於攻擊的對象選擇予以質疑<ref name = "沒在怕"/>。其中一部分意見認為攻擊中華民國教育部等網站無法造成影響，必須透過攻擊民生相關網站才能夠讓其他民眾有立即反應。但另一部分意見則認為「匿名者亞洲」癱瘓一些政黨網站以表達訴求即可，對於影響民生的重要網站則不要展開攻擊，此外也有部分人士認為攻擊課綱議題以外的網站將會模糊焦點。同時還有部分網友主張應當像[[愛德華·史諾登|愛德華·史諾登]]般公開機密數據，但是亦有網友擔心「匿名者亞洲」會入侵銀行等金融機構並且修改個人帳戶<ref name = "T客邦"/>。

對此「匿名者亞洲」在Facebook專頁上公開表示在這次協助臺灣的行動中獲得許多人意見，認為應該駭取特定資料或者批評手法技術層次較低；但是在文章中提到許多參與行動的成員亦也是普通生活的民眾，呼籲彼此之間應該要互相尊重<ref>{{Cite web |url = http://www.appledaily.com.tw/realtimenews/article/politics/20150803/661348/ |title = 網友下指導棋　「匿名者」籲彼此尊重 |author = 劉育良 |date = 2015年8月3日 |publisher = 《[[蘋果日報_(台灣)|蘋果日報]]》 |language = zh-hant |accessdate = 2015年8月5日 |quote =  }}</ref>，並且表示：「最低級的方式能引起國家注意事情嚴重性的時候，可能會是一個新的轉折點，改變到現在台灣的局面。」而在這篇公告發表後引起部分批評，但是也有網友留言表達感謝與支持<ref name = "網友建議紛沓而至">{{Cite web |url = http://news.ltn.com.tw/news/life/breakingnews/1399616 |title = 網友建議紛沓而至 匿名者籲互相尊重 |author =  |date = 2015年8月3日 |publisher = 《[[自由時報|自由時報]]》 |language = zh-hant |accessdate = 2015年8月5日 |quote =  }}</ref>。
