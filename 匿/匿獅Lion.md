'''匿獅Lion'''是[[香港特區政府|香港特區政府]][[衞生署_(香港)|衞生署]]為2018開展的「健康香港2025｜郁一郁．健康啲」宣傳運動而設計的[[吉祥物|吉祥物]]。他於2018年10月22日開設[[Facebook|Facebook]]專頁和[[Instagram|Instagram]]，從此亦以此日為「匿獅Lion生日」。

匿獅Lion於2018年12月8日的「健康香港2025｜郁一郁．健康啲」啟動禮隆重登場，並於同日在大氣電波中首次亮相。在半分鐘的電視宣傳廣告內，匿獅Lion反映香港人的生活習慣，例如一回家即躺在[[沙發|沙發]]上、玩[[手機|手機]]、乘搭[[電梯|電梯]]不走[[樓梯|樓梯]]，直接道出港人缺乏體能活動的情況。

== 背景 ==
* 廣大市民了解運動的好處，也清楚運動能改善身體狀況，然而數據反映港人缺乏體能活動。
* 在電視宣傳廣告內，匿獅Lion為了健康而作出改變去開始「郁一郁」，例如：做家務、與小朋友玩耍、行樓梯等等，從而鼓勵市民像匿獅Lion一樣踏出第一步「郁一郁」。
* 「健康香港2025｜郁一郁．健康啲」宣傳運動，旨在透過不同活動和媒介，並借助匿獅Lion的可愛形象，鼓勵市民將體能活動融入生活當中，做到隨時隨地都可以「郁一郁」。
* 除電視和電台宣傳廣告，匿獅Lion也在網上世界與大眾緊密接觸和互動。

== 設計概念 ==
* 匿獅Lion中的「匿」字，取粵語中「匿埋」之意，指匿獅Lion常足不出戶，在家中休息，自然缺乏體能活動及運動。
* 匿獅Lion和與大部份香港人一樣，了解做運動的好處，卻以很多藉口拒絕運動，如工作太忙、天氣惡劣，這正好反映香港人生活習慣。
* 匿獅Lion是一個反面教材，提醒香港人要擊退藉口，提高體能活動量，多點「郁一郁」。

== 匿獅Lion性格小檔案 ==
* 性別：男
* 生日日期：10月22日
* 星座：典型的天秤座
* 身型：微胖，有一個小肚腩
* 最愛睡覺和攤抖，慢活就是他的做「獅」宗旨。他認為上班上學令人身心疲倦，所以平時要多點休息，長年累月就變成懶惰。
* 但他明白運動帶來的好處，心態上亦想多點郁動，但卻討厭疲累的感覺，因而缺乏動力並給自己各種藉口懶去行動。
* 有點小聰明，會想盡辦法去偷懶，認為懶惰才是最愜意的生活方式。
* 喜歡交際，享受與朋友玩樂，不過聚會的內容不要與運動有關。

== 「健康香港2025｜郁一郁．健康啲」宣傳運動背景 ==
* 非傳染病是導致健康欠佳、傷殘和早逝的主要原因。 
* 香港政府確認預防和控制非傳染病是一個優先事項，並於2018年發表《邁向2025：香港非傳染病防控策略及行動計劃》以解決非傳染疾病日益嚴峻的威脅。文件列明了須在2025年或之前實現的9項本地目標，當中包括目標三「減少體能活動不足」 
* 為推動及鼓勵市民自覺或不自覺地把體能活動融入日常生活，衞生署開展「健康香港2025｜郁一郁．健康啲」宣傳運動，匿獅Lion也因而誕生，以推廣相關健康概念 
* 可從以下網頁獲取更多有關「健康香港2025｜郁一郁．健康啲」宣傳運動的背景資料：https://www.change4health.gov.hk/tc/saptowards2025/target3.html
