'''庾传素'''（{{bd|9世纪||10世纪|catIdx=Y}}），[[中国|中国]][[五代十国|五代十国]]时期[[政权|政权]][[前蜀|前蜀]][[官员|官员]]，两度官拜[[宰相|宰相]]。

== 王建年间 ==
庾传素生年及家世源流不详。效力前蜀开国皇帝[[王建_(前蜀)|王建]]，任[[蜀州_(唐朝)|蜀州]][[刺史|刺史]]。<ref name=SGCQ41>[http://archive.org/stream/06080952.cn#page/n160/mode/2up 《十国春秋》卷四十一]</ref>[[武成_(王建)|武成]]三年（910年）十二月，在[[户部侍郎|户部侍郎]]判度支任上被任为[[中书侍郎|中书侍郎]]、[[同中书门下平章事|同中书门下平章事]]，为[[实质宰相|实质宰相]]。<ref name=ZZTJ267>[[:s:資治通鑑/卷267|《资治通鉴》卷二百六十七]]</ref>

[[天汉_(王建)|天汉]]元年（917年），权势[[宦官|宦官]]飞龙使[[唐文扆|唐文扆]]与庾传素的政治盟友[[司徒|司徒]]、判枢密院事[[毛文锡|毛文锡]]争权。庾传素时任[[左仆射|左仆射]]兼中书侍郎、同平章事，毛文锡将要把女儿嫁给庾传素的儿子，七月，在枢密院召集亲族奏乐，但没有事先上表告知王建，王建听闻乐声，觉得奇怪，唐文扆趁机诬陷。八月，毛文锡被贬，庾传素也罢为[[工部尚书|工部尚书]]，而由庾传素的再从弟翰林学士承旨[[庾凝绩|庾凝绩]]权判内枢密院事。<ref name=ZZTJ270>[[:s:資治通鑑/卷270|《资治通鉴》卷二百七十]]</ref>不久庾传素又被任为[[兵部尚书|兵部尚书]]。<ref name=SGCQ41/>

== 王衍年间 ==
[[光天_(王建)|光天]]元年（918年）六月，王建崩，子[[王衍_(前蜀)|王宗衍]]继位，改名[[王衍_(前蜀)|王衍]]。七月，王衍任庾传素为[[太子少保|太子少保]]兼中书侍郎、同平章事，再拜为宰相。<ref name=ZZTJ270/>在相位无显著政绩。他和同僚宰相[[王锴|王锴]]都只顾自保，不能纠正年轻皇帝的浪费和轻率的作风。<ref name=ZZTJ272>[[:s:資治通鑑/卷272|《资治通鉴》卷二百七十二]]</ref>庾传素最初任蜀州刺史时，有唐兴县郎吏杨会事他甚谨，这时他就除杨会长马（[[长史|长史]]、[[司马|司马]]）一职以为酬劳，杨会坚辞：“我杨会的吏才远近皆知，不堪为官，宁可掩人口实。且舍弃数千家的款待而得一长马虚名，没有好处。”时人称其有识。[[咸康_(王衍)|咸康]]元年（925年），前蜀被东北邻国[[后唐|后唐]]魏王[[李继岌|李继岌]]所攻，<ref name=ZZTJ274>[[:s:資治通鑑/卷274|《资治通鉴》卷二百七十四]]</ref>庾传素随王衍降唐，[[同光|同光]]四年（926年）正月，<ref>《十国春秋》卷三十七误作同光三年（925年）事。</ref>李继岌遣[[凤翔节度使|凤翔节度使]][[李继曮|李继曮]]、客省使[[李严_(后唐)|李严]]部送王衍、前蜀宗族及宰相王锴、[[张格|张格]]、庾传素、[[许寂|许寂]]、[[翰林学士|翰林学士]][[李昊|李昊]]等及将佐家族数千人向东到都城[[洛阳|洛阳]]。三月，到[[长安|长安]]。庄宗正要东征平定兵变，担心王氏族党生变，遣中使[[向延嗣|向延嗣]]带诏书前去杀王衍，诏书称“王衍一行，并从杀戮。”诏书已盖玉玺，[[枢密使|枢密使]][[张居翰|张居翰]]将“行”字改为“家”字，前蜀百官才幸免于难。<ref>《十国春秋》卷三十七</ref>六月，庾传素等到洛阳，[[后唐明宗|后唐明宗]]宣王锴、张格、庾传素及原蜀[[御史中丞|御史中丞]][[牛希济|牛希济]]，各赐一韵，让他们试作蜀主降唐诗五十六字，王锴等都言及王衍僭号、荒淫、失国，只有牛希济不谤故主。<ref>《十国春秋》卷四十四</ref>庾传素被授刺史。<ref name=SGCQ41/>后事及卒年不详。

== 评价 ==
*《[[十国春秋|十国春秋]]》论曰：庾传素保位旷官（自保职位，不称职），一筹莫展，岂所云端揆（指相位）之佐哉！呜呼！择相顾可不慎邪！
