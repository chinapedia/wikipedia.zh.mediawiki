'''雞骨草'''（[[學名|學名]]：''{{lang|la|Abrus cantoniensis}}''，[[英文|英文]]名稱：''{{lang|en|Prayer-beads}}''）<ref>[http://www.hkherbarium.net/Herbarium/c_search_end.asp?id=3924 Abrus cantoniensis]香港植物標本室</ref>，別稱'''廣州相思子'''、'''廣東相思子'''、''' 大黃草'''、'''黃頭草'''、'''黃食草'''、'''黃仔強'''、'''豬腰草'''、''' 紅母雞草'''、'''小葉龍鱗草'''、''' 細葉龍鱗草'''、''' 假牛甘子'''、'''山彎豆'''、''' 石門檻'''及'''地香根'''等<ref>[http://www.cfh.ac.cn/22770.sp CFH物种信息卡]中國自然標本館</ref><ref name="意古中醫">[http://www.tcmlib.com/zy/html77/showdetail-373633362ce9b8a1e9aaa8e88d892c7a79.html 雞骨草] {{webarchive|url=https://web.archive.org/web/20090608012808/http://www.tcmlib.com/zy/html77/showdetail-373633362ce9b8a1e9aaa8e88d892c7a79.html |date=2009-06-08 }} 意古中醫</ref>，是[[豆科|豆科]][[相思子屬|相思子屬]]的一種[[植物|植物]]<ref name = "浸會大學">[http://libproject.hkbu.edu.hk/was40/detail?record=1&channelid=1288&searchword=%28Abrus+cantoniensis%29&extension=all&sortfield=%2Bname_formal_latin_sort 藥用植物圖像數據庫]香港浸會大學中醫藥學院</ref>，常見於[[中國|中國]][[華南|華南]]地區，具清熱利濕、益胃健脾的功能。因本種首先發現於[[廣州|廣州]][[白雲山|白雲山]]，故而有廣州相思子之稱<ref name = "當代藥用植物典4">{{cite book|title=《當代藥用植物典 4》|author= |series=蕭培根|publisher=香港賽馬會中藥研究院|page=第2-4頁|isbn=978-988-99226-3-4|date=2007年8月}}</ref>。

==分佈==
分佈於[[中國|中國]]、[[泰國|泰國]]等地。中國國內分佈於[[廣東|廣東]]、[[廣西|廣西]]、[[湖南|湖南]]等省份。多生長於[[海拔|海拔]]約200米的山地、疏林或[[灌木叢|灌木叢]]中<ref name="中國植物志">[http://frps.plantphoto.cn/pdf2.asp?id=9987 中國植物志]{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}第126頁</ref><ref name = "當代藥用植物典4"/>。

==形態特徵==
雞骨草是一種[[爬藤植物|爬藤植物]]，高約1－2[[米|米]]。主[[根|根]]粗壯，長約60厘米。[[莖|莖]]深紅紫色，枝條細直平滑，幼枝密披白色或黃褐色柔毛，老時柔毛脫落。[[羽狀複葉|羽狀複葉]]互生，[[偶數|偶數]]小葉6－11對，長圓形或倒卵狀長圓形，膜質，小[[葉柄|葉柄]]短，先端截形，有小銳尖，基部淺心形，[[葉脈|葉脈]]向兩面凸起，表面披疏毛，背面披糙伏毛，長約0.5－1.5厘米，寬約0.3－0.5厘米；[[托葉|托葉]]線狀披針形，成對著生。花為總狀[[花序|花序]]，腋生，蝶形花小，聚生於花序總軸的短枝上，長約6毫米；萼鐘狀；花梗短；[[花冠|花冠]]淡紫色或紫紅色，突出；[[雄蕊|雄蕊]]9枚，與旗瓣貼連，合生成管狀，上部分離；[[花柱|花柱]]短，[[子房|子房]]近無柄。果為[[莢果|莢果]]，長圓形，頂端具喙，扁平，披稀疏的白色或淡黃色糙伏毛，果熟時淺褐色，長約3厘米，寬約1.3厘米。[[種子|種子]]矩圓形，4－5顆，成熟時黑褐色，有明顯蠟黃色的種阜，中間有孔，邊緣具長圓狀環<ref name = "意古中醫"/><ref name = "浸會大學"/><ref name = "中國植物志"/>。

==醫藥用途==
{{medical}}
雞骨草的種子有毒須除去豆莢後，乾燥全草入藥。中藥名為雞骨草，始載於《[[嶺南採藥錄|嶺南採藥錄]]》，味甘、苦，性涼，無毒，歸心、[[肺|肺]]、[[肝|肝]]、[[胃|胃]]、[[腎|腎]]經，中醫歸類為[[清熱解毒藥|清熱解毒藥]]、[[疏肝藥|疏肝藥]]，虛寒體弱者慎用。藥材主產於[[廣東|廣東]]、[[廣西|廣西]]等地。有清熱利濕、疏肝止痛、活血散[[瘀|瘀]]等之功效，主治[[胃痛|胃痛]]、[[風濕骨痛|風濕骨痛]]、[[黃疸|黃疸]]型[[肝炎|肝炎]]、[[乳癰|乳癰]]、跌打瘀痛等等症狀。現代[[藥理|藥理]]研究表明雞骨草具[[抗炎|抗炎]]、保肝、促進[[免疫功能|免疫功能]]等作用，臨床則應用於肝硬化腹水、急慢性肝炎、[[膽囊炎|膽囊炎]]、[[瘰癧|瘰癧]]、外感風熱、蛇咬傷等治療上<ref name = "意古中醫"/><ref name = "浸會大學"/><ref name = "當代藥用植物典4"/>。雞骨草為中國[[嶺南|嶺南]]民間常用[[草藥|草藥]]，常作為[[湯料|湯料]]及[[涼茶|涼茶]]之用<ref name = "當代藥用植物典4"/>。

==藥材鑑定==
本品乾燥全草，全年均可採收，除去豆莢及雜質後曬乾；多纏繞成束狀；根圓柱形或圓錐形，質硬，表面有細縱紋，分枝長短粗細不一，直徑約3－15毫米；根莖短，結節狀；莖叢生，長藤狀，灰棕色至紫褐色，直徑約1.5－2.5毫米，最長可達1米，幼枝纖細，棕紅色，疏短柔毛；偶數羽狀複葉互生，小葉長圓形，背面披糙伏毛，長約8－12毫米；氣微，味微苦<ref name = "中藥材鑑別圖典">{{cite book|title=《常用中藥材鑑別圖典》|author= |series=趙中振、陳虎彪|publisher=萬里機構|page=第450頁|isbn= 978-962-14-4150-8|date=2010年1月}}</ref><ref>{{cite book|title=《香港中藥材圖鑑》|author= |series=趙中振|publisher=香港浸會大學中醫藥學院|page=第424頁|isbn= 988-97448-1-3|date=2003年10月}}</ref>。本種載錄於《[[中國藥典|中國藥典]]》2005年版，定為中藥雞骨草的法定原植物來源種，主要透過性狀、粉末特徵等鑒別資料，以控制[[藥材|藥材]]的質量<ref name = "當代藥用植物典4"/>。傳統經驗則認為以根莖結節、莖葉全者為佳<ref name = "中藥材鑑別圖典"/>。

==化學成份==
雞骨草全草含[[生物鹼|生物鹼]]、[[三萜類|三萜類]]、[[三萜皂苷類|三萜皂苷類]]、[[蒽醌類|蒽醌類]]等化學成分組成，如[[相思子鹼|相思子鹼]](abrine)、[[膽鹼|膽鹼]](choline)、[[大黃酚|大黃酚]](chrysophanol)、[[大黃素甲醚|大黃素甲醚]](physcion)、[[雞骨草二醇|雞骨草二醇]](cantoniensistriol)、[[槐花二醇|槐花二醇]](sophoradiol)、[[甘草次酸|甘草次酸]](glycyrrhetinic acid)等<ref name = "意古中醫"/><ref name = "當代藥用植物典4"/>。
