'''穢多'''（{{jpn|j=穢多|hgt=ゑた|hg=えた}}）乃是[[日本|日本]]中世紀時代兩種[[部落民|部落民]]之一，受到[[日本佛教|日本佛教]]與[[神道|神道]]中「污穢」觀念影響，對於「較污穢的工作」、「較污穢的人（[[犯罪|罪人]]）維生的行業」的稱呼。穢多的起源不明，有人推測是逃亡的[[農民|農民]]，後來變成從事皮革加工的「部民」；亦有人認為是對於[[非人|非人]]身分的俗稱，源自古代被征服的[[生口|奴隷]]。雖然穢多的起源有各式各樣的說法，但是跟非人的差別為[[世襲制|世襲制]]，子女必須繼承父母的職業。

== 關於階級制之議論 ==
=== 編入階級統治的解釋 ===
穢多乃成形於[[平安時代|平安時代]]、確立於[[江戶時代|江戶時代]]，雖然該稱呼廢止於[[明治|明治]]時代，但現代日本人對此蔑稱帶有「[[四民|士農工商]]之外的最下層身分」的歧視。[[關西大學|關西大學]]講師{{link-ja|上杉聰|上杉聰}}認為，[[鎌倉時代|鎌倉時代]][[奈良|奈良]]和[[京都|京都]]對於穢多便出現歧視，[[室町時代|室町時代]]的[[歷史|歷史]]文獻更出現「不要跟卑賤者[[結婚|結婚]]，一旦弄髒[[血液|血液]]的話就無法乾淨，穢多的子女永遠還是穢多」<ref>原文作：「卑しい者とは結婚しない。血は一度汚れるときれいにはならない。穢多の子はいつまでも穢多である」，參考{{ja}}[http://www.kyoken.jp/36/houkoku36/1houkoku36.htm 第36回部落解放研究京都市集会：部落の歴史 - 天皇制と部落問題]。</ref>等歧視性字句。[[京都大學|京都大學]]名譽教授、歷史學者{{link-ja|朝尾直弘|朝尾直弘}}則認為，「[[士大夫|士]]和農工商之間存有極大的身分差異，大體上農工商三者同列，也可視為[[平民|平民]]百姓的統稱。至於更下級則有稱為『穢多、[[非人|非人]]』的階層，因此士、農工商、『穢多、[[非人|非人]]』三種階層間存在明顯巨大的分隔線」<ref>參考《日本歴史の中の被差別民》，奈良人権部落解放研究所著，新人物往来社，2010年3月6日，ISBN 978-4404038241，頁100。</ref>。

=== 非編入階級統治的解釋 ===
不過，也有人認為[[幕府_(日本)|幕府]]為了[[政治|政治]]的考量劃分「士農工商穢多非人」等階級的說法是毫無根據的<ref>《身分差別社会の真実》，斎藤洋一、大石慎三郎合著，[[講談社|講談社]]，1995年7月17日，ISBN 978-4061492585。</ref>。[[日本歷史|日本史]]學者{{link-ja|田中圭一 (日本史學者)|田中圭一 (日本史学者)|田中圭一}}提出：「本來士農工商就是[[職業|職業]]的劃分，用此事來區隔身分是愚蠢的」<ref>《百姓の江戸時代》，田中圭一著，[[筑摩書房|筑摩書房]]，2000年11月，ISBN 978-4480058706，頁58。</ref>。

== 名稱的由來 ==
日本文獻上首見「穢多」一詞乃[[永仁_(日本年号)|永仁]]4年（1296年）的《天狗草紙繪卷》<ref>{{ja}}[http://www.emuseum.jp/detail/100262/000/000?mode=detail&d_lang=ja&s_lang=ja&class=%20&title=%20&c_e=%20&region=%20&era=%20&century=%20&cptype=%20&owner=%20&pos=177&num=8 e國寶：天狗草紙絵巻]。</ref>，編纂於[[承平_(朱雀天皇)|承平]]5年（935年）的第一部[[類書|類書]]《[[和名類聚抄|和名類聚抄]]》將「屠兒」[[訓讀|訓讀]]成「惠止利（{{lang|ja|えとり}}）」，解釋則是「屠宰[[牛|牛]][[馬|馬]]之[[肉|肉]]作為[[鷹|鷹]][[雞|雞]]之餌」。[[建治|建治]]元年（1275年）稻荷山的經尊法橋獻給[[北條實時|北條實時]]的語源[[詞典|辭書]]《名語記》也記載著：「住在河灘邊、食用牛馬的人為穢多（{{lang|ja|ゑた}}）」、「穢多乃餌取也，屠兒等同穢多」。同時期的另一本類書《{{lang-ja|塵袋|塵袋}}》則記錄：「根本上餌取乃以肉塊作為餵食[[鷹|鷹]]之餌」；[[正保|正保]]元年（1644年）以降的[[江户幕府|江户幕府]][[公文|公文]]則開始出現這個詞。由這些文獻記錄可知，穢多本為飼育鷹之[[主鷹司|主鷹司]]取餌之職業，後來轉換成對[[屠宰|屠夫]]的總稱<ref>《歴史のなかの米と肉―食物と天皇・差別》，原田信男著，平凡社，2005年6月，ISBN 978-4582765410。</ref>。受到佛教戒殺生和神道忌血污的思想所及，對於[[屠宰|殺生]]這種多污穢的工作遂以「穢多」稱之。

時至今日，有人以[[片假名|片假名]]「{{lang|ja|エッタ}}」稱呼，某些地方甚至出現「{{lang|ja|エッタボシ}}」（即讀成「穢多法師」）<ref>參看{{ja}}[http://www.mognet.org/hansen/japan/shinshu05.html 真宗（仏教）とハンセン病差別問題について]，「註2亀川村事件」之段落。</ref>的歧視稱呼。2011年1月5日[[奈良縣|奈良縣]][[御所市|御所市]]所設立的水平社博物館展出「[[日韓合併|日韓合併]]」的展覽期間，發生了抗議者以「{{lang|ja|エッタ}}」稱呼，結果鬧進法庭之事件<ref>詳情參看{{link-ja|水平社博物館前歧視抗議事件|水平社博物館前差別街宣事件}}或《奈良地方裁判所平成23年（ワ）第686号 平成24年6月25日民事部判決》本文。</ref>。

== 職業 ==
[[File:Eta.jpg|thumb]]
穢多的[[職業|職業]]從日本中世紀以來逐漸成形，直到[[江戶幕府|江戶幕府]]時期甫確立其制度。[[仁德天皇|仁德天皇]]時便以[[鷹|鷹]]捕捉其他[[鳥|鳥]]類，[[大宝_(日本)|大寶]]元年制定的《[[大寶令|大寶令]]》設置[[主鷹司|主鷹司]]，為了餵食鷹鳥，開始出現屠宰餌料之人。到了[[奈良時代|奈良時代]]穢多的原形亦曾見諸史料，此時代初期編纂的《[[播磨國風土記|播磨國風土記]]》〈飾磨郡大野里〉記載「惠多」<ref>按「穢多」、「惠多」二詞之日語發音相同。</ref>一詞。[[鎌倉時代|鎌倉時代]]的《天狗草紙繪卷》出現「穢多」一詞，描繪著著四條河灘上的穢多小孩抓住[[天狗_(日本)|天狗]]化身之[[鳶|鳶]]，將之扭斷[[頸|頸部]]殺害後處理其屍肉的畫面；身旁另一個小孩試圖以網捕捉另一隻鳶<ref>{{ja}}[http://tokyo-metanoia.cocolog-nifty.com/blog/2011/12/post-a46e.html 「天狗」の天敵]。</ref>。

[[平安時代|平安時代]]中期《[[延喜式|延喜式]]》雖然規定須供上[[豬|豬]]、[[鹿|鹿]]之肉給[[天皇|天皇]]，可是受到[[日本佛教|佛教]]禁[[殺生|殺生]]的思想影響，蔑視[[屠宰|屠夫]]的風氣也逐漸散播。這些人群居於[[京都|京都]][[鴨川_(淀川水系)|鴨川]]河灘[[沙洲|沙洲]]的小屋，從事其他人避之唯恐不及的賤業。因為[[沙洲|沙洲]]靠近[[賀茂御祖神社|下鴨神社]]，為了避免遭受穢氣波及，《[[延喜式|延喜式]]》特別規定該[[神社|神社]]南面附近禁止濫僧、屠夫居住<ref>{{ja}}[http://www.aozora.gr.jp/cards/001344/files/54861_49954.html 《濫僧考 - 河原者・坂の者・宿の者・非人法師》]，{{link-ja|喜田貞吉|喜田貞吉}}。</ref>。「濫僧」即[[非人|非人]]法師，本為平民但因無法忍受[[國司|國司]]的苛政，[[出家|出家]]脫離原籍。平安時代中期的[[公卿|公卿]]{{link-ja|三善清行|三善清行}}曾以《{{link-ja|意見十二箇條|意見十二箇条}}》上書給[[醍醐天皇|醍醐天皇]]云：「今天下之民三分之二乃禿首者（意指濫僧）」，可見當時濫僧和屠夫仍有所區隔。這些人也有從事打掃街道的工作，被稱作淨人（{{lang|ja|きよめ}}）。《[[壒囊鈔|壒囊鈔]]》記載：居住在河灘者稱作穢多，也被稱作坂者、散所者，其中以清水坂（位於今[[京都市|京都市]][[東山區_(日本)|東山區]]）比較出名。清水坂的坂者受到[[祇園|祇園]]感受院所屬的{{link-ja|犬神人|犬神人}}（清掃[[神社|神社]]內污穢或動物屍體的下級神官）召喚，在延喜寺僧兵出兵之際擔任先鋒。派駐到各部落警戒時接受居民給予的報酬，坂者的頭目也被稱作長吏法師<ref>{{ja}}[http://www.aozora.gr.jp/cards/001344/files/49813_44335.html 喜田貞吉：長吏名称考]。</ref>。各長吏法師圈繩定界，擁有自己的勢力範圍，[[寬元|寬元]]年間曾發生清水坂和奈良坂的長吏相互爭執的情事。

成書於[[明應|明應]]9年（1500年）的《{{link-ja|七十一番職人歌合|七十一番職人歌合}}》，其中第三十六番即描繪正在鞣皮的穢多，其形象為打赤膊、束髮、赤腳。直至[[战国时代_(日本)|戰國時代]]，[[皮革|皮革]]成為[[盔甲#日本的盔甲|鎧甲]]與馬具的主要原料，從事皮革加工相關人員被納入軍需產業而受到保護。甚至曾有東日本的[[大名_(称谓)|大名]]為了增加領地內的軍事需求，特地自西日本延請穢多過來定居的例子。[[江戶時代|江戶時代]]確立鎖國政策，禁止[[東南亞|東南亞]]輸入皮革製品，造成皮革來源不足。為此，殺牛宰馬的工作受到嚴格的控制，各農村配置穢多以便獲得皮革來源。這些穢多多半居住在農村外不適合農業耕作的河川兩側，後來逐漸被稱呼為「{{lang|ja|かわた}}」（[[漢字|漢字]]寫成皮田、皮多、川田、革多、革田、河田等）。例如《慶長播磨國圖》記載「{{lang|ja|かわた}}」的地區多達48處，《攝津國圖》中也出現7處記載成「皮田」、「皮多」、「川田村」、「河原村」、「{{lang|ja|カワラ}}村」的地方<ref>請參考{{ja}}[https://www.u-sacred-heart.ac.jp/library/display_collection/digital_gallaery/Wa291_038_Z3/0986291.html 聖心女子大学図書館デジタルギャラリー：摂津国図]。</ref>；[[寬永|寬永]]21年（1644年）《[[河内國|河内國]]更池村文書》第一卷〈河州丹北郡之内更池村家數人數萬改帳〉改用「河田」表示穢多。

演變到後來，處理擊斃牛馬（禁止屠殺）、獸皮加工、皮革製品的製造販售、祭禮的清掃工作、表演物品（包含藝人）的管理、草履雪駄、燈芯、筬（需要高度技術的紡織機零件）、竹精細加工等物品的製造販售……等，這些職業變成了由穢多寡占的家業。此外[[關東地方|關東地方]]的{{link-ja|彈左衛門|弾左衛門|淺草彈左衛門}}乃由[[非人|非人]]管理<ref>{{ja}}[http://www.asahi-net.or.jp/~mg5s-hsgw/tkburaku/history/cyouri.html 弾左衛門の支配下にあった、江戸の被差別民衆]。</ref>，故雖然皆稱穢多，卻東、西有別。17世紀後半以降，各幕藩領主遂定調以「{{lang|ja|かわた}}」稱呼穢多，各地的公文書類則記成皮多、皮田等。甚至像[[廣島藩|廣島藩]]到[[幕末|幕末]]為止，皆以「革田」作為指稱穢多的公家用語。

到了[[明治|明治]]4年8月28日（1871年10月12日）頒布{{link-ja|解放令|解放令}}，廢止穢多、[[非人|非人]]之身分和稱呼，同時此二者也喪失死牛馬取得權與前段所述長期寡占之職業<ref>{{ja}}[http://www.ranhaku.com/web06/02seiji_gyosei/01mei03.html 明治の身分解放令]。</ref>，於是陷入經濟困頓。加上解放令一併廢除地租改正、徴兵令、學制、{{link-ja|村請制度|村請制度}}等，引發民眾強烈反彈，終致發展成{{link-ja|解放令反對一揆|解放令反対一揆}}，各地有許多穢多、非人遭受襲擊。

== 外國類似的階層 ==
[[印度|印度]][[種姓制度|種姓制度]]中的[[贱民_(印度)|賤民]]（又稱達利特人）和日本的穢多一樣，也是處理[[皮革|皮革]]、[[屍體|屍體]]的低下階層。他們被視為不可接觸的人，絕對禁止與其他種姓接觸，甚至常發生賤民因接觸其他種姓而遭到虐待、殺害的事情<ref>[http://www.epochtimes.com/b5/8/7/16/n2193720.htm 大紀元：印度種姓制度陰影不散　賤民遇害事件頻仍]。</ref>。在[[韓國|韓國]]，[[李氏朝鮮|李氏朝鮮]]時代開始確立「八般私賤」中[[白丁|白丁]]為最卑下者。不過1910年[[日韓合併|日韓合併]]，[[大日本帝國|大日本帝國]]廢除身分制度、導入[[戶籍制度|戶籍制度]]；加上1950年爆發[[朝鮮戰爭|朝鮮戰爭]]，受到大規模的人口移動以及戶籍資料遺失等影響，許多人原本的身分階級不明。現在韓國的不平等階級待遇已漸漸消失，但仍有人在吵架爭論時以「白丁」羞辱對方。
