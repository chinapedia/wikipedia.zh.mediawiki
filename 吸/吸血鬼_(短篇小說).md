《'''吸血鬼'''》（[[英語|英語]]：{{lang|en|''The Vampyre''}}）是由[[約翰·威廉·波里杜利|約翰·威廉·波里杜利]]作於1819年的[[散文|散文]]式[[短篇小說|短篇小說]]，這部作品通常被認為是[[浪漫主義|浪漫主義]]{{tsl|en|Vampire literature|吸血鬼文學|吸血鬼}}[[奇幻作品|奇幻]]先驅。<ref name=":0">{{Cite journal|url = https://www.researchgate.net/publication/283318599_Vampire_Evolution|title = Vampire Evolution|last = Jøn|first = A. Asbjørn|date = 2003|journal = mETAphor|doi = |pmid = |access-date = 25 November 2015|pages = |page = 21|issue = 3}}</ref>{{tsl|en|Christopher Frayling|克里士多福·佛瑞靈}}描述此作為「首部成功融合了諸多{{tsl|en|Vampire lifestyle|吸血鬼式生活型態|吸血鬼}}元素而使其臻於一個成熟文學流派」的短篇小說。<ref name=Frayling_1992>

== 出場人物 ==
*{{tsl|en|Lord Ruthven (vampire)|盧希梵爵士}} — 一名優雅倜儻的[[英國人|英國]]貴族吸血鬼
*奧布瑞 — 一名富有的年輕紳士，也是一個孤兒
*蘭黛 — 奧布瑞和盧希梵結伴旅遊時認識的一名美麗[[希臘人|希臘]]女子
*奧布瑞的姊妹 — 與馬斯登伯爵訂婚
*馬斯登伯爵 — 即盧希梵爵士，他的另一個頭銜

== 電影改編 ==
{{lang|en|Britannia Pictures}} 在2016年公佈了將《吸血鬼》改編成電影的計劃，預計於2018年開始製作，將會在[[英國|英國]]、[[義大利|義大利]]和[[希臘|希臘]]為電影取景。<ref name="imdbpro2016">{{cite web|url=http://pro.imdb.com/title/tt4452568/ | accessdate=27 February 2017 | title=PRO The Vampyre}}</ref>羅溫·M·安希將擔任導演並預計於2018年10月上映。<ref name="imdb2016">{{cite web|url=http://www.imdb.com/title/tt4452568/ | accessdate=27 February 2017 | title=IMDb The Vampyre}}</ref>

1945年曾有一部[[英國電影|英國電影]]《{{tsl|en|The Vampire's Ghost|吸血幽靈}}》是改編自這部短篇小說，不過做了較大改動，比如由[[約翰·雅博|約翰·雅博]]飾演的盧希梵爵士易名為「韋柏·法隆」，故事場景也從英國和希臘變為[[非洲|非洲]]。同樣，一套1992年由[[BBC|BBC]]出品的[[電視電影|電視電影]]《{{tsl|en|The Vampyr: A Soap Opera|吸血殭屍：驚情二百年}}》是基於這部小說以及[[海因里希·馬施納|海因里希·馬施納]]的1828年歌劇《{{tsl|en|Der Vampyr|吸血鬼 (歌劇)|吸血鬼}}》改編，片中盧希梵爵士一角被易名為「雷普利」。這位雷普利在18世紀末陷入沈睡，於現代甦醒並搖身變為一個成功的[[實業家|實業家]]。
