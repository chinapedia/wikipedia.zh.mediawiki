{{日本作品|我命中-{注}-定的人<ref name="日本電視台"/>|ボク、運命の人です。|I'm Your Destiny}}，為[[日本電視台|日本電視台]]聯播網於「[[日本電視台週六連續劇|土十]]」時段播出的連續劇<ref name="sponichi278000">{{Cite news|url= http://www.sponichi.co.jp/entertainment/news/2017/02/18/kiji/20170218s00041000278000c.html|title=日テレ「土10」第1弾は亀梨＆山Pが12年ぶり共演 主題歌も|newspaper=[[Sponichi_Annex|Sponichi Annex]]|publisher=[[スポーツニッポン|スポーツニッポン]]|date=2017-02-18|accessdate=2017-02-20}}</ref>。由[[龜梨和也|龜梨和也]]主演。

2017年4月15日起每週六晚間10:00－10:54於[[日本電視台|日本電視台]]播出，首集為14分鐘加長版（晚間10:00－11:08）。主題曲由擔任主演的[[龜梨和也|龜梨和也]]與[[山下智久|山下智久]]組成的二人團體「龜與山P」合唱，這是兩人繼2005年合組「修二與彰」合唱日劇《[[改造野豬妹|改造野豬妹]]》主題曲之後，第二度以搭檔的方式演唱。

== 登場人物 ==
* 正木誠：[[龜梨和也|龜梨和也]]
: 主角。遇見謎樣男子之後，深信晴子是他命運之中邂逅的對象。
* 湖月晴子：[[木村文乃|木村文乃]]<ref name="oricon2087172">{{Cite web |date=2017-03-08 |url=http://www.oricon.co.jp/news/2087172/full/ |title=菜々緒、木村文乃と親友役 亀梨主演『ボク、運命の人です。』追加キャスト発表  |work=ORICON NEWS|publisher=[[オリコン|オリコン]] |accessdate=2017-03-19}}</ref>
: 個性務實的粉領族。其任職的公司位於誠任職的公司隔壁。
* 定岡光圀：[[滿島真之介|滿島真之介]]
: 晴子的同學。現為紅友貿易員工，在街上與晴子偶然相遇。
* 四谷三惠：[[菜菜緒|菜菜緒]]<ref name="oricon2087172" />
: 晴子的同事兼好友。
* 葛城和生：澤部佑
: 誠的同事。
* 鳩崎堇（{{lang|ja|鳩崎すみれ}}）：渡邊江里子（搞笑團體「阿佐谷姊妹」）
: 晴子任職的公司社長。
* 宇久森綠（{{lang|ja|宇久森みどり}}）：岡野真也
: 誠的同事。
* 湖月大地：[[杉本哲太|杉本哲太]]
: 晴子的父親。
* 關原卓：大倉孝二
: 誠的同事。
* 湖月善江：[[石野真子|石野真子]]
: 晴子的母親。
* 烏田翔吉：[[田邊誠一|田邊誠一]]
: 誠的上司。
* 謎樣男子／正木一郎：[[山下智久|山下智久]]
: 自稱神的謎樣男子。誠與晴子的兒子。30年後，地球因受隕石撞擊導致毀滅，為了避免如此人間慘事發生，他從未來來到現代，以「神」的身分出現在誠面前，運用自己的「『神』助攻」湊合誠與晴子。
* 「伊甸的園田」店長：村杉蟬之介
: 晴子經常光顧的小酒館店長。
* 富田：赤津大輔
: 相撲火鍋店「富田」的老闆。
