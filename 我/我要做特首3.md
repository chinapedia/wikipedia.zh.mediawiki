《'''我要做特首3'''》（'''{{lang|en|I want to be CE 3}}'''），是[[亞洲電視|亞洲電視]]製作的時事通識節目，是《[[我要做特首2|我要做特首2]]》的續集，節目集[[娛樂|娛樂]]、[[政治|政治]]、[[通識教育|通識教育]]於一身，節目由香港跨媒體文化人[[胡恩威|胡恩威]]主持。

== 概說 ==
現時香港社會問題日益嚴重，政府施政繼續頻頻失誤，民怨不斷升溫。為了讓香港市民能夠繼續對社會或政府提出建議，亞洲電視推出《'''我要做特首3'''》，繼續邀請社會各界別人士，包括政府官員、政商界名人、團體代表，以至基層市民等，跟主持人胡恩威深入探討社會問題。

新一屆特首政府經已誕生，《'''我要做特首3'''》會繼續跟進特首選舉後情況，包括新特首的政綱實施、新政府官員班底的安排，以及新政府的施政路向等，都會透過全新的《'''我要做特首3'''》陸續向觀眾詳細分析和介紹。節目亦會邀請嘉賓討論熱門時事及社會問題。
