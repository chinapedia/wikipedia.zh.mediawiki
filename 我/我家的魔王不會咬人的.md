《'''我家的魔王不會咬人的'''》（{{lang-ja|うちの魔王かみませんよ}}）是[[漫畫家|漫畫家]]おとうさん的[[日本漫畫|日本漫畫]]作品。

== 故事簡介 ==


== 登場角色 ==
以下中文名稱皆為民間通稱，並非官方翻譯。
=== 岸本家 ===
; {{nihongo|魔王／[[撒旦|撒旦]]|魔王／サタン}}
: 岸本家的寵物，魔王族，害怕勇者族，可以理解人類所說的話，喜歡甜食。非常喜歡媽媽。
; {{nihongo|兄／悠平|ユウヘイ}}
: 撒旦的飼主，初中開始打棒球，因為事故傷到了腳而無法前往甲子園。與撒旦從小感情就很好。
; {{nihongo|妹／悠希|ユウキ}}
: 悠平的妹妹，長相可愛卻非常[[腹黑|腹黑]]，自稱掌管了商店街。喜歡羅伊斯。
; {{nihongo|媽媽|はは}}
: 岸本家的媽媽，對撒旦很好，被撒旦尊稱「母親大人」
; {{nihongo|爸爸|ちち}}
: 岸本家的爸爸，是名幻獸學者，平常不在家裡，是個長相嚴肅卻對家庭很關心的好父親。

=== 四大天王 ===
; {{nihongo|比弗羅|ビフロ}}
: 四大天王之一，是名殭屍寵物，被羅伊斯稱為「[[腐女|腐女]]<ref>腐女原意為「喜歡[[BL|BL]]情節的女子」，但此處指「腐爛的女子（殭屍）」</ref>」
; {{nihongo|羅伊斯|ロイス}}
: 比弗羅的飼主，自稱「不小心搞錯性別的少女」。

=== 其他人物 ===
; {{nihongo|亞瑟|アーサー}}
: 勇者族。吉田婆婆家的寵物，聖誕夜也兼職聖誕老人。
; {{nihongo|吉田婆婆|吉田さん}}
: 亞瑟的飼主，是岸本家的鄰居。
; {{nihongo|佐藤將太|佐藤将太}}
: 自稱紅連大人，寵物為死之不死鳥，是一隻塑膠鴨，被悠希稱為職業級的笨蛋。
