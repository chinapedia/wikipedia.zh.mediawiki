== 概要 ==
此片導演石井裕也花費約3年的時間籌備<ref name="eigacom140530">{{Cite web| title = 石井裕也監督作「ぼくたちの家族」を原作者・早見和真はどう受け止めたか?| publisher = 映画.com| date = 2014-05-30| url = http://eiga.com/news/20140530/9/| accessdate = 2014-06-20}}</ref>，同年8月，受邀參與第38屆[[加拿大|加拿大]][[蒙特婁世界電影節|蒙特婁世界電影節]]的World Greats單元<ref>{{Cite web| title = 妻夫木聡が主演、石井裕也監督作『ぼくたちの家族』、モントリオール世界映画祭“ワールド・グレイツ部門”へ出品| publisher = CDJournal| date = 2014-05-07| url = http://www.cdjournal.com/main/news/-/58871| accessdate = 2014-05-08}}</ref><ref name="tvfan140514">{{Cite web| title = 妻夫木聡、「そのままでいいと言ってくれるのが家族」『ぼくたちの家族』がモントリオール世界映画祭へ| publisher = TVfanWeb| date = 2014-05-14| url = http://tvfan.kyodo.co.jp/news/704092| accessdate = 2014-05-15}}</ref>和第19屆[[釜山國際影展|釜山國際影展]]<ref>{{Cite web| author = 内田涼| title = 妻夫木、『ぼくたちの家族』で初共演の池松壮亮を絶賛| publisher = チケットぴあ| date = 2014-06-20| url = http://ticket-news.pia.jp/pia/news.do?newsCd=201406190008| accessdate = 2014-06-20}}</ref>。此片導演石井表示自己亦是年輕時母親即病逝，因而對原作感同身受；加上日本目前描繪此一議題的作者多數較年長，因此想和年輕的原著作者一樣，以自身30歲左右年齡階段的視角對此議題進行觀察，描寫疾病對家庭產生的改變<ref name="金馬影展"/>。

== 劇情 ==
原本是「[[隱蔽青年|家裡蹲]]」的若菜浩介（[[妻夫木聰|妻夫木聰]] 飾）在多年後走出自閉生活，終於娶妻，妻子深雪（{{link-ja|黑川芽以|黒川芽以}} 飾）亦懷孕3個月。浩介向母親玲子（{{link-ja|原田美枝子|原田美枝子}} 飾）報告此一好消息，浩介父親克明（[[長塚京三|長塚京三]] 飾）亦向玲子提議前往[[夏威夷|夏威夷]]旅遊，一家看似過著幸福的生活。但因健忘的症狀日益嚴重，玲子在和兒子浩介和丈夫克明陪同之下前往醫院就診。醫生告知，玲子的病症是[[腦腫瘤|腦腫瘤]]，且已大面積擴散，能正常說話、行動已是極為罕見的例子，原本以為自己只是[[阿茲海默症|阿茲海默症]]的玲子，聞言崩潰。醫生向浩介表示，母親玲子只剩一星期的壽命。對此一措手不及的惡耗，克明和浩介深受打擊，並堅決對玲子隱瞞此一事實。隨著病情逐日惡化，玲子亦開始認不得家人並胡言亂語，對著家人道出自己多年來的心聲。丈夫克明和兩位兒子此刻才知道母親內心真正想法。

為了治病，浩介要求平常漫不經心、經常向母親伸手要錢的弟弟俊平（[[池松壯亮|池松壯亮]] 飾）趕回家中，為了日後母親的醫療費用，兩人開始清查家中的財務狀況。俊平發現母親入院前向多家銀行申貸多筆借貸，竟達300萬。兩兄弟質問父親克明是否知道此事，父親竟將個人欠債3000萬的事和盤托出，驚訝的兄弟建議宣告[[破產|破產]]，但父親表示目前房子貸款1200萬，擔保人是浩介，宣告破產恐拖累浩介。一家看似陷入了絕望。

面對家中爆出的一連串問題，兄弟倆決定振作。玲子住院的醫生建議若菜家將玲子接回家，渡過人生最後的時光，但兄弟倆不放棄最後一絲機會，繼續分頭尋找肯接治母親的醫院。經過連日的奔波，俊平透過一位醫生（[[鶴見辰吾|鶴見辰吾]] 飾）的介紹，終於找到了一家肯為母親進行複檢的醫院，主治醫師（[[板谷由夏|板谷由夏]] 飾）表示雖然可能性極渺小，但若為惡性[[淋巴瘤|淋巴瘤]]，反而有治療和手術的機會。兄弟倆見到了一絲曙光……。

在生死當口，一家人從裂縫中被逼發出了蛻變和成長。

== 製作 ==
=== 拍攝 ===
主要拍攝外景地為[[山梨縣|山梨縣]][[上野原市|上野原市]]<ref name="sagamihara">{{Cite web| title = 映画「ぼくたちの家族」公開です!| publisher = さがみはらフィルムコミッション| date = 2014-05-21| url = http://www.sagamihara-fc.jp/archives/4110| accessdate = 2014-06-20}}</ref>，其他外景地包括[[神奈川縣|神奈川縣]][[相模原市|相模原市]]的[[藤野車站|藤野車站]]等地<ref name="sagamihara" />。故事背景的拍攝地之一「三好町」亦為原著作者{{link-ja|早見和真|早見和真}}的實際居住地，為忠於原作，住家場景未重新搭景，為直接使用該地既有的空餘物件來拍攝<ref>{{Cite web| author = 新居未希| title = 僕たちの世代(2)――家族を考える・早見和真さん| publisher = みんなのミシママガジン| date = 2014-05-22| url = http://www.mishimaga.com/special02/044.html| accessdate = 2014-06-20| deadurl = yes| archiveurl = https://web.archive.org/web/20140629224022/http://mishimaga.com/special02/044.html| archivedate = 2014年6月29日| df = }}</ref>。

=== 選角 ===
導演石井裕也表示，2011年即已選定由[[池松壯亮|池松壯亮]]演出片中二兒子一角，當時的池松有一種「由少年轉為成人」的氣味，和片中的俊平相同，而男主角一開始即想邀請[[妻夫木聰|妻夫木聰]]飾演。2014年，石井的另一部執導作品《[[球場上的朝陽|球場上的朝陽]]》亦是妻夫木聰和池松壯亮擔綱演出，石井表示，《[[球場上的朝陽|球場上的朝陽]]》的妻夫木聰為製片方的邀約，是巧合的結果；但池松演出此兩部片中都是自己邀約的<ref name="金馬影展"/>。
