《'''我的朋友世界第一可愛'''》是由三月創作的日本[[四格漫畫|四格漫畫]]作品。2015年4月号至9月号於《[[月刊Comic_Alive|月刊Comic Alive]]》中的一個四格漫畫欄目《[[COMIC_CUNE|COMIC CUNE]]》客串連載，後來轉移至於2015年8月27日正式創刊的四格漫畫雜誌《COMIC CUNE》2015年10月號發表<ref>{{cite web|url=http://natalie.mu/comic/news/158261|title=4コマ誌キューン創刊、なもりポスター付録＆「パンでPeace！」アニメ化|publisher=[[Natalie.mu|Natalie.mu]]|date=2015年8月27日|accessdate=2016年3月27日|language=ja}}</ref>。ComicWalker網站也發表該四格漫畫。<ref>{{cite web|url=http://natalie.mu/comic/news/164267|title=新4コマ誌コミックキューンの全連載作品が、ComicWalkerで無料配信|publisher=Natalie.mu|date=2015年10月28日|accessdate=2016年3月27日|language=ja}}</ref>第1本[[單行本|單行本]]於2016年3月26日發售。台灣地區由[[台湾角川|台湾角川]]出版。

==故事==
學生會長西園寺小春是學校裡人人憧憬的存在。有一天她遇到瑪麗亞，從此之後便被她纏上再也離不開。

; {{nihongo|西園寺小春|西園寺小春}}
: 學生會會長。在學校裡是人人憧憬的存在。
; {{nihongo|瑪麗亞|まりあ}}
: 樂天美少女。
; {{nihongo|和泉|和泉}}

; {{nihongo|結城|結城}}
: 認真的副會長，同樣為不服輸個性。視和泉為競爭對手。
; {{nihongo|沙織|沙織}}
: 學校老師，傲嬌微天然，是千歲從小到大的玩伴兼同居人。

: 學校老師，天然健氣，很保護沙織，是沙織從小到大的玩伴兼同居人。
