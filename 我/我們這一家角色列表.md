'''我們這一家角色列表'''記載[[日本|日本]]漫畫、改編動畫《[[我們這一家|我們這一家]]》的所有登場角色相關資料。

== 花家 ==
* 其姓氏「立花」在臺灣及香港的配音版本皆為「[[花姓|花]]」，下述所有內容以此為主。
* 家中的印章有三種，前期是「{{lang|ja|たちばな}}」，中期是「{{lang|ja|タチバナ}}」，後期是「立花」。
* 日語中「{{lang|ja|たちばな}}」（立花）亦指「橘」，故有些家庭成員以[[柑橘類|柑橘類]]命名。

=== 立花翠（花師奶） ===
Tachibana Midori
* 配音員：[[渡邊久美子|渡邊久美子]]（日本）；[[王瑞芹|王瑞芹]]（臺灣）；[[雷碧娜|雷碧娜]]（香港）
: 本名立花翠；日本原音通称{{tsl|ja|お母さん|お母さん}}，-{zh-hans:香港译為'''花师奶'''（花太太），台湾译为'''花妈'''; zh-hant:香港譯為'''花師奶'''（花太太）; zh-hk:臺灣譯為'''花媽''';}-，臺灣日販漫畫版無定名。
: 本劇中心人物，43歳。首次登場於漫畫第1回／動畫第1集單元A。角色形象參考自原作者[[螻榮子|螻榮子]]的母親。
: 身高：160公分，體重：75公斤左右（以體脂肪率乘3計算），體脂肪率：51.0%（超重），3月23日，血型O型。
: [[家庭主婦|家庭主婦]]，花老爺的妻子、橘子和柚子的母親。出身自[[九州|九州]][[大分縣|大分縣]]。
: 其體型和[[五官|五官]]特徵[[遺傳|遺傳]]自母親，似半[[人魚|人魚]]，且有著粗長的[[嘴唇|嘴唇]]。
: 有著不認輸的精神，對自己的外貌和所作所為往往頗具信心，凡有堅持的目標就要做到最後，而因此般堅持換來弄巧成拙的結果亦不在少數。矛盾的是本身也抱持「最好的人生有如平坦緩坡」與「凡事不該執著於某樣事物」的[[價值觀|價值觀]]。
: 習慣[[誇飾|誇大]]自己的[[情緒|情緒]]、[[言語|講話內容]]和[[肢體語言|動作]]；包括曾為教訓橘子，過分放大其舉止而造成家中驚動<ref name="48B">動畫版第48集單元B</ref>；另在生理亦有一些異於常人的部分，如很會流汗<ref name="248A">動畫版第248集單元A</ref>，對於無關自己痛癢的雜音（如[[機車|機車]][[引擎|引擎]]聲或抖腳聲）也格外敏銳。
: 秉持勤儉的生活態度，對物品錙銖必較，認為名牌是虛有其表，甚至會將貴重物品放很久或過期前夕才用；但偶為私心改變，例如有慣用卻較昂貴的[[衛生紙|衛生紙]]品牌<ref name="100B">動畫版第100集單元B</ref>。
: 隱約有[[重男輕女|重男輕女]]的傳統思維，尤其會反映在花爸和橘子身上，並偶有無意間偏袒柚子的情形（曾對橘子解釋是因為柚子的敏感性格常無法說教）；另外相當講求物品和造型的便利性，對時尚不但一竅不通，並排斥年輕族群所流行的許多穿著方式。
: 有許多力求方便的特殊行徑，如：以[[量杯|量杯]]代替易聚積[[茶垢|茶垢]]的[[茶壺|茶壺]]、用家中門旁[[瓦斯錶|瓦斯錶]]的剩餘空間放[[垃圾|垃圾]]並稱之為「垃圾表」等。
: 容易誤會或無法理解別人所表達的意思，並常誤用詞語與跟本人訴求相左的行為舉止或言論；若被家人發現並糾正，通常會口頭合理化、辯解或裝傻離開（哼歌）。
: 在平常生活中也很糊塗，經常做家事做到一半或忘記自己放東西的位置，而且會想到什麼就做什麼。
: 愛好的食物是[[布丁|布丁]]、[[煎餅|煎餅]]、[[香蕉|香蕉]]和[[五家寶|五家寶]]；對食物接受度高，幾無排斥自己的失敗料理。
: 平時喜歡哼歌，以原音角色曲〈热情的红[[玫瑰|玫瑰]]〉作為招牌曲（臺灣和香港以其它設定代之）。
: 興趣是撰寫[[書法|書法]]，其筆名為'''今治翠'''<ref name="97A">動畫版第97集單元A</ref>，並期望可以開個展。
: 害怕「真的」有[[雷公|雷公]]拿走[[肚臍|肚臍]]，故討厭雨天<ref name>首部劇場版</ref>，亦少於雨天出門；如有需要的東西，總會要求家人代買或乾脆不買並盡所能應對。另外也很怕[[狗|狗]]。
: 兒時綽號為'''大大的'''<ref name="101A">動畫版第101集單元A</ref>和'''阿肥'''<ref name>首部劇場版</ref>。
: 烹煮晚餐的菜色常受花爸的發薪日、花媽的心情或狀態和氣候影響，最常出現的料理是[[竹輪|竹輪]]和[[味噌湯|味噌湯]]。同時，很易煮出不合常理或差勁的餐點，因素包括從料理節目學習、缺用食材刻意應急等；要是引起家人抱怨，花爸對此多會叫外送。
: 偶像是'''橋本英二郎'''（藝名'''豬浦'''）<ref name="189A">動畫版第189集單元A</ref>，被她稱作'''豬老大'''。
: 雖然對橘子未來就讀的[[大學|大學]]以[[東京大學|東京大學]]為最高標準，但認為橘子可能無法考上，則將標準放低至[[早稻田大學|早稻田大學]]，同時也向橘子堅稱欲就讀沒有聽過的學校，即不支助學費。
: 由於長期做為家庭主婦的關係，對外偶有怕生的情形。
: 與花爸因為相親認識，曾一度想要拒絕這門親事<ref name="90B">動畫版第90集單元B</ref>。另特別在意花爸的感受，並多會包容他的特殊行為。
:口頭禪：「安捏喔」（[[閩南語|閩南語]]的「這樣喔」）

=== 立花蜜柑（花帶柑） ===
Tachibana Mikann
* 配音員：[[折笠富美子|折笠富美子]]（日本）；[[林美秀_(配音員)|林美秀]]（臺灣）；[[林元春|林元春]]（香港）
:-{zh-hans:港版为花带柑，台版为花橘子; zh-hant:港版為花帶柑; zh-hk:臺版為花橘子;}-，臺灣日販漫畫版為橘子。
: 首次登場於漫畫第1回／動畫第1集上半。
: 身高：155公分，體重：60公斤，體脂肪率：25.1%（稍微過重）生日為8月5日，血型O型。
: 花爸與花媽的女兒、柚子的姐姐、家中的長女，高中二年級學生（原作初期是高中一年級）。
: 外貌特徵是招牌[[白眼|白眼]]和綁[[髻|丸子頭]]的[[自然捲|自然捲]]長髮（讀[[中學|中學]]時是[[雙馬尾|雙馬尾]]）。另外在小時候，花媽打算幫她剪成短髮時，因為花爸說「女孩子留長頭髮比較可愛」，所以沒有剪掉。
: 個性幼稚不成熟，性格單純活潑、大而化之和小迷糊，無法有過於複雜的思想，時常不假思索地表達或從身體動作中透漏自己的想法，並且形容事物時習慣[[誇張|誇張]]化；性格與花媽有不少相似之處，甚至部分思維是幾乎相同，但經常在做出與花媽類似的事情後，反而感到懷疑和懊悔。
: 行為慣有模式是毫無章法和怠惰散漫，而在家時（尤其是寒、暑假）會邋遢到不修邊幅的程度。經常遲到或為學校的大考熬夜唸書，即使事先訂立計劃，也會因為自己不積極的態度而失敗收場。另有時會因為自己的隨便，而被弟弟柚子抱怨。偏胖的棉花糖身形，有粗手臂、小腹、蘿蔔腿，常常想著要減肥卻因為自己怠惰不積極的個性沒有一次成功過。然而花媽卻稱其身材有福相、健康，不需要減肥。
: 因為個子矮小的關係，曾多次被人誤認為是中學生，甚至是小學生<ref name="11A">動畫版第11集單元A</ref>。
: 時常為一些小事與花媽意見有異而鬥嘴，但也常會跟花媽訴說心事或討論問題。
: 喜歡[[泰迪熊|泰迪熊]]；在學校參加的社團是「泰迪熊研究社」，簡稱「熊研」。
: 患有輕微的[[近視|近視]]，出門時會戴[[隱形眼鏡|隱形眼鏡]]，甚至常在遲到沒時間準備的情形下戴普通眼鏡（漫畫版動畫版有描寫準備和戴隱形眼鏡的樣子，但後期少有描寫；）。個性冒冒失失，只要出遊一定會受很多傷。
: 暗戀同班同學岩木，而在橘子眼中，岩木彷彿是會發光的[[天使|天使]]。
: 曾做過不少工讀：其中一次是為了要買新手機就到車站前的麵包店打工<ref name="304A">動畫版第304集單元A</ref>；另一次是為了籌措熊研社出遊的旅費就於超級市場擔任新商品的試吃推銷。
: 電視動畫的臺灣配音版同時擔任下集預告旁白，日本方面另由[[萩野志保子|萩野志保子]]擔任。
: 口頭禪：「為什麼」。

=== 立花柚彥（花帶桔） ===
Tachibana Yuzuhiko
* 配音員：[[阪口大助|阪口大助]]／[[伊瀨茉莉也|伊瀨茉莉也]]（幼少期）／[[豬口有佳|豬口有佳]]<small>（0歲時）</small>（日本）；[[雷碧文|雷碧文]]／[[林佑俽|林佑俽]]<small>（劇場版[[台語|台語]]配音）</small>（臺灣）；[[周文瑛|周文瑛]]→[[陸惠玲|陸惠玲]]（香港）
: -{zh-hans:港版為花带桔，台版為花柚子; zh-hant:港版為花帶桔; zh-hk:台版為花柚子;}-；臺灣日販漫畫版為太郎。
: 首次登場於漫畫第1回／動畫第1集上半。
: 身高：151公分，體重：50公斤（以體脂肪率乘3計算），體脂肪率：16.8%（適中）生日為7月6日，血型B型。
: 花爸與花媽的兒子、橘子的弟弟、家中的老么，中學二年級學生（原作初期是中學一年級）。

: 有著成熟思想與深奧的內心世界，表面上是話不多（「內心話」卻特別多）和極端害羞的人，喜歡在私底下耍酷，甚至常為雞毛蒜皮的小事感到尷尬並逞強。

: 與其他家人不同於能吃很燙的食物，讓家人羨慕，但反而無法拿很燙的東西。吃東西喜歡配[[美乃滋|美乃滋]]，通常是[[鮪魚|鮪魚]][[罐頭|罐頭]]。會在半夜偷吃東西。
: 其實也有近視（只是動畫版從來沒有描寫他戴眼鏡的樣子）。
: 喜歡聽[[音樂|音樂]]，有時候會戴[[耳機|耳機]]。
: 偶像為'''丸野丸美'''，從無他人知曉，本身也非常保密，深怕被別人知道<ref name="44A">動畫版第44集單元A</ref>。
: 學校所屬社團為[[棒球|棒球]]社，擔任二壘手，但該社存在感非常低，很少被注意到。 
: 很會畫漫畫和翻書動畫，有回以川島的形象畫成漫畫，卻把川島弄哭。
: 口頭禪：「媽，晚飯好了沒？」

: 具有善於按摩的神手技，但對老婆卻毫無作用，甚至會讓她覺得很癢而大笑<ref name="163A">動畫版第163集單元A</ref>。

: 口頭禪：「哈哈哈」。

==昭和花家之澡堂物語篇登場角色==
*'''畫家新沼'''（{{lang|ja|新沼画家}}）
: {{配音2|JP|[[日野聰|日野聰]]}}
: 知名大畫家。一開始登場時是住在澡堂附近的某藝大學生。在思考畢業後未來的出路與靈感時來到澡堂泡澡，然後遇到橘子與柚子。
: 來到澡堂時正時逢花家因為請不到人畫澡堂壁畫而煩惱的時候，聽聞此事後就自願幫花家繪畫牆壁<ref name="294A">動畫版第294集單元A</ref>。經過幾天的構思，在無意中見到柚子的塗鴉而獲得靈感，替花家的澡堂畫上了塗鴉風格的壁畫。多年後自己如願成名且獲邀在新聞專題上接受訪談，表示當初為了感謝其恩情有贈了一幅畫給花家。

*'''坂本三郎'''（{{lang|ja|坂本三郎}}）
: {{配音2|JP|[[岩田光央|岩田光央]]}}
: 通緝犯。因為銀行搶案而四處躲藏，來到澡堂泡澡時被眼尖的柚子發現是通緝單上的人。
: 被橘子和柚子給試圖跟蹤，發現之後便請求兩人幫忙把禮物送給附近公寓的某位小孩。然後橘子和柚子以及其小孩認識後一起在空地玩捉迷藏，柚子意外地被廢棄的保險箱給困住時及時出現，於是就靠著過去的開鎖經驗幫柚子脫困<ref name="277A">動畫版第277集單元A</ref>。
: 該小孩的名字叫阿雄({{lang|ja|ヨシオ}})，是三郎的兒子。但因為不想連累到而沒有與他當面相認。但是在保險箱打開時兒子也在場，於是叫橘子與柚子把他帶走。接著一看到渡邊刑警出現在場就想自首，可是沒有當場逮捕，可能是知道他不想連累到孩子。

*'''渡邊刑警'''（{{lang|ja|渡辺刑事}}）
: {{配音2|JP|[[辻親八|辻親八]]}}
: 長期在追蹤通緝犯三郎動向的刑警，首次登場於時得知三郎將會接近兒子阿雄所在的公寓而在那裏跟監。
: 最後在柚子困在廢棄的保險箱，三郎正剛好在解圍時出現，沒有當場逮捕要自首的三郎。可能是知道他的兒子也在場的關係，不想連累到孩子。

*'''龜吉'''（{{lang|ja|カメ吉}}）
: 常常光顧花家澡堂的常客之一。
: 在橘子與柚子為了要在抽獎活動中抽到彩色電視機的時候曾經以手上的抽獎券來幫助兩人。

*'''田中千代'''（{{lang|ja|田中千代}}）
: 經營田中雜貨店的老婆婆。個性古怪且處事非常嚴格，附近認識她或曾光顧過的小孩都稱呼她"千代婆婆({{lang|ja|チョばあ}})"，同時也傳出不少相關的傳聞{{#tag:ref|例如在店內有一顆用來抽獎用的大糖果後面的繩子在過去就被剪掉，以致三年來都沒人抽到、可戳洞的抽獎用盒子裡面的獎品事先知道位置然後被取走等。|group="註"}}。
: 曾經有一位兒子，但是在二戰中躲避大轟炸的時候犧牲，因此非常愧疚<ref name="273A">動畫版第273集單元A</ref>。

*'''戶田主播'''（{{lang|ja|戸田アナ}}）
: 在畫家新沼的畫作獲獎的時候在電視節目裡做採訪的主播。

*'''門太'''（{{lang|ja|モン太}}）
: {{配音2|JP|[[松本梨香|松本梨香]]}}
: 住在澡堂附近的小孩。認識柚子以及藤野，有孩子王的氣概。

*'''小宮'''（{{lang|ja|みやちゃん}}）
: 橘子學校的同學之一。曾向家中開澡堂的橘子告知自家已經要安裝浴缸的事情。
