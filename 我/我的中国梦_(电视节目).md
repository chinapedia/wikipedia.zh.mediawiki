《'''我的中国梦'''》，2013年5月6日在[[湖南卫视|湖南卫视]]开播的[[纪录片|纪录片]]，该节目系湖南卫视响应[[中共中央总书记|中共中央总书记]][[习近平|习近平]]执政思想“[[中国梦|中国梦]]”而制作，让全中国吹响“中国梦”号角<ref>{{cite web |url=http://zixun.hunantv.com/hntv/20130530/1631347245.html |title=《我的中国梦》获赞誉 梦想践行我们在路上 |accessdate=2013年6月21日 |publisher= 金鹰网|work= |language=zh-cn}}</ref>，传递荧屏[[正能量|正能量]]，<ref>{{cite web |url=http://zixun.hunantv.com/hntv/20130428/0944196422.html |title=湖南卫视《我的中国梦》4月29日播出 传递荧屏正能量 |accessdate=2013年6月21日 |publisher=金鹰网 |work= |language=zh-cn}}</ref>官方宣传口号为“记录草根的平凡追梦故事，见证筚路蓝缕的中国梦”。<ref name=" fish" >{{cite web |url=http://zixun.hunantv.com/hntv/20130428/0944196422.html |title= 湖南卫视《我的中国梦》4月29日播出 传递荧屏正能量|accessdate=2013年6月21日 |publisher=金鹰网 |work= |language=zh-cn}}</ref>

== 概述 ==

*[[演员|演员]]：[[李亚鹏|李亚鹏]]、[[姜文|姜文]]、[[陈道明|陈道明]]<ref name=" fish" />
