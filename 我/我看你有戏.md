《'''我看你有戏'''》是[[浙江卫视|浙江卫视]]于2015年2月6日起每周五21：10播出的演艺[[真人秀|真人秀]]节目。由[[张国立|张国立]]、[[冯小刚|冯小刚]]、[[成龙|成龙]]、[[李冰冰|李冰冰]]担任评委及导师。<ref name = presentation></ref><ref name="vocal_concert">[http://news.jwb.com.cn/art/2015/1/1/art_247_5230188.html 李冰冰综艺首秀《我看你有戏》] {{webarchive|url=https://web.archive.org/web/20150101152946/http://news.jwb.com.cn/art/2015/1/1/art_247_5230188.html |date=2015-01-01 }}，今晚网，2015年1月1日</ref>此外，陈道明将担任该节目的节目顾问，将设置专业评审团。<ref name = tecent_news_2>[http://ent.qq.com/a/20141111/011071.htm 陈道明任《我看你有戏》节目顾问]</ref>白百何将率专业评审团。节目还将拍摄一部同名大电影。<ref name = tecent_news>[http://ent.qq.com/a/20141111/038585.htm 李冰冰加盟《我看你有戏》]</ref>
