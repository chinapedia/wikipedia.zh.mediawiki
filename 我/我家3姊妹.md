《'''我家3姊妹'''》是[[松本百力滋|松本百力滋]]在[[網誌|網誌]]中的小品文。其後在[[竹書房|竹書房]]的雜誌《{{lang|ja|[[すくすくパラダイス|すくすくパラダイス]]}}》（最初是《{{lang|ja|[[まんがくらぶ|まんがくらぶ]]}}》）以完全漫畫形式連載同名的短劇喜劇，之後[[日本|日本]]和[[韓國|韓國]]再合作，以原作為基礎的製作出電視動畫。

{{empty section}}

==角色==
===三姊妹一家===
; {{nihongo|小文|フー||配音員：[[大谷育江|大谷育江]]（日）／[[何璐怡|何璐怡]]（港）／[[陳曉月|陳曉月]]（台）}}
: 長女。2000年2月出生，動畫開始時5歲，血型為O型。基本服裝是粉紅色上衣和深紅色短裙。性格純真、無厘頭、好勝。喜歡幻想，討厭吃青椒，也害怕打雷。小時候性格火爆，因此有「惡魔」之稱。跟小千感情較好。
; {{nihongo|小鈴|スー||配音員：[[金井美香|金井美香]]（日）／[[楊善諭|楊善諭]]→[[陳皓宜|陳皓宜]]（港）／（台）}}
: 次女。2002年6月出生，動畫開始時3歲，血型為O型。基本服裝是藍色上衣和淺啡色褲子。其我行我素的性格被稱為「自由人」，口頭禪：「是旦啦，無所謂。」（いいでそべつに）小時候較乖巧，與小文差異極大，被稱為「菩薩」。
; {{nihongo|小千|チー||配音員：[[川田妙子|川田妙子]]（日）／[[林元春|林元春]]（港）／（台）}}
: 三女。2004年7月出生，動畫開始時1歲，血型為O型。基本服裝是紅色襯衫和黃色吊帶褲。性格自私大膽被稱為「社長」。十分調皮。對食物十分執著，食量也很大，把爸爸叫為「ぼん」（大餐或盛宴之意），也因此被稱為「-{zh-hans:饭饭小鬼;zh-tw:飯飯小鬼; zh-hk:為食妖怪}-」。
; {{nihongo|媽媽|お母さん||配音員：[[藤村知可|藤村知可]]（日）／[[朱妙蘭|朱妙蘭]]（港）／（台）}}
: 爸爸的高中同學。角色原型為作者[[松本百力滋|松本百力滋]]。
; {{nihongo|爸爸|お父さん||配音員：[[辻谷耕史|辻谷耕史]]，小學生時代：[[比嘉久美子|比嘉久美子]]（日）／[[張錦江|張錦江]]（港）／（台）}}
: 媽媽的高中同學。
; {{nihongo|小毛寶|モプシー||配音員：[[芝原千彌子|芝原千彌子]]（日）／[[黃玉娟|黃玉娟]]（港）／（台）}}
: 松本家的寵物兔。三姊妹出生前就開始飼養，於2008年11月逝世<ref>{{cite web
  | authorlink = [[松本ぷりっつ|松本ぷりっつ]]
  | title = 我家3姊妹網誌文章「モプちゃん」
  | url = http://ameblo.jp/3shimai/entry-10171515722.html }} </ref>。
; {{nihongo|爺爺・奶奶|おーちゃん・あーちゃん||配音員：[[福松進紗|福松進紗]]（日）／[[黃子敬|黃子敬]]（港）／（台）、配音員：[[西墻由香|西墻由香]]（日）／[[袁淑珍|袁淑珍]]（港）／（台）}}
; {{nihongo|外公・外婆|じいじ・ばあば||配音員：[[栗田圭|栗田圭]]（日）／[[譚炳文|譚炳文]]→[[陳曙光|陳曙光]]（第51集起）（港）／（台）、配音員：[[須藤繪里花|須藤繪里花]]（日）／[[沈小蘭|沈小蘭]]（港）／（台）}}

===小健一家===
; {{nihongo|小健|けんちゃん|配音員：[[世戶沙織|世戶沙織]]（日）／[[謝潔貞|謝潔貞]]（港）／（台）}}
: 小大的哥哥，三姊妹的朋友。
; {{nihongo|小大|だいちゃん|配音員：[[佐藤智惠|佐藤智惠]]（日）／[[袁淑珍|袁淑珍]]（港）／（台）}}
: 小健的弟弟，三姊妹的朋友，個性與小鈴相似。
