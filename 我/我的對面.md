《'''我的對面'''》（[[日語|日語]]：{{lang|ja|'''あたしの向こう'''}}），是[[日本|日本]]女性[[歌手|歌手]][[aiko|aiko]]的第32張[[單曲|單曲]]。2014年11月12日由[[PONY_CANYON|PONY CANYON]]發行。

== 背景 ==
距離前一張單曲「[[你的身旁|你的身旁]]」約9個半月，是繼專輯「[[泡沫般的愛情|泡沫般的愛情]]」發行5個半月後發表的新單曲。

== 製作 ==
於全國巡迴演唱會「LOVE LIKE POP vol.17」（2014年）的途中，接受了連續劇《[[了不起的選TAXI|了不起的選TAXI]]》的主題歌Tie-Up之請託而開始製作<ref name="orista141117p42">雑誌『オリ☆スタ』2014年11月17日号 p42</ref>。該連續劇的製作人以「就在快要停下腳步時能在背後推人一把的歌曲」為主題做出了請求<ref name="rankingbox7207">[http://rankingbox.jp/article/7207 【インタビュー】aiko あたしの新たな“選択” 新曲はトキメキがいっぱい]（rankingbox・2014年11月12日）</ref>。

本作對於aiko是相當的少見，在閱讀完連續劇的大綱之後便開始進行製作，一邊想著「如果能回到過去的話該怎麼辦才好」一邊寫下了歌詞<ref name="orista141117p42" />。然後，再將以前寫下的東西補上去便完成了歌詞<ref name="rankingbox7207" />。還有，也將「如果能忘記喜歡的人向前看的話就好了啊」這樣的想法一同放了進去<ref name="orista141117p42" />。

== 音樂錄影帶 ==
由[[丸山健志|丸山健志]]擔任音樂錄影帶導演<ref>[http://www.spaceshowertv.com/search/detail.cgi?mu=0104858&ch=0 ミュージックビデオサーチ aiko「あたしの向こう」]（SPACE SHOWER NETWORKS）</ref>。

於2014年11月6日在[[YouTube|YouTube]]公開了短版<ref>[http://www.barks.jp/news/?id=1000109454 aiko、ドラマ仕立てな「あたしの向こう」MV（ショートVer）を公開]（BARKS・2014年11月7日）</ref>，同月25日則在YouTube公開了完整版<ref>[https://youtube.com/watch?v=mZPbXTQWb0k あたしの向こう - aiko]（YouTube・2014年11月25日）</ref>。
