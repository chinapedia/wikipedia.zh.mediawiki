'''滦州兵变'''起於'''滦州兵谏'''，初爲[[辛亥革命|辛亥革命]]时期[[直隶|直隶]][[滦州|滦州]]发生的一场争取[[君主立宪|君主立宪]]政纲的[[兵變|兵變]]事件。其后轉爲支持同盟會的'''滦州起义'''。

== 滦州兵谏 ==
1911年10月10日，[[同盟會|同盟會]][[武昌起义|武昌发生革命武装起义]]。

10月29日，[[新軍|新军]][[陆军第二十镇|第二十镇]]统制[[张绍曾|张绍曾]]收到天津兵给司令部副官、同盟会秘密会员[[彭家珍|彭家珍]]急电，要张在滦州扣留由彭负责押运的军火。张见电即刻行事，共截获枪5000支，子弹500万发。之后，张联名第三十九协协统[[伍祥祯|伍祥祯]]、四十协协统[[潘榘楹|潘榘楹]]、[[陆军第二混成协|第二混成协]]协统[[蓝天蔚|蓝天蔚]]、[[陆军第三镇|第三镇]]第五协协统[[卢永祥|卢永祥]]等，在直隶[[滦县|滦州]]通电，向清政府提出类似[[最后通牒|最后通牒]]的“十二条政纲”<ref>李剑农《戊戌以后三十年中国政治史》第111页</ref>：
# 大清皇帝万世一系。
# 于本年内召集国会。
# 宪法由国会起草，以皇帝之名义宣布之，但皇帝不得加以修正或否认。
# 缔结条约及讲和，由国会取决，以皇帝之名义行之。
# 皇帝统率海陆军，但对国内用兵时，必经国会议决。
# 不得以命令施行“[[就地正法|就地正法]]，格杀勿论”之事。
# 特赦国事犯。
# 组织[[责任内阁|责任内阁]]，[[总理大臣|总理大臣]]由国会选举后，以皇帝敕任之，其他国务大臣由总理大臣推荐任之。[[皇族|皇族]]不得为国务大臣。
# 国会有[[修宪|修宪]]之提议权。
# 本年度预算未经国务议决，不得适用前年度之预算支出。
# 增重人民之负担，须由国会议决。
# 宪法及国会法之制定，军人有参与权。

同盟會滦州兵谏的同日，山西宣布独立，东西呼应，给清廷以极大威胁。清廷急忙将滦州以西的火车车皮（即車廂，多指貨車）全部集中于京，以防滦军乘车西进。

11月1日，清廷授[[袁世凯|袁世凯]]为[[内阁总理大臣|内阁总理大臣]]。11月3日清廷参考十二条政纲，制定并颁布《[[宪法重大信条十九条|宪法重大信条十九条]]》。

11月6日，[[溥仪|宣统帝]]下[[罪己诏|罪己诏]]，释放革命党刺客[[汪精卫|汪精卫]]、[[黄复生|黄复生]]。

11月7日，任命张绍曾为“长江宣抚大臣”，调离滦州，削去兵权。张绍曾交出兵权，避居[[天津英租界|天津英租界]]。同时朝廷提拔潘榘楹为第二十镇统制。将七十七标移至[[锦州|锦州]]，七十八标调往[[关外|关外]]，第七十九标团部及三营驻扎在滦州车沾，一、二营驻扎在滦州北关“直隶第三师范学校”院内，第八十标开到临榆（抚宁）。这样把二十镇分散调开，化整为零。将第三镇由东北调至廊坊，切断了第二十镇与[[陆军第六镇|第六镇]]会合的可能；调第六镇统制[[吴禄贞|吴禄贞]]署理山西巡抚，削去兵权。至此滦州兵谏平息。

== 滦州起义 ==
[[File:Memorial_park_to_the_martyrs_in_1911_Luanzhou_Revolution_(20170108153323).jpg|thumb]][[海淀区|海淀区]][[温泉地区|温泉地区]]显龙山的辛亥滦州革命先烈纪念园]]
[[陆军第二十镇|第二十镇]]七十九标一营管带[[王金铭|王金铭]]、二营管带[[施从云|施从云]]曾建议[[张绍曾|张绍曾]]，趁此时机与[[吴禄贞|吴禄贞]]、[[蓝天蔚|蓝天蔚]]联合起义，但张犹豫不决。后吴禄贞被袁世凯派刺客暗杀，蓝天蔚逃亡到[[上海|上海]]，此事未果。张绍曾被王金铭派人送到[[天津租界|天津租界]]避居。

此后，王金铭、施从云和第二十镇第八十标三营管带[[冯玉祥|冯玉祥]]等人和天津同盟會的革命党组织——北方[[共和会|共和会]]（会长为[[白雅雨|白雅雨]]）取得联系，秘密筹饷并策动起义。1912年1月，王、施等接受同盟会指示发动起义，1月3日宣布滦州独立，成立“北方革命军政府”，推王金铭为大都督、施从云为总司令、冯玉祥为总参谋长，通电全国。

清朝[[通永鎮|通永鎮]][[總兵|總兵]][[王怀庆|王怀庆]]率部于1月10日镇压了滦州起义。王、施等率部西进，在雷庄与清军遭遇，王、施被诱至清军营地谈判，被伏兵逮捕后遭枪决，白雅雨等兵败突围，也遭到枪杀。冯玉祥逃脱。
