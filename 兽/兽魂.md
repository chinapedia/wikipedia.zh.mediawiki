《'''兽魂'''》（モンスターソウル，MONSTER SOUL）是[[真島ヒロ|真岛浩]]创作的[[奇幻|奇幻]]、[[少年漫画|少年漫画]]。从2006年1月号到3月号集中连载于'''[[Comic_BomBom|Comic BomBom]]'''。2007年随着同作家同杂志的连载作品《'''[[普鲁的奇妙日记|普鲁的奇妙日记]]'''》的完结，该作品从2007年6月号到9月号的续篇再次被执笔。全7话被发布，单行本全2卷已发售。

== 概要 ==
自《RAVE》完结的2005年以后，连载作品仅限于《普鲁的奇妙日记》的真岛浩在[[周刊少年magazine|周刊少年magazine]]的下一部连载作品《[[FAIRY_TAIL|FAIRY TAIL]]》开始之前所描绘的作品。这个时刻，决定着被认为是全3次的集中连载的事。原本来自编辑部的提案是“新作2话的集中连载”，但由于世界观相对膨胀，所以参照了全3话的<ref>旧版1卷后记</ref>。

因为正好单行本一册创作了，从连载结束约1年后被单行本化，在那里象到那个为止的「[[圣石小子|哈鲁]]」一样的没登场人物和连载方案等被设计了。另外，在单行本上发表了“因为世界观大体上已经完成，说不定会做连载版的MONSTER SOUL”的计划，在《周刊少年Magazine》上也决定连载《FAIRY TAIL》，“包括周刊、月刊在内，连载3部作品都很困难”因为被明言，日程的情况，3回完成的状态持续着。

之后，2007年5月《普鲁奇妙日记》完结约1年半后，伴随着续篇执笔，同年的'''《Comic BomBom》'''6月号再次连载。与此同时，单行本表记也从「全1卷」变更为「1～以下续刊」，2卷发行后变更为「(1)(2)」。

单行本当初是BomBom等级被出版,不过，后边用ライバル漫画等级作为新装版被再版(正编没有变更,不过，后记的内容被改写)。

真岛自己认为“（我）作为原作者可以让其他人描绘（作画），但自己亲自描绘（因为月刊周刊同时连载的严峻性）已经不可能了”，因此单行本第2卷的书腰也被标记是“这里完结”。为《兽魂》担当原作，但因为当时作者招牌作品不足，实际上Comic BomBom的工作人员也相当具体地讨论推进着,不过，那个Comic BomBom本身的休刊决定了，计划成为了白纸<ref>新装版2卷引用后记</ref>。

再者，真岛浩从原作借鉴设想在之后的《[[FAIRY_TAIL|FAIRY TAIL]]》的提供角色原型系列的成为了现实基础。

== 主要内容 ==
; MONSTER SOUL
: 人类和魔物（怪物）共存的世界，艾尔文。以前，人类界和魔物界曾发生过二分之一的大战争。传说中的游击部队在那场战争中被称为魔物军最强，其名字也是暗黑之翼（ブラックエアーズ）。
: 比人类更强大，比人类更温柔，暗黑之翼的成员们向世界挥舞着炽热的拳头。
; MONSTER SOUL 2nd
: 曾经将世界分为两部分，魔物军与人类军的战争。在那场战争中被称为魔物军最强的传说中的游击部队，暗黑之翼。在他们面前被地狱种族（ヘルタールーツ）袭击的国家的少年，向他们求助。
: 失去的王国，为了救出的ドリオクラム人们，开始了暗黑之翼的地狱旅行。
