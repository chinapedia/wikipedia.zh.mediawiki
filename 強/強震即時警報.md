'''強震即時警報'''（{{lang-en|Earthquake Early Warning}}，縮寫：{{lang|en|EEW}}），又稱'''地震預警'''、'''地震速報'''，是[[臺灣|臺灣]]在[[地震|地震]]發生後發布的警報，針對部分地區利用[[地震波|地震波]]傳遞速度的差異，在較具威脅性但速度較慢的[[S波|S波]]、[[表面波|表面波]]到達前，透過[[電腦軟體|電腦軟體]]、[[行動應用程式|行動應用程式]]、[[災防告警細胞廣播訊息系統|災防告警細胞廣播訊息系統]]等方式預先通報，以爭取數秒到數十秒的預警時間。{{r|cwb_com|page=1}}<ref>{{Cite web |url = http://scweb.cwb.gov.tw/eew_DEMO/ |title = 計畫源起 |work = 地震速報系統-強震即時警報宣導網 |publisher = 中央氣象局地震測報中心 |language = zh-tw |accessdate = 2015-09-20 |archiveurl = https://web.archive.org/web/20151001040742/http://scweb.cwb.gov.tw/eew_DEMO/ |archivedate = 2015-10-01 |dead-url = yes}}</ref><ref name="ncdr_eew">{{Cite web |url = http://eew.ncdr.nat.gov.tw/Development_background02.html |title = 基本原理介紹 |work=強震即時警報應用交流平台 |publisher = 國家災害防救科技中心 |language = zh-tw |accessdate = 2015-09-21 |archiveurl = https://web.archive.org/web/20150925063751/http://eew.ncdr.nat.gov.tw/Development_background02.html |archivedate = 2015-09-25 |dead-url = no}}</ref>2017年[[交通部中央氣象局|交通部中央氣象局]]的強震即時警報系統總共發布56次警報，發生於臺灣島內或近海的地震可在地震發生後20秒內對外發布警報，對距離[[震央|震央]]70公里以外的地區提供預警。{{r|cwb_year_report106|page=46}}

==原理==
地震發生時，地震的[[能量|能量]]藉由地震波傳遞，地震波根據[[振動|振動]]方式分為[[P波|P波]]、[[S波|S波]]與[[表面波|表面波]]，P波傳遞速度約為每秒6到7公里，S波傳遞速度約為每秒3到4公里，表面波傳遞速度則小於S波。S波的[[振幅|振幅]]通常較P波大，且S波的水平向振動對房屋結構較具破壞性（表面波因包含較多水平向長週期能量也會影響結構物），因此S波的破壞性較P波強。透過傳遞速度較快而先抵達的P波預估破壞性較強的S波的振動大小，並在S波抵達前發出警報，爭取數秒到數十秒的預警時間，愈靠近震央的地區預警時間愈短。<ref name="ncdr_eew" />{{r|eewjournal|page=6}}

==作業流程==
中央氣象局強震即時警報系統的[[作業流程|作業流程]]分為觸發偵測、地震定位與發布決定三步驟。首先中央氣象局將[[地震儀|地震儀]]資料傳到中心站的[[電腦|電腦]]，利用地動振幅變化量與背景雜訊的[[比值|比值]]進行地震的觸發偵測，超過門檻時即為觸發狀態，當觸發測站超過5個時，地震觸發成立，紀錄觸發測站的P波到時與位移振幅。接著利用P波到時計算地震發生時間、震央位置與震源深度，地震規模則利用P波振幅計算{{efn|經驗關係式為<math>M_{Pd}=5.000+1.102\times\log_{10}(P_d)+1.737\times\log_{10}(r)</math>，<math>M_{Pd}</math>是警報使用的地震規模，<math>P_{d}</math>是P波垂直向位移的最大振幅（公分），<math>r</math>是震源距離（公里）}}，隨著測站資料傳回電腦持續計算與更新，當達到發布條件時即對外發布{{efn|對外發布的警報採用強震即時警報系統產生的第三報，第一報、第二報不採用<ref>{{Cite ==應用與合作==
強震即時警報可應用於民眾緊急避難應變、高速交通工具及時減速、電梯自動停止、瓦斯管線自動關閉等防災措施{{r|cwb_eew_school|page=14}}<ref>{{Cite web |url = http://scweb.cwb.gov.tw/eew_DEMO/DOC/%E5%BC%B7%E9%9C%87%E9%A0%90%E8%AD%A6Q&A.pdf |title = 強震即時警報Q&A |work = 地震速報系統-強震即時警報宣導網 |publisher = 中央氣象局地震測報中心 |format= PDF |language = zh-tw |accessdate = 2015-11-03 |pages= 9 |archiveurl = https://web.archive.org/web/20151003040028/http://scweb.cwb.gov.tw/eew_DEMO/DOC/%E5%BC%B7%E9%9C%87%E9%A0%90%E8%AD%A6Q&A.pdf |archivedate = 2015-10-03 |dead-url = yes}}</ref>。截至2017年，臺灣有4300個以上的學校、交通主管機關與防救災單位已安裝強震即時警報接收軟體，<ref>{{Cite web |url = https://www.motc.gov.tw/ch/home.jsp?id=724&parentpath=0,1,717 |title = 提升天災測報預警能力，建構客製化與行動化資訊服務 |publisher = 中華民國交通部 |language = zh-tw |date = 2018-06-20 |accessdate = 2018-06-28 |archiveurl = https://www.webcitation.org/70VX0hNMf |archivedate = 2018-06-28 |dead-url = no}}</ref>以學校而言，在接收到強震即時警報後，可利用廣播、[[電子看板|電子看板]]等方式發出警報，進行疏散或就地庇護。{{r|cwb_eew_school_apply|page=8}}2014年有9個單位（學術單位和民間廠商）與[[中央氣象局地震測報中心|中央氣象局地震測報中心]]簽訂「合作推動地震資-{}-訊傳遞服務契約」，提供強震即時警報傳遞服務與開發相關應用，包括警報廣播、地震警報器、行動應用程式等，{{r|cwb_year_report103|page=47}}2015年合計有12個單位與中央氣象局簽訂合作契約<ref name="week1041230" />{{r|cwbeew0206|page=47}}，2016年合計有15個單位與中央氣象局簽訂合作契約{{r|cwb_year_report105|page=49-50}}，2017年合計有11個單位與中央氣象局簽訂合作契約{{r|cwb_year_report106|page=50}}。國家地震工程研究中心亦與中華民國科技部、中華民國教育部及民間業者合作建置現地型地震預警系統，此系統配合演練與[[自動化控制|自動化控制]]，可降低傷亡與損失。<ref>{{cite journal en |author = 許丁友、林沛暘、王修賢、黃謝恭、江宏偉、盧恭君 |date = 2016-03 |title = 現地型強震預警系統於美濃地震之表現 |journal = 國家地震工程研究中心簡-{}-訊 |issue = 97 |language = zh-tw |publisher = 國家地震工程研究中心 |format = PDF |pages = 7 |url = http://www.ncree.org/Files/Publications/10003036.pdf |accessdate = 2017-05-08 |archiveurl = https://web.archive.org/web/20170508020456/http://www.ncree.org/Files/Publications/10003036.pdf |archivedate = 2017-05-08 |dead-url = no}}</ref>

==侷限==
===預警盲區===
由於強震即時警報系統需要收錄震央附近的地震波資料，且資料處理、通訊系統傳輸需要一定時間，此段時間內地震波仍持續傳播，所以接近震央的地區接收到警報時，S波可能已經到達，無法做出預警，這些沒有預警時間的地區稱為預警盲區。<ref name="ncdr_eew" /><ref name="cwb_eew" />區域型強震即時警報系統以發生於臺灣內陸的地震而言，平均約需要18到20秒才能發出警報，預警盲區為距離震央50－70公里以內。{{r|eewregionalonsite|page=5}}臺灣島的南北長約400公里，東西寬約150公里，以區域型系統而言，若大地震發生於島內，臺灣會有大範圍在盲區內，這些區域由於近震央，地震動強、災害嚴重。{{r|eewjournal|page=8}}以[[921大地震|921大地震]]的重災區為例，距離震央在100公里以內，區域型系統的預警時間不到10秒。{{r|onsite|page=38}}現地型強震即時警報系統的預警盲區為距離震央20－30公里以內<ref name="SANLIEN" /><ref name="eew921onsite">{{cite news |author = 湯佳玲、游明金 |title = 國家地震中心發表「現地型」新系統 ／強震即時警報 黃金應變時間增加 |url = http://news.ltn.com.tw/news/life/paper/470668 |work = 自由時報 |date = 2011-02-23 |accessdate = 2015-12-09 |language = zh-tw |archiveurl = https://web.archive.org/web/20151209121812/http://news.ltn.com.tw/news/life/paper/470668 |archivedate = 2015-12-09 |dead-url = no}}</ref>。以921大地震為例，現地型系統可以分別為[[臺中縣|臺中縣]]大里市（今[[臺中市|臺中市]][[大里區|大里區]]）、[[新竹科學園區|新竹科學園區]]、臺北市[[大安區_(臺北市)|大安區]]爭取7、17、27秒的預警時間，{{r|eewregionalonsite|page=6}}分別距離震央43、105、152公里<ref name="eew921onsite" />。

===誤報===
若短時間內地震連續發生，會影響強震即時警報的精確度，可能造成遺漏或誤報。測站背景雜訊、人為干擾、通訊品質不佳等因素也可能造成誤報，<ref name="cwb_eew" />例如因為落雷或監測儀器不正常運作而在沒有發生地震時發布警報<ref name="ncdr_eew" />。2016年11月12日上午11點30分左右，國家地震工程研究中心於嘉義30所中小學設-{}-置的地震預警系統在無地震的情況下發布警報，原因為港坪國小的跑道施工，重機具輾壓造成振動，觸發警報系統並傳到其他學校所造成。<ref>{{cite news |author = 王善嬿 |title = 〈中部〉強震來了？ 嘉市30校烏龍警報 |url = http://news.ltn.com.tw/news/local/paper/1051417 |work = 自由時報 |date = 2016-11-13 |accessdate = 2016-11-13 |language = zh-tw |archiveurl = https://web.archive.org/web/20161113034919/http://news.ltn.com.tw/news/local/paper/1051417 |archivedate = 2016-11-13 |dead-url = no}}</ref>中央氣象局的強震即時警報系統在[[2018年花莲地震|2018年2月花蓮地震序列]]中，2月4日下午9點56分41秒芮氏規模5.8级地震發生後1分鐘的芮氏規模5.1级地震漏發，2月6日下午11點50分42秒芮氏規模6.0级地震發生後3分鐘的芮氏規模5.0级地震漏發。{{r|tec02270102|page=21-24}}

===連線問題===
大地震發生時，電力、通訊可能中斷，造成觀測、計算與通訊系統無法正常運作，導致強震即時警報系統無法運作或產生更大誤差。<ref name="ncdr_eew" /><ref>{{cite journal en |author = 單誠基 |date = 2007-10 |title = 參加「台日強震即時警報系統技術交流研討會」─會後心得 |journal = 港灣報導 |issue = 78 |pages = 29 |publisher = 交通部運輸研究所 |language = zh-tw |url = http://www.ihmt.gov.tw/periodical/pdf/K078.pdf |format=PDF |issn = 1019-2603 |accessdate = 2015-12-11 |archiveurl = https://web.archive.org/web/20151214001551/http://www.ihmt.gov.tw/periodical/pdf/K078.pdf |archivedate = 2015-12-14 |dead-url = no}}</ref>在強震即時警報行動應用程式方面，接收訊息會受到開發者的技術能力、手機是否上線、網路是否暢通、距離震央遠近等影響，{{r|cwbeew0206|page=48}}2016年4月28日前後臺灣東部發生數十起地震，就有民眾反映程式失效，收不到訊息。<ref>{{cite news |author = 閻大富 |title = 地震APP收不到訊息 可能廠商或設定問題 |url =http://www.bcc.com.tw/newsView.2798923 |work = 中廣新聞網 |date = 2016-04-28 |accessdate = 2016-05-03 |language = zh-tw |archiveurl = https://web.archive.org/web/20160502235835/http://www.bcc.com.tw/newsView.2798923 |archivedate = 2016-05-03 |dead-url = yes}}</ref>在災防告警細胞廣播訊息系統方面，接收訊息會受到電信業者的技術、手機型號（是否符合規範）的影響。{{r|eq100|page=67}}
