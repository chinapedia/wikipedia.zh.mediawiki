'''-{zh-hant:強化子;zh-hans:增强子;zh-hk:增強子;zh-mo:增強子}-'''（{{lang-en|enhancer}}）又可譯為'''-{zh-hant:增強子;zh-hans:强化子;zh-hk:強化子;zh-mo:強化子}-'''，是[[DNA|DNA]]上一小段可與[[蛋白質|蛋白質]]（[[反式作用因子|反式作用因子]]；trans-acting factor）結合的區域，與蛋白質結合之後，基因的轉錄作用將會加強。強化子可能位於基因上游，也可能位於下游。且不一定接近所要作用的基因，甚至不一定與基因位於同一[[染色體|染色體]]<ref>{{cite journal en |authors = Spilianakis C, Lalioti M, Town T, Lee G, Flavell R |title = Interchromosomal associations between alternatively expressed loci |journal = [[自然_(期刊)|Nature]] |volume = 435 |issue = 7042 |pages = 637-45 |year = 2005 |PMID = 15880101}}</ref>。這是因為[[染色質|染色質]]的纏繞結構，使序列上相隔很遠的位置也有機會相互接觸。

== 增强子位置 ==
在[[真核生物|真核生物]][[细胞|细胞]]里，[[DNA|DNA]]的[[染色质|染色质]]复合体结构像[[原核生物|原核生物]]的[[超螺旋|超螺旋]]一样折叠。所以虽然增强子与[[基因|基因]]相距很多[[核苷酸|核苷酸]]，但在[[几何|几何]]上两者距离很近。使增强子与[[:en:General_transcription_factor|总转录因子]]及[[:en:RNA_polymerase_II|RNA多聚酶II]]的相互作用成为可能。增强子可以在被它调控基因的[[:en:Upstream_and_downstream_(DNA)|上游或下游]]。而且增强子不一定靠近[[转录|转录]]起始位点才能调控基因转录，已发现有些距离达几十万碱基对。增强子通过与激活蛋白的结合对启动子（不直接对启动子本身）起作用。这些激活蛋白与[[:en:Mediator_(coactivator)|中介复合物（辅激活物）]]相互作用，后者通过使用多聚酶II和总转录因子开始基因转录。曾经在[[内含子|内含子]]内发现增强子。有时增强子的方向颠倒后并不影响它的功能。而且增强子可被切除并插入到染色体的其他位置，仍然影响基因转录。这就是虽然内含子并不转录但其[[多态性|多态性]]会起作用。

目前对于增强子诱发的信息处理有两个不同的理论：<ref name="msu">{{cite journal |authors = Arnosti DN, Kulkarni MM |title = Transcriptional enhancers: Intelligent enhanceosomes or flexible billboards? |journal = J. Cell. * [[:en:Enhanceosome|增强子体]]——依赖高度合作协调行动，可被移动或去除单一蛋白结合位点的单一[[点突变|点突变]]破坏。
* 灵活的公告牌——整合性不强，多种蛋白独立调控基因表达，其作用的总和被基础转录机制读取。

== 特性 ==
# 與方向性無關：方向性不影響它的功能。
# 與位置無關：可位在轉錄起始點上游，也可以在下游。
# 與距離無關：即使調控的基因距離四個鹼基對，仍具有功能。
# 與組織特異性：某種增強子只會提高某種組織的基因進行轉錄。

== HACNS1 ==
HACNS1（又叫[[:en:CENTG2|CENTG2]]，位于[[:en:Human_Accelerated_Region_2|人加速区二]]）是对（独特相对人手指和可能改造脚踝以适应人类两腿行走）进化有贡献的一个基因增强子。如今有证据显示在人类基因组发现的十一万基因增强子中HACNS1在人类与祖先[[黑猩猩|黑猩猩]]分道扬镳之后的[[进化|进化]]过程中变化最大。<ref>[http://www.sciencecodex.com/gene_enhancer_in_evolution_of_human_opposable_thumb HACNS1: Gene enhancer in evolution of human opposable thumb]</ref>
