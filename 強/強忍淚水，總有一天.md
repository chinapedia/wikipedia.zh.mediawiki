{{Japanese|japanese=ミセナイナミダハ、きっといつか |romaji=Misenai Namida wa, Kitto Itsu ka}}
《'''強忍淚水，總有一天'''》（{{lang-ja|ミセナイナミダハ、きっといつか}}）是[[GReeeeN|GReeeeN]]的第15張單曲，於2012年2月29日由[[環球音樂_(日本)|NAYUTAWAVE RECORDS]]發行。

== 概要 ==
單曲距離前作《[[情書～LOVE_LETTER～|情書～LOVE LETTER～]]》發行約3個半月，為[[GReeeeN|GReeeeN]]於2012年發行的第1張單曲。單曲的「初回限定盤」附送收錄了A面曲《強忍淚水，總有一天》[[音樂影片|音樂影片]]的[[數碼多功能影音光碟|數碼多功能影音光碟]]和[[繪本|繪本]]。<ref name="oricon" />

== 歌曲 ==
《強忍淚水，總有一天》是一首正面的歌曲，歌詞描述主角在悲傷下仍然咬緊牙關，堅強地向前進發。<ref>{{ja icon}}{{cite web|url=http://artist.cdjournal.com/d/-/4112011415 |script-title=ja:GReeeeN ／ ミセナイナミダハ、きっといつか |publisher=CD Journal |date=2012 |accessdate=2012-07-15}}</ref>GReeeeN為這首歌下了以下的註腳：{{cquote|新歌《強忍淚水，總有一天》包含了「你的『逞强』十分強大和高尚，一定能成為對某個人的援助」的意思。而且當中的辛酸，將會成為可以引以為傲的事物。<ref>{{ja icon}}{{cite web|url=http://natalie.mu/music/news/62251 |script-title=ja:GReeeeNが竹内結子刑事ドラマ「ストロベリーナイト」主題歌 |publisher=[[Natalie_(網站)|Natalie]] |date=2012-01-05 |accessdate=2012-07-15}}</ref>}}

這首歌是[[富士電視台|富士電視台]][[富士電視台週二晚間九點連續劇|週二晚間九點連續劇]]《[[草莓之夜#電視劇|草莓之夜]]》的主題曲。<ref name="oricon" />

== 音樂影片 ==
《強忍淚水，總有一天》的[[音樂影片|音樂影片]]是GReeeeN首個以[[日本動畫|動畫]]製作的MV，由[[菊池久志|菊池久志]]、[[橋本三郎|橋本三郎]]等人負責製作。MV內容描述一個少女在被邪惡力量支配的世界中，苦於掙扎和勉強自己，去跟該力量抗爭。<ref name="oricon" />除了主角少女外，其他角色都不是人類，而是動物等等。<ref name="natalie" />故事最後以少女流下的淚水，將世界回復原狀（{{lang|ja|返（かえ）る}}，Kaeru）作結。<ref name="oricon">{{ja icon}}{{cite web|url=http://www.oricon.co.jp/news/music/2006954/full/ |script-title=ja:GReeeeN、初のアニメMV完成　菊池久志ら豪華クリエイターとタッグ |publisher=[[Oricon|Oricon]] |date=2012 |accessdate=2012-07-15}}</ref>敵方所有士兵都被變成青蛙（{{lang|ja|蛙（かえる）}}，也是Kaeru）。

動畫原作菊池久志表示他在想這部MV的原案時，是以想描繪從歌中感受到「人類的毅力」為開始的。他又稱將主角少女的感情以表情來表達，是他最留意的地方。<ref name="natalie">{{ja icon}}{{cite web|url=http://natalie.mu/music/news/64411 |script-title=ja:GReeeeN話題ドラマ主題歌PVは初のアニメーション仕立て |publisher=[[Natalie_(網站)|Natalie]] |date=2012-02-14 |accessdate=2012-07-15}}</ref>

== 排行榜成績 ==
單曲唱片的首周銷量約為2.2萬張，在[[Oricon公信榜|Oricon公信榜]]單曲周榜取得第6位。<ref name="oricon weekly" />在[[RIAJ付費音樂下載榜|RIAJ付費音樂下載榜]]上，《強忍淚水，總有一天》以第2位首次上榜，於次周登上首位。<ref name="riaj" />這首歌共連續上榜15星期，落榜前的排名為第92位。<ref>{{ja icon}}{{cite web |url=http://satsuki.musicdb.gr.jp/all_info/ranking_weekly/WeeklyRankingAction.do?term=2012f&weeklyCd=20120605&rankKbn=100 |script-title=ja:レコード協会調べ 2012年05月30日～2012年06月05日 ＜略称：レコ協チャート（「着うたフル(R)」）＞ |publisher=[[日本唱片協會|日本唱片協會]] |date=2012 |accessdate=2012-07-15 |deadurl=yes |archiveurl=https://web.archive.org/web/20141009053350/http://satsuki.musicdb.gr.jp/all_info/ranking_weekly/WeeklyRankingAction.do?term=2012f&weeklyCd=20120605&rankKbn=100 |archivedate=2014年10月9日 }}</ref>
