'''強弱法'''是用來表示一些特定[[音符|音符]]在[[演奏|演奏]]是的[[音强|強弱程度]]。

== 強弱符號 ==
兩個用來表示相對強弱程度的基本[[符號|符號]]是：
* <span style="font-family: 'Times New Roman', Times, serif; font-size: large;">'''''p'''''</span> —[[意大利語|意大利語]] {{lang|it|''piano''}} 的縮寫，表示弱
* <span style="font-family: 'Times New Roman', Times, serif; font-size: large;">'''''f'''''</span> —意大利語 {{lang|it|''forte''}} 的縮寫，表示強

在這兩個基本符號以上建立的另外一些符號包括：
* <span style="font-family: 'Times New Roman', Times, serif; font-size: large;">'''''mp'''''</span> —意大利語 {{lang|it|''mezzo-piano''}} 的縮寫，表示中等弱度（中弱）
* <span style="font-family: 'Times New Roman', Times, serif; font-size: large;">'''''mf'''''</span> —意大利語 ''mezzo-forte'' 的縮寫，表示中等強度（中強）

疊用{{serif|'''''p'''''}}和{{serif|'''''f'''''}}，可以衍生出以下符號：
* <span style="font-family: 'Times New Roman', Times, serif; font-size: large;">'''''pp'''''</span> —意大利語 {{lang|it|''pianissimo''}} 的縮寫，表示極弱
* <span style="font-family: 'Times New Roman', Times, serif; font-size: large;">'''''ff'''''</span> —意大利語 {{lang|it|''fortissimo''}} 的縮寫，表示極強

常用的強弱符號中，強弱的次序如下：

建基於以上的pianissimo和fortissimo，曾有[[音樂家|音樂家]]拓展至{{serif|'''''ffffffff'''''}}（{{lang|it|''fortissississississississimo''}} 的縮寫）和{{serif|'''''pppppppp'''''}}（{{lang|it|''pianissississississississimo''}} 的縮寫），雖然如此，但其實只是為了製造出氣氛，若是真的要演奏這種強弱則不太可能，因為{{serif|'''''ffffffff'''''}}的演奏強度會使[[打擊樂器|打擊類樂器]]或是部分[[弦樂器|弦樂器]]損壞或使其縮減使用年限。

== 突變符號 ==
另外，有一些[[符號|符號]]是用來針對某一個[[音符|音符]]：
* <span style="font-family: 'Times New Roman', Times, serif; font-size: large;">'''''sfz'''''</span> —[[意大利語|意大利語]] ''subito forzando'' 或 ''sforzando'' 的縮寫，表示對這個音符突別加強
* <span style="font-family: 'Times New Roman', Times, serif; font-size: large;">'''''sfp'''''</span> —意大利語 ''subito forzando-piano'' 或 ''sforzando-piano'' 的縮寫，表示在對這個音符突別加強後立即會到弱

== 漸變符號 ==
以下三個意大利語符號常用於表示音強的漸變：
*'''''crescendo'''''（簡作 ''cresc.''）：漸強
*'''''decrescendo'''''（簡作 ''decresc.''）：漸弱
*'''''diminuendo'''''（簡作 ''dim.''）：漸弱

此外，「＜」與「＞」符號也用於表現音強的漸變。向右開口的「＜」表示漸強，向左開口的「＞」表示漸弱。以下譜表中，音樂起以中強音，然後漸強，最後漸弱。

== 其他文字符號 ==
* '''''al niente'''''：漸弱至到無聲
* '''''calando'''''：漸弱
* '''''crescendo'''''：漸強
* '''''decrescendo''''' 或 '''''diminuendo'''''：漸弱
* '''''perdendo''''' 或 '''''perdendosi'''''：漸弱至無聲
* '''''morendo'''''：漸弱至無聲
* '''''marcato'''''：突強
* '''''sotto voce'''''：低聲地
* '''''in rilievo'''''：代表某一個樂器的音量比其他突出
