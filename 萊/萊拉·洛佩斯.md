'''萊拉·洛佩斯'''（'''Leila Luliana da Costa Vieira Lopes'''，{{bd|1986年|2月26日|||catIdx=L}}），2011年度英國安哥拉裔小姐、2011年度安哥拉小姐及[[2011年度環球小姐比賽|2011年度環球小姐]]。

== 個人生活 ==
洛佩斯在[[安哥拉|安哥拉]][[本吉拉|本吉拉]]出生，她是一名在英國就讀商科的學生。2010年10月8日奪得英國安哥拉裔小姐，代表安哥拉裔英國人參加2011年度安哥拉小姐。

== 2011年度英國安哥拉裔小姐 ==
洛佩斯在2010年10月8日奪得英國安哥拉裔小姐，代表安哥拉裔英國人參加2011年度安哥拉小姐。<ref>[http://club-k.net/diaspora/6139-comunidade-denuncia-irregularidades-no-miss-angola-reino-unido.html Angolan community denounces irregularities at Miss Angola UK] {{webarchive|url=https://web.archive.org/web/20140717140030/http://club-k.net/diaspora/6139-comunidade-denuncia-irregularidades-no-miss-angola-reino-unido.html |date=2014-07-17 }} retrieved September 13, 2011 (pt)</ref><ref name="angolauk"/>

== 2011年度安哥拉小姐 ==
洛佩斯於2010年12月18日奪得2011年度安哥拉小姐競選冠軍和最上鏡小姐，從而代表安哥拉參加[[2011年度環球小姐比賽|2011年度環球小姐競選]]。<ref name="results"/><ref name="venue"/>

== 2011年度環球小姐競選 ==
洛佩斯於2011年9月12日在[[巴西|巴西]][[聖保羅|聖保羅]]舉行的[[2011年度環球小姐比賽|2011年度環球小姐競選]]，奪得'''2011年度環球小姐'''。由上屆冠軍來自墨西哥的[[希梅納·納瓦瑞提|希梅納·納瓦瑞提]]在聖保羅為她加冕桂冠<ref>{{cite web|title=Miss Angola Leila Lopes is Miss Universe 2011!|url=http://www.beautymania.biz/2011/09/royals-update-miss-universe-2011.html|publisher=BeautyMania.biz|accessdate=12 September 2011|date=September 12, 2011}}</ref>，成為第60名奪得環球小姐殊榮的佳麗。洛佩斯是安哥拉歷史上第一位環球小姐，也是環球小姐歷史上第四位奪得環球小姐殊榮的非洲人 (南非曾在1978年奪得、納米比亞曾在1992年奪得、博茨瓦納曾在1999年奪得和安哥拉在2011年奪得)。另外，洛佩斯是環球小姐歷史上第二位來自非洲的黑人佳麗奪得環球小姐殊榮，對上一次是1999年由博茨瓦納小姐奪得。洛佩斯成為第一位安哥拉代表在世界四大選美比賽中奪得冠軍。
