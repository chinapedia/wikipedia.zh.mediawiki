'''萊娜·邁爾-蘭德魯特'''（Lena Meyer-Landrut，{{bd|1991年|5月23日|catIdx=Meyer-Landrut}}）<ref>{{cite web |author=Maier, Jens |url=http://www.stern.de/kultur/musik/interview-lena-meyer-landrut-an-kampf-zu-denken-liegt-mir-nicht-1550854.html |title=Interview Lena Meyer-Landrut: "An Kampf zu denken, liegt mir nicht" |publisher=Stern.de |date=2010-03-16 |accessdate=2010-04-09 }} {{de icon}}</ref> 。[[德國|德國]]女歌手。代表[[德國|德國]]參加[[2010年歐洲歌唱大賽|2010年歐洲歌唱大賽]]以歌曲《[[卫星_(歌曲)|衛星]]》取得冠軍，隔年再度代表[[德國|德國]]參加[[歐洲歌唱大賽|歐洲歌唱大賽]]以歌曲《被陌生人帶走》取得第十名。並在2013-2016年、2019年在德國版本的兒童好聲音節目上擔任1-4、7季導師。<ref>{{cite web |url=http://www.sat1.de/tv/the-voice-kids/coaches |title=Neues Jahr, neue Coaches!}} {{de icon}}</ref>

== 早年生活 ==
萊娜·邁爾-蘭德魯特出生于[[德国|德国]][[汉诺威|汉诺威]]<ref name="ad-hoc">Deutscher Depeschendienst（2010年3月12日）. [http://www.ad-hoc-news.de/lena-meyer-landrut-zwischen-grand-prix-und-schulbank--/de/Boulevard/Musik/21126433 Musik: Lena Meyer-Landrut: Zwischen Grand Prix und Schulbank]. Ad-hoc-news.de. 于2010年3月24日查阅. {{de icon}}</ref>，是前[[西德|西德]]駐[[蘇聯|蘇聯]][[莫斯科|莫斯科]]大使{{link-en|安德雷亚斯·邁爾-蘭德魯特|Andreas Meyer-Landrut}}的孫女<ref>{{cite web |url= http://www.bebraverlag.de/autoren/autor/197-Andreas_Meyer-Landrut |title= Andreas Meyer-Landrut |publisher=be.bra verlag |accessdate=2010-03-27 }} {{de icon}}</ref><ref name="bunte">Harders, Antje. "NACHGEFRAGT ... bei Nachwuchstalent Lena Meyer-Landrut". ''[[:en:Bunte|Bunte]]''. 2010/7, p. 110. {{de icon}}</ref><ref>{{cite web |author=Auswärtiges Amt |url=http://www.auswaertiges-amt.de/diplo/en/Infoservice/Presse/Reden/Archiv/2003/030217-10thAnniversaryDeutschRussischesForum.html |title=Germany – Speech on the 10th anniversary of the Deutsch-Russisches Forum |publisher=Auswaertiges-amt.de |date=2003-02-17 |accessdate=2010-03-24 }}</ref>。

作为家中的独女<ref name="ad-hoc"/>，莱娜由母亲独自在汉诺威的Misburg-Anderten区抚养长大<ref>{{cite web |author=Tina Stockhausen |url=http://nachrichten.rp-online.de/panorama/alle-lieben-lena-1.73039 |title=Alle lieben Lena |publisher=RP-Online |date=2010-04-21 |accessdate=2010-04-22 |deadurl=yes |archiveurl=https://web.archive.org/web/20100424070257/http://nachrichten.rp-online.de/panorama/alle-lieben-lena-1.73039 |archivedate=2010-04-24 }}</ref>，其父亲在她2岁大时就离开了这个家<ref>{{cite web |url=http://www.focus.de/panorama/boulevard/lena-meyer-landrut-vater-sieht-sich-nur-als-erzeuger_aid_507455.html |title=Seit 16 Jahren haben sich Lena Meyer-Landrut und ihr Vater Ladislas nicht gesehen |publisher=focus-online |date=2010-05-12 |accessdate=2010-05-12 }}</ref>。她5歳開始學習[[芭蕾舞|芭蕾舞]]，之後又學習了包括[[爵士舞|爵士舞]]和{{link-en|嘻哈舞|Hip-hop dance}}在內的多種现代舞蹈<ref name=profil>Norddeutscher Rundfunk. [http://eurovision.ndr.de/teilnehmer/lenameyerlandrut100.html Lena Meyer-Landrut] {{webarchive|url=https://web.archive.org/web/20100328115919/http://eurovision.ndr.de/teilnehmer/lenameyerlandrut100.html |date=2010-03-28 }}. Eurovision.ndr.de. 于2010年3月24日查阅. {{de icon}}</ref>。此后，莱娜渐渐对唱歌产生了浓厚的兴趣，并且作为[[臨時演員|臨時演員]]出现在一些德国电视剧中<ref name="ad-hoc"/>，而此前她从未接受过任何正规的表演或声乐训练<ref>{{cite web |author=Fraczek, Jennifer |url=http://www.welt.de/fernsehen/article6380364/Wie-aus-Lena-Meyer-Landrut-ploetzlich-ein-Star-wird.html |title=Kandidatin für Oslo: Wie aus Lena Meyer-Landrut plötzlich ein Star wird |publisher=Welt.de |date=2010-02-13 |accessdate=2010-03-24 }} {{de icon}}</ref>。

她就读于汉诺威的Roderbruch[[德國教育#第二級初階Sekundarbereich_I|综合中学]]（Integrierte Gesamtschule）<ref>{{cite web |author=Güngör Öztürker |url=http://www.welt.de/news/article6748117/Lena-Meyer-Landrut-auf-dem-Weg-nach-Oslo.html |title=USFO: Lena Meyer-Landrut auf dem Weg nach Oslo |publisher=Welt.de |date=2010-03-12 |accessdate=2010-03-12 }}</ref>，并于2010年4月参加了高中毕业考（[[:en:Abitur|Abitur]]）<ref name=profil/>。

== 媒体对她的印象 ==
