《'''戀愛暴君'''》（{{lang-ja|恋する暴君}}）為日本[[漫畫家|漫畫家]][[高永ひなこ|高永ひなこ]]的作品，以及[[跨媒體製作|跨媒體製作]]的[[廣播劇CD|廣播劇CD]]與[[OVA|OVA]]。於海王社發行BL漫畫雑誌《GUSH》上連載，連載至第10章尚未完結。2005年2月10日開始發行[[單行本|單行本]]，目前共11卷<ref>{{ja}}[http://www.gushnet.jp/search/a37891.html 作家別検索／高永ひなこ]</ref>。

== 故事簡介 ==
大學生的森永，暗戀非常討厭同性戀且個性凶暴蠻橫的學長巽，終於要步入第5年了，想要傳達自己的心意，甚至是一親芳澤就宛如是夢中之夢一般……而就在森永已經半死心的時候，一個絕佳的機會來了！森永多年的思念是否能得到回報呢？

== 登場人物 ==
; 森永 哲博（[[配音|配音]]：[[鳥海浩輔|鳥海浩輔]]）
: 於大學院農學部的碩士課程中研究生物技術。於大學入學時對宗一一見鍾情，從而開始了長達4年的暗戀。
:初次的隔天早上被宗一生氣地打了一拳後，消失了2周。
:因為廣人給的一瓶“酒”，而與學長有了更近一步的關係。被廣人叫做“小天使”，其原因是因為[[森永牛奶糖|某牛奶糖]]的商標。
; 巽 宗一（配音：[[緑川光|緑川光]]）
: 森永的戀人，與森永於同一個農學部的博士課程進修中，是森永的學長。有著修長身形和纖細的腰，有著與其秀麗的外表相反的凶暴蠻橫的個性。以前曾被研究室中的助教性騷擾過，在那之後變得非常討厭同性戀。在得知美國舊金山同意同性婚姻後，差點想炸了美國殺掉黑川。
; 巽 巴（配音：[[宮田幸季|宮田幸季]]）
: 宗一的弟弟、巽家的次男。現居於美國，與黑川相戀並已結婚。被某企業拉攏到美國。頭腦非常聰明，卻是個天然呆。有著奶油糖般的秀髮。
; 黒川 貢（配音：[[杉田智和|杉田智和]]）
: 巴的戀人。現在捨棄了[[上班族|上班族]]的生活與巴一同在美國生活中。
; 廣人'''（ヒロト）'''（配音：[[平川大輔|平川大輔]]）
: 森永的朋友，在同性戀酒吧工作。是森永困惑時的商量對象。稱呼森永為“小天使”。
; 巽 佳奈子（配音：[[許綾香|許綾香]]）
: 宗一與巴的妹妹、巽家的長女。是個只花了一晚就接受自己的哥哥週遭是同性戀者的國中生。
; 磯貝 太一郎（配音：[[關俊彥|關俊彥]]）
: 黒川的大學同學兼好友，大學時期曾一起打過工。
; 森永 國博（配音：[[成田劍|成田劍]]）
: 森永哲博的哥哥。
; 真崎 順也
: 森永哲博的初戀對象，國博的朋友。
