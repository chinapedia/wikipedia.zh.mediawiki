'''禮文機場'''（{{jpn|j=礼文空港|hg=れぶんくうこう|rm=Rebun kūkō|f=y}}，{{airport codes|RBJ|RJCR}}）是位於[[北海道|北海道]][[禮文郡|禮文郡]][[禮文町|禮文町]]（[[禮文島|禮文島]]）。根據日本空港整備法被分成地方管理機場。現在已經關閉。

== 概要 ==
*日本最北的機場。
*有機場前的禮文島，如果因天氣惡劣而渡輪停開的話，禮文島便沒有交通工具對外。因此，居民認為要盡快建設機場。

== 歷史 ==
*1978年6月：機場啟用
*2003年3月：取消航線往稚內機場。
*2009年4月9日：停用機場。

== 航空公司與航點 ==
*現在沒有定期班次，只有緊急運輸等。

== 如何前往 ==
*曾有[[宗谷巴士|宗谷巴士]]到達。不過，現在達乘出租車到達。
*由禮文町船泊分所徒步需時35分鐘。
