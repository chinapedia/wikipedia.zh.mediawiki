'''直島群島'''（{{jpn|j=直島諸島|hg=なおしましょとう|rm=Naoshima shotō|f=y}}）為位於[[瀨戶內海|瀨戶內海]]的島嶼群，共有27座島嶼，其中有人島5座（直島、向島、家島、牛首島、屏風島）、[[荒島|無人島]]22座，其主要島嶼[[直島町|直島]]位於香川縣[[高松市|高松市]]北方約13km，[[岡山縣|岡山縣]][[玉野市|玉野市]]南方約2km。行政區劃上除[[井島|井島]]的北半部屬於[[岡山縣|岡山縣]][[玉野市|玉野市]]以外，其他所有島嶼及井島的南半部皆屬[[香川县|香川県]][[直島町|直島町]]<ref>{{Cite book|和書 |author = 角川日本地名大辞典編纂委員会|title = 角川日本地名大辞典37 香川県|publisher = 角川書店|pages = 569頁|id = |isbn = 978-4-04-001370-1|date=1985年9月}}</ref><ref>{{Cite web |url = http://www.benesse-artsite.jp/about/island.html|title = 関連島について|publisher = ベネッセアートサイト直島|accessdate = 2014-08-15}}</ref><ref>{{Cite web |url = http://kotobank.jp/word/%E7%9B%B4%E5%B3%B6%E8%AB%B8%E5%B3%B6|title = 世界大百科事典 第2版「直島諸島」|publisher = kotobank|accessdate = 2014-08-15}}</ref><ref>{{Cite web |url = http://naoshima.cyapo.com/|title = 直島 - Naoshima - を知る|publisher = Navi! （タビナビ）|accessdate = 2014-08-15}}</ref>。

在[[江户时代|江戶時代]]與[[男木島|男木島]]、[[女木島|女木島]]合稱「直島3島」，最早屬於[[吉備國|吉備國]][[兒島郡|兒島郡]]，後被分出改隸屬[[讚岐國|讚岐國]]，並隸屬設置相當於郡級的單位「[[小豆島_(日本)|小豆島]]」，直到江戶時代末期，1872年被併入[[香川郡|香川郡]]<ref>{{Cite book|和書 |author = 角川日本地名大辞典編纂委員会|title = 角川日本地名大辞典37 香川県|publisher = 角川書店|pages = 795頁|id = |isbn = 978-4-04-001370-1|date=1985年9月}}</ref>。

此區域主要産業為[[渔业|漁業]]、[[旅游|觀光]]，直島還有金屬[[精錬|精煉]]工廠；近年來開始發展藝術觀光，直島上設有[[度假村|度假村]]及多個[[當代藝術|當代藝術]]作品，2010年起，與周邊數座島嶼成為[[瀨戶内國際藝術祭|瀨戶内國際藝術祭]]舉辦場地。<ref>{{Cite journal| journal=天下雜誌| url=http://www.cw.com.tw/article/article.action?id=5076979| title=直島　藝術逐夢瀨戶內海| issue=600| date=2016-06-21| author=周慧菁| accessdate=2017-04-28}}</ref>
