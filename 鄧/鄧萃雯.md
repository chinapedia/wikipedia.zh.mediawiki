'''鄧萃雯'''（'''Sheren Tang Shui Man'''；{{bd|1966年|3月2日||}}），[[香港|香港]]資深女艺人；曾為[[電視廣播有限公司|無綫電視]]及[[亞洲電視|亞洲電視]]合約女藝員。曾在2009年和2010年連續兩年奪得無綫[[萬千星輝頒獎典禮|萬千星輝頒獎典禮]]「最佳女主角」，史上第一位「連莊視-{后}-」。近年主要拍攝[[中國大陸電視劇|中國大陸電視劇]]。

==簡歷==
鄧萃雯曾就讀[[協恩中學附屬幼稚園|協恩中學附屬幼稚園]]、[[協恩中學附屬小學|協恩中學附屬小學]]，並畢業於[[協恩中學|協恩中學]]，後於1984年進入[[無綫電視藝員訓練班|無綫電視藝員訓練班]]第二期藝員招募（當年提名人為影帝[[劉德華|劉德華]]<ref>{{cite web | language =  | publisher = Youtube | title = 九姑娘自爆第一次獻華仔 | url = http://www.youtube.com/watch?v=oLRB6xvI4eA| date = }}</ref>），同期同學有[[邵美琪|邵美琪]]、[[郭富城|郭富城]]、[[陳庭威|陳庭威]]和[[黎美嫻|黎美嫻]]等人。她是一名虔誠的[[基督|基督]][[新教徒|新教徒]]，現為[[香港藝人之家|香港藝人之家]]主席。<ref>{{cite web | language = zh-hant | publisher =  | title = 虔誠基督徒 鄧萃雯 受訪記 | url = http://www.gnci.org.hk/beta/gospel/sheren_tang/index.php| date = }}</ref>

==演藝經歷==
===無綫早期（1985年－1995年）===
1985年獲監製[[蕭顯輝|蕭顯輝]]邀請演出她的第一部劇集《[[薛仁貴征東|薛仁貴征東]]》，她演出的第一部劇集便擔任第一女主角，當年該劇大受歡迎，平均收視60點。之後在多部劇集中擔演主要角色，如古裝武俠劇《[[俠客行|俠客行]]》中飾[[丁噹|丁噹]]、《[[倚天屠龍記|倚天屠龍記]]》中飾[[周芷若|周芷若]]，以青春可愛的少女形象打動觀眾，曾有「[[翁美玲|翁美玲]]的接班人」、「最佳古裝美人」等封號。<ref>{{cite web| language =| publisher =深圳網| title =44歲鄧萃雯被称"翁美玲接班人"| url =http://www.sznews.com/ent/content/2010-12/29/content_5212023.htm| date =2010年12月29日| deadurl =yes| archiveurl =https://web.archive.org/web/20110307064735/http://www.sznews.com/ent/content/2010-12/29/content_5212023.htm| archivedate =2011年3月7日| df =}}</ref>。同年，擔任第一女主角的作品有《[[遁甲奇兵|遁甲奇兵]]》等。後則在《[[開心女鬼|開心女鬼]]》中擔任第二女主角，飾演第一女主角[[鄭裕玲|鄭裕玲]]的妹妹楊可兒。

後期，鄧萃雯於每部劇集當中均擔任第一、二女主角，1989年，在《[[萬家傳說|萬家傳說]]》中與[[藍潔瑛|藍潔瑛]]第二次合作，並於劇中飾演多個隱蔽角色，亦和該時的一線小生[[郭晉安|郭晉安]]有許多對手戲，對她來說是一個大挑戰。1990年，她參演《又是冤家又聚頭》飾演情婦角色，與[[毛舜筠|毛舜筠]]飾演的正室有相當多的戲劇火花。

1992年鄧萃雯與[[無綫電視|無綫電視]]的基本合約正式完結，她決定前往美國修讀室內設計，之後幾年只在暑假期間回港替無綫拍劇，包括《血濺塘西》、《[[生死訟|生死訟]]》及《[[再見亦是老婆|再見亦是老婆]]》等作品。鄧萃雯表示在美國進修期間，看到同校的戲劇系同學如何投入學習演戲的態度，啟發了她以後的演藝道路。

===亞視時期（1996年－1997年）===
1995年鄧萃雯完成學業後，一度有離開演藝界打算的她卻再度重投演藝界。當時，鄧萃雯已沒有再次重返無綫，反而於1996年加入[[亞洲電視|亞洲電視]]，並開始演出亞洲電視的劇集，第一部接拍的劇集是《[[再見艷陽天|再見艷陽天]]》，飾演張文鳳一角，和[[陳秀雯|陳秀雯]]、[[馬景濤|馬景濤]]、[[林韋辰|林韋辰]]等合作，並擔任第二女主角。同年也參演《[[我和春天有個約會_(1996年電視劇)|我和春天有個約會]]》，在劇中飾演第一女主角姚小蝶，她在劇中的演出大獲好評，她在劇中所演唱的「離春天那麼遠」一炮而紅，如今不少的觀眾仍鍾意該首歌曲，當時紅極一時。1997年，鄧萃雯演出《[[國際刑警_(1997年電視劇)|國際刑警1997]]》後，與[[亞洲電視|亞洲電視]]的兩年部頭合約正式完結，從而離開。因此這部劇是她在亞視的告别作。

===大陸時期（1997年至今）===
鄧萃雯在[[亞洲電視|亞洲電視]]時，因是旗下部頭合約藝員，才能夠到處參演劇集。於是，雯女於1997年接拍[[台灣電視劇|台灣電視劇]]《[[施公奇案_(台灣)|施公奇案]]》，在此劇兩個單元中擔任女主角，與多位台灣藝人合作。隨後，鄧萃雯全力活躍在台灣電視劇中。2004年起，雯女開始進軍大陸，接演大陸劇集，在大陸劇集中都有出色的演出，著名作品包括，《[[青城之戀|青城之戀]]》、《[[玫瑰江湖|玫瑰江湖]]》、《[[牌坊下的女人|牌坊下的女人]]》、《[[還珠格格_(2011年電視劇)|新還珠格格]]》及《[[傾城雪|傾城雪]]》等劇集都深入民心。鄧萃雯於電視劇《牌坊下的女人》及《青城之戀》中，所飾演的角色大獲好評，將角色演得活靈活現。此後，鄧萃雯於中國電視劇中也佔「一姐」位置，獲多位新生代女演員當作學習榜樣。曾憑著《青城之戀》奪得金南方年度「南方盛典」金南方年度最佳电视剧女演员獎。

鄧萃雯在[[中國大陸電視劇|中國大陸電視劇]]，偶爾也獲監製提拔在一些劇中擔任第一女主角，擔任女一號的作品有，《[[少年英雄之洪文定|少年英雄之洪文定]]》、《[[青城之戀|青城之戀]]》、《[[牌坊下的女人|牌坊下的女人]]》等作品。

2011年憑瓊瑤一劇《[[還珠格格_(2011年電視劇)|新還珠格格]]》的[[繼皇后|繼皇后]]一角<ref>{{cite web | language = zh-hant | publisher = 騰訊娛樂 | title = 鄧萃雯成《新還珠格格》亮點 這個皇后很溫柔 | url = http://ent.qq.com/a/20110722/000313.htm| date = 2011年7月22日 }}</ref><ref>{{cite web | language = zh-hans | publisher = 網易娛樂 | title = 新還珠格格 皇后呼声最高 鄧萃雯：皇后不算狠 | url = http://ent.163.com/11/0720/03/79CJ2VQM00032DGD.html| date = 2011年7月20日}}</ref>，擊敗對手，成功奪得[[華鼎獎|華鼎獎]]「最佳女配角」獎項。<ref>{{cite web | language = zh-hans | publisher = 中國網絡電視台 | title = 2011華鼎獎祖峰鄧萃雯 分别獲最佳男女配角 | url = http://dianshiju.cntv.cn/20111208/100870.shtml| date = 2011年12月8日}}</ref>

2012年她參演了一部偶像劇《[[棋逢對手|棋逢對手]]》，是她第二次拍攝偶像劇<ref>{{cite web | language = zh-hant | publisher = 蘋果日報（香港） | title = 鄧萃雯私伙靚衫演惡魔 | url = http://hk.apple.nextmedia.com/entertainment/art/20120712/16505539| date = 2012年7月12日}}</ref><ref>{{cite web| language =zh-hans| publisher =新華娛樂| title =棋逢對手熱拍 鄧萃雯自備服裝變 “時尚女魔頭”| url =http://news.xinhuanet.com/ent/2012-07/11/c_123397782.htm| date =2012年7月11日| deadurl =yes| archiveurl =https://web.archive.org/web/20131007224623/http://news.xinhuanet.com/ent/2012-07/11/c_123397782.htm| archivedate =2013年10月7日}}</ref>。並在劇中飾演蘇珊娜一角，挑戰大惡鬥，此劇現今在[[河北衛視|河北衛視]]火熱播出。目前正在拍攝《[[風雷急|風雷急]]》，與實力派演員[[李立群|李立群]]及[[呂良偉|呂良偉]]等人合作，擔任第一女主角，並飾演戲班班主付玉霞一角。

===重返無綫（1999年－2012年）===
1999年，鄧萃雯選擇重返[[無綫電視|無綫電視]]，並參演《[[LovingYou我愛你|LovingYou我愛你]]》，亦再度和[[吳啟華|吳啟華]]合作，後也獲監製[[戚其義|戚其義]]邀請於《[[妙手仁心II|妙手仁心II]]》客串徐嘉詠一角。

2003年，鄧萃雯第一次參演偶像劇《[[戀愛自由式|戀愛自由式]]》，首次飾演女主角母親的角色，但鄧萃雯也用心去演活角色。恰好該年遇上無綫大裁員，鄧萃雯感到慶幸仍可再次於無綫劇集當中擔任重要角色。後憑同年劇集《[[LovingYou我愛你2|LovingYou我愛你2]]》的多變豐富演出打敗多位實力派演員奪得《[[2003年度萬千星輝賀台慶|萬千星輝頒獎典禮2003]]》的「本年度我最喜愛的實力非凡女藝員」獎項。（类似近期TVB頒獎典禮的最佳女配角獎項）。

2004年，鄧萃雯參演《[[金枝慾孽|金枝慾孽]]》，並在劇中飾演狠毒的鈕祜祿·如玥（如妃）一角，其演技大獲好評。其後因該劇在大陸首播以致她所飾演的「如妃」一角，深受許多大陸網民愛戴，她的演技已獲得整個大中華地區的觀眾認同，被許多觀眾封為「視壇天后」。因《金枝慾孽》的成功導致許多網民及觀眾盼無綫再製作《金枝慾孽》續集。憑《[[金枝慾孽|金枝慾孽]]》榮獲《[[2004年度萬千星輝賀台慶|萬千星輝頒獎典禮2004]]》的「本年度我最喜愛的實力非凡女藝員」及「本年度我最喜愛的電視角色」，也是無綫史上首位“連莊女配角”。除此之外，提名「本年度我最喜愛的女主角」（最後五強），並視為大熱門之一，惜最終敗給[[黎姿|黎姿]]。

2007年，在師奶劇集《[[師奶兵團|師奶兵團]]》中精湛的演出提名「最佳女主角」（最後五强）及「我最喜愛的電視女角色」，鄧萃雯亦藉60集的中港合拍劇《[[歲月風雲_(電視劇)|歲月風雲]]》中的汪紹芬（Fanny）一角而人氣急升，並首度和[[苗僑偉|苗僑偉]]合作演出兩夫妻。<ref>{{cite web| language =| publisher =新華網| title =鄧萃雯:用"心"演繹每個角色| url =http://news.xinhuanet.com/ent/2007-07/11/content_6359125.htm| date =2007年7月11日| deadurl =yes| archiveurl =https://web.archive.org/web/20160822191727/http://news.xinhuanet.com/ent/2007-07/11/content_6359125.htm| archivedate =2016年8月22日}}</ref>。並憑此劇提名「內地觀眾最喜愛TVB女藝人」。

2008年，鄧萃雯與2000年視帝[[歐陽震華|歐陽震華]]再度合作，在《[[尖子攻略|尖子攻略]]》中和歐陽震華扮演一對鬥氣冤家，於劇中擦出火花。鄧萃雯憑此劇中的嚴家勵（Kelly）一角提名《[[2008年萬千星輝頒獎典禮|2008年萬千星輝頒獎典禮]]》的「最佳女主角」（最後十强）。

2009年，鄧萃雯於《[[巾幗梟雄|巾幗梟雄]]》中飾演的康寶琦（四奶奶）一角，<ref>{{cite web | language =  | publisher = 新浪網| title = 鄧萃雯小女人大智慧 《巾幗梟雄》謀略當先 | url = http://dailynews.sina.com/bg/ent/tv/sinacn/file/20110110/18312149826.html| date = 2011年1月10日 }}</ref>演技精湛，深入民心，再次成為城中熱話，人氣急升，衆望所歸，憑大熱劇集於《[[2009年萬千星輝頒獎典禮|2009年萬千星輝頒獎典禮]]》中以大熱門姿態榮獲「最佳女主角」，<ref>{{cite web | language =  | publisher = 網易娛樂| title = 鄧萃雯憑 《巾幗梟雄》四奶奶奪最佳女主角 | url = http://ent.163.com/09/1204/22/5PNLSD1P0003401E.html| date = 2009年12月4日 }}</ref>也是首位讀無綫電視藝員訓練班奪得視-{后}-的女演員。

2010年，鄧萃雯在《[[巾幗梟雄|巾幗梟雄]]》的續集《[[巾幗梟雄之義海豪情|巾幗梟雄之義海豪情]]》中飾演鄭九妹（九姑娘）一角，再獲關注，憑此劇於《[[2010年萬千星輝頒獎典禮|2010年萬千星輝頒獎典禮]]》再獲「最佳女主角」，成為香港首屆的「連莊視-{后}-」。並成功打敗大熱門候選「三公主」[[佘詩曼|佘詩曼]]成為視-{后}-。<ref>{{cite web | language =  | publisher = 網易娛樂| title = 黎耀祥鄧萃雯憑 《巾幗梟雄之義海豪情》蟬聯視帝視后 | url = http://ent.163.com/10/1206/01/6N6ELEK100032DGD.html| date = 2010年12月6日 }}</ref>此外，她還在馬來西亞及中國大陸的殿堂級頒獎典禮當中以《[[巾幗梟雄之義海豪情|巾幗梟雄之義海豪情]]》奪得多個獎項，如《[[MY_AOD我的最愛頒獎典禮|MY AOD我的最愛頒獎典禮2010]]》奪得「我的最愛十大電視角色」、《[[演藝動力大獎|明報週刊演藝動力大獎]]》奪得「最突出電視女藝員」、《[[星和無綫電視大獎|星和無綫電視大獎]]》奪得「我最愛TVB電視女角色」、《TVshow2010娛樂電視大獎》奪得「年度我最愛的電視女角色」、《巴巴閉邊個夠我查篤撐》奪得「網民評選最佳女主角」及《[[2011年國劇盛典獲獎名單|2011國劇盛典]]》奪得「年度網路最受歡迎港台女演員」，是實至名歸的「七料視-{后}-」。

== 個人生活 ==
鄧萃雯有過兩段感情，鄧萃雯只承認過曾與[[萬梓良|萬梓良]]和[[鄭敬基|鄭敬基]]交往，後也是以分手收場。<ref>{{cite web| language = zh-hans| publisher = | title = 鄧萃雯經歷多個感情 坦言想結婚| url = http://et.21cn.com/star/zazhi/2010/12/10/7996648.shtml| date = 2010年12月10日| access-date = 2013年3月6日| archive-url = https://web.archive.org/web/20140116072240/http://et.21cn.com/star/zazhi/2010/12/10/7996648.shtml| archive-date = 2014年1月16日| dead-url = yes}}</ref>

在1985年，鄧萃雯和[[萬梓良|萬梓良]]拍攝劇集《[[薛仁貴（1985年电视剧）|薛仁貴征東]]》時，與大她9年的萬梓良「戲假情真」，當時萬梓良坦言會等她十年才結婚，但兩人拍拖一年半後，因性格不合而分手。2009年，鄧萃雯在接受[[陳志雲|陳志雲]]節目《[[志雲飯局|志雲飯局]]》時也透露曾與萬交往。

1998年，鄧萃雯和演員[[鄭敬基|鄭敬基]]交往，她表示鄭對她很好，但一年後都是以分手收場。

{{Who|有傳}}鄧萃雯在1996年拍攝《我》劇集時與男主角[[江華|江華]]交往。當時江華已婚，妻子為[[麥潔文|麥潔文]]。有傳因此鄧萃雯介入江華和麥潔文婚姻當第三者，後自動退出。
