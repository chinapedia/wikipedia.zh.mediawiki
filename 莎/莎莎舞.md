薩爾薩舞（Salsa Dance），风格平民化，自由气息浓郁，强调个性化、不受拘束。同时，因为脱胎于拉丁舞，它保留了典雅气质。这样一种雅俗共赏的激情热舞，自然吸引了众多年轻人

== 起源 ==

Salsa的词意是一种食物调料酱，西班牙发音为“萨尔萨”，故又称萨尔萨舞。它起源于古巴，是当今欧美非常流行的社交舞蹈之一。莎莎舞在纽约和福罗瑞达两地区的拉丁社区得到发展，逐渐发展出多种风格。不同地域的莎莎舞都有所不同，但其基本步伐节奏相似，有纽约莎莎舞（New York Salsa）、哥伦比亚莎莎舞（Colombia Salsa）、洛杉矶莎莎舞（L.A. Salsa）和古巴莎莎舞（Cuba Salsa）等，纽约跳法是目前的主流跳法。舞蹈配乐在曼波舞蹈的基础上进行改进，延用了Son乐的结构，每一音乐节拍为6至8拍。舞蹈动作花样繁多、变化快，以突出腰胯的“8”字形摆动为特色，具有很强的欣赏性和表演性，让众多时尚男女为之着迷。

== 类型 ==
在一些风格的莎莎舞中，例如洛杉矶风格和纽约风格，舞者通常在一个位置或一条线(交换位置)上运动；还有一些风格，如古巴莎莎舞，舞者通常互相绕圈，通常为三个点上运动。这种绕圈的风格受Cuban Son影响较大，尤其是20世纪20年代Son Montuno鼓点，然而Son Montuno作为流行音乐的一种，很适合即兴演奏，还在不断地发展之中。新的现代风格莎莎舞通常根据其起源的地理位置来命名，在起源地之外也有大量的追随者。借鉴其他舞蹈的风格与技巧融入到莎莎舞里非常普遍，用于区分这些风格的特征包括：节奏、基本步、身体律动、旋转、观念、舞蹈表现力以及舞伴之间握手的方式等。




==动作==
在許多莎莎舞的風格中，舞者通過踩踏來改變體重，上半身保持水平，幾乎不受體重變化的影響。重量變化導致臀部移動。手臂和肩膀的動作也被納入。古巴賭場的莎莎跳舞風格涉及腰部以上的顯著運動，上下肩部運動和胸腔移位。

“領舞者”使用這些手臂在“打開”或“關閉”位置傳達或發出“跟隨者”的信號。開放的姿勢要求兩個舞者握住一隻或兩隻手，特別是涉及轉身，背後放置手臂，或者相互移動的動作等等。在關閉的位置，領導者把右手放在跟隨者的背上，而跟隨者把左手放在領導者的肩上。

在原始的拉丁美洲形式，莎莎的前進/後退運動是在對角線或側向進行，三步重量變化完整。

在某些莎莎風格中，如紐約風格，舞者大多保持在彼此的前面（轉換場所），而在古典風格的拉丁美洲風格中，舞者圍繞著彼此，有時是3點。這種圓形風格受到古巴兒子的啟發，特別是20世紀20年代蒙托諾兒子的節奏。然而，由於它是一種流行的音樂，所以它對即興創作是開放的，因而不斷演變。現代莎莎風格與發展它們的原始地理區域相關聯並命名。在他們的家鄉以外，這些風格往往都是奉獻者。可以識別風格的特徵包括：時間，基本步驟，腳型，身體動作，輪次和數字，態度，舞蹈影響以及合作夥伴之間相互擁抱的方式。在音樂酒吧音樂中，採取稍微大一點的步驟（中斷步驟）以及步驟移動的方向通常可以用於識別風格。

==发展趋势==

薩爾薩舞蹈社交活動通常在夜總會，酒吧，宴會廳，餐廳和室外舉行，尤其是如果是戶外活動的一部分。 莎莎跳舞是一種國際舞蹈，可以在世界上大多數都市中找到。<ref> {site web |url =http：//www.salsadancecongress.com/ |title =莎莎和拉丁舞蹈大會|出版商= Salsa舞蹈協會 .com |accessdate = 2013-12-04}} </ref>節日每年舉行一次，通常稱為薩爾薩大會，在各個主辦城市旨在吸引來自其他城市和國家的各種莎莎舞者。 這些活動讓舞者們共同分享他們對舞蹈的激情，建立社區，並分享彼此的動作和技巧。 這些活動通常包括薩爾薩舞表演，現場莎莎音樂，工作坊，開放舞蹈和比賽。
