'''莎瑪·希恩·赫梅內斯'''（{{lang-es|Salma Hayek Jiménez}}，{{bd|1966年|9月2日|||catIdx=Hayek}}），[[墨西哥|墨西哥]]-[[美國|美國]]女[[演員|演員]]、[[導演|導演]]、影視製作人。她至今共出演、執導過30部[[電影|電影]]、[[電視節目|電視節目]]，著作有：《[[戀愛心曲|戀愛心曲]]》、《[[揮灑烈愛|揮灑烈愛]]》、《[[毒網|毒網]]》、《[[週六夜現場|週六夜現場]]》、《[[丑女贝蒂|丑女贝蒂]]》等。

莎瑪·希恩是首個被提名[[奧斯卡最佳女主角獎|奧斯卡最佳女主角獎]]的[[墨西哥人|墨西哥人]]，亦是第二個被提名這獎的[[拉丁美洲|拉丁美洲]]人。

== 生平 ==
莎瑪·希恩在1966年於[[墨西哥|墨西哥]][[韋拉克魯斯州|韋拉克魯斯州]][[夸察夸爾科斯|夸察夸爾科斯]]出生。父親是油公司的總裁，母親則是歌劇演唱者。有一弟Sami，是家具設計師。

== 演艺生涯 ==
早期是在[[墨西哥|墨西哥]]參演連續劇，開始嶄露頭角。1991年移居[[美國|美國]][[洛杉磯|洛杉磯]]，開始參演電影，但並沒有因此而大紅。

== 事业反弹 ==
2002年，海耶克终于等来了一部似乎是专门为她量身定做的剧本，在人物传记片《[[揮灑烈愛|揮灑烈愛]]》（Frida）中成功扮演了墨西哥超现实主义女画家[[芙烈達·卡蘿|芙烈達·卡蘿]]（Frida Kahlo）。

== 婚姻生活 ==
2007年3月，希恩宣布与法国亿万富翁、全球三大奢侈品集团之一的PPR总裁[[弗朗索瓦-亨利·皮诺|弗朗索瓦-亨利·皮诺]]订婚，同时对外公开怀孕的消息。同年9月，希恩诞下千金，取名为瓦伦蒂娜·帕洛玛·海耶克·皮诺（Valentina Paloma Hayek Pinault）。两人的发言人却分别对外宣布取消婚约。不过，经历了情变的两人于情人节当日在巴黎“秘密完婚”，不过还是在意大利威尼斯的凤凰剧院补办豪华婚礼。
