'''土客冲突'''专指[[明朝|明]][[清朝|清]]时期，在[[中国南方|中国南方]]的部分地区，本地人与客家人之间的激烈械斗冲突，其最高峰是[[清朝|清朝]]末年发生在[[广东|广东]]的土客械斗。土、客二字，土是指[[珠三角|珠三角]]本地人即[[廣府人|廣府人]]，客是指[[客家人|客家人]]。土客冲突的双方在不同地点均并不完全相同，例如广西的土客冲突，土方为壮族及当地的汉族人，客方为客家等民系。江西的土客冲突，土方为江西本地人，客方为客家人。

== 發生時間及地點 ==
土客冲突发生的地点很多，{{fact|在[[台灣|台灣]]、[[海南|海南]]、[[广东|广东]]、[[广西|广西]]、[[福建|福建]]、[[江西|江西]]、[[湖南|湖南]]的许多地方都曾经发生过|time=2009-12-30}}。土客冲突发生的时间跨越度也很长，从[[明朝|明朝]]中期到清末均有发生，但绝大多数土客冲突均发生在清末（十九世纪四十年代-十九世纪末）。

=== 衝突的成因 ===
土客衝突给当地人民，无论是土方还是客方，都造成了深重的灾难，归究其原因，不同研究者的说法都各异，但普遍接受的一点是：土客衝突的本质，在于争夺生存[[资源|资源]]；而其极端形式—土客械斗，则是乡村争夺资源的一个极端形式。此外，当时政府的挑动，以及地主土地制度的限制等，也是土客衝突的重要原因。土客冲突自明朝以前就在中国南方不时出现，但都规模不大。

但到了清末，清朝政府為要徹底清算消滅[[太平军|太平军]]等反清势力。同時，广东沿海等地的人口剧增（各地移民及自然人口增长），人民生活困苦，生存[[资源|资源]]处于极度紧张的状态。因為当时政治、经济都非常急剧变动，因此惨烈的土客械斗发生就在这種背景下。其中尤以[[四邑|四邑]]一帶的土客械鬥最為慘烈。

== 冲突过程 ==
由于清廷长期以来的[[一口通商|一口通商]]政策，促使广东经济以及人口快速增长，但[[第一次鸦片战争|第一次鸦片战争]]后签订的[[南京条约|南京条约]]因开放[[五口通商|五口通商]]，使得广东位置下降。原来累积的人口由经济动力变成经济压力。最终酝酿而成土客冲突。

其中客家方面据山为险，扼踞险要位置。加上筑有[[土楼|土楼]]等高防御设施，使得防御力战斗力并不低。而本地广府则因经济充裕，长期贸易使得广府占据着巨大资源，并且雇佣了欧洲[[雇佣军|雇佣军]]作战。促使双方战斗力平衡。由于客家人在战斗力据有上风，进而引发[[太平天国起义|太平天国起义]]。而广府雇佣欧洲雇佣军，也促使英法等欧洲国家扩张其贸易范围，进而引发[[第二次鸦片战争|第二次鸦片战争]]。

=== 四邑地区的土客械斗 ===

广东多地都发生过土客械斗，其中尤以“四邑地区”的土客械斗，最为惨烈，死伤最为惨重。[[四邑|四邑]]地区处于广东[[珠江三角洲|珠江三角洲]]西部，原本是本地人世代居住之地，明清年间，来自粤东地区的客家人涌入到四邑地区。在初期，由于涌入人口还不多，生存资源足够需求，一直相安无事，但慢慢随着客家人涌入越来越多，双方开始发生不同规模的衝突。發展到晚清[[咸丰_(年号)|咸丰]]、[[同治|同治]]年间，双方終於爆发了大規模的武裝械鬥（註：「土」於此例中指四邑本地廣府人）。最后客家人戰敗，大部分被赶回粤东梅州一帶的原居地。

== 後續影响 ==

=== 客家称谓的形成 ===

土客冲突的双方在不同地点均并不完全相同，例如广西的土客冲突，土方为[[壮族|壮族]]及当地的[[汉族|汉族]]人，客方为客家等民系。江西的土客冲突，土方为江西本地人，客方为客家人。而在四邑地区土客大械斗的双方之一的客家人，其原籍均来自[[赣南|赣南]]、[[闽西|闽西]]、[[粤东|粤东]]、[[粤北|粤北]]，使用客家话，有着相同的习俗。作为土客冲突的客方，他们处于少数，因此在土客冲突最激烈的几十年间，即清末土客械斗期间，这些移民到外地的人与原籍地的人相互之间有着强烈的民系认同，最终的结果是这些人普遍接受以'''客家'''作为民系通用的名字，这也就代表着'''[[客家民系|客家民系]]'''的正式形成。

== 學術研究及報告 ==
在中国，受政府意识形态的限制，土客冲突从清末到二十世纪末均未受到学者的重视，近年来才有《被遗忘的战争》等学术作品问世。
