== 位置 ==
[[图像:Tuerqi-ditu-zh.png|thumb]]

土耳其處於[[亞洲|亞洲]]與[[歐洲|歐洲]]連接處的重要位置，控制着[[黑海|黑海]]的入口，總面積達780576平方公里，略小于[[巴基斯坦|巴基斯坦]]，而稍大于[[中国|中国]][[青海省|青海省]]。北临黑海，南临[[地中海|地中海]]，东南与[[叙利亚|叙利亚]]、[[伊拉克|伊拉克]]接壤，西临[[爱琴海|爱琴海]]，并与[[希腊|希腊]]以及[[保加利亚|保加利亚]]接壤，东部与[[格鲁吉亚|格鲁吉亚]]、[[亚美尼亚|亚美尼亚]]、[[阿塞拜疆|阿塞拜疆]]和[[伊朗|伊朗]]接壤。

土耳其的西北边是[[色雷斯|东色雷斯]]（即[[巴尔干半岛|东南欧洲]]），剩余的部分则算是[[小亚细亚|小亚细亚]]（又称[[安那托利亚|安那托利亚]]）。安那托利亚[[半岛|半岛]]形状大致呈长方形，占土耳其总面积97％。色雷斯所占面积虽不超过3％，却包含土耳其人口10％以上，其中大部分住在[[伊斯坦堡|伊斯坦堡]]。色雷斯与小亚细亚被[[博斯普鲁斯海峡|博斯普鲁斯海峡]]、[[達達尼爾海峽|達達尼爾海峽]]和[[马尔马拉海|马尔马拉海]]分开，隔黑海与[[罗马尼亚|罗马尼亚]]、[[乌克兰|乌克兰]]、[[俄罗斯|俄罗斯]]和[[格鲁吉亚|格鲁吉亚]]相望。

== 气候 ==
沿海為[[地中海型氣候|地中海型氣候]]，年降水量600～1200毫米。内地向亚热带草原、沙漠气候过渡，为[[副热带干旱半干旱气候|副热带干旱半干旱气候]]，年降水量200～400毫米。

== 地形水文 ==
[[图像:Turkey_topo.jpg|thumb]]
三分之二的小亚细亚為[[高原|高原]]地形。最大的高原是[[安納陀利亞高原|安納陀利亞高原]]，高原的海拔高度最低1524米，最高1981米，地勢东高西低。

[[亞拉拉特山|亞拉拉特山]]的[[海拔|海拔]]高达5165米，是土耳其最高峰。

[[幼发拉底河|幼发拉底河]]与[[底格里斯河|底格里斯河]]从境内穿过，向東南流去，為沿岸[[农业|农业]]和[[城市|城市]]提供水源。

== 自然灾害 ==
土耳其地處[[地中海-喜馬拉雅地震帶|地中海-喜馬拉雅地震帶]]，是世界上[[地震|地震]]最頻繁的地區之一，北部马尔马拉海到[[凡湖|凡湖]]一带的地震最多。1999年8月17日土耳其发生7.4[[里氏地震规模|级]]地震，造成1.7万多人死亡，4万多人受伤，直接经济损失约200亿[[美元|美元]]。
