'''土卫九'''又稱為「菲比」（Phoebe），是环绕[[土星|土星]]运行的一颗[[卫星|卫星]]。

== 概述 ==

1898年，[[威廉·皮克林|威廉·皮克林]]发现土卫九。土卫九是使用照相技术发现的第一颗卫星。土卫九是土星距离较远的卫星之一，平均距离土星1295.2万千米。土卫九直径只有大约220千米，在轨道上逆向环绕土星运行。

== 基本参数 ==

* 发现者：[[威廉·皮克林|威廉·皮克林]]
* 平均公转轨道半径：12,952,000千米
* 公转周期：550.48天
* 自转周期: 0.4天
* 半径：110千米
* 逃逸速度：0.0697千米/秒
* 目视星等：16.45星等

== 表面特徵 ==
表面的反射率非常的低，和[[海衛八|海衛八]]一樣屬於十分黑暗的衛星，僅只反射了6%的太陽光。

平均直徑220公里，是[[諾爾斯衛星群|諾爾斯衛星群]]中最大的一顆[[衛星|衛星]]，像是歪斜的[[馬鈴薯|馬鈴薯]]外觀表面佈滿了大大小小的撞擊坑。
