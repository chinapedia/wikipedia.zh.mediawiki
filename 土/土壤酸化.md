'''土壤酸化''' ，是[[氫|氫]] [[陽離子|陽離子]]的積累，也稱為[[質子|質子]]，減少土壤pH。當質子供體加入土壤時，就會發生這種情況。給體可以是[[酸|酸]]，例如[[硝酸|硝酸]]和[[硫酸|硫酸]]（這些酸是[[酸雨|酸雨]]的常見組分）。它也可以是諸如[[硫酸鋁|硫酸鋁]]的化合物，其在土壤中反應以釋放質子。作為[[肥料|肥料]]添加的許多[[氮|氮]]化合物也長期酸化土壤，因為它們在該過程中被氧化時產生[[亞硝酸|亞硝酸]]和[[硝酸|硝酸]]。

當[[鈣|鈣]]，[[鎂|鎂]]，[[鉀|鉀]]和[[鈉|鈉]]的鹼性陽離子從土壤中浸出時，也會發生酸化。這種浸出隨著[[降水|降雨]]增加，[[酸雨|酸雨]]也會加速鹼性陽離子浸出。當植物生長時，植物從土壤中取出鹼，為每個鹼基[[陽離子|陽離子]]交換質子。當除去植物材料時，例如當森林被砍伐或作物收穫時，它們所佔據的基質永久地從土壤中喪失。

== 植物留在土壤上 ==
許多植物產生有機酸。當植物垃圾在土壤上積聚或結合到土壤中時，釋放這些酸（包括[[乙酸|乙酸]]、[[腐殖酸|腐殖酸]]、[[草酸|草酸]]和[[鞣酸|鞣酸]]）。這在[[針葉樹|針葉樹]]如[[松樹|松樹]]、[[雲杉|雲杉]]和[[冷杉|冷杉]]的土壤中尤其嚴重。

== 岩石在土壤中 ==
某些岩石也有助於土壤酸化。[[花崗岩|花崗岩]]和[[火成岩|火成岩]]他們有大量的[[石英|石英]]對抗風化。此外，它們具有相對低量的鈣和鎂。一些[[沉積|沉積]]岩石如[[頁岩|頁岩]]和[[煤|煤]]富含[[硫化物|硫化物]]，當水合和氧化時，產生比矽酸強得多的硫酸。海洋[[粘土|粘土]]在許多情況下也是富含硫的，並且如果這些粘土排出到氧化狀態，則其變得非常酸性。

== 污染 ==
酸化也可以從氮氣排放到空氣中發生，因為氮氣可能最終沉積到土壤中。<ref>USGS. [http://www.usgs.gov/newsroom/article.asp?ID=2075&from=rss_home Acid Soils in Slovakia Tell Somber Tale].</ref>
