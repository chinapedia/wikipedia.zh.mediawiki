'''土豆'''（前稱：'''土豆網'''）是[[阿里巴巴集團|阿里巴巴集團]]旗下[[合一集團|合一集團]]的[[影片分享網站|影片分享網站]]，在[[中國大陸|中國大陸]]提供的视频總計超過5500万，每天独立用户数超过2500万，注册用户超过8000万，用戶流量每月2亿<ref name="Shanghaiist_interview">{{cite web|url=http://shanghaiist.com/2007/10/11/tudou_now_bigge.php|title=Tudou now bigger than Youtube?|publisher=Shanghaiist|date=2007-10-11|language=en}}</ref>。

土豆成立的口號為“每个人都是生活的导演”，並聲稱土豆网相信年轻人的想象力、创造力，相信土豆的平台能帮助年轻人创造出更多，更好的影像作品，来表达对自己、生活和这个时代的看法。“让富有创造力的节目的创造者和分享者们能够自由地让自己的节目在用户面前出现，同时，也让每一个用户随时随地都能看到自己想看到的任何节目。”在集團的品牌定位中偏向年輕化與綜藝、動畫，形成與[[優酷網|優酷網]]不同的差異化內容。

2017年3月31日，在阿里巴巴召开的“阿里短视频战略暨新土豆发布会”上，阿里宣布土豆将转型成为短视频分发平台，公司将推出超过20亿补贴计划扶持短视频内容创作。<ref >{{cite web|url=http://news.mtime.com/2017/04/01/1567862.html | language= | title=阿里动用20亿“重启”土豆 |publisher=时光网 |date=2017-04-01 |accessdate= }}</ref>

== 歷史 ==
2005年4月15日，土豆网正式上线。<ref>{{Cite web|url=http://www.xinhuanet.com/zgjx/2010-08/13/c_13442924.htm|title=土豆网完成视频网站最大规模融资|accessdate=2018-05-27|work=www.xinhuanet.com}}</ref>

2009年，[[香港|香港]]商營電視台[[電視廣播有限公司|无线电视]]與土豆網簽署版權協議，要求土豆網刪除所有TVB的節目，尤其是近期播放的[[電視劇|電視劇]]、資訊節目。然而土豆網的刪除行為受到網民批評，現時土豆網以[[馬賽克_(影像處理)|馬賽克遮擋]]有關[[logo|logo]]作為措施，但若遇到投訴時仍作刪檔處理。

2009年1月，土豆网屏蔽了來自[[日本|日本]]的[[IP|IP]]<ref>{{cite web|url=http://www.bianews.com/news/47/n-58847.html|title="土豆网、优酷网屏蔽日本IP 或为规避版权风险|author=孟鸿|date=2009年1月5日|publisher=腾讯科技|deadurl=yes|archiveurl=https://web.archive.org/web/20101208040714/http://www.bianews.com/news/47/n-58847.html|archivedate=2010年12月8日|df=}}</ref>。4月18日，使用期长达3年的[[化妝舞会|化妝舞会]][[面具|面具]]Logo被笑脸面具所取代。同年11月，土豆網與[[中國電影集團公司|中國電影集團公司]]再次宣布合作方案，打算[[投資|投資]]土豆網中的[[播客|播客]]所自製的原創短劇，並同步使用[[手機|手機]]獨家首播，供土豆網用戶付費下載和點播<ref>{{cite web|url=http://tavis.tw/files/15-1000-10983,c1013-1.php|title=中國電影新模式－全土豆計劃|publisher=《國家影視產業資訊平臺》（TAVIS.tw）|date=2009-11-24|language=zh-tw}}</ref>。

2010年2月，土豆網撤下了大量已发布的涉嫌[[版權|版權]]爭議的视频<ref>{{cite web|url=http://ent.sina.com.cn/c/2010-02-05/16092869068.shtml|title= 优酷土豆宣布彻底删除盗版视频 将巨资购买正版|accessdate=2010年7月13日|author=徐力|date=2010年2月5日|publisher=现代快报}}</ref>。

2011年5月25日，土豆网推出原画平台，用户可以不受限制的上传任何清晰度的视频。如果某视频有原画版本，播放时可点击清晰度按钮选择原画版本<ref>[http://www.tudou.com/service/?type=originalquality 土豆帮助系统 - “原画”是什么？]</ref>。

2011年8月17日，土豆网在[[纳斯达克|纳斯达克]]上市<ref>{{Cite web|url=http://www.yicai.com/show_topic/1019497/|title=土豆网赴纳斯达克上市|accessdate=2018-05-27|work=www.yicai.com}}</ref>，发行价为29美元/股，融资1.74亿美元。同年8月31日[[新浪|新浪]]向土豆網投資6640萬美元，持股比例達9.05%，新浪成為土豆第五大股東，王微持股比例從12.7%下降至8.6%，但是，王微仍然擁有土豆最多的投票權。

2011年11月24日，日本[[東京電視台|東京電視台]]社長在例行記者會宣布，授權土豆網《火影忍者 疾風傳》等熱門動畫的播放权<ref>{{cite web|url=http://www.tv-tokyo.co.jp/contents/ir/jpn/getsurei/201111.html|script-title=ja:島田社長11月定例会見|author=|date=2011-11-24|5=|accessdate=2012-01-14|publisher={{lang|ja|テレビ東京}}|language=ja|deadurl=yes|archiveurl=https://web.archive.org/web/20120102121224/http://www.tv-tokyo.co.jp/contents/ir/jpn/getsurei/201111.html|archivedate=2012-01-02}}</ref><ref>{{Cite web |url=http://www.tv-tokyo.co.jp/contents/ir/jpn/press/pdf/111124.pdf |script-title=ja:テレビ東京が『NARUTO』ほか人気アニメを中国で即日配信開始！ |date=2011-11-24 |4= |accessdate=2012-01-14 |format=[[PDF|PDF]] |work= |publisher={{lang|ja|テレビ東京}} |pages= |language=Ja |deadurl=yes |archiveurl=https://web.archive.org/web/20120112100133/http://www.tv-tokyo.co.jp/contents/ir/jpn/press/pdf/111124.pdf |archivedate=2012-01-12 }}</ref><ref>{{cite web|url=http://china.kyodonews.jp/news/2011/11/20511.html|title= 东京电视台将向中国即时发布动画片|author=|date=2011-11-24||accessdate=2012-01-14|publisher= 共同网|language=zh-cn}}</ref>。同年11月28日，土豆网正式宣佈与东京电视台签独家协议、為期兩年<ref>{{cite web|url=http://www.tudou.com/groups/posts1962082p1.html |title= 土豆网与东京电视台签独家协议 同播日动画片|author=|date=2011-11-28||accessdate=2012-01-15|publisher= 土豆网|language=zh-cn}}</ref>，目前引進70多部動畫作品<ref>{{cite web|url=http://t.qq.com/k/%E5%9C%9F%E8%B1%86%E6%97%A5%E6%BC%AB%E7%8B%AC%E6%92%AD|title= 发微博看日漫赢Q币|author=|date=2012-01-13||accessdate=2012-01-15|publisher= 騰訊微博|language=zh-cn}}</ref>；12月22日，東京電視台7部熱門動畫在土豆網[[海外聯播動畫|即日播出]]<ref>{{Cite web |url=http://www.tv-tokyo.co.jp/contents/ir/jpn/press/pdf/111222.pdf |script-title=ja:中国における人気アニメの即日配信7作品で実施中！ |date=2011-12-22 |4= |accessdate=2012-01-14 |format=[[PDF|PDF]] |work= |publisher={{lang|ja|テレビ東京}} |pages= |language=Ja |deadurl=yes |archiveurl=https://web.archive.org/web/20160304121906/http://www.tv-tokyo.co.jp/contents/ir/jpn/press/pdf/111222.pdf |archivedate=2016-03-04 }}</ref>，過去作品和集數使用[[字幕組|字幕組]]的片源。

2012年3月12日，[[优酷|优酷]]与土豆宣布以换股方式合并，新公司将成立为[[合一集团|优酷土豆集团（优酷土豆股份有限公司）]]<ref>{{cite web|title=优酷与土豆宣布以换股方式合并|date=2012-03-12 |url=http://tech.sina.com.cn/i/2012-03-12/17036827996.shtml}}</ref>。同年9月，土豆网启用了新服务——豆泡，一款类似于[[NICONICO|NICONICO]]的弹幕视频播放器接入门户。采用[[Flash|Flash]]技术的嵌入式播放器，其最主要的功能是可以实现高级弹幕效果。将土豆网任意视频播放页面[[网址|网址]]中的“www”替换为“dp”（“豆泡”的汉语拼音首字母）即会进入该视频的豆泡播放器界面。主要功能包括：
* 弹幕发送：允许用户在通过网络观看视频时，在评论框中输入自己的评论，提交后评论会显示在视频中从屏幕中划过。网友可以通过这种方式[[吐槽|吐槽]]，并且也可以看到其他用户发送的评论。
* 微博发送：用户可将评论直接发送至[[新浪微博|新浪微博]]、[[腾讯微博|腾讯微博]]，直接生成微博内容。并且在播放器的右侧微博栏中会显示不同网友发送的关于此视频的微博，这样增加了网友的微博互动，促进形成以个以视频为中心进行交流和互动的圈子。
* 一键分享：用户可以选择不同的网络进行视频分享，如[[人人网|人人网]]，[[天涯论坛|天涯论坛]]，[[QQ空间|QQ空间]]等。
* 广告吐槽：允许用户在等待正片播放观看广告时，对广告进行吐槽。

=== 限制境外观看 ===
2011年8月开始至今，土豆网同[[优酷网|优酷网]]、[[爱奇艺|爱奇艺]]等中国视频网站一样，只允许[[中国大陆|中国大陆]]的[[IP地址|IP地址]]访问，[[港澳台|港澳台]]及海外IP均无法播放视频内容<ref>{{cite web|url=http://news.hk.msn.com/local/article.aspx?cp-documentid=5397493 |title=土豆網禁港人看 |accessdate= |author= |date=2011年8月28日 |publisher=明報 via MSN新聞 |archiveurl=https://web.archive.org/web/20131029195054/http://news.hk.msn.com/local/article.aspx?cp-documentid=5397493 |archivedate=2013年10月29日 |deadurl=yes }}</ref>，一度引起境外用户的不满。土豆网声称是因为版权问题而拒绝境外IP访问<ref>{{cite web|url=http://blog.ifeng.com/article/15749994.html|title=为看大陆电视剧 港台人民学“翻墙”|date=2012-01-05|publisher=鳳凰博報|deadurl=yes|archiveurl=https://web.archive.org/web/20120109085706/http://blog.ifeng.com/article/15749994.html|archivedate=2012-01-09}}</ref>，致使境外用戶不得使用大陸的代理或各種方法觀看影片，俗稱「翻牆進大陸」或「逆翻牆」<ref>{{cite web|url=https://www.google.com.hk/search?safe=off&q=翻牆看土豆&cad=h|title="翻牆看土豆"的相關Google搜尋結果|accessdate=2015-06-15}}</ref>。
