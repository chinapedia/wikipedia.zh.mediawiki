'''土耳其政治'''是以[[議會制|議會制]][[代議民主制|代議民主制]]的[[共和制|共和制]]為框架，[[土耳其總理列表|土耳其總理]]是政府的領導人，奉行多黨制。行政權由政府行使，立法權則屬於政府及[[土耳其大國民議會|土耳其大國民議會]]，司法則獨立於行政及立法。現行的憲法在1982年11月7日起實施，憲法奉行[[世俗主義|世俗主義]]。

2010年土耳其修憲公投通過了新的憲法，限制了軍隊的權力。<ref>{{Cite web|url=http://www.appledaily.com.tw/appledaily/article/headline/20100914/32811034 |title=司馬觀點：土耳其的修憲公投（江春男） |accessdate=2017-03-16 |author= |date= |work=蘋果日報 |publisher= |language=zh-TW |deadurl=no |archiveurl=https://archive.is/20120717021224/http://www.appledaily.com.tw/appledaily/article/headline/20100914/32811034 |archivedate=2012-07-17 }}</ref>土耳其的政治體制以分權為原則。

[[2017年土耳其修憲公投|2017年土耳其修憲公投]]以微弱优势通过，土耳其转变为[[总统制|总统制]]国家，总理一职也会废除。

== 行政部門 ==

國家首領稱為[[土耳其總統列表|總統]]。總統任期七年（2014年後總統任期由七年縮短至五年），由國民議會選舉產生。總統毋須是議會的議員。

現任總統[[雷傑普·塔伊普·埃爾多安|雷傑普·塔伊普·埃爾多安]]是在2014年8月28日由國會選舉產生的。行政權掌握在[[土耳其總理列表|總理]]及部長會議手裡。部長毋須是議會的議員（如凱莫爾·德維斯）。

總理由議會投信任票選舉產生，現任總理是[[艾哈迈德·达武特奥卢|艾哈迈德·达武特奥卢]]，他接任[[雷傑普·塔伊普·埃爾多安|雷傑普·塔伊普·埃爾多安]]，成为[[土耳其總理列表|總理]]。

議會主席比倫特·阿林克也是來自正義與發展黨。現任憲法法院院長是圖拉依·圖居，她是歷史上首位女性憲法法院院長。現時的土耳其軍事參謀長是亞薩爾·布余卡尼。

== 立法部門 ==
擁有550個議席的土耳其[[土耳其大国民议会|大國民議會]]掌有立法權，他們分別代表81個省份。議會議員每五年一任在2017年的修憲公投後，國會將有以下的權利:
* 制定，修訂，廢止[[法律|法律]]
* 接受國際協議
* 探討，增加或削減國家預算
* 對土耳其法官檢查官最高理事會中的7名成員進行任命
* 以及行使憲法賦予的其他權力

== 各級選舉系統 ==
各類選舉所用選舉制度的簡要概述如下：
* '''國會选举：'''採用[[名單比例代表制|封閉式政黨比例代表制]] ，550名议会议员大国民议会从85 个选区中選出 ，每個選區的選出的議員數量多寡取決與該選區的人口。惟黨派需獲得全國至少10%的選票才可獲得議席。獨立侯選人只須獲得其省份的10%選票便可當選。
* '''地方选举：''' 大都市和地區市長，市議員和省議員、[[ 村長| 村長]]和他们的[[村委会|村委会]]选举採用[[多數制|多數制]]選舉 ，候选人只要在其選區獲得相對多数的選票及勝出。
* '''总统选举：''' 採用 [[兩輪選舉制|兩輪選舉制]]，候選人在第一輪投票中沒有獲得絕對多數票；那麼得票數最多的兩名候選人，將在兩星期後的第二輪選舉中再次競爭。

== 重要的政治主張 ==

土耳其憲法及較活躍的政黨建基於以下的政治主張：

*[[政教分離|政教分離]]（Laicism）
*[[現代化|現代化]]
*土耳其[[民族主義|民族主義]]

其他的政治主張亦影響土耳其政治及其近代歷史，包括：

*[[資本主義|資本主義]]
*[[共產主義|共產主義]]
*[[伊斯蘭原教旨主義|伊斯蘭原教旨主義]]（Islamic fundamentalism）
*[[庫爾德人分離主義|庫爾德人分離主義]]（Kurdish separatism） 
*[[泛突厥主義|泛突厥主義]]
*[[社會主義|社會主義]]

== 政黨及選舉 ==
自1950年始，議會一直受到保守黨派把持。現時的執政黨[[正義與發展黨_(土耳其)|正義與發展黨]]，其核心都是穆斯林，他們傾向於把政黨定位為傳統民主黨派。而中左翼世俗黨派，最有名的是[[共和人民黨_(土耳其)|共和人民黨]]，該黨有穩定的支持者，大多來自大城市、沿岸地區、中產人士，還有少數是[[什葉派|什葉派]]教徒及[[庫爾德人|庫爾德人]]。

== 司法部門 ==
{{Main|土耳其司法制度}}

土耳其司法制度的自由及獨立受到憲法的保障。沒有任何組織、個人、機關可以干預法院的運作，行政和立法部門都必須遵從法院的裁決。法院獨立運作，其裁決必須根據憲法、法律及個人判決而來。

土耳其司法制度結構嚴謹。法院沒有陪審團制度，判決基於以律師及檢控官提出的證據確立事實後決定。小型的民事訴訟和案件可由太平紳士審理，這法院只有一名法官負責審理，對於輕罪及瑣碎案件有審理權，刑罰規限在小量罰款至短期監禁。由三名法官審理的法院對民事訴訟及嚴重罪案有裁判權。任何有罪判決都可以轉至上訴庭再審。

所有法院均對外公開。當案件需要閉門聆訊時，必須要對外解釋原因。判決和起訴制度受到憲法保障。除了自願性質外，裁判及檢控官也不會被解雇、強制退休及限制權力，但退休年齡限制仍適用。兒童法院則另有制度。

只有在司法部門的允許及法官不端正行為的情況下，法官才可被查核，在這個情況下，法律專家及高級法官會組成特別組織進行調查。

最高法官及檢控官議會是確保司法公正的主要機構，該議會還有考核專業法官及分配法院工作的權責。土耳其總理雷傑普·塔伊普·埃爾多安是該議會的主席。

土耳其現時採用國家司法電子系統，法院裁決及文件可以在網上查閱。

土耳其接受[[歐洲人權法院|歐洲人權法院]]的判決，亦同意受國際協議的約束。

== 軍事因素 ==

自1923年[[穆斯塔法·凱末爾·阿塔土克|穆斯塔法·凱末爾·阿塔土克]]建立現代的土耳其共和國後，雖然阿塔土克一直堅持軍政分離，但軍隊一直都自視為[[凱末爾主義|凱末爾主義]]（土耳其民族主義）的守護者。[[土耳其武裝部隊|土耳其武裝部隊]]對土耳其的政局及決策有一定程度的影響力，特別是國家防務方面，不過[[土耳其國家安全委員會|國家安全委員會]]的成立削弱了軍隊對此的影響。

軍方的政治角色較為通俗，他們捍衛[[世俗主義|世俗主義]]和[[共和制|共和制]]，並認為反世俗主義及擁護[[分離主義|分離主義]]的政黨可以透過憲法法院被取締

土耳其軍隊曾經干預政治，軍隊曾在1960年、1971年及1980年發動政變。1997年，軍隊策劃撤除親伊斯蘭總理[[内吉梅丁·埃尔巴坎|內吉梅丁·厄巴坎]]的職務<ref>{{Cite web|url=https://www.cia.gov/library/publications/the-world-factbook/geos/tu.html|title=The World Factbook — Central Intelligence Agency|accessdate=2017-03-16|work=www.cia.gov|language=en}}</ref>。

2007年4月27日，正值11月4日總統選舉的前夕，軍隊發表聲明，声称自己也是关于世俗主義“讨论”中的一方，這一聲明使人猜測軍隊即將會介入政治。聲明中提到：「土耳其武裝部隊會繼續履行他們的職務，他們對土耳其共和國的角色不會改變，他們對國家是絕對忠誠的。」<ref>{{Cite web|url=http://news.bbc.co.uk/2/hi/europe/6602775.stm|title=Excerpts of Turkish army statement|accessdate=2017-03-16|author=|date=|work=news.bbc.co.uk|publisher=BBC NEWS}} </ref><ref>{{Cite web|url=http://news.bbc.co.uk/2/hi/europe/6602375.stm|title=Army 'concerned' by Turkey vote|accessdate=2017-03-16|author=|date=|work=news.bbc.co.uk|publisher=BBC NEWS}}</ref>但[[2016年土耳其政變|2016年土耳其政變]]發生後顯見軍中有大量受外部勢力操控的單位。

而[[2016年土耳其政變|2016年土耳其政變]]未遂後與西方國家關係開始惡化，其認為歐美國家有嚴重嫌疑參與政變，之後反IS戰爭中西方支持庫德族武裝，連帶敲動了土境內的庫德族分裂勢力，雙方開始走遠，2017年9月底土宣布向俄羅斯購買[[S-400飛彈|S-400]]防空飛彈引起國際嘩然，因為北約的目的就是對抗[[俄羅斯|俄羅斯]]，而現在有北約國家開始與俄軍事合作，<ref>[http://tv.cctv.com/2017/09/29/VIDExXQQbOD6bt7WC5KCvMSB170929.shtml 央視-俄土走近]</ref>更擔心飛彈安裝的技術交流中俄方技術員可能得以接觸北約的軍事裝備與科技。
