'''土耳其航空5904號班機'''是一個由[[波音737|波音737-400]]執行的不載客調機航班，由[[阿达纳|阿达纳]]飛往[[吉达|吉达]]。1999年4月7日，這架編號為TC-JEP的波音737在土耳其南部[[阿达纳省|阿达纳省]]的[[傑伊漢|傑伊漢]]墜毀，機上6名機組人員全部死亡。

== 事发经过 ==
[[歐洲東部時間|歐洲東部時間]]當天凌晨0時36分，TK5904載著兩名飛行員和四名乘務員從阿達納起飛，前往吉達接載朝聖者，但在起飛8分鐘后墜毀於阿達納機場東北約{{convert|30|nmi|km}}處的 罕迪利村。<ref name="h">{{cite news |url=http://webarsiv.hurriyet.com.tr/1999/04/07/107253.asp |publisher=Hürriyet |title=THY uçağı Adana'da düştü |date=1999-04-07 |language=tr |accessdate=2009-01-04}}</ref><ref name="asn">{{cite web |url=http://aviation-safety.net/database/record.php?id=19990407-0 |publisher=Air Safety Network |title=Aircraft accident Boeing 737-4Q8 TC-JEP Ceyhan |accessdate=2009-01-04 }}</ref>飛機的撞擊使地面產生深達{{convert|10|m|ft}}的大坑，殘骸散落在周圍{{convert|42|m2|sqft}}的區域內。<ref name="asn" />

該機之前將150名[[朝觐_(伊斯兰教)|朝覲者]]接回阿達納，23時57分降落。<ref name="h" />飛行員稱沒有維修上的問題，降落後一切正常。

飛行記錄儀顯示，飛行員於23時22分43秒向附近{{link-en|因斯里克空军基地|Incirlik Air Base}}的航管員詢問其在雷達上發現的雷暴方向及速度。航管報告雷暴北移，而距機場西側以北{{convert|20|nmi|km}}的天氣晴朗。<ref name="shgm">{{cite web |url=http://www.shgm.gov.tr/kaza/tcjep.htm |publisher=Civil Aviation Authority |title=Accident report TC – JEP (07.04.1999) |language=tr |accessdate=2009-01-04 |deadurl=yes |archiveurl=https://web.archive.org/web/20080415001803/http://www.shgm.gov.tr/kaza/tcjep.htm |archivedate=2008年4月15日 }} </ref>

== 失事客機 ==

失事客機是一架[[波音737|波音737-4Q8]]，使用兩台[[CFM56|CFM56-3C1]]發動機，波音公司生產序列號25378/2732，1995年6月9日首飛。<ref name="asn" />

== 事發原因 ==
{{link-en|土耳其民用航空總局|Directorate General of Civil Aviation (Turkey)}}對此次空難的報告顯示，飛機墜毀原因如下：
{{quotation|

# 惡劣天氣可能造成了此次空難。
# 航前準備時可能沒有啟動[[皮托静压系统|皮托静压系统]]。
# 機組人員未能認識錯誤的空速指示。
# 機組未能藉助其他飛行儀表控制飛機。
# 乘務員在駕駛艙內可能分散了機師的注意力。<ref name="shgm" /><ref>{{cite web |url=http://www.ntsb.gov/ntsb/brief2.asp?ev_id=20001205X00446&ntsbno=DCA99RA053&akey=1 |publisher=NTSB |title=Identification: DCA99RA053 |accessdate=2009-01-04 }}</ref>
