'''土耳其航空1951號班機空難'''發生於2009年2月25日約9時31分（以下所提的時間均為UTC+2時區），一架註冊碼為'''TC-JGE'''、從[[土耳其|土耳其]][[伊斯坦堡|伊斯坦堡]]出發飛往荷蘭首都[[阿姆斯特丹|阿姆斯特丹]]的[[土耳其航空|土耳其航空]]1951號定期航班[[波音737-800|波音737-800]]客機。在下降过程中，由于左侧高度计故障<ref name=CNN4Mar>

，在阿姆斯特丹效區的[[史基浦機場|史基浦機場]]墜毀，機身分成三部分，沒有造成火災。<ref>http://avherald.com/h?article=41595ec3&opt=0</ref>飛機墜毀處離跑道約3[[公里|公里]]，A9公路以北<ref name="CNN article"/><ref name="bbc">[http://news.bbc.co.uk/2/hi/europe/7909683.stm Turkey plane crashes in Amsterdam]</ref>。9人死亡，55人受傷。

== 降落受損傷 ==


该[[波音737-800|波音737-800]]飞机严重损毁，断成三截，但沒有起火。机上共有127名乘客7名机组人员<ref>[http://www.thy.com/DarkSiteEN/crash_info.aspx Accident Information Page on THY Web Site]</ref><ref name="reuters">[http://www.reuters.com/article/newsOne/idUSTRE51O2J020090225 Turkish airliner crashes at Amsterdam airport, 1 dead]</ref><ref name ="minister1">http://www.hurriyet.com.tr/english/world/11081633.asp?gid=244</ref>。

== 事後 ==
救援人员迅速到达事故地点。大约20辆救护车，以及一些直升机和消防车到达现场。<ref>http://www.dailymail.co.uk/news/worldnews/article-1155022/One-dead-plane-carrying-135-passengers-crashes-breaks-pieces-Amsterdam-airport.html</ref>

起初，根据土耳其政府方面的消息和媒体报道，此事故未造成人员死亡，<ref>{{cite web |url=http://www.agi.it/world/news/200902251240-cro-ren0041-art.html |title=存档副本 |accessdate=2009-03-03 |deadurl=yes |archiveurl=https://web.archive.org/web/20090303145459/http://www.agi.it/world/news/200902251240-cro-ren0041-art.html |archivedate=2009-03-03 }}</ref>不过阿姆斯特丹机场当局否认了该说法。

根据一机场发言人，所有进出史基浦机场的航班被暂时中止。在国际协调时11时15分，有消息称至少Kaagbaan跑道(06/24)已经重新开放。A9公路也被关闭。A4和A13的车道也被关闭，便于紧急服务抵达事故现场。

== 事故原因 ==

3月4日，荷兰安全部门证实，飞行高度计故障是造成这次事故的直接原因<ref name=CNN4Mar/>。

当时飞机在自动驾驶状态下降落，无线电高度计出现了故障，致使自动驾驶仪将2100英尺高空误认为是地面（显示为-8英尺），并且自动油门亦是根据此数据操控飞机。而当时的雾天也使仍没完成降落检查的飞行员无法作出正确判断，机长数度关闭警报。这意味着他知道高度计的故障。但他并不知道由此造成的严重后果且并没有修正这一错误。自动油门在空中降低飞机速度，并拉起机头从而导致飞机[[失速|失速]]，飞行员却把这一行为误认为是自动驾驶仪的落地模式。在500英尺，当飞行员发现速度过低时，已经没有足够的时间与高度控制飞机。最终飞机在不到半分钟后，距离跑道500码的地方坠毁<ref name=CNN4Mar/>。
