'''力学'''是[[物理学|物理学]]的一个分支，主要研究[[能量|能量]]和[[力|力]]以及它们与物体的平衡、变形或运动的关系。
[[File:Mechanics_Overview_Table.jpg|right]]

== 发展历史 ==
人们在日常劳动中使用[[杠杆|杠杆]]、打水器具等等，逐渐认识物体受力，及平衡的情况。古[[希腊|希腊]]時代[[阿基米德|阿基米德]]曾对杠杆平衡、物体[[質心|重心]]位置、物体在[[水|水]]中受到的[[浮力|浮力]]等，作了系统研究，确定它们的基本规律，初步奠定了[[静力学|静力学]]，即'''平衡理论'''的基础，古希臘科學家[[亚里士多德|亞里斯多德]]也提出'''作用力造成運動'''的主張，即物體不受力，必將停止。

自[[文藝復興|文藝復興]]之后，[[科學革命|科學革命]]興起，[[伽利略|伽利略]]的[[自由落体运动规律|自由落体运动规律]]，以及[[艾萨克·牛顿|牛顿]]的[[牛顿运动定律|三大运动定律]]皆奠定了[[動力學|动力学]]的基础。'''力学'''从此开始成为一门科学。此后[[弹性力学|弹性力学]]和[[流体力学|流体力学]]基本方程的建立，使得力学逐渐脱离[[物理学|物理学]]而成为独立学科。到20世纪初，在[[流体力学|流体力学]]和[[固体力学|固体力学]]中，实际应用跟[[数学|数学]]理论的互相结合，使力学蓬勃起来，创立了许多新理论，同时也解决了[[工程技术|工程技术]]中大量关键性问题。

== 古典力学及量子力學 ==
力學主要可分為[[古典力学|古典力学]]及[[量子力學|量子力學]]。

若以發現的時間來看，古典力学較早被發現，啟源於[[艾萨克·牛顿|牛顿]]的[[牛顿运动定律|三大运动定律]]，量子力學則是20世紀初才由許多科學家所創立。

[[古典力学|古典力学]]主要研究低速或静止的宏观物体。[[开普勒|开普勒]]、[[伽利略|伽利略]]，尤其是牛顿是[[古典力学|古典力学]]的奠基人。

[[量子力學|量子力學]]應用範圍較廣，不過主要是針對[[微觀|微觀]]的物質。根據[[對應原理|對應原理]]，量子數相當大的量子系統可以與古典力学中的行為模式相對應，使得量子力學及古典力学不會相衝突。量子力學可以解釋及預測分子、原子及基本粒子的許多行為。不過針對一般常見的巨觀系統，若使用量子力學會複雜到無法處理粒子間的交互作用，因此，巨觀系統透過古典力学的方式處理仍較為恰當。

==[[相对论|相對論]]及牛頓力學 ==
量子力學將力学延伸到经典力學以外的範圍，而[[愛因斯坦|愛因斯坦]]的[[廣義相對論|廣義相對論]]及[[狹義相對論|狹義相對論]]也將原來牛頓及[[伽利略|伽利略]]的力學擴展到更大的範圍。在物體速度接近[[光速|光速]]時，因相對論而產生的效應會主導物體的行為，也會使其速度不會超過光速。[[量子力學|量子力學]]也需要配合相對論進行修正，[[量子场论|量子场论]]就是量子力学和狭义相对论的结合。不過[[量子场论|量子场论]]和[[广义相对论|广义相对论]]目前仍無法整合，這是[[大一統理論|大一統理論]]希望可以克服的問題。

== 依研究物體來分類 ==
在力學中，常用到一個名詞「[[物體|物體]]」。物體可能是[[質點|質點]]、[[拋體|拋體]]、[[太空船|太空船]]、[[星体|星体]]、某些[[流體|流體]]、某些[[固體|固體]]、某些[[機械|機械]]或某些[[土木|土木]][[建築|建築]]。

力學中的一些分支也和所探討的「物體」特性有關。例如質點就是小的物體，在古典力學中只視為一個有[[質量|質量]]的點。而[[剛體|剛體]]有固定的大小及尺寸，不允許形變，和質點比較，剛體增加了一些稱為[[自由度_(物理学)|自由度]]的[[參數|參數]]，例如在空間中的方向。

物體也有可能是可允許[[形變|形變]]的半剛體，如[[弹性_(物理学)|彈性體]]，或者根本沒有固定的形狀，如[[流體|流體]]。這些物體可利用古典力學的方式研究，也可以利用量子力學來分析。

例如一顆[[棒球|棒球]]的運動常使用古典力學來分析，而[[原子核|原子核]]內[[質子|質子]]及[[中子|中子]]的行為則通常會用量子力學來描述。
