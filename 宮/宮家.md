'''宮家'''，是[[日本皇室|日本皇室]]的一種制度，起源於12世紀的[[鎌倉時代|鎌倉時代]]。宮家是皇室的分家，天皇[[皇子|皇子]]、及宮家[[當主|當主]]的所有男性後裔（除了未來將繼承當主的長子）在成年、成婚後皆可創設宮家，並由[[天皇|天皇]]賜下宮號。依該宮家當主和[[天皇|天皇]]的血緣關係又有[[直宮家|直宮家]]和一般宮家的分別。當主，是指該宮家之主，通常情況下為男性，但在男性當主過世、又無男性後裔繼承當主時，則以已逝男性當主的[[正室|正室]]為新任當主。

== 概略 ==
{{see|世襲親王家}}
[[鎌倉時代|鎌倉時代]]以後，由於'''親王宣下'''的制度，使得本來沒有資格繼承皇位的二代以後的[[王|王]]、[[上皇|上皇]]養子，其所受封的[[親王|親王]]稱號被後世子孫承襲的情況非常普遍，此情況被後世稱為[[世襲親王家|世襲親王家]]，而這也是現在創設宮家的源流。例如[[鎌倉時代|鎌倉時代]]中期的[[順德天皇|順德天皇]]，他的兩個皇子[[岩倉宮忠成王|岩倉宮忠成王]]、[[四辻宮善統親王|四辻宮善統親王]]，他們的宮號就被其子孫所承襲下去。

真正被普遍認為是[[世襲親王家|世襲親王家]]之開端者，是[[室町時代|室町時代]]的[[龜山天皇|龜山天皇]]之皇子—[[恆明親王|恆明親王]]所創設的[[常盤井宮|常盤井宮]]，及[[後二条天皇|後二条天皇]]的皇子[[邦良親王|邦良親王]]所創設的[[木寺宮|木寺宮]]。上述兩位[[親王|親王]]，原本都有繼承皇位的可能性，但受到當時[[持明院統|持明院統]]及[[大覺寺統|大覺寺統]]的'''兩統迭立'''之影響，因此無法繼承皇位。而兩位親王所擁有的領地和莊園，皆被他們的後世子孫作為主要的經濟基礎。

[[常盤井宮|常盤井宮]]、[[木寺宮|木寺宮]]兩宮家，在[[室町時代|室町時代]]都因為沒有後嗣能繼承宮家而血統斷絕。而在創設這兩個宮家後，一直到二戰後脫離皇籍的宮家為止，在這長達約550年的時間裡，一直延續下來的宮家是[[伏見宮|伏見宮]]。

並且，在[[室町時代|室町時代]]時，[[小倉宮|小倉宮]]及[[玉川宮|玉川宮]]等，都被當作是[[舊南朝|舊南朝]]的後裔。在繼承皇位的爭奪戰中失意的皇族，或許可以說是為了和本家（天皇皇族）在政治上分庭抗禮而創設宮家，因此對那個時代的天皇而言，或許創設這樣的宮家並非天皇的本意，但日後世襲親王家便是由此開始。

之後到了[[安土桃山時代|安土桃山時代]]，因為朝廷及皇室的權力日漸衰退的影響，因此[[世襲親王家|世襲親王家]]的創設，如果沒有朝廷以外的經濟力量支援，將變的非常困難，對朝廷來說不如接受情況的改變。之後隨著政權的統一而創設的宮家有；[[桂宮|桂宮]]，-{[[有栖川宮|有栖川宮]]}-，[[閑院宮|閑院宮]]等三個宮家，而此三宮家再和早前的[[伏見宮|伏見宮]]，這四個[[世襲親王家|世襲親王家]]被稱為'''四親王家'''。

== 明治時代以後 ==

從[[德川幕府|德川幕府]]末期和[[明治時代|明治時代]]開始，新的宮家陸續創設，在[[文久|文久]]三年（1863年），[[中川宮|中川宮]]家創設（之後依序改號為[[賀陽宮|賀陽宮]]及[[久邇宮|久邇宮]]）、[[元治|元治]]元年（1864年）創設[[山階宮|山階宮]]、[[明治|明治]]三年（1870年），創設[[梨本宮|梨本宮]]、[[聖護院宮|聖護院宮]]、[[北白川宮|北白川宮]]、[[華頂宮|華頂宮]]、[[東伏見宮|東伏見宮]]（[[明治|明治]]十五年成為[[小松宮|小松宮]]家之養子）等宮家，明治25年（1892年）的[[賀陽宮|賀陽宮]]、明治36年的[[東伏見宮|東伏見宮]]、明治39年（1906年），作為明治天皇的皇子，又創設了[[朝香宮|朝香宮]]、[[竹田宮|竹田宮]]、[[東久邇宮|東久邇宮]]等三個宮家。

明治22年（1889年），根據[[皇室典範|皇室典範]]，永世皇族制復活，因'''親王宣下'''而出現的[[世襲親王家|世襲親王家]]的制度便被廢止，但仍有新的宮家創設，[[大正天皇|大正天皇]]的三個[[皇子|皇子]]分別創設了[[秩父宮|秩父宮]]、[[高松宮|高松宮]]和[[三笠宮|三笠宮]]。

在[[二戰|二戰]]之後，由於戰後的日本財政窘迫，無法繼續維持往昔的皇室規模，因此有許多宮家都依[[皇室典範|皇室典範]]第九条脫離皇籍，前述的眾多宮家中，除去[[大正天皇|大正天皇]]的三個[[直宮家|直宮家]]—[[秩父宮|秩父宮]]、[[高松宮|高松宮]]和[[三笠宮|三笠宮]]以外，總共有11個宮家、51位皇族成員脫離皇籍。

== 現今的問題 ==
秋篠宮家當主文仁親王及其妃紀子，在平成18年（2006年）產下現今日本皇室唯一的男嗣[[悠仁親王|悠仁親王]]；除此之外，其他宮家的男嗣自昭和29年（1954年）出生的[[高圓宮憲仁親王|高圓宮憲仁親王]]之後，直至現今沒有其他男性皇嗣誕生。而根據[[皇室典範|皇室典範]]第九条；「天皇或皇族可得[[領養|領養]][[養子|養子]]」所引申的意義，女性皇族也無法創設宮家。

現存的宮家普遍又有後嗣斷絕的問題，根据现行的皇室继承规章，四宫家中已有三宫家被认定将因为绝嗣而在未来断绝（分别是常陆宫、三笠宫和高圆宫）。因此為保持皇位的繼承，創設新的宮家有著急切的必要性，或修改[[皇室典範|皇室典範]]、允許女性皇族成立宮家；或由現存的宮家當主、迎[[明治維新|明治維新]]後脫離皇籍的男性皇族之子孫將其作為養子以繼承宮家等，上述的眾多說法同時爭論著，至今仍沒有一個統一的意識形成。
