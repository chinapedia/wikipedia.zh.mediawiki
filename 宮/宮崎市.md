'''宮崎市'''（{{jpn|j=宮崎市|hg=みやざきし|rm=Miyazaki shi|f=y}}）是位於[[日本|日本]][[九州_(日本)|九州地方]]東南部，[[宮崎縣|宮崎縣]]中部的[[城市|城市]]，也是宮崎縣縣政府所在地。宮崎市擁有[[喜凱亞|喜凱亞]]、[[青島_(宮崎縣)|青島]]、{{link-ja|日本職棒集訓|キャンプ (日本プロ野球)}}等觀光資源，是一座以服務業為主的城市。喜凱亞度假區內的[[喜來登頂級海洋度假大飯店|喜來登頂級海洋度假大飯店]]是九州最高的建築。宮崎市有人口約40萬人<ref>{{cite web |url=http://www.city.miyazaki.miyazaki.jp/fs/7131/02.pdf |script-title=ja:第１章 宮崎市の概要 |accessdate=2017-07-09|publisher={{lang|ja|宮崎市}} |language=ja}}</ref>，是九州次於[[長崎市|長崎市]]的人口第七多城市，自1998年開始是[[中核市|中核市]]。

== 歷史 ==
[[File:Founding_Ceremony_of_the_Hakko-Ichiu_Monument.JPG|thumb]]的竣工儀式]]
「宮崎」這一地名來自於宮崎郡，指宮崎莊、宮崎城的勢力範圍<ref name="Hyakunen">{{cite book |author={{lang|ja|平岡昭利}}|script-title=ja:《九州 地図で読む百年》 |year=1997 |publisher={{lang|ja|古今書院}} |location=東京都 |isbn=4-7722-1665-0|language=ja}}</ref>{{rp|125}}。位於宮崎市跡江地區的{{link-ja|生目古墳群|生目古墳群}}是九州最大的[[古墳時代|古墳時代]]前期的古墳群<ref>{{cite web |url=http://www.city.miyazaki.miyazaki.jp/culture/facilities/1819.html |script-title=ja:生目古墳群史跡公園 |accessdate=2017-07-08|publisher={{lang|ja|宮崎市}} |language=ja}}</ref>。但直到室町時代，現宮崎市轄區都是農村地區。在[[室町時代|室町時代]]的1551年（天文20年）時，太田七郎左衛門忠延在{{link-ja|大淀川|大淀川}}右岸的河口附近新建了名為「城崎」的城鎮。城崎是一座擁有{{link-ja|宮崎港|宮崎港|赤江港}}的小規模商業城市，和近畿地方之間有貿易往來。

在[[江戶時代|江戶時代]]，現宮崎市轄區的土地劃分十分複雜，大淀川左岸的現市中心地區和大淀川右岸的西部是延岡藩的飛地；日向灘沿岸的新別府、吉村、下別府、江田村等地是幕府直轄領地；山崎、鹽路等村是佐土原藩領地；西部的吉野、堤內、金崎村是高鍋藩領地；糸原、有田是鹿兒島藩領地；市轄區南部是飫肥藩領地<ref name="Hyakunen"></ref>{{rp|125}}。由於這些領地大多是距藩廳較遠的領地或飛地，因此大多都是農村地帶，只有城崎是小規模城鎮。江戶時代的宮崎雖然是平原地形，但卻因各藩領地錯綜的局面而未能形成較大的城鎮<ref name="Hyakunen"></ref>{{rp|125}}。

1871年[[廢藩置縣|廢藩置縣]]後，現宮崎縣分屬[[美美津縣|美美津縣]]、[[都城縣_(日本)|都城縣]]，現宮崎市在當時是兩縣交界處。1873年，美美津縣和都城縣合併為宮崎縣，縣廳設於宮崎郡上別府村<ref name="Hyakunen"></ref>{{rp|126}}。1876年，宮崎縣被併入[[鹿兒島縣|鹿兒島縣]]，宮崎縣一度降格為宮崎支廳。1883年，宮崎縣分縣運動取得成功，宮崎縣得以恢復，上別府村也再度成為宮崎縣的縣廳所在地。當時的上別府村仍是農村，縣廳附近就是農田<ref name="Hyakunen"></ref>{{rp|127}}。

1889年日本實施[[町村制|町村制]]之後，上別府村、江平町、川原町、松山町、瀨頭町、上野町合併為宮崎町<ref name="Hyakunen"></ref>{{rp|127}}。和九州其他縣廳所在地城市相比，宮崎町雖然規模較小但發展迅速。1923年[[日豐本線|日豐本線]]的全線通車更使得宮崎町的交通大幅改善。1924年，宮崎市正式設市，此後宮崎市的市區沿國道快速向[[宮崎站|宮崎站]]方向發展<ref name="Hyakunen"></ref>{{rp|127}}。由於日本神話傳說中的[[天孫降臨|天孫降臨]]之地高千穗位於宮崎縣，因此宮崎市在二戰之前是國家神道色彩濃郁的城市。1935年，[[昭和天皇|昭和天皇]]參拜[[宮崎神宮|宮崎神宮]]<ref>{{cite web |url=http://miyazakijingu.jp/modules/about/index.php?content_id=4 |script-title=ja:天皇陛下ご親拝と皇族ご参拝一覧（明治以降） |accessdate=2017-07-08|publisher={{lang|ja|宮崎神宮}} |language=ja}}</ref>。1940年，為紀念皇紀2600年，政府大幅擴建了宮崎神宮，並興建[[八紘一宇塔|八紘一宇塔]]和「皇軍發祥之地」的石碑<ref>{{cite web |url=http://www.miyazaki-city.tourism.or.jp/tourism/spot/20.html |script-title=ja:平和台公園 |accessdate=2017-07-08|publisher={{lang|ja|宮崎市観光協会}} |language=ja}}</ref>。在[[第二次世界大戰|第二次世界大戰]]中，宮崎市和[[日南市|日南市]]等宮崎縣內其他城市遭到了美軍的空襲<ref>{{cite web |url=http://www.soumu.go.jp/main_sosiki/daijinkanbou/sensai/situation/state/kyushu_10.html |script-title=ja:日南市における戦災の状況（宮崎県） |accessdate=2017-07-08|publisher={{lang|ja|総務省}} |language=ja}}</ref>。

[[File:Route220_TachibanaStreet.jpg|thumb]]
宮崎市在戰後的昭和大合併中亦進行了市町村合併，轄區範圍大幅擴大。1960年，島津久永和島津貴子夫妻（昭和天皇的第五皇女）選擇宮崎作為新婚旅行目的地，成為宮崎市觀光產業迅速發展的契機，新婚旅行選擇宮崎的風潮興盛一時<ref>{{cite web |url=http://www.kansai-u.ac.jp/Keiseiken/publication/seminar/asset/seminar11/s191_1.pdf |script-title=ja:メディアイベントとしての新婚旅行―1960 年から 1970 年代の宮崎を事例に― |accessdate=2017-07-08|publisher={{lang|ja|森津千尋}} |language=ja}}</ref>。然而1970年代中期之後，隨著[[沖繩返還|沖繩返還]]使得[[沖繩縣|沖繩縣]]成為日本國內新的南國風情代表地區，宮崎縣的觀光產業開始停滯。1980年代後海外旅行的普及更加劇這一狀況。為發展旅遊業，宮崎市在1980年代後期投資新設大型度假區[[喜凱亞|喜凱亞]]。然而因逢[[日本泡沫經濟|泡沫經濟]]崩潰導致喜凱亞經營困難，連年赤字亦拖累宮崎市財政<ref>{{cite web |url=http://www.kansai-u.ac.jp/Keiseiken/publication/report/asset/sousho143/143_04.pdf |script-title=ja:Ⅳ 観光振興の政策過程研究－宮崎県のリゾート計画を事例に－|accessdate=2017-07-08|publisher={{lang|ja|石橋章市朗}} |language=ja}}</ref>。但另一方面，戰前和延岡市、都城市人口數相當的宮崎市在戰後人口已大幅超過這兩個城市，且宮崎縣內{{link-ja|宮崎都市圈|宮崎都市圏}}的人口比例從1960年的30.5%增加到2005年的43.4%，顯示宮崎市在宮崎縣內的地位明顯提高<ref name="Chishi">{{cite book |author={{lang|ja|野澤秀樹・堂前亮平・手塚章}}|script-title=ja:《日本の地誌10　九州・沖縄》 |year=2008 |publisher={{lang|ja|朝倉書店}} |location=東京都 |isbn=978-4-254-16770-2|language=ja}}</ref>{{rp|435}}。1998年，宮崎市升格成為[[中核市|中核市]]<ref>{{cite web |url=http://www.city.miyazaki.miyazaki.jp/city/public_relations/relations/1997.html |script-title=ja:中核市みやざき|accessdate=2017-07-08|publisher={{lang|ja|宮崎市}} |language=ja}}</ref>。

== 地理 ==
[[File:青島パーキング_-_panoramio.jpg|thumb]]
宮崎市位於{{link-ja|宮崎平原|宮崎平野}}南部，除了市轄區南部的{{link-ja|鰐塚山地|鰐塚山地}}之外，大多數地區地形平緩。位於市轄區南部的{{link-ja|鰐塚山|鰐塚山}}是宮崎市的最高峰，海拔高度1,118米<ref>{{cite web |url=http://www.miyazaki-city.tourism.or.jp/tourism/spot/62.html |script-title=ja:鰐塚山（わにつかやま）|accessdate=2017-07-08|publisher={{lang|ja|宮崎市観光協会}} |language=ja}}</ref>。宮崎市中心地區大部分位於沖積平原和洪積台地之上<ref>{{cite web |url=http://www1.gsi.go.jp/geowww/landcondition/report/D001-temp-010u.pdf |script-title=ja:土地条件調査解説書「宮崎地区」|accessdate=2017-07-08|publisher={{lang|ja|国土地理院}} |language=ja}}</ref>。{{link-ja|大淀川|大淀川}}穿過宮崎市中心並注入{{link-ja|日向灘|日向灘}}。在修築人工堤壩之前，大淀川常有洪水發生<ref>{{cite web |url=http://www.mlit.go.jp/river/basic_info/jigyo_keikaku/gaiyou/seibi/pdf/ooyodo/h280714-2.pdf |script-title=ja:大淀川水系の流域及び河川の概要 |accessdate=2017-07-08|publisher={{lang|ja|国土交通省}} |language=ja}}</ref>。

{{link-ja|橘通 (宮崎市)|橘通り (宮崎市)|橘通}}和{{link-ja|國道10號 (日本)|国道10号|国道10号}}的路口是宮崎市區的商業中心，而橘通則是宮崎市南北方向的主幹道。宮崎市的大多數企業、銀行聚集在市區北部；大淀川沿岸則集中了政府部門和不少旅館；而宮崎神宮附近地區則是宮崎市的文教地區<ref name="Chishi"></ref>{{rp|437}}。

=== 氣候 ===
宮崎市年平均[[氣溫|氣溫]]在18℃左右，年均[[降雨量|降雨量]]超過2,000毫米，有時甚至在3,000毫米之上。但另一方面，宮崎市年均日照時間也大多超過2,000小時，特別是夏季強烈的日曬導致宮崎市雖然降水充沛卻時有乾旱發生，因此市內有眾多儲水池<ref name="Hyakunen"></ref>{{rp|128}}。宮崎市自3月中旬至4月天氣多變。5月下旬至7月上旬則是[[梅雨|梅雨]]季節。7月份的宮崎市受西南風影響而有[[焚風|焚風]]現象出現，是一年中最熱的時期。但8月份時宮崎市的平均氣溫低於[[福岡市|福岡市]]，在九州內並非酷暑地區。夏季的宮崎時有[[颱風|颱風]]吹襲，降雨較多。冬季的宮崎市則因[[九州山地|九州山地]]遮住雪雲使得天氣多晴天，很少有降雪<ref name="Chishi"></ref>{{rp|438}}。

== 人口 ==
[[File:Miyazaki,_Japan.jpg|thumb]]
宮崎町在設町之前的1887年只有人口2,618人，但在1891年時已增加到7,153人，1905年時增加到11,163人<ref name="Hyakunen"></ref>{{rp|129}}。1924年設市時，宮崎市面積45.15平方公里，人口42,920人<ref name="history"></ref>。隨著城市的發展和轄區範圍的擴大，宮崎市人口快速增加，在1950年人口普查時超過10萬人，並在1970年人口普查時超過20萬人<ref name="population">{{cite web |url=http://www.city.miyazaki.miyazaki.jp/fs/9/9/4/6/7/_/jinkoubijon.pdf |script-title=ja:宮崎市地方創生総合戦略 |accessdate=2017-07-08|publisher={{lang|ja|宮崎市}} |language=ja}}</ref>。1990年代後，隨著人口高齡化、少子化的進展，宮崎市人口增長趨緩。2010年，宮崎市人口超過40萬。但自2014年開始，宮崎市人口已開始減少<ref name="population"></ref>。2017年6月1日時，宮崎市有人口399,260人<ref>{{cite web |url=http://www.city.miyazaki.miyazaki.jp/fs/1/1/4/8/6/2/_/h2906jinkousetai.xls |script-title=ja:平成29年6月1日現在 推計人口・世帯数ー管内別 |accessdate=2017-07-08 |publisher={{lang|ja|宮崎市}} |language=ja }}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。宮崎市人口的自然增長率近年趨於零增長，而社會增長率雖在1997年之前長期是增長，1997年之後則時有減少<ref name="population"></ref>。2013年，宮崎市[[總和生育率|總和生育率]]有1.57，高於日本平均值（1.43）但低於宮崎縣平均值（1.72）<ref name="population"></ref>。

== 政治 ==
宮崎市在日本[[眾議院_(日本)|眾議院]]選舉中和[[綾町|綾町]]、[[國富町|國富町]]同屬於{{link-ja|宮崎縣第1區|宮崎県第1区}}<ref>{{cite web |url=http://www.pref.miyazaki.lg.jp/senkyo/kense/senkyo/syu-t.html |script-title=ja:衆議院小選挙区の区域 |accessdate=2017-07-08 |publisher={{lang|ja|宮崎県}} |language=ja}}</ref>。宮崎1區是[[自由民主黨_(日本)|自民黨]]有壓倒性優勢的{{link-ja|保守王國|保守王国}}，除了2009年[[第45屆日本眾議院議員總選舉|第45屆日本眾議院議員總選舉]]之外，自民黨壟斷了該選區的議席<ref>{{cite web |url=http://www.pref.miyazaki.lg.jp/senkyo/kense/senkyo/syu-k.html |script-title=ja:選挙の結果 |accessdate=2017-07-08 |publisher={{lang|ja|宮崎県}} |language=ja}}</ref>。宮崎市現任市長是{{link-ja|戶敷正|戸敷正}}，他在2010年首次當選，並在2014年順利連任<ref>{{cite web |url=http://www.city.miyazaki.miyazaki.jp/mayor/profile.html |script-title=ja:プロフィール |accessdate=2017-07-08 |publisher={{lang|ja|宮崎市}} |language=ja}}</ref>。宮崎市議會共有39名議員，以[[公明黨|公明黨]]和前新會的議員最多，均有6名議員<ref>{{cite web |url=http://www.city.miyazaki.miyazaki.jp/parliament/mechanism/424.html |script-title=ja:会派一覧 |accessdate=2017-07-08 |publisher={{lang|ja|宮崎市議会}} |language=ja}}</ref>。{{link-ja|宮崎縣議會|宮崎県議会}}中，宮崎市共有12名議員<ref>{{cite web |url=http://www.pref.miyazaki.lg.jp/gikai/about/members/area/miyazaki.html |script-title=ja:宮崎市選挙区 |accessdate=2017-07-08 |publisher={{lang|ja|宮崎県議会}} |language=ja}}</ref>。2015年度，宮崎市財政收入1,637.6億日元，支出1,599.8億日元<ref>{{cite web |url=http://www.pref.miyazaki.lg.jp/shichoson/kense/shichoson/documents/20479_20161220162933-1.pdf |script-title=ja:平成27年度市町村普通会計決算の概要 |accessdate=2017-07-08 |publisher={{lang|ja|宮崎県}} |language=ja |deadurl=yes |archiveurl=https://web.archive.org/web/20171022142302/http://www.pref.miyazaki.lg.jp/shichoson/kense/shichoson/documents/20479_20161220162933-1.pdf |archivedate=2017年10月22日 |df= }}</ref>。

== 經濟 ==
[[File:Tachibana_Department_store_Miyazaki_2008.jpg|thumb]]
2013年度，宮崎市市內生產總值13,317.07億日元，佔宮崎縣總量的約43%<ref>{{cite web |url=http://www.pref.miyazaki.lg.jp/tokeichosa/kense/toke/documents/23862_20160622150819-1.pdf |script-title=ja:市町村内総生産 |accessdate=2017-07-08 |publisher={{lang|ja|宮崎県}} |language=ja }}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。宮崎市耕地面積有8,700公頃，農業就業人口有7,304人。2015年的農業生產額有383.8億日元，其中蔬菜佔過半比例<ref name="agriculture">{{cite web |url=http://www.machimura.maff.go.jp/machi/contents/45/201/index.html |script-title=ja:宮崎県宮崎市 |accessdate=2017-07-08 |publisher={{lang|ja|農林水産省}} |language=ja}}</ref>。由於宮崎市氣候溫暖，因此蔬菜大棚種植繁盛，[[黃瓜|黃瓜]]的產量是日本首位，[[菜椒|青椒]]、[[番茄|番茄]]等蔬菜產量也是日本前列，並生產[[芒果|芒果]]等熱帶水果<ref>{{cite web |url=http://www.city.miyazaki.miyazaki.jp/business/agriculture_and_fisheries/agriculture/597.html |script-title=ja:宮崎市の野菜・果樹・花き |accessdate=2017-07-08 |publisher={{lang|ja|宮崎市}} |language=ja}}</ref>。宮崎市也是高檔和牛品種{{link-ja|宮崎牛|宮崎牛}}的產地之一<ref>{{cite web |url=http://www.miyazakibrand.jp/brand-list/01-beef/ |script-title=ja:宮崎牛 |accessdate=2017-07-08 |publisher={{lang|ja|みやざきブランド推進本部}} |language=ja}}</ref>。宮崎市漁港捕撈的魚類以[[金槍魚|金槍魚]]、[[沙丁魚|沙丁魚]]為主<ref name="agriculture"></ref>。宮崎市的製造業企業數和從業者數雖然佔宮崎縣總量的約二成，但產值只在宮崎縣內排名第三位，位居都城市、延岡市之後<ref>{{cite web |url=http://www.meti.go.jp/statistics/toppage/topics/kids/doc/45_miyazaki.pdf |script-title=ja:食料品製造業、電子部品・デバイス・電子回路製造業等が盛んな宮崎県 |accessdate=2017-07-08 |publisher={{lang|ja|経済産業省}} |language=ja}}</ref>。2012年度，宮崎市工業品生產額有約2,212億日元，其中電子元件部門比例最大，其次是食品製造業<ref>{{cite web |url=http://www.city.miyazaki.miyazaki.jp/fs/24994/05.xls |script-title=ja:工業 |accessdate=2017-07-08 |publisher={{lang|ja|宮崎市}} |language=ja}}</ref>。

總部設在宮崎市的銀行有{{link-ja|宮崎銀行|宮崎銀行}}<ref>{{cite web |url=http://www.miyagin.co.jp/kabunushi/profile/outline/ |script-title=ja:会社概要 |accessdate=2017-07-08 |publisher={{lang|ja|宮崎銀行}} |language=ja}}</ref>和{{link-ja|宮崎太陽銀行|宮崎太陽銀行}}<ref>{{cite web |url=http://www.taiyobank.co.jp/profile/?catid=50&blogid=11 |script-title=ja:会社概要 |accessdate=2017-07-08 |publisher={{lang|ja|宮崎太陽銀行}} |language=ja }}{{Dead link|date=2018年9月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。宮崎市集中了宮崎縣約六成的批發銷售額和約四成的零售銷售額<ref>{{cite web |url=http://www.meti.go.jp/statistics/tyo/syougyo/dms/2009/pdf/ken45.pdf |script-title=ja:宮崎県の年間商品販売額は卸売業の６割強、小売業の４割弱を宮崎市が占める |accessdate=2017-07-08 |publisher={{lang|ja|経済産業省}} |language=ja}}</ref>。{{link-ja|橘百貨店|橘百貨店}}是宮崎市最大的在地商業企業之一<ref>{{cite web |url=http://www.bonbelta.com/company/ |script-title=ja:会社案内 |accessdate=2017-07-08 |publisher={{lang|ja|橘百貨店}} |language=ja}}</ref>。和眾多日本中等規模城市類似，宮崎市亦因汽車的普及而出現市中心地區空洞化的問題。現在宮崎市已將市中心地區的活性化作為重要市政目標<ref>{{cite web |url=http://www.city.miyazaki.miyazaki.jp/fs/11301/gaiyoubann1203.pdf |script-title=ja:宮崎市中心市街地活性化基本計画 |accessdate=2017-07-08 |publisher={{lang|ja|宮崎市}} |language=ja}}</ref>。觀光產業在宮崎市經濟中有重要地位。2015年，因[[東九州自動車道|東九州自動車道]]宮崎至大分區間全線通車，來訪宮崎市的遊客數有614.1萬人，比前一年增加6.7%。夏季的8月和冬季的1、2月是宮崎市的觀光旺季。觀光客為宮崎市帶來超過799億日元的消費<ref name="tourism">{{cite web |url=http://www.city.miyazaki.miyazaki.jp/fs/7/7/4/7/0/_/77470.pdf |script-title=ja:平成２７年 宮崎市観光統計 |accessdate=2017-07-08 |publisher={{lang|ja|宮崎市}} |language=ja}}</ref>。1980年代後，政府在宮崎大學附近地區創建{{link-ja|宮崎学園都市|宮崎学園都市}}，試圖通過產學合作發展宮崎市的科技產業<ref>{{cite web |url=http://www.miyazaki-investment.com/cgi-bin/09_comp/view.cgi?sline=3&print=1&keys10=&tid=list2 |script-title=ja:宮崎学園都市ハイテクパーク |accessdate=2017-07-08 |publisher={{lang|ja|宮崎県}} |language=ja}}</ref>。

== 文化教育 ==

宮崎市使用的日語方言是{{link-ja|宮崎方言|宮崎弁}}，屬{{link-ja|豐日方言|豊日方言}}。宮崎方言雖然在文法等方面和同屬豐日方言的{{link-ja|大分方言|大分弁}}有相似之處，但在發音是{{link-ja|無重音|無アクセント}}方言，獨自特徵明顯<ref>{{cite book |author={{lang|ja|真田信治}}|script-title=ja:《方言学》 |year=2011 |publisher={{lang|ja|朝倉書店}} |location=東京都 |isbn=978-4-254-51524-4|language=ja}}</ref>{{rp|104-108}}。{{link-ja|宮崎神宮大祭|宮崎神宮大祭}}於每年10月26日舉行，自20世紀國家主義高揚之後規模日益盛大，現在已是宮崎縣內最大的節慶活動<ref>{{cite web |url=http://miyazakijingu.jp/modules/about/index.php?content_id=6 |script-title=ja:宮崎神宮とは |accessdate=2017-07-08 |publisher={{lang|ja|宮崎神宮}} |language=ja}}</ref>。

總部位於宮崎市的{{link-ja|宮崎日日新聞|宮崎日日新聞}}創刊於1940年，當時名為日向日日新聞，在1961年改為現名。2016年，宮崎日日新聞的日發行量約20.75萬份，在宮崎縣內的市場佔有率有67%<ref>{{cite web |url=http://www.the-miyanichi.co.jp/info/category_41/item_47.html |script-title=ja:会社案内 |accessdate=2017-07-08 |publisher={{lang|ja|宮崎日日新聞}} |language=ja}}</ref>。宮崎市內的電視台有{{link-ja|NHK宮崎放送局|NHK宮崎放送局}}；[[JNN|JNN]]系列的[[宮崎放送|宮崎放送]]<ref>{{cite web |url=http://mrt.jp/company/info/ |script-title=ja:会社情報 |accessdate=2017-07-08 |publisher={{lang|ja|宮崎放送}} |language=ja}}</ref>；同時加盟[[FNN|FNN]]、[[NNN|NNN]]、[[全日本新聞網|ANN]]的[[宮崎電視台|宮崎電視台]]<ref>{{cite web |url=http://www.umk.co.jp/company/info/ |script-title=ja:会社基礎情報 |accessdate=2017-07-08 |publisher={{lang|ja|テレビ宮崎}} |language=ja}}</ref>。宮崎電視台也是日本唯一同時加盟三個[[電視聯播網|電視聯播網]]的電視台。由於宮崎市氣候溫暖，因此在冬季時一些[[日本職棒|日本職棒]]、[[J聯賽|J聯賽]]球隊選擇宮崎市為冬季集訓的地點。2017年，[[讀賣巨人|讀賣巨人]]、[[歐力士野牛|歐力士野牛]]、[[福岡軟銀鷹|福岡軟銀鷹]]都選擇宮崎市作為冬季集訓地<ref>{{cite web |url=http://camp.miyazaki-city.tourism.or.jp/ |script-title=ja:2017年 プロ野球 宮崎春季キャンプ特設サイト |accessdate=2017-07-08 |publisher={{lang|ja|宮崎市観光協会}} |language=ja}}</ref>。這一時期也吸引大量球迷來到宮崎，促進觀光業的發展。

|language=ja}}</ref>。現在宮崎機場運行有前往[[東京國際機場|羽田]]、[[中部國際機場|名古屋]]、[[大阪國際機場|伊丹]]、[[關西國際機場|關西]]、[[福岡機場|福岡]]、[[那霸機場|那霸]]的日本國內定期航線，[[仁川國際機場|首爾]]、[[桃園國際機場|台北]]、[[香港國際機場|香港]]的國際定期航線<ref>{{cite web |url=http://www.miyazaki-airport.co.jp/flightinfo.html |script-title=ja:フライト情報 |accessdate=2017-07-08 |publisher={{lang|ja|宮崎空港}} |language=ja}}</ref>。因宮崎機場地理位置靠近宮崎市中心且宮崎市陸上交通不便，使得宮崎機場是日本乘客數較多的地方機場。2015年度，宮崎機場的乘客數達298.76萬人次<ref>{{cite web |url=http://www.miyazaki-apc.jp/overview/data.html |script-title=ja:データで見る宮崎空港 |accessdate=2017-07-08 |publisher={{lang|ja|宮崎空港振興協議会}} |language=ja}}</ref>。

== 觀光資源 ==
[[File:シーガイア2011.8.jpg|thumb]]
{{link-ja|青島神社 (宮崎市)|青島神社|青島神社}}是宮崎市遊客數最多的觀光景點<ref name="tourism"></ref>。青島神社的歷史至少可追溯至820年，在古代曾禁止普通人進入，直到1737年才解除禁令<ref>{{cite web |url=http://www.miyazaki-city.tourism.or.jp/tourism/spot/12.html |script-title=ja:青島神社 |accessdate=2017-07-08 |publisher={{lang|ja|宮崎市観光協会}} |language=ja}}</ref>。青島神社所在的[[青島_(宮崎縣)|青島]]擁有豐富的熱帶生物，並且以獨特的海蝕地貌「鬼之搓衣板」而著稱<ref>{{cite web |url=http://www.miyazaki-city.tourism.or.jp/tourism/spot/11.html |script-title=ja:鬼の洗濯板（波状岩） |accessdate=2017-07-08 |publisher={{lang|ja|宮崎市観光協会}} |language=ja}}</ref>。青島海水浴場是日本著名的海水浴場，被選入海水浴場百選<ref>{{cite web |url=http://www.welcomekyushu.jp/event/?mode=detail&isSpot=1&id=9999900046918 |script-title=ja:青島海水浴場 |accessdate=2017-07-08 |publisher={{lang|ja|九州観光推進機構}} |language=ja}}</ref>。[[宮崎神宮|宮崎神宮]]主祭[[神武天皇|神武天皇]]，雖然歷史較短但卻是[[官幣大社|官幣大社]]，在[[神道教|神道教]]中有崇高地位<ref>{{cite web |url=http://miyazakijingu.jp/modules/about/ |script-title=ja:ご祭神  |accessdate=2017-07-08 |publisher={{lang|ja|宮崎神宮}} |language=ja}}</ref>。[[宮崎縣立美術館|宮崎縣立美術館]]在收藏展示宮崎縣當地美術家的同時亦收集世界著名美術家的作品<ref>{{cite web |url=http://www.miyazaki-archive.jp/bijutsu/tour/about.html |script-title=ja:美術館について  |accessdate=2017-07-08 |publisher={{lang|ja|宮崎県立美術館}} |language=ja}}</ref>。[[宮崎縣綜合博物館|宮崎縣綜合博物館]]已有60年以上的歷史，展示宮崎縣的自然與文化<ref>{{cite web |url=http://www.miyazaki-archive.jp/museum/history/index.html |script-title=ja:総合博物館の歴史  |accessdate=2017-07-08 |publisher={{lang|ja|宮崎県総合博物館}} |language=ja}}</ref>。位於宮崎市沿海的[[喜凱亞|喜凱亞]]是日本規模前列的大型度假區，擁有高級酒店、溫泉<ref>{{cite web |url=http://www.seagaia.co.jp/japanese/hotel/ |script-title=ja:ホテル・スパ・温泉  |accessdate=2017-07-08 |publisher={{lang|ja|フェニックス・シーガイア・リゾート}} |language=ja}}</ref>、[[高爾夫球|高爾夫球]]場<ref>{{cite web |url=http://golf.seagaia.co.jp/ |script-title=ja:ゴルフ  |accessdate=2017-07-08 |publisher={{lang|ja|フェニックス・シーガイア・リゾート}} |language=ja}}</ref>等娛樂設施。
