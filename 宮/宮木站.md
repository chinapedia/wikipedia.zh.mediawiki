'''宮木站'''（{{jpn|j=宮木駅|hg=みやきえき|rm=Miyaki eki|f=y}}）是位於[[日本|日本]][[長野縣|長野縣]][[上伊那郡|上伊那郡]][[辰野町|辰野町]]大字伊那富的[[東海旅客鐵道|東海旅客鐵道]]（JR東海）[[飯田線|飯田線]]車站。

== 車站構造 ==
*單式月台1面1線
*[[地上車站|地上車站]]
*由[[伊那市站|伊那市站]]管理的[[無人站|無人站]]
*站舍與出入口在下行方向末端
*設有公廁。

== 車站周邊 ==
車站兩旁是三州街道和[[中央自動車道|中央自動車道]]，附近有條[[天龍川|天龍川]]。還有一些製紙和和菓子工場。

*辰野町立辰野中學
*[[信州豐南短期大學|信州豐南短期大學]]
*辰野醫院
*辰野宮本郵局
*[[長野縣辰野高等學校|長野縣辰野高等學校]]
*[[高原信金中央|高原信金中央]]宮木支店

== 历史 ==
* 1909年（[[明治|明治]]42年）12月28日 - 伊那電車軌道（1919年改名為[[伊那電氣鐵道|伊那電氣鐵道]]）松島（現在的伊那松島）至辰野（即是西町站）之間開業，新設'''宮木停車站'''。
* 1923年（[[大正|大正]]12年）3月16日 - 伊那松島至辰野之間建設新線，暫停使用。
* 1923年（大正12年）7月24日 - 宮木停車站重新營業。
* 1943年（[[昭和|昭和]]18年）8月1日 - 伊那電氣鐵道線國有化，成為[[日本國有鐵道|國鐵]]飯田線車站。同時升級為'''宮木站'''。
* 1954年（昭和29年）12月1日 - 東京都區内各站與[[長野站|長野站]]出發的旅客可以使用此車站。
* 1971年（昭和46年）4月1日 - 出發旅客車站的限制廢除。
* 1986年（昭和61年） - 成為無人站。
* 1987年（昭和62年）4月1日 - 因[[國鐵分割民營化|國鐵分割民營化]]，本站成為東海旅客鐵道的車站。
* 1998年（[[平成|平成]]10年）6月 - 中央主頁付近的業務施設撒去。
