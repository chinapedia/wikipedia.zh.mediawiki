[[美國|美國]][[華盛頓州|華盛頓州]]共有281個[[基层政权|'''地方政府''']]。州法律決定了各個地方政府的權力。

== 城市分級 ==
法律上來說，華盛頓州的一個城市可以以它的等級來作主要描述。華盛頓州的城市共分為五級：

* 一級市（First class city）（佔3.6%）
* 二級市（Second class city）（佔3.2%）
* 鎮（Town）（佔24.6%）
* 未分級市（Unclassified city）（只有一個）
* 編碼市（Code city）（佔68.3%）

'''一級市'''為分級當時人口超過10,000人並且在地方自治特許狀下運作的城市。這些城市可以執行任何《{{Link-en|華盛頓州修訂法|Revised Code of Washington}}》第35章特別允許的職責。包括[[西雅圖|西雅圖]]、[[塔科马_(华盛顿州)|塔科马]]、[[斯波坎|斯波坎]]、[[溫哥華_(華盛頓州)|溫哥華]]和[[雅基馬|雅基馬]]。

'''二級市'''為分級當時人口超過1,500人並且不具地方自治特許狀的城市。就像一級市一樣，這些城市可以執行任何《華盛頓州修訂法》第35章特別允許的職責。包括[[奥查德港_(华盛顿州)|奥查德港]]、[[瓦帕托_(华盛顿州)|瓦帕托]]、[[奥马克_(华盛顿州)|奥马克]]和[[科尔维尔_(华盛顿州)|科尔维尔]]。

'''鎮'''為分級當時人口低於1,500人的地方政府。鎮並不在特許狀下運作。就像前面的城市一樣，這些鎮可以執行任何《華盛頓州修訂法》第35章特別允許的職責。包括[[斯特拉昆_(華盛頓州)|斯特拉昆]]、[[星期五港_(華盛頓州)|星期五港]]、[[伊頓維爾_(華盛頓州)|伊頓維爾]]和[[沃特維爾_(華盛頓州)|沃特維爾]]。1994年，州議會規定1,500人為建制的人口下限。

'''未分級市'''並不在任何分級下運作。只有一個城市——[[瓦拉瓦拉縣_(華盛頓州)|瓦拉瓦拉郡]]的[[韦茨堡_(华盛顿州)|韦茨堡]]——是未分級市。該市在1881年的領地特許狀下成立，即華盛頓州加入聯邦的前八年。

'''編碼市'''為州議會所用來在州憲法看一般法下允許最大程度的地方自治。華盛頓州內最多城市採用這個分級。包括[[伦顿_(华盛顿州)|伦顿]]、[[贝尔维尤_(华盛顿州)|贝尔维尤]]、[[奧林匹亞_(華盛頓州)|奧林匹亞]]、[[長景市_(華盛頓州)|長景市]]、[[普尔曼_(华盛顿州)|普尔曼]]和[[大學城_(華盛頓州)|大學城]]。

編碼市（''非必要自治編碼市''（optional municipal code cities）的縮寫，由《華盛頓州修訂法》第35A章編碼）被州憲法或州法律授權得以執行任何沒有特別禁止的職責。這些城市可以執行任何其它分類的城市所被允許的職責。

== 政府形式 ==
華盛頓州的一個城市可以以它的政府形式來作次要描述。市與鎮的政府有三種不同的形式：

* {{Link-en|市委員會制政府|City commission government|市委員會制}}（只有一個）
* [[市长-议会制政府|市長-議會制]]（佔81.1%）
* [[議會-經理制政府|議會-經理制]]（佔18.5%）

'''市委員會制'''

[[雪爾頓_(華盛頓州)|雪爾頓]]是華盛頓州唯一使用三人委員會制政府的城市。

'''市長-議會制'''

華盛頓州內最多城市採用這種形式，這種政府需要一個民選[[市長|市長]]和一個民選市議會。包括[[西雅圖|西雅圖]]、[[斯波坎|斯波坎]]、[[肯特_(华盛顿州)|肯特]]、[[埃弗里特_(華盛頓州)|埃弗里特]]、[[布雷默顿_(华盛顿州)|布雷默顿]]和[[贝灵厄姆_(华盛顿州)|贝灵厄姆]]。

'''議會-經理制'''

這種形式的政府有一個民選的市議會和一個委任的市經理。包括[[雅基馬|雅基馬]]、[[溫哥華_(華盛頓州)|溫哥華]]、[[塔科马_(华盛顿州)|塔科马]]、[[贝尔维尤_(华盛顿州)|贝尔维尤]]和[[肯纳威克_(华盛顿州)|肯纳威克]]。
