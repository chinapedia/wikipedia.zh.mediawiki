'''華中橋'''是一座位於[[台灣北部|台灣北部]][[新店溪|新店溪]]之上的[[橋樑|橋樑]]，連結[[台北市|台北市]][[萬華區|萬華區]]與[[新北市|新北市]][[中和區|中和區]]，是目前中和區與台北市唯一聯絡的橋樑，也是兩地往來重要的交通要道。

== 沿革 ==
華中橋係由[[台灣省政府|台灣省政府]]和[[台北市政府|台北市政府]]共同投資興建，當時投資約新台幣四億九千多萬元，於1975年建造，並於1976年10月10號通車。

== 數據 ==
由於地處縱深最大的華中河濱公園，華中橋全長達1600[[公尺|公尺]]，寬約25公尺，主橋30跨29墩，長1263公尺，寬24.5公尺，採實心牆式橋墩，設有快車道四線，慢車道與人行道各兩線，為連接台北市及新北市最長的橋樑<ref name="廖耀宇"/>。台北市端增設汽車匝道銜接[[水源快速道路|水源快速道路]]淨寬5.5公尺，總長度約755公尺，改建機車道(含人行道)淨寬5.75公尺，總長度279公尺，可解決第一果菜市場於第一魚市場交通問題，已於2016年12月9日開工，預定2018年11月24日完工。

==下游側人行道(暨自行車道)拓寬工程==
本工程為配合[[臺北市政府|臺北市政府]]辦理雙北城市興建跨市橋樑自行車通行環境建置系統，並由[[臺北市政府交通局|臺北市政府交通局]]邀集[[新北市政府交通局|新北市政府交通局]]及相關單位研議跨市橋樑興建可行性，及研議跨市橋樑合作方案，以提供民眾友善安全之自行車通行環境。
本工程為105-106年度連續性計畫，將銜接聯絡[[中和區|新北市中和區]]與[[萬華區|臺北市萬華區]]華中橋下游側(西側)原1.5公尺寬人行道，再向外拓寬1.5公尺，拓寬後人行道寬度為3公尺，採人車共道方式規劃設計，拓寬之長度約920公尺。
<ref>[http://w2.dbas.taipei.gov.tw/budget/ofbudget/105budget/BUDGET/bfbud/105br5430_0707000m.pdf 工務局主管105年度施政及預算重點說明]</ref>
