'''華月'''（Kazuki，{{bd|1981年|4月7日|2000年|10月31日|catIdx=H}}），本名'''渡邊和樹'''（Watanabe Kazuki），出生於日本[[東京都|東京都]]，是[[日本|日本]]樂團[[Raphael|Raphael]]中的[[團長|團長]]，擔任[[吉他手|吉他手]]。他的[[吉他|吉他]]技術亦非常了得，他會不斷學習，磨練吉他技術。

在做[[美術|美術]]，[[封面設計|封面設計]]與[[形象設計|形象設計]]等相關工作時會另用化名'''華姬'''（Hanahime）。此外，華月想做女孩子，在穿女裝時會稱呼自己為「華壽美」（Kasumi），六弦舞姬。2000年10月31日，華月因壓力過大而服用過量[[鎮定劑|鎮定劑]]逝世。享年19歲。Raphael也在華月遺體被發現的當天過後五個月未發表任何活動。原訂的新世紀冬天巡迴在[[團長|團長]]不在的情況下正常巡迴，後正式發表活動休止聲明稿。華月的才華令人震驚，在他過世後,他的公司亦發行了諸多紀念華月的專輯<ref name="reunion">

==簡歷==
# 1981年4月7日 - 出生於東京都渋谷区
# 1992年 - 由於崇拜X JAPAN的hide，生日時買了第一把吉他
# 1994年 - 進入私立中學就讀
# 1995年 - 從相識的朋友那認識了yukito、結成THE BLUE HEARTS的cover band
# 1996年 - 到了YUKITO中學校慶參訪、由那得知YUKITO想要組成樂團。
# 1997年春 - Raphael結成。
# 1997年春 - 私立高中入學。因為髮禁的理由所以退學轉往公立高中。(日本私立高中優於公立高中)
# 1997年夏 - 和父親爭執所以離家出走。與當時同年的朋友同居。
# 1997年秋 - 認識了音樂製作人-平井光一。
# 1997年冬 - Raphael第一次演出。
# 1998年春 - 高中退學。
# 1999年夏 - Raphael出道主流
# 2000年春 - Raphael武道館公演。並且當作畢業典禮一般舉行
# 2000年10月31日 - 在自家發現遺體。死因是鎮定劑服用過量的中毒。
