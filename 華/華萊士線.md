'''華萊士線'''是[[生物地理學|生物地理學]]中，區分[[東洋區|東洋區]]（又名[[極東區|極東區]]{{r|shelldoc_000008|page1=60}}）和[[澳大拉西亞區|澳大拉西亞區]]的分界線。1854年到1862年英國動物地理學者[[亞爾佛德·羅素·華萊士|亞爾佛德·羅素·華萊士]]在馬來群島研究島嶼上的動物時，注意到[[婆羅洲|婆羅洲]]與[[蘇拉威西島|蘇拉威西島]]、[[峇里島|峇里島]]和[[龍目島|龍目島]]之間，似乎有一條隱形的界線將兩邊的生物分開；界線以西接近[[東南亞|東南亞]]的生物相，界線以東則接近[[新幾內亞|新幾內亞]]的生物相。華萊士注意到：峇里島的鳥類與爪哇島幾乎相同，但在距峇里島僅約30公里的龍目島，卻只有50%的鳥類相同。為紀念他的發現，科學界將劃分這兩區的界線稱為華萊士線。

== 延伸與修正 ==
[[File:Map_of_Sunda_and_Sahul.png|thumb]]
“华莱士线”这一名称最早是由托马斯·赫胥黎在1868年致伦敦动物学会的一封信件中提出，但是他延伸该线至菲律宾西侧。此后19世紀末、20世紀初的生物地理學者一直嘗試劃出一條明確的界限，以區分[[亞洲|亞洲]]與[[澳洲|澳洲]]的生物相；許多學者在華萊士線的基礎上提出延伸或修正，如[[韋伯線|韋伯線]]（Weber's Line）、[[里德克線|里德克線]]（Lydekker's Line）和[[赫胥黎線|赫胥黎線]]（Huxley's Line）等。
* 韋伯線：在華萊士線以東，通過[[摩鹿加群島|摩鹿加群島]]和蘇拉威西島，[[帝汶|帝汶]]與[[澳大利亞|澳大利亞]]之間。界線以西，東洋區動物成份超過50%；界線以東，澳洲區動物成份超過50%。
* 里德克線：從澳大利亞大陸北岸劃到[[新幾內亞島|新幾內亞島]]的西方邊緣，基本上是[[莎湖陸棚|莎湖陸棚]]（Sahul Shelf）的範圍。華萊士線與里德克線之間的區域稱為華萊士區。
* 赫胥黎線：由華萊士線向北延伸，通過[[巴拉望島|巴拉望島]]與[[菲律賓群島|菲律賓群島]]西部之間，並繞到[[呂宋島|呂宋島]]北側 (也就是將除了巴拉望島以外的菲律賓群島劃入澳大拉西亞區)。近年許多研究及發現，若再往北延伸，此線或者可能繞過[[臺灣島|臺灣島]]的東側，將[[蘭嶼|蘭嶼]]及[[綠島|綠島]]劃入與菲律賓群島同區，但此兩地的生物相同時一半是菲律賓相與一半是東亞相的，所以尚有保留。又被稱為新華萊士線。

== 華萊士線與冰河時期 ==
20世紀起隨著科技的進步、[[聲納|聲納]]的發明，人類開始對海洋進行海底探測，逐漸對[[海底地形|海底地形]]有更多了解，其中發現華萊士線與海平面下降約120 ~ 150公尺的陸地海岸線一致。推測在[[冰河時期|冰河時期]]，海平面下降，因此[[婆羅洲|婆羅洲]]及[[峇里島|峇里島]]等島嶼曾經與[[亞洲大陸|亞洲大陸]]相連，[[新幾內亞|新幾內亞]]和鄰近島嶼則曾與[[澳大利亞|澳大利亞]]相連；而華萊士線劃過的地方仍然有[[海洋|海洋]]的阻隔，如[[婆羅洲|婆羅洲]]與[[蘇拉威西|蘇拉威西]]、[[峇里島|峇里島]]與[[龍目島|龍目島]]之間。由於動植物多以陸行方式散布遷移，因此即使距離較近，多數動[[植物|植物]]也無法跨過[[海洋|海洋]]的阻隔。

== 其他應用 ==
[[马六甲海峽|馬六甲海峽]]最淺處僅深25公尺，[[巽他海峡|巽他海峡]]更淺，僅深20公尺，巨大船舶的擱淺事故時有發生；由於[[龙目海峡|龙目海峡]]與[[望加锡海峡|望加锡海峡]]水深均遠深於馬六甲海峽，因此華萊士線經過之處也可供24萬噸級 (Malacca-Max) 以上的[[超級油輪|超級油輪]]通行。
