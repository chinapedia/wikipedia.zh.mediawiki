'''華容道'''（{{lang-en|Klotski}}，來自[[波蘭文|波蘭文]]的{{lang|pl|klocki}}，意為'''木塊'''）是一種[[滑塊類遊戲|滑塊類遊戲]]，由放在方形盘中的10块方片拼成，目標是在只滑動方塊而不從[[棋盤|棋盤]]中拿走的情況下，將最大的一塊移到底部出口。流行於中國的[[華容道|華容道]]是由[[英國人|英國人]]John Harold Fleming在1932年所發明，然後本土化加上三國背景<ref name="两千年来的那些游戏 第2集 雅戏"></ref>。国内国外都有一些华容道的爱好者研究者<ref name="漫谈智力游戏华容道">{{cite book |author=杜焕生 |title=《漫谈智力游戏华容道》 |date=1986-04|publisher=宇航出版社|location=中國 |language=zh-hans |}}</ref>。
[[Image:NTPC-SHJH_HuaRongDao_2013-06.jpg|thumb]]

==玩法==
像其他的种类的滑块游戏，在一个方形盒子内放置了大小不同的方块，一般是4x5大小。在这些方块中有特殊的一个（一般是最大的）必须被移动到设计好的指定地点。玩家不允许拿起方块，但只可向平行或垂直的方向移动方块，常见的玩法是移動最少次數，或者用最少的時間來完成遊戲。

==歷史==
* Lewis W. Hardy在1909年获得了名为'''Pennant Puzzle'''的专利，是華容道的前身，并被芝加哥的OK Novelty公司生产。<ref>{{cite web |url=http://www.g4g4.com/MyCD5/SOURCES/SOURCE2.DOC |title=存档副本 |accessdate=2008-07-08 |deadurl=yes |archiveurl=https://web.archive.org/web/20071026082152/http://www.g4g4.com/MyCD5/SOURCES/SOURCE2.DOC |archivedate=2007-10-26 }}</ref>他还在1907-12-14申请了美国专利{{US patent|1017752}}并在1912-02-20通过<ref>[http://www.google.com/patents?id=xaoPAAAAEBAJ&pg=PA1&source=gbs_selected_pages&cad=2#v=onepage&q=%20&f=false U.S. Patent 1,017,752]</ref>。

*1932年，John Harold Fleming在英國申請專利（英國专利号411515），已與現代的華容道一模一樣，並也提供了橫刀立馬的解法<ref>[http://v3.espacenet.com/publicationDetails/mosaics?CC=GB&NR=411515A&KC=A&FT=D&date=19340607&DB=%20&locale=%20 An improved puzzle, and means therefor]</ref>。

===在中國的流傳===

*《[[科学消遣|科学消遣]]》（''Science Pastime''）是中國最早对于华容道游戏的记录之一，作者是一位中国教授姜长英，写于1949年。其称「估計它的歷史不過有幾十年。從前人的筆記中沒有發現有玩具華容道的記載。」<ref>[http://www.iis.sinica.edu.tw/Conference/ICGA2005/icga/andrea06-2.htm 第十一屆電腦賽局發展學術研討會]</ref>

* 40年代，华容道玩具在上海就很流行。50年代初期，上海地摊上有薄纸印制的华容道玩具出售。后来，上海文具店里也出售过木制的华容道玩具。60年代，上海玩具十四厂和上海长春塑料厂曾把华容道制作成塑料玩具，没有武將名字，只是薄的彩色塑料片，被命名为“船塢排檔”。

* 1943年，[[新四軍|新四軍]]文化教員梁青從蘇北的群眾那裡學會華容道，並將其在新四軍推廣。梁青還收集整理了華容道不同的佈局，並給每一個佈局都起了一個相應的名字。<ref>[http://www.chinesepuzzles.org/chinese/slidingblockpuzzles.php 華容道簡史]</ref>

*1956年8月号的《数学通讯》杂志上，在封面的小知识栏目中提出了“关羽放曹”游戏。1959年5月号和6月号的《辽宁画报》上也刊登了这个游戏，命名为“赶走纸老虎”。

*1982年9月号《我们爱科学》杂志刊登數學家[[谈祥柏|谈祥柏]]写的《华容道》。

*1984年9月，在[[北京市少年宫|北京市少年宫]]，由《我们爱科学》杂志社和[[首钢|首钢]]带钢厂粉末冶金厂联合举办少年华容道计时比赛，完成横刀立马局的最少时间是15秒。

*1985年1月5日，[[中国中央电视台|中国中央电视台]]在少年儿童节目里播放了《小游戏“华容道”》，由-{余}-俊雄供稿，介绍了华容道的规则和历史。

*1987年，余俊雄在《独粒钻石和华容道——迷人的智力游戏》認為華容道是來自[[洛書|洛書]]。之後人們就在填纵横图的基础上，发明了一种数学游戏——[[重排九宫|重排九宫]]。又说大约在元朝时傳到歐洲。但是这里的说法经不起推敲。首先重排九宫游戏在中國历史上从来没有任何文献记载或实物证据。至于元朝传入西方，这恐怕更是作者的凭空想象。余俊雄是北京玩具协会益智玩具委员会主任，热衷于推广中国古典数学玩具。从2002年起，北京玩具协会和中国科技馆举办了中国古典数学玩具展，把华容道列入其中，使得谬误传播开。<ref>[http://www.xys.org/xys/magazine/GB/2007/xys0712.txt “华容道”玩具不是中国人发明的]</ref>

*2002年著作《圖說中國古代遊藝》，介紹了中國自古傳統的智力遊戲七巧板、九連環，而無華容道<ref name=圖說中國古代遊藝">{{cite book |author=崔樂泉|title=《圖說中國古代遊藝》|date=2003-04-01|publisher=文津出版社|location=臺灣|language=zh-tw|isbn=9789576687037}}</ref>。

*2003年谈祥柏翻譯Elwyn R. Berlekamp、John Conway、Richard Guy在1982年合著《Winnings Ways:for your mathematical plays》，書名《稳操胜券》。書中引介了許多華容道的排法<ref name="稳操胜券">{{cite book |author=谈祥柏 |title=《稳操胜券》|date=2003-12-01|publisher=上海教育出版社|location=中國 |language=zh-cn |isbn=7532092208}}</ref>。

*2004年[[吴鹤龄|吴鹤龄]]在其著作《七巧板、九连环和华容道：中国古典智力游戏三绝》，認為「把華容道看做是從西方傳入以後本地化的產物，倒是比較合乎事實和邏輯的」。從無出現民國前的華容道玩具實體或文獻記載。

中國人之所以誤認华容道為其傳統遊戲，赋予这么悠久的历史，是因为它的故事背景出自於《三国演义》第五十回。<ref name="两千年来的那些游戏 第2集 雅戏">[http://bugu.cntv.cn/documentary/humanities/2000nianlaidenaxieyouxi/classpage/video/20100214/100146.shtml 《2000年来的那些游戏》第2集 雅戏]</ref>

==解法==

横刀立马布局最少的步數為81步，而且經電腦驗證過，已經不能再少。第一次有紀錄的81步走法是由[[马丁·加德纳|马丁·加德纳]]在1964年2月刊的《[[科学美国人|科学美国人]]》給出。在文章內他給了以下幾種類似華容道的遊戲的走法（括號內是Hordern分類的編碼）：Pennant Puzzle(C19),L'Âne Rouge(C27d),Line Up the Quinties(C4),Ma's Puzzle(D1),Stotts' Baby Tiger Puzzle (F10).

==變種==

===方塊名稱不同===

* 在[[第二次世界大戰|第二次世界大戰]]時期的[[日本|日本]]出現了排列幾乎一模一樣，但是方塊名稱改變了的遊戲，名稱為「{{lang|ja|[[:ja:箱入り娘_(パズル)|箱入り娘]]}}」。其中最大的方块称为“娘”（女儿），其他的方块也相应的以家庭成员来命名。[[File:Hakoiri3.jpg|缩略图]]

* 另有一種在日本出現的變種，每塊方塊都使用[[日本将棋|將棋]]棋子名稱。[[File:Hakoiri_Shogi.jpg|缩略图]]

===方塊排列不同===

====Pennant Puzzle====
[[File:Dad's_Puzzle.png|thumb]]
它還有很多名稱，例如'''Dad's Puzzler'''等等，由L. W. Hardy在1909年取得版權。它的玩法和華容道大致一樣，只是方塊排列和目的地位置有出入：
# 預設的方塊位置不同（見右圖），最大塊的正方形在左上角
# 除了2x2的正方形外，另外8塊分別為：兩塊1x2，四塊2x1，兩塊1x1
# 出口在圍欄的左下角，不是在底部正中

目前已知最少需要59步完成遊戲。

===電腦版本===

最早出現的電腦版本是包含在[[Microsoft_Windows_3.1|Microsoft Windows 3.1]] Game Pack內的其中一項小遊戲，需另外購置。

本遊戲在[[Gnome|Gnome]]有一個版本，由Lars Rydlinge開發；在其他distro尚有其他版本。

===方塊的形狀不同===
日本有一個變種，是將其中一個大塊變成兩個兵，難度比原來的容易。
最近几年出现了“不动兵”、将大方块变成曲尺形的“异形”类布局或“蝶舞”类联动布局，难度比传统华容道大得多。
