'''華仔天地'''（[[英語|英語]]：Andy World Club）是香港艺人[[刘德华|刘德华]]组建、由专业人士负责管理的愛好者组织，于1988年6月18日成立，至今成立已超過25年。宗旨是透过华仔天地发挥愛好者的团结力量，全心支持刘德华，以及号召会员发扬华仔努力不懈、服务社群的精神来回馈社会<ref>{{cite web |url=http://ent.xinmin.cn/2009/01/24/1539777.html |language= |title=刘德华上海成立首个“华仔天地”分会 |publisher=新民网 |date=2009-01-24 |accessdate= |deadurl=yes |archiveurl=https://web.archive.org/web/20100113231815/http://ent.xinmin.cn/2009/01/24/1539777.html |archivedate=2010-01-13 }}</ref><ref>{{cite web|url=http://sh.sina.com.cn/news/h/2009-02-11/1754107064.html | language= | title=刘德华歌迷会上海分会22日在东方艺术中心成立 |publisher=东方早报 |date=2009年2月11日 |accessdate= }}</ref>。会员分布在世界各地，主要在[[香港|香港]]、[[台湾|台湾]]、[[中国大陆|中国大陆]]、[[新加坡|新加坡]]和[[马来西亚|马来西亚]]等华人聚集区。

== 简介 ==
1988年華仔天地成立后，经常能在许多有劉德華出现的场所看到许多身着黄色会员服的会员，例如颁奖典礼、宣传活动现场等等<ref>{{cite web |url=http://www.jmnews.com.cn/c/2006/08/16/20/c_866425.shtml |language= |title=组图：台下“华仔天地”尖叫 刘德华：害怕衰老 |publisher=腾讯娱乐 |date=2006-08-16 |accessdate= |deadurl=yes |archiveurl=https://web.archive.org/web/20131224120030/http://www.jmnews.com.cn/c/2006/08/16/20/c_866425.shtml |archivedate=2013-12-24 }}</ref>。每年开支达600万元。

2009年2月22日，华仔天地[[中国大陆|中国大陆]]分会——'''上海"华仔天地"影音文化发展促进会'''，在[[上海|上海]][[东方艺术中心|东方艺术中心]]举行成立大典，这是中国首个经政府部门批准的社会团体组织，刘德华也成为在内地创立影音社团机构的第一位香港艺人<ref>{{cite web|url=http://ent.hunantv.com/y/l/20090222/209775.html | language= | title=“华仔天地”上海分会成立大典 歌迷力挺刘德华 |publisher=金鹰音乐 |date=2009-02-22 |accessdate= }}</ref>。為配合2014年華仔天地的發展策略及資源整合，自2013年12月15日起，上海“華仔天地”影音文化發展促進會停止運作及服務。

会员可通过华仔天地官方网站，了解到华仔最新动向及华仔天地的信息。

=== 主要活动 ===
华仔天地每年举办两次大型活动——华仔天地'''周年纪念日'''（618）<ref>{{cite web |url=http://www.yn.xinhuanet.com/ent/2008-06/20/content_13598532.htm |language= |title=刘德华"一家"庆华仔天地成立20周年 |publisher=新华网 |date=2008-06-20 |accessdate= |deadurl=yes |archiveurl=https://web.archive.org/web/20150402151408/http://www.yn.xinhuanet.com/ent/2008-06/20/content_13598532.htm |archivedate=2015-04-02 }}</ref>与华仔'''生日会'''（927）<ref>{{cite web|url=http://ent.qianlong.com/4543/2008/09/27/2500@4680340_2.htm | language= | title=刘德华生日会 举办“华仔牛一家聚歌舞乐烧摇” |publisher=千龙娱乐 |date=2008-09-27 |accessdate= }}</ref><ref>{{cite web|url=http://ent.163.com/photoview/00AJ0003/37301.html | language= | title=刘德华生日会父母现身 |publisher=网易娱乐图集 |date=2010-09-26 |accessdate= }}</ref><ref>{{cite web |url=http://ent.oeeee.com/a/20090923/353866.html |language= |title=刘德华生日会朱丽倩依旧无影 歌迷不想见刘太 |publisher=中国新闻网 |date=2009-09-23 |accessdate= |deadurl=yes |archiveurl=https://web.archive.org/web/20120212090856/http://ent.oeeee.com/a/20090923/353866.html |archivedate=2012-02-12 }}</ref><ref>{{cite web |url=http://news.xinhuanet.com/audio/2003-09/16/content_1083384.htm |language= |title=刘德华庆祝42岁生日 千名歌迷参加拍照会(组图) |publisher=东方网 |date=2003-09-16 |accessdate= |deadurl=yes |archiveurl=https://web.archive.org/web/20060509175941/http://news.xinhuanet.com/audio/2003-09/16/content_1083384.htm |archivedate=2006-05-09 }}</ref>，到时上千名来自世界各地的会员聚集在一起，与华仔一起欢度难忘时刻。

会员经常透过本会不时举行各种'''聚会'''，如会员生日会、海外梦想成真活动等，与华仔零距离接触。

为发扬华仔好学不倦的精神，天地会也举行如[[书法|书法]]、[[太极|太极]]、[[编织|编织]]、[[保龄球|保龄球]]<ref>{{cite web|url=http://pic.people.com.cn/GB/31655/10681996.html | language= | title=刘德华生日会惊喜不断 电话传情狂拼保龄球 |publisher=中国新闻网 |date=2009年12月30日 |accessdate= }}</ref>等各种'''兴趣班'''的学习与比赛，以让会员们陶冶情操、增进知识。

此外，华仔天地更积极带领会员参加社会'''[[公益活动|公益活动]]'''，为有需要的社群送上温暖与关怀。

== 会址 ==
香港总部现设于[[九龙湾|九龙湾]][[宏光道|宏光道]]，分为接待处、休闲区、演艺区和展览区（展出华仔历年所获得的逾百个各种奖项），以为华仔和会员们提供一个宽敞舒适的相聚之地<ref>{{cite web|url=http://www.dianping.com/shop/4724179/photos | language= | title=华仔天地图片 |publisher=大众点评网（子小立习习 上传） |date=2011-03-22 |accessdate= }}</ref>。
