[[File:Cube-vertex-figure-middle.svg|thumb]]
[[Image:Triangular_prism_vertfig.svg|thumb]]的頂點圖是一個[[等腰三角形|等腰三角形]]。其中等腰[[三角形|三角形]]的底邊位於三角柱的三角形面，另外兩腰位於[[正方形|正方形]]面上。為了要方便表達這個頂點圖的性質，我們可以使用{{link-en|頂點佈局|Vertex configuration}}符號3.4.4表是，其代表頂點圖的等腰三角形其中一條邊來自來源[[多面體|多面體]]的[[三角形|三角形]]、兩條來自來源多面體的正方形。]]
在[[幾何學|幾何學]]中，'''頂點圖'''是一種用於描述[[幾何圖形|幾何圖形]]之[[頂角|頂角]]特性的方式，大致上是將一個幾何圖形角被切去時所露出的[[形狀|形狀]]<ref name="Math_world">{{Mathworld | urlname = VertexFigure | title = Vertex figure}}</ref>。

== 定義 ==
先從[[多面體|多面體]]上選一個[[頂點|頂點]]，將該頂點的連出去的邊所連接到的頂點標記起來，將這些標記跨越相鄰面連接起來，這些線形成完整的一周，也就是一個環繞著該頂點的[[多邊形|多邊形]]，這個[[多邊形|多邊形]]即為該多面體的頂點圖<ref>[[哈罗德·斯科特·麦克唐纳·考克斯特|H. S. M. Coxeter]], ''Regular Polytopes'', Hbk (1948), ppbk (1973).</ref>。

== 正幾何圖形 ==
[[Image:Great_icosahedron_vertfig.png|left]]的頂點圖是五角星，可以用[[施萊夫利符號|施萊夫利符號]]計為{5/2}]]
若一個[[幾何圖形|幾何圖形]]是[[正圖形|正圖形]]，其[[原像_(幾何)|本身]]、[[胞|胞]]和頂點圖就都能夠使用[[施萊夫利符號|施萊夫利符號]]表示。

[[正圖形|正圖形]]的[[施萊夫利符號|施萊夫利符號]]一般會寫成 {''a'',''b'',''c'',...,''y'',''z''} 的形式，胞為 {''a'',''b'',''c'',...,''y''}，頂點圖則可以表示為 {''b'',''c'',...,''y'',''z''}。
#[[正多面體|正多面體]]在施萊夫利符號中計為{''p'',''q''}，其頂點圖就是一個[[正多邊形|正q邊形]]，在施萊夫利符號中計為{''q''}。
#*舉例來說，[[立方體|立方體]]在施萊夫利符號中計為 {4,3}，其頂點圖是[[正三角形|正三角形]]，在施萊夫利符號中計為 {3}。
#{{link-en|四維正圖形|regular 4-polytope}}或[[堆砌_(幾何)|三維空間填充]]在施萊夫利符號中計為{''p'',''q'',''r''}，其頂點圖在施萊夫利符號中就計為{''q'',''r''}.
#*舉例來說，[[超立方體|超立方體]]在施萊夫利符號中計為{4,3,3}，其頂點圖是[[正四面體|正四面體]]，在施萊夫利符號中計為{3,3}。
#*同樣的，立方體堆砌的施萊夫利符號為{4,3,4}，其頂點圖是施萊夫利符號計為{3,4}的[[正八面體|正八面體]]。

== 棱圖 ==
[[File:Truncated_cubic_honeycomb1.jpg|thumb]]的公共稜。可以以2個稜圖來表示，也就是說其頂點圖的頂點圖有兩種可能。]]
稜圖是頂點圖的頂點圖<ref>{{cite web | title = Klitzing:Vertex figures,etc. | url = http://www.bendwavy.org/klitzing/explain/verf.htm}}</ref>，可用於描述幾何圖形[[邊|稜]]的角（在[[三維空間|三維空間]]中可理解為[[二面角|二面角]]）的特性。

往更高的維度推廣，還有面圖、胞圖，面圖用於描述幾何圖形的四維面與面的交角，可以理解為堆砌體中，面與面接合的部分，雖然三維的面與面交會的部分都是[[平角|平角]]，但到四維空間就可以存在角度，類似二面角那樣，到五維空間就會需要類似頂點圖的面圖來描述其結構（類似於[[正多邊形鑲嵌|正多邊形鑲嵌]]的多邊形與多邊形棱的交會部分，因為是在平面上，因此這個二面角當然會是平角，但到了[[三維空間|三維空間]]，這種角就會出現角度、[[四維|四維]]以上就會有不止兩個圖形交會於此，因此需要棱圖來描述）。其他更高維度還有胞圖、n維胞圖等。

依此概念繼續推廣還有面圖、胞圖......以此類推。他們用來描述高維度的幾何體對應元素的結構。
