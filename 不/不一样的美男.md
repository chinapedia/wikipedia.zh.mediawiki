《'''不一样的美男'''》（{{lang-en|Lady Boy Friends}}；{{lang-th|เพื่อนกันมันส์ดี}}）是[[泰国|泰国]]的[[校园|校园]]题材[[电视剧|电视剧]]<ref>{{cite web |url= http://tv.mthai.com/digital-tv/88562.html|title= ซีรีส์แปลกที่แหวกขนบธรรมเนียม Lady Boy Friends เพื่อนกันมันส์ดี|author= |date= |website= mthai|publisher= |access-date= |quote=}}</ref>，由[[伍迪·古叻瓦|伍迪·古叻瓦]]（Wuddy Kullawas）导演；[[UTC+7|泰国时间]]2015年2月14日于{{link-th|Bright TV|ไบรต์ทีวี}}频道首播。在中国由[[天府泰剧|天府泰剧]]译制<ref>{{Cite web |url=http://www.bilibili.com/sp/%E4%B8%8D%E4%B8%80%E6%A0%B7%E7%9A%84%E7%BE%8E%E7%94%B7 |title=不一样的美男_专题_bilibili_哔哩哔哩弹幕视频网 |access-date=2018-10-05 |archive-url=https://web.archive.org/web/20171209095014/http://www.bilibili.com/sp/%E4%B8%8D%E4%B8%80%E6%A0%B7%E7%9A%84%E7%BE%8E%E7%94%B7 |archive-date=2017-12-09 |dead-url=yes }}</ref>，[[上海乐泰文化传媒|上海乐泰文化传媒]]代理。

第2季于2015年9月19日播出<ref>[https://web.archive.org/web/20151024043201/http://www.brighttv.co.th/th/program/Lady-Boy-Friends-%E0%B9%80%E0%B8%9E%E0%B8%B7%E0%B9%88%E0%B8%AD%E0%B8%99%E0%B8%81%E0%B8%B1%E0%B8%99%E0%B8%A1%E0%B8%B1%E0%B8%99%E0%B8%AA%E0%B9%8C%E0%B8%94%E0%B8%B5-season-2 Lady Boy Friends เพื่อนกันมันส์ดี season 2]</ref>。

== 剧情简介 ==
该剧以泰国外府的一所重点[[高级中学|高级中学]]“古叻瓦第一学院学校”（{{lang-th|โรงเรียนกุลวัสส์วิทยาลัย 1}}，简称{{lang|th|ก.ว.1}}）为背景，讲述的是来自泰国全国各地的学生和[[跨性別者|跨性別者]]之间发生的故事<ref>{{cite web |url= http://mt.sohu.com/20150305/n409351406.shtml|title=朋友圈都说泰国版的《不一样的美男》是今年必看神剧！ |author=视觉艺术|date= 2015-03-05|website= 搜狐|publisher= |access-date= 2017-09-23|quote=}}</ref>。一个新学期的开始，学校计划创建一个实验班，高一13班属于特殊班级；该班共有12名男生和8名跨性別者<ref>{{cite web |url= http://th.hujiang.com/new/p737245/|title= 年中福利：2015年好看泰剧大盘点|author=沪江泰语 |date=2015-07-03|website=沪江泰语|publisher= |access-date=2017-09-23|quote=}}</ref>。

==拍摄==
=== 第一季 ===
该剧实际为导演在早年学生时期时的故事，属于半[[自传|自传]]作品。其中的角色Frank性格和经历与导演相似。

第一季试拍于2014年10月16日<ref>{{cite web |url= https://www.instagram.com/p/BaYQMNLF3WT/|title= 16 Oct 2014 The first for shooting production “Lady Boy Friends The Series เพื่อนกันมันส์ดี” We miss every moments in that day. |author= ladyboyfriends|date= 2017-10-18|website=Instagram|publisher= |access-date=2017-10-19 |quote=}}</ref>，后于2014年11月24日开拍，演员大多为泰国各地大学学生。New和Peet来自曼谷[[阿他威商业技术学院|阿他威商业技术学院]]（{{lang|th|วิทยาลัยเทคโนโลยีอรรถวิทย์พณิชยการ}}），Frank、Tiw、Wut和Boss来自[[孔敬大学|孔敬大学]]，而Jet、Van和八位跨性别者学生等演员则来自[[玛哈沙拉堪大学|玛哈沙拉堪大学]]。

《不一样的美男》第一季播出当时正逢[[2015年诗琳通公主60寿辰|2015年诗琳通公主60寿辰]]，因此以标清信号播出时屏幕左上方有纪念公主寿辰的标志。

=== 第二季 ===
第二季于2015年9月开播<ref>{{cite web|url= http://tcctv.co.th/event/%E0%B8%9A%E0%B8%A7%E0%B8%87%E0%B8%AA%E0%B8%A3%E0%B8%A7%E0%B8%87%E0%B9%80%E0%B8%9B%E0%B8%B4%E0%B8%94%E0%B8%81%E0%B8%A5%E0%B9%89%E0%B8%AD%E0%B8%87%E0%B8%A5%E0%B8%B0%E0%B8%84%E0%B8%A3-lady-boy-friends-th/|title= บวงสรวงเปิดกล้องละคร Lady Boy Friends The Series2 เพื่อนกัน มันส์ดี season 2|author= |date= |website= 泰国中央电视台|publisher= |access-date= 2015-09-23|quote= |archive-url= https://web.archive.org/web/20170313143924/http://tcctv.co.th/event/%E0%B8%9A%E0%B8%A7%E0%B8%87%E0%B8%AA%E0%B8%A3%E0%B8%A7%E0%B8%87%E0%B9%80%E0%B8%9B%E0%B8%B4%E0%B8%94%E0%B8%81%E0%B8%A5%E0%B9%89%E0%B8%AD%E0%B8%87%E0%B8%A5%E0%B8%B0%E0%B8%84%E0%B8%A3-lady-boy-friends-th/|archive-date= 2017-03-13|dead-url= yes}}</ref>，但New的演员差亚通因学业原因而没有出演，只有以[[客串|客串]]形式在一两个场景中出现。

第二季原播出集数为24集，后由于审查原因而终止拍摄，导致第二季烂尾只播出前12集。[[UTC+7|泰国时间]]2017年4月1日晚上20:00，官方公布《不一样的美男》2.2季{{NoteTag|此前坊间流传的第三季其实是2.2季。}}，即之前第二季的未播出片段，于[[Facebook|Facebook Live]]和[[YouTube|YouTube]]等网络平台播放；2017年4月29日，《不一样的美男》2.2季因剧情和拍摄原因终结，只播出了4集<ref>{{cite web |url= https://www.facebook.com/LadyBoyFriendsTheSeries/photos/a.612916898838344.1073741833.556505651146136/1043257095804320/?type=3|title= สวัสดีครับทุกๆคน นี่พี่ Wuddy Kullawas...|author=Lady Boy Friends The Series |date= 2017-04-29|website= Facebook|publisher= |access-date= 2017-09-23|quote=}}</ref>。

=== 番外和电影版 ===
2016年9月29日与BeautiFarm（美丽农场）美容霜合作，公布其20分钟的中文字幕特别篇<ref>{{cite web |url= https://www.youtube.com/watch?v=_VE1MaOlnYs|title= Lady Boy Friends The Series Special Version |author= |date= |website=YouTube |publisher= |access-date= |quote=}}</ref>，另外也有4分26秒的短版本<ref>{{cite web |url= https://www.youtube.com/watch?v=9GLe8EvjXfk|title= Lady Boy Friends The Series Beauti Farm Version |author= |date= |website=YouTube |publisher= |access-date= |quote=}}</ref>；除New、Frank、Peet、Bell、Jeednie等几位老演员外，其他演员均为新人。

2016年8月28日宣布拍摄电影版；2017年1月16日于{{link-th|暹罗技术学院|วิทยาลัยเทคโนโลยีสยาม}}举行开机仪式，公布其名为《[[Destiny's_Boy|Destiny's Boy]]》。<ref>{{cite web |url= https://www.facebook.com/LadyBoyFriendsTheSeries/photos/a.612916898838344.1073741833.556505651146136/971755229621174/?type=3|title= บวงสรวงเปิดกล้องภาพยนตร์ Destiny's Boy...|author=Lady Boy Friends The Series |date= 2017-01-16 |website= Facebook |publisher= |access-date= 2017-09-23|quote=}}</ref>2017年10月16日，官方将《不一样的美男》电影版定于2018年上映，结果不明原因取消拍摄上映<ref>{{cite web |url= https://www.instagram.com/p/BaSLLDsF4K2/|title= Hi everyone!!! See you on 2018 |author= ladyboyfriends|date= 2017-10-16|website=Instagram|publisher= |access-date=2017-10-16 |quote=}}</ref>。

=== 取景地 ===
该剧拍摄取景地位于[[泰国|泰国]][[曼谷|曼谷]][[叻甲挽縣|叻甲挽縣]][[拉達那戈信頌朴叻甲邦學校|拉達那戈信頌朴叻甲邦學校]]（{{lang-en|Rattanakosin Somphot Lat Krabang School}}，{{lang-th|โรงเรียนรัตนโกสินทร์สมโภชลาดกระบัง}} {{coord|13.760084|100.731052}}）。在剧中角色们所读学校称为“古叻瓦第一学院学校”（{{lang-th|โรงเรียนกุลวัสส์วิทยาลัย 1}}）<ref>{{cite web |url= https://www.facebook.com/LadyBoyFriendsTheSeries/posts/567259440070757|title=4/13 โรงเรียนกุลวัสส์วิทยาลัย 1 (ก.ว.1)...  |author=Lady Boy Friends The Series |date= 2014-10-24|website= Facebook|publisher= |access-date=2017-09-23 |quote=}}</ref>，其中学校名称“{{lang|th|กุลวัสส์}}”源于导演的名字{{lang|th|'''กุลวัสส์''' เทพไพฑูรย์}}。

剧中学校的名字简称为{{lang|th|ก.ว.1}}，该校学生的衬衫右胸处均绣有此红色的[[泰文|泰文]][[缩写|缩写]]。

==迴響和評價==
==={{CHNML}}===
中文翻譯版由[[天府泰劇|天府泰劇]]（现“喜翻译制组”）製作，[[Blued_(應用程式)|Blued]]赞助，於彈幕視頻分享網站[[Bilibili|Bilibili]]发布；其翻譯內容逗趣和搞笑的台詞遭網友吐槽<ref>{{cite web |url= http://binzhou.dzwww.com/yvle/201502/t20150228_11956221.htm|title= 泰劇不一樣的美男雷哭網友：太duang了！ |author=前瞻网 |date=2015-02-28 |website=滨州网 |publisher= |access-date= 2017-09-23|quote=}}</ref>。同時，劇中部份演員誇張的表情、動作，再加上大量男同志間的劇情，閃瞎網友<ref>{{cite web |url=http://slide.ent.sina.com.cn/tv/w/slide_4_704_103379.html#p=1 |title= 泰劇《不一樣的美男》基情四射 閃瞎網友|author= |date= 2015-03-01|website= 新浪图片|publisher= |access-date= 2017-09-23|quote=}}</ref>。此外劇中有一重大突破就是沒有女主角，取而代之為出現帥哥與帥哥之間多對的愛情關係（例如Jet和New），三角戀（例如Tiw，Frank和Wut）和跨性別同學們的小圈子和對罵，再配合各種重口味和撕逼台詞，被中國內地網友評為大量金句和高能的劇集。同時，部份內地網友認為此劇背景音樂重複情況較為嚴重，場景不夠多元化，故事的主線，分支不夠明顯。<ref>{{cite web |url= http://ent.ifeng.com/a/20150310/42273114_0.shtml|title= 開扒高能神劇《不一樣的美男》|author=凤凰网娱乐 |date=2015-03-10 |website= 凤凰娱乐|publisher= |access-date= 2017-09-23|quote=}}</ref>

天府泰剧曾于2014年12月3日发布了一条关于该剧的微博，当时暂译名为《我的朋友是人妖》；由于剧名中含有“人妖”一词不被广泛接受，经投票后最终剧名为《不一样的美男》<ref>{{cite web |url= http://ent.163.com/15/0313/11/AKJ72TH800031GVS.html |title= 专访泰剧《美男》字幕组 看神翻译如何震碎三观|author= |date= 2015-03-13|website= 网易娱乐|publisher= |access-date= 2017-09-23|quote=}}</ref>。
