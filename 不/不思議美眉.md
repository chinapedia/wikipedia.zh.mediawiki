《'''不思議美眉'''》（{{lang-ja|不思議なソメラちゃん}}）是由{{tsl|ja|ちょぼらうにょぽみ|Choborau-nyopomi}}創作的[[日本|日本]][[四格漫畫|四格漫畫]]作品，連載於《{{tsl|ja|まんがぱれっとLite|漫畫調色盤Lite}}》（[[一迅社|一迅社]]）vol.6（2008年8月）至vol.38（最終號，2011年4月），[[單行本|單行本]]共2冊。

2015年4月宣佈製作電視動畫，並於同年10月至12月播出。此外，時隔四年的新續篇{{Nihongo|《'''不思議美眉 Haute Couture'''》|不思議なソメラちゃん オートクチュール}}也在《{{tsl|ja|まんが4コマぱれっと|漫畫四格調色盤}}》2015年6月號開始連載<ref>{{Cite web|url=http://natalie.mu/comic/news/145024|script-title=ja:ちょぼらうにょぽみ「不思議なソメラちゃん」続編始動！TVアニメ化決定で|date=2015-04-22|publisher={{lang|ja|コミックナタリー}}|accessdate=2016-11-10|language=ja}}</ref>。

== 故事簡介 ==
繼承了去世雙親的「野乃本魔法拳」的[[尼特族|尼特族]]少女野乃本索梅拉為了滿足眼前的慾望，將妹妹庫庫露和朋友天童雫等人牽連其中，引發各式各樣離奇怪異的事件。

== 登場人物 ==
; {{Nihongo|野乃本索梅拉|野乃本 ソメラ（ののもと ソメラ）|聲：[[仲谷明香|仲谷明香]]}}
: 本作主角，尼特族少女，是善於使用結合了[[魔法|魔法]]與[[拳法|拳法]]的「野乃本魔法拳」的高手，為了慾望與生活費（僅限能輕鬆取得的）而活著。

; {{Nihongo|野乃本庫庫露|野乃本 ククル（ののもと ククル）|聲：[[本渡楓|本渡楓]]}}
: 索梅拉的妹妹，比姐姐端莊乖巧，但也因此經常被姐姐使喚。

; {{Nihongo|天童雫|天童 雫（てんどう しずく）|聲：[[桃河Rika|桃河Rika]]<ref group="註">此角色在《[[漫研社|漫研社]]》中由茅野愛衣配音。</ref>}}
: 索梅拉的朋友和鄰居，頭腦程度和索梅拉差不多低。

; {{Nihongo|松嶋愛|松嶋 あい（まつしま あい）|聲：[[長繩麻理亞|長繩麻理亞]]}}
: 尋找失蹤的姐姐的少女。

; {{Nihongo|好實|コノミ|聲：[[茅野愛衣|茅野愛衣]]}}
: 庫庫露的朋友（自稱），如同跟蹤狂般地糾纏庫庫露。

; {{Nihongo|梅努斯|メヌースー}}
: 不可思議的生物。動畫於開頭有一小段介紹梅努斯的環節（由[[高橋花林|高橋花林]]擔任解說），聲優詳見[[#各話列表|各話列表]]。

; {{Nihongo|蛯原愛、麻衣、咪妮亞·梅妮希、波乃香前輩|蛯原愛、麻衣、ミィニャ・メーニッヒ、ぽのか先輩}}
: 同作者的漫畫《[[漫研社|漫研社]]》中的角色，以背景人物的形式登場。動畫中僅在第6話登場過一次，聲優同《漫研社》。

== 電視動畫 ==
以{{Nihongo|《Choborau-nyopomi劇場－新章－ 不思議美眉》|ちょぼらうにょぽみ劇場-新章- 不思議なソメラちゃん}}為題，於2015年10月至12月播出，為每話5分鐘的[[泡面番|短篇動畫]]。

此動畫的製作人員大致上與同作者的《[[漫研社|漫研社]]》相同，此外亦由{{tsl|ja|向清太朗}}持續擔任旁白及聲優。
