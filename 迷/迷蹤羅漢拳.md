'''迷蹤羅漢拳'''，是[[北少林|北少林]]迷蹤派支流。

== 歷史 ==
迷蹤羅漢拳從前只有家傳，自[[葉雨亭|葉雨亭]]開始公開傳授於河北省滄縣、南皮兩地。先後受聘於北平九門提督[[王淮慶|王淮慶]]、[[張學良|張學良]]及山東督軍[[張宗昌|張宗昌]]等，成為軍中總教練。三十年代南下上海精武，數年後轉聘為香港南華體育會少林拳班教師。抗戰時期，葉雨亭轉到廣州灣，任教西營新武堂。戰後，南華會員力懇回港復任。

== 拳術 ==
[[鐵腳羅漢拳|鐵腳羅漢拳]]、[[六合羅漢拳|六合羅漢拳]]、[[韋陀羅漢拳|韋陀羅漢拳]]、[[長拳|長拳]]、[[唐拳|唐拳]]、[[笑和緣|笑和緣]]、[[抹面拳|抹面拳]]、[[同背拳|同背拳]]、[[袍拳|袍拳]]等。

== 兵器 ==
[[虎頭雙鉤|虎頭雙鉤]]、[[緹袍劍|緹袍劍]]、[[斬馬刀|斬馬刀]]、[[梅花槍|梅花槍]]、[[綑羊棍|綑羊棍]]、[[雪花刀|雪花刀]]等。

== 對拆 ==
四攻四守、雙鉤進槍、單刀枴子進槍等。
