《'''迷糊軟網社'''》（{{lang-ja|そふてにっ}}）是[[あづち涼|あづち涼]]的一部以[[軟式網球|軟式網球]]為題材的漫畫作品<ref>{{cite web|title=北國的網球女孩們前進大賽《迷糊軟網社》將推出電視動畫作品|url=https://gnn.gamer.com.tw/5/48095.html|website=[[巴哈姆特電玩資訊站|巴哈姆特電玩資訊站]]|language=zh-hant|date=2010年10月28日}}</ref>，於月刊漫畫雜誌《[[月刊Comic_BLADE|月刊Comic BLADE]]》2008年10月号到2013年1月号連載，單行本由[[Mag_Garden|Mag Garden]]發售共8冊，中文版由[[東立出版社|東立出版社]]代理發售。<ref>{{cite web|title=Sofuteni (manga)|url=http://www.animenewsnetwork.com/encyclopedia/manga.php?id=12114|website=[[Anime_News_Network|Anime News Network]]|language=en}}</ref>2011年4月8日開始播放電視動畫共12集。<ref>{{cite web|title=Softenni! (TV)|url=http://www.animenewsnetwork.com/encyclopedia/anime.php?id=12115|website=Anime News Network|language=en}}</ref>

== 故事 ==
白玉初中女子軟網社的春風明日菜、澤夏琴音、秋山千歳、冬川来-{栖}-，還有新加入的留學生伊莉莎白・華倫，一起揮灑青春，以全國大賽為目標的歡樂生活就此展開了。

== 登場角色 ==
=== 白玉初中 ===
; {{nihongo|春風 明日菜|春風 明日菜（はるかぜ あすな）|聲優：[[伊藤加奈惠|伊藤加奈惠]]}}
:  白玉初中軟網社的社員，二年級，身高156cm，體重46kg，血型為B型。對於日常中的生活對話，常常會聯想到色色的方面，也就是有妄想的癖好。喜歡草莓牛奶，家裡是開牧場，網球實力普通，雙打時和澤夏琴音一組。

; {{nihongo|澤夏 琴音|澤夏 琴音（さわなつ ことね）|聲優：[[喜多村英梨|喜多村英梨]]}}
:  白玉初中軟網社的社員，二年級，身高156cm，體重48kg，血型為A型，髮型為雙馬尾。擅長空手道，家裡是空手道場，小學時曾在大會無傷三連霸，被稱為十年才出現一次的天才，被三島木以「不向別的更高目標前進嗎？」的話加入軟網社，運動神經很好，但是控球力很差，在合宿後有了進步。有著男孩子般的個性，喜歡軟網社顧問三島木，但是無法坦率的表現出來，看到三島木和伊莉莎白友好的時候會顯現出嫉妒的一面。
:雙打時和春風明日菜一組。

; {{nihongo|秋山 千歲|秋山 千歳（あきやま ちとせ）|聲優：[[伊藤靜|伊藤靜]]}}
:  白玉初中軟網社的社長，三年級，身高162cm，體重55kg，血型為A型，雙打與冬川來-{栖}-一組。

; {{nihongo|冬川 來-{栖}-|冬川 来栖（ふゆかわ くるす）|聲優：[[明坂聰美|明坂聰美]]}}
: 白玉初中軟網社的社員，一年級，擁有小麥色的膚色，實力不錯，雙打與秋山千歳一組。

; {{nihongo|伊莉莎白・華倫|エリザベス・ウォーレン|聲優：[[矢作紗友里|矢作紗友里]]}}
: 來自英國的留學生，目前加入白玉初中軟網社，二年級，小學時是打硬式網球，擁有全國大賽優勝的實力，在三島木去英國時認識三島木，從此便懷有好感，會說日語，但實際上相當缺乏日本的常識。

; {{nihongo|三島木 洋|三島木 洋（みしまぎ よう）|聲優：[[松本忍|松本忍]]}}
:  白玉初中軟網社的顧問，28歲，相當懶散，但是網球實力高強，曾經有過高中軟網全國三連霸的紀錄，也得過不少獎盃，喜歡吃辣，有一台[[速霸陸|速霸陸]]ALCYONE。

; {{nihongo|西岡 紫希|西岡 紫希（みしまぎ よう）|聲優：[[高垣彩陽|高垣彩陽]]}}
:  白玉初中的畢業生，也是軟網社的前社長，是三島木洋的表妹，在學中雙打是和秋山千歳一組。

=== 紅玉初中 ===
; {{nihongo|木場 澄乃|木場 澄乃（きば すみの）|聲優：[[井上麻里奈|井上麻里奈]]}}
:  女子軟網社的社員，有著男孩子般的性格，視琴音為對手，要在網球上分出勝負。雙打和雨宮芹葉一組。
; {{nihongo|雨宮 芹葉|雨宮 芹葉（あまみや せりは）|聲優：[[菊池心|菊池心]]}}
:  女子軟網社的社長，有著劉海遮住左眼的髮型，雙打和木場澄乃一組。

== 網路廣播 ==
