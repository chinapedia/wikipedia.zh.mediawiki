'''麟山郡'''（{{kor|諺=린산군|漢=麟山郡|羅=Rinsan gun|f=y}}）是[[朝鮮民主主義人民共和國|朝鮮民主主義人民共和國]][[黃海北道|黃海北道]]西南部的一個郡，南隔[[滅惡山脈|滅惡山脈]]與[[黃海南道|黃海南道]]相連。2008年人口73,626人。<ref>{{cite web |author=[[朝鮮中央統計局|朝鮮中央統計局]] |url=http://unstats.un.org/unsd/demographic/sources/census/2010_PHC/North_Korea/Final%20national%20census%20report.pdf |trans_title=2008年人口普查結果 |title=D P R Korea 2008 Population Census: National Report |year=2009年 |page=27}}</ref>下分1邑、19里。

== 地理 ==

北側是[[瑞興郡|瑞興郡]]和[[鳳山郡_(朝鮮)|鳳山郡]]，東側是[[平山郡|平山郡]]，西側是[[銀波郡|銀波郡]]和屬於黃海南道的[[新院郡|新院郡]]，南側是黃海南道的[[鳳川郡|鳳川郡]]和[[青壇郡|青壇郡]]。

== 歷史 ==
1952年行政區域改編時，從[[平山郡|平山郡]]和[[瑞興郡|瑞興郡]]的部份地域分離成為今日的麟山郡。
