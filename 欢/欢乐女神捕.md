《'''欢乐女神捕'''》，原名:欢乐无双，中国大陆的古装剧，导演钟澍佳，主演江铠同、熊乃瑾、翟天临、魏千翔。<ref>{{cite web |url=http://ent.qq.com/a/20140812/052258.htm |title=《欢乐无双》开启新风格喜剧 上演中国破产姐妹 |accessdate= 2014|publisher=腾讯网 |work= |language=zh-hans}}</ref><ref>{{cite web |url=http://ent.sina.com.cn/v/m/2014-03-11/10234109451.shtml |title=《欢乐无双》熊乃瑾变身毒舌女神捕 |accessdate=2014 |publisher=新浪网 |work= |language=zh-hans}}</ref><ref>{{cite web |url= http://ent.sina.com.cn/v/m/2014-03-12/11164110146.shtml|title= 《欢乐无双》欢乐开机 翟天临首演喜剧|accessdate=2014 |publisher=新浪网 |work= |language=zh-hans}}</ref>2014年3月开拍，6月杀青。於2018年6月6日在泰国電視台thairat tv 播出

==剧情==
该剧讲述“陆小欢”等[[明朝|明朝]]捕快的破案故事。
