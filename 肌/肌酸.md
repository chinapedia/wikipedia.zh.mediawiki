'''肌酸'''（{{lang-en|'''creatine'''}}），在[[生物化学|生物化学]]中，是一种自然存在于[[脊椎动物|脊椎动物]]体内的一种含氮的[[有机酸|有机酸]]，能够辅助为[[肌肉|肌肉]]和[[神经|神经]][[细胞|细胞]]提供能量。[[米歇尔·欧仁·谢弗勒尔|米歇尔·欧仁·谢弗勒尔]]于1832年首次在[[骨骼肌|骨骼肌]]中发现肌酸，而后，根据[[希腊语|希腊语]]「{{lang|el|κρέας}}」（{{lang|la|kreas}}，肉），命名为“{{lang|en|creatine}}”。

== 作用 ==
肌酸能够与{{link-en|磷酸肌酸|Phosphocreatine}}相互转化，因此可以作为作用于很多[[无脊椎动物|无脊椎动物]]体内的基于[[精氨酸|精氨酸]]和[[磷酸精氨酸|磷酸精氨酸]]的一个系统的一部分。由于这个能量运输器的存在，使得[[三磷酸腺苷|ATP]]与[[ADP|ADP]]的比例保持在一个很高的水平。这样就既保证了ATP中的自由能在一个很高的水平，又使得[[腺苷|腺苷]]的流失最小化，而腺苷流失可能会导致细胞功能紊乱。这样一个高能的磷酸[[缓冲溶液|缓冲溶液]]也被叫做[[磷酸原|磷酸原]]。

== 合成 ==
在人体内，肌酸主要在[[肝|肝]]中利用三个不同氨基酸合成，这三个[[氨基酸|氨基酸]]包括[[精氨酸|精氨酸]]、[[甘氨酸|甘氨酸]]和[[蛋氨酸|蛋氨酸]]。合成后95%的肌酸被贮存于骨骼肌中，剩下的存在于[[大脑|大脑]]、[[心脏|心脏]]和[[睾丸|睾丸]]中。

[[File:Creatine_biosynthesis.svg|thumb]]

GATM（甘氨酸脒基转移酶）也被叫做L-精氨酸（AGAT），{{EC number|2.1.4.1}}。它是一个线粒体酶，主要负责催化肌酸的生物合成中的第一个限速步骤，主要在肾脏中表达。

途径中的第二个酶，即GAMT, 胍基乙酸N-甲基转移酶，{{EC number|2.1.1.2}}，主要在肝脏中表达。

肌酸生物合成途径中的基因缺失会导致多种严重的神经学缺陷。

== 来源 ==
对于人类，大约体内贮存的一半的肌酸都是从食物中获取的（主要来源是[[肉|鲜肉]]和[[鱼|鲜鱼]]）。因为[[蔬菜|蔬菜]]不含肌酸，而肌肉肌酸含量随着肌酸摄入的增多而增多，所以[[素食主义|素食主义者]]通常比非素食主义者的肌肉肌酸含量低。

== 肌酸与肌肉疾病的治疗 ==
补充肌酸一直以来都被认为是治疗肌肉性、神经性或神经肌肉性疾病的一个可行的办法。包括治疗[[关节炎|关节炎]]、[[慢性充血性心力衰竭|慢性充血性心力衰竭]]、[[废用性萎缩|废用性萎缩]]、[[环状萎缩|环状萎缩]]、[[肌肉磷酸化酶缺乏症|肌肉磷酸化酶缺乏症]]、[[亨廷顿病|亨廷顿病]]、[[多样性神经肌肉疾病|多样性神经肌肉疾病]]、[[线粒体疾病|线粒体疾病]]、[[肌肉营养失调|肌肉营养失调]]和[[神经保护|神经保护]]等。

两个科学研究已经显示肌酸可能对神经肌肉紊乱有益。第一，[[康奈尔大学|康奈尔大学]]医学中心的M. Flint Beal，一个由MDA赞助的研究者，展示了在延长患有[[肌肉萎縮性側索硬化症|肌肉萎縮性側索硬化症]]的老鼠的肝寿命方面，肌酸的效果是现行处方药riluzole的两倍。他还认为肌酸之所以会对患有肌萎缩性侧索硬化症的老鼠的有着神经保护作用主要是两个原因，其一是受伤的神经细胞可用的能量增大，其二是阻断了导致细胞死亡的化学途径。

第二，一个由[[麦克马斯特大学|麦克马斯特大学]]医学中心两名加拿大籍研究者Mark Tarnopolsky和Joan Martin做的研究发现，肌酸可以使得患有多样的神经肌肉紊乱的人群适度的增加力量。最新的论文发表在1999年3月的《神经学》杂志。

== 在健身中的应用 ==
{{main|运动肌酸}}
可以说，肌酸在日常生活中最常见的用途就是健身运动员的训练补充。一般职业或者半职业健身运动员都会服用定量的肌酸来帮助肌肉的生长和提高肌肉的质量。

大部分情况下, 理想的服用时间是在健身之前半小时和健身之后半小时内服用肌酸。

如果当天没有大负荷的力量训练，可以在中午适当服用。

肌酸的使用量是根据体重计算的。在不参与大负荷运动的正常日补充量为：男性服用量为0.7-1.2毫克/磅，女性服用0.5-1.0毫克/磅。

== 副作用 ==
到目前为止还没有发觉可靠的证据来证明額外服用肌酸有很强烈的副作用，因为它本身就是身体分泌的物质之一。如果在过量使用的情况下会造成輕度[[腹泻|腹泻]]。

2015年3月。一篇刊登於[https://en.wikipedia.org/wiki/British_Journal_of_Cancer British Journal of Cancer]的研究發現男性使用含有肌酸的補給品或許會導致較高的睪丸癌發生機率。<ref>http://www.nature.com/bjc/journal/v112/n7/full/bjc201526a.html</ref><ref>http://www.webmd.com/cancer/news/20150416/muscle-building-supplements-cancer</ref>
