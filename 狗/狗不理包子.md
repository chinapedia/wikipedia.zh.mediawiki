'''狗不理包子'''创立于1858年，為[[天津|天津]]著名[[小吃|小吃]]，“'''[[天津三绝|天津三绝]]'''”之首，是[[中华老字号|中华老字号]]之一。在1960年代冷冻出口[[中国|中国]]国外，现在中国各地开有多家分店。

== 历史 ==
狗不理始于[[清朝|清朝]][[咸丰|咸丰]]年间（1858年），由直隶省[[武清县|武清县]]下朱庄人高贵友创始，高贵友乳名叫“狗子”，他自幼居於[[天津|天津]]，在当时[[南运河|南运河]]畔的[[刘家蒸吃铺|刘家蒸吃铺]]做帮工和学徒，高贵友用3年掌握做[[包子|包子]]和其他面食的手艺，然後开办一家名“德聚号”的小吃铺，专营[[包子|包子]]。

由于当时德聚号的包子色香味形俱佳，生意十分兴隆，高贵友忙得顾不上跟顾客说话，因此，很多顾客都戏称他为“狗子卖包子，不理人”。“狗不理”便因此得名，而他所经营的包子也被称作“狗不理包子”<ref>[http://q.sohu.com/forum/20/topic/5422550 天津小吃特色美食]</ref> 。

[[袁世凯|袁世凯]]任[[直隶|直隶]][[总督|总督]]期间在天津编练[[新军|新军]]时，曾把狗不理包子作为贡品进京献给[[慈禧太后|慈禧太后]]。慈禧太后尝后大悦，曰：“山中走兽云中雁，陆地牛羊海底鲜，不及狗不理香矣，食之长寿也。”从此，狗不理包子名声大振，逐渐在许多地方开设了分号。

1920年代，随着天津市商业中心南移并向[[租界|租界]]靠拢，狗不理包子店也搬至[[天津法租界|天津法租界]]的[[天祥市场|天祥市场]]（今[[天津劝业场|天津劝业场]]）附近，但搬迁后生意萧条甚至一度停业。[[中华人民共和国|中华人民共和国]]成立后，很多私人商铺纷纷[[公私合营|公私合营]]。1956年3月15日，天津市国有饮食公司在[[和平区_(天津市)|和平区]]山东路丰泽园饭庄旧址开设了天津狗不理包子铺，此后，[[党和国家领导人|中国国家领导人]]多次光顾<ref>[http://www.scprok.com/scgj/pr200602/001.htm 老字号面临新形势的挑战]</ref>。

== 近况 ==
[[File:Tianjin_lunch_of_Goubuli.jpg|thumb]]和[[煎饼果子|煎饼果子]]]]
*1988年，狗不理包子铺扩建并增加雅间，成为一家规格较高的酒楼。
*1992年，狗不理集团正式组建，成为天津市餐馆业的翘楚。
*2005年，拥有147年历史的天津狗不理包子店，被另外一家百年老店[[同仁堂_(天津)|同仁堂]]以一亿零六百万[[人民币|人民币]]投标并购。
*2008年2月25日，天津狗不理和平路大酒店英文'''“Go Believe”'''的招牌挂到狗不理的店门前，正式成为狗不理的英文店名。此举，一时掀起了老字号起洋名的舆论关注和讨论。不过[[美国|美国]]作家[[西德尼·谢尔顿|西德尼·谢尔顿]]1985年已经在他的名著《{{link-en|If Tomorrow Comes|If Tomorrow Comes (novel)}}》（假如明天来临）中将''Goubuli''，a kind of Chinese dumpling纳入到英文作品中。

== 制作方法 ==
[[File:天津狗不理包子.JPG|缩略图]]
狗不理包子高贵友以3比7的比例将肥瘦鲜猪肉和匀并加上适量的水、排骨汤和肚汤，并佐以[[香油|香油]]、[[酱油|酱油]]、[[薑|薑]]末、[[葱|葱]]末制作成馅。包子皮则将半发面搓条、放剂之后，擀成直径8.5厘米且薄厚均匀的圆形皮。包馅时用手指捏折并褶捻开，使18个褶花疏密一致，最后将包子用硬气蒸制而成。

不加[[酱|酱]]，用[[口蘑|口蘑]][[鸡汤|鸡汤]]和馅，皮用半发面（发面和硬面混合），而且比其他包子小巧，因此口味鲜美，大受欢迎。因为当时制作方法并没有[[专利权|专利权]]，被许多店家模仿，目前已成为天津包子的普通形象，但狗不理的包子更为精益求精。

== 品牌保护 ==

1980年在[[中华人民共和国|中华人民共和国]]恢复[[商标|商标]]管理之初，狗不理就完成了商品商标注册。1992年狗不理服务商标成为国家首批注册的服务商标。目前狗不理商品商标已在11项国际分类中注册，服务商标于1999年被[[国家工商行政管理总局|国家工商行政管理总局]]商标局认定为“[[中国驰名商标|中国驰名商标]]”，这是对其品牌的商业价值的肯定。

== 分店 ==

1980年代开始特许连锁经营。1980年，北京华天饮食集团公司将狗不理引入北京，并在[[地安门|地安门]]开设了狗不理在全国的第1家分店。<ref>[http://news.xinhuanet.com/food/2007-12/19/content_7280048.htm 新路春饭庄]</ref>2004年，狗不理在[[台湾|台湾]]开业。<ref>[http://www.china.com.cn/news/tw/2009-12/01/content_18986999.htm 天津“狗不理”包子进军台湾 明年日月潭开店]</ref>在天津，狗不理集团已拥有酒楼数十个，其中有十余家为直营酒楼，包括：山东路老店、和平路大酒店、水上大酒楼、食品街酒楼、津湾广场店、睦南道贵宾楼、十一经路店、中山路店、滨海大酒楼。此外，狗不理集团还在[[天津|天津]]开设多家狗不理快餐店。

== 媒体宣传 ==
