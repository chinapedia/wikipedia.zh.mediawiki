《'''侍戰隊真劍者'''》（原題：{{lang|ja|侍戦隊シンケンジャー}}<ref>日語讀作 Samurai Sentai ，即為「武士戰隊」之意，所以故事中會提及「武士」而不是「侍」。不過日本漢字的使用法與中國漢字的使用法有一定的差異，若是完全照搬原字可能會產生難以理解的狀況，在日文中的「侍」在某些情況下可指稱「武士」。由於歷史背景的變化，時常攜帶武器以便保護皇族與貴族的「侍」漸被改稱為「武士」。嚴格來說，「侍」不等同「武侍」的地位高過「武士」，但是台灣方面還是將它翻成了「武士戰隊真劍者」。（參考來源：岩波日本史辭典）香港方面則翻成「武侍戰隊」，估計是因「侍」和「士」的粵音一樣，而保留「侍」一字。</ref>）是日本[[東映|東映]]公司在2009年製作的「[[超級戰隊系列|超級戰隊系列]]」第33部[[特攝|特攝]]作品。2009年2月15日到2010年2月7日每週日早上7點30分到8點在[[朝日電視台|朝日電視台]]播出，[[台灣|台灣]]由[[東森幼幼台|東森幼幼台]]於2011年10月9日每週日起下午17:30播出中文配音版，但隨著《[[假面騎士W|假面騎士W]]》在本片原本的下午17:30時段開始上檔，所以從2012年7月1日起，移至每週日下午17:00播出，因此每周日下午17:00到18:00的特攝節目時段，由官方命名為「'''超級英雄特區'''」。[[香港|香港]]由[[翡翠台|無綫電視翡翠台]]於2012年3月4日起，每週日早上09:30播出粵語配音版，最初編組主要成員則為三男二女組合，顏色依序為：紅、藍、桃、綠、黃。

== 概述 ==
來自冥河的外道眾出現人間，對人類產生危害。打著「主公」名號的影武者'''志葉丈瑠'''將志葉家的家臣'''池波流之介'''、'''白石茉子'''、'''谷千明'''、'''花織琴羽'''集合在一起，為保護人類，以及阻止外道眾的破壞。這五名侍衛各自擁有自己專屬的折神，並且一路上找回了獨角仙、舵木（旗魚）、虎三個折神。之後丈瑠的朋友'''梅盛源太'''也加入他們的行列。此外侍戰隊也遭遇來自假面騎士世界的拜訪，此外，故事後半段少年榊原弘將家傳的牛折神送給丈瑠。

== 特色 ==
*本作是首次採用《[[武士|武士]]》做為主題的戰隊。
*風格取向為和風的日本武士造型，特別不同以往戰隊的配色形式，除了使用戰隊傳統的顏色組合：紅、藍、桃、綠、黃、金之外，這次配色反而以黑色為下半身基礎色調（真劍金是深藍色），搭配前述幾種色彩的上半身服裝，色彩對比強烈。
*在變身前的狀況下並無統一的制服，回歸早期戰隊穿著便服的方式，配色與個人的戰士色彩搭配，但無論色彩和比重都不明顯，譬如棗紅色、暗紅色、啡黃色等等，又譬如琴羽曾經只穿着黃色長襪。
*真劍者的組合並非平等階級的團隊，而是帶有日本傳統君臣關係的組合。當黑子們幫戰隊成員換上劍道服後，真劍紅說：「'''到此為止了，外道眾'''」（第26集真劍紅不在場時由真劍藍代替，下述的話則由全部五人一起說），然後六人進行變身及報完名號時，除真劍紅站著說：「'''天下御免的侍戰隊'''」外，其餘五人皆為半跪，說完後五人便站起來齊聲道：「'''真劍者，參見！'''」。另外由於侍戰隊六人均是武士，故報名時不同於以往先說出一堆形容詞再報上顏色稱號，相反是先報顏色稱號＋真實姓名，符合武士決鬥時要報上真姓名的傳統，亦沒有過去戰隊中的一堆自我形容詞句。
*變身後的隊呼不同以往，只有第一個會稱為「真劍」（真劍Red），其後的會以「同隸屬的（同じく）」代替（真劍）。例如平時六人變身，會是「'''真劍Red：志葉丈瑠，同隸屬的Blue：池波流之介，同隸屬的Pink：白石茉子……'''（下略）」（第26集真劍紅不在場時該內容則省略掉）。但是例如在第4集，只有真劍藍及真劍桃，則變作「'''真劍Blue：池波流之介，同隸屬的Pink：白石茉子'''」，此時「'''天下御免的侍戰隊'''」則會省略，而兩人直接說「'''真劍者，參見！'''」。代表著除君臣之別外還有前後之分，是很有紀律的侍（武士）。<ref>但是在台灣東森幼幼台配音的部分，不知道是因為台灣沒有這樣主從關係的觀念，為方便配音還是其他原因，所有人全都是「真劍〇〇（顏色），〇〇〇〇（名字）」。</ref>
*例外的是與假面騎士共演集數（《假面騎士Decade》第24集「見參侍戰隊」），該集中報名時所有人都以「'''真劍X，（本名）'''」自稱。
*當打倒敵人後，真劍紅會說道：「'''如此一來也就一件了結！'''（日本）；'''現在解決掉一個了！'''（台灣）；'''事情就此告一段落！'''（香港）」（真劍金加入後，會先帶頭比勝利手勢後，真劍紅才會說該句話）。
*本作品也是首度與「假面騎士系列」（《假面騎士Decade》）跨作品共同演出的戰隊，而且劇情產生橫跨關係，兩大特攝英雄同台出現成為難得一見的賣點，也是大突破。<ref>在二十、二十一集尾段出現。</ref>
*單人強化裝備（印籠丸）可以讓各戰隊成員個別使用。
*超級戰隊史上首次出現真正領導者是女性擔任的角色<ref>扮演真正殿下的女性是夏居瑠奈。</ref>，而且還是有史以來第一位女性紅色戰士。因此為增加曝光機會，安排在2011年第35作「[[海賊戰隊豪快者|海賊戰隊豪快者]]」登場的是志葉薰和丹波歲三，而非志葉丈瑠和日下部彥馬。
*繼前作後，本作品也是無司令設定的戰隊。

== 人物介紹 ==
=== 角色 ===
戰隊登場唱名：「'''天下御免的侍戰隊・真劍者，參見！'''」。<br/>
人物有很明顯的傳統日本武士風，成員組成的結構也帶有主從關係，從志葉家的主公真劍紅開始，其下面的家臣分別是池波家（真劍藍）、白石家（真劍粉紅）、谷家（真劍綠）、花織家（真劍黃）在中期還有一位非武士的梅盛源太（真劍金），比起其他系列上與下的分別較為明顯，雖然如此，角色還是有很鮮明的個性，而身為上位者的主公本人也帶有開明的胸襟。

第一人稱為「'''俺'''」或「'''僕'''」。志葉丈瑠被培育為志葉家族第18代繼承人，從小因雙親遭外道眾殺害，並由日下部彥馬一手帶大並授以技能。<br>
表面上冷酷嚴格又随意要求下屬，實際上卻熱心助人。處事穩重，各項技能在戰隊裡也是最好的。<br>
由於是家傳身分的領主，身分亦為侍戰隊的領袖人物。很照顧琴羽，對他很溫柔，第22集甚至自願當琴羽的隨從。第41集更因聽到琴羽的話，讓他再次領悟到自己該怎麼做。<br>
隨著故事發展揭露，丈瑠其實是被志葉家收留，奉命頂替當年尚未出生的公主志葉薰的替身，在本尊成人並習得封印文字之前，必須以主君身分自居，有朝一日帶領其他戰士與外道眾交戰。<br>
最後在公主使用封印文字失敗之後，以志葉薰的養子身份繼承志葉家當家之位，正式成為名副其實的第19代繼承人。<br>
私下很怕鬼，小時曾被帶進鬼屋被嚇到尿褲子，在第28集再次挑戰，也是落得被嚇昏的下場。不過奇怪的是丈瑠只對人造鬼屋感到害怕，遇到外道眾時則是毫無畏懼。持有秘傳光盤虎折神、牛折神以及恐龍折神。

*'''黑子（くろこ）'''（志葉家的家僕）：福澤博文、押川善文、人見早苗、竹內康博、橋口未和、岡元次郎

== 外道眾 ==
有時又稱為縫隙對面的妖怪，長居於[[三途川|三途川]]的魔界之水上。基本上除無名眾外，純外道的魔物設定是「在人間久留會缺水，必須返回三途川補充水份，亦只有三途川的河水能使其活下」和「本身就有兩條生命，第一條命死後，立即會使用第二條命，身型亦會變大」，不像之前的魔物要靠其他外力去加強能力變大。

*注：「三途川」是日本民間傳說中，從人間通往陰間地府的冥河。在日本文化中象徵著'''生與死'''的分界線。據說因為河水會根據亡者生前所為的善惡而以'''緩慢、普通、急速'''三種情況流動，故名為「'''三途'''」。

== 真劍者武器・技能 ==
=== 初期成員專用 ===
*'''書道手機'''
:真劍者五人有的變身電話，有手機和毛筆兩種設定，真劍者可通過毛筆書寫漢字來發揮自己的文字之力
:當真劍者要將武神巨大化，使用變身電話來書寫漢字“大”讓各自的武神放大。
:當武侍合體時，由真劍紅使用變身電話來書寫漢字“合”讓五武神合體，成為“真劍王”。
:當進行超武侍合體時，眾人使用變身電話來書寫漢字“超”使真劍王與大天空合體，成為“天空真劍王”。

*'''真劍丸'''
真劍者五人的共有武器，也是基本武器，可通過劍術斬殺敵人，也可以安裝各種密盤來發揮密盤本身蘊含的文字之力，手柄與刀刃連接處可安裝印籠丸。

*'''秘盤'''
文字之力或折神之力的載體，可濃縮文字之力，也可以發揮折神的能力或召喚折神，每位真劍者（六人）都持有部分密盤，也可用文字之力製造秘盤。

*'''印籠丸'''
;配音員：[[宮田浩德|宮田浩德]]（台：[[胡鎮璽|胡鎮璽]]／港：[[麥皓豐|麥皓豐]]）
:志葉家代代相傳的秘密武器，通過真劍金天才的頭腦完成，可安裝在真劍丸上使真劍者升級至超級模式，劍術和文字之力都可得到飛躍性提升，同時也只有在超級模式時才可使用部分高級秘盤，一般由真劍紅使用，但可以共用。
:集合所有光盤的文字力，可以讓真劍者成為超級型態，代表文字為“真”。
:進行真武侍合體時，將印籠丸與真劍丸結合，並在印籠丸中放入「真侍飛輪」即可使真劍王與大海王合體，成為“大海真劍王”。
:值得一提的是，真武侍合體時不需全員集合，合體過程只需變身為超級型態的真劍者操縱即可。

==折神==
侍戰隊成員各自有專屬的駕駛機械，這些機械稱之為折神<ref>本字“折神”是日本漢詞，意思是「折神」來自於「折り紙（摺紙）」與「式神（陰陽師的役使）」的混合詞彙。</ref>。當五架折神合體可成為武士合體──真劍王。其次真劍王除了以五架折神為基礎外，還可以和獨角仙折神、舵木折神、虎折神組裝成為不同作用的真劍王。此外獨角仙折神、舵木折神、虎折神又可以組裝成為飛行機大天空；大天空更可與真劍王合體成天空真劍王。

===折神===
侍戰隊的折神全部都是以動物為主要造型，每一隻折神都附帶有一片秘傳光盤。港譯'''武神'''。

*'''[[獅子|獅子]]折神'''
:第1集開始登場，真劍紅專用，代表「火」的折神，以撞擊作為攻擊。
:真劍王的頭部與軀幹。
:必殺技為「五角大火焰」。

*'''[[龍|龍]]折神'''
:第1集開始登場，真劍藍專用，代表「水」的折神，以口中吐出的龍息作為攻擊。
:真劍王的左腳、兜盔。

*'''[[龜|龜]]折神'''
:第1集開始登場，真劍粉紅專用，代表「天」的折神，以刮起的風作為攻擊。
:真劍王的右手。

*'''[[熊|熊]]折神'''
:第1集開始登場，真劍綠專用，代表「木」的折神，以手臂作為攻擊。
:真劍王的右腳。

*'''[[猿|猿]]折神'''
:第1集開始登場，真劍黃專用，代表「土」的折神，以手臂作為攻擊。
:真劍王的左手。

*'''[[獨角仙|獨角仙]]折神'''
:第5集開始登場，港譯'''甲鬥'''。真劍紅努力增強文字力後能使用。以可旋轉的大角和大砲作為攻擊。
:一開始為真劍紅所用，後來得到虎折神，策略重組後交予真劍粉紅。
:之後又因真劍綠的心有不甘及自我鍛鍊，後為真劍綠常用折神。
:後來真劍粉紅有借用，應該可以共用。

*'''[[劍魚|舵木]]折神'''<ref>“舵木”在日文用法是指稱旗魚。</ref>
:第7集開始登場，港譯'''劍魚'''。真劍藍用文字力「釣」回來的折神。以魚雷和吻作為攻擊。
:為真劍藍常用折神。
:後來真劍黃有借用，應該可以共用。

*'''[[虎|虎]]折神'''<ref>日本那邊有人另稱之為寅折神。</ref>
:第9集開始登場，真劍紅常用折神。以鑽頭狀的四肢作為攻擊。
:原本是志葉家落入外道眾手中的折神，後來回歸真劍者的折神。

*'''[[烏賊|烏賊]]折神'''
:第17集開始登場，真劍金專用，代表「光」的折神。以噴墨和觸手作為攻擊。
:事實上是真劍紅-志葉丈瑠兒時贈送的禮物，後為真劍金常用折神。

*'''[[龍蝦|海老]]折神'''<ref>日本漢詞中的“海老”指的是蝦子，這裏則特別指稱龍蝦。</ref>
:第19集開始登場，真劍金專用折神，且為真劍金使用壽司手機自行開發的折神。以雙鉗和光輪作為攻擊。

*'''[[恐龍|恐龍]]折神'''
:初出場於劇場版《天下分け目の戦》，之後電視版從第31集開始登場，真劍紅常用折神。以咬住作為攻擊。
:為首代真劍紅所使用的先祖秘傳飛輪，戰後遺失了，被放在外道眾所霸佔的幻龍神社。
:電視版最後期真劍綠有借用，DVD 特別篇《光侍驚變身》中真劍金也有用過。
:而在劇場版《侍戰隊真劍者 VS 轟音者銀幕BANG!!》中。真劍紅也有將恐龍DISC借給轟音紅使用，讓轟音紅變身為《Hyper轟音紅》，在與敵人戰鬥時，Hyper轟音紅的加油槍棍模式，也有出現像恐龍丸那樣的攻擊效果。

*'''[[牛|牛]]折神'''
:第32集開始登場，真劍紅常用折神。以撞擊和背上的大砲作為攻擊。
:牛折神是所有武的元祖，換言之就是最古老的折神。
:後來真劍金有借用。

==侍合體==
侍合體是這一代戰隊的機器人專有的稱呼，不過武侍合體不全然一定都是人型機器，例如大天空則是由獨角仙折神、舵木折神、虎折神組裝的鳥型飛行機器；其次烏賊天空破壞砲也不是人型機器，因此這裡的侍合體應該多指稱各折神合體的意思。

一般出場時的口號為「天下統一」，武裝出場時的口號為「天下無雙」，變形出場時的口號為「天下一品」。

*'''真劍王'''
:第2集開始登場，獅子折神、龍折神、龜折神、熊折神、猿折神合體。
:必殺技為「侍斬」。

*'''獨角仙真劍王'''
:第5集開始登場，獨角仙折神、真劍王合體。
:必殺技為「兜大迴轉砲」。

*'''舵木真劍王'''
:第7集開始登場，舵木折神、真劍王合體。
:必殺技為「舵木　一刀兩斷」。

*'''猛虎真劍王'''
:第9集開始登場，虎折神、真劍王合體。
:必殺技為「猛虎鑽突擊」。

*'''大天空'''
:第10集開始登場，獨角仙折神、舵木折神、虎折神合體飛行機。
:必殺技為「大天空　大激突」。

*'''天空真劍王'''
:第12集開始登場，大天空、真劍王合體。
:必殺技為「天空唐竹斬」。

*'''烏賊真劍王'''
:第18集開始登場，烏賊折神、真劍王合體。
:必殺技為「槍烏賊一閃」。

*'''大海王'''
:第20集開始登場，海老折神自行變化的侍合體，而且海老折神可以變化出東（雙鉗）南（雙刀）西（尾鰭扇）北（烏賊大海王）四種模式的大海王，其中「北」模式必須和烏賊折神的侍武裝。
:必殺技為「龍蝦鉗絕技返」。

*'''烏賊大海王'''
:第19集片頭曲登場，第22集正式登場，烏賊折神、大海王合體，大海王四種變形中「北」的模式。
:必殺技為「烏賊槍突刺」。

*'''大海真劍王'''
:第24集開始登場，真劍王、大海王合體。
:必殺技為「二天一流　亂斬」。

*'''烏賊天空破壞砲'''
:第24集開始登場，烏賊折神、大天空合體的巨型砲台，其配合大海真劍王一起使用。其缺點是使用時需要大量文字力，只可以使用一次。
:必殺技為「折神大開放」。

*'''大御用'''
:第28集開始登場，大御用變大後自行變形的折神機器人。
:必殺技為「秘傳盤亂射」。

*'''真劍大御用'''
:僅於第30、38集登場，變大了的大御用撞開了正要合體的獅子折神，最後變成大御用、龍折神、龜折神、熊折神、猿折神的合體。
:必殺技為「十手一直線」。

*'''恐龍真劍王'''
:第31集開始登場，恐龍折神、真劍王合體，初出場於劇場版《天下分け目の戦》。
:必殺技為「天地一線」。

*'''猛牛大王'''
:第33集開始登場，牛折神自行變化的侍合體，由真劍紅一人操控。
:必殺技為「猛牛大迴轉砲」。

*'''侍霸王'''
:第35集開始登場，11折神全合體；當搭配恐龍折神時則為12折神全合體。
:必殺技為「文字之力大彈圓」。
:搭配恐龍折神時的必殺技為「恐龍丸　大侍斬」。
:搭配恐龍折神的侍霸王，打破了2006年的[[轟轟戰隊冒險者|轟轟戰隊冒險者]]中，由究極大冒險所創下戰隊史上最多十機合體的數量紀錄。與前作[[炎神戰隊轟音者|炎神戰隊轟音者]]中出現，炎神王G12，一同成為超級戰隊系列史上最多合體機械數的十二機合體機械人，紀錄直到五年後被[[烈車戰隊特急者|烈車戰隊特急者]]的十四機合一打破。
