'''采样率'''（也称为'''采样速度'''或者'''采样频率'''）定义了每秒从[[连续信号|连续信号]]中提取并组成[[离散信号|离散信号]]的[[采样|采样]]个数，它用[[赫兹|赫兹]]（Hz）来表示。采样频率的倒数叫作'''采样周期'''或'''采样时间'''，它是[[采样|采样]]之间的时间间隔。注意不要将采样率与[[比特率|比特率]]（bit rate，亦称“位速率”）相混淆。

采样频率只能用于周期性采样的[[采样器|采样器]]，对于非周期性采样的采样器没有规则限制。

采样频率的常用的表示符号是<math>f_s\,</math>。

== 采样定理 ==
[[采样定理|采样定理]]表明采样频率必须大于被采样信号[[带宽|带宽]]的两倍，另外一种等同的说法是[[奈奎斯特频率|奈奎斯特频率]]必须大于被采样信号的带宽。

如果信号的带宽是100Hz，那么为了避免[[混叠|混叠]]现象采样频率必须大于200Hz。

换句话说就是采样频率必须至少是信号中最大频率分量频率的两倍，否则就不能从信号采样中恢复原始信号。

== 过采样 ==
在有些情况下，人们希望采样频率超出信号带宽的两倍这样就可以用[[数字滤波器|数字滤波器]]替换性能不好的模拟[[抗混叠滤波器|抗混叠滤波器]]，这个过程称为[[过采样|过采样]]。

== 视频系统 ==
在[[模拟视频|模拟视频]]中，采样率定义为帧频和场频，而不是概念上的像素时钟。图像采样频率是传感器积分周期的循环速度。由于积分周期远远小于重复所需时间，采样频率可能与采样时间的倒数不同。

* 50 Hz - [[PAL制式|PAL]]视频
* 60 / 1.001 Hz - [[NTSC|NTSC]]视频

当模拟视频转换为[[数字视频|数字视频]]的时候，出现另外一种不同的采样过程，这次是使用像素频率。一些常见的[[像素|像素]]采样率有：
* 13.5 MHz - [[CCIR_601|CCIR 601]]、[[D1_video|D1 video]]
高频[[Luminance_(video)|luminance]]成分的[[混淆现象|混淆现象]]作为[[moiré_pattern|moiré pattern]]出现。

== 音频 ==
在[[数字音频|数字音频]]领域，常用的采样率有：
* 8,000 Hz - [[电话|电话]]所用采样率，对于人的说话已经足够
* 11,025 Hz
* 22,050 Hz - [[无线电|无线电]][[广播|广播]]所用采样率
* 32,000 Hz - [[miniDV|miniDV]]数码视频[[camcorder|camcorder]]、[[Digital_Audio_Tape|DAT]]（LP mode）所用采样率
* 44,100 Hz - [[CDDA|音频CD]],也常用于[[MPEG-1|MPEG-1]]音频（[[VCD|VCD]], [[SVCD|SVCD]], [[MP3|MP3]]）所用采样率
* 47,250 Hz - [[Nippon_Columbia|Nippon Columbia]]（Denon）开发的世界上第一个商用[[脈衝編碼調變|PCM]]录音机所用采样率
* 48,000 Hz - miniDV、[[数字电视|数字电视]]、[[DVD|DVD]]、[[Digital_Audio_Tape|DAT]]、电影和专业音频所用的数字声音所用采样率
* 50,000 Hz - 二十世纪七十年代后期出现的[[3M|3M]]和[[Soundstream|Soundstream]]开发的第一款商用数字录音机所用采样率
* 50,400 Hz - [[三菱|三菱]]X-80数字录音机所用所用采样率
* 96,000或者192,000 Hz - [[DVD-Audio|DVD-Audio]]、一些[[LPCM|LPCM]] DVD音轨、[[Blu-ray_Disc|Blu-ray Disc]]（藍光碟）音轨、和[[HD-DVD|HD-DVD]]（高清晰度DVD）音轨所用所用采样率
* 2.8224 MHz - [[SACD|SACD]]、[[索尼|索尼]]和[[飞利浦|飞利浦]]联合开发的称为[[Direct_Stream_Digital|Direct Stream Digital]]的1位[[sigma-delta_modulation|sigma-delta modulation]]过程所用采样率。
