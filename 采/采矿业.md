'''采矿业'''是从地下开采有经济价值的矿物或其他物质的活动，开采的部位都是矿物比较集中的矿床，采矿业开采的物质包括[[铝|铝]]矾土、[[煤|煤]]、[[钻石|钻石]]、[[铁|铁]]、稀有金属、[[铅|铅]]、[[石灰石|石灰石]]、[[镍|镍]]、[[磷|磷]]、岩盐、[[锡|锡]]、[[铀|铀]]和[[钼|钼]]等，几乎任何不能由[[农业|农业]]生产的原始物质都是由矿物提供的，从广义来说，[[石油|石油]]、[[天然气|天然气]]、甚至地下[[水|水]]的开采都能算做采矿业范畴。

== 历史 ==
[[File:TamarackMiners_CopperCountryMI_sepia.jpg|thumb]][[密执安州|密执安州]]塔马拉克[[铜|铜]]矿1905年的矿工合影]]
[[File:Bagger-garzweiler.jpg|thumb]]
已知最早的人类采矿遗址大概是[[斯威士兰|斯威士兰]]的“狮穴”，根据放射性测定大约是属于43,000年前的矿穴，当时是开采赤铁矿用于做原始人图腾的[[颜料|颜料]]。在[[匈牙利|匈牙利]]也发现了一个原始人开采[[燧石|燧石]]用于制作武器和工具的矿场。另外在[[西奈半岛|西奈半岛]]也发现一个古[[埃及|埃及]]开采[[绿松石|绿松石]]的矿场。 

1627年，在[[斯洛伐克|斯洛伐克]]第一次使用[[黑色火药|黑色火药]]开矿。

== 采矿的步骤 ==
# 探矿，寻找可能存在的矿床；
# 勘探，确定矿床的蕴涵量和范围；
# 定量估算矿物的品质和埋藏范围；
# 进行可行性研究，决定是否有开采价值；
# 对矿山建设进行经济规划；
# 开发，建设矿山，开采矿物；
# 精选矿物；
# 恢复由于开采造成的土地破坏。

== 选矿 ==
选矿是一门专业的科学技术，从矿石中提炼浓缩有价值的矿物，有[[化学|化学]]方法和机械方法，包括[[粉碎|粉碎]]、研磨、重力選礦法、[[水|水]]选等多道工序，精选后残余部分为尾矿，需要抛弃。为了防止污染[[环境|环境]]，尾矿的处置也是需要研究的课题。

== 环境影响 ==
[[File:Iron_hydroxide_precipitate_in_stream.jpg|thumb]]
采矿业对[[环境|环境]]的影响包括水土流失、矿坑造成的地面沉降、生物多样性的破坏以及采矿过程中含化学物的废水对地下水的[[污染|污染]]等。

现代许多国家对采矿业有严格的环境保护和恢复地表状态的[[法律|法律]]法规，保证采矿区域要恢复原有的状态，甚至要求比采矿以前的环境更好。使用过去的方法采矿，对采矿业限制不严格的国家，采矿业会对环境造成无法弥补的伤害，同时会影响人类的健康。

采矿业会对周围环境和地下水造成相当大的影响，化学物质会集中污染大片地区，例如[[煤|煤]]矿就会产生大约20多种有毒化学物质，会经由排放水和供应水的渠道扩散污染范围。有严格控制法律的国家，采矿业必须有水文专家和[[地质学|地质学]]家仔细研究处理污染物，一般采用五种单独措施，处理水污染问题：水分离系统、含污染物废水集中存塘、地下水采集系统、地表水排放系统和隔离系统。对于含有酸性污染物的水要直接排放到水处理系统处理。

== 采矿事业 ==
以前可以由私人或小公司从事采矿事业，但现代采矿事业需要大量的资本投入，一般都是由国际知名的大企业从事。
== 采矿人员 ==
[[File:Gorskii_04428u.jpg|thumb]]
[[File:Coal-wpa.jpg|thumb]]
现代采矿事业不再仅仅是在地上挖掘，需要许多不同领域的专门人员。直接或间接地加入到采矿行业中来，包括[[工程师|工程师]]、[[实验室|实验室]]技术人员、[[地质学|地质学]]家、[[环境科学|环境科学]]家等，除了直接在[[矿山|矿山]]工作的人员外，还有许多专业人员为采矿业服务，例如[[会计|会计]]、[[律师|律师]]、营销人员、[[公关|公关]]人员等，还没有算上成千上万的矿山机械生产和维修人员。

采矿业雇员的工资一般较高，尤其是在边远地区的矿山工作，根据[[美国|美国]][[劳动统计局|劳动统计局]]的调查，[[煤|煤]]矿工人的工资要高出平均工资的30%，在采矿业工作的大学毕业生或地质工程师平均年工资为8万[[美元|美元]]。在采矿业一般只能工作到46-50岁，但目前到[[大学|大学]]学习采矿业的人越来越少。

== 矿山设计软件 ==
目前矿山设计最大的进步是采用了三维设计软件，不是再向以前人工设计那样耗费时间，只要勘探人员、地质学家、矿山工程师和其他方面的技术人员输入必要的数据，软件可以给出最优化的开采设计。

== 矿山机械 ==
矿山机械要能够[[粉碎|破碎]]和移走任何硬度的岩石，挖掘机、钻机、炸药和卡车是必不可少的，尤其对于露天开采。地下开采的机械更为复杂和昂贵，许多国际著名的这公司生产连续采掘的机械。
== 安全问题 ==
安全问题一直是采矿业最重要的问题，尤其对于地下开采，虽然现代采矿业的比几十年前要更为安全，但也经常发生[[矿难|矿难]]事故。2010年3月28日王家岭煤矿发生[[王家岭矿难|透水事故]]有超過100人被救出。<ref name="王家岭">{{cite news
