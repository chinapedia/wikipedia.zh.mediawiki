'''赵来思'''（'''Lensey Namioka''' née '''Chao'''），生于1929年6月14日，美籍华人。数学家，儿童与青少年文学作家。她是著名语言学家[[赵元任|赵元任]]先生的三女儿，美籍日裔数学家{{link-en|波岡維作|Isaac_Namioka}}的妻子。

== 生平与教育 ==
赵来思于1929年6月14日生于[[北京市|北平，中国]]。父亲是'''汉语言学之父'''[[赵元任|赵元任]]，母亲[[楊步偉|杨步伟]]。赵来思儿时同家人居住在中国，因战乱时常搬迁。1938年，为躲避战乱，九岁的赵来思随家人移居[[美国|美国]]。初到美国时赵来思不会讲英语，相较之下数学对她而言更容易。1947-1949年就读于[[拉德克利夫学院|拉德克利夫学院]]，后分别于1951年，1952年在[[加利福尼亞大學柏克萊分校|加州大学伯克利分校]]取得数学学士与硕士学位。她父亲时任[[加利福尼亞大學柏克萊分校|加州大学伯克利分校]]东亚系教授。她认识了同样在伯克利攻读数学的{{link-en|波岡維作|Isaac_Namioka}}，两人于1957年结婚。1963年因丈夫接受了[[华盛顿大学|华盛顿大学]]的职位，搬迁至西雅图。赵来思早期在大学任数学讲师，后从事了她儿时最爱的事业——创作冒险故事。

== 家庭 ==
[[赵元任|赵元任]]与[[楊步偉|杨步伟]]共育有四女，赵来思排行第三。大姐[[卞趙如蘭|赵如兰]]（Iris），二姐[[趙新那|赵新那]]（Nova），四妹赵小中（Bella）。

1957年9月9日，赵来思与丈夫{{link-en|波岡維作|Isaac_Namioka}}成婚，并育有两女 Aki （1959），Michi （1961）。

== 职业与写作 ==
赵来思早期在大学当数学讲师（数学讲师，[[:en:Wells_College|威尔斯学院]]，1957-58；数学讲师，[[康奈尔大学|康奈尔大学]]，1958-61；译者，[[美國數學學會|美国数学学会]]，1958-66；广播总监，日本广播公司，1969）。在上世纪七十年代中期成为了一名作家。赵来思在八岁时写了她的第一本书，名为《拿着竹剑的公主》。她在七十年代一次去日本的旅行中对日本武士有了更多的了解，回家后写下了《武士与长鼻子恶魔》。此书发表与1976年，后成为了她创作的武士系列的一部分。此外她创作了一系列以杨姓美籍华裔家庭展开的故事。同时她的写作也包括了年轻的女孩面对困境，中国与日本的旅游。

== 名字由来 ==
赵来思（Lensey）是唯一已知的以“Lensey”作名的人。她名字在中文里没有对应的汉字。她的父亲在归纳中文语音时发现“len”和“sey”两个音节可以出现在汉语言中却没有相应的汉字，因而她被命名为“Lensey”。
