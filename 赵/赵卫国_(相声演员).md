'''赵卫国'''（{{bd|1963年}}），[[陕西|陕西]][[西安|西安]]人，中国著名[[南派相声|南派相声]][[艺术家|艺术家]]。和[[任军|大兵]]搭档，被誉为“铂金组合”。<ref name="web200 " />

== 简介 ==
赵卫国从小就爱好[[戏曲|戏曲]]，受过严格的专业训练，戏曲基本功扎实。16岁时，他从[[西安|西安]]来到[[武汉|武汉]]，开始闯荡事业。<ref>{{cite web|url=http://happy.enorth.com.cn/system/2009/11/19/004281387.shtml|title=赵卫国、大兵一脚踹来的搭档|accessdate=2009-12-03|work=|publisher=北方网|language=zh-hans}}{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>1990年7月，赵卫国毕业于[[武汉市广播电视大学|武汉市广播电视大学]]文宣班。1995年，赵卫国获得了武汉市政府人才江花奖。2003年，[[任军|大兵]]、[[杨奇志|杨奇志]]分手后，赵卫国和[[任军|大兵]]组成了新搭档。此前，他在武汉的搭档是[[李道南|李道南]]。2003年8月7日，[[任军|大兵]]、赵卫国首次在[[蓝天大剧院|蓝天大剧院]]演出，正值非典期间，经过千方百计，两人的演出才获得成功。2003年10月12日，[[任军|大兵]]和赵卫国首次亮相[[大中华歌厅|大中华歌厅]]。2004年，赵卫国、大兵出席了[[湖南卫视|湖南卫视]][[春晚|春晚]]。2006年，赵卫国、大兵亮相[[央视|央视]][[春节联欢晚会|春节联欢晚会]]，演出《[[谁让你优秀|谁让你优秀]]》，夺得观众最喜爱的春晚节目二等奖。2006年5月，大兵和赵卫国的《[[免费电话|免费电话]]》亮相2007年[[央视|央视]][[春节联欢晚会|春节联欢晚会]]，再次获得最受观众喜爱的春晚节目二等奖。<ref name="web200 " >{{cite web |url=http://www.hsgd.net.cn/Article/HuangShiNews/wtyl/201002/20100209160748_91981.html|title=大兵、赵卫国联袂演绎新传奇 |accessdate=2010-02-09|work=|publisher=声屏网|language=zh-hans}}</ref>

== 公职 ==
[[国家一级演员|国家一级演员]]；[[中国曲协|中国曲协]]会员；湖北省曲协会员；武汉市相声学会会员。<ref name="web200 " />

== 作品 ==
*《借电话》
*《害人不浅》
*《归国记》
*《谁让你优秀》
*《免费电话》

== 奖项 ==
*《借电话》获全国曲艺小品决赛表演一等奖。<ref name="web200 " />
*《害人不浅》获湖北省第五届百花书会表演一等奖。<ref name="web200 " />
*《归国记》参加《[[在延安文艺座谈会上的讲话|在延安文艺座谈会上的讲话]]》发表50周年曲艺演唱获优秀奖、湖北省第二届[[屈原|屈原]]文艺创作奖。<ref name="web200 " />
