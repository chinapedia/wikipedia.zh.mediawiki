'''財華社集團有限公司'''(“'''財華社集團'''")（'''{{lang-en|Finet Group Limited}}''') 是一家於香港聯合交易所有限公司創業板上市的公司，股票代號為 [https://www.hkex.com.hk/chi/invest/company/quote_page_c.asp?WidCoID=8317&WidCoAbbName=%20&Month=1&langcode=c 8317-HK] 。

財華社集團是大中華區知名的財經媒體及財經資訊供應商之一，專為企業客戶及投資者開發、製作及提供財經訊息媒體服務。伴隨時代發展，財華社集團現已開拓財經視頻新媒體平台–現代電視（FinTV），致力於為全球華語觀眾及投資者提供專業、即時、深度的金融市場視頻資訊，為內地與香港的投資者建立起大中華地區財經新聞互通的橋樑，即時提供金融資訊、解決方案、財經新聞、財經視頻等綜合性金融資訊。同時，財華社集團正在向『移動端平台』與『互聯網+金融平台+金融應用』方向升級轉型。

== 業務範圍 ==
財華社集團的業務範圍包括：財華網上資訊產品，財經視頻新媒體平臺以及互聯網金融移動平臺。

== 財華網上資訊產品 ==
財華網上資訊產品主要透過集團旗下網站財華網（網址：www.finet.hk）、財華財經APP、微信小程序、公眾號等自媒體平台向全球投資者提供財經資訊。集團將股票、金融衍生產品、期權及期貨的實時市場數據，以及香港市場及美國市場等分析工具，整裝成模組式應用工具及資訊套裝，為投資者提供多項網上大利市及股票報價服務。

== 現代電視==
[http://www.fintv.hk 現代電視]（'''{{lang-en|FinTV}}''')，為財華社集團2011年創立的財經網路電視台，採用PC端、移動端多屏播出。以內地為主要市場，粵/普雙語播出，為華語體系的廣大投資者，提供即時報價、新聞、評論以及專題類節目。

== 互聯網移動平臺 ==
隨著滬港通、深港通和中港基金互認相繼開放，內地與香港聯動這一獨特優勢不斷突顯，財華社集團抓住機遇，正努力建設一個集資訊雲、金融雲、自選股、券商交易連結為一體的金融移動端平台，整合報價、資訊、分享、交易、互動，五個環節，將金融與移動相結合，提供完整服務鏈。為內地與香港的股民與投資者提供開放式財經資料、文字及視頻財經資訊、交流互動、連結交易平台等服務。

== 香港上市公司100強 ==
『香港上市公司100強』評選活動（簡稱『港股100強』），由港股100強研究中心主辦，經由專業評審委員會，以最先進的科學資料分析為依據，將在香港聯交所上市的公司其五大財務指標：營業額、總市值、除稅後溢利、股東回報率及股息回報率排行加權，根據每年度9月30日收市價或最接近6月30日公佈的年報數據計算，複合得分最高的100家上市公司最終入選主榜，當選為年度港股100強。
