'''现行盲文'''是一种用于拼写[[汉语普通话|汉语普通话]]的[[盲文|盲文]]，1953年由[[中华人民共和国教育部|中华人民共和国教育部]]颁布并在全国推行。虽然后来推出了[[汉语双拼盲文|汉语双拼盲文]]，在中国大陆，本方案仍更为通行。

现行盲文是[[表音文字|表音文字]]，汉语的每个音节由一至三方盲符组成，分別表示聲母、韻母和聲調。字母（声母与单元音韵母）的符号选择基于[[汉语拼音|汉语拼音]]所对应的盲文的拉丁字母表。

== 拼法规则 ==
声母、韵母、声调依次相拼以组成一个音节。声母zh ch sh r z c s可自成音节；所有韵母可自成音节。

以词为单位书写，词间空出一方。

原則上不標調，但部份情形例外。例如：

* 词语中的前字为自成音节的声母（zh ch sh r z c s）、后字为零声母时，前字'''必须'''标调，如「{{ruby-py|食|shí}}物（⠱⠂⠥）」。
* 生辟的词语或成语。
* 古汉语实词。
* 文内首次出现的专有名词（极普通的除外）。
* 區別同声同韵詞（同调或不同调）時，一些词应標調。
* 单音节词一般应标调，经常使用的除外。
* 双音节词或多音节词需标调时首先考虑第一个字；必要时也可在其他字上标调。
