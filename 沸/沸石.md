'''沸石'''是一种含有水架状结构的铝硅酸盐[[矿物|矿物]]，最早发现于1756年。[[瑞典|瑞典]]的矿物学家克朗斯提（Cronstedt）发现有一类天然铝硅酸盐矿石在灼烧时会产生[[沸腾|沸腾]]现象，因此命名为'''沸石'''（[[瑞典文|瑞典文]]：''zeolit''）。在希腊文中意为“沸腾的石头”。此后人们对沸石的研究不断深入。

沸石因成分不同分为'''方沸石'''（Na[AlSi<sub>2</sub>O<sub>6</sub>]·H<sub>2</sub>O）和'''钙沸石'''（Ca[Al<sub>2</sub>Si<sub>3</sub>O<sub>10</sub>]·3H<sub>2</sub>O）。其含水量与外界温度及水蒸气的压力有关，加热时水分可慢慢逸出，但并不破坏其结晶构造。

[[晶体|晶体]]结构中有许多空腔（笼）和连接空腔的通道，[[水分子|水分子]]位于其中，可由通道运输。晶体和集合体形态及[[解理|解理]]随着晶体结构的不同而异，一般呈浅色，玻璃光泽，硬度3－3.5，比重2.0－2.4。

沸石族矿物由低温热液作用形成，见于[[喷出岩|喷出岩]]，特别是[[玄武岩|玄武岩]]的孔隙中，也见于[[沉积岩|沉积岩]]、[[变质岩|变质岩]]及热液矿床和某些近代温泉沉积中。

1932年，McBain提出了“[[分子筛|分子筛]]”（Molecular sieve）的概念。表示可以在[[分子_(化学)|分子]]水平上筛分物质的多孔材料。沸石用作分子筛，可以吸取或过滤其他物质的分子。虽然沸石只是分子筛的一种，但是沸石在其中最具代表性，因此“沸石”和“分子筛”这两个词经常被混用。

除了天然产品外，也可由人工合成，人造沸石是：磺酸化聚苯乙烯，天然沸石：铝硅酸钠。

== A型沸石 ==
'''A型沸石'''，代号LTA，是应用最广的人造硅酸盐沸石。典型品种组成为|Na<sub>12</sub>(H<sub>2</sub>O)<sub>27/8</sub>|[Al<sub>12</sub>Si<sub>12</sub>O<sub>48</sub>]，|Na<sub>64</sub>(H<sub>2</sub>O)<sub>326.71</sub>|[Si<sub>96</sub>Al<sub>96</sub>O<sub>384</sub>]或|Na<sub>91.7</sub>|[Si<sub>96</sub>Al<sub>96</sub>O<sub>384</sub>]。孔穴十分巨大。该型始于立方晶系，具立方面心晶胞，晶胞参数<math>a</math> = 24.6Å左右。A型沸石具有吸附水分子的良好性能，干燥性能很高，易再生，可反复使用。5A型可以高效分立正丁烷和异丁烷，3A型可干燥乙烯、丙烯等气体。<ref>{{Cite book|title=无机化学|last=|first=|publisher=高等教育出版社|year=2002年第二版|isbn=978-7-04-011583-3|location=|pages=938页}}</ref>

== 磷酸铝沸石 ==
'''磷酸铝沸石'''，代号AFI，又称'''介孔磷酸铝(meso-porous aluminum phosphates)'''。典型组成是AlPO<sub>4</sub>。属于六方晶系，<math>a</math> = 13.827Å，<math>c</math> = 8.580Å。磷酸铝沸石具有优良的反应活性，可以制备单层碳纳米管；亦可用作催化剂，将乙烷氧化为乙二酸。<ref>{{Cite book|title=无机化学|last=|first=|publisher=高等教育出版社|year=2002年第二版|isbn=978-7-04-011583-3|location=|pages=939页}}</ref>
