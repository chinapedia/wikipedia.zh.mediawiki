'''鴨川市'''（{{jpn|j=鴨川市|hg=かもがわし|rm=Kamogawa shi|f=y}}）是[[千葉縣|千葉縣]][[房總半島|房總半島]]南部的一[[市|市]]。面[[太平洋|太平洋]]、屬[[安房國|安房地方]]東部，是南房總、外房地域的中核都市。

2004年，三亚与日本鸭川市缔结友好关系。<ref>[http://www.visitgd.com/news/sznews/163978.shtml 三亚与日本鸭川市缔结友好关系]{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

== 地理 ==
本市位於[[房總半島|房總半島]]東南部，面向太平洋（外房）<ref name="clair">{{cite web |url=http://www.clair.or.jp/j/forum/forum/articles/town/171_1/index.html |title=鴨川市の国際交流 |publisher=鴨川市企画振興課 |accessdate=2012-01-08 }}{{dead link|date=十一月 2017 |bot=InternetArchiveBot }}</ref>，與[[都道府縣廳所在地|縣廳所在地]]的[[千葉市|千葉市]]相距70公里。位於距離[[東京都|東京]]都心100公里的首都圈內。但是從東京經由[[東京灣橫斷道路|東京灣橫斷道]]最短距離仍然有85公里。

廣闊的市區中部稱作鴨川地區，東部稱做小湊地區，南部稱作江見地區，西部內陸地區稱作長狹地區。這和昭和大合併時成立的「鴨川市」之前的自治體行政區域一致。（一樣地，天津小湊地區在2005年2月11日合併前，和天津小湊町的區域一致）太平洋沿岸的國道128號，內陸地區往保田方向銜接的{{link-ja|長狹街道|長狭街道}}，和往千葉方向接續的{{link-ja|久留里街道|久留里街道}}縱貫全市。

鴨川地區（鴨川、田原、西條、東條）因為集中著以海洋世界為主的觀光地及市街，地區內的國道128號支道塞車情形顯著。天津小湊地區（天津、小湊）的北部丘陵帶，擁有清澄山、與日蓮聖人有因緣的清澄寺，小湊有誕生寺等而著名。沿岸地區有繁盛的漁業。江見地區（江見、太海、曾呂）有仁右衛門島、花卉中心，有繁盛的產花觀光設施。長狹地區（吉尾、主基、大山）為稻米產地長狹平野的延伸<ref name="furusato">{{cite web |url=http://www.furusato-kamogawa.net/grape2/ |title=鴨川市について |publisher=鴨川市ふるさと回帰支援センター |accessdate=2012-01-08 |deadurl=yes |archiveurl=https://web.archive.org/web/20110324102421/http://www.furusato-kamogawa.net/grape2/ |archivedate=2011-03-24 }}</ref>，四面環山，田地廣闊，有繁盛的農業。

== 沿革 ==
=== 歷史 ===
* 本市所在的安房地域在718年（[[養老|養老]]2年）脫離[[上總國|上總國]]，稱[[安房國|安房國]]。
* [[江戶時代|江戶時代]]的1614年，[[里見氏|里見氏]]{{link-ja|改易|改易}}，此地成為[[旗本|旗本]]領、[[天領|天領]]、小大名領地。
*1871年（明治4年）實施[[廢藩置縣|廢藩置縣]]以前，此地屬{{link-ja|花房藩|花房藩}}。
* 1878年（明治11年），郡區町編制法施行。1897年（明治30年），安房郡、平郡、朝夷郡、長狹郡4郡合併成立「安房郡」。
* 1971年（昭和46年）3月31日 - [[安房郡|安房郡]]鴨川町、{{link-ja|江見町 (千葉縣)|江見町 (千葉県)|江見町}}、{{link-ja|長狹町|長狭町}}合併施行市制，為'''鴨川市'''。
* 1971年7月 [[內房線|內房線]]電氣化
* 1972年（昭和47年）7月 [[外房線|外房線]]電氣化
* 1973年（昭和48年）3月 {{link-ja|鴨川繞道|鴨川バイパス}}全線通車
* 1980年（昭和55年）3月 {{link-ja|天津繞道|天津バイパス}}]全線通車
* 1986年（昭和61年）12月 [[長野縣|長野縣]][[大町市|大町市]]締結姊妹都市
* 1991年（平成3年10月） [[東京都|東京都]][[荒川區|荒川區]]締結友好都市<ref>{{cite web |url=http://www.city.arakawa.tokyo.jp/kusei/kokusai/yukokoryutoshi/kokunai/kamogawa.html |title=千葉県鴨川市 |publisher=荒川区 |accessdate=2012-01-07 |deadurl=yes |archiveurl=https://web.archive.org/web/20120128220214/http://www.city.arakawa.tokyo.jp/kusei/kokusai/yukokoryutoshi/kokunai/kamogawa.html |archivedate=2012-01-28 }}</ref>
*1993年（平成5年）11月5日 [[美國|美國]][[威斯康辛州|威斯康辛州]]{{link-en|馬尼托沃克 (威斯康辛州)|Manitowoc, Wisconsin}}締結國際姊妹都市<ref name="clair"/>
* 2005年（平成17年）2月11日 - 與安房郡{{link-ja|天津小湊町|天津小湊町}}合併為現在的'''鴨川市'''。

=== 合併過程 ===
2000年12月，千葉縣發表「合併推進要綱」，安房地域的合併模式有「（1）館山市、鴨川市、富浦町、富山町、鋸南町、三芳村、白濱町、千倉町、丸山町、和田町、{{link-ja|天津小湊町|天津小湊町}}」「（2）館山市、富浦町、富山町、鋸南町、三芳村、白濱町、千倉町、丸山町、和田町」「（3）鴨川市、{{link-ja|和田町 (千葉縣)|和田町 (千葉県)|和田町}}、天津小湊町」三種。

包含舊鴨川市與舊天津小湊町在內的安房郡市2市8町1村開始研究合併，2002年3月由千葉縣知事指定11市町村為合併重點支援地域，同年9月成立任意合併協議會。但因鴨川市身負巨額債務，2003年1月任意合併協議會解散<ref>{{cite web |url=http://gappei.city.kamogawa.lg.jp/upload/pdf/houkoku/houkoku-01.pdf |title=鴨川市・天津小湊町合併協議会設置までの経緯について |publisher=鴨川市 |format=PDF |date=2003-07-07 |accessdate=2012-01-08 |deadurl=yes |archiveurl=https://web.archive.org/web/20120117165727/http://gappei.city.kamogawa.lg.jp/upload/pdf/houkoku/houkoku-01.pdf |archivedate=2012-01-17 }}</ref>。

之後，鴨川市、天津小湊町成立合併協議會，決定新市名為「鴨川市」，2005年2月實施合併至今。

2004年7月11日，和田町針對與鴨川市合併進行公投，但未能通過<ref>{{cite web|url=http://www.pref.chiba.lg.jp/shichou/kouiki/gappei/documents/4syou7.pdf |title=（安房郡富浦町・同郡富山町・同郡三芳村・同郡白浜町・同郡千倉町・同郡丸山町・同郡和田町の合併 |publisher=千葉県 |format=PDF |accessdate=2012-01-08}}</ref>。

===衝浪===
從1964年[[橫須賀海軍設施|橫須賀美軍基地]]士兵與地方少年開始，1965年舉辦日本第一次全國大會。主要地點有前原海岸、東條海岸、江見海岸<ref>{{cite web |url=http://www.city.kamogawa.lg.jp/JP/0006/0088/00001114_6_88.html |title=サーフィン |publisher=鴨川市 |accessdate=2012-01-07 }}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>等。

作為日本衝浪發源地之一，1993年日本衝浪者基金會（SFJ）於此地成立。2010年10月8日至12日舉辦SurfriderFoundation國際會議2010<ref name="surfrider">{{cite web |url=http://www.surfrider.jp/activity/international/live.html |title=実況！SurfriderFoundation国際会議2010のすべて |publisher=SFJ |accessdate=2012-01-07 |deadurl=yes |archiveurl=https://web.archive.org/web/20120803230008/http://www.surfrider.jp/activity/international/live.html |archivedate=2012-08-03 }}</ref>。

以前曾收錄在美國電影{{link-en|The Endless Summer||The Endless Summer}}的著名巨浪浪點赤堤因港口拓寬工程而消失<ref name="surfrider"/><ref>{{cite web|url=http://www.pacific-gate.jp/culture/safari/california/ca01.html |title=～ 第1部 ランスカーソンからの手紙 ～ |author=すぎやまのぼる |accessdate=2012-01-07 }}</ref>。

每年6月第一個周日舉行鴨川市長杯（現鴨川城市杯）。
