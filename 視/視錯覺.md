[[File:Grey_square_optical_illusion.svg|220px]]：字母A與字母B所在方格實為同色。]]
'''视错觉'''是指通过几何排列、视觉成像规律等手段，製作有'''视觉欺骗'''成分的图像进行眼球欺骗，引起的[[视觉|视觉]]上的[[错觉|错觉]]，达到艺术或者类似魔术般的效果。

视错觉一般被分为：图像本身的构造导致的几何学错觉；由感觉器官引起的生理错觉；以及心理原因导致的认知错觉。特别是关于[[几何学|几何学]]的错觉，以其种类多而广为人知。

== 幾何學錯覺 ==
[[Image:Hering_illusion.svg|thumb]]
[[File:Jastrow_illusion.svg|thumb]]]]
[[File:Fraser_spiral.svg|thumb]]
視覺上的大小、長度、面積、方向、角度等幾何構成，和實際上測得的數字有明顯差別的錯覺，稱為幾何學錯覺。

最早開始研究幾何學錯覺，是在1855年Oppel所發表的分割距離錯覺；即沒被分割的，看起來比分割的面積來的小。
* {{tsl|en|Müller-Lyer illusion|謬勒·萊伊爾錯覺}}
* [[艾賓浩斯錯覺|艾賓浩斯錯覺]]
* {{tsl|en|Hering illusion|赫林錯覺}}：兩條[[平行線|平行線]]因受斜線的影響呈弯曲状。此种錯覺称为[[弯曲錯覺|弯曲錯覺]]。
* [[加斯特罗图形|加斯特罗图形]]：两扇形虽然大小形状完全相同，但是下方的[[扇型|扇型]]看似更大。
* {{tsl|en|Fraser spiral illusion|弗雷泽图形}}：[[英国|英国]][[心理学者|心理学者]]弗雷泽于1908年发表，这一图形是一个产生角度﹑方向錯覺的图形，被称作視錯覺之王。[[漩涡|漩涡]]状图形实際是[[同心圆|同心圆]]。
* {{le|謝帕德之桌|Shepard's Tables}}：垂直的桌面看起來比水平的細長，但兩者其實一模一樣。{{le|羅傑·謝帕德|Roger Shepard}}在 ''Mind Sights'' (1990) 中首度發表。

== 生理錯覺 ==
[[File:Grid_illusion.svg|thumb]]

主要來自人體的視覺適應現象，人的感覺器官在接受過久的刺激後會鈍化，也就造成了[[補色|補色]]及[[視覺暫留|視覺暫留]]的生理錯覺。由於白光是由不同波長的色光所組成的，所以任何兩種色光加在一起可成為白光者，這兩色就互為補色。

而[[視網膜|視網膜]]上的細胞受某種色光刺激後，會對該色產生疲勞，所以在視線離開該色後，該部分的細胞暫無法作用，而未受刺激的另一部分細胞開始活動，因而產生另一種視感，也就是補色的殘像。

另外還有因為[[視覺疲勞|視覺疲勞]]而產生的[[視覺暫留|視覺暫留]]現象也是，視覺暫留現象就是現今[[動畫|動畫]]的原理。

{{tsl|en|Hermann grid illusion|赫曼方格}}和{{tsl|en|Mach bands|馬赫帶}}是最常被用來解釋生理錯視的作品。

{{tsl|en|lateral inhibition|侧抑制}}一直被用來解釋為什麼人類在觀看馬赫帶時看到色塊交界处的色差。一旦受體被激活，这种刺激就會抑制它鄰近的受體被刺激后的效果。這種錯覺造成了相反及突出的邊緣。在赫曼方格的十字路口出現了灰點是因為dark surround的增加而出現的抑制反應。<ref>Pinel, J. (2005) Biopsychology (6th ed.). Boston: Allyn & Bacon. ISBN 0-205-42651-4</ref>侧抑制也被用來解釋赫曼的grid illusion，但最近已被反駁。<!--（參考{{le|disproved|grid illusion#The cause of both Scintillating and Hermann grid illusions]]）-->

== 认知错觉 ==
[[File:Duck-Rabbit_illusion.jpg|thumb]]，認知錯覺的名例]]
[[File:Rubin2.jpg|thumb]]
這主要來自於人類的[[知覺恆常性|知覺恆常性]]屬於[[認知心理學|認知心理學]]的討論範圍。
* [[ASCII立體圖|ASCII立體圖]]
* [[奈克方塊|奈克方塊]]
* 同化
* {{link-en|圖地反轉|Negative space}}：圖地反轉的現象，因為人的知覺具有組織性，會想辦法將視覺對象由背景中獨立出來，這個獨立出來的部分即為「圖」，周圍的部分則是「地」，這是由[[E.Rubin|E.Rubin]]提出來的，有名的作品「{{link-en|魯賓之盃|Rubin vase}}」則是研究圖地反轉的代表作。
:易成為圖的條件：面積小、水平或垂直、位於下方、往上突出比往下垂直容易、單純的形、反覆的、被包圍的、有動感的、對稱的、密度高或有質感者、紅色波長的顏色。
* [[鴨兔錯覺|鴨兔錯覺]]：由於人類對於已知物體的認知來自於特徵及主要輪廓的記憶，人腦會自動的將和腦中印象相似的形狀及物件做比對來判讀並賦與圖像的意義，所以只要該圖具有人腦中對該物的主要形象就會做出判讀，在不破壞主要認知特徵的情況下再加上另一個特徵，就會造成大腦的誤判，同類有名的創作者還有[[達利|達利]]。

== 著名的错觉 ==
[[File:Chubbillusion.gif|thumb]]
[[File:Dichotomy.svg|thumb]]
知觉恆常性是造成許多錯覺的原因，同化作用，兩張圖裡，左邊和右邊的底色其實都是一樣，不過因為受到前景的線條顏色的干擾，使人以為兩邊底色的深淺有所不同。{{le|色感一致性|color constancy}}和{{tsl|en|brightness constancy|亮度一致性}}的事實，無論什麼亮度，一個熟悉的物體會出現相同的顏色。錯覺的色差可以創造的，但當周圍不熟悉的物體亮度改變就不能。當物體的顏色會出現較深的黑色區域，便會反比較淺色的白色區琙，即使物體本身並沒有改變顏色。
* {{link-en|艾姆斯房間錯覺|Ames room illusion}}
* {{link-en|艾姆斯梯形窗錯覺|Ames trapezoid}}
有許多專門在創作這類錯覺幻覺的藝術家，包括[[毛瑞特斯·柯奈利斯·艾雪|毛瑞特斯·柯奈利斯·艾雪]]、 {{tsl|en|Bridget Riley|Bridget Riley}}、[[萨尔瓦多·达利|萨尔瓦多·达利]]、[[朱塞佩·阿尔钦博托|朱塞佩·阿尔钦博托]]、[[馬塞爾·杜象|馬塞爾·杜象]]、{{tsl|en|Oscar Reutersvärd|Oscar Reutersvärd}}及{{tsl|en|Charles Allan Gilbert|Charles Allan Gilbert}}。

也有一些當代藝術家正在試驗一種錯覺，包括{{tsl|en|Dick Termes|Dick Termes}}、[[福田繁雄|福田繁雄]]、[[Patrick_Hughes|Patrick Hughes]]、{{tsl|en|István Orosz|István Orosz}}、{{tsl|en|Rob Gonsalves|Rob Gonsalves}}及{{tsl|en|Akiyoshi Kitaoka|北岡明佳}}等。另外，錯覺也能運用於[[電影|電影]]中。

一些錯覺如{{tsl|en|Ponzo illusion|Ponzo illusion}}及{{tsl|en|Vertical-horizontal illusions|Vertical-horizontal illusions}}也可能發生於使用時由[[聽覺|聽覺]]到[[視覺|視覺]]感官替代裝置。

另外還有[[奧普藝術|奧普藝術]]，藝術家利用不同色彩的特定搭配，包括色彩重疊、圍繞、漸變等等，給視網膜帶來特殊的刺激，眼前的作品因而出現閃爍、甚至波動感等眩目的錯覺。
