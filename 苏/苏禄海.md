'''苏禄海'''（'''Sulu Sea'''），位于西南[[太平洋|太平洋]]的一个海域，周围被[[菲律宾|菲律宾]]的[[苏禄群岛|苏禄群岛]]、[[巴拉望岛|巴拉望岛]]、[[棉兰老岛|棉兰老岛]]以及[[马来西亚|马来西亚]]的[[沙巴|沙巴]]地区围绕，連接[[南海|南海]]和[[苏拉威西海|苏拉威西海]]。该片海域是[[海龟|海龟]]的繁殖场所，盛产海龟。海域面积约348000[[平方公里|平方公里]]，平均深度约1,570 米，最大深度5,576米，含盐度31-34‰。

除了马来西亚沿岸以外，苏禄海绝大部分是菲律宾[[内海|内海]]。

蘇祿海的內海地形為一個東北往西南走向的海底盆地，內海對外北部以民多羅海峽（Mindoro Strait）、西部以巴拉巴克海峽（Balabac Strait）與[[南海|南海]]流通，南部以錫布圖（Sibutu）通道與[[西里伯斯海|西里伯斯海]]流通。南海貫穿流（South China Sea Throughflow, SCSTF）即是藉由北方民多羅海峽與巴拉巴克海峽與南方錫布圖通道，透過蘇祿內海的蘇祿海貫穿流（Sulu Sea Throughflow），流通西北方的[[南海|南海]]與南方的西太平洋[[西里伯斯海|西里伯斯海]]，因此蘇祿內海影響了南海、西里伯斯海與''西太平洋環流''水量，為西太平洋上重要的[[邊緣海|邊緣海]]。

蘇祿內海的洋流有明顯季節變化，春季時蘇祿海貫穿流往北流，夏秋冬季時蘇祿海貫穿流往南流，貫穿流也受到科氏力與風力影響有偏向。<ref name = "蘇祿海表層環流之數值模式研究_2010"> {{ cite web | title = 蘇祿海表層環流之數值模式研究 | url = http://www.airitilibrary.com/Publication/alDetailedMesh?docid=U0021-1610201315210583 | author = 陳柏宇 | language = zh-tw,en | date = 2010-01-01 }} </ref>

== 內海潮汐 ==
根據[[新加坡大學|新加坡大學]]遙測及影像處理中心（Centre for Remote Imaging, Sensing and Processing (CRISP)）在1996年到2001年間的[[合成孔徑雷達|合成孔徑雷達]]影像顯示，蘇祿海潮汐來源為蘇祿內海東南方的三個地點，在不同潮汐源延伸後合併，並向西北方傳播，內海西邊有明顯潮汐折射與反射現象，東北方海域則無明顯潮汐，<ref name = "利用SAR 影像研究蘇祿海的內波_2003"> {{ cite web | title = 利用SAR 影像研究蘇祿海的內波 | url = http://www.airitilibrary.com/Publication/alDetailedMesh?docid=10218661-200309-8-3-1-14-a | publisher = 航測及遙測學刊 ； 8卷3期 (2003 / 09 / 01) ， P1 - 14 | author = 許明光(Ming-Kuang Hsu) ； 劉安國(Antony K. Liu) ； 李周衡(Chou-Han Lee)| language = zh-tw,en | date = 2003-09-01 }} </ref>

==文化==
据传在[[星际旅行|星际旅行]]中，[[苏鲁光|苏鲁光]]这一角色的名字就是来自于苏禄海，以代表[[亞洲人|亞洲人]]。

== 治安 ==
2017年6月19日，因蘇祿海附近[[海盜|海盜]]與[[阿布沙耶夫|阿布沙耶夫]]集團危害治安，菲律賓與[[印尼|印尼]]、[[馬來西亞|馬來西亞]]三國的國防部長在印尼北部港灣城市[[打拉根|打拉根]]會談，決定以打拉根市為海軍聯合指揮中心，展開以蘇祿海為中心的三國聯合海上巡邏，以解決恐怖份子及海盜的治安問題。<ref name = "印馬菲聯合巡邏蘇祿海 3國嚴防IS亞洲生根_2017"> {{ cite web | title = 印馬菲聯合巡邏蘇祿海 3國嚴防IS亞洲生根 | url = https://www.ettoday.net/news/20170620/949029.htm | publisher = ETtoday新聞雲 | language = zh-tw | date = 2017-06-20 }} </ref>
