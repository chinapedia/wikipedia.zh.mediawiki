'''苏缄'''（{{bd|1016年||1076年|3月1日}}），[[字|字]]'''宣甫'''，[[北宋|北宋]][[泉州|泉州]][[晋江|晋江]]出身，曾长期在[[华南|华南]]任地方官。在[[越南|越南]][[李朝_(越南)|李朝]]进攻[[邕州|邕州]]的战役中邕州城破，时任邕州知州的苏缄拒绝投降，率全家自尽。后被奉为邕州[[城隍|城隍]]。

== 出身和早年 ==
苏缄的曾祖父是[[唐朝|唐]]末[[五代|五代]]中的[[苏益|苏益]]，[[苏益|苏益]]隨著[[威武軍|威武軍]][[節度使|節度使]][[王信臣|王信臣]]入閩，從此一家居閩，傳至苏缄。苏缄是[[蘇結|蘇結]]的兄長，並且是[[苏颂|苏颂]]的堂叔。[[宝元|宝元]]元年（1038年）苏缄中进士，任[[南海区|南海]][[主簿|主簿]]，后调阳武尉。此间他以正直勇猛闻名。由于他颇有政绩，之后被提为秘书丞，后为[[英州|英州]]知州。

== 侬智高举事及其后 ==
北宋[[皇祐|皇祐]]四年（1052年）[[侬智高|侬智高]]举事，进攻[[广州|广州]]。苏缄认为广州是中心地区，不去救援就会无义，便将公务暂委下属，亲自招募数千兵士前往救援。苏缄捕杀了[[侬智高|侬智高]]手下谋士黄师宓之父，使对手丧失斗志，乘机狙击而立下战功。[[宋仁宗|宋仁宗]]为表嘉奖，任苏缄为供备库副使、[[广东|广东]]都监，管押两路兵甲，遣中使赐朝衣、金带。但后来在追击的战斗中失利，主将[[陈曙|陈曙]]被诛，苏缄也被降为[[房县|房州]]司马。后苏缄为著作佐郎，在监[[越州|越州]]税十余年后，才恢复原级官职。[[宋英宗|宋英宗]]即位后苏缄先后任[[廉州|廉州]]、[[鼎州|鼎州]]知州。

== 邕州之战 ==
=== 战前 ===
宋[[熙宁|熙宁]]四年（1071年）苏缄任皇城使[[邕州|邕州]]知州。时先后任[[桂州|桂州]]知州的[[沈起|沈起]]和[[刘彝|刘彝]]对[[交趾|交趾]][[李朝_(越南)|李朝]]采取强硬态度，拒绝通商，遭到苏缄的反对，但刘彝反过来斥责苏缄。当时宋越关系紧张至极，终于爆发战争。
=== 玉碎邕州之战 ===
{{see also|宋越熙宁战争}}
1075年12月，[[李朝_(越南)|李朝]]辅国太尉[[李常杰|李常杰]]率近十万军队进攻[[广西|广西]]，连破[[钦州|钦]][[廉州|廉]]二州。此时另一越将[[宗亶|宗亶]]已包围[[邕州|邕州]]，而邕州正规军不过2800余人。苏缄倾公库和己之所有招募兵士，进行顽强抵抗。[[李常杰|李常杰]]则参与合围邕州。

苏缄曾向[[沈起|沈起]]和[[刘彝|刘彝]]求救，但刘彝派出的张守节被[[李常杰|李常杰]]截杀，邕州孤立无援。<ref>[[李焘|李焘]]《[[续资治通鉴长编|续资治通鉴长编]]》记载：“辛酉，广南西路都监张守节为交贼所败于昆仑关。先是，苏缄遣使诣桂州请救，刘彝遣守节往援，守节闻贼十倍，逗留不即行，复迂取贵州路，驻兵康和驿以观胜负。缄又遣使持蜡书告急于提点刑狱宋球，球得书惊且泣，以便宜督守节进兵。守节惶遽不知所为，移屯火夹岭，回保昆仑关，猝遇贼，不及阵，一军皆覆，守节死之。”[[彭百川|彭百川]]《太平治迹统类》卷17：“时遣使请救于彝，彝遣都监张守节领三千兵以赴援。守节闻贼众十倍逗遛不即行，复迂途贵州驻兵康和驿，阴观胜负。缄又遣蜡丸告急于提点刑狱宋球，球得书惊泣，以便宜督，守节惶怖移军大夹岭，回保昆仑关，猝遇贼，不及阵，一军俱覆。”</ref>苏缄虽亲自上城射杀敌军，鼓舞士气，终是寡不敌众，城中粮尽水绝，在坚守42天之后，邕州于1076年3月1日（[[阴历|阴历]][[正月十二|正月十二]]）破城。苏缄在求援无望的情况下，决心“吾义不死贼手”，杀全家三十六人后自焚。仅其长子[[苏子元|苏子元]]因任职外地而在战前被苏缄遣走而幸免，但子元妻及子均未逃死难。李军找不到苏缄遗体，便在邕州大开杀戒，杀五万余人，但全城无人投降。

== 身后 ==
[[宋神宗|宋神宗]]在得知苏缄死节的消息后感到难过，召见[[苏子元|苏子元]]对苏缄的功劳给予很高的评价，认为他拖住李军而保住[[桂州|桂州]]、[[象郡|象郡]]，使局势不至于一边倒，是[[张巡|张巡]]、[[许远|许远]]也不能超过的功劳。苏缄获遗赠奉国军[[节度使|节度使]]，谥'''忠勇'''，赐都城甲第五；遗属获乡里上田十顷。苏子元以其父被任为西头供奉官、阁门祗候。

== 城隍身份 ==
《[[宋史|宋史]]》记载交趾军队在其后进攻[[桂州|桂州]]的途中远远见到有大军前来，军士均感到恐慌并认为是“苏皇城领兵来报怨”，被迫撤退。后来邕州人为苏缄立祠堂，尊他为当地[[城隍|城隍]]后，祠改为城隍庙。[[宋哲宗|宋哲宗]][[元祐|元祐]]中赐额'''怀忠'''。

宋[[绍兴_(南宋)|绍兴]]年间[[张汝锡|张汝锡]]在[[鳳山縣|鳳山]]麓建忠义庙，祀[[关羽|关羽]]、[[张巡|张巡]]、[[许远|许远]]，后增祀[[岳飞|岳飞]]、[[文天祥|文天祥]]、[[陆秀夫|陆秀夫]]、[[张世杰|张世杰]]、苏缄、[[赵昴发|赵昴发]]。另外邕州东街菜巷北口有为苏缄建的怀忠坊，忠义巷有忠义庙，奉祀苏缄等。在[[南安|南安]][[葵山|葵山]]有苏缄墓。1920年在苏缄一家殉难的地方竖立了一块刻有"宋苏忠勇公成仁处"的石碑。邕州知州苏缄殉难遗址于2002年被公布为市级文物保护单位。
