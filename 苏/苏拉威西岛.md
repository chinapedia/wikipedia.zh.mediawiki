'''苏拉威西岛'''（{{lang-id|Sulawesi}}）舊名'''西里伯斯島'''（{{lang-nl|Celebes}}）是[[印度尼西亚|印度尼西亚]]东部的一个大[[岛屿|岛屿]]。

== 地理 ==
[[File:Sulawesi_Topography.png|thumb]]
苏拉威西岛按[[面积|面积]]是世界第十一大岛，{{Fact|总面积为174,600[[平方公里|平方公里]]|time=2017-04-06T12:04:16+00:00}}，位于印度尼西亚东部，[[菲律宾|菲律宾]]南方，形状非常特别，类似一个大K字母，有四个[[半岛|半岛]]，岛中部是险峻的山区，因此四个半岛之间很少来往，从海路连接比从陆路方便。

苏拉威西岛的行政区域分为6个省：[[西蘇拉威西省|西蘇拉威西省]]、[[北蘇拉威西省|北蘇拉威西省]]、[[中蘇拉威西省|中蘇拉威西省]]、[[南蘇拉威西省|南蘇拉威西省]]、[[東南蘇拉威西省|東南蘇拉威西省]]和[[哥伦打洛省|哥伦打洛省]]。最大的城市是西南部的[[望加锡|望加锡]]，还有北部的[[万鸦老|万鸦老]]也是较大的城市。

== 动植物 ==

苏拉威西岛正位于[[华莱士线|华莱士线]]上<ref>丁楓峻(2010),  東南亞自助潛水趣.  華都文化. 頁30. ISBN 978-986-6860-93-5</ref>，因此既具有[[东洋界|东洋界]]也具有[[澳新界|澳新界]]的[[动物|动]][[植物|植物]]，其中大部分属于[[澳新界|澳新界]]的，有2,290[[平方千米|平方千米]]的区划被划归罗尔·林度国家公园。

苏拉威西岛目前已知有127种[[哺乳动物|哺乳动物]]，其中62%（79[[种|种]]）是本岛特有种，在世界其他地方不存在，其中最多的是一种矮水牛。[[鸟|鸟]]类有34%是当地特有种。

== 文化艺术 ==

苏拉威西人非常重视[[艺术|艺术]]，包括制造[[陶器|陶器]]、织物和[[舞蹈|舞蹈]]，他们制造的陶器非常精制；织物非常细致，有重复的图案。男人的舞蹈雄武有力，女人柔美流畅，舞蹈经常能讲述一个故事情节。

== 宗教 ==

大部分苏拉威西人信奉[[伊斯兰教|伊斯兰教]][[逊尼派|逊尼派]]，但有19%的人信奉[[基督宗教|基督宗教]]，其中2%信仰[[罗马天主教|罗马天主教]]，17%的人信奉[[基督新教|基督新教]]，主要集中在北部和中部，以及一些大城市，其中[[北蘇拉威西省|北蘇拉威西省]]的基督徒人口比例更是佔絕大多數，超過70%。但大部分人不管是信奉基督宗教还是伊斯兰教，仍然相信当地的原始信仰和当地的神祇。 

此外还存在一些外来人口的社区信仰[[印度教|印度教]]或[[佛教|佛教]]，主要在[[印度|印度]]人和华人之间。

== 自然災害 ==
2018年9月28日，蘇拉威西島遭逢規模7.5強震，隨後，中蘇拉威西島[[帕盧|帕盧]]市和[[棟加拉縣|棟加拉縣]]發生大規模[[海嘯|海嘯]]，造成逾千人罹難<ref> [http://news.ltn.com.tw/news/world/breakingnews/2569203 才剛遭強震、海嘯重創 印尼蘇拉威西島今早火山爆發。]自由時報 2018-10-03</ref>。5天後，2018年10月3日早上8點多，蘇拉威西島北方的[[索普坦火山|索普坦火山]]突然爆發，火山灰衝上4000公尺高，災害損失難以估計。
