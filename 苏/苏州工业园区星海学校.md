'''苏州工业园区星海学校'''始办于2000年9月，位于[[苏州|苏州]]城区东侧，[[苏州工业园区|苏州工业园区]]境内。是苏州工业园区管委会直属的第一所按照现代化标准建设的十二年一贯制公立学校，学校下辖[[苏州工业园区星海小学|苏州工业园区星海小学]]和[[苏州工业园区星海实验中学|苏州工业园区星海实验中学]]（初中、高中）。<ref>{{cite web |url=http://sipedu.sipac.gov.cn/School/SchoolDetail.aspx?ID=8 |title=存档副本 |accessdate=2011-04-23 |deadurl=yes |archiveurl=https://web.archive.org/web/20120531175450/http://sipedu.sipac.gov.cn/School/SchoolDetail.aspx?ID=8 |archivedate=2012-05-31 }}</ref><ref>{{cite web |url=http://sipedu.sipac.gov.cn/School/SchoolDetail.aspx?ID=7 |title=存档副本 |accessdate=2011-04-23 |deadurl=yes |archiveurl=https://web.archive.org/web/20120531175427/http://sipedu.sipac.gov.cn/School/SchoolDetail.aspx?ID=7 |archivedate=2012-05-31 }}</ref>
