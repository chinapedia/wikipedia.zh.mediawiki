'''蘇沃洛夫勳章'''（[[俄文|俄文]]：'''{{lang|ru|Орден Суворова}}'''）是一種[[蘇聯|蘇聯]]和[[俄羅斯|俄羅斯]]頒發的[[勳章|勳章]]，共分三級。该勳章是紀念[[亞歷山大·蘇沃洛夫|亞歷山大·蘇沃洛夫]]，於1942年7月29日（[[二戰|二戰]]期間）由[[蘇聯|蘇聯]][[蘇聯最高蘇維埃|最高蘇維埃]]主席團決定設立。在[[胜利勋章|胜利勋章]]诞生之前，该勋章中的第一级是等级最高的苏联軍功勋章。 <ref> {{cite book|author = 唐思、謝亮|title = 《號角：世界經典制服勳章藝術》| location = 北京|publisher = 中國長安出版社|year = 2013|page = 第166-172頁||isbn = 978-7-5107-0679-0}}</ref> <ref>[http://www.unitedcn.com/05JSZL/08xunzhang/new_page_2006.htm 苏沃洛夫勋章] {{webarchive|url=https://web.archive.org/web/20070806020938/http://www.unitedcn.com/05JSZL/08xunzhang/new_page_2006.htm |date=2007-08-06 }}</ref>


==頒授情況==
一級蘇沃洛夫勳章頒授了391枚，其中3枚屬部隊或兵團所有；二級蘇沃洛夫勳章頒授了2862枚，其中676枚屬部隊或兵團所有；三級蘇沃洛夫勳章頒授了4012枚，其中849枚頒授予部隊或兵團。

[[蘇聯元帥|蘇聯元帥]][[格奧爾基·康斯坦丁諾維奇·朱可夫|格奧爾基·康斯坦丁諾維奇·朱可夫]]為首位一級蘇沃洛夫勳章的獲得者

==頒授條件==
此勳章授予「指揮部隊，組織作戰行動並表現果斷、不懈而取得卓越成就，為祖國取得勝利的紅軍指揮員」，也可授予作戰部隊和兵團。

一級蘇沃洛夫勳章授予[[方面軍|方面軍]]、[[集團軍|集團軍]]司令、副司令、參謀長、作戰部長、作戰局長和兵種([[炮兵|炮兵]]、[[航空兵|航空兵]]、[[裝甲兵|裝甲兵]]、[[火箭炮|火箭炮兵]])主任；

二級蘇沃洛夫勳章授予[[軍長|軍長]]、[[師長|師長]]、[[旅長|旅長]]及副職、參謀長；

三級蘇沃洛夫勳章授予[[團長|團長]]、[[營長|營長]]及團參謀長。

==勳章樣式==
整章呈五角星形，中央圓框內為蘇沃洛夫頭像，上半部由俄文的[[亞歷山大·蘇沃洛夫|亞歷山大·蘇沃洛夫]]（{{lang|ru|Алекса́ндр Суво́ров}}）圍繞著；下半部則為月桂枝和及橡葉圖案，外圈是一道道五角劍芒。

一級蘇沃洛夫勳章對角距離為56毫米，章體分為兩片：金質頭像與圓盤、铂質劍芒。兩片由前後铆接固定，劍芒12點方向還另外铆接著一枚紅色琺瑯五角星。

二級蘇沃洛夫勳章對角距離為49毫米，章體分為兩片：銀質頭像與圓盤、金質劍芒。兩片由前後铆接固定。

三級蘇沃洛夫勳章對角距離為49毫米，章體全為銀質。1944年前仍分兩片，前後铆接；1944年後簡化為一整片。
