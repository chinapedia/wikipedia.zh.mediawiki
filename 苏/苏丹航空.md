'''苏丹航空'''（{{lang-ar|الخطوط الجوية السودانية}}）是一家[[苏丹|苏丹]]的[[国家航空公司|国家航空公司]]，总部设在[[苏丹|苏丹]][[首都|首都]][[喀土穆|喀土穆]]。该航空公司以[[喀土穆国际机场|喀土穆国际机场]]为[[枢纽机场|枢纽]]，主要经营国内航线以及[[北非|北非]]、[[中东|中东]]区域内的国际航线。苏丹航空是[[阿拉伯航空运输组织|阿拉伯航空运输组织]]的成员。

== 历史 ==
[[File:Sudan_Airways_A310-300_ST-AST_DXB_2006-11-17.png|缩略图]]]]
[[File:8dn_-_Fine_Air_DC-8-61F;_N30UA@MIA;24.01.1998_(5574216086).jpg|缩略图]]
苏丹航空成立于1947年，首批飞机是4架[[哈维兰鸽|哈维兰鸽]]8座飞机。1952年，苏丹航空引进7架28座的[[道格拉斯DC-3|道格拉斯DC-3]]。这些飞机在飞往[[亚丁_(也门)|亚丁]]、[[阿斯马拉|阿斯马拉]]、[[贝鲁特|贝鲁特]]、[[开罗|开罗]]、[[吉达|吉达]]等地的国际航线上服务。<ref name="history">{{cite web |url=http://www.sudanair.com/history.htm |title=History |publisher=苏丹航空 |date= |accessdate=2009年5月1日 |language=en |deadurl=yes |archiveurl=https://web.archive.org/web/20090420013036/http://www.sudanair.com/history.htm |archivedate=2009年4月20日 }}</ref>

1959年，苏丹航空加入[[国际航空运输协会|国际航空运输协会]]。后来，苏丹航空成为非洲最早引进[[福克F27|福克F27]]的航空公司。1970年代，苏丹航空开始引进[[波音707|波音707]]和[[波音737|波音737]]；1980年代引进[[福克F50|福克F50]]；1990年代引进[[空中客车A310|空中客车A310]]。<ref name="history"/>

2007年，苏丹政府对苏丹航空进行私有化改革。改革之后，政府仅持有苏丹航空30%的股份。新的投资者正在加强苏丹航空的预订系统、电子客票、电子值机等方面的技术升级。

2008年6月23日，苏丹民航管理局宣布对苏丹航空进行停飞，理由是违反民航安全要求。民航管理局表示这项决定与6月10日发生的[[苏丹航空109号航班空难|苏丹航空109号航班空难]]无关。<ref>{{cite web |url=http://news.bbc.co.uk/2/hi/africa/7467423.stm |title=Sudan crash airline is grounded |first=Amber |last=Henshaw |publisher=[[BBC|BBC]] |date=2008年6月21日 |accessdate=2009年5月1日 |language=en }}</ref>但数小时后，苏丹航空排除问题、申诉成功，获准恢复飞行。<ref>{{cite web |url=http://www.usatoday.com/travel/flights/2008-06-24-sudan-air-ground_N.htm |title=Sudan Airways resumes flights after brief grounding |publisher=usatoday.com引自[[美联社|美联社]] |date=2008年6月24日 |accessdate=2009年5月1日 |language=en }}</ref>

== 事故及意外 ==
* 1971年12月6日，苏丹航空一架[[航空器注册编号|注册号]]为ST-AAY的福克F27飞机被[[劫机|劫持]]后在苏丹城市蒂卡卡迫降，造成9名乘客和1名机组人员丧生，当时机上共载有37名乘客和5名机组人员。<ref>{{cite web |url=http://www.planecrashinfo.com/1971/1971-64.htm |title= ACCIDENT DETAILS |publisher=planecrashinfo.com |date= |accessdate=2009年5月1日 |language=en }}</ref>
* 1975年3月18日，苏丹航空一架注册号为ST-ADB的哈维兰客机在苏丹辛贾失事。4个乘客中仅有一人幸存，两名机组人员均遇难。<ref>{{cite web |url=http://www.planecrashinfo.com/1975/1975-22.htm |title= ACCIDENT DETAILS |publisher=planecrashinfo.com |date= |accessdate=2009年5月1日 |language=en }}</ref>
* 1986年8月16日，苏丹航空一架注册号为ST-ADY的福克F27飞机从苏丹[[马拉卡勒|马拉卡勒]]起飞，预定飞往喀土穆。飞机起飞后不久被[[苏丹人民解放军|苏丹人民解放军]]用[[地对空导弹|地对空导弹]]击落，机上57名乘客和3名机组人员全部遇难。<ref>{{cite web |url=http://www.planecrashinfo.com/1986/1986-37.htm |title= ACCIDENT DETAILS |publisher=planecrashinfo.com |date= |accessdate=2009年5月1日 |language=en }}</ref>
* 2003年7月8日，苏丹航空一架注册号为ST-AFK的[[波音737|波音737]]飞机从苏丹港起飞，预定飞往喀土穆。起飞10分钟后，机组人员报告引擎熄灭，返航时在距机场3英里处坠毁。机上载有106名乘客和11名机组人员，仅有一名3岁男孩幸存。<ref>{{cite web |url=http://www.planecrashinfo.com/2003/2003-31.htm |title= ACCIDENT DETAILS |publisher=planecrashinfo.com |date= |accessdate=2009年5月1日 |language=en }}</ref>
* 2008年6月10日，[[苏丹航空109号航班空难|苏丹航空109号航班]]由一架注册号为ST-ATN的空中客车A310-324飞机执行，由约旦安曼起飞、经停叙利亚大马士革后，抵达喀土穆。飞机在喀土穆国际机场降落时遇到恶劣天气，冲出[[跑道|跑道]]。机上214名乘客及机组人员中共有32人遇难。<ref>{{cite web |url=http://www.planecrashinfo.com/2008/2008-28.htm |title= ACCIDENT DETAILS |publisher=planecrashinfo.com |date= |accessdate=2009年5月1日 |language=en }}</ref>
