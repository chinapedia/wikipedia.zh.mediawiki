'''唯一的愛'''（[[日文|日文]]：{{lang|ja|たったひとつの恋}}），是從2006年10月14日開始，每週星期六在21:00-21:54(JST)在[[日本電視台|日本電視台]]及其聯播網（[[NNS|NNS]]）播出的連續電視劇。第一集加長20分，到22:14為止。平均[[收視率|收視率]]11.6%。

== 劇情簡介 ==

神崎弘人（龜梨和也 飾）父親去世，跟母親和病弱的弟弟3人相依為命，雖然繼承了父親的修船廠，但極為破舊且規模不大，瀕臨倒閉。弘人在與友人大澤亞裕太（平岡祐太 飾）、草野甲（田中聖 飾）前往出售自己所釣的漁獲的路上，與全國拓展連鎖的橫濱·元町星鑽珠寶店店長的千金月丘菜緒（綾瀨遙 飾）撞個滿懷，且因緣際會的得到[[慶應大學|慶應大學]]與[[橫濱女學館大學|橫濱女學館大學]]學生聯誼餐會的邀請。兩人在餐會上再度相遇，生活環境和性格都截然不同的兩人，以[[橫濱市|橫濱]]為舞台，展開刻骨銘心的苦戀……

== 主要角色 ==
*神崎弘人--[[龜梨和也|龜梨和也]]

:男主角，繼承父親遺留的神崎造船鐵工廠。外表乍看冷酷，但其實是個默默支撐家庭的溫柔男性，對女性相當專情，即使是死也要守護在菜緒身邊。高中時代參加棒球隊且獲得縣大會準決勝，一心想進入大學繼續棒球夢，但因父親突然死去而作罷。

*月丘菜緒--[[綾瀨遙|綾瀨遙]]

:女主角，橫濱·元町星鑽珠寶店店長的千金，就讀於橫濱女學館大學。個性坦率、情感豐富，中學時代曾因罹患血癌而進入[[特殊學校|養護學校]]就讀。

*草野甲--[[田中聖|田中聖]]

:弘人的好友，卡車司機，外表憨直、內心天真的浪漫主義者。

*大沢亜裕太--[[平岡祐太|平岡祐太]]

:弘人的好友，就讀於橫濱汽車修護專門學校。外表帥氣，個性卻優柔寡斷，但對朋友卻很講義氣，在露天咖啡廳打工。

*本宮裕子--[[戶田惠梨香|戶田惠梨香]]

:菜緒的好友，同樣就讀於橫濱女學館大學，老家在神戶開設醫院。與甲私底下密切交往，但在好友面前總對甲表現出冷淡的態度。

*月丘達也--[[要潤|要潤]]

:菜緒的哥哥，研究所在學中，是家族企業的內定接班人，但其實內心真正想要做的是研究人員。自從將骨髓捐贈給菜緒後，便對菜緒十分溺愛。

*月丘みつこ--[[田中好子|田中好子]]（特別出演）

:菜緒的母親，由於[[天然呆|天然呆]]（装傻/耗呆）的個性而成為月丘家的歡樂來源。

*月丘雅彦--[[財津和夫|財津和夫]]

:菜緒的父親。『星鑽珠寶』社長，希望達也與菜緒能繼承家業，堅決反對菜緒與弘人交往。

*神崎亜紀子--[[余貴美子|余貴美子]]

:弘人的母親，因家境貧寒而從事特種行業，對家中的事情經常袖手旁觀、漠不關心。

*神崎廉--[[齋藤隆成|齋藤隆成]]

:弘人的弟弟，個性樂觀開朗，但因為因為虛弱的體質而就讀養護學校。
