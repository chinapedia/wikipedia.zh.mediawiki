'''萍乡县'''，[[中国|中国]]旧[[县|县]]名。

[[三国|三国]][[孫吳|东吴]]政权，于[[宝鼎|宝鼎]]二年（267年）置县，县治在今[[芦溪县|芦溪县]]境。[[隋朝|隋朝]]时，属[[袁州_(古代)|袁州]]。[[唐朝|唐朝]][[武德|武德]]二年（619年），迁治于萍乡凤凰池（今萍乡市区）<ref name="a1"/>。属袁州、[[宜春郡|宜春郡]]<ref>《[[新唐书|新唐书]]·卷四十一·志第三十一》◎地理五</ref>。[[宋朝|宋朝]]时，仍属袁州<ref>《[[宋史|宋史]]·卷八十八·志第四十一》◎地理四</ref>。[[元朝|元朝]]时，升为[[萍乡州|萍乡州]]。[[明朝|明朝]][[洪武|洪武]]二年（1369年），复为县，属[[袁州府|袁州府]]。[[清朝|清朝]]时，考评：[[衝繁疲難|冲，繁]]<ref>《[[清史稿|清史稿]]·卷六十六·志四十一》◎地理十三......袁州府......萍乡冲，繁。府西一百四十里。西：徐仙山。南：笔架。东南：罗霄山，罗霄水出焉。分二支：一东流合牵水、渝水，折东迳宣凤汛，入宜春，为袁江；一西流合泉江，迳城南，会罗霄西北水，折西北，迳湘东镇，右合平溪岭水，入湖南醴陵，注渌江。四镇：宣风、芦溪、上栗、插岭关。巡司二：芦溪市、安乐。有草市废司。铁路达湖南醴陵。万载......</ref>。[[晚清|晚清]]时，萍乡境内的[[安源煤矿|安源煤矿]]得到开发。

民国初年，属[[庐陵道|庐陵道]]，民国十五年（1926年），直属于[[江西省_(中華民國)|江西省]]。民国二十年，属第八行政区。民国二十四年，属第二行政区<ref name="a1">{{Cite web |url=http://www.pingxiang.gov.cn/zjpx/sqgl/201206/t20120614_178830.htm |title=萍乡的历史沿革 |accessdate=2016-08-19 |author= |date=2016-01-03 |work= |publisher=萍乡市人民政府网站，来源：市史志办 |language=Zh-hans |deadurl=yes |archiveurl=https://web.archive.org/web/20160408065350/http://www.pingxiang.gov.cn/zjpx/sqgl/201206/t20120614_178830.htm |archivedate=2016-04-08 }}</ref>。

1949年，[[中国人民解放军|解放军]]取得[[渡江战役|渡江战役]]胜利后，国军[[白崇禧|白崇禧]]部在湘鄂赣边界布防，国军第46军驻守萍乡。7月，解放军发起[[湘赣战役|湘赣战役]]。19日，[[中国人民解放军第四野战军|四野]][[中国人民解放军第45军|第45军]]135师进入今萍乡市的桐木、上栗一带。20日在关下，135师与意图逃往湖南的国军第46军175师展开遭遇战。

此前，中共地下组织已逐步掌握萍乡境内的电厂、报刊、警察、自卫队等组织。中华民国萍乡县县长及县政府官员则已逃跑。23日，四野[[中国人民解放军第43军|第43军]]128师某侦察分队<ref>{{Cite web |url=http://xxgk.pingxiang.gov.cn/ayq/qzbmxxgk/qdaj/gzdt/zwdt/201603/t20160303_1460545.htm |title=解放初期的萍乡“青干班” |accessdate=2016-08-19 |author= |date=2016-02-19|work= |publisher=中国萍乡市人民政府网站，来源：安源区政府信息公开——区档案局|language=Zh-hans }}</ref>，由芦溪进入萍乡县城。26日，四野135师受军部命令驻守萍乡。29日，成立萍乡县人民政府<ref>{{Cite web |url=http://www.jx.chinanews.com/2015/0429/1765873.html |title=史海钩沉：萍乡解放始末 |accessdate=2016-08-19 |author=朱隆起 |date=2015-04-29 |work= |publisher=江西新闻网|language=Zh-hans }}</ref>。

1949年后，属[[袁州专区|袁州专区]]。1952年撤消袁州专区时，划入[[南昌专区|南昌专区]]。后归[[宜春专区|宜春专区]]。1960年改为萍乡市。
