'''袁象先'''（{{bd|860年代||924年|7月11日|catIdx=Y}}？<ref name=HFD32>据[[:s:舊五代史/卷32|《旧五代史》卷三十二]]，同光二年六月甲戌（924年7月11日），宋州奏报李绍安死讯，但消息传递需要时间，李绍安实际死亡的日期应在此之前。</ref>），[[后唐庄宗|后唐庄宗]]年间曾暫名'''李绍安'''，[[中国|中国]][[五代十国|五代十国]]时期[[后梁|后梁]]和[[后唐|后唐]]将领。他是后梁开国之君[[梁太祖|梁太祖]][[朱全忠|朱全忠]]的外甥，弑[[朱友珪|朱友珪]]、立[[朱友贞|朱友贞]]。后梁灭后降后唐。

== 家世 ==
袁象先生于[[唐懿宗|唐懿宗]][[咸通|咸通]]五年（864年）或六年（865年）。<ref name=HFD59>[[:s:舊五代史/卷59|《旧五代史》卷五十九]]</ref><ref name=NHFD45>[[:s:新五代史/卷45|《新五代史》卷四十五]]</ref>父袁敬初，自称[[唐中宗|唐中宗]]朝[[中书令|中书令]]、南阳郡王[[袁恕己|袁恕己]]之后。曾祖袁进朝，[[成都|成都]]少尹。祖父袁忠义，忠武军节度判官。母[[万安大长公主|朱氏]]，朱全忠妹，但在后梁建立前去世。袁象先为人宽厚慷慨。曾经射一只水鸟，不中，但入水的箭却射穿了一双鲤鱼，时人以为异象。袁象先显贵后，曾祖赠[[左仆射|左仆射]]，祖父赠[[司空|司空]]，父亲官至太府卿，赠[[司徒|司徒]]、[[驸马都尉|驸马都尉]]。<ref name=HFD59/>

== 唐朝年间效力朱全忠 ==
[[中和_(唐朝)|中和]]三年（883年）袁象先的舅父朱全忠被任为[[宣武军节度使|宣武军节度使]]，<ref name=ZZTJ255>[[:s:資治通鑑/卷255|《资治通鉴》卷二百五十五]]</ref>袁象先在此时或不久后开始军旅生涯，最初授[[银青光禄大夫|银青光禄大夫]]、[[检校|检校]]太子宾客、兼[[御史中丞|御史中丞]]，[[景福_(唐朝)|景福]]元年在检校左省常侍任上迁检校[[工部尚书|工部尚书]]，充宣武军马军指挥使兼左静边都指挥使。[[乾宁|乾宁]]五年，再迁检校右仆射、左领军卫将军同正，充宣武军内外马步军都指挥使。[[光化|光化]]二年（899年），权知[[宿州|宿州]]军州事。<ref name=HFD59/>当年正月，[[幽州节度使|幽州节度使]][[刘仁恭|刘仁恭]]攻[[魏州|魏州]]，三月，朱全忠派[[李思安|李思安]]等往救，屯[[内黄|内黄]]。刘仁恭派儿子[[义昌军节度使|义昌军节度使]][[刘守文|刘守文]]和女婿[[单可及|单可及]]攻之，李思安命袁象先伏兵清水之右，自己迎战于[[繁阳|繁阳]]，假装不胜退走，刘守文追杀到内黄以北，李思安回军与伏兵夹击之，幽州军大败，单可及被斩，被俘杀三万人，刘守文仅以身免，幽州军丧气。<ref>[[:s:舊五代史/卷135|《旧五代史》卷一百三十五]]</ref><ref name=ZZTJ261>[[:s:資治通鑑/卷261|《资治通鉴》卷二百六十一]]</ref>[[天复_(唐朝)|天复]]元年（901年），表授刺史，充宿州团练、埇桥镇遏都知兵马使。二年（902年），朱全忠的敌人[[淮南节度使|淮南节度使]][[杨行密|杨行密]]围宿州，袁象先因援兵未至颇为忧沮。一日登北城，在楼堞之上休息，失意似乎睡着了，梦见人告诉他：“我是陈璠，曾板筑这座城，旧府邸犹在，如今为军舍，可为我立庙，即用阴兵助公。”袁象先从之。淮南军急攻宿州营垒，州城几乎陷落，突然有大风雨，居民望见城上兵甲无数，淮南军不能进，即时退去。袁象先才相信鬼神之助，于是为之立祠，当地人一直祷祝。军校[[康怀贞|康怀贞]]奉命驰援宿州，淮南军退后，康怀贞权知宿州刺史，<ref>见《两五代史》康怀贞本传。</ref>袁象先离任。三年（903年），权知[[洺州|洺州]]军州事。[[天祐_(唐朝)|天祐]]三年（906年），授[[陈州|陈州]]刺史、检校[[司空|司空]]。<ref name=HFD59/>

== 后梁年间 ==
朱全忠篡唐建立后梁，为太祖皇帝后，[[开平_(后梁)|开平]]元年（907年）四月，袁象先献上白兔一只，被付史馆编录，兼示百官。<ref>[[:s:舊五代史/卷3|《旧五代史》卷三]]</ref>二年（908年），召回袁象先，授左英武军使，再迁左神武、右羽林统军。三年（909年），转右卫上将军，封'''汝南县男'''。四年（910年），权知宣武军留后，当时因宣武军前军部大梁已成为东都，宣武军军部已迁到[[宋州|宋州]]。<ref name=HFD4>[[:s:舊五代史/卷4|《旧五代史》卷四]]</ref>到任五个月，又改天平军两使留后。当时，[[郓州|郓州]]大饥荒，户民流散，袁象先开仓赈灾，百姓信赖他。五年（911年）二月，太祖北伐[[赵_(五代)|赵国]]和义武军（当时和后梁大敌[[后唐|晋]]都与赵结盟），以袁象先为镇定东南行营都招讨应接副使，进封开国伯，命他围[[脩县|脩县]]，不能克，后来后梁撤军。<ref name=ZZTJ268>[[:s:資治通鑑/卷268|《资治通鉴》卷二百六十八]]</ref>袁象先曾奉诏入朝，郓州人留他，毁石桥，他只得从其他门逃走。<ref name=HFD59/>

[[乾化|乾化]]二年（912年）二月，太祖大举河南之众以援正被晋国攻打的刘仁恭之子燕帝[[刘守光|刘守光]]，以[[平卢节度使|平卢节度使]][[贺德伦|贺德伦]]为应接使，袁象先为副。后袁象先回京任左龙武统军、在京马步军都指挥使。六月，太祖为子郢王[[朱友珪|朱友-{珪}-]]所弑。朱友-{珪}-杀养兄博王[[朱友文|朱友文]]，称朱友文弑君，夺位。但很多将领都知道朱友-{珪}-才是弑君者，袁象先很快联络大将[[魏博节度使|魏博节度使]][[杨师厚|杨师厚]]、太祖婿驸马都尉[[赵岩_(后梁驸马都尉)|赵岩]]、太祖子开封尹、东都留守、均王[[朱友贞|朱友贞]]及右监门上将军[[霍彦威|霍彦威]]，合谋推翻朱友-{珪}-。<ref name=HFD64>[[:s:舊五代史/卷64|《旧五代史》卷六十四]]</ref>赵岩回到洛阳与袁象先秘密定计，杨师厚遣小校王舜贤至洛阳，秘密与袁象先、赵岩图谋，并给袁象先和主军大将们写信。<ref>[[:s:舊五代史/卷22|《旧五代史》卷二十二]]</ref>[[鳳曆_(後梁)|鳳曆]]元年（913年）二月，朱友贞故意激怒在洛阳的左右龙骧军，称朱友-{珪}-要坑杀他们，告谕他们讨伐朱友-{珪}-，被奉为主后，遣人催促赵岩、袁象先、[[傅晖|傅晖]]、[[朱珪|朱珪]]等起事。当月，时任左龙虎统军、侍卫亲军都指挥使的袁象先等率禁兵数千人（或作千人）入宫，围朱友-{珪}-，朱友-{珪}-绝望与妻[[张皇后_(朱友珪)|张皇后]]和仆夫[[冯廷谔|冯廷谔]]一同自杀。<ref>[[:s:舊五代史/卷12|《旧五代史》卷一十二]]</ref><ref>[[:s:新五代史/卷23|《新五代史》卷二十三]]</ref>乱中袁象先所部禁兵重伤[[礼部尚书|礼部尚书]]、平章事、集贤殿大学士、判户部[[杜晓|杜晓]]致死，<ref>[[:s:舊五代史/卷18|《旧五代史》卷一十八]]</ref><ref>[[:s:新五代史/卷35|《新五代史》卷三十五]]</ref>并杀右散骑常侍侍讲[[李珽|李珽]]。<ref>[[:s:新五代史/卷54|《新五代史》卷五十四]]</ref>袁象先和赵岩控制西都[[洛阳|洛阳]]，袁象先派赵岩将[[传国玉玺|传国玉玺]]送到朱友贞所在的大梁，<ref>[[:s:新五代史/卷42|《新五代史》卷四十二]]</ref>请他去洛阳登基。朱友贞接受但选择在大梁登基，<ref>[[:s:舊五代史/卷28|《旧五代史》二十八]]</ref>改名[[朱瑱|朱瑱]]。<ref>[[:s:新五代史/卷3|《新五代史》卷三]]</ref><ref>[[:s:新五代史/卷13|《新五代史》卷一十三]]</ref>四月，袁象先因功以西京内外诸军马步军都指挥使、检校[[司徒|司徒]]、左龙虎统军、'''濮阳郡开国侯'''被授[[特进|特进]]、检校[[太保|太保]]、[[同平章事|同平章事]]，遥充镇南军（当时在敌国[[杨吴|吴]]管下）节度使、江南西道观察处置等使、行开封尹、判在京马步诸军事，进封'''开国公'''，<ref name=ZZTJ268/>增食邑一千户。<ref>[[:s:舊五代史/卷8|《旧五代史》卷八]]</ref>以[[进士|进士]][[萧希甫|萧希甫]]为书记。乾化四年（914年），授[[平卢节度使|平卢节度使]]，加检校[[太傅|太傅]]。<ref>《新五代史》误作贞明四年（918年）事，与袁象先在宣武军十年矛盾。[[钱大昕|钱大昕]]、[[吴兰庭|吴兰庭]]及[[吴廷燮|吴廷燮]]《五季方镇年表》等考证此事时皆误。详见朱玉龙《五代十国方镇年表》。</ref>又以萧希甫为巡官。萧希甫不乐，逃跑。<ref>[[:s:舊五代史/卷71|《旧五代史》卷七十一]]</ref><ref>[[:s:新五代史/卷28|《新五代史》卷二十八]]</ref>不久，袁象先移宣武军节度使，加检校[[太尉|太尉]]。在宣武军十年。[[贞明_(朱友贞)|贞明]]二年（916年）十一月，吴少摄政行军副使[[徐知训|徐知训]]为淮北行营都招讨使，围[[颍州|颍州]]。朱瑱命袁象先率援军相救，次年正月，袁象先援军到颍州，吴军实际主管军事的[[朱瑾|朱瑾]]早起看到西北昏黑，知道梁军将至，召行营都虞候[[刘权|刘权]]商议，刘权着幅巾锦袍前来，朱瑾认为他大敌当前还如此服饰，无军礼，叱斩之，徐知训徒步赤脚前去相救才得免。吴军与袁象先军交战，小有不利，撤退。<ref>《九国志·刘权传》</ref><ref name=ZZTJ269>[[:s:資治通鑑/卷269|《资治通鉴》卷二百六十九]]</ref>太祖年间，将领们贿赂横行，袁象先作为帝甥，在治下诸镇纳贿、盘剥尤其严重，因而家财巨万。<ref name=HFD59/>

== 后唐年间 ==
[[同光|同光]]元年（923年）四月，晋王[[李存勖|李存勖]]建立后唐，称帝，即庄宗。十月，发起突袭直取后梁都城大梁。梁主力远离都城不能相救，朱瑱在城陷前自杀，后梁灭亡。后唐皇帝入梁都，后梁节度使们成群来降。袁象先和镇国军留后[[霍彦威|霍彦威]]先到。袁象先以辇载所积累的珍货数十万贿赂庄宗宠妃[[刘玉娘|刘夫人]]和庄宗宠爱的权贵、伶官、宦官，使他们争相称赞自己。庄宗将汴州改回宣武军，<ref name=ZZTJ272>[[:s:資治通鑑/卷272|《资治通鉴》卷二百七十二]]</ref>而将袁象先治下治所在宋州的宣武军改名为归德军。当月，庄宗于崇元殿宴诸将，[[段凝|段凝]]、霍彦威、袁象先等都参加了。酒酣，庄宗举酒敬李嗣源：”此席宴客，皆吾前岁之劲敌，如今与吾同宴，是卿为前锋之效。”霍彦威等伏陛谢罪，庄宗说：“与卿话旧，别害怕。”赐御衣、器币，尽欢而罢。不久放他们归藩。<ref name=HFD64/><ref>[[:s:新五代史/卷46|《新五代史》卷四十六]]</ref>十一月，赐袁象先国姓李，赐名绍安，不久令他归镇。<ref>[[:s:舊五代史/卷30|《旧五代史》卷三十]]</ref>次年六月，卒于归德节度使任上，<ref name=HFD32/>赠太师。[[后周|后周]][[广顺|广顺]]中，赠[[中书令|中书令]]，追封'''楚国公'''。<ref name=HFD59/>

袁象先平生所积财产数千万，邸舍四千间，死后不分给诸子，都给儿子[[袁正辞|袁正辞]]。袁正辞以父荫为飞龙副使。<ref name=NHFD45/>
