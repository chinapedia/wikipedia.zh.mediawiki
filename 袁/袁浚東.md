'''袁浚東'''（'''Yuen Tsun Tung'''，{{bd|1991年|5月24日|||Y袁}}）出生於[[香港|香港]]，是一名已退役的職業職業足球員，可司職[[中堅|中堅]]、[[後衛_(足球)|清道夫]]、[[左閘|左閘]]、[[右閘|右閘]]、[[防守中場|防守中場]]。

他已於2012年退役，轉投懲教署工作，在2013年9月入職成為二級懲教助理<ref>{{cite web |url=http://news.takungpao.com.hk/hkol/topnews/2013-08/1873253.html |title=財爺勉懲署新丁公正執法 |publisher=香港大公報 |accessdate=2013年8月31日}}</ref>。他現時於工餘時間，仍會以業餘方式效力有多名政府紀律部隊在陣的[[香港甲組聯賽|香港甲組聯賽]]球隊[[永義足球隊|永義]]。

他有一名孖生弟弟，[[袁浚楠|袁浚楠]]，同樣效力於[[香港甲組聯賽|香港甲組聯賽]]球隊[[永義足球隊|永義]]，與他同樣司職[[中堅|中堅]]、[[清道夫|清道夫]]。

==球員生涯==
袁浚東出身自[[香港流浪足球會|流浪]]青年軍，2008年獲球會提升至一隊，轉往[[香港甲組足球聯賽|甲組]]的[[四海體育會|四海]]，司職後衛。

2010年夏天，離開[[四海體育會|四海流浪]]，轉投當時在[[香港乙組足球聯賽|乙組]]的[[元朗區體育會足球隊|元朗]]。

在2012年1月，被[[元朗區體育會足球隊|元朗]]外借到[[香港甲組足球聯賽|甲組]]的[[港菁|港菁]]效力。

2012年夏天，加盟當時另一支[[香港乙組足球聯賽|乙組]]球隊[[永義足球隊|永義]]。

==國際賽生涯==
2008年1月23日，袁浚東入選[[香港17歲以下足球代表隊|香港U–17]]<ref>{{cite web |url=http://www.hkfa.com/upload/pdf/2008/20080128c.pdf |title=2008國際青年足球邀請賽球隊名單 |publisher=香港足球總會 |accessdate=2008年1月28日}}</ref>，代表香港出席由在2008年1月25日至1月27日於藍田[[晒草灣遊樂場|晒草灣遊樂場]]草地足球場舉行的2008國際青年足球邀請賽<ref>{{cite web |url=http://www.hkfa.com/news_details/5579 |title=2008國際青年足球邀請賽 |publisher=香港足球總會 |accessdate=2008年1月28日}}</ref>。

2008年10月29日，首次入選[[香港20歲以下足球代表隊|香港U–20]]<ref>{{cite web |url=http://hk.apple.nextmedia.com/sports/art/20081029/11777762 |title=港全運隊旨吸取經驗 |publisher=香港蘋果日報 |accessdate=2008年10月29日}}</ref>，並代表香港出席在2009年5月25日至6月3日期間於中國大陸重慶舉行的[[中華人民共和國第十一屆全運會足球比賽|第十一屆山東濟南全運會足球預賽]]賽事<ref>{{cite web |url=http://www.hkfa.com/news_details/6809 |title=香港20歲以下青年隊第十一屆全運會預賽名單公佈 |publisher=香港足球總會 |accessdate=2009年5月23日}}</ref>。

2011年月6月10日，首次入選[[香港奧運足球代表隊|香港U–21]]，代表香港出席在2011年6月11日在[[小西灣運動場|小西灣運動場]]出戰[[港澳埠際足球賽|港澳埠際賽]]，並正選上陣<ref>{{cite web |url=http://www.hkfa.com/news_details/8705 |title=2011港澳埠際賽香港21歲以下代表隊名單公佈 |publisher=香港足球總會 |accessdate=2011年6月10日}}</ref>。
