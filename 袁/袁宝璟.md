'''袁宝璟'''（{{bd|1966年|2月16日|2006年|3月17日|catIdx=Y}}），中国億萬富翁，北京[[建昊集团|建昊集团]]董事长。

出生于[[辽宁省|辽宁省]][[辽阳市|辽阳市]]农民家庭。是中国首位“世界青年创业者大奖”得主；其捐资1000万元在[[全国学联|全国学联]]设立了“中国大学生跨世纪发展基金建昊奖学金”。2003年11月，袁氏四兄弟以雇凶杀人案被捕。2006年3月17日执行死刑。三兄弟都被判死刑，立即执行，与[[周永康案|周永康政法系统腐败案]]有关<ref>[http://news.163.com/14/0305/12/9MIR05H10001124J.html 刘汉攀上周滨:借周父之手"判死"商业对手]</ref>。

== 案件疑点 ==
檢方指控，袁寶璟涉嫌僱用「殺手」，將被害人汪興於2003年10月4日23時30分槍殺於遼陽市文聖區自家樓前。袁寶璟殺汪興的起因，是袁寶璟曾於1996年因生意損失而遷怒於四川商人[[刘汉_(四川商人)|劉漢]]，因此讓汪興去殺劉漢。汪興反以此要挾，袁寶璟才生殺人之意。 

袁寶璟的妻子卓瑪召開新聞發佈會，對當地警方的偵查程序提出質疑：如果認為袁寶璟讓自己的二哥袁寶琦找人殺汪興，袁寶琦又找袁寶森、袁寶福兩個堂兄去殺汪興，這是連環委託案。按常理，偵破程序是先抓直接行兇人袁寶森、袁寶福，審訊落實證據後，再抓袁寶琦，再到袁寶璟。這個過程需要時間。而警方抓人過程很奇怪：2003年11月23日下午6時，抓獲袁寶森、袁寶福，次日凌晨在廣西抓到袁寶琦，24日下午在北京抓到袁寶璟。沒有進行審訊，就抓捕連環案中所有嫌疑人，令人費解。不过也有人认为，先抓到行凶人，再抓到指使人和买凶者，从逻辑上完全可以说的过去<ref>{{cite news |url = http://news.xinhuanet.com/legal/2004-08/18/content_1811950.htm |title = 北京億萬富翁袁寶璟雇兇殺人案今日開庭審理 |author = 趙中鵬 |date = 2004年8月18日 |publisher = 新華網 |deadurl = yes |archiveurl = https://web.archive.org/web/20131002180946/http://news.xinhuanet.com/legal/2004-08/18/content_1811950.htm |archivedate = 2013年10月2日 |df =  }}</ref>。

袁寶璟、袁寶森、袁寶福、袁寶琦四兄弟因杀人犯刘汉的政法保护伞滥用职权，其中只有一人判[[死缓|死缓]]，其余仨人均被判死刑。

2005年10月14日，袁宝璟曾因揭发时任辽宁省常委、政法委书记[[李峰|李峰]]亲属在金龙股票上市隐性持股的经济犯罪事实被通知“暂缓死刑”。<ref>[http://news.sina.com.cn/o/2006-03-18/10208470368s.shtml 亿万富翁袁宝璟注射死刑 有重大检举情节被暂缓死刑]</ref><ref>[http://www.xinhuanet.com/chinanews/2006-03/18/content_6508049.htm 辽阳籍亿万富翁袁宝璟雇凶杀人被执行死刑]</ref>后来又捐一家开采石油价值495亿元人民币的印度尼西亚油田公司40%的股份，<ref>[http://news.sohu.com/s2006/06guxiongsharen/ 亿万富翁雇凶杀人案深层报告 袁宝璟被执行死刑]</ref>但袁寶璟仍然仓促地在2006年3月17日与其兄弟袁宝琦和袁宝森被执行死刑，袁寶璟全部财产均落入神秘人物手中。

2017年9月15日，据流亡海外的异议人士[[郭文贵|郭文贵]]爆料，袁氏家族的大部分财产为[[孟建柱|孟建柱]]及孙立军亲属所得。<ref>{{Cite web|url=https://hqsbonline.wordpress.com/2017/09/20/%E5%AE%B3%E6%AD%BB%E8%A2%81%E5%AE%9D%E7%92%9F%E7%9A%84%E6%98%AF%E5%AD%9F%E5%BB%BA%E6%9F%B1%E6%98%8E%E9%95%9C%E4%B8%93%E8%AE%BF%E9%83%AD%E6%96%87%E8%B4%B5%E7%AC%AC6%E6%9C%9F5-9%E2%80%A219%E6%96%87/|title=害死袁宝璟的是孟建柱明镜专访郭文贵第6期|accessdate=|author=|date=|publisher=}}</ref>
