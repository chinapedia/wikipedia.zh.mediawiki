'''袁纯清'''（{{bd|1952年|3月|catIdx=Y袁}}），[[湖南|湖南]][[汉寿|汉寿]]人；[[北京大学|北京大学]]法律系毕业，[[中国政法大学|中国政法大学]]法学硕士，[[湖南大学|湖南大学]]国际商学院[[管理学|管理学]][[博士|博士]]學位（[[在職研究生|在職研究生]][[學歷|學歷]]）。1971年10月加入[[中国共产党|中国共产党]]；是中共第十五届[[中央纪律检查委员会|中央纪律检查委员会]]委员、常委，第十六届中央候补委员，十七届、十八届中央委员。曾任[[中央农村工作领导小组|中央农村工作领导小组]]副组长、[[中国共产党山西省委员会|中共山西省委]]书记、省人大常委会主任，[[中国共产党陕西省委员会|中共陕西省委]]副书记、[[陕西行政长官列表|陕西省省长]]等职。

== 早期经历 ==
袁纯清出生在湖南山区；13岁时，进入[[汉寿县|汉寿县]]第一中学学习。“[[文化大革命|文革]]”开始后，学校停课，袁纯清被迫回乡务农。1971年10月至1977年2月间，他先后在家乡[[汉寿县|汉寿县]]和[[常德地区|常德地区]]的[[公安机关|公安局]]工作。1977年初，被推荐入[[北京大学|北京大学]]法律系成为最后一期[[工农兵学员|工农兵学员]]，是[[李克强|李克强]]的学长。在北大学习期间，袁纯清曾任[[中国共产主义青年团|共青团]][[北京大学|北京大学]]副书记，北大[[學生會|学生会]]主席，[[全国学联|全国学联]]副主席等职务。

== “[[團派|团派]]”经历 ==
袁纯清于1980年初自北大法律系毕业，并进入[[中国共产主义青年团|共青团]]中央工作。在团中央任职期间，袁历任学校部干事、副处长，学联办公室主任，学校部副部长、部长；团中央书记处书记，[[全国学联|全国学联]]秘书长、[[全国少工委|全国少工委]]主任，[[中华全国青年联合会|全国青联]]副主席，团中央[[机关党委|机关党委]]书记等职。共在共青团中央工作了十七年。

在此期间，袁纯清先后于1990年，获[[中国政法大学|中国政法大学]]的法学硕士学位；1997年又获[[湖南大学|湖南大学]]国际商学院的管理学博士学位（[[在職研究生|在職研究生]][[學歷|學歷]]）。1993年3月，他还当选为[[中国人民政治协商会议第八届全国委员会委员名单|第八届全国政协]]委员，兼任[[全国政协社会与法制委员会|全国政协社会与法制委员会]]常务副主任委员。<ref>[http://news.xinhuanet.com/ziliao/2007-02/05/content_5697974.htm 新华网资料：袁纯清简历] {{Wayback|url=http://news.xinhuanet.com/ziliao/2007-02/05/content_5697974.htm |date=20100731115315 }}</ref>

== 走出团中央 ==
从1997年10月开始，袁纯清正式离开[[中国共产主义青年团|共青团]]系统；在[[中国共产党第十五届中央委员会|中共十五届一中全会]]上，当选[[中国共产党中央纪律检查委员会|中纪委]]常委、兼秘书长。曾经轰动一时的[[厦门市|厦门]]“[[遠華案|远华走私案]]”就是由他作为新闻发言人对外公布的。1999年4月到2001年5月间，袁纯清又在[[北京大学|北京大学]]经济学院从事理论经济学博士后研究工作。<ref>[http://www.tycool.com/2007/11/11/00051.html 治陕西省长袁纯清，3,700万三秦儿女之大幸] {{Wayback|url=http://www.tycool.com/2007/11/11/00051.html |date=20080318015336 }} 海外中文网</ref>

2001年3月，袁纯清被外放[[陕西省|陕西]]，出任[[中国共产党陕西省委员会|中共陕西省委]]副书记；2004年1月起，又兼任[[中国共产党西安市委员会|中共西安市委]]书记的职务。在任期间，2005年2月，[[西安市|西安市]]推出了《西安国际化、市场化、人文化、生态化发展报告》，以白皮书的形式向社会报告[[西安市|西安]]的“发展理念”。而此前，袁纯清又被列为“全国受到关注的市委书记”之一。2006年6月，袁纯清接替已调中央工作的[[陈德铭|陈德铭]]，出任[[陕西省人民政府|陕西省人民政府]]代省长；次年2月3日，在第十届陕西省人民代表大会第五次会议上，正式当选为[[陕西行政长官列表|陕西省省长]]。 <ref>[http://www.gotopku.cn/data/detail.php?id=4087 学者型官员袁纯清接任陕西省代省长] {{Wayback|url=http://www.gotopku.cn/data/detail.php?id=4087 |date=20061208164943 }} [[北京大学|北京大学]]招生网</ref>

2010年5月31日，[[中国共产党中央委员会|中共中央]]宣布，袁纯清接任[[中国共产党山西省委员会|中共山西省委]]书记<ref>[http://news.ifeng.com/mainland/detail_2010_05/31/1568364_0.shtml 人民网：中共中央决定袁纯清出任山西省委书记]</ref>。7月起，兼任山西省人大常委會主任。因任职期间[[2014年山西官场地震|山西官场严重腐败]]，2014年9月1日，中共中央政治局常委[[刘云山|刘云山]]、中央组织部部长[[赵乐际|赵乐际]]亲自前往太原宣布山西省委领导调整<ref>{{cite web |title=袁纯清山西草草离场：谁来破解“晋官难当”魔咒 |url=http://www.chinanews.com/gn/2014/09-12/6586053.shtml |website=中国新闻网 |accessdate=2018-12-28 |date=2014-09-12}}</ref>。袁纯清被免去山西省委书记职务，转任[[中央农村工作领导小组|中央农村工作领导小组]]副组长。<ref>{{cite news |url = http://news.ifeng.com/a/20140901/41817246_0.shtml |title = 袁纯清任中央农村工作领导小组副组长 |date=2014-09-01 |publisher=凤凰网 |accessdate = 2014-09-01}}</ref>2017年10月，不再担任中央农村工作领导小组副组长职务<ref>{{cite news |url = http://www.thepaper.cn/newsDetail_forward_1891711|title = 袁纯清已不再担任中央农村工作领导小组副组长|date=2017-12-04 |publisher=澎湃新闻  |accessdate =2017-12-05}}</ref>。
