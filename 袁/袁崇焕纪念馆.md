'''袁崇焕纪念馆'''位于[[北京市|北京市]][[东城区_(北京市)|东城区]][[东花市东街|东花市斜街]]广东义园旧址，是在明末蓟辽督师[[袁崇焕|袁崇焕]]祠和墓的基础上建立的纪念馆。

==简介==
袁崇焕（1584年—1630年），字元素，号自如（或又字自如），原籍广东广州府东莞县，在广西梧州府藤县参加科举。明末蓟辽督师，因被疑通敌而被处以[[磔刑|磔刑]]。

清[[乾隆|乾隆]]年间，袁崇焕得到平反<ref>《清高宗实录》“谕军机大臣等：昨披阅明史，袁崇焕督师蓟辽，虽以我朝为难，但尚能忠于所事，彼时主暗政昏，不能罄其忱悃以致身罹重辟，深可悯测。现有无子孙，曾否出任，著传谕尚安，详细查明，遇便复奏。”</ref>，清政府在北京广东会馆所属的义园修建了袁崇焕的衣冠冢；道光年间，维修袁崇焕墓，南海[[吴荣光|吴荣光]]题写墓碑“有明袁大将军墓”。

[[File:Yuan_chonghuan-1.jpg|thumb]]

[[中华民国|民国]]初年，[[康有为|康有为]]出面，各界在墓旁建立袁崇焕祠<ref>“袁督师庙”不在此处，而在离此处不远的龙潭湖公园内，是1917年由康有为的学生张伯祯建立。</ref>。原墓堂廊柱曾悬有康有为手书“自坏长城慨今古，永留毅魄壮山河”对联。
1950年代，北京城区改造过程中，决定将城内墓葬迁出，袁崇焕墓也在其中。社会名士[[叶恭绰|叶恭绰]]、[[李济深|李济深]]、[[章士钊|章士钊]]、[[柳亚子|柳亚子]]等人联名呼吁保护。[[毛泽东|毛泽东]]于1952年5月25日批示予以保护。<ref>毛泽东复函叶氏：“接先生等四人来信，说明末爱国领袖人物袁崇焕先生祠庙事，已告彭真市长，如无大碍，应予保存。此时嗣后清于彭市长接洽为荷。”</ref>1954年，墓葬得到维修并保存下来。历史学家阎崇年曾指出：“文革时期袁崇焕在东花市斜街佘家馆的坟墓被挖开，里面未见其尸骨和遗物。<ref name="阎崇年">《明亡清兴六十年》第1讲《崇焕之死》。</ref>”他同时还说，“民国初年有人说袁崇焕的头颅被仆人窃走，私埋在自家的院子里。但是，在明清的官书、文集、笔记、方志、谱牒、档案里，见不到这方面的记载<ref name="阎崇年" />。”

2006年，袁崇焕墓和祠被[[中华人民共和国国务院|中华人民共和国国务院]]列入[[全国重点文物保护单位|全国重点文物保护单位]]名单。

馆内藏有传说是袁崇焕手迹的《听雨》以及[[康有为|康有为]]题写的《明袁督师庙记》手书等珍贵文物。

2017年5月23日，东城区官德教育基地在[[文天祥祠|文天祥祠]]、袁崇焕祠、[[于谦祠|于谦祠]]挂牌，东城区还下发通知要求全区各单位参观学习。其中，袁崇焕祠展出其家乡遗迹、任知县的[[福建省|福建省]][[邵武县|邵武县]]遗迹，守卫的[[辽宁省|辽宁省]][[兴城市|兴城]]遗迹<ref>{{Cite web |url = http://bj.people.com.cn/n2/2017/0615/c82840-30329538.html |title = 文保单位文天祥祠、袁崇焕祠和于谦祠变身官德教育基地|author =  |date = 2017-06-15 |publisher =人民网  |language =  |accessdate =  |quote =  }}</ref>。
