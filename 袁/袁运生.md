'''袁运生'''（{{bd|1937年|4月4日||}}）生于[[江苏|江苏]][[南通|南通]]，[[中华人民共和国|中华人民共和国]]画家、雕塑家、美术教育家<ref name=yszg>{{Cite web |url = http://art.china.cn/mjda/2009-06/30/content_2992197.htm |title = 袁运生 |author =  |date =2009-06-30  |publisher = 艺术中国 |language =  |accessdate =  |quote =  }}</ref><ref name=ifeng>{{Cite web |url = http://culture.ifeng.com/a/20170401/50876587_0.shtml |title =  袁运生：中国艺术对5000年文明的营养汲取还太浅|author =  |date = 2017-04-01 |publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref>。

==生平==
1955年，袁运生自[[江苏省南通中学|江苏省南通中学]]毕业，以油画系第一名的成绩考入[[中央美术学院|中央美术学院]]油画系。在校学习期间，[[墨西哥|墨西哥]]壁画家西盖洛斯来华讲学，给袁运生的启发和震动很大。1957年，袁运生因私下赞扬[[印象派|印象派]]绘画而在[[反右运动|反右运动]]中被打成“右派”。起初仍在校学习，后来被送到农场[[劳动改造|劳动改造]]。1961年底，被摘去“右派”帽子，从农场回到中央美术学院准备毕业创作<ref name=liuchun>刘淳，追索民族艺术精神——袁运生访谈录，民族艺术1998(4):8-13</ref>。

1962年，袁运生自[[中央美术学院|中央美术学院]]油画系[[董希文|董希文]]工作室毕业<ref name=yszg/>。毕业创作《水乡的记忆》在学校引发争论，留学[[苏联|苏联]]回来的一位老师只想给2分，理由是该作品未根据教学大纲规定的苏联（西方传统）雕刻的要求创作，而董希文想给5分，理由是突破了基础教学的制约而有新创造，最终该作品获4+<ref name=ifeng/>。同年，毕业后的袁运生被分配到[[吉林省|吉林省]][[长春|长春]]工人文化宫工作。1963年，[[毛泽东|毛泽东]]提出[[中国文联|中国文联]]各协会已到了[[修正主义|修正主义]]边缘，《美术》杂志便拿袁运生的《水乡的记忆》以及[[杨文秀|杨文秀]]的作品当作批判的典型，《水乡的记忆》被批判为资产阶级方向的代表、丑化劳动人民<ref name=liuchun/>。

1979年，袁运生被调到[[中央工艺美术学院|中央工艺美术学院]]任教<ref name=liuchun/>。1979年，[[首都国际机场|首都国际机场]]1号航站楼建成，需要艺术作品装饰，机场建设总指挥[[李瑞环|李瑞环]]找到[[中央工艺美术学院|中央工艺美术学院]]的[[张仃|张仃]]，他们决定为机场创作壁画。张仃随即组织了中国多位美术工作者成立了机场壁画小组。刚从[[云南|云南]]写生回到北京的袁运生成为小组成员。1979年10月，由7幅壁画组成的机场壁画群亮相，包括张仃的《哪吒闹海》、[[祝大年|祝大年]]的《森林之歌》、[[张国藩|张国藩]]的《民间舞蹈》、[[肖惠祥|肖惠祥]]的《科学的春天》、[[李化吉|李化吉]]和[[权正环|权正环]]的《白蛇传》，[[袁运甫|袁运甫]]的《巴山蜀水》，袁运生的《泼水节――生命的赞歌》。这是中国[[改革开放|改革开放]]后首批系列壁画作品，其中由袁运生创作的壁画《泼水节——生命的赞歌》里有几位正在沐浴的[[傣族|傣族]]裸体少女，该作品引起轩然大波，震动美术界，普通人也纷纷乘坐大巴来到机场餐厅看这幅壁画。海外媒体报道称：“中国在公共场所的墙壁上出现了女人裸体，预示了真正意义上的改革开放。”反对的意见则更为激烈，有人说这涉及民族问题，有人说绝对不可以有裸女出现，至少要加画个裤衩。[[中共中央宣传部|中共中央宣传部]]部长[[王任重|王任重]]找袁运生谈话，袁运生坚持不改画。最后[[邓小平|邓小平]]来机场视察时看到这幅壁画，非常高兴，看得很细，并表态支持保留该壁画。壁画风波这才告一段落。但1980年裸体少女还是被用布帘遮住，1982年又被用三合板封住钉死。1990年9月，袁运甫用从国外购买的壁画保护液，为机场壁画涂保护液以利长期保护，在涂装过程中拆开了三合板，从此该壁画中的裸体少女才重见天日，没有再被遮挡<ref name=ifeng/><ref name=yang>杨东恒，一幅壁画的三十年，新华月报2009:119-121</ref>。

1980年，袁运生被调回中央美术学院壁画系任教。除教学外，袁运生几乎无事可做，应邀设计的壁画几乎全被弃用。当时壁画系主任[[侯一民|侯一民]]说：“我们系里有一员大将，总是没事情可做，这怎么可以呢！”<ref name=yang/>1981年，袁运生用三个月在[[中国西北|中国西北]]考察中国传统文化和艺术。1982年，《美术》杂志第一期头条发表了袁运生的文章《魂兮归来》，文章认为在西方现代艺术冲击下，必须用民族艺术精神建起足够自信，找回失去的传统<ref name=liuchun/>。

1982年8月，袁运生应美国政府、[[哥伦比亚大学|哥伦比亚大学]]邀请访问[[美国|美国]]，在美国各地开展文化交流并讲学，随后定居美国并接受美国[[新墨西哥大学|新墨西哥大学]]、[[塔夫茨大学|塔夫茨大学]]、[[马萨诸塞州|马萨诸塞州]][[史密斯学院|史密斯学院]]、马萨诸塞州五所大学、[[哈佛大学|哈佛大学]]等多所大学的邀请做访问学者。1988年起在[[纽约|纽约]]当职业艺术家。其间，多次举办个人画展。1996年，为哈佛大学公共卫生学院创作大型丝织壁挂《人类的预言》。在美国的14年里，每次回中国，袁运生总要用一半时间考察[[河北|河北]]、[[山西|山西]]、[[陕西|陕西]]、[[河南|河南]]等地的石窟和寺庙<ref name=yszg/><ref name=yang/><ref name=liuchun/>。

1996年9月，袁运生回中国任中央美术学院油画系第四画室主任、教授<ref name=yang/>。2002年起，任中央美术学院学术委员会副主任、[[博士生导师|博士生导师]]。2002年，申报“中国传统雕塑的复制与当代中国美术教育体系的重建”课题，2003年取得[[中华人民共和国教育部|中华人民共和国教育部]]立项以及首次240万元人民币研究资金。该项目获[[中华人民共和国文化部|中华人民共和国文化部]]2005年首届创新奖。袁运生在中央美术学院研究生部创建以该课题为主题的研究中心，任研究中心主任。2003年起，研究中心队员先后对[[河南|河南]]、[[陕西|陕西]]、[[山东|山东]]、[[甘肃|甘肃]]、[[山西|山西]]、[[云南|云南]]等省开展系统性的石雕遗产考察。随后将研究成果编纂为美术学院教材<ref name=yszg/>。

==作品==
主要壁画作品有<ref name=yszg/>：
*首都机场壁画《泼水节，生命的赞歌》（1979年）
*美国塔夫茨大学图书馆壁画《红+蓝+黄=白？——关于中国两个神话故事》（1983年）
*哈佛大学公共卫生学院大型丝织壁挂《人类寓言》（1996年）
*[[香港城市大学|香港城市大学]]校门内壁画《夫子琴思》、《[[万户|万户]]飞天》（2005年）<ref name=yszg/>

主要绘画作品有<ref name=yszg/>：
油画《殉道者》（1981年）、《孤独》（1988年）、《涩笔》（1988年—1990年）、《搏》（1996年）《追——末班车》（1996年）、《乐》（1996年）等<ref name=yszg/>。
