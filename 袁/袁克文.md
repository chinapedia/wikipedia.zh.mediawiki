'''袁克文'''（{{bd|1889年||1931年|catIdx=Yuan,袁}}），字'''豹岑'''，别署'''寒云'''，[[中国|中国]][[河南|河南]][[項城|項城]]人，生於[[朝鮮國|朝鮮國]][[漢城|漢城]]，文學家，[[青幫|青幫]]大老、[[昆曲|昆曲]]名票，父為[[袁世凱|袁世凱]]。一生詩酒風流，故被稱為[[民國四公子|民國四公子]]之一。

== 简述 ==
[[中华民国|中华民国]][[大总统|大总统]][[袁世凯|袁世凯]]的次子，由其三姨太金氏（[[朝鮮王朝|朝鲜国]]人）生于朝鲜[[汉城|汉城]]（今[[首爾|首爾]]）。长兄[[袁克定|袁克定]]。袁克文师从[[方尔谦|方尔谦]]，熟读[[四书五经|四书五经]]、[[八比文|八比文]]，精通书法、绘画，頗能诗词歌赋，还极喜收藏书画、古玩等。后因妻妾成群，生活依然放浪不羁，又反对[[袁世凯称帝|袁世凯称帝]]，触怒其父，逃往上海，與袁世凱秘書步翔棻（字章五）一起加入[[青帮|青帮]]，靈前孝祖由師兄代收，與步翔棻一起成為興武六大字班人物，并在[[上海|上海]]、[[天津|天津]]等地开香堂广收门徒。1931年病逝于[[天津|天津]]。

== 妻妾子女 ==
除元配妻子[[刘梅真|刘梅真]]外，他还娶了五个姨太太，她们是：情韵楼、小桃红、唐志君、于佩文、亚仙。没有名分或“一度春风”的[[情妇|情妇]]那就更多了，据知情人讲有七八十个。 

袁克文有4子3女，袁家嘏（母劉梅真）、家彰（母劉梅真），家骝、家骥，女家华、家宜、家藏。家骝曾留学美国。家嘏1961年死去。三子[[袁家騮|袁家騮]]為有名[[高能物理|高能物理]]學家，家騮之夫人[[吳健雄|吳健雄]]也是[[核物理|核物理]]學家。

== 作品 ==
袁克文有詩文傳世，亦喜愛名人故事，曾作《[[紀曉嵐|紀曉嵐]]小傳》，另亦喜研究蒐集[[葉子戲|葉子戲]]資料，撰有《雀谱》一卷，由[[余大雄|余大雄]]作序，又编《叶子新书》，登载在《半月》杂志上，另還有《沿革表》、《角戏志》、《鴛鴦局圖經》。他說：“得明代叶子一局，从而略窥古法，复搜集天津、丹徒、临沂、歙县诸地之叶子，附以雀牌，作《沿革表》，纪其嬗变，作《角戏志》，疏其法例，合为一编，命曰《叶子新书》。戏虽无益，亦一代之文物也。”<ref name="袁克文传">{{cite book |author=王忠和 |title=《袁克文传》 |date=2006-06-01 |publisher=百花文艺出版社 |location=中國 |language=zh-hans |isbn=9787530643990}}</ref>
===《黃河篇》===
黃河遠上接星河，高掀銀浪下蒼波。天風吹塵寒四極，鬼母一笑驅千魔。壯士嗚咽拔劍起，戰場夜哭聞新鬼。風雨腥寒雪未昏，射狼噬逐狐狸死。君不見錦帳紅燭照美人，帝城歌舞春色新。列筵長侍君王笑，燕姬越女爭嬌顰。又不見野殍縱橫餘斷骨，將軍未老徵夫歿。瓊樓只乞夢溫柔，哪知蒼生涕淚竭。嗚呼！獨來徬徨叩九閽，四海八荒斷新魂。何年一瀉龍門水，滌蕩中原萬千里。
