'''袁紫僑'''（{{lang|en|'''CoCo Yuen Chi Kiu'''}}，{{bd|1991年|7月10日}}），原名'''袁聖殷'''，[[香港|香港]]女藝人。曾是前香港女子歌唱組合「LemonBeez」成員，現為另一女子歌唱組合「少女標本」成員之一。

== 簡歷 ==
袁紫僑畢業於[[悉尼科技大學|悉尼科技大學]]，2011年獨自從[[澳洲|澳洲]]返港任模特兒<ref>[http://the-sun.on.cc/cnt/entertainment/20130308/00470_055.html 袁紫僑青春大「模」畏]，[[太陽報|太陽報]]，2013-03-08</ref>，2012年與[[韋彤|韋彤]]組成女子歌唱組合「[[LemonBeez|LemonBeez]]」<ref>[http://hk.apple.nextmedia.com/entertainment/art/20111202/15854490 LemonBeez打孖出擊]，[[蘋果日報_(香港)|蘋果日報]] 娛樂，2011-12-02</ref>，發行過兩張迷你專輯，同時亦參演過多部影視作品。2015年分別與[[許雅婷|許雅婷]]（Kabby）、[[簡淑兒|簡淑兒]]（Jessica）和[[溫心|溫心]]（Swan）<ref>[http://orientaldaily.on.cc/cnt/entertainment/20150508/00282_076.html 少女標本出碟夠面子溫心等晒舞功]，on.cc，2015-05-08</ref> 等人，組成香港女子音樂組合「[[少女標本_(香港)|少女標本]]」<ref>[http://hk.apple.nextmedia.com/entertainment/art/20150507/19138216?top=12h 袁紫僑懶理周志文傳被炒]，蘋果日報，2015-05-07</ref><ref>[http://www.singpao.com/yl/mx/201505/t20150509_557682.html 少女標本分開錄單曲出道] {{webarchive|url=https://web.archive.org/web/20150518065702/http://www.singpao.com/yl/mx/201505/t20150509_557682.html |date=2015-05-18 }}，[[成報|成報]]，2015-05-09</ref>。

== 感情生活 ==
2016年10月30日，袁紫僑被記者揭發與[[何鴻燊|何鴻燊]]四房之子[[何猷君|何猷君]]舉止親暱，及後兩人承認拍拖<ref>[http://www.metrodaily.hk/metro_news/%E6%83%85%E4%BE%B6look%E6%B4%A9%E3%80%8C%E8%9C%9C%E3%80%8D-%E4%BD%95%E7%8C%B7%E5%90%9B%E8%A2%81%E7%B4%AB%E5%83%91%E8%AA%8D%E6%8B%8D%E6%8B%96-2/ 情侶Look洩「蜜」 何猷君袁紫僑認拍拖]，[[都市日報|都市日報]]，2016-11-01</ref>，兩人在2017年9月15日分手<ref>[https://news.mingpao.com/pns/dailynews/web_tc/article/20171015/s00016/1508003474526 何猷君認與袁紫僑分手 四太想兒子專心事業]，[[明報|明報]]，2017-10-15</ref>。

== 軼聞 ==
