'''袁姓'''是[[中文姓氏|中文姓氏]]，在《[[百家姓|百家姓]]》上排59名。袁姓人在[[四川|四川]]、[[华北|华北]]和[[江南|江南]]地区最具影响。总人口大约650万，占了当代[[中国|中国]]人口的0.54%。

== 来源 ==
[[File:Yuan_character_(1st_century).png|thumb]]
对于袁氏来源的记述，许多古籍内容差不多一致：袁氏是[[舜|舜]]的后裔，出自[[妫姓|妫姓]]。[[周武王|周武王]]灭[[商朝|商朝]]以后，追封前代圣王舜的后人[[胡公满|胡公满]]于[[陈国|陈]]。

胡公之十三世孙[[轅涛涂|轅涛涂]]（或作[[袁濤塗|袁濤塗]]）封在[[阳夏|阳夏]]（今河南太康），以其祖父庄爰的字为姓。袁氏自立姓之后，一直为陈国的大夫。[[春秋|春秋]]末年，袁侨、袁克都是轅涛涂的后裔。古时「袁」、「爰」、「辕」、「榬」、「溒」、「援」等字相通。正如《袁枢年谱》所云「一姓有六字五族之异」。据《[[新唐书|新唐书]]·宰相世系》，到秦末时，辕涛涂的裔孙[[辕告|辕告]]避难与[[黄河|黄河]]、[[洛河|洛河]]之间。[[辕告|辕告]]之少子政，以袁为氏，正式普遍地使用袁姓。

袁姓的第二个来源来自外姓和[[少数民族|少数民族]]的改姓。[[东汉|东汉]]末，分布在近[[四川|四川]]东部及[[重庆|重庆]]一代的[[板楯蛮|板楯蛮]]，有杜、朴、袁三大姓[[巴|巴]]夷王。袁姓同其他的姓氏一样，不断与其他民族的血缘交流。[[清朝|清朝]]初[[满洲八旗|满洲八旗]]中有袁姓，世居[[沈阳|沈阳]]。到清末[[民国|民]]初，旗人融化入汉族，袁姓旗人同样也回归了汉族，但其族人中已经混合了满族的血液。

== 历史分布和迁移 ==
自春秋末，袁姓一直活跃在河南一带。到[[汉朝|汉朝]]，形成了以[[汝南郡|汝南郡]][[汝阳县|汝阳县]]（今[[河南省|河南省]][[汝南县|汝南县]]）为中心的大姓。东汉初年，[[袁良_(汝南)|袁良]]仅为[[成武县|成武]][[县令|令]]，但从他的孙子[[袁安|袁安]]以下的四代人都做到[[太尉|太尉]]、[[司徒|司徒]]、[[司空|司空]]的高位。由此开创了[[汝南袁氏|汝南袁氏]]。与袁良同名的[[袁良_(陈郡)|袁良]]则成为[[陈郡袁氏|陈郡袁氏]]的先祖，孙[[袁滂|袁滂]]官至司徒。汝南袁氏和陈郡袁氏自[[东汉|东汉]]，[[曹魏|曹魏]]，[[晋朝|晋朝]]至[[南朝梁|南朝梁]]、[[南朝陈|南朝陈]]，人才辈出，权势显赫。至唐朝，与其它姓氏类似，袁姓者多以陈郡袁氏标榜<ref>《[[史通|史通]]·内篇·邑里第十九》爰及近古，其言多伪。至于碑颂所勒，茅土定名，虚引他邦，冒为己邑。若乃'''[[陈郡袁氏|称袁则饰之陈郡]]'''，[[京兆杜氏|言杜则系之京邑]]，[[彭城刘氏|姓卯金者咸曰彭城]]，[[钜鹿魏氏|氏禾女者皆云钜鹿]]。在诸史传，多与同风。此乃寻流俗之常谈，忘著书之旧体矣。</ref>。[[北宋|北宋]]时编撰的《[[新唐书|新唐书]]》<ref>《新唐书·卷七十四下·表第十四下》袁氏出自媯姓。陳胡公滿生申公犀侯......九世孫袁生生玄。孫幹，封貴鄉侯，復居'''陳郡陽夏'''。八世孫良，二子：昌、璋。昌，成武令，生'''漢司徒安'''，字邵公。三子：賞、京、敞。京，蜀郡太守......璋生'''司徒滂'''，字公熙。滂生渙，字曜卿，魏御史大夫。</ref>则将汝南袁氏和陈郡袁氏的同名先祖混为一人，至今仍有混淆<ref>{{Cite web |url=http://www.wccdaily.com.cn/shtml/hxdsb/20161022/14962.shtml |title=汉朝袁姓：陈郡和汝南家族最显赫 |accessdate=2017-10-14 |author= |date=2016-10-22 |work= |publisher=华西都市报网站|language=Zh-hans }}</ref>。

[[宋朝|宋朝]]时期，袁姓大约有26万人，以人数算宋朝第六十四位大姓。在全国的分布主要集中在于现在的[[四川|四川]]、[[浙江|浙江]]、[[山西|山西]]、[[江西|江西]]四省。其次分布于[[江苏|江苏]]、河南、[[安徽|安徽]]和[[湖南|湖南]]。

[[明朝|明朝]]，袁姓大约53万，为明朝第三十六大姓。袁姓人口增长高于全国人口的增长，宋、[[元朝|元]]、明600年袁姓人口主要向东南和南部地区迁移，而四川地区的袁姓人口萎缩严重。明末，袁姓也进入了[[台湾|台湾]]。最近600年期间，袁姓人口流动的程度和方向与宋，元，明间有了很大的区别，由东南部向西部、华中、华北回迁。

== 当代袁氏分布 ==
在全国的分布目前主要集中于四川，[[河北|河北]]两省，大约占袁姓总人口的23%。江西、江苏、河南、浙江、[[湖北|湖北]]，这五省又集中了38%。袁姓分布很广，但不均衡。

===歷史人物===
*[[袁盎|袁盎]]：西汉大臣
*[[袁安|袁安]]：汝南郡汝阳县（今河南汝南）人。中国东汉初期的政治家。官拜司徒
*[[袁賞|袁賞]]，汝南郡汝阳县（今河南汝南 ）人，[[袁安|袁安]]之子。漢和帝從竇氏手中奪回權力後追思袁安守正，任命他為郎。
*[[袁京|袁京]]，汝南郡汝阳县（今河南汝南）人，[[袁安|袁安]]之子。官至[[侍中|侍中]]、[[蜀郡|蜀郡]]太守。
*[[袁敞|袁敞]]，汝南郡汝阳县（今河南汝南）人，[[袁安|袁安]]之子。官至司空。
*[[袁彭|袁彭]]，汝南郡汝阳县（今河南汝南）人，[[袁京|袁京]]之子，官至[[光祿勳|光祿勳]]。
*[[袁湯|袁湯]]，汝南郡汝阳县（今河南汝南）人，[[袁京|袁京]]之子，桓帝時先後擔任司空、司徒和太尉。
*[[袁盱|袁盱]]，汝南郡汝阳县（今河南汝南）人，[[袁敞|袁敞]]之子，官至光祿勳。
*[[袁滂|袁滂]]，陈郡扶乐人（今河南太康），袁璋之子，[[袁涣|袁涣]]之父，官至東漢[[司徒|司徒]]。
*[[袁绍|袁绍]]：東汉末年军阀
*[[袁术|袁术]]：東汉末年军阀
*[[袁渙|袁渙]]，陳郡阳夏人。名門之後，東漢末人物，曾效力於袁術、呂布及曹操勢力之下。
*[[袁宏|袁宏]]：（328年－376年）[[东晋|东晋]]文学家、史学家。
*[[袁崧|袁崧]]：是袁涣的六世孙，袁宏的堂侄。[[东晋|东晋]]史学家。
*[[釋辯才|釋辯才]]：唐朝高僧
*[[袁崇焕|袁崇焕]]：明末抗清名將，军事家
*[[袁江|袁江]]：江苏扬州人，清朝画家
*[[袁耀|袁耀]]，江苏扬州人，清朝畫家。袁江之子。
*[[袁枚|袁枚]]：（1716年－1797年），清代诗人，散文家,文筆與當時的大臣尚书[[紀曉嵐|紀曉嵐]]齊名，時稱「南袁北紀」。
*[[袁世凯|袁世凯]]：（1859年9月16日－1916年6月6日），[[清末|清末]][[清朝內閣總理大臣|內閣總理大臣]],[[軍機大臣|軍機大臣]],[[中華民國大總統|中華民國第一任大總統]]。
*[[袁良|袁良]]：（1882年－1953年），中華民国政治家，中國政界的日本通。
*[[袁文才|袁文才]]（1898年－1930年），[[中国工农红军|中国工农红军]]早期领导人之一。
