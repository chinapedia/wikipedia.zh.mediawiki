'''彰化縣芬園鄉同安國民小學'''簡稱為'''同安國小'''，位於[[臺灣|臺灣]][[彰化縣|彰化縣]][[芬園鄉|芬園鄉]]，乃該鄉之縣立[[小学|國民小學]]，另設有安山分校。位處八卦山脈東側，海拔約二百公尺，屬丘陵地。行政隸屬彰化縣芬園鄉。<ref>參看{{cite web |url=http://163.23.94.129/new_page_25.htm |title=存档副本 |accessdate=2017年3月28日 |deadurl=yes |archiveurl=https://web.archive.org/web/20050213044211/http://163.23.94.129/new_page_25.htm |archivedate=2005年2月13日 |df= }}</ref>。建校於民國十一年四月一日（日據時大正二年），校名為芬園公學校同安寮分教場。校長為日籍內村英藏氏。同安村和中崙村居民為了讓學子方便就學，蓽路藍縷共創同安，成為芬園鄉第二所歷史悠久的小學，自日本時代起至今校友繁多。光復前(日據時代)有17屆，光復後有71屆，至今邁入95年歷史<ref>參看{{cite web |url=http://163.23.94.129/new_page_25.htm |title=存档副本 |accessdate=2017年3月28日 |deadurl=yes |archiveurl=https://web.archive.org/web/20050213044211/http://163.23.94.129/new_page_25.htm |archivedate=2005年2月13日 |df= }}。</ref>。近年因[[少子化|少子化]]問題，加上分校學區內的新社區增加，2012年曾發生本校僅30名學生、分校卻有51名學生的狀況<ref>參看{{cite web |url=http://mag.udn.com/mag/campus/storypage.jsp?f_ART_ID=417052 |title=存档副本 |accessdate=2012-11-15 |deadurl=yes |archiveurl=https://web.archive.org/web/20121015130359/http://mag.udn.com/mag/campus/storypage.jsp?f_ART_ID=417052 |archivedate=2012-10-15 }}。</ref>。

2016年雲美雪校長秉持積極辦學，力求革新的精神，帶領教職員團隊，努力整合校內外社區資源，積極爭取辦理多元教學計畫<ref>參看{{cite web |url=http://163.23.94.129/new_page_22.htm |title=存档副本 |accessdate=2017年3月28日 |deadurl=yes |archiveurl=https://web.archive.org/web/20170328195543/http://163.23.94.129/new_page_22.htm |archivedate=2017年3月28日 |df= }}。</ref>，一學年內辦理各種多元教學活動，教學次數及種類達到以往每年平均次數的數倍之多，為同安國小帶來大量的教學能量，期望重拾社區民眾對學校的信心與肯定。<ref>參看[https://www.facebook.com/taes502/]。</ref>。各界善心人士提供資源予同安國小學生<ref>參看[https://udn.com/news/story/3/2240335]。</ref>，期望學生能在資源平等的條件下，成長茁壯。
106年縣府評估現況，提出同安國小轉型建議，社區鄉親、家長及校友們也都凝聚回同安國小，向縣府提出對學校的看法，期望縣府能夠讓同安國小永續為社區的教育發揮功能，讓偏鄉的孩子們有一個安全的學習環境，並支持同安國小轉型，永續發展，成為社區的教育、情感以及歷史的重要中心。<ref>參看[http://news.ltn.com.tw/news/local/paper/1079985]。 </ref>。
教育處表示，同安國小新學年度（一○六）照常維持現行校務運作，學校要轉型，可朝發展學校特色或實驗性方案，例如蝴蝶園區或雙語實驗小學，並在新學年度課程、教學及教材，規劃混齡教學，教育處樂於提供學校諮詢輔導等相關資源。小校面對少子化，要有轉型才能生存的打算。同安國小未來轉型方向，將依照縣府輔導、建議以及家長溝通，多方商討，積極進行轉型。<ref>參看[http://news.ltn.com.tw/news/local/paper/1079985]。 </ref>。

== 校史沿革 ==
* 1922年：4月1日自[[彰化縣芬園鄉芬園國民小學|芬園公學校]]增設'''同安寮分教場'''，首任校長為日籍內村英藏氏。
* 1941年：4月1日獨立為'''同安寮國民學校'''。
* 1950年：改稱'''彰化縣同安國民學校'''。
* 1953年：增設安山分校。
* 1961年：增設大竹分校，1964年該分校更名成文德分校（於1969年獨立成為今[[彰化縣芬園鄉文德國民小學|彰化縣芬園鄉文德國民小學]]）。
* 1968年：6月1日因實施[[台灣九年國民義務教育|九年國民義務教育]]，校名改成'''彰化縣芬園鄉同安國民小學'''。
* 2012年：舉辦建校90周年校慶活動。
* 2015年：芬園鄉同安國民小學安山分校自104學年度起整併至同安國民小學。
* 2016年：安山分校校地移撥給縣警察局管理作為安山派出所使用。
* 2016年：學校轉型評估朝"六藝同安生態小學"特色學校經營發展。
