'''卢恒宇和李姝洁工作室'''是一家[[中国|中国]][[动画|动画]]工作室，工作室成立于2013年2月28日，最初只有五名成员<ref name=":0">{{Cite web|url = http://epaper.xkb.com.cn/view/895546|title = 一不小心这些漫画神剧火了，少儿请自动绕行2|accessdate = |author = |date = 2013-11-12|publisher = 新快报|deadurl = yes|archiveurl = https://web.archive.org/web/20160304122613/http://epaper.xkb.com.cn/view/895546|archivedate = 2016-03-04}}</ref>，创始人卢恒宇的目标是获得[[奥斯卡奖|奥斯卡奖]]<ref>{{Cite web|url = http://ent.sina.com.cn/m/c/2014-12-18/17394259612.shtml|title = 《十冷》导演：终极目标是拿到奥斯卡！|accessdate = |author = |date = 2014年12月18日|publisher = 新浪娱乐}}</ref>。
