'''盧承慶'''（{{bd|595年||670年||catIdx=L卢}}），字'''子餘'''，[[幽州|幽州]][[范阳郡|范阳郡]][[涿县|涿县]]（今[[河北省|河北省]][[涿州市|涿州市]]）人，出自[[范阳卢氏|范阳卢氏]]北祖大房，[[唐朝|唐朝]]官员，[[唐高宗|唐高宗]]年间任[[宰相|宰相]]。

== 背景 ==
卢承庆的祖父[[盧思道|盧思道]]是[[隋朝|隋朝]]武阳太守、散骑侍郎。[[隋炀帝|隋炀帝]]末年大将[[李渊|李渊]]在[[太原|太原]]起兵反隋时，卢承庆之父[[盧赤松|盧赤松]]任[[河東|河東]]令，先前就认识李渊，李渊兵到后立刻投降，成为李渊的僚属。李渊建立唐朝后，封卢赤松为范阳郡公。卢承庆相貌英俊，仪态大方，博学有才。卢承庆年少时，父亲去世，他继承了范阳郡公的爵位。<ref name=BT81>[[:s:旧唐书/卷81|《旧唐书》卷八十一]]</ref><ref name=NBT106>[[:s:新唐书/卷106|《新唐书》卷一百零六]]</ref>

== 唐太宗年间 ==
[[唐太宗|唐太宗]]继位初期，卢承庆任[[秦州|秦州]][[参军|参军]]，一次入朝奏报河西军情，说得很清楚，太宗称奇，升他为[[考功员外郎|考功员外郎]]，后又数次升迁至[[民部侍郎|民部侍郎]]。一次，太宗问及历代户口之事，卢承庆对以[[夏朝|夏朝]]、[[商朝|商朝]]乃至北周、隋朝的人口增减，太宗再次表示赞赏，不久让他兼任[[检校|检校]][[兵部侍郎|兵部侍郎]]，知五品选。卢承庆不接受，认为选官是尚书的职责，自己不能越权，但太宗没有答应，说：“朕相信你，你为什么不相信你自己呢？”后来卢承庆又历任[[雍州|雍州]][[别驾|别驾]]、[[尚书左丞|尚书左丞]]。<ref name=BT81/><ref name=NBT106/>

二十二年（648年），太宗封禅[[泰山|泰山]]，以时任户部侍郎卢承庆为检校封禅副大使。<ref name=CFYG>《[[册府元龟|册府元龟]]》</ref>

== 唐高宗年间 ==
[[贞观|贞观]]二十三年（649年）唐太宗驾崩，[[唐高宗|唐高宗]]继位。由于权相[[褚遂良|褚遂良]]诬告，卢承庆被贬为[[益州|益州]]大都督府[[长史|长史]]；褚遂良又指控卢承庆在雍州时失职，卢承庆又被贬为[[简州|简州]][[司马|司马]]。在这段时间，卢承庆结识了道因法师。<ref>[[:s:益州多宝寺道因法师碑文|李俨《益州多宝寺道因法师碑文》]]</ref>一年后，卢承庆转任[[洪州|洪州]]长史。高宗将要幸[[汝州|汝州]]温泉，擢升卢承庆为汝州刺史，<ref name=NBT106/>不久又召他回京任[[光禄卿|光禄卿]]。<ref name=BT81/>[[显庆|显庆]]二年（657年），大将[[苏定方|苏定方]]击破[[西突厥|西突厥]]，俘获其可汗[[阿史那賀魯|阿史那賀魯]]，十二月，高宗派卢承庆持册命去西突厥把西突厥的土地分给两位忠于唐朝的西突厥王子兴昔亡可汗[[阿史那彌射|阿史那彌射]]和继往绝可汗[[阿史那步真|阿史那步真]]，又命卢承庆和两位王子给各部首领封刺史以下官职。<ref>[[:s:旧唐书/卷194下|《旧唐书》卷一百九十四下]]</ref><ref>[[:s:新唐书/卷215下|《新唐书》卷二百一十五下]]</ref><ref name=ZZTJ200>[[:s:资治通鉴/卷200|《资治通鉴》卷二百]]</ref><ref>[[:s:分立弥射为兴昔亡可汗步真为继往绝可汗诏|:s:分立弥射为兴昔亡可汗步真为继往绝可汗诏]]</ref>

四年（659年），代[[杜正伦|杜正伦]]为[[度支尚书|度支尚书]]。五月，授[[參知政事|參知政事]]衔，为[[实质宰相|实质宰相]]。七月，因涉嫌谋反被流放的前宰相国舅[[长孙无忌|长孙无忌]]进一步遭到指控，卢承庆和其他宰相[[李勣|李勣]]、[[许敬宗|许敬宗]]、[[辛茂将|辛茂将]]、[[任雅相|任雅相]]奉命调查。其中许敬宗是[[武则天|武皇后]]的党羽，也是调查的发起者。长孙无忌最终被迫自杀。十一月，卢承庆被授予高一级的宰相衔[[同中書門下三品|同中書門下三品]]。但次年（660年）七月，他就因为对度支事务处理不当被免官，<ref name=ZZTJ200/><ref>[[:s:旧唐书/卷4|《旧唐书》卷四]]</ref><ref>[[:s:新唐书/卷3|《新唐书》卷三]]</ref>出为[[润州|润州]][[刺史|刺史]]。曾任右肃机（尚书右丞），[[麟德|麟德]]二年（665年）三月奉命为使去东都讯察记录囚犯的罪状。<ref name=CFYG/><ref>[[:s:遣使虑囚诏|:s:遣使虑囚诏]]</ref>约同年又迁雍州长史，<ref>[[郁贤皓|郁贤皓]]《[[唐刺史考全编|唐刺史考全编]]》</ref>授[[銀青光祿大夫|銀青光祿大夫]]。[[总章|总章]]二年（669年）二月，他又被任为[[司刑太常伯|司刑太常伯]]，<ref name=ZZTJ201>[[:s:资治通鉴/卷201|《资治通鉴》卷二百零一]]</ref>以代[[李乾祐|李乾祐]]。[[咸亨|咸亨]]元年（670年）闰九月，诏赠新亡的[[武则天|武皇后]]之母为[[孝明高皇后|鲁国太夫人]]，卢承庆摄同文正卿充监护大使，赐东园秘器，厚给官供。<ref>[[:s:大周无上孝明高皇后碑铭（并序）|武三思《大周无上孝明高皇后碑铭（并序）》]]</ref>不久即告老请求[[致仕|致仕]]。高宗批准了，加[[金紫光祿大夫|金紫光祿大夫]]衔。<ref name=BT81/><ref name=NBT106/>

他的弟弟[[卢承业|卢承业]]后来也担任雍州长史、检校尚书左丞，兄弟相继担任这些职位，时人以为荣。<ref name=BT81/>

当年，卢承庆卒，赠幽州都督，谥号'''定'''。临终时，他对儿子们说：“死生至理，亦犹朝之有暮。吾终，敛以常服；晦朔常馔，不用牲牢；坟高可认，不须广大；事办即葬，不须卜择；墓中器物，瓷漆而已；有棺无椁，务在简要；碑志但记官号、年代，不须广事文饰。”<ref name=BT81/><ref name=NBT106/>

[[唐德宗|唐德宗]]年间，卢承庆得以续图[[凌烟阁|凌烟阁]]。<ref>[[:s:新唐书/卷191|《新唐书》卷一百九十一]]</ref>

== 典故 ==
卢承庆任考功员外郎时，有一个监督[[漕运|漕运]]的官员因遭遇大风丢了粮食，卢承庆给他评为「中下」。这名官员神色自若，一言不发退下了。卢承庆看重他的气量，说“遭遇大风不是你所能阻止的”，改评为「中中」。这名官员既没有表现出高兴的样子，也没有表达惭愧。卢承庆对他的反应表示赞赏，又说他“宠辱不惊”，改评为「中上」。这是卢承庆能表彰别人的优点。<ref name=NBT106/><ref name=ZZTJ201/>
