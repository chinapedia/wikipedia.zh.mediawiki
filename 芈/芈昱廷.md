'''芈昱廷'''（{{bd|1996年|1月8日||}}），[[江苏|江苏]][[徐州|徐州]]人，[[围棋|围棋]]棋手。他11岁成为国内最年轻的职业棋手，12岁即获世少赛青年组亚军和首届全国智运会围棋青少年组冠军。曾豪取2011年全国围棋甲级联赛九连胜。2011年第二届全国智运会卫冕围棋少年赛。2013年夺得第一届[[梦百合杯世界围棋公开赛|梦百合杯世界围棋公开赛]]冠军。父亲芈智强。

== 职业生涯 ==
* 2007年7月定段。
* 2013年夺得第一届梦百合杯世界围棋公开赛冠军，直升九段。

== 国內比賽成绩 ==
* 2009年[[理光杯|理光杯]]新秀赛冠军。
* 2012年第12届[[理光杯|理光杯]]四强。
* 2012年[[全国围棋个人赛|全国围棋个人赛]]冠军。
* 2014年第10届[[威孚房开杯棋王争霸赛|威孚房开杯棋王争霸赛]]四强。
* 2015年第7届钻石杯[[中国龙星战|中国龙星战]]冠军。第29届[[中国围棋天元战|中国围棋天元战]]挑战者，决赛负于上届天元[[陈耀烨|陈耀烨]]。
* 2016年第6届“[[衢州烂柯杯|衢州烂柯杯]]”中国围棋冠军赛冠军<ref>[http://sports.sina.com.cn/go/2016-10-15/doc-ifxwvpaq1380772.shtml 衢州烂柯杯决赛芈昱廷胜柯洁夺冠 获50万奖金]</ref>。
* 2016年第12届[[威孚房开杯棋王争霸赛|威孚房开杯]]亚军。
* 2016年象屿杯[[中国电视围棋快棋赛|中国电视围棋快棋赛]]冠军<ref>[http://sports.sina.com.cn/go/2016-11-28/doc-ifxyawmp0421740.shtml 象屿杯决赛芈昱廷胜李钦诚 夺年度个人第三冠]</ref>。
* 2016年于[[中国围棋甲级联赛|中国围棋甲级联赛]]代表华泰证券江苏队，截至第20轮获12胜8负。

== 芈氏飛刀 ==
在[[2018年|2018年]][[8月1日|8月1日]]日[[中国围棋名人战|中國名人戰]]半決賽，芈昱廷執白對[[童夢成|童夢成]]的比賽中，星位點三三的定石下出了扳的新手(下圖第20手)，其後整個黑角被吃，甚至造成當時在野狐圍棋分析情勢的[[絕藝|絕藝]] AI 誤判角落的對殺。之後這一手及其後的變化造成流行，被媒體命名為「芈氏飛刀」<ref>{{Cite web|url=http://weiqi.qq.com/news/9558.html|title=双月排位赛潜伏成功复仇 顺利过关再进决赛|accessdate=2018-09-28|last=腾讯野狐围棋|work=weiqi.qq.com}}</ref><ref>{{Cite web|url=http://m.sohu.com/n/545376068/|title=名人战挑决芈昱廷完胜杨鼎新  连续两年挑战连笑-体育频道-手机搜狐|accessdate=2018-09-28|work=m.sohu.com}}</ref>。
