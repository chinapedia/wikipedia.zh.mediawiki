'''攜手扶弱基金'''（{{lang-en|'''Partnership Fund for the Disadvantaged'''}}）是[[香港特別行政區行政長官|香港特別行政區行政長官]]於2005年[[香港施政報告|香港施政報告]]中宣布設立二億港元的[[基金|基金]]。目的為推動[[社會福利|社會福利]]界、商界和[[政府|政府]]三方合作，建立伙伴關係，共同扶助弱勢社群<ref>{{cite web |language=zh-hk |url=http://www.singtao.com/archive/fullstory.asp?andor=or&year1=2004&month1=12&day1=11&year2=2004&month2=12&day2=11&category=all&id=20041211a10&keyword1=&keyword2= | title=二億元扶弱基金明春推出  |publisher=[[星島日報|星島日報]] |date=2004-12-11 |accessdate=2012-03-04}}</ref>。

[[香港特別行政區行政長官|香港特別行政區行政長官]]於2017年[[香港施政報告|香港施政報告]]中宣布在2018-19年度向攜手扶弱基金注資4億元，其中2億元主要用於推行課餘學習及支援項目，預計約13萬名基層兒童受惠。

==目的==
設立基金的目的︰
* 鼓勵社會福利界擴展網絡，以爭取商業機構參與扶助弱勢社群
* 鼓勵商界承擔更大的[[企業社會責任|企業社會責任]]，合力建立一個團結和諧、充滿愛心的社會

==諮詢委員會==
諮詢委員會由[[勞工及福利局|勞工及福利局]]局長委任，作為審核並向[[社會福利署|社會福利署]]署長建議應予批准的申請，以及就基金的各項運作事宜向社會福利署署長提供意見。

==申請資格==
* 根據香港《[[稅務條例|稅務條例]]》第112章第88條獲豁免繳稅的[[慈善|慈善]]非政府福利機構，均可申請此基金的等額資助
* 以下會被此基金的委員會優先考慮的項目<ref>{{cite web |language=zh-hk |url=http://news.hkheadline.com/instantnews/news_content/201005/31/20100531a182302.html?cat=a | title=攜手扶弱基金接受申請 |publisher=[[頭條日報|頭條日報]] |date=2010-05-31 |accessdate=2012-03-04}}</ref>︰
** 旨在提高就業技能/能力、提升生活質素、提升能力及倡導共融的計劃
** 可持續和商界維繫社會夥伴合作關係的計劃
** 能針對社會問題作出及早介入和提供預防措施的計劃及
** 如建議項目能夠顯示非政府福利機構，為配合受惠人士不斷轉變的服務需要，而提供適切的服務及擴展與商界的夥伴合作網絡

==資助額==
慈善非政府福利機構申請的資助，每宗上限為200萬港元。

==現時資助情況==
香港政府自2005年成立此基金後，至今已撥款超過2,940萬港元資助共97個項目，商界亦作出高達3,300萬元的現金及實物資助<ref>{{cite web |language=zh-hk |url=http://hk.apple.nextmedia.com/realtime/art_main.php?iss_id=20080223&sec_id=6996647&art_id=10785751 | title=攜手扶弱基金 撥3千萬資助近百項目 |publisher=[[蘋果日報_(香港)|蘋果日報 (香港)]] |date=2008-02-23 |accessdate=2012-03-04}}</ref>。

在[[2010年10月|2010年10月]]，[[電影演員|影視紅星]][[曾志偉|曾志偉]]和[[戚美珍|戚美珍]]為了協助聾人就業<ref>{{cite web |language=zh-hk |url=http://www.hkcd.com.hk/content/2010-12/06/content_2647626.htm |title=社企齊鋪樂農傷健共融 |publisher=[[香港商報|香港商報]] |date=2010-12-06 |accessdate=2012-03-04 |deadurl=yes |archiveurl=https://web.archive.org/web/20141108123046/http://www.hkcd.com.hk/content/2010-12/06/content_2647626.htm |archivedate=2014-11-08 }}</ref><ref>{{cite web |language=zh-hk |url=http://paper.wenweipo.com/2010/10/05/EN1010050038.htm |title=戚美珍助失聰人士就業 |publisher=[[文匯報_(香港)|文匯報 (香港)]] |date=2010-10-05 |accessdate=2012-03-04 |archive-url=https://web.archive.org/web/20111228092734/http://paper.wenweipo.com/2010/10/05/EN1010050038.htm |archive-date=2011-12-28 |dead-url=yes }}</ref>，他們得到該中心的協助<ref>{{cite web |language=zh-hk |url=http://hk.news.yahoo.com/%E6%BB%99%E8%B1%90%E5%B9%AB%E7%A4%BE%E4%BC%81%E5%81%9A%E7%94%9F%E6%84%8F-223000719.html | title=滙豐幫社企做生意 |publisher=[[星島日報|星島日報]] |date= 2011-09-08 | accessdate=2012-03-04}}</ref>，便成功開設社會企業素食餐廳《[[樂農|樂農]]》。該餐廳並透過慈善機構「[[龍耳|龍耳]]」聘請聾人<ref>{{cite web |language=zh-hk |url=http://www1.hk.apple.nextmedia.com/template/apple/art_main.php?iss_id=20101219&sec_id=4104&art_id=14780972 | title=生活存關愛：首間素食餐廳聘用聽障人士 |publisher=[[蘋果日報_(香港)|蘋果日報 (香港)]] |date=2010-12-19 |accessdate=2012-03-04}}</ref>，希望此舉能夠扶助弱勢社群。
