{{作品名稱|name=''Saving Private Ryan''|hk=雷霆救兵|tw=搶救雷恩大兵|cn=拯救大兵瑞恩}}是一部於1998年上映的[[美國|美國]][[戰爭片|戰爭片]]，由[[史蒂芬·史匹柏|史蒂芬·史匹柏]]執導，{{Link-en|羅勃·羅達特|Robert Rodat}}編劇。主演包括[[汤姆·汉克斯|汤姆·汉克斯]]、{{Link-en|湯姆·賽斯摩|Tom Sizemore}}、[[愛德華·賓斯|愛德華·賓斯]]及[[巴里·佩珀|巴里·佩珀]]，劇情描述[[諾曼第登陸|諾曼第登陸]]後，搶救二等兵雷恩的故事。

該片最為人津津樂道的是開頭的[[奧馬哈海灘|奧馬哈海灘]][[搶灘|搶灘]]場面，相當程度重現了當年的殘酷與慘烈。該片榮獲1998年[[奧斯卡最佳導演獎|奧斯卡最佳導演獎]]在內的5項大獎，票房收入也稱冠一時。

在获奖名单揭晓前，《拯救大兵瑞恩》被普遍公认最有希望夺得奧斯卡最佳影片奖，但最终败给了《[[莎翁情史|莎翁情史]]》。

== 劇情概述 ==
描述[[諾曼第登陸|諾曼第登陸]]後，-{zh-cn:瑞恩; zh-tw:雷恩; zh-hk:賴恩;}-家4名於前線參戰的兒子中，3個兒子皆已於兩週內於海外各戰區陸續[[陣亡|陣亡]]，其母親將在一天內接到三個兒子的死訊。而隸屬[[101空降師|101空降師]][[二等兵|二等兵]]的小兒子-{zh-cn:詹姆斯·法兰西斯·瑞恩; zh-tw:詹姆斯·法蘭西斯·雷恩; zh-hk:賴詹士;}-則在參與諾曼地的空降行動後下落不明，[[美國陸軍|美國陸軍]]參謀長[[乔治·卡特里克·马歇尔|马歇尔]][[上将|上将]]得知此事後出於人道考量，特令在法國前線作戰的[[美國陸軍遊騎兵|美國陸軍遊騎兵]]組織一支8人小隊深入敵後，只為在人海茫茫、槍林彈雨中找出生死未卜的二等兵雷恩，並將其平安送回後方。

== 製作 ==

=== 發展 ===
1994年，{{Link-en|羅勃·羅達特|Robert Rodat}}為電影撰寫了劇本，且將劇本交給了製片人{{Link-en|馬克·高登|Mark Gordon}}，受到高登的認可，接著，高登將劇本交給史匹柏執導。劇情根據{{Link-en|尼藍兄弟|Niland brothers}}的真實故事而改編，拍攝於1997年6月27日開始<ref name="ew">{{cite news|url=http://www.ew.com/ew/article/0,,284082,00.html|title=Message in a Battle|work=[[Entertainment_Weekly|Entertainment Weekly]]|date=July 24, 1998|accessdate=September 5, 2008|first=Jeff|last=Gordinier}}</ref>。

== 發行 ==

=== 專業評價 ===
[[爛番茄|爛番茄]]新鮮度高達92%，基於130條評論，平均分為8.6/10<ref>{{cite web |url=http://www.rottentomatoes.com/m/saving_private_ryan/ |title=Saving Private Ryan |date=July 24, 1998 |work=Rotten Tomatoes |accessdate=December 15, 2015}}</ref>，而在[[Metacritic|Metacritic]]上得到90分<ref>{{cite web |url=http://www.metacritic.com/movie/saving-private-ryan |title=Saving Private Ryan |work=Metacritic |accessdate=December 15, 2015}}</ref>，收穫普遍好評。

=== 票房 ===
電影於1998年7月24日在美國的2,463個劇院上映，首映週末票房總額為3050萬美元。最終，北美收入2.165億美元，其他地區收穫2.653億美元，其全球總票房高達4.818億美元<ref name=BOM-SPR>{{cite web|accessdate=April 8, 2010|url=http://boxofficemojo.com/movies/?id=savingprivateryan.htm|title=Saving Private Ryan |publisher=Box Office Mojo }}</ref>，次於《[[世界末日_(電影)|世界末日]]》位居1998年全球電影票房第二。[[Box_Office_Mojo|Box Office Mojo]]估計，《搶救雷恩大兵》在北美銷售了超過4574萬張門票<ref>{{cite web|accessdate=July 13, 2016|url=http://www.boxofficemojo.com/movies/?id=savingprivateryan.htm&adjust_yr=1&p=.htm|title=Saving Private Ryan (1998)|publisher=Box Office Mojo}}</ref>。
=== 在台 ===
本片在台公映时，不少家长带孩子去电影院观看本片。因为片中的血腥场景引起儿童不适而引发家长抗议，使得[[電影分級制度|电影分级制]]话题浮出水面。<ref>{{cite web |url=http://gb.cri.cn/27564/2010/04/07/4945s2808570.htm |title=《孤堡惊情》加速中国电影分级制度的产生 |date=2010-04-07 |accessdate=2017-01-18}}</ref>

== 家庭媒體 ==
電影的家庭媒體於1999年5月發行，[[VHS|VHS]]發行量超過4400萬美元<ref>{{cite news|url=http://variety.com/1999/film/news/ryan-s-next-attack-sell-through-market-1117744320/|title='Ryan's' next attack: sell-through market|work=[[Variety_(magazine)|Variety]]|date=July 29, 1999|accessdate=September 6, 2008 | first=Marc | last=Graser}}</ref>。DVD版本於同年11月發售，且售出超過150萬套。

== 相關史實 ==
電影改編自[[二戰|二戰]]真實故事，主角真正本名為美軍[[第101空降師|第101空降師]]501團H連士兵{{Link-en|尼藍兄弟|Niland brothers|弗里茨·尼藍}}（Fritz Niland），在美軍攻下卡倫坦數日後，弗里茨得到消息得知他在[[第82空降師|第82空降師]]的哥哥鲍勃在[[D日|D日]]中陣亡，同時他在[[美國陸軍第4步兵師|第4步兵師]]的另一位哥哥也在D日中陣亡於猶他灘頭，不久後法蘭西斯[[神父|神父]]也來陣地中告知他的大哥（[[轟炸機|轟炸機]]的機組成員）在D日的同一周中於[[中緬印戰區|中緬印邊界]]被擊落，同一天中尼藍夫人同時接到三封兒子陣亡的電報，佛利茲是家中剩下唯一生存的兒子，陸軍於是下令將他優先送回國。

電影後段小鎮攻防戰的情節則純屬虛構。另外本電影並非如之前傳言所稱取材於[[二戰|二戰]]時服役於[[美國海軍|美國海軍]]的[[蘇利文兄弟|蘇利文五兄弟]]。而最後證實弗里茨·尼藍的大哥並未陣亡，只是成為[[日軍|日軍]]的[[俘虜|俘虜]]，最後被[[英軍|英軍]]所救{{notetag|《BAND OF BROTHERS》（[[諾曼第大空降|諾曼第大空降]]）一劇中進攻卡倫坦的片段}}。

本片中的德军都剃平头，与史实不符。
