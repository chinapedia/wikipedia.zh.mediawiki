'''關山越嶺古道'''，[[臺灣|臺灣]]一條位在[[南臺灣|南部]]的古道。在[[臺灣日治時期|日治時期]]，則稱為'''關山越嶺道'''，其前身是由[[理蕃道路|理蕃道路]]修建而來，因此稱為'''關山越警備道'''、'''關山越嶺警備道'''。全線西起於[[高雄市|高雄市]][[六龜區|六龜區]]，橫跨[[中央山脈|中央山脈]]，東抵[[臺東縣|臺東縣]][[關山鎮|關山鎮]]，全長約171.06公里，也是全臺灣最長的古道。<ref name="《台灣古道特輯》 "/>[[二次大戰|二次大戰]]結束後，[[臺灣省政府|臺灣省政府]]以該古道作為原型路廊而興建了[[臺20線|臺20線]][[南橫公路|南橫公路]]，故其大部分原始路廊皆已消失，所以在目前保留下來的路廊最有名的是「'''中之關古道'''」，位在中之關至天池之間的路段。<ref name="《台灣古道特輯》 "> 《台灣古道特輯》，國民旅遊出版社，出版日期：1999/12/1 ISBN 9578554664</ref><ref name="《台灣的古道》 "> 《台灣的古道》，遠足文化，出版日期：民國91年5月 ISBN 9868015421</ref><ref name="中華民國行政院農業委員會林務局-臺灣山林悠遊網 關山─內本鹿越嶺國家步道系統 ">{{cite web |url=http://recreation.forest.gov.tw/Rt/RT_1_6_13.aspx |title=中華民國行政院農業委員會林務局-臺灣山林悠遊網 關山─內本鹿越嶺國家步道系統 | accessdate=2015-05-31}}</ref>

== 歷史 ==
[[嘉慶|嘉慶]]年間，居住在現今臺東與高雄交界處之中央山脈山區的[[臺灣原住民|原住民]]，當地的[[布農族|布農族]]由[[巒社群|巒社群]]與[[郡社群|郡社群]]自[[郡大溪|郡大溪]]向南跨過中央山脈後，來到現今在[[海端鄉|海端鄉]]的新武地區，這條遷徙路線即為關山越嶺古道最早的雛型。<ref name="《台灣古道特輯》 "/> <ref name="《台灣的古道》 "/>

[[清廷|清廷]]在[[甲午戰爭|甲午戰爭]]戰敗後，臺灣割讓給[[日本|日本]]後，自此臺灣便進入日治時期。1914年，時任[[臺灣總督|臺灣總督]]的[[佐久間左馬太|佐久間左馬太]]執行「[[五年計畫理蕃事業|五年計畫理蕃事業]]」，為防止山區滋事叛亂，因此繳收原住民所持有的[[獵槍|獵槍]]，導致當地原住民極度不滿，後續爆發[[逢阪事件|逢阪事件]]、[[大關山事件|大關山事件]]，及其他等多次反抗事件，爾後部分原住民便逃入山中，日本人為此決定興建一條警備道路，用以監視山區的[[布農族|布農族]]。<ref name="《台灣古道特輯》 "/> <ref name="《台灣的古道》 "/>

1926年，關山越嶺道開始興建，在1931年時全線完工，並且每5至6公里處便設有一處[[警察官吏駐在所|警察官吏駐在所]]，也設有多處[[砲台|砲台]]用以震懾原住民，其中當今然保存的砲台是臺東縣海端鄉]]的霧鹿砲台。 <ref name="《台灣的古道》 "/><ref name="高雄縣鄉土教育服務 中之關教育教材 ">{{cite web |url=http://content.ks.edu.tw/33k/006_historictrail/004_kno01.html |title=高雄縣鄉土教育服務 中之關教育教材 | accessdate=2015-05-31}}</ref>

[[臺灣戰後時期|戰後]]，[[國民政府|國民政府]]接收臺灣後，規劃興建[[南部橫貫公路|南部橫貫公路]]，於1968年7月動工，歷時4年4個月，1972年10月完工通車；其中，部分路廊依照原有的關山越嶺古道而興建，部分則為新闢道路，因此目前仍有約65公里八大段的古道路基尚存在，在天池至中之關路段由玉山國家公園改為健行步道，稱之中之關古道。<ref name="中華民國行政院農業委員會林務局-臺灣山林悠遊網 關山─內本鹿越嶺國家步道系統 "/><ref>{{cite web|url=http://www.thbu4.gov.tw/view01-01.asp|title=交通部公路總局第三區養護工程處－南橫之美}}{{dead link|date=十一月 2017 |bot=InternetArchiveBot }}</ref>

== 沿線景點 ==
=== 霧鹿砲台 ===
霧鹿砲台興建在臺灣日治時期[[昭和|昭和]]年間，與關山越嶺古道為同時期興建的，架設有一尊大砲，該大砲來自1903年[[日俄戰爭|日俄戰爭]]時由[[日本軍|日本軍]]繳獲，後續1927年輾轉運送至臺灣並架設在霧鹿部落的制高點以希望能夠震懾當地布農族原住民，周遭還設有薩苦（大崙）及馬典古魯（摩天）等兩座砲台。<ref name="高雄縣鄉土教育服務 中之關教育教材 "/><ref name="花東縱谷國家風景區 霧鹿砲台 ">{{cite web |url=http://www.erv-nsa.gov.tw/user/Article.aspx?Lang=1&SNo=03000154 |title=花東縱谷國家風景區 霧鹿砲台 |accessdate=2015-05-31 |archive-url=https://web.archive.org/web/20150518023714/http://www.erv-nsa.gov.tw/user/Article.aspx?Lang=1&SNo=03000154 |archive-date=2015-05-18 |dead-url=yes }}</ref>

第二次世界大戰結束後，日本無條件投降，霧鹿砲台的大炮並未被運回日本，而是存放在現今霧鹿國小後方的公園，這裡曾是日本人[[囚禁|囚禁]][[罪犯|罪犯]]的地方，在入口刻有於昭和年間的「[[南無阿彌陀佛|南無阿彌陀佛]]」石碑，據傳是為了安息布農族戰士之魂。<ref name="花東縱谷國家風景區 霧鹿砲台 "/><ref name="臺東縣政府 臺東觀光網 霧鹿砲台 ">{{cite web |url=http://tour.taitung.gov.tw/zh-tw/Travel/ScenicSpot/219/%E9%9C%A7%E9%B9%BF%E5%8F%A4%E7%A0%B2%E5%8F%B0 |title=臺東縣政府 臺東觀光網 霧鹿砲台 | accessdate=2015-05-31}}</ref>

目前存放在霧鹿國小後方公園的大砲共有兩座，其中一座便是霧鹿砲台的大砲，原本因當地布農族人加上當時臺灣省政府的反日情緒而拆掉了該砲台並移往臺東，後續又移回霧鹿。<ref>{{cite web|url=http://catalog.digitalarchives.tw/item/00/43/4a/b4.html|title=典藏臺灣 布農族--霧鹿古砲台}}</ref>另一座為原馬典古魯（摩天）砲台的大砲，馬典古魯（摩天）砲台的砲座則已改為[[公路總局|公路總局]]的養護所。<ref name="花東縱谷國家風景區 霧鹿砲台 "/><ref name="臺東縣政府 臺東觀光網 霧鹿砲台 "/><ref>{{cite web|url=http://www.walkingtaiwan.org/content/county/city_POI.asp?ids=546&jid=82&bid=8|title=走讀臺灣 海端鄉：霧鹿部落與古砲台}}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

=== 中之關古道 ===
中之關古道為關山越嶺古道至今尚存在的路廊之一，由[[玉山國家公園|玉山國家公園]]1988年7月到1989年5月，分12梯次勘查，後續耗時3年重建，於1997年完工開放，全長約3.5公里，占全關山越嶺古道的百分之二。古道中之關端之起點位在南橫公路130.5公里處至天池，終點處留有日治時期的駐在所殘跡、[[木炭|木炭]][[窯|窯]]與石造路基。<ref name="高雄縣鄉土教育服務 中之關教育教材 "/><ref name="玉山國家公園關山越嶺古道調查研究報告 ">{{cite web |url=http://www.ysnp.gov.tw/researchreport_messages.aspx?id=38 |title=玉山國家公園關山越嶺古道調查研究報告 | accessdate=2015-05-31}}</ref>

在古道海拔2,120公尺的中之關，此地可俯瞰布農族抗日領袖拉荷阿雷所在的玉穗社，因此當時日本人在此建有中之關駐在所，並除了派有全線最高階警官「警部補」外再加上派駐砲隊，為當時關山越嶺古道上極重要的軍事據點。<ref name="高雄縣鄉土教育服務 中之關教育教材 "/><ref name="玉山國家公園關山越嶺古道調查研究報告 "/>
