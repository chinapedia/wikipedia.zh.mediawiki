'''關信基'''{{BD|1940年|||}}是[[香港|香港]][[公民黨_(香港)|公民黨]]創黨主席，[[香港中文大學|香港中文大學]]政治與行政學系榮休講座教授，暱稱「阿關」。1973年加入[[香港中文大學|香港中文大學]]，曾擔任政治與行政學-{系系}-主任，{{Fact|被稱為系內「四大長老」之一（另外三位為[[鄭赤琰|鄭赤琰]]、[[翁松燃|翁松燃]]及[[李南雄|李南雄]]），同為香港社會科學界「四大巨頭」（另外三位為任教於[[香港中文大學|香港中文大學]]的[[劉兆佳|劉兆佳]]、[[香港理工學院|香港理工學院]]/[[香港理工大學|香港理工大學]]的[[李明堃|李明堃]]及[[香港大學|香港大學]]的[[黃紹倫|黃紹倫]]）|time=2009-03-07}}，於2006年退休，但退休以後仍兼職教授政治學科目——當代政治科學。

關信基1941年生於[[澳門|澳門]]，畢業於[[澳門|澳門]]名校[[粵華中學|粵華中學]]，1958年入讀[[台灣|台灣]][[國立政治大學|國立政治大學]]外交學系，1972年於[[慕尼黑大學|慕尼黑大學]]取得政治學博士學位，他本來打算返台灣任教，但機緣巧合下被介紹到[[香港中文大學|香港中文大學]]（中大）任教，直到今天。<ref>{{cite web |url=http://www.civicparty.hk/cp/pages/page-c/exco-c.php |title=存档副本 |accessdate=2011-07-31 |deadurl=yes |archiveurl=https://web.archive.org/web/20110315121458/http://www.civicparty.hk/cp/pages/page-c/exco-c.php |archivedate=2011-03-15 }}</ref><ref>{{cite web |url=http://www.civicparty.hk/cp/pages/page-c/exco-c.php |title=存档副本 |accessdate=2011-07-31 |deadurl=yes |archiveurl=https://web.archive.org/web/20110315121458/http://www.civicparty.hk/cp/pages/page-c/exco-c.php |archivedate=2011-03-15 }}</ref><ref>http://www.cuhk.edu.hk/cpr/pressrelease/090518c.htm</ref><ref>http://www.yzzk.com/cfm/Content_Archive.cfm?Channel=ae&Path=3625671052/15ae2b.cfm</ref>

除了中大政治與行政學-{系系}-主任外，他還曾出任該校教務長、社會科學院院長和新聞與傳播學-{系系}-主任，同時亦是中大教務會的成員。

「阿關」的學術領域相當廣闊，對政治文化、政治參與、政黨、比較政治、政治科學等均有深入的研究，是本港學界最受人尊崇的學者之一，[[蔡子強|蔡子強]]的碩士論文導師正是「阿關」。
