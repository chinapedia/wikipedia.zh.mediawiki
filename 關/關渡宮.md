'''關渡宮'''，位於[[中華民國|中華民國]]（[[臺灣|臺灣]]）[[臺北市|臺北市]][[北投區|北投區]][[關渡|關渡]]，俗稱'''關渡媽祖宮'''、'''關渡天后宮'''。 為主祀[[天上聖母|天上聖母]]媽祖之[[臺灣民間信仰|臺灣民間信仰]]廟宇，兼容[[佛教|佛教]]與[[道教|道教]]，為[[北臺灣|北臺灣]]歷史最悠久、香火最鼎盛的[[媽祖廟|媽祖廟]]，早期有「南有北港媽、北有關渡媽」一說，尊崇[[北港朝天宮|北港朝天宮]]與[[關渡宮|關渡宮]]天上聖母。關渡宮與北港朝天宮、[[鹿港天后宮|鹿港天后宮]]合稱「臺灣三大媽祖廟」。

北臺灣各地都有迎請關渡媽祖做客的習俗與傳統。甚至擴展到[[中臺灣|中臺灣]]與[[南臺灣|南臺灣]]，民間相傳，關渡宮以開基大媽與開基[[軟身神像|軟身]]二媽最為靈驗，也為各地信徒主要迎請巡庄繞境的神像。

== 沿革 ==
廟方表示該廟是[[南明|南明]][[永曆_(南明)|永曆]]十五年（1661年，[[清|清]][[順治|順治]]十八年），佛教[[臨濟宗|臨濟宗]]'''石興和尚'''從[[福建省|福建省]][[興化府|興化府]][[莆田縣|莆田縣]][[湄洲天后宮|湄洲天后宮]]分靈媽祖金身來臺(今開基大媽)，以茅草建屋，奉祀於干豆（[[關渡|關渡]]）山區{{r|關}}，但是黃富三〈北台首座媽祖廟關渡宮之起源與角色〉一文認為此一說法欠缺歷史文獻支持{{r|黃}}。目前就史料《[[諸羅縣志|諸羅縣志]]》來說，「干豆門天妃廟」是[[清|清]][[康熙|康熙]]五十一年（1712年），大雞籠社[[通事舍人|通事]][[賴科|賴科]]鳩眾所建於山上；康熙五十四年（1715年）時重修，易茅以瓦，建材自[[廈門|廈門]]運來，[[諸羅縣|諸羅縣]][[知縣|知縣]][[周鍾瑄|周鍾瑄]]命名曰「靈山廟」{{r|黃}}{{r|簡}}。該廟原建於山頂上，康熙五十八年（1719年）遷到山麓；乾隆四十七年（1782年）因廟宇方位不正，再次重修；道光三年（1823年）因風雨侵襲毀損，再次重修{{r|簡}}。

[[台灣日治時期|台灣日治時期]]，[[明治|明治]]卅年（1897年），關渡富商[[林大春|林大春]]因講究[[風水|風水]]學，收購廟地，改建為自己之住宅。後林大春捐了兩百八十[[銀元|銀元]]，並鳩資遷移「靈山天妃廟」到風水上稱作「萬水歸堂穴」的現址，坐西朝東，易名為「關渡祖宮」。

[[大正|大正]]十一年（1922年）再次重修，更名為「關渡宮」。

== 奉祀神佛 ==
'''關渡宮'''一樓正殿奉祀[[天上聖母|天上聖母]]、[[觀世音菩薩|觀世音菩薩]]、[[文昌帝君|文昌帝君]]。左側有「關渡延平郡王祠」，奉祀[[延平郡王|延平郡王]]與其屬神三位將軍，其原廟在1961年風災摧毀，[[檀越|檀越]]遷廟於此，意同於本廟的側殿。右側奉祀建廟大德。旁殿奉祀[[藥師佛|藥師佛]]、[[阿彌陀佛|阿彌陀佛]]、[[觀音菩薩|觀音菩薩]]、[[大勢至菩薩|大勢至菩薩]]等。二樓的「廣渡寺」奉祀[[西方三聖|西方三聖]]、[[地藏菩薩|地藏菩薩]]與信徒安奉的[[祖先|祖先]][[牌位|牌位]]。

後殿奉祀[[玉皇大帝|玉皇大帝]]、[[三官大帝|三官大帝]]、[[東華帝君|東華帝君]]、[[瑤池金母|瑤池金母]]、[[南斗星君|南斗星君]]、[[北斗星君|北斗星君]]等。

古佛洞奉祀[[千手觀音|千手觀音]]、[[羅漢|羅漢]]、[[四大天王|四大天王]]、[[天龍八部|天龍八部]]廿八位[[護法神|護法神]]等。財神洞奉祀[[財神|財神]]、[[福德正神|福德正神]]等。兩洞宗教雕塑繁多，為關渡宮一大殊景，頗為可觀。

== 遊覽特色 ==
關渡宮的後山（靈山公園）視野極佳，遠眺[[淡水河|淡水河]]、[[觀音山_(新北市)|觀音山]]、[[臺北盆地|臺北盆地]]；尤其是夕陽西下之景，景色優美。[[秋|秋]][[冬|冬]]時節，許多人會在此處觀賞[[候鳥|候鳥]]。靈山約有十餘株[[櫻花|櫻花]]，為日治時期所移植。早期的關渡三潮是關渡頗為奇特的景觀，但自從獅象守口的地形被炸開後，此景已不復在，但靈山公園中仍有由[[黃杰_(將軍)|黃杰]]將軍所提的「關渡三潮」紀念石碑。

關渡宮附近的名產以[[鹹鴨蛋|鹹鴨蛋]]、[[皮蛋|皮蛋]]、[[鐵蛋|鐵蛋]]聞名，許多香客皆會採購。

關渡宮旁有座「[[關渡玉女宮|玉女宮]]」，創建於道光十六年。
