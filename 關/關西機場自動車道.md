'''關西機場自動車道'''（{{jpn|j=関西空港自動車道|hg=かんさいくうこうじどうしゃどう|rm=Kansai Kūkō jidōshadō|f=y}}）是[[日本|日本]][[大阪府|大阪府]][[泉佐野市|泉佐野市]][[阪和自動車道|阪和自動車道]]至[[關西國際機場|關西國際機場]]的[[高速公路|高速公路]]（[[高速自動車國道|高速自動車國道]]），簡稱'''關西機場道'''（{{lang|ja|関西空港道／かんさいくうこうどう}}）、'''關空道'''（{{lang|ja|関空道／かんくうどう}}）。

高速公路路線編號與[[關西國際機場聯絡橋|關西國際機場聯絡橋]]共同被編為 {{Ja Exp Route Sign|E71}}<ref>{{Cite web|url=http://www.mlit.go.jp/road/sign/numbering/list/index.html|title=高速道路ナンバリング一覧|accessdate=2017年2月26日|publisher=国土交通省}}</ref>。

== 概要 ==
自[[阪和自動車道|阪和自動車道]]的泉佐野系統交流道分歧，在臨空系統交流道（{{lang|ja|りんくうJCT}}）與[[關西國際機場聯絡橋|關西國際機場聯絡橋（天空大橋<small><sup>R</sup></small>）]]連接。

本路線非由[[國土開發幹線自動車道|國土開發幹線自動車道]]指定，而為[[高速自動車國道路線指定政令|高速自動車國道路線指定政令]]指定的[[高速自動車國道|高速自動車國道]]。政令路線名稱是'''關西國際機場線'''（{{lang|ja|関西国際空港線}}）。

== 歷史 ==
* 1994年4月2日 : 全線通車。
* 2018年
**9月4日，[[颱風燕子_(2018年)|燕子颱風]]侵襲日本，在[[西日本|西日本]]登陸時刮起的強風使得一艘為關西機場補給航空油料的[[液貨船|液貨船]]「宝運丸」走錨漂流，並撞擊到關西國際機場聯絡橋南側車道，造成關西機場自動車道的一段橋面偏移。事故發生後，所有船員生還，鐵公路當時已關閉而沒有車輛受損及人員傷亡，但使得機場連外的鐵路與公路交通暫時中斷<ref>{{Cite web |url=https://www3.nhk.or.jp/news/html/20180904/k10011607811000.html# |title=関西空港の連絡橋にタンカー衝突 乗組員は全員無事救助 |access-date=2018-09-04 |archive-url=https://web.archive.org/web/20180904192123/https://www3.nhk.or.jp/news/html/20180904/k10011607811000.html# |archive-date=2018-09-04 |dead-url=yes }}</ref>，約3000人一時受困在機場內<ref>{{Cite web |url=https://www3.nhk.or.jp/news/html/20180904/k10011609201000.html# |title=関西空港 タンカー衝突で橋不通 利用客3000人が空港内に宿泊 |access-date=2018-09-04 |archive-url=https://web.archive.org/web/20180904192223/https://www3.nhk.or.jp/news/html/20180904/k10011609201000.html# |archive-date=2018-09-04 |dead-url=yes }}</ref>。
**9月5日，在檢查確認北側車道未受損後，以巴士將受困者從機場疏運<ref>{{Cite web |url=https://www3.nhk.or.jp/news/html/20180905/k10011609751000.html# |title=関西空港 バスでの輸送も始まる |access-date=2018-09-05 |archive-url=https://web.archive.org/web/20180905020223/https://www3.nhk.or.jp/news/html/20180905/k10011609751000.html# |archive-date=2018-09-05 |dead-url=yes }}</ref>，另外在海象穩定後，也以[[高速船|高速船]]載運受困者前往[[神戶機場|神戶機場]]<ref>{{Cite web |url=https://www3.nhk.or.jp/news/html/20180905/k10011609661000.html# |title=関西空港から神戸空港へ 3000人を高速船でピストン輸送 |access-date=2018-09-05 |archive-url=https://web.archive.org/web/20180905020103/https://www3.nhk.or.jp/news/html/20180905/k10011609661000.html# |archive-date=2018-09-05 |dead-url=yes }}</ref>。除了獲得許可通行的救災相關車輛及接駁巴士，機場自動車道暫不對其他車輛開放通行。
**9月14日，吊離遭撞損的南側公路橋面<ref>{{Cite web |url=https://www3.nhk.or.jp/news/html/20180912/k10011626081000.html# |title=関西空港連絡橋の橋桁撤去始まる 鉄道は今月中の復旧目指す |access-date=2018-09-18 |archive-url=https://web.archive.org/web/20180913134312/https://www3.nhk.or.jp/news/html/20180912/k10011626081000.html# |archive-date=2018-09-13 |dead-url=yes }}</ref>。
**9月21日，開放懸掛營業車牌的貨車、計程車等車輛通行。
**10月6日，開放一般自用車輛通行<ref>[http://www.kansai-airports.co.jp/news/2018/2637/J_181004_privatecar_hp.pdf 関西空港連絡橋 10月6日（土）0時よりマイカーの通行が可能になります。]（関西エアポート、2018年10月16日閲覧）</ref>。
