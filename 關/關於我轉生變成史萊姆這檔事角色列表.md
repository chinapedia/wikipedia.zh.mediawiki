== 朱拉·坦派斯特聯邦國（魔國聯邦） ==
; <span id="利姆路">{{nihongo|利姆路·坦派斯特|リムル＝テンペスト|Rimuru-Tempest|聲：[[岡咲美保|岡咲美保]]<ref name="character">{{Cite web|url=http://www.ten-sura.com/character/|title=角色|work=電視動畫「關於我轉生變成史萊姆這檔事」官方網站|accessdate=2018-06-08}}</ref>}}
: 統領朱拉大森林全域的新世代魔王，龍魔黏性星神體（究極史萊姆，為Web版龍魔大戰篇後的種族），稱號為「新星」，是現任「八星魔王」之一（Web版天魔大戰稱號為「大魔王」）。魔國聯邦的三大偶像之一。
: 前世是名為{{nihongo|'''三上悟'''|みかみ さとる|聲：[[寺島拓篤|寺島拓篤]]<ref>{{Cite Web|url=http://www.ten-sura.com/news/221/|title=追加角色&聲優情報發表！|work=電視動畫「關於我轉生變成史萊姆這檔事」官方公式網站|accessdate=2018-09-17}}</ref>}}的人類，原本是三十七歲的普通上班族，在公司後輩介紹他的女友認識時，為了保護後輩而被突然出現的持刀強盜（隨機殺人魔）刺殺。
: 轉世後成為[[史萊姆|史萊姆]]，獲得兩種特殊技能：分別是『捕食者』（後進化為究極技能《暴食之王 別西卜》）和『大賢者』（後進化為究極技能《智慧之王 拉斐爾》）。現世所用的名字是被維爾德拉所取的，但彼此共同的姓氏「坦派斯特（Tempest）」則是由利姆路所命名。<ref>【゠】（U+30A0）「リムル゠テンペスト」</ref>
: 為了協助維爾德拉解除勇者施加的技能『無限牢獄（Infinity Prison）』的封印，使用了捕食吸收維爾德拉進入胃袋之中進行解析。此後利姆路不斷地使用著『捕食者』吸收了許多魔獸強化了自己的能力，並獲得如哥布林、嵐牙狼、大鬼族等眾多魔物作為部下，而後在朱拉大森林裡建立起了魔物的城鎮。前後吞噬了豬頭魔王和暴風大妖渦而成為魔王種。
: 由於法爾姆斯王國軍的侵略事件，造成了城鎮內超過了百位居民的喪生，其中也包含了負責抵禦敵軍攻擊的紫苑。之後極其憤怒和悲傷的利姆路接受了愛蓮所說的建議，為了要讓死去的眾人復活，而挑戰了魔王進化。為了要獲取成為魔王所需的人類靈魂，無情地虐殺了前來侵略的法爾姆斯王國軍隊，甚至還因此獲得了恐怖的独有技能『心无者』，以此技能瞬間收割了敵軍所有人的兩萬條生命，並將兩萬具屍體做為祭品召喚出迪亞布羅及其兩名手下，命令他們活捉在屍體中裝死的拉贊。最終靠著獻祭兩萬條生魂，利姆路成為了覺醒魔王，復活了所有死去的魔物們，且成為了精神生命體，原本進化成覺醒魔王時並不會獲得究極技能，但因大賢者無數次的嘗試後成功進化成智慧之王，在用自身的權能幫利姆路獲得誓約之王、暴食之王，而後釋放維爾德拉並分析了其魔力殘渣，整合出了暴風之王。
: 在魔王們的饗宴中把「十大魔王」之一的克雷曼殺死後，得到了以奇伊為首的魔王們承認為魔王之一。自此之後，「十大魔王」這稱呼成為了歷史，而由於同為魔王的卡利翁和芙蕾的退位，以及利姆路的加入所誕生的「八星魔王」，這名號則被世人所認知。平時以任性、狡猾但為人溫厚、寬容且性格天真被大家所認知，但對待敵人與為了同伴而下達的命令又很殘酷，但在蹂躪了東之帝國近百萬的士兵後，又將其盡可能復活，但施以不容背叛的詛咒。（那是以擬似魂將其復活，雖保有人格和記憶，但無法復活肉體被消滅和喪失心靈的人，因並非以原本的靈魂復活，故會喪失靈魂之力而無法使用技能並喪失身體能力以外的力量，而原本那近百萬沒有心核的靈魂後來被利姆路拿來給部下進化了。）
: 在克羅諾亞記憶中的平行世界裡，利姆路因為在救助了孩子們後立刻回國因此沒有遭到日向襲擊，也因此在法爾姆斯的襲擊中迅速的將省吾等人徹底擊敗而錯失進化為覺醒魔王的時機，五年後便在實力不足的情況下面對帝國的侵略而戰死，同時這也促成了維爾德拉的復活。但是維爾德拉也因為利姆路的死而暴走，並將怒火指向人類，不只將帝國軍全滅，甚至連魔國聯邦也一併摧毀。但其實利姆路並未完全死去，透過『大賢者』復活後覺醒為真魔王，在早已改變的世界和毀滅的魔國聯邦中尋找倖存者，後來經過多次挑戰成功阻止克羅諾亞暴走，但世界的局勢已無可挽回且殘破不堪。據克羅諾亞所言，該平行世界的利姆路覺醒後比現在這個時間點還強。
: （Web版）龍魔激突中，利姆路在與維魯葛蓮多以及被操控的維爾多拉戰鬥中往龍種進化，力量大幅提升，但不是完全的龍種。頂上決戰後與成為「神人」的優樹對峙時，在停止的時間中，中了優樹的陷阱而被彈到遙遠的未來「時空的盡頭」，連時間也不復存在，利姆路經過了極悠久的時間醒了過來，在『虛數空間』儲蓄了龐大的『虛無崩壞』的能量，足以創造世界數萬次，後來拒絕再創造一個與原來的世界一樣的新世界，而利用穿越時空回到原來的時間點將優樹輕易擊敗後將其吞噬。最後成為「拯救世界的大魔王」被人們歌頌。結局時穿越世界與時空回到日本，在「三上悟」剛被歹徒刺死後的時間點，將還是「三上悟」的自己復活並複製自己的意識移到「三上悟」的自己身上後回到異世界，幾天後三上悟在醫院醒來，向後輩田村述說關於自己轉生變成史萊姆的那檔事。
: 其個性因為具有前世作為和平世界的日本人記憶，因此個性天真並且和善，容易大意而被算計，但是因為是從維爾德拉的妖氣中誕生出來的特殊個體，因此從誕生之初就擁有驚人的魔素量，總是被他人認為是強者的盈餘。而且因為有大賢者這種輔助思考能力因此所作所為都讓他人感受到一種深不可測的感覺，加上由於前世是死於隨機刺殺犯因此會堅守絕不退讓的底限，一旦觸及逆鱗就會變得冷酷，因此成為周遭的人們的警戒榜首，也時常有敵對方因為盲信個性天真的傳聞而踏過容許線，從而導致遭到毀滅性打擊。

=== 圣魔十二守护王 ===
即利姆路的部下中有資格被授魂覺醒為真魔王的十二人，此十二人最後都擁有覺醒魔王級甚至是在此之上的力量。

; {{nihongo|紅丸|ベニマル|benimaru|聲：[[古川慎|古川慎]]<ref name="character"/>}}
: 赫怒王（{{lang|ja|フレアロード}}）——赫怒王（Flare Lord）

== 其他角色 ==

; {{nihongo|田村|たむら|tamura|聲：[[鈴木崚汰|鈴木崚汰]]}}
: 在利姆路生前還是三上悟時，公司的後輩。二十八歲，艷福卻不淺，與澤渡美惠即將結婚。

; {{nihongo|澤渡美穗|さわたり みほ|sawatari miho|聲：[[貫井柚佳|貫井柚佳]]}}
: 在利姆路生前還是三上悟時，公司的後輩，與田村即將結婚。

; {{nihongo|蓋德（豬頭帝／豬頭魔王）|ゲルド（豚頭帝／豚頭魔王）|Geld|聲：[[乃村健次|乃村健次]]}}
: 原豬頭族自治區歐比克的豬頭人，在命危時得魔人喀爾謬德施救和命名而進化為豬頭帝。由於其特殊技能『饥饿者』的效果使全族都受到影響，豬頭族全員能將捕食對象的能力據為己有，但同時會受到永遠無法滿足的飢餓感折磨。
: 為解決飢荒問題而率領了全族往朱拉大森林進軍。在攻擊蜥蜴人族的聚落途中，與及時趕來的利姆路聯軍交戰。在與利姆路率領的聯軍交戰之時，捕食了突然闖進了戰場胡鬧的喀爾謬德而進化為了豬頭魔王。在進化後與利姆路兩人一對一進行決戰，最後不敵而敗北，在死前將全族託付給了利姆路。
::; {{nihongo|飢餓者|{{ruby|飢餓者|ウエルモノ}}}}『腐蚀、吸收、供给』
:: 在豬頭帝誕生之時必定會持有的技能，能影響所有處在支配下的存在，擁有能夠奪取補食對象的能力，但同時會被無法填補的飢餓感所折磨。

; <span id="卡律布狄斯">{{nihongo|卡律布狄斯|カリュブディス|Charybdis}}
: 稱為「暴风大妖涡」的魔王級精神生命體，每次死亡後隔段時間又會復活。因為是由維爾德拉的魔素而產生，又被稱作維爾德拉之子，外型是巨大的獨眼龍。並沒有堅實的自我，而是順從破壞的意志而行動。
: 據卡利斯所說，該性別似乎是女性。

; 勇者
: 300年前隻身對付『暴風龍』並成功封印的人類，人類歷史上最強大的「勇者」。自稱是從異世界來的「召喚者」。是黑銀長髮、身材嬌小、戴著面具、強大無比的美女。持有『絕對切斷』、『無限牢獄』等數種特殊能力。
: 名字是「{{ruby|克羅諾亞|破滅意志}}」，是克蘿耶在勇者活動時期所使用的的身份。穿越時空回到過去的克蘿耶，為了隱藏本名所以使用了化名，在日向的提議之下，決定使用「克羅諾亞」這個名字。
: 由於被日向取名的關係，逐漸有了與克蘿耶不同的人格。因為有著利姆路被打倒了的世界的記憶，後來成為了恶德的化身，變成只會憑著惡意破壞世界的存在，直到在克蘿耶體內的「印象風景」中與利姆路再次相見而恢復自我。
: 因為智慧之王能力的干涉而被變成了称为「神智核」的情报体，現寄宿在克蘿耶體內。非常喜歡利姆路。後來時常與克蘿耶互換，戰鬥中基本上由克羅諾亞負責或是輔助克蘿耶。與克蘿耶關係有如姐妹一般。
: 詳見『[[#克蘿耶|克蘿耶]]』。

; 維魯達納瓦 
: 被稱為『星王龍』的龍種之一，司掌星粒子等能量的創造。創造異世界，為了避免人類最後因為自相殘殺而滅亡，於是要金·克林姆茲擔任魔王，防止人類過度自相殘殺。
: 因為和魯德拉的妹妹露西亞結有了後代，違反自己所創的規則，間接造成了自己的死亡。重生後變成了自己的女兒蜜莉姆的寵物，但最後卻被長耳族殺害，從此沒有復活跡象，取而代之的是世界上除了龍種之外，也出現了許多龍族。
