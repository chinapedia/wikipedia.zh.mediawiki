'''關渡大橋'''是橫跨於[[台灣|台灣]][[北台灣|北部]]的[[淡水河|淡水河]]的[[橋樑|橋樑]]，為省道[[台15線|台15線]]首段，連結[[新北市|新北市]][[八里區|八里區]]、[[五股區|五股區]]與[[淡水區|淡水區]]、[[台北市|台北市]][[北投區|北投區]][[關渡|關渡]]，距離淡水河口約10公里 <ref name="淺談臺 15 線關渡大橋橋梁安全檢測工作(一)">{{cite web|url=http://www.thb.gov.tw/TM/Menus/Menu09/main0901/download/921/33/3302-2.pdf|title=淺談臺 15 線關渡大橋橋梁安全檢測工作(一)|author=王韻瑾|publisher=交通部公路總局}}</ref>，在[[淡江大橋|淡江大橋]]完工前為淡水河流域最靠近出海口的一座橋樑。在立委呂孫綾爭取下，促成開通以來首次耐震補強工程，於2019年進行。<ref>{{Cite web|title=呂孫綾|url=https://www.facebook.com/shirleylu312/posts/2335036579853401|work=www.facebook.com|accessdate=2019-02-27|language=zh-Hans}}</ref>

== 諸元 ==

* 橋重量：約6000[[公噸|公噸]]（主橋的部份）

== 簡史 ==
{{unreferencedsect | time = 2017-1-18}}
在[[清朝|清朝]][[雍正|雍正]]年間，逃避[[蕭隴社|蕭隴社]]追殺而遷居到[[小八里坌社|小八里坌社]]（今日[[新北市|新北市]][[淡水區|淡水區]]的竹圍一帶）的八里坌社[[原住民|原住民]]，為了能來往於淡水河兩岸，於是在今關渡一帶搭建了藤橋，此橋史稱為“關渡藤橋”或是“雍正藤橋”，藤橋存在的時間並未很久。此後要往來淡水河兩岸只能依靠[[渡船|渡船]]。

1976年，時任[[行政院院長|行政院院長]][[蔣經國|蔣經國]]，指示[[台灣省公路局|臺灣省公路局]]研究在[[關渡|關渡]]八里一帶興建一座可便利交通又兼顧[[國防|國防]]需要的橋樑。公開徵圖後，由[[林同炎|林同棪國際工程顧問公司]]以及[[中華顧問工程司|中華顧問工程司]]聯合設計的鋼構橋設計圖在競圖中脫穎而出。橋樑從1980年4月起動工，1983年10月31日完工通車。

== 工法 ==
由於關渡大橋靠近淡水河出海口，為了行船以及排洪需要，而捨棄了施工容易且省錢的鋼筋混凝土橋，以大跨距的鋼構橋為主橋。三座巨拱成為施工的難題，由於向國外租用可容納三座巨拱的駁船昂貴又緩不濟急，因此施工承包商[[唐榮公司|唐榮公司]]發明出「潮汐施工法」。先將三座巨拱分別在駁船上製造焊接，同時間進行橋墩的施工。橋墩施工完畢後，將承載巨拱的駁船利用漲潮時拖到定位點，待退潮後水面降低，則巨拱的基部自然就定位到橋墩，再撤走駁船，如此不需重型的起重設備。而主橋鋼材大部份也由國營的中國鋼鐵公司自行生產。

== 設計 ==
關渡大橋是首座由中華民國國人自力建造的五孔連續鋼繫拱橋，為中路式鋼拱橋（或稱穿越式鋼拱橋），是設計師[[林同棪|林同棪]]生前最滿意的作品之一，大橋呈拋物線的紅色鋼拱，高度遞減變化有序，橋姿巍峨拱肋造型輕巧優美，如同[[彩虹|彩虹]]跳躍水面，能與周遭的[[觀音山_(新北市)|觀音山]]背景達到相得益彰，並與附近之環境及景色相匹配，落成時為當時世界三大鋼橋之一，目前為台灣長跨徑拱橋之一<ref>{{cite web|language=zh-tw|url=http://www.twce.org.tw/info/%E6%8A%80%E5%B8%AB%E5%A0%B1/377-1-3.htm|title=橋樑大師林同棪|author=|publisher=技師報|date=|accessdate=2014-06-14}}</ref>。

== 耐震補強工程 ==
關渡大橋自1983年完工通車至今已36餘年，當時的設計與現行公路橋梁耐震設計規範已有出入，積年累月的使用，加上城際間來往交織的車流也不時造成路面變形；位處[[山腳斷層|山腳斷層]]帶的地理位置也致使橋體、橋墩潛藏安全疑慮。關渡大橋自2017年9月公開招標以來，歷經6次流標，期間立委[[呂孫綾|呂孫綾]]持續於[[立法院|立法院]]交通委員會提出質詢，並要求[[交通部|交通部]]就標案內容合理性重新檢討，研議招標策略。最後終於在2018年12月底有兩家廠商投標，2019年1月的審查會中，有一家廠商符合資格。2019年2月26日，關渡大橋耐震改善及維修補強工程完成決標，並將在開工後兩年內完工，完工後橋體可抗震達5級強度。<ref>{{Cite web|title=呂孫綾|url=https://www.facebook.com/shirleylu312/posts/2335036579853401|work=www.facebook.com|accessdate=2019-02-27|language=zh-Hans}}</ref><ref>{{Cite web|title=提高預算再招標 關渡大橋史上第一次大修 - 生活 - 自由時報電子報|url=https://news.ltn.com.tw/news/life/breakingnews/2478329|work=news.ltn.com.tw|accessdate=2019-02-27}}</ref>
