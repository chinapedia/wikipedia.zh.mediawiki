'''關節式轉向架'''是指一種應用在[[鐵路車輛|鐵路車輛]]上的[[轉向架|轉向架]]形式。它是將相鄰的[[車廂|車廂]]用同一個轉向架連接起來，故日文稱呼此種轉向架為「{{lang|ja|連接台車}}」。（註：「{{lang|ja|台車}}」為日文的轉向架）。

在英文，此類轉向架形式被稱為「雅各式轉向架」（Jacobs Bogie）；主要是為了紀念該轉向架形式是由 Wilhelm Jakobs（1858 - 1942）所設計。

== 特徵 ==
=== 優點 ===
[[File:Nebraska_Zephyr_articulation.JPG|thumb]]」也是採用「關節式轉向架」連接。]]
# 相鄰車廂以轉向架連結，不但減少[[車勾|車勾]]的使用，也減少其產生噪音的機會。
# 轉向架使用的總數減少，對列車整體的輕量化有一定貢獻。
# 車體相對於轉向架動作的角度變小，容易通過較彎曲的鐵道。
# 車輛通過彎道時兩旁所突出的空間較小；因此可以採用更廣的車體。
# 車室與轉向架距離較遠，車輪行走時的噪音較不易傳入車室內。
# 轉向架設於連結處，車身設計可大幅降低重心，以增加列車行駛的穩定性。
# 整體車軸的數量減少，能減低製作和維護成本。

=== 缺點 ===
# 當車廂要分開或加掛時，都必須進入鐵路機廠作業，大大影響了列車編成的自由度。
# 轉向架數量減少使單一轉向架所承受重量也隨之變大，必須增加其結構強度。但考量軸重過重也會對[[鐵軌|鐵軌]]造成影響，故縮短車身減重也是一個好方法。
# 如果該列車屬於[[動力分散式|動力分散式]]車輛，則容易因為車軸的總數量減少，而使列車總輸出功率下降。

=== 例子 ===
使用這種'''關節式轉向架'''的火車，以下列舉幾種：
#[[日本|日本]]：[[小田急電鐵|小田急電鐵]]大部份[[浪漫特快|浪漫特快（Romance Car）]]列車、[[JR東日本E331系電力動車組|JR東日本E331系電力動車組]]。
#[[法国|法国]]：[[高速鐵路|高速鐵路]]的所有列車（[[法国高速列车|TGV]]）。
#[[德国|德国]]：[[Talent柴聯車|Talent]] 系列的[[柴聯車|柴聯車]]，或者是[[德國鐵路|德國鐵路]]（DB）用於[[S-Bahn|S-Bahn]]的 [[德國鐵路423系|423系]] [[電聯車|電聯車]]。
#[[美國|美國]]：在二十世紀初期的時候，這種連結方式也被大量採用在當時流行的[[流線型火車|流線型火車]]：像是[[芝加哥，伯靈頓與昆奇鐵路公司|芝加哥，伯靈頓與昆奇鐵路公司]] 的「[[先鋒者微風號|先鋒者微風號]]」與[[聯合太平洋鐵路|聯合太平洋鐵路]]的「[[M-10000|M-10000]]」。
#貨運火車：美國的[[雙層貨櫃車|雙層貨櫃車（Pacer Stacktrain）]]，也是將五節貨櫃運輸平車，以關節式轉向架連結；其用來載運貨櫃的中央部位，才得以降低車身容納雙層貨櫃的高度。
