'''關山大圳'''為[[臺灣|臺灣]][[臺東縣|臺東縣]][[關山鎮_(台灣)|關山鎮]]境內的一座[[灌溉|灌溉]]水圳系統，該圳最早於西元1907年（民國前6年）由當地住民所開鑿，並引水自[[卑南溪|卑南溪]]上游進水口位於[[初來橋|初來橋]]下方。

== 沿革 ==

西元1907年由臺灣西部往臺灣東部開墾的漢人引卑南溪水開設里壟圳，是為最初的關山圳，灌溉農地約30公頃，但因所處地帶時常和原住民發生衝突，兩年後放棄里壟圳的管理使用。<ref name="關山鎮公所 全球資訊網 關山大圳">{{cite web |url=http://www.guanshan.gov.tw/page6-1c.php |title=關山鎮公所 全球資訊網 關山大圳 |accessdate=2014-12-06 |deadurl=yes |archiveurl=https://web.archive.org/web/20150817020823/http://www.guanshan.gov.tw/page6-1c.php |archivedate=2015-08-17 }}</ref>

日治時期大正4年，1915年後，台灣總督府對當地原住民的管理漸趨嚴格，並正式接管當時已遭放棄的里壟圳，外來移民增多，西元1923年[[里壟圳公共埤圳|里壟圳公共埤圳]]成立，1941年改為[[關山鎮_(臺灣)|關山]][[水利組合|水利組合]]。<ref name="關山鎮公所 全球資訊網 關山大圳" />

民國37年，[[關山鎮_(臺灣)|關山鎮]]前[[鎮長|鎮長]][[曾玉崙|曾玉崙]]當時擔任[[關山鎮_(臺灣)|關山]][[農田水利會|農田水利會]]會長，爭取興建大圳，希望藉由水圳的開闢，將關山地區的荒地化為良田。由前[[臺灣省|臺灣省]][[水利局|水利局]]斥資興建，自卑南溪初來段引水，並以[[日治時期|日治時期]]的里壟圳為基礎重建。<ref name="關山鎮公所 全球資訊網 關山大圳" />

工程自民國37年7月正式動工開鑿，歷時10年，於民國47年10月10日全面完工，圳道主幹線全長20公里，17條支線，[[排梳型|排梳型]]圳系，灌溉面積2,874公頃，，並改善因卑南溪湍急溪水易沖毀圳道的缺點，是臺灣[[東部地區|東部地區]]第二大的水圳。<ref name="關山鎮公所 全球資訊網 關山大圳" />

== 設施 ==

=== 水利公園 ===

自1999年(民國88年)關山圳開始綠化，在上游四公里導水路旁的管理道路兩側栽植[[楓樹|楓樹]]後，[[新武呂溪|新武呂溪]]初來段右岸出口的關山圳砂池周遭便建起假山、雙併式涼亭.戲水池和戲水空間做為進水口水利公園。<ref>{{cite web |url=http://www.ttia.gov.tw/water_page_ks.php |title=台東農田水利會 關山大圳水利公園 | accessdate=2014-12-06}}</ref>

=== 沉砂池 ===

為解決關山大圳淤沙問題，提高大圳的輸水功能，增加灌溉效益，及防範山洪暴發所挾帶的大量砂石流入渠圳幹道，於1989年(民國78年)[[臺灣省|臺灣省]]政府撥款興建沈砂池獲得補助款新臺幣3220萬元，建造[[沈砂池|沈砂池]]及其週邊設備，並於同年5月完工啟用。為加強[[水土保持|水土保持]]及水資源涵養效能，於1999年(民國88年)，由[[經濟部|經濟部]][[水資源局|水資源局]]再撥款補助新臺幣500萬元，整治沈砂池週邊環境。<ref>{{cite web |url=http://www.ttia.gov.tw/page4_kuanshan.php |title=台東農田水利會 關山工作站 | accessdate=2014-12-06}}</ref>

=== 關山大圳五雷鎮水碑及泰山石敢當 === 

* 由當地住民設立<ref name="文化部 文化資產局 關山大圳五雷鎮水碑及泰山石敢當">{{cite web |url=http://www.boch.gov.tw/boch/frontsite/cultureassets/caseBasicInfoAction.do?method=doViewCaseBasicInfo&caseId=VA09602000982&version=1&assetsClassifyId=1.2 |title=文化部 文化資產局 關山大圳五雷鎮水碑及泰山石敢當 |accessdate=2004-12-06 |deadurl=yes |archiveurl=https://web.archive.org/web/20141209151301/http://www.boch.gov.tw/boch/frontsite/cultureassets/caseBasicInfoAction.do?method=doViewCaseBasicInfo&caseId=VA09602000982&version=1&assetsClassifyId=1.2 |archivedate=2014-12-09 }}</ref>

因新武呂溪年年溢漲，導致無法拓地。為防治洪水氾濫，於1923年（日治大正12年）以及1937年（日治昭和12年）分別興建了里壠一號、二號堤防。 而在興建堤防之後，居民擔心潰堤，[[北庄|北庄]]鄉民提議求神庇祐，求符籤、取靈石，並請一道士畫五雷符一符，刻於石碑之上，豎於新武路驛東方50公尺溪岸，以鎮煞制水，是為'''五雷鎮水碑'''。 <ref name="文化部 文化資產局 關山大圳五雷鎮水碑及泰山石敢當" />

鎮水碑立碑時間說法有二，一為1923年（日治大正12年）一號時堤興建後所立；一為一號石堤興建前。據當地居民敘述此碑原立於一號石堤南側的聚落，當時石堤尚未興建，待一號石堤築畢，才遷至堤上，而第三次遷移至現[[海端車站|海端車站]]後方，最後才遷至二號堤上。故此碑因是立於1923年（日治大正12年）以前。 

1968-1973年間（民國57年至62年），因[[颱風|颱風]]多次來襲，而導致堤防潰決，[[德高|德高]]及新福一帶飽受水患，而後村民災後復耕，由一[[新福里_(臺東縣)|新福里]]里長提議特向[[豐泉里|豐泉里]][[代天府五府|代天府五府]]千歲求乞符籤，並致富里鰲溪取[[觀音石|觀音石]]，刻「'''泰山石敢當'''」一座，並於1974年（民國63年）5月[[端午節|端午節]]時連同舊設「五雷鎮水碑」一同遷設至海端二號石堤上。<ref name="文化部 文化資產局 關山大圳五雷鎮水碑及泰山石敢當" />

現今[[北庄社區|北庄社區]]村民每逢農曆7月29日及3月23日[[媽祖|媽祖]]生日時，都會有大規模的祭祀五雷鎮水碑及泰山石敢當的活動。<ref name="文化部 文化資產局 關山大圳五雷鎮水碑及泰山石敢當" />
