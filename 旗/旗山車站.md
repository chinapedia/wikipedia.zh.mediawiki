'''旗山車站'''，位於[[台灣|台灣]]高雄縣旗山鎮（今[[高雄市|高雄市]][[旗山區|旗山區]]），為[[台灣糖業股份有限公司|台灣糖業股份有限公司]][[旗尾線|旗尾線]]之[[鐵路車站|鐵路車站]]。今已廢止。

== 車站構造 ==
*洋風建築

== 利用情況 ==
*停駛後長期閒置，多次遭火災。相關當局一度計畫拆除。

== 車站週邊 ==
*[[旗山老街|旗山老街]]

== 歷史 ==
*明治43年（1910年）8月20日 - 高砂製糖株式會社鐵道線路九曲堂＝旗尾間運輸營業開始。設九曲堂、大樹脚、嶺口、溪洲、'''蕃薯藔'''、旗尾等站<ref>府報第三千三十號，告示第百七號，臺灣總督府</ref>，稱為旗尾線。稍後高砂製糖併入[[鹽水港製糖|鹽水港製糖]]。
*大正9年（1920年）10月1日 - 配合行政區改制，站名亦由蕃薯藔改稱'''旗山'''<ref>府報號外，告示第百七十四號，大正九年十月一日，[[臺灣總督府|臺灣總督府]]</ref>。
*戰後 - 成立[[台灣糖業公司|台灣糖業公司]]，成為其轄事業。
*1973年 - 除九曲堂＝溪埔間10km外，旗尾線餘段停辦客運<ref>民國六十二年度交通年鑑，中華民國交通部</ref>。
*1979年 - 全線廢止客運<ref>民國六十七、六十八年度交通年鑑，中華民國交通部</ref>。
*2005年7月14日 - 指定為縣歷史建築
*2009年7月25日 - 修復完成
*2016年4月1日 - 去年因天花板材塗料老化及風災外牆部分損壞等而暫停開放館舍。一直到開幕前一天修復完成，隔天重新開幕，新名為「糖鐵故事館」，並由高雄市政府文化局承接。<ref>{{cite web|url=http://udn.com/news/story/7314/1586593-%E7%B3%96%E9%90%B5%E6%97%97%E5%B1%B1%E8%BB%8A%E7%AB%99%E5%BE%A9%E6%B4%BB-%E3%80%8C%E8%80%81%E5%A4%A5%E4%BC%B4%E3%80%8D%E5%8A%A9%E9%99%A3 |title=糖鐵旗山車站復活 「老夥伴」助陣 |publisher=[[聯合報|聯合報]] | date=2016年3月25日 |accessdate=2016年4月28日}}</ref>
