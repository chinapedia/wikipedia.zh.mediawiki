'''甘油醛3-磷酸'''（{{lang-en|'''glyceraldehyde 3-phosphate'''}}，縮寫、{{lang|en|'''GADP'''}}、{{lang|en|'''GAP'''}}；或{{lang|en|'''3-phosphoglyceraldehyde''', '''PGAL'''}}）是生物體中的常見分子，是[[糖解作用|糖解作用]]的中間產物之一，以及[[卡爾文循環|卡爾文循環]]的醣類產物。這種分子擁有兩個[[光學異構物|光學異構物]]，在自然界中以D-甘油醛3-磷酸（D-glyceraldehyde 3-phosphate）佔大多數。

== 糖解作用 ==
在糖解作用中，此分子是[[果糖-1,6-雙磷酸|果糖-1,6-雙磷酸]]在[[醛縮酶|醛縮酶]]（aldolase）的催化下裂解產生；之後甘油醛3-磷酸會經由[[甘油醛-3-磷酸去氫酶|甘油醛-3-磷酸去氫酶]]（Glyceraldehyde 3-phosphate dehydrogenase）催化產生[[1,3-雙磷酸甘油酸|1,3-雙磷酸甘油酸]]。

在[[果糖-1,6-雙磷酸|果糖-1,6-雙磷酸]]裂解的過程中其實還有另一個產物，稱為[[二羥丙酮磷酸|二羥丙酮磷酸]]（Dihydroxyacetone phosphate），這種分子本身無法參與糖解作用，但是能夠經由[[三碳糖磷酸異構酶|三碳糖磷酸異構酶]]（Triose phosphate isomerase）的催化，生成甘油醛-3-磷酸。兩者之間的變換是可逆反應，當兩個分子濃度達到平衡時，其中將會有96%是二羥基丙酮磷酸。

甘油醛-3-磷酸也是糖解作用的籌備期（preparatory phase）與收益期（payoff phase）的分界，在甘油醛-3-磷酸之前的一系列反應中，每一分子的葡萄糖需要消耗2個[[三磷酸腺苷|ATP]]，之後則會得到4個ATP。

== 其它生成途徑 ==
除了在糖解作用中產生之外，[[果糖-1-磷酸|果糖-1-磷酸]]（Fructose 1-phosphate）也可經由[[果糖-1-磷酸醛縮酶|果糖-1-磷酸醛縮酶]]（Fructose 1-phosphate aldolase）的催化分成[[二羥丙酮磷酸|二羥丙酮磷酸]]與[[甘油醛|甘油醛]]（Glyceraldehyde），而這兩個分子又可以分別在其他酵素的催化下轉變成甘油醛3-磷酸。
