'''行動聯網裝置'''（'''M'''obile '''I'''nternet '''D'''evice，簡稱'''MID'''，又譯'''移动互联网设备'''）是[[英特尔|英特尔]]在2008年[[IDF|IDF]]大会上提出的一种电脑概念，为英特尔所设计的[[UMPC|UMPC]]类型草图，但英特尔称这个设计是UMPC的进化品。

英特尔将MID定义为一种体积小于[[笔记本电脑|笔记本电脑]]、但大于[[手机|手机]]的移动互联网装置，实际上其概念很接近、甚至可以说等同于[[个人数码助理|个人数码助理]]。

== 主要特徵 ==
* 尺寸-口袋大小
* 螢幕-介於4至6[[吋|吋]]之間，介於行動電話及[[小筆電|小筆電]]間<ref name="News3">{{cite web |url=http://www.ithome.com.tw/itadm/article.php?c=49248 |title=存档副本 |accessdate=2009年6月29日 |deadurl=yes |archiveurl=https://web.archive.org/web/20090703123945/http://www.ithome.com.tw/itadm/article.php?c=49248 |archivedate=2009年7月3日 |df= }} 英特爾MID新平台更小更省電 </ref>
* 價格-200-300美元<ref name="News5">{{cite web |url=http://www.ithome.com.tw/itadm/article.php?c=47894 |title=存档副本 |accessdate=2009年6月29日 |deadurl=yes |archiveurl=https://web.archive.org/web/20090703123939/http://www.ithome.com.tw/itadm/article.php?c=47894 |archivedate=2009年7月3日 |df= }} MIC：MID價格2~3百美元較有可為 </ref>
* 重量-400至800克<ref name="News5">{{cite web |url=http://www.ithome.com.tw/itadm/article.php?c=47894 |title=存档副本 |accessdate=2009年6月29日 |deadurl=yes |archiveurl=https://web.archive.org/web/20090703123939/http://www.ithome.com.tw/itadm/article.php?c=47894 |archivedate=2009年7月3日 |df= }} MIC：MID價格2~3百美元較有可為 </ref>
* 功能-透過內建的3G或Wi-Fi、WiMAX等功能上網，並提供更大的螢幕，讓使用者在網路上使用電子郵件、VoIP、線上音樂、影音內容等服務更容易。<ref name="News5">{{cite web |url=http://www.ithome.com.tw/itadm/article.php?c=47894 |title=存档副本 |accessdate=2009年6月29日 |deadurl=yes |archiveurl=https://web.archive.org/web/20090703123939/http://www.ithome.com.tw/itadm/article.php?c=47894 |archivedate=2009年7月3日 |df= }} MIC：MID價格2~3百美元較有可為 </ref>

== 市場定位 ==
MID的定位不是[[智慧型手機|智慧型手機]]、[[筆記型電腦|筆記型電腦]]或[[小筆電|小筆電]]。<ref name="News4">{{cite web |url=http://www.ithome.com.tw/itadm/article.php?c=55653 |title=存档副本 |accessdate=2009年6月29日 |deadurl=yes |archiveurl=https://web.archive.org/web/20090905084011/http://www.ithome.com.tw/itadm/article.php?c=55653 |archivedate=2009年9月5日 |df= }} 英特爾與Nokia結盟　要開發新型行動運算裝置 </ref>
據市場研究機構的使用者調查顯示由於新的MID皆已嵌入語音功能，有接近一半的受訪者認為MID可取代手機，這使得MID與智慧型手機之間最主要的差別僅在於尺寸。<ref name="News1">{{cite web |url=http://www.ithome.com.tw/itadm/article.php?c=53486 |title=存档副本 |accessdate=2009年6月29日 |deadurl=yes |archiveurl=https://web.archive.org/web/20090703205819/http://www.ithome.com.tw/itadm/article.php?c=53486 |archivedate=2009年7月3日 |df= }} 英特爾及LG將合作開發MID</ref>

== 市場表現 ==
根據市場研究機構的預測，MID出貨量將自2008年30.5萬台成長到2012年的4000萬台，達到120億美元的營收規模。<ref name="News1">{{cite web |url=http://www.ithome.com.tw/itadm/article.php?c=53486 |title=存档副本 |accessdate=2009年6月29日 |deadurl=yes |archiveurl=https://web.archive.org/web/20090703205819/http://www.ithome.com.tw/itadm/article.php?c=53486 |archivedate=2009年7月3日 |df= }} 英特爾及LG將合作開發MID</ref>

== 作業系統 ==
開發初期，一些[[媒体|媒体]]认为MID应该會採用[[Windows_XP|Windows XP]]或[[Windows_Vista|Windows Vista]]其中之一，但部份市場研究機構認為MID作業系統市場將由Linux主導。直至2009年2月英特爾公佈與LG合作後，LG宣稱將以Linux的Moblin 2.0軟體搭載Intel第二代MID平台Moorestown平台，才確認MID將採用Linux。<ref name="News1">{{cite web |url=http://www.ithome.com.tw/itadm/article.php?c=53486 |title=存档副本 |accessdate=2009年6月29日 |deadurl=yes |archiveurl=https://web.archive.org/web/20090703205819/http://www.ithome.com.tw/itadm/article.php?c=53486 |archivedate=2009年7月3日 |df= }} 英特爾及LG將合作開發MID</ref>至2009年英特爾表示，除UMPC使用Windows XP或Vista之外，較小螢幕的MID產品多使用Linux介面。<ref name="News3">{{cite web |url=http://www.ithome.com.tw/itadm/article.php?c=49248 |title=存档副本 |accessdate=2009年6月29日 |deadurl=yes |archiveurl=https://web.archive.org/web/20090703123945/http://www.ithome.com.tw/itadm/article.php?c=49248 |archivedate=2009年7月3日 |df= }} 英特爾MID新平台更小更省電 </ref>

== 發展平台 ==
=== 第一代 ===
Menlow為[[英特爾|英特爾]]第一代[[MID|MID]]平台。

=== 第二代 Moorestown ===
Moorestown為[[英特爾|英特爾]]第二代[[MID|MID]]平台。最大特色在於：
* 採用45奈米製程<ref name="News2">{{cite web |url=http://www.ithome.com.tw/itadm/article.php?c=55340 |title=存档副本 |accessdate=2009年6月29日 |deadurl=yes |archiveurl=https://web.archive.org/web/20130730072524/http://www.ithome.com.tw/itadm/article.php?c=55340 |archivedate=2013年7月30日 |df= }} 英特爾：新MID 平台Moorestown省電50倍 </ref>
* 在代號Lincroft的晶片中整合Atom處理器核心與繪圖功能，
* 待機較前一代的Menlow節省50倍的耗電
* 在代號Langwel I/O晶片，可支援各種I/O群組與無線傳輸解決方案。<ref name="News1">{{cite web |url=http://www.ithome.com.tw/itadm/article.php?c=53486 |title=存档副本 |accessdate=2009年6月29日 |deadurl=yes |archiveurl=https://web.archive.org/web/20090703205819/http://www.ithome.com.tw/itadm/article.php?c=53486 |archivedate=2009年7月3日 |df= }} 英特爾及LG將合作開發MID</ref>
* 將平台晶片組成從三個整合為二個，使MID可以設計的更小。
此平台預計於2010年問世。
