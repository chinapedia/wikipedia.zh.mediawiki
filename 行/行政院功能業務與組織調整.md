'''行政院功能業務與組織調整'''，通稱'''行政院組織改造'''、或簡稱為'''組改'''<ref>[https://web.archive.org/web/20150413124802/http://www.ndc.gov.tw/m1.aspx?sNo=0040001&ex=1&ic=0000015 組織改造主題網]</ref>，是[[中華民國政府|中華民國政府]]自2012年起針對[[行政院|行政院]]及其[[中華民國中央行政機關|所屬部會]]進行的一系列改革措施。

== 概述 ==
'''[[中華民國政府|中華民國政府]]'''自[[民國紀元|民國]]38年（1949年）遷往[[台灣|台灣]]之後，'''[[中央政府|中央]][[行政部門|行政機關]]'''的組織架構仍維持8部2會的建制；但進入1970年代之後，因應施政上的需要，國家最高行政機關[[行政院|行政院]]開始大量增設所屬機關，但開始擴編之後，過多的組織機關反而不利於政務的推行，例如行政院屬下具有部會地位的委員會便高達21個之多。

有鑑於此，1990年代起，行政院就著手規劃進行「'''政府改革工程'''」（又有'''中華民國政府改造'''、'''行政院組織改造'''、'''政府組織再造'''、'''政府再造'''等名稱）<ref>[http://www.dgbas.gov.tw/public/Data/23213572071.pdf 「創新前瞻‧繼往開來—行政院組織改造檔案展」，政府機關資訊通報第 293 期，中華民國 101 年 3 月]</ref>。而[[立法院|立法院]]於2004年6月11日通過《[[:s:中央行政機關組織基準法|中央行政機關組織基準法]]》（2004年6月23日施行）；爾後，立法院於2010年1月12日正式通過「政府組織再造四法」（2010年2月3日施行），做為中央各行政部門進行組織調整的基準法源，並將該改革措施的法定名稱定為「行政院功能業務與組織調整」，由[[公共行政|政府治理]]之主管機關負責執行（[[行政院研究發展考核委員會|行政院研究發展考核委員會]]→[[中華民國國家發展委員會|國家發展委員會]]）。經過22年的討論與規劃，整個組織調整計畫自2012年1月1日起分段實施。截至2018年7月止，共有34個機關{{NoteTag|此統計包含二級和三級機關，並隨行政院組織改造進程而改變。|name=統計}}（分為12部、8會、3委員會、1署、6獨立機關、1行、1院、2總處），最終將調整為33個機關{{NoteTag|name=統計}}（分為14部、8會、7獨立機關、1行、1院、2總處）。目前還有內政部等5部會組改法案未通過。

== 立法過程 ==
=== [[李登輝政府|李登輝政府]]時期 ===
民國77年（1988年）10月，將《行政院組織法修正草案》首度送交[[立法院|立法院]]審查，並曾於民國79年（1990年）元旦二讀；但行政院於該年6月將其撤回。當時的草案重點為：

將8部增加為12部,新設或改名重組的有：
# 衛生福利部
# 農業部
# 文化部
# 勞動部
# 經濟貿易部等5部

將委員會數量減為7會，共設：
# 蒙藏委員會
# 僑務委員會
# 經濟建設委員會
# 國軍退除役官兵輔導委員會
# 原子能委員會
# 國家科學委員會
# 研究發展考核委員會 

並增設2總署及1總處，分別是：
# 新聞總署
# 環境保護總署
# 主計總處

=== [[陳水扁政府|陳水扁政府]]時期 ===
民國93年6月制定《中央行政機關組織基準法》後，隨即擬定《行政院組織法修正草案》。此版本中央政府部門調整為'''十三部、四會和五獨立機關'''。重點為：

13部中有11個新設或更名重組，共有：
# 內政與國土安全部
# 外交及僑務部
# 國防及退伍軍人部
# 教育及體育部
# 經濟貿易部
# 交通及建設部
# 勞動及人力資源部
# 農業部
# 衛生及社會安全部
# 環境資源部
# 文化及觀光部

4會中新設2會：
# 國家發展及科技委員會
# 海洋委員會

5獨立機關中除將央行、中選會和公平會改制外並增設： 
# 金融監督管理委員會
#  國家通訊傳播委員會

但該版本未完成立法。

=== [[馬英九政府|馬英九政府]]時期 ===
行政院於民國97年（2008年）7月21日成立[[行政院組織改造推動小組|行政院組織改造推動小組]]，由[[行政院副院長|行政院副院長]]擔任該小組召集人。並於民國98年（2009年）4月9日通過《行政院組織法修正草案》擬將部會整併為十三部、九會、二總處、二其他機關及三獨立機關<ref>[http://history.ey.gov.tw/Organization/Outer 組織改造]，行政院</ref>。該版本重點為：

13部中新設及更名重組6部，分別是：
# 文化部
# 衛生福利部
# 勞動部
# 交通及建設部
# 農業部
# 環境資源部

委員會中增設國家發展委員會和海洋委員會。

3個獨立機關為：
# 中央選舉委員會
# 國家通訊傳播委員會
# 公平交易委員會

新增的2個總處是：
# 行政院主計總處
# 行政院人事行政總處

民國98年（2009年）10月12日，[[立法院司法及法制委員會|立法院司法及法制委員會]]完成《中央行政機關組織基準法修正草案》及《中央政府機關總員額法草案》初審<ref>{{Cite web |url = http://www.ly.gov.tw/02_introduce/0204_comm/business/businessFolderList.action?comtcd=36&id=4143&itemno=02083600&folderType=R |title = 立法院第7屆第4會期司法及法制委員會第6次全體委員會議議事錄 |author = 立法院 |date = 2010-01-06 |language = zh-tw }}</ref>；同年11月11日，《行政院功能業務與組織調整暫行條例草案》完成初審。<ref>{{Cite web |url= http://www.ly.gov.tw/02_introduce/0204_comm/business/businessFolderList.action?comtcd=36&id=4143&itemno=02083600&folderType=R |title = 立法院第7屆第4會期司法及法制委員會第16次全體委員會議議事錄 |author = 立法院 |date = 2010-01-06 |language = zh-tw }}</ref>同年12月14日《行政院組織法修正草案》完成初審，並展開朝野協商<ref>{{Cite web |url = http://www.ly.gov.tw/02_introduce/0204_comm/business/businessFolderList.action?comtcd=36&id=4143&itemno=02083600&folderType=R |title = 立法院第7屆第4會期司法及法制委員會第27次全體委員會議議事錄 |author = 立法院 |date = 2010-01-06 |language = zh-tw }}</ref>。

民國99年（2010年）1月12日，立法院[[三讀|三讀]]通過《行政院組織法》修正案、《中央行政機關組織基準法》修正案、《中央政府機關總員額法》、《行政院功能業務與組織調整暫行條例》等攸關政府組織再造的4項法案（合稱「政府組織再造四法」），並於同年2月3日經[[中華民國總統|總統]]正式公佈，自此終確立中央行政機關的新架構。行政院將於民國101年（2012年）1月1日起，由原有的37部會分階段整併為29個部會。分成'''14部、8會、3獨立機關、1行、1院、2總處'''，中央政府總員額上限則從現行的19萬200人減至17萬3000人<ref>{{cite news |author=曾韋禎、范正祥／台北報導 |url=http://www.libertytimes.com.tw/2010/new/jan/13/today-fo1.htm |title=政府再造4法通過 2012實施 |publisher=[[自由時報|自由時報]] |date=2010年1月13日 |accessdate=2010年1月16日 |deadurl=yes |archiveurl=https://web.archive.org/web/20100116174247/http://www.libertytimes.com.tw/2010/new/jan/13/today-fo1.htm |archivedate=2010年1月16日 }}</ref>。

2012年1月1日起，陸續啟動新的行政院及中央部會組織架構。原組織架構的11部會，原[[行政院新聞局|行政院新聞局]]、行政院消費者保護委員會、行政院青年輔導委員會、行政院體育委員會、[[行政院經濟建設委員會|行政院經濟建設委員會]]及[[行政院研究發展考核委員會|行政院研究發展考核委員會]]等已遭裁撤或降編；尚有行政院海岸巡防署、[[蒙藏委員會|蒙藏委員會]]、[[行政院公共工程委員會|行政院公共工程委員會]]及[[行政院原子能委員會|行政院原子能委員會]]等待新法源實施後也將裁撤或降編。

2012年10月1日，[[行政院人事行政總處|行政院人事行政總處]]「總處培字第10100518151號函」載明：「因應行政院組織改造，有關行政院所屬機關（構）考試職缺之實際用人機關、工作地點及內容，以分發機關辦理分配作業時為準；另該職缺可能配合行政院組織調整，移撥至他機關<ref>{{Cite web |url = http://wwwc.moex.gov.tw/main/Exam/wFrmPropertyDetail.aspx?m=2091&c=102010 |title = 考選部選高一字第1011400766號公告 |author = 考選部 |date=2012-10-12 |language = zh-tw }}</ref>。」

=== [[蔡英文政府|蔡英文政府]]時期 ===
2016年10月11日[[蔡英文|蔡英文]]總統在執政決策協調會中指出，剩下六部會組改將分成二階段進行，首先將處理陸委會和經濟及能源部組織法和裁撤[[蒙藏委員會|蒙藏委員會]]。<ref>[http://www.cna.com.tw/news/aipl/201610125005-1.aspx 總統開執政協調會推蒙藏委員會裁撤　中央社 2016-10-12]</ref>

2017年9月15日，蒙藏委員會裁撤；12月23日廢止組織法。

2018年4月28日，[[海洋委員會|海洋委員會]]在高雄市成立。原行政院海岸巡防署同時降為該會所屬機關。同年5月3日，行政院會通過重擬的«內政部組織法草案»、«交通及建設部組織法草案»、«經濟及能源部組織法草案»、«環境資源部組織法草案»、«農業部組織法草案»及«核能安全委員會組織法草案»。<ref>[https://www.ey.gov.tw/Page/9277F759E41CCD91/926f7832-73de-48a5-9066-1dca898b6a20  行政院會通過「內政部組織法」修正草案等43項組織法草案 行政院2018-5-3]</ref>
