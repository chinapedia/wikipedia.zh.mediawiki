'''行政院農業委員會家畜衛生試驗所'''（簡稱'''家畜所'''），位於[[臺灣|臺灣]][[新北市|新北市]][[淡水區|淡水區]]，成立於[[台灣日治時期|日治時期]]的1904年，負責臺灣的[[家畜|家畜]]衛生試驗研究工作。

==沿革==
*1904年於[[屏東縣|屏東]]成立「'''牛瘟血清作業所'''」。
*1919年改制為「'''獸疫血清製造所'''」。
*1923年遷至[[臺北市|臺北市]][[樺山町|樺山町]]。
*1931年再遷至臺北縣淡水鎮（今[[新北市|新北市]][[淡水區|淡水區]]）現址。
*1949年改銜為「'''臺灣省農林處獸疫血清製造所'''」。
*1962年改銜為「'''臺灣省家畜衛生試驗所'''」。
*1988年增設「動物用藥品檢定分所」於[[苗栗縣|苗栗縣]][[竹南鎮|竹南鎮]]。
*1999年因[[精省|精省]]而改隸於[[行政院農業委員會|行政院農業委員會]]，更銜為「'''行政院農業委員會家畜衛生試驗所'''」。
*未來，農委會改組為[[中華民國農業部|農業部]]後，將改制為「'''農業部獸醫試驗所'''」<ref>[http://www.rdec.gov.tw/ct.asp?xItem=4531410&ctNode=11573&mp=14 組織改造主題網-農業部獸醫試驗所組織法草案]{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>。

== 組織 ==
=== 所本部 ===
*生物研究組
*豬瘟研究組
*疫學研究組
*製劑研究組
*人事室
*會計室
*秘書室

=== 所屬機關 ===
*動物用藥品檢定分所

==未來工作==
未來該所的工作職掌為:
# 獸醫科技試驗研究計畫與方案規劃及建議。
# 動物疾病防治、檢診技術研究及應用。
# 新興動物傳染病診斷、鑑定、監控及防治研究。
# 獸醫流行病學、獸醫病理生物學及獸醫公共衛生學研究。
# 動物用疫苗、診斷試劑開發研究、製造及供應。
# 獸醫科技之訓練、研習、合作與防疫技術推廣及宣導。
# 動物用藥品檢定技術研發、檢驗服務、訓練及推廣。
# 其他有關事項。
