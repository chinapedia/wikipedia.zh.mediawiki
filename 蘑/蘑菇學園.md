《'''蘑菇學園'''》是香港作家[[狄芬妮琳|狄芬妮琳]]著作，[[瑞雲|瑞雲]]作畫的[[少女漫畫|少女漫畫]]作品，由[[天下出版社|天下出版社]]發行，是繼《[[我的王子殿下|我的王子殿下]]》後另一個瑞雲跟狄芬妮琳合作的[[香港漫畫|香港漫畫]]作品。

本作品在香港及中國內地都有不錯的人氣，被認為是「港台少女漫畫的經典之作」。<ref>{{Cite web |url=http://www.comicyu.com/Html2010/2/2009/34648.html |title=《蘑菇学园》单行本VOL1-2隆重登场！|date=2009-11-25 |work=漫域}}</ref><ref>{{Cite web |url=http://www.it.com.cn/games/dm/zb/2009/12/25/10/706328.html |title=向幸福开跑《蘑菇学园》单行本VOL3-4登场 |work=IT游戏频道 |date=2009-12-25 }}{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>

== 故事簡介 ==
故事是關於校園愛情。<ref>{{Cite web |url=http://comic.people.com.cn/GB/122409/8883237.html |title=4月号上《可爱100》五大法宝 漫画连连看 |date=2009-02-27 |work=动漫频道 |publisher=人民网 }}</ref>小琳平凡的生活將會掀起變化，一切開始都會在私校蘑菇學園。小琳在中四開學禮見到恭介，戀次默默守護小琳。

== 登場人物 ==
;小琳
:本故事的女主角，憧憬[[童話式愛情|童話式愛情]]的平凡女生。小琳雖然內向怕事，但單純及充滿愛心的她卻非常惹人喜愛。
;恭介
:樂壇炙手可熱的[[歌手|歌手]]。外表冷酷，內心卻像小孩般天真可愛。
;戀次
:公認的壞學生，喜愛獨來獨往，跟小琳非常有緣。
;舅父仔
:小琳的舅父仔，在[[意大利|意大利]]深造回來的著名時裝[[設計師|設計師]]Rossi。
;阿猶
:崇拜戀次的小男孩。
;博先生
:恭介的[[經理人|經理人]]，對舅父仔懷有怨恨。
;阿柏
:小琳的同學，個性隨和且友善，是小琳的好友阿嘉的小學同學。
;嘉莎
:與恭介合作[[拍戲|拍戲]]的人氣[[女藝人|女藝人]]。外表溫純可愛，但內裡卻是充滿心機的人。
