'''宫女'''，也称'''宫人'''，狭义上指[[君主|君主]]及其家庭的[[女僕|女僕]]或[[侍女|侍女]]，广义上指君主[[后宫|后宫]]包括[[妃嫔|妃嫔]]的所有女性。較為高級的宮女則稱為[[女官|女官]]。

在不同的国家、朝代，宫女主要的职能是为君主及其家庭的做杂役，以及为皇家提供[[音乐|音乐]]、[[舞蹈|舞蹈]]等娱乐。在君主專制政體中，宫女被视为[[君主妾室|君主的准妾室]]。宫女被君主临幸后，一般会进行[[册封|册封]]，获得妃嫔的[[位号|位号]]。[[中国历史|中国历史]]上，亦有宫女被皇帝临幸后，雖被視為皇帝的妾室，但未受册封為妃嬪的例子，如明朝的[[戴银娘|戴银娘]]、[[胡侍御_(明神宗)|胡氏]]等人。有些更沒有妾室地位，只維持宮中僕役身份。

宫女的来源可能是强制，例如奴隶、战俘、罪犯或其眷属，或[[宮女采選|君主从民间选女]]。也有一些情况下是自愿的，如明代宣宗起秀女多選幼女，養之教之，以禮教陰禮御婢。

== 中国 ==
{{main|中國後宮}}
年華老去的宮女，可以[[退宮|退宮]]。帝王登基時，往往都會把宮人放出民間。在唐代，有些宮女退宮後會進入道觀修道。<ref>李豐楙：《憂與遊：六朝隋唐遊仙詩論集》（臺北：臺灣學生書局，1996），頁325。</ref>
明代洪武朝即欲規範退宮制度，洪武廿二年曾下誥敕令其外有家者，或五載或六載，歸其父母，聽從婚嫁。除新帝登基，常以災異大放宮女，若太長未放宮女，儒臣必以陰陽之理規勸皇帝，以陰氣怨氣不滯留宮中，加上紫禁城空間最多就容納不到3000宮女，放年長宮女進年輕的宮女，少有皇帝不允的。

== 朝鲜 ==
[[李氏朝鮮|李氏朝鮮]]每年从[[兩班|兩班]]、[[中人|中人]]、[[常民|常民]]、[[白丁|白丁]]阶层的女孩中挑选宫女。入宫后为見習的小宫女，小宫女通過考試後成為正式的宫女，稱為[[內人|內人]]，资深宫女通過考核可升为尚宫。最高为[[提调尚宫|提调尚宫]]，掌管一座[[宫殿|宫殿]]的事务。宫女在年老体衰的时候，或者主人去世守丧3年后可以退宫。退宫的宫女不得婚嫁。 

在朝鮮王朝實錄中，可能為了稱呼方便，廣義的將嬪妃與宮女都稱為宮人。

== 土耳其 ==
{{main|宮女 (鄂圖曼帝國)}}
[[奥斯曼帝国|奥斯曼帝国]]的宫女来源为[[女奴|女奴]]或自愿。宫女一般是在后宫做杂役，通常由[[太后|太后]]管辖。非常美丽或有才艺的宫女，可能会被训练之后引见给[[蘇丹_(稱謂)|蘇丹]]，得到苏丹宠幸后，有機會升为[[嫔妃|嫔妃]]。
