'''平內町'''（'''ひらないまち'''）是[[青森縣|青森縣]][[東津輕郡|東津輕郡]]夏泊半島的一[[町|町]]。
扇貝養殖的發祥地。
本地的淺所海岸是除了夏泊半島之外，另一處吸引天鵝飛來的地方。亦擁有很多如夜越山滑雪場等的休閒及觀光施設。

== 町名的由來 ==
* 平內這個名字被認為是來自[[阿伊努語|阿伊努語]]的pira-nay。pira有懸崖的意思，而nay則有河流的意思。合起來就有河流向山及山與山之間流動著這樣的意義。

== 歷史 ==
* 1889年（明治22年）4月1日 - 町村制施行，設立中平內村、東平內村、西平內村。
* 1928年（昭和3年）10月1日 - 中平內村町制施行為小湊町。
* 1955年（昭和30年）3月31日 - 小湊町、東平內村、西平內村合併為'''平內町'''。

== 地理 ==
本町的東西被國道4號所貫穿，東北本線的沿線附近為人口集中的地方。小湊川的下流的周邊被設置町役場。本町的南部主要為山地。
* 河川：小湊川、清水川
