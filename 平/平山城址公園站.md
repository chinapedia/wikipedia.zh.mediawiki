'''平山城址公園站'''（{{jpn|j=平山城址公園駅|hg=ひらやまじょうしこうえんえき|rm=Hirayama-jōshi-kōen eki|f=y}}）是一個位於[[東京都|東京都]][[日野市|日野市]]平山五丁目，屬於[[京王電鐵|京王電鐵]][[京王線|京王線]]的[[鐵路車站|鐵路車站]]。[[車站編號|車站編號]]是'''KO31'''。

== 年表 ==
* 1925年3月24日，作為玉南電氣鐵道的車站啟用，最初名為'''平山站'''。
* 1926年12月1日，玉南電氣鐵道併入京王電氣軌道，平山站成為京王的車站。
* 1944年5月31日，京王與[[東京急行電鐵|東京急行電鐵]]合併後，平山站作為京王線的車站運行。
* 1948年6月1日，東急與京王帝都電鐵分拆，平山站成為京王的車站。
* 1955年9月11日，改稱'''平山城址公園站'''。
* 1976年10月23日，搬至現址。
* 2013年2月22日，以'''KO31'''作為平山城址公園站的編號。

== 車站構造 ==
本站是[[相對式月台|相對式月台]]2面2線[[地面車站|地面車站]]，站房位於一號月台側。1號月台與2號月台之間設有地下道與電梯作為聯繫。

廁所在1號月台。

== 站名由來 ==
開設當時的站名「'''平山'''」是以車站所在地的地名「日野市'''平山'''」命名。當地是{{link-ja|武藏七黨|武蔵七党}}西黨日奉氏一族平山氏的發源地。在平山城址公園完成後改為現名。
