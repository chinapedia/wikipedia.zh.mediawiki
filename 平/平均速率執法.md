'''平均速率執法'''指的是一種測量車輛在固定距離區間的平均速率，以判斷車輛是否超速的執法方式。<ref name = "平均速率執法驚人　萬里隧道每日罰近百車_2018"> {{ cite web | title = 平均速率執法驚人　萬里隧道每日罰近百車 | url = https://tw.appledaily.com/new/realtime/20180724/1397316/ | publisher = 蘋果 日報 | language = zh-tw | date = 出版時間：2018-07-24 }} </ref><ref name = "小心！萬里隧道「平均速率」執法　22天開2140罰單_2018"> {{ cite web | title = 小心！萬里隧道「平均速率」執法　22天開2140罰單 | url = https://www.setn.com/News.aspx?NewsID=407399 | publisher = SETN 三立新聞網 | language = zh-tw | date = 2018-07-24 }} </ref><ref name="鼎文書局編者：廖震2015">{{cite book|author1=鼎文書局|author2=編者：廖震|title=高普特考【交通行政（含概要）】（考點完善整理，試題詳盡解析）: 3版|url=https://books.google.com/books?id=J_tRBgAAQBAJ&pg=PA384|date=5 January 2015|publisher=鼎文書局|isbn=978-957-454-771-5|pages=384–}}</ref>

== 執法方式 ==
在[[台灣|台灣]][[新北市|新北市]]的萬里隧道，警方開始施行一種舉發車輛違規超速的方式，因為隧道內的行車距離固定，執法單位在隧道入口與出口測量車輛進入與離開的時間，再以此時間及距離計算車輛速率，檢查車輛速率是否高於速限，如果平均速率高於速限則算超速違規<ref name="鼎文書局編者：廖震2015" />，但實務上為避免誤差，在計算超速速率時會以高於實際速限的時速作為執法依據。

== 執法成效 ==
2018年7月1日執法到2018年7月22日為止，台灣新北市萬里隧道的速限每小時50公里，但車輛平均速率高於每小時70公里才舉發，以平均速率執法總共開出罰單2,140張，每張罰單1,800元新台幣以上，共開出價值新台幣385萬元罰單；台灣以外的[[英國|英國]]、[[義大利|義大利]]與[[澳大利亞|澳大利亞]]也有執行平均速率執法取締車輛超速違規。<ref name = "萬里隧道 22天開出2140張罰單_2018"> {{ cite web | title = 萬里隧道 22天開出2140張罰單 | url = http://www.chinatimes.com/newspapers/20180724000594-260102 | publisher = 中時電子報 | author = 葉 書宏 | language = zh-tw | date = 2018-07-24 }} </ref><ref name = "注意！平均速率執法　這隧道22天開2140張罰單_2018"> {{ cite web | title = 注意！平均速率執法　這隧道22天開2140張罰單 | url = https://news.tvbs.com.tw/life/960837 | publisher = TVBS 新聞 | author = 張中宜 | language = zh-tw | date = 2018-07-24 }} </ref>
