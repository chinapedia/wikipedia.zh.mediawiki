'''平汝铁路'''自[[宁夏回族自治区|宁夏回族自治区]][[石嘴山市|石嘴山市]][[平罗县|平罗县]][[火车站镇|火车站镇]]至[[汝箕沟|汝箕沟]]，全长82.2[[公里|公里]]，于1971年11月全线通车，素有“太西煤走廊”之称<ref>{{cite web |url=http://www.nxnews.cn/zhuanti/nxzt/tlnx/tlzxl/201101/t20110110_952976.htm |language=zh-hans |title=平汝支线铁路：素有“太西煤走廊”之称 |publisher=宁夏新闻网 |date=2011-01-10 |accessdate=2011-03-23 |archive-url=https://web.archive.org/web/20160310112237/http://nxnews.cn/zhuanti/nxzt/tlnx/tlzxl/201101/t20110110_952976.htm |archive-date=2016-03-10 |dead-url=yes }}</ref>。全线属[[兰州铁路局|兰州铁路局]]银川铁路办事处管辖，并由平汝支线公司负责管理，为电气化铁路，年运送能力330万吨-700万吨。该线虽为货运线路，但开行一对[[银川站|银川站]]至汝箕沟站的普客列车。

==路线概况==
该线自[[包兰铁路|包兰铁路]]平罗站引出，向西北进入大武口[[石炭井矿区|石炭井矿区]]，转正北进入[[贺兰山|贺兰山]]。在[[内蒙古自治区|内蒙古自治区]][[阿拉善左旗|阿拉善左旗]][[呼鲁斯太镇|呼鲁斯太镇]]向西进入贺兰山北段腹地，穿过[[贺兰山隧道|贺兰山隧道]]后转向南到达平罗县[[汝箕沟镇|汝箕沟镇]]。另外各矿区支线铁路在大磴沟站与平汝铁路接轨，矿区支线总长20.1公里。<ref>[[#魏卓等|魏卓等，第188页]]</ref>

线路自起点至11公里处地形平坦，偶有沙丘，此后进入贺兰山前的[[洪积平原|洪积平原]]和有较大起伏的山麓[[洪积裙|洪积裙]]。沿[[大武沟|大武沟]]河谷进入贺兰山区，九跨大武沟至石炭井矿区。<ref>[[#马千里|马千里 (1983)，第129页]]</ref>

== 历史 ==
1959年，为了转用石炭井、白芨沟、大峰沟的煤炭和物资，以大武口站为中心点修筑平汝铁路。全线分为两期施工。一期工程于1959年开工，1961年7月完成开始临时运营。一期工程自[[石嘴山站|石嘴山站]]（原名平罗站）始，止于[[大磴沟站|大磴沟站]]。全长34.7公里，称“朝石支线”，为一级专用线，非平行图区间通过能力为22.5对。<ref>[[#魏卓等|魏卓等，第188页]]</ref>

二期工程于1964年开始，1969年完工。二期工程自大磴沟站延伸至[[汝箕沟|汝箕沟]]，长47.5公里，称“汝箕沟支线”，为二级专用线，非平行图区间通过能力为20.5对。朝石支线与汝箕沟支线并称为平汝铁路（又名平汝支线），于1971年11月通车。<ref>[[#魏卓等|魏卓等，第188页]]</ref>

==事故和灾难==
* 1975年8月14日，贺兰山暴发山洪，平汝线40处被冲毁，中断行车163小时。<ref>{{cite web|url=http://ningxia.mofcom.gov.cn/aarticle/sjdixiansw/200507/20050700182514.html | language=zh-hans | title=宁夏石嘴山市：星海湖的变迁 |publisher=宁夏回族自治区商务之窗 |date=2005-07-20 |accessdate=2010-08-30 }} </ref>
