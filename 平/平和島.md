'''平和島'''（{{jpn|j=平和島|hg=へいわじま|rm=Heiwajima|f=y}}）是[[東京都|東京都]][[大田區|大田區]]的[[人工島|人工島]]與同區町名。現行行政地名為平和島一丁目至平和島六丁目。人口308人，僅有平和島二丁目、平和島五丁目有居民（2013年8月1日）<ref>[http://www.city.ota.tokyo.jp/kuseijoho/suuji/jinkou/setai_jinkou/oota_suji2508.html 大田区ホームページ：平成25年8月1日現在]</ref>。[[填海|填海地]]名為京濱第2區埋立地。現在已與陸地相連。

== 概要 ==
位於[[東京都|東京都]][[大田區|大田區]]東部。北鄰[[品川區|品川區]][[勝島|勝島]]。東鄰大田區[[東海_(大田區)|東海]]（大井埠頭），南鄰大田區[[昭和島|昭和島]]，西鄰大田區[[大森本町|大森本町]]、大田區[[平和之森公園_(大田區)|平和之森公園]]。與東部的大田區東海之間為[[京濱運河|京濱運河]]。

區內大半為[[工業|工業]]用地。[[幹線道路|幹線道路]]聚集，因此吸引[[東京流通中心|東京流通中心]]等各企業物流中心、貨車總站、倉庫、冷藏倉庫等進駐，是具有強烈流通業務性質的區域。[[平和島競艇場|平和島競艇場]]也在本地。

1939年起開始進行填海工程，第二次世界大戰期間曾中斷施工。戰時是[[美國|美國]]等[[同盟國_(第二次世界大戰)|同盟國]][[俘虜|俘虜]]收容地（東京捕虜收容所）。戰時曾作為[[東條英機|東條英機]]等[[戰犯|戰犯]]的暫時收容所。因為有著這段歷史，為祈求和平而命名為「平和島」。昭和40年代以後逐漸成為流通業務重鎮。

== 地名由來 ==
因此地曾是俘虜收容地、戰犯收容所，為祈求和平而命名為「平和島」。

== 年表 ==
* 1939年1月31日 - 京濱第2區埋立地取得許可。事業主體為東京都。
* 1967年
** 7月18日 - 竣工。竣工面積1,176,588 m²
** 9月 - 編入東京都大田區。
** 10月 - 町名命名為「平和島」。
* 1968年
** 4月15日 - 實施[[住居表示|住居表示]]<ref>1968年（昭和43年）7月23日[[自治省|自治省]][[告示|告示]]第158號「住居表示が実施された件」</ref>。
* 1988年4月1日 - 二丁目擴張<ref>1991年（平成3年）2月16日自治省告示第9號「住居表示が実施された件」</ref>。

== 地域內的町名 ==
* 平和島一丁目 - 地域西北部。[[東京都道316號日本橋芝浦大森線|東京都道316號日本橋芝浦大森線]]（[[海岸通_(東京都)|海岸通]]）以西的區域。區內有各企業物流中心與停車場，運河旁有[[平和島競艇場|平和島競艇場]]與[[BIG_FUN平和島|BIG FUN平和島]]。也是平和觀音像所在地。
* 平和島二丁目 - 地域東北部。海岸通以東的區域。區內有[[京濱貨車總站|京濱貨車總站]]、各運送業者事務所、貨物站、物流中心等相關設施。
* 平和島三丁目 - 地域中東部。[[東京都道318號環狀七號線|環七通]]北側、海岸通東側的區域。區內有企業倉庫、物流中心。
* 平和島四丁目 - 地域中西部。環七通北側、海岸通西側的區域。區內有企業物通中心，占據大部分的大田區立[[平和島公園|平和島公園]]、[[東京都下水道局|東京都下水道局]]平和島抽水站、[[京濱急行巴士|京濱急行巴士]][[京濱急行巴士大森營業所|大森營業所]]等。
* 平和島五丁目 - 地域西南部。環七通南側、海岸通西側的區域。區內有各企業物通中心、[[哈多巴士|哈多巴士]]本社與車庫。
* 平和島六丁目 - 地域東南部。環七通南側、海岸通東側的區域。區內有[[東京流通中心|東京流通中心]]與各企業冷藏工廠、冷藏倉庫群。[[東京單軌電車羽田線|東京單軌電車羽田線]][[流通中心站|流通中心站]]也在此地。

== 交通 ==
勝平橋連結北邊的品川區勝島。新平和橋與大和大橋連結東邊的大田區東海（大井埠頭）。南海橋連結南邊的昭和島。都大橋連結西邊的平和之森公園，但現在陸地已相連。

南北向的[[東京都道316號日本橋芝浦大森線|東京都道316號日本橋芝浦大森線]]（海岸通）與平行的[[首都高速1號羽田線|首都高速1號羽田線]]貫穿本地中央，並設有首都高速道路[[平和島出入口|平和島出入口]]與[[平和島PA|平和島服務區]]。此外，也與通過本地中央的東西向[[東京都道318號環狀七號線|環七通]]交會。道路也設計成適合大型車的形態。

鐵路車站方面，本地有[[東京單軌電車羽田線|東京單軌電車羽田線]][[流通中心站|流通中心站]]。另外也可利用步行20分-30分的[[京急本線|京急本線]][[平和島站|平和島站]]（平和島站位於大田區[[大森北|大森北]]）。此外，還可搭乘[[大森站_(東京都)|大森站]]出發的[[京濱急行巴士|京濱急行巴士]]路線。
