[[数学|数学]]中，术语'''平凡'''或'''平凡的'''经常用于结构非常简单的[[范畴论|对象]]（比如[[群|群]]或[[拓扑空间|拓扑空间]]），有時亦會用'''明顯'''或'''乏趣'''這兩個詞代替，但对非数学工作者来说，它们有时可能比其他更复杂的对象更难想象或理解。

例如：
* 明顯[[因數|因數]]：對於每個正整數 n 來說，1、-1、n 和 -n 都是它的明顯因數。
* [[空集|空集]]：不包含任何元素的[[集合_(数学)|集合]]；
* [[平凡群|平凡群]]：只含[[单位元|单位元]]的[[群|群]]；
* {{le|平凡环|Zero ring}}：定义于[[单元素集合|单元素集合]]的环。

==平凡解==
'''平凡'''也用于一个[[方程|方程]]具有非常简单结构的解，但是为了完整性不能省略。这种解称为'''平凡解'''。例如，考虑[[微分方程|微分方程]]

:<math>y'=y</math>
这里 ''y'' = ''f''(''x'') 为[[函数|函数]]，其[[导数|导数]]为 ''y''′。
:''y'' = 0，[[0|0 函数]]是平凡解；
:''y'' (''x'') = e<sup>''x''</sup>，[[指数函数|指数函数]]是一个'''非平凡'''解。

类似地，数学家经常将[[费马大定理|费马大定理]]描述为方程 <math>a^n + b^n = c^n</math> 对 ''n'' > 2 没有非平凡解。
显然，这个方程确实有解。比如 <math>a=b=c=0</math> 对任何 ''n'' 都是解，''a'' = 1, ''b'' = 0, ''c'' = 1 也一样。但是这种解是显然而无趣的，从而称为'''平凡'''。

==數學推理==
'''平凡'''也经常指证明中容易的[[穷举法|情形]]，为了完整性而不能省略。比如，[[数学归纳法|数学归纳法]]证明分为两部分：“奠基步驟”是对一个特殊起始值比如 ''n'' = 0 或 ''n'' = 1 证明定理；然后归纳步骤证明如果定理对特定值 ''n'' 成立，那么对 ''n''+1 也成立。奠基情形经常是显然的。（但是，也有归纳步骤是平凡的而奠基情形却困难的例子。关于多项式的定理经常是这种类型，证明对变元的个数用归纳法。证明如果系数环 ''A'' 是唯一分解整环那么 ''A''[''X''<sub>1</sub>,...,''X''<sub>''n''</sub>] 是[[唯一分解整环|唯一分解整环]]，归纳步骤只要简单的写成 ''A''[''X''<sub>1</sub>,...,''X''<sub>''n''</sub>] = ''A''[''X''<sub>1</sub>,...,''X''<sub>''n''-1</sub>][''X''<sub>''n''</sub>]，而一个变元的奠基情形是困难的。）类似地，我们可能想证明某种性质对一个集合中所有元素都成立。证明的主要考虑非空集合，详细检验其元素是否具有該性質；但如果集合是空集，則性质对其所有元素都成立，因为没有元素需要檢驗。（参见{{le|空洞的事實|Vacuous truth}}）
  
数学界一个常见的笑话是说“平凡”和“被[[证明|证明]]了的”是[[同义词|同义词]]——这就是说，任何定理如果已知成立就可以认为是“平凡”的。另一个笑话是关于两个数学家讨论一个定理。第一个数学家说某个定理是“平凡的”。另一个要求一个解释，然后他进行了 20 分钟的解说。解说完了之后，第二个数学家同意这个定理是平凡的。这个笑话指出对平凡性判断的主观性。举个例子，对[[微积分|微积分]]熟练的人，会认为这个定理

:<math>\int_0^1 x^2\, dx = 1/3</math> 
是平凡的。但对初学者来说，可能一点也不显然。

值得注意的是，平凡性也取决于语境。[[泛函分析|泛函分析]]中的证明可能会给出一个数，平凡地假设存在这样的大数。在[[初等数论|初等数论]]中证明自然数的基本结论时，证明也许會與「每個自然数都有一个后继」息息相關，但此點需加以證明，或者将其作为一个[[皮亚诺公理|公理]]。
