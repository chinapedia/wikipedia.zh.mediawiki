'''平野美宇'''（{{Lang-ja|平野 美宇}}，{{bd|2000年|4月14日|c=Hirano, Miu}}），一名[[日本|日本]]女子[[乒乓球|乒乓球]]運動員<ref>{{cite web|url=http://www.ittf.com/ittf_stats/stats_per_player.asp?player_id=117820|title=Stats Records - ITTF Statistics|deadurl=yes|archiveurl=https://web.archive.org/web/20140407130840/http://www.ittf.com/ittf_stats/stats_per_player.asp?player_id=117820|archivedate=2014-04-07}}</ref><ref>{{cite web|url=http://www.ittf.com/ittf_ranking/world_ranking_per_name.asp?Siniors=1&Player_ID=117820|title=World ranking Record for HIRANO Miu (JPN)|deadurl=yes|archiveurl=https://web.archive.org/web/20141006135201/http://www.ittf.com/ittf_ranking/world_ranking_per_name.asp?Siniors=1&Player_ID=117820|archivedate=2014-10-06}}</ref><ref>{{cite web |url=http://www.ittfranking.com/cgi-bin/ttplayer.cgi?number=117820&lang=0&format=detailed|title=HIRANO Miu (JPN) - ITTF Rankings}}</ref>，目前效力於JOC學院俱樂部。她出生於[[靜岡縣|靜岡縣]][[沼津市|沼津市]]，但在[[山梨縣|山梨縣]][[中央市|中央市]]長大。2016年她以16歲之齡成為史上最年輕的[[世界杯乒乓球賽|世界杯乒乓球賽]]女子單打冠軍，同時亦為首位非中國籍得主。

== 生平 ==
她的家人幾乎全部都和桌球運動有關<ref>{{cite web|url=http://midasjunior.sblo.jp/archives/20101010-1.html
 |title=「ニックのJr卓球練習日記」 |author= |date=2009-01-30 |accessdate=2010-01-23}}</ref>。父母是前[[筑波大學|筑波大學]]乒乓球隊的隊長<ref name="sponichi"/>，兩個妹妹也都參與了相關賽事。祖父為[[山梨大學|山梨大學]]名譽教授[[平野光昭|平野光昭]]，他也曾擔[[山梨大學|山梨醫科大學]]<ref>已合併成[[山梨大學|山梨大學]]。</ref>的乒乓球隊顧問一職<ref>{{cite web|url=http://air.ap.teacup.com/awatenai/881.html |title=「天才卓球少女『平野美宇ちゃん』のおじいさん」 |author=平方眞 |date=2009-01-30 |accessdate=2010-01-23}}</ref>。

她從兩歲開始居住於[[山梨縣|山梨縣]][[中央市|中央市]]，小學進入[[中央市立田富北小學校|中央市立田富北小學校]]就讀<ref name="sponichi">卓球：愛ちゃん超え!8歳美宇ちゃん最年少勝利 スポーツニッポン 2009年1月15日</ref>。由於從小就擁有超齡的乒乓球球技，加上輸球後時常因不甘心而哭泣等特點與[[福原愛|福原愛]]相似，而被譽為「小愛二世」。

2012年，以11歲的年齡首次參加{{link-ja|日本Top 12乒乓球大會|ジャパントップ12卓球大会}}<ref>{{cite web |url=http://www.sankei.jp.msn.com/sports/news/120127/oth12012721080026-n1.htm |title=11歳平野と伊藤が初出場 卓球ジャパン・トップ12 |publisher=MSN産経ニュース |date=2012-01-27 |accessdate=2012-03-31 |deadurl=yes |archiveurl=https://web.archive.org/web/20120130040905/http://sankei.jp.msn.com/sports/news/120127/oth12012721080026-n1.htm |archivedate=2012-01-30 }}</ref>。

2014年3月30日，她和[[伊藤美誠|伊藤美誠]]兩人以十三歲的年齡在[[國際桌總職業巡迴賽|世界巡迴賽]]德國公開賽獲得女雙冠軍，成為了世界巡迴賽史上第三年輕的冠軍<ref>平野奪冠時的年齡為13歲又350天。[[伊藤美誠|伊藤美誠]]則是13歲又160天，創下最年輕紀錄。</ref>。

緊接著在下週的西班牙公開賽，她不但與伊藤的組合再度奪冠，單打方面也闖入決賽，最後不敵[[李芬_(乒乓球运动员)|李芬]]獲得亞軍。

2016年，她首次參加[[世界盃乒乓球賽|世界盃乒乓球賽]]。在中國選手[[丁寧|丁寧]]和[[劉詩雯|劉詩雯]]皆因傷病退賽的情况下，平野美宇先後淘汰了隊友[[伊藤美誠|伊藤美誠]]及新加坡名將[[馮天薇|馮天薇]]進入決賽，在決賽4：0擊敗了中華台北的[[鄭怡靜|鄭怡靜]]，成為史上最年輕的世界單打冠軍，也打破了中國選手一直壟斷[[世界盃乒乓球賽|世界盃]]女單冠軍的局面。

2017年，平野美宇出戰在[[無錫|無錫]]舉行的[[亞洲乒乓球錦標賽|亞洲乒乓球錦標賽]]。在中國隊佔盡主場之利的情况下，她接連擊敗了[[丁寧|丁寧]]、[[朱雨玲|朱雨玲]]和[[陳夢|陳夢]]三位中國選手（後兩者更是以3:0橫掃），摘得女單桂冠，震驚乒壇，使其成為對中國隊最大威脅的球員。

亞錦賽之後，平野美宇出戰在[[德國|德國]]舉行的[[世界乒乓球錦標賽|世乒賽]]，為賽會的女單八號種子，被編入了唯一没有中國選手的四強賽區。一路順利晉級，在八強賽4-0橫掃馮天薇殺入四強。半決賽遭丁寧以4-1復仇，獲得銅牌，是自1969年以來日本在世乒賽的首面女單奬牌，意義重大。女雙方面，則夥拍石川佳純，在十六強止步於丁寧和劉詩雯的拍下。
