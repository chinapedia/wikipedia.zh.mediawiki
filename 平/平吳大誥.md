《'''平吳大誥'''》{{NoteTag|以往，越南人亦稱中國的明朝為「吳」，如20世紀初的漢文歷史小說《[[後陳逸史|後陳逸史]]》（[[潘珮珠|潘珮珠]]撰）中有謂：「我國[[越南陳朝|陳朝]]之季，閏[[胡朝|胡]]失正，'''吳'''賊蟠蜛我國，十有餘年。」見潘珮珠《後陳逸史》，收錄於《越南漢文小說叢刊》第二輯第三冊《歷史小說類》，臺灣學生書局校點本第293頁。}}（{{lang-vi|'''''Bình Ngô đại cáo'''''}}），[[越南|越南]]古代的重要政治文告，作者是[[越南|越南]][[後黎朝|後黎朝]]的開國功臣兼文人[[阮廌|阮廌]]，以[[漢語|漢語]][[文言文|文言文]]寫成。該文是在[[明朝|明朝]][[安南屬明時期|佔領越南]]結束時，抗明勢力中的[[黎利|黎利]]已擊敗明軍，取得勝利，為宣示[[大越|大越國]]（[[越南|越南]]）脫離明朝統治、建立後黎朝的統治而寫的。《平吳大誥》全文貫穿著民族情感及去暴安民的思想，既在政治上具有重要含義，亦在文學上有相當成就。在越南後世，《平吳大誥》有「千古雄文」、「第二個[[越南独立宣言|獨立宣言]]」等稱譽。

== 寫作背景 ==
[[文件:Nguyen_Trai.jpg|thumb]]]]
《'''平吳大誥'''》的寫成，與15世紀初期的[[越南|越南]]政局有關。1407年，[[中國|中國]][[明朝|明朝]]率軍南下，攻滅越南的[[胡朝|胡朝]]。明軍將領[[張輔|張輔]]發表[[:wikisource:zh:平南露布|平南露布]]<ref>嚴從簡《殊域周咨錄·安南》，北京中華書局點校本，183─185頁。</ref>、[[明成祖|明成祖]]皇帝頒下[[:wikisource:zh:平安南詔|詔書]]<ref>《明實錄·太宗實錄》卷六十八，永樂五年六月癸未條，茲參考李國祥主編《明實錄類纂·涉外史料卷》，武漢出版社版，597─599頁。</ref>，宣稱已佔據越南，並要進行[[安南屬明時期|直轄統治]]。其後，越人的反抗行動此起彼落，特別是來自[[藍山鄉|藍山]]（在今[[清化省|清化省]]）的[[黎利|黎利]]（即[[後黎太祖|後黎太祖]]）最具影響力，並於1427年擊退明軍。

當明軍在1428年[[農曆|農曆]]十二月十七日（[[西曆|西曆]]1月3日）<ref>{{中西曆轉換}}</ref>撤退後，黎利便發表由阮廌所寫的《平吳大誥》，宣佈[[大越國|大越國]]（越南）在黎利的領導下，脫離明朝的統治而獨立。<ref>吳士連等《大越史記全書·本紀實錄·黎紀·太祖高皇帝》，東京大學東洋文化硏究所校合本，546頁；越南社會科學委員會《越南歷史》第二部份第六章，北京人民出版社中譯本，293頁。</ref>

== 內容概述 ==
《'''平吳大誥'''》的內容，主要為突出[[黎利|黎利]]軍隊與[[中國|中國]][[明朝|明朝]]之間的戰爭，以及黎利帶領[[大越|大越國]]獨立的合理性。在全文一開始，就提出「仁義之舉，要在安民，吊伐之師，莫先去暴」的除暴安民思想。繼而提出「惟我大越之國，實為文獻之邦，山川之封域既殊，南北之風俗亦異」，以彰顯越人獨立行動的合理性。

隨後，文章大部份的篇幅均述及中越間的交戰史，先略述[[吳權|吳權]]擊敗[[南漢|南漢]]、[[宋越熙寧戰爭|李宋之戰]]、[[蒙越戰爭|蒙越戰爭]]當中越人的戰績，繼而提到明朝佔領大越國後「連兵結釁」的亂亡時期，歷數明人在當地所作的劣行。不過後來[[黎利|黎利]]「[[藍山起義|奮跡藍山]]」，積極進行抗明行動，「發憤忘食，每研談韜略之書，即古驗今，細推究興亡之理，圖回之志，寤寐不忘」，經過一段拼搏後，黎利勢力先後於[[崒洞之役|崒洞之役]]（崒洞又作𡨧洞）及[[支棱之役|支棱之役]]裡取得決定性的軍事勝利。結果，明朝政府決定撤兵，黎利勢力亦願意與明軍「修好有誠」。

最後，文中提到黎利勢力為大越國爭取到獨立復國，實為功勞殊鉅，「社稷以之尊安，山川以之改觀，乾坤既否而復泰，日月既晦而復明。于以開萬世太平之基，于以雪千古無窮之耻」，所以要在這個歷史時刻，「誕布維新之誥，布告遐邇，咸使聞知」。<ref>吳士連等《大越史記全書·本紀實錄·黎紀·太祖高皇帝》，東京大學東洋文化硏究所校合本，546─548頁。</ref>

== 寫作特點 ==
據[[中國|中國]]學者[[王曉平|王曉平]]的分析，本文有以下的特別之處。
*吸收了中國古代辭賦的簡潔寫法：例如[[北宋|北宋]][[蘇洵|蘇洵]]的《[[六國論|六國論]]》中，以[[秦國|秦國]]為一方，以六國（[[田齊|齊]]、[[楚國|楚]]、[[燕國|燕]]、[[韓國_(戰國)|韓]]、[[趙國|趙]]、[[魏國_(戰國)|魏]]）為一方，展現出[[戰國時代|戰國時代]]錯綜複雜的形勢，《平吳大誥》亦採用此一技巧，把[[明軍|明軍]]及[[黎利|黎利]]軍之間歷時十年的戰事，以文約事豐的方式記述出來。但同時，《平吳大誥》對各場重要戰役記述得十分具體，一一點明日期，既能通報戰況，又營造出捷報頻傳的效果。
*注入個人的情感體驗：阮廌參與了黎利的抗明起義，又為黎利代寫《平吳大誥》，他在文中透過帶有君王口吻的「予」（指黎利），帶出其對[[十年抗明戰爭|抗明戰爭]]的情感體驗。例如文中提到「予奮跡藍山，棲身荒野，念世讐豈可共戴，誓逆賊難與俱生。痛心疾首者垂十餘年，嘗膽臥薪者蓋非一日。發憤忘食，每研談韜略之書，即古驗今，細推究興亡之理，圖回之志，寤寐不忘」，便是以作者本人的思想感情為基礎而下筆，因而讀來真實感人。<ref>王曉平《亞洲漢文學‧歌詩之橋‧越南黎朝漢文學的鼎盛》，天津人民出版社版，100─103頁。</ref>

== 後世評價 ==
[[越南|越南]]後世對《'''平吳大誥'''》有相當高的評價。1970年代，[[越共|越共]]學者提出該文是越南民族的「第二個獨立宣言」<ref>越南社會科學委員會《越南歷史》第二部份第六章，北京人民出版社中譯本，296頁。</ref>，是「一首舉世無雙的英雄歌，是一幅天才的、生動的圖畫，它忠實地描繪出一個正在奮起的、決心克服一切艱難險阻的、並為我國（指越南）歷歷史寫下最光輝一頁的英雄民族的形象。這部作品還概括地總結了歷史和人民群眾的創造力以及在抗戰中得到高度發揚的民族的高貴品質，提高了我國人民的獨立自由的意志、英勇不屈的傳統和人道主義的精神。」<ref>越南社會科學委員會《越南歷史》第二部份第六章，北京人民出版社中譯本，302頁。</ref>

在[[中國|中國]]，《平吳大誥》曾遭到批評，如[[清|清]]人[[徐延旭|徐延旭]]在其著作《[[越南輯輅|越南輯輅]]》裡提到[[黎利|黎利]]勢力建政後「於其國作《平吳大誥》，宣示其民，語尤悖逆」。<ref>徐延旭《越南輯略·世系沿革》，清光緒三年（1877年）刻本，12頁。</ref>到現代，學術界出現肯定《平吳大誥》重要性的論調，例如學者[[郁龍余|郁龍余]]、[[孟昭毅|孟昭毅]]等的《東方文學史》中，提到該文的意義：「《平吳大誥》是他（[[阮廌|阮廌]]）代表[[黎利|黎利]]寫的佈告越南全國百姓的開國文獻，有很高的藝術價值和文獻價值，被譽為『千古雄文』」。<ref>郁龍余、孟昭毅主編《東方文學史》第八章《中古越南文學》，北京大學出版社版，240頁。</ref>
