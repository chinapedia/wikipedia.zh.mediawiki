'''平手政秀'''（{{bd|1492年|6月4日|1553年|2月25日}}）是[[日本戰國時代|日本戰國時代]]的武將。仕於[[織田信秀|織田信秀]]、[[織田信長|信長]]兩代，[[尾張國|尾張國]][[春日井郡|春日井郡]][[志賀城|志賀城]]城主。幼名'''狛千代丸'''。初名是'''清秀'''。別名'''五郎左衛門'''。官位是[[監物|監物]]、[[中務丞|中務丞]]。

追溯到[[平手氏|平手氏]]的起源，與[[三河|三河]][[松平氏|松平氏]]有共同的先祖，屬於清和[[源氏|源氏]][[新田氏|新田氏]]後裔的一支。

== 生平 ==
在信秀時代已經是重臣，主要負責財政和外交，亦擅長[[和歌|和歌]]、[[茶道|茶道]]，受到在[[天文_(後奈良天皇)|天文]]2年（1533年）到訪尾張國的[[山科言繼|山科言繼]]讚賞。在天文12年（1543年）5月代表信秀[[上洛|上洛]]，為朝廷付出4千貫修補[[御所|御所]]的費用，負責與朝廷的交涉活動。

[[天文|天文]]3年（1534年），在信長出生的同時成為其傅役，擔任次席[[家老|家老]]。天文16年（1547年），在信長的[[初陣|初陣]]中擔當輔助角色。為了強化信秀一族在尾張[[織田氏|織田氏]]中的競爭優勢及地位，在翌年（1548年）協助主家與[[美濃國|美濃]]的[[齋藤道三|齋藤道三]]結盟，使信長與道三之女[[濃姬|濃姬]]締結婚姻。據說當時雙方各懷心機，道三賭上了自己的愛女，希望能藉由此樁婚姻實現奪取尾張一國的野心；而政秀則看準擁有道三這名岳父的撐腰，信長即能坐穩主家繼承人的位子，並藉此平息家中因信長荒誕行為而醞釀的反對聲音。

在[[三河|三河]]的[[安城合戰|安城合戰]]中率領援軍救援[[織田信廣|織田信廣]]。

政秀對於被說成傻瓜、年輕的信長的荒誕行為始終感到憂心及憤慨，時常對信長勸諫，在信秀死去以及織田家中不穩的情況下，於天文22年（1553年）閏1月13日[[切腹|切腹]]自盡。享年62歲。戒名是'''政秀寺殿功菴宗忠大居士'''。

== 切腹的原因 ==
政秀切腹的理由有以下多種說法：
*為了勸諫信長的奇行，這是有名的美談。
*因為信長的行為沒有改變，於是很自責。
*拒絕獻上平手五郎右衛門的駿馬。
*信長放棄在[[萬松寺|萬松寺]]的葬禮。
*與其他重臣的權力鬥爭。與推舉信長之弟（[[織田信行|信行]]）為[[家督|家督]]繼承者的[[林秀貞|林秀貞]]、[[林美作守|通具]]兄弟和信行的[[後見|後見]]人[[柴田勝家|柴田勝家]]對立。

== 死後 ==
政秀死後，信長的行為仍然沒有改變，但是信長在政秀死後命[[澤彦宗恩|澤彦宗恩]]在[[春日井郡|春日井郡]][[小木村|小木村]]建立[[政秀寺|政秀寺]]來記念政秀。

政秀家中在『[[信長公記|信長公記]]』中記載有3個兒子（五郎右衛門、監物、甚左衛門），但族譜中則只有兒子[[平手久秀|平手久秀]]、孫兒[[平手汎秀|平手汎秀]]。至於『信長公記』中列出的3人是誰則有不同的見解。族譜中位置不明的[[平手長政|平手長政]]（孫右衛門）應該是長男五郎右衛門，五郎右衛門是政秀之弟[[平手政利|政利]]的兒子，後來被政秀收為養子。

女兒（雲仙院）成為信長之弟[[織田長益|織田長益]]（有樂齋）的正室。
