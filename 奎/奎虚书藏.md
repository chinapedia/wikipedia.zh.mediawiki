'''奎虚书藏'''位于[[中国|中国]][[山东省|山东省]][[济南市|济南市]][[历下区|历下区]]大明湖路257号，北临大明湖，东临遐园，为原'''山东省立图书馆'''藏书楼，建于1935-1936年，为现代建筑风格。

== 历史 ==

山东省图书馆的前身为山东省立图书馆，馆址位于[[大明湖|大明湖]]西南岸，落成于1909年12月，建筑布局仿浙江宁波的藏书楼[[天一阁|天一阁]]，当时被称为遐园，主要建筑包括海岳楼、宏雅阁等<ref>{{cite web |url=http://www.sdwht.gov.cn/whjg/12.jsp|title=《机构设置-山东省图书馆》|accessdate=2009-12-12|work=|publisher=山东省文化厅网}}</ref>。

1928年日本以保护日侨为由出兵济南，山东省立图书馆因与中国国民党山东省党部接壤而受到战火波及，大部分建筑被毁，原藏书楼楼顶被洞穿，基础亦遭到震动破坏。1929年后，藏书量亦多至原藏书楼不能容纳<ref name="multiple1">{{cite book |author=王献唐 |title=《季刊奎虚书藏落成纪念专集抽印本-奎虚书藏营建始末记》|year=1936 |publisher=山东省立图书馆 |location=济南 |isbn=}}</ref>。因而于1934年筹建新藏书楼，命名为“奎虚书藏”，其意为“奎星主齐，虚星主鲁，以二星之分野，括齐鲁之疆域”<ref>{{cite web |url=http://www.gmw.cn/01gmrb/2003-04/02/18-6A1975B9A788083D48256CFB007FB645.htm|title=《遐园 奎虚书藏》|accessdate=2009-12-12|work=韦力|publisher=光明网}}</ref>。

在1984年新阅览楼建成前，奎虚书藏一直为该馆的主体建筑，长期用于图书的采编、收藏和借阅，其藏书包括善本古籍、普通古籍、[[海源阁|海源阁]]藏书及其故物等（现多迁至二环东路的山东省图书馆新馆）<ref>{{cite web|url=http://www.cpin.com.cn/NewsCenter/NewsView.aspx?NewsID=11562|title=《开民智 保国粹》|accessdate=2009-12-12|work=|publisher=新华书目报}}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。2004年，该建筑曾被整修维护，其功能也由藏书楼向自习室、阅览室转换<ref name="multiple2">{{cite web |url=http://www.dzwww.com/xinwen/xinwenzhuanti/jnby/zxbd/200703/t20070304_2029147.htm|title=《奎虚书藏 分野齐鲁》|accessdate=2009-12-12|work=任鹏、杨黎明|publisher=大众网}}</ref>。

另外，该建筑在1945年曾作为第十一战区副司令长官司令部大礼堂使用，并在此举行了侵华日军山东战区签降仪式，当时的战区副司令长官[[李延年|李延年]]曾题有“我武维扬”匾以纪念这一历史事件<ref name="multiple2" />。

== 建筑 ==

建筑坐西面东，平面呈山字形，楼后使用回廊相连，钢筋混凝土结构。建筑分为两层（当时亦考虑便于随时增筑），上层最初为书库六间，书架为木质，下层最初为阅览室、展览室等共十六间，正中的大阅览室可容纳400人<ref name="multiple1" />。东立面呈五段式划分，稍突出南、北两端和中间部分，中部[[女儿墙|女儿墙]]为叠落状[[马头墙|马头墙]]处理，正中为由[[傅增湘|傅增湘]]题写的“奎虚书藏”四字<ref>{{cite web|url=http://travel.e23.cn/Content/2009-09-22/200992200432.html|title=《大明湖新景区八大景观亮相》|accessdate=2009-12-12|work=|publisher=舜网-济南日报|deadurl=yes|archiveurl=https://web.archive.org/web/20100521061438/http://travel.e23.cn/Content/2009-09-22/200992200432.html|archivedate=2010-05-21}}</ref>，南、北两端的女儿墙亦采用相似的手法，与中间部分取得了呼应<ref name="multiple2" />。
