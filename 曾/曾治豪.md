'''曾治豪'''（Gary，{{bd|1986年|2月8日}}），[[歌手|歌手]]與詞／曲創作人，常態在音樂餐廳駐唱。2009年首度參加大型歌唱選秀節目[[超級星光大道_(第六屆)|第六屆《超級星光大道》]]獲得第十名的成績，為[[星光六班|星光六班]]成員之一。

== 演藝歷程 ==
曾治豪受到同為駐唱歌手的母親影響，從小就喜歡唱歌，並嚮往進入演藝圈從事自己最熱愛的表演，但知名的資深藝人父親[[賀一航|賀一航]]（本名曾新民）<ref>{{cite news |language =zh | url = https://star.ettoday.net/news/1160914 | title = 除夕夜才能看到賀一航　兒子曾治豪：只知道他叫做爸爸 | publisher = ETtoday星光雲 | date = 2018-05-01}}</ref>
認為演藝圈不能算是一份穩定的工作，並不贊成兒子踏進這行，曾治豪於是私下偷偷報名[[超級星光大道_(第六屆)|第六屆《超級星光大道》]]歌唱比賽，一路過關斬將，直至擠進賽程前10強決定賽，父親看到其毅力和堅持，才逐漸轉變態度支持，更主動報名參加節目，與曾治豪進行踢館PK賽，以行動激勵其演藝之路和表達父愛（見[[超級星光大道_(第六屆)|第六屆《超級星光大道》]]第21集)。

曾治豪鮮少對外透露父親是知名藝人[[賀一航|賀一航]]，甚至首次參加2009年[[超級星光大道_(第六屆)|第六屆《超級星光大道》]]大型歌唱比賽時，也未在個人背景資料上提及<ref>{{cite web |publisher= [[Yahoo!奇摩|Yahoo!奇摩]]|url=https://tw.celebrity.yahoo.com/news/%E6%98%8E%E6%98%9F%E6%98%9F%E4%BA%8C%E4%BB%A3-%E6%9B%BE%E6%B2%BB%E8%B1%AA%E4%B8%8D%E7%95%B6%E9%9D%A0%E7%88%B8%E6%97%8F-%E9%9D%A0%E6%AF%94%E8%B3%BD-%E5%9C%93%E6%98%9F%E5%A4%A2-081629197.html | language=| title=明星星二代／曾治豪不當靠爸族 靠比賽一圓星夢 |date=2013-11-01}}</ref>，反倒是由擔任首集評審的[[康康|康康]]（康晉榮）在節目中公開兩人父子關係。曾治豪低調而不願依仗父蔭的性格，使得父親[[賀一航|賀一航]]也經常於受訪時表示，不會主動幫曾治豪介紹工作或爭取表演舞台<ref>{{cite web |publisher= [[蘋果日報|蘋果日報]]|url=http://www.appledaily.com.tw/realtimenews/article/new/20140801/443720/ | language=| title=我兒不靠爸！ 賀一航拒兒同台 |date=2014-08-01}}</ref>。

起初，[[賀一航|賀一航]]並不看好兒子曾治豪的音樂夢想，經過多年靜默和觀察，發覺曾治豪仍專注且持續在演藝圈尋求表現機會，且有不錯的成績，對此感到欣慰，開始逐漸展現行動上的支持<ref>{{YouTube|YfNVOcWaIc8| 20131102曾治豪＆賀一航@萬秀豬王}}</ref>，2013年再度參加[[超級歌喉讚|超級歌喉讚]]節目擔任關主，與曾治豪同台對戰，在光榮戰敗後對曾治豪表示：爸爸能做的就這麼多了，加油！（見[[超級歌喉讚|超級歌喉讚]]第19集）。經擔任節目評審的亞洲電台DJ咖啡糖賢齡揭露，曾在搭捷運時，巧遇西裝筆挺剛面試完工作的曾治豪才得知，除了晚上的音樂餐廳駐唱之餘，曾治豪仍想在白天找份正職，多賺點錢以支持自己的夢想。曾治豪自己則表示，堅持想走的路，多付出一些是應該的，此舉低調地沒有讓家人知道<ref>{{cite web| publisher=Yes娛樂| url=http://tw.twent.chinayes.com/Content/20130802/kh9ljjiarsjx4.shtml| language=| title=戰場無父子 曾治豪挑戰「秀場天王」賀一航| date=2013-08-02| deadurl=yes| archiveurl=https://web.archive.org/web/20150705150818/http://tw.twent.chinayes.com/Content/20130802/kh9ljjiarsjx4.shtml| archivedate=2015-07-05}}</ref>。

[[賀一航|賀一航]]在接受媒體採訪時，曾表示若曾治豪再沒有一點成績，希望他改走幕後<ref>{{cite web |publisher= [[自由時報|自由時報]]|url=http://ent.ltn.com.tw/news/breakingnews/1311029 | language=| title=賀一航下最後通牒 曾治豪別肖想靠爸|date=2015-05-08}}</ref>，曾治豪除一邊尋求演出機會，也一邊感受到壓力，當台灣最高收視率的綜藝節目[[中視|中視]]《[[綜藝玩很大|綜藝玩很大]]》邀請十多年不再接外景工作的[[賀一航|賀一航]]與曾治豪至[[馬來西亞|馬來西亞]][[檳城|檳城]]參加通告時，[[賀一航|賀一航]]看著兒子曾治豪確實在演藝這條路上極為認真努力而答應陪同，曾治豪在錄影過程中則是盡全力配合、拚命表現<ref>{{cite web|publisher=[[中國報|中國報]]|url=http://www.chinapress.com.my/node/632937|language=|title=《綜藝玩很大》吳宗憲、曾治豪撞車|date=2015-06-25|deadurl=yes|archiveurl=https://web.archive.org/web/20150804005041/http://www.chinapress.com.my/node/632937|archivedate=2015-08-04}}</ref><ref>{{cite web |publisher= [[星洲日報|星洲日報]]|url=http://mykampung.sinchew.com.my/node/367839 | language=| title=有人隱形眼鏡掉落　甚至乾嘔…憲哥《綜藝玩很大》太拼命！|date=2015-06-27}}</ref>，主持人[[吳宗憲|吳宗憲]]更表示，[[賀一航|賀一航]]與曾治豪兩父子在玩遊戲時，父子情深的一面令人感動<ref>{{cite web|publisher=[[時華資訊|時華資訊]]|url=http://www.news.seehua.com/archives/68926|language=|title=難忘乘船8小時《綜藝玩很大》憲哥來檳開咖啡館|date=2015-06-27|deadurl=yes|archiveurl=https://web.archive.org/web/20150722003630/http://www.news.seehua.com/archives/68926|archivedate=2015-07-22}}</ref>。<br><br>

*2008年初試啼聲，與[[于立成|于立成]]（大山）合唱《大人小孩》單曲，收錄於[[于立成|于立成]]（大山）《愛情片段》絕版經典專輯。
*2009年是[[超級星光大道_(第六屆)|第六屆《超級星光大道》]]參賽者之一，最終獲得第十名的佳績。
*2010年參加選秀節目《[[拉芳星光大會|拉芳星光大會]]》，名次為第九名。
*2011年開始嘗試在網路上發表個人音樂作品，全創作單曲《小朋友》為2013【橙心做好事－香橙檸檬蛋糕】公益活動主題曲正式發表。
*2012年擔當尚和歌仔戲團《半人》台味歌仔戲舞台劇男主角，初次嘗試舞台劇表演。
*2013年參加[[超級歌喉讚|超級歌喉讚]]，因首次參賽表現優異，之後受邀為關主身份。
*2014年8月起，參與[[東森幼幼台|東森幼幼台]]《[[萌學園六復活之戰|萌學園六復活之戰]]》演出，就此在演藝圈展開戲劇之路。
*2015年7月底，加入了三立台灣台 《[[甘味人生|甘味人生]]》劇組，創造出個人多元的戲劇風格，獲得好評。

== 駐唱 ==
曾治豪同時身兼[[歌手|歌手]]、[[演員|演員]]、[[主持人|主持人]]的多重身份，在音樂餐廳駐唱時，除了演唱歌曲之外，也擅長炒熱現場氣氛與賓客互動，更因[[作詞人|作詞人]]、[[作曲人|作曲人]]的創作音樂人背景，曾治豪在表演當中，經常帶來即性與爆發力的演出。在[[台灣|台灣]]曾駐唱的音樂餐廳：Amigo Live House、China Pa、三四味屋、大牛西餐廳、新竹探索魅力、桃園瓦奇餐廳；另在[[馬來西亞|馬來西亞]][[檳城|檳城]]M2 CLUB表演。目前在[[台灣|台灣]]駐唱於EZ5 Live House及EZ Corner Live House。

=== [[東南衛視|東南衛視]]：《[[拉芳星光大會|拉芳星光大會]]》 ===
【賽程成績 '''第九名'''】
*在第1集中，首位以"沒有一個X"順利升級的參賽者；節目[[主持人|主持人]][[陶晶瑩|陶晶瑩]]（陶子）稱曾治豪為「台灣小王子」，並介紹其在[[超級星光大道_(第六屆)|第六屆《超級星光大道》]]獲得第十名的實力。
*在第2集中，評審[[黃舒駿|黃舒駿]]點評：曾治豪的表演具有辨識度，而此特質正是新一代藝人最缺乏的。
*在第3集中，評審[[康康|康康]]點評：曾治豪的起承轉合掌握得很好…。同為駐唱歌手的母親現身，提醒音樂這條路，不好走，要自己多努力，並為曾治豪加油打氣。
*在第5集中，評審[[黃舒駿|黃舒駿]]點評：負責和聲部份的曾治豪與[[張智成|張智成]]的合唱完美，可考慮組成雙人美聲團體。由於表現達資優生（90分以上）水準，同時獲得該集的「拉芳美麗之星」<ref>[[:拉芳星光大會|拉芳星光大會 wiki]]</ref>頭銜（同集另位得獎者為[[吳斌|吳斌]]）。
*在第8集中，評審[[陳建寧|陳建寧]]點評：曾治豪小小年紀，可以完美詮釋歌曲當中糾葛的情感面，很令人訝異…有那麼一剎那，有點感覺這像是曾治豪的歌，表現超越個人平常水準。
*在第9集中，同為《[[拉芳星光大會|拉芳星光大會]]》學員的好友[[易慧|易慧]]、[[余超穎|余超穎]]特地到曾治豪駐唱的音樂餐廳探班，[[余超穎|余超穎]]更現場與曾治豪合唱《屋頂》。
*在第10集中，評審[[黃大煒|黃大煒]]點評：為曾治豪的整體改造，多加了1分，而曾治豪的選歌很聰明，雖然前段有些失誤，但結束是完美的，因此，評審個人給予19分（評審當中給分最高的）做為鼓勵。
