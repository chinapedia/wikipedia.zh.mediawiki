'''曾國雄'''是[[中華民國|中華民國]][[國立台北大學|國立台北大學]]的現任講座教授，亦為[[國立交通大學|國立交通大學]]終身講座教授。曾擔任[[交通大學|交通大學]]科技管理研究所所長、以及[[開南大學|開南大學]]校長<ref>{{cite web |url=http://www.old.knu.edu.tw/knu/page/POR/media.htm |title=存档副本 |accessdate=2016-08-29 |deadurl=yes |archiveurl=https://web.archive.org/web/20160913231951/http://www.old.knu.edu.tw/knu/page/POR/media.htm |archivedate=2016-09-13 }}</ref>。他曾經擔任過[[國家科學委員會|國家科學委員會]]特約研究員兩次，並獲頒傑出特約研究員的榮譽<ref>http://www.ntpu.edu.tw/up/?p=1869&lang=zh-tw</ref><ref>http://www.ntpu.edu.tw/ntpuhistory/Alumni/alumni-more.php?id=436</ref>。

曾國雄基於[[ANP|ANP]]及[[DEMATEL|DEMATEL]]的技術，研發出[[DANP|DANP]]的分析方法<ref>Resources, Conservation and Recycling 期刊: http://www.sciencedirect.com/science/article/pii/S0921344912000365</ref>。

==重要研究與學說==
===改良的混合式多評準決策分析===
相較於傳統的多評準決策分析，曾國雄強調五項改良的重點<ref>https://www.ceeol.com/search/article-detail?id=340641</ref><ref>http://ghtzeng.weebly.com/comments-on-ldquomultiple-criteria-decision-making-mcdm-methods-in-economics-an-overviewrdquo.html</ref>：
# 多目標決策中用於分析的變數，應考量現實世界的情況，即變數並非相互獨立、而是交互影響。
# 不能只從現有方案中「爛蘋果裡挑最好的」，而是應該要選擇理想解(aspired solution)<ref>http://ghtzeng.weebly.com/new-books.html</ref>。
# 決策不應只是方案的排序(ranking & selection)，應該要想辦法增進到更好(improve)。
# 資訊的融合應是模糊區間，為非加法或是超加法型。
# 決策空間與目標空間是可變的。

===DANP方法===
在過去[[ANP|ANP]]之分析方法，必須先透過[[DEMATEL|DEMATEL]]等方法確定準則之間的相依性，再由ANP計權重。曾國雄等人<ref>{{cite web |url=http://www.orstw.org.tw/IJOR/vol5no3/paper-3-ijor-vol5_3_-yu-ping_ou_yang.pdf |title=存档副本 |accessdate=2016-08-31 |deadurl=yes |archiveurl=https://web.archive.org/web/20160824174904/http://www.orstw.org.tw/IJOR/vol5no3/paper-3-ijor-vol5_3_-yu-ping_ou_yang.pdf |archivedate=2016-08-24 }}</ref>則基於ANP的概念，研發新的方法DANP，計算層級間、層級內不同要素的權重，而減少不同層級要素的複雜程度。
其優點在於專家問卷的設計上能夠省去大量的題項，專家填答時亦能夠避免邏輯上的不一致情況。

===VIKOR與TOPSIS之比較 ===
在比較不同方案與理想解的距離所使用的方法中，分析並發現[[VIKOR|VIKOR]]能夠避免[[TOPSIS|TOPSIS]]方法中產生誤判的情況。<ref>http://www.sciencedirect.com/science/article/pii/S0377221703000201</ref>
