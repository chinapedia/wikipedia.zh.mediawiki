'''曾努·穆赫辛'''（'''Jugnu Mohsin'''，{{bd|1958|||}}<ref>{{cite web |url=http://www.highbeam.com/doc/1P2-28007304.html |title=Butterfly effect |author=Parul |date=February 21, 2011 |work=India Today |publisher= |accessdate=8 September 2012 |archive-url=https://web.archive.org/web/20140611124121/http://www.highbeam.com/doc/1P2-28007304.html |archive-date=2014-06-11 |dead-url=yes }}</ref>）是一位[[巴基斯坦|巴基斯坦]]出版人暨編輯，其總部設在[[拉合爾|拉合爾]]的巴基斯坦首份英文獨立新聞周刊《[[星期五時報|星期五時報]]》（The Friday Times）工作。她亦在新聞周刊上有一個每月一次的諷刺專欄。於1999年，她的丈夫兼《星期五時報》主編[[納詹姆·塞提|納詹姆·塞提]]因其工作而被[[三軍情報局|三軍情報局]]綁架一個月。期間，因為三軍情報局在沒有起訴塞西的情況下拘留他，因此國際上不少組織均發起了要求釋放塞西的運動。同年，穆赫辛和塞西獲[[保護記者委員會|保護記者委員會]]頒發[[国际新闻自由奖|国际新闻自由奖]]。

== 創辦《星期五時報》 ==
根據塞西指出，他們有了創辦巴基斯坦獨立報紙的靈感，原因是挫折。於1984年，塞西因「莫須有」式的版權指控而被監禁時，沒有任何一份報紙願意為他伸冤。因此，他們才有了創辦獨立報紙的靈感。翌年，他們以穆赫辛的名義申請報紙牌照，全因塞西在政府眼中已經「臭名昭著」，以其名義申請牌照是不可能的。儘管如此，政府仍然拖延至1987年才批准申請。可是，穆赫辛請求政府延後一年才發牌，以等待巴基斯坦獨裁者[[穆罕默德·齐亚·哈克|穆罕默德·齐亚·哈克]]的倒台。此報紙於1989年5月首次發行。<ref name=TFT>{{cite web |url=http://www.thefridaytimes.com/beta3/tft/aboutus.php |title=The good ol' bad days |author=Najam Sethi |work=The Friday Times |archivedate=2012-09-09 |deadurl=no |archiveurl=https://www.webcitation.org/6AY63D230?url=http://www.thefridaytimes.com/beta3/tft/aboutus.php |accessdate=9 September 2012 }}</ref>

== 1999年綁架事件==
於1999年初，[[英國廣播公司|英國廣播公司]]電視節目《通訊員》（Correspondent）訪問了穆赫辛的丈夫塞西，並準備報道巴基斯坦政府的腐敗。<ref name=CPJ2>{{cite web |url=http://cpj.org/1999/10/veteran-journalist-najam-sethi-arrested.php |title=Veteran Journalist Najam Sethi Arrested |date=10 May 1999 |author=Ann K. Cooper |publisher=Committee to Protect Journalists |deadurl=no |archivedate=2012-09-07 |archiveurl=https://www.webcitation.org/6AV3ITYaF?url=http://cpj.org/1999/10/veteran-journalist-najam-sethi-arrested.php |accessdate=7 September 2012 }}</ref>於五月初，聯繫人警告塞西政府有意介入他與《通訊員》製作組的合作，且官員亦在計劃拘捕他。<ref name=CPJ2 />於5月8日，政府間諜在塞西家中拘捕了他。<ref name=CPJ />穆赫辛指出，當時至少有8位持械人員破門入屋，攻擊居所的保安人員。當他們被問及有沒有拘捕令時，其中一人便恐嚇會槍殺塞西。之後，穆赫辛被綁起來，並被鎖在另一間房間中。<ref name=CPJ2 />

三軍情報局在沒有起訴塞西的情況下拘留了他達1個月。他被單獨監禁於拉合爾一個拘留中心中。<ref name=CPJ3>{{cite web |url=http://cpj.org/1999/03/najam-sethi-editor-of-thethe-friday-times-released.php |title=Najam Sethi, editor of the The Friday Times. Released |date=3 June 1999 |author=Ann K. Cooper |publisher=Committee to Protect Journalists |archivedate=2012-09-07 |deadurl=no |archiveurl=https://www.webcitation.org/6AV4MTi5i?url=http://cpj.org/1999/03/najam-sethi-editor-of-thethe-friday-times-released.php |accessdate=7 September 2012 }}</ref>為此，穆赫辛發起了一個釋放塞西的運動，並與一些國際[[非政府組織|非政府組織]]接觸。期間，她仍然繼續出版《星期五時報》。[[國際特赦組織|國際特赦組織]]公開表示他們相信塞西被捕與其對巴基斯坦政府腐敗的調查有關，並把他稱為[[良心犯|良心犯]]。<ref>{{cite web |url=http://www.amnesty.org/es/library/asset/ASA33/015/1999/es/f42c6c5d-e12e-11dd-b6eb-9175286ccde2/asa330151999en.html |title=Further information on UA 107/99 (ASA 33/11/99, 14 May 1999) and follow-up (ASA 33/13/99, 21 May 1999) – Prisoner of conscience/fear of torture |date=3 June 1999 |publisher=Amnesty International |accessdate=14 June 2012}}</ref> 總部設在美國的[[保護記者委員會|保護記者委員會]]亦為此向巴基斯坦總理[[納瓦茲·謝里夫|納瓦茲·謝里夫]]發了一封抗議信，<ref name=CPJ/>[[世界銀行集團|世界銀行集團]]主席[[詹姆斯·沃尔芬森|詹姆斯·沃尔芬森]]亦致電謝里夫請求釋放塞西。<ref>{{cite web |url=http://www.pbs.org/newshour/media/j_awards/sethi_11-23.html |title=Najam Sethi and Jugnu Mohsin |author=Terence Smith |date=23 November 1999 |work=NewsHour |publisher=PBS |archivedate=2012-09-09 |deadurl=no |archiveurl=https://www.webcitation.org/6AY9hUKzr?url=http://www.pbs.org/newshour/media/j_awards/sethi_11-23.html |accessdate=9 September 2012 }}</ref>

於6月1日，政府正式控訴塞西，並把他移交警方羈押。可是，[[巴基斯坦最高法院|巴基斯坦最高法院]]在翌日指出政府沒有提供足夠證據證明塞西有罪。塞西馬上獲釋，而政府亦撤消了控訴。<ref name=CPJ3 />於1999年，[[保護記者委員會|保護記者委員會]]頒予穆赫辛和塞西[[国际新闻自由奖|国际新闻自由奖]]。<ref name=CPJ>{{cite web |url=http://cpj.org/awards/1999/awards.php |title=1999 Awards – Announcement |year=1999 |publisher=The Committee to Protect Journalists |archivedate=2012-09-07 |deadurl=no |archiveurl=https://www.webcitation.org/6AV2DmmJd?url=http://cpj.org/awards/1999/awards.php |accessdate=7 September 2012 }}</ref>

== 《我的領主》 ==
於1991年6月，穆赫辛和塞西出版公司萬佳圖書（Vanguard Books）出版了特赫米娜·杜蘭尼的《我的領主》（My Feudal Lord），一本關於杜蘭尼和政治家穆斯塔法·哈爾婚姻的書。書中，杜蘭尼指出自己曾被哈爾虐待。此書很快成為一本暢銷書。杜蘭尼之後與穆赫辛簽約，並給予她50%外國特許權使用費。<ref name=I>{{cite web |url=http://www.highbeam.com/doc/1P2-5001473.html |title=My feudal lords Amnesty honoured him with its Journalism Under Threat award, but in Pakistan Najam Sethi is still persecuted |author=Peter Popham |date=20 July 1999 |work=The Independent |publisher= |accessdate=8 September 2012 |archive-url=https://web.archive.org/web/20130523032017/http://www.highbeam.com/doc/1P2-5001473.html |archive-date=2013-05-23 |dead-url=yes }}</ref>

可是，於1999年5月19日，杜蘭尼召開記者招待會並宣佈《我的領主》為她帶來的收入全部被塞西騙去。之後，杜蘭尼起訴塞西在精神上的折磨她，而塞西亦反訴杜蘭尼誹謗。最後，塞西獲判無罪。<ref name=I />

== 諷刺 ==
在總統[[佩尔韦兹·穆沙拉夫|佩尔韦兹·穆沙拉夫]]的統治期間，穆赫辛寫了一個名為「濃粥和布什」（Mush and Bush）的諷刺專欄。她在專欄中寫了一段穆沙拉夫和[[美國總統|美國總統]][[喬治·W·布什|喬治·W·布什]]的虛構對話。早前，她亦寫了一個名為「昏暗和專制的個性，他不能容忍不同政見」（dim and authoritarian personality, his intolerance of dissent）的諷刺專欄，諷刺總理[[納瓦茲·謝里夫|納瓦茲·謝里夫]]。<ref name=WP>{{cite web |url=http://www.highbeam.com/doc/1P2-11344641.html |title=How Pakistan's Satirists Poke Fun, Politically |author=Emily Wax |date=28 November 2007 |work=The Washington Post |publisher= |accessdate=8 September 2012 |archive-url=https://web.archive.org/web/20140611052911/http://www.highbeam.com/doc/1P2-11344641.html |archive-date=2014-06-11 |dead-url=yes }}</ref>她的姐姐莫尼·穆赫辛亦有一個名為「交際花的日記」（Diary of a Social Butterfly），專門諷刺該國的社會精英。<ref name=WP />

== 觀點 ==
穆赫辛倡導自由巴基斯坦的概念，並且反對[[原教旨主義|原教旨主義]]。於2006年1月，她主張女人有權在參與[[馬拉松|馬拉松]]時穿著短褲而非[[沙麗克米茲|沙麗克米茲]]。<ref>{{cite web |url=http://www.highbeam.com/doc/1P3-982300041.html |title=Pak women run to shake off their country's Taliban past |date=30 January 2006 |work=Hindustan Times |publisher= |accessdate=8 September 2012 |archive-url=https://web.archive.org/web/20140611052916/http://www.highbeam.com/doc/1P3-982300041.html |archive-date=2014-06-11 |dead-url=yes }}</ref>她亦反對前[[板球|板球]]運動員[[伊姆蘭·罕|伊姆蘭·罕]]加入政壇，全因他沒有經驗。<ref>{{cite web |url=http://www.highbeam.com/doc/1G1-285874086.html |title=King Khan |author=Mike Giglio |date=18 April 2012 |work=Newsweek |publisher= |accessdate=8 September 2012 |archive-url=https://web.archive.org/web/20140611052915/http://www.highbeam.com/doc/1G1-285874086.html |archive-date=2014-06-11 |dead-url=yes }}</ref>
