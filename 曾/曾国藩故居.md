'''曾国藩故居'''位于[[湖南省|湖南省]][[娄底市|娄底市]][[双峰县|双峰县]][[荷叶镇|荷叶镇]]，包括[[富厚堂|富厚堂]]、[[白玉堂|白玉堂]]、黄金堂、大夫第（敦德堂、奖善堂）、万宜堂、修善堂、有恒堂、华祝堂、文吉堂等，建造于[[清朝|清朝]][[同治|同治]]年间，总面积约4万平方米，建筑面积约1万平方米。<ref>{{cite web |url=http://news.163.com/13/0331/20/8RAQOCQV00014JB6.html |title= 探访“湘军首领”曾国藩故居|accessdate=2013-03-31  |publisher=网易 |work= |language=zh-cn}}</ref><ref>{{cite web|url=http://news.rednet.cn/c/2013/02/19/2913641_2.htm|title=湖南娄底市双峰县 曾国藩故居富厚堂|accessdate=2013-02-19|publisher=红网|work=|language=zh-cn|deadurl=yes|archiveurl=https://web.archive.org/web/20130927002524/http://news.rednet.cn/c/2013/02/19/2913641_2.htm|archivedate=2013-09-27}}</ref><ref>{{cite web |url=http://news.sina.com.cn/o/2012-11-19/212725613670.shtml |title=品湖湘名人文化 红网网友自驾曾国藩故居归来 |accessdate=2012年11月19日 |publisher= 新浪网|work= |language=zh-cn}}</ref>

== 沿革 ==
1865年（大清同治四年），曾国藩故居建立，为清式建筑。
