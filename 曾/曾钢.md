'''曾钢'''（{{bd|1952年|}}），出生于[[北京|北京]]，[[中华人民共和国|中华人民共和国]]政治人物、外交官。曾担任外交部拉美司司长。<ref>{{cite news |language = zh-hans |url = http://www.fmprc.gov.cn/mfa_chn/wjb_602314/zzjg_602420/ldmzs_602900/xwlb_602902/t325367.shtml |title = 第二次中日韩外交部拉美事务司级磋商在韩国举行 |publisher = 中华人民共和国外交部 |date = 2007-05-29 |accessdate = 2014-11-13}}</ref>

2002年，接替[[刘峻岫|刘峻岫]]，担任[[中国驻厄瓜多尔大使列表|中华人民共和国驻厄瓜多尔大使]]。<ref>{{cite web| title = 驻厄瓜多尔共和国历任大使 |url = http://www.fmprc.gov.cn/mfa_chn/ziliao_611306/wjrw_611324/2167_611334/2174_611348/t9284.shtml |publisher = 中华人民共和国外交部 |accessdate = 2014-11-13}}</ref>2007年，接替[[张拓|张拓]]，担任[[中国驻阿根廷大使列表|中华人民共和国驻阿根廷大使]]。<ref>{{cite web| title = 驻阿根廷共和国历任大使 |url = http://www.fmprc.gov.cn/mfa_chn/ziliao_611306/wjrw_611324/2167_611334/2174_611348/t9278.shtml |publisher = 中华人民共和国外交部 |accessdate = 2014-11-13}}</ref>2010年，由[[殷恒民|殷恒民]]接任，其改任担任[[中国驻墨西哥大使列表|中华人民共和国驻墨西哥大使]]。<ref>{{cite web| title = 驻墨西哥历任大使 |url = http://www.fmprc.gov.cn/mfa_chn/ziliao_611306/wjrw_611324/2167_611334/2174_611348/t9289.shtml |publisher = 中华人民共和国外交部 |accessdate = 2014-11-13}}</ref>2013年，由[[邱小琪|邱小琪]]接任。<ref>{{cite web| title = 国家主席习近平任免驻外大使 |url = http://www.fmprc.gov.cn/mfa_chn/zyxw_602251/t1081502.shtml |publisher = 中华人民共和国外交部 |date = 2013-09-26 |accessdate = 2014-11-13}}</ref>
