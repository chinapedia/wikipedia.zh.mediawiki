'''曾志豪'''，為臺灣音樂工作者。[[國立台灣師大附中|國立台灣師大附中]]、[[義守大學|高雄工學院]]資訊工程學系畢業，自學音樂成材，曾任高雄工學院電子音樂社創社社長，現任大宇資訊音樂音效課課長兼音序作業員，主要負責大宇資訊大多數電玩產品的配樂和音效的製作。

== 電玩配樂創作歷史 ==

1997年進入大宇資訊工作，擔任遊戲配樂製作人員，配樂作品包含[[仙劍奇俠傳系列|仙劍奇俠傳系列]]、[[軒轅劍系列|軒轅劍系列]]、[[大富翁系列|大富翁系列]]、[[明星志願|明星志願系列]]、[[天使帝國|天使帝國]]系列、[[正宗台灣麻將|正宗台灣麻將]]系列，作品超過一千首，並與許多遊戲製作人合作。<ref> {{cite web|url=http://www.midifan.com/modulearticle-detailview-2931.htm |title=仙劍系列音樂團隊主創，台灣知名遊戲音樂製作人曾志豪專訪 |accessdate=2012-12-02 |date=2012-12-02 }}</ref>

== 和演藝界的合作 ==

因為遊戲專案合作，結識繆思特音樂工作室的[[呂聖斐|呂聖斐]]、駱集益。

2008年，獲得時任電影[[海角七號|海角七號]]音樂總監的呂聖斐推薦，擔任主題曲「國境之南」作曲者。同年，與國境之南原唱[[范逸臣|范逸臣]]、作詞者嚴云農，共同獲頒第45屆[[金馬獎|金馬獎]]最佳原創電影歌曲獎項。<ref> {{cite web |url=http://iservice.libertytimes.com.tw/liveNews/news.php?no=157046 |title=第45屆金馬獎入圍與得獎名單 |author=自由時報 |accessdate=2009-04-02 |date=2008-12-09 }}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref> {{cite web|url=http://gnn.gamer.com.tw/3/33923.html |title=跨足《海角七號》大宇曾志豪榮獲金馬獎最佳原創電影歌曲獎 |accessdate=2009-04-02 |date=2008-12-09 }}</ref>

2010年，大宇資訊與無限延伸音樂事業有限公司旗下歌手[[陳思涵|陳思涵]]合作，曾志豪負責其所演唱的《[[軒轅劍外傳_雲之遙|軒轅劍外傳 雲之遙]]》的主題歌《淚狂奔》等歌曲的作曲。

2011年，大宇資訊與歌手[[陳依婷|陳依婷]]合作，曾志豪負責其所演唱的《[[仙劍奇俠傳五|仙劍奇俠傳五]]》的主題歌《心願》和宣傳歌《少年情》的作曲。

2012年底，大宇資訊第二次與歌手[[蕭人鳳|蕭人鳳]]合作，曾志豪負責其所演唱的《[[仙劍奇俠傳五前傳|仙劍奇俠傳五前傳]]》的主題歌《牽絆》的作曲。

2013年，大宇資訊與大陸歌手[[廖羽|廖羽]]合作，曾志豪負責其所演唱的《[[軒轅劍陸_鳳凌長空千載雲|軒轅劍陸_鳳凌長空千載雲]]》的歌曲《千年醉》的作曲(PRODUCER：靳鐵章，編曲：陳飛午)。
