'''陰蝨'''（''Pthirus pubis''）是一種[[寄生|寄生]]於[[人體|人體]][[毛髮|毛髮]]的[[寄生虫|寄生虫]]，長約1至3毫米，無[[翼|翼]]。因常見於[[阴部|阴部]]，故稱陰蝨。另外，由於陰蝨身體扁平，远看如同皮屑，细看则如同小[[螃蟹|螃蟹]]，故在英語中又稱蟹蝨 （Crab louse）。

== 簡介 ==
雖然最常見於陰部，但陰蝨在絕大部份人類毛髮上均可生存。它的身體較[[頭蝨|頭蝨]]寬闊，腳部較寬廣，使它可於疏落的毛髮間攀爬。除了[[陰毛|陰毛]]外，阴虱也可在[[头发|头发]]、[[睫毛|睫毛]]、[[鬍鬚|鬍鬚]]和[[腋毛|腋毛]]处寄生。

和其他[[蝨|蝨子]]一樣，陰蝨以人類的[[血液|血液]]為[[食物|食物]]，因此陰蝨棲息的患處會不時感到痕癢。另外，陰蝨亦可能帶有其他[[病菌|病菌]]，使患者產生[[感染|感染]]。

陰蝨的傳播途徑主要是[[性交|性交]]，因此陰蝨感染被認為是[[性传播疾病|性传播疾病]]的一種。[[衣物|衣物]]和[[床|床]]鋪也是陰蝨的重要傳播途徑。[[廁所|廁所]]則'''不會'''傳染陰蝨，因為廁所的平面光滑，缺乏可讓蝨子停留的條件。如廁前不久如患有陰蝨患者剛使用過,亦有被傳染之可能。

== 生命週期 ==
阴虱为[[不完全变态|不完全变态]]发育，其生活史分为：卵、若虫和成虫三个阶段。

# '''[[卵|卵]]'''：陰蝨的蟲卵一般都緊附在阴毛的毛幹根部上，呈[[淡红色|淡红色]]或[[铁锈|铁锈]]色的[[橢圓形|橢圓形]]，类似于点状血痂。孵化期約為一星期。
# '''[[若虫|若虫]]'''：若虫是阴虱自卵孵化后，成长为成虫之前的形态。若虫的样子与成虫相似，但是要小很多。若虫的成長期亦為一星期左右。
# '''[[成虫|成虫]]'''：阴虱的成蟲呈[[灰白色|灰白色]]或[[褐色|褐色]]，[[雌性|雌蟲]]一般比[[雄性|雄蟲]]為大。

一般陰蝨的[[壽命|壽命]]是30天，但所有脫離人體的陰蝨均會在2天內[[死亡|死亡]]。

== 症状 ==
被阴虱寄生后，通常会出现如下症状：

# 阴部搔痒，特别是夜间搔痒更甚；
# 阴部出现红疹、丘疹、血痂或青斑；
# 内裤上有铁锈色粉末状或颗粒状虱粪；
# 阴毛根部可发现铁锈色或红褐色椭圆形虫卵；
# 阴毛处发现阴虱活动。

== 治療 ==
# 剃净阴毛及肛周毛发并将其焚烧；
# 将被污染的衣物、床单、被罩蒸煮或开水浇烫消毒，杀灭虫卵及成虫；
# 使用热水及肥皂（[[硫磺皂|硫磺皂]]）清洗阴部；
