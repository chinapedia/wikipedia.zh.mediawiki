'''沧源佤族自治县'''（[[佤语|佤语]]：Qang Yin Vax Qux Zi Zi Sin）简称'''沧源'''，是[[中华人民共和国|中华人民共和国]][[云南省|云南省]][[临沧市|临沧市]]下属的一个[[自治县|自治县]]，也是中国两个纯佤族自治县之一<ref group="注">另一个是[[西盟佤族自治县|西盟佤族自治县]]</ref>，县政府驻[[勐董镇|勐董镇]]。[[佤族|佤族]]是沧源县内的主体民族，全县18.56万人口中有14.7万是佤族，占总人口的80%<ref name="沧源县政府网">{{cite web |url=http://www.cangyuan.gov.cn/list-6-1.html |title=沧源概况 |last= |first= |date= |website=沧源县政府网 |publisher=沧源县人民政府 |access-date=2017-10-22 |quote=}}</ref>，通用[[佤语|佤语]]。[[沧源口岸|沧源口岸]]是国家二类陆路[[口岸|口岸]]，有三条口岸通道通往缅甸<ref>{{cite web |url=http://www2.customs.gov.cn/tabid/30040/Default.aspx |title=沧源口岸简介 |last= |first= |date= |website=www2.customs.gov.cn |publisher=中华人民共和国沧源海关 |access-date=2017-10-22 |quote= |archive-url=https://web.archive.org/web/20180201002302/http://www2.customs.gov.cn/tabid/30040/Default.aspx |archive-date=2018-02-01 |dead-url=yes }}</ref>。

==历史==
沧源古称佧佤山区或[[阿佤山|阿佤山区]]，也称“葫芦王地”。[[汉朝|汉]]至[[东晋|东晋]]时期，沧源属[[哀牢|哀牢地]][[永昌郡|永昌郡]]，[[南朝|南朝]]时属[[宁州|宁州]]，[[南诏|南诏]]时期属[[南诏行政区划|银生节度]]管辖，[[大理国|大理国]]时属裸黑部，[[元朝|元朝]]时属[[孟连路|孟连路]]和[[孟定路|孟定路]]，[[明朝|明朝]]属[[孟连司|孟连司]]和[[耿马司|耿马司]]，[[清朝|清朝]]属[[永昌府|永昌府]][[镇边抚彝直隶厅|镇边抚彝直隶厅]]。民国初年设'''沧源行政委员会'''，隶属于[[普洱道|普洱道]]，1915年划归[[澜沧拉祜族自治县|澜沧县]]第十一区，1937年成立'''沧源设治局'''，治所在[[勐董镇|勐董]]，1940年后沧源设治局先后隶属云南省第一、第四和第九行政督察区。1949年成立临时人民政府，属[[思茅专区|宁洱专区]]，1951年成立'''沧源县'''人民政府，属[[思茅专区|普洱专区]]<ref group="注">1951年宁洱专区更名普洱专区</ref>，驻地在[[岩帅镇|岩帅镇]]。1952年，[[缅宁专区|缅宁专区]]成立，沧源县划归缅宁专区管辖，同时驻地从岩帅迁往勐董。1954年，缅宁专区更名[[临沧专区|临沧专区]]<ref group="注">临沧专区于1970年改为临沧地区，后在2003年撤地设市，即今[[临沧市|临沧市]]</ref>。1958年，沧源县改设'''沧源佧佤族自治县'''。1963年，沧源佧佤族自治县更名沧源佤族自治县<ref group="注">1963年系批准时间，正式更名时间为1964年</ref>。<ref name="中国地名掌故词典"></ref><ref>{{cite book |last=陈 |first=潮 |date= |title=中华人民共和国行政区划沿革地图集 |url= |location=[[北京|北京]] |publisher=[[中国地图出版社|中国地图出版社]] |page=P127-P128 |isbn=978-7-5031-3184-4 |author-link= }}</ref><ref name="简介"></ref><ref>{{cite web |url=http://www.xzqh.org/html/show/yn/19061.html |title=沧源县历史沿革 |last= |first= |date=2014-10-28 |website=www.xzqh.org |publisher=行政区划网 |access-date=2017-10-22 |quote=}}</ref>

1934年，沧源西部的[[班洪乡|班洪]]等地曾发生当地佤族反抗[[英国|英国]]武装侵占班洪和班佬等部落共管地区银矿的[[班洪事件|班洪事件]]<ref>{{cite web |url=http://history.kunming.cn/index/content/2009-09/07/content_1962138.htm |title=1934年 震惊中外的沧源“班洪事件” |last=陈 |first=江涛 |date=2009-09-07 |website=昆明信息港 |publisher=优游旅行网 |access-date=2017-10-22 |quote=}}</ref>。

1960年，[[中华人民共和国总理|中华人民共和国总理]][[周恩来|周恩来]]与[[缅甸总理|缅甸联邦总理]][[吴努|吴努]]签订《[[中华人民共和国和缅甸联邦边界条约|中华人民共和国和缅甸联邦边界条约]]》，缅甸将今属[[怒江傈僳族自治州|怒江州]][[泸水市|泸水市]]的[[片马镇|片马]]、古浪、岗房地区约153平方公里的土地归还中国，而中国政府则同意按照1941年6月18日中英两国政府换文来划定从[[南帕河|南帕河]]和[[南定河|南定河]]汇合处到[[南卡江|南卡江]]和[[南永河|南永河]]汇合处的一段边界；中国政府将[[瑞丽市|瑞丽]]以西面积220平方公里的猛卯三角地移交给缅甸，作为交换获得面积189平方公里的班洪、班老部落辖区，即今属沧源县的[[班洪乡|班洪乡]]、[[班老乡|班老乡]]。<ref>[[维基文库|维基文库]]中的《[[:s:中华人民共和国和缅甸联邦边界条约|中华人民共和国和缅甸联邦边界条约]]》</ref>

2015年3月1日，[[南汀河断裂带|南汀河断裂带]]沧源县[[芒卡镇|芒卡镇]]段发生5.5级地震，震源深度11千米，造成沧源、[[耿马傣族佤族自治县|耿马]]和[[镇康县|镇康]]三县69,657人受灾、50人受伤，其中8人重伤<ref>{{cite web |url=http://www.chinanews.com/gn/2015/03-02/7092715.shtml |title=云南沧源地震救援全面展开 |last=胡 |first=远航 |date=2015-03-02 |website=中国新闻网 |publisher=中国新闻网 |access-date=2017-10-22 |quote=}}</ref><ref>{{cite web |url=http://news.ifeng.com/a/20150301/43243837_0.shtml |title=云南沧源地震造成孟定镇贺海村14人受轻伤 |last= |first= |date=2015-03-01 |website=凤凰资讯 |publisher=人民网 |access-date=2017-10-22 |quote=}}</ref><ref>{{cite web |url=http://www.cea.gov.cn/publish/dizhenj/468/553/101700/101701/20150304104912320289170/index.html |title=云南共安排应急资金950万元用于抗震救灾 |last= |first= |date=2015-03-04 |website=中国地震局 |publisher=人民网 |access-date=2017-10-22 |quote=}}</ref>。

==地理==
沧源佤族自治县东接[[双江拉祜族佤族布朗族傣族自治县|双江拉祜族佤族布朗族傣族自治县]]和[[普洱市|普洱市]][[澜沧拉祜族自治县|澜沧拉祜族自治县]]，北靠[[耿马傣族佤族自治县|耿马傣族佤族自治县]]，西部和南部邻[[缅甸|缅甸]][[掸邦|掸邦]][[佤邦自治区|佤邦自治区]]。沧源县地处滇西[[横断山脉|横断山脉]]纵谷地带南端，地势北、中高，东、西、南三面低，由北向南倾斜，地形复杂多变，深切的中山与河谷交错，山区面积占99.2%，坝区仅占0.8%，坝子主要有勐角，勐董，勐省等小平坝。沧源县内的最高峰是窝坎大山海拔2605米，最低点为芒卡坝南汀河出境处，海拔为450米，县城海拔1,270米。主要河流有[[小黑江_(瀾滄江右支流)|小黑江]]和[[南滚河|南滚河]]。[[南滚河国家级自然保护区|南滚河国家级自然保护区]]位于县西部。<ref name="简介">{{cite web |url=http://www.yn.xinhuanet.com/nets/cy/ |title=沧源简介 |last=木 |first=胜玉 |date= |website=新华网云南频道 |publisher=沧源县人民政府 |access-date=2017-10-22 |quote=}}</ref><ref name="云南省地图册">{{cite book |last= |first= |date= |title=云南省地图册 |url= |location= |publisher=[[星球地图出版社|星球地图出版社]] |page=P94-P95 |isbn=978-7-5471-1018-8 |author-link= }}"</ref><ref name=地理与资源>{{cite web |url=http://ein.ynnu.edu.cn/wasearch/articleview.aspx?id=628 |title=沧源佤族自治县的地理与资源 |last=施 |first=玉 |date=2013-01-07 |website=ein.ynnu.edu.cn |publisher=佤族大辞典 |access-date=2017-10-22 |quote=}}</ref>

沧源县共有森林面积53万亩，占总面积的12%，森林蓄积量为539万立方米，已探明的矿产有煤、铁、铅、银、石膏、硫磺等，其中仅煤和铅有开采记录<ref name=地理与资源></ref>。

气候上，沧源属[[副热带季风气候|南亚热带季风气候]]，气候温和干湿分明，降雨量充沛，日照充足，因地形、海拔的不同，气温呈垂直立体分布，年、月平均气温差异较大，一般西部深切河谷和低海拔地带气温高于其他地区，境内山间谷盆地带的气温高于东北部，年均日照时数为1126.9至1905.3小时，历年平均霜期为48至50天<ref name=地理与资源></ref>。2015年统计，沧源县年均气温18.4
℃，全年降水量为1608.9 ㎜<ref>{{cite book |author=云南省统计局 |date= |title=2016云南统计年鉴 |url=http://www.yearbookchina.com/FileDownload.aspx?type=xls&file=n2017020253000293%3d555-558 |location=北京 |publisher=中国统计出版社 |page=P555-P558 |isbn=978-7-5037-7903-9 |author-link= }}</ref>。

==人口==
2015年统计，沧源佤族自治县有农村人口14.4万人，共3.71万户，乡村从业人员9.14万人<ref name="农村和农业生产基本情况">{{cite book |author=云南省统计局 |date= |title=2016云南统计年鉴 |url=http://www.yearbookchina.com/FileDownload.aspx?type=xls&file=n2017020253000305%3d591-594 |location=北京 |publisher=中国统计出版社 |page=P591-P594 |isbn=978-7-5037-7903-9 |author-link= }}</ref>。

==交通==
沧源县境内没有[[高速公路|高速公路]]和[[铁路|铁路]]，[[214国道|214国道]]经过沧源县东部[[岩帅镇|岩帅镇]]的[[小黑江_(瀾滄江右支流)|小黑江]]西岸。县内主要公路有[[231省道_(云南)|231省道]]和[[236省道_(云南)|236省道]]，其余皆为[[中华人民共和国公路分类|县乡级公路]]。<ref name="云南省地图册"></ref>正在规划建设中的[[芒孟高速|芒孟高速]]（[[芒市|芒市]]至[[孟连傣族拉祜族佤族自治县|孟连]]）经过沧源县，将在[[糯良乡|糯良乡]]设置糯良[[立交桥|互通]]，并建设连接沧源县城与糯良互通的一级路<ref>{{cite web |url=http://xw.kunming.cn/a/2016-11/22/content_4430970.htm |title=芒市至孟连高速耿马至沧源段拟于2017年1月动工 |last=昝 |first=娟娟 |date=2016-11-22 |website=昆明信息港 |publisher=昆明信息港 |access-date=2017-10-22 |quote=}}</ref>。

[[沧源佤山机场|沧源佤山机场]]位于县城以北36公里<ref name="沧源机场"></ref>，于2016年12月8日正式建成通航<ref>{{cite web|url=http://www.yn.xinhuanet.com/2016info2/20161206/3564154_c.html|title=沧源佤山机场通航力助文化旅游发展|author=赵淑芳 爱星·西涅 李树鑫|date=2016-12-06|publisher=新华网|deadurl=yes|archiveurl=https://web.archive.org/web/20161207165031/http://www.yn.xinhuanet.com/2016info2/20161206/3564154_c.html|archivedate=2016-12-07}}</ref>，现有飞往[[昆明市|昆明]][[昆明长水国际机场|长水国际机场]]的航班，并计划于2018年开通飞往[[成都双流国际机场|成都]]与[[重庆江北国际机场|重庆]]的航线<ref name="沧源机场">{{cite web |url=http://www.cnta.gov.cn/xxfb/hydt/201611/t20161130_808097.shtml |title=“云端上的机场”云南沧源佤山机场将于12月8日通航 |last= |first= |date=2016-12-01 |website=国家旅游局信息中心 |publisher=中国新闻网 |access-date=2017-10-21 |quote= }}{{Dead link|date=2018年10月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。
