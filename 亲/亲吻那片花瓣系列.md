{{vgname|親吻那片花瓣|ja=その花びらにくちづけを}}，是由[[同人社團|同人社團]]{{lang|ja|[[ふぐり屋|ふぐり屋]]}}製作發行的[[戀愛冒險|戀愛冒險]]類型[[日本成人游戏|成人]][[同人遊戲|同人遊戲]]系列，後來陸續改編成小說、OVA。此系列遊戲主要是描敘“聖米卡艾魯女子學院”（{{lang|ja|聖ミカエル女子学園}}，簡稱為米卡女校）的女性師生們的[[百合_(同人)|百合]]故事。

;{{nihongo|蓬莱泉瑠奈|蓬莱泉 瑠奈（ほうらいせん　るな）||聲：[[上田幸姫|上田幸姫]]}}

== 相關作品 ==
=== OVA ===
OVA是從第三作《{{lang|ja|その花びらにくちづけを あなたと恋人つなぎ}}》改編的成人動畫，2010年7月30日發售DVD版，2016年5月27日发售BD版。

=== 小说 ===
本作的番外篇小説均由原作遊戲劇本作者佐野晋一郎以及同人小説社團{{lang|ja|ルミノシティ}}合作編著<ref>[http://lumino.sakura.ne.jp/doujin.html ルミノシティ - 同人誌情報]</ref>，因爲加入了[[性愛|性愛]]場面的描寫，所以被分類為[[成人小説|成人小説]]。以[[同人志|同人志]]的方式發行，售價均為735[[日元|日元]]。

;親吻那片花瓣——二人的聖誕節
原名《{{lang|ja|その花びらにくちづけを 二人のクリスマス}}》，于2006年12月31日，即[[Comic_Market|Comic Market 71]]舉辦時發行。講述了優菜在平安夜之前再次邀請七海去她的家裏舉行夜宿派對的故事。<ref>[http://fuguriya.sakura.ne.jp/c71.html ふぐり屋 Official Web Site - 初めての、二人きりのクリスマス♪]</ref>

;親吻那片花瓣——二人的假日時光
原名《{{lang|ja|その花びらにくちづけを 二人のバカンス}}》，于2007年8月19日，即[[Comic_Market|Comic Market 72]]舉辦期間發行。講述楓與紗良爲了化解學業的壓力利用假期去南方海邊的事情。<ref>[http://fuguriya.sakura.ne.jp/c72_a.html ふぐり屋 Official Web Site - 夕日に染まる海岸で、身を寄せ合う二人のバカンス♪]</ref>

;親吻那片花瓣——二人的炎熱夏天
原名《{{lang|ja|その花びらにくちづけを 二人の熱い夏}}》，與上作同日發行。講述了為麻衣與玲緒兩人前往玲緒家鄉下別墅的快樂時光。<ref>[http://fuguriya.sakura.ne.jp/c72_b.html ふぐり屋 Official Web Site - 玲緒の、『麻衣と二人きりでラブラブ大作戦！』]</ref>

;親吻那片花瓣——二人的情人節
原名《{{lang|ja|その花びらにくちづけを 二人のバレンタイン}}》，于2007年12月22日，即[[Comic_Market|Comic Market 73]]舉辦時發行。故事情節為七海和優菜交往之後的第一個情人節前夕，七海苦惱于送給姐姐大人巧克力的事情。<ref>[http://fuguriya.sakura.ne.jp/c73.html ふぐり屋 Official Web Site - 二人きりで過ごす、初めてのバレンタイン]</ref>

;親吻那片花瓣——不是劇終的謝幕
原名《{{lang|ja|その花びらにくちづけを カーテンコールは終わらない}}》，于2008年8月8日，即[[Comic_Market|Comic Market 74]]舉辦時發行。講述楓與紗良作爲校内的人氣人物應邀參加校慶中的話劇表演的故事。<ref>[http://fuguriya.sakura.ne.jp/c74.html ふぐり屋 Official Web Site - 王子様役は、楓ちゃんで決まりだよね！]</ref>

;親吻那片花瓣——再次的深情之吻
原名《{{lang|ja|その花びらにくちづけを 愛のキスをもういちど}}》，于2009年2月20日發行。故事發生的時間與上一冊相同為校慶之前，講述了麻衣意外地證實了玲緒的歌唱才能（在其房間内找到了她錄製的[[CD|CD]]），因此説服她參加校慶合唱的事情。<ref>[http://fuguriya.sakura.ne.jp/c75.html ふぐり屋 Official Web Site - 学園祭では合唱をするのはどうかなっ！？]</ref>

=== 廣播劇CD ===
本作的廣播劇的發售時間和小説同時，同樣因爲有性愛情節，只能以CD為媒介發售而不能公開播放。廣播劇的錄製均由原作遊戲的聲優完成，售價統一為1,900日元。

;親吻那片花瓣——幸福之吻
原名《{{lang|ja|その花びらにくちづけを ずっと幸せなキス}}》，于2007年12月22日，即[[Comic_Market|Comic Market 73]]舉辦時發行。講述七海與她的姐姐大人的初次約會的事情。<ref>[http://fuguriya.sakura.ne.jp/cd01.html ふぐり屋 Official Web Site - お姉さまと見知らぬ街での初デート！もう思いっきりラブラブしちゃうの♪]</ref>

;親吻那片花瓣——小楓的Super Mode
原名《{{lang|ja|その花びらにくちづけを 楓ちゃんスーパーモード}}》，于2008年2月20日發行。情節為身為戀人的北嶋姐妹前往溫泉旅館的故事。<ref>[http://fuguriya.sakura.ne.jp/cd02.html ふぐり屋 Official Web Site - いつもは地味な私だけど……旅先では大胆にいちゃラブしちゃうの♪]</ref>

=== 主題歌CD ===
[[File:甘い花园～コンプリートMIX～.jpg|320px]]
2008年12月28日，即[[Comic_Market|C75]]舉辦期間，由[[5pb.|5pb.]]發佈了遊戲的主題歌CD《{{lang|ja|甘い花園～コンプリートMIX～}}》。售價1,050日元。除《甘い花園～コンプリートMIX～》外還有其原版和伴奏、旋律皆為相同的其他兩個主題曲版本《潤う夕空》以及《誘惑のEtude》。<ref>[http://5pb.jp/records/release/detail/detail.php?records_product_code=FPBD-0076 5pb. Records：甘く切ない思いの全てを…聴いて…]</ref>

整張專輯的詞曲作者為濱田智之，編曲者為大川茂伸，歌曲演唱則由聲優[[春奈有美|春奈有美]]完成。
