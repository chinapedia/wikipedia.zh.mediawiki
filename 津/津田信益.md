'''津田信益'''（生年不詳－1633年7月7日）是[[江戶時代|江戶時代]]初期的[[旗本|旗本]]。[[越前|越前]]北之庄藩（[[福井藩|福井藩]]）藩士。父親是[[織田信清|織田信清]]。

== 生平 ==
生年不詳。父親信清是[[尾張|尾張]][[犬山城|犬山城]]城主，在反抗信長後逃亡，而信益則被信長原諒。[[天正_(日本)|天正]]9年（1581年），以[[連枝眾|連枝眾]]身份參加閱兵儀式（{{Link-ja|京都御馬揃|京都御馬揃え}}）。天正10年（1582年），[[本能寺之變|本能寺之變]]前，在信長[[上洛|上洛]]之際，與[[蒲生賢秀|蒲生賢秀]]等人一同成為[[安土城|安土城]]本丸的留守眾（『[[信長公記|信長公記]]』）。後來仕於[[豐臣秀吉|豐臣秀吉]]，因為[[茶道|茶道]]關係而在[[片桐且元|片桐且元]]之下。晩年仕於[[越前|越前]]北之庄藩（[[福井藩|福井藩]]），成為向[[德川將軍家|德川將軍家]]的連絡役。

== 子孫 ==
長女奈和（[[長壽院|長壽院]]）成為[[結城秀康|結城秀康]]的側室，在北之庄藩領內的[[永平寺町|永平寺町]]於信益的屋敷內，誕下在後來成為[[越前|越前]][[大野藩|大野藩]]藩主的六男[[松平直良|松平直良]]。在[[慶長|慶長]]14年（1609年）死去。

次女於佐井（[[貞正院|貞正院]]，1624年－1684年）因為[[德川家康|德川家康]]關係，仕於家康的嫡男[[德川秀忠|秀忠]]的女兒[[東福門院|東福門院]]，後來成為[[德川義直|德川義直]]的側室，在義直的正室[[淺野氏|淺野氏]]死後，以繼室身份成為正室，誕下女兒[[京姬|京姬]]（[[廣幡忠幸|廣幡忠幸]]室）。相傳容姿和才知優秀。

長男[[津田信總|信總（九郎次郎）]]和三男[[津田信勝|信勝（圖書）]]仕於松平直良。信勝因為直良勸說而回復[[織田氏|織田姓]]，成為大野藩（後來的[[明石藩|明石藩]]）[[家老|家老]]織田家的家祖。信總的兒子[[津田直信|直信]]成為[[尾張藩|尾張藩]]藩士津田家的家祖。

== 其他 ==
一說指與[[織田信長|織田信長]]的嫡男[[織田信忠|信忠]]的次男[[織田秀則|織田秀則]]是同一人物。根據這個說法，在[[關原之戰|關原之戰]]中加入西軍並敗北後，與兄長[[織田秀信|秀信]]分離並逃亡，並被[[結城秀康|結城秀康]]（因為秀康曾成為[[豐臣秀吉|豐臣秀吉]]的養子，所以越前松平家的立場被認為是親豐臣派）匿藏，並改姓為織田家的連枝「津田」，以「津田左衛門信益」為名，但亦有可能是同名同姓的其他人。
