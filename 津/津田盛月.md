'''津田盛月'''（1534年－1593年）是[[日本戰國時代|日本戰國時代]]至[[安土桃山時代|安土桃山時代]]的武將。[[織田氏|織田氏]]一族。名諱是'''信勝'''、'''信重'''。通稱'''左馬允'''。官位是[[隼人正|隼人正]]。父親是[[織田刑部大輔|織田刑部大輔]]，哥哥是[[中川重政|中川重政]]，祖父是[[織田信長|織田信長]]的叔父[[織田信次|織田信次]]（『[[織田系圖|織田系圖]]』），不過在年代上有矛盾（『織田信長家臣人名辭典』，[[吉川弘文館|吉川弘文館]]出版，2010年，第2版，作者：[[谷口克廣|谷口克廣]] ）。妻子是[[織田勝左衛門尉|織田勝左衛門尉]]（『織田系圖』）。兒子有[[津田信任|信任]]、[[津田信成|信成]]。女兒是[[織田忠辰|織田忠辰]]妻。

== 生平 ==
在[[天文_(後奈良天皇)|天文]]3年（1534年）出生。仕於[[織田信長|織田信長]]，幫助信長統一[[尾張|尾張]]。天文21年（1552年），在信長與[[織田信友|織田信友]]的戰鬥（[[萱津之戰|萱津之戰]]）中，殺死[[坂井五郎|坂井五郎]]（『[[織田系圖|織田系圖]]』）。[[弘治_(後奈良天皇)|弘治]]2年（1556年），在信長與[[織田信行|織田信行]]（信勝）的戰鬥（[[稻生之戰|稻生之戰]]）中，殺死[[柴田勝家|柴田勝家]]手下的[[鎌田勘之丞|鎌田勘之丞]]（『[[信長公記|信長公記]]』）。因為這些功績而成為[[黑母衣眾|黑母衣眾]]之一（『高木文書』）。從[[永祿|永祿]]12年（1569年）開始因為信長的命令而擔任[[足利義昭|足利義昭]]的守備。

不過此後，哥哥[[中川重政|中川重政]]與[[長光寺城|長光寺城]]城主[[柴田勝家|柴田勝家]]因為領地合組的問題而出現爭執，盛月更斬殺勝家的[[代官|代官]]，於是與兄長一同被改易並追放。

後來被[[羽柴秀吉|羽柴秀吉]]召見而前往[[姬路|姬路]]，此時改名為'''外峯四郎左衛門'''並出仕。以後，在秀吉之下在進攻[[備中|備中]]等戰事中立下戰功。而在秀吉之下的事被信長發現，於是被下令[[切腹|切腹]]，不過因為[[本能寺之變|本能寺之變]]而作罷（『[[武家事紀|武家事紀]]』）。

[[天正_(日本)|天正]]12年（1584年），參加[[小牧長久手之戰|小牧長久手之戰]]，因為這次功績而敘任[[從五位下|從五位下]][[隼人正|隼人正]]，此時改為津田姓。弟弟[[木下雅樂助|木下雅樂助]]於此戰中戰死。在戰後[[德川家康|德川家康]]與秀吉的妹妹[[朝日姬|朝日姬]]的婚事，有說法指這是[[蜂須賀正勝|蜂須賀正勝]]和盛月的功績（『[[川角太閤記|川角太閤記]]』）。

在天正15年（1587年）參加[[九州征伐|九州征伐]]，被秀吉賜予所領3萬5千石而成為[[大名|大名]]。

天正18年（1590年）10月，在[[後北條氏|後北條氏]]的[[北條氏邦|北條氏邦]]的家臣[[豬俁邦憲|豬俁邦憲]]奪取[[名胡桃城|名胡桃城]]後，與[[富田一白|富田一白]]一同被派遣向北條氏要求引渡和處罰涉事人。北條氏拒絕要求，於是秀吉和北條氏的關係完全決裂，導致秀吉下令全國討伐北條氏（[[小田原征伐|小田原征伐]]）。

在[[文祿|文祿]]2年（1593年）於[[伏見|伏見]]死去。享年58歲（『[[兼見卿記|兼見卿記]]』）。繼任人是長男[[津田信任|信任]]。

== 逸話 ==
