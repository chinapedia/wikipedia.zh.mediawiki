'''津島之宮站'''（{{jpn|j=津島ノ宮駅|hg=つしまのみやえき|rm=Tsushimanomiya eki|f=y}}）是一位於[[日本|日本]][[三豐市|三豐市]]三野町大見，隸屬於[[四國旅客鐵道|四國旅客鐵道]]（JR四國）的臨時[[鐵路車站|鐵路車站]]，並不設車站編號。

本站和位於[[德島縣|德島縣]]的[[田井之濱站|田井之濱站]]，均屬JR四國（亦是全[[四國島|四國島]]內）長期休止的車站，相比田井之濱站一年營業23日，津島之宮站每年只營業2天，分別為8月4日全日及5日上午。以日數計，本站和[[會津鐵道|會津鐵道]][[會津線|會津線]]的{{link-ja|一之堰六地藏尊站|一ノ堰六地蔵尊駅}}並列全日本全年營業日數最少的臨時站；<ref>不計算全年所有列車全通過的臨時站。</ref> 但如以總營業時間計算，本站則比一之堰六地藏尊站還短；而當以停靠列車數目來計算時，本站來回停靠的班次都比一之堰六地藏尊站為多。

車站的設立，是為了配合車站附近的津嶋神社（又稱津島神社）每年兩天的夏季大祭才使用的。<ref>{{cite web |url=http://www.shikoku.org.uk/JR/LINE/yosan/ST-TSUSHIMANOMIYA.htm |title=存档副本 |accessdate=2013-03-24 |deadurl=yes |archiveurl=https://web.archive.org/web/20140810011902/http://www.shikoku.org.uk/JR/LINE/yosan/ST-TSUSHIMANOMIYA.htm |archivedate=2014-08-10 }} 非官方車站介紹。</ref>

==車站結構==
設有一座單層木製的建築物，設於月台末端向[[多度津站|多度津站]]方向，主要是用作處理站務及儲存開站時所需要的各類物品及配備。除了在售票口部份加設有窗口外，其他窗口部份皆採用開放式的設計，當車站營業時，位處兩側及前端的售票窗口的圍板可以撐開<ref>{{cite web |url=http://www.shikoku.org.uk/JR/LINE/yosan/ST-TSUSHIMANOMIYA_c.jpg |title=存档副本 |accessdate=2013-03-24 |deadurl=yes |archiveurl=https://web.archive.org/web/20140810082217/http://www.shikoku.org.uk/JR/LINE/yosan/ST-TSUSHIMANOMIYA_c.jpg |archivedate=2014-08-10 }} 車站站舍營業時的情形，取自非官方車站介紹。</ref>，而當車站停止使用時，圍板會蓋上，成為密封的建築物<ref>{{cite web |url=http://www.shikoku.org.uk/JR/LINE/yosan/ST-TSUSHIMANOMIYA_e.jpg |title=存档副本 |accessdate=2013-03-24 |deadurl=yes |archiveurl=https://web.archive.org/web/20140810075607/http://www.shikoku.org.uk/JR/LINE/yosan/ST-TSUSHIMANOMIYA_e.jpg |archivedate=2014-08-10 }} 車站停止營業時的模樣，取自同一個非官方車站介紹。</ref>。

[[file:JRShikoku-Yosan-line-Tsushimanomiya-station-platform-20100804.jpg|thumb]]
本站只設有[[側式月台|側式月台]]1個，闊度比一般月台為長，並與附近的建築物用地相連在一起，因而可提供一個較大的月台空間擺設小攤檔之用，有效長度為4輛。月台邊會臨時標示列車所停靠的位置。列車停靠時，往高松方向為左側開門，往詑間方向則為右側開門。

又由於本站屬臨時車站，並不如其他一般恆常車站般，為對應新型號列車而進行月台加高工程；而車站位處彎位，列車駛經時會輕微往內側傾斜，再加上使用車站乘客的有大量的小童、長者及持[[手推車|手推車]]的家長，所以為顧及安全，所以當車站營業時，會由[[多度津站|多度津站]]或其他車站抽派職員在本站駐守，並安排當列車到達時，每一趟門都會有職員駐守，以協助乘客上落。而所有停靠本站的列車，亦會暫時取消單人駕駛制度並臨時加設車掌協助。

站舍及部份設施乃來自舊[[箕浦站|箕浦站]]站舍的物資。

==歷史==
* 1915年5月7日：以「臨時乘降場」開業（比臨時站的級別再低）。
* 1969年10月1日：臨時站。
* 1987年4月1日：[[日本國鐵分割民營化|日本國鐵分割民營化]]，車站的營運由JR四國繼承。
* 2013年12月22日：為慶祝「予讚線（多度津～觀音寺站）開通100周年記念音樂會in津嶋之宮」於{{link-ja|津嶋神社|津嶋神社 (三豊市)}}舉行，特別開辦臨時列車『故鄉列車「三豐市民號」』來往本站至[[本山站_(香川縣)|本山站]]，另外上、下行合共再停靠開設14班列車，是罕有地在其他時間開站營業<ref>[http://www.jr-shikoku.co.jp/03_news/press/13-11-29/02.htm 「予讃線（多度津駅～観音寺駅間）開通１００周年記念コンサートin津嶋ノ宮（みとよ）」の開催について] {{webarchive|url=https://web.archive.org/web/20140730211015/http://www.jr-shikoku.co.jp/03_news/press/13-11-29/02.htm |date=2014-07-30 }}。JR四國，2013年11月25日。</ref>。
* 2015年5月10日：為紀念車站開業100周年，當日臨時開業一天<ref>[http://www.jr-shikoku.co.jp/03_news/press/15-03-20/02.htm 予讃線「津島ノ宮駅開業１００周年記念イベント」　みとよシーサイドコンサートｉｎ津嶋ノ宮（つしまのみや）の開催について，] {{webarchive|url=https://web.archive.org/web/20150406012601/http://www.jr-shikoku.co.jp/03_news/press/15-03-20/02.htm |date=2015-04-06 }}JR四國，2015年3月20日。</ref>。
