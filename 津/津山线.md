'''津山線'''（{{jpn|j=津山線|hg=つやません|rm=Tsuyama sen|f=y}}）是一條連結[[日本|日本]][[岡山縣|岡山縣]][[岡山市|岡山市]][[北區_(岡山市)|北區]]的[[岡山站_(日本)|岡山站]]至[[津山市|津山市]]的[[津山站|津山站]]，屬於[[西日本旅客鐵道|西日本旅客鐵道]]（JR西日本）的鐵路線（[[地方交通線|地方交通線]]）。

== 概要 ==
津山線和[[吉備線|吉備線]]原本同樣是中國鐵道（現在的[[中鐵巴士|中鐵巴士]]）依照[[私設鐵道法|私設鐵道法]]開業的私鐵，大半是在1944年6月1日因[[戰時買收私鐵|戰時買收]]而國有化的路線。和[[因美線|因美線]]共同構成[[陰陽連絡路線|陰陽連絡線]]，岡山車站 - [[鳥取車站|鳥取車站]]間有急行「砂丘」等之[[優等列車|優等列車]]運轉，但是，[[智頭急行智頭線|智頭急行智頭線]]開業後，津山線作為陰陽連絡線的角色被智頭線的特急列車取代，成為都市間連絡線。全線由岡山支社管轄。

全線由{{link-ja|西日本旅客鐵道岡山支社|西日本旅客鉄道岡山支社|岡山支社}}管轄，2016年3月26日起引入路線顏色與路線記號，路線顏色是山吹色，路線記號是'''T'''<ref>{{Cite web |url=http://www.westjr.co.jp/press/article/2015/09/page_7661.html |title=～お客様によりわかりやすく鉄道をご利用いただくために～岡山・福山エリアの主な路線に「路線記号」「ラインカラー」を導入します |publisher=西日本旅客鉄道 |accessdate=2016-05-27}}</ref><ref>{{Cite web |url=http://www.westjr.co.jp/press/article/2016/01/page_8222.html |title=「吉備線・宇野線・岡山駅出入口の愛称名」ならびに「路線記号・ラインカラー」の使用開始について |publisher=西日本旅客鉄道 |accessdate=2016-05-27}}</ref>。另外岡山站－{{link-ja|法界院站|法界院駅}}間位於[[IC卡|IC]][[乘車卡|乘車卡]]「[[ICOCA|ICOCA]]」的岡山、廣島地域的岡山、福山地區<ref>[http://www.jr-odekake.net/icoca/area/map/okayama.html ご利用可能エリア 岡山・広島エリア｜ICOCA｜ICOCA：JRおでかけネット] - 西日本旅客鉄道</ref>。

== 運行形態 ==
正式的起點是岡山車站，但是，列車運行上津山車站往岡山車站的列車是下行，反方向是上行。因為為了配合在岡山車站接續的[[山陽新幹線|山陽新幹線]]・[[山陽本線|山陽本線]]・[[宇野線|宇野線]]・吉備線和在津山車站接續的[[姫新線|姫新線]]的方向。

=== 廢站 ===
（）內是岡山站起計的營業距離。

* 岡山荷扱所（舊、岡山市站）：1944年廢除、岡山站－法界院站間（0.5公里）
* 博覽會場前臨時停留場：1928年廢除、岡山站－法界院間（約1.2公里）
* 法界院臨時停車場：1908年廢除、岡山站－法界院站間（約3.2公里）
* 南臨時乘降場：1934年廢除、建部站－福渡站間（29.3公里）
* 北臨時乘降場：1934年廢除、建部站－福渡站間（29.8公里）
* 高尾臨時停留場：1937年廢除、佐良山站附近（53.3公里）
