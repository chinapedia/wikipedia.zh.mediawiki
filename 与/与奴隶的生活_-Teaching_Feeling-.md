{{vgsname|《與奴隸的生活 -Teaching Feeling-》|ja=奴隷との生活 -Teaching Feeling-}}是由日本同人社團FreakilyCharming製作的[[養成|養成]]類型[[日本成人遊戲|成人遊戲]]，2015年10月27日於[[DMM.com|DMM.com]]旗下網站DMM同人.R18及[[DLsite.com|DLsite.com]]上以[[數位下載|數位下載]]方式發售。在遊戲中玩家將扮演一名醫生並照顧一名傷痕累累的女孩。

== 系統 ==
《與奴隸的生活 -Teaching Feeling-》是一款美少女養成遊戲，遊戲裡的每一天被劃分為數個階段，玩家可以執行不同的指令與女主角互動，這些指令包括摸頭、上床、泡茶、換衣服與出門，玩家可以透過帶希爾薇出門來取得新衣服和泡茶的材料。每項指令花費的時間不一，而且會提升或降低希爾薇的兩個數值：{{nihongo|親密度|親愛度|}}與{{nihongo|淫亂度|淫乱度}}，隨著數值提升、希爾薇的個性會逐漸外向，並且會觸發一些遊戲事件，並影響希爾薇在進行性行為時的表現<ref name="dlsite販售"/>，在性行為場景中，會以擬音、斷面圖和漫畫形式呈現<ref name="dlsite販售"/>。遊戲並沒有結局。<ref name="dlsite販售"/>

== 故事 ==
曾被主角救過、給人可疑感覺的商人會帶著飽受前主人虐待、渾身是傷的女孩希爾薇（{{jpn|j=シルヴィ}}）拜訪玩家扮演的醫生，之後主角便開始和希爾薇共同生活一段時間<ref name="dlsite販售"/>。

== 製作 ==
遊戲在推出後仍持續釋出更新檔，包括新增新衣服、劇情與修正錯誤，至2018年11月16日為止更新至 2.4.0。<ref name="官方tumblr"/>

== 評價 ==
遊戲在推出之後爆紅，開賣後第17天，銷售量就超越DLsite當年度排行榜上的所有其他遊戲，至開賣後第71天，銷售量達總排行榜第一名。在遊戲受歡迎後推出了抱枕套<ref name="抱枕"/>，此外該社團先前的作品也受到關注，因而使dlsite以半價促銷FreakilyCharming先前的作品<ref name="官方dlsite"/>。
