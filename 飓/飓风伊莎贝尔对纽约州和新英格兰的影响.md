'''飓风伊莎贝尔对纽约州和新英格兰的影响'''相对较小，并且主要局限于风害。[[飓风伊莎贝尔|飓风伊莎贝尔]]于2003年9月6日在[[大西洋|大西洋]]热带海域的[[佛得角|佛得角]]附近由一股[[东风波|东风波]]发展而成。系统朝西北方向前进，进入一片水温较高且[[风切变|风切变]]很弱的环境后稳步加强，于9月11日达到风速每小时270[[公里|公里]]的最高强度。经过四天的强度波动后，飓风逐渐减弱，于9月18日在[[北卡罗莱纳州|北卡罗莱纳州]]的[[外滩群岛|外滩群岛]]以风速每小时165公里的强度[[登陆_(气象学)|登陆]]。风暴登陆后迅速减弱，于次日在[[宾夕法尼亚州|宾夕法尼亚州]]西部转变为[[温带气旋|温带气旋]]。

[[纽约州|纽约州]]受到的损失共计达到9000万美元（2003年美元，相当于{{Formatprice|{{Inflation|US|90000000|2003}}}}{{#time:Y}}年美元）<ref name="tcr">{{cite web |author=Jack Beven & Hugh Cobb |year=2003 |title=Hurricane Isabel Tropical Cyclone Report |publisher=National Hurricane Center |accessdate=2014-02-07 |archiveurl= https://web.archive.org/web/20131114014935/http://www.nhc.noaa.gov/2003isabel.shtml |deadurl=no |archivedate=2013-11-14 |url=http://www.nhc.noaa.gov/2003isabel.shtml?}}</ref>，[[佛蒙特州|佛蒙特州]]也报告了约10万美元损失（2003年美元，相当于{{Formatprice|{{Inflation|US|100000|2003}}}}{{#time:Y}}年美元）。中等强度的大风刮断了该地区的许多供电线路，导致局部地区停电，还有两人因风暴产生的大浪而遇难。

== 防灾措施 ==
飓风伊莎贝尔在北卡罗莱纳州登陆前数小时，[[国家飓风中心|美国国家飓风中心]]向[[新泽西州|新泽西州]]和纽约州边界到[[苏福克县_(纽约州)|苏福克县]][[莫里切斯_(纽约州)|莫里切斯]]（{{lang|en|Moriches}}）之间的[[长岛_(纽约)|长岛]]南部海岸发布了[[热带气旋警告|热带风暴警告]]<ref name="tcr" />。飓风登岸约一天前，气象预测部门估计风暴有10%的可能性会在距[[纽约市|纽约市]]不到120公里处经过。估计[[新英格兰|新英格兰]]南部其他地区也有类似的概率，伊莎贝尔还有3%的可能性会在距[[缅因州|缅因州]]的[[东港_(缅因州)|东港]]不到120公里处经过<ref name="prob45">{{cite web|author=Beven |year=2003 |title=Hurricane Isabel Probabilities 45 |publisher=National Hurricane Center |accessdate=2014-02-15 |archivedate=2013-06-17 |deadurl=no |archiveurl=https://web.archive.org/web/20130617202131/http://www.nhc.noaa.gov/archive/2003/prb/al132003.prblty.045.shtml |url=http://www.nhc.noaa.gov/archive/2003/prb/al132003.prblty.045.shtml? }}</ref>。从飓风吹袭陆地约两天前开始直到其登岸数小时后，气象预测部门都估计风暴将在穿越纽约州西部上空的过程中逐渐转变成[[温带气旋|温带气旋]]，然后以温带系统形式进入[[加拿大|加拿大]]<ref name="archive">{{cite web |author=National Hurricane Center |year=2003 |title=Hurricane Isabel Advisory Archive |accessdate=2014-02-15 |url=http://www.nhc.noaa.gov/archive/2003/ISABEL.shtml |archivedate=2013-10-01 |archiveurl=https://web.archive.org/web/20131001132223/http://www.nhc.noaa.gov/archive/2003/ISABEL.shtml |deadurl=no}}</ref>。

[[纽约州州长|纽约州州长]][[乔治·保陶基|乔治·保陶基]]敦促居民购买紧急救援物资，并给汽车加满[[汽油|汽油]]。纽约州应急管理办公室从风暴登岸前一星期就开始着手准备应对这场飓风。该办公室还发布了第一级应急启动，由策划小组准备好应急预案，并努力协调州内其他部门。纽约州[[美国国民警卫队|国民警卫队]]开始进行前期准备工作，检查能够在紧急情况下动员的人员名单。陆地和空中国民警卫队官员确定紧急情况下需要动用的设备，如[[直升机|直升机]]、[[发电机|发电机]]、高车轴车辆和通讯器材等。州警务人员根据需要制订了人员和设备的应急计划。纽约州公园、休闲和历史保护办事处确保需要的设备都能正常使用，还用沙袋对建筑物进行固定来应对可能出现的洪灾。<ref name="nysdot">{{cite web |author=Office of Media Relations |date=2003-09-16 |title=Governor Pataki Announces State Gearing Up for Isabel |publisher=New York State Department of Transportation |accessdate=2014-02-15 |url=https://www.dot.ny.gov/news/press-releases/2003/2003209}}</ref>七家航空公司允许可能会受到飓风影响的旅客推迟登机日期<ref name="airtravel">{{cite web |author=Air Travel.com |archiveurl=https://web.archive.org/web/20130515054337/http://airtravel.about.com/cs/airtravellinks/a/hurricane.htm |year=2003 |title=Changing Tickets in the Path of a Hurricane |accessdate=2014-02-15 |archivedate=2013-05-15 |deadurl=no |url=http://airtravel.about.com/cs/airtravellinks/a/hurricane.htm}}</ref>。
