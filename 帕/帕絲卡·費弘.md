'''帕絲卡·費弘'''（{{lang-fr|Pascale Ferran}}，{{bd|1960年|4月17日|catIdx=Pascale Ferran}}）是出生於[[巴黎|巴黎]]的法國女性導演、電影劇本家。

== 經歷 ==
個人首次發表的作品為1983年短片《{{lang|fr|Souvenir de Juan-Les-Pins}}》。而帕絲卡·費弘1994年以電影《{{lang|fr|Petits arrangements avec les morts}}》得到當年[[坎城影展|坎城影展]][[金攝影機獎|金攝影機獎]]；<ref>{{cite web|url=http://www.viennale.at/en/films/petits-arrangements-avec-les-morts |title=PETITS ARRANGEMENTS AVEC LES MORTS |work=Vienna International Film Festival |accessdate=2016年4月29日}}</ref>並在2006年推出以改編自英國作家[[大衛·赫伯特·勞倫斯|大衛·赫伯特·勞倫斯]]的著作《[[查泰萊夫人的情人|查泰萊夫人的情人]]》之電影《[[查泰萊夫人|查泰萊夫人]]》後；帕絲卡·費弘奪下該年度[[路易·德呂克獎|路易·德呂克獎]]、<ref>{{cite web|url=http://www.lemonde.fr/cinema/article/2006/12/18/le-prix-louis-delluc-2006-recompense-lady-chatterley-de-pascale-ferran_846937_3476.html |script-title=fr:Le prix Louis-Delluc 2006 récompense "Lady Chatterley", de Pascale Ferran |work=Le Monde.fr |author=Le Monde.fr avec AFP |date=2006年12月18日 |accessdate=2016年4月29日}}</ref>以及2007年[[凱撒電影獎|凱撒電影獎]]中最佳電影、最佳改編劇本等成績。<ref>{{cite web|url=http://en.unifrance.org/movie/25995/lady-chatterley |title=Lady Chatterley |work=UniFrance |accessdate=2016年4月29日}}</ref>

後續帕絲卡·費弘有與荷蘭動畫導演[[麥可·度德威特|麥可·度德威特]]合作，與對方一同替法國製片商Why Not Productions、Wild Bunch、以及日本[[吉卜力工作室|吉卜力工作室]]合力製作的動畫片《[[紅龜|紅龜]]》裡撰寫劇本。<ref>{{cite web|url=http://www.awn.com/news/micha-l-dudok-de-wit-s-red-turtle-set-september-2016-release-date |title=Michaël Dudok de Wit’s 'The Red Turtle' Set for September 2016 Release Date |work=AWN |author=Jennifer Wolfe |accessdate=2015年12月29日}}</ref>
