'''帕布罗·拉拉因'''（Pablo Larraín Matte，1976年8月19日－）是一名[[智利|智利]]电影导演，他已执导了五部电影，并联合执导了一部电视剧。<ref>{{cite web |url=http://www.list.co.uk/article/16867-profile-pablo-larrain/ |title=Pablo Larraín |date=2 April 1009 |author=Kaleem Aftab |work=[[The_List_(magazine)|List.co.uk]] |accessdate=7 February 2013 }}</ref>

== 生平简介 ==
他生于智利首都[[圣地亚哥|圣地亚哥]]。毕业于首都的 University for the Arts, Sciences, and Communication 的传播学专业。

他妻子是女演员[[Antonia_Zegers|Antonia Zegers]]，两人育有2个孩子: Juana (2008)<ref>{{cite web |url=http://www.terra.cl/entretencion/index.cfm?id_cat=2471&id_reg=940700 |title=Se acabó la dulce espera de Antonia Zegers: Fue mamá |work=Terra.cl |date=19 March 2008 |accessdate=7 February 2013 |language=es}}</ref>和 Pascual (2011)。

2012年执导的[[政治|政治]]题材电影《[[向政府說不|向政府說不]]》获得[[第85届奥斯卡金像奖|第85届奥斯卡金像奖]]最佳外语片提名，为首部获得[[奥斯卡最佳外语片|奥斯卡最佳外语片]]提名的智利作品。2015年执导的[[天主教|天主教]]题材电影《[[神父俱乐部|神父俱乐部]]》赢得[[第65届柏林电影节|第65届柏林电影节]]评审团大奖，获得[[第73届金球奖|第73届金球奖]]最佳外语片提名。2016年执导的美国传记片《[[第一夫人的秘密|第一夫人]]》入选[[第73届威尼斯影展|第73届威尼斯影展]]主竞赛片。
