'''弗朗西斯科·馬丁尼茲·羅卡'''（{{lang-es|Francisco Martínez Roca}}，{{bd|1969年||catIdx=Paco Roca}}）是[[西班牙|西班牙]][[瓦倫西亞|瓦倫西亞]]出身；以「'''帕科·羅卡'''」（{{lang|es|Paco Roca}}）為筆名的男性漫畫家、插畫家。<ref name="lam">{{Cite web |url=http://www.lambiek.net/artists/r/roca_paco.htm |title=Paco Roca (b.1969 Spain) |publisher=Lambiek |accessdate=2014年1月28日}}</ref>

== 經歷 ==
在投入漫畫家生涯前、帕科·羅卡於廣告界内工作，並在其中學習美術技巧方法。<ref name="SPC">{{Cite web |url=http://www.spainisculture.com/en/artistas_creadores/paco_roca.html |title=Paco Roca |publisher=Spain Culture |accessdate=2014年1月28日}}</ref>

1994年帕科·羅卡開始將創作的漫畫投稿，起初於《Kiss Comix》及《El Víbora》等兩份偏向成人情色題材內容的雜誌上發表作品。<ref name="SPC"/>隨後帕科·羅卡也嘗試繪製過3D畫面風格的漫畫系列《Road Cartoons》，<ref name="lam"/>和以家鄉西班牙歷史上人事物為題材改編的作品，如將[[薩爾瓦多·達利|薩爾瓦多·達利]]為故事人物的《陰險遊戲》（{{lang|es|El juego lúgubre}}）<ref>{{Cite web |url=http://www.newspanishbooks.com/book/el-juego-lugubre |script-title=es:El juego lúgubre The Sinister Game |publisher=New Spanish books |accessdate=2014年1月28日}}</ref><ref name="lam"/>、或是以[[西班牙內戰|西班牙內戰]]事件作為劇情的《燈塔》（{{lang|es|El Faro}}）等等。<ref name="lam"/>

帕科·羅卡在2007年發表以老年人為題材的漫畫《歡樂皺紋》（{{lang|es|Arrugas}}），該部作品除了獲得2008年西班牙國家漫畫獎外、<ref name="Cine"/>也於2011年則由動畫導演[[伊格納西奧·費雷拉斯|伊格納西奧·費雷拉斯]]改編成[[歡樂皺紋|同名動畫]]，<ref name="Cine"/>而帕科·羅卡亦在該片負責編劇方面處理。
