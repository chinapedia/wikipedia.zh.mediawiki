'''帕特里克·帕特森'''（{{lang-en|'''Patrick Patterson'''}}，{{bd|1989年|3月14日|||catIdx=P}}），[[美国|美国]]职业[[篮球|篮球]][[运动员|运动员]]，出生于美国[[华盛顿特区|华盛顿特区]]，现效力于[[NBA|NBA]]<ref name="虎扑">{{cite web | url=http://www.hoopchina.com/nba/players/patrickpatterson_3462.html| title=帕特里克·帕数据 | publisher =虎扑| accessdate=2010-11-04}}</ref>。

==大学篮球生涯==
帕特里克·帕特森在其大学期间被[[肯塔基大学|肯塔基大学]]教练塔比·史密斯（Tubby Smith）看中，加入了肯塔基大学篮球队。

===菜鸟赛季===
帕特里克·帕特森的第一个[[NCAA|NCAA]]赛季是2007-08赛季，在这个赛季中，他场均可以拿到16.4分、7.7个篮板、1.2次盖帽和0.8次抢断，投篮命中率达到57％。但因帕特森左脚踝[[应力性骨折|应力性骨折]]，他错过了赛季的后半程比赛。赛季结束后，帕特森表示自己将留在肯塔基大学队，不会参加[[NBA选秀|NBA选秀]]<ref name=heralddpatchoice>{{cite web  |title = Patterson's choice a surprise to the end  |publisher = ''The Herald-Dispatch''  |date = 2007-05-18  |url = http://www.herald-dispatch.com/apps/pbcs.dll/article?AID=/20070518/SPORTS02/705180358/1014  |accessdate = 2007-05-24 }}</ref><ref name="Scout.com.">{{cite web|url=http://kentucky.scout.com/a.z?s=48&p=9&c=8&cfg=bb&yr=2007|title=Basketball Recruiting: Kentucky|date=2007|work=Scout.com.|publisher=Scout.com.|accessdate=2009-04-11}}</ref>。

===大二赛季===
在帕特森的大二赛季中，他与队友[[约迪·米克斯|约迪·米克斯]]（Jodie Meeks）成为该赛季最成功的二人组合。2008年12月22日，在对阵田纳西州立大学队的比赛中，帕特森拿到了其职业生涯最高的33分，并带领球队取得了胜利。在这个赛季中，他场均能拿到17.9分、9.3个篮板、2.1次盖帽和0.6次抢断，投篮命中率达到60％。2009年4月18日，帕特森决定参加[[NBA选秀|NBA选秀]]，但一个月后他又反悔，并表示下个赛季不会参加NBA选秀，而是会留在肯塔基大学<ref name="体讯网">{{cite web | url=http://basketball.titan24.com/app/nbadraft/profile.php?id=7 | title=帕特里克·帕资料 | publisher =体讯网| accessdate=2010-11-04}}</ref>。

===大三赛季===
大三赛季初期，由于肯塔基内线球员众多，帕特森曾短暂的尝试打[[小前锋|小前锋]]位置，但控球能力匮乏和缺少面框进攻能力让他曾遭遇批评。不过最终回到大前锋位置的帕特森还是展露了他在进攻端的能力，在这个赛季中，他的三分球命中率达到了40％<ref name="新浪">{{cite web | url=http://sports.sina.com.cn/star/PatrickPatterson/index.shtml | title=新浪篮球明星资料 | publisher =新浪网| accessdate=2010-11-04}}</ref>。2010年4月23日，帕特森宣布参加[[NBA选秀|NBA选秀]]<ref>[http://sports.espn.go.com/espn/page2/story?page=simmons/100625 Action: Draft Diary XIV]</ref>。
