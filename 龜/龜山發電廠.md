'''龜山發電廠'''，又稱為'''龜山發電所'''或'''龜山水力發電所'''，位於[[臺灣|臺灣]][[新北市|新北市]][[新店區|新店區]][[南勢溪|南勢溪]]與[[北勢溪|北勢溪]]匯流處，鄰近[[翡翠水庫|翡翠水庫]]與[[桂山發電廠|桂山發電廠]]，是[[台灣|台灣]]第一座[[水力發電廠|水力發電廠]]<ref name="產業遺產，再生發電─台灣第一座水力發電廠的過去與未來">{{cite web|language=zh-tw|url=http://energymonthly.tier.org.tw/outdatecontent.asp?ReportIssue=200410&Page=5|title=產業遺產，再生發電─台灣第一座水力發電廠的過去與未來|author=趙俊祥|publisher=能源報導|date=2004年10月|accessdate=2013-12-21}}</ref>，也是台灣進入[[電力|電力]]時代的重要史證。

== 沿革 ==
1902年，日本人{{link-ja|土倉龍治郎|土倉龍治郎|土倉龍治郎}}向[[台灣總督府|台灣總督府]]提出於龜山開發水力發電之計畫，成立「台北電氣株式會社」<ref name="龜山水力發電所—台灣第一">{{cite web|language=zh-tw|url=http://pylin.kaishao.idv.tw/?p=121|title=龜山水力發電所—台灣第一|author=林炳炎|publisher=|date= 2007年9月28日|accessdate=2013-12-21}}</ref>。1903年11月改為官營，並另成立「台北電氣作業所」<ref name="產業遺產，再生發電─台灣第一座水力發電廠的過去與未來"/>，同年開始建造<ref name="新店區龜山發電廠">{{cite web|language=zh-tw|url=http://www.boch.gov.tw/boch/frontsite/cultureassets/caseBasicInfoAction.do?method=doViewCaseBasicInfo&caseId=IA09606000010&version=1&assetsClassifyId=1.2&menuId=310|title=新店區龜山發電廠|author=|publisher=行政院文化部文化資產局|date=|accessdate=2013-12-21|deadurl=yes|archiveurl=https://web.archive.org/web/20131224110550/http://www.boch.gov.tw/boch/frontsite/cultureassets/caseBasicInfoAction.do?method=doViewCaseBasicInfo&caseId=IA09606000010&version=1&assetsClassifyId=1.2&menuId=310|archivedate=2013-12-24}}</ref>。1904年底，龜山發電所的土木工程與室內輸電線路完工，1905年7月工程正式完工<ref name="新店區龜山發電廠"/>，並架設龜山至台北間電壓11kV、長16.5公里的輸電線路，送至古亭變電所降壓為100V後送進台北的用戶端<ref name="產業遺產，再生發電─台灣第一座水力發電廠的過去與未來"/>。1905年8月17日，台北三市街（[[台北城|台北城]]、[[大稻埕|大稻埕]]、[[艋舺|艋舺]]）開始試送電，9月11日漸增加點燈，10月15日三市街皆大放光明<ref name="龜山水力發電所—台灣第一"/>，台北成為台灣第一個有電燈的城市<ref name="百年滄桑 全台首座水力發電所倒塌"/>。1913年為了避免水患而改建，加高地基並強化結構<ref name="老龜山發電所">{{cite web|language=zh-tw|url=http://info.taipower.com.tw/TaipowerWeb//upload/files/10/539_35-39.pdf|title=老龜山發電所|author=|publisher=台電月刊539期|date=2007年11月|accessdate=2013-12-21|deadurl=yes|archiveurl=https://web.archive.org/web/20131224105740/http://info.taipower.com.tw/TaipowerWeb//upload/files/10/539_35-39.pdf|archivedate=2013-12-24}}</ref>。1943年龜山發電所撤廢停止運轉，由1939年興工、1941年竣工之「新龜山發電所」（今[[桂山發電廠|桂山發電廠]]桂山機組）正式取代龜山發電所，而龜山發電所3部發電機組拆除後，其中2號機移至[[臺東縣|臺東]][[大南發電所|大南發電所]]（今[[東興發電廠|東興發電廠]]）繼續運轉發電<ref name="產業遺產，再生發電─台灣第一座水力發電廠的過去與未來"/>。1961年電廠建築登記為[[台灣電力公司|台灣電力公司]]，1968年台電以無業務需求標售發電所後，目前由私人取得所有權<ref name="新店區龜山發電廠"/><ref name="百年滄桑 全台首座水力發電所倒塌"/>。2006年，經台北縣文史學會爭取，登錄為台北縣歷史建築（今為[[新北市文化資產列表|新北市歷史建築]]）。

== 設施 ==
龜山發電廠採用川流式發電，自今龜山-{里}-栗子園處築壩取[[南勢溪|南勢溪]]水，鑿四段隧道，以明渠暗渠接續共2.5公里水路，輸水至電廠後方前池接壓力鋼管，再利用落差能量帶動電廠內[[水輪機|水輪機]]發電，並經尾水道排入[[新店溪|新店溪]]<ref name="產業遺產，再生發電─台灣第一座水力發電廠的過去與未來"/>。壓力鋼管直徑1.4公尺，由原本的二支增為三支供水輪機使用，另有一支直徑0.48公尺的壓力鋼管供勵磁機水車用<ref name="產業遺產，再生發電─台灣第一座水力發電廠的過去與未來"/>，最大引水量為每秒6.954立方公尺，有效落差14.85公尺<ref name="產業遺產，再生發電─台灣第一座水力發電廠的過去與未來"/>。水輪機組為橫軸二輪（單流）前口雙子[[法蘭西斯水輪機|法蘭西斯式水車]]，[[發電機|發電機]]為三相交流，出力250kVA。原裝置二部機組，1907年再增設一部機組<ref name="新店區龜山發電廠"/>，原裝置容量為600kW，1930年擴充為750kW（0.75[[MW|MW]]）<ref name="產業遺產，再生發電─台灣第一座水力發電廠的過去與未來"/>。電廠主建築物的屋頂[[山牆|山牆]]有[[台灣電力株式會社|台灣電力株式會社]]之正倒三角形「台」字標誌及電光紋路，其中「台」字代表台北電氣作業所，而電紋則是山牆上左右兩個曲折的斜邊，象徵[[閃電|閃電]]<ref name="百年滄桑 全台首座水力發電所倒塌"/>。攔水壩則為台灣首座鋼筋[[混凝土|混凝土]]構造物<ref name="新店區龜山發電廠"/><ref name="百年滄桑 全台首座水力發電所倒塌"/>。

== 現況 ==
2012年3月14日傍晚時分，龜山水力發電所由於廠房老舊，屋頂桁架鏽蝕嚴重，不堪近百年的負荷突然倒塌，現場散落一塊塊磚牆，僅剩墊高地基的基座牆<ref name="百年滄桑 全台首座水力發電所倒塌">{{cite web|language=zh-tw|url=http://www.libertytimes.com.tw/2012/new/mar/26/today-life10.htm|title=百年滄桑 全台首座水力發電所倒塌|author=記者潘杏惠／新北報導|publisher=[[自由電子報|自由電子報]]|date=2012-03-26|accessdate=2013-12-21}}</ref>。
