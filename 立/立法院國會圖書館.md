'''立法院國會圖書館'''，是[[中華民國立法院|中華民國立法院]]附屬的[[圖書館|圖書館]]，位於[[台灣|台灣]][[台北市|台北市]][[中正區_(臺北市)|中正區]][[中山南路|中山南路]]一號。由於立法院本身空間狹小，因此該館無法擁有獨立的館-{舍}-，其[[閱覽室|閱覽室]]分布三個樓層。其服務對象主要為[[立法委員|立法委員]]、國會助理以及其他[[政府機關|政府機關]]、[[學術團體|學術團體]]、國會[[記者|記者]]等。

== 沿革 ==
立法院國會圖書館前身是'''立法院秘書處圖書資料室'''，成立於1999年1月。於該年修正的「國會五大改革法案」裡，於《[[:s:立法院組織法|立法院組織法]]》第十五條將圖書資料室提升為國會圖書館。

== 掌理事項 ==
根據《立法院組織法》第二十二條規定，國會圖書館掌理下列事項：
#關於立法[[書刊|書刊]][[光碟|光碟]]資料之蒐集、管理及運用事項
#關於立法報章資料之蒐集、管理及運用事項
#關於立法資料之分析、研究、檢索及參考事項
#關於立法[[出版品|出版品]]之編纂及交換事項
#關於國會圖書館館際合作事項
#其他有關圖書館研究、發展及服務事項。

== 各項功能與特色 ==
立法院國會圖書館的立法院[[智庫|智庫]]，是立法院圖書館自行研發的一系列[[資訊系統|資訊系統]]，是立法知識管理的具體呈現，可追蹤法案、掌握最新立法動態與國會相關資訊，可以分享即時性、便捷性與公開化的立法資訊。

以傳統圖書館角色而言，立法院國會圖書館提供服務圖書文獻、期刊資料、報紙、[[政府公報|政府公報]]、新聞資料、[[參考工具書|參考工具書]]、立法院公報、立法院議事日程、法律政治類書籍、中外文法規等資料。

立法院國會圖書館的網站於1999年9月開始架設，其內容包括十大項：立法資源、立委問政、焦點議題、國會新聞、館藏資源、立法智庫、數位資訊、線上服務、參考指南、隨選訂閱等。自2002年改版之後的網站增加[[英文|英文]]版，其內容有：國會資訊、國會圖書館介紹、組織與架構、核心服務、網站地圖等。

== 國會圖書館的出版品==
國會圖書館所發行的出版品有《立法院國會圖書館服務簡介》、《國會圖書館館訊》、《立法報章資料專輯》、《法律沿革全書補編》、《立法報章資料專題索引》等。
