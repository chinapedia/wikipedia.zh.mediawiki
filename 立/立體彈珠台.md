'''Full Tilt! Pinball'''（常稱'''Full Tilt!'''）是一款[[電腦|電腦]][[視訊|視訊]][[彈珠臺|彈珠臺]][[遊戲|遊戲]]，由[[Cinematronics_Games|Cinematronics]][[工作室|工作室]]開發，由[[Maxis|Maxis]][[公司|公司]]於1995年發行。這款彈珠臺電腦遊戲中共有3張「彈珠臺」，分別名為「太空軍校生」（“Space Cadet”）、「Skulduggery」以及「Dragon's Keep」。當時絕大部分的同類型的電腦遊戲都是採用[[二維|二維]]畫面渲染。而Full Tilt! Pinball則是採用[[三維|三維]]預渲染繪圖技術，來「構造」三張不同的「彈珠臺」，有別於其它模式類似的電腦遊戲，在同類遊戲中尚屬首次。每張「彈珠臺」的右側欄還會展示玩家的得分、彈珠數量、玩家數量等資訊以及「彈珠臺」的[[縮圖|縮圖]]。
在[[Windows_8|Windows 8]]的[[Windows_Store|Windows Store]]中，有一款类似于「彈珠臺」的游戏，名称叫[[Pinball_FX2|Pinball FX2]]。

== 彈珠臺 ==

=== 太空軍校生（Space Cadet） ===

在「太空軍校生」彈珠臺上，玩家扮演一定數量的[[太空|太空]][[艦隊|艦隊]]，通過完成一定數量的任務來提升玩家的階級。玩家級別從低到高被劃分為9個級別，分別是：見習軍校生（Cadet）、海軍[[少尉|少尉]]（Ensign）、海軍[[中尉|中尉]]（Lieutenant）、[[艦長|艦長]]（Captain）、[[少校|少校]]（Lieutenant Commander）、海軍指揮官（Commander）、海軍[[準將|準將]]（Commodore）、艦隊[[司令|司令]]（Admiral）、[[五星上將|五星上將]]（Fleet Admiral）。玩家擊球命中「任務目標」來選擇任務，然後進入「發射坡道」將彈珠發射出去。每個任務會要求玩家達成一些目標，例如「attack bumpers」（共一組四隻柱塞位於彈珠臺的上部）8次（「target practice」任務）。當目標達成，或發射坡道的指示燈（如「燃料指示燈」）全部亮起時，任務結束。在一定時間以後，「燃料指示燈」將關閉，並且可以在彈珠經過某個指示燈後重新點亮，當彈珠重新發射時全部指示燈點亮。接近完成任務時，一些在彈珠臺中部以環形排列的藍色指示燈（圓心是黑洞）會被點亮。當環形中全部藍色指示燈全部亮起時，玩家級別晉升，一個在藍色指示燈圓環內的橙色指示燈圓環中的一個指示燈會被點亮。

該遊戲在台灣當年由[[智冠科技|智冠科技]]代理並引進，譯名為「歡樂彈珠台」。

「太空軍校生」彈珠臺後來獨立發展為另一個彈珠臺遊戲，即後來的「立體彈珠台-太空軍校生」，內建於[[Microsoft_Windows_NT_4.0|Microsoft Windows NT 4.0]]、[[Microsoft_Windows_2000|Microsoft Windows 2000]]、[[Microsoft_Windows_ME|Microsoft Windows ME]]、[[Microsoft_Windows_XP|Microsoft Windows XP]]中，可以在以上[[作業系統|作業系統]]的「開始」功能表上找到。

=== Skulduggery ===

[[Image:Skulduggery.png|thumb]]
「Skulduggery」彈珠臺中，玩家扮演一名寶藏[[獵人|獵人]]，尋找Peg Leg的[[戰利品|戰利品]]。玩家可以通過以下兩種方法達成目標：通過拼湊出一張[[藏寶圖|藏寶圖]]或通過啟動並完成彈珠臺上一系列的迷你遊戲模式。後者類似於另外兩套彈珠臺的「任務」設定。這些迷你遊戲模式以[[海盜|海盜]]為主題，例如船戰、酒館鬥毆、逃離[[百慕大三角|百慕大三角]]、叛變以及劍鬥。

=== Dragon's Keep ===

「Dragon's Keep」彈珠臺中，玩家需在「奇幻境界」中完成許多探索任務，以將[[怪獸|怪獸]]殺死。這些任務包括「Dragon Hoard」、「Fire Lizard Attack」、「Rescue Maiden」、「Dragon Pass」、「Wizard's Fury」以及「Slay Dragon」。玩家在遊戲時可以獲得[[咒語|咒語]]、[[武器|武器]]和[[裝甲|裝甲]]。所謂的武器是為玩家增加遊戲得分，裝甲和咒語可以使一些特殊的[[閘門|閘門]]、[[磁鐵|磁鐵]]以及斜槽開啟一段時間來改變遊戲。

'''3D Pinball for Windows – Space Cadet'''，或者稱'''3D Pinball'''、'''Pinball'''，是微軟Windows作業系統小遊戲元件中的一員，微軟官方中文名：[[正體中文|繁體]]「-{'''立體彈珠台-太空軍校生'''}-」或簡稱「-{'''立體彈珠台'''}-」，而[[简体中文|簡體]]為「-{'''三维弹球-太空军校生'''}-」或簡稱「-{'''三维弹球'''}-」。最早出現在Windows作業系統主題擴展包[[Windows_Plus!|Windows Plus! 95]]中，後來的[[Windows_NT_4.0|Windows NT 4.0]]、[[Windows_ME|Windows ME]]、[[Windows_2000|Windows 2000]]、[[Windows_XP|Windows XP]]（包括32位元版本和64位元版本的[[Windows_XP_Professional_x64_Edition|Windows XP Professional x64 Edition]]）中內建於作業系統中。在[[Windows_98|Windows 98]]中微軟也提供「立體彈珠臺」的安裝指導教程。<ref>{{cite web|url=http://support.microsoft.com/kb/188204|title=How to Install 3D Pinball Using Windows 98 CD-ROM|date=2007-01-23|work=Help and Support |publisher=Microsoft|accessdate=2008-06-18}}</ref>然而後來的Windows XP 64位元版本以及[[Windows_Vista|Windows Vista]]（Beta 1）開始，這個小遊戲由於原始碼的[[相容性|相容性]]問題、[[BUG|BUG]]以及微軟與Cinematronics失去聯繫，而從Windows系統的元件中移除了。<ref>[http://blogs.msdn.com/b/oldnewthing/archive/2012/12/18/10378851.aspx Why was Pinball removed from Windows Vista?]</ref><ref>[http://vga.zol.com.cn/342/3428914.html win7小秘密 微软解答"弹珠-{台}-"消失原因] - zol.com.cn</ref>

Full Tilt! Pinball的原始碼中已包含大量的[[x86|x86]][[組合語言|組合語言]]，要移植這個遊戲到當年微軟Windows的NT系列操作系统仍然支援的[[MIPS架構|MIPS架構]]、[[DEC_Alpha|Alpha架構]]以及[[PowerPC|PowerPC架構]]上的话，需要以[[C語言|C語言]]來重寫程式码。成為Windows元件以後，不但只有「太空軍校生」這張彈珠臺被保留，名称有所变更以外，遊戲的最大[[顯示解析度|顯示解析度]]也只有640×480像素，原Full Tilt! Pinball可以支援640×480像素、800×600像素以及1024×768像素三種不同的顯示解析度。遊戲畫面右邊栏的弹珠台缩略图也只是二維圖像而不是預渲染的三維圖像。遊戲[[開發商|開發商]]和發行商的名稱「Maxis」和「Cinematronics」雖得到完整保留，但字體顏色從原來的[[黃色|黃色]]改為[[深紅色|深紅色]]，不加注意是很容易被忽略的。另外遊戲開始時會有「3D Pinball」字樣和彈珠柱塞略縮圖的全熒幕啟動畫面（Flash Screen）。

「立體彈珠台-太空軍校生」和原版「Full Tilt! Pinball」相比十分相似，但原版擁有更流暢的過場動畫、更多的遊戲音樂、更多的熒幕解析度可供選擇。Windows版本的「立體彈珠台」預設只有遊戲聲音而沒有[[遊戲音樂|遊戲音樂]]。Full Tilt! Pinball原版的附帶回放功能且顯示色彩豐富的「光榮榜」，在Windows系統內建的版本上變成了顯示十分簡陋的得分排行榜。完成任務以及獲得加分時彈珠臺的彩燈閃耀效果，以及回放功能也一併遭「[[閹割|閹割]]」。其它的不同還有柱塞的黃色指示燈：當彈珠經過後原本熄滅的會點亮，反之亦然，這個特性是Full Tilt! Pinball原版所沒有的，Full Tilt! Pinball原版是一直點亮的。Windows內建的版本中「[[燃料|燃料]]」的使用時長比原版的要更長。遊戲特效聲音上，Windows內建的版本和Full Tilt! Pinball原版的相近，但不同的是Windows內建的版本使用了[[MIDI|MIDI]]來製作遊戲特效聲音。
