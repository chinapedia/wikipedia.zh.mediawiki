'''立花氏'''（日語漢字：{{lang|ja|立花氏}}，假名：{{lang|ja|たちばなし}}，羅馬字：''Tachibanashi''）是[[大友氏|大友氏]]分流的[[武家|武家]]。與在[[日文|日文]]讀音上相同的[[橘氏|橘氏]]沒有關係。

== 出身 ==
在[[日本南北朝時代|日本南北朝時代]]，[[大友貞宗|大友貞宗]]的兒子[[大友貞載|大友貞載]]以[[筑前國|筑前國]][[糟屋郡|糟屋郡]][[立花城|立花城]]為據點並稱立花為開始。以後成為大友氏的重臣，但是在[[立花鑑載|立花鑑載]]一代因為背叛了[[大友義鎮|大友義鎮]]（宗麟）而被同樣是大友氏支流的[[戶次鑑連|戶次鑑連]]（立花道雪）攻滅，於是在鑑載的兒子[[立花親善|立花親善]]一代斷絕。

== 戶次鑑連（立花道雪）和誾千代 ==
[[大友氏|大友氏]]的一族[[大友義鎮|大友義鎮]]令[[戶次鑑連|戶次鑑連]]代替立花鑑載・親善入主[[立花城|立花城]]並繼承立花氏的名跡（鑑連後來入道並號道雪）。因為鑑連自身是主家大友家的關係而被禁止使用立花姓，因此沒有以立花姓自稱，不過以立花道雪的名稱而為人所知。而由於道雪沒有兒子，於是把立花城讓給女兒[[立花誾千代|誾千代]]，道雪在晩年讓同為大友氏庶流的[[高橋紹運|高橋紹運]]的兒子[[高橋統虎|統虎]]迎娶誾千代並收統虎為養子。統虎之後改名為立花宗茂並在支撐衰退後的大友氏而與[[島津氏|島津氏]]的戰鬥中相當活躍，在[[豐臣秀吉|豐臣秀吉]]的[[九州征伐|九州征伐]]後被賜予[[筑後國|筑後國]]柳川13萬2千石。

== 立花宗茂及柳川藩主家立花氏 ==
不過[[立花宗茂|立花宗茂]]因為在[[關原之戰|關原之戰]]中加入西軍而被沒收領地並成為[[浪人|浪人]]，後來在[[慶長|慶長]]9年（1604年）被[[德川氏|德川氏]]收留，在慶長11年（1606年）被賜予[[陸奥國|陸奥國]]棚倉1萬石而復歸[[大名|大名]]身份。後來宗茂在[[大坂之役|大坂之役]]中立下戰功，在[[元和|元和]]6年（1620年）因為在關原之戰以來支配筑後柳川32萬石的[[田中氏|田中氏]]絶家而被賜予[[柳川藩|柳川藩]]10萬9千石，於是返回舊領柳川。在關原之戰後被改易的武將可以再度成為「大名」的例子相當少，其中能夠復歸舊領的就只有立花宗茂。

[[江戶時代|江戶時代]]的立花氏作為柳川藩主並一直持續到[[明治維新|明治維新]]，在維新後被列為[[華族|華族]]並授予[[伯爵|伯爵]]。現在的立花家在[[柳川市|柳川市]]改装自邸[[立花氏庭園|立花氏庭園]]（松濤園）並經營著料亭・旅館・宴會場「御花」｡

而第4代柳川藩主[[立花鑑任|立花鑑任]]的同母弟弟[[立花貞晟|立花貞晟]]（彈正）成為[[旗本寄合席|旗本寄合席]]並分立出'''旗本立花彈正家'''，不過在第2代的養嗣子[[立花清直|清直（後來的貞俶）]]繼承藩主家而絕家，兩家再度合而為一。

== 高橋統增及三池藩主家立花氏 ==
還有[[立花宗茂|立花宗茂]]的實弟‧繼承實家高橋家的[[高橋統增|高橋統增]]（直次）在[[豐臣政權|豐臣政權]]下領有1萬8千石，但是在[[關原之戰|關原之戰]]中與兄長宗茂一同加入西軍而失去領地，後來被[[德川氏|德川氏]]賜予5千石。直次亦跟隨兄長改稱立花氏。

之後在立花直次的兒子[[立花種次|種次]]一代被賜予[[三池藩|三池藩]]1萬石而復歸成[[大名|大名]]。'''三池藩立花氏'''在[[立花種周|立花種周]]一代成為[[外樣大名|外樣大名]]和[[若年寄|若年寄]]，但是後來失勢而被轉封[[陸奥國|陸奥國]]內5千石並降格為[[旗本|旗本]]。後來被加增5千石而復歸大名，在[[慶應|慶應]]4年（1868年）再度返回三池。此家在維新後被授予[[子爵|子爵]]。而末裔之一是被稱為日本農政之祖的[[加納久宜|加納久宜]]，子孫有[[麻生太郎|麻生太郎]]（[[日本|日本]]前[[總理大臣|總理大臣]]）和[[科幻小說|科幻小說]]作家[[野田昌宏|野田昌宏]]。

而直次的次男[[立花種吉|種吉]]及[[立花種長|種長]]的次男[[立花種澄|種澄]]則成為[[江戶幕府|江戶幕府]]的旗本。
