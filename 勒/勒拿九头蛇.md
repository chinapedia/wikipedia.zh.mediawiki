'''勒拿九头蛇'''（{{lang-el|'''Λερναῖα Ὕδρα'''}}，{{lang-en|'''Lernaean Hydra'''}}，义为“[[海蛇|水蛇]]”），又稱'''海卓拉、许德拉'''，是古[[希腊神话|希腊神话]]裡一种传说中有九个头的大蛇，存在於印歐的许多子嗣文化當中。

== 外貌 ==
古希腊的九头蛇也称為'''勒拿九头蛇'''（Lernaean Hydra或 Hydra of Lerna），因傳說中其所棲身的沼泽位在[[勒拿湖|勒拿湖]]附近而得名。它是体形硕大的水蛇之属，关于其头的个数，说法不一。[[保薩尼亞斯_(地理學家)|保薩尼亞斯]]（Pausanias）认为只有一个，[[西蒙尼特斯|西蒙尼特斯]]（Simonides）说有五个，[[托普塞|托普塞]]（Edward Topsell）增加至七个，到了[[狄奥多罗斯|狄奥多罗斯]]（Diodorus Siculus）的版本中甚至多达一百个。但大部分典籍都以九头为准，尤其指出最中心的那个头刀枪不入（也有说是金色的）。

== 家世 ==
九头蛇的[[家谱|家谱]]有两个版本，一说它为[[堤丰|堤丰]]和[[厄客德娜|厄客德娜]]（Echidna）所生，如此它便是[[刻耳柏洛斯|刻耳柏洛斯]]、[[客迈拉|客迈拉]]、[[涅墨亚狮子|涅墨亚狮子]]、甚至[[哈耳庇厄|哈耳庇厄]]的姐妹（九头蛇为雌性）。另一说它的父母是[[泰坦巨人|泰坦巨人]][[帕拉斯|帕拉斯]]（Pallas）与[[冥河|冥河]]女神[[斯堤克斯|斯堤克斯]]（Styx）。那样一来，它的兄弟姐妹就包括了[[宙斯|宙斯]]的扈从[[泽洛斯|泽洛斯]]、[[胜利女神|胜利女神]][[尼刻|尼刻]]、还有[[战神|战神]]的两员大将[[克拉托斯|克拉托斯]]与[[比亚|比亚]]。

== 九头蛇与海格力斯 ==
[[File:Antonio_del_Pollaiolo_-_Ercole_e_l'Idra_e_Ercole_e_Anteo_-_Google_Art_Project.jpg|left]]作]]
九头蛇因[[海格力斯|海格力斯]]而出名。那位[[英雄|英雄]]十二大伟绩中的第二件便是杀死这[[妖魔|妖魔]]。九头蛇剧毒无比，[[沼泽|沼泽]]的水也因此长年恶臭，四周土地变为褐色。它还经常吞食过往行人牲口，为害一方。赫拉克勒斯与侄子[[伊奥劳斯|伊奥劳斯]]（Iolaus）驾马车来到沼泽。他命伊奥劳斯看住车辆，自己用火箭将九头蛇引出。双方展开大战，每当英雄用剑（一说為镰刀）砍掉那魔怪的一个头，另一个新头（一说两个）又重新生出。海格力斯于是喊侄子来帮忙。伊奥劳斯用火炬烧灼断颈，头便无法重生。最后海格力斯将八头一一砍下，又用巨棍打落正中的主头，埋于土中，用大石压住，才算铲除这一祸害。后来宙斯为纪念海格力斯的丰功伟绩，将被他杀死的九头蛇升到空中，变成了[[长蛇座|长蛇座]]。

根据其它版本所述，双方搏斗正酣之际，[[赫拉|赫拉]]派来一只巨蟹来协助魔物。巨蟹钳了海格力斯一下，旋即被英雄杀死。这只巨蟹成为了天空中[[巨蟹座|巨蟹座]]的来源。公元前二世纪古勒拿的石版上也刻画有这一场景。

九头蛇死后，海格力斯将箭头-{zh-hans:喂; zh-hant:餵;}-上其胆部的毒汁。这些毒箭后来误伤了[[人马|人马]][[喀戎|喀戎]]。喀戎是不死之身，但蛇毒入骨，疼痛难当，他便放弃了永生，宁愿一死，飞上天空成了[[人马座|人马座]]。毒箭还埋下了海格力斯自己的死因。他完成十二项任务后，携娇妻回家安顿。路上用毒箭射死了另一人马[[涅索斯|涅索斯]]（Nessos）。人马死前诓骗海格力斯的妻子[[德伊阿妮拉|德伊阿妮拉]]（Deianeira），说自己的血是催生爱情的圣药，涂抹一点在衣服上，给谁穿上，谁便会一辈子忠贞不渝。当时有谣言说海格力斯将迎娶[[罗尔|罗尔]]（Lole）。为了挽回丈夫的心，得伊阿尼拉便照人马所言行事。海格力斯穿上妻子所赠的新衣，九头蛇的毒液便通过涅索斯的血渗入肌肤。他想扯下衣服，但已与皮肉粘连，再无法分开。一代英雄，就此殒命。这段掌故在《[[变形记_(奥维德)|变形记]]》中记载得最为详尽生动。

== 其它传说 ==
传说九头蛇死后与地狱犬一起为[[冥王|冥王]]看守门户。《[[伊利亚特|伊利亚特]]》便这样写道：「（[[哈得斯|哈得斯]]的）大门下盘踞着不少魔物……勒拿九头蛇嘶嘶作声。」

希腊[[厄利斯|厄利斯]]城（Elis）附近的[[阿尼格罗斯河|阿尼格罗斯河]]（Anigros River）污浊不堪，水族难存。据说便是因为涅索斯中了毒箭，在此洗濯伤口，令得河水受蛇毒污染所致。
