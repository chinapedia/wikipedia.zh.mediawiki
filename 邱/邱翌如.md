'''邱翌如'''（{{lang|en|'''CHIU Yi Ju'''}}，{{bd|1984年|3月19日|||catIdx=Chiu邱}}），為[[台灣|台灣]]女子[[羽球|羽球]]運動員，畢業於[[國立台灣師範大學|國立台灣師範大學]]運動競技學系<ref>{{zh-tw}}{{cite web|title=APAR適應體育研究群 - 研究群成員：邱翌如|url=http://apar.ntupes.edu.tw/IRyu.htm|agency=國立臺灣體育運動大學競技運動研究所|accessdate=2017-07-03|deadurl=yes|archiveurl=https://web.archive.org/web/20170703011718/http://apar.ntupes.edu.tw/IRyu.htm|archivedate=2017-07-03}}</ref>，現為台中市豐原國中體育組長兼羽毛球教練<ref>{{zh-tw}}{{Cite web |url = http://www.fyjh.tc.edu.tw/web02/index.php?id=19 |title = 葫蘆墩創始中學----豐原國中教育紀要 |author =  |date = 2014-12-25 |publisher = 台中市立豐原國民中學 |accessdate = 2016-07-10 |quote = 後記：已逾一甲子的老學校...；體育組邱翌如組長執筆體育團隊部份；... |archive-url = https://web.archive.org/web/20160726144233/http://www.fyjh.tc.edu.tw/web02/index.php?id=19 |archive-date = 2016-07-26 |dead-url = yes }}</ref><ref>{{zh-tw}}{{cite news |author = 黃秀仁 |coauthors =  |url = http://www.ltsports.com.tw/other/complex/110325-u15-u17-11 |title = 羽球》亞青U15、U17國手選拔賽11日西螺東南國中燃戰火 |publisher = 麗台運動報 |pages =  |date = 2015-04-11 |accessdate = 2016-07-10}}</ref>。

== 簡歷 ==
2004年12月5日，邱翌如在[[2004年中華台北羽球公開賽|中華台北羽球公開賽]]女子單打項目決賽中，以0比2（9-11、8-11）不敵日本的[[梅津知惠|梅津知惠]]，屈居亞軍，仍寫下台灣選手在該賽事女單項目的最佳成績<ref>{{zh-tw}}{{cite news |author = 劉忠杰 |coauthors =  |url = http://www.appledaily.com.tw/appledaily/article/property/20041206/1431261/ |title = 心急誤事 邱翌如亞軍 |publisher = 蘋果日報 |pages =  |date = 2004-12-06 |accessdate = 2016-07-10}}</ref>。

2012年10月2日，邱翌如在[[2012年中華台北羽球公開賽|中華台北羽球公開賽]]會外賽中以2比0（21-17、21-14）擊敗另一名台灣選手[[宋碩芸|宋碩芸]]，連續九年從該賽事的會外賽晉級會內賽<ref>{{zh-tw}}{{cite news |author = 劉長鑫 |coauthors =  |url = http://www.tsna.com.tw/index.php?q=node/35436 |archiveurl = https://archive.fo/20160710060748/http://www.tsna.com.tw/index.php?q=node/35436 |title = 〈台北羽球公開賽〉勇哉邱翌如　連9年從會外晉級 |publisher = TSNA |pages =  |date = 2012-10-02 |archivedate = 2016-07-10 |deadurl = yes }}</ref>。
