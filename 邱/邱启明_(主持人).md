== 简介 ==
1971年，邱启明出生于出生于[[江苏省|江苏省]][[宿迁市|宿迁市]]。<ref name="irs" />早年邱启明在[[南京电视台|南京电视台]]担任主持人，被誉为“南京十大名嘴”。<ref name="irs" />

2005年，邱启明受邀到[[上海|上海]][[东方卫视|东方卫视]]担任主持人。<ref name="rmws" />

2009年8月10日，邱启明担任[[中国中央电视台|中国中央电视台]]《[[24小时|24小时]]》栏目主持人。<ref name="irs" />并于同年通过[[优秀人才入境计划|优秀人才入境计划]]获得[[香港身份证|香港身份证]]。<ref name="搜狐"/>

2011年8月7日，邱启明和[[柴静|柴静]]搭档担任[[中国中央电视台|中国中央电视台]]《[[看见|看见]]》栏目主持人。<ref name="irs" />

2012年3月23日，邱启明在[[新浪微博|新浪微博]]上怒斥央视制片人是傀儡，随后在接受《[[南方人物周刊|南方人物周刊]]》媒体的采访时暗示离职已经成为定局，邱启明说：“[[体制|体制]]不能硬碰硬，只能匍匐前进。”，“我永远不跟风，不做墙头草。我的精神领袖是[[陈少敏|陈少敏]]（1968年第八届十二中全会上，唯一一位不举手表决开除[[刘少奇|刘少奇]]党籍的人），独立思想、独立作为、爱土地、爱国家，在太多大是大非的问题上她坚信自己的信念。”<ref>{{cite web |url=http://wei.sohu.com/20120429/n341999319.shtml?pvid=tc_news&a=&b=%E9%82%B1%E5%90%AF%E6%98%8E%E8%B0%88%E5%87%BA%E8%B5%B0%E5%A4%AE%E8%A7%86%EF%BC%9A%E6%B0%B8%E8%BF%9C%E4%B8%8D%E5%81%9A%E5%A2%99%E5%A4%B4%E8%8D%89%20%E8%A6%81%E6%91%B8%E7%9D%80%E8%89%AF%E5%BF%83%E5%81%9A%E4%BA%8B |title=邱启明谈出走央视:要摸着良心回报一些正的东西 |publisher= 搜狐网|accessdate= 2012年4月29日|work= |language =}}</ref>

2012年6月，邱启明离开央视后，加盟湖南卫视，主持《我们约会吧》。2013年5月9日入职[[亚洲联合卫视|亚洲联合卫视]]担任副台长及新闻主播。2014年11月5日离开亚洲联合卫视，出任[[搜狐|搜狐]]新闻财经中心[[制片人|制片人]]及[[总监|总监]]<ref name="搜狐"/>。同年年底参加明星舞蹈真人秀《[[与星共舞中国版|与星共舞]]》。

== 个人生活 ==
邱启明的妻子名叫[[李菡_(主持人)|李菡]]，2004年，两人在[[南京电视台|南京电视台]]工作时互相认识。<ref name="rmws" />2004年3月13日，在《20世纪经典演唱会》节目上，邱启明向观众介绍了李菡的简历，介绍词中说“将有一个酷似[[王小丫|王小丫]]，但比[[王小丫|王小丫]]更有气质，更骨感的女主持（李菡）”，两人在主持节目时，邱启明大声呼喊“小李菡，小李菡”，两人关系由此拉近和升温。<ref name="rmws" />2005年，两人受邀去[[上海|上海]][[东方卫视|东方卫视]]工作，在[[上海|上海]]后，两人关系进展迅速。<ref name="rmws" />2007年12月8日，两人走入婚姻的殿堂。<ref name="rmws" />2009年，李菡生下一个男孩叫“邱乘木”。<ref name="rmws" />

== 风格 ==
邱启明为人正直善良，主持风格犀利而外表俊朗，因而被誉为“央视最帅新闻主播”。<ref name="sa" />

== 评价 ==
央视男主持[[朱军|朱军]]对邱启明又赞又讽：“他不存在[[跳槽|跳槽]]的情况，因为邱启明从来没有真正加入央视主持人的行列。我很欣赏邱启明，他是非常优秀的主持人，他有自己的想法，在个性张扬的时代应该有这样的主持人。”，同时拿自己和邱启明对比说，“这个平台是央视给我的。20年前，从[[西北|西北]]闯到北京，我没有根基，因为央视全国人民都认识我了，要是今天拿跳槽去要挟东家的话，这非常不仗义。”<ref>{{cite web |url= http://ent.ifeng.com/idolnews/mainland/detail_2012_06/17/15357175_0.shtml|title=朱军谈主持人邱启明跳槽：他从没真正加入央视 |accessdate=2012年6月17日  |publisher=凤凰网 |work= |language =zh-cn}}</ref>
