'''邱进'''，男，1954年12月26日生，[[中国人民大学|中国人民大学]]毕业，大学学历。现任[[中华人民共和国国家安全部|中华人民共和国国家安全部]]副部长<ref>{{cite news|author1=王增宁|title=全国地理信息市场专项整治工作进入落实整改阶段|url=http://www.zgchb.com.cn/article/2009/1016/article_14814.html|accessdate=2015-06-07|agency=中国测绘报|publisher=中国测绘新闻网|date=2009-10-16|location=北京|language=zh-hans|deadurl=yes|archiveurl=https://web.archive.org/web/20150607212814/http://www.zgchb.com.cn/article/2009/1016/article_14814.html|archivedate=2015-06-07}}</ref>，[[副总警监|副总警监]]警衔。

1987年6月起任[[中国共产主义青年团北京市委员会|共青团北京市委]]副书记，[[中华全国学生联合会|中华全国学生联合会]]副秘书长。

2002年起任国家安全部[[部长助理|部长助理]]、八局局长，[[国家安全部|国家安全部]]副部长。2012年处理[[王立军事件|王立军事件]]时，邱进从成都带[[王立军|王立军]]回京，因邱进一行人的航班乘客信息被网民曝光，邱进闻名全国。

邱进是[[中共十八大代表|中共十八大代表]]。2008年，当选[[第十一届全国政协委员|第十一届全国政协委员]]<ref>{{cite news |title = 第十一届全国政协委员名单 |url = http://www.indaa.com.cn/zt/2009qglh/zxzs/200903/t20090303_127573.html |accessdate = 2013-04-09 |work = 新华社北京1月25日电 中国人民政治协商会议第十一届全国委员会委员名单(共2237人，2008年1月25日政协第十届全国委员会常务委员会第二十次会议通过) |date = 2008年1月25日 |deadurl = yes |archiveurl = https://web.archive.org/web/20130928014403/http://www.indaa.com.cn/zt/2009qglh/zxzs/200903/t20090303_127573.html |archivedate = 2013年9月28日 }}</ref>，代表特别邀请人士，分入第五十七组。<ref>{{cite web |title = 全国政协信息-邱进 |url = http://www.cppcc.gov.cn/CMS/wylibary/showJcwyxtInfoWylibary.action?tabJcwyxt.guid=11W0020850&wytype=wjwy |accessdate=2013-04-09 |publisher = 全国政协}}</ref>
