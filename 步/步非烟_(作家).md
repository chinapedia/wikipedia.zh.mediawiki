'''步非烟'''（{{bd|1981年|7月11日|catIdx=B}}），本名'''辛晓娟'''，出生于[[四川省|四川省]][[成都市|成都市]]，当代[[武侠小说|武侠小说]]作家，被网友称为“北大才女”。其笔名来自于[[皇甫枚|皇甫枚]]所著的唐代[[传奇|传奇]]《[[步飞烟|步飞烟]]》。1999年步非烟考入[[北京大学中文系|北京大学中文系]]，2006年被授予古代文学硕士学位，2012年被授予北京大学中文系博士学位。其作品大气磅礴、汪洋恣肆，想象力神秘奇特、笔风变化万端，她开创的女子武侠更被誉为新神话主义、大陆武侠奇幻界想象力的代表<ref>{{Cite web|url = http://finance.ifeng.com/roll/20091216/1586800.shtml|title = 美女作家步非烟：武侠小说终将复兴|accessdate = 2015-08-03|author = |date = 2009年12月16日|publisher = 凤凰网财经|language = zh-cn}}</ref>。

== 主要作品 ==
*《[[华音流韶|华音流韶]]》系列：《紫诏天音》、《风月连城》、《彼岸天都》、《海之妖》、《曼荼罗》、《天剑伦》、《雪嫁衣》、《梵花坠影》
*《[[武林客栈|武林客栈]]》系列：《武林客栈·日曜卷》、《武林客栈·月阙卷》、《武林客栈·星涟卷》
*《天舞》系列：《天舞·摩云》、《天舞·御龙》、《天舞·.魅月》、《天舞·葬雪》、《云中漪兰》（外传）
*《人间六道》系列：《修罗道》
*《九阙梦华》系列：《解忧刀》《绝情蛊》
*《昆仑传说》系列：《昆仑劫灰》、《月影传说》、《天烬云殇》
*《舞阳风云录》系列：《月出秋山》《芙蓉花落》《塞上秋风》《梵天宝卷》《长空剑诀》《血令梅香》《天罗宝藏》（外传）
*《七宗罪》系列：《青螺髻》、《堕天翼》、《玲珑心》、《丁香舌》、《春葱指》、《阴耳风》、《沉脂腕》
*《[[玫瑰帝国|玫瑰帝国]]》系列（未完）：
**正传：《堕天使之心》、《潘多拉之盒》，《荆棘鸟之冠》，《黑羽蝶之翼》，《白蔷薇之祭》
**外传：《龙之羽翼》、《无冕之后》、《师》、《凯撒》
*《云天之外·木兰花开》系列（未完）
*其他作品：《剑侠情缘》、《玄武天工》、《飞羽天下》、《鬼族的公主》、《乌衣风流》、《黄金时代》、《华音阁十二月花》、《诗剑七夜谈》
*文集：《中国新武侠典藏书系·步非烟专卷》

== 争议事件 ==
2006年，在第三届今古传奇武侠文学奖暨黄易武侠文学特别奖颁奖仪式上，步非烟放言，“试图要超越金庸，革掉金庸们的武侠命，突破传统武侠小说堡垒，创作出新时代的武侠小说。”此语引发了中国大陆媒体及网民的关注，引出种种真假传言。有人认为她这样做"勇气可嘉，但实在无知"，也有人直指她“不自量力，太想出名，借此炒作”<ref>{{Cite web|url = http://news.cctv.com/performance/20061127/101929.shtml|title = 北大女生PK金庸 称要革掉金庸们的武侠命|accessdate = 2015-08-03|author = |date = 2006年11月27日|publisher = 华西都市报|language = zh-cn}}</ref>。
