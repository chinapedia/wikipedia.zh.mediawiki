== 人物简介 ==

　'''　农新瑜'''     现为湖北日报传媒集团资深编辑记者。现为湖北日报传媒集团资深编辑记者。社会任职有：武汉音乐家协会会员（2006年起）、湖北省葫芦丝巴乌专业委员会副会长兼秘书长（2013年起）、湖北省民族管弦乐学会理事（2008年起）、湖北省笛子专业委员会副秘书长（2013年起）、中国陶笛学会理事（2010年起）、武汉高校笛箫联谊会常务副主席（2001年起）、武汉岳飞文化研究会理事（2011年起）等。

== 简要履历 ==

1975年生于广西，壮族。<br>
1997年毕业于武汉科技大学，1998年招考进入湖北日报社，任《市场指南报》记者。<br>
2001年参与《楚天金报》创业，历任记者、要闻责编、时尚白领部副主任、编委办公室副主任、爱心工作部主任、楚天金报网站总监等职。<br>
2011年9月， 调任湖北日报传媒集团《前卫》杂志总编助理。<br>
2012年，调湖北日报传媒集团新媒体发展中心。<br>
2013年，调任《湖北日报》主编。<br>

== 工作成就及荣誉 ==
　　
    新闻工作
从事新闻工作十多年，具有较丰富新闻采编经验，多次荣获省级以上新闻奖，主要有：<br>
2003、2004年度，获湖北新闻奖三等奖各1件；2005年度，获湖北省报纸副刊作品年赛金奖1件；<br>
2006年度，获湖北新闻奖一等奖1件、二等奖1件，湖北新闻奖报纸版面复评金奖、银奖各1件；<br>
2007年度，获湖北新闻奖一等奖2件、二等奖3件、三等奖2件，湖北省报纸副刊作品年赛金奖1件，湖北新闻奖报纸版面复评金奖1件，湖北新闻宣传好稿奖2件；<br>
2008年度，湖北新闻奖三等奖1件；<br>
2009年度，湖北新闻奖二等奖、三等奖各1件等。<br>
曾获楚天金报总编辑特别奖、十大金报之星、集团年度优秀员工、集团十大杰出青年等荣誉。参与组织策划“金报周末晚会”、“湖北首届金报时尚文化节”等品牌活动。率团队打造的“金报读者爱心自驾乡村行”活动，引巨大社会反响，荣获国家级奖励，被全国妇联授予“维护儿童权益全国先进集体”称号。<br>

== 艺术人生 ==
　 　　
 艺术简历
业余酷爱艺术，擅长笛、箫、埙、葫芦丝等民族乐器，1995年就读大学期间首创武汉科技大学笛箫艺术协会。1997年毕业后业余时间致力于宣传弘扬民族音乐，2005年元月，在母校武汉科技大学举办“笛声萦梦.农新瑜笛子独奏音乐会”，成为湖北业余笛界举办独奏音乐会第一人。<br>
业余时间致力于宣传祖国传统文化和民族音乐，参与筹建多个协会，担任华中师大、中南财经政法大学、湖北大学、武汉工业学院等多所高校笛箫协会常年义务指导老师。<br>
积极投身各类民乐活动，撰写了大量音乐文章，策划主持了“龙笛凤箫情武汉高校巡演”等大量音乐活动，主持（导聆）过“李志成笛子独奏音乐会”、“谢闻吉笛子独奏音乐会”、“况禄师笛子独奏音乐会”等多场音乐会，在湖北民乐界具有一定影响力。<br>
农新瑜的笛箫演奏，在业余自学基础的基础上，受教于笛子大师孔建华先生、笛箫教育家瞿同庆教授，得到笛子演奏家蔡冬至先生、张重堂先生等人的指导和教诲。其演奏风格纯正，韵味浓厚，崇尚情感真挚，激情与细腻并重。
