'''农业'''是[[第一級產業|第一級產業]]，在現代有廣狹之分。廣義上的農業是[[種植業|種植業]]、[[林業|林業]]、[[畜牧業|畜牧業]]、[[漁業|漁業]]的總稱，而狹義上的農業則是純粹指[[種植業|種植業]]（而這亦符合中文裡「農」字的意思）。农業的產品一般會是[[食物|食物]]、[[纖維|纖維]]、[[生物燃料|生物燃料]]、[[藥物|藥物]]或是其他利用自然資源而來，可以維持或提昇人類生活的物品<ref name="Office1999">{{cite book |author= |title=Safety and health in agriculture |url=http://books.google.com/books?id=GtBa6XIW_aQC&pg=PA77 |publisher=International Labour Organization |isbn=978-92-2-111517-5 |pages=77– |accessdate=13 September 2010|date=1999}}</ref>。此專頁的內容為廣義上的農業。

農業屬[[初級生產|初級生產]]，為[[人類|人類]]最大和最重要之[[經濟|經濟]]活動之一。簡單地說是人類運用其[[智慧|智慧]]去改變[[自然環境|自然環境]]，利用動植物的生長繁殖來獲得產品，更進一步換取經濟收益的一種系統。農業是人類定居[[文明|文明]]興起的關鍵因素，種植或養育[[馴養|馴化]]後的物種，可以增加食物，有助於[[文明|文明]]的發展。有關農業的知識稱為[[農業科學|農業科學]]。[[農業史|農業史]]可以追溯到數千年前，農業的發展隨著不同地區的[[氣候|氣候]]、[[文化|文化]]及技術有很大的不同。不過所有的農業都需要技術可以擴展及維持可以種植作物或養殖動物的土地。若是種植植物，一般就需要[[灌溉|灌溉]]技術，不過也有些植物可以{{link-en|旱地耕作|dryland farming}}。動物畜養是個龐大的產業，約佔了地球不被冰或是水覆蓋的面積的三分之一。在[[已開發國家|已開發國家]]，一般是採取[[單一作物|單一作物]]的[[集約農業|集約農業]]方式，不過藥物與肥料使用量巨大，進入21世紀後[[可持續農業|可持續農業]]的比例也在漸漸提高，包括[[樸門|樸門]]（Permaculture）及[[有機農業|有機農業]]，著重在生態平衡與就近百里飲食。

依據[[中華人民共和國|中華人民共和國]][[農業法|農業法]]第二條定義，本法所称農業，是指种植业、林业、畜牧业和渔业。

== 詞源 ==

無論是根據東漢時期的《[[說文解字|說文解字]]》抑或清康熙時期的《[[康熙字典|康熙字典]]》的解釋，「農」字都是「耕種」的意思。這表示在[[中國|中國]]古代就只有[[種植業|種植業]]才會被稱作「農業」。換言之，現代人將[[畜牧業|畜牧業]]和[[漁業|漁業]]都併入「農業」，實際上是扭曲了中文裡「農」字原來的意思。

== 歷史 ==
{{Main|农业史}}
農業起源於至少一萬年前，並且從最早的耕作時代起有很大的發展。一些證據指出：一萬一千年前中東的[[新月沃土|新月沃土]]是最早出現有計畫的播種和收成原本應在野外採集之植物的地區；也有人認為，農業的獨立發展發生於中國北方和南方、非洲的[[薩赫勒|薩赫勒地區]]、[[新幾內亞|新幾內亞]]和美洲的一些地區。5000年前，中東農耕族群男性來到歐洲，與當地狩獵、採集族群中的女性通婚後產生現代歐洲人；這過程亦將農業傳播至歐洲。<ref>{{Cite news|url=http://pansci.asia/archives/7796|title=農業的傳播，移民與通婚|last=|first=|date=2011-10-02|work=|newspaper=[[PanSci_泛科學|PanSci 泛科學]]|accessdate=2017-03-09|language=zh-TW}}</ref>

中國[[南北朝|南北朝]]時期出版的《[[齊民要術|齊民要術]]》為中國最完整最古老的農牧著作<ref>{{cite journal zh| authors=劉仲康 鐘金湯| title=賈思勰與《齊民要術》| journal=科學月刊| volume=0346| url=http://210.60.224.4/ct/content/1998/00100346/0016.htm| accessdate=2013-12-03| date=1998年10月| deadurl=yes| archiveurl=https://web.archive.org/web/20130323050757/http://210.60.224.4/ct/content/1998/00100346/0016.htm| archivedate=2013-03-23}}</ref>，收錄1500年前中國農藝、[[園藝|園藝]]、造林、蠶桑、畜牧、[[獸醫|獸醫]]、配種、釀造、[[烹飪|烹飪]]、儲備，以及治荒的方法，書中援引古籍近200種，所引《氾勝之書》、《四民月令》等現已失傳的漢晉重要農書，並探討農業和商業、政治間的關係，等於一種[[農業社會|農業社會]]的全盤研究。

上個世紀的農業特色在於提高生產率：利用合成[[肥料|肥料]]、[[農藥|農藥]]、[[選育|選育]]和[[农业机械化|機械化]]，取代了人類的勞動。最近農業和行政議題緊密連結，像是：[[水污染|水汙染]]、[[生質燃料|生物燃料]]、[[转基因生物|基因改良作物]]、[[关税|關稅]]和[[農業補貼|農業補貼]]。近年來，出現了反對機械化農業對外在環境的破壞和支持[[有機農業|有機農業]]和[[可持续发展|永續]]農業的聲浪。另外還有一種科技化農業，多層大樓中垂直型態的農田除了可以自由調控生長環境（光照、濕度、滴灌、培養基）之外，甚至這些溫室用了[[機器人|機器人]]來生產，產量與品質都是最高的，也幾乎對自然生態無破壞，可以直接建在都市內節省運送的浪費，目前受限於成本，除了[[日本|日本]]、中東、北歐、俄羅斯等國家正在試驗中外還未普及，但在2050年以後世界上預計會多出30億人，這種「農廠」、「農作大厦」在今日可種植土地幾乎全被開發下就很需要，很有可能成為未來主要的生產方式，包含更久以後前往其他行星居住的人們所利用，例如國際太空站中植物在低重力的生長實驗。

== 農業系統之結構 ==
農業是人類有目的，有計劃地參與通过改造自然環境来进行生產的活動，因此它是一類組織系統；人類把技術應用到自然環境中，獲取經濟效益，所以它也是一個人類干擾系統，[[農作物|農作物]]－環境－人三元結構系統。這與自然系統生物－環境二元結構的本質區別，而人工控制、利用和改造是系統的特徵。其結構包括：

=== 投入 ===
'''自然環境投入（natural environment input）'''
農業的自然環境是指來自自然界的各圈，影響農業生物生命活動的各種自然條件總和。主要分為生物環境與非生物環境兩方面。

生物環境投入：包括了農業生物周圍之植被、雜草、病蟲、昆蟲等要素和培植物質，如種子、牲畜等。它們互相制約，互相依存共同組成生態系統中的生產者、捕食者和競爭者等之關係。

非生物環境投入：指日照、[[氣溫|氣溫]]、水汽、地形、[[土壤|土壤]]等條件。它們為農作物生長、發育、繁殖提供了空間、物質和能源，是農業不可少的條件。

'''人文環境投入（human environment input）'''
勞力（labour）、[[資本|資本]]（capital）、[[市場|市場]]（market）、[[運輸|運輸]]（transport）、技術（technology）、[[土地|土地]]（land）、[[政府|政府]]（government）、個人觀感（perception）。

=== 過程 ===
'''農業生態系統的運作'''
既不同於它所從屬的全球生態系統中的其他子系統，如各種自然[[生態系統|生態系統]]，任何以植物爲基礎的自然生態系統雖然都能積蓄太陽能，但它們並不需要投入勞動。

'''農業社會經濟系統的運作'''
也不同於它所從屬的國民經濟系統中的其他子系統，如[[工業|工業]]，它是利用生物的生命過程來取得人們所需要的產品。

== 農業對環境影響 ==
[[File:Erosion.jpg|thumb]]。]]
農業有時會產生環境問題因為農業改變了其自然環境及產生有害的副產品。一些對環境有負面影響例如：
* [[生物多樣性|生物多樣性]]的減少
* [[森林砍伐|森林砍伐]]（Deforestation）

* 各种农药的污染以及有害影响
* 耕地扩大造成的生态失衡
* [[动物|动物]]粪便中包含[[氨|氨]]和[[铵|铵]]的颗粒物质被扬起到空中造成的[[空气污染|空气污染]]。
* 牛隻的反芻行為引致甲烷的釋放<ref>[http://spaces.isu.edu.tw/upload/19434/5/files/dept_5_lv_2_2250.pdf 溫室效應為何？]</ref>
* 耗盡土壤中的礦物
* [[土壤侵蝕|土壤侵蝕]]
* [[土壤鹽化|土壤鹽化]]

== 農業系統之研究 ==
自19世紀末至20世紀，已經有地理學家對農業推行深入研究。例如英國的G. G. Chisholm及德國的H. Engelbrecht。以屈圖士（Whittelesey）的分類法最完善。

=== 范杜能的孤立國理論 ===
[[范杜能|范杜能]]在1826年出版的著作《孤立國》（The Isolated State）中，分析了一系列影響農業土地利用的因素。他是首位認真地研究空間經濟學的學者，憑著經營農莊的經驗，並將其連繫至租值理論上，成為後世地理學者研究農業和土地利用的基礎。

范杜能模式有一些簡化了的假設<ref>{{Cite web
 | url= http://content.edu.tw/senior/geo/ks_ks/book1/industry/non_geo/ps1.htm
 | title= 邱念圈
 | accessdate=2013-12-04
 | work= 數位教學資源入口網
 | publisher= 中華民國教育部
}}</ref>：
* 孤立國
* 均質平原
* 位於城市中央的單一[[市場|市場]]
* 市場價格一致
* 農夫是「[[經濟人|經濟人]]」，追求利潤極大化
* 無外在影響力，沒有[[科技|科技]]革新
* 只有一種運輸形式，交通費用呈[[線性|線性]]關係

他的「孤立國」（isolated state）理論源於[[亞當·斯密|亞當·斯密]]的「[[經濟人|經濟人]]」（economic men）概念，即所有[[農民|農民]]均欲把農田的利潤極大化。

=== 辛克萊模式 ===
[[File:New_crops-Chicago_urban_farm.jpg|240px]]
辛克萊發現美國中西部城市的擴張影響周圍農業土地利用的發展，他認為范杜能模式在已發展國家中已過時，原因是交通運輸改變、大量生產及市場擴大及城市擴張<ref>{{Cite web
 | url= http://content.edu.tw/senior/geo/ks_ks/book1/industry/non_geo/ps2.htm
 | title= 辛克萊模式
 | accessdate=2013-12-04
 | work= 數位教學資源入口網
 | publisher= 中華民國教育部

*城市地價較農村高，形成城市用地與農業用地的直接競爭
*城市外圍的農地地價較高，會被投機者收購
*愈接近城市，用作農地之價值下降，因為城市化的可能增加；若距離市區愈遠，價值增加，直至到達一點已無機會變作城市土地，用作農地的價值便會穩定。

== 政策 ==
農業政策集中於農業生產的目標及方法。在政策程度的一般目標例如：
* 食物安全：確保食物避免受到污染。
* 食物監管：確保食物應付人們需要。
* 食物質素：確保食物符合質素。
* 保護
* 環保影響
* 經濟穩定
