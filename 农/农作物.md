'''農作物'''，或常被稱為'''作物'''，又称'''农艺作物'''，俗称'''庄稼'''，是泛指在大量培植供人食用或做[[工业|工业]]原料的物種，是由野生[[植物|植物]]经过人类不断的选择、驯化、利用、演化而来的具有经济价值的被人们所栽培的一切植物。目前世界上被人们所栽培的植物约1500种，依其應用及研究方式不同，常見多種不同的分類方式，例如食用作物、特用作物、雜用作物等。

广义上农作物可分为[[大田作物|大田作物]]、[[园艺作物|园艺作物]]、[[树|林木]]三类，包括所有栽培植物，如大田作物，果树，蔬菜，观赏、药用植物，林木等<ref name="中国作物学会">{{cite web|url=http://www.chinacrops.org/zwxh/kp/农作物百科/N/N7.htm|title=农作物(field crops)|publisher=[[中国作物学会|中国作物学会]]|date=2006年11月20日|accessdate=2014年7月25日|archive-url=https://web.archive.org/web/20150604062137/http://www.chinacrops.org/zwxh/kp/%E5%86%9C%E4%BD%9C%E7%89%A9%E7%99%BE%E7%A7%91/N/N7.htm|archive-date=2015-06-04|dead-url=yes}}</ref>。对这种广义的作物进行栽培与管理即[[种植业|种植业]]。

狭义上农作物专指大田作物（field crops）<ref name="中国作物学会" />，即在田间进行大面积栽培的农艺作物，包括粮、棉、油、麻、丝、茶、糖、烟和饲料等。对这种狭义的作物进行栽培与管理即作物栽培。

== 定義 ==
「作物」本為[[日本汉字|日本名詞]]，20世纪初期在中国农业文献中开始引用<ref name="中国作物学会" />，指[[耕作|耕作]]所得之物也。<ref name="翁1947">，翁德齊，1947《作物學概論》。</ref> <ref name="翁1947a">，翁德齊，1947《作物學概論》，第1頁。編按：此說法可能需要確認，參見[http://ja.wikipedia.org/wiki/%E8%BE%B2%E7%94%A3%E7%89%A9 日文維基百科/農產物]</ref>故作物云者，即經人工管理栽培之植物是也。舉凡直接供給人類衣食住之農業植物，或供給家畜之飼料植物皆屬之。<ref name="翁1947" />

目前估計全世界的野生植物約在30萬種以下，其中人類選擇有利用價值的植物給予保護、栽培、馴化和改良，使其成為具有經濟價值的植物，這種植物名之為作物。例如[[水稻|水稻]]，野生稻的原產地可能在[[東南亞|東南亞]]，經數千年的栽培和馴化，才變成重要的農作物。<ref name="盧1994">，盧英權，1994《作物學》。</ref>

由[[農學|農藝學]]之觀點，作物可分為食用作物（food crops）與特用作物或工藝作物（special crops, industrial crops）。<ref name="高景輝1984">高景輝，1984。作物分類。第1-8頁。In:國立台灣大學農藝學系（編），1984。作物學實習手冊。臺北市：國立編譯館。</ref>

食用作物供應人類糧食或[[飼料|牲畜之食料]]為主，具有單位面積產量高與適應力強易於栽培二種特性。<ref name="高景輝1984" />

特用作物為供應特殊用途而栽培的作物，其供應範圍不包括於食用方面。一般以工業原料為主，大多必須經加工後方能應用。特用作物經過加工後，價格提高，又耐儲藏，因此可以當作貿易品，故特用作物也可稱為貿易作物（commercial crops）。<ref name="高景輝1984" />

目前所稱作物其形態及性質由於栽培馴化的結果，與野生種類已大不想同，由於人類在其作物中尋找出變異，並選擇有利的表現型予以繁殖、改良。作物常是具高收穫量，但相對的對經常也失去一些野生品種所具的特性，如抗病性。這種現象在粟與狗尾草、稻與野生稻、燕麥與野生燕麥中，可見作物種類之穀粒較大，但比起野生種的耐旱力、抗病力則較弱。<ref name="盧1994" />

==範圍==
廣義而言，作物之範圍，包含所有之栽培作物，如食用作物、特用作物、[[森林|森林]]作物、及[[園藝|園藝]]作物均屬之。<ref name="翁1947" />

狹義而言，僅限於食用作物及特用作物而已；或合併二類作物而名之曰普通作物。<ref name="翁1947" />

普通作物似與森林作物及園藝作物略有區別。惟嚴格言之，則其界限尚難顯然劃分。例如某些国家多以馬鈴薯為日常食品，而廣為種植，是則馬鈴薯乃為普通作物之一。然在其他地区通常之情形下，多以之為副食品，故又可稱為園藝作物。<ref name="翁1947" />

==歷史==
=== 起源 ===
[[人類|人類]]開始栽培[[作物|作物]]推測可能有一萬多年。據Helbaek在1959年發表，[[伊拉克|伊拉克]]北部Jarmo所發現的[[素燒|素燒]][[陶罐|陶罐]]等，所附著的[[穀類|穀類]][[種子|種子]]或[[小穗|小穗]]痕跡，經[[放射性|放射性]][[碳同位素|碳同位素]]測定其年代在7000年以前已存在。<ref name="盧1994" />

[[西歐|西歐]]地區藉由[[花粉分析|花粉分析]]的方法，推斷西歐大部份地區在[[冰期|冰期]]後期，即距今8000年前，才開始有人類遷入，由[[湖泊|湖泊]][[沉積物|沉積物]]的花粉圖譜顯示，[[新石器時代|新石器時代]]，至晚在[[青銅器時代|青銅器時代]]中期之前（西元前1600年）已有穀類栽培的證據。<ref name="盧1994" />

現今世上所栽培的作物均由野生[[植物|植物]]逐漸演進而來。據De Candolle《[[栽培植物的起源|栽培植物的起源]]》（1882年），記錄了共247種作物的發祥地，其中舊大陸產作物199種，新大陸作物45種，及不明者3種。其後經Vavilov在1951年，以[[遺傳學|遺傳學]]及[[植物地理學|植物地理學]]的方法，發展出[[遺傳中心說|遺傳中心說]]（Gene center hypothesis）。提出八個原始中心，稍後不斷修正有十個起源中心。<ref name="盧1994" />

==重要性==
作物提供[[人類|人類]]每日的[[衣|衣]][[食|食]]之原料、[[實業|實業]]之原料、並顯響到[[國家|國家]]之盛衰。<ref name="翁1947" />

==分類==
分類方式多種，常依其所欲利用的方式不同，而有不同的方式，顯得有些紛亂。例如，按總類可以分為“[[糧食|糧食作物]]”、“[[綠肥作物|綠肥作物]]”、“[[經濟作物|經濟作物]]”等；根據其使用方向可以分為“[[纖維|纖維]]作物”、“[[糖|糖]]料作物”、“[[澱粉|澱粉]]作物”、“[[飲料|飲料]]作物”、“[[橡膠|橡膠]]作物”、“[[染料|染料]]作物”、“藥用作物”、“芳香油作物”等；根據其成分可以分為“穀[[麥|麥]]類作物”、“[[豆|豆]]類作物”、“薯類作物”等；根據其生長條件可以分為“旱地作物”、“[[水田|水田]]作物”、“覆蓋作物”等。

===按植物學之分類法===
作物依照[[植物學|植物學]]的[[科學分類法|科學分類法]]最主要的作物概屬於[[禾本科|禾本科]]及[[豆科|豆科]]。其他如[[馬鈴薯|馬鈴薯]]、[[菸草|菸草]]等屬[[茄科|茄科]]；[[棉花|棉花]]屬[[錦葵科|錦葵科]]等。<ref name="翁1947" />

===按普通用途之分類法===
此種方法乃按照[[農學|農學]]分類法（Agronomic classification）<ref name="翁1947" /><ref name="高景輝1984" />。但不同學者仍常見差異，例如：盧英權將作物概分為食用作物、工藝作物及飼料作物（含綠肥作物），而未提及速成作物、保護作物等特殊作物<ref name="盧1994" />。下列種類概屬於食用作物及工藝作物。
# 禾穀類或穀實類作物（Cereals或Grain crops）：以[[種子|種子]]為食用目的而栽培之[[禾本科|禾本科]]作物，如[[稻|稻]]、[[小麥|小麥]]、[[大麥|大麥]]、[[燕麥|燕麥]]、[[玉米|玉米]]、[[高梁|高梁]]、[[黍|黍]]、[[粟|粟]]（含[[稷|稷]]）、[[蜡燭稗|蜡燭稗]]、[[薏苡|薏苡]]、[[蕎麥|蕎麥]]等。
# [[豆類|豆類]]作物（Legume for seed）或稱豆菽類作物：以種子為栽培目的之[[豆科|豆科]]作物，如[[菜豆|菜豆]]、[[豌豆|豌豆]]、[[蠶豆|蠶豆]]、[[大豆|大豆]]、[[落花生|落花生]]、[[綠豆|綠豆]]、[[豇豆|豇豆]]、[[扁豆|扁豆]]等。
# 飼料作物（Forage crops）：供[[牲畜|牲畜]]食用而栽培之作物，普通均指牧草而言，牧草可分為：1.禾草如[[狼尾草|狼尾草]]、[[梯牧草|梯牧草]]。2.豆草如[[三葉草|三葉草]]、[[苜蓿|苜蓿]]、[[紫雲英|紫雲英]]等。3.其他如[[十字花科|十字花科]]之[[蕪菁|蕪菁]]、及[[藜科|藜科]]之[[甜菜|甜菜]]。
# 根類作物（root crops）：以其膨大的[[根|根]]為栽培目的之作物，例如[[甘藷|甘藷]]、[[甜菜|甜菜]]、[[胡蘿蔔|胡蘿蔔]]、[[菊芋|菊芋]]等。
# 纖維作物（fiber crops）：以採收[[纖維|纖維]]為目的之作物，採收纖維的部位各有不同：採自[[種子|種子]]者如[[棉|棉]]；採自[[莖|莖]]上之[[韌皮部|韌皮部]]者如[[黃麻|黃麻]]；採自[[葉|葉]]部者如[[瓊麻|瓊麻]]；採自莖部全體者如[[藺草|藺草]]。依用途又可分為[[紡織|紡織]]類、[[編組|編組]]類、與製[[紙|紙]]類。<ref name="盧1994" />
# 塊莖作物（tuber crops）：採收膨大的地下莖（或作[[塊莖|塊莖]]）為目的之作物，如[[馬鈴薯|馬鈴薯]]、[[芋|芋]]、[[薯蕷|薯蕷]]。
# 糖料作物（sugar plants）：供作製[[糖|糖]]的作物，以[[甘蔗|甘蔗]]和[[甜菜|甜菜]]為主。
# 嗜好類或刺激類作物（stimulants）：這類作物經食用後具有刺激和興奮[[神經|神經]]的作用，長久食用即成為一種[[嗜好|嗜好]]。如[[菸草|菸草]]、[[茶|茶]]、[[咖啡|咖啡]]、[[可可|可可]]、[[罌粟|罌粟]]等。
# 油料作物（oiling crops）：種子供榨油之作物，如[[大豆|大豆]]、[[花生|花生]]、[[亞麻|亞麻]]、[[油菜|油菜]]、[[棉|棉]]籽等。 
# 染料作物（coloring crops）：凡是作物的[[根|根]]、[[莖|莖]]、[[葉|葉]]、[[花|花]]等部份可調製[[染料|染料]]供染色用者。如[[薑黃|薑黃]]、[[山藍|山藍]]、[[薯榔|薯榔]]。
# 香料作物（aromatics）：如[[香水茅|香水茅]]可製造[[香水|香水]]；[[茉莉|茉莉]]的花可以薰[[茶|茶]]（[[香片|香片]]），增加茶葉[[香味|香味]]；製[[香|香]]的原料如[[檀香|檀香]]、[[沉香|沉香]]等。
# 藥用作物（drugs）：供[[药用植物|藥用]]，大多數為野生的植物，僅少數人栽培，如[[除蟲菊|除蟲菊]]、[[魚藤|魚藤]]、[[薄荷|薄荷]]、[[當歸|當歸]]等。

===按作物特別用途之分類法===
部份作物的用途特殊，但仍屬於作物的範圍<ref name="翁1947" /><ref name="高景輝1984" />： 
# 覆土作物（cover crops），或覆蓋作物：這類作物長成後，莖葉覆蓋地面，具有防止[[土壤|土壤]][[侵蚀作用|沖刷]]的功用。覆土作物耕翻土中後，亦可增加土壤養分而為綠肥作物。如[[野豌豆|野豌豆]]、[[紫苜蓿|紫苜蓿]]等。
# 綠肥作物（green manure crops）：栽培的目的在於利用它的莖葉翻埋土中，以改善[[土壤|土壤]]理化性質，增加土壤肥力的作物。如[[大豆|大豆]]、[[紫雲英|紫雲英]]、[[苜蓿|苜蓿]]等。
# 速成或過渡作物（catch crops）：遇天時不利、水份不足，或主要作物播種期已過，或栽種失敗時可種植[[黑麥|黑麥]]、[[粟|粟]]、[[蕎麥|蕎麥]]等作物以資補救。例如[[華北地區|華北地區]]高梁未能如期播種，多改種粟；倘若粟亦不能如期播下，則可改播綠豆。又，秋季播種黑麥，來春將其翻入土中另植他種作物，此種黑麥稱為過渡作物，能保護土壤，亦可作為綠肥，還可充放牧之用。
# 鮮飼作物（soiling crops）：此類作物刈割後未經貯藏而直接飼畜者。
# 青藏作物（silage crops, ensilage crops）：作物刈取後藏於不透氣的箱中或秣窖中經發酵再行飼畜者。玉蜀黍、蜀黍、向日葵等是美國主要的青藏作物。
# 保護作物（nurse crops）、共榮作物（nurse crops）
