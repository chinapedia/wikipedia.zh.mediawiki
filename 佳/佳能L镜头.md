'''L镜头'''是[[佳能|佳能公司]]为[[单镜反光相机|单镜反光相机]]生产的顶级镜头系列。

字母“L”是英文单词“豪华”（Luxury）的第一个字母，L镜头的品質與價格都十分突出。佳能的[[變焦鏡|变焦]]和[[定焦镜头|定焦]]L镜头應用于已经淘汰的FD接環以及目前所有EOS相机使用的EF接環。

截至2015年，佳能还没有为EF-S接環推出L镜头。有些人认为，佳能的鏡頭產品是否具有“L”的定位，是一个商业推广的问题<ref name="photozone">{{cite web
|url=http://www.photozone.de/Reviews/Canon%20EOS%20Lens%20Tests/45-canon-eos-aps-c/178-canon-ef-s-17-55mm-f28-usm-is-test-report--review 
|title=Canon EF-S 17-55mm f/2.8 USM IS - Test Report / Review
|accessdate=2008-02-25
|work=[http://photozone.de/ photozone]
}}</ref>；譬如EF-S 17-55mm和EF-S 10-22mm两款镜头都配備有L镜头所使用的高品質光學鏡片，並具有与某些L镜头不相上下的画质，却没有被标上“L”。

雖然L鏡頭一般應用於單鏡反光機上，但佳能的消費級半專業數碼相機PowerShot Pro1卻是使用一個非更換式L鏡頭，成為了一個特殊例子。

==特征==
[[File:Canon_EF_70-200mm.jpg|thumb]]]]
L镜头的光学性能都十分突出，一般來說，比佳能的普通镜头產品更坚固耐用，更能承受恶劣环境。大部分新生产的L镜头都能防尘和防水滴。由于价格昂贵，体积较大并较重，L镜头往往更适合专业摄影师或者资深爱好者。

大部分L镜头都有如下特征：
* 适合在恶劣环境中使用，如野外。（部分镜头具有防尘防水滴结构）。
* 至少一片[[萤石|萤石]]或低[[色散|色散]]（UD或S-UD）镜片，以及[[非球面镜片|非球面镜片]]。
* 镜头前端对焦时不旋转，适合使用濾鏡（比如[[偏振光镜片|偏振光镜片]]）。
* 相比于同一焦段的其他佳能镜头，[[光圈|光圈]]更大，变焦镜头采用恒定光圈，大部分L镜头都是圆形光圈。
* 以环形超声波对焦马达（USM）驅動對焦，並有全时手动对焦（Full-Time Manual Focus）或电子手动对焦的功能。
一些非L级镜头也带有一项或多项上述特征。一些L镜头还配有镜头图像稳定器（'''I'''mage '''S'''tabilizer，IS）。某些L变焦镜头采用了内变焦的方式来加强防尘防水滴性能。

较大的L镜头，比如70-200mm或者100-400mm变焦镜头，以及更长（300mm或以上）的定焦镜头，外觀都是[[象牙|象牙]]白色塗裝，使镜头在陽光直射的情况下，鏡筒內也能保持较低的温度，以保证成像品質；因此“白镜头”也成为了佳能长焦镜头的标志。除了已停产的200mm f/1.8L和新推出的200mm f/2.0L IS以外，所有焦距在300mm以下的定焦镜头都是黑色的。另外，红圈是L镜头上的标志性特征。

==镜头编号==
佳能镜头尾端的六位编号包含了产地和生产日期信息。

比如，UV1212中的字母“U”表示生产厂。第一个字母有三种可能：

#U，表示[[宇都宫|宇都宫]]分厂（Utsunomiya）
#F，表示[[福岛|福岛]]分厂（Fukushima）
#O，表示[[大分|大分]]分厂（Oita）

第二个字母，比如“V”，代表生产年份：

*A = 2012年，1986年，1960年
*B = 2013年，1987年，1961年
*C = 2014年，1988年，1962年
*D = 2015年，1989年，1963年
*E = 2016年，1990年，1964年
*F = 1991年，1965年
*G = 1992年，1966年
*H = 1993年，1967年
*I = 1994年，1968年
*J = 1995年，1969年
*K = 1996年，1970年
*L = 1997年，1971年
*M = 1998年，1972年
*N = 1999年，1973年
*O = 2000年，1974年
*P = 2001年，1975年
*Q = 2002年，1976年
*R = 2003年，1977年
*S = 2004年，1978年
*T = 2005年，1979年
*U = 2006年，1980年
*V = 2007年，1981年
*W = 2008年，1982年
*X = 2009年，1983年
*Y = 2010年，1984年
*Z = 2011年，1985年

接下来的两位数字是生产月份。最后两位数字用于内部分类。

例如：編號''UV1212''，表示这颗镜头于2007年12月产自宇都宫分厂。
