'''蕭家祖屋'''<ref name="客委會-聯合報">{{Cite web |url=http://www.hakka.gov.tw/ct.asp?xItem=3298&ctNode=308&mp=20 |title=《三級古蹟蕭家古厝整修完成.全台最完整五堂大屋古厝,開放參觀》，客家委員會官網轉載自聯合報 |access-date=2014-08-31 |archive-url=https://web.archive.org/web/20150924025448/http://www.hakka.gov.tw/ct.asp?xItem=3298&ctNode=308&mp=20 |archive-date=2015-09-24 |dead-url=yes }}</ref>，又名'''蕭家古屋'''、'''蕭屋伙房'''、'''佳冬蕭宅'''<ref name="文化資產局">[https://nchdb.boch.gov.tw/assets/overview/monument/19851127000088 《佳冬蕭宅》，文化部文化資產局] </ref>，位於台灣[[屏東縣|屏東縣]][[佳冬鄉|佳冬鄉]]，是台灣唯一[[客家|客家]]圍龍夥房五堂大屋，1985年[[中華民國內政部|中華民國內政部]]列為[[三級古蹟|三級古蹟]]，並由蕭家子孫所組成的「佳冬蕭家祖屋管理委員會」負責管理<ref name="客委會-聯合報"/>。

== 蕭家發展史 ==
蕭氏祖籍[[廣東|廣東]][[梅縣|梅縣]]，自松源蕭家第十九世遷台第一世祖蕭達梅渡海來台至今近兩百餘年。最初落腳於[[台南|台南]]，最後定居於目前的佳冬鄉。以釀酒為業，逐漸累積財富，並購置大批田產。其後人來台三世祖蕭光明更以「蕭協興」為商號名稱在台南、[[東港鎮_(臺灣)|東港]]、佳冬一帶經營貿易活動，並從事碾米及染布生意，經營有成，逐漸成為佳冬地區的一大家族。<ref name="文化資產局"/>

== 工藝技術 ==
[[File:佳冬蕭宅合院.JPG|thumb]]
蕭家請來唐山師傅負責興建，並全部採用中國的建材，從安平港轉運東港再分送到六根庄(今佳冬)，自清[[咸豐_(年號)|咸豐]]十年（1860年）開始，陸續興建了第二堂至第四堂，左右橫屋、染房、馬鹿廊部分，至[[光緒|光緒]]元年（1875年），又興建了第一堂屋及其左右橫屋，於光緒六年（1880年）增建了第五堂，並將左右橫屋聯結成完整的防禦外牆。

蕭屋使用的是馬背山牆，呈現樸質的客家人不好浮誇的表現。由於前後修改建的時間有好幾個段落，建材的使用也有所不同。第一堂、第二堂與步月樓牆體的土角磚外覆薄磚的「金包玉」砌築，且因時制宜，工匠施作用其用料、施工的精細度，和雕工的精良，都具有高度的成就，足以作為傳統建築藝術價值的代表。

空間使用上最大的特色就是室內外交互滲透。五堂雖各自獨立，但是堂屋與左右橫屋間四通八達，或漏明窗、八卦門轉化空間的區隔方式，豐富不同空間層的組合。室內外空間近密接合，左右兩條前後貫通天井，提供前後堂次要路徑，往來於堂屋內埕、橫屋間相當方便。

堂屋和橫屋之間則利用「騎馬廊」連接，平時家族會在這一個具開放性的半戶外空間圍坐聊天，或為婦女從事家務、縫補女紅之所在，形成休閒或工作空間。

== 防禦形式 ==
[[File:屏東縣佳冬鄉蕭家古厝的步月樓.jpg|thumb]]
[[康熙|康熙]]六十年（1721年）[[朱一貴|朱一貴]]戰事，屏東客家庄為了自衛而組織義軍，各處客庄劃分為前、後、左、右、中及先鋒等「[[六堆|六堆]]」，並成立地方自治組織，互為守望相助，自此六堆中的民團即擔負起地方治安的重要角色。

[[光緒|光緒]]二十一年（1895年），日軍犯台，自枋寮登陸而入侵佳冬，蕭光明當時擔任六堆副總理、左堆總理，率眾據守蕭屋之「步月樓」，現在可看見當時抗戰時所留下來的彈痕，見證客家先民保衛鄉土精神。

蕭屋位踞佳冬的市中心區，為當地聚落的防禦要角。蕭屋佔地0.4公頃，面寬29.6公尺、深67公尺，採雙「回」字型建構一完整封閉的圍龍屋。在整個平面的安排上，藉由外圍連貫的左右橫屋與第一堂門堂、第五堂住家共同為成「厝包」，內圈由堂屋與騎馬廊相連而成。這要嚴密的建築配置，顯示客家民居所強調的防禦功能。至今在蕭屋裡通道側，仍可以看到許多昔日具有防衛功能的銃孔。
