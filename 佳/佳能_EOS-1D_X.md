'''佳能 EOS-1D X'''是[[佳能|佳能]]下一代[[全画幅数码单镜反光相机|全画幅数码单镜反光相机]]，并计划用它来替代[[佳能_EOS-1D_Mark_IV|佳能 EOS-1D Mark IV]]和[[佳能_EOS-1Ds_Mark_III|佳能 EOS-1Ds Mark III]]两个产品。<ref name = "announcement">{{cite news|url=http://www.cameragearguide.com/5327/canon-announces-new-flagship-18mp-eos-1d-mark-x-dslr-camera/|title=Canon announces New Flagship 18MP EOS 1D Mark X DSLR Camera|publisher=cameragearguide.com|date=2011-10-18|accessdate=2011-10-18}}</ref>它具有1,810万[[像素|像素]]的CMOS感光元件，与双核DIGIC 5+微处理器，其中每一个都能提供DIGIC 4微处理器17倍的处理能力，另有一枚专门用于测光和对焦的DIGIC 4处理器。有61个高密度网状阵列自动对焦点（其中41点为十字型）。这款相机最高能够达到每秒14幅（JPEG，锁定反光板及对焦距离）的连拍速度。常用[[感光度|感光度]]覆盖100至51,200（扩展感光度包括50至204,800）。另外，相机还提供[[吉比特以太网|千兆以太网]]接口以方便图片的传输。还可选配无线局域网适配器或GPS模块。该相机已于2012年6月20日赶在2012年伦敦夏季奥运会前上市。在香港的价格为51,800港币。<ref name = "announcement"/><ref>{{cite news|url=http://www.popphoto.com/gear/2011/10/new-gear-canon-1d-mark-x|title=New Gear: Canon EOS-1D X Full-Frame Pro DSLR|publisher=POPPHOTO.com|date=2011-10-17|accessdate=2011-10-18}}</ref>
