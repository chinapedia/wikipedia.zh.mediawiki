'''鹿港街長宿舍'''位於臺灣彰化縣[[鹿港鎮|鹿港鎮]]，是日治時期[[臺中州|臺中州]][[彰化郡|彰化郡]][[鹿港街|鹿港街]]街長的宿舍，於民國九十一年（2002年）11月20日公告為歷史建築{{r|文}}。

== 沿革 ==
鹿港街長宿舍於昭和十年（1935年）4月1日動工，同年10月20日落成{{r|彰}}。鹿港街首任街長[[陳懷澄|陳懷澄]]在任時該宿舍尚未建立，所以首位入住的街長不是他而是第二任的吉田秀次郎{{r|彰}}。

二次大戰後，街長宿舍閒置，曾作為鹿港鎮公所的倉庫{{r|彰}}。而在該建築指定為歷史建築後，於隔年（2003年）9月17日開始修復工程，於民國九十三年（2004年）7月完工，民國九十四年（2005年）1月30日舉行修復工程落成典禮{{r|彰}}。

== 建築 ==
街長宿舍為木建築，基座抬高以防潮與防蛀蟲{{r|彰}}。其外牆為雨淋板，室內隔間牆則為編竹夾泥牆{{r|彰}}。其平面呈L形，內部有座敷、居間、茶間等設施{{r|文}}。
