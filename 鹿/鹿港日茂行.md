'''鹿港日茂行'''是位在臺灣[[彰化縣|彰化縣]][[鹿港鎮|鹿港鎮]]的縣定古蹟，是原籍福建[[永寧鎮_(石獅市)|永寧]]的[[林振嵩|林振嵩]]所開設之船頭行，為鹿港乾隆至道光年間（1785－1845左右<ref>[http://idbpark.moeaidb.gov.tw/%E5%85%A8%E5%9C%8B%E5%B7%A5%E6%A5%AD%E5%8D%80/%E5%B7%A5%E6%A5%AD%E5%8D%80%E4%BB%8B%E7%B4%B9/%E5%BD%B0%E6%BF%B1%E5%B7%A5%E6%A5%AD%E5%8D%80/08.asp?Zone=%B9%FC%C0%D8%A4u%B7~%B0%CF&UnitCode=NA0000&Section=%A4%A4%B3%A1%A4u%B7~%B0%CF 台灣工業用地供給與服務資訊網——彰濱工業區——自然環境]{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>）全盛時期之鹿港八郊中的泉郊裡最大的船頭行，乾隆至道光二十年（1840年）之間為鹿港首富{{r|文}}{{r|彰}}。

1998年時尚未指定為古蹟的日茂行因鹿港鎮公所的道路開闢工程遭到拆除，引起林家後裔與當地有心保護古蹟者反彈而向彰化縣政府反應，工程因而暫緩，最後日茂行在2000年7月14日正式公告為古蹟，2003年進行修復，於2005年12月完工{{r|文}}{{r|日}}{{r|彰}}。

== 沿革 ==
[[File:日茂行日茂埕.JPG|200px]]

日茂行創辦人林振嵩於乾隆三十年（1765年）時來到鹿港，最初販鹽致富，後來改從事貿易，創辦「日茂行」{{r|日}}。清乾隆四十九年（1784年）鹿港成為正口，與泉州蚶江通商{{r|彰}}，林振嵩可能於此時於今址建設日茂行之建築{{r|日}}。此時林振嵩三子[[林文濬|林文濬]]來臺協助父親經營日茂行業務，使得日茂行蓬勃發展，成為鹿港首富，日後在乾隆五十一年（1786年）時日茂行已有雄厚經濟實力可以承擔[[鹿港龍山寺|鹿港龍山寺]]的遷移工程{{r|日}}。

乾隆五十三年（1788年）林振嵩因母喪而返鄉，日茂行則交由三子林文濬負責，其業務日益發展，在嘉慶二十一年（1816年）翻新改建{{r|日}}。而林文濬於道光六年（1826年）逝世後，由其五子[[林廷璋|林廷璋]]繼承日茂行，然而此時鹿港已經因為港口淤塞逐漸沒落，日茂行也由於鹿港失去航運功能而衰微{{r|日}}。

二次大戰之後，日茂行曾在民國五十六年（1967年）進行整修{{r|文}}。民國六十三年（1974年）時[[漢寶德|漢寶德]]等學者專家曾受委託對鹿港古老建築評鑑，此時日茂行並未被列為古蹟，後來在民國八十七年（1998年）5月17日時鹿港鎮公所開闢道路時拆除了日茂行部分結構，引起林家後代與當地有心人士不滿並向彰化縣政府反應，而使得工程暫停{{r|日}}。而在當年10月1日，在日茂行舉辦了「歷史之心」活動，企圖結合前衛藝術家的表現能力來再現歷史空間，但過程中卻因為裝置藝術之設置而引發地方居民與學者、藝術家之間的爭議<ref>{{cite book|author=廖昭雅|title=《政府指定古蹟對古蹟既有利益者之[[利益衝突|利益衝突]]之研究——以彰化縣鹿港鎮日茂行為例》 |publisher=逢甲大學土地管理研究所 |location= |isbn= |pages=3頁 |date=2007年6月}}</ref>{{r|日}}。後來在文化資產保存法修正後的民國八十九年（2000年）6月20日，於彰化縣縣定古蹟審查會上將日茂行列為縣定古蹟，並於同年7月14日公告{{r|日}}。

== 建築 ==
[[File:日茂行小合院.JPG|200px]]
日茂行為磚木建築，原有三進，但第一進部分已經不存，今只餘後兩進，均面寬三開間，旁邊有一小合院{{r|文}}。門廳前有一大埕，擺有許多磉石，是過去郊商往來所用的旗杆座及酬神時演戲用的戲臺石{{r|彰}}。

其內文物有林文濬所書的「日茂行」門額與「鰲波東注」、「大觀」等匾{{r|彰}}，此外還有民國五十六年（1967年）重修時由鹿港匠師施金義承做之正廳彩繪、和美陳穎派施作之彩繪與書法家歐陽錦華的墨跡，但其中「日茂行」門額在道路工程中受損斷成兩截，現由其後人保管{{r|文}}。
