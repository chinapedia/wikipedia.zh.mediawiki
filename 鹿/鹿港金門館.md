'''鹿港金門館'''位於臺灣[[彰化縣|彰化縣]][[鹿港鎮|鹿港鎮]]，是清代的班兵伙館{{r|卓|page=14}}，同時供奉分香自[[金門|金門]]東門內校場觀德堂的[[蘇府王爺|蘇府王爺]]{{NoteTag|1=再往上追溯可到金門的新頭伍德宮{{r|卓|page=4}}。}}{{r|彰}}{{r|研|page=6}}，於民國八十九年（2000年）10月25日公告為古蹟{{r|文}}。該建築原為[[許樂三|許樂三]]的宅邸，後來此宅在清[[嘉慶|嘉慶]]十年（1805年）時薄賣改為浯江館，後來最晚在清末時改稱金門館{{r|研|page=31}}。

而金門館所在的地區，原是水師汛與理番同知署等官兵居住區，所形成的街廓型聚落與附近並未有血緣淵源，類似後來的[[眷村|眷村]]{{r|卓|page=28}}。而除了在鹿港有金門館外，在[[艋舺|艋舺]]（今[[臺北市|臺北市]][[萬華區|萬華區]]）與安平（[[臺南市|臺南市]][[安平區|安平區]]）亦設有金門館{{r|卓|page=47}}。

== 沿革 ==
[[File:鹿港金門館浯江館匾額.jpg|thumb]]
鹿港金門館在清末之前原稱「浯江館」，本為仕紳許樂三的宅邸，在[[乾隆|乾隆]]五十一年（1786年）前已存在{{NoteTag|1=許樂三在乾隆五十一年（1786年）與鹿港富商[[林振嵩|林振嵩]]一同經理[[鹿港龍山寺|鹿港龍山寺]]的重修事宜，可見此前許樂三已居於鹿港{{r|卓|page=11}}。而〈重建浯江館碑記〉用「宇」來稱呼其宅邸，且其宅邸曾作為[[福康安|福康安]]辦公之衙署，可知其宅邸有相當程度的規模{{r|卓|page=11}}。}}{{r|研|page=11}}。在乾隆五十二年（1787年）福康安率大軍自鹿港登陸處理[[林爽文事件|林爽文事件]]以及乾隆五十三年（1788年）安平水師左營遊擊一員移駐鹿港後，鹿港增加了不少來自金門的兵丁，遂有建館並奉祀其神祇蘇王爺之需要{{r|卓|page=12}}。嘉慶十年（1805年）許樂三遺命其子將住居薄賣改為浯江館，但並非只有[[金門|金門]]人士可在此休憩，〈重建浯江館碑記〉便提到說「故凡官兵斯鎮及弁丁輿夫、彼都人士，無不憩息其間」{{r|研|page=11}}。

[[道光|道光]]十一年（1831年）時，鹿港[[游擊_(官名)|遊擊]][[溫兆鳳|溫兆鳳]]倡議重建，但未動工便陞任艋舺營[[參將|參將]]，後來繼任的劉光彩{{NoteTag|1=福建同安人，行伍出身，道光十一年（1832年）調任安平水師協左營鹿港遊擊，道光十四年（1834年）仍在任{{r|卓|page=13}}。}}繼續倡議，該工程由[[鄭用錫|鄭用錫]]擔任總理{{NoteTag|1=鄭用錫之先祖來自金門{{r|卓|page=14}}。}}，於道光十二年（1832年）二月動工，道光十四年（1834年）四月落成，耗資2600銀元{{r|彰}}{{r|研}}{{r|卓|page=14}}。此次重建工程有劉光彩的〈重建浯江館碑記〉為記錄，且附有捐題碑{{r|卓|page=15}}。由碑文之記載可以發現，當時捐款贊助的將兵全是水師，而且範圍遍及今天的臺南、澎湖、鹿港、萬華（艋舺）、淡水（滬尾），而且絕大部分並非金門人，故學者卓克華認為金門館雖冠以「浯江」或「金門」二字，但其實不只是金門會館而是水師會館或水師伙館{{r|卓|page=17、18}}。此外卓克華指出，此次金門館的重建，表面上是因為棟宇摧殘，但其實另一個目的是為了解決班兵運配的問題，使金門館可以作為班兵在等候船隻時短期居留的場所{{r|卓|page=25}}。而這次的重建，大致奠定今日的廟貌{{r|卓|page=49}}。

清[[咸豐|咸豐]]五年（1855年）再次重修，一說是受到道光二十八年（1848年）[[1848年彰化地震|彰化地震]]影響之故{{r|彰}}，另一說則認為此次重修並未動到主體建築，而是金門館擁有的店屋{{NoteTag|1=根據咸豐五年（1855年）重修的捐題碑，可知當時金門館的公業有「新興街頭瓦蓋公店」（每月收2000文）、金門館旁公店（每年收一次，專用於劉光彩之華誕）、地藏王廟口瓦蓋公店與灰窯（每年稅銀24元）{{r|卓|page=35}}。}}與廟旁邊室，並新建劉光彩的祠堂{{r|研|page=31}}{{r|卓|page=37}}。[[同治|同治]]年間據說又有重修過一次{{NoteTag|1=根據日治時期在明治卅年（1897年）所作的調查記錄（出自《臺灣總督府公文類纂宗料史料彙編》），金門館在同治十三年（1874年）重修；另根據大正年間的《寺廟臺帳》，則記載是在同治四年（1865年）由郭祈盛發起募捐改築{{r|卓|page=40、41}}。但由於兩者年代相距過近，此段時間鹿港又無重大事件，不太符合一般廟宇的重修間隔，學者卓克華認為在無其他史料佐證之下，僅稱在同治年間有重修過較好{{r|卓|page=41}}。}}{{r|研|page=31}}。此外約在咸豐年間，浯江館可能已經改名成今天的「金門館」，光緒年間「金門館」之名已正式出現在文書上{{r|卓|page=48}}。

日治時期，根據大正年間的《寺廟臺帳》，金門館曾在[[明治|明治]]四十一年（1908年）於管理人郭文獻倡修下募集400餘金進行重修{{r|卓|page=40}}。此外《寺廟臺帳》提到金門館過去因治病靈驗而有眾多信徒，且金門移民信仰深厚，但由於後來當地移出者多，靈顯事蹟少，導致信徒日益減少{{r|卓|page=43}}。

二次大戰結束初期，金門館的信徒侷限在龍山寺附近的30多戶人家，香火不盛{{r|卓|page=44}}。民國六十一年（1972年）一度重修，且有當時的金門縣長郝成璞贈「宏揚先緒」匾{{r|卓|page=44}}，卓克華據此推測此時金門館可能已經和金門祖廟有所往來{{r|卓|page=45}}。三年後（1975年）管理人卓神保發起重修，修復正殿、廟房、前殿等處{{r|彰}}{{r|研|page=30}}{{r|卓|page=45}}。民國八十三年（1994年）6月再次重修並在民國八十八年（1999年）9月1日舉行安座大典{{r|彰}}{{r|研|page=30}}，期間金門館組成奠安香火隊回到祖廟金門新頭伍德宮祭拜{{r|卓|page=45}}。

== 建築 ==
金門館為兩進三開間的建築，坐北朝南，與鹿港其他寺廟朝向河口的布局不同{{r|研|page=43}}。此外金門館與一般寺廟不同，在三川殿與正殿兩側皆設有房室，供班兵移防候船暫居{{r|研|page=51}}。

== 祭祀活動 ==
根據咸豐五年（1855年）重修的捐題碑，可知當時金門館的祭祀活動有農曆四月十二日的「王爺聖誕」、七月十二日的普渡「增普」，此外在八月廿二日還會慶祝「劉協臺華誕」以感念劉光彩之功績{{NoteTag|1=使用「華誕」一詞表示當時劉光彩仍在世{{r|卓|page=36}}。}}{{r|卓|page=35、36}}。
