'''鹿野圳'''為[[臺灣|臺灣]][[臺東縣|臺東縣]][[鹿野鄉|鹿野鄉]]境內的一座[[灌溉|灌溉]][[水圳|水圳]]系統，其最早興建於西元1953年(民國42年)耗時約4年於1957年(民國46年)完工通水，目前由[[臺東縣|臺東]][[農田水利會|農田水利會]][[鹿野鄉|鹿野]][[工作站|工作站]]進行管理與維護。

== 沿革 ==

=== 鹿野圳 ===
鹿野圳於西元1953年(民國42年)於由當時[[台灣省政府|台灣省政府]][[水利局|水利局]]規畫並興建，在1957年(民國46年)全數完工，總工程時數耗時約4年，然自西元1970年(民國59年)起，鹿野圳下游的[[鹿寮圳|鹿寮圳]]因進水口河床刷深無法順利取水，同時因鹿野大圳進水口至鹿寮圳進水口[[鹿寮溪|鹿寮溪]]河段於枯水期流量滲漏率平均高達30％，為方便取水及提高水源利用效率，遂兩條水圳之進水全面改由鹿野圳進水口共同取水，並經由鹿野大圳第1支線供水至鹿寮圳之灌溉區，因此鹿寮與鹿野兩圳灌溉系統合而為一。<ref name="台東農田水利會 專輯報導 鹿野大圳灌溉用水有效調配營運成果">{{cite web |url=http://www.ttia.gov.tw/page13-2.php |title=台東農田水利會 專輯報導 鹿野大圳灌溉用水有效調配營運成果 | accessdate=2014-12-10}}</ref>

鹿野工作站[[台灣日治時期|日治時期]]原屬[[關山鎮_(臺灣)|關山]][[水利組合|水利組合]]；[[臺灣戰後時期|二戰後]]於1946年(民國35年)改組為[[關山鎮_(臺灣)|關山]][[水利協進會|水利協進會]]；於1948年(民國37年)再度改為[[關山鎮_(臺灣)|關山]][[水利委員會|水利委員會]]；1956年(民國45年)正式改為台東農田水利會鹿野工作站。<ref>{{cite web |url=http://www.ttia.gov.tw/page4_luye.php |title=台東農田水利會 鹿野工作站 | accessdate=2014-12-10}}</ref>

== 設施 ==
=== 導水設施 ===
鹿野圳共有7條支線包含第 1、2、3、4 支線與鹿寮圳灌區幹線屬於北方區， 5、6、7支線屬南方區。<ref name="台東農田水利會 專輯報導 鹿野大圳灌溉用水有效調配營運成果" />

2、3、4 支線灌溉範圍在'''鹿野鄉永安地區'''，第5、7支線則是在'''鹿野鄉高台龍田地區'''，第6支線目前已停止施工與通水。<ref name="台東農田水利會 專輯報導 鹿野大圳灌溉用水有效調配營運成果" />

=== 攔河堰 ===

鹿野圳攔河堰設置於進水口旁，以利於水流匯集至進水口。攔河堰靠近進水口一邊缺口為半自動倒伏壩，而攔河堰全長53.6公尺，寬9.3公尺，厚度12公分，原為混凝土鋼筋構造，後因颱風幾度侵襲，導致攔河堰主體時常遭洪水挾帶大量砂石磨損，於1999年(民國88年)台東農田水利會耗資新台幣3,922,000元進行改善，將鋼板加封在原有的混凝土鋼筋構造之上。<ref>{{cite web |url=http://student.kmes.tc.edu.tw/taiculture/water/14/C/04/11/C-14-04-11-01.htm |title=排灌系統 台東農田水利會 排灌系統 鹿野大圳 |accessdate=2015-01-22 |deadurl=yes |archiveurl=https://web.archive.org/web/20150122171314/http://student.kmes.tc.edu.tw/taiculture/water/14/C/04/11/C-14-04-11-01.htm |archivedate=2015-01-22 }}</ref>

=== 調整池 ===
* 馬背調整池

馬背調整池興建緣由因鹿野大圳第5、7支線的高台龍田灌區因圳路漸呈老化，滲漏情形嚴重，常無水供灌，前台灣省水利局於1988~1989年(民國77~78年)辦理「鹿野大圳、鹿寮圳灌溉用水調配及其灌溉系統改善規劃」；配合該灌溉系統改善規劃，自民國80年度起施做6萬噸調整池。<ref name="台東農田水利會 馬背調整池" >{{cite web |url=http://www.ttia.gov.tw/water_page_luye.php |title=台東農田水利會 馬背調整池 | accessdate=2014-12-10}}</ref>

調整池水引自鹿野大圳；池長115公尺、寬105公尺、深8公尺，面積約1.2公頃，總蓄水容量約60000公噸，為目前全臺灣[[噴灌|噴灌]]區最廣的人造[[蓄水|蓄水]][[農塘|農塘]]。<ref name="台東農田水利會 馬背調整池" />

=== 水利公園 ===
* 武陵親水步道

鹿野大圳自完工後通水至今，因[[武陵社區|武陵社區]]圳路漸呈老化，易崩塌且滲漏嚴重，不足以維下游約一千公頃農田灌溉之需；又因部分圳路穿過武陵社區，將該社區一分為二，且該段圳路荒草及垃圾充斥，雜亂無章，甚至有居民跌落而溺斃之事發生，固於2001年(民國90年)辦理鹿野大圳幹線改善工程前，經地方建議並與當地居民訪談溝通後，台東農田水利會決定在不影響原有灌溉管理需求下，在圳道上方加蓋[[水泥|水泥]]板，並配合社區特色並融入[[部落|部落]]文化，整建為綠意盎然，且充滿生命力的[[親水公園|親水公園]]。

* 馬背調整池水利公園

馬背調整池在興建時一併規劃了在周遭設置水利公園，水利公園包含一條以石組自然生態工法堆而成的溪道，流水清澈悠然，以及一座涼亭和花草營造的生態池和親水池帶。<ref>{{cite web |url=http://tour.taitung.gov.tw/zh-tw/Travel/ScenicSpot/275/%E9%A6%AC%E8%83%8C%E8%AA%BF%E6%95%B4%E6%B1%A0%E6%B0%B4%E5%88%A9%E5%85%AC%E5%9C%92 |title=台東縣政府 台東觀光網 馬背調整池水利公園 | accessdate=2014-12-10}}</ref>

== 改善工程 ==

* 鹿野大圳幹線改善工程

前台灣省水利局於1988~1989年(民國77~78年)對鹿寮溪水源之灌溉區域做通盤檢討，並進行辦理'''「鹿野大圳、鹿寮圳灌溉用水調配及其灌溉系統改善規劃」'''；將鹿野大圳灌溉系統，由後單期水田灌溉用水改為一年二期旱作[[噴灑灌溉|噴灑灌溉]]，其中第1～第4支線保持地表[[溝灌|溝灌]]，第5及第7支線則為旱作噴灑灌溉，期以少許之水量獲致全區域灌溉之功效。<ref name="鹿野大圳幹線改善工程">{{cite web |url=http://www.ttia.gov.tw/page6-4-5.php |title=台東農田水利會 鹿野大圳幹線改善工程 | accessdate=2014-12-10}}</ref>

因早期圳路設施老舊，皆為乾砌塊石，且年久失修時易崩塌外並滲漏嚴重，影響後半旱作灌溉區灌溉用水。台東農田水利會即擬定計畫分年分期予以改善，自1993年(民國82年)起至2007年(民國96年)止，分年編列預算辦理更新。<ref name="鹿野大圳幹線改善工程" />
