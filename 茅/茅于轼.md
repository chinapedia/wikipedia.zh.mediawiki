'''茅于轼'''（{{bd|1929年|1月14日|catIdx=M茅}}），[[江苏|江苏]][[镇江|镇江]]人，生于[[江苏|江苏]][[南京|南京]]，中国经济学家。2012年3月，茅于轼获得[[米尔顿·弗里德曼自由奖|米尔顿·弗里德曼自由奖]]。<ref name="fhwsa " /><ref name="dfcfwa ">{{cite web|url= http://finance.eastmoney.com/news/1365,20120405199395471.html|title= 茅于轼：我获奖说明中国成功 奖金全捐感言已写好|publisher= 东方财富网|accessdate= 2012年|work= |language= zh-cn|archive-url= https://web.archive.org/web/20120407221947/http://finance.eastmoney.com/news/1365,20120405199395471.html|archive-date= 2012-04-07|dead-url= yes}}</ref>他是天则经济研究所荣誉理事长，人文经济学会理事长。代表作有《择优分配原理》、《中国人的道德前景》。

== 生平 ==
1929年1月14日，茅于轼出生于[[江苏省|江苏省]][[南京市|南京市]]。他的祖父[[茅乃登|茅乃登]]是[[清朝|清朝]][[江浙联军|江浙联军]]革命军司令部秘书部的副长，父亲[[茅以新|茅以新]]是著名的铁路工程师，母亲是[[苏州|苏州]]大家闺秀，二伯父[[茅以昇|茅以昇]]是著名的桥梁专家。<ref name="fhwsa " >{{cite web |url= http://finance.ifeng.com/news/special/maoyushi/20120405/5858942.shtml|title=茅于轼：我获奖说明中国成功 奖金全捐获奖感言已写好 |publisher=凤凰网 |accessdate= 2012年|work=徐伟、巫秋君 |language =zh-cn}}</ref><ref name="dfcfwa " />
儿时，茅于轼的父亲[[茅以新|茅以新]]因为工作调动，全家经常搬迁，从小学到中学，他辗转读过13所学校。<ref name="fhwsa " /><ref name="dfcfwa " />
1946年，茅于轼考上[[南开中学|南开中学]]，<ref name="fhyb " />接着又考入[[国立交通大学|国立交通大学]]工商管理系，大二时转入机械系。<ref name="fhwsa " /><ref name="dfcfwa " />

1949年，茅于轼来到[[黑龙江省|黑龙江省]][[齐齐哈尔市|齐齐哈尔市]]的[[齐齐哈尔铁路局|齐齐哈尔铁路局]]工作，历任机车司炉、司机、技术员、工程师。<ref name="fhwsa " /><ref name="dfcfwa " />

1955年，调入铁道部[[中国铁道科学研究院|铁道科学研究院]]，任助理研究员，从事机车车辆性能研究。<ref name="sohu ">{{cite web |url= http://green.sohu.com/20090708/n265082028.shtml|title=经济学家、天则经济研究所所长茅于轼简介
 |publisher=搜狐网 |accessdate= 2009年|work= |language =zh-cn}}</ref>
1958年，[[反右倾|反右倾]]斗争开始，茅于轼被划分为“[[右派|右派]]”，他被降职降薪，妻子[[赵燕玲|赵燕玲]]被停薪留职。<ref name="fhwsa " /><ref name="dfcfwa " />
1960年，茅于轼被下放到[[山东省|山东省]][[滕县|滕县]][[劳动改造|劳动改造]]。<ref name="fhwsa " /><ref name="dfcfwa " />
1966年，[[文化大革命|文化大革命]]开始后，茅于轼和妻子一起被挂牌、[[批斗|批斗]]、强迫劳动，家被抄，并且被剃成光头，母亲也被牵连进来剃成“[[阴阳头|阴阳头]]”。<ref name="fhwsa " /><ref name="dfcfwa " />之后被调至[[大同机车厂|大同机车厂]]工作10年。
从1960年到1976年，茅于轼在劳动改造期间学习，在[[数学|数学]]、[[力学|力学]]、[[英语|英语]]方面有所領悟。<ref name="fhwsa " /><ref name="dfcfwa " />1975年开始从事运输经济、数理经济等微观经济学研究，1979年提出择优分配原理。<ref name="sohu " />

1984年9月，茅于轼从铁道部科学研究院调到[[中国社会科学院|中国社会科学院]][[美国研究所|美国研究所]]，1985年，任副研究员、研究员。<ref name="fhyb " />1986年赴[[美国|美国]][[哈佛大学|哈佛大学]]做注册访问学者。<ref name="fhyb " />1987年又担任[[非洲能源政策研究网|非洲能源政策研究网]]顾问。<ref name="fhyb " />1990年，担任[[澳大利亚|澳大利亚]][[昆士兰大学|昆士兰大学]]经济系讲师。<ref name="fhyb " >{{cite web |url= http://finance.ifeng.com/people/economist/maoyushi.shtml|title=茅于轼详细资料 |publisher=凤凰网 |accessdate= 2010年|work= |language =zh-cn}}</ref>
1993年，茅于轼和[[盛洪|盛洪]]、[[张曙光|张曙光]]等经济学家共同创建了由美国基金会以及私人资本资助的民间智库“[[天则经济研究所|天则经济研究所]]”<ref name="diaoxiao"/>{{noteTag|1=“天则”出自《[[诗经|诗经]]》“天生烝民，有物有则”，意思是：“合乎天道自然之制度规则”。 }} ，任所长、理事长，现为理事长、常务理事。<ref name="fhwsa " /><ref name="dfcfwa " />

== 家庭 ==
茅于轼保持着良好的生活习惯，每天早上6点起床，晚上十点就寝，他的日常生活排满了接待、采访、写文章和会议。<ref name="fhwsa " /><ref name="dfcfwa " />

1953年，出身于南京名门世家24岁的茅于轼在苏州认识了出身富商之家的助产学校学生19岁的赵燕玲。1955年茅于轼与赵燕玲结婚。夫妇俩育有一儿一女。儿子[[茅为星|茅为星]]出生于1957年10月。

== 奖项 ==
2012年3月下旬，美国智库卡托研究所(Cato Institute)宣布，83岁的茅于轼获得该所颁发的2012“[[米尔顿·弗里德曼自由奖|米尔顿·弗里德曼自由奖]]”，奖金为25万美元。茅于轼在接受《中国经济学人》采访时表示：“CATO给我这个奖，说明中国在自由争取方面有了很大成功。跟其他发展中国家比，中国在这方面的成功是很了不起的。其中有我们的一份力量吧，所以这个奖，不光是给我，也是给中国所有促进自由的人一个鼓励。”<ref>{{cite web|url=http://www.zaobao.com/zg/zg120331_007.shtml | language= | title=茅于轼获弗里德曼自由奖；官媒称自由派促中国多元化 |publisher=联合早报 |date=2012-03-31 |accessdate= }}</ref>茅于轼及夫人于5月4日在华盛顿出席颁奖仪式<ref>{{cite web|url=http://www.thechinatimes.com/online/2012/05/3350.html|title=Economist Mao Yushi Awarded $250,000 Milton Friedman Liberty Prize|language=en|publisher=The China Times|accessdate=2013-07-02}}</ref>，并发表获奖感言<ref>{{cite web|url=http://www.thechinatimes.com/chinese/online/2012/05/1437.html|title=茅于轼弗里德曼促进自由奖获奖感言|language=zh-cn|publisher=The China Times|accessdate=2013-07-02}}</ref>。

== 争议 ==

=== 钓鱼岛 ===
2012年8月，茅于轼在[[凤凰卫视|凤凰卫视]]《[[一虎一席谈|一虎一席谈]]》时说：“……老百姓关心的是什么，生命财产的安全，希望能够交朋友，到[[日本|日本]]到[[台湾|台湾]]去旅游，做生意，交换信息，互相交朋友享受人生，这个是我们老百姓最关心的事，而且我觉得[[钓鱼岛|钓鱼岛]]没有常住人口，没有[[GDP|GDP]]，更没有税收，如果海水涨了把它淹了对全世界任何一个国家任何一个老百姓不会有任何的影响，我们为了这件事造成冲突我觉得这个太不合算了。”<ref>{{cite web |url=http://news.ifeng.com/mainland/special/diaoyudaozhengduan/content-2/detail_2012_08/20/16933765_0.shtml |title=茅于轼：钓鱼岛没有GDP 中日为此冲突不合算 |accessdate=2012年8月20日 |publisher=凤凰网 |work= |language=zh-hans}}</ref><ref>{{cite web |url=http://news.sina.com.cn/w/2012-08-16/172424981208.shtml |title= 网友总结名人钓鱼岛不当言论 痛斥“汉奸”|accessdate= 2012年8月16日|publisher=新浪网 |work= |language=zh-hans}}</ref>又在[[新浪微博|新浪微博]]说：“……请问，是领土完整重要，还是百姓的生命财产重要？我认为当然是百姓的生命财产更重要。领土不完整，少了一块，于我何干呢？当然如果那块土地上有我们的同胞，这块土地被别人拿去后这些同胞要做亡国奴，那么我们有义务保卫国土。如果那是一块连人都没有的荒岛，争这块领土就毫无意义。或者这块土地上的百姓归属别人管理之后，生活反而提高了，自由反而扩大了，那么这种领土主权的转移，不但不必反对，还值得欢迎。”<ref>{{cite web |url=http://finance.jrj.com.cn/book/2012/08/25171014263469.shtml |title=茅于轼：钓鱼岛只是荒岛 丢掉了于我何干？  |accessdate=2012年8月25日 |publisher=金融界 |work= |language=zh-hans}}</ref>

=== 航母 ===
茅于轼反对造[[航母|航母]]，“我不同意拿国家的钱去造航空母舰。造航空母舰增加国家的国防力量这是好事，但是你没有看到如果你增加国防力量，别人也增加国防力量，这个力量就抵消了，你还是没有占到便宜。不造航母，钱可以用于消除贫困、[[艾滋病|艾滋病]]治疗、改善[[气候变暖|气候变暖]]等等困扰的问题。”<ref name=" tangji" >{{cite web |url=http://finance.jrj.com.cn/people/2011/04/0315149639755.shtml |title= 茅于轼：贫富最大差异反映在住房上 |accessdate= 2011年4月3日|publisher=金融界 |work= |language=zh-hans}}</ref>

=== 经济适用房 ===
茅于轼反对建造[[经济适用房|经济适用房]]，“这几年政府用了很多心思，建了公租房、廉租房等等，城市户口住的问题确实有很大的改善，但是仅限于户籍人口，我觉得这个政策是有很大毛病的。所以我反对经济适用房，这是我一直反对的，经济适用房第一没有效率，第二没有公平。”<ref name=" tangji" />

=== 毛泽东 ===
2011年4月26日，茅于轼發表《把毛泽东还原成人》，提出把[[毛泽东|毛泽东]]还原成人，不再是神，接受公正的评定。<ref>{{cite web |url=http://finance.jrj.com.cn/people/2011/06/22095410264920.shtml |title=茅于轼：对毛泽东的再认识  |accessdate= 2011年6月22日|publisher=金融界 |work= |language=zh-hans}}</ref>2013年8月31日，茅於軾發表《把毛澤東還原成人》修改版，認為毛澤東因為迷信權力，喪失了起碼的理性，從一個政治家淪落為和人民對立的人民公敵。<ref>{{cite web | url=http://history.dwnews.com/news/2013-09-02/59324170-all.html | title=茅於軾：把毛澤東還原成人（修改稿）| date=2013-09-02}}</ref>

===  教育 ===
茅于轼认为，教育问题的最好的方法应该是“提高学费，通过提高的学费，以增加更多的奖学金和助学贷款，来解决穷人上学的问题。”<ref>[http://xbsb.gansudaily.com.cn/system/2008/01/07/010567907.shtml 茅于轼提高学费的主意有点馊]</ref>

=== 不支持计划生育 ===
茅于轼不支持中国大陆实施[[计划生育|计划生育]]政策，反而“主张放开三胎四胎，随便生，现在的计划生育政策太荒谬了。”<ref>{{cite news|author1=2012-03-02|title=两会问道之茅于轼论放开二胎|url=http://view.news.qq.com/a/20120228/000085.htm|accessdate=2017-10-10|agency=国际在线|publisher=腾讯网}}</ref>，同时他还认为“未来中国进入老龄化社会的速度比哪个国家都快，可能造成的问题也特别大。一胎化政策对当前的经济有很大的好处，降低了家庭的负担，提高了社会的储蓄率。这就是所谓的人口红利。但是这一份红利是寅吃卯粮得来的，将来老龄化社会降临时劳动力供给不足，欠帐是要归还的。”<ref>{{cite news|author1=王达水|title=茅于轼无视中国人口总量的全球性压力|url=http://opinion.hexun.com/2007-09-27/100864572.html|accessdate=2017-10-10|publisher=和讯网|date=2007-09-27}}</ref>。

===政治立场反复===
2013年，在被中国军方反美宣传片《[[较量无声|较量无声]]》直接点名后，茅于轼公开力挺[[中国共产党|中共]]，并删除了微博中不利于当局的言论，而且在接受媒体采访时表示自己希望共产党政权能够稳定地保持下去，并对以[[习近平|习近平]]为核心的现任领导“充满希望”。此事被一些网民讥讽为“叛变”和“神速跳忠”<ref>[http://china.dwnews.com/news/2013-11-01/59343046.html 茅于轼公开力挺中共 被指“叛变”] {{Wayback|url=http://china.dwnews.com/news/2013-11-01/59343046.html |date=20131221175904 }}.多维新闻网.[2013-11-01]{{zh-hans}}</ref>。
