'''茅港尾天后宮'''位於[[臺灣|臺灣]][[臺南市|臺南市]][[下營區|下營區]]茅港里，亦稱'''茅港尾堡天后宮'''{{r|張}}。據說該廟媽祖乃[[臺灣明鄭時期|明鄭時期]]來臺，廟史可追溯至永曆卅一年（1677年{{NoteTag|1=清康熙十六年。}}）{{NoteTag|1=相良吉哉《臺南州祠廟名鑑》則將創立年代記為康熙五十二年（1713年）{{r|相}}。}}{{r|張}}。

[[林爽文事件|林爽文事件]]中因[[梁朝桂|梁朝桂]]感念茅港尾天后宮媽祖保佑，奏請乾隆皇帝嘉勉，而後除敕封「天后宮」與賜匾外，並允許茅港尾媽祖前往[[臺灣府城|府城]]大天后宮時，有「[[拱辰門_(臺南市)|大北門]]進，大北門出」的特權{{r|張}}。此外在當地，還有關於「茅港尾媽祖」為了救民而洩漏天機，最後被[[上天|上天]]懲戒，[[禁錮|禁錮]]百年的傳說{{r|張2}}。

== 沿革 ==
[[File:茅港尾天后宮內景.JPG|thumb]]
[[File:茅港尾天后宮神昭海表匾額.JPG|thumb]]
[[File:茅港尾天后宮咸樂碑記.JPG|150px]]

茅港尾天后宮的歷史可追溯到明鄭時期，據說在永曆卅一年（1677年）時正式建廟{{r|張}}。而後據說在康熙四十六年（1707年）改建，康熙五十二年（1713年）竣工{{r|何|page=139}}。所在地茅港尾在過去是[[倒風內海|倒風內海]]熱鬧的港口之一，在明鄭時期的《梅氏日記》中被記作「Hoem Cangbooij」{{r|張}}，明鄭軍備圖寫作「梅港尾」{{r|張2}}。入清之後，茅港尾街成為[[臺灣府城|臺灣府城]]與[[諸羅縣城|諸羅縣城]]之間的重要市鎮與中繼站{{NoteTag|1=《[[諸羅縣志|諸羅縣志]]》中，寫說「邑治至府一路市鎮，此為最大」{{r|張}}。}}，諸羅縣知縣[[周鍾瑄|周鍾瑄]]在康熙五十五年（1716年）設有茅港尾公館{{r|張2}}。

而由於茅港尾街在交通要道上，清朝時的民變多會波及到茅港尾街{{r|張}}。林爽文事件時，梁朝桂領兵到灣裡街（臺南善化），茅港尾居民則組織義軍，以天后宮與觀音亭作基地，協助清軍收復[[鐵線里_(臺南市)|鐵線橋]]與[[鹽水區|鹽水港]]一帶{{r|張}}{{r|張2}}。事後梁朝桂感念媽祖庇佑與當地居民協助，奏請[[乾隆帝|乾隆帝]]嘉勉，於是在乾隆五十五年（1790年）敕封「天后宮」，賜「神昭海表」匾與「護國庇民」詔書，且給予茅港尾媽祖到府城時可以「[[拱辰門_(臺南市)|大北門]]進，大北門出」的特權{{NoteTag|1=臺灣府城鎮北坊為軍事重地，一般進香廟宇多從[[臺灣府城小南門|小南門]]進出{{r|張}}。而除了茅港尾媽祖之外，[[北港朝天宮|北港媽祖]]照例是從「小北門進，大北門出」，據說要到鎮北坊的「鎮臺衙」讓鎮臺參香後方可出城{{r|張|page=75}}。}}{{r|張}}。

道光廿六年（1846年）天后宮進行重修，當時所刻的〈咸樂碑記〉提到茅港尾媽祖相當靈驗，是他處媽祖所不能比擬{{r|張}}。而在道光、咸豐年間的[[府城迎媽祖|府城迎媽祖]]活動中，北港媽祖南下時必留宿於茅港尾天后宮{{r|張}}。然而在同治元年五月十一日（1862年6月7日）發生大地震，茅港尾街亦遭到重創，天后宮也因而傾圮並在日後衍生出傳說{{r|張2}}{{r|張}}。地震後，天后宮董事許媽超發起重建，同治十二年（1872年）許媽超因案被捕，由洪窮接任董事{{r|何|page=140}}。光緒六年（1880年）天后宮重建竣工{{r|張2}}，但原本三落的規模縮減為二落{{r|何|page=140}}。

[[臺灣日治時期|臺灣日治時期]]，由於茅港尾從昔日官道上的重要市鎮演變成[[曾文郡|曾文郡]][[下營庄|下營-{庄}-]]底下的一個[[大字|大字]]，地方財力大不如前，僅能集資進行小規模修繕，[[皇民化運動|皇民化運動]]期間則欠缺整修{{r|張}}。

[[二次大戰|二次大戰]]後，茅港尾天后宮於民國五十一年（1962年）在主事余滄波倡議下，由馮振安擔任主任委員，余滄波等六人擔任委員，進行天后宮的重建{{r|何|page=140}}。該年8月便拆除舊廟，9月於原址旁增購廟地六厘多{{r|何|page=140}}。而後在10月8日寅時動工，午時定分金，12月4日上樑{{r|何|page=140}}。工程先蓋後殿，在蓋前殿，於民國六十四年（1975年）8月竣工，隔年（1976年）建醮入火安座{{r|張}}{{r|張2}}{{r|何|page=140、141}}。

而在天后宮重建的同一時期，「重建茅港尾橋鐵線橋碑記」石碑也被人發現。據張耘書《臺南媽祖信仰研究》所寫，相傳約在民國五十年（1961年），媽祖指示信徒劉復朝，在他園中將會出土神器，而後在劉復朝園中挖出康熙五十七年（1718年）的「重建茅港尾橋鐵線橋碑記」石碑{{r|張}}。而隨著石碑的出土，當地出現重建天后宮的聲音，且當地傳說昔日茅港媽祖因洩漏天機而被禁錮，如今受戒百年期滿將再顯神威{{r|張}}。而何厚增《被遺忘的茅港尾——四百年史話》則寫說臺灣鄉土史家黃清淵曾欲尋找該石碑而不可得，後來劉復朝之女婿於民國五十二年（1963年）7月5日在田園挖土整地時，發現底下有什麼像是大石塊的東西，而後與路過的陳進德小心翼翼地整理周遭，才發現原來是座石碑{{r|何|page=101}}。在想起有此處藏有寶貝的傳聞後，他們走告村民合力將石碑挖出，放到天后宮前的大榕樹下{{r|何|page=101}}。而當時茅港尾天后宮已經在大興土木，石碑出土後每天下午廟頂會有一道光順著屋瓦由上而下慢慢移動而消失，圍觀的民眾都說這是茅港尾天后宮正「興」咧{{r|何|page=101}}。

茅港尾天后宮由於茅港尾的沒落，少有舉辦大型繞境活動{{r|張}}。民國九十九年（2010年）在茅港尾媽祖旨意下，才舉辦為期三日的「巡香賜福」遶境，並依清朝舊例前往臺南[[大天后宮|大天后宮]]會香{{r|張}}。

== 轄境 ==
茅港尾天后宮昔日的祭祀圈主要以茅港尾堡為主，有「堡頭廟」之稱，包括火燒珠、麻豆寮、洲仔、橋仔頭、甲中、十六甲、紅毛厝、下營、大屯寮、舊西寮、連表、右武衛、中營、南廍等聚落，相當於現在下營區的大屯-{里}-、大埤里、甲中里、紅厝里、賀建里、西連里與官田區的南廍里一帶{{r|張}}。古今相比之下，轄域變化不大，但人口數較有明顯變化，這主要與倒風內海消失，茅港尾失去港口機能，再加上天災等因素影響有關{{r|張}}。在同治、光緒年間，茅港尾街的地位便被麻豆給取代{{r|張}}。

== 傳說 ==
[[File:重建茅港尾橋鐵線橋碑記.jpg|150px]]
傳說在清同治元年（1862年）某日，天后宮突然[[降乩|降乩]]，指示近日將有大難，要街民當心{{r|張}}{{r|張2}}{{r|何|page=72}}。由於當時正逢[[戴潮春事件|戴潮春事件]]，茅港尾街民以為是[[戴潮春|戴潮春]]的部隊欲來入侵，遂躲在家裡嚴守家門，並在庭院與通道灑下豆粒，要阻撓入侵者{{r|張}}{{r|張2}}{{r|何|page=72}}。結果發生了大地震，居民因通道受阻，逃生不及，導致茅港尾街死傷慘重{{r|張}}{{r|張2}}。茅港尾媽祖則因為洩漏天機，遭[[上天|上天]]禁錮百年{{r|張}}{{r|張2}}{{r|何|page=72}}。而後在茅港尾媽祖禁錮期滿後，信徒們發起天后宮的重建{{r|何|page=72}}。
