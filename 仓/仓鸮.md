'''仓鸮'''（[[学名|学名]]：''{{lang|la|Tyto alba}}''）是分布最广的[[猫头鹰|猫头鹰]]种类，也是所有鸟类中分布最广的之一，属于[[鸮形目|鸮形目]]中两科之一的[[草鸮科|草鸮科]]。它广布于世界各地，除了[[極地|極地]]、[[荒漠|荒漠]]、喜马拉雅山脉以北、[[印度尼西亚|印度尼西亚]]的大部分地区和一些[[太平洋岛屿|太平洋岛屿]]<ref name = bruce1999>Bruce (1999) pp. 34–75</ref>。[[系统发生学|谱系学]]证据显示，仓鸮至少有三支主要世系：欧亚大陆和非洲一支，大洋洲一支，美洲一支，以及岛屿上极度趋异的[[分類單元|分类群]]。共约28个亚种间的体型和颜色差异很大，但体长大多在33至39厘米之间，翼展在80到95厘米之间。头背部的羽毛为斑驳的灰色或棕色，下体呈白色至棕色，有时带暗色斑点。面盘是标志性的心形，大部分亚种为白色。

绝大部分仓鸮为[[夜行性|夜行性]]动物，但在英国和一些太平洋岛屿，它们也在日间狩猎。仓鸮擅长猎杀地面上的动物，几乎所有的猎物为小型哺乳动物。它们的听觉十分敏锐，通过听声辩位的方式捕捉猎物。它们为终身配偶制，若配偶被杀，新的配对才可能会形成。繁殖时间视所处地域而定，每窝平均产4只蛋，营巢于空树洞、旧房屋或壁岩洞中。雌鸟负责孵蛋，雄鸟为雌鸟和雏鸟提供食物。当大量小型猎物能被轻易取得时，仓鸮的数量会快速增长。它们在全球范围被评估为[[无危物种|无危物种]]，但部分地区的一些亚种受威胁的程度较高。

== 生态环境 ==
倉鸮生活在高大乔木林中。

== 分布地域 ==
分布于[[南亚|南亚]]、[[北美洲|北美]]、[[南美洲|南美]]、[[中美洲|中美]]、[[欧洲|欧洲]]（东北部除外）、[[大洋洲|大洋洲]]、[[非洲|非洲]](撒哈拉沙漠地区除外)和[[马达加斯加岛|马达加斯加岛]]。在[[中国|中国]]，仓鸮分布在[[云南|云南]]（[[昆明|昆明]]、[[勐海|勐海]]）。

== 特征 ==
颜色为苍白色，翅膀和腿都较长，体长33-39厘米，双翅展开有80-95厘米。

== 食物 ==
主要以[[啮齿类|啮齿类]]动物为食，昼伏夜出，食量很大，一夜可以吃掉七只鼠类。

== 繁殖 ==
倉鴞每年繁殖2次，第一次從3月到6月初，第2次從9月到12月。野生仓鸮一般活到5－11年。
