'''湯前站'''（{{jpn|j=湯前駅|hg=ゆのまええき|rm=Yunomae eki|f=y}}）是一位於[[日本|日本]][[熊本縣|熊本縣]][[球磨郡|球磨郡]][[湯前町|湯前町]]，由[[球磨川鐵道|球磨川鐵道]]所經營的[[鐵路車站|鐵路車站]]，是該公司所經營的[[湯前線|湯前線]]之[[終點站|終點站]]。

== 車站構造 ==
月台2面1線的終點站、北側的站舍是有人站、車站下設有電話。
平常多使用北側月台、站舍和站舍以交流中心｢湯～とぴあ｣為出入口。
南側的月台有弧線的裝飾，平常較少使用，不過南側月台也有剪票口等設施。

== 車站週邊 ==
站前有[[九州產交巴士|產交巴士]]的湯前站前公車站和[[西米良村|西米良村營巴士]]的湯前公車站。

本站位於湯前町中心，附近有許多商店。

===鄰近===
*交流中心｢湯～とぴあ｣ - 和站舍相接。從這裡也可以進入月台。
*湯前町商工會館 - 約30公尺。
*湯前町立湯前漫畫美術館（[[那須良輔|那須良輔]]記念館、{{tsl|ja|くまもとアートポリス|熊本藝術建築}}（KAP）參賽建築）- 站東方約50公尺。步行1分鐘。
:展示[[諷刺|諷刺]][[漫畫家|漫畫家]]那須良輔的原畫等。
*[[國道219號|國道219號]] - 本站北側約100公尺。
*縣道43號錦湯前線 - 本站東側約100公尺。
*湯前[[郵局|郵局]] - 約400公尺。國道219號上。

===遠方===
*明導寺 - 約700公尺。熊本縣道43號錦湯前線上。
:重要文化財阿彌陀堂，為日本的佛教寺院，擁有特殊的建築。
*湯前町公所 - 約80公尺。位於熊本縣道43號錦湯前線上。
*肥後銀行湯前分行

== 歷史 == 
*1924年3月30日 設立。
*1987年4月1日 - [[國鐵分割民營化|國鐵分割民營化]]後成為九州旅客鐵道（JR九州）轄下的車站。
*1989年10月1日 - 湯前線交由[[球磨川鐵道|球磨川鐵道]]營運。
