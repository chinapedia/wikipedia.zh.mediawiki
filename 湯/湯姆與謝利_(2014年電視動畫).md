《'''湯姆與謝利'''》2014年系列（'''The Tom and Jerry Show'''，[[央视|央视]]引进版本翻译为“'''新版猫和老鼠'''”）是由Renegade Animation及[[華納兄弟動畫公司|華納兄弟動畫公司]]製作的[[美國|美國]][[電視動畫|電視動畫]]，建基於1940年[[湯姆與謝利|同名作品]]的角色和故事進行重製。節目於2014年3月1日率先透過[[Teletoon|Teletoon]]於加拿大推出，其後於2014年4月9日在美國[[Cartoon_Network|Cartoon Network]]首播。

==製作==
2012年，[[Cartoon_Network|Cartoon Network]]宣佈開始籌備一部《湯姆與謝利》的新系列電視動畫，預定52集，每集片長11分鐘，後來確認會遷就海外發行26集22分鐘版本。比起前作《[[湯姆傑利小故事|湯姆傑利小故事]]》較偏向兒童市場，官方宣稱今次製作會比過去更忠於1940年的早期電影[[短片|短片]]風格，主攻合家歡市場。<ref name="bcdb">{{Cite news|url=http://blog.bcdb.com/tom-jerry-show-coming-cartoon-network-4850/|title=The Tom and Jerry Show Coming to Cartoon Network|publisher=Big Cartoon News|date=2012年10月8日|deadurl=yes|archiveurl=https://web.archive.org/web/20121025033439/http://blog.bcdb.com/tom-jerry-show-coming-cartoon-network-4850/|archivedate=2012年10月25日}}</ref>

實質動畫製作是由Renegade Animation負責，主要利用[[Adobe_Flash|Adobe Flash]]軟件繪製，由公司主席之一達雷爾·溫·薜他斯身兼執導和監製工作。節目原定於2013年推出，其後因為製作進度落後而順延。<ref>{{cite web|title=Cartoon Network Upfront Presentation 2013|url=http://animatedtv.about.com/od/picturegallerie1/ss/Cartoon-Network-Upfront-Presentation-2013_9.htm|publisher=About.com|accessdate=2014年3月10日|deadurl=yes|archiveurl=https://web.archive.org/web/20140103190551/http://animatedtv.about.com/od/picturegallerie1/ss/Cartoon-Network-Upfront-Presentation-2013_9.htm|archivedate=2014年1月3日}}</ref>內容於美國境內版定為「TV-PG」級別，破壞效果的誇張程度介乎於[[短片|短片]]和《湯姆傑利小故事》之間。<ref>{{cite web |url=http://www.cartoonnetwork.com/tv_shows/tomjerry/video/new-tom-and-jerry-show-clip.html |title=New Tom and Jerry Show - The Tom and Jerry Show Videos â€“ Cartoon Network |publisher=Cartoonnetwork.com |date= |accessdate=2014年4月2日 |deadurl=yes |archiveurl=https://web.archive.org/web/20140403035252/http://www.cartoonnetwork.com/tv_shows/tomjerry/video/new-tom-and-jerry-show-clip.html |archivedate=2014年4月3日 }}</ref>片中貫徹湯姆和謝利沉默，而其他角色可以隨便出聲的風格，但特別沿用短片導演[[威廉·漢納|威廉·漢納]]當年為湯姆灌錄的經典呼叫聲效，以及片頭主題音樂。

== 故事 ==
兩位角色－－'''湯姆'''和'''謝利'''會和其他短片出現過的角色，一同被放置在各個不同的常設世界觀當中。縱使背景和基本人物關係相同，每集世界觀相同的故事均會在既有設定下重設，並不存在一條連貫的時間線。目前常設的世界觀例如：

; 住家
: 在現代小鎮中，湯姆和鬥牛犬史派克住在一對夫妻主人的獨立屋，謝利則住在居所的老鼠洞中。湯姆最初是跟隨女主人小姜，而史派克則是跟隨男主人里克，並有一名兒子泰克。住在他們附近有家貓圖黛絲，以及野貓布奇。

; 魔法
: 湯姆是一對中世紀姊妹魔女——比媞與曉蒂的隨從，謝利則住在工房的老鼠洞中。長期有一隻蠑螈紐特作為實驗對象同住。工房所處的森林外方，有著各種中世紀庶民和貴族。

; 偵探
: 湯姆是一名人類偵探在辦公室飼養的寵物，於夜裡會伴隨在老鼠洞的謝利，一同化身協助城中動物的偵探拍檔。全篇有著旁白引導。

; 科學
: 謝利跟一隻名為拿坡崙的老鼠、以及一隻倉鼠，被科學家畢格比博士作為智慧稍高的對象，在大廈裡的一間實驗室裡安樂生活。湯姆則住在實驗室樓層下的垃圾房，不時被清潔機械人招呼。
