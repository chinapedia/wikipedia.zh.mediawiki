《'''湯姆克蘭西：全境封鎖'''》（{{lang-en|''Tom Clancy's The Division''}}）是一款[[第三人称射击游戏|第三人稱]][[战术射击游戏|战术射击]][[動作角色扮演遊戲|動作角色扮演]][[電子遊戲|遊戲]]，由[[Ubisoft_Massive|Ubisoft Massive]]、[[Ubisoft_Reflections|Ubisoft Reflections]]和[[红色风暴娱乐|Ubisoft Red Storm]]共同開發，屬於[[湯姆·克蘭西|湯姆·克蘭西]]系列品牌遊戲，支援[[Microsoft_Windows|Microsoft Windows]]、[[PlayStation_4|PlayStation 4]]和[[Xbox_One|Xbox One]]平台<ref name="NBCNews">{{cite web|title=Ubisoft unveils Tom Clancy game 'The Division' |first=Yannick |last=LeJacq| publisher=NBC News|url=http://www.nbcnews.com/technology/ubisoft-unveils-tom-clancy-game-division-6C10272293|date=2013-06-10|accessdate=2013-06-10}}</ref>。遊戲最初是在2013年[[E3|E3]]的[[Ubisoft|Ubisoft]]記者會上公佈並同時播放一段7分鐘長的遊戲影片。最終於2016年3月8日发售，並在此後的一年中先後推出了三個大型DLC擴展包「地下世界」、「生存模式」及「最后一战」。

續作《[[湯姆克蘭西：全境封鎖2|湯姆克蘭西：全境封鎖2]]》安排於2019年3月發售。

== 遊戲概要 ==
《全境封鎖》的故事取材於{{link-en|暗冬行動|Operation Dark Winter}}和{{link-en|第51號總統指令|National Security and Homeland Security Presidential Directive}}。遊戲中一場在[[黑色星期五_(购物)|黑色星期五]]購物季期間藉由[[鈔票|鈔票]]等媒介散佈的疾病，讓[[美國|美國]]在5日之內崩解癱瘓<ref>{{cite web|last=Goldfarb|first=Andrew|title=E3 2013: Open World Tom Clancy RPG The Division Announced |url=http://www.ign.com/articles/2013/06/10/e3-2013-open-world-tom-clancy-rpg-the-division-announced |publisher=IGN |date=2013-06-10|accessdate=2013-08-20}}</ref>。玩家扮演的角色隸屬於一個名為「國土戰略局」（Strategic Homeland Division - SHD，正好與DHS - [[美國國土安全部|美國國土安全部]]的縮寫相反）的組織，簡稱「The Division」。該組織的目的是要擊退威脅，並儘一切手段來「保存剩下的事物」，并逐步逼近幕后黑手。在遊戲中玩家將面臨来自[[人工智慧|人工智能]]（AI）敌人（[[PVE|PVE]]）和其他玩家（[[PVP|PVP]]）的挑戰。

== 多人模式 ==
在2015年6月18日 Ubisoft SEA在其官方Youtube頻道發佈一條影片名為：《湯姆克蘭西：全境封鎖》揭露「暗區」，
探索 Tom Clancy's The Division《湯姆克蘭西：全境封鎖》當中的 PVP 地區：暗區（Dark Zone），一個位於曼哈頓中心一帶、充滿恐懼與背叛的區域，玩家將可隨時以無接縫地­圖形式在任何時間進入該區域。

在「暗區」中所有的特工都只是為了自己：留意敵方陣營並且保持對其他特工的警覺。­但既然這個區域如此危險、又為何要進入呢？一部份的答案是為了搜刮當初JTF(聯合特遣部隊)退出暗區而遺留下來之稀少和高價值的裝備，可以背叛同路人獨佔所有的物品<ref>[https://www.youtube.com/watch?v=IDQyKaUgr2A Tom Clancy's The Division《湯姆克蘭西：全境封鎖》揭露「暗區」/ Dark Zone Reveal [中文字幕<nowiki>]</nowiki> - Ubisoft SEA]</ref>。

在2017年2月更新的1.6版本中，游戏增加了新的追加下载内容“最后一战”（The Last Stand），并在其中增加了新的多人联机模式。在“最后一战”里玩家将被分为两个派系相互对抗，将在暗区内占领战术终端来获得胜利。游戏类似其他第一人称射击游戏的抢点模式。

== 組織 ==

===  國土戰略局(Strategic Homeland Division) ===
特工（玩家）所屬組織，各特工在平時以各行各業身份蟄伏在民間。爆發危機後，全員出動導正社會混亂秩序並回復政府機關的機能。特工執行任務時獨來獨往，必要時組成小隊行動。

在Green Poison疫情爆發期間，第一波出動後卻遇上極大困境並遭到某些特工背叛而行動失敗，導致聯合特遣部隊全面退出暗區。

===  聯合特遣部隊(Joint Task Force) ===
政府的統合任務部隊，由美軍特戰部隊、紐約州國民兵、警察、消防隊、醫療人員、環保局清潔隊、自願義工、民間工程運輸人員所組成，負責維持紐約的治安，物資運輸，提供醫療和安全區等所有活動。

===  暴徒(Rioters) ===
無所事事的難民，為生存不惜搶劫殺人，並無領導及組織性的集團。

===  淨化者(Cleaners) ===
天花疫情爆發後由駐守在[[紐約|紐約]]的環保局清潔隊和防疫人員所組成的集團，因在疫情爆發後被政府遺棄在曼哈頓，不再聽從政府，不論此人是否遭受感染，一律用火焰噴射器燒死，自認“淨化世界”是自身的使命。領袖為喬·費洛。後期分裂成數個派系，部分成員脫離組織加入聯合特遣部隊。

===  賴克斯幫(Rikers) ===
從賴克斯島監獄逃獄的囚犯，穿著服飾皆為獄服，具有極度暴力傾向，毫不在意深陷於水深火熱中的民眾。領袖為拉瑞·巴雷特。後期分裂成數個派系，部分成員脫離組織成為暴徒。

===  倖存者軍隊(Last Man Battalion) ===
先前受雇於高階[[華爾街|華爾街]]企業的[[私人軍事服務公司|私人軍事服務公司]]，但天花疫情爆發期間僱主不願撤離將他們棄在紐約，被拋棄的他們極度氣憤，以優越軍武及訓練企圖控制紐約。指揮官为查爾斯·碧利斯。後期分裂成數個派系，部分成員脫離組織加入聯合特遣部隊。

== 開發 ==
《全境封鎖》最初以[[遊戲機歷史_(第八世代)|第八世代遊戲主機]]為獨佔平台開發。然而在遊戲發表後不久，Ubisoft就在遊戲的官方[[Twitter|Twitter]]中表示：「我們針對次世代主機的體驗進行最佳化，但我們也並未排除未來其它的平台<ref>{{cite web|title=Tweet about other platforms|url=https://twitter.com/TheDivisionGame/status/344682013430804481|work=Twitter|accessdate=2013-06-12}}</ref>。」

2013年8月20日，Ubisoft宣布在使用者社群的建議下，《全境封鎖》將支援[[Microsoft_Windows|Microsoft Windows]]平台<ref>{{cite web |url= http://www.polygon.com/2013/8/20/4639462/the-division-confirmed-for-windows-pc |title= The Division confirmed for Windows PC |date=2013-08-20 |accessdate=2013-08-20 |work= Polygon |first= Jenna |last= Pitcher}}</ref>。在2014年游戏开发者大会上，育碧展出了為《全境封鎖》開發的Snowdrop Engine遊戲引擎。該引擎由Massive Entertainment負責開發，其特點是簡化遊戲開發中繪制貼圖，AI調節等流程，並且該引擎可以支持“所建即所見”，而且擁有強大的自動生成工具。<ref>{{cite web |url= http://gnn.gamer.com.tw/9/94809.html |title= “更好，而不是更大”《湯姆克蘭西：全境封鎖》引擎「Snowdrop」影像公開 |date=2014-03-24 |accessdate=2014-04-17 |work= 巴哈姆特 |first= 九郎十三 |last= }}</ref>

2016年2月9日，Ubisoft宣布《全境封锁》 开放测试将在2016年2月18日在XBOX ONE，2016年2月19日在PC和 PS4平台开放，2016年2月22日全平台开放测试结束，《全境封锁》将在2016年3月8日在[[PS4|PS4]]、[[XBOX_ONE|XBOX ONE]]和PC平台发售。

== 發行 ==
《全境封鎖》原定於2014年發售，<ref>http://www.vg247.com/2014/03/07/the-division-the-crew-still-slated-for-2014-release-by-ubisoft/</ref>不過在2014年5月時育碧宣布本作將延期到2015年發售。<ref>{{cite web|last=Corriea |first=Alexa Ray |title=Tom Clancy's The Division delayed to 2015 |url=http://www.polygon.com/2014/5/15/5720806/tom-clancys-the-division-delayed-2015|date=2014-05-15|accessdate=2014-05-19}}</ref> UBISOFT在2015年5月12號宣布遊戲確定將在2016年3月8日推出。而中文版亦同步上市。<ref>http://www.ubisoft.com.tw/tctd/</ref>

《全境封鎖》在發售后獲得了商業上的成功，在發售后24小時便超過了原先的紀錄保持者《[[看门狗_(游戏)|看门狗]]》打破了育碧自己的多項銷售紀錄，首周便獲得了超過3.3億美元的收入成績。成為育碧新打造的又一重量級[[知识产权|知识产权]]。不過育碧並没有透露實際銷量數據<ref>[http://www.polygon.com/2016/3/9/11188792/the-division-ubisoft-sales-records-day-one The Division breaks multiple Ubisoft sales records in its first 24 hours].Polygon.2016-03-09.[2016-03-09].</ref><ref>[http://www.polygon.com/2016/3/15/11240460/the-division-launch-week-sales-record-new-ip The Division eclipses Destiny with $330M opening week, best ever for a new franchise].Polygon.2016-03-15.[2016-03-09].</ref>
