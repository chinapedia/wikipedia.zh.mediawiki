'''仲川遙香'''（{{lang-ja|仲川 遥香}}，{{bd|1992年|2月10日|catIdx=Nakagawa haruka}}）是[[日本|日本]]女[[藝人|藝人]]，為女子偶像團體[[AKB48|AKB48]]、[[JKT48|JKT48]]前成員，出生於[[東京都|東京都]]。2007年以AKB48第3期生身分出道，2012年從AKB48移籍至JKT48後，轉以JKT48所在的[[印尼|印尼]]從事演藝活動。

== 來歷 ==
'''2006年'''
* 12月3日，參加AKB48第三期AKB48追加成員面試，並合格。

'''2008年'''
* 10月，與同屬Team B且同經紀公司的[[渡邊麻友|渡邊麻友]]、[[多田爱佳|多田爱佳]]和[[平嶋夏海|平嶋夏海]]組成「[[走廊奔跑隊|走廊奔跑隊]]」，並開始活動。

'''2009年'''
* 1月28日，以走廊奔跑隊的單曲《[[初戀衝刺／青色未來|初戀衝刺／青色未來]]》出道<ref>由於當時仲川還未曾被選為AKB48單曲的選拔成員，《初戀衝刺／青色未來》是其首張CD。</ref>。
* 8月23日舉行的「[[AKB48全國巡演_(2009年)|AKB104選拔成員組閣祭]]」演唱會中，宣布從12月開始改組至Team A。

'''2010年'''
* 6月，在[[AKB48第17張單曲選拔總選舉「向母親發誓，這是真的」|AKB48第17張單曲選拔總選舉「向母親發誓，這是真的」]]得到第20位，進入單曲選拔組，是仲川加入AKB48後第4年首次進入選拔。
* 9月21日，「[[AKB48第19張單曲選拔猜拳大會|AKB48第19張單曲選拔猜拳大會]]」得到第4位，進入單曲選拔組。

'''2011年'''
* 6月9日，「[[AKB48第22張單曲選拔總選舉|AKB48第22張單曲選拔總選舉]]」中得到第24位，成為[[AKB48#Under_Girls、Theater_Girls|Under Girls]]<ref>{{ja icon}}{{cite web |url=http://ameblo.jp/akihabara48/entry-10918551053.html |title=AKB48 22ndシングル選抜総選挙結果 |work=AKB48官方部落格 |publisher=[[AKB48|AKB48]] |date=2011-06-10  |accessdate=2012-04-16}}</ref>。

'''2012年'''
* 2月29日起，因聲帶結節的手術入院<ref>{{ja icon}}{{cite web |url=http://www.sponichi.co.jp/entertainment/news/2012/02/29/kiji/K20120229002733290.html |title=ＡＫＢ４８仲川遥香　声帯手術で休養「すぐに戻ってきますね」 |publisher=Sponichi Annex |date=2012-02-29 |accessdate=2012-04-16}}</ref>。3月7日出院，繼續演藝活動<ref>{{ja icon}}{{cite web |url=http://blog.oricon.co.jp/watarirouka/archive/9290/0 |title=みなさーん！はるごんぶろぐ(^ 3^)/ |work=[[走廊奔跑隊|走廊奔跑隊]]官方部落格 |publisher=[[Oricon|Oricon]] |date=2012-03-07 |accessdate=2012-04-16 |deadurl=yes |archiveurl=https://web.archive.org/web/20120314033310/http://blog.oricon.co.jp/watarirouka/archive/9290/0 |archivedate=2012-03-14 }}</ref>。
* 6月6日，「[[AKB48第27張單曲選拔總選舉|AKB48第27張單曲選拔總選舉]]」中得到第44位。
* 12月23日，移籍至JKT48。成為JKT48 Team J成員。

'''2013年'''
* 4月3日，在Google+上表明不參選「[[AKB48第32張單曲選拔總選舉|AKB48第32張單曲選拔總選舉]]」<ref>{{ja icon}}{{cite web |url=https://plus.google.com/102249965218267255722/posts/QSY4YWb8jrN |title=仲川遥香Google+ |work=仲川遥香Google+ |publisher=[[仲川遥香Google+|仲川遥香Google+]] |date=2013-04-03  |accessdate=2013-04-07}}</ref>{{R|Sponichi20130416}}。

'''2014年'''
* 4月26日，於「第一屆JKT48單曲選拔總選舉」中獲得13,276票，得第3位<ref>{{Cite web|url=http://jkt48.com/pemilihan2014/sokuho3?lang=id |title=PEMILIHAN MEMBER SINGLE KE-6 JKT48 |publisher=JKT48官方網站 |language=id |date=2014-04-26 |accessdate=2014-04-27}}</ref>。
* 6月8日，在Google+上宣布以后将不再作为AKB48成员活动，完全移籍至JKT48。

'''2015年'''
* 5月2日，在『JKT48第二回單曲選拔總選舉』以21882票獲得第2名
* 6月13日，被任命为Team T的队长。

'''2016年'''
* 2月28日，在《JKT48 Request Hour 30》中宣布畢業，將活動至2016年年底，而在結束JKT48活動後，會繼續留在印尼發展。
* 12月1日，在《JKT48 第一屆大組閣》移籍至JKT48 Team J成員。
* 12月30日，從JKT48正式畢業。

== 人物 ==
*是家中五個兄弟姊妹中的次女<ref>《[[duet|duet]]》2009年11月号</ref>。
*是[[早安家族|早安家族]]的團體[[℃-ute|℃-ute]]（特別是[[萩原舞_(歌手)|萩原舞]]）的粉絲，因此BLOG中常常有有關該團體的話題。
*2011年7月27日於《[[AKBINGO!|AKBINGO!]]》中透露父親為職業摔角選手。
*特長是運動能力強，在讀書時有玩棒球。
*與[[前田敦子|前田敦子]]為高中時代的同班同學，出道後仍是室友。
