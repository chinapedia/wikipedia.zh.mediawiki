{{作品名稱|name=''The Help''|hk=寫出友共鳴|tw=姊妹|cn=相助}}是一部2011年上映的美国电影。根据[[凱瑟琳·史托基特|凱瑟琳·史托基特]]（Kathryn Stockett）2009年出版的美国畅销同名小说改编而成，是[[塔特·泰勒|塔特·泰勒]]执导的第一部电影。

本片具有浓郁的地域特色与强烈的种族特色，揭示了[[美国黑人民权运动|美国黑人民权运动]]之前的社会面貌。

== 剧情 ==
背景为1960年代的[[密西西比州|密西西比州]]，讲述一位女大学生因看不惯美国社会对[[黑人|黑人]][[女佣|女佣]]的歧视现象，毅然执笔帮助这些女佣争取权利的故事。

1962年的密西西比州，黑佣在许多家庭依旧遭遇着不公平的待遇。22岁的白人女孩史基特·费兰刚完成[[大学|大学]]学业回到家乡。她接受了良好教育与思想启蒙，并梦想成为出色[[作家|作家]]。但她思想保守的母亲认为，女人最靠谱的归宿是有一桩好婚姻。史基特从小由黑人女佣带大，因此她和黑佣是好朋友，但这次回家她却发现女佣不告而别。 

在史基特的朋友家当帮佣的黑佣爱比琳·克拉克机灵能干、内敛沉着。另一名个性泼辣的黑佣米妮有着一手好厨艺，两人是最好的朋友。

个性善良的史基特一直对黑佣充满同情和爱心，当她发现黑佣在生活中时常得不到平等的待遇后，她决意实施一个计划：通过亲身采访在白人家庭工作的黑佣们的甘苦遭遇，将这些情况写成一本书，以一种平静而同样充满力量的行动去帮助黑人发起民权运动。

== 制作 ==
本片于2010年7月至10月间拍摄，大部分场景是在密西西比州[[利福勒縣_(密西西比州)|利福勒縣]]的Greenwood镇摄制。

== 评价 ==
代表性评价有：“一曲惊艳、温馨且甜美的希望之歌”，“《姊妹》绝不应被定位为那种揭露时代丑恶的经典，而是从丑恶中追寻美丽的颂歌。”，“纵观历史上任何一个暑假档期，都匮乏这种塞满了幽默与心碎的动人诗篇，也正是因此，才让本片成为一份惊喜之礼”，“出色的表演与深沉的剧情交响辉映，煲煮出一窝浓香醇厚的心灵鸡汤”<ref>[http://ent.sina.com.cn/m/f/2011-08-15/ba3386860.shtml 本周新片《相助》《死神来了5》《惊魂半小时》]</ref>。

== 票房 ==
首周末收2552万次于《[[猩球崛起|猩球崛起]]》排名第二，之后分别以2001万、1433万和1420万连续三周蝉联冠军，成为自2010年7月的《[[全面啟動|全面啟動]]》之后，时隔一年多的首部三连冠的影片<ref>[http://ent.qq.com/a/20110905/000418.htm 北美票房：《帮助》勇夺三连冠]</ref>。第五个周末收获869万次于新片《[[全境擴散|全境擴散]]》名列第二。
