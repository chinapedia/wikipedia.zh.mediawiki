'''相同方式共享'''（{{lang-en|Share-alike}}）是[[著作權|版权授权]]方面的术语之一，最初由[[创作共用|知识共享]]计划开始使用，其要求对作品进行重利用或分发时，采用与原作品相同或相似的许可协议<ref>{{cite web|url=http://opendatahandbook.org/en/glossary.html#term-share-alike-license|title=Glossary|accessdate=2012-03-05|language=en}}</ref>。带有相同方式共享要求的[[自由內容|自由内容]]或[[自由软件|自由软件]]许可称为[[Copyleft|Copyleft]]许可协议。

当前受支持的知识共享许可协议中，包含相同方式共享条件的有2种：知识共享 署名-相同方式共享（[[copyleft|copyleft]]的[[自由內容|自由内容]]许可协议）和知识共享 署名-非商业性使用-相同方式共享（专有许可协议）。

除版权外，[[授權_(法律)|专利授权]]领域中的一个类似计划也有使用这一术语<ref>{{Cite web|url=http://blog.iandavis.com/2011/08/08/share-alike-patents/|title=Share-Alike Patents|accessdate=2012-03-05|language=en}}</ref>。

== Copyleft ==
{{Main|Copyleft}}
'''Copyleft'''协议，或{{tsl|en|Gratis versus libre|免费与自由|自由}}的相同方式共享协议，是相同方式共享协议中最大的子类别，可分为[[自由內容|自由内容]]许可协议（如知识共享 署名-相同方式共享）和[[自由软件|自由软件]]许可协议（如[[GNU通用公共许可证|GNU通用公共许可证]]）。在作品中使用copyleft材料时，使用者通常会被要求将整个作品以copyleft形式发布，因此有批评者称这类许可协议为{{tsl|en|Viral license|传染性许可协议}}。Copyleft协议属于'''互惠许可协议'''，后者还包括一些非自由协议，如[[共享源代码|微软有限互惠许可证]]。

一些自由内容和自由软件许可协议不包括相同方式共享要求，这些许可协议属于[[寬鬆自由軟體授權條款|宽松许可协议]]。

== 知识共享 ==
现有的6种知识共享协议中，有2种相同方式共享协议（CC 署名-相同方式共享和CC 署名-非商业性使用-相同方式共享），它们均要求{{tsl|en|Attribution (copyright)|姓名標示}}。根据知识共享的说法，相同方式共享协议的作用包括防止用户在未来对衍生作品添加新的限制条款，以及衍生作品需采用与原作品相同或相似的许可协议。<ref>{{Cite web|url=http://wiki.creativecommons.org/Share_Alike|title=Share Alike|accessdate=2010-12-17|language=en}}</ref>

3.0和4.0版本的相同方式共享协议中包含有[[许可证兼容性|兼容性]]条款，允许知识共享声明其他许可协议与该版协议兼容，衍生作品可以不使用原作品的协议，而是使用这些兼容协议进行授权。

== 版本历史 ==
知识共享共发布了5个版本的BY-SA和BY-NC-SA协议（1.0、2.0、2.5、3.0和4.0）。
* 署名-相同方式共享 1.0 通用<ref>{{Cite web|url=http://creativecommons.org/licenses/by-sa/1.0/|title=Creative Commons — Attribution-ShareAlike 1.0 Generic — CC BY-SA 1.0|accessdate=2013-10-06|publisher=Creativecommons.org|language=en}}</ref>；署名-非商业性使用-相同方式共享 1.0 通用<ref>{{Cite web|url=http://creativecommons.org/licenses/by-nc-sa/1.0/|title=Creative Commons — Attribution-NonCommercial-ShareAlike 1.0 Generic — CC BY-NC-SA 1.0|accessdate=2013-10-06|publisher=Creativecommons.org|language=en}}</ref> – 发布于2002年12月
* 署名-相同方式共享 2.0 通用<ref>{{Cite web|url=http://creativecommons.org/licenses/by-sa/2.0/|title=Creative Commons — Attribution-ShareAlike 2.0 Generic — CC BY-SA 2.0|accessdate=2013-10-06|publisher=Creativecommons.org|language=en}}</ref>；署名-非商业性使用-相同方式共享 2.0 通用<ref>{{Cite web|url=http://creativecommons.org/licenses/by-nc-sa/2.0/|title=Creative Commons — Attribution-NonCommercial-ShareAlike 2.0 Generic — CC BY-NC-SA 2.0|accessdate=2013-10-06|publisher=Creativecommons.org|language=en}}</ref> – 发布于2004年5月
* 署名-相同方式共享 2.5 通用<ref>{{Cite web|url=http://creativecommons.org/licenses/by-sa/2.5/|title=Creative Commons — Attribution-ShareAlike 2.5 Generic — CC BY-SA 2.5|accessdate=2013-10-06|publisher=Creativecommons.org|language=en}}</ref>；署名-非商业性使用-相同方式共享 2.5 通用<ref>{{Cite web|url=http://creativecommons.org/licenses/by-nc-sa/2.5/|title=Creative Commons — Attribution-NonCommercial-ShareAlike 2.5 Generic — CC BY-NC-SA 2.5|accessdate=2013-10-06|publisher=Creativecommons.org|language=en}}</ref> – 发布于2005年6月
* 署名-相同方式共享 3.0 未本地化版本<ref>{{Cite web|url=http://creativecommons.org/licenses/by-sa/3.0/|title=Creative Commons — Attribution-ShareAlike 3.0 Unported — CC BY-SA 3.0|accessdate=2013-10-06|publisher=Creativecommons.org|language=en}}</ref>；署名-非商业性使用-相同方式共享 3.0 未本地化版本<ref>{{Cite web|url=http://creativecommons.org/licenses/by-nc-sa/3.0/|title=Creative Commons — Attribution-NonCommercial-ShareAlike 3.0 Unported — CC BY-NC-SA 3.0|accessdate=2013-10-06|publisher=Creativecommons.org|language=en}}</ref> – 发布于2007年3月
* 署名-相同方式共享 4.0 国际<ref>{{Cite web|url=http://creativecommons.org/licenses/by-sa/4.0/|title=Creative Commons — Attribution-ShareAlike 4.0 International — CC BY-SA 4.0|accessdate=2014-05-15|publisher=Creativecommons.org|language=en}}</ref>；署名-非商业性使用-相同方式共享 4.0 国际<ref>{{Cite web|url=http://creativecommons.org/licenses/by-nc-sa/4.0/|title=Creative Commons — Attribution-NonCommercial-ShareAlike 4.0 International — CC BY-NC-SA 4.0|accessdate=2014-05-15|publisher=Creativecommons.org|language=en}}</ref> – 发布于2013年11月

== 运用 ==
2009年6月，[[维基人|维基百科社群]]与[[维基媒体基金会|维基媒体基金会]]理事会同意将知识共享 署名-相同方式共享（CC BY-SA）协议用作维基百科和基金会其他网站的主要内容协议。知识共享称赞了此决定，认为是[[自由文化運動|自由文化]]和远见领导力的胜利。<ref>{{Cite web|url=http://creativecommons.org/weblog/entry/15411|title=Wikipedia + CC BY-SA = Free Culture Win!|language=en}}</ref>
