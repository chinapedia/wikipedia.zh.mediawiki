'''相對性理論'''（{{lang|ja|'''相対性理論'''}}），是來自日本東京的樂團，成立於2006年九月。自稱「Youtube影像時代的流行藝術家」<ref>[http://natalie.mu/artist/show/id/1194 ナタリー - 相対性理論のプロフィール]</ref>，積極進行LIVE活動與音樂合作。

== 簡介 ==
早期不論專輯封面或宣傳影片，皆不直接拍攝樂團成員面貌，除LIVE表演以外幾乎不在公開媒體露面。直到2009年音樂雜誌「STUDIO VOICE」製作了『相對性理論』特集，書中刊載了主唱{{tsl|ja|やくしまるえつこ}}的照片以及附錄{{tsl|ja|やくしまるえつこ}}的朗讀CD。<ref>[http://www.amazon.co.jp/dp/B002B69ID6 STUDIO VOICE (スタジオ・ボイス) 2009年 07月号]</ref>其後樂團成員才各自向外發展、開始接受普通媒體採訪。

早期製作以真部脩一為中心，專輯製作表多數標示真部脩一為作曲和作詞，實際上大部分歌曲為四位團員共作，純粹是當初登記錯誤。<ref>[http://www.cinra.net/interview/2010/03/17/000000.php 永井聖一インタビュー ：CINRA.NET] </ref>後期樂團成員們也漸漸發表各自作曲、作詞的作品。

樂團成員為流動制，比起一般樂團更強調企劃性質，不過樂團前期成員相當固定，直到2010年貝斯手真部脩一與鼓手西浦謙助不參加往後活動<ref>[http://kensukenishiura.blogspot.jp/2012/06/blog-post_08.html 西浦謙助official website - 2012年6月8日金曜日「相対性理論を聴いて下さっている皆様へ、大切なお知らせ」]</ref>，參與者才有較大的變動。主唱やくしまるえつこ在訪談表示「相對性理論就像[[軟體|軟體]]（Software）」。<ref>[http://musicmagazine.jp/mm/mm201104.html ミュージック・マガジン2011年4月号]</ref>

== 簡歷 ==
* 2006年9月　やくしまるえつこ、永井聖一、真部脩一、西浦謙助，四位成員組成『相對性理論』。
* 2007年6月15日　發行獨立製作專輯『{{link-ja|戚風主義|シフォン主義}}』。僅僅在LIVE活動和郵寄通路販售便賣出4000張專輯。<ref>「[http://natalie.mu/news/show/id/5892 相対性理論の「シフォン主義」がリマスター全国流通]」</ref>
* 2008年5月　[[MySpace|MySpace]]×特別企劃「Myx」特集介紹。
* 2008年5月8日　首張專輯『戚風主義』重新整理母盤，再次發行流通日本全國。<ref>[http://www.bounce.com/news/daily.php/13302 相対性理論、廃盤となっていた自主制作盤『シフォン主義』の全国流通が決定] {{webarchive|url=https://web.archive.org/web/20100601092739/http://www.bounce.com/news/daily.php/13302 |date=2010-06-01 }}</ref><ref>[http://natalie.mu/news/show/id/5892 相対性理論の「シフォン主義」がリマスター全国流通]</ref>
* 2008年5月11日　獲得[[TOWER_RECORDS|TOWER RECORDS]]週間獨立音樂榜第一名。<ref>[http://natalie.mu/news/show/id/7060 快挙！相対性理論がタワレコ全店でウィークリー1位]</ref>
* 2009年1月7日　發行第二張專輯『{{link-ja|Hi-Fi新書|ハイファイ新書}}』。獲得[[Oricon|Oricon]]週間排行榜第七名。
* 2009年5月12日　『戚風主義』獲得日本唱片店員主辦的「[[第一回_CD_Shop_大賞|第一回 CD Shop 大賞]]」。
* 2009年6月　音樂雜誌「STUDIO VOICE」七月號刊載相對性理論特集，隨誌附贈主唱やくしまるえつこ的「朗讀CD」。<ref>http://www.amazon.co.jp/dp/B002B69ID6 STUDIO VOICE (スタジオ・ボイス) 2009年 07月号</ref>
* 2010年1月6日　發行相對性理論＋{{link-ja|涉谷慶一郎|渋谷慶一郎}}合作單曲『{{link-ja|Notre Musique|アワーミュージック (相対性理論の曲)}}』。
* 2010年4月7日　發行第三張專輯『{{link-ja|Syncronicity|シンクロニシティーン}}』，獲得[[Oricon|Oricon]]週間排行榜第三名、當日排行第二名。
* 2010年4月　音樂雜誌「Sound&Recording Magazine」五月號刊載相對性理論特集，隨書附錄ＣＤ「シンクロニシティーン / リコンストラクチャーズ」（參與者包括鈴木慶一、渡邊琢磨、大谷能生、渋谷慶一郎）<ref>[http://natalie.mu/music/news/30018 サンレコ付録CDで相対性理論を鈴木慶一ら4名が再構築]</ref>
* 2010年9月29日　發行相對性理論與{{link-ja|大谷能生|大谷能生}}合作單曲『{{link-ja|暴亂與待機|乱暴と待機 (曲)}}』。
* 2011年4月27日　發行第四張專輯『{{link-ja|正確的相對性理論|正しい相対性理論}}』，獲得[[Oricon|Oricon]]週間排行榜第八名。
* 2012年6月9日　在官方網站未告知的情況下，貝斯手真部脩一與鼓手西浦謙助退出LIVE活動、而後也未參與樂團活動。僅西浦謙助在網誌表示「由於樂團成員是流動制的，因此不參與LIVE表演。」<ref>[http://kensukenishiura.blogspot.jp/2012/06/blog-post_08.html 西浦謙助official website - 2012年6月8日金曜日「相対性理論を聴いて下さっている皆様へ、大切なお知らせ」]</ref>
* 2013年7月10日　預計發行第五張專輯。
