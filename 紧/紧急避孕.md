{{Medical}} 
'''紧急避孕'''（{{lang-en|emergency contraception}}），指无保护措施的[[性生活|性生活]]后，或[[避孕|避孕]]失败后几小时或几日内，妇女防止非意愿性[[妊娠|妊娠]]的发生，而采取的补救避孕法，其中包括放置[[宫内节育器|宫内节育器]]和口服紧急[[避孕药|避孕药]]。

== 定义 ==

== 适应症 ==
# 避孕失败，包括[[安全套|安全套]]破裂、滑脱；未能做到体外排精；错误计算[[安全期|安全期]]；漏服[[短效避孕药|短效避孕药]]；[[宫内节育器滑脱|宫内节育器滑脱]]。
# 性生活未使用任何避孕方法。
# 遭到[[性暴力|性暴力]]。

== 方法 ==
# 宫内节育器：在无保护性生活后5日（120小时）之内放入，有效率达95%以上。
# 紧急避孕药种类及用法：主要有[[雌孕激素|雌孕激素]]复方制剂，[[单孕激素|单孕激素]]制剂及抗孕激素制剂3大类。
