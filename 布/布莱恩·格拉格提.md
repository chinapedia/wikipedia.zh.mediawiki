'''布莱恩·蒂莫西·格拉格提'''（{{lang-en|'''Brian Timothy Geraghty'''}}，{{bd|1975年|5月13日||}}）是一名美国电影和电视剧演员，知名于其在[[奥斯卡奖|奥斯卡奖]]获奖电影《[[拆弹部队|拆弹部队]]》中所扮演的角色，与[[丹泽尔·华盛顿|丹泽尔·华盛顿]]共演的广受好评的电影《[[迫降航班|迫降航班]]》，以及受到讚賞的[[HBO|HBO]]電視劇《[[酒私風雲|酒私風雲]]》中的角色<ref>{{Cite web |url = https://deadline.com/2013/02/brian-geraghty-joins-boardwalk-empire-441523/ |title = Brian Geraghty Joins ‘Boardwalk Empire’ |author = Nellie Andreeva |date = 2013-02-27 |publisher = Deadline Hollywood |language = en}}</ref><ref>{{Cite web |url = http://collider.com/ron-livingston-brian-geraghty-boardwalk-empire-season-4/ |title = Ron Livingston and Brian Geraghty Join Cast Of HBO's BOARDWALK EMPIRE |author = Allison Keene |date = 2013-02-27 |publisher = Collider |language = en}}</ref>。
