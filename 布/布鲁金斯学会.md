'''布鲁金斯学会'''（Brookings Institution），又稱'''布鲁金斯研究所'''或'''布魯金斯研究院'''<ref>[http://www.mac.gov.tw/ct.asp?xItem=62725&ctNode=6228&mp=1]</ref>，是[[美国|美国]]著名[[智库|智库]]之一<ref>[http://udn.com/NEWS/WORLD/WOR3/4682740.shtml 布魯金斯研究所 全球最頂尖智庫]</ref>，主要研究[[社会科学|社会科学]]尤其是[[经济|经济]]与发展、都市政策、政府、外交政策以及全球经济发展等议题<ref>[http://www.britannica.com/eb/article-9016635 Brookings Institution] ''Encyclopædia Britannica''.</ref><ref name="autogenerated1">{{cite web |url=http://www.brookings.edu/about/RobertSBrookings_bio.aspx |title=Robert Somers Brookings – Brookings Institution |publisher=Brookings.edu |date= |accessdate=2010-04-29 |deadurl=yes |archiveurl=https://archive.is/20121212145724/http://www.brookings.edu/about/RobertSBrookings_bio.aspx |archivedate=2012-12-12 }}</ref>，总部位于[[华盛顿特区|华盛顿特区]]。

1993至2002的國會記錄學術分析指出，布魯金斯研究院報告被美國[[保守派|保守派]]政治人物引用的次數幾乎和[[自由派|自由派]]政治人物一樣多；在[[宾夕法尼亚大学|宾夕法尼亚大学]]发布的2012年[[全球關鍵智庫指標|全球關鍵智庫指標]]报告中，布鲁金斯学会被评为全球最有影响力的智库<ref>University of Pennsylvania [http://www.gotothinktank.com/2012-global-goto-tank-index-report 2012 Global Go To Think Tanks Ranking Released] {{webarchive|url=https://web.archive.org/web/20130302102430/http://www.gotothinktank.com/2012-global-goto-tank-index-report/ |date=2013-03-02 }} 21 January 2013</ref>。

== 历史 ==
学会成立于1916年。现任学会主席为美国[[外交政策|外交政策]]专家、前[[美国副国务卿|美国副国务卿]]（1994-2001）Strobe Talbott。2010年学会的预算为9000万美圆。<ref>{{cite web |title= The Brookings Institution on Charity Navigator | url=http://www.charitynavigator.org/index.cfm?bay=search.summary&orgid=3390 | publisher=charitynavigator.org | accessdate=March 4, 2012}}</ref>

==中國==
布鲁金斯学会與[[清華大學|清華大學]]在[[北京|北京]]共同成立清华-布鲁金斯公共政策研究中心該中心成立于2006年，位于[[清华大学公共管理学院|清华大学公共管理学院]]，研究項目為中国经济社会变革並有维系良好的[[中美关系|中美关系]]等重要领域的目的。
