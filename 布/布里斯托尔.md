'''布里斯托'''（{{lang-en|Bristol}}，[[國際音標|讀音]]：{{Audio|en-uk-Bristol.ogg|/{{unicode|ˈbrɪstəl}}/}}），或译'''布里斯托尔'''，香港有戰前街道譯作碧仙桃<ref>見[[香港|香港]][[尖沙咀|尖沙咀]]街道[[碧仙桃路|碧仙桃路]] ''Bristol Ave''</ref>，[[英國|英國]][[英格蘭|英格蘭]][[西南英格蘭|西南]]區域的[[名譽郡|名譽郡]]、[[單一管理區|單一管理區]]、[[英國城市地位|城市]]。布里斯托西臨[[愛爾蘭海|愛爾蘭海]]，[[人口|人口]]410,500（[[英格蘭|英格蘭]]第8、英國第11多），自[[中世紀|中世紀]]起已是一個重要的商業港口，地位一度僅次於[[倫敦|倫敦]]，直到1780年代才被[[利物浦|利物浦]]、[[曼徹斯特|曼徹斯特]]、[[伯明翰|伯明翰]]超過。現今的布里斯托乃英國重要的[[航天|航天]]、高科技及金融貿易中心，擁有一個國際機場。全市有兩所大學，[[布里斯托大学|布里斯托大学]]和[[西英格兰大学|西英格兰大学]]。

布里斯托北、東、南三方都被其他的單一管理區包圍：東南與[[萨默塞特|萨默塞特]]郡的[[巴斯和東北萨默塞特|巴斯和東北萨默塞特]]相鄰，西南與[[萨默塞特|萨默塞特]]郡的[[北萨默塞特|北萨默塞特]]相鄰，東北與[[格洛斯特郡|格洛斯特郡]]的[[南格洛斯特郡|南格洛斯特郡]]相鄰。布里斯托距離西面的[[威爾斯|威爾斯]][[首府|首府]][[卡迪夫|卡迪夫]]約41[[公里|公里]]（26[[英里|英里]]），而英國首都倫敦，則位於該市東北方約131公里（81英里）的位置處。

2012年11月15日，民選布里斯托市長，不再是「市長閣下」（Lord Mayor）。<ref>{{PDF|[https://www.bristol.gov.uk/sites/default/files/documents/council_and_democracy/elections/Bristol%20Mayoral%20Election%20booklet%20%28Chinese%20version%29.pdf 2012布里斯托市長選舉手冊]}}</ref>

== 历史 ==
[[File:Robert_Ricart's_map_of_Bristol.png|left]]
布里斯托早在11世紀已建立了獨立的城堡，12世紀則發展成一個貿易港口。由於對外貿易興旺，13世紀布里斯托已是造船中心。在15至16世紀，歐洲展開了開發美洲新大陸的風潮，而對正大西洋的布里斯托，更加是奴隸貿易中心，大量奴隸被販賣到美洲。自1700年到1807年，超過二百艘販奴船停靠在布里斯托，販賣了超過五十萬名[[非洲|非洲]]黑人到[[美洲|美洲]]做[[奴隸|奴隸]]。

自十九世紀起布里斯托開始受到鄰近[[利物浦|利物浦]]市的競爭，布里斯托的對外港口地位已有所下降。這時布里斯托轉變為一個工業中心，並且開始有鐵路連接至首都[[倫敦|倫敦]]。隨著二次大戰後工業式微，加上戰後的破壞，布里斯托已不再是世界級都會。但經過戰後的重建，布里斯托卻反而成為英國的航天中心。

== 經濟 ==
[[File:Bristol_International_Balloon_Fiesta_August_14_2004.JPG|thumb]]
布里斯托的[[传统产业|传统产业]]以[[农业|农业]]、[[纺织业|纺织业]]和工程业为主。[[航空|航空]]工业始于1910年，历史悠久，实力强大。[[协和飞机|协和飞机]]的制造商之一、著名的[[英国航空公司|英国航空公司]]在布里斯托有超过5000員工；而国际[[航空|航空]]和[[工业|工业]]动力界巨头[[劳斯莱斯|劳斯莱斯]]公司在当地也有近5000名员工和超过75年的[[发动机|发动机]]生产历史。2008年在保持传统产业优势的基础上，该市的产业结构正向[[金融业|金融业]]、[[保险|保险]]业、[[服务业|服务业]]和[[高科技|高科技]]产业转变。布里斯托是一批跨国公司的欧洲总部所在地。许多高科技公司如[[惠普|惠普]]、[[意法半導體|意法半導體]]都在当地投资设厂。[[电信|电信]]业尤为发达，[[多媒体|多媒体]]技术的研发与应用在世界居领先地位。布里斯托的[[旅游业|旅游业]]也十分强大，每年该市仅旅游收入就达3亿英镑，並以每年八月的[[布里斯托國際熱氣球節|布里斯托國際熱氣球節]]在全球享譽盛名。{{link-en|布里斯托鎊|Bristol Pound}}是布里斯托特有的{{link-en|社區流通貨幣|Community currency}}（Community currency）。

== 汽车 ==
{{main|布里斯托汽車}}
[[英国航空公司|英国航空公司]]在本市加入製作汽車，在1940年代成立[[布里斯托汽車|布里斯托汽車]]（Bristol Cars），在設於[[佛利頓|佛利頓]]的廠房手造豪華房車。1960年布里斯托汽車脱离航空公司独立。這也使得布里斯托爾成為重要的汽車輸出港。

== 教育 ==
布里斯托有2个重要的高等教育机构：成立於1909年，[[紅磚大學|紅磚大學]]之一的[[布里斯托尔大学|布里斯托尔大学]]，被誉为英国的麻省理工（MIT），以及[[西英格兰大学|西英格兰大学]]（前稱布里斯托理工學院，在1992年成為大學）。布里斯托也設有2間[[進修|進修]]（Further Education）學府，[[布里斯托市書院|布里斯托市書院]]（City of Bristol College）及[[佛利頓書院|佛利頓書院]]（Filton College），還有一間[[神學院|神學院]]，叫做[[三一学院_(布里斯托)|聖三一學院]]。该市有129家小學和3个市学习中心。该市还有许多独立的高质量学校，包括[[伊丽莎白女王医院_(布里斯托)|伊丽莎白女王医院]]（實質上是一間男校）等。

== 交通 ==

布里斯托有2个主要[[火车站|火车站]]：[[布里斯托客運站|布里斯托客運站]]和[[法學院草原站|法學院草原站]]。在布里斯托設置[[輕軌運輸|輕鐵]]系統的計畫並不成功，因此市內[[公共交通|公共交通]]仍然依靠[[巴士|巴士]]。市內的巴士主要由[[第一巴士集團|第一巴士集團]]營運。市中心也設有[[水上交通|水上交通]]（[[布里斯托渡輪|布里斯托渡輪]]），設有通勤和康樂用路線。

布里斯托位於[[倫敦|倫敦]]和[[威爾斯|威爾斯]]之間的中軸線上，有[[M4公路|M4公路]]連接，並有[[伯明翰|伯明翰]]至[[埃克塞特|埃克塞特]]的[[M5公路|M5公路]]途經，也有[[32公路|32公路]]連接[[市中心|市中心]]M4公路間。 位於[[路斯基_(布里斯托)|路斯基]]的[[布里斯托機場|布里斯托機場]]是本市主要的[[機場|機場]]。

== 地理 ==

布里斯托西岸臨[[愛爾蘭海|愛爾蘭海]]，北、東、南三方被[[單一管理區|單一管理區]]包圍：東南與[[薩默塞特郡|薩默塞特郡]]的[[巴斯-東北薩默塞特|巴斯-東北薩默塞特]]相鄰，西南與[[薩默塞特郡|薩默塞特郡]]的[[北薩默塞特|北薩默塞特]]相鄰，東北與[[格洛斯特郡|格洛斯特郡]]的[[南格洛斯特|南格洛斯特]]相鄰。
