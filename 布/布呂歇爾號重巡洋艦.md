'''布呂歇爾號重巡洋艦'''（[[德語|德語]]：'''Blücher'''）是[[納粹德國海軍|納粹德國海軍]]的一艘[[希佩尔将军级重巡洋舰|希佩尔将军级重巡洋舰]]，也是二戰爆發初期德國最新的一艘船。在布呂歇爾號服役後僅6個月，它就在入侵挪威的[[威瑟演習作戰|威瑟演習作戰]]第一天，於[[德勒巴克海峡战役|德勒巴克海峡战役]]中被其岸防要塞擊沉。

== 設計 ==
[[希佩尔将军级重巡洋舰|希佩尔将军级重巡洋舰]]主要是用來對抗世界主要海軍強權、超過他國簽訂的[[华盛顿海军条约|華盛頓條約]]艦隻以及突破[[凡爾賽條約|凡爾賽條約]]的限制，以擴充德國海軍的實力。

== 服役歷史 ==
{{main|德勒巴克海峡战役}}
布呂歇爾號以第5海軍艦隊的[[旗艦|旗艦]]身份，與[[裝甲艦|裝甲艦]][[德國號裝甲艦|呂佐號]]（前德國號）、[[輕巡洋艦|輕巡洋艦]][[埃姆登号轻巡洋舰|埃姆登號]]、3艘小型魚雷艇和8艘小型佈雷艦一起由海軍少將[[奧斯卡·庫莫茲|奧斯卡·庫莫茲]]指揮，進行入侵挪威的[[威瑟演習作戰|威瑟演習作戰]]第一階段—運輸德國陸軍部隊佔領[[奧斯陸|奧斯陸]]。

布呂歇爾號的姊妹艦[[希佩尔将军级重巡洋舰|希佩爾將軍號]]同樣參與了入侵挪威的行動，負責讓部隊登陸[[特倫汗|特倫汗]]（挪威的第3大城市，也位於挪威西海岸線的中部），儘管在登陸過程中遭遇到英國皇家海軍[[螢火蟲號驅逐艦|螢火蟲號驅逐艦]]的攻擊與衝撞而受損<ref name="Petrow53">Richard Petrow, ''The Bitter Years'', 1974, William Morrow Co., p.53</ref>，登陸部隊仍成功抵達上岸，並在當日清晨於城內的[[克利斯汀史坦堡壘|克利斯汀史坦堡壘]]（Kristiansten Fortress）與大部分行政大樓升起[[德國國旗|納粹國旗]]，而大多數挪威居民都還尚未清醒。

布呂歇爾號的攻擊行動就沒如此順利了，德國艦隊被挪威岸防巡邏艦[[波爾三世號巡邏艦|波爾三世號]]所發現，並在1940年4月8日就通知了防務人員。載有1000名士兵的布呂歇爾號在前頭帶領德國艦隊駛入奧斯陸峽灣的[[德勒巴克海峡|德勒巴克海峡]]（Drøbak narrows），並接近到[[奧斯卡斯堡要塞|奧斯卡斯堡要塞]]（Oscarsborg Fortress）南方。德國艦隊司令命令其麾下船艦將前後主砲皆排成一直線，表示出對挪威岸防工事的輕蔑<ref>Petrow 1974, p. 65</ref>。

[[File:28_cm_gun_at_Oscarsborg_Fortress.jpg|thumb]]
在清晨4點21分（挪威時間），挪威的要塞對布呂歇爾號開火。要塞上的3門德製[[克虜伯|克虜伯]]280毫米砲（由於缺乏訓練有素的砲手，僅有2門火砲有人在操作）裝設於1893年（分別取名為Moses、Aaron和Josva），已經過時，因此要塞砲手直到平射距離才開火。多數消息來源表示，要塞砲於1600至1800公尺內開火，第一枚280毫米砲彈擊中布呂歇爾號的船尾桅杆前部<ref name="Ribsskog 50">Ribsskog 1998: 50</ref>，並造成船體中央至桅杆燃起大火與濃煙<ref>Ribsskog 1998: 50-51</ref>。第2門要塞砲接著也開火，擊中布呂歇爾號的203毫米砲塔底座，使其在進入奧斯陸峽灣時艦體已有部分燃燒著<ref name="Ribsskog 51">Ribsskog 1998: 51</ref>。然而挪威砲兵在那短暫的時間裡也僅能以2門砲攻擊該艦，他們當時只有30名訓練不足的砲兵，極為缺乏人手<ref name="Ribsskog 50"/>，而且也沒有時間裝填第3發砲彈。另外，第3座「約書亞」（Joshua）要塞砲已裝填彈藥，但卻無人看守開火。

布呂歇爾號試著反擊，但因為其艦上輕型火砲射角太高，而203毫米主砲則因為奧斯卡斯堡第2波的280毫米要塞砲砲擊所造成的傷害而無法開火。

[[File:Karte_Oscarsborg.png|thumb]]的地圖與奧斯卡斯堡要塞]]
儘管布呂歇爾號船面已大火一片，挪威的57毫米口徑輔助[[海岸砲|海岸砲]]於[[胡斯維克|胡斯維克]]（Husvik）上仍不停地朝該艦開火，大砲一直轟擊船舷，而57毫米砲則集中攻擊船艦的上部結構和防空武器<ref name="Fjeld 36">Fjeld 1999:36</ref>，但以輕型火砲壓制部份火力的布呂歇爾號仍慢慢地往要塞駛去。當布呂歇爾號已駛到胡斯維克火砲陣地前面並以防空砲開火攻擊時，該陣地不得不被挪威砲兵所放棄<ref name="Fjeld 36"/>。在東側岸防砲兵的一座科帕斯（Kopås）150毫米砲擊中了布呂歇爾號的轉向系統，使得其船員必須掌舵與操作螺旋槳以免擱淺。

德軍不知道在奧斯卡斯堡要塞砲附近有座魚雷發射陣地，其自1901年建造，有3座岸用雙聯式水下魚雷發射管，裝備的魚雷是[[奧匈帝國|奧匈帝國]]製造的[[白頭魚雷|白頭魚雷]]（製造於[[匈牙利王國|匈牙利王國]]的阜姆，為現在[[克羅埃西亞|克羅埃西亞]]的[[里耶卡|里耶卡]]），為該世紀初的老古董。這些魚雷已經測試發射過超過200次，但仍不確定其可靠程度皆拿來使用<ref name="Hauge 36">Hauge 1995: 36</ref>。挪威岸防兵成功發射了魚雷，直接擊中布呂歇爾號2枚，第1枚擊中前方的「Anton」砲塔，第2枚擊中[[機艙|機艙]]，使得該艦失去控制並在狹窄的峽灣中漂流。艦隊的其他艦隻見布呂歇爾號爆炸，錯誤地以為被水雷所傷，使其艦隊陣型轉而密集，導致奧斯陸之後未有如預期地在黎明前佔領。在其他入侵艦隊撤退前，[[德意志號裝甲艦|呂佐號裝甲艦]]被科帕斯砲擊中了3次，它的「Anton」和「Bruno」砲塔受損無法使用。受損的呂佐號全速撤退，退出了挪威岸防砲的射界範圍。

[[File:BildeB3.jpg|thumb]]
德軍試著讓布呂歇爾號擱淺至[[納索德登半島|納索德登半島]]（Nesodden），但失敗了。 

6點，受創且下沉中的布呂歇爾號拋錨於[[阿斯克霍姆|阿斯克霍姆]]（Askholmen），試著令船身較穩以救助更多士兵與船員。布呂歇爾號將它僅剩的魚雷全數射出以免船體爆炸傷擊乘員。6點23分，艦上的火勢蔓延到了105毫米砲塔的彈藥庫並且引爆彈藥。7點，見艦上火勢已無法控制，艦長下達棄船命令。到了7點22分，布呂歇爾號傾覆沉沒。全艦2,202名的船員與登陸部隊中有830名陣亡（其中至少有320名船員），不是淹死就是被船身浮油燃起的火焰燒死。倖存者紛紛往峽灣兩側的沿岸游去。布呂歇爾號的水手們被命令讓出他們的救生衣給船上部隊（所有水手都被預估會游泳），因而挽救了大量士兵的生命。艦長[[海因里希·沃尔达克|海因里希·沃尔达克]]（Heinrich Woldag）上校雖然成功從這場戰鬥中逃脫，但在8天後因為飛機失事而陣亡。

[[File:BlucherAnker.JPG|thumb]]所保有的布呂歇爾號的一具錨。]]
德軍登陸奧斯陸行動的延遲使得挪威皇室成員、內閣和政府要員逃脫，並帶走了挪威的黃金儲備，之後用於海外流亡政府繼續在戰時反抗。

== 現存的殘骸 ==
現在，布呂歇爾號仍沉沒在[[奧斯陸峽灣|奧斯陸峽灣]]，其主砲位於[[奧斯卡堡要塞|奧斯卡堡要塞]]以北3,250公尺處。船身約在90公尺深度並呈側傾狀態，其A、B、C和D砲塔之砲座皆已崩毀，可能105毫米高砲也如此。

後來布呂歇爾號的兩個[[錨|錨]]皆被[[打撈|打撈]]了起來，其中一個於[[阿克爾碼頭|阿克爾碼頭]]（Aker Brygge）所展示，另一個則在[[德勒巴克|德勒巴克]]。

當布呂歇爾號離開原港口[[希維諾烏伊希切|希維諾烏伊希切]]執行作戰時，它約載有2670立方公尺（約2500噸）的油料。其中已有部份在前往挪威的途中消耗掉了，而在它沉沒後仍不斷地在漏油，到了1991年已達到每天漏50公升的油，對當地環境造成污染。挪威政府決定從布呂歇爾號殘骸裡盡可能的打撈其油料。1994年10月，[[石水公司|石水公司]]（Rockwater AS）與石油營運的深海[[潜水|潛水員]]至[[北海_(大西洋)|北海]]於其船體的133處燃料庫，取出1000噸的油料，另外有47處油料庫無法進入，故仍保存著油料。被取出的油料在經過清洗後，又再被賣出<ref name="Binder 180"/>。

該油料開採作業使得布呂歇爾號上的兩架[[Ar_196水上侦察机|Ar 196水上侦察机]]的其中一架得以重見天日，該機於1994年11月9日被打撈上岸，並現存於[[斯塔万格|斯塔万格]]的[[航空史博物館|航空史博物館]]<ref>[http://www.flymuseum-sola.no/sider/hoved.html ''Flyhistorisk Museum Sola''] {{webarchive|url=https://web.archive.org/web/20081102045317/http://www.flymuseum-sola.no/sider/hoved.html |date=2008-11-02 }}</ref><ref name="Binder 180">Binder: 180</ref>

== 同名的其他船隻 ==
布呂歇爾號是第三艘以[[普魯士王國|普魯士]][[德國元帥列表|陸軍元帥]][[格布哈德·勒伯莱希特·冯·布吕歇尔|格布哈德·勒伯莱希特·冯·布吕歇尔]]命名的德國海軍艦艇（該人最有名的事蹟是與[[威灵顿公爵|威灵顿公爵]]於[[滑铁卢战役|滑铁卢战役]]擊敗[[拿破崙|拿破崙]]）。第一艘以其命名的船艦為[[北德意志造船廠|北德意志造船廠]]（Norddeutsche Schiffbau AG，後來變成[[克虜伯|克虜伯]]麾下企業的一支）所建造的[[基爾港|基爾港]]的護衛艦，並於1877年3月20日下水。該船在1907年鍋爐爆炸後從軍方退役，並於[[西班牙|西班牙]][[維戈|維戈]]擔任運煤船直到其生涯結束。1908年4月11日，裝甲巡洋艦[[布呂歇爾號裝甲巡洋艦|布呂歇爾號]]於基爾港的帝國海軍造船廠下水。該船於[[一戰|一戰]]1915年1月24日的[[多格灘海戰|多格灘海戰]]中，被英國海軍中將[[戴维·贝蒂|戴维·贝蒂]]（David Beatty）率領的優勢戰鬥巡洋艦兵力擊沉。因此，在兩次世界大戰中的兩艘「布呂歇爾號」皆在戰初的8個月內被敵人擊沉。

== 歷任艦長 ==
[[海因里希·沃尔达克|海因里希·沃尔达克]]（Heinrich Woldag）海軍上校，於1939年9月20日至1940年4月9日期間擔任布呂歇爾號艦長（沃塔戈於1940年4月17日在搭乘前往奧斯陸峽灣的飛機中，墜機陣亡<ref>Deutsche-kriegsschiffe.de: [http://www.deutsche-kriegsschiffe.de/Schiffe/schwere-kreuzer/bluecher.htm Schwerer Kreuzer ''Blücher''] {{de icon}}</ref>。
