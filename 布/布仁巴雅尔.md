'''布仁巴雅尔'''（{{mong|m=ᠪᠦᠷᠢᠨᠪᠠᠶᠠᠷ|k=Бүрэнбаяр|ZMH=Bürinbayar}}{{efn|“布仁”（{{MongolUnicode|ᠪᠦᠷᠢᠨ}}）意思是“完全的”（《蒙汉词典》，内蒙古大学蒙古学研究院蒙古语文研究所编，内蒙古大学出版社1999年出版，第518页），“巴雅尔”（{{MongolUnicode|ᠪᠠᠶᠠᠷ}}）意思是“喜悦”。（《蒙汉词典》，433页）。西里尔转写参见[http://dic.on-toli.com/modules/japan/ 在线日语-蒙古语词典]}}；{{bd|1960年|3月6日|2018年|9月19日|catIdx=B}}），[[蒙古族|蒙古族]][[布里亚特人|布里亞特部]][[巴爾虎|巴爾虎]]人，[[中华人民共和国|中国]][[歌手|歌手]]、[[作曲家|作曲家]]，[[中国国际广播电台|中国国际广播电台]][[蒙语|蒙语]]节目的主持人，[[国家一级演员|国家一级演员]]<ref name="一级">{{cite web |title=布仁巴雅尔与妻子乌日娜获国家一级演员称号 |url=http://ent.sina.com.cn/y/2009-04-16/18362476531.shtml |website=新浪娱乐 |accessdate=2018-09-20 |date=2009-04-16}}</ref>。因与妻子[[乌日娜|乌日娜]]、侄女[[英格玛_(内蒙古)|英格玛]]在[[2006年中国中央电视台春节联欢晚会|2006年中国中央电视台春节联欢晚会]]上演唱《[[吉祥三宝|吉祥三宝]]》得到关注。

== 生平 ==
1960年，布仁巴雅尔出生于[[内蒙古|内蒙古]][[呼伦贝尔盟|呼伦贝尔盟]]的[[新巴尔虎左旗|新巴尔虎左旗]]。“布仁巴雅尔”的蒙古语意为“全喜、吉祥”<ref>{{cite web|url=http://www.cctv.com/program/czzx/20051026/101613.shtml|title=《吉祥三宝》 2005年10月25日|publisher=cctv.com|author=责编：白秀颖|date=2005-10-26|accessdate=2011-05-11|archive-url=https://web.archive.org/web/20160304215031/http://www.cctv.com/program/czzx/20051026/101613.shtml|archive-date=2016-03-04|dead-url=no}}</ref>。13岁时，布仁外出离家上学。1978年，布仁巴雅尔由[[鄂温克族自治旗|鄂温克族自治旗]][[乌兰牧旗|乌兰牧旗]]推荐到[[呼和浩特市|呼和浩特市]][[内蒙古艺术学院|内蒙古艺术学校]]学习[[蒙古长调|蒙古长调]]歌曲和[[马头琴|马头琴]]演奏。1980年，到内蒙古艺术学校呼伦贝尔盟分校继续学习，并担任舞蹈班的语文科教学工作，在校期间与后来的妻子，[[鄂温克族|鄂温克族]]歌手乌日娜相识相恋。1983年，到鄂温克族自治旗乌兰牧骑担任独唱演员和马头琴演奏员。1984年，乌日娜考上位于北京的[[中央民族大学|中央民族学院]]，两人开始异地恋。1985年，布仁巴雅尔到呼伦贝尔盟电影发行放映公司蒙古语电影配音科工作，从事配音和剧本翻译。1989年2月，布仁巴雅尔与乌日娜结婚<ref>{{cite web |title=“吉祥三宝”背后的至爱亲情 |url=http://www.qingdaonews.com/gb/content/2006-05/09/content_6825973.htm |website=青岛新闻网 |date=2006-05-09}}</ref>。1990年，布仁到北京[[中国国际广播电台|中国国际广播电台]]蒙古语部工作，从事新闻时事节目和专题栏目的播音及文字翻译、采访等工作，并担任蒙语主持人。业余时间则一直坚持创作并演唱蒙语歌曲。1991年6月21日，他们女儿诺尔曼出生<ref>{{cite web |url=http://www.ynet.com/view.jsp?oid=7248646 |title=吉祥三宝做客北青网 |accessdate=2010-01-24 |deadurl=yes |archiveurl=https://web.archive.org/web/20070309210410/http://www.ynet.com/view.jsp?oid=7248646 |archivedate=2007-03-09 |website=北青网}}</ref><ref>{{cite web |title="The Sun, the Moon and the Stars" |url=http://en.chinabroadcast.cn/2242/2005-3-17/81@217473.htm |website=China Radio International |accessdate=2018-09-20 |archiveurl=https://web.archive.org/web/20050328102314/http://en.chinabroadcast.cn/2242/2005-3-17/81%40217473.htm |archivedate=2005-03-11 |date=2005-03-11}}</ref><ref name="成长">{{cite web |author1=成长 |title=“吉祥三宝”爸爸布仁巴雅尔去世 享年58岁 |url=http://bjwb.bjd.com.cn/html/2018-09/20/content_283408.htm |website=北京晚报 |accessdate=2018-09-20 |date=2018-09-20}}</ref><ref name="wutongzi">{{cite web |author1=温瞳soul |title=呼伦贝尔——布仁巴雅尔 |url=http://www.wutongzi.com/a/136304.html |website=梧桐子网}}</ref>。

1994年，布仁巴雅尔创作了歌曲《吉祥三宝》，并将其作为生日礼物送给了女儿<ref name="成长"/>。1998年，布仁巴雅尔出访[[法国|法国]]、[[德国|德国]]、[[瑞士|瑞士]]、[[荷兰|荷兰]]等国，进行独唱和马头琴独奏演出，2003年，应邀到俄罗斯[[布里亚特共和国|布里亚特共和国]][[乌兰乌德|乌兰乌德]]市担任“今夜群星灿烂”国际声乐大赛评委<ref name="wutongzi"/>。2005年，布仁巴雅尔推出个人第一张音乐专辑《天边》，《吉祥三宝》是其中的主打歌。2006年，布仁巴雅尔与妻子乌日娜、侄女英格玛登上央视春晚演唱《吉祥三宝》<ref name="成长"/>。
《吉祥三宝》走红后，布仁巴雅尔夫妇与[[凤凰卫视|凤凰卫视]]中文台台长[[王纪言|王纪言]]和呼伦贝尔市市委共同发起创建“五彩呼伦贝尔传说”儿童合唱团，并担任合唱团的艺术总监<ref name="一级"/>。布仁巴雅尔到[[鄂温克族|鄂温克]]民族聚居区采风，制作并于2008年发行了原生态唱片《历史的声音》。2010年，他创作的原生态舞台剧《敖鲁古雅》在北京首演，创作该剧过程中，布仁曾多次去呼伦贝尔敖鲁古雅民族乡拜访使鹿鄂温克部落的老人，从老人口中记录了大量濒临失传的民歌<ref name="成长"/>。

2016年起，布仁巴雅尔自费发起了“拍摄百位呼伦贝尔百岁老人”的项目。2017年，其摄影作品作为内蒙古自治区成立70周年献礼作品在北京展出。2018年，布仁巴雅尔《呼伦贝尔·万岁》人物摄影作品集由新世界出版社出版，9月15日，布仁巴雅尔出席了《呼伦贝尔·万岁》新书发布会暨同名主题影展开幕<ref name="成长"/>。

2018年9月19日下午3点15分，布仁巴雅尔因突发[[心肌梗塞|心肌梗塞]]在呼伦贝尔市[[海拉尔区|海拉尔区]]家中去世，享寿58岁{{efn|内蒙古[[呼伦贝尔学院|呼伦贝尔学院]]所发讣告称59岁<ref>{{cite web |author1=布仁巴雅尔同志治丧委员会办公室 |title=讣   告 |url=http://news.hlbrc.cn:81/bencandy.php?fid-82-id-11018-page-1.htm |website=呼伦贝尔学院 |date=2018-09-20}}</ref><ref>{{cite web |author1=张恩杰 |title=蒙古族歌唱家布仁巴雅尔遗体告别式明早举行 |url=http://www.fawan.com/2018/09/21/1434732t185.html?kuaixun |website=法制晚报 |date=2018-09-21}}</ref>}}<ref>{{Cite web|url=http://www.chinanews.com/yl/2018/09-20/8631952.shtml|title=“吉祥三宝”父亲布仁巴雅尔因心梗去世 享年58岁|website=中国新闻网|author1=张曦|date=2018-09-20|accessdate=2018-09-20|archive-url=https://web.archive.org/web/20180920112054/http://www.chinanews.com/yl/2018/09-20/8631952.shtml|archive-date=2018-09-20|dead-url=no}}</ref><ref>{{Cite news|url=http://szb.nmgcb.com.cn/html/2018-09/21/content_43130.htm|title=“天边”不再歌声悠扬|publisher=内蒙古晨报|author1=张昊文|date=2018-09-21|accessdate=2018-09-21}}</ref>。9月22日上午10时，布仁巴雅尔的遗体告别仪式在海拉尔区殡仪馆举行，现场一直循环播放着布仁巴雅尔生前的音乐作品<ref>{{cite web |author1=杨畅 |title=吉祥三宝“爸爸”布仁巴雅尔遗体告别仪式举行 女儿：希望所有人记住爸爸的美好 |url=http://www.bjnews.com.cn/ent/2018/09/22/507217.html |website=新京报 |date=2018-09-22}}</ref>，多名演员、歌手、主持人，以及众多粉丝自发到场送别<ref>{{cite web |title=布仁巴雅尔告别仪式举行 妻子乌日娜面容憔悴 |url=http://ent.163.com/18/0922/14/DSAK3G1O00038FO9.html |website=网易娱乐 |date=2018-09-22}}</ref>。

== 荣誉 ==
1987年，获第一届内蒙古自治区蒙语歌曲电视大奖赛二等奖<ref name="成长" />。

1988年，获第一届全国少数民族青年歌手大奖赛一等奖<ref>{{cite news |title=吉祥三宝父亲去世 感谢因有爱而感动的世界 |url=http://www.ln.chinanews.com/news/2018/0920/177531.html |accessdate=2018-09-20 |agency=中新社 |publisher=中新网 |date=2018-09-20}}</ref>。

1992年，参加全国“歌王歌后”大奖赛，获通俗组“挑战者奖”（三等奖）<ref name="wutongzi" />。

2006年，《吉祥三宝》在“我最喜爱的央视春晚节目”评选中获歌舞类二等奖<ref>{{cite web |title=还记得那个唱《吉祥三宝》的小女孩吗？她如今亭亭玉立，才貌双全! |url=http://www.dzwww.com/xinwen/zhongqiqu/zj/201603/t20160316_14002530.htm |website=大众网 |date=2016-03-16 |access-date=2018-09-21 |archive-url=https://web.archive.org/web/20161007075108/http://www.dzwww.com/xinwen/zhongqiqu/zj/201603/t20160316_14002530.htm |archive-date=2016-10-07 |dead-url=no }}</ref>。

2007年，《吉祥三宝》获得第十届精神文明建设“[[五个一工程|五个一工程]]”优秀作品奖<ref>{{cite news |title=第十届“五个一工程”获奖作品 |url=http://archive.wenming.cn/zt/2007-09/08/content_17337477.htm |publisher=中国文明网 |date=2007-09-08}}</ref>。
