'''布農語'''（[[原住民族語羅馬字|布農文]]：{{lang|bnn|'''Bunun'''}}）是[[南島語系|南島語系]]的一種，與[[噶瑪蘭語|噶瑪蘭語]]並列，用[[拉丁文字|拉丁文字]]書寫，為[[台灣|台灣]][[布農族|布農族]]的民族語言<ref>齊莉莎(Zeitoun, Elizabeth)，"布農語參考語法"，遠流，臺北，p.40,2000。</ref>。 目前台灣[[原住民|原住民]]布農語認證考試分為[[卓群布農語|卓群布農語]]、[[卡群布農語|卡群布農語]]、[[丹群布農語|丹群布農語]]、[[巒群布農語|巒群布農語]]、[[郡群布農語|郡群布農語]]等五群<ref>國立台灣師範大學進修推廣學院,"98年度原住民族語言能力認證考試"{{cite web |url=http://140.122.109.96/ipt/viewpage.php?page_id=11 |title=存档副本 |accessdate=2017-03-01 |deadurl=yes |archiveurl=https://web.archive.org/web/20091217013326/http://140.122.109.96/ipt/viewpage.php?page_id=11 |archivedate=2009-12-17 }},臺北市,2009.</ref>。

==方言群==
布農語在族群遷移之歷史上發展為6個[[方言|方言]]社群，分別為：
* [[郡群|郡群]]（Isbukun）
* [[巒群|巒群]]（Takbanuaz）
* [[丹群|丹群]]（Takivatan）
* [[卡群|卡群]]（Takibakha）
* [[卓群|卓群]]（Takitudu）
* [[蘭群|蘭群]]（Takopulan）

除在20世紀30年代蘭社群被毗鄰的[[鄒族|鄒族]]所同化外、其餘5個社群為現代布農語群的基本組成單元。布農語群分裂的大致過程為郡社群先獨立出來，再而巒社群分裂出丹社群及卡社群，後來卡社群分裂出卓社群。也因大約從17世紀末、18世紀初族群開始分裂而進行南北不同方向的遷移，以致到21世紀布農語形成5大方言語群的區別。

卓社群、卡社群兩者為台灣北部主要通行的布農語。丹社群、巒社群布農語通行于台灣中部地區。而郡社群的布農語則是布農族裡的主導語言，主要通行在台灣南部地區。古老的布農族曾採用類似[[象形文字|象形文字]]的[[畫曆|畫曆]]來記事，現行採用[[拉丁字母|拉丁字母]]來拼寫<ref>江慧真／調查採訪,"李健秋借助科技 打造原住民語典"{{cite web |url=http://news.chinatimes.com/focus/501011672/112012080500112.html |title=存档副本 |accessdate=2012-08-05 |deadurl=yes |archiveurl=https://web.archive.org/web/20120810033408/http://news.chinatimes.com/focus/501011672/112012080500112.html |archivedate=2012-08-10 }},[[中國時報|中國時報]],2012-08-05/00:59.</ref><ref>原民風味館,"布農族象形文字"{{cite web |url=http://taiwanpaf.org/web/?attachment_id=1046 |title=存档副本 |accessdate=2012-12-22 |deadurl=yes |archiveurl=https://web.archive.org/web/20160304232649/http://taiwanpaf.org/web/?attachment_id=1046 |archivedate=2016-03-04 }},臺北市政府原住民族事務委員會,2012年查閱.</ref>。

===族名表示格式===
{{see also|臺灣原住民族人名}}

一般布農族在自我介紹的簡歷上都會註明寫著《某某人個人名 + 氏族名(某社群)》之方言語群格式，例：《"卜袞 伊斯瑪哈單 伊斯立 端"(郡社群)》。

==語音系統==
{{main|布農語音系}}

布農語的音系之[[音位|音位]]大都使用適當的[[Unicode|Unicode]]符號來標示。在[[台灣南島語|台灣南島語]]的[[書寫系統|書寫系統]]的訂定上，元輔音之音系表原則上都是先「[[發音部位|發音部位]]」（橫列）、後「[[發音方法|發音方法]]」（縱列）、再考量「[[清濁音|清濁音]]」，來訂定其音系之音位架構。<ref name=shaolatin>行政院原住民族委員會,"原住民族語言書寫系統"[http://www.edu.tw/files/list/M0001/aboriginal.pdf],台語字第0940163297號,原民教字第09400355912號公告,中華民國94年12月15日.</ref>

==布農語語法==
{{main|布農語語法}}

在語法的分類上[[台灣南島語|台灣南島語]]並不同於一般的[[分析語|分析語]]或其它[[综合语|综合语]]裡的動詞、名詞、形容詞、介詞和副詞等之基本[[詞類|詞類]]分類。比如台灣南島語裡普遍沒有副詞，而副詞的概念一般以動詞方式呈現、可稱之為「副動詞」，類之於[[俄语|俄语]]裡的副動詞。<ref name=diunnbbunhua>張永利,"台灣南島語言語法：語言類型與理論的啟示(Kavalan)"[http://vietnam.nsc.gov.tw/public/Data/012714292271.pdf]{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }},語言學門熱門前瞻研究,2010年12月/12卷1期,pp.112-127.</ref> 布農語語法分類是將基礎的語法之詞類、詞綴、字詞結構及分類法，對比[[分析語|分析語]]等之詞類分類法加以條析判別。
