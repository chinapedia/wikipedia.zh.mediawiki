'''布拉索夫'''（[[羅馬尼亞語|羅馬尼亞語]]：'''{{lang|ro|Brașov}}'''；[[德語|德語]]：'''{{lang|de|Kronstadt}}，'''中文译为'''喀琅施塔特'''；[[匈牙利語|匈牙利語]]：'''{{lang|hu|Brassó}}'''；{{lang-la|'''Corona'''}}）為[[羅馬尼亞|羅馬尼亞]][[布拉索夫縣|布拉索夫縣]]的首府，位於[[特兰西瓦尼亞|特兰西瓦尼亞]]南部，距離羅馬尼亞首都[[布加勒斯特|布加勒斯特]]約166公里。而這座城市更被南[[喀爾巴阡山脈|喀爾巴阡山脈]]包圍著。而這座城市最著名的是{{link-en|金鹿國際流行音樂節|Golden Stag International Music Festival}}，吸引各國參加。

== 地理及氣候 ==
布拉索夫位處羅馬尼亞中部，在南[[喀爾巴阡山脈|喀爾巴阡山脈]]的[[奧穆峰|奧穆峰]]東北約20公里，接近[[雅洛米察河|雅洛米察河]]（{{lang|ro|Ialomiţa}}）。在5月至9月間，平均溫度約為23 °C / 75 °F，氣候溫和。

== 歷史 ==
布拉索夫的歷史最早可以追溯至[[新石器時代|新石器時代]]。19世紀後期，考古學家在布拉索夫一帶發掘出原始人類活動及生活的痕跡。此外，他們在此處也挖掘出[[達基亞|達基亞人]]的軍事堡壘和[[羅馬人|羅馬人]]的建築物，證明布拉索夫一帶在遠古已經有人居住。考古學家相信在[[青铜时代|青铜时代]]的後期，最早的羅馬尼亞人就是在南[[喀爾巴阡山脈|喀爾巴阡山脈]]一帶群居，並形成逐漸部落式的社會制度。而一些出土文物如硬幣、飾物、金屬工具、陶器及居所遺址可以鑑定這些人大約生活在公元2世紀至8世紀。另外可以確定的是，這些住民都信奉基督教（在[[東西教會大分裂|東西教會大分裂]]後，他們大多奉[[東正教|東正教]]）。此外在13世紀時，一些來到羅馬尼亞的[[萨克森人|開拓者]]到達此地後，幫助當地居民建立一些學校、教堂等建築，更把居民們分為不同職業，如商人、木匠等，令這片土地得以發展。

而德意志人的到來就為布拉索夫的發展起了很大作用。1141至1161年間，[[匈牙利君主列表|匈牙利國王]][[蓋薩二世|蓋薩二世]]（{{lang|ro|II. Géza}}）就請來了這些德意志人到來協助發展城鎮、建立礦井、耕種土地，以幫助當地的文明發展。這些開拓者大多來自[[萊茵蘭|萊茵河區]]、[[佛拉芒|佛拉芒]]、[[摩泽尔省|摩澤爾]]，亦有些來自[[图林根|图林根]]、[[巴伐利亞|巴伐利亞]]、[[瓦隆|瓦隆]]及[[法國|法國]]等地。這些人後來在當地混居，成為當地的居民之一。

1211年，匈牙利國王[[安德烈二世|安德烈二世]]下令一部份[[十字軍|十字軍]][[條頓騎士團|條頓騎士團]]的騎士到[[布尔岑兰|布尔岑兰]]（{{lang|de|Burzenland}}）加強邊界的防禦，以抵抗敵人入侵。雖然1225年條頓騎士團被教宗遣往[[普魯士|普魯士]]，但這些騎士們卻留了下來，並建立起布拉索夫城。這些德國人在當地進行商貿及手工業，令布拉索夫城成為連接[[歐洲|歐洲]]和[[奧斯曼帝國|奧斯曼帝國]]的一個重要商站。此外，這個城市的免稅政策令當地的薩克森人得以獲取暴利，並加強他們在當地的政治影響力。而這些富庶的商人，更是當地具獨殊風格建築的始作俑者。當地的不少建築至今仍保存下來，一部份更被列入[[联合国教育科学文化组织|聯合國教科文組織]][[世界遗产|世界遗产]]之中，如當地最著名的「黑教堂」（{{lang|ro|Biserica Neagră}}），就是[[東歐|東歐]]最大的[[哥德式建築|哥德式建築]]。

布拉索夫後來成為德意志騎士們的封地。自此，新殖民者開始不承認當地居民的特權，更不再認同他們作為公民的身份。這使得他們不能再在當地經營手工業的生意。此外，他們的宗教－東正教也不再被承認，尤其是在15世紀時最為明顯。後來，新殖民者們卻為當地居民建立起第一所石磚教堂，以及在特蘭西瓦尼亞發行第一份活字印刷品，不久更建起第一所圖書館。為當地文教發展提供很大的幫助。此外，[[俄羅斯|俄羅斯]][[俄國君主列表|女沙皇]][[伊丽莎白·彼得罗芙娜|伊麗莎白]]、[[摩爾多瓦|摩爾多瓦]]總督及[[瓦拉奇公国|瓦拉奇]]統治者對此城的捐贈更令當地教育和宗教得到大力發展。18世紀時，[[神聖羅馬帝國皇帝列表|神聖羅馬皇帝]][[约瑟夫二世_(神圣罗马帝国)|約瑟夫二世]]更向居民開放了公民權利，形成更民主的政治。

1918年，[[第一次世界大戰|第一次世界大戰]]結束，[[羅馬尼亞|羅馬尼亞]]獨立，特蘭西瓦尼亞成為羅馬尼亞的一部份，直至[[第二次世界大戰|第二次世界大戰]]前，布拉索夫城的經濟穩步發展。二戰時，羅馬尼亞成為[[軸心國|軸心國]]的成員，與[[同盟國_(第二次世界大戰)|同盟國]]對抗。後來，二戰結束，盟國勝利，布拉索夫城的德國人被放逐至[[蘇聯|蘇聯]]，一部份在羅馬尼亞變成[[羅馬尼亞社會主義共和國|共產政權]]後逃到[[西德|西德]]。1987年，布拉索夫城內出現了反對政府的罷工潮，間接促成羅馬尼亞在1989年爆發了[[1989年羅馬尼亞革命|革命]]，結束統治者[[尼古拉·齊奧塞斯庫|尼古拉·齊奧塞斯庫]]的政權。

== 人口 ==
根據2002年調查所得，布拉索夫的總人口大約有284,734人。當地的種族如下（2002年為準）：

* [[羅馬尼亞|羅馬尼亞人]]：258,042人（90.62%）
* [[匈牙利|匈牙利人]]：23,204人（8.15%）
* [[德國人|德國人]]：1,717人（0.6%）
* [[罗姆人|罗姆人]]：762人（0.26%）
* [[猶太人|猶太人]]：138人（0.07%）
* 其他：871人（0.3%）

布拉索夫的都市區在2005年建成，預計會有350,000至400,000名居民。

== 經濟 ==
在1918至1922年間的內戰後，布拉索夫開始發展[[工業|工業]]，後來更建立起一間全羅馬尼亞最大規模的飛機製造公司''{{lang|ro|Industria Aeronautică Română}}''。在[[第二次世界大戰|第二次世界大戰]]間，布拉索夫就為羅馬尼亞政府生產了不同種類的[[戰鬥機|戰鬥機]]以應付[[蘇聯|蘇聯]]的攻勢。戰後羅馬尼亞[[羅馬尼亞社會主義共和國|共產政府]]成立，這個飛機製造場被強制改成[[農業|農業]]設備生產場，並改名''{{lang|ro|Uzina Tractorul Brașov}}''。

然而，布拉索夫的工業化進程在共產政府統治的年代開始加速，這吸引了很多來自外地的勞工。而政府更大力發展當地的重工業，如[[德國|德國]][[MAN股份公司|MAN公司]]的貨車生產線就設在布拉索夫。雖然在近年來，布拉索夫的工業發展有所減退，但它仍生產農業用的大型[[拖拉機|拖拉機]]及機械、液壓變速器、汽車零件、滾珠軸承、[[直升機|直升機]]、建築物料、[[工具|工具]]、[[傢具|傢具]]、[[紡織品|紡織品]]、[[鞋|鞋]]及[[化妝品|化妝品]]。此外，當地的食品加工業有一定規模，如[[巧克力|巧克力]]製造廠、[[啤酒|啤酒]]廠等。

== 交通 ==
布拉索夫城內的交通已完善的發展，城市具備多條[[公共汽车|公共汽车]]及[[無軌電車|無軌電車]]路線。而一條全國性的鐵路更把布拉索夫和[[錫比烏|錫比烏]]、[[阿爾巴尤利亞|阿爾巴尤利亞]]、[[勒姆尼庫沃爾恰|勒姆尼庫沃爾恰]]、[[梅迪亞什|梅迪亞什]]等城市連結起來。

== 旅遊 ==
當地有很多歷史建築，當中不乏成為[[联合国教育科学文化组织|聯合國教科文組織]][[世界遗产|世界遗产]]之作。如當地著名的「[[黑教堂_(布拉索夫)|黑教堂]]」（{{lang|ro|Biserica Neagră}}），就吸引了很多外國遊客到來參觀。此外，布拉索夫保留著一個傳說是「吸血鬼[[德古拉|德古拉]]」住過的城堡，令很多遊客紛紛慕名而來，一探這個傳說的究竟。此外，由於布拉索夫座落於南[[喀爾巴阡山脈|喀爾巴阡山脈]]的山腳，所以當地的山景也成為遊客的觀光熱點。

而這城市自1968年起成為「金鹿國際流行音樂節」的主辦城市，迄今已舉辦十六屆。而這個活動每年也吸引很多外國人參與。
