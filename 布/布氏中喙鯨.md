'''布氏中喙鯨'''或'''柏氏中喙鯨'''（學名''Mesoplodon densirostris''），旧称'''布兰氏喙鲸'''，也名'''钝喙鲸'''，'''大西洋喙鲸'''，'''热带喙鲸'''和'''瘤齿喙鲸'''<ref name='id581'>{{cite web| url =http://www.bioinfo.cn/db05/BjdwSpecies.php?action=view&id=581| title =瘤齿喙鲸| work =《中国动物物种编目数据库》| publisher =中国科学院微生物研究所| author =中国科学院动物研究所| accessdate =2009-03-27| deadurl =yes| archiveurl =https://web.archive.org/web/20160305075354/http://www.bioinfo.cn/db05/BjdwSpecies.php?action=view&id=581| archivedate =2016-03-05}}</ref>，似乎是中喙鯨屬中分布最廣泛的[[物種|物種]]，也是本屬唯一已有確實海中觀測研究資料者。已知經常傳出目擊記錄的地點至少有三處：[[夏威夷|夏威夷]][[歐胡島|歐胡島]]外海、南[[太平洋|太平洋]][[社會群島|社會群島]]外海、與[[巴哈馬|巴哈馬]]東北部。其種名源自[[拉丁文|拉丁文]]的'''''densus'''''（意為「'''厚'''」或「'''密'''」）與'''''rostrum'''''（意為「'''嘴喙'''」）。

== 外型特徵 ==

柏氏中喙鯨的體型寬厚而粗壯，背鰭小，呈[[三角形|三角形]]或鐮刀形，大約位於身長2/3處。嘴喙長度中等，[[額隆|額隆]]外觀上較小而扁平。嘴部曲線特殊，先是水平往後延伸，至中段急劇高起呈一圓弧狀。成年雄鯨在下顎隆起處有2顆大型[[牙齒|牙齒]]，末端略微突出，會往前傾斜越過上顎。牙齒上常附著有成串的[[鵝頸藤壺|鵝頸藤壺]]。

背部與身體側面皆為深藍灰色，至腹部轉為淺灰色。頭部可能呈淺褐色，上脣與下顎邊緣為淺灰色。牠們身上常有許多橢圓形傷痕，特別是在[[生殖器|生殖器]]附近，可能是遭[[雪茄鮫|雪茄鮫]]咬傷。成年雄鯨身上常有白色長條擦傷痕跡。

== 分布 ==

柏氏中喙鯨廣泛分布於各大洋[[熱帶|熱帶]]與暖[[溫帶|溫帶]]海域，偶爾會進入高[[緯度|緯度]]海域，原因可能與[[暖流|暖流]]有關，例如北[[大西洋|大西洋]]的[[墨西哥灣流|墨西哥灣流]]（Gulf Stream）與[[非洲|非洲]]南部的[[厄加勒斯洋流|厄加勒斯洋流]]（Agulhas Current）。已知的擱淺記錄有：
*北大西洋：[[加拿大|加拿大]]的[[諾瓦斯科西亞|諾瓦斯科西亞]]、[[冰島|冰島]]、[[英國|英國]]、[[美國|美國]]東岸、[[巴哈馬|巴哈馬]]、[[葡萄牙|葡萄牙]]、[[西班牙|西班牙]]、[[法國|法國]]。
*北[[太平洋|太平洋]]：[[夏威夷|夏威夷]]、[[日本|日本]]、[[加州|加州]]南部、[[台灣|台灣]]。
*南大西洋：[[巴西|巴西]]、[[南非|南非]]。
*南太平洋：[[智利|智利]]中部、[[澳洲|澳洲]]的[[塔斯馬尼亞州|塔斯馬尼亞州]]、[[紐西蘭|紐西蘭]]北部。
尚無證據顯示牠們有季節性的移動或遷徙。柏氏中喙鯨經常在500至1,000[[公尺|公尺]]深的大陸坡或深層[[海溝|海溝]]出現。

== 習性、生殖、食性 ==

柏氏中喙鯨多以3至7頭的小群體出現。通常會緩慢浮至海面，先是嘴喙突出水面，接著出現不明顯的噴氣，以15-20秒的間隔浮出水面數次後，會進行20-45分鐘的深潛。牠們主要以[[魷魚|魷魚]]和小型[[魚類|魚類]]為食。對其生殖狀況了解甚少。

== 現狀 ==

柏氏中喙鯨偶爾會被捕殺，但從未成為主要捕鯨對象。2000年3月在巴哈馬有數頭個體擱淺，確實原因不明，由於該海域不久前曾舉行過[[海軍|海軍]]演習，部分學者認為可能與聽覺系統受損有關。
