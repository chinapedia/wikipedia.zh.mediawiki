'''布干维尔岛'''（英语：Bougainville Island），[[巴布亚新几内亚|巴布亚新几内亚]]东端一[[岛屿|岛屿]]，地理上属于[[所罗门群岛|所罗门群岛]]的一部分，因此连同附近的[[布卡岛|布卡岛]]有时也被称为“北所罗门群岛”。布干维尔岛全岛面积为9318平方公里，为[[布干维尔自治区|布干维尔自治区]]的主体部分，全省人口400，000人（2014年）。

== 歷史 ==
{{main|布干維爾歷史}}
布干維爾島為1768年為[[法國|法國]]航海家[[路易斯·安東尼·布干維爾|布干維爾]]發現，並以其名命名。1885年，[[德国|德国]]占领了北所罗门群岛。1900年，[[英国|英国]]以其在[[萨摩亚|萨摩亚]]的权利，以换取除'''布干维尔岛'''之外的全部所罗门群岛，布干维尔岛遂成为[[德属新几内亚|德属新几内亚]]的一部分。[[一战|一战]]后，[[澳大利亚|澳大利亚]]受[[国联|国联]]委托统治新几内亚，并在[[二战|二战]]后将其与原澳属巴布亚合并。1976年，[[巴布亚新几内亚|巴布亚新几内亚]]宣布独立。

1970年代开始，[[布干维尔铜矿公司|布干维尔铜矿公司]]开始开采岛上蕴藏的大规模[[铜|铜]]矿，很快该铜矿便成为巴布亚新几内亚的经济支柱之一。由于巴新政府和铜矿公司在开采过程中并不十分注意对当地环境的保护，加之没有有效的利益回馈机制，当地岛民多有反抗情绪。1975年，当地在巴新独立前夕自行宣布成立[[北所罗门共和国|北所罗门共和国]]，不久取消。1990年，弗朗西斯.奥纳（{{lang|en|Francis Ona}}）率领的反抗组织再度宣布独立，巴新政府对该岛实施经济封锁，并派兵加以镇压，酿成了旷日持久的武装冲突。 從1989年開始到2001年巴政府與布各派達成全面和平協議前，布干维爾估計有一萬二千人死亡，占總人口的近7％。

1994年[[布干維爾革命軍|布干維爾革命軍]]、[[布干維爾臨時政府|布干維爾臨時政府]]、巴布亞紐幾內亞政府和[[所羅門群島|所羅門群島]]政府簽訂停火協定。1995年雙方於[[澳洲|澳洲]]進行和談。1996年巴布亞紐幾內亞政府再度攻擊布干維爾島。1997年在紐西蘭斡旋下，雙方正式宣布停戰。1998年，和平觀察團終抵布干維爾島。2001年，巴布亞紐幾內亞和布干維爾簽訂[[布干維爾和平協定|布干維爾和平協定]]，結束了長達十二年的戰爭，並允其在未來舉行獨立公投和成立自治政府。根據該條約，布干維爾島於2015年至2020年期間將舉行獨立公投，決定是否脫離巴布亞紐幾內亞正式獨立。但強硬派的[[法蘭西斯·歐那|法蘭西斯·歐那]]拒绝，他在其控制區自立為王，並對大選予以抵制。

[[2005年1月|2005年1月]]，中央政府批准《[[布干维爾憲法|布干维爾憲法]]》，同年5月20日，布自治政府大選开始，6月4日，布干維爾人民代表大會黨候選人[[約瑟夫·卡布伊|約瑟夫·卡布伊]]當選自治政府主席。行政長官為查馬利利和布自治区議会議長為[[潘尼埃|潘尼埃]]。同一時間，法蘭西斯·歐那宣佈聯合國接納了他所創建的王國，但事後證實為一場騙局。

同年七月，法蘭西斯·歐那死於傷寒，全島再度陷入動亂。[[2008年6月|2008年6月]]，自治政府主席約瑟夫·卡布伊去世，年底補選由[[詹姆斯·泰尼斯|詹姆斯·泰尼斯]]當選並於隔年一月就職。

目前，布干維爾是巴布亞紐幾內亞唯一的[[自治區|自治區]]。根据与中央政府达成的协议，自治区政府决定于2019年6月15日举行[[独立公投|独立公投]]。

== 地理 ==
{{main|布干維爾地理}}
[[File:BougainvilleTopography.png|thumb]]
布干維爾位於[[西南太平洋|西南太平洋]]上所羅門群島的最北端，也是該群島中最大島嶼，呈提琴狀。在政治區劃上，它屬巴布亞紐幾內亞的北所羅門省，省會在島東岸的[[阿拉瓦|阿拉瓦]]。全島呈西北至東南走向，長約120公里，寬約從64公里至96公里，總面積約一萬平方公里。人口連同附屬島[[布卡島|布卡島]]在內約十三萬人，主要人種屬[[美拉尼西亞人|美拉尼西亞人]]，白人亦約有6000人左右。所羅門群島位在[[環太平洋地震帶|環太平洋地震帶]]上，因此布干維爾島上多[[火山|火山]]、[[地震|地震]]，沿海多[[珊瑚礁|珊瑚礁]]。西部最大的海湾是[[奥古斯塔皇后湾|奥古斯塔皇后湾]]。

除南部有狹窄的沿海平原外，全島多山，最高峰為位於全島最北的[[巴爾比火山|巴爾比火山]]，[[海拔|海拔]]2745公尺。[[氣候|氣候]]上屬[[熱帶雨林|熱帶雨林]]型，常年高溫多雨。沿海平原上壤肥沃，內部則雨林密布。

由於布干維爾島位置較偏向所羅門群島，因此當其受劃分為巴布亞紐幾內亞領土時引起爭議，日後該島居民的武裝衝突自號[[北所羅門群島|北所羅門群島]]即可略之一二。

== 經濟 ==
{{main|布干維爾經濟}}
島上居民大多從事[[農業|農業]]，主要農產有[[甘薯|甘薯]]、[[香蕉|香蕉]]、[[椰子|椰子]]、[[可可|可可]]及[[咖啡|咖啡]]等。在中部的山區中，二十世紀的六十年代於東南部發現豐富的銅礦，蘊藏量估計約在九億噸。

近年來銅礦開採量迅速增長，已為首要出口商品；其他出口產品尚有椰乾及可可等。

== 人口 ==
{{main|布干維爾人口}}
連同屬島[[布卡島|布卡島]]在內，布干維爾島約人口12.9萬人。 主要為美拉尼西亞人，另有來自[[歐洲|歐洲]]的[[僑民|僑民]]約6000人左右。
