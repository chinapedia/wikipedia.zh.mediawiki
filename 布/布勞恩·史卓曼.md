'''亞當·謝爾'''（{{lang-en|Adam Scherr}}，{{bd|1983年|9月6日|||Braun Strowman}}）是出生於[[美國|美國]][[北卡羅萊納州|北卡羅萊納州]]的職業摔角手，他所較廣為人知的擂台名稱為「布勞恩·史卓曼」（Braun Strowman）。<ref>{{cite web |url=http://www.profightdb.com/wrestlers/braun-stowman-9702.html |title=Braun Strowman |work=Internet Wrestling Database |language=en |accessdate=2016年11月18日}}</ref>

== 經歷 ==
=== 力氣競賽 ===
在進入摔角聯盟[[WWE|WWE]]以前，亞當曾多次拿下各種力量競賽項目的優勝。其中包括了2011年的美國業餘全國錦標賽（US Amateur National Championships ）獲勝者、<ref>{{cite web |url=http://www.ironmind.com/news/Adam-Scherr-From-NAS-to-the-Pros/ |title=Adam Scherr: From NAS to the Pros |work=Ironmind |author=Randall J. Strossen |language=en |date=2011年11月7日 |accessdate=2016年11月18日}}</ref>2012年阿諾業餘強人錦標賽（Arnold Amateur Strongman Championships）冠軍、<ref>{{cite web |url=http://www.ironmind.com/news/Adam-Scherr-Wins-Arnold-ASC-Amateur-Strongman-World-Championships/ |title=Adam Scherr Wins Arnold ASC Amateur Strongman World Championships |work=Ironmind |author=Randall J. Strossen |language=en |date=2012年3月4日 |accessdate=2016年11月18日}}</ref>以及2013年阿諾強人經典賽第10名成績等等。<ref>{{cite web |url=http://www.strongman.org/news/arnold-strongman-2013-results/ |title=Arnold Strongman 2013 Results |work=Strongman.org |author=Strongman.org |language=en |date=2013年3月5日 |accessdate=2016年11月18日 |deadurl=yes |archiveurl=https://web.archive.org/web/20130907063530/http://www.strongman.org/news/arnold-strongman-2013-results/ |archivedate=2013年9月7日 }}</ref>

=== WWE ===
[[File:Braun_Strowman.jpg|thumb]]
==== 初期 ====
2013年5月12日亞當與摔角聯盟WWE完成簽約，<ref>{{cite web |url=http://www.dailywrestlingnews.com/wwe-signs-new-powerlifter-john-cena-injury-update-slaughter/ |title=WWE Signs New Powerlifter, John Cena Injury Update, Slaughter |work=Daily Wrestling News |author=Eric Arrington |language=en |date=2013年5月12日 |accessdate=2016年11月18日}}</ref>之後前往位於[[奧蘭多_(佛羅里達州)|奧蘭多]]一帶的中心接受摔角訓練，並開始採用「布勞恩·史卓曼」為擂台名稱。

此期間布勞恩也有在WWE節目上參與一些配角人物的演出，例如於2014年5月間擔任在摔角選手[[亞當·羅斯|亞當·羅斯]]入場時；站在一旁替對方歡呼加油的派對團體成員之一。<ref>{{cite web |url=http://www.wrestlezone.com/news/479465-several-nxt-stars-play-rosebuds-on-raw-jr-reacts-to-boxing-commentary-debut-kevin-steenwwe-update |title=Several NXT Stars Play Rosebuds on Raw, JR Reacts to Boxing Commentary Debut, Kevin Steen/WWE Update |work=Wrestle Zone |author=Nick Paglino |language=en |date=2014年5月27日 |accessdate=2016年11月18日}}</ref>

==== NXT時期 ====
2014年12月19日，布勞恩於WWE旗下的[[NXT_Wrestling|NXT]]聯盟賽事節目上首次以摔角手身份正式登場。在當時位於[[佛羅里達州|佛羅里達州]][[傑克遜維爾_(佛羅里達州)|傑克遜維爾]]舉辦的比賽中，布勞恩擊敗了對手[[查德·蓋博|查德·蓋博]]。<ref>{{cite web |url=http://www.wrestleview.com/wwe-news/52246-12-19-nxt-results-jacksonville-florida-tag-match/ |title=12/19 NXT Results: Jacksonville, Florida (Tag Match) |work=Wrestle View |author=Jason Namako |language=en |date=2014年12月22日 |accessdate=2016年11月18日}}</ref>

==== 外耶特家族時期 ====
之後在2015年8月24日，WWE讓布勞恩升格於主要節目《[[RAW|RAW]]》裡頭登場；並將他在節目上定位為摔角團隊「[[外耶特家族|外耶特家族]]」的新成員。在當天由外耶特家族的[[布雷·外耶特|布雷·外耶特]]與[[盧克·哈波爾|盧克·哈波爾]]；面對由[[迪恩·安布羅斯|迪恩·安布羅斯]]和[[羅曼·瑞恩斯|羅曼·瑞恩斯]]的二對二主賽中，當在布雷·外耶特一方處於不利時，戴著黑色羔羊面具的布勞恩則突然從場邊闖入；攻擊了迪恩·安布羅斯與羅曼·瑞恩斯倆人。<ref>{{cite web |url=http://wrestlingrumors.net/braun-stowman-debuts-on-monday-night-raw-joins-wyatt-family/39927/ |title=Braun Stowman Debuts on “Monday Night Raw,” Joins Wyatt Family |work=Wrestle Rumors |author=Derek Stoughton |language=en |date=2015年8月24日 |accessdate=2016年11月18日 |archive-url=https://web.archive.org/web/20160305025947/http://wrestlingrumors.net/braun-stowman-debuts-on-monday-night-raw-joins-wyatt-family/39927/# |archive-date=2016-03-05 |dead-url=yes }}</ref>

加入外耶特家族後，布勞恩初次在《RAW》上的個人賽事勝利，是在8月31日上與迪恩·安布羅斯的一對一比賽中；在對方使用犯規攻擊的情況下取勝。<ref>{{cite web |url=http://www.wwe.com/shows/raw/raw-march-21-2016/article/dean-ambrose-braun-strowman |title=Braun Strowman def. Dean Ambrose via Disqualification |work=[[WWE|WWE]] |author=Anthony Benigno |language=en |date=2016年3月21日 |accessdate=2016年11月18日}}</ref>而他所首次參與的[[世界摔角娛樂付費收看節目|PPV]]大型賽事，則是9月的《[[冠軍之夜|冠軍之夜]]》裡與外耶特家族成員；面對由迪恩·安布羅斯、羅曼·瑞恩斯、[[克里斯·傑利可|克里斯·傑利可]]所組成團隊的三對三比賽。<ref>{{cite web |url=http://411mania.com/wrestling/411s-wwe-night-of-champions-2015-report-09-20-15/ |title=411’s WWE Night of Champions 2015 Report 09.20.15 |work=411Mania |author=Scott Slimmer |language=en |date=2015年9月20日 |accessdate=2016年11月18日}}</ref>之後布勞恩參與的2016年第29屆《[[皇家大戰|皇家大戰]]》的壓軸賽多人混戰（Battle Royal）中，雖然布勞恩並非最後優勝人選；但過程中與羅曼·瑞恩斯並列為當天在擂台上淘汰最多人數的選手。<ref>{{cite web |url=http://www.skysports.com/wwe/news/14203/10142960/wwe-royal-rumble-2016-triple-h-wins-wwe--heavyweight-title |title=WWE Royal Rumble 2016: Triple H wins WWE World Heavyweight Title |work=Sky Sports |language=en |date=2016年1月25日 |accessdate=2016年11月18日}}</ref>

==== 單飛時期 ====
WWE在2016年夏季期間開始將旗下選手各分配到《RAW》、與《[[SmackDown|SmackDown]]》節目裡，布勞恩當時則被安排從外耶特家族拆夥；單飛至《RAW》當中。<ref>{{cite web |url=http://www.wwe.com/article/2016-wwe-draft-results |title=2016 WWE Draft results: WWE officially ushers in New Era |work=[[WWE|WWE]] |author=WWE.com Staff |language=en |date=2016年7月19日 |accessdate=2016年11月18日}}</ref>

在要以獨立選手身份在《RAW》演出後，WWE開始規劃一連串讓布勞恩對抗各地獨立圈選手的比賽；來表現出布勞恩的強度。之後包括7月25日的與[[詹姆斯·艾爾斯沃茲|詹姆斯·艾爾斯沃茲]]、<ref>{{cite web |url=http://heavy.com/sports/2016/09/james-ellsworth-wwe-jobber-wrestling-wrestler-match-braun-strowman-monday-night-raw-smackdown-the-miz-background/ |title=James Ellsworth: 5 Fast Facts You Need to Know |work=Heavy |author=Brendan Morrow |language=en |date=2016年9月15日 |accessdate=2016年11月18日}}</ref>或8月1日對上厄文·安德霍姆（Evan Anderholm）、<ref>{{cite web |url=http://allwrestling.com/newswireitem/78887/braun-strowman-def-evan-anderholm/ |title=Braun Strowman def. Evan Anderholm |work=All Wrestling |language=en |date=2016年8月2日 |accessdate=2016年11月18日}}</ref>到10月10日和浪花兄弟（The Splash Brothers）的一對二、<ref>{{cite web |url=http://www.pw-core.com/news/Braun_Strowman_def_The_Splash_Brothers/89608 |title=Braun Strowman def. The Splash Brothers |work=PW-Core |language=en |date=2016年10月10日 |accessdate=2016年11月18日}}</ref>以及甚至10月17日與高亢邁爾三重奏（The Mile-High Trio）的一對三賽事，<ref>{{cite web |url=http://www.wwe.com/shows/raw/2016-10-17/article/braun-stroman-def-the-mile-high-trio |title=Braun Stroman def. The Mile-High Trio |work=[[WWE|WWE]] |author=Anthony Benigno |language=en |date=2016年10月17日 |accessdate=2016年11月18日}}</ref>都是以壓倒性的過程取勝。

隨後WWE逐步安排讓布勞恩對於都是和弱小選手對賽一事；感到抱怨不滿，而與主線選手[[山米·贊恩|山米·贊恩]]對抗的新情節。之後在10月31日；一場優勝者能於11月第30屆《[[強者生存|強者生存]]》大賽上代表節目《RAW》一方出戰的多人混戰裡，布勞恩在最後淘汰掉山米·贊恩，獲得了在《強者生存》登場的資格。<ref>{{cite web |url=http://www.wrestleview.com/wwe-wrestling-results/wwe-raw-results/68935-wwe-raw-results-103116-live-hartford-goldberg-returns-hell-cell-fallout-survivor-series-teams/ |title=WWE RAW Results – 10/31/16 (Live from Hartford, Goldberg returns, Hell in a Cell fallout, Survivor Series teams) |work=Wrestle View |author=Mike Tedesco |language=en |date=2016年10月31日 |accessdate=2016年11月18日}}</ref>
