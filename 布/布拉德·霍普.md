'''布拉德利·邦特·霍普'''（'''Bradley Bonte Hawpe'''，{{bd|1979年|6月22日|||Hawpe, Brad}}）出生于[[美国|美国]][[德克萨斯州|德克萨斯州]][[沃思堡|沃思堡]]，是[[美国职棒大联盟|美国职棒大联盟]][[聖地牙哥教士|聖地牙哥教士]]的[[內野手|內野手]]。

== 高中及大学时期 ==
霍普曾就读於沃思堡的博斯维尔高中，当时他的学校棒球队中充当一垒手和投手。他还和球队一起获得过德州的冠军。

高中毕业後霍普进入[[路易斯安那州立大学|路易斯安那州立大学]]就读，他和路易斯安那州立大学老虎队一起赢得了{{by|2000年}}的大学世界系列赛。

== 职业生涯 ==
霍普在[[1998年美国职棒大联盟选秀|1998年大联盟选秀大会]]上被[[多伦多蓝鸟|多伦多蓝鸟]]在第47轮选中，但他并没有和球队签约。两年後他再度进入[[2000年美国职棒大联盟选秀|大联盟选秀]]，这次被[[科罗拉多落矶|科罗拉多落矶]]在第11轮选中，并和球队签约。

霍普在小联盟的四个赛季表现出色。{{by|2000年}}他入选了[[西北联盟|西北联盟]]的全明星赛，{{by|2002年}}获得[[卡罗莱纳联盟|卡罗莱纳联盟]]MVP，{{by|2003年}}获得[[德州联盟|德州联盟]]全明星赛MVP。

{{MLBy|2004年}}5月1日，霍普首次登上大联盟，那个赛季他在大联盟出场42次。

{{MLBy|2005年}}是霍普在大联盟的第一个完整赛季，在101场比赛中击出9支全垒打和47分打点，他的[[保送|保送]]和[[打点|打点]]分别排在国联新秀的第三和第四位。霍普在守备上也表现出他的实力，这个赛季一共有10次外野助杀，排名国联第五位，同时创造了落矶队的新人纪录，在队史上也排名第八位。

{{MLBy|2006年}}霍普几乎每项数据都创下生涯新高。这个赛季他出场150次，打击率2成92，并有22支全垒打和84分打点。他也成为队史上第四位击出超过20支全垒打的左打者。他的16次外野助杀排名大联盟第二，仅次於[[华盛顿国民|华盛顿国民]]的[[阿方索·索里亚诺|阿方索·索里亚诺]]，在队史上也排名第二，仅次於[[丹特·比切特|丹特·比切特]]（[[:en:Dante_Bichette|Dante Bichette]]）在{{MLBy|1999年}}的17次外野助杀。

{{MLBy|2007年}}是霍普迄今为止打得最好的一个赛季，29支全垒打、11分打点、3成87上垒率和5成39长打率均创下生涯新高。他在两出局後击出66分打点，创造队史新纪录。他随球队一起获得国联冠军，并打入[[2008年世界大赛|世界大赛]]，但以0比4负于[[波士顿红袜|波士顿红袜]]。

尽管季中由于[[腿后腱|右腿后腱]]拉伤而错过了一部分比赛，霍普在{{MLBy|2008年}}还是继续保持着不错的状态，击出25支全垒打和85分打点。但守备方面表现不够理想，守备率（.956）和守备范围因子（1.50）都是所有大联盟右外野手中最低的，失误数（9次）则是右外野手中最高的<ref>{{cite web | url = http://espn.go.com/mlb/stats/fielding/_/position/rf/order/true | title = MLB Player Fielding Stats - As rf - 2009 | publisher = [[ESPN|ESPN]]}}</ref>。

{{MLBy|2009年}}霍普首次入选[[2009年美国职棒大联盟全明星赛|全明星赛]]<ref>{{cite web | last = Renck | first = Troy E. | url = http://www.denverpost.com/sportsheadlines/ci_12759442?source=rss | title = Marquis, Hawpe earn All-Star bids for Rockies | publisher = [[丹佛邮报|丹佛邮报]] | date = July 6, 2009}}</ref>，他还和一同入选全明星赛的队友[[杰森·马奎斯|杰森·马奎斯]]一起买了30个球棒和50件全明星赛T恤送给队友<ref>{{cite web | url = http://www.denverpost.com/rockies/ci_12839939 | last = Renck | first = Troy E. | title = Renck: Hawpe gets ball from First Fan | publisher = [[丹佛邮报|丹佛邮报]] | date = 2009-07-15 | accessdate = 2009-07-18}}</ref>。他在比赛的七局下半险些击出全垒打，但被[[卡尔·克劳福|卡尔·克劳福]]接杀出局。

== 私人生活 ==
霍普和妻子金拥有一个女儿艾弗莉·格雷斯（Avery Grace），她在2004年10月12日出生。
