'''布鲁斯·斯特拉利'''（{{lang-en|Bruce Straley}}，{{bd|1978年}}）是美国电子总监、美术师与设计师，就职于电子游戏公司[[顽皮狗|顽皮狗]]，知名作品为《[[最后生还者|最后生还者]]》和《[[神秘海域4：盗贼末路|神秘海域4：盗贼末路]]》。斯特拉利刚进入电子游戏业时就职于西部技术公司，他在该公司以美术师身份参与了两部作品。此后他在多家公司担任设计师，期间他为[[水晶动力|水晶动力]]设计了《[[超级杰克龙|超级杰克龙]]》和《[[超级杰克龙3|超级杰克龙3]]》。

斯特拉利在1999年进入顽皮狗，并起先负责《古惑狼赛车》的[[杰克与达斯特系列|杰克与达斯特系列]]的美术工作。之后他担任了《[[秘境探險：黃金城秘寶|秘境探險：黃金城秘寶]]》的共同艺术总监，并被点为《[[秘境探險2：盜亦有道|秘境探險2：盜亦有道]]》游戏总监。此后他以游戏总监身份，领衔开发了《最后生还者》，接下来他又监督了《神秘海域4：盗贼末路》。斯特拉利的作品获得称赞，其中《最后生还者》得到高度评价，获得一系列奖项与提名。

== 职业生涯 ==

斯特拉利早先就职于西部技术，在艺术设计部门制作过两款游戏：1992年的Menacer光枪6合1游戏和1993年《X-Men》{{efn|name=Menacer}}<ref name="X-Men" />。接着他先后为Pacific Softscape和Zono Incorporated设计了《Generations Lost》（1994）与《Mr. Bones》（1996）<ref name="Generations" /><ref name="Bones" />。他后就职于[[水晶动力|水晶动力]]，并和后来的[[神秘海域系列|神秘海域系列]]创意总监[[艾米·汉尼恩|艾米·汉尼恩]]<ref name="Crystal" />、后[[顽皮狗|顽皮狗]]联合总裁伊万·威尔斯<ref name="New Tricks" />，及多名后顽皮狗雇员有过合作<ref name="Crystal">{{cite web |url=http://www.gameinformer.com/b/features/archive/2013/02/25/something-in-the-water-crystal-dynamics-in-the-90s.aspx?PostPageIndex=7 |title=Something In The Water: Crystal Dynamics In The '90s |page=7 |last=Helgeson |first=Matt |work=[[Game_Informer|Game Informer]] |publisher=[[GameStop|GameStop]] |date=February 25, 2013 |accessdate=July 6, 2015 |archiveurl=https://www.webcitation.org/6ZoO9ziLv?url=http://www.gameinformer.com/b/features/archive/2013/02/25/something-in-the-water-crystal-dynamics-in-the-90s.aspx?PostPageIndex=7 |archivedate=2015-07-06 |deadurl=no }}</ref>。斯特拉利在水晶动力期间设计了《[[超级杰克龙|超级杰克龙]]》（1998）<ref name="Gex" />，并为《[[超级杰克龙3|超级杰克龙3]]》（1999）绘制了附加原画<ref name="Gex 3" />。斯特拉利在1999年3月《超级杰克龙3》发行后进入了顽皮狗<ref>{{cite web |url=http://www.naughtydog.com/crash/crash/timeline.htm |title=Crash Bandicoot – Time Line |publisher=[[顽皮狗|Naughty Dog]]. [[索尼電腦娛樂|Sony Computer Entertainment]] |accessdate=July 6, 2015 |archiveurl=https://web.archive.org/web/20080729045219/http://www.naughtydog.com/crash/crash/timeline.htm |archivedate=2008-07-29 |dead-url=no }}</ref>。

斯特拉利起初在顽皮狗担任美术工作，参与了1999年游戏《[[古惑狼赛车|古惑狼赛车]]》的制作<ref name="CTR">{{cite web |url=http://naughtydog.com/corporate/kennel/bSTRALEY/bruceSTRALEY.htm |title=Bruce Straley |publisher=[[顽皮狗|Naughty Dog]]. [[索尼電腦娛樂|Sony Computer Entertainment]] |accessdate=July 6, 2015 |archiveurl=https://web.archive.org/web/20040520231757/http://naughtydog.com/corporate/kennel/bSTRALEY/bruceSTRALEY.htm |archivedate=2004-05-20 |dead-url=yes }}</ref>。斯特拉利虽然以纹理美术师身份受雇，但限于团队规模，设计、背景建模、前景动画等各项工作由他们共同完成。在工作室规模逐渐扩大后，任务也开始专门化<ref name="House">{{cite web |url=http://www.gameinformer.com/b/features/archive/2012/02/24/expanding-the-house-naughty-dog-39-s-second-team.aspx |title=Expanding The House: Naughty Dog's Second Team |last=Turi |first=Tim |work=[[Game_Informer|Game Informer]] |publisher=[[GameStop|GameStop]] |date=February 24, 2012 |accessdate=July 6, 2015 |archiveurl=https://www.webcitation.org/6VOg9Z8eX?url=http://www.gameinformer.com/b/features/archive/2012/02/24/expanding-the-house-naughty-dog-39-s-second-team.aspx |archivedate=2015-01-07 |deadurl=no }}</ref>。斯特拉利在《[[杰克与达斯特：先驱者的遗产|杰克与达斯特：先驱者的遗产]]》（2001）<ref name="Jak" />、《[[杰克II|杰克II]]》（2003）<ref name="Jak2" />和《[[杰克3|杰克3]]》（2004）中负责美术工作<ref name="Jak2" /><ref name="Jak3" />。斯特拉利还创造了[[杰克与达斯特系列|杰克与达斯特系列]]画面处理技术，并以技术与艺术特色知识、为两部门架起沟通桥梁<ref name="New Tricks" />。斯特拉利在《[[神秘海域：德雷克的宝藏|神秘海域：德雷克的宝藏]]》（2007）中，和参与团队[[PlayStation_2|PlayStation 2]]到[[PlayStation_3|PlayStation 3]]技术改进的[[鲍勃·拉菲|鲍勃·拉菲]]，共同担任艺术总监。斯特拉利在2011年发行的《[[神秘海域2：纵横四海|神秘海域2：纵横四海]]》中担任游戏总监<ref name="House" />。

继《神秘海域2》开发后，顽皮狗将团队分成两支，同时开发项目。一支团队制作《[[神秘海域3：德雷克的诡计|神秘海域3：德雷克的诡计]]》，斯特拉利和[[尼尔·德鲁克曼|尼尔·德鲁克曼]]则应伊万·威尔斯和克里斯托夫·巴莱斯特拉要求，带领另一支团队开发新游戏；斯特拉利能获任担当总监领导项目，应归于他的经验与既往项目成果<ref name="New Tricks">{{cite web |url=http://www.gameinformer.com/b/features/archive/2012/02/22/the-past-present-and-future-of-naughty-dog-an-interview-with-the-co-presidents.aspx |title=New Tricks: An Interview With Naughty Dog's Co-Presidents |last=Hansen |first=Ben |work=[[Game_Informer|Game Informer]] |publisher=[[GameStop|GameStop]] |date=February 22, 2012 |accessdate=July 6, 2015 |archiveurl=https://www.webcitation.org/6VOeIaJhr?url=http://www.gameinformer.com/b/features/archive/2012/02/22/the-past-present-and-future-of-naughty-dog-an-interview-with-the-co-presidents.aspx |archivedate=2015-01-07 |deadurl=no }}</ref>。他们原打算开发杰克与达斯特新作，但团队认为他们“现时不为系列爱好者服务”，而决定制作一款[[知识产权|新游戏]]，题为《[[最后生还者|最后生还者]]》<ref>{{cite web |url=http://www.gamespot.com/articles/naughty-dog-reveals-how-axed-jack-and-daxter-reboot-led-to-the-last-of-us/1100-6415331/ |title=Naughty Dog reveals how axed Jack and Daxter reboot led to The Last of Us |last=Gaston |first=Martin |publisher=[[GameSpot|GameSpot]].
