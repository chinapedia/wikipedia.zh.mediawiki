'''布施 辰治'''（{{bd|1880年|（[[明治|明治]]13年）11月13日{{refnest|group="注"|戶籍中是14日生，但布施本人說自己是13日生<ref>『布施辰治研究』 299頁</ref>。}}|1953年|（[[昭和|昭和]]28年）9月13日|catIdx=B}}）是[[日本|日本]][[宮城縣|宮城縣]]出身的[[律師|律師]]及社会運動家。以唯一一位獲得大韓民国建国勲章的日本人而得名。

== 前歷 ==
出生宮城縣[[牡鹿郡|牡鹿郡]][[蛇田村|蛇田村]]（現在是[[石卷市|石卷市]]一部分）的農家。少年時接觸[[墨子|墨子]]的[[兼愛|兼愛]]思想<ref>『弁護士布施辰治』 16 頁</ref>，長大後接受[[日本正教会|正教会]]的洗禮 <ref>『弁護士布施辰治』 18頁</ref>並就讀神田[[復活大教堂_(東京)|復活大教堂 (東京)]]附屬神学院，雖然從年輕開始就親近[[和平主義|和平主義]]，但就讀神学院3個月後就遭退學，以後因為對於基督教派派系的[[托爾斯泰運動|托爾斯泰運動]]產生共鳴而以此為自身之典範並一生持續參與社會運動，宗教上則因為結婚而改信為妻子信仰的[[日蓮正宗|日蓮正宗]] <ref>『弁護士布施辰治』 38頁</ref><ref>『大日蓮』764号 50-58頁</ref>。

最初到[[東京專門学校|東京專門学校]]（現在的[[早稻田大学|早稻田大学]]）就讀<ref>『布施辰治（明治篇）』 34頁</ref>，之後轉學到[[明治法律学校|明治法律学校]]（現在的 [[明治大学|明治大学]]）就讀， 1902年畢業，22歳時通過判事檢事登用考試，成為候補司法官。之後在[[宇都宮地方裁判所|宇都宮地方裁判所]]擔任代理[[檢察官|檢察官]]，但由於不忍[[起訴|起訴]]某個[[殉情|殉情]]未遂事件的犯人，因此就任不到一年就辭去候補司法官的工作。

== 作為律師==
辭去檢察官工作後而改擔任律師，但工作上一直無法步上軌道而只好一直更換事務所。不過在1917年的[[死刑|死刑]]事件中，成功讓[[鈴ヶ森おはる殺し事件|鈴ヶ森おはる殺し事件]]的被告獲得無罪判決，並因此以刑事律師的身分而成名，1920年時，負責處理250個案件，1天平均要出庭4次 <ref>『弁護士布施辰治』 90頁</ref>。

以後，除了刑事事件外，也積極參與許多爭議事件、廢娼運動，[[普選運動|普選運動]]等社会問題，但使他名聲大噪的就是在 [[韓國獨立運動|韓國獨立運動]]當中，擔任辯護律師。戰前因為替[[二重橋炸彈事件|二重橋炸彈事件]]，[[朴烈事件|朴烈事件]]，[[朝鮮共產黨|朝鮮共產黨]]等事件辯護以及[[關東大地震|關東大地震]]時發生虐殺韓國人的事件而對當局進行嚴厲的批判，戰後也在[[評定河原事件|評定河原事件]]，[[阪神教育事件|阪神教育事件]]，[[平事件|平事件]]，[[台東会館事件|台東会館事件]]等多數與韓國人有關的事件中進行辯護，因此讓布施在[[韓国|韓国]]受到很高的評價。2000年2月29日時，在[[韓国文化放送|韓国文化放送]]的1小時節目「[[PD手帳|PD手帳]]」中，就以「發掘　日本的[[辛德勒|辛德勒]]布施辰治」為題播放。然後2004年時，韓国政府授與大韓民国建国勲章中的愛族章給他，讓布施成為日本人當中唯一獲得大韓民国建国勳章的受勳者。
在有關普選運動中，布施屬於[[政治研究会|政治研究会]]及[[労働農民黨|労働農民黨]]等左派政黨，1928年[[第16屆日本眾議院議員總選舉|第1回普通選舉]]中，以労働農民黨候選人身分出馬競選而落選<ref>『ある弁護士の生涯』 73-75頁</ref>。之後以律師的身分為作為同黨，卻受[[三・一五事件|三・一五事件]]影響而遭命解散，並在當時成為非法政黨的 [[日本共産黨|日本共産黨]]出庭進行辯護，但在1929年，卻被以「不當辯護」為由，遭東京[[控訴院|控訴院]]向[[懲戒裁判所|懲戒裁判所]]起訴，1932年遭[[大審院|大審院]]（現在日本的[[最高裁判所|最高裁判所]]）判決剝奪其律師資格。隔年所屬的 [[日本労農弁護士團|日本労農弁護士團]]也遭人檢舉，造成布施成為共同被告中唯一因為違反 [[治安維持法|治安維持法]]而受刑的人，並因此在[[千葉刑務所|千葉刑務所]]服刑一年多。

戰後，布施在自己所創的 [[自由法曹團|自由法曹團]]復活時，擔任顧問。1949年[[三鷹事件|三鷹事件]]發生，在主嫌竹内景助維持自己的[[自白|自白]]時，他就堅持自白是遭人偽造，竹內是無罪的主張，並因此與其他律師產生激烈的衝突而退出。之後主張竹內是無罪而正準備要替他辯護的不久，卻於 1953年9月13日因為[[癌症|癌症]]病逝。布施葬於[[東京都|東京都]][[豐島區|豐島區]]的[[常在寺|常在寺]]，常在寺中，建立一座刻有他座右銘「生則與民眾同在，死則為民眾而死」的彰顯碑。

== 思想 ==
作為刑事律師，從處以被告無法接受的刑罰是沒有道德上的效果 <ref>『ある弁護士の生涯』 37頁</ref>，死刑是國家為了維持秩序而以 [[恐怖份子|恐怖份子]]自居<ref>『布施辰治研究』 70頁</ref>等觀點而主張[[廢除死刑|廢除死刑]]。另外受[[吉野作造|吉野作造]]的[[民本主義|民本主義]]影響，因此也支持可以代表一般民衆意志的[[陪審制|陪審制]] <ref>[http://kindai.ndl.go.jp/info:ndljp/pid/1108929 『法廷実話　審く者，審かれる者』] 2頁</ref>。

身為和平主義者而相當厭惡軍人，但對於樸實嚴謹的 [[乃木希典|乃木希典]]則是以教育家的身分尊敬他 <ref>『ある弁護士の生涯』 43頁</ref>。

雖然主張要廢除[[天皇制|天皇制]]，但認為作為廣受人民支持的制度並不需要人為的解體<ref>[http://kindai.ndl.go.jp/info:ndljp/pid/1281948 『打倒？支持？天皇制の批判，憲法改正（私案）』]　11頁</ref>，只要等到[[日本皇室世系圖|皇室血脈]]斷絕就會自然消滅<ref>『弁護士布施辰治』 217頁</ref>。另外對於[[昭和天皇|昭和天皇]]接受[[波茨坦公告|波茨坦公告]]以避免 [[本土決戰|本土決戰]]的發生，而對他有相當高的評價 <ref>[http://kindai.ndl.go.jp/info:ndljp/pid/1281948 『打倒？支持？天皇制の批判，憲法改正（私案）』]　8頁</ref>，因此1946年3月1日時，在[[皇居外苑|皇居外苑]]舉辦感謝集会<ref>『弁護士布施辰治』 224頁</ref>。

== 逸事 ==
*辭去檢察官後，由於有一段時間擔心自己的出路，因此想要成為[[將棋|將棋]][[棋士|棋士]]，最終也獲得三段的段位<ref>『弁護士布施辰治』 24頁</ref>。
* 擁有先天性[[色盲|色盲]]<ref>『弁護士布施辰治』 3頁</ref>。
* 曾在[[二林事件|二林事件]]當中，來台參加1927年的公審而為相關被告進行辯護，並且於來臺灣不到十天內，在225小時中就奔走臺灣南北21處，演講32場<ref>http://blog.yam.com/lawpaper/article/72326538</ref>。
