'''布拉格15区'''位于[[布拉格|布拉格]]东南部，面积1024.94公顷，由上梅涅赫卢比（Horní Měcholupy）和豪斯吉瓦希（Hostivař）两个[[地籍|地籍]]区组成，[[1994|1994]]年11月18日成立。之前，豪斯吉瓦希属于[[布拉格10区|布拉格10区]]，上梅涅赫卢比则是独立的[[自治市区_(捷克)|自治市区]]。.
 
== 布拉格15区行政区 ==
布拉格15区政府受[[布拉格|布拉格]]市的委托，为附近的布拉格-下梅涅赫卢比（Dolní Měcholupy）、布拉格-杜布齐（Dubeč）、布拉格-贝德罗维采（Petrovice）、布拉格-史杰尔博赫利（Štěrboholy）等[[自治市区_(捷克)|自治市区]]执行扩大行政职责。1994—2001年，布拉格15区也为布拉格-贝尼采（Benice）、布拉格-格勒夫拉蒂（Kolovraty）、布拉格-内德维希（Nedvězí）和布拉格-乌赫希涅维斯（Uhříněves）执行扩大职责，这些市区2001年合并为新的[[布拉格22区|布拉格22区]]。

== 教育 ==
布拉格15区境内有5所小学（包括初中），1所文艺小学，3所私立中学（师范中专，查佩克兄弟高中和私立技术中专学校），1所公立中专学校和8所幼儿园。

== 自然环境 ==
布拉格15区包括豪斯吉瓦希水库的一部分和哥辛耐茨丘（Kozinec）。哥辛耐茨上发现了古代[[斯拉夫|斯拉夫]]族的城堡遗址。水库建在博吉溪（Botič）上，从水库坝到旧豪斯吉瓦希的博吉溪曲流地带被列为自然保护区。15区还设有[[鸟类标号环|鸟类标号环]][[加环站|加环站]]和窦乐茨园Toulcův dvůr[[布拉格|布拉格]]市[[生态|生态]]教育中心。

== 文物和纪念碑 ==
* 窦乐茨园：14世纪建立的小贵族堡垒，后经过晚期[[哥特式|哥特式]]和[[文艺复兴|文艺复兴]]风格的改造，目前是[[生态|生态]]教育中心。
* [[聖若翰洗者|圣施洗者约翰]]被斩首教堂：13世纪的早期[[哥特式|哥特式]]教堂，围墙和木制钟楼有18世纪的基础。
* 豪斯吉瓦希的古老民居、农园和住宅。
* 豪斯吉瓦希墓地：19世纪以来的塑像，包括安多宁•施韦拉和弗•萨•沙勒达的坟墓。
* 豪斯吉瓦斯和上梅涅赫卢比各有两次[[世界大战|世界大战]]烈士纪念碑。
* 上梅涅赫卢比的铁质钟楼。

== 名人 ==
* 安多宁•施韦拉（Antonín Švehla 1873–1933），豪斯吉瓦希人，政治人物。1918年10月28日参加了[[捷克斯洛伐克|捷克斯洛伐克]]共和国的建立，后任农民党主席、[[捷克斯洛伐克|捷克斯洛伐克]][[总理|总理]]（1922-1929）。
* 弗•萨•沙勒达（F. X. Šalda 1867–1937），著名[[文学评论|文学评论]]家。他的父母晚年在豪斯吉瓦希定居，创建了家庭坟墓，沙勒达的遗体后来也埋葬于此。
* 历任区长：杨•纳德沃尼克（Jan Nádvorník 1994–2006，前参议院议员），伊瑞•贝德稀释（Jiří Petříš 2006–2010，已故），巴维•科勒伽（Pavel Klega 2010–2014），米兰•温茨乐（Milan Wenzl，现任）。
