'''筍崗站'''，又稱'''筍崗貨運站'''，原称'''深圳北站'''，位於[[中國|中國]][[广东省|广东省]][[深圳市|深圳市]][[罗湖区|罗湖区]][[筍崗街道|筍崗街道]]，距离[[深圳站|深圳站]]3公里，是[[广深铁路股份有限公司|广深铁路股份有限公司]]直属的货运[[一等站_(中国大陆)|一等站]]，主要为深圳地区到发的货物列车进行装运、编组作业，办理整车、零担、集装箱货物到发，建有52条货运股道、366个货位，货场建筑面积为190,000[[平方米|平方米]]，此站目前还被用于存放深圳和深圳东站始发和终到的车底。

==历史==
笋岗站始建于1962年2月，[[广州铁路局|广州铁路局]]在[[深圳站|深圳站]]、[[深圳东站|布吉站]]间，增设'''笋岗站'''，为[[四等站_(中国)|四等站]]，分担深圳站部分货运业务，1973年6月进行了一次擴建。1977年1月1日起，笋岗站更名为'''深圳北站'''，升级为[[三等站_(中国)|三等站]]，同时深圳站改为纯客运站，深圳北站成为当时深圳地区最主要的货运车站，担负对香港及深圳地区的全部铁路货运业务<ref name=gtjtz>{{cite book |author=广州铁路（集团）公司史志编委会 |title=广铁集团志 |year=2002 |publisher=中国铁道出版社 |location=北京 |isbn=7-113-03637-6 |page=}}</ref>。

随着中国从1970年代末开始实行[[改革开放|改革开放]]，深圳市于1979年11月升级成为[[地級市|地區一級的省轄市]]，1980年设立[[深圳经济特区|深圳经济特区]]，经济发展十分迅速，迫切需要一个集货物到发、解编、过境交接等多项功能的大型货运站，广州铁路局于1982年决定对深圳北站进行改扩建。1984年开始，随着广深铁路的[[复线铁路|复线]]建设，深圳北站改造为双向一级二场区段[[编组站|编组站]]，担任货物列车的到发及编解任务，设有正线2股、站线47股，包括下行编-{zh-hans:发;zh-hk:發;zh-sg:发;zh-tw:發;}-线14股、上行编-{zh-hans:发;zh-hk:發;zh-sg:发;zh-tw:發;}-线24股、货物线9股，其中下行编组场设有半自动驼峰调车设备<ref name=gtjtz />。

1986年，深圳北站扩建改造完成，货物运输和列车解编能力大为提高，成為當時[[華南地區|華南地區]]最大的口岸編組貨運站，并在1987年升级成为一等货运站。至1992年，深圳北站有货物仓库18座，货物站台8座，装卸线8股。从1998年底起，由于[[京九铁路|京九铁路]]的配套项目[[平湖南站|平湖南编组站]]全面投入使用，深圳北站地位逐步下降，成为只承担货运作业和客车技术作业的车站<ref name=shenzhenjtz>{{cite book |author=閻志禎、李漢君 |title=深圳市交通運輸志 |year=2001 |publisher=方志出版社 |location=北京 |isbn=7-80122-699-2 |page=}}</ref>。與此同時，由於廣深鐵路完成電氣化工程，並逐步開行電力機車及電力動車組，此站亦曾一度升格為「'''深圳機務段'''」（廣鐵深段）及「'''廣州車輛段深圳北運用車間'''」（廣鐵廣段（深北）），專門用作整備[[城際直通車|城際直通車]]及[[廣深動車組列車|廣深城際列車]]用車（包括所有早年廣深鐵路公司向廣州中車租用的車輛），但目前已經撤銷。于1980年代，深圳北站建设了多条支线线路通往邻近的[[筍崗街道|筍崗]]、[[清水河街道_(深圳市)|清水河]]物流園區货物仓库以便装卸货物，但由于铁路货物运量的减少，加上相關貨倉陸續被重建成商住項目，这些支线已极少被利用并被拆除<ref>{{Cite web |url=http://www.fangwang.com/news/index.asp?id=531473 |title=废弃多年铁轨咋还不拆? 影响居民交通生活 |accessdate=2011-07-22 |publisher= |date= 2009-04-09 }}</ref>。

在当时，深圳北站也是香港与中国大陆之间铁路货物运输的中转站，过去每天也有数趟货物列车来往[[香港|香港]]和深圳在站，由于中港两地之间的铁路货运量从1990年代末开始持续减少，[[港铁|港铁]]于2010年决定停止办理货运业务。而在深圳北站内也设有筍崗出入境邊防檢查站，为进港的“[[三趟快車|三趟快車]]”进行边检、活畜防疫等工作。

由于[[广深港客运专线|广深港客运专线]]、[[厦深铁路|厦深铁路]]的兴建，深圳将相继出现[[深圳北站|新深圳站]]、[[深圳东站|深圳东站]]、[[福田站|福田站]]等新火车站，[[中华人民共和国铁道部|铁道部]]在2008年中对深圳的火车站名称进行了梳理，对于位于[[龙华区_(深圳市)|龙华区]]（当时为[[宝安区|宝安区]]）[[民治街道|民治街道]]的广深港客运专线新深圳站正式命名为“[[深圳北站|深圳北站]]”，而位于罗湖区笋岗片区的原深圳北站恢復命名为“'''笋岗站'''”<ref>{{Cite web |url=http://www.szxf.net/help/helpnews/200811/20081123171730.html |title=投资43亿元 深圳北站2011启用 |accessdate=2011-07-22 |publisher= |date=2008-11-23 }}</ref><ref>{{Cite web|url=http://www.szpl.gov.cn/main/jqdt/main.html|title=关于我市境内新建铁路车站站点标准名称的通告|accessdate=2011-07-22|publisher=深圳市规划和国土资源委员会|date=|deadurl=yes|archiveurl=https://www.webcitation.org/6Cc2p5AjS?url=http://www.szpl.gov.cn/main/jqdt/main.html|archivedate=2012-12-02}}</ref>。

==客技站==
筍崗[[客技站|客技站]]，全称筍崗客车技术作业整备站，设置在笋岗站正线、上行场西侧，笋岗立交桥以北。由于京九铁路开通运营，深圳站始發长途列车数量逐步增加，为满足客车整备的需要，深圳市政府于1996年决定兴建深圳北客技站，确定选址在笋岗村西南侧，之后由市领导协调进行拆迁补偿评估，并决定补偿金额由铁路部门出资8000万元人民币支付，其余从市国土基金内支付，后又有深圳市人大代表、政协委员对选址提出异议，经过有关部门重新组织调研、论证及环保部门的审议，深圳市政府于2001年7月6日再次确定同意铁路客技站选址按照《笋岗片区法定图则》不变，确定在深圳北站西侧。尽管客技站选址已经确定，但建设进度一直十分缓慢，主要原因在于拆迁征地问题无法妥善解决，部分土地因拆迁补偿费偏高，市政府部门未批复，导致建设用地长时间未能移交广深铁路股份有限公司，直接影响到铁路客技站用地审批程序<ref>{{Cite web |url=http://www.sztb.gov.cn/gov/detail.aspx?paraID=1474&clsID=131&pid=187 |title=深交函226号(提案20040461关于尽快新建惠(州)深(圳)铁路的提案) |accessdate=2011-07-22 |publisher=深圳市交通运输委员会 |date=2005-11-20 }}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref>{{Cite web |url=http://www.sztb.gov.cn/gov/detail.aspx?paraID=1235&clsID=159&pid=187 |title=深交函308号(建议第20040032号关于尽快建成深圳市铁路客技站的建议答复) |accessdate=2011-07-22 |publisher=深圳市交通运输委员会 |date=2005-11-19 }}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref>{{Cite web |url=http://news.sina.com.cn/c/2005-08-27/14346796978s.shtml |title=广深四线铁路重要一环深圳北客技站将在年内动工 |accessdate=2011-07-22 |publisher=深圳晚报 |date=2005-08-27 }}</ref>。

2006年末，深圳北客技站作为广深铁路四线建设的配套项目正式全面动工，2008年竣工。目前，笋岗客技站占地面积30万平方米，设置有客车技术整备作业线及列检线共14条股道、洗车场、检修场以及上水站、加油站、食品供应站等设施。
