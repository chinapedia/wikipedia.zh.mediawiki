'''成龙英皇影业有限公司'''('''Jackie Chan Emperor Movies Limited''')，簡稱'''JCE'''，[[香港|香港]][[电影制作|电影制作]]及发行公司，始建于2003年8月，总部设立在[[湾仔|湾仔]]，隶属于[[英皇集团|英皇集团]]，由演员[[成龙|成龙]]和英皇集团总裁[[杨受成|杨受成]]共同建立。该公司成立以来，主要是制作和销售成龙电影为主，迄今已发布了《[[新警察故事|新警察故事]]》（2004年）、《[[神话_(电影)|神话]]》（2005年）等。它还接受发布其他香港电影。

*《[[新警察故事|新警察故事]]》（2004年）
*《[[精武家庭|精武家庭]]》（2005年）
*《[[神话_(电影)|神话]]》（2005年）
*《[[海南雞飯_(電影)|海南雞飯]]》（2005年）
*《[[長恨歌_(電影)|长恨歌]]》（2005年）
*《[[宝贝计划|宝贝计划]]》（2006年）
*《[[一个好爸爸|一个好爸爸]]》（2008年）
*《[[新宿事件|新宿事件]]》（2009年）{{citation needed}}
*《[[大兵小将|大兵小将]]》（2010年）<ref>{{cite web |title = Little Big Soldier |url = http://www.hkcinemagic.com/en/movie.asp?id=11017 |publisher = HKCinemagic |accessdate = 2009-06-11}}</ref><ref>{{cite web |title = First look at Jackie Chan's 'Little Big Soldier'|url = http://www.kungfucinema.com/first-look-at-jackie-chans-little-big-soldier-7445 |publisher = Kung Fu Cinema |accessdate = 2009-08-28}}</ref>
*《[[辛亥革命_(电影)|辛亥革命]]》（2011年）
