'''成功火力發電廠'''為一座位於[[臺灣|臺灣]][[臺東縣|臺東縣]][[成功鎮|成功鎮]]的[[火力發電廠|火力發電廠]]。成功火力發電廠自[[台灣電力公司|台灣電力公司]]完成台灣[[花東縱谷|花東縱谷]]地區的[[電力|電力]]輸送線路後，成功火力發電廠遭到裁撤，因此目前該發電廠已停機廢棄，原址改為'''台灣電力公司台東營業區成功服務所'''。

== 沿革 ==
=== 起源 ===
台東成功新港地區最初是在[[台灣日治時期|日治時期]]1935年（昭和10年）時，因當時的[[新港漁港|新港漁港]]建設工程完工，一併也將[[漁港|漁港]]所需的'''新港製冰工場'''完工，而為了製冰工場所需電力而設置了兩座發電機組，在消耗製冰所需的電量後，多餘的電力則轉賣供給民營的'''新港電氣組合'''讓當時新港地區居民使用。 然而當時因發電機組發電量有限，所以當時僅有194戶人家擁有電力可用，並且限制每天僅晚間供電5個小時。<ref name="台灣社區通 台東縣成功鎮成功愛鄉協會 成功供電滄桑史">{{cite web |url=https://sixstar.moc.gov.tw/blog/aisang/myBlogArticleAction.do?method=doListArticleByPk&articleId=46388 |title=台灣社區通 台東縣成功鎮成功愛鄉協會 成功供電滄桑史 |accessdate=2015-02-19 |deadurl=yes |archiveurl=https://web.archive.org/web/20150219161943/https://sixstar.moc.gov.tw/blog/aisang/myBlogArticleAction.do?method=doListArticleByPk&articleId=46388 |archivedate=2015-02-19 }}</ref>

=== 興建 ===
後來，隨著新港地區居民增加，原先新港製冰工場的多餘電力已不敷當時新港地區的居民使用，因此1940年（昭和15年）4月新港電氣組合在現今台灣電力公司台東營業區成功服務所現址興建火力發電廠，當時發電量20K.W.，是為新港火力發電所。<ref name="台灣社區通 台東縣成功鎮成功愛鄉協會 成功供電滄桑史"/><ref>{{cite web |url=https://twstudy.iis.sinica.edu.tw/twstatistic50/INDUSTRY/Mt287.xls |title=台灣電力公司發展經過 |accessdate=2015-02-19 |deadurl=yes |archiveurl=https://web.archive.org/web/20160304121125/http://twstudy.iis.sinica.edu.tw/twstatistic50/INDUSTRY/Mt287.xls |archivedate=2016-03-04 }}</ref>

然而因為新港地區居民數量仍不斷增加，導致新港火力發電所很快就再次不敷新港地區居民使用，因此，民國30年後，新港電氣組合原本籌畫在成功鎮[[三仙里|三仙里]][[富家溪|富家溪]]上游利用水位高低落差進行[[水力發電|水力發電]]。在水力發電廠廠房等土木工程都完工等待發電機組到位之際，卻因[[第二次世界大戰|第二次世界大戰]][[太平洋戰爭|太平洋戰爭]]，從[[日本|日本]]利用船運運送過來的發電機組連續兩度都遭到[[美國海軍|美國海軍]]的[[潛水艇|潛水艇]]擊沉。因此該水力發電廠之工程計畫至此告吹。<ref name="台灣社區通 台東縣成功鎮成功愛鄉協會 成功供電滄桑史"/>

1940年（昭和15年）6月，[[台灣總督府|台灣總督府]]接管台灣全島所有的發電與供電業務，同年8月'''[[花蓮港|花蓮港]][[電氣|電氣]][[株式會社|株式會社]]'''（東部電氣株式會社的前身）再將新港電氣組合收購，至此，新港地區發電業務收歸官方所有。<ref>{{cite web |url=http://www.airiti.com/teps/ec/ecjnlarticleView.aspx?jnlcattype=1&jnlptype=1&jnltype=462&jnliid=5002&issueiid=95202&atliid=1973808 |title=台灣電子期刊服務網 日治時期東台灣的民營電力事業(1917~1944) | accessdate=2015-02-19}}</ref> <ref name="台灣社區通 台東縣成功鎮成功愛鄉協會 成功供電滄桑史"/>

=== 燒毀 ===
1943年（昭和18年）時，一場大火將新港火力發電所燒毀，而當時也因日本第二次世界大戰戰況吃緊，日本台灣總督府已無力進行修復，因此，自此新港地區居民再次陷入夜間僅能使用[[煤油燈|煤油燈]]進行照明的黑暗期。<ref name="台灣社區通 台東縣成功鎮成功愛鄉協會 成功供電滄桑史"/>

=== 重建 ===
1945年（民國34年），[[臺灣戰後時期|中華民國接管臺灣]]，當時台灣電力公司接手東部電氣株式會社之發電與供電業務，並針對已燒毀的新港火力發電所重建。在重建後的新港火力發電所中建置5部發電機組，夜間供電時間也從原先的5個小時增加到6個小時。<ref name="臺灣電力公司 台東區營業處沿革"/><ref name="台灣社區通 台東縣成功鎮成功愛鄉協會 成功供電滄桑史"/>

後來隨著台灣電力公司增加成功火力發電廠的發電機組，讓成功火力發電廠的總發電量到達511K.W.，供電時間也6個小時改完全天候供電。 <ref name="臺灣電力公司 台東區營業處沿革">{{cite web |url=http://branch.taipower.com.tw/Content/Messagess/contents.aspx?SiteID=564732647050047760&MmmID=564744743017677204 |title=臺灣電力公司 台東區營業處沿革 | accessdate=2015-02-19}}</ref>

=== 停機 ===
台灣電力公司因應業務發展需求，開始興建花東輸電線路，整體工程在1962年（民國51年）間完成，使原本屬於單獨發電個體的台東地納入了台灣全島的台灣電力公司整體電力系統之中，而隨之的成功火力發電廠也在此之後結束發電並裁撤。 <ref name="臺灣電力公司 台東區營業處沿革"/>
