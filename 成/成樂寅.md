'''成樂寅'''（[[韩语|韩语]]：'''성낙인'''，{{bd|1950年|8月24日|||catIdx=吴}}），[[韩国|韩国]]的的行政学者兼大学教授，2014年任[[首尔大学|首尔大学]]校长。

==学历==
* 京畿高中毕业。
* 汉城大学法律系法学学士
* 汉城大学研究生院法学硕士及博士课程进修
* 法国[[巴黎第二大学|巴黎第二大学]]研究生院国内公法（宪法学）法学博士

== 主要经历 ==
* 1980年 ~ 1999年 : [[岭南大学|岭南大学]]法律学院教授
* 1991年 ~ 1997年 : [[言论仲裁委员会|言论仲裁委员会]]委员 
* 1995年 ~ 1997年 : 总统咨询教育改革委员会委员
* 1996年 ~ 2005年 : 国务总理行政审判委员会委员(信息公开专门委员长)
* 1999年 ~ 2002年 : 首尔大学的法律大学副教授
* 2000年 ~ 2002年 : 首尔大学的法律大学教务事务副院长兼法学部长
* 2001年 ~ 2003年 : [[行政自治部|行政自治部]]司法考试委员会委员
* 2001年 ~ 2002年 : [[韩国宪法学会|韩国宪法学会]]副会长
* 2002年 : [[韓國敎員團體總聯合會|韓國敎員團體總聯合會]] 교권지원지위원회 위원장, 교권옹호위원회 위원장  
* 2002年 ~ 2004年 : [[大韩民国信息通讯部|信息通讯部]]网络地址资源管理法制定推进委员会委员长
* 2002年 ~ 2004年 : 首尔大学教授协会理事
* 2003年 ~ 2004年 : 首尔大学教授伦理委员会委员
* 2003年 ~ 2004年 : 首尔大学人事委员会及两性平等委员会委员
* 2004年 ~ 2005年: 大检察厅尊重人权调查制度惯行改善委员会委员长
* 2004年 ~ 2006年 : 首尔大学 法律大学校长
* 2005年 ~ 2007年 : 韩国公法学会会长
* 2005年 ~ 2011年 : [[宪法裁判所|宪法裁判所]]咨询委员
* 2008年 ~ 2010年 : 首尔大学评议委员会委员
* 2008年 ~ 2010年 : 首尔大学 법인화위원회 거버넌스분과위원장
* 2008年 ~ 2010年 : 国会宪法研究咨询委员会副委员长
* 2009年 ~ 2012年 : 大法院大法官候选人推荐委员会委员
* 2009年 ~ 2013年 : 韩国法学教授会会长
* 2010年 ~ 2013年 : 国会公职人员伦理委员会委员长
* 2011年 ~ 至今 : 内容纠纷调整委员会委员长
* 2012年~ 2014年: 警察委员会委员长(第8届) 
* 2014年 ~ 至今 : 首尔大学校长(第26届)

== 获奖经历 ==
* 1989年 :韩国公法学会学术鼓励奖[<ref>目前的新进学术奖.</ref> 正式奖(宪法部分)
* 2005年 : 大韩民国黄条勤政勋章(总统)
* 2010年 : 第20届相许可对象法律部门对象
* 2012年 : 韩国宪法学会学术奖
* 2014年 : 第6届大韩民国法律对象(学术领域)
