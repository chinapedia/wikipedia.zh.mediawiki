'''成之凡'''{{BD|1928年||||catIdx=C成}}祖籍[[湖南省|湖南省]][[湘乡市|湘乡市]]苏坡乡普利堂，[[北京人|北京人]]，[[法国华人|法国华人]]音乐家、国画家、文物收藏家、哲学家、社会活动家，毕业于上海音乐专科学校，曾三度宣布竞选[[法国总统|法国总统]]，但因未能获得法国500名议员和地方民选代表的联署签名认可，无法参加。<ref name="sina" >{{cite web |url=http://news.sina.com.cn/richtalk/news/world/9903/031724.html |title=特写：三次竞选法国总统的华裔女人 |accessdate=1999 |publisher=新浪网 |work= |language=zh-hans}}</ref><ref name="pig">{{cite web|url=http://www.xxtzb.gov.cn/a/tongzhanshihua/201001%7D/220.html|title=竞选过法国总统的华裔女士成之凡|accessdate=2010|publisher=湘乡市统一战线|work=|language=zh-hans|deadurl=yes|archiveurl=https://web.archive.org/web/20161005180819/http://www.xxtzb.gov.cn/a/tongzhanshihua/201001%7D/220.html|archivedate=2016-10-05}}</ref><ref name="long" >{{cite web |url=http://www.china.com.cn/chinese/news/1171481.htm |title=法籍女华人成之凡曾三次参加法国总统竞选(图) |accessdate=2006 |publisher=中国网 |work= |language=zh-hans}}</ref>

成之凡曾在巴黎多次举办个人画展，其作品收入[[巴黎东方博物馆|巴黎东方博物馆]]、[[法国国家现代博物馆|法国国家现代博物馆]]。<ref name="sina" /><ref name="pig" /><ref name="long" />

==生平==
1928年，成之凡出生在[[北京|北京]]。成之凡的父亲[[成舍我|成舍我]]是中华民国报人，母亲杨王蕃曾就读北平女子师范大学。<ref name="sina" /><ref name="pig" /><ref name="long" />

11岁时，成之凡考入上海音乐专科学校，毕业之后在[[上海音乐学院|上海音乐学院]]教书。<ref name="sina" /><ref name="pig" /><ref name="long" />

1951年5月，一名法国钢琴家到[[中华人民共和国|中华人民共和国]]访问，收她做徒弟，11月，将她带到[[法国|法国]][[巴黎|巴黎]]。<ref name="sina" /><ref name="pig" /><ref name="long" />

1973年4月，成之凡以“巴黎[[欧洲音乐学院|欧洲音乐学院]]教授、法国神圣画沙龙理事、法国国家现代博物馆艺术委员会委员”的身份在巴黎建立“[[法国道教协会|法国道教协会]]”，她担任主席。<ref name="sina" /><ref name="pig" /><ref name="long" />

1981年10月，成之凡开始竞选法国总统，她主张“一元二位制”，即法国应该一男一女分别担任总统，但是她的参选只到了准候选人的阶段。<ref name="sina" /><ref name="pig" /><ref name="long" />

1988年，成之凡再度竞选法国总统，提出“一元三位制”，即法国总统应该三人分担，得到了5位数的选票。同年，成之凡回到湘乡市家乡访问。<ref name="sina" /><ref name="pig" /><ref name="long" />法国内政部没有其参加总统选举的证明，1988年法国总统选举第5位为法国共产党参选人André Lajoinie。<ref name="wiki" >{{cite web |url=https://en.wikipedia.org/wiki/French_presidential_election,_1988 |title=1988年法国总统大选结果}}</ref>

1995年，成之凡第三次竞选法国总统，提出“天下为公”、“世界大同”（孙中山思想），但是仍然失败告终。<ref name="sina" /><ref name="pig" /><ref name="long" />法国内政部没有其参加总统选举的证明。<ref name="wiki2" >{{cite web |url=https://en.wikipedia.org/wiki/French_presidential_election,_1995|title=1995年法国总统大选结果}}</ref>

==家庭==
成幼殊的母亲名叫“杨王蕃”，父亲“成舍我”是中华民国著名报人，一生追随[[孙中山|孙中山]]“[[三民主义|三民主义]]”理想，后[[中国国民党|中国国民党]]统治的[[中国大陆|中国大陆]]沦陷，远走[[台湾|台湾]]，曲线救国，建立台湾[[世界新聞職業學校|世界新聞職業學校]]（今 [[世新大學|世新大學]]）。<ref name="lion" >{{cite journal | language =zh-hans | author =崔述伟 | title =《成幼殊：诗歌伴随我一生》| journal =《老年人》 | volume =第4期 | year = 2012年 | pages =第24、25、26页  | issn = 1007-2116 }}</ref><ref name="cat" >{{cite web |url= http://news.hexun.com/2010-03-10/122924725.html|title=成舍我与成露茜 父女独立报人的一个世纪 |accessdate=2010 |publisher=和讯网 |work= |language=zh-hans}}</ref><ref name="ten" >{{cite web |url= http://news.qq.com/a/20100311/003284.htm|title=成舍我与成露茜：独立报人的一个世纪 |accessdate= 2010|publisher=腾讯网 |work= |language=zh-hans}}</ref>

成之凡有一个妹妹[[成幼殊|成幼殊]]，是中华人民共和国外交官和诗人；她的弟弟[[成思危|成思危]]，是中华人民共和国政治人物。同父异母妹妹[[成嘉玲|成嘉玲]]、[[成露茜|成露茜]]。<ref name="lion" /><ref name="cat" /><ref name="ten" />

1953年，成之凡和贝尔覃（Philipie Bertrand）在[[基督教|基督教]]教堂结婚。<ref name="sina" /><ref name="pig" /><ref name="long" />
