'''成都战役'''是[[第二次国共内战|第二次国共内战]]后期的一次战役，发生于1949年12月，通过此次战役，[[刘伯承|刘伯承]]、[[邓小平|邓小平]]率领[[中国人民解放军第二野战军|中国人民解放军第二野战军]]在[[成都平原|成都平原]]歼灭[[胡宗南|胡宗南]]所率领的[[国军|国军]]30余万人，和平进占[[成都|成都]]。

== 战役前期 ==

1949年12月，解放军第二野战军攻占[[重庆|重庆]]后，自重庆西进，以切断国军退往[[西昌|西昌]]的通路，实现自己歼灭国军主力于[[成都平原|成都平原]]的战役目标。

南路解放军令2个军进军[[乐山|乐山]]、2个军进军[[邛崃|邛崃]]、[[新津|新津]]，北路解放军令第18兵团自川北分路南下。12月11日，[[郭汝瑰|郭汝瑰]]率部在[[宜宾|宜宾]]宣布起义。其后，南路解放军于12月13日攻占[[宜宾|宜宾]]，15日攻占[[简阳|简阳]]，16日攻占[[乐山|乐山]]，18日攻占[[彭山|彭山]]，19日攻占[[邛崃|邛崃]]并在[[峨眉山市|峨眉]]俘虏[[宋希濂|宋希濂]]，20日攻占[[大邑|大邑]]。

北路解放军于12月14日攻占[[广元|广元]]，18日攻占[[剑阁|剑阁]]，21日攻占[[绵阳|绵阳]]、[[巴中|巴中]]，22日攻占[[江油|江油]]。

自此，解放军从南北两面完成了对成都平原的包围。

== 战役中期 ==

12月21日，[[蒋介石|蒋介石]]令[[胡宗南|胡宗南]]率部自成都地区向[[西康|西康]]、[[云南|云南]]地区突围。22日，胡宗南在新津召开军事会议，决定第5兵团向[[西昌|西昌]]突围，第18兵团向[[云南|云南]][[昭通|昭通]]突围，第7兵团向[[贵州|贵州]][[威宁|威宁]]突围，第15、20兵团向贵州[[毕节|毕节]]突围。次日，胡宗南乘飞机前往西昌，飞机最终却降落到了[[海南岛|海南岛]]，导致国军在突围过程中失去了统一指挥，陷入一片混乱之中，败局已定。

== 战役后期 ==

12月21日，川鄂边区绥署副主任[[董宋珩|董宋珩]]和第16兵团在[[什邡|什邡]]宣布投共，24日，第15兵团司令[[罗广文|罗广文]]和第20兵团司令[[陈克非|陈克非]]分别在[[郫县|郫县]]和[[安德|安德]]宣布投共，25日，第7兵团司令[[裴昌会|裴昌会]]在[[德阳|德阳]]宣布投共，26日，第20军军长[[杨汉烈|杨汉烈]]和第127军军长[[赵子立|赵子立]]分别在[[金堂|金堂]]、[[巴中|巴中]]宣布投共。

到此时，国军只剩下第5兵团司令[[李文_(中將)|李文]]决定执行胡宗南的命令，自24日起向邛崃攻击前进，希望经此到达[[雅安|雅安]]，于27日被解放军全歼，李文被俘。当日，第18兵团司令[[李振|李振]]在[[成都|成都]]宣布投共，第二野战军第18兵团进占成都。至此，国军胡宗南部30余万主力在成都平原基本被歼灭，成都战役结束。

== 评价 ==

通过成都战役，解放军基本歼灭了退往西南地区的国军主力，国军在成都战役后不久最终退出大陆，蒋介石自成都飞往[[台北|台北]]后，终生没有再回到大陆。
