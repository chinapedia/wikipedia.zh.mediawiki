'''双色球'''是由[[中国福利彩票|中国福利彩票]]发行管理中心[[组织|组织]]的，全国范围[[销售|销售]]发行的[[博彩|乐透型]](Lottery)[[彩票|彩票]]。由[[民政部|民政部]]的直属事业单位"[[中国福利彩票发行管理中心|中国福利彩票发行管理中心]]"负责发行销售的彩票。其发行的宗旨是向社会募集资金用于福利事业，统一每注售价[[2￥|2元]]人民币。

双色球为目前中国（大陆）范围内销售量及关注度最高的彩票，每期投注额大概3.5亿。

== 规则 ==
双色球每注投注号码由6个红色球号码和1个蓝色球号码组成。红色球号码从1--33中选择；蓝色球号码从1到16中选择。每注需要选择6个红色球号码，1个蓝色球号码。

每注统一投注金额为2元。采用全国统一[[奖池|奖池]]计奖。

双色球每周销售三期，分别在周二、周四、周日晚间开奖。

== 兑奖 ==
兑奖期限为开奖后一日起记60天内，除一、二等奖须到指定中心兑现外，其余奖金可在投注站中直接兑现。

一、二等奖须到各地省中心验证、登记和兑奖。

== 票面 ==
正面使用统一的打印格式，但各地稍有不同，主要有：投注日期、投注号码、投注注数、投注额、暗码等。

背面各地区可自行设计相关样式，但不得有商业广告内容。主要题材有本地区风观、捐助项目展示厅等。

== 质疑 ==
===作弊传闻===
双色球福彩不乏作弊传闻，尤其在[http://v.youku.com/v_show/id_XMTI4MjM3MjQ=.html 2004009期双色球的开奖]片段中，摇奖器中滑落下来的奖球位置，与小屏幕中的奖球有明显差别。被网友揭露为作弊。但[[中国福利彩票|中国福利彩票]]官方对此回应在录制双色球第2004009期开奖过程时，由于拍摄特写镜头的摄像机设置错误，导致录制过程中无法切换到特写画面，没能采集到特写画面。而摇奖一旦开始，中间不能停顿，也不能重新录制。考虑到观众的收视习惯，制作人员按照已经摇出并经过公证的号码，补拍了特写镜头<ref>[http://www.china.com.cn/chinese/2004/Feb/496813.htm 双色球摇奖画面造假 中国福彩中心向彩民道歉]</ref>。 
===延迟开奖事件===
双色球开售后共发生8起延迟开奖事件，但有5起原因不明，最长的长达4小时。2015年1月25日，原本于晚上9时30分在[[中国教育电视台|中国教育电视台]]一频道（中国教育1）直播的双色球第2015011期开奖节目，在没有事先预告的情况下突然取消，改为播出[[广告|广告]]，导致当期开奖延迟，此为2014年“双色球5亿派奖”的最后一期。晚上9时52分，中国教育电视台一频道以[[游动字幕|字幕飞播]]方式播出公告开奖延迟情况。晚上9时54分，中彩网官方微博称“由于销售数据通讯传输故障，导致本期双色球摇奖无法按时进行，待故障排除后再在公证员的监督下进行开奖”。直至晚上11时20分故障排除后，晚上11时55分开奖。1月26日1时30分，中国教育电视台播出了该期双色球开奖的实况录像。1月26日，[[中华人民共和国民政部|中华人民共和国民政部]]办公厅副主任[[陈日发|陈日发]]在新闻发布会表示，由于[[重庆|重庆]]福彩销售系统在数据汇总过程中出现数据索引异常，未能及时完成数据汇总和数据传输，导致无法统计全国销量，因此该期双色球摇奖未按时进行。有[[网友|网友]]表示，延迟开奖背后可能隐含暗箱操作等造假行为，并将此问题向[[中纪委|中纪委]]进行了举报。而另外2起延迟开奖的原因分别是通讯故障（2009年12月1日第2009141期）、因湖南暴雨致该地区数据上传较晚，无法按时整合统计汇总（2013年5月7日第2013052期）<ref>{{cite news|title=福彩双色球开奖突取消 官方回应：数据传输故障 |url=http://news.sohu.com/20150126/n408064698.shtml|publisher=搜狐 |date=2015-01-26 |accessdate=2015年1月27日}}</ref><ref>{{cite news|title=盘点近年8起双色球延迟开奖：5起原因不明|url=http://news.sina.com.cn/c/2015-01-27/023931446454.shtml|publisher=新浪 |date=2015年1月27日 |accessdate=2015年1月27日}}</ref>。
