'''双龙棋具杯中韩青年擂台赛'''，是[[中国|中国]]和[[韩国|韩国]]之间的一项[[围棋|围棋]]赛事，于2010年3月10日开战。由韩国ORO围棋网、[[新浪|新浪]]围棋网主办，双龙棋具、中国福建制茶世家、韩国三千里自行车行提供赞助。

== 比赛时间 ==
2010年3月10日起，每周三、四进行。
== 比赛形式与规则 ==
中韩每方10人进行擂台赛，采用网络对弈。黑贴6目半，每方保留时间30分钟，3次30秒读秒。断线四次系统判负。
== 奖金 ==
胜方1500万韩元，负方500万韩元。
