'''双溪大年站'''（{{lang-ms|Stesen Kereta Api Sungai Petani}}）位于[[马来西亚|马来西亚]][[吉打州|吉打州]][[双溪大年|双溪大年]]市，属于[[KTM电动列车服务|KTM电动列车服务]]（ETS）和[[KTM通勤铁路北马区|KTM通勤铁路北马区]]的其中一站。[[马来亚铁道公司|马来亚铁道公司]]是该车站的主要经营者。

*'''双溪大年'''至[[曼谷|曼谷]]
