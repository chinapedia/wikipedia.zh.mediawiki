'''鴿石斛'''（[[学名|学名]]：''{{lang|la|Dendrobium crumenatum}}''），又名'''木石斛'''，是[[蘭科|蘭科]][[石斛蘭屬|石斛蘭屬]]的一种热带附生兰，广泛分布於台湾[[绿岛|绿岛]]、[[东南亚|东南亚]]和[[太平洋|太平洋]]的各大岛屿。

== 特征 ==
[[File:Dendrobium_crumenatum_2.jpg|thumb]]
鴿石斛会开出白色的花朵，花非常芬芳，唇盘为淡黄色。花开需由突然的降温引起，至少降温5.5℃才能使其开花，如雨後或其他人工条件都能办到。其花壽很短，只有2日。从侧面观察，鴿石斛的花很像飞翔的[[鸽子|鸽子]]。在英文中，[[鸽子兰|鸽子兰]]（dove orchid）与鴿石斛（pigeon orchid）由於英文名很相似，有时会混淆。

== 分布 ==
[[File:Dendrobium_crumenatum_4.jpg|thumb]]
鴿石斛常见於[[马来西亚|马来西亚]]和[[新加坡|新加坡]]，自然生长在[[热带雨林|热带雨林]]中。鴿石斛常被种植在街道两旁以美化环境，适於布置花境和花坛，也适宜作为背景花卉材料。在[[马来西亚普渡大学|马来西亚普渡大学]]（University Putra Malaysia）中，鴿石斛常见於校园中，一般生长在街道两旁[[雨豆樹|雨豆樹]]的树干上。在野外，鴿石斛常见於[[槟城|槟城]]的[[阿依淡水坝|阿依淡水坝]]（Air Itam Dam）和[[升旗山|升旗山]]（Penang Hill）。

== 文化象征 ==
在[[马来西亚|马来西亚]]的一些州，如[[霹靂州|霹靂州]]和[[彭亨州|彭亨州]]，鴿石斛与精神信仰有关。马来人社区会将鴿石斛种植在房前，一般是大门附近，用於避邪，保护家庭的安全。人们相信这种植物能阻挡邪恶的力量，防止它们进入屋内。
