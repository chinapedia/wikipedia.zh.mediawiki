'''亳州学院'''是一所位于[[中华人民共和国|中华人民共和国]][[安徽省|安徽省]][[亳州市|亳州市]]的公立普通本科院校。学校前身是创立于1909年的蒙城师资讲习所，2002年升格为亳州师范高等专科学校，2016年在亳州师专的基础上成立亳州学院，成为本科层次的高等院校<ref>{{cite news|url=http://ah.anhuinews.com/system/2016/05/20/007353698.shtml?qgdxuoargwqqnhmp|title=亳州学院5月20日正式揭牌（图）|date=2016-05-20|publisher=中安在线}}</ref>。

== 历史沿革 ==
* 1909年，创办蒙城师资讲习所
* 1952年，建立[[蒙城师范学校|蒙城师范学校]]
* 2002年，升格为亳州师范高等专科学校，成为专科层次的高等院校
* 2011年，学校从[[蒙城县|蒙城县]]整体搬迁到亳州市区<ref>{{cite news|url=http://www.eol.cn/anhui/anhuinews/201604/t20160412_1385766.shtml|title=亳州师专升为本科更名亳州学院 结束亳州无本科历史|date=2016-04-12|publisher=中国教育在线}}</ref>
* 2016年5月，升格为亳州学院，成为本科层次的高等院校，结束了亳州市无本科院校的历史<ref>{{cite news|url=http://ah.sina.com.cn/news/g/2016-05-16/detail-ifxsenvn7193573.shtml|title=亳州师范高等专科学校升格为亳州学院|date=2016-05-16|publisher=新浪网}}</ref>

== 现状 ==
2017年学院面向全国11个省市招生，招生专业33个，其中本科专业9个：学前教育、小学教育、文化产业管理、电子信息工程、汉语言文学、经济与金融、生物工程、制药工程、运动康复<ref name=jj>{{cite web|url=http://www.bzuu.edu.cn/10/list.htm|title=学院简介|publisher=亳州学院|accessdate=2017-12-22}}</ref>。

== 组织机构 ==
学校现设有10个教学系部<ref name=jj/>：
* 教育系
* 生物与化学工程系

== 学生与教师 ==
学校现有全日制在校[[专科生|专科生]]2400余人，成人[[大专|大专]]生369人。学校现有在职教职工249人，专任教师171人，教师[[本科|本科]]学历100%。[[硕士|硕士]]生5人，[[研究生|研究生]]课程班结业者27人；两人次获[[曾宪梓|曾宪梓]]教师奖，一人享受省政府特殊津贴；现有[[副教授|副教授]]以上职称19人，[[高级讲师|高级讲师]]4人，[[讲师|讲师]]57人。

==文化传统==
* 学校坚持以人为本，以学生发展为本的办学思想，全面实施素质教育，形成了“培养综合素质，发展个性特长”的培养模式。
* 学校以“善学善教、育己育人、德高才博、献身教育”为[[校训|校训]]，以提高教学质量为宗旨，确立了“以课堂教学改革为突破口，学法改革与教法改革同步，全面实施素质教育”的教改目标；加强教育实践，提倡并实践改一次性实习为分段实习，培养学生学以致用的能力。
* “严谨、垂范、求真、拓新”的教风和“勤学、善思、励志、进取”的学风铸就了[[亳州师专|亳州师专]]光荣的历史，52年来，共培养1.4万余名师德高尚、师益精良，具有创新精神和实践能力的基础教育师资。
