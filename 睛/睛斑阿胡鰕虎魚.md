'''睛斑阿胡鰕虎魚'''，为[[輻鰭魚綱|輻鰭魚綱]][[虾虎魚亚目|虾虎魚亚目]][[鰕虎魚科|鰕虎魚科]]的其中一[[種|種]]

== 分布 ==
本魚分布於[[亞洲|亞洲]]及[[大洋洲|大洋洲]]的[[半淡鹹水|半淡鹹水]]水域。

== 特徵 ==
本魚體延長略呈圓柱狀，後部側扁，頭中大，吻較長，吻背平直。眼小，上側位，尾柄較短。體被小型的櫛鱗，第一背鰭前至眼後方被細小鱗片，尾形略為截平，體色呈黃棕色到淡棕色，體側中央有約6-7個黑褐色圓斑。第一背鰭後方具一大型黑色圓斑，第二背鰭及尾鰭具有3-6條垂直於鰭條之深褐色點紋。背鰭硬棘7枚，背鰭軟條10枚，臀鰭硬棘1枚，臀鰭軟條10枚，體長可達13公分。

== 生態 ==
本魚為溯河洄游的底棲性魚類，成魚在溪流中產卵繁殖，多棲息在較緩流的潭區裡，仔魚孵化後，隨水流漂至河口，幼魚則再成群洄溯至溪流中，屬雜食性，以底藻、水深[[昆蟲|昆蟲]]及小型[[甲殼類|甲殼類]]等為食。

== 經濟利用 ==
可做為食用魚。
