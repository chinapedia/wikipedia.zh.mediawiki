'''祁东四明山国家森林公园，'''位于[[湖南省|湖南省]][[衡阳市|衡阳市]][[祁东县|祁东县]]的四明山。

== 沿革 ==
1996年8月10日，祁东四明山被祁东县人民政府批准为县级森林公园<ref name=":0">{{Cite book|title=《衡阳市志：1978~2008》|last=|first=|publisher=方志出版社|year=2008年|isbn=9787514412246|location=|page=1253|pages=|chapter=第十五编 农业与农村经济}}</ref>。

2004年12月20日，祁东四明山森林公园由县级森林公园升格为省级森林公园<ref name=":0" />。

2014年12月，祁东四明山森林公园经经中国森林风景资源评价委员会审议，成功获评为[[中华人民共和国国家森林公园列表|国家级森林公园]]<ref>{{Cite news|url=http://hn.rednet.cn/c/2014/12/18/3553095.htm|title=祁东县四明山森林公园获评国家级森林公园（图）|last=段|first=海桥|date=2014年12月18日|work=红网|accessdate=}}</ref>。

== 地理位置 ==
祁东四明山国家森林公园，位于衡阳市祁东县西部边陲，西北与[[邵阳县|邵阳县]]五丰铺、[[东安县|东安县]]的南镇乡毗邻，南与[[永州市|永州市]][[冷水滩区|冷水滩区]]黄茶园接壤，东面以祁东县[[太和堂镇|太和堂镇]]为界<ref name=":0" />。

== 景观 ==
祁东四明山国家森林公园由四明山片区和鼎新湖片区两部分组成。

公园内包括长龙江大峡谷、腾云岭、登仙崖、绝情谷、碧玉湖、野牛坪瀑布等景点<ref>{{Cite news|url=http://hn.rednet.cn/c/2016/12/09/4158898.htm|title=祁东县四明山国家森林公园开园|last=陈|first=鸿飞|date=2016年12月9日|work=湖南日报|accessdate=}}</ref>。

公园内还有万福寺、紫云寺、腾云祖师殿等寺庙<ref>{{Cite web|url=http://smsfp.forestry.gov.cn/30015/96326.html|title=公园简介|accessdate=2017年2月13日|author=|date=2015年4月25日|publisher=}}</ref>。
