[[英國|英國]]'''橡樹大賽'''（Oaks, 又稱橡樹錦標、葉森橡樹大賽）是一場只限三歲純種雌馬參與的國際一級賽。賽事於每年6月初在'''[[葉森賽馬場|葉森賽馬場]]'''舉行，路程為一哩4化郎6碼（2420米）。雖然賽事的正式名稱是The Oaks，但因為'''[[葉森打吡|葉森打吡]]'''的關係，英國國內以至全球都已慣於稱它為'''葉森橡樹大賽'''（Epsom Oaks）。

這是英國馬季第三場經典賽，亦是季內第二場雌馬限定的經典賽。它同時是英國雌馬三冠的次關（首關為'''[[一千堅尼錦標|一千堅尼錦標]]'''，尾關為'''[[聖烈治錦標|聖烈治錦標]]'''）。至目前為止，史上只出現過9匹英國三冠雌馬，第一匹是1868年的Formosa，最後一匹是1985年的Oh So Sharp。

== 歷史 ==
這場歷史悠久的賽事，是以葉森鎮東部一座由'''[[爱德华·史密斯-斯坦利，第十二代德比伯爵|第十二代德比伯爵]]'''所擁有的莊園'''橡樹園'''（The Oaks）來命名。他和賓客們在1778年的一場派對上設想了這場比賽，並在1779年以'''橡樹錦標'''之名開始舉行。首屆頭馬Bridget正是由伯爵本人擁有。

橡樹大賽後來成為英國馬壇最重要的三歲賽事之一。至1860年代中期，這個年齡組別最頂級的五場比賽（一千堅尼錦標、'''[[二千堅尼錦標|二千堅尼錦標]]'''、橡樹大賽、葉森打吡、聖烈治錦標，特色是距離一場比一場長），漸被人們稱為'''經典賽'''。經典賽的概念，後來更傳到其他國家。

歐洲不同國家都有舉辦橡樹大賽，包括愛爾蘭的'''[[愛爾蘭橡樹大賽|愛爾蘭橡樹大賽]]'''、德國的Preis der Diana、法國的Prix de Diane和義大利的Oaks d'Italia；而在歐洲以外的國家，也有澳洲的澳洲橡樹、新西蘭橡樹和日本的'''[[優駿牝馬|優駿牝馬]]'''。

在最初的百餘屆，馬匹在橡樹大賽的負磅曾經多次更改，直至1892年才確立9[[英石|英石]]（126磅）的標準負磅。<ref name="waterman">{{Template:Cite book|title=The Punter's Friend|publisher=Queen Anne Press|year=1999|isbn=1852916001|location=Harpenden, Herts, UK|last1=Waterman|first1=Jack}}</ref>

在兩次世界大戰期間，橡樹大賽曾移師新市場馬場舉行，並改名為「新市場橡樹錦標」<ref name="waterman">{{Template:Cite book|title=The Punter's Friend|publisher=Queen Anne Press|year=1999|isbn=1852916001|location=Harpenden, Herts, UK|last1=Waterman|first1=Jack}}</ref> 。2014年為紀念去年過世的八屆橡樹冠軍練馬師Sir Henry Cecil而冠名「Sir Henry Cecil橡樹大賽」。<ref>{{Template:Cite |title=Investec Oaks to be run in memory of Sir Henry|url=http://www.racingpost.com/news/horse-racing/epsom-2014-oaks-to-be-run-in-memory-of-sir-henry-cecil/1609264/#newsArchiveTabs=last7DaysNews}}</ref>
