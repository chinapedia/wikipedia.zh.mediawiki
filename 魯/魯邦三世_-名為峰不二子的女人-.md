《'''魯邦三世 -名為峰不二子的女人-'''》（{{lang-ja|LUPIN the Third -峰不二子という女-}}）是以[[Monkey_Punch|Monkey Punch]]的漫畫《[[魯邦三世|魯邦三世]]》為原型改編的[[日本|日本]][[電視動畫|電視動畫]]。2012年4月至6月在[[日本電視台|日本電視台]]播出，全13話；[[Funimation|Funimation]]也在[[美國|美國]]網路播出了本作，附英文字幕。

== 概要 ==
本作是電視動畫《[[魯邦三世_(TV第1期)|魯邦三世 (TV第1期)]]》開始放映40周年的紀念作品。這是繼1985年《{{link-ja|魯邦三世 PartIII|ルパン三世 PartIII}}》播映完畢後，27年以來「魯邦三世」系列的新電視動畫。

不同於歷來的作品，以系列中居女主角地位的[[峰不二子|峰不二子]]的視點來展開故事，並將時間點前推到魯邦、次元、石川三人尚未結識的時期，以獨自觀點講敘正篇故事前傳意味濃厚的故事。此外，在保留系列動畫強烈冒險色彩的同時，對於性的暗示十分露骨，風格上明顯地面向成年觀眾。

本作的日文標題並未使用通常的「{{lang|ja|ルパン三世}}」，而是用英文「{{lang|en|LUPIN the Third}}」表記。

2012年12月，本作獲得第16回[[文化廳媒體藝術祭|文化廳媒體藝術祭]]動畫部門新人獎<ref>{{cite web |url=http://j-mediaarts.jp/awards/new_face_award?locale=ja&section_id=3 |title=アニメーション部門｜新人賞 |publisher=第16回文化庁メディア芸術祭 |accessdate=2012-12-14 |language=ja |deadurl=yes |archiveurl=https://web.archive.org/web/20121218233417/http://j-mediaarts.jp/awards/new_face_award?locale=ja&section_id=3 |archivedate=2012-12-18 }}</ref>。

2012年9月19日，DVD和BD版開始發售<ref>{{cite web | url=http://www.vap.co.jp/lupin/ | title=LUPIN THE THIRD -ルパン三世- | work=VAP | accessdate=2013-07-18 | language=ja}}</ref>。

=== 本作原創 ===
; {{nihongo|奧斯卡[[警部補|警部補]]|オスカー警部補||聲：[[梶裕貴|梶裕貴]]}}
; {{nihongo|路易斯·勇·阿爾梅德伯爵|ルイス・勇・アルメイダ伯爵||聲：{{link-ja|古舘寛治|古舘寛治}}}}
; {{nihongo|貓頭鷹|梟頭||聲：[[長克巳|長克巳]]、[[橋本一子|橋本一子]]（愛莎的母親）}}
: 自稱是阿爾梅德伯爵的執事。戴著茶色的貓頭鷹面具，阿爾梅德伯爵死後主管各項事務，負責與魯邦及相關人物接觸。真實身份為愛莎的母親「米奈娃」（{{lang-ja|ミネルバ}}），與丈夫弗里茲·凱撒博士一起進行研究的科學家。
; {{nihongo|弗里茲·凱撒博士|Dr.フリッツ・カイザー||聲：{{link-ja|高岡瓶瓶|高岡瓶々}}（<small>演職員表中以「博士」記載</small>）}}
; {{nihongo|愛莎|アイシャ||聲：[[小林沙苗|小林沙苗]]}}
: 弗里茲·凱撒博士的女兒。
