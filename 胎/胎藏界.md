'''胎藏界'''（{{lang-sa|{{IAST|garbhakośa-dhātu}}}}），又名'''行續'''、'''行部'''、'''行怛特羅'''（{{lang-sa|caryā-tantra}}，{{lang-bo|spyod-pahi rgyud}}），密宗術語，以漢傳密宗經典《[[大日經|大日經]]》為中心所形成的一個[[密宗|密宗]]流派。

它與[[金剛界|金剛界]]合稱兩部大法，或兩部純密，形成[[唐密|唐密]]的主體。在它之前形成的密法則稱為[[雜密|雜密]]。此宗在[[漢地|漢地]]，經[[唐武宗滅佛|唐武宗滅佛]]後失傳；然此前已及早傳入[[日本|日本]]，而在日本茁壯，並約莫自20世紀起，唐密教法反哺回漢地。

胎藏界與金剛界形成日本[[東密|東密]]及[[台密|台密]]的核心教義，在日本信奉及修行者眾多。

== 藏傳密宗觀點 ==
[[藏傳密宗|藏傳密宗]]稱胎藏界為'''行續'''、'''行怛特羅'''、'''二俱續'''、或'''事二俱瑜伽'''；依照其[[教判#藏传佛教九乘判教|判教]]，胎藏界密法屬下三部密法中的第二部。

== 釋義 ==
[[File:Taizokai.jpg|thumb]]。中央方形的中心為早期的[[毗盧遮那佛|毗盧遮那佛]]；周匝圍繞著八尊[[佛|佛]]與[[菩薩|菩薩]]（自頂按順時鐘方向依序為：[[寶幢佛|寶幢如來]]、[[普賢菩薩|普賢菩薩]]、[[開敷華王如來|開敷華王如來]]、[[文殊師利菩薩|文殊師利菩薩]]、[[阿彌陀佛|無量壽如來]]、[[觀世音菩薩|觀世音菩薩]]、[[天鼓雷音如來|天鼓雷音如來]]、[[彌勒菩薩|彌勒菩薩]]。）|left]]
胎{{ruby|藏|zàng}}界的梵文，源自於梵文字根garbha，有隱藏、包含之意，在密宗之中，意指為[[佛性|佛性]]、或[[如來藏|如來藏]]。胎藏界以《大日經》「[[菩提心|菩提心]]為因，大悲為根本，以[[方便|方便]]而至究竟」三句義為根本。認為人皆具備[[如來藏|如來藏]]佛性，是一切眾生本有的清淨菩提心，真常不變，胎藏心中；[[大日如來|大日如來]]代表的就是這個本有的清淨菩提心。修行真言密法的目的，就是在於發揚這個清淨菩提心，使自己與大日如來合一。

胎藏界密法，結合了[[真言|真言]]咒語，與[[曼荼羅|曼荼羅]]。[[藏傳佛教|藏傳佛教]][[宗喀巴|宗喀巴]]認為，[[行續|行續]]外在的真言儀軌與內在的禪定修持是同等重要的，所以稱為行續<ref>《密宗道次第廣論》卷二：「若待外事內定等分，非待極多外事，即是行部之機。」「事定等行故名行續。」</ref>。它與[[事續|事續]]不同的地方在於，雖然它也重視咒語儀軌，但是它已經進入理論層次，更為重視[[般若|般若]]智慧的層面，因此，它可以被視為是[[事續|事續]]的進一步發展<ref>《密宗道次第廣論》卷二：「若內正定與諸外事等分開示立為行續。如《大日經》註云：『此續雖是方便智慧為主之瑜伽續，然為攝受信解外事諸眾生故，開示隨順外事諸行。故共稱事續或二俱續。』又《金剛藏莊嚴續》云：『事二俱瑜伽。』亦說行續為二俱續。」</ref>。

== 歷史源流 ==
胎藏界密法可能源起於南印度[[如來藏學派|如來藏學派]]，略早於[[金剛界|金剛界]]傳承，後以[[那爛陀寺|那爛陀寺]]為中心，向全印度散播。因玄奘與義淨至求學時，胎藏界密法尚未傳入那爛陀寺，因此它的興起時間，可以推估是在[[武周|武周]]至開元之間。

在金剛界傳承興起之後，它吸納了胎藏界成為它的一部份。多數唐密修行者，會同時修行二者，而以金剛界傳承為主。

=== 中國 ===
唐開元年間，由[[善無畏|善無畏]]傳入中國，其弟子[[一行|一行]]承繼之，作[[大日經疏|大日經疏]]。這是胎藏界密法在中國的第一個傳承。

後[[不空|不空]]三藏至[[錫蘭|錫蘭]]學習密法時，在普賢[[阿闍黎|阿闍黎]]座下，受金剛界與胎藏界兩部灌頂，後傳回中國，形成第二個傳承。因其弟子眾多，唐密以不空傳承為主，也因此形成金剛界與胎藏界兩部密法融合的風格。

=== 日本 ===
==== 真言宗（東密） ====
[[空海|空海]]（弘法大師）接受[[青龍寺|青龍寺]][[惠果|惠果]][[阿闍梨|阿闍梨]]的指導，屬於[[不空|不空]]三藏的傳承，他之後將將胎藏界與金剛界密法傳入日本。

==== 台密 ====
[[最澄|最澄]]至中國留學，受法於當時天台山修禪寺座主道邃上人，並從越州龍興寺順曉阿闍梨得结缘灌頂並傳胎藏界，學习此法，但由于时间紧迫并未學全。

弘仁三年，最澄親至乙訓寺拜訪[[空海|空海]]，以空海為阿闍梨，接受空海的胎藏界和[[金剛界|金剛界]]结缘灌頂。但最澄要求空海传法阿闍梨灌頂時，空海要求他先接受三年修行，拒絕他的請求。最澄要求借閱[[金剛界|金剛界]]《[[理趣經|理趣經]]釋》也遭到拒絕。於是最澄以[[天台宗|天台宗]]，結合密宗。又最澄弟子[[圓仁|圓仁]]、徒孫[[圓珍|圓珍]]亦曾入唐，求學密宗，形成台密的傳統。
