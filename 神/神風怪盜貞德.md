《'''神風怪盜貞德'''》（{{Lang-ja|神風怪盗ジャンヌ}}），又稱《'''神風怪盜'''》，是日本漫畫家[[種村有菜|種村有菜]]原作的怪盜漫畫，在[[集英社|集英社]]漫畫月刊《[[Ribon|Ribon]]》1998年2月號至2000年7月號連載，共30話（另有番外篇3話），並發行了7冊[[單行本|單行本]]。該作品後來被[[日本電視動畫|電視動畫]]化，電視動畫有44集，自1999年2月13日至2000年1月29日在日本[[朝日電視台|朝日電視台]]首播，[[台灣|台灣]]由[[台灣電視公司|台灣電視公司]]首播，[[香港|香港]]由[[無綫電視|無綫電視]]首播。

== 故事簡介 ==
主角日下部瑪倫是[[高中|高中]]女生，暗中與準天使妃茵（香港譯為阿-{菲}-，台灣譯為-{妃}-茵）一起合作對付惡魔。小麗藉由妃茵的力量可以變身成怪盜貞德，封印那些附著在物品上使擁有人魔化的惡魔，把惡魔封印後由妃茵收集。小麗在每次行動前會發出預告狀。和小麗從小一起長大的東大寺都是警察的女兒，夢想成為一名警察，一直很想逮捕怪盜貞德，一直不知道貞德的真正身份，好幾次都差點把貞德逮捕歸案。

== 登場角色 ==
'''日下部-{栗}-(台灣譯為日下部-{麻呂}-;香港譯為日下部瑪倫)／怪盜貞德'''（日下部まろん（くさかべ まろん）／怪盗ジャンヌ）[[配音員|聲]]：[[桑島法子|桑島法子]]（日）；[[曾秀清|曾秀清]]（港） ;[[陶敏嫻|陶敏嫻]] (台)
: 本作的主角。
:生日：5月30日
:年齡：17歲
:身高：154cm
:體重：43kg
:血型：A型         
:專長：韻律體操
:興趣：運動
:桃-{栗}-學園高中二年級。表面上是普通的女高中生，學校內新體操（即[[藝術體操|藝術體操]]）部成員，在準天使菲的幫助下可以利用[[十字架|十字架]]變身成為怪盜貞德，前世是[[聖女貞德|聖女貞德]]。她的任務是把即將來臨現世的惡魔封印，利用招式把惡魔「將軍」成[[西洋棋|西洋棋]]的棋子，及後因惡魔力量加強而要使用「強力將軍」封印惡魔。雖然自己一個人生活，性格仍開朗爽直，但暗裏其實是個固執的人，常常口是心非。父母離婚後被遺棄，經常躲起來，感覺孤單落寞，感受不到自己身邊其實有很多照顧自己的朋友（包括菲）。常常說自己受不了稚空，但心底是很喜歡他的。
: 曾經因為發現稚空的真實身分為怪盜辛巴達而心情低落，但在稚空的解釋下誤會冰釋。劇終兩人結婚育有一女魚月（轉世後的菲）。

: 與稚空一起行動的黑天使。在過去和菲一起成長、喜歡菲，漫畫版轉世為都與大和的獨生子心時，保留前世的記憶。

* 數位攝影 - 東映化学デジタルラボテック

* 音樂 - 加藤道明
