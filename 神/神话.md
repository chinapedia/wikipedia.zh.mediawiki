{{神話}}

在[[民俗学|民俗学]]上，'''神话'''是指关于[[人类|人类]]和[[世界|世界]]变迁的[[神圣|神圣]]故事。<ref name = "dundesintro">Dundes, Introduction, p. 1</ref>在广义上，“神话”可以指任何古老传说，<ref>Kirk, "Defining", p. 57; Kirk, ''Myth'', p. 74; Simpson, p. 3</ref> 藉由故事的形式來表達[[民族|民族]]的[[意识形态|意识形态]]。<ref>{{cite journal |last= Lincoln |first= Bruce |year= 2006 |title= An Early Moment in the Discourse of "Terrorism": Reflections on a Tale from Marco Polo |url= http://www.jstor.org/stable/3879351 |journal= [[Comparative_Studies_in_Society_and_History|Comparative Studies in Society and History]] |volume= 48 |issue= 2 |pages= 242-259 |quote= More precisely, mythic discourse deals in master categories that have multiple referents: levels of the cosmos, terrestrial geographies, plant and animal species, logical categories, and the like. Their plots serve to organize the relations among these categories and to justify a hierarchy among them, establishing the rightness (or at least the necessity) of a world in which heaven is above earth, the lion the king of beasts, the cooked more pleasing than the raw. }}</ref>
神话来源于[[原始社会|原始社会]]时期，[[人类|人类]]通过推理和想象对[[自然|自然]]现象作出解释。但是由于这时的[[知識|知識]]水準非常低下，因此经常笼罩着一层神秘的色彩。

神话中[[神|神]]的形象大多具有超人的力量，是原始人类的认识和愿望的理想化。

许多民族的原始社会的[[历史|历史]]，都是从神话故事开头的。神话中的人物大多来自原始人类的自身形象。[[狩猎|狩猎]]比较发达的[[部落|部落]]，所创造的神话人物大多与狩猎有关；[[农耕|农耕]]发达的部落所创造的神话人物多与[[农业|农业]]有关。神话中的[[英雄|英雄]]也以刀斧、[[弓箭|弓箭]]为[[武器|武器]]。从神话中，可以看到先民的一些事迹。

不论是世界[[文明|文明]]发生最早地区的原始社会民族，还是当今世界上还处在原始社会的民族，他们流传的许多神话故事都大同小异。

神话也是[[文学|文学]]的先河，是人类最早的口头散文作品。例如《[[庄子|庄子]]·應帝王》中说：“泰氏，其卧徐徐，具觉于于，一以己为马，一以己为牛。”

神话具有一定的地域性和区域性，不同的文明或者民族都有自己所理解的神话含义<ref>[[胡适|胡适]]在《[[白话文学史|白话文学史]]》里说：“‘三百篇’里……没有神话的遗迹。”“中国古代民族没有故事诗，仅有简单的神歌与风谣而已。”[[茅盾|茅盾]]曾经指出：“中国神话之系统的记述，是古籍所没有的。”又说：“神话这个词，中国向来是没有的，但神话的材料，虽然只是些片段，却见于古籍甚多，并且成为中国古代文学中的色彩鲜艳的一部分。”（茅盾《中国神话研究》）</ref>。

== 定義 ==
在學術上，學者所說的神話，必須具有幾個條件：
* 敘述人類原始時代或人類演化初期的單一事件或故事。<ref name = "mythnature">Bascom, p. 9</ref> <ref name = "mythfolk">"myths", ''A Dictionary of English Folklore''</ref> <ref>O'Flaherty, p.78: "I think it can be well argued as a matter of principle that, just as 'biography is about chaps', so mythology is about gods."</ref>
* 承傳者對這些事件、故事必須信以為真。<ref name = "mythnature"/> <ref name = "mythfolk"/> <ref name = "eliademyth">Eliade, ''Myths, Dreams and Mysteries'', p. 23</ref> <ref>Pettazzoni, p. 102</ref>傳承者自己也知道是虛構的故事被稱為“寓言”。<ref>Eliade, ''Myth and Reality'', p. 10-11; Pettazzoni, p. 99-101</ref>
* 必須是遠古族群的人們集體創造並且流傳下來，如果是個人創造，並且沒有透過傳承而且群眾的成員對其創造的參與，這故事再怎麼神奇均不屬於神話。<ref name = "mythnature"/>

和神话相近的概念有[[传说|传说]]和[[民间故事|民间故事]]。这三个词指不同类型的传统故事。<ref>Bascom, p. 7</ref>不同于神话，民间故事可以发生在任何时代、任何地点；传承民间故事的人也不把它们当成真实、神圣的。<ref name = "mythnature"/>传说则一般被当做是真实的，但是传说一般发生在更近的事件段内，传说中的世界和现在的世界几乎一样。<ref name = "mythnature"/>传说的主角一般是人类，神话的主角一般是超自然角色。<ref name = "mythnature"/>

定义神话、传说和民间故事这三个概念，只是为了把传统故事归类。<ref name = "mythlegendfolk">Bascom, p. 10</ref>在许多文化中，神话和传说之间不一定有明确界限。<ref>Kirk, ''Myth'', p. 22, 32; Kirk, "Defining", p. 55</ref>有些文化中只把传统故事分为两类：民间故事和神话传说。<ref>Bascom, p. 17</ref>甚至神话和民间故事也不能完全区分开，一个民族认为是[[虚构|虚构]]的故事（民间故事），可能被另一个民族当成史实（神话）。
<ref>Bascom, p. 13</ref> <ref name = "doty">Doty, p. 114</ref>事实上，当一个神话失去其[[宗教|宗教]]地位后，它更像是民间故事，例如以前[[欧洲|欧洲]]的[[巨人|巨人]]、[[仙女|仙女]]和人类英雄带有[[神性|神性]]，在[[基督教|基督教]]传播开后全被当成民间故事。<ref name = "mythfolk"/>

== 分類 ==
神話大致分為三類：[[創世神話|創世神話]]、[[神祇神話|神祇神話]]、[[英雄神話|英雄神話]]。其中以創世神話最為重要。

* 創世神話—講述人類原始時期，記載事務、制度起源的神話。又可粗分為[[世界起源|世界起源]]神話、[[人類起源|人類起源]]神話、[[文化|文化]]起源神話。
但不表示每一個[[民族|民族]]就一定會同時擁有上述幾種神話，有些民族可能講到人類起源，卻沒說到文化起源，但儘管如此，都仍可歸納一個共同點，就是講述「宇宙初開」演進到「秩序規則」。

* 神祇神話和英雄神話—就是講述神佛和英雄們的種種事蹟。

== 不同學術範圍對神話的理解 ==
[[File:Idun_and_the_Apples.jpg|288px]]
[[英文|英文]]中的神話（myth）一詞從[[希臘文|希臘文]]的mythos一詞而來，其原文含意包括「[[語詞|語詞]]」、「[[言說|言說]]」、「[[故事|故事]]」及「虛構故事/[[小說|小說]]」等；因其不可質疑的有效性而往往與希臘文中的「[[道|道]]/[[科學|科學]]」（logos）相對，在近代的學術研究中，神話往往不是「虛假故事」、「誤解」的同義詞，在不同的學術研究中，「神話」這術語有三種主要用法，即[[禮儀|禮儀]]/[[人類學|人類學]]用法、[[文學|文學]]用法及[[符號學|符號學]]用法：

* 禮儀/人類學用法認為神話是一種匿名創作的敘事，它提供了[[世界|世界]]為何是現在這樣及人們為何如此行事的解釋，它是將[[自然|自然]]變為[[文化|文化]]的一種重要的手段，是人類嘗試對[[自我身份|自我身份]]的一種解釋，故有其當中的文化意涵。

* 在文學理論中，神話是某種永恆的人類[[真理|真理]]的故事或[[象徵|象徵]]，這種永恆真理通常屬於[[道德|道德]]的或[[美學|美學]]的範疇，這種象徵或[[原型|原型]]具有[[跨文化|跨文化]]的意義。[[宗教|宗教]]與文學有極其相似的[[社會功能|社會功能]]，就是把這種永恆的真理的價值觀透過故事的形式傳遞。

* [[符號學|符號學]]對神話的理解有別以上兩種用法，它是指一系列連接而不相合的相關概念，某種文化中的成員靠著這種概念才能理解某些對人類重要的主題，神話在[[無意識|無意識]]與[[主體間性|主體間性]]的狀態運作，使文化[[自然化|自然化]]。

==起源==

===真实事件===
一种理论认为，神话是从真实[[历史|历史]]事件演变而来。<ref name = "bulfincheuhemerism">Bulfinch, p. 194</ref> <ref name = "honkoancienttheories">Honko, p. 45</ref>根据这一理论，古代讲故事的人反复述说一段历史，年深日久，故事中的人物被神格化。<ref name = "bulfincheuhemerism"/> <ref name = "honkoancienttheories"/>例如一些学者认为，风神[[埃俄罗斯|埃俄罗斯]]的神话就来自一个古代真实的国王，他教人民使用[[帆船|帆船]]，掌握用风的技巧，后来被流传成[[风神|风神]]。<ref name = "bulfincheuhemerism"/>[[宙斯|宙斯]]是一个英明的[[国王|国王]]，死后被葬在[[克里特|克里特]]岛。<ref>''Zeus Is Dead: Euhemerus and Crete'', S. Spyridakis, The Classical Journal, Vol. 63, No. 8, May, 1968, pp. 337-340.</ref>公元前5世纪的[[古希腊|古希腊]]作家[[希罗多德|希罗多德]]和[[普罗迪克斯|普罗迪克斯]]都支持这种主张。<ref name = "honkoancienttheories"/>希腊神话学家[[犹希迈罗斯|犹希迈罗斯]]（{{lang-grc|Εὐήμερος}} [''Euhēmeros'']，生于前4世纪）是持这一理论最有名的人，此类思想也被称为“犹希迈罗斯主义”。<ref name = "honkoancienttheories"/><ref>"Euhemerism", ''The Concise Oxford Dictionary of World Religions''</ref>

===寓言===
还有一些人认为神话来自[[托寓|托寓]]。例如把古希腊众神看做对自然现象的解读和[[比喻|比喻]]，[[阿波罗|阿波罗]]代表[[太阳|太阳]]，[[波塞頓|波塞頓]]代表[[水|水]]。<ref name = "honkoancienttheories"/>相似的理论认为，神话是[[哲学|哲学]]或[[灵性|灵性]]概念的载体：[[雅典娜|雅典娜]]代表睿智和公正，[[阿芙洛狄忒|阿芙洛狄忒]]代表欲望等。<ref name = "honkoancienttheories"/>19 世纪的[[梵语|梵语]]学者[[马克斯·缪勒|马克斯·缪勒]]支持神话托寓理论。他相信神话最初是对大自然的比喻性描述，后来逐渐被人按字面意思理解。例如，人们常用修辞手法描述大海“躁动不安”，后来神话中的[[海神|海神]]就被描述为暴躁易怒。<ref>Segal, p. 20</ref>

===拟人化===
一些学者认为神话是对无生命物体和力量的[[拟人|拟人]]化。例如古人崇拜[[火|火]]、[[空气|空气]]等自然力，后逐渐把它们想像成[[神明|神明]]。这一理论被称为“造神思维”（Mythopoeic thought）。<ref>Bulfinch, p. 195</ref>根据这一理论，古人常把没有生命的东西看做有主观[[意识|意识]]的人。<ref>Frankfort, p. 4</ref>因此他们把自然事件描述为神明人为造成的，这些神往往有人类的性格。这些故事集中起来就形成了神话。<ref>Frankfort, p. 15</ref>

===神话-仪式理论===
根据神话-[[仪式|仪式]]理论，所有存在的神话都和仪式密不可分。<ref>Segal, p. 61</ref>最极端的的说法是，所有神话都是早期人类对仪式的解释。<ref>Graf, p. 40</ref>这一理论最早被[[圣经|圣经]]学者威廉姆·史密斯提出。<ref>Meletinsky pp.19-20</ref>根据史密斯的理论，人们最初进行的仪式和神话没有关系。后来我们忘记了最初举行仪式的原因，就尝试编一些故事来解释仪式，并声称举行仪式是为了纪念神话中描述的事件。<ref>Segal, p. 63</ref>[[人类学|人类学]]家[[詹姆斯·弗雷泽|詹姆斯·弗雷泽]]也持有类似理论。他认为原始人最早信仰[[魔法|魔法]]，后来人们对魔法失去信心，就发明了神话，并且说他们之前举行的魔法仪式为了安抚众神而进行的[[宗教|宗教]]仪式。<ref name="Frazer, p. 711">Frazer, p. 711</ref>

==目的==
一些[[宗教历史学|宗教历史学]]家认为，神话最重要的职能之一是建立行为模式<ref name = "honkomythfunction">Honko, p. 51</ref>并提供宗教体验。<ref>Eliade, ''Myth and Reality'', p. 8</ref>通过互相复述神话，传统社会中的成员暂时把自己和当前的世界分离，让心灵返回到神话时代，从而使自己更接近神聖的世界。<ref name = "eliademyth"/><ref name = "honkomythfunction"/><ref>Eliade, ''Myth and Reality'', p. 19</ref> 

在某些情况下，社会在复述神话的时候会试图重现神话中的场景。例如，人们会重现神在创世时使用的治愈手段，希望以此治愈目前生病的人。<ref>Honko, p. 49</ref>[[罗兰·巴特|罗兰·巴特]]也认同这一观点。他认为现代文化也做着类似的事情。由于[[科学|科学]]无法定义人类[[道德|道德]]，人们仍然借由宗教体验来与一个更远古的时代建立联系。那些远古时代在人们心目中有更高的道德标准，和现今的科技时代完全不同。<ref>Roland Barthes, Mythologies</ref>

宗教学家 [[Joseph_Campbell|Joseph Campbell]] 定义神话有四个基本功能：神秘功能（使人感受宇宙的威严）、科学功能（解释世界运作方式）、社会学功能（支持和验证社会秩序）、教育功能（说明人类在各种条件下应该如何生活）。<ref>Campbell, p. 22-23</ref>

在一個文化中，神話通常扮演著四種功能<ref>Campbell, Joseph (1970), Myths, Dreams,and Religion, New York: E. P. Dutton</ref>：

1.形而上的功能：可以用來解釋存在的根源

2.宇宙論的功能：強調宇宙萬物都是某個大畫面的一部分

3.社會性的功能：透過神話可以促進成員對社會規範的遵守

4.心理的功能：提供一種個人行為的模範

== 比較神話學 ==
比較神話學是指將不同的神話進行比較以找出它們共同的主題和特點，或者同體系而分支的神話，例如[[蒙古国|蒙古神話]] 與 突厥神話，兩者的起源都是[[額爾古納昆|額爾古納昆]]，可能因為是遊牧民族互相交流或分離兩地產生的。
