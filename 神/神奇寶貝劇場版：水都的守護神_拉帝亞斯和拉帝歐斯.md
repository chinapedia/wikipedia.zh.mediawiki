《'''水都的守護神 拉帝亞斯和拉帝歐斯'''》（{{lang-ja|劇場版ポケットモンスター 水の都の護神 ラティアスとラティオス}}），是神奇寶貝動畫的第五部電影版、同時也是城都聯盟及無印系列的最後一部電影。標語為「心之水滴，呼喚傳說神奇寶貝」。

票房收入達26.7億[[日元|日元]]，在《[[Pokemon_the_movie_XY_光輪的超魔神_胡巴|光環的超魔神 胡帕]]》上映前是全系列電影票房中最低的一部，然而其在觀眾間的評價普遍偏正面，也是一部表現充滿特色的作品。

== 場景 ==
* 奧多馬雷：取材自[[意大利|意大利]]的水都[[威尼斯|威尼斯]]。

== 劇情簡介 ==
水都奧多馬雷，相傳在這裡有兩隻傳說神奇寶貝——[[拉帝歐斯|拉帝歐斯]]與[[拉帝亞斯|拉帝亞斯]]，秘密地看守著守護這座城市的寶石：心之水滴。小智一行人正好造訪這裡，參加完水上競速賽後，並從當地居民口中得知這項傳說。這時，聞名世界的怪盜姊妹出現了，她們為了奪取心之水滴，利用探測器，找出了變成人型的拉帝亞斯，想藉此查出心之水滴的下落，正好被小智撞見。怪盜姊妹想利用心之水滴的力量掌控整個水都奧多馬雷，小智該如何制止她們的計畫？

== 備註 ==
* 本片是寵物小精靈劇場版中，第一次有寵物小精靈死亡的一作。
* 片尾中親吻小智的是卡儂還是拉帝亞斯，官方並沒有說明。
* 片尾曲當中出現「[[神奇寶貝劇場版：夢幻之神奇寶貝_洛奇亞爆誕|洛奇亞爆誕]]」內出場的收藏家。
* 本部電影一直直到2006年才隨著「神奇寶貝樂園（Poké Park）」的宣傳活動被引進台灣，並且未在電影院上映。
