'''神宮前'''（{{jpn|j=神宮前|hg=じんぐうまえ|rm=Jingūmae|f=y}}）是[[東京都|東京都]][[澀谷區|澀谷區]]的一個町。有一丁目至六丁目。2013年9月30日的人口有12,338人<ref>[http://www.city.shibuya.tokyo.jp/data/statics/tokei/02jinko/h25r01.xls 渋谷区／住民基本台帳・外国人登録による人口] {{webarchive|url=https://web.archive.org/web/20131013081847/http://www.city.shibuya.tokyo.jp/data/statics/tokei/02jinko/h25r01.xls |date=2013-10-13 }}</ref>。

== 歷史 ==
{{see also|原宿#歷史}}
{{see also|表参道 (原宿)#歷史}}
* 1889年（[[明治|明治]]22年）5月1日，[[町村制|町村制]]施行，千駄谷村、原宿村及穩田村合併，形成[[東京府|東京府]][[南豐島郡|南豐島郡]][[千駄谷町|千駄谷村]]。
* 1896年（明治29年）4月1日，南豐島郡和[[東多摩郡|東多摩郡]]合併，形成[[豐多摩郡|豐多摩郡]]。
* 1907年（明治40年）4月1日，千駄谷村施行町制，形成[[千駄谷町|千駄谷町]]。
* 1932年（[[昭和|昭和]]7年）10月1日，豐多摩郡千駄谷町編入[[東京市|東京市]]，成為[[澀谷區|澀谷區]]的一部份。
* 1943年（昭和18年）7月1日，[[東京都|東京都]]制施行。東京市澀谷區變為東京都澀谷區。
* 1965年（昭和40年）3月1日，当地區實施[[住居表示|住居表示]]。原先的原宿一至三丁目為中心的區域改町名作神宮前一丁目至神宮前六丁目<ref>同年6月15日、自治省告示第91号「住居表示が実施された件」</ref>

== 概要 ==
町內主要是[[商業用地|商業用地]]和[[高級住宅街|高級住宅街]]。從西北部順時針依次為一丁目至六丁目。

=== 神宮前一丁目 ===

東部是[[明治通_(東京都)|明治通]]、神宮前二丁目、神宮前三丁目、神宮前四丁目。西部是JR[[山手線|山手線]]和[[代代木公園|代代木公園]]。南部是表参道、神宮前六丁目。北部與千駄谷三丁目相接。

; 主要設施
* [[原宿站|原宿站]]
* [[明治神宮前站|明治神宮前站]]（[[東京地下鐵千代田線|千代田線]]）
* [[澀谷區立中央圖書館|澀谷區立中央圖書館]]
* [[澀谷區立原宿外苑中学校|澀谷區立原宿外苑中学校]]
* 千駄谷區民會館
* [[東鄉神社_(澀谷區)|東鄉神社]]（東鄉記念館）
* 太田紀念美術館
* [[SECOM|SECOM]]本社大廈
* [[ラフォーレ原宿|ラフォーレ原宿]]
* [[警視廳|警視廳]][[原宿警察署|原宿警察署]]
* [[八千代銀行|八千代銀行]]

=== 神宮前二丁目 ===
[[File:Street_in_Sendagaya.JPG|thumb]]二丁目（左手）]]

東部與[[明治神宮外苑|明治神宮外苑]]相接。西部是明治通和神宮前一丁目。南部是神宮前三丁目。北部與千駄谷二丁目相接。
; 主要設施
* [[土耳其共和国|土耳其共和国]][[大使館|大使館]]
* [[愛沙尼亞共和国|愛沙尼亞共和国]]大使館
* [[東京都立青山高等學校|東京都立青山高等學校]]
* [[國學院高等學校|國學院高等學校]]
* 海外子女教育研究所
* 澀谷神宮前[[郵便局|郵便局]]

=== 神宮前三丁目 ===
[[File:Watarium.jpg|thumb]]
東部與外苑西通、[[北青山|北青山]]二丁目相接。西部與明治通、神宮前一丁目相接。南部與神宮前四丁目相接。北部與神宮前二丁目相接。
; 主要設施
* 澀谷區立心身障害者福祉中心
* [[亘理美術館|亘理美術館]]
* [[妙圓寺_(澀谷區)|妙圓寺]]
* [[長安寺_(澀谷區)|長安寺]]
* [[原宿團地|原宿團地]]
* 原宿[[變電所|變電所]]
* [[美好年代美容專門學校|美好年代美容專門學校]]

=== 神宮前四丁目 ===
[[File:Omote-sando03s3200.jpg|thumb]]
東與北青山三丁目相接。西部是明治通及神宮前一丁目。南部是表參道、神宮前五丁目。北部是神宮前三丁目。
; 主要設施
* 澀谷區立神宮前小學校
* [[表參道之丘|表參道之丘]]
* [[東急廣場|東急廣場]]表參道原宿店
* 表參道廣場
* [[西武信用金庫|西武信用金庫]]
* 原宿警察署神宮前[[交番|交番]]

=== 神宮前五丁目 ===

東部與北青山三丁目、[[青山通|青山通]]及澀谷四丁目相接。西部與神宮前六丁目相接。南部與澀谷一丁目相接。北部與表参道、神宮前四丁目相接。
; 主要設施
* [[國際連合大學|國際連合大學]]本部
* [[兒童城|兒童城]]
* [[青山劇場|青山劇場]]
* 澀谷青山通郵便局
* 東京都住宅供給會社
* [[天理教|天理教]]東中央大教會
* 東京第一科學者基督教會

=== 神宮前六丁目 ===
[[File:Miyashita_Park_Shibuya_Tokyo.jpg|thumb]]]]
東部與神宮前五丁目相接。西部與JR山手線和[[國立代代木競技場|國立代代木競技場]]、神南一丁目相接。南部是[[宮下公園|宮下公園]]。北部與表参道、神宮前一丁目相接。
; 主要設施
* [[麒麟啤酒|麒麟啤酒]]原宿本社大廈
* 明治神宮前站（[[東京地下鐵副都心線|副都心線]]）
* 神宮前六郵便局
* 澀谷區神宮前隠田區民會館
* [[喬納森_(日本飯店)|喬納森]]
* [[三菱東京UFJ銀行|三菱東京UFJ銀行]]原宿支店
* [[郵貯銀行|郵貯銀行]]原宿八角館内出張所
* 宮下公園
* [[Colombin|Colombin]]原宿本店沙龍
