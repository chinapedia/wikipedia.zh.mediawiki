'''神威蓝光'''（{{lang-en|Sunway BlueLight MPP}}）是[[中国|中国]][[国家并行计算机工程技术研究中心|国家并行计算机工程技术研究中心]]制造的一台[[超級计算机|超級计算机]]。部署於[[山東省|山東省]][[濟南市|濟南市]]的[[国家超级计算济南中心|国家超级计算济南中心]]。

== 系统性能 ==
运算能力峰值达到1.07016[[PFlops|PFlops]]，持续性能达到795.9[[TFlops|TFlops]]，Linpack效率为74.37%，总功耗为1074[[千瓦|KW]]<ref>[http://tech.163.com/digi/11/1031/07/7HM5JAVR001618J1.html 国产CPU掀热潮揭秘神威蓝光来龙去脉]</ref>。存储容量2PB，最高带宽69.6TB/s<ref>[http://news.ifeng.com/mil/2/detail_2012_10/20/18401801_0.shtml 14万3千枚CPU：探访中国首台自主超级计算机 (图)]</ref>。采用多种节能技术实现绿色指标741.06MFlops/W。

== 系统硬件 ==
神威蓝光超级计算机共有8704个[[CPU|CPU]]。这些CPU全部采用中国购买许可后自主设计生产的CPU，即[[申威|申威]]3代“[[申威1600|申威1600]]”处理器（ShenWei processor SW1600）。这是该超级计算机最大的突破。该处理器是16核64位处理器，每个核心都是[[DEC_Alpha|DEC Alpha处理器]]的现代化增强版，以最高频率1.1[[GHz|GHz]]运行时，16个核总共提供最大141[[GFlops|GFlops]]的双精度浮点处理能力。这使中国成为继美国和日本之后世界上第三个独立研发千万亿次超级计算机的国家。

神威蓝光采用[[水冷|水冷]]散热系统。
