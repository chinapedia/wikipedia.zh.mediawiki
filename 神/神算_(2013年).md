《'''神算'''》是[[公共電視文化事業基金會|公共電視文化事業基金會]]《公視學生劇展》一部作品，描述了一位[[通靈|通靈]]女孩的生活故事。<ref>{{Cite news|url=http://story.pts.org.tw/innovative/episode/53 |title=公共電視台 公視學生劇展 |date=2017-05-06 |newspaper=archive.is |accessdate=2017-05-06 |deadurl=bot: unknown |archiveurl=https://archive.is/20170506154701/http://story.pts.org.tw/innovative/episode/53 |archivedate=2017-05-06 }}</ref><ref>{{Cite news|url=https://www.inmovie.com.tw/review/3911 |title=神算 超能少女繁忙的一天 |author=鄭秉泓 |date=2015-04-20 |work= |newspaper=in微創 影像創作網 |accessdate=2017-05-06 |deadurl=bot: unknown |archiveurl=https://archive.is/20170316100250/https://www.inmovie.com.tw/review/3911 |archivedate=2017-03-16 }}</ref>

== 緣起 ==
陳和榆在看到一篇登載於[[臺灣蘋果日報|臺灣蘋果日報]]的[[面談|訪談]][[报告文学|報導]]<ref>[https://archive.is/20170316110545/http://www.appledaily.com.tw/appledaily/article/headline/20100715/32662173/ 人間異語：成功靠自己 跟神通沒關係] 臺灣蘋果日報 2010-07-15</ref>後，以其為基底開始發想關於[[靈媒|靈媒]]的故事[[情節|情節]]。後來，陳和榆訪問了前述報導中的受訪者[[劉柏君|劉柏君]]，並前往拜訪許多[[通靈|通靈]]人士與相關人物，深入研究[[宮廟|宮廟]][[文化|文化]]與其[[信徒|信徒]]之間的互動關係<ref>[https://theinitium.com/article/20170313-taiwan-The-Teenage-Psychic/ 從學生短片到HBO劇集，台灣「通靈少女」的越洋旅程] 呂苡榕 端傳媒 2017-03-13</ref>。

== 故事大綱 ==
謝君雅是個熱愛[[壘球|壘球]]的高中學生，她因天生可以感知[[鬼魂|鬼魂]]與[[神靈|神靈]]而與[[道場|道場]]產生緣分；在道場裡，她希望藉著[[修行|修行]]改善個人困擾，但是，她更常因[[通靈|通靈]]能力而遭受[[信仰者|信眾]]求問與[[崇拜|崇拜]]。更糟的是，她因為靈媒工作而無法專心練習壘球，也失去和[[同學|同學]]在放學後互動的機會。

== 製作發行 ==
2011年，陳和榆以[[研究|研究]]蒐集所得之[[資訊|資訊]]與個人發想[[編劇|編寫]]並[[導演|導演]]了一部30分鐘[[短片|短片]]《神算》；他將該短片投稿參加《公視學生劇展》，並在獲選後於2013年首度放映於[[公視主頻|公視主頻]]。
