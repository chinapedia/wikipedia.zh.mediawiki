《'''降世神通：最後的氣宗'''》（{{lang-en|'''''The Last Airbender'''''}}）是一部2010年[[美国|美国]]出品的一部史诗冒险[[奇幻电影|奇幻电影]]，于2010年7月1日在北美地区上映。本片改编自[[电视动画|电视动画]]《[[降世神通：最後的氣宗|降世神通：最後的氣宗]]》，由著名印度裔导演[[奈特·沙马兰|奈特·沙马兰]]执导。美國於2010年7月1日上映。

== 情節 ==
[[水|水]]族部落（Water Tribes）凱塔拉和她的哥哥蘇卡在極冰的河岸狩獵，追尋野獸足跡意外發掘[[冰河|冰河]]底下在發光，鑿冰後冰層龜裂冒出巨大冰球。冰球破裂出現名為安的小孩和一隻飛天野牛阿霸（Appa）受困當中。在不知情況下，安正是失落已久的神通－維持世界平衡的精神領袖。由於它的消失，世界四個種族中的烈火國（Fire Nation），向其他三族-大氣牧族（Air Nomads）、水族部落（Water Tribes）以及大地之國（Earth Kingdom）宣戰。

烈火國的王子朱克正找尋失落的神通，將他帶回給父親烈火國國王。朱克察覺光源與火國士兵們來到水族部落南方。他假定神通應有歲數，故下令帶走部落所有的長者。朱克發現安有著氣宗的[[刺青|刺青]]引起他的揣測，消失的神通正是大氣牧族的最後的氣宗。安願意主動交出，只要朱克同意放走部落的長者們。

在船上，朱克的大伯愛和向安測試，發現安確實是神通。通過測試證實後，安趁機使用[[滑翔翼|滑翔翼]]逃脫，與凱塔拉和蘇卡搭著阿霸會合。安和他的新朋友來到安曾經修行的大氣寺廟南方，才知道他離開且冰封已近整[[世紀|世紀]]，大氣牧族早已被烈火國滅族，包括他的氣宗僧侶師父。絕望之下，他進入了神通狀態，發現他自己身在心靈境界，有個聲音引導他在何方。

之後，他們飛到已被烈火國控制的大地國小鎮，凱塔拉因幫忙小男孩逃脫而遭到火兵們捕抓。安鼓動大地國的運土神通們群起反抗，別再受制於烈火國的威嚇之下，當地運土神通成功地驅趕烈火士兵。安因當年不願當神通，逃避之下使得他僅會截氣術，依照氣水土火的轉世[[輪迴|輪迴]]，下個要學的則為截水術。凱塔拉給予截水神通[[卷軸|卷軸]]，幫忙安學習截水術，且前往水族部落北方找尋截水神通大師，沿途也解放了許多大地國的鄉村。

過程中，安的學習不順獨自偷溜回到北方大氣寺廟，卻在神通廟裡遭到烈火國趙將軍的埋伏遭到逮捕。然而，一個蒙面客「藍魔」（Blue Spirit）潛入，協助安從牢獄中逃脫。最後發現蒙面客竟是朱克，趙將軍策劃殺掉王子，將王子的船引爆。倖存的朱克藉由愛和的幫助，隱藏身分藏匿在趙將軍正前往水族部落北方的船艦裡。趙將軍從圖書館-萬知堂得知水族部落的神靈弱點，趁神通前往之時，逮回神通順道攻打北方水族部落。

安一群抵達北方水族部落，受到水族人們的歡迎，蘇卡也和水族公主浩月（Yue）一見鍾情，截水神通大師帕古（Pakku）開始教導安截水術。很快地，趙將軍所帶領的烈火國船艦抵達，朱克趁此時從冰層下方潛入皇宮內尋找安。船艦發動第一波火球攻勢，安要求要在幽靜的地方冥想，去見神靈指示。冥想之時，朱克找到安，與凱塔拉水火對決大打出手，並且擊倒帶走安。安收到指示回神後，再與朱克氣火對抗，最後凱塔拉趕到將朱克冰封。

烈火國攻勢加劇，愛哈發現趙將軍溜到水族聖地，將海水與明月神靈中的明月神靈殺掉，導致截水神通們喪失所有截水的能力。水族公主表示她的生命是月靈所賦予，於是犧牲自己的命讓月靈重生。發怒的愛和與朱克流放趙將軍，趙將軍則被回復能力的截水神通們，用海水包覆溺死身亡。覺醒的安懂得截水術，運用海水升起一片巨大的[[海嘯|海嘯]]，成功地驅趕走烈火船艦。安完全地承認作為神通王的命運，與凱塔拉和蘇卡準備繼續旅程，前往大地之國找尋運土神通大師來教導安。趙將軍身亡和愛和的背叛消息傳回烈火國，國王同意他的女兒亞蘇拉（Azula）去追捕朱克，且預備在索辛彗星的到來展開攻擊。

== 製作 ==
* 電影原本英文片名為《Avatar: The Last Airbender》，與改編動畫影集同名。然而，[[詹姆斯·卡麥隆|詹姆斯·卡麥隆]]和[[20世紀福斯|20世紀福斯]]已經先註冊電影標題《Avatar（[[阿凡達|阿凡達]]）》 ，避免爭議故將「Avatar」刪去僅留下《The Last Airbender》作為片名。
* [[傑西·麥卡尼|傑西·麥卡尼]]原為烈火國朱克王子預選人員，因檔期衝突由《[[貧民百萬富翁|貧民百萬富翁]]》男主角[[戴夫·帕托|戴夫·帕托]]（Dev Patel）取代。
* 本片為導演[[奈特·沙馬蘭|奈特·沙馬蘭]]首部非原創改編電影作品。自1998年的《[[小鬼一籮筐|小鬼一籮筐]]》（''Wide Awake''）以來，首部分級為PG級的電影。
* [[奈特·沙馬蘭|奈特·沙馬蘭]]表示部分角色名稱「發音有誤」（mispronunciation），像是安、蘇卡及神通的念法等，其實是他希望「尊重來源素材」並採用亞式發音。另一重大更動，[[奈特·沙馬蘭|奈特·沙馬蘭]]從動畫系列改編自大螢幕中，限制了火宗的御火來源。只有高等的火宗神通才能從氣中控制火焰。一些橋段像是京島戰士（Kyoshi Warriors）和其領袖杉子（Suki，蘇卡的愛意對象）都在電影定剪版本刪去，導演認為這些橋段是支線故事，僅直接地保留電影的主要故事線。
* [[奈特·沙馬蘭|奈特·沙馬蘭]]第七次與電影作曲家[[詹姆斯·紐頓·霍華|詹姆斯·紐頓·霍華]]合作，也是當今[[好萊塢|好萊塢]]電影導演與音樂作曲合作關係第四長的夥伴。排在前頭分別為[[史蒂芬·史匹柏|史蒂芬·史匹柏]]和[[約翰·威廉斯|約翰·威廉斯]]超過36年，[[羅拔·湛米基斯|勞勃·辛密克斯]]和[[亞倫·席維斯崔|亞倫·席維斯崔]]（Alan Silvestri）有26年，[[提姆·波頓|提姆·波頓]]和[[丹尼·葉夫曼|丹尼·葉夫曼]]（Danny Elfman）有14年，屏除1994年《[[艾德·伍德_(電影)|-{zh-cn:艾德·伍德; zh-hk:艾活傳; zh-tw:艾德伍德;}-]]》不算的話則有25年。
公佈演員名單後，於電影公司的論壇受網民猛烈批評，其後公司關閉該電影的討論分頁了事。
