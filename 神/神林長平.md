'''神林長平'''（かんばやし ちょうへい、本名:高柳 清、1953年7月10日 - ）、[[日本|日本]][[作家|作家]]。神林長平曾8度獲得[[星雲賞|星雲賞]]<ref>{{cite web |url=http://www.sf-fan.gr.jp/awards/list.html |script-title=ja:星雲賞リスト |language=ja |accessdate=2009-05-10}}</ref>。

== 生平 ==
* 1953年 - 出生於[[新潟縣|新潟縣]][[新潟市|新潟市]]。
* 1972年 - [[長岡工業高等専門学校|長岡工業高等専門学校]]畢業。
* 1979年 - 「狐と踊れ」入選第5回[[早川・SF大獎|早川・SF大獎]]佳作<ref>{{nihongo|"Dance with Fox"|狐と踊れ|Kitsune to odore}} (1979).  In a collection of the same title (1981).<!-- {{cite book |script-title=ja:狐と踊れ |author=神林長平 |publisher=早川書房 |language=ja |year=1981 |isbn=4-15-030142-5}} --></ref>。
* 1983年 - 獲得第14回[[星雲賞|星雲賞]]日本短編部門：『言葉使い師』。
* 1984年 - 獲得第15回星雲賞日本長編部門：『敵は海賊・海賊版』、日本短編部門を『スーパー・フェニックス』。
* 1985年 - 獲得第16回星雲賞日本長編部門：『[[戰鬥妖精雪風|戰鬥妖精雪風]]』。
* 1987年 - 獲得第18回星雲賞日本長編部門：『プリズム』。
* 1995年 - 獲得第16回[[日本SF大賞|日本SF大賞]]：『言壺』<ref>{{nihongo|''Word Pot''|言壷|''Kototsubo''}} (1994)</ref>。
* 1998年 - 獲得第29回星雲賞日本長編部門：『敵は海賊・A級の敵』。
* 2000年 - 獲得第31回星雲賞日本長編部門：『グッドラック 戦闘妖精・雪風』。
* 2001年 - 就任[[日本科幻小說作家俱樂部|日本科幻小說作家俱樂部]]会長<ref>{{cite web |url=http://www.sfwj.or.jp/club.html |title=History of SFWJ |language=ja |accessdate=2009-05-10}}</ref>。
* 2013年 - 獲得第44回[[星雲賞|星雲賞]]日本短編部門：『いま集合的無意識を、』。
