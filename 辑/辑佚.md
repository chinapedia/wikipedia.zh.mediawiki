'''辑佚'''是对已经失传，但其内容等仍以引用的形式保存在其它存世[[文献|文献]]中的文献材料加以搜集整理，使得佚失的书籍文献恢复原貌的行为。<ref name="zsh">{{Cite book zh|author= 张舜徽 |title= 中国文献学 |publisher= 上海古籍出版社 |id= ISBN 7-5325-4127-4}}</ref>通过辑佚得到的文献，称为'''辑本'''或者'''辑佚本'''，研究辑佚的历史、方法、原则和其他相关问题的[[学科|学科]]，称为'''辑佚学'''。

辑佚的产生，与[[中国古代|中国古代]]文献及典籍频繁而大量的佚失此一情况密不可分；90%以上的[[先秦|先秦]][[两汉|两汉]]文献、98%以上的[[曹魏|魏]][[晉朝|晋]][[南北朝|南北朝]]文献、94%以上的[[隋朝|隋]][[唐朝|唐]]文献均已无法查考。<ref name="zsx">{{Cite book zh|author= 张三夕 |title= 中国古典文献学 |publisher= 华中师范大学出版社 |id= ISBN 7-5622-2650-4}}</ref>

但是，由于中国古代书籍有[[引用|摘引]]、抄辑其它文献的习惯，所以一些佚失的书籍中的只言片语得以通过引用的形式保存在其它文献中。<ref name="yyq">{{Cite book zh|author= 杨燕起、高国抗 |title= 中国历史文献学（修订本） |publisher= 北京图书馆出版社|id= ISBN 7-5013-2173-6}}</ref>辑佚就是將上述的只言片语加以搜集及整理，使佚失的书籍得以重见天日。它与[[校勘学|校勘学]]、[[训诂学|训诂学]]、[[考据学|考据学]]及[[注释学|注释学]]都有着密切的关系，是研究中国古代文化及历史的基本功之一。<ref name="wjj">{{Cite book zh|author= 王俊杰 |title= 中国古典文献学概论|publisher= 齐鲁书社|id= ISBN 7-5333-1721-1}}</ref>

==历史 ==
辑佚起源于何时，学术界有不同的说法。有些学者认为可以追溯到魏晋甚至更远古的时候，但是学术界一般认为有明确目的和理论指导的辑佚活动起于[[宋代|宋代]]。<ref name="psg">{{Cite book zh|author= 潘树广、黄镇伟、涂小马|title= 文献学纲要|publisher= 广西师范大学出版社|id= ISBN 7-5633-3061-5}}</ref>
[[File:Lu_Xun_was_wearing_a_sweater.jpg|thumb]]也曾从事辑佚工作，辑出《会稽郡故书杂集》、《古小说钩沉》等书。]]
[[南宋|南宋]]时期，[[王应麟|王应麟]]通过辑佚，整理出[[郑玄|郑玄]]的《周易注》、《尚书注》和《三家诗》辑本，<ref name="wf">{{Cite book zh|author= 吴枫|title= 中国古典文献学|publisher= 齐鲁书社|id= ISBN 7-5333-1474-3}}</ref>有学者认为这是有意识的辑佚的起源，并且认为王应麟是辑佚的开创者。这种说法得到了许多学者的认同。同时期的[[郑樵|郑樵]]在《[[通志|通志]]·校雠略》提出“书有名亡实不亡”的观点，被认为是在辑佚学发展初期所提出的指导性理论。但也有学者认为郑樵的这种说法有“言之过易”之嫌。<ref name="zsh" />

清代中期，在[[文字狱|文字狱]]的高压下，一批学者借考据以遁世，躲进小楼成一统，埋首旧纸堆。<ref name="wf" />在这段时期，[[乾嘉汉学|乾嘉汉学]]得到蓬勃发展，辑佚也成为了一门[[显学|显学]]。<ref name="psg" />这段时期的民间辑佚代表作有由[[马国翰|马国翰]]辑出的《[[玉函山房辑佚书|玉函山房辑佚书]]》、由[[黄奭|黄奭]]辑出的《[[汉学堂丛书|汉学堂丛书]]》、由[[严可均|严可均]]辑出的《[[全上古三代秦汉三国六朝文|全上古三代秦汉三国六朝文]]》等等。<ref name="jbq">{{Cite book zh|author= 蒋伯潜|title= 校雠目录学纂要|publisher= 北京大学出版社|id= ISBN 7-301-00265-3}}</ref>

另一个对清代的辑佚活动产生巨大影响的事件是乾隆年间《[[四库全书|四库全书]]》的修撰。乾隆三十八年，安徽[[学政|学政]][[朱筠_(清朝)|朱筠]]奏请从内库所藏的《[[永乐大典|永乐大典]]》中辑出佚书。在[[乾隆皇帝|乾隆皇帝]]的首肯下，《四库全书》的编撰们，从《永乐大典》中辑出经部书籍66种、史部书籍41种、子部书籍130种、集部书籍175种，合共375种，4926卷。今天《四库全书》中所有标有“永乐大典本”的书籍，都是通过辑佚的方法，从《永乐大典》中辑出的佚书。<ref name="zsh" />

近代以来，汉学学者也多有辑佚之作，如[[罗振玉|罗振玉]]曾根据[[敦煌文书|敦煌文书]]，辑出《鸣沙石室佚书》、《鸣沙石室遗书续编》等书，[[鲁迅|鲁迅]]曾辑出《[[会稽郡故书杂集|会稽郡故书杂集]]》、《[[古小说钩沉|古小说钩沉]]》等书，[[赵万里|赵万里]]曾辑出《[[大元一统志|大元一统志]]》、《[[析津志|析津志]]》等书。<ref name="yyq" />

== 类型 ==
辑佚的类型根据不同的分类依据，有不同的说法。

从所要辑佚的文献类型划分，辑佚可以分为辑佚书、辑佚文和辑佚诗三种。因为辑佚文和辑佚诗的数量极少，所以辑佚通常是指辑佚书。<ref name="wjj" />

从所要辑佚的文献存世情况划分，辑佚也可以分为三类。第一类是全书已经完全亡佚，通过辑佚，让失传的文献重现于世；第二类是全书基本存世，但有少量缺失，通过辑佚，把缺失的部分补全；第三类是全书已经完全亡佚，但是还有前人的辑本存世，通过辑佚，把前人辑本漏辑的部分补上，让辑本更加完善。后两类辑佚是广义的辑佚，它们的实质是[[补遗|补遗]]。<ref name="psg" />

== 方法 ==
辑佚的具体方法学术界有不同的说法，但是就大致的阶段而言，辑佚的全过程可分为准备、辑录、整理和综结四个阶段。<ref name="zsx" />准备阶段主要是指确定古籍是否佚失，以及确定辑佚的材料来源；辑录则是指将分散在不同书籍的佚文加以收集；整理主要解决佚文的编排次序，对佚文进行[[校勘|校勘]]、[[补缀|补缀]]和[[辨伪|辨伪]]。综结则主要进行确定编目和编制索引的工作。<ref name="psg" />

== 材料来源 ==
辑佚的材料来源很广，[[类书|类书]]、[[古注|古注]]、[[史书|史书]]、[[地志|地志]]、[[字书|字书]]、[[杂钞|杂钞]]、<ref name="zsx" />[[金石|金石]]、[[书目|书目]]、[[报刊|报刊]]，甚至[[日记|日记]]、[[书信|书信]]、[[档案|档案]]等等，<ref name="psg" />凡是有转引佚书的文献，都可以作为辑佚的材料来源。

但是，在具体的辑佚上，不同的书籍，佚文可能藏身的材料来源往往具有一定的规律性。比如上古时期的书籍，大多散见于[[诸子百家|诸子百家]]的著作和汉代的[[笺注|笺注]]里面，两汉三国时期的书籍大多散见于唐代的[[义疏|义疏]]里面。<ref name="zsh" />把握这些规律，能够更加高效地确定辑佚的材料来源。

== 意义 ==
通过辑佚，可以让已经佚失的古文献重现于世，有利于文献的保存和流通。比如[[二十四史|二十四史]]中的《[[旧五代史|旧五代史]]》在清代已经佚失，但是通过对《永乐大典》的辑佚，今天的我们才有幸能够看到了这部长达一百五十卷的恢宏巨著。同时，通过辑佚产生的辑本，也为学者提供的较为可靠的文字材料，节约了学者检索文献的时间。<ref name="wjj" />

通过辑佚活动的实践和对辑佚学的研究，可以推动[[校勘学|校勘学]]、[[训诂学|训诂学]]、[[考据学|考据学]]、[[注释学|注释学]]等相关学科的发展。学者在从事辑佚的过程中，也可以在这些方面得到充分的锻炼。<ref name="wjj" />

== 日本辑佚 ==
在日本，也利用辑佚的方法来研究和收集古代的资料。例如[[古风土记|古风土记]]，据称原有四十八[[日本令制国列表|國]]风土记，如今完整保存的只有《[[出雲國風土記|出雲國風土記]]》，而《[[播磨國風土記|播磨國風土記]]》、《[[肥前國風土記|肥前國風土記]]》、《[[常陸國風土記|常陸國風土記]]》、《[[豐後國風土記|豐後國風土記]]》等書則留有殘章；其余有約三十國风土记以佚文方式流传至今，但有人认为这部分佚文中可能有成文于[[奈良时代|奈良时代]]的部分。又如[[日本后纪|日本后纪]]，至15世紀初时尚有40卷，完整流传至今的仅有10卷，其余部分零散地以佚文方式存在。
