'''參議院'''（{{lang-es|Senado}}）是[[西班牙議會|西班牙議會]]的上議院。西班牙參議院現在總共有266名議員，上一次選舉在2011年11月20日舉行。現在西班牙參議院的最大黨派是[[人民黨_(西班牙)|人民黨]]。
== 歷史 ==
當前參議院為[[1978年西班牙憲法|1978年西班牙憲法]]頒布時成立。

== 議會組織 ==
參議員依照黨派組成黨團。參與人數少於10人的政黨組成混合黨團。
== 選舉制度 ==
=== 直選 ===
目前266席次中有208席由直選選出。

原則上每個省分都會有4個席位，[[休達|休達]]和[[梅利利亞|梅利利亞]]則有兩個席位。而島嶼部分會有特殊分配：[[巴利阿里群島|巴利阿里群島]]（Baleares）和[[加那利群島|加那利群島]]（Canarias）較大的島嶼-  馬略卡島，大加那利島和特內里費島 - 三個席位，而較小的7個島（ 較小的島嶼 - 梅諾卡島，伊比沙島 -福門特拉島，富埃特文圖拉島，戈梅拉島，耶羅島，蘭薩羅特島和拉帕爾馬島 ）會獲得一個席位。[[休達|休達]]和[[梅利利亞|梅利利亞]]共有兩名。這種分配在很大程度上有利於小省份：擁有約600萬人口的馬德里和擁有10萬居民的索里亞都是由四名參議員代表。

=== 區域任命 ===
目前266席次中有58席由區域議會任命。

== 地位 ==
在西班牙，只有參議院能管理各地方政府主席（《[[西班牙宪法|西班牙宪法]]》第155條），也只有參議院能夠暫停地方議會的權力（《[[地方管理法|地方管理法]]》第61條）<ref name="boeLBRL">{{es icon}}[[Boletín_Oficial_del_Estado|Spanish Official Gazette]]: [http://www.boe.es/aeboe/consultas/bases_datos/doc.php?coleccion=iberlex&id=1985/05392 Local Regime Framework Act (Law 7/1985)]{{Dead link|date=2018年9月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

2017年10月27日，參議院通過議案動用[[1978年西班牙憲法|《西班牙憲法》第155條]]凍結[[加泰隆尼亞|加泰隆尼亞]]自治權，並強行解散自治政府以及取消加泰隆尼亞獨派政治人物職務。當時214票贊成，47票反對，1票棄權。
