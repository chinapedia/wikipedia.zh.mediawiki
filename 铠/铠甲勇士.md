《'''鎧甲勇士'''》是一部[[中國大陸|中國大陸]]攝製的[[特攝劇|特攝劇]]。該劇結合中國傳統的[[五行|五行]]創造出了「金、木、水、火、土」五俠，而五俠的真人[[姓名|姓名]]則用了「東、西、南、北、中」五個[[方向|方向]]命名。據悉，此片投資總額達2000萬元[[人民幣|人民幣]]。2009年3月28日哈哈少儿频道首播。

《鎧甲勇士》是繼《[[變身戰士阿龍|變身戰士阿龍]]》和《[[金甲戰士|金甲戰士]]》後的第三套通過[[廣電總局|廣電總局]]審查的中國本土特攝，但《變身戰士阿龍》中途被廣電總局下令禁播。

由於《鎧甲勇士》的皮套跟[[日本|日本]]的[[假面騎士系列|假面騎士系列]]設計相似，故此有人估計此片由製作假面騎士系列及[[超級戰隊系列|超級戰隊系列]]的[[東映|東映]]幕後協助，但製作公司[[奧飛動漫文化|奧飛動漫文化]]及東映雙方都未就此傳言作任何評論。而僅在第一部片尾的製作名單中，合作署名出現[[上海圓谷策劃公司|上海圓谷策劃公司]]。不過，該片的部份皮套演員均出自是前日本動作組織（[[JAC|JAC]]）的成員[[柴原孝典|柴原孝典]]開設的[[Wild_Stunt|Wild Stunt]]公司，而且柴原孝典本人亦擔任動作指導，故打鬥場面可媲美假面騎士系列。

== 故事簡介 ==
五千年前，[[宇宙|宇宙]]的最大黑暗勢力'''暗影大帝'''對[[太陽系|太陽系]]有旺盛生命體的[[地球|地球]]發動侵略，並派出了'''暗影五行魔'''率領了50個'''魔靈石'''（魔靈石能與地球上的生命結合為'''暗影異能獸'''）來到了地球。暗影五行魔當時選中了有高度文明的古中國作為首要毀滅的對象，太陽系的守護者'''太陽星'''為了預防地球被暗影勢力所吞噬毀滅，於是向[[古代中國|古代中國]]發射了五顆'''光影石'''。這五顆光影石分別能與形成地球萬物的五行元素結合，形成五套能在光和影中穿梭戰鬥的鎧甲─'''光影鎧甲'''；而能召喚這些光影鎧甲的人，就是當初這五顆光影石分別落地的五個村落中的人們。不管時代的變遷，只要這些村人[[遺傳|遺傳]]的[[血液|血液]]仍在，他們就能透過光影石的力量，召喚光影鎧甲來執行與暗影集團的神聖戰鬥！

==概要==
鎧甲勇士是由流淌著光影村血統的召喚人與由光影石構成的光影鎧甲合體而成的戰士的總稱，除了帝皇俠以外共有五位。鎧甲會根據召喚者的體質而發揮出不同的威力，所以召喚者越強，鎧甲的威力便越強。『<u>每套光影鎧甲都有著一定戰鬥時限，當光影鎧甲能量不足時，胸口的信號燈便會閃爍起來。</u>（存疑，无印象电视剧中有铠甲勇士有战斗时限的介绍，无任何因能量不足而导致胸前的灯闪烁，一般情况下重伤直接强制解除变身。）』
;鎧甲召喚器
:變身成鎧甲勇士的道具，召喚者可借由召喚器發射光影石之光能量，接受鎧甲完成合體，成為鎧甲勇士<ref>但是，鎧甲召喚器並不是召喚鎧甲的必要條件。只要有光影石在召喚人的身旁，召喚人便可召喚鎧甲。</ref>。具有通訊和傳信機能，在鎧甲召喚人的日常生活中經常代替[[手機|手機]]使用。由一隊與二隊曾同時佩戴召喚器看來，每種鎧甲召喚器不止一個。
;乾坤腰帶
:架於腰部的裝備，亦是整套光影鎧甲最主要的部份。左右兩方各有一個轉輪，左方為發動必殺技用，右方則為召喚武器用。召喚武器時，從腰帶中心中心的太極陣則用於把封印帖轉移到光影石的所在地。
;禪定印
:鎧甲勇士共通的特殊技能，發動方法為將兩手於腰前上下合掌。能讓指定範圍的時間停止流動。
;光影駒
:五位鎧甲勇士的專用座騎，以龍為藍本的三輪型機車，平常透過轉動乾坤腰帶左方的轉輪召喚，具飛行能力。駕駛方法跟[[摩托車|摩托車]]無異，但加速的方法則是把手柄向內拉。武器為從眼部發出的光彈和口部發出的炮擊。

== 用語 ==
'''組織'''
*'''ERP'''
:當初只是在研究並分析地球上所出現的特殊異能量、不明[[放射線|放射線]]以及怪異污染物的研究單位。在一次次的深入研究過程中，意外地得到了兩顆光影石：火影石與木影石。接著並發現了古代傳說中之光影鎧甲的秘密。加魯博士立刻決定，將ERP研究室改造成鎧甲勇士的基地，並陸續開發了能聚集和傳送的光影石特殊能量的光影能量室。ERP研究室專屬戰鬥衛星。先進的能量傳輸系統以及進入召喚者的深層潛意識中，並結合召喚者的腦波和磁場，從而開發召喚者的特殊能量的虛擬戰鬥室系統。
:基地入口在一間經常表示「休息中」的漢堡店─幸福堡裡的洗手間。

*'''影界'''
:利用魔靈石的異能獸破壞地球，背後由暗影大帝和暗影五行魔所操縱。基地位置位於一座長年被黑雲遮蓋的山上的山洞，但沒有任何守衛把守。

'''名詞'''
*'''光影石'''
:由太陽系的守護者太陽星，為了預防地球被暗影勢力所吞噬毀滅，於是向古中國發射的能量，能與形成地球萬物的五行元素結合，成為光影鎧甲。
:共有五顆，分別為：'''火影石、木影石、水影石、金影石、土影石'''。

*'''魔靈石'''
:來自黑暗宇宙的魔靈石，是一種異能量，能輕易地附身於地球上的單細胞或多細胞生物而成為'''異能獸'''。

*'''黑域'''
:暗影五行魔的所在地，會使光影鎧甲的能量急劇下降。

*'''光影帖／封魔盒'''
:把魔靈石封印起來的裝備，書本形的盒子。當魔靈石完全被封印時，便會在鎧甲勇士面前出現。

*'''RT血型'''
:光影村後人特有的血型，亦是召喚鎧甲的主要條件。

== 登場人物 ==
===鎧甲勇士召喚人===
;铠甲勇士一队
:;炘南
::鋼琴家一名，自小父母雙失，居住於蕙姨的「幸福餃子館」中。雖然是鎧甲勇士，但卻以自己的鋼琴比賽行先。經常因練習及比賽而不參與戰鬥，惹來北淼的批評，亦曾經有一段時間沒有參與戰鬥。
::夢想為取得全國大賽冠軍，完成母親的願望。性格深思熟慮，但容易受壓力影響。

:;東杉
::科學工作者一名，性格優柔寡斷。在堂弟啟東的點化下，決心繼承木之鎧甲。
::一直對合作問題意見不大，只想一心一意戰鬥。同時對美真有興趣，希望跟她交往。

:;北淼
::孤兒一名，長大後成為富家子弟。對影界的人極為反感，同時對五位戰士由誰當領導諸多意見。
::生性猜忌以及偏激，為了取得領導權不惜打傷炘南，同時亦排斥最後加入的西釗。但後來因張健的背叛，以及得知西釗的過去後，與他冰釋前嫌。
::一直為小時候拯救不了小雪(冰兒)而後悔不已，而這亦被奪得冰兒肉體的惡水利用。
::在失去變身能力期間一直協助美真調查陽光孤兒院的事件,喜欢着冰儿

:;西釗
::孤兒一名，及後異能獸襲擊弧兒院，被界王帶走。長大後便為影界效力，與鎧甲勇士為敵，但本性不壞。後來被坤中感化，更帶領其餘戰士直搗影界巢穴。

::五人中最先回復變身能力的一人。喜欢冰儿

:;坤中
::五位戰士中年紀最小，酷愛籃球。在美真發現土影石後，檢查得知是適合召喚鎧甲的人。於打籃球時認識西釗，並勸他退出影界加入鎧甲勇士行列。
::經驗尚淺，性格中庸，偶爾跟西釗的問題而跟北淼鬧意見，亦曾因一時大意而被界王控制。

;吳啟東
:東杉的堂弟，就讀大學歷史系三年級。木村的後人，亦是促成東杉成為風鷹俠的人。

;敏慈
:於「幸福餃子館」工作，原為富家女一名。經常為炘南做衣服給他比賽，亦暗戀著他。是促成炘南成為孤兒的那場車禍的引發者兼目擊者。

;蕙姨
:「幸福餃子館」老闆。認識炘南的媽媽，後者去世後便收留了炘南。以“幸福餃子”作為招牌菜。

;小蘭
:蕙姨的女兒，對炘南這種氣質的男子十分仰慕。夢想成為炘南的新娘。

;小嵩
:初期是「八天刊」記者，致力報導鎧甲勇士的消息。更目睹東杉等人變身過程，更打算公諸於世，不過在美真的請求下把這念頭打消。
:後來辭去記者的工作，繼而轉到外國修讀時裝設計，還練得一身好廚藝。

;上班族/胖主婦/重考生/模型迷/漫畫男/追星族
:小嵩在「宅人聊天室」的好友，曾提供他不少情報。

;冰兒(小雪)
:孤兒一名，小時候被界王帶走，北淼的義妹。初時想跟鎧甲勇士和好，但卻聽到北淼說對影界的人的不信任立場後。
:接到從丑將手上得知她小時候跟北淼的關係，並離開界王。後期被丑將(惡木)利誘成為惡水的的軀體，由于最后恶水被封印，因此她成功摆脱了恶水的控制而恢复了自我。


;陽光孤兒院院長
:炘南五人以及冰兒曾待過的孤兒院院長。知道封印異能獸的第二種方法。

===暗影集團===
;黑帝
:原本只是一名潦倒的拾荒者，因意外的拾得光影帖，並解放了暗影五行魔，更許下變成富豪的願望，繼而成為影界的支配者。
:最後因打開了五行魔給他的錦囊而被五行魔佔領了身體，成為'''黑魔獸'''，最后被打败。

;暗影五行魔/魔靈護法(惡火，惡水，惡木，惡金，惡土)
:暗影大帝的手下，其屬性跟五位鎧甲勇士一樣以五行為主，本身並無肉體。因此，會以“實現”宿主所許的惡願作為利誘從而取得其肉體。
:最後被另一梯隊的鎧甲勇士全部封印。

;四大遠古異能獸(饕餮，混沌，窮奇，檮杌)
:當五行魔全被封印後，黑帝派出的異能獸。除了能自由變回封印帖外，還能抵擋鎧甲勇士的必殺技。
:後被帝皇俠全部封印。

;影霸
:第一代影界領頭，實為暗影五行魔的傀儡。本是土村的後人，所以也能變身成地虎俠，但卻誤入歧途，踏上不歸路。
:最後在跟異能獸戰鬥後，把土影石交給風鷹俠/東杉後力歇而亡。

;丑將
:聽命於影霸及界王，主要作前線行動。在中期以冰兒跟北淼的關係來威脅冰兒，但反被黑犀俠打至重傷。
:其後為得到強大的力量而把肉體出賣於暗影五行魔，成為惡木的軀體，最后被封印。

;界王
:本是科學家一名，因一次實驗意外後使左眼及右手殘廢，其後生活潦倒。被丑將帶走改造成為了第二代統領。
:在故事時間點的10年前指揮異能獸攻擊「陽光福利院」，並帶走了西釗，小雪(冰兒)和金影石。最後把自己作為異能獸的肉體，但卻被雪獒俠等人打敗。
