'''姬路市'''（{{jpn|j=姫路市|hg=ひめじし|rm=Himeji shi|f=y}}）是位於日本[[近畿地方|近畿地方]]西部、[[兵庫縣|兵庫縣]]西南部（[[播磨地方|播磨地方]]）的都市，也是播磨地方的經濟、文化及教育中心。人口超過53萬，是兵庫縣內次于[[神戶市|神戶市]]的第二大城市，同時也是日本首批[[中核市|中核市]]<ref name="Chukakushi">{{cite web|url=http://www.city.himeji.lg.jp/info/profile/kernel.html　|script-title=ja:地方分権時代の旗手・中核市|accessdate=2017-04-06|publisher={{lang|ja|姫路市}}|language=ja}}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。姬路市和周圍的城市共同組成了人口超過70萬人的{{link-ja|姬路都市圏|姫路都市圏}}<ref>{{cite web|url=http://www.city.himeji.lg.jp/var/rev0/0091/9508/2016329184247.pdf　|script-title=ja:播磨圏域連携中枢都市圏ビジョン|accessdate=2017-04-14|publisher={{lang|ja|姫路市}}|language=ja}}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

姬路市有著悠久的歷史，擁有已被列為[[世界文化遺產|世界文化遺產]]的[[姬路城|姬路城]]和[[圓教寺|圓教寺]]等眾多文化財產。「姬路」這一地名來自於姬路城所在的姬山的古名「{{lang|ja|日女路（ひめじ）の丘}}」。這個地名起源於這一帶在古代養蠶業繁盛，而在古代[[蠶|蠶]]的讀法是{{lang|ja|ヒメコ}}。在《[[播磨國風土記|播磨國風土記]]》中，也有名為「{{lang|ja|日女道丘（ひめじおか）}}」的地名出現<ref>{{cite web|url=http://www.city.himeji.lg.jp/guide/castle/about/history.html　|script-title=ja:姫路城の歴史|accessdate=2017-04-14|publisher={{lang|ja|姫路市}}|language=ja}}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

==歷史==
姬路市周邊地區於[[繩文時代|繩文時代]]已有人類居住。自7世紀[[大化改新|大化改新]]之後，因[[播磨國|播磨國]]的國府置於這裡，姬路成為播磨地方的中心<ref name="History">{{cite web |url=http://www.city.himeji.lg.jp/info/profile/history.html　 |script-title=ja:姫路の歴史 |accessdate=2017-04-18 |publisher={{lang|ja|姫路市}} |language=ja }}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。8世紀中期時，[[聖武天皇|聖武天皇]]在姬路建造[[國分寺|國分寺]]，同一時期還建造了[[广峰神社|广峰神社]]、书写山[[圆教寺|圆教寺]]等重要宗教建築<ref name="History"></ref>。[[室町時代|室町時代]]時期，播磨國豪族[[赤松氏|赤松氏]]在姬山修建要塞。在[[安土桃山時代|安土桃山時代]]，[[豐臣秀吉|豐臣秀吉]]於1581年（[[天正_(日本)|天正]]9年）修建了具[[天守|天守]]的姬路城，此為姬路的城下町歷史之始<ref name="Kinki">{{cite book |author={{lang|ja|平岡昭利・野間晴雄}} |script-title=ja:《地図で読む百年　近畿Ⅱ》 |year=2006 |publisher={{lang|ja|古今書院}} |location=東京都 |isbn=978-4772230544 |language=ja}}</ref>{{rp|81}}。

[[File:Old_painting_of_Himeji_castle.jpg|thumb]]

[[關原之戰|關原之戰]]之後，德川家康的女婿[[池田輝政|池田輝政]]被移封至此，成立了[[姬路藩|姬路藩]]，現在的[[姬路城|姬路城]]就是興建於池田輝政統治時期<ref name="History"></ref>。1617年由於繼承的[[池田光政|池田光政]]過於年幼，池田氏被轉封至[[鳥取藩|鳥取藩]]，由[[德川家康|德川家康]]重臣[[本多忠勝|本多忠勝]]之子[[本多忠政|本多忠政]]接替姬路藩；在池田輝政和本多忠政統治時期，除了姬路城竣工之外，姬路的城下町格局亦在這一時期基本完成，確立了現在姬路市中心地區的城市格局<ref name="Kinki"></ref>。一直到[[江戶時代|江戶時代]]結束，姬路藩的領主經過多達九次的變動，但此後皆由親藩大名或譜代大名治理。

[[明治維新|明治維新]]之後，日本政府在1871年實施[[廢藩置縣|廢藩置縣]]，姬路藩改設為姬路縣（後改名為[[飾磨縣|飾磨縣]]），姬路也跟著成為姬路縣的縣廳所在地，直到1876年8月，飾磨縣被併入[[兵庫縣|兵庫縣]]<ref>{{cite web |url=http://web.pref.hyogo.lg.jp/kk32/pa13_000000013.html　 |script-title=ja:県域の変遷 |accessdate=2017-04-18 |publisher={{lang|ja|兵庫県}} |language=ja }}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，姬路的地理位置雖然位於新兵庫縣的中部，但因在江戶時代是譜代大名領地，在新政府中發言力較低，因此兵庫縣縣廳所在地拱手讓給[[神戶市|神戶市]]<ref name="Kinki"></ref>{{rp|81}}。

1889年，日本實施[[市制_(日本制度)|市制]]，姬路市成為日本第一批的市之一<ref name="Chishi">{{cite book |author={{lang|ja|石川義孝・金田章裕}}|script-title=ja:《日本の地誌8　近畿圏》 |year=2006 |publisher={{lang|ja|朝倉書店}} |location=東京都 |isbn=4-254-16768-7 |language=ja}}</ref>{{rp|270}}。近代姬路是一座軍事都市，1898年，[[大日本帝國陸軍|大日本帝國陸軍]][[第10師團|第10師團]]設置在姬路，標誌姬路獲得了「軍都」的地位<ref name="Kinki"></ref>{{rp|84}}。1888年，姬路到{{link-ja|兵庫車站|兵庫駅}}之間的鐵路開通，當地的交通條件大幅改善，亦加速了近代工業的發展<ref name="Kinki"></ref>{{rp|84}}。1923年神戶明石電鐵（現[[山陽電氣鐵道|山陽電鐵]]）和1936年姬津線（現[[姬新線|姬新線]]）的開通使得姬路成為鐵路交通樞紐<ref>{{cite web |url=https://www.westjr.co.jp/press/article/2016/09/page_9255.html　 |script-title=ja:「みんな大好き姫新線80周年記念イベント」などの実施について |accessdate=2017-04-18 |publisher={{lang|ja|JR西日本}} |language=ja }}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。鐵路的開通使得姬路的都市發展主軸從東西方向的西國街道轉為南北方向的御幸通，可見鐵路對姬路城市發展的巨大作用<ref name="Kinki"></ref>{{rp|84}}。

在[[第二次世界大戰|第二次世界大戰]]期間，因姬路市的「軍都」地位，加上姬路是重要的鐵路樞紐和工業都市，使得姬路成為美國{{link-en|第21轟炸機司令部|XXI Bomber Command}}的轟炸目標。姬路在二戰末期的1945年6月22日和7月3日兩次遭到大規模空襲，其中又以7月3日空襲的規模最大。美軍在這場空襲中擲下了767噸炸藥，摧毀了姬路市63.3%的建築區<ref>{{cite web |url=https://ja.scribd.com/document/61702152/21st-Bomber-Command-Tactical-Mission-Report-247-250-Ocr　|script-title=en:21st Bomber Command Tactical Mission Report 247, 250, Ocr |accessdate=2017-04-18 |publisher={{lang|en|Scribd}} |language=en}}</ref>，約170人在空襲中遇難，總戶數的40%被燒毀<ref>{{cite web |url=http://www.soumu.go.jp/main_sosiki/daijinkanbou/sensai/situation/state/kinki_08.html　 |script-title=ja:姫路市における戦災の状況（兵庫県） |accessdate=2017-04-18 |publisher={{lang|ja|総務省}} |language=ja }}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。但幸運的是姬路城在空襲中逃過一劫<ref>{{cite web |url=http://himejicastle-tour.jp/?page_id=850　|script-title=ja:戦争の傷跡 |accessdate=2017-04-18 |publisher={{lang|ja|姫路城完全観光案内所}} |language=ja}}</ref>。

戰後姬路市在1930年代至1950年代期間合併了附近的多個行政區劃，使得轄區面積倍增，人口也從約11萬人增加到17萬人<ref name="Kinki"></ref>{{rp|85}}。1948年，姬路市實施戰災復興事業，其代表工程之一是在姬路城和姬路車站之間修建了寬50米的道路，都市再開發迅速進展，姬路市的城市面貌在戰後發生巨大變化<ref name="Kinki"></ref>{{rp|85}}。1964年，在經過八年的施工之後，姬路城的昭和大修理工程竣工<ref>{{cite web |url=http://www.city.himeji.lg.jp/jyokakuken/repair/　 |script-title=ja:姫路城・昭和の大修理写真 |accessdate=2017-04-18 |publisher={{lang|ja|姫路市}} |language=ja }}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。兩年之後，姬路市政府為紀念修理工程舉辦{{link-ja|姬路大博覽會|姫路大博覧会}}，象徵姬路市進入新的時代<ref>{{cite web |url=http://www.kk-kanzaki.jp/castle/　 |script-title=ja:姫路城とのあゆみ |accessdate=2017-04-18 |publisher={{lang|ja|神崎組}} |language=ja }}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。1996年，姬路市升格為中核市<ref name="Chukakushi"></ref>。2006年，周邊的{{link-ja|家島町|家島町}}、{{link-ja|夢前町|夢前町}}、{{link-ja|香寺町|香寺町}}、{{link-ja|安富町|安富町}}被併入姬路市，形成現在姬路市的範圍<ref>{{cite web |url=http://www.sichouyouran.jp/document/list/03.html　 |script-title=ja:市町合併の状況 |accessdate=2017-04-18 |publisher={{lang|ja|兵庫県市町村振興協会}} |language=ja }}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。2015年，姬路城的平成大修理工程竣工，使得前往姬路市觀光的遊客數大增，亦讓姬路城首次成為日本觀光客人數最多的城堡<ref>{{cite web |url=http://www.sankei.com/west/news/160510/wst1605100021-n1.html　 |script-title=ja:大修理後の姫路城…経済効果６１５億円ナリ！城郭入場者数も初の全国一に |accessdate=2017-04-18 |publisher={{lang|ja|産経WEST}} |language=ja }}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

== 地理 ==
[[File:Himeji_city_center_area_Aerial_photograph.1980.jpg|thumb]]
姬路市位於[[兵庫縣|兵庫縣]]西部[[播磨平原|播磨平原]]中西部，是播磨地方（舊飾磨縣）的中心城市。姬路市轄區內的主要河川包括位於中東部的{{tsl|ja|市川_(兵庫県)|市川 (兵庫縣)|市川}}、中部的{{tsl|ja|船場川}}、{{tsl|ja|野田川_(兵庫県)|野田川 (兵庫縣)|野田川}}（{{tsl|ja|外堀川}}、三左衛門堀）、中西部的{{tsl|ja|夢前川}}、{{tsl|ja|大津茂川}}、西部的{{tsl|ja|揖保川}}<ref>{{cite web |url=https://www.kkr.mlit.go.jp/himeji/torikumi/river/about/about_ibo.html　|script-title=ja:揖保川の概要 |accessdate=2017-04-14|publisher={{lang|ja|近畿地方整備局　姫路河川国道事務所}} |language=ja}}</ref>，這些河流都是往南匯入[[播磨灘|播磨灘]]。姬路市域內的主要山峰則有市中心的姬山、市中心北部的{{link-ja|廣峰山|広峰山}}和{{link-ja|增位山 (兵庫縣)|増位山 (兵庫県)|增位山}}、市區西北部的{{link-ja|書寫山|書写山}}、市區北部的{{link-ja|明神山 (兵庫県)|明神山 (兵庫縣)|明神山}}和[[雪彥山|雪彥山]]。其中市中心的姬山是姬路城的所在地，姬路城下町亦是圍繞姬山開始發展。[[山陽新幹線|山陽新幹線]]和[[山陽本線|山陽本線]]橫貫姬路市中心。

姬路市的中心地區是以[[江戶時代|江戶時代]]姬路城下町為基礎發展，使得1970年代之前姬路市主城區集中在[[姬路車站|姬路車站]]以北地區。但在1972年[[山陽新幹線|山陽新幹線]]開通之後，姬路車站新設南口，車站南側地區開始有顯著發展<ref name="Kinki"></ref>{{rp|86}}。近年[[姬路車站|姬路車站]]附近地區進行了土地區劃整理，市中心地區正進行大規模的再開發<ref>{{cite web|url=http://www.city.himeji.lg.jp/s70/2212598/_10616/_10617.html　|script-title=ja:姫路駅周辺土地区画整理事業の概要|accessdate=2017-04-14|publisher={{lang|ja|姫路市}}|language=ja}}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。2006年市町村合併之後，位於[[瀨戶內海|瀨戶內海]]中的[[家島群島|家島群島]]也成為姬路市的一部分。家島群島由40多個島嶼組成<ref>{{cite web|url=http://www.city.himeji.lg.jp/s20/3251001/_16957/_16958.html　|script-title=ja:家島地域の概要|accessdate=2017-04-14|publisher={{lang|ja|姫路市}}|language=ja}}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，是[[近畿地方|近畿地方]]重要的建設用石材供給地之一<ref name="Kinki"></ref>{{rp|85}}。

===氣候===
姬路市的氣候屬於[[瀨戶內海式氣候|瀨戶內海式氣候]]，按[[柯本氣候分類法|柯本氣候分類法]]則為冬冷夏熱的[[副熱帶濕潤氣候|副熱帶濕潤氣候]]（Cfa）。瀨戶內海式氣候的特徵是晴朗天氣日數多，降水量相對偏少。姬路市年均降水量是1199.0mm，低於日本平均水準<ref>{{cite web|url=https://weather.time-j.net/Stations/JP/himeji　|script-title=ja:兵庫県 姫路の気温、降水量、観測所情報|accessdate=2017-04-18|publisher={{lang|ja|気温と雨量の統計}}|language=ja}}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。姬路市在夏季時多被太平洋高氣壓覆蓋，有時會發生瀨戶內海沿岸地區特有的天氣現象{{link-ja|凪|凪}}，導致氣溫急劇升高。冬季的姬路市則多晴天，較少發生積雪的情況。位於瀨戶內海之中的家島群島則比姬路市本土的瀨戶內海氣候特徵更加顯著。

== 人口 ==
姬路市在1889年剛成立時，面積只有3.03平方公里，人口24,958人<ref name="Kinki"></ref>{{rp|81}}；此後隨著都市化的進展和轄區面積的擴大，姬路市的人口亦穩定增加。1968年時，姬路市的人口已超過40萬人<ref name="Chishi"></ref>{{rp|271}}。然而在2003年，姬路市人口達到536,562人的高峰之後已開始減少<ref name="Population">{{cite web |url=http://www.city.himeji.lg.jp/var/rev0/0100/0528/2015713155814.pdf　 |script-title=ja:ひめじ創生戦略〔人口ビジョン〕・〔総合戦略〕 人口データ編 |accessdate=2017-04-18 |publisher={{lang|ja|姫路市}} |language=ja }}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>；至2017年4月1日時，姬路市人口為533,077人<ref>{{cite web|url=http://www.city.himeji.lg.jp/toukei/hmj/hmj17/hmj1704.pdf　|script-title=ja:姫路市の推計人口　（平成29年4月1日現在）|accessdate=2017-04-18|publisher={{lang|ja|姫路市}}|language=ja}}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

雖然姬路市的總和生育率達1.56，略高於日本平均水準，但由於人口持續流出，姬路市的人口仍然持續減少<ref name="Population"></ref>。現在姬路市正試圖通過改善育兒環境等方式試圖降低人口減少速度，並促進姬路市和鄰近區域的人口交流來抑制人口減少對經濟造成的負面影響<ref>{{cite web|url=http://www.city.himeji.lg.jp/var/rev0/0099/5821/2016331194630.pdf　|script-title=ja:ひめじ創生戦略〔人口ビジョン〕・〔総合戦略〕|accessdate=2017-04-18|publisher={{lang|ja|姫路市}}|language=ja}}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

== 經濟 ==
=== 農業 ===
和日本大多數城市地區一樣，農業只在姬路市經濟中佔據很小的比例。2010年時，姬路市的農家數共有9,158戶，其中只有761戶是專業農家。2014年時，姬路市的農地面積有4,642公頃<ref name="Agriculture">{{cite web |url=http://www.city.himeji.lg.jp/var/rev0/0097/5664/201541311353.pdf　 |script-title=ja:姫路の農林水産業 |accessdate=2017-04-18 |publisher={{lang|ja|姫路市}} |language=ja }}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。姬路市所在的[[播磨平原|播磨平原]]在江戶時代生產的主要產品是「播磨三白」（米、棉、[[鹽|鹽]]）<ref name="Chishi"></ref>{{rp|276}}。現在雖然姬路的[[棉花|棉花]]生產並不興盛，但[[稻米|稻米]]仍然是姬路市的主要農產品之一。此外，姬路的主要農產品還有[[無花果|無花果]]、[[蔥|蔥]]、[[蜜瓜|蜜瓜]]等產品<ref name="Agriculture"></ref>。姬路市的林業和漁業亦有一定規模。姬路市的森林面積位居兵庫縣第七位；漁獲量則位居兵庫縣第二位，僅次於[[淡路市|淡路市]]<ref name="Genjou">{{cite web |url=http://www.city.himeji.lg.jp/var/rev0/0098/6899/6_genjou.pdf　 |script-title=ja:第3章 姫路市の現状 |accessdate=2017-04-18 |publisher={{lang|ja|姫路市}} |language=ja }}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

=== 工業 ===
[[File:KEPCO_Himeji-daini_Power_Plant.JPG|thumb]]
姬路市是兵庫縣內重要的工業都市。2012年，姬路市的製造業產品生產金額在日本全國排名第16位，在兵庫縣內則僅次於[[神戶市|神戶市]]，位居次席<ref name="Economic">{{cite web |url=http://www.city.himeji.lg.jp/var/rev0/0069/2660/201533082424.pdf　 |script-title=ja:姫路市経済振興ビジョン |accessdate=2017-04-18 |publisher={{lang|ja|姫路市}} |language=ja }}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。姬路的近代工業開始於1878年官營姬路紡織所的創建<ref name="Kinki"></ref>{{rp|84}}。播磨地方曾是日本重要的棉花產地，給姬路的紡織業發展提供了良好環境。進入[[大正|大正]]時代之後，因姬路市「軍都」的特徵，使得重工業在姬路工業中的地位迅速提高。1939年，位於姬路市南部沿海地區的廣畑製鐵所開始投入生產，是當時日本最大規模的煉鋼廠之一<ref>{{cite web |url=http://www.nssmc.com/works/hirohata/about/outline.html　 |script-title=ja:製鉄所概要 |accessdate=2017-04-18 |publisher={{lang|ja|新日鉄住金}} |language=ja }}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。同一時期，{{link-ja|山陽特殊製鋼|山陽特殊製鋼}}等企業也開始在姬路設廠。這些鋼鐵工廠大多位於沿海區域，使得姬路成為日本一大鋼鐵冶煉業中心<ref name="Kinki"></ref>{{rp|85}}。但重工業的發展同時也伴隨著紡織工業的蕭條。二戰時期姬路的眾多紡織工廠都被軍需工廠取代<ref name="Chishi"></ref>{{rp|278}}。

姬路的工業在二戰和戰後曾陷入停滯，但在1950年代中期之後再次發展。1957年，播磨臨海工業地帶被日本政府認定為繼京濱、中京、阪神、北九州之後的第五個工業地帶<ref name="Chishi"></ref>{{rp|278}}。此後在1964年，姬路市被列入工業整備特別地域<ref>{{cite web |url=https://www.city.himeji.lg.jp/kouhou/kouhoushi/backpdf/s30/pdf_s39/19640715344.pdf　 |script-title=ja:広報ひめじ |accessdate=2017-04-18 |publisher={{lang|ja|姫路市}} |language=ja }}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，姬路市的沿海地區因此興建眾多填海地並設立火力發電廠，成為近畿地方的主要鋼鐵和能源供給地。現在鋼鐵工業、電機工業、化學工業在姬路市工業中佔比重最大<ref name="Economic"></ref>。在姬路設有工廠的主要大企業包括[[三菱電機|三菱電機]]<ref>{{cite web |url=http://www.mitsubishielectric.co.jp/corporate/gaiyo/network/randd/detail/#P000000020　|script-title=ja:姫路製作所 |accessdate=2017-04-18 |publisher={{lang|ja|三菱電機}} |language=ja}}</ref>、{{link-ja|大賽璐|ダイセル}}<ref>{{cite web |url=https://www.daicel.com/profile/workplace.html　|script-title=ja:事業場案内 |accessdate=2017-04-18 |publisher={{lang|ja|ダイセル}} |language=ja}}</ref>、{{link-ja|日本化藥|日本化薬}}<ref>{{cite web |url=http://www.nipponkayaku.co.jp/company/network/factory/himeji.html　 |script-title=ja:姫路工場 |accessdate=2017-04-18 |publisher={{lang|ja|日本化薬}} |language=ja }}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>、{{link-ja|住友精化|住友精化}}<ref>{{cite web |url=https://www.sumitomoseika.co.jp/company/map/factory_himeji.html　 |script-title=ja:姫路工場 |accessdate=2017-04-18 |publisher={{lang|ja|住友精化}} |language=ja }}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>等企業。姬路市的傳統產業亦具有特色，代表的傳統產業有[[皮革|皮革]]等產業<ref>{{cite web |url=http://himeji.jibasan.jp/leather/himekawa/index.html　 |script-title=ja:姫路革細工 |accessdate=2017-04-18 |publisher={{lang|ja|電子じばさん館}} |language=ja }}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

=== 服務業 ===
服務業在姬路市經濟總量中佔據最大比重。日本主要乳幼兒服裝用品企業之一的{{link-ja|西松屋|西松屋}}創業於姬路，現在總部仍然位於姬路市<ref>{{cite web |url=https://www.24028.jp/company/history　 |script-title=ja:沿革 |accessdate=2017-04-18 |publisher={{lang|ja|西松屋チェーン}} |language=ja }}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。姬路市是播磨地方的商業中心，{{link-ja|山陽百貨店|山陽百貨店}}<ref>{{cite web |url=http://www.sanyo-dp.co.jp/company/info/index.html　 |script-title=ja:会社概要 |accessdate=2017-04-18 |publisher={{lang|ja|山陽百貨店}} |language=ja }}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>和{{link-ja|大和屋敷|ヤマトヤシキ}}<ref>{{cite web |url=http://www.yamatoyashiki.co.jp/corporate/corporate01　 |script-title=ja:会社概要 |accessdate=2017-04-18 |publisher={{lang|ja|ヤマトヤシキ}} |language=ja }}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>是兩大姬路發祥的零售業企業。由於擁有日本最大的[[現存天守|現存天守]]，姬路市是兵庫縣吸引遊客最多的城市之一。2015年，到訪姬路市的觀光客數超過1190萬人，和2014年比增加了30.1%<ref>{{cite web |url=http://www.city.himeji.lg.jp/var/rev0/0086/3884/reportH27.pdf　 |script-title=ja:姫路市入込客数・観光動向調査 |accessdate=2017-04-18 |publisher={{lang|ja|姫路市}} |language=ja }}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，顯示完成平成大修理之後的姬路城對姬路的觀光產業起到巨大振興作用。現在姬路市已將觀光產業作為姬路市的重點發展產業，以姬路城作為姬路市的最重要觀光資源，期待能吸引更多觀光客<ref>{{cite web |url=http://www.city.himeji.lg.jp/var/rev0/0096/6728/plan.pdf　 |script-title=ja:姫路市観光戦略プラン（案） |accessdate=2017-04-18 |publisher={{lang|ja|姫路市}} |language=ja }}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。 

== 交通 ==
[[File:170107_Himeji_Station_Himeji_Hyogo_pref_Japan01s3.jpg|thumb]]

姬路市是兵庫縣西部的鐵路交通中心，市內鐵路由[[西日本旅客鐵道|西日本旅客鐵道]]（JR西日本）和[[山陽電氣鐵道|山陽電氣鐵道]]兩家鐵路公司運營。[[姬路車站|姬路車站]]是姬路市的鐵路交通中心，[[JR神戶線|JR神戶線]]、[[山陽本線|山陽本線]]、[[播但線|播但線]]、[[姬新線|姬新線]]四條[[在來線|在來線]]路線及[[山陽新幹線|山陽新幹線]]在此交會<ref>{{cite web |url=https://www.jr-odekake.net/eki/timetable.php?id=0610619　 |script-title=ja:時刻表 |accessdate=2017-04-18 |publisher={{lang|ja|JRおでかけネット}} |language=ja }}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。山陽電氣鐵道的姬路車站也靠近JR的姬路車站。1972年，[[新大阪車站|新大阪車站]]至[[岡山車站|岡山車站]]之間的[[山陽新幹線|山陽新幹線]]開通，途徑姬路車站，標誌姬路市進入新幹線時代，至[[東京|東京]]及[[山陽地方|山陽地方]]、[[九州_(日本)|九州]]的陸路交通大幅改善<ref>{{cite web |url=http://www.ikkyo-tekken.org/studies/1993/1993_22.pdf　 |script-title=ja:第2章 山陽新幹線の歴史 |accessdate=2017-04-18 |publisher={{lang|ja|一橋大学鉄道研究会}} |language=ja }}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

[[山陽電氣鐵道|山陽電氣鐵道]]在姬路市有[[山陽電氣鐵道本線|本線]]和{{link-ja|網干線|山陽電気鉄道網干線}}兩條路線<ref>{{cite web |url=http://www.sanyo-railway.co.jp/railway/station/index.html　|script-title=ja:路線図・駅情報 |accessdate=2017-04-18 |publisher={{lang|ja|山陽電気鉄道}} |language=ja}}</ref>。途徑姬路市的高速公路有[[山陽自動車道|山陽自動車道]]和[[中国自動車道|中国自動車道]]<ref name="Traffic">{{cite web |url=http://www.city.himeji.lg.jp/var/rev0/0083/8259/20164819614.pdf　 |script-title=ja:公共交通を中心とした姫路市総合交通計画（改訂二版） |accessdate=2017-04-18 |publisher={{lang|ja|姫路市}} |language=ja }}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。姬路市內的巴士交通則主要由{{link-ja|神姬巴士|神姫バス}}運營<ref>{{cite web |url=http://www.shinkibus.co.jp/info/aboutSHINKI.html　 |script-title=ja:業務内容 |accessdate=2017-04-18 |publisher={{lang|ja|神姫バス}} |language=ja }}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。自姬路市有前往[[小豆島_(日本)|小豆島]]<ref>{{cite web|url=https://www.shikokuferry.com/timetable/himeji-fukuda　|script-title=ja:時刻表・運賃　姫路～小豆島（福田）|accessdate=2017-04-18|publisher={{lang|ja|四国フェリー}}|language=ja}}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>、[[家島群島|家島群島]]等地的渡船<ref>{{cite web |url=http://www.bouze-kisen.sakura.ne.jp/jikoku.26.01.14.html |script-title=ja:時刻表|accessdate=2017-04-18 |publisher={{lang|ja|坊勢汽船}}  |language=ja}}</ref>。

姬路市內過去在1970年代在姬路車站至{{tsl|ja|手柄山}}之間曾有[[單軌鐵路|單軌鐵路]]運行，但因票價昂貴導致乏人乘坐，僅通車8年之後就告關閉<ref>{{cite web |url=http://toyokeizai.net/articles/-/136449　 |script-title=ja:｢大将軍駅｣が語る｢短命モノレール｣の大構想 |accessdate=2017-04-18 |publisher={{lang|ja|東洋経済ONLINE}} |language=ja }}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。自1980年代起，也曾提出建立「播磨機場」的計畫，但至今仍未能成案<ref>{{Cite web| url=https://www.city.himeji.lg.jp/kouhou/kouhoushi/backpdf/s60/pdf_s62/19871201706.pdf| format=PDF| title=廣報姬路市　1987年12月號（706號）| language=ja| date=2087-11-15| accessdate=2017-04-19| publisher=姬路市政府}}</ref><ref>{{Cite web| url=http://www.e-himeji.com/enj00.htm| title=盛り上がらないまま白紙撤回へ| work=ほりほりの姫路ガイド| accessdate=2017-04-19| language=ja}}</ref>。

== 文化教育 ==
姬路市地處日本傳統文化中心地區之一的[[近畿地方|近畿地方]]的外緣區，擁有獨特的地方文化。姬路市的方言屬於{{link-ja|播州方言|播州弁}}，雖然屬於[[近畿方言|近畿方言]]，但和[[中國地方|中國地方]]及[[四國地方|四國地方]]的方言有較多近似之處<ref>{{cite book |author={{lang|ja|飯豊毅一・日野資純・佐藤亮一}}|script-title=ja:《講座方言学〈8〉中国・四国地方の方言》 |year=1982|publisher={{lang|ja|国書刊行会}} |location=東京都 |isbn=978-4-336-01979-0 |language=ja}}</ref>{{rp|44}}。

姬路市主要的祭典活動有{{link-ja|姬路浴衣祭|姫路ゆかたまつり}}、姬路城祭、以{{link-ja|灘之喧嘩祭|灘のけんか祭り}}為代表的{{link-ja|播州秋祭|播州の秋祭り}}等活動。姬路浴衣祭於每年6月舉行，是姬路城守護神長壁神社的夏祭，亦是關西地區規模最大的浴衣文化活動之一<ref>{{cite web |url=https://www.hyogo-tourism.jp/event/index.php?act=detail&id=9914 |script-title=ja:姫路ゆかたまつり |accessdate=2017-04-18 |publisher={{lang|ja|兵庫県観光情報サイト ひょうごツーリズムガイド}} |language=ja}}</ref>。播州秋祭是播磨地方一帶神社在秋季舉行的例大祭的總稱，其中灘之喧嘩祭於每年10月14日和10月15日舉行，是姬路市白濱町{{link-ja|松原八幡神社|松原八幡神社}}的秋季例大祭，在播州秋祭中最為知名<ref>{{cite web |url=http://www.nadamatsuri.jp/outline/index.html |script-title=ja:灘のけんか祭りとは |accessdate=2017-04-18 |publisher={{lang|ja|灘のけんか祭り}} |language=ja}}</ref>。

姬路市的主要文化設施有{{link-ja|兵庫縣立歷史博物館|兵庫県立歴史博物館}}<ref>{{cite web |url=http://www.hyogo-c.ed.jp/~rekihaku-bo/official/about.html |script-title=ja:施設紹介 |accessdate=2017-04-18 |publisher={{lang|ja|兵庫県立歴史博物館}} |language=ja}}</ref>、利用舊第十師團兵器部倉庫改建的{{link-ja|姬路市立美術館|姫路市立美術館}}<ref>{{cite web |url=http://www.city.himeji.hyogo.jp/art/gaiyo.html |script-title=ja:姫路市立美術館の概要 |accessdate=2017-04-18 |publisher={{lang|ja|姫路市立美術館}} |language=ja |title=存档副本 |archive-url=https://web.archive.org/web/20150419220711/http://www.city.himeji.hyogo.jp/art/gaiyo.html |archive-date=2015-04-19 |dead-url=yes }}</ref>。

姬路市內的高等教育機構有[[兵庫縣立大學|兵庫縣立大學]]姬路校區（前身是姬路工業大學）<ref>{{cite web |url=http://www.u-hyogo.ac.jp/outline/about/history/himeji.html |script-title=ja:兵庫県立大学の沿革 |accessdate=2017-04-18 |publisher={{lang|ja|兵庫県立大学}} |language=ja}}</ref>、{{link-ja|姬路大學|姫路大学}}<ref>{{cite web |url=http://himeji.koutoku.ac.jp/category/university/progress.html |script-title=ja:学園のあゆみ |accessdate=2017-04-18 |publisher={{lang|ja|姬路大學}} |language=ja}}</ref>、{{link-ja|姬路獨協大學|姫路獨協大学}}<ref>{{cite web |url=http://www.himeji-du.ac.jp/univ/history.html |script-title=ja:獨協学園の沿革 |accessdate=2017-04-18 |publisher={{lang|ja|姫路獨協大学}} |language=ja}}</ref>等學校。
