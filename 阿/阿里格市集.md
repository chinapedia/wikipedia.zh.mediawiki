阿里格市集除了星期一之外每天早上皆有開張， place d'Aligre和 rue d'Aligre位於巴黎第12區。

== 情況 ==
阿里格市集是由兩個市場所組成：有屋頂的市場是 marché Beauvau 或稱 Beauvau-Saint-Antoine，位於Aligre廣場西半邊的市場是沒有屋頂的，一路沿著 rue d'Aligre到廣場的另一半邊。

== 名稱起源 ==
阿里格市場的名稱源自於Étienne François d'Aligre ([[1727年|1727]]-[[1798年|1798]])，當這個市場正在建設時他是巴黎議會的的第一任主席。至於市場被覆蓋的部分則是根據Charlotte de Beauvau，最后一個女修道院院長Saint-Antoine-des-Champs和 Marc de Beauvau-Craon的女兒。

== 历史 ==
[[File:P1010872_Paris_XII_Place_Aligre_Marché_Beauvau_reductwk.JPG|左]]
有頂的Beauvau-Saint-Antoine市場是由 Samson-Nicolas Lenoir 在西元1779年在向 Saint-Antoine修道院修女所購買的土地上建造。，在西元1843年由 Marc-Gabriel Jolivet一個巴黎的城市建築師所重建。
