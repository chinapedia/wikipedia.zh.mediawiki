'''阿姆斯特丹港'''（{{lang-nl|Haven Amsterdam}}），是[[荷兰|荷兰]]首都[[阿姆斯特丹|阿姆斯特丹]]的[[港口|港口]]，位于{{Link-en|IJ湾|IJ (bay)}}与{{Link-en|北海运河|North Sea Canal}}沿岸，通过北海运河与[[北海_(大西洋)|北海]]相连。这座港口建于13世纪，在17世纪时是[[荷兰东印度公司|荷兰东印度公司]]的一个重要港口。今天，阿姆斯特丹港是荷兰第二大港，仅次于[[鹿特丹港|鹿特丹港]]。2008年，阿姆斯特丹港货运吞吐量达到7580万吨，其中大部分为散装货物。

== 历史 ==
[[File:VROOM_Hendrick_Cornelisz_The_Harbour_in_Amsterdam.jpg|thumb]]
阿姆斯特丹最早的港口活动可追溯到公元13世纪。阿姆斯特丹港于1342年首次被记载，见于阿姆斯特丹获得城市权的相关资料中。<ref name="history">{{cite web|url=http://www.amsterdamports.nl/smartsite1167.dws |title=The port of Amsterdam |publisher=阿姆斯特丹港 |date= |accessdate=2009年8月9日 |language=en |deadurl=yes |archiveurl=https://web.archive.org/web/20100309210802/http://www.amsterdamports.nl/smartsite1167.dws |archivedate=2010年3月9日 }}</ref>

在[[荷兰黄金时代|荷兰黄金时代]]，阿姆斯特丹港发展成为荷兰东印度公司的一个重要港口。<ref name="history"/>

连接阿姆斯特丹与[[登海尔德|登海尔德]]的{{Link-en|北荷兰运河|Noordhollandsch Kanaal}}建于1819年至1824年间；而连接阿姆斯特丹与{{Link-en|艾默伊登|IJmuiden}}的北海运河建于1865年至1876年间。<ref name="history"/>

== 地理 ==
[[File:IJ_4.88665E_52.39839Nconverted.JPG|thumb]]
[[File:Amsterdam_Noordzeekanaal_Afrikahaven_Amerikahaven_Westhaven_luchtfoto.JPG|thumb]]
阿姆斯特丹港坐落在北海运河与IJ湾沿岸，通过北海运河连接北海，通过北荷兰运河连接登海尔德，通过IJ和IJmeer湖连接马克美尔湖，通过{{Link-en|阿姆斯特丹－莱茵运河|Amsterdam-Rhine Canal}}连接[[莱茵河|莱茵河]]。

阿姆斯特丹港的水域面积为600[[公顷|公顷]]（1500[[英亩|英亩]]），陆地面积为1900公顷（4700英亩），包括了不动产、码头、道路、铁轨、沟渠和绿地。<ref name="annualreport2008">{{cite web|url=http://poa.80db.nl/assets/Jaarverslag_POA_V1_ENG_1704.pdf |title=Annual report 2008 |publisher=阿姆斯特丹港 |date= |accessdate=2009年7月17日 |language=en |format=PDF |deadurl=yes |archiveurl=https://web.archive.org/web/20110724143331/http://poa.80db.nl/assets/Jaarverslag_POA_V1_ENG_1704.pdf |archivedate=2011年7月24日 }}</ref>

== 商业运营 ==
阿姆斯特丹港在货物吞吐量上仅次于[[鹿特丹港|鹿特丹港]]，位居荷兰第二。<ref>{{cite web |url=http://www.havenraad.nl/images/Jaarverslag%202008%20definitief%20compleet_tcm226-246236.pdf |title=Jaarverslag 2008 |publisher=Nationale Havenraad |date= |accessdate=2009年7月23日 |language=nl |format=PDF |deadurl=yes |archiveurl=https://web.archive.org/web/20110724153457/http://www.havenraad.nl/images/Jaarverslag%202008%20definitief%20compleet_tcm226-246236.pdf |archivedate=2011年7月24日 }}</ref>

2008年，共有6029艘次船舶停靠阿姆斯特丹港，产生了7580万吨的货物吞吐量，其中大部分为散装货物。同年，集装箱吞吐量达435129[[20呎标准货柜|标准货柜]]。船舶数量和货物吞吐量均比2007年有所上升。<ref name="annualreport2008"/>

2008年，海运方面有117艘次[[邮轮|邮轮]]停靠阿姆斯特丹港，运送旅客22.6万人次；而河运方面则有998艘次邮轮运送17.0万人次的旅客。与2007年相比，海运方面呈现大幅增长，增幅达50%，而河运方面则略有减少。<ref name="annualreport2008"/>

2008年，阿姆斯特丹港总[[收入|收入]]为12530万[[欧元|欧元]]，实现[[利润|利润]]4500万欧元。收入与利润均低于2007年水平。<ref name="annualreport2008"/>

2008年，阿姆斯特丹港的直接雇员仅有361人，但间接雇员达5.5万人。<ref name="annualreport2008"/>2009年7月7日，阿姆斯特丹政府任命Dertje Meijer为港口主管。<ref>{{cite web|url=http://www.amsterdamports.nl/smartsite22206.dws |title=Dertje Meijer, new director Port of Amsterdam |publisher=阿姆斯特丹港 |date=2009年7月7日 |accessdate=2009年8月9日 |language=en }}{{dead link|date=六月 2017 |bot=InternetArchiveBot }}</ref>
