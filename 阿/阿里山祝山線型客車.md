'''祝山線型客車'''是[[阿里山森林鐵路|阿里山森林鐵路]]的[[鐵路客車|鐵路客車]]，現為祝山線、眠月線、神木區間線專用車。在[[林務局|林務局]]的紀錄中，祝山線型客車最多共有二十六輛。

== 歷史 ==
*1994年-1996年：祝山線型客車投入服務。
== 塗裝 ==
祝山線型客車另一大特點在於其塗裝，它的塗裝一改當時過去林鐵只有紅白色系的傳統，因此除紅底單白線外，於車窗窗帶改為墨綠色，車門不再牽就車體而改為銀色，整體塗裝源自[[瑞士|瑞士]][[冰河列車|冰河列車]]RhB（Rhtische Bahn）之Bernina Express登山客車，除車頂外，可謂RhB客車翻版，而內裝仿自台鐵[[EMU500型|EMU500型]]電車。

== 特殊構造 ==
車門一改過去鋼體客車側面單門的傳統，比照木造客車改為雙門設計，加快上下車速率，如此設計可能是為了因應[[祝山|祝山]]觀[[日出|日出]]與[[眠月|眠月]]觀[[一葉蘭|一葉蘭]]與石猴的大量人潮，但內裝仍沿襲今日SP客車長條椅設計。車窗改良過去巴士窗型式，以七個大窗搭配上方可滑動鋁窗呈現，亦是台灣首見。
