'''阿姆羅·雷'''（{{lang|ja|アムロ・レイ}}，Amuro Ray）是[[GUNDAM系列作品|GUNDAM系列作品]]當中由[[機動戰士GUNDAM|初代GUNDAM]]開始登場的人物。同時也是[[機動戰士GUNDAM|機動戰士GUNDAM]]的主角。他也作為[[機動戰士GUNDAM_逆襲的夏亞|機動戰士GUNDAM 逆襲的夏亞]]的主角之一出現，此外，他也有在機動戰士z GUNDAM中以配角的方式登場，[[聲優|聲優]]是[[古谷徹|古谷徹]]，-{阿寶和阿姆羅}-分別成為[[香港|香港]]和[[台灣|台灣]]高達迷對初代主角的常用稱呼，而[[中國|中國]]高達迷稱阿姆羅為騾子。

阿姆羅·雷在第2次新吉翁战争后[[任務中失蹤|下落不明]]（MIA），联邦军在搜索3个月后宣布放弃搜索，并判定其为[[陣亡|陣亡]]（KIA）。之后因其在第2次新吉翁战争的重要贡献而被特别晋升3级为上校。

== 人物介紹 ==
=== 從出生到一年戰爭前期 ===
[[宇宙世紀_(GUNDAM)|U.C.]]0063年11月4日出生，父親為提姆·雷，母親為卡瑪莉亞·雷。阿姆羅為兩人的獨子，出生地點和小時候生長的地方有很多種說法，電視版的故鄉是日本的[[山陰地方|山陰地方]]某座城市，電影版的故鄉是[[北美洲|北美洲]][[太平洋|太平洋]]的沿海城市[[魯珀特王子港|魯珀特王子港]]。

小時候與母親離別，跟著父親移民到宇宙。此時的居所也有很多說法，由於Side7的移民時間開始是0078年5月，所以在這之前應該是住在別的[[太空殖民地|太空殖民地]]。

由於父親忙於工作時常獨自一人在家。因而養成了喜歡操作電腦與機械的[[內向|內向]][[性格|性格]]。移民到Side7之後與經常受到鄰家的少女[[芙勞‧波|芙劳‧宝]]家族的照顧，此外他憑著自己在機械方面的獨特天賦獨自設計並組裝了高智能寵物機器人[[Haro|Haro]] (順帶一提在安彥良和的The Origins比改成Haro是父親買給阿寶的高智能玩具)。

由於Side7沒有比較明確的產業，大部分的移民住戶是靠生活保證金在生活著。因為父親在軍方的工作所得比較高所以阿姆羅是過著比較富裕的生活，因此阿姆羅在[[國小|國小]]畢業<ref>機動戦士ガンダム第30話「小さな防衛線」</ref>之後就沒有再繼續升學或找工作。

一年戰爭結束時的最後階級有兩種設定，電視版為[[上士|上士]]，電影版為[[少尉|少尉]]，目前以電影版為官設，電視版則當成同人或異說。

== 經歷以及在劇中的活躍 ==
=== 一年戰爭後期（[[機動戰士GUNDAM|初代GUNDAM]]） ===
; 與GUNDAM的相遇
[[宇宙世紀_(GUNDAM)|U.C.]]0079年9月18日，為了追擊[[地球聯邦軍|地球聯邦軍]]新造戰艦[[白色基地|白色基地]]，[[吉翁公國|吉翁軍]]姆塞級宇宙巡洋艦法爾梅爾號侵入Side7附近空域並且派遣三台[[MS-06系列機動戰士|薩克II]]進入殖民地內進行偵察([[強襲|強襲]])。當時15歲的阿姆羅在避難當中偶然間拿到了聯邦「[[V作戰|V作戰]]」的機密資料（[[RX-78系列機動戰士|RX-78-2 GUNDAM]]的操作說明書）並且啟動正待命中的GUNDAM。啟動之後，使用[[火神炮|火神炮]]和[[光束軍刀|光束軍刀]]以壓倒性的實力在瞬間擊毀兩架薩克II，這場戰鬥同時也是歷史上首次MS之間的對戰（實戰），父親提姆·雷因其中一台薩克II的爆炸，使殖民地有一個破洞而被外太空吸出去。[[File:AmuroRayUC0079pilot.jpg|thumb]]

此後以民間人身份成為聯邦駕駛員，並且被視為聯邦軍正規駕駛。當初因為找不到戰鬥的意義所在，與白色基地艦長[[布萊特·諾亞|布萊特·諾亞]]發生過多次衝突，但在與吉翁軍人[[藍巴·拉魯|兰巴·拉尔]]遭遇之後，在精神上漸漸從少年成長為青年。

另外，與宿敵[[夏亞·阿茲納布爾|夏亞·阿茲納布爾]]為首的諸多吉翁官兵的戰鬥當中，被發現是[[新人類|新人類]]。參加[[敖德薩作戰|敖德薩作戰]]與[[宇宙殖民地_(GUNDAM世界)#所羅門|所羅門]]攻略戰等大型戰役獲得了非常大的戰果。而被吉翁士兵稱之為「聯邦的白色惡魔」。雖然GUNDAM在最終決戰（[[宇宙殖民地_(GUNDAM世界)#阿·巴瓦·空(A_Baoa_Qu)|阿·巴瓦·空]]攻略作戰）當中擊毀了夏亞的乘機MSN-02吉翁號，聯邦最終紀錄的擊墜數為142台ms，9艘戰艦，排名第2<ref>gundam official s file page 42(本書另一處p152上提及據說阿姆羅擊墜數高達200，但真偽不明)</ref>，僅次於榮格。(有一說是200台)<ref>gundam official s file page 152(原文說據說是200台，但真偽不明)</ref>,此外阿姆羅還有一年戰爭唯一一個MA的擊墜紀錄。

在阿姆羅的成長過程當中，[[雪拉·瑪斯|雪拉·瑪斯]]等白色基地乘組員，還有他憧憬的女性軍官[[瑪琪露達|瑪琪露達]]等等都留下了不可磨滅的影響。但是最具有決定性的，應當是他在Side6所遇見的少女'''[[拉拉·遜|拉拉]]'''。阿姆羅與身為新人類的拉拉由於相遇而使得新人類的能力得以成長，在戰場上與拉拉的交戰過程中，使他初次體驗到新人類的互相理解。[[拉拉·幸|拉拉]]最後並介入惱怒的夏亞與阿姆羅之間，為夏亞擋下了阿姆羅的攻擊而死。這件事情成為阿姆羅人生最大的遺憾，並且讓他長期感到痛苦。

阿姆羅在劇中這種類似[[御宅族|御宅族]]的個性描寫，與《GUNDAM》之前的[[機器人動畫|機器人動畫]]當中幾乎清一色明朗堅強的主角個性大相逕庭，就連身為配音員的古谷徹也曾回憶提到：當時為了演出這樣的角色而吃了很多苦。在當時的青少年世代當中，認為阿姆羅忠實的反映了他們的心境，而成為GUNDAM愛好者的不在少數。

=== 格利普斯战役（[[機動戰士Z_GUNDAM|機動戰士Z GUNDAM]]） ===
阿姆羅在一年戰爭後被視為英雄，有許多記者曾經問他「到底什麼是新人類」？他的發言對大眾來說也許過於抽象，因此並沒有得到廣泛的認知。雖然被升為上尉，在北美的夏恩基地服勤，但事實上是被軟禁的狀態（當時地球聯邦政府將新人類視為危險人物），阿姆羅也因為擊落拉拉的遺憾而過著憂鬱自閉的生活。
[[File:100px-AmuroRayUC0087.jpg|thumb]]
到了[[宇宙世紀_(GUNDAM)|U.C.]]0087年，由於前白色基地乘組員小林隼人的養子卡茲的激勵，阿姆羅與卡茲一同逃離聯邦軍的監視，並加入[[A.E.U.G.|A.E.U.G.]]的支援組織[[卡拉巴|卡拉巴]]，並在此時與七年不見的[[夏亞·阿茲納布爾|夏亞]]（當時化名克瓦特羅·巴茲拿）重逢，單憑新人類互相感應已知克瓦特羅真實身分。在碰見令他想起以前自己的[[卡密兒·維丹|卡密兒·維丹]]以及貝托蒂嘉·伊爾瑪並且受到他們的刺激之後，讓阿姆羅決心再度搭乘[[機動戰士|MS]]戰鬥，雖然自稱駕駛技術大不如前(聲稱自己以前可以在1分內擊毀6台ms，現在卻花了2分鐘才勉強打贏六個新兵)<ref>機動戦士Ζガンダム Define</ref>，但是表現仍是強於早期的[[卡密兒·維丹|卡密兒·維丹]](機動戰士z鋼彈的主角)，第一場復出戰中以自己駕駛的運輸機墜毀為代價成功與夏亞及卡密兒兩人合力擊退亞席馬，之後駕駛著力克·戴亞斯（Rick Dias）與迪謝（DiJeh，新劇場版未出場）等機體參加了A.E.U.G.的吉力馬札羅基地攻略作戰以及達卡作戰。但是在克瓦特羅建議他上宇宙時，卻因為錯手殺死拉拉的陰影，而拒絕了這項提議，也因此沒有參加最終的格里普斯攻防戰。

=== 第二次新吉翁抗爭（[[機動戰士GUNDAM_逆襲的夏亞|機動戰士GUNDAM 逆襲的夏亞]]） ===
[[宇宙世紀_(GUNDAM)|U.C.]]0093年，阿姆羅以聯邦軍外圍部隊[[隆德·貝爾|隆德·貝爾]]艦隊的MS部隊指揮官身份，駕駛Re-Gz（靈格斯）與[[RX-93系列机动战士|Nu GUNDAM]]與布拉度一同對抗夏亞所領導的[[新吉翁|新吉翁]]，當時的階級是跟七年前一樣仍是上尉。

在[[宇宙世紀_(GUNDAM)|U.C.]]0093年3月12日企圖阻止馬沙投下[[阿克西斯|阿克西斯]]的攻防戰中，秒殺夏亞的MS[[MSN-04|MSN-04 沙撒比]]，並且帶著馬沙的逃生艙一起，企圖用υ GUNDAM將阿克西斯推離地球軌道。在當時地球軌道上突然觀測到規模包圍整個地球的虹光，阿克西斯隨著虹光的遠離地球，夏亞與阿姆羅則失蹤。戰後三個月聯邦軍官方記錄上記載阿姆羅戰死，並破將阿姆羅連升三級成為上校，此後都沒有阿姆羅的消息，但仍然有不少人相信這位傳說中的英雄仍然活著。

== 映像作品與小說版的相異點 ==
在小說版的初代GUNDAM當中，阿姆羅在最後因為夏亞以和解的名義放鬆了阿姆羅戒心，最終被新人類部隊以[[MS-09|Rick DOM]]發射的強化型流彈偷襲而死。但是由於作者[[富野由悠季|富野由悠季]]本人曾說並無意以此小說否定[[機動戰士Z_GUNDAM|機動戰士Z GUNDAM]]以後的諸作品，所以應視為富野所構想的另一個不同的故事。另外，在[[機動戰士GUNDAM_ZZ|機動戰士GUNDAM ZZ]]小說版當中，阿姆羅也曾經搭乘[[疾风·戴亚斯|疾风·戴亚斯]]（Schuzrum Dias）（亦有版本寫為Sturm Dias，Sturm於德文中原意為的「風暴」，一般在軍武方面多譯為「突擊」）為[[傑特·亞希達|傑特·亞希達]]斷後，幫助-{zh-hans:捷多;zh-hk:捷度;zh-tw:傑特;}-上宇宙。但在電視版GUNDAM ZZ當中Schuzrum Dias是被AE社轉贈於新吉翁的機體。又，在小說《貝托蒂嘉的子嗣》中，阿姆羅駕駛的最終機體是Hi-ν GUNDAM(平行世界中的νGUNDAM，而非其衍生型)。
