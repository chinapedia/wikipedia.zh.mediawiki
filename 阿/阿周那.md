'''阿周那'''（{{lang-sa|अर्जुन}}，{{IAST|Arjuna}}），又作'''阿尔诸纳'''及'''阿朱那'''，古印度史诗[[摩诃婆罗多|摩诃婆罗多]]中的核心人物之一，亦于[[薄伽梵譚|薄伽梵譚]]中出场。其父母分别为众神之主[[因陀羅|因陀羅]]与[[貢蒂|貢蒂]]（为[[俱卢|俱卢]]王[[般度|般度]]之首位妻子）。在其前世的生涯中，其名为那罗（Nara），为[[毗湿奴|毗湿奴]]化身之圣人[[那罗延|那罗延]]的一生挚友。

《摩诃婆罗多》中普遍被认为最重要的一部分《[[薄伽梵歌|薄伽梵歌]]》便是阿周那与化身为其车夫的[[黑天|黑天]]（[[毗湿奴|毗湿奴]]的一个[[化身|化身]]）进行的对话。
== 词源与别名 ==
阿周那之名意为“白色”或“银色”，亦有“洁净”之意。

据《夺牛篇》所载，阿周那共有九个别名：

*翼月生（{{lang|sa|फाल्गुन}}）：翼宿（即颇勒古尼）为其运星；

*吉湿奴（{{lang|sa|जिष्णु}}，意为“胜利者”）：阿周那南征北战，克敌制胜；

*有冠者（{{lang|sa|किरीटिन्}}）：阿周那与檀那婆作战时，因陀罗予其一顶金冠；

*驾驭白马者（{{lang|sa|श्वेतवाहन}}）：阿周那作战时常驭白马；

*毗跋蔟{{notetag|也译“有修”}}（{{lang|sa|बीभत्सु}}，意为“厌恶者”）：阿周那作战时从不做令人厌恶之事；

*维阇耶{{notetag|也译“毗阇耶”}}（{{lang|sa|विजय}}，意为“胜利”）：阿周那在作战时，若未战胜敌人，绝不返回；

*黑王子（{{lang|sa|कृष्ण}}）：阿周那肤色黝黑，因陀罗颇为爱之；

*左手开弓者（{{lang|sa|सव्यसाचिन्}}，意为“左右开弓”）：阿周那之双手都可拉开神弓[[甘提婆|甘提婆]]；

*胜财（{{lang|sa|धनञ्जय}}）：阿周那战胜诸多国家并抢夺其财产。<ref>{{Cite web |url = http://www.sacred-texts.com/hin/m04/m04044.htm |title = The Mahabharata, Book 4: Virata Parva: Go-harana Parva: Section XLIV |language = en |accessdate = 2018-06-25 }}</ref>
