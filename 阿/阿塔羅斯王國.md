'''阿塔羅斯王國'''，或稱'''阿塔羅斯王朝'''、'''帕加馬王國'''為小亞細亞一個希臘化王國在[[亞歷山大大帝|亞歷山大大帝]]的東征之后，[[地中海|地中海]]地區進入了所謂的[[希腊化时代|希腊化时代]]，帕加馬則在[[繼業者|繼業者戰爭]]之後變成了一個由獨立王公統治的地方勢力，由[[阿塔羅斯一世|阿塔羅斯一世]]於[[帕加馬|帕加馬]]正式稱王。。在[[阿塔羅斯王朝|阿塔羅斯王朝]]（前283年~前129年）統治下，王國一度成为一個相當強盛的國家。

== 歷史 ==

阿塔羅斯王朝得名於[[阿塔羅斯一世|阿塔羅斯一世]]，他的家族之前就統治帕加馬，家族的第一位的統治者是[[菲萊泰羅斯|菲萊泰羅斯]]，菲萊泰羅斯於前283年開始統治帕加馬。在眾多的希腊化國家中，他所建立的阿塔羅斯王朝，日後成為新興霸權的[[羅馬共和国|羅馬]]最熱心的支持者和盟友。在[[阿塔羅斯一世|阿塔羅斯一世]]統治時期，曾打敗了侵入[[小亞細亞|小亞細亞]]的[[凯尔特人|凯尔特人]]，並開始使用國王頭銜，開創阿塔羅斯王朝並建立阿塔羅斯王國。帕加馬在[[第一次馬其顿戰爭|第一次馬其顿戰爭]]和[[第二次馬其顿戰爭|第二次馬其顿戰爭]]中都与罗马人结盟，共同反对[[马其顿|马其顿]]国王[[腓力五世_(马其顿)|腓力五世]]。在[[歐邁尼斯二世|歐邁尼斯二世]]统治时代，帕加马又在[[第三次马其顿战争|第三次马其顿战争]]中站在罗马一边反对马其顿国王[[珀尔修斯_(马其顿)|珀尔修斯]]。作为一支相对弱小的国际力量，阿塔罗斯王朝诸王选择与罗马结盟无疑是明智的。在罗马人打垮了最大的希腊化国家[[塞琉古帝国|塞琉古帝国]]后，阿塔羅斯王國获得了前塞琉古帝国在安纳托利亚的全部领土作为长期支持罗马的回报。

在[[阿塔罗斯三世|阿塔罗斯三世]]于前133年年去世时，他没有留下任何男性子嗣。他在遗嘱中将帕加马“赠予罗马”，以避免发生内战或是罗马干脆以军事入侵吞并这个国家。 

== 文化 ==
[[File:Pergamon_Trajan_Temple_RB2.jpg|190px]]
阿塔罗斯王朝的统治比较开明。许多保存下来的古文献显示，帕加马的国王鼓励文化，他们向城镇派遣能干的技工并且征收较低的税赋。他们并且允许在其辖境内的其它希腊城邦保持名义上的独立地位。帕加马与主要的希腊文化中心如[[德尔斐|德尔斐]]，[[提洛|提洛]]和[[雅典|雅典]]保持往来。而阿塔罗斯诸王仿照[[雅典卫城|雅典卫城]]建造了帕加马的[[卫城|卫城]]。

帕加马拥有古代世界第二好的[[图书馆|图书馆]]，也许仅次于[[埃及|埃及]][[亞歷山卓|亚历山大]]的[[亚历山大图书馆|亚历山大图书馆]]。

== 影響 ==
在埃及的[[托勒密王朝|托勒密王朝]]禁止出口[[莎草纸|莎草纸]]之后，帕加马人发明的[[羊皮纸|羊皮纸]]成了地中海地区纸张的唯一替代物，影響後世。
