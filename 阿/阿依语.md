{{Infobox language|familycolor=American|name=阿依语|nativename=''A'ingae''|states=[[厄瓜多尔|厄瓜多尔]], [[哥伦比亚|哥伦比亚]]|region= [[亚马逊雨林|亚马逊雨林]]|ethnicity=[[阿依人|阿依人]]|speakers=2,400|date=2001–2008|ref=e18|iso3=con|family=[[孤立语言|孤立语言]]|nation=[[厄瓜多尔|厄瓜多尔]]: 部落领地官方语言}}'''阿依语''' (A'ingae，西班牙语称作 '''柯梵语（Cofán）''') 是[[厄瓜多尔|厄瓜多尔]]东北部和[[哥伦比亚|哥伦比亚]]南部亚马孙流域土著居民[[阿依人|阿依人]]使用的[[语言|语言]]。阿依语在形态上属于[[黏着语|黏着语]]，主要依靠不同的词缀来实现各种语法功能。

==音系==

阿依语的[[音节结构|音节结构]]为CV，CVʔ或CVN，也就是说，[[音节|音节]]末尾的[[辅音|辅音]]必须是[[喉塞音|喉塞音]]或[[鼻塞音|鼻塞音]]。

===元音===

===辅音===

阿依语的[[塞音|塞音]]分为[[清不送气|清不送气]]，[[清送气|清送气]]和[[鼻冠浊音|鼻冠浊音]]三种。其中鼻冠浊音在词首一般实现为浊塞音。塞擦音有清浊对立，浊塞擦音在非词首位置时实现为[[鼻冠|鼻冠]]浊音。擦音只有清音。[x]和[h]为自由变体。

== 数词 ==
阿依语目前常用的数词为1~5，5以上的数字一般用[[西班牙语|西班牙语]]表示。5以内的数字以[[二进制|二进制]]位基础：

1. fue'cco (二进制：1）

2. ccoangi'cco（二进制：10）

3. coanifue'cco（二进制：10 + 1 = 11）

4. ccattufayi'cco（二进制：100）

5. fuefayi'cco（二进制：1 + 100 = 101）

跟据M. B. Borman编写的''Vocabulario Cofan''（《柯梵语-西班牙语，西班牙语-柯梵语词典》）<ref>https://www.sil.org/resources/archives/10958</ref>，6~10可以用以下方式表达：

6. ccafaiseyi'cco（字面含义：换到另一只（手））

7. ccafaise(yi)ccoangi'cco（字面含义：换到另一只（手）数到二（二进制：10））

8. ccafaise(yi)ccoanifue'cco（字面含义：换到另一只（手）数到三（二进制：11））

9. ccafaise(yi)ccattufayi'cco（字面含义：换到另一只（手）数到四（二进制：100））

10. tive pa'tssi（字面含义：两手用全）

又根据Marcelino de Castellvi 1938年的记录<ref>http://www.persee.fr/doc/jsa_0037-9174_1938_num_30_2_1975</ref>，4可以被称为fue'fayi（1*100），5可以被称为fue'faye（1+100），6可以被称为fue'faise-ccoangi（100 + 10），7可以被称为fue'faiseccoanifue （100 + 11), 8可以被称作fue'faiseiccattufayi (100 + 100)。这体现出了阿依语曾经有过更为严密的[[二进制|二进制]]体系。当然，日常生活中可能很少会真正用到5以上的数词，所以Borman以及de Castellvi所提供的5以上数字的说法或许并不是非常通行。在de Castellvi的记录中，5以上的数字都有至少三四种说法，这也反映了5以上数字用词的不稳定性。
