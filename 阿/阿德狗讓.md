'''阿德狗讓'''（A Tek Kaujong，{{bd|1592年||1661年||catIdx=A}}）是台灣中部[[拍瀑拉族|拍瀑拉族]]的戰士，領導族人與[[明鄭時期|東寧王朝]][[鄭成功|鄭成功]]軍隊作戰。最後中伏擊而死。
歷史文件中以'''阿德狗讓'''之名紀錄；中村孝志則認為阿德狗讓並非人名，而是「大肚番」的別稱<ref>中村孝志。《荷蘭時代臺灣史研究下卷》。台灣：稻鄉，2002年。頁92。</ref>。<br />

== 事蹟 ==
西元1661年6月時，明軍將領張志與黃昭遣發屯墾北路。因為縱管事楊高凌削土番，激發大肚番阿德狗讓殺楊高並圍攻軍營，以標槍殺死左先鋒楊祖。7月，[[鄭成功|鄭成功]]派黃安、陳瑞二將領征討。最後以伏擊戰術斬殺阿德狗讓。

== 地名說 ==
中村孝志則認為a-tek Kaujong之a-tek為無意義的發語音，Kaujong則為大肚番的別稱Hajovan，並非人名<ref>中村孝志。《荷蘭時代臺灣史研究下卷》。台灣：稻鄉，2002年。頁92。</ref>

== 其他 ==
近代作家考慮到將"狗"字硬嵌在名字中有種族歧視疑慮，且避免原住民抗議。於是以類似譯音取代。

如小說"大語符紋路"中，參考鄒族朋友建議用"阿德克故浪(A Tek Kaujong)"取代。
