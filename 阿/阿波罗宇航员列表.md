==曾接近月球但未登月的宇航员==

在上列的6次[[阿波罗计划|阿波罗]]任务中，每一次都有一名宇航员留在[[阿波罗指令/服务舱|指令/服务舱]]中，在另外两名搭档登月时绕月球轨道航行。此外，[[阿波罗8号|阿波罗8号]]、[[阿波罗10号|阿波罗10号]]、[[阿波罗13号|阿波罗13号]]三次任务中均有3名宇航员接近了月球(前两次任务中飞船按计划绕月球轨道航行，阿波罗13号仅从月球绕过，未进入轨道)。如下是曾离月球仅数百公里的宇航员名单：

# '''[[吉姆·洛威尔|吉姆·洛威尔]]''' - 阿波罗8号, [[阿波罗13号|阿波罗13号]]：计划登月（在世）

# '''[[托马斯·斯塔福德|托马斯·斯塔福德]]''' - [[阿波罗10号|阿波罗10号]]（在世）

洛威尔和海斯曾在[[阿波罗13号|阿波罗13号]]中被安排执行登月任务，但是在抵达月球前的事故让登月计划被放弃。3人后来成功返回地球。

===未执行过阿波罗任务的替补宇航员===
每次阿波罗任务都有替补成员接受任务训练，以保证在主力成员无法执行任务时接替。真正被使用的替补成员只有一位：[[阿波罗13号|阿波罗13号]]发射前两天，[[杰克·斯威格特|杰克·斯威格特]]替换了接触了[[风疹|风疹]]的[[肯·马丁利|肯·马丁利]]。

# '''[[戈尔登·库勃|戈尔登·库勃]]''' - [[阿波罗10号|阿波罗10号]]替补指令长，1970年从美国国家航空航天局辞职。
# '''[[约瑟夫·恩格|约瑟夫·恩格]]''' - [[阿波罗14号|阿波罗14号]]替补登月舱驾驶员，原计划中的[[阿波罗17号|阿波罗17号]]登月舱驾驶员，后来由哈里森·施密特执行。
# '''[[文斯·布兰德|文斯·布兰德]]''' - [[阿波罗15号|阿波罗15号]]替补指令舱驾驶员，计划中的[[阿波罗19号|阿波罗19号]]指令舱驾驶员。
