'''阿塞拜疆驻华大使馆'''（{{lang-az|Azərbaycanın Çin səfirliyi}}）是[[亞塞拜然共和國|亞塞拜然共和國]]在[[中華人民共和國|中華人民共和國]]所設的[[外交代表機構|外交代表機構]]，位于中国[[北京市|北京市]][[朝阳区|朝阳区]][[建国门外大街|建国门外大街]]上的齐家园外交公寓B-3号<ref>{{Cite web |url=http://www.azerbembassy.org.cn/index.php?m=content&c=index&a=lists&catid=51 |title=联系我们 |publisher=阿塞拜疆驻华大使馆 |accessdate=2015-07-14}}</ref>。

== 历史 ==
1992年4月2日，[[亞塞拜然共和國|亞塞拜然共和國]]與[[中華人民共和國|中華人民共和國]]正式建交。中方於同年8月份於亞塞拜然首都[[巴庫|巴庫]]設立[[中国驻阿塞拜疆大使馆|大使館]]，而亞塞拜然駐華大使館則於1993年9月3日於北京開館<ref>{{Cite web |url=http://www.azerbembassy.org.cn/index.php?m=content&c=index&a=lists&catid=19 |title=大使致辞 |publisher=阿塞拜疆驻华大使馆 |accessdate=2015-07-14}}</ref>。
