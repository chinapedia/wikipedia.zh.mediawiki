'''阿侬语'''（'''ɑ³¹nuŋ³¹'''）是[[怒族|怒族]]支系[[阿侬人|阿侬人]]使用的一种[[藏缅语族|藏缅语言]]。<ref>{{cite journal|author=段玲|title=怒族阿侬语概说|journal=大理师专学报|volume=1997年|issue=第2期}}</ref>它与同语支的[[独龙语|独龙语]]较接近，同源词高达33.3%，而与怒族[[怒苏语|怒苏语]]区别很大。现时大部分[[阿侬人|阿侬人]]都已转用[[傈僳语|傈僳语]]和[[汉语|汉语]]，阿侬语在日常生活中已很少使用。<ref>孙宏开，刘光坤，16－20页．</ref><ref>{{cite journal|author=孙宏开|title=记阿侬语—对一个逐渐消亡语言的跟踪观察|journal=中国语文|volume=1999年|issue=第5期}}</ref>

== 词汇 ==
阿侬语的词汇根多为单音节，但实际词汇里单音节词不多。在种种词语中，和[[捕鱼|捕鱼]]、[[采集|采集]]、[[狩猎|狩猎]]相关的词汇很多，而表示竹子的词汇甚至有几十种。或许由于保留了一些古老构词特点，阿侬语中存在很多名、动同音词。

从词的来源看，阿侬语有固有词、外来词两大类。固有的核心词一般来自原始藏缅语或汉藏语。阿侬语与[[彝语|彝语]]同源词约有14.8%，与傈僳语同源词约16.1%，与景颇语同源词则约占18.5%。另一些固有词则为阿侬语独有，大多为阿侬人在长期生活实践中陆续创造。至于阿侬语的借词，主要来自[[傈僳语|傈僳语]]和汉语，少数来自[[缅甸语|缅甸语]]、[[白语|白语]]、[[藏语|藏语]]。

早期汉语借词语音已适应阿侬语特点，如lo<sup>55</sup>tsɯ<sup>55</sup>（骡子）、da<sup>31</sup>mo<sup>55</sup>（帽子），有些可能是通过傈僳语转借，新借词则与当地汉语方言近似，如ɕan<sup>35</sup>fɑ<sup>31</sup>（宪法）。有些汉语借词阿侬语中有类似的固有词，借词与固有词并存。
阿侬语中傈僳语借词更加普遍。而由于从缅甸传来的宗教读物，阿侬语也出现了一些缅语借词。

从词的结构来看，阿侬语可非为单纯词和复合词。单音节单纯词约占10%，低于其他汉藏语言。<ref>孙宏开，刘光坤，46－51页．</ref>

== 语法 ==
=== 词类 ===
阿侬语的词可大略分为名词、数词、量词、代词、动词、形容词、副词、连词、语气词、叹词和助词。

==== 名词 ====
名词的语法特点有：
# 若表示复数，可以再名词后加zɪ<sup>31</sup>ɳɯ<sup>31</sup>或mɯ<sup>53</sup>。
# 人称领属：名词前加ɑ<sup>31</sup>表示“我的”，ɳɯ<sup>31</sup>表“你的”，ŋ<sup>31</sup>表“他/她的”。
# 名词后加tɕhɛn<sup>31</sup>（本意“儿子”）表示小。
# 后缀phɯ<sup>31</sup>表阳性。mɯ<sup>31</sup>/mɑ<sup>55</sup>表阴性。

==== 数词 ====
数词常与量词连用，置于量词之前：
# 基数词：thi<sup>55</sup>（一）、ɑ<sup>31</sup>ɳi<sup>55</sup>（二）、ɑ<sup>31</sup>som<sup>53</sup>（三）、bɹi<sup>53</sup>（四）、phɑŋ<sup>31</sup>（五）、kuŋ<sup>55</sup>（六）、sɪȵ<sup>35</sup>（七）、ɕɛȵ（八），dɯ<sup>31</sup>gɯ<sup>31</sup>（九）
# 合成数词：由基数词加tsha<sup>55</sup>（十）、ɕɑ<sup>33</sup>（百）、jɑ<sup>55</sup>/tu<sup>31</sup>（千）、mɯ<sup>31</sup>构成。
# 分数：两个数量词中间加ɖuŋ<sup>31</sup>khɑ<sup>31</sup>（××分中××分）。
# 倍数：数词后加khɑ<sup>55</sup>（倍）。
# 序数：数词加kɯ<sup>31</sup>phɑŋ<sup>55</sup>/ʔi<sup>55</sup>phɑŋ<sup>55</sup>前缀。

==== 量词 ====
分为名量词、动量词两类。

==== 代词 ====
分人称代词、指示代词、疑问代词、反身代词、泛指代词5类。

==== 动词 ====
# 人称和数：
## 单数：第一人称加后缀ŋ<sup>55</sup>、第二人称加前缀ŋ<sup>31</sup>。
## 双数：第一人称加后缀sɛ<sup>55</sup>/so<sup>33</sup>、第二人称加ɳ<sup>31</sup>前缀和sɛ<sup>55</sup>后缀。
## 多数：第一人称加后缀i<sup>31</sup>、第二人称加前缀ɳ<sup>31</sup>、ɳɯ<sup>31</sup>后缀。
## 第三人称的动词均不加改动。阿侬语动词的人称和数有时还要与宾语或宾语的定语相关联。
# 体：分将行、进行、已行、曾行和完成体。
## 动词后bɯ⁵⁵、uɑ⁵⁵加表即将进行。
## 动词后加ɛ⁵⁵、no³¹ɛ³¹表正在进行。
## 动词后加dʑɛ³¹表已进行。
## 动词后加ie³¹表曾进行过。
# 态：分自动、使动和互动。
# 式：陈述、命令和祈求。
## 陈述式：加后缀ɛ⁵⁵或ɛ³¹。
## 命令式：动词声母为清音、浊塞擦音、复辅音，加o³¹表示单数命令式，加so³¹表示双数命令式，加ɳo³¹表示多数命令式。声母为浊擦音、浊鼻音、浊边音，声母变为清音再加前面所说的后缀；动词词根为开元音打头，除了加前述后缀外，另加phɯ³¹前缀。
## 祈求式：动词加前缀lɛ⁵⁵或lɑ³¹，句末加语气词lie³¹。
# 方向：根据行为的朝向加不同后缀。
# 名物化：加后缀使得动词或宾动词组作名词用。
## 表示“做××用的”：加dɛm⁵⁵。
## 表示“做××的地方”：加zɑ⁵⁵。
## 表示“做××的人”：加su⁵⁵。

==== 形容词 ====
前加uɑ⁵⁵、tɕhi⁵⁵分别表示两种反义的状貌形容词；词根重叠来表达程度加深；可以名物化（后加o⁵⁵、前加ŋ³¹或ʔŋ³¹）。

==== 副词 ====
修饰谓语。

==== 助词 ====
大致分为领属、施动、受动、工具、时间、地点、从由、比较、随同、定指等类。

==== 叹词 ====
表达应答、惊讶、叹息、提示等语气。
# 提示说话人注意：ʔo⁵⁵和nɑ⁵³，放句首。
# 表无可奈何：ħɛ³³，放句首;
# 表惊愕、奇怪、受刺激等：ɕɑ⁵⁵、ɑ⁵⁵mi⁵⁵ 、ɑ⁵⁵tɕɯ³³等。
# 表痛楚：ɑ³¹lɑ⁵⁵和ɑ⁵⁵iɯŋ⁵⁵，放句首。
# 表轻蔑：phe⁵³，放句首。
# 打招呼：ɑ⁵⁵ɣɯ⁵⁵
# 表领悟：ɑ̃³¹
# 表不满：ŋ³¹ŋŋ³⁵
# 表称许、羡慕：ʔo⁵⁵ue³¹
# 表慨叹：ẽ³¹
# 表惋惜、同情：tse³¹tse³¹
# 表否定：pfu̥³¹<ref>孙宏开，刘光坤，64－120页．</ref><ref>{{cite journal|author=孙宏开|title=阿侬语概况|journal=民族语文|volume=2000年|issue=第4期|pages=68－80页}}</ref>

=== 句法 ===
阿侬语的句子主要为[[主宾谓|主宾谓]]结构，没有[[被动句|被动句]]。加了[[结构助词|结构助词]]的句子，[[宾语|宾语]]可置于[[主语|主语]]前。定语为名词、代词时，置于中心词前；定语为形容词、数词、指量词组时置于中心词后。[[状语|状语]]一般加在[[谓语|谓语]]前，少数副词可置于谓语之后。

阿侬语单句的类别有：
# 陈述句：用陈述后缀（如ɛ³³）或其他体后缀。
# 疑问句：用疑问代词（如khɑ³¹ioʔ⁵⁵[谁]）、疑问语气词（ʠɑ⁵³）、是非问或选择疑问构成。
# 命令句：动词使用命令式
# 祈求句：动词祈求式
# 惊叹句：使用表惊叹的叹词/语气词。
# 拟测句：使用拟测语气词。（如tha³¹luʔ⁵⁵[大概]）

阿侬语复句用来表达完整概念，有并列和偏正两大类。<ref>孙宏开，刘光坤，123－129页．</ref>

== 阿侬语的流变 ==
阿侬语是一个急剧变化中的语言。词汇方面，外来词急速增长；语音方面，增加了一些复合元音和鼻化元音，复辅音逐渐消失，卷舌音正在消失，出现了紧喉元音、中平调，语音自由边读较普遍；语法上名词动词系统都发上了很大变化。<ref>孙宏开，刘光坤，133－154页．</ref>
