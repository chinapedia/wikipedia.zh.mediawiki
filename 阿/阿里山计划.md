[[八二三砲戰|1958年第二次台海危機]]白熱化之際，美軍為表示捍衛[[中華民國|中華民國]]的決心，在1958年9月自加州本土緊急增派{{link-en|第83戰鬥機中隊|83d Fighter Weapons Squadron}}協防臺灣，這是F-104戰鬥機與中華民國首次的交集。

該中隊為首批操作F-104A攔截機的單位，而當時[[中華民國空軍|中華民國空軍]]對F-104的了解不多，只知道該機極速達2.0馬赫，足足快出現役的F-86進倍；823砲戰結束之後，情報顯示[[中国人民解放军空军|中国人民解放军空军]]已逐步換裝俄製[[MiG-19|MiG-19]]戰機，因此國軍方面向美軍顧問團申請更多F-100超級軍刀機軍援，F-104並非國軍方面要求的增援機種。不過美國在允諾增援更多的F-100戰機同時，且主動要求提供國軍換裝F-104，這個消息大出國軍意外，也因此在[[國防部|國防部]]內出現了意外的反對聲浪。

美軍軍援並非空白支票可以無限額度運用，每年會有計劃額度，超越額度後便須由軍援國自行吸收經費。在當時，換裝一支F-104[[中隊|中隊]]需要的經費，與換裝三個人數破萬的陸軍師相等，在經費拮据戰略仍需反攻大陸的時代，分配資源的比重不只考慮實質戰術要求，也涉及政治智慧的配重；但當時當時國防部長[[俞大維|俞大維]]認為陸軍師再多對空防也沒有助益，因此決定先換裝一個中隊的F-104，使中華民國空軍正式邁入倍音速時代，換裝代稱就是所謂「'''阿里山計劃'''」，後續接收F-104仍以相同代稱延續。

== 阿里山1號 ==
民國49年（1960年）5月28日，空軍正式實施阿里山一號計劃，換裝美國軍援的一批21架F-104A單座（一說25架）及5架F-104B雙座（最初為4架，但1963年10月因墜毀1架B型，美軍後來又軍援1架同型機頂替）戰機，駐地選定位於[[台中|台中]]的公館基地（美軍擴建後的[[清泉崗空軍基地|清泉崗空軍基地]]）。

F-104戰機在飛行速度於[[馬赫|馬赫]]0.9、1.4、1.7及2.0時，[[剩餘推力|剩餘推力]]最大；最能發揮該型機的優勢，攔截進犯台灣北部政經中樞的敵機。當時決定由空軍第三大隊第8[[中隊|中隊]]為第一個換裝F-104的中隊，飛行員由各戰鬥機部隊遴選。換裝F-104的飛行員不只要合乎空勤標準，甚至於連長像都列為遴選的條件之一，經過兩個多月的挑擇優選出了22位官校22期至39期、在當時正處於巔峰狀態，全都參加過台海戰爭的飛行員，有些甚至還在1958年甫告一段落的台海危機中擊落過MiG-17。

第一批到達台灣為2架F-104B，在1960年5月17日由美軍[[道格拉斯C-124|道格拉斯C-124]][[運輸機|運輸機]]自[[琉球|琉球]]空運抵台，8中隊位於基地西區，於是便將該區域劃為F-104的專用廠區，除8中隊的空地勤人員外，任何閒雜人等一律不准進入。而當時F-104的裝配廠，俗稱阿里山廠，也就是現在[[漢翔|漢翔]]位於清泉崗基地的裝配廠。這批F-104A型編號為4201至4222，B型為4101至4105；不過阿里山一號星式機只使用6年就歸還美國，美國以全新的G型機（阿里山5號）替換這批飛機，而這批A/B型機則轉讓給{{link-en|約旦皇家空軍|Royal Jordanian Air Force}}，有1架A型則讓渡到給[[巴基斯坦空軍|巴基斯坦空軍]]序列，運用至1977年除役。

== 阿里山2號 ==
接收首批F-104A/B之後，美國依海外軍事援助計劃（Military Associate Program，MAP）再軍援一批55架F-104G給台灣，前20架由[[洛克希德|洛克希德]]原廠製造，其餘則由[[加拿大飛機公司|加拿大飛機公司]]（Canada-ir）生產。

民國52年（1963年）7月16日，實施阿里山二號計劃，首次接收F-104G。接收此批戰機的單位為三大隊7中隊。
== 阿里山3號 ==
民國52年8月12日，進行阿里山三號計劃，由第六大隊12中隊換裝8架RF-104G偵察機及2架TF-104G雙座戰鬥教練機。

12中隊RF-104G編號分別為5626、5628、5630、5632、5634、5636、5638、5640等8架。特點是均採偶數碼編號，安排此特色的理由是因為空軍歷來損失的偵察機編碼多為奇數號機，為使飛行員安心，在寧可信其有的心態下編號也就從善如流避開單數號。

== 阿里山4號 ==
民國53年（1964年）5月1日，實施阿里山四號計劃，接收了23架F-104G、1架TF-104G；由原先操作F-86F的空軍三大隊28中隊換裝，三大隊也成為空軍第一支全大隊運用F-104的單位。

== 阿里山5號 ==
由於美軍強化對第三世界國家的軍援政策，東南亞國家的援助，美軍收回阿里山一號的F-104A/B讓渡給約旦及巴基斯坦。
為此中華民國空軍在民國56年（1967年）5月11日實施阿里山五號計劃，原本採用A/B型機8中隊從美方獲得全新的F-104G；這批飛機總共有11架G型機與2架RF-104，參與113空戰的F-104G也多為此批戰機；換裝單位為阿里山1號時取得F-104A的三大隊8中隊。

== 阿里山6號 ==
民國59年6月，空軍向美國爭取F-4戰機未果，但得到美國提供兩個中隊的F-100A與一個中隊的F-104A/B。民國59年12月1日換裝，此為阿里山六號計劃，共接收了25架F-104A/B、F-104A編號由4241至4262、F-104B由4121至4123。由十一大隊41中隊接受換裝。

雖然看似美國只提供了較早出廠的二手A型機，但較不為人知的是這批A型機在機體性能的突出之處；該批F-104A為F-104A-20-LO批次機體、1956年出廠，與1958年增援臺灣的第83攔截機中隊運用機型相同，但這批飛機在美國空軍操作時是由1969年12月解散的{{link-en|第319攔截機訓練中隊|319th Fighter Interceptor Training Squadron}}運用，單位解散後轉移給中華民國空軍；在1967年底，319中隊的26架戰機為協助{{link-en|F-104S戰鬥機|Aeritalia F-104S}}開發，故改裝上S型機使用的J79-GE-19型發動機（後燃推力17,900磅），推力數值甚至高於F-104G，又比阿里山一號的F-104A/B推力提升了1,900磅，加上比較輕為攔截任務設計的A型機身，使得這批F-104A推重比表現是F-104家族中最佳的一種子型號。這批F-104A與阿里山一號外觀上的不同處在於發動機噴嘴，後期加裝了上下防撞燈、機首下方的[[太康天線|太康天線]]及主輪蓋攔截器。

推力增強後的F-104A飛行表現，據服役過的飛官回憶加速性極佳、飛行操作性也相當優秀，因此飛過的飛行員個個讚不絕口；由0.9馬赫加速至2.0馬赫只需2分多，1.4馬赫還可以帶桿轉彎，+2G的動作下，後燃器全開飛行速度仍可到達馬赫2.0。當戰機无外挂、機內燃油3000磅、推重比接近1時，加速度更快！

== 阿里山7號 ==
因雙座機的嚴重短缺，空軍遂於民國65年5月13日實施阿里山七號計劃，購置美軍除役的F-104D共七架，由三大隊接收。機號4161至4166，其中一架做為拆零機，至民國77年除役。

== 阿里山8號 ==
1980年代初期，美國雷根政府與中共簽訂[[中美三个联合公报|817公報]]後，空軍向美國提出購置 XF 戰機（主要是 F-16/79、及 [[F-20|F-20]]）便受到強大政治阻礙而未能通過。但仍得到美國政府同意，將[[西德空軍|西德空軍]]於[[:en:Luke_Air_Force_Base|美國路克空軍基地（LUKE AFB）]]的一批66架TF/F-104G售予中華民國。這批戰機共有29架F-104G、37架TF-104G 、分為23架及43架兩批運交。計劃中尚包括26具J79發動機、17具[[後燃器|後燃器]]。

這批飛機中，TF-104G由當時仍運用F-104的三、六、十一大隊下轄之5個戰術/偵照中隊均分；F-104G則由十一大隊的42、48中隊換裝，十一大隊成為第二支全大隊運用F-104的單位，在阿里山八號完成之際，中華民國F-104機隊進入服役高峰期，當時有兩支大隊、一支偵照中隊共7支中隊運用F-104。

民國72年7月20日，由十一大隊41中隊實施阿里山八號F-104G戰機戰備訓練計劃，換裝TF-104G。第一架組裝完成的是4186號機。民國73年1月1日，42中隊換裝F-104G，同年9月5日48中隊換裝F-104G，這批戰機也是最後除役的一批，共服役十五年。

== 阿里山9號 ==
為了維持日趨老舊的F-104機群及補充因損失而減少的數量，空軍決定向美方再增購一批F-104戰機，而美國便將號稱在[[亞利桑那州|亞利桑那州]]沙漠儲存場封存，由日本製造，並在航空自衛隊服役的F-104J/DJ轉售中華民國。日本製造的星式機會封存在美國是因為這批戰機購置時為美國以海外有償軍援的方式出資協助日本自製，產權上屬於日美共有，有必要也可逃避[[:ja:武器輸出三原則|武器輸出三原則]]規範轉讓，因此航空自衛隊在1986年將星式機完全除役後挑選了狀況較好的飛機歸還美國，美國則利用規範漏洞轉讓給中華民國，實際上這批飛機可能根本連封存廠都沒進去過。

中華民國空軍於民國76年7月6日實施阿里山九號計劃購置30架F-104J與5架F-104DJ，但只組裝21架F-104J 及5架F-104DJ，其餘充做拆零機。F-104J機號由4501至4521、F-104DJ則由4591至4595，其中4520於飛交CCK時墜毀在半途中，於是再組裝一架4522出來。本批戰機由阿里山2號接裝的三大隊7中隊使用，而該中隊的G型機則遭汰換。

J型機與 G 型機的不同主要受到日本國情限制，J型在製造時便取消了M-2轟炸計算機，火控雷達也是比較陽春的NASARR F15J，雖然服役時有配備2.75吋火箭，但嚴格來講並不具有對地攻擊能力。但因為取消了相關機能，飛機整體重量略輕，飛行性能也略勝G型機一籌。反映在外觀上的差距為主輪較G/S型窄，起落架艙蓋並無像G/S型一樣隆起一塊方形。

雖然日本挑選了飛行時數較低且較新的飛機轉讓。但是終究是日本授權生產的特規機，許多零件即使和G型看似共通，但實際混合運用時仍會出問題，也因此發生墜機意外；發生事故後J型在後勤保修上的困難日趨嚴重，因此加速了J型機的除役進度，使得這批戰機在空軍內服役不到5年就在民國80年3月1日除役。

最廣為人知的零件相容性問題是發生在1993年3月4日，一架編號5664的RF-104G偵察機發動機，其中的J79-IHI-11B發動機高壓段(後級)與J79-GE-11B低壓段(前級)相接造成的失事意外，飛行員在跳傘三天後才獲救。

== 阿里山10號 ==
空軍在民國73年6-8月連摔三架F-100超級軍刀機之後，決定汰除48中隊的F-100機群，為補充11大隊戰機的不足，於是再向美國購置了一批F-104G。這批F-104G為美國依海外軍援計劃軍援[[丹麥|丹麥]]的戰機，除役之後歸還美國轉售台灣。

空軍在民國77年1月15日實施阿里山十號計劃，共購置18架，分別為F104G 4411~4425及TF-104G 4151~4153 。這批F-104G為加拿大飛機公司（Canadair）生產、與阿里山二號部分F-104G是同一個生產批號，二十餘年後這些戰機竟又在台灣相逢。但和阿里山八號的F-104G在外形及內部結構都有些許不同，而這批F-104G準備運交台灣時，美方先行予以整修將F-104G的輕翼結構改為與TF-104G相同的重翼結構，以增加其強度及使用壽命。

== 阿里山11號 ==
因洛克希德公司宣佈停止供應F-104零組件，空軍最後透過美國國防部於前後購入16架F-104G與8架TF-104G。這批原為比利時空軍的F-104戰機並未成軍服役，全數充作拆零機使用。
