'''阿坂城'''（{{jpn|j=阿坂城|hg=あざかじょう|rm=Azakajō}}），又稱'''白米城'''（{{jpn|j=白米城|hg=はくまいじょう|rm=Hakumaijō}}）或'''椎之木城'''（{{jpn|j=椎之木城|hg=しいのきじょう|rm=Shīnokijō}}），是[[伊勢國|伊勢國]]{{Link-ja|一志郡|一志郡}}阿坂（現[[三重縣|三重縣]][[松阪市|松阪市]]大阿坂町）曾經存在的一座{{Link-ja|山城 (日本)|山城|山城}}。城池是伊勢[[國司|國司]]及後成為[[戰國大名|戰國大名]]的[[北畠家|北畠氏]]在伊勢北部的重要據點<ref name="fs1">三重県観光連盟 監修（1989）：第181頁</ref><ref name="kk">「角川日本地名大辞典」編纂委員会 編（1983）：第78頁</ref>。在[[南北朝時代_(日本)|南北朝]]時，城池乃南朝北畠氏的勢力範圍，在[[稱光天皇|稱光天皇]]即位後北畠氏在城池舉兵，並且與[[足利幕府|足利幕府]]軍隊交戰，雖然最終戰敗，但是以米洗馬的傳說卻流存至今。儘管，北畠氏在阿坂城戰況失利，但最終成功與幕府和解而存續下來。進入[[戰國時代_(日本)|戰國時代]]後，[[織田氏|織田氏]]家臣[[瀧川一益|瀧川一益]]曾經再三拒絕攻打阿坂城，而[[豐臣秀吉|豐臣秀吉]]攻打城池期間亦曾經被箭射中，這是他生涯唯一一次在戰事中受傷。1982年，城跡以'''阿坂城跡附高城跡枳城跡'''（{{jpn|j=阿坂城跡 附 高城跡 枳城跡|hg=あざかじょうあと つけたり たかんじょうあと からたちじょうあと|rm=Azakajōato Tsuketari Takanjōato Karatachijōato}}）之名獲指定為日本國[[史跡|史跡]]<ref name="bkc">{{cite web |url=http://kunishitei.bunka.go.jp/bsys/maindetails.asp?register_id=401&item_id=1536 |title=国指定文化財等データベース:主情報詳細/阿坂城跡 附 高城跡 枳城跡 |publisher=[[文化廳|文化廳]] }}</ref>。

== 概要 ==
城池在《[[滿濟|滿濟准后]]日記》中寫作'''淺香城'''，而在《南方紀傳》中則寫作'''阿射賀城'''<ref name="kk"/>。城池建於大阿坂町的西邊高約312米的山上<ref name="kk"/>，面積約為東西180米乘以南北330米<ref>三重県観光連盟 監修（1989）：第179和181頁</ref>。城池的東邊可以看到[[伊勢灣|伊勢灣]]、[[伊勢志摩國立公園|伊勢志摩國立公園]]和[[三河國|三河]]的山群<ref name="fs9">三重県観光連盟 監修（1989）：第179頁</ref>，西邊則可以看到[[室生赤目青山國定公園|室生赤目青山國定公園]]的山群<ref name="fb2">ワークス 編（1997）：第52頁</ref>。

城跡分為南郭和北郭，南郭稱為白米城，北郭則稱作椎之木城<ref name="kk"/>。築城主是[[北畠親房|北畠親房]]<ref name="fb1">ワークス 編（1997）：第51頁</ref>或{{Link-ja|北畠滿雅|北畠満雅}}。城池曾經以難以攻下的城池之名以聞名，後來由奉[[織田信長|織田信長]]之命的木下藤吉郎（其後的[[豐臣秀吉|豐臣秀吉]]）拿下<ref name="fb2"/>。

== 城郭構造 ==
城跡大致分為南郭和北郭，南郭的範圍是東西25米乘以南北30米，高12米的台狀地，北郭則是70米乘以70米的{{#tag:ref|按《美杉村史》的記載<ref name="mvh">美杉村史編集委員会（1981）：第264頁</ref>。《角川日本地名大辭典》則稱城池面積是東西40至70米乘以南北150米<ref name="kk"/>。|group="注"}}台狀地，兩郭相距約250米<ref name="mvh"/>。城池的南郭稱作白米城，北郭稱為椎之木城<ref name="kk"/>。

椎之木城的規模較大而且較新，由護城河和{{Link-ja|土壘|土塁}}等組成，由於結構複雜，因此椎之木城可能是城池的主郭<ref name="hb">平凡社（1983）：第524頁</ref>。椎之木城以兩個狹長的台狀地為中心，建造護城河來包圍台狀地之間和兩側<ref name="zcj">村田 編（1987）：第254頁</ref>，其中北面的西下斜面處建有兩處豎堀<ref name="zcj"/>。相對於椎之木城，白米城處於的位置較高，能夠看到松阪市內<ref name="zcj"/>。白米城的台狀地是45米乘以60米的梯形基座，上方則是20米乘以35米的平面圓橢形，四角均有小型的台狀地<ref name="zcj"/>。

白米城的東、北和西三個方位的護城河沿[[山脊|山脊]]興建，並且建有小型的台狀地，北方和西方的山脊則設有兩條護城河，在桝形山的山頂上則建有護城河、空堀和[[狼煙_(軍事)|狼煙]]台<ref name="mvh"/>。

城池附近有高城和枳城等支城<ref name="fs1"/>，其中高城是由北畠氏家臣大宮氏建造<ref name="hb"/>，建有{{Link-ja|虎口 (出入口)|虎口|虎口}}，屬典型的日本[[戰國時代_(日本)|戰國時代]]風格建構<ref name="zcj"/>。枳城則具備日本[[南北朝時代_(日本)|南北朝時代]]的特色<ref name="fg">ふぁみんぐ編集室（2012）：卷頭特集</ref>。

== 歷史 ==
[[File:Kitabatake_Chikafusa.svg|thumb]]
[[建武_(後醍醐天皇)|建武]]2年（1335年），北畠親房與其子[[北畠顯家|北畠顯家]]、[[北畠顯信|北畠顯信]]和{{Link-ja|北畠顯能|北畠顕能}}進駐[[南勢|伊勢國南部]]後，建造了阿坂城等數處城池<ref name="fb1"/>。阿坂城首次在史料上出現是在[[文和|文和]]元年或[[正平_(後村上天皇)|正平]]7年10月23日<ref name="fs1"/>（1352年11月30日）的《鷲見加加丸軍忠狀》中，該文書在描述[[北朝_(日本)|北朝]]軍的{{Link-ja|土岐賴康|土岐頼康}}援助{{Link-ja|鷲見氏|鷲見氏}}的段落中提及了阿坂城<ref name="kk"/>。

其後[[應永|應永]]22年（1415年）春天，{{Link-ja|北畠滿雅|北畠満雅}}不滿[[稱光天皇|稱光天皇]]違犯[[明德和約|明德和約]]中定下的[[兩統迭立|兩統迭立]]而即位為由<ref name="mh1">西垣和松島（1974）：第81頁</ref>，在阿坂城舉兵<ref name="kk"/><ref name="mh1"/>。滿雅獲得原本南朝勢力下的[[大和國|大和]]、[[紀伊國|紀伊]]和[[河內國|河內]]的武士支持，而[[室町幕府|幕府]]方面則以{{Link-ja|一色義貫|一色義貫}}為總大將<ref name="mh1"/>，聯同[[土岐氏|土岐氏]]和[[京極氏|京極氏]]應戰，導致北畠軍處於守城的形勢<ref name="kk"/>，期間幕府軍將阿坂城包圍，並且截斷城內的水源<ref name="mh1"/>，導致北畠軍陷入困境<ref name="fs1"/>，而實際上城內的水源亦因而所淨無幾<ref name="fg"/>，儘管如此北畠軍為了向幕府軍營造城內水源仍然相當豐富的假象，便把以白米當成水來清潔馬匹，從以欺騙幕府軍<ref name="fs1"/><ref name="mvh"/>。因此，在遠處的幕府軍無法分辨出洗馬的其實是米而非水，誤以為城池仍然水源充足，便隨即放棄斷水的作戰，並且解除對阿坂城的包圍<ref name="mh1"/><ref name="fg"/>。最終，雖然城池被攻下，但是滿雅通過中間人的協助，成功與幕府和解<ref name="mh1"/>，這也是別稱白米城的由來<ref name="fs1"/><ref name="kk"/>。白米城傳說記載於《{{Link-ja|櫻雲記|桜雲記}}》中<ref name="fg"/>，該書寫成於[[江戶時代|江戶時代]]前期，雖然包含以{{Link-ja|軍記物|軍記物}}形式寫成的部份，但是同時也有真偽難辨的地方<ref name="fg"/>。按《滿濟准后日記》記載，城池在應永22年5月15日（1415年6月21日）被攻陷<ref name="hb"/>。其後的[[文明_(後土御門天皇)|文明]]10年（1478年），在[[北畠材親|北畠材親]]的協助下，於建有阿坂城的桝形山的山麓處建成了作為[[菩提寺_(日本)|菩提寺]]的淨眼寺<ref name="hb"/><ref name="zcj"/>。從此，北畠氏便以淨眼寺作為居所，戰鬥時才移師阿坂城，從而建構出根小屋式山城的形態<ref name="hb"/>。

白米城傳說後，再未有城池的紀錄，直至[[永祿|永祿]]8年（1565年）和[[天正_(日本)|天正]]8年（1580年），城池的名字才重現於史料<ref name="kk"/>。當時的城主由大宮入道含忍齋擔任<ref name="kk"/><ref name="mvh"/>。永祿10年（1567年），由於[[織田氏|織田氏]]家臣[[瀧川一益|瀧川一益]]再三拒絕織田信長命其攻擊城池的要求，自此便以牢不可破的城池而聞名<ref name="fb2"/>。其後的永祿12年8月（1569年9月），信長軍包圍阿坂城，雖然曾經勸含忍齋投降，但是未獲回應，後來通過木下藤吉郎（其後的豐臣秀吉）的計策才令到其開城投降<ref name="fb2"/>，然後城池成為{{Link-ja|廢城|廃城}}<ref name="fs1"/><ref name="fg"/>。期間，北畠軍的大宮大之丞吉行向木下藤吉郎放箭，導致藤吉郎的左邊腋下和左腳均受傷，也是秀吉生涯中唯一一次在戰事中負傷<ref name="fb2"/><ref name="fg"/><ref name="rj">{{cite web |url=http://www.rekishijin.jp/rekishijinblog/suzuki/%E5%8F%B2%E8%B7%A1/ |title=第8回 史跡めぐりは足腰鍛錬とカーナビ 秀吉生涯一度の戦傷・伊勢阿坂城 |author={{Link-ja|鈴木輝一郎|鈴木輝一郎}} |publisher=歴史人 |date=2011-09-09 |accessdate=2015-03-17 |archive-url=https://web.archive.org/web/20150402113146/http://www.rekishijin.jp/rekishijinblog/suzuki/%E5%8F%B2%E8%B7%A1/# |archive-date=2015-04-02 |dead-url=yes }}</ref>。進攻阿坂城是作為[[大河內城之戰|大河內城之戰]]而發動的前哨戰<ref>西垣和松島（1974）：第100頁</ref>，同時亦是信長給予當時尚未建立軍功的秀吉的一次機會<ref name="rj"/>。

1982年4月7日，城跡獲日本指定為國的史跡<ref name="bkc"/>。

== 現狀 ==
城跡現在只剩餘平坦的城郭遺跡，昔日的形態已經變得模糊<ref name="fb2"/>。護城河和土壘仍然存在，在南郭則豎立了標誌為阿坂城遺址的石碑<ref name="fg"/>。在城跡成為日本國指定史跡之前的一年開始，當地大阿坂町民開始定期除草，並且整頓城跡<ref name="fg"/>。

在[[東海旅客鐵道|JR東海]][[紀勢本線|紀勢本線]]、[[名松線|名松線]]、[[近畿日本鐵道|近鐵]][[近鐵山田線|山田線]]{{Link-ja|松阪站|松阪駅}}下車，轉乘[[三重交通|三重交通]]{{Link-ja|路線巴士|路線バス}}48號系統嬉野一志町方向巴士，於岩倉口巴士站下車，然後徒步15分鐘，便會到達前往城跡的登山口<ref name="fb1"/>，從登山口至城跡的山路，需要步行大約40分鐘<ref name="fb2"/>，慢行的話則大約花費一小時才能到達<ref name="fg"/>。

由於從城跡遠望的景色不俗，在春天的時候有不少人為此登山<ref name="fs9"/>，也有順道在附近的{{Link-ja|瑞巖寺 (松阪市)|瑞巌寺 (松阪市)|瑞巖寺}}、橫瀧寺和不動瀑布等地方散步的登山人士<ref name="fb2"/>。
