'''阿克利夫站'''（{{lang-en|Arncliffe Station}}）是[[雪梨|雪梨]][[城市鐵路|城市鐵路]]的[[東郊及伊拉瓦拉線|東郊及伊拉瓦拉線]]的一個沿線車站，它服務悉尼的的一個往宅區—[[阿克利夫_(悉尼)|阿克利夫]]。

==月台服務==
大部份時間每小時會在2班列車，星期一至五的繁忙時間會加設班次。

'''1號月台'''
 	 
* 東郊及伊拉瓦拉線 — 往邦迪聯運。

'''2號月台'''
	 
* 東郊及伊拉瓦拉線 — 往瀑布。

'''3號月台'''
  	 
* 東郊及伊拉瓦拉線 — 往邦迪聯運。

'''4號月台'''
	 
* 東郊及伊拉瓦拉線 — 往瀑布。

== 傷殘人士設施 ==

該站不設升降機等傷殘人士設施，但車站在開放時間會在職員當值。<ref>{{cite web en|work=City Rail| title=Arncliffe station facilities|url=http://www.cityrail.info/stations/station_details?stationId=72 | accessdate=28 June 2009}}</ref>

== 接駁交通 ==

* Route 471 - South to [[Rockdale_railway_station,_Sydney|Rockdale station]],  North to [[Five_Dock,_New_South_Wales|Five Dock]] via [[Turrella|Turrella]], [[Bardwell_Park|Bardwell Park]], [[Earlwood|Earlwood]], [[Canterbury,_New_South_Wales|Canterbury]], [[Ashbury,_New_South_Wales|Ashbury]], [[Ashfield,_New_South_Wales|Ashfield]], [[Haberfield|Haberfield]], [[Rodd_Point|Rodd Point]]
