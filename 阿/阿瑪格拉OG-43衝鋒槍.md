'''阿瑪格拉OG-43衝鋒槍'''（{{Lang-en|Armaguerra OG-43}}）及其以後的版本，OG-44衝鋒槍是一款在[[第二次世界大战|二战]]時期由[[意大利社会共和国|薩羅共和國]]槍械設計師喬瓦尼·奧利亞尼（Giovanni Oliani）所設計的[[冲锋枪|冲锋枪]]，僅停留在原型槍的水平，發射[[9×19mm魯格彈|9×19毫米魯格]][[口徑|口徑]][[手枪|手枪]][[子彈|子彈]]。

== 起源和發展 ==
由位於[[克雷莫納|克雷莫納]]的阿瑪格拉公司的設計師喬瓦尼·奧利亞尼（Giovanni Oliani）所研製的OG-43，是一枝受到最大的經濟性產品和最高隱蔽性的啟發以下研製的短距離推進用途[[自動火器|自動]]突擊武器。這是在意大利社會共和國緊急情況以下生產的一系列武器，並與{{Link-en|TZ-45衝鋒槍|TZ-45}}、{{Link-en|FNAB-43衝鋒槍|FNAB-43}}伊索塔弗拉斯基尼構成其中的一部分。接下來的版本，OG-44，卻具有一個更傳統的形狀。雖然該槍得到了意大利社会共和国戰爭部批准生產，但該國已到了事件的邊緣而從未成事。

== 概述 ==
OG-43具有一個在尊重當代型號的非常現代化的線條。該槍由壓制鋼製成，連[[彈匣|彈匣]]也充當了其握把。L形槍機與其復進簧在[[槍管|槍管]]上方的圓筒裡滑動。這設計可減少該槍點發期間的槍管傾斜。{{Link-en|拉機柄|Cocking handle}}在[[機匣|機匣]]的上方沿著復進簧保持圓筒滑動。[[槍托|槍托]]支架可向槍身折疊或向後方展開。該槍亦在[[扳機護環|扳機護圈]]前方、穿孔的槍管罩的下方，設置了置於前方、斜倚的[[木材|木]]製[[輔助握把|握把]]。該武器還採用了與{{Link-en|貝瑞塔M38衝鋒槍|Beretta Model 38|貝瑞塔MAB 38}}相同的雙排式10、20、30、40發可拆卸式[[彈匣|彈匣]]供彈。該槍為[[反沖作用_(槍械)|短行程反沖作用]]原理，在[[擊發調變槍械|半自動和全自動模式]]以下都是[[開放式槍機|開放式槍機]]。瞄準具固定在100公尺和200公尺＃。

=== OG-44 ===
OG-44版本具有更為傳統的線條。它採用了以[[铰链|铰链]]連接到{{Link-en|手槍握把|Pistol grip}}的傳統型木製[[槍托|槍托]]，定位在扳機和彈匣固定插槽的後方。重量比OG-43稍高，而總長度亦加長至787毫米。
