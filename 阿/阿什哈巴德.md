'''阿什哈巴德'''（[[土庫曼語|土庫曼語]]：{{lang|tk|'''Ashgabat'''}}；[[俄語|俄語]]：{{lang|ru|'''Ашхаба́д'''}}），[[土庫曼|土庫曼]]首都，人口909,000（2009年），面积约300km²。

== 歷史 ==
阿什哈巴德位於古代[[安息王國|安息王國]]最初的首都[[尼薩_(土庫曼斯坦)|尼薩]]東南方18公里處。位處於當時[[絲綢之路|絲綢之路]]的路線上，當時被稱為[[科奇卡拉|科奇卡拉]]，在公元前2世紀是一個產酒的村落。後來在公元前1世紀的一次大地震，整條村被地震所毀滅，但其優越的地理位置，使村落其後又再重建，直到13世紀[[蒙古帝國|蒙古帝國]]的興起時，連同[[花剌子模|花剌子模]]一起被毀滅。<ref>{{cite web |url=http://www.geographicbureau.com/trips/central_asia/turkmenistan/info/brief_description_of_the_main_s.jdx |title=Konjikala: the Silk Road precursor of Ashgabat |language=en |deadurl=yes |archiveurl=https://web.archive.org/web/20141029043151/http://www.geographicbureau.com/trips/central_asia/turkmenistan/info/brief_description_of_the_main_s.jdx |archivedate=2014-10-29 }}</ref><ref>{{cite book |url=http://books.google.com/books?id=UGanxmJgQNIC&pg=PA40&lpg=PA40&dq=konjikala&source=bl&ots=76-v-N-YOn&sig=_SxnsotPclSEHqnFjepj4qQhlzc&hl=en&sa=X&oi=book_result&resnum=3&ct=result#PPA41,M1 |chapter=Konjikala |first1=MaryLee |last1=Knowlton |title=Turkmenistan |publisher=Marshall Cavendish |year=2006 |pages=40-41 |isbn=978-0-7614-2014-9 }}</ref>

而「阿什哈巴德」這個名字早在19世纪中叶的历史文献出現，意思为“爱之城”，当时为土库曼人的一支[[捷詹|捷詹人]]的城堡<ref name=tmmogcom20020710 />。1881年，基於《[[阿克哈條約|阿克哈條約]]》，阿什哈巴德連同花剌子模其他地方被[[波斯|波斯]]割讓與[[沙俄|沙俄]]，沙俄立即组建[[後裡海軍區|後裡海軍區]]，並在阿什哈巴德建立要塞，作為軍區的行政中心<ref name=tmmogcom20020710 />。1885年～1889年建成通向[[裡海|裡海]]和[[塔什干|塔什干]]的铁路，发展迅速，成为俄国和伊朗的贸易中心。1925年成为[[土库曼苏维埃社会主义共和国|土库曼苏维埃社会主义共和国]]的首都<ref name=tmmogcom20020710 />。1948年10月，阿什哈巴德遭受大地震破坏<ref name=tmmogcom20020710 />，16萬人於地震中死亡<ref name=tmmogcom20020711>{{cite news |url=http://tm.mofcom.gov.cn/aarticle/ddgk/zwshoudu/200207/20020700011025.html |title=首都－阿什哈巴德 |date=2002-07-11 |accessdate=2012-09-28 |publisher=中国驻土库曼斯坦使馆经商参处 |language=zh-hans |archive-url=https://web.archive.org/web/20070822092639/http://tm.mofcom.gov.cn/aarticle/ddgk/zwshoudu/200207/20020700011025.html |archive-date=2007-08-22 |dead-url=yes }}</ref>，之後重建。1991年，土庫曼斯坦獨立，作為新國家的首都及政治、經濟、文化的新中心，阿什哈巴德大興土木，以迎合將來的位置與機遇<ref name=tmmogcom20020711 />。

== 地理 ==
阿什哈巴德是[[卡拉库姆沙漠|卡拉库姆沙漠]]中的一个[[绿洲|绿洲]]城市，位於土庫曼斯坦南部，卡拉库姆沙漠和[[科佩特山|科佩特山]]交界处<ref name=tmmogcom20020711 />。

阿什哈巴德距[[伊朗|伊朗]]边境30多公里。在[[第一次世界大戰|第一次世界大戰]]爆發後，成为了沙俄与伊朗之間的贸易重镇<ref name=tmmogcom20020710 />。

由於阿什哈巴德位處於卡拉库姆沙漠和科佩特山交界处<ref name=tmmogcom20020711 />，阿什哈巴德的北部與沙漠接壤，距離科佩特山也只有25公里，因此當地的氣候為典型[[大陆性气候|大陆性气候]]<ref name=tmmogcom20020710 />的[[沙漠气候|沙漠气候]]，夏季漫长，炎热干燥；冬季短暂，温和而少雪。昼夜温差较大，日照极为充分<ref name=tmmogcom20020710 />。全年的總降雨量也只是227毫米，當中以三月和四月最潮濕<ref>{{cite web |url=http://www.weatherbase.com/weather/weather.php3?s=8883 |title=Historical Weather for Ashgabat, Turkmenistan |publisher=weatherbase.com |accessdate=2008-12-18 }}</ref>。

=== 人口 ===
阿什哈巴德人口結構主要為[[土庫曼族|土庫曼族]]，其次則為[[俄羅斯人|俄羅斯人]]、[[亞美尼亞人|亞美尼亞人]]與[[亞塞拜然族|亞塞拜然族]]。
2000年10月的人口有63.7万<ref name=tmmogcom20020710 />，2009年統計有90.9萬人。每年人均人口增長約為5%<ref name=tmmogcom20020802>{{cite web |url=http://tm.mofcom.gov.cn/aarticle/ddgk/zwrenkou/200208/20020800034971.html |title=土库曼人口增长率达5% |date=2002-08-02 |accessdate=2012-09-28 |language=zh-hans }}</ref>。

== 交通 ==
[[外裡海鐵路|外裡海鐵路]]在本市設站。[[薩帕爾穆拉特·尼亞佐夫國際機場|薩帕爾穆拉特·尼亞佐夫國際機場]]位在本市約10公里處。[[土庫曼航空|土庫曼航空]]的總部也設於本市。
