'''阿拉伯航空'''，或译'''阿拉比亚航空'''（{{lang-ar|العربية للطيران}}），是一家总部设在[[阿联酋|阿联酋]][[沙迦|沙迦]]的[[廉价航空公司|廉价航空公司]]，是[[阿拉伯航空运输组织|阿拉伯航空运输组织]]的成员。该公司以[[沙迦国际机场|沙迦国际机场]]和[[穆罕默德五世国际机场|穆罕默德五世国际机场]]为[[枢纽机场|枢纽]]，经营[[中东|中东]]、[[北非|北非]]、[[印度次大陆|印度次大陆]]、[[中亚|中亚]]和[[欧洲|欧洲]]等地共计51多个目的地的航线。

== 历史与发展 ==
2003年2月，沙迦管理者、阿联酋最高委员会成员，谢赫·苏丹·本·穆罕默德·卡西米颁布[[埃米尔|埃米尔]]法令，成立了中东地区首家廉价航空公司——阿拉伯航空。<ref>{{cite web |url=http://www.thaindian.com/newsportal/business/middle-east-set-for-low-cost-aviation-boom_10030279.html |title=Middle East set for low cost aviation boom |publisher=Thaindian News |date=2008年3月23日 |accessdate=2009年3月10日 |language=en }}</ref>阿拉伯航空在2003年10月28日开始运营，首飞航线为沙迦至[[巴林|巴林]]。2007年初，阿拉伯航空成功面向公众发售其55%的股份。<ref>{{En icon}}{{cite journal en |date=2007年3月27日 |title=Directory: World Airlines |journal=Flight International |pages=52 }}</ref><ref name="history">{{cite web |url=http://www.airarabia.com/timeline.html |title=Air Arabia Timeline - Corporate History |publisher=阿拉伯航空 |date= |accessdate=2009年3月10日 |language=en |deadurl=yes |archiveurl=https://web.archive.org/web/20090217072052/http://airarabia.com/timeline.html |archivedate=2009年2月17日 }}</ref>

2009年4月29日，摩洛哥阿拉伯航空在[[摩洛哥|摩洛哥]][[达尔贝达|达尔贝达]]成立，宣布自同年5月起以穆罕默德五世国际机场为枢纽，经营达尔贝达与若干欧洲城市之间的航线。<ref name="maroc">{{cite web |url=http://www.airarabia.com/AirArabiaMarocOfficiallyLaunches.html |title=Air Arabia Maroc officially launches |publisher=阿拉伯航空 |date=2009年4月29日 |accessdate=2009年4月30日 |language=en |deadurl=yes |archiveurl=https://web.archive.org/web/20090503085745/http://www.airarabia.com/AirArabiaMarocOfficiallyLaunches.html |archivedate=2009年5月3日 }}</ref>摩洛哥阿拉伯航空由阿拉伯航空与摩洛哥当地的一家私营航空公司[[摩洛哥皇家快运航空|摩洛哥皇家快运航空]]以及巴林的Ithmaar银行共同合作创办、阿拉伯航空进行管理，专门经营阿拉伯航空的北非与欧洲之间的航线。<ref name="maroc"/><ref>{{cite web |url=http://www.ameinfo.com/137523.html |title=Air Arabia to open second hub in Morroco to serve Europe, Middle East and Africa |publisher=ameinfo |date=2007年11月8日 |accessdate=2009年3月10日 |language=en |deadurl=yes |archiveurl=https://web.archive.org/web/20090324073553/http://www.ameinfo.com/137523.html |archivedate=2009年3月24日 }}</ref>

[[南亚|南亚]]方面，阿拉伯航空以[[加德满都|加德满都]]为新的次要枢纽，为该地区与中东之间的航线服务，并与[[尼泊尔|尼泊尔]]的[[雪人航空|雪人航空]]签署了合资协议。根据协议，两家公司合资创办一家新的廉价航空公司，称为“Fly Yeti”。受尼泊尔动荡的政局影响，Fly Yeti自2008年暂停营业至今。<ref>{{cite web |url=http://www.aviationrecord.com/news-articles.aspx?articleType=ArticleView&articleId=700 |title=Nepal's Fly Yeti to suspend flights |publisher=AviationRecord |date=2008年6月30日 |accessdate=2009年3月10日 |language=en }}{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

在2008年和2009年连续两届的[[Skytrax|Skytrax]]全球最佳廉价航空公司评选中，阿拉伯航空均位居中东地区第一位。<ref>{{cite web |url=http://www.worldairlineawards.com/Awards_2008/Lowcost-08.htm |title=EASYJET WINS AWARD FOR WORLD'S BEST LOW-COST AIRLINE |publisher=[[Skytrax|Skytrax]] |date= |accessdate=2009年3月10日 |language=en |deadurl=yes |archiveurl=https://web.archive.org/web/20090305051144/http://www.worldairlineawards.com/Awards_2008/Lowcost-08.htm |archivedate=2009年3月5日 }}</ref><ref>{{cite web |url=http://www.worldairlineawards.com/Awards_2009/Lowcost-09.htm |title=AIR ASIA WINS AWARD FOR WORLD'S BEST LOW-COST AIRLINE |publisher=[[Skytrax|Skytrax]] |date= |accessdate=2009年4月30日 |language=en |deadurl=yes |archiveurl=https://web.archive.org/web/20090426093107/http://www.worldairlineawards.com/Awards_2009/Lowcost-09.htm |archivedate=2009年4月26日 }}</ref>

== 目的地 ==
2003年10月28日，阿拉伯航空成功首航巴林首都[[麦纳麦|麦纳麦]]之后，又在当月先后开通沙迦至[[大马士革|大马士革]]、[[科威特城|科威特城]]、[[马斯喀特|马斯喀特]]、[[贝鲁特|贝鲁特]]等四条航线。2004年上半年，阿拉伯航空将自己的网络扩展到[[卡塔尔|卡塔尔]]、[[埃及|埃及]]、[[斯里兰卡|斯里兰卡]]、[[苏丹|苏丹]]、[[沙特阿拉伯|沙特阿拉伯]]等国，年底开通了至[[也门|也门]]首都[[萨那|萨那]]的航线。随后几年，其通航目的地均逐年增长。<ref name="route">{{cite web |url=http://www.airarabia.com/timeline2.html |title=Air Arabia Timeline - Route Launches |publisher=阿拉伯航空 |date= |accessdate=2009年3月10日 |language=en |deadurl=yes |archiveurl=https://web.archive.org/web/20090830221121/http://www.airarabia.com/timeline2.html |archivedate=2009年8月30日 }}</ref>截至2014年2月，阿拉伯航空已经通达28个国家的52个目的地（含部分即将开通航线）。<ref name="destinations">{{cite web |url=http://www.airarabia.com/destinations.html |title=Destinations |publisher=阿拉伯航空 |date= |accessdate=2009年4月30日 |language=en |deadurl=yes |archiveurl=https://web.archive.org/web/20090311022839/http://airarabia.com/destinations.html |archivedate=2009年3月11日 }}</ref>
