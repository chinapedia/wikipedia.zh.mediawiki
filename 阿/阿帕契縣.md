'''阿帕奇縣'''（{{lang-en|Apache County}}）是[[美國|美國]][[亞利桑那州|亞利桑那州]]東北部的一個縣，東鄰[[新墨西哥州|新墨西哥州]]，北鄰[[猶他州|猶他州]]，東北鄰[[科羅拉多州|科羅拉多州]]。面積29,056平方公里，是州中面積第三大的縣份，亦是[[美國面積最大縣份列表|美國面積第六大縣份]]。根據[[美國2010年人口普查|2010年人口普查]]，阿帕契縣共有人口71,518人。阿帕契縣的縣治為[[聖約翰斯_(亞利桑那州)|聖約翰斯]]。

阿帕契縣成立於1879年2月24日，得名於當地的[[印第安人|印第安人]]部落[[阿帕契族|阿帕契族]]。<ref>{{英文}}[http://www.arizonagenealogy.com/apache/ Apache County, Arizona History]</ref>

===2010年===
根據[[2010年美國人口普查|2010年人口普查]]，阿帕契縣擁有71,518居民。阿帕契縣的人口是由23.3%[[歐裔美國人|白人]]、0.2%[[非裔美國人|黑人]]、72.9%[[美國土著|土著]]、0.3%[[亞裔美國人|亞洲人]]、0.0%[[太平洋岛民|太平洋岛民]]、1.3%其他[[種族|種族]]和2.0%[[混血|混血]][[种族构成|构成]]。而[[西班牙裔美国人|西班牙裔]]或[[拉丁美洲人|拉丁美洲人]]佔了人口5.8%。<ref>{{英文}}[http://www2.census.gov/census_2010/01-Redistricting_File--PL_94-171/California/ "2010 Census P.L. 94-171 Summary File Data". ] United States Census Bureau</ref>阿帕契縣是美國少數土著人口比白人人口多的縣份。

===2000年===
根據[[2000年美國人口普查|2000年人口普查]]{{GR|2}}，阿帕契縣擁有69,423居民、19,971住戶和15,257家庭。<ref>{{en}}[http://www.census.gov/population/www/pop-profile/profile2000.html Population Profile of the United States: 2000]</ref>。其[[人口密度|人口密度]]為每平方英里6居民（每平方公里2居民）。阿帕契縣擁有31,621間房屋单位，其密度為每平方英里3間（每平方公里1間）。阿帕契縣的人口是由19.50% [[歐裔美國人|白人]]、0.25%[[非裔美國人|黑人]]、76.88%[[美國土著|土著]]、0.13%[[亞裔美國人|亞洲人]]、0.06%[[太平洋岛民|太平洋岛民]]、1.75%其他[[種族|種族]]和1.43%[[混血|混血]][[种族构成|构成]]。而[[西班牙裔美国人|西班牙裔]]或[[拉丁美洲人|拉丁美洲人]]佔了人口4.49%<ref>{{en}}[http://www.census.gov/main/www/cen2000.html Census 2000 gateway]</ref>。阿帕契縣的居民中，有58.39%母語為[[納瓦荷語|納瓦荷語]]、38.39%為[[英語|英語]]以及2.71%為[[西班牙語|西班牙語]]<ref>{{英文}}[http://www.mla.org/map_data_results&state_id=4&county_id=1&mode=geographic&order=r MLA Language Map Data Center]</ref>。阿帕契縣是主要語言非英語的38個縣份之一。阿帕契縣亦是主要語言既非英語，亦非西班牙語的3個縣份之一，更是州中唯一一個第一語言並非英語或西班牙語的縣份。<ref>{{英文}}[http://www.mla.org Modern Language Association]</ref>

在19,971住户中，有43.80%擁有一個或以上的兒童（18歲以下）、49.30%為夫妻、21.40%為單親家庭、而有23.60%為非家庭。其中21.20%住户為獨居，6.90%為同居長者。平均每戶有3.41人，而平均每個家庭則有4.04人。在69,423居民中，有38.50%為18歲以下、9.40%為18至24歲、25.10%為25至44歲、18.70%為45至64歲以及8.30%為65歲以上。人口的年齡中位數為27歲，每100個女性就有98.20個男性。而每100個成年女性（18歲以上）就有94.50個成年男性。<ref name="Census">{{英文}}{{cite web |url=http://2010.census.gov/2010census/data/apportionment-pop-text.php |title=Resident Population Data: Population Change |date=2010-12-23 |publisher=United States Census Bureau |accessdate=2010-12-23 |archiveurl=https://web.archive.org/web/20101225031104/http://2010.census.gov/2010census/data/apportionment-pop-text.php |archivedate=2010-12-25 |deadurl=no }}</ref>

阿帕契縣的住戶收入中位數為$23,344，而家庭收入中位數則為$26,315。男性的收入中位數為$30,182，而女性的收入中位數則為$22,312。阿帕契縣的[[人均收入|人均收入]]為$8,986。約33.50%家庭和37.80%人口在[[貧窮線|貧窮線]]以下。<ref name="Census"/>

於2000年，阿帕契縣最多人相信的宗教為[[天主教|天主教]]（19,965人）<ref name="thearda">{{英文}}{{cite web |url=http://www.thearda.com/mapsReports/reports/counties/04001_2000_Adherents.asp|title=County Membership Reports |accessdate=2011-08-22 |format= |work= thearda.com }}</ref>。而最大的宗教機構則分別為[[天主教會|天主教會]]（19,965成員）和[[耶穌基督後期聖徒教會|耶穌基督後期聖徒教會]]（8,947成員）。<ref name="thearda"/>
