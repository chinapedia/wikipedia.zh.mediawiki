'''阿布·努瓦斯'''（{{lang-ar|ابونواس}}；{{lang-fa|ابونواس}}；[[拉丁化|拉丁化]]：{{unicode|Abū-Nuwās}}<ref name="Gp2">[[#GA|Garzanti]]</ref>，756年-814年<sup id="fn_a_back">[[#fn_a|a]]</sup> ）是[[阿拉伯|阿拉伯]][[詩人|詩人]]。

生于[[波斯|波斯]][[阿瓦士|阿瓦士]]，在[[巴士拉|巴士拉]]长大，有[[波斯|波斯]]和[[阿拉伯|阿拉伯]]血統。幼时曾随香料商学艺，但志在诗文。后在[[巴士拉|巴士拉]]和[[库法|库法]]求学，先后师从诗人[[瓦利卜·伊本·侯巴卜|瓦利卜·伊本·侯巴卜]]和[[海莱夫·艾哈迈尔|海莱夫·艾哈迈尔]]。他研读过《古兰经》、圣训，通晓阿拉伯文法、修辞、诗律。初到巴格达[[阿拔斯王朝|阿拔斯王朝]]王宫时，与某些大臣多有牵连，在政局发生突变时潜往埃及避难。30岁时辗转返回[[巴格达|巴格达]]，撰写颂诗多篇，博得[[哈伦·拉希德|哈伦·拉希德]]和[[阿布·阿卜杜拉·穆罕默德·阿明·本·哈伦|艾敏]]两位哈里发的赏识，成为宫廷诗人。长于作饮酒歌，有“酒诗人”之称<REF name="咏酒诗人"/>。思想豁朗，性格豪放，主张充分享受现实人生的欢乐，反对宗教禁欲和苦行。阿布·努瓦斯也寫了一些給年輕男子的情詩<REF>

诗作题材广泛，除歌颂阿拔斯朝廷和劝善的诗篇外，尤以赞美青春、美酒、爱情的抒情诗和描写宫殿、花园的写景诗以及调侃权贵、嘲弄群小的讽刺诗最为著名。语法严谨，辞藻优美，热情奔放，富于想象，诙谐生动，突破了传统诗歌的题材和形式，为阿拉伯诗歌的发展开辟了新天地，对当时和后世的诗人影响很大。阿拉伯文学评论界认为他是阿拉伯文学黄金时代最伟大的新派诗人。遗留的诗篇经10世纪文学家{{le|伊斯巴哈尼|Abu al-Faraj al-Isfahani}}搜集整理，传世的有12000多句<ref name="咏酒诗人">

== 早年及作品 ==
阿布·努瓦斯的父亲哈尼（Hani）是阿拉伯人，是[[吉赞省|吉赞省]]Banu Hakam部落的后裔，是{{link-en|迈尔万二世|Marwan II}}军队中的一名士兵。努瓦斯从来没有见过自己的父亲。他的波斯裔母亲哥爾巴（Golban）是一名裁缝。对于阿布·努瓦斯的出生年份，说法不一，跨度从747年至762年。有人说他生于[[巴士拉|巴士拉]]，<ref name="Gp2"/>但也有称其生于大马士革。他的[[阿拉伯人名|名字]]为“al-Hasan ibn Hani al-Hakami”，阿布·努瓦斯只是他的昵称，意为“有长发穗的父亲”，指其两侧垂至肩膀的头发。

阿布·努瓦斯还处在少年时期时，他的母亲把他卖给了[[巴士拉|巴士拉]]的一个杂货商Sa’ad al-Yashira。[[巴士拉|巴士拉]]搬到[[巴格达|巴格达]]居住，同行的可能还有Walibah ibn al-Hubab。

== 放逐和入狱 ==
由于撰写了一篇悼念被[[哈里发|哈里发]][[哈伦·拉希德|哈伦·拉希德]]清剿的知名波斯政治家族[[巴爾馬克家族|巴爾馬克家族]]的诗歌，阿布·努瓦斯被迫逃到[[埃及|埃及]]躲避一段时间。809年，在哈伦·拉希德死后，他回到巴格达。随后，哈伦·拉希德22岁的儿子穆罕默德·[[阿布·阿卜杜拉·穆罕默德·阿明·本·哈伦|阿明]]（阿布·努瓦斯过去的门生）地位的上升对于阿布·努瓦斯来说是十分幸运的。许多学者认为，阿布·努瓦斯的大部分诗歌都撰写与阿明在位期间（809年-813年）。

尽管如此，阿布·努瓦斯酗酒成性、放荡不羁的作为最终还是让阿明忍无可忍，他因此锒铛入狱。阿明后来被自己的弟弟[[阿布·阿拔斯·阿卜杜拉·马蒙·本·哈伦|马蒙]]篡位，而马蒙对于阿布·努瓦斯没有任何耐心。

据称，马蒙的秘书Zonbor趁着阿布·努瓦斯醉酒时，欺骗他撰写了一篇嘲讽[[阿里·本·阿比·塔利卜|阿里·本·阿比·塔利卜]]（第4任[[正统哈里发|正统哈里发]]，先知[[穆罕默德|穆罕默德]]的堂弟及女婿）的作品。之后，Zonbor故意当众大声朗读了这篇诗歌，使得努瓦斯永无翻身之地。根据来源不同，阿布·努瓦斯可能死于狱中，或被Ismail bin Abu Sehl毒死<ref>

== 影响 ==

《巴格达历史》一书作者Al-Khatib al-Baghdadi在书中记载，阿布·努瓦斯埋葬于巴格达的Shunizi墓园。<ref>{{cite book|url=http://books.google.com/books?id=5DlbAAAAQAAJ&lpg=PA394&ots=4twy8GhFuH&dq=Abu%20Nuwas%20buried%20cemetery&pg=PA394#v=onepage&q=Abu%20Nuwas%20buried%20cemetery&f=false |title=Ibn Khallikan's biographical dictionary - Google Books |publisher=Books.google.com |date= |accessdate=2010-09-12}}</ref>

阿布·努瓦斯被认为是古典阿拉伯文学的一位巨匠，他影响到后世的许多作家，例如波斯詩人[[欧玛尔·海亚姆|欧玛尔·海亚姆]]及[[哈菲茲_(詩人)|哈菲茲]]。在众多《[[一千零一夜|一千零一夜]]》故事中嘲讽了他的享乐主义。

他百无禁忌的言论表达，尤其是在一些被伊斯兰传统所禁止的问题上，激励着人们反对审查的斗争。尽管在20世纪早年以前，他的作品都是可以自由流通的，1932年，第一部现代删减版阿布·努瓦斯作品集出现在开罗。在2001年1月，埃及文化部命令燒毀六千份阿布·努瓦斯{{le|同性性欲|homoerotic}}詩歌的複本<ref>''Al-Hayat'', 2001-01-13</ref><ref>''Middle East Report'', 219 Summer 2001</ref>

== 斯瓦希里文化 ==

在东非的[[斯瓦希里文化|斯瓦希里文化]]中，阿布·努瓦斯的名字被写作Abunuwasi，十分流行。在该文化中，这个名字出现在许多有关伊斯兰社会的民谣或文学作品中，在其他地方的类似故事中可能会用[[纳斯尔丁|纳斯尔丁]]、Guha或者“毛拉”（对老师的尊称）。在这些故事中，阿布·努瓦斯会智斗贪婪的财主，救助穷苦的民众。

[[坦桑尼亚|坦桑尼亚]]艺术家Godfrey Mwampembwa创作了名为《Abunuwasi》的[[斯瓦希里语|斯瓦希里语]]漫画书，这是改编自三篇有关阿布·努瓦斯的故事的。<ref name="Pilcher">Pilcher, Tim and Brad Brooks. (Foreword: Dave Gibbons). ''The Essential Guide to World Comics''. ''Collins and Brown''. 2005. 297.</ref>此書已由Sasa Sema出版社在1996年出版<ref>[http://www.amazon.com/Abunuwasi-Gado/dp/9966960902 Amazon.com: Abunuwasi (Swahili Edition) (9789966960900): Gado: Books<!-- Bot generated title -->]</ref>。
