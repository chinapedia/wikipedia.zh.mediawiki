'''阿卜德卡德·穆罕默德·格扎尔'''（{{lang-ar|عبد القادر غزال}}，{{bd|1984年|12月5日|}}）是一名在法国出生的[[阿尔及利亚人|阿尔及利亚裔]]足球运动员，司职前锋，目前效力[[意大利足球甲级联赛|意大利足球甲级联赛]]的[[巴里体育俱乐部|巴里]]<ref>{{cite web |url=http://www.dzfoot.com/article-1330.php |title=存档副本 |accessdate=2010-07-16 |deadurl=yes |archiveurl=https://web.archive.org/web/20100409044223/http://www.dzfoot.com/article-1330.php |archivedate=2010-04-09 }}</ref>。

== 俱乐部生涯 ==
格扎尔在[[法国|法国]]的一些小规模足球俱乐部开始他的职业生涯。在2005年，他搬到[[意大利|意大利]]居住，并加入[[克罗托内足球俱乐部|克罗托内]]。在那里他突破了自己，在[[意大利足球丙一级联赛|意大利足球丙一级联赛]]的[[2007年至2008年意大利足球丙一級聯賽|07至08赛季]]为球队攻入20球，协助球队升上[[意大利足球乙级联赛|意大利足球乙级联赛]]。

在2008年6月13日，格扎尔和[[意大利足球甲级联赛|意大利足球甲级联赛]]的[[热那亚板球与足球俱乐部|热那亚]]。其后，俱乐部马上把他外借到[[意大利足球乙级联赛|意大利足球乙级联赛]]的球队。为此，他在2008年7月2日转投另一间意甲俱乐部[[锡耶纳足球俱乐部|锡耶纳]]。当时，两间俱乐部各有加扎尔一半的拥有权。

在2010年7月1日，格扎尔和[[巴里体育俱乐部|巴里]]签订4年合约。<ref>[http://www.barilive.it/sport/news.aspx?idnews=17768 Pulzetti e Ghezzal sono del Bari]</ref>

== 国家队生涯 ==
在2008年10月23日，在[[锡耶纳足球俱乐部|锡耶纳]]的官方网站中，格扎尔宣布在和[[阿尔及利亚国家足球队|阿尔及利亚国家足球队]]主教练[[拉巴·萨阿丹|拉巴·萨阿丹]]倾谈过后，他决定代表阿尔及利亚在国际比赛中上阵。由于格扎尔在法国出生，因此他也可以选择代表[[法国国家足球队|法国]]上阵。<ref>http://www.acsiena.it/newsdetail.php3?id=6738</ref>

在2008年11月12日，格扎尔应主教练拉巴·萨阿丹的征召，在2008年11月18日一场于[[法国|法国]][[鲁昂|鲁昂]]举行的友谊赛中上阵，对手是[[马里国家足球队|马里]]。<ref>{{cite web |url=http://www.faf.org.dz/equipe_nationale-2008/point%20de%20presse%20Saadane%20121108.htm |title=存档副本 |accessdate=2008-11-16 |0=2009-02-26 |deadurl=yes |archiveurl=https://archive.is/20090226111702/http://www.faf.org.dz/equipe_nationale-2008/point%20de%20presse%20Saadane%20121108.htm |archivedate=2009-02-26 }}</ref>

在2009年2月11日，格扎尔射入他在国家队的首个入球，协助国家队以2-1击败[[贝宁国家足球队|贝宁]]。

在[[2010年世界杯足球赛|2010年世界杯足球赛]]的[[2010年世界杯足球赛决赛圈C组|分组赛C组]]中，格扎尔上阵了2次。不过在对[[斯洛文尼亚国家足球队|斯洛文尼亚]]的一仗中，他仅仅上阵了14分钟就领了两黄一红牌被驱逐离场。<ref>{{cite web
