'''阿桑'''（{{bd|1975年|2月28日|2009年|4月6日|catIdx=A阿桑}}），本名'''黃嬿璘'''，[[台灣|台灣]]女歌手。2009年4月6日上午8時30分，阿桑因[[乳癌|乳癌]]末期，病逝於臺北縣新店市（今[[新北市|新北市]][[新店區|新店區]]）慈濟醫院，得年34歲。

==生平==
=== 早年生活 ===
阿桑本名黃嬿璘，年幼家贫，小学六年级时父母离异。读国中时阿桑开始住在姨妈<ref>{{cite web | url=http://tc.wangchao.net.cn/baike/detail_1890386.html | title=[阿桑] 我對大家的感謝 汪一平 | publisher=批踢踢實業坊  -  A-Sun看板 | accessdate=2009年4月7日}}</ref>家。为减轻姨妈家的负担，阿桑一直帮忙做手工补贴家用，从折塑膠花、包巧克力包装纸，到给电子工厂拼凑主机板零件等各种工作都做过。<ref name="mies " >{{cite web |url=http://music.yule.sohu.com/20090408/n263257552_1.shtml |title=歌手阿桑走得很安静 身世和歌声一样沧桑 |accessdate=2009年4月8日 |work=|publisher=搜狐网 |language=zh-hans}}</ref><ref>{{cite web |url=http://music.yule.sohu.com/20090407/n263234109.shtml |title=阿桑身世和歌声都沧桑 从小和爸妈躲债生活 |accessdate=2009年4月7日 |work=|publisher=搜狐网 |language= zh-hans}}</ref> 阿桑后来说：“从我懂事开始，父母就不停地吵架。厌烦透了这种不快乐的生活，我特别想逃避。我找到的逃避方法就是唱歌，这让单亲家庭的我不那么压抑。”， <ref name="wnly " >{{cite web |url=http://ent.163.com/ent/editor/music/050601/050601_417861.html |title=阿桑：我和刘若英完全不同 我的坚强来自真实 |accessdate=2005年6月1日 |work=|publisher= 网易|language=zh-hans}}</ref>“在我的人生中，对我最好的就是母亲，从很小的时候，我就想，我该如何回报我的母亲呢，后来我在[[酒吧|酒吧]]中唱歌，我开口唱的那一刻，脑海中就会出现万人听我唱歌的场面，于是我知道我的梦想就是成为一个非常有名的[[歌手|歌手]]，让我的母亲明白，她有一个值得骄傲的女儿。”<ref name="wnly " /> 她常在外賺錢貼補家計<ref>{{cite web| url=http://www.yn.xinhuanet.com/ent/2009-04/07/content_16176699.htm| title=歌迷忆阿桑：愿天堂里有你一片叶子| publisher=新华网| date=2009年4月7日| accessdate=August 6, 2011| deadurl=yes| archiveurl=https://web.archive.org/web/20140718032151/http://www.yn.xinhuanet.com/ent/2009-04/07/content_16176699.htm| archivedate=2014年7月18日}}</ref>。

=== 事業開始 ===
天生愛表演的阿桑，[[高中|高中]]選讀了[[南強工商|南強工商]]影劇相關科系，畢業後考入藝工隊。在藝工隊待了五年，和孫協志是同事，離開之後和朋友的朋友，一些原本不認識的人，組樂團，到[[酒吧|酒吧]]去打拚<ref name="mies " /><ref>{{cite web| url=http://www.yn.xinhuanet.com/ent/2009-04/08/content_16188129.htm| title=阿桑走的很安靜　成長舊照曝光| publisher=新華網| date=2009年4月8日| accessdate=August 6, 2011| deadurl=yes| archiveurl=https://web.archive.org/web/20111104040901/http://www.yn.xinhuanet.com/ent/2009-04/08/content_16188129.htm| archivedate=2011年11月4日}}</ref>。阿桑在[[酒吧|酒吧]]認識了[[經理人|經理人]]的[[汪一平|汪一平]]，她說：「我在酒吧唱歌的時候，認識了我現在經理人的助理，因為他的緣故，我簽了經理人合約。雖然我的經理人對我很好，但我們錄的Demo帶找不到唱片公司發行，拖了很久，當時很多人都對我講，『阿桑，你還是換個工作吧。』」<ref name="wnly " /> 阿桑曾考慮轉行，後來被經理人勸阻作罷。<ref name="wnly " /> 

2003年夏，她於[[華研國際音樂|華研唱片]]發行的「薔薇之戀」電視原聲帶中以片尾曲「葉子」一曲初試啼聲，略帶滄桑的磁性嗓音吸引了觀眾的注目。此時阿桑出道的年齡已經28歲。<ref>{{cite web |url=http://music.yule.sohu.com/20090407/n263238578.shtml |title= 台女歌手阿桑患乳癌病逝 原本今年打算結婚|accessdate=2009年4月7日 |work=|publisher=搜狐網 |language= zh-hans}}</ref> 藝名的由來是因為唱片公司為了要幫她取個比本名更好叫的小名，而唱片公司的人一開始聽到她的歌聲時，就覺得非常滄桑感人，從她的歌聲中想到了「桑」所以幫她取了這個藝名。<ref name="asvm " >{{cite web |url=http://ent.163.com/ent_2003/editor/news/star/040301/040301_224924.html |title=磁性新女聲：阿桑 |accessdate= |work=|publisher=網易 |language=zh-hans}}</ref> 同時在台灣紅起來之後大多數人稱她為「療傷歌手」。同年底發行首張個人專輯《[[受了點傷|受了點傷]]》，獲得2004年第十五屆[[金曲獎|金曲獎]]「最佳新人」入圍。2005年初發行第二張專輯《[[寂寞在唱歌|寂寞在唱歌]]》，2006年底與[[華研國際音樂|華研唱片]]合約期滿。2006年11月赴往中國大陆發展演藝事業。

=== 因病逝世 ===
2008年10月發現罹患乳癌末期。2009年4月6日上午8時30分，阿桑因[[乳癌|乳癌]]末期病逝於[[新北市|臺北縣]][[新店區|新店]][[佛教慈濟綜合醫院|慈濟醫院]]，得年34歲<ref>{{cite web | url=http://www.yn.xinhuanet.com/newscenter/2009-04/08/content_16189550.htm | title=歌手阿桑患癌去世　让歌迷唏嘘不已 | date=2009年4月8日 | accessdate=August 6, 2011 | deadurl=yes | archiveurl=https://web.archive.org/web/20090412232426/http://www.yn.xinhuanet.com/newscenter/2009-04/08/content_16189550.htm | archivedate=2009年4月12日 }}</ref>。阿桑逝世之时，美国华侨男友未能来得及见其最后一面，阿桑遗嘱称想要出书。<ref>{{cite web |url=http://music.yule.sohu.com/20090408/n263257684.shtml |title=阿桑遗愿想出书 前东家协办后事 |accessdate=2009年4月8日 |work=|publisher=搜狐 |language= zh-hans}}</ref> 阿桑的遗体告别仪式，[[S.H.E|S.H.E]]、师兄[[张智成|张智成]]、[[动力火车|动力火车]]、[[华研唱片|华研唱片]]公司都送去花篮表示哀悼。阿桑遺體火化后长眠于[[新北市|新北市]][[万里區|万里區]]的天祥宝塔。

==音樂經歷==
===專輯===
因演唱電視劇[[薔薇之戀|薔薇之戀]]片尾曲《葉子》走紅的阿桑，憑藉著淒美的曲風、動人的旋律，加上本身獨具穿透力的樂聲，立刻唱紅樂壇。特別是正式發表專輯作品《受了點傷》時，再次將此曲重新編曲，並找來了[[薔薇之戀|薔薇之戀]]導演瞿友寧與劇中擔任主角角色之一的[[任家萱|任家萱]]與[[鄭元暢|鄭元暢]]演出近九分鐘長的音樂錄影帶。這首歌至今仍是樂壇傳唱度極高的一首阿桑代表作。

阿桑的《[[寂寞在唱歌|寂寞在唱歌]]》在[[中国大陆|中国大陆]]销售不错，一上市就突破了15万的大关。阿桑说：“我第一张专辑的主打歌叫《[[受了点伤|受了点伤]]》，那其中蕴含的全部都是我的经历，我觉得在我发第一张专辑以前，真的是很不幸，我也不知道为什么自己的生活会那么不顺利！”<ref name="wnly " />  虽然父母离异很久了，母亲再嫁，但是母亲始终都关爱着阿桑，“我的每一首歌母亲都会认真地听，给我提意见，这一次我推出《[[寂寞在唱歌|寂寞在唱歌]]》，母亲也给了我很多的指导。我能有现在的成绩，都是母亲在背后支持我的缘故。”<ref name="wnly " /> 

《[[叶子|叶子]]》被[[中国|中国]]的[[超级女声|超级女声]]选手[[韩真真|韩真真]]翻唱，唱红了阿桑在中國歌迷心中的地位。

*2003年11月《受了點傷》
*2005年2月《寂寞在唱歌》

阿桑去世前留下两首她生前创作的歌曲《[[爱情美的样子|爱情美的样子]]》和《[[双鱼女子|双鱼女子]]》，两首遗作由阿桑生前的好友兼助手[[张湘怡|张湘怡]]演唱。<ref>{{cite web| url=http://www.yn.xinhuanet.com/ent/2009-04/09/content_16204455.htm| title=阿桑遗作《爱情美的样子》即将公开| publisher=新华网| date=2009年4月9日| accessdate=August 6, 2011| deadurl=yes| archiveurl=https://web.archive.org/web/20111104094138/http://www.yn.xinhuanet.com/ent/2009-04/09/content_16204455.htm| archivedate=2011年11月4日}}</ref>。

== 评价 ==
阿桑自谓：“我很喜欢[[刘若英|刘若英]]，但是我从来没有想过歌风会和她一样，因为我们的生活背景、成长经历是完全不同的，我只是唱出了我真实的情感历程而已。”，<ref name="wnly " /> “人生是很难的，但不管怎样都还要走下去！我希望我的感受可以被人听到，也希望这种经历成为对同道中人的鼓励！”<ref name="wnly " /> 

阿桑弟弟(黃俊仁)：“她很坚强很勇敢。”<ref>{{cite web |url=http://v.sohu.com/20090408/n263251375.shtml |title= 视频：阿桑乳腺癌病逝 其弟：她很勇敢很坚强|accessdate=2009年4月8日 |work=|publisher=搜狐网 |language= zh-hans}}</ref>

[[巫启贤|巫启贤]]：“阿桑是个调皮女孩，带着傻劲，希望她在天堂一切都好。”<ref name="mies " />

[[叶蓓|叶蓓]]：“阿桑在[[北京|北京]]居住房屋都是自己帮找的，因为她是一个善良的女孩，一位真正喜欢音乐的女孩，是一位知道感恩的女孩，所以大家都愿意帮助她。她笑的总是那么甜美。”<ref>{{cite web |url=http://music.yule.sohu.com/20090406/n263227728.shtml |title= 叶蓓哭泣阿桑：经常网上聊天 笑的总是那么甜美|accessdate= 2009年4月6日|work=|publisher= 搜狐网|language= zh-hans}}</ref>

[[黑楠|黑楠]]：“她很追求音乐。阿桑不是随随便便对待自己的人。”<ref>{{cite web |url= http://ent.sina.com.cn/y/2009-04-06/23062458240.shtml|title=黑楠追忆阿桑：她一直很努力在追求音乐 |accessdate=2009年4月6日 |work=|publisher=新浪网 |language= zh-hans}}</ref>
