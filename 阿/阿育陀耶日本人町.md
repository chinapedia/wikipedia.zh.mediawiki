'''阿育陀耶日本人町'''（{{jpn|j=アユタヤ日本人町|rm=Ayutaya Nihonjin-machi}}；[[泰文|泰文]]：หมู่บ้านญี่ปุ่น）位於[[泰國|泰國]]中部的[[阿瑜陀耶府|阿瑜陀耶府]]挽巴茵縣，是[[昭披耶河|昭披耶河]]畔閣雲區（Tambon Ko Rian）的一個名勝古蹟。

== 起源 ==
它是從14世紀中葉開始，至18世紀的一個[[日本人|日本人]]聚居地。從15世紀後期直到16世紀初的[[暹羅|暹羅]]大城皇朝處處充滿商機，尤其以沿[[昭披耶河|昭披耶河]]的西岸之南部為核心，在此570公尺 x 230公尺的南北寬長地域裡，在高峰期曾經有1,000至1,500位日本人聚居。根據16世紀 [[日本|日本]]出版的《暹羅國風土軍記》的文章記載甚至估計有8,000名居民，他們的身份大多為僱傭兵、[[商人|商人]]、[[基督教徒|基督教徒]]，以及他們的隨員及家人等。

===軍事需要===
[[Image:NagamasaArmy.jpg|thumb]]的軍隊]]
[[阿瑜陀耶府|阿瑜陀耶府]]的日本人村在14世紀開始形成，原因在[[戰國_(日本)|日本戰國時代]]，許多武士因戰亂失去主人而變成[[浪人|浪人]]。在[[關原之戰|關原之戰]]及[[大坂之役|大坂之役]]後，此現象更迅速擴大。而[[暹羅|暹羅]]的大城皇朝正受[[緬甸|緬甸]]軍事威脅，正好聘用此批經驗豐富的日本浪人作[[僱傭兵|僱傭兵]]，導致大批[[日本|日本]]的浪人流入[[阿瑜陀耶府|阿瑜陀耶府]]。暹羅的{{link-en|頌曇|Songtham}}君主統治時代，亦有聘用[[葡萄牙人|葡萄牙人]]的炮火隊作僱傭兵，因此需要以日本僱傭兵作為剋制，帶動更多的日本浪人湧入。

暹羅的《三印法典》（กฎหมายตราสามดวง）列出：「日本人義勇兵局」（กรมอาสาญี่ปุ่น）為[[暹羅|暹羅]]的官位制度裡的第三高位，[[山田長政|山田長政]]被封名為「奥亚社那披穆」（ออกญาเสนาภิมุข）的爵位。

== 商業發展 ==
有別於[[日本|日本]]的[[朱印船|朱印船]]，暹羅進口大量的日本制造的劍作武器，現時還能夠在曼谷的大皇宮裡看到、陶瓷、高達20萬張[[皮革|皮革]]（主要是[[鹿|鹿]]及[[鯊魚|鯊魚]]）及[[银|白銀]]。大量的商機，以致於在1620年對日本的貿易額,已高於其它國家的總和。

== 基督宗教 ==
此外，[[阿瑜陀耶王國|阿瑜陀耶王國]]對管制宗教的法例寬鬆，吸引許多[[基督徒|基督徒]]逃離了[[幕府|幕府]]。在1627年的調查已有400位[[神父|神父]]的記錄。

== 衰落 ==
1629年，[[阿瑜陀耶王國|阿瑜陀耶王國]]的{{link-en|巴沙·通|Prasat Thong}}君主繼位，為了牽制日本人的勢力，由暹羅皇室重新掌管当地貿易。此外，暹羅南部的洛坤府直轄縣更發起反對山田長政的事件。在1630年，日本人村更發生暴動，居民被屠殺。因此日本僱傭兵在大城皇朝的政治及軍事影響盡失。與此同時，在1632年，日本人村火災后約有400人再次聚集來重建。雖然日本人已失去政治及軍事力量，但由于從事商業貿易，因此做了很多[[暹羅|暹羅]]南部[[也拉府|也拉府]]所生產的[[錫|錫]]之生意。在18世紀的初期還存在日本人村，其後則已逐步融入[[暹羅|暹羅]]社會，終於被自然同化。
