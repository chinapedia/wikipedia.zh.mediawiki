'''阿不都拉·阿巴斯'''（{{bd|1951年||}}）[[塔塔尔族|塔塔尔族]]，[[新疆|新疆]][[乌鲁木齐|乌鲁木齐]]人，[[中华人民共和国|中华人民共和国]]生物学家、[[全国人民代表大会|全国人民代表大会]]新疆地区代表。

==生平==
毕业于[[新疆大学|新疆大学]]生物学专业。加入[[中国共产党|中国共产党]]。任新疆大学生命科学与技术学院教授、副院长。2003年获聘任为新疆维吾尔自治区第十届人民政府特邀行政执法监督员，聘期五年。他连任第九、十、十一、十二届全国人大代表。<ref name=xzb>{{cite news |language =  |author = |coauthors =  |url = http://www.110.com/fagui/law_177068.html |title = 新疆维吾尔自治区人民政府办公厅关于聘请自治区第十届人民政府特邀行政执法监督员的决定（新政办发〔2003〕134号）|publisher = 新疆维吾尔自治区人民政府办公厅 |pages =  |date = 2003-10-14 |accessdate = }}</ref><ref>{{cite news|title=全国人大代表信息-阿不都拉·阿巴斯|url=http://www.npc.gov.cn/delegate/viewDelegate.action?dbid=112631|accessdate=2013-04-05|newspaper=全国人大网}}</ref><ref>{{cite news|title=全国人大代表信息-阿不都拉·阿巴斯|url=http://www.npc.gov.cn/delegate/viewDelegate.action?dbid=122632|accessdate=2013-04-05|newspaper=全国人大网}}</ref>任内提出过有关新疆教育、环境等方面的许多议案。他从十一届全国人大代表任期内开始提[[南疆|南疆]]高中阶段免除学费的建议，连提多年，最终得到国家各部委重视，该问题得以解决。<ref>{{cite news |language =  |author = |coauthors =  |url = http://news.ifeng.com/a/20160310/47769212_0.shtml |title =阿不都拉·阿巴斯代表：“我要把群众的心声说出来”  |publisher = 凤凰网 |pages =  |date = 2016-03-10 |accessdate = }}</ref>
