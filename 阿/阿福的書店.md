阿福的書店([[英語|英语]]：A-Fu's Bookstore)是[[臺灣|台灣]]小型[[獨立書店|獨立書店]]之一<ref>{{Cite web|url = http://old.ohmygod.org.tw/goodtohaveyou/good170.htm|title = 5%的觀念，阿福的良心書店|accessdate = 2010/9|author = 陳中勳|date = 2010-10-03|publisher = 生命力新聞}}</ref><ref>{{Cite web|url = http://www.parenting.com.tw/article/5049977-%E9%98%BF%E7%A6%8F%E7%9A%84%E6%9B%B8%E5%BA%97+%E7%82%BA%E5%AD%A9%E5%AD%90%E8%80%8C%E5%AD%98%E5%9C%A8+/|title = 阿福的書店 為孩子而存在|accessdate = 2013-03|author = 王曉晴|date = 2013-05|publisher = 親子天下}}</ref>，由[[蕭文福|蕭文福]]於2000年5月15日創辦<ref>{{Cite web|url = http://cclitier.blogspot.tw/2014/03/blog-post.html|title = 喵的打字房 永遠的阿福爸爸，記「阿福的書店」老闆蕭文福|accessdate = 2014年2月|author = clitier|date = 2014年3月|publisher = }}</ref>，坐落於[[新北市|新北市]][[蘆洲區|蘆洲區]]，原先開設於[[成功里_(新北市蘆洲區)|成功里]]永樂街31巷，後因房東不續租而遷至[[中原里_(新北市蘆洲區)|中原里]]中正路233號1樓現址，鄰近[[捷運三民高中站|捷運三民高中站]]1號出口，步行約10分鐘，[[國立空中大學|國立空中大學]]對面。

書店創辦之初，是為了提供附近社區的親子有空間可以閱讀、舉辦親子共讀讀書會，而展售的書籍大多以兒童繪本為主，致力推廣閱讀從小開始的理念。店長[[蕭文福|蕭文福]]於2014年2月16日因心肌梗塞辭世，現由妻子[[楊雪惠|楊雪惠]]經營<ref>{{Cite web|url = http://www.chinatimes.com/newspapers/20140622001160-260102|title = 阿福的書店沒了福爸 福媽堅守書童天堂|accessdate = 2014年6月|author = 白宜君|date = 2014年6月22日|publisher = 中國時報}}</ref>。
