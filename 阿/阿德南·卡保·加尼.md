'''阿德南·卡保·加尼'''{{efn|或簡稱為'''阿·卡·加尼'''（{{lang-id|A. K. Gani}}）；稱謂一般帶上他的頭銜、學位、尊稱等，為阿德南·卡保·加尼副博士（{{lang-id|Drs. Adnan Kapau Gani}}）{{sfn|夏基安|1965|p=7}}。}}（{{lang-id|Adnan Kapau Gani}}；{{bd|1905年|9月16日|1968年|12月23日|CatIdX=Gani, Adnan Kapau}}），已故[[印度尼西亞|印度尼西亞]]（印尼）政治人物，[[穆斯林|穆斯林]]，曾出任印尼副總理、經濟部長。他在現屬[[西蘇門答臘省|西蘇門答臘省]]的地區出生，並在[[爪哇|爪哇]]島度過大部分青年時光，還在留居爪哇時開始學醫和參與當地[[印度尼西亞民族覺醒|正在萌芽的民族主義運動]]，直至他到[[南蘇門答臘省|蘇門答臘島南部]]行醫為止。他曾在[[印度尼西亞獨立革命|印度尼西亞獨立革命]]時期三度出任經濟部長（或譯社會福利部長），並在後兩段經濟部長任期期間兼任副總理。雖然加尼隨後遷往南蘇門答臘省[[巨港|巨港]]定居，不過他在去世之前仍然一直活躍於當地政壇。他在2007年11月獲追封為[[印度尼西亞民族英雄|印度尼西亞民族英雄]]。

== 早年生涯 ==
加尼在1905年9月16日生於荷屬東印度[[蘇門答臘西岸駐紮地|蘇門答臘西岸駐紮地]][[巴倫巴揚|巴倫巴揚]]（Palembajan, Residentie Sumatra's Westkust；今屬[[南蘇門答臘省|南蘇門答臘省]]，位於[[武吉丁宜_(西蘇門答臘省)|武吉丁宜]]以西）{{sfn|The Jakarta Post 2007, Four forgotten independence}}{{sfn|JCG, Adnan Kapau Gani}}，是一名教師的兒子。他於1923年在武吉丁宜完成初等教育，並於其後前往巴達維亞（今[[雅加達|雅加達]]）完成中學教育和學醫{{sfn|JCG, Adnan Kapau Gani}}{{sfn|KR 1947, Kabinet Sjarifoeddin}}。他在1926年從專門培育未來醫生的[[土著醫師訓練學校|土著醫師訓練學校]]（School tot Opleiding van Indische Artsen，STOVIA）畢業{{sfn|KR 1947, Kabinet Sjarifoeddin}}。

加尼自青年時期起便已經活躍於政治活動和社會組織。他是「爪哇青年」（Jong Java）、「蘇門答臘青年」（Jong Sumatera）等數個[[印尼原住民|土著青年]]社團的成員{{sfn|JCG, Adnan Kapau Gani}}。他曾在1920年代末經營數家企業（當中包括一家寄宿公寓和二手書店），並捐出這些企業的盈利，為1928年的印尼青年大會籌措經費。他參加了這次青年大會；這次大會發布了[[青年誓言|青年誓言]]，也是《[[偉大的印度尼西亞|偉大的印度尼西亞]]》（印尼國歌）首次演奏的場合{{sfn|The Jakarta Post 2007, Four forgotten independence}}{{sfn|JCG, Adnan Kapau Gani}}{{sfn|Mufid 2012, Menyelundup untuk Kemakmuran Republik}}。他在1931年加入在[[蘇卡諾|蘇卡諾]]被荷蘭殖民政府拘禁後不久從[[印度尼西亞民族黨|印度尼西亞民族黨]]（{{lang|id|Partai Nasional Indonesia}}, PNI）分裂出來的印尼黨（Partindo），又於翌年蘇卡諾出獄後結識蘇卡諾，並和蘇卡諾一起加入[[印度尼西亞政治聯盟|印度尼西亞政治聯盟]]{{sfn|Mufid 2012, Menyelundup untuk Kemakmuran Republik}}{{sfn|KR 1947, Kabinet Sjarifoeddin}}。

加尼一直對戲劇很感興趣{{sfn|Mufid 2012, Menyelundup untuk Kemakmuran Republik}}，他還在1941年獲導演[[拉登·阿里芬|拉登·阿里芬]]邀請，參演由阿里芬執導，由[[友聯影業|友聯影業]]出品的電影《[[純潔的愛情|純潔的愛情]]》。當時荷屬東印度的電影業正開始迎合教育程度高的觀眾。雖然部份觀眾認為加尼參演《純潔的愛情》會損害獨立運動，不過加尼卻認為這是有必要的，因為這樣做可以改善人們對荷屬東印度本土電影的觀感。這部電影是加尼唯一一部參演的電影，也是一部成功贏利的電影{{sfn|Biran|2009|pp=262–263}}。他也在同年獲頒醫學學位{{sfn|KR 1947, Kabinet Sjarifoeddin}}。

1942年[[印度尼西亞日治時期|日本佔領荷屬東印度]]後，加尼因為拒絕和日本當局合作而在1943年9月被捕，直至翌年10月才獲釋。此後他便以私人醫生的身分度日，直至1945年日治時期結束為止{{sfn|KR 1947, Kabinet Sjarifoeddin}}。

== 革命時期生涯 ==
[[File:AK_Gani_KR_3_October_1946_p2.jpg|thumb]]
[[印度尼西亞獨立宣言|印尼獨立]]後，[[印度尼西亞獨立革命|獨立革命]]隨即爆發；加尼在此期間不但在軍隊服役，還得到更大的政治勢力。他曾在1945年至1947年期間擔任印尼民族黨中央執行委員會委員兼南蘇門答臘省黨部主席，也曾在當時負責協調革命軍在該省的軍事行動{{sfn|JCG, Adnan Kapau Gani}}{{sfn|Hägerdal|2009|p=156}}。他認為對印尼這個剛剛獨立的國家而言，[[巨港|巨港]]有條件成為印尼的經濟重鎮，並認為印尼可以[[石油政治|憑藉石油爭取國際社會支持]]。他一方面和國際石油集團（包括荷蘭人擁有的[[蜆殼公司|荷蘭皇家蜆殼]]公司）展開石油買賣談判，另一方面又突破荷蘭封鎖，走私武器和軍用物資{{sfn|Mufid 2012, Menyelundup untuk Kemakmuran Republik}}。他和[[新加坡華人|新加坡華人]]社群之間有着千絲萬縷的聯繫，而新加坡華人也有協助他完成這些任務{{sfn|Yong|2003|p=3}}。

他在1946年10月2日加入由印尼總理[[蘇丹·夏赫里爾|蘇丹·夏赫里爾]]領導的[[第三屆夏赫里爾內閣|第三屆夏赫里爾內閣]]，擔任經濟部長，至1947年6月27日離任{{sfn|Suprapto|1985|p=51}}；夏赫里爾後來談到他覺得加尼是時任總統蘇卡諾強行安插到他的內閣任職的閣員人選{{sfn|Mrázek|1994|p=364}}。加尼在出任經濟部長期間曾與夏赫里爾和[[穆罕默德·鲁姆|穆罕默德·鲁姆]]代表印尼出席《[[林牙椰蒂協定|林牙椰蒂協定]]》談判代表團第三次全體會議，並於1947年簽署這份協定{{sfn|Suprapto|1985|pp=54–55}}。荷蘭人認為加尼既不善交際，也容易暴露自己的情緒{{sfn|Mufid 2012, Menyelundup untuk Kemakmuran Republik}}。他也一度參與銀行貿易公司（Banking and Trading Company, BTC）的籌建工作，不過後來他擔心銀行貿易公司會壟斷印尼共和國的對外貿易，因此他便運用身為經濟部長的便利協助一名爪哇商人另起爐灶，並控制這家新成立的外貿企業，以打通新加坡華商、蘇門答臘和爪哇共和國控制區之間的經貿往來，宣傳革命和援助在外的印尼僑民{{sfn|Mufid 2012, Menyelundup untuk Kemakmuran Republik}}{{sfn|Yong|2003|pp=49–50}}{{sfn|Lindblad|Post|2009|p=78}}。

在[[第一屆阿米爾·謝里夫丁內閣|下一屆內閣]]中，加尼、塞蒂亞吉（Setijadji）和總理[[阿米爾·謝里夫丁|阿米爾·謝里夫丁]]都是[[組閣者|組閣者]]，他們組建的內閣在1947年7月3日獲各黨授權，開始執政{{sfn|Suprapto|1985|p=62}}。在此期間，加尼除了繼續擔任經濟部長，也兼任謝里夫丁手下的副總理{{sfn|Suprapto|1985|p=63}}。荷蘭殖民當局在當年7月向共和國控制區發動攻擊，而加尼則成為首位被捕的印尼內閣部長，不過他在其後仍能獲釋{{sfn|JCG, Adnan Kapau Gani}}。他也參加了一場在[[古巴|古巴]][[哈瓦那|哈瓦那]]舉行的貿易會議{{sfn|Yong|2003|p=78}}。他在[[第二屆阿米爾·謝里夫丁內閣|第二屆阿米爾·謝里夫丁內閣]]當中仍然留任原職{{sfn|Suprapto|1985|p=71}} ，直至這個內閣在1948年1月29日因為部分黨派對《倫維爾協定》的不滿而倒台為止{{sfn|Suprapto|1985|p=77}}。

== 後革命時期生涯 ==
加尼在1949年獨立革命結束後成為南蘇門答臘省軍政長官。他在1954年仍然活躍於政壇，並於當年重新入閣，擔任[[第一屆阿里·沙斯特羅阿米佐約內閣|第一屆阿里·沙斯特羅阿米佐約內閣]]的交通部長；他也在擔任交通部長期間受聘擔任南蘇門答臘省[[巨港|巨港]][[史利威查雅大學|史利威查雅大學]]校長。此後他一直活躍於南蘇門答臘省，直至他在1968年12月23日在巨港逝世為止。加尼身後安葬於巨港錫貢堂英雄公墓（Siguntang Heroes' Cemetery）{{sfn|JCG, Adnan Kapau Gani}}{{sfn|Suprapto|1985|p=161}}{{sfn|Kepustakaan Presiden2 RI, Adenan Kapau Gani}}。加尼身故時遺下妻子瑪斯杜拉（Masturah）；兩人沒有生育子女{{sfn|Penetapan 0073|pp=2–3}}。

== 榮譽 ==
2007年11月9日，印尼總統[[蘇西洛·班邦·尤多約諾|蘇西洛·班邦·尤多約諾]]頒布2007年第66/TK號總統令，向他和另外3人追授[[印度尼西亞民族英雄|印度尼西亞民族英雄]]稱號{{sfn|The Jakarta Post 2007, Four forgotten independence}}{{sfn|Suara Merdeka 2007, Presiden Anugerahkan Gelar}}。
