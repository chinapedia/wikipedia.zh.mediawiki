'''阿魯渾'''（{{mong|m=ᠠᠷᠭᠦᠨ|k=Аргун|z=Arghun}}，{{Lang-de|Arγun}}<ref>根据Herzog August Bibliothek Wolfenbüttel，1987年出版的《[[中國圖說|中國圖說]]》第15页记载</ref>,{{bd|1258年||1291年|3月10日}}），[[蒙古|蒙古]]人，[[阿八哈|阿八哈]]之子，在1284年推翻叔父[[貼古迭兒|貼古迭兒]]，继任[[伊兒汗國|伊兒汗國]]的第四任君主。1284年8月11日<ref>根据商务印书馆，1986年出版的《[[史集|史集]]》第三卷，第187页记载</ref>至1291年3月10日<ref>根据商务印书馆，1986年出版的《史集》第三卷，第213页记载</ref>在位。他在位時試圖阻止其汗國走向伊斯蘭化。

== 生平 ==

阿鲁浑是阿八哈的儿子。阿鲁浑和他最心爱的妻子[[忽勒塔黑哈敦|忽勒塔黑哈敦]]([[卜鲁罕|卜鲁罕]])生下了长子[[合赞|合赞]]，他和妻子[[兀鲁克哈敦|兀鲁克哈敦]]生下了第三子[[完者都|完者都]]<ref>根据商务印书馆，1986年出版的《史集》第三卷，第186页记载</ref>，[[合赞|合赞]]和[[完者都|完者都]]二者后来都成为伊儿汗国的国君。当阿八哈去世时，原本是指定阿鲁浑为继承人的。但是此时他的叔父[[贴古迭儿|贴古迭儿]]成为了汗位的强力竞争者，阿鲁浑只得退让了。他被任命为[[大呼罗珊|呼罗珊]]地区的管理者。贴古迭儿上台后，推行国家[[伊斯兰|伊斯兰]]化的执政方略，激起了国内保守派贵族的不满。于是他们投靠阿鲁浑而对贴古迭儿发动了内战。最终阿鲁浑成功击败了叔父而登上了汗位。

== 统治 ==

阿鲁浑在位时期，阻止了国家向伊斯兰教转化的倾向。他虽然是一个虔诚的[[佛教|佛教]]徒。但是和大多数蒙古人一样，他对各种宗教信仰都很宽容。他委任了很多[[基督教|基督教]]徒和[[犹太教|犹太教]]徒担任国内的文职官员。他甚至允许[[穆斯林|穆斯林]]按照[[沙里亚法规|沙里亚法规]]而不是蒙古律法来处理穆斯林之间的诉讼。他通过削弱军队将领的权力来树立政府的权威，并试图把正规的民政管理引入蒙古人的纯军事统治中。

== 军事 ==

阿鲁浑统治期间，伊儿汗国基本处于和平状态，很少发生军事冲突。1289年-1290年，他平息了一场[[瓦剌|瓦剌]]动乱。1290年，他在[[高加索|高加索]]地区击退了[[金帐汗国|金帐汗国]]的入侵。此外，在他统治的这段时间里，[[马木留克王朝|马木留克王朝]]持续向[[叙利亚|叙利亚]]地区增兵。1289年，马木留克[[苏丹|苏丹]][[嘉拉温|嘉拉温]]攻占了蒙古人的[[附庸国|附庸国]]，[[十字军|十字军]]控制的[[的黎波里|的黎波里]]。

== 外交 ==

阿鲁浑在位期间一直致力建立一个由[[法蘭克人|法蘭克人]]與[[蒙古族|蒙古]]組成的「[[法兰克-蒙古同盟|法兰克-蒙古同盟]]」，以联手对抗伊儿汗国的宿敌马木留克。他曾四次派遣使节分别出访[[教皇|教皇]][[洪诺留四世|洪诺留四世]]、[[英国|英国]]国王[[爱德华一世|爱德华一世]]、[[法国|法国]]国王[[腓力四世_(法兰西)|腓力四世]]和教皇[[尼古拉四世|尼古拉四世]]。他一直以夺取[[耶路撒冷|耶路撒冷]]作为换取西方世界出兵的条件。他甚至曾许诺如果攻占了耶路撒冷，他本人也可以接受[[洗礼|洗礼]]。可惜13世纪末期，西方世界对于发动[[十字军东征|十字军东征]]已经失去了兴趣，阿鲁浑心目中的联盟最终没有能够建立起来。

== 逝世 ==

阿鲁浑于1291年3月10日逝世。13世纪时，蒙古统治者们在西方世界成为了热门的话题。在[[意大利|意大利]]，许多新生的婴儿被以蒙古君王的名字来命名。由于阿鲁浑汗与西方世界的联系加深，有许多新生儿被命名为“'''Argone'''”。
