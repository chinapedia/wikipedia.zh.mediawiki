現代阿瓦爾語（{{lang|ru|Авар мацӀ}} [{{lang|en|awar maʦʼ}}]，自稱為 {{lang|ru|магӀарул мацӀ}}、[{{lang|ru|maʕarul maʦʼ}}] “山中的語言”、Avar language）屬於[[高加索語言|高加索語言]]之一，隸屬 Alarodian Northeast-Caucasian (或 Nakh-Dagestani) 的 Avar-Andi-Tsez 組。

== 地理分佈 ==
阿瓦爾語分佈於俄羅斯[[達吉斯坦共和國|達吉斯坦共和國]]東部和南部、[[阿塞拜疆|阿塞拜疆]] Zakatala 西北部的 Balaken。<ref name="e18" />有些阿瓦爾語居於俄羅斯其他地方，亦有些居於[[車臣共和國|車臣共和國]]、[[卡爾梅克共和國|卡爾梅克共和國]]、[[格魯吉亞|格魯吉亞]]、[[約旦|約旦]]和[[土耳其|土耳其]][[馬摩拉海|馬摩拉海]]區。全球有1.4百萬使用者。

=== 官方地位 ===
阿瓦爾語是[[達吉斯坦|達吉斯坦]]六種語言之一；除[[阿瓦爾人|阿瓦爾人]]外，不同族裔間也會以阿瓦爾語溝通。

=== 方言 ===
阿瓦爾語有兩個主要方言區，北部（阿瓦爾書面語），包括 Khunzakh, Kazbek, Gunib Gumbet 和其他地區；南部（方言區），被不少語言學家視為另一語言：阿瓦爾、 Bagulal, Chamalal, Budukh, Botlikh, Andi, Godoberi, Tindi, Karati, Akhvakh, Tsez (又稱 Dido), Khwarshi, Hinukh, Hunzib, Bezhta，全都是以說那些話的部落來命名。

== 書寫系統 ==
阿瓦爾語由15世紀開始有書寫文字，以古[[格魯吉亞字母|格魯吉亞字母]]書寫。由17世紀起，又以修改[[阿拉伯字母|阿拉伯字母]]書寫，稱為 ''Ajam''，一直用至現代。亦由於達吉斯坦曾被蘇聯統治，在1928年蘇聯推行文字改革時，以[[拉丁字母|拉丁字母]]書寫，又再在1938年改為[[西里爾字母|西里爾字母]]。該套西里爾字母是從[[俄語字母|俄語字母]]中加上一個字母 ''[[Ӏ|palochka]]'' ({{unicode|Ӏ}})。由於很多電腦沒有這個字母的字體，很多時以大寫 I 代替。

== 歷史 ==
阿瓦爾的書面語來自 '''болмац{{unicode|Ӏ}}''' (bolmac{{unicode|ʼ}}) — ''bo'' = “軍隊”或“國”，''mac{{unicode|ʼ}}'' = "語言" — 是不同方言和語言使用者之間的共同語。bolmac{{unicode|ʼ}} 再往上推，來自 Khunzakh 的方言，Khunzakh （高加索阿瓦爾），阿瓦爾區的首都和文化中心，部分亦受南部方言影響。現在，阿瓦爾的書面語亦影響着方言，致使方言和書面語的差異減少。

最著名的現代阿瓦爾語文學作者是 Rasul Gamzatov 。他是達吉斯坦的人民詩人，於2003年11月3日逝世。他的著作被翻譯成俄語後，在前蘇聯享負盛名。
