== 早年 ==
她 13 歲時，在家鄉[[巴西|巴西]][[巴伊亞|巴伊亞]]的某個大型休閒購物賣場逛街時被星探發現。兩年後，她贏得了福特超級模特兒大賽巴西賽區的大獎，並有幸被評為福特全球超級模特兒大賽亞軍佳麗。

== 模特兒生涯 ==
[[File:Adriana-Lima_Chgo_2010-12-10_photoby-Bielawski.jpg|thumb]]
亞莉安娜13歲時在商場行行下街被星探發掘的，不久她移居到紐約市，與[[Elite_Model_Management|Elite Model Management]]簽約，並成為當界Guess麗人。接著就不斷接到著名時尚期刊的邀約，成為封面人物或是內頁模特兒，如：《Elle》、《Vogue》、《Marie Claire》、義大利版《Vogue》和《Harper's Bazaar》。她的過人天賦以及表演才能在 E! Entertainment、Travel Channel、Extra 等電視節日，以及匯集明星的寶馬短片——由王家衛執導的《跟蹤》影片中為世人所領略。2006年由她作封面的《GQ》更是全年銷量的第一。 亞莉安娜現時則是Victoria's Secret 'Angel'和Maybelline的代言人。

== 義務工作 ==
亞莉安娜善於運用自己的名氣及財力幫助他人了，特別值得一提的是位於[[巴伊亞|巴伊亞]]的Caminos Do Luz[[孤兒院|孤兒院]]，正是她運用自己的聲望及財富創建起來的。
