'''阿拉木图地铁'''（{{lang-kk|Алматы метрополитені}}），是[[哈萨克斯坦|哈萨克斯坦]][[阿拉木图|阿拉木图]]的城市轨道交通系统，首条线路在經過超過23年的建設後，于2011年12月1日开通<ref>[http://news.xinhuanet.com/fortune/2011-12/02/c_111210895.htm 哈萨克斯坦第一条地铁在阿拉木图开通] {{webarchive|url=https://web.archive.org/web/20150402171653/http://news.xinhuanet.com/fortune/2011-12/02/c_111210895.htm |date=2015-04-02 }}，[[新华网|新华网]]，2011年12月2日，于2011年12月2日查阅</ref>。阿拉木图地铁是[[中亚|中亚]]地区继[[乌兹别克斯坦|乌兹别克斯坦]][[塔什干地鐵|塔什-{干}-地鐵]]后的第二个城市轨道交通系统，也是前[[苏联|苏联]]地区的第十六个城市轨道交通系统。值得一提的是，和大多數俄系地鐵一般，阿拉木图地鐵各車站也是繼承了前蘇聯別出心裁的設計，不過以獨立的哈薩克民族和伊斯蘭風格為主。

由於[[阿拉木圖|阿拉木圖]]是哈萨克的第一大城市，在册人口為150多万，常住人口為200万至300万。由於人口集中，為交通和空氣質素帶來不少壓力，建設地鐵可望解決問題<ref>[http://www.52bus.com/news/track/world/2011/1212/14857.html 【哈萨克斯坦】历经23载,阿拉木图地铁终于开通]{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

== 建設 ==
工程建設在1988年開始，當時[[哈薩克斯坦|哈薩克斯坦]]仍然是[[苏联|苏联]]的一部分，當時項目計劃在2007年完成，開支由苏联中央政府承担。可是隨著[[苏联|苏联]]解體，[[莫斯科|莫斯科]]提供的資金開始乾涸，哈薩克政府並沒有能力繼續完工，唯有盡力保存已建成的部分。因為工人債務的累積，而導致花費有所上升。2003年，地鐵項目劃入国家预算，哈薩克政府注入資金，項目重新啟動。2010年，哈薩克总统[[努尔苏丹·纳扎尔巴耶夫|努尔苏丹·纳扎尔巴耶夫]]下令，地鐵必須在國家獨立20周年前夕啟用<ref>[http://world.people.com.cn/GB/13040899.html 阿拉木图圆了地铁梦（第一现场）]</ref>。最後地鐵在2011年12月1日開通。

== 系統 ==
當建設完成後，阿拉木圖地鐵會有三段，總長度為45公里。第一线第一段的長度為8.56公里，有7個車站，有7列列車提供服務。每一列列車有4個車箱，總載客量為940人，試驗營運速度為80公里。列車由[[韓國|韓國]][[現代Rotem|現代Rotem]]建造<ref>[http://eltonzeng.blog.hexun.com.tw/72369702_d.html 剛剛開放的阿拉木圖地鐵]</ref>。地鐵線从北部的赖雷别克大街沿富尔曼诺夫大街向南至阿拜大街，再向西至加加林大街。建設費用為1659億堅戈（1美元约合147.69坚戈）<ref>[http://world.people.com.cn/GB/57507/16468703.html 哈萨克斯坦阿拉木图地铁正式通车]</ref>。系統在將來會延長至莫斯科站，於2011年開工建設，並於2015年4月投入營運。

== 批評 ==
當地專家批評，地铁站布局並不合理。地铁站遠離大型商場等人流集中的地方，不能起到減輕交通壓力的作用。另外，地铁票价只佔營運成本的五分之一，政府可能要作出大額補貼。此外中亞的小规模地震，亦威脅到地鐵的安全。

== 票价 ==
為單一票價，每程80[[哈薩克斯坦坚戈|哈薩克斯坦坚戈]]<ref>[http://www.52bus.com/news/track/world/2011/1205/14734.html 【哈萨克斯坦】第一条地铁在阿拉木图开通] {{webarchive|url=https://web.archive.org/web/20140714182115/http://www.52bus.com/news/track/world/2011/1205/14734.html |date=2014-07-14 }}</ref>。
