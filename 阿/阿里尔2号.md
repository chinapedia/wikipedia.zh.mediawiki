'''阿里尔2号（{{lang|en|Ariel 2}}），也称之为UK-C，'''是[[英国|英国]]发射的一个航天[[人造卫星|人造卫星]]。该发射隶属于[[阿里尔项目|阿里尔项目]]，并由英国科学与工程研究委员会运作<ref>{{cite web|url = http://www.astronautix.com/craft/ariel.htm|title = Ariel|first = Mark|last = Wade|work = Encyclopedia Astronautica|accessdate = 2009-09-07}}</ref>。该人造卫星由美国[[西屋公司|西屋公司]]制造<ref>{{cite news|title = Design of a Spacecraft|date = 1965-01-21|work = Flight International|page = 115}}</ref>，负重68公斤（150磅）<ref>{{cite web|url = http://space.skyrocket.de/doc_sdat/ariel-1.htm|first = Gunter|last = Krebs|publisher = Gunter's Space Page|title = Ariel 1, 2|accessdate = 2009-09-07}}</ref><ref name="WCS">{{cite web|url = http://www.spacesecurity.org/files/WorldCivilSats2006.xls|title = World Civil Satellites 1957-2006|publisher = Space Security Index|work = |accessdate = 2009-09-07|deadurl = yes|archiveurl = https://web.archive.org/web/20110718022840/http://www.spacesecurity.org/files/WorldCivilSats2006.xls|archivedate = 2011-07-18}}</ref>。1964年，它是第一艘被使用于射电天文学运作的人造卫星。

阿里尔2号由[[美国国家航空航天局|美国国家航空航天局]]使用侦察兵X-3号发射成功，1964年3月27日由{{le|瓦罗普斯飞行研究所|Wallops Flight Facility}}3号发射台发射成功<ref name="LaunchLog">{{cite web|url = http://planet4589.org/space/log/launchlog.txt|title = Launch Log|publisher = Jonathan's Space Page|first = Jonathan|last = McDowell|accessdate = 2009-09-07}}</ref>。1964年11月终结人物<ref>{{cite web|url = http://centaur.sstl.co.uk/SSHP/micro/micro60s.html|title = Traditional Micro-satellites list: 1957-1969|publisher = Surrey Satellite Technology Ltd|accessdate = 2009-09-07|deadurl = yes|archiveurl = https://web.archive.org/web/20090605053312/http://centaur.sstl.co.uk/SSHP/micro/micro60s.html|archivedate = 2009-06-05}}</ref>，并在11月18日解体。
