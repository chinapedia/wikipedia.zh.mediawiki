'''阿拉伯语'''（'''{{rtl-lang|ar|العَرَبِيَّة}}''' al-ʻarabīyah [alʕaraˈbijja] 或者 {{rtl-lang|ar|عربي/عربى}} ʻarabī  [ˈʕarabiː]）是除了[[英語|英語]]和[[法語|法語]]之外最多國家使用的[[官方語言|官方語言]]。阿拉伯語源自[[公元6世纪|公元6世纪]]的[[古典阿拉伯语|古典阿拉伯语]]。它包括书面语及流通于[[中东|中东]]、[[北非|北非]]和[[非洲之角|非洲之角]]（即[[索马里半岛|索马里半岛]]）的[[阿拉伯語變體|各种口语]]。阿拉伯语属于[[亚非语系|亚非语系]]。

阿拉伯语的书面语称为“[[现代标准阿拉伯语|现代标准阿拉伯语]]”或“[[现代标准阿拉伯语|书面阿拉伯语]]”。书面阿拉伯语是目前唯一在官方及正式场合使用的阿拉伯语，用于大多数书面文件和讲座、新闻广播等正式讲话。但这亦因国家而异。1912年，在[[摩洛哥|摩洛哥]]加入[[阿拉伯国家联盟|阿拉伯国家联盟]]之前，[[摩洛哥阿拉伯语|摩洛哥阿拉伯语]]曾在正式场合使用过一段时间。

阿拉伯语属于[[中部闪米特语|中部闪米特语]]，与[[亚拉姆语|亚拉姆语]]、[[希伯来语|希伯来语]]、[[烏加里特語|乌加里特语]]和[[腓尼基语|腓尼基语]]相近。阿拉伯语书面语不同于其所有地方的口语，且更为传统和保守。两者是[[双层语言|双层语言]]的关系，用于不同的场合。

一些地方的阿拉伯语无论是书写还是口头形式，都无法[[相互理解性|互通]]<ref>"Arabic language." Encyclopædia Britannica. 2009. Encyclopædia Britannica Online. Retrieved on 29 July 2009.</ref>。而所有地方的阿拉伯语被当作是一个整体。即是说，纯粹从语言学的角度来说，它们是不同的语言；但是从政治及民族的角度来说，他们又是一个整体。如果阿拉伯语被当作一个整体，则世界上估计有4.22亿人<ref>"World Arabic Language Day". UNESCO. UNESCO. 1995–2012. Retrieved 27 May 2013.</ref>以其为母语。如果各地的阿拉伯语当作是不同的语言，则很难估计到底有多少种，因为它们是[[方言连续体|方言连续体]]，之间没有明确的界线。其中[[埃及阿拉伯語|埃及阿拉伯语]]的使用人数最多，大约五千四百万人<ref>Egyptian Arabic reference at Ethnologue (16th ed., 2009)</ref>以其为母语——多于其他任何一种闪米特语言。

现代的书面语（[[现代标准阿拉伯语|现代标准阿拉伯语]]）源于[[古兰经|古兰经]]的语言（即[[古典阿拉伯语|古典阿拉伯语]]），用于学校教学及工作、政府、媒体等场合。两者合起来被称为书面阿拉伯语，是[[伊斯兰教|伊斯兰教]]的{{link-en|宗教語言|Liturgical language}}。现代标准阿拉伯语的语法与古典阿拉伯语大体相同，词汇也有相同之处。但古典阿拉伯语的一些语法结构在现代标准阿拉伯语中不再使用，在口语中不使用的词汇也不在现代书面语中使用。而且现代书面语从口语中借入了一些词汇和语法现象。新的词汇大多用来表达近现代出现的概念。

阿拉伯语用[[阿拉伯字母|阿拉伯字母]]从右往左书写。有时在非正式场合也可用[[拉丁字母|拉丁字母]]从左往右书写，但没有统一的形式。

阿拉伯语往[[伊斯兰世界|伊斯兰世界]]的语言（如[[波斯语|波斯语]]、[[土耳其语|土耳其语]]、[[索马里语|索马里语]]、[[波斯尼亞語|波斯尼亞語]]、[[哈萨克语|哈萨克语]]、[[孟加拉语|孟加拉语]]、[[乌尔都语|乌尔都语]]、[[马来语|马来语]]和[[豪萨语|豪萨语]]）輸出了大量词汇。[[中世纪|中世纪]]时期，书面阿拉伯语成了欧洲文化的重要载体，特别是在科学、数学和哲学领域。这导致许多欧洲语言也从阿拉伯语中借入了大量词汇。阿拉伯语在词汇和语法方面对[[羅曼語族|羅曼語族]]的语言（特别是[[西班牙语|西班牙语]]、[[葡萄牙语|葡萄牙语]]、[[加泰羅尼亞語|加泰羅尼亞語]]和[[西西里語|西西里語]]）影响很大。

阿拉伯语也从其他语言中借入了大量词汇，如早期从[[希伯来语|希伯来语]]、[[希腊语|希腊语]]、[[波斯语|波斯语]]、[[叙利亚语|叙利亚语]]，中期从[[土耳其语|土耳其语]]，当代从欧洲语言（主要是[[英语|英语]]和[[法语|法语]]）。

== 历史 ==
阿拉伯语源自古语言闪米特语，源自[[阿拉伯半島|阿拉伯半島]]<ref name="中國大百科全書">張甲民，《[[中國大百科全書|中國大百科全書]]》-阿拉伯語</ref>，於5世紀時在北方方言基礎上形成統一的文學語言<ref name="中國大百科全書"/>，从公元6世纪开始便有古阿拉伯语的文献，公元7世纪开始，随着[[伊斯兰帝国|伊斯兰帝国]]的扩张，及[[阿拉伯人|阿拉伯人]]和伊斯兰教传入其他国家，阿拉伯语完全取代了[[伊拉克|伊拉克]]、[[叙利亚|叙利亚]]、[[埃及|埃及]]和[[北非|北非]]从前使用的语言。许多语言学家认为阿语是闪语系中最接近[[闪米特祖语|闪米特祖语]]的。

== 特点 ==
阿语依照各国、各地的不同存在方言，在[[阿拉伯半岛|阿拉伯半岛]]的南部沿海，阿语有若干方言，这些方言统称为[[南阿拉伯语|南阿拉伯语]]，南阿拉伯语与[[北阿拉伯语|北阿拉伯语]]差异之大以至人们认为是一门独立语言。另外西北非的[[马格里布|马格-{里}-布]]方言和中东方言交谈也不能互相理解。但各个方言区的人能採用《古兰经》使用的[[古典阿拉伯语|古典阿拉伯语]]（书面语或标准语）作為標準，進而溝通。

使用较多的方言有埃及方言、叙利亚方言及伊拉克方言。由于埃及的文化产业特别是电影业（及相关行业）较之其他阿拉伯国家比较发达。以及阿拉伯小说家[[塔哈·侯赛因|塔哈·侯赛因]]的小说广为流传。所以，埃及方言通过电影、歌曲、小说等形式传播到阿拉伯各地。

阿拉伯语的標準形式稱作“夫斯哈”（{{rtl-lang|ar|الفصحى}}，[[现代标准阿拉伯语|现代标准阿拉伯语]]或[[古典阿拉伯语|古典阿拉伯语]]），使得各地住民能夠交流。

阿拉伯语是世界上的一种重要语言，也是[[联合国|联合国]]的工作语言之一。

==阿拉伯语语法系统==
{{main|阿拉伯语语法}}
'''阿拉伯语语法'''（{{ lang|ar|النحو}}）是[[阿拉伯语语言学|阿拉伯语语言学]]的一个重要分支。研究按《[[古兰经|古兰经]]》中确定的用法来运用的[[词类|词类]]、[[词的屈折变化|词的屈折变化]]或表示相互关系的其他手段以及词在句中的功能和关系。包含词的[[构词|构词]]、[[构形|构形]]的规则和[[组词|组词]]成句的规则。

== 宗教意义 ==
阿拉伯语是[[伊斯兰教|伊斯兰教]]的通用语言，《[[古兰经|古兰经]]》是以古典阿拉伯语撰写的，伊斯兰教徒传统上认为古兰经是不可能准确翻译的，有些教徒甚至认为试图翻译经文是不应该的。随之时代的变化和传播伊斯兰教的需要《古兰经》被翻译成多种语言文字，但是在伊斯兰活动里会力求使用阿拉伯语。

虽然阿语与伊斯兰教关系密切，但许多穆斯林并不会说阿拉伯语。他们之中大多数都会認得阿拉伯文中的阿拉伯字母，但不知文意。

== 中国穆斯林的宗教语言 ==
在很多中国穆斯林聚居区所使用的语言中有很多阿拉伯语及波斯语单词。但由于波斯人和突厥人早于阿拉伯人来到中国，所以[[明朝|明朝]]之前[[突厥语|突厥语]]和波斯语对中国穆斯林的影响相对更大，但在宗教上阿拉伯语影响更大，以致一些文献提到[[回族|回族]]的宗教语言便是阿拉伯语。但[[明朝|明朝]]统治期间，[[朱元璋|朱元璋]]实行民族同化政策，回民逐渐使用[[汉语|汉语]]和[[汉字|汉字]]。

在使用汉语的穆斯林的的词汇中如“赛摆布”——“缘由”的意思，就是来源于阿拉伯语“{{rtl-lang|ar|سبب}}”。

在[[维吾尔语|维吾尔语]]、[[哈萨克语|哈萨克语]]、[[吉尔吉斯语|吉尔吉斯语]]（柯尔克孜语）词汇中阿拉伯语词也占有相当的比例。

== 字母 ==
{{seealso|阿拉伯字母}}
阿拉伯语文字（[[字母|字母]]）源于古代的[[阿拉米文|阿拉米文]]字，可能从古埃及文字的一种变体演变而来；阿语书写分为数种字体，书写方向从右至左。由于伊斯兰教禁止偶像崇拜，禁止圖畫等藝術形式，书法成为阿拉伯人最重要的艺术形式，演化出许多十分复杂的阿拉伯文字体。其中比较流行的是库法体、三一体、誊写体、公文体等。阿语字母28个，全部为[[辅音|辅音]]字母；[[元音|元音]]通过由加在字母上方或下方的标音符号来表示，但这些符号通常是省略，只在初级启蒙书中和《[[古兰经|古兰经]]》版本中出现。另外世界上使用阿拉伯字母的语言有[[波斯语|波斯语]]、[[普什图语|普什图语]]、[[乌尔都语|乌尔都语]]、一部分[[突厥语|突厥语]]、[[柏柏尔语|柏柏尔语]]以及中国境内的[[维吾尔语|维吾尔语]]、[[哈萨克语|哈萨克语]]、[[乌兹别克语|乌兹别克语]]等。另外还有一些原本使用阿拉伯字母的语言在现代化模仿西方的过程中转而使用[[拉丁字母|拉丁字母]]，如[[土耳其语|土耳其语]]、[[斯瓦西里语|斯瓦西里语]]、[[土庫曼語|土庫曼語]]和[[阿塞拜疆语|阿塞拜疆语]]。亦有少數阿拉伯語的變體不使用阿拉伯文，例如[[馬爾他語|馬爾他語]]是使用[[拉丁字母|拉丁字母]]。
