== 人體的體溫調節 ==
人體的體溫是恆定的，其內在溫度約維持在37℃左右，也就是所謂[[核心溫度|核心溫度]]（core body temperature），通常指人體的直腸溫度，位於人體的內部中心，繞著臟器，其外圍就是[[體壁|體壁]]與[[皮膚|皮膚]]。 	

體溫若高到41～42℃（如：劇烈[[運動|運動]]、[[發燒|發燒]]）時，體內許多[[蛋白質|蛋白質]]分子（尤以[[酵素|酵素]]分子對生命非常重要）會很快地分解，而細胞內許多代謝反應卻需要在恆定的溫度下才能順利地進行，酵素是[[代謝|代謝]]過程的重要[[催化劑|催化劑]]，沒有了酵素，細胞也就無法進行代謝作用。

== 內溫動物的體溫調節 ==
'''[[內溫動物|內溫動物]]'''(舊稱恆溫動物)，通常是指[[鳥類|鳥類]]和[[哺乳類|哺乳類]]，牠們的下視丘內有體溫調節中樞，具有自行調節體溫的功能，通常體表具有良好的保溫構造，例如：[[羽毛|羽毛]]、[[毛髮|毛髮]]。

內溫動物在天氣炎熱或激烈運動時，皮膚的[[血管|血管]]會擴張，使[[血液|血液]]量增加，加速體熱在體表散失，食慾也同時減少，以減少[[熱量|熱量]]產生。有時，內溫動物的汗腺分泌量增加促使體熱快速散失。另外，呼氣、[[排泄|排泄]]及[[排遺|排遺]]也可散失一部分的體熱。

內溫動物在天氣寒冷時，皮膚的血管會收縮，使血液量減少，抑制體熱在體表散失的速度，食慾也同時增加，配合[[甲狀腺|甲狀腺]]分泌[[甲狀腺素|甲狀腺素]]的量增加，加速細胞的[[呼吸作用|呼吸作用]]，以加速熱量產生。有時，內溫動物的[[肌肉|肌肉]]會不自主地顫抖，皮膚表層產生「[[雞皮疙瘩|雞皮疙瘩]]」，以增加體熱。

== 外溫動物的體溫調節 ==
'''[[外溫動物|外溫動物]]'''(舊稱變溫動物)，通常是指[[魚類|魚類]]、[[兩生類|兩生類]]和[[爬蟲類|爬蟲類]]等動物，由於缺乏體溫調節中樞，無法自行調節體溫，因此體溫會隨著外界[[環境|環境]][[溫度|溫度]]改變而有明顯的變動。

外溫動物在環境溫度大幅改變時，會更換[[棲息地|棲息地]]或[[休眠|休眠]]的外在行為協助體溫不至於變化過大，例如：天氣熾熱時，牠們會躲在陰涼處避免體溫過高；天氣寒冷時，牠們會外出曬太陽，以獲取熱量。

== 植物的體溫調節 ==
'''[[植物|植物]]'''的體溫過高時，[[葉綠素|葉綠素]]會被破壞而成白色，即「[[葉片白化|葉片白化]]」。因此植物也須進行體溫調節。

植物可藉由[[蒸散作用|蒸散作用]]將過多的體熱帶走，[[葉片|葉片]]的[[角質層|角質層]]可以反射陽光，以減少熱的吸收。
