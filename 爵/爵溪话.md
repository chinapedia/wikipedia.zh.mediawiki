'''爵溪话'''又名'''所里话'''、'''城里话'''<ref>{{cite web|url=http://cmspub.cnnb.com.cn/xslyw/system/2008/02/18/010029630.shtml|title=趣谈象山方言|publisher=象山旅游网|date=2008-02-18|accessdate=2011-04-26}}</ref>，是[[中国|中国]][[浙江省|浙江省]][[象山县|象山县]]的一个[[官话|官话]][[方言岛|方言岛]]，位于[[爵溪街道|爵溪街道]]，2000年使用人口约9500人。<ref name="main">{{cite conference|title=浙江象山县爵溪“所里话”音系|author=黄晓东|date=2006-03|conference=第四届国际吴方言学术研讨会|place=[[宁波|宁波]]}}</ref>

== 语言来源 ==
据考证，爵溪话来源于[[明代|明代]]爵溪驻军，属于[[军话|军话]]的一种。明代洪武三十年（1397年）设立爵溪[[千户所|千户所]]，隶属于昌国卫，建置之初，军士多来自北方，而前往当地流徙充军之人也多来自北方。而在清代[[海禁|海禁]]之时，当地居民并未内迁，因而驻军官逐渐受到当地[[吴语|吴语]]影响，逐渐形成了爵溪话。<ref name="main" />

== 比较研究 ==
经过黄晓东等学者研究认为，爵溪话属于吴语化的官话。在与[[杭州话|杭州话]]和[[苍南|苍南]][[金乡话|金乡话]]对比研究后发现，这三种语言所具备的特征高度相同，这说明三者之间可能具备某种联系。<ref name="research">{{cite journal|title=吴徽语区内的方言岛|author=曹志耘|coauthor=黄晓东|journal=语言的多视角考察|publisher=南洋理工大学中华语言文化中心|year=2007}}</ref>

== 现状 ==
由于爵溪街道经济发达，对外交往日益频繁，因此爵溪话目前正迅速向吴语[[宁波话|宁波话]]靠拢。<ref name="research" />
