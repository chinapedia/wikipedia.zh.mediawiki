'''吏讀'''（[[北韓|北韓]]读作：{{lang|ko|리두}}，[[南韓|南韓]]读作：{{lang|ko|이두}}），是以[[漢字|漢字]]表記[[朝鮮語|朝鮮語]]（韓語）的一種方法。別名有'''吏書'''、'''吏道'''、'''吏刀'''、'''吏吐'''等。[[三國_(朝鮮)|三國時代]]開始使用，直至西元十九世紀末。是研究[[古朝鮮語|古代朝鮮語]]的資料之一。
{{漢字}}
{{Korean writing}}

== 概說 ==
廣義上吏讀又包含「[[鄉札|鄉札]]」、「[[口訣_(朝鮮語)|口訣]]」等，指代所有使用漢字紀錄朝鮮語的方法。狹義上指「吏讀文」中使用的朝鮮語的漢字表記法。「吏讀文」主要指官府的行政文書中使用漢字表記的散文，而類似文體的民間文章也可稱為「吏讀文」。此處主要敘述狹義的「吏讀」。

吏讀的形成時期尚不明確，但據推測大致始於[[三國_(朝鮮)|三國時代]]，至[[新羅|統一新羅時代]]確立。有的文獻資料記載吏讀為[[新羅人|新羅人]][[薛聰|薛聰]]所創，但並無充足的證據證明此事，薛聰可能僅為當時的表記法的整理者。

== 表記法 ==
吏讀文之中名詞、動詞詞根等實義部分使用[[漢字|漢字]]，表示語法功能的部分主要使用吏讀。但名詞、動詞部分也有使用吏讀的例子。朝鮮半島剛傳入漢字的一段時間使用正規的[[文言文|漢文]]，但其後出現了按朝鮮語語序排列漢字的「[[誓記體|誓記體]]」（{{lang|ko|서기체}}）模擬漢文。可以認為在此種模擬漢文中補充語法要素後，即形成吏讀。

吏讀之中出現的漢字讀法主要是從古代傳承的習慣讀法，但也存在不同於[[中世朝鮮語|中世朝鮮語]]的吏讀特有的讀法。例如，表示處格（朝鮮語[[格_(語法)|格]]的一種）的「良中」（－{{古韓語|아ᄒᆡ}}），中世朝鮮語為「{{lang|ko|-애/-에}}」。

以下為出現於《養蠶經驗撮要》（1415年）中的吏讀用例：

* 漢文：{{lang|ko|蠶陽物，大惡水，故食而不飲。}}
* 吏讀文（下劃線部份為吏讀）：{{lang|ko|蠶<u>段</u>陽物<u>是乎等用良</u>，水氣<u>乙</u>厭却，桑葉<u>哛</u>喫破<u>爲遣</u>，飲水<u>不冬</u>。}}（漢語直譯：蠶為陽物，因而厭却水氣，喫破桑葉，而不飲水。）
* 將吏讀部分替換為[[諺文|諺文]]（現代式拼法）：{{lang|ko|蠶딴 陽物이온들쓰아，水氣을 厭却，桑葉뿐 喫破하고，飲水안들。}}
* 对训读的汉字也标注：{{lang|ko|{{ruby|蠶|누에}}는 {{ruby|陽|양}}{{ruby|物|물}}{{ruby|是|이}}므로, {{ruby|水|물}}{{ruby|氣|기}}를 {{ruby|厭|싫}}어해, {{ruby|桑|뽕}}{{ruby|葉|잎}}뿐 {{ruby|喫|먹}}고, {{ruby|水|물}}을{{ruby|飲|마시지}}않는다.}}
