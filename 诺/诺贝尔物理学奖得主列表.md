'''[[诺贝尔物理学奖|诺贝尔物理学奖]]'''是[[诺贝尔奖|诺贝尔奖]]的六个奖项之一，由[[瑞典皇家科学院|瑞典皇家科学院]]每年颁发给在[[物理|物理]]科学领域做出杰出贡献的科学家。<ref>{{cite web |title = Alfred Nobel – The Man Behind the Nobel Prize |publisher = Nobel Foundation |url = http://nobelprize.org/alfred_nobel/ |accessdate=2008-10-29}}</ref> 根据[[阿尔弗雷德·诺贝尔|阿尔弗雷德·诺贝尔]]的遗愿，该奖由[[诺贝尔基金会|诺贝尔基金会]]管理，由瑞典皇家科学院选出5名成员组成一个委员会来评选出获奖者。<ref>{{cite web|title=The Nobel Prize Awarders |publisher=Nobel Foundation |url=http://nobelprize.org/prize_awarders/ |accessdate=2008-10-29 |deadurl=yes |archiveurl=https://web.archive.org/web/20131012052515/http://www.nobelprize.org/prize_awarders/ |archivedate=2013-10-12 }}</ref> 诺贝尔物理学奖於1901年第一次頒發，由[[德国|德国]]的[[威廉·伦琴|威廉·伦琴]]獲得。每个获奖者会得到一块奖牌，一份获奖证书，以及一笔不菲的奖金，奖金的数额每年会有变化。<ref>{{cite web |title = The Nobel Prize |publisher = Nobel Foundation |url = http://nobelprize.org/nobel_prizes/ |accessdate=2008-10-29}}</ref>1901年，伦琴得到150,782[[瑞典克朗|瑞典克朗]]，相当于2007年12月的7,731,004瑞典克朗。2008年，三位获奖者（[[小林誠_(物理學家)|小林诚]]、[[益川敏英|益川敏英]]和[[南部阳一郎|南部阳一郎]]）分享了总额为1千万瑞典克朗的奖金（略多于100万[[欧元|欧元]]，或140万[[美元|美元]]）。<ref>{{cite web|title=The Nobel Prize Amounts |publisher=Nobel Foundation |url=http://nobelprize.org/nobel_prizes/amounts.html |accessdate=2008-10-29 |deadurl=yes |archiveurl=https://web.archive.org/web/20080731233358/http://nobelprize.org/nobel_prizes/amounts.html |archivedate=2008-07-31 }}</ref>该奖每年于12月10日，即[[阿尔弗雷德·诺贝尔|阿尔弗雷德·诺贝尔]]逝世周年纪念日，以隆重的仪式在[[斯德哥尔摩|斯德哥尔摩]]音乐厅颁发。<ref>{{cite web|title=The Nobel Prize Award Ceremonies |publisher=Nobel Foundation |url=http://nobelprize.org/award_ceremonies/ |accessdate=2008-10-29 |deadurl=yes |archiveurl=https://web.archive.org/web/20100131133649/http://nobelprize.org/award_ceremonies/ |archivedate=2010-01-31 }}</ref>

[[约翰·巴丁|约翰·巴丁]]是唯一两次获得该奖的得主，他于1956年和1972年獲獎。[[威廉·劳伦斯·布拉格|威廉·劳伦斯·布拉格]]是至今最年轻的诺贝尔物理学奖奖得主，也是诺贝尔三项科学奖项中的最年轻得主，他在1915年获奖时仅有25岁。<ref>{{cite web|title=Nobel Laureates Facts |publisher=Nobel Foundation |url=http://nobelprize.org/nobel_prizes/nobelprize_facts.html |accessdate=2008-10-29 |deadurl=yes |archiveurl=https://web.archive.org/web/20070202025710/http://nobelprize.org/nobel_prizes/nobelprize_facts.html |archivedate=2007-02-02 }}</ref> 至今共有三位女性获得过该奖，分别是[[玛丽·居里|玛丽·居里]]（1903年）、[[玛丽亚·格佩特-梅耶|玛丽亚·格佩特-梅耶]]（1963年）和{{link-en|唐娜·斯特里克蘭|Donna Strickland}}（2018年）。在六个诺贝尔奖项中，这是女性获奖人次第二少的奖项（只多於僅一位女性得主的[[諾貝爾經濟學獎|諾貝爾經濟學獎]]）。<ref>{{cite web |title = Women Nobel Laureates |publisher = Nobel Foundation |url = http://nobelprize.org/nobel_prizes/lists/women.html |accessdate = 2008-10-29}}</ref>截至2018年，共有209人获得过该奖。诺贝尔物理学奖有6年因故停发（1916、1931、1934、1940－1942年）。
