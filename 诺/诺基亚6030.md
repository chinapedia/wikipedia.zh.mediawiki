'''诺基亚 6030'''是由[[诺基亚|诺基亚]]公司出品的一款[[移动电话|移动电话]]产品，是一款入门级商务手机。外壳颜色主要有[[黑色|黑色]]和[[银色|银色]]两种。支持 Java MIDP 2.0，支持通过“空中传送”下载基于 Java 的应用软件和游戏。
