'''诺福克县'''（{{lang-en|Norfolk County}}），[[加拿大|加拿大]][[安大略省|安大略省]][[南安大略|南部]]的一个单层次政区，位于[[伊利湖|伊利湖]]北岸，西臨[[埃爾金縣|埃爾金縣]]，西北鄰[[牛津縣_(安大略省)|牛津縣]]，北及[[布蘭特縣|布蘭特縣]]，東面則與[[哈爾迪曼德縣|哈爾迪曼德縣]]為鄰。总人口63,175（2011年）。行政中心位于[[锡姆科|锡姆科]]。

== 历史 ==
1792年，诺福克建置为县。1800年，哈尔迪曼德县（Haldimand）设立，析置自诺福克县。1974年，两县重新合并为哈尔迪曼德-诺福克地区自治体<ref>{{cite web |url=http://www.norfolkcounty.ca/download/government/countystratplan0910.pdf |title=Norfolk County Council Strategic Plan 2009-2010 |publisher=諾福克縣政府 |accessdate=2014-02-22 |deadurl=yes |archiveurl=https://web.archive.org/web/20140228071228/http://www.norfolkcounty.ca/download/government/countystratplan0910.pdf |archivedate=2014-02-28 }}</ref>。2001年，哈尔迪曼德、诺福克再次分设。德里与诺福克所辖的乡、锡姆科镇、楠蒂科克市的西半部分合并为诺福克镇。新设政区的条例则将诺福克镇更名为诺福克县。虽然由于历史的原因，诺福克保留县的名称，但是其并没有县政府，而作为单层次政区来管理<ref>{{cite web |url=http://www.mah.gov.on.ca/AssetFactory.aspx?did=6212 |title=Municipal Restructuring Activity Summary Table |publisher=安大略省政府 |accessdate=2014-02-22}}</ref>，法律上归类为市（city）。

1974年之前，诺福克县划分为8个乡，分别为夏洛特维尔、霍顿、米德尔顿、北沃尔辛厄姆、南沃尔辛厄姆、汤森、温德姆、伍德豪斯。虽然行政区划改革后，8个乡的政治实体不复存在，但有时仍作为地理划分的依据。
