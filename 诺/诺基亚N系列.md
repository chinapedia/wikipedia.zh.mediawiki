'''[[諾基亞|諾基亞]]N系列'''是[[多媒体|多媒体]][[手机|手机]]家族的一员。这个系列的手机都支持数字多媒体服务比如音乐播放，视频采集，照相，手机游戏和因特网服务。并支持至少一种高速无线技术，例如[[3G|3G]]或[[无线局域网|无线局域网]]。

== 历史 ==
诺基亚于2005年4月27日在[[阿姆斯特丹|阿姆斯特丹]]举行的手机制造商[[记者招待会|记者招待会]]上宣布了一个新的多媒体设备品牌。<ref>[http://press.nokia.com/PR/200504/991467_5.html Nokia launches Nokia Nseries branded multimedia device range] {{Wayback|url=http://press.nokia.com/PR/200504/991467_5.html |date=20090401231907 }}</ref>最初于记者会上发布的N系列手机是[[诺基亚_N70|N70]]、[[诺基亚_N90|N90]]和[[诺基亚_N91|N91]]。2005年11月2日，诺基亚发布了[[诺基亚_N71|N71]]、 [[诺基亚_N80|N80]]和[[诺基亚_N92|N92]]。2006年4月25日，诺基亚发布了[[诺基亚_N72|N72]]、[[诺基亚_N73|N73]]和[[诺基亚_N93|N93]]。2006年9月26日诺基亚发布[[诺基亚_N75|N75]]和[[诺基亚_N95|N95]]。2007年1月8日，诺基亚发布了[[诺基亚_N76|N76]]、[[诺基亚_N77|N77]]和[[诺基亚_N93i|N93i]]。2007年8月29日，诺基亚发布了[[诺基亚_N95_8GB|N95 8GB]]、[[诺基亚_N81|N81]]、[[诺基亚_N81_8GB|N81 8GB]]，并于同年11月14日发布了[[诺基亚_N82|N82]]，它是第一部有[[氙气闪光灯|氙气闪光灯]]的诺基亚手机。2008年巴塞罗纳的全球移动通信协会上，诺基亚发布了[[诺基亚_N96|N96]]和[[诺基亚_N78|N78]]。同年8月末，诺基亚又发布了两款新N系列手机[[诺基亚_N79|N79]]和[[诺基亚_N85|N85]]。12月2日，诺基亚发布了[[诺基亚_97|N97]]。<ref>http://www.nseries.com/index.html#l=products,n79 {{webarchive|url=https://web.archive.org/web/20080430035008/http://www.nseries.com/index.html |date=2008-04-30 }}</ref><ref>http://www.nseries.com/index.html#l=products,n85 {{webarchive|url=https://web.archive.org/web/20080430035008/http://www.nseries.com/index.html |date=2008-04-30 }}</ref>2009年2月17日，诺基亚发布了N86 8MP，它是诺基亚公司第一部有800万像素的手机。

== 特征 ==
N系列被規劃為諾基亞公司中四大營業單位 (Business Units) 的"多媒體事務部"中，而諾基亞也使用See New、Feel New、Hear New、Watch New、Play New與Do New等總共6個New有關的辭彙，期待N系列能從視覺、聽覺、觸覺等元素帶給使用者全新的使用經驗。
诺基亚N系列的目标人群是那些想把尽可能多的功能装入一部设备的人。N系列通常有比较好的摄像头（其中有很大一部分使用了[[蔡司公司|卡尔蔡司]]光学元件）就是一个这样的例子。
除了N81/N818G，從N95開始，之後的N系列產品都有內建[[GPS|GPS]]、WLAN等功能。另外，除了N72之外，所有的N系列產品皆內建3G(或者更好的)通訊。

== 操作系统 ==

N系列的第一部手机N90，使用旧款的[[Symbian|Symbian]] 8.1 OS，N70也使用这个系统。后来诺基亚推出的N系列手机都转而使用SymbianOS 9(除了基于N70的N72)。新的N系列手机将使用加入了更多新特性的SymbianOS 9系统。

[[诺基亚_N800|N800]][[诺基亚_N810|N810]]以及[[诺基亚N900|诺基亚N900]]网络终端是唯一几部不用Symbian OS系统的N系列手机。它们使用基于[[Linux|Linux]]的[[Maemo|Maemo]]操作系统。<ref>[http://maemo.org/ maemo.org]</ref>
