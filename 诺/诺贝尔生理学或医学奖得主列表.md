'''诺贝尔生理学或医学奖得主列表'''，是[[诺贝尔生理学或医学奖|诺贝尔生理学或医学奖]]的得主列表。

诺贝尔生理学或医学奖于1901年首次颁发，得主是德国科学家[[埃米尔·阿道夫·冯·贝林|埃米尔·阿道夫·冯·贝林]]。每一位获奖者都会得到一块[[奖牌|奖牌]]，一份获奖证书，以及一笔不菲的奖金，奖金的数额每年会有变化<ref>{{cite web |title = The Nobel Prize |publisher = [[诺贝尔基金会|Nobel Foundation]] |url = http://nobelprize.org/nobel_prizes/ |accessdate = 2008-11-21}}</ref>。例如，1901年，冯·贝林得到的奖金为150,782[[瑞典克朗|瑞典克朗]]，相当于2008年12月的7,731,004瑞典克朗；而2008年，[[哈拉尔德·楚尔·豪森|哈拉尔德·楚尔·豪森]]、[[弗朗索瓦丝·巴尔-西诺西|弗朗索瓦丝·巴尔-西诺西]]和[[吕克·蒙塔尼|吕克·蒙塔尼]]分享了总数为一千万瑞典克朗的奖金（略多于100万[[欧元|欧元]]，或140万[[美元|美元]]）<ref>{{cite web |title = The Nobel Prize Amounts |publisher = Nobel Foundation |url = http://nobelprize.org/nobel_prizes/medicine/amount.html |accessdate = 2008-11-21}}</ref>。该奖于每年12月10日，即[[阿尔弗雷德·诺贝尔|阿尔弗雷德·诺贝尔]]逝世周年纪念日，以隆重的仪式在[[斯德哥尔摩|斯德哥尔摩]]颁发<ref>{{cite web|title=The Nobel Prize Award Ceremonies |publisher=Nobel Foundation |url=http://nobelprize.org/award_ceremonies/ |accessdate=2008-11-21 |deadurl=yes |archiveurl=https://web.archive.org/web/20100131133649/http://nobelprize.org/award_ceremonies/ |archivedate=2010-01-31 }}</ref>。

诺贝尔生理学或医学奖得主的研究领域分布相当广。截至2000年，有13名获奖者来自[[神经生物学|神经生物学]]领域，而有13名则在[[代谢|中间代谢]]研究中做出贡献。1939年的获奖者，德国人[[格哈德·多馬克|格哈德·多馬克]]，被其政府禁止领奖。虽然后来他得到了奖牌和获奖证书，却没有得到奖金<ref>{{cite web|title=Nobel Laureates Facts |publisher=Nobelprize.org |url=http://nobelprize.org/nobel_prizes/nobelprize_facts.html |accessdate=2008-11-21 |deadurl=yes |archiveurl=https://web.archive.org/web/20070202025710/http://nobelprize.org/nobel_prizes/nobelprize_facts.html |archivedate=2007-02-02 }}</ref>。截至{{CURRENTYEAR}}年，共有12位女性获得该奖项，人數僅次於16名的和平獎和13名的文學獎，是女性得主第三多的諾貝爾獎項，她们是[[格蒂·科里|格蒂·科里]]（1947年）、[[罗莎琳·萨斯曼·耶洛|罗莎琳·萨斯曼·耶洛]]（1977年）、[[巴巴拉·麦克林托克|巴巴拉·麦克林托克]]（1983年）、[[丽塔·列维-蒙塔尔奇尼|丽塔·列维-蒙塔尔奇尼]]（1986年）、[[格特魯德·B·埃利恩|格特魯德·B·埃利恩]]（1988年）、[[克里斯汀·紐斯林-沃爾哈德|克里斯汀·紐斯林-沃爾哈德]]（1995年）、[[琳达·巴克|琳达·巴克]]（2004年）、[[弗朗索瓦丝·巴尔-西诺西|弗朗索瓦丝·巴尔-西诺西]]（2008年）、[[伊麗莎白·布萊克本|伊麗莎白·布萊克本]]（2009年）、[[卡羅爾·格雷德|卡羅爾·格雷德]]（2009年）、[[邁-布里特·莫澤|邁-布里特·莫澤]]（2014年）和[[屠呦呦|屠呦呦]]（2015年）<ref>{{cite web |title = Women Nobel Laureates |publisher = Nobel Foundation |url = http://nobelprize.org/nobel_prizes/lists/women.html|accessdate=2008-11-21}}</ref>。截至2018年，共有216人获得过诺贝尔生理学或医学奖。该奖有9年因故停发（1915－1918年、1921年、1925年、1940－1942年）。<ref>{{cite web|url=http://www.nobelprize.org/nobel_prizes/medicine/laureates/|title=All Nobel Laureates in Physiology or Medicine|work=Nobelprize.org|accessdate=2014-10-10}}</ref>
