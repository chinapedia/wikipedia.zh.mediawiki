'''诺伊斯'''（Neuss）是[[德国|德国]][[北莱茵-威斯特法伦州|北莱茵-威斯特法伦州]]的一座城市，坐落在[[莱茵河|莱茵河]]下游的西岸，与[[杜塞尔多夫|杜塞尔多夫]]相望，是[[诺伊斯莱茵县|诺伊斯莱茵县]]最大的城市。诺伊斯的人口在1963年超过10万，成为[[大型城市|大型城市]]，它也是德国面积最大的非直辖市（即隶属于县的市）。诺伊斯以它的古罗马历史、莱茵河港口和诺伊斯射击节（Neusser Bürger-Schützenfest）而闻名，在1984年庆祝建城2000周年。

== 地理 ==

诺伊斯坐落在[[莱茵河|莱茵河]]下游的西岸，与[[杜塞尔多夫|杜塞尔多夫]]相望，[[埃尔夫特河|埃尔夫特河]]（Erft）在此汇入莱茵河。海拔最高67.5米，最低位置在港口入口处，海拔30米。诺伊斯市南北向最长距离为13.2千米，东西向最长距离为12.8千米。

== 历史 ==

诺伊斯是德国历史最悠久的城市之一，早在公元前16年，古罗马士兵就在埃尔夫特河汇入莱茵河处建造了一座木质的堡垒。据推测，[[凯尔特人|凯尔特人]]曾经在如今的诺伊斯地区居住，这里的地理优势明显，一方面位于一条重要陆路的终点，这条道路从被[[恺撒|恺撒]]征服的[[高卢|高卢]]一直通往莱茵河，另一方面又处于交通便利的水路，包括莱茵河、埃尔夫特河、[[利珀河|利珀河]]（Lippe）、[[鲁尔河|鲁尔河]]（Ruhr）和[[乌珀河|乌珀河]]（Wupper）。

公元1世纪中期古罗马士兵在埃尔夫特河河口附近建筑了一座石质军营（Castra Novesia），可容纳6500名士兵，在军营附近则建立起来战壕和一座战地城市，供士兵的家人居住，以及其他商人、旅馆主和军械工人工作。公元2世纪初军队撤走后成为居民区，又逐渐建造了防波堤和教堂等建筑。

1190年诺伊斯第一次正式被官方称为城市，神圣罗马帝国皇帝[[亨利六世_(神圣罗马帝国)|亨利六世]]授予诺伊斯关税自由，1200年左右建筑起5扇城门的城墙，1209年Quirinus-Münster大教堂奠基。由于诺伊斯在1474年至1475年间抵抗了勃艮第公爵[[查理_(勃艮第公爵)|大胆的查理]]的长期占领，作为奖赏，[[腓特烈三世_(神圣罗马帝国)|腓特烈三世]]皇帝又授予诺伊斯铸币权、使用红色印章的权利，诺伊斯同时成为一座[[汉萨同盟|汉萨同盟]]城市，并拥有了新的城徽，这些奖赏在诺伊斯的发展中起到了重要的作用，但是1583年至1588年[[科隆战争|科隆战争]]的战败和1586年的一场大火使得诺伊斯一切尽失。

1794年至1814年诺伊斯被[[拿破仑|拿破仑]]占领，1816年成为[[普鲁士|普鲁士]]一个县的首府。19世纪，诺伊斯建造了莱茵河上的铁路桥，连通了莱茵河右岸的杜塞尔多夫和整个铁路网，加上莱茵河港口的建成，使得诺伊斯经历了新的经济复苏。

== 经济 ==

诺伊斯的经济支柱产业主要是港口工业、加工业和服务业。港口主营炼油和集装箱货物运输，以前曾经重要的农业机械制造现在已经不存在。2003年起，诺伊斯港和杜塞尔多夫港统一由一家企业管理，驻地在诺伊斯。除了港口外，城市南部还有大规模的工业区，主要为钢铁、铝制品和纸张加工业，以及食品工业。在过去的几十年里，科技、物流和贸易逐渐赢得更大的比重。诺伊斯的铁路也在城市经济中起很大的作用，使得与[[杜塞尔多夫国际机场|杜塞尔多夫国际机场]]的空中交通枢纽连接便利。2005年诺伊斯居民的平均购买力是德国平均水平的116%，2006年12月统计的失业率为7.9%。
