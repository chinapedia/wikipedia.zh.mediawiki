[[诺贝尔经济学奖|诺贝尔经济学奖]]，正式名称为“瑞典中央银行纪念阿尔弗雷德·诺贝尔经济学奖”（{{lang-sv|Sveriges riksbanks pris i ekonomisk vetenskap till Alfred Nobels minne}}），是由[[瑞典皇家科学院|瑞典皇家科学院]]每年颁发给经济学领域研究人员的奖项<ref name="NobelPrizeSelectionProcess">{{cite web |url=http://nobelprize.org/nomination/economics/process.html |title=Nomination and Selection of the Laureates in Economics |publisher=Nobel Foundation |accessdate=2010-01-10 |archiveurl=https://web.archive.org/web/20100110122603/http://nobelprize.org/nomination/economics/process.html |deadurl=yes |archivedate = 2010-01-10}}</ref>。首次颁奖是在1969年，得奖人有两位，分别是[[朗纳·弗里施|朗纳·弗里施]]和[[扬·廷贝亨|扬·廷贝亨]]<ref name="Winners1969">{{cite web |url=http://www.nobelprize.org/nobel_prizes/economic-sciences/laureates/1969/ |title=The Sveriges Riksbank Prize in Economic Sciences in Memory of Alfred Nobel 1969 |publisher=Nobel Foundation |accessdate=2014-02-13 |archiveurl= https://web.archive.org/web/20140125000312/http://www.nobelprize.org/nobel_prizes/economic-sciences/laureates/1969/ |archivedate=2014-01-25 |deadurl=no}}</ref>。每位获奖者将得到一枚奖牌，一份证书和奖金，奖金的数额已经有过多次调整<ref name="NobelPrizeDescription">{{cite web |url=http://www.nobelprize.org/nobel_prizes/ |title=The Nobel Prize |archiveurl=https://web.archive.org/web/20140209163927/http://www.nobelprize.org/nobel_prizes/ |publisher=Nobel Foundation |accessdate=2014-02-13 |archivedate=2014-02-09 |deadurl=no}}</ref>。1969年时弗里施和廷贝亨一共获得了37万5000[[瑞典克朗|瑞典克朗]]，相当于2007年12月的287万1041瑞典克朗。这个奖项于每年的12月10日，[[阿尔弗雷德·诺贝尔|诺贝尔]]的忌日在[[斯德哥尔摩|斯德哥尔摩]]举行的仪式上颁发<ref>{{cite web | title = The Nobel Prize Award Ceremonies | publisher = Nobel Foundation | url = http://www.nobelprize.org/award_ceremonies/ |accessdate=2010-01-31 |archiveurl = https://web.archive.org/web/20100131133649/http://nobelprize.org/award_ceremonies/ |archivedate = 2010-01-31}}</ref>。

截至2017年，诺贝尔经济学奖一共颁发了49次，共有79人获奖<ref name="NobelLaureatesList">{{cite web |url=http://www.nobelprize.org/nobel_prizes/economic-sciences/laureates/ |title=All Laureates in Economics |publisher=Nobel Foundation |accessdate=2014-10-13}}</ref>。获奖者研究领域包含[[宏观经济学|宏观经济学]]的占9次，比其它任何领域都要多<ref name="FieldsTable">{{cite web |url=http://www.nobelprize.org/nobel_prizes/themes/economic-sciences/lindbeck/table.html |title=Economics Table |archiveurl=https://web.archive.org/web/20131016080213/http://www.nobelprize.org/nobel_prizes/themes/economic-sciences/lindbeck/table.html |archivedate=2013-10-16 |deadurl=no |publisher=Nobel Foundation |accessdate=2014-02-13}}</ref>。有28位经济学奖得主来自[[芝加哥大学|芝加哥大学]]，超过其他任何一个机构<ref name="LaureatesbyUniversity">{{cite web |archiveurl=https://web.archive.org/web/20140122073956/http://www.uchicago.edu/about/accolades/22/#economics |deadurl=no |url=http://www.uchicago.edu/about/accolades/22/#economics |title=Nobel Laureates of UChicago |publisher=The University of Chicago |accessdate=2014-02-13 |archivedate=2014-01-22}}</ref>。
