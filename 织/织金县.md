'''织金县'''是[[中华人民共和国|中华人民共和国]][[贵州省|贵州省]][[毕节市|毕节市]]下属的一个[[县|县]]。面积2868平方公里，2002年人口91万。[[邮政编码|邮政编码]]552100，县政府驻[[文腾街道|文腾街道]]。

== 历史 ==
根据[[清朝|清朝]][[道光|道光]][[进士|进士]][[諶厚光|谌厚光]]编纂的《[[平远州志|平远州志]]》所记载：“织金[[秦朝|秦]]属[[巴郡|巴郡]]；[[汉朝|汉]]属[[牂牁郡|牂牁郡]]；为[[仡佬族|仡佬族]]世居；[[晋朝|晋]]以后[[彝族|彝族]]入主；[[宋朝|宋]]、[[元朝|元]]、[[明朝|明]]为毗喇郡。”<ref>{{Cite web|url=http://www.gz.xinhuanet.com/zfpd/bj/zj/2004-11-26/2004-11-26-2317.htm|title=历史文化名城——织金|accessdate=2018-11-04|date=2004-11-26|publisher=新华网|archiveurl=https://web.archive.org/web/20100209023312/http://www.gz.xinhuanet.com/zfpd/bj/zj/2004-11-26/2004-11-26-2317.htm|archivedate=2010-02-09}}</ref>清[[康熙|康熙]]四年（1665年），设平远府。二十二年（1683年），降为平远州。民国2年（1913年）废州，置平远县。民国3年（1914年）1月以平远州城东有织金街、织金关而改名为织金县。

== 地理与气候==
织金县位于贵州中部偏西，毕节地区的东南，[[乌江|乌江]]上游[[六冲河|六冲河]]与[[三岔河|三岔河]]交汇的三角地带。东距[[贵阳市|贵阳]]151公里，南距[[安顺市|安顺]]95公里，北距[[毕节市|毕节市]]129公里。境内东西长82.5公里，南北宽66公里，总面积2868平方公里。东边与[[平坝|平坝]]、[[清镇|清镇]]相邻；南面接[[普定县|普定]]、[[安顺市|安顺]]；西方接[[纳雍县|纳雍]]、[[六枝特区|六枝]]；北部接[[大方县|大方]]、[[黔西县|黔西]]。

境内山峦起伏，多为[[喀斯特地形|喀斯特地貌]]，[[溶洞|溶洞]]遍布。距县城22公里的[[织金洞|织金洞]]，有“天下第一洞”之称。附近有[[洪家渡水库|洪家渡水电站]]，是[[西电东输|西电东输]]的第一站。地势西南高，东北低。最高[[海拔|海拔]]2262米，最低海拔1319米。

属[[副热带季风气候|亚热带季风气候]]，年平均气温14.7℃，[[降雨量|降雨量]]938.4毫米。全年日照1172小时，[[无霜期|无霜期]]327天。

== 人口 ==
县内有[[汉族|汉]]、[[苗族|苗]]、[[彝族|彝]]、[[回族|回]]、[[布依族|布依族]]等23个民族，少数民族（含[[穿青人|穿青人]]）占48.02％。

== 经济 ==
旅游业发达。旅游区主要分为织金古城、织金洞、结河峡谷、洪家渡四大部分。其中织金洞最为著名，有“天下第一洞”之称。

== 文化 ==
县境内有古寺、古墓、古桥、古塔等多处，但损毁比较严重。县城内的财神庙是全国罕见的古建筑，与[[日本|日本]][[大阪|大阪]]的[[天守|天守阁]]相似，为省级文物保护单位。

=== 著名人物 ===
*[[丁宝桢|丁宝桢]]：曾任[[山东巡抚|山东巡抚]]，斩杀[[安德海|安德海]]。
*[[丁道衡|丁道衡]]：地质学者，发现[[白云鄂博铁矿|白云鄂博铁矿]]。
*[[朱厚泽|朱厚泽]]：原[[中宣部|中宣部]]部长，作风开明。
