[[File:Pell_s_equation.gif|thumb]]
若一個[[丢番图方程|丢番图方程]]具有以下的形式：

:<math>x^2 - ny^2= 1</math>

且<math>n</math>为[[正整数|正整数]]，则称此二元二次不定方程为'''佩尔方程'''（英文：Pell's equation德文：Pellsche Gleichung）。

若<math>n</math>是[[完全平方数|完全平方数]]，则这个方程式只有[[平凡_(數學)|平凡]]解<math>(\pm 1, 0)</math>（实际上对任意的<math>n</math>，<math>(\pm 1, 0)</math>都是解）。对于其余情况，[[拉格朗日|拉格朗日]]证明了佩尔方程总有非平凡解。而這些解可由<math>\sqrt{n}</math>的[[連分數|連分數]]求出。

== 佩尔方程的解 ==

设<math>\tfrac{p_i}{q_i}</math>  是<math>\sqrt{n}</math>的连分数表示：<math>[a_{0}; a_{1}, a_{2}, a_{3}, \,\ldots ]</math>的渐近分数列，由连分数理论知存在<math>i</math>使得(''p''<sub>i</sub>,''q''<sub>i</sub>)为佩尔方程的解。取其中最小的<math>i</math>，将对应的 (''p''<sub>i</sub>,''q''<sub>i</sub>)称为佩尔方程的'''基本解'''，或'''最小解'''，记作(''x''<sub>1</sub>,''y''<sub>1</sub>)，则所有的解(''x''<sub>i</sub>,''y''<sub>i</sub>)可表示成如下形式：
:<math>x_i + y_i\sqrt n = (x_1 + y_1\sqrt n)^i</math>。

或者由以下的[[遞迴關係式|遞迴關係式]]得到：
:<math>\displaystyle x_{i+1} = x_1 x_i + n y_1 y_i,</math>
:<math>\displaystyle y_{i+1} = x_1 y_i + y_1 x_i</math>。

== 例子 ==
===标准型===

:<math>\displaystyle x^2 - 7 y^2 = 1</math>。

首先根据根号7的[[渐进连分数表示|渐进连分数表示]]，找出前几项，察看（分子，分母）是否是一组解。

:第一项：<math>\tfrac{h}{k} = \tfrac{2}{1}</math>，<math>h^2 - 7 k^2 = -3,\,\!</math>不是解；
:第二项：<math>\tfrac{h}{k} = \tfrac{3}{1}</math>，<math>h^2 - 7 k^2 = 2,\,\!</math>不是解；
:第三项：<math>\tfrac{h}{k} = \tfrac{5}{2}</math>，<math>h^2 - 7 k^2 = -3,\,\!</math>不是解；
:第四项：<math>\tfrac{h}{k} = \tfrac{8}{3}</math>，<math>h^2 - 7 k^2 = 1.\,\!</math>是解。于是最小解是(8,3)。计算<math>x_1 + y_1\sqrt n</math>的各次乘方，或者用递推公式（不能直接得出某一项）就可以得到接下来的各组解

:<math>x_n=\frac{(8+3\sqrt{7})^n+(8-3\sqrt{7})^n}{2},y_n=\frac{(8+3\sqrt{7})^n-(8-3\sqrt{7})^n}{2\sqrt{7}}</math>

:(x,y)=(8,3)、 (127,48)、 (2024,765)、 (32257,12192)、 (514088,194307)、 (8193151,3096720)、 (130576328,49353213) ......

===非标准型===
*对于方程<math>x^2-ny^2=k</math>，利用{{link-en|條目名稱|Brahmagupta's identity|Brahmagupta's identity}}找出方程解。

:<math>(x_1^2 - Ny_1^2)(x_2^2 - Ny_2^2) = (x_1x_2 + Ny_1y_2)^2 - N(x_1y_2 + x_2y_1)^2 = (x_1x_2 - Ny_1y_2)^2 - N(x_1y_2 - x_2y_1)^2</math>

例如<math>x^2-7y^2=2</math>有解(3,1)。

<math>r^2-7s^2=1</math>时，有<math>2=(3r+7s)^2-7(3s+r)^2=(3r-7s)^2-7(3s-r)^2</math>

(r,s)=(8,3)、 (127,48)、 (2024,765)、 (32257,12192)、 (514088,194307)、 (8193151,3096720)、 (130576328,49353213) ......

(x,y)=(3,1)、 (45,17)、 (717,271)、 (11427,4319)、 (182115,68833)、 (2902413,1097009)、 (46256493,17483311) ......

*对于方程<math>ax^2-by^2=c</math>，两边乘上a，求出<math>(ax)^2-aby^2=ac</math>的解。

例如<math>5x^2-2y^2=3</math>有解(1,1)。

设<math>z=5x</math>，<math>(5x)^2-10y^2=z^2-10y^2=15</math>

<math>r^2-10s^2=1</math>时，有<math>15=(5r+10s)^2-10(5s+r)^2=(5r-10s)^2-10(5s-r)^2</math>

(r,s)=(19,6)、 (721,228)、 (27379,8658)、 (1039681,328776)、 (39480499,12484830) ......

(z,y)=(5,1)、 (35,11)、 (155,49)、 (1325,419)、 (5885,1861)、 (50315,15911)、 (223475,70669) ......

(x,y)=(1,1)、 (7,11)、 (31,49)、 (265,419)、 (1177,1861)、 (10063,15911)、 (44695,70669) ......

== 与代数数论的联系==

佩尔方程与代数数理论有紧密联系，因为公式<math>x^2 - n y^2 = (x + y\sqrt n)(x - y\sqrt n)</math>给出了环<math>\mathbb{Z}[\sqrt{n}]</math>（即[[二次域|二次域]]<math>\mathbb{Z}(\sqrt{n})</math>）上的[[范数|范数]]。因此(''x'',''y'')是佩尔方程的解当且仅<math>x+y \sqrt{n}</math>的范数是一，即是域上的一个'''单元'''。根据[[狄利克雷单位定理|狄利克雷单位定理]]，<math>\mathbb{Z}[\sqrt{n}]</math>的所有单元都可以表示为同一个'''基本单元'''的乘方形式。这就是说一个佩尔方程的所有的解都是一个基本解的乘方。单元总可以通过解一个类似佩尔方程而得到，但这时的基本解并不一定就是基本单元。

== 与切比雪夫多项式的联系 ==

'''佩尔方程'''和[[切比雪夫多项式|切比雪夫多项式]]有内在的联系：若''T<sub>i</sub>'' (''x'')和''U<sub>i</sub>'' (''x'')分别是第一类和第二类切比雪夫多项式的相应项，那么它们是佩尔形式方程<math>T_i^2 - (x^2-1) U_{i-1}^2 = 1</math>的解。于是第一类和第二类切比雪夫多项式可以通过展开基本解的乘方得到。

:<math>T_i + U_{i-1} \sqrt{x^2-1} = (x + \sqrt{x^2-1})^i</math>。

进一步有：如果(''x<sub>i</sub>'',''y<sub>i</sub>'')是佩尔方程的第''i''个解，那么
:  ''x<sub>i</sub>'' = ''T<sub>i</sub>'' (''x''<sub>1</sub>) 
: ''y<sub>i</sub>'' = ''y''<sub>1</sub>''U''<sub>''i'' - 1</sub>(''x''<sub>1</sub>)。
