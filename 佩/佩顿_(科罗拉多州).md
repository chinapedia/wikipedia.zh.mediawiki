'''佩顿'''（{{lang-en|Peyton}}）是[[美国|美国]][[科羅拉多州|科羅拉多州]][[厄爾巴索縣_(科羅拉多州)|厄爾巴索縣]]的一个[[普查规定居民点|普查规定居民点]]和[[美國郵政署|郵政投递点]]，[[2010年美國人口普查|2010年的人口]]为250人<ref>[http://mcdc.missouri.edu/webrepts/pl94trends/Colorado_stplace2.html Colorado Trend Report 2: State and Complete Places (Sub-state 2010 Census Data).] {{webarchive|url=https://archive.is/20120711184447/http://mcdc.missouri.edu/webrepts/pl94trends/Colorado_stplace2.html |date=2012-07-11 }} </ref>。佩顿与其附近的[[法尔孔|法尔孔]]共用[[美國郵區編號|邮政编码]]80831<ref name="ZIPcode">{{Template:Cite web|date = January 2, 2007|url = http://zip4.usps.com/zip4/citytown.jsp|title = ZIP Code Lookup|format = [[JavaScript|JavaScript]]/[[HTML|HTML]]|publisher = United States Postal Service|accessdate = January 2, 2007}}</ref>。佩顿与其附近的城镇，法尔孔、[[卡尔汉_(科罗拉多州)|卡尔汉]]、[[拉玛_(科罗拉多州)|拉玛]]、[[西姆拉_(科罗拉多州)|西姆拉]]和[[马西森_(科罗拉多州)|马西森]]都散落在[[24号美国国道|24号美国国道]]上，[[科罗拉多斯普林斯|斯普林斯]]和[[利蒙_(科罗拉多州)|利蒙]]之间。
