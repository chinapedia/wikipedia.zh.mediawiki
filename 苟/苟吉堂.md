'''苟吉堂'''（又名'''苟在安'''，{{bd|1898年||1976年}}）[[国民革命军|国民革命军]][[中将|中将]]， [[日本|日本]]士官22期步科。

== 早年 ==
[[四川|四川]][[宣汉|宣汉]]人，十岁丧父失学，从事[[農業|农耕]]。十六岁得到推荐，在[[宣汉中学校|宣汉中学校]]里面当杂工，同时间得到免费在教室外旁听的机会，如此完成[[基础教育|基础教育]]。而后经商，从事[[盐|盐]][[鐵|鐵]]贸易，小有所成。1928年因为兵乱而放弃商业，到[[日本士官学校|日本士官学校]]学习军事。
== 抗日戰爭期間 ==
[[九一八事變|九一八事變]]发生時，苟吉堂参与组织[[中国留日学生总退学|中国留日学生总退学]]，为请愿代表之一，抗议日本侵华。被捕并投入[[日本|日本]][[东京若松町宪兵监狱|东京若松町宪兵监狱]]，拒写悔过书。而后返国。

1931年-1936年，在[[潘文华|潘文华]]部任职，驻防嘉定（现[[乐山|乐山]]）等地。

1937年后，在[[汤恩伯|汤恩伯]]部长期供职，历任第13军[[参谋|参谋]]（军长為[[汤恩伯|汤恩伯]]）、第89师参谋长、第110师参谋长、第85军参谋长、第20军团参谋处长、第31集团军参谋处长、第29军副军长、第3方面军副参谋长、第3方面军前方指挥所主任等职。 

[[卢沟桥事变|卢沟桥事变]]前夕，译释日[[军事|军事]]文本《察哈尔兵要地志》，此文是十三军缴获的，有[[日本|日本]]华北作战的部分[[军事|军事]]要点，为汤部[[华北|华北]]作战提供了重要情报。

卢沟桥事变后，参与了[[南口|南口]]，[[台儿庄|台儿庄]]，[[湘西|湘西]]等第十三军、第二十军团、第三十一集团军、第三方面军之历次战役。  

1945年，任[[上海|上海]]淞沪警备部副参谋长，并受命撰写《中国陆军第三方面军抗战记实》。

1947年，任[[湘南|湘南]]师管区司令。《中国陆军第三方面军抗战记实》于上海出版。

1948年，任[[重庆|重庆]]师管区司令。

1949年，辞去军队职务。

== 晚年 ==
1949年－1951年，在[[重庆|重庆]]租地从事农耕和养殖，并再次尝试[[商业|商业]]。

1951年－1959年，经同乡长者[[西南军政委员会|西南军政委员会]]副主席[[王维舟|王维舟]]介绍，到位于[[成都|成都]]的[[西南民族学院|西南民族学院]]工作。任股长，负责修建教学楼和[[宿舍|宿舍]]、种花木。又在位于[[成都|成都]]红牌楼和西昌县的校办农场工作。后因伤不能干重农活，被令退职。

1966年，[[文革|文革]]爆发，抄家、劳动、被令迁出校园，复又令迁回。

1976年，10月8日晨，从床边落下，股骨颈骨折。10月15日傍晚，在家中逝世。

== 榮譽 ==
作为[[抗日|抗日]]有功人员，获[[光华奖章|光华奖章]]、[[忠勤勋章|忠勤勋章]]、[[云麾勋章|云麾勋章]]、[[美国自由勋章|美国自由勋章]]等。
