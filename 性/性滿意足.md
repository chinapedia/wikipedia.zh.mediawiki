《'''性滿意足'''》（{{lang-en|''Satisfaction''}}）是一部在2014年7月17日晚間10點於[[USA電視台|USA電視台]]首播的影集。<ref>{{Cite web |url=http://www.deadline.com/2014/04/usa-network-summer-rush-satisfaction-royal-pains |title=USA Network Set Summer Premiere Dates |last=Andreeva |first=Nellie |publisher=[[Deadline.com|Deadline.com]] |date=April 23, 2014}}</ref>本劇獲得第二季續訂，並同樣預訂播出10集。<ref>{{cite web|url=http://insidetv.ew.com/2014/10/02/usa-network-renews-satisfaction-cancels-rush/|title=USA Network renews 'Satisfaction,' cancels 'Rush'|work=[[娛樂周刊|娛樂周刊]]|accessdate=24 August 2015}}</ref>

2016年2月26日，USA宣布取消本劇。<ref>{{cite web|url=http://deadline.com/2016/02/satisfaction-canceled-usa-after-two-seasons-1201710182/|title=‘Satisfaction’ Canceled By USA After Two Seasons|work=[[娛樂周刊|娛樂周刊]]|date=February 26, 2016|accessdate=February 26, 2016}}</ref>

== 故事 ==
正當基金經理人尼爾（[[麥特·帕斯莫|麥特·帕斯莫]] 飾演）與妻子葛蕾絲（{{link-en|史黛芬妮·紹姿塔克|Stephanie Szostak}} 飾演）在面對他們之間的夫妻感情與和人生議題時，尼爾意外發現妻子與牛郎發生關係。<ref>{{Cite news |url=http://www.hollywoodreporter.com/live-feed/usa-summer-premiere-dates-satisfaction-698323 |title=USA Debuts First Trailer for Matt Passmore's 'Satisfaction,' Set Summer Premieres |first=Denise |last=Petski |work=[[荷里活報道|好萊塢報導]] |date=April 23, 2014}}</ref> 在偶然的機會下，尼爾開始瞞著妻子當起牛郎，同時，尼爾也開始省思自己的婚姻與人生。

== 製作 ==
本劇由尚恩·雅布朗斯基開創，並由勞斯·克雷斯納弗擔任執行製作。本劇取景於[[亞特蘭大|亞特蘭大]]。<ref>{{cite news |url=http://variety.com/2014/tv/news/usa-network-sets-summer-premiere-dates-for-rush-satisfaction-and-returnees-1201161441 |title=USA Network Sets Summer Premiere Dates for 'Rush,' 'Satisfaction' and Returnees |last=Kissell |first=Rick |work=[[綜藝_(雜誌)|綜藝雜誌]] |date=April 23, 2014 |accessdate=2014-04-23 }}</ref>

== 評價 ==
本劇獲得許多正面評價，於[[爛番茄|爛番茄]]整合的評論中獲得82％的新鮮度<ref name=rotten>{{cite web|title=Satisfaction Season 1|url=http://www.rottentomatoes.com/tv/satisfaction/|website=[[爛番茄|爛番茄]]|accessdate=19 September 2014}}</ref>，在[[Metacritic|Metacritic]]整合的評論中，19位評論者給予本劇62分的分數。<ref name=meta>{{cite web|title=Satisfaction: Season 1|url=http://www.metacritic.com/tv/satisfaction-2014|website=[[Metacritic|Metacritic]]|accessdate=19 September 2014}}</ref>
