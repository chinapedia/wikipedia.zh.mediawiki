'''擁抱'''是可以表達[[情感|情感]]的一種[[身體親密|肢體接觸]]，是指兩個或以上的[[人|人]]相互用[[手|手]]圍著對方的[[身體|身體]]抱起來。每個[[文化|文化]]都對擁抱有著不同的解釋和定義。

== 擁抱文化 ==
在[[西方文化|西方文化]]裡，一般上[[父母|父母]]、[[家人|家人]]和[[朋友|朋友]]之間很久沒見面 和/或 快將告別的時候，都會給對方一個很深的擁抱。而這樣的擁抱也表達了兩個人之間對對方的感覺。

在[[東方文化|東方文化]]裡，由於不鼓勵人與人身體間的親密接觸，故擁抱在東方文化非常罕見。大部分的擁抱只能在[[夫妻|夫妻]]、[[情侶|情侶]]或[[父母|父母]]與年幼子女之間見到。朋友間或成年家人間的擁抱是非常少見的。但在[[中亞|中亞]]國家里，異性之間的擁抱受[[伊斯蘭|伊斯蘭]][[律法|律法]]影響一般不允許在[[公共場所|公共場所]]進行，而同性之間的擁抱卻相當普遍。

在[[中華文化|中華文化]]裡，一般人都不會主動去擁抱，朋友之間忽然在路邊邂逅也一般不會擁抱，古時候，不少人以[[鞠躬|鞠躬]]或抱拳拱手的方式問好，現在取而代之是[[握手|握手]]或[[招手|招手]]。因為大部分的人認為擁抱很[[肉麻|肉麻]]，也覺得在大庭廣眾面前很不好意思。

== 表達愛意 ==
一般上夫妻或情侶間互相擁抱表達[[愛|愛]]意，這樣的擁抱除了簡單的身體碰觸外，也不時出現[[接吻|接吻]]，不少時候也是發生[[愛撫|性行為的前奏]]。

== 表達人情溫暖 ==
當人與人很久沒有見面 和/或 快將告別的時候，雙方互相擁抱以表達對彼此之間的思念。當人在感動、緊張、情緒受到極大的波動的時候，人與人之間也會互相擁抱給對方人情上的溫暖。

== 擁抱風氣 ==
一般上東方國家沒有擁抱的風氣，西方國家的擁抱文化相當普遍。

== 擁抱的方法 ==

== 擁抱與健康 ==
不少人認為由於擁抱直接碰觸對方的身體，而這樣的碰觸對雙方的身體都會產生生理影響，這些生理影響包括突然覺得放鬆，因此擁抱被認為有助於一個人的健康。

2005年，[[美國|美國]][[北卡羅來納州立大學|北卡羅來納州立大學]]的一組[[科學家|科學家]]在38對[[情侶|情侶]]的身上做了擁抱的實驗。<ref>[http://news.bbc.co.uk/2/hi/health/4131508.stm 擁抱如何幫助女性的心臟]</ref>這研究顯示，當情侶擁抱的時候，男女雙方的[[大腦|大腦]]里的[[腦下垂體|腦下垂體]]製造的[[催生素|催生素]]會突然增加，由於催生素能降低[[血壓|血壓]]，因此也能夠減低患[[心臟病|心臟病]]的機率。這項也指出，女性在擁抱過程中，體內的[[皮質醇|皮質醇]]會降低，由於皮質醇會增加血壓，因此，女性在擁抱時候，血壓的降低程度比男性要來得多。

而[[心理學|心理學]]家證實，擁抱具有心理治療的能力，可促進健康、快樂、安定感。擁抱接觸，已被視為醫療的基本工具。<ref>John Alfred Rowe, Minedition (2007), '' I Want a Hug''. ISBN 9780698400641</ref><ref>約翰．艾爾菲．羅伊 ， 格林文化 (2008) ， ''抱抱我'' 。 ISBN 9789861890739</ref>

== 自由擁抱運動 ==
[[自由拥抱运动|自由拥抱运动]]，是指在主动上街拥抱陌生人的一个活動。由[[澳大利亚|澳大利亚]]人Juan Mann的“Free Hugs（自由拥抱）”提出，以拒绝冷漠，通过拥抱向陌生人传递温暖。
