'''旋花'''（[[学名|学名]]：{{lang|la|''Calystegia sepium''}}）为[[旋花科|旋花科]][[打碗花属|打碗花属]]的植物。

== 形态 ==
多年生缠绕[[草本|草本]]。卵形叶子互生，基部为戟形；夏季开淡红色花，花单生于叶腋，花冠漏斗状；球形[[蒴果|蒴果]]；黑褐色卵圆状三棱形种子。

== 分布 ==
分布在[[印度尼西亚|印度尼西亚]]、[[西伯利亚|西伯利亚]]、[[欧洲|欧洲]]、[[澳大利亚|澳大利亚]]、[[北美洲|北美洲]]、[[阿尔泰|阿尔泰]]以及[[中国|中国]]部分地区，生长于海拔140米至2,600米的地区，一般生于路旁、溪边草丛、农田边及山坡林缘，目前尚未由人工引种栽培。

== 别名 ==
蕣华（说文解字）、旋葍（蜀本草）、旋花、旋葍、筋根、续筋根、鼓子花、美草、天剑草（本草纲目） 葍子根（救荒本草） 吊茄子（江西） 篱天剑、饭藤子、饭豆藤、野苕（湖北） 包颈草（湖南） 面根藤、打碗花（贵州） 狗儿弯藤（四川） 打破碗花（云南）
