'''适航指令'''（{{lang-en|Airworthiness Directive}}，通常简写成'''AD'''）是为了通知[[型别验证|已认证飞机]]的所有者和操作者，特定型号的飞机、发动机、航空电子设备或其他系统存在已知的[[空中安全|安全]]隐患，且必须予以纠正。<ref name="TCAIM"> {{cite web|url = http://www.tc.gc.ca/eng/civilaviation/publications/tp14371-lra-2-0-2566.htm#2-7|title = Aeronautical Information Manual, LRA - 2.0 Aircraft Airworthiness, Airworthiness Directives|accessdate = 2008-11-03|last = [[加拿大交通部|加拿大交通部]]|authorlink = |date = October 2008|deadurl = yes|archiveurl = https://web.archive.org/web/20110417023605/http://www.tc.gc.ca/eng/civilaviation/publications/tp14371-lra-2-0-2566.htm#2-7|archivedate = 2011-04-17}}</ref><ref name="CAR605.84"> {{cite web|url = http://www.tc.gc.ca/CivilAviation/Regserv/Affairs/cars/PART6/605.htm#605_84|title = Canadian Aviation Regulation 605.84 Aircraft Maintenance - General|accessdate = 2008-11-03|last = [[加拿大交通部|加拿大交通部]]|authorlink = |archiveurl = https://web.archive.org/web/20081207121257/http://www.tc.gc.ca/CivilAviation/Regserv/Affairs/cars/Part6/605.htm#605_84|archivedate = 7 十二月 2008|deadurl = no|date = January 2008}}</ref> 

如果一架已认证的飞机尚未遵守重要的适航指令，那么它将不适合飞航。<ref name="TCAIM" /><ref name="CAR605.84" />因此，适航指令对于飞机操作者是强制性的。

== 目的 ==
适航指令通常来自營運商报告的运行困难，或航空事故的调查结果。适航指令通常由飞机制造或注册国的民航当局发布。适航指令发布时，注册国与制造国的民航当局几乎总会协调，以确保发布的适航指令没有冲突。

具体而言，适航指令的目的是通知飞机所有者：
* 飞机可能有不安全的因素，或<ref name="CAR593.02"> {{cite web|url = http://www.tc.gc.ca/eng/civilaviation/regserv/cars/part5-standards-593s-1829.htm|title = Canadian Aviation Regulation Standard 593.02 Airwothiness Directives|accessdate = 2008-11-03|last = [[加拿大交通部|加拿大交通部]]|authorlink = |date = January 2008|deadurl = yes|archiveurl = https://web.archive.org/web/20130518090436/http://www.tc.gc.ca/eng/civilaviation/regserv/cars/part5-standards-593s-1829.htm|archivedate = 2013-05-18}}</ref>
* 飞机可能无法符合[[型别验证|认证基础]]或其他条件，影响飞机的适航性，或<ref name="CAR593.02" />
* 必须采取必要行动，以确保持续的安全飞行，或<ref name="CAR593.02" />
* 在一些紧急情况下，飞机将被禁止飞行，直到设计并实施整改行动为止。<ref name="CAR593.02" />

适航指令在大多数地区是强制性的，且往往有遵守指令的限期或飞机飞行小时。

适航指令可以分为两大类：{{Fact|date=2009年4月}}
# 需要在下一次飞行前立即遵守的紧急指令，和
# 需要在指定时间内遵守的较不紧急指令。

== 各国程序 ==
=== 美国 ===
美国联邦航空管理局通过三个不同的程序发布适航指令：<ref name="FAAAD">{{cite web|url = http://www.faa.gov/aircraft/air_cert/continued_operation/ad/type_pub/|title = Types of Airworthiness Directives|accessdate = 2009-10-04|last = [[美国联邦航空管理局|美国联邦航空管理局]]|authorlink = |archiveurl = https://web.archive.org/web/20091016183528/http://www.faa.gov/aircraft/air%5Fcert/continued%5Foperation/ad/type%5Fpub/|archivedate = 16 十月 2009|deadurl = no|date = October 2009}}</ref> 
* 标准适航指令程序：先通告规则拟议（NPRM），其次是最终规则
* 最终规则及征求意见
* [[紧急适航指令|紧急适航指令]]—不征求意见而发布。这只是用来发布“当存在不安全的因素，而需要所有者/操作者立即采取行动……以迅速纠正紧急飞行安全状况。”<ref name="FAAAD" />
