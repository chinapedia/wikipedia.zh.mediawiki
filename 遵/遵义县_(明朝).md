'''遵义县'''，[[明朝|明朝]][[万历|万历]]年间设置的[[县|县]]，今[[贵州省|贵州省]][[遵义市|遵义市]]前身。

明朝初，为播州[[长官司|长官司]]。[[明朝政府|明朝政府]]在[[播州之役|播州之役]]获胜，消灭[[播州杨氏|播州杨氏]]土司政权后，于万历二十九年（1601年）四月设置[[遵义军民府|遵义军民府]]，同时改播州长官司为遵义县，为[[遵义府|遵义府]]的[[附郭县|附郭县]]。县治与府治同迁原司城之西的白田坝，时属[[四川承宣布政使司|四川省]]<ref>《[[明史|明史]]·卷四十三·志第十九》◎地理四○四川、江西 四川......遵义军民府，元播州宣慰司......万历二十九年四月改置遵义军民府。领州一，县四。西北距布政司千七百里。遵义，倚。元播州总管。洪武五年正月改为播州长官司。万历二十九年四月改县，与府同徙治白田坝，在故司城之西。北有龙岩山。其东为定军山，又有大楼山，上有太平关，亦曰楼山关。又东有乌江，源自贵州水西，即涪陵江上源，中有九接滩，其南有乌江关。又东南有仁江，东有湘江、洪江，皆流合於乌江。又西南有落闽水，东有乐安水，亦俱流入焉。又东南有河度关。西南有老君关。又东有三度关。西有落濛关。西北有崖门关、黑水关。北有海龙囤，有白石口隘。桐梓......</ref>。[[清朝|清朝]][[雍正|雍正]]七年（1729年），随遵义府改属贵州省，考评：[[衝繁疲難|冲繁难]]<ref>《[[清史稿|清史稿]]·卷七十五·志五十》◎地理二十二△贵州......遵义府......雍正五年改隶......领厅一，州一，县四。遵义，冲，繁，难。倚。顺治初，因明隶四川。雍正七年，同府改隶。东：香风、三台。西：洪关、元宝、大水田山、娄山。北：大楼、龙岩、定军山。西北：永安山。乌江缘开州入，中渡河、乐闽河及二小水南流注之，又东南会清水江，入瓮安。湘江出县西北龙岩山，二源合南流，洪江合凤凰溪来会，南迳湄潭，至瓮安注乌江。赤水河自仁怀入，沙坝河合数小水北流注之，又纳盐井河，错入桐梓。东：三渡关。西：乌江关、落濛关。北：太平关。驿四：乌江、播川、松坎、湘川。桐梓......</ref>。[[辛亥革命|辛亥革命]]后，于1912年废县。1913年，[[废府州厅改县|全国废府]]，复置遵义县。

1935年1月间，[[红一方面军|红一方面军]]曾攻占遵义县城，[[中共中央政治局|中共中央政治局]]在此召开[[遵义会议|遵义会议]]。1949年时，属[[貴州省_(中華民國)|贵州省]]第五行政督察区。1949年11月，解放军发起[[西南战役|西南战役]]。21日，[[中国人民解放军第10军|解放军第10军]]攻占遵义<ref>{{Cite web |url=http://sichuandaily.scol.com.cn/2009/09/25/20090925559144445813.htm|title=大军进川 我看见红色的潮水涌向四川|accessdate=2016-09-29 |author=记者：覃丹（资料整理），记者：梁现瑞、余向华、王毅 |date=2009-09-25 |work= |publisher=四川在线，来源：四川日报电子版，资料来源：《刘邓大军征战记》第三卷|language=Zh-hans }}</ref>。25日设置遵义县人民政府<ref name="a1">{{Cite web|url=http://epaper.zyol.gz.cn/zyrb/html/2016-05/29/content_301601.htm|title=播州名称来历考|accessdate=2016-09-29|author=|date=2016-05-29|work=|publisher=遵义日报多媒体数字报|language=Zh-hans}}{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>。

1950年，遵义县属[[遵义专区|遵义专区]]，拆分出遵义市（或说1949年11月后<ref name="a1"/>）。次年，遵义市并入，1952年再度拆分出遵义市。1958年，遵义县并入遵义市。1961年，复置。1970年，属[[遵义地区|遵义地区]]。1992年时，辖38个乡镇。1997年后，其下辖乡镇多次划出，分别划入[[红花岗区|红花岗区]]、[[汇川区|汇川区]]、[[新蒲新区|新蒲新区]]。2016年，[[中华人民共和国国务院|中国国务院]]同意遵义县撤县设区，改为[[播州区|播州区]]，并将遵义县部分乡镇划入红花岗区和汇川区<ref>{{Cite web |url=http://gz.people.com.cn/n2/2016/0406/c194827-28087256.html|title=国务院批复同意省政府关于调整遵义市部分行政区划的请示 再见，遵义县 你好，播州区 |accessdate=2016-09-29 |author=胡锐 |date=2016-04-06 |work= |publisher=人民网，来源：商报|language=Zh-hans }}</ref>。
