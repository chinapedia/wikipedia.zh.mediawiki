'''初田牛車站'''（{{jpn|j=初田牛駅|hg=はったうしえき|rm=Hattaushi eki|f=y}}）為一已廢站的車站，位於[[北海道|北海道]][[根室市|根室市]]大字初田牛的[[北海道旅客鐵道|北海道旅客鐵道]]（JR北海道）[[根室本線|根室本線]]（花咲線）的[[鐵路車站|鐵路車站]]。電報略號為'''{{lang|ja|ウシ}}'''。

車站名稱來自[[阿伊努語|阿伊努語]]的「{{lang|ja|オ・ハッタラ・ウシ}}」（o-hattar-us-i，擁有深河口的場所），再簡化成「{{lang|ja|ハッタラウシ}}」（hattar-us-i）並取漢字諧音成為「初田牛」。

== 歷史 ==
* 1920年（[[大正|大正]]9年）11月10日 - 以[[日本國有鐵道|國有鐵道]]車站開始營業。為一般車站。
* 1944年（[[昭和|昭和]]19年）12月2日 - 站房改建。
* 1962年（昭和37年） 12月1日 - 取消整理車攜貨物服務。
* 1964年（昭和39年）4月1日 - 變成業務委託車站。
* 1971年（昭和46年）10月2日 - 廢除貨物、行李處理。同時車站無人化。
* 1977年（昭和52年）10月24日 - 站房改建。
* 1987年（昭和62年）4月1日 - [[國鐵分割民營化|國鐵分割民營化]]後，由JR北海道繼承。
* 2019年（平成31年）3月16日 - 因使用旅客過少而廢站<ref>{{cite web|url=https://www.jrhokkaido.co.jp/CM/Info/press/pdf/20181214_KO_H31Kaisei.pdf|title=2019年３月ダイヤ改正について|publisher=JR北海道|date=2018-12-14|accessdate=2019-03-11}}</ref>。

== 車站構造 ==
初田牛車站為1面1線、[[側式月台|側式月台]]的地面車站。而等候室亦與[[姉別車站|姉別車站]]及[[新吉野車站|新吉野車站]]有相同的設計。亦是無人及秘境車站。

== 車站周邊 ==
位於[[北海道道1127號初田牛厚床線|北海道道1127號初田牛厚床線]]分岔路、礫石路的盡頭。而初田牛地區的會議所及十分稀少的民居則位於等候室旁的森林之中。而[[北海道道142號根室濱中釧路線|北海道道142號根室濱中釧路線]]附近與等候室後面有道路連接，不過由於路軌沒有設置道路，需要用較迂迴的通道前往<ref>車站月台的圖片右邊的人正在用較迂迴的方法穿過路軌。</ref>。
