'''初來橋'''為[[臺灣|臺灣]]一座[[臺東縣|臺東縣]][[海端鄉|海端鄉]]境內的重要聯繫橋梁，由海端鄉公所以及[[交通部公路總局|交通部公路總局]]共同進行管理與維護。其橫跨於[[新武呂溪|新武呂溪]]之上，是[[池上鄉|池上鄉]]進入[[南橫公路|南橫公路]]的重要橋梁。

該橋樑在2010年完工後，全長870公尺，最大淨寬12公尺，設有雙向快車道與慢車道，橋樑結構為鋼箱樑。<ref name="自由時報 〈南部〉池上通往海端 初來新橋今天通車">{{cite web |url=http://news.ltn.com.tw/news/local/paper/424778 |title=自由時報 〈南部〉池上通往海端 初來新橋今天通車  | accessdate=2014-12-05}}</ref>昔日[[池上大橋|池上大橋]]在尚未興建完工時，位在上-{游}-的初來橋是做為[[台9線|台9線]]聯繫[[花東縱谷|花東縱谷地區]]的橋梁，不僅是串起海端與池上的交通，也是當時池上唯一進入南橫公路的橋梁。<ref>{{cite web |url=http://link.safetaiwan.tw/link/04_Bridge/47BridgesRebuildPlan-2011/%E5%88%9D%E4%BE%86%E6%A9%8B/%E5%88%9D%E4%BE%86%E6%A9%8B%E4%BB%8B%E7%B4%B9.htm |title=初來橋介紹 | accessdate=2014-12-05}}</ref>

== 歷史 ==
初來橋最初始建於1977年，而在2009年[[莫拉克風災|莫拉克風災]]後，[[行政院|行政院]]實施全國橋樑總體檢，由於初來橋橋齡已高達33年，加上早因橋寬不足，河床長年下刷，基礎裸露嚴重，危及橋樑安全，因此將其編入[[臺灣省道|省道]]老舊受損橋樑緊急改建計畫，並於2009年7月起在初來橋原址旁動工改建。<ref>{{cite web |url=http://www.pcc.gov.tw/epaper/10003/old.htm |title=公共工程電子報 老舊橋梁全數通車囉！ |accessdate=2014-12-05 |deadurl=yes |archiveurl=https://web.archive.org/web/20141208063758/http://www.pcc.gov.tw/epaper/10003/old.htm |archivedate=2014-12-08 }}</ref>

初來橋工程包含橋梁RC結構、鋼構及橋面版、地面道路工程三大部份，三項主要部份分別施工，惟因初來橋主線高架橋樑P7~P11橋墩位置座落於既有新武呂溪行水區之上，又適逢汛期期間故須先行施作高灘地A1~P6墩柱基礎，以免影響前述整體施工時程，故後期施作[[台20線|台20線]]A1橋台引道道路，全長計200公尺。[[錦屏村|錦屏]]的防汛道路，全長計165公尺。[[台20線#甲線|台20甲線]]A2橋台引道道路，全長計300公尺。三項總工程經費斥資[[新台幣|新台幣]]5億餘元。<ref>{{cite web |url=http://www.thb.gov.tw/tm/Menus/Menu12/object/(1)/(3)-12.html |title=初來橋 工程概況 | accessdate=2014-12-05}}</ref>

初來橋新橋順利於2010年9月3日完工通車，而旁邊的舊橋基於通洪安全考量下，在45天內拆除完畢。<ref name="自由時報 〈南部〉池上通往海端 初來新橋今天通車" />
