'''親吻'''是指用[[唇|嘴唇]]觸碰其他事物，通常是另一個人。特别的，{{lang|ja|'''接吻'''}}<ref>[[漢語中的日語借詞|由日語引入的借詞]]</ref>是指两人的嘴唇互相接触。親吻是一種經過學習的行為，從其他動物之間的行為学习而來。許多[[靈長類|靈長類]]動物也有親吻的行為。據研究，親吻可能也具備了嗅取[[荷爾蒙|荷爾蒙]]的原始[[生物|生物]]意義。<ref>參看 [http://www.sciscape.org/news_detail.php?news_id=1002 一嗅鍾情——費洛蒙與性] {{webarchive|url=https://web.archive.org/web/20070918214022/http://www.sciscape.org/news_detail.php?news_id=1002 |date=2007-09-18 }}{{Zh icon}}</ref>

== 表達愛意 ==
[[File:KISS-キッス8139606A.jpg|thumb]]的[[西洋人|西洋人]]的親吻]]

在現代[[西方文化|西方文化]]中，親吻是一種常見的表達[[愛意|愛意]]的方式。只是在全球168種人類文化中，只有46%把接吻視為一種浪漫的親密行為。人類的祖先可能都沒有相互接吻的行為。<ref>[http://www.solidot.org/story?sid=44936 为什么只有人类接吻]</ref>

在彼此熟悉的兩個人之間，相互親吻是一種见面打招呼或者說[[再見|再見]]的方式。通常，這種親吻表现为短暫地用噘起的嘴唇觸碰[[臉頰|臉頰]]，或是只是單單用臉頰互相接触。在[[歐洲|歐洲]]和[[拉丁美洲|拉丁美洲]]，这是男女之间和女人与女人之間常見的打招呼方式。长辈和[[儿童|孩子]]之间也可以通过親吻来表达感情。

作為一種[[愛情|愛情]]或者[[性慾|性慾]]的表达，親吻表现为两人嘴唇与嘴唇的接触，通常會更加强烈，並且持續更长的[[時間|時間]]。熱情的[[情侶|情侶]]或[[配偶|配偶]]可能會吮吸彼此的[[嘴唇|嘴唇]]，或者將[[舌頭|舌頭]]放到对方的嘴里（[[法式舌吻|法式舌吻]]）。帶有[[性意味|性意味]]的親吻可以是一个人親吻另一个人身體的各个部位。在[[浪漫關係|浪漫]]和[[性感|性感]]的親吻中，[[身體|身體]][[感觉|感觉]]是最重要的，不少時候也會有進一步的行為。

== 象徵意义 ==
{{See also|吻手礼}}
若不是为了表达爱意，亲吻是一种常见的象征性肢体语言，可以表达问候或者服从。这时，亲吻并不是为了体验其带来的身体感觉。
* 在[[欧洲大陆|欧洲大陆]]的许多地方，亲吻脸颊是表示问候的传统[[社交|社交]][[禮儀|禮儀]]。亲吻的次数，接触脸颊的顺序，在不同地区有所不同。
* 亲吻也可以表示敬意和服从，比如亲吻[[君主|君主]]或者[[教宗|教宗]]。
* 有强烈预示意味的亲吻如“告别之吻”或者“死亡之吻”。
* [[英语|英语]]中<ref>英语中，親吻称为{{lang|en|kiss}}，可能是个[[拟声|拟声]]词，研究親吻的科學稱為{{lang|en|Philematology}}</ref>，[[吻手禮|吻手禮]]（{{lang|en|Kissing Hands}}）表示[[英国王室|英国王室]]正式任命国家高级官员。尽管过去官员们真的能够亲吻君主的手，但现在已经不这么做了。当[[英国首相|英国首相]]，内阁成员等被正式任命，他们被称为拥有{{lang|en|Kissed Hands}}。
* 网络上亲吻的符号为 <code>xx</code> 、<code> :-*</code>或<code>>3<</code>。多見於[[IRC|IRC]]、[[即時通訊軟體|即時通訊軟體]]、[[部落格|部落格]]等。

== 其他用途 ==
[[File:GGrandKiss.jpg|right]]親吻，1900年]]
亲吻这个词也可以指不接觸嘴唇而表達情感的動作。
* [[飞吻|飞吻]]只是手和嘴的一组动作。当两人之间的距离比较远，但互相可以看到对方，就可以用飞吻来传递感情。一个人面对一群人时，也可以飞吻。
* {{link-en|爱斯基摩式吻|Eskimo kissing}}（{{lang-en|Eskimo kiss}}）是指兩個人輕輕地觸碰鼻尖，在[[紐西蘭|紐西蘭]]的[[毛利人|毛利人]]文化之中被稱為「Hongi」。
* 英语{{lang|en|Butterfly Kiss}}是指兩個人將眼睛靠近在一起，並且在另一人上面揮動睫毛。

== 姿势 ==
为了避免[[鼻子|鼻子]]的碰撞，接吻双方往往会将他们的脸转向一边，这样他们的头就和对方成一定的角度。有时，为了使接吻有个舒适的姿势，其中一个人会坐下并支撑另一个，也可能将他们的手臂或大腿缠在一起，这样就变成了拥抱并接吻。在接吻中支撑另一方的人往往更主动。《[[自然_(雜誌)|自然]]》杂志报道，通过观察人们在公共场所，诸如[[飞机场|飞机场]]、[[公园|公园]]的接吻，可以发现人们将头偏向右边的比左边的多，比例大约为2:1。<ref>Nature 421, 711 (13 February 2003); doi:10.1038/421711a</ref>

== 接吻的益处和害处 ==
[[File:Samurai_kiss.jpg|right]]
=== 益处 ===
一次接吻可以使面部29块[[肌肉|肌肉]]处于紧张状态，29块肌肉包括12种唇部及17种[[舌头|舌头]]部位的肌肉。这样的效果是使得[[皮肤|皮肤]]光滑、预防[[皱纹|皱纹]]、增加[[血液循环|血液循环]]。<ref name="webxinhua11 ">{{cite web|url=http://news.xinhuanet.com/fashion/2012-01/12/c_122576962.htm|title=蜜吻可以防龋齿 关于接吻的10大健康趣事|accessdate=2012年1月12日|work=|publisher=新华网|language=zh-hans}}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

接吻可以交换[[唾液|唾液]]，包括以下物质：[[水分|水分]]、[[酶|酶]]、[[细菌|细菌]]、[[脂肪|脂肪]]、[[天然盐|天然盐]]以及[[蛋白质|蛋白质]]，使人[[免疫系统|免疫系统]]产生[[抗体|抗体]]。<ref name="webxinhua11 " />

接吻一次燃烧2到3[[卡路里|卡路里]]的[[热量|热量]]，可以达到[[减肥|减肥]]的目的。<ref name="webxinhua11 " />

=== 害处 ===
接吻使得双方[[口腔|口腔]]中的[[细菌|细菌]]通过[[唾液|唾液]]传播，经常性的接吻容易使人得“急性[[接吻病|接吻病]]”，造成[[口疮|口疮]]、[[喉咙|喉咙]][[发炎|发炎]]，更甚者造成[[坏血病|坏血病]]。<ref>{{cite web|url=http://news.xinhuanet.com/health/2004-08/25/content_1879508.htm|title=接吻也会得病 年轻人要警惕"接吻病"|accessdate=2004年8月25日|work=|publisher=新华网|language=zh-hans|deadurl=yes|archiveurl=https://web.archive.org/web/20110326055238/http://news.xinhuanet.com/health/2004-08/25/content_1879508.htm|archivedate=2011年3月26日}}</ref>

== 著名的吻 ==
[[File:The_Kiss_-_Gustav_Klimt_-_Google_Cultural_Institute.jpg|thumbnail]] ]]
* [[基督教|基督教]][[聖經|聖經]][[新約|新約]]中写到，[[加略人猶大|加略人猶大]]亲吻了[[耶稣|耶稣]]并背叛了他。
* 英国海军统帅[[霍雷肖·纳尔逊|霍雷肖·纳尔逊]]的临终遗言就是对他的下属说“吻我，哈代”（“{{lang|en|Kiss me Hardy!}}”，也有另一个版本说他说的是“{{lang|en|Kismet, Hardy}}”）。
* 童话《[[睡美人|睡美人]]》、《[[白雪公主|白雪公主]]》和一些其他的的一些神话故事中，男主人公通过一个浪漫的吻唤醒了女主人公。在《[[黑客帝国|黑客帝国]]》中，Trinity通过吻将Neo救活了。
* 童话《[[青蛙王子|青蛙王子]]》中，一个浪漫的吻将王子从青蛙变回了人。
* [[奥古斯特·罗丹|奥古斯特·罗丹]]创作了雕塑“吻”（''Le Baiser''）。
* 《[[亂世佳人|亂世佳人]]》中[[克拉克·蓋博|克拉克·蓋博]]一吻[[費雯麗|費雯麗]]，事後費雯麗卻說十分厭惡蓋博的假牙臭味。
* 《[[小姐与流浪汉|小姐与流浪汉]]》中，当小姐和流浪汉同时吃一根[[面条|面条]]时，他们的嘴唇最终碰到了一起。
* 1945年8月14日（北京时间8月15日）发生在纽约时代广场的一幕亲吻。时值日本宣布无条件投降，纽约民众纷纷走上街头庆祝胜利。一位水兵在时代广场的欢庆活动中亲吻了身旁的一位女护士，这一瞬间被《生活》 杂志的摄影师阿尔弗雷德·艾森施泰特抓拍下来，成为传世的经典历史画面（[[胜利之吻|胜利日之吻]]）。从此以后，每年8月14日都有数百对男女在时代广场重现“胜利日之吻”，以纪念二战结束。<ref>[http://news.sohu.com/20120504/n342300317.shtml 二战“胜利之吻”男女主角真实身份被揭开(图)] 搜狐网{{Zh icon}}</ref><ref>[http://news.sohu.com/20100623/n273021972.shtml 美国经典“二战胜利之吻”照片女主角去世] 搜狐网{{Zh icon}}</ref>
* 世界上有记录的最长的吻发生在1999年4月5日，[[以色列|以色列]]的[[特拉维夫|特拉维夫]]。这一纪录为30小时45分钟，由 Karmit Tsubera 和 Dror Orpaz 创造。
