'''親泊朝省'''（{{jpn|j=親泊朝省|hg=おやどまり ちょうせい|rm=Oyadomari Chōsei}}、{{bd|1903年|9月18日|1945年|9月3日}}），[[琉球族|琉球族]]，本姓向，[[沖繩縣|沖繩縣]]出身。是[[大日本帝國陸軍|大日本帝國陸軍]][[軍人|軍人]]，最終軍階為陸軍[[大佐|大佐]]。[[陸軍士官學校_(日本)|陸軍士官學校]]37期騎兵科第一名畢業。曾任陸軍士官學校[[教官|教官]]、[[大本營_(大日本帝國)|大本營]]陸軍部報道部部長、[[大本營_(大日本帝國)|大本營]]情報官。終戰後攜同妻兒自殺身亡<ref name="略歴">{{Cite web |title=親泊朝省|url=http://www.asahi-net.or.jp/~un3k-mn/ziketu-oyadomari.htm}}</ref><ref name="親泊朝擢">{{Cite web |title=親泊朝擢|url=http://www6.plala.or.jp/guti/cemetery/PERSON/A/oyadomari_ct.html}}</ref><ref name="親泊朝省">{{Cite web |title=親泊朝省|url=http://www6.plala.or.jp/guti/cemetery/PERSON/A/oyadomari_c.html}}</ref><ref name="大本営">{{Cite web |title=大本営陸軍報道部部長　親泊朝省大佐|url=http://midorikaze.com/midori.html}}</ref>。

==家世==

[[File:菅波三郎と親泊朝省.jpg|thumb]]
親泊朝省為[[琉球國|琉球國]][[第二尚氏王朝|第二尚氏王朝]]第三任國王[[尚真王|尚真王]]之長子、廢世子[[尚維衡|尚維衡]]（浦添王子朝滿）的後裔，為[[向氏親泊家|向氏親泊家]]成員。父親[[親泊朝擢|親泊朝擢]]和二弟[[親泊朝晉|親泊朝晉]]為教育家，二弟婦為琉球國末代國王[[尚泰王|尚泰王]]之四子[[尚順|尚順]]男爵之五女[[尚芳子|尚芳子]]<ref name="親泊朝晋">{{Cite web |title=親泊朝晋|url=http://www6.plala.or.jp/guti/cemetery/PERSON/A/oyadomari_cs.html}}</ref>。三弟名[[親泊朝圭|親泊朝圭]]。長姊[[宮城鶴|親泊鶴]]嫁水產事業家{{link-ja|宮城新昌|宮城新昌}}，育有兩位女兒{{link-ja|岸朝子|岸朝子|朝子}}和[[尚道子|道子]]皆為[[美食家|美食家]]，道子之夫為尚泰王之孫{{link-ja|尚明|尚明}}。親泊朝省娶其[[摯友|摯友]]、[[大和族|大和族]]的陸軍[[大尉|大尉]][[菅波三郎|菅波三郎]]之妹[[菅波英子|菅波英子]]為妻，夫婦育有一女[[親泊靖子|親泊靖子]]和一子[[親泊朝邦|親泊朝邦]]<ref name="親泊朝擢" /><ref name="親泊朝省" /><ref name="大本営" />。

親泊朝省出生於原屬琉球國領土的日本沖繩縣[[首里|首里]][[大中町|大中町]]，是父親親泊朝擢和母親牛（{{lang-ryu|ウシ}}）的第三個孩子，有兩姊二弟，故為長子。他最初就讀於父親任校長的[[大宜味尋常小學校|大宜味尋常小學校]]，幼年時的他好讀書，[[作文|作文]]出色，性格溫順。後來升讀[[沖繩縣立第一中學校|沖繩縣立第一中學校]]（現為{{link-ja|沖繩縣立首里高等學校|沖縄県立首里高等学校}}），兩年後轉校就讀{{link-ja|熊本陸軍地方幼年學校|熊本陸軍地方幼年学校}}，並在校內認識到終生之友菅波三郎，後來一起升讀位於[[東京市|東京市]]（今屬[[東京都|東京都]]）{{link-ja|市谷|市谷}}的陸軍士官学校。1923年預科畢業，並成為羅南騎兵第二十七聯隊第二中隊士官候補生。1925年以第一名的成績畢業於騎兵科，並回歸原隊<ref name="略歴" /><ref name="大本営" /><ref name="親泊朝省" />。

1930年，親泊朝省迎娶菅波三郎之妹英子。1931年，[[九一八事變|九一八事變]]爆發，他隨軍出征[[中國|中國]]。1934年成為陸騎兵第25連隊中隊長，翌年長女靖子出生，一年後擔任[[陸軍大學校|陸軍大學校]][[馬術|馬術]]教官，次年升為[[參謀本部_(大日本帝國)|參謀本部]]副官。1939年晉升為騎兵[[少佐|少佐]]。翌年他在陸軍大學校專科畢業，8月就任[[第38師團_(日本陸軍)|第38師團]]作戰主任參謀，在[[中國|中國]][[廣東|廣東]]進行戰爭，同年長男朝邦出生。次年隨軍[[香港保衞戰|攻打香港]]，翌年再到[[荷屬東印度|荷屬東印度]]的[[爪哇島|爪哇島]]（今屬[[印尼|印尼]]）參與{{link-en|爪哇戰役|Battle of Java (1942)}}，後來因為[[美軍|美軍]]投入[[前線|前線]]，日軍在該年秋季後死守[[瓜達爾卡納爾島|瓜達爾卡納爾島]]，親泊朝省亦隨軍留守當地。由於當地[[糧食|糧食]]極度不足，加上[[疫症|疫症]]流行，軍隊決定在來年2月撤退，身為陸軍[[中佐|中佐]]的親泊朝省的身體也較以往瘦削不少<ref name="大本営" />。

1943年，親泊朝省返回日本東京，擔任陸軍士官學校教官，次年被任命進入[[大本營_(大日本帝國)|大本營]]情報，他擔任大本營陸軍部報道部部長以及{{link-ja|內閣情報局|内閣情報局}}情報官，其任務包括向民間報導機關提供戰況消息。這段時間他因為要保守軍隊機密，未能向國民報導實際情況，內心持續感到不安和抱歉，也令他覺得這工作極為艱辛<ref name="略歴" /><ref name="親泊朝省" />。

1945年，親泊朝省升任陸軍[[大佐|大佐]]。同年8月15日，[[昭和天皇|昭和天皇]]頒下《[[終戰詔書|終戰詔書]]》，身為陸軍軍人的親泊朝省深切反省日軍在[[大東亞戰爭|大東亞戰爭]]中的暴行，開始撰寫批判日軍暴行和[[軍國主義|軍國主義]]的《[[草莽之文|草莽之文]]》（{{lang-ja|草莽の文}}），並於8月20日付印。[[大日本帝國|大日本帝國]]於9月2日[[日本投降|宣佈投降]]，翌日早上，他在位於[[東京都|東京都]]{{link-ja|小石川區|小石川区}}的寓所中身穿{{link-ja|軍服 (大日本帝國陸軍)|軍服 (大日本帝国陸軍)|軍服}}，[[注射|注射]][[氰化鉀|氰化鉀]]殺死年僅十歲的女兒靖子以及年僅五歲的兒子朝邦，再槍殺身穿盛裝、化淡妝的妻子英子，然後對準[[心臟|心臟]]開槍自殺身亡。後來與妻兒合葬在位於東京都[[府中市|府中市]]和[[小金井市|小金井市]]的{{link-ja|多磨靈園|多磨霊園}}裡面之親泊家墓地<ref name="大本営" /><ref name="親泊朝省" />。

==《草莽之文》內容簡介==
《草莽之文》共二萬一千餘字，以53-54張[[原稿紙|原稿紙]]寫成，分為六章，是親泊朝省的[[遺書|遺書]]。書中表達了一個親身參與戰爭的日本陸軍軍人對日軍行為的反省以及對日軍暴行的批判。他對日本遠征軍、尤其是1937至1938年在中國濫殺無辜、姦淫擄掠、對「同民族」的[[中國人|中國人]]{{noteTag|原文為「[[支那人|支那人]]」，「同民族」是指同屬東洋民族<ref name="大本営" />}}之蔑視，感到十分遺憾，其中在第四章提及關於[[南京大屠殺|南京大屠殺]]的內容。又批判了日軍的殘暴、專制，指[[九一八事變|九一八事變]]、[[七七事變|七七事變]]等都是軍方上層獨行獨斷而引發。他認為日軍的暴行只對軍方的上層人員有利，卻令皇軍[[權威|權威]]掃地，引發陸軍和[[大日本帝國海軍|海軍]]對立，官、軍、民心皆逐漸失去，並且世界各國因看見日軍的暴行而對日本不信任。書中揭露了一些日本政府隱瞞的事實，並且證明了南京大屠殺之真實性<ref name="大本営" />。
