'''厭跨女症'''（{{lang-en|Transmisogyny或Trans-misogyny}}），是[[跨性別恐懼症|跨性別恐懼症]]和[[厭女症|厭女症]]的交集。[[跨性別恐懼症|跨性別恐懼症]]被定義為“對跨性別者的無理恐懼，厭惡或歧視”。<ref>Merriam Webster [https://www.merriam-webster.com/dictionary/transphobia]</ref>[[厭女症|厭女症]]被定義為“對女性的仇恨”。<ref> Merriam Webster[https://www.merriam-webster.com/dictionary/misogyny]</ref>因此，[[厭跨女症|厭跨女症]]包括對性別頻譜偏向女性一側的[[跨性別|跨性別]]人士或[[變性人|變性人]]持負面的態度，仇恨和[[歧視|歧視]]，特別是[[跨性別女性|跨性別女性]]。<ref name="fmn">{{cite web|url=http://everydayfeminism.com/2014/01/transmisogyny/|title=Transmisogyny 101: What It Is and What Can We Do About It|date=2014-01-27|publisher=everyday feminism|accessdate=2015-01-27}}</ref>

== 狀況 ==

[[跨性別女性|跨性別女性]]因為轉換為性別階級低的[[女性|女性]]而被[[歧視|歧視]]<ref name="ethic" /><ref name="fmn" />，[[跨性別男性|跨性別男性]]相對沒有這個問題，由父權文化下的[[厭女|厭女]]與[[恐跨|恐跨]]交織性歧視，同時面臨[[女性|女性]]與[[跨性別|跨性別]]的雙重弱勢身分<ref name="ethic">{{Cite book |title=Sexual deceit: the ethics of passing |url=http://books.google.com/books?id=m4OGpJXp_oQC&pg=PA12 |page=12 |year=2013 |first=Kelby |last=Harrison |publisher=Lexington Books}}</ref>，像是半數以上的跨性別女性都在生命中遇過各種形式的[[性暴力|性暴力]]經驗，具有[[陰柔氣質|陰柔氣質]]的[[性別酷兒|性別酷兒]]也深受此害<ref name="fmn" />。

== 與恐跨症之間的差異 ==

厭跨女症與[[恐跨症|恐跨症]]並不一樣，恐跨症是指對於所有[[跨性別者|跨性別者]](包含[[跨性別男性|跨性別男性]]與[[跨性別女性|跨性別女性]])的恐懼與歧視，厭跨女症則針對[[跨性別女性|跨性別女性]]，這牽涉到[[順性別主義|順性別主義]]與[[男性沙文主義|男性沙文主義]]下，[[性別|性別]]實際存在著不平等問題，普遍上社會[[歧視|歧視]]女性，女性被視為次等的，[[男跨女|跨女]]便是從原本應為既得利益者的身分成為性別階級較為低階的女性，承襲對男性的期待，與對女性的貶抑，因此受到嚴重的貶抑。

== 排除 ==

跨性別女性常常被排除在對女性的保護與保障之外，特別是女性空間的使用，因此更容易遭到針對女性的歧視、騷擾與暴力，並成為被高度針對的目標。原因常是跨性別女性常常被誤認為是別有所圖的順性別男性，與被部分順性別女性主義者認為帶有男性優勢，抑或著認為社會性別與性別認同是父權的產物，所以跨性別女性不如順性別女性一樣是「天生自然」的女性。

== 樣貌歧視 ==

我們可以發現不符合主流審美觀與[[異性戀|異性戀]][[男性凝視|男性凝視]]的[[跨性別女性|跨性別女性]]，遭到更加惡劣的[[歧視|歧視]]與異樣眼光；更難以取得社會上的資源，而且也很難有辦法避免遭受暴力。

== 不孕歧視 ==

跨性別女性常因為無法依照自我認同的性別角色生育，或者是經過絕育手術，特別是因為無法[[懷孕|懷孕]]，而遭到歧視。像是基於此原因而被認為不是「真正的女人」，或是在強調[[傳宗接代|傳宗接代]]的文化下，被認為是次等的女性。

== 跨性別女同志 ==
還有一種針對跨性別女同志（喜歡女性的[[跨性別女性|跨性別女性]]）的歧視，稱作'''恐跨女同'''或'''恐跨拉'''（Translesbophobia），同時是[[厭女|厭女]]、[[恐同|恐同]]、[[恐跨|恐跨]]，以及[[恐拉|恐拉]]與'''厭跨女'''等交叉[[歧視|歧視]]，多半是社會假定跨性別者一定是異性戀者（喜歡的[[性別|性別]]一定與[[性別認同|性別認同]]相反），作為一種[[異性戀主義|異性戀主義]]與[[性別二元|性別二元]]的展現。

== 跨性別娼妓 ==
許多[[跨性別女性|跨性別女性]]會因為社會壓迫，諸如：就學就業[[歧視|歧視]]、原生家庭的[[經濟|經濟]]斷絕、[[醫療|醫療]]需求（[[賀爾蒙|賀爾蒙]]、[[整形|整形]]、[[性別重置手術|性別重置手術]]）等需求，因此從事性產業。但同時這也是由於社會將跨性別女性給[[物化|色情化]]與[[獵奇|獵奇化]]所造成的，嫖客許多為異性戀男性覺得好玩、新奇、嘗鮮等因素而進行交易，因此常有不尊重甚至對跨性別娼妓[[施暴|施暴]]的現象。或者有些跨性別娼妓會隱瞞身分為[[順性別|順性別]][[女性|女性]]，但被發現時就很容易因客人惱羞成怒或感覺被欺騙，而遭受暴力。
