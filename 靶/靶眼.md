'''靶眼'''（{{lang-en|Bullseye}}），本名'''萊斯特'''，是一名在[[漫威漫畫|漫威漫畫]]出版的漫畫中登場的[[超級反派|超級反派]]。由Marv Wolfman和John Romita, Sr.所創作，首次出現於《[[夜魔俠|夜魔俠]]》#131（1976年3月）。

靶眼在IGN前100名漫畫書惡棍中排行第二十<ref>{{cite web |url=http://comics.ign.com/top-100-villains/20.html |title=Bullseye is Number 20 |publisher=Comics.ign.com |date= |accessdate=2011-02-06 |deadurl=yes |archiveurl=https://web.archive.org/web/20110203091526/http://comics.ign.com/top-100-villains/20.html |archivedate=2011-02-03 }}</ref>，他同時也在前五十位復仇者排名中名列第三十五名<ref>{{cite web |last= |first= |date=April 30, 2012 |title=The Top 50 Avengers |url=http://www.ign.com/top/avengers/35 |publisher=[[IGN|IGN]] |accessdate=July 28, 2015}}</ref>。

==簡介==
靶眼是個來歷不明的殺手，他接受[[金霸王_(漫威漫畫)|金霸王]]的雇傭成為其頭號殺手，替他剷除在擴張其犯罪帝國的道路上所會遇到的阻礙，也就是如此，靶眼與和金霸王作對的夜魔俠開始了爭鬥，直到他被捕入獄為止。

在入獄後，金霸王改雇傭[[幻影殺手_(漫畫)|幻影殺手]]成為他的頭號殺手，逃獄後的靶眼為了奪回自己的地位而殺死了她，這引來了曾是幻影殺手戀人的夜魔俠的憤怒，於是夜魔俠找上了靶眼，在決鬥後靶眼從高處摔落並摔斷了他的脊椎導致癱瘓。

之後日本的一名罪犯科學家小山賢二找上了癱瘓的靶眼，並用亞德曼金屬修復了他的脊骨，原本只是普通人類的靶眼不但因此恢復了健康，在亞德曼金屬的協助下更令他有了強悍的肉體。於是他再次和夜魔俠展開爭鬥，並殺了他的戀人[[凱倫·佩吉|凱倫·佩吉]]。

秘密入侵事件後，[[綠惡魔|諾曼·奧斯朋]]因立下大功而取代[[鋼鐵人|東尼·史塔克]]成為美國政府的代言人，並改組神盾局為天錘局，他找來靶眼成為其組建的[[黑暗復仇者|黑暗復仇者]]的一員，並讓靶眼以[[鷹眼_(漫畫)|鷹眼]]的身分出現<ref>[http://www.comicbookresources.com/?page=article&id=19654 THE OSBORN SUPREMACY: Dark Avengers], [[Comic_Book_Resources|Comic Book Resources]], 2008-01-22</ref>。

==能力==
靶眼沒有任何超能力，他有著優越的體能、敏捷、耐力、速度與反應。

他最著名的是擁有超人等級的投擲技術，他可以用[[撲克牌|撲克牌]]撕裂目標的喉嚨，也能用牙籤殺死一百碼（約91公尺）外的對手。

靶眼同時也是一個神射手，不管是使用常規武器還是冷兵器，他都能精準地命中目標。除此之外也是近身搏鬥的高手。

在和夜魔俠決鬥而導致癱瘓後，小山賢二教授用亞德曼金屬修復了他的脊椎，這使他擁有了更耐打的肉體。

在行動之前靶眼會仔細研究目標的一切，好讓他在展開暗殺行動時會更加的順利。
