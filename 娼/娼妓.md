'''娼妓'''，又稱'''妓女'''、'''婊子'''、'''窯姐'''，是一種以提供陪伴及[[娛樂|娛樂]]（其目的可能包括[[心靈|精神上]]或[[性慾|肉體上]]的滿足）為主的[[行業|行業]]。傳統上許多娼妓以奉客、陪客喝酒解苦、[[表演|表演]]為主，具有較高的[[藝術|藝術]]素養，即「賣藝」不「賣身」的[[交際花|交際花]]；但由於娼妓業工作的演變，近代娼妓一詞專指提供[[性交易|性服務]]的[[女性|女性]][[性工作者|性工作者]]。妓女因工作內容社會地位低，可為中性用法或貶義詞。

== 歷史 ==
据史家研究，各大文明古国的娼妓都发源于[[宗教|宗教]]儀式。最早流行于[[古巴比伦|古巴比伦]]，古巴比伦不同地位的女子，不管尊卑贵贱，都必须在神庙里当一段时间的[[庙妓|庙妓]]，在神庙里贡献自己的[[贞操|贞操]]，以示对神的虔诚。中国[[殷商|殷商]]时期，也有一群类似“宗教卖淫”的女子，名字叫巫娼，她们借着与神灵相通的名义，通过妖艳的外表，能够魅惑男子<ref>{{cite news|url=http://news.ifeng.com/history/shijieshi/detail_2013_10/15/30328972_0.shtml|title=娼妓的起源是什么？古巴比伦女子不论贵贱都须当庙妓|date=2013-10-15|publisher=凤凰历史|accessdate=2014-10-26}}</ref>。後來這些娼妓開始世俗化，變成以表演、陪侍客人為業。妓女也分有很多种，有的不一定提供性服務，有些是賣藝不賣身的[[藝妓|藝妓]]或[[歌妓|歌妓]]，專門為客人彈唱、獻藝。唐代著名女詩人[[薛濤|薛濤]]出身官宦世家，其父病亡，便身入樂籍，“言语巧偷鹦鹉舌，文章分得凤凰毛”，<ref>[[元稹|元稹]]《寄赠薛涛》：“锦江滑腻峨眉秀，幻出文君与薛涛。言语巧偷鹦鹉舌，文章分得凤凰毛。纷纷词客皆停笔，个个公侯欲梦刀。别后相思隔烟水，菖蒲花发五云高。”</ref>後來[[元稹|元稹]]、[[白居易|白居易]]、[[杜牧|杜牧]]都是她的貴賓，日後被稱為“校書”，<ref>[[何光远|何光远]]《鉴戒录》谓: “自韦皋镇成都日，令入乐籍，呼为女校书。”[[陳振孫|陳振孫]]《[[直齋書錄解題|直齋書錄解題]]》卷十九-{云}-：“號薛校書，世傳奏授，恐無是理，殆一唐文拔萃，公侯皆欲往成都任職。”</ref>可說是是妓女中的極品。時人[[王建_(唐朝)|王建]]有《寄蜀中薛涛校书》詩：“萬裏樓臺女校書，琵琶花下閉門居，掃眉才子知多少，管領春風總不如。”《[[全唐诗|全唐诗]]》收薛涛诗八十九首。宋朝文人[[柳永|柳永]]一生都在[[花柳|妓院]]巷裡親熱唱和，大部分的词诞生在[[青樓|青樓]]笙歌艳舞、锦榻绣被之中<ref>[[張舜民|張舜民]]《畫墁錄》載：[[晏殊|晏殊]]斥柳永“彩線慵拈伴伊坐”乃俗艷之詞。</ref>，柳永晚年窮愁潦倒，在潤州去世時一貧如洗<ref>[[叶梦得|叶梦得]]《避暑录话》卷下记载：“永终屯田员外郎，死旅，殡润州僧寺。王和甫为守时，求其后不得，乃出钱葬之。”</ref>，是他的歌妓姐妹們集資營葬。<ref>[[祝穆|祝穆]]《方輿勝覽》記載：“柳永卒於襄陽，死之日，家無餘財，群妓合資葬於南門外。每春日上塚，謂之‘吊柳七’。”</ref>

近代以來的妓女仍有分等級，書寓是上海最高級的妓女，也叫詞史、校書，長三是與書寓同等級的高級妓女，但不避諱性交易，<ref>[[徐珂|徐珂]]《[[清稗類鈔|清稗類鈔]]·娼妓類》，卷三十八-{云}-：“論滬妓之差等，輒曰書寓、長三、么二，是固然矣。”</ref>晚清名妓李萍香出身望族，“工吟咏，擅书画……女红之余，兼攻词翰”。至於下等妓院的妓女為求生意，多當街拉客，許多文人礙於面子亦避之唯恐不及。清末戰亂頻仍，難民多避難租界，同治年間上海租界的妓院達千餘家。<ref>[[陳其元|陳其元]]《庸閒齋筆記》：“娼寮妓館，趁風駢集，以洋人為護符......余攝縣事時，欲稍稍裁抑之，而勢有不能，嘗飭洋租地保密稽之。蓋有名數者，計千五百餘家，而花煙館及鹹水妹、淡水妹等等，尚不與焉。”</ref>妓女從業的原因大都是家境貧寒。妓院也有供應瓜果、茶水，稱為「打茶圍」，召妓侍座侑酒稱「叫局」，打麻將稱「碰和」。

清初著名文人[[袁枚|袁枚]]、[[李寧|李寧]]等人都是妓院常客。[[俄羅斯|俄羅斯]]文学家[[托尔斯泰|托尔斯泰]]年輕時常去妓院，他在1847年的日记记录“染上了淋病”，正在接受治疗，[[水银|水银]]的副作用让他承受著说不出的痛苦。歐洲短篇小說之王[[莫泊桑|莫泊桑]]（Guy de Maupassant）也是妓院常客，他的小說〈泰利葉之家〉（La Maison Tellier）便是以妓院為題材，由於縱情聲色，莫泊桑因此染上[[梅毒|梅毒]]，導致腦病變瘋狂，1893年在精神病院割喉了結一生。<ref>《莫泊桑的生命悲歌》，顏坤琰，《世界文化》2003年第4期</ref>至於高級妓女接待客人也分等級，身份不高也進不了場子，若想一親芳澤，可謂難上加難。高級妓女對象多為文人或權貴，但其身份仍受盡外界歧視，即便兩廂有情，礙於低下的社會地位也很難開花結果。也有從良的妓女最後因夫妻之間缺乏真實的情感，或沉溺於過往的紙醉金迷，而被迫返回妓業。“閉門羹”的典故則是來自於唐代名妓史鳳，[[馮贽|馮贽]]《雲仙雜記》引《常新錄》：“史鳳，宣城妓也。待客以等差……下列不相見，以閉門羹待之。”

東亞傳統娼妓由見習至正式娼妓的通過儀禮稱為[[梳攏|梳攏]]，本來是女性的成人禮[[笄禮|笄禮]]的別稱。對於娼妓來說，梳攏不但代表成年，還表示已經接受了足夠的訓練，可以正式接待客人，也常常包括出賣[[初夜|初夜]]。

== 現狀 ==
近代至現代除了打正旗號賣淫的性工作者外，還有些女性從事陪侍工作，也屬於廣義的娼妓。例如伴舞的[[舞女|舞女]]、卡拉OK夜總會的[[伴唱|伴唱]]、[[夜總會|夜總會]]的[[坐檯小姐|坐檯小姐]]等，這些娼妓未必會與客人發生性行為，但有些於陪侍時容許客人觸摸身體。

一些地區的傳統娼妓在現代社會仍尊循古法，作古裝打拌賣藝表演，如日本的[[藝妓|藝妓]]、朝鮮的[[妓生|妓生]]。但因培育與工作辛苦且客群有限，已愈來愈少女性願意投身這類傳統表演。

== 地位 ==
由於世界上多數社會的傳統觀念認為女性不應出賣肉體賺錢，因此娼妓這種需要與眾多男性應酬、交流的行業，無論是否「賣身」，均被視為低下階層，不受人尊重。
