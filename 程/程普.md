'''程普'''{{BD|？年||210年||catIdx=C}}，字'''德谋'''，[[右北平|右北平]]土垠人（今河北[[丰润|丰润]]）。[[東漢|東漢]]末年[[孫堅|孫堅]]麾下將軍，[[孙吴|東吳]]孫氏三代元勳之首；是[[江表之虎臣|江東十二虎臣]]之首。因為軍中最為年長，所以被人們尊稱為'''程公'''。

== 生平 ==
===犬馬之力===
{{see also|董卓討伐戰|孫策平江東之戰}}
早年在州郡擔任官吏，後跟隨[[孙坚|孙坚]]四處征戰，黄巾餘賊於[[宛|宛]]、[[邓|邓]]，破董卓於[[阳人|阳人]]等都有參加，數有戰功。

191年，孙坚戰死，繼續跟随其子[[孙策|孙策]]到[[淮南|淮南]]起兵，攻克[[庐江|庐江]]，後跟隨孫策到[[横江|横江]]、[[当利|当利]]，大破[[張英_(東漢)|张英]]、[[于麋|于麋]]等，又展转進攻[[秣陵|秣陵]]、[[湖孰|湖孰]]、[[句容|句容]]、[[曲阿|曲阿]]，程普皆有戰功，孙策增其二千兵，馬骑五十匹。後又进破[[乌程|乌程]]、[[石木|石木]]、[[波门|波门]]、[[陵传|陵传]]、[[馀亢|馀亢]]，程普立功最多。孫策入主[[会稽|会稽]]，任程普为吴郡都尉，治於钱唐。

===豪勇絕倫===
{{see also|曹操討伐袁術之戰|孫策襲劉勛皖城之戰|孫策攻黃祖沙羡之戰}}
後為丹杨都尉，居於[[石城|石城]]。再讨伐[[宣城|宣城]]、[[泾縣|泾縣]]、[[安吴|安吴]]、[[陵阳|陵阳]]、[[春谷|春谷]]亂贼。跟隨孫策進攻[[祖郎|祖郎]]，可是孫策反而被围，程普与一骑持矛殺入重圍，策马疾呼，敵軍大敗，孫策解圍而出<ref>《[[江表傳|江表傳]]》：「策旣平定江東，逐袁胤。袁術深怨策，乃陰遣間使齎印綬與丹楊宗帥陵陽祖郎等，使激動山越，大合衆，圖共攻策。」</ref>。後拜為[[荡寇中郎将|荡寇中郎将]]，领[[零陵郡|零陵郡]][[太守|太守]]，又隨軍讨[[刘勋_(东汉)|刘勋]]於[[寻阳|寻阳]]，进攻[[黄祖|黄祖]]於[[沙羡|沙羡]]，最後还镇石城。

===久經持重===
{{see also|夏口之戰|江夏之戰|赤壁之戰||江陵之戰 (208年)|夷陵之戰 (208年)}}
200年，孫策被刺殺身亡，[[孙权|孙权]]登位，程普巡視三郡，讨平不服之人。後隨孙权征伐[[江夏|江夏]]，到達[[豫章|豫章]]，另行率軍進攻[[乐安|乐安]]。平定後，轉為代[[太史慈|太史慈]]守备於[[海昏縣|海昏]]，後与[[周瑜|周瑜]]同为左右都督，於208年大破[[曹操|曹操]]於[[烏林|烏林]]（''詳見[[赤壁之戰|赤壁之戰]]''），乘勝进攻[[南郡|南郡]]，經過多月戰爭，打敗[[曹仁|曹仁]]。拜為[[裨将军|裨将军]]，领[[江夏郡|江夏郡]][[太守|太守]]，治於[[沙羡|沙羡]]，食邑四县。

210年，周瑜逝世，代领[[南郡|南郡]][[太守|太守]]。孫权與[[劉備|劉備]]結盟，將南郡分給劉備，程普还领江夏，升為[[荡寇将军|荡寇将军]]，不久於同年逝世<ref>《三國志·周瑜傳》載“（瑜）時年三十六。（孫）權素服舉哀。感動左右。喪當還吳，又迎之蕪湖，眾事費度，一為供給。後著令曰：‘故將軍周瑜、程普，其有人客，皆不得問。’”</ref>。《吳書》記載曰：程普斬殺叛亂者，并親自把斬殺死去的人投在火中，此後得病，數百日後死去。

按《三国志·[[步骘|步骘]]传》，在孙权称尊号后都督[[西陵|西陵]]的步骘曾向太子[[孙登_(太子)|孙登]]言及十一个就职于荆州的人，其中有程普。[[陈景云|陈景云]]认为此时程普去世已久，而同样就职荆州的[[吕岱|吕岱]]却不在此十一人之中，故此处“程普”或为“吕岱”之误。

== 特徵 ==
程普有容貌、计略，善於应对。愛好施与，喜歡與士大夫交往。曾因輩份關係而出言羞辱[[周瑜|周瑜]]，不過最後都因周瑜心胸開闊，而受到感動。而因程普在[[孫堅|孫堅]]帳下最為年長，時人皆稱「程公」。

==艺术形象==
===三國演義===
在小說《[[三國演義|三國演義]]》中，[[董卓討伐戰|董卓討伐戰]]中登場，與[[祖茂_(三国)|祖茂]]、[[韓當|韓當]]、[[黃蓋|黃蓋]]時為孫堅最早的部下，描寫為孫堅的參謀。使用'''鐵脊蛇矛'''作為武器，孫堅與[[華雄|華雄]]作戰時，斬殺華雄副將[[胡軫|胡軫]]。孫堅發現玉璽後，提議孫堅回家開拓的計策。當孫堅和[[袁紹|袁紹]]對立時，與黃蓋、韓當一起大戰袁紹手下大將[[顏良|顏良]]，[[文丑|文丑]]。孫堅原屬[[袁術|袁術]]麾下，孫堅死後韓當、黃蓋、程普歸附袁術；三人聽到孫策舉兵消息，并得到[[呂範|呂範]]的勸說，與黃蓋、韓當投歸到孫策軍中。程普在爾后與[[劉繇|劉繇]]部下[[太史慈|太史慈]]單挑而不分上下，孫權時代討伐[[黃祖|黃祖]]，程普報孫堅之仇，使用弓箭瞄準黃祖的脖子射殺，[[甘寧|甘寧]]斬去黃祖首級，作為成功征討的戰功。周瑜時期為東吳的副都督，與周瑜發生不和，對孫家有歷久的貢獻。

== 評論 ==
*[[孫權|孫權]]：“故將軍周瑜、程普，其有人客，皆不得問。”
*[[陳壽|陳壽]]《[[三國志|三國志]]》：“先出諸將，普最年長，時人皆呼程公。性好施與，喜士大夫。”“凡此诸将，皆江表之虎臣，孙氏之所厚待也。”
*[[呂蒙|呂蒙]]：“昔周瑜、程普為左右部督，共攻江陵，雖事決於瑜，普自恃久將，且俱是督，遂共不睦，幾敗國事。”
*[[陆机|陆机]]：“甘宁、[[凌统|凌统]]、程普、[[贺齐|贺齐]]、[[朱桓|朱桓]]、[[朱然|朱然]]之徒奋其威。”
*[[章如愚|章如愚]]：“如程普、黄盖、甘宁、徐盛、潘璋、朱然、朱桓、贺齐、凌统、全琮、吕范，皆智足以御众，勇足以却敌，未有不为守令之职者。”
*[[程公許|程公許]]：“蜀将如关、张、[[庞统|庞统]]，吴将如[[周瑜|周瑜]]、[[鲁肃|鲁肃]]，志长命短，天下重惜之。而[[马超|马超]]、[[黄忠|黄忠]]、[[赵云|赵云]]、[[费祎|费祎]]、[[吕蒙|吕蒙]]、程普、[[步骘|步骘]]、[[甘宁|甘宁]]辈皆智勇绝伦，足以当一面。”
*[[郝经|郝经]]：“程普诸将皆江表虎臣，鏖兵卫主，攻坚轧敌，兴王定覇，孙氏兄弟卒立国建号，诸将之力也。”“吴将剽轻，殆多谲计。莫肯下人，卒自称帝。摩创抚孤，动辄流涕。驾驭有术，驱策有方。果保江东，不负桓王。”
