'''程家桥街道'''，是{{PRC admin/fullname|31/01/05/010/000|3}}下辖的一个[[街道办事处|街道办事处]]<ref>{{PRC admin/ref sgc|{{PRC admin/parent|31/01/05/010/000}}}}</ref>，因长宁区境内原有河流王满泗泾上南北的向石桥“程家桥”而得名<ref name="shxz">{{cite web|title=上海县志 >> 第一篇建置 >> （六）集镇、新村 >> 程家桥镇|url=http://www.shtong.gov.cn/newsite/node2/node4/node2250/shanghai/node53607/node53624/node53626/userobject1ai35471.html|website=上海通网站|publisher=上海市地方志办公室|accessdate=2017-08-06}}</ref>。面积7.6平方公里，户籍人口1.98万人（2008年）。
== 历史沿革 ==
程家桥集镇原本位于于今虹桥路、哈密路口南侧的新泾港、王满泗泾两岸。明末清初，此地逐渐成为村落；清朝晚期，居民逐渐增加，程家桥地区便形成了集镇<ref name="cnqz" />。

清光绪二十六年（1900年），英国一侨民在程家桥建立老裕泰马房，民国3年(1914年)，老裕泰马房主人病死，马房为怡和洋行、太古洋行、汇丰银行等8家大班购下，改建成高尔夫球场，名虹桥抛球总会球场<ref name="zoo">{{cite web|title=认识我们-我们的历史|url=http://www.shanghaizoo.cn/sites/shanghaizoo/Us.aspx?ctgid=74EA9A66-B625-4351-95EE-5AB40AED137C&id=1E4349A8-D01E-4C19-B1B6-8822C1BBA5E1|website=上海动物园}}</ref>。

光绪二十七年（1901年），[[工部局|工部局]]越界辟筑[[虹桥路|虹桥路]]和罗别根路（即[[哈密路|哈密路]]）。随后，外国人在虹桥路沿线建了一系列花园、别墅，其中[[沙逊别墅|沙逊别墅]]、[[朱家花园|朱家花园]]、[[孔家花园|孔家花园]]、[[罗别根花园|罗别根花园]]、[[霍更斯别墅|霍更斯别墅]]（空管局退休干部活动中心）等具有美国、英国、意大利等国建筑风格的花园别墅座落在程家桥区域附近<ref name="cnqz" />。

民国时期，当地被划入[[蒲淞区|蒲淞区]]。民国31年（1942年），日军“清乡”期间，小贩偷越“请乡”封锁线，从青浦、松江贩米至此出售；王满泗桥、老程家桥商店二十余家陆续迁此，渐成小镇，1946年有商店36家<ref name="shxz" />。

1949年4月，程家桥划属[[上海特别市|上海市]][[新泾区|新泾区]]管辖<ref name="xzqh">{{cite web|title=上海县志 >> 第一篇建置 >> （三）行政区划 >>  上海市新泾区、龙华区、西郊区行政区划|url=http://www.shtong.gov.cn/newsite/node2/node4/node2250/shanghai/node53607/node53615/node53617/userobject1ai35421.html|website=上海通网站|publisher=上海市地方志办公室|ref=xzqh}}</ref>。1949年5～7月，[[上海市军事管制委员会|上海市军事管制委员会]]新泾区接管委员会先后设5个办事处，下辖6镇，其中便有程家桥镇，后程家桥镇归入新泾乡；1951年5月，又复设程桥乡。

1954月5月21日，为配合[[上海动物园|西郊公园]]开放<ref>{{Cite web|url=http://www.shtong.gov.cn/Newsite/node2/node4/node2250/shanghai/node53953/node53967/node53969/userobject1ai36110.html|title=上海县志->--第二十一篇交通运输->--五公路客运->--解放后公交客运|accessdate=2017-07-08|author=|date=|publisher=上海市地方志办公室}}</ref>，由市区通往程家桥的[[上海公交57路|公共汽车线路57路]]开通，线路由静安寺起，经南京西路、延安西路、虹桥路到西郊公园<ref name="jtyd">{{cite web|url=http://www.shtong.gov.cn/newsite/node2/node4/node2249/changning/node15386/node42474/node63012/userobject1ai28063.html|title=长宁区志 >> 第十三编交通邮电 >> 第一章道路客运 >> 第二节 公交车|accessdate=2017-07-07|publisher=上海市地方志办公室 上海通网站|website=上海通网站}}</ref>。同年5月25日，[[西郊公园|西郊公园]]改建工程完工，开放参观；同年8月，市政府决定将西郊公园扩建为[[上海动物园|上海动物园]]<ref name="zoo" />。一时，游客大增，市面益振。时有商店44家<ref name="cnqz" />。

1956年，程家桥再度被划入北新泾镇；同年3月，程家桥地区随着新泾区撤销而并入[[西郊区|西郊区]]<ref name="xzqh" />。1958年8月西郊区撤销，同年11月份，程家桥随北新泾镇再划入[[上海县|上海县]]<ref name="shxz-jjbh">{{cite web|title=上海县志 >> 第一篇建置 >> （二）县域 >>  境域变化|url=http://www.shtong.gov.cn/newsite/node2/node4/node2250/shanghai/node53607/node53612/node53614/userobject1ai50766.html|website=上海通网站|accessdate=2017-07-31}}</ref>。

[[File:HuazhuHotelsHQ.jpg|thumb]]和[[华住酒店集团|华住酒店集团]]总部]]
1981年北新泾镇、新泾公社、虹桥公社部分地共0.45平方公里划属长宁区<ref name="shxz-jjbh" />。1982年全镇道路铺设下水道，铺筑沥青路面，沿新泾港砌石驳岸4.10公里；淮阴路、[[虹井路|虹井路]]、[[虹梅路|虹梅路]]及[[程家桥路|程家桥路]]均翻修成沥青路。1985年，在上海动物园门前建造上海最大的人行地道<ref name="cnqz" />。1986年，程家桥始设[[街道办事处|街道办事处]]。1987年虹桥路拓宽改建，全镇房屋全部拆除<ref name="cnqz" />。于此同时，1981年11月至1992年底，新泾供销社、农口、轻工、虹桥国际机场等系统、单位在程家桥建设了一批公房小区，形成程桥一村、二村、虹桥机场新村和上航新村等4个住宅小区。另建有绿谷别墅、松园别墅及皇朝、明鸿等别墅群。1984年起，程家桥地区从有300余户的生活区发展成为有7537户居民的生活区<ref name="cnqz" />。

1996年12月2日，辖区南部的[[延安高架路|延安高架路]]西段建成通车<ref>{{cite web|title=上海年鉴1997->--十四、城市建设、重大工程->（二）市政基础设施建设|url=http://www.shtong.gov.cn/Newsite/node2/node19828/node19886/node24125/node64389/userobject1ai62566.html|website=上海通网站|publisher=上海市地方志办公室|accessdate=2017-08-06}}</ref>。2010年4月10日，[[上海地铁10号线|上海地铁10号线]][[龙溪路站|龙溪路站]]以东及支线先期开通试运营，主线龙溪路站以西于2010年11月30日开通。10号线在境内设[[龙溪路站|龙溪路站]]、[[上海动物园|上海动物园]]、[[虹桥1号航站楼|虹桥1号航站楼]]等三个站点<ref>{{cite web|title=上海轨道交通路网总里程突破400公里 10号线4月10日提前开通试运营|url=http://www.shmetro.com/node49/201004/con103539.htm|website=上海地铁|publisher=上海地铁|accessdate=2017-08-06|date=2010-04-08}}</ref>。

== 现行行政区划 ==
目前，街道辖区东起俞家浜，折西郊宾馆东围墙、虹桥路、虹梅路；西至虹桥国际机场（扩建前）{{NoteTag|name="hqjc"|虹桥机场扩建为[[虹桥综合交通枢纽|虹桥综合交通枢纽]]后，长宁、闵行两区界限未曾因之改变<ref>{{cite book|author1=上海市测绘院|title=上海分区地图——闵行区地图|publisher=中华地图学社|isbn=978-7-80031-838-2|edition=2017年1月第五版}}</ref>。}}西围墙；南沿虹桥国际机场（扩建前）{{NoteTag|name="hqjc"}}南围墙，折沪杭铁路徐虹支线；北临虹桥国际机场北、东围墙，折夏家浜、新泾港、淮阴路、西郊宾馆北围墙<ref name="cnqz">{{cite web|title=长宁区志 >> 第二编街道、镇、乡 >> 第二章街道、镇、乡分述 >> 第十节 程家桥街道|url=http://www.shtong.gov.cn/newsite/node2/node4/node2249/changning/node15187/node42031/node62980/userobject1ai27624.html|website=上海通网站|publisher=上海市地方志办公室|accessdate=2017-08-06}}</ref>。
