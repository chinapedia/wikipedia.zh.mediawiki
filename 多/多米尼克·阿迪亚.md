'''多米尼克·阿迪亚'''（{{lang-en|'''Dominic Adiyiah'''}}，{{bd|1989年|11月29日|catIdx=Adiyiah, Dominic}}），是一名[[加纳|加纳]][[足球|足球]][[运动员|运动员]]，司職[[前鋒_(足球)|前鋒]]、[[翼鋒|翼鋒]]，現時效力于乌克兰球会[[基辅阿森纳足球俱乐部|基辅阿森纳]]。

== 球會生涯 ==
艾迪耶亞在{{tsl|en|Feyenoord Academy|費耶諾德學院}}開始他的職業生涯，於2006年升上一隊。不久便轉投[[獅心足球俱乐部|獅心]]，在那裡他在聯賽合共上陣24次取得11個入球，並入選加納聯賽全明星隊，他更成為2006-07年聯賽[[最有價值球員|最有價值球員]]。2008年8月6日，艾迪耶亞轉會至[[挪威足球超级联赛|挪威足球超级联赛]]球隊[[費德列斯達足球會|費德列斯達]]。8月30日，在對陣[[阿里辛特足球會|阿里辛特]]的賽事中獲得上陣機會。<ref>{{cite web |url=http://www.transfermarkt.de/de/spielbericht/918273/aalesunds-fk_fredrikstad-fk/spielverlauf/index.html |title=存档副本 |accessdate=2009-11-11 |deadurl=yes |archiveurl=https://web.archive.org/web/20091104061359/http://www.transfermarkt.de/de/spielbericht/918273/aalesunds-fk_fredrikstad-fk/spielverlauf/index.html |archivedate=2009-11-04 }}</ref>

在[[2009年世界青年足球錦標賽|2009年世界青年足球錦標賽]]中艾迪耶亞表現出色，吸引了[[意甲|意甲]]勁旅[[AC米蘭|AC米蘭]]注意。<ref>http://www.ontheminute.com/news/news.php?news=18604</ref>2009年10月，報導指艾迪耶亞將於冬季轉會期加盟[[AC米蘭|AC米蘭]]。2009年11月1日，[[AC米蘭|AC米蘭]]總經理[[阿德里亚诺·加利亚尼|加利亞尼]]證實了這消息。<ref>{{cite web |url=http://www.acmilan.com/NewsDetail.aspx?idNews=93141 |title=存档副本 |accessdate=2009-11-11 |deadurl=yes |archiveurl=https://web.archive.org/web/20091104042503/http://www.acmilan.com/NewsDetail.aspx?idNews=93141 |archivedate=2009-11-04 }}</ref><ref>http://soccernet.espn.go.com/news/story?id=692382&sec=transfers&cc=4716</ref>只等待冬季轉會期開放及通過體檢後便會正式加盟。<ref>{{cite web |url=http://www.fredrikstadfk.no/content/view/2540/372/ |title=存档副本 |accessdate=2009-11-11 |deadurl=yes |archiveurl=https://web.archive.org/web/20091106235722/http://www.fredrikstadfk.no/content/view/2540/372/ |archivedate=2009-11-06 }}</ref>
但2011年居然沦落到土乙卡西亚卡踢球，2012年夏天米兰将其出售给乌克兰首都球会[[基辅阿森纳足球俱乐部|基辅阿森纳]]。

== 國家隊 ==
自2008年開始，艾迪耶亞已入選{{tsl|en|Ghana national under-20 football team||加納U-20}}<ref>http://www.modernghana.com/news/193787/1/none.html</ref>。他的第一場比賽是於08年3月30日對[[尼日爾國家足球隊|尼日爾U-20]]。<ref>{{cite web |url=http://www.ghanafa.org/blackstars/200803/2782.asp |title=存档副本 |accessdate=2009年6月18日 |deadurl=yes |archiveurl=https://web.archive.org/web/20090604082053/http://www.ghanafa.org/blackstars/200803/2782.asp |archivedate=2009年6月4日 |df= }}</ref>艾迪耶亞入選{{tsl|en|Ghana national under-20 football team||加納U-20}}參加在埃及舉行的[[2009年世界青年足球錦標賽|2009年世界青年足球錦標賽]]。<ref>http://www.kicker.de/news/fussball/junioren/startseite/515539/artikel_Adiyiah-wird-zum-Helden.html</ref>他出色的表現協助球隊晉身決賽，2009年10月16日決賽對陣[[巴西國家足球隊|巴西U-20]]，雙方在法定時間打成0-0，要以互射[[十二碼|十二碼]]分勝負，最終{{tsl|en|Ghana national under-20 football team||加納U-20}}以4-3擊敗[[巴西國家足球隊|巴西U-20]]首次奪得[[世界青年足球錦標賽|世界青年足球錦標賽]]冠軍。<ref>http://de.fifa.com/u20worldcup/matches/round=253219/match=300090123/summary.html</ref>艾迪耶亞以8個入球羸得[[金靴獎|金靴獎]]<ref>http://de.fifa.com/u20worldcup/news/newsid=1119558.html#adiyiah+ghanas+goldjunge</ref>，其出色的表現也羸得賽事的[[金球獎|金球獎]]及[[最有價值球員|最有價值球員]]。<ref>{{cite web |url=http://www.transfermarkt.de/de/news/32096/aegypten-adiyiah-raeumt-ab.html |title=存档副本 |accessdate=2009-11-11 |deadurl=yes |archiveurl=https://web.archive.org/web/20091020015911/http://transfermarkt.de/de/news/32096/aegypten-adiyiah-raeumt-ab.html |archivedate=2009-10-20 }}</ref>

2009年11月3日，艾迪耶亞首次成為加納大國腳，入選了於11月15號友誼賽對[[馬里國家足球隊|馬里]]的名單。<ref>http://www.theghanaianjournal.com/2009/11/03/dede-ayew-adiyiah-and-others-join-black-stars/</ref>他也入選了[[2010年非洲國家盃|2010年非洲國家盃]]的名單，並協助球隊奪得亞軍。

[[2010年南非世界杯|2010年南非世界杯]]1/4决赛对乌拉圭的比赛里阿迪亚替补上阵，表现尚可，但是在点球大战里他罚丢了加纳队的最后一个点球，最终球队饮恨出局。
