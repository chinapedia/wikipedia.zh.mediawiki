'''多头政体'''（{{lang-en|Polyarchy}}），又译“'''多头政治'''”、“'''多元民主'''”、“'''多元政體'''”，指一种充分发展的[[代议制|代议制]][[政体|政体]]，可以视作[[现代|现代]][[代议民主|代议民主]]制的替代词，其最重要的特点是公开就某政治议题进行争论的[[政治自由|政治自由]]，以及[[选举|选举]]和被选举权为代表的[[政治参与|政治参与]]权。美国著名政治学家[[羅伯·道爾|罗伯特·达尔]]在其于1953年出版的《[[政治、经济与福利|政治、经济与福利]]》一书提出了“多元民主”这一政治学概念。1971年出版的《[[多头政体：政治参与和反对派|多头政体：政治参与和反对派]]》深刻阐述“多头政体”这一观点。达尔认为，政府决策的制定并不是就某些基本政策事宜统一起来的多数人的庄严进程，它是对相对少的群体的安抚。民主与专制的区别不在于是否是“[[多数人的统治|多数人的统治]]”，而在于是否是“[[多重少数人的统治|多重少数人的统治]]”。<ref>{{cite news|url=http://xuebao.ruc.edu.cn/CN/abstract/abstract8586.shtml|title=现代民主理论的兴起及其演进历程——从人民主权到表达民主再到协商民主|date=|publisher=中国人民大学学报}}</ref>

达尔非常强调[[利益集团|利益集团]]对于[[多元民主|多元民主]]的重要性。通过考察竞争性选举与有组织的利益集团的活动之间的关系，达尔改进了[[熊彼特|熊彼特]]的精英民主理论。达其看来，民主不是人民的主权，不是多数人的统治，也不是“单一”的少数人的统治，而是“多重”少数人的统治。<ref name="政治学系政治科学系2002">{{cite book|author1=中国人民政治大学. 政治学系|author2=中山大学 (Guangzhou, China). 政治科学系|title=转型中的中国政治与中国政治学发展国际研讨会论文汇编|url=https://books.google.com/books?id=nUYjAQAAMAAJ|year=2002|publisher=中国人民政治大学国际关系学院政治学系, 中山大学政治和公共事务管理学院政治科学系}}</ref>一个政体的民主性质是由多重的集团或多重的少数人来决定的。<ref name="贺文萍2005">{{cite book|author=贺文萍|title=非洲国家民主化进程研究|url=https://books.google.com/books?id=FoQ2AQAAIAAJ|year=2005|publisher=时事出版社|isbn=978-7-80009-926-7}}</ref>为了避免语义上的混乱，达尔主张将Democracy改为Polyarchy（多头政体）。<ref>{{cite news|url=http://theory.people.com.cn/GB/40557/145802/145822/8795058.html|title=精英民主与多元民主|date=2008年11月3日|publisher=人民网}}</ref>

==概要==
达尔认为，习用的“[[民主|民主]]”是一个不精确的[[术语|术语]]。在他看来，民主制最重要的特点是，“可以完全或近乎完全地对所有公民的要求做出反应”。达尔承认，这个定义过于理想化，以此衡量，当今世界没有一个政府可称得上“民主”。为了能够更准确清晰地分析现实政治，达尔认为需要定义一个新的词语来描述当今大多数西方发达[[资本主义|资本主义]]国家的政治体制。

一个能实现充分反映公民意愿的政府，需要在保证公民政治上的[[平等权|平等权]]外，还需要保障公民充分的[[政治参与|政治参与]]机会：
# 系统表述自身意愿的机会；
# 通过个人或集体行动向其他公民及政府公开表明自身意愿的机会；
# 公民的意见必须得到政府在施政过程中的同等重视，也就是说，不因意见的来源或内容而忽视或无视意见本身。
以上只是判断一个政府是否民主的[[必要条件|必要条件]]而非[[充分条件|充分条件]]。在人口众多的现代国家，民主政府还要至少有八项制度保证。

根据公开争论和选举参与两条标准，可以将政府大致划归为四类。
# 封闭的专制政体。该政体禁止人民参与政治议题的讨论（缺乏政治自由），缺乏公开公正的选举制，不保证公民选举权（缺乏政治参与）。
# 包含的专制政体。该政体在一定程度上保证公民的选举权，但缺乏政治自由。
# 竞争的寡头政体。该政体在一定程度上保证公民的政治自由，但缺乏完善的选举制度（缺乏政治参与）。
# 多头政体。该政体保障了政治自由，实现了公民对政治的广泛参与。
