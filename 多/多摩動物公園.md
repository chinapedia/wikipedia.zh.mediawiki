'''多摩動物公園'''（{{jpn|j=多摩動物公園|hg=たまどうぶつこうえん}}）是[[東京都|東京都]][[日野市|日野市]]的[[動物園|動物園]]，也稱'''多摩動物園'''，是與[[恩賜上野動物園|恩賜上野動物園]]並列的日本知名動物園。作為都市公園的正式名稱為'''都立七生公園'''。

==概要==
1958年5月5日開園。最初是作為恩賜上野動物園的分園，園區比恩賜上野動物園大上約4倍。

由於位於[[丘陵地|丘陵地]]，地勢起伏多，加上展示動物區之間相隔較遠，園內設有接駁巴士供遊客搭乘。

過去本園由東京都直營，在導入{{link-ja|指定管理者制度|指定管理者制度}}後，2006年4月1日起與恩賜上野動物園、{{link-ja|葛西臨海水族園|葛西臨海水族園}}、[[井之頭自然文化園|井之頭自然文化園]]一同移交給[[公益財團法人|公益財團法人]]{{link-ja|東京動物園協會|東京動物園協会}}營運。

==園內組成==
===亞洲園===
; [[紅毛猩猩|紅毛猩猩]]天空步道（オランウータンのスカイウォーク）
: 由鐵塔與鋼索組成，全長150公尺。
; 猛禽舍
: 半圓形巨籠中飼養著[[白尾海雕|白尾海雕]]與[[金鵰|金鵰]]。
; 亞洲山岳區（アジアの山岳ゾーン）
: 展示[[雪豹|雪豹]]、[[小貓熊|小貓熊]]、[[羚牛|羚牛]]。
; [[馬來貘|馬來貘]]水邊（マレーバクの水辺）
: 透過玻璃觀察馬來貘在水中活動的情形。
; 鼴鼠之家（モグラの家）
: 透過玻璃觀察[[鼴鼠|鼴鼠]]在土中的巢穴。也展示日本最小的哺乳類{{link-ja|東京尖鼠|トウキョウトガリネズミ}}。
; 亞洲沼地（アジアの沼地）
: 重現[[亞洲小爪水獺|亞洲小爪水獺]]、[[印度犀牛|印度犀牛]]、[[水牛|水牛]]、水鳥的棲息地。
; 亞洲平原（アジアの平原）
: 展示[[狼|狼]]、[[蒙古野馬|蒙古野馬]]。

===非洲園===
[[File:多摩動物公園ライオンバス_(7917055360).jpg|thumb]]
; 獅子巴士（ライオンバス）
: 獅子巴士可近距離觀察獅子，是日本國內最早引進此服務的動物園。巴士由[[京王電鐵巴士|京王電鐵巴士]]運行。
; 大草原（サバンナ）
: [[長頸鹿|網紋長頸鹿]]、[[細紋斑馬|細紋斑馬]]、[[彎角劍羚|彎角劍羚]]、[[鴕鳥|鴕鳥]]、[[鵜鶘|鵜鶘]]等。
; [[黑猩猩|黑猩猩]]之森（チンパンジーの森）
: 高15公尺的塔與各式設施，可觀察黑猩猩的運動能力與智力。

===澳洲園===
; 無尾熊館（コアラ館）
: 展示[[無尾熊|無尾熊]]為、[[蜜袋鼯|蜜袋鼯]]與環尾袋貂等有袋類。

===昆蟲園===
[[File:Great_Orange_Tip_(Hebomia_glaucippe).jpg|thumb]]
; 昆蟲園本館
: 模仿[[蜻蜓|蜻蜓]]形象打造的本館建築，展示昆蟲與標本。
; 昆蟲生態園
: 進行溫度調節，重現河川與森林等環境的展示空間，飼養[[蝴蝶|蝴蝶]]、負蝽、[[獨角仙|獨角仙]]等昆蟲。
: 以蝴蝶為造型的場館在1989年入選{{link-ja|日本建築學會賞|日本建築学会賞}}。

==歷史==
*1958年 - 開園。代表動物有[[黑猩猩|黑猩猩]]、印度象、[[印度犀牛|印度犀牛]]等。
*1960年 - [[長頸鹿|網紋長頸鹿]]來園。開始建設非洲園。日本首次成功繁殖[[雙峰駱駝|雙峰駱駝]]
*1962年 - 日本首次成功繁殖[[羱羊|羱羊]]、豚鹿。
*1964年 - 獅子巴士啟動。
*1965年 - 日本首次成功繁殖[[犛牛|犛牛]]。
*1966年 - 「草蜢與蝴蝶溫室（昆蟲生態園前身）」完成。日本首次成功繁殖[[馬來熊|馬來熊]]。
*1967年 - [[非洲象|非洲象]]、[[獵豹|獵豹]]、[[彎角劍羚|彎角劍羚]]來園。日本首次成功繁殖[[馴鹿|馴鹿]]。
*1968年 - 日本首次成功繁殖[[斑馬|查普曼斑馬]]、白羚羊。
*1969年 - 昆蟲園本館開館。日本首次成功繁殖夜猴、[[狷羚|狷羚]]等。
*1970年 - 日本首次成功繁殖[[馬來貘|馬來貘]]、圓盾大袋鼠、[[雕鴞|雕鴞]]。
*1965年 - 日本首次成功繁殖{{link-en|歐洲盤羊|Mouflon}}。
*1973年 - 日本首次成功繁殖印度象。
*1976年 - 日本首次成功繁殖[[白尾海雕|白尾海雕]]。
*1977年 - 大草原放飼場完成。
*1978年 - 日本首次成功繁殖[[麋鹿|麋鹿]]、白頭鳶、[[美洲紅䴉|美洲紅䴉]]等。
*1980年 - 日本首次成功人工繁殖[[丹頂鶴|丹頂鶴]]。
*1982年 - 日本首次成功繁殖[[兔屬|北海道雪兔]]。
*1984年 - 澳洲[[塔龍加動物園|塔龍加動物園]]無尾熊來園。無尾熊館完成。
*1985年 - 猛禽舍完成。日本首次成功繁殖 長鼻袋鼠、短鼻袋狸等。
*1987年 - 世界首次成功繁殖[[日本睡鼠|日本睡鼠]]。
*1988年 - 昆蟲生態園完成。日本首次成功繁殖[[東方白鸛|東方白鸛]]。
*1990年 - 獵豹、大草原舍完成。
*1991年 - 亞洲山岳區完成。日本首次成功繁殖[[蜂鳥|蜂鳥]]。
*1993年 - 觀察中心（ウォッチングセンター）完成。
*1996年 - 非洲象舍改建。世界首次成功繁殖[[黑面琵鷺|黑面琵鷺]]。日本首次成功繁殖羚牛、[[紅頭美洲鷲|紅頭美洲鷲]]、[[白鶴|白鶴]]。
*1997年 - 世界最高齡的水牛「ミタケ」死亡（36歲）。
*1998年 - 非洲象繁殖成功。
*2000年 - 黑猩猩之森、馬來貘水邊完成。
*2002年 - 昆蟲館本館改建。
*2005年 - 紅毛猩猩天空步道完成。
*2007年 - 佐渡朱䴉保護中心的朱䴉來園。
*2008年 - 朱䴉首度在佐渡以外的設施繁殖。亞洲沼地區完成。
*2011年 - 3月17日至3月31日因東日本大地震影響暫時休園。
*2013年 - 亞洲平原區完成。
