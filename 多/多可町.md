'''多可町'''（{{jpn|j=多可町|hg=たかちょう|rm=Taka chō|f=y}}）是位於日本兵庫縣中部的行政區劃。轄區全境位於[[中國山地|中國山地]]內，79.%的區域為山林地，杉原川自北往南貫穿轄區。<ref name="Brief">{{Cite web| url=http://www.takacho.jp/town_guide/machi_gaiyou.html| title=多可町的概要| publisher=多可町公所| language=ja| accessdate=2017-01-19| archive-url=https://web.archive.org/web/20170131191024/http://www.takacho.jp/town_guide/machi_gaiyou.html| archive-date=2017-01-31| dead-url=yes}}</ref>

== 地理 ==
* 山：{{tsl|ja|千ヶ峰|千峰}}（1005m）、{{tsl|ja|笠形山}}（939m）、{{tsl|ja|妙見山}}（693m）
* 河川：{{tsl|ja|杉原川 (兵庫県)|杉原川 (兵庫縣)|杉原川}}、野間川 （{{tsl|ja|加古川}}支流）
* 湖沼：糀屋水庫（別名：{{tsl|ja|翠明湖}}）

== 歷史 ==
1879年實施郡區町村編製法時，多可郡的郡役所曾設在現在町內的中村地區。1889年日本實施[[町村制|町村制]]時，現在的轄區在當時分屬中村、{{tsl|ja|松井庄村|松井庄村|松井-{庄}-村}}、{{tsl|ja|杉原谷村}}、{{tsl|ja|野間谷村}}、{{Tsl|ja|大和村 (兵庫県)|大和村 (兵庫縣)|大和村}}，共5村；經過1950年代的[[市町村合併|昭和大合併]]後，被整併為{{tsl|ja|中町 (兵庫県)|中町 (兵庫縣)|中町}}、{{tsl|ja|加美町 (兵庫県)|加美町 (兵庫縣)|加美町}}、{{tsl|ja|八千代町 (兵庫県)|八千代町 (兵庫縣)|八千代町}}，3町。2005年11月1日，這三個町再次被整併為現在的'''多可町'''。<ref name="Brief"/>

== 交通 ==
過去曾經有[[西日本旅客鐵道|西日本旅客鐵道]][[鍛冶屋線|鍛冶屋線]]從[[西脇市|西脇市]]野村車站（現已更名為{{tsl|ja|西脇市駅|西脇市車站}}）連結至町內的中區；但此路線已於1990年停止營運。目前轄區內無鐵路路線通過。
