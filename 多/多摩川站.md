'''多摩川站'''（{{jpn|j=多摩川駅|hg=たまがわえき|rm=Tamagawa eki|f=y}}）是一個位於[[東京都|東京都]][[大田區|大田區]][[田園調布|田園調布]]一丁目，屬於[[東京急行電鐵|東京急行電鐵]]的[[鐵路車站|鐵路車站]]<ref name="eki">{{Cite web |url=http://www.tokyu.co.jp/railway/station/info/?id=9 |title=多摩川駅｜各駅情報 |publisher=東京急行電鉄 |accessdate=2015-01-13}}</ref>。

== 停靠路線 ==
在軌道名稱上，此站只有[[東橫線|東橫線]]與[[東急多摩川線|東急多摩川線]]2條路線停靠。但是東橫線的[[田園調布站|田園調布站]]－[[日吉站_(神奈川縣)|日吉站]]之間的複複線是由東橫線列車與[[目黑線|目黑線]]列車使用，在旅客資訊上視為兩個不同的路線停靠。因此，在旅客資訊上此站是東橫線、目黑線、東急多摩川線3條路線的轉乘站。3條路線各有車站編號，東橫線是'''TY09'''、目黑線是'''MG09'''、東急多摩川線是'''TM01'''<ref name="eki"/>。

== 歷史 ==
* 1923年（[[大正|大正]]12年）3月11日 - 目黑蒲田電鐵目黑 - 丸子站（現[[沼部站|沼部]]）區間通車，'''多摩川站'''啟用<ref name="jtb98">[[#jtb|東急の駅]]、p.98。</ref>。
* 1926年（大正15年）
** 1月1日 - 車站往目黑方向移動，站名改稱'''丸子多摩川站'''<ref>[{{NDLDC|2956145/5}} 「地方鉄道駅名改称」『官報』1925年12月16日]（国立国会図書館デジタル化資料）</ref><ref name="jtb98"/><ref name="jtb97">[[#jtb|東急の駅]]、p.97。</ref>，月台為3面4線配置<ref name="jtb52">[[#jtb|東急の駅]]、p.52。</ref>。
** 2月14日 - 東橫線通車<ref name="jtb97"/><ref name="jtb52"/>。
* 1931年（[[昭和|昭和]]6年）1月1日 - 以附近的「多摩川園遊園地」更名為'''多摩川園前站'''<ref name="jtb97"/><ref name="jtb52"/>。
* 1935年（昭和10年）7月 - 廢除站內平交道，新建地下通道<ref name="jtb97"/>。
* 1977年（昭和52年）12月16日 - 站名改稱'''多摩川園站'''<ref name="jtb97"/><ref name="jtb52"/>。之後多摩川園遊園地在1979年（昭和54年）6月閉園し<ref name="jtb98"/>。
* 1988年（昭和63年）11月 - 田園調布 - 多摩川間改良工程動工<ref name="kairyo">{{Cite press release |url=http://www.tokyu.co.jp/file/000207.pdf |title=営団地下鉄南北線・都営三田線と相互直通運転を開始 |publisher=東京急行電鉄 |date=2000-02-07 |accessdate=2015-01-14 |format=pdf}}</ref>。
* 1997年（[[平成|平成]]6年）6月 - 改良工程完工<ref name="kairyo"/>。
* 2000年（平成12年）8月6日 - 目蒲線分割為目黑線、東急多摩川線，站名改為'''多摩川站'''<ref name="jtb97"/><ref name="jtb52"/><ref name="henko">{{Cite press release |url=http://www.tokyu.co.jp/file/000328.pdf |title=目蒲線の運行系統変更による線名変更などを実施 |publisher=東京急行電鉄 |date=2000-03-28 |accessdate=2015-01-14 |format=pdf}}</ref>。並成為東橫線急行停靠站。

=== 站房改良 ===
1988年11月至1997年6月，因東橫線複複線化事業的一環，進行田園調布 - 多摩川間改良工程<ref name="kairyo"/>，本站站房進行大規模改造。

東橫線與[[目蒲線|目蒲線]]是兩條不同線路，其中田園調布與多摩川（當時名為多摩川園）之間是呈現[[複複線|複複線]]的形式，且兩站皆為3面4線形態。改良工程後，兩站月台改為2面4線構造。目蒲線從本站往[[沼部站|沼部]]方向至多摩川第一平交道之間是平緩的下坡。東橫線月台往橫濱方向是連結{{link-ja|多摩川橋梁 (東急東橫線)|多摩川橋梁 (東急東横線)|多摩川橋梁}}的急轉彎，車輛在此須減速慢行。另外，上行線最後一輛車廂會在傾斜狀態下停靠月台，對於乘客上下車也十分危險。

隨著複複線化事業進行，目蒲線從本站分割，目黑方向改為與東橫線採取同方向同月台設計，月台位置往田園調布移動50公尺，改建為高架月台2面4線構造，消除過去的急轉彎月台，蒲田方向則新建1面2線的地下月台。目蒲線的地下線路替換工程在一夜完成。

靠近多摩川橋梁的{{link-ja|東京都道11號大田調布線|東京都道11号大田調布線}}（多摩堤通）鐵路橋高度僅有3.3公尺，也在本次工程中進行改良。

此改良工程最初未規劃在田園調布、多摩川兩站設置電梯，但在地方居民的請願下增設，開啟了日本鐵道站的無障礙化浪潮。

由於多摩川橋梁改建、增設工程（1997年12月完成）與新丸子⇔武藏小杉間增線工程（2000年9月完成）先行完成，在2000年8月6日目黑線通車以前，目蒲線暫時使用新的地下站，可直通田園調布方向。該線路在之後作為連絡線使用。

==車站結構==
本站為高架的2面4線島式月台與地下1面2線組成的車站<ref name="map">{{Cite web|url=http://www.tokyu.co.jp/railway/station/yardmap/?id=9|title=多摩川駅｜駅構内図|publisher=東京急行電鉄|accessdate=2015-01-13|deadurl=yes|archiveurl=https://web.archive.org/web/20140429121458/http://www.tokyu.co.jp/railway/station/yardmap/?id=9|archivedate=2014-04-29}}</ref>。高架的部份，外面2線是東橫線，內側2線是目黑線，地下2線則為東急的多摩川線。目黑線月台已設置[[月台門|月台門]]。

廁所在付費區內<ref name="map"/>。電梯設於月台的橫濱、蒲田側，連通高架月台與地下月台<ref name="map"/>。

站房在高架下，設有一處剪票口，出入口分為東口、西口、南口<ref name="map"/>。

東橫線、目黑線月台在彎道上，月台間隙較大。

=== 巴士路線 ===
[[File:TokyuBus-TamagawaSta.jpg|thumb]]
站前道路狹窄，因此選擇在南口前方多摩堤通交差點附近的舊東急游泳學校停車場前設置廣場，站名為「'''多摩川園'''」，在車站更名後也改為「'''多摩川站'''」。本站可搭乘[[東急巴士|東急巴士]]運行的路線巴士<ref name="bus">{{Cite web |url=http://www.tokyubus.co.jp/jikoku/dia/cgi/search_route.cgi?stcode=1725816 |title=東急バス時刻表（多摩川駅） |publisher=東急バス株式会社 |accessdate=2015-01-14}}</ref>。

* 多摩01 - 經[[都立大學站|都立大學站]]前往{{link-ja|國立醫院機構東京醫療中心|国立病院機構東京医療センター|東京醫療中心}}<ref name="bus"/>
