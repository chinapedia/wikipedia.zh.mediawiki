'''多氯聯苯'''（{{lang-en|'''polychlorinated biphenyl'''}}，簡稱'''PCB'''，[[CAS號|CAS號]]{{CAS|1336-36-3}}），又稱'''多氯聯二苯'''或'''二聯酚'''，是許多含氯數不同的[[聯苯|聯苯]]含氯化合物的統稱<ref>{{harvnb|陳怡儒|2011|pp=191-193}}</ref><ref name="周琦淳2013-p108">{{harvnb|周琦淳|2013|pp=110-111}}</ref>。在多氯聯苯中，部份[[苯環|苯環]]上的[[氫原子|氫原子]]被[[氯原子|氯原子]]置換，一般式為C<sub>12</sub>H<sub>(10-n)</sub>Cl<sub>n</sub>（1≦n≦10）。依氯原子的個數及位置不同，多氯聯苯共有209種[[異构体|異构体]]存在，與1,4-戴奧辛 （[[二噁英|二噁英]]，Dioxin）同屬{{tsl|en|Polyhalogenated compound|多鹵代化合物}} (Polyhalogenated compounds，PHCs)，或稱[[類戴奧辛物質|類戴奧辛物質]] (Dioxins and dioxin-like compounds)。

== 特性 ==
多氯聯苯在[[常溫|常溫]]下是比[[水|水]]重的[[液體|液體]]，多氯聯苯耐[[熱|熱]]性及[[電|電]][[絕緣|絕緣]]性能良好，[[化學性質|化學性質]]穩定。多氯聯苯不溶於水，易溶於[[有機溶劑|有機溶劑]]及[[脂肪|脂肪]]，常用作加熱或冷卻時的熱[[載體|載體]]、[[電容器|電容器]]及[[變壓器|變壓器]]內的絕緣液體 (變壓器油)，也常作為[[切削液|切削液]]、[[潤滑油|潤滑油]]、[[液壓油|液壓油]]、除塵劑、防水化合物、鑄劑、真空泵流體、[[塗料|塗料]]及[[溶劑|溶劑]]使用，應用的範圍很廣。

多氯聯苯混合物，其特徵在於氯的含量，少氯的多氯聯苯是無臭、無味，清澈到暗黃色粘稠液體。而高氯化的混合物是更粘稠、更深的黃色。

多氯联苯异构体有209种可能，这些异构体从单个氯原子的取代到全取代十氯联苯。<ref name="pcb">{{cite web|title=多氯联苯|url=http://www.caslab.com/Chinese/PCB-Congeners.php|accessdate=2013-07-13}}</ref>

== 歷史 ==
多氯聯苯是[[德國|德國]]人[[H·施米特|H·施米特]]和[[G·舒爾茨|G·舒爾茨]]在1881年首次[[化學合成|合成]]的。1892年，[[美國|美國]]開始[[工業|工業]]生產多氯聯苯，1930年代美國[[孟山都|孟山都]]公司 (Monsanto Company)開始大量生產並廣泛應用於工業上。1968年及1979年，[[日本|日本]]及[[台灣|台灣]]分別出現[[米糠油中毒|米糠油中毒]]事件，原因是在生產過程中有多氯聯苯漏出，污染[[米糠油|米糠油]]。

由於多氯聯苯為持久性有機污染物和其環境毒性，於1979年[[美國國會|美國國會]]和於2001年通過的[[斯德哥爾摩|斯德哥爾摩]]關於持久性有機污染物公約，已經禁止了多氯聯苯的生產。因此各國紛紛禁止多氯聯苯生產及使用。

== 危害及影響 ==

多氯聯苯屬於[[致癌物質|致癌物質]]<ref name="toxic">{{cite web|title=多氯聯苯的毒性及急救方式|url=http://www.eric.org.tw/Chm_/92Manual/s001.htm|accessdate=2013-07-13|deadurl=yes|archiveurl=https://web.archive.org/web/20071008185038/http://www.eric.org.tw/Chm_/92Manual/s001.htm|archivedate=2007-10-08}}</ref>，容易累積在[[脂肪組織|脂肪組織]]，造成[[腦部|腦部]]、[[皮膚|皮膚]]及[[內臟|內臟]]的[[疾病|疾病]]，並影響[[神經|神經]]、[[生殖|生殖]]及[[免疫系統|免疫系統]]。

據[[美國環境保護署|美國環境保護署]]（EPA）的報告指出，多氯聯苯已被證明會導致動物，包括人類的癌症，其毒害作用也包括內分泌紊亂和神經毒性。 

個人可以通過呼吸被污染的空氣，進食受污染的食物，以及經皮膚接觸老舊含多氯聯苯的電力設備中接觸到多氯聯苯。一旦暴露，一些多氯聯苯在身體內可改變成其他化學品。這些化學物質或保持不變的多氯聯苯可以被排泄於糞便中，或保持在脂肪或其他器官中幾個月。多氯聯苯也可存在[[牛奶|牛奶]]的脂肪中並通過哺乳傳遞給嬰兒。

在人們接觸到極高水平的多氯聯苯而產生最常見的健康影響是皮膚狀況，如[[氯痤瘡|氯痤瘡]]和[[皮疹|皮疹]]，但這些被稱為是急性全身中毒。有研究發現人員暴露於多氯聯苯中顯示血液尿象的變化，也可能導致[[肝臟|肝臟]]損害。

常見的症狀包括皮膚和眼部病變、不規則的[[月經|月經]]週期和降低[[免疫反應|免疫反應]]。其他症狀包括疲勞、頭痛、咳嗽，和不尋常的皮膚潰瘍。此外，在兒童有認知發展較差的報告。

根據美國環境保護署（EPA）報告指出，多氯聯苯已被證明會導致動物患上癌症，並支持在人類是一種致癌物的證據。在2013年國際研究機構對癌症分類，多氯聯苯為人類致癌物。研究發現在多氯聯苯工人中[[惡性黑素瘤|惡性黑素瘤]]和稀有[[肝癌|肝癌]]，發生率增加。

如所討論的多氯聯苯顯示出廣泛的毒性作用。這些效果可以根據各種不同的多氯聯苯而不同。類似二噁英，多氯聯苯毒性被認為主要是通過結合芳香烴受體（AHR）介導的。干擾細胞內鈣的信號轉導；這可能會導致神經毒性。正多氯聯苯 (Ortho-PCBs)可通過結合到轉甲狀腺素，干擾[[甲狀腺激素|甲狀腺激素]]的運輸。多氯聯苯可在[[免疫系統|免疫系統]]癌症的產生中發揮作用，因為在非常高劑量的多氯聯苯[[動物實驗|動物實驗]]中顯示效果。在一些有關人類的研究聲稱，發現人類免疫反應和環境中多氯聯苯水平之間的關聯。

== 處理及銷毀方法 ==

===物理===

*焚化 - 儘管多氯聯苯不自燃，它們可以被極端和小心控制的條件下燃燒。當前要求多氯聯苯，在1200℃溫度下由燃油和大量氧下進行焚燒至少兩秒鐘。缺氧可導致二噁英、[[呋喃|呋喃]]或多氯聯苯的不完全破壞。

*輻照 - 如果在[[異丙醇|異丙醇]]或礦物油多氯聯苯的脫氧混合物進行照射[[γ射線|γ射線]]，那麼多氯聯苯將被脫氯形成聯苯和無機氯化物。最近在美國往往使用用過的核燃料作為輻射源。 

*熱解 - 銷毀使用等離子弧處理與熱解多氯聯苯。在惰性環境中由[[電弧|電弧]]提供了極高溫度使長鏈分子斷裂。

===微生物===

最近的許多工作都集中在的微生物的研究，其能夠分解多氯聯苯。在一般情況下，這些生物體的工作在兩者之一：無論他們使用的多氯聯苯作為碳源或破壞通過還原脫氯，在聯苯骨架上由氫置換氯。一種新的[[維生素|維生素]]稱為B12s，這是一個功能強大的親核體與還原催化劑。這可以被用在多氯聯苯，以快速和選擇性的方式讓它脫氯。 

===化學===

許多化學方法可用來破壞或減少多氯聯苯的毒性。芳香族親核取代是破壞低濃度多氯聯苯，如變壓器油的方法。在[[氮氣|氮氣]]覆蓋下不到兩小時即可產生聚乙二醇取代氯，無催化劑在700〜925℃下，氫(H2)裂解碳 - 氯鍵，裂解聯苯環形成苯，產生[[鹽酸|鹽酸]]。這可以在較低溫度下用銅催化劑來進行，並得到聯苯。然而，由於這兩種途徑都需要氫氣和相對高的溫度中，它們是昂貴的。高正電的金屬，或強還原劑如[[萘鈉|萘鈉]]，在非質子溶劑產生的[[電子|電子]]轉移到多氯聯苯，氯離子的排出，並在多氯聯苯的結合。其效果是聚合的許多分子，從而減少該混合物的揮發性、溶解性與毒性。這種方法是在低強度多氯聯苯混合物最成功。

多氯聯苯光化學的解決方案是基於傳輸一個電子，如[[胺|胺]]，到光化學興奮的多氯聯苯，給予負離子[[自由基|自由基]]。這兩種排出氯離子和芳基中提取氫原子或立即變為質子化，從而導致氯原子的失去。它僅用於水溶性多氯聯苯。

多氯聯苯破壞大氣的主要途徑是通過攻擊OH自由基。直接光解可以發生在上層大氣層，但激發多氯聯苯的[[紫外線|紫外線]]波長的光，被在[[對流層|對流層]]的[[臭氧|臭氧]]所屏蔽。然而研究表明，在[[光敏劑|光敏劑]]的存在下，例如[[丙酮|丙酮]]，較高波長（> 300 nm）的光可降解多氯聯苯。 

施瓦茨的反應是許多研究的主題，並具有比其他路線顯著好處。這是有利的，因為通過它在還原過程進行，因此，通過氧化反應部會產生二噁英。

所提出的反應方案包括從[[鈦|鈦]]（Ⅲ）的有機金屬種類的電子轉移而形成在多氯聯苯分子的自由基陰離子，其中排出氯，最終形成的相對無毒的聯苯。
