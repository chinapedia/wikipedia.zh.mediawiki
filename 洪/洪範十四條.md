'''洪范十四条'''（{{kor|k=홍범 14 조|hanja=洪範十四條}}）是1894年底[[朝鲜王朝|朝鲜王朝]][[金弘集_(朝鮮)|金弘集]]、[[朴泳孝|朴泳孝]]联立内阁颁布的誓告文。它以[[日本帝国|日本帝国]]驻朝鲜公使[[井上馨|井上馨]]所拟定的《内政改革纲领二十条》为蓝本，由金弘集、朴泳孝等人起草，并在1895年1月7日（朝鲜开国五百三年甲午十二月十二日）由[[朝鲜高宗|朝鲜高宗]]率[[朝鲜纯宗|王世子]]、[[兴宣大院君|大院君]]和文武百官在[[宗廟_(首爾)|宗庙]]向列祖列宗宣誓。'''洪范十四条'''是朝鲜[[甲午改革|甲午改革]]的纲领性文件，意味着朝鲜近代化政治经济改革的全面开启。同时，它的第一条“'''割断附依清国虑念，确建自主独立基础'''”也标志[[朝鲜半岛|朝鲜半岛]]与[[中华帝国|中华帝国]]上千年的[[宗藩关系|宗藩关系]]正式终结。
== 全文 ==
{{quote|
# 割断附依清国虑念，确建自主独立基础。 　　
# 制定王室典范，以昭大位继承暨宗戚分义。 　　
# 大君主御正殿视事，政务亲询各大臣裁决，后嫔宗戚不容干预。 　　
# 王室事务与国政事务，须即分离，毋相混合。 　　
# 议政府及各衙门职务权限，明行制定。 　　
# 人民出税，总由法令定率，不可妄加名目，滥行征收。 　　
# 租税课征及经费支出，总由度支衙门管辖。 　　
# 王室费用，率先减节，以为各衙门及地方官模范。 　　
# 王室费用及各官府费用，预定一年额算，确立财政基础。 　　
# 地方官制，亟行改定，以限节地方官吏职权。 　　
# 国中聪俊子弟，广行派遣，以传习外国学术技艺。 　　
# 教育将官，用征兵法，确定军制基础。 　　
# 民法刑法严明制定，不可滥行监禁惩罚，以保全人民生命及财产。 　　
# 用人不拘门第，求士遍及朝野，以广人才登庸。<ref>参看《朝鲜王朝实录·高宗实录》三十一年十二月十二日条。</ref>}}
