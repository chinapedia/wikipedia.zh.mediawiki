'''洪原郡'''（{{kor|諺=홍원군|漢=洪原郡|羅=Hongwŏn gun|f=y}}）位於[[朝鮮|朝鮮]][[咸鏡南道|咸鏡南道]]中南部，臨近[[朝鮮東海|朝鮮東海]]。洪原郡的面積有905平方公里，郡廳位於[[洪原邑|洪原邑]][[新興里|新興里]]。根據1990年的人口統計，洪原郡有居民100,893人<ref>根據《[[大英百科全書|大英百科全書]]》的數據</ref>。

== 地理 ==
洪原郡位於[[咸鏡山脈|咸鏡山脈]]的西北部，山脈有多座山都位於郡的境內，高度從300米到1300米不等，最高的山峰是有1681米的[[八峰|八峰]]。

郡內的主要河流有起源於[[豆蕪山|豆蕪山]]的[[西大川|西大川]]及起源於[[香坡山|香坡山]]的[[東大川|東大川]]。

洪原郡的海岸線比較平坦，環繞着洪原灣({{lang|ko|홍원만}})。洪原灣位於郡的東南部，面向朝鮮東海。

由於有海風的調節，洪原郡在海岸的氣候與內陸山區的氣候的差別很大。

== 經濟 ==

洪原郡的經濟歷代都是以農業為主，主要的產物有[[大米|大米]]、[[黃豆|黃豆]]、[[珍珠米|珍珠米]]、[[大麥|大麥]]及[[馬鈴薯|馬鈴薯]]。此外，沿岸亦有漁業發展。雖然洪原郡有礦產，但佔生產總值的比例不大，主要的礦產有：[[金|金]]、[[銀|銀]]、[[石灰|石灰]]、[[螢石|螢石]]及[[石墨|石墨]]。

== 歷史 ==

在洪原郡曾發現有[[舊石器時代|舊石器時代]]及[[新石器時代|新石器時代]]的遺物，可見人類在當地生活的歷史很悠久；在景雲面曾發現青銅器時代的物件。

公元6世紀，[[新羅|新羅]][[真興王|真興王]]佔領[[漢江_(韓國)|漢江]]流域後繼續北進，佔領了今日咸鏡南道的部份地區，包括今日的洪原郡。到了公元9世紀，今日的[[咸鏡道|咸鏡道]]及[[平安道|平安道]]都是[[渤海國|渤海國]]的領土，由[[南京南海府|南京南海府]]管治。其後，渤海國為[[契丹|契丹]]所滅，而[[契丹|契丹]]後來亦被[[金朝|金朝]]所滅。到了元朝時，洪原郡成為了[[高麗|高麗]]的一部份。

* 1914年：行政區域改編，原來的社及面合併，成為[[州翼面|州翼面]]({{lang|ko|주익면}})、[[景浦面|景浦面]]({{lang|ko|경포면}})、[[雲浦面|雲浦面]]({{lang|ko|운포면}})、[[龍源面|龍源面]]({{lang|ko|용원면}})、[[龍川面|龍川面]]({{lang|ko|용천면}})、[[平浦面|平浦面]]({{lang|ko|평포면}})、[[好賢面|好賢面]]({{lang|ko|호현면}})、[[希賢面|希賢面]]({{lang|ko|희현면}})、[[鶴泉面|鶴泉面]]({{lang|ko|학천면}})、[[龍雲面|龍雲面]]({{lang|ko|룡운면}})、[[甫青面|甫青面]]({{lang|ko|보청면}}) 等11個面。
* 1936年：州翼面改稱為[[洪原面|洪原面]] ({{lang|ko|홍원면}})，景浦面與雲浦面合併成為[[景雲面|景雲面]] ({{lang|ko|경운면}})，龍川面與平浦面合併成為[[龍浦面|龍浦面]] ({{lang|ko|룡포면}})、部份地區劃入龍源面，好賢面與希賢面合併成為[[普賢面|普賢面]] ({{lang|ko|보현면}})，甫青面與龍雲面一部份及原被劃入[[咸州郡|咸州郡]]、1920年併入的[[東退潮面|東退潮面]]合併成為新的[[三湖面|三湖面]] ({{lang|ko|삼호면}})，鶴川面與龍雲面合併成為[[雲鶴面|雲鶴面]] ({{lang|ko|운학면}})
* 1941年：洪原面昇格成為[[洪原邑|洪原邑]]。
* [[光復節_(朝鮮半島)|朝鮮光復]]前的1944年，洪原郡分為以下1個邑6個面：洪原邑、景雲面、龍源面、龍浦面、普賢面、雲鶴面和三湖面，並設有洪原警察署。
* 1952年12月：洪原郡的洪原面、景雲面、普賢面及雲鶴面合併進洪原郡裡，原各面所屬的里現在直屬洪原郡。
* 1988年7月：신성리及景興里合併進景浦里，成為今日的[[景浦勞動者區|景浦勞動者區]]。
