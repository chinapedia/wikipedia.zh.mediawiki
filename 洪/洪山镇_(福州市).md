'''洪山镇'''是[[中华人民共和国|中国]][[福建省|福建省]][[福州市|福州市]][[鼓楼区|鼓楼区]]下辖的一个[[镇|镇]]。

== 得名 ==
洪山镇得名于境内的[[洪山桥|洪山桥]]。

== 行政区划 ==
中华人民共和国在1949年建国后，洪山镇曾使用了洪山区、红卫公社、洪山公社、洪山乡等地名。<ref name="hszz">{{cite book |author1=《洪山镇志》编撰委员会|title=洪山镇志 |date=1998 |publisher=福建教育出版社 |isbn=7533425243 |accessdate=2018-08-03}}</ref>1992年，洪山乡改名为洪山镇。1995年底，洪山镇从福州郊区（现[[晋安区|晋安区]]）划归鼓楼区。

洪山镇共辖12个[[居委会|居委会]]及7个[[经合社|经合社]]。分别是
*社区居委会：{{PRC admin/children|35/01/02/100/000}}。<ref>{{PRC admin/ref sgc|35/01/02/100/000}}</ref>
*经合社：黎明、西郊、国光、保福、晨光、梅峰、双下。

== 旅游 ==
洪山镇的旅游景点有[[福州左海公园|福州左海公园]]、[[西禅寺|西禅寺]]、[[金牛山公园|金牛山公园]]等。

== 教育科研 ==
洪山镇内的教育及科研机构有[[福州大学|福州大学]]怡山校区、[[闽江学院|闽江学院]]（旧校区）、[[福州黎明职业技术学院|福州黎明职业技术学院]]、[[福建省物构所|福建省物构所]]、福建省高新技术创业园、福建省留学人员创业园、福州市科技园区管委会、福州市创业中心等。
