《'''洪範'''》，《[[尚書|尚書]]》篇名，「洪」是大的意思，「範」是法的意思<ref name="屈">{{cite book|author1=屈萬里|title=尚書今註今譯|publisher=臺灣商務印書館|url=https://books.google.com.tw/books?id=cWvGztwCQ6IC&lpg=PT97&ots=kpn2HNjFLM&hl=zh-TW&pg=PT89#v=onepage&q&f=false}}</ref>。用現代語言來說，洪範就是「大憲」，近似於「國家[[憲法|憲法]]」之意<ref>{{cite book|author1=谢选骏|title=第一期中国文明的精神}}</ref>。

尚書序說《洪範》為[[箕子|箕子]]向[[周武王|周武王]]陳述的「天地之大法」。漢朝時人把《洪範》當作傳說中的「[[洛書|洛書]]」<ref>[[朱自清|朱自清]]《經典常談·周易》：“那講五行的《洪範》，據說也是大禹治水時在洛水中從一隻神龜背上得著的，也出於天賜。所謂‘洛書’，便是那個。”</ref>，《[[漢書|漢書]]·五行志》曰：「禹治洪水，賜《洛書》，法而陳之，《洪範》是也。」，認為禹治洪水時，上天就賜其《洪範》，幫助他治水。[[屈萬里|屈萬里]]認為《洪範》是[[戰國|戰國]]初年的作品<ref name="屈"/>。

《洪範》說：“肅，時寒若”，“乂，時暘若”，意思是說君主施政態度能影響天氣的變化。汉代[[董仲舒|董仲舒]]在《洪範》的理论基础上，提出了“[[天人感應|天人感應]]”之說。

==內容==
傳說[[夏禹|夏禹]]治洪水時，天賜予洪範，分為九疇（九類），助其治水。[[周武王|周武王]]滅殷之後，向殷商遺臣[[箕子|箕子]]問道，箕子認為「天以是道畀之禹，傳至於我，不可使自我而絕，以武王而不傳，則天下無可傳者矣」，於是為武王陳述了這篇建國君民的大則大法<ref>{{cite encyclopedia|author=井敏珠|title=洪範九疇|encyclopedia=教育大辭書|url=http://terms.naer.edu.tw/detail/1307541/?index=97}}</ref>：

# 五行：水、火、木、金、土
#:身為一國之君，須明察[[五行|五行]]之理，善於調和運用這五種類型的資材（material），來促進人民生活安樂。
# 敬用五事：貌、言、視、聽、思
#:國君治理人民要具備這五樣修養：容貌恭敬才能表現嚴肅；言論正當辦事才會順利；視察清楚才可明辨一切；聽聞聰敏謀事才易成功；思慮通達成為聖人。
# 農用八政：食、貨、祀、[[司空|司空]]、[[司徒|司徒]]、[[司寇|司寇]]、賓、師
#:為民設置職掌食糧、財貨、祭祀、營建、戶口、刑獄、朝覲、軍事的官員。
# 協用五紀：歲、月、日、星辰、曆數
#:君王的施政要協天時以敬人事，了解這五種紀天時的方法，才可獲致最大成效。
# 建用皇極：大中至正之[[王道_(儒家思想)|王道]]
#:人君治民、教民、養民所應遵守的法度：無偏無陂，遵王之義；無有作好，遵王之道；無有作惡，尊王之路。無偏無黨，王道蕩蕩；無黨無偏，王道平平；無反無側，王道正直。
# 乂用三德：正直、剛克、柔克
#:治理人民所當採取的三種方法。平和安寧之世，用正直之道來治理；若有頑劣倔強的人，用強硬的武力制服，使其收斂；若對溫馴柔順的人，則用迂迴友善的態度，贏取信賴。
# 明用稽疑：謀及乃心，謀及卿士，謀及庶人，謀及卜筮（雨，霽，蒙〔霧〕，驛〔圛〕，克，貞，悔）
#:國君倘若遇到重大疑難，自己要好好思量，其次與[[卿士|卿士]]商議，然後徵求人民的意見，最後再[[卜筮|卜筮]]尋求天意。卜兆有五：雨，霽，蒙〔霧〕，驛〔圛〕，克；占筮有二：貞，悔
# 念用庶徵：雨、暘、燠、寒、風；休徵、咎徵
#:觀察「雨」(下雨)、「暘」（晴天）、「燠」（炎熱）、「寒」（寒冷）、「風」（刮風）的天象，來驗證施政得失的徵兆；人君須時時修德執中，勤政愛民，不可有稍忽逸樂與怠惰。
# 嚮用五福：壽、富、康寧、攸好德、考終命；威用六極：凶短折、疾、憂、貧、惡、弱
#:君行王道，天即賜五福；不行王道，天則降六極。
