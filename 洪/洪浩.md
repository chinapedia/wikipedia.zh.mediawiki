'''洪浩'''（{{bd|1955年|5月|}}），[[浙江省|浙江省]][[鄞县|鄞县]]人。曾任[[上海市人民政府|上海市人民政府]][[秘书长|秘书长]]，高级[[经济师|经济师]][[职称|职称]]。

==个人简历==

毕业于上海[[复旦大学管理学院|复旦大学管理学院]]工业企业管理专业，获得[[学士|学士]][[学位|学位]]。

1972年12月至1977年3月，在[[中国人民解放军|中国人民解放军]]服役。期间获得嘉奖多次，并获得“钢铁战士”[[荣誉称号|荣誉称号]]。

1977年4月至1994年7月，先后服务于上海市电车1场和4场，终任常务副场长。

1977年4月至1994年7月，服务于[[上海市公交公司|上海市公交公司]]，终任常务副总经理。

1994年7月至2000年4月，曾服务于[[上海市公用事业局|上海市公用事业局]]，担任常务副局长。并曾服务于[[上海市建设委员会|上海市建设委员会]]，担任常务副主任。

2000年4月至2007年2月，服务于[[上海市环保局|上海市环保局]]，担任环保局局长。并曾担任[[上海市人民政府|上海市人民政府]]常务副秘书长。

2007年2月26日，除上海市政府的公务外，并兼任[[中国2010年上海世界博览会|上海世博会]]事务协调局（简称上海“世博局”）局长，负责[[中国2010年上海世界博览会|2010年上海世博会]]的统筹规划工作。

2011年4月，被任命为[[上海市人民政府|上海市人民政府]][[秘书长|秘书长]]<ref name="新浪上海新闻">{{cite web |url = http://sh.sina.com.cn/news/s/2011-04-13/0811179267.html |title=姜平当选上海市副市长 洪浩受任上海市政府秘书长 |publisher=[[新浪网|新浪网]]上海新闻 |date= 2011年4月13日 |accessdate= 2011年5月16日}}</ref><ref name="上海人大新闻">{{cite web|url=http://www.spcsc.sh.cn/cwhgz/content/2011-05/27/content_73864.htm|title=十三届常委会第二十六次会议任命洪浩为上海市人民政府秘书长|publisher=上海人大公众网|date=2011-05-27|accessdate=2011年5月16日|deadurl=yes|archiveurl=https://web.archive.org/web/20111123071121/http://www.spcsc.sh.cn/cwhgz/content/2011-05/27/content_73864.htm|archivedate=2011年11月23日}}</ref>。

2013年3月，当选[[上海市人大常委会|上海市人大常委会]]副主任、[[上海市总工会|上海市总工会]]主席<ref name="洪浩工会">{{cite web |url = http://news.ifeng.com/mainland/renshi/detail_2013_05/31/25938291_0.shtml |title=洪浩当选上海市总工会主席 |publisher=凤凰网 资讯 |date=2013年5月31日 |accessdate= 2013年9月19日}}</ref>。
