'''筑波'''（{{lang|ja|つくば}}）是[[大日本帝國海軍|大日本帝國海軍]]的[[巡洋戰艦|巡洋戰艦]]。[[筑波級巡洋戰艦|筑波級巡洋戰艦]]一號艦。艦名是以[[茨城縣|茨城縣]]的「[[筑波山|筑波山]]」來命名。在日本海軍中為第2隻艦船使用該名稱。

== 概要 ==
起初以擁有[[戰艦|戰艦]]砲力的巡洋艦而計劃・急造，於1912年新設巡洋戰艦艦種之前一直是一等巡洋艦（[[裝甲巡洋艦|裝甲巡洋艦]]）。而在[[簡氏海軍年鑑|簡氏海軍年鑑]]中，在艦種類別變更之前被分類為裝甲巡洋艦而在[[華盛頓裁軍條約|華盛頓裁軍條約]]中與同型艦[[生駒號巡洋戰艦|生駒]]則被當作為戰艦，成為條約中所限制的對象。

== 建造經緯 ==
由於在[[日俄戰爭|日俄戰爭]]初期喪失了戰艦[[初瀬號戰艦|初瀨]]與[[八島號戰艦|八島]]，因而需以[[日俄戰爭|日露戰爭]]中的臨時軍費建造替代艦。其後本艦在自行設計下、並由[[吳工廠|吳工廠]]建造，成為最初國產大艦。艦體・引擎及防御力為裝甲巡洋艦，而主砲則與戰艦同樣搭載12吋砲。

== 略歷 ==
* 1905年 於吳工廠動工，同年下水。
* 1907年 竣工、並航向歐美米作訪問 。
* 1914年 [[第一次世界大戰|第一次世界大戰]]時被派遣到[[西太平洋|西太平洋]]。參加佔領南洋諸島。
* 1917年 於[[橫須賀港|橫須賀港]]時火藥庫發生爆炸。大破著底。除籍
