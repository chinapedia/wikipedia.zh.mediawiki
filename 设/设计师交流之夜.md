'''设计师交流之夜'''（英文：PechaKucha 或Pecha Kucha，日文：{{lang|ja|ペチャクチャ}}<ref>[http://www.forvo.com/word/pecha_kucha/ Japanese pronunciation at Forvo]</ref>） 是一种演讲形式，该演讲规定使用20张[[演示文稿|演示文稿]]，并用20秒讲述一张的时间完成（即共6分40秒的演讲时间），旨在保证演讲内容简洁和快节奏的演讲形式。<div><ref name="ipo1">{{Template:Cite web|publisher = Intellectual Property Office|location = United Kingdom|title = Case details for Community Trade Mark E5374426|url = http://www.ipo.gov.uk/ohim?ohimnum=E5374426|date = 11 February 2008}}</ref><ref name="ipo2">{{Template:Cite web|publisher = Intellectual Property Office|location = United Kingdom|title = Case details for Trade Mark 2417930|url = http://www.ipo.gov.uk/domestic?domesticnum=2417930|date = 9 June 2006}}</ref></div>

''设计师交流之夜''这样的演讲形式是由[[东京|东京]][[建筑设计|建筑设计]]公司Klein-Dytham Architecture (KDa) <ref>{{Template:Cite journal|journal = [[时代_(杂志)|时代]]|author = Jean Snow|title = All Talk|url = http://www.time.com/time/magazine/article/0,9171,501060724-1214999,00.html|date = July 17, 2006|accessdate = 2012-07-04}}</ref><ref>{{Template:Cite web|url = http://www.pechakucha.org/cities/tokyo/events/50b3feecf3b43bdb0b0000f0|title = PechaKucha 20x20 - Tokyo - Vol. 1|accessdate = 23 May 2014}}</ref> 的Astrid Klein 和 Mark Dytham于2003年设计出的，起初的目的是吸引年轻的[[设计师|设计师]]去位于[[六本木|六本木]]的SuperDeluxe参加他们的实验性活动，在那里可以展示设计师们的作品和交流想法。<ref name="metropolis"><cite class="citation web">Jean Snow (June 9, 2006). [http://archive.metropolis.co.jp/tokyo/637/feature.asp "20/20 Vision: The Tokyo-born Pecha Kucha phenomenon has the global creative community hooked"] {{webarchive|url=https://web.archive.org/web/20091226133403/http://archive.metropolis.co.jp/tokyo/637/feature.asp |date=2009-12-26 }}. </cite></ref>

2004年，欧洲的几个城市也开始尝试该活动。<ref name="pk">{{Template:Cite web|url = http://pechakucha.org|title = PechaKucha 20x20}}</ref><ref name="wired">{{Template:Cite web|url = http://www.wired.com/techbiz/media/magazine/15-09/st_pechakucha#|title = Pecha Kucha: Get to the PowerPoint in 20 Slides Then Sit the Hell Down|author = [[Daniel_H._Pink|Daniel H. Pink]]|accessdate = 2009-09-29|publisher = [[Wired_(magazine)|Wired (magazine)]]|date = 2007-08-21}}</ref> 截止到2014年5月，PKNs已经在全球700个城市里举办过。 <ref>{{Template:Cite web|title = PechaKucha 20x20 - Attend|url = http://www.pechakucha.org/attend|archiveurl = http://www.webcitation.org/6PWN6wy0o|archivedate = 12 May 2014}}</ref>

== 活动形式 ==
一个标准的设计师交流之夜通常会邀请8-14个演讲者。每位演讲者准备20张演示文稿(PPT)，并用20秒时间讲述一页。不同城市的主办方或许会加上他们自己的特色和形式。
在[[荷兰|荷兰]]的[[格罗宁根|格罗宁根]]，演讲的空隙会有现场乐队演出，而在每个演讲的最后20秒，会有主办方邀请的评论人进行即时评论。 

演讲的听众一般都是来自[[设计|设计]]，[[建筑|建筑]]，[[摄影|摄影]]，[[艺术|艺术]]和创意领域，也有来自院校的学生<ref>{{Template:Cite news|url = http://www.boston.com/ae/theater_arts/articles/2010/06/28/pecha_kucha_designed_to_inspire/|work = The Boston Globe|title = Designed to inspire|first = Sam|last = Allis|date = 2010-06-28}}</ref> 多数演讲者是专业设计师来展示他们的设计作品，但也有演讲者讲述的主题是他们的旅行，研究课题，学习项目，兴趣爱好，收藏，或是其他的兴趣。有些活动中也会用到视频作品作为演讲素材。

== 如何发起一个设计师交流之夜 ==
由于PechaKucha是一个被Klein-Dytham Architecture (KDa)[[注册商标|注册的商标]]。因此通常需由主办方联系''设计师交流之夜''（PechaKucha）委员会来发起一个活动，完成非正式的申请程序之后，双方达成相关的“握手”协定便可。设计师交流之夜并没有形式上的限制。任何受邀请人都可以准备一个演讲。<ref>{{Template:Cite web|url = http://www.pechakucha.org/start-a-city|title = PechaKucha 20x20 - Start a City|accessdate = 23 May 2014}}</ref>
