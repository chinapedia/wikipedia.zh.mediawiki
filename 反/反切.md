{{音韻學}}
'''反切'''（又稱'''反語'''、'''音反'''）是中國自古以來[[漢字|漢字]]（廣義的）[[標音|注音]]方法，一般取兩個常用字：前者稱'''反切上字'''，取其雙聲之[[聲母|聲母]]；後者稱'''反切下字'''，取其疊韻之[[韻母|韻母]]和[[聲調|聲調]]{{NoteTag|所謂上、下本是豎書時之稱謂，雖然漢文化圈主要已採用橫書，但並無「左字」「右字」等稱呼。}}<ref>{{Cite book|title=切韻考|last=陳|first=灃|publisher=|year=|isbn=|location=|pages=}}</ref>。反切是中國古代最主要和使用時間最長的注音方法之一。

至於反切法之發明有說啟發自西域[[梵文|梵文]]字<ref>{{Cite book|title=通志・藝文略|last=鄭|first=樵|publisher=|year=|isbn=|location=|pages=}}</ref>，因此狹義上講，規律整齊的反切注音系統是[[拼寫字母|拼音字母]]系統<ref name=":0">{{Cite book|title=《中國語言學史》|last=王|first=力|publisher=中國圖書刊行社 (China Book Press)|year=1984|isbn=9620403282|location=香港}}</ref>。

== 起源考 ==

=== 反切以前的注音方法 ===
在反切法誕生之前，[[直音法|直音法]]（即是「讀若某」、「如某」等）是常用的字書注音方法。這種方法在東漢之前佔據字書之主流，例如《[[說文解字|說文]]・玉部》：「珣，讀若宣」；《[[說文解字|說文]]・宀部》：「宋，居也，從宀木，讀若送。」等。一般是用一個字注另一個同音字。

也因此，直音法的侷限十分明顯：依時空之不同，每個字的發音都存在差異，且一些情況下，使用某字的假借字（如廣韻裡「行」字和「絎」、「胻」字等）或[[生僻字|生僻字]]來注音，有時反而會造成不解<ref name=":0" />。這種情況在其他注音方法出現之後得到了改善，而韻書也常常把多種方法一起使用以追求準確。

=== 反切興廣的年代 ===
關於反切法之源起，學界說法不一，但可以確定的是，它出現於佛教傳入中原後的東漢，魏晉開始盛行{{NoteTag| 「至於魏世，此事大行」此說法主要見於《顏氏家訓・音辭》，是學界的主要觀點。王力的《中國語言學史》中記錄了關於此論點的辯證，此不贅述，詳查原本。}}<ref>{{Cite book|title=[[顏氏家訓|顏氏家訓]]・音辭|last=顏|first=之推|publisher=|year=|isbn=|location=|pages=}}</ref>。至於誰是第一個使用反切的人，自古以來便無定論，但總結起來不外乎三類：
# 一部分人認為是著《[[爾雅音注|爾雅音注]]》一書的<u>[[孙炎|孫炎]]</u>。比如<u>顏之推</u>《[[顏氏家訓|顏氏家訓]]·音辭篇》有：「孫叔然（[[孫炎|孫炎]]）創《[[爾雅音義|爾雅音義]]》，是漢末人獨知反語」；還有<u>[[陸德明|陸德明]]</u>之《[[經典釋文|經典釋文]]・敘錄・條例》：「古人音書，止為譬況之說，<u>孫炎</u>始為反語」。
# 一部分人提出在<u>孫炎</u>之前便可見使用此方法（雖然並未按照定律）的例子，但具体始于何人也有争议。比如<u>[[章太炎|章炳麟]]</u>的《[[國故論衡|國故論衡]]・音理論》就有：「造反語者，非始<u>孫叔然</u>也。」又說：「又尋漢地理志...<u>[[應劭|應劭]]</u>注：『垫音徒浹反』」指向了<u>應劭</u>；《一切经音译》又说：「古来音反，多以旁纽为双声，始自<u>[[服虔|服虔]]</u>」，<u>[[服虔|服虔]]</u>和<u>[[應劭|應劭]]</u>同是漢末人，與<u>孫炎</u>是同時代。
#  一部分人則認為反切的初創跟[[佛教|佛教]]傳入有關。例如[[陳振孫|陳振孫]]《直齋書錄題解·韻補》：「反切之學，自西域入中國」；<u>[[鄭樵|鄭樵]]</u>《[[通志|通志]]・藝文略》：「切韻之學起於西域」等。這種說法有正史《[[隋書|隋書]]・經籍志》的支撐，進一步指出是[[汉明帝|漢明帝]]時隨佛經同入中國。

此三類說法在年代上並無太大出入，然而第三種「反切西域說」卻收到了很大的反對。例如以<u>[[胡以魯|胡以魯]]</u>為首的人指出，漢代以前所見的「合音」現象，如「不可」為「叵」，「何不」為「盍」，「如是」為「爾」，「而已」為「耳」，「之乎」為「諸」，「者焉」為「旃」等，就可以做為反對的證據。

[[钱大昕|钱大昕]]说《[[诗经|诗经]]》已有反切，並且驳斥反切受[[佛经|佛经]]的影响，“岂古圣贤之智乃出梵僧下耶”，“吾于是知六经之道，大小悉备，后人詹詹之智，早不出圣贤范围之外也”，但不能因此就否定反切普遍使用與佛教傳入有關<ref>钱大昕：《潜研堂文集》卷十五，《答问十二》。</ref>。

但是雖然合音和反切都是二字得出一音，但是卻有本質的不同。合音完全出於自然，例如是先有“不可”，快讀時自然合音為“叵”，當中並無對字音作出有意識的分析，“不可”也不是為標注“叵”的讀音而出現。而反切則是自覺地分析漢字讀音的結果。

=== 由反切所孕育出的韻書時代 ===
{{Main articles|韻書}}

反切乃中國第一個有系統、科學化的拼音方法，是古典[[聲韻學|聲韻學]]的一大里程碑。它對後世的影響是巨大的且不可否認的。

例如《[[切韵考|切韻考]]》云：「蓋有反語，則類聚之即成韻書，此自然之勢也」，言反切是韻書之基礎也。自漢代出現反切以來，大大便利了文人對古籍的學習和在創作，由於當時學術派別與師承之傳統，後來所編排的韻書也各有差異。而以《[[切韻|切韻]]》為首的韻書打破了這樣的局面，開啟了中國古代語言學的一個新時代<ref>{{Cite book|title=漢語等韻學|last=李|first=新魁|publisher=中華書局|year=1983年11月|isbn=|location=|pages=4}}</ref>。

隋唐的反切是研究[[中古漢語|中古漢語]]語音系統的基本資料，是[[漢語音韻學|漢語音韻學]]的基礎之一。

== 拼讀方法 ==
中國古代因為沒有拼音，所以用兩個漢字來注一個字的音。第一個字（反切上字）注[[聲母|聲母]]，第二個字（反切下字）注[[韻母|韻母]]和[[聲調|聲調]]。例如：'''𠁁，徒口切。'''“'''徒'''”的聲母為'''定母'''，中古擬音[d]，“'''口'''”的韻母為'''侯韻'''，中古擬音為[əu]，聲調為'''上聲'''，故'''𠁁'''為定母侯韻上聲，中古擬音為上聲[dəu]。

但是古字典的反切只適用於當時的古音，不可單憑古代字典的反切記錄套用于現代音。因爲古音和現代音有較複雜的演變規律，且不同的漢語族語言也有不同的演變。比如上面的'''𠁁'''，如果儘憑古字典“徒口切”，取“徒”的[[普通話|普通話]]聲母t，和"口"的普通話韻母ou3，推斷為普通話讀做tou3，就錯了。因爲“徒”中古音聲母是全濁音[d]，該聲母的非平聲字按規律在普通話中聲母要演變為清不送氣音d[t]，同時這個中古全濁聲母影響了聲調演變，使中古上聲一律變為普通話去聲，即'''濁上變去'''律。所以'''𠁁'''的現代普通話讀音是dou4[təu˥˩]。

可見按照「上字取聲，下字取韻」的原則，古人或能很準確地讀出一字字音。然而，由於古今音變的關係，古代所製定的反切，到後世已不能再簡單地以「上字取聲，下字取韻調」的原則讀出該字正確讀音。
依《[[廣韻|廣韻]]》一書，除了「上字取聲母，下字取韻母」之原則，還有一口訣：「上字辨陰陽{{NoteTag|聲母的清音濁音，見[[平分陰陽|平分陰陽]]}}，下字辨平仄{{NoteTag|韻母，見[[四聲|四聲]]、[[入聲韻|入聲韻]]}}」。

== 反切的局限 ==
#須先懂得一部份漢字才可以用反切法讀出字音
#用作反切的上、下字太多，共有400多上字，1000多個下字，作為5000多反切字。如：東為德紅切十七，即有十七個同音字。
#反切上下字聲韻調交雜，不便於拼音。上字有聲無韻、下字有韻無聲，完全依賴連讀，拼音方法並不方便。
#在《[[廣韻|大宋重修廣韻]]》一書，簡單易讀的字，往往用複雜的字作切語，如：「一」為-{於}-悉切；「八」為博撥切。
