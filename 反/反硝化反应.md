'''脫硝反应'''（{{Lang-en|'''denitrification'''}}，亦称为'''脫硝作用'''、'''脫氮作用'''）是指细菌将[[硝酸盐|硝酸盐]]（NO<sub>3</sub><sup>−</sup>）中的[[氮|氮]]（N）通过一系列中间产物（NO<sub>2</sub><sup>−</sup>、NO、N<sub>2</sub>O）还原为[[氮气|氮气]]分子（N<sub>2</sub>）的生物化学过程。参与这一过程的细菌统称为'''脫硝細菌'''或'''脫氮細菌'''。

脫硝化菌在无氧条件下，以将[[硝酸盐|硝酸盐]]（NO<sub>3</sub><sup>−</sup>）为{{le|电子受体|Electron acceptor}}完成[[呼吸作用|呼吸作用]]（respiration）以获得能量。这一过程是[[硝酸盐呼吸|硝酸盐呼吸]]（nitrate respiration）的两种途径之一，另一种途径是另一类是[[硝酸异化还原成铵盐|硝酸异化还原成铵盐]]（Dissimilative reduction of nitrate to ammonia, DRNA）。

== 反应 ==
总的脫硝过程可以用以下方程式表示：
:2 NO<sub>3</sub><sup>−</sup> + 10 e<sup>−</sup> + 12 H<sup>+</sup> → N<sub>2</sub> + 6 H<sub>2</sub>O, ΔG<sup>0</sup> = −333 kJ/mol
其中包括以下四个[[还原反应|还原反应]]还原反应：
# [[硝酸盐|硝酸盐]]（NO<sub>3</sub><sup>−</sup>）还原为[[亚硝酸盐|亚硝酸盐]]（NO<sub>2</sub><sup>−</sup>）：2 NO<sub>3</sub><sup>−</sup> + 4 H<sup>+</sup> + 4 e<sup>−</sup> → 2 NO<sub>2</sub><sup>−</sup> + 2 H<sub>2</sub>O
# [[亚硝酸盐|亚硝酸盐]]（NO<sub>2</sub><sup>−</sup>）还原为[[一氧化氮|一氧化氮]]（NO）：2 NO<sub>2</sub><sup>−</sup> + 4 H<sup>+</sup> + 2 e<sup>−</sup> → 2 NO + 2 H<sub>2</sub>O
# [[一氧化氮|一氧化氮]]（NO）还原为[[一氧化二氮|一氧化二氮]]（N<sub>2</sub>O）：2 NO + 2 H<sup>+</sup> + 2 e<sup>−</sup> → N<sub>2</sub>O + H<sub>2</sub>O
# [[一氧化二氮|一氧化二氮]]（N<sub>2</sub>O）还原为[[氮气|氮气]]（N<sub>2</sub>）：N<sub>2</sub>O + 2 H<sup>+</sup> + 2 e<sup>−</sup> → N<sub>2</sub> + H<sub>2</sub>O

以上四个反应均为放热反应，所以在无氧或缺氧条件下，细菌可以将[[硝酸盐|硝酸盐]]（NO<sub>3</sub><sup>−</sup>）作为[[电子传递链|电子传递链]]（ETC）的最终电子受体（TEA, terminal electron acceptor），来完成物质能量交换。

== 脫硝菌 ==
脫硝菌在自然界以各种形式广泛存在，如：
* ''[[Paracoccus_denitrificans|Paracoccus denitrificans]]''（[[自养|自养]]，氧化[[氢气|氢气]]H<sub>2</sub>）
* ''[[Thiobacillus_denitrificans|Thiobacillus denitrificans]]''（[[自养|自养]]，氧化[[硫化物|硫化物]]（S<sup>2−</sup>）或者[[硫代硫酸盐|硫代硫酸盐]]（S<sub>2</sub>O<sub>3</sub><sup>2−</sup>））
* ''[[Pseudomonas_stutzeri|Pseudomonas stutzeri]]''（[[异养|异养]]，氧化有机碳）
脫硝菌主要为原核生物，大量存在于在α-, β- 和γ-变形菌纲中。已知的脫硝菌的属有''Achromobacter, Acinetobacter, Agrobacterium, Bacillus, Chromobacterium, Flavobacterium, Spirillum, Vibrio, Halobacterium, Methanomonas, Pseudomonas''等。

尽管已经发现了[[自營|自營]]脫硝菌，但上述脫硝过程主要由[[异營|异營]]脫硝菌来完成。

== 应用及意义 ==
脫硝反应在自然界具有重要意义，是[[氮循环|氮循环]]的关键一环，它和[[無氧氨氧化|無氧氨氧化]]（Anammox）一起，组成自然界被固定的氮元素重新回到大气中的途径。

在环境保护方面，脫硝化反应和[[硝化反应|硝化反应]]一起可以构成不同工艺流程，是[[生物除氮|生物除氮]]的主要方法，在全球范围内的[[污水处理厂|污水处理厂]]中被广泛应用。污水处理中所利用的脫硝菌为[[异养菌|异养菌]]，其生长速度很快，但是需要外部的有机碳源，在实际运行中，有时会添加少量[[甲醇|甲醇]]等有机物以保证反硝化过程顺利进行。
