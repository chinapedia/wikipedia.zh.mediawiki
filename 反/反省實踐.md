'''反省實踐（'''{{Lang-en|reflective practice}}）又譯做'''反思實踐，'''是在學習的過程中能夠對行動進行反思的能力<ref name="Schon">{{cite book |last = Schön|first = Donald A.|authorlink = Donald Schön|date = 1983|title = The reflective practitioner: how professionals think in action|location = New York|publisher = Basic Books|isbn = 046506874X|oclc = 8709452|ref = harv}}</ref>。 有人定義反省實踐是「透過仔細與自我反省的方式檢視平日實踐，來對日常生活中實務上遭遇的價值觀與理論給予批判性的注意。這樣的練習能夠培養洞見。」<ref>{{cite book |last = Bolton|first = Gillie|date = 2010|origyear = 2001|title = Reflective practice: writing and professional development|edition = 第3版|location = Los Angeles|publisher = Sage Publications|isbn = 9781848602113|oclc = 458734364|page = [http://books.google.com/books?id=XSNiMpiHuwcC&pg=PR19 xix]}}</ref>培養反省實踐能力的重於原因，在於單獨只有經驗無法使學習發生；在得到經驗的同時也有進行{{Link-en|人類的自省行為|Human_self-reflection|反省}}是十分必要的<ref name="Loughran">{{cite journal |last = Loughran|first = J. John|date = 2002-01|title = Effective reflective practice: in search of meaning in learning about teaching|journal = Journal of Teacher Education|volume = 53|issue = 1|pages = 33–43|doi = 10.1177/0022487102053001004|url = http://oneteacher.global2.vic.edu.au/files/2014/09/Loughlan-242761j.pdf}}</ref><ref>{{cite journal |last1 = Cochran-Smith|first1 = Marilyn|last2 = Lytle|first2 = Susan L.|date = 1999-01|title = Relationships of knowledge and practice: teacher learning in communities|journal = Review of Research in Education|volume = 24|issue = 1|pages = 249–305|doi = 10.3102/0091732X024001249|jstor = 1167272}}</ref>。

反省實踐能夠成為{{Link-en|實踐本位專業學習|practice-based professional learning}}裡十分重要的工具之一，人們能夠在執業經驗中學習，而不只從{{Link-en|正式學習|formal learning}}或{{Link-en|知識轉移|knowledge transfer}}裏頭獲得。 反省實踐可能是在個人的{{Link-en|專業發展|professional learning}}以及修正中最重要的源頭。同時，它也是可以將理論與實踐結合起來的一種很重要的方法；透過反省，人可以發現並標記出在他工作成果的脈絡之中的各類想法或理論<ref>{{cite journal |last = McBrien|first = Barry|date = July 2007|title = Learning from practice—reflections on a critical incident|journal = [[Accident_and_Emergency_Nursing|Accident and Emergency Nursing]]|volume = 15|issue = 3|pages = 128–133|doi = 10.1016/j.aaen.2007.03.004|pmid = 17540574}}</ref>。一個在實踐中不斷反思的人不只是回顧過去的行動與事件，同時也要有意識去照看情緒、行動、與反應，使用這些資訊來添增到他既有知識庫中以達成更高層次的理解<ref name="paterson-chapman">{{cite journal |last1 = Paterson|first1 = Colin|last2 = Chapman|first2 = Judith|date = August 2013|title = Enhancing skills of critical reflection to evidence learning in professional practice|journal = Physical Therapy in Sport|volume = 14|issue = 3|pages = 133–138|doi = 10.1016/j.ptsp.2013.03.004|url = http://www.researchgate.net/profile/Judith_Chapman4/publication/236637498_Enhancing_skills_of_critical_reflection_to_evidence_learning_in_professional_practice/links/540992db0cf2822fb739aff7.pdf|pmid = 23643448}}</ref>。
