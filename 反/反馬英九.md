'''反馬英九'''，又稱'''倒馬'''、'''反馬'''、'''罷馬'''或'''嗆馬'''，係專指在[[馬英九|馬英九]]總統在任期間，發生於[[臺灣|臺灣]]的多起[[社會運動|社會運動]]，多以要求時任[[中華民國總統|中華民國總統]]的[[馬英九|馬英九]][[辭職|辭去總統職務]]為訴求，同類的臺灣政治運動有[[百萬人民倒扁運動|倒扁]]。

反馬運動的時間主要發生在[[馬英九政府|馬英九政府]]任期期間，主要是[[泛綠|泛綠]]支持者推動，但也有不少同樣反對泛綠政黨的第三勢力聲音。在其社會運動的成員隨著時間推移，有擴大至臺灣社會各層面的趨向，見下文所述。

== 背景 ==
{{seealso|馬英九#連戰內閣時期}}
起初，在馬英九於[[中華民國法務部|中華民國法務部]]部長任內期間，嚴打[[臺灣黑金政治|黑金]]，得罪不少同屬[[中國國民黨|中國國民黨]]的黨內同志，後索性辭官，獲得當時民眾的好評<ref>當時內閣滿意度第一二名為馬英九與劉兆玄，分任法務部長與交通部長。</ref>；其後，[[1998年中華民國直轄市市長暨市議員選舉#.E8.87.BA.E5.8C.97.E5.B8.82|1998年臺北市長選舉]]，在[[宋楚瑜|宋楚瑜]]宣布不參選<ref>[http://news.cts.com.tw/cts/general/199802/199802230012949.html 宋楚瑜不參選臺北市長]華視新聞，1998-02-23。</ref>後投入選戰，擊敗當時尋求連任的[[民主進步黨|民主進步黨]]籍候選人[[陳水扁|陳水扁]]與[[新黨|新黨]]的[[王建煊|王建煊]]後，其聲勢扶搖直上；[[2000年中華民國總統選舉|2000年中華民國總統選舉]]，分裂的國民黨([[連戰|連戰]]、[[宋楚瑜|宋楚瑜]])被民進黨擊敗，臺灣在無預期下完成第一次[[政黨輪替|政黨輪替]]，在野的[[泛藍|泛藍]]陣營尋求整合，以期奪回執政權，時任[[臺北市市長|臺北市市長]]的馬英九以清廉形象與高個人魅力被視為未來可以擊敗陳水扁的政治明星；[[2004年中華民國總統選舉|2004年中華民國總統選舉]]，國民黨的連宋配以微弱差距再度敗給民進黨，馬英九被推為「泛藍共主」；2005年，馬英九擊敗立法院長[[王金平|王金平]]，成為[[中國國民黨主席|中國國民黨主席]]，其後，陳水扁爆發[[國務機要費案|國務機要費案]]，引起了社會不滿，導致[[倒扁運動|倒扁運動]]正式產生，[[泛綠|泛綠]]陣營支持度大幅低落，終於在[[2008年中華民國總統選舉|2008年中華民國總統選舉]]時，馬英九擊敗了民進黨的[[謝長廷|謝長廷]]與[[蘇貞昌|蘇貞昌]]，成為[[中華民國總統|中華民國總統]]，達到其從政生涯最高峰，臺灣完成第二次政黨輪替。

== 不支持罷免馬英九的看法 ==
由於按照現行[[中華民國憲法|中華民國憲法]]規定，當總統於任期未滿時缺位時，由副總統繼位接任至任期屆滿後再改選，而時任副總統為[[吳敦義|吳敦義]]。

因此，不支持罷免馬英九的人士，也有一部分是屬於反對吳敦義的，而非等於認同馬英九及國民黨政府的一切作為。

* [[蔣孝嚴|蔣孝嚴]]（前[[國民黨副主席|國民黨副主席]]）：「臺灣已是非不分道德淪喪」，「現在狀況的確是很混亂，[[烏賊|烏賊]]滿街跑，但只要有是非的人，都會力挺馬英九。」<ref>[http://big5.gwytb.gov.cn/xuan/201309/t20130929_4956362.htm 蔣孝嚴力挺馬英九：臺灣已是非不分道德淪喪] {{Wayback|url=http://big5.gwytb.gov.cn/xuan/201309/t20130929_4956362.htm |date=20160309062458 }}，中共中央臺灣工作辦公室、國務院臺灣事務辦公室，2013-09-29。</ref>

* [[謝長廷|謝長廷]]（前[[行政院院長|行政院院長]]）：「罷免馬英九，讓副總統變總統，這…。」<ref>[http://www.chinatimes.com/realtimenews/20130915001107-260407 罷免馬？謝長廷：副總統變總統…這…]，中時電子報 ，2013-09-15。</ref>
