'''反町站'''（{{jpn|j=反町駅|hg=たんまちえき|rm=Tammachi eki|f=y}}）是一個位於[[神奈川縣|神奈川縣]][[橫濱市|橫濱市]][[神奈川區|神奈川區]]上反町，屬於[[東京急行電鐵|東京急行電鐵]][[東橫線|東橫線]]的[[鐵路車站|鐵路車站]]。[[車站編號|車站編號]]是'''TY20'''。

==歷史==
* 1926年（[[大正|大正]]15年）2月14日 - 此站啟用<ref name="jtb68">[[#jtb|東急の駅]]、p.68。</ref>。
* 1960年（[[昭和|昭和]]35年） - 車站改建<ref name="jtb68"/>。
* 2004年（[[平成|平成]]16年）1月31日 至 2005年（平成17年）2月1日 - 為配合與此線與[[港未來線|港未來線]]進行[[直通運轉|直通運轉]]，車站進行地下化工程。<ref>[https://ja.wikipedia.org/wiki/%E6%96%B0%E4%B8%B8%E5%AD%90%E9%A7%85#jtb 東急の駅、p.68。]</ref><ref> [http://hot.tokyu.co.jp/railway/hot/2004/2004_02.pdf 「横浜駅・反町駅が変わります」 (pdf) 、『HOT ほっと TOKYU』第277号、東京急行電鉄、2004年1月20日、2015年1月14日瀏覽。] {{webarchive|url=https://web.archive.org/web/20150113185343/http://hot.tokyu.co.jp/railway/hot/2004/2004_02.pdf |date=2015年1月13日 }}</ref>
* 2006年（平成16年）3月28日 - 地下車站開始使用，同時更改出入閘機及自動售票機之位置。<ref>[{{cite web |url=http://hot.tokyu.co.jp/railway/hot/0604/0604.pdf |title=存档副本 |accessdate=2017-01-22 |deadurl=yes |archiveurl=https://web.archive.org/web/20150113185142/http://hot.tokyu.co.jp/railway/hot/0604/0604.pdf |archivedate=2015-01-13 }} 『HOT ほっと TOKYU』第308号、東京急行電鉄、2006年3月20日、2015年1月14日瀏覽。」</ref>

==車站結構==
本站是[[島式月台|島式月台]]1面2線（月台有效長度可停放20公尺級車輛八輛編組，同時有月台延長段可增加兩輛編組。月台延長段平時以柵欄隔開，沒有[[月台門|月台門]]）的[[地下車站|地下車站]]<ref>{{Cite web |url=http://www.tokyu.co.jp/railway/station/info/?id=20|title=反町駅各駅情報|publisher=東急電鉄|accessdate=2015-01-14}}</ref>。本站原是[[側式月台|側式月台]]2面2線[[地面車站|地面車站]]，後於2004年（[[平成|平成]]16年）1月31日地下化。剪票口在地上1樓，月臺在地下四樓。大廳層有電扶梯與電梯通往月台層。

本站地下化後，舊站房仍持續使用至2006年（平成18年）3月28日新站房啟用為止。舊站房在同年4月中拆除改建為綠地與步道。另外，新站房旁也新建商業設施。

地面車站時代的廁所在入口處，2004年12月於一樓付費區內新設廁所。廁所前方還設有休息區。

== 站名由來 ==
開業時以所在地橫濱市青木町字'''反町'''命名<ref name="jtb68"/>。

「反町」的由來有多種說法，一說是「段町」轉字而來，也有源自一反二反的「反」（{{jpn|j=反|hg=たん}}，古代面積單位），還有說是源自此地定期在休耕地[[刀耕火耨|燒田]]的「ソリ」、「ソリマチ」<ref name="jtb68"/>。
