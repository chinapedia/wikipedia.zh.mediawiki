《'''再见巴士'''》（{{jpn|j=サヨナラバス|rm=sayonarabasu}}）是[[日本|日本]]二人组合[[柚子_(組合)|柚子]]（ゆず）的第5张单曲。<ref>{{Cite web|url = http://www.senha-yuzu.jp/discography/single/index10.php|title = ゆず Official Website | ディスコグラフィ | シングル|accessdate = 2016-02-28|author = |date = |publisher = |language = ja|deadurl = yes|archiveurl = https://web.archive.org/web/20150706144939/http://www.senha-yuzu.jp/discography/single/index10.php|archivedate = 2015-07-06}}</ref><ref>{{Cite web|url = http://www.oricon.co.jp/prof/30824/products/225295/1/|title = サヨナラバス/シャララン | ゆず | ORICON STYLE|accessdate = 2016-02-28|author = |date = |publisher = |language = ja}}</ref>1999年3月17日由其所属公司{{link-ja|Senha&Company|セーニャ・アンド・カンパニー}}发售。
