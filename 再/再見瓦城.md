{{engname|《再見瓦城》|''The Road to Mandalay''}}，是出身於[[緬甸|緬甸]]的華裔[[導演|導演]][[趙德胤|趙德胤]]執導的[[電影|電影]]，首部從獨立製片轉為商業長片電影<ref>{{Cite web |url = http://www.chinatimes.com/newspapers/20150523000679-260112 |title = 趙德胤奪創投獎 4000萬資金到位 |author = 許世穎 |date = 2015-105-23 |publisher = 自由時報 |language = zh-hant |accessdate = 2015-05-23}}</ref>，由[[柯震東|柯震東]]、[[吳可熙|吳可熙]]領銜主演。[[第53届金马奖|第53届金马奖]]共6项提名。

== 劇情 ==
描寫從緬甸偷渡泰國的陌生男女，產生情感卻受限彼此對未來不同的憧憬<ref>{{Cite web |url = http://ent.ltn.com.tw/news/breakingnews/1325382 |title = 趙德胤奪獎 坎城影展第一炮 |author = 徐卉 |date = 2015-05-23 |publisher = 中國時報 |language = zh-hant |accessdate = 2015-05-23}}</ref>。
== 演員 ==
*[[柯震東|柯震東]] 飾演 阿國：投靠泰國郊區紡織工廠擔任主管的表哥，願望是賺夠錢回臘戍開服飾店。
*[[吳可熙|吳可熙]] 飾演 蓮青：偷渡時與阿國相識，對都市生活有憧憬，希望拿到泰國護照後去台灣工作。

== 拍攝 ==
《再見瓦城》將是趙德胤從獨立製片轉為商業作品的首部長片，從[[釜山影展|釜山影展]]開始至[[坎城影展|坎城影展]]，已經在世界各影展活動拿到8個創投、劇本的獎項，當中有法德公共電視台（ARTE）的「國際ARTE獎」（Prix ARTE International），坎城影展創投單元工作坊（Atelier de la Cinefondation）法國ARTE集團創投獎金，及2014金馬創投會議「臺北文創劇作獎」、「傳翼數位與錄音獎」，也為後續投資、合作留下契機。開拍所需的資金新台幣4000萬元已到位，籌備在2015年底開拍，以及挑選劇組演員<ref>{{Cite web |url = http://www.cna.com.tw/news/amov/201505230011-1.aspx |title = 趙德胤的再見瓦城 獲歐洲電視台獎 |author = 曾依璇 |date = 2015-05-23 |publisher = 中央社 |language = zh-hant |accessdate = 2015-05-23}}</ref>。

根據《[[蘋果日報_(台灣)|蘋果日報]]》的報導，[[柯震東|柯震東]]經紀人[[柴智屏|柴智屏]]在[[第68屆坎城影展|第68屆坎城影展]]密會導演趙德胤，因此傳出他有機會演出吸毒犯。趙德胤曾讚柯震東是「不錯的演員」，柯震東日前受訪也曾表示在《[[念念|念念]]》首映上見過趙導，若有機會合作也很好<ref>{{Cite web |url = http://ent.ltn.com.tw/news/breakingnews/1339392 |title = 柯震東拚復出 演吸毒犯也OK |date = 2015-06-05 |publisher = 自由時報 |language = zh-hant |accessdate = 2015-06-10}}</ref>。
