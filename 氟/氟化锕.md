'''氟化锕'''是一种无机化合物，化学式AcF<sub>3</sub>，有强放射性。

== 制备 ==
氟化锕可由[[氢氧化锕|氢氧化锕]]和[[氟化氢|氟化氢]]在700℃反应得到<ref name=wj />：
: Ac(OH)<sub>3</sub> + 3 HF → AcF<sub>3</sub> + 3 H<sub>2</sub>O
由于氟化锕难溶于水，因此水溶液中的复分解反应也能得到它<ref name=wj />：
: Ac<sup>3+</sup> + 3 F<sup>−</sup> → AcF<sub>3</sub>↓
