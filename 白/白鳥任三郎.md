'''白鳥任三郎'''（{{lang|ja|白鳥 任三郎}}/{{lang|ja|しらとり にんざぶろう}}，Shiratori Ninzaburō），是[[日本|日本]][[漫画|漫画]]和[[动画|动画]]作品《[[名侦探柯南|名侦探柯南]]》中的[[名偵探柯南角色列表|虚构角色]]。

命名來自於日本推理小說的著名刑警'''白鳥'''完市及推理電視劇《'''紳士刑警'''》中的[[古畑任三郎|古畑'''任三郎''']]。

== 人物 ==
警视厅搜查一课強行犯搜查三系的[[警部|警部]]，是[[目暮十三|目暮十三]]的同事，[[高木涉_(名偵探柯南)|高木涉]]、[[佐藤美和子|佐藤美和子]]的上司；初戀情人是任教於帝丹小學1年B班的'''小林澄子'''。

家境相當富裕，管家是'''鸭井五十吉'''；在[[輕井澤|輕井澤]]擁有大型的別墅，地下室為儲存葡萄酒的酒窖<ref group="參" name="OVA2">OVA2《16個嫌疑犯》</ref>。

== 形象 ==
個性較為沉默寡言；雖然不善於槍法<ref group="參">M2《[[名偵探柯南：第十四號獵物|第十四號獵物]]》</ref>，但对於[[建筑|建筑]]<ref group="參" name="M1"/>和[[葡萄酒|葡萄酒]]<ref group="參" name="OVA2"/>有很深的研究。

== 情感 ==
小時候原先是想成為一名優秀的律師，但受到小林的感染而立志當上警官。曾將長相極為相似的[[佐藤美和子|佐藤]]誤認為小林而不斷追求之，也曾與其他男性警員成立「'''佐藤美和子防線'''」，並擔任最高負責人<ref group="參">TV 358-359《本廳刑事戀愛物語5》</ref>；卻也因此招致小林的誤會，所幸之後得以冰釋前嫌而成為一對戀人<ref group="參">TV 568-569《白鳥警官 櫻花的回憶》、583《小林老師的戀情》、584《白鳥警官的失戀》、585《超越時空的櫻花之戀》</ref>。
