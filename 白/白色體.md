[[File:Potato_-_Amyloplasts.jpg|thumb]][[塊莖|塊莖]]細胞內的[[澱粉體|澱粉體]]（屬於白色體的一種）]]

'''白色體'''（Leucoplast），泛指一種存在[[植物|植物]]細胞中，不含[[色素|色素]]的[[色素體|色素體]]，功能為儲存[[養分|養分]]或執行[[生化|生化]][[合成|合成]]作用，儲存養分的白色體會依儲存的養分類別而特化成[[澱粉體|澱粉體]]、[[油粒體|油粒體]]及[[蛋白質體|蛋白質體]]。廣義的白色體也包括未經照光而[[黃化葉綠體|退化的葉綠體]]（Etioplast）。

== 特徵 ==
{{File|zh-hant=Plastids_types_zh.svg|zh-hans=Plastids_types_zhs.svg|thumb|220px|desc=各種[[色素體|質粒體]]的關係圖}}

跟其他的質粒體比較下，白色體缺乏[[色素|色素]]並呈現無色，大部分存在於植物的[[根|根部]]、[[莖|莖部]]等陽光照不到的地方，或存在於一些癒合組織及[[分生組織|分生組織]]，在暗室生長時也會存在於[[表皮細胞|表皮細胞]]。外型通常呈不規則，可如[[變形蟲|變形蟲]]似的改變外型。白色體的大小通常比[[葉綠體|葉綠體]]小得多<ref name="chi">{{cite web|url=http://ap6.pccu.edu.tw/Encyclopedia/data.asp?id=2119&forepage=1|title=中華百科全書 白色體|author=林明煊|publisher=中華百科全書|language=Zh-tw|access-date=2011-01-04|archive-url=https://web.archive.org/web/20160304194203/http://ap6.pccu.edu.tw/Encyclopedia/data.asp?id=2119&forepage=1#|archive-date=2016-03-04|dead-url=no}}</ref>。

== 功用 ==
白色體的功能主要是儲存養分，有時也可進行一些[[生化|生化]][[有机合成|合成]]反應。

=== 儲存物質 ===
白色體常特化為儲存[[營養素|營養素]]的胞器，可儲存[[澱粉|澱粉]]、[[脂肪|脂肪]]、[[蛋白質|蛋白質]]和[[鞣质|鞣质]]等，並可依其儲存的物質分為[[澱粉體|澱粉體]]、[[油粒體|油粒體]]、[[蛋白質體|蛋白質體]]和[[单宁体|单宁体]]<ref name="chi" />。

==== 澱粉體 ====
{{main|澱粉體}}
澱粉體（amyloplast）是一種白色體，負責將多餘的[[葡萄糖|葡萄糖]]合成[[澱粉|澱粉]]的形式儲存，留到需要時使用。常見於[[水果|水果]]及地下[[塊根|塊根]]、[[塊莖|塊莖]]等儲存部位，如[[番薯|番薯]]、[[馬鈴薯|馬鈴薯]]和[[芋頭|芋頭]]等。除了儲存澱粉之外，澱粉體還被認為和植物[[根|根]]部的[[向地性|向地性]]有關，當澱粉體感受[[重力|重力]]變化而在[[根冠|根冠]]（root cap）中的平衡細胞（statocyte）沉澱時，會帶動[[肌動蛋白|肌動蛋白]]的活動，刺激其[[細胞膜|細胞膜]]的[[離子通道|離子通道]]，這些重力產生的訊號最後造成[[生長素|生長素]]（IAA）分部的不對稱性，使根部產生向地性。而[[莖|莖]]則出現相反的情形，因為生長素分部不均而產生[[負向地性|負向地性]]<ref>{{cite web|url=http://highscope.ch.ntu.edu.tw/wordpress/?p=1079|title=植物根的向地性（Root Gravitropism）|author=劉玉山|publisher=國科會高瞻計畫資源平台 中學教材示範網頁|date=2009-07-10|language=Zh-tw}}</ref>。

==== 油粒體 ====
{{main|油粒體}}
油粒體（elaioplasts）也是一種白色體，特化為儲存[[脂質|脂質]]的胞器，脂質被儲存在油粒體內的許多[[色素體|色素體]]小球（plastoglobuli）中，色素體小球的主要成分為油滴，也含有少量色素體[[醌|醌類]]（plastoquinone）。油粒體在[[單子葉植物|單子葉植物]]和[[苔蘚植物|苔蘚植物]]較為普遍<ref>{{cite web|url=http://www.encyclo.co.uk/define/Plastoglobuli|title=Encycopedia Plastoglobuli|language=en|access-date=2011-01-06|archive-url=https://web.archive.org/web/20120211102949/http://www.encyclo.co.uk/define/plastoglobuli#|archive-date=2012-02-11|dead-url=no}}</ref><ref>{{cite web|url=http://www.mondofacto.com/facts/dictionary?plastoglobuli|title=mondofacto Plastoglobuli|language=en|access-date=2011-01-06|archive-url=https://web.archive.org/web/20111106121941/http://mondofacto.com/facts/dictionary?plastoglobuli#|archive-date=2011-11-06|dead-url=no}}</ref>。

==== 蛋白質體 ====
{{main|蛋白質體}}
蛋白質體（proteinoplasts）也被認為是一種白色體，特化為儲存[[蛋白質|蛋白質]]的胞器。其中蛋白質在其中可能以[[結晶|結晶]]的形式儲存。蛋白質體常存在於植物的[[種子|種子]]中，如[[巴西堅果|巴西堅果]]、[[花生|花生]]等。不同於澱粉體、油粒體的是，我們對於蛋白質體的實際型態及機制還不甚了解<ref name="wise">{{cite book|last=Wise|first=Robert|title=The Diversity of Plastid Form and Function|publisher=Springer|date=2007|chapter=1|url=http://www.springerlink.com/index/qp032630631337u6.pdf}}</ref><ref>{{cite web|url=http://www.mondofacto.com/facts/dictionary?proteinoplasts|title=mondofacto proteinoplasts|language=en|access-date=2011-01-07|archive-url=https://web.archive.org/web/20160304191010/http://www.mondofacto.com/facts/dictionary?proteinoplasts#|archive-date=2016-03-04|dead-url=no}}</ref>。

==== 单宁体 ====
{{main|单宁体}}
单宁体（tannosome）是一种制造并存储[[鞣质|鞣质]]（又称单宁）和[[多酚|多酚]]的白色体。存储的鞣质和多酚可以作为化学物质帮助植物驱赶[[食草动物|食草动物]]，如[[蚜虫|蚜虫]]。鞣质还可以保护植物免受[[病原体|病原体]]和[[紫外线|紫外线]]的危害。<ref>{{Cite web |url = http://www.the-scientist.com/?articles.view/articleNo/37597/title/New-Organelle--The-Tannosome/ |title = New Organelle: The Tannosome |author = Kerry Grens |date = 2013-09-23 |publisher = The Scientist Magazine® |accessdate = 2014-08-22 |archive-url = https://web.archive.org/web/20140307150925/http://www.the-scientist.com/?articles.view%2FarticleNo%2F37597%2Ftitle%2FNew-Organelle--The-Tannosome%2F# |archive-date = 2014-03-07 |dead-url = no }}</ref>

=== 生化合成 ===
除了儲存養分的功能之外，白色體有時還有進行[[生化合成|生化合成]]的功能，可合成[[脂肪酸|脂肪酸]]、[[胺基酸|胺基酸]]和一些[[四吡咯|四吡咯]]化合物（Tetrapyrroles）如[[血紅素|血紅素]]。

== 轉換 ==
{{main|黃化葉綠體}}
若植物生長在缺乏光線的暗室中，其[[葉綠體|葉綠體]]及葉綠體的前導物原質粒體（proplastids）將轉換成黃化葉綠體（Etioplast）<ref>{{cite web|url=http://www.cella.cn/book/07/02.htm|title=細胞生物學教程|language=zh-hans|access-date=2011-01-07|archive-url=https://web.archive.org/web/20120224103825/http://www.cella.cn/book/07/02.htm#|archive-date=2012-02-24|dead-url=no}}</ref>
，它們缺乏活化的[[色素|色素]]，會使葉片呈現淡黃色而非綠色，由於缺乏色素，黃化葉綠體也能夠被認為是廣義的白色體。在數分鐘的光線照射後，黃化葉綠體會逐漸轉化為葉綠體<ref name="wise"></ref>。

黃化葉綠體沒有[[類囊體|類囊體]]，只有原片層體（prolamellar body），為管狀的[[膜|膜]]聚合而成，主要構造為四條管狀的膜聚合於同一個三角型接點<ref>原片層體（prolamellar body）的主要構造如此圖：{{cite web |url=http://www.rsbs.anu.edu.au/profiles/Brian_Gunning/Web%20PCB/Images/Prolamellar%20body.htm |title=存档副本 |accessdate=2011-01-07 |deadurl=yes |archiveurl=https://web.archive.org/web/20090828023636/http://www.rsbs.anu.edu.au/profiles/Brian_Gunning/Web%20PCB/Images/Prolamellar%20body.htm |archivedate=2009-08-28 }}</ref>，管中含有[[葉綠素|葉綠素]]的前導物質。

在接受照光之後，黃化葉綠體將在[[細胞分裂素|細胞分裂素]]（CK）的作用下轉變成正常葉綠體，類囊體也會在此時由原片層體轉變而成<ref>{{cite web|url=http://phylogenomics.berkeley.edu/mcp.pdf|title=Proteome Analysis of the Rice Etioplast|author=Anne von Zychlinski‡§¶|language=en|deadurl=yes|archiveurl=https://web.archive.org/web/20100623210408/http://phylogenomics.berkeley.edu/mcp.pdf|archivedate=2010-06-23}}</ref>。
