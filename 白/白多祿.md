'''聖白多禄'''，亦译为'''聖白伯铎'''、'''聖桑实'''（{{lang-es|'''San Pedro Sans y Jordá'''}}，{{bd|1680年|9月22日|1747年|5月26日}}）是18世紀到[[中國|中國]][[福建|福建]]傳教的[[天主教|天主教]][[傳教士|傳教士]]。他生於[[西班牙|西班牙]][[塔拉戈纳|塔拉戈纳]]，为[[罗马天主教|罗马天主教]][[多明我会|多明我会]]会士，1732年到1747年间担任天主教福建教区主教，1747年因传播天主教而在[[福州|福州]]被杀，2000年10月1日獲[[教宗|教宗]][[若望保祿二世|若望保祿二世]]冊封為聖人，為[[中華殉道聖人|中華殉道聖人]]之一。

== 早年 ==
1680年9月22日，白多禄生于西班牙塔拉戈纳省[[多笃撒教区|多笃撒教区]]的[[阿斯科|阿斯科镇]]，出生时母亲和孪生妹妹俱亡，次日在本村教堂领洗，取名伯鐸·若瑟·安德肋。白多祿的舅父何弥额神父是[[萊里達|萊里達]]當地教區的[[司鐸|司鐸]]，白多祿幼年時就被其父托付給舅父進行教育。17岁時，白多祿进入莱里达城的多明我会修道院學習，1698年7月6日，18歲的白多祿發愿後成為修士，并取了“[[殉道者伯鐸|殉道者伯鐸]]”的會名。6年后的1704年9月24日，他又晋升為[[神甫|神甫]]。[[1707|1707]]年的[[西班牙王位繼承戰爭|西班牙王位繼承戰爭]]中，萊里達城受到攻擊，白多祿參與了救治傷員的工作。1709年，他被派往[[薩拉戈薩|薩拉戈薩]]做教學工作，并成為了當地[[聖母玫瑰會|聖母玫瑰會]]的指導司鐸。1712年秋天，他加入由多明我会玫瑰会省42名神甫组成的传教团，离开西班牙[[加的斯|加的斯]]港先赴[[墨西哥|墨西哥]]，再由墨西哥[[阿卡普爾科|阿卡普爾科]]港出發前往[[菲律宾|菲律宾]]，經過全程11個月的旅行，終于到達[[馬尼拉|馬尼拉]]。<ref name="《流血记》">郑天祥，《流血记》，澳门慈幼印书馆，1950</ref>

== 入闽传教 ==
當時，多明我會已經成為羅馬天主教在華傳教的主力，由于多明我會反對[[祭孔|祭孔]][[祭祖|祭祖]]而導致了[[禮儀之爭|禮儀之爭]]，被[[清朝|清朝]]禁止傳教，很多傳教士遭羈押或驅逐，就在這種背景下，白多祿還是決定到中國傳教。經過在菲律賓1年零8個月的等待後，1715年6月22日，白多禄與[[华雅敬|华雅敬]]神父一起由馬尼拉出發前往中国，6月29日到達[[廣東|廣東]]，為躲避口岸的官兵，白多祿又轉往[[廈門|廈門]]登陸，但仍被中國士兵抓獲，不過由于負責的軍官不愿追究，白多祿被釋放。隨后，他经[[泉州|泉州]]、[[兴化|兴化]]一线北上，前往當時福建天主教的中心地區[[閩東|閩東]][[福宁州|福宁州]][[福安县|福安县]]傳教。這時他開始使用自己近音的[[漢語|漢語]]名字“白多祿”，來取代[[西班牙文|西班牙文]]名伯铎·桑实。<ref name="《流血记》"/><ref name="福安市志">福建省福安市地方志编纂委员会，福安市志，方志出版社，1999年</ref>

初到閩東，白多祿奔波于福安的各個鄉村間傳教，成立了圣多明我第三会，并使600多當地的文人士紳皈依天主教，還組織了公教仕绅协进会，這些在地方上頗具影響的人物大大幫助了白多祿的傳教行動，曾經有士兵進入鄉村民居內搜捕傳教士，結果當地信教的士紳到縣衙控告士兵擾民，還獲得勝訴。同時，白多祿在當地成立了聖母玫瑰會，勸教友勤念[[玫瑰經|玫瑰經]]。此外，他還在[[穆阳|穆阳]]建立了玫瑰圣母堂，在康家板（康厝）建立了天主圣三堂。在此期間，他的逐漸適應并接受了中國的生活方式、學習了當地的文化風俗，爭取到許多教友的支持。在被選為福建传教区的区会长之後，白多祿苦于傳教人手不夠，請马尼拉会省加派傳教士赴閩，但從馬尼拉出發到澳門的4位西班牙籍傳教士被[[葡萄牙人|葡萄牙人]]所阻撓而無法進入中國境內，只得在一年後喬裝成漁民從[[廣州|廣州]]登陸再輾轉來福建。不久，华雅敬神父被調至福建任区会长，白多祿就南下到[[閩南地區|閩南地區]]傳教。<ref name="《流血记》"/>

== 禁教高峰時期 ==
1723年（[[雍正|雍正]]元年）[[雍正皇帝|雍正皇帝]]上台后，[[清政府|清政府]]加大了禁教的力度，严厉禁止天主教活动，并悬赏传教士的人头，<ref name="Pedro Sanz y Jordá, Santo">[http://es.catholic.net/santoral/articulo.php?id=9797 Pedro Sanz y Jordá, Santo], Catholic.net</ref>當時白多禄仍在[[漳州|漳州]]，只得隱居起來，晝伏夜出，在晚上去見教友。這種情形持續了6年。1724年，清政府下令將全國的傳教士都集中於[[广州|广州]]，最后驅逐到[[澳门|澳门]]，不許回中國傳教。福建的[[天主教堂|天主教堂]]被政府没收充公，教徒也被勒令放弃信仰。這時，福建的华雅敬会长決定讓一部分傳教士去廣州，以迷惑官府，剩下的傳教士則留下并隱藏起來。如此，福安一带的天主教仍然在地下秘密传播，许多传教士躲到[[宁德县|宁德县]]岚口村，1726年时福安还有18座教堂，教徒超过一万名。不久，教廷使者來到中國和清政府商談這次危機，儘管沒有解決問題，但局勢也緩和下來。1728年1月29日，白多禄被委任为福建辅理代牧。1729年冬，因为禁教局势再度紧张，白多禄离开[[漳州府|漳州府]][[龙溪县|龙溪县]][[石码镇|石码镇]]，到广州避难。1730年2月24日，他在廣州被[[祝圣|祝圣]]为主教。1732年1月3日，他又担任福建代牧，同年8月，他隨同其他傳教士一起被驅逐到澳門。<ref name="福安市志"/><ref name="白多禄">[http://www.chinesemartyrs.ca/Saints/ViewSaints.aspx?id=89 白多禄]{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}，天主教中華殉道聖人堂</ref><ref name="宁德地区志">福建省宁德地区地方志编纂委员会，宁德地区志，方志出版社，1998年</ref>

1735年，[[雍正帝|雍正帝]]去世，白多祿于是考慮重返福建。經過幾度嘗試，1738年6月，他和施方济神父成功離開澳门，乘船潛回福建，經過1個月才返回福安，隐藏在信徒的家中秘密传教，負責闽、浙、赣三省教务，并巡回各縣堂口。此時的白多祿已經熟練掌握了[[閩東語|閩東語]]，能夠流利的向各階層人士[[布道|布道]]。<ref name="《流血记》"/>

== 殉教 ==
1746年（[[乾隆|乾隆]]十一年）四月，福宁[[知府|知府]][[董启祚|董启祚]]、福宁镇臣[[李有用|李有用]]侦查出福安县穆洋村一带有传教士活动，而且教徒还利用暗室、地窖、重墙复壁来隐匿传教士，于是便报告福建[[巡抚|巡抚]][[周学健|周学健]]，周学健格外重视禁教事宜，立即派兵将费若望神父和十几个教徒逮捕，并上奏[[乾隆帝|乾隆帝]]，而乾隆帝则赞许逮捕教士教徒一事。于是周学健加大了搜捕力度，1746年6月28日，德方济和施方济神父被捕，1746年7月2日，白多禄在穆阳被搜捕入狱。<ref name="Pedro Sanz y Jordá, Santo"/>经过多方审讯，周学健了解到福安的天主教传播甚广，城镇乡间从士绅到庶民，非天主教徒的竟然只占少数，这种情况让他十分吃惊。于是周学健再次上奏关于福建天主教秘密传播之事，并希望朝廷严加惩处。乾隆帝对此情况也十分震惊和重视，并在全国范围内再次大规模禁教。这次在福安被逮捕的教徒多被治罪，而周学健特别按首犯责任追究白多禄，1746年九月十二日，周学健专折上奏要求判处白多禄斩首，乾隆帝初批“未免言之过当，然照律定拟，自所应当”，而交由三法司复议后最终还是下诏处斩。在经受狱中长期的折磨之后，白多禄在1747年5月26日（[[乾隆|乾隆]]十二年），被斩首于福州西门外。时年66岁。<ref name="《流血记》"/>在他殉道之后一年的1748年，另外四名先后被捕的西班牙神父[[施方濟|施方濟]]、[[華雅敬|華雅敬]]、[[德方濟|德方濟]]，及[[費若望|費若望]]，也被处决。<ref name="从清代档案看雍正乾隆时期福建地区的传教与禁教">黄爱平，从清代档案看雍正乾隆时期福建地区的传教与禁教，中国人民大学清史研究所，中華文史網</ref>

== 影响和评价 ==
清代中期时，政府认为白多禄所代表的[[天主教会|天主教会]]居心叵测、扰乱中国的礼教、威胁政府统治，甚至认为教会用幻术使女性终身不思匹偶而成为[[修女|修女]]，因而认定白多禄传播异端，危害巨大。<ref name="从清代档案看雍正乾隆时期福建地区的传教与禁教"/>而[[中国共产党|中国共产党]]方面认为白多禄属于“作為近代[[殖民主義|殖民主義]]、[[帝國主義|帝國主義]]侵略中國工具的外國傳教士”一类的人物，并对[[梵蒂冈|梵蒂冈]]在2000年为包括白多禄在内的120名在中国殉教的天主教徒封圣这一行为表示强烈不满。<ref name="揭開所謂“聖人”的面目">史岩，揭開所謂“聖人”的面目，《人民日報》（2000年10月03日第二版）</ref>但福建的天主教徒对白多禄评价甚高，在闽东有很多关于白多禄为善的轶事和白主教的[[圣迹|圣迹]]，白多禄匿居的山洞也成为天主教徒的[[朝圣地|朝圣地]]。<ref name="充满传说的一方教会">缪增荣，[http://www.chinacatholic.org/XindeNews.asp?Id=5263 充满传说的一方教会]{{dead link|date=July 2015}}，信德报（第268期）</ref>教宗[[本篤十四世|本篤十四世]]在1752年1月24日的樞機公開會議中，高度表揚了包括白多禄在内的五位殉教的[[多明我会|多明我会]]傳教士。1757年，天主教[[玫瑰會|玫瑰會]]省開始派傳教士到[[福州府|福州府]]及[[福宁府|福宁府]]，用了十年时间搜集白多禄等人的資料，調查證實他們有資格承當[[真福者|真福者]]。1893年5月14日，白多禄獲教宗[[良十三世|良十三世]]晋封為[[真福|真福]]，2000年10月1日，教宗[[若望保祿二世|若望保祿二世]]冊封白多禄為聖人，白多禄成為[[中華殉道聖人|中華殉道聖人]]之一。<ref name="《流血记》"/><ref name="">[http://www.catholic.org.tw/dominicanfamily/saints_china.htm#%E8%81%96%E7%99%BD%E5%A4%9A%E7%A5%BF%E4%B8%BB%E6%95%99 聖白多祿主教 Pedro Martir Sanz Jorda 1680-1747] {{webarchive|url=https://web.archive.org/web/20081203161040/http://www.catholic.org.tw/dominicanfamily/saints_china.htm |date=2008-12-03 }}，道明中华殉教圣人，天主教会台湾地区主教团网站</ref>
