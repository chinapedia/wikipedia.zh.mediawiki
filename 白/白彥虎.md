'''白彥虎'''（[[東干語|東干語]]：'''{{lang|ru|Биянхў}}'''；{{bd|1830年|2月8日|1882年|7月26日|catIdx=Bai, Yanhu}}），[[小名|小名]]'''白素'''，[[經名|經名]]'''穆罕默德·阿尤布'''，[[陕西人|陕西人]]，具体籍贯仍无定论{{NoteTag|白彥虎具体[[籍贯|籍贯]]有多种说法，一说为西安本地人，一说为[[陝西|陝西]][[泾陽|泾陽]]人，一说为陕西[[大荔|大荔]]人；其中泾阳说又有泾阳县瑞安乡包方里焦村（今陕西省泾阳县崇文镇焦村）人的说法。}}，是领导[[同治陕甘回变|同治陕甘回变]]的主要人物，為回民軍[[十八大營元帥|十八大營元帥]]之一，從1862年至1877年與[[清軍|清軍]]作戰。后被[[左宗棠|左宗棠]]打败，失敗後带领2000多回民逃至[[中亞|中亞]]<ref name=xx /> ，仍不断骚扰中国西北边疆地区<ref>魏光焘，《戡定新疆记》：“戕官弁，杀行客，掠台马”。</ref>。他被中亞的[[東干族|東干族]]人称为“'''虎大人'''”，尊為「'''东干族之父'''」。

== 生平 ==
=== 陝甘回乱 ===
[[File:Dungan_Revolt_Map.png|thumb]]
由于清政府常年民族歧视压迫政策，导致社会、民族矛盾尖锐，[[1862年|1862年]]，趁清政府平定地方叛乱、镇压各地农民、各民族起义；西北地方空虚之机，陝西回民發動[[同治陕甘回乱|同治陕甘回乱]]。不识字，且不是[[阿訇|阿訇]]的白彥虎於[[1866年|1866年]]被推舉為回民十八大營元帥之一，但骁勇善战，{{fact|甘肃地区33次可考证的战役中，有29次有白的参与。}}（以後十八大元帥裡有七位戰死：[[馬正和|馬正和]]、[[楊文治|楊文治]]、[[張四明|張四明]]、[[王明章|王明章]]、[[沙學德|沙學德]]、[[赫明堂|赫明堂]]、[[馮君幅|馮君幅]]；其他投降的有九位：[[禹得彦|禹得彦]]、[[崔偉|崔偉]]、[[余彦禄|余彦禄]]、[[藍明泰|藍明泰]]、[[閻興泰|閻興泰]]、[[鄒保和|鄒保和]]、[[陳林|陳林]]、[[馬生彦|馬生彦]]、[[華大才|華大才]]）。1868年3月，十八營再整編為四大營。

左宗棠挥军西进，十余万回军且战且走。

=== 逃往中亞 ===
1872年白彥虎在[[西寧|西寧]]被清軍擊敗後率軍轉向[[河西|河西]]，之後再逃往[[新疆|新疆]]投靠[[阿古柏|阿古柏]]。1877年，[[左宗棠|左宗棠]]率大軍擊溃阿古柏，白彥虎回民軍被左追擊，于是率殘部2000多人<ref name=xx>{{cite book|title=《续修陕西省通志稿》卷一百七十八|pages=第32页}}</ref>。越過[[恰克馬克山口|恰克馬克山口]]逃入中亞[[楚河|楚河]]流域，後于今[[吉尔吉斯斯坦|吉尔吉斯斯坦]][[比什凱克|比什凱克]]因[[肺炎|肺炎]]病亡。

== 歷史定位 ==
從歷史的角度來看，對於白彥虎的判斷大約還有幾點爭議：
# 一則白彥虎本人曾加入[[清朝|清朝]]軍隊有兩年之久，也參加過征討[[太平天國|太平天國]]之役，角色定位顯得尷尬。
# 再來，身為統帥的白彥虎為了使殘餘回軍逃到[[中亞|中亞]]，滅親的手段是否過於激烈。以至到今天中亞東干老人還傳有「白彥虎殺嫂」之說。
# 三則於西寧战役的慘敗中，白彦虎与[[禹得彦|禹得彦]][[阿訇|阿訇]]发生過劇烈矛盾的争辩，以至另有一批人随禹得彦阿訇降清。中國有人认为他与[[阿古柏|阿古柏]]勾結，从事分裂国家行动，要受抨击。

== 评价 ==
* [[左宗棠|左宗棠]]：贼中所称虎元帅即白彦虎，盖贼中十八酋之一，凶悍素著者也<ref>《左文襄公全集·卷三十·奏稿四·<截剿逆回大胜折>》。</ref>。
* [[易孔昭|易孔昭]] ：白彦虎，狡黠异常<ref>《平定关陇纪略》。</ref>。
* 《平回志》：唯白彦虎党最狡黠。
* [[胡绳|胡绳]]：白彦虎原是陕西回民起义的一个头子，起义失败后逃到甘肃，又在同治十二年（1873年）出关到了新疆北部。他率领手下数千兵力，投靠了阿古柏，成为阿古柏的得力打手。<ref>胡绳．《从鸦片战争到五四运动》：人民出版社，1981年：第389页。</ref>。
* [[王震|王震]]：阿古柏是从新疆外部打进来的，其实他是沙俄、英帝的走狗，左公带兵出关，消灭阿古柏、白彦虎，收复失地，得到了新疆各族人民的支持，这是抗御外侮，是值得赞扬的。<ref name="徐文钦2015">{{cite book |author=徐文钦 |title=跟高层领导学读书 |url = https://books.google.com/books?id=elP8CAAAQBAJ&pg=PT220 |date = 2015-05-07 |publisher=南文博雅 |page = 220 |id = GGKEY:CKPS58DLTEQ }}</ref><ref>{{cite web |url = http://news.ifeng.com/history/1/renwu/200809/0908_2665_770257_8.shtml |title=王震将军评左宗棠：屠夫还是功臣？(图) |date=2008年9月8日 }}</ref>
