'''白龍珠'''( 1906年 - 1970年2月4日 )，原名：麥龍珠，[[粵劇|粵劇]]行裡人稱他為「阿蛇」，[[籍貫|籍貫]]：[[廣東省|廣東省]][[南海縣|南海縣]]，15歲少年時代已從事[[粵劇|粵劇]]，他是[[薛覺先|薛覺先]]的首徒，最初擔任小生；[[第二次世界大戰|第二次世界大戰]]期間，白龍珠跟隨[[薛覺先|薛覺先]]到[[廣西省|廣西省]]的[[桂林|桂林]]及[[柳州|柳州]]演出，有一晚表演《[[四大美人|四大美人]]》，原來演[[伍員|伍員]]的武生醉酒失場，被[[薛覺先|薛覺先]]臨時拉夫出臺，自此被[[薛覺先|薛覺先]] 要求轉型作武生<ref>1960年5月23日([[星期一|星期一]]) ，[[香港|香港]]《[[華僑日報|華僑日報]]》，第二張，第七版。</ref><ref>1970年2月8日(星期日) ，[[香港|香港]]《[[華僑日報|華僑日報]]》，第三張，第四頁。</ref>。白龍珠亦是[[香港|香港]][[電影|電影]][[演員|演員]]，自銀幕[[處男|處男]]《歌唱新美人計》(1952年)作至息影作《[[樊梨花|樊梨花]]》(1968年，飾演薛仁貴)，一生參演約90齣[[電影|電影]]。

== 社交圈子 ==
* 1964年7月6日([[星期一|星期一]])，在[[九龍|九龍]][[油麻地|油麻地]][[志和街|志和街]]的「百老匯酒店」(現在的「聖地牙哥酒店(新館)」)的502號房，白龍珠與其它[[梨園|梨園]]子弟[[鄧偉凡|鄧偉凡]]、[[何驚凡|何驚凡]]、[[蕭仲坤|蕭仲坤]]、[[林蛟|林蛟]]、[[蘇少棠|蘇少棠]]、[[陳錦棠|陳錦棠]]、[[少新權|少新權]]及[[羅家權|羅家權]]等組成長期性的“先生俱樂部”，性質在聯絡[[感情|感情]]與守望相助，談古說今，研傳統[[藝術|藝術]]，及搓[[麻將|麻將]]<ref>1964年7月7日(星期二) ，[[香港|香港]]《[[華僑日報|華僑日報]]》，第六張，第二頁。</ref>。

* 白龍珠習慣每天下午在「戲人茶座」喝下午茶，然後在[[九龍|九龍]][[油麻地|油麻地]][[彌敦道|彌敦道]]363號的「新新酒店」(現在的「恒成大廈 Hang Shing Building」)六樓與[[朋友|朋友]]搓八圈[[麻將|麻將]]<ref>1967年6月14日([[星期三|星期三]]) ，[[香港|香港]]《[[華僑日報|華僑日報]]》，第四張，第一頁。</ref>。

== 桃色事件 ==
1951年與[[情婦|情婦]][[同居|同居]]，1955年3月被30歲[[情婦|情婦]]胡寶珠起訴「故意忽略供給」，由於白龍珠尚有48歲的髮妻關玉秋，白龍珠被判無罪，然而自願給予[[情婦|情婦]]胡寶珠一次性港幣300元「撫慰金」(掟煲費)<ref>1955年3月17日(星期四) ，[[香港|香港]]《[[華僑日報|華僑日報]]》，第一張，第四頁。</ref>。

== 麥廬 ==
* 1960年6月，白龍珠租住[[油麻地|油麻地]][[炮台街|炮台街]]面向「油麻地新街市」的3房樓宇，自己住在頭房，其餘兩房分租予別人<ref>1960年6月16日(星期四) ，[[香港|香港]]《[[大公報|大公報]]》，第二張，第六版。</ref>。

* 1964年6月購入[[新界|新界]][[青山|青山]][[屯門|屯門]]，近[[后角天后廟|后角天后廟]]附近的1,500多平方英呎土地，建築以『麥廬』命名的[[別墅|別墅]]，飼養[[白鴿|白鴿]]及[[蜜蜂|蜜蜂]]，廣植果樹<ref>1964年6月16日(星期二) ，[[香港|香港]]《[[華僑日報|華僑日報]]》，第八張，第一頁。</ref>。

== 掛靴息影 ==
* 從15歲踏入戲行至63歲(1969年)已做八任細班(粵劇行規每6年等一任細班)，總共演出48年，接受兒媳的建議「掛靴」(息影退休)，把戲服割讓別人，以表決心<ref>1969年2月6日(星期四) ，[[香港|香港]]《[[華僑日報|華僑日報]]》，第四張，第四頁。</ref>。

== 逝世 ==
* 1970年2月4日([[星期三|星期三]])晚上9時，在[[九龍|九龍]][[廣華醫院|廣華醫院]]病逝，享齡64歲；1970年2月8日(星期日)下午2時，在[[九龍殯儀館|九龍殯儀館]]舉殯，[[遺體|遺體]]卜葬[[九龍|九龍]][[長沙灣天主教墳場|長沙灣天主教墳場]]<ref>1970年2月8日(星期日) ，[[香港|香港]]《[[華僑日報|華僑日報]]》，第三張，第四頁。</ref>。
