'''白栎'''（[[学名|学名]]：{{lang|la|''Quercus fabri''}}）为[[壳斗科|壳斗科]][[栎属|栎属]]的植物。

== 形态 ==
[[落叶乔木|落叶乔木]]，高达20米。小枝密生粗毛；倒卵形叶互生，每边具有6-10个波状钝缺齿，下面有灰色粗毛，细脉明显；长卵形[[坚果|坚果]]，生于杯状壳斗中，壳斗外被细小鳞片。

== 分布 ==
分布于[[朝鲜|朝鲜]]以及[[中国大陆|中国大陆]]的[[四川|四川]]、[[贵州|贵州]]、[[广东|广东]]、[[陕西|陕西]]、[[广西|广西]]、[[河南|河南]]、[[湖北|湖北]]、[[云南|云南]]、[[安徽|安徽]]、[[福建|福建]]、[[湖南|湖南]]、[[浙江|浙江]]、[[江西|江西]]、[[江苏|江苏]]等地，生长于海拔50米至1,900米的地区，一般生于山地杂木林中和丘陵。

== 别名及异名 ==
别名：小白栎（云南植物志）

异名：
* Quercus fabri Hance var. multinervis C. J. Qi
