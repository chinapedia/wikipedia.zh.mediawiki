'''白象勳章'''（{{lang-th|เครื่องราชอิสริยาภรณ์อันเป็นที่เชิดชูยิ่งช้างเผือก}}），[[泰國|泰國]]的一種[[勳章|勳章]]，1861年由[[拉瑪四世|拉瑪四世]]創始，為外籍人士可獲得的最高榮譽。

== 等級 ==
本[[勳章|勳章]]分為以下八個等級
* [[File:Order_of_the_White_Elephant_-_Special_Class_(Thailand)_ribbon.png|File:Order of the White Elephant - Special Class (Thailand) ribbon.png]] 大授騎士章 （特等）
:: {{lang-th|มหาปรมาภรณ์ช้างเผือก}} (''má-hăa bpon maa pon cháang-pèuak'')
:: 簡稱：{{lang|th|ม.ป.ช.}}
* [[File:Order_of_the_White_Elephant_-_1st_Class_(Thailand)_ribbon.png|File:Order of the White Elephant - 1st Class (Thailand) ribbon.png]] 大十字騎士章 （一等）
:: {{lang-th|ประถมาภรณ์ช้างเผือก}} (''bprà-tà-măa-pon cháang-pèuak'')
:: 簡稱：{{lang|th|ป.ช.}}
* [[File:Order_of_the_White_Elephant_-_2nd_Class_(Thailand)_ribbon.png|File:Order of the White Elephant - 2nd Class (Thailand) ribbon.png]] 指揮者騎士章 （二等）
:: {{lang-th|ทวีติยาภรณ์ช้างเผือก}} (''tá-wee dtì yaa pon cháang-pèuak'')
:: 簡稱：{{lang|th|ท.ช.}}
* [[File:Order_of_the_White_Elephant_-_3rd_Class_(Thailand)_ribbon.png|File:Order of the White Elephant - 3rd Class (Thailand) ribbon.png]] 指揮者章 （三等）
:: {{lang-th|ตริตาภรณ์ช้างเผือก}} (''dtrì dtaa pon cháang-pèuak'')
:: 簡稱：{{lang|th|ต.ช.}}
* [[File:Order_of_the_White_Elephant_-_4th_Class_(Thailand)_ribbon.png|File:Order of the White Elephant - 4th Class (Thailand) ribbon.png]] 從者章（四等）
:: {{lang-th|จัตุรถาภรณ์ช้างเผือก}} (''jat-dtoo rá-tăa-pon cháang-pèuak'')
:: 簡稱：{{lang|th|จ.ช.}}
* [[File:Order_of_the_White_Elephant_-_5th_Class_(Thailand)_ribbon.png|File:Order of the White Elephant - 5th Class (Thailand) ribbon.png]] 成員章（五等）
:: {{lang-th|เบญจมาภรณ์ช้างเผือก}} (''ben-jà maa pon cháang-pèuak'')
:: 簡稱：{{lang|th|บ.ช.}}
* [[File:Order_of_the_White_Elephant_-_Medal_(Thailand)_ribbon.png|File:Order of the White Elephant - Medal (Thailand) ribbon.png]] 金質章（六等）
:: {{lang-th|เหรียญทองช้างเผือก}} (''rĭan tong cháang-pèuak'')
:: 簡稱：{{lang|th|ร.ท.ช.}}
* [[File:Order_of_the_White_Elephant_-_Medal_(Thailand)_ribbon.png|File:Order of the White Elephant - Medal (Thailand) ribbon.png]] 銀質章（七等）
:: {{lang-th|เหรียญเงินช้างเผือก}} (''rĭan ngern cháang-pèuak'')
:: 簡稱：{{lang|th|ร.ง.ช.}}
