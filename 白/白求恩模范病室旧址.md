'''白求恩模範病室舊址'''，位於[[山西省|山西省]][[忻州市|忻州市]][[五台县|五台县]][[耿镇镇|耿镇镇]]松岩口村，是[[第二批全國重點文物保護單位|第二批全國重點文物保護單位]]的革命遺址及革命紀念建築物之一。病室由[[寺廟|寺廟]]改建而成，是典型的四合院，共建有三所[[正廳|正廳]]、四所[[耳房|耳房]]和七所房屋，在[[中國抗日戰爭|中國抗日戰爭]]時曾經是[[白求恩|白求恩]]執業的地方，因被[[侵華日軍|侵華日軍]]焚燬而在1974年重建。白求恩模範病室舊址在1982年被列为[[全国重点文物保护单位|全国重点文物保护单位]]。

== 歷史 ==
[[File:Bethune_ward.jpg|left]]
在[[中國抗日戰爭|中國抗日戰爭]]期間的1938年3月，[[加拿大|加拿大]]醫生[[白求恩|白求恩]]率领医疗队至中國[[延安|延安]]，並在同年5月轉到[[晋察冀边区|晋察冀边区]]五台县工作<ref name="杨继红"/>。他视察了當地的医院，發現[[药物|药物]]和[[醫療器械|醫療器械]]严重短缺、医护人员的技术水平也很低，令他感到难以开展醫療工作<ref name="杨继红"/>。於是白求恩決定成立一所标准化的示范医院，以作為醫療和教学的用途<ref name="杨继红"/>。他就此向[[毛澤東|毛澤東]]撰寫報告，並向[[聂荣臻|聂荣臻]]提出設立示范医院的建議；聂荣臻起初认为示范医院會因前线形势的变化而难於维护，但他與[[叶青山|叶青山]]商議後同意了白求恩的建議<ref name="党史博览"/><ref name="五台山"/><ref name="司苏实"/>。

该医院由白求恩親自設計和施工，把一座[[龍王廟|龍王廟]]改建為[[外科|外科]]病室，設有[[手術室|手術室]]、消毒室、醫務室和洗滌室，用以救治戰爭的傷員、培訓醫務幹部；同时他让村內的工匠也参与到修建工程当中，修建人员一起在白求恩的指导下改建房舍、修葺门窗、製造医疗器械<ref name="五台山"/><ref name=華夏經緯網/><ref name="山西晚报"/>。医院在1938年9月15日落成，此後被[[晉察冀軍區後方醫院|晉察冀軍區後方醫院]]稱為「白求恩模範病室」；聂荣臻、[[宋劭文|宋劭文]]、[[胡仁奎|胡仁奎]]等人出席了模範病室的落成典禮，白求恩被任命為院长<ref name="五台山"/><ref name=華夏經緯網/><ref name="山西晚报"/><ref name="党史博览"/>。模范病室用以救治戰爭的傷員，但主要的用途是培訓醫務人员，向各个根据地的医疗人员提供临床教学<ref name=華夏經緯網/><ref name="齐鲁网"/>。

在白求恩模範病室落成典礼的一个星期後，[[侵华日军|侵华日军]]进攻晋察冀边区，模範病室組織了多個战地医疗队，在抗日前线工作<ref name="人民网"/>。過了不久，白求恩模範病室因[[三光政策|三光政策]]而遭到日军焚毁，只剩下一棵松树和门楼未被毀壞<ref name="人民网"/><ref name="山西晚报"/>。中华人民共和国成立后，1974年根據原貌得以重建<ref name="齐鲁网"/>。

== 結構 ==
[[File:Bethune_and_a_patient.jpg|缩略图]]伤兵看病]]
白求恩模範病室是一座典型的[[四合院|四合院]]，長41公尺、寬28公尺，面積為1,148平方公尺，共建有三所正厅、四所耳房和七所房屋<ref name="山西晚报"/><ref name="晋祠"/>。白求恩模范病室旧址的大门朝南，院墙前方有一塊写有「全国重点文物保护单位」的石刻牌，墙上寫着「白求恩模范病室旧址」<ref name="山西晚报"/>。四合院东面的两所耳房是医务室，由财神庙改建而成，修復後陳設了送药箱，而院门东面的小耳房則用作洗涤室；西面的两所耳房是消毒室，原是瘟神庙，修復後放置了消毒笼<ref name="山西晚报"/>。四合院的东侧有七所病房，中间的三所病房住了刚完成[[外科手術|外科手術]]的伤者，兩所病房住了重伤的伤者，另外兩所住了轻伤的伤者；西侧没有房屋<ref name="山西晚报"/>。北面三所正厅是龙王庙改建而成的手术室，修復後有白布包裹四周，房的中間有一张手术台，並陈列了一些医疗器械；南侧本來是龙王庙的戏台，修復後放置了白求恩的雕像<ref name="山西晚报"/>。四合院中間有一棵大[[松樹|松樹]]<ref name="華夏經緯網" />。

== 保護和開發 ==
中華人民共和國成立後，[[國家文物局|國家文物局]]在1974年撥出款項修復白求恩模範病室舊址，按原貌重建之，而室內的陳設亦是按照原樣<ref name="華夏經緯網" /><ref name="晋祠" />。除了修復工程外，病室舊址的東面設立了「纪念白求恩陈列室」，陈列室後來改为「白求恩纪念馆」<ref name="忻州"/>。纪念馆有一個六米多高的[[汉白玉|汉白玉]]题字塔，刻上了毛澤東《[[纪念白求恩|纪念白求恩]]》全文，以及[[徐向前|徐向前]]、聂荣臻和[[薄一波|薄一波]]褒揚白求恩的題字；纪念馆有白求恩的雕像、關於白求恩行醫的图片，以及工作汇报、信件、日记、医疗器械等文物<ref name="忻州"/>。

白求恩模範病室舊址在1982年2月23日被列入第二批全國重點文物保護單位的「革命遺址及革命紀念建築物」，国家文物局由此呼籲各地政府做好文物保护和管理工作，為文物划出保护范围、作出标志说明、建立纪录。截至2015年，白求恩模範病室舊址是中国境内保护规格最高的白求恩相關文物保护单位<ref name="国家文物局"/><ref name="党史博览" />。為了配合[[红色旅游|红色旅游]]的構想，參觀白求恩模範病室舊址並不需要購買门票；白求恩模範病室舊址以往參觀人數不多，但在2005年因「红色旅游年」的安排而吸引了更多參觀人次<ref name="杨继红" />。2013年，[[白求恩国际和平医院|白求恩国际和平医院]]舉行了名為「重走白求恩路」的义诊活动，其中一個站點為白求恩模範病室舊址，在此為264人提供了診治服務<ref name="新华网"/>。
