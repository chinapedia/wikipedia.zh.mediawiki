'''白尾海雕'''（[[學名|學名]]：''{{lang|la|Haliaeetus albicilla}}''<ref>註釋：在
'''[[語源學|語源]]'''上，「''Haliaeetus''」為[[新拉丁語|新拉丁語]]，意為「海雕」。「''albicilla''」意為「白尾」，由[[拉丁語|拉丁語]]中的「''albi-''」（白色的）加「''cilla''」（尾巴）而成。</ref>）又稱白尾海鷲<ref>此名極為少用，但白尾海鵰在英語中亦可為「'''Erne'''」或「'''Ern'''」，在[http://hk.dictionary.yahoo.com/search.html?q=1&s=ern 雅虎香港字典]{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}中被譯為「白尾海鷲」</ref>，是[[鷹科|鷹科]]中一種非常大型的[[猛禽|猛禽]]。很多人認為牠們是[[美國|美國]][[白頭海雕|白頭海雕]]的近親，並深信牠們在[[舊大陸|舊大陸]]居住在同一個[[生態位|生態位]]。

== 外觀 ==
白尾海雕是一種大型的鳥類，長76到100[[公分|公分]]，[[翼展|翼展]]長200至250[[公分|公分]]，[[體重|體重]]介乎於3.5到7.5[[公斤|公斤]]之間。[[雌性|雌性]]的比[[雄|雄]]性的大很多。白尾海雕的翅膀張開時非常闊，頭部很大，嘴部很厚，很輕易就能把肉切開。成年的白尾海雕除了較淺色的頭部和頸部、白色的尾部、黃色的嘴和腳外，基本上全身的主要顏色為棕色。幼體的嘴和腳較成體的深色，在半成年時，牠們的尾部才變為帶黑色條紋的[[白色|白色]]。

== 分佈情況及分類學 ==
白尾海雕在北[[歐亞大陸|歐亞大陸]]進行繁殖。1975年，白尾海雕被成功引入[[蘇格蘭|蘇格蘭]][[小三島|小三島]]（Small Isles）的[[拉姆島|拉姆島]]（Isle of Rum），現時牠們在整個[[外赫布里底群島|外赫布里底群島]]（Outer Hebrides）和[[韋斯特羅斯|韋斯特羅斯]]（Wester Ross）的海岸進行繁殖。而在英國，白尾海雕仍然鮮少到那裡進行繁殖。全歐洲最大的白尾海雕族群位於[[挪威|挪威]]海岸。

2006年5月22日，有消息指有一對白尾海雕在[[荷蘭|荷蘭]]Oostvaardersplassen自然保護區內進行繁殖。牠們是自行進入該自然保護區的，而不是由人類引入。這是首次有白尾海雕自行進入荷蘭進行繁殖。在2007年，這對白尾海雕回到自己的巢去了<ref>荷蘭Oostvaardersplassen自然保護區的擁有者攝得白尾海雕築巢的片段，並利用[[網路攝影機|網路攝影機]]把圖像轉為電腦數據，放在網上，見[http://www.staatsbosbeheer.nl/actueel/zeearend/default.asp?id={8301F776-F5D0-46A9-8432-99B08D34A3AC} 此]</ref>。

2007年夏天，白尾海雕開始被引入[[愛爾蘭共和國|愛爾蘭共和國]]。來自挪威的15隻年輕白尾海雕將被放於愛爾蘭西南部的[[凱里郡|凱里郡]]（County Kerry）[[基拉尼國家公園|基拉尼國家公園]]（Killarney National Park）。這個大型的項目將會持續數年，希望將更多的白尾海雕引至基拉尼國家公園。白尾海雕在愛爾蘭已有悠久的歷史，但在1800年代卻因為受到當地的人捕獵而逐漸消失了。在之後的5年都會每年有15隻小白尾海雕被放在基拉尼國家公園。<ref>[http://www.rte.ie/news/2007/0816/eagles.html RTÉ：「稀有的鷹被引進愛爾蘭共和國」]</ref>

白尾海雕大多並不會[[鳥類遷徙|遷徙]]，只有最北部的白尾海雕族群，像是在[[西伯利亞|西伯利亞]]的白尾海雕才會在每年的冬季遷至南方。

在[[格陵蘭|格陵蘭]]的最西南端及[[冰島|冰島]]西部有一個不遷徙的小族群。這個白尾海雕族群的成員體型特別大，因此以前有人認為這小族群是另一[[亞種|亞種]]，稱之為「''groenlandicus''」，現時科學家認為這族群是單型種（monotypic species），而牠們的體型之所以如此大是因為牠們為[[漸變群(群體遺傳學)|漸變群]]（Cline），即如[[伯格曼氏法則|伯格曼氏法則]]（Bergmann's Rule）所述<ref>關於伯格曼氏法則可參考[http://bbs3.nsysu.edu.tw/txtVersion/treasure/nature-ecology/M.892912918.A/M.1032665503.A.html 中山大學－西子灣站之「nature-ecology」，淺談鳥類的分類原則]——亞種分化的原則之一：根據伯格曼氏法則，同一物種中，生活於較寒冷地區的亞種的體型較大</ref>。

白尾海雕與[[白頭海雕|白頭海雕]]是一個[[複合種|複合種]]（species complex）。牠們從[[中新世|中新世]]早期開始出現差別。這個複合種由一個白頭的物種（白頭海雕）和一個黑頭的物種（白尾海雕）組成。像[[虎頭海雕|虎頭海雕]]（Steller's Sea-eagle）般，牠們都擁有黃色的爪和嘴，若是成體的話，牠們的眼睛亦然。

== 食物 ==
白尾海雕的食物很多，有[[魚類|魚類]]、[[鳥類|鳥類]]和[[腐肉|腐肉]]，有時牠們亦會捕獵一些小型的[[哺乳動物|哺乳動物]]。很多鳥類只吃腐肉維生，有時去搶[[水獺|水獺]]和其他鳥類的食物，而白尾海雕亦是一個很強的獵手。有些白尾海雕會跟[[金鷹|金鷹]]爭奪[[兔|兔]]和[[野兔|野兔]]，因為牠們兩者都會吃捕食這些動物。在繁殖季節期間，若牠們有了胎兒，牠們每天需要500至600克的食物。在冬季的月份，牠們不大想活動，這時牠們的食量就會降至每天吃200到300克的食物。

== 繁殖 ==
[[File:White-tailed_Eagle_with_2_Buzzards.jpg|250px]]Canna島上空被兩隻[[鵟|鵟]]襲擊]]
白尾海雕在4或5歲時已性成熟。牠們找到配偶後就會跟對方共處一輩子，但若有其中一方死了，牠們仍舊會很快找到新的配偶。牠們會在空中示愛。白尾海雕發出的聲音要比[[金鷹|金鷹]]多得多，特別是在繁殖季節期間，若雄性接近雛鳥時，雄性就會發出很多聲音。 

白尾海雕的巢很大，一般是位於一棵[[樹|樹]]上的樹枝或在海邊的[[懸崖|懸崖]]構成的。牠們的巢通常會再使用，有時，一個巢甚至可用數十年。在[[斯堪地納維亞|斯堪地納維亞]]，不斷有很多樹木塌下來，就是因為長期負荷不住白尾海雕巨大的鳥巢。

一般而言，白尾海雕的巢穴位於隱蔽的懸崖上，但亦有時會在湖邊或河邊。白尾海雕的地盤可與金鷹的重疊，但是這兩個物種之間偶爾也會產生競爭。金鷹較喜歡山脈及荒野，而白尾海雕則較喜歡海岸和海洋。

一對白尾海雕每年可生出1到3隻[[蛋|蛋]]。[[雄性|雄性]]和[[雌性|雌性]]均會協助孵蛋的工作，而蛋一般會在離開母體後38天才孵化。在孵出後，幼鳥一般都很會互相遷就，但通常第一隻孵出的幼鳥體型都會較大，且在進食時佔有較大的優勢。雌性白尾海雕會完成大部分的孵蛋工作及較常餵幼鳥。在出生5到6個星期後，幼鳥一般就能自行進食，在出生11到12個星期後幼鳥才會長出羽毛，但牠們仍會一直留在巢中，依賴父母捕獵維生，再過6至7個星期牠們才開始懂得自立。

== 將近滅絕及後來在歐洲的復原 ==
白尾海雕是[[頂級掠食者|頂級掠食者]]。在1959到60年代期間，白尾海雕在歐洲多處地方的數量急遽下跌，在一些地區甚至滅絕了。因此，很多地方都開始設立法律，希望能減少對白尾海雕的捕獵及保護繁殖區和冬季攝食區。後來，白尾海雕在歐洲的數量終於得以回升。因此，歐洲從前的白尾海雕繁殖區現時也重新引入了一些白尾海雕，但其數量直到今天仍在上升。

最近在歐洲中北部對白尾海雕進行的[[微隨體|微隨體]]（microsatellite）[[粒線體DNA|粒線體DNA]]有關研究<ref>該項研究為「Hailer ''et al.'' 2005, 2006」</ref>顯示，白尾海雕的數量仍在上升，仍然保持著可觀的[[遺傳多樣性|遺傳多樣性]]（genetic diversity）數量，這暗示著牠們的[[近親交配衰退|近親交配衰退]]（inbreeding depression）的風險很低<ref>此為一個很嚴重的問題，愈長壽的動物密度就愈低</ref>，這一再證明了白尾海雕的數量在歐洲已成功復原。
