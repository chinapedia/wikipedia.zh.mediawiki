'''白玉瀑布'''是一座位在[[臺灣|臺灣]][[臺東縣|臺東縣]][[卑南鄉|卑南鄉]][[知本溫泉|知本溫泉]]區內的[[瀑布|瀑布]]，其原名為'''白玉瀧'''（[[日文|日文]]：白玉の瀧）<ref name="白玉の瀧">{{cite web|language=zh-tw|url=http://memory.ncl.edu.tw/tm_cgi/hypage.cgi?HYPAGE=image_photo_detail.hpg&project_id=twpt&dtd_id=10&xml_id=0000363603&subject_name=%E6%97%A5%E6%B2%BB%E6%99%82%E6%9C%9F%E8%87%BA%E7%81%A3%E5%9C%96%E5%83%8F%E5%AF%AB%E7%9C%9F|title=臺東名所 知本溫泉白玉瀑布|author=|publisher=臺灣記憶圖像|date=|accessdate=2015-08-12}}</ref>，目前已改為白玉瀑布，是知本溫泉區的景點之一。<ref name="白玉瀧">{{cite web|language=zh-tw|url=http://tour.taitung.gov.tw/zh-tw/Travel/ScenicSpot/279/%E7%99%BD%E7%8E%89%E7%80%91%E5%B8%83|title=白玉瀑布|author=|publisher=臺東縣政府臺東觀光網|date=|accessdate=2015-08-12}}</ref>

== 沿革 ==
[[File:Bai-Yu_Waterfall02.jpg|thumb]]
白玉瀑布最早在[[臺灣日治時期|臺灣日治時期]]由[[日本人|日本人]]所發現，並且當時稱之為白玉瀧。[[臺灣戰後時期|中華民國接管臺灣後]]，[[臺東縣政府|臺東縣政府]]將白玉瀧改名為白玉瀑布，由於瀑布與鄰近的樂山產業道路之間，尚有一段無法行車之山徑，故在近年已興建步道於瀑布與產業道路之間。<ref name="白玉瀧"/><ref name="白玉瀑布">{{cite web|language=zh-tw|url=http://www.chu.edu.tw/~s09303070/writer.html|title=白玉瀑布|author=|publisher=中華大學|date=|accessdate=2015-08-12}}</ref>

== 特色 ==
白玉瀑布高約50公尺，並且，白玉瀑布旁尚有一座小瀑布，就位在樂山產業道路旁，不須經由步道抵達，其落差高度高於主瀑布也就是白玉瀑布，但水量則還是主瀑布大，目前主瀑布水量之多寡受限於當地[[氣候|氣候]]之影響，因此瀑布的豐水/枯水期十分明顯。<ref name="白玉瀧"/><ref name="白玉瀑布"/>

== 交通資訊==
私人轎車：行經[[File:TW_CHW194.svg|23px]][[縣道194號|194縣道]]自知本溫泉區外溫泉方向往內溫泉方向，當看到''白玉瀑布''之指示牌後，左轉上切接入樂山產業道路，續行即可到達白玉瀑布下方之停車場，再從停車場經步道向溪流步行至步道終點即可抵達白玉瀑布下方。目前已封路只能步行進入。
