'''白蘭高·伊歷'''（{{lang-en|'''Branko Ilič'''}}，{{bd|1983年|2月6日|catIdx=Branko Ilič}}），是一名[[斯洛文尼亞|斯洛文尼亞]]足球运动员，司职後衛，目前效力洛文尼亞球队[[奧林比查|奧林比查]]。

他通常擔任左[[後衛|後衛]]，他也可以擔任中後衛。

== 球會生涯 ==

伊歷的職業生涯開始於[[奧林比查|奧林比查]]，在2003年對[[高柏|高柏]]的比賽中首次亮相。因球會的嚴重的財政困難，2005年1月，他離隊加盟[[當姆薩尼|當姆薩尼]]，他在陣中踢了1年半。

2007年1月，伊歷被租借到[[貝迪斯|貝迪斯]]，成為球會第一名斯洛文尼亞籍球員。他第一次露面是於國王杯決賽對同城死敵西維爾。 2月4日，他首次為球隊出踐聯賽比賽，對陣畢爾包，他助攻予羅拔使球隊在客場勝2-1。

經過半個多賽季，該安達盧西亞球會2007年夏天以€1.5m買斷他<ref>[http://www.nzs.si/index.php?t=reprezentancaPortret&id=19912 El Betis se queda con Ilic (Betis keeps Ilic)] {{es icon}}</ref>，但伊歷表現最終都回落了下來，在2008-09球季，球隊在賓菲加收購尼爾遜，在護級的球季，他只上陣三場比賽。2009年9月6日，FC莫斯科向貝迪斯外借得到這名斯洛文尼亞的右後衛。

== 國際賽生涯 ==

2004年8月18日，伊歷首次代表在一場友誼賽與鄰國塞爾維亞替國家隊出賽。
