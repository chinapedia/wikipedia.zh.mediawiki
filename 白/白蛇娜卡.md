'''白蛇娜卡'''（'''白蛇のナーガ'''）為[[日本|日本]][[輕小說|輕小說]]《[[秀逗魔導士|秀逗魔導士]]》外傳系列當中的主要角色之一。<ref>{{cite web|url=http://www.mania.com/slayers-great_article_76719.html |title=Slayers Great |publisher=Mania.com |date= |accessdate=2013-09-25}}</ref><ref>{{cite web|url=http://www.animefringe.com/magazine/01.02/reviews/2/index.php3 |title=Reviews - Slayers: The Book of Spells DVD |publisher=Animefringe |date= |accessdate=2013-09-25}}</ref><ref>{{cite web|url=http://www.animenewsnetwork.com/review/slayers-return/dvd |title=Slayers Return DVD - Review |publisher=Anime News Network |date=2004-01-13 |accessdate=2013-09-25}}</ref><ref>''Gamers' Republic'' issue 4, page 88.</ref>

== 角色介紹 ==
: 娜卡被設定為胸大無腦的魔導士，但胸藏知識卻很豐厚，自稱是[[莉娜·因巴斯|莉娜·因巴斯]]最大的對手，登場時通常伴隨著「哦～呵呵呵呵呵呵」的高笑聲出現。其名稱是取自於[[印度神話|印度神話]]的蛇神「[[那伽|那伽]]（Nagas）」。
: 娜卡在正傳小說裡只有回想而沒實際登場，但在外傳中卻很活躍，跟小說裡的幾個人物有密切的關係；擅長黑魔術、精靈魔術、白魔術，以冰系魔法為主要攻擊魔法，本身也偏愛召喚魔法跟泥巨人生成術。身為魔導士的資質極高，只要看過一次該魔法以自學的方式就學會了。
: 娜卡跟莉娜一樣是大胃王，酒量也很好，就算是用大啤酒杯裝著[[白蘭地|白蘭地]]一口氣喝下肚也面不改色。十分出眾的容貌並有一頭黑髮的美人，但是裝扮讓人看起來會聯想到「邪惡的女黑魔導士」。經常穿著黑色緊身衣型態（[[比基尼泳衣|比基尼泳衣]]樣式）的衣服與附有錐型尖刺的護肩（母親的遺物），即使在[[冬天|冬天]]或十分惡劣的天氣穿這樣子也不會[[感冒|感冒]]。即使是在如何失敗也好，被魔法炸飛也好，被打飛、被冰住、被龍踩過也好，都能以極強的恢復力馬上復活而自豪（這種「[[非常識|非常識]]」的設定跟動畫版的[[高里·加布列夫|高里·加布列夫]]十分相同）。
: 娜卡有著異於常人的審美觀和[[價值觀|價值觀]]。但對各國上流統治階級的情況和黑社會的消息十分清楚，而且身上散發著領導人的魅力（雖然本人並沒這意思）；令人意外的是，娜卡對家務事十分拿手。
: 娜卡真實身份是阿梅莉亞的親生姐姐，本名為「葛蕾希亞·烏爾·娜卡·賽倫（グレイシア·ウル·ナーガ·セイルーン）」，是賽倫王家第一王女，也是國王的繼承人第一王子菲爾王子的長女。娜卡一身接近[[全裸|全裸]]的服飾其實屬於母后，也是菲爾王子的亡妻隱藏的[[興趣|興趣]]。由於曾親眼目睹母后浴血身亡的情形，只要見血就昏厥。以「要修行」的理由，離開了賽倫；其實是由於母后被暗殺，而體認到自己的不足。該暗殺者似乎是スレイヤーズすぺしゃる第三集提到的ブーレイ......不過阿梅莉亞並不擔心，只當她是迷路所以遲遲沒回家。

== 備考 ==
: 娜卡與莉娜認識的原因：
::當初跟莉娜打過一次架而輸了，之後把要打倒莉娜當作理由就像「金魚大便」一樣黏著莉娜，與其一起旅行，一起吃飯（雖然大部分都是莉娜買單）。
: TV版刻意修改幾個娜卡應該出場的設定：
:*當初設定是莉娜將惡人們連帶城市一起用龍破斬吹飛，之後娜卡來責罵莉娜（但是娜卡被阿梅莉亞替代了）。
:*莉娜休息的旅店失火（娜卡放的火），而一不小心被捲入火場，再登場的時候是以全身包著紗布的木乃伊的狀態出現（這被TV版第一季的三流魔導士替代，但第二部OVA版則還原。）
