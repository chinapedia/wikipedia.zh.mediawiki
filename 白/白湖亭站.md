'''白湖亭站'''位于[[福州市|福州市]][[仓山区|仓山区]][[盖山镇|盖山镇]]则徐大道，是[[福州地铁|福州地铁]][[福州地铁1号线|1号线]]的一座中间站，于2016年5月18日开通试运营。<ref>{{Cite web|url=http://www.xinhuanet.com/local/2016-05/18/c_1118890602.htm|title=福州首条地铁南段开通-新华网|accessdate=2019-01-02|work=www.xinhuanet.com}}</ref>

== 站名来源 ==
在古代时该地曾有一湖，名为“白湖”，明朝时期，人们在白湖边修筑一栋[[亭_(東亞)|亭子]]并取名“白湖亭”，随后白湖亭便成为该地地名并沿用至今。地铁站所处位置即为白湖亭，终定名白湖亭站。<ref>{{Cite web|url=http://dzb.hxnews.com/2016-05/18/content_380411.htm|title=地铁1号线 9站名起底_海都报电子版_海峡都市报电子版_海峡网|accessdate=2019-01-02|work=dzb.hxnews.com}}</ref>

== 车站结构 ==
白湖亭站位于仓山区则徐大道下则徐广场西侧，基本呈东南至西北走向，主体结构为双柱三跨地下二层车站，岛式站台，地下一层为站厅层，地下二层为站台层。车站全长172米，宽19.2米。<ref>{{cite news|url=http://mag.fznews.com.cn/shtml/fzwb/20150403/178691.shtml|title=地铁1号线白湖亭站 今晚二期施工围挡|publisher=[[福州晚报|福州晚报]]|date=2015-04-03|accessdate=2016-06-03}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

=== 站厅 ===
{{ multiple image | align = right | direction = vertical | width = 220

白湖亭站站厅位于则徐大道下方。车站站厅采用白色直线型吊顶与白色搪瓷板装修，设客服中心1处，分布于站厅中心，服务设施设置智能导乘机1处、自动售货机2处，C出入口通道旁规划设有便利店，D出入口通道设有洗手间。

== 出入口与周边 ==
白湖亭站共设4个出入口，各分布于则徐大道与盖山路交叉路口的四个象限，无障碍电梯设于C出入口。<ref>{{Cite web |url=http://www.fzmtr.com/frmDetailPage.aspx?key=df81a003a4b143e68cacb2d5e5ebf08e |title=白湖亭站 |accessdate=2016-07-08 |archiveurl=https://web.archive.org/web/20160630175044/http://www.fzmtr.com/frmDetailPage.aspx?key=df81a003a4b143e68cacb2d5e5ebf08e |archivedate=2016-06-30 |deadurl=yes }}</ref>

=== 周边 ===
白湖亭周边主要为住宅用地与待开发地块，D口所在地块为则徐广场。白湖亭公交枢纽站可由C口出向南步行150米到达；福建商业广场可由A口出向北步行200米到达；盖山镇政府可由B口出向西步行150米到达。
