'''白花蛇舌草'''（[[學名|學名]]：''{{lang|la|Hedyotis diffusa}}''<ref name = "浸會大學">[http://libproject.hkbu.edu.hk/was40/detail?channelid=1288&searchword=herb_id=D01227 藥用植物圖像數據庫]香港浸會大學中醫藥學院</ref>），別稱'''白花十字草'''、'''蛇舌草'''、'''蛇舌癀'''、'''蛇針草'''、'''蛇總管'''、'''龍舌草'''、'''鶴舌草'''、'''龍吐珠'''、'''矮腳白花蛇利草'''、'''尖刀草'''、'''珠仔草'''、'''[[羊鬚草|羊鬚草]]'''、'''羊須草'''、'''目目生珠草'''、'''節節結蕊草'''、'''二葉葎'''、'''南地珠'''、'''千打捶'''、'''竹葉草'''等<ref name = "浸會大學"/><ref name = "實用中草藥圖典">{{cite book|title=《實用中草藥圖典》|author= |series=宋立人|publisher=商務印書館|page=第76頁|isbn= 978-962-07-5525-5|date=2010年1月}}</ref><ref>[http://www.cfh.ac.cn/37521.sp 物種信息卡]中國自然標本館</ref>，為[[茜草科|茜草科]][[耳草屬|耳草屬]]植物。花期7－9月，果期8－10月<ref name = "浸會大學"/>。

==分佈及生境==
分佈於[[中國|中國]]、[[尼泊爾|尼泊爾]]、[[日本|日本]]等地，中國境內分佈於[[廣東|廣東]]、[[廣西|廣西]]、[[海南|海南]]、[[福建|福建]]、[[安徽|安徽]]、[[雲南|雲南]]及[[香港|香港]]等地<ref name = "當代藥用植物典">{{cite book|title=《當代藥用植物典》|author= |series=蕭培根|publisher=香港賽馬會中藥研究院|page=第342-345頁|isbn=978-988-99226-3-4|date=2007年8月}}</ref>。多生長於濕潤的田邊、路旁、溝邊、草地及農舍旁，喜溫暖濕潤的環境，不耐旱<ref name = "浸會大學"/><ref name = "實用中草藥圖典"/><ref name = "本草圖錄 ">{{cite book|title=《中國本草圖錄 卷二》|author= |series=連文琰、胡廷松|publisher=商務印書館、人民衛生出版社|page=第184頁|isbn=962-07-3079-8|date=1988年11月}}</ref>。

==形態特徵==
白花蛇舌草是一種一年生披散[[草本植物|草本植物]]，高約15－50厘米。[[根|根]]細長，白色，有分枝。[[莖|莖]]略帶方形或扁圓柱形，纖細，中央有白色的髓部，具縱棱，光滑無毛，從基部分枝。[[葉|葉]]線形至線狀披針形，單葉，對生，薄紙質，無葉柄，先端急尖，側脈不明顯，與托葉相連，上面光滑，下面有時稍粗糙，長約1－3.5厘米，寬約1－3毫米；托葉膜質，基部合生成鞘狀，頂端有小齒，長約1－2毫米。花單生或成對生於葉腋；花白色；多具花梗，花梗略短粗；[[花冠|花冠]]4裂；[[花萼|花萼]]管與[[子房|子房]]合生，球形，頂端4裂，裂片長圓狀披針形，邊緣具睫毛，宿存，長約1.5－2毫米；花冠漏斗形，白色，長約3.5－4毫米，先端4深裂，裂片卵狀長圓形，禿淨，長約2毫米；[[雄蕊|雄蕊]]4枚，長於冠管喉部，與花冠裂片互生；花絲扁形；花藥卵形；子房下位，2室；柱頭半球形，2淺裂。[[果|果]]為[[蒴果|蒴果]]，雙生，膜質，扁球形，頂端具宿萼4齒裂，成熟時室背開裂，直徑約2－2.5毫米。[[種子|種子]]棕黃色，細小，具3個棱角。<ref name = "浸會大學"/><ref name = "實用中草藥圖典"/><ref name = "本草圖錄 "/><ref name = "中藥材圖鑑">{{cite book|title=《香港中藥材圖鑑》|author= |series=趙中振|publisher=香港浸會大學中醫藥學院|page=第100頁|isbn= 988-97448-1-3|date=2003年10月}}</ref><ref name = "常用中藥材鑑別圖典">{{cite book|title=《常用中藥材鑑別圖典》|author= |series=趙中振、陳虎彪|publisher=萬里機構|page=第415頁|isbn= 978-962-14-4150-8|date=2010年1月}}</ref><ref>{{cite book|title=《郊野的閒花野草》|author= |series=陳永鏗|publisher=萬里機構、明天出版|page=第26-29頁|isbn= 978-962-14-3767-9|date=2008年5月}}</ref>

==醫藥用途==
{{medical}}
白花蛇舌草全草入藥，味苦、甘，性寒，無毒，歸[[手少陰心經|心]]、[[手太陰肺經|肺]]、[[足厥陰肝經|肝]]、[[手陽明大腸經|大腸]]經。能清熱解毒、活血止痛、利尿消腫、利濕通淋、[[抗氧化|抗氧化]]、抗腫瘤、抗炎、抗化學誘變、保護[[胃黏膜|胃黏膜]]損傷及增強免疫力等。主治[[肺癰|肺癰]]、咽喉腫痛、肺熱喘咳、水腫、[[肝炎|肝炎]]、[[闌尾炎|闌尾炎]]、[[扁桃體炎|扁桃體炎]]、[[泌尿系統|泌尿系統]]感染、[[腸癰|腸癰]]、[[痢疾|痢疾]]、[[黃疸|黃疸]]、[[惡性腫瘤|惡性腫瘤]]等，外用可治毒蛇咬傷。[[中藥|中藥]]名為白花蛇舌草。<ref name = "浸會大學"/><ref name = "實用中草藥圖典"/><ref name = "當代藥用植物典"/><ref name = "本草圖錄 "/><ref name = "中藥材圖鑑"/><ref name = "常用中藥材鑑別圖典"/>

==醫藥典故==
傳說古時一位善者，因得肺癰而病重，群醫束手無策，家屬邀得一位名醫診治，名醫診病閱方一時亦未能找出治療良方，倦極伏案小睡，夢見一位白衣女子託夢對他說：「此君乃是大好人，樂善懷仁，惠及生物，見有捕蛇者，他即買下放生，先生務必精心施治，救他一命。」名醫於是向其討教治療良方，白衣女子帶名醫往戶外，在白衣女子所到之處有一條[[白花蛇|白花蛇]]，蛇舌伸吐處化作小草，其後名醫醒來在田邊發現夢中小草，採下煎服給病者，病者服後痊癒。名醫其後查遍當時的歷代本草，也未能查出此草為何種草藥，有感吟詩：「白花蛇舌草纖纖，伏地盤桓農舍邊，自古好心多善報，靈蟲感德藥流傳。」白花蛇舌草因而得名。<ref name = "實用中草藥圖典"/><ref>[http://www.brion.org.tw/index.php?option=com_content&task=view&id=648&Itemid=11 財團法人 台灣必安研究所]</ref>本種民間應用於清朝末年，始於中國[[廈門|廈門]]、[[汕頭|汕頭]]一帶，其後大量出口至[[東南亞|東南亞]]地區。六十年代，逐漸由民間草藥轉型成為中成藥原料藥。<ref name = "當代藥用植物典"/>

==藥材鑑定==
本種乾燥全草，灰綠至灰棕色，扭纏成團狀；莖質脆易折斷，幼細捲曲狀；主根細長1條，鬚根纖細；以莖葉完整，[[果實|果實]]飽滿者為佳。但本種尚未載入《中國藥典》，故而沒有評價標準。<ref name = "實用中草藥圖典"/><ref name = "當代藥用植物典"/><ref name = "中藥材圖鑑"/><ref name = "常用中藥材鑑別圖典"/>

==化學成份==
全草含環烯醚萜類、三萜類、[[黃酮類|黃酮類]]、蒽醌類化合物、有機酸及其酯類等化學成分組成，當中以[[熊果酸|熊果酸]](ursolic acid)及[[齊墩果酸|齊墩果酸]](oleanolic acid)常作為指標性成分參考，以便於控制其藥材質量。<ref name = "實用中草藥圖典"/><ref name = "當代藥用植物典"/>
