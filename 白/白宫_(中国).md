'''白宫'''在[[中华人民共和国|中华人民共和国]]境内，通常是指那些与[[美国国会大厦|美国国会大厦]]建筑风格相近的[[中华人民共和国地方政府|地方政府]]建筑。中国民众通常会称其为[[白宫|白宫]]，虽然这些建筑风格与美国国会大厦更为接近。

位于[[安徽省|安徽省]][[阜阳市|阜阳市]][[颍泉区|颍泉区]]的颍泉区政府建筑——阜阳“白宫”是较为知名者。有报道认为，这座建筑“象征着当地领导干部的狂妄与贪婪”<ref name="a1">{{Cite web |url=http://www.ftchinese.com/story/001019763/ce |title=中国的“白宫” |accessdate=2012-07-25 |author= |date=2008年6月3日 |work= |publisher=FT中文网|language=Zh-hans }}</ref>。而这座建筑物的造价，约占颍泉区全年[[财政收入|财政收入]]的三分之一。

== 背景 ==

在中国城市中，用以显示[[中国共产党|中国共产党]]和[[中华人民共和国政府|中国政府]]政绩的公共建筑中，很少是学校或医院<ref name="a1"/>。中国学校房屋所普遍存在的[[校舍|校舍]]安全问题，一直使中国政府受到外界指责<ref>{{Cite web |url=http://www.stnn.cc/china/200812/t20081229_951531.html|title=中国校舍安全隐患大 危房面积相当134个鸟巢  |accessdate=2012-07-25 |author= 编辑：崔艳玲 |date=2008-12-29 |work= |publisher=星岛环球网|language=Zh-hans }}</ref>。[[中华人民共和国中央政府|中央政府]]认为地方政府要为校舍安全负责<ref>{{Cite web |url=http://news.southcn.com/china/zgkx/content/2009-05/18/content_5159375.htm|title=教育部：校舍因质量问题遇灾垮塌 要问责当地政府 |accessdate=2012-07-25 |author=编辑：赵殿川 |date=2009-05-18|work= |publisher=南方网，来源: 中国新闻网|language=Zh-hans }}</ref>，而现实中，则是奢侈的地方政府建筑与破旧校舍的对比<ref>{{Cite web |url=http://www.cqn.com.cn/news/xfpd/szcj/dflb/518183.html|title=河南上蔡:国贫县的豪华衙门与寒酸校舍|accessdate=2012-07-25 |author= |date=2012-01-10|work= |publisher=中国质检新闻网，来源：中国新闻网|language=Zh-hans }}</ref>。
