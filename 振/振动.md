{{distinguish|振盪}}
'''振动'''（{{lang-en|vibration}}），指一个物体相对于静止参照物或处于平衡状态的物体的往复运动。一般来说振动的基础是一个系统在两个能量形式间的能量转换，振动可以是周期性的（如单摆）或随机性的（如轮胎在碎石路上的运动）。

== 简谐振动 ==

[[简谐振动|简谐振动]]又称'''谐振'''，应作为一种重要的特殊情况来讨论：

右图所示为无[[阻尼|阻尼]]的[[简谐振动|简谐振动]]（参见[[简谐运动|简谐运动]]），与[[位移|位移]]<math>y(t)</math>，[[振幅|振幅]]<math>y_0</math>和[[周期|周期]]<math>T</math>相关。

某一时刻<math>t</math>的位移<math>y(t)</math>达到最大值<math>y_0</math>。周期是一次振动的时间，也就是[[系统|系统]]在振动中两次相同状态的间隔。周期''T''的[[倒数|倒数]]是[[频率|频率]]''f'',即：<math>f = {1 \over T} \quad</math>.<br>
频率的另一个表达符号为<math>\nu</math>（读音："nü"），[[计量单位|计量单位]]为“[[赫兹|赫兹]]”（簡寫：Hz）。

当[[回复力|回复力]]与[[振幅|振幅]]成[[正比|正比]]，此振动称为[[简谐振动|简谐振动]]（注意：[[单摆|单摆]]只是近似的简谐振动）。

这里借助一个[[线性方程组求解|线性系统]]，由于回复力随振幅变化：振幅扩大两倍回复力也随之扩大两倍。

一个这样的振动描述为：
:<math>
y(t)=y_0\cdot\sin(2\pi f t+\varphi_0) \,
</math>

其中
:<math>
y_0</math>  = [[振幅|振幅]]
:<math>\varphi_0</math>  = 振动的初[[相位|相位]]

由
:<math>
\varphi (t) = 2 \pi f t+\varphi_0 \,
</math>
描述出[[相位|相位]]，''f''或<math>\nu</math>为振动的[[频率|频率]]。

<math>2\pi</math>倍的频率，<math>\omega = 2\pi \cdot f</math>,  为振动的[[角频率|角频率]]。
通过[[角频率|角频率]]简写为：
:<math>
y(t)=y_0\cdot\sin(\omega\,t+\varphi_0) \,
</math>

对时间求导得到：
:<math>
v(t)=\omega\cdot y_0\cdot \cos(\omega\,t+\varphi_0) \,
</math>

其中
:<math>v(t)</math> = [[振子|振子]]的速度。

再次求导：
:<math>
a(t)=-\omega^2\cdot y_0\cdot \sin(\omega\,t+\varphi_0) \,
</math>

其中
:<math>a(t)</math> = [[振子|振子]]的[[加速度|加速度]]。

简谐振动的特点是：
# 有一个[[平衡位置|平衡位置]]（[[动能|动能]]耗尽之后，[[振子|振子]]应该[[静止|静止]]的唯一位置）。
# 有一个[[大小|大小]]和[[方向|方向]]都作[[周期|周期]]性变化的[[回复力|回复力]]的作用。
# [[频率|频率]]单一、[[振幅|振幅]]不变。

振动可分為：
*''阻尼''和 ''无阻尼''振动,
*''自由''、''受迫''、''自发''和 ''诱发''振动,
*''线性''和 ''非线性''振动,
*''单自由度的''、''有限多个自由度的''、''无限多个自由度的''振动。

所有这些属性可以平行出现。

== 阻尼振动 ==
{{main|阻尼振动}}
{|style="border: 1pt solid" | class="floatright"
|[[File:Damped_oscillation_graph.svg|File:Damped oscillation graph.svg]][[File:damped_spring.gif|62px]]
|-
|自由阻尼振动
|}

真实情况中物理系统总是[[阻尼|阻尼]]的，因为系统总是通过诸如[[摩擦|摩擦]]等原因向外界放出[[能量|能量]]。放任一个这样的系统自己运动（自由振动），最终会达到“静止状态”，这是[[热力学第二定律|热力学第二定律]]所描述的，[[永动机|永动机]]不存在(参见[[能量守恒|能量守恒]]）。

设一个自由阻尼振动在某一瞬时平衡，得到下面的总运动方程：
:<math>
\mathit{m} \ddot x + \mathit{R} \dot x + \mathit{D} x = 0 \,
</math>

m: [[质量|质量]]<br>
R: [[阻尼系数|阻尼系数]]（不同于[[摩擦系数|摩擦系数]]）<br>
D: [[倔强系数|倔强系数]]（[[劲度系数|劲度系数]]）

(对于[[扭转振动|扭转振动]]<math>m</math>替换为<math>J</math> ([[转动惯量|转动惯量]])

；<math>x</math>替换为<math>\varphi</math> (偏转角) )

借助解法：
:<math>
x (t) = x_0 e^{i \alpha t} \,
</math>

得到[[微分方程|微分方程]]的[[解|解]]：
:<math>
x (t) = x_0 e^{- \frac{\mathit{R}}{2\mathit{m}}t} e^{\mp i\sqrt{\frac{\mathit

{D}}{\mathit{m}} - \frac{R^2}{4 \mathit{m}^2}}t} \,
</math>

借助[[欧拉变形|欧拉恒等式]]把上面的解从[[复数|复数]]带入[[实数|实数]]，并设<math> \frac{\mathit{R}}{2\mathit{m}} = \delta </math>，即得到以下振动函数：
:<math>
x(t)=2x_0\,e^{-\delta t}\sin(\omega\, t+ \varphi_0) \,
</math>

这个有两个[[实数解|实数解]]的函数可分为两个[[子函数|子函数]]，在下面可以详细阐述。

如果阻尼系数为零，则振幅永远不会减小。振动会以相同的摆幅永久持续。这里同样可以看出，阻尼系数不可过大，否则不会发生实际意义上的振动（摆动），而是系统在原地“[[蠕变|蠕动]]”。两种情况之间的界限造成了[[频率|频率]][[临界|临界]]点。

'''衰减期'''<math>\mathbf{\tau}</math>为振幅不断降低到[[e_(数学常数)|<math>\mathbf{e}</math>]]分之一（<math>\approx{0{,}368}</math>）的时间。如振幅函数方程所示，<math>\tau</math>等于函数指数的倒数。
衰减期表示为：
:<math>
\tau=\frac{2\mathit{m}}{\mathit{R}}
</math>

阻尼振动也经常以包含衰减期的形式表达。即：
:<math>
x(t)=2x_0\,e^{-\frac{t}{\tau}}\sin(\omega\, t+\varphi_0) \,
</math>

<math>\mathbf{\tau}</math>也同样用于表示[[松弛时间|松弛时间]]（或[[阻尼时间|阻尼时间]]）。表示系统能量（不是振幅）衰减到[[e_(数学常数)|<math>\mathbf{e}</math>]]分之一（<math>\approx{0{,}368}</math>）的时间。
因此能量与振幅的[[平方|平方]]成[[比例|比例]]，相应的[[半衰期|半衰减期]]的松弛时间：
:松弛时间<math>
\tau=\frac{\mathit{m}}{\mathit{R}} \,
</math>

总解的非阻尼部分可以写作：
:<math>
\,e^{i \omega t} \,
</math>,

因子
:<math>\omega = \sqrt{\frac{\mathit{D}}{\mathit{m}} - \frac{R^2}{4 \mathit{m}^2}}</math>

在指数中称为[[角频率|角频率]]。对于一个非阻尼振动（技术上不可能实现）的阻尼系数<math>\mathit{R} = 0</math>只对'''自角频率'''<math>\mathbf{\omega_0}</math>有意义：
:<math>
\mathbf{\omega_0} = \sqrt{\frac{\mathit{D}}{\mathit{m}}} \,
</math>.

==自由或受迫振动==
作振动的[[系统|系统]]在[[外力|外力]]的作用下物体离开[[平衡|平衡]]位置以后就能自行按其固有[[频率|频率]]振动，而不再需要[[外力|外力]]的作用，这种不在外力的作用下的振动称为[[自由振动|自由振动]]．理想情况下的自由振动叫[[无阻尼自由振动|无阻尼自由振动]]．自由振动时的[[周期|周期]]叫[[固有周期|固有周期]]，自由振动时的[[频率|频率]]叫[[固有频率|固有频率]]．它们由振动系统自身条件所决定，与[[振幅|振幅]]无关．

[[振子|振子]]受到周期性变化的外力持续作用而进行振动称为[[受迫振动|受迫振动]]。实际意义上是指所有通过周期性刺激的正弦状简谐振动。这个对振动进行周期性刺激的频率称作[[刺激频率|刺激频率]]。另外也有多频率的刺激。刺激也通过随机过程[[随机振动|随机振动]]被研究。

在规律刺激下一个系统同时产生两种振动：
*自由振动（一个或多个固有频率），大小定义自[[初始条件|初态条件]]并在单位振动时间内发生阻尼
*狭义上的带有常数振幅的刺激频率的受迫振动。刺激频率（或多个中的一个）与固有频率（或多个中的一个）的关系及振动系统的阻尼可以通过[[放大函数|放大函数]]定量。

在[[工程力学|工程力学]]中[[距离刺激|距离刺激]]，[[力刺激|力刺激]]，[[不平衡刺激|不平衡刺激]]都是重要因子。

振幅在[[共振|共振]]的条件下达到最大值。在无阻尼和刺激频率与固有频率相等时，振幅为无限大。阻尼增大，共振振幅减小。

==自激振动==
通过自身振动产生的能量满足振动所需的能量的振动成为自激振动，称作[[振荡器|振荡器]]。在微分方程中这种现象的阻尼为负数。这方面的一个典型的例子就是[[小提琴|小提琴]]的[[琴弦|琴弦]]。这是由于[[琴弓|琴弓]]琴弦间的[[静摩擦|静摩擦]]等于[[动摩擦|动摩擦]]，并且动摩擦在[[差速|差速]]增大的同时不断减小。另外一个例子是摩擦玻璃杯的边缘会发出声响。

自激振动在实际中是由振幅界定的，另外一种情况，当无止境给与刺激时，振幅也是无限的，系统将被破坏。

==参数振动==
当一个振动系统的参数（阻尼数，倔强系数）周期变化的时候，称作参数振动。因此在[[蒸汽机车|蒸汽机车]]中可以通过周期性的参数变化驱动系统持续运动。

==线性与非线性振动==
在描述振动系统的微分方程中，振动的单位和时间微分之间所有的关系为线性的，称为线性振动。反之称为非线性振动。非线性自由振动和周期刺激的非线性强迫振动不再是正弦状，而是高次[[谐波|谐波]]状。在实际意义上，强迫振动的共鸣关系是变化的，自激振动的振幅是受限的。

==单自由度和有限多个自由度的振动==
可以用一个振动单位完全描述的振动称为单[[自由度|自由度]]振动。例如'''平面的'''[[单摆|单摆]]。让单摆做空间运动，则是双自由度振动。当一个工程振动系统有多个振子而必须为描述每一个振子而建立一个坐标系统的话，即称作多自由度振动。如一个[[曲柄轴|曲柄轴]]的扭曲振动或在[[地震|地震]]中多[[塔楼|塔楼]]建筑的[[水平|水平]]振动。

"n"自由度振动可以通过"n"个二[[阶|阶]][[微分方程|微分方程]]描述。按照振动单位，与其一阶或/和二阶导数相关。线性振动系统可以通过所谓[[主坐标|主坐标]]借助一个[[缩放|坐标变形]]与此坐标的微分方程及其二阶导数相[[相关|耦合]]。多数情况下把一阶导数的影响作为[[不相关|不相关]]来考虑，也不是严重的错误。不相关微分方程可以确定系统的[[固有频率|固有频率]]

解微分方程后可以通过[[反向变换|反向变换]]得到原坐标系的[[时间|时间]]关系。

非线性振动系统中[[封闭形式|封闭形式]]的不相关是不可能的。存在一个[[近似|近似]]过程，使[[微分方程|微分方程]]组的[[线性化|线性化]]末端有[[重根|重根]]。

==无限多个自由度的振动==

[[真空振动|真空振动]]有无限个[[自由度|自由度]]和[[无限|无限]]个[[固有频率|固有频率]]，实际应用中用来描述工程上的[[弦|吊索]]，[[柱|柱]]，[[层|层]]和[[拱|拱]]。

==举例==

日常生活中典型的振动有[[石英钟|石英钟]][[钟摆|钟摆]]的摆动，[[秋千|秋千]]的摆动等等很多。严格来说[[原子|原子]]在[[晶体|晶体]]中的[[平衡振动|平衡振动]]，[[四季|四季]]变化，[[地球自转|地球自转]]，[[心跳|心跳]]和[[风|风]]中抖动的[[叶子|叶子]]都是振动。这里广义上指所有随[[时间|时间]]改变[[状态|状态]]的[[过程|过程]]。

一个[[单摆|单摆]]始于破坏[[平衡|平衡]]的[[能|能]]（如推动钟摆即给其[[势能|势能]]），使其具有[[初始速度|初始速度]]（[[动能|动能]]）。

此处的[[回复力|回复力]]即为将钟摆向下拉的[[重力|重力]]。在理想的无[[摩擦|摩擦]]条件下钟摆可以回复到原来初始的平衡位置，这时[[势能|势能]]完全转化为[[动能|动能]]，其势能在平衡时达到最小值。

[[单摆|单摆]]的振幅很小，可以近似看作[[谐振|谐振]]。
