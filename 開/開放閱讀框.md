'''開放閱讀框'''（英語：'''Open reading frame'''；縮寫：'''ORF'''；其他譯名：'''開放閱讀框架'''、'''開放讀架'''等）是指在給定的[[閱讀框架|閱讀框架]]中，不包含终止密碼子的一串序列。這段序列是生物個體的[[基因組|基因組]]中，可能作為[[蛋白質|蛋白質]]編碼序列的部分。[[基因|基因]]中的ORF包含並位於開始[[密碼子|編碼]]與終止編碼之間。由於一段[[DNA|DNA]]或[[RNA|RNA]]序列有多種不同讀取方式，因此可能同時存在許多不同的開放閱讀框架。有一些[[電腦程式|電腦程式]]可分析出最有可能编码蛋白质的序列。

== 例子 ==
一段5'-UC'''UAA'''AGGUCCA-3'序列。此序列共有3種讀取法：

# UCU AAA GGU CCA
# CUA AAG GUC
# '''UAA''' AGG UCC

由於UAA為終止密码子，因此第三種讀取法不具編譯出蛋白質的潛力，故只有前兩者為開放閱讀框架。

== 找ORF的软件 ==
在线寻找ORF的软件：[http://www.ncbi.nlm.nih.gov/orffinder/ ORF Finder]
=== 功能 ===
ORF Finder 被用来预测已存在的编码区的小基因序列。也可用于大规模的开放式阅读框寻找。
