《'''开展境内个人直接投资境外证券市场试点方案'''》俗稱為'''港股直通车'''、'''港股自由行'''，是[[中華人民共和国政府|中華人民共和国政府]]开放[[中国大陆|中国大陆]]境内个人直接对外[[证券|证券]]投资业务试点。居民个人可在试点地区通过相关渠道，以自有[[外汇|外汇]]或[[人民币|人民币]]购汇直接对外证券投资。而境外目标地暂定为[[香港|香港]]。

== 背景 ==
港股直通车，这一政策最初的设想来源于[[天津滨海新区|天津滨海新区]]与[[中银国际|中银国际]]的业务交流，双方提出由[[中国银行|中国银行]]开办这一业务。

== 日程 ==
# 2007年8月下旬，[[中国国家外汇管理局|中国国家外汇管理局]]宣布《开展境内个人直接投资境外证券市场试点方案》，开放中国散户资金自由行投资香港股票，人称「港股直通车」，试点初期只在[[天津|天津]]。
# 次日，传媒报导，内地投资人涌往[[中国银行|中国银行]]天津分行，查询及开户。 [[香港|香港]][[恒生指数|恒生指数]]急升。
# 2007年10月26日星期五，恒生指数突破三万点，以30,405点收市。
# 2007年11月4日，[[国务院总理|国务院总理]][[温家宝|温家宝]]发言，暂缓实行该方案。次日11月5日星期一，恒生指数急跌1526点，收28,942点。 <ref>[http://hk.biz.yahoo.com/071105/239/2ivip.html 温家宝总理发言，有关港股直通车的言论，港股急挫。] {{webarchive|url=https://web.archive.org/web/20071107072021/http://hk.biz.yahoo.com/071105/239/2ivip.html |date=2007-11-07 }}</ref>

==暂停原因==
国务院总理温家宝向香港传媒暗示“直通车”短期不会启动，主要是有四大考虑<ref>[http://www.hkheadline.com/news/headline_news_detail.asp?id=37138&section_name=wnn 「直通车」急煞停 温总：有四大考虑]</ref>：国内自由资金流动的法规监管、资金流入对香港及内地市场的影响、向内地股民进行[[风险管理|风险]]教育，以及咨询各方意见，以免内地资本市场出现大波幅及影响港股稳定。
外界一向认为，[[中央政府|中央政府]]一直不开动「直通车」，是基于内地人缺乏投资教育，盲目入市都可以赚钱，若投入完全陌生的境外市场，随时接来港股的「[[接火棒|火棒]]」，就会造成[[套牢股票|亏损]]。<ref>[http://paper.wenweipo.com/2007/08/31/HT0708310002.htm 港股直通车热炒 另类忧虑：谁接火棒？]</ref>

==传重提计划==
2008年1月末，内地多份主要报章昨日刊载外汇管理局官员的访问，提到正研究个人境外直接投资管理，被市场诠释为内地将于短期之内「排洪」，憧憬港股直通车快在落实，刺激恒指及相关受惠股份饱受美国次级按揭问题的港股从大跌市中大幅抽升。[[东亚银行|东亚银行]]主席[[李国宝|李国宝]]指「直通车」2008年4月至5月有望开通。不过，[[中国内地|中国内地]][[外管局|外管局]]否定「直通车」传闻，称现在还在考虑可行性问题，还没有考虑到时间表问题。2010年3月10日，天津市市长黄兴国介绍，天津市在港股直通车等创新金融领域做了大量基础工作，不过目前港股直通车没有立即启动迹象，港股直通车问题仍在研究当中。<ref>{{cite web |url=http://finance.people.com.cn/GB/11118435.html |language=中文 |title=天津市市长黄兴国：港股直通车无启动迹象 |publisher=新华网 |date=2010-03-11 |accessdate=2011-03-12 |deadurl=yes |archiveurl=https://web.archive.org/web/20160304081606/http://finance.people.com.cn/GB/11118435.html |archivedate=2016-03-04 }}</ref>2011年3月11日，天津市市长黄兴国表示，天津市已经做好准备，正在等待国家批复。
