'''舒城中学'''，始创于1902年，为[[安徽省|安徽省]]中等教育的三所源头学校之一。1952年正式定名为现名。于2012年入选第二届《中华百年名校》<ref>{{Cite web|url = http://www.top100zx.cn/DetailArticle.aspx?id=8988|title = 2012（第二届）中学百年名校论坛在京举办《中学百年名校》正式更名为《中华百年名校》|accessdate = 2012-07-16}}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref>{{Cite web|url = http://gk.shucheng.gov.cn/DocHtml/12694/2012/7/23/18555930109532.html|title = 舒城中学荣获全国“中华百年名校”称号|accessdate = 2012-07-23}}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

== 现状 ==
舒城中学校园占地面积近170亩，校园绿化率达到80%以上，共有3幢教学楼及3幢学生公寓，另有双层的餐厅。同时还配有科学馆、微机室、语音室、天文馆、图书馆、实验室等教学配套设施。校内还拥有一座塑胶跑道及人造草坪结构的标准运动场。学校现有教职工242人（截至2014年9月），其中专任教师202人，包括特级教师2人，硕士研究生24人，高级教师80人，中级教师93人。

== 教学情况 ==
学校现仅高中一个教学部。

== 校园环境 ==
校园内有多处标志性建筑：沧桑南楼；古雅仁风亭；浴德园；现代钢雕。
