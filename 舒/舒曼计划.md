'''舒曼计划（又稱舒曼宣言）'''是1950年5月9日[[法国外交部|法国外交部]]部长[[罗贝尔·舒曼|罗贝尔·舒曼]]在法国外交部驻地[[奥赛码头|奥赛码头]]时钟沙龙（{{lang|fr|Salon de l'Horloge}}）在一次记者招待会上公布的一个计划。在这个计划中他建议将[[德国|德国]]与法国的煤和钢铁生产融合到一起：
:“法国政府建议设立一个共同的高级公署（{{lang|fr|Haute Autorité}}）来管理法国和德国的煤和钢铁的生产。其它欧洲国家也可以参加这个机构的组织。”
[[File:Salon_de_l'Horloge.jpg|thumb]]

舒曼计划是法国外交部非常保密的一份文件，它是由[[让·莫内|让·莫内]]为首写出来的。只有德国总理[[康拉德·阿登纳|康拉德·阿登纳]]在文件被公布前数小时内被知会。阿登纳立刻表示同意。

该高级公署的迫切任务包括：
* 生产现代化，提高品質
* 向法国、德国和其它参加国的市场以同条件提供煤和钢铁
* 共同向其它国家出口
* 改善这些工人的生活条件

舒曼在美国、英国和法国外交部长共同在伦敦举行会议商讨新的欧洲政策策略前数天内公布了舒曼计划，虽然在此前其它国家的政治家不知道这个计划的存在，但是在伦敦会议上它成为讨论最多的文件。

== 背景 ==
由于[[冷战|冷战]]的加剧法国政府认识到它无法阻止[[德国再武装|德国再武装]]，也无法阻止德国政府获得其主权。同时[[西德|西德]]的经济发展也使得法国无法继续限制德国的煤和钢铁工业的发展。而法国本身也对德国的这些资源感兴趣，因此法国政府改变了它至此为止的压抑政策，提出这个合作政策来协调两国之间在这些重要资源上的合作，避免过去互相之间的竞争和不信任。

== 后来的发展 ==
1951年4月18日簽訂的[[巴黎條約_(1951年)|巴黎條約]]是基于舒曼计划制定的，根據本條約成立[[歐洲煤鋼共同體|歐洲煤鋼共同體]]。让·莫内也是煤钢共同体的首任主席（1952年至1955年）。1967年欧洲煤钢共同体和[[欧洲经济共同体|欧洲经济共同体]]以及[[歐洲原子能共同體|歐洲原子能共同體]]按照[[合并条约|合并条约]]合并为[[欧洲共同体|欧洲共同体]]。

== 舒曼宣言全文 ==
沒有與威脅的大小相應的創造性努力，便無法維護世界的和平。


一個有組織的、有生命力的歐洲所能為文明帶來的貢獻，在維護和平關係上不可或缺。在二十多年前就作為統一歐洲之先鋒的法國一直以來都將為和平服務當成一個重要的目標。統一的歐洲尚未達成，而我們經歷了戰爭。


統一的歐洲並非一蹴可幾，也不是透過單純的統合就能達成：這將會透過那些首先作出事實上團結的具體事實來實現。歐洲國家的統一需要消除德法兩國之間長達數世紀的對立。德國與法國必須站在第一線展開工作。


為此，法國政府提議在有限但決定性的條件下立即採取行動。


法國政府提議，將法國與德國的煤鋼生產整合由一個共同的高級機構管理，一個開放其他歐洲國家加入的組織。煤鋼生產的合併將立即確保為經濟發展建立共同基礎——歐洲聯邦的第一階段，並且改變各個長久以來致力於生產武器而成為最大受害者的地區的命運。


由此所達成的生產團結將表示，任何法國與德國之間的戰爭都不但不可想像，而且實際上也不可能。這個強大，且基於供應所有成員國於工業生產所必需的原物料之目的，對所有願意加入的國家開放的生產共同體，將為其經濟聯盟奠定真正的基礎。


這個生產將無差別、無例外地供給全世界，以便提高生活水準及促進和平工作。屆時，歐洲將能以更多的資源實踐其最重要任務之一：非洲大陸的發展。如此將可簡單且快速地統合對創建一個經濟共同體所必需之利益，包括將那些長久以來因為世仇而分隔的國家結合為更廣更深的共同體。


透過整合基礎產業以及建立一個將連繫法國、德國與其他成員國的新高級管理局，這項提議將為守護和平所必需的歐洲聯邦打下第一塊基石。


為了實踐這項明確的目標，法國政府已經準備好在以下基礎上進行協商。


要交付給聯合高級管理局的任務，便是在最短的期限內確保下列事項：生產的現代化與其品質的提升、法國及德國和其他成員國市場上鋼鐵與煤炭相同條件的供應、共同出口至其他國家的發展、以及在過程中產業工作者生活條件的衡平。


為了達成這些目標，有鑑於成員國實際上參差不齊的生產條件，必須暫時採取一定的預防措施，更精確來說是：應用生產與投資計畫、建立價格衡平機制、設立有助於生產合理化的自由兌幣基金等。在成員國之間煤炭和鋼鐵的進出口將立即免除關稅，並且不得基於不同運費作不同處理。漸漸地便能塑造可保障在最佳效率水準上最合理的生產分配的光景。


相對於一個透過有限作法來瓜分與剝削國內市場並追求長久高利潤的跨國壟斷企業聯盟，本項計畫中的組織將保障市場的融合以及生產的擴張。


在此所述的原則和關鍵的條約要點，應為一份由各國簽署並由各國國會通過的條約所規範。這場對於制定施行條例而言不可或缺的協商會議將有賴於一名由共同協議任命的仲裁人。這名仲裁人必須監督這項條約是否符合這些原則，並且在發生難以調和的對立僵局時決定一項終局的解決方案以供隨後採用。


這個負責整體行政職能的聯合高級管理局將由在平等的基礎上受各國政府任命的獨立個人所組成。主席則將經由各國政府共同協議選任，其決定將於法國、德國以及其他成員國內享有拘束力。而相應的配套措施也將保障對於高級管理局所作之決定提出抗議的可能性。


各個國家在聯合國的代表將被授命每年兩度向聯合國提交一份關於這個新機構活動的報告，特別是關於維持和平目標的方面。


高級管理局的設立絕對不會預先判斷企業的所有權問題。高級管理局在履行其任務時將考慮到授予國際魯爾區管制局的權力，任何對德國課予的義務亦同，只要這些權力義務仍然有效。
