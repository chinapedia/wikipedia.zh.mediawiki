'''共和党'''，[[中华民国|中华民国]]初年建立政党之一。成立于1912年5月9日[[上海张园|上海张园]]，由[[民社|民社]]、[[统一党|统一党]]、[[国民协进会|国民协进会]]、[[民国公会|民国公会]]、[[國民黨_(上海)|国民党]]等合并组成。其组织拥有国内支部27个，国外支部7个。党员约10万。[[黎元洪|黎元洪]]为理事长，[[张謇|张謇]]、[[章太炎|章太炎]]、[[伍廷芳|伍廷芳]]、[[那彦图|那彦图]]等4人为理事。该党党员先后成为阁员的有：[[熊希龄|熊希龄]]、[[范源濂|范源濂]]、[[张元奇|张元奇]]、[[应德闳|应德闳]]、[[章宗元|章宗元]]。参加该党的地方都督有[[黎元洪|黎元洪]]、[[蒋尊簋|蒋尊簋]]、[[朱瑞|朱瑞]]、[[庄蕴宽|庄蕴宽]]、[[吕公望|吕公望]]、[[赵惟熙|赵惟熙]]、[[杨增新|杨增新]]、[[胡景伊|胡景伊]]、[[程德全|程德全]]等人。不久，[[章太炎|章太炎]]辞去共和党理事，发宣言维持[[统一党|统一党]]。

== 成立緣起<ref>《共和黨資料選》 饒懷民整理 中南地區辛亥革命史研究會、武昌辛亥革命研究中心編. 《辛亥革命史叢刊》第8輯. 北京市：[[中華書局|中華書局]], 1991.09. 第171-188頁</ref> ==

共和黨由統一黨、民社、國民協進會、民國公會、國民黨五團體合併而成，於民國元年五月初九日在滬開成立大會，通過規約，選舉理事長、理事、幹事，並將五團名義同時取消，在京設立本部，迭經登報布告。

共和黨成於五大政團：
*(一)'''[[統一黨|統一黨]]'''，即前之中華民國聯合會，創辦者為章炳麟、張謇、[[程德全|程德全]]等，皆有功民國之人，其黨員咸一時材俊，今[[章炳麟|章炳麟]]以他故未加入。張、程諸先生均在本黨。
*(二)'''[[民社|民社]]'''，即武昌起事諸人，武漢舊有共進會，光復後，[[孫武_(民國)|孫武]]、[[張振武|張振武]]、 [[劉成禺|劉成禺]]、[[張伯烈|張伯烈]]等組織政黨，推黎元洪為領袖，其黨員皆共進會之創造民國者。
*(三)'''[[國民協進會|國民協進會]]'''，乃[[黃河|黃河]]以北最初發生之政黨，其中黨員成為才識閎達，熱心共和之士。
*(四)'''[[民國公會|民國公會]]'''，為[[揚子江|揚子江]]一帶革命團體之光復會中穩健分子及其他政客學子所組織。
*(五)'''[[國民黨_(上海)|國民黨]]'''，為江浙間樸學士人素昔從事教育及地方公益者所組織。

=== 上海合併之情形 ===
合併之議始於上海，民國元年4月15日在商學公會開第一次會議，蒞會者共有七政團，計<br />
*'''[[統一黨|統一黨]]'''代表[[孟森|孟森]]、[[黃雲鵬|黃雲鵬]]，
*'''[[民社|民社]]'''代表[[孫發緒|孫發緒]]、[[胡鈞|胡鈞]]、[[項驤|項驤]]，
*'''[[民國公會|民國公會]]'''代表[[陳敬第|陳敬第]]、[[黃群|黃群]]，
*'''[[國民協進會|國民協進會]]'''代表[[周大烈|周大烈]]、[[籍忠寅|籍忠寅]]，
*'''[[國民協會|國民協會]]'''代表[[張嘉璈|張嘉璈]]、[[張國淦|張國淦]]、[[袁思永|袁思永]]，
*'''[[國民黨_(上海)|國民黨]]'''代表[[沈彭年|沈彭年]]、[[張毓英|張毓英]]，
*'''[[共和建設討論會|共和建設討論會]]'''代表[[孫洪伊|孫洪伊]]、[[湯化龍|湯化龍]]、[[林長民|林長民]]。
自是日始，逐日開會討論黨名、黨義、政綱、組織諸問題，並將所有擬議條件陸續報告各團，經各團自行開會議決承認後，復由代表報告，其中惟共和建設討論會代表、國民協會代表報告本黨意見，對於理事問題稍有異議，於是七政團變為五政團，仍繼續開議，於4月24日始將規約決議並作成合併決議書。

== 黨義 ==

*保持全國統一，取國家主義。

*以國家權力扶持國民進步。

*應世界大勢以平和實利立國。

== 再次合併 ==
1913年5月又与[[民主党_(1912年中华民国)|民主党]]、[[统一党|统一党]]合并，组成[[进步党_(中国)|进步党]]，成为国会仅次于[[国民党_(1912–1913)|国民党]]的第二大党。主张国权主义，反对国民党的[[民权主义|民权主义]]。
