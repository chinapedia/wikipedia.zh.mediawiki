'''共产主义受害者纪念雕像'''（{{lang-en|'''Victims of Communism Memorial'''}}）位於[[美國|美國]][[華盛頓哥倫比亞特區|華盛頓哥倫比亞特區]]，座落於[[麻薩諸塞州|麻薩諸塞]]大道及[[新澤西州|新澤西]]大道附近的G街西北角。

2005年11月，[[美國國家首都規劃委員會|美國國家首都規劃委員會]]批准該紀念碑的設計，該設計乃仿製自1989年中國[[六四事件|八九民運]]中，由學生建造的[[民主女神像|民主女神像]]（原作在軍隊鎮壓平民時用坦克輾過倒塌），標明是為了「紀念超過一億名[[共產主義|共產主義]]之下的受害者」，「不容[[列寧|列寧]]、[[史達林|史達林]]、[[毛澤東|毛澤東]]、[[波爾布特|波爾布特]]、[[胡志明|胡志明]]、[[金日成|金日成]]、[[門格斯圖|門格斯圖]]和[[菲德尔·卡斯特罗|卡斯楚]]的暴行淡出歷史背景之中」<ref>{{cite web |url=http://www.victimsofcommunism.org/about/ |title=存档副本 |accessdate=2011-07-25 |deadurl=yes |archiveurl=https://web.archive.org/web/20110728155348/http://www.victimsofcommunism.org/about/ |archivedate=2011-07-28 }}</ref>。该纪念碑于2007年6月12日落成，美国总统[[乔治·沃克·布什|小布希]]在落成典礼上发表讲话<ref>{{cite web |url=http://appledaily.atnext.com/template/apple/art_main.cfm?iss_id=20070613&sec_id=4104&subsec_id=11866&art_id=7208496 |title=存档副本 |accessdate=2007-06-13 |deadurl=yes |archiveurl=https://web.archive.org/web/20070614075732/http://www1.appledaily.atnext.com/template/apple/art_main.cfm?iss_id=20070613&sec_id=4104&subsec_id=11866&art_id=7208496 |archivedate=2007-06-14 }}</ref>。當日同時也是已故總統[[罗纳德·里根|罗纳德·里根]]在[[柏林圍牆|柏林圍牆]]前發表《[[拆掉這堵牆|拆掉這堵牆]]》（{{lang|en|Tear down this wall}}）講話20週年。

== 背景 ==
1993年，[[美國眾議院|美國眾議院]]議員Dana Rohrabacher、[[美国参议院|參議院]]議員Claiborne Pell及Jesse Helms提出的興建紀念碑議案（H. R. 3000），於12月17日獲得一致通過，並由總統[[比爾·克林頓|比爾·克林頓]]簽字，成為《公共法》第905條第103-199節。由於紀念碑需延期成立，因此於1998年10月21日根據《公共法》第326條第105-277節，把工程順延至2007年12月17日以前。

紀念碑成立的目的，是要「把共产主義暴政的歷史教育下一代」，基金會於計劃第一階段需籌集資金及指導，現已籌得825,000[[美元|美元]]後。紀念碑奠基儀式已於2006年9月27日完成，當日有100多人參加。

該紀念碑的落成也引起中国当局的不滿，[[中华人民共和国外交部发言人|中国外交部發言人]][[秦剛|秦剛]]聲稱這是「出於[[冷戰|冷戰]]思維和政治需要，挑動意識形態、社會制度對立」<ref>[http://news.xinhuanet.com/world/2007-06/13/content_6238272.htm]</ref>。

== 揭幕儀式 ==
[[File:President_Bush_dedicates_the_Victims_of_Communism_Memorial.jpg|upright=1.2]]在揭幕式上的讲话]]
2007年6月12日，共产主义受害者纪念雕像正式揭幕。揭幕儀式上邀請了很多遭受共產黨政權迫害的人士作為嘉賓，例如[[越南|越南]]詩人[[阮志天|阮志天]]、美國籍反[[中共|中共]]社會活動家[[吳弘達|吳弘達]]和[[拉脫維亞|拉脫維亞]]記者Nijolė Sadūnaitė。

小布什總統在揭幕儀式上的演講中這樣形容受難者<ref name=youpai>[http://www.youpai.org/read.php?id=1689 布什在共产主义受难者纪念碑落成仪式上的演讲]</ref>：
{{cquote|这些牺牲者的幽灵常徘徊在历史当中——在他们背后，还有数百万或更多的籍籍无名的人死于共产主义的毒手。他们有在[[斯大林|斯大林]]的[[烏克蘭大饑荒|大饥荒]]中活活饿死的无辜的[[乌克兰人|乌克兰人]]，有死于斯大林[[大清洗|清洗]]的[[俄国人|俄国人]]，有将全部家当装上牛车背井离乡被流放到[[北极|北极]]这一苏联共产主义死亡[[集中营|集中营]]的[[立陶宛|立陶宛]]人、[[拉托维亚|拉托维亚]]人和[[爱沙尼亚|爱沙尼亚]]人。有在[[大跃进|大跃进]]和[[文化大革命|文化大革命]]中无辜受害的[[中国人|中国人]]。有在[[波尔布特|波尔布特]]的[[杀戮战场_(柬埔寨)|杀戮战场]]中被屠杀的[[柬埔寨|柬埔寨]]人，有为了投奔自由试图挖掘[[柏林圍墙|柏林圍墙]]而被射杀的[[德意志民主共和國|东德]]人，有在[[卡廷大屠殺|卡廷森林]]被屠杀的[[波兰|波兰]]人，有在“[[红色恐怖|红色恐怖]]”中被屠杀的[[埃塞俄比亚|埃塞俄比亚]]人，有在[[尼加拉瓜|尼加拉瓜]][[桑地诺民族解放阵线|桑地诺]]独裁政权中被谋杀的[[米斯基托人|摩斯基多印地安人]]，有为逃离暴政而溺死在海上的[[古巴|古巴]]人。我们从来都不知道那些死难者的人的名字，但在这个神圣的地方，无名的共产主义受难者的名字将被祭献于历史，永远被铭记。}}

小布什總統接著將[[恐怖主義|恐怖主義]]、[[激進主義|激進主義]]一齊列為美國今日必須要面對的敵人<ref name=youpai/>：
{{cquote|那些袭击我们国家的[[恐怖分子|恐怖分子]]和激进分子都是冷血的意识形态的追随者，他们鄙视[[自由|自由]]，压制异己，野心勃勃，谋求建立[[極權主義|极权统治]]。同共产党一样，我们新的敌人相信，可以通过滥杀无辜来实现他们激进的目标。同共产党一样，我们新的敌人对自由人民不屑一顾，声称我们这些享有自由的人是懦弱的，声称我们缺乏捍卫自由生活方式的决心。同共产党一样，那些暴力的[[伊斯蘭法西斯主義|伊斯蘭法西斯主義]]的追随者注定要灭亡！}}
