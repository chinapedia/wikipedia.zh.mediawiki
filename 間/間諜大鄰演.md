《'''-{zh-hk:兩公婆決戰特務王;zh-cn:邻家大贱谍;zh-tw:間諜大鄰演;}-'''》（{{lang-en|'''''Keeping Up with the Joneses'''''}}）是一部2016年的美國[[動作片|動作]][[喜劇片|喜劇片]]，由[[格雷格·莫托拉|格雷格·莫托拉]]執導，[[查克·葛里芬納奇|查克·葛里芬納奇]]、[[喬·漢姆|喬·漢姆]]、[[艾拉·費雪|艾拉·費雪]]與[[蓋兒·加朵|蓋兒·加朵]]主演。該片於2016年10月21日在美國上映。

== 劇情 ==
故事敘述一對郊區夫婦發現他們看似完美的新鄰居其實是政府的特務，為此共同捲入了國際間諜的危險任務中。

== 發行 ==
該片原定於2016年4月1日在美國上映<ref>{{cite web|url=http://deadline.com/2015/03/eddie-the-eagle-fox-taron-egerton-hugh-jackman-1201392499/|title=‘Eddie The Eagle’ Has Landed At Fox; Biopic Starring Taron Egerton, Hugh Jackman Set For 2016|website=[[Deadline.com|Deadline.com]]|first=Erik|last=Penderson|date=March 13, 2015|accessdate=May 29, 2016}}</ref>，後延遲至10月21日<ref>{{cite web|url=http://deadline.com/2016/05/warner-bros-maze-runner-the-death-cure-dylan-obrien-1201764118/|title=‘Maze Runner: The Death Cure’ Release Moved To 2018 As Dylan O’Brien Recovers|website=[[Deadline.com|Deadline.com]]|first=Ross|last=A Lincoln|date=May 27, 2016|accessdate=May 29, 2016}}</ref>。臺灣原定2016年12月2日上映，但後來撤了檔。

=== 評價 ===
《間諜大鄰演》獲得負面居多的評價。[[爛番茄|爛番茄]]基於76條評論，新鮮度17%，平均得分為4.5/10<ref>{{cite web|url=https://www.rottentomatoes.com/m/keeping_up_with_the_joneses |title= Keeping Up with the Joneses (2016) |publisher=Rotten Tomatoes |accessdate=October 20, 2016}}</ref>。[[Metacritic|Metacritic]]得分31（滿分100），評價不佳<ref>{{cite web|url=http://www.metacritic.com/movie/keeping-up-with-the-joneses |title=Keeping Up with the Joneses |publisher=[[Metacritic|Metacritic]]|accessdate=October 19, 2016}}</ref>。

A Madea Halloween' at box office |work= [[Los_Angeles_Times|Los Angeles Times]]}}</ref>。美國首映週末票房收入僅560萬美元，位居當週票房排名第七。
