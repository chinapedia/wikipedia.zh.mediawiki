{{Vgname|勇者鬥惡龍 節奏劇場|ja=シアトリズム ドラゴンクエスト，Theatrhythm Dragon Quest}}是[[任天堂3DS|任天堂3DS]]獨佔遊戲，預定於2015年3月26日在日本發行。這是[[勇者鬥惡龍系列|勇者鬥惡龍系列]]首款[[音樂遊戲|音樂遊戲]]<ref>{{Cite web |date=2014-12-10 |url=http://www.famitsu.com/news/201412/10067540.html |title=『ドラゴンクエスト』シリーズ初の音楽ゲーム『シアトリズム ドラゴンクエスト』が発売決定！ |work=ファミ通.com |accessdate=2015-02-26}}</ref>。

== 玩法 ==

《勇者鬥惡龍 節奏劇場》收錄勇者鬥惡龍十部本傳遊戲的60餘首歌曲。玩家可在挑戰模式中解鎖歌曲<ref>{{Cite web |date=2015-01-29 |url=http://www.famitsu.com/news/201501/29069783.html |title=『シアトリズム ドラゴンクエスト』収録楽曲ほか続報をお届け |work=ファミ通.com |accessdate=2015-02-26}}</ref>，並在賭場解鎖操作角色<ref>{{Cite web |date=2015-02-26 |url=http://www.famitsu.com/news/201502/26071929.html |title=『シアトリズム ドラゴンクエスト』"すごろく場"など新情報が一挙判明 |work=ファミ通.com |accessdate=2015-02-26}}</ref>。

== 评价 ==

日本杂志《[[Fami通|Fami通]]》打出35/40分。中国大陆杂志《[[游戏机实用技术|游戏机实用技术]]》给出25/30，称游戏充分融入了勇者斗恶龙元素，但有缺乏新意，是“换了套《勇者斗恶龙》外衣”的《[[最终幻想_节奏剧场|最终幻想 节奏剧场]]》<ref>{{cite journal | title = 黄金眼 | journal = 游戏机实用技术 | issue = 368 | year = 2015 | page = 29 }}</ref>。
