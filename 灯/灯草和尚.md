《'''灯草和尚'''》（又名：《'''灯草和尚传'''》、《'''和尚缘'''》、《'''灯花梦全传'''》），作者署名为[[元朝|元朝]][[临安|临安]][[高明_(元朝)|高则诚]]。具体作者学术界尚有疑问和争议。

== 内容 ==
全书讲述[[元朝|元朝]]末年，一个叫杨知县的妻子汪氏及其婢女暖玉、女儿长姑和女婿李可白、灯草幻化的[[和尚|和尚]]、[[道士|道士]]周自如之间混乱的多角[[性关系|性关系]]故事<ref name="multipletx1 ">{{cite web|url=http://news.hainan.net/newshtml08/2008w5r26/400131f1.htm|title=看看中国古代的和尚是如何偷情的|accessdate=2008-05-26|work=|publisher=海南新闻网|language=zh-hans}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

== 思想 ==
{{fact|全书旨要在于宣扬[[性欲|性欲]]的合理性和反对[[封建礼教|封建礼教]]。}}

== 版本 ==
全书一共六卷十二回。

== 禁毁 ==
[[清朝|清朝]][[统治者|统治者]]为了稳定政权，控制文化，列出了各种各样的[[禁书|禁书]]，其中，[[性文学|性文学]][[小说|小说]]之首就是《灯草和尚》。其次有六大禁书：《[[载花船|载花船]]》、《[[蜃楼志|蜃楼志]]》、《[[品花宝鉴|品花宝鉴]]》、《[[闹花丛|闹花丛]]》、《[[株林野史|株林野史]]》、《[[綠野仙蹤_(清朝小說)|綠野仙蹤)]]》<ref name="multipletx21 ">{{cite web|url=http://book.qq.com/a/20070208/000024.htm |title=清朝被查禁的六大两性小说 情色描写露骨 |accessdate=2007年2月8日 |work= |publisher=腾讯网 |language=zh-hans |deadurl=yes |archiveurl=https://web.archive.org/web/20070310175945/http://book.qq.com/a/20070208/000024.htm |archivedate=2007年3月10日 }}</ref>。
