[[File:Habsburg_Persian_alliance.jpg|thumb]]和[[伊斯瑪儀一世|伊斯瑪儀一世]]率先試圖建立哈布斯堡-波斯同盟抵抗[[奧斯曼帝國|奧斯曼帝國]]。]]

為了對抗共同的敵人[[奧斯曼帝國|奧斯曼帝國]]，[[哈布斯堡君主國|哈布斯堡君主國]]與[[波斯帝國|波斯帝國]]在16世紀組成'''哈布斯堡-波斯同盟'''<ref>{{harvnb|Kearney|2004|p=112}}</ref>。

== 首次接觸 ==

在對抗突厥人的問題上，波斯沙阿[[伊斯瑪儀一世|伊斯瑪儀一世]]一直與羅馬帝國的[[查理五世_(神聖羅馬帝國)|查理五世]]和匈牙利的[[拉約什二世|拉約什二世]]保持著交流。後來成為西班牙國王的查理五世在1516年至1519年間向波斯派遣使節，冀能建立同盟關係<ref>{{harvnb|Anderson|1998|p=224}}</ref>。拉約什二世亦以同一目的在1516年也派遣一名叫[[彼得魯斯·德·蒙特·利巴諾|彼得魯斯·德·蒙特·利巴諾]]（Petrus de Monte Libano）的修士到波斯<ref>{{harvnb|Jackson|Lockhart|1986|pp=381-382}}</ref>。

出使的成效不得而知，但伊斯瑪儀一世在1523年以拉丁文寫了一封信函給查理五世，提議組成軍事行動對抗奧斯曼人，並再以彼得魯斯·德·蒙特·利巴諾為使者。使節在1524年3月來到[[布爾戈斯|布爾戈斯]]向查理五世請求「組成同盟對抗突厥人」<ref>{{harvnb|Gmara|2009|p=70}}</ref>，但並沒有得出任何具體的結論<ref name="Reston(2009)359">{{harvnb|Reston|2009|p=359}}</ref>。查理五世在原則上同意組成同盟，但伊斯瑪儀一世在1524年逝世，使協議告吹。

== 奧斯曼-薩非戰爭 ==
[[File:Husain_Ali_Beg.jpg|thumb]]的使節侯賽因·阿里·比格。]]

1529年2月18日，出於對[[維也納之圍|奧斯曼帝國侵攻維也納]]的擔憂，查理五世在[[托萊多_(西班牙)|托萊多]]致函予伊斯瑪儀一世，請求波斯以軍事手段分散奧斯曼帝國的注意力，而伊斯瑪儀一世已於1524年逝世，王位由[[塔赫瑪斯普一世|塔赫瑪斯普一世]]繼承<ref name="Reston(2009)359"/><ref name="Vehse(2004)71">{{harvnb|Vehse|2004|p=71}}</ref>。傳達這個訊息的是一位騎士巴爾比，建立同盟的目的是要在兩年內使兩國分別由東西兩面夾擊奧斯曼帝國<ref>{{harvnb|Kearney|2004|p=112}}</ref>，塔赫瑪斯普一世也向查理五世表達了「善意」<ref name="Vehse(2004)71"/>。雙方已決定分別進攻奧斯曼帝國，巴爾比花了超過一年時間才回國，但當時的情況已有所不同，由於受到[[昔班王朝|昔班]][[烏茲別克族|烏茲別克人]]叛亂的影響，波斯被迫與奧斯曼帝國議和<ref>{{harvnb|Papadopoulo|1979|p=594}}</ref>。

差不多在同一時間，神聖羅馬帝國皇帝[[斐迪南一世_(神聖羅馬帝國)|斐迪南一世]]也派遣使者到波斯，但沒有取得任何成效。他在1532年及1533年也派遣過使者。即使在漫長的[[奧斯曼-薩非戰爭_(1532年-1555年)|奧斯曼-薩非戰爭]]裡，雙方也保持著交流<ref>{{harvnb|Lambton|Lewis|1978|p=330}}</ref>。自此以後，奧斯曼帝國一旦入侵歐洲，他們的領土東部便可能受到波斯的襲擊，迫使奧斯曼帝國君主蘇萊曼要火速返回首都<ref>{{harvnb|Garnier|2008|p=16}}</ref>。

哈布斯堡的敵人法國[[法蘭索瓦一世|法蘭索瓦一世]]與[[蘇萊曼一世_(奧斯曼帝國)|蘇萊曼一世]]在1536組成[[法國-奧斯曼帝国联盟|法國-奧斯曼帝国联盟]]，以應對哈布斯堡的威脅<ref>{{harvnb|Topping|2004|p=26}}</ref>。1547年，[[蘇萊曼一世_(奧斯曼帝國)|蘇萊曼一世]]入侵波斯，法國派遣使者[[加布里埃爾·德·呂茨|加布里埃爾·德·呂茨]]（Gabriel de Luetz）協助奧斯曼帝國作戰<ref>{{harvnb|Jackson|Lockhart|1986|p=382}}</ref>，他向蘇萊曼一世給予了一些軍事建議，如在[[凡城圍城戰|凡城圍城戰]]裡提議大炮放置的位置。

== 再度接觸 ==
[[File:Persian_embassy_at_the_Quirinale.jpg|thumb]][[奎里納爾宮|奎里納爾宮]]，顯示波斯使團在羅馬拜訪[[保祿五世|保祿五世]]。]]

波斯在[[奧斯曼-波斯戰爭|奧斯曼-波斯戰爭]]裡五度入侵奧斯曼帝國，每次都能在一定程度上削弱了奧斯曼帝國，而且能夠在奧斯曼帝國忙於歐洲戰事的時候開啟第二條戰線，以呼應哈布斯堡。這使哈布堡的壓力得以舒緩，顯然是這個同盟的策略<ref name="Reston(2009)359"/>。

波斯經歷了一段紛爭時期後，奧地利與波斯在1593年恢復聯繫，[[布拉格|布拉格]]的[[魯道夫二世_(神聖羅馬帝國)|魯道夫二世]]通過身處[[莫斯科|莫斯科]]的波斯官員向[[阿拔斯一世_(波斯)|阿拔斯一世]]傳達訊息，英國冒險家[[安東尼·舍利|安東尼·舍利]]（Anthony Shirley）也在這個同盟裡起著穿針引線的作用<ref>{{harvnb|Savory|2008|p=109}}</ref>。到17世紀，類似的外交活動依然持續活躍，以抗衡奧斯曼帝國，如[[波斯使團出使歐洲_(1599年至1602年)|1599年至1602年波斯出使歐洲]]及[[波斯出使欧洲使团_(1609年-1615年)|1609年至1615年波斯出使歐洲]]。
