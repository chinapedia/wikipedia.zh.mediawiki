'''哈图沙'''（[[赫梯楔形文字|赫梯楔形文字]]：{{Cuneiform|𒌷𒄩𒀜𒌅𒊭}} {{unicode|''Ḫa-at-tu-ša''}}；{{lang-tr|Hattuşaş}}）是古代[[赫梯王国|赫梯王国]]的首都，位于[[土耳其|土耳其]]首都[[安卡拉|安卡拉]]东偏北约164千米乔鲁姆省松古尔卢地区的勃尕卡尔村，在[[克澤爾河|克澤爾河]]弯道环绕的区域。

[[联合国教科文组织|联合国教科文组织]]於1986年將哈圖沙列入[[世界遗产|世界遗产]]名录。

== 地理位置 ==

城市遗址被农田、山坡草地和小片[[森林|森林]]所包围，古代森林面积可能更大，周围农田为城市提供了[[小麦|小麦]]、[[大麦|大麦]]和[[豆|豆]]类[[粮食|粮食]][[食品|食品]]，森林提供了建筑[[材料|材料]]，居民的主要[[服装|服装]]材料是[[羊|羊]]毛，放牧的牲畜也提供了主要的肉食。贵族们可能还可以在森林中以猎[[鹿|鹿]]为乐。[[河流|河流]]水量太小，不足以通航，运输主要依靠陆路交通。

== 早期历史 ==
[[File:Yazilikaya.12gods.jpg|thumb]]
在公元前6000年这里已经有人类居住的痕迹，大约在公元前2000年，赫梯人已经在这里定居，公元前19世纪至前18世纪，[[亚述|亚述]]商人在这里建立了贸易点和居民区，考古发现有[[楔形文字|楔形文字]]的铭文开始引入到哈图沙。规模之大，被誉为当时中东四大帝国之一。

大约公元前1700年，哈图沙曾被烧毁，可能是被[[阿尼塔|阿尼塔]]国王攻陷的，他并立了一个碑文说：“我在夜晚用武力攻陷这座城市，在这里撒播了杂草的种子，我以后如果有任何国王想重建哈图沙，必将会被天神所击垮。”
[[File:Hattusa.yenicekale.jpg|thumb]]

== 赫梯帝国 ==
只过了一代人的时间，一位[[西臺|赫梯人]]国王到达这里，建立都城，自称[[哈图西里一世|哈图西里一世]]，是有27位现在已知名称的赫梯王国的第一位国王。后来由于外敌入侵，赫梯王国几次迁都，最后[[穆尔西里三世|穆尔西里三世]]将首都迁回哈图沙，以后哈图沙直到帝国于前1180年灭亡以前一直是赫梯王国的首都。

在最繁荣时期，哈图沙面积达1.8平方千米，由[[苏庇路里乌玛一世|苏庇路里乌玛一世]]（约公元前1375年-前1335年）建造的内外两道城墙包围，内城有0.8平方千米，有宫殿和庙宇，外城城门在内城南若1千米处，有勇士、[[狮子|狮子]]和[[斯芬克斯|斯芬克斯]]的[[浮雕|浮雕]]装饰。外城内有四座寺庙，居民房屋是用木料和土坯搭建的，现在已经只留有地基，估计有4万至5万居民，其中有1/3居住在内城。目前还有石建的宫殿和庙宇遗迹。

城北墙外有一座石刻陵墓和有宏大的铭文巨石的石窟神庙，庙墙上有精美的浮雕；城西北有供奉[[太阳|太阳]]神和风暴神的神庙。

大约在公元前1180年，随着赫梯王国的灭亡，哈图沙也被毁灭和遗弃，直到公元前1世纪，才有一些其他民族的人在这里定居。

== 考古发现 ==
从1906年开始，[[德意志帝国|德意志帝国]]考古协会开始在这里发掘，发现了许多[[楔形文字|楔形文字]]碑片，包括有契约、[[法律|法律]]、预言等，最重要的发现是大约在公元前1283年赫梯王国和[[古埃及|古埃及]][[第十九王朝|第十九王朝]]法老[[拉美西斯二世|拉美西斯二世]]的和平条约，现在有一个复制品作为最早的国际和平条约范例放在[[纽约|纽约]][[联合国|联合国]]总部。

在这里已经发现了大约30,000件泥版文书，大部分存放在[[安卡拉|安卡拉]]和[[伊斯坦布尔|伊斯坦布尔]]的[[博物馆|博物馆]]内。
