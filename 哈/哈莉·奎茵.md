'''哈莉·奎茵'''（{{lang-en|Harley Quinn}}，又譯作'''小丑女'''），原名哈琳·法蘭西斯·奎澤（{{lang-en|Harleen Frances Quinzel}}），是一位於[[DC漫畫|DC漫畫]]旗下的[[虛構|虛構]][[超級反派|超級反派]]角色，存在於[[DC宇宙|DC宇宙]]中，最初在《[[蝙蝠俠_(動畫影集)|蝙蝠俠動畫]]》<ref>《The Bat Man Adventure》</ref>（1992年）中登場。

2008年，在[[IGN|IGN]]的漫畫書惡棍全部列表中她排名第45<ref>{{cite web| year = 2009| title = Top 100 Comic Book hero| url = http://comics.ign.com/top-100-villains/index.html}}</ref><ref>{{cite web| url = http://comics.ign.com/top-100-villains/45.html| title = Harley Quinn is Number 45| work = [[IGN|IGN]]| publisher = [[News_Corporation|News Corporation]]| deadurl = yes| archiveurl = https://web.archive.org/web/20130716154409/http://comics.ign.com/top-100-villains/45.html| archivedate = 2013-07-16}}</ref>，在《Comics Buyer's Guide》的100位最性感女性漫畫人物中排名第16位<ref>{{cite book| last = Frankenhoff| first = Brent| authorlink = Brent Frankenhoff| title = Comics Buyer's Guide Presents: 100 Sexiest Women in Comics| publisher = [[Krause_Publications|Krause Publications]] | year = 2011| page = 19| url = | isbn = 1-4402-2988-0}}</ref>。

== 人物 ==
哈莉·奎茵是蝙蝠俠的頭號敵人「[[小丑_(角色)|小丑]]」的跟班，也是「[[毒藤女|毒藤女]]」的閨蜜，而哈莉·奎茵的「抗毒體質」就是毒藤女賦予的。

哈莉·奎茵原本是一名「[[阿卡漢精神病院|阿卡漢精神病院]]」的心理醫生，但就在她治療小丑的期間，卻對小丑的過去深感同情且被感動，從此她便開始無法自拔的迷戀起小丑這個人，之後哈琳·奎澤更是為了愛而開始幫助小丑逃離阿卡漢精神病院，最後哈琳·奎澤也跟著離開阿卡漢，並開始跟著穿上小丑的服飾成為後來的「哈莉·奎茵」。
