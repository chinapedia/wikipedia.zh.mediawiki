'''哈兹拉特苏丹清真寺'''是[[哈萨克斯坦|哈萨克斯坦]]首都[[阿斯塔纳|阿斯塔纳]]的一座清真寺。由[[努尔苏丹·纳扎尔巴耶夫|努尔苏丹·纳扎尔巴耶夫]]命名为哈兹拉特苏丹，意为“神圣的苏丹”。哈兹拉特苏丹也是[[艾哈迈德·亚萨维|霍贾·艾哈迈德·亚萨维]]的尊称<ref name=huanqiu/>。

==简介==
该清真寺于2009年6月动工，2012年7月6日12时30分正式启用，其间还发生过造成1人死亡的火灾。属于传统的伊斯兰风格建筑，大量使用了[[哈萨克族|哈萨克族]]装饰图案。殿堂地面全部用整洁的大理石铺就，地上铺满各式花毯<ref name=huanqiu>{{cite news |language = 中文 |author =  |coauthors =  |url = http://go.huanqiu.com/asia/2013-06/4031321.html |title = 探寻中亚最大清真寺 哈兹拉特苏丹 |publisher = [[环球时报|环球时报]] |pages =  |date = 2013-06-17 |accessdate = 2015年12月28日}}</ref>。

哈兹拉特苏丹清真寺占地面积超过11公顷，拥有可容纳745辆汽车的开放式停车场，以及5座大型喷泉，建筑面积17700平方米，可容纳5000人至1万人同时做礼拜<ref>{{Cite web |url = http://www.crntt.com/doc/1047/3/5/1/104735182.html?coluid=209&kindid=1290&docid=104735182&mdate=0706003541 |title = 中评镜头：走近哈萨克斯坦最大清真寺 |author =  |date = 2017-07-06 |publisher = 中评网 |language =  |accessdate =  |quote =  }}</ref>。该清真寺对游客开放，寺内专门为女游客提供了着装，参观的女士可以在门口处穿上长袍和披肩。清真寺大楼分三层，分别有礼拜大厅，图书馆，新媒体设备齐全的教室和用于开斋和慈善晚宴等宗教活动的餐厅等设施<ref>{{cite news |language = zh-hant |author = 努尔黑沙 |coauthors =  |url = http://finance.people.com.cn/BIG5/n/2014/0908/c387602-25620911.html |title = 摩登城市中的清真寺——哈茲拉特蘇丹清真寺 |publisher = [[人民網|人民網]] |pages =  |date = 2014-09-08 |accessdate = 2015-12-28}}</ref>。
