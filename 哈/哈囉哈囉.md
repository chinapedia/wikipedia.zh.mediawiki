'''哈囉哈囉'''（{{lang-tl|Halo-halo}}）是一種把甜豆、果凍等東西一同摻在沾著煉奶的碎冰裡食用的[[菲律賓|菲律賓]]甜點。<ref>{{Cite web |url=http://foodcitations.com/halo-halo-the-best-way-to-cool-down-this-summer/ |title=Halo-Halo: The best way to cool down this Summer |work=Food Citations |author=Ringo |date=2012年3月23日 |accessdate=2015年5月20日}}</ref>在[[他加祿語|他加祿語]]裡哈囉哈囉有「把東西混合在一起」的意思。<ref>{{Cite web |url=http://www.nytimes.com/2010/07/18/us/18bcintel.html?_r=0 |title=Halo-Halo |work=[[紐約時報|紐約時報]] |author=HANK PELLISSIER |date=2010年7月17日 |accessdate=2015年5月20日}}</ref>

== 起源 ==
此甜點的實際起源不明確，有一說是在[[二次大戰|二次大戰]]前有個來自[[日本|日本]]的移民在菲律賓販售混合著甜豆、牛奶的刨冰，該點心受到當地人民的歡迎後；菲律賓人也開始如法炮製並添加水果等配料方面的改良。<ref name="ffmagazine">{{Cite web |url=http://ffemagazine.com/filipino-icon-halo-halo/ |title=Filipino Icon: Halo-Halo |work=FFE Magazine |author=FFE PKM Staff |date=2012年12月2日 |accessdate=2015年5月20日}}</ref>

== 方法 ==
製作哈囉哈囉的方式是將碎冰放進容器裡淋上煉奶或椰奶後；灑上像是[[紅豆|紅豆]]、[[鷹嘴豆|鷹嘴豆]]、椰肉、水果切片、果凍等各式五花八門的甜零嘴，而在最後成品上加上一球[[冰淇淋|冰淇淋]]也是常見的方法。<ref name="ffmagazine"/>
