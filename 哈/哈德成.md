'''哈德成'''（{{bd|1888年||1943年|10月25日}}）经名'''希拉伦丁'''（Hilal al—Din，意为宗教的新月），字'''国桢'''，[[回族|回族]]，[[陕西省|陕西省]][[南郑县|南郑县]]人，[[中华民国|中华民国]]大[[阿訇|阿訇]]、经学教育家，与[[王静斋|王静斋]]、[[马松亭|马松亭]]、[[达浦生|达浦生]]并称中国现代“四大阿訇”<ref name=ah>{{Cite web |url = http://www.chinaislam.net.cn/cms/zjjy/jsrw/201205/25-685.html|title = 中国现代伊斯兰教四大阿訇之哈德成|author =  |date = 2012-05-25 |publisher = 中国伊斯兰教协会 |language =  |accessdate =  |quote =  }}</ref><ref name=ma>马全仁，我国穆斯林学者哈德成，阿拉伯世界1984年03期</ref>。

==生平==
哈德成的原籍是陕西省南郑县，早年随父亲哈希龄迁居[[上海|上海]]。自幼学习儒书，并随父亲学习[[阿拉伯语|阿拉伯语]]。16岁到[[江苏|江苏]][[镇江|镇江]]以及[[河南|河南]]向伊斯兰教经师学习。后来任上海[[浙江路清真寺|浙江路清真寺]]（俗称“外国寺”）教长，其间自学阿拉伯语、[[波斯语|波斯语]]、[[英语|英语]]、[[乌尔都语|乌尔都语]]。1913年到[[麦加朝觐|麦加朝觐]]，1914年回国，发起成立协兴公司，利用该公司开展海外贸易，盈利用来培养中国伊斯兰教人材。1919年，任驻[[锡兰|锡兰]]（今[[斯里兰卡|斯里兰卡]]）、[[埃及|埃及]]经理。1924年回到上海，和[[马刚侯|马刚侯]]等人筹组“[[中国回教学会|中国回教学会]]”，并且出版该会《月刊》，还翻译《[[古兰经|古兰经]]》，先后出版《古兰经》第1至3卷中文译本。1928年，和[[达浦生|达浦生]]等人创办上海伊斯兰师范学校，并派优秀毕业生留学埃及<ref name=ah/><ref name=ma/>。

1937年[[抗日战争|抗日战争]]全面爆发后，哈德成在[[上海租界|上海租界]]筹办慈善事业。1941年化名“马国成”，独自逃到[[重庆|重庆]]，并谢绝国民政府聘任，赴[[云南省|云南省]][[沙甸|沙甸]]，帮助其学生[[马坚|马坚]]翻译全本《古兰经》，并且在养正阿拉伯语专科学校高级班任教<ref name=ah/>。

1943年10月25日，哈德成在沙甸病逝，享年56岁<ref name=ah/><ref name=ma/>。
