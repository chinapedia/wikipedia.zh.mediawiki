《'''哈利·波特与火焰杯'''》（{{lang-en|''Harry Potter and the Goblet of Fire''}}）是[[哈利波特|哈利波特]]電影系列的第四集，由[[麥可·紐威爾|迈克尔·尼维尔]]导演，於2005年11月18日上映。这部电影长度为157[[分钟|分钟]]，比系列电影的[[哈利·波特与密室_(电影)|第二部]]短3分钟。在[[美国|美国]]，这部电影被评为'''PG-13'''级别，在[[英国|英国]]和[[爱尔兰|爱尔兰]]是'''12A'''级别，在[[澳大利亚|澳大利亚]]是'''M15'''级别。上映三天后，北美票房总额达到1.03亿[[美元|美元]]，為《'''[[哈利·波特|哈利·波特]]'''》系列电影中最高的首映记录。

== 故事大纲 ==
:<small>可参见{{哈利波特4}}或[[哈利·波特与火焰杯_(电影)#电影改编|本条目“电影改编”一节]]</small>

电影开始的画面是一条恐怖巨大的蛇（[[佛地魔|佛地魔]]的宠物）缓慢地匍匐在一个荒凉的墓地和荒凉的庄园（里德尔府）中。管理人员法蘭克·布萊斯发现了里德尔房间中有光芒出现，他认定有人闯入了庄园，便进入了房屋，登上楼梯。他听到楼上中有三个人谈话，他走到更近的地方，画面中出现了[[彼得·佩迪魯|虫尾]]和一个陌生人（从后面得知他是[[小巴堤·柯羅奇|小巴堤·柯羅奇]]），他们正在和[[伏地魔|伏地魔]]（没有显现）谈话。那条蛇爬过管理员到达楼上，用爬說语和伏地魔说一个“[[麻瓜|麻瓜]]”管理者正在楼梯上，伏地魔命蟲尾用索命咒“[[啊哇呾喀呾啦|啊哇呾喀呾啦]]”杀掉了管理者。

时空转换，[[哈利·波特_(小說角色)|哈利·波特]]在[[榮恩·衛斯理|榮恩·衛斯理]]的家中，從夢中驚醒，[[赫敏·格兰杰|赫敏·格兰杰]]来向他问好。韦斯莱家（韦斯莱先生、罗恩、金妮、弗雷德、乔治）、哈利、赫敏与迪哥里父子（阿莫斯·迪戈里和儿子[[塞德里克·迪戈里|塞德里克·迪戈里]]）在拂晓之时就来到了魁地奇世界杯比赛地。他们是通过触摸一个被设置成[[门钥匙|门钥匙]]（可以实现瞬间传送的魔法物品）来到比赛地点旁边的荒野上的。來自各國的球迷盛裝出席（比如说可以看到一面[[意大利|意大利]]旗帜，还有身穿[[部落|部落]]服装的[[非洲人|非洲人]]。）而比赛场地是一个巨大的球场。这一段非常简短，只能看到两队（爱尔兰和保加利亚）入场介绍，最明显的球星是保加利亚队找球手[[威克多尔·克鲁姆|威克多尔·克鲁姆]]，一出場現場就歡騰無比。

世界盃过后宿营地出现一阵骚乱，戴着面具的食死人出现，点燃了帐篷，在一陣仓皇之中，哈利与朋友们分开並跌撞在地上，昏過去了。过一阵宿营地烧光了的时候，小巴蒂·克劳奇来了，并且在漆黑的夜晚中释放了[[黑魔标记|黑魔标记]]，哈利醒来之后看到他时他逃跑了。罗恩、赫敏找到哈利时，魔法部工作人员来了，他們以為哈利是犯人，后来韦斯莱先生趕來說明情況解開誤會。下個場景哈利和朋友们搭乘[[霍格華茲|霍格華茲]]特快上前往学校，哈利在買零食車上的點心初遇張秋，並對她有好感。

当大家来到城堡，[[阿不思·邓布利多|阿不思·邓布利多]]校长宣布，[[三巫鬥法大賽|三巫鬥法大賽]]要在霍格沃茨举办，这是一个由欧洲三所学校聯合舉辦的传统赛事，每校選出一名代表参与比赛，選出冠军，获得荣誉。邓布利多介绍两个来访的学校：[[波巴洞魔法學校|波巴洞魔法學校]]（电影中是女校），优雅的校长[[马克西姆夫人|马克西姆夫人]]和[[花兒·戴樂古|花兒·戴樂古]]领头的漂亮女孩把周圍的男生迷得神魂顛倒；[[德姆蘭魔法學校|德姆蘭魔法學校]]（电影中像是一所军校），校长[[依果·卡卡夫|依果·卡卡夫]]和维克多‧喀浪最先登场（这令罗恩非常兴奋，他是喀浪的球迷）。隨后火盃出现，邓布利多說明規則並附註只有年滿17歲學生才能參加比賽。

哈利和朋友们在[[瘋眼穆敵|瘋眼穆敵]]执教的黑魔法防御术科目中学到了关于[[不赦咒|不赦咒]]的知识。幾天后火焰盃陸續选出三位学生代表：维克多‧喀浪（德姆蘭）、花兒‧戴樂古（波巴洞）、[[西追·迪哥里|西追·迪哥里]]（霍格沃茨），当人们认为火焰杯已经结束选择时，却又再选出一名选手，竟然是年僅14歲的哈利·波特，这引起一阵骚动。但是，按照规则，被火焰盃選出的人選必須參賽，哈利因此成為[[三巫鬥法大賽|三巫鬥法大賽]]的第四位參賽者。

圣诞晚会上，哈利、榮恩、妙麗分别提出或接受异性的邀请，与舞伴一起参加晚会。最令人意外的是，邀請妙麗參加舞會的是维克多‧喀浪。

第一个比赛项目是从[[龙|龙]]的看管下取一只包含下一个任务线索的金蛋。他們分別選出即將要對付的龍：西追抽到瑞典短吻龍、花兒抽到威爾士綠龍，喀浪抽到中國火球龍，而哈利要對付的是兇暴的匈牙利角尾龍。

在一場艱辛奮鬥後，哈利成功得到金蛋，金蛋中的人魚歌聲线索指引，第二个任务是从禁湖地下救出最重要的朋友。哈利需要救罗恩，喀浪需要救妙麗，西追需要救[[秋·张|秋·张]]，而花兒則是妹妹[[加布丽·德拉库尔|加布丽·德拉库尔]]。后来哈利不仅救了罗恩还救了加布丽·德拉库尔，尽管是最后一个完成任务的，但是他因为奮不顧身的精神被奖励为第二名（书中是第一名）。

第二个任务之后，哈利在森林里发现了[[巴堤·柯羅奇|巴堤·柯羅奇]]的尸体，之後他到校長室时听到了邓布利多和魔法部部长[[康奈利·福吉|康奈利·福吉]]的谈话。谈话结束后，邓布利多让哈利在办公室中等待。哈利无意间发现了[[哈利·波特中的魔法物品|冥想盆]]，他看到了邓布利多的回忆：伊果·卡卡夫因为忠于伏地魔在魔法部受审而後供出小巴蒂·克劳奇曾參與綁架及虐待隆巴頓夫婦。

第三个也是最后一个项目是在[[迷宫|迷宫]]中寻找奖杯。塞德里克和哈利共同先于别人到达了奖杯处，他们决定共同触摸奖杯。奖杯本应把他们带到迷宫入口处，但却把他们带到了里德尔府，虫尾彼得杀了想反抗的塞德里克，伏地魔借助父親屍骨、一個忠心僕人的肢體（蟲尾剁了自己的一隻手）和哈利的血液而得到复活，他召唤了所有食死徒，哈利看到卢修斯·马尔福也是其中之一。伏地魔強制要哈利與他對戰並想杀掉他，但不僅徒勞無功，過程中雙方魔杖還出現奇特反應，哈利意外地释放了一个咒语（呼呼‧前咒現），所有被伏地魔杀掉的人的灵魂都显现出来，哈利也在這時见到父母（也曾被伏地魔杀死）和得到其幫助，最后他趁此机会带着塞德里克尸体触摸奖杯，回到霍格沃茨。

觀賽人群被塞德里克的尸体震惊了，悲傷萬分的哈利被疯眼汉穆迪強行带回城堡。在一些對話之後細心的哈利發現穆迪知道很多自己經歷的事情，甚至哈利沒有告訴他的事情，教授居然也知道。終於在關鍵時刻，鄧布利多、石內卜、麥奈米娃三位教授趕到救下了哈利。原來疯眼汉穆迪是小巴蒂·克劳奇喝下變身水而變形的，真正的穆迪教授被鎖在了一個箱子裡。他策划了这场“哈利卷入比赛获得冠军，到达伏地魔处使其复活”的阴谋。

== 與原著相異之處==
* 在小說裡，一開始為旁白描述[[湯姆瑞斗|湯姆瑞斗]]家過去發生的故事,電影裡則沒有演出旁白描述過去所發生的故事內容,電影開頭就直接是大蛇出現往前竄行,接著就是管理員法蘭克抬頭看到[[湯姆瑞斗家|湯姆瑞斗家]]有燈光懷疑有人闖入而開始往後的劇情內容。
* 在小說裡，[[魚鰓草|魚鰓草]]是[[多比|多比]]給的；但在電影裡，卻是[[奈威·隆巴顿|奈威]]給的。(由於本集小說裡的中要角色小精靈多比與眨眨，電影中全部被刪除)
* 在小說裡，[[弗雷和喬治·衛斯理|弗雷和喬治]]長出鬍子後，是互相大笑；在電影裡，卻是互相拉扯鬍子，而且是將紙條丟入火盃後才被彈開，小說裡兩人嘗試穿越鄧不利多的年齡界線就被彈開。
* 在小說裡，[[芭蒂|芭蒂]]跟哈利跳完舞後，就跟[[波巴洞|波巴洞]]的一名男子跳舞；但在電影裡，芭蒂卻只是走開而已，而且電影裡的[[波巴洞|波巴洞]]是女校，沒有男生。

== Weird Sisters的诉讼 ==
有一个叫作[[Weird_Sisters|Weird Sisters]]的女子乐队，他们向制片公司等相关人员提起了要求赔偿四千万[[美元|美元]]的[[诉讼|诉讼]]。因为他们在影片中使用了Weird Sisters这个名字作为乐队。

实际上原著中有关于这个组合的说法（翻译为“古怪姐妹”演唱组），而电影拍摄时，制作公司请了几位歌手来装扮成古怪姐妹演唱组成员，这招致了“真的”古怪姐妹乐队的诉讼。

== 反响 ==
这部电影得到了大多数评论家的肯定。在[[Rotten_Tomatoes|Rotten Tomatoes]]上得到了89%的级别。人们表扬这一部电影比前作更加巧妙和诙谐。

不论如何，抱怨还是有的。一些人认为这个电影过于沉闷和粗略（比如第三个项目就没怎么表现）。一些人称赞[[拉尔夫·费恩斯|拉尔夫·费恩斯]]饰演的伏地魔，一些人认为他的表演不能带来强烈的恐怖感。而[[阿倫·烈卡文|阿倫·烈卡文]]的“[[爱好者|爱好者]]”则抱怨说他所扮演的西弗勒斯·石內卜出场时间太少了。

还有，對於扮演邓布利多的米高·詹邦，他那好鬥的表演风格也令一些人失望。

== 商业成功 ==
首映第一天，这部电影的北美[[票房|票房]]就达到了四千万美元。而根據這部電影在[[中國大陸|中國大陸]]的發行商[[中影集团|中影集团]]表示，《哈利·波特与火焰杯》在中国内地上映10天（截至11月27日）总票房为6656万元[[人民币|人民币]]，这其中北京票房总额达到1000万元，着相当高。可见这部电影的商业运作很成功。
