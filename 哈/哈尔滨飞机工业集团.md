'''哈尔滨飞机工业（集团）有限责任公司'''（简称'''哈飞'''）[[中國航空工業集團|中國航空工業集團]]旗下一員。原名'''哈尔滨飞机制造公司''' 之前的名是伟建机器制造厂，成立于1952年4月。<ref>{{cite web|url=http://www.hafei.com/company1_1_1.htm |title=集团概况 |accessdate=2007-12-18 |publisher=哈尔滨飞机工业集团有限责任公司 |language=zh-cn |deadurl=yes |archiveurl=https://web.archive.org/web/20141226164439/http://www.hafei.com/company1_1_1.htm |archivedate=2014-12-26 }}</ref>。1998年9月9日，哈尔滨飞机制造公司改名为哈尔滨飞机工业集团有限责任公司<ref> {{cite web |url=http://www.hafei.com/hahang_history_90.htm |title=90年代大事记 |accessdate=2007-12-18 |publisher=哈尔滨飞机工业集团有限责任公司 |language=zh-cn |deadurl=yes |archiveurl=https://web.archive.org/web/20071127113028/http://www.hafei.com/hahang_history_90.htm |archivedate=2007-11-27 }}</ref>现位于[[黑龙江省|黑龙江省]][[哈尔滨市|哈尔滨市]]。

==历史沿革==
*1952年4月1日，作为中华人民共和国航空工业第一批六大主机厂之一的'''122厂'''正式成立，創辦初期僅負責飞机維修業務。
*1953年12月，122厂在原有的基础上扩建成为大型[[轰炸机|轰炸机]]工厂。
*1957年5月14日，正式启用'''国营伟建机器厂'''为第二厂名。
*1978年9月1日，部明确122厂为对外开放工厂，[[轰5|轰5]]、[[直5|直5]]、[[运11|运11]]为对外开放机种，对外厂名为'''哈尔滨飞机制造厂'''。
*1986年11月1日，更名为'''哈尔滨飞机制造公司'''。
*2000年12月18日，"哈飞"[[A股|A股]]在[[上海证券交易所|上海证券交易所]]挂牌上市。
*2003年12月16日，与[[巴西航空工业公司|巴西航空工业公司]]合资创办'''哈尔滨安博威飞机工业有限公司'''。
*2004年4月30日，由哈尔滨飞机工业集团与[[哈尔滨东安发动机集团|哈尔滨东安发动机集团]]整合重组后的[[哈尔滨航空工业(集团)有限公司|哈尔滨航空工业(集团)有限公司]]正式挂牌成立。<ref>{{cite news| first=| last=| coauthors=| title=哈尔滨航空工业(集团)有限公司| date=2004-05-11| publisher=新浪网| url=http://auto.sina.com.cn/news/2004-05-11/64302.shtml| work=| pages=| accessdate=2007-12-18| language=zh-cn| deadurl=no| archiveurl=https://web.archive.org/web/20051117022727/http://auto.sina.com.cn/news/2004-05-11/64302.shtml| archivedate=2005-11-17| df=}}</ref>

== 研发大事 ==
[[File:DATANSHAN_CHINA_AVIATION_MUSEUM_BEIJING_OCT_2012_(8272484990).jpg|缩略图]]
[[File:Eurocopter_EC_175.jpg|缩略图]]
[[File:038_Harbin_H-5_PLAAF_(3223304620).jpg|缩略图]]
*1958年1月19日，開始生產[[苏联|苏联]]授權生產之軍用機。第一批預定生產機種為[[直-5|直五]]军用[[直升机|直升机]]（仿Mi-4）以及[[Il-28轟炸機|轟五]]噴射轰炸机([[Il-28轟炸機|Il-28轟炸機]]授權生產版本)。
*1958年9月29日，“松花江一号”小型旅游[[運輸機|運輸機]]试飞成功。
*1958年12月16日，中国第一架[[直升机|直升机]]--[[直-5|直5]]首飞成功。
*1959年9月27日，中国第一架中型[[喷气轰炸机|喷气轰炸机]]--[[轰6|轰6]]飞机首飞成功。当年12月交付部队。
*1965年5月14日，中国第一架参与投放[[原子弹|原子弹]]的[[飞机|飞机]]—[[轰6|轰6]]飞机成功实现中国第一次用飞机进行运载、投放[[原子弹|原子弹]]的试验。
*1966年9月25日，中国第一架噴射轟炸機—[[Il-28轟炸機|轟五]]飞机首飞成功。
*1967年4月，[[Il-28轟炸機|轟五]]型轰炸机通过測試，正式量产。
*1968年12月9日，[[毛泽东|毛泽东]]、[[周恩来|周恩来]]将[[水轟五|水轟五]]研制生产定点于122厂。
*1969年12月10日，[[直-6|直-6]]机02架试飞成功。
*1970年11月12日，[[Il-28轟炸機|轟教五]]首飞成功。
*1971年，创造年产[[直-5|直5]]、[[Il-28轟炸機|轟五]]各100架的记录。
*1974年，哈飞开始研制[[运-11|运-11]]
*1975年12月30日，[[运11|运11]]（Y-11）飞机首飞成功，Y-11为双发动机多用途飞机。
*1976年4月3日，自行研制的第一代大型水上[[反潛機|反潜轰炸机]][[水轰5|水轰5]]飞机首飞成功。同年，成功研制了涡桨五航空发动机并转入小批量生产。
*1980年初，设计、研制生产了民用飞机[[运-12|运-12]]。
*1980年10月15日，引进法国"海豚"直升机生产专利，开始研制生产“海豚”直升机。
*1982年7月14日，[[运-12|运-12]]Ⅰ型首次试飞成功。
*1983年，直9机被[[总参|总参]]正式列装。
*1984年6月15日，[[运-12|运-12]]Ⅱ型首次试飞成功。
*1985年2月12日，[[运-12|运-12]]飞机首次夜航试飞成功。
*1985年5月20日，运12飞机首飞成功。
*1985年12月24日，[[运-12|运-12]]Ⅱ型飞机获得[[中国民航总局|中国民航总局]]首次对国产[[民用飞机|民用飞机]]颁发的第一个型号合格证。
*1986年12月24日，[[运-12|运-12]]Ⅱ型飞机获得[[中国民航总局|中国民航总局]]对国产[[民用飞机|民用飞机]]颁发的第一个生产许可证。
*1987年5月，中国第一架用于海洋监测的[[运-12|运-12]]Ⅱ型飞机交付[[国家海洋局|国家海洋局]]正式投入使用。该架机获得[[中国民航总局|中国民航总局]]签发的单机适航证。
*1987年6月11日，[[运-12|运-12]]飞机、[[直-9|直-9]]机首次参加了在[[法国|法国]]举办的第37届[[布尔日国际航天航空博览会|布尔日国际航天航空博览会]]。
*1987年12月16日，成功改装了[[水轰5|水轰5]]森林灭火机。

==主要产品==
哈飞主要产品包括<ref>{{cite web |url=http://news.xinhuanet.com/mil/2006-04/18/content_4440167_1.htm |title=中国航空工业创建55周年 9成军机自主研发生产 |accessdate=2007-12-18 |publisher=新华网 |language=zh-cn |deadurl=no |archiveurl=https://web.archive.org/web/20060422213642/http://news.xinhuanet.com/mil/2006-04/18/content_4440167_1.htm |archivedate=2006-04-22 |df= }}</ref>：
*[[直-5|直-5]] （Z-5）多用途直升机，初期代号“旋风25”，原型为前苏联[[米-4|米-4]]（Mil Mi-4）直升机，1958年12月14日首次试飞，1959年初投入批生产，已退役
*[[Il-28轟炸機|轟五]]（H-5），噴射戰術战术轰炸机，苏联伊尔-28(IL-28)喷气轰炸机授權生產，1966年9月25日试飞成功，1967年4月量产，1984年停产。
*[[水轰-5|水轰-5]] 水轰-5型水上反潜轰炸机（PS-5），1971年装出第一架原型01号，1976年4月3日首次进行水上起降试飞，1986年服役。
*[[直-9|直-9]] （Z-9） 直-9轻型多用途直升机是引进法国[[歐直AS365海豚直升機|SA365“海豚”型]]直升机的生产专有权合同，由哈飞生产，1982年装配首架。1992年1月16日国产化直-9首飞，该型号定名直-9B。
*[[直-15|直-15]]（Z-15）和歐洲直升機公司聯合開發多用途直升机，2016年12月20日首飛。
*[[运-11|运-11]] Y-11为双发动机多用途运输机。
*[[运-12|运-12]] 运-12运输机是在运-11基础上更换发动机后研制的小型涡桨多用途运输机。
