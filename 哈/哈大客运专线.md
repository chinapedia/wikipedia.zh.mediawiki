'''哈大客运专线'''，又稱'''哈大高鐵'''<ref>[http://news.sina.com.tw/article/20121201/8444459.html 哈大高鐵今日正式營運 系世界首條高寒區高鐵]</ref>，是[[中国|中国]][[黑龙江省|黑龙江省]][[哈尔滨市|哈尔滨市]]与[[辽宁省|辽宁省]][[大连市|大连市]]之间的[[高速铁路|高速铁路]]，[[哈尔滨西站|哈尔滨西站]]至[[沈阳北站|沈阳北站]]为[[京哈客运专线|京哈客运专线]]的一部分。

== 概况 ==
哈大客运专线线路正线全长904公里，途径[[中国东北地区|东北]]3省4个[[副省级市|副省级市]]和6个[[地级市|地级市]]，其中辽宁省境内553公里，吉林省境内270公里，黑龙江省境内81公里。哈大客运专线采用复线[[電氣化鐵路|電氣化鐵路]]，[[基础设施|基础设施]]按350公里/时速建设，区间运行列车均为[[动车组|动车组]]，平均时速200公里以上。哈大客运专线经过10座城市，由南向北依次为：[[大连市|大连市]]、[[营口市|营口市]]、[[鞍山市|鞍山市]]、[[辽阳市|辽阳市]]、[[沈阳市|沈阳市]]、[[铁岭市|铁岭市]]、[[四平市|四平市]]、[[长春市|长春市]]、[[松原市_(吉林省)|松原市]]、[[扶余市|扶余市]]、[[哈尔滨市|哈尔滨市]]，是世界首条達到時速300公里的[[寒带|高寒区]]高铁<ref name= "试运行"/>。哈大客运专线工程将新建[[火车站|火车站]]18座，改造现有火车站6座。全线有2/3以上的路段在高架桥上，共有162座桥梁，总长663公里。

哈大客运专线项目在项目建议书阶段预计投资是820亿元，到可行性研究报告阶段提升至950亿，而网易声称其开通后实际的造价“远远超过1000亿元[[人民币|人民币]]”<ref>[http://news.163.com/13/0322/01/8QHLJRCJ0001124J.html]</ref>，平均每[[公里|公里]]造价达1亿元人民币。建设资金来源主要分为铁路建设基金、地方政府资金和[[银行|银行]][[贷款|贷款]]三部分，具体项目出资比例为：[[沈阳铁路局|沈阳铁路局]]77.73%，[[哈尔滨铁路局|哈尔滨铁路局]]8.64%，辽宁哈大高铁投资公司9.02%，吉林省交通投资开发公司3.37%，黑龙江省哈大高铁有限责任公司1.24%。<ref>[http://www.ztxygs.com.cn/Article_72/2007831152569-1.html 哈大铁路拉动铁道建设市场] {{webarchive|url=https://web.archive.org/web/20090108042212/http://www.ztxygs.com.cn/Article_72/2007831152569-1.html |date=2009-01-08 }}</ref>
== 支线==
*长春联络线： 长春西站与长春站之间。全长20km。速度160km/h。设有杨家粉坊线路所、崔家营线路所。总投资92699万元。施工单位中铁建十三局。

== 运营 ==
{{main|哈大高速动车组列车|哈大动车组列车}}
哈大高铁运营初期，实行冬季和夏季“两张列车运行图”，分别按时速200公里和300公里两个速度等级开行动车组列车，同时实行与两个速度等级相对应的票价。冬季运行图实施阶段为每年12月1日至次年4月20日，共5个月。夏季运行图实施阶段为每年4月21日至11月30日，共7个月<ref>{{cite news|title=铁道部：哈大高铁12月1日开通运营 冬夏速度不同|url=http://www.dfdaily.com/html/8757/2012/11/24/899189.shtml|accessdate=2012-11-24|newspaper=中国网|date=2012-11-24}}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref> 。2015年起，哈大高铁全年按照时速300公里开行动车组列车，即“冬夏一张列车运行图”。<ref>{{cite web|title=哈大盘营两高铁明起 实行冬夏一张运行图|url=http://finance.sina.com.cn/roll/20151130/055923884672.shtml|publisher=经济日报|date=2015-11-30|accessdate=2016-08-13}}</ref>

== 列车 ==
[[File:CRH380BG_20151103.JPG|thumb]]担当，图为一组CRH380BG动车组列车进入[[滨海站|滨海站]]]]
{{See also|CRH380BG|中国标准动车组}}
其使用的[[CRH380B|CRH380B]]型高寒动车组由[[长春轨道客车股份有限公司|长春轨道客车股份有限公司]]设计制造，具备防雪、抗寒性能，可在零下38摄氏度低温环境下持续运行（可适应最低温度为-40℃）。<ref name="试运行">{{cite web|url=http://news.163.com/12/1008/09/8D9KAVKM0001124J.html | language= | title=哈大高铁今日全线试运行 为世界首条高寒区高铁 |publisher=人民网 |date=2012-10-08 |accessdate= }}</ref>。
2016年8月15日6时10分，由中国标准动车组担当的G8041次列车，从[[大连北站|大连北站]]出发，沿哈大高铁开往[[沈阳站|沈阳站]]，这是中国标准动车组首次载客运行<ref>{{cite news|title=中国标准动车组首次载客运行|url=http://news.xinhuanet.com/politics/2016-08/15/c_129230065.htm|publisher=新华网}}</ref>。

== 爭議 ==
2011年6月，哈大客专公司总经理[[杜厚智|杜厚智]]因哈大线基建工程和物资招投标方面的经济问题被免职接受调查<ref>{{cite news|title=哈大铁路客运专线总经理杜厚智被免职接受调查|url=http://news.ifeng.com/mainland/detail_2011_07/13/7649869_0.shtml|accessdate=2012-11-24|newspaper=21世纪经济报道|date=2011年7月13日}}</ref>。

2012年9月，“21世纪经济报道”声称哈大客运专线路基“发生大面积[[冻胀|冻胀]]”<ref>{{cite web |url=http://epaper.21cbh.com/html/2012-09/14/content_33722.htm?div=-1 |title=存档副本 |accessdate=2012-11-24 |deadurl=yes |archiveurl=https://web.archive.org/web/20120919060427/http://epaper.21cbh.com/html/2012-09/14/content_33722.htm?div=-1 |archivedate=2012-09-19 }}</ref>。网易等媒体声称其原因是“投资方和施工方为了节省成本，没有在施工过程中使用高寒地区施工应有的防冻技术”<ref>[http://news.qq.com/a/20120903/001349.htm]</ref><ref>[http://news.163.com/13/0328/17/8R2PVCTF0001124J.html] </ref>。

哈大客运专线原定于2013年4月1日起按照夏季运行图提速至300公里每小时的计划被延迟至2013年4月21日实行，网易声称其原因为此前因高管贪腐而引发的“工程偷工减料的质量问题尚未得到根本性的整改”。<ref>[http://news.163.com/13/0322/01/8QHLJRCJ0001124J.html] </ref>也有意见认为延迟是因为铁道部改制。
