'''哈尔滨国际会议展览体育中心'''，是位于[[哈尔滨市|哈尔滨市]][[南岗区|南岗区]]的一处综合功能建筑群，包括[[體育場|體育場]]、[[體育館|體育館]]、[[展覽館|展覽館]]、[[酒店|酒店]]、[[商場|商場]]和[[超市|超市]]等。整片区域位于黄河路、长江路、南直路和红旗大街的包围之中。

== 体育场 ==
[[哈尔滨国际会展中心体育场|哈尔滨国际会展中心体育场]]是哈尔滨市最大的综合性体育场，可容纳50000人，主要进行足球比赛、举办演唱会等活动，[[哈尔滨毅腾|哈尔滨毅腾]]、[[黑龙江火山鸣泉|黑龙江火山鸣泉]]等俱乐部均曾或正将此体育场作为主场。

== 体育馆 ==
[[哈尔滨国际会展体育中心体育馆|哈尔滨国际会展体育中心体育馆]]曾举办过[[2009年冬季世界大學生運動會|2009年冬季世界大學生運動會]]的開幕式和闭幕式<ref>{{cite web|title=大冬会场馆介绍——哈尔滨国际会展中心体育馆|url=http://sports.163.com/09/0212/22/5201P5U700051CAQ.html|website=网易体育|accessdate=2016-03-05}}</ref>，也曾举办过2007年[[国际滑冰联盟花样滑冰大奖赛中国站|花样滑冰大奖赛中国站]]的比赛。
== 展览厅 ==
哈尔滨国际会议展览体育中心展厅分为A、B、C、D共4个大厅，且常年承办各种展会活动。
===举办展览===
*哈尔滨车展
*哈尔滨文博会
*[[中国-俄罗斯博览会|中国-俄罗斯博览会]]
*[[哈尔滨青少年ACG展会|哈尔滨青少年ACG展会]]
*[[中国冰雪ACG展会|中国冰雪ACG展会]]

== 其它 ==

=== 酒店 ===
哈尔滨华旗饭店位于此区域。

=== 商业 ===
哈尔滨红博购物中心、红博会展购物公园(地下)、艺汇家商场等均位于此区域。

此区域内有一家[[家乐福|家乐福]]超市。
