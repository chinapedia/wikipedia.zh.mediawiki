'''帆船運動'''（{{lang-en|'''Sailing'''、'''Yachting'''}}<ref>[http://gongjushu.cnki.net/refbook/BasicSearch.aspx Yachting]，中國大陸：中国工具书网络出版总库。</ref><ref>[http://140.128.103.128/web/Content.asp?ID=3779 帆船運動]，台灣：中國大百科全書。</ref>）為戶外[[水類運動|水上運動]]之一，主要活動地點為海上，帆船運動員藉由自然風力作用於[[帆|船帆]]，駕駛帆船或{{link-en|帆艇|Sailboat}}向前移動，是以技巧為主，體能為輔的一種運動。<ref name = First>温贺宝、陈建华（2009）：《水上运动竞技与休闲》，第143頁，中國大陸：哈尔滨地图出版社。</ref>

帆船運動最早起源於17世紀的[[歐洲|歐洲]]，在1900年[[法國|法國]][[巴黎|巴黎]]舉辦的[[1900年夏季奧林匹克運動會|第二屆夏季奧運]]中，帆船運動開始被列為正式比賽項目，至今，這項運動在世界各沿海國家或地區，逐漸成為普遍而且廣受歡迎的水上體育活動之一。<ref name = First/>
