'''合作金庫商業銀行'''（[[金融機構代號|金融機構代號]]：006），簡稱'''合作金庫'''或'''合庫'''，為[[合作金庫金融控股|合作金庫金融控股]]旗下的主要[[子公司|子公司]]，是[[臺灣|臺灣]]大型[[商業銀行|商業銀行]]及傳統八大公股行庫之一，其官股比例僅次於[[台灣銀行|台灣銀行]]、[[台灣土地銀行|台灣土地銀行]]，同時也是臺灣營業據點最多的商業銀行，國內據點270間，海外據點20間。<ref>[http://tw.rter.info/bank/detail.php?code=006 合作金庫商業銀行各分行電話、地址與匯款轉帳代碼表]</ref>

== 歷史 ==
合作金庫的前身為'''臺灣產業組合協會'''，創立於1923年，由全台灣290餘間[[合作社|合作社]]聯合組成。1942年，改組為'''台灣產業組合聯合會'''。1944年，因[[太平洋戰爭|太平洋戰爭]]，再度改組，分成'''台灣產業金庫'''與'''台灣農業會'''兩個機構，台灣產業金庫專職於金融事業。

1945年[[第二次世界大戰|第二次世界大戰]]結束，中華民國接收台灣。1946年10月5日，中華民國政府接收台灣產業金庫，改組為'''[[臺灣省|臺灣省]]合作金庫'''，屬性為特種經營的金融機構（不具[[銀行|銀行]]身分），至1985年才取得法人資格。2001年元旦，合作金庫配合[[陳水扁政府|陳水扁政府]]的[[台灣金融改革|金融改革（金改）]]政策，改制為'''合作金庫銀行股份有限公司'''；2004年11月，合作金庫在[[臺灣證券交易所|臺灣證券交易所]]正式[[首次公開募股|掛牌上市]]。

2006年5月1日，合作金庫與同為[[中華民國公營事業|公營]]行庫的[[中國農民銀行|中國農民銀行]]合併，以合作金庫為存續銀行，中國農民銀行因合併而消滅，合作金庫同時更名為'''合作金庫商業銀行股份有限公司'''。目前合作金庫為中華民國國內外分支機構最多的金融機構（共301家），總資產排名則僅次於[[臺灣銀行|臺灣銀行]]。

2011年12月1日，合作金庫改組為'''[[合作金庫金融控股公司|合作金庫金融控股股份有限公司]]'''，下轄合庫銀行、保險、證券、票券、投資信託等公司。

== 沿革 ==
*1944年，'''台灣產業金庫'''成立。
*1946年10月5日，台灣產業金庫改組為'''臺灣省合作金庫'''，主要[[股東|股東]]包含[[臺灣省政府|臺灣省政府]]以及各[[合作社|合作社]]、[[農會|農會]]、[[漁會|漁會]]、[[農田水利會|農田水利會]]等民間社團，[[股本總額|股本總額]]為[[舊臺幣|舊臺幣]]二千五百萬元，以每股舊臺幣一百元分為二十五萬股，其中政府股（公股）十五萬股、各社團股（社股）十萬股。
*1979年5月31日，總部大樓「臺灣省合作金庫大樓」奠基。
*1985年5月，依據《[[銀行法|銀行法]]》第52條，取得法人資格。
*1985年10月16日，奉命概括承受'''[[十信案|台北市第十信用合作社]]'''。
*1995年8月4日，奉命概括承受'''[[彰化市|彰化市]]第四信用合作社'''（金融代號：160）。
*2001年1月1日，奉准改制為'''合作金庫銀行股份有限公司'''，各地「支庫」改為「分行」。
*2001年9月14日，概括承受'''[[台中市|台中市]]第一信用合作社'''（金融代號：145）、'''台中市第五信用合作社'''（金融代號：149）、'''台中市第九信用合作社'''（金融代號：153）與'''台中市第十一信用合作社'''（金融代號：154）。
*2002年7月26日，概括承受'''[[台中縣|台中縣]][[神岡區|神岡鄉]]農會'''、'''[[彰化縣|彰化縣]]彰化市農會'''和'''[[雲林縣|雲林縣]][[林內鄉|林內鄉]]農會'''等三家農會之信用部。
*2003年6月，[[股票|股票]]公開發行。
*2004年11月17日，在[[臺灣證券交易所|臺灣證券交易所]][[股票上市|股票上市]]（[[股票代號|股票代號]]：5880）。
*2006年5月1日，配合[[陳水扁政府|陳水扁政府]]金融改革政策，合作金庫銀行與[[中國農民銀行|中國農民銀行]]（金融代號：002）合併，同時更名為'''合作金庫商業銀行股份有限公司'''。
*2009年1月，合作金庫銀行與[[法國巴黎銀行|法國巴黎保險集團]]正式簽約合資成立[[合作金庫人壽|合作金庫人壽]]，合作金庫將持有合庫人壽51%之股權，法國巴黎保險則持有49%<ref>[http://www.bnpparibas.com.tw/cn/2009/01/13/%E5%90%88%E4%BD%9C%E9%87%91%E5%BA%AB%E9%8A%80%E8%A1%8C%E8%88%87%E6%B3%95%E5%9C%8B%E5%B7%B4%E9%BB%8E%E4%BF%9D%E9%9A%AA%E9%9B%86%E5%9C%98%E6%AD%A3%E5%BC%8F%E7%B0%BD%E7%B4%84%E5%90%88%E8%B3%87%E6%88%90/ 合作金庫銀行與法國巴黎保險集團正式簽約合資成立「合庫人壽」]，2009-01-03，法國巴黎銀行台灣業務新聞中心</ref>。
*2010年9月24日，在位於台北市[[松山區_(臺北市)|松山區]][[長安東路|長安東路]]二段及[[八德路_(臺北市)|八德路二段]]交界處的原[[臺灣汽車客運公司|臺灣汽車客運公司]]中崙車站汽車修理廠土地興建新總行大樓<ref>田裕斌 台北2010年9月24日電，[http://news.cts.com.tw/cna/money/201009/201009240570036.html 〈合庫新總行大樓動土〉] {{webarchive|url=https://web.archive.org/web/20150705012032/http://news.cts.com.tw/cna/money/201009/201009240570036.html |date=2015-07-05 }}，[[中央通訊社|中央通訊社]]2010年9月24日。</ref>，由[[三門聯合建築師事務所|三門聯合建築師事務所]]辦理規劃設計及監造工作，[[互助營造|互助營造]]施工，包括一棟地下5層加地上20層之辦公大樓、一棟12層之資訊大樓及一棟7層之會議大樓，總樓地板面積約3萬2千餘坪，預定2014年完工<ref>合作金庫銀行 新聞稿，[http://www.tcb-bank.com.tw/brief_introduction/index_news/Documents/NewsData/26-990924.htm 〈合作金庫銀行總行大樓新建工程動土典禮〉]，2010年9月24日。</ref>。
*2011年12月1日，與[[合作金庫資產管理公司|合作金庫資產管理公司]]及[[合作金庫票券金融公司|合作金庫票券金融公司]]以股份轉換方式共同設立[[合作金庫金融控股公司|合作金庫金融控股公司]]。
*2013年9月11日，舉行新總行大樓「合庫金控暨合作金庫商業銀行總行大樓」上樑典禮<ref>合作金庫銀行 新聞稿，[http://www.tcb-bank.com.tw/brief_introduction/index_news/Documents/News102/31-1020911.htm 〈合庫總行大樓上樑典禮 啟動資產活化〉]，2013年9月11日。</ref>。
*2017年10月24日，合庫金控總部暨合作金庫商業銀行總行大樓啟用。
