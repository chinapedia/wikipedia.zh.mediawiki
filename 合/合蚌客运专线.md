'''合蚌客运专线'''，是[[中国|中国]]位于[[安徽省|安徽省]]中北部，连接[[合肥|合肥]]与[[蚌埠|蚌埠]]两座城市的一条[[中國高速鐵路|客运专线]]。

== 概述 ==
合蚌客运专线是[[京台高速铁路|京台高速铁路]]的一部分，同时兼有合肥、[[淮南|淮南]]、[[滁州|滁州]]、蚌埠四个城市间的城际铁路功能。<ref>[http://www.ah.xinhuanet.com/news/2008-06/16/content_13556635.htm 合蚌铁路项目通过专家评估] {{webarchive|url=https://web.archive.org/web/20110718220443/http://www.ah.xinhuanet.com/news/2008-06/16/content_13556635.htm |date=2011-07-18 }}</ref>

合蚌客运专线自合肥铁路枢纽引出，沿[[淮南铁路|淮南铁路]]经[[长丰县|长丰县]]水家湖至淮南市九龙岗，沿[[蚌合高速公路|蚌合高速公路]]跨窑河、既有[[水蚌铁路|水蚌线]]引入蚌埠高速客站，全长130.67公里，其中新建线路长度为120公里，该工程采用双线，近期速度300km/h, 远期速度350km/h, 采用电力牵引。由[[上海铁路局|上海铁路局]]负责管辖，已于2009年1月8日开工建设<ref name="r1">[http://news.xinmin.cn/domestic/shehui/2008/11/17/1434597.html 合蚌铁路客运专线计划年底开工建设]</ref><ref name="r2">{{Cite web |url=http://ah.anhuinews.com/system/2009/01/08/002188555.shtml |title=合蚌客运专线今天开工 |access-date=2009-01-08 |archive-url=https://web.archive.org/web/20110718223007/http://ah.anhuinews.com/system/2009/01/08/002188555.shtml |archive-date=2011-07-18 |dead-url=yes }}</ref>。建成之后，将会实现从合肥4个小时抵达[[北京|北京]]<ref>[http://www.ah.xinhuanet.com/ahzw2006/2008-06/16/content_13556448.htm 4小时到北京 合蚌铁路鸣响江淮城市群"集结号"] {{webarchive|url=https://web.archive.org/web/20160304141106/http://www.ah.xinhuanet.com/ahzw2006/2008-06/16/content_13556448.htm |date=2016-03-04 }}</ref>。投资估算总额136亿元人民币，由铁道部和安徽省政府共同出资建设，施工总工期为3.5年。受7·23事故影响，原定2011年12月通车的合蚌客运专线推迟正式运营时间。

《蚌埠日报》报道，合蚌客运专线2012年10月16日正式通车。通車后從合肥到北京坐火車最快時間由9.5小時左右縮短到4小時。

在规划合蚌客运专线之前，合蚌铁路一般指的是[[水蚌铁路|水蚌线]]和[[淮南铁路|淮南线]]的一段（水家湖至合肥）组成的铁路线。

== 车站 ==
合蚌线全线将共设9个车站和一个线路所，分别为：<ref name="r1"/>
* [[合肥站|合肥站]]
* [[合肥北城站|合肥北城站]]
* [[大包郢线路所|大包郢线路所]]
* [[双墩集站|双墩集站]]
* [[下塘集站|下塘集站]]
* [[水家湖站|水家湖站]]
* [[淮南东站|淮南东站]]
* [[刘府南站|刘府南站]]
* [[蚌埠南站|蚌埠南站]]
* [[蚌埠站|蚌埠站]]
其中，合肥、合肥北城、水家湖、淮南东和蚌埠南5个为客车站。
