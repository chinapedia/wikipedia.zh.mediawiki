'''合齒魚科'''是[[輻鰭魚綱|輻鰭魚綱]][[仙女魚目|仙女魚目]][[合齒魚亞目|合齒魚亞目]]的其中一[[科|科]]。

== 分布 ==
本[[科|科]]魚類分布在全球各大海域中。

== 深度 ==
水深約10至80公尺。

== 特徵 ==
本[[科|科]]魚種，體橢圓形或延長，略側扁，頭與軀幹具被圓鱗。口極寬，背鰭短，有脂鰭，齒尖利，在上下頷、腭骨及舌面多成毛刷狀，極少犬齒，較大的齒通能壓伏。尾鰭分叉，無發光器，脊椎數多。

== 生態 ==
本[[科|科]]魚類，棲息在沙泥地，甚或埋藏在沙中僅露出頭部。只有少部分出現在珊瑚礁，其餘多生活在較深水域，運動緩慢，以寬大而銳利的牙齒來追捕小魚，只要被牠咬到的獵物很少有在逃脫的機會，亦捕食被海流衝過來的[[浮游生物|浮游生物]]。

== 經際利用 ==
體型小無重大經濟價值，主因是本科魚類肉鬆散無味，小且多刺。故一般多用於製造[[魚鬆|魚鬆]]或[[魚漿|魚漿]]。
