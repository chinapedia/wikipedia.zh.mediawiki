'''合'''（conjunction，亦稱合日）是[[位置天文學|位置天文學]]的一個名詞，它的意義是從一個選定的特定天體（通常是[[地球|地球]]）觀察到二個天體在天空上的位置彼此非常靠近。較嚴謹的說法是這兩個天體在天球上有相同的[[赤經|赤經]]或[[黃經|黃經]]，而通常對太陽系內的天體都會使用黃經。這種現象有時稱為{{lang|en|appulse}}：'''兩星漸近'''（台灣用法<ref>[http://terms.naer.edu.tw/detail/697158/ 國家教育研究院雙語詞彙、學術名詞暨辭書資訊網]{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>）或'''最小角距'''（中國大陸用法<ref>[http://www.lamost.org/astrodict/search.php 中國的天文名詞]</ref>）。

== 近距離通過 ==
[[File:Lune-Venus-Jupiter.JPG|thumb]]、[[木星|木星]]合月]]一般來說，兩顆行星的'''合'''意味著它們有相同的[[赤經|赤經]]（或是[[時角|時角]]），這稱為'''赤經合'''。當然，兩顆行星也會有'''黃經合'''，而在[[星曆表|星曆表]]上的'''合'''通常都是前者（因為觀測都是使用赤道座標）。由於黃道與赤道之間有傾角，所以黃經合與赤經合的位置與時間都不一樣，但差異不會太大。有時也會發生'''三星合'''的事件，但只會在黃道或赤道其中之一發生。無論是赤經合還是黃經合，看起來都是這兩個天體在[[天球|天球]]上靠得很近，並且其中一顆會從另一顆的北方或南方掠過。

== 更近的距離通過 ==
然而，如果這兩個天體在合的時候還有相同的[[緯度|緯度]]，則較靠近地球的這一顆天體會從較遠那一顆的前方經過。在這種情況下，是在所謂的[[食_(天文現象)#對衝點|對衝點]]，如果其中一顆的影子能投射在另一顆的表面，就能發生[[食_(天文現象)|食]]的現象。例如，[[月球|月球]]經過地球的陰影時就會發生[[月食|月食]]。如果接近地球的天體盤面遠小於較遠的天體，這種現象稱為[[凌|凌]]。當[[水星|水星]]經過太陽前方時稱為[[水星凌日|水星凌日]]，[[金星|金星]]經過太陽前方時稱為[[金星凌日|金星凌日]]。反過來的話，接近地球的天體盤面遠大於較遠的天體，就可能將後者完全遮蔽掉，這種現象稱為[[掩星|掩]]。例如，每年都會發生幾次月球經過行星的前方，將行星[[掩星|遮蔽]]的現象；而像月球經過太陽與地球之間，將太陽遮蔽的情形極為罕見，特別稱為[[日食|日食]]（雖然這是一個不適當的名稱）。

== 觀測者的位置 ==
合的現象不僅與天體的相對位置有關，也與觀測者的位置相關。就以從地球上觀察太陽和月球做說明，在合的時刻，太陽和月球看起來是在相同的方向上（或是說有相同的黃經），而當太陽和月球在相對的位置時則是在[[衝_(天體位置)|衝]]（觀測所見的太陽和月球分開在相反的方向上或是黃經相差180度）。

== 上合和下合 ==
當一顆[[內側行星|內側行星]]的位置與地球不在太陽的同一側且三者成一直線時，稱為'''上合'''（ superior conjunction），或是'''外合'''，而這時從太陽看內側行星與地球猶如處在對立的[[衝_(天體位置)|衝]]（opposition）位上；當一顆內側行星與[[地球|地球]]在太陽的同一側且三者成一直線時，則稱為'''下合'''（inferior conjunction），或是'''內合'''。當下合時，從內側的行星觀察在外側的行星（地球）時，外側行星（地球）是與太陽處在對立的'''衝'''位上。
