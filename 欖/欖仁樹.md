'''榄仁树'''（[[學名|學名]]：''{{lang|la| Terminalia catappa }} ''）別稱'''大葉欖仁樹'''<ref name = "台灣行道樹圖鑑">{{cite book|title=《台灣行道樹圖鑑》|author=陳俊雄、高瑞卿|series= |publisher=貓頭鷹出版|page=第86-87頁|isbn= 978-986-6651-02-1|date=2008年3月}}</ref>、'''涼扇樹'''<ref name = "台灣行道樹圖鑑"/>、'''枇杷樹'''<ref name="國立台灣師範大學">[http://biogeo.geo.ntnu.edu.tw/service/biogeodb/plants/tempplq.asp?status=hplant&cluster=num&numland=TW_E002&key_words=233-006 植物查詢報表] {{webarchive|url=https://web.archive.org/web/20040912211522/http://biogeo.geo.ntnu.edu.tw/service/biogeodb/plants/tempplq.asp?status=hplant&cluster=num&numland=TW_E002&key_words=233-006 |date=2004-09-12 }}國立台灣師範大學地理學系生物地理與自然保育研究</ref>、'''山枇杷樹'''<ref name="中國植物誌">[http://www.plant.ac.cn/pdf.asp?pg=zwz/53(1)/010b.PDF 中國植物誌 第十頁]{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}植物通</ref>、'''法國枇杷'''、'''楠仁樹'''、'''雨傘樹'''<ref name = "台灣國家公園">[http://np.cpami.gov.tw/index.php?option=com_content&task=view&id=495&Itemid=10088 欖仁巨傘撐藍天，紅葉飄零話冬意]台灣國家公園</ref>、'''島朴'''及'''古巴梯斯樹'''<ref name = "國立台灣師範大學"/>等，為[[使君子科|使君子科]][[欖仁樹屬|欖仁樹屬]]<ref>[http://lovetree.forest.gov.tw/web/know.asp?OP=SHOW&RECID=6 樹木小常識] {{webarchive|url=https://web.archive.org/web/20050330225021/http://lovetree.forest.gov.tw/web/know.asp?OP=SHOW&RECID=6 |date=2005-03-30 }}台灣一人一樹認養網站</ref>植物。

== 形態 ==
欖仁樹因果實的形狀貌似[[橄欖|橄欖]]的核，故而得名<ref name = "台灣行道樹圖鑑"/><ref name = "台灣國家公園"/><ref name="香港自然網">[http://www.hknature.net/hdiary/week17.html 菠蘿壩自然教育徑－欖仁樹]{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}郊遊日記，二月二十日，香港自然網</ref>。花期3<ref name = "中國植物誌"/>－7月<ref name = "香港自然網"/>，果期7－9月<ref name = "中國植物誌"/>。

欖仁樹是一種落葉[[喬木|喬木]]<ref>[http://www.hktree.com/tree/Indian_almond.htm 欖仁樹]樹木谷</ref>，高可達15米或更高。[[樹皮|樹皮]]縱裂呈剝落狀，褐黑色。枝幹平展，側枝輪生<ref name = "真理大學"/>，近頂端密披黃色絨毛，落葉後有密而明顯的[[葉痕|葉痕]]<ref name="高雄市珍貴樹木">[http://dongsha.kcg.gov.tw/tree/Terminalia%20catappa.htm Terminalia catappa L.]{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}高雄市珍貴樹木</ref>
，[[樹冠|樹冠]]層傘形<ref name = "真理大學"/>，老樹有明顯板根<ref name = "高雄市珍貴樹木"/>。葉倒卵形，互生，叢生枝頂層次分明，基部截形或狹心形，先端鈍圓或短尖，中部以下漸狹，全緣，葉兩面無毛或幼時背部披疏軟毛，葉基背偶有一對腺體<ref name = "台大農業陳列館"/><ref name="台南縣政府">[http://green.tainan.gov.tw/green/main.asp?SysID=200782412058478191554 綠化資訊站]{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}台南縣政府</ref>，主脈粗壯，表面下陷形成一淺槽，背面凸起，基部[[葉柄|葉柄]]處披絨毛，側脈10－12對，網脈稠密，長約12－22厘米，寬約8－15厘米；葉柄披毛，短而粗壯，長約10－15毫米；秋季轉紅葉，冬季落葉<ref name = "台灣國家公園"/><ref name = "高雄市珍貴樹木"/>。花為[[穗狀花序|穗狀花序]]長而纖細，腋生，雌雄異花植物<ref name = "台灣行道樹圖鑑"/>，[[雄花|雄花]]長於上部，[[雌花|雌花]]或兩性花長於下部<ref name = "台灣行道樹圖鑑"/><ref name = "真理大學"/>，長約15－20厘米；花多數，綠色或白色，長約10毫米；花盤由5個腺體組成，披白色粗毛；[[花苞|花苞]]片小，早落；缺乏[[花瓣|花瓣]]<ref name = "高雄市珍貴樹木"/>；花萼筒杯狀，外面無毛，內面披白色柔毛，長約8毫米；萼齒5裂，白色<ref name = "真理大學"/>，三角形，幾乎與萼筒等長；[[雄蕊|雄蕊]]10枚，伸出花萼之外，長約2.5毫米；[[子房|子房]]呈圓錐形，幼時披毛，成熟時近無毛；[[花柱|花柱]]粗壯，胚球2顆，倒懸於室頂。果為[[核果|核果]]，扁橢圓形，兩端稍漸尖，兩邊具梭，梭上具翅狀的狹邊像船隻的龍骨狀構造<ref name = "台灣行道樹圖鑑"/>；果皮無毛含纖維質<ref name = "台大農業陳列館"/>，堅硬，初為綠色，成熟時呈咖啡色<ref name="臺灣原住民數位博物館">[http://www.dmtip.gov.tw/event/dye/03/03_0.htm 染料家族] {{Wayback|url=http://www.dmtip.gov.tw/event/dye/03/03_0.htm |date=20080924032438 }}臺灣原住民數位博物館</ref>，內果皮堅硬而質輕，能浮於水面<ref name = "台灣行道樹圖鑑"/><ref name = "台大農業陳列館"/>，長約3－4.5厘米，寬約2.5－3.1厘米。種子矩圓形，1顆，含油質。<ref name = "中國植物誌"/>

== 分佈 ==
欖仁樹常生於氣候溫熱的海邊沙灘<ref name = "中國植物誌"/>或向陽處山坡草地上<ref name="海南省疾病預防控制中心">[http://www.hncdc.cn/webstar/zwinfo.asp?id=16 海南省常見有毒野果圖鑑數據庫]{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}海南省疾病預防控制中心</ref>，現常栽培作行道樹。原產於[[臺灣島|臺灣島]]<ref name="台大農業陳列館">[http://www.aeh.ntu.edu.tw/main2/plant/plant01_04.htm 欖仁樹] {{webarchive|url=https://archive.is/20050315014844/http://www.aeh.ntu.edu.tw/main2/plant/plant01_04.htm |date=2005-03-15 }}台大農業陳列館</ref>、[[馬來西亞|馬來西亞]]<ref name="自由時報">[http://www.libertytimes.com.tw/2003/new/mar/13/life/article-4.htm 《植．物．語》] {{webarchive|url=https://web.archive.org/web/20050406201808/http://www.libertytimes.com.tw/2003/new/mar/13/life/article-4.htm |date=2005-04-06 }}自由時報，蘇枋</ref>、[[印度|印度]]、[[巴基斯坦|巴基斯坦]]及[[太平洋諸島|太平洋諸島]] <ref name="植物資料庫">[http://tpbg.tfri.gov.tw/Plants/plants_info.asp?rid=279欖仁Terminalia catappa L.]{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}植物資料庫</ref>。在中國境內，分佈於[[海南|海南]]、[[廣東|廣東]]、[[四川|四川]] <ref name = "海南省疾病預防控制中心"/>、[[湖北|湖北]] <ref name = "海南省疾病預防控制中心"/>、[[雲南|雲南]]東南部。於[[越南|越南]]、 [[大洋洲|大洋洲]]、及[[南美|南美]]熱帶海岸<ref name = "中國植物誌"/>等地均有分佈。

== 生長 ==
欖仁樹喜生長於高溫濕潤的氣候，光線充足的環境，不拘土質<ref>[http://www.hualien-innocuous.hl.gov.tw/Cikasoan/big5/DB/DB_view.asp?id=498&cid=6&urlID=%20&DBtype=1 植物資料庫] {{webarchive|url=https://archive.is/20120801230112/http://www.hualien-innocuous.hl.gov.tw/Cikasoan/big5/DB/DB_view.asp?id=498&cid=6&urlID=&DBtype=1 |date=2012-08-01 }}花蓮縣政府農業局知卡宣森林公園</ref>，但種植於肥沃排水良好的砂質土壤中最佳，能耐旱抗風耐鹽性強<ref name = "自由時報"/>。主要使用播種方式繁殖<ref name="真理大學">[http://www.au.edu.tw/ox_view/admin/kb/school's%20plant/plant/Plant%20Web/new_page_309.htm 欖仁樹] {{Wayback|url=http://www.au.edu.tw/ox_view/admin/kb/school's%20plant/plant/Plant%20Web/new_page_309.htm |date=20080915212425 }}台灣真理大學校園植物網</ref>。同時果實能浮於水面，故亦具有海漂植物散佈的特性<ref name = "台灣行道樹圖鑑"/><ref name = "台大農業陳列館"/><ref name = "植物資料庫"/>。

== 用途 ==
{{medical}}
欖仁樹的樹皮性味苦、性涼，有收斂之效<ref name = "數位典藏國家型科技計畫">[http://catalog.digitalarchives.tw/item/00/15/3e/52.html 欖仁樹] 數位典藏聯合目錄</ref>，對解毒止瘀、化痰止咳、[[痢疾|痢疾]]、痰熱咳嗽及[[瘡瘍|瘡瘍]]有治療功效<ref>[http://www.zysj.com.cn/zhongyaocai/yaocai_l/lanrenshupi.html 欖仁樹皮]中醫世家</ref>。葉及嫩葉對疝痛、頭痛、發熱、風濕關節炎有治療功效<ref name = "數位典藏國家型科技計畫"/>。葉汁對[[皮膚病|皮膚病]]、[[痲瘋|痲瘋]]及[[疥癬|疥癬]]有治療功效<ref name = "數位典藏國家型科技計畫"/>。種子性味苦、澀、性涼，可清熱解毒，對咽喉腫痛、痢疾及腫毒有治療功效<ref>[http://www.pharmnet.com.cn/tcm/knowledge/detail/112003.html 欖仁樹子]中國醫藥網</ref><ref>[http://www.zysj.com.cn/zhongyaocai/yaocai_l/lanrenshuzi.html 欖仁樹子]中醫世家</ref>。

欖仁樹的種子含油份，油份芳香含杏仁味，可供製成食品添加調味料<ref name = "香港自然網"/>及藥用<ref name = "台灣國家公園"/>；果皮含[[鞣質|鞣質]]，可製成黑色[[染料|染料]]<ref name = "香港自然網"/>；果皮、成葉及落葉均可染出黃褐與綠褐色系<ref name = "臺灣原住民數位博物館"/>；而樹皮亦含有單寧成份，可製成黑色染料<ref name = "台南縣政府"/>；木材可作船隻及家具等的用材<ref name = "台灣國家公園"/>。

欖仁樹的葉子枯萎後，收集起來煮過、泡製過後，可用水族中所謂的黑水，可加強[[泰國鬥魚|泰國鬥魚]]發情,提高受孕,尤其可驗證在[[泰國鬥魚|泰國鬥魚]]的繁殖中，因欖仁葉會釋放出色素;單寧酸，使水色變深，增加水色模擬黑水環境條件，可穩定魚兒的情緒,盡快解除緊迫,適應不良，加強提高體;尾和背鰭色澤，降低 PH值（增酸)，它含有大量的丹寧酸;腐植酸;葉酸等,可調節水的酸鹼值，軟水的作用，可以使水變稍為軟,不過也得看原水的軟硬度,如硬就得用更多葉子浸泡以達同樣效果，有大葉欖仁葉浸泡的魚缸，有生物死掉後會比較晚發霉，因此推斷大葉欖仁葉應該或多或少有些抑制黴菌生長的防腐功能。
