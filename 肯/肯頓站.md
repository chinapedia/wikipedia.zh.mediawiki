'''肯頓站'''（{{lang-en|Kenton station}}）是一個設有[[倫敦地鐵|倫敦地鐵]]（[[貝克盧線|貝克盧線]]）和[[倫敦地上鐵|倫敦地上鐵]]（[[沃特福德直流電鐵線|沃特福德直流電鐵線]]）列車服務的鐵路車站，位於{{link-en|肯頓路|Kenton Road}}南側，距離倫敦地鐵[[大都會線|大都會線]]的[[諾斯維克公園站|諾斯克公園]]站只有很短的步行距離。

== 歷史 ==
此站是其中一個由{{link-en|倫敦及西北鐵路|London and North Western Railway}}興建的[[沃特福德直流電鐵線|「新線」]]，由卡姆登到[[沃特福德交匯站|沃特福德交匯]]，容許短距離列車由[[沃特福德交匯站|沃特福德交匯站]]開往[[尤斯頓車站|尤斯頓車站]]和{{link-en|寬街站 (倫敦)|Broad Street railway station (London)|寬街站}}。新線基本上與1837年{{link-en|倫敦及伯明翰鐵路|London and Birmingham Railway}}主線平行。

[[File:Kenton_railway_station_MMB_04_350266.jpg|thumb]]{{link-en|英國鐵路350型電力動車組|British Rail Class 350|350型列車}}在[[西岸主線|西岸主線]]上高速通行肯頓站]]
肯頓站於1912年6月15日開放<ref name=CULG>{{cite web |url=http://www.davros.org/rail/culg/bakerloo.html#dates |title=Bakerloo Line, Dates |work=Clive's Underground Line Guides |accessdate=2008-07-21 }}</ref>。此站只有新線設有月台，平行的主線鐵路只停靠北面一個站的[[哈羅及威爾德斯通站|哈羅及威爾德斯通站]]，部分也停靠南面三個站的[[溫布利中央站|溫布利中央站]]。

[[貝克盧線|貝克盧線]]服務於1917年4月16日開展<ref name=rose>{{cite book |last=Rose |first=Douglas |title=The London Underground, A Diagrammatic History |year=1999 |publisher=Douglas Rose/Capital Transport |isbn=1-85414-219-4 }}</ref>。1982年9月24日，貝克盧線不再停靠[[石橋公園站|石橋公園]]以北的車站<ref name=rose/>，但1984年6月4日又延長貝克盧線回哈羅及威爾德斯通<ref name=rose/>。

車站位於鐵路東側的舊煤炭貨場現時已經不再使用，並由一間[[森寶利|森寶利]]的超級市場取代。

== 服務 ==
*各方向每小時3班倫敦地上鐵（沃特福德直流電鐵線）列車。
*各方向每小時6班倫敦地鐵（貝克盧線）列車。

== 接駁交通 ==
[[倫敦巴士|倫敦巴士]]114、183、223、H9、H10、H18、H19-{zh-hans:路; zh-hk:號線; zh-tw:路}-服務此站。

== 無障礙通行 ==
根據倫敦交通局2011年4月的地圖<ref>{{cite web|url=http://www.tfl.gov.uk/gettingaround/14091.aspx|title=''Getting Around''|publisher=Transport for London}}</ref>，此站不可無障礙通行。最近可以無障礙通行的車站包括[[哈羅及威爾德斯通站|哈羅及威爾德斯通]]、[[金斯伯里站|金斯伯里]]和[[溫布利公園站|溫布利公園]]，一些行程可通過使用倫敦巴士183-{zh-hans:路; zh-hk:號線; zh-tw:路}-前往[[格德斯綠地站|格德斯綠地站]]或[[平納站|平納站]]，並從該處繼續。
