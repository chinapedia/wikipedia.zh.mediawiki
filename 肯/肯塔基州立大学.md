'''肯塔基州立大学'''（Kentucky State University，缩写：KSU）是一所位于美国[[肯塔基州|肯塔基州]][[法兰克福_(肯塔基州)|法兰克福]]的[[州立大學|州立大學]]、[[传统黑人大学|传统黑人大学]]。该大学创始于1886年，当时为有色人种州立师范学校（State Normal School for Colored Persons），1972年升格为大學。2017年《[[美國新聞與世界報道|美國新聞與世界報道]]》将其排在“南部地区大學”中第16位。<ref name="USnews">{{cite web|title=Kentucky State University|url=https://www.usnews.com/best-colleges/kentucky-state-1968|publisher=[[美國新聞與世界報道|美國新聞與世界報道]]|accessdate=2016-10-28}}</ref><ref name="KSUHistory">{{cite web|url=http://kysu.edu/about-ksu/history/ |title=History of Kentucky State University |publisher=Kentucky State University |accessdate=2014-07-14}}</ref><ref name="demospring2014">{{cite web|url=http://kysu.edu/wp-content/uploads/2013/10/Enrollment-Snapshot-Spring2014.pdf |title=Enrollment Snapshot Spring 2014 |publisher=Kentucky State University |accessdate=2014-07-14}}</ref>

==历史==
[[File:KYState-Frankfort1898.jpg|thumb]]
[[File:Jackson_Hall,_Kentucky_State_University.JPG|thumb]]
1886年5月大学前身州立有色人种师范学院（State Normal School for Colored Persons）获办学许可，是肯塔基州成立的第二所州立高等教育机构<ref name="KSUHistory" />。1887年10月11日正式开学，第一届学生共55名，老师三人，校长是约翰·H·杰克逊（John H. Jackson）<ref name="KSUHistory" />。学校逐渐扩大，1902年改名肯塔基有色人种师范和工业学院（Kentucky Normal and Industrial Institute for Colored Persons），1926年又改作肯塔基有色人种工业学院（Kentucky State Industrial College for Colored Persons）。1930年代早期一度废校，1938年该做肯塔基州立黑人学院（Kentucky State College for Negroes）<ref name="KSUHistory" />。1952年去掉黑人一词，1972年成为大學，1973年接受第一个研究生。2017年该大学教员对董事会投出不信任票<ref>{{cite web |url=http://www.state-journal.com/2017/03/25/facultys-vote-of-no-confidence-in-kentucky-state-university-board-of-regents-a-crossroad-for-school/ |title=Faculty's vote of no confidence in Kentucky State University Board of Regents 'a crossroad' for school |author=Rosalind Essig |publisher=[[The_State_Journal_(Frankfort)|The State Journal (Frankfort)]] |date=March 25, 2017 |accessdate=April 8, 2017}}</ref>，但一些黑人员工表示在升职和任期上受到歧视<ref>{{cite web |url=http://www.chronicle.com/article/No-Confidence-Vote-Exposes/239614 |title=No-Confidence Vote Exposes Racial Tensions Among Faculty at Kentucky State U. |author=Adam Harris |date=March 28, 2017 |accessdate=April 8, 2017}}</ref>，他们表示信任董事会，而不信任学术评议会和其他教员的投票结果<ref>{{cite web |url=http://www.chronicle.com/blogs/ticker/black-professors-at-kentucky-state-vote-no-confidence-in-faculty-senates-president/117672 |title=Black Professors at Kentucky State Vote No Confidence in Faculty Senate's President |author=Adam Harris |date=April 7, 2017 |accessdate=April 8, 2017}}</ref>。
