'''陈中孚'''（{{bd|1882年||1958年5月24日|}}）[[字|字]]'''奇曽'''，[[江蘇省|江蘇省]][[蘇州府|蘇州府]][[吴县|吴县]]人，[[中華民国|中華民国]]政治人物。<ref name=rg>{{Cite book<!--|和書-->|author = 徐友春主編|title = 民国人物大辞典 増訂版|year = 2007|publisher = 河北人民出版社|isbn = 978-7-202-03014-1}}；<br/>{{Cite book<!--|和書-->|author = 劉国銘主編|title = 中国国民党百年人物全書|year = 2005|publisher = 团結出版社|isbn = 7-80214-039-0}}；<br/>{{Cite book<!--|和書-->|author = 東亜問題調査会|title = 最新支那要人伝|year = 1941|publisher = [[朝日新聞社|朝日新聞社]]|isbn = }}</ref>

==生平==
[[清|清]]末，他赴日本留学，毕业于[[法政大学|法政大学]]。归国後，他和[[商震|商震]]到[[東三省|東三省]]从事革命派的秘密活動。[[辛亥革命|辛亥革命]]爆发后，他担任東三省革命派的联络員。<ref name=rg/>

[[中華民国|中華民国]]成立後，他参加[[二次革命|二次革命]]，革命失敗后他逃亡到日本，後任[[中華革命党|中華革命党]]总務部第二科科長。1915年他归国，奉[[孫文|孫文]]（孫中山）的指示在[[山東省_(中華民国)|山東省]]从事党務和軍務。同年末，[[護国战争|護国战争]]爆发，他任東北革命軍总司令部参謀兼運糧局局長，後代理总司令职务。1917年（[[民国紀元|民国]]6年）他参加[[護法運動|護法運動]]，任大元帥府軍事委員会委員。<ref name=rg/>

[[国民政府|国民政府]]成立後，陳中孚成为[[中国国民党|中国国民党]]中的[[西山会議派|西山会議派]]成員，被视为接近元老[[許崇智|許崇智]]。1926年（民国15年），他任[[广州国民政府|广州国民政府]]委員会秘書。翌年10月，他任[[安徽省_(中華民国)|安徽省]]政府財政厅厅長，12月转任该省国民政府財政特派員。1928年（民国17年），他任[[青島市|青島市]]接收专員，翌年7月改任[[江蘇省_(中華民国)|江蘇省]]政府委員。同年中，他被任命为青島市長，但实际未就任。<ref name=rg/>

1931年（民国20年）5月，陳中孚作为許崇智的代理人参加反[[蒋介石|蒋介石]]派的广州国民政府，任政務委員会委員。1935年（民国24年），[[冀察政務委員会|冀察政務委員会]]（委員長：[[宋哲元|宋哲元]]）成立，他任该委員会的外交委員会主席委員，因同宋的意見不合而于翌年辞任。<ref name=rg/>

[[抗日战争|抗日战争]]爆发後的1938年（民国27年）3月，陳中孚任[[中華民国維新政府|中華民国維新政府]]行政院長[[梁鴻志|梁鴻志]]的顧問。他还开始了拥立[[吴佩孚|吴佩孚]]任親日政府首脑的活动。1939年（民国28年）他赴[[漢口|漢口]]为拥吴而设立办事处。但因为吴拒绝参加亲日政府（同年末，吴死去）该活動遭到重大挫折。[[汪精卫政权|汪精卫政权]]（南京国民政府）成立後的1941年（民国30年）5月，他任[[国民政府委员会_(汪精卫政权)|国民政府委員]]。後来他任（汪派）中国国民党中央監察委員。<ref name=rg/>

汪精卫政权倒台后，陳中孚赴日从事“日华亲善”活动，并担任“亚洲交友会”顾问。1958年5月24日，在东京都因[[心肌梗死|心肌梗死]]逝世。享年77（满75岁）。<ref>《朝日新闻》1958年6月22日</ref>
