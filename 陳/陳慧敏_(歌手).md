'''陳慧敏'''（{{lang-en|'''Vivian Chan'''}}，{{bd|1989年|4月3日}}<ref>[http://the-sun.on.cc/cnt/entertainment/20130426/00470_071.html 陳慧敏牛一 粉絲開P慶祝]《太陽報》2013年4月26日</ref>），[[澳門居民|澳門人]]，曾參與由[[澳門廣播電視股份有限公司|澳門廣播電視股份有限公司]]舉辦的《Sing擂台》，取得亞軍；另於2010年參加由[[香港無綫電視|香港無綫電視]]舉辦的《[[超級巨聲2|超級巨聲2]]》，獲得第十名的佳績。2013年其首支單曲《排隊》派台，陳慧敏正式於[[香港|香港]]出道。她同時為[[澳門演藝人協會|澳門演藝人協會]]的副會長<ref>{{cite news |author1=澳門特別行政區政府新聞局 |title=譚俊榮會見澳門演藝人協會一行 |url=https://www.gcs.gov.mo/showNews.php?DataUcn=95266&PageLang=C |date=2015-12-11}}</ref>。她與[[林欣彤|林欣彤]]為現實生活中的好友，二人於參加《超級巨聲2》時結識<ref>{{cite web|title=林欣彤陳慧敏恨嫁　擔心長住「姑婆屋」|url=https://www.hk01.com/article/9847/|publisher=香港01|date=2016-03-01}}</ref>。

==簡歷==
陳慧敏活躍於港澳樂壇，有多首受歡迎的得獎作品，包括《小人物》、《始終》、《自拍》 、《城市人》等。熱愛音樂的她曾參與多個大型歌唱比賽，更入選《超級巨聲 2》十強，2013 年簽約太陽娛樂文化，同年推出首張同名個人專輯，正式於香港發展。

2015年，陳慧敏在音樂及影視發展作出全新突破，先推出了第二張個人專輯《小人物》，主打歌《小人物》更登上多個流行榜。<ref>{{cite web |author1=獨驍天下 |title=小人物 陳慧敏 |url=http://m.dooland.com/index.php?s=/article/id/653213/from/faxian.html}}</ref><ref>{{cite web |author1=東周網 |title=分手的禮物 陳慧敏 |url=https://eastweek.my-magazine.me/main/42589}}</ref><ref>{{cite web |author1=論盡音樂 |title=[ 論盡短打 ] 陳慧敏第二張專輯「小人物」 |url=https://www.facebook.com/stupidmusichk/photos/a.514427802001498/711158302328446/?type=1&theater}}</ref><ref>{{cite web |author1=香港01 |title=【好聲音．Live】 陳慧敏完美演繹《小人物》　聽得巧感動 |url=https://www.hk01.com/%E9%9F%B3%E6%A8%82/41123/%E5%A5%BD%E8%81%B2%E9%9F%B3-live-%E9%99%B3%E6%85%A7%E6%95%8F%E5%AE%8C%E7%BE%8E%E6%BC%94%E7%B9%B9-%E5%B0%8F%E4%BA%BA%E7%89%A9-%E8%81%BD%E5%BE%97%E5%B7%A7%E6%84%9F%E5%8B%95}}</ref><ref>{{cite web |author1=U Blog |title=【碟評講場】《小人物》@ 陳慧敏 |url=https://blog.ulifestyle.com.hk/blogger/shingster1412/2015/06/%E3%80%90%E7%A2%9F%E8%A9%95%E8%AC%9B%E5%A0%B4%E3%80%91%E3%80%8A%E5%B0%8F%E4%BA%BA%E7%89%A9%E3%80%8B-%E9%99%B3%E6%85%A7%E6%95%8F/}}</ref>

在9月份舉行了首個公開售票音樂會《飛航模式音樂會》，全場坐無虛席。<ref>{{cite news |title=《飛航模式》音樂會精華片段 |url=https://www.facebook.com/vivianchanwaiman/posts/1005397469481156?__tn__=-R |work=陳慧敏FB貼文 |date=2015-09-21}}</ref>

在影視上，陳慧敏更首次擔任銀幕電影《燈塔下的戀人》女主角<ref>{{cite web |author1=澳門基金會 |title=澳基會支持文創　《燈塔下的戀人》放映會永樂舉行 |url=https://www.fmac.org.mo/emphasisProjects/showInfo_4717}}</ref>，更參與多部電影演出如《十月初五的月光》<ref>{{cite news |title=原班人馬延《十月初五的月光》 |url=https://www.cyberctm.com/zh_TW/news/detail/1032028#.W5eg0OgzbIV |work=澳門日報 |date=2015-11-04}}</ref>、《29+1》等等，也為電影《沖天火》女主角作粵語配音。她同時為擔任澳門演藝人協會的副會長。
