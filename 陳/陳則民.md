'''陳則民'''（{{bd|1881年||1951年|}}），[[江蘇省|江蘇省]][[蘇州府|蘇州府]][[吴县|吴县]]人。[[中華民国|中華民国]]政治家、[[律师|律师]]。<ref name=xu>{{Cite book|author = 徐友春主編|title = 民国人物大辞典 増訂版|year = 2007|publisher = 河北人民出版社|isbn = 978-7-202-03014-1}}</ref>

==生平==
他早年留学日本，[[日本大学|日本大学]]法科毕业。归国後，他当选参議院議員、政治討論会副会長。后来他任[[北京政府|北京政府]]的大总統府顧問。1920年（[[民国紀元|民国]]9年）之后的3年間，他任[[蘇州市|蘇州]]電气公司[[董事長|董事長]]。此后他在[[上海市|上海]]任开业律师。他曾任律師公会会長、上海各路商会連合会会長、納税華人理事会理事、蘇州電气廠总办。<ref name=xu/>

[[抗日战争|抗日战争]]爆发後，日军攻占[[蘇州|蘇州]]，陳則民任[[蘇州地方自治委員会|蘇州地方自治委員会]]会長。1938年（民国27年）3月，他参加[[中華民国維新政府|中華民国維新政府]]，任教育部部長。5月，他改任[[江蘇省_(中華民国)|江蘇省]]省長。在江蘇省省長任上，1940年（民国29年）3月[[中華民国維新政府|中華民国維新政府]]同[[汪精卫政权|汪精卫政权]]合流後，他获得留任，任职至同年6月。此後，他任監察院監察区監察使。<ref name=xu/><ref name=liu>{{Cite book|author = 劉寿林等編|title = 民国職官年表|year = 1995|publisher = [[中華書局|中華書局]]|isbn = 7-101-01320-1}}</ref>

日本投降後，陳則民以[[漢奸|漢奸]]罪被[[蒋介石|蒋介石]]的[[国民政府|国民政府]]逮捕，后来被判处无期徒刑。1949年10月，[[中華人民共和国|中華人民共和国]]成立後，他继续被收监。1951年，他在蘇州的監獄内病逝。享年71岁。<ref name=xu/>
