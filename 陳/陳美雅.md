'''陳美雅'''（{{bd|1970年|11月12日}}），臺灣政治人物，[[中國國民黨|中國國民黨]]籍，現任高雄市議員。生於[[高雄市_(省轄市)|高雄市]]，已婚，丈夫：夏仕欽<REF>[http://hk.crntt.com/crn-webapp/touch/detail.jsp?coluid=7&kindid=0&docid=103881969。高雄博士女議員嫁小12歲隨扈]</REF>。

畢業於[[日本|日本]][[早稻田大學|早稻田大學]]法學博士，追隨曾任第4屆高雄市議員的父親[[陳明吉|陳明吉]]從事政治活動<ref>{{Cite web |url = http://udn.com/news/story/6656/1035646 |title = 讓陳菊頭痛的美女議員 定情小12歲基層警 |author = 陳維鈞、楊濡嘉 |date = 2015年7月5日 |publisher = 聯合報 |language = zh-hant |accessdate = 2015年10月19日 |quote = 陳美雅的父親陳明吉曾當選縣市合併之前第4屆高雄市議員，她擁有日本明治大學法律學士、早稻田大學法律碩士和博士的學歷，繼承父志，競選第7屆高市議員和合併之後第1屆議員。 }}</ref>，自2006年開始擔任高雄第七屆高雄市議員<ref>{{Cite web |url = http://www.toughguy.tw/modules/tadnews/index.php?nsn=946 |title = 大高雄第一屆市議員選舉第一選區應選5席 |author =  |date = 2013年4月23日 |publisher = 很角色週報 |language = zh-hant |accessdate = 2015年10月19日 |quote = 現任5名市議員，國民黨2席市議員當中，陳美雅在第7屆選舉雖然是新人，但以1萬8043票最高票當選。}}</ref>，並連任高雄縣市合併後第六選區的第1、2、3屆議員。

==服務政見==
*設立農十六凹子底森林公園、凹子底停車場、帶動內惟整體發展
*成功拆除大公路陸橋、興建鹽埕綜合社會福利館、推動駁二文化產業發展
*活化旗津，興建新旗津醫院、新行政中心、海岸線保護工程<ref>{{Cite web |url = http://www.kcc.gov.tw/PeriodMembers/Introduction.aspx?KeyID=1787 |title = 高雄市議會本屆議員 |author =  |date = |publisher =  |language = zh-hant |accessdate = 2015年10月19日 |quote = 美雅宣言：國際視野專業問政，捍衛婦幼安全，保障學童教育環境，打造高雄海空經貿國際城，增加就業機會，爭取國家圖書館南部分館在美術館區，爭取大型體育賽事在高雄，提高高雄能見度。 }}</ref>
*成功推動台泥遷廠、都市更新、興建滯洪池，強化愛河流域防洪工程
*要求市府針對全市埋設地下管線全面檢修
*爭取社區監視器及全面汙水下水道整治
*減輕年輕父母負擔，爭取廣設公立托嬰中心、公立幼稚園
