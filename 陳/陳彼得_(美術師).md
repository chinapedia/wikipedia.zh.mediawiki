'''陳彼得'''（{{lang-en|Peter Chan}}）是美國的男性電子遊戲及電影美術師。

== 經歷 ==

在高中時期陳彼得從父親買來的介紹科幻電影《[[星際大戰|星際大戰]]》雜誌中，開始對[[概念藝術|概念美術]]產生興趣。<ref name="bio">{{cite web |url=http://peterchanconceptart.com/bio.html |title=Bio |work=P.CHAN |accessdate=2015年9月16日 |deadurl=yes |archiveurl=https://web.archive.org/web/20160304125132/http://peterchanconceptart.com/bio.html |archivedate=2016年3月4日 |df= }}</ref>

之後陳彼得進入拍攝《星際大戰》的[[盧卡斯影業|盧卡斯影業]]旗下之電子遊戲開發公司[[LucasArts|LucasArts]]就職，主要為LucasArts發行的冒險遊戲設計美術內容，包括《[[猴島小英雄2：老查克的復仇|猴島小英雄2：老查克的復仇]]》、《[[瘋狂時代|瘋狂時代]]》、<ref>{{cite journal |date=1993年12月 |title=卡通之後─瘋狂時代的美術創作 |journal=[[電腦玩家|電腦玩家]] |volume=29期 |pages=115頁 |publisher=電腦玩家雜誌社 |location=台灣 }}</ref><ref name="bio" />或是由[[提姆·謝弗|提姆·謝弗]]以[[阿茲特克|阿茲特克]]死後世界觀改編的《[[神通鬼大|神通鬼大]]》等遊戲作品。<ref>{{cite web |url=https://books.google.com.tw/books?id=AY5zAgAAQBAJ&pg=PA209&dq=peter+chan+lucasarts&hl=zh-TW&sa=X&ved=0CCAQ6AEwAWoVChMI9o26_df7xwIVQ8COCh3JCAfG#v=onepage&q=peter%20chan%20lucasarts&f=false |title=Graphic Adventures |author=Philipp Lenssen |page=209頁 |accessdate=2015年9月16日}}</ref>

離開LucasArts後陳彼得改進入提姆·謝弗另成立的遊戲公司[[Double_Fine_Productions|Double Fine Productions]]，為裡頭製作的《[[瘋狂世界|瘋狂世界]]》、《[[破碎時光|破碎時光]]》等遊戲處理概念美術事務，<ref>{{cite web |url=http://mixnmojo.com/features/previews/Psychonauts-Preview |title=Psychonauts Preview |work=The International House of Mojo |date=2012年2月21日 |accessdate=2015年9月16日}}</ref>另外也開始接洽各電影的美術案件，像奇幻電影《[[哈利波特：神秘的魔法石_(電影)|哈利波特─神秘的魔法石]]》、《[[波特萊爾的冒險|波特萊爾的冒險]]》，或是[[皮克斯|皮克斯]]的3D動畫電影《[[怪獸大學|怪獸大學]]》等。<ref>{{cite web |url=http://www.lucasdelirium.it/bio.php#chan |script-title=it:PETER CHAN (illustratore) |work=Lucasdelirium |accessdate=2015年9月16日}}</ref>
