'''陳佐才'''法政牧師（{{lang-en|Rev. Canon Alan Chor-Choi CHAN}}，{{bd|1935年||||catIdx=C}}），聖公會[[法政牧師|法政牧師]]，[[崇基神學組|崇基神學組]]榮休主任，曾任[[香港中文大學|香港中文大學]][[崇基學院|崇基學院]]署理院長（1991年1月至7月）。

==簡歷==
陳佐才，廣東新會人，1935年生於廣東省廣州市，廣州初中畢業時成績優異，後轉到香港[[聖保羅男女中學|聖保羅男女中學]]就讀。1958年畢業於[[崇基學院|崇基學院]]英國語文系。之後入讀香港協和神學院，1961年畢業。1966年獲美國{{le|太平洋神學院|Church Divinity School of the Pacific}}神道學碩士學位，1985年獲美國{{le|太平洋神學院|Church Divinity School of the Pacific}}頒授榮譽神學博士。他在1975年起在崇基神學組（現名：[[香港中文大學崇基學院神學院|香港中文大學崇基學院神學院]]）任教20年，並兩度擔任神學組主任（1982-1987 及 1992-1995），直至1995年退休。<ref>{{ cite web |url=http://www.grateful-heart.org/WonderfulGrace/content.php?article=2765 |title=陳佐才牧師談情 |date=1989-08-27 |publisher=身心美 |author=陳珊珊 }}</ref><ref>{{ cite web |url=https://www.ct.org.tw/1125978 |title=牧者情懷陳佐才牧師著施義勝 |date=1989-08-27 |publisher=基督教論壇報 |author= }}</ref>

陳佐才於1962年被按立成為牧師，之後成為[[聖公會港澳教區|聖公會港澳教區]][[法政牧師|法政牧師]]、諸聖堂義務聖品及佳寶教育基金會監督。此外，亦曾擔任《時代論壇》董事會主席（1999-2002）。<ref>{{ cite web |url=http://christiantimes.org.hk/Common/Reader/News/ShowNews.jsp?Nid=152490&Pid=5&Version=0&Cid=2196&Charset=big5_hkscs |title=守望我城——《論壇》三十年．回歸二十年，陳佐才〈在破碎中踐行不離地的真理〉 |date=2017-09-01 |publisher=時代論壇 |author=陳盈恩 }}</ref>

陳佐才善於融會中西文化，講課以有趣生動見稱，他喜愛閱讀[[莎士比亞|莎士比亞]]、[[蘇東坡|蘇東坡]]和[[金庸|金庸]]小說。他特別喜歡《[[笑傲江湖|笑傲江湖]]》中的令狐沖和《聖經》中的耶利米。<ref>{{ cite web |url=http://eastweek.my-magazine.me/main/66761 |title=陳法政的七色人生 |date=2017-05-29 |publisher=東周刊 |author=關慧玲 }}</ref>他尤其善於將金庸武俠小說的人物和情節融入神學討論之中，著有《武俠靈修》<ref>{{ cite book |url= |title=《武俠靈修》 |date=2009 |publisher=道風山基督教叢林 |author=陳佐才 |location=香港 | pages=  |ISBN=9789628294312 }}</ref>、《武俠人生：金庸小說的宗教情懷》<ref>{{ cite book |url= |title=《武俠人生：金庸小說的宗教情懷》 |date=1990 |publisher=突破出版社 |author=陳佐才 |location=香港 | pages=  |ISBN=9789622641075 }}</ref>、《天龍八部的閃閃靈光》<ref>{{ cite book |url= |title=《天龍八部的閃閃靈光》 |date=1998 |publisher=明窗出版社 |author=陳佐才 |location=香港 | pages=  |ISBN=9789629731014 }}</ref>等書。
