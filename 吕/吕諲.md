'''吕諲'''（{{bd|712年||762年||catIdx=L吕}}），[[蒲州|蒲州]]河东（今山西[[永济|永济]]）人，爵封'''须昌县开国伯'''，谥号'''肃'''，[[唐朝|唐朝]]官员，[[唐肃宗|唐肃宗]]年间拜相。史学家往往认为他后来担任地方官时比担任宰相更出色。<ref>如[http://www.sidneyluo.net/a/a17/140.htm 《新唐书》卷一百四十] {{webarchive|url=https://web.archive.org/web/20080921200856/http://www.sidneyluo.net/a/a17/140.htm |date=2008-09-21 }}</ref>

== 拜相前 ==
吕諲生于[[唐玄宗|唐玄宗]]登基前后的[[先天_(年号)|先天]]元年（712年），[[蒲州_(北周)|蒲州]]人氏，是[[春秋时期|春秋时期]][[姜姓齐国|姜姓齐国]]的后裔。<ref>[http://www.sidneyluo.net/a/a17/075.htm 《新唐书》卷七十五] {{webarchive|url=https://web.archive.org/web/20091219203329/http://www.sidneyluo.net/a/a17/075.htm |date=2009-12-19 }}</ref>吕諲年少时有大志，好学，但家贫不能自给。同乡富人[[程楚宾|程楚宾]]被吕諲的才华打动，相信他将来能出人头地，把女儿嫁给了吕諲，并和儿子[[程震|程震]]一起资助他，使他声誉日广，也得以在[[开元|开元]]末年游历京城[[长安|长安]]。<ref name=BT185>[http://www.sidneyluo.net/a/a16/185b.htm 《旧唐书》卷一百八十五下] {{webarchive|url=https://web.archive.org/web/20080917221804/http://www.sidneyluo.net/a/a16/185b.htm |date=2008-09-17 }}</ref><ref name=NBT140>[[:s:新唐书/卷140|《新唐书》卷一百四十]]</ref>

玄宗[[天宝_(唐朝)|天宝]]（742年 - 756年）初年，吕諲中[[进士|进士]]，调授[[宁陵|宁陵]]尉。采访使[[韦陟|韦陟]]欣赏吕諲的才华，署任他为支使。<ref>韦陟任陈留采访使约在天宝五载（746年）。</ref>天宝十三载（754年）三月，[[陇右|陇右]]、[[河西节度使|河西节度使]][[哥舒翰|哥舒翰]]邀请吕諲任[[度支判官|度支判官]]。<ref>[[:s:资治通鉴/卷217|《资治通鉴》卷二百一十七]]</ref>累兼卫佐、太子通事舍人。在哥舒翰手下，吕諲谨慎而努力地工作，在当时就知道诸将的长处和短处。有时同僚们都出去游赏了，他还在官署复查文件。他的勤勉使哥舒翰更喜欢他了。累兼虞部员外郎、[[殿中侍御史|殿中侍御史]]。<ref name=BT185/><ref name=NBT140/>

十四载（755年），[[安禄山|安禄山]]在[[范阳|范阳]]叛乱并迅速南进，攻陷唐朝东都[[洛阳|洛阳]]，建立[[大燕|大燕]]。哥舒翰奉命前往[[潼关|潼关]]阻击，却在十六载（756年）战败，唐玄宗只得弃长安逃往[[成都|成都]]。但太子[[李亨|李亨]]没有随驾，而是逃往[[灵武|灵武]]称帝，即[[唐肃宗|唐肃宗]]，唐玄宗后来也予以认可。当哥舒翰战败时，吕諲仍在他帐下，[[至德_(唐朝)|至德]]元载（756年）七月也骑马西逃到灵武[[行在|行在]]。在[[宦官|宦官]]中人尉[[朱光輝|朱光輝]]和[[李遵驟|李遵驟]]推荐下，肃宗召见了吕諲，欣赏他有才，超拜他为[[御史中丞|御史中丞]]，<ref>[[:s:资治通鉴/卷218|《资治通鉴》卷二百一十八]]</ref>对他的进奏无不允准。吕諲与逃奔灵武的奉先令[[崔器|崔器]]关系好，引为[[御史中丞|御史中丞]]兼[[户部侍郎|户部侍郎]]。二载（757年），肃宗移驾[[凤翔府|凤翔府]]，任吕諲为[[武部侍郎|武部侍郎]]，允许他穿高官的金紫袍服。十月，肃宗收复长安、洛阳，设置三司使，以[[御史大夫|御史大夫]]兼[[京兆尹|京兆尹]][[李岘_(梁国公)|李岘]]、吕諲、崔器、[[刑部侍郎|刑部侍郎]]兼御史中丞[[韩择木|韩择木]]、大理卿[[严向|严向]]五人为之，处置投降大燕的原唐朝官员。<ref>[[:s:旧唐书/卷50|《旧唐书》卷五十]]</ref>吕諲又荐崔器为[[吏部侍郎|吏部侍郎]]、御史大夫。<ref>[[:s:旧唐书/卷115|《旧唐书》卷一百一十五]]</ref>十二月，吕諲受命和时任礼部尚书李岘同为详理使，与崔器等官员一同负责处置陷燕罪臣。吕諲和崔器严厉，上言都应处死，肃宗几乎同意，而李岘比他们识大体，也较为仁慈，争执称“諲、器守文，不达大体”。<ref>[[:s:请宥陷贼官寮奏|:s:请宥陷贼官寮奏]]</ref>最终肃宗从李岘议，对这些官员以六等定罪，<ref>[[:s:资治通鉴/卷220|《资治通鉴》卷二百二十]]</ref>保全了很多人，<ref>[[:s:新唐书/卷209|《新唐书》卷二百零九]]</ref>如因故宰相[[张说|张说]]于肃宗有旧恩，其子燕[[中书令|中书令]][[张均|张均]]得免死。<ref name=NBT140/><ref>[[:s:旧唐书/卷97|《旧唐书》卷九十七]]</ref><ref name=BT112>[[:s:旧唐书/卷112|《旧唐书》卷一百一十二]]</ref><ref name=NBT131>[[:s:新唐书/卷131|《新唐书》卷一百三十一]]</ref>吕諲的严厉为自己招来了多人的鄙视。<ref name=BT185/>李岘也因此声誉高于吕諲、崔器。<ref>[[:s:新唐书/卷146|《新唐书》卷一百四十六]]</ref>

吕諲为河陇判官时，善于看相的[[金梁凤|金梁凤]]也在河陇，对他说：“判官的骨相，应该做宰相。吃一大惊怖，就得。”大约[[乾元_(唐朝)|乾元]]元年（758年），吕諲至驿站，责让驿长，笞打之。有驿吏是武将，性粗猛，持弓箭突入射吕諲两箭，几乎中其面，吕諲跳墙得免。报知金梁凤，金梁凤说：“这必是要入相了。”<ref>[[:s:旧唐书/卷191|《旧唐书》卷一百九十一]]</ref><ref>[[:s:新唐书/卷204|《新唐书》卷二百零四]]</ref>

== 拜相 ==
乾元二年（759年）三月，吕諲在武部侍郎任上被授[[同中書門下平章事|同中書門下平章事]]，<ref>[[:s:授吕諲同平章事制|:s:授吕諲同平章事制]]</ref>为[[实质宰相|实质宰相]]，知[[门下省|门下省]]事，尽管他本人并未在门下省任职。这是肃宗重组政府工作的一部分，肃宗同时也任李岘、[[李揆|李揆]]、[[第五琦|第五琦]]为相而罢免了原宰相[[苗晉卿|苗晉卿]]、[[王璵|王璵]]。累加[[银青光禄大夫|银青光禄大夫]]、'''东平男'''。李岘位望稍高，军国大事公卿不敢言，都独决于李岘，吕諲等因此不平怀恨。<ref name=BT112/><ref name=NBT131/>七月，吕諲荐时任太常卿的韦陟为礼部尚书、东京留守，判尚书省事，兼东京畿观察处置等使。<ref>[[:s:旧唐书/卷92|《旧唐书》卷九十二]]</ref><ref>[[:s:新唐书/卷122|《新唐书》卷一百二十二]]</ref>同月，吕諲因丁母忧而去职，但十月即被起复原职，<ref name=BT10>[[:s:旧唐书/卷10|《旧唐书》卷十]]</ref><ref name=NBT6>[[:s:新唐书/卷6|《新唐书》卷六]]</ref>十二月，兼任[[度支使|度支使]]，<ref name=ZZTJ221>[[:s:资治通鉴/卷221|《资治通鉴》卷二百二十一]]</ref>代第五琦。<ref name=BT49>[[:s:旧唐书/卷49|《旧唐书》卷四十九]]</ref>封爵须昌县伯，任[[黃門侍郎|黃門侍郎]]。[[上元_(唐肃宗)|上元]]元年（760年）初，肃宗将吕諲的宰相衔升为[[同中書門下三品|同中書門下三品]]，<ref>但《新唐书·宰相表》无此记载。参见[http://www.sidneyluo.net/a/a16/185b.htm 《旧唐书》卷一百八十五下] {{webarchive|url=https://web.archive.org/web/20080917221804/http://www.sidneyluo.net/a/a16/185b.htm |date=2008-09-17 }}、《新唐书》卷一百四十和[http://www.sidneyluo.net/a/a17/table/form02.htm 《新唐书》卷六十二] {{webarchive|url=https://web.archive.org/web/20080922174032/http://www.sidneyluo.net/a/a17/table/form02.htm |date=2008-09-22 }}的不同记载。</ref>赐[[门戟|门戟]]。有人对吕諲说不宜身着丧服接受如此吉庆之事，吕諲于是暂时褪去缞麻丧服，当中拜受，人们都笑他失礼。<ref name=BT185/><ref name=NBT140/>

吕諲拜相时，任岳父程楚宾为[[衛尉少卿|衛尉少卿]]，内兄弟程震為[[員外郎|員外郎]]，更加招致了批评，他本人又和出纳诏命的宦官[[馬上言|馬上言]]（或作马尚言）关系亲密。有人为了当蓝田尉而行贿马上言，吕諲在马上言要求下就补那人为蓝田尉。五月，事发，肃宗震怒，笞杀马上言，让其从官分食其肉。吕諲没有被杀，但被罢相，<ref name=BT10/><ref name=NBT6/>贬为太子[[李豫|李豫]]的宾客。<ref name=BT185/><ref name=NBT140/><ref name=ZZTJ221/>

== 拜相后 ==
七月，授吕諲[[荆州_(古代)|荆州]]大都督府[[长史|长史]]、兼御史大夫，充澧、朗、忠、硖五州节度观察处置等使。<ref name=BT10/>八月，他刚到荆州，便请求肃宗将军部[[江陵|江陵]]改成府并定为南都。九月，肃宗同意了，于是升江陵为江陵府，以吕諲为府尹。在吕諲要求下，他还派置永平军团练兵三千驻守江陵，以遏吴、蜀之冲，<ref name=ZZTJ221/>防当地发生叛乱。因吕諲称本镇无兵，在[[泌阳|泌阳]]有兵的监察御史里行[[元结|元结]]被进为水部员外郎，佐吕諲。<ref>[[:s:新唐书/卷143|《新唐书》卷一百四十三]]</ref><ref>[[:s:唐故容州都督兼御史中丞本管经略使元君表墓碑铭（并序）|颜真卿《唐故容州都督兼御史中丞本管经略使元君表墓碑铭（并序）》]]</ref>吕諲又奏取有才者数十人总管牙兵，威惠两行。二年（761年）正月，奏请将岳、潭、衡、連、道、邵、永、郴和黔中的[[涪州_(唐朝)|涪州]]共九州划给江陵府，增置永平军万人，获准。<ref name=NBT140/><ref>[[:s:旧唐书/卷39|《旧唐书》卷三十九]]</ref><ref name=ZZTJ222>[[:s:资治通鉴/卷222|《资治通鉴》卷二百二十二]]</ref>又从江陵府析置[[长宁县|长宁县]]。<ref name=BT185/>

吕諲任相时与同僚李揆不合。二月，吕諲在荆州得到美誉后，李揆不快，担心他再入相，声称湖南不需要士兵，建议解散吕諲手下的军队，还派人来刺探吕諲的过错。吕諲上奏肃宗为自己辩解并弹劾李揆。最后李揆被罢相贬为[[袁州|袁州]]長史。<ref name=BT185/><ref name=NBT140/><ref name=ZZTJ222/><ref>[[:s:旧唐书/卷126|《旧唐书》卷一百二十六]]</ref><ref>[[:s:新唐书/卷150|《新唐书》卷一百五十]]</ref>

奏于[[灌阳县|灌阳县]]故城复置其县。<ref>[[:s:旧唐书/卷40|《旧唐书》卷四十]]</ref>

吕諲的前任荆州长史[[張惟一|張惟一]]也是防御使，被属下司马[[陳希昂|陳希昂]]威胁。陈希昂是[[衡州|衡州]]蛮酋，有私军千人，把衡州当作自己的私人封地来管治。一次，他因和张惟一亲将[[牟遂金|牟遂金]]不和，率部卒闯入张惟一的官邸，要他下令处决牟遂金。张惟一出于恐惧下令处决了牟遂金，从此陈希昂掌控了荆州军政。吕諲到任后，先讨好陈希昂，上表荐举他去长安为侍御史，出为[[常州|常州]]刺史、本州防御使，却在其行经江陵拜谒时伏兵突然将其袭杀，尽斩其部下数十人，积尸于府门。府中慑服，<ref name=NBT140/>吕諲才奏其罪。吕諲自此得以控制江陵府。<ref name=BT185/>表大理司直[[严郢|严郢]]为判官。<ref name=NBT145>[[:s:新唐书/卷145|《新唐书》卷一百四十五]]</ref>

当时又有妖人[[申泰芝|申泰芝]]以妖术谄事李辅国，被任为[[道州|道州]]军校。申泰芝为官腐败，向当地群蛮勒索财物。邻近的[[潭州_(隋朝)|潭州]]刺史[[龐承鼎|龐承鼎]]对他早有不满，趁他来潭州时将他拘押，起获赃物，上表弹劾。申、庞都被带到长安，李辅国不但撇清自己，还反诬庞承鼎，命吕諲调查。吕諲命[[判官|判官]]、[[監察御史|監察御史]]嚴郢调查后上疏为庞承鼎辩白，证实申泰芝有罪。但肃宗深受李辅国蛊惑，下令[[赐死|赐死]]庞承鼎，流放严郢到[[建州_(唐朝)|建州]]。吕諲坚决反对，不顾危险，虽没法帮到庞、严二人，却赢得了百姓的尊敬，其守正刚断不挠得到人们看重。最后，申泰芝罪行大白而伏诛，庞承鼎得以平反。<ref name=BT185/><ref name=NBT140/><ref name=NBT145/>

[[宝应_(年号)|宝应]]元年（762年），肃宗召[[山南东道节度使|山南东道节度使]][[来瑱|来瑱]]赴京师，来瑱乐于在军部[[襄阳|襄阳]]，也为将士所爱，就指使所部将吏上表留之，行到[[邓州|邓州]]，肃宗复令他还镇。吕諲、[[淮西节度使|淮西节度使]][[王仲昇|王仲昇]]及往来中使说：“来瑱曲收众心，恐久后难制。”于是肃宗割商、金、均、房四州另置观察使，以来瑱为[[邓州|邓州]]刺史，只领六个州。当时王仲昇被燕帝[[史朝义|史朝义]]将谢钦让围攻数月，吕諲已久病，来瑱怀恨不救，三月，王仲昇兵败被俘。<ref name=ZZTJ222/><ref>[[:s:旧唐书/卷114|《旧唐书》卷一百一十四]]</ref><ref>[[:s:新唐书/卷144|《新唐书》卷一百四十四]]</ref>之前，二月，吕諲已卒，赠[[吏部尚書|吏部尚書]]，谥肃。时严郢为度支员外郎，请谥以二字“忠肃”，太常博士[[独孤及|独孤及]]坚议以“肃”字为当，<ref>[[:s:故江陵尹兼御史大夫吕諲谥议|:s:故江陵尹兼御史大夫吕諲谥议]]</ref><ref>[[:s:驳太常拟故相国江陵尹谥议|:s:驳太常拟故相国江陵尹谥议]]</ref>获准。<ref>[[:s:新唐书/卷162|《新唐书》卷一百六十二]]</ref><ref>[[:s:故朝散大夫使持节常州诸军事守常州刺史充本州团练守捉使赐紫金鱼袋独孤公谥议|权德舆《故朝散大夫使持节常州诸军事守常州刺史充本州团练守捉使赐紫金鱼袋独孤公谥议》]]</ref><ref>[[:s:常州刺史独孤及集后序|梁肃《常州刺史独孤及集后序》]]</ref><ref>[[:s:朝散大夫使持节常州诸军事守常州刺史赐紫金鱼袋独孤公行状|梁肃《朝散大夫使持节常州诸军事守常州刺史赐紫金鱼袋独孤公行状》]]</ref><ref>[[:s:故常州刺史独孤公神道碑铭（并序）|崔祐甫《故常州刺史独孤公神道碑铭（并序）》]]</ref>吕諲治理江陵三年，号为良守。起初郡人问他立生祠，吕諲死后一年多，江陵将吏合钱十万，于府西高爽干燥的地方大立祠宇，四时祠祷。<ref name=BT185/><ref name=NBT140/>元结作《[[:s:吕公表|吕公表]]》以示哀悼。《舆地碑记目》卷二《江陵府碑记》有《唐江陵尹吕諲庙碑》《唐江陵尹吕公表》，并注引《集古录》称是元结所撰。<ref>《集古录》误作肃宗元年立。</ref>

《新唐书》如是评价吕諲：

“吕諲为相时，不称职。但到了荆州以后，号令明确，赋税合理。治理时崇尚威信，所以军士用命，全境没有盗贼，百姓也歌咏他。自至德（肃宗继位初的年号，756年 - 758年）以来的几十个地方官中，吕諲最有名声。他在世时荆州百姓就为他建了祠堂，他死后下属为他建了一个更大的。”<ref name=NBT140/>

在荆州期间，吕諲了解到[[杜鴻漸|杜鴻漸]]和[[元载|元载]]的才干，举荐他们，后来他们都成为宰相。<ref name=NBT140/>其幕府还曾有时任尚书郎的[[源休|源休]]等。<ref>《全唐诗·元结<寄源休并序>》</ref>吕諲去世当年五月，时任[[中书侍郎|中书侍郎]]的元载即代其为度支使。<ref name=BT49/>元结后来作《[[:s:举吕著作状|举吕著作状]]》举荐吕諲时任秘书省著作郎的侄子吕季重时，称吕諲“立身无私，历官清俭，身殁之后，家无余财”。吕諲死后四年，有人提出给吕諲改谥号。<ref>[[:s:驳太常停谥陇右节度使郭知运议|独孤及《驳太常停谥陇右节度使郭知运议》]]</ref>
