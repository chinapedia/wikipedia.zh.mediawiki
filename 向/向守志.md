'''向守志'''（{{bd|1917年|11月28日|2017年|9月2日|X向}}），原名'''向守芝'''，[[四川省|四川省]][[宣汉县|宣汉县]]南坝场人，[[中国人民解放军上将|中国人民解放军上将]]，[[中国共产党第十二届中央委员会|中国共产党第十二届中央委员会]]委员，[[中国共产党中央顾问委员会|中国共产党第十三届中央顾问委员会]]委员<ref name=shishi>{{Cite web |url =http://news.ifeng.com/a/20170902/51846411_0.shtml |title = 开国少将、原南京军区司令员向守志逝世 终年100岁 |author = |date =2017-09-02|publisher = 凤凰网 |language = |accessdate = |quote = }}</ref>。

== 生平 ==
=== 民国时期 ===
向守志出生在川东北[[大巴山|大巴山]]区[[宣汉县|宣汉县]]双河区南坝场一个贫苦农民家庭。1933年参加[[中国少年先锋队|少年先锋队]]。1934年7月参加[[红四方面军|红四方面军]]，1935年加入[[中国共产主义青年团|中国共产主义青年团]]。历任[[红九军|红九军]]二十七师七十六团二营四连战士、班长。参加了[[川陕苏区|川陕苏区]]反“六路围攻”作战。1935年3月28日夜，[[徐向前|徐向前]]指挥红九军与[[红三十军|红三十军]]、[[红三十一军|红三十一军]]主力，兵分三路，强渡[[嘉陵江|嘉陵江]]西进，拉开了红四方面军长征的序幕。向守志随部参加了[[长征|长征]]。1935年6月18日，向守志所在的红四方面军部队与[[中央红军|中央红军]]在小金川会师，向守志首次见到了[[毛泽东|毛泽东]]、[[张闻天|张闻天]]、[[周恩来|周恩来]]、[[朱德|朱德]]等中央领导。1936年夏升任副排长。1936年秋，部队抵达[[甘肃|甘肃]]南部。在[[岷县|岷县]]围歼敌军的战斗中，因作战勇敢而受师领导表扬。长征结束后，调陕北红军大学（后来改成[[庆阳市|庆阳]]红军步兵学校）初级科学习，担任学员队的班长兼学习小组长，并在1936年9月转入[[中国共产党|中国共产党]]。其间，参加了[[山城堡战役|山城堡战役]]的战斗准备。在一年的学习中，因成绩优良而受奖励<ref name=shishi/><ref name=dazhou/><ref name=dadi>{{Cite web |url = http://dangshi.people.com.cn/GB/165617/165618/166491/167909/12080640.html |title = 共和国老将军的情怀——访南京军区原司令员向守志上将 |author = 张德、陈松鹤 |date = |publisher = 大地2007年第16期 |language = |accessdate = |quote = }}</ref><ref name=shiwusui>{{Cite web |url =http://news.sina.com.cn/c/nd/2017-09-02/doc-ifykpuui0462313.shtml |title = 开国将军向守志:15岁扛起红缨枪 曾亲历上甘岭|author = |date = 2017-09-02 |publisher = 新浪 |language = |accessdate = |quote = }}</ref>。

1937年毕业后，向守志被分配到[[八路军|八路军]][[129师|一二九师]]三八六旅七七一团任机枪连副连长。[[抗日战争|抗日战争]]全面爆发后，1937年9月6日，部队在[[陕西省|陕西省]][[三原县|三原县]]举行誓师大会，会后北上渡过[[黄河|黄河]]赴[[山西省|山西]]抗日前线[[平定县|平定县]]东。1937年10月中旬，[[日军|日军]]一个旅团的兵力自[[井陉县|井陉]]向南由平定县进至阳泉镇，企图切断[[旧关|旧关]]和[[娘子关|娘子关]]八路军的后路，进犯[[太原市|太原]]，八路军一二九师以三八六旅七七一团为主在七亘村两次伏击日军，此即[[七亘村战斗|七亘村战斗]]，毙敌400多人，缴获骡马300多匹，向守志随部队参战。1938年2月，向守志升为三八六旅七七一团二营机枪连连长。1938年3月，师长[[刘伯承|刘伯承]]命旅长[[陈赓|陈赓]]在神头岭伏击日军，即[[神头岭战斗|神头岭战斗]]，毙伤俘日军1500余人，缴获长短枪500余支、骡马600余匹，向守志在战斗中指挥机枪连的6挺[[马克沁机枪|马克沁机枪]]向日军猛烈射击。半个月后，刘伯承、[[邓小平|邓小平]]、[[徐向前|徐向前]]决定以3个主力团在[[河北省|河北]][[涉县|涉县]]以西、山西[[黎城县|黎城县]]东阳关以东的响堂铺地段切断日军的运输线[[邯郸市|邯（郸）]][[长治市|长（治）]]大道。1938年3月31日进行了[[响堂铺战斗|响堂铺战斗]]，歼日军400余名，日军汽车全部被毁，向守志参战。战后被提升为特务营营长，随后改任第二营营长<ref name=shishi/><ref name=dazhou>{{Cite web |url = http://www.dzrbs.com/html/2017-06/23/content_265777.htm |title =闪耀的星群 |author = |date = 2017-06-23 |publisher = 达州日报网 |language = |accessdate = |quote = }}</ref>。后来升为副团长，[[太行军区|太行军区]]第十团团长<ref name=lao>{{Cite web |url = http://dangshi.people.com.cn/n1/2016/1010/c85037-28765702.html|title = 参加长征的老红军向守志：“归根结底，胜利靠党的领导” |author = |date =2016-10-10 |publisher = 中国共产党新闻网 |language = |accessdate = |quote = }}</ref>。先后参加了[[林南战役|林南战役]]、对日大反攻战役<ref name=shiwusui/>。

[[第二次国共内战|第二次国共内战]]时期，任太行军区第一支队副支队长，[[中国人民解放军第二野战军|晋冀鲁豫野战军]]六纵十八旅副旅长，太行军区独立第二旅旅长。1947年4月，在[[豫北战役|豫北战役]]中，率太行军区独立第二旅担负阻击任务，促进友邻部队歼灭[[中華民國國軍|国军]]第二快速纵队，全旅受到晋冀鲁豫军区通令嘉奖。1947年8月，改任晋冀鲁豫、[[中国人民解放军第二野战军|中原野战军]]九纵二十六旅旅长，随陈谢兵团南渡[[黄河|黄河]]，参加[[洛阳战役|洛阳战役]]、[[淮海战役|淮海战役]]等。1949年，任[[中国人民解放军第二野战军|中国人民解放军第二野战军]][[中国人民解放军空降兵军|第十五军]]四十四师师长兼政委<ref name=shiwusui/><ref name=lao/>。

=== 共和国时期 ===
[[中华人民共和国|中华人民共和国]]成立后，1951年任[[中国人民志愿军|中国人民志愿军]]第十五军四十四师师长，参加[[朝鲜战争|抗美援朝战争]]。1952年10月12日晨，向守志命八十七团三营400余人在发起攻击前秘密潜伏在距391高地前沿阵地数百米开阔地的蒿草丛和小杂树中，该营战士[[邱少云|邱少云]]遭敌燃烧弹延烧时，为不暴露目标而犧牲，当天傍晚三营战士夺取391高地。向守志先后参加了[[抗美援朝第五次战役|第五次战役]]和防御作战，亲历[[上甘岭战役|上甘岭战役]]。1951年冬，在第五次战役期间第二阶段作战中，向守志头部负伤，1954年回国<ref name=dadi/><ref name=shiwusui/><ref name=lao/>。

1954年起，向守志历任第十五军参谋长、第一副军长兼参谋长、军长。1957年8月向守志进入[[中国人民解放军高等军事学院|中国人民解放军高等军事学院]]基本系一班学习，学期三年。1960年8月，毕业后的向守志出任[[西安炮兵学校|西安炮兵学校]]校长，该任命由[[中华人民共和国国务院总理|中华人民共和国国务院总理]][[周恩来|周恩来]]签发。1962年，该校改为中国人民解放军第二炮兵工程技术学院，向守志改任院长<ref name=dadi/><ref name=lao/>。

1965年8月，由[[毛泽东|毛泽东]]点将，国务院总理周恩来任命向守志为[[中国人民解放军炮兵|军委炮兵]]副司令员，主管[[导弹|导弹]]部队建设。1966年6月6日，中央军委决定组建[[中国人民解放军火箭军|中国人民解放军第二炮兵部队]]领导机关，由向守志和[[李天焕|李天焕]]负责筹建。1967年7月4日，毛泽东亲自签发任命书，由中央军委下达，任命向守志为第二炮兵司令员，李天焕为政委。向守志成为首任第二炮兵司令员。43天后，[[林彪|林彪]]签发命令，撤消对向守志的任命<ref name=lao/><ref name=dadi/>。1967年，受到[[文化大革命|文化大革命]]影响，被关押。此后6年多时间里，向守志在批斗、游街、隔离、劳动改造中度过。他的亲人无一幸免。他和妻子张玲4年半不准通信<ref name=lao/><ref name=dadi/><ref name=dazhoue/>。1969年底，林彪发出“[[林副主席指示第一号令|林副主席指示第一号令]]”，以战备为由，命令所有在北京的军队高级干部及家属全部撤离北京。张玲和80岁的老母亲及4个子女不得不到[[四川省|四川省]][[开州区|开县]]（今属[[重庆市|重庆市]]）毕山农村向守志的外甥家“投亲靠友”，向守志则先后辗转[[天津市|天津]]、[[郑州市|郑州]]、[[叶县|叶县]]等五个农场从事养猪、种菜等劳动。直到林彪在[[九一三事件|九一三事件]]垮台后的1972年初，向守志才获得恢复党籍、补发工资的通知，并带着家人回到北京<ref name=dazhoue/>。 

1974年11月的一天，中央军委副主席[[叶剑英|叶剑英]]召见已回炮兵（又称一炮）工作的向守志，叶剑英说：“任命你为第二炮兵司令员这件事我是知道，撤消你的职务我不知道。”叶剑英还表示军委准备仍派向守志担任第二炮兵司令员。1975年3月25日晚，[[中国人民解放军总参谋部|中国人民解放军副总参谋长]]兼[[中央军事委员会办公厅|中央军委办公厅]]主任[[胡炜|胡炜]]打电话给向守志说：“叶帅说了，你明天就去二炮报到上班。”向守志再度出任第二炮兵司令员，并兼任第二炮兵党委第一书记。1975年8月30日，中央军委发出通知，公布了经[[中国共产党中央委员会|中共中央]]、毛泽东主席批准的各总部、各军兵种、各大军区主官的任命，其中第二炮兵司令员仍为向守志<ref name=dadi/>。

1977年9月，调任[[中国人民解放军南京军区|南京军区]]副司令员兼党委常委。1982年10月升任南京军区司令员兼党委书记。1990年5月离休<ref name=lao/>。

向守志离休后，坚持[[书法|书法]]、[[绘画|绘画]]、赋诗，并热衷少年儿童教育以及慈善事业<ref name=dadi/>。2002年11月8日，[[中国共产党第十六次全国代表大会|中共十六大]]时作诗：“盛世盛会十六大，“[[三个代表|三个代表]]”是灯塔，昂首阔步奔[[小康|小康]]，前程辉煌富中华。”2003年6月为纪念毛泽东诞辰110周年作诗：“丰功伟绩千秋颂，开天辟地东方红。典范长存励后人，盛世腾飞慰毛公。”<ref name=dadi/>2015年3月，向守志接受《[[解放軍報|解放軍報]]》记者采访时称，“[[习近平|习主席]]在[[中华人民共和国国家主席新年贺词|新年贺词]]中深情地说，‘我要为我们伟大的人民点赞’，我深受感染，也想说句心里话——我为我们伟大的党点赞！”并评论[[周永康|周永康]]、[[徐才厚|徐才厚]]等人“抛弃信仰，背叛党旗下的誓言，走到党和人民的对立面，可耻可悲。”<ref>{{cite news |title=“我为我们伟大的党点赞”——访98岁老红军、南京军区原司令员向守志|url=http://www.81.cn/jfjbmap/content/2015-03/21/content_105341.htm |publisher=解放军报|date=2015-03-21|accessdate=2015-03-21 }}</ref>

1955年被授予[[中国人民解放军少将|少将]]军衔。1988年9月被授予[[中国人民解放军上将|上将]]军衔。曾获三级[[八一勋章_(1955年)|八一勋章]]、二级[[独立自由勋章|独立自由勋章]]、二级[[解放勋章|解放勋章]]、一级[[红星功勋荣誉章|红星功勋荣誉章]]。他是[[中国共产党第十一次全国代表大会|中共十一大]]代表，[[中国共产党第十二届中央委员会委员列表|十二届中央委员]]，十三届[[中国共产党中央顾问委员会|中顾委]]委员。[[中国共产党第十六次全国代表大会|中共十六大]]列席代表<ref name=dadi/>。

2017年9月2日，向守志在[[南京市|南京]]病逝，享年100岁<ref name=shishi/>。9月10日，在南京西天寺殡仪馆博爱厅向向守志同志遗体最后致哀，根据[[中共中央|中共中央]]丧事从简的规定，不邀请南京以外的单位和个人来参加悼念活动<ref>{{Cite web |url = http://www.thepaper.cn/newsDetail_forward_1790007 |title = 原南京军区司令员向守志同志遗体最后致哀仪式今日在南京举行 |author =  |date = 2017-09-10 |publisher = 澎湃新闻 |language =  |accessdate =  |quote =  }}</ref>。

==家庭==
*妻子：张玲。[[河南省|河南]][[开封市|开封]]人，1919年生，1935年参加[[一二·九运动|一二·九运动]]，1937年12月参加革命，1938年加入[[中国共产党|中国共产党]]。历任县委常委、[[东平县|东平县]]委书记、[[宜宾市|宜宾市]]委书记、[[湖北工程学院|孝感大学]]党委书记等职。1965年因病离休。抗日战争中，1944年在太行区党委党校[[延安整风运动|整风]]学习班学习期间与向守志结识。1945年初，经刘伯承和邓小平同意，太行军区政治部批准了向守志和张玲的结婚申请。1945年5月25日结婚<ref name=dadi/><ref name=hbgc/>。1960年，41岁的张玲罹患癌症。1994年被评为全国“抗癌明星”<ref name=kangai>{{Cite web |url = http://news.163.com/10/1225/15/6OOQ9DFE00014AED.html |title = 我的妈妈张玲 |author = 向安黎 |date = 2010-12-25 |publisher = 网易 |language = |accessdate = |quote = }}</ref>。向守志有4个儿女，成年后两个在北京工作，一个在[[香港|香港]]工作，一个在[[安徽省|安徽]]工作<ref name=dadi/>。
