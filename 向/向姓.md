'''向姓'''是中文的姓氏之一，在[[百家姓|百家姓]]中排第129位，按人口计算排名第99位。

== 起源 ==
;漢族
*出自[[姜姓|姜姓]]，为炎帝神农氏之后。神农氏有裔孙名向，被封为诸侯，其后代子孙以向为姓，遂成向姓。
*出自[[祁姓|祁姓]]，为帝尧的后裔，以国为姓。周代有向国，故址在今山东莒县南部。后来国灭，向国国君的后代就以原国名为姓，成为向姓的一支。
*出自[[子姓|子姓]]，为汤王的后代。春秋时期，宋国国君宋桓公有子名[[向父肹|肹]]，字向父，世代为宋国卿士，其后亦姓向，成为向氏。
*出自[[相姓|相姓]]，由土家族相姓演變，因「向」與「相」原是同音異譯，故土家人訛「相」為「向」。

;土家族
向姓為[[土家族|土家族]]大姓，由相姓演变而来，因“向”与“相”原是同音异译，故土家人讹“相”为“向”。复旦大学谭其骧教授在《近代湖南人之蛮族血统》一文中说：“向氏为湖南蛮姓中之最早见于记载者。后汉建武二十三年，武陵蛮精夫[[相单程|相单程]]作乱。相氏疑即向氏，相、向同音而异译也”。

[[北宋|北宋]]时期，澧水流域车溪洞土酋[[向克武|向克武]]率领土人归附宋朝。宋代开宝五年（公元972年），朝廷下旨改车溪洞为柿溪州，授土酋向克武为柿溪州宣抚使，允其子孙世代承袭。向克武遂设柿溪州宣抚司衙署（在今桑植县上洞街乡政府）。

[[桑植县|桑植县]]《向氏族谱》记载了这一史实：“肇自祖公，向姓讳克武，于唐末宋初客游南楚，因入溪峒。是时大乱，土宇瓜分。流寇马殷占据湖南，称为楚王。公遂隐居山峒。公本盛德长者，乃为土众推（戴）为部落（酋）长。越十余载，天下宗宋，四海一家，万姓一体，公率各峒酋首倡向化，调征苗叛，得蒙嘉奖‘忠顺’，以为各峒酋长，仍住车溪峒，改为湖南柿溪州，（钦授向克武）为（柿溪州）军民宣抚使。”向克武死后，其子[[向万民|向万民]]袭职，忠心辅国，仁德爱民。后因无嗣，传弟向万才<ref>[http://www.tujiazu.org.cn/contant.asp?channelid=1&classid=52&id=2104 张家界市土家姓氏源流综述]</ref>。

;琉球族
向姓為[[琉球族|琉球族]]大姓，是[[琉球國|琉球國]]尚氏王室的旁支。

== 郡望 ==
河南郡：汉高帝置，在今河南省洛阳市一带。	

== 歷史名人 ==
*[[向朗|向朗]]：東漢末年三國時蜀漢官員
*[[向寵|向寵]]：東漢末年三國時蜀漢將領。向朗之姪
*[[向秀|向秀]]：魏末晋初竹林七贤之一
*[[向海明|向海明]]：隋末农民起义军领袖。他原为扶风（今陕西凤翔）僧人，自称"弥勒"出世。大业九年（613年）率众起义，后为隋将杨义臣所败
*[[向子韶|向子韶]]：开封人，宋朝元符年间进士。建炎初年任淮宁知府
*[[向士壁|向士壁]]：常州人，南宋名将，绍定年間进士，文官至大理卿直龙图阁，武官至兵部侍郎兼转运使，后因贾似道妒其功，被诬告杀害
*[[向侃|向侃]]：巢县人，明成祖永乐年间舉人，宣德中任靖宁州判官，后升任监察御史，一度又出为知府
*[[向警予|向警予]]：原名向俊贤，土家族，中共早期著名的妇女运动领导人之一
*[[向忠发|向忠发]]：中共中央总书记
*[[向華勝|向華勝]]，香港電影出品人
*[[向柏榮|向柏榮]]，香港搏擊拳王

*[[向艳梅|向艳梅]]，中國舉重運動員
*[[向洋|向洋]]，新加坡男歌手、演员
