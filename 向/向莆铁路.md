'''向莆铁路'''是联络[[中国|中国]][[江西省|江西省]][[南昌市|南昌市]]和[[福建省|福建省]][[莆田市|莆田市]]以及[[福州市|福州市]]的铁路，因原计划从江西省南昌市[[南昌县|南昌县]][[向塘镇|向塘镇]]的[[向塘西站|向塘西站]]到福建省莆田市而得名，后因优化调整改为从[[昌九城际铁路|昌九城际铁路]]的[[乐化东站|乐化东站]]接轨到[[南昌西站|南昌西站]]，再向东南经[[南昌县|南昌县]][[广福镇_(南昌县)|广福镇]]，绕开向塘镇直达[[抚州|抚州]]，再经[[南城|南城]]、[[南丰|南丰]]以及福建[[建宁|建宁]]、[[泰宁|泰宁]]、[[将乐|将乐]]、[[沙县|沙县]]、[[尤溪|尤溪]]、[[永泰|永泰]]，在永泰分岔后，一条线路通往福州，另一条线路通往莆田<ref>{{cite web |url=http://www.fjsen.com/fujian/2009-06/10/content_720227.htm |title=向莆铁路预计2013年通车 |accessdate=2009-12-08 |author= |publisher=东南新闻网 |language=zh-hans |deadurl=yes |archiveurl=https://web.archive.org/web/20110610012027/http://www.fjsen.com/fujian/2009-06/10/content_720227.htm |archivedate=2011-06-10 }}</ref>。

== 概况 ==
全线设计时速为200公里。线路长度共计635.861公里（含永泰至[[福州|福州]]支线），其中江西段线路全长245.045公里，福建段线路总长390.816公里，预计总投资约518亿元。向莆铁路从南昌到福州沿线共设20个车站，营业里程为547公里，运营时速为200公里，而在福建永泰分线至莆田，沿线则设涵江北、莆田2个车站，营业里程为59公里，运营时速为200公里。

向莆铁路全线隧道共计118座<ref>[http://www.jx.xinhuanet.com/news_center/2007-11/23/content_11748627.htm 总投资５８１亿元 向莆铁路明日正式开工] {{webarchive|url=https://web.archive.org/web/20150223090555/http://www.jx.xinhuanet.com/news_center/2007-11/23/content_11748627.htm |date=2015-02-23 }}</ref>，福建境内的桥隧比堪称亚洲最高<ref>{{Cite web |url=http://news.66163.com/2007-11-24/63304.shtml |title=向莆线福州桥隧比密度是亚洲最高 |access-date=2015-02-23 |archive-url=https://web.archive.org/web/20150223091545/http://news.66163.com/2007-11-24/63304.shtml |archive-date=2015-02-23 |dead-url=yes }}</ref>福建段桥隧占总路段9成<ref>[http://news.sina.com.cn/c/2007-11-24/000812959510s.shtml 向莆铁路福建段 桥梁隧道占9成]</ref>。

== 历史 ==
=== 前期准备 ===
该铁路的建设最早于2004年11月13日由江西省政府向国务院上报《关于恳请建设向莆铁路的请示》而提出的<ref>[http://news.fznews.com.cn/zt/2007/xpxstl/zxbd/2007-11-23/20071123qk-xj_qEgg135741.shtml 从２００４年至２００７年 向莆铁路建设大事记]</ref>，该项目的早期规划是从南昌[[向塘|向塘]]枢纽引出途径[[抚州|抚州]]、[[三明|三明]]最终到达[[莆田|莆田]]<ref>[http://jiangxi.jxnews.com.cn/system/2005/01/25/000789734.shtml 网友与人大代表热议：向莆铁路该不该建？]</ref>；经过了一年的时间铁道部在2005年12月6日正式确认该项目名称为“向莆铁路”，同年12月24日至26日铁道部就向莆铁路进行勘测<ref>[http://www.jxnews.com.cn/jxrb/system/2005/12/25/002181830.shtml 铁道部就向莆铁路建设来赣踏勘]</ref>，这时规划在福建三明分开后一条通往莆田另一条通往福州<ref>[http://jiangxi.jxnews.com.cn/system/2005/12/26/002182155.shtml 江西将出资50%与福建、铁道部共建向莆快速铁路]</ref>。

2006年8月25日国家发改委研究批准了向莆铁路项目建议书，位于南昌的终点站将改到[[南昌西站|新南昌站]]<ref>[http://jtyss.ndrc.gov.cn/gzdt/200609/t20060913_84254.html 我委批复新建向塘至莆田铁路项目建议书]</ref><ref>[http://news.jxnews.com.cn/system/2006/08/27/002324131.shtml 向莆铁路获国家批准立项]</ref>，2006年10月10日《新建铁路向莆线向塘至莆田（福州）段环境影响报告书（简本）》公示<ref>[https://web.archive.org/web/20061116163523/http://www.crfsdi.com/hg/index.htm 新建铁路向莆线向塘至莆田（福州）段环境影响报告书（简本）]</ref>；<ref>[http://www.jxdpc.gov.cn/departmentsite/tb/zdgc/zdxmxx/200612/t20061204_55263.htm 向塘至莆田铁路]</ref>。

2007年4月确认向莆铁路在抚州的走向<ref>[http://jxfz.jxnews.com.cn/system/2007/04/12/002465274.shtml 向莆铁路经我市线路基本确定]</ref>；2007年8月29日国家发改委批准可行性研究报告，这时确认向莆铁路在福建的分叉点由三明沙县移动到[[永泰县|永泰]]<ref>[http://jtyss.ndrc.gov.cn/gzdt/200710/t20071008_163604.html 新建向塘至莆田铁路可行性研究报告通过批准]</ref><ref>[http://www.jxnews.com.cn/jxrb/system/2007/08/31/002555982.shtml 向莆铁路建设项目获国家批准]</ref>；而岗上越行站被取消，而黎川站（西城站）将作为预留<ref>[http://jiangxi.jxnews.com.cn/system/2007/09/27/002576042.shtml 赣江再添大桥贯通向莆铁路(图)]</ref>。

=== 开工建设 ===
2007年11月24日向莆铁路江西段开工<ref>[http://jiangxi.jxnews.com.cn/system/2007/11/23/002617848.shtml 向莆铁路（江西段）今日开工（组图）]</ref><ref>[http://www.jxgdw.com/jxgd/news/jszg/userobject1ai753858.html 向莆铁路江西段开工 全线长约603.6公里 ] {{webarchive|url=https://web.archive.org/web/20131017193106/http://www.jxgdw.com/jxgd/news/jszg/userobject1ai753858.html |date=2013-10-17 }}</ref>。全线最长的[[青云山隧道|青云山隧道]]（长约22公里）于2008年8月8日动工<ref>[http://news.fznews.com.cn/fuzhou/2008-8-10/2008810e_1k_4es--20305.shtml 向莆铁路青云山隧道８日动建 总投资１５.１６亿]</ref>；2008年11月全线正式开工<ref>[http://www.chnrailway.com/news/20081127/200811272132613936556.shtml 向莆铁路尤溪至永泰段开工]</ref>。

2012年6月底，全线116条隧道仅剩联络线隧道一座（九峰山隧道）未贯通，正线隧道已全部贯通，247座桥梁已完工242座，铺轨完成设计的54.7%<ref>[http://www.hinews.cn/news/system/2012/07/25/014686423.shtml 向莆铁路铺轨完成过半 福建到湖南最便捷通道]</ref>；2012年11月29日全线贯通<ref>[http://jndsb.jxnews.com.cn/system/2012/11/30/012195649.shtml 向莆铁路全线铺轨贯通[图<nowiki>]</nowiki>]</ref>。

=== 联调联试 ===

2013年1月7日向莆铁路南昌西线开始调试<ref>[http://www.jxnews.com.cn/jxrb/system/2013/01/08/012242146.shtml 向莆铁路进行调试[图<nowiki>]</nowiki>]</ref>，1月10日铁道部正式批复向莆铁路线路正式命名，南昌西至福州为[[昌福线|昌福线]]，永泰至莆田为[[永莆线|永莆线]]，而乐化东站至南昌西站归入[[昌九城际线|昌九城际线]]<ref>[http://jxfz.jxnews.com.cn/system/2013/01/16/012253427.shtml 1月10日铁道部批复向莆铁路线路正式命名]</ref><ref>[http://www.gov.cn/jrzg/2013-01/17/content_2314240.htm 南昌铁路局：向莆铁路正式命名为昌福线、永莆线]</ref><ref>[http://news.163.com/13/0116/20/8LCA44AH00014JB6.html 向莆铁路正式命名为昌福线和永莆线年内将全线通车]</ref>；1月17日行走[[向乐铁路|向乐铁路]]的7206次列车从三江镇驶入向莆铁路行走至抚州北站，原向乐铁路三江镇到抚州北区间在建设向莆铁路时给予向莆铁路重新建设使用<ref>[http://jiangxi.jxnews.com.cn/system/2013/01/18/012256308.shtml 向乐支线列车在向莆铁路运行]</ref>。

2013年1月30日江西段最后一座桥梁架设完毕<ref>[http://jiangxi.jxnews.com.cn/system/2013/02/01/012275621.shtml 桥通啦 向莆铁路昌福线江西段最后一座桥梁架通]</ref>，4月28日江西乐化到福建尤溪段正式送电<ref>[http://jiangxi.jxnews.com.cn/system/2013/04/25/012390597.shtml 向莆铁路28日起开始送电 铁路部门提醒勿在沿线放风筝]</ref><ref>[http://jiangxi.jxnews.com.cn/system/2013/04/28/012396388.shtml 向莆铁路今起正式送电 沿线居民注意安全防范]</ref>，8月9日开始全线联调联试<ref>[http://www.gov.cn/jrzg/2013-08/09/content_2464041.htm 向莆铁路全线联调联试开始 设计时速200公里]</ref><ref>[http://www.jxnews.com.cn/jxrb/system/2013/08/09/012564689.shtml 今天坐动检车游向莆铁路去]</ref>。

=== 开通运营 ===
2013年9月21日起开始发售26日车票<ref>[http://www.nctlj.com.cn/Show.aspx?TID=3250 南昌火车站今日发售首张向莆线动车车票]</ref><ref>[http://www.12306.cn/mormhweb/zxdt/201309/t20130920_952.html 关于昌福、永莆线旅客列车车票发售的公告]</ref>，9月23日在新闻发布会确认将在26日运行<ref>[http://www.nctlj.com.cn/Show.aspx?TID=3255 向莆铁路9月26日开通运营（新闻通稿）]</ref><ref>[http://jxfz.jxnews.com.cn/system/2013/09/26/012677808.shtml 向莆铁路大事记]</ref>，9月26日正式通车<ref>[http://tv.cntv.cn/video/C10616/188b50697f974b229b9005bea1fc4cfc 《新闻直播间》 20130926 09:00]</ref><ref>[http://www.jxnews.com.cn/jxrb/system/2013/09/26/012675274.shtml 向莆铁路正式开通运营[图<nowiki>]</nowiki>]</ref><ref>[http://stock.sohu.com/20130922/n386971396.shtml 向莆铁路将于26日开通 南昌至福州3小时直达] {{Wayback|url=http://stock.sohu.com/20130922/n386971396.shtml |date=20130927202824 }}</ref>。

运行初期安排开行动车组列车18.5对，直通动车组为5对，另外有9对货车。

== 影响 ==
向莆铁路的通车使得千里赣闽一日往返成为现实，南昌与福州的最快铁路旅行时间由原来的11小时压缩至3小时12分，南昌与厦门的最快铁路旅行时间由原来的17小时压缩至4小时23分<ref>[http://www.gov.cn/jrzg/2013-09/26/content_2495277.htm 向莆铁路正式开通运营 将实现千里赣闽一日往返]</ref>，向莆铁路结束了[[南城|南城]]、[[南丰|南丰]]、[[建宁县|建宁]]、[[泰宁|泰宁]]、[[将乐|将乐]]、[[尤溪|尤溪]]、[[永泰县|永泰]]七个县不通火车的历史<ref>[http://jiangxi.jxnews.com.cn/system/2013/09/26/012675301.shtml 向莆铁路今日正式开通运营[图<nowiki>]</nowiki>]</ref>；铁路还结束了江西唯一一个地级市抚州的市区只有断头铁路的历史（[[向乐铁路|向乐铁路]]是地方工业支线，到抚州[[乐安县|乐安县]]断头）。

向莆铁路的通车也使得[[中国高速铁路|中国高速铁路]]总里程突破1万公里<ref>[http://ce.jxcn.cn/system/2013/09/26/012677261.shtml 向莆铁路开通运营 中国高铁里程破万]</ref>，而铁路的通车也导致南昌前往厦门的飞机票价打到2折<ref>{{Cite web |url=http://news.jxgdw.com/jxxw/sh/2254683.html |title=向莆铁路运行通车后 民航和汽运日子难过了 |access-date=2013-10-11 |archive-url=https://web.archive.org/web/20131012180128/http://news.jxgdw.com/jxxw/sh/2254683.html |archive-date=2013-10-12 |dead-url=yes }}</ref>，福州或泉州到南昌的航班取消<ref>[http://jiangxi.jxnews.com.cn/system/2013/10/16/012720339.shtml 向莆线动车逼停南昌至福州航线]</ref>，但是铁路的通车使得福州的国际航班需求量增加<ref>[http://www.taihainet.com/news/xmnews/cjdc/2013-10-25/1154663.html 向莆铁路开通冲击航空市场 厦航取消福泉飞南昌航班]</ref>。

由于[[向乐铁路|向乐铁路]]只适合运行低等级的普通旅客列车，这不利于向莆铁路的运输安全，因此向乐铁路唯一的一班列车7205/6次列车已于2013年5月5日起停运<ref>[http://jiangxi.jxnews.com.cn/system/2013/08/30/012609406_02.shtml 鹿心社与网民在线交流 37个问题落实情况公布]</ref>。

有乘客由于发音不准，在购买福州或抚州的车票时容易造成售票员误解<ref>[http://weibo.com/2072751944/Af3imro0j 【福州？抚州？】]</ref>。
