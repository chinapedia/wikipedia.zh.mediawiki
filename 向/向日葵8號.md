'''向日葵8號'''（{{lang-ja|ひまわり8号}}）是日本[[氣象衛星|氣象衛星]]之一，為[[向日葵系列衛星|向日葵系列衛星]]其中之一款[[人造衛星|人造衛星]]，由[[氣象廳_(日本)|日本氣象廳]]開發，[[三菱電機|三菱電機]]製造。於2014年10月7日由[[三菱重工业|三菱重工业]]<ref name="mhi"/>與[[宇宙航空研究開發機構|宇宙航空研究開發機構]]<ref name="jaxa"/>將其送上軌道，屬於[[地球同步衛星|地球同步衛星]]。向日葵8號於2015年7月7日測試完成正式投入運作，目的是取代於2006年投入服務的[[多功能运输卫星#MTSAT-2|MTSAT-2]]（向日葵7號）。

== 發射 ==
裝載向日葵8號的[[H-IIA|H-IIA202]]火箭在日本時間2014年10月7日14時16分00秒於[[種子島宇宙中心|種子島宇宙中心]]發射升空<ref name="jaxaH2A"/>，其後14時43分57秒衛星與火箭成功分離<ref name="jma20141007"/>，同月16日19時00分確認其進入[[地球靜止軌道|靜止軌道]]<ref name="jma20141017"/>。

== 目的 ==
其作用為提供[[日本|日本]]、[[東亞|東亞]]及[[太平洋|西太平洋]]之週邊的[[天氣預報|天氣預報]]、[[颱風|颱風]]、[[暴雨|暴雨]]、[[氣候變化|氣候變化]]的[[觀察|觀察]]、[[預測|預測]]等相關之報告，同時負責確保[[船隻|船隻]]與[[航空|航空]]的安全以及觀察地球之環境<ref name="jma"/>。

== 運行計劃 ==
向日葵8號設計的目的是作為{{link-ja|多功能運輸衛星|MTSAT|MTSAT-2}}的接替衛星，在送上軌道後經過一系列的測試後於2015年7月7日正式開始運作<ref name="jma20150527"/><ref name="sorae20150707"/>。而採用相同設計的[[向日葵9號|向日葵9號]]於2016年11月2日升空，8號與9號將會替換現有的MTSAT-1R（向日葵6號）與MTSAT-2（向日葵7號）<ref name="jma20131224"/>。

=== 成像儀之波段 ===
向日葵8號內置的高像素紅外線成像儀（AHI: Advanced Himawari Imager）<ref name="AHI"/>的[[可见光|可視波長]]與[[紅外線|近紅外線]]達3波段，同時內置的[[传感器|传感器]]能感應紅外線10至16波段，與MTSAT-1R與MTSAT-2的1波段可視波長、4波段紅外線波長相比，大幅增加了可感應的波段數。

=== 圖像傳送 ===
現在MTSAT-1R的高速 (HRIT)與低速(LRIT)情報傳送之接收站於2015年11月已停止服務，其後於2015年1月起，衛星圖像均由通信衛星HimawariCast負責傳送，初期只有{{link-ja|JCSAT|JCSAT|JCSAT-2A}}投入服務，2015年第四季再加入JCSAT-2B<ref name="himawari_cast"/>。

=== 自動太陽回避功能 ===
[[春分|春分]]與[[秋分|秋分]]期間的午夜，衛星都會自動與太陽和地球形成一直線，為預防太陽直接照射衛星的成像儀造成加速損壞，便會使用此功能避免陽光直射，但也因此造成觀測畫面的部分缺損<ref name=taiyo/><ref name="taiyo002"/><ref name="taiyo4">{{Cite web
|url=http://www.data.jma.go.jp/mscweb/ja/operation8/equinox/plan.html
|title=春分期・秋分期の観測について（太陽自動回避及び太陽迷光）
|accessdate=2015-09-03
|publisher=[[気象衛星センター|気象衛星センター]]}}</ref>
<ref name="taiyo002">{{Cite web
|url=http://www.data.jma.go.jp/mscweb/ja/operation8/foundation/image/image_info.html#002
|title=春分期・秋分期の太陽自動回避による画像欠損
|accessdate=2015-09-03
|publisher=[[気象衛星センター|気象衛星センター]]}}</ref>。此外此功能也可避免成像儀射入陽光後出現帶狀的太陽光線而使圖像失真<ref name=taiyo3/><ref name="taiyo003"/>。
