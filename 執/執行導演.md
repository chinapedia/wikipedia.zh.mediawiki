'''執行導演'''（{{lang-en|Executive Director}}）是指影視作品在拍攝階段有兩名或以上[[聯合導演|聯合導演]]時，其中負責現場拍攝工作的那位[[導演|導演]]，因此執行導演又被稱為「現場導演」，並且在拍攝現場掌握著每個镜頭的通過權。<ref>{{cite web | url=http://edu.1905.com/archives/view/201/ | title=1905電影網・電影學院 - 基礎知識 - 執行導演 | publisher=國家廣播電影電視總局電影衛星頻道節目製作中心 (CCTV6)}}</ref><ref>{{cite web | url=http://baike.baidu.com/view/375101.htm | title=相關執行導演釋義 | publisher=百度百科 (參考資料：夏衍《電影藝術詞典》第136頁，北京中國電影出版社2005年版)}}</ref>

執行導演有時會被誤看成劇組中的副導演，而不是導演，但其實在劇組人員中執行導演被視為導演<ref>{{cite web | url=http://www.createhk.gov.hk/fffi/download/summary_app_guide_c.pdf | title=第三屆首部劇情電影計劃電影創作及製作計劃比賽 - 計劃摘要及報名須知 | publisher=創意香港}}</ref>。在中國大陸、台灣、香港、澳門、新加坡等地以中文放映的相關影視作品例子包括：[[錢國偉_(香港)|錢國偉]]執行導演《[[美人魚_(2016年電影)|美人魚]]》<ref>{{cite web | url=http://xw.qq.com/ent/20160118001037 | title=《美人魚》執行導演錢國偉：擁香港半壁娛樂圈 | publisher=騰訊娛樂 | accessdate=2016-01-18}}</ref>、[[陳嘉上|陳嘉上]]執行導演《[[三人世界|三人世界]]》<ref>{{cite web | url=http://news.xinhuanet.com/ent/2014-11/06/c_127185383.htm | title=陳嘉上做客《全民電影》大師班 盛讚“周迅最棒” | publisher=新華網 | accessdate=2014-11-06}}</ref>，以及[[連奕琦|連奕琦]]執行導演《[[左耳_(電影)|左耳]]》<ref>{{cite web | url= http://star.ettoday.net/news/573434 | title=《左耳》入圍蘇有朋反被黑？連奕琦：做人不能吃乾抹淨 | publisher=ETtoday東森星光雲 | accessdate=2015-10-02}}</ref>，等等。

== 注意事項 ==
* 執行導演（Executive Director）不同於[[副導演|副導演]]（Assistant Director，AD），通常为導演的分身，负责指挥一个分剧组的摄制工作。<ref>{{cite web | url=https://lyratheinsider.wordpress.com/credits-eng/ |title=電影工作人員職稱 / Credits（中英文對照）}}</ref>
