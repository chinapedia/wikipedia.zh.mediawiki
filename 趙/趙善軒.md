'''趙善軒'''博士（{{lang-en|Dr. '''Gavin Chiu Sin-hin'''}}）歷任新亞研究所博士後研究員，[[明愛專上學院|明愛專上學院]]人文及語言學院助理教授、[[樹仁大學|樹仁大學]]經濟系客席講師、[[臺灣|臺灣]][[中央研究院|中央研究院]]近代史研究所訪問學人（2017年1月10日 — 2017年3月10日）<ref>、[http://www.mh.sinica.edu.tw/UserDetail.aspx?userID=478&mid=30&tmid=9 中央研究院近代史研究所 -來訪學侶趙善軒]</ref>、美國[[布萊恩特大學|布萊恩特大學]]及[[北京理工大學珠海學院|北京理工大學珠海學院]]合聘副教授<ref>{{cite web |url=http://bryantc.zhbit.com/Sz/20150603/63.html |title=布萊恩特學院中外合作辦學 -師資隊伍. 2015-06-03（存檔副本）|accessdate=2017年2月9日 |deadurl=yes |archiveurl=https://web.archive.org/web/20170211075321/http://bryantc.zhbit.com/Sz/20150603/63.html |archivedate=2017年2月11日 |df= }}</ref>、[[饒宗頤文化館|饒宗頤文化館]]研究員，教授全球化、近代中國史。他亦曾任[[香港中國婦女會馮堯敬紀念中學|香港中國婦女會馮堯敬紀念中學]]校董、網上媒體[[謎米香港|謎米香港]]節目主持和行政總裁，曾為《周末趙集》主持及《蕭遙遊》客席主持，人稱趙博。

== 簡歷 ==
趙善軒於[[香港中國婦女會馮堯敬紀念中學|香港中國婦女會馮堯敬紀念中學]]畢業後，升讀[[香港樹仁學院|香港樹仁學院]]（現[[香港樹仁大學|香港樹仁大學]]）歷史系，先後獲榮譽文憑及榮譽文學士學位。之後在[[新亞研究所|新亞研究所]]及[[暨南大學|暨南大學]]兼讀學位，以《上海機器織布局(1878-1893)發展史研究》取得新亞研究所文學碩士（2008）<ref>[https://newasia.org.hk/%E7%A2%A9%E5%A3%AB%E7%8F%AD%E7%AC%AC%E5%9B%9B%E5%8D%81%E4%B8%80%E5%B1%86%E8%87%B3%E4%BB%8A/ 新亞研究所 -歷屆畢業論文碩士班（第四十一屆至今）]</ref>，及以〈歷史文學與經濟思想：《史記：貨殖列傳》研究〉獲得暨南大學中國古代文學博士。

英國皇家歷史學會會員<ref>{{cite web |url=http://www.royalhistoricalsociety.org/rhsmembers.pdf |title=存档副本 |accessdate=2013-02-16 |deadurl=yes |archiveurl=https://web.archive.org/web/20130614115358/http://www.royalhistoricalsociety.org/rhsmembers.pdf |archivedate=2013-06-14 }}</ref>及歐洲漢學學會會員。研究領域為中國文化史、經濟思想史、中國經濟史。出版著作十九本，論文三十篇。

善軒曾創辦補習社「趙博教室」、亦曾任[[現代教育|現代教育]]中文科補習導師及[[OurRadio|OurRadio]]「通識天地」節目主持。

趙善軒曾為[[明愛專上學院|明愛專上學院]]人文及語言學院助理教授，並同時擔任[[香港中國婦女會馮堯敬紀念中學|香港中國婦女會馮堯敬紀念中學]]校董及[[謎米香港|謎米香港]]節目主持。2015年7月，他在《蕭遙遊》節目中透露，因參與佔領運動，他任職的學校收到數十封投訴信，所以他決定辭去網台節目、所有公職及學術工作。<ref>{{cite news |title=受政治打壓　網台主持趙善軒全退 |date=2015-07-31 |work=謎米新聞 |url=http://news.memehk.com/posts/9282 }}</ref>

== 個人生活 ==
2017年7月1日「7.1大遊行後」，[[學聯|學聯]]午夜在遮打道預演[[和平佔中|和平佔中]]，警方圍捕511人參與者，趙善軒是其中之一。<ref>{{cite news |title=7.2被捕者難忘警鼓掌送行 |work=蘋果日報 |date=2014-07-23 |url=https://hk.news.appledaily.com/local/daily/article/20140723/18808929 }}</ref>

趙善軒在2014年11月24日的〈蕭遙遊〉節目中透露，他因與[[謎米香港|謎米香港]]的女主持Kate交往而疑遭「[[香港製造網絡電視|低俗頻道]]」的金主「鱷魚」刑事恐嚇。<ref>{{cite web |title=〈蕭遙遊〉低俗金主疑刑事恐嚇趙博﹗ |work=youtube |date=2014-11-24 |url=https://www.youtube.com/watch?v=sPF595jjO5s }}</ref>

2015年趙善軒發現女友意外懷孕，之後在同年仲夏舉行婚禮<ref>{{cite web |title=太太懷孕記(一) 懷孕初期 |author=趙善軒 |work=facebook |url=https://zh-hk.facebook.com/ZhaoShanXuanBoShi/photos/a.559934464017844.132058.476400069037951/1015298275148125/}}</ref>。太太任職鋼琴導師，二人育有一女。

於2017年11月左右，趙善軒發現淋巴結，之後證實患上癌症，屬最晚期的第四期，經過手術、電療、中醫、斷糖等治療後，於2018年9月康復。<ref>{{cite web |title=我的患癌、治療、康復之經歷（一） |work=youtube |date=2018-09-19 |author=趙善軒 |url=https://www.youtube.com/watch?time_continue=1&v=nRgOfJHyXCg }}</ref>
