'''你'''（[[現代標準漢語|現代標準漢語]]：[[汉语拼音|汉语拼音]]：nǐ、[[IPA|IPA]]：{{IPA|[ni˨˩˦]}}；[[粵語|粵語]]：nei<sup>5</sup><ref>{{cite web|author=粵語審音配詞字庫|title=你|url=http://arts.cuhk.edu.hk/Lexis/lexi-can/search.php?q=%A7A|accessdate=2010年7月2日|authorlink=粵語審音配詞字庫|language=zh-hk}}{{dead link|date=2018年10月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>），是一個[[代名詞|代名詞]]，中文的[[第二人稱|第二人稱]]。

== 詞義 ==
{{wiktionary}}
「你」是汉语的第二人称代词，用以称对方，多是指单个人，有时也指示若干人，如：你方。<ref>{{Cite web |url=http://www.zdic.net/zd/zi/ZdicE4ZdicBDZdicA0.htm |title=字典中 你 字的解释 |publisher=汉典}}</ref>  普通話及部份方言中，「您」為“你”的尊稱。此外，如果对方是[[女性|女性]]，个别时候还会写成“-{妳}-”（在中國大陸不规范）。

== 文言 ==
“你”不存在于古代[[汉语|汉语]]中，[[康熙字典|康熙字典]]沒有收录该字。在文言中“你”用其他字眼代替：
*汝（古作女）、爾、若、乃、而等。這些字事實上皆與「你」為同源字，例如-{爾}-加上人字旁即為「-{儞}-」，发展到现代簡写成「你」（例如傳統漢字「-{爾}-」的對應[[簡化字|簡化字]]就是「-{尔}-」）。汝、你在部份漢語方言仍然同音。
*閣下、子、公、台端、大駕等。多用於書面文。香港書面文或告示多用「閣下」，臺灣的公文書則用「台端」

== 各地同義用字 ==
[[吴语|吴语]]：「侬」。

[[闽南语|闽南语]]：用「汝」或「你」（[[臺灣閩南語推薦用字|臺灣閩南語推薦用字]]），依各地腔調不同有lú、lí或lír（皆用[[台羅拼音|台羅拼音]]）的讀法。<ref>{{cite web|url=http://www.edu.tw/FileUpload/3677-15601/Documents/700iongji_990915.pdf|title=臺灣閩南語推薦用字700字表|publisher=教育部|date=2010-09}}</ref>

膠遼官話：'''您'''、'''你'''共存，分用於不同語境，不能混用。您，或恁，并不表尊稱，而是在不同語境中有“''你們/你（們）的/你（們）''”之義。

== 其他語言中的「你」 ==
=== 英文 ===
{{Wiktionary|you|3=en}}

在[[標準英語|標準英語]]中，用“you”表示「你」，表达的可以是單數意义，也可以是複數意义。通常這個字之後都是一些[[複數|複數]]詞，例如''you are''，但這項也不是一定。早期的英文則分為[[複數|複數]]是''you''，[[單數_(語言學)|單數]]則是''thou''。

由於「you」可以是單數也可以是複數，故此各英文方言試圖修復這個問題以避免混淆。這通常是新增一個複數的字詞，例如y'all或者you-all、you guys、you lot（[[英國|英國]]）、youse（[[蘇格蘭|蘇格蘭]]）、yous（[[英國|英國]][[利物浦|利物浦]]）、youse guys、you-uns/yinz等等。在這些情況下，you不再是常用字詞。除了使用y'all的方言外，you guys是[[美國|美國]]最常用的複數方式，甚至連[[白宮|白宮]]也在使用<ref>{{cite news

==== 歷史 ====
在[[中古英語|中古英語]]，''You''是詞語“你”的[[賓格|賓格]]，[[主格|主格]]則是'''''Ye'''''。Ye是由[[古英語|古英語]]中的ge或ȝe演變而成的。

==== 現代寫法 ====
為了節省時間，有时部份人會將You寫成发音相同的字母[[U|U]]，例如''How R U?''（你最近如何？），這樣的簡寫已開始變得廣泛使用；但在正式的書信上，一般均不會使用這個簡寫，因為U不是正式的寫法。

=== 荷蘭語 ===
[[荷蘭語|荷蘭語]]的「你」是{{lang|nl|''u''}}，像英文，這既是[[複數|複數]]又是[[單數_(語言學)|單數]]。同時，jij（單數）及jullie（複數）都隨之出現了，但這兩個都是非正式的，''u''才比較正式、有禮的。

=== 法語 ===
[[法語|法語]]中，作主语时的''{{lang|fa|vous}}''是表示眾數或者单数的尊称形式的“你”（或者“您”），而非尊称的單數則是用{{lang|fa|''tu''}}。作前置宾语时用''{{lang|fa|vous}}''或''{{lang|fa|te}}''（对应{{lang|fa|''tu''}}），而强调、单独指称或同位语时则使用''{{lang|fa|vous}}''或''{{lang|fa|toi}}''（对应{{lang|fa|''tu''}}）。

=== 俄語 ===
[[俄语|俄语]]的“你”的单数第一格形式为{{lang|ru|ты}}，复数为{{lang|ru|вы}}。值得注意的是，表示尊敬的“您”是用“你”的复数形式加首字母大写，也就是{{lang|ru|Вы}}。由于俄语句子第一个单词也是需要首字母大写，因此必须通过上下文才能判断Вы是单数的“您”还是复数的“你们”。
*Вы студент?（您是大学生吗？）
*Вы студент'''ы'''?（你们是大学生吗？）

=== 波斯語 ===
[[波斯語|波斯語]]使用相同的方式，{{lang|fa|ʃomɒ}}是正式的複數而{{lang|fa|to}}是非正式的單數。

=== 瑞典語 ===
[[瑞典語|瑞典語]]的{{lang|sv|ni}}應該是複數，但通常作為單數使用，而正式的單數應該是{{lang|sv|du}}。
