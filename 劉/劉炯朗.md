'''劉炯朗'''（{{bd|1934年|10月25日|catIdx=L劉}}），生於[[广东省_(中华民国)|廣東]][[廣州市_(中華民國)|廣州]]，台湾資訊學者，曾於[[美國麻省理工學院|美國麻省理工學院]]、[[伊利諾大學|伊利諾大學]]任教，並曾擔任過[[伊利諾大學香檳分校|伊利諾大學香檳校區]]助理副校長職務，2000年時當選[[中央研究院院士|中央研究院院士]]。1998—2002年間，放棄美國籍而返臺擔任[[國立清華大學|國立清華大學]]校長。現為集邦科技股份有限公司董事長、[[香港城市大學|香港城市大學]]訪問學者、[[國立成功大學|國立成功大學]]電機工程學系李國鼎科技講座教授與國立清華大學蒙民偉榮譽講座教授。

== 求學時期 ==
劉炯朗民國廿三年十月在廣東出生，年僅四歲就碰上[[對日抗戰|對日抗戰]]，父親劉沛然是國軍飛行員, 母親則帶著三個兒子從廣州逃到澳門，一面在小學任教，一面悉心教養三子。劉炯朗在[[聖公會澳門蔡高中學|聖公會澳門蔡高中學]]完成中、小學教育。十一歲抗戰勝利後，父親隨空軍部隊到台灣，他們則繼續留在澳門唸書。因為父親在台灣擔任軍職，遂前來台灣報考大學，並考入[[成功大學|成功大學]]（當時的台南工學院）電機系就讀。

民國四十一年，由澳門來台進入成功大學前，曾隨父親住在高雄岡山眷村樂群村<ref>[http://tw.myblog.yahoo.com/rocafes-nca/article?mid=767&sc=1 岡山軍眷村發展史]{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。當時台灣的大學不多，僅有台灣大學、省立師範學院、省立台南工學院、以及省立台中農學院，沒有聯招，劉炯朗當時只報考台灣大學與省立台南工學院。在考上台南工學院電機系後，就決定在台南唸書，不再考台灣大學。1956年大學畢業，報考[[国立清華大學|国立清華大學]]原子科學研究所第一屆入學考試，並以正取第二名的成績獲得錄取。但因同時取得美國麻省理工學院提供的獎學金，所以選擇從軍擔任陸軍少尉預官。1958年退伍後，便隻身負笈留美，於1960年及1962年分別取得[[美國麻省理工學院|美國麻省理工學院]]電腦[[碩士|碩士]]及[[博士|博士]]學位。

== 旅美時期 ==

劉炯朗自麻省理工學院取得博士學位後，分別在麻省理工學院與伊利諾大學香檳分校擔任教職三十多年:
*美國[[麻省理工學院|麻省理工學院]]電機工程系助理教授及副教授（1962—1972）
*美國[[伊利諾大學香檳分校|伊利諾大學香檳校區]]資訊工程系教授（1972—1998）
*美國[[伊利諾大學香檳分校|伊利諾大學香檳校區]]助理副校長（1995—1998）
*美國[[伊利諾大學香檳分校|伊利諾大學香檳校區]]榮譽退休教授（1998-）
此外，亦曾擔任哈佛大學、墨西哥城城市大學、柏林工學院及巴黎大學等知名學府與機構訪問學者、從事短期研究。

== 國立清華大學校長時期 ==
劉炯朗旅居美國四十年，在此之前也僅在臺六年，與清大並無淵源。然而，清大校長為劉炯朗最為台灣大眾所知的職位。曾任中研院資訊所所長的[[李德財|李德財]]曾經表示，劉炯朗放棄美籍返臺接掌清大校長是促使他決定返台服務的原因之一<ref>[http://www.libertytimes.com.tw/2004/new/jul/15/today-life6.htm 《國內新院士人物素描》品質第一李德財自我要求高] {{webarchive|url=https://web.archive.org/web/20101023130037/http://www.libertytimes.com.tw/2004/new/jul/15/today-life6.htm |date=2010-10-23 }}</ref><ref>[http://president.nchu.edu.tw/dtlee/?p=1301 2014年6月6日 博士班畢業典禮校長致詞稿]</ref>(對於台灣的國立大學校長的國籍限制，已於2005年放寬為外籍或雙重國籍者，亦可擔任<ref>[http://www.news.high.edu.tw/monthly178/content02.htm 從大學法修法談國內大學教育的改革] {{webarchive|url=https://web.archive.org/web/20080330014925/http://www.news.high.edu.tw/monthly178/content02.htm |date=2008-03-30 }}</ref>)。

劉炯朗校長四年任內〔1998/2/4—2002/2/3〕，由[[台積電|台積電]]募款新台幣一億五千萬元<ref>[http://www.tsmc.com/tsmcdotcom/PRListingNewsArchivesAction.do?action=detail&newsid=713&language=C 持續投入人才培育台積公司捐款新台幣一億五千萬元協助清華大學成立全國首座科技管理學院]</ref>，支持清大籌建台積館並成立科技管理學院。亦從旺宏、臺達電、威盛、凌陽、合勤、台灣應用等諸多公司為清大募得約五億捐款<ref>[http://www.sinorama.com.tw/show_issue.php?id=2002119111030c.txt&table=0&h1=%E6%95%99%E8%82%B2%E6%96%87%E5%8C%96&h2=%E5%A4%A7%E5%AD%B8 校長加油！ ──籌款興學時代來臨]</ref>。同時促成與[[國立交通大學|國立交通大學]]、[[國立中央大學|國立中央大學]]、[[國立陽明大學|國立陽明大學]]籌組[[臺灣聯合大學系統|臺灣聯合大學系統]]合作方案。而兩岸清華大學與[[北京大學|北京大學]]、[[復旦大學|復旦大學]]、[[蘇州大學|蘇州大學]]、[[蘭州大學|蘭州大學]]的交流成果，亦頗可觀。

劉炯朗接任清華大學校長剛滿一個月，清大校園內發生刑事案[[洪曉慧事件|洪曉慧事件]]，校園氣氛低迷，校譽深受打擊。但是在[[遠見雜誌|遠見雜誌]]所做的學術聲望調查，清華大學在他任內於1998、1999連續兩年學術聲望超越台大，排名第一。

== 專業貢獻 ==
劉炯朗為國際知名之計算機科學家，對於[[即時系統|即時系統]]、[[電腦輔助設計|電腦輔助設計]]、[[VLSI佈局|VLSI佈局]]、[[組合最佳化|組合最佳化]]、[[離散數學|離散數學]]等領域均有傑出貢獻，其於1973年發表在''Journal of the [[美國計算機協會|ACM]]''之論文「Scheduling Algorithms for Multiprogramming in a Hard-Real-Time Environment」，至今已被引用超過8162次(至2013年為止)，影響極大，並且在2011年獲得有「EDA界的諾貝爾獎」之稱的[[卡夫曼獎|卡夫曼獎]]。另外，其所撰寫的離散數學教科書「Elements of Discrete Mathematics」被世界許多大學資訊工程系（例如：美國伊利諾大學芝加哥分校、加拿大皇后大學、台灣大學）採用為離散數學主要教材或是重要參考教材。劉炯朗曾在[[美國麻省理工學院|美國麻省理工學院]]、美國[[伊利諾大學|伊利諾大學]]任教，返回[[台灣|台灣]]後亦擔任[[國立清華大學|國立清華大學]]校長及榮譽講座教授，培育英才無數、桃李滿天下。[[圖靈獎|圖靈獎]]唯一的華人得主[[姚期智|姚期智]]是其門生。

== 業餘貢獻 ==

劉炯朗自清大校長退休後，持續因為媒體的訪問、演講邀約、著作出版、廣播節目、文章在網路上的轉載、擔任大專院校和科技公司榮譽職位，繼續在台灣與華人社會保持影響力<ref>[http://www.libertytimes.com.tw/2011/new/jul/15/today-life9.htm 中研院士劉炯朗︰破除排名迷思 教育才會變好] {{webarchive|url=https://web.archive.org/web/20110902063753/http://www.libertytimes.com.tw/2011/new/jul/15/today-life9.htm |date=2011-09-02 }}</ref><ref>[http://blog.chinatimes.com/1234lyl/archive/2008/12/31/364442.html 訪劉炯朗之一：處亂世 誠信為重 (影音)]</ref><ref>[http://mag.udn.com/mag/campus/storypage.jsp?f_ART_ID=184759 東海博雅書院 劉炯朗任院長] {{webarchive|url=https://web.archive.org/web/20090328175150/http://mag.udn.com/mag/campus/storypage.jsp?f_ART_ID=184759 |date=2009-03-28 }}</ref><ref>[http://www.businesstoday.com.tw/test/content.aspx?a=W20061200164&p=1 不要學生第一名 只要謹守工作倫理]{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。他目前也是國際研究機構TrendForce(集邦科技)的董事長，推動商業智庫與智慧財在台灣與國際市場的能見度。<ref>[http://news.cnyes.com/Content/20091026/KAPDKJ0YYABPY.shtml 集邦科技智慧財 獲聯電集團投資10%股份]</ref>

== 家庭 ==
劉炯朗父親劉沛然以美國歸僑身分加入廣東空軍, 在1932年第一次淞滬戰爭時被編入混合機隊（由驅逐機與轟炸機配合）北上參戰, 對日抗戰期間曾派駐四川、雲南昆明與新疆伊寧等地, 經歷多次空戰並擔任過飛行教官、偵查班副主任等職, 空軍中校退伍。劉炯朗三兄弟都是成大校友，一門三傑，哥哥劉炯炎讀機械、弟弟劉炯權念化工，赴美進修後都擔任大學教職，劉炯權在生物晶片領域的表現相當傑出，現為[[凱斯西儲大學|凱斯西儲大學]](Case Western Reserve University，CWRU)化工系教授兼電子設計中心(Electronic Design Center)主任。劉炯朗的妻子[[張韻詩|張韻詩]]為知名資訊學者，現任[[中央研究院|中央研究院]]資訊科學研究所特聘講座，研究專長為即時系統、作業系統、計算機網路，以及效能分析評估。

== 趣事 ==
* 劉炯朗弟弟劉炯權為知名材料科學學者, 每年固定自美返台進行學術交流, 協助台灣相關研究。他在一次訪問曾笑說, 他在台灣常被人介紹是劉校長的弟弟。<br>
* 劉炯朗母語為粵語, 澳門中小學也是粵語教學, 所以他曾自我打趣說: 他說著有中文腔的英語, 有廣東腔的中文。<br>
