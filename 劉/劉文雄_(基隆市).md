'''劉文雄'''（{{bd|1954年|9月18日|2017年|7月31日|catIdx=L刘}}）[[中華民國|中華民國]]政治人物，[[河南|河南]][[開封|開封]][[回族|回族]]，[[臺灣|臺灣]]戰後中國大陆移民第2代，生於[[基隆|基隆]]，[[臺灣伊斯蘭教|臺灣伊斯蘭教]]信徒（[[穆斯林|穆斯林]]），曾任[[中国國民黨|中国國民黨]]籍[[臺灣省議員|臺灣省議員]]、[[親民黨|親民黨]][[立法委員|立法委員]]、[[親民黨|親民黨]]副秘書長等。2017年獲[[蔡英文政府|蔡英文政府]]提名為[[監察委員|監察委員]]，由於送交[[國會|國會]]審理中而尚未通過此人事案。7月21日凌晨，劉文雄因酒後嘔吐[[窒息|窒息]]而緊急住院，10日後不治。

任職省議員期間，劉文雄與時任[[臺灣省長|臺灣省長]][[宋楚瑜|宋楚瑜]]非常友好，日後宋楚瑜創建親民黨，劉文雄從國民黨轉投親民黨，並成為黨內重要領袖之一，此後一直留在親民黨，未如許多親民黨政治人物重回[[國民黨|國民黨]]。

== 家世與宗教 ==
劉文雄祖先是[[中國|中國]]邊疆的[[回族|回族]]（血統上可能是[[漢人|漢人]]或[[東干人|東干人]]）<ref>邱慕天.[https://anntw.com/articles/20170803-6rix 土葬、速葬、簡葬 穆斯林喪禮送劉文雄].台灣醒報.2017-08-03 </ref>，大約[[清末|清末]]遷徙至[[河南|河南]][[開封|開封]]，而劉自幼信仰[[伊斯蘭教|伊斯蘭教]]，是[[臺灣政治|臺灣政界]]少有的[[穆斯林|穆斯林]]。雖因從政之故，在[[戒律|戒律]]上頗多扞格，但從未[[叛教_(伊斯蘭教)|背棄伊斯蘭教]]。任職[[國會|國會]]時，其[[同事|同事]][[穆閩珠|穆閩珠]]、[[林忠正|林忠正]]、[[丁守中|丁守中]]等人的家族都是[[伊斯蘭教|伊斯蘭教]]背景，不過還保持穆斯林身份的只剩下劉文雄一個<ref>程嘉文.[http://www.epochtimes.com/b5/5/2/9/n808155.htm 劉文雄是唯一回教立委].大紀元.2005-02-09 </ref>。過去他擔任立委時，都會在辦公室門口掛上《[[可蘭經|可蘭經]]》，但外界皆知他喝酒，已違反回教[[教義|教義]]，他也坦承，作為政治人物，他只有遵守「不吃[[豬肉|豬肉]]」，其他很多[[戒律|戒律]]，如「不許[[飲酒|飲酒]]」、「[[拜功|每天祈禱五次]]」，他都無法遵守了，但他依然心中虔信伊斯蘭教，他說他對死亡無所畏懼，因為死亡是回到[[真主|真主]]的身邊<ref>[https://tw.news.yahoo.com/%E5%8A%89%E6%96%87%E9%9B%84%E6%98%8F%E8%BF%B7-%E9%80%B1-%E6%9D%8E%E9%B4%BB%E9%88%9E-%E8%AB%8B%E5%A4%A7%E5%AE%B6%E9%9B%86%E6%B0%A3-080152353.html 劉文雄昏迷一週 李鴻鈞：請大家集氣].新頭殼.2017-07-29</ref><ref>翁嫆琄.[http://www.ettoday.net/news/20170731/978782.htm 身為回教徒愛喝酒　劉文雄曾說：死亡是回到真主身邊].ETtoday新聞雲.2017-07-31</ref>。

== 經歷 ==
劉文雄是[[政治大學|政治大學]]地政系畢業，原本從商，27歲時於青商會結識[[基隆市長|基隆市長]][[張春熙|張春熙]]，因機智、口才便給，張春熙拔擢之為基隆市政府機要秘書，因而步入政壇。而後擔任[[臺灣省議會|臺灣省議會]]第9、10屆省議員，連任第4、5、6屆[[立法委員|立法委員]]。<ref name="蘋果日報">{{cite news |url = http://www.appledaily.com.tw/realtimenews/article/politics/20170731/1172748/applesearch/%E3%80%90%E4%B8%80%E8%B7%AF%E5%A5%BD%E8%B5%B0%E3%80%91%E5%8A%89%E6%96%87%E9%9B%84%E6%98%8F%E8%BF%B710%E5%A4%A9%E4%BB%8A%E6%99%9A%E8%BE%AD%E4%B8%96%E3%80%80%E4%BA%AB%E5%B9%B464%E6%AD%B2 |title = 【一路好走】劉文雄昏迷10天今晚辭世　享年64歲 |last=戴之聖|publisher = 蘋果日報 |date = 2017-07-31 |accessdate = 2017-08-19}}</ref>

[[1997年中華民國縣市長選舉|1997年中華民國縣市長選舉]]中，[[中國國民黨|國民黨]]提名劉文雄參選基隆市長，時為國民黨籍的基隆市議員[[許財利|許財利]]執意違反黨紀退黨參選，導致[[藍軍|藍軍]]票源分裂，由[[民進黨|民進黨]]提名的[[李進勇|李進勇]]當選基隆市長。鑒於1997年[[許財利|許財利]]脫黨事件，2001年劉文雄禮讓許財利參選，而許財利也公開承諾倘若當選只當一屆，次屆市長選舉將全力輔選劉文雄。

[[2005年中華民國地方公職人員選舉|2005年三合一選舉]]，時任基隆市長許財利違背當初僅任一屆市長的承諾，再度參選。劉文雄憤而參選，可惜不敵[[中國國民黨主席|國民黨主席]][[馬英九|馬英九]]輔選而引發的[[棄保效應|棄保效應]]，最後以敗選收場。但許財利任期未過半即因[[糖尿病|糖尿病]]、[[蜂窩性組織炎|蜂窩性組織炎]]、[[敗血症|敗血症]]病逝。而當時的基隆市議會議長[[張通榮|張通榮]]即宣布參選市長補選。2007年3月11日（許財利喪禮次日）劉文雄也在基隆[[碧砂漁港|碧砂漁港]]的[[蔣經國|蔣經國]]銅像前宣布參選，但劉文雄因不欲破壞[[泛藍|泛藍]]團結，宣佈以[[無黨籍|無黨籍]]身分參與補選，但還是敗給了有國民黨奧援的張通榮。最終劉文雄三度問鼎市長寶座都落敗。<ref name="蘋果日報"/>

2008年，國民黨、親民黨原本有共識，要讓劉文雄轉任監委，最後卻因當時國民黨主席馬英九反對而破局，劉文雄因而轉行當名嘴，時常上政論節目猛烈批評馬政府。<ref name="蘋果日報"/>

2016年劉文雄代表親民黨參選[[2016年中華民國立法委員選舉|立委選舉]]（[[基隆市選舉區|基隆市選舉區]]），與[[民國黨|民國黨]]的[[楊石城|楊石城]]、[[國民黨|國民黨]]的[[郝龍斌|郝龍斌]]、[[民進黨|民進黨]]的[[蔡適應|蔡適應]]一同角逐，結果不敵蔡適應，亦輸給郝龍斌，只拿下第3名。

2017年3月，總統[[蔡英文|蔡英文]]提名劉文雄為[[監察委員|監察委員]]，但人事案在[[立法院|立法院]]引發爭議。據透露，是部分綠委對於劉文雄身為親民黨的泛藍血統「很有意見」。劉文雄本不願回應，後來表示他自己「從頭到尾都是被動」、「能為國家做事情的話就做」。<ref>{{cite news |url = http://www.appledaily.com.tw/appledaily/article/headline/20170530/37666348/ |title = 不滿劉文雄 綠委擋監委名單 |last=符芳碩、陳郁仁|publisher = 蘋果日報 |date = 2017-05-30 |accessdate = 2017-08-19}}</ref>

2017年7月21日，劉文雄凌晨酒醉嘔吐窒息，引發心肌梗塞送醫。<ref>{{cite news |url = https://udn.com/news/story/11333/2615457 |title = 劉文雄走完一生 沒選上市長是他最大的缺憾 |last=吳淑君|publisher = 聯合報 |date = 2017-07-31 |accessdate = 2017-08-19}}</ref>家人發現後立刻通報[[緊急救助電話|119專線]]，救護員施以[[心肺復甦術|心肺復甦術]]、[[電擊|電擊]]才恢復[[心跳|心跳]]、[[呼吸|呼吸]]，但依然無[[意識|意識]]，緊急送往[[長庚醫院|基隆長庚醫院]]搶救<ref>{{cite web |publisher=自由電子報 |title=親民黨副秘書長劉文雄一度命危 緊急手術搶救 |last=林嘉東|url=http://news.ltn.com.tw/news/politics/breakingnews/2138625 |date =2017-07-21 |accessdate=2017-07-21}}</ref>，期間曾以[[葉克膜|葉克膜]]維繫生命，但最終依然在10日因[[心肌梗塞|心肌梗塞]][[逝世|逝世]]，享壽62歲。親民黨主席[[宋楚瑜|宋楚瑜]]、總統[[蔡英文|蔡英文]]等多位政要皆前往致哀。8月3日於[[臺北清真寺|臺北清真寺]]舉行喪禮，並[[土葬|土葬]]於回教公墓<ref>{{Cite news|url=https://udn.com/news/story/6656/2615443|title=劉文雄病逝長庚醫院享壽64歲 將依伊斯蘭教葬禮|author=曾健祐|date=2017-07-31 |newspaper=聯合新聞網|accessdate=2017-07-31}}</ref>。
