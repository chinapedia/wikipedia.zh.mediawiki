'''劉青雲'''（{{bd|1894年|9月14日|1982年|3月9日}}），小名'''主燈'''，為[[台灣|台灣]][[台灣日治時期|日治]]及[[台灣戰後時期|戰後]]時期的[[作家|作家]]。[[台南|台南]]望族出身，自中學時代起留學[[日本|日本]]，[[慶應義塾大學|慶應義塾大學]]畢業，於留學期間改信[[基督教|基督教]]。著有《[[羅華改造統一書翰文|羅華改造統一書翰文]]》、《[[大學精詳|大學精詳]]》及《[[天國奧秘全照射|天國奧秘全照射]]》等書。<ref name="史話313">{{cite web |url=http://www.laijohn.com/book4/313.htm |title=史話313  劉主燈學業成績單 |accessdate= |last= |first= |author=賴永祥 |coauthors= |date=1995-06-18 |work=教會史話 |publisher= }}</ref><ref name="史話314">{{cite web |url=http://www.laijohn.com/book4/314.htm |title=史話314  羅華改造統一書翰文 |accessdate= |last= |first= |author=賴永祥 |coauthors= |date=1995-06-25 |work=教會史話 |publisher= }}</ref><ref name="史話315">{{cite web |url=http://www.laijohn.com/book4/315.htm |title=史話315《天國奧秘全照射》跋 |accessdate= |last= |first= |author=賴永祥 |coauthors= |date=1995-07-02 |work=教會史話 |publisher= }}</ref><ref name="壹葉通訊12">{{cite web |url=http://www.laijohn.com/works/aleaf/12.htm |title=劉青雲先生（1894-1982）略歷 |accessdate= |last= |first= |author=賴永祥 |coauthors= |date=1983-03 |work=壹葉通訊 |publisher= }}</ref>

==生平==
劉青雲，[[台南|台南]]望族出身，為[[劉瑞山|劉瑞山]]之長男，生母為[[劉謝就|劉謝就]]。生長於篤信基督教之家庭中，幼年時期就讀於太平境基督長老教會附屬小學。1907年進入台南[[長老教會中學|長老教會中學]]。於長榮中學期間，劉青雲成績優異，於1909年三年級下半年「總結考」時，以97分（及第分數70）之高分順利成為四年級生。<ref name="史話313"/>1910年前往[[日本|日本]]求學。1912年喪母。自日本[[京都|京都]]同志社中學普通部畢業後，於1915年進入[[慶應義塾大學|慶應義塾大學]]就讀。1920年聖誕節，劉青雲在[[東京|東京]]靈南坂基督教會，由小崎弘道牧師受洗入教。1921年，取得慶應義塾大學理財學士。<ref name="畢業證書">{{cite web |url=http://www.laijohn.com/Laus/Lau,Chun/diploma-Keio.htm |title= 劉青雲 慶應義塾大學 畢業證書 |accessdate= |last= |first= |author= |coauthors= |date=1921-03-31 |work=台南劉家雜錄 |publisher= }}</ref>留學期間，劉青雲與妹妹劉秀琴的學姊本目貞認識，當時本目貞正就讀於東京基督教所創辦之[[青山學院大學|青山女學院]]英文科。1922年，兩人於東京靈南坂基督教會結婚。婚後，劉青雲返回台灣協助父親經營家業，經營三多商會，從事實業，後任劉家和源農場管理人20多年。此外亦曾任台南市教育委員、台南市協議會員等公職。1924年，劉青雲建築一幢兩層樓英國式的洋樓，媲美當時的[[台南州知事官邸|台南州知事官邸]]。<ref name="壹葉通訊12"/>

1925年，劉青雲以[[白話字|白話字]]依照[[廈門話|廈門話]]語音，編纂並出版了《[[羅華改造統一書翰文|羅華改造統一書翰文]]》一書，共513頁，內容為羅馬字與漢字並用之對照文，是一部撰寫[[文言文|漢文]][[信|書翰文]]的指南用書。1942年，出版《[[大學精詳|大學精詳]]》，以漢文、日文以及白話字精解《[[大學_(經傳)|大學]]》，共48頁。在這之後，劉青雲轉而勤讀《[[聖經|聖經]]》，並於1965年12月聖誕期間發表《耶穌是誰？》一著作。1970年，完成聖書題典《[[天國奧秘全照射|天國奧秘全照射]]》上冊，隨後於1978年在完成該書之下冊，共計10,179題聖。<ref name="史話314"/>
