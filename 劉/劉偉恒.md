'''劉偉恒'''（'''Benny Lau'''，{{bd|1975年|5月6日|catIdx=L劉偉恒}}），別名「恆隊長」、「後波劉」，香港導演和編劇，曾為[[英皇娛樂|英皇娛樂]]旗下藝人，現任[[香港電台|香港電台]]節目主持，在商台任職時期曾推廣[[大韓民國|韓國]]文化，有「韓流第一人」之稱。

== 簡歷 ==
劉偉恆曾就讀聖貞德小學及[[聖貞德中學|聖貞德中學]]，在中學高中時讀理科班，後來於中學會考中取得16分的成績，當中包括中文及英文科各C級的成績。他讀到中六之後{{noteTag|劉偉恆本人於2015年3月31日在香港電台第二台廣播節目《守下留情》中透露自己中學會考成績。}}，即十多歲的時候跟隨家人移民美國，並於三藩市完成大學課程。1997年7月，劉氏畢業返港，加入[[香港商業電台|香港商業電台]]，主持深宵節目《[[一切從音樂開始|一切從音樂開始]]》。1998年，在一次經南韓轉機回三藩市探親的航程上，初次接觸韓國音樂，隨後取得當時上司[[鄭丹瑞|鄭丹瑞]]的批准，於其節目《[[我恆我訴|我恆我訴]]》中每周加插15分鐘播放韓語歌曲的環節《[[韓流襲港|韓流襲港]]》。

初期，韓國歌曲並未為香港聽眾接受，唯亦逐漸取得不少捧場客，《[[韓流襲港|韓流襲港]]》及後發展成獨立節目，並於2002年開設專門介紹韓國潮流文化的網站《韓流網》。他也為香港及[[中國大陸|中國大陸]]的報刊撰寫與韓國文化有關的專欄，亦曾訪問過近百位知名[[韓國|韓國]][[藝人|藝人]]。

踏入30歲後，劉氏有感於廣播界可進一步發展的空間有限，尤其不喜歡主持如「巴巴閉 Afternoon D」一類的節目 (尤其不喜歡唱那些改編歌詞作品)，故向商台提出只主持《[[韓流襲港|韓流襲港]]》節目，騰出時間向外發展。唯商台不同意此建議，故劉氏終決定離開商台。於2007年7月28日晚播出預錄的《[[韓流襲港|韓流襲港]]》代表他正式離開商台。

現時'''劉偉-{恒}-'''全力於電影圈發展，並加盟由[[張家振|張家振]]及[[楊紫瓊|楊紫瓊]]創立的[[星城娛樂|星城娛樂]]，成為旗下的導演及編劇。2007年，首個電影劇本《木人巷》，入選韓國釜山國際電影節的Pusan Promotional Plan (PPP)。2008年，他執導首部短片《[[Mr_Right|Mr Right]]》，由譚耀文及陳逸寧主演，[[彭浩翔|彭浩翔]]監製，此片入選<韓國[[釜山國際電影節|釜山國際電影節]]>, <[[香港國際電影節|香港亞洲電影節]]>及法國<One Country One Film Festival>。同年，另一劇本《[[王家欣|王家欣]]》入選台灣[[金馬獎|金馬獎]]創投會。

此外，劉偉恆亦擔任MV導演，作品包括：[[李克勤|李克勤]]<他慈我悲>、〈一個都不能少〉，<再見穿越機>，<兩人行>，[[Soler|Soler]]<風的季節>，[[溫家恆|溫家恆]]<同窗密友>，[[歐陽靖|歐陽靖]],[[陳奐仁|陳奐仁]]<買一送一>，[[C_AllStar|C AllStar]]<We Are Kitchee!><ref name ="ct-2014-01143"/>。

另外，劉偉-{恒}-於2009年開始以替工身份於香港電台擔任電台節目主持。2010年9月接替[[梁東|梁東]]，2011年10月1日開始與[[梁禮勤|梁禮勤]]正式主持《[[守下留情|守下留情]]》節目。<ref name ="ct-2014-01143"/>

2010年12月25日，劉偉恆接受浸禮加入[[九龍城浸信會|九龍城浸信會]]。<ref name ="ct-2014-01143"/>

2012年2月25日，劉偉恆與[[有線新聞台|有線新聞台]]主播/記者[[王沛然|王沛然]]於[[九龍城浸信會|九龍城浸信會]]結婚。<ref name ="ct-2014-01143"/>

2013年3月2日，長女出生，取名劉子晴(小饅頭)。

2015年1月2日，幼女出生，取名劉子悅(包包)。

2018年11月，劉遠赴四川拍攝第三部電影[[你的世界|你的世界]]，「你」為全資中國內地電影，演員包括[[張宥浩|張宥浩]]、[[薜昊婧|薜昊婧]]、[[向傲宇|向傲宇]]、[[呂頌賢|呂頌賢]]、[[祝延平|祝延平]]等等，電影預計在2019年下半年上映。
