'''劉立方'''（{{bd|1946年|12月11日|||catIdx=劉}}），出身[[台灣|台灣]]的美籍華裔[[海洋學|海洋學]]家，研究[[海嘯|海嘯]]、[[計算流體力學|計算流體力學]]的專家，以發展[[COMCOT模式|COMCOT模式]]('''Cornell Multi-grid Coupled Tsunami model''')及[[COBRAS模式|COBRAS模式]]('''COnstraint-Based Reconstruction and Analysis System model''')聞名於海洋工程界及水利工程界，曾擔任美國[[康乃爾大學|康乃爾大學]]土木與環境工程研究所主任，現擔任[[新加坡國立大學|新加坡國立大學]]行政副校長(2016年)、[[美國國家工程院|美國國家工程院]]院士(2015年)、台灣[[中央研究院|中央研究院]]院士(2016年)、[[美國|美國]][[康乃爾大學|康乃爾大學]]1912級工學講座教授。

== 生平 ==
劉立方於1968年獲得[[國立台灣大學|國立台灣大學]]土木工程學系學士，隨即考取公費留美，分別1971年及1974年獲得[[美國|美國]][[麻省理工學院|麻省理工學院]]土木及環境工程研究所碩士及博士，師事水動力學者，亦來自台灣的[[梅強中|梅強中]]教授。

畢業後，被美國[[康乃爾大學|康乃爾大學]]土木與環境工程學系聘任[[助理教授|助理教授]]一職。1979年升格為[[副教授|副教授]]，1983年正式升格為[[教授|教授]]，1985年到1986年間，劉教授出任土木與環境工程學系副主任，至1987年，他擔任土木與環境工程學系本科生課程副院長，在擔任土木與環境工程學院副院長期間，他亦擔任荷蘭[[代爾夫特理工大學|代爾夫特理工大學]]水工實驗室、[[丹麥技術大學|丹麥技術大學]]水動力學系、[[大阪市立大學|大阪市立大學]]等校的[[訪問學者|訪問學者]]。

2007年4月被台灣[[國立中央大學地球科學學院|國立中央大學地球科學學院]]水文科學研究所（2008年改名為水文與海洋科學研究所<ref>[http://www.ihs.ncu.edu.tw/people/bio.php?PID=5 國立中央大學水文與海洋科學研究所劉立方的個人資料]</ref>）聘為中大國鼎講座教授。

2009年至2015年6月30日間，他擔任美國[[康乃爾大學|康乃爾大學]]土木與環境工程學系主任，2015年被[[美國國家工程院|美國國家工程院]]延攬擔任院士一職，並擔任康乃爾大學1912級工學講座教授，2016年被台灣[[中央研究院|中央研究院]]延攬擔任院士一職，后又被[[新加坡國立大學|新加坡國立大學]]聘為行政副校長一職至今。

== 研究 ==
劉立方的研究興趣為海岸與海洋工程学，專攻海水波動理論、[[海啸動力學|海啸動力學]]、[[碎波|碎波]]過程、泥沙輸送過程、[[波浪|波浪]]與結構物相互作用等領域。{{fact|他在康乃爾大学依據非線性淺水波理論(NLSW)研制的海啸數值模擬COMCOT被多國採用}}，開發海啸預警系统和海啸高風險區溢淹圖以及用以估算海嘯災害造成的損失。

劉立方研制的另一個數值模擬軟體COBRAS是建立在雷诺時間-平均方程(Reynolds Averaged Navier-Stokes (RANS) equations)的基礎上，{{fact|也廣為國際採用為海岸結構工程初步設計的工具以及包括由山體滑坡引發的水波等的波浪與固体相互作用的研究工具。}}
