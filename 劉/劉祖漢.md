'''劉祖漢'''（1894年—1985年），[[中華民國|中華民國]]政治軍事人物。学名'''西屏'''，浙江[[奉化|奉化]]人。

==生平==

清[[光緒|光緒]]二十年，生於[[浙江省|浙江省]]奉化縣大桥镇北街村一個士绅家庭<ref>據《奉化縣志》記載</ref>。早年在奉化凤麓学堂唸書，與[[周淡游|周淡游]]、[[王恩溥|王恩溥]]、[[何禄山|何禄山]]、[[俞镇臣|俞镇臣]]、[[朱孔阳|朱孔阳]]、[[陈泉卿|陈泉卿]]等交好，是是“凤麓十兄弟”之一<ref>{{cite web |url=http://www.shtzb.org.cn/shtzw/node70/node222/u1a1746277.html |title=俞良钢教授与奉化俞氏家族 |publisher=上海統一戰線 |date=2012年1月31日 |accessdate=2014-08-18 |language=zh-CN |deadurl=yes |archiveurl=https://web.archive.org/web/20150924101517/http://www.shtzb.org.cn/shtzw/node70/node222/u1a1746277.html |archivedate=2015年9月24日 }}</ref><ref>{{cite web |url=http://fhnews.cnnb.com.cn/gb/node2/node472/node991/node1002/userobject1ai42644.html |title=少年蒋介石之四：接受新学 |publisher=奉化新聞網 |date=2006年5月26日 |accessdate=2014-08-18 |language=zh-CN |deadurl=yes |archiveurl=https://web.archive.org/web/20120617194245/http://fhnews.cnnb.com.cn/gb/node2/node472/node991/node1002/userobject1ai42644.html |archivedate=2012年6月17日 }}</ref>。毕业于浙江讲武学堂。1914年赴[[日本|日本]]留學，入讀[[明治大学|明治大学]]，翌年回國<ref>{{cite book|volume= |title=《太蔣介石的親情、愛情與友情》|author=[[呂芳上|呂芳上]] 等/合著  |year=2011 |accessdate=2014-08-18 |location=台北 |page=114 |isbn=9789571353579 |publisher=[[時報出版|時報出版]]}}</ref>。1921年，任粤军东路军工兵大队长。後入[[黃埔軍校|黃埔軍校]]，經[[孫中山|孫中山]]任命，與[[赵澄志|赵澄志]]等任招兵委員<ref>{{cite web|url=http://www.hoplite.cn/templates/hpshshwx0025.html |title=黄埔军校教导团组建及早期人事分析 |publisher=中国黄埔军校网 |accessdate=2014-08-18 |language=zh-CN}}</ref>。1925年，参加[[蔣介石|蔣介石]]討伐[[陳炯明|陳炯明]]叛變之[[东征|东征]]。[[白沙戰役|白沙戰役]]中，因保護蒋介石而左臂中弹致残，做了截肢手術。蒋因此送他再次赴日本養傷进修。回國後旋任广东省[[惠阳县|惠阳县]]县长，今[[惠州市|惠州市]]市長相當<ref name="liu">{{cite web|url=http://xnc.zjnm.cn/zdxx/twjs/view.jsp?zdid=8791&lmid=10&id=41342|title=刘祖汉旧宅|publisher=浙江農民信箱|accessdate=2014-08-18|language=zh-CN|deadurl=yes|archiveurl=https://web.archive.org/web/20140819084756/http://xnc.zjnm.cn/zdxx/twjs/view.jsp?zdid=8791&lmid=10&id=41342|archivedate=2014-08-19}}</ref>。

[[國民革命軍北伐|國民革命軍北伐]]初期，任國軍第二十六军第一师党代表，兼任政治部主任。后歷任两浙缉私统领，[[中華民國財政部|中華民國財政部]]皖南缉私局长，浙江省内河水上警察局长等职<ref name="liu"/>。

1944年，當選奉化县临时参议会议长。

1949年，移居[[台湾|台湾]]。1985年，病逝於[[台北|台北]]，享年91歲<ref name="liu"/>。

==故居==

1949年後，刘祖汉旧居被奉化地方接管徵用。2008年，浙江省文物普查時發現。刘祖汉台灣后人曾拜訪故居<ref>{{cite web|url=http://www.zjww.gov.cn/sjxmInfo/2008-10-14/22685628.shtml |title=奉化市第三次全国文物普查简报(第13期) |publisher=浙江文物網 |date=2008-10-14 |accessdate=2014-08-18 |language=zh-CN}}</ref><ref>{{cite web |url=http://www1.fhnews.com.cn/gb/node2/node356/node3974/node5197/node5236/node5239/userobject1ai211562.html |title=城里民国建筑的故事 |publisher=奉化日報 |date=2008年9月10日 |accessdate=2014-08-18 |language=zh-CN |deadurl=yes |archiveurl=https://web.archive.org/web/20140819090226/http://www1.fhnews.com.cn/gb/node2/node356/node3974/node5197/node5236/node5239/userobject1ai211562.html |archivedate=2014年8月19日 }}</ref>。據劉氏後人，劉生前曾回憶，蔣介石少年時愛好舞龍<ref>{{cite web|url=http://www.ihningbo.cn/info.jsp?aid=1377 |title=奉化发现蒋介石与奉化布龙的众多轶事 |publisher=寧波市非物質文化遺產保護網 |date=2013-04-22 |accessdate=2014-08-18 |language=zh-CN}}</ref>。
