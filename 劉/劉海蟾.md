'''劉海蟾'''，名'''操'''，字'''宗成'''，又字'''昭元'''、'''昭遠'''，道號'''海蟾子'''。後人俗稱'''劉海'''，是[[道教|道教]][[傳說|傳說]]人物。[[五代十国|五代十國]]時期[[廣陽|廣陽]]<ref>即[[幽州|幽州]]，古稱[[廣陽郡|廣陽郡]]，[[宋|宋]]曾置[[燕山府|燕山府]]，今[[北京市|北京市]][[豐臺區|豐臺區]][[宛平|宛平]]</ref>人，著名[[道教|道教]][[祖師|祖師]]，[[全真道|全真道]][[全真五祖|五陽祖師]]之一。以其[[广阳郡|廣陽]]人，人稱'''廣陽先生'''、'''廣陽劉真人'''。<ref>{{Cite web|url=http://www.etaoist.org/taoist/index.php/2012-08-16-20-59-99/item/1189-%E9%81%93%E5%AE%B6%E4%BB%8A%E6%97%A5-20130715 |title=道家今日 |work=道家身心靈線上雜誌－開運篇 |publisher=國際道家學術總會|date=2013-07-17|accessdate=2013-07-17檢索|quote=[[六月初十|六月初十]]是【劉海蟾帝君聖誕】}}</ref>劉真人道號海蟾子，道教界以'''刘海蟾'''呼之，但民間流傳「'''劉海'''戲金'''蟾'''」，被後人讹作「'''刘海'''」，'''刘海'''也成為世人對劉海蟾最普遍的稱呼。

相传为[[道教|道教]][[全真五祖|全真五祖]]之一，[[全真教|全真教]]「海蟾派」始祖。[[元朝|元]]世祖[[忽必烈|忽必烈]]封其为“'''海蟾明悟弘道真君'''”，[[元武宗|元武宗]]加封「'''海阳嗣派九天扶正帝君'''」，[[泰定帝|泰定帝]]封為「'''廣陽啟南弘道純佑帝君'''」。

相傳著有《還金篇》、《黃帝陰符經集解》等，與[[張無夢|張無夢]]、[[種雲溪|種雲溪]]、[[陳希夷|陳希夷]]等為道友。弟子[[张伯端|張紫陽]]、[[董凝陽|董凝陽]]。劉海蟾一直被質疑是[[虛構人物|虛構人物]]，其人生平眾說紛紜，因為有關劉海蟾的描述只出現在[[野史|野史]]、[[傳奇|傳奇]]、[[戲劇|戲劇]]、[[小說|小說]]等[[民間文學|民間文學]]，與道教典籍如《呂帝聖蹟紀要》等，[[正史|正史]]並無其人之記載。

== 出身 ==
一说刘海蟾名'''哲'''，或作'''喆'''，字'''玄英'''，又作'''元英'''，[[五代十国|五代]]時期[[后梁|后梁]][[燕_(五代)|燕國]][[广阳郡|廣陽]]（今[[北京市|北京市]][[大興區|大興區]]）人，又說[[滨州_(古代)|濱州渤海縣]]（今山東省[[滨州市|濱州市]]）人，[[寄籍|寄籍]]廣陽。或曰[[遼國|遼國]][[燕州|燕州]]廣陽（今北京）人。也有古籍說他是後梁[[廣陵|廣陵]]（今[[河南|河南]][[息縣|息縣]]）人，一說[[長安|長安]]人。

== 介紹 ==
相傳海蟾出身[[門閥|門閥]]，為[[傾奇者|傾奇者]]，喜歡將頭髮梳至額前，長度齊眉。長於[[經|經]][[史|史]]，少舉[[明經|明經]]甲科，为[[劉仁恭|劉仁恭]]禮聘為[[盧龍節度使|盧龍節度使]]鎮府[[司馬|司馬]]、都知兵馬使、檢校[[大理寺|大理寺]][[少卿|少卿]]、[[國子監|國子監]][[祭酒|祭酒]]，劉仁恭之子[[劉守光|劉守光]]囚禁劉仁恭，殺死兄長[[劉守文|劉守文]]，自立為王，而後[[稱帝|稱帝]]，升刘海蟾為[[相國|相國]]，但海蟾好[[黃老|黃老]]之術，治民寬簡，因守光攻擊父兄，重役[[軍夫|軍夫]]，施政無道，海蟾屢勸不聽，頗為失意。一日，遇[[八仙|八仙]]之中的[[鍾離權|鍾離權]]點化，[[鍾離權|鍾離權]]取[[銅錢|銅錢]]與[[雞蛋|雞蛋]]各十枚相疊（這是春秋時期[[晉靈公|晉靈公]]「危如累卵」的典故），問：「危乎？」刘答曰：「危甚矣。」鍾離曰：「相公性命，俱危更甚。」言畢，打破雞蛋，辭去。海蟾知功名難恃，災禍將至，於是棄官修道，入山時遇[[吕洞宾|吕洞宾]]，拜为师表，成為了點化他的[[钟离权|钟离权]]的再传弟子，自號'''海蟾子'''，至[[華山|華山]]隱居，又入[[代州|代州]][[鳳凰山|鳳凰山]]、[[長安|長安]][[终南山|终南山]]、[[福州|福州]][[武夷山|武夷山]]潛心修煉。

另說有位姓劉名玄英的讀書人，原名操，字宗成，號海蟾。此人好談[[性命雙修|性命雙修]]，崇拜[[黃老之術|黃老之術]]。傳說劉海蟾兩次遇到神仙，第一次遇到「正陽祖師」[[鍾離權|鍾離權]]，第二次遇到「呂祖」。據《神仙通鑑》記載：「初遇正陽祖師，授以金液還丹之旨，遂棄官學道。後遇呂祖，乃改名玄英，號海蟾子。复授以金丹之要，遁跡終南，修真成道。」有人說他是道教北宋第四祖。元代至元年間被封為「明悟弘道真君」。武宗時加封為「純佑帝君」。<ref>周新國《武陵藏珍》</ref>

另說劉喆是個富貴中人，有一天一位自稱「正陽子」（「真陽子」）的道人前來拜訪，和他大談「[[道德經|一生二，二生三，三生萬物]]」的「道」，劉喆於是大徹大悟，改號海蟾子，棄官學道，遁跡終南山下，後登仙班，駕鶴而去。道教[[金丹派|金丹派]]把他奉為祖師。

== 寶誥 ==
燕山名相，遼左才人。身處富貴之場，性好清虛之道。雲房點化以危機，感悟潛身而解組。三百口之火宅俱拋，十萬家之門兵盡屏。修真於武夷山中，顯化於王屋嶺上。濟度洪深，玄功廣博。大悲大願，大聖大慈。海陽嗣派，九天扶正帝君，啟南興教天尊。

== 軼事 ==
=== 瀏海 ===
{{Main|瀏海}}
道教全真派全真五祖之一的劉海蟾喜歡將頭髮梳至額前，長度齊眉，故稱此種頭髮為「劉海」，後為了區別，改稱「瀏海」。

==== 劉海戲金蟾 ====
俗有「劉海戲金蟾，一步一吐錢」之說。 [[神話|神話]]中，劉海蟾之親人為官甚貪，但因尚知修[[道|道]]，死後未被貶入[[地獄|地獄]]，被化作三足[[金蟾|金蟾]]投入[[東海|東海]]之中，歸[[龍王|龍王]]管轄。劉海蟾得道後，想要解救金蟾，以一串金錢釣三足蟾出海，（蓋因三足蟾性貪，見金錢便咬）負于肩上，是為「劉海戲金蟾」，而這[[金蟾|金蟾]]（傳說中三足的[[蟾蜍|蟾蜍]]名叫「金蟾」），走一步，能吐一枚[[銅錢|銅錢]]。劉海蟾走到哪，就把錢撒到哪，周濟窮人，而劉海蟾也被視為有[[財神|財神]]的能力。

據湖南常德的周新國考證，劉海戲金蟾這個故事，人物原型出自後梁[[燕山府|燕山府]]的劉海蟾，後人把劉海蟾這個字號一分為二：劉海、蟾，又改為「劉海戲金蟾」。

==== 劉海砍樵 ====
{{Main|劉海砍樵}}
[[湖南人|湖南人]]稱劉海是[[湖南|湖南]][[常德|常德]]人，是個孝順母親的樵夫，後感動狐仙，共結連理，並經過一連串的傳奇之後才終於得道成仙。其故事經過民間各種[[演戲|演戲]]與[[說書|說書]]，版本甚多，姑且錄一較通行的版本。

[[五代|五代]]時，常德絲瓜井（今常德市[[武陵區|武陵區]]）邊，住著劉海一家三口，劉海是個讀書人，父親是個樵夫。劉海二十歲考上[[秀才|秀才]]那一年，父親病逝，劉海母親因思念亡夫，日夜啼哭而失明，劉海找不到教書的工作，毅然決然放棄[[科舉|科舉]]，每天砍柴奉養母親，但因自己向來是讀書人，砍柴砍得不好，收入也不穩定，又加上把所有的收入都奉養母親，劉海常常餓得不成人形。

一日，劉海在山上遇到了一個衣不蔽體的美女，名喚「胡秀英」，秀英自稱父母雙亡，自己遭到[[強盜|強盜]]擄走，[[人口販賣|人口販賣]]多次，來到此地時終於脫逃，孤苦無依，劉海忙著把自己的衣服給她穿，兩人相談甚歡，秀英於是日日到劉海家中幫忙，並照顧劉母，由於秀英貼心又勤快，很會[[織布|織布]]，劉母非常喜歡秀英，於是鼓吹兩人結成連理。秀英同意後，兩人到城中的鷄鵝巷買新衣、[[妝奩|妝奩]]。其實秀英是一個美麗的狐仙，只需再修道三十年，就可以拜謁[[驪山老母|驪山老母]]，成為[[仙女|仙女]]。但她因前世與劉海有緣，又感念劉海的孝順，將自己苦練的「內丹寶珠」含在口中，化成人形，要嫁給劉海。

鷄鵝巷有一座阿羅寺，供奉[[十八羅漢|十八羅漢]]，[[羅漢|羅漢]]或乘龍或駕鶴，分庭抗禮，各有千秋，當中第十位羅漢踩的是「金蟾」，金蟾甚貪，起了惡念，展現神通，盜走了秀英的寶珠，秀英現出原形，也只好一五一十地向劉海哭訴，並要[[自盡|自盡]]，劉海勸阻了秀英，拿起斧頭去找「金蟾」決鬥，大罵如果找不到「內丹寶珠」，就打倒第十羅漢的[[偶像崇拜|金身]]。金蟾欺騙主人，說劉海受狐仙迷惑，要欺凌自己，羅漢大怒而直接[[顯靈|顯靈]]，祭起衣缽，用[[袈裟|袈裟]]蓋住劉海頭顱，縮小其身，壓在玉缽之下，並派遣[[護法|護法]][[伽藍神|伽藍神]]日夜看守，要引秀英出來，也把秀英收了。

秀英自知不敵佛法，正在萬分苦惱。此時斧頭居然說話，原來劉海仙風道骨，上天也派了一位神靈守護「斧頭」，是為斧頭神。斧頭神連忙帶著秀英找到了[[鍾離權|鍾離權]]與[[呂洞賓|呂洞賓]]。鍾離權與呂洞賓用[[法術|法術]]召來了[[護法|護法]][[伽藍神|伽藍神]]，但劉海依然被困在佛家的玉缽之下，兩個仙人用盡各種鈍器都無法打破玉缽，呂洞賓知道「解鈴還須繫鈴人」，佛家之物必須以佛家之物相剋，又駕雲到[[西天|西天]][[須彌山|須彌山]]向[[釋迦佛|釋迦佛]]求問。釋迦佛不發一語，只拿了一個金缽給身旁的[[目連尊者|目連尊者]]，目連尊者笑了一笑，已領會佛意，於是尊者隨著兩個仙人到了當地，尊者持當年[[破地獄|破地獄]]的錫杖一打，打破玉缽，劉海跳了出來，回復原來身材大小，但「內丹寶珠」非常火燙，劉海於是用[[袈裟|袈裟]]蓋在寶珠之上，拿回了「內丹寶珠」。第十羅漢發現後劉海被刼，袈裟被盜，玉缽被破，大怒，要與鍾離權與呂洞賓鬥法，目連尊者還未走，恰好告訴第十羅漢事實真相，把袈裟物歸原主，還送了一個釋迦佛用過的「金缽」給第十羅漢，知道自己受金蟾所騙的第十羅漢，笑著把金蟾送給了劉海。

由於金蟾每走一步就吐出一個[[銅錢|銅錢]]，劉海與秀英成為一方富戶。秀英在劉海母親年邁病逝後，與劉海分別，成仙而去。劉海將所有[[財產|財產]]拿出來，全部[[布施|布施]]給村中的貧病之人，也皈依[[道教|道教]]，帶著金蟾，隨著鍾離權與呂洞賓到[[終南山|終南山]]修道，最終成仙，[[鄉民|鄉民]]在劉海大戰羅漢的地方，建造了「劉海廟」以紀念之。

劉海砍樵的傳說在[[北宋|北宋]]時已經成型，至清代中葉已形成了今天流傳的[[傳奇|傳奇]]的主要版本。今天在常德還能找到傳說中的絲瓜井，但井口石板風化剝落，井壁石磚殘缺破損，井口四周雜草叢生。傳說中劉海大戰「第十羅漢」的「劉海廟」也早已損毀，雖然如此，劉海砍樵傳說在今天仍有深刻的現實意義，劉海夫妻身上的勤勞、正直、孝順等美德和追求幸福的美好願望，真實地表現了勞動人民的願望。傳說中的人物個性鮮明，堪稱湖南[[民間文學|民間文學]]的精品。劉海砍樵傳說保存了常德人民的生活狀態和民俗民情。
