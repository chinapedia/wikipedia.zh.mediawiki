'''可变数据打印'''（{{lang-en|Variable-data printing}}，缩写VDP）是一种数字印刷方式，也包括[[按需印刷|按需印刷]]，可变数据打印是指，在不停机或降低印刷速度的情况下，一个印刷页面中的元素如[[文本|文本]]、[[图形|图形]]和[[图像|图像]]在下一个印刷页面中可以立即改变，且使用的信息来自[[数据库|数据库]]或外部文件。<ref>{{cite web |title=  "ABC’s of VDP - A Variable Data Printing Basics Guide" |url=  http://www.efi.com/documents/promo/abc/ABCs_VDP_Letter.pdf |author= |work= Electronics for Imaging,Inc. | accessdate= 3 June 2011}}</ref>例如，一组具有相同排版的个性化字母，可以用不同的名字和地址来印刷。可变数据打印主要应用于[https://en.wikipedia.org/wiki/Direct_marketing 直接营销]、[[客户关系管理|客户关系管理]]、[[广告|广告]]、发票和邮件的群发、<ref name="multiple">{{cite web |title=  "Saving Mailing Costs with Variable Data Printing" |url=  http://www.ondemandexpo.com/on-demand-newsletter/saving-mailing-costs-with-variable-data-printing?utm_source=OnDemand&utm_medium=email&utm_campaign=eNewsletter_05_25_2011&utm_content=saving-mailing-costs-with-variable-data-printing |author=   |work=  On Demand Expo eNewletter |date=  23 May 2011 |accessdate=  1 June 2011 |deadurl=  yes |archiveurl=  https://web.archive.org/web/20120324211052/http://www.ondemandexpo.com/on-demand-newsletter/saving-mailing-costs-with-variable-data-printing?utm_source=OnDemand&utm_medium=email&utm_campaign=eNewsletter_05_25_2011&utm_content=saving-mailing-costs-with-variable-data-printing |archivedate=  2012年3月24日 }}</ref>
宣传册或明信片活动。

== 可变数据打印 ==
可变数据打印是[[數碼印刷|数字印刷]]的直接产物，它利用计算机数据库、数字印刷设备和高效的软件来创建高品质的全彩色文档，外观媲美传统胶印。相比于[[胶印|平版胶印]]“[[大量生產|大规模生产]]”的单个产品，可变数据打印通过数字印刷技术使印刷品的大规模定制成为可能。不同于生产10,000份相同的产品并提供其给10,000名客户，可变数据打印可以印刷10,000份独特的、带有定制信息的产品。

可变打印的赢利是浮动的，从一般费用的双倍到10-15倍都是有可能的。当然，这取决于印刷的内容和内容的相关性，但该技术为市场营销活动提供了一个有效的工具来增加[[投资回报率|投资回报率]]。

可变打印分为几个层次。最基本的层面包括更改每个印品上的称呼或姓名，这很像[[邮件群发|群发邮件]]。其次是更复杂的可变数据打印使用不同“版本”，在不同的市场也许存在各种不同的定制，基于不同的客户群来改变文本和图像。最后是完全可变打印，文本和图像都可以为每位客户改变。所有的可变数据打印都基于一条基础的设计理念，即一个页面上的静态元素（不变元素）和可变区域都要被打印。而不变的元素在每一页面中的位置实际上是相同的，可变区域被文本或图像填满，且受一系列的应用、页面的风格和数据库中的数据所限制。

可变数据打印包含三种主要的操作方法。<ref>{{cite web|title="Variable-Data Printing: Personalized Delivery of Information for the 21st Century"|url=http://www.digitaloutput.net/content/ContentCT.asp?P=78|author=|work=Rockport Custom Publishing, LLC|date=|accessdate=4 June 2011|deadurl=yes|archiveurl=https://web.archive.org/web/20111001035330/http://www.digitaloutput.net/content/ContentCT.asp?P=78|archivedate=2011年10月1日}}</ref>


第一个方法，加载一个静态文件到打印机内存，打印机服从一下指令，即通过[[打印机驱动程序|打印机驱动程序]]或[[RIP|光栅图像处理器]]（RIP）来打印任何发送到到打印机驱动程序或RIP的静态文件。可变数据可以被打印在静态文件的上面。这种方法是执行VDP最简单的方法，但是它的功能小于典型的群发邮件。

第二个方法是在打印之前使用标准的软件将静态和可变元素结合在打印文件中。这就产生了一个传统的（潜在的、巨大的）打印文件，<ref name="multiple 1"> {{cite web |title=   "Speaking In Tongues: Sorting Out Variable Data Printing Languages" |url=  http://www.fujixerox.com.au/products/image/media/TSR-0906-Speak-Tongues-reprint.pdf |volume= 7 |number= 17 |author= |work= The Seybold Report |date=  |accessdate= 4 June 2011}}</ref>
其中每个图像被合并到每一页。这种方法的缺点是运行很多非常大的打印文件会使RIP的处理能力崩溃。当发生这种情况，打印速度可能变得非常慢，对于超过几百页的印刷品来说是不切实际的。

第三个方法是在打印前使用专门的VDP软件将静态和可变元素结合在打印文件中。这将产生最佳的打印文件，如[[PDF/_VT|PDF/ VT]]，PostScript或[https://en.wikipedia.org/wiki/PPML PPML]，<ref name="multiple 1"/>打印速度可以达到最高，因为RIP只需要处理静态元素一次。
<ref>{{cite web |title=   "Managing Variable Data Full-Color Printing at High Speeds" |url=  http://www.infoprintsolutionscompany.com/internet/comnelit.nsf/Files/ManagingPrintatHighSpeedInfoPrint/$File/ManagingPrintatHighSpeedInfoPrint.pdf |Volume=  |Numbber=  |author=Hohensee, Reinhard |work= InfoPrint Solutions |date=  |accessdate= 5 June 2011}}</ref>

== VDP软件与服务 ==
许多软件程序包可以用来合并文本和图像并转换成的VDP打印文件。有些是独立的软件包，但是大部分优秀的可变数据打印软件程序包实际上是一个或多个排版软件如A[[Adobe_Creative_Suite|Adobe Creative Suite]]的插件模块。<ref>{{cite web|title="A New Generation of Variable Data Publishing Solutions"|url=http://www.adobe.com/products/vdp/pdfs/vdp_whitepaper.pdf|Volume=|Numbber=|author=|work=Adobe Systems Incorporated|date=|accessdate=5 June 2011|deadurl=yes|archiveurl=https://web.archive.org/web/20121002170743/http://www.adobe.com/products/vdp/pdfs/vdp_whitepaper.pdf|archivedate=2012年10月2日}}</ref>

除了VDP软件，VDP印刷程序可能也需要其它的软件包辅助。邮件发送软件在美国是必要的（[[美国邮政服务|美国邮政服务]]），加拿大利用[https://en.wikipedia.org/wiki/Bulk_mailing 批量邮寄]降低邮资。<ref name="multiple"/>使用邮件软件在创建VDP打印文件前进行预先分类和验证，并生成邮件地址的条形码。然后页面会按照正确分类的邮政编码顺序被打印。也许还需要使用软件来管理数据的质量（如对于删除重复信息）和一致性。<ref>{{cite web |title=  "Upstream Database and Digital Asset Management in Variable Data Printing" |url= http://printinthemix.com/ |author= Frey, Franziska |work= Printing Industry Center at RIT |date= |accessdate= 5 June 2011}}</ref>在软件购买方面，<ref name="multiple 2">{{cite web |title= "Designing for Digital: Skill Sets Needed to Design for Variable Data" |url= https://ritdml.rit.edu/bitstream/handle/1850/10895/35274_pdf_707B6614-DF7A-11DE-A4BD-8F409E1A67F9.pdf?sequence=1 |author= Jordan, Jessica |work= Rochester Institute of Technology |date=  |accessdate= 5 June 2011 |deadurl= yes |archiveurl= https://web.archive.org/web/20111002125833/https://ritdml.rit.edu/bitstream/handle/1850/10895/35274_pdf_707B6614-DF7A-11DE-A4BD-8F409E1A67F9.pdf?sequence=1 |archivedate= 2011年10月2日 }}</ref>许多公司提供分类服务，如VDP相关的打印文件，邮件和数据服务。

== 优势 ==
可变数据打印和传统印刷的区别在于其中所包含的个性化。个性化使企业连接到客户。可变数据打印不仅是页面中名称或地址的改变；以前名称变更是很有用的，因为当时这是一个新的概念。而现在个性化反映的是客户的价值。为了使VDP获得成功，企业必须了解客户。例如，喜爱棒球运动的客户收到一张来自VDP的印着自己喜欢的棒球运动员的明信片。明信片是有效的，因为客户很有可能阅读上面的内容。一个无用功的VDP明信片例子如下，邮寄一张足球运动员的明信片给这个客户，如果客户在足球没有兴趣，那么他可能就不会注意的这张明信片。这个想法是通过与客户联系的能力附加价值给他们。个性化允许公司联系、沟通客户，可与潜在客户建立联系，并保持与现有客户的关系。一个潜在的客户可以转化为客户甚至是忠诚的客户。企业要获得忠诚的客户，可变数据印刷可以凭借优质的工作和服务帮助他们达成目标。<ref name="multiple 3">{{cite web |title= "Variable data printing update" |url=http://multichannelmerchant.com/printchannel/0301-variable-data-printing-update/index.html |author= Tierney, Jim |work=  |date= |accessdate= 22 February 2013}}</ref>

另一个优势是可变数据打印提高了反应速度和反应时间。个性化吸引消费者的关注，通过邮寄使得反应速度提高。个性化同时也增加了响应时间，因为邮寄的宣传单对消费者的影响使消费者更快作出反应。顾客不会看一看放下稍后就忘记了，所以可能需要几个星期才能收到回应。<ref name="multiple 3"/>

== 消费者 ==
消费者是可变数据印刷的最为重要的部分。为了使活动取得成功，企业必须拥有客户的必要数据。数据非常重要，因为它决定什么时候开始营销产品。如果企业不了解它们的潜在消费者或客户，可能就会失去这些客户或没有意识到选错了目标人群。一个产品并不适用于每个人，所以了解目标消费者的需求、期望和爱好很重要。例如，想要卖狗给一个养猫的人并不可行。这些产品对消费者没有什么用，所以重要的是在营销前了解目标消费者。市场调研是经常被用来获取消费者的信息；它也是营销计划的第二步。下面有个例子关于市场如何运用问卷调查来了解消费者。问卷调查刺激消费者完成他们，所以通常有较高的回应率，因为消费者认为完成调查是有价值的。一旦目标消费者被发现，营销所需要的数据可以被使用。这个数据是可变的。研究获得的数据将存入数据库，在这里数据将被放入可变模块。可变元素的一个例子是年龄。年龄决定特定的年龄人群收到何种信息。<ref>{{cite web |title= "ABC's of VDP" |url=  http://www.efi.com/documents/promo/abc/ABCs_VDP_Letter.pdf |author=  |work=  |date= |accessdate= 22 February 2013}}</ref>

== 一体化 ==
可变数据打印可以与其他平台相结合，如PURLS、email blasts和 QR codes；这三个平台是公认的营销手段。许多人都发现为了成功进行营销，综合使用所有的平台非常有优势。群发邮件和PURLS使企业顾客的信息。群发邮件能包含一定的个性化信息，但通常不太多。PURL中包含大量的个性化信息，它是个性化的统一资源定位（URL）。总之，这是一个过度平台，它也是获得消费者大多数信息的地方。群发邮件包含一个PURL，它引导消费者进入个性化的页面。PURL使企业通过所要求的信息获得消费者的信息。 QR码可以被添加到一个明信片上。它的工作原理和群发电子邮件相同。它引导消费者到网页。综合三个平台有助于营销。<ref name="multiple 3" />

== 概念的起源 ==
可变数据打印这个术语来源于弗兰克•罗马诺（[[罗彻斯特理工学院|罗彻斯特理工学院]]影像艺术与科学学院印刷媒体学校名誉教授），罗马诺先生并没有因为塑造了这个术语得到广泛赞誉，，但重要的是早在1969年他就是用了这个术语。<ref name="multiple 4">{{cite web |title= "Frank Romano on VDP" |url=  http://www.efi.com/documents/promo/abc/ABCs_VDP_Letter.pdf |author= |work= Digital Formatting Services |date= 17 January 2013 |accessdate= 18 January 2013}}</ref>在1999年出版的“个性化和数据库打印”就是他和David Broudy共同撰写的。<ref name="multiple 4"/>

合并静态文档元素和可变文档元素的概念早于这个时间，那时就出现了各种应用，从简单的桌面“[[邮件群发|邮件合并]]”到金融业和银行业复杂的[[大型计算机|主机]]应用等等。过去VDP常常与数码印刷机器联系在一起。但是近年来这种技术的应用已普及到网页，电子邮件，和移动信息。
