'''可足渾皇后'''（？－386年前），[[十六国|十六国]]时期，[[前燕|前燕]]景昭帝[[慕容儁|慕容儁]]的皇后。生子[[慕容晔|慕容晔]]、[[慕容暐|慕容暐]]、[[慕容冲|慕容冲]]。

== 皇后 ==
可足渾氏开始是文明王[[慕容皝|慕容皝]][[世子|世子]][[慕容儁|慕容儁]]的夫人。[[元玺|元玺]]二年（353年）[[二月十七|二月十七]]，慕容儁立可足渾氏为[[皇后|皇后]]，立她的儿子[[慕容瞱|慕容瞱]]为[[皇太子|皇太子]]。元玺五年（356年）[[七月十二|七月十二]]，慕容瞱去世，慕容儁再以她次子慕容暐为皇太子。光寿二年（358年），出身段部鲜卑的吴王慕容垂王妃[[成昭皇后|段氏]]，自恃贵族，看不起可足渾皇后，可足渾皇后大怒，指使宦官中常侍涅浩指控段氏以巫蛊诅咒。段氏被捕，慕容儁又欲借此机会诬弟弟[[慕容垂|慕容垂]]谋反，段氏抵死不肯累及丈夫，结果被活活打死于狱中。慕容垂再娶了段氏之妹为新王妃，可足渾皇后强迫慕容垂废掉段氏，以她的妹妹长安君可足渾氏作吴王妃。慕容垂虽不敢拒绝，但表现得十分不悦，可足渾皇后更加讨厌慕容垂。

== 皇太后 ==
[[光寿|光寿]]四年（360年）[[正月廿一|正月廿一]]，慕容儁去世，太子[[慕容暐|慕容暐]]即位，改元[[建熙|建熙]]。二月，尊母亲可足渾皇后为[[皇太后|皇太后]]。可足渾太后开始干预朝政，当时，在慕容儁的叔叔上庸王[[慕容评|慕容评]]和大将[[慕輿根|慕輿根]]的协助下，慕容儁的弟弟太原王[[慕容恪|慕容恪]]执政。慕輿根想独揽大权，劝慕容恪夺位称帝，慕容恪当即拒绝。慕輿根又向可足渾太后诬告慕容恪和慕容评意图谋反。太后相信了慕輿根，准备命他缉捕二王，但十岁的小皇帝慕容暐不相信，从而终止了这项行动。不久慕容恪得知此事，便处死了慕輿根。慕容恪一直为前燕尽忠，建熙八年（367年）五月[[壬辰|壬辰]]，慕容恪去世。慕容恪临终遗言，推荐慕容垂总领兵权。但可足渾太后讨厌慕容垂，而慕容评怕慕容垂和自己争权，所以都没有同意，最终被任为大司马接手兵权的竟是年幼的皇弟慕容冲。

此后，可足渾太后、慕容评主持朝廷，二人信任腐败的官员，使慕容儁、慕容恪兄弟建立的前燕帝国元气大伤。

建熙十年（369年）[[四月十五|四月十五]]，根据可足渾太后的旨意，慕容暐立太后堂弟豫章公[[可足渾翼|可足渾翼]]的女儿可足浑氏为皇后。稍后[[东晋|东晋]]大将[[桓温|桓温]]北伐前燕，慕容暐之兄乐安王慕容臧战败。慕容暐向[[前秦|前秦]][[苻坚|苻坚]]求救，同时准备放弃[[中原|中原]]，逃回故都和龍（今[[辽宁|辽宁]][[锦州|锦州]]）。慕容垂劝阻了慕容暐，并自愿与桓温一战。不久，慕容垂联合前秦苟池的援军，几乎全歼桓温的军队，前燕幸免于难。

然而，可足渾太后依旧怨恨慕容垂，拒绝向慕容垂和他的军队给予赏赐，并想处死慕容垂。慕容垂听说后，立即投奔前秦苻坚，被苻坚封为大将。
 
前燕又违背与前秦的约定，没有将洛阳地区割让给前秦。建熙十一年（370年），前秦苻坚以此为借口，派[[宰相|宰相]][[王猛|王猛]]统兵东征前燕，王猛以少胜多，击败了慕容评的军队，很快攻克燕都邺城，前燕灭亡。但整个过程中，史书没有提到可足渾太后。

可足浑太后的幼子慕容冲因貌美被苻坚占作娈童，经过王猛劝谏，慕容冲才被送出宫。此后可足浑太后去世，以燕国皇后的礼仪埋葬。

== 身后 ==
[[建興_(慕容垂)|建興]]元年（386年），慕容垂自称[[后燕|后燕]]皇帝，恢复燕国宗庙。为给发妻复仇，他以可足渾太后倾覆社稷为由将其追废为庶人，改由没有儿子的[[段昭儀_(前燕景昭帝)|段昭仪]]配享慕容儁。
