'''肉孜·吾守尔'''（{{bd|1939年|7月15日|2011年|5月9日}}）笔名'''多浪'''，[[维吾尔族|维吾尔族]]，[[新疆|新疆]][[麦盖提|麦盖提]]人。[[中国人民解放军少将|中国人民解放军少将]]，[[中华人民共和国|中华人民共和国]]作家<ref name=zjw>{{Cite web |url = http://www.chinawriter.com.cn/zxhy/member/9850.shtml |title =肉孜·吾守尔 (1939～2011.5.9)  |author =  |date =  |publisher =中国作家网  |language =  |accessdate = 2017-02-04 |quote =  }}</ref><ref name=cpc>{{Cite web |url =http://cpc.people.com.cn/GB/64093/87393/14632258.html  |title =  肉孜·吾守尔同志逝世|author =  |date = 2011-05-13 |publisher =中国共产党新闻网  |language =  |accessdate =  |quote =  }}</ref>。

==生平==
肉孜·吾守尔是大专毕业。1956年1月加入[[中国共产党|中国共产党]]。1957年1月应征参加[[中国人民解放军|中国人民解放军]]，历任战士、班长、排长、副指导员、指导员，[[新疆军区|新疆军区]]8138部队政治处副主任、副政委，[[乌鲁木齐军区|乌鲁木齐军区]]军政干（部）校政治部副主任，[[南疆军区|南疆军区]]副政委兼喀什军分区政委、南疆军区纪委副书记，新疆军区副政委。他还是第九、十届全国政协委员<ref name=zjw/><ref name=cpc/>。

肉孜·吾守尔是新疆作协委员。1958年开始发表文学作品。1997年加入[[中国作家协会|中国作家协会]]<ref name=zjw/>。

2011年5月9日，肉孜·吾守尔在兰州军区乌鲁木齐总医院病逝，享年72岁<ref name=zjw/><ref name=cpc/>。
