'''促卵泡激素'''（{{Lang-en|'''follicle-stimulating hormone''', '''FSH'''}}-{zh-cn:，亦称为'''卵泡刺激素'''）; zh-hk:，亦稱為'''卵泡刺激素'''）; zh-tw:）}-是一种由[[脑垂体|脑垂体]]合成并分泌的[[激素|激素]]，属于糖基化[[蛋白质激素|蛋白质激素]]，因最早发现其对女性卵泡成熟的刺激作用而得名。后来的研究表明，促卵泡激素在男女两性体内都是很重要的激素之一，调控着发育、生长、青春期性成熟、以及生殖相关的一系列生理过程。促卵泡激素和[[黄体化激素|黄体化激素]]在生殖相关的生理过程中协同发挥着至关重要的作用。

== 结构 ==
促卵泡激素是一种[[糖蛋白|糖蛋白]]，活性形式是[[糖基化|糖基化]]的异源[[二聚体|二聚体]]，由α和β两条[[多肽|多肽]]组成。[[黄体化激素|黄体化激素]]，[[甲状腺激素|甲状腺激素]]，[[人绒毛膜促性腺激素|人绒毛膜促性腺激素]]等糖蛋白激素采用了与促卵泡激素相似的结构。它们共享同样的α亚基（含92位[[氨基酸|氨基酸]]残基），而β亚基则随激素的不同而不同。促卵泡激素的β亚基含有118位[[氨基酸|氨基酸]]残基，负责与[[促卵泡激素受体|促卵泡激素受体]]的相互作用。促卵泡激素表面的糖基化涉及[[海藻糖|海藻糖]]、[[半乳糖|半乳糖]]、[[甘露糖|甘露糖]]、[[半乳糖胺|半乳糖胺]]、[[葡萄糖胺|葡萄糖胺]]、以及[[硅铝酸|硅铝酸]]。其中，硅铝酸与促卵泡激素的生物[[半衰期|半衰期]]紧密相关。促卵泡激素的半衰期为3至4小时。促卵泡激素的分子量约为30000Da。

== 基因 ==
促卵泡激素α亚基的基因位于染色体6p21.1-23，在多种不同细胞中有表达；β亚基的基因位于染色体11p13，在脑垂体细胞中表达，受[[促性腺激素释放激素|促性腺激素释放激素]]的控制，被抑制素所抑制，被[[活化素|活化素]]所增强。

== 活性 ==
促卵泡激素调控人体的发育、生长、青春期性成熟、以及生殖相关的一系列生理过程，特别是刺激[[生殖细胞|生殖细胞]]的成熟。

=== 在女性体内 ===
在[[卵巢|卵巢]]中，促卵泡激素刺激尚未成熟的[[卵泡|卵泡]]的生长，直至成熟为[[格拉夫卵泡|格拉夫卵泡]]。卵泡在生长过程中会释放[[抑制素|抑制素]]以阻断促卵泡激素的进一步合成。这一机制保证了排卵的选择性。在黄体化阶段的末尾，促卵泡激素水平也有小幅度提升，可能与下一个排卵周期的开始有关。

=== 在男性体内 ===
在[[睾丸|睾丸]]中，促卵泡激素提高[[塞爾托利氏細胞|塞爾托利氏細胞]]合成[[男性激素|男性激素]]结合蛋白的水平，诱发塞尔托利细胞的紧密结合，同时分泌抑制素，在[[成精子|成精子]]过程中起到至关重要的作用。

== 作用机理 ==
促卵泡激素的目标细胞表面表达有[[促卵泡激素受体|促卵泡激素受体]]，属于[[G蛋白偶联受体|G蛋白偶联受体]]家族。促卵泡激素与其受体蛋白的结合将导致后者的构象变化。作为跨细胞膜的[[膜蛋白|膜蛋白]]，促卵泡激素受体胞外的变构将引发胞内的变构，改变其与[[G蛋白|G蛋白]]的结合状态，并通过其它蛋白的参与，进一步诱发[[环磷酸腺苷|环磷酸腺苷]]等[[第二信使|第二信使]]的生成，将信号传至细胞核内，实现对蛋白表达和细胞发育进程的调节。

== 相关疾病 ==
促卵泡激素水平在儿童期较低，而在女性的[[更年期|更年期]]之后则很高。

=== 高促卵泡激素水平 ===
促卵泡激素的高水平预示着性腺引发的限制性反馈缺失，导致脑垂体不断合成促卵泡激素。这在女性接近或处于更年期时属于正常现象，但对于处于生育年龄的女性则是不正常的，可能是以下疾病的信号：

# [[过早绝经|过早绝经]]也称为[[卵巢早衰|卵巢早衰]]
# [[性腺障碍|性腺障碍]]或[[Turner综合征|Turner综合征]]
# [[阉割|阉割]]
# [[斯外尔综合征|斯外尔综合征]]
# [[先天肾上腺增生|先天肾上腺增生]]的某些病例
# 睾丸失效

=== 低促卵泡激素水平 ===
促卵泡激素分泌水平的降低将导致性腺功能的缺失。在男性精子数量不足的病症中尤为典型。在女性中表现为生育周期的停止，具体相关的疾病有:

== 医疗应用 ==
促卵泡激素可在绝经期女性的尿液中提取得到，也可以通过[[基因工程|基因工程]]的方法[[重组|重组]]表达得到。临床用于对不育症患者的治疗，用以刺激卵泡的发育成熟，同时也用于[[体外人工受精|体外人工受精]]和[[体内人工受精|体内人工受精]]的相关治疗中。
