'''顧愷之'''（{{bd|約348年||405年|catIdx=G}}），[[表字|字]]長康，东晋人，是一位出身[[晋陵郡|晉陵]]無錫（今江蘇[[无锡市|無錫]]）的[[畫家|畫家]]與[[官員|官員]]，以其人物畫作品著稱。<ref name=":0">{{Cite web|url=http://www.aerc.nhcue.edu.tw/2-0-2/dung-jin/Gu-Kaizhi/main.htm|title=顧愷之|accessdate=2017-03-02|work=www.aerc.nhcue.edu.tw|deadurl=yes|archiveurl=https://web.archive.org/web/20160520121455/http://www.aerc.nhcue.edu.tw/2-0-2/dung-jin/Gu-Kaizhi/main.htm|archivedate=2016-05-20}}</ref><ref>{{Cite web|url=https://www.npm.gov.tw/dm/album/htm/name096_txt.htm|title=顧愷之|accessdate=2017-03|author=|date=|publisher=國立故宮博物院}}</ref>

== 生平 ==

出生年份没有记载<!-- 生年无具体记载。如果有新的发现，请提供来源。 -->。约364年在[[南京|南京]]为[[瓦官寺|瓦官寺]]画[[维摩诘|维摩诘]]像，引起轰动。366年当上[[大司马|大司马]][[大司马参军|参军]]，392年为[[殷仲堪|殷仲堪]]参军，405年升为[[散骑常侍|散骑常侍]]。

顾恺之多才，工[[诗|诗]][[赋|赋]]，善[[书法|书法]]，被时人称为“才绝、画绝、痴绝”，他的画线条连绵流畅，如“春蚕吐丝”。著有《[[s:論畫|论画]]》、《[[s:魏晉勝流畫贊|魏晋胜流画赞 (摹搨妙法)]]》和《[[s:畫雲臺山記|画云台山记]]》三本[[绘画|绘画]]理论书籍(以上三篇文章現今存在最早版本由[[唐朝|唐]][[張彥遠|張彥遠]][[s:歷代名畫記|歷代名畫記]]抄錄以傳)，提出“以形写神”、“尽在阿堵中”的传神理论。其[[陆探微|陆探微]]、[[张僧繇|张僧繇]]合称“六朝三大家”。<ref name=":0" />

== 繪畫 ==
顾恺之画迹甚多，有《司马宣王像》《谢安像》《刘牢之像》《王安期像》《阮脩像》《阮咸像》《晋帝相列像》《司马宣王并魏二太子像》《桂阳王美人图》《荡舟图》《虎豹杂鸷鸟图》《凫雁水鸟图》《庐山会图》《水府图》《行三龙图》《夏禹治水图》等。传世作品有《[[女史箴图|女史箴图]]》、《[[列女仁智图|列女仁智图]]》卷、《[[洛神赋图|洛神赋图]]》卷。<ref name=":0" />

===女史箴图===
《[[女史箴图|女史箴图]]》长卷，傳顧愷之的作品，現剩[[唐|唐]](8世紀)絹臨本藏於[[大英博物館|大英博物館]]，原為[[清|清]]宮藏畫，在[[英法聯軍|英法聯軍]]火燒[[圓明園|圓明園]]時被搶劫到[[英國|英國]]。另[[宋代|宋代]]白描紙臨本（一說為[[李公麟|李公麟]]筆）於[[北京故宮博物院|北京故宮博物院]]藏。

[[晉惠帝|晉惠帝]]時，[[賈后|賈后]]專權善妒，當時大文學家[[張華|張華]]作《[[女史箴|女史箴]]》一文來諷刺她，並借此教育宮廷婦女。顧愷之根據張華原作的主要內容作畫内容是所谓妇女应遵守的清规戒律。一共12段(現剩9段)，每段包含一个古代宫廷女人的“模范”故事，并在每段前面自己抄录张华的一段赋。唐人摹本上面盖满了从8世纪起，历代[[收藏家|收藏家]]和[[皇帝|皇帝]]的[[印章|印章]]，后面有[[金章宗|金章宗]]和[[清朝|清朝]][[乾隆|乾隆]]皇帝的以及其他人的[[题跋|题跋]]和[[乾隆皇帝|乾隆皇帝]]亲自画的一束[[兰花|兰花]]。画后有顾恺之自己的签字，这幅画可能是世界上最早的有画家签字的画。人物線條圓轉，後人稱之為「春蠶吐絲」，又叫「高古游絲描」，技法上受篆書影響。氣味古樸，其用筆的功力，線條的質量，都是後人很難達到的。

女史箴图于1900年[[八国联军|八国联军]]进攻[[北京|北京]]时被一名不知情的印度英兵（一說是英军大尉基勇松）為了玉畫扣從某貴婦廉價買下，1903年僅以二十五英鎊賣給[[大英博物馆|大英博物馆]]。前两段已经流失。大英博物馆曾经请[[日本|日本]]专家进行修复，為了不使畫作在捲動的過程當中受損，修復後只供摊平展览。<ref name=":0" /><ref>{{Cite news|url=http://trendfoundation.pixnet.net/blog/post/26018752-%E3%80%8A%E6%96%87%E5%AD%B8%E8%88%87%E7%B9%AA%E7%95%AB%EF%BC%9A%E9%A1%A7%E6%84%B7%E4%B9%8B%E2%94%80%E5%A5%B3%E5%8F%B2%E7%AE%B4%E5%9C%96%E3%80%8B|title=《文學與繪畫：顧愷之─女史箴圖》 @ 趨勢教育基金會 :: 痞客邦 PIXNET ::|last=(TRENDFOUNDATION)|first=TRENDFOUNDATION|newspaper=趨勢教育基金會|accessdate=2017-03-02|language=zh-TW}}</ref>

===洛神賦圖===
現剩两幅[[宋朝|宋朝]]人临摹的《[[洛神賦圖|洛神賦圖]]》，是根据[[曹植|曹植]]的[[洛神賦|洛神賦]]画的，画中曹植和随从在岸上遥望水上飘逸窈窕的洛神和各种[[神仙|神仙]][[怪物|怪物]]。其中一幅现藏[[北京|北京]][[故宫博物院|故宫博物院]]，另一幅被末代皇帝[[溥仪|溥仪]]卷逃到东北，[[終戰詔書|日本投降]]后散落民间，后被[[辽宁博物馆|辽宁博物馆]]收藏。<ref name=":0" />

===列女仁智图===
《[[列女传|列女传]]》是[[汉|汉]]代的名[[儒家|儒家]]學者[[刘向|刘向]]所写。顾恺之以绘画的方式传达了书中列女的故事。《[[列女仁智图|列女仁智图]]》是[[宋|宋]]人摹本。但仔细端详，保留了顾恺之过人的画技。现存《列女仁智图》仅余二十八个人物。在顾恺之的笔下，画卷中人物的神态、气质、身份被描绘得生动有趣，恰如其分。顾恺之对生活观察十分细致。这样才可以将每一个人物的神态、气质描绘得如此传神。《列女仁智图》省掉背景平列人像的画法，是[[汉|汉]]代同类题材大多采用的画法，称作"平列构图布局法"，。顾恺之的画技，通过人物的姿态表情，使平摆浮搁的人物之间有了内在联系。现藏[[北京故宫博物院|北京故宫博物院]]。<ref name=":0" />

===斫琴图/斲琴图===
《[[斫琴图|斫琴图]]/[[斲琴图|斲琴图]]》传为顾恺之之[[宋|宋]]摹絹本。此图描绘古代文人学士制琴的场景。畫中有14人，或斷板、或制弦、或試琴、或旁觀指揮，還有幾位侍者（或學徒）執扇或捧場，但从画面的空间布局上看，工艺流程的先后次序不明显，而且人物之间的关系也缺乏确定性与连续性。由于没有具体的历史故事背景，也没有相关的文字说明，因此很可能是脱离文字的故事性而强调人物的各自特征。在古代的长卷人物画中，通常都注重故事的敷演，表达一定的伦理道德观念。此画恰在这一点上忽略不计。此图虽不及《[[洛神赋图|洛神赋图]]》有名，及具代表性，但在风格特征上仍凸现出顾恺之的千古一绝。現藏[[北京故宮博物院|北京故宮博物院]]。<ref name=":0" />
