'''{{ruby-py|涪|-{zh-cn:fú; zh-tw:ㄈㄨˊ; zh-hk:fau<sup>4</sup>}-}}陵区'''是[[中華人民共和國|中華人民共和國]][[重庆市|重庆市]]下辖的[[市辖区|市辖区]]，面积2945平方公里，总人口120万，是重庆第三大城市。涪陵位于重庆市中部，城市建立在[[长江|长江]]、[[乌江|乌江]]两岸，因当地[[榨菜|榨菜]]产业发达，又有“榨菜之乡”的说法。

涪陵区，涪陵——因乌江古称“涪水”，[[巴国|巴国]]先王陵墓多葬于此而得名。地处重庆中部，邑枕长、乌两江，素为乌江流域物资集散地，有渝东门户之称，是闻名遐迩的“中国榨菜之都”。

涪陵，古代巴国故都、军事重镇，长江中上游地区区域中心城市之一，千里乌江第一大城市，重庆市省域副中心城市，[[成渝经济区|成渝经济区]]四大中心城市之一和主要工业城市，重庆第三大城市。

全境东西宽74.5公里，南北长70.8公里，幅员面积2941.46平方公里。截至2013年末涪陵区辖4个园区、8个街道、12个镇、 6个乡，总人口116.50万，城区常住人口67万，是重庆第三大城市、成渝经济区东部中心城市。涪陵区自然条件好，人文资源丰富。涪陵榨菜、涪陵水牛、涪陵红心萝卜是闻名海内外的三大特产；水底碑林白鹤梁、程朱理学点
易洞是名闻中外的名胜古迹。

== 历史沿革 ==
* 涪陵因[[乌江|乌江]]古称涪水，[[巴国|巴国]]先王陵墓多葬于此而得名，巴国在此设郡。
* [[秦昭王|秦昭王]]三十年（前227年）置枳县，为区境置县之始。隋置涪陵县，隶属巴郡；[[元朝|元朝]]入涪州。明清均屬[[重慶府|重慶府]]。民国初年（1913年），改涪州为涪陵县，先后隶属川东道和四川省第八区。
* 中华人民共和国成立后，1950年初，四川省第八行政督察區析置涪陵專區和[[酉陽專區|酉陽專區]]，涪陵專区辖涪陵縣、[[南川縣|南川縣]]、[[酆都縣|酆都縣]]、[[武隆縣|武隆縣]]、[[彭水縣|彭水縣]]5縣，並將[[大竹專區|大竹專區]][[长寿縣|长寿縣]]、[[萬縣專區|萬縣專區]][[石砫縣|石砫縣]]劃入，隶川东行署区。
* 1952年12月20日，[[酉陽專區|酉陽專區]]撤銷，所屬的[[酉陽縣|酉陽縣]]、[[秀山縣|秀山縣]]、[[黔江縣|黔江縣]]劃歸涪陵專區<ref>1952年12月20日政務院【政政齊字第169號】批復</ref>。
* 1952年9月27日，長壽縣劃歸重慶市直轄<ref>1952年9月27日西南軍政委員會【會民令字第0033號】命令，1953年1月16日內政部【內民字第32號】批復</ref>。
* 1953年1月16日，[[重慶市|重慶市]]長壽縣劃歸涪陵專區<ref>1953年1月16日內務部【內民字第32號】批准</ref>。
* 1953年3月10日，[[大竹專區|大竹專區]]墊江縣劃歸涪陵專區<ref>1953年3月10日政務院【政政孫字第50號】批復</ref>。
* 1959年4月13日，長壽縣劃歸重慶市<ref>四川省人民委員會1959年3月5日報告</ref><ref>1959年4月13日四川省人民委員會【川議字第0152號】批復</ref>。
* 1968年9月，[[涪陵專區|涪陵專區]]改名[[涪陵地區|涪陵地區]]。
* 1983年9月9日，撤銷[[涪陵縣|涪陵縣]]，設立[[涪陵市|涪陵市]]（縣級）<ref>1983年9月9日國務院【國函188號】批復：3．撤銷涪陵縣，設立涪陵市（縣級）。以涪陵縣的行政區域為涪陵市的行政區域。</ref>。
* 1988年5月18日，[[石柱縣|石柱縣]]，[[秀山縣|秀山縣]]，[[黔江縣|黔江縣]]，[[酉陽縣|酉陽縣]]和[[彭水縣|彭水縣]]從涪陵地區劃出，單獨設立[[黔江地區|黔江地區]]<ref>同意你省的石柱土家族自治縣，秀山，黔江，酉陽土家族苗族自治縣和彭水苗族土家族自治縣從涪陵地區劃出，單獨設立黔江地區。地區行政公署駐黔江土家族苗族自治縣聯合鎮。1988年5月18日國務院【國函76號】批復</ref>
* 1995年11月5日，撤銷涪陵地區、縣級涪陵市，設立地級涪陵市，新設立枳城區和李渡區，涪陵市轄原涪陵地區的墊江縣、武隆縣、豐都縣和新設立的枳城區、李渡區。原涪陵地區的南川市由省直轄<ref>1995年11月5日國務院【國函106號】批復；1995年12月5日四川省人民政府【川府函518號】批復：1．同意撤銷涪陵地區、涪陵市，設立地級涪陵市，市人民政府駐新設立的枳城區。2．涪陵市新設立枳城區和李渡區。枳城區轄原縣級涪陵市的敦仁、崇義、荔枝、江東4個街道辦事處和黃旗、龍橋、白濤、焦石、珍溪、清溪、南沱、百勝8個鎮及山窩、羅雲、卷洞、叢林、仁義、中峰6個鄉，區人民政府駐中山東路；李渡區轄原縣級涪陵市的李渡、藺市、馬武、新妙、龍潭、青羊、石沱、堡子、鎮安、義和、致韓11個鎮和明家、增福、兩匯、石龍、同樂、酒店6個鄉，區人民政府駐李渡鎮。3．涪陵市轄原涪陵地區的墊江縣、武隆縣、豐都縣和新設立的枳城區、李渡區。原涪陵地區的南川市由省直轄。1995年12月6日四川省人民政府【川府民政87號】批復：涪陵市將枳城區的龍渡鎮劃歸李渡區管轄；將李渡區的酒店鄉劃歸枳城區管轄。</ref>。
* 1996年9月15日，涪陵市委託重慶市代管<ref>將萬縣市、涪陵市和黔江地區委託重慶市代管。1996年9月5日黨中央、國務院批復；9月15日正式實行</ref>。
* 1997年3月14日，劃入重慶直轄市<ref>第八屆全國人民代表大會第五次會議通過：1.設立重慶直轄市，撤銷原重慶市。2.重慶直轄市轄原四川省的重慶市、萬縣市、涪陵市、黔江地區所轄行政區域。</ref>。
* 1997年12月20日，撤销原地级涪陵市和枳城区、李渡区，设立重庆市涪陵区，重庆市涪陵区辖原枳城区、李渡区的行政区域。原涪陵市所辖（代管）的南川市、武隆县、丰都县、垫江县改归重庆市直接管辖<ref>1997年12月20日，中共中央辦公廳、國務院辦公廳批復：2.撤銷涪陵市（及所轄枳城區、李渡區），設立重慶市涪陵區。1998年2月22日掛牌；原涪陵市所轄的南川市、豐都縣、墊江縣、武隆縣由重慶市直接管理。</ref>。

* 1953年9月16日，涪陵縣第十三區的江家、大石、雲集、燕石、安順、石回、兩桂、白林、大水、福盛、石塔、青平、華中等13個鄉，第十四區的同興、瑪瑙、板橋、鐵牛、飛龍、群策、萬壽、禦龍等8個鄉，共21個鄉劃歸[[長壽縣|長壽縣]]；第十四區的綠柏、雲山、觀鬥、沙坪、保和、百家、古福、龍塘等8個鄉，第十五區的水井、武功、打鼓、土豐、龍井、包家、鳳宜、安樂、土主等10個鄉，第十六區的箐口、三匯、巧嶺、勝利、裴江、石廟、平溪、永平、雞棲、東村、老君、沈家、東匯等13個鄉，共31個鄉劃歸[[墊江縣|墊江縣]]<ref>1953年9月16日涪陵專署【民行字第21號】通知</ref>。

== 地理 ==
地理坐标介于东经106°56'—107°43'，北纬29°21'—30°01'之间。地形以低山浅丘为主。

===气候===
区内属中亚热带湿润季风气候，四季分明，常年平均气温18.1℃，年均降水量1072毫米，无霜期317天，日照1248小时。

{{涪陵区气候数据}}

==交通==
已建成水路、公路、铁路交通枢纽。区内水泥公路总长1477公里，有直抵主城区的[[渝涪高速公路|渝涪高速公路]]，[[渝怀铁路|渝怀铁路]]2005年通车。规划中的渝长(沙)高速公路及南涪、渝利两条铁路贯穿全境。[[长江|长江]]横贯东西，[[乌江|乌江]]纵穿南北。2013年重庆到涪陵的南线高速公路通车，主城到涪陵1小时左右的车程，同时高速可以到丰都。

==宗教==
涪陵民风纯朴，宗教信仰主要有佛教、道教、天主教、基督教。 

==资源==
*植被：森林90万亩，植物4000余种，中药材946种，野生动物750余种。
*矿产：26种，储量大，品位高，易开采。

==旅游==
巴文化、易文化、[[榨菜|榨菜]]文化相互交融，有[[国家级风景名胜区|国家级风景名胜区]]2处，省级3处：水底碑林[[白鹤梁|白鹤梁]]，[[巴国|巴国]]故里小田溪，理学圣地点易洞，全国罕见、重庆唯一的野生植物带石夹沟，雄奇峻秀的乌江画廊。另有近年来新开发出武陵山大裂谷风景区，位于重庆市涪陵区城东南约40公里的武陵山乡境内，武陵山系西北尾端，乌江下游东岸，景区创建面积9平方公里。景区平均海拔1300米，最高处1980米，山势奇峻多姿，原生植被丰富，种类繁多，共有2000余种野生植物和200余种野生动物；空气清新宜人，生态环境极其优良，山、林、泉、洞、瀑、崖、湖、潭、峡、坑、缝一应俱全，旅游资源极为丰富。十里大峡谷雄阔壮美，谷底奇石叠垒，溪流淙淙；天然绝壁宛如万里长城，绵延数里；地缝、暗河神奇壮观。其中铜墙铁壁、如来神掌、将军石、青天峡地缝等景点堪称一绝。

== 经济 ==
涪陵经济在重庆直辖时划入之“新重庆地区”中，处于较领先的位置，其中[[太极集团|太极集团]]、[[重庆乌江实业集团|重庆乌江实业集团]]为[[上市公司|上市公司]]。

已与50多个国家和地区进行了贸易。
