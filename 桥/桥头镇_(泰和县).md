'''桥头镇'''是[[江西省|江西省]][[泰和县|泰和县]]下辖的一个镇，位于泰和县西部，距县城49公里、[[井冈山机场|井冈山机场]]20公里，总面积253平方公里，人口1.5万。境内有[[白鹭湖|白鹭湖国家森林公园]]、[[楠木林度假村|楠木林度假村]]、朝仙崖文天祥抗元议事旧址等风景名胜，是泰和县新兴的旅游热点区。

== 地理与交通 ==

桥头镇位于泰和县西部，北邻[[吉安县|吉安县]]，南毗[[遂川县|遂川县]]。境内地形主要为丘陵和山地，南高北低。[[灉水|灉水]]（境内称六八河）和六七河和分别自西北、西南入境，横穿全镇，在东部的湛口村汇合后，称为牛吼河，向东流入[[禾市镇|禾市镇]]。六八河上建有大型水库[[南车水库|南车水库]]。

行政中心和主要村镇分布于六七河两岸，[[319国道|319国道]]沿河而行，[[泰井高速公路|泰井高速公路]]横穿北部，镇政府距[[井冈山机场|井冈山机场]]30公里、县城48公里、[[泰和火车站|泰和火车站]]52公里。

== 沿革 ==

现桥头镇辖境在1950年时属津洞、桥头等乡，1956年合并设立桥头乡，1958年改为桥头公社，1984年撤消人民公社，恢复为桥头乡<ref>李宝库 主编：《中国政区大典·二》，杭州：浙江人民出版社，1999年，ISBN 7-213-01789-6，第1185页</ref>。2000年6月27日撤消桥头乡，改设桥头镇<ref name="jaxz">〈[http://www.jxqhdm.com/NEWS/newshow.aspx?ID=2200 吉安市乡镇行政区划变更]{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}〉，江西区划地名网，2007-10-16，于2009年4月26日查阅</ref>。

== 行政区划 ==

桥头镇现辖1个[[居委会|居委会]]和12个[[村委会|村委会]]：桥头镇居委会、津洞村、中坪村、西池村、店前村、小山村、黄陂村、水坑村、石壁村、湛口村、东山村、春和村、毛家村，镇政府驻小山村<ref>〈[http://www.jxqhdm.com/index-ja.aspx 2009年泰和县行政区划] {{webarchive|url=https://web.archive.org/web/20091025153207/http://www.jxqhdm.com/index-ja.aspx |date=2009-10-25 }}〉，吉安区划地名网（江西区划地名网子网站），于2009年4月1日查阅。</ref>。
