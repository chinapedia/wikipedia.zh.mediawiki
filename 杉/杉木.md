'''杉木'''（[[學名|學名]]：''{{lang|la|Cunninghamia lanceolata}}''）又稱'''福州杉'''、'''刺杉'''，为[[柏科|柏科]][[杉木屬|杉木屬]]植物。原產於[[中國|中國]]及[[越南|越南]]，並有一[[變種|變種]][[巒大杉|巒大杉]]，分布在[[台灣|台灣]]，模式標本採自[[浙江|浙江]][[舟山|舟山]]。

== 形态特征 ==
常绿[[乔木|乔木]]，高可达30米以上；树冠常呈尖塔形；树皮鳞状，褐色，内皮红色；大枝平展，小枝近轮生；线状披针形叶子，革質，稍硬而刺手，先端漸尖，少微鈍，葉緣有細齒，葉面深綠色，有光澤，除葉先端及基部外兩側有窄氣孔帶，微具白粉或白粉不明顯，葉底淡綠色，沿中脈兩側各有1條明顯白色氣孔帶；老樹之葉通常較窄短、較厚，上面無氣孔線。<ref>[http://frps.eflora.cn/frps/%E6%9D%89%E6%9C%A8]中国植物志  第7卷 杉木
Cunninghamia lanceolata</ref>长2.5～6厘米，基部扭转成二列；雌雄同株，花期4月；圆卵形球果，当年十月下旬成熟，每种鳞有种子3粒，暗褐色，有光澤，种子两侧有翅，果期10月。<ref name="中药大辞典" /><ref name="邓莉兰" />

== 分布 ==
天然分布于中国[[秦岭-淮河线|秦岭、淮河]]以南地区，[[越南|越南]]、[[老挝|老挝]]北部，變種[[巒大杉|巒大杉]]則分布在[[台湾|台湾]]山区。<ref name="邓莉兰">{{cite book|author=邓莉兰|title=常见树木.2.南方|publisher=[[中国林业出版社|中国林业出版社]] |location= |language=zh-hans |isbn=978-7-5038-4590-1|pages=40页 |date=2007年1月}}</ref>中國垂直分佈的上限隨地形和氣候條件的不同而有所差異。在東部[[大別山|大別山]]區海拔700米以下，[[福建|福建]][[戴雲山|戴雲山]]區1000米以下，在[[四川|四川]][[峨眉山|峨眉山]]海拔1800米以下，[[雲南|雲南]][[大理|大理]]則在海拔2500米以下。

==習性==
喜溫暖濕潤氣候及深厚、肥沃、排水良好、含[[腐殖质|腐殖质]]较多的酸性土壤中，幼樹較耐陰。<ref>[http://www.eflora.cn/spjjzw/%E6%9D%89%E6%9C%A8]《中国经济植物志》第 1039 页</ref>，不耐水淹和鹽鹼，在陰坡生長較好，淺根性，生長較快。<ref>[http://www.eflora.cn/sp/%E6%9D%89%E6%9C%A8]Cunninghamia lanceolata (Lamb.) Hook. 園林樹木1200種</ref>

==繁殖==
主要利用扦插及播種繁殖。

== 用途 ==
杉木生长迅速，是山区绿化的重要树种，材质优良，木材黄白色，有时心材带淡红褐色，有香气，材質軟硬適中，紋理直，易加工，較耐腐，供建築、橋樑等用，15至20年即可成材。<ref name="邓莉兰" />杉木的心材及树枝、根、树皮、叶、种子、油脂均可入药。<ref name="中药大辞典" />樹皮及根、葉能祛風燥濕、收斂止血；種子含油約20%，供製肥皂。<ref>[http://www.eflora.cn/sp/%E6%9D%89%E6%9C%A8]Cunninghamia lanceolata (Lamb.) Hook. 中國高等植物圖鑑</ref>由於杉木用途眾多，被稱為'''萬能之木'''。

== 异名 ==
杉木首见于《[[唐本草|唐本草]]》，别名杉（《[[南方草木状|南方草木状]]》），柀、煔（《[[尔雅|尔雅]]》），沙木、檠木（《[[本草纲目|本草纲目]]》），杉树、沙树、刺杉、廣州杉、福州杉、廣葉杉、正杉等。<ref name="中药大辞典">[[江苏新医学院|江苏新医学院]]. [[中药大辞典|中药大辞典]]. [[上海科学技术出版社|上海科学技术出版社]]. 1986.5. 1037页. ISBN 7-5323-0842-1</ref> <ref>[http://plant.tesri.gov.tw/plant100/WebPlantDetail.aspx?tno=208002020] 台灣野生植物資料庫-杉木</ref>
