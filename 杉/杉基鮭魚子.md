'''杉基鮭魚子'''<ref>台灣角川翼文庫（Wing Books）所採之譯名。</ref>（{{lang-ja|杉基 イクラ}}，Sugimoto Iqura，{{bd||8月17日|catIdx=Sugimoto Ikura}}），是日本[[北海道|北海道]]出身的女性[[漫畫家|漫畫家]]。現居住在[[關東|關東]]。

== 人物 ==
小學時代即對喜歡的機器人動畫描繪，高中開始畫[[同人誌|同人誌]]和商業漫畫選集，就讀[[短期大學|短期大學]]時出道。<ref>{{lang|ja|{{Cite web |url = http://tokyomangalab.com/?interviews=iqwura-sugimoto |title = 杉基イクラ先生インタビュー【ナナマル サンバツ／サマーウォーズ】 |publisher = 東京マンガラボ |accessdate = 2013-03-17 |deadurl = yes |archiveurl = https://web.archive.org/web/20130304074715/http://tokyomangalab.com/?interviews=iqwura-sugimoto |archivedate = 2013-03-04 }}}}</ref>

已婚，家中育有一兒。早期以[[筆名|筆名]]{{nihongo|「久遠摩緒」|くおん摩緒}}從事[[BL|BL]]漫畫活動，同人誌則使用{{nihongo|「久遠寺摩緒子」|久遠寺摩緒子}}名義。之後改為現在的筆名，「杉基」取自[[本名|本名]]漢字讀音{{nihongo|「sugimoto」|スギモト}}、{{nihongo|「鮭魚子」|{{lang|ja|イクラ}}}}是愛吃的東西<ref>{{lang|ja|{{cite web|url=https://twitter.com/iqura_s/status/105946971042361344 |title=<nowiki>杉基=漢字は違うけど本名。今となっては旧姓。イクラ=好物。</nowiki> |publisher=[[Twitter|Twitter]] |author=@iqura_s（杉基イクラ）||accessdate=|date=2011-08-23 |language= }}}}</ref><ref>{{lang|ja|[http://theinterviews.jp/iqura_s/181368 はい！はいせんせぇ！(｀ ◕ ω ◕)ﾉ PNを変えたのはなんでですかー？]. THE INTERVIEWS. 2011-09-06.}}</ref>，並走一般作品及插畫的風格。

曾於2006年第14屆[[台北國際書展|台北國際書展]]來台舉行《寄生少女》的簽名會。<ref>[http://www.tibe.org.tw/tibe/2006/index.php?language=ch&page_code=press_detail&new_no=166 寄生少女漫畫家杉基來台參加台北國際書展 魔怪畫風令粉絲讚嘆]. 台北國際書展. 2006-02-08.</ref>
