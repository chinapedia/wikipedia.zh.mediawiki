《'''禅真后史'''》，[[明朝|明朝]]署名[[方汝浩|方汝浩]]的[[长篇小说|长篇小说]]，是《[[禅真逸史|禅真逸史]]》的续书。<ref name="multiplebk562 " >{{cite book |language =zh-hans|author=方汝浩|title=《禅真后史》|year=2006-06-01|publisher=太白文艺出版社|location=陕西|pages=|isbn=9787806058954}}</ref>

== 内容 ==
全书承接《[[禅真逸史|禅真逸史]]》而来，讲述[[唐朝|唐朝]][[贞观|贞观]]年间的[[宫廷|宫廷]][[政治|政治]]、[[军事|军事]][[战争|战争]]、[[社会生活|社会生活]]动态。故事讲述[[玉帝|玉帝]]排遣薛举下凡尘，俗名叫做瞿琰。经过仙道指点而能文能武，先后除妖降魔、锄奸除恶、整顿乾坤，后出任官职，不久后隐退，随僧人林澹然遁入禅门，成佛升天。<ref name="multiplebk562 " />

== 思想 ==
全书仍旧继承了《[[禅真逸史|禅真逸史]]》的思想，作者希望诞生一个理想化的人物来拯救社会，治理时政，实现清平太平的生活。<ref name="multiplebk562 " />

== 回目 ==
全书一共六十回。<ref name="multiplebk562 " />

== 禁毁 ==
[[同治|同治]]七年（1868年），[[丁日昌|丁日昌]]下令严禁“淫词小说”，查禁[[小说|小说]][[戏曲|戏曲]]书目122种，“淫词”[[唱片|唱片]]114种，续查“淫书”34种，分2批共计268种，《禅真后史》即为其中之一。<ref name="multiplexlw11 " >{{cite web |url= http://book.sina.com.cn/nzt/sex/fushichunmeng/32.shtml|title=对性文学的查禁 |accessdate=2006年1月24日 |work=|publisher=新浪网 |language=zh-hans}}</ref>
