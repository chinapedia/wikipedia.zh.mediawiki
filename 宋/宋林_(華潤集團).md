'''宋林'''（{{bd|1963年|2月3日||}}），[[山東|山東]][[乳山|乳山]][[曲水村|曲水村]]（今属[[威海市|威海市]]）人，出生于山东省[[济南市|济南市]]，曾为中共黨員、驻港[[中资|中资]][[央企|央企]]高管、企业家、全国政协委员。他是前[[香港中國企業協會|香港中國企業協會]]會長，前[[華潤集團|華潤集團]]、[[華潤信託|華潤信託]]、[[漢威資本|漢威資本]]董事長等職位。香港中國企業協會在特首選舉委員會，坐擁16票。2014年4月17日因涉嫌严重违纪违法被调查，2015年9月11日被开除党籍。2018年5月18日被撤销[[太平紳士|太平紳士]]委任。

== 简历 ==
宋林于1963年出生于济南，当时他的父母都在[[山东师范大学|山东师范大学]]教书，1970年宋林被送回老家曲水农村读书。大学时期，他获得[[同济大学|同济大学]][[工程力学|工程力学]]学士學位。

宋林在1985年加入[[華潤集團|華潤集團]]若干旗下企業公司管理層部份，其主要工作職位经理、高级经理、助理总经理、副总经理、董事、企发部总经理，包括[[華潤創業有限公司|華潤創業有限公司]]、[[万科企业股份有限公司|万科企业股份有限公司]]、[[華潤微電子|華潤微電子]]（已私有化）、[[華潤石化|華潤石化]]（已結業後出售給[[中國石化|中國石化]]）、[[华润励致|华润励致]]（[[華潤燃氣|華潤燃氣]]前身）、[[華潤電力控股有限公司|華潤電力控股有限公司]]等旗下企業。也是[[東亞銀行（中國）|東亞銀行（中國）]]，以及[[吉利汽車|吉利汽車]]獨立非常務董事。2004年宋林升任华润集团[[总经理|总经理]]；2008年升任华润集团董事长。在威海当地，宋林被称作“'''胶东宋氏骄子'''”。<ref name="网易">[http://money.163.com/14/0422/02/9QDD24I200253B0H.html 宋林家世起底：出身寒微 妻女早已远走美国]</ref>他在《[[财富|财富]]》2012中国最具影响力的50位商界领袖排行榜中排名第20位。

值得一提的是，他是在2002年收購[[萬佳百貨|萬佳百貨]]（現并入[[華潤萬家|華潤萬家]]）時，同時被委任[[董事長|董事長]]一職。

==落马==
2013年7月17日和2014年4月15日，[[新华社|新华社]]《[[经济参考报|经济参考报]]》首席记者王文志以个人身份，两次向[[中纪委|中纪委]]实名举报宋林在华润收购山西金业资产过程中存在严重的渎职行为，造成巨额国有资产流失，还举报宋林包养情妇并涉嫌贪腐。<ref >{{cite web |url = http://news.163.com/api/14/0415/20/9PT95QPH0001124J.html |language = zh-cn |title=记者再次实名举报华润董事长宋林包养情妇及贪污 |publisher = 扬子晚报 |date=2014-04-15 |accessdate= }}</ref>宋林对此予以否认，认为举报内容纯属捏造和恶意中伤<ref>[http://news.163.com/14/0416/06/9PUBNAKQ00014JB6.html 华润董事长声明否认贪腐 称举报纯属捏造中伤]</ref>。中央纪委、监察部网站于4月17日宣布宋林涉嫌严重违纪违法，目前正接受组织调查。4月19日宋林涉嫌严重违纪违法被免职<ref>{{cite web |url = http://news.163.com/14/0419/10/9Q6GG9I70001124J.html |language = zh-cn |title=华润集团董事长宋林涉嫌严重违纪违法被免职 |publisher=新华网 |date=2014-04-19 |accessdate= }}</ref>。宋林被免職后，[[招商局集团|招商局集团]]董事长[[傅育宁|傅育宁]]接任华润集团董事长一职。

宋林案曝光后，幕后支持他的[[曾庆红|曾庆红]]势力受到重创；此外华润系企业股价遭受全面暴跌。<ref>[http://www.epochtimes.com/gb/14/4/23/n4138564.htm宋林案累华润股价全线暴跌.html?photo=2 华润董事长宋林案累华润股价全线暴跌]  </ref>据传宋林的案件涉及[[贺国强|贺国强]]的家人、长子贺锦涛<ref>[http://www.zaobao.com/special/report/politic/cnpol/story20140420-334483 宋林案 传牵扯贺国强家人]</ref><ref >{{cite web |url = http://china.dwnews.com/news/2014-05-06/59469749.html | language= | title=既得利益集团缩水 中共破官商联袂藩篱 |publisher=多维新闻 |date=2014-05-06 |accessdate= }}</ref>。

另外，有媒体预测中纪委已经将宋林案列为继[[周永康案|周永康案]]之后的第三号大案<ref>[http://www.epochtimes.com/gb/14/4/20/n4135745.htm华润宋林被速免职-背后三“大老虎”呼之欲出.html?p=all 华润宋林被速免职-背后三“大老虎”呼之欲出] </ref>。另据香港《[[苹果日报|苹果日报]]》消息，宋林被爆出豪饮豪食行为。宋林在职期间，常在香港湾仔的华润总部[[宴会厅|宴会厅]]宴请政商权贵，豪吃22头极品干[[鲍鱼|鲍鱼]]、[[苏眉|苏眉]]、[[冬虫草|冬虫草]]炖汤等名贵菜，更豪饮每支约8万至12万[[港元|港元]]的法国[[勃艮第|勃艮第]]顶级红酒，估计每餐最少花费60万元港元。而此宴会厅亦被指是领导设宴的场所，有指费用由华润支付。其净系食物成本价每人最少1,000至2,000港元。<ref>[http://business.sohu.com/20140423/n398709754.shtml 宋林招待权贵极尽奢华:12万元红酒一餐6支]</ref>

2014年5月20日，[[中国人民政治协商会议第十二届全国委员会|中国人民政治协商会议第十二届全国委员会]]第十五次主席会议宣布，宋林不再担任[[第十二届全国政协委员|第十二届全国政协委员]]<ref>[http://news.ifeng.com/a/20140521/40391682_0.shtmlhttp://news.ifeng.com/a/20140521/40391682_0.shtml 原华润集团董事长宋林被撤销全国政协委员资格]</ref>。2015年9月11日，中央纪委监察部网站宣布宋林被开除党籍。<ref>{{cite news|url=http://www.ccdi.gov.cn/xwtt/201509/t20150911_61775.html|title=华润(集团)有限公司原党委书记、董事长宋林严重违纪违法被开除党籍|publisher=中央纪委监察部网站|date=2015-09-11|accessdate=2015-09-11}}</ref>

2017年2月27日，[[广州|广州]]中院开庭审理了宋林贪污、受贿一案，认定他贪污受贿3300多万元。6月1日，广州中院一审宣判，宋林犯贪污罪，判处有期徒刑十年六个月，并处罚金人民币一百五十万元；犯受贿罪，判处有期徒刑十二年，并处罚金人民币二百五十万元。决定执行有期徒刑十四年，并处罚金人民币四百万元；贪污所得财物返还华润（集团）有限公司；受贿所得财物予以没收，上缴国库<ref>{{cite news|url=http://news.youth.cn/jsxw/201706/t20170602_9945124.htm|title=宋林一审被判十四年|publisher=新华社广州6月1日电|date=2017-06-02 |accessdate=2017-06-02 }}</ref>。

== 家庭 ==
* 宋林的祖父解放前在[[地主|地主]]家做[[长工|长工]]，积极参与革命，打[[土豪|土豪]]分田地，后在村大队担任干部。祖父膝下四子，二子早年夭折，剩下长子宋吉清（大学学历）、三子宋吉彬（喜欢务农，和父亲一起供养两兄弟读书）、四子宋吉波（中学学历）。<ref name="网易" />
* 父亲宋吉清，1929年出生，14岁闹革命当了[[儿童团|儿童团]]团长，17岁受父母之命，和年长4岁的同村姑娘结婚，18岁参军，参加[[解放战争|解放战争]]，在莱阳解放中曾上前线作战。解放后，20岁的宋吉清回家念书，在烟台牟平念中学，受到新思想影响，与原配妻子离婚。后来考上山东[[文登|文登]]师范，毕业后留校任教三年，与自己的学生林氏（宋母）结婚，后调任济南教书。他于2002年因病在济南去世。宋家族人说，宋吉清一直希望宋林能进入仕途，当个大官。<ref name="网易" />

宋林目前已婚，他的妻子带着孩子远走[[美国|美国]]，和宋一直处于分居状态。他最后一次回曲水村是2013年[[清明节|清明节]]的前一天，宋林回去在村西头宋家的墓地上祭拜过世的父亲。另外有媒体报道宋林发迹源于有深厚的家庭背景，宋吉彬否认，曲水村多位村民表示宋家祖上三代都是普通人，没有任何背景。<ref name="网易" />
