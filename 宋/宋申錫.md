'''宋申錫'''（{{bd|8世纪||833年|8月18日|}}<ref name=BT17-2>[[:s:舊唐書/卷17下|《旧唐书》卷一十七下]]</ref>），字'''慶臣'''，[[唐朝|唐朝]]官员，在[[唐文宗|唐文宗]]年间短暂担任[[宰相|宰相]]。他意图帮助文宗清除当权[[宦官|宦官]]，反被诬指企图推翻文宗改立其皇弟（漳王[[李凑|李凑]]）而遭贬死。

== 家庭背景 ==
宋申錫生年不详，在《新唐书》宰相世系表中，他的家族被称为[[廣平|廣平]]（今[[河北|河北]][[邯郸|邯郸]]）宋氏，他本人的传则称这一脉的起源已失考。<ref name=NBT152>[[:s:新唐書/卷152|《新唐书》卷一百五十二]]</ref>他的祖父叫宋素，父亲叫宋叔夜。<ref name=BT167>[[:s:舊唐書/卷167|《旧唐书》卷一百六十七]]</ref><ref name=NBT75>{{cite web |url=http://www.sidneyluo.net/a/a17/075.htm |title=存档副本 |accessdate=2010-02-07 |0=2009-12-19 |deadurl=yes |archiveurl=https://web.archive.org/web/20091219203329/http://www.sidneyluo.net/a/a17/075.htm |archivedate=2009-12-19 }}{{cite web |url=http://www.sidneyluo.net/a/a17/table/form92.htm |title=存档副本 |accessdate=2009-11-16 |0=2009-12-20 |deadurl=yes |archiveurl=https://web.archive.org/web/20091220200255/http://www.sidneyluo.net/a/a17/table/form92.htm |archivedate=2009-12-20 }}</ref>堂姑姑叫[[宋若昭|宋若昭]]，曾為其撰寫墓誌銘。

== 早期经历 ==
宋申錫早年丧父，家境贫寒，但文才很好。中[[进士|进士]]后，他在[[秘书省|秘书省]]任[[校书郎|校书郎]]。[[元和_(唐朝)|元和]]十一年（816年），失宠的宰相[[韋貫之|韋貫之]]因反对[[唐宪宗|唐宪宗]]对[[藩镇|藩镇]]作战而被罢相、逐出[[长安|长安]]改任[[湖南觀察使|湖南觀察使]]后，<ref>[[:s:資治通鑑/卷239|《资治通鉴》卷二百三十九]]</ref>邀请宋申錫做他的从事。此后宋申锡就在藩镇工作。<ref name=NBT152/><ref name=BT167/>

== 唐穆宗、唐敬宗年间 ==
[[唐穆宗|唐穆宗]][[长庆|长庆]]（821年 - 824年）初年，宋申锡被时任[[御史中丞|御史中丞]][[牛僧孺|牛僧孺]]拜为[[监察御史|监察御史]]。<ref>[[杜牧|杜牧]]《唐故太子少师奇章郡开国公赠太尉牛公墓志铭（并序）》</ref><ref>[[:s:张彻宋申锡可并监察御史制|白居易《张彻宋申锡可并监察御史制》]]</ref>任上奉诏查验秘书监分司东都[[元锡|元锡]]赃罪，得其罪行属实，使其被贬为[[壁州|壁州]][[刺史|刺史]]。<ref name=CFYG>《[[册府元龟|册府元龟]]》</ref>二年（822年），任起居舍人。四年（824年）十一月，[[淮南节度使|淮南节度使]][[王播|王播]]厚赂权贵要员请求领盐铁使，十二月，时任起居郎的宋申锡与[[谏议大夫|谏议大夫]][[独孤朗|独孤朗]]、[[张仲方|张仲方]]、起居郎[[孔敏行|孔敏行]]、[[柳公权|柳公权]]、[[补阙|补阙]][[韦仁实|韦仁实]]、[[刘敦儒|刘敦儒]]、[[拾遗|拾遗]][[李景让|李景让]]、[[薛廷老|薛廷老]]等伏延英殿上疏反对，请求在延英殿面奏王播奸邪、交结宠幸以求大用。<ref>[[:s:资治通鉴/卷243|《资治通鉴》卷二百四十三]]</ref>但[[唐敬宗|唐敬宗]]年幼不能用其言。<ref>[[:s:旧唐书/卷164|《旧唐书》卷一百六十四]]</ref><ref>[[:s:新唐书/卷167|《新唐书》卷一百六十七]]</ref>[[宝历_(唐朝)|宝历]]二年（826年），宋申锡累转禮部員外郎，很快又任翰林侍講學士。<ref name=BT17-1>[[:s:舊唐書/卷17上|《旧唐书》卷一十七上]]</ref>宋申锡行事谨慎，不结党，与朝廷官员间激烈的党争形成对比，时人认为他的升官是为了让别人学习。<ref name=NBT152/><ref name=BT167/>

== 唐文宗年间 ==
十二月，敬宗被弑，弟唐文宗继位。<ref>[[:s:資治通鑑/卷243|《资治通鉴》卷二百四十三]]</ref>宋申锡充书诏学士。<ref name=BT17-1/>任戶部郎中，知制诰。[[太和_(唐朝)|太和]]二年（828年），任[[中書舍人|中書舍人]]、朝议郎、行尚书礼部员外郎、充[[翰林学士|翰林学士]]、[[上柱国|上柱国]]，赐紫金鱼袋。<ref name=NBT152/><ref name=BT167/>推荐和自己交好的[[李让夷|李让夷]]为[[右拾遗|右拾遗]]。<ref>[[:s:新唐書/卷181|《新唐书》卷一百八十一]]</ref>并奉敕为[[义成军节度使|义成军节度使]]凉国公[[李听|李听]]撰写德政碑。<ref>[[:s:赐义成军节度使凉国公李听立德政碑敕|:s:赐义成军节度使凉国公李听立德政碑敕]]</ref><ref>[[:s:赐李听敕|:s:赐李听敕]]</ref><ref>[[:s:义成军节度郑滑颍等州观察处置等使金紫光禄大夫检校司徒使持节滑州诸军事兼滑州刺史御史大夫上柱国陇西县开国公食邑一千八百户李公德政碑铭（并序）|:s:义成军节度郑滑颍等州观察处置等使金紫光禄大夫检校司徒使持节滑州诸军事兼滑州刺史御史大夫上柱国陇西县开国公食邑一千八百户李公德政碑铭（并序）]]</ref>后被授守尚书户部侍郎知制诰，充翰林学士。<ref>[[:s:授学士王源中等中书舍人制|李虞仲《授学士王源中等中书舍人制》]]</ref>

当时，文宗忌惮当权宦官，认为是宦官害死了他的祖父宪宗和兄长敬宗。掌握[[神策军|神策军]]的神策中尉[[王守澄|王守澄]]就是一个有权势的大宦官，领禁兵，尤其跋扈，他的门人[[郑注|郑注]]公开受贿，令文宗很不快。宋申锡作为学士，常在内廷，文宗察觉宋申锡忠厚，认为他沉厚有方略，可以和宋申锡商量，并借机私下命他与外廷朝臣合谋夺王守澄的权，并许诺封他为宰相，宋申锡顿首致谢。<ref name=BT169>[[:s:旧唐书/卷169|《旧唐书》卷一百六十九]]</ref><ref name=BT175>[[:s:旧唐书/卷175|《旧唐书》卷一百七十五]]</ref>四年（830年）六月，宋申锡拜右丞，七月以朝议郎、守尚书右丞、上柱国、赐紫金鱼袋的身份被任为正议大夫、行尚书右丞、同中书门下平章事，为实质宰相。<ref name=BT17-2/><ref name=BT184>[[:s:旧唐书/卷184|《旧唐书》卷一百八十四]]</ref><ref name=NBT8>[[:s:新唐书/卷8|《新唐书》卷八]]</ref><ref name=NBT82>[[:s:新唐书/卷82|《新唐书》卷八十二]]</ref><ref name=NBT208>[[:s:新唐書/卷208|《新唐书》卷二百零八]]</ref><ref name=ZZTJ244>[[:s:資治通鑑/卷244|《资治通鉴》卷二百四十四]]</ref><ref>[[:s:授宋申锡行尚书右丞平章事制|:s:授宋申锡行尚书右丞平章事制]]</ref>由于他的良好声誉，很多人希望他拜相，但他到任后，只是循例工作，能力其实不足与声望匹配。<ref name=NBT152/><ref name=BT167/>

作为清除当权宦官计划的一部分，七月，宋申锡推荐朋友[[王璠_(唐朝)|王璠]]任[[京兆尹|京兆尹]]兼御史大夫，并秘密向王璠透露了文宗的密旨，授意王璠捕杀郑注。<ref>《新唐书·王璠传》《资治通鉴》都称御史中丞[[宇文鼎|宇文鼎]]与宋申锡同受文宗密旨及指使王璠捕杀郑注。但宇文鼎升御史中丞在太和三年（829年），非宋申锡荐引，且其为御史中丞时为王璠下级，且在后来的宋申锡冤案中未被牵连，可见其与宋申锡同受密旨的记载尚存疑，且未必参与宋申锡、王璠的谋划。详见[[卞孝萱|卞孝萱]]《[http://www.doc88.com/p-5823921595981.html 唐宋申锡冤案研究]》。</ref>但王璠秘密把堂帖给王守澄看，于是泄密，使王守澄和郑注有所警觉。<ref name=NBT208/><ref>[[:s:新唐書/卷179|《新唐书》卷一百七十九]]</ref><ref name=ZZTJ245>[[:s:資治通鑑/卷245|《资治通鉴》卷二百四十五]]</ref>因漳王李凑贤明在朝内外有人望，郑注指使神策军同正将兼衙前虞候[[豆盧著|豆盧著]]诬指宋申锡和李凑图谋推翻文宗，立李凑为帝，十六宅宫市典晏敬则、朱训与宋申锡亲吏王师文同谋，宋申锡秘密给李凑送金币，晏敬则于十六宅拿出漳王吴绫汗衫一领、熟线绫一匹以答宋申锡，<ref name=BT175/><ref name=BT184/>意在让宋申锡获罪被族诛。<ref name=NBT82/>五年（831年）二月，当王守澄于浴堂将豆盧著的弹劾报告给文宗时，文宗震惊了，信以为真，下令调查。<ref name=BT169/>王守澄希望派二百骑神策军去靖恭里宋宅立刻将宋申锡族灭，但被一同入见的内官[[馬存亮|馬存亮]]制止：“谋反者只是宋申锡，为什么不召南司会议，现在突然如此，京师要乱了。”王守澄只得作罢。文宗也立刻下令召集所有的宰相来[[延英殿|延英殿]]。<ref name=ZZTJ244/>当宋申锡、[[路隨|路隨]]、[[李宗闵|李宗闵]]、牛僧孺四位原本都在私宅休假的宰相都来到中书省东门时，东门的宦官说宋申锡不在被召之列。路随等愕然地互相看，宋申锡便知道自己一定是获罪了，望着延英殿用笏板叩头退回家待罪。他怡然不以为意，回府后停在外舍不入内，素服待罪。夫人说：“你是宰相，位极人臣，为什么要背叛天子而谋反呢？”宋申锡答：“我本是孤儿，承蒙皇上厚恩，当上宰相，不能锄奸臣乱党，反被罗织罪名陷害，夫人看我宋申锡像是谋反的人吗？”于是夫妇一同哭泣。<ref name=NBT152/><ref name=BT167/>

其余三位宰相在延英殿见了文宗，看了王守澄的奏报，也震惊了，面面相觑，一言不发。同时，文宗命王守澄逮捕晏敬则、王师文。神策军在宋宅捕到孔目官张全真、家人买子缘信等，又于十六宅及市肆追捕胥吏。<ref name=BT167/>晏敬則被抓，王師文逃走。<ref name=ZZTJ244/>朱训被下神策军黄门狱，刑讯逼供得状称宋申锡多年来对他说皇帝多病、太子年幼，其次当立漳王等，及去年十二月、闰十二月各人商议谋反事。<ref name=NBT152/><ref name=CFYG/><ref name=BT175/><ref name=NBT82/>

2天后，宋申锡被贬为太子右庶子。<ref name=NBT8/><ref>当时并没有太子，这只是名誉职位。</ref>没人敢公开说他是被诬告的，仆射[[窦易直|窦易直]]说宋申锡当诛，闻者愕然。但京兆尹[[崔琯|崔琯]]、大理卿[[王正雅|王正雅]]指出王师文尚在逃，奏请由外廷审讯宋申锡、豆卢著，<ref name=BT175/><ref name=NBT82/><ref>[[:s:新唐书/卷143|《新唐书》卷一百四十三]]</ref><ref name=NBT182>[[:s:新唐書/卷182|《新唐书》卷一百八十二]]</ref>才使事件没有进一步升级。<ref name=NBT152/><ref name=BT167/>经过审讯，晏敬則称宋申锡事先派王师文联络漳王，商议谋反。<ref name=ZZTJ244/>

文宗召师保、仆射、尚书丞郎、常侍、给事、谏议、舍人、御史中丞、京兆尹、大理卿于中书省及集贤院审验此事。又2天后，文宗召集宰相、议事官于延英殿询问意见，并召三师议论如何处刑。左常侍[[崔玄亮|崔玄亮]]与给事中[[李固言|李固言]]、谏议大夫[[王质_(唐朝)|王質]]、补阙[[卢钧|卢钧]]、[[舒元褒|舒元褒]]、[[罗泰|罗泰]]、[[蔣係|蔣係]]、[[裴休|裴休]]、[[窦宗直|窦宗直]]、[[韋溫_(宣歙觀察使)|韋溫]]、拾遗[[李群|李群]]、[[韦端符|韦端符]]、[[丁居晦|丁居晦]]、[[袁都|袁都]]都伏在玉阶下乞求文宗重新和宰相商议，<ref name=NBT182/><ref name=BT149>[[:s:旧唐书/卷149|《旧唐书》卷一百四十九]]</ref><ref>[[:s:旧唐书/卷173|《旧唐书》卷一百七十三]]</ref><ref name=NBT132>[[:s:新唐书/卷132|《新唐书》卷一百三十二]]</ref>并由适当的部门来调查，而非在宫中问讯。韦温说：“宋公履行有素，身居台辅，不当有此，是奸人陷害。吾辈谏官，岂避一时之雷电，而让圣君贤相陷于被蒙蔽迷惑之罪？”<ref name=BT168>[[:s:旧唐书/卷168|《旧唐书》卷一百六十八]]</ref><ref name=NBT169>[[:s:新唐书/卷169|《新唐书》卷一百六十九]]</ref>文宗以已和公卿大官商议过为由拒绝，并因震怒四次叱退谏官。崔玄亮下跪，哭着说：“处决一个百姓要谨慎，处决一个宰相更要谨慎。我不是为宋申锡说话，是为陛下可惜天下之法。”<ref name=BT165>[[:s:旧唐书/卷165|《旧唐书》卷一百六十五]]</ref><ref name=NBT164>[[:s:新唐书/卷164|《新唐书》卷一百六十四]]</ref>文宗有所感悟，怒气稍解，重新召集宰相。牛僧孺说：“位极人臣无非宰相，宋申锡已经是宰相，就算谋反成功，他仍然只能是宰相，他谋反图什么呢？他肯定没有谋反。”起初，京师都以为宰相真的联同亲王谋逆了，三四天后才知道是诬陷，人士都怨恨王守澄、郑注，于是谏官们才号泣相争。郑注本来希望处死宋申锡，担心宰相的意见胜出，而再次调查会让真相大白，于是建议王守澄不再坚持处死宋申锡，改为贬到岭外，<ref name=BT17-2/><ref name=NBT152/><ref name=BT167/><ref name=BT175/><ref name=ZZTJ244/>及请求下诏贬李凑。<ref name=NBT82/>

第二天，李凑被贬为巢县公，<ref name=BT175/>宋申锡贬为[[开州_(唐朝)|开州]]（今[[重庆|重庆]]）司马同正。<ref name=BT17-2/><ref name=BT184/><ref>[[:s:舊唐書/卷36|《旧唐书》卷三十六]]</ref>这是王守澄的意思，文宗不敢违。<ref>[[:s:新唐書/卷178|《新唐书》卷一百七十八]]</ref>宋申锡任相期间，恨要员受贿，拒绝了全国各地的贿赂，当他被抄家时，能抄到的只有他收到和拒绝贿赂的文书，晏敬则等被处死及一同被流放而死的有上百人，<ref name=ZZTJ244/>李凑傅母[[杜仲阳|杜仲阳]]被流放[[金陵|金陵]]。<ref name=ZZTJ245/>朝野都为宋申锡被贬而叹息，以为冤枉。<ref name=BT167/>豆卢著则以右神策军散兵马使、检校太子宾客被提拔为守本官兼[[殿中侍御史|殿中侍御史]]。<ref name=NBT152/>

因力争救下宋申锡，天下称崔琯贤，蒋系等被时人所称道，<ref name=BT149/><ref name=NBT132/>王正雅也被时人推重，崔玄亮也因坚持而在朝堂得到名声，<ref name=BT165/>卢钧、韦温也由此显名。<ref name=NBT182/><ref name=BT168/><ref name=NBT169/><ref>[[:s:旧唐书/卷177|《旧唐书》卷一百七十七]]</ref>而王质则为宦官所恨，被宰相出为[[虢州|虢州]]刺史。<ref name=NBT164/><ref>[[:s:旧唐书/卷163|《旧唐书》卷一百六十三]]</ref>崔玄亮也因害怕宦官报复，当天就主动请辞致仕。<ref name=ZZTJ244/><ref>[[白居易|白居易]]《唐故虢州刺史赠礼部尚书崔公墓志铭（并序）》</ref>

六年（832年）六月，天久旱，文宗下诏求下雨的办法，司门员外郎[[李中敏|李中敏]]上言：“天旱不是圣德不至，而是因为宋申锡蒙冤，郑注行奸。现在下雨的办法，莫如斩郑注而昭雪申锡。”指出宋申锡不受贿，遭奸人诬陷，天旱是上帝的灵验，天下人都知道宋申锡是冤枉的。士大夫都觉得此举危险，此奏疏也被留中。李中敏也在次年称病归[[洛阳|洛阳]]。<ref name=ZZTJ245/><ref>[[:s:旧唐书/卷171|《旧唐书》卷一百七十一]]</ref><ref>[[:s:新唐书/卷118|《新唐书》卷一百一十八]]</ref><ref>[[:s:太和六年大旱上言|:s:太和六年大旱上言]]</ref>文宗也很厌恶郑注诬陷宋申锡，想令京兆尹杖杀他，但正逢郑注的药对文宗的[[中风|中风]]病有了效果，文宗开始善待郑注。<ref>[[:s:旧唐书/卷174|《旧唐书》卷一百七十四]]</ref>后来文宗也下诏赦免宋申锡余党，只捕捉王师文一人。<ref>[[:s:赦宋申锡馀党诏|:s:赦宋申锡馀党诏]]</ref>

宋申锡被终身禁止返回长安。七年（833年），他在开州任上去世，<ref name=ZZTJ244/>文宗允许把他的尸体运回长安安葬。宋申锡获罪后，宦官愈发强横，文宗表面上包容，其实不能忍受。九年（835年）五月，文宗下诏称西川监军[[杨承和|杨承和]]庇护宋申锡，下令安置杨承和于[[驩州|驩州]]，令西川军用囚车押送之，不久[[赐死|赐死]]。[[开成|开成]]元年（836年）九月时，因改变立场也开始帮助文宗对付当权宦官的郑注已在[[甘露之变|甘露之变]]中被杀，文宗从容对宰相们说：“朝廷岂有遗事乎？”宰相[[李石_(唐朝)|李石]]在延英殿进言宋申锡忠直被贬死，蒙冤未雪，其他宰相[[郑覃|郑覃]]、李固言也都回答宋申锡之冤，文宗也表示自己被骗了，为奸人所逼，俯首哭泣，数行眼泪流下：“当时兄弟都保不住，何况申锡？内臣外廷都有为此推波助澜的，是朕不明，假如遇到[[汉昭帝|汉昭帝]]，必无此冤情。有司为我褒显他。”又说：“[[李德裕|德裕]]也是申锡一类。”起用被郑注诬陷而被贬的[[太子宾客|太子宾客]]分司东都[[李德裕|李德裕]]为[[浙西观察使|浙西观察使]]。<ref>[[:s:新唐书/卷180|《新唐书》卷一百八十]]</ref>于是宋申锡得以平反，追复正议大夫、尚书右丞（或误作左丞）、同中书门下平章事、上柱国，赐紫，赠兵部尚书，谥号'''文懿'''。他的儿子[[宋慎微|宋慎微]]被任命为[[城固县尉|城固县尉]]。<ref name=BT17-2/><ref name=BT167/><ref name=ZZTJ245/>开成年间，文宗还在麟德殿感慨宋申锡堪任御史中丞，可惜了。<ref>[[钱易|钱易]]《[[南部新书|南部新书]]》</ref>[[唐武宗|唐武宗]][[会昌_(唐朝)|会昌]]元年（841年）三月，宰相李德裕、[[崔珙|崔珙]]、[[崔郸|崔郸]]、[[陈夷行|陈夷行]]邀[[枢密使|枢密使]]至中书省一同上奏，引宋申锡等案例阻止武宗处决前宰相[[杨嗣复|杨嗣复]]、[[李珏|李珏]]。<ref>[[:s:資治通鑑/卷246|《资治通鉴》卷二百四十六]]</ref><ref>[[:s:论救杨嗣复李珏陈夷直状|:s:论救杨嗣复李珏陈夷直状]]</ref><ref>[[:s:第三状|:s:第三状]]</ref>二年（842年），赐谥'''贞'''，<ref name=NBT152/>一作'''穆'''。<ref name=CFYG/>

[[许浑|许浑]]有诗《太和初靖恭里感事》《闻开江宋相公申锡下世二首》，即叹宋申锡事。

后来[[宋朝|宋朝]]殿中侍御史[[段少连|段少连]]曾上疏[[宋仁宗|宋仁宗]]引用崔元亮叩殿陛理宋申锡事为美事。<ref>[[:s:宋史/卷297|《宋史》卷二百九十七]]</ref>

== 评价 ==
*《旧唐书》
**史臣曰：'''申锡'''小器大谋，贬死为幸。
**赞曰：[[李程|（李）程]]、'''（宋申）锡'''弼谐，于道难周。<ref name=BT167/>
*《新唐书》赞曰：'''申锡'''谋小任大，颠沛从之，惜乎！<ref name=NBT152/>
*《资治通鉴》：文宗深愤其然，志欲除之（宦官势力），以'''宋申锡'''之贤，犹不能有所为，反受其殃。<ref>[[:s:資治通鑑/卷263|《资治通鉴》卷二百六十三]]</ref>

== 子孙 ==
据《新唐书卷七十五‧表第十五[[新唐書宰相世系表|宰相世系]]》，除了宋慎微外，宋申锡还有长子宋球，宋球的儿子叫宋绚，字韬文。

== 轶闻 ==
[[卢肇|卢肇]]《[[逸史|逸史]]》称宋申锡死后托梦于其妻，称已诉冤于上帝，王璠必遭腰斩，一同被杀的还有数人，都埋在城外。历史上王璠亦死于甘露之变。
