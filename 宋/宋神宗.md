'''宋神宗赵顼'''（{{bd|1048年|5月25日|1085年|4月1日|catIdx=Z趙}}），本名'''趙仲鍼'''，[[宋英宗|宋英宗]]的长子，[[北宋|北宋]]第六代皇帝，1067年1月25日-1085年4月1日在位。

== 早年 ==
宋仁宗慶曆八年（1048年）四月十日，出生在濮安懿王宮邸睦親宅。宋英宗趙曙和宣仁聖烈皇后高氏所生長子。八月仁宗賜名為仲針，後被授為率府副率，後三次升遷至右千牛衛將軍。 

嘉佑八年（1063年），趙頊與其父趙曙一起入居慶甯宮。三月二九日仁宗趙禎逝世，趙頊父趙曙即位，授趙頊為安州觀察使，封光國公。五月趙頊受經於東宮。宋英宗趙曙見了他如此好學常場派遣內侍去阻止他。侍講王陶進入宮內，趙頊率弟弟趙顥向他參拜，可見對師傅的尊重。九月加封忠武軍節度使、同中書門下平章事，封淮陽郡王改名為趙頊。  

治平元年（1064年），進封潁王。治平三年（1066年）三月納故相向敏中的孫女為夫人。十月英宗病重，趙頊按照宋仁宗時舊制，請求兩日一到邇英閣講讀，以安朝廷百官之心。十二月趙頊被立為皇太子。趙頊太子時喜讀《韓非子》，對法家「富國強兵」之術有趣；還讀過王安石的《上仁宗皇帝言事書》，讚賞王安石的理財治國思想。

== 朝政 ==
=== 熙寧變法與新舊黨爭 ===
神宗即位後，對北宋積貧積弱深感憂心，而他素來都欣賞[[王安石|王安石]]的才幹，故即位後命王安石推行變法，振興北宋王朝，是為王安石变法，又稱[[熙寧變法|熙寧變法]]。

在王安石的主持下，[[均输法|均输]]、[[青苗法|青苗]]、[[农田水利法|农田水利]]、[[免役法|免役]]、[[市易法|市易]]、[[保甲法|保甲]]、[[方田均税法|方田均税]]、[[保马法|保马]]等新法相继出籠。新法几乎涵盖社会的各个方面，惟操之過急，利弊互見。北宋学者[[陆佃|陆佃]]说：“造元丰间，积票塞上，盖数千万石，而四方常平之钱，不可胜计。”當時垦田面积大幅度增加，全国高达7億畝，城镇商品经济取得了空前发展。但是变法受到守旧派激烈的反對，朝中的[[司马光|司马光]]、[[范镇|范镇]]、[[赵瞻|赵瞻]]纷纷上书陈述对新法的不滿，司马光与[[呂惠卿|吕惠卿]]为了[[青苗法|青苗法]]在皇帝面前争辩，新法維持了将近二十年，直到[[司馬光|司馬光]]盡罷新法為止。此一時期，面临朝廷和后宫的双重阻力，神宗受到的打擊可想而知，[[高滔滔|高太后]]更是对神宗说：“王安石是在变乱天下呀！”岐王[[赵颢|赵颢]]也从旁劝说神宗应该遵从皇太后的懿旨，神宗心烦意乱，怒斥歧王說：「那你來當皇帝好了。」岐王诚惶诚恐，失声痛哭。

=== 烏臺詩案 ===
發生於宋神宗[[元豐|元豐]]二年（1079年），[[蘇軾|蘇軾]]於當年移知[[湖州|湖州]]，到任後上表謝恩，朝臣以其上表中用語，暗藏譏刺。御史[[何正臣|何正臣]]上表彈劾蘇軾，指其「愚弄朝廷，妄自尊大」，又以蘇軾動輒歸咎新法，要求朝廷明正刑賞。御史[[李定|李定]]曾因不服母孝，受蘇軾譏諷，於此案中也指蘇軾有「悛終不悔，其惡已著」、「傲悖之語，日聞中外」、「言偽而辯，行偽而堅」、「怨己不用」等四大可廢之罪。

御史[[舒亶|舒亶]]尋摘蘇軾詩句，指其心懷不軌，譏諷神宗青苗法、助役法、明法科、興水利、鹽禁等政策。神宗下令拘捕，太常博士[[皇甫遵|皇甫遵]]奉令前往逮人。[[蘇轍|蘇轍]]時在商丘已預知消息，託[[王適|王適]]協助安置蘇軾家屬，並上書神宗陳情，願以官職贖兄長之罪。
蘇軾在9月被捕後，寫信給蘇轍交代身後之事，長子[[蘇邁|蘇邁]]則隨途照顧。押解至太湖，蘇軾曾意圖自盡，幾經掙扎，終未成舉。捕至御史臺獄下，御史臺依平日書信詩文往來，構陷牽連七十餘人。

後因[[慈聖皇后|太皇太后曹氏]]、[[王安禮|王安禮]]等人出面力挽，前宰相[[王安石|王安石]]也說：「豈有聖世而殺才士者乎？」蘇軾終免一死，貶謫為「檢校尚書水部員外郎黃州團練副使本州安置」，前往[[黄州_(隋朝)|黃州]]。蘇轍被貶江西[[筠州_(唐朝)|筠州]]任酒監，平日與[[蘇軾|蘇軾]]往來者，如[[曾鞏|曾鞏]]、[[李清臣|李清臣]]、[[張方平|張方平]]、[[黃庭堅|黃庭堅]]、[[范鎮|范鎮]]、[[司馬光|司馬光]]等29人亦遭處分。[[張方平|張方平]]、[[司馬光|司馬光]]和[[范鎮|范鎮]]罰紅銅三十斤，其餘各罰紅銅二十斤。[[烏臺詩案|烏臺詩案]]於十二月結束。

=== 與西夏的關係 ===
時值[[李秉常|夏惠宗]]在位，母黨[[梁皇后_(西夏毅宗)|梁氏]]專權，[[西夏|西夏]]國勢日非，圖一舉殲滅羌夏。[[王韶_(宋朝)|王韶]]在庆州（今[[甘肃|甘肃]][[庆阳|庆阳]]）大破夏军，占领西夏二千里土地。不过后来在[[永乐城之战|永乐城之战]]中惨败，“厥后兵不敢用于北，而稍试于西，灵武之役，丧师覆将，涂炭百万。帝中夜得报，起，环榻行，彻旦不寐。”灭夏之举未能实现。事後，宋神宗在朝中當眾痛哭。<ref>《邵氏聞見錄‧卷第五》載，元丰五年噩耗在深夜傳回京裏的。</ref>他有抱負，勵精圖治，想滅西羌，惜壯志未酬，於[[元豐_(宋神宗)|元豐]]八年抱憾而歿，享年38歲。其子[[宋哲宗|宋哲宗]]親政後，竭盡所能完成父親遺志。

== 因病去世 ==

元丰八年三月初五日（1085年4月1日），宋神宗在福宁殿去世，享年38岁，殡于殿西阶，庙号神宗，群臣上谥号为「英文烈武'''圣孝皇帝'''」，十月二十四日，葬于永裕陵。绍圣二年（1095年）九月，加谥为「绍天法古运德建功英文烈武钦仁'''圣孝皇帝'''」。崇宁三年（1104年）十一月，改谥为「体天显道帝德王功英文烈武钦仁'''圣孝皇帝'''」。政和三年（1113年）十一月，加谥为「体元显道法古立宪帝德王功英文烈武钦仁'''圣孝皇帝'''」。

== 評價 ==
===正面评价===
* [[元朝|元朝]]官修[[正史|正史]]《[[宋史|宋史]]》[[脱脱|脱脱]]等的評價是：“帝天性孝友，其入事两宫，必侍立终日，虽寒暑不变。尝与岐、嘉二王读书东宫，侍讲王陶讲谕经史，辄相率拜之，由是中外翕然称贤。其即位也，小心谦抑，敬畏辅相，求直言，察民隐，恤孤独，养耆老，振匮乏。不治宫室，不事游幸，历精图治，将大有为。未几，王安石入相。安石为人，悻悻自信，知祖宗志吞幽蓟、灵武，而数败兵，帝奋然将雪数世之耻，未有所当，遂以偏见曲学起而乘之。青苗、保甲、均输、市易、水利之法既立，而天下汹汹骚动，恸哭流涕者接踵而至。帝终不觉悟，方断然废逐元老，摈斥谏士，行之不疑。卒致祖宗之良法美意，变坏几尽。自是邪佞日进，人心日离，祸乱日起。惜哉！”<ref>
[http://www.guoxue123.com/shibu/0101/00songs/015.htm 《宋史·神宗本紀》]</ref>

* 重用名將[[王韶|王韶]]，發動『[[熙河之役|熙河之役]]』，率軍擊潰[[羌|羌]]人和[[西夏|西夏]]的軍隊，置[[熙州|熙州]](今甘肅臨洮)，收復河、洮、岷、宕、亹五州，對西夏形成包圍的之勢。熙宁五年（1072年）王韶收復今臨洮與臨夏，升镇洮军为熙州，設熙河路。王韶为龙图阁待制、熙河路馬步軍都总管、经略安抚使兼知熙州<ref>《宋会要辑稿》蕃夷六</ref>。熙宁六年（1073年）、夏天率兵攻占武胜城（今甘肃临洮），乘胜追擊，进攻河州（今甘肃东乡西南），直捣定羌城 （今甘肃广河）。熙寧七年，收回被[[吐蕃|吐蕃]]侵略的二十萬平方公里故土，以功迁礼部司郎中、枢密院直学士<ref>《東都事略》卷八十二</ref>，史稱：「宋幾振矣！」。[[熙宁|熙宁]]八年（1075年）宋廷在[[熙州|熙州]]（今甘肃[[临洮|临洮]]）、[[河州|河州]]（今甘肃[[临夏|临夏]]）、[[洮州|洮州]]（今甘肃[[临潭|临潭]]）、[[岷州|岷州]]（治所今甘肃[[西和|西和]]）、[[永宁寨|永宁寨]]（今甘肃[[甘谷|甘谷]]）等地设州、买马，进行民族贸易，此舉受到了邊境各族的热烈欢迎，史載“熙河人情甚喜”<ref>《续资治通鉴长编》卷241，宋神宗熙宁五年（1072年）十二月丁酉条。</ref>。朝廷以[[王韶|王韶]]为经略安抚使，“通远军自置市易司以来，收本息钱五十七万余缗”<ref>《续资治通鉴长编》卷250，宋神宗熙宁七年（1074年）二月庚辰条。</ref>，[[吐蕃|吐蕃]]政权逐渐瓦解，被稱為『[[熙河開邊|熙河開邊]]』。

===负面评价===
* 明末清初思想家[[王夫之|王夫之]]《[[宋論|宋論]]》稱“宋政之乱，自神宗始”<ref>《宋论》卷六《神宗一》</ref>。

* [[烏臺詩案|烏臺詩案]]是宋神宗執政生涯中，最大的政治汙點。因[[蘇軾|蘇軾]]當年移知湖州，到任後上表謝恩，卻被宰相[[王珪|王珪]]等朝臣認為有貶新政與朝廷之意，御史[[李定|李定]]和[[舒亶|舒亶]]尋摘[[蘇軾|蘇軾]]詩句，指其心懷不軌，譏諷神宗青苗法、助役法、明法科、興水利、鹽禁等政策。神宗下令拘捕，太常博士[[皇甫遵|皇甫遵]]奉令前往逮人。捕至御史臺獄下，御史臺依平日書信詩文往來，構陷牽連七十餘人，引起朝野不小震動。在當時[[蘇軾|蘇軾]]已成為繼[[歐陽脩|歐陽脩]]之後的宋朝文壇泰斗，卻因為一句詩下獄，讓時人議論紛紛，而宰相[[王珪|王珪]]等朝臣又想置[[蘇軾|蘇軾]]於死，讓時人對朝局頗為不滿，其中神宗祖母太皇太后曹氏對此事有所發言，認為[[蘇軾|蘇軾]]是先帝[[宋仁宗|宋仁宗]]所選定的太平宰相。而曾任神宗宰相的大臣[[王安石|王安石]]則向神宗上書，認為：「豈有聖世而殺才士者乎？」神宗迫於壓力，在查清[[蘇軾|蘇軾]]並未有詆毀朝廷之意，將其貶至黃州，但已為神宗的執政留下汙點。

* [[永樂城之戰|永樂城之戰]]是宋神宗執政以來，最大規模的征伐行動，也是神宗執政以來，最大的政治挫折。在當時因[[西夏|西夏]][[外戚|外戚]][[梁太后|梁太后]]與其弟[[梁乙埋|梁乙埋]]姐弟當權，國勢衰落，政治腐敗，[[西夏|西夏]]舉國上下怨聲載道，民不聊生。梁太后多次出兵攻宋，想提高國內[[政治|政治]]威望，卻都慘敗而歸。神宗認為[[西夏|西夏]]無理，下令攻滅[[西夏|西夏]]。宋軍於[[元豐|元豐]]四年（1081年）11月在慶州（今[[甘肅|甘肅]][[慶陽|慶陽]]）擊潰夏軍，占領西夏兩千多里土地。神宗大喜，命給事中[[徐禧|徐禧]]、鄜延道總管[[种諤|种諤]]於元豐五年（1082年）9月帶兵攻夏，準備一舉滅夏。[[徐禧|徐禧]]為了建戰功，拒聽[[种諤|种諤]]的建議([[种諤|种諤]]反對興建[[永樂城|永樂城]])，築[[永樂城|永樂城]]，與[[种諤|种諤]]發生內鬨。由於[[徐禧|徐禧]]好大喜功，聽不進諫言，導致宋軍內鬨，[[永樂城之戰|永樂城之戰]]大敗，宋軍損失二十萬軍士，[[徐禧|徐禧]]、[[李舜舉|李舜舉]]、[[李稷|李稷]]、[[高永能|高永能]]等人死難，被宋人批評為「永樂之恥」。經過[[永樂城之戰|永樂城之戰]]的戰敗，宋神宗開始悔悟，而不再輕言開戰。

==衍生文化==
历史穿越小说《[[新宋|新宋]]》主人公石越便是被安排穿越到到神宗的时代。

== 趣闻轶事 ==
宋神宗赵顼被《时代》杂志认为是有史以来第三富有的人，也是最有权势的人物之一。尽管他在位仅18年，去世时才30多岁，但积累的财富不容小觑。在很强的科技创新和税收能力的帮助下，宋神宗在位期间的北宋国内生产总值占到全世界的25%至30%。
