'''宋卫平'''（{{bd|1957年||catIdx=S}}），[[浙江|浙江]][[绍兴|绍兴]]人，[[中国|中国]][[企业家|企业家]]和房地产商。

==生平==
1957年，出生于[[浙江省|浙江省]][[绍兴市|绍兴市]][[嵊县|嵊县]]。1982年，毕业于[[杭州大学|杭州大学]]（今[[浙江大学|浙江大学]]）历史系。1982年至1987年，在某学校教书。1987年，辞职下海，就职于[[珠海市|珠海市]]某计算机企业担任[[秘书|秘书]]。1994年，从[[广东|广东]]回到[[杭州|杭州]]，投资15万元[[人民币|人民币]]创建[[中国绿城集团|中国绿城集团]]，总部在杭州。

宋卫平并热爱[[足球|足球]]运动，亦投资球赛球队，其创建了[[浙江绿城足球俱乐部|浙江绿城足球俱乐部]]。

2007年，宋卫平夫妇在[[福布斯|福布斯]]中国大陆富豪榜中位列第41位，身价97.5亿人民币。2006年位列第27位。

2013年以12.2亿美元身家位居胡润中国富豪榜第212名。<ref>{{cite web |url=http://hurun.net/EN/HuList.aspx |title=China Rich List 2013 |accessdate=4 September 2014 |publisher=Hurun Report |deadurl=yes |archiveurl=https://web.archive.org/web/20140831000018/http://hurun.net/EN/HuList.aspx |archivedate=2014年8月31日 }}</ref>
