'''宋慈'''（{{bd|1186年||1249年|catIdx=S宋}}），字'''惠父'''，[[南宋|南宋]][[建宁府|建宁府]][[建陽|建陽]]（今[[福建省|福建省]][[南平市|南平市]][[建阳区|建阳区]]）人，官至[[朝议大夫|朝议大夫]]、[[直焕章阁|直焕章阁]]、知[[广州|广州]]、充[[广南东路|广南东路]][[经略安抚使|经略安抚使]]、[[马步军|马步军]][[都总管|都总管]]，著有《[[洗冤集录|洗冤集录]]》，是世界历史上第一本以死亡方式系统编辑的法医学著作。

== 生平简历 ==

少时受学于[[朱熹|朱熹]]弟子同乡[[吴雉|吴雉]]<ref>《[[宋经略墓志铭|宋经略墓志铭]]》：“公少耸秀轩豁，师事考亭高第吴公雉。”</ref>，后入[[太学|太学]]师事[[太学博士|太学博士]]<ref>《[[宋史|宋史]]·真德秀传》：“召为太学正，嘉定元年迁博士。”</ref>[[真德秀|真德秀]]<ref>《宋经略墓志铭》：“暨入太学，西山真公德秀衡其文，见谓有源流出肺腑，公因受学其门。”</ref>。[[宋宁宗|宋宁宗]][[嘉定|嘉定]]十年（1217年），中[[進士|進士]][[乙科|乙科]]，授[[鄞县|鄞县]][[县尉|县尉]]（掌管治安，从九品），因父喪而未赴任<ref>《宋经略墓志铭》：“丁丑，南宫奏赋第三，中乙科。调鄞尉，未上，丁外艰。”</ref>。服丧期满，授[[信丰|信丰]][[主薄|主薄]]（掌管财务，从九品，位在县尉之上）。先后知[[长汀|长汀]]、[[通判|通判]]（掌管财赋）[[邵武軍|邵武軍]]，又通判[[南剑州|南剑州]]，辞官不就，改差干办诸军粮料院<ref>《宋经略墓志铭》：“再调信丰簿。……辟知长汀县。通判邵武军，摄郡有遗爱。通判南剑州，不就。杭相李公宗勉擢贰天府，除诸军粮料院。”</ref>（掌管军队粮草分拨，号为储才之地）。后知常州，赈济饥荒有功，进一官<ref>《[[续资治通鉴|续资治通鉴]]》：“（宋理宗淳祐元年）七月甲辰，以知婺州赵与懃、常州宋慈、江阴军尹焕、广德军康植济籴有劳，各进一秩。”</ref>。以[[司农丞|司农丞]]（[[阶官|阶官]]，正八品）知[[赣州|赣州]]，后被弹劾免职<ref>《宋经略墓志铭》：“擢司农丞，知赣州。当路以要官勾致，公不答。迁劾，免。”</ref>。[[起家|起家]]知[[蕲州|蕲州]]，赴任途中改任提点[[广南东路|广南东路]][[刑狱|刑狱]]公事。广东官吏多不奉法，案犯羁押数年都不能处理，宋慈到任八月，即将二百余囚犯判处完毕<ref>《宋经略墓志铭》：“起知蕲州，道除提点广东刑狱。……南吏多不奉法，有留狱数年未详复者。公下条约，立期程，阅八月，决辟囚二百余。”</ref>。再改任提点[[江南西路|江南西路]]刑狱，江西百姓多于农闲时武装贩运私盐，宋慈增设关卡严加禁止，收到成效，甚至有人建议将其经验推行各地<ref>《宋经略墓志铭》：“移节江西，赣民遇农隙，率贩盐于闽粤之境，名曰盐子，各挟兵械，所到剽掠，州县单弱，莫敢谁何。公鳞次保护，讯其出入，奸无所容。举行之初，人持异议。事定，乃大服。谏者奏乞，取宋某所行，下浙右以为法。”</ref>。后兼知[[赣州|赣州]]。又调任提点[[广南西路|广南西路]]刑狱，境内大治。以朝散大夫（阶官，从六品）、直秘阁（贴职，正八品）提点[[荆湖南路|荆湖南路]]刑狱。湖南安抚大使[[陈韡|陈韡]]<ref>《宋史·理宗纪》：“（淳祐七年四月辛丑）陈韡知枢密院事、湖南安抚大使、知潭州。”</ref>辟为行府参议官，以参谋功进直宝谟阁（贴职，从七品）<ref>《宋经略墓志铭》：“兼知赣州……蜀相游公侣大拜以公按刑广右，循行部内，所至雪冤禁暴，虽恶溺出所，辙迹必至。除直秘阁，核湖南。会陈公以元枢来建大阃，兼制广西，辟公参谋。……进直宝谟阁。</ref>。再以[[朝议大夫|朝议大夫]]（阶官，正六品）、[[直焕章阁|直焕章阁]]（贴职，从七品）知[[广州|广州]]、充[[广南东路|广南东路]][[经略安抚使|经略安抚使]]、[[马步军|马步军]][[都总管|都总管]]<ref>《[[宋史|宋史]]·职官志》：“旧制，安抚总一路兵政，以知州兼充，太中大夫以上，或曾历侍从乃得之，品卑者止称主管某路安抚司公事。中兴以后，职名稍高者出守，皆可兼使，如系二品以上，即称安抚大使。广东、西、荆南、襄阳仍旧制加“经略”二字。凡帅府皆带马步军都总管。……（庆元）五年，臣僚言：‘遴选帅才，除尝任执政外，两制从官必曾经作郡、庶官必曾任宪漕实有治绩者。’从之。惟广南东、西两路则带经略安抚使。”</ref>（掌管本路军事行政），卒于任上，年六十四岁<ref>《宋经略墓志铭》：“拔直焕章阁、知广州、广东经略安抚，持大体，宽小文，威爱相济，开阃属两。忽感末疾，犹自力视事。学宫释菜，宾佐请委官摄献，毅然前往。由此委顿，以淳佑六年三月七日终于州治，年六十四岁，秩止朝议大夫。”</ref>。

== 家庭 ==

元配余氏，继室连氏，均封宜人<ref>《宋经略墓志铭》：“娶余氏，继连氏，皆封□人。”宋制，阶官朝散大夫以上官吏妻母封宜人。</ref>。

== 个人成就 ==
宋慈在二十餘年的宦海生涯中，先後四次出任[[广东|广东]]、[[江西|江西]]、[[广西|广西]]、[[湖南|湖南]]四处刑事长官，以断狱著称，在长期的断案过程了积累了丰富的法医检验经验，并与当时已有的法医学著作《[[内恕录|内恕录]]》等（时有[[疑狱集|疑狱集]]、[[疑狱事类|疑狱事类]]、[[决狱龟鉴|决狱龟鉴]]、[[检验格目|检验格目]]）加以綜合、核定和提煉。[[宋理宗|宋理宗]][[淳祐|淳祐]]七年（1247年），时任[[朝散大夫|朝散大夫]]（[[阶官|阶官]]，从六品）、新除[[直秘阁|直秘阁]]（贴职，正八品）、提点[[荆湖南路|荆湖南路]]刑狱公事、湖南安抚大使行府参议官的宋慈创作完成了《[[洗冤集錄|洗冤集錄]]》這部系統的[[法醫學|法醫學]]著作《洗冤录》<ref>《[[四库全书|四库全书]]总目提要》：“<洗冤集录>，宋宋慈撰。慈字惠父，始末未详。是书自序题淳祐丁未，结衔题朝散大夫，新除直秘阁，湖南提刑，充大使行府参议官。序中称四权臬司，於狱案审之又审，博采近世诸书，自《内恕录》以下凡数家，薈稡厘正，增以己见为一编，名曰《洗冤集录》。刊於湖南宪治。后来检验诸书，大抵以是为蓝本。而递相考究，互有增损，则不及后来之密也。”</ref>。宋慈一生平反冤案無數，他認為“獄事莫重於大辟，大辟莫重於初情，初情莫重於檢驗”（出自《[[洗冤集录|洗冤集录]]》序），堅持“審之又審”，重視現場堪驗。时人[[劉克莊|劉克莊]]在[[墓誌銘|墓誌銘]]中稱他“聽訟清明，決事剛果，撫善良甚恩，臨豪猾甚威。属部官吏以至穷闾委巷，深山幽谷之民，咸若有一宋提刑之临其前。”

== 宋慈生卒年份争议==
时人[[劉克莊|劉克莊]]《後村先生集》卷一五九《宋經略墓志銘》中說是“以浮祐六年三月七日終於州治，年六十四。”宋無浮祐年號，當為坊刻誤。若為淳祐六年（1246年），然《洗冤集錄》宋慈自序在淳祐七年丁未，顯誤。明嘉靖二十年（1541年）《建寧府志》卷二十《丘墓》记载“淳祐十二年敕葬樂田里之昌茂，特贈朝議大夫、直煥章閣，御出神道門以旌。”嘉靖三十二年（1553年）《建陽縣志》卷七《墳墓》记载“朝請大夫宋公墓在洛田里，地名昌茂。公諱慈，字惠父，仕宋，累官至朝請大夫、直煥章閣，致仕，淳祐十二年（1252年）卒，葬於此，御出神道門。”<ref>「宋慈生卒年份考辨」，諸葛計，《社會科學戰線叢刊》，1980年第2期</ref>

普遍接受的觀點認為：《宋經略墓志銘》中「浮祐六年」當為「淳祐九年（1249年）」之訛。淳祐十二年僅為死後敕葬之年，嘉靖《建陽縣志》誤。

'''宋慈墓'''，位于中国[[福建省|福建省]]南平市建阳区崇雒乡崇雒村昌茂坊（距离武夷山东站仅13公里），为一个[[福建省文物保护单位|省级文物保护单位]]，类型为古墓葬，为第一批福建省文物保护单位，公布时间为1961年5月10日<ref>{{cite book |author=福建省地方志编纂委员会编 |title=福建省志: 文物志 |publisher=方志出版社 |year=2002 |ISBN=7-80122-638-0}}</ref>。

宋慈墓的历史年代为宋。

== 《洗冤集錄》 ==
{{main|洗冤集录}}

《[[洗冤集錄|洗冤集錄]]》的宋刊本迄今尚未發現，現存最早的版本為元刻本《宋提刑洗冤集錄》。內容自“條令”起，至“驗狀說”終，共5卷，53條。從目錄來看，本書的主要內容包括：宋代關於檢验尸傷的法令；[[验尸|验尸]]的方法和注意事項；[[尸体|尸体]]現象；各種機械性[[窒息|窒息]]死；各種鈍器損傷；銳器損傷；交通事故損傷；高溫致死；中毒；病死和急死；屍體發掘等等。

《[[洗冤集錄|洗冤集錄]]》是集宋慈以前外表屍體檢驗經驗之大成的著作。作者在書中開篇即提出不能輕信口供，認為“告狀切不可信，須是詳細檢驗，務要從實”，對疑難案件尤“須是多方體訪，務令參會歸一，切不可憑一、二人口說，便以為信”。他還提出檢驗官必須親臨現場、屍格必須由其親自填寫的屍體檢驗原則。

《洗冤集錄》被譽爲世界上最早的法醫學專著，是中國法醫學的里程碑。由於受限於當時的科學水平，其內容難免有錯誤。但整體而言，瑕不掩瑜，是一部符合科學精神的傑出作品。[[清|清]][[同治|同治]]六年，[[荷蘭|荷蘭]]人首先將這本書翻譯成荷蘭文，傳入西方。
