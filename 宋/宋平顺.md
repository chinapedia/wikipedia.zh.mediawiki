'''宋平顺'''（{{bd|1945年|7月|2007年|6月3日|catIdx=Song宋}}<ref>{{cite web|url=http://news.bbc.co.uk/chinese/simp/hi/newsid_6730000/newsid_6736500/6736551.stm|title=天津政协主席宋平顺自杀身亡|date=2007年6月9日|publisher=BBC中文网|accessdate=2007年6月11日}}</ref>），[[中华人民共和国|中华人民共和国]]政府官员，祖籍[[河北省|河北省]][[深泽县|深泽县]]，于[[天津市|天津市]]出生与成长，1960年7月参加工作，1970年9月加入中国共产党，天津市和平区新华职工大学英语专业毕业，在职大专学历。宋平顺长期掌控天津政法系统，曾任中共天津市委副书记、天津市政协主席、政法委书记等要职，于2007年6月3日在天津市委大楼内自杀身亡<ref name=xinhua>[http://news3.xinhuanet.com/local/2007-07/05/content_6333952.htm 天津政协主席宋平顺严重违纪违法被开党籍] {{webarchive|url=https://web.archive.org/web/20160304200209/http://news3.xinhuanet.com/local/2007-07/05/content_6333952.htm |date=2016-03-04 }}，新华网，2007年7月5日发表，2010年11月27日查阅</ref>。

== 履历 ==
宋平顺长期在天津市公安系统工作，植根天津市政法界数十年。他早年曾任天津市公安局一处副科长、副处长。1983年起一路升迁，历任[[天津市公安局|天津市公安局]]副局长、党委常委，党委副书记。1988年5月，任天津市公安局局长、党委书记。1990年6月，升任天津市副市长、中共天津市委政法委副书记，兼市公安局局长、党委书记、武警天津市总队第一政委。1993年5月，升任中共天津市委常委、政法委书记，并继续担任天津市副市长、市公安局局长、武警天津市总队第一政委。1998年5月，升任中共天津市委副书记。2003年1月，转任天津市政协主席，但仍破例兼任天津市委政法委书记，直至2006年3月才卸去政法委书记一职，专职任市政协主席直至自杀<ref name=xinhuaziliao>[http://news.xinhuanet.com/ziliao/2003-02/20/content_737742.htm 新华网：宋平顺官方简历]，新华网，2010年11月27日查阅</ref>。

== 贪腐 ==
20世纪90年代初，在天津轰动一时的“[[永基非法集资案|永基非法集资案]]”丑闻与宋平顺有关，此事在天津市可谓家喻户晓<ref name="中国新闻网：宋平顺迷案">[http://www.chinanews.com.cn/gj/kong/news/2007/09-26/1036609.shtml 中国新闻网：宋平顺迷案]，中国新闻网，2007年09月26日发表，2010年11月27日查阅</ref>。宋平顺的情妇[[徐敏|徐敏]]，经宋平顺授意成立天津顺风公司，承包天津所有公交道路上的交通信号灯设备以及天津市公、检、法系统所有的警服、警具器械的生意<ref>[http://www.znxy.org.cn/snnc/ncml_con.asp?id=112 《三农内参》：“问题官员”和情妇们的众生相]{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}，《三农内参》，2010年11月27日查阅</ref>。

此外，著名记者[[罗昌平|罗昌平]]指出，天津市机动车驾驶适应性检测中心亦由宋平顺情妇徐敏承包，该检测中心强制天津市机动车驾驶者体检才能换证，但检测中心的人只管收钱、盖章，并不进行实际体检工作，被指滥用权力和贪腐，但就在宋平顺自杀次日，该检测中心大门紧闭、停止业务，改由医院进行体检<ref name="中国新闻网：宋平顺迷案"/>。

2004年3月15日，曾有1000多名来自天津的民众在北京[[中国银行业监督管理委员会|中国银行业监督管理委员会]]打出“打倒[[张立昌|张立昌]]，打倒大恶霸宋平顺！”的横幅表示对其不满。在天津广泛流传的民谣“两顺一昌，唬弄中央”便是对其的讽刺{{notetag|另一个“顺”指天津市公安局原局长[[武长顺|武长顺]]，2014年7月20日因涉嫌严重违纪违法而接受组织调查。}}<ref>[http://news.creaders.net/china/newsViewer.php?nid=331896&id=774510 天津书记张立昌病逝 官褒民贬]，2010年11月27日查阅</ref>。

2007年6月10日，中共天津市委书记[[张高丽|张高丽]]表示，宋平顺被查出涉及权钱交易及包养情妇等问题，具体内容是利用职权为港商谋取利益；包养情妇，并生有一子<ref name="天津通报市政协主席宋平顺服毒身亡">[http://gb.chinareviewnews.com/doc/1003/8/4/8/100384837.html?coluid=7&kindid=0&docid=100384837 天津通报市政协主席宋平顺服毒身亡]，中国评论社，2007年6月9日发表，2010年11月27日查阅</ref>。

== 自杀 ==
2007年6月4日，天津市政协主席宋平顺缺席其本该出席的会议。当日晚上9时左右，[[中共天津市委|中共天津市委]]接到工作人员报告，称宋平顺在其办公室内死亡。天津市委遂立即向[[中共中央|中共中央]]报告。随后，[[中华人民共和国公安部|公安部]]刑侦局立即派员到天津的现场进行勘查，并确定宋平顺为自杀身亡。此前，已有传言中央纪律检查委员会正在对其进行调查<ref name="天津通报市政协主席宋平顺服毒身亡"/>。事件由[[香港|香港]]《[[明报|明报]]》首先报道。据媒体报道，宋平顺在死前因涉重大[[贪污|贪污]]案件，正接受[[中纪委|中纪委]]调查。7月5日，[[新华社|新华社]]正式公布宋平顺严重违纪的官方消息，并开除其党籍，在公告裡透露了其死于自杀。最初媒体报道其自杀地点为[[天津市政协大楼|天津市政协大楼]]的办公室，后《[[南方都市报|南方都市报]]》记者报道实际自杀地点并不在政协大楼，而是在市委大楼的一间办公室内。宋平顺遂成为自中国大陆“[[文化大革命|文化大革命]]”后首名自杀身亡的中国[[正部级|正部级]]高官。

== 身後 ==
据[[新华社|新华社]]7月5日电报说，[[中共中央纪律检查委员会|中共中央纪律检查委员会]]会同有关部门对宋平顺严重违纪违法问题进行了调查。现已查明，宋平顺道德败坏，包养情妇；滥用手中权力，为情妇谋取巨额不正当利益。情节严重，影响恶劣，其行为严重违反了中共的纪律。根据《中国共产党纪律处分条例》的有关规定，中共中央纪委决定并报经中共中央批准，开除宋平顺党籍<ref name="xinhua"/>。2007年7月6日，天津市委先后召开常委扩大会议和全市区县局领导干部会议，坚决拥护中央开除宋平顺党籍决定<ref>[http://news.sohu.com/20070706/n250929069.shtml 天津市召开干部大会表示，坚决拥护中央开除宋平顺党籍决定]，2010年11月28日查阅</ref>。

== 评价 ==
天津一度流传讽刺其与前[[中共天津市委|中共天津市委]][[政法委|政法委]]书记、市政协主席宋平顺及天津市委原书记[[张立昌|张立昌]]的民谣“两顺一昌，糊弄中央”<ref>{{cite web |title = 天津书记张立昌病逝 官褒民贬 |url = http://news.creaders.net/china/newsViewer.php?nid=331896&id=774510 |publisher= |accessdate=2014年7月28日}}</ref>。（“两顺”是指前宋平顺和其下属时任[[天津市公安局|天津市公安局]]长[[武长顺|武长顺]]，“昌”是指时任中共天津市委书记[[张立昌|张立昌]]。）

财新传媒在《武长顺起底》中提到，“公安交管系统凡是有利可图处，均可见武氏（武长顺）家族以权谋利的身影和手脚，他们通过频繁的股权变更转换，于公私间腾挪转换，游刃有余。此外，[[武长顺|武长顺]]还和上司宋平顺默契配合，心照不宣，分享寻租空间。”<ref name="武长顺起底">{{cite news|author=财新记者包志明、见习记者郭清媛|title=武长顺起底|url=http://china.caixin.com/2014-10-15/100738621_all.html|accessdate=2015-02-14|newspaper=财新传媒|date=2014-10-15}}</ref>
