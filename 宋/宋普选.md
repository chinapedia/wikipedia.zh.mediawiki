'''宋普选'''（{{bd|1954年|2月18日||}}），原名'''宋建河'''，[[山东省|山东]][[博兴|博兴]]乔庄镇七合村人，生于[[山东|山东]][[宁津|宁津]]，[[中国人民解放军上将|中国人民解放军上将]]，现任[[中央军委后勤保障部|中央军委后勤保障部]]部长。

== 生平 ==
=== 早期经历 ===
宋普选的父亲宋玉章于1946年参加[[八路军|八路军]]，曾担任过时属[[河北|河北]]的[[宁津县|宁津县]]民政局局长。宋普选出生在宁津，并在宁津生活、上学，一直到他参军入伍。1969年11月，宋普选参加[[中国人民解放军|中国人民解放军]]，此后他长期在[[中国人民解放军济南军区|济南军区]]任职，先后担任战士、班长、排长、作训参谋、团作训股长、师作训科长、团长、军作训处长。此后，他担任济南军区司令部作战部作战处处长一职十余年之久，晋升为副师级干部。这一时期，宋普选和作战部参谋[[冯玉军|冯玉军]]先后合写过一些专著文章，如“落差咏叹调”、“现地侦察并未过时”等，对新时期军事学术研究等课题提出了自己的看法，获得好评<ref name="五虎将">{{cite web |title=战区五虎将：“谋”将宋普选 |url=https://news.qq.com/cross/20160225/8EVTL570.html |website=腾讯 |accessdate=2019-01-23 |date=2016-02-25 |archive-url=https://web.archive.org/web/20190123223356/https://news.qq.com/cross/20160225/8EVTL570.html |archive-date=2019-01-23 |dead-url=no }}</ref>。

=== 铁军成长 ===
1989年7月，宋普选毕业于[[中国人民解放军国防大学|国防大学]]基本系。1995年3月，升任驻山东[[泰安|泰安]]的[[中国人民解放军陆军第六十七集团军|陆军第67集团军]]200师代理师长，1996年5月，调到[[河南|河南]][[新乡市|新乡]]的[[中国人民解放军陆军第五十四集团军|陆军第54集团军]]任副参谋长。2001年3月，升任集团军参谋长，晋升为副军级干部。2003年12月，升任济南军区副参谋长。同年，宋普选再次前往国防大学深造，就读高级轮训班。在学习时，宋普选应军事科学院院长[[葛振峰|葛振峰]]上将约稿，撰写《着眼军事战争准备需要，下大力培养高素质参谋人才队伍》，收集在《参谋长论参谋修养》一书中。宋普选在特稿中强调，参谋要“甘当无名英雄”，学会“慎独”，要耐得住“寂寞”，要注重德、能、勤、绩，必须做到“作战决策科学化、情报获取和处理实时化与自动化、信息传输网络化、战场控制全员化”。这篇论文在全书中最长，见解独特，得到了当时众多首长的一致赞赏<ref name="五虎将" />。

2006年8月，宋普选回到54集团军，出任集团军军长。[[汶川大地震|汶川大地震]]期间，其所率部队曾参与抢险救灾。宋普选在汶川大地震抢险形势分析会上，喊出了“铁军就要啃最硬的骨头，抢最艰巨的任务”的口号。据统计，54集团军在救援期间成功抢救出240名幸存者，抢救遇险群众6546人，转移受灾群众31.1万人，医治受伤群众29万余人，安置灾民77万余人，抢修道路8000多公里，转运物资45.5万余吨。宋普选在抗震救灾中表现出的能力，受到军委领导的赞扬<ref name="五虎将" />。2009年2月，他接替退役的原副司令员[[徐承云|徐承云]]中将，任[[中国人民解放军南京军区|南京军区]]副司令员<ref>{{cite web|url=http://cn.chinagate.cn/politics/2009-02/24/content_17327213.htm|title=解放军高层人事调整 大批救灾将领获任新职|publisher=中国发展门户网|date=2009-02-24|access-date=2011-01-02|archive-url=https://web.archive.org/web/20150930173333/http://cn.chinagate.cn/politics/2009-02/24/content_17327213.htm|archive-date=2015-09-30|dead-url=no}}</ref>。

=== 执掌战区 ===
2013年7月，宋普选升任[[中国人民解放军国防大学|中国人民解放军国防大学]]校长，成为正大军区职将领<ref>{{cite web|url=http://china.caixin.com/2013-07-26/100561045.html|title=宋普选升任国防大学校长|accessdate=2013-07-26|author=|date=2013-07-26|publisher=财新网|archive-url=https://web.archive.org/web/20130801200206/http://china.caixin.com/2013-07-26/100561045.html|archive-date=2013-08-01|dead-url=no}}</ref>。上任后，宋普选和政委[[刘亚洲|刘亚洲]]即提出了“从严治校”12条廉政纪律规定，强调“治军先治官”，着力整治“四风”（教风、学风、演风、考风），坚持“从严治教”、“从严治考”、“从严治研”，取得卓越成效<ref name="五虎将" />。2014年12月，宋普选与[[张仕波|张仕波]]中将对调，任[[中国人民解放军北京军区|北京军区]]司令员<ref>{{cite web|url=http://news.163.com/15/0105/22/AF7QPU6900014SEH.html|title=宋普选任北京军区司令员 白建军任副司令员|accessdate=2015-01-05|author=|date=2015-01-05|publisher=网易|archive-url=https://web.archive.org/web/20150108062429/http://news.163.com/15/0105/22/AF7QPU6900014SEH.html|archive-date=2015-01-08|dead-url=no}}</ref>。2015年9月3日，宋普选在[[纪念中国人民抗日战争暨世界反法西斯战争胜利70周年大会|纪念中国人民抗日战争暨世界反法西斯战争胜利70周年大会]]中担任阅兵总指挥，陪同[[习近平|习近平]]检阅三军<ref>{{cite web |title=延续国庆阅兵惯例，北京军区司令员宋普选担任抗战阅兵总指挥 |url=http://news.163.com/15/0903/11/B2J9JH6T00014AED.html |website=网易 |accessdate=2019-01-24 |date=2015-09-03}}</ref>。

2016年1月，宋普选任新成立的[[中国人民解放军北部战区|中国人民解放军北部战区]]司令员<ref>{{cite web|url=http://news.ifeng.com/a/20160201/47322408_0.shtml|title=五大战区司令员、政委公布|publisher=凤凰网|date=2016-02-01|access-date=2016-02-01|archive-url=https://web.archive.org/web/20160201114731/http://news.ifeng.com/a/20160201/47322408_0.shtml|archive-date=2016-02-01|dead-url=no}}</ref>，领导和指挥[[黑龙江省|黑龙江]]、[[吉林省|吉林]]、[[辽宁|辽宁]]、[[内蒙古|内蒙古]]、[[山东|山东]]等5个[[中国一级行政区|省级行政区]]的武装力量，负责应对[[朝鲜半岛|朝鲜半岛]]方向的突发事件。在出任北部战区司令员后，宋普选在2016年全国两会期间接受采访，表示“战区要有战略视野”<ref>{{cite web |title=北部战区司令员宋普选：战区要有战略视野 履行好主战使命 |url=http://www.81.cn/bbzq/2016-03/15/content_6959409.htm |website=中国军网 |accessdate=2019-01-23 |date=2016-03-15}}</ref>。2017年8月，任[[中央军委后勤保障部|中央军委后勤保障部]]部长<ref name="buzhang">{{Cite web|url=http://news.ifeng.com/a/20170913/51981437_0.shtml|title=宋普选任军委后勤保障部部长|accessdate=2017-09-13|author=|date=2017-09-13|publisher=凤凰网|language=|quote=|archive-url=https://web.archive.org/web/20170913095256/http://news.ifeng.com/a/20170913/51981437_0.shtml|archive-date=2017-09-13|dead-url=no}}</ref>。但在之后的[[中共十九大|中共十九大]]上，宋普选意外落选中央委员<ref>{{cite web |title=中国共产党第十九届中央委员会委员名单 |url=http://www.xinhuanet.com//politics/19cpcnc/2017-10/24/c_1121846551.htm |website=新华网 |accessdate=2019-01-23 |date=2017-10-24 |archive-url=https://web.archive.org/web/20190123223635/http://www.xinhuanet.com//politics/19cpcnc/2017-10/24/c_1121846551.htm |archive-date=2019-01-23 |dead-url=no }}</ref>。

宋普选是[[中国共产党第十七次全国代表大会|中共十七大]]、[[中国共产党第十九次全国代表大会|十九大]]代表，第[[第十二届全国人民代表大会代表名单|十二届]]、[[第十三届全国人民代表大会代表名单|十三届]]全国人大代表。2002年7月，晋升少将军衔<ref name=buzhang/>。2010年7月，晋升中将军衔<ref>{{cite web|url=http://www.zaobao.com/wencui/2010/08/hongkong100809d.shtml|title=中国新晋陞21名中将 多为专家学者型将领|accessdate=2010-08-09|author=|date=2010-08-09|publisher=联合早报|archive-url=https://web.archive.org/web/20100811091148/http://zaobao.com/wencui/2010/08/hongkong100809d.shtml|archive-date=2010-08-11|dead-url=no}}</ref>。2015年7月31日，晋升上将军衔<ref>{{cite web|url=http://news.163.com/15/0731/18/AVSE9V3M0001124J.html|title=中央军委在北京举行晋升上将军衔警衔仪式|accessdate=2015-07-31|author=|date=2015-07-31|publisher=网易|archive-url=https://web.archive.org/web/20150802045934/http://news.163.com/15/0731/18/AVSE9V3M0001124J.html|archive-date=2015-08-02|dead-url=no}}</ref>。

== 个人 ==
* 宋普选出身作战参谋，被称为「谋将」<ref>{{cite web |title=不在中委名单内 宋普选落选引关注 |url=http://news.dwnews.com/china/news/2017-10-24/60019502.html |website=多维新闻 |accessdate=2019-01-23 |date=2017-10-24 |archive-url=https://web.archive.org/web/20171230140132/http://news.dwnews.com/china/news/2017-10-24/60019502.html |archive-date=2017-12-30 |dead-url=no }}</ref>。他始终坚持「充分发挥司令部机关职能的作用，大力培养高科技参谋人才，重点在信息作战力量、计算机网络骨干、电磁频谱管控人才」的理念，并将之践行于带兵实践。<ref name="五虎将" />
* 宋普选在原济南军区任职时，他的一个堂弟也参军入伍，但担心体检过不了。叔婶专门去找宋普选，希望能走走关系顺利些，但宋普选说：「要是体检没问题，那就堂堂正正当兵；要是体检过不了，找我也没有用。」<ref name="">{{cite web |title=这位曾担任重要任务的解放军上将 如今再次被重用 |url=https://military.china.com/important/11132797/20170913/31388005_all.html |website=中华网 |accessdate=2019-01-23 |date=2017-09-13 |archive-url=https://web.archive.org/web/20171122121121/http://military.china.com/important/11132797/20170913/31388005_all.html |archive-date=2017-11-22 |dead-url=no }}</ref>
