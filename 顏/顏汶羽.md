'''顏汶羽'''（'''Frankie NGAN Man-yu'''，1986年8月31日－），現任觀塘區（佐敦谷區）區議員，青年民建聯主席，民建聯政策副發言人，九龍社團聯會青年事務委員會主席，牛頭角街坊福利會理事長。

== 個人簡歷 ==


顏汶羽畢業於新生命教育協會呂郭碧鳳中學，繼而升讀嶺南大學，修讀社會科學學系，其後於香港城市大學完成公共政策及管理文學碩士課程，現於嶺南大學就讀經濟學博士課程。大學二年級時透過實習計劃擔任立法會議員陳鑑林的議員助理，展開其政治生涯。

== 從政生涯 ==

2007年，顏汶羽到立法會議員陳鑑林的辦事處實習，並加入民建聯，實習期後為民建聯陳百里助選區議會選舉。

2008年，加入民建聯不足幾個月，成為民建聯人力事務的副發言人至今。<ref>顏汶羽﹕冀帶領青民「向右企」 http://std.stheadline.com/daily/news-content.php?id=87246&target=2</ref>

2008年，畢業於嶺南大學社會科學學系，其畢業論文主題是最低工資，並得到最佳論文獎項。<ref> 政壇初哥蛻變獨當一面區議員　http://paper.takungpao.com/resfile/PDF/20140518/PDF/a14_screen.pdf</ref>

2011年，代表民建聯參選觀塘區佐敦谷區的區議會選舉，成為了當屆民建聯最年輕的區議員。

2012年，首次加入李慧琼團隊參選立法會，競逐超級區議會議席。<ref>民建聯53人參選打硬仗 http://paper.takungpao.com/resfile/2012-07-16/A16/A16.pdf</ref>

2015年，顏汶羽獲委任為第六屆青年民建聯主席。<ref>顏汶羽﹕冀帶領青民「向右企」http://std.stheadline.com/daily/news-content.php?id=87246&target=2</ref>　同年，於區議會選舉中，成功連任觀塘區佐敦谷區議員，以2,638票擊敗只有1,400票的黃偉達。<ref>2015年區議會選舉結果 http://www.elections.gov.hk/dc2015/chi/results_kwun_tong.html?1503030349217</ref>

2016年，第二次參選立法會，與黨主席李慧琼參選「超級區議會」的區議會（第二）功能界別，名單排名第四，代表九龍東。<ref> 顏汶羽建團隊慧攻超區

2017年，顏汶羽連任青年民建聯主席，在4月25日被民建聯中委會續委任為第七屆青民主席。<ref>顏汶羽連任青年民建聯主席 http://www.takungpao.com.hk/hongkong/text/2017/0426/77068.html</ref>

2018年，顏汶羽於七月中獲得行政長官社區服務獎。<ref>潮總宴賀獲勳21鄉彥 http://paper.wenweipo.com/2018/07/19/AY1807190001.htm</ref>


== 政治理念 ==

對於政黨，他認為從政最大目標是將自己對政策的想法實踐於社會，亦坦言建制派在這方面有優勢，舉例建制派可直接與官員游說，「好過連見你也不見」，加上民建聯在政策研究的資源投入較多，自己最終選擇加入民建聯，並坦言外界對民建聯有誤解，指民建聯並非永遠與政府站在同一綫。

他認為只是民建聯作為建制大黨，政府推出政策前總會先與民建聯商議，自然被貼上「保皇黨」的標籤。作為青年民建聯主席，他期望青民走出傳統左派的固有印象，帶頭向右靠攏，拉闊民建聯政治光譜。


== 地區服務 ==

2011年，顏汶羽代表民建聯成為觀塘區佐敦谷區的議員候選人。儘管民建聯在該區的選舉戰績是 「三連敗」，但顏汶羽他不辭勞苦， 「堅持每天做早站 ，6點鐘就落區派單張，有一段時間還試過從早上5點到9 點，下午3點再繼續到10點」，憑藉着勤奮和實幹，終於逆轉局勢，贏得市民支持，成功地成為了最年輕的區議員。

秉持着「民生無小事」的信念，積少成多，點滴為民。 親自落區，由下而上地了解情況為宗旨，凝造出「上能議政論政，下能通廁修渠」的個人形象，意將政策研究和地區工作結合。

== 民主概念 ==

顏汶羽強調民主就是包容及尊重，若衝擊法治只會適得其反，期望社會理性討論，不令政改原地踏步。他亦指香港不是一個獨立國家，民主制度設計必須符合 「一國兩制」下特別行政區的地位，並不可能和一些獨立國家的民主制度相比，因此社會需要一套適合本港作為特別行政區憲政地位的民主政制，必須能平衡本港選民選舉權和中央政府任命權；若本港追求民主制度時不顧 「一國兩制」，政改之路只會原地踏步。

被問及小撮激進「本土派」甚至鼓吹「港獨」者參選立法會，顏汶羽強調，任何政團都要正視為何他們仍有支持者，而民建聯作為跨階層的大政黨，必須正視所有意見，「但涉及一些違法主張與行為，便必須堅決反對，這個立場是相當清晰的。」<ref> 顏汶羽建團隊慧攻超區
