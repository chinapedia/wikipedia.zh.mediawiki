'''凱撒沙拉'''是一道[[沙拉|沙拉]]。
== 歷史 ==
[[File:HotelCaesar.JPG|thumb]]
{{le|凱撒·卡狄尼|Caesar Cardini}}在1924年發明了凱撒沙拉。卡狄尼是個[[墨西哥|墨西哥]][[蒂華納|蒂華納]]（Tijuana）的[[義大利菜|義式]]餐館老闆兼[[主廚|主廚]]，他住在[[聖迭戈|聖迭戈]]，但為了避開[[禁酒令|禁酒令]]（Prohibition）而在蒂華納工作。關於沙拉發明的故事有許多種版本，最常見的說法是，在有次[[美國國慶日|七月四日]]那天，餐廳耗盡了廚房內所有的東西，因此卡狄尼用他僅剩的食材，並以他的天賦異稟，製作出了凱撒沙拉。另一個說法是說，沙拉是為了一群[[好萊塢|好萊塢]]明星在週末派對後所製作出的。在現今的墨西哥蒂華納，凱撒飯店（Hotel Caesar）已經不再經營餐館，但重新裝修的飯店旁邊的酒吧與餐館近來也按照傳統開始提供凱撒沙拉（2007年1月10日）。

== 食譜 ==
一份凱撒沙拉通常由下列食材料理製作：
* [[生菜|長葉萵苣]]（romaine lettuce）
* [[焗|焗]][[麵包|麵包]]粒
* [[檸檬|檸檬]]汁
* 剛壓扁的[[蒜頭|蒜頭]]
* [[芥末|芥末]]，粉狀或膏狀
* [[橄欖油|橄欖油]]
* 磨碎的[[帕馬森乾酪|帕馬森乾酪]]（Parmesan cheese）
* 生的、[[半熟|半熟]]或滾熟的[[蛋黃|蛋黃]]
* 磨碎的[[黑胡椒|黑胡椒]]
* [[喼汁|喼汁]]或稱英國黑醋

在《来自茱莉亚·蔡尔德的厨房》（''From Julia Child's Kitchen''）這本書中，[[茱莉亚·蔡尔德|茱莉亚·蔡尔德]]敘述了在1924年當她還小時，在卡狄尼餐廳吃到凱撒沙拉的經驗。幾年後，為了重現原版的食譜，她找到了卡狄尼的女兒，羅莎·卡狄尼（Rosa Cardini）。羅莎·卡狄尼的食譜與下列連結的那些食譜所有不同。特別的是，[[萵苣|萵苣]]是蓋滿整盤的，因為莖葉的關係會看起來鼓鼓的，而且得用手拿著吃，沙拉頂多加上白煮蛋，但不會有鯷魚。與多數人認知不同的是，原本的凱撒沙拉[[食譜|食譜]]並沒有包含[[鳳尾魚|鯷魚]]，淡淡的鯷魚味是來自[[喼汁|喼汁]]的關係，卡狄尼事實上不喜歡在他的沙拉中加入鯷魚，但有些現今的食譜上都包含了鯷魚切片或是鯷魚醬的材料。

有些食譜包含了多種食材，如[[芥末|芥末調味料]]、[[酪梨|酪梨]]、[[蕃茄|蕃茄]]、[[培根|培根]]碎片、[[大蒜|大蒜]]丁香，或鯷魚。許多餐廳提供更有料的沙拉，像是灑滿[[烤雞|烤雞]]肉、烤[[鮭魚|鮭魚]]或烤[[蝦|蝦]]的凱撒沙拉。今日，在許多意式餐廳和一些墨西哥餐館都會提供凱撒沙拉，有些墨西哥餐館可能偶爾會將麵包丁換成[[玉米餅|玉米餅]]條（tortilla），或是用[[Cotija起司|Cotija起司]]來替代巴馬乾酪（味道相似）。

卡狄尼家族早期將原版的食譜申請註冊，而且瓶裝的卡狄尼凱撒沙拉醬仍然可以買得到，但沒有鯷魚。其他瓶裝的形式現在也有在販售，包括[[Morgan's|Morgan's]]和[[Newman's_Own|Newman's Own]]。

== 生蛋和沙門氏菌 ==
許多人擔心凱撒沙拉醬的衛生安全，因為其中的食材：生蛋，其常被發現有感染沙門氏菌的危險，這跟許多類似的調味醬如[[蛋黃醬|蛋黃醬]]也有一樣的顧慮，雖然在大多數的情況下，酸性的[[pH|pH]]值認為足以殺死細菌。然而，一些後來版本的食譜講求使用[[半熟蛋|半熟蛋]]或[[低溫殺菌|低溫殺菌]]的蛋，但即使餐廳製作的凱撒沙拉換成切片、煮熟的蛋，仍無法避免偶爾發生的沙門氏菌感染。今日，許多食譜省去了蛋，並調理成凱撒[[油醋|油醋]]（vinaigrette），優格也可以用來替代蛋並保有奶油狀的質感。
