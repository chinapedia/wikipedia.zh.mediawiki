'''凱莉·麥當勞'''（{{lang-en|'''Kelly Macdonald'''}}，{{bd|1976年|2月23日}}）是一名蘇格蘭女演員，最知名於演出電影《[[迷幻列車|迷幻列車]]》（1996年）、《[[高斯福大宅謀殺案|高斯福大宅謀殺案]]》（2001年）、《{{link-en|玩命911|Intermission (film)}}》（2003年）、《{{link-en|魔法保姆麥菲|Nanny McPhee}}》（2005年）、《[[哈利波特：死神的聖物Ⅱ|哈利波特：死神的聖物Ⅱ]]》（2011年）和《[[勇敢傳說之幻險森林|勇敢傳說之幻險森林]]》（2012年）。2005年的電視電影《{{link-en|咖啡館裡的女孩|The Girl in the Café}}》為她提名一項[[金球獎|金球獎]]及獲得[[黃金時段艾美獎|黃金時段艾美獎]]。她在著名迷你劇《{{link-en|政局密云|State of Play}}》中飾演Della Smith。

她於2007年電影《[[二百萬奪命奇案|二百萬奪命奇案]]》中被提名[[英國電影學院獎最佳女配角|英國電影學院獎最佳女配角]]。她也在[[HBO|HBO]]的電視劇《[[酒私風雲|酒私風雲]]》全五季（2010年–2014年）中飾演[[瑪格麗特·湯普森|瑪格麗特·湯普森]]<ref>{{Cite web |url = http://www.telegraph.co.uk/culture/tvandradio/8812019/Steve-Buscemi-and-Kelly-Macdonald-Boardwalk-Empire-interview.html |title = Steve Buscemi and Kelly Macdonald: Boardwalk Empire interview |author = Jane Mulkerrins |date = 2011-10-07 |publisher = The Telegraph |language = en}}</ref><ref>{{Cite web |url = https://www.rollingstone.com/movies/movie-news/qa-boardwalk-empires-kelly-macdonald-on-margarets-independent-streak-97534/ |title = Q&A: ‘Boardwalk Empire’s' Kelly Macdonald on Margaret's Independent Streak |author = Sarene Leeds |date = 2012-09-18 |publisher = Rolling Stone |language = en}}</ref>，為她得到2011年[[黃金時段艾美獎|艾美獎]]提名。
