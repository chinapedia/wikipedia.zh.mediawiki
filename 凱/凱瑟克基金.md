'''凱瑟克基金'''是[[香港|香港]]認可的[[慈善機構|慈善機構]]，這基金是由[[約翰．凱瑟克|約翰．凱瑟克]][[爵士|爵士]]（Sir John Keswick）及其女兒[[美琪．凱瑟克|美琪．凱瑟克]]（Maggie Keswick Jencks）於1979年創辦。

該基金現時是由一班專業人士及義工管理，秉承凱瑟克爵士和美琪的意願，資助香港及中國內地的社會福利。

== 背景 ==
約翰．凱瑟克爵士在1952年至1956年期間出任[[怡和管理有限公司|怡和管理有限公司]]的主席，他長期在[[中國|中國]]居住，對中國及人民有深厚的感情。他的遺願是回饋中國及人民。

凱瑟克爵士創立基金的目的，是向社會福利制度所忽略的範疇提供資助，例如精神病康復服務及資助一些計劃的經常性開支。凱瑟克爵士1982年逝世後，女兒美琪．凱瑟克秉承他的意願，致力支持[[香港|香港]]及中國內地的社會福利計劃。她的童年曾居於香港，也和父親一樣熱愛中國文化。在她的管理下，基金逐步擴展項目的資助範圍。最後，她於1995年因癌病而逝世。

== 焦點項目 ==
* [[Maggie's_癌症關顧中心|Maggie's 癌症關顧中心]]
** 該中心位於香港[[屯門醫院|屯門醫院]]內，是怡和前主席約翰．凱瑟克爵士命名的凱瑟克基金支持成立，而中心名「Maggie」則為他的女兒的名字美琪．凱瑟克，她在生時曾患上乳癌，故希望其他患癌病人也能積極參與治療<ref> {{cite web |language=zh-hk |url=http://news.hkheadline.com/dailynews/content_hk/2010/01/26/100958.asp | title=末期鼻咽癌女 活在當下 |publisher=[[頭條日報|頭條日報]] |date=2010-01-26 |accessdate=2013-07-29}}</ref><ref> {{cite web |language=zh-hk |url=http://orientaldaily.on.cc/cnt/news/20100126/00176_059.html | title=癌症關顧中心年半後落成 |publisher=[[東方日報|東方日報]] |date=2010-01-26 |accessdate=2013-07-29}}</ref>。
** 在凱瑟克基金及[[美琪凱瑟克癌症關顧中心基金有限公司|美琪凱瑟克癌症關顧中心基金有限公司]]的贊助下，成立香港第一間 Maggie's 癌症關顧中心，配合屯門醫院的癌症治療，提供一站式服務，協助癌症患者及親友能開懷面對患病對生活的影響<ref> {{cite web |language=zh-hk |url=http://the-sun.on.cc/cnt/news/20100126/00410_032.html | title=醫知健：愛心小屋為癌病人打氣 |publisher=[[太陽報|太陽報]] |date=2011-01-26 |accessdate=2013-07-29}}</ref>。
* [[白普理寧養中心|白普理寧養中心]]
** 凱瑟克基金和[[善寧會|善寧會]]成立香港第一間獨立寧養中心，位於[[沙田|沙田]][[亞公角山路|亞公角山路]]，在1992年正式啟用，為26名患有癌症的人士提供居所。該中心其後在1995年轉交[[醫院管理局|醫院管理局]]管理。
