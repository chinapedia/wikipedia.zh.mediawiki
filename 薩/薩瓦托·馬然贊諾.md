'''薩瓦托·馬然贊諾'''（'''Salvatore Maranzano'''，{{IPA-it|salvatore marandzano}}，{{bd|1886年|7月31日|1931年|9月10日}}）是一名來自[[西西里島|西西里島]][[卡斯泰拉姆馬雷德爾戈爾福|卡斯泰拉姆馬雷德爾戈爾福]]鎮的[[黑社會|有組織犯罪]]人物，以及一名早期的[[西西里黑手黨|西西里黑手黨]]老大，他領導了後來成為在美國的{{link-en|博納諾犯罪家族|Bonanno crime family}}。他發動了{{link-en|卡斯泰拉姆馬雷戰爭|Castellammarese War}}以奪取[[美國黑手黨|美國黑手黨]]活動的控制，並短暫成為黑手黨的「{{link-en|所有老大的老大|Capo di tutti capi}}」（ capo di tutti capi）。他後來在[[查理·盧西安諾|查理·「幸運」·盧西安諾]]的命令下被謀殺，盧西安諾建立了一個權力分享的協議來防止未來的地盤爭奪戰。<ref>{{Cite web |url = http://americanmafiahistory.com/salvatore-maranzano/ |title = Salvatore Maranzano, and the Castellamarese War |publisher = americanmafiahistory.com |language = en}}</ref>

== 早年 ==
作為一名年輕人，馬然贊諾想成為一名[[神父|神父]]，甚至學習成為一名神父，但後來變成與他家鄉的[[西西里黑手黨|黑手黨]]有聯繫。<ref>Critchley, p. 144</ref>馬然贊諾擁有威嚴的儀態，並受到他的黑社會同行的極大尊重。他對[[尤利烏斯·凱撒|尤利烏斯·凱撒]]和[[羅馬帝國|羅馬帝國]]入迷，並喜歡與受過低教育的美國黑社會的相似人士談論這些話題。<ref>Raab, pp. 26-27</ref>正因為如此，他被黑社會同輩給予綽號為「小凱撒」（Little Caesar）。

== 早年生涯 ==
馬然贊諾在[[第一次世界大戰|第一次世界大戰]]後不久移居至美國，定居[[布魯克林|布魯克林]]。在建立合法生意而成為[[地產代理|地產代理]]的同時，他也維持越來越多的非法私酒業務，使用房地產公司作為其非法業務的前台。不久後，馬然贊諾參與了[[賣淫|賣淫]]和非法走私[[麻醉品|麻醉品]]，他也成為一名年輕的{{link-en|約瑟夫·博納諾|Joseph Bonanno}}的導師，並晉升他成為他的小老大。

== 卡斯泰拉姆馬雷戰爭 ==
為了保護和維持馬然贊諾逐步建立的犯罪帝國的福祉，他於1930年向對手[[喬·馬塞里亞|喬·馬塞里亞]]（所有老大的老大）宣戰，開始了{{link-en|卡斯泰拉姆馬雷戰爭|Castellammarese War}}。1931年4月15日，馬塞里亞被殺，而馬然贊諾在黑社會衝突中獲勝。

== 所有老大的老大 ==
馬然贊諾現在是紐約最強大的黑手黨成員（Mafioso）。馬塞里亞被謀殺的兩週後，馬然贊諾召集幾百名黑手黨成員到一個在[[紐約上州|紐約上州]]未公開地點的宴會廳。馬然贊諾確認並宣布戰後倖存的犯罪家族老大——[[查理·盧西安諾|查理·「幸運」·盧西安諾]]、[[湯米·加利亞諾|湯米·加利亞諾]]、[[喬·普羅法斯|喬·普羅法斯]]、{{link-en|文森特·曼加諾|Vincent Mangano}}和他自己。他還為自己創造了一個額外的地位，即「老大們的老大」。由於馬然贊諾以前聲稱自己想要終結老大的統治，所以對被召集的黑手黨成員來說，這是意想不到的。<ref name="ff28">Raab, pp. 28-29</ref><ref name="MafEnc">{{cite book |title=''The Mafia Encyclopedia'' |last=Sifakis |first=Carl |authorlink= |coauthors= |year=1987 |publisher=Facts on File |location=New York City |isbn=0-8160-1856-1 }}</ref>

然而，馬然贊諾的詭計，他傲慢對待他的下屬，以及他喜歡將他的組織與羅馬帝國（他鑒於凱撒的一系列軍事指揮而試圖對組織進行模仿）作比較，並不會與盧西安諾和他雄心勃勃的朋友，如[[維多·吉諾維斯|維多·吉諾維斯]]、[[法蘭克·卡斯特羅|法蘭克·卡斯特羅]]和其他人相處得好。實際上，盧西安諾相信馬然贊諾比馬塞里亞更加迂腐守舊和渴望權力。儘管他擁護現代的組織方式，包括全體士兵在受到{{link-en|頭目 (黑手黨)|Caporegime|頭目}}監督的情況下進行家族的非法工作，但本質上馬然贊諾是一個「{{link-en|小鬍子皮特|Mustache Pete}}」——一個老派的黑手黨成員，過於沉浸在舊世界方式中。例如，他反對盧西安諾與猶太黑幫，如[[邁爾·蘭斯基|邁爾·蘭斯基]]和[[本傑明·西格爾|本傑明·「畢斯」·西格爾]]的合作關係。事實上，盧西安諾和他的同行在擺脫馬然贊諾前，一直打算等待他們的時機。<ref name="MafEnc" />

馬然贊諾馬上就意識到這一點，並開始計劃謀殺盧西安諾、吉諾維斯、卡斯特羅等人。<ref>Maas, p. 80</ref>可是馬然贊諾沒有採取足夠的行動，當他僱用{{link-en|瘋狗科爾|Mad Dog Coll}}謀殺盧西安諾和吉諾維斯的時候，盧西安諾已經發現了馬然贊諾的計劃。

== 逝世 ==
盧西安諾在1931年9月10日安排了[[本傑明·西格爾|畢斯·西格爾]]、{{link-en|薩繆爾·萊文|Samuel Levine (mobster)}}和另外兩名黑幫前往馬然贊諾的辦公室，假裝成會計師或稅務人員。在{{link-en|漢姆斯利大廈|Helmsley Building|紐約中央大樓}}的9樓辦公室裡，他們立即解除了馬然贊諾守衛的武裝。四名刺客然後射殺並刺死了馬然贊諾。當他們下樓時，他們遇到往樓上與馬然贊諾約會的科爾。他們警告他這裡有突襲，而科爾逃跑了。

在馬然贊諾去世後，盧西安諾廢除了「capo di tutti capi」（所有老大的老大）的地位。馬然贊諾的小老大{{link-en|約瑟夫·博納諾|Joseph Bonanno}}接管了馬然贊諾大部分的非法勾當，演變成{{link-en|博納諾犯罪家族|Bonanno crime family}}。雖然博納諾否認知道有關盧西安諾殺死馬然贊諾的計劃，但如果博納諾仍然支持他的前任老大，那麼博納諾將不大可能會活著留下來。

馬然贊諾被埋葬在紐約[[皇后區|皇后區]]的{{link-en|聖約翰公墓|St. John Cemetery (Queens)}}，靠近盧西安諾的墳墓。<ref>{{Cite web |url = http://www.nydailynews.com/new-york/queens/cemetery-mob-mafiosi-article-1.312368 |title = Cemetery has a mob of mafiosi |author = Mehrnoosh Torbatnejad |date = 2008-02-26 |publisher = New York Daily News |language = en}}</ref><ref>{{Cite web |url = https://www.findagrave.com/cgi-bin/fg.cgi?page=gr&GRid=5678 |title = Salvatore Maranzano |publisher = Find a Grave |language = en}}</ref>唯一知名的馬然贊諾的照片來自他死時的現場。（作者David Critchley確認那張通常被聲稱是馬然贊諾的面部照片，其實是扎根於倫敦的黑幫{{link-en|梅希納兄弟|Messina Brothers|薩瓦托·梅希納}}。<ref>{{Cite journal|url = http://www.magcloud.com/browse/issue/23974|title = Maranzano Muddle|last = Critchley|first = David|date = July 2009|journal = Informer|doi = |pmid = |access-date = |pages = 22-23}}</ref>）

== 流行文化 ==
* 馬然贊諾在[[馬里奧·普佐|馬里奧·普佐]]的小說《[[教父_(小說)|教父]]》中扮演著一個小小的虛構化角色。
* 1972年的電影《[[賊殺賊|賊殺賊]]》中，由{{link-en|約瑟夫·懷斯曼|Joseph Wiseman}}飾演馬然贊諾。
* 1991年的電影《[[江湖好漢|江湖好漢]]》中，由[[米高·甘邦|米高·甘邦]]飾演馬然贊諾，但被稱為Faranzano。
* 1999年的電影《''Lansky''》中，由Ron Gilbert飾演馬然贊諾。
* 1999年{{link-en|Lifetime (TV network)|Lifetime (TV network)|Lifetime}}電視網的電影《''Bonanno: A Godfather's Story''》中，由[[愛德華·詹姆斯·奧莫斯|愛德華·詹姆斯·奧莫斯]]飾演馬然贊諾。
* 2011年的電視劇《[[超疑特工|超疑特工]]》的集數"{{link-en|Immortal Sins|Immortal Sins}}"中，由Cris D'Annunzio飾演馬然贊諾。
* 電視劇《[[酒私風雲|酒私風雲]]》的第五季中，由Giampiero Judica飾演馬然贊諾。在劇中，他的暗殺被描述為由[[努基·湯普森|努基·湯普森]]所命令的，而努基的弟弟Eli是參與的槍手之一，槍擊馬然贊諾的頭部來解決他。
