'''萨姆索岛'''({{lang|da|Samsø}})是[[丹麥|丹麥]]的一個[[島|島]]。

遊客常在6、7月時來採摘[[草莓|草莓]]。在丹麥，這裏以早熟的[[馬鈴薯|馬鈴薯]]馳名。

== 地理 ==
它在[[北海_(马来西亚)|北海]][[卡特加特海峽|卡特加特海峽]]，離[[日德蘭半島|日德蘭半島]]15公里。

島東的海峽稱為珊索海峽({{lang|da|Samsø Bælt}})，它將萨姆索岛和西蘭島分開了。最接近的縣是西面，在日德蘭半島上的[[奧德縣|奧德縣]]({{lang|da|Odder}})。

萨姆索岛面積為112平方公里，約有四千名居民。

== 能源 ==
1997年，萨姆索岛在一個比賽獲勝了。她要開始一個計劃：在十年之內，全面以[[可再生能源|可再生能源]]發電。1998年，島上只有8%的[[電力|電力]]和15%的熱能來自可再生能源。2000年，借助11個新[[風車|風車]]，島的電力已能自給自足。全島近二十戶人家中，有440人佔有風車的股份。[http://ide.idebanken.no/bibliotek_engelsk/ProsjektID.asp?ProsjektID=189]{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}

萨姆索岛的能源為以木、太陽能和稻草為基礎的城市供熱系統、風車和太陽能。

實踐這個全面使用可再生能源的最大的困難來自交通。2003年，島上只有三輛[[電力車|電力車]]。

有些人已經將萨姆索岛視作未來能源供求上的「[[示範單位|示範單位]]」。

== [[北歐神話|北歐神話]] ==
[[File:Maarup_Østerstrand_mod_nord.jpg|thumb]]

[[薩克索·格拉瑪提庫斯|薩克索·格拉瑪提庫斯]]指，傳說中瑞典的英雄{{lang|da|Hjalmar}}和他的朋友{{lang|da|Orvar-Odd}}曾在這個島上對抗狂戰士{{lang|da|Arngrim}}的12個兒子。這場戰爭記錄了在{{lang|da|Faroese ballads}}、{{lang|da|Orvar-Odd}}[[薩迦_(文學)|薩迦]]和{{lang|da|Hervarar}}薩迦內。

== 行政 ==
萨姆索岛屬[[中日德兰大区|中日德兰大区]]的'''[[萨姆索自治市|萨姆索自治市]]'''。這個縣的範圍除了萨姆索岛，還有位於萨姆索岛東北面的小島{{lang|da|Hjortholm}}，整體面積為114平方公里。縣會議在{{lang|da|Tranebjerg}}舉行。縣長是{{lang|da|Carsten Bruun}}，屬[[自由黨_(丹麥)|自由黨]]。
