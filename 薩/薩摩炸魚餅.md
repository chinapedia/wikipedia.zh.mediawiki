'''薩摩炸魚餅'''（{{lang-ja|薩摩揚げ}}）是一種起源[[日本|日本]][[九州|九州]]南方[[鹿兒島|鹿兒島]]一帶，以[[魚漿|魚漿]]製成的油炸食品。

日本[[關西|關西]]地方多以原指日式油炸物料理「[[天婦羅|天婦羅]]」來稱呼薩摩炸魚餅，<ref>{{Cite book |author=吉川誠次、大堀恭良 |year=2002年 |script-title=ja:日本・食の歴史地図 |publisher=生活人新書 |isbn=4-14-088016-3 |pages=81、88頁}}</ref>而台灣民間也受其影響將天婦羅日語發音轉換成的字詞「'''甜不辣'''」來稱呼之。但如今台灣的甜不辣名詞意義已經和日本的天婦羅定義不同，台灣甜不辣是指白色條狀的麵粉混製的魚漿條，而日本天婦羅一般是指蔬菜、魚肉裹上麵包粉和蛋液油炸沾柴魚醬汁的油炸食物<ref>{{Cite web |url=http://worldfoods.tabi-navis.com/asia/taiwan/tianbula.html |script-title=ja:甜不辣（天婦羅）／黒輪 tián bú là [台湾風薩摩揚げ |work={{lang|ja|世界の食べ物}} |accessdate=2015年6月12日 }}</ref>

== 方法 ==
調理方式先以[[鯖科|鯖魚]]、[[鯛科|鯛魚]]、[[鱈科|鱈魚]]等食用魚做成魚漿，之後在加入調味料的魚漿裡混合一些像是切碎後的[[青蔥|青蔥]]、[[洋蔥|洋蔥]]、[[胡蘿蔔|胡蘿蔔]]之類蔬菜或是其它海鮮類，然後弄成圓餅狀、或長條等其它模樣拿去油炸。<ref name="kagoshimakenjinkai">{{Cite web |url=http://www.kagoshimakenjinkai.ne.jp/satsumaage.html |script-title=ja:もっと鹿児島が知りたい［3］さつま揚げの話 |work={{lang|ja|関東鹿児島県人会連合会}} |accessdate=2015年6月12日 }}</ref><ref name="living-eye">{{Cite web |url=http://www.living-eye.com/contents_culture.php?maingenreid=10000067&subgenreid=10000151&contentsid=10002387 |script-title=ja:つけあげ（薩摩揚げ、さつまあげ） |work=Living Eye |accessdate=2015年6月12日 }}</ref>

== 來源 ==
此食品的由來有一說是早年統治九洲南方[[薩摩國|薩摩國]]的[[島津氏|島津氏]]第二十八代當主[[島津齊彬|島津齊彬]]為了發展當地漁村經濟，鼓勵地方人民將獲補的魚群加工作成此種油炸食物販賣。<ref name="living-eye"/>另一說是[[江戶時代|江戶時代]]時期；[[中國|中國]][[福建省|福建省]]的一些油炸魚肉料理方法有流傳到[[琉球|琉球]]一帶，當時琉球人將魚肉混合澱粉之類的油炸物叫為「{{lang|ja|チキアギ}}」，隨後此類食品傳到靠近琉球的鹿兒島後改以「{{lang|ja|つけ揚げ}}」稱呼，並演變成現今的薩摩炸魚餅。<ref name="nmnweb">{{Cite web |url=http://www.nmnweb.net/shokuzai/satsuma.html |script-title=ja:薩摩揚げとは |work=Natural Medicine Network |accessdate=2015年6月12日 }}</ref>

== 養分 ==
薩摩炸魚餅內含大量[[不飽和脂肪|不飽和脂肪]]與作為腦部養分的[[二十二碳六烯酸|DHA]]，而因多以[[菜籽油|菜籽油]]油炸較不易引發提高血液中[[膽固醇|膽固醇]]的問題。<ref name="living-eye"/><ref name="nmnweb"/>
