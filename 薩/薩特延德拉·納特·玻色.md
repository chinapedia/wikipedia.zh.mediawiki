'''薩特延德拉·納特·玻色'''（{{lang-bn|'''সত্যেন্দ্র নাথ বসু'''}}，{{lang-en|'''Satyendra Nath Bose'''}}，{{bd|1894年|1月1日|1974年|2月4日|catIdx=B}}），[[印度|印度]]物理學家，專門研究[[數學物理|數學物理]]。他最著名的研究是1920年代早期的[[量子物理|量子物理]]研究，該研究為[[玻色-愛因斯坦統計|玻色-愛因斯坦統計]]及[[玻色-愛因斯坦凝聚|玻色-愛因斯坦凝聚]]理論提供了基礎。[[玻色子|玻色子]]就是以他的名字命名的。

儘管[[玻色子|玻色子]]、[[玻色-愛因斯坦統計|玻色-愛因斯坦統計]]及[[玻色-愛因斯坦凝聚|玻色-愛因斯坦凝聚]]概念的相關研究所獲得的[[諾貝爾獎|諾貝爾獎]]不止一個──最近的是2001年的[[物理學|物理學]]獎，因對[[玻色-愛因斯坦凝聚|玻色-愛因斯坦凝聚]]的理論進展有貢獻而獲獎的──但玻色本人從未獲得過[[諾貝爾物理學獎|諾貝爾物理學獎]]（1956年、1959年與1962年獲得諾貝爾物理學獎提名）。他多才多藝，能說多國語言之餘，還會彈{{le|埃斯拉古琴|Esraj}}（Esraj，一種跟[[小提琴|小提琴]]相近的樂器）。

著名物理學家[[贾扬特·纳利卡|贾扬特·纳利卡]]在他的《科學邊緣》一書中寫道：“S·N·玻色的粒子物理研究（約1922年），其中闡明了[[光子|光子]]的表現，並為統計遵從量子規則的微系統提供了機會，是二十世紀印度科學貢獻的前十名之一，是可被視為諾貝爾獎級數的研究。”<ref> ''The Scientific Edge'' by Jayant V. Narlikar, Penguin Books, 2003, page 127.其他納里卡認為其研究達到諾貝爾獎級數的二十世紀印度科學家還有[[拉馬努金|拉馬努金]]、[[錢德拉塞卡拉·拉曼|錢德拉塞卡拉·拉曼]]和[[梅格納德·薩哈|梅格納德·薩哈]]（Megn Nad Saha）</ref>

== 早年及生平 ==
玻色生於[[印度|印度]][[西孟加拉邦|西孟加拉邦]]的[[加爾各答|加爾各答]]，是七名孩子中的長子。他的父親蘇倫特拉納特·玻色（Surendranath Bose）曾任職於[[東印度鐵路|東印度鐵路]]工程部。

玻色就讀於加爾各答{{le|Hindu School, Kolkata|Hindu School, Kolkata|印度教學校}}（Hindu School），後就讀於也位於加爾各答的[[院長學院|院長學院]]（Presidency College），他在這兩所當地知名學府時都獲得了最高分。他接觸了一些優秀的老師，如[[賈加迪什·錢德拉·玻色|賈加迪什·錢德拉·玻色]]（Jagdish Chandra Bose，無血緣關係）及{{le|普拉富爾拉·錢德拉·羅伊|Prafulla Chandra Roy}}（Prafulla Chandra Roy），他們都鼓舞了玻色要立好遠大志向。他於1911年至1921年任[[加爾各答大學|加爾各答大學]]物理學[[科系|系]]講師。他於1921年轉到了當時成立不久的[[達卡大學|達卡大學]]物理學系（現位於[[孟加拉|孟加拉]]境內），也是任職講師。

玻色於1924年寫了一篇推導[[馬克斯·普朗克|普朗克]][[普朗克黑體輻射定律|量子輻射定律]]的論文（詳見下節〈[[薩特延德拉·納特·玻色#沒錯的錯誤|薩特延德拉·納特·玻色#沒錯的錯誤]]〉），當中並沒有提到任何古典物理。在開始時未能發表的挫折下，他把論文直接寄給身在[[德國|德國]]的[[艾爾伯特·愛因斯坦|艾爾伯特·愛因斯坦]]。愛因斯坦意識到這篇論文的重要性，不但親自把它翻譯成德語，還以玻色的名義把論文遞予名望頗高的《[[德國物理學刊|德國物理學刊]]》（''Zeitschrift für Physik''）發表。就是因為此次賞識，玻色能夠第一次離開印度，前往[[歐洲|歐洲]]並逗留兩年，期間與[[路易·德布罗意|路易·德布罗意]]、[[居里夫人|居里夫人]]及愛因斯坦工作過。

玻色於1926年回到達卡，任教授兼物理學系主任，並繼續留在達卡大學教學至1945年。那時候他回到了加爾各答，在加爾各答大學教學至1956年，他退休時被授予名譽教授頭銜。

有一次玻色在達卡大學講課，課題是[[光電效應|光電效應]]及[[紫外災變|紫外災變]]，玻色打算向學生展示當時理論的不適之處，因為理論預測的結果跟實驗不符。在講課期間，玻色在應用理論時犯了錯，意想不到的是居然得出一個跟實驗一致的預測。（他後來將講課內容改寫成了一篇短文，叫《普朗克定律與光量子假說》。）

那錯誤是一個很簡單的錯──跟認為擲兩枚硬幣得兩正面的概率是三分之一是一樣的──任何對[[統計學|統計學]]有一點基礎理解的人都知道有問題。然而預測結果跟實驗吻合，使玻色意識到事有蹊蹺。他首次提出[[麦克斯韦-玻尔兹曼分布|麦克斯韦-玻尔兹曼分布]]對微觀粒子不會成立，這是因為由[[海森堡測不準原理|海森堡測不準原理]]所導致的變動此時會大得足夠構成影響。故此他強調在每個體積為'''h<sup>3</sup>'''的[[相空間|相空間]]中找到粒子的概率而捨棄粒子不同的位置和[[動量|動量]]。

好幾份[[物理|物理]][[學刊|學刊]]都沒有為玻色發表論文。他們認為他所展現的是一個簡單錯誤，而且玻色的發現被忽略了。灰心的他寫了封信給[[愛因斯坦|愛因斯坦]]，愛因斯坦馬上就同意他的觀點。愛因斯坦寫了一篇支持玻色理論的論文，遞予《德國物理學刊》發表，並要求將這兩篇論文一同發表，此時玻色的理論終於受到推崇。這是1924年的事。玻色早前曾經把愛因斯坦的[[廣義相對論|廣義相對論]]論文從[[德語|德語]]翻譯成[[英語|英語]]。有人說玻色把愛因斯坦當成他的“導師”。

玻色的“錯誤”能得出正確結果，這是因為光子們是不能被分辨出來的，也就是不能把任何兩個同能量的光子當作兩個能被明確識別的光子。比方說，如果在另一個宇宙裏，硬幣表現得像光子及其他[[玻色子|玻色子]]一樣，擲出兩正的機率的確是三分之一（正反=反正）。玻色的“錯誤”現在被稱為[[玻色-愛因斯坦統計|玻色-愛因斯坦統計]]。

愛因斯坦採取了這個概念，並把它延伸到原子去。這為預測某個現象的存在舖好了路，這個現象就是現在的[[玻色-愛因斯坦凝聚|玻色-愛因斯坦凝聚]]，在這現象中一組高密度的[[玻色子|玻色子]]（[[自旋|自旋]]為整數的粒子，以玻色命名）在超低溫狀態中會成為[[玻色-愛因斯坦凝聚|玻色-愛因斯坦凝聚體]]，於1995年被實驗所證實。

== 往後的研究 ==
在這以後玻色的概念在世界物理學界廣受好評，達卡大學於1924年允許他休假到[[歐洲|歐洲]]去旅行。他在[[法國|法國]]度過了一年，跟[[居里夫人|居里夫人]]共事，也跟多位知名[[科學家|科學家]]見過面。之後他又多遊學一年，在[[柏林|柏林]]跟愛因斯坦共事。在1926年他回到達卡大學之後，就立即於被擢升為[[教授|教授]]。他並沒有[[博士學位|博士學位]]，一般來說他是不夠資格當教授的，但是愛因斯坦還是推薦了他。他的研究範圍很廣，從[[X射線晶體學|X射線晶體學]]到[[統一場理論|統一場理論]]都有涉獵。他還跟[[梅格納德·薩哈|梅格納德·薩哈]]一起發表了[[氣體|真實氣體]]用的一條[[狀態方程|狀態方程]]。

除物理以外，他還研究過[[生物化學|生物化學]]及[[文學|文學]]（[[孟加拉語|孟加拉語]]及英語）。他還深入地學習過[[化學|化學]]、[[地質學|地質學]]、[[動物學|動物學]]、[[人類學|人類學]]、[[工程學|工程學]]及其他[[科學|科學]]。作為一個有[[孟加拉|孟加拉]]背景的印度人，他花了不少時間把[[孟加拉語|孟加拉語]]推廣為教學語言，將孟加拉語應用到他所翻譯的科學上的論文，以及推廣該地區的發展。

玻色於1944年被選為[[印度科學促進協會|印度科學促進協會]]主席，1958年獲選為[[英國皇家學會|英國皇家學會]]會員。

== 軼事 ==
#有一次大科學家[[尼尔斯·玻尔|尼尔斯·玻尔]]正在講課。玻色列席。講課者講着講着，中途在解釋某一點時有難處。他一直都在黑板上寫着；他停下來，轉向玻色，問道，“玻色教授能幫我個忙嗎？”講課期間玻色都在閉着眼坐着。聽眾們都忍不住向波耳教授的話報以微笑。令他們驚奇的是，玻色張開了眼睛；一下子就把講課者的難題給解決了，之後他坐下來又把眼睛閉上了。<ref>[http://calcuttaweb.com 加爾各答網]上的傳記。</ref>
#1927年在意大利科莫举行了科莫会议，除了[[爱因斯坦|爱因斯坦]]、[[薛定谔|薛定谔]]和[[狄拉克|狄拉克]]以外，当代最著名了物理学家，包括[[玻尔|玻尔]]、[[海森堡|海森堡]]、[[普朗克|普朗克]]、[[洛倫茲|洛倫茲]]、[[德布罗意|德布罗意]]等都出席了。但是玻色却没有能够出席，原因很离奇。因为当时大会向远在印度的玻色教授发出了邀请函，寄往了加尔各答大学，署名“寄给加尔各答大学的玻色教授”。但是当时玻色已经离开加尔各答大学去了达卡大学，而加尔各答大学还有一位姓玻色，全名叫做D.M.玻色的教授，而当时的通讯并不如现在发达，于是这位名不见经传的玻色就代替了当时已经很有名望的S.N.玻色，参加了众星云集的科莫大会。
