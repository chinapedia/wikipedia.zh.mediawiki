《'''-{信義}-'''》（{{lang-ko|'''신의'''}}，{{lang-en|'''Great Doctor'''}}）為[[韓國|韓國]][[SBS_(韩国)|SBS]]自2012年8月13日起播出的[[SBS月火連續劇|月火連續劇]]，由《[[黎明的眼睛|黎明的眼睛]]》、《[[沙漏_(電視劇)|沙漏]]》、《[[太王四神記|太王四神記]]》等電視劇的[[金鍾學|金鍾學]]導演和[[宋智娜|宋智娜]]編劇攜手打造，此劇亦是金鍾學的遺作。以[[恭愍王|恭愍王]]元年為時代背景。

==劇情==
講述[[高麗|高麗]]時代的武士[[崔瑩|崔瑩]]（[[李敏鎬|李敏鎬]] 飾）和現代（2012年）[[整形|整形]]外科女醫生柳恩修（[[金喜善|金喜善]] 飾），穿越時空相遇，之後共同輔助造就賢明君王治理天下之過程。

== 關於劇名 ==
前名《神醫》是以現代醫生穿越時空來到高麗時代行醫之內容所訂定的名字，然而[[文化廣播_(韓國)|MBC]]曾提出此劇與《[[Dr._JIN|Dr. JIN]]》的故事核心設定類似，有剽竊的嫌疑。在重新編寫劇本後人物佈局有所變動，比起醫生劇情更聚焦在武士[[崔瑩|崔瑩]]身上。因此劇名《-{信義}-》比《神醫》更符合故事內容，雖然韓文劇名「{{lang|ko|신의}}」並無變動，但意思卻相差甚遠，不過在[[大中華地區|大中華地區]]有些電視台仍舊以《神醫》為劇名播出。

== 榮譽記事 ==
*2014.11.13 2012年播出由李敏鎬主演的SBS TV電視劇《信義》於11月11日在日本東京澀谷Hikarie Hall舉行的「GAYO Entertainment Awards 2014」韓國電視劇部門被評為第一名。為2014年度最佳韓劇。<ref>[https://www.koreastardaily.com/tc/photo/51103 李敏鎬主演《信義》成日本觀眾最喜歡的韓劇]2014.11.13韓星網</ref>

* 2017 日本週刊雜誌KPOP影音銷售-韓國戲劇（藍光DVD銷售前20名排行榜）李敏鎬和金喜善主演的"信義"2017年7月份奪銷售冠軍。<ref>[http://www.koretame.com/user_data/rankingpage.php?mode=hanryu&term=month_rireki 韓國·韓國戲劇排名（最佳第20名）2017年7月份]</ref>

== 其他 ==
2012年10月19日，[[李必立|李必立]]因眼睛受傷，不得不終止拍攝，決定提前退出劇組並於隔週入院進行手術。受傷是因為意外事故，和《-{信義}-》的拍攝無關<ref>[http://ent.sina.com.cn/v/j/k/2012-10-19/10383767252.shtml 韓星李必立眼睛受傷退出《-{信義}-》拍攝]新浪娛樂{{zh}}</ref>。劇組安排在第28集府院君派火修人二人前往搜捕醫仙時大肆屠殺典醫寺，並透過對話交待張御醫為保蜱蟲的解藥而為二人所殺。

第一集的現代拍攝場地位於首爾江南區的[[奉恩寺|奉恩寺]]及[[COEX|COEX]]。
