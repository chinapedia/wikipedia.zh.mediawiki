'''信興酒樓'''（'''{{lang-en|Shun Hing Restaurant}}'''），是[[香港|香港]]一間已結業的[[粵式酒樓|酒樓]]，位於[[深水埗|深水埗]][[桂林街|桂林街]]95A號。早在1936年由羅氏家族開業。到2016年因考慮翻新困難，宣佈屹立深水埗80年的酒樓於12月29日下午5時結業。該酒樓和[[信興集團|信興集團]]並無關係。

== 歷史 ==
[[File:Shun_Hing_Restaurant_Interior_201612.jpg|270px]]
1936年，酒樓在深水埗桂林街一座戰前樓宇的地舖開業。到1950年代因[[唐樓|唐樓]]拆卸重建一度停業，重建後酒樓返回原址繼續經營，其內部裝潢一直使用至今。

該處1970年代仍是「信興茶室」，到1980年初才轉為酒樓經營模式，於購入[[基隆街|基隆街]]舖面以打通並擴張店面名後則行號先後改名為「信興樓」及「信興酒樓」。

到1990年代吸引客人來吃[[魚翅撈飯|魚翅撈飯]]及擦鮑魚（前者於2010年代初取消供應）。踏入2000年代中期以降客人以中老年男士為主<ref>{{cite news|author1=林可欣|title=【獨家】深水埗信興酒樓年底結業　老店不敵新招牌規定及時代轉變|url=http://www.hk01.com/%E7%A4%BE%E5%8D%80/54524/-%E7%8D%A8%E5%AE%B6-%E6%B7%B1%E6%B0%B4%E5%9F%97%E4%BF%A1%E8%88%88%E9%85%92%E6%A8%93%E5%B9%B4%E5%BA%95%E7%B5%90%E6%A5%AD-%E8%80%81%E5%BA%97%E4%B8%8D%E6%95%B5%E6%96%B0%E6%8B%9B%E7%89%8C%E8%A6%8F%E5%AE%9A%E5%8F%8A%E6%99%82%E4%BB%A3%E8%BD%89%E8%AE%8A|accessdate=2016-12-20|publisher=香港01|date=2016-11-18}}</ref>。

== 特色 ==
[[File:Shun_Hing_Restaurant_Dinsum_kiosk_201612.jpg|270px]]
[[File:Shun_Hing_Restaurant_Interior2_201612.jpg|270px]]
1958年，該店聘請[[霓虹燈|霓虹燈]]公司訂造樓高四層的霓虹招牌。

但根據[[屋宇署|屋宇署]]2010年新制度下，招牌不能伸逾4.2米，離地不能少於3.5米。超過上述規限便是違例，因此到2016年6月被迫拆卸，街坊對此感到可惜<ref>{{cite news|url=http://www.hk01.com/%E7%A4%BE%E5%8D%80/30270/%E6%B7%B1%E6%B0%B4%E5%9F%97%E7%92%80%E7%92%A8%E6%A8%99%E8%A8%98%E5%85%89%E8%BC%9D%E5%88%B0%E6%AD%A4-60%E5%B9%B4%E8%80%81%E6%8B%9B%E7%89%8C-%E5%9F%8B%E8%91%AC-%E5%A0%86%E5%A1%AB%E5%8D%80 |title=深水埗璀璨標記光輝到此 60年老招牌「埋葬」堆填區|publisher=香港01|date=2016-07-09}}</ref>。

酒樓招牌菜是[[棉花雞|棉花雞]]、[[鵪鶉蛋燒賣|鵪鶉蛋燒賣]]、[[燒賣|燒賣]]、[[牛肉腸粉|牛肉腸粉]]、[[燒味飯|燒味飯]]等<ref>{{cite news|url=http://stanksh4food.blogspot.hk/2014/06/blog-post_19.html |title=老去的深水埗，老去的《信興酒樓》|date=2014-06-19}}</ref><ref>{{cite news|url=http://lamhome2002.blogspot.hk/2016/04/1936.html |title=香港老酒家 信興酒樓|date=2016-04-24}}</ref><ref>[http://news.chilin.hk/hk/?job=body&sec=eJzbcGxpFAAGugJ2&page=9&path=Q29udGVudF9OZXdzLmNmbT9DaGFubmVsPWphJlBhdGg9MjAxOTEwMTE0MjA2L3ZwYTMuY2Zt&pic=big&date=20150710 信興——還是點心車最好][[明報|明報]]，2015年7月10日</ref>。

== 結業 ==
2016年11月中，該店因為考慮翻新困難、成本高昂、拆卸霓虹招牌等問題，由美國回流香港的羅梓洋宣佈屹立深水埗80年的酒樓於2016年12月29日結業<ref>{{cite news|url=http://www.hk01.com/%E7%A4%BE%E5%8D%80/54524/-%E7%8D%A8%E5%AE%B6-%E6%B7%B1%E6%B0%B4%E5%9F%97%E4%BF%A1%E8%88%88%E9%85%92%E6%A8%93%E5%B9%B4%E5%BA%95%E7%B5%90%E6%A5%AD-%E8%80%81%E5%BA%97%E4%B8%8D%E6%95%B5%E6%96%B0%E6%8B%9B%E7%89%8C%E8%A6%8F%E5%AE%9A%E5%8F%8A%E6%99%82%E4%BB%A3%E8%BD%89%E8%AE%8A |title=獨家】深水埗信興酒樓年底結業　老店不敵新招牌規定及時代轉變|date=2016-11-18|publisher=[[香港01|香港01]]}}</ref><ref>{{cite news|url=http://hk.apple.nextmedia.com/etw/%E6%B5%81%E8%A1%8CGossip/article/20161118/3_55934162/%E5%82%B3%E7%B5%B1%E8%8C%B6%E6%A8%93%E5%8F%88%E5%B0%91%E4%B8%80%E9%96%93-%E6%B7%B1%E6%B0%B4%E5%9F%97%E4%BF%A1%E8%88%88%E9%85%92%E6%A8%93%E5%B9%B4%E5%BA%95%E7%B5%90%E6%A5%AD|title=傳統茶樓又少一間！深水埗信興酒樓年底結業|publisher=[[蘋果日報|蘋果日報]]（飲食男女）|date=2016-11-18}}</ref><ref>{{cite news|url=http://hk.on.cc/hk/bkn/cnt/news/20161218/bkn-20161218120003358-1218_00822_001.html|title=港故：80年酒樓結業　海歸少東堅持風光告別|publisher=[[on.cc|on.cc]]|date=2016-12-18}}</ref>。
