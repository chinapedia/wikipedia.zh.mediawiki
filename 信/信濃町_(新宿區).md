'''信濃町'''（{{jpn|j=信濃町|hg=しなのまち|rm=Shinanomachi|f=y}}）是[[日本|日本]][[東京都|東京都]][[新宿區|新宿區]]的[[町|町]]名。町域[[人口|人口]]為957人<ref>[http://www.city.shinjuku.lg.jp/content/000130038.xls 新宿区　住民基本台帳の町丁別世帯数及び男女別人口（日本人と外国人の合計）]</ref>。未實施[[住居表示|住居表示]]，[[郵遞區號|郵遞區號]]160-0016。

町域內有[[JR|JR]][[信濃町站|信濃町站]]，[[外苑東通|外苑東通]]西側有{{link-ja|慶應義塾大學醫院|慶應義塾大学病院|慶應醫院}}，東側以[[創價學會|創價學會]]相關設施聚集而聞名。

== 概要 ==
位於新宿區南部。北接[[須賀町_(新宿區)|須賀町]]與[[左門町|左門町]]，東與東南部鄰[[南元町_(新宿區)|南元町]]，東一部分接[[若葉_(新宿區)|若葉]]三丁目，南與[[霞丘町|霞丘町]]之間以JR[[中央線快速|中央線]]路廊為界，西通[[大京町|大京町]]。

外苑東通西側是{{link-ja|慶應義塾大學醫院|慶應義塾大学病院}}、[[慶應義塾大學|慶應義塾大學]]信濃町校區（醫學部）與[[東京電力醫院|東京電力醫院]]、民音音樂博物館等。地域南部的外苑東通東側是[[創價學會|創價學會]]本部等創價學會相關設施，有「學會村」之稱<ref name =systemtositeno>[[古川利明|古川利明]]　『システムとしての創価學會＝公明党』（[[第三書館|第三書館]]　1999年10月25日）　ISBN 978-4807499229 </ref>。地域南端附近是[[信濃町站|信濃町站]]。

[[外苑東通|外苑東通]]貫通町域中央。幹線道路沿線大樓林立，巷內與北部以住宅地為主。
*當談論到政治和宗教時，創價學會因本部位於此地而常以「信濃町」代稱，政治上則是指[[公明黨|公明黨]]（日本政界常用各黨本部所在地做為代稱）。
* 「信濃町商店振興會」加盟店的貼紙是與創價學會三色旗一樣的紅、黃、藍三色<ref name =systemtositeno/>。此外，這三色也常見於町内商店的看板。

== 歷史 ==
=== 地名的由來 ===
當地是[[江戶時代|江戶時代]][[幕臣|幕臣]][[永井尚政|永井尚政]]（[[信濃守|信濃守]]）的屋敷<ref name =systemtositeno/>。1873年（[[明治|明治]]6年），以現在的外苑東通為界，西是「千駄谷西信濃町」，東為「四谷東信濃町」。

=== 沿革 ===
* 1873年（[[明治|明治]]5年）　外苑東通為界成立千駄谷西信濃町與四谷東信濃町。
* 1943年（[[昭和|昭和]]18年）兩町合併為信濃町。
