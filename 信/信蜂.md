《'''信蜂'''》（{{lang-ja|テガミバチ}}）是[[浅田弘幸|浅田弘幸]]的[[少年漫畫|少年漫畫]]作品，信蜂與作者前作《[[夢幻高手|夢幻高手]]》的不同，是一部奇幻風格的漫畫，講述幻想世界中郵差的故事。

== 概要 ==
於《[[月刊少年Jump|月刊少年Jump]]》2006年10月號開始連載，隨著《[[月刊少年Jump|月刊少年Jump]]》的停刊，暫時連載於《[[週刊少年Jump|週刊少年Jump]]》，其後於《[[Jump_Square|Jump Square]]》繼續連載。18本漫画单行本陆续于2007年1月4日至2014年8月4日发售，2011年3月漫畫銷量累計突破400萬冊<ref>{{cite web|url=http://mantan-web.jp/2011/03/09/20110309dog00m200049000c.html |title=テガミバチ ： 作者のデビュー25周年でイラスト集発売 |publisher=[[每日新聞|每日新聞]] |date=2011年3月9日 |accessdate=2014年7月10日 |language=Ja |deadurl=bot: unknown |archiveurl=https://archive.is/20120715033028/mantan-web.jp/2011/03/09/20110309dog00m200049000c.html |archivedate=2012年7月15日 }}</ref>，已完结。

本作動畫版先在台灣播放，之后香港於2011年8月4日在[[Animax|Animax]]播出。2012年1月30日台灣Animax開始在週一到週五22:30進行第二季首播。[[香港電視網絡|香港電視網絡]]於2015年1月22日播放第一季及第二季，主要人物名稱及用語依照臺灣漫畫出版社所用譯名，而不是依照香港漫畫出版社所用的譯名

== 故事 ==
在沒有黎明的行星上，有一片稱為「琥珀之地（Amber Ground）」的大陸。有一班於[[首都|首都]]上的人工[[太陽|太陽]]照射不到的危險地域執勤國家公務員，國民把他們的「心」託付於重要的「信」中運送出去。這就是被「信蜂」的工作。

少年拉格以「信件」的身分遇上信蜂葛修，兩人於旅程中互相扶持克服逆境，逐漸培養出真摯的感情，促使拉格有了要成為信蜂的目標，並藉此獲得首都曉的通行証以尋找自己的母親……

== 用語說明 ==
; {{nihongo|信蜂|テガミバチ|LETTER BEE}}
: 國家公務[[郵差|郵件速遞員]]，俗稱「BEE」，工作是在人工太陽的光線照射不到的地方送「信」。
: 他們擁有打倒鎧蟲的武器「心彈鎗」（注意：日本[[明治政府|明治政府]]郵件速遞員鎗所持歴史有），無論遇上任何危險都會把「信」送達目的地。「信」不僅是明信片和包裹等細小的物件，也包含像拉格一樣的「人」和達溫一樣的「動物」。

; {{nihongo|領蜂|ヘッド・ビー|HEAD BEE}}
: 信蜂最高職位的稱號。據說沒有不能投遞的信件，是葛修和拉格的目標。
: 現役的首領蜂只有一人。負責政府的重要信件，更在首都執行工作。

; {{nihongo|Dingo|相棒|ディンゴ}}
: 為「信蜂」工作術語的一種，意指共同行動的搭檔。
: 除了狗外，也可以訓練[[獅子|獅子]]或[[豹|豹]]等。如果有錢還可以雇用格鬥家。

; {{nihongo|鎧蟲|鎧虫|ガイチュウ}}
: 棲息於尤沙里和猶達加等城鎮陰暗處，是有一層強韌堅硬的鎧甲覆蓋著身體的殘暴危險生物。他們被認為沒有意志，但有被人們的心吸引而襲擊人們的習性。在太古時好像尋求著某些東西的意志。打倒鎧蟲的唯一方法就是要在鎧蟲內令「心」產生迴響，但一般武器無法對它造成傷害，因此需要心彈鎗為武器。雖然尤沙里的學者發表過－鎧蟲並不存在「心臟」，但拉古卻確實曾聽到了鎧蟲悲嗚的聲音，後來才知道那是葛修的記憶。
: 鎧蟲的名字是酒精飲料或原料命名。

; 精靈琥珀
: 能代替太陽的能源資源，從太古開始沈睡在大地之中。「[[精靈|精靈]]」是寄宿在樹木內擁有大地靈性力量的蟲，其後與[[琥珀|琥珀]]化為精靈琥珀。「信蜂」以精靈琥珀與其他道具組合為武器使用。

; 摩訶
: 與龍相似的生物，金色的鬃毛以及海藍色的瞳孔。雖然生性殘酷，但身姿美麗，在一部分地域被視為神聖的象徵。
: 看守著精靈庭院。

; 蜂巢
: 尤沙里中央的郵政館，俗稱「BEE-HIVE」。

; 鈴
: AG的貨幣單位。

; 閃爍之日（瞬きの日）
: 12年前的第311日，人工太陽的光消失之日。在這個瞬間，葛修失去母親的記憶。政府表示這是「進行人工太陽的調查及調整之日」，亦是拉格和希爾貝特出生的日子。

== 琥珀之地（AG） ==
四面環海、終日黑夜的國家，生活各種各樣的人種和種族。本島以河川分割，以階級分為「亞卡茲克」、「尤沙里」及「猶達加」三個區域。

=== 區域 ===
; 亞卡茲克（拂晓）
: AG的首都，被人工太陽照射著。是特級及上流階層的人生活的都市。
; 尤沙里（黄昏）
: 中產階級的人生活的地域。郵政館與「信蜂」的據點。
;;夜想道（Nocturne hood）
::尤烏薩利中央的主要幹道，亦是「蜂巢」的所在地。
;;木莓之丘（Raspberry Hill）
::位於尤烏薩利西部，拉格在信蜂面試審查時，遞送郵件的目的地，為鎧蟲古廉暮思的棲息地。
;;祈禱之丘
::尤烏薩利裡最能感受到人工太陽的「光」，同時也是葛修和絲露貝蒂喜愛的地方。附近有鎧蟲雷特。
;;薄荷雪芭冰鎮
::尤烏薩利東北方山頂的小村，鎧蟲棲息地區非常廣，鎧蟲茜貝魯棲息在此地的薩爾橋崖下。
;;灰鋼城
::動畫原創城鎮，能從近路哈爾多沙漠和遠路修羅峽谷抵達。
;;哈爾多沙漠
::動畫原創，極危險的地方，是很多鎧蟲的棲息地，曾經出現鎧蟲阿布山的變種。
;;甜蜜之水
::當地的水會使飲用者麻醉，是一個充滿反政府活動的城鎮。附近有鎧蟲山崎，曾受到鎧蟲席魯多攻擊。
; 猶達卡（深夜）
: 工人和下等階級人們主要生活的地域，亦是鎧蟲棲息的地方，差不多全部地域都是危險地帶。居民的收入都被政府吸乾，也傳聞有人從外海走私而過著富裕的生活。
;;哥薩·貝爾
::位於連古司鎮，是拉古與他的母親安居住的地方，亦是拉古成為「信」的地方。
;;布爾‧柏普堅山脈
::鎧蟲黛貴莉棲息的地方。
;;喬澤白沙漠
::空中飛舞著美麗的綿毛般小花，亦是鎧蟲布卡治棲息之處。
;;坎培爾·里度斯
::拉格被送到的港口城市，是猶達加最南端，亦是有外海海盜停泊的違法港。
;;蘭多市
::雜技團「Love Someone Down」曾經公演的地方。
;;愛敦煤礦坑鎮
::拉格與妮綺相遇的地方，擁有出租馬車與火車站的城鎮。
;;貝哥洛里森林
::鎧蟲科亞露西斯棲息的地方。
;;基利埃鎮
::別稱絕望之城，連接猶達加與尤沙里之間的「畢夫雷特橋」附近的城鎮。
;;溪谷之城「斯蘭斯奧」
::愛麗娜的故鄉，拉路克曾經前往當地的小村莊喬治購買飾品，因為山體崩壞，鎧蟲亞普珊出現在此地。
