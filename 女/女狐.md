〈'''女狐'''〉（[[英語|英]]：MEGITSUNE、[[日語|日]]：メギツネ）是[[BABYMETAL|BABYMETAL]]在2013年6月19日發行的第五張單曲。也是從[[櫻花學院|櫻花學院]]獨立後的首張音樂作品。

== 發行 ==
2013年4月1日於網路上公布消息<ref>{{cite news|url=http://natalie.mu/music/news/87796|title=BABYMETALが生バンドと“修行”ツアー、6月には新曲発売|newspaper=音楽[[Natalie_(網站)|Natalie]]|publisher=株式会社ナターシャ|date=2013-04-01|accessdate=2016-01-11|language=ja}}</ref>，2013年6月19日[[TOY'S_FACTORY|TOY'S FACTORY]]（BMD FOX RECORDS）正式發行，包括三種首批限量版《キ》版、《ツ》版、《ネ》版及普通版四種版本<ref name="natalie20130510">{{cite news|url=http://natalie.mu/music/news/90354|title=BABYMETAL、新作アートワークで妖艶メギツネ姿に|newspaper=音楽Natalie|publisher=株式会社ナターシャ|date=2013-05-10|accessdate=2013-11-12|language=ja}}</ref>。首批限量版的DVD分別收錄2012年10月至2013年2月間舉行的專場公演「[[LIVE_〜LEGEND_I、D、Z_APOCALYPSE〜|I、D、Z〜LEGEND]]」演唱會影像片段<ref name="natalie20130510"/>。此外，首批限量版內含演唱會參與資格的抽選券，當選者可參加2013年7月14日在[[東京都|東京都]]{{link-ja|目黒鹿鳴館|目黒鹿鳴館}}舉行的「狐狸祭」限定公演<ref>{{cite news|url=http://www.barks.jp/news/?id=1000092314|title=【Kawaii girl Japan】全員キツネ？！ BABYMETAL、数百人のキツネ面のファンと「キツネ祭り」開催|newspaper=BARKS|publisher=ジャパンミュージックネットワーク株式会社|date=2013-07-15|accessdate=2016-01-11|language=ja}}</ref>。

=== 銷售成績 ===
在日本[[Oricon公信榜|公信榜]]中，以22000張銷售量登上週榜第7名<ref>{{cite web|url=http://biz-m.oricon.co.jp/news/data/1242.shtml|title=客層広がるbabymetal、海外展開の道しるべとなるか|work=ORICON BiZ online|publisher=[Oricon株式会社|date=2014-08-18|accessdate=2016-01-11|language=ja}}</ref>。

== 封面設計 ==
單曲封面的插圖是由導演田中紫紋設計，以櫻花為背景，化著狐狸妝的日本古裝女性背著樂器出嫁，畫面是一個新娘穿著喪服的錯亂感。加入了將{{link-en|Flying V|Gibson Flying V}}吉他融合[[三味線|三味線]]、把衣服上的[[家紋|家紋]]變成魔法陣等BABYMETAL元素<ref>《{{link-ja|ヘドバン|ヘドバン (雑誌)}}》2013年7月4日 VOL.1 出版社：{{link-ja|SHINKO MUSIC|シンコーミュージック・エンタテイメント}}（ISBN： 978-4-40-163846-8）</ref>。

== 音樂類型 ==
; 主打歌〈女狐〉
: 運用西洋的[[民謠金屬|民謠金屬]]、[[金屬蕊|金屬蕊]]與[[力量金屬|力量金屬]]，加入東洋的[[日本民族樂器|日本民族樂器]]太鼓與三味線，並融入[[江戶時代|江戶時代]]民謠〈[[樱花_(民谣)|櫻花]]〉等元素，是一首[[和洋折衷|和洋折衷]]的音樂。它的主題是女權，為如[[大和撫子|大和撫子]]這樣的傳統女性發聲，歌詞提到「女人如戲子」、「臉上笑著 心裡卻痛哭著」。此曲錄製了36種不同版本後才定案發表<ref>{{cite web|url=http://www.cinra.net/interview/201605-djtekinasomething|title=ベビメタで評価されても飽き足りない。ゆよゆっぺの核心を突く|publisher=CINRA.NET|date=2016-05-27|accessdate=2016-07-05|language = ja}}</ref>。

; 普通版B面曲〈紅月-Akatsuki-〉
: SU-METAL的獨唱曲<ref name="barks20130510">{{cite news|url=http://www.barks.jp/news/?id=1000090162|title=【Kawaii girl Japan】BABYMETAL、”祭りメタル”「メギツネ」の最新アートワーク、CD詳細解禁|newspaper=BARKS|publisher=ジャパンミュージックネットワーク株式会社|date=2013-05-10|accessdate=2016-01-11|language=ja}}</ref>。融合力量金屬、[[速度金屬|速度金屬]]以及[[X_JAPAN|X JAPAN]]式的[[敘事曲|敘事曲]]、雙吉他對位演奏法，被普遍認為是向X JAPAN的〈[[紅_(X單曲)|紅]]〉致敬<ref name="whiplash">{{Cite web|url=http://whiplash.net/materias/cds/198699-babymetal.html|title=Babymetal: Mais headbangers que metade das bandas de hoje|work=
whiplash|date=2014-02-26|accessdate=2016-08-20|language=pt}}</ref>。

; 首批限量版B面曲〈撒嬌大作戰〉

== 現場表演 ==
[[File:Babymetalfondatheatrelosangeles8.jpg|thumb]]
現場演出中除了主唱[[中元鈴香|SU-METAL]][[演歌|演歌]]式的唱法，副唱[[水野由結|YUIMETAL]]與[[菊地最愛|MOAMETAL]]在表情上也十分嚴肅，並有日本[[祭_(日本)|祭典]]舞蹈、化妝撲粉、面具遮臉等動作<ref>{{cite news|url=http://gekirock.com/news/2013/04/babymetal_2ndsingle.php|title=BABYMETAL、2ndシングル『メギツネ』を6/19にリリース決定！さらに全公演生バンド演奏によるライヴ・ツアーの追加公演も発表！！|newspaper=激ロック|publisher=激ロックエンタテインメント株式会社|date=2013-04-15|accessdate=2016-01-11|language=ja}}</ref>。

〈女狐〉的現場首演於2013年6月22日淘兒唱片澀谷店「『女狐』發行紀念 “狐仙”祈願奉納會＆迷你演唱會」上<ref>{{cite news|url=http://www.oricon.co.jp/news/2025891/full/|title=BABYMETALが新曲ヒット祈願「ライブで盛り上がって」|newspaper=ORICON STYLE|publisher=ORICON|date=2013-06-22|accessdate=2016-01-11|language=ja}}</ref>。

電視首演於2013年6月21日[[日本電視台|日本電視台]]的音樂節目「{{link-ja|〜音樂龍〜|ミュージックドラゴン}}」<ref>{{cite news|url=http://natalie.mu/music/news/93337|title=BABYMETAL「音楽龍」でタカトシとヘドバン披露|newspaper=音楽Natalie|publisher=株式会社ナターシャ|date=2013-06-21|accessdate=2016-01-11|language=ja}}</ref>。之後在[[NHK綜合頻道|NHK綜合頻道]]7月16日播出的「[[MUSIC_JAPAN|MUSIC JAPAN]] ANNEX」亮相<ref>{{cite news|url=http://tower.jp/article/news/2013/07/09/n02|title=7月15日MJ特番にベビメタ、しゃちほこ、FTISLAND、赤い公園、在日ファンクら|newspaper=TOWER RECORDS ONLINE|publisher=[[淘兒唱片|淘兒唱片]]|date=2013-07-09|accessdate=2016-01-11|language=ja}}</ref>。

合作演出於2014年2月2日，在[[台灣|台灣]][[台北市|台北]]的「完全櫻樂團 第壹團 重音偶像大對決」中，與台灣的金屬樂團[[閃靈樂團|閃靈]]同台演奏<ref>{{cite news|url=http://www.metalsucks.net/2014/02/10/video-babymetal-performs-chthonic/|title=Video: Babymetal Performs with Chthonic|newspaper=MetalSucks|author=Axl Rosenberg|date=2014-02-10|accessdate=2016-01-11|language=en}}</ref>。

現場表演歌曲到一半時，固定會穿插SU-METAL拿狐神面具遮臉、高舉，然後在[[Breakdown_(音樂)|Breakdown]]結束的霎那將面具拋下舞台的橋段<ref>{{Cite web |url=http://www.barks.jp/news/?id=1000093230|title=【Kawaii girl Japan/ライブレポート】BABYMETAL降臨!<SUMMER SONIC 2013>を興奮の渦へ|accessdate=2015-10-29|work=BARKS|publisher=ジャパンミュージックネットワーク株式会社|date=2013-08-12|language=ja}}</ref>。
