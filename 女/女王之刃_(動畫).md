{{日本作品|女王之刃|クイーンズブレイド|Queen's Blade}}是同名視覺對戰型遊戲書系列《[[女王之刃|女王之刃]]》中的基本設計為基礎發展而成的動畫系列；但動畫當中的設定、故事發展和原作情節並不相同的。而《女王之刃 叛亂》則是原創續集。

《女王之刃》電視動畫由三期故事構成，第一期副標題是「流浪戰士」（{{lang|ja|流浪の戦士}}），第二期副標題是「王位繼承者」（{{lang|ja|玉座を継ぐ者}}）。2010年8月至2011年3月發售續作，全6卷OVA《女王之刃 美麗的鬥士們》（{{lang|ja|美しき闘士たち}}）。

於2011年底至2012年初發售的兩卷OVA，是《女王之刃》過渡至《女王之刃 叛亂》的故事。OVA第一話附錄在2011年10月29日發售的《女王之刃Premium Visual Book》中，OVA第二話則附錄在2012年1月29日發售的《女王之刃 叛亂Premium Visual Book》之中。

2012年電視動畫推出第三期續篇，副標題是「叛亂」（{{lang|ja|リベリオン}}）。2013年發售外傳OVA《戰敗女王》（{{lang|ja|ヴァンキッシュド・クイーンズ}}）講述各美鬥士的敗北。

2016年紀念對戰類視覺遊戲書《女王之刃》刊行10周年紀念而推出的OVA動畫。OVA《女王之刃 魔法之書》上下卷動畫與2016年1月29日所發售的《鏡之魔術姬 SNOW WHITE》同捆限定版中一同推出。

2018年宣布重啟系列《女王之刃 無限（Unlimited）》企劃，OVA《女王之刃 無限》新系列角色和世界觀的設定幾乎與以前一致，不過會展開新的故事，新作的主題為「帥氣與色氣」。

== 戰敗女王 ==

== 魔法之書 ==
《'''女王之刃 魔法之書'''》（{{lang|ja|クイーンズブレイド グリムワール}}紀念對戰類視覺遊戲書《女王之刃》刊行10周年紀念而推出的OVA動畫。集結了動畫《女王之刃》第一季與第二季的製作團隊，其中包括了擔當總監督兼系列構成的吉本欣司、負責角色設計的鈴木真矢等人，以見證新一代美鬥士們的活耀物語為目標，除此之外，曾登場過的美鬥士們也將預定作為嘉賓登場。

OVA《女王之刃 GRIMOIRE》與2016年1月29日所發售的《鏡之魔術姬 SNOW WHITE》同捆限定版中一同推出。
