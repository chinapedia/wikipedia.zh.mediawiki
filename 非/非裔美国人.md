'''非裔美国人'''（{{lang-en|African American}}）指[[美國人口|美國人口]]中的[[黑人|黑人]]，又稱為'''美国黑人'''。以前[[人种学|人种学]]家称呼亞洲人、白人、黑人三个人种为“[[蒙古人种|蒙古人种]]”、“[[高加索人种|高加索人种]]”和“[[尼格罗人种|尼格罗人种]]”。尼格罗（negro）是[[拉丁语|拉丁语]]，詞義为“黑色的”，在美国[[种族主義|种族主義]]猖獗時，逐渐演变成对黑人的誣衊语言，等同於“[[黑鬼|黑鬼]]”。[[非裔美国人民权运动|非裔美国人民权运动]]崛起後，这个单词被摒棄，黑人采用“黑色的”（black）自称。不過1960年代後，因为以肤色称呼人种被认为是不够[[政治正确|政治正确]]，这个单词也被摒棄，而改为“非洲的”，因为黑人的祖先都来自[[非洲|非洲]]。

== 社会观念 ==
大多美国非裔祖先都是在17世紀和18世纪[[西非|西非]]被绑架后引入美国而压迫为[[奴隶|奴隶]]。在19世纪中期，他们由美国[[南北战争|南北战争]]而获得自由，獲憲法保障平等的權利，但此後因[[美國最高法院|美國最高法院]][[普莱西诉弗格森案|普莱西诉弗格森案]]，有關[[隔離但平等|隔離但平等]]的判決，以及美國各州陸續出現不少[[種族隔離|種族隔離]]的法律，導致非裔美国人的公民權利不受保障。非裔美国人在20世纪中期，他们由[[美国民权运动|美国民权运动]]而最終获得全面的[[公民權|公民權]]。2008年，[[巴拉克·歐巴馬|巴拉克·歐巴馬]]贏得[[2008年美國總統選舉|總統選舉]]，成為[[美國歷史|美國歷史]]上首位有黑人血统的[[美國總統|總統]]。

非裔美国人一般也指後來移民到美國的非洲人。其他族裔的美国人是自愿来到美国的，而他们的祖先是被迫的，和其他族裔不同，所以他们倒宁可和自己的非洲同胞一样，被称为黑人（black），反而不愿意被称为非裔美国人。另外，這也與1960年起的一股黑人自我認同運動有關，這股運動與當時狂飆的[[民權運動|民權運動]]相結合，強調「黑就是美」（Black is beautiful），不再將"Black"看做是一種較為低等的形容詞。一般而言，稱為非裔美国人實際上帶有[[政治正確|政治正確]]的意味，因為這樣比較可以包含平等的意義，避免直接以膚色及種族稱呼黑人，與白人相對。

== 政治倾向 ==
非裔美国人在早期絕大部分一直支持[[共和黨_(美国)|共和黨]]，因為共和黨總統[[亞伯拉罕·林肯|亞伯拉罕·林肯]]支持解放黑奴及種族平等。但自從1930年代[[民主黨_(美国)|民主黨]]政府實行[[羅斯福新政|羅斯福新政]]以及提倡和支持民權運動後，他們轉而支持[[民主黨_(美国)|民主黨]]。現時美國有80-90%以上的非裔美国人支持民主黨，但也有約7-10%的非裔美国人支持共和黨。共和黨總統在過去五十年只獲不到15%的非裔美国人選票。

== 社会问题 ==
