《'''非法圖利'''》（{{lang-en|'''''The Racket'''''}}）是一部1928年的美國[[犯罪電影|犯罪電影]]，由[[劉易斯·邁爾斯通|劉易斯·邁爾斯通]]執導，主演有{{link-en|Thomas Meighan|Thomas Meighan}}、{{link-en|瑪麗·普雷沃斯特|Marie Prevost}}、{{link-en|路易斯·沃海姆|Louis Wolheim}}和{{link-en|喬治·E·史東|George E. Stone}}。本片由[[侯活·曉治|侯活·曉治]]監製，{{link-en|巴特利特·科馬克|Bartlett Cormack}}和Tom Miranda編劇，由[[派拉蒙電影|派拉蒙電影]]發行。<ref>{{Cite web |url = http://www.afi.com/members/catalog/DetailView.aspx?s=&Movie=11516 |title = The AFI Catalog of Feature Films:''The Racket'' |publisher = AFI |language = en}}</ref>本片改編自Cormack的1927年[[百老匯劇院|百老匯]]戲劇《''{{link-en|The Racket (play)|The Racket (play)|The Racket}}''》。<ref>{{Cite web |url = https://www.ibdb.com/broadway-show/the-racket-7342 |title = ''The Racket'' play produced on Broadway at the Ambassador Theatre |publisher = IBDB.com |language = en}}</ref>

== 背景 ==
[[File:The_Racket_(1928)_film_poster.jpg|thumb]]
由於描述腐敗的警察部隊和市政府而引起爭議，本片和戲劇於那時候的[[芝加哥|芝加哥]]被禁。<ref>{{Cite web |url = http://www.tcm.com/this-month/article/85281%7C0/The-Racket.html |title = ''The Racket'' at Turner Classic Movies, by Brian Cady |publisher = Turner Classic Movies |language = en}}</ref>
