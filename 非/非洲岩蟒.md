'''非洲岩蟒'''（學名：''Python sebae''）是[[蛇亞目|蛇亞目]][[蟒科|蟒科]][[蟒屬|蟒屬]]下的一種無毒蟒蛇，主要分布於[[非洲|非洲]]南部[[撒哈拉以南非洲|撒哈拉地區]]，是[[非洲|非洲]]體型最巨大的蛇類。目前共有兩個亞種已被確認。<ref name="ITIS">{{ITIS|ID=634785|taxon=''Python sebae''|year=2007|date=12 September}}</ref>

== 特徵 ==
成年的非洲岩蟒體長可達6[[米_(單位)|米]]，是全球最巨型的蛇類之一（其它巨蛇亦是同屬[[蟒屬|蟒屬]]下的蛇類）<ref name="Meh87">Mehrtens JM. 1987. Living Snakes of the World in Color. New York: Sterling Publishers. 480 pp. ISBN 0-8069-6460-X.</ref>目前為止，有紀錄最長的非洲岩蟒是於1932年於[[科特迪瓦|科特迪瓦]]所發現的，其體長達9.72米。<ref>Wood, The Guinness Book of Animal Facts and Feats. Sterling Pub Co Inc (1983), ISBN 978-0-85112-235-9</ref>

非洲岩蟒的體色主要由[[棕色|棕色]]構成，身體上有許多[[橄欖色|橄欖色]]的不規則圖紋。其體紋與顏色均與[[緬甸蟒|緬甸蟒]]十分相似，因此容易被混淆。然而這兩種蟒蛇並沒有緊密的關連。

== 地理分布 ==
非洲岩蟒主要分布於[[非洲|非洲]][[撒哈拉以南非洲|撒哈拉地區]]，包括[[坦桑尼亞|坦桑尼亞]]、[[剛果民主共和國|剛果民主共和國]]、[[肯雅|肯雅]]、[[贊比亞|贊比亞]]、[[赤道畿內亞|赤道畿內亞]]、[[莫桑比克|莫桑比克]]等地方。其標準產地曾被標示為「[[美國|美國]]」與及「[[巴西|巴西]]」，均屬訛誤。<ref name="McD99"/>

== 生態行為 ==
非洲岩蟒活躍於一般草原和[[熱帶乾濕季氣候|熱帶草原]]，不過其活動區域並不會遠離水源，有時候也會闖進森林邊緣地帶，但很少深入其中。<ref name="Meh87"/>

非洲岩蟒頗依賴水源，當天氣太熱或太乾燥的時候，牠們會強行躲進其它動物所挖掘的洞穴裡，並進入[[夏眠|夏眠]]狀態。不過非洲岩蟒性情較為凶惡，脾氣不佳，若受騷擾，容易發動咬擊，這一點與普遍較為溫馴的[[緬甸蟒|緬甸蟒]]迥然不同（二者外貌頗為相似）。

== 進食習慣 ==
非洲岩蟒在捕食方面屬於機會主義者，牠能以憑身體使出強勁的擠壓力，將獵物殺死。幼年的非洲岩蟒主要捕食[[鼠類|鼠類]]，對於一些為農作物被田鼠肆意啃食的農夫而言，這些小非洲岩蟒頗為幫上了忙。可是，成年後的非洲岩蟒卻往往要捕食大型的動物，例如[[羊|羊]]、[[羚羊|羚羊]]等，甚至亦會捕食小型的[[鱷魚|鱷魚]]，這一點為畜牧業帶來相當威脅。

== 繁殖 ==
非洲岩蟒多於[[春天|春天]]開始進行繁殖，雌蛇每次約能誕下100枚蛇卵，亦會負責守護及孵育其蛇卵。初生幼蛇約有45至60[[公分|公分]]長。

== 保育狀況 ==
非洲岩蟒有者漂亮的紋路，因此他的皮成為了皮革市場的上等材料，而且非洲岩蟒的肉在非洲是許多人的重要食物之一，另外非洲岩蟒雖然在寵物市場是冷門物種〈太兇暴〉，但仍然有著一定的市場需求。食用市場、皮革市場和寵物市場的需求使得非洲岩蟒在原生地遭到大獵捕捉，加上棲地破壞使得數量的減少更加快速。目前本種族群雖然不至於有滅絕的危險，但非洲岩蟒的數量已不再普遍，族群狀態變成了只有某些地方還很多的區域性普遍，而且大型個體越來越罕見。目前本種被列入華盛頓公約CITES附錄II保育類動物，野生族群貿易量受到管制。

== 飼育 ==
非洲岩蟒亦是常見的受飼養蛇類之一，願意接受定時餵食[[鼠類|鼠類]]及[[兔子|兔子]]。但是由於與其它蟒蛇相比，非洲岩蟒的身體顏色較為陰暗，性格也較不溫馴，故此其價值一般較低；而其巨大的體格及其貪吃的性情，亦令牠變得較難飼育，不適合初學者購養。

== 對人類的傷害 ==
非洲岩蟒甚少攻擊人類，雖然牠們擁有足夠殺死成年人的能力，但迄今為止由非洲岩蟒所造成的致命事件僅有很少，而且受害者大多是小孩。在1951年，一份[[烏干達|烏干達]]報章報導了一名13歲男童被非洲岩蟒所吞噬，不過該蟒蛇亦被外力迫其將男童吐出；1973年另一份報章指一名[[葡萄牙|葡萄牙]]籍軍人亦被發現遭蟒蛇所吞。1979年一條長4.5米的蟒蛇試圖將一名13歲男童（身高1.3米，體重45公斤）吞食，但因遭到男童友伴以石頭擲擊而將男童吐出；<ref name="Bra80">Branch WR, Hacke WD. 1980. A Fatal Attack on a Young Boy by an African Rock Python (''Python sebae''). Journal of Herpetology. 14:305-307.</ref>最後一宗較為人所知的事件發生於2002年的[[南非|南非]]，當時的受害人是一名10歲小孩。<ref name="Age">[http://www.theage.com.au/articles/2002/11/24/1037697985131.html Children terrified as python eats boy] at [http://www.theage.com.au/ The Age]</ref>
