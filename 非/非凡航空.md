'''非凡航空'''（'''{{lang|en|VIVA Macau}}'''）是一間曾以[[澳門|澳門]]為基地且已被停牌的[[廉價航空公司|廉價航空公司]]<ref>{{cite web |url=http://www.macaodaily.com/html/2010-03/31/content_447750.htm |title=存档副本 |accessdate=2010-03-30 |deadurl=yes |archiveurl=https://web.archive.org/web/20100405010127/http://www.macaodaily.com/html/2010-03/31/content_447750.htm |archivedate=2010-04-05 }}</ref>。

2009年，有非凡公司內部知情人士向[[澳門立法會|澳門立法會]][[議員|議員]][[區錦新|區錦新]]透露公司行政混亂且經營不善，公司內部有不少人收受高薪但無所作為，且不斷接收政府數以千萬計的「資助」（實際為貸款，屬政府公帑）以維持公司運作。7月31日，區錦新就政府資助非凡航空的問題向行政當局提出書面質詢，後來當局迫於議會的壓力停止繼續對非凡航空提供「資助」<ref name="sp01">{{cite news |author1=區錦新 |title=非凡事件暴露特區財政漏洞 兩億學費讓我們可學到甚麼？ |url=https://www.facebook.com/sonpou/posts/2087995481466682 |agency=訊報 |date=2018-08-02}}</ref><ref name="正01">{{cite news |title=區錦新促交代非凡事 |url=http://www.chengpou.com.mo/news/2010/5/10/2085.html |agency=正報 |date=2010-05-10}}</ref>。

由於[[澳門航空公司|澳門航空公司]]在2010年3月28日終止與非凡航空的分專營合約，非凡航空已失去空運經營人的牌照，亦被[[澳門民航局|澳門民航局]]要求停止銷售機票等商業活動<ref>{{Cite web |url=http://www.gcs.gov.mo/showNews.php?DataUcn=43787 |title=存档副本 |access-date=2012-05-21 |archive-url=https://web.archive.org/web/20160304204327/http://www.gcs.gov.mo/showNews.php?DataUcn=43787 |archive-date=2016-03-04 |dead-url=no }}</ref><ref>{{Cite web |url=http://www.gcs.gov.mo/showNews.php?DataUcn=43785 |title=存档副本 |access-date=2012-05-21 |archive-url=https://web.archive.org/web/20160304115507/http://www.gcs.gov.mo/showNews.php?DataUcn=43785 |archive-date=2016-03-04 |dead-url=no }}</ref>。2010年4月12日，非凡航空向[[澳門初級法院|澳門初級法院]]提出[[破產|破產]]請求，其後法官於同年9月13日宣告其破產。非凡航空拖欠總金額為11億4078萬5000多[[澳門元|澳門元]]的債務，當中拖欠澳門工商業發展基金的債務總額為2億1200萬元<ref name="正報01">{{cite news |title=「非凡」破產歸檔了！兩億公帑盡報銷 |url=http://www.chengpou.com.mo/cp-page/dailynews/?d=122766 |agency=正報 |date=2018-07-23 |access-date=2018-07-23 |archive-url=https://web.archive.org/web/20180723152204/http://www.chengpou.com.mo/cp-page/dailynews/?d=122766 |archive-date=2018-07-23 |dead-url=no }}</ref>。

非凡航空破產8年後，主理案件法官於2018年1月23日作出批示，根據訴訟法律規定，宣告將案件歸檔。同年7月，初級法院於公佈非凡航空破產案的判決已轉為確定，表示[[澳門政府|澳門政府]]已無法收回曾經向非凡航空借出的2.12億[[澳門元|澳門元]]公帑，而初級法院暗批特區工商基金不抓緊時機追究責任而導致如此後果<ref name="正報01"/><ref name="論盡1">{{cite news |title=公帑又冇咗兩億！ 初院︰非凡航空破產案已歸檔 |url=https://aamacau.com/2018/07/22/%e5%85%ac%e5%b8%91%e5%8f%88%e5%86%87%e5%92%97%e5%85%a9%e5%84%84%ef%bc%81-%e5%88%9d%e9%99%a2%ef%b8%b0%e9%9d%9e%e5%87%a1%e8%88%aa%e7%a9%ba%e7%a0%b4%e7%94%a2%e6%a1%88%e5%b7%b2%e6%ad%b8%e6%aa%94/ |agency=論盡媒體 |date=2018-07-22 |access-date=2018-07-23 |archive-url=https://web.archive.org/web/20180727133930/https://aamacau.com/2018/07/22/%e5%85%ac%e5%b8%91%e5%8f%88%e5%86%87%e5%92%97%e5%85%a9%e5%84%84%ef%bc%81-%e5%88%9d%e9%99%a2%ef%b8%b0%e9%9d%9e%e5%87%a1%e8%88%aa%e7%a9%ba%e7%a0%b4%e7%94%a2%e6%a1%88%e5%b7%b2%e6%ad%b8%e6%aa%94/ |archive-date=2018-07-27 |dead-url=no }}</ref>。目前該案件已由廉政公署接手調查。

== 歷史 ==
[[File:b-mav.jpg|200px]]
非凡航空的前身為[[澳門鷹揚航空服務有限公司|澳門鷹揚航空服務有限公司]]轄下的「WOW!Macau」，及後於2005年12月9日改名為「Viva Macau」。2006年5月，非凡航空公司與[[澳門航空|澳門航空]]之間的分專營權達成共識並獲通過，於2006年的12月16日啟航。此外，非凡航空正與澳門航空公司協商，以提供代碼共享航班服務，擴展大中華地區乘客量。

===政府貸款支持營運===
[[澳門特區政府|澳門特區政府]]通過工商業發展基金，一共向非凡航空貸款近2億元澳門幣<ref name=tdm20100115_140939>{{cite news|url=http://www.tdm.com.mo/c_radio/news/index.php?id=109132|title=政府向非凡航空貸款近二億|work=即時新聞|publisher=澳門廣播電視股份有限公司|accessdate=2010-03-27|archive-url=https://web.archive.org/web/20160305065953/http://www.tdm.com.mo/c_radio/news/index.php?id=109132|archive-date=2016-03-05|dead-url=no}}</ref>。澳門政府表示支持以澳門為基地的航空公司運作。時任財政司司長譚伯源指出，政府貸款非凡航空具有抵押，保證公帑安全，未來會視乎非凡的發展，考慮是否需要再支持。他又說，澳門作為旅遊城市，航空網絡的穩定性非常重要，保證客源不受影響。

4月6日下午6時起，[[旅遊危機處理辦公室|旅遊危機處理辦公室]]終止非凡航空事件的應變機制，關閉設於[[澳門機場|澳門機場]]處理非凡航空事宜的櫃台，同時停止本地及海外熱線服務。而退回機票款項追討事宜，由[[澳門消費者委員會|澳門消費者委員會]]提供協助<ref>{{cite web |url=http://www.macaodaily.com/html/2010-04/08/content_450762.htm |title=存档副本 |accessdate=2010-04-10 |deadurl=yes |archiveurl=https://web.archive.org/web/20100411051102/http://www.macaodaily.com/html/2010-04/08/content_450762.htm |archivedate=2010-04-11 }}</ref>。4月8日，[[經濟財政司|經濟財政司]]司長[[譚伯源|譚伯源]]表示，非凡航空已向[[中級法院|中級法院]]遞交索償申請<ref>{{cite web |url=http://www.macaodaily.com/html/2010-04/09/content_451053.htm |title=存档副本 |accessdate=2010-04-10 |deadurl=yes |archiveurl=https://web.archive.org/web/20100412140901/http://www.macaodaily.com/html/2010-04/09/content_451053.htm |archivedate=2010-04-12 }}</ref>。

2018年，主理案件法官於1月23日作出批示，根據訴訟法律規定，宣告將案件歸檔。同年7月，初級法院於公佈非凡航空破產案的判決已轉為確定，表示[[澳門政府|澳門政府]]已無法收回曾經向非凡航空借出的2.12億[[澳門元|澳門元]]公帑<ref name="論盡1">{{cite news |title=公帑又冇咗兩億！ 初院︰非凡航空破產案已歸檔 |url=https://aamacau.com/2018/07/22/%e5%85%ac%e5%b8%91%e5%8f%88%e5%86%87%e5%92%97%e5%85%a9%e5%84%84%ef%bc%81-%e5%88%9d%e9%99%a2%ef%b8%b0%e9%9d%9e%e5%87%a1%e8%88%aa%e7%a9%ba%e7%a0%b4%e7%94%a2%e6%a1%88%e5%b7%b2%e6%ad%b8%e6%aa%94/ |agency=論盡媒體 |date=2018-07-22 |access-date=2018-07-23 |archive-url=https://web.archive.org/web/20180727133930/https://aamacau.com/2018/07/22/%e5%85%ac%e5%b8%91%e5%8f%88%e5%86%87%e5%92%97%e5%85%a9%e5%84%84%ef%bc%81-%e5%88%9d%e9%99%a2%ef%b8%b0%e9%9d%9e%e5%87%a1%e8%88%aa%e7%a9%ba%e7%a0%b4%e7%94%a2%e6%a1%88%e5%b7%b2%e6%ad%b8%e6%aa%94/ |archive-date=2018-07-27 |dead-url=no }}</ref>。初級法院認為，特區工商基金其實早應該於2010年在澳門提起有關執行案的同時馬上抓緊時間到[[香港|香港]]，以透過司法途徑向被執行人追究有關的承兌保證責任，暗指特區工商基金因不抓緊時機到香港追究責任而導致債務無法被追回<ref name="正報01"/>。區錦新則質疑為何政府會選擇借貸給一間一直都是靠政府資助的公司，又認為當年有人揭發事件，迫使政府中止對非凡的資助，否則賠上的公帑不會只有兩億，是「不幸中的大幸」<ref>{{cite web |author1=李俊傑 |author2=蔡立盛 |title=區錦新:非凡航空一直靠政府資助過活 |url=http://www.mastvnet.com/news/Macao/2018-08-05/166469.html |website=澳亞網 |date=2018-08-05}}</ref>。

目前該案件已交由廉政公署接手調查。
