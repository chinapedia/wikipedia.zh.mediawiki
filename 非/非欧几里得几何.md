== 几何原本第五公设 ==
[[古希腊|古希腊]]数学家[[欧几里得|欧几里得]]的《[[几何原本|几何原本]]》提出了五条[[公设|公设]]：

# 从一[[點|點]]向另一[[點|點]]可以引一条直线。
# 任意[[线段|线段]]能无限延伸成一条直线。
# 给定任意线段，可以以其一个端点作为[[圆心|圆心]]，该线段作为[[半径|半径]]作一个[[圆|圆]]。
# 所有[[直角|直角]]都[[相等|相等]]。
# 若两条直线都与第三条直线相交，并且在同一边的[[内角|内角]]之和小于两个直角，则这两条直线在这一边必定相交。

长期以来，[[数学家|数学家]]们发现[[平行公設|第五公设]]和前四个公设比较起来，显得文字叙述冗长，而且也不那么显而易见。有些数学家还注意到欧几里得在《几何原本》一书中直到第29个命题中才用到，而且以后再也没有使用。也就是说，在《几何原本》中可以不依靠第五公设而推出前28个命题。因此，一些数学家提出，第五公设能不能不作为公设，而作为定理？能不能依靠前四个公设来证明第五公设？这就是几何发展史上最著名的，争论了长达两千多年的关于“平行线理论”的讨论。由于证明第五公设的问题始终得不到解决，人们逐渐怀疑证明的路子走的对不对？第五公设到底能不能证明？

== 罗巴切夫斯基几何 ==
1820年代，[[沙俄|俄国]][[喀山大学|喀山大学]]教授[[罗巴切夫斯基|罗巴切夫斯基]]在证明第五公设的过程中，他走了另一条路子。他提出了一个和欧氏[[平行公設|平行公設]]相[[矛盾|矛盾]]的[[命题|命题]]，用它来代替第五公设，然后与欧氏几何的前四个公设结合成一个[[公理系统|公理系统]]，展开一系列的推理。他认为如果这个系统在基础的推理中出现矛盾，就等于证明了第五公设。此即数学中的[[反证法|反证法]]。但是，在他极为细致深入的推理过程中，得出了一个又一个在直觉上匪夷所思，但在[[逻辑|逻辑]]上毫无矛盾的命题。最后，罗巴切夫斯基得出两个重要的结论：

# 第五公设不能被证明。
# 在新的公理体系中展开的一连串推理，得到了一系列在逻辑上无矛盾的新的定理，并形成了新的理论。这个理论像欧氏几何一样是完善的、严密的[[几何学|几何学]]。

这种几何学被称为罗巴切夫斯基几何，简称[[罗氏几何|罗氏几何]]。这是第一个被提出的非欧几何学。从罗氏几何学中，可以得出一个极为重要的、具有普遍意义的结论：逻辑上不矛盾的一些[[公理|公理]]都有可能提供一种几何学。

== 鲍耶氏和高斯的贡献 ==
几乎在罗巴切夫斯基创立非欧几何学的同时，匈牙利数学家[[鲍耶·雅诺什|鲍耶·雅诺什]]也发现了第五公设不可证明和非欧几何学的存在。鲍耶在研究非欧几何学的过程中也遭到了家庭、社会的冷漠对待。他的父亲——数学家{{le|鲍耶·法尔卡什|Farkas Bolyai}}认为研究第五公设是耗费精力劳而无功的蠢事，劝他放弃这种研究。但鲍耶·雅诺什坚持为发展新的几何学而辛勤工作。终于在1832年，在他的父亲的一本著作裡，以附录的形式发表了研究结果。

[[高斯|高斯]]也发现第五公设不能证明，并且研究了非欧几何。但是高斯害怕这种理论会遭到当时教会力量的打击和迫害，不敢公开发表自己的研究成果，只是在书信中向自己的朋友表示了自己的看法，也不敢站出来公开支持罗巴切夫斯基、鲍耶他们的新理论。

== 非欧几何分类 ==
[[File:Triangle_on_spherical_plane.png|thumb]]
按几何特性（曲率），现存非欧几何的类型可以概括如下：

* 坚持第五公设，引出'''[[欧几里得几何|欧几里得几何]]'''。
* 以“可以引最少兩条平行线”为新公设，引出'''[[罗氏几何|罗氏几何]]'''（双曲几何）。
* 以“一条平行线也不能引”为新公设，引出'''[[黎曼几何|黎曼几何]]'''（椭圆几何）。

这三种几何学，都是常曲率空间中的几何学，分别对应曲率为0、负常数和正常数的情况。

如果完全去掉第五公设，就得到更加一般化的{{le|绝对几何|Absolute geometry}}。这种几何不仅可以囊括前面提到的三种几何，而且允许空间的不同位置有不同的曲率。[[黎曼几何|黎曼几何]]是描述任意维数任意弯曲的绝对几何空间的一种微分解析几何学。

一般来讲，非欧几何有广义、狭义、通常意义三个不同含义：
* 广义的非欧几何：泛指一切和欧几里得几何不同的几何学。
* 狭义的非欧几何：只是指[[罗氏几何|罗氏几何]]或[[黎曼几何|黎曼几何]]。
* 通常意义的非欧几何：指[[罗氏几何|罗氏几何]]和[[黎曼几何|黎曼几何]]二者。
