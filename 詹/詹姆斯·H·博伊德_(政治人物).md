'''詹姆斯·H·「吉米」·博伊德'''（{{lang-en|'''James H. -{“Jimmy”}- Boyd'''}}，{{bd|1906年|11月5日|1974年|4月11日}}）是一名[[大西洋城|大西洋城]][[政客|政客]]和犯罪{{link-en|政治老闆|Political boss}}，他為腐敗的[[大西洋縣|大西洋縣]]{{link-en|財務主管|Treasurer}}[[伊努·L·強森|伊努·L·強森]]和[[新澤西州參議院|新澤西州參議員]][[法蘭克·S·法利|法蘭克·S·法利]]工作。

== 早年 ==
詹姆斯·博伊德在1906年11月5日出生於[[美國|美國]][[新澤西州|新澤西州]]的大西洋城。<ref name="Is Jimmy Darmody based on a real person?">{{Cite web |url = http://www.historyvshollywood.com/reelfaces/boardwalkempire.php |title = BOARDWALK EMPIRE (2010-2014) |publisher = HistoryvsHollywood.com |language = en}}</ref>

== 禁酒令時期 ==
博伊德於1920年[[禁酒令|禁酒令]]生效期間在[[大西洋城麗思卡爾頓酒店|大西洋城麗思卡爾頓酒店]]當一名服務生。但不久後，博伊德開始走上腐敗的大西洋城財務主管和{{link-en|南澤西|South Jersey}}[[愛爾蘭幫|愛爾蘭幫]]老大[[伊努·L·強森|伊努·L·強森]][[努基·強森的組織|組織]]的指揮鏈中。博伊德和強森在強森和[[查理·盧西安諾|查理·盧西安諾]]正在組成{{link-en|聯合會 (集團)|The Combined (Group)|七大集團}}的那時會面，他們便立刻喜歡彼此。強森開始培養博伊德成為他組織中的下一個老大，他後來成為強森的得力助手。他管理[[地下酒吧|地下酒吧]]、[[妓院|妓院]]、非法賭場和{{link-en|彩票賭博|Numbers game|彩票勾當}}。博伊德也是強森和未來老大法利的最高{{link-en|黑幫執法者|Mob enforcer|執法者}}。

=== 個人生活 ===
博伊德嫁給了Marie E. Boyd。<ref name="$10,000 Scholarship" />

== 法蘭克·法利 ==
強森在1941年被判逃稅罪後，[[新澤西州參議院|新澤西州參議員]][[法蘭克·S·法利|法蘭克·S·法利]]便控制了大西洋城的政治機器。為了在新舊政權之間建立一條「橋樑」，博伊德被任命為在大西洋城中所有業務的監督者。他還成為第四選區共和黨俱樂部（Fourth Ward Republican Club）的執行主席、Freeholders大西洋委員會助理書記和書記員，以及第四選區的老大。在法利的政權下，博伊德被稱為一個凶狠的人。

=== 第二次世界大戰 ===
博伊德在[[第二次世界大戰|第二次世界大戰]]期間參軍。 從1942年至1945年，博伊德是{{link-en|第79步兵師 (美國)|79th Infantry Division (United States)|第79步兵師}}的士兵。 他曾在[[大君主作戰|諾曼底戰役]]，[[諾曼第戰役|北法戰役]]，[[盟軍由巴黎到萊茵的推進|萊茵蘭戰役]]和[[中歐會戰|中歐]]服役。

== 逝世 ==
博伊德於1974年4月11日在家中因癌症去世，享年67歲。<ref>{{Cite web |url = https://news.google.com/newspapers?nid=1955&dat=19740412&id=KgQrAAAAIBAJ&sjid=x5cFAAAAIBAJ&pg=6073,895020&hl=en |title = "Political Figure-James Boyd dies" |date = 1974-04-12 |language = en}}</ref><ref name="$10,000 Scholarship" /><ref name="Is Jimmy Darmody based on a real person?" />他的妻子在2006年去世之前，在{{link-en|大西洋海角社區學院|Atlantic Cape Community College}}設立了「詹姆斯·H·博伊德紀念獎學金」（James H. Boyd Memorial Scholarship），價值10,000美元。

== 流行文化 ==
[[HBO|HBO]]電視劇《[[酒私風雲|酒私風雲]]》首兩季中的其中一個主要角色[[吉米·達摩狄|吉米·達摩狄]]，某程度上根據博伊德而創作出來的虛構化版本，由[[麥可·彼特|麥可·彼特]]飾演。他是一名[[第一次世界大戰|第一次世界大戰]]老兵，有一位妻子和兒子。在劇集中，他的父親是Louis "Commodore" Kaestner（基於[[路易斯·庫爾勒|路易斯·「準將」·庫爾勒]]），而他的導師是腐敗的[[大西洋縣|大西洋縣]]財務主管和政治老闆[[努基·湯普森|伊努·「努基」·湯普森]]（基於[[伊努·L·強森|伊努·L·強森]]）。

劇集中，達摩狄的伙伴及朋友包括：[[查理·盧西安諾|查理·盧西安諾]]、[[邁爾·蘭斯基|邁爾·蘭斯基]]、[[艾爾·卡彭|艾爾·卡彭]]、[[強尼·托里奧|強尼·托里奧]]、[[理查·哈洛|理查·哈洛]]和[[米奇·道爾|米奇·道爾]]（基於[[米奇·達菲|米奇·達菲]]）。
