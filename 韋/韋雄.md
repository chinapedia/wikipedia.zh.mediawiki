'''韋雄'''（{{lang|en|'''Philip Wai}}，{{bd|1981年|8月3日|catIdx=韋}}），[[香港|香港]]男歌手、演員及音樂人。

== 簡歷 ==
韋雄三歲時隨家人移民到[[加拿大|加拿大]][[多倫多|多倫多]]定居，於2005年畢業於[[多倫多大學|多倫多大學]]經濟系。其後在2007年回流[[香港|香港]]發展。2008年，他簽約[[香港|香港]][[星娛樂|星娛樂]]而出道，為2009年樂壇新人，曾經奪得2002年《[[新秀歌唱大賽多倫多選拔賽|新秀歌唱大賽多倫多選拔賽]]》冠軍。香港音樂監製、作曲家及編曲家[[韋景雲|韋景雲]]是其哥哥<ref>{{cite web |title=專訪加拿大幫創作人 - 韋雄、韋景雲 |url=http://teenpower1.rthk.hk/tptv/inside.php?id=267 |work=[[TeenPower|TeenPower]] TV |publisher=[[香港電台|香港電台]] |accessdate=2013-10-30 |date=2010-02-15}}</ref>。

韋雄曾與[[鍾嘉欣|鍾嘉欣]]合唱《戀愛令人心痛》一曲，讓他獲得《[[新城勁爆頒獎禮|新城勁爆頒獎禮]]》「新登場男歌手」獎項。2012年，他更獲邀赴[[台灣|台灣]]參加歌唱選秀節目《[[華人星光大道2|華人星光大道2]]·強敵踢館賽》<ref>{{cite web |title=韋雄台灣Sing戰 壓力大 |url=http://orientaldaily.on.cc/cnt/entertainment/20121207/00282_085.html |work=[[東方日報_(香港)|東方日報]] |publisher=東方日報 |accessdate=2012-12-07 |date=2012-12-07}}</ref>。

另外，他曾獲[[香港交通安全隊|香港交通安全隊]]委任為交通安全大使 - 推廣及宣傳署高級助理隊監<ref>{{cite web|title=香港交通安全隊 - 推廣及宣傳署 |url=http://www.rsa.org.hk/market.php |publisher=香港交通安全會}}</ref>。2014年11月，他加盟 [[Unleash_Entertainment|Unleash Entertainment]] ，成为旗下的声乐指导及配唱监制。
