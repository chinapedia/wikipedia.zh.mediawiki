'''深夜宅配'''是在[[ラジオ関西|ラジオ関西]]的[[アニたまどっとコム|アニたまどっとコム]]內、從2013年4月5日到2015年12月26日放送的[[廣播電台|廣播電台]]。

== 概要 ==

節目被命名為深夜的宅配、由化身為「宅配員」的主持人內山及山本、為聽眾送上各式各樣的內容<ref>{{citenews | url = http://anitama.com/saishininfo/index.html#spring2013 | title = アニたまどっとコム２０１３春の新番組情報② | publisher = ラジオ関西 | date = 2013-04-05| accessdate = 2013-05-13}}</ref> 。

節目的標誌、由內山喜歡的動物「鸚鵡」乘坐在山本喜歡的動物「猩猩」的肩上所型成。

== 單元 ==
; '''まよデリ業務聯絡'''（第1回 - ）<br />
（日文：まよデリ業務連絡）
: 介紹普通信件的單元。


; のじょみ米其林指南（第1回 - 第97回）<br />
（日文：のじょみしゅらん）
: 募集各地區美食情報、試吃有辦法調貨過來的食物、由山本吃完的感想來決定是否記載在原創小冊子『のじょみしゅらん』的單元。


; HOT or COOL便（第1回 - 第96回）<br />
（日文：ホット or クール便）
: 為了需要努力卻沒有動力的人送上加油訊息的單元。鼓勵方式分為超熱情的「HOT」及非常冷靜的「COOL」兩種。會是哪一種由聲音轉盤決定。


; 為您宅配難以出口的話語（第1回 - 第97回）<br />
（日文：いいにくいメッセージお届けします）
: 有想要說出來的話卻無法說出口、由主持人代替您說出口的單元。


; 嚴禁敗北 宵夜決鬥（第98回 - ）<br />
（日文：負け厳禁 お夜食バトル）
: 賭上宵夜美食家的名譽、兩位主持人進行決鬥的單元。贏家享受宵夜、輸家必須接受收拾殘局的懲罰。


; 傳遞到心中 まよVoice （第98回 - ）<br />
（日文：ハートに届け まよボイス）
: 兩位主持人各自所寫的文章出來的單元。哪一篇文章更加容易傳遞到內心深處由聽眾來決定、結果會在下下回發表。


; 深夜的垃圾回收（第99回 - ）<br />
（日文：まよなかの廃品回収）
: 聽眾透過郵件寄來不需要的過去・想要消除的過去(垃圾)、由主持人發送鼓勵的回信的單元。

=== 試作單元 ===
; まよデリ 夜禮<br />
（日文：まよデリ 夜礼）
: 主持人・聽眾發表從播放日開始算起一週間的目標、並驗證是否如預期達成的單元。

; 宵夜限定 銀座一丁目 心血來潮食堂<br />
（日文：夜食限定 銀座1丁目 気まぐれ食堂）
: 挑戰料理宵夜的單元、誰來挑戰由抽籤決定。
