《'''深夜烘焙坊'''》（{{lang-ja|真夜中のパン屋さん}}），是日本女作家大沼紀子於2011年以深夜開始營業的麵包店為背景創作的小說。<ref name="ddnavi71629">{{lang|ja|{{Cite web |url = http://ddnavi.com/news/71629/ |title = ラノベとも違う！ 今人気の“キャラ立ち小説”とは？ |publisher = ダ・ヴィンチ電子ナビ - メディアファクトリー |accessdate = 2013-05-07 }}}}</ref>2013年改編成電視劇。

== 劇情概要 ==
每天深夜11點至清晨5點營業的「暮林烘焙坊」，提供各式美味麵包，等待著顧客上門品嚐。麵包店營業時間相當特殊，上門的顧客也都不是普通的人物，各自有不同的煩惱，諸凡事業不順、感情問題、與社會脫節…。這裡的麵包彷彿有一股神奇的魔力，當他們享用麵包的同時，也會說出自己的心事，藉此紓發自己鬱悶的心情…<ref>[http://www.goldsuntv.com.tw/program_deils.php?n_id=22 深夜中的麵包店]</ref>

2013年改編電視劇正式開拍，4月28日至6月16日於NHK BS Premium頻道播出，同年11月5日至12月6日NHK綜合台播出，由[[瀧澤秀明|瀧澤秀明]]、[[桐山照史|桐山照史]]及[[土屋太鳳|土屋太鳳]]主演。國興衛視於2014年5月19日起以《[[深夜中的麵包店|深夜中的-{zh-hans:面;zh-hant:麵;}-包店]]》的劇名在台灣播出，播出時間為每週一至週四晚間10點至11點。

=== 登場人物 ===
==== 主要人物 ====
* 暮林陽介：[[瀧澤秀明|瀧澤秀明]]（香港配音：[[梁偉德|梁偉德]]） 
: 暮林烘焙坊負責人兼麵包學徒。
* 柳弘基：[[桐山照史|桐山照史]]（關西傑尼斯Jr.）（香港配音：[[曹啟謙|曹啟謙]]） 
: 暮林烘焙坊麵包師傅。
* 篠崎希實：[[土屋太鳳|土屋太鳳]]（少女時期：[[信太真妃|信太真妃]]）（香港配音：[[羅孔柔|羅孔柔]]） 
: 高中生。母親律子為了與新男友展開新生活，拜託美和子以同父異母姐姐的身分照顧希實。

==== 暮林烘焙坊 ====
* 暮林美和子：[[伊藤步|伊藤步]]（香港配音：[[柚子蜜_(配音員)|柚子蜜]]） 
: 暮林烘焙坊麵包師傅。本姓久瀨，暮林的妻子。在希實來到烘焙坊的半年前因意外過世。

==== 烘焙坊的客人 ====
* 水野木靈：藤野大輝（香港配音：[[凌晞|凌晞]]） 
: 三軒茶屋南小學3年2班學生。單親家庭出身，與母親同住。
* 斑目裕也：六角精兒（香港配音：[[陳永信|陳永信]]） 
: 烘焙坊的常客，電視劇劇本作家。暗戀著住在附近的高遠由香里（平川由梨飾演），經常用望遠鏡監視她的一舉一動。
* 蘇非亞：[[室剛|室剛]]（香港配音：[[黃啟昌|黃啟昌]]） 
: 人妖酒吧老闆。
* 老年客人：[[五月晴子|五月晴子]]（香港配音：[[陸惠玲|陸惠玲]]）
* 松川幸夫：菜枝豆（香港配音：[[盧國權|盧國權]]）
* 高橋祐二：中嶋班恩（香港配音：[[張炳強|張炳強]]）
* 池澤誠：深澤敦（香港配音：[[劉奕希|劉奕希]]）
: 以上3人為到烘焙坊買麵包的上班族。

==== 都立櫻原高中 ====
* 三木涼香：[[小島藤子|小島藤子]]（香港配音：[[何寶珊|何寶珊]]） 
: 希實的幼時玩伴。班上呼風喚雨的風雲人物，但心中其實相當空虛寂寞。
* 蒲原瑞希：若菜葵（香港配音：[[陳皓宜|陳皓宜]]） 
* 久保田舞：朝日梨帆（香港配音：[[黃昕瑜|黃昕瑜]]）
: 涼香的朋友。
* 弓坂悦郎：[[竹内壽|竹内壽]]（香港配音：[[黃積權|黃積權]]） 
: 班長。希實與涼香的同班同學，很喜歡希實。
* 長谷川公男：[[大内厚雄|大内厚雄]]（香港配音：[[張炳強|張炳強]]） 
: 班導師。

==== 其他 ====
* 水野織繪：[[前田亞季|前田亞季]]（香港配音：[[曾佩儀|曾佩儀]]）
: 木靈的母親。護理師。
* 三木淑枝：[[田中律子|田中律子]]（香港配音：[[沈小蘭|沈小蘭]]）
: 涼香的母親。相當溺愛自己的女兒。
* 篠崎律子：[[友坂理惠|友坂理惠]] （香港配音：[[鄭麗麗|鄭麗麗]]）
: 希實的母親。

=== 各集登場人物 ===
* 一郎：中原和宏（第3－4集）
* 由井綾乃：野村麻純（第5－6集、完結篇）（香港配音：[[劉惠雲|劉惠雲]]）
: 佳乃的雙胞胎姊姊。
* 由井佳乃：野村麻純（一人分飾二角）（第6集）（香港配音：[[劉惠雲|劉惠雲]]）
: 弘基的前女友，綾乃的雙胞胎妹妹。刻意以美色接近多名男性，從事假結婚真詐欺的非法行為。最後向警方自首。
* 男子A：相島一之（第5－6集）（香港配音：[[潘文柏|潘文柏]]）
* 男子B：三谷昌登（第5－6集）（香港配音：[[劉奕希|劉奕希]]）
: 以上2人皆為結婚詐欺案的受害人。
* 美作元史：[[植草克秀|植草克秀]]（第7集）（香港配音：[[葉振聲|葉振聲]]）
: 醫師。木靈的父親，與織繪為外遇關係，實為有婦之夫。
