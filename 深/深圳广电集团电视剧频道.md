'''深圳广电集团电视剧频道'''（简称'''深圳电视剧'''）是[[深圳广播电影电视集团|深圳广播电影电视集团]]旗下的一条本土化影视剧[[电视|电视]]频道，每天24小时不间断广播。该频道于[[中华人民共和国|中华人民共和国]][[广东省|广东省]][[深圳市|深圳市]]播放<ref>{{Cite web|url=http://www.s1979.com/shenzhen/201012/3010068130.shtml|title=电视剧频道节目直播|publisher=中国时刻网|deadurl=yes|archiveurl=https://web.archive.org/web/20141224080559/http://www.s1979.com/shenzhen/201012/3010068130.shtml|archivedate=2014-12-24}}</ref>。

== 历史 ==
* 1994年，本台开播，当时为深圳电视台第二套节目，使用48频道播出。
* 2002年，本台更名为深圳电视台电视剧频道。
* 2008年8月，深圳市国家标准[[地面数字电视|地面数字电视]]开播，该台数字版本可于深圳大部分地区以无线方式接收，其频率为562[[兆赫|兆赫]]<ref>{{Cite web |url=http://news.sznews.com/content/2008-08/05/content_3160075.htm |title=深圳国标地面数字电视开通 市民可无线收看高清电视 |date=2008-08-05 |publisher=深圳新闻网 |accessdate=2014-12-13 }}</ref>。
* 2014年，该频道实现高标清同播。
* 2016年12月尾，该频道与[[深圳广电集团都市频道|都市频道]]的标清版改为16:9格式播出。

== 播出节目==
除早晨七点左右播出[[生活|生活]]类节目，并在每节电视剧之间插入[[电视购物|电视购物]]节目外，该频道几乎全日播出影视剧。[[凌晨|凌晨]]时段主要播出怀旧影片。

===电视剧时段<ref>[http://tv.cutv.com/index.html?channel=ZwxzUXr 城视网]</ref>===
*《老片坊通宵》：每天凌晨时段播出，主要播放中国[[战争片|战争片]]、革命片。
*《榕树海剧场》：每天早晨时段播出。
*《青春互动剧场》：每天上午时段播出。
*《月亮湾剧场》：每天下午时段播出。
*《薰衣草剧场》：每天下午时段播出。
*《黄金剧场》：每天晚间黄金时段播出，多为首播电视剧。
*《满天星剧场》：每天晚间时段播出，部分为[[韩国|韩国]]、[[香港|香港]]外购剧。
