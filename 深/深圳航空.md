'''深圳航空有限责任公司'''（'''简称：深圳航空，英文名：Shenzhen Airlines'''）（IATA：'''ZH'''，ICAO：'''CSZ'''）是一家位于[[广东|广东]][[深圳|深圳]]的航空公司，其基地为[[深圳宝安国际机场|深圳宝安国际机场]]，现为[[中国国际航空|中国国际航空]]最大规模的控股子公司，也与母公司一样为[[星空联盟|星空联盟]]成员。

==历史==
=== 早期经营及发展 ===
'''深圳航空'''於1985年9月23日开始筹办，1989年10月11日以“'''深圳市航空公司'''”名字注册，1992年12月24日定名为“'''深圳航空公司'''”，1993年9月17日正式开航，当时的航司代码为IATA：'''4G'''<ref>{{Cite === 李泽源入主时期（2005-2010） ===
2005年5月29日[[李泽源|李泽源]]所拥有的深圳匯潤投资联合哈尔滨億陽集團以27.2億元擊敗二股东国航等對手，取得深圳航空65％的控股權，令深圳航空一度成为一家民营航空公司。6月10日，[[波音公司|波音公司]]在[[西雅图|西雅图]]向深圳航空公司交付了第一架[[波音737|波音737-900]]，由此深航成为第一家购买该款机型的中国航空公司。2007年传出意向购买中国拥有自主知识产权的[[ARJ21|ARJ21]]支线飞机，但告吹。

深航在李泽源掌舵期间，高速扩张，先是与汉莎航空等企业开办了中国第一家中外合资的货运航空——[[翡翠航空|翡翠航空]]，又与美国梅莎航空等合资成立了[[鲲鹏航空|鲲鹏航空]]；然而子公司经营都不甚理想。再加上公司管理混乱，导致深航由盈转亏。

截至2008年12月底，深航全年除稅前及除稅后净虧損分別7,457萬元及3,126萬元，而截至2009年12月底的全年除稅前及除稅后净虧損大幅增加逾10倍，分別為7.69億元及8.64億元，而期內深航的股東權益净虧損達20.67億元。

2009年11月30日下午，新华网发布简短消息称，深航高级顾问李泽源因涉嫌经济犯罪，正在接受公安机关调查。在深航内部，李泽源被员工称为“大老板”，但他的头衔是高级顾问。深航董事长由李泽源的儿子李默担任。

=== 2010年及之后发展 ===
2010年3月22日中国国际航空出資6.82億元人民幣增持深圳航空股權，由原先25%增至51%，重新成為其控股股東。[[深圳國際|深圳國際]]屬下全資子公司全程物流亦同時斥資3.48億元注入深圳航空，持股量由10%提高至25%。深航也由此变回国有股份制航空公司。至於原大股東匯潤投資，其破產管理人因未有注資，股權被攤薄至24%。在[[民航局|民航局]]局长李家祥于2010年8月26日主持召开的全行业航空安全紧急电视电话会议上，首次披露了在2008年至2009年的行业调查中检查出一批资质造假的飞行员情况。其中，涉及深圳航空飞行员103名。<ref>{{cite web |url=http://www.lsol.com.cn/html/zhongwairedian/gangaotairedian/20100907/34857.html |title=《103名深航飞行员被曝资质造假 深航称不知情》 |accessdate=2011-12-04 |publisher=浙江在线 |language=zh-hans }}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

2011年5月9日[[深圳國際|深圳國際]]發出公告，旗下全資子公司全程物流已全面收購匯潤投資持有的24%股權，作價為788萬元人民幣，令其持股量提高至49%，成為深圳航空第二大股東。而匯潤投資宣告正式破產。同年，深圳航空獲選為[[2011年世界大學生夏季運動會|第26届世界大学生运动会]]指定航空公司。

[[星空聯盟|星空聯盟]]在2011年7月6日和深圳航空簽署了初步協議，正式邀請深航加入星空聯盟，預計將於2012年年底正式加盟，成為自母公司[[中国国际航空|中国国际航空]]外另一位來自中國的星空聯盟成員。<ref>{{cite web|url=http://cn.wsj.com/gb/20110706/BCHc-001897.asp|title=《星空聯盟與深圳航空簽署初步加盟協議》|accessdate=2011-07-06|publisher=華爾街日報中文版|language=zh-hans|archiveurl=https://web.archive.org/web/20160306070221/http://cn.wsj.com/gb/20110706/BCHc-001897.asp|archivedate=2016-03-06}}</ref>

2012年11月29日深圳航空正式加入[[星空聯盟|星空聯盟]]，並於同日將常旅客計劃由自有的尊鹏俱乐部（更早之前为金鹏知音）<ref>{{Cite web|title=常旅客|url=https://web.archive.org/web/20010710050425/http://www.shenzhenair.com/wsdp/clk/index.htm|work=web.archive.org|accessdate=2019-04-06}}</ref>變更為国航系的鳳凰知音。<ref name=":0" />

2018年第九屆世界航空公司排行榜，深圳航空獲「世界十佳美麗空姐」榜首。<ref>[http://www.chinatimes.com/realtimenews/20180516004517-260409 世界航空排行]</ref>

== 股东及控股权变迁 ==
1992年11月23日，[[中国国际航空公司|中国国际航空公司]]，深圳市南方通发实业公司（代表[[深圳市政府|深圳市政府]]）、[[华侨城集团|深圳经济特区华侨城经济发展总公司]](代表[[香港中旅集团|香港中旅集团]])和中银信托投资公司（[[中国银行|中国银行]]下属信托公司）签订了《联合经营深圳航空公司合同》，正式创立深圳航空。最初深圳航空公司注册资本为人民币5000万元，总投资额为人民币3亿元，其中深圳市南方通发实业公司占总投资额的10％，华侨城公司占40％，中银信托占25％，中国国航占25％。 <ref name=":1">{{Cite book|title=深圳市交通运输志|last=深圳市交通运输志编纂委员会|first=|publisher=方志出版社|year=2001年09月01日|isbn=7-80122-699-2|location=|pages=233}}</ref>

第一次股份变动发生在1996年底，当时深圳市南方通发实业公司所持有的股份转让给[[深圳市国资委|深圳市投资管理公司]]，另由于中银信托投资公司由于经营不善造成资不抵债，于1995年10月被[[人民银行|人民银行]]接管<ref>{{Cite web|title=信托业发展血泪史！五次清理整顿，中银信托...|url=http://trust.jrj.com.cn/2018/01/10081523922215.shtml|work=金融界|accessdate=2019-04-08}}</ref>，其所持有深航股份后被[[广东发展银行|广东发展银行]]收购。

第二次次股份变动发生在2000年8月，华侨城持有的40%股份被转让给当时广东发展银行旗下资产经营公司广东广控集团，广东发展银行连同旗下的广控集团遂一度成为深圳航空公司的大股东。这也开创了中国内地金融企业控股航空企业的先河<ref name=":1" />

到了2004年2月，新修订的《中华人民共和国商业银行法》规定，商业银行系统不得向非银行金融机构和企业投资，深航原实际大股东——广东发展银行不得不转让手中的股份。<ref name=":4" />2005年5月，深航开始改制并转让占比多达65%的股权（这部分股权原为广东发展银行旗下广东广控集团及其关联公司所有<ref name=":2">{{Cite web|title=深航去民营化 民营航空进退失据-搜狐财经|url=http://business.sohu.com/20091210/n268828078.shtml|work=business.sohu.com|accessdate=2019-04-08}}</ref>），两家名不见经传的民营公司深圳汇润投资有限公司（汇润投资是李泽源专为竞拍深航股份而成立的项目公司，注册资本仅1000万元，由李宜时（即李泽源）、赵丽、秦畹江、宋祖玉四名自然人股东发起，李宜时个人持股89%。）和哈尔滨亿阳集团（以通信产业起家，旗下拥有上市公司[[亿阳信通|亿阳信通]]。其老板邓伟是全国政协委员。）以27.2亿元的价格通过竞拍，一举击败包括中国国航、深圳国资委、[[中信集團|中信集团]]等大型国企以及淡马锡、花旗银行在内的诸多巨头，获得深航65%的控股权，深航由此从国有地方航空公司变身成为中国最大的民营航空公司。李泽源也由此正式走上前台，一度成为深航的大股东及掌舵者，在2009年11月30日之前的深航重大运作中，李泽源都是幕后操盘者。<ref name=":3">{{Cite web|title=李泽源事件追踪：或起因深圳航空“民营化”--中国民航新闻网--人民网|url=http://finance.people.com.cn/caac/GB/114104/10526059.html|work=finance.people.com.cn|accessdate=2019-04-08}}</ref>

但不久爆发弊案，李泽源能入主深航，靠的是其幕后人是[[新華保險|新华人寿]]原董事长关国亮（2007年11月，因违规挪用资金等问题被司法机关调查）。2005年，关国亮通过其控制的隆鑫公司，巨额拆借8.16亿元给汇润投资，用以交付收购深航股权的首期款。<ref name=":3" />之后李泽源除了利用个人关系延交本应在2005年8月15日前支付高达13.6亿元50%的股权款外，又在2005年~2009年之间涉嫌与5位高管多次利用职务上的便利挪用深航资金，累计高达20.3亿元，用于为汇润等公司偿还债务（主要是收购深航应交的股权款）以达到其“空手套白狼”的目的，这直接造成深航在这之间几年的巨额亏损<ref>{{Cite web|title=深圳航空原实际控制人李泽源涉嫌挪用20亿被公诉_财经_凤凰网|url=http://finance.ifeng.com/roll/20130407/7868472.shtml|work=finance.ifeng.com|accessdate=2019-04-08}}</ref>。截至李泽源被调查时，汇润投资尚有20%的深航股权款5.44亿元未能付清而仍被抵押在广控集团。<ref name=":2" /><ref name=":4">{{Cite web|title=深航27亿收购案解谜：李泽源关国亮空手套白狼|url=http://finance.sina.com.cn/chanjing/gsnews/20130418/133415193075.shtml|work=finance.sina.com.cn|accessdate=2019-04-08}}</ref>

2009年11月30日下午，李泽源因涉嫌经济犯罪，被公安机关带走调查。同年12月1日，深航当时二股东——国航的副总裁樊澄被任命为深航党委书记，全面接管党委领导班子。此后深圳航空实际由中国国航的控制管理。

2010年3月22日中国国际航空正式宣布出資6.82億元人民幣增持深圳航空股權，由原先25%增至51%，重新成為其控股股東。[[深圳國際|深圳國際]]（[[深圳市国资委|深圳市国资委]]所属企业）屬下全資子公司全程物流亦同時斥資3.48億元注入深圳航空，持股量由10%提高至25%。深航也由此变回国有股份制航空公司，并正式成为中国国际航空的子公司。至於原大股東匯潤投資，其破產管理人因未有注資，股權被攤薄至24%。而到了2011年5月9日，[[深圳國際|深圳國際]]發出公告，旗下全資子公司全程物流已全面收購匯潤投資持有的24%股權，作價為788萬元人民幣，令其持股量提高至49%，成為深圳航空第二大股東。

目前深航的股权结构为：中国国际航空占有51%，代表深圳市政府的[[深圳國際|深圳國際]]屬下全資子公司全程物流则占有49%。

=== 航徽释义 ===
[[File:B-2971_Boeing_B.737_Shenzhen_Airlines_(7182839429).jpg|缩略图]]
深圳航空的现有標誌是“民族之鵬”。“民族之鵬”是中國傳統文化和現代文化集合的圖騰，標誌造型氣勢磅礴，沉著矯健，呈高瞻遠矚，胸懷萬物，根基穩固之三態，一為睿智定乾坤，二是同心創輝煌，三生萬物盛千里。代表深圳航空“沉穩，誠信，進取”的理念。

而深航的第一代航徽也是取意鲲鹏，其恰似鲲鹏两只巨大的翅膀，双翼合并，作飞翔之势，成锐意进取之态。双翼宛如飞机前缘剖面造型，双翼腾起，直击长空，形成稳定、流畅、动态十足的气流形态，充分反映了深航的行业特征。环形象征太阳，喻义阳；双翼代表月亮，喻义阴。阴阳相融，日月同辉，泽及四方，体现了深航"服务大众，奉献社会"的宗旨。以清新的绿色为背景蕴育着无限生机，展示出深航生机勃勃，欣欣向荣的景象。外面的环形同时象征着深航团结协作，万众一心的企业文化。航徽总体如"鲲鹏展翅同风起，扶摇直上九万里"的意境，象征深航前途不可限量。<ref>{{Cite web|title=航徽释义|url=https://web.archive.org/web/20010422031044/http://www.shenzhenair.com/ljsh/hhsy/hhsy.html|work=web.archive.org|date=2001-04-22|accessdate=2019-04-06}}</ref>

=== 企业精神及口号 ===
深航第一代的企业口号是"'''深情一片，航程万里'''"，该口号凝聚着员工对公司的认同，社会对公司的支持以及公司对员工和社会的回报，这样一种解不开的情结，昭示着深航前途无可限量<ref>{{Cite web|title=航徽释义|url=https://web.archive.org/web/20010422031044/http://www.shenzhenair.com/ljsh/hhsy/hhsy.html|work=web.archive.org|date=2001-04-22|accessdate=2019-04-06}}</ref>。其后在2003年调整为更具文采的“'''深情无限，航程万里'''”。2008年，深航企业口号变为“'''任何时候、自然体贴、深圳航空'''”至今。

== 事件及意外 ==
*2009年2月15日，由北京至深圳的ZH9852航班因为左机翼故障中途迫降郑州<ref>{{cite web|url=http://udn.com/NEWS/MAINLAND/MAI1/4740226.shtml|title=《飞机迫降，深航班机无伤亡》 |accessdate=2010-04-28 |publisher=联合新闻网报导网页 |language=zh-hans }}</ref>。
*2015年7月26日，由台州至广州的ZH9648航班发生[[深圳航空9648号班机纵火事件|纵火事件]] 。   
*2016年6月26日早上大約10時，一架[[空中客车A320|空中客车A320]]執飞深圳航空9041號航班由[[福建|福建]][[泉州|泉州]][[晉江機場|晉江機場]]飛往[[香港|香港]]，在降落[[香港國際機場|香港國際機場]]北跑道時需要復飛，但復飛時錯誤地選擇轉向右邊，向[[大嶼山|大嶼山]]方向復飛，其後還在[[天壇大佛|天壇大佛]]頭頂飛過，低於安全高度，之後才緩緩爬升至6000英尺，繞了360度大圈，大約20分鐘後安全降落在香港國際機場北跑道。香港[[民航處|民航處]]承認該航班是偏離正常飛行航道，要求涉事航空公司提交報告<ref>{{Cite web |url=https://www.hk01.com/article/28687/ |title=深航客機掠過天壇大佛險撞山　民航處：降落香港偏離航道 |date=2016-06-28 |publisher=香港01}}</ref><ref>{{Cite web |url=http://hk.on.cc/hk/bkn/cnt/news/20160628/bkn-20160628195417410-0628_00822_001.html |title=深航客機大嶼山低過安全高度飛行　險撞大佛 |date=2016-06-28 |publisher=東網}}</ref>。
*2018年7月7日下午1時43分，一架[[波音737-800|波音737-800]]（註冊號：B-5412）執飛深圳航空9127號航班由[[武漢|武漢]]至[[呼和浩特|呼和浩特]]，在正常降落後滑出快速脫離道，機上124名旅客及9名機組人員無恙，事件具體成因需調查<ref>{{Cite web |url=http://news.southcn.com/gd/content/2018-07/07/content_182498140.htm |title=深航一航班降落呼和浩特时偏离跑道，致机场暂时关闭 |date=2018-07-07 |publisher=央视网}}</ref>。客機降落時現場下小雨<ref>{{Cite web |url=http://news.sina.com.cn/o/2018-07-07/doc-ihexfcvm3461956.shtml |title=深航航班降落呼和浩特时冲入草坪 部分航班取消 |date=2018-07-07 |publisher=新浪 中国新闻网}}</ref>。
