'''深圳东站'''<ref>[http://www.sznews.com/news/content/2012-09/15/content_7202402.htm 布吉火车站更名深圳东站 预计10月底通车]</ref>，原名'''布吉站'''（注：当地人仍称其为布吉站），位于[[中國|中國]][[广东省|广东省]][[深圳市|深圳市]][[龙岗区|龙岗区]][[布吉街道|布吉街道]]，为[[廣深鐵路|廣深鐵路]]的客货运[[三等站_(中国)|三等站]]，始建于1911年，由[[广州局集团|广州局集团]][[广深铁路股份有限公司|广深铁路股份有限公司]][[深圳站|深圳车站]]管辖<ref>{{Cite web |url=http://ilonghua.sznews.com/content/2018-02/05/content_18416754.htm|title=深圳站和深圳东站21趟临客明日开行 |accessdate=2018-07-03|publisher=晶报|date=2018-02-05 }}</ref>。

深圳东站是成为[[布吉综合客运枢纽|布吉综合客运枢纽]]的一部分，为深圳市“两主三辅”的铁路客运站布局的其中一个辅助客运站，有长途旅客列车到发，旅客并可以经[[深圳地铁|深圳地铁]][[布吉站_(地鐵)|布吉站]]转乘地鐵[[深圳地铁3号线|深圳地铁3号线（龍崗线）]]和[[深圳地铁5号线|5号线（环中线）]]<ref>[http://www.southcn.com/news/dishi/shenzhen/ttxw/200411250064.htm 南方网：布吉火车站将成深综合交通枢纽中心]</ref>。

==历史==
布吉站始建于1911年，是[[广深铁路|广九铁路华段]]首批建成的车站之一，车站本身就与“布吉”这一地名有密切关系。于[[清朝|清朝]]时期，在如今广深铁路布吉段西侧有一个名叫“莆隔村”的村落，属[[新安县_(广东历史)|新安县]]管辖。由于[[客家话|客家话]]中“莆”与“布”的发音十分相似，自清代中叶以后，莆隔村便渐被当地人称为“布隔村”。咸丰二年（1852年），当地人在莆隔村南面建起一个墟市，名叫“丰和墟”，并逐渐成为商品、农贸产品的集散地。1911年，广九铁路华段在丰和墟设置了一个车站，并命名为“布吉站”，自此“布隔”遂称之为“布吉”<ref>{{Cite web |url=http://news.sznews.com/content/2011-04/05/content_5501573_5.htm |title=布吉农批市场该搬不该搬？ 菜篮子与通途孰轻孰重 |accessdate=2011-07-22 |publisher=晶报 |date=2011-04-05 |archive-url=https://web.archive.org/web/20110410003542/http://news.sznews.com/content/2011-04/05/content_5501573_5.htm |archive-date=2011-04-10 |dead-url=yes }}</ref><ref>{{Cite web|url=http://www.szdm.gov.cn/dmwh/dmgs/201005/t20100527_55881.html|title=破译深圳地名密码·龙岗|accessdate=2011-07-22|publisher=深圳地名网|date=|deadurl=yes|archiveurl=https://web.archive.org/web/20151222094817/http://www.szdm.gov.cn/dmwh/dmgs/201005/t20100527_55881.html|archivedate=2015-12-22}}</ref>。

1949年10月后，广九铁路华段改称[[广深铁路|广深铁路]]。布吉站也是广深铁路双线改造（1984年至1987年）及准高速改造（1991年至1994年）施工时唯一沒有同時进行站房扩建的车站；在建设广深铁路三线时，由于所需站场面积扩大，布吉站相应进行了扩建，建有一座基本站台及一座中间站台，共有正线3条、到发线6条、货物线1条，站房面积100多[[平方米|平方米]]。随着广深铁路历次提速，停靠布吉站的旅客列车数量越来越少，至2000年代初，每天仅余下一对旅客慢车停靠，而[[春运|春运]]时也会有少数临时旅客列车以布吉站到发；随着广深铁路最后一对慢车8443／8444次于2003年停运，布吉站再没有旅客列车停靠。货运方面，布吉站的规模远远小于其他车站；与深圳地区其他铁路货运站相比，包括[[笋岗站|笋岗站]]（原深圳北站）、[[平湖站|平湖站]]、[[平湖南站|平湖南站]]，布吉站的货运量是四个车站中最少的。截至2007年，布吉站仅办理整车货物发到，到达货物量、发送货物量分别仅为254260吨、10095吨。

=== 改造规划 ===
[[File:Buji_(2148019440).jpg|thumb]]
[[File:Buji_Railway_Station_in_construction.jpg|thumb]]
早于1995年，深圳市政府确定了深圳市中短期铁路规划，当时提出“一主两辅”的深圳地区铁路客运站布局形式，“一主”即[[深圳站|深圳站]]，“两辅”即布吉站和[[深圳西站|深圳西站]]；深圳站主要承担广深铁路本线城际列车及全国主要城市始发和到达长途列车，而布吉站、深圳西站主要承担全国各地次要城市（例如安徽、江西、湖南、湖北、四川等民工比较集中的省份）始發和到达长途列车；当时规划布吉站每天将始發长途旅客列车28趟，客流量约3万人，“一主两辅”的格局也于1998年列入铁道部制定的全国铁路总图规划。

2002年，在[[香港特区政府|香港特区政府]]、广东省政府的推动下，中央政府、[[中华人民共和国铁道部|中华人民共和国铁道部]]正式对[[广深港高速铁路|广深港高速铁路]]立项研究。同年5月，深圳向铁道部提出将[[京广客运专线|京广客运专线]]延伸至深圳，并与[[杭福深客运专线|杭福深客运专线]]在深圳交汇。2004年1月，[[中华人民共和国国务院|中国国务院]]原则通过了《国家中长期铁路网规划》，规划明确了京广深客运专线、杭福深客运专线将在深圳交汇。2005年3月，国家铁道部组织举行了《深圳铁路枢纽总图规划》专家评审会，明确提出深圳将逐步形成由广深铁路、京港客运专线和杭福深客运专线以及相关支线（平南、平盐铁路）构成的“双十字形枢纽”格局，深圳市的铁路客运将形成“两主一辅”的格局，其中[[深圳北站|新深圳站]]（今深圳北站）、深圳站为主要客运站，布吉站为辅助客运站<ref>{{Cite web |url=http://gdjsb.ycwb.com/2006-09/12/content_1211319.htm |title=深圳逐步确立地区交通枢纽地位 |accessdate=2011-07-22 |publisher=金羊网 |date=2006-09-12 |deadurl=yes |archiveurl=https://web.archive.org/web/20160307203229/http://gdjsb.ycwb.com/2006-09/12/content_1211319.htm |archivedate=2016-03-07 }}</ref>，主要承担全国各地次要城市始发和到达长途旅客列车。2006年8月，铁道部与深圳市政府就广深港客运专线在深圳境内设站达成协议，决定于[[福田区|福田区]]增设[[福田站|福田站]]；同年[[厦深铁路|厦深铁路]]深圳段车站完成选址，决定于龙岗区设'''深圳东站'''，因此深圳市政府对原有《深圳铁路枢纽总图规划》作出了修订，调整为“两主三辅”的铁路客运站布局，即在“两主一辅”的基础上新增福田站、深圳东站两个辅助客运站<ref>{{Cite web |url=http://city.finance.sina.com.cn/city/2009-06-18/113144.html |title=深圳西站的去与留引发各方关注 |accessdate=2011-07-22 |publisher= 深圳新闻网 |date= 2009-06-18 }}</ref>。

2004年11月，深圳市规划局及[[广州铁路集团|广州铁路集团]]联合進行“布吉火车站规划及设计方案招标”活动，规划将布吉火车站改造成一个集铁路站、地铁站、公交站等多种交通功能为一体的综合客运交通枢纽，共有5家单位参加了综合枢纽的建筑方案设计投标，其中[[华南理工大学|华南理工大学]]建筑设计研究院和[[铁道第四勘察设计院|铁道第四勘察设计院]]提交的方案经过专家审议后被选为入围方案，分列第一、二名，而最终的方案会综合两个入围方案各自的优点<ref>{{Cite web |url=http://news.sina.com.cn/c/2004-11-25/06294339589s.shtml |title=90岁布吉火车站将变脸 |accessdate=2011-07-22 |publisher=深圳商报 |date= 2004-11-25 }}</ref>。2007年9月，规划部门完成了《布吉客运综合枢纽工程的可行性研究报告》，同年10月通过评审<ref>{{Cite web |url=http://www.szlgnews.com/lgnews/content/2007-10/17/content_1584020.htm |title=布吉客运枢纽工程可行性报告通过专家评审 |accessdate=2011-07-22 |publisher=龙岗新闻网 |date=2007-10-17 |archive-url=https://web.archive.org/web/20081120103256/http://www.szlgnews.com/lgnews/content/2007-10/17/content_1584020.htm |archive-date=2008-11-20 |dead-url=yes }}</ref>。2008年4月，完成了布吉客运枢纽建设工程范围内的拆迁工作，正式开工建设，并将于2012年12月21日启用。<ref>[http://sz.bendibao.com/news/2012128/459448.htm 深圳东站或21日启用]</ref>

2012年12月5日，布吉站正式更名为深圳东站<ref>{{Cite web |url=http://native.cnr.cn/city/201212/t20121204_511482403.html |title=深圳布吉站5日起正式更名为深圳东站 |accessdate=2012-12-05 |publisher= 中国广播网  |date=2012-12-04 }}</ref>。布吉综合客运枢纽竣工后，深圳东站将成为交通枢纽的一部分，占地13.71万平方米，具体范围位于吉华路和深惠路一帶，站房为跨线式，由东、西站房及高架候车厅组成，设有6条有效长度650米的客车到发线、1条货车到发线和4条正线，设旅客基本站台和中间站台各两座，主要为[[京九线|京九线]]、[[广茂铁路|广茂线]]及[[广西|广西]]、[[云南|云南]]方向始发终到的长途旅客列车<ref>{{Cite web |url=http://news.sznews.com/content/2011-01/12/content_5252856_5.htm |title=深圳北站主体工程完工 将成全国重要铁路枢纽 |accessdate=2011-07-22 |publisher=深圳特区报 |date=2011-01-12 }}</ref>。

==营运==
该站于2012年12月21日正式开始營運，首班列車開往天津，但候车厅、停车场等设施施工尚未完成。
