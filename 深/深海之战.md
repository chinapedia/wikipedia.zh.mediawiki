《'''深海之战'''》（{{kor|k=7광구|hanja=七鑛區|rm=Chilgwanggu}}，{{lang-en|'''''Sector 7'''''}}）是2011年上映的一部[[韩国|韩国]][[三维电影|3D]]電影，由[[金志勋|金志勋]]执导，[[河智苑|河智苑]]、[[安圣基|安圣基]]、[[吴智昊|吴智昊]]主演。<ref>{{cite web |url=http://movie.now.com/home/movie?id=Sector7 |title=存档副本 |accessdate=2011年12月11日 |deadurl=yes |archiveurl=https://web.archive.org/web/20111201165755/http://movie.now.com/home/movie?id=Sector7 |archivedate=2011年12月1日 |df= }}</ref>该片号称是[[亚洲|亚洲]]首部3D惊悚、悬疑动作片<ref>{{cite web | language = zh-cn | publisher = 网易| title =《深海之战》发中文海报 主打亚洲首部3D惊悚片| url = http://ent.163.com/11/1115/07/7ISRLMJP000300B1.html| date =  | accessdate = 2011-11-15 }}</ref>。

== 劇情介紹 ==
講述一艘在[[济州岛|济州岛]]南端2500米的深海中进行石油探测的探测船“日食号”，遇到深海中的不明生命体的怪物，随后一船科学家便与怪物展开了生死殊斗的故事。
