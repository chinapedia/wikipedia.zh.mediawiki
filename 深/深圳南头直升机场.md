'''深圳南头直升机场'''（又称：'''深圳直升机场'''，'''南头机场'''，'''深圳(中信)海洋直升机场'''）位于[[中国|中国]][[广东省|广东省]][[深圳市|深圳市]][[南山区|南山区]][[南海大道_(深圳市)|南海大道]]与[[北环大道|北环大道]]交界处，拥有一条跑道。与[[深圳宝安国际机场|深圳宝安国际机场]]直线距离约为15公里，两机场跑道方向交叉。机场总建筑面积为3.2万平方米，设有机场候机楼、16个停机坪和设备完善的指挥塔，可同时起落16架[[直升机|直升机]]。<ref name="nfdaily100503">{{cite news | url=http://epaper.oeeee.com/H/html/2010-05/03/content_1070928.htm | title=南头直升机场屡建救援奇功 | work=[[南方都市报|南方都市报]] | date=2010-05-03 | accessdate=2014-12-05 | author=康殷 | deadurl=yes | archiveurl=https://web.archive.org/web/20141208175155/http://epaper.oeeee.com/H/html/2010-05/03/content_1070928.htm | archivedate=2014-12-08 }}</ref>

== 概要 ==
=== 建设 ===
深圳直升机场由[[中国海洋航空公司|中国海洋航空公司]]、[[中国直升飞机公司|中国直升飞机公司]]、[[中国海洋石油总公司|中国海洋石油总公司]]、[[南海石油服务总公司|南海石油服务总公司]]和[[深圳工业发展公司|深圳工业发展公司]]联合兴建，总投资额为5000万，1983年3月，由基建工程兵负责建设的深圳直升机场在[[南头|南头]]正式开工，同年10月17日建成使用，为深圳首座直升机场。

=== 定位 ===
筹建之初，深圳直升机场主要为南海油气开发服务，直升机以深圳作为母港，飞抵南海协助勘探石油。就在[[深圳机场|深圳机场]]选址悬而未决之际，拥有16个直升机坪的深圳直升机场，曾一度考虑引入短程起降飞机，在深圳机场建成之前，以直升机场之名暂代深圳机场之实。

从建成发展到现在，深圳直升机场的功能也日益多样化，开展各类[[通用航空|通用航空]]业务，包括国内陆上石油服务、海上石油服务、人工降水、医疗救护、航空探矿、直升机引航作业、通用航空包机飞行、公务飞行、空中游览、出租飞行、直升机机外载荷等。而深圳直升机场，亦一直作为[[中信海直|中信海直]]公司的基地进行运作。

=== 跑道 ===
深圳直升机场共有一条跑道：方向11/29，长585米，宽18米。<ref name="gochina120622">{{Cite web |url = http://www.gachina.net/NeiRong_WenZhang_229692_443960_cn.html |title = 深圳南头直升机场 |author = admin |date = 2012-06-22 |publisher = 中国通用航空门户网 |accessdate = 2014-12-05 }}</ref>

== 空中交通管制 ==
南头机场与宝安机场划为同一个管制空域，由于11跑道延长航线与宝安机场33跑道下滑道交叉，进出南头机场的[[航空器|航空器]]必须注意观察，严格听从管制指挥，防止空中危险进近和相撞。

南头机场拥有独立管制台，设立[[塔台|塔台]]席位，塔台呼号为“南头”，负责该机场的放行、地面、起降，离场航空器离地后联系[[珠海|珠海]]进近，离开珠海进近管制圈后移交回南头机场进行程序管制，进场航空器下降至南头机场塔台空域内后由珠海进近移交至本场塔台指挥。由本场离场的航空器需由宝安塔台同意，否则只能在本场做[[五边飞行|五边飞行]]等。<ref name="gochina120622"/>

== 机场迁移规划 ==
经过20多年发展，1983年建成的深圳直升机场所在的偏远地带已经变成南山中心区，直升机场附近居民和商业活动越发频繁。直升机虽然为深圳城市的灾难救援、石油勘探提供大量服务，但直升机起降噪音日益扰乱周边居民正常生活；此外，由于安全需要周围建筑物必须限高，严重制约当地片区发展。

2012年2月，在南山区五届人大六次会议上，有人大代表建议从城市发展和资源充分利用角度出发，应将直升机场搬迁至离市中心较远地方，促进当地社区经济发展。<ref name="nfdaily100503"/>

2018年5月9日，《市规划国土委关于观澜樟坑径机场用地选址方案、规划设计条件及基本生态控制占用方案公示展示的通告》发布，其中显示新机场选址。樟坑径机场用地规划总用地面积29.2公顷，包括机场运行区25.52公顷、油库区0.6公顷、运行保障区3.1公顷。运行区大部分用地及油库用地均位于[[龙华区|龙华区]][[观湖街道|观湖街道]]，少量用地就横跨于[[龙岗区|龙岗区]][[平湖街道|平湖街道]]<ref>{{Cite web |url=https://m.mp.oeeee.com/a/BAAFRD00002018051079007.html |title=深圳南头直升机场搬迁龙华正式公示 |date=2018-05-10 |publisher=南方都市报}}</ref><ref>{{Cite web |url=http://finance.sina.com.cn/stock/s/2018-05-10/doc-ihaichqz9146478.shtml |title=深圳新建第二机场？ 不过与中信海直密切相关 |date=2018-05-10 |publisher=新浪财经 新浪证券综合}}</ref>。

== 机场事迹 == 
*1983年，深圳直升机场在深圳市南头镇东北约2公里建成并部分交付使用，可同时起落16架直升飞机。主要为华南各海域勘探、开发海洋石油运送各种器材、物资，为海上工作人员提供生活服务。

*1983年，由中国海洋航空公司等五家大公司共同注资的[[中信海直|中国海洋直升机专业公司]]正式成立，简称中海直，总部设在深圳直升机场。

*1993年8月5日，深圳[[清水河大爆炸|清水河大爆炸]]事故发生后，有重伤员需要紧急送到[[广州|广州]]治疗，深圳直升机场即刻起飞两架直升机，在卫生系统医务人员的配合下，将4名重伤员及时送抵广州。

*2008年5月12日中国[[四川|四川]][[汶川|汶川]]发生里氏8.0级[[汶川大地震|地震]]。5月15日凌晨，总部位于深圳的中信海直公司，接到征用命令后，在3个小时内从深圳等7个地区，调集12架直升机飞赴汶川参与抗震救灾。来自深圳的两名直升机飞行员救起67名地震伤者。<ref name="nfdaily100503"/>
