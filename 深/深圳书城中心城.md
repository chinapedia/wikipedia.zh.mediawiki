'''深圳書城中心城'''位於[[中國|中國]][[廣東省|廣東省]][[深圳市|深圳市]]中心區（[[CBD|CBD]]）北中軸線上，[[福田區|福田區]]福中一路，北鄰[[蓮花山_(深圳)|蓮花山]]，南接[[深圳市民中心|深圳市民中心]]，西鄰[[深圳音乐厅|深圳音乐厅]]、[[深圳图书馆|深圳图书馆]]，東毗[[深圳市少年宫|深圳市少年宫]]，是「[[深圳書城|深圳書城]]」品牌的旗艦店。深圳書城中心城於2004年7月5日開工建設，2006年11月6日正式開業。深圳書城中心城隸屬深圳市發行集團公司，是國有全資、獨立核算的文化公司。

== 建築 ==
深圳書城中心城佔地8.7[[公頃|公頃]]，建築面積8.2萬[[平方米|平方米]]，為單層框架結構，由[[日本|日本]]建築師[[黑川紀章|黑川紀章]]設計，室內設計由「{{Link-en|凱里森建築事務所|Callison}}」負責，經營面積4.2萬平方米，是現時全世界單店經營面積最大的書城<ref>[http://www.southcn.com/news/gdnews/sd/200611070012.htm 深圳書城中心城開業 匯集30萬種圖書成亞洲最大單體書店]</ref>，被列入2003-2004年深圳市政府重點工程。

建築物包括地下一層、夾層和地上一層共三層空間。地下層主要為停車區，設有近600個停車位。地面建築分南北兩區，[[紅荔路|紅荔路]]和[[福中一路|福中一路]]之間為「北區」，福中一路以南為「南區」，兩區通過跨福中一路的連接體連接。書城東西兩邊是“詩、書、禮、樂”四個面積各1萬平方米的綠色文化公園，書城的屋面除25米寬的步行軸外，兩側滿布著綠色的植物，屋頂的覆土層可以讓草坪和樹木生長。首層和夾層擁有龐大的縱向共用大廳，兩側的屋頂有縱向延伸的天窗，南、北兩區分別有一方一圓兩處庭園，取「天圓地方」之意，直通天面，接引天然光和綠色植物景觀的作用。

== 經營 ==
深圳書城中心城為首家體驗式書城，引入「書店商場二合一」（BookMall）概念。在大賣場中規劃出近2萬平方米用於展示與銷售出版物及文化用品，除了設有中心書店外，還開設了「人文社科書店」（北區首層）、「經濟管理書店」（北區首層）、「藝術設計書店」（南區首層）、「生活閒暇書店」（北區首層）、「科學技術書店」（北區負一層）、「音悅時空主題店」（北區首層）、「青少年主題店」（北區負一層）、「外文原版書店」、24小時營業的「星光悅讀棧」（南区首层）等特色店，組合近30萬品種的產品。還整合了品位餐飲（[[星巴克|星巴克]]、illy coffee、紫苑茶館、[[面點王|面點王]]、[[味千拉麵|味千拉麵]]、可頌坊、[[仙蹤林|仙蹤林]]、[[必勝客|必勝客]]等）、休閒娛樂、創意產品和文化精品等專案，提供一站式、多元化的文化消費選擇與體驗。

== 小資料 ==
* 中心城提供的紙質購物袋需要收費，大、中、小不同型號價格分別為人民幣1.5元、1元及5角。
* 中心城洗手間男女蹲位比例近1：3，並設定專門的殘障人洗手間，提供[[手語|手語]]服務。
* 中心城的背景音樂由專業人士精心編排，背景音樂因季節、天氣、時段而不同。
* 中心城的播音服務採用[[普通話|普通話]]、[[英語|英語]]、[[粵語|粵語]]三種語言。
