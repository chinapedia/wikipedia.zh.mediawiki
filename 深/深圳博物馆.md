'''深圳博物馆'''，位于[[中国|中国]][[广东省|广东省]][[深圳市|深圳市]][[福田区|福田区]]福中路[[市民中心_(深圳)|市民中心]]A区，是深圳市属综合性[[博物馆|博物馆]]。

==简介==
深圳博物馆原址位于深南中路同心路6号。福中路市民中心的现址于2008年12月25日16点（[[UTC|UTC]]+8）举行开幕典礼，正式对外开放。

深圳博物馆是一座以地方性为主的综合类博物馆，其建筑主体是深圳市民中心的一部分，位于其东翼。总占地面积12500平方米，建筑面积32000平方米，博物馆总投资达2.4亿元[[人民币|人民币]]。<ref>{{cite web

2012年，深圳博物馆被[[国家文物局|国家文物局]]核定为第二批[[国家一级博物馆|国家一级博物馆]]<ref>{{Cite web |url = http://www.gov.cn/zwgk/2012-12/07/content_2285163.htm |title =关于发布第二批国家一级博物馆名单的通知（文物博函〔2012〕2237号）  |author =  |date =2012-12-07  |publisher = 中国政府网 |language =  |accessdate =  |quote =  }}</ref>。

==展览与陈列==
深圳博物馆共有四项基本陈列：

*深圳改革开放史展厅：位于三楼，展览面积3235平方米，共设四个展厅。主要展示深圳自1978年[[改革开放|改革开放]]至2008年30年间深圳取得的巨大建设成就及发展。
*古代深圳展厅：位于二楼，展览面积713平方米。主要展示深圳的[[考古|考古]]发现及深圳地区自人类活动开始到[[清朝|清朝]][[鸦片战争|鸦片战争]]前夕6000年历史。
*近代深圳展厅：位于二楼，展览面积1237平方米。主要展示1839年深圳自鸦片战争前夕至1949年[[中华人民共和国|中华人民共和国]]成立的百余年历史。包括割让[[香港|香港]]、[[惠州起義|三洲田起义]]、[[省港大罢工|省港大罢工]]、[[抗日战争|抗日战争]]等重要历史事件，另有专门部分展示100多年来反映深圳地区[[民生|民生]]情况的图片、实物及再现实景。
*深圳民俗展厅：位于二楼，展览面积1860平方米。主要展示深圳地区传统的民俗文化，包括地区民俗、[[广府民俗|广府民俗]]、[[客家民俗|客家民俗]]及海洋文化民俗。

此外，一楼还设有儿童专题厅，面积为482平方米。博物馆临时展览也设在一楼。
