'''深圳公交B847路'''，是西乡地铁站顺时针行驶循环线，性质是支线[[公交|公交]]，由[[深圳市西部公共汽车有限公司|深圳市西部公共汽车有限公司]]运营。

== 线路简介 ==
* '''深圳公交B847路'''由[[深圳市西部公共汽车有限公司|深圳市西部公共汽车有限公司]]营运。
* B847路是西乡地铁站顺时针行驶循环线，全长7.7公里，配车3部，车型为宇通客车ZK6608DM；单程行驶时间为30分钟。
* 服务时间：6:30-22:30
* 发车间隔：20分钟一班

== 途经站点 ==
*顺时针行驶：[[西乡站|西乡地铁站]] - 华盛辉大厦 - 西乡客运站 - 大益广场 - 新安市场 - 西乡立交 - 西乡人民医院 - 西乡路口 - 径贝村 - 西乡码头 - [[坪洲站|坪洲地铁站]] - 劳动路口 - [[西乡站|西乡地铁站]] (13站)

== 历史 ==
*2011年9月，由于碧海片区在该线开通之前没有一条直达G107的公交线路，遂该线路首次出现于2011年度第二批公共汽车线路规划方案。<ref>http://www.sztb.gov.cn/gov/detail.aspx?paraID=12879&clsID=394&pid=390{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }} 关于征求《2011年度第二批公共汽车线路规划工作方案》（征求意见稿）意见的通告</ref>
*2012年1月6日，深圳公交B847路开通，用车为宇通客车ZK6720DF。<ref>{{cite web |url=http://www.baoannet.com/content/2012-01/06/content_6369739.htm |title=存档副本 |accessdate=2012年2月26日 |deadurl=yes |archiveurl=https://web.archive.org/web/20120119205025/http://www.baoannet.com/content/2012-01/06/content_6369739.htm |archivedate=2012年1月19日 |df= }} B847环线开通 西乡地铁口往西乡大门有直达公交</ref>
*2014年5月，由于B847路客流量一直比较低，B847路开线时使用的2台宇通ZK6720DF调至B630路，B630路抽取3台宇通ZK6608DM于B847路。
