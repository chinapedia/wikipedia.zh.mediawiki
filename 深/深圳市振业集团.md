'''深圳市振业(集团)股份有限公司'''，证券简称：'''G深振业'''、证券代码：'''000006'''，曾用简称：深振业Ａ。公司总股本2.54亿股（2006年2季度），总部位于中国深圳市宝安南路振业大厦27-32层，现任董事长为李永明，总经理为李富川。

== 历史沿革 ==
公司前身为成立于1984年8月的深圳市建设(集团)公司开发部，1989年4月改组为股份有限公司，同年8月发行股票69万股，筹资690万元，1992年4月27日在[[深交所|深交所]]上市，上市首日开盘价33元。

== 股权分置改革 ==
2006年1月12日实施[[股权分置改革|股权分置改革]]，非流通股股东向全体流通股股东每10股送2股。

== 業務 ==
公司主要在深圳從事房地產開發，較著名的物業包括[[六約|六約]][[振業城|振業城]]、和[[深圳市地鐵集團|深圳市地鐵集團]]合作發展的[[橫崗車輛段|橫崗車輛段]]上蓋項目。
