'''隶定'''，指在[[出土文獻|出土文獻]]整理和古文字研究上，将古汉字摹拟成[[隶书|隶书]]，并保留原字体结构的作法。

== 概要 ==
《[[偽古文尚書|偽古文尚書]]·尚書序》：「科斗書廢已久，時人無能知者，以所聞伏生之書考論文義，定其可知者，為'''隸古定'''。」[[孔颖达|孔颖达]]《尚書正義》：「言『隸古』者，正謂就古文體而從'''隸定'''之。存古為可慕，以隸為可識，故曰『隸古』，以雖隸而猶古。」

孔壁[[古文经|古文经]]被發現後，由於記載[[先秦|先秦]]文字難以解讀，而採當時[[漢朝|漢朝]]通行的[[隸書|隸書]]寫定辨識，此為「隸古定」。即使後世[[楷書|楷書]]通行仍將這種考釋[[古文_(文字)|古文]]方式稱作「隸定」，大致可分為筆劃隸定法、偏旁隸定法、字形轉換法、綜合隸定法等等。也因此隸定常依各家治學有所不同，沒有標準可循。<ref>{{cite web|author=李守奎|title=《曹沫之陳》之隸定與古文字隸定方法初探|url=http://www.bsm.org.cn/show_article.php?id=482|website=簡帛網|date=2006-12-12|accessdate=|language=}}</ref>

== 举例 ==
* 更—㪅：小篆“更”从丙、攴，隶变之后粘合成为独体字。其隶定字模拟小篆结构作“㪅”。
* 侯—矦：小篆“侯”从人、-{厂}-、矢，隶变之后重组成为左右结构的合体字。其隶定字模拟小篆结构作“矦”。

== 隶定字列表 ==
以《[[说文解字|说文解字]]》[[小篆|小篆]]为收字范围，列出[[通用字符集|通用字符集]]中相应的隶变字、隶定字及以之作为部件类推出的其他隶定字。此外，[[Unicode|Unicode]]也收錄[[金文|金文]]等出土文物的隸定字。

并非所有经过隶变的汉字都有对应的隶定字，也不是所有隶定字都能够类推完全。单字的使用频率、笔画的多少、隶变部件与隶定部件的区别度，都可能影响隶定字的有无。
