《'''戲劇性謀殺'''》（{{lang-en|'''DRAMAtical Murder'''}}），是由[[Nitro_+_CHiRAL|Nitro + CHiRAL]]在2012年發行的一款[[十八禁遊戲|18禁]][[BL遊戲|BL遊戲]]，並於2013年發佈續作DRAMAtical Murder re:connect，並於2014年7月6日日本時間深夜時段由日本[[東京電視台|東京電視台]]開始播放電視動畫，同時另有[[NICONICO|NICONICO]]網路直播隔周放送，還出了psv 版本，內含水紀線（非友情）。

== 故事簡介 ==
在日本群島附近的一座小島「碧島」，是主角蒼葉所生活的地方，一日，東江財團收購了大片的土地，進而興建一座名叫「白金牢籠」的遊樂區，而原本生活的島民，一些接受資助搬離碧島，而剩餘不接受的人被迫聚集到邊邊，進而形成的舊民區。
蒼葉工作在「平凡」，真的很平凡的生活中，一位位人物，以及自己的身世，頭痛，接踵而來，到底有甚麼故事等著蒼葉去解決呢？

: 本作主角。
: 官網介紹句：「怎——麼——會——那種東西最招人厭了。」
: 樂觀，愛照顧人，被拜託也不會嫌麻煩的好人性格，很少生氣。
: 面對敵人的時候會猛踢對方，殺傷力足以踢爛下頜骨。
: 必殺技被水紀叫做飛狐劈擊。
: 從過去開始就被不明原因的頭痛困擾著。
: （劇透：實際上擁有以聲音為媒介的能力「暴露」，頭痛亦是這個原因。體內曾有三個人格──「理性」、「慾望」和「抑制力」。）
: 動畫第五集發現闖入家中並綁走多惠婆婆的黑衣人就是Dry juice的人，而水紀也在其中，但個性變得十分殘暴。
: 據多惠婆婆所說蒼葉之前是有玩過萊姆的(萊姆ID：Sly blue)，但發生了意外(其實是Virus和Trip製造的意外)而導致蒼葉遺忘曾經玩過萊姆的這件事。
: 在綁架婆婆的事件落幕後，收到來自Virus的訊息說警察正趕往他家並叫蒼葉趕緊逃走，此時和紅雀一同收到了白金牢籠的通行許可證，並一同前往白金牢籠。
: 刺青的事件之後跟Noiz一同出門買食物，途中遇到了隸屬Morphine的黑衣人們，Clear出手相救，所性並無大礙。
: 檢查完Clear有無受傷後一同在客廳看新聞，當新聞播到特別紀念活動表演時突然痛苦起來，Clear隨即關掉電視才好了起來。Clear說剛剛那首歌大概是死亡之曲。
: Clear待機後，回想起多惠婆婆說過“你的聲音有操縱他人的力量，也就是說你也能破壞對方的意識”後，開始覺得“是不是因為我操縱了紅雀他們的意識，才導致他們這麼痛苦？”
: 正當蒼葉覺得不能再連累其他人時，這時Mink打電話說“我來找你討人情了(幫助蒼葉救回婆婆)，帶著你的智能夥伴出來。”便偷偷得出去了。在進入了Mink的內心世界後，蒼葉在被熊熊大火燃燒的Mink的家鄉中，看到了Delivery work的多江小姐、多惠婆婆、羽賀老闆和孩子們等人。
: Mink說“你的身上有著跟我一樣的味道，死亡的氣味。”
: 蒼葉發現蓮在散播破壞中樞塔的武裝系統後有些奇怪。
: 中樞塔中被Mink壓迫並扯痛頭髮而讓「破壞」出現，才脫離險境。跟Mink走散後遇到了Sei「生」，隨後遭黑衣人打昏，醒來後見到了也是Morphine的一員的Virus和Trip。從他們口中得知Sei「生」是自己的雙胞胎哥哥，是在東江研究所誕生的實驗品，經過設計後天生就具備操縱人心的力量，出生時蒼葉跟生的頭髮是相連的且是有神經的，兩個人出生後皆沒有呼吸，剪斷頭髮後生就有呼吸了，但蒼葉還是沒有呼吸。後來被多惠婆婆以「不想他再成為實驗材料」為由給帶走了。但是後來卻奇蹟似的活了過來。
: 為了救蓮進入了自己的內心世界(蓮透過咬將自己的意識注入蒼葉腦中)。蓮說在蒼葉心中的「理性」、「慾望」和「抑制力」是三個各自獨立的個體，而蓮自己就是那股「抑制力」。
: 被東江拉入去除保護機制的萊姆中對決，發現東江的智能伴侶就是模仿Sei而創造出來的，後來因為知道不可能打贏東江他們而選擇與Sei「生」對話後贏了對決。見到生後想和他一同逃離這裡，但生說他已經撐不住了，希望蒼葉破壞他，並叫蒼葉也該承認「慾望」(也就是「破壞」)的存在了。最後在Sei「生」苦苦哀求說“對我來說是救贖”下破壞了他。
: 在中樞塔倒塌時，被Mink事先將智能伴侶植入的機車所救。
: 在中樞塔倒塌後一年，接到來自醫院的電話(畫面確認是Virus和Trip所打)說“有您的家人正在本院治療，如果方便的話是否能來探望？”，聽到他們提及可能是應該死亡的「生」而決定過去。
: 到醫院前又再度收到了原本由生所發送的遊戲訊息。
: 後來發現躺在病床上的是以為已經回到自己身體裡的蓮(長相是Sei)。

; {{nihongo|紅雀|紅雀（こうじゃく）(Koujaku) ||聲：沖野靖広／[[高橋廣樹|高橋廣樹]]}}
: 官網介紹句：「都說到這份上了再拒絕的話，不是太不像男人了嗎？」 
: 不定期在街頭擺攤的理髮師，人氣很高。
: 蒼葉的青梅竹馬，擅長照顧人。
: 有抽菸的習慣，根據蒼葉所說紅雀只有在他房間外的陽台才會抽菸。
: 交際手腕靈活，在外人面前掛著營業式笑容，很受女子歡迎。
: 組織了街鬥幫派“紅時雨”，幫派裡都是熱情的好人。
: 幾年前曾離開碧島回本州去過，回來以後總覺得有什麼瞞著蒼葉。
: 多惠婆婆的綁架事件後，和蒼葉都收到了白金牢籠通行許可證，並一同前往白金牢籠。
: 於白金牢籠中看見龍峰後，憤怒的追了上去。
: 情緒相當憤怒時，背後的紅花的刺青會慢慢的浮現，據龍峰所說紅雀越憤怒，刺青越會侵蝕他的心靈。
: 在龍峰對蒼葉說出紅雀自己殺死了母親後，便失去理性的拔刀衝向龍峰，甚至連蒼葉也想斬殺。
: 隨後蒼葉使用能力進入了紅雀的內心世界，才知道原來紅雀是側室的孩子，因正室生不出孩子而被迫繼承家業(黑道)。而刺青是一種象徵，是紅雀父親叫龍峰硬刺上去的，在刺青完成最後修飾的日子裡，紅雀受不了痛苦而暈了過去，等他醒來看見的是身旁一片血海，連自己的母親也倒在其中，而自己則是握著滿是鮮血的刀子，紅雀說他以為只要見到蒼葉就會沒事而來到碧島，但還是會不經意的想起那天的血海。就在紅雀打算放棄時，蒼葉說：“也許你覺得這樣就好，但你身邊的人呢？紅時雨的夥伴呢？還有我呢？你的一生都得背負著這個刺青，但我們都清楚你的為人，不管你背負著多大的罪孽跟宿命，你依然是我所熟悉的紅雀。當你迷失自我時，不管幾次我都會告訴你我心中的紅雀是個怎樣的傢伙，還有我一直都相信著你。所以不要輸了，紅雀，不要輸給刺青跟過去的記憶。紅雀，我現在要破壞你的心，把禁錮你心靈的過去給破壞掉。”使紅雀恢復了以往的冷靜。(GE)
: 在蒼葉遭Virus和Trip綁架時前來搭救。

; {{nihongo|Noiz|ノイズ(諾伊茲) ||聲：野次馬根性／[[日野聰|日野聰]]}}
: 官網介紹句：「那，就把你重要的東西毀掉。」
: 萊姆遊戲高手，似乎對萊姆有特別的感情。
: 說話口氣淡薄，有時會做出意想不到的動作。
: 據他所說自己一直是只吃外送披薩和義大利麵活過來的。
: 智慧伴侶是腰間的立方體“擬兔”，有多個可拆卸單獨使用。
: 動畫第三集闖入倉葉家，從口袋拿出兔子形狀的吊飾後，確認蒼葉就是Sly Blue本人。蒼葉也才知道Noiz就是對他強制開戰的那個人。
: 自從跟蒼葉對戰過後，便一直想叫蒼葉加入王牌兔子。
: 對於蒼葉不回答是怎麼在萊姆打贏他的相當氣憤，以折斷手臂威脅蒼葉再跟他打一場，在Clear和紅雀相繼闖入後做罷。
: 後與紅雀發生爭執並打了起來，在蒼葉大喊後停止打架，最後在多惠婆婆生氣的大吼後四人一同被叫下樓吃東西。
: 因為親了來打擾蒼葉工作的小女孩的手，而有了“穿孔色狼”的稱號，隨後便突然親了蒼葉，使得蒼葉不知所措，甚至懷疑Noiz的性向。
: 多惠婆婆的綁架事件後，收到了白金牢籠通行許可證，在蒼葉在房間休息時來到了白金牢籠。
: 萊姆幫派“Ruff Rabbit 王牌兔子”的首領。
: 不知為何對於身體受傷毫不在意，跟蒼葉在白金牢籠中遇襲受了傷，蒼葉問他你都不會痛嗎，他回答“別說拳打腳踢，甚至用刀刺也沒感覺，我根本感覺不到痛。”
: 隨後發現疑似殘黨的純白人物便追了過去，被對方放出的音樂給催眠而攻擊隨後追來的蒼葉，蒼葉同樣使用能力進入他的內心世界。看到年幼的Noiz躺在床上，床邊疑似父母的人說：“為什麼會發生這種事，這樣一輩子都別讓他出門反而比較好。”“真是丟光我們家的臉，世世代代都會以你為恥，要是你沒出生就好了。”
: 隨後在眼前出現的是被數條鎖鍊拴著且蜷縮在地的Noiz，他一開口便說：“我被這個世界所隔離了，”(背景跑出“被父母拋棄”“不得不一個人生活”“沒有痛覺，沒任何感覺，什麼都看不見”“一個人留在孤獨的黑暗空間”等話)此時蒼葉才知到這就是Noiz所感覺的世界。

: 在中樞塔中終於見到了蒼葉，向蒼葉坦承之前那些如預言般的遊戲畫面和白金牢籠通行證是自己發送的，其實是想要蒼葉來到他身邊並用“破壞”的力量毀滅自己。
: 在蒼葉進入自己的內心世界後，以自己已經要撐不住為由，再次提出希望蒼葉破壞自己。
: 在消失前表示自己不恨東江，說實驗的一部分自己也有責任，最後心情平和的被蒼葉給破壞了。

; {{nihongo|龍峰|リョホウ||聲：一条光／[[笹沼晃|笹沼晃]]}}
: 穿著藍色和服，戴著骷髏裝飾，脖子上刺有青色海馬。
: 刺青師，刺青狂熱者。

: 受東江邀請加入東江財團。

== 用語 ==
作品中的用語。
; {{nihongo|碧島|碧島（みどりじま）}}
: 一座架空的島嶼。
: 位於日本列島的西南方，是遊戲的舞台。
: 被東江財團買下。

; {{nihongo|東江財團|東江財閥||東江コンツェルン}}
: 收購了整個碧島，並曾以許多好處誘惑島上居民離開。
: 將剩下的居民全部堆在島的一邊，建立一座頂級的遊樂設施──白金牢籠。
: 對舊居民區的居民採放任管理，並只提供最低限度的生活物資。（水、電、瓦斯等）
: 似乎在計畫著什麼。

; {{nihongo|舊居民區|旧住民区}}
: 不願離開島上的居民所居住的地方，分為東西南北四個區域。
: 管理混亂，現在是黑幫和虛有其名的警察為非作歹的地帶。
: 東區是治安較好的一區，由各色商店和民宅組成。
: 西區是純居民區，幾乎沒有商店。
: 南區是年輕人很多的地方，有許多的服裝店或娛樂設施。
: 北區則被封鎖，現為禁止進入的區域。也是治安最差的區域。
: 北區的最北端便是白金牢籠。

; {{nihongo|「平凡」雜貨店|「平凡」ジャンクショップ}}
: 蒼葉打工的地點，出售各類零件。
: 由於其低廉的價格，廣受各方客戶喜愛。

; {{nihongo|白金牢籠|プラチナ・ジェイル||Platinum Jail}}
: 東江財團建立的大型遊樂設施，占了島上1/3的面積。
: 只有得到邀請才可入場，據說門票十分地昴貴，從外無法看見其全貌。
: 在蒼葉等人進入之後，漸漸發現這裡面的人都只是供東江財闊作研究的白老鼠而已。

; {{nihongo|中樞塔|オーバルタワー||Oval Tower}}
: 位於白金牢籠中央，將其分為五個區域。
: 是控制一切機能的中心，也是白金牢籠的象徵。

; {{nihongo|便攜終端|コイル}}
: 被廣泛使用的攜帶式電腦。
: 蒼葉使用的是手錶型。
: 除如手機般的撥接功能，亦有收信、付費、上網、路線指引、身份證明等多項功能。

; {{nihongo|智能伴侶|オールメイト||Allmates}}
: 一種人工生命體，擁有人工智慧。
: 能和其對話交流，外型各式各樣，因此亦被當作寵物。
: 也能在「萊姆」中做為戰鬥伙伴。

; {{nihongo|萊姆|ライム||Rhyme}}
: 在年輕族群中擁有高人氣的虛擬遊戲，只要擁有智能伴侶便可以進行遊戲。
: 對戰時會進入虛擬的對戰空間，並能實際感受到對戰的快感。
: 開始地點受「卯水」的限制，「卯水」出現的地點才可以開始萊姆對戰。

; 卯水
: 萊姆的裁判，亦是遊戲開始的指標，
: 是在東江將此遊戲買下後才加入的規則。
: 其實是東江的智能伴侶，是以生散播的意志作為原型製造的，有著壓倒性的實力。

; 街頭格鬥
: 與萊姆相對，是現實世界中的戰鬥。
: 常以幫派做為劃分地盤的依據，幫派間常有以武力爭奪地盤的行為。

; 安全機制
: 安裝在Clear和他的兄弟腦中的系統。
: 其主要作用為絕對禁止同伴間互相傷害也不可以攻擊主人。

; {{nihongo|強制開戰|辻斬りライム}}
: 最近新出現的一種萊姆形式。
: 會將毫無防備的人突然間拉入萊姆對戰，亦沒有「卯水」做裁判。
: 速度和力量被人為地大幅提升，因此快感更勝一籌。
: 但在強制開戰的萊姆中身體所受的傷害會殘留到現實世界，因此十分危險。

; 染色音樂
: 東江用來使聽到的人被洗腦的歌曲，讓他得以進行操縱人心的實驗。
: 普通人只要一聽到便會喪失心智變成聽從東江命令的傀儡，只有像蒼葉等有特殊力量之人才可稍稍抵抗。

; 神隱
: 於動畫第一集提到，從水紀口中得知在10年前發生了整個幫派消失的事件，但蒼葉認為那不過是都市傳說罷了。
: 在10年後的現在神隱再度發生，不同的是消失的成員過了幾天又回來了，但都沒有失蹤期間的記憶，連自己是誰都忘了。
: 水紀形容：就好像腦袋裡的東西被破壞一樣。

; {{nihongo|暴露|暴露（スクラップ）}}
: （劇透）蒼葉擁有的能力，以聲音為媒介。本質是「破壞」、「死亡」。
: 能夠控制人心，或進一步進入對方無防備的內心，便可看見對方不願面對的回憶。
: 只要蒼葉帶著感情說話，對方便會無條件的聽從。
: 能夠對這些加以傷害，甚至可以使對方成為廢人。
: 也可以幫助對方，使其面對這些回憶，並消除其影響。
: 實際上是蒼葉體內代表「破壞」的人格所擁有的能力，
: 當蒼葉使用它，身體被會逐漸被該人格侵占。
: 蒼葉的頭痛便是「破壞」的人格為證明自己的存在的舉動。
: 蒼葉的雙生哥哥「生」亦有此種能力，只不過其媒介是眼神，本質則是「創造」、「新生」。
