-{zh-hans:'''仁波切''';zh-hk:'''寧波車'''或'''仁寶哲''';zh-tw:'''仁波切'''}-（{{bo|z=རིན་པོ་ཆེ་|p=rinboqe|w=rinpoche}}），是對[[藏传佛教|藏传佛教]][[上師|上師]]的一种尊称。

== 詞語釋義 ==
仁波切源于[[藏语|藏语]]，原義是“珍宝”，意義與[[和尚|和尚]]、[[阿闍黎|阿闍黎]]、[[喇嘛|喇嘛]]、高[[僧|僧]]相近。-{zh-hans:在[[香港|港]][[澳門|澳]]译作'''宁波车'''或'''仁宝哲''';zh-tw:在[[香港|港]][[澳門|澳]]譯作'''寧波車'''或'''仁寶哲''';zh-hk:在[[中國大陸|中國大陸]]和[[台灣|台灣]]譯作'''仁波切'''}-。在汉语中仁波切一词的用法并不普遍；尽管在意义上仁波切与[[上師|上師]]和[[活佛|活佛]]并不完全相同，但在汉语中大众常用上師和活佛来称呼仁波切。

僧人在三種情形下會被稱作仁波切，一是轉世高僧（即[[活佛|活佛]]）并被認證；二是學問堪為世人楷模者；三是此世有很高修行的成就者。也就是說轉世高僧一定是仁波切，但被稱為仁波切的人不一定是轉世高僧。

在[[西藏|西藏]]與[[不丹|不丹]]，此詞單獨使用時，亦指[[蓮花生|蓮花生]]。

这个称号并不一定用作称呼人物，有些物也称作“仁波切”。例如，西藏[[大昭寺|大昭寺]]的释迦牟尼像也被称为“觉窝仁宝哲”（Jowo Rinpoche）。<ref>[http://khejokrinpoche.blogspot.com/2011/08/blog-post_8297.html 狮吼棒喝--大藏寺祈竹仁波切问答选录]</ref> 又如，[[西藏|西藏]]三大神山之首[[冈底斯山|冈底斯山]]主峰被称为[[冈仁波钦|冈仁波钦]]，既“雪山珍宝”之意。

== 具备仁波切的条件 ==
# 须有正统传承的根本金刚上师之密法[[灌顶|灌顶]]。
# 从[[金刚萨埵|金刚持]]至自己的根本上师，其间所有密法之传承灌顶皆须圆满，不可间断。
# 须受[[阿闍黎|阿闍黎]]灌顶，精通显密[[佛法|佛法]]及[[菩提心学|菩提心学]]與[[怛特羅|怛特羅]]，并具备[[火坛仪式|火供]]等修法材料，熟悉所有经文仪规中说的修行方法。
# 自己有能力传授密法时，须经根本金刚[[上师|上师]]许可，方可對人传法灌顶。

== 冒充现象 ==
{{seealso|泉州仁波切}}
两岸三地娱乐圈和商界的部分人物信仰[[藏傳佛教|藏傳佛教]]，且一般都拜有[[上师|上师]]。<ref>{{cite web|author=李春晖|url=http://www.iceo.com.cn/mag2013/2014/0429/288367.shtml|title=活佛告诉你企业家为何要拜上师求佛法|publisher=中国企业家|date=2014-04-29|accessdate=2015-12-25}}</ref><ref name="wyb15516">{{cite web|url=http://www.guancha.cn/local/2015_05_16_319768_s.shtml|title=为什么朝阳区盛产仁波切？ 因为那是中产阶级的“私人订制”|publisher=[[观察者网|观察者网]]|date=2015-05-16|accessdate=2015-12-25}}</ref>受此风气影响，中国大陆的中产阶级也开始推崇[[藏传佛教|藏传佛教]]，以寻求心灵上的抚慰，但这种现象有时被视为自我标榜身份的工具<ref name="wyb15516"/>，而非纯粹信仰。<ref>{{cite web|url=http://www.ycwb.com/ePaper/xkb/html/2014-08/06/content_511151.htm?div=-1|title=追随那些未经审视的仁波切有时只是一种时尚，与信仰无关|publisher=[[新快报|新快报]]|date=2014-08-06|accessdate=2015-12-25|deadurl=yes|archiveurl=https://web.archive.org/web/20140806184008/http://www.ycwb.com/ePaper/xkb/html/2014-08/06/content_511151.htm?div=-1|archivedate=2014-08-06}}</ref>对活佛的迷信导致“仁波切”被讹传为活佛的代名词<ref>{{cite news|url=http://news.163.com/15/0818/13/B1A8UHHM00014AEE.html|title=“仁波切”乱象 如何辨别山寨“仁波切”|publisher=中国西藏网|date=2015-08-18|accessdate=2015-12-25}}</ref>，也导致了假冒活佛的出现，这些假冒活佛并未完整地看过一本[[佛经|佛经]]，很少遵守[[佛教戒律|佛教戒律]]，但往往自称“仁波切”且有一定数目的信徒，并与弟子存在供养关系，被网民调侃为“'''散养仁波切'''”或是“[[朝阳区_(北京市)|朝阳区]]仁波切”（因[[北京市|北京市]]朝阳区为[[中产阶级|中产阶级]]聚居区）<ref name="wyb15516"/><ref>{{cite web|url=http://news.sohu.com/s2015/dianji-1759/index.shtml|title=“仁波切”们的生财之道|publisher=搜狐新闻|date=2015-12-02|accessdate=2015-12-25}}</ref>。这些冒牌活佛中不乏假借宗教之名骗财骗色者。<ref>{{cite web|author=刘毅|url=http://news.eastday.com/eastday/13news/auto/news/china/u7ai4168849_K4.html|title=朝阳群众大有可为|publisher=中国西藏网|date=2015-06-23|accessdate=2015-12-25}}</ref>
