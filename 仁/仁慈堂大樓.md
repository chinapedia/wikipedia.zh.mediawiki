'''仁慈堂大樓'''（{{lang-pt|Santa Casa da Misericórdia}}） 是位於[[澳門特別行政區|澳門特別行政區]][[議事亭前地|議事亭前地]]旁邊的磚石[[建築物|建築物]]，為澳門慈善機構[[仁慈堂|仁慈堂]]的物業。2002年獲[[澳門政府|澳門政府]]授予[[仁愛功績勳章|仁愛功績勳章]]<ref>{{cite journal|language=zh-hant |journal=[[澳門特別行政區政府公報|澳門特別行政區政府公報]] |date=2002-12-23 |issue=51/2002 |page=1364 |title=第57/2002號行政命令 |url=http://images.io.gov.mo/bo/i/2002/51/orde-57-2002.pdf }}</ref>；2005年以[[澳門歷史城區|澳門歷史城區]]的部分被列入[[世界文化遺產|世界文化遺產]]名錄內。仁慈堂大樓的地下是澳門政府部門[[公證署|公證署]]的辦公室，二楼則為[[仁慈堂博物馆|仁慈堂博物馆]]。

== 歷史 ==
[[天主教澳門教區|天主教澳門教區]]的首任[[主教|主教]]賈耐勞 (D. Belchior Carneiro) 於1569年創立澳門[[仁慈堂|仁慈堂]]<ref name="Yip">{{cite web |language=zh-hant |publisher=[[澳門日報|澳門日報]] |date=2005-12-25 |page=C01 |title=賈尼路主教與「仁慈堂」 |author=葉迅中 }}</ref>，負責慈善救濟的工作。仁慈堂大樓建於18世紀中叶，建築物於19世紀後期加建拱廊<ref name="Yip" />，其工程於1905年完工而成為富有[[新古典主義|新古典主義]]建築風格的歐陸式建築物<ref>{{cite book | language=zh-hant |title=澳門世界遺產 |publisher=[[澳門文化局|澳門文化局]] |location=[[澳門|澳門]] |url=http://edocs.icm.gov.mo/Heritage/MWHC3.pdf }}</ref>。

昔日的仁慈堂又被稱為'''支糧廟'''，據《澳門記略校注》一書所記載，因昔日仁慈堂屬下的老人院及[[醫院|醫院]]等[[慈善機構|慈善機構]]的職員都會到該處支取薪水。<ref>{{cite web |language=zh-hant |publisher=[[澳門日報|澳門日報]] |date=2004-10-17 |page=C02 |title=仁慈堂 }}</ref>

==仁慈堂博物館==
{{main|仁慈堂博物館}}

仁慈堂博物館於2001年12月14日正式揭幕，展出了大部份來自仁慈堂值理會主席飛安達的私人珍藏。<ref>{{cite web |language=zh-hant |publisher=[[澳門日報|澳門日報]] |date=2001-12-14 |page=B03 |title=仁慈堂博物館今開幕成旅遊景點春節前對外開放 }}</ref>
