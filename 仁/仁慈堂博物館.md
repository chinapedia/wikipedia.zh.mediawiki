'''仁慈堂博物館'''（{{lang-pt|'''Núcleo Museológico'''}}）是位於[[澳門|澳門]][[議事亭前地|議事亭前地]]仁慈堂右巷二號[[仁慈堂大樓|仁慈堂大樓]]二樓的一所以[[天主教|天主教]]為主題的[[博物館|博物館]]，由澳門慈善機構[[仁慈堂|仁慈堂]]值理會管轄。

==歷史==
[[File:Santa_Casa_da_Misericórdia_de_Macau.JPG|200px]]
於1569年成立的[[仁慈堂|仁慈堂]]為了讓居民了解仁慈堂的宗旨和歷史以及其發揚助人仗義的仁愛精神<ref name="Yip">{{cite web |language=zh-hant |publisher=[[澳門日報|澳門日報]] |date=2005-12-25 |page=C01 |title=賈尼路主教與「仁慈堂」 |author=葉迅中 }}</ref><ref>{{cite web |language=zh-hant |publisher=[[澳門日報|澳門日報]] |date=2001-12-14 |page=B03 |title=仁慈堂博物館今開幕成旅遊景點春節前對外開放 }}</ref>，因而耗資約200萬[[澳門幣|澳門幣]]在建於18世紀中葉的[[仁慈堂大樓|仁慈堂大樓]]進行裝修及改善保安系統以開設'''仁慈堂博物館'''<ref name="Yip" /><ref name="wkp" />；博物館於2001年12月14日由[[澳門行政長官|澳門行政長官]][[何厚鏵|何厚鏵]]、[[天主教澳門教區主教|天主教澳門教區主教]][[林家駿|林家駿]]、仁慈堂值理會主席[[飛安達|飛安達]]等主持揭幕<ref>{{cite web |language=zh-hant |publisher=[[澳門日報|澳門日報]] |date=2001-12-15 |page=A06 |title=仁慈堂博物館昨揭幕 何厚鏵方達舟林家駿飛安達歐安利剪綵 }}</ref>；此博物館是繼[[天主教藝術博物館與墓室|天主教藝術博物館與墓室]]及[[聖物寶庫|聖物寶庫]]後，澳門第三所以及至今唯一一所非公營機構開設以宗教為題材的博物館。

為了展出更多藏品以及提升博物館質量，仁慈堂博物館於2011年上半年決定擴充博物館<ref name="novo">{{cite news |language=zh-hant |page=P03 |publisher=[[市民日報|市民日報]] |date=2011-12-07 |title=仁慈堂博物館新展廳增120件展品 }}</ref>。澳門行政長官[[崔世安|崔世安]]、[[社會文化司司長|社會文化司司長]][[張裕|張裕]]等於2011年12月6日為仁慈堂博物館新展廳主持落成揭幕禮，為該博物館增添展出超過120件藏品<ref>{{cite news |language=zh-hant |publisher=[[澳門日報|澳門日報]] |date=2011-12-07 |page=B01 |title=特首主禮 增百二珍貴展品 仁慈堂博物館新展廳揭幕 }}</ref>。

==設施==
仁慈堂博物館位於[[仁慈堂大樓|仁慈堂大樓]]二樓，博物館設有與[[天主教|天主教]]、[[耶穌會|耶穌會]]以及[[仁慈堂|仁慈堂]]相關的的瓷器、書籍文獻等合共2,000多件收藏品<ref name="wkp">{{cite web |language=zh-hant |publisher=[[華僑報|華僑報]] |date=2001-09-05 |page=14 |title=仁慈堂博物館快將開放 二千展品門票撥充善舉 }}</ref>；展品當中展有於1627年1月草擬通過並於1662年手抄而成的《澳門仁慈堂章程》正本、繪於18世紀的身兼首任[[天主教澳門教區主教列表|天主教澳門教區主教]]的仁慈堂創始人賈耐勞主教全身油畫、[[頭顱|頭顱]]遺骨和其陪葬的[[十字架|十字架]]、[[聖辣非醫院|聖辣非醫院]]內的銅鐘外，也有展出大部分由[[土生葡人|土生葡人]]飛安達及[[馬若龍|馬若龍]]捐贈從[[歐洲|歐洲]]、[[日本|日本]]及[[中國大陸|中國大陸]]等地搜購天主教傳教士過去在[[中國|中國]][[江西|江西]][[景德鎮|景德鎮]]製造到的天主教東方禮的祭獻儀式所用的器具以及刻有耶穌會徽號的[[陶瓷|陶瓷]]器皿等藏品<ref name="Yip" /><ref>{{cite web |language=zh-hant |publisher=[[澳門日報|澳門日報]] |date=2001-12-14 |page=B03 |title=館內展出文物祭器三百多件仁慈堂章程手抄本珍貴 }}</ref>。

於2011年年底開設新展廳佔地800[[平方呎|平方呎]]<ref name="novo" />；其展品大多突顯天主教在東西文化影響而製成的瓷器、象牙、木雕、銀器物品，也展有104.5厘米高於[[同治|同治]]年間燒製瓷器粉彩的[[耶穌聖心|耶穌聖心]]雕像、直徑達38厘米也於同治年間燒製的[[耶穌受洗|耶穌受洗]]青花碟以及於1936年以[[中文|中]]、[[法文|法文]]繪製的中華天主教教區全圖等展藏品<ref>{{cite web |language=zh-hant |publisher=[[澳門日報|澳門日報]] |date=2011-12-02 |page=A07  |title=特首將親臨主禮 仁慈堂博物館新廳六日開幕 }}</ref>。
