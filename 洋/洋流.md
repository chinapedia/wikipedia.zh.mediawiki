'''洋流'''亦称'''海流'''，是具有相对稳定流速和流向的大规模[[海水|海水]]运动。惟有在陸地沿岸，會因[[潮汐|潮汐]]、地形及河水的注入等影響其變化。

== 分类 ==
; 按成因分类
* 风海流（[[吹送流|吹送流]]、漂流）：亦称吹送流，漂流：在[[风|风]]力作用下形成的。
* [[密度流|密度流]]：在[[密度|密度]]差异作用下引起。
* [[傾斜流|傾斜流]]：海面因風、氣壓、降水或河水流入等原因而傾斜，其所引起的海流。
* [[補償流|補償流]]（[[湧升流|湧升流]]和下降流）：因为海水挤压或分散引起。<ref>{{Cite book | title = 地理图册 普通高中课程标准实验教材| publisher = 浙江教育出版社 | language = zh-hans }} </ref>

; 按冷暖性质分类
* [[暖洋流|暖流]]：本身水温比周围水温高。
* [[寒洋流|寒流]]：由高緯度流向中高緯度，本身水温比周围水温低。
* [[涼流|涼流]]：由中高緯度流向低緯度，本身水溫比周圍水溫低，較寒流高溫。

; 按地理位置分类
[[赤道流|赤道流]]、[[大洋流|大洋流]]、[[极地流|极地流]]及[[沿岸流|沿岸流]]等。

== 洋流受到的力 ==
* 风的应力：包括风对洋流的摩擦力和风对洋流的[[压力|压力]]
* 海水内部的[[壓強梯度力|壓強梯度力]]和[[摩擦力|摩擦力]]
* 海水密度差異：包括溫度和海水鹽度差异所造成的力，通常發生在極區
* [[地转偏向力|地转偏向力]]
* 海陸相接地形：影響其流向

==== 反气旋型大洋环流 ====
[[信风带|信风带]]作用下的[[信风漂流|信风漂流]]（南、北赤道暖流）向西流动，遇[[大陆|大陆]]后，一部分海水因信[[风切应力|风切应力]]南北向速度分量不均和补偿作用而折回，便形成了自西向东的[[赤道逆流|赤道逆流]]和[[赤道潜流|赤道潜流]]；另一部分信风漂流向[[高纬|高纬]]的南北分流，在北太平洋形成[[黑潮|黑潮]]、在南太平洋形成[[东澳大利亚洋流|东澳大利亚洋流]]、在南大西洋形成[[巴西洋流|巴西洋流]]、在北大西洋形成[[北大西洋湾流|北大西洋湾流]]、在南印度洋形成[[莫桑比克洋流|莫桑比克洋流]]。<ref name="XJ">{{Cite book | title = 地理 普通高中课程标准实验教材| publisher = 湖南教育出版社 | language = zh-hans }} </ref>

[[西风带|西风带]]作用下的[[西风漂流|西风漂流]]向东流动，遇大陆后，向两侧的高纬低纬分流，形成补偿流，向[[低纬|低纬]]流的洋流有：北太平洋的[[加利福尼亚洋流|加利福尼亚洋流]]、南太平洋的[[秘鲁洋流|秘鲁洋流]]、北大西洋的[[加那利洋流|加那利洋流]]、南大西洋的[[本格拉洋流|本格拉洋流]]、南印度洋的[[西澳大利亚洋流|西澳大利亚洋流]]。<ref name="XJ"/>

信风漂流、信风漂流遇大陆后向高纬转向的补偿流、西风漂流、西风漂流遇大陆后向低纬转向的补偿流，便构成各大洋副热带海区（仅指大洋的如下海区：北太平洋、南太平洋、北大西洋、南大西洋、南印度洋）的反气旋型大洋环流。

==== 气旋型大洋环流 ====
由西风漂流、西风漂流遇到陆地后向北分支形成的补偿流、[[极地东风带|极地东风带]]形成的中高纬大洋西岸的洋流组成北半球中高纬海区的[[气旋|气旋]]型大洋环流。<br />该环流在北太平洋上有：[[北太平洋暖流|北太平洋暖流]]、[[阿拉斯加洋流|阿拉斯加洋流]]、[[千岛寒流|千岛寒流]]；在北大西洋上有：[[北大西洋暖流|北大西洋暖流]]、[[挪威暖流|挪威暖流]]、[[东格陵兰寒流|东格陵兰寒流]]。<ref name="XJ"/>

==== 北印度洋季风漂流 ====
北印度洋受南亚季风的影响，冬半年盛行东北季风，形成东北季风漂流，夏半年盛行西南季风，形成西南季风漂流。

==== 南极绕极环流 ====
在极地东风带的吹拂下形成环绕南极洲大陆一周的南极绕极环流，再往低纬方向为环绕南极大陆一周的西风漂流<ref name="XJ"/>，因本海区自然特征比较一致，有些学者把南极外围海区称为[[南冰洋|南冰洋]]，另一部分学者认为大洋应有其对应的[[大洋中脊|大洋中脊]]而不承认“南冰洋”这一称谓。

== 洋流的各種影響 ==
=== 对[[气候|气候]]的影响 ===
总体来说，暖流增加溫度和溼度，寒流降低溫度和溼度。
==== 对[[气温|气温]]的影响 ====
* 洋流使低纬度的热量向高纬度的热量传输，特别是暖流的贡献。
* 洋流对同纬度大陆两岸气温的影响：暖流经过的大陆沿海气温高，寒流经过的大陆沿海气温低。
==== 对[[降水|降水]]和[[雾|雾]]的影响 ====
* 暖流上空有[[热量|热量]]和[[水氣|水氣]]向上输送，使得[[层结|层结]]不稳定、空气湿度增大而易产生降水。而寒流产生[[逆温|逆温]]，层结稳定，水氣不易向上输送，蒸发又弱，下层[[相对湿度|相对湿度]]有时虽然很大，但只能成雾，不能成雨。
* 寒流表面多[[平流雾|平流雾]]，在以下几种情况出现：
# '''[[海陆风|海陆风]]雾：'''陆风在白天流到寒流表面而形成平流雾；
# '''海雾：'''在寒暖流交汇处，风自暖流表面吹至寒流表面而形成平流雾。

=== 对海洋生物的影响 ===
*在寒暖流交汇处或上升补偿流区，往往形成较大的[[渔场|渔场]]<ref name="XJ"/>，世界四大渔场及其洋流成因如下：
#[[北海道渔场|北海道渔场]]：位于[[日本|日本]][[北海道岛|北海道岛]]附近，日本暖流和千岛寒流交汇。
#[[北海渔场|北海渔场]]：位于欧洲[[北海_(大西洋)|北海]]，北大西洋暖流与极地东风带带来的北冰洋南下冷水交汇。
#[[秘鲁渔场|秘鲁渔场]]：海岸盛行[[东南信风|东南信风]]，为[[离岸风|离岸风]]，导致[[上升补偿流|上升补偿流]]（亦称涌流）。
#[[纽芬兰渔场|纽芬兰渔场]]：[[加拿大|加拿大]][[纽芬兰岛|纽芬兰岛]]附近，墨西哥灣流和[[拉布拉多寒流|拉布拉多寒流]]交汇。
*[[赤道|赤道]]地区的[[企鹅|企鹅]]：在太平洋东部赤道地区的[[科隆群岛|科隆群岛]]（又名加拉帕戈斯群岛），有企鹅分布，是秘鲁寒流的因故。

=== 对海洋污染的作用 ===
有利也有弊：既可以使污染物因迅速扩散而加快其稀释和净化的速度，也相应地使污染范围扩大。<ref name="XJ"/>

==== 洋流重要性 ====

#將多個不同洋域的熱能傳送至不同洋區（熱能上的平衡）； 
#將多個不同洋域的養分送往不同的洋區；
#將多個不同洋域含氧量不同的海水因洋流分佈往不同洋區。

因此洋流在地球的生物圈和物理環境上起了重要而積極的平衡和帶動作用，對大部分生物（包括陸地上）有存活上的積極幫助。
