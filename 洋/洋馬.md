'''洋馬'''（{{lang-ja|ヤンマー}}，{{lang-en|Yanmar}}），是[[日本|日本]]'''洋馬控股株式會社'''所領導企業集團總稱，以及[[工程作業車輛|工程作業車輛]]、[[農業機械|農業機械]]、[[柴油引擎|柴油引擎]]及小型[[船舶|船舶]]的[[品牌|品牌]]。

== 沿革 ==
*1912年－3月，創立{{nihongo|'''山岡發動機工作所'''|山岡發動機工作所|Yamaoka Hatsudoki Kosakusho}}，生產煤氣發動機。
*1920年－11月，農用立形石油發動機試作完成。
*1921年－3月，「{{lang|ja|ヤンマー}}」商標登錄，農用横形變量式石油發動機製造販售。
*1925年－輸出石油發動機至[[中國|中國]]南部。
*1928年－1月，設置[[臺灣|臺灣]][[臺北市|臺北]]辦事處。
*1929年－5月，設置中國[[上海|上海]]辦事處，以「洋-{}-馬牌」在[[江蘇|江蘇]]、[[浙江|浙江]]、[[山東|山東]]3省通路販售。
*1930年－立形[[两冲程循环|-{zh-cn:两冲程循环;zh-hk:二衝程循環;zh-tw:二行程循環;}-]][[柴油引擎|柴油發動機]]試作完成。
*1931年－2月，工作所公司化。與中國[[大連|大連]]的福昌公司合作，擴展[[满洲地区|满洲地区]]販售通路。 
*1935年－1月，與[[北京|北京]]的三昌洋行合作，擴展中國北部販售通路。
*1936年－1月，{{nihongo|'''山岡内燃機株式会社'''|山岡內燃機株式會社|Yamaoka Nainenki Kabushikigaisha}}設立。　
*1940年－11月，山岡內燃機吸收合併山岡發動機工作所。
*1952年－2月，社名變更{{nihongo|'''洋馬柴油發動機株式會社'''|ヤンマーディーゼル株式会社|Yanmar Diesel Co., Ltd.}}<ref>在臺灣登記「日商野-{}-馬柴油引擎股份有限公司」。</ref>。
*1960年－8月，與[[臺灣銀行|臺灣銀行]]合辦的中國農業機械股份有限公司在臺北設立。
*1993年－11月，開設駐[[中國大陸|中國大陸]]上海事務所。
*1994年－9月，在中國大陸[[江蘇省|江蘇省]][[江陰市|江陰市]]成立江蘇洋-{}-馬農機有限公司 。
*1997年－10月，在中國大陸江蘇省[[無錫市|無錫市]]合資成立洋-{}-馬農機（中國）有限公司。
*2001年－9月，在臺灣註冊日商野-{}-馬柴油機工程股份有限公司，11月於[[高雄市|高雄市]]正式成立。
*2002年－7月，洋馬柴油發動機轉型[[控股公司|控股公司]]，更名{{nihongo|'''洋馬株式會社'''|ヤンマー株式会社|Yanmar Co., Ltd.}}。
*2003年－1月，在中國大陸設立洋-{}-馬發動機（上海）有限公司。2月，與山東時風集團合辦的山東時風洋-{}-馬發動機有限公司設立。 
*2007年－2月，山東時風洋-{}-馬發動機更名洋-{}-馬發動機（山東）。3月，在[[香港|香港]]註冊洋-{}-馬工程（香港）有限公司，翌年11月正式成立。
*2011年－11月，在中國大陸[[黑龍江省|黑龍江省]][[哈爾濱市|哈爾濱市]]成立哈爾濱洋-{}-馬農業機械有限公司。
*2013年－4月，以股票轉移方式新設控股公司{{nihongo|'''洋馬控股株式會社'''|ヤンマーホールディングス株式会社|Yanmar Holdings Co., Ltd.}}。

== 名稱及商標 ==
