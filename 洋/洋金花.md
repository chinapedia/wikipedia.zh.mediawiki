'''洋金花'''（[[學名|學名]]：''{{lang|la|Datura metel}}''）屬[[茄科|茄科]]植物，是[[中醫|中醫]]學常用的草藥，但具有毒性，不能過量服用。洋金花多數生長於村邊、路旁、荒地，在濕潤向陽的土地較為常見。此花有可能原產自東[[印度|印度]]<ref>來源：{{cite web |url=http://www.ars-grin.gov/cgi-bin/npgs/html/taxon.pl?13315 |title=Datura metel L.
 |accessdate=20/10/07 |work=Germplasm Resources Information Network |publisher=United States Department of Agriculture |language=en}}</ref>或[[美洲|美洲]]，在[[北美洲|北美洲]]視為[[觀賞植物|觀賞植物]]，而香港則稱之為「[[香港|香港]]四大毒草」之一。<ref>來源：{{cite web |url=http://sc.afcd.gov.hk/gb/hknature.net/chi/ecodiary/diary_30a.html |title_i= |title=《Ecological Diary 生態日記》 |accessdate=20/10/07 |author=漁農自然護理署 |date=20/9/01 |work=香港自然網 }}{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

== 形態特徵 ==
洋金花為[[一年生植物|一年生]]直立[[草本|草本]]，約高0.5-1.5米，全株近乎無毛，莖呈深紫色。

葉互生，在莖上部呈假對生，葉柄長2-6厘米，葉身成卵形或寬卵形，長5-20厘米，寬4-l5厘米，葉身基部不對稱，頂端漸尖，側[[葉脈|脈]]每邊約4-6條；葉邊全緣、微波狀、或具不規則短齒。

花直立而單生，一般生長於葉腋或枝叉間，花梗長約1厘米，花萼圓筒形，長4-9厘米，直徑2厘米；花冠呈漏斗形，有白色、淡黃色、淡紫色之分，長14-20厘米，檐部直徑6-10厘米，野生類種為單[[花瓣|瓣]]，培植的種類則有二重瓣或三重瓣，裂片有短尖，短尖下有3條縱脈紋。

雄蕊有5枚（培植的種類可達15枚），花絲貼生於花冠筒內，長1-1.2厘米，而雌蕊則有1枚，子房疏生短剌毛，花柱11-16厘米，頭呈棒狀。蒴果近球形，直徑約3cm，外殼刺疏而短，不規則4瓣裂。種子近腎形，顏色淡褐，直徑3毫米。花期及果期為3月至12月。<ref>來源：{{cite book |author=中國科學院中國植物志編輯委員會 |title=《中國植物志 第67卷 第1分冊》 |url=http://cvh.ibcas.ac.cn/zhiwuzhi/page/67(1)/147.pdf |format=PDF |accessdate=20/10/07 |location=北京 |publisher=科學出版社 |year=1978 |isbn=9787030024565 |pages=147-148 |deadurl=yes |archiveurl=https://web.archive.org/web/20071025194149/http://cvh.ibcas.ac.cn/zhiwuzhi/page/67(1)/147.pdf |archivedate=2007-10-25 }} </ref>

=== 與曼陀羅花的分別 ===
洋金花（Datura metel L）與[[曼陀羅花|曼陀羅花]]（Datura stramonium）是[[生物分類法|生物分類法]]中的同一[[属_(生物)|屬]]，外形亦十分相似。不過，曼陀羅花的蒴果是規則4瓣裂，花萼有5棱，洋金花蒴果則為不規則4瓣裂，花萼沒有棱。另外曼陀羅花的花冠亦比較細小。<ref>來源：{{cite book|author=中國科學院中國植物志編輯委員會|title=《中國植物志 第67卷 第1分冊》|url=http://cvh.ibcas.ac.cn/zhiwuzhi/page/67(1)/144.pdf|format=PDF|accessdate=20/10/07|location=北京|publisher=科學出版社|year=1978|isbn=|pages=144-146|deadurl=yes|archiveurl=https://web.archive.org/web/20071025194129/http://cvh.ibcas.ac.cn/zhiwuzhi/page/67(1)/144.pdf|archivedate=2007-10-25}} </ref>

== 醫藥用途及其毒性 ==
{{medical}}
洋金花全株皆含莨菪烷類[[生物鹼|生物鹼]]，包括[[東莨菪鹼|東莨菪鹼]]（hyoscine）、[[莨菪鹼|莨菪鹼]]（hyoscyamine）、[[阿托品|阿托品]]（Atropine）這些物質有麻醉、止痛、鬆弛肌肉等作用。經製煉的洋金花的花可用作麻醉，只需口服3-5[[克|克]]，5分鐘就會發作，效力可長達5-6小時。如果服用過多，就會出現幻覺、中毒等病徵，嚴重者會致命。由於使用稍微過量便可致毒，故必須慎用。
從[[中醫|中醫]]角度，多數以花入藥，果、葉、根亦可，常於夏秋採集，洗淨曬乾備用。洋金花味苦、辛，性溫，有大毒，主治麻醉鎮痛、平喘止咳。而在古時[[印度|印度]]，因為洋金花含有麻醉作用，能減輕症狀，所以會廣泛用作治療[[癲癇症|癲癇症]]、精神病、[[心臟病|心臟病]]，甚至[[腹瀉|腹瀉]]等病症。而在[[越南|越南]]有人將洋金花的干花和葉子卷成[[香煙|香煙]]吸食。<ref>來源：{{cite web |url=http://www.pfaf.org/database/plants.php?Datura+metel |title=Datura metel |accessdate=20/10/07 |work=Plants For A Future database report |publisher=Plants For A Future |language=en}}</ref>

== 分佈 ==
野生的洋金花常見於[[熱帶|熱帶]]和[[亞熱帶|亞熱帶]]地區，而[[溫帶|溫帶]]地區則以人工栽種為主。在中華地區，[[台灣|台灣]]、[[香港|香港]]、[[福建|福建]]、[[廣東|廣東]]、[[廣西|廣西]]、[[雲南|雲南]]、[[貴州|貴州]]等地都常見野生洋金花，而江蘇、浙江，甚至北方城市都有人工培植。
