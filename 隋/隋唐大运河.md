'''隋唐大运河'''是[[隋代|隋代]]统一南北以后，将以前已有的自然水道加上隋代开凿的运河组成了一条自江南一直到京城洛阳的路上水道，主要目的是运送首都所需的物资。运河以[[洛阳|洛阳]]为中心，分别由公元584年开凿的向西到陕西西安的[[廣通渠|廣通渠]]，603年开凿的经山东临清北上至河北涿郡的[[永濟渠|永濟渠]]，公元605年向南开凿到江苏清江的[[通濟渠|通濟渠]]和公元610年连接江苏镇江和浙江杭州的[[江南运河|江南运河]]构成。隋唐大运河是世界史上最长的运河。但是随着时间的推移，隋朝大运河部分河段失去通航功能，随后被元世祖[[忽必烈|忽必烈]]所修的[[京杭大运河|京杭大运河]]（仅古[[邗沟|邗沟]]、[[江南运河|江南运河]]等河段与隋朝大运河有重合）取代。

== 概况 ==
隋唐大运河以洛阳为中心，南起杭州，北到北京，隋朝开凿全长2700公里，跨越地球10多个纬度，纵贯在中国最富饶的东南沿海和华北大平原上，经过今浙江、江苏、安徽、河南、山东、河北、北京七个省市，通达黄河、淮河、长江、钱塘江、海河五大水系，是中国古代南北交通的大动脉，在中国的历史上产生过巨大的作用，是中国古代劳动人民创造的一项伟大的水利建筑工程，也是世界上开凿最早、规模最大的运河。后经元朝取直疏浚，全长1794公里，成为现今的京杭大运河。京杭大运河利用了隋朝大运河不少河段，缩短了900多公里的航程。 自清末改漕运为海运，大运河才不再是国家经济的大动脉。

隋代的政治和军事中心位于目前的中国渭河平原，尤其是[[中原|中原]]地区，在平定[[南陈|南陈]]期间，为了运兵南下，开始修建运河。待[[开皇|开皇]]九年（589年）三月平定[[南陈|南陈]]后，为了将江南的粮食和纺织品运送到北方，又陆续开凿了数条运河。最终形成洛阳为中心，首尾相连的几条运河。

== 历史 ==
[[隋文帝|隋文帝]]开皇七年（587年），隋为兴兵伐陈，从今淮安到扬州，开[[山阳渎|山阳渎]]，和[[广通渠|广通渠]]后又整治取直，中间不再绕道[[射阳湖|射阳湖]]。[[隋炀帝|隋炀帝]][[大业_(年号)|大业]]元年605年，开[[通济渠|通济渠]]，工程西段自今洛阳西郊引谷、洛二水入[[黄河|黄河]]，东段自[[荥阳|荥阳]]汜水镇东北引黄河水，循[[汴水|汴水]]（原[[淮河|淮河]]支流），经[[商丘|商丘]]、[[宿县|宿县]]、[[泗县|泗县]]入淮通济渠，又名汴渠，是漕运的干道。隋炀帝大业四年608年又开[[永济渠|永济渠]]，引黄河支流[[沁水|沁水]]入今[[卫河|卫河]]至[[天津|天津]]，继溯[[永定河|永定河]]通今[[北京|北京]]。公元610年继开江南运河，由今[[镇江|镇江]]引江水经[[常州市|常州]]、[[无锡|无锡]]、[[苏州|苏州]]、[[嘉兴|嘉兴]]至[[杭州|杭州]]通[[钱塘江|钱塘江]]。至此，建成以[[洛阳|洛阳]]为中心，由永济渠、通济渠、邗沟和江南运河连接而成，西接大兴，南通余杭，北通涿郡，全长折今2700余公里的大运河。

唐興，還都大興城，渭水猶不足用，高宗間，常就食洛陽。天寶元載，以韋堅為使治渠。乃佔咸陽，壅渭為堰，絕灞、滻而東，因古渠，過望春樓，注永豐倉下，復與渭合，二年而成，名曰廣運潭。是歲，渠漕山東粟四百萬石。天寶中，每歲水陸運二百五十萬石入關。[[唐代|唐代]]诗人[[皮日休|皮日休]]作诗道：“万艘龙舸绿丛间，载到扬州尽不还。应是天教开汴水，一千余里地无山。尽道隋亡为此河，至今千里赖通波。若无水殿龙舟事，共禹论功不较多。”（《[[汴河|汴河]]怀古二首》）

== 意义 ==
大运河为后世经济发展起到了很大的促进作用，在大运河修筑成功和此后发挥作用的五百余年时间之内成为沟通运河沿线的重要政治、经济、文化的纽带。但在[[隋炀帝|隋炀帝]]大业年间修建过程中过重的劳役加重了普通百姓的负担，隋炀帝太急著建運河使得民怨沸腾，成为[[隋末民变|隋末民变]]的诱因之一。<ref>[http://www.apdnews.com/news/35167.html 隋唐大运河“申遗” 结果明年出炉]，亚太日报，2013年9月26日</ref>

== 遗址 ==
隋唐大运河通济渠、大部分河段已经湮没成为遗址，仅存若干河段。
