'''岩燕'''（''Ptyonoprogne rupestris''），一種生活在山地的小型[[燕科|燕科]]鳥類。

== 生态环境 ==
海拔1500-5000米的高山峡谷，喜在[[湖|湖]]、鱼池、沼泽、水库、江河等的水面迥翔。

== 分布地域 ==
岩燕分布在[[南欧|南欧]]和[[南亚|南亚]]的广大地区，冬天会迁往[[北非|北非]]和[[印度|印度]]等地，在[[中国|中国]]，它们的分布地区西起[[新疆|新疆]]、[[青海|青海]]、[[甘肃|甘肃]]、[[宁夏|宁夏]]、[[内蒙古|内蒙古]]、[[四川|四川]]、[[西藏|西藏]]、[[云南|云南]]、东抵[[山西|山西]]、[[河北|河北]]和[[辽宁|辽宁]]。

== 特征 ==
体长14-15厘米，上身褐色，下身暗黑色。

== 食物 ==
主要以昆虫为食。

== 繁殖 ==
每次产3-5卵。
