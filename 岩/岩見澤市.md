'''岩見澤市'''（{{jpn|j=岩見沢市|hg=いわみざわし|rm=Iwamizawa shi|f=y}}）為位於[[日本|日本]][[北海道|北海道]][[空知綜合振興局|空知綜合振興局]]南部的城市，位於[[石狩平原|石狩平原]]的東部，也是空知綜合振興局辦公室的所在地。轄區內的水稻耕作面積及收穫為北海道第一。

由於過去在北海道開拓時期，為了開採[[幌內煤田|幌內煤田]]（位於現在的[[三笠市|三笠市]]），興建了[[札幌|札幌]]到[[幌內|幌內]]之間的道路，當時在[[幾春別川|幾春別川]]旁（位於現在的岩見澤市轄區北部）建了讓建築工人休息的場所，稱為「浴澤」（{{lang|ja|ゆあみさわ}}、Yuamisawa），後逐漸被稱為「岩見澤」（{{lang|ja|いわみざわ}}、Iwamizawa），也成為現在城市的名稱。<ref>{{cite web |url=http://www.city.iwamizawa.hokkaido.jp/i/sisei/iwa/chimei.htm |title=地名的由來 |publisher=岩見澤市政府 |accessdate=2008年1月29日 |language=ja |deadurl=yes |archiveurl=https://web.archive.org/web/20070927030228/http://www.city.iwamizawa.hokkaido.jp/i/sisei/iwa/chimei.htm |archivedate=2007年9月27日 }}</ref>

== 歷史 ==
=== 沿革 ===
* 1878年：開拓使在延著[[幾春別川|幾春別川]]在通往[[幌內煤礦|幌內煤礦]]的地方設置旅人休息所。<ref>{{cite web |url=http://www.city.iwamizawa.hokkaido.jp/i/sisei/iwa/rekisi.htm |title=市的簡介 |publisher=岩見澤市政府 |accessdate=2008年1月29日 |language=ja |deadurl=yes |archiveurl=https://web.archive.org/web/20071102170912/http://www.city.iwamizawa.hokkaido.jp/i/sisei/iwa/rekisi.htm |archivedate=2007年11月2日 }}</ref>
* 1882年11月13日：[[幌內鐵道|幌內鐵道]]的幌內（[[三笠市|三笠市]]）至手宮（[[小樽市|小樽市]]）間路段通車。
* 1883年：開始勸導[[士族|士族]]移民，並開始有大量士族進入此地。
* 1884年10月6日：設置岩見澤村並設置戶長役場設置。
* 1891年：[[北海道煤礦鐵道|北海道煤礦鐵道]]岩見澤至歌志內（現在的[[歌志內市|歌志內市]]）間路段通車通（舊[[歌志內線|歌志內線]]、現在的[[函館線|函館線]]）。
* 1892年2月：栗澤村（後來的[[栗澤町|栗澤町]]）從岩見澤村分村。
* 1897年10月：[[北海道廳|北海道廳]]在此設置[[空知支廳|空知支廳]]。
* 1900年7月1日：成為北海道一級村制，同時[[北村_(北海道)|北村]]從岩見澤村中分村。
* 1906年2月1日：改制為岩見澤町。
* 1906年4月1日：栗澤村成立為北海道二級村。
* 1909年4月1日：栗澤村改為北海道一級村。
* 1919年4月1日：北村成為北海道二級村。
* 1943年4月1日：改制為岩見澤市，為北海道第10個、全日本第201個城市。
* 1949年4月1日：栗澤村改制為栗澤町。
* 1985年：[[國鐵|國鐵]][[萬字線|萬字線]]停駛。
* 1986年：北海道21世紀博覽會開始。
* 1987年：[[北海道旅客鐵道|北海道旅客鐵道]][[幌內線|幌內線]]停駛。
* 2006年3月26日：[[栗澤町|栗澤町]]和[[北村_(北海道)|北村]]併入岩見澤市。

== 姊妹市・友好都市 ==
=== 日本 ===
* [[南阿爾卑斯市|南阿爾卑斯市]]（[[山梨縣|山梨縣]]）：由於北村最早的開墾者[[北村雄治|北村雄治]]來自[[若草町|若草町]]，2000年[[北村_(北海道)|北村]]與若草町締結友好都市；在北村和若草町各自被合併之後，2004岩見澤市與南阿爾卑斯市重新締結為姊妹都市。
* [[讚岐市|讚岐市]]（[[香川縣|香川縣]]）：由於[[栗澤町|栗澤町]]小西地區最早的開墾者來自[[長尾町|長尾町]]，1992年栗澤町與長尾町締結友好都市。
* [[砺波市|砺波市]]（[[富山縣|富山縣]]）：由於栗澤町砺波地區最早的開墾者來自[[砺波市|砺波市]]，1992年起栗澤町與砺波市開始有固定交流活動。

=== 海外 ===

* {{flagicon|USA}} {{Tsl|en|Canby, Oregon|坎比}}（美國 [[奧勒岡州|奧勒岡州]]）：於1989年由栗澤町與其締結姊妹都市。
* {{flagicon|PRC}} [[玉泉鎮|玉泉鎮]]（[[中華人民共和國|中華人民共和國]] [[黑龍江省|黑龍江省]] [[阿城市|阿城市]]）於1982年由栗澤町與簽訂交流議定書，進行農業技術交流。
