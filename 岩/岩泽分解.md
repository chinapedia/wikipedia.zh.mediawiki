数学中，[[半单李群|半单李群]]的'''岩泽分解''' KAN 推广了实方阵能写成一个[[正交矩阵|正交矩阵]]和[[上三角矩阵|上三角矩阵]]的乘积（[[格拉姆-施密特正交化|格拉姆-施密特正交化]]之推论）。以创立者[[日本|日本]][[数学家|数学家]][[岩泽健吉|岩泽健吉]]命名。

==定义==

*''G'' 是一个连通半单实李群。
*<math> \mathfrak{g}_0 </math> 是 ''G'' 的[[李代数|李代数]]。
*<math> \mathfrak{g} </math> 是 <math> \mathfrak{g}_0 </math> 的[[复化|复化]]。
*θ 是 <math> \mathfrak{g}_0 </math> 的一个[[嘉当对合|嘉当对合]]。
*<math> \mathfrak{g}_0 = \mathfrak{k}_0 \oplus \mathfrak{p}_0 </math> 是相应的[[嘉当分解|嘉当分解]]。
*<math> \mathfrak{a}_0 </math> 是 <math> \mathfrak{p}_0 </math> 的一个极大阿贝尔子空间。
*Σ 是 <math> \mathfrak{a}_0 </math>的限定根，对应于 <math> \mathfrak{a}_0 </math> 作用在 <math> \mathfrak{g}_0 </math> 上的特征值。
*Σ<sup>+</sup> 是 Σ 的正根。
*<math> \mathfrak{n}_0 </math> 是由 Σ<sup>+</sup> 的根空间的和给出的幂零李代数。
*''K'',''A'', ''N'' 分别是由 <math> \mathfrak{k}_0, \mathfrak{a}_0 </math> 和 <math> \mathfrak{n}_0 </math>生成的子群。

那么，<math> \mathfrak{g}_0 </math> 的'''岩泽分解'''为
:<math>\mathfrak{g}_0 = \mathfrak{k}_0 + \mathfrak{a}_0 + \mathfrak{n}_0</math>，
''G'' 的岩泽分解为：
:<math>G=KAN</math>.

''A'' （或等价的 <math> \mathfrak{a}_0 </math>）的[[维数|维数]]称为 ''G''的'''实秩''' 。

岩泽分解对一些不连通半单李群''G'' 也成立，此时 ''K'' 为（不连通）[[极大紧子群|极大紧子群]]并假定 ''G'' 的[[中心|中心]]有限。

==例子==
如果 ''G''=''GL<sub>n''</sub>('''R''')，那么可取 ''K'' 为正交矩阵，''A'' 为正对角矩阵，''N'' 为[[幂幺群|幂幺群]]（对角元全1的上三角矩阵）。
