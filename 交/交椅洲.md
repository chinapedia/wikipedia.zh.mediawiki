'''交椅洲'''（{{lang-en|Kau Yi Chau}}，亦作{{lang|en|Kau I Island}}或{{lang|en|Kau-i-chau}}），又稱'''大交椅洲'''（{{lang-en|Tai Kau Yi Chau}}），是[[香港|香港]]一座無人[[島嶼|島嶼]]。

交椅洲外型像[[交椅|交椅]] (古代中國傳統椅子)，因而得名。

== 地理位置 ==
交椅洲屬於[[離島區|離島區]]，位於[[坪洲|坪洲]]及[[小交椅洲|小交椅洲]]以東，[[喜靈洲|喜靈洲]]及[[周公島|周公島]]以北。

== 島上設施 ==
交椅洲並沒有民居。近岸有一座建在礁石上的編號81[[燈塔|燈塔]]。山上有由[[海事處|海事處]]設立，用作海上交通控制的遙距[[雷達|雷達]]站<ref>{{cite web |author1=香港海事處 |title=遙距雷達站 |url=https://www.mardep.gov.hk/hk/pub_services/ocean/vts_radarstn.html |accessdate=2018-12-21}}</ref>和[[差分全球定位系統|差分全球定位系統]]中央基準站<ref>{{cite web |author1=香港海事處 |title=差分全球定位系統修正服務 |url=https://www.hydro.gov.hk/hk/dgps.php |accessdate=2018-12-21}}</ref>，和一公共[[直升機|直升機]]坪<ref>{{cite web |title=香港地方 - 地方 - 公共直升機坪 |url=http://www.hk-place.com/view.php?id=160 |accessdate=2018-12-21}}</ref>。

==美國軍艦擱淺==
[[Image:USS_Regulus_(AF_57).jpg|thumb]]
[[File:Typhoon_Rose_(1971)_ship_damage.JPG|thumb]]
1971年8月17日[[颱風露絲|颱風露絲]]吹襲期間，美國軍艦{{link-en|王子號糧食補給艦|USS Regulus (AF-57)|Regulus}}在交椅洲海面擱淺，當局在接著三周試圖將之重新浮起，惟失敗告終。

Regulus遂在1971年9月10日退役和拆毀。<ref>Ross M. Contini; EM 5 Aboard the Regulus at the time of wreck.</ref>

==大小交椅洲發展計劃==
{{see also|香港10號幹線#歷史}}
在2005年，[[香港政府|香港政府]]曾經擬議進行交椅洲[[填海|填海]]計劃，將大小交椅洲連接起來，發展該地為全新[[貨櫃碼頭|貨櫃碼頭]]和港口後勤用地，而且是建議中的[[10號幹線|10號幹線]]連接[[香港島|香港島]]和[[大嶼山|大嶼山]][[香港迪士尼樂園度假區|香港迪士尼樂園度假區]]的[[海底隧道|海底隧道]]中途站，但是因為受到環境保護團體的強烈反對下，政府才被迫擱置大小交椅洲發展計劃。

== 東大嶼都會 ==
{{Main|東大嶼都會}}
香港政府於2012年公佈《優化土地供應策略：維港以外填海及發展岩洞》增加[[香港土地供應|土地供應]]，除了5個近岸填海選址，另外提出於中部水域填海建造[[人工島|人工島]]。人工島範圍包括於[[喜靈洲|喜靈洲]]及交椅洲附近水域填約1,000公頃土地，並保留原先島嶼之[[海岸線|海岸線]]。2016年底《[[香港2030+|香港2030+：跨越2030年的規劃遠景與策略]]》研究提出整合[[梅窩|梅窩]]未被充分利用的土地，並加上人工島之新增土地，開拓作香港未來第三個[[中央商务区|核心商業區]]，容納最多70萬人口及20萬職位。
