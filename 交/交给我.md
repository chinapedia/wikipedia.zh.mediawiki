《'''交給我'''》（{{lang-en|'''Give It 2 Me'''}}）是美國女歌手[[瑪丹娜|瑪丹娜]]在2008年6月所發行的單曲，收錄於她第十一張[[錄音室專輯|錄音室專輯]]《'''[[娜式糖|娜式糖]]'''》（{{lang-en|'''Hard Candy'''}}）中，該單曲在[[Billboard_Hot_100|美國單曲榜]]最高名次為第57名<ref>[http://www.billboard.com/artist/308786/madonna/chart?f=379 美國告示牌官網瑪丹娜單曲資料庫]</ref>。

== 歌曲簡介 ==
〈交給我〉由[[瑪丹娜|瑪丹娜]]和音樂創作高手菲董（Pharrell Williams）兩人共同譜寫編製，在[[電子合成器|電子合成器]]層層堆疊的音樂下，聽起來卻又層次分明，帶點[[龐克|龐克]]的快版主節奏，在間奏之後又在歌曲中融入Hip Hop風味，讓聽者不由自主的隨之起舞。〈交給我〉是一首標準的娜式[[電音舞曲|電音舞曲]]，它在專輯《[[娜式糖|娜式糖]]》（{{lang-en|Hard Candy}}）當中算是耐聽度較高的一首舞曲。


== 商業成績 ==
〈交給我〉是專輯《娜式糖》（{{lang-en|Hard Candy}}）所推出的第二首單曲，雖然上一首單曲〈4 Minutes〉拿下了第3名，但似乎並未影響到第二首單曲的成績。〈交給我〉在[[Billboard_Hot_100|美國單曲榜]]最高名次僅獲得第57名，在榜內也只有停留1週的時間，不過它在舞曲榜則到達了榜首位置<ref>[http://www.billboard.com/charts/dance-club-play-songs/2008-08-09 2008-08-09告示牌舞曲榜]</ref>，成為瑪丹娜在該榜獲得的第39首冠軍單曲<ref>[http://www.billboard.com/artist/308786/madonna/chart?f=359 美國告示牌官網瑪丹娜舞曲榜資料庫]</ref>。〈交給我〉在英國單曲榜也進入了前十名，最高成績為第7名<ref>[http://www.officialcharts.com/Artist/28948/MADONNA/ 英國排行榜瑪丹娜專輯單曲資料庫]</ref>。
