'''交城县'''是[[中華人民共和國|中国]][[山西省|山西省]][[吕梁市|吕梁市]]所辖的一个[[县|县]]。位於[[呂梁山|呂梁山]]東麓，[[太原盆地|太原盆地]]西緣，[[307國道|307國道]]經過<ref>http://www.sx.chinanews.com/news/2009/1230/15965.html</ref>。

== 歷史 ==
縣境內發現有新石器時代[[瓦窯遺址|瓦窯遺址]]、[[磁窯遺址|磁窯遺址]]<ref>http://www.sx-jc.gov.cn/sub/zjjc_show.asp?id=5105&cataid=A00620005{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

[[隋|隋]][[開皇|開皇]]十六年（596年）置縣，因县治居[[汾河|汾]][[孔河|孔]]二河相交处，故名。

[[唐|唐]][[天授|天授]]二年（691年）縣治由今[[太原|太原]][[古交|古交]]移置今址。

[[明代|明代]]隸屬[[太原府|太原府]]。

[[民国|民国]]三十年（1941年）十一月划为交城、交西两县，民国三十三年（1944年）复并为交城县。

1949年9月隶属[[榆次专区|榆次专区]]，10月改属[[汾阳专区|汾阳专区]]；1951年复属榆次专区。

1958年属[[晋中专区|晋中专区]]，同年9月，该县与[[文水|文水]]、[[汾阳|汾阳]]并为汾阳县

1959年9月复为交城县

1971年划属吕梁地区恢复交城县建制至今。

== 地理 ==
西部为[[吕梁山|吕梁山脉]]，东部为[[太原盆地|太原盆地]]。

== 经济 ==
交城縣的經濟以煤炭、焦化、建材等重工業為主，因此污染嚴重。2004年9月，[[山西省环保局|山西省环保局]]將交城縣劃為全山西省第一個“循环经济建设试点县”，試圖減輕污染，發展[[循環經濟|循環經濟]]。<ref>{{Cite web |url= http://www.people.com.cn/GB/paper83/15416/1366088.html|title=交城：步入大循环经济时代|accessdate=2014-11-17 |work= |publisher=人民網|date=2005}}</ref> 2006年9月，[[國家發改委|國家發改委]]宣佈在交城開設省級經濟開發區，即山西交城经济开发区。此開發區包括當地的夏家營和天寧鎮兩鎮，是原來2002年呂梁地委開設的“夏家营生态工业园区”的一部分，而後者本來就是交城縣的主要經濟區，其工業產值占全縣的70%以上。<ref>{{Cite web|url=http://kfq.ce.cn/gdkfq/shx/201405/05/t20140505_1590882.shtml|title=山西交城经济开发区|accessdate=2014-11-17|work=|publisher=中国经济网|date=2014-05-05|deadurl=yes|archiveurl=https://web.archive.org/web/20141129060813/http://kfq.ce.cn/gdkfq/shx/201405/05/t20140505_1590882.shtml|archivedate=2014-11-29}}</ref>
