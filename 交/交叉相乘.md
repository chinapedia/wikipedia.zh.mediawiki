[[数学|数学]]上，尤其是在[[四则运算|四则运算]]和[[初等代数|初等代数]]中，给定一个两边各一个[[分式|分式]]或[[比|比]]的[[等式|等式]]，就可以用'''交叉相乘'''来[[化简|化简]][[等式|等式]]或求出[[变量|变量]]。
给定一个这样的[[等式|等式]]：
:<math>\frac a b = \frac c d</math>
(当{{math|''b''}}和{{math|''d''}}都不等于[[0|零]]时)，可以交叉相乘来得到：<ref>一个对于这种方法的介绍[[视频|视频]] https://www.youtube.com/watch?v=8eqx8EM-Yto</ref>
:<math>ad = bc \qquad</math>或<math> \qquad a = \frac {bc} d.</math>
[[欧几里得几何|欧几里得几何]]中，相同的运算可以通过[[相似三角形|相似三角形]]得到。

==过程==
实践中，''交叉相乘''的方法就是将两边的分子各乘以另一边的分母。<ref>http://zh.wikihow.com/交叉相乘</ref>
:<math>\frac a b \nwarrow \frac c d \quad \frac a b \nearrow \frac c d.</math>
这种方法的[[数学证明|数学证明]]是由下列[[数学|数学]]过程推导而来的。我们从这个简单的[[等式|等式]]开始：
:<math>\frac a b = \frac c d</math> ({{math|''b''}}和{{math|''d''}}都不等于[[0|零]])
我们可以两边同乘以[[相等|相等]]的[[数|数]]而两边仍然相等，所以如果我们在这个[[等式|等式]]两边同乘以{{math|''bd''}}，我们就得到了:
<math>\frac a b \times bd = \frac c d \times bd.</math>
我们可以把[[等式|等式]]左边的两个{{math|''b''}}和右边的两个{{math|''d''}}约去，剩下
:<math>ad = bc</math>
我们在这里也可以两边同除以:<math>\frac {ad} {bd} = \frac {cb} {db}.</math>来得到：
:<math>a = \frac {bc} d</math>
我们也可以两边同乘以{{math|{{sfrac|''d''|''d''}}}}和{{math|{{sfrac|''b''|''b''}}}} (都等于1)，得到：
:<math>\frac a b \times \frac d d = \frac c d \times \frac b b</math>
所以：
:<math>\frac {ad} {bd} = \frac {cb} {db}.</math>
两边同除以{{math|''bd''}}得到：
:<math>ad = cb.</math>
这些步骤中的单独的每一步都基于[[等式|等式]]性质，交叉相乘是一条[[捷径|捷径]]，也是一个易于理解的，可以教给学生的过程。

==使用方法==
这是一个用来[[化简|化简]][[等式|等式]]或求出[[变量|变量]][[数值|数值]]的[[数学|数学]]方法。如果我们遇到一个这样的方程：
:<math>\frac x b = \frac c d</math>
我们可以用交叉相乘解出
:<math>x = \frac {bc} d.</math>
举个例子，如果我们要求出一辆[[汽车|车]]在7[[小时|小时]]内能开多远，我们如果知道它是[[匀速|匀速]]的，且它已在之前的3[[小时|小时]]内开了90[[英里|英里]]，将这个问题转为[[比例|比例]]我们得到：
:<math>\frac x {7\ \mathrm{hours}} = \frac {90\ \mathrm{miles}} {3\ \mathrm{hours}}.</math>
交叉相乘得到：
:<math>x = \frac {90\ \mathrm{miles}\times7\ \mathrm{hours}} {3\ \mathrm{hours}}.</math>
所以
:<math>x = 210\ \mathrm{miles}</math>
注意即使是这样的方程
:<math>a = \frac {x} {d}</math>
将{{math|''b''}}部分视为1，也可以视为下列方程来用''交叉相乘''来解决
:<math>\frac a 1 = \frac x d.</math>
任何含有[[分式|分式]]的[[等式|等式]]也都可以用两边同时乘以分母的[[最小公倍数|最小公倍数]]来[[化简|化简]]
