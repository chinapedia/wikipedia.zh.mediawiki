'''交趾等處承宣布政使司'''（「'''交趾'''」又寫成「'''交阯'''」、「'''交址'''」），簡稱交趾布政司，是[[明朝|明朝]]于[[永乐|永乐]]五年（1407年）攻灭[[越南|越南]][[胡朝|胡朝]]之后在越南设置的一个地方[[行政机构|行政机构]]，推行与[[内地|内地]]相同的管理方式，下设十五[[府|府]]、卅六[[州|州]]、两百余[[县|县]]，统制范围大致为现在[[越南|越南]][[北纬|北纬]][[北纬17度|十七度]]以北[[地区|地区]]。[[黎利|黎利]]建立[[後黎朝|後黎朝]]後，於[[明朝|明朝]][[宣德|宣德]]三年（1428年）廢止。

== 成立背景 ==
{{Main|明入越}}

[[中國|中國]]於[[秦朝|秦]][[漢朝|漢]]至[[五代十國|五代十國]]時期，曾在[[越南|越南]]地區設置[[交趾郡|交趾郡]]等[[官署|官署]]，進行[[越南北屬時期|直接統治]]。其後越南於10世紀中葉起獨立建國。<ref>[[郭振鐸|郭振鐸]]、[[張笑梅|張笑梅]]《[[越南通史|越南通史]]》第一編第一章第一節，[[中國人民大學|中國人民大學]]出版社，8至13頁。</ref>14世紀後期，[[越南|越南]]（又稱安南國，當時國土約相當於現時[[北中部|中、北部地區]]）[[陳朝_(越南)|陳朝]]接受[[明朝|明朝]]冊封，雙方處於[[朝貢體系|宗藩關係]]。[[明成祖|明成祖]]時，陳朝被權臣[[黎季犛|黎季犛]]（即胡季犛）所篡，明廷乃起兵南下征討，將之消滅（時在1407年）。據越南史籍《[[大越史記全書|大越史記全書]]》記載，當時明廷遍求陳氏子孫，欲立為國王，安南官吏耆老說已被胡季犛所「滅盡，無可繼承陳後」，並向明廷提出「安南國本交州，願復古郡縣，與民更新」。<ref>吳士連等《大越史記全書·本紀全書·後陳紀·簡定帝》，[[東京大學|東京大學]]東洋文化硏究所，494頁。</ref>其後，明成祖頒下《[[wikisource:zh:平安南詔|平安南詔]]》，聲稱為了回應安南[[官吏|官吏]][[耆老|耆老]]的請求，便將當地改名「交址」，再次进行[[安南屬明時期|直接统治]]，依照內地的[[明朝行政區劃|行政區劃]]模式，設立[[承宣布政使司|承宣布政使司]]，[[都指揮使司|都指揮使司]]、[[提刑按察使司|提刑按察使司]]等「三司」。<ref>《[[明實錄|明實錄]]·太宗實錄》卷六十八，永樂五年六月癸未條，茲參考李國祥主編《明實錄類纂·涉外史料卷》，武漢出版社，598頁。</ref>

== 轄境建置 ==
'''交趾承宣布政使司'''治交州府，辖交州府、北江府、谅江府、三江府、建平府、新安府、建昌府、奉化府、清化府、镇蛮府、谅山府、新平府、演州府、乂安府、顺化府等十五府、三十六州、一百八十多個縣。又置太原直隶州、宣化直隶州、嘉兴直隶州、归化直隶州、广威直隶州等五個直隸布政司的州，分轄二十九縣，在要地則設十一[[衛所制|衞]]、三[[衛所制|所]]，一市舶司。<ref>郭振鐸、張笑梅《越南通史》第四編第十一章第二節，[[中國人民大學出版社|中國人民大學出版社]]，398頁。</ref>。後太原直隶州、宣化直隶州分别升为[[太原府|太原府]]、[[宣化府|宣化府]]；九年末（1412年）置宁化直隶州；十三年（1415年）置升华府；降演州府为演州直隶州。

== 廢棄 ==
{{Main|安南屬明時期|後陳朝|藍山起義}}

'''交趾承宣布政使司'''自成立後，當地民眾起事不斷，[[明|明]]廷歲歲用兵。在[[明宣宗|明宣宗]][[宣德|宣德]]二年（1427年），明軍總兵官[[柳升|柳升]]在當地遭到重大挫敗，明廷乃允許撤兵。越人[[黎利|黎利]]建立[[後黎朝|後黎朝]]，改國號[[大越|大越]]，於是安南復國。<ref>郭振鐸、張笑梅《越南通史》第五編第十二章第一節，中國人民大學出版社，419─420頁。</ref>到宣德三年（1428年），正式废除交趾承宣布政使司。<ref>張廷玉《明史‧職官志四》，北京中華書局，1840頁。</ref>
