'''失信被执行人'''，俗称'''老赖'''，是指经[[中华人民共和国法院列表|中华人民共和国各级人民法院]]认定的“被执行人具有履行能力而不履行生效法律文书确定的义务”的人员。根据相关规定，所有被人民法院纳入“失信被执行人名单”的人员，在高消费及非生活或者经营必需的有关消费会受到限制<ref name=rmfyb>{{Cite web|url=http://www.nbd.com.cn/articles/2018-12-01/1277957.html|title=人民法院报：“限高”不等于纳入“失信人名单”|publisher=人民法院报|date=2018-12-01|access-date=2018-12-23|author=楚仑 孔颖丽|language=zh-hans|archive-url=https://web.archive.org/web/20181203194325/http://www.nbd.com.cn/articles/2018-12-01/1277957.html|archive-date=2018-12-03|dead-url=no}}</ref>。

==背景==
据[[最高人民法院|最高人民法院]]统计，在2008年至2012年全国各级人民法院执结的被执行人有财产的案件中，70%以上的被执行人存在逃避、规避甚至暴力抗拒执行的行为，只有不到30%的被执行人自动履行了义务。另据报道称，因“老赖”造成的痼疾，严重影响了“社会的和谐稳定”。为此，最高人民法院在2013年8月底发布《关于公布失信被执行人名单信息的若干规定》<ref name=byt>{{Cite web|url=http://www.chinanews.com/fz/2013/08-30/5224203.shtml|title=7成多被执行人逃避抗拒执行 老赖将被纳入黑名单|publisher=半月谈|date=2013-08-30|access-date=2018-12-23|author=杨维汉 徐硙|language=zh-hans|archive-url=https://web.archive.org/web/20140821061109/http://www.chinanews.com/fz/2013/08-30/5224203.shtml|archive-date=2014-08-21|dead-url=no}}</ref><ref name=jrj>{{Cite web|url=http://bank.jrj.com.cn/2017/08/30101823027746.shtml|title=古稀老赖欠8亿巨款：6次进失信名单 到底拿她怎么办?|publisher=金融界网站综合|date=2017-08-30|access-date=2018-12-23|author=|language=zh-hans|archive-url=https://web.archive.org/web/20180215023516/http://bank.jrj.com.cn/2017/08/30101823027746.shtml|archive-date=2018-02-15|dead-url=no}}</ref>。

==惩戒方式==
===惩戒对象===
根据2013年7月1日[[最高人民法院审判委员会|最高人民法院审判委员会]]第1582次会议通过及根据2017年1月16日最高人民法院审判委员会第1707次会议通过的《最高人民法院关于修改〈最高人民法院关于公布失信被执行人名单信息的若干规定〉的决定》修正的《[[:s:最高人民法院关于公布失信被执行人名单信息的若干规定|最高人民法院关于公布失信被执行人名单信息的若干规定]]》规定，各级人民法院应当将其纳入失信被执行人名单，依法对其进行信用惩戒<ref name=gd>{{Cite web|url=http://cq5zy.chinacourt.org/article/detail/2017/03/id/2559732.shtml|title=最高人民法院关于公布失信被执行人名单信息的若干规定|publisher=最高人民法院|date=2017-03-02|access-date=2018-12-23|author=|language=zh-hans|archive-url=https://web.archive.org/web/20180405203728/http://cq5zy.chinacourt.org/article/detail/2017/03/id/2559732.shtml|archive-date=2018-04-05|dead-url=no}}</ref>：
# 有履行能力而拒不履行生效法律文书确定义务的；
# 以伪造证据、暴力、威胁等方法妨碍、抗拒执行的；
# 以虚假诉讼、虚假仲裁或者以隐匿、转移财产等方法规避执行的； 
# 违反财产报告制度的；
# 违反限制消费令的；
# 无正当理由拒不履行执行和解协议的。

根据《规定》，被执行人具有以上规定第2项至第6项规定情形的，纳入失信被执行人名单的期限为2年；被执行人以暴力、威胁方法妨碍、抗拒执行情节严重或具有多项失信行为的，可以延长1至3年。另外，各级人民法院不得在具有下列情形之一的情况下，依据本规定第1条第1项的规定将被执行人纳入失信被执行人名单<ref name=gd/>：
# 提供了充分有效担保的；
# 已被采取查封、扣押、冻结等措施的财产足以清偿生效法律文书确定债务的；
# 被执行人履行顺序在后，对其依法不应强制执行的；
# 其他不属于有履行能力而拒不履行生效法律文书确定义务的情形。

另外，若被执行人为未成年人的，各级人民法院不得将其纳入失信被执行人名单<ref name=gd/>。

===惩戒措施===
根据《最高人民法院关于公布失信被执行人名单信息的若干规定》，失信被执行人将在政府采购、招标投标、行政审批、政府扶持、融资信贷、市场准入、资质认定等方面受到信用惩戒<ref name=gd/>；根据2010年5月17日最高人民法院审判委员会第1487次会议通过及根据2015年7月6日最高人民法院审判委员会第1657次会议通过的《最高人民法院关于修改〈最高人民法院关于限制被执行人高消费的若干规定〉的决定》修正的《[[:s:最高人民法院关于限制被执行人高消费及有关消费的若干规定|最高人民法院关于限制被执行人高消费及有关消费的若干规定]]》规定，被纳入失信被执行人名单的人员（自然人）不得有以下高消费及非生活和工作必需的消费行为<ref>{{Cite web|url=https://www.chinacourt.org/law/detail/2015/07/id/148347.shtml|title=最高人民法院关于限制被执行人高消费及有关消费的若干规定|publisher=最高人民法院|date=2015-07-20|access-date=2018-12-23|author=|language=zh-hans|archive-url=https://web.archive.org/web/20170225222739/http://www.chinacourt.org/law/detail/2015/07/id/148347.shtml|archive-date=2017-02-25|dead-url=no}}</ref>：
# 乘坐交通工具时，选择[[飞机|飞机]]、列车软卧、[[轮船|轮船]]二等以上舱位；
# 在星级以上[[宾馆|宾馆]]、[[酒店|酒店]]、[[夜总会|夜总会]]、[[高尔夫球场|高尔夫球场]]等场所进行高消费；
# 购买[[不动产|不动产]]或者新建、扩建、高档装修房屋；
# 租赁高档[[写字楼|写字楼]]、宾馆、公寓等场所办公；
# 购买非经营必需车辆；
# 旅游、度假；
# 子女就读高收费私立学校；
# 支付高额保费购买保险理财产品；
# 乘坐[[高速动车组列车|G字头动车组]]列车全部座位、其他动车组列车一等以上座位等其他非生活和工作必需的消费行为。（该规定同样适用于往来[[香港西九龙站|香港西九龙站]]的列车车次<ref>{{Cite web|url=http://www.chinanews.com/ga/2018/09-06/8620196.shtml|title=香港特区政府运房局：正研究乘客用内地居住证购高铁票|publisher=中国新闻网|date=2018-09-06|access-date=2018-12-23|author=曾平|language=zh-hans|archive-url=https://web.archive.org/web/20180909072556/http://www.chinanews.com/ga/2018/09-06/8620196.shtml|archive-date=2018-09-09|dead-url=no}}</ref>）

除上述措施外，被执行人一旦被纳入失信被执行人名单，其名下的住房、银行账户、[[养老金|养老金]]、[[移动支付|移动支付]]账户（如[[支付宝|支付宝]]、[[微信支付|微信支付]]等）将被冻结查封，且不得在全国范围内担任任何公司的法定代表人、董事、监事和高级管理人员，也不得为子女报名就读[[私立学校|私立学校]]，其炒股、出境、在金融机构[[贷款|贷款]]或办理[[信用卡|信用卡]]亦遭到限制。同时，被执行人名下车辆不得驶入[[中华人民共和国高速公路|高速公路]]，一旦被执行人名下车辆进出高速路收费站，车辆将被暂扣，由高速执法大队移交法院处理。根据2015年11月1日起实施的《[[中华人民共和国刑法|中华人民共和国刑法]]修正案（九）》，人民法院的判决、裁定有能力执行而拒不执行，将以“拒不执行判决罪”论处。对于情节严重的，处3年以下有期徒刑、拘役或者罚金；情节特别严重的，处3年以上7年以下有期徒刑，并处罚金<ref>{{Cite web|url=http://hbqlfy.chinacourt.org/article/detail/2018/04/id/3254663.shtml|title=纳入被执行人失信名单和限制高消费的后果|publisher= 青海省祁连县人民法院|date=2018-04-02 |access-date=2018-12-23|author=执行庭 |language=zh-hans }}</ref><ref>{{Cite web|url=http://politics.people.com.cn/n/2014/0117/c1001-24145319.html|title=八部门：老赖等失信人禁乘飞机禁办信用卡|publisher=新京报|date=2014-01-17|access-date=2018-12-23|author=邢世伟|language=zh-hans|archive-url=https://web.archive.org/web/20150418085352/http://politics.people.com.cn/n/2014/0117/c1001-24145319.html|archive-date=2015-04-18|dead-url=no}}</ref>。

从2015年7月起，[[蚂蚁金服|蚂蚁金服]]旗下的[[芝麻信用|芝麻信用]]和最高人民法院实现系统连接，实时更新失信被执行人数据。一旦支付宝用户被纳入失信被执行人名单，其芝麻信用分将被扣减，且在芝麻信用合作商户处的消费购物行为也将受到约束<ref>{{Cite web|url=http://www.xinhuanet.com//tech/2015-07/02/c_127974718.htm|title=最高法和芝麻信用专线连接 "老赖"今后网购将受限|publisher= 京华时报|date=2015-07-02 |access-date=2018-12-23|author=马文婷 |language=zh-hans }}</ref>。另外，也有一些地方与通讯运营商合作，为失信被执行人设置专属[[回铃音|彩铃]]，且开通后未经法院同意不得取消。民众如拨打失信被执行人名下的电话号码，将会报出机主被列为失信被执行人的提示<ref>{{Cite web|url=http://news.cctv.com/2018/04/19/ARTIvK8PQxGTKFZtX0UAjdc0180419.shtml|title=“老赖”不堪忍受专属彩铃乖乖还钱：我快名誉扫地|publisher=中安在线|date=2018-04-19|access-date=2018-12-23|author=李后祥|language=zh-hans|archive-url=https://web.archive.org/web/20181223211523/http://news.cctv.com/2018/04/19/ARTIvK8PQxGTKFZtX0UAjdc0180419.shtml|archive-date=2018-12-23|dead-url=no}}</ref>。在北京，被纳入失信被执行人名单的人员，不得参加小客车摇号<ref>{{Cite web|url=http://news.haiwainet.cn/n/2018/0831/c3541083-31387052.html?baike|title=关注个人信用信息!北京限制失信人摇号1.4万人次|publisher=北京晨报|date=2018-08-31 |access-date=2018-12-23|author= |language=zh-hans }}</ref>。

==信息公示==
根据《最高人民法院关于公布失信被执行人名单信息的若干规定》，记载和公布的失信被执行人名单信息应当包括如下内容<ref name=gd/>：
# 作为被执行人的法人或者其他组织的名称、[[统一社会信用代码|统一社会信用代码]]（或组织机构代码）、法定代表人或者负责人姓名；
# 作为被执行人的自然人的姓名、性别、年龄、身份证号码；
# 生效法律文书确定的义务和被执行人的履行情况；
# 被执行人失信行为的具体情形；
# 执行依据的制作单位和文号、执行案号、立案时间、执行法院；
# 人民法院认为应当记载和公布的不涉及国家秘密、商业秘密、个人隐私的其他事项。

2013年10月24日，全国法院失信被执行人名单信息公布与查询平台（现为[[中国执行信息公开网|中国执行信息公开网]]）面向社会开通。民众可输入被执行人姓名或名称来查询该失信被执行人的信息，上述信息向社会对外公布<ref>{{Cite web|url=http://www.chinacourt.org/article/detail/2013/10/id/1112516.shtml|title=最高人民法院网开通公布与查询平台 失信被执行人名单信息随时可查|publisher=中国法院网|date=2013-10-24|access-date=2018-12-23|author=|language=zh-hans|archive-url=https://web.archive.org/web/20150923202931/http://www.chinacourt.org/article/detail/2013/10/id/1112516.shtml|archive-date=2015-09-23|dead-url=no}}</ref>。除此之外，地方法院也可以通过公告栏、报纸、广播、电视、网络、召开新闻发布会等方式公布失信被执行人名单信息<ref name=byt/>。近年来也有法院通过影院放映电影的垫片广告<ref>{{Cite web|url=http://www.xinhuanet.com/local/2018-06/19/c_1123004608.htm?baike|title=“老赖”名单搬上大银幕 网友直呼“干得漂亮”！|publisher=东南网|date=2018-06-19 |access-date=2018-12-23|author= |language=zh-hans }}</ref>、[[抖音短视频|抖音]]<ref>{{Cite web|url=http://world.chinadaily.com.cn/a/201807/05/WS5bebdea1a3101a87ca938bb3.html|title=外媒：中国利用抖音APP抓“老赖” 效果不错|publisher=中国日报网|date=2018-07-05 |access-date=2018-12-23|author= |language=zh-hans }}</ref>等方式公布失信被执行人名单信息。2014年7月，[[最高人民法院执行局|最高人民法院执行局]]与人民网联合推出“失信被执行人排行榜”<ref>{{Cite web|url=http://www.xinhuanet.com/overseas/2015-11/21/c_128452202.htm|title=外媒称中国打击"老赖":禁止坐飞机高铁及出国旅游|publisher=参考消息网|date=2015-11-21 |access-date=2018-12-23|author= |language=zh-hans }}</ref>。

==影响==
据2018年7月10日最高人民法院召开的新闻发布会上的消息，截至2018年7月，中国大陆处于发布中的失信被执行人共789万例，涉及失信被执行人440万个。在惩戒方面，已限制1222万人次购买机票，限制458万人次购买动车、高铁票，限制失信被执行人担任企业法定代表人及高管28万人。全国280万失信被执行人迫于信用惩戒压力自动履行义务<ref>{{Cite web|url=http://news.cnr.cn/native/gd/20180711/t20180711_524296865.shtml|title=最高法：已有280万失信被执行人自动履行义务|publisher=人民日报|date=2018-07-11 |access-date=2018-12-23|author= |language=zh-hans }}</ref>。失信被执行人履行法院生效判决也促使“维护社会的和谐稳定”<ref>{{Cite web|url=http://jszx.court.gov.cn/2550/ExecuteNewsletterImage/117617.jhtml|title=广东河源：剑指“老赖” 执结案件93件|publisher=广东省法院决胜“基本解决执行难”信息网|date=2018-10-08  |access-date=2018-12-23|author= |language=zh-hans }}</ref>。

==轶事==
