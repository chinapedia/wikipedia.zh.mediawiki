《'''失落的羽翼'''》是由EXE-Create發售的PC單機遊戲，是Frane系列的第一作。繁體中文版由[[協和國際多媒體|協和國際多媒體]]代理。

== 故事簡介 ==
== 登場角色 ==
=== 主角 ===
; {{nihongo|聖迪‧西理克|セディ}}
; {{nihongo|芙蘭|フラン| CV：[[米田美咲|米田美咲]]}}
: 由天界墮下到凡間的天使，因失去羽毛而失去感情和記憶。
=== 伙伴 ===
; {{nihongo|路比|ルピー| CV：[[與島ぬぐみ|與島ぬぐみ]]}}
: 天界的妖精。
; {{nihongo|多藍|ドラン| CV：[[宮城誠|宮城誠]]}}
: 芙蘭所使役的龍，一旦離開使役者身邊就會變成玩偶。 
=== 叛軍 ===
; {{nihongo|尤洛.里朗傑尼杜|ユーロ| CV：[[伴宗德|伴宗德]]}}
: 叛軍首領。外貌與30年前一樣，沒老去。希望改變世界，認為神所做出來的世界不合理，故向神宣戰，引起戰亂。
; {{nihongo|艾士庫德|エスクード| CV：[[恒松步|恒松步]]}}
: 尤洛所愛著的女性。原為天界的天使，把其中一隻翅膀給了尤洛，令尤洛不老。  
; {{nihongo|迪魯哈姆|ディルハム| CV：[[足立博明|足立博明]]}}
: 王立騎士團一等騎士。希望世界變得更好，故將希望賭在尤洛身上。
; {{nihongo|阿典|アデン| CV：[[竹本良造|竹本良造]]}}
: 王立騎士團其下成員。
; {{nihongo|奈拉|ナイラ| CV：[[前田愛美|前田愛美]]}}
: 惡魔族少女。與尤洛約定要創做一個理想世界。由於性格天真而不知情地被尤洛利用。使役龍名阿彿隆。
=== 其他 ===
; {{nihongo|巴路博亞‧西理克|バルボア| CV：竹本良造}}
: 聖迪父親。與尤洛曾經是有著共同理想的戰友。
; 歐蕾兒
: 聖迪已故的母親，曾幫助托瓦依村。
; {{nihongo|雷亞路|レアル| CV：恒松步}}
: 托瓦依村的村民。當地村民都被給予恩惠，認為自己是神所選中的人。其後村子因惡魔族奈拉給予試練，而導致村子出現魔物。與其他村民對話時不和，覺得村民都自私自利，後來外出與魔物談判，希望能互相理解。最後被聖迪從魔物巢穴中救出。
; 馬洛奇
: 拉多魯村的村民，在聖迪從橋上掉到河中時救助聖迪，帶到拉多魯村醫療。經常到維魯修廢棄礦坑遊玩，故對當中地形很熟悉。
; 健
: 錫安村的村民。村子都被給予試練，以前村民雖然不幸，但會同心協力，會未來從滿希望。其後因尤洛在附近興建軍營，要求村子供應食物，導致村子苦上加苦。
; 雷夫
: 住在達魯谷（被稱為「神的爪跡」）的木匠。
; 迪拿盧 
: 住在巴賽特村。劍術高超，足已與迪魯哈姆匹敵。妹妹因戰爭死去。
