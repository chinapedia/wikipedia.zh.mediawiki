《'''失控'''》（英語：Ingobernable）是以[[墨西哥|墨西哥]]做為故事背景的一部以[[政治|政治]]為題材的網路電視影集，由{{tsl|en|Kate del Castillo||凱特·戴·卡斯提洛}}主演，第1季全15集於2017年3月24日在網路電視媒體網站 [[Netflix|Netflix]] 全球播出<ref>{{Cite web |url = https://media.netflix.com/en/press-releases/ingobernable-netflixs-second-original-series-from-mexico-premieres-march-24 |title = Ingobernable, Netflix's Second Original Series from Mexico, Premieres March 24 |author = Beverly Hills |date = 2017年2月2日 |publisher = Netflix |language = en |accessdate = 2017年3月25日 |quote = Netflix confirms March 24, 2017 as the debut for the first season of the anticipated Mexican original series, Ingobernable. The fictional piece will debut at 12am (CST) and feature a total of 15 episodes that will tell the story of a Presidential family set in present day Mexico.}}</ref><ref>{{Cite web |url = http://www.economist.com/blogs/prospero/2017/03/mexican-drama |title = Mexican drama
Netflix hopes for another Spanish-language hit with Ingobernable |author = TW |date = 2017年3月24日 |publisher = The Economist |language = en |accessdate = 2017年3月25日 |quote = On March 24th Netflix is launching a new series that it hopes will pull off the same trick. Ingobernable stars Kate del Castillo as a Mexican first lady who is on the run following the mysterious death of her husband, the president.}}</ref>，故事講述[[墨西哥|墨西哥]][[第一夫人|第一夫人]]因為丈夫橫死而遭到通緝，她冒險深入未知的領域，好證明自己的清白<ref>{{Cite web |url = https://twitter.com/khootiatling/status/845291826063990784 |title = 從墨西哥總統夫人突然變成通緝犯的故事 |author = Khóo Tiat-lîng |date = 2017年3月24日 |publisher = Twitter |language = zh |accessdate = 2017年3月25日 |quote = Netflix 中譯為《失控》，墨西哥第一夫人因為丈夫橫死而遭到通緝，她冒險深入未知的領域，好證明自己的清白。}}</ref><ref>{{Cite web |url = https://twitter.com/Ingobernable/status/837286378152972289 |title = Ingobernable 24 de marzo |author = Ingobernable‏ |date = 2017年3月25日 |publisher = Twitter |language = sp |accessdate = 2017年3月25日 |quote = El destino de México está por cambiar.
