'''萝扎·伊萨科芙娜·奥通巴耶娃'''（{{lang-ky|'''Роза Исаковна Отунбаева'''}}；{{bd|1950年|8月23日|catIdx=O}}）又称“奥坦巴耶娃”，為前任[[吉爾吉斯斯坦總統|吉爾吉斯斯坦總統]]。曾是前[[吉爾吉斯斯坦|吉爾吉斯斯坦]]外长，並在[[2010年吉尔吉斯斯坦革命|2010年吉尔吉斯斯坦革命]]以前是反对派领导人。奥图巴耶娃2010年4月8日宣布，她组建的临时政府将在6个月内履行政府职能。她表示，临时政府准备在半年之内修改宪法、选举法和政党法，以符合民主发展的需求。<ref>{{cite web |url = http://news.163.com/10/0409/01/63PSQPQC000146BC.html|title=吉尔吉斯斯坦反对派称临时政府将履职半年}}</ref>

== 生平 ==
1950年8月23日出生在吉爾吉斯坦[[奥什|奥什]]，1972年从[[莫斯科国家大学|莫斯科国家大学]]哲学系毕业。之后在[[吉爾吉斯坦国家大学|吉爾吉斯坦国家大学]]担任了6年高级教授及哲学系系主任职位。

1981年她担任[[吉爾吉斯坦共产党|吉爾吉斯坦共产党]][[比什凱克|伏龙芝市]]列宁区区委第二书记，从此便开始了她的政治生涯。1980年代做为[[苏联|苏联]]的代表被派往[[巴黎|巴黎]]的[[联合国教科文组织|联合国教科文组织]]，随后成为苏联驻[[马来西亚|马来西亚]]的大使。1992年，她成为独立后的吉爾吉斯坦首位驻[[美国|美国]]大使。两年后，她被总统[[阿卡耶夫|阿卡耶夫]]任命为外交部长。1998年至2001年期间担任该国驻[[英国|英国]]大使。

2004年，她与[[阿斯卡爾·阿卡耶夫|阿卡耶夫]]决裂，成立了一个[[吉爾吉斯坦社会民主党|新政党]]。

在2005年的“[[郁金香革命|郁金香革命]]”中，她帮助[[库尔曼别克·巴基耶夫|巴基耶夫]]取得了政权。随后，她被巴基耶夫任命为外长。

2010年4月巴基耶夫被推翻後，她代理總統一職。成為吉國首位女性總統。
