'''萝芙木'''（[[学名|学名]]：{{lang|la|''Rauvolfia verticillata''}}）是[[夹竹桃科|夹竹桃科]][[萝芙木属|萝芙木属]]的植物。

== 形态 ==
常绿小[[灌木|灌木]]，多分枝，具乳质。叶对生或3～5枚轮生，长椭圆形至广披针形。夏季开白花；聚伞花序顶生；花萼分裂，花冠高脚蝶状，花冠筒中部膨大，花冠裂片5枚，向左覆盖；雄蕊5枚，着生于花冠筒中部；心皮离生。卵形或椭圆形[[核果|核果]]离生，成熟时为紫黑色。

== 分布 ==
分布在[[台湾岛|台湾岛]]、[[越南|越南]]以及[[中国大陆|中国大陆]]的[[西南|西南]]、[[华南|华南]]等地。一般生于丘陵地带的林中、林边及溪边较潮湿的灌木丛中。喜[[热带|热带]]、[[亚热带|亚热带]]温暖、潮湿环境。在年均温20℃以上、低温不低于-4℃、雨量1200毫米以上、湿度75～80％地区生长良好。

== 药用 ==
根、茎、叶都可药用，含利血平、萝芙木甲素等多种生物碱，有比较持久的降血压作用，可用于治疗[[高血压|高血压]]病。
