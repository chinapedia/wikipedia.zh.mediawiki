《'''混合世界BIORG TRINITY'''》是[[舞城王太郎|舞城王太郎]]原作，[[大暮維人|大暮維人]]繪畫的[[日本|日本]][[漫畫|漫畫]]作品，[[集英社|集英社]]出版，是完成了《[[飛輪少年|飛輪少年]]》之後大暮維人的新作<ref>{{cite web|author=漫遊者|title=漫國博覽會：瘋狂的世界－《BIORG. TRINITY 混合世界》|url=https://hk.lifestyle.yahoo.com/%E6%BC%AB%E5%9C%8B%E5%8D%9A%E8%A6%BD%E6%9C%83%EF%BC%9A%E7%98%8B%E7%8B%82%E7%9A%84%E4%B8%96%E7%95%8C----%E3%80%8Abiorg--trinity-%E6%B7%B7%E5%90%88%E4%B8%96%E7%95%8C%E3%80%8B-033502166.html|accessdate=2015年2月2日|date=2014年4月30日}}</ref>，以未來大戰之後的世界為時代背景，2013年1月開始在《Ultra Jump》連載，至2018年1月結束。話數單位是「bug.○」（○為數字）。

== 故事簡介 ==
糟糕，我真的會因為太喜歡榎本芙三步喜歡到掛掉。雙手掌心出現洞穴、吸入喜歡的東西並與其融合的疾病──「生化蟲」。
在這個構築於危險平衡上的世界，戀愛與青春會以什麼樣貌呈現？令人意外的兩人所描繪的前衛青春群像劇，開幕啦！

== 登場角色 ==
=== 主要人物 ===
; {{nihongo|藤井|藤井（ふじい）}}
: 喜歡芙三步，雖然和穗坂是情敵，但一直把他當成好朋友。
; {{nihongo|穗坂正路|穂坂 正路（ほさか しょうじ）}}
: 和芙三歩是青梅竹馬，藤井的情敵，真實身分是生造獵人9號，與8號發生爭執後重傷在生死邊緣，被藤井用蟲洞吸入，兩人融合一體，目前在藤井的身體裡住著，有事情會用電話與藤井聯絡，口頭禪是2秒內。
; {{nihongo|榎本芙三步|榎本 芙三歩（えのもと ふみほ）}}
: 被藤井與穗坂喜歡著，看起來是普通的少女，其實代表的是整個世界，有時會用不可思議的力量解決難題，又被叫做座標0，是非常重要的存在，是世界分裂成三部分中的靈魂，不過本人似乎是不知道這些事，
; {{nihongo|井本極子|井本 極子（いもと きわこ）}}
: 為人豪爽活潑，和藤井從初中就認識，非常關心藤井。
