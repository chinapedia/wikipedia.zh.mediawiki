'''渦蟲類'''（{{Lang-la|Turbellaria}}）属[[扁形动物门|扁形动物门]]，生活在池塘、溪流的石块下，或者在海水中也有些生活在土中。体表具[[纤毛|纤毛]]，摆动时积水呈涡状。多数为自由生活种类。繁殖方式使用[[斷裂生殖|斷裂生殖]]，是[[無性生殖|無性生殖]]的一種。

==分類==
原渦蟲綱包括所有扁形動物門之下的非寄生物種。但由於渦蟲類物種已被證實為[[並系群|並系群]]，原來屬於寄生物種的三個綱原來都是渦蟲類物種的直系分支，加上綱內三個目各自獨立成為一個門，較新的文獻已不再使用本分類。

大多數一般人接觸最多的渦蟲類物種都屬於[[三歧腸目|三歧腸目]]。2008年以後的研究確認了：[[無腸目|無腸目]]其實並不屬於扁形動物門；而[[小鎖狀目|小鎖狀目]]（亦作[[鏈渦蟲目|鏈渦蟲目]]）的物種亦較早與其他扁形動物門的物種分化。這兩個目的物種在較新的文獻已不再被認為是渦蟲類物種。然而在[[医学寄生虫学|医学寄生虫学]]，基本上仍然沿用舊有的分類。

== 食物 ==
主要吃小的[[水生动物|水生动物]]、[[动物|动物]][[尸体|尸体]]及[[动物|动物]][[内脏|内脏]]。寄生的吸收[[动物|动物]]或[[植物|植物]]体内的营养。

== 體型 ==
大部分渦蟲的長度小於1cm，但熱帶的陸生種類長度可達60cm。

== 形状 ==
多为黑色，体前端有两个可感光的眼体，两侧对称。吻位于身体的三分之二处，且无肛门。有背、腹之分。
