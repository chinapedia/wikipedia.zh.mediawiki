'''全錄公司'''（'''Xerox'''，{{nyse|XRX}}），又稱為-{zh-hans:'''全录'''（台湾）;zh-hk:'''全錄'''（香港）;zh-tw:'''施樂'''（中國大陸）;}-，是一家[[美国|美国]]文案管理、處理技術公司，產品包括[[打印机|打印機]]、[[复印机|複印機]]、數字印刷設備以及相關的服務和耗材供應。全錄公司的總部位于[[美國|美國]][[康涅狄格州|康涅狄格州]][[費爾菲爾德縣_(康乃狄克州)|費爾菲爾德縣]][[诺沃克_(康涅狄格州)|諾沃克]]（於2007年10月自[[費爾菲爾德縣_(康乃狄克州)|費爾菲爾德縣]][[史丹福_(康乃狄克州)|史丹福]]搬遷）。[[纽约州|纽约州]][[罗切斯特_(纽约州)|羅切斯特]]是全錄公司的誕生地，現在仍是全錄公司最主要的辦公地點。

由於全錄在辦公室光學設備、尤其是影印機方面的成功，在[[美式英語|美式英語]]中甚至有將其品牌名稱「xerox」直接當作「影印」（photocopy）的[[動詞|動詞]]或[[名詞|名詞]]之代用詞的非正式習慣。<ref>{{Cite web |url =http://www.eng.fju.edu.tw/etc/quiz/common%20files/X.htm |title = xerox/copy/replicate「影印」、「抄寫」與「複製」|publisher =[[輔仁大學|輔仁大學]]英國語文學系|accessdate = 2014-03-21}}</ref>此外現在電腦中隨處可見的[[图形用户界面|圖型化介面]]也是由全錄公司和旗下的[[帕羅奧多研究中心|帕羅奧多研究中心]]發明的。

==歷史==

1973年全錄開發出带有[[GUI|GUI]]與滑鼠的[[Xerox_Alto|Xerox Alto]]，但是董事會沒看出商機，不看好並未推出商品化。

1979年[[史蒂夫·乔布斯|喬布斯]]參觀了全錄的[[帕羅奧多研究中心|PARC研究所]]看到全錄原型機[[Xerox_Alto|Xerox Alto]]，喬布斯決定開發圖形使用者介面的新電腦。1983年[[苹果公司|蘋果公司]]推出了[[Apple_Lisa|Apple Lisa]]，首次採用GUI的[[商品化|商品化]]電腦。1985年[[微软|微軟公司]]也應用GUI推出了[[Windows_1.0|Windows 1.0]]。

2009年9月28日全錄以64億美元，收購美國電腦服務公司Affiliated Computer Services（ACS）。

香港公司在1964年成立，原稱蘭克全錄（香港）有限公司，2000年12月成為富士全錄株式會社的附屬機構（由[[富士軟片|富士軟片FUJIFILM]]株式會社（日本）和全錄有限公司（美國）按75:25股權比例組成），辦事處位於太古城中心第四期（即溜冰場隔籬）。

2018年1月31日，[[富士菲林|富士菲林]]宣布與[[施樂|施樂]]達成協議，兩者合營企業[[富士施樂|富士施樂]](Fuji Xerox)將與施樂合併，富士菲林將擁有合併後公司50.1%權益。施樂股東將獲派每股9.8美元特別派息，並擁有其餘49.9%權益。但收购案遭到施乐大股东达尔文·迪森（Darwin Deason）的反对。他要求叫停收购，并向纽约州法院提出了多起诉讼。4月27日，美国纽约州高级法院命令施乐暂停被收购相关程序<ref>[https://cn.nikkei.com/industry/management-strategy/30287-2018-04-28-13-22-06.html 富士胶片对施乐的收购案被美国法院叫停].日经中文网.</ref>。5月，施乐单方面退出了与[[富士菲林|富士菲林]]的合并计划。6月18日，[[富士菲林|富士菲林]]向美国曼哈顿地方法庭提起诉讼，指控施乐公司单方面撕毁并购交易，要求后者赔偿10亿美元<ref>[http://www.ce.cn/cysc/newmain/yc/jsxw/201806/20/t20180620_29472118.shtml 富士胶片起诉施乐索赔10亿美元].中国经济网.</ref>。

== 發明圖形使用者介面 ==
全錄發明了電腦的[[圖形使用者介面|圖形使用者介面]]，卻未將之发扬光大，反而讓蘋果與微軟開啟圖形使用者介面的商機並應用在其新商品。
