'''綠色四步'''（Four Green Steps）是一間[[互聯網|互聯網]][[環保主義|推動環保]]的組織，由亞羅吉和亞羅標於2008年創立，總部位於[[魁北克|魁北克]]的[[蒙特利爾|蒙特利爾]]。<ref name="The Concordian">{{cite web |publisher = The Concordian |author = Nekhim, Valeria |title = "Four Steps" to a "Green" future |url = http://www.theconcordian.com/four-steps-to-a-green-future-1.1077045 |date = 2010-01-19 |deadurl = yes |archiveurl = https://web.archive.org/web/20100302114712/http://www.theconcordian.com/four-steps-to-a-green-future-1.1077045 |archivedate = 2010-03-02 }}</ref>主席亞羅吉畢業於[[协和大学|协和大学]]電影動畫系。<ref name=LinkedIn>{{cite web |publisher = LinkedIn |title = Jaye Yarrow |url = http://ca.linkedin.com/pub/jaye-yarrow/19/9ba/64a |accessdate = 2010-08-10 }}{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>該組織是由以下四個部分組成：學校活動，社區，購物中心以及信息空間。網站提供各種有關環保的資訊和學習教材。<ref name = "The Concordian"/>與此同時，該網站也是您獲取身體健康，心理健康<ref name = "Fit Perez">{{cite web |publisher = Fit Perez |author = Perez Hilton|title= 25 Natural Stress Remedies |url = http://fitperez.com/2011-01-20-25-natural-stress-remedies |date=2011-01-20}}</ref>，瘦身竅門<ref name = "Fit Perez2">{{cite web |publisher = Fit Perez |author = Perez Hilton|title= Fitness Tips To Help Save Time And Money |url =http://fitperez.com/2011-01-20-fitness-tips-to-help-save-time-and-money |date=2011-01-20}}</ref>以及排毒食譜<ref name="Fit Perez3">{{Cite web| url = http://fitperez.com/2011-03-25-detox-guide | title = Detox Guide | publisher = Fit Perez | last = Hilton | first = Perez | date = 2011-03-25 }}</ref>的絕佳場所。

==產品和服務==
===學校活動===

學校活動這一板塊，不僅為全世界的小學及中學提供了一份教學參考，也為各個年齡段的學生提供了參加環境保護活動的機會。該公司提供的活動包含了從利用藝術形式展示環境問題，到學習如何記錄生態日誌。<ref name="Journey">{{cite web |publisher = A Journey to Zero |title = Generation ZE: Teaching us how to live in a green world |url = http://journey-to-zero.posterous.com/generation-ze-teaching-us-how-to-live-in-a-gr |date = 2010-04-15 |accessdate = 2010-08-10 |archiveurl = https://archive.is/20130104210314/http://journey-to-zero.posterous.com/generation-ze-teaching-us-how-to-live-in-a-gr |archivedate = 2013-01-04 |deadurl = yes }}</ref>於2009年10月開始啟動的學校活動，在第一年之中已經吸引了十五個國家的四十所學校參與。<ref name="Teacher Magazine">{{cite web |publisher = Pacific Edge Publishing Ltd |title = Canadian Teacher Magazine |url = http://www.canadianteachermagazine.com/pdf/CTM-Nov10.pdf |date = 2010-11-01 |accessdate = 2010-11-09 |deadurl = yes |archiveurl = https://web.archive.org/web/20110409154809/http://www.canadianteachermagazine.com/pdf/CTM-Nov10.pdf |archivedate = 2011-04-09 }}</ref>此外，學校之間的互動，例如環保科技競賽，也隨著展開。

===社區===

社區專區是由與環境有關的[[博客|博客]]，視頻，活動和食譜組成的。此外，公眾也可以在社區上發表自己的觀點與想法。綠色四步是洛杉磯綠色節的官方合作夥伴。演員Booboo Stewart和妹妹Fivel Stewart在2011年加入了綠色四步。

===購物中心===

購物中心板塊的目標，是為了提供給公眾一個可以購買到比傳統購物中的商品更加[[可持續|可持續]]、[[有機|有機]]、環保的替代商品的平台。購物平台中的產品包含但不局限於服裝，[[有機食物|有機食物]]，家具以及水處理裝置。產品種類繁多，不能一一介紹。產品的價格由兩加幣的肥皂盒到數千加幣的專業淨水系統不等。<ref name = "The Concordian"/>2011年10月，綠色四步在洛杉磯綠色節上推出了他們的購物中心。

===信息空間===

信息空間提供了一系列討論環境問題的文章，其中包含很多主題例如[[科學|科學]]与[[技術|技術]]、[[商業|商業]]以及世界新聞。信息空間的文章會在每天不同時段及時更新。在2010年11月，綠色四步在其信息空間宣布了電子回收運動的開始，該組織希望能夠憑藉這個運動來減少由廢舊電子，家電及工具產品累積而成的垃圾。<ref name="e-waste">{{cite web |publisher = Four Green Steps |author = Kozlan, Melanie |title = What is 'E-Waste' & How Can I Get Rid Of It?! |url = http://www.fourgreensteps.com/infozone/featured/features/what-is-e-waste-a-how-can-i-get-rid-of-it |date = 2010-11-02 |accessdate = 2011-01-11 |deadurl = yes |archiveurl = https://web.archive.org/web/20101130085009/http://www.fourgreensteps.com/infozone/featured/features/what-is-e-waste-a-how-can-i-get-rid-of-it |archivedate = 2010-11-30 }}</ref>

=== 流行度 ===

製造和提供有道德的產品和服務現今正變得越來越流行與必須，尤其是在英國，在2008年有價值120億道德產品被賣出。<ref name="Vancouver">{{cite web |publisher = The Vancouver Sun |author = Morton, Brian |title = New online store puts the ethical into shopping |url = http://www.vancouversun.com/business/Northerners+weigh+against+Site+proposal/2481355/online+store+puts+ethical+into+shopping/2937410/story.html?id=2937410 |date = 2010-04-22 }}{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

截止到2010年8月10日，已經有七十二個國家在綠色四步的學校活動專區中註冊。<ref name="School Program">{{cite web |publisher = Four Green Steps |title = School Program |url = http://www.fourgreensteps.com/school-program/ |accessdate = 2010-08-10 |deadurl = yes |archiveurl = https://web.archive.org/web/20100617224102/http://www.fourgreensteps.com/school-program/ |archivedate = 2010-06-17 }}</ref>而該項目也已經被[[新西蘭|新西蘭]] 教育部認可。<ref name="New Zealand">{{cite web|publisher = New Zealand Ministry of Education|title = Four Green Steps School Program|url = http://www.edgazette.govt.nz/Notices/Notice.aspx?NoticeId=613508|accessdate = 2010-08-10}}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

綠色四步致力於與[[麥吉爾大學|麥吉爾大學]]<ref>{{cite web|publisher = McGill University |author = |title= Green Week - Working in the Environment |url = http://www.mcgill.ca/channels/events/item/?item_id=112877 |accessdate = 2010-08-16}}</ref>，康考迪亞大學<ref>{{cite web|publisher = Career and Placement Services - Concordia University |title= 4 Green Steps - Sustainable Café |url = http://caps.concordia.ca/archives/016208.php |accessdate = 2010-08-16}}</ref><ref>{{cite web|publisher = Concordia Career Management Services |title = JMSB Sustainable Internship Program |url = http://johnmolson.concordia.ca/en/career-services/employers/sustainable-internship-program |accessdate = 2010-10-25}}</ref>的學生合作進行關於環境的倡議與研究。綠色四步也被ECO Canada評為一個真正的環保機構。<ref>{{cite web|publisher = ECO Canada|title= ECO Job Board|url = http://www.eco.ca/viewjobs/viewJob.aspx?jobID=13544&contentM=no| date=2010-06-21 |accessdate = 2010-08-16 }}</ref>在《暮光之城》中出演的Booboo Steward和他的妹妹Fivel一起加入了綠色四步。
