== 主要角色 ==
{{main|橘郡风云}}

== 客串角色（按出場序） ==
* [[:en:Michael_Cassidy_(actor)|Michael Cassidy]] 飾演蕭笙（[[:en:Zach_Stephens|Zach Stephens]]），在[[高申|高申]]於第一季末離開後和[[樂心瑪|樂心瑪]]談戀愛，之後發覺高申和樂心瑪二人仍相愛對方，故退出這三角關係。創作者說「他去了[[马林县_(加利福尼亚州)|马林县]]（Marin County）替[[佐治·魯卡斯|佐治·魯卡斯]]的電影公司工作」。
* [[:en:Nicholas_Gonzalez|Nicholas Gonzalez]] 飾演 DJ，李家的園丁，於第二季初和[[顧瑪莎|瑪莎]]相戀。
* [[:en:Shannon_Lucio|Shannon Lucio]] 飾演靈思（[[:en:Lindsay_Gardner|Lindsay Gardner]]），[[李奇立|李奇立]]私生女及[[高姬絲|高姬絲]]的同父異母妹妹，曾與[[歐華恩|歐華恩]]相戀，後和其母 Renee Wheeler 搬至[[芝加哥|芝加哥]]居住。
* [[:en:Kathleen_York|Kathleen York]] 飾演韋詠蘭[[:en:Renee_Wheeler|Renee Wheeler]]，和李奇立發生婚外情，誕下靈思。
* [[:en:Olivia_Wilde|Olivia Wilde]] 飾演雅莉（[[:en:Alex_Kelly|Alex Kelly]]），在魚餌店（The Bait Shop）工作，分別為高申及顧瑪莎的前女友。
* [[:en:Kim_Delaney|Kim Delaney]] 飾演布慧冰（[[:en:Rebecca_Bloom|Rebecca Bloom]]），[[高誠|高誠]]舊情人，曾介入高氏夫婦的婚姻。
* [[:en:Billy_Campbell|Billy Campbell]] 飾演畢卡特（[[:en:Carter_Buckley|Carter Buckley]]），雜誌《紐波特生活（Newport Living Magazine）》前編輯，轉職前和高姬絲的關係曖昧。
* [[:en:Johnny_Messner|Johnny Messner]] 飾演歷斯（[[:en:Lance_Baldwin|Lance Baldwin]]），[[顧茱莉|顧茱莉]]的初戀情人，一個[[機會主義|機會主義]]者。
* [[:en:Logan_Marshall-Green|Logan Marshall-Green]] 飾演[[歐程|歐程]]（前譯歐格雷）（[[:en:Trey_Atwood|Trey Atwood]]），歐華恩親兄，剛刑滿出獄。第二季末被顧瑪莎槍傷，及後私自離開醫院前往[[拉斯維加斯|拉斯維加斯]]。
* [[:en:Marguerite_Moreau|Marguerite Moreau]] 飾演簡蘊德（[[:en:Reed_Carlson|Reed Carlson]]），壞科學出版社行政人員，負責出版高申及簫笙合著的[[繪本小說|繪本小說]]。
* [[:en:Nikki_Griffin|Nikki Griffin]] 飾演沙靜思（[[:en:Jess_Sather|Jess Sather]]），一名有[[可卡因|可卡因]][[毒癮|毒癮]]的性感女郎，歐程對她亦大有好感。在第三季回來，歐華恩替她處理感情問題。

== 重要情節 ==
* 翠莎向歐華恩謊稱她意外流產。
* 高申在第一季末遠走到波特蘭，暫住胡洛及其父的家，高誠來說項時他拒絕回去。
* 高申和歐華恩最終都返回紐波特灘。
* 兩名新男友出場，分別為樂心瑪男友簫笙（白晢版高申）及顧瑪莎男友 D.J.（後因得悉顧瑪莎利用他來激怒顧茱莉，而與顧瑪莎分手）。
* 歐華恩與同班的高材生靈思來往，後來發現原來靈思是高姬絲同父異母的妹妹。
* 李奇立被控[[賄賂|賄賂]]罪及其他相關罪名，因其前僱員 Renee Wheeler 承認他們之前有姦情而[[無罪釋放|無罪釋放]]。
* 李奇立辭去紐波特集團[[行政總裁|行政總裁]]之位，指派顧茱莉任[[行政總裁|行政總裁]]及高姬絲任[[財政總裁|財政總裁]]。
* 17歲的雅莉和高申一起在魚餌店工作，二人繼而產生感情。
* 李奇立和 Renee Wheeler 的戀情曝光，他們的女兒靈思希望李奇立可以收養她，從而接近及了解她親父。
* 顧占美和顧茱莉舊情後熾，但當前者得悉顧茱莉想和李奇立維持婚姻關係後，便離開紐波特到[[夏威夷|夏威夷]] [[:en:Maui|毛伊島]] 發展事業。
* 顧茱莉計劃出版一本名為《紐波特生活》的生活潮流雜誌，但被認為用來表現自我居多。
* 簫笙和高申（和樂心瑪）根據他們的家人朋友的生活，創作了一本[[漫畫|漫畫]]（及後[[繪本小說|繪本小說]]）。
* 雅莉和顧瑪莎短暫熱戀，之後雅莉靜悄悄地離開紐波特比奇。
* 李奇立一次輕微[[心臟病|心臟病]]發，令他將家庭放回第一位。
* 布慧冰（高誠舊情人）希望高誠幫她洗脫多年前犯的罪名，後因布父（高誠的[[大學|大學]]教授）逝世而選擇離開紐波特比奇。
* 基因測試證實靈思確為李奇立親女，但靈思卻決定跟其母搬往[[芝加哥|芝加哥]]。
* 樂心瑪發覺她仍深愛高申，遂與簫笙分手，而和高申復合。
* 歷斯（顧茱莉的初戀情人）利用八十年代時顧茱莉及他合演的小電影《The Porn Identity》向顧茱莉[[勒索|勒索]]，勒索不成後歷斯在《紐波特生活》發佈會上當眾發佈該錄影帶。
* 畢卡特（《紐波特生活》編輯）與高姬絲之間有著曖昧關係。
* 歐程刑滿出獄後暫住高家，及後搬進雅莉舊居。
* 在歐程的廿一歲生日派對中，沙靜詩服食過量[[毒品|毒品]]，差點在泳池中溺斃。此事驚動了警方，歐程因而被拘捕（後又無罪釋放）。而他和沙靜詩亦開始來往。
* 李奇立和顧茱莉關係變差，李奇立更要脅要[[離婚|離婚]]（部份原因是顧茱莉與前夫舊情復熾），但未及簽紙李奇立便去世。
* 受[[毒品|毒品]]影響下，歐程企圖[[強姦|強姦]]顧瑪莎。
* 高姬絲對[[酒精|酒精]]上癮，引致車禍發生。
* 由原本高申與[[佐治·魯卡斯|佐治·魯卡斯]]見面而簫笙與樂心瑪去高中舞會，變成最後簫笙往見佐治·魯卡斯而高申與樂心瑪一起。
* 李奇立因心臟病發逝世。

季末：
* 顧占美和李凱莉回來出席李奇立的[[葬禮|葬禮]]。
* 高姬絲又再[[酗酒|酗酒]]，高誠決定送她到[[戒酒中心|戒酒中心]]，並聯同家人一起說服高姬絲接受治療。
* 歐程和沙靜詩在魚餌店牽涉[[毒品|毒品]]買賣。
* 歐華思發現歐程企圖[[強姦|強姦]]顧瑪莎。他倆兄弟由質問變成打架，顧瑪莎趕到時歐程正勒著歐華思的頸。顧瑪莎向歐程開槍，歐程生死未卜。
