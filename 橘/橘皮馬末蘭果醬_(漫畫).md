== 劇情簡介 ==
300年前，對[[吸血鬼|吸血鬼]]進行無差別攻擊的'''殺戮政策'''開始施行，後因有[[绝灭|物種滅絕]]的危機，所以人類取消了政策，並於200年前與吸血鬼簽訂'''和平協議'''。現時的吸血鬼以喝豬血取代人血，隱藏自己的[[本能|本能]]並潛伏在人類社會。可是由於他們[[捕食|獵食]]的本能，吸血鬼對於喝人血還是有一定的渴望。

女性吸血鬼'''白瑪麗'''一直努力隱藏自己的身份，[[模仿|模仿]]人類的生活習慣，每天忍受著同學們對吸血鬼的各種[[偏見|偏見]]及[[歧視|歧視]]。直至某一天，擁有香甜血液的男性人類'''鄭在民'''出現，導致白瑪麗失去控制並吸吮了他的脖子。從此鄭在民就對白瑪麗很在意，並逐漸接近她。隨著時間推移，兩人之間漸漸產生微妙的感情。

==特殊用語==
; 優美高中（{{lang|ko|요미고교}}，{{lang|en|Yomi High School}}）
: 瑪麗及在民等人就讀的學校，為故事發生的主要舞台{{sfn|橘皮馬末蘭果醬|loc=第4話}}{{sfn|橘皮馬末蘭果醬|loc=第106話}}。

; 現代吸血鬼（{{lang|ko|현대의 뱀파이어}}，{{lang|en|Modern Day Vampire}}）
: 現代的吸血鬼，外表與人類無異。

; 身體特徵
: 經過長年[[演化|演化]]後，現代吸血鬼的大部分特徵及能力都已經退化，他們不再擁有尖銳的[[犬齒|犬齒]]，對陽光有[[免疫力|免疫力]]，[[肌肉|肌肉]]力量已退化，不再擁有強大的力氣及飛簷走壁的能力，壽命也縮短，已發現90歲就因老化而死亡的吸血鬼，對寒冷非常敏感，但嗅覺仍然靈敏，恢復能力也很強，而且不再害怕[[十字架|十字架]]及[[大蒜|大蒜]]，[[皮膚|皮膚]]比人類白皙。{{sfn|橘皮馬末蘭果醬|loc=第1話}}{{sfn|橘皮馬末蘭果醬|loc=第26話}}

:; 眼睛
:: 進食時及空腹時眼睛會變成紫色。

:; 牙齒
:: 如果不管犬齒會漸漸長出來並變尖，每個月都需要打磨。{{sfn|橘皮馬末蘭果醬|loc=第40話}}

; 飲食習慣和嗜好
: 為了遵守跟人類的和平協議，大部分吸血鬼以喝豬血為生。可是由於他們[[捕食|獵食]]人血的本能，當遇上擁有香甜血液的人類時，吸血鬼還是難以抗拒，甚至會失去理性地捕食。但由於犬齒已退化，一般都不會成功。對吸血鬼來說，人血比豬血更好喝。無法分辨人類食物的味道，即使吃下去也無法消化，所以必須在事後吐出來。吃下大蒜會感到難受。

:; 喝人血
:: 雖然吸血鬼喝豬血也能存活，但由於人血比較好喝，部分吸血鬼仍在偷偷地喝人血。另外的原因是喝人血能喚醒吸血鬼的能力，以韓施厚為例，他在喝了人血一年後，開始對寒冷沒那麼敏感，恢復能力也變得更強，喝了兩年後甚至擁有一拳打斷樹幹的強大力氣。但由於身體變得更接近吸血鬼的本質，皮膚也會輕易被陽光灼傷。{{sfn|橘皮馬末蘭果醬|loc=第104話}}{{sfn|橘皮馬末蘭果醬|loc=第106話}}{{sfn|橘皮馬末蘭果醬|loc=第112話}}

; 心理特徵
: 生來的意識和感情都跟人類相同，能融入人類社會生活。可是大部分人類都對吸血鬼有很多誤解及偏見，導致各種歧視，吸血鬼會因此而覺得委屈及難受。即使如此，有些吸血鬼仍希望能跟人類和平共處，希望找出即使知道自己是吸血鬼，但仍願意留在自己身邊的人類朋友；但有些吸血鬼認為人類只會把吸血鬼當成怪物，根本沒可能跟自己和平共處，於是只把人類當成獵物。

; 生活和文化
: 吸血鬼一般在人類社會中隱藏自己的身分生活，可是一旦被人類發現就有被驅逐的危險，需要不停地搬家，一直過著飄忽不定的生活。300年前，人類對吸血鬼施行殺戮政策，導致吸血鬼數量大幅減少。雖然200年前已簽訂和平協議，只要吸血鬼不再喝人血，人類就不會獵殺吸血鬼，可是現時吸血鬼的數量仍是非常少，有[[绝灭|物種滅絕]]的危機。但是一般吸血鬼對於從事生產活動沒有興趣，性格自由奔放，缺乏團體意識，即使自己的種族有滅絕的危機，仍沒打算延續家族。但也有部分吸血鬼的婚事在年輕時早已被父母安排好，以延續下一代。{{sfn|橘皮馬末蘭果醬|loc=第26話}}
