'''琛航岛'''（[[英语|英语]]：Duncan Island），是[[西沙群岛|西沙群岛]]中的一个岛屿。现由[[中华人民共和国|中华人民共和国]]政府实际控制，行政上划归[[海南省|海南省]][[三沙市|三沙市]]管辖。[[越南|越南]]政府亦声称拥有其主权。

== 地理位置 ==
琛航岛位于[[中国南海|中国南海]][[西沙群岛|西沙群岛]]的[[永乐群岛|永乐群岛]]中[[永乐环礁|永乐环礁]]南部，[[晋卿岛|晋卿岛]]以西约1海里。

== 地理地形 ==
琛航岛面积0.28平方公里。该岛有沙堤环绕，岛上植被茂盛。琛航岛坐落在[[环礁|环礁]]上，与处于同一个珊瑚[[礁盘|礁盘]]上的[[广金岛|广金岛]]（琛航岛以西约370米）有一条长堤相连接，环礁内有[[潟湖|潟湖]]，环礁北部开口形成航道连接外海，使琛航岛成为航行条件很好的港口。

== 历史 ==
1909年[[广东水师|广东水师]]提督[[李准|李准]]巡海时以其随行军舰“琛航号”命名为“琛航岛”。

== 命名 ==
*1935年公布的名称为“坛坚岛”。
*1947年和1983年公布名称为“琛航岛”。

中国渔民俗称“三脚”、“大三脚岛”、“三脚岛”<ref>{{cite book zh |author=广东省地名委员会 |title=《南海诸岛地名资料汇编》 |format=M |edition=1 |location=广州 |publisher=广东省地图出版社 |date=1987 |pages=171 |id= ISBN 80522-023-9}}</ref>。

== 岛上状况 ==
琛航岛是永乐群岛交通中心。琛航岛上修有纪念1974年中国人民解放军海军[[西沙之战|西沙之战]]阵亡烈士的纪念碑。

岛上建有[[中國航天測控站列表|中國航天測控站]]西沙站。
