'''鶯歌溪'''，又名'''兔子坑溪'''<ref name="北桃地區鶯歌溪排水整治及環境營造規劃"/>、'''大湖溪'''<ref name="鶯歌鄉土誌">《鶯歌鄉土誌》，今澤正秋，台北州海山郡鶯歌庄役場，昭和9年</ref>，為[[臺灣北部|臺灣北部]]的一條[[河川|河川]]，屬於[[大漢溪|大漢溪]]支流，集水區範圍包括[[新北市|新北市]][[鶯歌區|鶯歌區]]及[[桃園市|桃園市]][[龜山區|龜山區]]，主流長度約為8.5公里<ref name="北桃地區鶯歌溪排水整治及環境營造規劃"/>，主要支流有鶯歌支線、尖山支線、圳子頭坑支線、福源排水及鶯歌分線<ref name="北桃地區鶯歌溪排水整治及環境營造規劃"/>，集水區域範圍計21.01平方公里<ref name="鶯歌溪排水治理計畫"/>，目前是[[經濟部水利署|經濟部水利署]]第十河川局管理的中央管區排，為龜山、鶯歌地區重要之區域排水<ref name="北桃地區鶯歌溪排水整治及環境營造規劃"/>。

== 水文地理 ==
鶯歌溪水源來自[[桃園市|桃園市]][[龜山區|龜山區]]兔子坑一帶[[丘陵|丘陵]]山地，地質上屬於龜山丘陵（龜崙嶺、山仔角山塊），標高400公尺，地勢自西北向東南遞降<ref name="鶯歌溪排水治理計畫"/>。鶯歌溪下游流經鶯歌市區及[[大湖_(新北市)|大湖]]鳳鳴都市計畫區，地面坡度較緩<ref name="鶯歌溪排水治理計畫"/>。鶯歌溪最後穿過景觀橋[[三鶯龍窯橋|三鶯龍窯橋]]後，於[[三鶯大橋|三鶯大橋]]下游處匯入[[大漢溪|大漢溪]]左岸。

== 名稱 ==
*兔子坑溪：鶯歌溪發源自龜山兔子坑地區，故又名兔子坑溪。
*大湖溪：鶯歌溪流貫鶯歌[[大湖_(新北市)|大湖地區]]的中央，故又名大湖溪<ref name="鶯歌鄉土誌"/>。

== 支流 ==
匯入鶯歌溪的主要支流，由上游至下游依序為：
*福源排水（桃園市龜山區福源國小附近）
*圳子頭坑支線（新北市鶯歌區中湖國小附近）
*鶯歌支線（鶯歌區尖山地區）
*尖山支線（鶯歌區尖山地區）
