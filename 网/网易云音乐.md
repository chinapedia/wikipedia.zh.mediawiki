'''网易云音乐'''（NetEase Cloud Music）是中国[[网易|网易]]公司自2013年4月23日基於[[網易雲|網易雲]]服務推出的一个[[音乐|音乐]]平台，除了提供主要的音乐播放、下载与查找服务外，还提供音乐[[社交|社交]]功能（歌曲评论、[[声学指纹|声学指纹]]（听歌识曲）、根据历史播放记录的歌单推荐、地理位置识别等）。

== 功能 ==
* 更大程度地利用歌单进行音乐的整理而非传统意义上的专辑与表演者。<ref>{{cite web|author1=阿盏|title=网易云音乐|url=http://music.163.com/#/playlist?id=156577738|website=music.163.com|publisher=网易云音乐|accessdate=2017-08-27|quote=让人无限怀念的经典，永远不会消失的旋律，童年的回忆怎么可以少了这些！[[萝卜片|萝卜片]]，老物比较多，燃向为主，并非全收op和ed，多是单主喜欢的歌，如果混有一些奇怪的东西=-=不要在意细节啦}}</ref>
* [[个性化|个性化]]推荐（包括每日推荐、私人FM、心动模式）<ref>{{cite web|author1=娱乐资本论|title=网易系口碑最好产品 云音乐原来是这么突围的|url=http://news.mydrivers.com/1/504/504734.htm|website=快科技|accessdate=2017-09-11|date=2016-10-26}}</ref><ref>{{cite web|title=网易云音乐：3年用户增长2亿，爆棚的口碑才是最好的营销工具|url=http://www.mediaclub.cc/News/infos/id/6695|website=麦乐网|accessdate=2017-09-11|date=2016-12-01}}</ref><ref>{{cite web|title=网易云音乐再遭下架 情怀难过版权大山|url=http://www.techweb.com.cn/viewpoint/2017-09-11/2583687.shtml|website=techweb|accessdate=2017-09-11|date=2017-09-11}}</ref>
* 精彩的用户评论。评论及热评功能是云音乐不可分割的一部分。边听歌边看评论，用户可以在评论区找到听歌的共鸣。<ref>{{Cite web|url=https://music.163.com/#/topic?id=212001 |title = 新人必看！云音乐iPhone特色功能指南！ |author = 云音乐小秘书 |date = 2015-10-13 |accessdate = 2017-12-30}}</ref>在2017年10月份的4.2.1更新后，移动端用户可以在自己的用户主页查看自己发布的评论，包括动态评论、歌曲评论、歌单和专辑评论等等。2017年12月上线近期热评功能。
* 独立音乐人入驻。从2014年开始公开招募独立音乐人。<ref>{{Cite web|url=http://music.163.com/nmusician/web/retrieval|title=网易云音乐|accessdate=2017-10-28|work=music.163.com}}</ref>并在2016年11月22日启动“石头计划”，投资两亿扶持独立音乐人。<ref>{{Cite web|url=http://www.bjnews.com.cn/ent/2016/11/22/424495.html|title = 网易云音乐启动“石头计划” 投资2亿扶持独立音乐人 |author =新京报新媒体 作者：杨畅 |date =  2016-11-22 |accessdate = 2017-12-30}}</ref>
* 主播电台。用户不仅可以收听大牌DJ的优质节目，还可以创建属于自己的节目频道。<ref>{{Cite web|url=http://www.gd.chinanews.com/2014/2014-10-29/2/333948.shtml|title = 网易云音乐“主播电台”上线 人人都可当“主播” |author = 广东新闻网 |date =  2014年10月29日 |accessdate = 2017-12-30}}</ref>
* 短视频功能。在2018年3月移动端的5.0.0更新后，网易云重构了移动端的用户界面，将原本不太引人注意的视频模块放置到了首页。这意味着网易开始重视云音乐的短视频功能。<ref>{{Cite web|url=http://tech.ifeng.com/a/20180321/44914293_0.shtml|title=网易云音乐为什么要主推短视频？|accessdate=2018-05-28|work=tech.ifeng.com}}</ref>
* 音效功能。在5.4.0的内测版中，网易云在其音乐播放界面添加了音效入口和封面动效，有十几种音效可供选择。

现支持[[iOS|iOS]]、[[Android|Android]]、[[Windows|Windows]]、 [[Linux|Linux]]在内的多个操作系统<ref name="新浪网网易云音乐正式发布">{{Cite_news |author1=孟鸿 |date=2013年<!--0-->4月23日<!-- 16:29--> |title=网易云音乐正式发布 丁磊自述：除了摇滚都听 |script-title= |trans-title= |url=http://tech.sina.com.cn/i/2013-04-23/16298270391.shtml |dead-url=no |format=shtml |editor1-last= |editor1-first= |department= |newspaper=tech.sina.com.cn |type= |series= |language=zh-Hans |volume= |issue= |others= |edition= |location=北京 |publisher=[[新浪|新浪]]科技 |publication-date=2013年4月23日 |agency=新浪科技 |page=1 |id= |archive-url= |archive-date= |access-date=2017-03-13 |via= |registration= |subscription= |lay-url= |lay-source= |lay-date= |quote=网易CEO丁磊表示，数字音乐正被移动互联网改变，PC时代的模式即将过时。 |postscript= |ref= }}</ref>。截至2017年11月，网易云音乐用户数量超过4亿，用户创建歌单达4亿个，用户评论数达4亿个<ref>{{Cite web|url=https://m.sohu.com/a/204880748_161623/?pvid=000115_3w_a|title=网易云音乐用户数突破4亿 宣布与KKBOX达成战略合作|accessdate=2017-12-23|author=|date=2017-11-15|publisher=搜狐首页·新闻}}</ref>。

由于许多[[唱片公司|唱片公司]]（如[[King_Records|King Records]]）要求向用户收取数字专辑的费用，网易云音乐上越来越多的音乐开始转变为付费收听和下载的模式，无损格式的音乐则全部收费，还有一部分音乐由于网易暂未获取授权而不能播放；另外，中国大陆以外的用户无法使用其完整服务。

== 相关争议 ==

=== 周杰伦下架事件 ===
4月1日，网易云音乐突然下架周杰伦歌曲一事引起热议。原来，下架之前，周杰伦歌曲版权授权已到期7小时，且网易云音乐在3月31日告知用户可花费400元购买周杰伦200首热门歌曲合辑。就此事，网易云音乐在4月1日，4月2日共发布两个声明。4月2日晚上，网易云音乐副总裁王诗沐通过视频形式，直接面向站内用户进行回应和说明。<ref>{{Cite web|url=http://tech.sina.com.cn/roll/2018-04-06/doc-ifyteqtq4799791.shtml|title=“周杰伦下架事件”复盘：网易云音乐做错了什么？|accessdate=2018-06-13|work=tech.sina.com.cn}}</ref><ref>{{Cite news|url=https://36kr.com/p/5127564.html|title=“周杰伦下架事件”复盘：网易云音乐做错了什么？|accessdate=2018-06-13|language=zh-CN}}</ref>

2017年2月27日，网易云音乐与日本最大娱乐集团[[Avex_trax|爱贝克思]]（avex）联合宣布，双方已达成独家音乐版权战略合作，将共同致力于[[日本流行音樂|日本音乐]]在中国大陆的传播推广和正版化进程。合作达成后，网易云音乐已获得爱贝克思旗下日本音乐全面授权在中国大陆地区的独家版权，成为中国大陆日本音乐版权曲库最丰富的音乐平台。同时，网易云音乐将全面代理这批版权曲库在中国大陆地区的独家推广、版权管理和分销等业务，助力爱贝克思拓展中国大陆市场。<ref>{{Cite web|url=http://ent.163.com/17/0227/14/CE9OR8UN00038793.html|title=网易云音乐获日本最大娱乐集团独家音乐版权|accessdate=|author=网易娱乐|date=2017-02-27|publisher=网易}}</ref><ref>{{cite web|author1=音乐先声|title=网易云音乐得救了，是谁救了它？|url=http://tech.sina.com.cn/csj/2017-09-14/doc-ifykywuc3096303.shtml|website=新浪专栏|accessdate=2017-09-14}}</ref>

2018年3月6日，网易云音乐与[[阿里音乐|阿里音乐]]共同对外宣布，双方达成音乐版权互相转授权的合作。至此，两大音乐平台的网络音乐版权结构得到进一步完善，用户的视听体验将更加丰富。一方面，网易云音乐将[[天娱传媒|天娱传媒]]、[[Avex_trax|爱贝克思]]（avex）、[[丰华唱片|丰华唱片]]、[[華研國際音樂|华研国际]]等优质音乐版权转授给阿里音乐；同时，阿里音乐将[[滚石唱片|滚石唱片]]、[[SM娱乐|SM娱乐]] 、[[贝塔斯曼音乐集团|BMG]]等优质音乐版权转授给网易云音乐。<ref>{{Cite news|url=http://tech.163.com/18/0306/11/DC7A4GKD00097U7R.html|title=网易云音乐和阿里音乐宣布达成版权互授合作|author=网易科技|date=2018-03-06|work=|accessdate=2018-05-23}}</ref>
