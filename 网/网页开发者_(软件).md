'''網頁開發者'''（Web Developer）是給以[[Mozilla_Firefox|Mozilla Firefox]]核心為基礎[[網頁瀏覽器|網頁瀏覽器]]的一個強大的-{zh-hans:扩展; zh-hant:附加元件;}-。它加入了幾個選單與一條工具列以及多種網頁開發者工具。本插件已測試相容於[[Firefox|Firefox]]、[[Flock|Flock]]、以及[[SeaMonkey|SeaMonkey]]。該-{zh-hans:扩展; zh-hant:附加元件;}-原本由派屈克（Chris Pederick）<ref>{{en}}[https://addons.mozilla.org/en-US/firefox/user/45 Chris Pederick]</ref> 開發，他也是User Agent Switch <ref>{{en}}[https://addons.mozilla.org/en-US/firefox/addon/59 User Agent Switcher]</ref> -{zh-hans:扩展; zh-hant:附加元件;}-作者。目前穩定版本是 1.1.4，於2007年6月5日發表。該-{zh-hans:扩展; zh-hant:附加元件;}-是Mozilla 2006年「Extend Firefox」競賽的大獎得主 <ref>{{en}}[http://developer.mozilla.org/contests/extendfirefox/index.php "Extend Firefox" competition results]</ref> 。並且它是少數幾個特別被Firefox開發者所推薦的[[網頁程式設計|網頁程式設計]]-{zh-hans:扩展; zh-hant:附加元件;}-之一 <ref>{{en}}[https://addons.mozilla.org/en-US/firefox/recommended Firefox -{zh-hans:扩展; zh-hant:附加元件;}-: 推薦-{zh-hans:扩展; zh-hant:附加元件;}-]</ref> 。
