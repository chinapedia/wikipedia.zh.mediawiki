'''网易微博'''是一个由[[网易|网易]]推出，提供[[微型博客|微型博客]]服务的类[[Twitter|Twitter]]网站。用户可以通过网页、[[WAP|WAP]]页面、外部程序和手机短信、彩信等发布消息。与一般微博服务不同，网易微博的消息最大长度限制为163个字而不是通常的140个字<ref name="cb">{{cite web | url=http://www.cnbeta.com/articles/102568.htm | title=“网易微博”截图首次亮相 可以一次输入163个字 | accessdate=2011-07-14}}</ref>，与网易的[[域名|域名]]163.com一致。并可上传图片。

== 历史 ==
网易微博于2010年1月20日上线内测。在内测期间，网易发放邀请码使用户逐步增多。

3月20日，网易微博公测版1.0上线。<ref>{{cite web | url=http://tech.163.com/10/0320/14/627OI254000915BF.html | title=网易微博公测上线 更开放更去中心化 | accessdate=2011-07-14}}</ref>

[[UTC+8|UTC+8]]时间7月13日19：00，网易微博暂停服务，没有给出重启服务的时间。与此同时，[[新浪微博|新浪微博]]、[[搜狐微博|搜狐微博]]、[[凤凰微博|凤凰微博]]、[[人民微博|人民微博]]等微博在logo处添加“测试版”字样。此前的7月9日至12日，[[搜狐微博|搜狐微博]]也曾进行维护。<ref>{{cite web| url=http://nf.nfdaily.cn/nfdsb/content/2010-07/14/content_13754608.htm| title=网易微博也开始"维护" 各网站推出"测试版"| accessdate=2011-07-14| deadurl=yes| archiveurl=https://web.archive.org/web/20110810042753/http://nf.nfdaily.cn/nfdsb/content/2010-07/14/content_13754608.htm| archivedate=2011-08-10}}</ref>搜狐和网易均回应称维护原因是升级微博系统，然而有消息称，或许是中国的各个微博服务正在接受监管部门的整顿。<ref>{{cite web | url=http://finance.ifeng.com/money/roll/20100715/2409372.shtml | title=网易微博突然关闭 各大微博齐变“测试版” | accessdate=2011-07-14}}</ref>网易微博于7月15日恢复。

7月31日，网易微博2.0上线，重新设计了首页并更换了logo<ref>{{cite web | url=http://tech.163.com/10/0803/18/6D6C3FCS00094IMI.html | title=网易微博2.0说明3：新首页 | accessdate=2011-07-14}}</ref>。该版本将“回复”重命名为“评论”，使用引用框在一个页面呈现全部对话<ref>{{cite web | url=http://tech.163.com/10/0803/18/6D6BCGET00094IMI.html | title=网易微博2.0版特性1：新评论 | accessdate=2011-07-14}}</ref>。转发功能也进行了改进。改进后的功能类似于[[Twitter|Twitter]]的retweet：合并被多次转发的微博、不再允许对原微博内容进行更改、原作者删除某条微博时，转发的该条微博也会消失。<ref>{{cite web | url=http://tech.163.com/10/0803/18/6D6C16CN00094IMI.html | title=网易微博2.0版说明2：新转发 | accessdate=2011-07-14}}</ref>

12月28日，微博2.5版上线。

2012年3月16日，网易微博将与新浪、搜狐和腾讯微博共同正式实行微博[[中国网络实名制|实名制]]。<ref>[http://internet.solidot.org/article.pl?sid=12/02/07/0856203 四大微博将在3月16日全部实名制]</ref>

2012年7月15日13：35网易微博暂停服务。[[File:2012年7月15日13：35网易微博暂停服务.JPG|thumb]]

2014年11月5日，网易微博将正式关闭，网易微博用户将迁移至其轻博客[[LOFTER|LOFTER]]，以保存用户的原内容。<ref>[http://pcedu.pconline.com.cn/568/5683517.html 微博黄金时代结束？网易微博将正式关闭]</ref>

== 特色 ==
=== i 达人认证 ===
与[[新浪微博|新浪微博]]邀请明星和名人开设微博客，并对他们进行实名认证的策略不同，网易微博的自身定位是“有态度的微博”，这和网易的自身定位——“有态度的网站”相符<ref>{{cite web | url=http://tech.163.com/11/0704/14/784IQNA6000915BF.html | title=报告称“有态度”已成网易微博形象特征 | accessdate=2011-07-14}}</ref>。因此网易推出了“i 达人认证 ”，引导用户创造有价值的内容。<ref>{{cite web | url=http://3c.ycwb.com/2011-04/29/content_3421610.htm | title=网易微博：有态度的用户打造价值的内容 | accessdate=2011-07-14 }}{{Dead link|date=2018年9月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

“i 达人认证”是授予某个领域的行家或潮流的风向标身份标识。认证主要考虑微博账户主的真实身份、专业能力和网络表现力准。目前，i 达人认证仅授予给部分能够确认身份的用户。<ref name="iVerified">{{cite web | url=http://t.163.com/Verified | title=网易微博i 达人认证 | accessdate=2011-07-14 | archiveurl=https://web.archive.org/web/20110821121652/http://t.163.com/Verified | archivedate=2011-08-21 | deadurl=yes }}</ref>

=== 产品整合 ===
网易用户可在[[网易新闻|网易新闻]]评论页点选“转发到微博”按钮将任意评论转发到自己的微博。<ref name="qq">{{cite web | url=http://tech.qq.com/a/20100517/000051.htm | title=新浪搜狐网易三门户网站打通新闻评论和微博 | accessdate=2011-07-14}}</ref>2011年7月6日更开始默认在网易门户新闻跟帖的用户评论左侧显示用户的微博头像以及微博加关注按钮。<ref>{{cite web | url=http://www.cnbeta.com/articles/147962.htm | title=网易跟帖全面整合网易微博 发帖可激活微博账号 | accessdate=2011-07-14}}</ref>

网易将微博整合进入网易邮箱，允许用户在网易邮箱中查看朋友的微博状态。此外，还会推荐用户通讯录中联系人的微博账户。<ref>{{cite web | url=http://tech.163.com/10/0803/18/6D6ANJEH00094IMI.html | title=网易定义的微博：与网易邮箱深度整合 | accessdate=2011-07-14}}</ref>

网易[[有道|有道]]实时搜索中包含了来自网易微博和新浪微博的内容。<ref>{{cite web | url=http://tech.qq.com/a/20100416/000174.htm | title=有道实时搜索测试版上线 整合新浪网易微博 | accessdate=2011-07-14}}</ref>

=== 第三方应用 ===
网易微博开放了[[API|API]]，这让开发者能够发布各种第三方应用。
目前除移动客户端之外，还有一些趣味第三方应用推出。<ref>{{cite web | url=http://open.t.163.com/ | title=开放平台 | accessdate=2011-07-14 | archiveurl=https://web.archive.org/web/20110711143210/http://open.t.163.com/ | archivedate=2011-07-11 | deadurl=yes }}</ref>

== 争议 ==
=== 账户 ===
目前，网易微博用户无法注销自己的账户。其他一些中国的微博客服务也存在同样的问题。<ref name="delete">{{cite news| url=http://www.morningpost.com.cn/bjcb/html/2011-07/12/content_104844.htm| title=不可注销的微博| work=北京晨报| date=July 12, 2011| accessdate=2011-07-14| deadurl=yes| archiveurl=https://web.archive.org/web/20110715220655/http://www.morningpost.com.cn/bjcb/html/2011-07/12/content_104844.htm| archivedate=2011年7月15日}}</ref>

=== 内容审查 ===
与其他中国的微博服务一样，网易微博有内容审查。2010年7月15日，网易微博从维护中恢复服务后，内容搜索功能一度取消，而内容审查人手则翻倍。<ref>{{cite web | url=http://www.cbnweek.com/Details.aspx?idid=1&nid=3817 | title=微博客大战 | accessdate=2011-07-14}}</ref>
