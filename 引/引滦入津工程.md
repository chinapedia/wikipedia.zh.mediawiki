'''引滦入津工程'''是[[中华人民共和国|中国]][[滦河|滦河]]、[[海河|海河]]流域的大型跨流域引水工程。以[[河北省|河北省]]境内的滦河[[潘家口水库|潘家口水库]]为龙头，经[[大黑汀水库|大黑汀水库]]调节后，通过输水干渠经[[迁西|迁西]]、[[遵化|遵化]]进入[[天津市|天津市]][[蓟县|蓟县]][[于桥水库|于桥水库]]，再经[[宝坻区|宝坻区]]至[[宜兴埠泵站|宜兴埠泵站]]，全长234千米<ref name="纪炳纯">{{cite journal |title=引滦入津流域底栖动物研究及水质评价 |author=纪炳纯，秦保平等 |journal=南开大学学报：自然科学版 |year=2002 |volume=35 |issue=2 |pages=106-112 |url=http://www.cqvip.com/qk/93854x/2002002/6492024.html }}</ref>。工程于1982年5月11日动工，于1983年9月11日建成<ref>[http://news.qq.com/a/20081124/002592.htm 引滦入津：改变一座城市命运的水利工程]</ref>。

== 背景 ==
[[File:Tianjin_haihe.jpg|right]]
20世纪70年代末，由于天津市人口剧增，用水量加大，而主水源[[海河|海河]]因上游环境破坏、森林减少、修建[[水库|水库]]、[[灌溉|灌溉]]农田等原因，导致下游水量大幅度减少，造成天津供水严重不足。当时，天津市用水量由原来的每天180万方降到100万方，后又压缩到70万方。天津市民生活用水由原来每人每天70公升降到65公升，而且是每公升含1000多毫克[[氯化物|氯化物]]的咸水。天津工业生产用水由原来日用77万方降到45万方。此外，天津第一发电厂由于缺水被迫停止发电，天津纺织、印染、造纸等工业面临停产威胁。<ref>[http://news.enorth.com.cn/system/2008/10/28/003745306.shtml 引滦入津：远水解近渴 史册载功勋]</ref>

== 建设 ==
1981年8月，[[中华人民共和国|中华人民共和国]][[国务院|国务院]]决定兴建引滦入津工程，将其列为国家重点建设项目，工程的勘测、设计、施工任务由天津市负责<ref name="光明日报">[http://kaoshi.gmw.cn/01gmrb/2004-09/29/content_108752.htm 引滦入津：载入史册的水利工程]{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。该工程是一个跨海河、滦河流域的巨大工程群，包括在[[迁西县|迁西县]]的滦河干流修建[[潘家口水库|潘家口水库]]和[[大黑汀水库|大黑汀水库]]，并通过引滦入津（北线）和[[引滦入唐工程|引滦入唐]]（南线）两个巨大的输水工程，沟通滦河、[[州河|州河]]、[[还乡河|还乡河]]、[[陡河|陡河]]等河系，把已成的[[于桥水库|于桥]]、[[邱庄水库|邱庄]]及[[陡河水库|陡河]]等大型水库串联起来形成一个具有蓄水、壅水、分水、输水、配水、净水等功能的完整的水源体系，以解决天津市、唐山市的城市用水困难，并兼供发电和农业灌溉。<ref name="搜狐">[http://news.sohu.com/20080904/n259383296.shtml 1983年9月5日 引滦入津工程全线通水]</ref>

1982年5月11日，工程正式开工。<ref name="gov.cn">[http://www.gov.cn/jrzg/2008-11/24/content_1158184.htm 引滦入津：改变一座特大型城市命运的水利工程]</ref>

输水工程从河北省境内的滦河大黑汀水库开始，通过输水干渠经[[迁西|迁西]]、[[遵化|遵化]]入[[黎河|黎河]]至天津[[蓟县|蓟县]][[于桥水库|于桥水库]]，经新建州河暗渠，转输水明渠，注入[[尔王庄水库|尔王庄水库]]<ref name="与时俱进的引滦入津工程">{{cite journal |title=与时俱进的引滦入津工程 |author= |journal=海河水利 |year=2004 |volume= |issue=4 |pages= |url=http://www.cqvip.com/qk/96399x/2006004/22620011.html }}</ref>，经泵站分两路进入天津市：一路从明渠入[[北运河|北运河]]和[[海河|海河]]；另一路由暗渠入[[天津水厂|天津水厂]]。年平均输水量10亿立方米，最大输水能力为60～100立方米／秒。

引滦河道自大黑汀水库坝下至天津全长234公里，全线隧洞、泵站、管道桥梁等工程项目共有215个，共治理河道100多公里，开挖64公里的专用水渠，全部工程开凿出的岩石达140万立方米。其间还需要在滦河和蓟运河的分水岭处穿越[[燕山|燕山]]开凿一条12.394千米长的引水隧洞。工程需要完成土方3460万，石方166.7万，浇筑混凝土约80万立方米，修建各类建筑物面积12万多平方米，安装设备2776台件。此外，还有进水闸枢纽、加压泵站、平原水库、大型倒虹吸、暗管、明渠、暗渠、净水厂，公路桥，以及农田水利配套、供电、通讯工程等。<ref name="光明日报"/><ref name="gov.cn"/><ref name="天津通志水利志">{{Cite web |url=http://www.tjdfz.org.cn/tjtz/slz/ |title=天津通志水利志 |access-date=2010-08-29 |archive-url=https://web.archive.org/web/20160315042107/http://www.tjdfz.org.cn/tjtz/slz/ |archive-date=2016-03-15 |dead-url=yes }}</ref>

1983年9月5日上午8时，潘家口水库、大黑汀水库和引滦枢纽闸依次提闸放水，9月11日，引滦入津工程全线通水。<ref name="gov.cn"/>

工程总投资11.34亿元。<ref name="天津通志水利志"/>

== 后绪及影响 ==
引滦入津工程建成后，在天津城市的发祥地——[[三岔河口|三岔河口]]建设了引滦入津工程纪念碑，由[[邓小平|邓小平]]于1986年8月20日题字。<ref name="光明日报"/>

引滦入津工程扭转了天津的缺水局面，保证了几百万城市居民生活用水和工业用水，减少了地下水开采，有效控制了地面沉降，市区年平均沉降量已经由1985年的86毫米减缓到2002年的17毫米。<ref name="光明日报"/>

2002年天津市利用[[亚洲开发银行|亚洲开发银行]]贷款，投资24亿元开始实施引滦水源保护工程，年节水量可达2600万立方米以上。<ref name="光明日报"/><ref name="与时俱进的引滦入津工程"/>

2002年对引滦入津输水工程沿线水域调查发现，调查水域可初步划定为中－富营养型过渡类型，并已呈现明显富营养化加剧的趋势<ref name="纪炳纯"/>。
