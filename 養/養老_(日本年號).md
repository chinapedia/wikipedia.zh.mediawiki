'''養老'''（[[717年|717年]][[十一月十七|十一月十七]]—[[724年|724年]][[二月初四|二月初四]]）是[[日本|日本]][[奈良時代|奈良時代]][[元正天皇|元正天皇]]之[[年號|年號]]。

==改元==
*[[灵龟|灵龟]]三年十一月十七（717年12月24日）：[[改元|改元]]养老。
*养老八年二月初四（724年3月3日）：改元[[神龟_(圣武天皇)|神龟]]。

== 來源 ==

== 大事記 ==
*養老二年（718年）：建立[[能登國|能登國]]和[[安房國|安房國]]。[[藤原不比等|藤原不比等]]修訂[[養老律令|養老律令]]。
*養老四年（720年）：二月發生[[隼人叛亂|隼人叛亂]]。五月《[[日本書紀|日本書紀]]》完成。
*養老五年（721年）：建立[[諏方國|諏方國]]。
*養老八年（723年）：實施{{tsl|ja|三世一身法|三世一身法}}。
*養老八年（724年）：[[元正天皇|元正天皇]]讓位於[[聖武天皇|聖武天皇]]。

== 出生 ==
*養老二年（718年）：[[孝謙天皇|孝謙天皇]]／[[稱德天皇|稱德天皇]]，第46/48代[[天皇|天皇]]。

== 逝世 ==
*養老四年（720年）：藤原不比等。
*養老五年（721年）：[[元明天皇|元明天皇]]，第43代天皇。
