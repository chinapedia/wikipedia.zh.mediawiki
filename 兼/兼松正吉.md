'''兼松正吉'''（{{lang-ja|兼松正吉 かねまつ まさよし}} {{bd|1542年||1627年|10月13日}}），[[安土桃山時代|安土桃山時代]]到[[江戸時代|江戸時代]]初期的武将。兼松清秀之子，[[尾張国|尾張国]]葉栗郡島村（現在的[[愛知県|愛知県]][[一宮市|一宮市]]島村）出生。亦称「金松」，后证实是误记。通称千熊、又四郎、修理亮。<ref>http://www.asahi-net.or.jp/~jt7t-imfk/taiandir/x153.html</ref>
== 生平 ==
出仕[[織田信長|織田信長]]，[[桶狹間之戰|桶狭间之战]]中出阵。最初是下級武士，因为战功升至信长的马廻众。

本能寺之変后，返回尾张出仕[[織田信雄|織田信雄]]。[[小牧、长久手之战|小牧・長久手之战]]时也作为信雄的下属出战。信雄没落以后，作为[[羽柴秀吉|羽柴秀吉]]的[[黃母衣眾|黄母衣众]]。曾臣属于[[豐臣秀次|豊臣秀次]]，秀次死后又臣属于秀吉。秀吉死后从属于[[德川家康|徳川家康]]。慶長5年（1600年）、随家康征伐会津，转进向西进攻岐阜城，参加[[关原之战|关原之战]]。战后成为家康之子[[松平忠吉|松平忠吉]]的与力，知行2600石。忠吉早逝之后，从属于[[德川義直|徳川義直]]，成为[[尾張藩|尾張藩]]士。墓所位于政秀寺。<ref>http://www.gokuh.jp/ghp/busho/oda_022.htm</ref>

== 逸话 ==
天正元年（1573年）8月的[[一乘谷之戰|刀根坂之战]]中，正吉拿着敌人的首級拜见信長，因为赤着脚在山中奔跑的关系脚被血染红了。信长对他的奋战大为感动，把自己日常带着的草鞋赐给了他。传说这双草鞋就成为了兼松家的传家宝。

元龟元年（1570年）正月时正在参加[[姊川之戰|姉川之战]]，正吉采来河原中的芦苇临时做了装饰用以祈求武运。传说这就是门松的起源。

[[石山合戰|石山合战]]中的天满之森战役中和[[毛利秀賴|毛利秀赖]]一同伏击石山本愿寺方的敌将・长末新七郎，因为两人互相谦让取得敌将首级的机会最终失去时机两人都没有拿到首级。

庆长5年（1600年）[[关原之战|关原之战]]的前哨战・米野之战中，在战场上跟从小就交情深厚的津田元纲重逢，而元纲当时是敌方[[織田秀信|织田秀信]]的家臣。双方进行了单挑但彼此都没有互相为敌的打算，所以挑了一个适当的时候互相称赞了对方的勇武就各自撤退了。元纲在战后成为了鸟取藩的家老，保存了家业。
