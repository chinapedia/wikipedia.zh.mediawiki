'''隆頭魚科'''（[[学名|学名]]：{{lang|la|Labridae}}）為'''隆頭魚目'''（{{lang|la|Labriformes}}）的唯一一個[[科|科]]{{r|Deepfin}}，属[[輻鰭魚綱|輻鰭魚綱]][[真骨下纲|真骨下纲]]，传统上也被归类于广义的[[鱸形目|鱸形目]]。 71属，约500种{{r|FishBase}}。广布于全世界热带和温带海域，以珊瑚礁中为最丰富。体长{{convert|5|-|200|cm}}不等。

== 分布 ==
本[[科|科]]魚類廣泛分布於全球熱帶、副熱帶之[[珊瑚礁|珊瑚礁]]海域。{{r|FishBase}}

== 深度 ==
從淺的潮間帶到水深50至60公尺處，甚至有些魚種可棲息於200公尺或更深的海域。{{r|FishBase}}

== 特徵 ==
本[[科|科]]在體形及體長大小的變異大，體型由長橢圓形到延長型及側扁程度大小亦不一。體被圓鱗，頭部完全裸出，唯頰部鰓蓋有時被鱗。口小型或中等，端位，可伸縮，唇上通常有肉。體唯一列或兩列之圓錐齒，前方通常為犬齒，有時亦長於嘴角處，各齒均不癒合。喉部具有上下兩列之咽喉齒。背鰭1枚且長；臀鰭有2至3枚硬棘。側線鱗列連續或中斷。{{r|FishBase}}

== 生態 ==
大部分種類生活於淺海，但亦有些種類可居於200公尺以上的海底。通常以[[珊瑚礁|珊瑚礁]]為底質，有些則生活於沙地上，但當有危險時，則鑽入沙中躲避。屬於日行性魚類，夜間睡於沙地。具有強而有力的咽喉齒，故大部分之種類以具堅硬外殼的[[貝類|貝類]]、[[甲殼類|甲殼類]]、[[海膽|海膽]]等為食，但亦有其他食性，如[[浮游生物|浮游生物]]食者、[[珊瑚蟲|珊瑚蟲]]食者、魚食者及以清潔行為取得之際生性[[無脊椎動物|無脊椎動物]]為食。他們大都體色鮮艷，且圖案複雜。幼魚體色往往和成魚不同，而且許多種類之雌雄之體色亦有所不同。通常性別兩色的情形，伴隨著由雌魚成長後，性別轉變成雄魚。即開始型的雄魚，具較平淡的體色，經性轉變後，成為體色鮮艷的終端型雄魚。終端型雄魚通常有領域性，在其領土內，存有一與之交配的成熟雌魚及數條性別不明顯的幼魚。終端型雄魚通常一夫一妻之生殖行為。而不經性轉變的雌魚，則是成群的產卵。

== 經濟利用 ==
多為不具經濟重要性的食用魚，有些種類體色鮮艷，可當作觀賞魚。
