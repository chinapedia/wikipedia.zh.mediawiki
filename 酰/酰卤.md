'''酰卤''' (或'''酸卤''') 是[[含氧酸|含氧酸]]的[[羟基|羟基]]被[[卤素|卤素]]取代的衍生[[化合物|化合物]]。<ref>{{GoldBookRef | title = acyl groups | file = A00123}}</ref><ref name="IUPAC">{{GoldBookRef|title=acyl halides|file = A00124}}</ref>

若酸为[[羧酸|羧酸]]，则酰卤化合物含有一个'''–COX'''[[官能团|官能团]]：由[[羰基|羰基]]连接一个[[卤素|卤素]]原子组成。这类酰卤的通式为：'''RCOX'''（R可以是烷基，CO代表羰基，X代表卤素如氯原子）。酰卤可分为：[[酰氟|酰氟]]、[[酰氯|酰氯]]、[[酰溴|酰溴]]和[[酰碘|酰碘]]（不稳定）。其中酰氯为最常用之酰卤，也是常用的[[酰化剂|酰化剂]]，而酰碘也曾是工业生产中规模最大的一类酰卤化合物。利用酰卤制备乙酸的方法每年的产量约数十亿公斤。
<ref>Hosea Cheung, Robin S. Tanke, G. Paul Torrence “Acetic Acid” in Ullmann's Encyclopedia of Industrial Chemistry 2002, Wiley-VCH, Weinheim. {{DOI|10.1002/14356007.a01_045}}</ref>

[[磺酸|磺酸]]的羟基可同样被卤素取代而生成相应的磺酰卤。实际应用中，通常是氯原子作为卤素取代形成[[磺酰氯|磺酰氯]]。

== 制备 ==
酰卤的常用实验室制法：
* 相应的[[羧酸|羧酸]]与[[氯化亚砜|氯化亚砜]]或[[五氯化磷|五氯化磷]]制备酰氯。
* 相应的[[羧酸|羧酸]]与[[五溴化磷|五溴化磷]]制备酰溴。
* 相应的[[羧酸|羧酸]]与[[氟化氰|氟化氰]]制备酰氟。
* [[芳香化合物|芳香]]族酰氯可通过[[氯甲酰化|氯甲酰化]]（类似于[[傅－克酰基化反应|傅－克酰基化反应]]，使用[[甲醛|甲醛]]为酰化试剂）制备。

== 反应 ==
酰卤是一种非常活泼的化合物，可作为中间体合成其他有机化合物。例如，酰卤可发生下列[[化学反应|化学反应]]：

* [[水解|水解]]：与[[水|水]]反应生成[[羧酸|羧酸]]。该反应广泛应用于[[乙酸|乙酸]]的工业合成。
:[[File:Hydration_of_Acyl_Halide.PNG|400px]]

* [[醇解|醇解]]：与[[醇|醇]]反应生成[[酯|酯]]。
:[[File:Acyl_Halide_plus_Alcohol.PNG|400px]]

* [[氨解|氨解]]：与[[氨|氨]]或[[胺|胺]]反应生成[[酰胺|酰胺]]。
:[[File:Acyl_Halide_plus_Amine.PNG|400px]]

* [[傅-克反应|傅-克酰基化反应]]：[[路易斯酸|路易斯酸]][[催化|催化]]下与[[芳香族化合物|芳香族化合物]]反应生成芳[[酮|酮]]。（参见[[傅-克酰基化反应|傅-克酰基化反应]]）

[[卤化氢|卤化氢]]会在上述所有反应中生成，如若酰卤为酰氯，则反应会生成[[氯化氢|氯化氢]]。

== 多重官能团 ==
[[Image:Adipoyl_chloride.png|thumb]]

分子内可允许存在超过一个酰卤官能团，如“[[己二酰氯|己二酰氯]]”，它具有两个酰氯[[官能团|官能团]]。己二酰氯是[[己二酸|己二酸]]的两个羟基被氯原子取代的产物。它的重要用途之一是与有机二氨基化合物发生[[聚合反应|聚合反应]]，形成聚酰氨化合物，也称[[尼龙|尼龙]]；还可和一些其他的有机化合物形成[[聚酯|聚酯]]。

[[光气|光气]] (羰基二酰氯 Cl–CO–Cl)是一种剧毒的气体，属[[碳酸|碳酸]](HO–CO–OH)的二氯化衍生物。光气中两个氯自由基都可进行类似于酰氯的反应。光气可作为反应物制备[[聚碳酸酯|聚碳酸酯]]，还可用于其他工业应用中。

== 毒性 ==
挥发的酰卤都具有[[催泪|催泪]]效果，因为酰氯会与眼睛表面的水份产生卤化氢和有机酸，从而刺激眼睛。若吸入酰卤蒸气会导致相似的问题。大多数的酰卤（甚至包括非挥发性酰卤，如[[苯磺酰氯|苯磺酰氯]]）都会对眼睛、皮肤和[[粘膜|粘膜]]产生有[[刺激性|刺激性]]。
