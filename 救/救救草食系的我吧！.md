《'''救救草食系的我吧！'''》（{{lang-ja|れすきゅーME!}}）是日本漫畫家[[卷田佳春|卷田佳春]]所創作的[[青年漫畫|青年漫畫]]作品<ref>[http://www.animenewsnetwork.com/encyclopedia/manga.php?id=15127 Rescue Me! (manga)][[Anime_News_Network|Anime News Network]]{{en}}</ref>，在[[秋田書店|秋田書店]]《Champion RED Ichigo》雜誌連載，臺灣是由[[東立出版社|東立出版社]]代理發售正體中文版。2013年6月20日發售第3集[[單行本|單行本]]限定版附贈[[OVA|OVA]]。<ref>[http://www.animenewsnetwork.com/encyclopedia/anime.php?id=15128 Rescue Me! (OAV)]Anime News Network{{en}}</ref>

== 故事簡介 ==
水谷正行是一位普通的高中生，因為父母在國外工作而習慣獨自一人生活。有一天家裡突然來一位同年級的女學生清水沙也加。沙也加自稱自己是正行的父母僱用的女傭而同居，正行也因此開始新的同居生活。

== 登場人物 ==
聲優部份左邊是廣播劇CD，右邊是OVA。
; {{nihongo|水谷正行|水谷 正行（みずたに まさゆき）|聲優：[[豊永利行|豊永利行]]／[[逢坂良太|逢坂良太]]}}
:本作的男主角。時常對於沙也加的一些脫序行為感到困擾。
; {{nihongo|清水沙也加|清水 さやか（しみず さやか）|聲優：[[稲村優奈|稲村優奈]]／[[野村真悠華|野村真悠華]]}}
:和正行同年級但不同班。如同妻子般照料著正行的生活，對於接近正行的女性都會特別警戒。
; {{nihongo|篁結花|篁結 花（たかむら ゆか）|聲優：[[高森奈津美|高森奈津美]]}}
:正行母親的妹妹也是正行的叔母，但卻是國中學生。對正行當成哥哥而仰慕著並將沙也加視為競爭對手。
; {{nihongo|高木麻衣||高木 麻衣（たかぎ まい）|聲優：長谷川知子／松本ふくみ}}
:和結花同年級的學生。暗戀著結花也因此敵視正行。
; {{nihongo|小園杏奈||小園 杏奈（おぞの あんな）|聲優：[[河原木志穗|河原木志穗]]／[[三宅麻理惠|三宅麻理惠]]}}
:正行的學妹也是美化委員會的成員。
; {{nihongo|松下|松下||聲優：[[興津和幸|興津和幸]]（OVA版）}}
:正行的同學。羨慕正行的同居生活也因此時常容易產生一些誤解和妄想。
