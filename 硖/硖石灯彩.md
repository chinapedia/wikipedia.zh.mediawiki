'''硖石灯彩'''为流行于[[中国|中国]][[浙江省|浙江省]][[嘉兴|嘉兴]][[海宁市|海宁市]][[硖石街道|硖石街道]]（原硖石镇）一带的一种传统[[灯笼|灯彩]]<ref group="注">又称花灯、彩灯，多于传统节日、婚寿吉庆之时悬挂，以烘托喜庆气氛。</ref>，分为座灯、提灯、壁灯、挂灯和礼品灯五类，作为浙江灯彩的杰出代表现已被列为[[中国国家级非物质文化遗产列表|中华人民共和国国家级非物质文化遗产]]。每年元宵节和中秋节当地均会举办灯会展示彩灯。

硖石灯彩始于唐代，宋代已盛行，至清代时造型突破了灯笼的基本样式，向大型化、多样化发展，出现了亭台楼阁、宝塔等建筑模型及龙舟、采莲船等大型灯彩<ref>{{cite web |url=http://www.cnjxol.com/topic/content/2012-09/21/content_2158763.htm|title=《硖石灯彩》|accessdate=2014-09-13|work=|publisher=嘉兴在线}}</ref>。其原材料有宣纸、竹蔑和铅丝等，做法以竹篾为骨架造型，外加糊纸绘图，以灯映画，一般采用国画绘制方法绘以花鸟、人物、山水等，并手工针刺花纹图案，融工艺、书画于一体，主要工艺技法包括针、拗、结、扎、刻、绘、糊和裱八类，其中以针工为主要特色，灯片的针刺密度可达每平方厘米十八至三十二孔，一件成品针孔多达几百万个<ref>{{cite web |url=http://hnnews.zjol.com.cn/hnnews/system/2010/07/28/012430693.shtml|title=《硖石灯彩的工艺特色》|accessdate=2014-09-13|work=|publisher=海宁新闻网}}</ref>（现状也可采用电脑针刺机代替手工针刺，进行半机械化生产<ref>{{cite web |url=http://hnrb.zjol.com.cn/hnrb/html/2014-02/13/content_259012.htm|title=《硖石灯彩如何大放异彩?》|accessdate=2014-09-13|work=张会、王超英|publisher=海宁日报}}</ref>）。主要的[[燈藝師|燈藝師]]有陈伟炎、胡金龙、沈祖生和宋振华等<ref>{{cite web |url=http://www.jiaxing.gov.cn/mljx/jxwh_36090/lswh/fwzwhyc_36096/gjjfyml/201401/t20140113_308275.html|title=《硖石灯彩》|accessdate=2014-09-13|work=|publisher=嘉兴市政府}}</ref>。
