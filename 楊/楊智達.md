'''楊智達'''（{{bd|1991年|9月4日|catIdx=Yang楊}}）是生於[[臺灣|臺灣]][[台南市|台南市]]的[[社會運動|社會運動]]及政治工作者，長期為各項社會議題發聲，參與高雄和台南地方議題。他曾任 2015南台灣[[臺灣反核遊行|廢核大遊行]]執行統籌<ref>{{cite news|publisher = 地球公民基金會|title=廢核重生 2015南台灣廢核大遊行誓師 |url=http://www.cet-taiwan.org/node/2082|date=2015-03-05}}</ref>，亦投入空污防制、文史保存等[[社會運動|社會運動]]<ref>[http://zdayoung.blogspot.tw/2015/05/blog-post.html 「高雄『不只同志』大遊行」- 關於我與同遊]，楊智達 部落格，2015年5月19日</ref>。

[[2016年中華民國立法委員選舉|2016年立委選舉]]，楊智達參選台南市第四選區（東區、南區、安平區）[[立法委員|立法委員]]<ref>{{Cite web |url = http://news.ltn.com.tw/news/local/paper/893304 |title = 〈南部〉《太陽花世代來挑戰》綠黨楊智達 參選南市立委 |author = 王俊忠 |date = 2015年6月28日 |publisher = 自由時報 |language = zh-tw |accessdate = 2015年12月24日 |quote = 第三政治勢力進軍台南！綠黨台南支黨部秘書長、二十三歲青年楊智達，昨宣布參選南市第四選區立委；坦承自己是同志身分的楊智達表示，過去在高雄、台南參與不少社會運動，深感台灣政治被國民黨與民進黨兩大黨綁架，他打出「政治不只有兩種選擇」的競選主軸，希望與公民站在一起，監督兩大黨、挺好的綠黨！ }}</ref>，主打[[環保運動|環保]]、勞工、青年、平權等議題，是台灣選舉史上最年輕的已出櫃[[同志_(LGBT)|同志]]候選人<ref>[http://twunbound.github.io/view/zdayoung 楊智達：相信時間的人]，郝明義、 陳季芳，2015年12月30日</ref><ref>{{cite news|publisher = 民報|title=23歲綠黨「同志」楊智達角逐台南立委 |url=http://www.peoplenews.tw/news/a347f944-6874-46b5-a9f9-916770a3ad4b|date=2015-06-08}}</ref><ref>{{cite news|publisher = 苦勞網|title=第三勢力進軍台南！綠黨23歲青年楊智達參選立委 |url=http://www.coolloud.org.tw/node/82700|date=2015-06-08}}</ref>。選後，回歸[[NGO|NGO]]工作者身分，擔任[[台南新芽|台南新芽]]創會秘書長<ref>{{cite news|publisher = 自由時報|title=要讓台南更好！ 在地青年組新芽協會參與市議政 |url=http://news.ltn.com.tw/news/politics/breakingnews/1851194|date=2016-10-09}}</ref>，在2018年3月擔任第三屆[[台南彩虹遊行|台南彩虹遊行]]總召。<ref>{{Cite web|url=https://vita.tw/%E5%8F%B0%E5%8D%97%E5%BD%A9%E8%99%B9%E9%81%8A%E8%A1%8C%E8%81%AF%E7%9B%9F-%E5%90%91%E4%B8%8A%E9%80%A3%E7%B5%90%E5%90%91%E4%B8%8B%E7%B4%AE%E6%A0%B9-f122914c3b37|title=台南彩虹遊行聯盟 向上連結向下紮根|accessdate=2018-7-13|author=輔大新聞傳播系實驗媒體|date=2018-4-27|publisher=生命力新聞}}</ref>

== 生平經歷 ==
幼時曾在台南麥得心智教育機構培訓，遭受暴力虐待經驗，被關在密閉空間進行吞火、劈木板等訓練。他表示因為這段經驗讓他從小就對弱勢議題特別有感，因而長大後投入社會運動領域幫助弱勢。在學時期曾擔任學生議會議長、系學會要職，畢業後於[[台灣綠黨|台灣綠黨]][[高雄市|高雄市]][[三民區|三民區]]市議員參選人[[梁益誌|梁益誌]]競選辦公室擔任組織部主任，同時投入[[中央公園護樹護地運動|中央公園護樹護地運動]]。輔選工作結束後，陸續擔任南台灣廢核大遊行<ref>[https://www.flyingv.cc/project/5948 《廢核重生》314南台灣廢核大遊行]，flyingV，2015年</ref>執行統籌、高雄同志大遊行交通組長、聲援圖博抗暴遊行籌備等各項社運工作。

2015年6月27日宣布在家鄉[[臺南市第四選舉區|臺南市第四選舉區]]參選立法委員，成為該屆選舉南台灣最年輕候選人，最終雖未當選，但仍拿下九千餘票，為該屆選舉台南第三勢力候選人得票最高一位。

選後與多位在地青年組成台南新芽協會，投入市政監督、議題倡議工作，並身兼台南性平推動聯盟召集人，後續亦投入在選區重劃、舊魚市場保存、學童自學等議題。2017年6月因發文關注韓國選秀節目[[PRODUCE_101_第二季|PRODUCE 101]]，被網友誤認為是立法委員關注韓國娛樂動態引起大量轉載甚至傳到韓國，雖事後發表聲明表示非當選人，但仍備受關注<ref>{{Cite news|url=http://star.ettoday.net/news/944519|title=前立委候選人楊智達也追《101》　評論轉韓網引朝聖 {{!}} ETNEWS星光雲 {{!}} ETNEWS新聞雲|newspaper=ETNEWS星光雲|accessdate=2017-07-26|language=zh-TW}}</ref>。2018年4月宣布加入[[時代力量|時代力量]]。
