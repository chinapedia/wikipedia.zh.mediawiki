'''楊家正'''{{BD|1987年|1月15日}}，職業[[電子競技|電子競技]]選手。前[[星海爭霸：怒火燎原|星海爭霸：怒火燎原]]選手，後為[[星海爭霸II|星海爭霸II]]職業選手，擅長種族為[[蟲族_(星海爭霸)|蟲族]]。以網路代號'''RRB'''及'''Sen'''聞名於[[星海爭霸|星海爭霸]]社群。2016年5月9日宣布將於完成本季台灣區《WCS世界盃聯賽》賽事後退休<ref>{{Facebook post |story = |photo = |video = y | user =HongKongEsports |id = 1010943808995560 |title =【隊伍資訊－選手退休聲明】 |name =HKEsports 香港電子競技有限公司 Hong Kong Esports Limited
 |date = 2016-05-09 |accessdate = 2016-05-17 |language= zh-tw}}</ref><ref>{{Facebook post |story = |photo = |video = |user = YangChiaCheng |id = 1125552174154008 |title =哈摟 大家好 我是Sen 要跟大家宣布的事情是我決定退役了. |name =Sen |date = 2016-05-09 |accessdate = 2016-05-17 |language= zh-tw}}。</ref>

== 生涯經歷 ==

在星海爭霸：怒火燎原生涯時，因曾於[[WCG|WCG]]賽事中擊敗[[韓國|韓國]]職業選手，而有了「最強非韓選手」的美譽；之後轉戰星海爭霸II，由於表現仍然出色，被臺灣星海社群稱為「'''臺灣蟲王'''」。<ref>[http://www.cna.com.tw/topic/newsworld/29-1/201211270005-1.aspx Sen：當電競英雄，講天分的]，中央社，2012年11月27日</ref>

曾隸屬於[[台灣電競聯盟|台灣電競聯盟(TeSL)]]的[[橘子熊|橘子熊]]隊，擔任戰隊主將，在《[[暴雪嘉年華|暴雪嘉年華]] 2011》獲得季軍的好表現，也讓聯盟決定要將他納入2011台灣電競準名人堂，於[[電競五年|電競五年賽季]]創下61勝9敗的佳績，之後得到年度MVP、最佳蟲族選手、最高人氣等獎項。

「2014台灣電子競技公開賽」星海爭霸II冠軍戰，「台灣蟲王」楊家正（Sen）以4：3擊敗WCS世界盃聯賽積分暫居第一的韓國選手高錫賢（HyuN），首次在台灣主場拿下國際賽冠軍。<ref>[http://news.ltn.com.tw/news/focus/paper/803553 台灣蟲王「驅韓」 《星海II》奪冠]，[[自由時報電子報|自由時報電子報]]，2014年8月11日</ref>
