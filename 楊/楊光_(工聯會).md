'''楊光'''{{大紫荊}}（{{lang-en|Yeung Kwong}}，{{bd|1926年||2015年|5月16日|catIdx=Y楊}}<ref>{{cite web |url=http://rthk.hk/rthk/news/expressnews/20150516/news_20150516_55_1100064.htm |title=工聯會前會長楊光病逝 享年89歲 |publisher=[[香港電台|香港電台]] |date=2015年5月16日 |deadurl=yes |archiveurl=https://web.archive.org/web/20150518103427/http://rthk.hk/rthk/news/expressnews/20150516/news_20150516_55_1100064.htm |archivedate=2015年5月18日 }}{{zh-hant}}</ref>）是[[香港|香港]]左派工會[[香港工會聯合會|香港工會聯合會]]（工聯會）的前[[政黨領袖|領導人]]，先後任[[理事長|理事長]]和[[會長|會長]]。女兒在英國讀書，女婿是英國白人．在[[香港回歸|香港回歸]]後於2001年獲[[香港特別行政區政府|香港特別行政區政府]]頒授[[大紫荊勳章|大紫荊勳章]]，表揚其為勞工界的貢獻。社會大眾一般指其在1967年[[六七暴動|六七暴動]]期間擔任左派勢力的領導角色，組織及策劃一連串包括暗殺和放置炸彈的暴力事件<ref>[http://www.chinayouth.org.hk/20070506/catalogue.html 神州青年服務社「六七暴動40年」圖片展]神州青年服務社</ref>。

[[香港日治時期|香港日治時期]]，楊光工作於當年香港最大的[[香港海軍船塢工會|香港海軍船塢工會]]。1962年，楊光由工聯會副理事長正式擔任理事長，及後在1980年接任會長位置，直至1988年方由[[李澤添|李澤添]]繼任。1975年至1988年期間，楊光自[[張振南|張振南]]以後，成為第二名擔任[[港區全國人大代表|港區全國人大代表]]的工聯會理事長，此後更三次連任。他亦是[[香港特別行政區第一屆政府推選委員會|香港特別行政區第一屆政府推選委員會]]之委員，以勞工界身份成為400名推選委員之一<ref>[http://www.takungpao.com/news/09/07/31/zuguo01-1119983.htm 楊光──發展騰飛大功臣] {{webarchive|url=https://web.archive.org/web/20091028020629/http://www.takungpao.com/news/09/07/31/zuguo01-1119983.htm |date=2009-10-28 }}，《大公報》，2009-7-31</ref>。楊光亦曾任第7屆[[中國人民政治協商會議全國委員會香港地區委員|全國政協委員]]<ref>[http://cn.chinagate.cn/indepths/2009-02/18/content_17299113_2.htm 第七届全国政协委员名单] 中國發展門戶網 2009年02月18日</ref>。

== 加入電車工會 ==
楊光於1948年加入[[香港電車有限公司|電車公司]]工作，1954年起擔任左派工會「香港電車職工會」副主席，反對外籍經理莊士頓開除工人，而楊光本人亦於1954年7月1日，與左派電車工會主席陳耀材等31人遭電車公司解僱。1960年代曾參與「港九車床打磨職工會」。

== 領導左派工會暴動 ==
{{seealso|六七暴動}}
1967年5月16日，[[香港親共人士|香港親共人士]]宣佈成立[[港九各界同胞反對港英迫害鬥爭委員會|港九各界同胞反對港英迫害鬥爭委員會]]（簡稱「各界鬥委會」）由工聯會理事長楊光出任主任。鬥委會隨即以「反英抗暴」為口號，聯合各親共團體數百人，手持《[[毛主席語錄|毛主席語錄]]》前往[[港督府|港督府]]（今禮賓府）示威遊行，並在港督府門外張貼[[大字報|大字報]]。港督府一度成為集體抗議示威的主要目標。港府重申要維持法律秩序，限制到港督府請願人數，抗議行動於是轉到[[中環|中環]][[花園道|花園道]]和[[皇后像廣場|皇后像廣場]]一帶。港九各地陸續有集會和示威，參加者除工人外，亦有學生及其他群眾，期間拱北行政府新聞處舊址（今長江集團大樓）播放歐西流行曲去鎮壓示威者。

== 獲特區政府頒授大紫荊勳章爭議 ==
2001年，[[董建華|董建華]]領導的[[香港特別行政區政府|香港特別行政區政府]]當局向[[六七暴動|六七暴動]]領袖楊光，頒發最高榮譽的[[大紫荊勳章|大紫荊勳章]]，亦引起過爭議。時任特首董建華在事件後曾表示，讚揚楊光對勞工界等貢獻，故頒授勳章給他作為肯定，但就迴避了楊光在六七暴動中的角色這個極受爭議的部分。<ref>[http://www.singtao.com/archive/fullstory.asp?andor=or&year1=2001&month1=7&day1=10&year2=2001&month2=7&day2=10&category=all&id=20010710a09&keyword1=%20&keyword2=%20 董讚楊光有貢獻避談暴動][[星島日報|星島日報]] 2001年7月</ref>於[[沙頭角槍戰|沙頭角槍戰]]中殉職的警目馮燕平兒子馮金堂，就楊光授勳一事，怒斥特區政府冷酷<ref>父親生忌日楊光授勳 痛怒攻心 殉職警員子斥特區政府冷酷 《明報》 2001年7月7日</ref>。暴動期間[[石寶德|石寶德]]勳爵訪港時，因拆彈遭炸彈炸傷的警務督察[[周湛樵|周湛樵]]指，楊光在暴動期間可能對他的所屬機構有貢獻，但對香港政府作出何種貢獻就不知道了。

曾經入獄的已故左派人士[[翟暖暉|翟暖暉]]指出，政府以「工運」作為頒獎予楊光的理據，[[劉千石|劉千石]]及[[李卓人|李卓人]]亦有從事工運，如果劉、李不能得到勳章，那麼楊光獲獎便是明顯存在政治因素<ref>[http://www.alliance.org.hk/info/news/wp/?p=632]{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

楊光逝世之後，曾任第四至十屆[[中華人民共和國|中華人民共和國]][[香港特別行政區全國人民代表大會代表|全國人大會議香港特區代表]]、香港[[培僑中學|培僑中學]]校長的[[吳康民|吳康民]]，2015年在香港[[明報|明報]]撰文表示，六七暴動時，楊光的角色「...『一切聽指揮』，自己並無決策權。事無大小，都得聽新華社的。而且事實上鬥委成立不久，楊光就已成港英當局監視以至追捕對象。他率眾到[[香港總督府|總督府]]示威以後，就只能躲進[[中國銀行大廈_(香港)|中國銀行（舊址）大廈]]頂樓，作為『寓公』達數月之久。」至於當時的暴力行動，包括「放真假炸彈，以至炸死[[北角|北角]]無辜小童，暗殺在電台上叫囂反共的廣播員林彬」，吳康民指出：「都不是楊光作出的決定。也許執行者是某些工聯會的屬員，但指揮行動的都另有其人。楊光當時只是一面空頭旗幟，甚至可說是個傀儡，並未參與實際工作。」對於楊光個人的評價，吳康民認為他「...是一個平和但缺乏主見的人，因為他聽話，所以才能擔任這個『反英抗暴」的『旗手』。」此外，吳康民認為楊光為人「...一生淡泊自持，絕對不是一個『工人貴族』，是真正的[[無產階級|無產階級]]......由於他平實、聽話，為工人階級所接受。但又因他『文化不高，缺乏辯才』，所以也不算是一位出色的工會領袖。」<ref>[http://news.mingpao.com/pns/%E5%90%B3%E5%BA%B7%E6%B0%91%EF%B9%95%E7%82%BA%E6%A5%8A%E5%85%89%E8%BE%A8%E8%AA%A3%20%E9%82%84%E3%80%8C%E5%85%AD%E4%B8%83%E3%80%8D%E7%9C%9F%E7%9B%B8/web_tc/article/20150519/s00012/1431972430734 吳康民﹕為楊光辨誣 還「六七」真相 - 20150519 - 觀點 - 觀點 - 明報新聞網]</ref>

== 林彬被殺案 ==
1967年[[六七暴動|六七暴動]]期間，[[商業電台|商業電台]]的節目主持[[林彬|林彬]]在節目中批評發動該次暴動的左派人士，<ref>[http://hk.apple.nextmedia.com/template/apple_sub/art_main.php?iss_id=20080701&sec_id=38164&subsec_id=11236448&art_id=11294678  十八樓C座 拗足40年]《蘋果日報》，2008年7月1日</ref>林在《欲罷不能》節目內對左派的行為及目的更是諸多嘲諷，指他們擾亂香港秩序、發動[[罷工|罷工]]、[[罷課|罷課]]及[[罷市|罷市]]失敗，更多次譴責身為“[[鬥委會|鬥委會]]”核心人物的楊光。

1967年8月24日早上8時許，林彬偕堂弟駕着藍色福士私家車於[[何文田|何文田]][[窩打老道山|窩打老道山]]文福道嘉鳴閣住所附近，被偽裝成修路工人的兇徒攔截。兇徒向車上潑以電油縱火，林彬滾出車後再被淋電油，臉部燒焦，頭髮燒光。他在救護車一度甦醒，並向妻子大喊：「左仔害死我咯！」結果林彬與其同車的堂弟林光海被燒死。事發當天「地下鋤奸突擊隊」司令部發表公告，指事件是「向林逆作出民族紀律的處分，嚴厲懲罰，以示儆戒」。《[[大公報|大公報]]》在他遇害翌日以「地下突擊隊鋤奸　敗類林彬受重傷」為題，譴責他為「對於愛國同胞極盡污辱誣蔑之能事」、對殖民地政府「認賊作父」。該報又稱呼林彬為「林逆」，細數其「罪狀」，「此人一向在反華宣傳中充當爛頭蟀角色……他天天叫囂，用盡無恥無良低能辣撻下流賤格的語言，把祖國的革命群眾稱為暴民。」林彬被殺後，左派稱林彬為「民族敗類，[[港英|港英]]走狗」；謀殺是「執行民族紀律」，並聲稱仍會繼續「制裁其他敗類」。事件中無人被捕，至今仍為懸案。雖然並無證據說明楊光指使謀殺林彬，但社會一般認為左派組織如「鬥委會」及其負責人楊光需要為事件負責<ref>[http://www.ngensis.com/mad-ref/mad-2.htm 林彬遇害]《文化大革命志 補卷一: 赤禍香港》</ref>。

== 去世 ==
2015年5月16日凌晨2時40分，楊光在[[沙田|沙田]][[威爾斯親王醫院|威爾斯親王醫院]]病逝，享年89歲，工聯會對楊光離世感到哀痛。署理行政長官、政務司司長[[林鄭月娥|林鄭月娥]]代表[[香港特別行政區政府|香港特別行政區政府]]指楊光「多年來積極參與香港工運，對勞工福利有突出建樹。他長期為廣大基層服務，貢獻良多」<ref>{{cite web|title=林鄭月娥哀悼楊光逝世|url=http://www.news.gov.hk/tc/categories/school_work/html/2015/05/20150516_182233.shtml|website=香港政府新聞網|date=2015-05-16}}</ref>

工聯會的悼詞中以「光叔」稱呼楊光，讚揚他「想工人群眾所想，急工人群眾所急」，除領導多場工運及關心工人福利外，在1960年代初香港水荒時，更聯合[[香港中華總商會|香港中華總商會]]向粵政府求助，爭取由[[東江|東江]]水輸港，指「經[[周恩來|周恩來]]總理特批和內地政府、民眾的大力支持下」，工程於1965年峻工，並且「解決香港生存危機」<ref>{{cite news|title=【楊光出殯】工聯會讚引東江水解決香港生存危機|url=http://hk.apple.nextmedia.com/realtime/news/20150614/53852475|agency=蘋果日報|date=2015-06-14}}</ref>。

同年6月14日出殯中，示威者帶同標語和「[[菠蘿|菠蘿]]」到[[紅磡|紅磡]][[世界殯儀館|世界殯儀館]]外示威，「菠蘿」放在寫有「同胞勿近」的紙皮箱上，諷刺工聯會當年以相同手法在街頭隨處放置[[手榴彈|手榴彈]]，連累不少無辜市民被炸死。示威者批評楊光當年放炸彈，是[[六七暴動|六七暴動]]時發動連串襲擊是殺人犯，他的死不值得政府官員高度讚揚，也不是對香港有極大貢獻<ref>{{cite news|title=【楊光出殯】本土派「菠蘿」諷指示放炸彈|url=http://hk.apple.nextmedia.com/realtime/news/20150614/53851840|agency=蘋果日報|date=2015-06-14}}</ref>。
