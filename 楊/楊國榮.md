'''楊國榮'''博士（{{lang-en|Dr.  Anthony Kwok-Wing YEUNG}}，{{bd|1967年|1月30日|2010年|7月29日||catIdx=Yeung楊}}），是香港從事哲學和通識教育的學者，[[香港中文大學|香港中文大學]]文學士、哲學碩士，加拿大[[英屬哥倫比亞大學|英屬哥倫比亞大學]]哲學博士。曾任教於[[香港中文大學|香港中文大學]]哲學系及[[香港理工大學|香港理工大學]]通識教育中心。

==簡歷==
楊國榮，1967年1月30日生於[[香港|香港]]，父親楊玉槐，母親余麗華。1986年入讀[[香港中文大學|香港中文大學]][[崇基學院|崇基學院]]，主修哲學，副修中國語言及文學，1990年以一級榮譽畢業，同年入讀香港中文大學研究院，師從[[陳特|陳特]]，研究倫理學，1992年獲哲學碩士學位<ref>{{cite book |title=A Comparison of the moral theories of Kant and Sartre |author=Yeung, Kwok-Wing Anthony |date=1992-07-22 |publisher=The Chinese University of Hong Kong }}</ref>。之後赴笈加拿大卑斯省的[[英屬哥倫比亞大學|英屬哥倫比亞大學]]，1997年獲哲學博士學位。

1997年學成回港後，楊氏於香港中文大學任人文學科研究所副研究員和哲學系兼任講師，1999年轉任[[香港理工大學|香港理工大學]]通識教育中心講師，任教課程包括「生與死」、「中國智慧之道」、「社會倫理」和「世界問題與人類責任」等。楊氏並且是哲學論壇「愚公沙龍」之發起人。<ref>{{ cite web |url=http://yksalon.debate.com.hk/ |title=「愚公沙龍」成立於二oo二年 |publisher=愚公沙龍 }}</ref>

楊氏著有《青紅皂白：從社會倫理到倫理社會》<ref>{{ cite web |url=http://www.cp1897.com.hk/product_info.php?BookId=9789620427657 |title=《青紅皂白：從社會倫理到倫理社會（修訂本）》 }}</ref> 和《顯魅與和樂：對生命意義的逆流探索》<ref>{{ cite web |url=http://www.jointpublishing.com/publishing/catalogue/macau-and-hong-kong/hong-kong-now-and-then/people/9789620430244.aspx |title=顯魅與和樂——對生命意義的逆流探索 |publisher=三聯書店 }}</ref>等書，編作有《陳特文集》等。他曾領導編寫[[香港教育統籌局|香港教育統籌局]]中四至中五「綜合人文學科」中「個人發展課程單元」教材套。

2008年7月29日，楊氏確診患上末期腎癌，2010年7月29日於香港浸信會醫院辭世，享年43歲。<ref>{{ cite web |url=http://www.jointpublishing.com/publishing/catalogue/macau-and-hong-kong/hong-kong-now-and-then/people/9789620430244.aspx |title=作者簡介|date=2010 |publisher=三聯書店 }}</ref>

== 個人生活 ==
楊國榮與妻子何杏楓是香港中文大學的同學，初時兩人同屬崇基學院的哲學系，後來何杏楓轉至中國語言及文學系，同於1990年本科畢業。1992年碩士畢業後，兩人結伴同赴加拿大[[英屬哥倫比亞大學|英屬哥倫比亞大學]]進修，同獲博士學位。何杏楓博士現為香港中文大學中國語言及文學系副教授，兩人育有一女靜得。<ref>{{ cite book |url= |title=楊國榮：《顯魅與和樂——對生命意義的逆流探索》|date=2010 |publisher=三聯書店 |author=楊鍾基 |location=香港 |pages=9-12 |chapter=〈序一〉 |ISBN=9789620438424}}</ref>

何杏楓：「我們總是稱他『楊過』，也說不清是因為『國榮』的『國』跟『過』近，還是因為他那少年輕狂的身影一直沉澱在記憶裏。」<ref>{{ cite book |url= |title=楊國榮：《顯魅與和樂——對生命意義的逆流探索》|date=2010 |publisher=三聯書店 |author=何杏楓 |location=香港 |pages=461 |chapter=〈跋：新生〉 |ISBN=9789620438424}}</ref>

楊國榮在〈我離開的時候：給杏兒〉寫給妻子的話：「記著，死後是有生命的。我會在死後一個很不錯的世界裏，一天到晚看著你。你要好好的活著，我會看得很有趣味的。」<ref>{{ cite book |url=http://www.cuhk.edu.hk/ccc/alumni/newsletter/newsletter201009.pdf |title=《崇基校友》2010年9月號 |date=2010 |publisher=香港中文大學崇基學院 |location=香港 |page=16}}</ref>

==同名人士==
因姓名相同，同有博士學位，並同於香港理工大學工作，或同從事哲學教育，常有人將本文的楊國榮博士與以下的人士混淆：

* 曾任香港理工大學前副校長(學術發展)的楊國榮博士，他1978年獲Queen's University of Belfast頒發之博士學位；他活躍於紡織及製衣界，曾任香港紡織及服裝學會會長。

* 華東師範大學哲學系的楊國榮博士，他於1957年10月生於上海，1988年獲哲學博士學位，任教於華東師範大學哲學系。
