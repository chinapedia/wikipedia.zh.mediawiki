'''楊肇嘉'''（{{bd|1892年|10月13日|1976年|4月19日|catIdx=Y楊}}），[[中華民國|中華民國]]政治人物、社會運動家，[[臺中市|臺中]][[清水區_(臺灣)|清水]]人。楊肇嘉為知名[[門閥|門閥]][[士紳|士紳]]，投身台灣民族運動，鼓吹地方[[自治|自治]]，多次被[[日本政府|日本政府]]「關切」。其為人急公好義，於[[台灣日治時期|日治時期]]助成無數台灣青年，如獎勵青年畫家（如幫助[[李石樵|李石樵]]參加「[[帝展|帝展]]」）、資助飛行家、鼓勵文學青年翻譯《[[紅樓夢|紅樓夢]]》等。[[中華民國政府|中華民國政府]]播遷來台後，擔任[[臺灣省政府|臺灣省政府]]民政廳長，1962年起擔任[[總統府國策顧問|總統府國策顧問]]。

== 早年經歷與努力 ==
出生於台中牛罵頭牛埔子（[[清水區_(臺灣)|清水區]]秀水）[[佃農|佃農]][[楊送|楊送]]之家，本名番兒，兄弟共十三人，排行第七。1897年，[[過繼|過繼]]給台中縣清水鎮首富、前清誥授[[奉政大夫|奉政大夫]][[楊澄若|楊澄若]]為養子，改名楊肇嘉，排行第三，養母陳氏春玉無子嗣。因養父家境寬裕，1901年，楊肇嘉得以進入[[牛罵頭公學校|牛罵頭公學校]]就讀。1908年（明治41年）赴日本[[東京市|東京市]][[黑田高等小學校|黑田高等小學校]]就讀，隔年再轉入[[東京京華商業學校|東京京華商業學校]]。1917年（大正6年）楊澄若任牛罵頭區長，一切實務均由楊肇嘉代理。1920年（大正9年）地方制度改制，楊肇嘉任[[清水街|清水街]]長，向當局爭取海線鐵路經清水、沙鹿之建築計劃。

== 投身台灣民族運動 ==
楊氏投身台灣民族運動，主要是受[[蔡惠如|蔡惠如]]影響。1925年（大正14年）楊氏不顧[[台灣總督府|總督府]]之壓迫，代表[[台灣議會設置請願運動|台灣議會設置請願運動]]赴東京請願。

1926年，入[[早稻田大學|早稻田大學]]攻讀政治經濟，同時任東京新民會常務理事，並結交日本政界開明人士，厚植台灣民族運動之根基。1930年，楊氏於[[東京|東京]]反對台灣總督府重發「阿片吸食牌照」，由[[新民會_(臺灣)|新民會]]刊印《台灣阿片問題》發佈各界，迫使總督府收回成命。1930年[[林獻堂|林獻堂]]等人致書楊氏回台主持[[台灣地方自治聯盟|台灣地方自治聯盟]]。1933年楊肇嘉偕同[[葉清耀|葉清耀]]、[[葉榮鐘|葉榮鐘]]赴朝鮮考察[[地方自治|地方自治]]制度。自1931年春[[台灣民眾黨|台灣民眾黨]]被禁後，台灣地方自治聯盟即擔負台灣民族運動之責。[[九一八事變|九一八事變]]後，以[[荻洲立兵|荻洲立兵]]為首之軍部，對台人壓迫日深，至1934年台灣議會請願運動亦被迫停止。1935年總督府實施[[第一屆市會及街莊協議會員選舉|第一屆市會及街莊協議會員選舉]]，地方自治聯盟獲勝，不過仍未達成實質的地方自治。同年8月，林獻堂在地方自治聯盟會員大會中，提議改組為政黨，未獲通過。1936年林獻堂引發「[[祖國事件|祖國事件]]」。1937年4月，台灣報刊全面廢止中文版，在此情勢下，楊氏遷居東京，與林獻堂、[[吳三連|吳三連]]共謀他計。1939年總督府提出[[米穀管理案|米穀管理案]]，楊氏與吳三連等人策劃抵制。

1940年1月18日，[[蔡培火|蔡培火]]、吳三連以「反軍思想」之嫌遭逮捕，楊氏為躲避風頭而前往中國。1941年，擬取道[[朝鮮|朝鮮]]，轉[[北京|北京]]，往[[上海|上海]]，但於[[新義州|新義州]]時，被疑為[[重慶|重慶]]間諜而遭到逮捕，後由[[吳金川|吳金川]]營救而出。

== 戰後經歷 ==
戰後，楊氏自上海回台，於民國38年底（1949年）出任省府委員，民國39年初兼任民政廳廳長。民國39年（1950年）辦理第一屆縣市議員選舉，其後於民國42年（1953年）辭去職位。民國45年（1956年）擔任[[林獻堂|林獻堂]]公祭副主委，並與林獻堂夫人及兩子護送其遺體回台中霧峰安葬。<ref>[http://catalog.digitalarchives.tw/item/00/31/9a/24.html 台灣耆宿林獻堂先生身後哀榮]，典藏台灣</ref>民國51年（1962年）獲聘為總統府國策顧問，之後便漸漸退出政壇，並著手撰寫回憶錄（著有《楊肇嘉回憶錄》）。退休後，居於清水六然居（1937年落成），楊氏取《[[菜根譚|菜根譚]]》古訓：「自處毅然、處人藹然、有事嶄然、無事澄然、得意冷然、失意泰然」為居所命名，並自號六然居士。楊肇嘉於民國65年（1976年）病逝，享壽85歲。
