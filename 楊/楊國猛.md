'''楊國猛'''，[[TOM集團|TOM集團]][[執行長|執行長]]兼執行董事。

==工作==
楊國猛先生自2008年3月起獲委任爲[[TOM集團|TOM集團]]首席執行官兼執行董事CEO<ref name="任命CEO">{{cite web|url=http://b2b.netsun.com/detail--2599462.html|title=TOM集团宣布委任管理层 首席执行官易人|accessdate=2009-05-20}}</ref><ref name="CEO App">{{cite web|url=http://www.nba.com/techsummit/bio_yeung.htm|title=Ken Yeung|accessdate=2009-05-20}}</ref><ref name="Executive Profile">{{cite web| url=http://savvisg3uk.in.reuters.com/money/quotes/officerProfile?symbol=2383.HK&officerId=| title=Executive Profile| accessdate=2009-05-20}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref name="McKinsey">{{cite web |url=https://alumni.mckinsey.com/alumni/default/public/content/jsp/alumni_news/20080326_KenYeungLBN.jsp |title=Ken Yeung New CEO of TOM Group Limited |accessdate=2009-05-20 |deadurl=yes |archiveurl=https://web.archive.org/web/20110714064030/https://alumni.mckinsey.com/alumni/default/public/content/jsp/alumni_news/20080326_KenYeungLBN.jsp |archivedate=2011-07-14 }}</ref>加入TOM集團前，楊先生爲[[維港投資集團|維港投資集團]]有限公司董事，維港是一家私人投資控股公司，于全球各地主要投資科技及其它業務。楊先生亦曾爲中華關懷投資控股有限公司之首席執行官。

楊國猛曾于[[悉尼|悉尼]]、[[墨爾本|墨爾本]]及[[大中華|大中華]]的[[Mckinsey_&_Company|Mckinsey & Company]]工作超過6年，主要爲電訊、電子、傳媒及電子商務業界，制訂業務發展策略及提供企業改造和營運管理等方面的顧問服務。此後，楊先生在[[通用電氣|通用電氣]]亞太區和[[可口可樂|可口可樂]]中國有限公司擔任多個管理職位。

加入TOM集團後，楊國猛堅持其科技為主導的管理風格，重點拓展公司的[[互聯网|互聯网]]業務<ref name="Facebook">{{cite web |url=http://tech.sina.com.cn/i/2008-07-23/19582346610.shtml|title=TOM集团CEO杨国猛：与Facebook有合作可能|accessdate = 2009-05-20 }}</ref>。在多個媒體訪問中，他認為中國有打造跨媒體市場的潛力。而作為TOM集團的新任舵手，他也重申了TOM集團的跨媒體策略，並以「結合產品，聚合夥伴」<ref>{{cite web |url=http://www.cnadtop.com/brand/renwu/2008/9/24/567245b9-fba8-45d8-bb83-61f2a6b92322.htm | title = TOM集团倾力打造大中华跨媒航母：对内整合 对外聚合 | accessdate = 2009-05-20 }}</ref><ref name="Integration">{{cite web |url=http://www.ycwb.com/myjjb/2008-05/13/content_1885769_2.htm |title=如果“打通”就有赢的机会 |accessdate=2009-05-20 |archive-url=https://web.archive.org/web/20110222024822/http://www.ycwb.com/myjjb/2008-05/13/content_1885769_2.htm |archive-date=2011-02-22 |dead-url=yes }}</ref>的營運模式實施對內整合，打通資源帶包括產品、用户以及業務在内的全面整合。對外，楊國猛強調聚合國內外之內容、技術以及電訊營運夥伴，藉各自優勢共同研發和引進創新的無線互聯网產品，致力為用戶提供線上、線下以至無線的產品和服務。

楊國猛於任內促成多個合作項目，顯示其以科技為主導策略上的決心。TOM在2008年第一季度與[[Stardoll|Stardoll]]合作<ref name="Stardoll">{{cite web |url=http://www.worldsinmotion.biz/2008/08/tom_online_launches_stardoll_i.php | title =TOM Online Launches Stardoll In China|accessdate =2009-05-20}}</ref><ref name="Stardoll2">{{cite web |url=http://en.ce.cn/Business/Enterprise/200808/22/t20080822_16583284.shtml| title = Stardoll and Tom embrace | accessdate = 2009-05-20}}</ref>，為用戶提供明星裝扮、時尚娛樂和在線交友服務。其後於第三季，TOM集團又與美國職業籃球協會（[[NBA|NBA]]）中國公司共同宣布結成長期戰略聯盟，成為NBA官網大中華區的獨家官方合作夥伴，攜手建造中國內地、香港及台灣的NBA中文官方網站<ref name="NBA HK">{{cite web |url=http://news.xinhuanet.com/internet/2009-04/28/content_11274265.htm|title=TOM结合无线与互联网为广告商增值|accessdate =2009-05-20}}</ref><ref name="NBA HK-Marketing">{{cite web |url=http://www.marketing-interactive.com/news/12447 | title = NBA fever spreads to Hong Kong's savvy youth |accessdate = 2009-05-20}}</ref><ref name="NBA CN">{{cite web |url=http://enterprise.dbw.cn/system/2009/04/27/051882895.shtml|title =TOM携手NBA 合力打造全方位体育盛宴|accessdate =2009-05-20}}</ref><ref name="NBA HK-SINA">{{cite web |url=http://news.sina.com.cn/c/2009-04-29/122515547142s.shtml | title =杨国猛担任TOM集团CEO 打造全方位体育盛宴|accessdate =2009-05-20}}</ref><ref name="NBA CH Onscreenasia">{{cite web |url=http://www.onscreenasia.com/article-3922-nbachinaandtomtounveilnbacomwebsitesingreaterchina-onscreenasia.html |title=NBA China and TOM to unveil NBA.COM web sites in Greater China |accessdate=2009-05-20 |deadurl=yes |archiveurl=https://web.archive.org/web/20100128095334/http://www.onscreenasia.com/article-3922-nbachinaandtomtounveilnbacomwebsitesingreaterchina-onscreenasia.html |archivedate=2010-01-28 }}</ref><ref name="NBA HK-3">{{cite web |url=http://www.china.com.cn/news/txt/2009-04/28/content_17690727.htm | title = TOM打造线上、线下、以至手机的全方位体育盛宴 | accessdate = 2009-05-20}}</ref>。NBA的中國內地官網已於2008年10月上線。於2009年初，TOM集團與[[中國郵政集團|中國郵政集團]]公司共同簽訂合作協議，宣布結成戰略合作夥伴，共同發展國內電子商務。

作為上市公司的CEO，楊國猛的舉手頭足都為公眾關注。TOM集團公布2008年年報後，他的薪酬也惹來外界關心。當眾人認為上市公司CEO定必收取豐厚俸祿，楊與其管理團隊則自願放棄[[年終花紅|年終花紅]]或只作象徵式收取低微的花紅，以顯示其與公司共同進退的決心<ref>{{cite web| url = http://active.zgjrw.com/News/2009424/zgjrw/520164783100.html| title = TOM集团CEO杨国猛决心与集团共进退| accessdate = 2009-05-20}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

2010年8月10日，中国邮政集团公司与TOM集团有限公司联合推出的合资[[B2C|B2C]]网站“邮乐网”在北京正式启动上线。TOM集团会与[[中国邮政|中国邮政]]全力合作，将“邮乐网”发展成为买家和卖家双方欢迎的网购平台。<ref name="Ule">{{cite web|url =http://www.ce.cn/cysc/tech/07ityj/guonei/201008/11/t20100811_20457672.shtml|title =中国邮政进军电子商贸 TOM集团全力技术支持|accessdate =2010-08-11|archive-url =https://web.archive.org/web/20100904145421/http://www.ce.cn/cysc/tech/07ityj/guonei/201008/11/t20100811_20457672.shtml|archive-date =2010-09-04|dead-url =yes}}</ref>

==個人及家庭==
楊國猛於[[香港|香港]]出生及長大，操流利[[英語|英語]]、[[普通話|普通話]]及[[粵語|粵語]]，已婚，育有两子，杨竣凯和杨竣谦。

==學歷==
楊在學時期也以專修與電腦和科技有關科目為主。楊畢業於美國[[麻省理工學院|麻省理工學院]]，擁有[[電機工程|電機工程]][[學士|學士]][[學位|學位]]和電機及電腦工程[[碩士|碩士]]學位。楊亦曾獲[[香港政府|香港政府]]頒授爲期兩年的奬學金攻讀美國[[聯合世界書院|聯合世界書院]]，在學期間不負眾望，憑個人努力於國際文憑試中考獲全科甲等成績畢業。
