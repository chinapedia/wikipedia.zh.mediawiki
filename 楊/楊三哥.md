'''楊三哥'''（{{vie|v='''Dương Tam Kha'''|hn=楊三哥}}，生卒年不詳），又稱為'''楊平王'''（{{vie|v='''Dương Bình vương'''|hn=楊平王}}），[[越南|越南]][[吳朝|吳朝]]時期君主，本為[[交趾|交趾]]統治者[[楊廷藝|楊廷藝]]的兒子。吳朝君主[[吳權|吳權]]去世後，憑著篡奪行為獲取權力，於945年至950年在位。其後，遭吳權之子[[吳昌文|吳昌文]]廢黜。

== 生平 ==
=== 奪取權力 ===
楊三哥是10世紀時期[[交趾|交趾]]統治者[[楊廷藝|楊廷藝]]的兒子，他的其中一名姊妹，是[[吳朝|吳朝]]君主[[吳權|吳權]]的王后。據《[[越史略|越史略]]》及《[[大越史記全書|大越史記全書]]》等史料所載，當吳權病重時，委託楊三哥輔助其子。及至吳權去世，945年，楊三哥篡位，僭稱「'''平王'''」，吳權長子[[吳昌岌|吳昌岌]]懼怕大禍臨頭，逃到南冊江（今屬[[越南|越南]][[海陽省|海陽省]]），投靠茶鄉（[[海陽省#行政區劃|金城縣]]）的范令公家中隱居。楊三哥隨即找來吳權第二子[[吳昌文|吳昌文]]，收為己子。

楊三哥對吳昌岌窮追不捨，派指揮使[[楊吉利|楊吉利]]、[[杜景碩|杜景碩]]到范令公處，「索取昌岌，凡三反命」，搜捕數回仍未果。范令公最後唯有「匿昌岌山洞間」，使楊三哥找不著。<ref name="越史略571">《越史略》卷上（收錄於《欽定四庫全書·史部》第466冊），上海古籍出版社書影本，571頁。</ref><ref name="吳士連172-173">吳士連等《大越史記全書·外紀全書·吳紀》，東京大學東洋文化硏究所校合本，172-173頁。</ref><ref name="陳仲金57">陳仲金《越南史略》第三卷第一章，北京商務印書館中譯本，57頁。</ref>

=== 被吳昌文推翻 ===
950年，楊三哥派[[吳昌文|吳昌文]]、[[楊吉利|楊吉利]]、[[杜景碩|杜景碩]]三人征討太平、唐阮二村的動亂。途中，吳昌文勸告楊吉利、杜景碩二人說：「[[吳權|我先王]]之德，治于民心，凡所施令，罔不悅從。不幸違棄群臣，'''平王'''自行不義，奪我兄弟之位，罪莫大焉。今又使我等征無辜之邑，幸而勝之則已，彼如不服，為之奈何？」二人認為有理，吳昌文便即時決定「還師揜襲平王」。結果吳昌文計成，執獲三哥，「衆欲殺之」，吳昌文卻認為「平王於我有恩，豈忍加刑？」最後，楊三哥被降為「'''張楊公'''」，以張楊一地為食邑。<ref name="越史略571"/><ref>吳士連等《大越史記全書·外紀全書·吳紀》，東京大學東洋文化硏究所校合本，173頁。</ref><ref>陳仲金《越南史略》第三卷第一章，北京商務印書館中譯本，58頁。</ref>

== 關於楊三哥的名稱 ==
[[中國|中國]]古籍《[[宋史|宋史]]·外國列傳·交阯》有這樣的記載：「後有[[楊廷藝|楊廷藝]]、紹洪皆受廣南署，繼為[[交阯|交阯]][[節度使|節度使]]。紹洪卒，州將[[吳昌岌|吳昌岌]]遂居其位。」<ref>脫脫等《宋史·外國列傳·交阯》，北京中華書局校注版，14057頁。</ref>當中的「紹洪」（{{vie|v=Dương Thiệu Hồng|hn=楊紹洪}}）大概是指楊三哥。此外，《[[越史略|越史略]]》則稱楊三哥「諱'''主將'''」（{{vie|v=Dương Chủ Tướng|hn=楊主將}}），亦即把「主將」視作他的名。<ref name="越史略571"/>

== 家庭 ==
* 父：[[楊廷藝|楊廷藝]]，曾於931年至937年成為[[交趾|交趾]]的統治者。
* 姊妹：[[吳權|吳權]]王后楊氏，《[[大越史記全書|大越史記全書]]》說楊三哥是「楊后之兄（原注：一作弟）」。<ref>吳士連等《大越史記全書·外紀全書·吳紀》，東京大學東洋文化硏究所校合本，172頁。</ref>

== 評價 ==
* [[越南|越南]][[封建時代|封建時代]]史家[[黎嵩|黎嵩]]批評「楊三哥以外戚之親，受托孤之寄，然廢主自立，難逃篡弒之誅。」<ref>黎嵩《越鑑通考總論》（《大越史記全書》卷首），東京大學東洋文化硏究所校合本，86頁。</ref>
