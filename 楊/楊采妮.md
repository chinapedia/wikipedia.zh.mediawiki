'''楊采妮'''（{{lang|en|'''Charlie Young'''}}，{{bd|1974年|5月23日|catIdx=Y楊}}），出生於[[台灣|台灣]][[台北縣|台北縣]]，[[香港|香港]]及[[台灣|台灣]][[女演員|女演員]]、[[女歌手|女歌手]]。

== 演藝發展 ==
楊采妮出生於[[台灣|台灣]]新北市，父親是上海人，母親是台灣人。三歲時隨父母移居香港。中學時就讀於[[德望學校|德望學校]]，在15歲時前往美國唸書，三年後回到香港發展演藝事業。

1992年，楊采妮初次踏足香港演藝界，被發掘與[[郭富城|郭富城]]合作拍攝香港[[謝瑞麟|謝瑞麟]]珠寶首飾店的電視廣告<ref>{{cite web|title=香港中古廣告: TSL謝瑞麟(郭富城,楊采妮)1992|url=https://www.youtube.com/watch?v=fgYPliXbEuI}}</ref>，推出後楊采妮開始廣受觀眾注目。楊采妮的名字一瞬間在香港年青群眾間火熱起來，成為當時在港坊間旋風式流行的《[[Yes!Card|Yes!Card]]》肖像收藏熱門人物(Yes!Card是由90年代香港年青人潮流刊物《Yes!雜誌》推出的收藏商品)，被香港傳媒冠以「閃卡皇后」封號(閃卡是指《Yes!Card》的最難抽中的珍貴"閃版")。其後，楊采妮開始在不少平面廣告、報刊訪問及電視電台活動亮相，又參與歌手[[郭富城|郭富城]]、[[張學友|張學友]]、[[李克勤|李克勤]]等的音樂錄影帶演出，並開始電影拍攝發展。

1993年，電影方面，楊采妮有機會與當時眾位超級巨星諸如[[張國榮|張國榮]]、[[林青霞|林青霞]]、[[梁家輝|梁家輝]]、[[張曼玉|張曼玉]]、[[梁朝偉|梁朝偉]]、[[劉嘉玲|劉嘉玲]]、[[張學友|張學友]]合作拍攝[[王家衛|王家衛]][[導演|導演]]的經典電影《[[東邪西毒|東邪西毒]]》，成為其重要作品之一。同年，楊采妮簽約[[EMI|EMI]]唱片公司發展歌唱事業成為歌手，並於1993年底推出了首支粵語歌《不會哭於你面前》及首張粵語唱片《[[愛的感覺|愛的感覺]]》，歌曲推出後反應甚好，而此唱片專輯短時間內已獲金唱片成績，更使她勇奪1993年度香港[[無線電視|無線電視]]頒發的[[十大勁歌金曲頒獎典禮|十大勁歌金曲頒獎典禮]]「最受歡迎新人獎－金獎」，成功地開展了歌影視三方面的發展。

1994年，她與臺灣歌手[[吳奇隆|吳奇隆]]合作，拍攝[[徐克|徐克]]導演的古典民間爱情悲剧電影《[[梁祝_(1994年電影)|梁祝]]》，及主唱劇中電影主題曲《梁祝》(與吳奇隆合唱)、《原來是您》等。歌唱方面，年初夥拍[[梁朝偉|梁朝偉]]參與[[商業電台|商業電台]]叱咤903廣播劇《初戀無限次》聲音演出，及主唱劇中主題曲《初戀》，又得到同公司的歌手[[巫啟賢|巫啟賢]]為她作曲－《祝福一句》及兩人合唱的《愛情來的時候》，歌曲皆流行一時，並於4月推出全新粵語EP《初戀》；下半年更推出了一首大熱歌曲《但願你明白》，此曲非常流行更成為她首支四台冠軍歌，11月推出的粵語專輯《[[毋忘我_(楊采妮專輯)|毋忘我]]》更獲得白金唱片銷量，人氣甚高。楊采妮是年亦推出了兩首頗深入民心的動畫主題曲－《愛的魔法》(日本動畫「幻法小魔星」主題曲)及《全城來吧Cha Cha Cha》(日本動畫《大耳鼠之比》主題曲)，在電視台的配合下，頗受小朋友群眾喜愛。1994年算是楊采妮歌唱電影事業的豐收年，不僅勇奪[[商業電台|商業電台]]頒發的[[1994年度叱咤樂壇流行榜頒獎典禮得獎名單|1994年度叱咤樂壇流行榜頒獎典禮得獎名單]]「叱吒樂壇新力軍女歌手－金獎」，《愛情來的時候》一曲更奪得[[十大勁歌金曲頒獎典禮|十大勁歌金曲頒獎典禮]]「最受歡迎男女合唱歌曲獎銅獎」。自歌手[[周慧敏|周慧敏]]被喻為「玉女掌門人」後，楊采妮更被香港傳媒封為「玉女接班人」人選。

1995年4月，楊采妮推出首張國語專輯《[[笑著流淚|笑著流淚]]》，反應甚好，成功開拓了台灣的知名度；此後她於95-97年間也推出了《[[為所欲為|為所欲為]]》、《[[懂不懂|懂不懂]]》、《[[神話_(楊采妮專輯)|神話]]》、《[[采妮|采妮]]》等多張國、粵語專輯，又拍攝了多部電影作哈包括：《[[墮落天使_(1995年電影)|墮落天使]]》([[王家衛|王家衛]]導演)、《[[94獨臂刀之情|94獨臂刀之情]]》、《[[花月佳期_(1995年電影)|花月佳期]]》、《[[自梳_(電影)|自梳]]》等，兼顧港、台兩地的歌影發展，深受歡迎。

1997年時，從事演藝工作5年後的她突然宣佈引退，決定與其新加坡籍男友[[邱韶智|邱韶智]]共同發展形象顧問事業，並於1997年中推出最後粵語專輯《[[采妮|采妮]]》以及最後一張新曲精選專輯《[[離別之前|離別之前]]》作為退出演藝圈的最後作品。

在2003年上旬，楊采妮與邱韶智經營的形象顧問公司[[清盤|倒閉]]，兼且她公開宣佈與男友邱韶智分離。2003年下旬，楊采妮再次復出演藝圈拍攝電影《[[新警察故事|新警察故事]]》、然後她再接拍電影《[[七劍|七劍]]》、《[[再說一次我愛你_(電影)|再說一次我愛你]]》、《[[父子|父子]]》、《[[浮城_(電影)|浮城]]》與《[[寒戰|寒戰]]》。

2009年，Charlie在演藝事業上展開了新的冒險，用了3年時間編寫了第一部電影劇本《[[聖誕玫瑰_(電影)|圣诞玫瑰]]》，亦因為這個劇本被第七屆香港亞洲電影投資會選出的劇本頒發卓越劇本獎。並成功製作及導演了她的第一部電影《[[聖誕玫瑰_(電影)|圣诞玫瑰]]》於2013年分別在香港、澳門、台灣、中國、新加坡等地上映。《[[聖誕玫瑰_(電影)|圣诞玫瑰]]》成功更因為Charlie 各方好友傾力相助，如[[徐克|徐克]]及[[张之亮|张之亮]]導演擔任其監製，又得到一眾演技出眾的好友郭富城、桂綸鎂、張震、秦海璐等。

2014年《[[聖誕玫瑰_(電影)|圣诞玫瑰]]》也獲邀請參加第四屆土耳其犯罪與懲罰的電影節，入圍該電影節的獎項提名。

== 慈善工作 ==
Charlie 希望藉以藝人的身份，希望社會各界關注價慈善活動，對於眾多的慈善機構她不唯餘力，她由2004年直到現在一直擔任大使為明愛、[[奧比斯|奧比斯]]香港大使及[[聯合國兒童基金會|聯合國兒童基金會]]大使。她多年來一直不斷地參與不同的慈善探訪，如落後的中國[[寧夏|寧夏]]、[[貴州|貴州]]、[[雲南|雲南]]以、因受天災而嚴重受創的[[四川|四川]]、衛生落後的[[尼泊爾|尼泊爾]]、飽受饑餓疾病困擾的[[非洲|非洲]]吉布提和[[斯里蘭卡|斯里蘭卡]]等地，提供救助和支持受災者使走過他們從心靈創傷。

熱愛海洋的Charlie 同樣熱衷關心於野生動物保育，因此獲得[[世界自然基金|世界自然基金]]和[[海洋公園海洋保育基金|海洋公園海洋保育基金]]邀請成為保育大使，希望透過Charlie 對海洋發動物的熱愛同時喚起社會人仕的關注並教育下一代。同樣非常關心如何保護弱勢社群的兒童及發展教育。2013年，Charlie以助養者的身份隨[[宣明會|宣明會]]訪問了[[格魯吉亞|格魯吉亞]]和[[亞美尼亞|亞美尼亞]]因戰火而變得貧窮的國家。此外，還與好朋友[[梁詠琪|梁詠琪]]、[[許茹芸|許茹芸]]、[[李心潔|李心潔]]成立了[[小黃花慈善教育基金|小黃花慈善教育基金]]以協助不同地區貧困兒童得到更多接受教育和學習的機會。

== 現况 ==
一代資深“玉女”楊采妮和43歲圈外男友邱韶智情牽19年，據悉，2013年5月邱韶智求婚成功，當時他說：“無論順境或逆境、富裕或貧窮、健康或疾病、快樂或憂愁，我將永遠愛著你、珍惜你，嫁給我吧！”深深感動楊采妮，答應求婚，小倆口確定2013年11月2日在[[新加坡|新加坡]]完成終身大事。楊采妮和邱韶智分分合合，2人當年高調相戀，1997年楊采妮更為愛息影從商，和邱韶智合開公司，怎知事業不順，2003年公司倒閉，她又復出拍片，並透露已分手。前年傳出分手8年的楊、邱復合，楊采妮對戀情轉趨低調，2013年5月初到台灣宣傳導演處女作《[[聖誕玫瑰_(電影)|聖誕玫瑰]]》，被問婚事還笑說：“沒那個感覺。”

其實早在前年2人舊情重燃後，男方就多次向她求婚，但楊采妮一心想當導演，希望圓夢再說，2013年5月23日是她39歲生日，邱韶智趁機再度求婚，終於打動芳心。據悉，楊采妮對婚事相當保密，因丈夫是新加坡人，貼心選在新加坡某五星級酒店完婚，1個多月前已經飛去試完婚紗。小倆口低調未印喜帖，透過e-mail或親口邀約，演藝圈受邀朋友僅約20人，除[[袁詠儀|袁詠儀]]夫婦、[[郭富城|郭富城]]、[[徐克|徐克]]等人，還有她的姊妹淘：[[梁詠琪|梁詠琪]]、[[李心潔|李心潔]]及[[許茹芸|許茹芸]]，[[許茹芸|許茹芸]]更會擔任她的伴娘。
