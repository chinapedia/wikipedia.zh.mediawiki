{{作品名稱|name=''The BFG''|hk=吹夢的巨人|tw=吹夢巨人|cn=圆梦巨人}}是一部2016年[[美國|美國]][[奇幻片|奇幻]][[冒險片|冒險片]]，由[[史蒂芬·史匹柏|史蒂芬·史匹柏]]執導和監製。電影改編自[[羅爾德·達爾|羅爾德·達爾]]創作的1982年[[吹夢巨人|同名小說]]。由{{link-en|露比·巴恩希爾|Ruby Barnhill}}、[[馬克·勞倫斯|馬克·勞倫斯]]、[[比爾·哈德爾|比爾·哈德爾]]和[[蕾貝卡·霍爾|蕾貝卡·霍爾]]主演。

[[華特迪士尼工作室電影|華特迪士尼工作室電影]]將電影定於2016年7月1日在美国上映，並同時釋出迪士尼數位3D與[[RealD_戲院|RealD 3D]]版本。儘管電影獲得正面居多的評價，但全球票房總計僅1.83億美元。

== 劇情 ==
在巨人國裡，所有巨人都會吃人。唯獨身長八公尺，在群體中顯得較矮的BFG不吃人。善良的他會在小朋友睡覺的時候把好夢用小喇叭吹進小朋友的腦袋裡，所以又被稱為「吹夢巨人」。 直到有一天一位名叫蘇菲的小女孩發現了他，之後便與英格蘭女王，三人一同成功制伏侵略人類世界的吃人巨人<ref>{{@movies|id=fben23691740|title=吹夢巨人 The BFG}}<!-- 2015-12-19 accessdate --></ref>。

== 發行 ==
電影在[[美國|美國]]發行於2016年7月1日<ref>{{cite news|last1=McNary|first1=Dave|title=Tom Hanks-Steven Spielberg Cold War Thriller Set for Oct. 16, 2015|url=http://variety.com/2014/film/news/tom-hanks-steven-spielberg-cold-war-thriller-set-for-oct-16-2015-1201221269/|accessdate=June 6, 2014|work=Variety|date=June 16, 2014}}</ref>，並由[[華特迪士尼工作室電影|華特迪士尼工作室電影]]發行於世界各地，除了歐洲、非洲和中東地區是由[[夢工廠|夢工廠]]負責。[[夢工廠|夢工廠]]的合作夥伴{{link-en|Reliance Entertainment|Reliance Entertainment}}將負責印度的發行權，而Entertainment One則定於2016年7月22日在[[英國|英國]]上映<ref>{{cite news|title=Steven Spielberg’s ‘The BFG’ Primed for U.K. Release With eOne|url=http://variety.com/2014/film/news/steven-spielbergs-the-bfg-primed-for-u-k-release-with-eone-1201261149/|accessdate=July 12, 2014|work=Variety|date=July 11, 2014}}</ref>。

=== 評價 ===
《吹夢巨人》獲得了普遍好評。[[爛番茄|爛番茄]]根據285位專業影評人持有75%的新鮮度，平均得分6.78／10<ref name="RT">{{cite web |url=http://www.rottentomatoes.com/m/the_bfg_2016/?search=BFG |title= The BFG (2016) |website=[[Rotten_Tomatoes|Rotten Tomatoes]] |accessdate=May 13, 2016}}</ref>，而在[[Metacritic|Metacritic]]上得到66分，[[IMDB|IMDB]]上得6.4分，正面評價居多<ref name="Meta">{{cite web |url=http://www.metacritic.com/movie/the-bfg |title=The BFG reviews |website=[[Metacritic|Metacritic]] |accessdate=May 15, 2016}}</ref>。

=== 票房 ===
美國於2016年7月1日上映，首映週末票房僅1960萬美元，位居當週票房排名第四，表現不佳。[[Deadline.com|Deadline.com]]認為，這是一個可怕的開始，因電影的成本高達1.4億美元<ref>{{cite web|url=http://www.forbes.com/sites/scottmendelson/2016/07/02/box-office-disneys-the-bfg-earns-not-so-huge-7m-friday/#415ea2a85279|title=Box Office: Disney's 'The BFG' Earns Not-So-Huge $7M Friday|author=Scott Mendelson|work=[[Forbes|Forbes]]|date=July 2, 2016|accessdate=July 3, 2016}}</ref><ref name="DEADopening">{{cite web|url=http://deadline.com/2016/07/independence-day-box-office-legend-of-tarzan-the-purge-election-year-the-bfg-1201782489/|title=‘Dory’ Dominates, ‘Tarzan’ Improves, ‘Purge’ Excels As ‘BFG’ Falls: July 4th Weekend B.O. – Sunday Update|author=Anthony D'Alessandro|work=''[[Deadline.com|Deadline.com]]''|date=July 3, 2016|accessdate=July 3, 2016}}</ref>，台灣則於8月11日上映，首映週末獲466萬新台幣位居當週第三，次週獲259萬降至第六，上映第3週以120萬位居第九。
