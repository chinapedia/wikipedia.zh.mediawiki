《'''吹雪'''》（{{lang-ja|ふぶき}}）是[[西澤幸奏|西澤幸奏]]的第1張單曲，於2015年2月18日由[[FlyingDog|FlyingDog]]發售。

== 概要 ==
《吹雪》為西澤幸奏的出道曲，為[[電視動畫|電視動畫]]《[[艦隊Collection|艦隊Collection]]》的[[片尾曲|片尾曲]]<ref>{{Cite web |url = http://headlines.yahoo.co.jp/hl?a=20150115-00065581-lisn-musi |title = 西沢幸奏「艦これ」ED曲のミュージッククリップ公開 |author = minp! |date = 2015年1月15日 |publisher = [[雅虎日本|雅虎日本]] |accessdate = 2015年4月11日 |deadurl = yes |archiveurl = https://web.archive.org/web/20150416171859/http://headlines.yahoo.co.jp/hl?a=20150115-00065581-lisn-musi |archivedate = 2015年4月16日 }}</ref>。

《吹雪》的歌詞中提到了[[吹雪型驅逐艦|吹雪型]]的部分艦艇（[[吹雪號驅逐艦_(吹雪型)|吹雪]]、[[白雪號驅逐艦_(吹雪型)|白雪]]、[[初雪號驅逐艦_(吹雪型)|初雪]]、[[深雪號驅逐艦_(吹雪型)|深雪]]、[[叢雲號驅逐艦_(吹雪型)|叢雲]]、{{link-ja|磯波號驅逐艦 (吹雪型)|磯波 (吹雪型駆逐艦)|磯波}}）{{註|白雪、初雪、深雪、叢雲與磯波均沒有在動畫中登場}}，且以上的艦娘都是由[[上坂堇|上坂堇]]配音<ref>{{Cite web|url=http://aniapp.animate.tv/archives/80334 |title=『艦隊これくしょん -艦これ-』美少女に擬人化した戦艦「艦娘」でオリジナル艦隊を編成! 艦娘のボイスには日高里菜さんや井口裕香さんといった人気声優陣が続々と参加 |publisher=animate TV anime app! |date=2013-04-22 |accessdate=2013-09-15 |deadurl=yes |archiveurl=https://web.archive.org/web/20130424041052/http://aniapp.animate.tv/archives/80334 |archivedate=2013-04-24 }}{{ja}}</ref>。

在單曲插圖中艦娘為吹雪。該曲與艦隊Collection的[[主題曲|主題曲]]《[[海色|海色]]》同時發售。

== 反應 ==
《海色》推出後在僅僅幾天之內便與《吹雪》一起於[[秋葉原|秋葉原]]「消失」，各售賣點均貼出售罄字樣，同時[[亞馬遜日本|亞馬遜日本]]的庫存亦全數被訂購完畢<ref>{{Cite web |url = https://www.facebook.com/fmg2010/photos/pb.178979035485497.-2207520000.1427608356./895614233821970/?type=3&permPage=1 |title = 模物語 |author = 模物語 |date = 2015年2月24日 |publisher = [[Facebook|Facebook]] |accessdate = 2015年3月29日}}</ref>。最後經計算後首週賣出了超過2.8萬張<ref name=yahoo>{{Cite web |url = http://headlines.yahoo.co.jp/hl?a=20150224-00069555-lisn-musi |title = 「艦これ」シングルでAKINOが通算2作目のTOP10入り、西沢幸奏は今年デビューのソロアーティストで最高位 |author = minp! |date = 2015年2月24日 |publisher = [[雅虎日本|雅虎日本]] |accessdate = 2015年3月29日}}</ref>。

[[日本告示牌#Hot_Animation|Billboard Hot Animation]]方面，《吹雪》初登場便登上第2位，僅次於《海色》<ref>{{Cite web |url = http://www.billboard-japan.com/d_news/detail/26154/2 | title =『艦これ』主題歌がビルボードアニメチャート1、2フィニッシュで“ぱんぱかぱーん！” |publisher = [[Billboard_JAPAN|Billboard JAPAN]] |date = 2015年2月24日}}</ref>。另外吹雪和海色亦取得了[[iTunes|iTunes]]日本排行榜日榜（2月19日）<ref>{{cite web |url=http://www.tnsori.com/20150218-itunes-topsongs-daily/2|title= 15/02/18付iTunesトップソング：「艦これ」初日1位・2位独占！1位AKINO from bless4、2位西沢幸奏|publisher=tnsori.com|date=2015年2月19日|accessdate=2015年3月29日}}</ref>與週榜<ref>{{cite web |url=http://www.tnsori.com/20150302-itunes-topsongs-weekly/2|title= 15/03/02付　オリジナルiTunes週間トップソング50|publisher=tnsori.com|date=2015年2月25日|accessdate=2015年3月29日}}</ref>的冠亞軍。而在3月2日公佈的[[Oricon公信榜|Oricon公信榜]]週榜中，《吹雪》取得第6位，緊貼在《海色》之後<ref name=yahoo/>。
