'''鐵路便當'''，為[[鐵路車站|鐵路車站]]或列車車廂內所販售的[[便當|便當]]，在[[日語|日語]]中稱作「{{lang|ja|駅弁}}」（Ekiben）。
在日本，不同車站與行駛在特定地區的列車，經常會販售融入當地特有食材的鐵路便當。

== 定義 ==

{{Fact|在車站內部月台上販售或是在列車上販售者，以[[米飯|米飯]][[主食|主食]]才算是便當。歐美地區以對於Lunch（午餐）和Dinner（晚餐）較為模糊，通常是以[[麵包|麵包]]、[[三明治|三明治]]、[[點心|點心]]為主，雖裝在盒內販售，但並非便當。|time=2019-03-10T12:41:00+00:00}}

== 起源 ==

搭乘火車時，會長時間被拘束在[[鐵路列車|列車]]上，依搭車時間有時會需要用餐，因此出現了在車站或車廂販售便當的型態。

[[日本|日本]]最早的鐵路便當於[[東京|東京]][[上野|上野]]至[[櫔木縣|櫔木縣]][[宇都宮|宇都宮]]1885年7月16日全線通車後出現宇都宮車站，由白木屋旅館出售，售價5錢，便當只有撒上[[芝麻|芝麻]]的[[飯糰|飯糰]]和[[蘿蔔乾|蘿蔔乾]]、[[酸梅|酸梅]]，用竹葉包裹。

另一種説法是1877已經有紀錄於[[大阪火車站|大阪火車站]]和[[神户火車站|神户火車站]]有小販將便當翹在胸前在[[月台|月台]]叫賣。

== 日本 ==

日本的駅弁通常專指於火車站售賣的便當，可能會依季節變動駅弁的內容，根據銷售策略更換名稱和包裝。

早期的駅弁容器以薄木片(經木)為主，後以紙和塑膠為主，陶、木、竹等亦為主流物料。

駅弁通常為冷食，第1款熱食駅弁於1988年3月新神戶站出售。

== 台灣 ==

{{main|臺鐵便當}}

[[File:福井食堂鐵路便當01.jpg|thumb]]
[[File:THSR_Bento_Shop_in_HSR_Chiayi_Station.jpg|thumb]]的高鐵便當]]
臺鐵-{zh-hans:盒饭（当地称为“便当”）;zh-hant:便當}-是指[[臺灣鐵路管理局|臺灣鐵路管理局]]於車站或列車內販售之鐵路便當，今由該局[[臺灣鐵路管理局|臺灣鐵路管理局]]餐旅服務總所負責統製。

非官方的小販便當在2000年重新販賣前，部分車站出現小販，兜售便當，補足餐飲縫隙，提供乘車者裹腹之慾。而不在車站或列車賣的鐵路便當，在電視傳播下，1990年代起台灣興起鐵路便當風潮。[[統一超商|統一超商]]為搶攻外食族用餐的市場，推出[[奮起湖車站|奮起湖]]、[[福隆車站|福隆]]、[[關山車站_(臺灣)|關山]]、[[池上車站_(臺灣)|池上]]、[[高雄車站|高雄]]等車站的鐵路便當，更在2007年引進[[日本鐵道|日本鐵道]]便當。並非在車站或是列車內販賣，只有參考鐵路便當菜色與販售命名的「鐵路便當店」，更在各地陸續出現。棒球選手[[黃平洋|黃平洋]]退休後也在[[天母棒球場|天母棒球場]]附近開起便當店，名稱為「黃平洋鐵路便當」。也在[[彰化縣|彰化縣]][[社頭鄉|社頭鄉]]便當店，名稱為「福井食堂」。

== 中国 ==
[[File:201704_Package_delivering_truck_on_the_platform_of_Shanghai_Station.jpg|thumb]]站台上的动车组列车送餐车]]
中国大陆的铁路-{zh-hans:盒饭;zh-hant:便當（當地稱為"盒飯"）}-分为车站出售的盒饭以及列车出售的盒饭两种。从前，车站在出售盒饭的同时还会推销一些当地的特色食品，比如[[天津站|天津站]]的[[狗不理包子|狗不理包子]]、[[嘉兴站|嘉兴站]]的[[嘉興粽子|嘉興粽子]]等等。铁路的发展使得部分地方食品逐渐风行全国：比如四大铁路鸡（[[德州扒鸡|德州扒鸡]]、[[道口烧鸡|道口烧鸡]]、[[符离集烧鸡|符离集烧鸡]]、[[沟帮子熏鸡|沟帮子熏鸡]]<ref>{{cite news|title=每个坐火车的人都需要一只德州扒鸡|url=http://www.sohu.com/a/156561525_409307|publisher=搜狐|date=2017-07-12}}</ref>）等。同时也催生出了一些只有在火车站才售卖的特色热食，比如[[上饶站|上饶站]]出售的鸡腿<ref>{{cite news|title=上饶特色美食 肉质鲜美 肥而不腻 上饶鸡腿|url=http://jx.sina.com.cn/sr/eat/2014-03-05/224.html|publisher=新浪江西|date=2014-03-05}}</ref>、[[韶关东站|韶关站]]（现韶关东站）的[[煲仔饭|煲仔饭]]<ref>{{cite news|author=曾胜泉|url=http://news.sina.com.cn/c/177939.html|title=春运写真：广东韶关每天30万民工南下|publisher=[[羊城晚报|羊城晚报]]|date=2001-02-03|accessdate=2017-04-01}}</ref>。随着铁路的发展，列车停站时间减少；加上食品安全问题，在站台上出售盒饭或热食的车站越来越少<ref>{{cite news|title=韶关火车站的卤鸡腿又有得吃了吗？|url=http://news.dayoo.com/guangzhou/201105/11/73437_16618978.htm|publisher=大洋网|date=2011-05-11|deadurl=yes|archiveurl=https://web.archive.org/web/20170118142614/http://news.dayoo.com/guangzhou/201105/11/73437_16618978.htm|archivedate=2017-01-18}}</ref>。

中国旅客列车餐饮经营分为既有线餐车经营和动车组航空式配餐2种方式。

=== 餐车经营 ===

主要由铁路局客运段直接经营、铁路内部职工承包经营或餐饮服务公司经营三种。有[[T65/66次列车|T65/66次列车]]的牛肉面<ref>{{cite news|title=餐车大厨的招牌牛肉面|url=http://www.js.xinhuanet.com/2014-04/11/c_1110203629.htm|date=2014-04-11|publisher=新华网|deadurl=yes|archiveurl=https://web.archive.org/web/20170116155942/http://www.js.xinhuanet.com/2014-04/11/c_1110203629.htm|archivedate=2017-01-16}}</ref>、南宁客运段的[[南寧米粉|南寧米粉]]、成都客运段的[[川菜|川菜]]等等<ref>{{cite web|title=舌尖上的春运!这些火车餐对得起回家路|url=http://auto.dahe.cn/2016/01-28/106386691.html|date=2016-01-28|publisher=大河网}}</ref>。

=== 动车组列车 ===
[[File:Steamed_pork_belly_set_meal_on_G27_(20160116114435).jpg|thumb]]高速动车组列车上售卖的一种冷链盒饭]]
中国铁路动车组列车除部分运营线路（如[[广深动车组列车|广深动车组]]）外大多不设餐座，因此盒饭存储主要有热链、冷链和常温链三种存储方式。为保障食品安全，动车组列车一般冷链配餐为主，但因此导致的口感下降，以及过高的价格常常引起旅客的诟病<ref>{{cite news|title=火车盒饭何以又贵又难吃？|url=http://zhenhua.163.com/16/0126/17/BE9912P90004662N.html|publisher=网易新闻|date=2016-01-26|access-date=2017-01-14|archive-url=https://web.archive.org/web/20170119004626/http://zhenhua.163.com/16/0126/17/BE9912P90004662N.html|archive-date=2017-01-19|dead-url=yes}}</ref><ref>{{cite news|url=https://news.rthk.hk/rthk/ch/component/k2/1419574-20180923.htm|title=高鐵往北京列車有午市套餐　售價65元人民幣|publisher=[[香港電台|香港電台]]|date=2018-09-23|accessdate=2019-03-17|archiveurl=https://web.archive.org/web/20180924115700/http://news.rthk.hk/rthk/ch/component/k2/1419574-20180923.htm|archivedate=2018-09-24}}</ref><ref>{{cite news|author=黎仕南; 盧熙林|url=https://hk.news.appledaily.com/local/daily/article/20180924/20506866|title=港式食品首日無供應|publisher=[[蘋果日報_(香港)|蘋果日報]]|date=2018-09-24|accessdate=2019-03-17|archiveurl=https://web.archive.org/web/20190317071014/https://hk.news.appledaily.com/local/daily/article/20180924/20506866|archivedate=2019-03-17}}</ref><ref>{{cite web|url=http://views.ce.cn/view/ent/201901/29/t20190129_31389110.shtml|title=高铁外卖哥让服务与体验无缝衔接|publisher=中国经济网|date=2019-01-29|accessdate=2019-03-17|archiveurl=https://web.archive.org/web/20190317071957/http://views.ce.cn/view/ent/201901/29/t20190129_31389110.shtml|archivedate=2019-03-17}}</ref>。

=== 互聯網訂餐 ===
[[File:Mao-style_braised_pork_lunch_box_ordered_from_Changshanan_(20181216113449).jpg|thumb]]预订的一种社会品牌餐食]]
2015年起，部分铁路局与网络平台合作，推出利用互联网订餐的服务。重庆客运段、西安客运段推出微信订餐服务<ref>{{cite news|title=成渝动车首推微信订餐等系列“微服务”|url=http://www.huaxia.com/jxtf/jrsc/xwsc/2015/02/4278511.html|date=2015-02-13|publisher=华夏经纬网}}</ref>。[[上海铁路局|上海铁路局]]华铁旅服于2017年1月13日正式上线[[饿了么|饿了么App]]，在运行初期可以提供共25趟列车<ref>{{cite news|title=25列高铁可用饿了么点外卖|url=http://finance.sina.com.cn/roll/2017-01-12/doc-ifxzqhka2835071.shtml|publisher=新浪财经|date=2017-01-12}}</ref>。2017年7月17日，27个于各[[省会|省会]]及[[计划单列市|计划单列市]]所在地的车站开始试点[[高速动车组列车|G]]/[[普通动车组列车|D字头]]动车组列车互联网订餐服务，乘客可通过[[中国铁路客户服务中心|中国铁路客户服务中心]]网站或手机客户端预订沿途车站餐厅供应的社会品牌餐食<ref>{{cite news|url=http://www.stardaily.com.cn/2017/0711/58274.shtml|title=动车、高铁17日起可以点外卖 12306网站、手机APP等方式均可预订|publisher=[[北京娱乐信报|北京娱乐信报]]|date=2017-07-11|accessdate=2017-10-11}}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，此类餐食预订服务于2018年1月18日增加沿线车站地方特产预订功能，订餐时限由2小时缩短为1小时<ref>{{cite news|url=https://zj.zjol.com.cn/news.html?id=851914|title=今日起高铁外卖升级 开车前1小时可订餐还能点地方特产|publisher=浙江新闻客户端|date=2018-01-18|accessdate=2019-03-08}}</ref>。
