== 劇情簡介 ==
為了商談而前往[[箱根|箱根]]山中的骨董商'''今川雅澄'''在居住的旅社——仙石樓遇到了在系列作的《[[姑獲鳥之夏|姑獲鳥之夏]]》亦有出場的老人'''久遠寺嘉親'''。以及為了採訪禪寺『明慧寺』隨後而來的《稀譚月報》採訪團——'''中禪寺敦子'''、'''飯窪'''和以攝影師身分參與的'''鳥口守彥'''等人，在旅社庭院中發現了一個突然出現的僧侶屍體。

另一方面'''京極堂'''為了應經營書店的老前輩之邀，前往箱根查勘意外被發現的書庫，以及為了旅行而順道隨行的京極堂之妻和'''關口'''夫妻也到了同座山上……

== 登場人物 ==
; {{nihongo|[[百鬼夜行系列#中禪寺秋彥|中禪寺 秋彥]]|{{ruby|中禅寺|ちゅうぜんじ}} {{ruby|秋彦|あきひこ}}|Chuuzenji Akihiko}}
: 二手書店「京極堂」經營者，也是「武藏晴明神社」代代相傳的神主和陰陽師的怪人，親朋好友習慣稱呼他「京極堂」。性好讀書，也對科學和宗教有著平等的熱愛。口頭禪是「這世上沒有不可思議的事」。
: 詳情請參閱「[[百鬼夜行系列#中禪寺秋彥|百鬼夜行系列#主要登場人物]]」。

; {{nihongo|[[百鬼夜行系列#關口巽|關口 巽]]|{{ruby|関口|せきぐち}} {{ruby|巽|たつみ}}|Sekiguchi Tatsumi}}
: 出過幾本選集的小說家，和京極堂是讀書時代的好友。
: 詳情請參閱「[[百鬼夜行系列#關口巽|百鬼夜行系列#主要登場人物]]」。

; {{nihongo|[[百鬼夜行系列#榎木津禮二郎|榎木津 禮二郎]]|{{ruby|榎木津|えのきづ}} {{ruby|礼二郎|れいじろう}}|Enokizu Reijirou}}
: 經營「薔薇十字偵探社」的偵探。京極堂的友人。雖然稱為偵探，但是行事隨便，而且容易忘記他人的名稱。有著俊美的外表，和看到他人記憶的特殊能力。
: 詳情請參閱「[[百鬼夜行系列#榎木津禮二郎|百鬼夜行系列#主要登場人物]]」。

; {{nihongo|[[百鬼夜行系列#中禪寺敦子|中禪寺 敦子]]|{{ruby|中禅寺|ちゅうぜんじ}} {{ruby|敦子|あつこ}}|Chuuzenji Atsuko}}
: 京極堂之妹，《稀譚月刊》的編輯。
: 詳情請參閱「[[百鬼夜行系列#中禪寺敦子|百鬼夜行系列#主要人物的家人]]」。

; {{nihongo|[[百鬼夜行系列#鳥口守彥|鳥口 守彥]]|{{ruby|鳥口|とりぐち}} {{ruby|守彦|もりひこ}}|Toriguchi Morihiko}}
: 關口偶爾撰稿的雜誌《月刊實錄犯罪》之編輯。極度路癡，且容易用錯成語及典故。對攝影有著興趣。
: 詳情請參閱「[[百鬼夜行系列#鳥口守彥|百鬼夜行系列#主要登場人物]]」。

; {{nihongo|[[百鬼夜行系列#今川雅澄|今川 雅澄]]|{{ruby|今川|いまがわ}} {{ruby|雅澄|まさすみ}}|Imagawa Masasumi}}
: 古董店「待古庵」的經營者，出自[[蒔繪|蒔繪]]世家的次子，為了逃出次子的身分的壓力而接手分家經營的古董店，對於自己古董的辨別能力感到沒有自信。
: 戰時是榎木津的部隊手下，深受榎木津的無常個性所害。這次為了商談而前往「仙石樓」。
: 詳情請參閱「[[百鬼夜行系列#今川雅澄|百鬼夜行系列#主要人物的交友關係]]」。

; {{nihongo|[[百鬼夜行系列#久遠寺嘉親|久遠寺 嘉親]]|{{ruby|久遠寺|くおんじ}} {{ruby|嘉親|よしちか}}|Kuonji Yoshichika}}
: 原本經營醫院的老人，在[[姑獲鳥之夏|姑獲鳥之夏]]的事件之後，放棄醫院院長的身分，到山中旅社「仙石樓」過著隱居生活。
: 詳情請參閱「[[百鬼夜行系列#久遠寺嘉親|百鬼夜行系列#主要人物的交友關係]]」。

===明慧寺===
; <span id="圓覺丹">{{nihongo|圓 覺丹|{{ruby|円|まどか}} {{ruby|覚丹|かくたん}}|Madoka Kakutan}}</span>
: 明慧寺貫首，68歲的老人。

; {{nihongo|大西 泰全|{{ruby|大西|おおにし}} {{ruby|泰全|たいぜん}}|Oonishi Taizen}}
: 明慧寺最年長的僧侶，88歲，被眾人尊稱為「老師」。

; {{nihongo|小坂 了稔|{{ruby|小坂|こさか}} {{ruby|了稔|りょうねん}}|Kosaka Ryounen}}
: 明慧寺四知事之一的值歲，60歲，明慧寺對外的唯一窗口。和今川經營的古董店前店主有過幾次的生意往來，也是這次邀約今川前往商談者，行事胡作非為，被叫做「破戒僧」。本事件的第一名受害者。

; {{nihongo|和田 慈行|{{ruby|和田|わだ}} {{ruby|慈行|じあん}}|Wada Jian}}
: 明慧寺四知事之一的監院，同時也兼任知客的職位，28歲。是個外表俊美的年輕僧侶，有著嚴格的個性，看不起了稔的胡作非為。

; {{nihongo|中島 佑賢|{{ruby|中島|なかじま}} {{ruby|佑賢|ゆうけん}}|Nakajima Yuuken}}
: 明慧寺四知事之一的維那，56歲。值得讚賞的禪師，和了稔關係較好。

; <span id="桑田常信">{{nihongo|桑田 常信|{{ruby|桑田|くわだ}} {{ruby|常信|じょうしん}}|Kuwada Joushin}}</span>
: 明慧寺四知事之一的典座，48歲。和慈行及了稔交惡。

; {{nihongo|[[姑獲鳥之夏#菅野博行|菅野 博行]]|{{ruby|菅野|すがの}} {{ruby|博行|はくぎょう}}|Sugano Hakugyou}}
: 明慧寺的前任典座，70歲的老人，因為發狂而被監禁在土牢中。實際是「[[姑獲鳥之夏|姑獲鳥之夏]]」事件中失蹤的小兒科醫生「菅野博行」。

; {{nihongo|加賀 英生|{{ruby|加賀|かが}} {{ruby|英生|えいしょう}}|Kaga Eishou}}
: 慈行的侍僧，18歲。在不久之前還只是暫到，由了稔引介入寺。

; <span id="牧村托雄">{{nihongo|牧村 托雄|{{ruby|牧村|まきむら}} {{ruby|托雄|たくゆう}}|Makimura Takuyuu}}</span>
: 常信的侍僧，22歳。以前是博行的侍僧。

; <span id="杉山哲童">{{nihongo|杉山 哲童|{{ruby|杉山|すぎやま}} {{ruby|哲童|てつどう}}|Sugiyama Tetsudou}}</span>
: 由仁秀老人撿到的嬰兒，由於[[關東大地震|關東大地震]]而失去家人，28歳。在明慧寺剃度。為人木訥，幾近遲緩。

===其他===

===警察===
; {{nihongo|[[百鬼夜行系列#石井寬爾|石井 寬爾]]|{{ruby|石井|いしい}} {{ruby|寛爾|かんじ}}|Ishii Kanji}}
: 日本國家警察神奈川縣本部搜查一課警部，為了代替山下管理這次事件而來。

; {{nihongo|[[百鬼夜行系列#山下德一郎|山下 德一郎]]|{{ruby|山下|やました}} {{ruby|徳一郎|とくいちろう}}|Yamashita Tokuichirou}}
: 日本國家警察神奈川縣本部搜查一課警部補，本次事件的搜查主任。不善管理，對這次事件的關係人抱著懷疑之心。

; {{nihongo|[[百鬼夜行系列#益田龍一|益田 龍一]]|{{ruby|益田|ますだ}} {{ruby|龍一|りゅういち}}|Masuda Ryuuichi}}
: 日本國家警察神奈川縣本部搜查一課刑警，階級是巡查。山下的部下，為人親和力高。

; {{nihongo|菅原 剛喜|{{ruby|菅原|すがわら}} {{ruby|剛喜|たけよし}}|Sugawara Takeyoshi}}
: 日本國家警察神奈川縣箱根署刑警，階級是巡查長。慣於用逼問自白的方式查案。
