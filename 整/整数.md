'''整数'''，是[[序列|序列]]<math>\{\ldots, -4, -3, -2, -1, 0, 1, 2, 3, \ldots\}</math>中所有的[[数|数]]的统称，包括[[负整数|负整数]]、[[0|零]]（0）与[[正整数|正整数]]。和[[自然數|自然數]]一樣，整數也是一個[[可數|可數]]的[[無限集合|無限集合]]。這個[[集合_(数学)|集合]]在数学上通常表示[[粗體|粗體]]<math>Z</math>或<math>\mathbb{Z}</math>，源于[[德语|德语]]单词''Zahlen''（意为“[[数|数]]”）的[[首字母|首字母]]。

在[[代數數論|代數數論]]中，這些屬於[[有理數|有理數]]的一般[[整數|整數]]會被稱為'''有理整數'''，用以和[[高斯整數|高斯整數]]等的概念加以區分。

== 正整数与负整数 ==
{{main|正整數|负整数}}
整數是一个[[集合_(数学)|集合]]，通常可以分为[[正整數|正整數]]、[[零|零]]（0）和[[負整數|負整數]]。'''正整數'''（符号：'''Z'''<sup>+</sup>或<math>\mathbb{Z}^{+}</math>）即[[大於|大於]]0的整數，是[[正数|正数]]与整数的[[交集|交集]]。而'''負整數'''（符号：<math>Z^-</math>或<math>\mathbb{Z}^{-}</math>）即[[小於|小於]]0的整數，是[[负数|负数]]与整数的交集。和整數一样，两者都是[[可數|可數]]的[[無限集合|無限集合]]。除正整數和負整數外，通常将0與正整數统称为'''非負整數'''（符号：'''Z'''<sup>+</sup><sub>0</sub>或<math>\mathbb{Z}^{+}_{0}</math>），而将0與負整數统称为'''非正整數'''（符号：'''Z'''<sup>-</sup><sub>0</sub>或<math>\mathbb{Z}^{-}_{0}</math>）。在[[数论|数论]]中[[自然数|自然数]]通常被视为与正整數等同，即1，2，3等，但在[[集合论|集合论]]和[[计算机科学|计算机科学]]中自然数则通常是指非负整数，即0，1，2等。

全体'''整数'''关于[[加法|加法]]和[[乘法|乘法]]形成一个环。[[环论|环论]]中的[[整环|整环]]、[[无零因子环|无零因子环]]和[[唯一分解域|唯一分解域]]可以看作是整数的[[抽象化|抽象化]]模型。

'''<math>\mathbb{Z}</math>'''是一个加法[[循环群|循环群]]，因为任何整数都是若干个1或-1的和。1和-1是'''<math>\mathbb{Z}</math>'''仅有的两个生成元。每个元素个数为无穷个的循环群都与<math>(\mathbb{Z},+)</math>[[同构|同构]]。

== 有序性质 ==

'''Z'''是一个[[全序集|全序集]]，没有上界和下界。<math>\mathbb{Z}</math>的序列如下：
:<math>\ldots < -2 < -1 < 0 < 1 < 2 < \ldots</math>

一个整数大于零则为正，小于零则为负。零既非正也非负。

整数的序列在代数运算下是可以比较的，表示如下：

#若<math>a < b</math>且<math>c < d</math>，则<math>a + c < b + d</math>
#若<math>a < b</math>且<math>c > 0</math>，则<math>a * c < b * c</math>；若<math>c < 0</math>，则<math>a * c > b * c</math>.

整数环是一个[[欧几里德域|欧几里德域]]。

== <math>\mathbb{Z}</math>的基數 ==

<math>\mathbb{Z}</math>的[[基数_(数学)|基數]]（或勢）是[[阿列夫数|ℵ]]<sub>0</sub>，與<math>\mathbb{N}</math>相同。這可以從'''<math>\mathbb{Z}</math>'''建立一[[雙射函數|雙射函數]]到'''<math>\mathbb{N}</math>'''來證明，亦即該函數要同時滿足[[單射|單射]]及[[滿射|滿射]]的條件，例如：

:<math>f(x) = \begin{cases} 2x + 1, & \mbox{if } x \ge 0 \\ 2|x|, & \mbox{if } x < 0 \end{cases}</math>

當該函數的[[定義域|定義域]]僅限於'''<math>\mathbb{Z}</math>'''，則證明'''<math>\mathbb{Z}</math>'''與'''<math>\mathbb{N}</math>'''可建立一一對應的關係，即兩集[[等勢|等勢]]。
