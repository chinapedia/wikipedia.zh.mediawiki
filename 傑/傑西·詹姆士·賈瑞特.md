'''傑西·詹姆士·賈瑞特'''（{{lang-en|Jesse James Garrett}}）是一名[[用戶體驗設計|用戶體驗設計]]領域的設計師、網頁體驗設計公司Adaptive Path的創辦人，且為在使用[[瀏覽器|瀏覽器]]瀏覽[[網頁|網頁]]時；於頁面更新內容時的過程中減低需消耗伺服器傳送資源之[[非同步|非同步]]更新技術「[[AJAX|AJAX]]」概念的命名者。<ref>{{Cite web |url=http://blog.jjg.net/weblog/2005/02/ajax.html |title=Ajax |work=Jjg.Net |author=傑西·詹姆士·賈瑞特 |accessdate=2014年10月24日 }}</ref>

==生平==
出生於[[加拿大|加拿大]]的[[渥太華|渥太華]]，在年幼時移居至[[美國|美國]]的[[洛杉磯|洛杉磯]]居住，於19歲後則改搬至[[舊金山|舊金山]]。<ref name="About">{{Cite web |url=http://www.jjg.net/about/ |title=About Me |work=Jjg.Net |author=傑西·詹姆士·賈瑞特 |accessdate=2014年10月24日 }}</ref>

從[[佛羅里達大學|佛羅里達大學]]畢業後1995年傑西開始從事於網路介面的開發、即科技作家方面的職務，之後2001年另成立了專門處理[[使用者經驗|使用者經驗]]的公司Adaptive Path。<ref name="About"/>在[[Google|Google]]公司的[[電子地圖服務|電子地圖服務]]「[[Google地圖|Google地圖]]」於2005年上線後，當年2月傑西發表出一篇談論此讓網頁透過[[Javascript|Javascript]]以[[XML|XML]]格式來回傳資料、達到非同步更新網頁內容技術的文章，並將此技術以「[[AJAX|AJAX]]」來簡稱之、來代表原先含意「Asynchronous JavaScript and XML」的縮寫，<ref>{{Cite web |url=http://www.wrox.com/WileyCDA/Section/What-is-Ajax-.id-303217.html |title=What is Ajax? |work=Wrox |author=Chris Ullman |date=2008年10月17日 |accessdate=2014年10月24日 }}</ref>而傑西所提出的內容讓他獲得2006年雜誌《Wired Magazine》頒發的Wired Magazine's Rave大獎，<ref>{{Cite web |url=http://www.leighbureau.com/speakers/JGarrett/ |title=Jesse James Garrett President and co-founder, Adaptive Path Author, The Elements of User Experience Developed the web concept "Ajax" |work=Leighbureau  |accessdate=2014年10月24日 }}</ref>也讓往後資訊界以AJAX來稱呼此網頁設計技術。

之後2008年傑西與[[Mozilla公司|Mozilla公司]]合作，構思名為「Aurora」的案子來設想未來使用者在操控[[瀏覽器|瀏覽器]]可能會接觸到的使用體驗，<ref>{{Cite web |url=http://hci.stanford.edu/courses/cs547/abstracts/08-09/081017-garrett.html |title=Aurora: Envisioning the Future of the Web |work=[[史丹佛大學|史丹佛大學]] |date=2008年10月17日 |accessdate=2014年10月24日 }}</ref>而他主導讓使用者透過網路能更便利分享訊息的「iWitness」專案也獲得2011年邁阿密奈特基金會的創新大獎優勝者。<ref>{{Cite web |url=http://www.knightfoundation.org/press-room/press-release/knight-foundation-media-innovation-contest-announc/ |title=Knight Foundation media innovation contest announces 2011 winners |work=Knightfdn |date=2011年6月22日 |accessdate=2014年10月24日 |deadurl=yes |archiveurl=https://web.archive.org/web/20110801115036/http://www.knightfoundation.org/press-room/press-release/knight-foundation-media-innovation-contest-announc/ |archivedate=2011年8月1日 }}</ref>
