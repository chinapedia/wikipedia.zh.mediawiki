'''綠黨社會民主黨聯盟'''，簡稱'''綠社盟'''，是[[臺灣|臺灣]]的一個[[選舉聯盟|選舉聯盟]]，成立于2015年8月31日，由[[綠黨_(臺灣)|綠黨]]和[[社會民主黨_(臺灣)|社會民主黨]]（簡稱社民黨）共同組成，集結社會高度理想型力量，以建立綠色[[社會民主|社會民主]]理想國為目標。

==歷史==
1996年1月25日，[[綠黨_(臺灣)|綠黨]]建黨，定位為[[公民社會|公民社會]]開放的參政平台，鼓勵具有共同政治理念、價值與目標的積極公民加入；名稱中之「[[綠色#象徵意義|綠]]」係指[[綠色政治|綠色政治]]，而非[[民主進步黨|民主進步黨]]所屬之[[泛綠陣營|泛綠陣營]]。綠黨的政治主張包含[[環境保護|環境保護]]、[[生態平衡|生態平衡]]、注重[[草根民主|草根民主]]、堅持[[社會平等|社會平等]]與[[世界和平|世界和平]]。在選舉方面，因長期關心勞工、弱勢、環保議題等，得以在社會上維持一定的影響力；直至2006年參選臺北市議員後，在數次的選舉中得票數始顯著成長，後更於[[2012年中華民國立法委員選舉|2012年立委選舉]]上超越[[新黨|新黨]]取得第五多政黨票。2014年[[2014年中華民國地方公職人員選舉|九合一選舉]]推出10人參與地方選舉，分別於[[桃園市議會|桃園市議會]]、[[新竹縣議會|新竹縣議會]]各獲得一席議員，首次進入[[地方議會|地方議會]]<ref>[http://www.ettoday.net/news/20141129/432467.htm 快訊／綠黨議員候選人 竹縣周江杰、桃市王浩宇當選]，[[ETtoday新聞雲|ETtoday新聞雲]]，2014年11月29日</ref>。

2015年3月29日，[[社會民主黨_(臺灣)|社會民主黨]]建黨，由[[國立臺灣大學|國立臺灣大學]]社會學系副教授[[范雲_(臺灣)|范雲]]、律州聯合法律事務所律師[[賴芳玉|賴芳玉]]、[[國立中正大學|國立中正大學]]政治學系助理教授[[陳尚志|陳尚志]]、[[臺灣少年權益與福利促進聯盟|臺灣少年權益與福利促進聯盟]]秘書長[[葉大華|葉大華]]、前《[[四方報|四方報]]》總編輯[[張正|張正]]、[[中央研究院政治學研究所|中央研究院政治學研究所]]博士後研究員[[伍維婷|伍維婷]]、[[台灣同志諮詢熱線協會|台灣同志諮詢熱線協會]]文宣部主任[[呂欣潔|呂欣潔]]、國立臺灣大學政治學研究所博士生[[江俊宜|江俊宜]]、社會民主黨籌備處秘書長[[嚴婉玲|嚴婉玲]]等人發起籌組，為標舉[[社會民主主義|社會民主主義]]理念的新政黨。

2015年4月9日，綠黨批評，台灣過去16年來[[政黨輪替|政黨輪替]]的經驗證明，[[中國國民黨|中國國民黨]]與民進黨不論在朝或在野，執政者皆無法深得民心，在野者亦無法形成有效的監督力量，[[兩黨制|兩大黨的政治結構]]已一再證明無法為台灣政治形成良好的競爭關係；真正符合台灣人民利益與期待的[[第三勢力|第三勢力]]，既不會是民進黨的側翼，也不會是[[投機|投機]]的新瓶舊政客，更不會是從現有的政商結構中長大的[[政二代|政二代]]；綠黨不會與民進黨就2016年區域立委選舉進行任何協商<ref>{{cite news|author=綠黨|title=綠黨不會與民進黨進行協商 綠黨不會是民進黨的側翼|url=http://www.coolloud.org.tw/node/82090|newspaper=[[苦勞網|苦勞網]]|date=2015-04-09|accessdate=2016-04-29}}</ref>。

2015年8月17日，社會民主黨與綠黨合開記者會宣布共組參政聯盟，綠黨召集人[[李根政|李根政]]、[[張育憬|張育憬]]與社民黨召集人[[范雲_(臺灣)|范雲]]等人簽署〈參政聯盟合作備忘錄〉，參政聯盟定名為「綠黨社會民主黨聯盟」共同爭取立法委員席次，兩黨人士現場高喊「人民團結站出來，第三勢力進國會，淘汰國民黨、制衡民進黨」<ref>{{Cite web|url=http://www.coolloud.org.tw/node/83263|author=宋小海|language=zh-tw|title=綠黨社民黨宣佈共組參政聯盟|publisher=苦勞網|date=2015-08-17|accessdate=2015-08-20}}</ref>；並在立法委員選舉後，將朝二黨合併為一個黨的方向進行<ref>{{Cite web|url=http://pnn.pts.org.tw/main/2015/08/17/%E7%B6%A0%E9%BB%A8%E3%80%81%E7%A4%BE%E6%B0%91%E9%BB%A8%E7%B0%BD%E5%90%88%E4%BD%9C%E5%82%99%E5%BF%98%E9%8C%84|author=邱彥瑜|language=zh-tw|title=綠黨、社民黨共組參政聯盟 選後討論實質整合|publisher=[[公視新聞|公視新聞]]議題中心|date=2015-08-17|accessdate=2016-05-08|archive-url=https://web.archive.org/web/20160503012642/http://pnn.pts.org.tw/main/2015/08/17/%E7%B6%A0%E9%BB%A8%E3%80%81%E7%A4%BE%E6%B0%91%E9%BB%A8%E7%B0%BD%E5%90%88%E4%BD%9C%E5%82%99%E5%BF%98%E9%8C%84/|archive-date=2016-05-03|dead-url=yes}}</ref>。

2016年1月16日，[[2016年中華民國立法委員選舉|2016年立委選舉]]投票，范雲在開票時間說綠社盟等許多新興政黨會跨過選舉門檻<ref>{{Cite web|url=http://www.appledaily.com.tw/realtimenews/article/new/20160116/776567|author=沈能元|language=zh-tw|title=周子瑜事件影響 范雲：國民黨將繼續挫敗|publisher=台灣蘋果日報即時新聞|date=2016-01-16|accessdate=2016-01-17}}</ref>，結果是綠社盟全軍覆沒，雖然得票退居第六，不過與上屆僅綠黨不分區得票相比，投票率降低卻成長了近八萬票，甚至以不到兩千五百票之差超越上屆跨越5%門檻的[[臺灣團結聯盟|臺灣團結聯盟]]；在區域立委候選人的得票也大有長進，尤其[[臺北市|臺北市]]及[[新北市|新北市]]的部分候選人得票超過10%，絕大多數選將都有4%至5%以上的得票；下屆立委選舉起，因政黨得票超越2%，綠社盟不須提名10席區域立委候選人，便可直接提出不分區立委名單。同日晚間，李根政在綠社盟記者會表示，綠社盟確實得票不如預期，但不會停止參政，會強力監督[[蔡英文政府|蔡英文政府]]的施政，「就像2000年到2008年，我們監督民進黨的力道也沒有鬆懈過……現在喊『國民黨不倒、台灣不會好』的人，很多人沒經歷過[[陳水扁政府|民進黨執政期間]]；我們跟他們（民進黨）近身肉搏了八年，（瞭解）民進黨財團化的問題非常嚴重，不會樂觀地以為蔡英文執政就可解決問題」<ref>{{Cite web|url=http://e-info.org.tw/node/112814|author=陳文姿、詹嘉紋、洪郁婷、彭瑞祥|language=zh-tw|title=從22萬到38萬 環保政黨票持續成長 席次仍掛零|publisher=[[台灣環境資訊協會|台灣環境資訊協會]]環境資訊中心|date=2016-01-17|accessdate=2016-01-17}}</ref>。

2016年1月18日，李根政、張育憬與范雲為綠社盟選舉失利負起責任，相繼宣布辭職；范雲表示，她將繼續協助社民黨在地方扎根；李根政與張育憬表示，綠黨不會放棄社運參政之路、也不會忘記初衷<ref>{{Cite web|url=http://www.appledaily.com.tw/realtimenews/article/new/20160118/777902|author=林靖堂|language=zh-tw|title=為敗選負責 范雲等綠社盟召集人皆辭職|publisher=[[台灣蘋果日報|台灣蘋果日報]]即時新聞|date=2016-01-18|accessdate=2016-01-19}}</ref>。2016年1月22日，范雲承認，「我的確把事情想得太單純了」，社民黨與綠黨對民進黨的態度不完全一樣，社民黨在[[2016年中華民國總統選舉|2016年總統選舉]]中支持蔡英文而引發綠黨不滿，「可能綠黨部分朋友對民進黨痛恨的程度比我多吧」<ref>{{cite news|language=zh-tw|author=方德琳|title=專訪范雲：社民黨不會變泡沫|url=https://www.twreporter.org/a/2016election-thirdparty-fan|publisher=[[報導者_(臺灣媒體)|報導者]]|date=2016-01-22|accessdate=2017-03-25}}</ref>。

2016年1月29日，社民黨全國委員會選出陳尚志擔任代理召集人；社民黨秘書長[[葉虹靈|葉虹靈]]說，社民黨與綠黨討論後，預計[[農曆年|農曆年]]後將公布綠社盟2016年選舉檢討報告<ref>{{Cite web|url=http://www.ettoday.net/news/20160129/640256.htm|author=楊佳穎|language=zh-tw|title=范雲請辭！陳尚志接社民黨代理召集人 苗博雅任發言人|publisher=ETtoday新聞雲|date=2016-01-29|accessdate=2016-04-17}}</ref>。

2016年7月1日，曾為綠社盟站台助講的[[第三社會黨|第三社會黨]]發起人[[周奕成|周奕成]]表示，「如果要幫台灣培養一個未來的反對黨，理想上，社會基礎和民進黨分開的綠社盟比較可能」；綠社盟是綠黨與社民黨的結合，反而導致兩黨互相牽制（例如社民黨無法施展綠黨不同意的事），結果是甩不開社運界的[[意識形態|意識形態]]包袱及隨之而來的過度[[潔癖|潔癖]]以致寸步難行，「如果當初不結合，我覺得會比較好」；綠社盟花費太多力氣講述個別政策，例如長照、社福或地方環境問題，「小黨提了一百分的政見，但沒有能力去實現它；民進黨提的政見只有六十分，卻有能力百分之百做到」<ref>{{cite news|language=zh-tw|author=房慧真|title=周奕成：台灣需要第三社會|url=https://www.twreporter.org/a/2016election-thirdparty-chou|publisher=報導者|date=2016-07-01|accessdate=2017-03-25}}</ref>。

2018年5月12日，綠黨與[[樹黨|樹黨]]於[[新竹縣|新竹縣]]與[[新竹市|新竹市]]成立「綠樹雙結盟」。

2018年7月19日，綠黨、社民黨與[[基進黨|基進黨]]成立「社會福利國家連線」。<ref>{{Cite web |url = https://udn.com/news/story/6656/3261000 |title = 不滿時代力量 社會福利國家連線拚「五市五黨團」 |author = 劉宛琳 |date =2018-07-19  |publisher =聯合報 |language = zh-hant |accessdate = 2018年10月30日 |quote = }}</ref><ref>{{Cite web |url = https://www.storm.mg/article/464995 |title =「若不做第三勢力整合很不好」 社會福利國家連線盼與時力合作 |author = 林瑋豐 |date =2018-07-19 |publisher =風傳媒 |language = zh-hant |accessdate = 2018年10月30日 |quote = }}</ref>由於基進黨的右派立場與其他兩黨的左派立場差異甚大，綠黨、社民黨黨員及支持者皆對此產生疑慮。<ref>{{Cite web |url = https://www.cmmedia.com.tw/home/articles/10474 |title = 第三勢力小黨組策略聯盟 只是形式大於實質？ |author = 孫偉倫 |date = 2018-06-20 |publisher = [[信傳媒|信傳媒]] }}</ref><ref>{{Cite web |url = https://www.socialism.org.tw/2018/07/06/tp0706/ |title = 風雨欲來－台灣國內外的資本主義危機將至 |author = [[國際社會主義前進|國際社會主義前進]] |date = 2018-07-06 |quote = 基進黨是一個具右翼民粹傾向的政黨，過往不時發表排外言論（例如將反中共與反中國人混雜起來，以「中國白蟻」這種國族歧視謔稱中國人），也支持不少新自由主義政策。基進黨這種民粹組織沒有一貫的立場，立場往往隨著民情和局勢搖擺，現在為了選票而打出左傾福利綱領。}}</ref>

2018年12月2日，[[左翼聯盟_(臺灣)|左翼聯盟]]秘書長[[黃德北|黃德北]]表示，綠黨在台灣第三勢力政黨之中比較接近歐洲的左翼綠黨，但2016年大選失利後許多優秀幹部淡出政治，現任領導人高度向民進黨靠攏、處處為民進黨的政策辯護；社民黨更接近歐洲的自由主義政黨，更重視民主、人權、自由等價值，較少關懷勞動與階級議題，與歐洲的社會民主黨性質上差異頗大<ref>{{Cite web|url=https://www.thenewslens.com/feature/2018-2020/109299|author=黃德北|language=zh-tw|title=台灣選舉只有右派價值的藍綠輪替，而主要的第三勢力都不算是左翼政黨|work=[[關鍵評論網|關鍵評論網]]|publisher=|date=2018-12-02|accessdate=2019-01-09}}</ref>。

==主張==
綠黨社會民主黨聯盟共同政見，是以推動台灣經濟、社會、政治之改革，並強調建立生態、永續、自由、平等、團結等社會進步價值等，為綠色社會民主理想為宗旨。

綠黨社會民主黨聯盟推動[[LGBT權益|LGBT權益]]，[[多元性別|多元性別]]主流化，反對[[歧視|歧視]]，尊重差異，強化性別平等教育，確保[[跨性別|跨性別]]之[[性別認同|性別認同]]及醫療權益，推動[[婚姻平權|婚姻平權]]制度。<ref>{{cite news|title=綠黨社會民主黨聯盟官方網站|url=http://www.greensdparty.org.tw|deadurl=yes|archiveurl=https://web.archive.org/web/20151120101137/http://www.greensdparty.org.tw/|archivedate=2015-11-20}}</ref>
