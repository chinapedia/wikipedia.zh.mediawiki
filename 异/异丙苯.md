'''异丙苯'''（[[化学式|化学式]]：C<sub>9</sub>H<sub>12</sub>），俗称'''枯烯'''、'''𬜢'''<ref>{{Cite web |url = http://terms.naer.edu.tw/terms/manager_admin/new_file_download.php?Pact=FileDownLoad&button_num=g5&source_id=160&Pval=68 |title = 化學命名原則第四版較第三版新增部分 |language = zh-tw |accessdate = 2018-09-09 }}</ref>（Cumene），是难溶于水的无色液体。可溶于[[乙醇|乙醇]]、[[乙醚|乙醚]]、[[苯|苯]]、[[四氯化碳|四氯化碳]]，存在于[[原油|原油]]中，具可燃性。

工业上用[[苯|苯]]与[[丙烯|丙烯]]发生[[傅-克反应|傅-克反应]]生产异丙苯，所用的酸性催化剂是载于[[氧化铝|氧化铝]]上的固体[[磷酸|磷酸]]催化剂。<ref>[http://www.the-innovation-group.com/ChemProfiles/Cumene.htm The Innovation Group website, page accessed 15/11/07]</ref> 用此法制得的异丙苯绝大部分都用于生产[[苯酚|苯酚]]、[[丙酮|丙酮]]，称为[[异丙苯法|异丙苯法]]，中间体是[[过氧化氢异丙苯|过氧化氢异丙苯]]。少量用于合成香料、聚合反应引发剂等。
