'''尚湖镇'''，中国[[浙江省|浙江省]][[金华市|金华市]][[磐安县|磐安县]]下辖的一个[[镇|镇]]。总面积106.42平方公里，人口1.8万。<ref name="gov">[http://www.panan.gov.cn/shz/jgxx_2820/xzgk_2821/ 乡镇概况—尚湖镇] {{Wayback|url=http://www.panan.gov.cn/shz/jgxx_2820/xzgk_2821/ |date=20170330014328 }} 磐安县人民政府门户网站</ref>

原名尚坞，后改为尚湖。1950年11月由九和乡分设尚湖乡<ref name="gov" />，1956年为尚湖乡，1961年改为尚湖公社，1983年复改为尚湖乡，1986年为尚湖镇。<ref name="gov" />。产业有茶叶、香菇、药材、纺织、机械等。<ref>{{cite book|editor=戴均良 |title=中国古今地名大词典 |year=2005 |publisher=上海辞书出版社 |isbn= 9787532617432 |pages=1767}}</ref>

[[曹娥江|曹娥江]]发源于尚湖镇王村的大盘水脉长坞。<ref>[http://td.sx.gov.cn/art/2016/8/30/art_26372_1002821.html 水文] {{Wayback|url=http://td.sx.gov.cn/art/2016/8/30/art_26372_1002821.html |date=20170329234908 }} 绍兴市国土资源局</ref>[[东阳江|东阳江]]（[[金华江|金华江]]支流）的主流北江发源于尚湖镇的龙乌尖。<ref>[http://www.jhszb.cn/newsitem/277368477 水系] {{webarchive|url=https://web.archive.org/web/20170329234559/http://www.jhszb.cn/newsitem/277368477 |date=2017-03-29 }} 金华市地方志编纂委员会办公室</ref>
