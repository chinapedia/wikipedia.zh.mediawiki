'''尚方劍'''，又稱'''尚方寶劍'''，中國古代寶劍，由少府尚方監鑄造，供[[天子|天子]]及皇室使用，可用以代表[[皇帝|皇帝]]旨意，是一種權力和榮譽的象徵。亦稱為'''勢劍'''<ref>{{Cite web|url=http://www.guoxuedashi.com/kangxi/pic.php?f=dcd&p=2560|title=《汉语大词典》第2560页 第2卷 816}}</ref>，[[大臣|大臣]]官員若持有皇帝御賜的尚方寶劍不但代表著大臣獲得皇帝寵信，也是大臣官員的殊榮。這也可以被作為一種軍事授權，是由皇帝賜與武將，給與戰場上可以直接誅殺低層軍官及士兵，不用特別上奏[[朝廷|朝廷]]，俗稱「先斬後奏」。

== 簡介 ==
起源於[[漢朝|漢朝]]，尚方寶劍，意指由[[少府|少府]]尚方令鑄造，供皇室使用的刀劍。相傳尚方令在[[月蝕|月蝕]]時鑄劍，以供天子使用<ref>[[宋朝|宋]]薛季宣《浪語集》卷31〈記漢尚方劍〉：「尚方，漢工官也。漢法，常以月食時鑄劍，供天子服用。」</ref>。當時稱[[斬馬劍|斬馬劍]]。《漢書》中有[[朱雲|朱雲]]向[[漢成帝|漢成帝]]請斬馬劍，以斬[[張禹_(西汉)|張禹]]的故事<ref >《漢書》卷67〈楊胡朱梅雲列傳〉：「至成帝時，丞相故安昌侯張禹以帝師位特進，甚尊重。雲上書求見，公卿在前。雲曰：『今朝廷大臣上不能匡主，下亡以益民，皆尸位素餐，孔子所謂「鄙夫不可與事君」，「苟患失之，亡所不至」者也。臣願賜尚方斬馬劍，斷佞臣一人以厲其餘。』上問：『誰也？』對曰：『安昌侯張禹。』」顏師古註：「尚方，少府之屬官也，作供御器物，故有斬馬劍，劍利可以斬馬也。」</ref>。之後成為典故，文臣在文章或詩詞中，常以尚方劍作為一種象徵，代表自己獲得皇帝親信，可以逕行誅殺不良官員。如明朝[[劉伯溫|劉伯溫]]《贈周宗道六十四韻》：「先封尚方劍，按法誅奸贓。」

在明朝時，尚方寶劍是由皇帝賜與臣子的劍，擁有此劍的臣子，可以代表皇帝，直接誅殺官員以至百姓，而不需要經三法司會審。通常這都是由皇帝賜與武將，在戰場中使用，用來整肅[[軍紀|軍紀]]。[[明朝|明]]末的[[薊州|蓟]][[遼寧|辽]][[督師|督師]][[袁崇煥|袁崇煥]]即是用御賜尚方寶劍，將[[皮島|東江]][[总兵|总兵]][[毛文龍|毛文龍]][[斬首|斬首]]，然事實上毛文龍本人也有尚方寶劍，明代記載尚方劍僅能斬[[監司|監司]]、[[副將|副將]]以下官員，並不能斬殺[[總兵官|總兵官]]<ref>《明史·杨嗣昌列传》：巡抚不用命，立解其兵柄，简一监司代之。总兵不用命，立夺其帅印，简一副将代之。监司、副将以下，悉以尚方剑从事。</ref>。但明末[[袁崇煥|袁崇煥]]也能據此劍殺東江鎮總兵毛文龍，雖然這也引起了[[崇禎帝|崇禎帝]]不滿，而成為袁日後被[[處決|處決]]的原因之一。尚方劍雖然只有一把，但受賜的官員可以將其生殺大權分配屬下，例如[[播州之役|播州之役]]中，李化龍請得[[萬曆帝|萬曆帝]]之尚方寶劍後，給屬下總兵及監軍各發一劍，囑咐他們也可以用收到的劍去行使尚方寶劍同樣權力。明中葉後，尚方寶劍大量賜與官員，甚至有流落外地而朝廷不知的情況。

清代主要以沿用自明代的王命旗牌，行類似尚方寶劍的權力，但先斬後奏大權每面旗牌只限殺一人，不像尚方寶劍般不限人數，亦不能故意繞過三法司而濫用。但尚方寶劍形式的信物在清朝仍有出現，最著名的是遏必隆刀，此刀為清初重臣[[遏必隆|遏必隆]]之腰刀，在其死後收入皇宮，因人而名。[[乾隆帝|乾隆帝]]曾將此刀賜经略大学士傅恒征大金川，以其斬作戰不力之[[訥親|訥親]](巧合地訥親正是遏必隆之孫)、另外大学士赛尚阿讨太平天国時，[[咸丰帝|咸丰帝]]亦赐遏必隆刀以壯軍威，兵敗而轉賜[[徐廣縉|徐廣縉]]。此外在太平天國時期，咸豐帝分別賜銳捷刀<ref>《清史稿》本紀二十、列傳八</ref>、訥庫尼素刀<ref>《清史稿》本紀二十、志六十五</ref>、神雀刀<ref>《清史稿》本紀二十、列傳一百九十、二百零九</ref>於惠親王[[綿愉|綿愉]]、科爾沁親王[[僧格林沁|僧格林沁]]、欽差大臣[[勝保|勝保]]，都有先斬後奏之權。此外，[[道光帝|道光帝]]在世時，亦曾賜當時仍是皇子之恭親王[[奕訢|奕訢]]白虹刀<ref>《清史稿》列傳八</ref>，此刀傳至其孫[[溥偉|溥偉]]，欲以此刀斬[[袁世凱|袁世凱]]為[[光緒帝|光緒帝]]報仇，以其有先斬後奏之權，[[慈禧|慈禧]]亦無法禁止。<ref>{{Cite web|url=http://blog.sina.com.cn/s/blog_470bf257010008nr.html|title=《三聯生活周刊》2006年第3期 毓嶦:我所知道的溥仪}}</ref>

此外，清朝仍有賜正式尚方寶劍的紀錄，為光緒三十二年(1906年)由慈禧太后下旨賜與主持修建[[新寧鐵路|新寧鐵路]]的旅美華僑陳宜禧，准其先斬後奏，斬殺任何阻撓修建鐵路的村民。此劍據聞在中共[[土地改革运动|土地改革]]時流失，後來由廣東省博物館收藏。<ref>{{Cite web|url=http://www.chens.org.cn/sept2009/Article/ShowArticle.asp?ArticleID=2122|title=陳氏宗親網：名人介紹—陳宜禧|access-date=2017-08-08|archive-url=https://web.archive.org/web/20170808115259/http://www.chens.org.cn/sept2009/Article/ShowArticle.asp?ArticleID=2122|archive-date=2017-08-08|dead-url=yes}}</ref><ref>{{Cite web|url=http://www.tszhengxie.com/Disp.Aspx?ID=1290&ClassID=10|title=台山政協 文史資料：陈宜禧：台山人精神的典型代表}}</ref>

[[中國文學|中國文學]]中，傳統[[戲曲|戲曲]]或[[小說|小說]]中常可見持有御賜「勢劍」的大臣，有“先斬後奏”之權，可以先將造成重大危害之人處死，再稟報[[朝廷|朝廷]]。除了“势剑”外，还有「[[金牌|金牌]]」，金牌乃「皇帝頒發，可奉旨便利行事的信物」，稱“勢劍金牌”，在[[元代|元代]]戲曲小說中時常出現。如[[孫仲章|孫仲章]]《河南府张鼎勘头巾》的王小二就被手持“势剑金牌”的[[府尹|府尹]]下令斩首，幸好被一位爱管闲事的官员所救。<ref>{{wikisource-inline|河南府張鼎勘頭巾}}</ref>

== 參見 ==
* [[符节|符节]]：为史书更多记载的授权形式，持节者有先斩后奏的生杀之权。
* [[丹書鐵券|丹書鐵券]]：是[[天子|天子]]頒發給大臣的一種獎賞性憑証，有的甚至可以抵免[[死刑|死刑]]，稱免死金牌。
* [[黃馬褂|黃馬褂]]：只有[[皇帝|皇帝]]近身的侍衛[[武士|武士]]，或獲[[朝廷|朝廷]]特別賞賜者才可以穿著的一種榮譽服飾。
* 金[[鐧|鐧]]、金[[鞭|鞭]]、龍頭拐杖：傳說中「上打昏君、下打讒臣」的名器。
* 《[[九品芝麻官|九品芝麻官]]》：喜剧電影，[[清朝|清朝]][[巡按|巡按]]包龍星（[[周星馳|周星馳]]飾）持尚方寶劍欲殺犯人，卻被揭發是[[明朝|明朝]]末代皇帝[[崇禎帝|崇禎]]賞賜的遺物。而電影裡的背景時代是清朝[[同治帝|同治]]年間。尚方寶劍早已失效。
* 《[[鹿鼎記2神龍教|鹿鼎記2神龍教]]》：喜剧電影，公主護衛大臣多隆（[[陳百祥|陳百祥]]飾），欲以「上斬昏君、下斬讒臣」的尚方寶劍教訓平西王[[吳三桂|吳三桂]]（[[秦沛|秦沛]]饰），王爺以先帝御賜的「免死金牌」對抗之。
* [[代天巡狩|代天巡狩]]
