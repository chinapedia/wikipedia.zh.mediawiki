'''埃里克·奧洛夫·梅尔贝里'''（{{lang-sv|'''Erik Olof Mellberg'''}}，{{IPA-all|ˈuːˌlɔf ˈmɛlˌbærj}}，{{bd|1977年|9月3日|catIDx=Mellberg, Olof}}）是一名[[瑞典|瑞典]]足球運動員，司職[[后卫_(足球)|中后卫]]。他在2003年被评选为{{Link-sv|瑞典足球先生|Guldbollen}}。<ref>{{cite web |url=http://svenskfotboll.se/arkiv/tidigare/2004/02/guldbollen-2003-olof-mellberg/ |title=Guldbollen 2003 - Olof Mellberg |publisher=瑞典足球协会 |date= |accessdate=2009年8月15日 |language=sv |archive-url=https://web.archive.org/web/20090701093222/http://svenskfotboll.se/arkiv/tidigare/2004/02/guldbollen-2003-olof-mellberg/ |archive-date=2009-07-01 |dead-url=yes }}</ref>

== 俱乐部 ==
梅尔贝里年少时效力于家乡的古尔斯蓬队，1996年被[[瑞典足球超级联赛|瑞典足球超级联赛]]的{{Link-sv|代格福什IF|Degerfors IF}}选中。1997年，代格福什降级后，梅尔贝里转会至[[AIK索尔纳|AIK索尔纳]]。但他在AIK索尔纳也仅仅效力了10个月，之后便加盟[[西甲|西甲]]球队[[桑坦德竞技|桑坦德竞技]]。尽管刚开始并不十分适应[[西班牙|西班牙]]足球，他还是在第一个赛季就进入主力阵容。

伴随着梅尔贝里在桑坦德竞技的出色表现，外界不断传出他将转会西甲强队的消息，涉及到的球队包括[[巴塞罗那足球俱乐部|巴塞罗那]]和[[瓦伦西亚足球俱乐部|瓦伦西亚]]。然而，他最终在2001年转会的球队却是[[英超|英超]]的[[阿斯顿维拉足球俱乐部|阿斯顿维拉]]，转会费为500万[[英镑|英镑]]。<ref>{{cite web |url=http://news.bbc.co.uk/sport2/hi/football/teams/a/aston_villa/1444827.stm |title=Mellberg joins Villa |publisher=[[BBC|BBC Sport]] |date=2001年7月18日 |accessdate=2009年8月15日 |language=en }}</ref>在阿斯顿维拉期间，他曾长期担任[[队长_(足球)|队长]]，直至[[2006年世界杯足球赛|2006年世界杯足球赛]]后辞去队长职务。2006年8月19日，阿斯顿维拉客场挑战[[阿森纳足球俱乐部|阿森纳]]。这是[[酋长球场|酋长球场]]举行的首场竞赛性质的比赛及首场英超联赛。梅尔贝里在比赛中率先进球，成为在这座球场打进第一个英超进球的球员，并帮助本队与阿森纳战成1比1。<ref>{{cite web |url=http://sports.sina.com.cn/g/2006-08-21/16062410266.shtml |title=新鲜空气吹过酋长球场上空 阿森纳战平温格看到新希望 |author=姜轶 |publisher=[[新浪网|新浪网]]（转自《足球-劲体育》） |date=2006年8月21日 |accessdate=2009年8月15日 |language=zh }}</ref>2007－2008赛季，阿斯顿维拉引进[[扎特·奈特|扎特·奈特]]，与[[马丁·劳尔森|马丁·劳尔森]]构成中后卫组成。梅尔贝里因此改打右后卫。在2008年3月15日与[[朴茨茅斯足球俱乐部|朴茨茅斯]]的比赛中，梅尔贝里因累积两张黄牌被罚下场。

[[File:Rio_Juventus.jpg|thumb]]比赛中]]
2008年初，梅尔贝里与[[尤文图斯足球俱乐部|尤文图斯]]达成初步协议，定于赛季结束后根据[[博斯曼法案|博斯曼法案]]、以自由球员的身份加盟尤文图斯。5月3日，梅尔贝里迎来自己在阿斯顿维拉的最后一场主场比赛，对阵[[威根竞技足球俱乐部|威根竞技]]。为了感谢他对球队的贡献，阿斯顿维拉将这个比赛日定为“梅尔贝里日”。而客场对阵[[西汉姆联足球俱乐部|西汉姆联]]的比赛则成为他在阿斯顿维拉的告别赛。梅尔贝里向每位到[[厄普顿公园球场|厄普顿公园球场]]观看比赛的阿斯顿维拉球迷赠送一件印有他的姓名和号码、以及“Thanks 4 Your Support”字样的阿斯顿维拉主场或客场队服。他还表示，他将把自己在这场告别赛上所穿的队服装裱起来。

2008年夏天，梅尔贝里如期转会尤文图斯。他在7月15日与[[皮亚琴察足球俱乐部|皮亚琴察]]的[[友谊赛|友谊赛]]上首次代表新球队出场。<ref>{{cite web |url=http://sports.sina.com.cn/g/p/2008-07-16/07573786251.shtml |title=[热身赛]尤文0-1皮亚琴察 梅尔贝里防守稳健 |publisher=[[新浪网|新浪体育]] |date=2008年7月16日 |accessdate=2009年8月15日 |language=zh }}</ref>2009年1月18日，梅尔贝里在与[[拉齐奥足球俱乐部|拉齐奥]]的比赛中首次为尤文图斯进球。<ref>{{cite web |url=http://news.xinhuanet.com/sports/2009-01/19/content_10681333.htm |title=意甲-铁卫处子球 尤文1-1平拉齐奥落后国米3分 |publisher=[[新华网|新华网]]（转自[[搜狐网|搜狐网]]） |date=2009年1月19日 |accessdate=2009年8月15日 |language=zh }}</ref>

2009年6月23日，梅尔贝里以250万[[欧元|欧元]]的身价转会奥林匹亚科斯，签订了为期三年的合同。<ref>{{cite web|url=http://www.olympiacos.org/index.html?lang=en#/Article/26001/ |title=Press Release |publisher=[[奥林匹亚科斯足球俱乐部|奥林匹亚科斯足球俱乐部]] |date=2009年6月23日 |accessdate=2009年8月15日 |language=en |deadurl=yes |archiveurl=https://web.archive.org/web/20090629194254/http://www.olympiacos.org/index.html?lang=en |archivedate=2009年6月29日 }}</ref><ref name="titan">{{cite web |url=http://soccer.titan24.com/2009-06-23/32300.html |title=尤文送别防线替补老将 梅尔贝里正式加盟希腊豪门 |author=慕捷 |publisher=[[体坛网|体坛网]] |date=2009年6月23日 |accessdate=2009年8月15日 |language=zh }}</ref>如果奥林匹亚科斯获得下一赛季希腊超级联赛冠军冠军，他们将再支付50万欧元给尤文图斯。<ref name="titan"/>

== 国家队 ==
[[File:Olof_Mellberg_070822.jpg|thumb]]
梅尔贝里自2000年开始为[[瑞典国家足球队|瑞典国家队]]效力。他曾代表瑞典队参加过[[2000年欧洲足球锦标赛|2000年]]、[[2004年欧洲足球锦标赛|2004年]]、[[2008年欧洲足球锦标赛|2008年]]和[[2012年欧洲足球锦标赛|2012年]]四届[[欧洲足球锦标赛|欧洲足球锦标赛]]，以及[[2002年世界杯足球赛|2002年]]和[[2006年世界杯足球赛|2006年]]两届[[世界杯足球赛|世界杯足球赛]]。在其国家队生涯中，梅尔贝里共出场117次，打进8球。

2002年世界杯前，在瑞典队的一次公开训练上，梅尔贝里凶狠的铲球引起队友[[弗雷德里克·永贝里|永贝里]]的不满。双方便发生争吵，扭打在一起，后被队友拉开。<ref>{{cite web |url=http://news.bbc.co.uk/hi/chinese/news/newsid_2008000/20082712.stm |title=從永貝里打架看世界杯壓力 |author=自平 |publisher=[[BBC|BBC]] |date=2002年5月25日 |accessdate=2009年8月15日 |language=zh }}</ref><ref name="cctv">{{cite web |url=http://www.cctv.com/news/sports/worldcup/20060614/100446.shtml |title=瑞典更衣室内讧 永贝里与梅尔贝里大打出手 |author=毛小东 |publisher=[[中国中央电视台|中国中央电视台]]（转自《[[新京报|新京报]]》） |date=2006年6月14日 |accessdate=2009年8月15日 |language=zh }}</ref>从此，梅尔贝里与永贝里一直关系不佳。2006年世界杯，瑞典与[[特立尼达和多巴哥国家足球队|-{zh-hans:特立尼达和多巴哥;zh-hk:千里達;zh-tw:千里達及托巴哥;}-]]战平之后，这两名球员在更衣室里再次发生激烈争吵及斗殴。<ref name="cctv"/>

2006年9月，瑞典与[[列支敦士登国家足球队|列支敦士登]]的欧锦赛预选赛前，梅尔贝里与[[兹拉坦·伊布拉希莫维奇|伊布拉希莫维奇]]、[[克里斯蒂安·威廉松|威廉松]]因违反队规，未在晚上11点之前归队，被开除出队。<ref>{{cite web |url=http://news.cctv.com/sports/football/integrate/20060906/102133.shtml |title=备战欧洲杯预选赛 伊布梅尔贝里晚归被除名 |publisher=[[中国中央电视台|中国中央电视台]]（转自《京华时报》） |date=2006年9月6日 |accessdate=2009年8月15日 |language=zh }}</ref>10月7日，梅尔贝里重新回到队中，参加了与[[西班牙国家足球队|西班牙]]的比赛，并帮助球队以2比0取胜。<ref>{{cite web |url=http://www1.uefa.com/tournament/qualifying/matches/match=83714/index.html |title=Sweden too strong for sorry Spain |publisher=[[UEFA|UEFA]] |date=2006年10月7日 |accessdate=2009年8月15日 |language=en |deadurl=yes |archiveurl=https://web.archive.org/web/20091001071203/http://www1.uefa.com/tournament/qualifying/matches/match%3D83714/index.html |archivedate=2009年10月1日 }}</ref>
