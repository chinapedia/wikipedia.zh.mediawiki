'''奥沙唑仑'''（{{lang|en|Oxazolam}}）为一种[[神经系统|神经系统]]用药、镇静[[催眠药|催眠药]]、抗焦虑药物<ref name="MED126">{{cite web | url=http://www.med126.com/pharm/2009/20090107123454_48584.shtml | title=奥沙唑仑 | publisher=医学全在线 | date=2009 | accessdate=2011-09-27 | author=医药数据库中心 | deadurl=yes | archiveurl=https://web.archive.org/web/20101005065451/http://www.med126.com/pharm/2009/20090107123454_48584.shtml | archivedate=2010-10-05 }}</ref>。此药是[[地西泮|地西泮]]、氯氮的主要代谢产物。他的药理作用和[[地西泮|地西泮]]、氯氮类似但弱些，属于短到中效[[苯二氮平类药物|苯二氮平类药物]]。市面上的主要规格为胶囊10毫克、片剂（5mg、10mg、20mg）、10%散剂<ref name="honghaizi">{{cite web| url=http://www.med126.com/pharm/2011/20110222090004_432495.shtml| title=奥沙唑仑| publisher=医学全在线| accessdate=2011-09-27}}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

== 药理与药动 ==
奧沙唑侖是苯二氮卓类药，主要作用在[[杏仁核|杏仁核]]、[[下丘脑|下丘脑]]的大脑边缘系统，对[[大脑皮质|大脑皮质]]的激活系统无影响。有和[[地西泮|地西泮]]类同的安静平稳功效和化学结构，作用强但毒性低，對催眠、肌松、步履失调等一些的抑制作用小。[[动物实验|动物实验]]方面大鼠有躯体依赖，对狗有躯体依赖但依赖程度低于地西泮<ref name="MED126"/>。

口服快速吸收，4个小時裡血药浓度达顶峰，[[半衰期|半衰期]]是六到二十四小時，持续长度最少48小时。分布在各脏器，其中[[肝脏|肝脏]]和[[肾脏|肾脏]]分布数达总量50%以上。代谢产物是[[N-去甲安定|N-去甲安定]]与[[二苯甲酮|二苯甲酮]]类结合物，二苯甲酮类结合物从排尿清除，N-去甲安定的半衰期是61小时<ref name="zysj">{{cite web | url=http://www.zysj.com.cn/lilunshuji/jingshenyaopinlinchuangyingyong/1019-6-9.html | title=奥沙西泮　Oxazepam | publisher=中医世家 | accessdate=2011-09-27}}</ref>。

== 适用范围 ==
用作治疗焦虑、紧张、抑郁等一些情感障碍，还可用在神经症、植物神经症状、心身疾病和小儿癫痫等，也用作麻醉前用药。但大剂量持续服用不可超过2周，用药期间不适合饮酒。
外国的1548例报道证明神经症有效率是70.2％，植物神经功能失调是77.3％，心身疾病是85.7％，麻醉前给药是72.9％<ref name="MED126"/>。
==用法及用量==
若用于抗焦虑则口服每天3次，每顿10～20mg。麻醉前给药一般于进行手术前1～2毫克每千克。年长者和体质差者抗焦虑时起始剂量是7.5毫克1天三顿，可逐渐根据实情增至15mg1日三到四顿<ref name="MED126"/>。吸烟的人与不吸烟者相比使用剂量要比较多<ref name="zysj"/>。和中枢抑制药、单胺氧化酶抑制剂一起用或喝酒能增强此药作用，所以应该停服本品或减少用量，和西咪替丁一起用会影响清除率。长期用此药会产生依赖性。

== 副作用 ==
比较少但常见的有嗜睡、易疲倦、视力模糊、眩晕、低血压、心动过速、步履瞒珊、胃部不适和皮疹、恶梦少见共济失调，偶而会有白细胞减少、肌张力低下和药物依赖。
== 注意事项 ==
有药物滥用或成瘾史、肝肾功能损害者、ADHD病患、低蛋白血症、严重慢性阻塞性肺病、重症肌无力、同时使用神经系统药物、闭角型[[青光眼|青光眼]]的人应该慎用<ref name="zysj" />。

而需要禁用的人为使用此药过敏、睡眠呼吸暂停综合征、[[怀孕|怀孕]]早期孕妇、严重呼吸功能不全、急性闭角型青光眼<ref name="zysj" />。

用药这段时间或及用药之後不能开车和操作机器<ref name="zysj" />。
