'''奥兰多桑福德国际机场'''（{{airport codes|SFB|KSFB|SFB}}）（[[英文|英文]]：'''{{lang|en|Orlando Sanford International Airport}}'''）是[[美国|美国]][[佛罗里达州|佛罗里达州]][[桑福德_(佛罗里达州)|桑福德]]的[[国际机场|国际]][[机场|机场]]，也兼为同州的[[奥兰多_(佛罗里达州)|奥兰多]]服务。

== 设施 ==
奥兰多桑福德国际机场占地{{convert|2010|acre|ha|0|lk=on}}，并建有四条[[跑道|跑道]]：
* 跑道9L/27R: 9,600 x 150 英尺. (2,926 x 46[[米|米]]), 表面: [[沥青|沥青]]
* 跑道9C/27C: 3,578 x 75 英尺. (1,091 x 23米), 表面: 沥青
* 跑道9R/27L: 6,647 x 75 英尺. (2,026 x 23米), 表面: 沥青
* 跑道18/36: 6,002 x 150 英尺. (1,829 x 46米), 表面: 沥青/[[混凝土|混凝土]]
