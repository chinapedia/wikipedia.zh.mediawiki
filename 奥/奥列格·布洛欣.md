'''奥列格·布洛欣'''（{{lang-uk|'''Олег Володимирович Блохін'''}}，{{bd|1952年|11月5日|catIdx=Блохін}}），是一名足球教练，具有乌克兰和[[俄罗斯|俄罗斯]]的[[混血儿|混合血统]]，现任[[乌克兰国家足球队|乌克兰国家足球队]]主教练。他退役之前曾是[[苏联国家足球队|苏联国家足球队]]的射手，是1975年[[金球奖_(足球)|欧洲金球奖]]（即“欧洲足球先生”）获得者。

== 运动员 ==
布洛欣球员生涯的绝大部分时间都为[[基辅迪纳摩足球俱乐部|基辅迪纳摩]]效力，是这支苏联冠军球队的领袖和头号射手，出场次数和进球数分别达到创纪录的432场和211球。他与基辅迪纳摩一起赢得了8次苏联冠军。1975年和1986年，布洛欣两次带领球队取得[[欧洲优胜者杯|欧洲优胜者杯]]冠军；他本人在两次决赛中各打进一球。

在苏联国家队中，布洛欣保持着最高的出场纪录（112场）和进球纪录（42球）。他随队参加了[[1982年世界杯|1982年]]和[[1986年世界杯|1986年]]两届[[世界杯|世界杯]]，并各打进一球。

1988年，布洛欣转会[[奥地利|奥地利]]的{{Link-en|施泰尔福尔韦茨|SK Vorwärts Steyr}}，成为苏联最早加盟国外球队的球员之一。一年后，他又转投[[塞浦路斯|塞浦路斯]]球队{{Link-en|利马索尔艾利斯|Aris Limassol F.C.}}，并于1990年在那里退役。

== 教练员 ==
结束球员生涯后，布洛欣随即在[[希腊|希腊]]开始了自己的教练生涯，1990年至1993年间执教[[奥林匹亚科斯足球俱乐部|奥林匹亚科斯]]。期间，他带队夺得两次[[希腊杯|希腊杯]]和一次[[希腊超级杯|希腊超级杯]]。之后的近10年时间里，他先后执教[[帕奥克足球俱乐部|PAOK]]、[[伊翁尼科斯足球俱乐部|伊翁尼科斯]]和[[雅典AEK足球俱乐部|雅典AEK]]等球队。

2003年9月，布洛欣成为[[乌克兰国家足球队|乌克兰国家队]]的主教练。在他的带领下，乌克兰打进[[2006年世界杯|2006年世界杯]]八强，在1/4决赛中败给当届冠军[[意大利国家足球队|意大利队]]。由于未能将球队带入[[2008年欧洲杯|2008年欧洲杯]]，布洛欣于2007年12月辞去乌克兰国家队主教练职务。<ref>{{cite web |url=http://www.chinanews.com.cn/ty/gjzq/news/2007/12-07/1098057.shtml |title=布洛欣为欧预赛失利负责 乌克兰教练组集体辞职 |author=宋宗利 |publisher=[[中国新闻网|中国新闻网]] |date=2007年12月7日 |accessdate=2009年8月1日 |language= }}</ref>

几天后，布洛欣出任[[莫斯科足球俱乐部|FC莫斯科]]的主教练。<ref>{{cite web |url=http://soccernet.espn.go.com/news/story?id=490600&cc=5739 |title=FK Moscow hire former Ukraine manager Blokhin |publisher=[[ESPN|ESPN]]soccernet |date=2007年12月14日 |accessdate=2009年8月1日 |language=en }}</ref>赛季结束后，球队名列第9，俱乐部方面宣布将布洛欣解雇。<ref>{{cite web |url=http://www.sports.ru/football/6293328.html |title=Блохин освобожден от обязанностей главного тренера «Москвы» |publisher=sports.ru |date=2008年11月26日 |accessdate=2009年8月1日 |language=ru }}</ref>此前布洛欣表示，如果他事先知道俱乐部方面会放走多名主力球员、同时又要求取得好成绩，那他当时绝不会与俱乐部签约。<ref>{{cite web |url=http://www.sports.ru/football/6206389.html |title=Олег Блохин: «Если бы знал, как будут делаться дела в команде, то весной бы не пришел в «Москву» |publisher=sports.ru |date=2008年11月16日 |accessdate=2009年8月1日 |language=ru }}</ref>也有观点认为，布洛欣在莫斯科失败的原因是与媒体关系欠佳，从而导致媒体持续对他进行攻击，影响了他在球队里的威信。<ref>{{cite web |url=http://www.sports.ru/football/6358902.html |title=Геннадий Литовченко: «Говорил Блохину, что его будут травить» |publisher=sports.ru |date=2008年12月4日 |accessdate=2009年8月1日 |language=ru }}</ref>

2011年4月，布洛欣再次出任乌克兰国家队主教练，带领球队备战由波兰和乌克兰联合主办的[[2012年欧洲杯|2012年欧洲杯]]。<ref>{{cite web |url=http://cn.uefa.com/uefaeuro/news/newsid=1622547.html |title=布洛欣重掌乌克兰教鞭 |publisher=[[UEFA|UEFA]] |date=2011年4月21日 |accessdate=2011年12月3日 |language= |deadurl=yes |archiveurl=https://archive.is/20120711084518/http://cn.uefa.com/uefaeuro/news/newsid=1622547.html |archivedate=2012年7月11日 }}</ref>

== 从政 ==
布洛欣退役后也曾步入政界。2002年，他成功连任乌克兰议会议员；同年10月，加入乌克兰联合社会民主党。近几年，由于专注于足球教练事业，布洛欣淡出政界。

== 家庭 ==
布洛欣曾与蘇聯/乌克兰著名[[艺术体操|艺术体操]]运动员伊琳娜·杰留金娜（Irina Deriugina）结婚，并育有一女，名为伊琳娜。但两人已于1990年代初[[离婚|离婚]]。
