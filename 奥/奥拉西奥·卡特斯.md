== 生平 ==
卡特斯的父親為[[西斯納|西斯納]]特許經營控股並控股的飛機製造公司擁有人，而奧拉西奧·卡特斯便在這般背景下被送往美國研究航空工程。在19歲時他在[[阿曼拜銀行|阿曼拜銀行]]（Banco Amambay）貨幣貨幣兌換業務並很快讓銀行價值快速增值，在接下來的幾年中卡特斯購買或者協助創立了25家公司，包括有巴拉圭最大的捲菸製造商[[塔巴克萊拉埃斯特|塔巴克萊拉埃斯特]]（Tabacalera del Este）和數家主要的果汁裝瓶公司<ref name = "英國廣播公司">{{Cite web |url= http://www.bbc.co.uk/news/world-latin-america-22247484 |title= Profile: Horacio Cartes |author=  |date= 2013年4月22日 |publisher= [[英國廣播公司|英國廣播公司]] |language= en |accessdate= 2013年4月22日 |quote=  }}</ref>。

== 政治生涯 ==
奧拉西奧·卡斯特一直到2008年以前並沒有接觸過政治領域，甚至也尚未登記成為有選舉權利之民眾。2009年時他加入立場右翼的[[紅黨_(巴拉圭)|紅黨]]，並且希望能夠稍微抑制逐漸左傾的拉丁美洲政局。儘管紅黨由於在1989年支持[[阿爾弗雷多·斯特羅斯納|阿爾弗雷多·斯特羅斯納]]軍事獨裁政權而不斷受到批判，然而卡特斯成功將其名聲塑造成可靠人物並且成為新一代紅黨代表人物。最後他的經濟實力促使得其贏得黨內初選，成為紅黨在[[2013年巴拉圭大選|2013年巴拉圭大選]]中所提名的候選人<ref name = "英國廣播公司"/>。

在其競選承諾中提到將會募集私人基金以升級巴拉圭基礎建設且讓公共設施更為現代化，之後透過吸引國際投資來創造就業機會。在競選期間卡特斯也提出數個激進關於同性戀者的話語，除了以猴子來跟同性戀者比較外，還表示如果自己兒子為同性戀的話將會拿槍射擊自己的睪丸<ref>{{Cite web |url= http://www.nytimes.com/2013/04/17/world/americas/candidate-in-paraguay-stirs-controversy-with-remarks-on-gay-marriage.html?_r=2& |title= Candidate Disparages Gays in Paraguay, Stirring Dispute |author= [[西蒙·羅梅羅|西蒙·羅梅羅]]（Simon Romero） |date= 2013年4月16日 |publisher= 《[[紐約時報|紐約時報]]》 |language= en |accessdate= 2013年4月22日 |quote=  }}</ref>。

2013年4月21日舉行的[[2013年巴拉圭大選|巴拉圭大選]]中，卡特斯最後以獲得1,089,247張票（佔全部45.81%）正式當選為[[巴拉圭總統|巴拉圭總統]]<ref name = "英國廣播公司"/>。
