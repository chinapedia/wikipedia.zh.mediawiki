'''奥本市'''<ref>{{cite web |url=http://www.auburn.nsw.gov.au/uploadedFiles/AuburnWeb/Development/Info%20brochure_Traditional%20Chinese.pdf |title=奧本地方環保計劃2009草案 |year=2009 |language=zh-hant }}</ref>（又譯'''欧本市'''；[[英文|英文]]：'''Auburn City'''）是[[澳大利亞|澳大利亞]][[雪梨|雪梨]]地區的一個[[地方政府區域|地方政府區域]]，地處[[雪梨中央商業區|雪梨中央商業區]]以西15公-{里}-。奥本市基本是城郊住宅區，也有較大規模的輕工業、服務業和商業。

20世紀後半葉，奥本開始成為[[移民|移民]]聚居區，開始有一波又一波來自各國的新移民搬入此地區，然後在該族群在澳洲扎根後　又逐漸搬出，被另一族群代替。根據2011年的人口普查，現在的歐本市居民中第一大族群（佔人口20.1%）為[[華裔|華裔]]。其他較大族群依次為：7.5%自認為是“澳大利亞裔”，7.0%為[[英格蘭|英裔]]，5.9%为[[土耳其|土耳其]]人，6.3%為[[黎巴嫩|黎巴嫩]]裔。由於过去相當長一段時期裏歐本的最大少数民族是土耳其人，因此虽然华裔已跃升第一大族群，在歐本街头仍随处可见土耳其式餐厅、特色商店。歐本最著名的地标之一加里波里清真寺的建築原型即是[[伊斯坦布尔|伊斯坦布尔]]的[[聖索菲亞大教堂|聖索菲亞大教堂]]。

轄境內的[[:en:Homebush_Bay,_New_South_Wales|康宝树湾]]是[[2000年夏季奥林匹克运动会|2000年雪梨奧運]]的主場地所在地。

== 议会 ==
奥本市议会有議席10人固，全市分2個選區，各區選出議員5名代表市民和納稅人問政；市長非直選。<ref name=EC>{{cite web|url = http://www.lg.elections.nsw.gov.au/LGE2008/result.Auburn.html|title = Auburn Council|work = 2008 Election results| publisher = Electoral Commission NSW| accessdate = 2009-06-15}}</ref>
