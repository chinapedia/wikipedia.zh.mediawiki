== 奖项 ==
* 1985年：金球奖最佳女主角奖、奥斯卡最佳女主角提名-《[[紫色姐妹花_(電影)|紫色姊妹花]]》
* 1990年：金球奖最佳女配角奖、奥斯卡最佳女配角奖、[[英国电影学院奖|英国电影学院奖]]最佳女配角奖-《[[人鬼情未了|人鬼情未了]]》

== 所出席典礼、节目 ==
她担任1994、1996、1999和2002年奥斯卡颁奖典礼主持人。在2009年[[第81届奥斯卡金像奖|第81届奥斯卡金像奖]]上，她与其他四位往届[[奧斯卡最佳女配角獎|最佳女配角]]上台，负责赞赏[[艾美·亞當斯|爱美雅当斯]]在《[[圣诉|圣诉]]》中的出色表现。
