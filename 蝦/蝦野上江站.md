'''蝦野上江站'''（{{jpn|j=えびの上江駅|kyu=|hg=えびのうわええき|hgt=|rm=Ebino-Uwae eki|f=y}}）位於[[宮崎縣|宮崎縣]][[蝦野市|蝦野市]]，是[[九州旅客鐵道|九州旅客鐵道]]（JR九州）
[[吉都線|吉都線]]上的車站。

== 車站構造 ==
岸式月台1面1線的地上站，沒有站房。有候車室，附近設有腳踏車停車場。

== 車站周邊 ==

* 蝦野市立上江小學
* 上江保育園
* 蝦野上江簡易郵局

== 历史 ==

* 1957年（昭和32年）7月5日 - 設立，當時稱為上江站。
* 1987年（昭和62年）4月1日 - [[國鐵分割民營化|國鐵分割民營化]]後成為九州旅客鐵道（JR九州）轄下的車站。
* 1990年（平成2年）11月1日 - 改稱為蝦野上江站
