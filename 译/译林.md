《'''译林'''》是一本[[中国|中国]]当代的[[翻译|翻译]][[文学|文学]][[杂志|期刊]]，由[[凤凰出版传媒集团|凤凰出版传媒集团]]的[[译林出版社|译林出版社]]于[[南京|南京]]出版，1979年10月1日创刊。杂志现为双月刊，16[[开本|开本]]版式，每期224页。《译林》每期的文学内容一般有一篇长篇小说，数篇短篇小说以及几篇诗歌、散文——这些均为流行的外文作品的中文翻译；另一方面，杂志还有关于外国文学的新闻、介绍、评论、当期作品的评析以及专题。

1979年创刊至1997年的20年中，《译林》为季刊，1997年改为双月刊<ref name="新华网">[[新华网|新华网]]，2003年9月26日：首期《译林》增刊9月下旬上市。{{cite web |url=http://news.xinhuanet.com/book/2003-09/26/content_1099706.htm |title=存档副本 |accessdate=2007-06-13 |deadurl=yes |archiveurl=https://web.archive.org/web/20050317191102/http://news.xinhuanet.com/book/2003-09/26/content_1099706.htm |archivedate=2005-03-17 }} </ref>。2003年9月，首次推出增刊<ref name="新华网"/>，其后，译林杂志社又陆续推出《译林》的衍生杂志产品：2005年起，《译林》每年有[[春季|春]]、[[秋季|秋]]两[[季节|季]]增刊；2006年底，出版《译林·文摘版》第1期的试刊，2007年2月正式出版第1期，杂志为双月刊。

《译林》的中国国内刊号为CN32-1029/I，国际刊号ISSN1001-1897。

== 创刊经过 ==
1979年改革初期，为解决书荒局面，[[江苏省出版局|江苏省出版局]]决定由[[江苏人民出版社|江苏人民出版社]]创办《译林》杂志，以介绍外国当代文学、特别是健康的通俗文学为重点，由[[李景端|李景端]]负责筹办。初版20万册，后加印20万册，收到1万多封读者来信，引起很大轰动。<ref>施亮，《译林》事件始末，炎黄春秋2008年第6期</ref>


== 风波 ==
1980年4月中旬，[[李景端|李景端]]听闻中央负责意识形态工作的领导人[[胡乔木|胡乔木]]对译林杂志提出点名批评，后得知是时任[[中国社会科学院|中国社会科学院]]外国文学研究所所长的[[冯至|冯至]]于当年4月7日给胡乔木写了一封长信，表达了对译林刊登《[[尼罗河上的惨案|尼罗河上的惨案]]》的不满。胡乔木收到信后，加批语转发江苏省委研究处理，江苏省委研究后批示“……译林还是应该办下去，但选稿应当坚持党的文艺方针……”，随后根据省委的精神，江苏省出版局和译林编辑部做了自查报告，后中宣部没有追究责任，冯至和译林编辑部负责人见面沟通，互相谅解。<ref>施亮，《译林》事件始末，炎黄春秋2008年第6期</ref>

1983年，在文艺战线提出“[[清除精神污染|清除精神污染]]”口号后，有人指责，译林杂志主要传播西方商业化的小说，对广大读者是很大的精神污染。译林杂志经受了各种压力，受到了排挤和冷落。<ref>施亮，《译林》事件始末，炎黄春秋2008年第6期</ref>
