'''壬生忠見'''（{{jpn|j=壬生 忠見|hg= みぶ の ただみ |rm= }}，生卒年不详），是日本[[平安時代|平安時代]]中期的[[歌人|歌人]]。[[壬生忠岑|壬生忠岑]]之子。初名'''多'''、後改'''忠实'''、最后改为'''忠見'''。与父同属[[三十六歌仙|三十六歌仙]]。[[官位|官位]]为[[正六位|正六位上]][[伊予国|伊予]][[掾|掾]]。

== 生平 ==
初居[[攝津|攝津]]，虽然生活貧苦，但他很早就以和歌著称。

954年（[[天历|天历]]8年），就职于御厨子所定外膳部。958年（[[天德|天德]]2年）升六位攝津大目，除此之外再没有对他官职的记载。

曾参加953年（[[天历|天历]]7年）10月的内裏菊合、960年（[[天德|天德]]4年）的内裏歌合等。活跃于村上朝的歌坛，有许多屏风画歌流传。

《後撰和歌集》收录其1首和歌，之后的勅撰和歌集收录36首。个人歌集为《忠見集》。

== 逸事 ==
=== 竹馬 <ref>《袋草子》 </ref>===
壬生忠见少年时家贫，但有和歌才能优异而为人所知，一次受召入宫，没有可以乘坐的车，天皇戏言道：“不如骑竹马来”，壬生忠见歌道：

}}

=== 与[[平兼盛|平兼盛]]对决 ===
在天徳内裏歌合中，壬生忠見与[[平兼盛|平兼盛]]对决。壬生忠見吟咏道： 
{{Quote box |width=500px |align=center|quoted=true |bgcolor=#FFFFF0 |salign=right|source =——《拾遺和歌集·恋一·621》《百人一首·41》
 |quote  ={{lang|ja|恋すてふ我が名はまだき立ちにけり人知れずこそ思ひそめしか}}  <br />  暗情人尽知，唯吾窃恋不闻噂，尚思萌芽情
}}
可惜未能得到天皇赏识，败于[[平兼盛|平兼盛]]。《[[沙石集|沙石集]]》记载其不久郁郁而终，但《[[袋草紙|袋草紙]]》中提到壬生忠见个人歌集中仍有不少老年时期的作品，故对《[[沙石集|沙石集]]》的记载发出质疑。
