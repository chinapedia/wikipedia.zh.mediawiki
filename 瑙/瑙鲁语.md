'''瑙鲁语'''(''dorerin Naoero'')又稱'''諾魯文'''，为一[[波利尼西亚|波利尼西亚]][[语言|语言]]，属于[[南岛语系|南岛语系]]，主要使用者分布于[[瑙鲁|瑙鲁]]。目前瑙鲁语的使用者约有7,000人，其中约50%分布在瑙鲁。大部分使用者都能兼用[[英语|英语]]。

== 发音 ==
瑙鲁语有16-17个[[辅音|辅音]]，12个[[元音|元音]]。

== 书写 ==
目前瑙鲁语使用如下29個[[拉丁字母|拉丁字母]]：
* [[元音|元音]]: a, ã, e, i, o, õ, u, ũ
* [[半元音|半元音]]: j
* [[辅音|辅音]]: b, c, d, f, g, h, j, k, l, m, n, ñ, p, q, r, s, t, w, y, z
