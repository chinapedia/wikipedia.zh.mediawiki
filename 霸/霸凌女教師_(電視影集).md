《'''霸凌女教師'''》（{{lang-en|'''''Bad Teacher'''''}}）為美國[[情境喜劇|情境喜劇]]系列，由希拉蕊·溫斯頓（Hilary Winston）創作，根據2011年的同名電影《{{link-en|霸凌女教師|Bad Teacher}}》改編。本[[单镜头模式|單鏡頭]]影集於2014年4月24日在[[CBS_(電視網)|CBS]]首映，為[[2013年－2014年美国电视网节目表|2013年－2014年美國電視網節目表]]之一。<ref>{{cite web |url=http://www.thefutoncritic.com/news/2014/02/14/mid-season-premiere-dates-announced-for-bad-teacher-and-unforgettable-619415/20140214cbs01/ |title=Mid-Season Premiere Dates Announced for ''Bad Teacher'' and ''Unforgettable'' |date=February 14, 2014 |publisher=The Futon Critic |accessdate=April 26, 2014}}</ref>

==故事大意==
梅雷迪思·戴維斯（Meredith Davis）與她的先生離婚，由於簽署[[婚前協議書|婚前協議書]]的緣故，因此她什麼都得不到。梅雷迪思被迫搬到她朋友的家中同住，在帶朋友的女兒莉莉（Lily）上下學時，想到可以在莉莉的學校擔任老師並且釣多金的單身父親，可讓她回到過去貴婦的生活。因此她假造履歷欺騙該學校的校長卡爾·蓋恩斯雇用她。故事就發生在她與學校的其他老師：體育老師喬爾（Joel）、艾琳（Irene）；金妮（Ginny），以及她朋友的女兒莉莉以及其他學生之間展開。

==演員==

===主要===
*[[艾莉·葛瑞那|艾莉·葛瑞那]] 飾演 梅雷迪思·戴維斯（Meredith Davis），剛離婚的前釣金女，由於缺錢，因此成為國小老師，並試圖釣一位多金的單身父親與其結婚。在看似放蕩的外表，其實很重視學生之間的霸凌問題。
*[[莎拉·吉爾伯特|莎拉·吉爾伯特]] 飾演 艾琳（Irene）學校的老師之一，外表不佳，有社交問題，因此期望與梅雷迪思成為好朋友。
*[[Ryan_Hansen|Ryan Hansen]] 飾演 喬爾·寇斯（Joel Kotsky），學校的體育老師，並且也是梅雷迪思的高中同學，對梅雷迪思有明顯的好感，然而梅雷迪思很明顯看不上他。
*Sara Rodier 飾演 莉莉（Lily），梅雷迪思朋友的女兒，與梅雷迪思在同一間小學念書，並擔任校園交通指揮，有被霸凌的問題。
*[[克莉絲汀·戴維斯|克莉絲汀·戴維斯]] 飾演 金妮·泰勒-克拉普（Ginny Taylor-Clapp），學校的老師之一，自我要求嚴格，並強烈對梅雷迪思抱持懷疑。
*{{link-en|大衛·亞倫·基亞|David Alan Grier}} 飾演 卡爾·蓋恩斯（Carl Gaines），學校的校長，最近剛離婚，目前室友同居。經常被梅雷迪思的花言巧語所欺騙。

===配角===
*Caitlin Kimball 飾演 金（Kim），一位實習老師，目前跟在金妮老師身邊學習。

==製作與發展==
本劇首次於2012年10月5日發表，並於2013年1月23日預訂製作首播集。<ref>{{cite web |url=http://www.deadline.com/2012/10/bad-teacher-comedy-from-hillary-winston-gene-stupnitsky-lee-eisenberg-to-cbs/ |title=''Bad Teacher'' Comedy From Hilary Winston, Gene Stupnitsky & Lee Eisenberg To CBS |last=Andreeva |first=Nellie |date=October 5, 2012 |publisher=[[Deadline.com|Deadline.com]] |accessdate=April 26, 2014}}</ref><ref>{{cite web |url=http://www.thefutoncritic.com/news/2013/01/23/development-update-wednesday-january-23-700403/10178/ |title=Development Update: Wednesday, January 23 |date=January 23, 2013 |publisher=The Futon Critic |accessdate=April 26, 2014}}</ref> 2013年3月13日，宣布由唐斯‧卡爾迪諾（Don Scardino）成為首播集的導演。<ref name="devUpdate-3.13">{{cite web |url=http://www.thefutoncritic.com/news/2013/03/13/development-update-wednesday-march-13-836125/10246/ |title=Development Update: Wednesday, March 13 |date=March 13, 2013 |publisher=The Futon Critic |accessdate=April 26, 2014}}</ref>

2013年2月18日，宣布由大衛·亞倫·基亞加入劇組並飾演卡爾（Carl）。<ref>{{cite web |url=http://www.thefutoncritic.com/news/2013/02/18/development-update-monday-february-18-17405/10212/ |title=Development Update: Monday, February 18 |date=February 16, 2013 |publisher=The Futon Critic |accessdate=April 26, 2014}}</ref> 同年2月19日，宣布由艾莉·葛瑞那飾演本劇女主角梅雷迪思（Meredith）。<ref>{{cite web |url=http://www.thefutoncritic.com/news/2013/02/19/development-update-tuesday-february-19-12300/10213/ |title=Development Update: Tuesday, February 19 |date=February 19, 2013 |publisher=The Futon Critic |accessdate=April 26, 2014}}</ref> 2月21日，宣布Ryan Hansen 飾演喬爾(Joel)。<ref>{{cite web |url=http://www.thefutoncritic.com/news/2013/02/21/development-update-thursday-february-21-607103/10215/ |title=Development Update: Thursday, February 21 |date=February 21, 2013 |publisher=The Futon Critic |accessdate=April 26, 2014}}</ref> 3月6日，宣布莎拉·吉爾伯特飾演艾琳（Irene）。<ref>{{cite web |url=http://www.deadline.com/2013/03/the-talks-sara-gilbert-to-co-star-in-cbs-pilot-bad-teacher/ |title=''The Talk''{{'}}s Sara Gilbert To Co-Star In CBS Pilot ''Bad Teacher'' |last=Andreeva |first=Nellie |date=March 6, 2013 |publisher=[[Deadline.com|Deadline.com]] |accessdate=April 26, 2014}}</ref> 3月13日，宣布克莉絲汀·戴維斯飾演金妮（Ginny）。<ref name="devUpdate-3.13" /> Sara Rodier 飾演莉莉Lily，不過並未特別宣布。2013年3月19日，宣布Caitlin Kimball加入本劇組飾演配角金（Kim）。<ref>{{cite web |url=http://www.thefutoncritic.com/news/2013/03/19/development-update-tuesday-march-19-508213/10254/ |title=Development Update: Tuesday, March 19 |date=March 19, 2013 |publisher=The Futon Critic |accessdate=April 26, 2014}}</ref>

2013年5月13日，報導指出本劇將不被放入2013年CBS秋季的預定表中。<ref>{{cite web |url=http://www.thefutoncritic.com/news/2013/05/13/development-update-monday-may-13-979113/10377/ |title=Development Update: Monday, May 13 |date=May 13, 2013 |publisher=The Futon Critic |accessdate=April 26, 2014}}</ref> 2013年5月15日，宣布本劇將排除在CBS 2013-14年度節目預定表當中，然而一星期後，2013年5月22日，又宣布本劇重新進入節目預定表裡。<ref>{{cite web |url=http://www.thefutoncritic.com/news/2013/05/15/cbs-announces-2013-2014-primetime-schedule-659502/20130515cbs01/ |title=CBS Announces 2013-2014 Primetime Schedule |date=May 15, 2013 |publisher=The Futon Critic |accessdate=April 26, 2014}}</ref><ref>{{cite web |url=http://www.thefutoncritic.com/news/2013/05/22/cbs-orders-single-camera-comedy-bad-teacher-for-2013-2014-season-237315/20130522cbs03/ |title=CBS Orders Single-Camera Comedy ''Bad Teacher'' for 2013-2014 Season |date=May 22, 2013 |publisher=The Futon Critic |accessdate=April 26, 2014}}</ref>
