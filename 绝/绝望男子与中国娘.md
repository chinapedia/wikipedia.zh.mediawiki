《'''绝望男子与中国娘'''》（{{lang-ja|絶望男子と中国娘}}）是由[[日本|日本]][[漫画家|漫画家]][[理央|理央]]创作的[[漫画|漫画]]作品<ref>{{Cite web|url = http://www.akitashoten.co.jp/comics/4253130941|title = 絶望男子と中国娘 第1巻 {{!}} 秋田書店|accessdate = 2015年8月22日|author = |date = |publisher = 秋田书店|language=ja}}</ref>。

== 创作背景 ==
该作品是作者理央的首个商业连载漫画，最初于[[pixiv|pixiv]]网站上以《'''交流障碍男子与中国娘'''（{{lang-ja|コミュ障男子と中国娘}}）》的标题进行黑白连载。因反响较好，该作品后来转移到了《{{lang|ja|Championタップ！}}》杂志进行连载，并转为彩色漫画，同时标题也改为现在的《绝望男子与中国娘》<ref name=":0" />。

== 故事介绍 ==
本作品讲述了一位没有恋人、朋友，且无法正常与他人交流的日本男子根子太郎的故事。有着交流障碍的他，每天的唯一消遣就是在夜深之时，只身一人前往[[便利店|便利店]]。然而，在某天晚上，他遇到了一位说着[[中文|中文]]的神秘女孩阎龙龙，故事也就由此展开<ref name=":0">{{Cite web|url = http://comic.pixiv.net/works/1400|title = 絶望男子と中国娘 - pixivコミックで漫画を無料試し読み|accessdate = 2015年8月22日|author = |date = |publisher = [[Pixiv|Pixiv]]|language = ja}}</ref>。作品中，根子太郎逐渐被阎龙龙的可爱所着迷，两人的交流也越来越多，在根子太郎遇到困难时，阎龙龙也会给予鼓励，逐步帮助他开口说话。
