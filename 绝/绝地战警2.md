{{作品名稱|name=''Bad Boys II''|hk=重案夢幻組2|tw=絕地戰警2|equiv-cn=tw}}是一部2003年的[[动作片|动作]][[喜剧|喜剧]]电影，由[[迈克尔·贝|迈克尔·贝]]执导，[[傑瑞·布洛克海默|傑瑞·布洛克海默]]担任监制，主演为[[威尔·史密斯|威尔·史密斯]]和[[马丁·劳伦斯|马丁·劳伦斯]]。该片是1995年的电影《[[绝地战警|绝地战警]]》的续篇，讲述两位警探追查进入[[迈阿密|迈阿密]]的[[毒品|毒品]]地下交易网的经过。该片的总体评价不佳，但是票房收入很好，在全球收益为2.73億美元。<ref name="boxoffice" />[[絕地戰警3|續集]]定於2020年1月17日上映。

== 剧情大纲 ==
在第一部电影之后8年，警探麦克·劳里（Mike Lowrey，[[威尔·史密斯|威尔·史密斯]]饰演）和马奎斯·巴涅特（Marcus Burnett，[[马丁·劳伦斯|马丁·劳伦斯]]饰演）正在追查进入[[迈阿密|迈阿密]]的[[迷幻药|迷幻药]]源头。他们追踪着几只来自[[古巴|古巴]]，来到一个[[三K党|三K党]]聚会，毒品也在这里进行交易。在交火中，麦克意外射伤了马奎斯的臀部，使得马奎斯开始考虑自己是否想要和麦克继续搭档。

同时，有点[[神经症|神经症]]的古巴毒枭约翰尼·塔皮亚（Johnny Tapia）要求手下再次更换运毒船的日程。两个[[俄罗斯黑帮|俄罗斯黑帮]]成员阿列克谢（Alexei）和约瑟夫（Josef）从塔皮亚那里得到毒品，以经营他们的夜总会生意，但是利润中的近半数都归入塔皮亚的囊中。两人决定和塔皮亚谈判，收回部分利润份额，但是塔皮亚派手下杀死了约瑟夫，并拿阿列克谢的妻儿做人质，威逼他把自己的夜总会拱手让出。

麦克和马奎斯的妹妹希德（Syd）从之前在[[纽约|纽约]]见面后便发生了恋情，而希德正巧是[[美国缉毒署|美国缉毒署]]的卧底，正在调查俄罗斯黑帮的[[洗钱|洗钱]]行为。在她第一次执行任务时，一个[[海地|海地]]帮派企图阻断交易，杀死希德。帮派成员同迈阿密警察/缉毒署探员发生交火。马奎斯和麦克因此得知了希德的真实工作，而马奎斯对此并不开心。

马奎斯和麦克前去和海地帮头目对峙，结果发生枪战，而帮派头目则告诉他们关于交易的信息来自于他朋友给他的一段录影。在看完这段视频后，马奎斯和麦克意识到，由塔皮亚母亲持有的西班牙棕榈树殡仪馆（Spanish Palms Mortuary）很可能被用作掩盖毒品交易。伪装成除虫公司的雇员，他们进入塔皮亚的宅邸，并发现塔皮亚是用殡仪馆里的死尸将毒品和金钱走私到古巴。在逃跑时，前来为约瑟夫复仇的阿列克谢杀掉了塔皮亚的绝大部分保安，但在交火中被警方射杀。

麦克和马奎斯以及志愿前往的[[特種武器和戰術部隊|SWAT]]小队前往古巴解救被塔皮亚绑架的希德。一场枪战过后，古巴军队来到现场。塔皮亚的新房子被毁，母亲和女儿困在其中，愤怒的塔皮亚在麦克、马奎斯、希德身后穷追不舍。最后，他们来到[[关塔纳摩湾|关塔纳摩湾]]的美国海军基地。马奎斯和希德向基地的士兵求助，而塔皮亚和麦克之间发生交火。马奎斯抓住机会，射出自己最后一发子弹，正中塔皮亚头部。塔皮亚倒地，触发了一枚地雷，他的尸体在爆炸中灰飞烟灭。

之后，在巴涅特的家中，麦克为马奎斯买了一个新泳池，而马奎斯也终于默许了麦克和希德的恋情。他将自己本准备上交的调职申请撕掉。最后，泳池还是坏掉了，两人在被水淋透后，唱起电视剧《Cops》的主题曲《Bad Boys》。<ref>{{cite web|url=http://www.imdb.com/title/tt0172156/plotsummary|title=Plot Summary for Bad Boys II|publisher=imdb.com}}</ref>

== 反响 ==
=== 评论界 ===
该片在评论界反响不佳。该片因其冗长、充满暴力且臃肿的故事线，贬低女性的论调以及充满暴力的笑料成为评论界炮轰的对象。电影评论网[[烂番茄|烂番茄]]征集了149位影评人的意见，只有23%的影评人对该片持肯定态度，平均评分仅为4.1（满分10分）。<ref>{{cite web |title = Bad Boys II (2003) |url = http://www.rottentomatoes.com/m/bad_boys_ii/ |work = [[烂番茄|烂番茄]] |accessdate =2009-09-01 }}</ref>

《[[芝加哥太阳时报|芝加哥太阳时报]]》的[[罗杰·艾伯特|罗杰·艾伯特]]给了该片1星（满分4星），并且对该片中一个小男孩的场景以及反复使用“黑鬼”（nigga）一词感到不满，他认为：“这一场景不必要的残酷性让我脱离了-{影片}-，而试图理解制作人的想法。他们到底在想什么呢？他们怎么会如此不了解人之本性，而觉得这样的场景会受到观众的喜爱？”<ref>{{cite web |first = Roger |last = Ebert |title = Bad Boys II Review |url = http://rogerebert.suntimes.com/apps/pbcs.dll/article?AID=/20030718/REVIEWS/307180301/1023 |archiveurl = http://www.amo.org.au/artist.asp?id=3858 |publisher = ''Chicago Sun-Time'' |date = 2003-07-18 |archivedate = 2009-09-01 |accessdate = 2009-09-01 }}</ref>在影评节目《同厄伯特与罗珀一起看电影》（At the Movies with Ebert & Roeper》中，影评人{{link-en|理查德·罗珀|Richard Roeper}}在评选自己心目中2003年度烂片榜时，将该片放在第一名，而同年臭名昭著的《鸳鸯绑匪》（Gigli）反而居于第三位。在2010年元旦，罗珀评选的10年内最烂电影中，该片再次上榜，位列第9。<ref>{{cite web|url=http://www.suntimes.com/news/roeper/1967978,roeper-worst-movies-of-the-year-010110.article|title=Roeper's worst movies of the decade|author=Richard Roeper|date=2010-01-01|deadurl=yes|archiveurl=https://web.archive.org/web/20100419130657/http://www.suntimes.com/news/roeper/1967978,roeper-worst-movies-of-the-year-010110.article|archivedate=2010-04-19}}</ref>

而《[[西雅图邮讯报|西雅图邮讯报]]》的影评人艾伦·A·基姆（Ellen A. Kim）对该片的评价较为正面，他认为该片包含有“一种盲目的乐趣……如果你喜欢这一类型的电影，那它就是你的选择。”<ref>{{cite web|url=http://www.seattlepi.com/movies/131238_badboys18q.html|title=Plot, schmot. 'Bad Boys II' is an unabashed guilty pleasure|author=Ellen A. Kim|date=2003-07-18|publisher=Seattle Post-Intelligencer}}该片的动作场面和视觉特效也获得了部分评论家和观众的好评。</ref>

=== 票房 ===
该片的票房获得巨大成功。在[[北美|北美]]，该片票房成绩为138,608,444美元；在其他地区，票房为134,731,112美元，全球票房共计273,339,556美元，这是第一部毛收益的两倍。<ref name="boxoffice" />

=== 奖项 ===
在2004年度[[MTV电影大奖|MTV电影大奖]]上，该片因在高速上的追击场景，获“最佳动作场景”提名，但是败给了《[[指环王：王者归来|指环王：王者归来]]》。[[威尔·史密斯|威尔·史密斯]]和[[马丁·劳伦斯|马丁·劳伦斯]]获得了“最佳荧幕组合”奖。<ref>{{cite web|url=http://www.mtv.com/ontv/movieawards/2004/|title=2004 MTV Movie Awards|publisher=MTV.com}}</ref>

=== 游戏 ===
该电影的[[电子游戏|电子游戏]]版名为《[[絕地戰警：邁阿密行動|絕地戰警：邁阿密行動]]》（Bad Boys: Miami Takedown），于2004年在北美發售，基于[[PS2|PS2]]、[[Xbox|Xbox]]、[[GameCube|GameCube]]以及[[Windows|Windows]]等平台。该游戏本来准备在2003年末发布（和电影的[[DVD|DVD]]同期发售），但是最后不得不推迟数月。由于游戏开发不佳，并没有获得销量或评论上的任何成功，诸多游戏站点都给了它较低的评分。<ref>{{cite web|url=http://www.gamespot.com/ps2/action/badboys2/review.html|title=Bad Boys: Miami Takedown Review|date=2004-09-24|publisher=Gamespot|author=Alex Navarro}}</ref>

== 续集 ==
