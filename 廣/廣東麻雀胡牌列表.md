== 零番 ==
*'''雞胡（無番胡）'''：和牌後數不出任何番種分（花牌不計算在內）。（註：個別玩家視無番胡為一番種，為五番）

== 一番 ==
*'''自摸'''：靠自己摸牌使之胡。
*'''三元牌（箭刻）'''：牌型擁有一副中、發、白的刻子，各一番。若攞齊任兩副刻字又叫雙箭刻。
*'''明槓'''：有暗刻，碰別人打出一張相同的牌開槓；或自己抓進一張與碰明刻相同的牌開槓。
*'''無花'''：牌型沒有花牌。
*'''正花'''：拿到與自己門風相符的花牌。
*'''門清'''：全副牌靠自己摸，無碰上可暗槓，最後靠人出衝(出銃)。
*'''海底撈月（妙手回春）'''：抽最後一張牌自摸。(可定為五番)
*'''海底撈魚（一炮而紅）'''：別人打最後一張胡（出衝(出銃)）。
*'''絕張'''：胡的牌只剩下一隻未亮相。(可定為五番)
*'''生張'''：胡的牌未曾亮相。
*'''坎張（卡窿）'''：食兩張牌之間的牌。
*'''將'''：一對眼為「二」、「五」或「八」
*'''缺五'''：牌型沒有「五」此數字。
*'''花<nowiki>幺</nowiki>'''：牌型有「一」或「九」的刻子。
*'''無字'''：牌型沒有番子（即所有字牌，包括白板）。
*'''莊家'''：只適用於莊家。莊家胡糊多算一番，出衝也要多賠一番。
*'''[[一般高|般高]]'''：拿到兩副相同的順子。
*'''卡五'''：自己拿着「四」同「六」、卡窿叫「五」（另計坎張）。
*'''單釣（獨聽）'''：胡糊只叫一番(缺將牌)。
*'''邊張'''：單釣「一二三」的「三」或「七八九」的「七」。
*'''老少'''：同花色的1.「一二三」和「七八九」（老少副），或2.「一一一」和「九九九」（老少碰）。
*'''缺門（缺一門）'''：牌型只有兩種花色，無字牌。另計無字(不同玩家視缺門不需無字)。
*'''彩龍（三色通貫/花龍）'''：牌型有三種花色之「一二三」「四五六」「七八九」。
*'''[[搶槓|搶槓]]'''：別人加槓時該牌是胡牌，通常運用於十三幺。(可定為五番)
*'''兩槓'''：槓出兩張牌，明、暗或加槓不限。
*'''清四碰'''：光靠碰牌而有四副刻子，包明或加槓，另計對對胡。(可定為五番)
*'''四歸一'''：四張同樣的牌分為一刻子和一順子。(可定為三番)
*'''半求人'''：全部牌靠人上或碰，最尾自摸。
*'''兩色兩順（喜相逢/半杯口）'''：有兩副不同花色的順子。
*'''連六（半龍）'''：同一花色有六張相連的牌。

== 三番 ==
*'''門風'''：風刻子剛好是自己的門風位。
*'''圈風'''：風刻子剛好是該圈的圈風。
*'''[[平胡|平胡]]'''：全部牌為順子組成，但眼不能為字牌。（不計無字）
*'''一台花（花槓）'''：集齊四季「春夏秋冬」（1234）或四君子「梅蘭菊竹」（一二三四）四種花牌。(可定為兩番)
*'''暗槓'''：在自抓四張相同的牌開槓。只有暗槓無其他上、碰、明或加槓。可另計門清。
*'''全求人'''：全部牌靠人上或碰，最尾靠出衝。
*'''門清自摸（不求人/門清一摸三）'''：牌型為門清再自摸。
*'''[[杠上开花|槓上開花（槓上自摸/嶺上開花）]]'''：明、暗或加槓補牌時，補牌為胡牌。
*'''雙暗刻'''：自己儲齊兩個暗刻、兩暗槓或一暗刻一暗槓。(可定為五番)
*'''[[断幺九|斷<nowiki>幺</nowiki>（斷<nowiki>幺</nowiki>九/短<nowiki>幺</nowiki>）]]'''：牌中沒有「一」、「九」、或字牌。
*'''[[一条龙_(麻将)|一條龍（清龍/一氣通貫）]]'''：擁有同一花色「一」至「九」的牌。(可定為七番)
*'''[[三相逢|三相逢（三色同順/三色三同順）]]'''：牌型有三副同數不同花色之順子。(可定為五番)
*'''五門齊（五福臨門/五族共和）'''：牌型筒、索、萬、風牌、三元牌五種牌全部集齊。(可定為五番)
*'''四歸二'''：四張同樣的牌分為兩順子和一對眼。
*'''[[浑带幺|混帶<nowiki>幺</nowiki>（全帶九、全帶<nowiki>幺</nowiki>）]]'''：全部順子、刻子同眼都有「一」、「九」或「字牌」。
*'''雙同刻'''：牌型有兩副相同數字的刻子。
*'''三尾（三公）'''：1.同一花色，連續數字的兩副刻子和眼；或2.任意風牌兩副刻子加眼。
*'''三料牌'''：同時有上、碰同槓（非暗刻、順）。
*'''斷紅'''：牌型無萬子、135679 筒、1579 索和紅中。即248筒、23468索和东南西北发白

== 五番 ==
*'''[[三暗刻|三暗刻]]'''：靠自己儲齊三副刻子。(可定為七番)
*'''[[小三元|小三元]]'''：有中、發、白其中兩款刻子，餘下一款作眼。(可定為二十番)
*'''雞胡（無番胡）'''：和牌後數不出任何番種分（花牌不計算在內）。（註：個別玩家視無番胡為零番）
*'''[[对对和|對對胡（碰碰胡）]]'''：全副牌無順子。
*'''[[混一色|混一色（湊一色/半一色）]]'''：牌型由單一花色和字牌組成。(可定為三番)
*'''[[清带幺|清帶<nowiki>幺</nowiki>（全帶<nowiki>幺</nowiki>/純全帶九）]]'''：全部順子、刻子同眼都有「一」或「九」，且無字牌。
*'''組合龍'''：由三種花色做出「一四七」、「二五八」、「三六九」組合之牌。(可定為七番)
*'''三色三步高（迷你橋）'''：三種花色做出三組遞增一位或二位數字的順子。
*'''三色同刻（三相刻）'''：有筒、索、萬三種花色的同數字刻子。(可定為三番或七番)
*'''三色連刻（三色三節高/半連刻）'''：三種花色做出三組遞增數字的刻子（計槓）。
*'''[[推不倒|推不倒]]'''：牌型由旋轉對稱，上下沒有區別的牌組成。牌型即由一二三四五八九餅、二四五六八九索同白板組成。
*'''孔雀的飛'''：1.一索刻子加兩個任何風刻；2.一索刻子加兩個箭刻，但無風刻。

== 七番 ==
*'''三風刻'''：牌型有三個風刻。
*'''[[七對子|七對子]]'''：門清取得七對對子（有些玩法未開槓的四張相同的牌不能算是兩個對子）。(可定為十番)
*'''大於五'''：牌型由「六」至「九」組成。
*'''小於五'''：牌型由「一」至「四」組成。
*'''三元七對（三分天下）'''：有「中」、「發」、「白」在內的七對子。（另計七對子）

== 八番 ==
*'''七搶一'''：自己摸到七張花牌，另一家一有花牌，即可搶牌胡牌，計出衝。
*'''全不靠（十三不靠/十三天使）'''：三種花色的「一四七」、「二五八」、「三六九」，不能錯位，加上東南西北中發白，單一任何牌組成十四張牌（無眼）。
*'''單龍抱'''：般高加般低（即遞增一位的一色二步高，本身無番），須門清。
*'''亂蛇（包羅萬象）'''：兩個般低，須門清。

== 十番 ==
*'''[[三杠子|三槓]]'''：槓出三副牌。
*'''[[大三元_(麻将)|大三元]]（三元及第/鼎足三分）'''：牌型拿到「中」、「發」、白」刻子。(可定為四十番)
*'''雙槓自摸（槓上槓自摸/連槓）'''：連續兩槓後自摸。中間補花即不計。明槓出衝包自摸。
*'''[[清一色|清一色]]（鐵騎兵）'''：牌型由同一花色組成。(可定為八番)
*'''[[流局满贯|流局十番（<nowiki>幺</nowiki>九振切）]]'''：流局時門清且捨牌全為「一」「九」或字牌無被上碰槓。計自摸。
*'''配牌花胡'''：起手拿到1.八隻花牌，或2.七隻花牌和別人一隻。按正常流程，手牌足十四張時胡牌1.八仙過海，或2.七搶一，另加計十番。
*'''[[一色三節高|一色三節高（姊姊碰/三連刻）]]'''：同一花色的三副連續遞增的刻子，計槓。
*'''[[一色三同順|一色三同順]]'''：同一花色的三副相同順子。
*'''雙飄'''：牌型拿到兩副不同花色的老少。
*'''[[两般高|雙龍抱（二杯口）]]'''：牌型拿到兩副般高，須門清。
*'''三色雙龍會'''：兩種花色的老少，餘下一種的「五」作眼（另計雙飄）。(可定為七番)
*'''二不槓（雙鐵支）'''：門清時，以兩組沒開槓之牌胡牌（無眼）。
*'''全大/全中/全小'''：牌型由「七八九」/「四五六」/「一二三」組成。
*'''九蓮花燈'''：三種花色的搭子及眼配成「一一一二三四五六七八九九九」的牌型。
*'''全雙刻（偶數）'''：牌型由「二」、「四」、「六」、「八」組成。
*'''全單刻（奇數）'''：牌型由「一」、「三」、「五」、「七」、「九」組成。
*'''頂三刻'''：牌型有同一花色的「一」、「五」「九」刻子，計槓。
*'''跳三刻'''：牌型有同花色的一三五、二四六、三五七、四六八、五七九刻子，計槓。
*'''筋三刻'''：牌型有同花色的「一四七」、「二五八」、「三六九」刻子，計槓。
*'''四喜七對（混地雙）'''：七對子有齊東南西北的對子（另計七對子）。
*'''雙色同刻'''：兩種花色兩組相同刻子。
*'''三步刻（天下名士）'''：牌型由「一四七」、「二五八」或「三六九」組成。
*'''四歸四'''：四隻相同的牌落到四個順子上。

== 十五番 ==
*'''全帶（滿庭芳）'''：牌型所有順子、刻子和眼都有同一數字（一些玩法僅能全帶「五」）。(可定為七番)
*'''三數'''：牌型由三個數字組成（風牌、三元牌各自當一個數字）。
*'''[[混老頭|混老頭]]（混<nowiki>幺</nowiki>九/地九）'''：牌型由「一」、「九」及字牌組成。(可定為十番)
*'''一色三步高（半步橋/萬乘之國）'''：同一花色三副遞增一位或二位的順子。(可定為七番)
*'''客風三刻（萬馬奔騰）'''：牌型有三副風刻子，但不能有圈風牌。
*'''二色同對（清杯口）'''：同順子的雙龍抱。
*'''鏡胡糊'''：兩種花色兩種相同順子。
*'''一半數字'''：牌型中單數牌同雙數牌數目一樣。
*'''連六對（柳丁）'''：包含同一花色連續六對的七對子（另計七對子）。

== 二十番 ==
*'''天聽（[[雙立直|雙立直]]）'''：閒家開牌（計補花）後即宣佈听牌，不計門清，自摸計不求人。中途換牌或有得食不食，天聽即解除。
*'''地聽'''：打出第一張牌即宣佈听牌。本身須門清但不計，自摸計不求人。中途換牌或有得食不食，地聽即解除。
*'''[[四喜和|小四喜]]'''：東、南、西、北其中三副為刻子，餘下一副做眼。
*'''[[四暗刻|四暗刻]]（坎坎糊）'''：牌型有四副自己摸的刻子。單釣可食出衝，否則要靠自摸。
*'''兩數'''：牌型由兩個數字組成（數字計法跟三數一致）。
*'''八仙過海（大花糊/全花牌）'''：拿到八張花牌，可不理牌型立即糊牌。
*'''萬炮齊發（海底搶槓）'''：海底撈月加搶槓。
*'''滿分花槓'''：三槓之中有一暗槓、一明槓，剩下的須是槓上開花或搶槓。
*'''喊九蓮'''：以非門清方式拿到九蓮寶燈牌型加其中一隻做眼。
*'''七星不靠（十三魔王）'''：任意單一的三種花色的「一四七」、「二五八」、「三六九」，不能錯位，加上全部東南西北中發白，組成十四張牌（無眼）。(可定為十番)
*'''三不槓（三鐵支）'''：門清以三組無開槓之牌胡牌。
*'''全將碰（將帥領兵共和國）'''：牌型由「二」、「五」、「八」的刻子組成，不計三數。
*'''四字刻'''：牌型由四副字牌刻子，加上非字牌做眼。
*'''新幹線'''：牌型一條龍加兩個風牌胡牌；或紅中白板。
*'''雙龍搶珠（雙龜搶珠）'''：萬、索兩副連六加上一對筒子。
*'''十三魔殿（七星王國）'''：四七萬、二五八筒、三六索、東南西北中發白（另計七星不靠）。

== 三十番 ==
*'''人胡（天聽一發）'''：閒家起手天聽，食別家打出的第一張牌。不計天聽、門清。(可定為四十番)
*'''地聽一發'''：地聽後，食別家打出的第一張牌或自摸。不計地聽、門清。
*'''三連槓自摸'''：連續開三個槓之後胡牌。中間補花即不計。明槓出衝包自摸。
*'''鐘鼓齊鳴（槓爆/海底槓自摸）'''：海底撈月加槓上開花。
*'''一色四步高（段橋）'''：一種花色四副依次遞增一位或兩位的順子。
*'''風花雪月'''：一筒（月）、五筒（花）、任何風刻（風）同白板（雪）四副刻子。
*'''花鳥風月'''：一筒（月）、五筒（花）、任何風刻（風）同一索（鳥）四副刻子。
*'''萬綠一點紅（紅色森林）'''：由紅中代替發財的綠一色。
*'''萬綠叢中一點紅（環保車/綠一色車輪）'''：有五索在內的綠一色。
*'''十三金字塔'''：門清取得全部幺九牌，加上四七萬、二五八筒，三六索，加其中一隻做眼。
*'''自動車道'''：牌型由「二」、「四」、「六」筒加上兩個風牌，或紅中白板組成。
*'''青洞門'''：牌型由「二」、「四」、「八」筒加上一風牌和發財組成。
*'''十三騎士（七星對）'''：四七萬、二五八筒、三六索的七對子。

== 四十番 ==
*'''[[天和_(麻将)|天胡]]'''：莊家起手（計補花）即胡牌。不計門清一摸三。暗槓、連續暗槓照計。
*'''[[地和|地胡]]'''：閒家起手天聽後第一張牌自摸，不理別家上碰或莊家槓上開花。
*'''[[四喜和|大四喜]]（天雙喜/四喜臨門）'''：牌型有東、南、西、北四副刻子。
*'''四方大發'''：以發財為將的大四喜。(可再另計大四喜)
*'''[[四杠子|十八羅漢（四槓/十八學士/槓槓胡）]]'''：槓出四張不同的牌。
*'''[[清老头|清老頭（清<nowiki>幺</nowiki>九/全<nowiki>幺</nowiki>碰/天九牌）]]'''：牌型由「一」和「九」組成。(可定為二十番)
*'''[[字一色|字一色]]'''：牌型由字牌組成。可計七對子。(可定為二十番)
*'''[[十三幺|十三<nowiki>幺</nowiki>（國士無雙）]]'''：門清取得全部<nowiki>幺</nowiki>九牌加字牌各一，其中一牌做眼。
*'''一色四順（同般高）'''：四張相同的牌分成兩副般高、或一副般高加一對順子。
*'''十三不搭'''：第一圈有全不靠的牌型，加上當中作一對子胡糊。（如做出全不靠，加五翻）
*'''同般低'''：般低牌型以般高牌組合。
*'''四連刻（一色四節高）'''：同一花色四副連續刻子。
*'''一色雙龍會'''：一種花色兩副老少，「五」為眼。不計七對。
*'''藍一色'''：牌型由風牌、白板或八筒組成。
*'''[[綠一色|綠一色]]（滿園春色）'''：牌型由二、三、四、六、八索或發財組成。可計清一色。
*'''紅孔雀（紅一色）'''：牌型由一、五、七、九索或紅中組成。若其中湊成雙鐵支可另計清一色。
*'''黑一色'''：牌型由二、四、八筒刻子加任何風牌組成。
*'''黃一色'''：牌型由任何萬子、五六七筒和紅中白板組成。
*'''[[大车轮|連七對（車輪滾滾/竹林種瓜/家庭祝福）]]'''：同花色連續的七對子。
*'''百萬石（百萬雄兵）'''：清一色狀態所有牌數加埋等於或超過一百。
*'''[[九莲宝灯|九子連環（九蓮寶燈/至尊清龍）]]'''：門清牌型為同花色「一一一二三四五六七八九九九」，再加當中任何一隻使胡牌。
