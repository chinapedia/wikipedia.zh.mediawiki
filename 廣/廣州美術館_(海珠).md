'''廣州美術館（海珠）'''，是[[廣東|廣東]][[廣州|廣州]]籌建中的新[[廣州美術館|廣州美術館]]，計劃於2019年落成啟用。

== 位置 ==
新美術館位於海珠區雙塔路南，赤崗北路西，藝苑東路東所包圍地皮。

== 籌備 ==
2012年[[广州市委|广州市委]]和[[广州市人民政府|市政府]]提出在[[海珠区|海珠区]]建设文化行政区，到2013年[[海珠湖|海珠湖]]北侧被规划發展為广州四大馆——广州博物馆新馆、广州美术馆、广州文化馆和广州科学馆<ref>[http://news.ycwb.com/2017-03/22/content_24499993.htm 广州新美术馆与新文化馆将于2019年完工并择日迎客]，大洋网 2017-03-22 </ref>。

新美術館原初建設方案，是'''廣州畫院暨廣州當代美術館'''項目，選址[[白云新城|白雲新城]]雲城西路，後根據市政府常務會議討論，廣州畫院分拆入[[广州大学城|廣州大學城]]的廣州文學藝術創作中心，廣州美術館另在[[广州城市中轴线|廣州新中軸線]]南段選址<ref>[http://news.sohu.com/20120822/n351220545.shtml 广州美术馆博物馆选址有变] 2012年08月22日 人民网</ref>。 

== 建築計劃 ==
新館2012年時的計劃建築面積擬為6萬平方米，計劃投資13.2億元。在建築定案由德国赫尔佐格与合伙人建筑设计事务所设计時，計劃建筑面积8万平方米，层高约40米。設施包括藏品库房、展陈区(含布展工程)、文化教育与公共服务区、业务科研用房、设备用房、地下停车场及室外配套设施等，投入财政资金近10億<ref>[http://www.gz.gov.cn/gzgov/s20584/201408/2711097.shtml 建设“四馆一园”等重点文化基础设施 广州美术馆] 广州市人民政府 2015-02-16</ref><ref>[http://news.artintern.net/html.php?id=48648 广州美术馆文化馆将动工]，2014年09月15日 广州日报</ref>。由於在征地拆迁、供电方案变更等方面的談判，發展进度缓慢。

廣州市重点办副主任胡妙杰表示，如果美术馆2018年5月动工，按计划会在2019年底完工。
