'''廣西省'''是[[中華人民共和國|中華人民共和國]]成立後所成立的[[中国一级行政区|省級行政區]]之一，前後存在9年多。1957年7月15日，[[第一屆全國人民代表大會|第一屆全國人民代表大會]]第四次會議決定撤銷廣西省；1958年3月5日，[[廣西僮族自治區|廣西僮族自治區]]成立。

== 人口 ==
中華人民共和國國家統計局於1954年11月1日公報的「第一次全國人口調查登記」，統計至1953年6月30日為止，廣西省人口為1956萬0822人。

== 歷史 ==
1949年12月11日，[[中国人民解放军|解放軍]]攻佔[[鎮南關|鎮南關]]，[[廣西戰役|廣西戰役]]結束。1950年2月，[[廣西省人民政府|廣西省人民政府]]在南寧成立。1951年5月，國民黨在廣西全省的勢力基本被消除。1952年12月10日，在邕寧、宜山、百色3個專區成立[[桂西僮族自治區|桂西僮族自治區]]；1956年3月改為[[桂西僮族自治州|桂西僮族自治州]]；1956年10月，中共中央提出了建立廣西僮族自治區的倡議，1957年6月[[中华人民共和国国务院|國務院]]決定建立廣西僮族自治區，並在同年7月召開[[第一屆全國人民代表大會|第一屆全國人民代表大會]]第四次會議上通過相應的決議，9月4日成立籌備委員會。1958年3月5日，廣西第一屆人民代表大會第一次會議召開，宣佈成立廣西僮族自治區。

== 行政區劃 ==
1956年底，廣西省轄1自治區（行政公署級），3專區、4省轄市、2地區，63縣、7自治縣、2市轄區、1縣級市，712區、20區級鎮，8830鄉、99鄉級鎮。
