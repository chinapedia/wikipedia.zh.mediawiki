'''廣見線'''（{{jpn|j=広見線|hg=ひろみせん|rm=Hiromi Sen|f=y}}）是一條連接[[日本|日本]][[岐阜縣|岐阜縣]][[可兒郡|可兒郡]]、[[可兒市|可兒市]]與[[愛知縣|愛知縣]][[犬山市|犬山市]]的[[鐵路|鐵路]]路線，由[[名古屋鐵道|名古屋鐵道]]擁有及營運。該線名稱取自可兒市的舊名{{tsl|ja|広見町 (岐阜県)|廣見町 (岐阜縣)|廣見町}}。
== 路線資料 ==
* 路線距離（{{link-ja|營業距離|営業キロ}}）：22.3公里
* [[軌距|軌距]]：1,067毫米
* 站數：11個（包括起終點站）
* 複線路段：犬山站－新可兒站間
* 電氣化路段：全線電氣化（[[直流電|直流電]]1500V）
* [[閉塞_(鐵路)|閉塞方式]]：自動閉塞式
* 可[[列車交會|交會]]車站：明智站
* 最高速度：90公里/小時（犬山站－新可兒站間）
* 最小曲線半徑：160公尺（犬山站構內）
== 歷史 ==
現在的廣見線是由兩條鐵路合併而成的。

* 往返御嵩的一段原為{{tsl|ja|東濃鉄道・東美鉄道|東美鐵道|東濃鐵道}}的一部份，於1920年通車<ref group="參">{{cite web|url={{NDLDC|2954536/10}}|title=地方鉄道運輸開始|work=官報|publisher=[[大藏省|大藏省]]印刷局|date=1920-08-27|accessdate=2017-03-22}}</ref>，使用762毫米[[窄軌|窄軌]]。東濃鐵道於1926年以廣見站為界分拆為兩段，南段[[日本國鐵|國有化]]為[[太多線|太多線]]<ref group="參">{{cite web|url={{NDLDC|2956375/2}}|title=鉄道省告示第176号|work=官報|publisher=[[大藏省|大藏省]]印刷局|date=1926-09-22|accessdate=2017-03-22}}</ref>，北段則由新成立的東美鐵道接收<ref group="參">{{cite web|url={{NDLDC|2956364/4}}|title=鉄道譲渡|work=官報|publisher=[[大藏省|大藏省]]印刷局|date=1926-09-09|accessdate=2017-03-22}}</ref>。1928年，太多線搬遷廣見站，東美鐵道改經現今路線以配合之，並更改軌距為1067毫米。支線{{tsl|ja|名鉄八百津線|八百津線}}於1930年通車。
* 往返犬山的一段原稱今渡線，由名鐵<ref group="註" name="名鐵">有關名古屋鐵道曾經使用的名稱，請參閱[[名古屋鐵道|相關條目]]</ref>興建，於1925年通車。該線最初從今渡站（即今{{tsl|ja|日本ライン今渡駅|日本萊茵河今渡站}}）經東犬山站直接連接往名古屋方向的[[犬山線|犬山線]]，四年後延長至廣見站並改稱廣見線。次年名岐鐵道<ref group="註" name="名鐵" />及東美的廣見站皆改名為新廣見站。

1942年，因應[[大平洋戰爭|戰時精簡鐵路管理的需要]]，名古屋鐵道<ref group="註" name="名鐵" />接管了東美鐵道<ref group="參" name="活化">{{cite web|url=http://www.town.mitake.gifu.jp/kasseika/rekisi/pdf/rekisi_3.pdf|title=名鉄広見線の歴史③：広見線の変遷|publisher=名鉄広見線活性化協議会（新可児駅～御嵩駅）|accessdate=2017-03-27}}</ref>，東美旗下所有路線皆稱為東美線。1946年，廣見線改接[[犬山站|犬山站]]，近東犬山站一段廢止。兩年後，往返御嵩的一段東美線併入廣見線，八百津線則獨立成線。

御嵩一段廣見線於1952年延長至現在的御嵩站，舊御嵩站則改稱御嵩口站。至於犬山至新廣見站之間於1967至1969年間擴闊為[[複線鐵路|複線]]。新廣見站於1982年更名為新可兒站。

八百津線於2001年廢止，而新可兒至御嵩之間的廣見線亦因長期赤字而一度受廢止的威脅<ref group="參" name="GFN">{{cite news|url=http://www.gifu-np.co.jp/blog/tetsu/2007/12/12/187|title=広見線存廃で名鉄が打診|author=「鉄」記者ブログ|newspaper={{tsl|ja|岐阜新聞}}|date=2007-12-12|accessdate=2017-03-27}}</ref>，後來因[[可兒市|可兒市]]、[[御嵩町|御嵩町]]及[[八百津町|八百津町]]政府給予補貼而得以繼續營運<ref group="參" name="ASN">{{cite web|url=http://iwase-akihiko.hatenablog.com/entry/20110211/1297419792|title=定期券 半額 モニター 制度も 苦戦　〜名鉄 広見線〜|date=2011-11-02|accessdate=2017-03-27}}</ref>。

== 服務詳情 ==
廣見線日間有以下列車服務<ref group="名" name="新可兒">{{cite web|url=http://trainbus.meitetsu.co.jp/pc/T2?sf=%E6%96%B0%E5%8F%AF%E5%85%90&rsf=4397-%E6%96%B0%E5%8F%AF%E5%85%90|title=新可兒駅|work=駅時刻表|accessdate=2017-03-27}}</ref><ref group="名" name="犬山平">{{cite web|url=http://www.meitetsu.co.jp/ekibetsu/timetable/timetable/tt313841.html|title=犬山站往新可兒方向平日時刻表|accessdate=2017-03-27}}</ref><ref group="名" name="犬山休">{{cite web|url=http://www.meitetsu.co.jp/ekibetsu/timetable/timetable/tt313842.html|title=犬山站往新可兒方向假日時刻表|accessdate=2017-03-27}}</ref>：
* 新可兒站至御嵩站設普通列車，每小時兩班，早上繁忙時間加至一小時四班
* 新可兒站至犬山站設普通列車，每小時四班，其中兩班列車在犬山站成為準急列車，沿[[犬山線|犬山線]]、[[名古屋本線|名古屋本線]]、[[常滑線|常滑線]]及[[機場線_(名古屋鐵道)|機場線]]往返中部國際機場站
此外，在繁忙時間有數班「μSky」（早上由新可兒站往中部國際機場，晚上回程）或「特急」（假日早上由新可兒站往豐橋站，不設回程）列車<ref group="名" name="新可兒" />，不停富岡前及善師野站<ref group="名" name="ST" />。
