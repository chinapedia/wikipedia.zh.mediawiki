'''廣瀨鈴'''（{{jpn|j=広瀬 すず|hg=ひろせ すず|rm=Hirose Suzu|f=y}}，{{bd|1998年|6月19日|||catIdx=Hirose Suzu}}）是[[日本|日本]][[女演員|女演員]]及[[模特兒|時裝模特兒]]，出身[[靜岡縣|靜岡縣]][[靜岡市|靜岡市]][[清水區_(日本)|清水區]]，所屬經紀公司為{{link-ja|FOSTER|フォスター (芸能プロダクション)}}旗下。姐姐是隸屬同一經紀公司的[[廣瀨愛麗絲|廣瀨愛麗絲]]。

== 經歷 ==
2012年，當時中學二年級14歲的廣瀨參加了女性潮流雜誌《[[Seventeen_(雜誌)|Seventeen]]》舉辦的「Miss Seventeen」模特兒甄選會，跟{{link-ja|藤井Sachi|藤井サチ}}、{{link-ja|岡崎紗繪|岡崎紗絵}}和{{link-ja|高堰麗|高堰うらら}}一起從6,515名報名者中脫穎而出，獲得大獎<ref>{{cite web|url=http://mantan-web.jp/2012/08/24/20120824dog00m200026000c.html |language=ja |title=<span lang="ja">ミスセブンティーン ： 6515人から選ばれた4人　読者イベントに初登場</span> |work=MANTANWEB |publisher=[[每日新聞|每日新聞]] |date=2010-08-18 |accessdate=2013-08-09}}</ref>。她因此跟在2009年加入《Seventeen》的姐姐[[廣瀨愛麗絲|廣瀨愛麗絲]]一起當上該雜誌的專屬模特兒<ref name="sanspo130407">{{cite web |url=http://www.sanspo.com/geino/news/20130407/oth13040705050006-n1.html |language=ja |title=<span lang="ja">広瀬すず、女優デビュー！「いつかは姉妹共演」</span> |publisher=sanspo.com |date=2013-04-07 |accessdate=2013-08-09 |deadurl=yes |archiveurl=https://web.archive.org/web/20130824010936/http://www.sanspo.com/geino/news/20130407/oth13040705050006-n1.html |archivedate=2013-08-24 }}</ref>。其後除了模特兒的工作外，也有參與廣告演出<ref>{{cite web|url=http://www.sponichi.co.jp/entertainment/news/2013/02/18/kiji/K20130218005222660.html |language=ja |title=<span lang="ja">広瀬すず　ＯＫまで100回以上「下です」</span> |publisher=[[體育日本|體育日本]] |date=2013-02-18 |accessdate=2013-08-09}}</ref>。

2013年春季，廣瀨在《[[幽靈女友|幽靈女友]]》之中初次出演電視戲劇<ref name="sanspo130407" />。2015年1月，於[[日本電視台|日本電視台]][[日本電視台週六連續劇|土九]]時段的《[[學校的階梯_(電視劇)|學校的階梯]]》初次擔任連續劇主演<ref>{{cite web|url=http://www.sanspo.com/geino/news/20141107/oth14110705030012-n1.html |language=ja |title=<span lang="ja">長ゼリフにも意欲！広瀬すず「学校のカイダン」で連ドラ初主演</span> |publisher=sanspo.com |date=2014-10-07}}</ref><ref>{{cite web|url=http://www.sankei.com/entertainments/news/141107/ent1411070009-n1.html |language=ja |title=<span lang="ja">日テレ幹部が１６歳少女泣かした？！　広瀬すず、ドッキリで呼び出され、連ドラ初主演オファーで…</span> |publisher=[[產經新聞|產經新聞]] |date=2014-10-07}}</ref>。

2017年11月20日，NHK公布由廣瀨主演2019年春季NHK第100部 [[連續電視小說|晨間劇]]《[[夏空|夏空]]》<ref>{{cite web |title=広瀬すず、朝ドラ100作目ヒロインに抜てき 2019年度前期放送「夏空－なつぞら－」 |url=https://mdpr.jp/news/detail/1729208 |publisher=[[modelpress|modelpress]]|date=2017-11-20|accessdate=2018-11-09|language=ja}}</ref>

2018年11月9日，NHK公布由廣瀨擔任第69回《[[NHK紅白歌唱大賽|NHK紅白歌唱大賽]]》紅組主持人。<ref name="2018-11-09">{{cite web |title=紅白歌合戦の司会者を発表 紅組・広瀬すず×白組・櫻井翔 総合司会は内村光良 |url=https://mdpr.jp/news/detail/1803767 |publisher=[[modelpress|modelpress]]|date=2018-11-09 |accessdate=2018-11-09|language=ja}}</ref>
