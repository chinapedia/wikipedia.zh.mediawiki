'''廣佛同城化'''或称'''廣佛一體化'''，是[[中华人民共和国|中国]][[广东省|广东省]]两个大型城市[[廣州|廣州]]與[[佛山|佛山]]進行城市一體化的相關措施。把廣州和佛山這個[[雙城|雙城]]合二為一，成爲[[廣州大都會區|廣州大都會區]]。

== 背景 ==
* 1919年，孫中山在其所著《[[建國方略|建國方略]]·實業計劃》中提出發展中國經濟的六大計劃。其中的第三計劃是建設「'''南方大港'''」，其要點是將廣州建設成為'''世界大港'''。孫中山把建設「南方大港」的位置選在[[黃埔|黃埔]]深水灣一帶，規劃建設一個由黃埔到佛山，包括[[沙面|沙面]]水路在內的「新廣州市」。新廣州市的商業區規劃在河南島（今[[海珠區|海珠區]]），工業區分佈在[[花地街道|花地]]至佛山之間，碼頭在後航道至黃埔一帶建設。
* 2008年4月16日廣州市社科院發佈《2008中國廣州經濟發展報告》，建議佛山放棄做廣東第三大城市的定位，承接廣州的影響力輻射，成為廣州的'''衛星城'''<ref>[http://cn.chinagate.cn/economics/2008-04/17/content_14970400.htm 广州社科院发报告建议佛山做广州卫星城]</ref>。
* 2009年有人提出“广佛合并”，即把佛山市整体并入广州市，原佛山五区由广州市管辖<ref>{{cite web |title=网友建议广佛合并超越北京上海 当中国第一城 |author= |url=http://gz.house.163.com/09/1015/08/5LLFLMBM00873C6D.html |publisher=网易，原载南方日报 }}</ref>。2011年省政协十届四次会议上，又有政协委员提出相关提案。<ref>{{cite web|title=广东省政协委员戴庆元建言“广佛合并”把佛山五区划入广州行政管辖|author=孔华, 赖南辉|url=http://gdjsb.ycwb.com/2011-01/26/content_3317608.htm|publisher=金羊网，原载广东建设报|deadurl=yes|archiveurl=https://web.archive.org/web/20160309234816/http://gdjsb.ycwb.com/2011-01/26/content_3317608.htm|archivedate=2016-03-09}}</ref>。

== 措施 ==
=== 交通 ===
1997年，[[廣州地鐵|廣州地鐵]]開通。其後佛山市政府曾一度發展其自己的地下鐵路網絡，但因资金问题一直未有實施。2004年12月，[[深圳地鐵|深圳地鐵]]也完成通車。佛山市方面才開始進行建造地鐵的工程，但維持不久又再不了了之。直至2008年，广州市政府决定由廣州地鐵公司作为出资方与佛山市政府共同投资，由广州承建佛山地鐵，第一期地鐵於2010年[[广州亚运|广州亚运]]前落成通車來往廣州和佛山，但名字上不是稱「佛山地鐵」，而是「'''[[廣佛地鐵|廣佛地鐵]]'''」。

在[[計程車|計程車]]方面，廣州和佛山兩地的計程車都可以互通。但由於地鐵的開通，加上價錢不菲，所以兩地之間的計程車很少能離開所屬市轄區。

=== 電訊 ===
廣州和佛山之間的電話通訊可以以本地電話的收費形式互通，但至今依然未有实质行动。
