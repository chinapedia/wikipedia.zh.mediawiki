'''广州地铁三號線列车'''是[[广州地铁|广州地铁]]的[[电动车组|电动车组]]车款之一，目前在[[广州地铁3号线|3号线]]运营，內部型號另称'''B1型'''。本列车是中国大陆首款营运时速为120km/h的地铁列车。

== 简介 ==
该批列车由[[南车株洲电力机车|南车株洲电力机车]]进行制造，与[[西门子|西门子]]合作设计生产。共生产120辆，40列（3节编组）。在2005年随[[广州地铁3号线|3号线]]运营开始时使用。

列车在运营初期以3节编组营运，后来由于编组过短造成乘车拥挤导致外界舆论压力才到2010年4月20日将3节编组连挂成6节编组。现时列车在[[機場北站_(廣州)|机场北]]-[[体育西路站|体育西路]]和[[天河客运站_(地铁)|天河客运站]]-[[番禺广场站|番禺广场]]均可运营。

现时列车全数配属于[[厦滘车辆段|厦滘车辆段]]和[[嘉禾车辆段|嘉禾车辆段]]，並不固定。

== 历史 ==
*2003年5月19日下午，[[广州地铁|广州地铁]]与[[西门子|西门子]]在[[白天鹅宾馆|白天鹅宾馆]]举行广州地铁三号线车辆采购项目的签约仪式，全数列车将在[[株洲电力机车|株洲电力机车]]生产。
*2005年11月18日，B1型于湖南株洲电力机车有限公司下线。<ref>[http://www.most.gov.cn/dfkjgznew/200512/t20051219_26927.htm 我国设计时速最快的地铁列车在湖南株洲下线]</ref>同年12月26日，首通段正式开通，列车以3节编组营运。
*2010年1月8日，03001号车在厦滘车辆段大修库启动自动架修工作，为国内首条最高时速120km/h地铁线路列车架修工作拉开序幕<ref>[http://gd.news.sina.com.cn/news/2010/01/07/779257.html 三号线首列车运行5年大体检]</ref>。目前，03001车已在2010年2月中下旬重新投入运营，其余列车将陆续进行大修。
*2010年4月28日，为配合2010年4月28日起3号线全線升级為六節，本列车由原来单列3节编组连挂成两列变成6节编组运营。

== 列车细节概要 ==
=== 外观 ===
沿用一号线[[安达-西门子模块化列车|安达-西门子模块化列车]]的梯型结构，车身涂装方面两边颜色以橙色为主，另各有一条白色饰带。本列车车门与4号线列车车门均乃广州地铁首次使用的[[內崁式門|塞拉门]]，有利于降低列车的风阻。

=== 内部 ===
每个单元车设6张长椅子，4张双人椅子并设有设备区。照明使用[[荧光灯|荧光灯光管]]及不透明的塑料燈罩。車窗上方配备4:3的显示器，播放地铁电视。车门旁两侧亦设有紧急开门装置，部分设有紧急求助按钮。

值得留意的是，由于列车是双列重联，C车与C车之间车厢不贯通，乘客如需从奇数C车（或偶数C车）前往偶数C车（或奇数C车），需要等列车停站开门时繞經站台前往。

==== 乘客信息显示系统 ====
所有列车车门顶部设有胶板，胶板顶部设有LED显示屏以中英双语显示运营信息（如终点站、下一站、可换乘线路等），底部则是3号线的路线图。

===== 改造 =====
[[File:Guangzhou_Metro_L3-CSR_B1_Train_Infornation_System.JPG|thumb]]
[[File:Guangzhou_Metro_L3-CSR_B1_Train_Map.JPG|thumb]]
在所有列车的乘客信息显示系统进行改造之前，列车车门上方均設單點式閃燈路綫圖，只有天河客運站—番禺廣場一段和體育西路—廣州東站一段。后期随3号线新车投放以及配合北延段通车，但由于北延段信号问题而无法載客運營，导致體育西路—廣州東站被覆蓋，不使用。

为了配合「Y」型交路實施，B1型全部40列（3节编组）列车已改造广播系统，涉及列车广播，改设膠板路線圖及LED显示屏。改造完成後，該種列車可以在3號綫全線（含北延段）營運。<ref>[http://www.gzzb.gd.cn/cms/wz/view/other/jgy.jsp?infoId=227936 广州市轨道交通三号线B1型车过三北线运营列车广播系统改造项目招标公告]</ref>

目前所有列車已全部改造完成。

== 列车编组 ==
列車以3节為一列，編組方式為A-B-C（= Mcp + T + Mcp =），3卡编组自3号线开通至2010年4月20日使用，2010年4月20日至今使用6节编组，編組方式為A-B-C-C-B-A（= Mcp + T + Mcp = Mcp + T + Mcp =）。现时在列车检修或大修时两节列车才会进行解挂。
