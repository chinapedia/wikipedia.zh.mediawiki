《'''推拿'''》（{{lang-en|'''See Without Looking'''}}），是2013年在[[中国大陆|中国大陆]]上映的一部电视剧，由中国导演[[康洪雷|康洪雷]]执导，演员[[濮存昕|濮存昕]]、[[张国强_(中国大陆演员)|张国强]]、[[李菁菁|李菁菁]]、[[刘威葳|刘威葳]]、[[高亚麟|高亚麟]]主演。剧情改编自作家[[毕飞宇|毕飞宇]]的同名小说，小说曾获[[茅盾文学奖|茅盾文学奖]]，故事讲述盲人推拿师们的励志故事。该剧于2013年8月15日登陆[[中央电视台综合频道|央视一套]]晚间黄金档。<ref>[http://ent.sina.com.cn/v/m/2013-08-15/16283988603.shtml 《推拿》今晚登陆央视 濮存昕恋上李菁菁]法制晚报.2013年8月15日.</ref>

== 剧情梗概 ==
讲述了一群在“沙宗琪”推拿院工作的盲人推拿师们故事。

== 製作 ==
=== 前期籌備 ===
導演康洪雷曾表示自己喜歡這部原著小說，2010年就曾稱會將《推拿》改編成電視劇，但因為劇本原因，項目遲遲沒有開機<ref name="改編研討會">{{cite web |url=http://gzdaily.dayoo.com/html/2011-12/15/content_1559066.htm |language= |title="电视剧《推拿》明年开拍 |publisher=广州日报 |date=2011-12-15 |accessdate= |deadurl=yes |archiveurl=https://web.archive.org/web/20120203092155/http://gzdaily.dayoo.com/html/2011-12/15/content_1559066.htm |archivedate=2012-02-03 }}</ref>。後來，康洪雷找到了他的老搭檔編劇陳枰改編劇本<ref name="開機新聞">{{cite web|url=http://ent.sina.com.cn/v/m/2012-03-08/09463575374.shtml | language= | title="电视剧《推拿》开机 濮存昕重回荧屏演盲人 |publisher=新浪娱乐 |date=2012-03-08 |accessdate= }}</ref>。編劇陳枰直言，因為劇本講述的內容，所以當初她並不看好把《推拿》做成電視劇“既沒有狗血也沒有分崩離析的段落，電視台很難買賬”<ref name="改編研討會"/>。但是她最終將18萬字的小說改成30集電視劇版內容<ref name="開機新聞"/>。2011年12月14日導演康洪雷出席《推拿》改編研討會，透露目前劇本第一稿已完成<ref name="改編研討會"/>，2012年2月7日《推拿》劇組正式成立，並於3月正式開機拍攝<ref>{{cite web |url=http://njrb.njnews.cn/html/2012-02/08/content_1157715.htm |language= |title="毕飞宇《推拿》开拍电视剧 |publisher=南京日报数字报刊 |date=2012-02-08 |accessdate= |deadurl=yes |archiveurl=https://web.archive.org/web/20140102191637/http://njrb.njnews.cn/html/2012-02/08/content_1157715.htm |archivedate=2014-01-02 }}</ref>。
=== 劇組安排 ===
《推拿》的主演在導演康洪雷的安排曾下幾次去北京盲人學校體驗生活<ref name="開機新聞"/>，學習盲人推拿，了解盲人的生活常態、技能訓練、盲杖的使用方法等<ref name="高亞麟">{{cite web|url=http://ent.sina.com.cn/v/m/2012-04-19/15143609817.shtml | language= | title="电视剧《推拿》火热开机 高亚麟首演弱视盲人 |publisher=新浪娱乐 |date=2012-04-19 |accessdate= }}</ref>。《推拿》劇組為了在外觀上讓演員更接近盲人，特意從曾負責《愛麗絲夢遊記》製作的美國好萊塢專業公司，定制了特殊的隱形眼鏡<ref name="劇組安排">{{cite web|url=http://ent.sina.com.cn/m/c/2012-03-14/15333580384.shtml | language= | title="濮存昕接演《推拿》赴盲校体验生活 |publisher=新浪娱乐 |date=2012-03-14 |accessdate= }}</ref>，這種隱形眼鏡不但能在視覺上起到仿真效果，還能讓演員一戴上它就看不清外面的事物，很方便主演們演绎盲人<ref>{{cite web|url=http://ent.sina.com.cn/v/m/2013-08-19/11403990586.shtml | language= | title="《推拿》编剧“潜伏”推拿院成半个专家 |publisher=新浪娱乐 |date=2013-08-19 |accessdate= }}</ref>。開機後，劇組還專門聘請一名推拿師，全程跟組指導演員的動作<ref name="劇組安排"/>。

=== 參演演員 ===
電視劇版《推拿》主創人員於2012年3月7日出席電視劇新聞發布會，宣佈濮存昕確定出演劇中推拿店掌門人“沙復明”<ref name="開機新聞"/>，張國強、劉威葳、王亞彬、繆俊傑、高亞麟飾演的盲人推拿師，而李菁菁、李博等飾演明眼人<ref name="高亞麟"/><ref>{{cite web|url=http://ent.sina.com.cn/v/m/2012-03-09/11083576401.shtml | language= | title="濮存昕张国强加盟《推拿》演绎盲人 |publisher=新浪娱乐 |date=2012-03-09 |accessdate= }}</ref>。

== 侵權事件 ==
2013年9月，《推拿》原小說作家畢飛宇與人民文學出版社聯合向北京市東城區法院提出訴訟，狀告電視劇版《推拿》編劇陳枰未經畢飛宇許可，擅自授權西苑出版社出版劇本《推拿》<ref name="侵權事件">{{cite web|url=http://ent.qq.com/a/20140401/020908.htm | language= | title="《推拿》作者毕飞宇：作家遭侵权不能总忍着 |publisher=腾讯娱乐 |date=2014-04-01 |accessdate= }}</ref>。2014年3月19日，北京市東城區法院一審判決畢飛宇勝訴：認定被告陳枰、西苑出版社侵犯畢飛宇《推拿》著作權，劇本《推拿》下架，賠償畢飛宇經濟損失5萬元<ref name="侵權事件"/>。編劇陳枰對一審判決不服，表示準備提起上訴<ref name="侵權事件"/>。同時人民文學出版社工作人員向記者表示，原小說作家畢飛宇得知結果但並不滿意，將和畢飛宇於15日內上訴<ref>{{cite web|url=http://ent.qq.com/a/20140320/014349.htm | language= | title="剧版《推拿》侵权案原著毕飞宇胜诉 获5万赔偿|publisher=腾讯娱乐|date=2014-03-20 |accessdate= }}</ref>。

== 收视率 ==
该剧首播后，收视率并不理想，但是网络上关于该剧的评价却相当之高，甚至有网友称这是一部零差评的电视剧。「叫好不叫座」的现象引起网友及业内人士的争议与思考。<ref>[http://ent.sina.com.cn/v/m/2013-08-30/10593998818.shtml 《推拿》“零差评”为啥搞不定收视率？ ]北国网-辽沈晚报.2013年08月30日.</ref><ref>[http://media.people.com.cn/n/2013/0830/c40606-22745790.html "零负评"电视剧《推拿》仅达央视收视达标线一半]北京晚报.2013年08月30日.</ref>
