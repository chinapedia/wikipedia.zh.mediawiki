'''豐田村'''，是[[台灣日治時期日本移民村|台灣日治時期日本移民村]]之一，位於[[台灣|台灣]][[花蓮港廳|花蓮港廳]][[壽庄|壽-{庄}-]]（現今[[花蓮縣|花蓮縣]][[壽豐鄉|壽豐鄉]]豐裡村、豐山村、豐坪村三村）<ref name="壽豐鄉文史館"/>。

== 沿革 ==
日治時期，台灣總督府曾在花、東一帶進行大規模的「移民政策」，從1911年到1924年間，在[[東台灣|東台灣]]建立了數處頗具規模的移民村，如[[吉野村|吉野村]]（今[[花蓮縣|花蓮縣]][[吉安鄉|吉安鄉]]）、豐田村（今[[花蓮縣|花蓮縣]][[壽豐鄉|壽豐鄉]]）、[[林田移民村|林田]]（今[[花蓮縣|花蓮縣]][[鳳林鎮_(臺灣)|鳳林鎮]]）、[[瑞穗移民村|瑞穗]]（今[[花蓮縣|花蓮縣]][[瑞穗鄉|瑞穗鄉]]）、[[鹿野村|鹿野]]（今[[臺東縣|台東縣]][[鹿野鄉|鹿野鄉]]）等等共十餘處移民村，這些移民村規劃完善，村內道路設計成寬敞有序的[[棋盤|棋盤]]狀，村內設有移民指導所、[[派出所|派出所]]、醫療所、[[神社|神社]]與佈教所，以及教育移民子弟的[[小學校|小學校]]等單位，其中豐田移民村是目前保存較為良好的一處<ref name="壽豐鄉文史館"/>。

豐田是日本在台灣所設的第二個移民村，日治時代被選作日本移民村的示範基地<ref name="豐田移民村"/>。豐田村始於大正2年(1913年)，舊地名為鯉魚尾，以水田多而土地肥沃出名，原為[[賀田金三郎|賀田金三郎]]之事業地，明治43年(1910年)因經營不善為[[臺東拓殖合資會社|臺東拓殖合資會社]]承繼，1911年開始由會社自熊本縣與宮城縣招募移民入住，墾殖百餘甲甘蔗園 <ref name="張素玢">{{cite book |last=張素玢 |first= |date= |title=《未竟的殖民-日本在臺移民村》 |url= |location= |publisher=衛城出版 |page= |isbn= |author-link= }}</ref>。大正元年(1911年)臺東拓殖合資會社與鹽水港株式會社合併後，以歸還之總督府許可地作為豐田村之用地，大正2年(1913年)移民開始入住。

移民村內設有豐田小學校（今豐裡國小）、移民指導所（今豐田社區中心）、「醫生的家」、豐田派出所廳舍（今壽豐文史館）、[[豐田神社|豐田神社]]（今碧蓮寺）<ref name="豐田移民村"/><ref>[http://books.google.com/books?id=_OtLAQAAIAAJ&q=%E4%B8%B0%E7%94%B0%E7%A7%BB%E6%B0%91%E6%9D%91&dq=%E4%B8%B0%E7%94%B0%E7%A7%BB%E6%B0%91%E6%9D%91&hl=en&sa=X&ei=B6adU7upLYeIogTs2oLQAg&ved=0CDkQ6AEwCQ 日據時期台灣總督府經濟政策研究: (1895-1945)P344]</ref>。
== 聚落 ==
豐田村位於知亞干溪出山口北側，至大正6年住戶約180戶人口912人，村中聚落可分為<ref name="張素玢"/> :
*大平
*中里 : 位於豐田村中央，移民指導所、醫療所、小學校、警察官吏派出所及布教所皆位於此。
*森本 : 以林木茂盛之神社而得名。
*山下
== 水路 ==
豐田圳在1916年開鑿幹線和支線，於1917年完工，是豐田村唯一的水利設施，其水源引知亞干溪水（今[[壽豐溪|壽豐溪]]），灌溉耕地600甲<ref name="豐田圳"/>。另外，[[臺灣總督府|台灣總督府]]從1915年起，分三年進行排水道修築工程，興建排水幹線和支線，以解決豐田村山下聚落積水問題<ref name="豐田圳"/>。1941年設[[東部發電廠溪口機組|溪口發電所]]，發電尾水接豐田圳供豐田移民村的灌溉民生用水<ref name="溪口發電廠小而美 支援灌溉有功"/>。
==組織==
豐田村的社會組織有 : <ref name="張素玢"/>
* 恆心會儲金 : 備不時之需與會員救濟。
* 豐田村水利組合 : 成立於大正6年(1917年)，分配灌溉用水與水渠修繕。
* 青年會 : 於農閑進行補習教育與協同耕作。
