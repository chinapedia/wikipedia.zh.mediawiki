'''豐中市'''（{{jpn|j=豊中市|hg=とよなかし|rm=Toyonaka shi|f=y}}）是位在[[大阪府|大阪府]]的北部，北攝地區裡的一個[[城市|城市]]，距離大阪市中心約只有十五[[公里|公里]]，是[[大阪都市圈|大阪都市圈]]近郊的一個[[衛星城市|衛星城市]]，西北部還有[[大阪國際機場|大阪國際機場]]，由於交通便利，在大阪府中為人口第五多的行政區劃，次於[[大阪市|大阪市]]、[[堺市|堺市]]、[[東大阪市|東大阪市]]、[[枚方市|枚方市]]。2001年升格成為[[特例市|特例市]]，2012年進一步升格成為[[中核市|中核市]]。<ref>{{Cite web| url=http://www.city.toyonaka.osaka.jp/joho/shoukai/miryoku/index.html| title=豊中の魅力| publisher=豐中市政府| accessdate=2019-01-07| language=ja}}</ref>

轄區北部為{{Tsl|ja|千里丘陵}}地勢較高，南部則為[[大阪平原|大阪平原]]，地層屬於洪積層結構，主要以砂石和粘土交互形成。轄區內除了位於大阪國際機場周邊的緩衝綠地以外，大部分區域都是已完成實施都市計劃的市街地區，最主要的綠地則是位於東部的{{Tsl|ja|服部緑地|服部緑地}}；另外由於南部的{{Tsl|ja|高川}}和{{Tsl|ja|天竺川}}由於歷史上曾多次發生洪水，在多次築堤之下，現在在部分河段已經成為了[[懸河|懸河]]。<ref>{{Cite web| url=http://www.pref.osaka.lg.jp/attach/4127/00100766/08h24_01shiryo4-2.pdf| title=淀川水系神崎川ブロック河川整備計画（変更原案）| publisher=大阪府| date=2012-06-12| page=3| language=ja| accessdate=2019-01-07}}</ref>

==历史==

現在的豐中市大部分區域在在令制國時代屬於[[攝津國|攝津國]][[豐島郡_(大阪府)|豐島郡]]，僅東北部區域屬於[[三島郡_(大阪府)|三島郡]]（8世紀後屬於[[岛下郡|岛下郡]]）；在文獻紀錄中，「豐島」的名稱最早出現在712年編撰的[[古事記|古事記]]中巻，之後的[[續日本紀|續日本紀]]中也有記載到769年[[稱德天皇|稱德天皇]]為承認豐島本地15為豪族的地位，曾賜[[姓_(日本政治結構)|姓]]他們。<ref name="History">{{Cite web|url=http://www.city.toyonaka.osaka.jp/joho/shoukai/history/history.html| title=豊中の歴史と歩み（市の歴史）| date=2016-01-28| publisher=豐中市政府| language=ja| accessdate=2019-01-07}}</ref>

8世紀至11世紀期間，此地曾先後設立了椋橋莊、利倉莊等[[莊園_(日本)|莊園]]，11世紀末期後，本地連同包括整個豐島地方大多成為[[藤原氏|藤原氏]]的私有領地；15世紀[[應仁之亂|應仁之亂]]中，本地開始陷入長期的戰亂，直到16世紀[[織田信長|織田信長]]的勢力到達攝津國之後，本地才隨著整個攝津國成為[[荒木村重|荒木村重]]的領地，但之後又一度因為荒木村重的叛亂而遭到戰火波及，但之後由於成為織田信長及後繼的豐臣秀吉統一全日本的主要據點而迅速復興。<ref name="History"/>

[[江戶時代|江戶時代]]此區域被分割得很零碎，大部分區域為幕府直屬或旗本的領地，[[岡田藩|岡田藩]]、{{Tsl|ja|飯野藩}}、[[麻田藩|麻田藩]]和{{Tsl|ja|淀藩}}在此也有領地，其中麻田藩的領地雖然大多不在此，但其藩廳{{Tsl|ja|麻田陣屋}}是設於現在豐中市西北部的螢池地區。<ref name="History"/>

19世紀[[明治維新|明治維新]]後，本地曾先後隸屬大坂裁判所司農局、大阪府司農局、大阪府北司農局、[[攝津縣|攝津縣]]、[[豐崎縣|豐崎縣]]、[[兵庫縣|兵庫縣]]、麻田縣、岡田縣、淀縣、深津縣，直到1871年第一次府縣整併才全數併入大阪府。

1889年實施[[町村制|町村制]]後，現在的豐中市轄區在當時分屬{{Tsl|ja|麻田村}}、'''豐中村'''、{{Tsl|ja|桜井谷村|櫻井谷村}}、{{Tsl|ja|熊野田村}}、{{Tsl|ja|中豊島村|中豐島村}}、{{Tsl|ja|南豊島村|南豐島村}}、{{Tsl|ja|庄内_(豊中市)|庄內_(豐中市)|-{庄}-內村}}、{{Tsl|ja|小曽根村|小曾根村}}、{{Tsl|ja|新田村 (大阪府)|新田村 (大阪府)|新田村}}共九個行政區劃；豐中村先在1927年升格改制為'''{{Tsl|ja|豊中町_(大阪府)|豐中町_(大阪府)|豐中町}}'''，1936年進一步與麻田村、櫻井谷村、熊野田村[[市町村合併|合併]]為'''豐中市'''，1947年至1955年期間又陸續併入其餘地區，確認了現今豐中市的轄區範圍。<ref name="History"/>

1950年代末期，當時豐中市的市街地約僅占兩成，其餘地區多為農地與山林；1960年代{{Tsl|ja|千里ニュータウン|千里新市鎮}}開始開發後，人口快速成長，產業迅速往商業、服務業發展，逐漸成為現在市街地面積超過七成，農地及山林僅餘下不到5%。

== 交通 ==

[[File:Osaka_International_Airport_Aerial_photograph_1985.jpg|thumb]]

大阪過去的主要機場[[大阪國際機場|大阪國際機場]]位於轄區的西北部，機場本身的東半部與主要航廈設施都位於豐中市轄區，但在[[關西國際空港|關西國際空港]]啟用後，這裡的航班數量銳減，但由於仍然具有距離大阪市中心非常近的優勢，仍為大阪市區內搭乘國內航線的首要選擇。

轄內最主要的軌道交通市南北貫穿整個轄區的[[阪急電鐵|阪急電鐵]][[寶塚本線|寶塚本線]]，往北可到[[池田市|池田市]]、[[川西市|川西市]]、[[寶塚市|寶塚市]]，往南可進入大阪市區；北部還有[[北大阪急行電鐵|北大阪急行電鐵]][[南北線_(北大阪急行電鐵)|南北線]]和[[大阪高速鐵道|大阪高速鐵道]][[大阪單軌電車線|大阪單軌電車線]]通過{{Tsl|ja|千里ニュータウン|千里新市鎮}}，最南端阪急電鐵[[神戶本線|神戶本線]]約有一公里的路段其實也通過轄內，但此區間並未設有車站。

高速公路部分，在南部的舊市區有[[名神高速道路|名神高速道路]]和[[阪神高速11號池田線|阪神高速11號池田線]]通過，後者因為可連接[[大阪國際機場|大阪國際機場]]，也曾被稱作「機場線」；在北部則有[[中國自動車道|中國自動車道]]通過。

轄內客運則以{{Tsl|ja|阪急バス|阪急巴士}}所經營的路線為主，阪急巴士的公司也設於轄內，{{Tsl|ja|阪急バス豊中営業所|阪急巴士豐中営業所}}也就設於公司本社所在處。

== 觀光資源 ==

[[File:151128_Hattori_Ryokuchi_Park_Osaka_pref_Japan13bs3.jpg|thumb]]

== 教育 ==

位於豐中市的高等教育學校包括位於轄區北部的[[大阪大學|大阪大學]]豐中校區和位於轄區南部的{{Tsl|ja|大阪音楽大学|大阪音樂大學}}；此外也有大量私立中小學，包括與曾經因日本首相[[安倍晉三|安倍晉三]]引起爭議的[[学校法人森友学园|学校法人森友学园]]設立瑞穗之國國紀念小學院一案，引起爭議的校地也是位於豐中市。

== 姊妹、友好城市 ==

=== 日本 ===

* [[沖繩市|沖繩市]]（[[沖繩縣|沖繩縣]]）
*: 1964年豐中市前往當時仍為[[美國統治琉球時期|美軍治理]]之下的[[胡差市|胡差市]]弔念二次大戰期間[[沖繩戰役|沖繩戰役]]中在當地戰死的豐中市出身的士兵，此後兩地市政府間開始密切交流，豐中市也開始支援胡差市職員的教育訓練；1972年沖繩管轄權從美國移交回日本，胡差市也在1974年[[市町村合併|合併]]為沖繩市，兩地遂於同一年11月3日締結為兄弟都市。<ref>{{Cite web| url=https://www.city.toyonaka.osaka.jp/miryoku/relife/okinawai_ikisatu.html| title=沖縄兄弟都市提携のいきさつと交流| date=2017-12-20| publisher=豐中市政府| accessdate=2019-01-09| language=ja}}</ref>
