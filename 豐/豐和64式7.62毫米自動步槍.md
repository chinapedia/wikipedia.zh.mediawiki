'''豐和64式步槍'''，簡稱「64式」，使用於[[日本自卫队|日本自卫队]]與[[日本海上保安廳|日本海上保安廳]]。氣動式[[戰鬥步槍|戰鬥步槍]]，64式使用[[7.62×51mm_NATO|7.62×51 NATO]]彈藥及可卸式的20發彈匣。由於日本法令的限制，64式從未輸出到其他國家，僅有日本使用。目前已經開始逐漸被較先進的[[豐和89式5.56毫米突擊步槍|89式]]取代，不過仍有大批服役於自衛隊與日本海上保安廳。

== 研發歷史 ==
日本自衛隊成立後決定自行設計一型適合日本士兵體型的自動步槍以取代當時使用的美製[[M1加兰德步枪|M1加兰德步枪]]。

主要設計師為銅金義一、横井由之介、岩下源蔵、津野瀬光男，由豊和工業負責開發。

64式步槍在測試靶場內的準確度表現優於美製[[M14自动步枪|M14自动步枪]]，不過部隊實際使用時反映準確度時好時壞非常不穩定，且由於過於複雜的內部結構導致過多的故障，使用的部隊對其可靠度大加懷疑。

64式步槍使用減量10%的[[7.62×51mm_NATO|7.62×51 NATO]]特製彈藥以滿足日本士兵對後座力的要求，同時在自動射擊時的{{Link-en|槍口上揚|Muzzle rise}}也大幅度的降低。


== 特徵 ==
* '''射擊方式'''：64式的發射模式為：ア（安全）→タ（半自動）→レ（全自動）。ア（アンゼンソウチ（安全装置）／安全／Safety device（Safety position））→タ（タンシャ（単射）／半自動／semi-auto) → レ（レンシャ（連射）／全自動／Full auto）或アタレ（Atare）;日文的Atare（当たれ）意為「命中目標」（Hit the target）。

== 狙擊型 ==
64式步槍可加裝光學式瞄準鏡成為狙擊步槍，不過由於鏡座設計不良只有一隻固定[[螺絲|螺絲]]，[[瞄準鏡|瞄準鏡]]極易移位，影響準確性。加上日本自衛隊的狙擊步槍與瞄準鏡集中使用，狙擊手沒有個人專屬的步槍、瞄準鏡，使得[[狙擊手|狙擊手]]無法控制瞄準鏡保持於歸零狀態。目前日本自衛隊狙擊步槍正處於換裝美製[[M24狙擊步槍|M24]]階段中。

== 目前發展 ==
由於預算的限制，日本海上保安廳與日本自衛隊的二线後備部隊仍舊使用64式步槍，第一线戰鬥部隊已換裝[[89式5.56毫米步槍|豐和89式]]步槍。部分日本自衛隊的成員認為64式步槍的傳統铣製鋼質槍身與木質[[槍托|槍托]]是過時與過重的設計。日本警察的[[特殊急襲部隊|特殊急襲部隊]]曾短暫的使用過64式作為狙擊步槍。
