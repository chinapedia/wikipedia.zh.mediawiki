'''豐原北號誌站'''位於[[台灣|台灣]][[臺中市|臺中市]][[豐原區|豐原區]]，為[[臺灣鐵路管理局|臺灣鐵路管理局]][[舊山線|臺中線（舊線）]]、[[東勢線|東勢線]]（已廢線）的鐵路號誌站，也是[[舊山線|臺中線（舊線）]]、[[東勢線|東勢線]]的實際分歧點。

== 車站構造 ==
*此站控管[[東勢線|東勢線]]合併入[[舊山線|臺中線（舊線）]]路軌的區間，往[[豐原車站|豐原]]的方向為雙線，往[[后里車站|后里]]及[[朴口車站|朴口]]的方向為單線。

== 利用狀況 ==
*東勢線廢線後改建成[[單車徑|自行車道]]（[[東豐自行車綠廊|東豐自行車綠廊]]）。
*臺中線（舊線）廢線後改建成[[單車徑|自行車道]]（[[后豐鐵馬道|后豐鐵馬道]]）。

== 車站週邊 ==
*[[東豐自行車綠廊|東豐自行車綠廊]]起點。
*[[后豐鐵馬道|后豐鐵馬道]]起點。
*舊山線[[大甲溪鐵橋|大甲溪鐵橋]]。

== 歷史 ==
*1959年1月12日：東勢線開始營運。
*1966年7月20日：本站開始使用。
*1991年9月1日：東勢線停止營業。
