'''豐里橋'''為一座[[臺灣|臺灣]][[臺東縣|臺東縣]][[臺東市|臺東市]]境內橫跨[[太平溪|太平溪]]的[[橋樑|橋樑]]，由[[臺東縣政府|臺東縣政府]]進行管理與維護，前身為[[台11線|台11線]][[省道|省道]]經過的主要主線橋樑，後於1998年，台11線進行外環道工程，[[交通部公路總局|交通部公路總局]]於太平溪靠出海口處另建新豐里橋，自此，豐里橋轉由臺東縣政府進行管理與維護。<ref>{{cite web |url=http://tour.taitung.gov.tw/zh-tw/Travel/ScenicSpot/488/%E8%B1%90%E9%87%8C%E6%A9%8B |title=台東縣政府 台東觀光網 豐里橋 }}</ref>

== 歷史 ==
{{fact|豐里橋最初建造年代目前尚無數據考察。}}{{fact|最早有紀錄時，為1965年（民國54年）6月18日，豐里橋遭到[[1965年太平洋颱風季#超級颱風_Dinah|黛納颱風]]災害沖毀。}}{{fact|1969年（民國58年），豐里橋重建完成。}}1990年（民國79年），因交通流量不足，進行第一次改建。<ref name="ttct">{{cite web |url=http://www.boe.ttct.edu.tw/front/bin/partprint.phtml?Part=11040010&Category=0&Style=1 |title=臺東縣政府 教育處 再見豐里橋 }}</ref>

原有豐里橋已使用42年，由於過於老舊，因此臺東縣政府配合[[太平溪整體整治計畫|太平溪整體整治計畫]]順勢編列計畫，進行第二次改建。<ref name="ttct"/>{{fact|其改建工程於2011年（民國100年）1月29日開工，工期500天。}}由於這次改建工程的影響龐大，為了減少[[交通黑暗期|交通黑暗期]]的不便，並因應鄰近學校學生上下學及附近居民一般生活等需求，臺東縣政府也於豐里橋上游50公尺處搭設4公尺寬之鋼便橋。<ref>{{cite web |url=http://www.cna.com.tw/postwrite/Detail/103697.aspx#.VDjSTvmSzxx |title=中央通訊社 太平溪豐里橋改建工程5月9日起將試通車 黃縣長感謝民眾及用路人配合 現鋼便橋將暫放供拍照 }}</ref>

== 諸元 ==
{{fact|豐里橋改建計畫為，拆除舊橋後，於原址重建三垮鋼橋乙座，改為橋寬20公尺橋兩端簡支鋼鈑樑垮度各24公尺，中間提籃式拱橋垮度80公尺鋼橋總長128公尺，兩端引道寬20公尺長約102公尺及其寬度輔附屬設施工程。}}
