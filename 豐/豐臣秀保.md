'''豐臣秀保'''（1579年－1595年5月24日／1594年6月4日）是[[安土桃山時代|安土桃山時代]]的武将和[[大名|大名]]。[[豐臣氏|豐臣氏]]家臣。父親是[[三好吉房|木下彌助（三好吉房）]]。母親是[[豐臣秀吉|豐臣秀吉]]的姐姐[[瑞龍院|瑞龍院]]，後來成為[[豐臣秀長|豐臣秀長]]的養子。因為是[[大和國|大和國]]領主，而官位是[[中納言|中納言]]，因為通稱'''大和中納言'''。

== 生平 ==
在[[天正_(日本)|天正]]7年（1579年）出生，家中三男。幼名是'''辰千代'''。

因為是[[豐臣氏|豐臣氏]]的血脈，所以在幼少時已經不斷昇進。天正19年（1591年）1月，因為叔父[[豐臣秀長|秀長]]沒有繼嗣，於是與秀長年約4至5歳的女兒結婚並成為秀長的養子。同月，因為秀長死去而繼承並成為[[郡山城|郡山城]]城主。同時領有[[大和|大和]]、[[紀伊|紀伊]]兩國，[[後見役|後見役]]是[[藤堂高虎|藤堂高虎]]和[[桑山重晴|桑山重晴]]。同年被任命為[[從四位下|從四位下]][[参議|参議]]並被下賜豐臣姓。

在[[文祿|文祿]]元年（1592年）的[[文祿之役|文祿之役]]中率領1萬5千名士兵在[[名護屋城|名護屋城]]参陣，不過自身沒有渡海，而是在名護屋城下建起[[陣屋|陣屋]]並逗留2年。而配下的[[紀伊國|紀伊國]][[國人眾|國人眾]]（[[堀內氏善|堀內氏善]]）和藤堂高虎、[[桑山元晴|桑山元晴]]、[[桑山一晴|一晴]]、[[杉若氏宗|杉若氏宗]]、[[杉若無心|無心]]等人以水軍身份渡海參戰。還有[[本多俊政|本多俊政]]在[[壹岐|壹岐]][[勝本城|勝本城]]率領5百名士兵擔任在番，為朝鮮渡海軍在海上輸送兵站物資和維持島上治安。

在文祿2年（1593年）閏9月25日與藤堂一同從名護屋返回[[下關|下關]]並前往[[京都|京都]]，於同年成為[[從三位|從三位]][[權中納言|權中納言]]，以後被稱為'''大和中納言'''、'''郡山中納言'''。文祿3年（1594年）2月，在[[吉野|吉野]]的櫻花觀賞會中與[[豐臣秀吉|秀吉]]、[[豐臣秀次|秀次]]、[[菊亭晴季|菊亭晴季]]等人一同舉行[[和歌會|和歌會]]。

在文祿4年（1595年）4月16日死去。享年17歲。戒名是'''瑞光院華嶽春英'''（『[[系圖纂要|系圖纂要]]』）。

== 死因 ==
在俗説中有許多橫死的說法，例如在『[[武德編年集成|武德編年集成]]』中記述了惡評後，在[[吉野川|吉野川]]上流為了測試[[小姓|小姓]]的勇氣而命令他跳崖，小姓在秀保跳下後跟著跳下，最後雙雙溺死，不過此事的日期被記錄為文祿3年，因此在時間上顯得不合理。

除了前述的死因外，根據[[桑田忠親|桑田忠親]]的研究，由同時代的[[駒井重勝|駒井重勝]]的『[[駒井日記|駒井日記]]』等史料中，可以得知秀保在[[文祿|文祿]]4年（1595年）4月為了養病而前往[[大和國|大和國]][[十津川|十津川]]，不過在4月10日期間被發現身上有一種[[麻疹|麻疹]]的疱瘡，於是在13日晚上向[[吉田淨慶|吉田淨慶]]和16日晚上向[[曲直瀨正琳|曲直瀨正琳]]等醫師求醫，於14日曾一時間回復，但是病情在15日早上再度惡化，最後在16日死去。

秀保迎娶了表親阿菊（「{{lang|ja|おきく}}」，秀長的女兒）為正室，不過沒有誕下子女，曾經成為秀長養子的[[藤堂高吉|仙丸]]亦已經成為了[[藤堂高虎|藤堂高虎]]的養子並改名為藤堂高吉，於是令大和豐臣家斷絕。

== 人物逸話 ==
;與哥哥[[豐臣秀次|豐臣秀次]]同様有「'''殺生關白'''」的惡名。但與與秀次的風評同様，這個惡名不是事實的可能性被認為相當高。

*被評為以殺人為嗜好等，不斷重複著無道行為的暴君。

*秀保捕捉快要生產的婦人並命令「割開腹部讓我看看胎兒吧」。這個婦人感到驚恐，於是以削髮為尼的方式希望秀保能放過自己，不過秀保沒有接受，因此這個婦人在割開腹部並取出胎兒後就跳入池中身亡。

*秀保命令忍術師「在這個池中叫出大蛇來看看」，忍術師在唱出呪文後，空池被裝滿水，大蛇出現後像要把秀保呑下肚，於是秀保非常驚慌並逃回[[郡山城|郡山城]]。因此家臣為了讓大蛇不要再出現而用大石埋下池底的洞穴。
