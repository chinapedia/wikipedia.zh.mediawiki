'''豐田線'''（{{jpn|j=豊田線|hg=とよたせん|rm=Toyota sen|f=y}}）為連接從[[愛知縣|愛知縣]][[日進市|日進市]]之[[赤池車站_(愛知縣)|赤池站]]至愛知縣[[豐田市|豐田市]]之{{tsl|ja|梅坪駅|梅坪站}}的[[名古屋鐵道|名古屋鐵道]]（名鐵）路線。在本線剛開通時，曾稱作'''豐田新線'''。

本路線亦為通往[[名古屋市|名古屋市]]或是前往[[豐田市|豐田市]]之通勤、通學路線。另外，在沿線有許多的大學院校，因此有許多人利用本路線來通學。全線為立體交叉之路線，雖沒有設立[[平交道|平交道]]，本路線並非全為高架行駛模式，沿途中通過不少丘陵地形，因此有不少隧道穿插其中。

此外，雖然本線正式之起點為赤池站，但在列車行進與車內廣播中，由梅坪站往赤池站方向的列車均為下行列車，反方向則為上行列車。
== 路線資料 ==
* 路線距離（{{link-ja|營業距離|営業キロ}}）：15.2公里
* [[軌距|軌距]]：1067毫米
* 站數：8個（包括起終點站）
* 複線路段：全線
* 電氣化路段：全線（直流1500V）
* [[閉塞_(鐵路)|閉塞方式]]：自動閉塞式
* 最高速度：100公里/小時
* 最大坡度：34.5‰（三好丘站－黑笹站間）
* 軌條：50kgN（開通當初一部以20m軌道鋪設）
* 主要橋樑、隧道
** 日進車庫跨線橋（赤池－日進間、全長487.7公尺、{{link-ja|預力混凝土橋|プレストレスト・コンクリート橋|PC桁}}、鋼合成桁）<ref name="Rp370_p18">{{Cite journal|和書|author = 山下武|title = 豊田新線の計画と建設について|date = 1979-12 |publisher = 電気車研究会 |journal = 鉄道ピクトリアル |volume = 370|page=18}}</ref>
** 愛知池橋樑（米野木－黑笹間、全長256公尺、PC桁）<ref name="Rp370_p18"/>
** 東名高速架道橋（米野木－黑笹間、全長49.6公尺、{{link-ja|Truss橋#Pratt|トラス橋#プラットトラス|鋼Pratt Truss}}）<ref name="Rp370_p18"/>
** 福谷隧道（三好丘－淨水間、505公尺）<ref name="Rp370_p18"/> - 名鐵最長的山岳隧道<ref>{{Cite journal|和書|author = 高橋大介|title = 線路と保線|date = 2009-03 |publisher = 電気車研究会 |journal = 鉄道ピクトリアル |volume = 816|page=101}}</ref>。

== 營運型態 ==
基本上[[三河線|三河線]]與[[名古屋市營地下鐵鶴舞線|名古屋市營地下鐵鶴舞線]]採直通運行方式，於豐田市站至[[上小田井車站|上小田井站]]之間來回運行。但是在平常日幾乎所有時段，包括周六、假日的早晨與深夜時段都有往[[名鐵犬山線|名鐵犬山線]][[犬山車站|犬山站]]之直通列車。在白天時段毎小時雖有4趟班車（平日每隔15分鐘、周六假日每隔13或17分鐘），在早晨傍晚時段均會加開班車。在周六假日時，因地下鐵鶴舞線每隔10分鐘均有1班車次，所以為了調整運行間隔，有過半數的直達車都會在赤池站停約3分鐘。另外，由於赤池站為[[名古屋市交通局|名古屋市交通局]]所管理之車站，所以本站列為無站務員管理之車站。
