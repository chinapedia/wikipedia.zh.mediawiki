{{nihongo|'''96式40毫米自動榴彈發射器'''|96式40mm自動てき弾銃|96-shiki 40-miri jidōtekidanjū}}是一挺由[[日本|日本]]{{Link-en|豐和工業|Howa}}自1996年開始研製和生產的40毫米[[榴弹发射器#全自動型榴弹发射器|自動榴弹发射器]]，發射[[40毫米榴彈|40×56毫米榴彈]]。

== 歷史 ==
知道美軍裝備的[[Mk_19自動榴彈發射器|Mk 19自動榴彈發射器]]在實戰中提供的面殺傷支援能力以後，[[日本|日本]][[陸上自衛隊|陸上自衛隊]]也計劃裝備類似武器。而針對自衛隊對於使用[[住友62式7.62毫米機槍|住友62式]][[通用機槍|通用機槍]]和[[白朗寧M2重機槍|住友M2HB]][[重機槍|重機槍]]兩款重火力支援武器以外的需要，豐和工業公司於1996年自行研製和生產了96式40毫米自動榴彈發射器，並從1996年開始陸續裝備陸上自衛隊，用作車載武器。另外，96式自動榴彈發射器必要時還可以安裝在M2HB重機槍的[[M3三腳架|M3]][[三腳架|三腳架]]之上，藉而架設在地面上使用。<ref name="History">{{cite web|url=http://www.masdf.com/spec/land/japan/96apc.html|title=96式装輪装甲車|language=ja|accessdate=2009-01-13}}</ref><ref name="spearhead11">96式自動てき弾銃 荒木雅也 スピアヘッドNo.11 P80-85 アルゴノート社</ref>豐和96式自動榴彈發射器主要裝備陸上自衛隊[[步兵|步兵]]（日本稱為普通科）部隊的車輛，而住友M2HB重機槍則裝備[[装甲兵|装甲兵]]（日本稱為機甲科）以及中央快速反應部隊（中央即應集團）的車輛。

== 設計及操作細節 ==
豐和96式採用彈鏈供彈。在豐和96式的左側是一個整合式供彈托架，讓40毫米榴彈可以順暢地為該全自動型榴彈發射器裝填。<ref>{{cite web|url=http://militaryroom.chips.jp/newpage39.html|title=９６式４０㎜自動てき弾銃|accessdate=2009-01-13|archiveurl=https://web.archive.org/web/20110722084034/http://militaryroom.chips.jp/newpage39.html|archivedate=2011-07-22|language=ja|deadurl=yes}}</ref>扳機組件和握把安裝在[[機匣|機匣]]尾端，扳機造型類似[[白朗寧M2重機槍|白朗寧M2重機槍]]的蝶狀扳機，不過握把並非垂直設置。平時，握把呈“一”字形，使用時將握把向下旋轉，使其呈“八”字形。在機匣後部左右兩側均設置有[[擊發調變槍械|快慢機]]柄，具有保險、半自動、全自動三檔位置。容量為50發的彈鏈裝載箱掛在發射器左側。其使用與Mk 19類似的立框式表尺照門，表尺最大射程為1,500公尺（1,640.42码，4,921.26英尺，0.93英里））。

豐和96式發射由[[大金工業|大金工業]]所研製和生產的[[40毫米榴彈|40×56毫米]]「破甲及殺傷兩用榴彈」，[[彈藥|彈藥]]全長112毫米（4.41英吋），重量為371[[克|克]]（13.09[[盎司|盎司]]），其與美軍Mk 19自動榴彈發射器所使用的40×53毫米榴彈並不通用。<ref name="spearhead11"/>該榴彈發射器也可發射「40毫米訓練榴彈」。

== 使用 ==
豐和96式可被[[步兵|步兵]]和[[装甲车|裝甲車輛]]所使用，前者會架設在[[三腳架|三腳架]]上使用，而後者則架設在一個[[武器架座|武器架座]]上。<ref name="Type 96"/>

它最突出之處就是它是[[96式裝甲運兵車|96式]][[装甲运兵车|装甲运兵车]]「A型」搭載的主要武器之一（「B型」則是白朗寧M2）。<ref name="History"/><ref name="spearhead11"/><ref>{{cite web|url=http://tag.vis.ne.jp/jsdf/soubi/jgsdf/96teki.html|title=９６式40mm自動てき弾銃|language=ja|accessdate=2009-01-27}}</ref>

在平成23年度（2011年）富士綜合火力演習之中，出現了追加[[紅點鏡|紅點鏡]]的型號。<ref name="spearhead11"/>
