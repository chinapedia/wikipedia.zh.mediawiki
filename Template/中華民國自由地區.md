# 「自由地區」是指[[中華民國政府]]的有效統治範圍，又稱「臺澎金馬」或「臺灣地區」。
# 依照《[[中華民國憲法增修條文]]》與《[[s:地方制度法|地方制度法]]》之規定，省虛級化改為非地方自治團體。
# 代管[[海南特別行政區]][[東沙群島]]與[[南沙群島]]。
# 代管[[莆田縣]][[烏坵鄉]]。
# 僅轄[[馬祖列島]]，常稱作「馬祖」。代管羅源縣東引鄉與長樂縣莒光鄉。
