==快捷碼：请求保护页面==
;已完成
# semi 或 s（半保护）
# full 或 p（全保护）
# move（移动保护）
# salt 或 t 或 create （白纸保护）
# done 或 do （完成）
# unpr 或 u （已解除）
# ispr 或 ap （已保护）
# isun 或 au （已解除保护）
;拒绝请求
# deny 或 decline （拒绝请求）
# nact 或 nea （可通过其他办法解决）
# npre （不会做出预见性的保护）
# mpfa 或 fa （首页的特色条目不会被保护）
# disp 或 dr 或 cont 或 dispute （考虑争论的解决）
# usta 或 ut （用户讨论页不会被保护）
# bloc 或 b （用户已被封禁）
# tabl 或 tb 或 rb （用户已被重新封禁）
# noun 或 nu （未解除）
# nucr （请在子页面创建条目有可靠来源的版本）

==快捷碼：破坏==
# i 或 inactive 或 old 或 ins （近期編輯未足以用於判斷是否封禁用戶）
# stopped 或 stop 或 nfw （最近警告后无破坏）
# mon 或 monitor （将关注此用户）
# an3 或 3rr 或 anew 或 ew （提报3RR）
# a 或 complex 或 ani （提报争议）
# u 或 uaa （提报不妥当用户名）
# rpp （提报保护页面）
# sp 或 ssp 或 spi （提报CU）
# np 或 notpunish （封禁乃用以预防破壞而非惩罚）
# nv 或 notvandal （编辑並非破坏）
# del （相关页面已删除）
# e 或 sr 或 stale （用户并未编辑相关页面）
# sn 或 nowstale （用户最后编辑距今已有较长时间）
# disp 或 dr 或 cont 或 dispute （考虑争论的解决）
# b 或 bloc （封禁）
==快捷碼：上诉==
# re （恢复）
# deny 或 decline （未恢复）

# done 或 do 或 ok （完成）
# deny 或 decline （未完成）

# rb （已是回退员）
# grb （已是全域回退员）

# ipb （已是IP封禁例外者）
# cu （已是確認用戶）
# acu （已是自动確認用戶）
