'''上瓦爾登州'''（[[德語|德語]]：Obwalden;[[法語|法語]]：Obwald;[[意大利語|意大利語]]：Obvaldo;[[羅曼語|羅曼語]]：Sursilvania）是[[瑞士|瑞士]]中部的一個州。人口33,330，首府位於[[薩爾嫩|薩爾嫩]]（Sarnen）。傳統上是個「半州」，但憲法中已無此分類。

== 地理 ==
* 最高點：[[鐵力士峰|鐵力士峰]]（Titlis），海拔高度3238米。
* 最低點：[[琉森湖|琉森湖]]（Vierwaldstättersee），海拔高度434米。
* [[湖泊|湖泊]]：州境包含小部份的[[琉森湖|琉森湖]]，完全在境內的湖有[[薩爾嫩湖|薩爾嫩湖]]（Sarnersee）及[[龍疆湖|龍疆湖]](Lungerersee) 。

== 歷史 ==
上瓦爾登州境內最早的聚落是[[凱爾特|凱爾特]]人和[[古羅馬人|羅馬人]]。[[阿勒曼尼|阿勒曼尼]]人在公元[[八世紀|八世紀]]開始移居此地。1291年，由上瓦爾登及[[下瓦爾登半州|下瓦爾登]]組成的[[翁特瓦爾登|翁特瓦爾登]]與[[施維茨州|施維茨]]（Schwyz）及[[烏里州|烏里]]（Uri）為抵抗[[哈布斯堡王朝|哈布斯堡王朝]]，於[[琉森湖|琉森湖]]畔的[[呂特利|呂特利]]（Rütli）締結盟約，[[瑞士|瑞士]]邦聯從此誕生。[[十三世紀|十三]]及[[十四世紀|十四世紀]]期間，上瓦爾登州擁有自己的政府，並與[[下瓦爾登半州|下瓦爾登]]有一個聯合議會。

1403年上瓦爾登為了擴張領土去建立[[乾酪|乾酪]]及[[牲畜|牲畜]]的市集，遂與[[烏里州|烏里州]]一起侵略勒文蒂那地區（Leventina，即現在的[[提契諾州|提契諾州]]）。[[十六世紀|十六世紀]]初，很多上瓦爾登人去當僱傭兵，因此累積了大筆財富。1798至1803年間，[[拿破崙|拿破崙]]佔領[[瑞士|瑞士]]並建立[[赫爾維蒂共和國|赫爾維蒂共和國]]，上瓦爾登失去了獨立地位，但人民首次能享有基本的政治權利。1815年，英格爾堡（Engelberg）修道院和同名的市鎮加入上瓦爾登州。

== 政治 ==
上瓦爾登在[[瑞士|瑞士]]邦聯內是一個半州，能享有和其他全州一樣的權利與義務，與別不同的地方是半州在[[瑞士聯邦議會|瑞士聯邦議會]]中只能有一名議員。由於上瓦爾登面積很小，人口又稀少，所以州政府內只有五名官員。

== 經濟 ==
中小型企業主導著上瓦爾登的[[經濟|經濟]]，行業包括小型機械、合成材料、醫療設備及[[納米技術|納米技術]]等範疇。

傳統行業仍然很重要，尤其是伐木業和農業。上瓦爾登的農業專注於蓄牧業，供應奶類及肉類。多數農莊是家族經營。

[[旅遊|旅遊]]業是另一經濟支柱，本地企業和居民都受惠於旅遊業的設施，約有四分之一人口直接或間接從事旅遊業。

== 文化 ==
當地團體致力保護上瓦爾登的傳統文化，例如是傳統音樂、舞蹈、服飾、戲劇、節日及慶典等。上瓦爾登出了一些現代藝術家例如Josef Garovi（作曲家）, Caspar Diethelm（作曲家）, Julian Dillier（詩人）, Franz Bucher（畫家）, Kurt Sigrist（雕塑家）及Alois Spichtig（雕塑家）。

== 行政區劃 ==
[[File:Karte_Gemeinden_des_Kantons_Obwalden.png|thumb]]
上瓦爾登半州由於面積太小，所以沒有區級劃分，而轄下有七個市級行政區：
*[[薩爾嫩|薩爾嫩]]（Sarnen）
*[[克恩斯|克恩斯]]（Kerns）
*[[薩希塞恩|薩希塞恩]]（Sachseln）
*[[阿爾普那克|阿爾普那克]]（Alpnach）
*[[吉斯韋爾|吉斯韋爾]]（Giswil）
*[[龍疆|龍疆]]（Lungern）
*[[恩格尔贝格|恩格尔贝格]]（Engelberg）上瓦爾登的市級行政區享有很大程度的自治權，三分之二的稅收直接交到市政府手上。
