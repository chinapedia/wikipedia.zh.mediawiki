'''上茄苳顯濟宮'''位於臺灣[[臺南市|臺南市]][[後壁區|後壁區]]，是上茄苳的庄廟，主祀[[保生大帝|保生大帝]]（保生三真人）{{r|李}}{{r|周}}。廟中有「澤被生民」與「保我後生」兩塊古匾{{r|李}}。

== 沿革 ==
該廟的信仰源流據說是康熙年間{{NoteTag|李麗菁《臺1線的鄉愁 從上茄苳到二層行》寫說為康熙廿八年（1689年）{{r|李}}。}}賴國永、賴國福等人來臺到「北隙」一帶拓墾時，自家鄉迎奉的保生二大帝孫真人及中壇元帥{{NoteTag|賴國永、賴國福是福建漳州府南靖縣永豐里梧宅社賴仕勇公之後的第十代{{r|李}}{{r|周}}。}}{{r|李}}{{r|周}}{{r|縣}}。後來因為「上茄苳埤」常崩堤，北隙常常淹水，遂在雍正六年（1728年）南遷上茄苳{{NoteTag|據說南遷上茄苳的原因是保生二大帝降乩指示上茄苳北東南三面環山，西面可遙望唐山的祖家，為一風水寶地{{r|周}}。}}{{r|周}}。

到了嘉慶十年（1805年），正式興建廟宇供奉{{r|李}}{{r|周}}，工程在二年後（1807年）完成{{r|縣}}。而後又歷經多次擴建重修，逐漸擴大規模{{r|李}}。

二次大戰後，民國四十二年（1953年）由周邊13庄{{NoteTag|詔安厝、新岩腳、北埔、木屐寮、三角仔、蓮潭、虎仔墓、檨仔林、杭內、內洲仔、十七戶、上茄苳{{r|李}}。}}再次募資重建，之後在民國六十六年（1977年）翻修廟蓋，民國七十七年（1988年）增修鐘鼓樓，民國八十七年（1988年）將廟蓋翻修為混擬土，並增設金爐{{r|李}}。民國九十六年（2007年）首次舉行「刈水火活動」，前往白河北埔刈水火招軍旗，是要讓溪面漂流的無主靈魂投入保生大帝旗下修行，待功果圓滿後便可投胎轉世{{r|李}}。

== 祀神 ==
上茄苳顯濟宮所供奉的「保生三真人」分別是保生大大帝[[保生大帝|吳真人]]、保生二大帝[[孫思邈|孫真人]]與保生三大帝[[許遜|許真人]]{{r|周}}。

== 傳說 ==
據說在嘉慶十年（1805年）左右，因為[[分類械鬥|分類械鬥]]、盜賊侵入與瘟疫蔓延等問題，賴國永之曾孫賴開捷乃祈求保生二大帝與中壇元帥指點迷津{{r|周}}。而在五月初旬某夜，保生大大帝雲遊至上茄苳，見居民誠心祈求而降乩，並以符水治病{{r|周}}。而後八月中旬保生二大帝降駕指示保生大大帝與保生三大帝將選在上茄苳莊駐驆並護境祐民，十一月時賴姓家族討論神明增置問題時，賴開捷提議建築公廟奉祀{{r|周}}。之後在保生大大帝指示下，於今址興建了顯濟宮{{r|周}}。
