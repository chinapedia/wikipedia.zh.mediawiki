'''上圯乡'''是[[江西省|江西省]][[泰和县|泰和县]]下辖的一个[[乡|乡]]，位于泰和县东南部山区，全乡面积90.3平方公里，人口8千余人。大型水库[[老营盘水库|老营盘水库]]位于境内。

== 交通地理 ==
上圯乡位于泰和县东南部山区，[[319国道|319国道]]和[[京九铁路|京九铁路]]穿境而过，距县城40公里，距[[井冈山机场|井冈山机场]]47公里。全境主要为中、高[[丘陵|丘陵]]地形，[[云亭河|云亭河]]（缯水）自东南向西北纵贯中部，在境内建有大（二）型水库老营盘水库。林木资源丰富，盛产竹木、油茶。

== 沿革 ==
上圯乡辖境1950年为回龙、上圯、西岗等乡，1956年合并为回龙乡，1958年小庄乡并入、合设回龙人民公社，1968年更名为老营盘人民公社。1984年老营盘公社撤消，分设老营盘乡（现[[老营盘镇|老营盘镇]]）和上圯乡<ref>李宝库主编：《中国政区大典》第2册，杭州：浙江人民出版社，1999年，ISBN 7-213-01789-6，第1186页</ref>。

== 行政区划 ==
上圯乡现辖7个[[村委会|村委会]]：北坑村、石田村、上圮村、芫头村、西岗村、回龙村、沔口村<ref>〈[http://www.jxqhdm.com/index-ja.aspx 2009年泰和县行政区划] {{webarchive|url=https://web.archive.org/web/20091025153207/http://www.jxqhdm.com/index-ja.aspx |date=2009-10-25 }}〉，吉安区划地名网（江西区划地名网子网站），于2009年4月1日查阅。</ref>。
