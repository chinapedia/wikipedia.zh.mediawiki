{{中国菜}}

'''上海菜'''，简称'''沪菜'''，而'''本帮菜'''狹義即指上海菜（[[中國菜系|菜系]]又稱「幫菜」，若以上海人為視角，自然稱上海菜為「本幫菜」），为发源于上海本地的一种菜系，广义的上海菜是以「本帮菜」为主吸收各派之长形成的综合性、广泛性的菜系，特別是[[江|江]][[浙|浙]]一帶料理與上海菜有時難以二分<ref name="厦门日报">{{cite web |url = http://www.china.com.cn/zhuanti2005/txt/2002-12/25/content_5252259.htm |title=花样年华上海菜 |publisher=厦门日报|author=}}</ref>。上海菜以红燒、生煸見長，濃油赤醬为典型特征，吸取了[[杭州菜|杭州菜]]、[[宁波十大名菜|寧波菜]]、[[徽菜|徽州菜]]、[[淮扬菜|淮扬菜]]、[[蘇錫菜|蘇錫菜]]等'''「江浙菜系」'''的做法，借鉴了西菜、西点的做法，挖掘其他帮别菜肴之长，逐渐形成兼容并蓄且不失开拓创新的海派风格菜肴，主要可以总结为湯卤醇厚，濃油赤醬，糖重色豐，咸淡適口<ref name="蔡培谊">{{cite web|url=http://newspaper.jfdaily.com/xwwb/html/2012-10/18/content_901336.htm |title=糟钵斗 |publisher=新民晚报 |author=蔡培谊}}</ref>。選料注重活、生、寸、鲜；调味擅長咸、甜、糟、酸。

广义上说，上海菜主要由发源于上海本土的本帮菜、[[海派西餐|海派西餐]]，以及改良自上海的「海派粤菜」、「海派川菜」、以及江浙等菜系所组成，这些菜系在上海餐饮中都占有较大的市场份额<ref name="中国东方文化研究会">{{cite web |url = http://www.chinesefolklore.com/news/news_detail.asp?id=509|title=上海都市饮食的泛民俗现象及其思考|publisher=中国民俗网|author=中国东方文化研究会}}</ref>。

==历史沿革==
[[南宋|南宋]]末年，上海十六铺出现了酒肆。清初，[[十六铺|十六铺]]洋行街（今[[黄浦区|黄浦区]][[小东门_(上海)|小东门]]阳朔路）有菜馆六七家。上海开埠后，徽、苏锡、宁帮随商人来沪开设菜馆。<ref name="地方志1">{{cite web|url=http://www.shtong.gov.cn/newsite/node2/node2247/node79045/node79394/node79408/userobject1ai104452.html|title=酒菜行业|publisher=上海地方志|author=地方志}}</ref>

===十六帮别闹申城===
1843年上海开埠后，「十六帮别闹申城」，本、宁、徽、京、川、苏锡、镇扬、豫、鲁、湘、津、粤、闽、清真、素菜等十六总不同的餐饮流派汇聚上海摊。最早在上海流行起来的是安徽菜，<ref name="看看新闻"/>随后苏锡菜、粤菜、淮扬菜和京菜逐渐兴盛，1930年代苏锡菜馆几占上海酒菜馆之半，而广东大菜则成为了上海番菜的代名词，深得上海市民及外侨的喜爱，<ref name="周松芳">{{cite web|url=http://www.21cbh.com/2012/nfcity_820/284677.html|title=海派粤菜新时代|publisher=南方都市报|author=周松芳|deadurl=yes|archiveurl=https://web.archive.org/web/20140903190526/http://www.21cbh.com/2012/nfcity_820/284677.html|archivedate=2014-09-03}}</ref>[[新雅粤菜馆|新雅粤菜馆]]、[[上海杏花楼酒家|上海杏花楼酒家]]等海派粤菜名店成为了上海中菜馆中的头牌并传承至今。此外，[[国际饭店|国际饭店]]、燕云楼的京菜；[[扬州饭店|扬州饭店]]、老半斋的淮扬菜；梅龙镇酒家、四川饭店、洁而精、绿杨村的海派川菜；[[老正兴|老正兴]]的苏锡菜；大富贵酒楼的徽帮菜；[[洪长兴|洪长兴]]的清真菜；[[功德林|功德林]]的素菜等皆为上海餐饮名店。

===上海本帮菜===
{{main|本帮菜}}
本帮起源于本地家常菜，以家常、平民化为自身的特色，常用的烹调方法以红烧、煨、糖为主，烹饪上善于用糟，以浓油赤酱、咸淡适中、醇厚鲜美著称。早期的本帮菜更是以价廉、味重、量足为鲜明特点，圈子、腌笃鲜、黄豆汤这些平民原料，皆为本帮菜的生力军。最早的本帮菜源于浦东三林人，于清嘉庆光绪年间在上海老城区开设的饭摊，称作“铲刀帮”。1930年后，上海工商业快速发展，本帮菜的顾客变成以公司职员为主的新兴阶层，廉价菜在本帮菜中的比例开始下降。<ref name="看看新闻">{{cite web|url=http://shanghai.kankanews.com/c/2014-05-04/0014691475.shtml|title=看懂上海：上海本帮菜|publisher=看看新闻网|author=看看新闻}}</ref>

===海派西餐===
{{main|海派西餐}}
海派西餐为[[上海|上海]]特有的、中西结合的一种菜式流派，是[[上海人|上海人]]按照自身的口味及本地食材的特性自己创造出来的[[西餐|西餐]]。<ref name="lp">{{Cite book |title = 话说沪商 |publisher = 中华工商联合出版社 |pages = 第136-138页 | ISBN =  | accessdate = 2014-01-02 | author = 潘君祥，段炼| url = http://books.google.com.hk/books?id=Ocpibm_gSMYC&pg=PA37&dq=%E6%B5%B7%E6%B4%BE%E8%A5%BF%E9%A4%90&hl=zh-CN&sa=X&ei=1yn2UrqGIomUiQe3lYDADg&ved=0CC0Q6AEwAA#v=onepage&q=%E6%B5%B7%E6%B4%BE%E8%A5%BF%E9%A4%90&f=false}}</ref> [[上海开埠|上海开埠]]后，西方文化逐渐开始影响上海，西菜馆也随之在上海开立。据文献记载，上海的第一家西菜馆为位于[[福州路_(上海)|福州路]]的一品香番菜馆<ref name="lp"/>。当时西菜又称为番菜。虽然在当时吃西餐是非常时髦的生活方式，但是中国人一开始难以适应西菜的烹饪方式，就比方说这半生半熟的[[牛排|牛排]]<ref name="lp"/>。上海西餐于是便汲取各国西餐的精髓而逐渐形成了自身的特色，俄式海派西餐一菜一汤（罗宋汤、[[面包|面包]]加[[黄油|黄油]]）因经济实惠瞬时风靡上海。1937年前，上海有西菜馆200多家<ref name="地方志2">[http://www.shtong.gov.cn/newsite/node2/node2245/node74959/node74963/node74981/node74989/userobject1ai90737.html 上海饮食服务业志-饮食业的形成和发展] {{webarchive|url=https://web.archive.org/web/20150924130355/http://www.shtong.gov.cn/newsite/node2/node2245/node74959/node74963/node74981/node74989/userobject1ai90737.html |date=2015-09-24 }} 上海市地方志办公室.</ref>，尤以霞飞路和福州路最为集中。其中福州路（旧称四马路）更是因此被称为“四马路大菜”<ref name="地方志2"/>。当时有名的有红房子、[[德大西菜社|德大西菜社]]、凯司令西菜社、蕾茜饭店、复兴西菜社和天鹅阁西菜馆等。

===本帮菜“老八样”===
本帮走油肉、咸肉扣水笋、扣鸡、本帮蒸三鲜、红烧鳊鱼、扣三丝、小葱肉皮、扣蛋卷。

===海派西餐===
罗宋汤、[[上海炸猪排|炸猪排]]、[[上海色拉|上海色拉]]（即洋山芋色拉）、[[烙蛤蜊|烙蛤蜊]]、烙蟹斗、金必多浓汤。

===点心类===
著名的上海點心有[[小籠馒头|南翔小笼馒头]]、[[生煎馒头|生煎馒头]]、[[粢饭糕|粢饭糕]]、[[油墩子|油墩子]]、[[老虎脚爪|老虎脚爪]]、[[蟹壳黄|蟹壳黄]]、[[八宝饭|八宝饭]]、[[排骨年糕|排骨年糕]]、双档、[[柴爿馄饨|柴爿馄饨]]、[[油豆腐粉丝汤|油豆腐粉丝汤]]、[[鸡鸭血汤|鸡鸭血汤]]、上海米糕、高桥松饼、[[豆沙鍋餅|豆沙鍋餅]]、[[兩面黃_(上海)|两面黄]]、[[葱油拌面|葱油拌面]]。
