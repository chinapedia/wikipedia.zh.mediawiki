《'''上位'''》（{{lang-en|'''Struggle'''}}）导演和编剧都是[[管晓杰|管晓杰]]，主演赵奕欢、文卓、胡悦、文梦洋、游艺湉、若绮。<ref name="bee" /><ref>{{cite web |url=http://ent.sina.com.cn/m/c/2013-01-15/13243835736.shtml |title=《上位》继续青春故事 赵奕欢演艺校生 |accessdate=2013 |publisher=新浪网 |work= |language=zh-hans}}</ref>2013年2月28日网络上映。

==剧情==
该片讲述中国大陆[[北漂|北漂]]一族中的青年女子为了演艺事业在肮脏的中国大陆娱乐圈摸爬滚打，她们被潜规则、自曝、[[炒作|炒作]]等，她们忍辱负重地挣扎和沉浮，只为了某一天可以出镜，一夜成名，成就影星梦。<ref name="bee" />

== 花絮 ==
该片上映之后一个月内的票房是2亿元人民币。<ref>{{cite web |url=http://mm.lzep.cn/2013/0704/175071.html |title=《上位》票房破2亿 赵奕欢等主创庆功 |accessdate=2013 |publisher= |work= |language=zh-hans |deadurl=yes |archiveurl=https://web.archive.org/web/20140322121540/http://mm.lzep.cn/2013/0704/175071.html |archivedate=2014年3月22日 |df= }}</ref><ref>{{cite web |url= http://news.xinhuanet.com/ent/2013-04/01/c_124528380.htm|title=《上位》破2亿成行业标杆 赵奕欢等主创庆功(图) |accessdate=2013 |publisher=新华网 |work= |language=zh-hans}}</ref>

为了让演出更加真实，赵奕欢让剧组掴了一个耳光，符合剧本里的要求。<ref>{{cite web |url=http://news.xinhuanet.com/ent/2013-03/07/c_124427780.htm |title=赵奕欢在《上位》剧组受虐 被掌掴后坚持上夜戏  |accessdate=2013 |publisher=新华网 |work= |language=zh-hans}}</ref>

该片辛辣揭露和讽刺了中国大陆娱乐圈的“[[潜规则|潜规则]]”，女艺人为了出境，必须出卖肉体给导演及剧组等。<ref name="bee" />
