'''上大崎'''（{{jpn|j=上大崎|hg=かみおおさき|rm=Kami-ōsaki|f=y}}）是[[東京都|東京都]][[品川區|品川區]]的地名，設一丁目至四丁目。2013年（平成25年）8月1日為止的人口有11,069人<ref>[http://www.city.shinagawa.tokyo.jp/ct/other000044500/201308a.xls 世帯と人口‐例月表 【地区別、町丁別、地域センター別】 2013年8月1日現在]</ref>。[[郵遞區號|郵遞區號]]141-0021。

== 地理 ==
位於品川區北部，鄰[[港區_(東京都)|港區]]、[[目黑區|目黑區]]、[[澀谷區|澀谷區]]。北接目黑區[[三田_(目黑區)|三田]]、澀谷區[[惠比壽|惠比壽]]，東鄰港區[[白金台|白金台]]，東南部連品川區[[東五反田|東五反田]]，西南通品川區[[西五反田|西五反田]]，西接目黑區[[目黑_(東京都)|目黑]]、[[下目黑|下目黑]]。

[[山手線|山手線]]貫穿町內，並設置[[目黑站|目黑站]]。此外地域內有東西向的[[目黑通|目黑通]]通過。目黑站周邊商業設施聚集，其他地區多為大樓、公寓、住宅混雜。上大崎一丁目附近靠近白金台與池田山，上大崎二丁目附近的高速道路旁俗稱「（白金）長者丸」<ref>[http://tokyo.txt-nifty.com/tokyo/2003/09/1.html 東京些末観光]　2011年10月21日閲覧。</ref>，上大崎三丁目附近多大使館，俗稱「花房山」，為知名[[高級住宅地|高級住宅地]]。

== 歷史 ==
[[File:Meguro_Station_in_Pre-war_Showa_era.JPG|thumb]]初期的[[目黑站|目黑站]]周邊]]
上大崎二丁目的[[都市再生機構|UR]]大規模集合住宅一帶，在太平洋戰爭敗戰前為[[海軍大學校|海軍大學校]]，後曾改為[[國立感染症研究所|國立預防衛生研究所]]與[[日映新社攝影所|日映新社攝影所]]<ref>[http://golgo13.main.jp/060212/060212.htm 折りたたみ自転車で野山を走ろう]　2011年10月21日閲覧。</ref>。

品川區上大崎的寺院群是源自[[芝|芝]][[增上寺|增上寺]]的下屋敷，幕末為江戶七大火葬場（荼毘所）之一。

=== 地名由來 ===
舊大崎村後來劃分為上大崎村與下大崎村。實施住居表示時保留「上大崎」作為此地地名。

== 再開發計畫 ==
上大崎二丁目與三丁目，[[目黑站|目黑站]]東南側3.2公頃的地區進行大規模再開發<ref name="nikkei20120712">目黑站前で大規模開発・東京建物など，5年後完成めざす・マンション2棟920戸　『日本経済新聞』　平成24年7月12日　東京・首都圏経済面</ref>。此地原為[[都營巴士|都營巴士]]車庫，2012年（[[平成|平成]]24年）7月[[東京都|東京都]]核准[[東京建物|東京建物]]、[[第一生命保險|第一生命保險]]、[[大成建設|大成建設]]、[[竹中工務店|竹中工務店]]組成「目黑站前地區市街地再開發組合」，開發工程將在2014年（[[平成|平成]]26年）8月動工<ref name="nikkei20120712"/>。

根據計畫，預計在最靠近目黑站的建地北側建造由40層的超高層公寓大廈（510戶）與27層的高層公寓大樓（[[總樓面面積|總樓面面積]]8萬平方公尺）所組成的雙子塔，南側將建造38層的超高層公寓<ref name="nikkei20120712"/>。計畫總工程費為747億日圓，預定2017年（[[平成|平成]]29年）12月完工<ref name="nikkei20120712"/>。

伴隨此開發計畫，目黑站側將整備為[[廣場|廣場]]，[[山手線|山手線]]旁的道路進行拓寬<ref name="nikkei20120712"/>。

== 交通 ==
町域內的目黑站有[[山手線|JR山手線]]、[[東急目黑線|東急目黑線]]、[[東京地下鐵南北線|地下鐵南北線]]等路線通過，為交通要衝。此外也有許多行駛目黑通的巴士路線。2005年（[[平成|平成]]17年）以前的目黑站前設有[[都營巴士目黑營業所|都營巴士目黑營業所]]。
