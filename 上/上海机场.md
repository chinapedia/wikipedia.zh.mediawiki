目前[[上海市|上海市]]拥有两座民用机场，[[浦东国际机场|浦东国际机场]]与[[虹桥国际机场|虹桥国际机场]]，两座机场分摊且分工地起降上海繁忙的国际与国内航班。浦东国际机场为上海最大的机场，主要承担进出上海的国际航班，但同时经营一部分国内航班。虹桥国际机场主要承担进出上海的国内航班，但同时也经营部分国际包机航班。此外，上海市内还拥有2座军用机场，1座直升机机场。

==发展历史==
{{参见|勒内·瓦隆}}
1911年，[[法国|法国]]飞行家[[环龙|环龙]]驾苏姆式双翼飞机来[[上海|上海]]作飞行表演，但是不幸机裂殒命。这是上海人民第一次目睹到飞机的风采。1912年，留学[[英国|英国]]学习飞行的[[厉汝燕|厉汝燕]]学成回国，驾驶爱特立克式单翼机在上海上空作试飞表演，这是中国人第一次在上海领空驾驶飞机。

===民国时期===
上海的第一座民用机场，是[[上海虹桥国际机场|虹桥机场]]，位于上海市西郊，1921年3月10日开始动土兴建，主要工程于是年6月底基本完成。虹桥机场工程竣工之后，却因为航空署因经费不足，闲置土地，原规划京沪航线未能按期通航。南京政府成立后，随着沪蓉航空线管理处的筹建，以及其上海至南京航线的开航，虹桥机场才正式被使用。1934年，虹桥机场进行了一次扩建，征地890亩。1937年8月13日，[[淞沪会战|淞沪会战]]爆发，导火线就是[[虹桥机场事件|虹桥机场事件]]，日本军队占领上海之后，进行了大规模改建和扩建使虹桥机场的占地面积达2513余亩，但大部分为强占土地。

上海的第二座民用机场，是位于龙华地区的[[上海龙华机场|龙华机场]]，龙华机场的建造日期比虹桥机场要早，但是在建成伊始，一直用作军用。龙华机场也是上海市第一个由陆军管辖的军用机场。1929年，国民政府航空署决定将龙华机场改为民用机场，由于当时虹桥机场闲置土地，所以到1936年淞沪抗战前龙华机场经不断修建，已成为当时中国最好的一个民用机场。日军占领后，也对龙华机场进行了一系列的扩建，主要为满足日军的战斗机停机需要。

1945年日本二战投降后，虹桥和龙华两大民用机场正式划归国民政府空军管辖。不久，两座机场相继重新划归民用，[[上海虹桥国际机场|虹桥机场]]因为政府内部诸多原因未能得到很好的建设，而相反的，[[上海龙华机场|龙华机场]]则经扩建，已成为中国民用航空运输的枢纽机场，而且在当时远东地区也是屈指可数的一个国际机场。

===中华人民共和国===
1949年中华人民共和国成立后，虹桥机场成为空军机场，龙华机场则继续作为民用机场。随著上海经济和民用航空事业的发展，到1960年代中期，龙华机场由于紧靠市区，受净空条件等限制，已无法适应大型飞机起降，1963年經[[中华人民共和国国务院总理|國務院總理]][[周恩来|周恩来]]批准，虹桥机场扩建为国际机场，准备将民航用途转至虹桥机场，但在起初为军民合用机场。1966年8月，原在龙华机场起降的国内航班飞机转到虹桥国际机场起降，同时规定龙华机场为“训练基地和航班飞机的备降场”。1972年，驻虹桥机场空军撤离。以后，虹桥机场作为上海的航空枢纽，进行了不断的扩建与改造。而龙华机场则承担上海起降小型固定翼飞机、飞艇、直升机的通用航空的机场（2B级[[通用机场|通用机场]]）。

1990年代起，上海经济高速发展，虹桥国际机场已不堪重负，配合著[[浦东新区|浦东新区]]的设立与崛起，上海市政府决定建设[[浦东国际机场|浦东国际机场]]，这也是上海第三座民用机场。浦东国际机场位于浦东的江镇、施湾、祝桥滨海地带。一期工程于1997年10月全面开工，并于1999年9月建成开始使用，进出上海的国际航班开始转交浦东国际机场经营，尔后浦东国际机场还经营进出上海一部分的国内航班。浦东机场二期工程已于2007年完工并投入使用。浦东机场超越虹桥机场成为上海最大最重要的航空枢纽。目前，浦东机场是中国大陆最繁忙的机场之一。

龙华机场已没有起降民用航班，并最终于2008年后被拆除。上海机场形成浦东与虹桥两座机场分工使用的格局，为满足上海发展及2010年世博会，虹桥机场于2006年开始规划建设二期工程扩建，这项扩建是[[虹桥综合交通枢纽|虹桥综合交通枢纽]]工程的一部分，于2010年初完工投入使用，使虹桥机场拥有更强的航空货客吞吐量。

由于中国商飞[[ARJ21|ARJ21]]与[[C919|C919]]组装生产地都在上海，目前上海的两座机场承担着中国商飞的测试及交付场地功能。其中，ARJ21在大场机场测试交付<ref name="ARJ21交付运营 回顾国产ARJ21交付、首飞历程">[http://news2.jschina.com.cn/system/2016/06/28/029044757.shtml ARJ21交付运营 回顾国产ARJ21交付、首飞历程]</ref>，而C919则在浦东国际机场测试<ref name="C919大型喷气客机上海滑行测试顺利完成 首飞在望">[http://news.163.com/17/0302/11/CEH6KJDB00018AOQ.html#p=CECE9MQH56NT0001 C919大型喷气客机上海滑行测试顺利完成 首飞在望]</ref>。

==运输规模==
2017年，上海机场共接待进出港旅客1.12亿人次（其中浦东7000.43万人次，虹桥4191.13万人次），实现货邮吞吐424.36万吨（其中浦东383.56万吨，虹桥40.80万吨），起降航班76.06万架次（其中浦东496879架次，虹桥263720架次）。上海是全球第5个年创造航空客运量过亿人次纪录的城市（前四个分别是[[英国|英国]][[伦敦|伦敦]]、[[美国|美国]][[纽约|纽约]]、[[日本|日本]][[东京|东京]]、[[美国|美国]][[亚特兰大|亚特兰大]]），也是全球第3个创造年航空货量400万吨以上纪录的城市（前两个分别是[[香港|香港]]、[[美国|美国]][[孟菲斯_(田納西州)|孟菲斯]]）。其中，浦东机场旅客吞吐量稳居中国大陆第二，出入境人数稳居中国大陆空港口岸第一，货邮吞吐量稳居中国大陆第一，世界第三<ref>{{cite web |title = 上海浦东机场年旅客量突破7000万 仅次于北京香港 |publisher = 航空圈 |date = 2018-02-01 | url=http://www.sohu.com/a/214253753_600491|accessdate =2018-02-04|language = zh }}</ref><ref>{{cite web |title = 上海浦东机场口岸出入境人数首破3500万 连续15年稳居全国空港口岸之首 |publisher = 看看新闻  |date = 2017-12-27 | url=http://news.sina.com.cn/o/2017-12-27/doc-ifyqcsft7253618.shtml|accessdate =2018-02-04|language = zh }}</ref>。

===民用机场===
*[[上海浦东国际机场|上海浦东国际机场]]
**{{Airport codes|PVG|ZSPD}}。位于上海[[浦东新区|浦东新区]]东部与[[东海|东海]]相邻。现在是上海市最重要的[[机场|机场]]，经营进出上海绝大部分的定期国际航班，同时也经营一部分国内航班，其航班量占整个上海机场的60%左右，目前建有2座[[航厦|航厦]]；4条起降[[跑道|跑道]]。
*[[上海虹桥国际机场|上海虹桥国际机场]]
**{{Airport codes|SHA|ZSSS}}。位于上海西郊[[长宁区|长宁区]]。浦东国际机场使用前，上海唯一的[[国际机场|国际机场]]，现为上海第二大机场，主要经营国内航班，同时也經營至[[东京|东京]][[羽田机场|羽田机场]]、[[首尔|首尔]][[金浦机场|金浦机场]]、[[台北|台北]][[台北松山机场|松山机场]]、[[香港|香港]][[香港国际机场|赤鱲角机场]]、[[澳门|澳门]][[澳门国际机场|机场]]的定期国际航班。目前建有2座[[航厦|航厦]]；2条起降[[跑道|跑道]]，2号航厦是[[虹桥综合交通枢纽|虹桥综合交通枢纽]]的一部分。

===已拆除废弃机场===
*[[上海龙华机场|上海龙华机场]]
**位于上海市中心内[[徐汇区|徐汇区]]，邻近[[黄浦江|黄浦江]]旁，为中国最早期的民用机场之一。1966年8月后停止起降民用航班，但仍有小型固定翼飞机、[[飞艇_(轻航空器)|飞艇]]、[[直升机|直升机]]起降，2008年拆除。
*[[上海江湾机场|上海江湾机场]]
**位于上海北部[[杨浦区|杨浦区]]北部，[[二战|二战]]期间日军建成后一直作为军事机场使用。1994年停止使用，1997年正式废除并转交上海市政府规划使用。
*[[上海丁家桥机场|上海丁家桥机场]]
**位于上海北部[[宝山区_(上海市)|宝山区]]丁家桥，[[二战|二战]]期间日军建成后一直作为军事机场使用。1978年拆除。
