'''上海亚细亚大楼'''（{{lang-en|'''Asia Building, Shanghai'''}}），位于[[上海市|上海市]][[黄浦区|黄浦区]][[中山东一路|中山东一路]]1号，也因此被称为“外滩第一楼”<ref name="abs">{{cite web|url=http://www.shtong.gov.cn/newsite/node2/node71994/node81772/node81774/node81784/userobject1ai108913.html |title=外滩第一楼：亚细亚大楼（中国太平洋保险公司总部） |publisher=上海地方志办公室 |accessdate=2013-08-04 }}</ref>。该楼由麦边洋行出资、[[马海洋行|马海洋行]]设计、[[裕昌泰营造厂|裕昌泰营造厂]]于1913年兴建，1916年竣工。初建时因投资方而被称作麦边大楼（现译麦克波恩大楼），1917年[[壳牌公司|亚细亚火油公司]]租入大楼后，因其在华的影响，而逐渐被人称作亚细亚大楼<ref>{{cite web|url=http://www.shanghai.gov.cn/shanghai/node2314/node2315/node5827/userobject21ai358855.html |title=“外滩1号”--中国太平洋保险公司总部大楼穿上“新衣” |publisher=中国上海 | date=2009-08-31 |accessdate=2013-08-04 }}</ref>。大楼总高八层，外立面为折衷主义风格<ref>{{cite web|url=http://sh.sohu.com/20120724/n348930503.shtml |title=岁月相框：看老照片，读旧上海滩城事（11） |publisher=搜狐上海 | date=2012-07-24 |accessdate=2013-08-04 }}</ref>。1989年，被选入[[上海市优秀历史建筑|上海市第一批优秀历史建筑]]<ref>{{cite web|url=http://www.shtong.gov.cn/newsite/node2/node71994/node81772/node81779/node81794/userobject1ai108894.html |title=三、1989年、1993年以优秀近代建筑列入上海市文物保护单位一览表（第一批） |publisher=上海地方志办公室 |accessdate=2013-08-04 }}</ref>。

==历史==
亚细亚大楼所处土地，自[[上海开埠|上海开埠]]以后，曾先后为[[兆丰洋行|英商兆丰洋行]]、[[旗昌洋行|美商旗昌洋行]]所有<ref>{{cite web |url=http://www.ct66.com/ProShow.asp?Proid=1273 |title=光华火油“民国的中石化” |publisher=中国股票博物馆 |accessdate=2013-08-04 }}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。1891年，旗昌洋行因经营不善而宣告破产。其位于外滩的数幅土地为[[轮船招商局|轮船招商局]]购入，外滩1号所在地块也在其中。之后，轮船招商局又将此地转手，几经易主于1913年为麦边洋行购得。麦边洋行随后在原址拆除旧屋，并聘请马海洋行设计新楼，裕昌泰营造厂稍后接手了新楼的建设项目。工程于1916年竣工，麦边洋行随即将该楼出租。翌年，英国壳牌运输贸易公司与荷兰皇家石油公司的子公司，亚细亚火油公司租入该楼除二、三层以外的所有楼面。并被允许于门外挂起亚细亚火油的铭牌<ref>{{cite web |url=http://s07.static-shell.com/content/dam/shell/static/chn-zh/downloads/news-and-library/china-history-brochure2004ch.pdf |title=壳牌在中国的故事 |publisher=壳牌中国 |accessdate=2013-08-04 |deadurl=yes |archiveurl=https://web.archive.org/web/20131102045718/http://s07.static-shell.com/content/dam/shell/static/chn-zh/downloads/news-and-library/china-history-brochure2004ch.pdf |archivedate=2013-11-02 }}</ref>。此前，亚细亚火油从设在上海的办事处到北中国分公司一直设立在[[九江路_(上海)|九江路]]7号的大楼内。十年间该公司在华业务迅速扩张，石油销量占据中国市场的四分之一<ref name="abs"/>。并迅速成为与[[美孚石油|美孚]]、[[德士古|德士古]]齐名的石油公司<ref>{{cite web|url=http://www.changjiangtimes.com/2012/11/423179.html |title=“亚细亚”带来汉口照明时代 |publisher=长江商报 | date=2012-11-30 |accessdate=2013-08-04 }}</ref>。因此，改变办公条件和强大经济实力，最终促成公司租入了位于外滩1号的大楼。而因其声名在外，大楼也逐渐被人改称为亚细亚大楼<ref name="abs"/>。

亚细亚火油公司一直在内办公到1941年。当年12月，随着[[太平洋战争|太平洋战争]]的爆发，作为英商资产的亚细亚火油公司被[[侵华日军|日军]]强行接收，而大楼也成为日方资产。1945年，抗战胜利后，大楼发还原业主，亚细亚公司重新租回此处。1949年上海易主后，亚细亚大楼业主于第二年改为华东石油公司，而亚细亚火油公司则迁入[[圆明园路|圆明园路]]，成为中国境内唯一外商石油公司<ref>{{cite web |url=http://www.shell.com.cn/zh/aboutshell/who-we-are/history/china.html |title=关于壳牌中国 |publisher=壳牌中国 |accessdate=2013-08-04 |deadurl=yes |archiveurl=https://web.archive.org/web/20130807191105/http://www.shell.com.cn/zh/aboutshell/who-we-are/history/china.html |archivedate=2013-08-07 }}</ref>。1966年，[[文化大革命|文化大革命]]爆发，办事处撤离。而在早先的1959年，[[上海市冶金设计院|上海市冶金设计院]]、上海市房地产管理局和上海市丝绸公司等单位迁入亚细亚大楼。直至1996年，外滩总体进行房屋置换，原有单位迁出，大楼后为[[中国太平洋保险(集团)有限公司|中国太平洋保险公司]]总部所在<ref name="abs"/>。2009年，为迎接翌年召开的[[上海世界博览会|上海世界博览会]]，[[上海市人民政府|上海市政府]]出资对外滩建筑进行整修，亚细亚大楼亦在其列<ref name="abs1"/>。

==建筑风格==
亚细亚大楼占地面积1739平方米，建筑面积11984平方米。总体为[[钢筋混凝土|钢筋混凝土]]的框架结构，但外立面使用[[花岗岩|花岗岩]]和清水砖砌成。东、南两侧的外立面沿东南角呈对称结构，建筑风格属折衷主义。另外从横向和竖向而言均依据三段式结构进行设计建造，横向上，底部与顶部均吸收[[巴洛克建筑|巴洛克风格]]，而中部又采用了[[现代主义|现代主义]]的设计。竖向上，位于中央的底部与顶部均使用贯高两层的双[[爱奥尼柱式|爱奥尼柱式]]。在功能设置上，中央除底部外，中部和顶部均设置内凹阳台<ref name="abs1">{{cite web|url=http://2010.qq.com/a/20090831/000026.htm |title=外滩第一楼完成装修 提升美观和历史厚重感 |publisher=腾讯网 | date=2009-08-31 |accessdate=2013-08-04 }}</ref>。

另外，大楼为调和视觉感官，在东南角即[[延安东路_(上海)|延安东路]]中山东一路路口也使用了凹面设计。1938年加建的第八层，其四角也采用了凹面式。平面上，建筑则为回字形，中间设计有天井。为确保建筑物的采光，整幢大楼共有大小窗户100余个。另外，在二层、五层和八层的窗、阳台廊柱均按半圆形设计<ref name="abs"/>。
