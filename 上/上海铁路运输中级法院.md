'''上海铁路运输中级法院'''（简称'''上铁中院'''），位于[[上海市|上海市]][[虹口区|虹口区]][[武进路_(上海)|武进路]]527号，是[[中华人民共和国|中华人民共和国]][[上海市|上海市]]的一所[[铁路运输法院|铁路运输中级法院]]。

== 沿革 ==
1954年4月28日，'''上海铁路沿线专门法院'''成立，负责审理南京铁路分局、上海铁路分局、杭州铁路分局、上饶铁路分局沿线辖区内案件。1955年4月1日，该院更名为'''上海铁路运输法院'''。1957年10月20日，该院撤销。

1981年12月15日，'''上海铁路运输中级法院'''成立，1982年5月1日开始正式受理案件。<ref>{{cite web |url = http://www.shtong.gov.cn/newsite/node2/node2245/node81324/node81329/node81362/node81368/userobject1ai101256.html|title=上海审判志：第八节　专门人民法院 |publisher=上海通 |date= |accessdate=2012-03-06}}</ref><ref>{{cite web |url = http://www.shtong.gov.cn/newsite/node2/node2245/node68716/node68729/node68808/index.html |title=上海铁路志  第九篇政法、武装 第三章法院 |publisher=上海通 |date= |accessdate=2012-03-06}}</ref>

2012年6月5日，上海铁路运输中级法院、[[上海铁路运输法院|上海铁路运输法院]]签署改制移交协议，两级铁路运输法院自此脱离铁路系统，移交上海市进行属地管理，纳入[[中华人民共和国司法体制|国家统一司法体系]]。<ref name="zgxww">[http://www.chinanews.com/fz/2013/09-25/5323036.shtml 上海铁路法院将受理涉轨交、高架道路部分案件]，中国新闻网，2013-09-25</ref><ref name="zonghe">[http://www.shjcw.gov.cn/node2/zzb/shzfzz/jrgz/node1745/u1ai431317.html 上海铁路法院管辖范围扩大 将受理涉轨交高架部分案件]，上海政法综合网，2013-09-26</ref>

2012年12月20日，上海铁路法院系统干部大会暨铁路中院主要领导交接仪式举行。仪式由沈志先主持，夏明林宣读了[[中共上海市委|中共上海市委]]关于陈永华同志到龄退休、[[许一新|许一新]]同志任上海铁路运输中级法院党组书记的决定，吴偕林宣读了铁路两级法院党组成员以及相关法律职务的任命。许一新接替陈永华担任上海铁路运输中级法院党组书记、院长。<ref>[http://www.shenbanglawyer.com/html/xinwendongtai/xingyedongtai/20121224/12031.html 铁路法院干部大会暨铁路中院主要领导交接仪式举行] {{webarchive|url=https://web.archive.org/web/20141229180920/http://www.shenbanglawyer.com/html/xinwendongtai/xingyedongtai/20121224/12031.html |date=2014-12-29 }}，上海申邦律师事务所，2012-12-24</ref>

根据2012年8月1日起实施的《最高人民法院关于铁路运输法院案件管辖范围的若干规定》（法释〔2012〕10号），经最高人民法院批准，[[上海市高级人民法院|上海市高级人民法院]]对上海两级铁路运输法院受理指定管辖案件下达正式批复，自2013年9月26日起，“[[上海铁路运输法院|上海铁路运输法院]]依法受理上海市轨道交通运营区域内发生的民事一审案件、[[上海市公安局|上海市公安局]]交警高架支队管辖区域内发生的道路交通事故引起的人身、财产损害赔偿一审案件，以及上海市轨道交通运营区域内发生的行政一审案件；上海铁路运输中级法院依法受理对上述一审案件提起的上诉案件，以及[[上海市北新泾监狱|上海市北新泾监狱]]刑事[[减刑|减刑]]、[[假释|假释]]案件。”这是全国铁路运输法院改制移交后，全国首家依据《最高人民法院关于铁路运输法院案件管辖范围的若干规定》由最高人民法院批准扩大民事案件指定管辖范围的铁路运输法院。同时，[[上海市高级人民法院|上海市高级人民法院]]根据上海市本地实际，依法指定部分行政案件和减刑假释案件由铁路运输法院受理。<ref name="zgxww" /><ref name="zonghe" />

2014年12月25日，上海市第十四届人大常委会第十七次会议审议并表决通过多条人事任免案，决定任命[[吴偕林|吴偕林]]为[[上海市第三中级人民法院|上海市第三中级人民法院]]院长、[[上海知识产权法院|上海知识产权法院]]院长、上海铁路运输中级法院院长，免去其[[上海市高级人民法院|上海市高级人民法院]]副院长、审判委员会委员、审判员职务。<ref name="shzw">[http://shzw.eastday.com/shzw/G/20141225/u1ai141785_K194.html 吴偕林任上海市三中院、知识产权法院、铁路运输中院院长上海法院检察院系统提拔一批新干部]，上海政务，2014-12-25</ref>

2014年12月28日，上海市第三中级人民法院、上海知识产权法院在[[上海市|上海市]][[浦东新区|浦东新区]][[张衡路|张衡路]]988号挂牌成立。上海市第三中级人民法院依托上海铁路运输中级法院设立，同时组建上海知识产权法院，与上海市第三中级人民法院[[合署办公|合署办公]]，实行“[[一个机构多块牌子|三块牌子一个机构]]”。和上海市第三中级人民法院、上海知识产权法院同日成立的还有[[上海市人民检察院第三分院|上海市人民检察院第三分院]]。自2015年1月1日起，上海市第三中级人民法院、上海知识产权法院正式受理案件。<ref name="tengxun">[http://news.qq.com/a/20141229/001142.htm 全国首个跨行政区法院和检察院在上海成立]，腾讯，2014-12-29</ref><ref name="tianjin">[http://www.tianjinwe.com/rollnews/201412/t20141229_789349.html 三中院和知识产权法院合署办公] {{webarchive|url=https://web.archive.org/web/20141229063534/http://www.tianjinwe.com/rollnews/201412/t20141229_789349.html |date=2014-12-29 }}，天津网，2014-12-29</ref>

== 管辖 ==
本法院司法管辖[[上海铁路局|上海铁路局]]管区。自2013年9月26日起，本法院还受理对[[上海铁路运输法院|上海铁路运输法院]]依法受理的“[[上海地铁|上海市轨道交通]]运营区域内发生的民事一审案件、[[上海市公安局|上海市公安局]]交警高架支队管辖区域内发生的道路交通事故引起的人身、财产损害赔偿一审案件，以及上海市轨道交通运营区域内发生的行政一审案件”提起的上诉案件。<ref name="zgxww" /><ref name="zonghe" />
