'''上海市政府飞行队'''（{{lang-en|'''Shanghai Government Aviation Force'''}}），又名'''上海市公安局警务航空队'''（{{lang|en|'''Shanghai Municipal Public Security Bureau Police Aviation Force'''}}）。成立于2009年5月20日<ref>{{cite web |title = 市政府飞行队、市公安局警务航空队成立 韩正出席揭牌仪式 |publisher = [[上海市人民政府|上海市人民政府]] |date = 2009-05-21 |url = http://www.shanghai.gov.cn/shanghai/node2314/node2315/node4411/u21ai339188.html |accessdate = 2014-09-22 |language = zh }}</ref>，目前共拥有5架[[直升机|直升机]]。
