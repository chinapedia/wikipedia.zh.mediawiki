'''上海市徐汇中学'''是位于上海虹桥路68号的著名[[完全中学|完中]]学校，在[[徐家汇天主教堂|徐家汇天主教堂]]旁，始名'''聖依納爵公學'''（{{lang-fr|Collège Saint Ignace}}），由[[天主教|天主教]][[耶稣会|耶稣会]]建于1850年（清[[道光|道光]]三十年）<ref>{{Cite web |url = http://news.xinhuanet.com/edu/2010-11/06/c_13594231.htm |title = 上海“西学东渐第一校”徐汇中学迎来建校160周年 |publisher = 新华网 |date = 2011-11-06}}</ref>，有拉丁文校训：{{lang|la|LABORA SICVT BONVS MILES CHRISTI}}。1931年组织校董会，向[[中華民國教育部|中華民國教育部]]办理立案，易名“'''徐汇中学'''”，改“[[二三制|二三制]]”。至今历史已超过160年。现为徐汇区实验性示范性高中之一<ref>{{cite web |url = http://www.xhedu.sh.cn/cms/data/html/doc/2007-12/24/145839/index.html |publisher = 徐汇教育 |title = 实验性示范性高中 |date = 2007-12-24 |author = 徐汇区教育局 |access-date = 2012-09-02 |archive-url = https://web.archive.org/web/20120831083412/http://www.xhedu.sh.cn/cms/data/html/doc/2007-12/24/145839/index.html |archive-date = 2012-08-31 |dead-url = yes }}</ref>。

== 简介 ==

1849年法籍耶稣会士[[南格禄|南格禄]]在徐家汇招收了二三十名中国儿童开始启蒙讲学，1850年（清道光三十年）正式取名为'''徐汇公学'''，也称'''圣依纳爵公学'''，1931年定名为'''徐汇中学'''。坐落在明末大学士[[徐光启|徐光启]]的故里[[徐家汇|徐家汇]]，受其影响，开启了中国中西教育交汇的先河，在[[中国教育史|中国教育史]]上拥有一个特殊的地位，被誉为中国西洋办学第一校。<ref>{{Cite web |url = http://www.shtong.gov.cn/newsite/node2/node71994/node81772/node81774/node81784/userobject1ai108989.html |publisher=上海市地方志办公室 |title = 中国"西洋办学第一校"：徐汇公学楼（徐汇中学）}}</ref>中共建政后，1951年由政府接收成为公办学校。而在[[新北市|新北市]][[蘆洲區|蘆洲區]]的另一所[[新北市私立徐匯高級中學|私立徐滙高級中學]]，由1949年到[[臺灣|臺灣]]的徐滙中學校友于1963年以原上海徐匯中學名義[[在臺復校|在臺復校]]。

徐汇公学的内部组织和管理方法都沿袭法国的教会学校制度，由耶稣会院长总揽全权。教学上首开[[数理化|数理化]]教学，设置音体美课程；与[[土山湾油画院|土山湾油画院]]有密切的亲缘关系，学生很早就接受西方绘画训练；1862年前已开始组建学生西洋乐队，好评如云；1947年辟建英语语音室。徐汇中学素以教育、教学严谨著称，形成崇尚科学、爱国荣校、个体发展的办学传统，培养了很多优秀人才，历届校友遍及世界各地。曾经在徐汇中学就读的著名校友有：教育家"爱国老人"[[马相伯|马相伯]]、[[地质学|地质学]]家[[翁文灏|翁文灏]]院士、“近代雕塑之父”[[张充仁|张充仁]]、中国原[[卫生部|卫生部]]长[[陈敏章|陈敏章]]、[[主教|主教]]张家树金鲁贤、中科院副院长[[严义埙|严义埙]]、世界著名[[原子物理|原子物理]]学家[[法国科学院|法国科学院]]院士[[周知行|周知行]]、[[中国科学院|中国科学院]]院士[[周兴铭|周兴铭]]、中国科学院院士[[汪应洛|汪应洛]]、[[中国工程院|中国工程院]]院士[[魏敦山|魏敦山]]、[[翻译家|翻译家]][[傅雷|傅雷]]、现代作家[[叶辛|叶辛]]、游泳世界冠军[[杨爱华|杨爱华]]、香港[[厂主联合会|厂主联合会]]会长[[黄鉴|黄鉴]]博士、美国苏浙同乡会会长[[朱元忠|朱元忠]]等人。 学校在海内外久负盛名，在西欧、北美、东南亚有很大影响。

今日徐汇中学多彩教育办学理念的简要表述：

学校营造丰富多彩的教育天地，使每个学生丰富多彩的兴趣爱好、个性特长得到充分发展，[[创新|创新]]意识和创新能力得到充分展现，从而培养出大批丰富多彩的人才，使未来的世界更精彩。

徐汇中学道德教育的指导思想是：“爱家、爱校、爱国，爱心奉献社会；敬德、敬学、敬业，敬意图报人民”。

徐汇中学多年来德育工作的特点是：“使学生的脉搏与祖国的心脏一起跳动”，从而培养一种为中华复兴而不懈奋斗的精神。

徐汇中学奋斗教育的重点培养学生具有：“好强的个性、竞争的意识、不服输的精神、不断进取的追求”，在矛盾、困难面前决不示弱，在激烈竞争中有通过自我努力超越别人的决心，决不服输，有爬起来再干的勇气，取得成绩决不自满，有努力创造新成绩的愿望和行动。

== 办学理念 ==
*'''办学传统'''
崇尚科学、爱国荣校、多彩发展
 
*'''办学特色'''
营造多彩天地，发展多彩个性，培养多彩人才

*'''办学目标'''
经过努力，把徐汇中学建设成办学有特色，学生有特长，教师有专长，教育教学有质量，社会声誉鹊起的市实验性示范性中学，从而树立社会主义公办学校为国为民育人才的模范形象，建成徐汇区品牌教育高地
  
*'''德育目标'''
学会感恩、关爱他人、懂得责任、恪守诚信

==== 崇思楼 ====
始称新校舍，建于1917年。[[孔子|孔子]]曰：“学而不思则罔”，故名“崇思”。这幢建于1917年属法国文艺复兴时期建筑风格的古建筑，其外立面为砖石结构，内部则以木结构为主，共4层。原有三楼大礼堂，法國Mansard式屋顶于1990年代改为普通屋顶。2009年进行大修復原，楼内一楼有小礼堂、汇学博物馆，二楼有校长室、图书馆，三楼有校史陈列室、徐汇画苑、手风琴、音乐、舞蹈教室，四楼则为美术工作室、会议室。该建筑是[[上海市优秀历史建筑|上海市优秀历史建筑]]。

==== 重德楼 ====
2000年建于西校舍旧址，崇思楼西。《[[易经|易经]]》倡导“厚德载物”，遂名“重德”。2010年翻新，现有六层楼，有高中年级教室（18间）、办公室、3间计算机教室、2间会议室还有心理咨询室。

==== 尚学楼 ====
1991年在老校舍原址上再建，崇思楼南。《[[论语|论语]]》云：“思而不学则殆。”因称“尚学”。五层楼，为六、七、八年级的教学楼。

==== 砺行楼 ====
2004年整修、收回。“子别根犹牵，母盼月同圆。海阔水相连，潮平帆可悬。”2012年再次整修，现有六层楼，有[[生命科学|生命科学]]创新实验中心、工程素养类实验室（高铁模拟实验室、高铁模拟驾驶实验室、3D打印实验室）、身体素质评估中心，同时也是教师发展中心（教导处）、学生发展中心（学生处）、科研和课程发展中心（教务处）、总务处等行政部门的办公室所在地。

==== 汇学长廊 ==== 
通过艺术的手法再现了这所百年老校的风雨历程。汇学长廊已成为徐家汇文化根脉的注脚之一。

==== 空中篮球场 ==== 
位于尚学楼5楼屋顶，供初中学生使用。

==== 大操场 ====
辟于1905年。当年凡有足球比赛即“万人空巷，观者如潮”。2000年重建，铺有[[人造草坪|人造草坪]]。2016年重新铺设人造草坪。2017年9月将原先的草坪改为了篮球场。

==== 惜馔楼 ====
“谁知盘中餐，粒粒皆辛苦。”特名为“惜馔楼”。目前为教师食堂。

==== 体育馆 ====
1996年建成。馆内可举行多种活动。2014年开始改造翻建，因为资金原因暂停了半个学期。已于2018年初全面完工，命名为“勤体馆”。设有大礼堂、乒乓房、羽毛球场等等。

==== 网球场==== 
2017年将原先的篮球场改为网球场。

==== [[游泳池|游泳池]] ====
最深2米，暑假期间对外开放。上海市教委指定中小学生学游泳单位之一。
目前已不开放使用

==== [[马相伯|马相伯]]铜像 ====
“崇思楼”前，竖有中国近代教育改革的先驱、徐汇中学历史上第一位华人校长[[马相伯|马相伯]]先生（1840年~1939年）的塑像。

=== 西校区 ===
位于番禺路、南丹路口。原为[[凯旋中学|凯旋中学]]，2004年并入徐汇中学。为初三年级所在地。

=== 北校区 ===
淮海中路1788号，原南模初级中学，2009年开始使用。并在2010年停用。

=== 南校区 ===
华发路68号，华发路龙吴路口，2018年9月正式启用。
