'''上海市历史博物馆'''（'''上海革命历史博物馆'''），位于[[上海市|上海市]][[南京西路_(上海)|南京西路]]325号原[[上海跑马总会|上海跑马总会]]大楼，是介绍[[上海历史|上海历史]]的上海市级地方性历史博物馆<ref>{{cite web|title=上海市历史博物馆|url=http://www.historymuseum.sh.cn/index.php/aboutlb/index?pa=1|website=www.historymuseum.sh.cn|deadurl=yes|archiveurl=https://web.archive.org/web/20170527212237/http://www.historymuseum.sh.cn/index.php/aboutlb/index?pa=1|archivedate=2017-05-27}}</ref><ref>{{cite web|title=上海市历史博物馆|url=http://www.historymuseum.sh.cn/index.php/aboutlb/index?pa=8&resID=10|website=www.historymuseum.sh.cn|deadurl=yes|archiveurl=https://web.archive.org/web/20161220104711/http://www.historymuseum.sh.cn/index.php/aboutlb/index?pa=8&resID=10|archivedate=2016-12-20}}</ref>。

==历史==
上海市自1950年代开始筹建上海历史与建设博物馆，于1983年建成上海历史文物陈列馆，1991年7月更名为上海市历史博物馆，但是没有对外开放的永久展馆。1994年10月，上海市历史博物馆在[[虹桥路_(上海)|虹桥路]]1286号再次开馆，于1999年3月关闭。上海市历史博物馆的展品在[[东方明珠广播电视塔|东方明珠广播电视塔]]裙房内的上海城市历史发展陈列馆向社会开放，展区分五部分，分为“华亭溯源”、“城厢风貌”、“开埠掠影”、“十里洋场”、“海上旧踪”五部分<ref>{{cite web  | url=http://www.sh-aiguo.gov.cn/node2/node4/node7/userobject1ai539.html |title=上海市历史博物馆  |publisher= 中共上海市委宣传部 |date=  | accessdate=2012-05-14 }}</ref>。

2015年底，经[[中共上海市委|中共上海市委]]、[[上海市人民政府|上海市人民政府]]正式批复，“[[十三五规划|十三五]]”期间规划建设的上海市级重大文化设施之一“上海市历史博物馆/上海革命历史博物馆”建设项目正式启动，将位于[[南京西路_(上海)|南京西路]]325号的[[上海市文物保护单位|上海市文物保护单位]][[上海跑马总会|跑马厅]]大楼修缮后作为馆址。2017年10月1日起，该馆东楼开始了内部征求意见的试开放。2017年11月1日起，该馆内部试运行<ref>{{Cite web |url =http://www.thepaper.cn/newsDetail_forward_1844575  |title = 上海市历史博物馆/上海革命历史博物馆明起内部试运行 |author =  |date = 2017-10-31  |publisher = 澎湃新闻 |language =  |accessdate =  |quote =  }}</ref>。2018年3月26日整個博物館免費向市民開放<ref name=":0">[http://shanghai.xinmin.cn/xmsq/2018/03/10/31367600.html "跑马总会"修缮改建中 上海历史博物馆月底正式开放]  李争 看看新闻网  2018-03-10</ref><ref>[https://www.thepaper.cn/newsDetail_forward_2043306 探访今天免费开放的上海历史博物馆].澎湃新闻.[2018-03-26].</ref>。

== 展览 ==
博物馆展品总数11万件，分书画、金属、陶瓷、工艺、证章、文献、印刷、纺织品、石刻、钱币、照片、剪纸、邮票、唱片和杂项共15类。珍贵文物有[[明代|明代]]韩希孟顾绣花卉虫鱼册、侯峒曾行书轴、七宝寺藏金字写经、近代陈化成抗英遗物“振远将军”铜炮、[[太平天国|太平天国]]大花钱、原[[汇丰银行|汇丰银行]]门前铜质对狮、 1893年英美公共租界及法租界界碑等。目前展览分常设展和临时展两类。
