'''上海前滩国际商务区'''，是在[[中華人民共和國|中華人民共和國]][[上海市|上海市]]兴建中的世界级[[中心商务区|商务中心]]，位于[[浦东新区|浦东新区]]原[[上海世博会|世博会]]场馆区“后滩”地区南侧，是[[陆家嘴金融贸易区|陆家嘴金融贸易区]]的姊妹城<ref>{{cite news |url = http://news.cqnews.net/html/2012-11/25/content_21819992.htm |title = 上海前滩国际商务区全面开工基础建设 |publisher = 中国新闻网 |date = 2012年11月25日 |accessdate = 2013-05-04 |deadurl = yes |archiveurl = https://web.archive.org/web/20131016054805/http://news.cqnews.net/html/2012-11/25/content_21819992.htm |archivedate = 2013年10月16日 }}</ref>。

== 地理位置 ==
“前滩国际商务区”（下称前滩）位于上海市黄浦江中部地区东岸ES4地块，原曾规划建设“环球影城”，后因故停止。因该地块位于上海世博会后滩的上游，开发主体陆家嘴集团的总经理杨小明将其命名为“前滩”，得到市领导的认可，并为世人所熟知。前滩地区的范围主要包括[[黄浦江|黄浦江]]以东、[[川杨河|川杨河]]以南、济阳路以西、华夏西路以北的区域，北接世博后滩拓展区和耀华地块、西临黄浦江和[[徐汇区|徐汇]]滨江地区，其中，沿黄浦江岸线长约2.3公里，沿川杨河岸线长0.8公里，总面积2.83平方公里<ref>{{cite news |url = http://sh.eastday.com/m/20121126/u1a7019236.html |title =前滩国际商务区基础建设开工 道路、学校等各就各位 |publisher = 东方网 |date = 2012年11月26日 |accessdate = 2013-05-04}}</ref>。

== 功能规划 ==
当前，前滩地块规划居住人口2.5万，工作人口为15万-20万，其中外籍人口预计占比60%以上。占地约283.17公顷，其中，供办公、商业、文化娱乐所用的建筑将占总比重之50%，学校、医院等公共服务设施占10%，其余40%则用于建造国际社区及高端白领居住区<ref>{{cite news|url =http://house.ifeng.com/cre/detail_2012_07/12/15970827_0.shtml|title =上海前滩：世界级CBD十年后成形|publisher =时代周报|date =2012年7月12日|accessdate =2013-05-04|deadurl =yes|archiveurl =https://web.archive.org/web/20120717105539/http://house.ifeng.com/cre/detail_2012_07/12/15970827_0.shtml|archivedate =2012年7月17日}}</ref>。

=== 主体设施 ===
区别于已基本建成的[[陆家嘴金融贸易区|陆家嘴金融贸易区]]，前滩商务区主要为上海建成国际金融中心提供诸如会计、咨询、法律等等各项现代服务，也是跨国企业地区总部集聚地。整个规划的固定资产投资约1000亿元人民币，总建筑面积超过350万平方米<ref>{{cite news |url =http://news.online.sh.cn/news/gb/content/2012-11/26/content_5701914.htm |title =前滩国际商务区4项目 2013年底完成市政工程建设 |publisher =新闻晨报 |date =2012年11月26日 |accessdate =2013-05-04 |deadurl =yes |archiveurl =https://archive.is/20130628085043/http://news.online.sh.cn/news/gb/content/2012-11/26/content_5701914.htm |archivedate =2013年6月28日 }}</ref>。其中，有两栋高度为133.5米的办公塔楼被称为前滩的“双塔”；自西向东由一个主景观轴构建，依次排布有：下沉式庭院、林荫道、两层楼的商业架空走廊和休息平台等，接着是一个两层同高的高层塔楼门厅，最东侧为水面景观<ref>{{cite news |url = http://www.shanghai.gov.cn/shanghai/node2314/node2315/node18454/u21ai731718.html |title = 建工设计院设计前滩国际商务区 “双塔” 项目 |publisher = 中国上海-政务新闻-部门动态 |date = 2013年4月2日 |accessdate = 2013-05-04}}</ref>。

=== 配套设施 ===
为配合主体商务功能，前滩商务区规划配备有道路和管线基建，其中：市政道路及管线工程20条、长约12.4公里；[[黄浦江|黄浦江]]河道整治：将完成1.75公里、总面积7.35万平方米河道改移，以形成“亲水景观”，以上两项工程均将在2013年完成。另外设有：国际学校、滨江公园、体育场馆、桥梁、防汛墙、亲水平台等等生活用的商务配套设施

=== 区域交通 ===
[[File:凌兆新村站.jpg|thumb]]]]

区域内交通较为便捷，有三条[[上海轨道交通|轨道交通线]]附近通过，分别是[[上海轨道交通6号线|轨交6号线]]、[[上海轨道交通8号线|轨交8号线]]和建设中的[[上海轨道交通11号线|轨交11号线南延伸段]]，还有[[上海轨道交通7号线|轨交7号线]]从该区域北边约2公里远附近通过；另有世博大道—耀龙路的延伸线、[[南北高架|南北高架]]南延伸段济阳路、[[中环线|中环线]]—[[上中路隧道|上中路隧道]]等多条主干道或延伸线及多条[[上海公交|市内公共交通线]]经过<ref name="大公报" />。

== 影响及效益 ==
据估算，前滩商务区建成运营后，每年将产生超千亿元的产值，并与陆家嘴金融贸易区形成互补，成为上海城市的副中心，未来将成为一个24小时的国际城市区域，与[[纽约|纽约]]、[[伦敦|伦敦]]等无缝接轨，从而进一步拉动浦东及上海整体经济的发展和城市功能升级<ref>{{cite news |author = 李雯露 |url = http://house.hexun.com/2012-11-26/148331650.html |title = 上海前滩国际商务区进入开发阶段 投资千亿再造陆家嘴 |publisher = 观点地产网 |date = 2012年11月26日 |accessdate = 2013-05-04}}</ref>。同时，由于文化、体育和休闲等配套设施建成后，有助于改善区域文化环境，提升居民生活水平。但也有评论指出，前滩地区的发展可能会成为新一轮的“[[房地产泡沫|地产盛宴]]”，因为商务区的建设会推高区域内及周边地区的[[商品房|商品房]]价格<ref>{{cite news |url = http://news.online.sh.cn/news/gb/content/2012-11/26/content_5701914.htm |title = 前滩国际商务区4项目 2013年底完成市政工程建设 |publisher = 新闻晨报 |date = 2012年11月26日 |accessdate = 2013-05-04 |deadurl = yes |archiveurl = https://archive.is/20130628085043/http://news.online.sh.cn/news/gb/content/2012-11/26/content_5701914.htm |archivedate = 2013年6月28日 }}</ref>。
