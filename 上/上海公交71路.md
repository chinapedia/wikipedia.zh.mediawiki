'''上海公交71路'''，有时又称'''延安路中运量公交线'''，简称'''中运量71路'''或'''71路中运量'''{{NoteTag|“71路中运量”主要用于在2017年初的开通前调试，以防与调度系统中[[上海公交1250路|“原71路”]]（双流路天山路—延安东路外滩）产生冲突。}}，是[[中国|中国]][[上海市|大陆上海市]]的一条地面公共电车路线。目前，该线路行驶于[[延安路中运量公交系统|延安路中运量公交系统]]内，西起[[申昆路枢纽站|申昆路枢纽站]]，东至[[延安东路外滩站|延安东路外滩站]]，主要途经沪青平公路与延安路。全程线路总长17.5 km，设站25组；区间车线路长15.5 km，设站22组。

==历史沿革==
''注：71路公交车原先是由双流路天山路至延安东路中山东一路的日间公共汽车线路，而现有中运量线路与之为线路名及部分线路段（天山路—外滩）上的继承关系<ref name="R71" />（原71路在天山路的线路段由[[上海公交1250路|上海公交1250路]]继承）<ref name="R71-2">{{cite web|author1=[[上海巴士三公司|-{巴士}-三公司]]|title=原71路缩线同步更名1250路|url=http://www.84000.com.cn/NewsDetail.aspx?actionNewsID=3231|website=[[上海久事公交集团|上海-{巴士}-集团]]|date=2017-01-24|quote=经过行业管理部门许可，[[上海巴士三公司|-{巴士}-三公司]]71路从2017年1月28日（周六）首班车起缩线营运并更名为1250路，首末站调整为凯旋路延安西路站至天山路双流路站。}}</ref>。''

民国38年（1949年）1月，中华民国上海市政府开通了3路公共汽车，在中正东路(今[[延安东路_(上海)|延安东路]])到番禺路间行驶。同年，[[上海戰役|上海战役]]爆发，中共占领上海市，其当局接手运营后，更名71路。至1992年时，该线路已经延伸到水城路(即水城路威宁路，原127终点站)。1992年，上海市公交总公司职工投股组成独立经营的股份制车队曾在市内开辟ZX（专线）71路（水城路—延安东路外滩）公交车。<ref name="cnqz">{{cite web|title=长宁区志 >> 第十三编交通邮电 >> 第一章道路客运 >> 第二节 公交车|url=http://www.shtong.gov.cn/newsite/node2/node4/node2249/changning/node15386/node42474/node63012/userobject1ai28063.html|website=上海通网站|publisher=上海市地方志办公室 上海通网站|accessdate=2017-07-07}}</ref>

1975年12月26日，为改善市民的出行条件，方便西部区域职工上下班，71路增开与74路联运的高峰车[[上海公交202路|上海公交202路]]，起始站从西藏路至汤更浪（北翟路绥宁路），后调整为农科院{{NoteTag|即上海农业科学院北门，位于北翟路2901号，今申昆路与七莘路之间。农科院原本占据华漕镇南部及今虹桥机场西部大片土地作为其农场，后部分实验农田被征用为[[虹桥枢纽|虹桥枢纽]]。}}至普安路，仅停靠大世界等主要站点，分流了71路的长途客流<ref name="cnqz"></ref>。2010年3月13日，高峰联运线路202路停运。<ref>{{cite web|title=202路自2010年3月13日起暂停营运|url=http://www.84000.com.cn/NewsDetail.aspx?actionNewsID=2550|website=[[上海久事公交集团|上海-{巴士}-集团]]}}</ref>

1994年10月起，71路实行准无人售票，即每辆铰接车保留一位乘务员监票的同时不再负责票务<ref name="cnqz"></ref>。1995年5月，带有后置客车底盘的SK6104HP型无人售票公共汽车批量投入71路营运 <ref name="gysyz">{{cite web|title=上海公用事业志 >> 第六篇公用事业设备生产 >> 第四章客车底盘与电车控制设备 >> 第三节大客车底盘|url=http://www.shtong.gov.cn/newsite/node2/node2245/node4516/node55033/node55148/node55162/userobject1ai42470.html|website=上海通网站|publisher=上海市地方志办}}</ref>。

2005年8月13日，71路终点站由水城路威宁路改为双流路天山路。2009年3月7日，撤销威宁路仙霞路站点<ref>{{cite web|title=优化路线少走弯路 71路明起部分改道|url=http://61.152.242.65/bssq/servlet/Srv?action_code=viewCorpMovement&Id=13041|website=上海巴士四汽公共交通有限公司|accessdate=2017-07-08|date=2009-03-06|quote="71路西端始末站自2005年8月13日由仙霞路威宁路迁至双流路。"}}</ref>。

2006年9月1日，71路大站车开通，上行经中山东一路、金陵东路、河南中路、延安东路、延安中路（茂名路上高架，凯旋路下高架）、延安西路、天山路、威宁路、新渔东路、双流路，设浙江中路站、西藏中路站、重庆北路站、石门一路站、凯旋路站、遵义路站、古北路站、曹家宅站、双流路站；下行经天山路、延安西路（凯旋路上高架，茂名路下高架）、延安中路、延安东路外滩，设芙蓉江路站、娄山关路站、中山西路站、凯旋路站、重庆北路站、西藏中路站、[[延安东路外滩站|延安东路外滩站]]。该大站快车采取定时不定车的办法，以保障发车的准点率。逢双休日、节假日停驶。<ref>{{cite web|title=71路9月1日起开设大站快车|url=http://61.152.242.65/bssq/servlet/Srv?action_code=viewCorpMovement&Id=10029|website=上海巴士四汽公共交通有限公司|accessdate=2017-07-08|date=2006-08-29}}</ref>

2013年，71路在公交车上安装了多套车载“电子警察”设备，查处占用公交车道违法行为。<ref>{{cite web|title=专拍违法占道 全市公交专用车道的电子警察监控总数达200套|url=http://www.shtong.gov.cn/Newsite/node2/node70344/u1ai136464.html|website=上海通网站|publisher=上海市地方志办公室}}</ref>
[[File:71路在延安东路西藏中路站点载客.jpg|thumb]]
2016年11月，上海市交通委员会发布《延安路中运量系统公交线路优化调整计划》，征求市民意见。根据该方案，71路将转为[[延安路中运量公交系统|延安路中运量公交系统]]内运行，并新开辟到凯旋路延安西路的短驳公交替代该线路在天山路的线路段，是为1250路。<ref>{{cite web|title=《延安路中运量系统公交线路优化调整计划》网上征询市民意见的公告|url=http://www.jt.sh.cn/jtw/gsgg/n39/u1ai52106.html|website=上海交通网站|publisher=上海市交通委员会}}</ref>

2017年1月28日，替代原71路在天山路线路段的短驳线路1250路(双流路天山路—凯旋路延安西路)开通。2017年2月1日0时10分，原71路末班车驶离外滩终点站；同日清晨4点30分，转至中运量公交系统内运行的71路无轨电车由申昆路枢纽站始发，开始对公众试运营。<ref name="paper_kt">{{cite news|title=上海延安路中运量公交开通试运营|url=http://www.thepaper.cn/newsDetail_forward_1610252|publisher=澎湃新闻|date=2017-02-01}}</ref>

2018年[[中国国际进口博览会|中国国际进口博览会]]期间，71路区间车临时改线，不停靠申昆路枢纽站，改为从沪青平公路直达[[国家会展中心（上海）|国家会展中心（上海）]]的P1停车场<ref>{{cite news |author1=曹刚 |title=71路延伸线从P1停车场直接出发 从进博会场馆直通市中心更方便 |url=http://newsxmwb.xinmin.cn/chengsh/2018/11/05/31450479.html |accessdate=2019-04-09 |work=新民晚报 |date=2018-11-05}}</ref>。

== 运营组织 ==

[[File:Z5A-0004_at_W_Yanan_Rd,_Jiangsu_Rd_(20170203111906).jpg|thumb]]附近]]
[[File:A_ZK5120C-Type_Trolleybus_at_Hangdong_Road_Station,_Yan'an_Road_Bus_Medium_Transit_System.jpg|thumb]]
试运行期间安排全程车（71路）和至黄陂北路的区间车（71区间）。全程车高峰间隔4分、低谷间隔8分；区间车高峰间隔5分、低谷间隔10分。全程、区间线隔班交替运营，可以实现高峰期小于3分钟的平均间隔。

运营单位购买了40辆18米宇通牌ZK5180A型电车和28辆12米宇通牌ZK5120C型电车<ref>{{cite web | title = 延安路中运量纯电动城市客车采购及电池租赁项目成交公告  | publisher = 中国政府采购网 | date = 2016-10-12 | url=http://www.ccgp.gov.cn/cggg/dfgg/cjgg/201610/t20161012_7421783.htm | accessdate = 2016-11-30 | language = zh }}</ref>，目前有37辆ZK5180A<ref>{{Cite web|url=http://zh.84000.com.cn:1917/bimp-base/map/lineMap.jsp?lineName=71|title=公交线路信息-71路|accessdate=2017-07-07|author=|date=|publisher=[[上海久事公交集团|上海巴士集团]]}}</ref>{{NoteTag|其中一辆因2018年3月5日发生的意外被砸而无法投入运营，故减去一辆。}}和27辆ZK5120C<ref>{{Cite web|url=http://zh.84000.com.cn:1917/bimp-base/map/lineMap.jsp?lineName=71%E5%8C%BA%E9%97%B4|title=公交线路信息-71区间|accessdate=2017-07-07|author=|date=|publisher=[[上海久事公交集团|上海巴士集团]]}}</ref>获得公共汽电车运营证，投入实际运行。

71路全程为单一票价2元，享受交通卡换乘优惠政策。支付车资方式与常规线路相同，为上车投币或刷卡，而非站台售票或购票进站。线路每车配备乘务员一名，负责引导乘客、解答乘客咨询、协助驾驶员关注[[集电杆|集电杆]]、帮助身心障碍者等，但不负责现金售票事务。

== 线路与站点 ==
=== 线路走向 ===
==== 71路 ====
* 上行：自延安东路外滩（延安东路中山东一路）始，循延安东路、延安中路、延安西路、沪青平公路、申昆路、高虹路至申昆路枢纽站（高虹路申昆路）。
* 下行：自申昆路枢纽站（高虹路申昆路）始，循高虹路、申昆路、沪青平公路、延安西路、延安中路、延安东路至延安东路外滩（延安东路中山东一路）。

==== 71区间 ==== 
* 上行：自黄陂北路下客站（人民广场武胜路）始，循武胜路、延安东路、延安中路、延安西路、沪青平公路、申昆路、高虹路至申昆路枢纽站（高虹路申昆路）
* 下行：回程为自申昆路枢纽站（高虹路申昆路）始，循高虹路、申昆路、沪青平公路、延安西路、延安中路、延安东路、黄陂北路、武胜路至黄陂北路下客站（人民广场武胜路）。

== 客运量 ==
据2018年3月31日《[[新闻晨报|新闻晨报]]》，过去的2017年中，71路客流量不断上升，单日最高客运量已经突破5万乘次。<ref>{{cite news|work=[[新闻晨报|新闻晨报]]|date=2018-03-31|title=上海正研究开辟新中运量公交线 71路中运量单日最高客流突破5万乘次，传统公交线面临较大转型挑战|author=钟晖|page=A01|accessdate=2018-04-02}}</ref>

== 运营事故 ==
[[File:延安路中运量公交系统工程浙江路口非专用道段.jpg|发生事故的位置，正是图中公交车所处的位置。背景就是涉事的海洋大厦。]]
2018年3月5日6时30分许，位于[[延安东路_(上海)|延安东路]]近[[浙江中路|浙江中路]]附近海洋大厦二十九层户外招牌被大风吹落，正巧砸中途经的一辆71路公交车上，车顶被部分砸断，车厢侧面玻璃被砸烂，所幸未造成人员伤亡。上午9时许，事故车辆拖离现场，现场清理完毕。<ref>{{cite news|url=http://newsxmwb.xinmin.cn/shanghaitan/2018/03/05/31365521.html|website=[[新民晚报|新民晚报]]（新民网）|date=2018-03-05|title=延安东路一大厦招牌从楼顶掉下 砸中路过中运量公交车|author=夏韵}}</ref>

== 形象标志 ==
上海公交71路的[[吉祥物|吉祥物]]被称为“量量”，是以该线路使用的宇通牌客车车头为灵感创作的，着绿色T恤；并设计了一个英文翻译错误的形象标志<ref>{{cite web |title=呆萌的方头娃娃：上海首个公交吉祥物即将亮相71路_大申网_腾讯网 |url=http://sh.qq.com/a/20171103/005372.htm |website=sh.qq.com |language=zh-CN}}</ref>。

==评价==
{{See Also|延安路中运量公交系统工程#评价}}
===正面评价===
[[上海市交通委|上海市交通委]]认为，经过运营单位的优化，71路总体运营情况良好，各项指标基本达到预期<ref>{{cite web|author1=新民晚报|title=上海首条中运量公交71路客流稳步增长_央广网|url=http://www.cnr.cn/shanghai/tt/20170521/t20170521_523765302.shtml|website=央广网|language=zh-CN}}</ref><ref>{{cite web|author1=解放网|title=上海延安路中运量71路试运营三个月_央广网|url=http://news.cnr.cn/native/city/20170519/t20170519_523763958.shtml|website=央广网|language=zh-CN}}</ref>。

2017年5月，该系统的设计和EPC总承包单位[[上海市政工程设计研究总院|上海市政工程设计研究总院]]向媒体表示，在3个月的试运营期间，工作日期间71路客流稳定在4万以上，高峰日客流接近5万；早晚高峰平均时速达到每小时18公里以上。试运营初期的单程开行时间为80分钟，目前提速在60分钟左右。71路车速逐步加快，班次准点，“获得较多市民的认可”<ref>{{cite web|author1=解放网|title=中运量公交71路 高峰日客流约5万_央广网|url=http://www.cnr.cn/shanghai/tt/20170416/t20170416_523709764.shtml|website=央广网|language=zh-CN}}</ref>，达到设计初期运营目标。

2018年11月，因[[中国国际进口博览会|中国国际进口博览会]]期间的工作，71路中运量班组获[[上海市总工会|上海市总工会]]颁发的上海[[五一劳动奖状|五一劳动奖状]]<ref>{{cite web |title=上海市五一劳动奖状（奖章）、 工人先锋号候选对象公示 |url=http://www.shzgh.org/renda/node5902/node6520/node6525/u1ai6226221.html |website=上海市总工会 |date=2018-11-15 |quote=上海巴士第三公共交通有限公司71路中运量}}</ref>。

===负面评价===
[[File:上海127路缩线通告.jpg|thumb]]
中运量尚在建设时，有关部门开始调整相关公交线网<ref>{{cite web|author1=上海市交通委员会|title=《延安路中运量系统公交线路优化调整计划》网上征询市民意见的公告|url=http://www.jt.sh.cn/jtw/gsgg/n39/u1ai52106.html|website=上海交通网|accessdate=2017-07-10|date=2016-11-17}}</ref>。根据相关方案，[[上海久事公交集团|上海巴士集团]]将缩线并裁撤925路、[[上海公交936路|936路]]、[[上海公交1250路|原71路]]和[[上海公交1251路|127路]]；合并[[上海公交911路|911路和911区间]]<ref name="R71">{{cite web|author1=李蕾|title=延安路中运量公交拟沿用“71路”，调整22条、新辟4条公交线--上观|url=http://www.shobserver.com/news/detail?id=36720|website=上海观察|accessdate=2016-11-17}}</ref>。由于925路、936路<ref>{{cite web|title=925、936等公交撤销计划遭质疑 乘客担心成本上升_新浪上海_新浪网|url=http://shanghai.sina.com.cn/news/2011-02-15/0243172562.html|website=新浪网上海频道}}</ref>、127路<ref>{{cite web|title=830路缩线至静安寺127路拟全线撤销 A03-上海-新闻晨报|url=http://newspaper.jfdaily.com/xwcb/html/2014-12/21/content_49403.htm|website=新闻晨报网络版|access-date=2017-07-04|archive-url=https://web.archive.org/web/20170728203645/http://newspaper.jfdaily.com/xwcb/html/2014-12/21/content_49403.htm|archive-date=2017-07-28|dead-url=yes}}</ref>、911路和911路区间车<ref name=":6">{{Cite web|url=http://newspaper.jfdaily.com/jfrb/html/2015-12/10/content_156321.htm|title=911路调整之难的改革逻辑 13-解放财经-解放日报|accessdate=2017-10-21|work=newspaper.jfdaily.com|archive-url=https://web.archive.org/web/20171022142416/http://newspaper.jfdaily.com/jfrb/html/2015-12/10/content_156321.htm|archive-date=2017-10-22|dead-url=yes}}</ref>等曾多次列入撤销、缩线计划表，这一行为引发市民不满。有市民因此向公交车或公交车站张贴“大字报”或涂抹张贴在公共区域的有关通告文书等以表不满。

71路开通前夕，有本土派微博用户指责其为“政绩工程”，不过相关提问很快被东方报社隐藏，未予以显示<ref>{{cite web|title=这里是上海-{巴士}-集团，关于71路中运量公交，提问吧!|url=http://www.thepaper.cn/asktopic_detail_10007449|publisher=澎湃新闻|accessdate=2017-02-21}}</ref>。

71路主线与相邻新辟接驳线路换乘站点并非同站，因而导致换乘步行距离较长。原本依靠925路直接前往中心城区的航华新村居民在71路开通、925路撤销后并无新辟接驳线路，出行不便<ref name=xmwktdr>{{cite web|url=http://shanghai.xinmin.cn/msrx/2017/02/01/30800930.html|title=中运量公交车来啦！乘坐舒适获肯定|website=新民网|author=董怡虹|publisher=[[新民晚报|新民晚报]]}}</ref>。

2018年5月11日，有署名为“周元飞”的[[久事公交集团|久事公交集团]]员工向公司公开反应，宣称71路的逃票率达到4成，尤其全程车前半节车存在监票盲区；71路运营单位[[上海巴士三公司|-{巴士}-三公司]]回复称，中运量车队要求乘务员提醒监票，但乘务员也无法对无视提醒者执行强制索要票款、驱赶乘客下车等售票员职权<ref>{{cite web |author1=周元飞 |author2=[[上海巴士第三公共交通有限公司|上海巴士第三公共交通有限公司]] |title=巴士论坛（投诉单号 108216） |url=http://www.84000.com.cn/BbslistView.aspx?conParm=b_Type=3&categoryID=81&Div=Menu36_div |website=[[上海久事公交集团|上海久事公交集团]] |accessdate=2018-08-14 |archiveurl=https://web.archive.org/web/20180814122411/http://www.84000.com.cn/BbslistView.aspx?conParm=b_Type=3&categoryID=81&Div=Menu36_div |archivedate=2018-08-14 |date=2018-05-11}}</ref>。
