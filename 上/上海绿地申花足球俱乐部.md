'''上海绿地申花足球俱乐部'''（{{lang-en|Shanghai Greenland Shenhua FC}}），是位于[[中国|中国]][[上海|上海]]的一家[[职业|职业]][[足球俱乐部|足球俱乐部]]，简称'''上海申花'''。其前身是成立于1951年11月1日的上海市足球代表队，是中国最早的职业足球俱乐部之一。

上海申花足球俱乐部成立于1993年12月10日，<ref>{{cite news|url=http://sh.xinmin.cn/special/wbty/article/2009/06/04/2049201.html|title=足坛闹新事 企业“购”球队 上海申花足球俱乐部今晚挂牌|publisher=新民晚报|author=葛爱平|date=1993-12-10|accessdate=2010-06-17}}</ref><ref>{{cite book|author=韩勇|title=《中国足球俱乐部内幕》|date=1998年11月|publisher=中国城市出版社|isbn=9787507410693|chapter=21 理顺机制，轻装上阵}}</ref> 是[[中国足球甲A联赛|中国足球甲A联赛]]和[[中国足球超级联赛|中国足球超级联赛]]的创始会员之一，并夺得1995年[[中国足球甲级A组联赛|中国足球甲A联赛]]冠军，是一支在中国足球坛上享有盛誉的足球俱乐部，拥有广泛的球迷群体。申花为第一个赞助球队的公司名稱，此后俱乐部认为申花一词存有比较好的寓意（「申」是上海的别称，可称之为「申城之花」），而且知名度比较高，所以一直沿用此名稱，尽管球队的投资方历经变化，球队名称中始终保留了“申花”二字。

2007年2月8日，足球俱乐部被[[第九城市|第九城市]]董事长[[朱骏|朱骏]]收购，与朱骏拥有的另外一支同级联赛足球俱樂部[[上海联城|上海联城]]合并。2014年1月30日，[[绿地控股集团|绿地集团]]宣布正式接手上海申花，朱骏时代宣告终结。<ref>{{cite news|url=http://sports.163.com/14/0131/15/9JU8EDUS00051C89.html|title=绿地宣布接手申花 朱骏时代宣告终结|author=朱翃|publisher=网易转[[新华网|新华网]]|date=2014-01-31|accessdate=}}</ref>2014年2月6日，绿地集团宣布成立上海绿地足球俱乐部。2014年7月起，绿地集团将申花字样以括弧形式加入俱乐部名称中，并表示将于2015年重新注册时正式正名为上海绿地申花足球俱乐部。2014年10月30日，上海绿地申花足球俱乐部证实，经过将近一年的努力，绿地集团已经正式走完了对上海申花联盛足球俱乐部所有的收购流程。<ref>{{cite news |language=zh-hans |url=http://sports.sina.com.cn/j/2014-10-31/10527390413.shtml |title=申花证实彻底完成转让 明年俱乐部名加回申花 |publisher=[[新浪|新浪]] |date=2014-10-31 |accessdate=2014-10-31}}</ref>2015年1月4日，“申花”两个字正式加入到俱乐部的名称当中，更名为上海绿地申花足球俱乐部。<ref>{{cite news |language=zh-hans |url=http://sports.sina.cn/csl/lvdi/2015-01-05/detail-iavxeafr9646899.d.html?from=mbaidu&stun=20007&vt=4|title=绿地正式将申花加入俱乐部名|publisher=[[新浪|新浪]] |date=2015-01-05|accessdate=2015-01-05}}</ref>
2017年11月26日，上海绿地申花比分3比3打平上海上港，因客场进球原则，获得2017年足协杯冠军，时隔19年之后再次获得中国足协杯冠军<ref>{{Cite web|url=http://www.sohu.com/a/206771631_463728|title=解气！申花时隔19年终再夺足协杯 5进决赛夺两冠|accessdate=2018-02-19|date=2017-11-26|work=www.sohu.com}}</ref>。

== 主场与训练基地 ==
[[File:虹口足球场-中国足球超级联赛2019年第一轮.jpg|缩略图]]
其主场位于中国第一座专业足球场[[虹口足球场|虹口足球场]]，有时亦会改在上海其他体育场或者专业足球场舉辦賽事，例如於1998赛季改在[[上海体育场|上海体育场]]，2006赛季亚冠联赛及2007赛季大部分比赛改在[[源深体育中心|源深体育中心]]体育场，於2007赛季女子世界杯期间改在[[金山足球场|金山足球场]]；此外，俱乐部也曾經将部分足协杯主场比赛改在[[江湾体育场|江湾体育场]]、[[闸北体育场|闸北体育场]]、芜湖奥体中心以及部分中超杯主场比赛改在嘉定体育场等。2001年，球队一线队住宿及训练由[[江湾体育场|江湾体育场]]搬到了目前使用的[[康桥基地|康桥基地]]；此外，原[[上海联城足球俱乐部|上海联城足球俱乐部]]的[[青浦华新基地|青浦华新基地]]曾在朱骏时代（2007年-2013年）负责俱乐部预备队的住宿及训练。

== 球队历史 ==
{{Main|上海申花足球俱乐部历史}}

1993年12月10日，上海足球史上第一家职业足球俱乐部──'''上海申花足球俱乐部'''成立。是中国第一家完全脱离体委、足协等政府行政机制的职业足球俱乐部。俱乐部的第一任主教练[[徐根宝|徐根宝]]提出“抢逼围”的口号，力争改变传统的上海足球技术意识出色但是身体拼抢吃亏的情况。

1995年，这一举措获得了巨大的成功，申花取得了联赛十连胜，提前两轮夺得联赛冠军，并涌现出了[[范志毅|范志毅]]、[[谢晖|谢晖]]、[[祁宏|祁宏]]、[[吴承瑛|吴承瑛]]等一大批著名球星。上海二纺机足球俱乐部（原上海体育学院）1994，1995年两次参加乙级联赛后并入上海申花。

1997年，俱乐部为了改变球队的打法，开始坚持聘请外籍主教练，其中包括[[拉扎罗尼|拉扎罗尼]]、[[伊利亚·佩特科维奇|伊利亚·佩特科维奇]]等带队参加[[世界杯足球赛|世界杯决赛圈]]的主帅，以及获得过[[欧洲冠军杯|欧洲冠军杯]]的[[柳普科·彼德洛维奇|柳普科·彼德洛维奇]]。不过申花在此期间始终未能压倒中国足坛的霸主[[大连海昌足球俱乐部|大连队]]，最终只获得了四个联赛亚军和一个[[中国足协杯|足协杯]]冠军，但是球队的风格则成功向技术流派转型。

2000年，赛季开始前，俱乐部股权发生变化，原先的投资方上海申花集团退出，由七家[[上海|上海]]大型[[国有企业|国有企业]]共同参股组建新的上海申花股份有限公司，原投资者申花集团不占资产，不过新组建的公司保留了“申花”的名称，并继续留用了原来经营俱乐部的董事长[[郁知非|郁知非]]为总经理。

2001年12月19日，俱乐部所有权再次发生变化，并更换了管理层并对教练组和球队进行了大换血，上海广电集团和[[上海文广集团|上海文广集团]]这两家[[上海|上海]]大型国有企业接手俱乐部，俱乐部更名为'''上海申花SVA文广足球俱乐部'''。此后，同为上海大型国企的[[上海电气集团|上海电气集团]]也收购俱乐部部分股权。俱乐部兼并了第一任主帅徐根宝创立的[[上海有线02足球俱乐部|上海有线02足球俱乐部]]，从而获得了[[杜威_(足球运动员)|杜威]]、[[孙祥|孙祥]]等年轻球员。但是球队原有的核心[[申思|申思]]、[[祁宏|祁宏]]、[[吴承瑛|吴承瑛]]却也先后转会到新出现的同城对手[[陕西中新|上海中远国际队]]，上海足坛形成了两强争霸的局面。

2003年，申花以一分优势压倒国际，夺得了[[2003年中国足球甲A联赛|最后一届]][[甲A联赛|甲A联赛]]的冠军。但是由于当时球队为了夺冠而行贿，冠军头衔于2013年2月18日被收回。<ref>{{cite web |url = http://sports.sina.com.cn/c/2013-02-18/18356426082.shtml |title = 反赌处罚:申花泰达扣6分罚100万 剥夺申花03冠军 |publisher = 新浪网 |date = 2013-02-18 |accessdate = 2013-02-18}}</ref>

2007年2月8日，俱乐部的主管者发生了第二次重大变更，另一支中超球队[[上海联城足球俱乐部|上海联城]]的投资人[[朱骏|朱骏]]收购申花俱乐部，将[[申花联城合并|两支球队合并]]，俱乐部更名为'''上海申花联盛足球俱乐部'''。新成立的申花俱乐部获得了2007年[[A3联赛|A3联赛]]冠军以及[[2008年中国足球超级联赛|2008年中国足球超级联赛]]亚军，这也是俱乐部历史上第八次获得职业联赛亚军。

2009年，赛季结束后，俱乐部董事局主席[[朱骏|朱骏]]亲赴欧洲，聘请到了曾获得世界杯季军的名帅[[米洛斯拉夫·布拉泽维奇|布拉泽维奇]]出任2010赛季的主教练。同时俱乐部签入了多名年轻球员，开始阵容重组。年轻的申花队赛前并不被人看好，但是却在布拉泽维奇的带领下表现异常出色，在第一阶段联赛休战时处于积分榜领先位置。不过在夏季引援中球队未能有效增补外援，下半赛季随着停赛及伤病等情况的反复出现，暴露出明显的板凳深度不足问题，被山东鲁能反超，最终获得季军，重返亚冠赛场。赛季中外援[[杜维尔·里亚斯科斯|里亚斯科斯]]表现突出，打破了申花球员单季入球纪录，[[冯仁亮|冯仁亮]]、[[吴曦|吴曦]]、[[宋博轩|宋博轩]]等新加盟且第一次参加顶级联赛的球员也凭借自己的表现入选了国字号球队。

2011年，失去了布拉泽维奇的申花由原中方助理教练奚志康出任主教练，征战亚冠与中超联赛。新加盟的阿根廷射手[[路易斯·萨梅隆|路易斯·萨梅隆]]表现突出，一度长期占据中超联赛射手榜第一。球队在2011赛季亚冠联赛小组赛中以6战0胜2平4负仅积2分小组垫底的成绩惨遭淘汰，同时该战绩也是历届亚冠联赛中国球队的最差战绩。虽然球队在联赛第一阶段进入三甲，但是赛程进入夏季之后，申花战绩出现巨大滑坡，经历了俱乐部历史上战绩最差的一段时期。其间主教练奚志康离任重新担任助理教练，由去年辅助布拉泽维奇的德拉岑·拜塞克接任主教练，最终名列第十一名，为队史第二差名次。

2012赛季开赛前，上海申花俱乐部大手笔地引进了法国球星[[尼古拉·阿内尔卡|尼古拉·阿内尔卡]]与法国名帅[[让·蒂加纳|让·蒂加纳]]，使得球队的联赛夺冠呼声甚至超过了卫冕冠军广州恒大。但由于球队战绩不佳以及与俱乐部高层矛盾激化，蒂加纳宣告辞职。球队助理教练让·弗洛伦特·伊本吉成为代理主教练，阿内尔卡成为球队队长兼助理教练。5月30日，上海申花队宣布前阿根廷国家队主教练[[塞尔吉奥·巴蒂斯塔|塞尔吉奥·巴蒂斯塔]]成为申花队主教练。6月20日，英超豪门切尔西的当家球星[[迪迪埃·德罗巴|迪迪埃·德罗巴]]通过自己的官方网站宣布了加盟申花的消息，成为[[阿内尔卡|阿内尔卡]]后同一赛季中上海申花引进的又一位国际巨星。在德罗巴与另一位新外援哥伦比亚国脚[[吉奥瓦尼·莫雷诺|吉奥瓦尼·莫雷诺]]帮助下，上海申花成绩出现了反弹，联赛排名升至第8名，依然保有冲击亚冠资格的机会。但联赛第27轮上海申花2-4惨败贵州人和，亚冠梦想提前破灭。最终申花队以联赛第9名收场。

2013赛季开始之前，申花斥巨资引进的两名外援[[阿内尔卡|阿内尔卡]]与[[德罗巴|德罗巴]]相继转会离开。2月18日，中国足协收回球队2003年中国足球甲A联赛冠军头衔并让上海申花队从负6分开始比赛。虽然申花再陷“欠薪门”，三名南美外援曾因欠薪拒绝比赛，主帅[[塞尔吉奥·巴蒂斯塔|巴蒂斯塔]]也被欠薪，但申花在联赛中保持着良好的上升势头，联赛前八轮五次在落后的情况下完成逆转，上海申花也被媒体冠以“落后的申花惹不起”之名。赛季中期，申花宣布巴蒂斯塔因为个人原因辞职。随后也爆出申花俱乐部对巴蒂斯塔欠薪的消息，中方教练组组长[[沈祥福|沈祥福]]即时出任申花主教练。最终，从负六分起步得上海申花以38分的积分排名第八。（如加上6分扣分，申花队排名中超第4位）

2013赛季结束后，球队球队一度传闻朱骏将带队西迁云南，入主红塔基地，但受到了来自其他五大股东以及来自政府和上海市体育局的阻止而未果。2014年1月30日，俱乐部投资人朱骏退出，上海的国有企业[[绿地控股集团|绿地控股集团]]接手球队。2月6日，俱乐部正式更名为'''上海绿地足球俱乐部'''。2014赛季以'''上海绿地申花队'''的名义参加中超联赛。由于「申花」两字被从俱乐部名称中正式剔除，在球迷中反响激烈，许多上海球迷无法接受这样的队名变动，并且疑似违反了《中国足球协会关于中超俱乐部产权转让的规定》<ref>{{cite web |url = http://sports.sina.com.cn/j/2014-02-07/01227007448.shtml |title = 再见了申花!上海申花俱乐部名字没了 球迷流泪无眠 |publisher = 新浪体育 |date = 2014-02-07 |author = 邰晓鹏}}</ref><ref>{{cite web |url = http://sports.sina.com.cn/j/2004-10-10/14501187021.shtml |title = 《中国足球协会关于中超俱乐部产权转让的规定》  |publisher = 中超委员会 |date = 2013-02-18 |accessdate = 2013-02-18}}</ref>。迫于球迷压力，2014年7月起，绿地集团将申花字样以括弧形式加入俱乐部名称中，并在赛季中期更改了球队队徽。球队在赛季中的表现也差强人意，先后在主场0-3负于老对手[[北京国安|北京国安]]，终结了20年来虹口对国安的不败纪录；在足协杯半决赛中又以两回合0-5惨败[[江苏舜天足球俱乐部|江苏舜天]]。在最后一场主场比赛中更是以2-5惨败[[天津泰达|天津泰达]]，最终球队在联赛中排名第9，杯赛中止步半决赛，球队的糟糕表现也导致了再度接手球队的主教练[[塞尔吉奥·巴蒂斯塔|巴蒂斯塔]]在赛季结束后下课，法国人[[弗兰西斯·吉约|弗兰西斯·吉约]]接任球队主帅。

2014年10月30日，上海绿地申花足球俱乐部证实，经过将近一年的努力，绿地集团已经正式走完了对上海申花联盛足球俱乐部所有的收购流程。<ref>{{cite news |language = zh-hans |url = http://sports.sina.com.cn/j/2014-10-31/10527390413.shtml |title = 申花证实彻底完成转让 明年俱乐部名加回申花 |publisher=[[新浪|新浪]] |date = 2014-10-31 |accessdate=2014-10-31}}</ref>2015年1月4日，“申花”两个字正式加入到俱乐部的名称当中，更名为'''上海绿地申花足球俱乐部'''<ref>{{cite news |language = zh-hans |url = http://sports.sina.cn/csl/lvdi/2015-01-05/detail-iavxeafr9646899.d.html?from=mbaidu&stun=20007&vt=4 |title = 绿地正式将申花加入俱乐部名 |publisher=[[新浪|新浪]] |date=2015-01-05 |accessdate = 2015-01-05}}</ref>。

2017赛季的中国足协杯两回合决赛当中，上海绿地申花在11月19日的主场比赛以1-0战胜上海上港。一周之后11月26日的客场比赛上海申花虽以2-3负于上海上港，打成总比分3-3，但因客场进球规则，获得2017年足协杯冠军。这是上海申花时隔19年之后再次获得中国足协杯冠军，同时获得了队史第七次参加亚足联冠军联赛的资格。

于2018年初，上海绿地申花开始了收购[[根宝足球基地|崇明根宝足球俱乐部]]99/00年龄段的梯队球员的行动。在经数月的与同城对手的竞标之下，于2018年4月成功将该梯队纳入球队当中。梯队中青年才俊众多，数人代表上海全运会U18队获得冠军，且在代表中国国家U19代表队的几场比赛中，也有上佳的表现，被认为是带来希望的梯队。2018年7月的夏季转会窗口报名中，四名来自该梯队的球员获得了一线队报名<ref>{{Cite web|url=http://shenhua.online.sh.cn/content/2018-03/16/content_8817925.htm|title=申花收购根宝9900梯队花费将超1亿 恒大报价1.5亿——上海热线申花频道|accessdate=2018-07-18|last=小飞龙|work=shenhua.online.sh.cn}}</ref>。
