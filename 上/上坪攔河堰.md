'''上坪攔河堰'''，又稱為'''燥樹排攔河堰'''，簡稱'''上坪堰'''，是[[臺灣|臺灣]]的一座[[攔河堰|攔河堰]]，位於[[頭前溪|頭前溪]]支流[[上坪溪|上坪溪]]下游，行政區位處[[新竹縣|新竹縣]][[竹東鎮|竹東鎮]]燥樹排與[[橫山鄉_(臺灣)|橫山鄉]]田寮村上頭排交界附近，主要功能是提供[[寶山水庫|寶山水庫]]、[[寶山第二水庫|寶山第二水庫]]、[[台灣電力公司|台灣電力公司]][[桂山發電廠|桂山發電廠]]軟橋機組與[[竹東圳|竹東圳]]所需水源供公共給水、[[灌溉|灌溉]]與[[水力發電|水力發電]]之用<ref name="北水局"/>，目前由新竹農田水利會、台灣自來水公司及經濟部水利署北區水資源局共同組成「上坪攔河堰管理小組」，負責上坪堰操作、維護、搶修及更新管理<ref name="上坪堰水門操作規定"/>。

== 沿革 ==
1981年[[寶山水庫|寶山水庫]]興建時，[[台灣省政府|台灣省政府]]水利局在新竹縣竹東鎮燥樹排興建「燥樹排攔河堰」，提供[[竹東圳|竹東圳]]灌溉用水來源，以及軟橋水力發電廠發電用水，並提供寶山水庫水源。1996年[[賀伯颱風|賀伯颱風]]時，燥樹排攔河堰受損，經[[台灣自來水公司|台灣自來水公司]]修復。2004年8月[[艾利颱風|艾利颱風]]來襲帶來豪雨與洪水，又造成燥樹排攔河堰毀損，壩基砂礫石基礎流失而懸空，溢流堰除靠近左、右岸之分塊外，縱向收縮縫後段大致均流失，下游混凝土固床工及右岸砌卵石護坡亦遭洪水沖毀<ref name="基本概況"/>，風災後築設緊急導水堤以順利輸水<ref name="復水又停水 民眾氣炸"/><ref name="水利署因應新竹地區用水需求已積極調配水源，保證不影響竹科用水"/>。[[經濟部水利署北區水資源局|經濟部水利署北區水資源局]]於2006年辦理上坪堰復建工程，位於左堰座上游之進水口則配合[[寶山第二水庫|寶山第二水庫]]之興建而辦理改建，改建後之設計取水量為20 cms，復建工程於2007年完工使用<ref name="基本概況"/>。

== 諸元 ==
上坪堰壩長70.5公尺，壩堰高10.5公尺，屬於[[混凝土|混凝土]][[重力壩|重力壩]]<ref name="新竹縣現有水庫壩堰"/>。攔河堰上游左岸設有沉砂池及進水口，連接取水隧道，供應現有竹東圳、寶山水庫及寶二水庫用水<ref name="上坪堰水門操作規定"/>。而竹東圳並兼具提供民生用水與[[新竹科學園區|新竹科學園區]]工業用水之水源外，在竹東軟橋里附近利用圳路落差在軟橋地方設置一座小型[[水力發電廠|水力發電廠]]，是為[[軟橋發電廠|軟橋發電廠]]，由[[台電公司|台電公司]][[桂山發電廠|桂山發電廠]]遙控管理，是一座迷你型發電廠，可供竹東地區用電<ref name="竹東圳的歷史與新生"/>。此外為了維護生態，上坪堰設有[[魚道|魚道]]。
