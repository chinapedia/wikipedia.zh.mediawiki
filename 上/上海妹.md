'''上海妹'''（{{bd|1909年||1954年|9月16日}}）原名'''顏思莊'''，[[廣東省|廣東省]][[中山縣|中山縣]]人，出生於[[新加坡|新加坡]]，是一名[[粵劇|粵劇]]演員。上海妹的[[父親|父親]]是名伶[[顏傑卿|太子友]]，其夫是-{丑}-生[[半日安|半日安]]，而其兄是班政家[[顏蝦|顏蝦]]。上海妹擅演溫柔賢淑的悲劇人物，演出細膩，咬字清晰，自創的反線中板，婉轉幽怨，稱為「妹腔」。 

== 演出經歷 ==
上海妹生於梨園世家，幼從父學，亦曾受學於男花旦[[余秋耀|余秋耀]]及女花旦[[醒醒群|醒醒群]]，6歲初踏臺板，自小已在星馬[[南洋|南洋]]各地走埠演出。

1931年受聘[[馬師曾|馬師曾]]到[[美國|美國]]登台，這段時間結識馬師曾的徒弟半日安，日後更結為夫婦。1930年代初[[香港|香港]]粵劇男女同班解禁，上海妹與[[譚蘭卿|譚蘭卿]]一同投身馬師曾組織的[[太平劇團|太平劇團]]，演出《龍城飛將》、《國色天香》等劇目。後獲[[唐雪卿|唐雪卿]]賞識，招攬上海妹及[[半日安|半日安]]一同加盟[[覺先聲劇團|覺先聲劇團]]，與[[薛覺先|薛覺先]]合作演出，後更成為劇團唯一的正印花旦，演出劇目包括《[[胡不歸|胡不歸]]》、《玉梨魂》、《嫣然一笑》、《前程萬里》、《[[王昭君|王昭君]]》、《[[藍袍惹桂香|藍袍惹桂香]]》、《[[西施|西施]]》、《含笑飲砒霜》等，其中上海妹、薛覺先及半日安合演的《胡不歸》大受歡迎，被觀眾稱為「原裝胡不歸」。

1941年[[聖誕節|聖誕節]][[香港淪陷|香港淪陷]]，翌年上海妹半日安夫婦演罷[[明治劇團|明治劇團]]的班期後，終於有機會離開香港到當時[[中立|中立]]的[[澳門|澳門]]，先後組織或加入國華、[[國壽年劇團|國壽年]]、新馬及好景等劇團演出。不久夫婦兩人離開澳門返回內地，與覺先聲第二小生[[呂玉郎|呂玉郎]]、[[呂雁聲|呂雁聲]]成立大中華劇團，以演薛派劇目為主。和平後，夫婦兩人才返回香港，曾參加新聲、覺先聲、覺華、[[錦添花劇團|錦添花]]、[[麗春花劇團|麗春花]]等劇團演出，她的最後粵劇舞臺演出為1953年10月為[[麗春花劇團|麗春花]]演出《七星伴月》（[[紅樓夢|紅樓夢]]），自始一病不起至病逝<ref>1954年9月17日(星期五)，香港《工商日報》，第五頁。</ref>。

== 香港電影明星 ==
上海妹自銀幕處女作《沙三少》（1935年）至她的電影遺作《女兒香》（1951年），演出13齣電影，全部均為黑白粵語電影。
* 1935年9月：《沙三少》，飾演銀姐；
* 1937年5月：《神秘之夜》；
* 1937年11月：《七十二行》；
* 1939年1月：《第八天堂》；
* 1939年6月：《半面西施》；
* 1939年11月：《醫死閻羅王》；
* 1941年3月：《左慈戲曹操》，飾演崔鳳嬋；
* 1941年7月：《冰山火線》，飾演夏家璧；
* 1948年7月：《好女兩頭瞞》；
* 1948年10月：《雙人頭賣武》；
* 1949年12月：《大鬧粉粧樓》，反串飾演龍標；
* 1951年3月：《銀海春光》，飾演粵劇紅伶，此片紀念《伶星日報》創刊19週年攝製，邀得30多位影星及導演客串演出。
* 1951年10月：《女兒香》，飾演粵劇紅伶。

== 上海妹的演出特點 ==
[[譚伯葉|譚伯葉]]先生對上海妹的演出評語<ref>1960年7月9日（星期六），香港《大公報》，第二張，第七版。</ref>：
* 上海妹擅演悲劇、賢妻良母型的文場戲。
* 唱法，字露、腔好；然而聲、氣都不足，但她能巧妙掩蓋，善於藏拙。
* 做手「乾淨俐落」，臺步更是一絕。
* 上海妹的劇路與另一名伶[[千里駒|千里駒]]類似。

== 肺病逝世 ==
1954年9月16日（星期四）下午4時50分在[[九龍|九龍]][[尖沙嘴|尖沙嘴]][[柯士甸道|柯士甸道]]108號四樓的住所，因[[肺病|肺病]]逝世，享年45歲，彌留時丈夫半日安尚在[[越南|越南]]登臺表演，遺下當時19歲的長女李若玲及6歲的幼子李舜華。1954年9月18日（星期六）下午2時在[[萬國殯儀館|萬國殯儀館]]出殯，依她的[[遺言|遺言]]在[[鑽石山|鑽石山]]舉行火葬<ref>1954年9月17日（星期五），香港《工商日報》，第五頁。</ref>。
