《'''衛斯理'''》（'''{{lang|en|The 'W' Files}}'''），[[香港|香港]][[無綫電視|無綫電視]][[翡翠台|翡翠台]][[劇集|劇集]]，於2003年6月2日至2003年7月11日首播，共30集，由[[羅嘉良|羅嘉良]]及[[蒙嘉慧|蒙嘉慧]]領銜主演，由[[郭晉安|郭晉安]]及[[向海嵐|向海嵐]]特別演出，並由[[唐文龍|唐文龍]]、[[楊怡|楊怡]]、[[楊明|楊明]]及[[麥長青|麥長青]]聯合主演，監製[[張乾文|張乾文]]。

故事改編自[[科幻小說|科幻小說]][[衛斯理系列|衛斯理系列]]。八個被改編的故事包括《[[紙猴|紙猴]]》（《地底奇人》）、《[[屍變_(小說)|屍變]]》、《[[木炭_(小說)|木炭]]》、《[[尋夢|尋夢]]》、《[[蠱惑_(小說)|蠱惑]]》、《[[神仙_(小說)|神仙]]》、《[[鬼混|鬼混]]》和《[[盜墓_(小說)|盜墓]]》。故事背景則設定為民初的[[上海|上海]]。

==與原著不同處==
本劇故事背景設定為1930年代的[[上海|上海]]，有別於原著的1960至80年代的[[香港|香港]]，而當中角色和情節都與原著也很大的差異：

# 原著中衛斯理與白素一早結為夫妻，但劇中他們尚未結婚。
# 在原著中的[[尋夢|尋夢]]一書內，劉麗玲是模特兒，而非電視劇中的歌手；白素一直沒有向衛斯理提出分手，但電視劇卻出現了此情節。
# 《[[蠱惑_(小說)|蠱惑]]》、《[[神仙_(小說)|神仙]]》兩個故事在劇中同時進行，但原著兩個故事發生的時間相距超過三十年 (《蠱惑》寫於1968年，但故事卻發生在1940年代末；而《神仙》寫於1982年，故事發生在1980年代初)。兩個故事之間並無關連。
# 《蠱惑》劇中被芭珠落蠱毒的是白奇偉，但原著中被芭珠落蠱毒的是衛斯理大學同學葉家祺。而原著中白奇偉沒有娶妻，亦不認識芭珠。
# 《神仙》所處的山洞，則在芭珠族人居地附近，他們所指的神山中。白奇偉身上的蠱毒被賈玉珍所吸的仙氣所化解。而賈玉珍則在原著中需要修練百多年才能成正果，但劇中只是一陣子就修成正果，而神仙在原著是不願意見人，並不懂現代的說話，而劇中則是兩名道人，並能與衛斯理直接溝通。
# 《[[鬼混|鬼混]]》中原著是[[何藍絲|何藍絲]]是猜王徒弟及與[[溫寶裕|溫寶裕]]破鬼混降，劇中變成[[黃紅紅|黃紅紅]]。
# 《[[盜墓_(小說)|盜墓]]》中原著與劇中的意義完全不同，原著中古墓中的只是外星人的屍體，但劇中真的到古墓中救出外星人。
# 故事裏铁蛋成了权倾一时的大将军剧中变成了一個警官。
# 小說中，溫寶裕的女友為何藍絲，而非黃紅紅。同時，胡說並未有追求過白素。
# 小郭於小說中的確為出色的偵探，而他正是偵探社的老闆。
# 小說中：衛斯理是出入口公司的董事長：而非偵探社及當舖老闆。
