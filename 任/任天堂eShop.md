'''任天堂eShop'''（{{lang-ja|ニンテンドーイーショップ}}，{{lang-en|Nintendo eShop}}）是由[[任天堂|任天堂]]建立，由[[任天堂Network|任天堂Network]]提供支持的[[数字发行|数字发行]]平台；目前主要为[[任天堂3DS家族|任天堂3DS]]、[[Wii_U|Wii U]]和[[任天堂Switch|任天堂Switch]]提供軟體銷售下載等服務。除遊戲以外，它也提供應用程式、以及3D版电影等下载。任天堂eShop於2011年6月在美國、日本、欧洲和澳洲开始服务。<ref name=":2">{{cite web|url=http://www.nintendo.co.jp/3ds/eshop/about/index.html|title=ニンテンドーeショップで、できること|accessdate=2012-10-02|publisher=[[任天堂|ニンテンドー]]|archiveurl=https://web.archive.org/web/20121016060759/http://www.nintendo.co.jp/3ds/eshop/about/index.html|archivedate=2012-10-16|deadurl=yes}}</ref>

任天堂3DS透過系統更新後，主畫面上會多出任天堂eShop的選項。Wii U和任天堂Switch均在推出時即有任天堂eShop图标选项，但都需要經過系統更新後才能連接。<ref name=":0">{{Cite news|url=https://www.nintendo.co.jp/support/switch/system_update/index.html|title=本体更新情報｜Nintendo Switch サポート情報｜Nintendo|newspaper=任天堂ホームページ|accessdate=2017-11-06}}</ref>Wii U和任天堂Switch均支援多工處理，玩者在遊戲進行中也可進入任天堂eShop，並能夠在背景中進行下載。<ref name=":1">[http://www.nintendolife.com/news/2012/11/wii_u_will_require_day_one_update_for_key_features Wii U Will Require Day One Update for Key Features - Wii U News @ Nintendo Life]. Nintendolife.com. Retrieved on 2013-08-23.</ref>

== 功能特色 ==
任天堂eShop的商標按鈕會出現在任天堂3DS、Wii U和任天堂Switch的主選單畫面中，在主機連線至網際網路時才可使用。

eShop會紀錄所有的下載和購買記錄，玩者可重新下載曾經購買過的商品而不需額外付費。購買後可立即開始下載，或加入排程中等到主機閒置時再行下載。<ref name="Kotaku">{{cite web|url=http://kotaku.com/5854187/nintendo-adding-playable-demos-eshop-web-interface-and-sleep-mode-downloads-to-3ds|title=Nintendo Adding Playable Demos, eShop Web Interface and Sleep Mode Downloads to 3DS|author=Evan Narcisse|work=Kotaku.com|date=2011-10-28|accessdate=2011-11-07}}</ref>曾擁有[[任天堂DSi|任天堂DSi]]主機的玩者可將[[DSiWare|DSiWare]]的購買記錄轉移至任天堂3DS，但部分DSi專用的軟體則無法轉移（例如活動日誌（{{lang|ja|うごくメモ帐}}）或DSi的網路瀏覽器）。<ref>{{cite web|url=http://www.nintendo.com/consumer/systems/3ds/en_na/gi_index.jsp?menu=applications%20and%20features&submenu=ctr-gi-apps-dsiware-wont-transfer|title=Nintendo 3DS - General Information|publisher=Nintendo|accessdate=2011-08-06}}</ref> 在2011年12月推出的系統更新後，玩者也可以在3DS主機間轉移購買記錄。<ref>[http://www.ripten.com/2011/10/21/3ds-november-firmware-update-details/ 3DS November Firmware Update Detailed | RipTen Videogame Blog]. Ripten.com (2011-10-21). Retrieved on 2013-08-23.</ref>

=== 貨幣與支付方式 ===
任天堂過往的網路商店[[Wii_Shop_Channel|Wii Shop Channel]]和[[DSi_Shop|DSi Shop]]使用[[任天堂點數|任天堂點數]]（Nintendo Point）作為價格的標示方式，而任天堂eShop則直接以各地區的貨幣來標示軟體價格。<ref>{{cite web|url=http://www.gamespot.com/features/6316665/the-3ds-eshop-what-you-need-to-know/?tag=updates%3Beditor%3Ball%3Btitle%3B4|title=The 3DS eShop: What You Need to Know|author=Giancarlo Varanini|work=GameSpot.com|publisher=CNet|date=2011-06-02|accessdate=2011-11-07}}</ref>玩者可使用[[信用卡|信用卡]]或商店內銷售的[[預付卡|預付卡]]來將金錢儲值至帳號內。<ref>{{cite web|url=http://www.nintendo.com/consumer/systems/3ds/en_na/ht_apps.jsp|title=Nintendo 3DS - How To - Applications & Features|work=support.nintendo.com|accessdate=2011-11-07}}</ref>也可以不对账号充值，直接使用信用卡来购入游戏。

玩家也可以在其他购物平台（包括实体商店和网络商店）直接购入游戏的下载码（包括实体下载卡和数字下载码），然后在eShop中输入它来进行下载。

在任天堂Switch平台上，目前已有多个国家的eShop新增了对[[PayPal|PayPal]]的支持。

=== 多工處理 ===
Wii U上的任天堂eShop可在任何時間透過主畫面進入，即使在遊戲進行的過程中也可連接。透過[[SpotPass|SpotPass]]則可進行背景下載，玩者可同時繼續遊玩遊戲或使用其他的Wii U功能。[[任天堂3DS|任天堂3DS]]則支援在睡眠模式中進行背景下載。<ref>{{cite web|last=Narcisse|first=Evan|title=Nintendo Adding Playable Demos, eShop Web Interface and Sleep Mode Downloads to 3DS|url=http://kotaku.com/5854187/nintendo-adding-playable-demos-eshop-web-interface-and-sleep-mode-downloads-to-3ds|publisher=Kotaku}}</ref>目前下載排程中每次最多可排入10個項目。下載的進度可在主畫面中的「下載管理」中確認，當下載完成後畫面的右上角也會出現訊息提示。

任天堂Switch的eShop也支持后台下载功能，但是当玩家游玩需要联机的游戏时，后台下载会被停止。

=== 評分系統 ===
任天堂eShop可讓使用者對遊戲、應用程式和其他媒體內容發表評分。使用者可在最低一顆星至最高五顆星間對軟體進行評價。遊戲作品可依照適用的年齡、性別或玩家屬性進行分類。評分系統在Wii U也與[[Miiverse|Miiverse]]整合。使用者必須至少使用軟體超過一個小時後才能對其發表評分。

任天堂3DS和任天堂Switch的eShop不支持评分功能。

=== 任天堂Network Premium ===
在2012年9月13日的[[任天堂Direct|任天堂Direct]]發表影片中，社長[[岩田聰|岩田聰]]宣布了「Nintendo Network Premium」（北美地區稱為「Deluxe Digital Promotion」）的會員計劃。購買Wii U 32G豪華版（Premium Pack，北美洲稱為Deluxe Pack）玩家可免費獲得兩年的會員資格。會員在Wii U的任天堂eShop購買遊戲後可享有「原商品售價10%」的任天堂點數回饋，集滿500點後即可至活動網頁兌換500円回饋金，之後便可在Wii U和3DS的eShop中使用。

這項促銷活動計劃至2014年12月為止，點數回饋金的兌換則到2015年3月底。

== 適用地区 ==

在任天堂3DS和Wii U平台上，玩家可登陆的eShop和主机版本绑定，只能登陆主机所属的国家/地区的eShop。而在任天堂Switch平台上，玩家可登陆的eShop并不和主机绑定，也不会由玩家所在地区的IP来决定，而是根据玩家任天堂账户的设置来选择适用地区，解除了任天堂多年来的“锁区”政策。玩家可以随意变更自己的任天堂账户设置，在不同地区的eShop购入游戏。

香港地区的任天堂Switch eShop于2018年4月3日上线；但香港版在开放之初不能像其他地区的eShop一样浏览游戏信息或进行联机游戏；仅能使用下载码来下载数字版游戏和DLC（需至其他指定网上商店购买下载码），或是重新下载曾有的数字版游戏。<ref>{{Cite web|url=http://www.nintendo.com.hk/pressrelease/switch_eshop_20180306.htm|title=Nintendo Switch的Nintendo eShop使用指引|accessdate=2018-03-07|author=任天堂(香港)有限公司|date=|work=www.nintendo.com.hk|publisher=}}</ref><ref>[https://www.g-cores.com/articles/96749 任天堂NS港服将于4月3日上线].机核网.2018-03-06.[2018-03-06].</ref>韩国区的任天堂Switch eShop也以和香港版类似的形式于同日上线。截至2019年4月，任天堂Switch台湾地区的eShop尚未开放。
