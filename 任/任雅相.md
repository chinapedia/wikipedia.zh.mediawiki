'''任雅相'''（{{bd|?||662年|3月9日|catIdx=R任}}<!-- 生年无具体记载。如果有新的发现，请提供来源。 -->），[[唐朝|唐朝]]将官，[[唐高宗|唐高宗]]年间短暂拜相，封'''安乐县公'''。

== 家世 ==
尽管任雅相官拜宰相，他的家世根据出土的其父墓志，父名任绪，京兆渭南人，隋朝时任西戎牙市监，唐武德二年（619年）正月十四日卒，年四十九。任雅相为其第二子，还有一子名雅顺。但他未拜相时的仕途都几乎无载。《[[旧唐书|旧唐书]]》《[[新唐书|新唐书]]》也没有他的传。他是[[渭南|渭南]]人。<ref name=ZZTJ200>[[:s:资治通鉴/卷200|《资治通鉴》卷二百]]</ref>

== 效命唐太宗 ==
[[唐太宗|唐太宗]][[贞观|贞观]]十三年（639年）二月，任雅相为节度副使朝散大夫，与节度大使右武卫大将军慕容宝册授突厥可汗[[阿史那薄布|阿史那薄布]]。<ref>[[:s:可汗山铭|谢偃《可汗山铭》]]</ref>

== 效命唐高宗 ==
唐高宗[[永徽|永徽]]六年（655年）三月，高宗有意讨伐[[高句丽|高句丽]]，于屯营教舞，召[[李义府|李义府]]、任雅相、[[许敬宗|许敬宗]]、[[许圉师|许圉师]]、[[张延师|张延师]]、[[苏定方|苏定方]]、[[阿史那忠|阿史那忠]]、[[于阗|于阗]]王[[伏阇|伏阇]]、[[上官仪|上官仪]]等赴洛城门观乐，乐曲名《一戎大定乐》，并赐观乐者杂彩有差。<ref>[[:s:旧唐书/卷028|《旧唐书》卷二十八]]</ref>

[[显庆|显庆]]二年（657年）闰正月，时任[[燕然都护|燕然都护]]的任雅相被任为伊丽道行军总管右屯卫将军苏定方的副总管，随苏定方、[[程知节|程知节]]等讨伐[[西突厥|西突厥]]沙钵罗可汗[[阿史那賀魯|阿史那賀魯]]，参与作战，破阿史那贺鲁于金牙山，<ref>[[:s:新唐书/卷217上|《新唐书》卷二百一十七上]]</ref>十二月，苏定方命燕然副都护[[萧嗣业|萧嗣业]]、另一副总管[[婆闰|婆闰]]率胡人军队奔赴邪罗斯川追杀阿史那贺鲁，他本人与任雅相率新附众军为其后继，最终使阿史那贺鲁被萧嗣业所俘。<ref name=ZZTJ200/><ref>[[:s:旧唐书/卷083|《旧唐书》卷八十三]]</ref><ref>[[:s:旧唐书/卷194下|《旧唐书》卷一百九十四下]]</ref><ref>[[:s:旧唐书/卷195|《旧唐书》卷一百九十五]]</ref><ref>[[:s:新唐书/卷111|《新唐书》卷一百一十一]]</ref><ref>[[:s:新唐书/卷215下|《新唐书》卷二百一十五下]]</ref>

任雅相后为[[兵部尚书|兵部尚书]]，曾伐高句丽，无大功而返。<ref>[[:s:旧唐书/卷199上|《旧唐书》卷一百九十九上]]</ref>四年（659年）五月，任雅相被授[[參知政事|參知政事]]，成为[[宰相|宰相]]。<ref name=BT4>[[:s:旧唐书/卷004|《旧唐书》卷四]]</ref><ref name=NBT3>[[:s:新唐书/卷003|《新唐书》卷三]]</ref>七月，因涉嫌谋反被流放的[[太尉|太尉]]、国舅[[长孙无忌|长孙无忌]]进一步遭到指控，任雅相和其他宰相[[李勣|李勣]]、许敬宗、[[辛茂将|辛茂将]]、[[卢承庆|卢承庆]]奉命调查。其中许敬宗是[[武后|武后]]的党羽，也是调查的发起者。长孙无忌最终被迫自杀。<ref name=ZZTJ200/>

[[龙朔|龙朔]]元年（661年）四月，[[唐高宗|唐高宗]]派兵东征高句丽，任雅相以兵部尚书、[[同中书门下三品|同中书门下三品]]、乐安县公为[[浿江|浿江]]道行军总管，与平壤道行军总管苏定方、扶余道行军总管萧嗣业、辽东道行军总管[[契苾何力|契苾何力]]、镂方道行军总管右骁卫将军[[程名振|程名振]]、沃沮道行军总管左骁卫将军[[庞孝泰|庞孝泰]]一起率三十五军东征。二年（662年）二月甲戌，任雅相在军中病逝于司戎太常伯任上，<ref name=BT4/><ref name=NBT3/>赠荆州大都督，谥曰'''敬'''。<ref name=ZZTJ200/>

==后事==
任雅相为将，没有推荐过亲戚故吏从军，都是把名额给有关部门补授，他对人说：“官无大小，皆国家公器，岂可苟便其私！”于是军中赏罚公平，大家对他的公正很服气。<ref name=ZZTJ200/>征高句丽期间，表贤而有谋的[[萧晶|萧晶]]为记室。<ref>[[:s:新唐书/卷202|《新唐书》卷二百零一]]</ref>

[[唐德宗|唐德宗]]年间，任雅相得续图于[[凌烟阁|凌烟阁]]。<ref>[[:s:新唐书/卷191|《新唐书》卷一百九十一]]</ref>

== 子孙 ==
曾孙[[任鹏|任鹏]]后为[[陵州|陵州]]刺史。任鹏子[[任迪簡|任迪簡]]为[[易定節度使|易定節度使]]。任迪简之子[[任宪|任宪]]字亚司。
