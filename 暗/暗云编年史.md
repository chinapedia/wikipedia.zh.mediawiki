《'''暗雲編年史'''》（{{lang-ja|ダーククロニクル}}，[[平文式羅馬字|羅馬字]]：{{transl|ja|Dāku Kuronikuru}}，Dark Chronicle），在北美發售時名為“'''暗雲2'''”（{{lang|en|'''''Dark Cloud 2'''''}}），是一款[[PlayStation_2|PlayStation 2]]平臺[[動作角色扮演遊戲|動作角色扮演遊戲]]。遊戲由[[Level-5|Level-5]]和[[索尼電腦娛樂日本工作室|索尼電腦娛樂日本工作室]]開發，[[索尼電腦娛樂|索尼電腦娛樂]]于2002年11月在日本、2003年2月在北美、2003年9月在歐洲發行<ref>{{cite web |url = http://www.mobygames.com/game/ps2/dark-cloud-2/credits |title=Dark Cloud 2 (2002) PlayStation 2 credits |publisher=[[MobyGames|MobyGames]] |accessdate=May 31, 2012}}</ref><ref>{{cite web |url = http://level5ia.com/product/dark-cloud-2-us-february-2003/?page=1 |title=Dark Cloud 2 - February 2003 |publisher=[[Level-5|LEVEL-5 International America]] |accessdate=May 31, 2012}}</ref>。

遊戲和前作《[[暗雲|暗雲]]》使用了相似的[[動作角色扮演遊戲|動作角色扮演]]與[[城市建造遊戲|城市建造]]機制。玩家控制分別來自現在和未來的[[主角|主角]]尤利斯和莫妮卡，并阻止希望從過去毀滅現在的[[反派|反派]]格里芬皇帝。遊戲中設定架空現實發生在《暗雲》結局500年之後。
