《'''暗物質'''》（{{lang-en|''Dark Matter''}}）是一部[[加拿大|加拿大]]科幻電視劇，由{{link-en|喬瑟夫·馬洛茲|Joseph Mallozzi}}和{{link-en|保羅·穆利|Paul Mullie}}創作，改編自他們的同名漫畫《{{link-en|暗物質 (漫畫)|Dark Matter (comics)|暗物質}}》，Prodigy Pictures聯同[[SPACE_(電視頻道)|SPACE頻道]]開發本劇。<ref>{{Cite web |url = http://www.bellmedia.ca/pr/press/new-sci-fi-series-dark-matter-exerts-gravitational-pull-june-12-space/ |title = All-New Sci-Fi Series DARK MATTER Exerts Its Gravitational Pull, June 12 on Space |author = Andrea Allen |date = 2015-05-20 |publisher = Bell Media |language = en}}</ref>第一季共有13集，於2015年6月12日首播。<ref>{{Cite web |url = http://globalnews.ca/news/1724345/sci-fi-series-dark-matter-to-be-filmed-in-toronto/ |title = Sci fi series 'Dark Matter' to be filmed in Toronto |author = John R. Kennedy |date = 2014-12-12 |publisher = Global News |language = en}}</ref><ref>{{Cite web |url = http://www.thetvjunkies.com/dark-matter-premiere-dates-set/ |title = DARK MATTER PREMIERE DATES SET |author = Bridget Liszewski |date = 2015-05-20 |publisher = TheTVJunkies.com |language = en}}</ref>2015年9月，續訂第二季共13集，在2016年7月1日播放。<ref>{{Cite web |url = http://deadline.com/2015/09/killjoys-dark-matter-renewed-season-2-syfy-1201511362/ |title = 'Killjoys' & 'Dark Matter' Renewed For Season 2 By Syfy |author = Denise Petski |date = 2015-09-01 |publisher = Deadline Hollywood |language = en}}</ref><ref>{{Cite web |url = http://tvline.com/2016/04/29/killjoys-season-2-premiere-date-syfy-dark-matter/ |title = Killjoys, Dark Matter Get Season 2 Premiere Dates at Syfy |author = Ryan Schwartz |date = 2016-04-29 |publisher = TVLine |language = en}}</ref>本劇於2016年9月續訂第三季，於2017年6月9日播放。<ref>{{Cite web |url = http://deadline.com/2016/09/killjoys-dark-matter-renewed-third-season-syfy-1201812120/ |title = 'Killjoys' & 'Dark Matter' Renewed For Third Season By Syfy |author = Denise Petski |date = 2016-09-01 |publisher = Deadline Hollywood |language = en}}</ref>2017年9月1日，Syfy在第三季後取消本劇。<ref>{{cite web |url=http://deadline.com/2017/09/dark-matter-canceled-syfy-after-three-seasons-1202159804/ |title='Dark Matter' Canceled After Three Seasons At Syfy |work=[[Deadline_Hollywood|Deadline Hollywood]] |first=Denise |last=Petski |date=2017-09-01}}</ref>

== 劇情 ==
六個人自一艘太空船（Razr，刀鋒）的休眠艙中甦醒，卻沒有任何記憶。經過一連串事件後，他們發現自己是一群作奸犯科被通緝的傭兵及海盜，而且他們已經接了任務要替一間邪惡的星際公司剷除一個不合作的殖民地。他們試圖找回或逃避各自的過去、學習信任彼此、並在各星際勢力的包夾中努力生存。
