'''織田信包'''（{{bd|1543年|8月17日|1614年|8月22日}}）是[[日本戰國時代|日本戰國時代]]武將以及[[江戶時代|江戶時代]]的武將、[[大名|大名]]。丹波柏原藩初代藩主。[[織田氏|信包系織田氏]]初代。父親是[[織田信秀|織田信秀]]。[[織田信長|織田信長]]的弟弟。通稱'''三十郎'''。曾成為[[長野工藤氏|長野工藤氏]]養子，成為第17代當主。

== 生平 ==
在[[天文_(後奈良天皇)|天文]]12年（1543年）7月17日出生，家中四男（有異說）。父親信秀是[[尾張國|尾張國]]的[[戰國大名|戰國大名]]。[[永祿|永祿]]11年（1568年）2月，因為兄長信長的命令，成為支配北[[伊勢|伊勢]]的長野工藤家（長野家）的養子，並以伊勢國[[上野城|上野城]]為居城，不過後來因為信長的命令，解除養子關係，返回織田家。永祿12年（1569年）10月，伊勢[[大河內城|大河內城]]被攻下，[[北畠家|北畠家]]向信長臣服，此時被信長任命為伊勢[[安濃津城|安濃津城]]城主。

此後跟隨信長在各地轉戰。在北[[近江|近江]][[小谷城|小谷城]]的[[淺井長政|淺井長政]]被攻滅後，保護妹妹兼長政的正室[[織田市|市]]和他們的女兒[[淀殿|茶茶]]、[[常高院|初]]、[[崇源院|江]]。[[天正_(日本)|天正]]3年（1575年），參加鎮壓[[越前一向一揆|越前一向一揆]]。天正5年（1577年），參加進攻[[雜賀黨|雜賀黨]]（[[紀州征伐|紀州征伐]]）。以織田一族重鎮的身份，受到厚遇。後來被任命輔助信長的長男[[織田信忠|信忠]]。以原尾張[[守護|守護]][[斯波義銀|斯波義銀]]的女兒為長男[[織田信重|信重]]的正室，令織田家和[[斯波家|斯波家]]加深關係。

天正10年（1582年）6月，信長和信忠在[[本能寺之變|本能寺之變]]中被殺後，跟隨[[豐臣秀吉|豐臣秀吉]]，領有伊勢[[津城|津城]]15萬石，被稱為「'''津侍從'''」。天正11年（1583年），與姪兒[[織田信孝|織田信孝]]等人對立，並攻擊[[柴田勝家|柴田勝家]]和[[瀧川一益|瀧川一益]]。

天正18年（1590年），在[[小田原征伐|小田原征伐]]中因為替[[北條氏政|北條氏政]]、[[北條氏直|氏直]]父子求情，於是觸怒秀吉，在[[文祿|文祿]]3年（1594年）9月被改易。改易理由是在[[檢地|檢地]]後，雖然令石高增加，但是沒有增加稅額。改易後，剃髮並自號'''老犬齋'''，在[[京都|京都]]的[[慈雲院|慈雲院]]隱居。

此後被賜予近江國內2萬石，並成為秀吉的[[御伽眾|御伽眾]]。[[慶長|慶長]]3年（1598年）6月，被賜予[[丹波國|丹波國]][[冰上郡|冰上郡]]柏原（現今[[兵庫縣|兵庫縣]][[丹波市|丹波市]]柏原）3萬6千石。

慶長5年（1600年），在[[關原之戰|關原之戰]]中加入西軍，參與攻擊丹後田邊城等戰事，不過在戰後沒有被[[德川家康|德川家康]]問罪，所領受到保證。此後在[[大坂城|大坂城]]輔助甥孫[[豐臣秀賴|豐臣秀賴]]，慶長19年（1614年）7月17日，在[[大坂冬之陣|大坂冬之陣]]前於大坂城中吐血死去。享年72歲。有傳言指是被[[片桐且元|片桐且元]]毒殺，不過沒有定論。[[家督|家督]]由三男[[織田信則|信則]]繼承。

== 關於生母、兄弟長幼 ==
生母不明。不過因為負責照顧父親[[織田信秀|信秀]]的正室兼[[織田信長|信長]]的生母[[土田御前|土田御前]]，以及後來又負責保護土田御前的女兒[[織田市|市]]的3個女兒，因此很可能與信長和市同樣是土田御前的兒子。而且在信長生前，於織田一門中地位亦相當高，亦顯示出信包是信長的同母弟的可能性。

被認為是信秀的四男，不過加上比信包年長的[[織田信廣|信廣]]、信長、[[織田信行|信勝（信行）]]，以及在[[弘治_(後奈良天皇)|弘治]]元年（1555年）死去時有約15歲的[[織田秀孝|秀孝]]和比信長年長的[[織田信時|信時]]，則算是六男，實況不明。

== 在織田一門中的立場 ==
在[[天正_(日本)|天正]]9年（1581年）的{{tsl|ja|京都御馬揃え|京都御馬揃}}中，[[織田信忠|織田信忠]]率領80騎，[[織田信雄|織田信雄]]率領30騎，接著信包率領10騎（之後[[織田信孝|織田信孝]]亦率領10騎）。而且在『[[信長公記|信長公記]]』中，有織田一門名稱相連的情況下，信包一定是在信雄和信孝之間，因此被推測在一門中，是排行第3的地位。
