'''貴陽街'''是一條位於[[台北市|台北市]]的街道，路名源自[[貴州省|貴州省]][[貴陽市|貴陽市]]，行經[[中正區_(台北市)|中正區]]和[[萬華區|萬華區]]，以[[中華路_(台北市)|中華路]]分為兩段，二段為[[單行道|單行道]]。其中西園路以西的部分是[[艋舺|艋舺]]的起點，被稱作'''台北第一街'''，是台北市最古老的街道<ref>{{cite web|url=https://whdo.gov.taipei/News_Content.aspx?n=BA44C2AB122C71F4&sms=865D02859464D466&s=9C708D6CE5AA3292|title=番薯市街 |publisher=萬華區公所}}</ref>。過去分為大溪口街、歡慈市街、直興街、草店尾街、祖師廟橫街等<ref>{{cite web |title=1903 最近實測臺北全圖 |url=https://www.historygis.udd.gov.taipei/urban/map/ |website=臺北市歷史圖資展示系統}}</ref>，直到1947年改為今名。

== 沿革 ==
{{主條目|艋舺}}
[[淡水河|淡水河]]第一水門附近，因[[新店溪|新店溪]]、[[大漢溪|大漢溪]]匯流而擴大河面，過去被稱為'''大溪口'''。[[臺灣清治時期|清代]][[康熙|康熙]]年間[[漳州|漳]]、[[泉州|泉]]漢人在台北開墾時，在此與行[[獨木舟|獨木舟]]的[[平埔族|平埔族]]族人交易[[番薯|番薯]]，形成了俗稱番薯市的市集，以及老街'''番薯市街'''。後來大溪口邊平埔族語稱為「莽葛」的獨木舟群聚，附近便取讀音近似的「艋舺」為[[街庄社鄉|-{庄}-名]]。

[[乾隆|乾隆]]年間，[[台北天后宮|媽祖廟]]、土地公廟、[[清水巖|清水祖師廟]]出現，老街也跟著向東擴展。西園路至西昌街為直街仔或稱'''直興街'''，即[[台北天后宮|新興宮]]前直往東的街道。土地公廟至康定路口清水祖師廟一帶為'''草店尾街'''，因早年以草厝作為店舖。後泉州同安人在祖師廟後方，今日貴陽街、柳州街、永福街一帶形成八甲庄，以祖師廟前街、祖師廟後街與老街相交<ref name="臺北畫刊">{{cite book |title=《臺北畫刊》貴陽街 |date=2003-03 |publisher=臺北市政府新聞處 |url=http://blind.tpml.edu.tw/ct.asp?xItem=545510&ctNode=346&mp=10}}</ref>。1877年牧師[[馬偕|馬偕]]來到艋舺傳教，並在此建立艋舺教會<ref>{{cite web |title=547 草店尾街租屋談起 |url=http://www.laijohn.com/book6/547.htm |website=賴永祥長老史料庫-教會史話}}</ref>。

[[臺灣日治時期|日治時期]]因艋舺街道狹窄，空襲時容易引起大火，因此於1908年實施[[市區改正|市區改正]]，番薯市街拓寬取直，接通至第一高女校門口，並雅化改為'''歡慈市街'''{{r|臺北畫刊}}。
