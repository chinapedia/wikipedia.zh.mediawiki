'''貴族'''指的是[[奴隶制|奴隶制]]、[[封建制|封建制]]、[[君主制|君主制]]、[[政教合一|教会制]]、[[种姓制度|种姓制]]等国家和地区的一些特有的群体，通过血缘、姓氏等某种特有的制度来继承知识、权力、财富而形成的传统。通常贵族财富多于其他人。在現時，一些国家和地区依然有完整的贵族传统。

貴族同其他阶级的最明显区别是拥有世袭[[爵位|爵位]]或者拥有对世袭爵位的继承权，其特點是[[世襲|世襲]]以及擁有與爵位相對應的特權，受到國家政權和君主的保證。終身[[爵位|爵位]]的擁有者一般不被認為是貴族；其他勋位、荣誉称号的拥有者也不被认为是贵族。高级[[官僚|官僚]]往往可以凭借贡献获得赐爵，大多是终身爵位；军事将领可以凭借战功获得封爵或赐爵，或是世袭爵位，或是终身爵位。由貴族祖先衍生出來的家族，僅擁有貴族血統但無世袭爵位，則通常被看做是[[世族|世家]]或[[士族|士族]]；他們或許仍然具有崇高地位與影響力，卻因為沒有實質性的爵位和相應的特權，而成為貴族階層中的邊緣者。

==字源==

== 東方 ==
東方由於各個文明發端較早、文明發展呈連續性，各地保留有相當多、種類各異的貴族等級制度相關記載及歷史遺留。如歷史上日本的[[日本皇室|皇室]]、[[公家_(日本)|公家]]分謂；[[朝鮮|朝鮮]]的[[兩班|兩班]]制度；[[中國|中國]]的[[中国爵位|世爵]]制度；到今天還有遺留的[[印度|印度]][[種姓制度|種姓制度]]、日本泰國等君主制國家的皇室和爵位制度、[[東南亞|東南亞]]地區的各世襲[[蘇丹_(稱謂)|蘇丹]]、[[台灣原住民|台灣原住民]]各部落的[[頭目|頭目]]等。

中國自[[三代|三代]]施行[[諸侯|諸侯]][[封建|封建]]制度，和後世[[西歐|西歐]]在[[中世紀|中世紀]]時期大分裂的社會有共同點，貴族擁有爵位、領地，在自己的領地內作為君主，而同時又對高一等級的君主效忠服役。其代表性制度為[[周代|周代]]的[[五等爵|五等爵]]制，主要規定了在[[天子|天子]]（在周朝指整個古代中國的君主）之下有公、侯、伯、子、男五等爵位<ref>《[[禮記|禮記]]·王制》曰：「王者之制祿爵，[[公爵|公]][[侯爵|侯]][[伯爵|伯]][[子爵|子]][[男爵|男]]，凡五等。」</ref>，天子是天下共主、除了作為貴族擁有自己的直轄領地外，統轄所有的諸侯，而諸侯按照爵位等級治理領地、統轄下級附庸貴族<ref>「[[天子|天子]]之田方千里，公侯田方百里，伯七十里，子男五十里。不能五十里者，不合於天子，附於諸侯曰附庸。」</ref>。諸侯封建制度到了[[東周|東周]]開始逐漸乃至全面崩潰，如[[春秋時期|春秋時期]]的楚國，本為子爵，因不滿其國大爵小，要求周王升其爵位未竟，於西元前704年稱王；其後其他六個主要諸侯國也紛紛稱王，周天子僅為名義上的“天子”。自[[秦|秦]]代中國開始實行完全的[[郡縣制|郡縣制]]之後，後世朝代皆以郡縣制為主；然而歷朝為安撫[[革命|改朝換代]]之功臣、嘉獎有特殊貢獻之文武官員，仍時有分封之舉，然而此後封爵貴族對於領地事務的控制已大大不如[[三代|三代]]時期的自主性、絕對性。

日本的貴族封建制度亦久。如[[公家_(日本)|公家]]（公卿）即中央之貴族；[[大名_(称谓)|大名]]即地方之諸侯，有權分封土地及享有資源；下屬的[[日本武士|武士]]，等同於西方中世紀貴族所自擁的武裝軍隊（[[騎士|騎士]]）。自1868年[[明治維新|明治維新]]之後，日本效法[[英国授勋及嘉奖制度|英制貴族系統]]下達[[華族令|華族令]]，將原本的貴族階層重新定義為[[華族_(日本)|華族]]，使原來的貴族能在[[君主立憲|君主立憲]]下的上議會保有一席之地。此一貴族制在[[日本投降|日本投降]]後的1947年取消。

[[東南亞|東南亞]]地區的[[蘇丹_(稱謂)|蘇丹]]為當地的世襲統治者，目前已多無實權，成為[[虛位元首|虛位元首]]，例如[[馬來西亞最高元首|馬來西亞最高元首]]即是從當地各世襲蘇丹中選出的。

[[台灣原住民|台灣原住民]]各族普遍以部落為基礎社會單位，有些族群會設有頭目一職為部落統一之政治領袖。頭目職位有些為以有才幹者任之（如泰雅族），有些則是世襲，為貴族階層。目前具有較完備貴族制度者有[[排灣族|排灣族]]和[[魯凱族|魯凱族]]等，然而隨著時代變遷，部落貴族的政經實力已漸不如往昔。

== 西方 ==
=== 早期 ===
[[File:Italian_Nobleman_of_the_Fifteenth_Century_From_a_Playing_card_engraved_on_Copper_about_1460_Cabinet_des_Estampes_National_Library_of_Paris_edit2.png|right]]
[[古羅馬|古羅馬]]時期，有爵位的貴族是享有特权的公民。他们和[[平民階級|平民階級]]相對立、形成特權階級的公民家庭集團。在公元前400年之前，封爵贵族曾垄断一切官职和圣职；[[罗马皇帝|罗马皇帝]]必须是封爵贵族；贵族免服[[兵役|兵役]]。4世纪以后的羅馬，封爵贵族的頭銜逐渐演变成非世袭的荣誉头衔。

=== 中世紀以後 ===
中世紀以後的歐洲由於沒有形成統一的帝國，君主國林立，受到歷史上的[[羅馬帝國|羅馬帝國]]等大國的影響，其贵族制度各國皆有不同。依照英文資料，歐洲中世紀貴族頭銜由上至下可概分如下：（男性／女性）
* Emperor / Empress —— 相當於古代中國的“[[天子|天子]]”、“[[皇帝|皇帝]]”、“[[可汗|可汗]]”等稱號；現代漢語通常翻譯為“皇帝”/“女皇（帝）”。其對應的統治區域稱為Empire。只有Empire的君主才有權擁有Emperor的頭銜。
* King / Queen —— 相當于古代中國的“[[王|王]]”級別，有些是獨立的，有些則從屬與某一帝國；現代漢語通常翻譯為“國王”/“女（國）王”。
* Prince / Princess —— 根據其來源，分別相當於古代中國的宗室王爵（如：皇子、[[公主|公主]]、[[親王|親王]]等）；或者作為某些獨立小國君主的稱號，相當於古代中國賜封的海外國王爵位；亦或者指伯爵或伯爵級別以上的眾封爵貴族（伯爵、侯爵、公爵皆可稱Prince）。亦因其來源、含義各有不同，現代翻譯可能有多個對應稱呼，如：[[王子|王子]]/[[公主|公主]]，[[親王|親王]]/王妃，[[大公|大公]]，眾諸侯（作複數princes）等。
* Grand Duke / Grand Duchess —— 現代通常翻譯為“[[大公|大公]]”/“女大公”。
* Archduke / Archduchess —— 現代通常翻譯為“[[大公|大公]]”/“女大公”。
* Duke / Duchess —— 現代譯名見下文。
* Marquess / Marchioness —— 現代譯名見下文。
* Margrave / Margravine —— 現代譯名見下文。
* Count, Earl / Countess —— 現代譯名見下文。
* Viscount / Viscountess —— 現代譯名見下文。
* Baron / Baroness —— 現代譯名見下文。

英國貴族系統概分為五等，翻譯成漢字時，因參考中國古代源遠流長的[[五等爵|五等爵]]制，按照《禮記》之'''公'''、'''侯'''、'''伯'''、'''子'''、'''男'''五等封爵的名號，專為其特製以下譯名：
* Duke——譯作“公爵”，其等級相當於中國五等爵中的'''公'''（“公爵”作為頭銜使用時，須與[[中國爵位|中國爵位]]名稱的“公”相區別：中國的爵位名稱是一個單字詞彙“公”，可以用於頭銜，並且其用于頭銜時，不可以和“爵”字連用，而僅以“封號（+諡號）+爵位名稱”相稱，如：魏文侯，誠意伯等；而“公爵”作為一個特定的翻譯詞彙，是一個雙字詞，可以按照英文的習慣直接用於人名、地名）。
* Marquess——譯作“侯爵”
* Earl——譯作“伯爵”
* Viscount——譯作“子爵”
* Baron——譯作“男爵”

当代英國的[[英国授勋及嘉奖制度|勛位榮譽制度]]對於[[世襲貴族|世襲貴族]]及[[終身貴族|終身貴族]]作了區分。世袭贵族是传统意义上的贵族階層（nobility），其爵位頭銜世代相傳，相當於中國古代封爵的“世襲罔替”；终身贵族的爵位一般用於封賞进入上议院的非贵族成员、高级法院法官等，拥有者不属于贵族，並且不得世襲；更低级的爵位“Sir（現代翻譯作[[爵士|爵士]]）”則只是荣誉称号，拥有者也不是贵族。

== 贵族后裔 ==

当[[政权|政权]]被推翻之时，贵族和[[王族|王族]]所拥有的[[头衔|头衔]]、[[财富|财富]]、[[土地|土地]]会被剥夺。各国情况亦不尽相同。在实行共和制的欧洲国家中，贵族、王室后裔使用原来的头衔是常见的现象，虽然已失去法律效用。

[[辛亥革命|辛亥革命]]，建立[[中华民国|中华民国]]后，根据《[[清室優待條件|清室優待條件]]》，[[清朝皇族|皇族]]、[[回部王公|回部]]、[[蒙古王公|蒙古王公]]及其他贵族头衔、[[清朝爵位|爵位]]得以保留。[[北洋政府|北洋政府]]执政时期，回部、蒙古王公仍有袭封和晋封。他们与西南边疆地区的世袭[[土司|土司]]类似，对其[[领地|领地]]仍有管辖权。1920年代至1940年代，中国时局动荡，政权更迭频繁。清朝皇族和[[旗人|旗人]]的政治、经济优待在1920年代后丧失。边疆地区王公、土司对领地的统治，尚未被[[中华民国政府|中华民国政府]]顾及。

1949年[[中华人民共和国|中华人民共和国]]建国，取代中华民国政府在大陆执政。在[[中国共产党|中国共产党]]领导下，建立起中央集权型单一制国家，掌控[[地方政治|地方政治]]。随着[[土地改革运动|土地改革运动]]、[[民族区域自治|民族区域自治]]的推进，王公、土司的统治在1950年代彻底结束。此后，中国大陆由于强调[[阶级斗争|阶级斗争]]，包括清朝皇族在内的贵族及其后代对自身身份较为忌讳，大都较少提及。改革开放后，政治氛围宽松，贵族及其后代身份逐渐成为一件值得炫耀的事。其中亦不乏夸大或伪造身份者<ref>{{Cite web |url=http://news.ifeng.com/a/20140911/41937752_2.shtml |title=学者：正支后裔很低调 旁支就喜欢管自己叫爱新觉罗 |accessdate=2017-12-23 |author= |date=2014-09-11 |work= |publisher=凤凰网，来源：中国网|language=Zh-hans }}</ref>，而单纯伪造贵族后裔身份并不触及法律。

[[印度独立|印度独立]]后，[[土邦|土邦]]被并入联邦，王公贵族头衔得以保留。但在1970年代贵族头衔被废除。
