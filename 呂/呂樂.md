'''呂樂'''，<span style="font-size:smaller;">[[CPM|CPM]]</span>（{{bd|1920年|5月16日|2010年|5月13日|catIdx=Lu呂樂}}），人稱'''樂哥'''、'''阿叔'''，[[廉政公署|廉政公署]]稱之為'''呂老虎'''，生于[[廣東省|廣東省]][[海豐縣|海豐縣]]，香港警務處刑事偵緝處前[[總探長|總探長]]，在任時期建立了一套與[[黑社會|黑社會]]制度相關的[[貪污|貪污]]機制，終淪為[[香港|香港]][[通緝犯|通緝犯]]。呂樂和[[藍剛|藍剛]]、[[韓森_(香港)|韓森]]及[[顏雄|顏雄]]4名被通緝的前探長，並稱「[[四大探長|四大探長]]」。

== 生平 ==
呂樂原名呂慕樂<ref>{{cite news|author=|title=祖母逝世後呂家所登訃文|publisher=華僑日報|date=1959-07-29|language=zh-hant}}</ref>，幼時於[[廣東省|廣東省]][[潮循道|潮循道]][[海豐縣|海豐縣]][[附城鎮_(海豐縣)|附城鎮]]鄉間為[[漁民|漁民]]，來到[[香港|香港]]後於[[長洲_(香港)|長洲]]長大。[[當差|當差]]前曾經以[[擦鞋工|擦鞋]]、派報及拉[[人力車|人力車]]等[[工作|工作]]為生。

1940年11月9日，呂樂加入了[[香港警隊|香港警隊]]。初時為[[軍裝巡邏小隊|軍裝巡邏小隊]]的[[警員|警員]]，不久後轉任為[[探員|探員]]。於[[雜差房|雜差房]]駐守時，獲得資深[[探長|探長]]陳立的賞識和提攜，於1951年由探員升任為[[探目|探目]]，同時調任於[[深水埗警署|深水埗警署]]，於1956年升任為[[高級探目|高級探目]]。<ref>{{cite journal|title=追踪呂樂神秘資產|publisher=《壹週刊》|date=2010年5月27日|language=zh-hant}}</ref>擔任期間，最為人稱道的是於1955年大破[[鑽石山|鑽石山]][[14K|14K]]群英大會。當時有百餘名[[14K|14K]]黑社會份子在鑽石山一所[[學校|學校]]內大排筵席，呂樂不動聲色，帶隊一網打盡。自此呂樂以掃蕩[[黑社會|黑社會]]而聞名，亦成為他晉升的功績及基石。<ref>工商日報 1976年11月6日</ref>

1956年，香港發生[[右派|右派]][[雙十暴動|雙十暴動]]。由於呂樂對[[14K|14K]]及[[新義安|新義安]]等黑社會的[[组织_(社会学)|組織]]及人士十分熟悉，故此其技大派用場，受到高層關注及重用。由於呂樂表現出色，同年獲晉升至[[新界區|新界區]][[探長|探長]]，駐守[[荃灣警署|荃灣警署]]。<ref>《香港6000年》第935頁</ref>此後，荃灣警署的[[雜差房|雜差房]]被用作為[[新界區|新界區]]的刑事偵緝處辦事處。

1958年，調至[[油麻地警署|油麻地警署]]接任退休的[[劉福_(探長)|劉福]]。1962年，警隊重置[[總探長|總探長]]一職，從10個[[環頭|環頭]]的10名探長中，選出兩位人選。亦有傳聞指出總探長的重置，實際上是為了呂樂和藍剛而專門設的。同年4月1日，呂樂與深水埗警署探長[[藍剛|藍剛]]一同獲晉升為總探長，前者駐守於香港區，後者駐守於九龍區<ref>工商日報 1962年4月20日</ref>兼及新界區<ref>大公報 1962年4月20日</ref>，同時獲[[英女皇|英女皇]][[伊莉莎白二世|伊莉莎白二世]]頒予[[殖民地警察獎章|殖民地警察獎章]]。

1967年，警隊為了壓制猖獗的[[貪污|貪污]]活動，對調了呂樂與藍剛的轄區，以避免他們「在地生根」，發展成為盤根錯節的關係。翌年，呂樂提前[[退休|退休]]。於1973年偕同[[妻子|妻子]]與8名子女[[移民|移民]][[加拿大|加拿大]]。

1974年[[廉政公署_(香港)|廉政公署]]成立後，因呂樂資產高達五億[[港元|港元]]，[[財富官職不相稱|財富官職不相稱]]，隨即成為[[通緝|追緝]]的最大目標。1977年香港法院正式向已经移居台湾的呂樂追讨部分财产，合计港幣九百九十三萬八千二百二元<ref>{{cite document|title=廉政公署年报|url=https://www.icac.org.hk/filemanager/tc/content_27/1977.pdf|publisher=[[香港廉政公署|香港廉政公署]]|date=1977|language=zh-hant}}</ref>。

1979年，呂樂於[[臺灣|臺灣]][[臺北市|臺北市]][[台北東區|東區]][[敦化南路|敦化南路]]購買了一所公寓豪宅，自此於[[臺北|臺北]]長居30多年，生活低調樸實且親切有禮，鄰居多半不知道他的過去與來歷，在先前，他居住過[[北投區|北投區]][[關渡|關渡]][[聖景路|聖景路]]上之別墅。他的八名子女中，女兒取得中華民國國籍，擔任臺灣公務員。七名兒子都在加拿大發展，呂樂期間亦有往返[[加拿大|加拿大]]。

2010年5月13日，患[[胃癌|胃癌]]末期的呂樂於[[加拿大|加拿大]]病逝，終年89歲。即使[[廉政公署_(香港)|廉政公署]]早在1974年已成立並通緝呂樂，但始終無法將他引渡回港。後葬於[[加拿大|加拿大]][[溫哥華|溫哥華]]。<ref>[http://hk.apple.nextmedia.com/news/art/20100521/14052905 廉署通緝34年 五億探長呂樂去世]</ref>

== 個人榮譽 ==
* 於1958年10月30日，獲頒授警察勳章。

== 軼事 ==
[[File:Lui_Kwan_Pok_Lutheran_Day_Nursery,_Cheung_Chau_(Hong_Kong).jpg|thumb]]教堂路的路德會呂君博幼兒園]]
呂樂相信自己是[[道教|道教]][[全真派|全真派]][[祖師|祖師]][[呂祖|呂祖]]之後，其父[[呂君博|呂君博]]，又名[[呂古|呂古]]，人稱「五叔」，是一位讀書人，其母為[[顏譚妹|顏譚妹]]。於長洲[[教堂路|教堂路]]7號的路德會呂君博幼稚園乃以其父為名的幼兒學院，為呂樂捐建的。

相傳呂樂從警時原為[[文盲|文盲]]，所以簽名皆為兩個圓圈（「呂」），一個向下的箭頭（「落」，諧音「樂」），而後才苦學而[[識字|識字]]。呂樂對此傳言很生氣，還說自己在[[香港日治時期|香港日治時期]]辦過《香港人報》，但有關此報紙的資料甚少。

呂樂有一位有[[黑社會|黑社會]]背景的[[親戚|親戚]]──[[姑丈|姑丈]]，[[新義安|新義安]]龍頭坐館[[向華炎|向華炎]]，向華炎是呂樂叔祖父[[呂六|呂六]]之[[女婿|女婿]]，呂樂的[[姑母|姑姑]]為[[呂杏華|呂杏華]]。

呂樂一家共有7子1女，7子全部居於[[加拿大|加拿大]]，其女為[[中華民國|中華民國]]國籍，現任[[公務員|公務員]]。

[[污點證人|污點證人]][[韓德_(香港)|韓德]]指出，呂樂擁有至少5億[[港元|港元]][[身家|身家]]，因此，電影有「五億探長」之名。污點證人[[吳錫豪|吳錫豪]]指出，當年他每1至2個月，需要向呂樂進貢30,000到40,000港元。

呂樂不單對外威風八面，於[[香港警隊|香港警隊]]裡亦十分風光，當[[警務處處長|警務處處長]]巡訪[[警署|警署]]時，他於列隊中站在頭位。當[[警務處處長|警務處處長]]與呂樂談過警務後，他便不會再理睬其他人，包括其他外籍[[高級警官|高級警官]]，鋒頭一時無兩。

呂樂與[[香港足球圈|香港足球圈]]人士相熟，曾經長期包下[[灣仔|灣仔]][[白宮酒店|白宮酒店]]517號房，在房內竹戰及會見各方人馬，並以該酒店房間號碼為名，於1950年代創辦[[五一七|五一七足球會]]。<ref>{{cite book|author=韋基舜 著|title=「吾土吾情：白宮酒店517」}}</ref>

呂樂為掩飾來歷不明的[[財富|財富]]，於1959年至1968年間，以父母名義，先後在[[尖沙咀|尖沙咀]]、[[筲箕灣|筲箕灣]]、[[觀塘|觀塘]]、[[沙田|沙田]]、[[港島|港島]][[半山|半山]]及[[灣仔|灣仔]]等，購入當時合共價值約300多萬港元的[[物業|物業]]。其中於1967年以父母、太太及其中兩名仍在求學的兒子──[[呂誠|呂誠]]和[[呂實|呂實]]的名義，成立[[渭濱置業有限公司|渭濱置業有限公司]]，並以上述公司之名購買位於沙田的[[博雅山莊|博雅山莊]]。除[[地產|地產]]外，呂樂離開香港後，仍然有[[遙控|-{zh-hans:遥控; zh-hant:遙距控制;}-]]經營。於1970年代開了15間，於加拿大也有分店<ref>「追踪呂樂神秘資產」《壹週刊》2010年5月27日]</ref>。

呂樂於沙田[[大圍|大圍]]的物業[[博雅山莊|博雅山莊]]已被[[政府|政府]]沒收，及後大部份的[[建築物|建築物]]均因為[[城門隧道|城門隧道]]工作而被拆卸，現時只餘下門樓及部份圍牆。
