'''段彦谟'''（{{bd|?||882年|7月17日|catIdx=D段}}<ref name=ZZTJ255>[[:s:資治通鑑/卷255|《资治通鉴》卷二百五十五]]</ref>），晚唐军阀，880年 - 882年与[[宦官|宦官]][[朱敬玫|朱敬玫]]联手控制[[荆南|荆南]]，但后来与朱闹翻并被朱所杀。

== 家世 ==
段彦谟家世不详，《[[旧唐书|旧唐书]]》《[[新唐书|新唐书]]》均无他的传。[[唐僖宗|唐僖宗]]初年，他作为[[泰宁|泰宁]]都将参与对[[黄巢|黄巢]]的农民大军作战。[[乾符|乾符]]六年（879年）十一月，朝廷命他取代江西招讨使[[曹全晸|曹全晸]]。当时，曹全晸刚和[[山南东道节度使|山南东道节度使]][[刘巨容|刘巨容]]大败黄巢，正在追击。朝廷让段彦谟取代曹全晸使曹停止了追击，<ref>《旧唐书·僖宗纪》作广明元年三月段彦谟受任为江西节度使。</ref>黄巢及其主将[[尚讓|尚讓]]得以逃脱。<ref name=ZZTJ253>[[:s:资治通鉴/卷253|《资治通鉴》卷二百五十三]]</ref>

== 控制荆南 ==
刘巨容击败黄巢后回师山南东道，而前宰相诸道行营[[都统|都统]]兼[[荆南节度使|荆南节度使]][[王铎_(唐朝)|王铎]]早在曹、刘击败黄巢前都逃跑了，荆南节度使空缺，无人管领。荆南监军宦官[[楊復光|楊復光]]让[[忠武军|忠武军]]都将[[宋浩|宋浩]]在朝廷任命前权知荆南府事，段彦谟以兵守城，随后朝廷下诏任宋浩为荆南安抚使。段彦谟耻于居宋浩之下，因此怨恨。宋浩不敬杨复光，杨复光唆使段彦谟杀宋浩。[[广明|广明]]元年（880年）四月，宋浩下令士兵不许砍伐军部[[江陵|江陵]]街上的树木，段彦谟的一些士兵却违反了这一规定，宋浩鞭打了他们的背，段、宋矛盾终于激化。段彦谟认为宋浩是在侮辱自己，引壮士杀入宋浩大营，杀宋浩及其二子。杨复光又上表称宋浩太残酷，所以被杀。杨复光、段彦谟以杨复光宾客[[常滋|常滋]]为节度留后。<ref>[[:s:新唐书/卷9|《新唐书·僖宗纪》]]作正月事。</ref>因杨复光所荐，僖宗敕段彦谟为[[朗州|朗州]][[刺史|刺史]]，改以[[工部侍郎|工部侍郎]][[鄭紹業|鄭紹業]]为新任节度使。<ref name=ZZTJ253/>段彦谟显然无意去朗州赴任，而郑绍业害怕控制荆南的段彦谟，拖延半年之久才赴任。十二月，黄巢攻占京城[[长安|长安]]，僖宗被迫逃往[[成都|成都]]，<ref>[[:s:资治通鉴/卷254|《资治通鉴》卷二百五十四]]</ref>召回郑绍业，从杨复光所请，以段彦谟代为荆南节度使。<ref name=ZZTJ255/><ref name=NBT186>[[:s:新唐书/卷186|《新唐书》卷一百八十六]]</ref>段彦谟假装巡边，拜见杨复光，以黄金数百两为谢。<ref>[[:s:新唐書/卷207|《新唐书》卷二百零七]]</ref>

== 身亡 ==
段彦谟和新任监军宦官朱敬玫很快发展成敌对关系。朱敬玫招募三千精兵号为[[忠勇军|忠勇军]]，亲自统辖。段彦谟生气了，决定杀掉朱。但中和二年（882年）六月，朱敬玫先发制人，攻入段彦谟府中，段彦谟正在睡觉，拔剑缒城而出逃奔亲军营垒，不被接纳，说“尔等负我！”于是被杀，亲属僚佐一并遇害，朱敬玫以江陵少尹[[李燧|李燧]]为[[留后|留后]]，诬陷段彦谟有罪。僖宗再次任郑绍业为荆南节度使，但郑犹豫着没去荆南赴任，朱敬玫让押牙[[陈儒_(唐朝)|陈儒]]领荆南事。<ref name=ZZTJ255/><ref name=NBT186/>
