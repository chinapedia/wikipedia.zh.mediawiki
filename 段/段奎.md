'''段奎'''('''Đoàn Khuê''') ({{bd|1923年|10月29日|1999年|1月16日|}}), 別名(Võ Tiến Trình) ，越南人民军大将，[[越南共产党中央政治局|越南共产党中央政治局]]委员、中央軍事委員会副書記，1991年至1997年任[[国防部长|国防部长]]。<ref>Đại tướng Đoàn Khuê Vịên lịch sử quân ṣư Vịêt Nam - 2002 Reminiscences of General Đoàn Khuê, 1923-1999; Includes biographical sketch; collection of articles.</ref>

== 生平 ==
段奎1923年10月29日出生于越南[[廣治省|廣治省]][[肇豐縣|肇豐縣]], 1945年加入印度支那共产党（今[[越南共产党|越南共产党]]），6月任廣治省臨時政治委員会軍事委員，同年9月加入卫国团（后为[[越南人民军|越南人民军]]）。1946年6月1日被任命为[[廣治陸軍中学|廣治陸軍中学]]政治委員<ref>井川（2005年）、17ページ</ref>。同年12月[[第一次印度支那战争|印度支那战争]]爆发，他历任步兵营教导员，团政委，步兵325师副政委等职。

1955年段奎任炮兵第351师政委，1958年任步兵师政委，同年授予上校军衔。1960年升第4軍区副政治委員、第5軍区司令，1960年和1961年曾两次到中国军事学院学习军事理论。[[越南战争|越南战争]]时期在南部地区指挥作战。1974年晋升为少将。

1976年10月任第5军区政委，同年12月在第4届越南劳动党大会（同大会改名共産党）上当选为中央委員<ref>[http://123.30.190.43:8080/tiengviet/tulieuvankien/details.asp?topic=105&subtopic=211&leader_topic=505&id=BT26120660704 第4期党中央執行委員会（1976-1982年）] {{webarchive|url=https://web.archive.org/web/20140228210333/http://123.30.190.43:8080/tiengviet/tulieuvankien/details.asp?topic=105&subtopic=211&leader_topic=505&id=BT26120660704 |date=2014-02-28 }} {{Vi icon}}</ref>。1980年晋升[[中将|中将]]。

1977年任第5军区司令兼政委，1978年底指挥第5军区部队参加了[[柬越战争|侵柬作战]]。1982年3月、第5届越南共産党大会再次当选中央委員<ref>[http://123.30.190.43:8080/tiengviet/tulieuvankien/details.asp?topic=105&subtopic=211&leader_topic=505&id=BT20120660994 第5期党中央執行委員会（1982-1986年）] {{webarchive|url=https://web.archive.org/web/20140113145802/http://123.30.190.43:8080/tiengviet/tulieuvankien/details.asp?topic=105&subtopic=211&leader_topic=505&id=BT20120660994 |date=2014-01-13 }} {{Vi icon}}</ref>。1984年12月21日、晋升[[上将|上将]]<ref>[[:s:vi:Nghị_quyết_số_597_NQ/HĐNN7|Nghị quyết số 597 NQ/HĐNN7]] {{Vi icon}}</ref>。1986年12月接替[[黎德英|黎德英]]任侵柬越军（478兵团）司令。

1986年12月、第6届越南共産党大会当选为政治局委员、党内序列第12位<ref>[http://123.30.190.43:8080/tiengviet/tulieuvankien/details.asp?topic=105&subtopic=211&leader_topic=505&id=BT20120661010 第6期党中央執行委員会（1986-1991年）] {{webarchive|url=https://web.archive.org/web/20140113144231/http://123.30.190.43:8080/tiengviet/tulieuvankien/details.asp?topic=105&subtopic=211&leader_topic=505&id=BT20120661010 |date=2014-01-13 }} {{Vi icon}}</ref>。1987年2月16日担任[[越南人民军|越南人民军]]总参謀長兼国防部副部长、中央军委委员<ref>[[:s:vi:Nghị_quyết_số_782B_NQ/HĐNN7|Nghị quyết số 782B NQ/HĐNN7]] {{Vi icon}}</ref>。1990年1月10日晋升为大将<ref>{{cite web |url=http://d-arch.ide.go.jp/browse/html/1990/201/1990201DIA.html |title=存档副本 |accessdate=2014-02-23 |deadurl=yes |archiveurl=https://web.archive.org/web/20130121082445/http://d-arch.ide.go.jp/browse/html/1990/201/1990201DIA.html |archivedate=2013-01-21 }}</ref>。

1991年6月、越共“7大”上再次选为政治局委員、序列第5位<ref>[http://123.30.190.43:8080/tiengviet/tulieuvankien/details.asp?topic=105&subtopic=211&leader_topic=505&id=BT20120661038 第7期党中央執行委員会（1991-1996年）]{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }} {{Vi icon}}</ref>。同年8月、担任[[武文杰|武文杰]]政府的国防部长，兼中央軍事委員会副書記。1996年6月越共“八大”再次当选为中央政治局委员、序列第6位<ref>[http://123.30.190.43:8080/tiengviet/tulieuvankien/details.asp?topic=105&subtopic=211&leader_topic=505&id=BT20120661062  第8期党中央執行委員会（1996-2001年）] {{webarchive|url=https://web.archive.org/web/20131012011058/http://123.30.190.43:8080/tiengviet/tulieuvankien/details.asp?topic=105&subtopic=211&leader_topic=505&id=BT20120661062 |date=2013-10-12 }} {{Vi icon}}</ref>。1997年9月卸任国防部部长，留任中央军事委員会副书记。

第7・8・9期国会議員。

1999年1月16日去世，年75。

== 家族 ==
段奎之母（Nguyễn Thị Dương, 1902年生)。其所生的子女有5人战死沙场，被授予[[越南英雄母亲|越南英雄母亲]]称号。

他的两个弟弟:

* Đoàn Chương, [[中将|中将]]，军事战略院院長
* Đoàn Thúy, 陆军上尉
