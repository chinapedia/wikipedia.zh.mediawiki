'''藥師經'''，全稱'''藥師琉璃光如來本願功德經'''（梵文：Bhaiṣajya-guru-pūrva-praṇidhāna-viśeṣa-vistara），收於《[[大正藏|大正藏]]》經集部，又名《十二神將饒益有情結願神咒經》。本經敍述[[藥師佛|藥師佛]]於過去世修[[菩薩行|菩薩行]]，為利導眾生發十二大願，著重於對眾生現世病苦災厄的救濟和獲得福樂<ref name="bha">{{cite web|title=藥師琉璃光如來本願功德經解題|url=http://www.gaya.org.tw/library/readers/guide-ysj.htm}}</ref>。

== 題解 ==
藥師琉璃光如來又稱藥師佛，是救濟世間疾苦的大醫王。「藥師」，佛教說佛陀為無上醫王或大藥師，「琉璃」，也寫作“瑠璃”，是梵語“{{zy|薜|bì}}琉璃”的略譯，是一種極希有的寶物<ref name="bha"/><ref name="bra2">{{cite web|author1=李玉况|title=敦煌藥師經變研究|url=http://enlight.lib.ntu.edu.tw/FULLTEXT/JR-MISC/mag12477.htm}}</ref>。

「本願功德」，願是願欲，[[本願|本願]]即菩薩所發弘願<ref name="bha"/>。

== 傳譯 ==
本經傳來中國，共有四次翻譯<ref name="bha"/><ref name="bra2"/>：

* 東晉‧[[帛尸梨蜜多羅|帛尸梨蜜多羅]]所譯《[[佛說灌頂拔除過罪生死得度經|佛說灌頂拔除過罪生死得度經]]》。
* 隋‧[[達磨笈多|達磨笈多]]所譯《藥師如來本願經》。
* 唐‧[[玄奘|玄奘]]所譯《藥師琉璃光如來本願功德經》，本經本無神咒以及[[八菩薩|八菩薩]]的各別名號<ref>{{cite web|author1=李玉况|title=敦煌藥師經變研究|url=http://enlight.lib.ntu.edu.tw/FULLTEXT/JR-MISC/mag12477.htm|quote=隋代敦煌流行的《藥師經》是帛尸梨蜜多羅和慧簡兩種譯本。'''《藥師經》諸漢文譯本中，唯有此二譯本詳述藥師八菩薩的名號'''，說明這八位菩薩在當時的藥師信仰中有著重要的地位。受到這兩本《藥師經》的影響，有些隋代藥師經畫藥師八菩薩。此外，《藥師經》諸漢文譯本中，也只有帛尸梨蜜多羅和慧簡二譯本論及禮敬藥師佛，可往生兜率天，面謁彌勒。所以隋代畫師多在藥師經變的對應或相關位置上作彌勒上生經變，以表示藥師信仰和彌勒信仰的密切關係。'''隋代以後，其他譯本日益流行。這些譯本不但未載藥師八菩薩的名號，也將禮敬藥師佛，可往生兜率天這段文字刪去'''，所以上述兩種隋代藥師經變的特色，在隋代以後的作品中遂很少發現。}}</ref><ref>{{cite web|title=藥師琉璃光如來本願功德經似乎有缺八大菩薩名號|url=http://forum.cbeta.org/index.php?topic=504.0;wap2}}</ref>；今之八菩薩名，乃後人依《[[佛說灌頂拔除過罪生死得度經|佛說灌頂拔除過罪生死得度經]]》添入成現今流通的譯本。此本流通最廣。
* 唐‧[[義淨|義淨]]所譯，經名《藥師琉璃光七佛如來本願功德經》，內容較前譯為廣。前者只說藥師[[淨土|淨土]]，義淨譯則有'''吉祥王如來、寶月智嚴光音自在王如來、金色寶光妙行成就如來、無憂最勝吉祥如來、法海雷音如來、法海勝慧遊戲神通如來、藥師琉璃光如來'''，七如來佛淨土。

西藏譯本有二種，一種相當於[[玄奘|玄奘]]所譯，另一譯相當於[[義淨|義淨]]譯本。一九三一年在[[吉爾吉特|吉爾吉特]]附近一個[[卒塔婆|卒塔婆]]中，考古學家發現了幾部梵本《藥師經》。

== 內容 ==
本經為佛應曼殊室利（[[文殊師利菩薩|文殊師利菩薩]]）的啟請，說明[[東方淨琉璃世界|東方淨琉璃世界]]藥師如來的功德，並詳述藥師如來因地所發的十二大願：生佛平等願、開曉事業願、無盡資生願、安立大道願、戒行清淨願、諸根具足願、身心康樂願、轉女成男願、回邪歸正願、從縛得脫願、得妙飲食願、得妙衣具願<ref name="bha"/>。

此十二大願是藥師如來在因地修行的廣大行願，而其果德，也就是受報國土[[東方淨琉璃世界|東方淨琉璃世界]]。藥師佛注重為眾生求得現世的安樂，以念佛、持咒、供養等善巧方便度化眾生，經中並宣說為現世眾生救病、救國難、救眾難，用以消災延壽的法門<ref name="bha"/>。

== 注解 ==
本經之註疏有《藥師經疏》（大正藏 No. 2766）、《藥師經疏》（大正藏 No. 2767）、 《藥師經古跡》二卷（新羅．太賢撰）、《藥師經燈燄》一卷（清．淨挺著）、《藥師經直解》一卷（清．靈耀撰）、《藥師經講記》（民國．太虛大師講，竺摩記）、《藥師經講記》（民國．印順導師講，妙峰、常覺記）等<ref name="bha"/><ref name="bra2"/>。並傳有眾多念頌、觀行儀軌等行法。

* 政本優生，使一切人民身心美善、
* 發揚慧力，使一切人民本力充實、
* 四攝六度，使一切人民萬事咸宜、
* 服務社會，使一切人民捨身救世、
* 精嚴戒律，使一切人民身心清淨、
* 重視醫衛，使一切人民疾苦悉得救治、
* 設醫施藥，使一切人民脫貧離病、

* 樹立正法，使一切正法育行、
