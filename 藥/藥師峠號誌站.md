'''藥師峠號誌站'''（{{jpn|j=薬師峠信号場|hg=やくしとうげしんごうじょう|rm=Yakushitōge shingōjō|f=y}}）位於[[新潟縣|新潟縣]][[十日町市|十日町市]]犬伏，是[[北越急行|北越急行]][[北北線|北北線]]的[[號誌站|號誌站]]。

== 車站構造 ==
* 位於[[藥師峠隧道|藥師峠隧道]]內，設置一主線與一待避線。

== 用途 ==
* 作為列車交會的場所，待避時會進行車內驗票的工作。

== 歷史 ==
* 1997年3月22日　北北線通車時設立。
