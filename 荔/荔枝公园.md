'''荔枝公园'''是地处中国[[广东省|广东省]][[深圳市|深圳市]][[福田区|福田区]]的一所市政公园，总面积为28公顷。于1982年在原地原有589株荔枝和低洼稻田基础上改建成园，主要景点为荔香阁、荔湖、揽月桥等。

==位置==
荔枝公园位于深圳市福田区红岭中路1001号，由东面的红岭路、南面的深南大道、西面的同心路、北面的红荔路夹构而成，紧邻[[深圳博物馆|深圳博物馆]]、[[深圳市少年儿童图书馆|深圳市少年儿童图书馆]]、[[深圳市青少年活动中心|深圳市青少年活动中心]]。
==历史==
荔枝公园得名源于1983年市规划局根据园内有589株荔枝树，并将它们作为公园的主景树，因而将原名新湖公园改名为荔枝公园<ref>{{cite web | url=http://design.yuanlin.com/HTML/Opus/2010-6/Yuanlin_Design_3213.HTML |title=深圳荔枝公园|accessdate=2015-06-05 |publisher=中国园林网}}</ref>。
<br>公园内种植有荔枝树，2014年6月份曾发生荔枝成熟时市民攀爬采摘事件，后被公园保安人员阻止<ref>{{cite web | url=http://news.163.com/14/0613/09/9UK2DMRP00014AEE.html |title=深圳荔枝公园成“果园” 荔枝遭游客乱采摘 |date=2014-06-13|accessdate=2015-06-05|work=|publisher=网易新闻}}</ref>。园方表示园内荔枝成熟后会进行统一采摘并送到福利院和养老院给孩子和老人们吃，且攀爬可能会导致危险意外发生，因此希望市民不要贸然采摘<ref>{{cite web | url=http://finance.qq.com/a/20140614/012390.htm |title=荔枝公园采摘荔枝送福利院 |date=2014-06-14|accessdate=2015-06-05|work=腾讯网|publisher=深圳新闻网}}</ref>。
