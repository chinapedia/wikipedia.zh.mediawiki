'''桂枝'''（拉丁文：Ramulus Cinnamomi、桂心）是[[中藥材|中藥材]]，是[[樟科|樟科]]常綠[[喬木|喬木]]桂樹（[[肉桂|肉桂]]，''Cinnamomum cassia''）的嫩枝。產於[[廣西|廣西]]、[[廣東|廣東]]、[[雲南|雲南]]、[[福建|福建]]等地，尤以[[廣西|廣西]]為多。春夏采收，晒干或阴干，切片生用。

== 性味歸經 ==
*味辛、甘
*性温
*入[[手太陰肺經|肺]]、[[手少阴心经|心]]、[[足太陽膀胱經|膀胱]]經

== 功效應用 ==
*發散[[風寒|風寒]]：'''桂枝'''能透達營衛，發散肌表風寒，用於[[感冒|感冒]]風寒表証，症見惡寒、[[發熱|發熱]]、[[頭痛|頭痛]]、身疼、脈浮等。但發汗作用比[[麻黃|麻黃]]較為緩和。若配[[麻黃|麻黃]]同用則增強發汗散寒之力，用於治外感風寒無汗的表實証，如[[麻黃|麻黃]]湯；配斂陰和營之[[白芍|白芍]]同用則能調和營衛，且發汗之力更緩，用於治外感風寒有汗的表虛証，如'''桂枝'''湯。

*溫經散寒：'''桂枝'''味辛辣而性溫行，有溫經散寒、袪風止痛的功效。用於風寒濕邪阻滯經絡所致肢體[[關節|關節]]疼痛，常以'''桂枝'''湯配[[防風|防風]]、[[威靈仙|威靈仙]]等同用。'''桂枝'''味兼甘，並略有強壯作用，在婦科和其他雜病應用很廣，如血虛寒凝引起的[[月經|月經]]不調、[[閉經|閉經]]、[[痛經|痛經]]，常與[[吳茱蓃|吳茱蓃]]、[[當歸|當歸]]、[[白芍|白芍]]、[[川芎|川芎]]等同用，如溫經湯。又用於血虛心悸、脈結代，常配[[炙甘草|炙甘草]]、[[黨參|黨參]]、[[阿胶|阿胶]]等同用，如復脈湯。

*通陽化氣：'''桂枝'''有溫中陽、化痰飲、化[[膀胱|膀胱]]之氣而行水之效，用於陽氣不舒、水濕停留，或為水腫，或為痰飲之証，如治痰飲之苓桂甘湯，治水腫之五苓散等。

== 用量用法 ==
煎服３～１０克；用於解表者，一般６～１０克；若用於[[風濕|風濕]][[關節|關節]]痹痛，有時須用較大劑量，約１２～１５克。

== 使用注意 ==
'''桂枝'''性較溫熱，溫熱病、陰虛陽盛証及血熱吐衄者均忌用；且其性兼溫通，孕婦及[[月經|月經]]過多者慎用。

== 其他資料 ==
*《[[神農本草經|神農本草經]]》謂'''桂枝'''「主上氣咳喘，結氣，喉痹吐吸，利[[關節|關節]]。
*《[[珍珠囊|珍珠囊]]》：“去伤风头痛，开腠理，解表发汗，去皮肤风湿。”
*《[[本经疏证|本经疏证]]》：“能利关节，温经通脉……其用之道有六：曰和营、曰通阳、曰利水、曰下气、曰行瘀、曰补中，为桂枝六大功效。其功最大，施之最广，无如桂枝汤，则和营其功也。”

*據化學分析，'''桂枝'''所含揮發油中的主要成分為桂皮醛、桂皮酸，并含有少量乙酸桂皮醛，乙酸苯丙酯。尚含粘液质、鞣质及树脂等。

*據藥理研究，'''桂枝'''所含揮發油，能抑制[[腸道|腸道]]多種[[細菌|細菌]]發育，如[[痢疾|痢疾]]杆菌、[[霍亂|霍亂]]弧菌、沙門氏菌等。對流感病毒亦有抑制作用，又能刺激[[汗腺|汗腺]]，使[[皮膚|皮膚]][[血管|血管]]擴張，故有發汗作用。其揮發油味辛辣，氣芬香，能促進消化液的分泌及增加胃腸蠕動，有助於胃腸積氣的排除，故有健胃袪風作用。此外，'''桂枝'''並能強心鎮痛。
