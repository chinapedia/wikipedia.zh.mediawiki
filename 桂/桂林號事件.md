'''桂林號事件'''（{{lang-en|'''Kweilin Incident'''}}）是發生於1938年8月24日、[[廣東|廣東]]上空的一場[[空難|空難]]。當日上午，一架[[中國航空|中國航空]]的[[道格拉斯DC-2|道格拉斯DC-2]]客機由[[香港|香港]][[啟德機場|啟德機場]]起飛，原定途經[[廣西|廣西]][[梧州|梧州]]及[[四川|四川]][[重慶|重慶]]，前往[[成都|成都]]，但客機在離開香港空域後，旋即遭到[[日本海軍|日本海軍]]戰機追擊。客機在廣東迫降河面後再遭到日本戰機多次掃射，最終沉沒水底，機上18人只有美籍機師、華裔無線電員及另一華裔乘客三人生還。這場空難也被視為民航史上首次有客機遭到軍機擊毀，並引起一定的國際關注。{{cfn|Crouch|y=2012|p=158}}

== 經過 ==

桂林號（機身編號32）是[[中國航空|中國航空]]屬下的[[道格拉斯DC-2|道格拉斯DC-2]]客機。該機負責來往[[香港|香港]]－[[廣西|廣西]][[梧州|梧州]]－[[四川|四川]][[重慶|重慶]]－[[成都|成都]]的定期航線，而航線則由[[泛美航空|泛美航空]]承包營運。<ref>{{cite web |url=http://www.cnac.org/pilots01.htm |title=CNAC pilots |website=CNAC.org |author= |date= |accessdate=August 28, 2014}}</ref>

1938年8月24日上午8時3分，桂林號由香港[[啟德機場|啟德機場]]起飛，當時機上共有14名乘客，以及美裔機師活士（Hugh Leslie Woods，又譯伍茲）、華裔副機師劉崇佺、無線電員羅昭明、及侍者武慶華4名機組人員。{{cfn|Crouch|y=2012|pp=155-156}}約半小時後，桂林號進入中國空域，機師活士發現8架日本[[水上飛機|水上飛機]]排出攻擊陣形而來，即把桂林號駛入雲層躲避，然而日本軍機從後追上，並開始射擊。活士將桂林號俯衝急降，並安全降落到一條河流之上。{{cfn|Crouch|y=2012|p=157}}

日機攻擊桂林號之時，無線電員羅昭明已即向外拍[[電報|電報]]，報稱「日機追擊」。後來桂林號迫降河面漂浮，機上乘客人員也沒有傷亡，故此羅昭明又電報「乘客均安」。這使到香港在24日的晚報俱指桂林號的人員安然無恙。不過，日本戰機很快隨後趕上，並多次以機槍掃射桂林號的機身，令到桂林號開始入水下沉。由於水流湍急，逃生困難，桂林號最終只有機師活士、無線電員羅昭明及男乘客樓兆念三人生還。{{cfn|Crouch|y=2012|p=157-158, 165}}

三名生還者對事件的經過有不同描述。活士稱桂林號在起飛後20分鐘於6,000呎高空進入中國空域，已經看到日本的8架水上飛機在前方巡邏。當時[[廣州戰役|廣州戰役]]雖然尚未爆發，但[[日本海軍|日本海軍]]的第五艦隊已經在華南海域部署，並且有一艘[[水上飛機母艦|水上飛機母艦]]派駐香港外海。活士將飛機折返香港[[新界|新界]]並爬升至8,000呎，等待日本巡哨飛走，在數分鐘後才再次進入中國空域，不料日軍飛機已經爬升至超過11,000呎，並從後追擊，阻止活士折返香港。{{cfn|Crouch|y=2012|p=156}}桂林號迫降後，活士又稱曾經進入機艙，並叫侍者武慶華打開機門，但機上乘客卻沒有離開。活士因此假定機上乘客不諧泳術，而返回駕駛室，最後泅水逃生。{{cfn|Crouch|y=2012|p=157}}乘客樓兆念及無線電員羅昭明，則指活士從未通知乘客離開，後來機門遭人打開後，致使機艙大量入水，眾人才嘗試泅水逃生，然而飛機很快便向下沉沒。

== 空難原因與影響 ==
[[File:Hu_Yun.jpg|thumb]]]]
日軍軍方從未公布截擊桂林號的原因，引發不少猜測。其一說法指日軍是打算行剌[[立法院院長|立法院院長]][[孫科|孫科]]。在香港發行的日文報章《[[香港日報|香港日報]]》曾經承認孫科是這次攻擊的目標，並指日軍飛機本來打算活捉孫科；而孫科原先的行程是在8月24日乘坐桂林號離開香港，但他最後卻乘搭了另一間航空公司（[[歐亞航空|歐亞航空]]）。孫科的女秘書後來聲稱自己錯誤安排行程，意外令到孫科避過一劫。亦有猜測認為孫科是刻意發布錯誤資訊，令到日軍截擊錯誤飛機。{{cfn|Crouch|y=2012|pp=164-165}}

[[File:Xu_Xinliu.jpg|thumb]]]]

桂林號事件對中國銀行業亦有一定衝擊。當時[[交通銀行|交通銀行]]董事長[[胡筠_(銀行家)|胡筠]]、[[浙江興業銀行|浙江興業銀行]]行長[[徐新六|徐新六]]、[[中華民國中央銀行|中華民國中央銀行]]機要科主任[[王宇楣|王宇楣]]三人俱在空難中喪生。<ref name=sina>胡卓然 (2014年07月18日). "[https://www.webcitation.org/6S9qPEOni?url=http://history.sina.com.cn/his/zl/2014-07-18/112995625.shtml 侵華日軍曾擊落中國民航客機]". 新浪歷史.</ref>

桂林號事件是民航史上首次有客機遭到軍機擊毀，在當時獲得廣泛報導。空難後中航及其他客運公司開始將航線改為晚間飛行，並且引進德國的無線電返航系統，協助制定航程。桂林號事件也加劇美國輿論對日本的敵意，但並未足以促使美國決定支援中國抗戰。{{cfn|Crouch|y=2012|pp=167-169}}

== 桂林號的下場 ==
[[File:Qian_Changgan.jpg|thumb]]]]
桂林號在空難後被中航打撈修理，然後更名為重慶號（機身編號改為39），繼續用作客機。{{cfn|Crouch|y=2012|p=218}}為免引起乘客不安，中航在宣傳廣告中並未提及重慶號的前身。{{cfn|Crouch|y=2012|pp=211-212}}1940年10月29日，重慶號在[[雲南|雲南]][[沾益縣|沾益縣]]降落後遭到日本飛機攻擊爆炸焚毀，機上12人有9人死亡，包括建築師[[錢昌淦|錢昌淦]]。{{cfn|Crouch|y=2012|pp=218-219}}錢昌淦參建了[[錢塘江大橋|錢塘江大橋]]，並在抗戰期間於[[滇緬公路|滇緬公路]]建造橋樑，身後民國政府將滇緬公路其一橋樑命名為「昌淦橋」，以茲紀念。<ref name=sina/>
