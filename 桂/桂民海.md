===瑞典介入===
2015年10月18日，桂敏海到内地投案，宁波市中级人民法院撤销缓刑，对其交通肇事罪收监执行有期徒刑2年<ref name=xinjb/>。2017年10月17日，桂敏海交通肇事罪刑满释放，但他涉及的涉嫌非法经营犯罪一案仍在侦查，尚未终结。根据《中华人民共和国出境入境管理法》第二十八条规定，被判处刑罚尚未执行完毕或者属于刑事案件被告人、犯罪嫌疑人的外国公民不准出境。10月16日他承诺“不擅自离开宁波，如要离开宁波，会及时向办案单位报告自己的去向。”<ref name=qizi/>12月12日桂敏海向宁波公安机关提出在宁波居留的申请，公安机关为其办理了居留许可<ref name=xinjb/>。在宁波居住期间，他租住住房，和三个姐姐陪伴年迈的母亲<ref name=zjdc>{{Cite web|url = http://news.sina.com.cn/c/nd/2018-02-10/doc-ifyrmfmc1313331.shtml|title = 瑞典籍华裔香港书商桂敏海再次被拘留真相调查|author = |date = 2018-02-10|publisher = 新浪|language = |accessdate = |quote = |archive-url = https://web.archive.org/web/20180211072938/http://news.sina.com.cn/c/nd/2018-02-10/doc-ifyrmfmc1313331.shtml|archive-date = 2018-02-11|dead-url = no}}</ref>。2017年11月26日、12月11日他两次到上海分别申请、领取新的身份证件，其妻钟宁君也曾自[[德国|德国]]回宁波与其共同生活将近一个月<ref name=qizi/>。

2017年11月10日，桂敏海家人帮他联系了[[三甲医院|三甲医院]][[宁波市医疗中心李惠利医院|宁波市医疗中心李惠利医院]]做全面检查，医生诊断他[[颈椎|颈椎]]第六、七节变形导致压迫神经受损引起肌肉萎缩。疾病诊断意见书称，桂敏海患有脊髓型[[颈椎病|颈椎病]]。在2018年1月桂敏海再度被刑事拘留后，上海、宁波5位骨科和神经科专家诊断了他手部肌肉萎缩的病情，认为是颈椎压迫神经引发肌肉萎缩，基本排除[[肌萎缩性脊髓侧索硬化症|肌萎缩性脊髓侧索硬化症]]（“渐冻症”）<ref name=xinjb/>。

自他2017年10月获释后，瑞典方面几乎每天都和他联系，不停劝说他回瑞典，并提供了多套方案<ref name=qizi/>。针对他手部肌肉萎缩的情况，瑞典方面称他得了“渐冻症”，并告诉他不要在中国就医，会尽快将他送回瑞典治疗。经多次拒绝后，桂敏海最终同意<ref name=zjdc/>。1月20日，瑞典驻上海领事馆两名外交官秘密将他接上领事馆挂有外交牌照的汽车带离宁波，从上海转乘高铁赴北京<ref name=xinjb/>。据[[法新社|法新社]]报道，其女安吉拉（Angela）称，其父2018年1月20日在两名瑞典外交官陪同下乘火车赴北京途中，在火车驶近北京时被约10名便衣警员登上车厢强行带走。她称其父此行赴北京是为“接受一名瑞典医生的检查”，治疗“渐冻症”。桂敏海被带走时，两名瑞典外交官都在场<ref>{{Cite web |url=https://cn.nytimes.com/china/20180123/china-police-bookseller-train-gui-minhai/ |title=瑞典籍香港书商桂民海在火车上被中国警方带走 - 纽约时报中文网 |access-date=2018-01-23 |archive-url=https://web.archive.org/web/20180123050459/https://cn.nytimes.com/china/20180123/china-police-bookseller-train-gui-minhai/ |archive-date=2018-01-23 |dead-url=no }}</ref><ref name=qizi/>。事后瑞典政府沉默数日。据[[英国|英国]]《[[卫报|卫报]]》2月6日报道，瑞典外交部长[[瑪戈特·瓦爾斯特倫|瓦尔斯特伦]]5日声明称，桂敏海持续被拘“事态严重”，并指责中方对桂敏海和两名瑞典外交官的行为是“粗暴干涉”，违反了领事协助的基本国际规则，“我们要求（中方）让我们的公民跟瑞典外交与医务人员见面，并且获释以便和他的女儿、家人团聚”。《卫报》认为“该声明意味着瑞典自2015年桂第一次失踪后对该案保持的审慎态度有明显升级”<ref name=qizi>{{Cite web|url = http://news.sina.com.cn/c/nd/2018-02-10/doc-ifyrmfmc1315473.shtml|title = 香港书商桂敏海讲述“失踪”:我成了瑞典人的棋子|author = |date = 2008-02-10|publisher = 新浪|language = |accessdate = |quote = |archive-url = https://web.archive.org/web/20180211074426/http://news.sina.com.cn/c/nd/2018-02-10/doc-ifyrmfmc1315473.shtml|archive-date = 2018-02-11|dead-url = no}}</ref>。

中国公安机关则称，1月20日上午8时前后，桂敏海离开宁波去[[上海|上海]][[上海虹橋站|虹桥火车站]]，随后乘中午11时10分G126次[[高铁|高铁]]赴北京，此行途中桂敏海携带多份涉及国家秘密情报资料。此前宁波公安机关已发现桂敏海有涉嫌从事为境外非法提供国家秘密、情报，危害国家安全的违法犯罪活动。在他赴北京途中，公安机关多次联系他本人，要求他返回宁波接受调查，“但现场陪同的外籍人员要求桂不配合”。因此当G126次高铁在[[济南西站|济南西站]]停留时，公安人员依法将他带离列车，以涉嫌为境外非法提供国家秘密、情报罪对他采取[[刑事拘留|刑事拘留]]强制措施。1月20日他被刑事拘留后，当天公安机关即通过中国外交部向瑞典驻华使馆通报了办案单位及联系电话，并向瑞典驻华大使馆一秘何某通报了被拘留原因。随后中国外交部门也向瑞典外交部门进行领事通报，通报了涉嫌罪名、被刑事拘留情况及办案单位。1月30日，宁波市公安局专门到上海向瑞典驻上海总领事通报了案情近况，并转交了桂敏海和其家人分别致瑞典驻华大使[[林戴安|林戴安]]的信，两封信均对瑞典方面做法表示了不满<ref name=qizi/>。

2018年2月6日，[[中華人民共和國外交部發言人|中國外交部發言人]][[耿爽|耿爽]]在例行記者會上回应称，“桂敏海雖然是瑞典公民，但他涉及的案件，必須依照中國法律處理，瑞典當局應該知道案件的嚴重性質，以及瑞方某些人在其中扮演不光彩的角色，絕不接受瑞方無視中方通報，一再發表不負責任的言論，強烈要求瑞方不要做損害雙方互信，以及影響兩國關係大局的事。”<ref>{{Cite web|url = https://tw.appledaily.com/new/realtime/20180206/1293248/|title = 中國外交部回應桂民海案：瑞典某些人「不光彩」|author = |date = 2018-02-06|publisher = 蘋果日報|language = |accessdate = |quote = |archive-url = https://web.archive.org/web/20180207021923/https://tw.appledaily.com/new/realtime/20180206/1293248/|archive-date = 2018-02-07|dead-url = no}}</ref><ref>{{Cite web|url = http://news.ltn.com.tw/news/world/breakingnews/2334679|title = 桂民海「被失蹤」 中國外交部證實：違反刑法被捕|author = |date = 2018-02-06|publisher = 自由時報電子報|language = |accessdate = |quote = |archive-url = https://web.archive.org/web/20180206175214/http://news.ltn.com.tw/news/world/breakingnews/2334679|archive-date = 2018-02-06|dead-url = no}}</ref><ref name=xinjb>{{Cite web|url = http://epaper.bjnews.com.cn/html/2018-02/11/content_711276.htm?div=-1|title = 瑞典籍华裔香港书商桂敏海再次被拘|author = |date = 2018-02-11|publisher = 新京报|language = |accessdate = |quote = |archive-url = https://web.archive.org/web/20180211131612/http://epaper.bjnews.com.cn/html/2018-02/11/content_711276.htm?div=-1|archive-date = 2018-02-11|dead-url = no}}</ref>

据中国公安机关通报，桂敏海因涉嫌为境外非法提供国家秘密、情报罪被羁押在[[宁波市|宁波市]]公安局看守所。2018年2月9日，桂敏海在宁波市公安局看守所露面并接受境内外媒体采访。他表示，此次采访是他本人主动向办案机关提出，以便澄清“事情经过和真相”。他称自己此前赴北京是为在瑞典驻华大使馆的安排下“以看病为名”，先到该使馆“面见一个瑞典的汉学家，还有一个[[美国|美国]]基金会的人员”，再“伺机出境回瑞典”。他说“到现在为止，没有医生诊断我得了渐冻症。”“瑞典人现在说我得了渐冻症，我觉得是夸张了事实，把这个事情作为借口，好尽早把我带到瑞典。”他说自己虽有瑞典国籍，但在近十年时间里生活在德国，并未获得过瑞典政府的特别关注，此次瑞典政府是在“炒作”、“利用”他，并最终导致他再度被拘留，“尤其是这一次，我不知道他们到底是关注我，还是在利用我这件事情。2018年是瑞典的大选年，我的事情对于瑞典某些政治人物来说，也许对他们在大选年中的一些政治活动会有加分。”“假如瑞典政府还要继续拿我的事情说事，我可能考虑放弃瑞典国籍。”<ref name=qizi/>

2018年2月2日，[[國際出版人協會|国际出版家协会]]表示将为其颁发2018年伏尔泰奖，以向“他的勇气致敬”。对此桂敏海2月9日称，自己“不想也不会接受这个奖，否则我又犯傻被别人利用了。”<ref name=qizi/>

2018年6月5日，37家瑞典報紙共同呼籲中國釋放出版商桂民海，重要的學術界人士、編輯、記者、政界人士和演員等多人也參與連署<ref>{{Cite web|url=https://newtalk.tw/news/view/2018-06-05/126867|title=銅鑼灣書商桂民海再傳「被失蹤」 瑞典37大報聲援|date=2018-06-05|publisher=新頭殼newtalk|accessdate=2018-12-28}}</ref><ref>{{Cite web|url=https://www.cna.com.tw/news/aopl/201806050357.aspx|title=桂民海遭陸羈押 37家瑞典日報呼籲釋放|date=2018-06-05|publisher=中央社|accessdate=2018-12-28}}</ref>。

家屬瑞典華使館抗議|date=2018-07-15|publisher=蘋果日報 (香港)|accessdate=2018-12-28}}</ref><ref>{{Cite web|url=http://m.trad.cn.rfi.fr/中國/20180713-被捕1000天-桂民海親朋要求中國當局將其釋放|title=被捕1000天 桂民海親朋要求中國當局將其釋放|date=2018-07-13|publisher=法國國際廣播電台|accessdate=2018-12-28}}</ref>。

2018年8月13日，瑞典外交部長[[瑪戈特·瓦爾斯特倫|瑪戈特·瓦爾斯特倫]]表示瑞典醫護人員獲准探訪桂民海，她強調瑞典正繼續進行「密集工作」，並將以「幾種不同的方式」進行醫療訪問。除此之外，華爾史特龍繼續要求北京當局釋放桂敏海，不過並未提及桂民海的病情或醫療訪問詳細資訊<ref>{{Cite web|url=https://m.nownews.com/news/2800951|title=瑞典獲准醫療探視桂民海 異議人士籲公布病況|date=2018-08-14|publisher=NOWNews|accessdate=2018-08-15|archive-url=https://web.archive.org/web/20180814135204/https://m.nownews.com/news/2800951|archive-date=2018-08-14|dead-url=yes}}</ref><ref>{{Cite web|url=https://news.rti.org.tw/news/view/id/420759|title=瑞典：中國准許瑞典醫生探視桂民海|date=2018-08-14|publisher=中央廣播電臺|accessdate=2018-08-15|archive-url=https://web.archive.org/web/20180814134955/https://news.rti.org.tw/news/view/id/420759|archive-date=2018-08-14|dead-url=no}}</ref>。
