'''搜狗云输入法'''是2009年11月<ref>[http://pinyin.sogou.com/cloud/log.html 搜狗云输入法更新历史]</ref> 由[[搜狐|搜狐]]（{{nasdaq2|SOHU}}）公司推出的一款基于[[JavaScript|JavaScript]]，利用[[Ajax|Ajax]]通信原理，[[B/S结构|B/S结构]]，通过[[云计算|云计算]]技术的概念性[[汉语拼音输入法|汉语拼音输入法]]产品。

== 运行平台 ==
搜狗云输入法可运行在多种[[操作系统|操作系统]]的[[浏览器|浏览器]]中。官方的搜狗云输入法只能在[[网页|网页]]上或通过[[搜狗拼音输入法|搜狗拼音输入法]]使用，通过在浏览器中运行一段[[JavaScript|JavaScript]]，就可以使用搜狗云输入法进行[[中文|中文]]输入；[[搜狗拼音输入法|搜狗拼音输入法]]中也整合了搜狗云输入法，在用户通过[[搜狗拼音输入法|搜狗拼音输入法]]输入文本时，它会自动通过网络得到运算结果，并作为第二候选词。为了方便用户的使用，官方也推出了搜狗云输入法的[[Firefox|Firefox]]插件与[[Chrome|Chrome]]插件。

到目前为止，搜狗云输入法未开放任何[[API|API]]。不过，已经出现了一些第三方的[[输入法|输入法]]实现，如[[跨平台|跨平台]]的[[小小输入法|小小输入法]]，以及[[Fcitx|Fcitx]]输入法的云模块，通过将用户输入的[[拼音|拼音]]发送给搜狗云输入法的[[服务器|服务器]]，以达到在[[桌面|桌面]]上使用搜狗云输入法的效果。

== 特点 ==
搜狗云输入法不依赖任何本地[[语言模型|语言模型]]和[[词库|词库]]，完全靠[[服务器|服务器]]提供运算支持。由于[[服务器|服务器]]采用的是共4GB[[三元模型|三元模型]]和[[trigger模型|trigger模型]]，200万词的词库，语言模型和词库均非常庞大，因此和一般[[拼音输入法|拼音输入法]]相比，准确率较高。[[搜狐|搜狐]]自称其“短句输入正确率为94%，长句输入正确率84%”。<ref>[http://pinyin.sogou.com/cloud/question.html 搜狗云输入法的帮助与介绍]</ref>。

== 评价 ==
搜狗云输入法精确率高，[[词汇|词汇]]量大，可以方便输入生僻词汇，提高了打字效率。具有[[跨平台|跨平台]]的特征，而且不受本地[[操作系统|操作系统]]的限制，方便了使用多种[[操作系统|操作系统]]或本地操作系统受到限制 （如[[网吧|网吧]]）的用户。<ref>[http://news.newhua.com/news/2009/1110/78760.shtml 感受搜狗云输入的威力]</ref>然而搜狗云输入法依赖[[网络|网络]]，而且只能在[[网页|网页]]上使用，在网络状况不好时将难以使用，输入效率不如一般的本地输入法。<ref>[http://linuxtoy.org/archives/sogou-cloud-input-method.html 搜狗云输入法！]</ref> 也不能在办公软件或[[即时通讯软件|即时通讯软件]]上使用，没有很大的应用价值。还有一部分人IT博客作者与开发者认为，搜狗云输入法根本不是真正的[[云计算|云计算]]，本质上只是[[Web|Web]][[应用程序|应用程序]]，是[[云计算|云计算]]的概念[[炒作|炒作]]。<ref>[http://mimi19730930.blog.163.com/blog/static/437984200910710260838/ 搜狗云输入法 又一个云忽悠？]</ref><ref>[http://home.donews.com/donews/article/1/137979.html 搜狗云本质——Web应用程序]</ref>
