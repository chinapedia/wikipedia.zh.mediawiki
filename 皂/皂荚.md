==形態特徵==
皂莢是一種落葉[[喬木|喬木]]或小喬木[[植物|植物]]，高約5－10米<ref name = "青海植物誌"/>，最高可達30米<ref name = "安徽植物誌"/><ref name = "北京植物誌"/><ref name = "浙江植物誌"/>，[[胸徑|胸徑]]120厘米<ref name = "安徽植物誌"/>，[[樹冠|樹冠]]扁球形<ref name = "上海植物園"/>。

[[樹皮|樹皮]]粗糙不裂，[[暗灰色|暗灰色]]或[[灰黑色|灰黑色]]。[[樹幹|樹幹]]及枝條[[灰色|灰色]]至深褐色，具分枝的圓刺，嫩枝灰綠色或有時帶[[褐色|褐色]]，有縱棱；刺圓柱形，粗壯，並常分枝，中部至頂端呈[[圓錐狀|圓錐狀]]，基部至頂端橫切面呈圓形，[[紅褐色|紅褐色]]，無毛，具光澤，長可達16厘米。[[葉|葉]]為一回[[偶數|偶數]][[羽狀複葉|羽狀複葉]]，互生或於短枝上簇生，長約10－27厘米，幼樹及萌芽枝條有二回偶數羽狀複葉；小葉3－9對，下部的較小，卵形、卵狀披針形、長圓形至長橢圓形，紙質，先端鈍或漸尖，具短尖頭，基部斜楔形或斜圓形，有時稍歪斜，[[葉緣|葉緣]]具細鋸齒或近全緣，兩面均無毛，或僅在兩面中脈上稍披短柔毛，表面淡黃綠色，無光澤，背面網脈明顯，兩面均突起，長約2－12.5厘米，寬約1－4厘米；小[[葉柄|葉柄]]披黃色短柔毛，長約1－5毫米；[[葉軸|葉軸]]具槽，疏披短柔毛。[[花|花]]為總狀[[花序|花序]]，[[雜性花|雜性花]]；花序頂生或腋生，不分枝，披短柔毛，長5－14厘米。[[花冠|花冠]]淡綠色、白色或黃白色，披短柔毛；[[花梗|花梗]]長約2－10毫米。[[雄花|雄花]]直徑約9－10毫米；[[花托|花托]]深棕色，外披柔毛，長約2.5－3毫米；[[萼片|萼片]]4枚，三角狀披針形，兩面披柔毛，長約3毫米；[[花瓣|花瓣]]4枚，長圓形，披微柔毛，長約4－5毫米；[[雄蕊|雄蕊]]6－8枚，[[花絲|花絲]]基部披長柔毛；退化[[雌蕊|雌蕊]]長約2.5毫米。[[兩性花|兩性花]]直徑約10－12毫米；花梗長約2－5毫米；萼片花瓣與雄花相似；但兩性花萼片長約4－5毫米，花瓣長約5－6毫米；雄蕊8枚；[[子房|子房]]條形，微彎，沿縫線上及基部披白色短柔毛或全體披毛，具短柄，柱頭淺2裂；[[胚珠|胚珠]]多數。[[果|果]]為木質[[莢果|莢果]]，條形或鐮刀形，但不扭轉，基部漸狹成長柄狀，果肉稍厚，兩面臌起，長約5－37厘米，寬約1－4厘米；或有部份莢果短小，呈柱形，長約5－30厘米，寬約1－3.5厘米，彎曲作新月形的常稱為豬牙皂<ref name = "山東植物誌"/>，內無[[種子|種子]]；[[果頸|果頸]]長約1－3.5厘米；[[果瓣|果瓣]]紅褐色或棕褐色，革質，常披白色粉霜；果經冬不落<ref name = "上海植物園"/>。[[種子|種子]]長圓形或橢圓形，扁平，亮棕色，具光澤，長約10－13毫米，寬約8－9毫米。

==一般用途==
皂莢樹因其能抗風、抗寒、耐[[酸鹼|酸鹼]]、適應性強等特性，適合作綠化樹種<ref name="當代藥用植物典">{{cite book|title=《當代藥用植物典 1》|author=|series=趙中振、蕭培根|publisher=香港賽馬會中藥研究院|page=第428-431頁|isbn=978-988-99226-0-3|url=http://www.hkjcicm.org/cm_database/plants/detail.aspx?herb_id=374|date=2006年8月}}{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>；果實能提出[[植物膠|植物膠]]，能提高麵粉的品質<ref name = "當代藥用植物典"/>；種仁因含豐富的[[微量元素|微量元素]]、[[氨基酸|氨基酸]]及[[半乳甘露聚糖|半乳甘露聚糖]]，可供製作保健餅乾、麵包及飲料等食用品<ref name = "當代藥用植物典"/>；莢果煎汁能作天然[[洗滌劑|洗滌劑]]，為洗髮產品的天然原料<ref name = "中國的野菜"/>；木質堅硬但易開裂，耐腐耐磨供製作傢俱及建築中的柱與樁<ref>[http://scitech.people.com.cn/BIG5/25509/99037/99062/99084/99345/6145829.html 皂莢 皂角(Gleditsia sinensis)]人民網</ref><ref name="中國園林建設網">[http://www.china-landscape.net/wz/913.htm 皂莢 （拉丁名：Gleditsia sinensis）]{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}中國園林建設網</ref>。

==醫藥用途==
{{ctm}}
===根皮或莖皮===
皂莢的[[根皮|根皮]]或[[莖皮|莖皮]]入藥，味辛，性溫。[[中藥|中藥]]名為皂莢木皮，秋冬兩季採收，切片曬乾，具解毒散結、祛風殺蟲等功效，主治無名腫毒、[[疥癬|疥癬]]、惡瘡、[[淋巴|淋巴]]結核、[[風濕|風濕]]骨痛等<ref name = "浸會大學"/>。  

===葉===
皂莢的[[葉|葉]]入藥，味辛，性溫。[[中藥名|中藥名]]為皂莢葉，[[春季|春季]]採收，曬乾，具祛風解毒、生髮等功效，主治風熱瘡癬、毛髮不生等<ref name = "浸會大學"/>。 

===棘刺===
皂莢的乾燥[[棘刺|棘刺]]入藥，味辛，性溫<ref name = "浸會大學"/><ref name = "中藥彩色圖集"/><ref name = "常用中藥材鑑別圖典">{{cite book|title=《常用中藥材鑑別圖典》|author= |series=趙中振、陳虎彪|publisher=萬里機構|page=第218頁|isbn= 978-962-14-4150-8|date=2010年1月}}</ref>，[[藥材|藥材]]主產於[[河南|河南]]、[[江蘇|江蘇]]、[[湖北|湖北]]、[[廣西|廣西]]、[[安徽|安徽]]、[[四川|四川]]、[[湖南|湖南]] 等地<ref name = "當代藥用植物典"/>。中藥名為皂莢刺<ref name = "浸會大學"/>，具祛[[痰|痰]]止咳、搜風、消腫透[[膿|膿]]、抗菌、殺蟲散結等功效<ref name = "浸會大學"/><ref name = "當代藥用植物典"/>，主治產後缺乳、[[胎衣|胎衣]]不下、[[瘡疹|瘡疹]]頑癬、癰疽腫毒、排膿、瘰鬁、癘風等<ref name = "浸會大學"/>，外用可治疥癬麻風<ref name = "中藥彩色圖集"/>，現代[[藥理|藥理]]表明皂莢刺具抗菌、祛痰、抗癌、抑癌等作用<ref name = "當代藥用植物典"/>。

===不育果實===
皂莢的乾燥不育[[果實|果實]]入藥<ref name = "浸會大學"/>，味辛、咸，性溫<ref name = "常用中藥材鑑別">{{cite book|title=《常用中藥材鑑別圖典》|author= |series=趙中振、陳虎彪|publisher=萬里機構|page=第373頁|isbn= 978-962-14-4150-8|date=2010年1月}}</ref>，有小毒<ref name = "常用中藥材鑑別"/>，[[藥材|藥材]]主產於[[山東|山東]]、[[四川|四川]]、[[雲南|雲南]]、[[貴州|貴州]]、[[陝西|陝西]]、[[河南|河南]]等地<ref name = "當代藥用植物典"/>。中藥名為豬牙皂、小牙皂、眉皂<ref name = "香港常用有毒中藥"/>，為皂角樹因受外傷、衰弱等原因而結成的不育果實<ref name = "香港常用有毒中藥"/>，具祛痰止咳、殺蟲散結、散結消腫、開竅通閉等功效，主治痰咳喘滿、痰涎壅盛、神昏不語、[[中風|中風]]口噤、[[癲癇|癲癇]]、喉痹、二便不通、癰腫疥癬等<ref name = "浸會大學"/>。 

===成熟果實===
皂莢的乾燥成熟[[果實|果實]]入藥<ref name = "浸會大學"/>，味辛、咸，性溫，有小毒（《[[名醫別錄|名醫別錄]]》）<ref name = "本草綱目白話精譯本"/>，歸肺、大腸經<ref name = "香港常用有毒中藥"/>，[[藥材|藥材]]產於[[河北|河北]]、[[山西|山西]]、[[河南|河南]]、[[山東|山東]]等地<ref name = "香港常用有毒中藥"/>，全國各地均產，多為栽培。中藥名為皂莢、大皂角、山皂莢、皂節（江西）、懸刀（山西、廣西、《[[外丹本草|外丹本草]]》）、雞棲子（《[[廣志|廣志]]》）、烏犀（《[[曾氏方|曾氏方]]》）等<ref name = "本草綱目白話精譯本"/><ref name = "香港常用有毒中藥"/>，皂莢為中醫臨床用藥，藥用之名始載於《[[神農本草經|神農本草經]]》，列為下品<ref name = "香港常用有毒中藥"/>；《[[傷寒總病論|傷寒總病論]]》中記載，蘄、黃兩地，某年從[[春季|春季]]至[[秋季|秋季]]均流行急性[[咽喉|咽喉]]腫痛，引致大量病人死亡，及後[[黃州|黃州]]的[[潘昌言|潘昌言]]使用黑龍膏方救活病人，膏方當中的主要藥材為皂莢<ref name = "圖解神農本草經">{{cite book|title=《圖解神農本草經》|author= |series=陶隱夕|publisher=山東美術出版社|page=第475-476 頁|isbn=978-7-5330-2529-8|date=2008年8月}}</ref>；《[[本草綱目|本草綱目]]》中記載，皂莢能殺蟲、殺精、[[墮胎|墮胎]]，浸於酒中再煎成膏狀，塗抹於布上能敷貼各種腫痛<ref name = "圖解神農本草經"/>；於潮濕久雨天氣時，與[[蒼朮|蒼朮]]同燒能辟瘟疫及濕邪<ref name = "圖解神農本草經"/>；皂莢單獨燃燒時，可燻久痢[[脫肛|脫肛]]，及能通肺及大腸氣，治療痰氣喘咳、瘡瘍癬疥等<ref name = "圖解神農本草經"/>；《[[名醫別錄|名醫別錄]]》中記載，皂莢能治腹部脹滿、消食積、咳嗽、明目益精、婦女胞衣不下等，用於沐浴，但不入湯劑<ref name = "本草綱目白話精譯本"/>；《[[日華諸家本草|日華諸家本草]]》中記載，皂莢能消除頭風、通利關節、消痰、殺蟲、治療骨蒸潮熱、中風牙關緊閉、增進食慾等<ref name = "本草綱目白話精譯本"/>；《[[藥性論|藥性論]]》中記載，破堅癥、腹中痛、能墮胎<ref name = "香港常用有毒中藥"/>。

皂莢具祛頑[[痰|痰]]、祛風殺蟲、通竅開閉、利大小便等功效<ref name = "香港常用有毒中藥"/><ref name = "圖解神農本草經"/>，主治風邪痺阻、喘咳痰多、痰厥昏迷、癲癇、[[中風|中風]]、口噤不語、消腫止等<ref name = "圖解神農本草經"/><ref name = "浸會大學"/>，外用可治風癬疥癩等<ref name = "香港常用有毒中藥"/>，非邪痰痞滿者、虛弱者及[[孕婦|孕婦]]禁服<ref name = "香港常用有毒中藥"/><ref name = "圖解神農本草經"/>。現代臨床應用於[[慢性阻塞性肺病|慢性阻塞性肺病]]、[[哮喘|哮喘]]、[[支氣管|支氣管]]炎、慢性傳染性[[肝炎|肝炎]]、[[陰道炎|陰道炎]]、面神經炎、面神經麻痺、 [[腸梗阻|腸梗阻]]、[[骨癌|骨癌]]、高脂血症等治療上<ref name = "當代藥用植物典"/><ref name = "本草綱目白話精譯本"/>。但服用過量可引致中毒，中毒症狀常於服後1－3小時出現，症狀包括煩躁不安、[[嘔吐|嘔吐]]、惡心、胃脘飽滿及灼熱等；8－10小時後症狀包括頭暈、四肢酸麻、乏力、腹瀉水樣泡沫狀物、嚴重者出現醬油色血性小便等。中毒初期可使用0.02%[[高錳酸鉀|高錳酸鉀]]溶液或[[鞣酸|鞣酸]]溶液洗胃，及需要時導瀉；出現脫水時可輸液，以維持酸鹼平衡；服用[[阿托品|阿托品]]或複方[[樟腦酊|樟腦酊]]；出現煩躁者症狀者可使用[[鎮靜劑|鎮靜劑]]；症狀嚴重至出現溶血現象時，可小量[[輸血|輸血]]<ref name = "香港常用有毒中藥"/><ref name = "中國園林建設網"/>。

===種子===
皂莢的[[種子|種子]]入藥<ref name = "浸會大學"/>，味辛，性溫，無毒<ref name = "本草綱目白話精譯本"/>。中藥名為皂莢子，秋季果實成熟時採收，曬乾，具祛風散熱、化痰散結、潤腸通便等功效<ref name = "浸會大學"/>，主治痰喘腫滿、大便燥結、腸風下血、[[疝氣|疝氣]]疼痛、瘰鬁、腫毒、瘡癬等，體虛者及孕婦禁服<ref name = "浸會大學"/> 。

==藥材鑑定==
本品種載錄於《[[中國藥典|中國藥典]]》2005年版，定為[[中藥|中藥]]皂莢刺及豬牙皂的法定原植物[[來源種|來源種]]，附錄中定為[[中藥|中藥]]大皂角的法定原植物[[來源種|來源種]]，主要透過性狀及顯微鑒別等資料，以控制[[藥材|藥材]]的質量<ref name = "當代藥用植物典"/>。 

===棘刺===
[[中藥|中藥]]皂莢刺以皂莢的乾燥棘刺入藥<ref name = "浸會大學"/>，全年均可採收<ref name = "浸會大學"/><ref name = "常用中藥材鑑別圖典"/>，以9月至翌年3月間採收為宜<ref name = "浸會大學"/>，採收後乾燥或趁鮮切後乾燥<ref name = "常用中藥材鑑別圖典"/>；完整的主棘及1－2次分枝的棘刺，扁圓柱形，末端尖銳；主刺長約3－15厘米或更長，直徑約0.3－1厘米；分枝棘刺長約1－6厘米；表面紫棕色或棕褐色，光滑或有細皺紋；木部黃白色；髓部淡紅棕色，鬆軟；體輕，質地堅硬，不易折斷。氣微，味淡；切片厚約0.1－3厘米，常帶尖細的刺端；粉狀具刺激性，誤吸能使人作嚏<ref name = "中藥彩色圖集"/><ref name = "常用中藥材鑑別圖典"/>。傳統經驗則認為以無枝梗，紫棕色；髓部切片紅棕色，鬆軟者為佳<ref name = "常用中藥材鑑別圖典"/>。

===不育果實===
[[中藥|中藥]]豬牙皂以皂莢的乾燥不育果實入藥<ref name = "浸會大學"/>，秋季果實成熟變黑時採收<ref name = "浸會大學"/><ref name = "常用中藥材鑑別"/>，採收後除去雜質後乾燥<ref name = "常用中藥材鑑別"/>；扁圓柱形，多彎曲成[[鐮刀|鐮刀]]狀，頂端漸尖，有[[鳥喙|鳥喙]]狀[[花柱|花柱]]殘基，基部驟細狹長，有[[果梗|果梗]]殘痕；表面黑紫色或紫棕色，披灰白色蠟質粉霜，擦去粉霜後具光澤，部份現細小的疣狀突起及腺體，淡黃色短裂紋；腹縫線突起呈稜脊狀；質地堅硬而脆，易折斷， 斷面棕黃色，中間疏鬆，有淡綠色或淡棕黃色的絲狀物，呈絲網樣；縱向剖開可見整齊的凹窩，偶有發育不全的[[種子|種子]]<ref name = "香港常用有毒中藥"/><ref name = "常用中藥材鑑別"/>。氣微，有刺激性，味先甜後辣<ref name = "常用中藥材鑑別"/>；粉狀紅棕色具刺激性，誤吸能使人作嚏<ref name = "香港常用有毒中藥"/>。傳統經驗則認為以紫褐色、個小、飽滿、有光澤、無果梗者為佳<ref name = "常用中藥材鑑別"/>。

===果實===
皂莢的乾燥成熟果實入藥<ref name = "浸會大學"/>，秋季果實成熟時採收<ref name = "浸會大學"/>，採收後除去雜質後乾燥<ref name = "浸會大學"/>；扁長呈劍鞘狀或畸彎曲；表面紫褐色或黑紫色，光滑，具不規則的網狀皺紋；頂端稍尖，基部鈍圓，有果梗或斷痕；兩側有明顯的縱稜線；種子所在處隆起；質地堅硬，折斷處纖維性，斷面黃色或灰黃色；種子扁橢圓形，黃褐色，光滑，多數。長約2－25厘米，寬約2－3厘米，厚約1厘米<ref name = "香港常用有毒中藥"/>。氣微，味辛辣，搖之有響聲<ref name = "香港常用有毒中藥"/>；粉狀具刺激性，誤吸能使人作嚏<ref name = "香港常用有毒中藥"/>。

==典故==
相傳古時有一農家少女，長得如花似玉，一天少女於野外打柴，不幸遇上村外一惡少，此惡少在鄰近數村壞事做盡，惡少對少女頓起色心，將少女強姦了。少女受辱後，自覺丟人，在一顆大皂角樹上自縊身亡。女兒突然死去，痛不欲生，哭得淚也乾了，嗓門也叫破了，盼望女兒能起死回生。突然一位滿頭白髮的老翁出現，向農戶兩夫婦說道：「老翁自有還魂之術，請用皂角末吹入少女鼻孔，方能起死回生！」， 少女父親淚眼抬望，發現老翁悄然融入皂角樹之中，方知樹神顯靈，父母兩人立刻依照老翁的吩咐，摘下皂角碾成粉末，輕輕吹入愛女鼻孔，過了一會兒，少女鼻孔微動，接着猛然一聲噴嚏後，漸漸恢復蘇醒過來。自此皂角便成為了民間的靈丹妙藥<ref>{{cite book|title=《208種中藥典故》|author= |series=張虹、湯金城|publisher=中原農民出版社 |page=第364-367頁|isbn= 978-7-80739-716-8|date=2009年8月}}</ref><ref>{{cite book|title=《中藥的故事》|author= |series=陳沬金|publisher=百花文藝出版|page=第211-212頁|isbn= 978-7-5306-5563-4|date=2010年4月}}</ref>。
