'''音樂記譜法'''（樂譜）是一種以印刷或手寫製作，用[[符號|符號]]來記錄[[音樂|音樂]]的方法。不同的文化和地區發展了不同的記譜方法，除了現今常見音樂的[[五線譜|五線譜]]和[[簡譜|簡譜]]外，在[[傳統音樂|傳統音樂]]中還有[[工尺譜|工尺譜]]，以及[[古琴|古琴]]特有的[[減字譜|減字譜]]。這些記譜法各有其特色，有些雖然對樂曲中的裝飾音、力度變化都做了標記，但其實只是供初學者作為參考<ref>

== 历史 ==
不同樂種因為各有其特點，所發展出來的記譜法往往有所差異，而隨著樂種本身的持續發展，記譜法也可能不斷地發生變化。如五線譜，除了紀錄樂曲旋律的音高、時值外，也對力度做了標記，然而隨著音樂家對力度、音色的日益重視<ref>{{GroveOnline|Dynamics|January 28|2015  }}</ref>，發現這些細緻的變化其實難以精確紀錄，且這些變化又具有一定的即興演奏成分<ref>{{GroveOnline|Improvisation, §II: Western art music|January 28|2015  }}</ref>，於是又發展出在一定框架內自由發揮的記譜法，以使音樂更富有彈性與活力，而這恰如[[古琴|古琴]]或其它傳統音樂的記譜特色<ref>

=== 歐洲 ===
[[File:Bachlut1.gif|thumb]]
1985年，[[柏克萊加州大學|柏克萊加州大學]]的Anne Draffkorn Kilmer曾發表文章論證[[楔形文字|楔形文字]]板上發現了最早的記譜法。

[[希臘羅馬時期|希臘羅馬時期]]，已經有簡單的符號粗略地代表音的高低，不少詩歌會有節奏標記，如長短格（trochaic）和短長格（iambic）等等。

[[中世紀|中世紀]]時代，樂譜從早期的[[紐姆記譜法|紐姆譜]]（Neumatic notation）逐漸發展到現代的五線譜。樂譜從單聲部逐漸增加到多聲部的分譜，亦開始有不同節奏的標記。

從[[巴洛克音樂|巴洛克音樂]]到現在，樂譜出版事業蓬勃發展，大大推動了音樂的傳播，並因為[[樂隊|樂隊]]的發展以及各種新穎樂器技巧的出現而增添了更多的符號。[[作曲家|作曲家]]會精細地標記音量、速度和音色變化。

隨著[[爵士樂|爵士樂]]和[[流行音樂|流行音樂]]的發展，不少樂譜同時標有旋律、和弦標記以及吉他指法。

=== 中國 ===
[[File:Elegant_Orchid.jpg|right]]
中國留傳下來最早的樂譜是[[南北朝|南北朝]]時代的[[古琴譜|古琴譜]]《[[碣石調·幽蘭|碣石調·幽蘭]]》。早期的樂譜為文字譜，將指法詳細地以文字敍述。後來為了簡化記錄方法，[[唐朝|唐朝]]後期出現了[[減字譜|減字譜]]，記錄指法動作、弦序和徽位。敦煌的[[莫高窟|莫高窟]]收藏了不少唐宋以來的樂譜，其中包括不少琵琶樂譜。宋代的[[姜白石|姜白石]]道人歌曲出現了記錄音高的[[俗字譜|俗字譜]]，證明當時的[[宋詞|宋詞]]是可以歌唱的。明清兩代的[[戲曲|戲曲]]，包括崑劇、京劇、粵劇以及當時不少器樂曲都應用了[[工尺譜|工尺譜]]。[[潮州音樂|潮州音樂]]有其獨特的[[二四譜|二四譜]]。

[[民國|民國]]年間，受到西方以及日本的影響，中國的學堂樂歌開始應用[[簡譜|簡譜]]以及[[五線譜|五線譜]]。直至今天，在流行歌曲中，以[[簡譜|簡譜]]和[[五線譜|五線譜]]居多。在戲曲，工尺譜也依然有人使用。在瑤琴領域減字譜仍在繼續使用（一般是與簡譜或者五線譜同時使用）。在南音中仍然在使用傳統的工尺譜。傳統音樂的一些樂器也使用簡譜或者五線譜。

簡譜、五線譜由於一些根本性的缺陷，無法在傳統音樂中代替減字譜、工尺譜。在瑤琴領域，使用簡譜（五線譜）記錄的某些曲子，只代表這個記錄者的彈奏方法，其它的琴人仍然可以按照古譜作出自己的彈法。之所以出現這種現象，是因為減字譜本身比較模糊，允許實際演奏者做一些自己的發揮，自由度比較大。南音中工尺譜也有類型情形。

==== 大譜表 ====
鋼琴音樂的樂譜最初是紀錄在十一條線的大譜表上。
音符由字母表的前七個字母來命名並反覆運用。

由於在如此多的線條上閱讀音符比較困難，就用譜號把F、C、和G線標記出來。

在現代音樂中，為了方便讀譜，將大譜表分成兩部分，每部份由五條平行橫線組成。

中間的一條橫線省去。於是，中央C記在中間一條短線上，稱為加線，並且在需要的時候才使用。<ref> {{Cite book | author = 威拉德·阿·帕爾默、莫頓·曼努斯、阿曼達·維克·萊斯科 | title = 快樂鋼琴基礎教程 樂理·1B級（Alfred's Basic Piano Course） | publisher = 上海音乐出版社 | date = 2015年7月1日 | pages = 2 | ISBN = 9787552307313 | accessdate = 2016-7-31 language = 中文 }} </ref>

==== 拍号 ====
{{Main|拍号|拍子}}
节拍是音乐中有规律地强拍和弱拍的反复，如果我们跟随音乐的进行用脚一下一下地点地，每一下就是一拍，打击乐器在音乐中就是配合节拍演奏的。

在乐谱中，拍号是用分数的形式来标画的。每一个乐谱前面都有拍号，拍号如同分数'''{{nowrap|{{su|p=3|b=4}}}}'''，'''{{nowrap|{{su|p=4|b=4}}}}'''，'''{{nowrap|{{su|p=2|b=2}}}}'''。

分母表示拍子的时值也就是说用几分音符来当一拍

分子代表每一小节有多少拍子

例如：

<score>
  \relative c'' {
     \time 3/4
     d,
  }
</score>

'''{{nowrap|{{su|p=3|b=4}}}}'''表示每小节有三拍，以四分音符（<span style="font-size: xx-large">{{Unicode|♩}}</span>）为一拍

==== 力度记号 ====
{{Main|强弱法}}
力度是指在曲谱或音乐表演中音的强弱程度。通常音的强弱变化对音乐形象的塑造也起着非常重要的作用。

<score>
\new GrandStaff <<
  \new Staff = "violin" {
 {
      \set Staff.midiInstrument = #"violin"
      % not strictly necessary, but a good reminder
      \transposition c'
      \key c \major 
       c'\mp d' e' d' e' f' g' f' g'\p a' b' a'
    }
  }
  \new Staff = "clarinet" {
\clef bass
      \relative c'' {
      \set Staff.instrumentName = \markup { Cl (B\flat) }
      \set Staff.midiInstrument = #"clarinet"
      \transposition bes
      \key c \major
      c,,1 \mf e1 a1\f
    }
  }
>>
</score>

兩個用來表示相對強弱程度的基本[[符號|符號]]是：
* <span style="font-family: 'Times New Roman', Times, serif; font-size: large;">'''''p'''''</span> —[[意大利語|意大利語]] ''piano'' 的縮寫，表示弱
* <span style="font-family: 'Times New Roman', Times, serif; font-size: large;">'''''f'''''</span> —[[意大利語|意大利語]] ''forte'' 的縮寫，表示強

在這兩個基本[[符號|符號]]以上建立的另外一些[[符號|符號]]包括：
* <span style="font-family: 'Times New Roman', Times, serif; font-size: large;">'''''mf'''''</span> —[[意大利語|意大利語]] ''mezzo-forte'' 的縮寫，表示中等強度
* <span style="font-family: 'Times New Roman', Times, serif; font-size: large;">'''''mp'''''</span> —[[意大利语|意大利语]] ''mezzo-piano'' 的缩写，表示中等弱度

===指法譜（Tablature或者Tab）===
{{le|指法譜|Tablature}}是根據樂器本身的特點標出如何在該樂器奏出每個音（有些包括指法）的記譜法。音符標記並不代表音高。這種記譜法多用在有琴桁（也叫品）的弦樂樂器上。目前最為廣泛使用的，是吉他六線譜。六根線代表吉他的六根琴弦。線上的數字代表用右手撥響該琴弦時左手需要按住該琴弦的第幾品。

== 樂譜製作程式 ==

常用的五線譜電腦製作程式有[[MuseScore|MuseScore]]、[[Sibelius|Sibelius]]、[[Finale|Finale]]、[[Lilypond|Lilypond]]、[[Encore_(軟件)|Encore]]、[[Overture|Overture]]等等。亦有專門製作吉他樂譜的程式[[Guitar-pro|Guitar-pro]]。簡譜製作有兩種常用的方法，一種是以一般文書處理程式製作，如[[Microsoft_Word|Microsoft Word]]，配上簡譜字體，如[[彩虹簡譜字庫|彩虹簡譜字庫]]；另一種為專門製作簡譜的程式，如中國開發的[[作曲大師|作曲大師]]。
