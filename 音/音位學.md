'''音位學'''（{{lang-en|Phonology 或 Phonemics}}），又稱'''音系學'''（中國）、'''音韻論'''（日本）、'''音素學'''、'''音韻學'''（台灣）<ref>{{Cite web|url =http://linguist.ccu.edu.tw/ezcatfiles/b004/img/img/40/MAlxcoursemap_chinese.pdf|title =國立中正大學語言學研究所語言學組碩士班課程地圖|accessdate =2015年10月27日|deadurl =yes|archiveurl =https://web.archive.org/web/20160322000536/http://linguist.ccu.edu.tw/ezcatfiles/b004/img/img/40/MAlxcoursemap_chinese.pdf|archivedate =2016年3月22日}}</ref> <ref>{{Cite web |url = http://ling.nccu.edu.tw/course/pages.php?ID=course |title = 國立政治大學語言學研究所【碩士班】專業必修科目一覽表 |accessdate =2015年10月27日}}</ref>，為[[語言學|語言學]]分支學科，是研究某一[[語言|語言]]的[[音位|音位]]的學問<ref name=YGX/>。

音位學家針對某一語言的[[語音系統|語音系統]]進行科學分析，研究不同的[[音素|聲音元素]]是如何被串接、組合起來而形成一個特定的語言（如[[英語|英語]]），並且表達出特定的意義。在[[漢語|漢語]]語境下的[[音位學|音位學]]通常指[[漢語音韻學|漢語音韻學]]，然而其內涵並非針對[[漢語|漢語]]所做之音位學，而是漢語之[[歷史語言學|歷史語言學]]<ref name=JS/>。

[[音位學|音位學]]是研究[[音|音]]的結構的兩大學科分支之一，另一分支為[[語音學|語音學]]，二者的區別在於[[音位學|音位學]]側重研究語音的功能，[[語音學|語音學]]側重研究語音的產生<ref name=web1/>。

== 概念 ==

* [[語音|語音]]：即[[音|音]]，泛指[[人類|人類]]所發出的[[聲音|聲音]]
* [[音位|音位]]：亦稱[[音素|音素]]、[[語素|語素]]，某一[[語言|語言]]或[[方言|方言]]中用於區別意義的最小[[語音單位|語音單位]]
* [[音系|音系]]：某一[[語言|語言]]中[[語音|語音]]的[[發音規律|組織規律]]，包括語音的對立及互補分佈、[[:en:phonotactics|配列規則]]{{en icon}}及[[語素|語素]]的[[變體|變體]]等<ref>Zhang, J. [http://www.linguistics.ku.edu/people/faculty/zhang_pdf/experimental-phonology.pdf Experimental Phonology] {{webarchive|url=https://web.archive.org/web/20131203011320/http://www.linguistics.ku.edu/people/faculty/zhang_pdf/experimental-phonology.pdf |date=2013-12-03 }}, [[堪萨斯大学|University of Kansas, USA]].</ref>{{rp|2-3}}
* [[語音學|語音學]]：研究[[人類|人類]][[語音|語音]]的學問

=== 音與音位 ===

[[結構語言學|結構語言學派]]將[[音位|音位]]定義為「能夠區別意義的最小語音單位」。音位是抽象的且本身不具有含義；音位可具化為實際的[[語音|音]]，該過程稱為音位的[[:en:phonetic_realization|語音體現]]{{en icon}}。一個音位可能包含著幾個不同[[音值|音值]]的音，因而可以體現為一個音、兩個音或更多的[[同位音|同位音]]。<ref name=web2/><ref>洪惟仁. [http://www.uijin.idv.tw/doc/%E7%B7%9A%E4%B8%8A%E6%95%99%E5%AD%B8%E8%88%87%E5%AD%B8%E7%94%9F%E4%BD%9C%E5%93%81/%E6%BC%A2%E8%AA%9E%E8%AA%9E%E9%9F%B3%E8%88%87%E9%9F%B3%E9%9F%BB/2%E8%AC%9B%E7%BE%A9%E6%95%99%E6%9D%90/02-0%E9%9F%B3%E4%BD%8D%E5%AD%B8.pdf ''第二章 音位學''], 漢語語音與音韻.</ref>{{rp|3}}

=== 與語音學的關係 ===

[[語音學|語音學]]主要處理語音的物理性質描述，而音位學主要處理一個語言系統中語音的功能。二者亦有聯繫，如在音位學的分析中會利用語音學的基本知識，[[拼字學|拼字學]]是在音位學的分析基礎上產生的。<ref name=web2/>

* [[語音學|語音學]]：
# 研究方向：發音與人體生理結構的關係，較近於語音的具體層面<br>即人類如何運用其口、鼻等器官與組織來發出各種不同的[[音素|聲音元素]]，如英語的各個母音與子音<ref name=JS/>
# 側重於發音部位、發音方式，以及語音特徵的研究
# 特徵：只注意[[語音|語音]]的實質不同，不考慮語音在整個[[語音系統|語音系統]]裡有無意義

* 音位學：
# 研究方向：[[發音|發音]]差異與[[語義|語義]]的關係，較近於語音的抽象層面<br>以英語可數名詞"書本"'''''book''''' 為例，當說話者要表達書本的數目大於一本的意思時，就要在其字尾多發一個'''''s''''' 的聲音元素（[[子音|子音]]）<ref name=JS/>
# 側重於各個獨立的基本音在語音系統的重要性，語音結構及語音組合規律
# 特徵：[[發音|發音]]差異在整個語音系統裡是否有意義
