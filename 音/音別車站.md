'''音別車站'''（{{jpn|j=音別駅|hg=おんべつえき|rm=Ombetsu eki|f=y}}）是位於[[北海道|北海道]][[釧路市|釧路市]]音別町本町1丁目的[[北海道旅客鐵道|北海道旅客鐵道]]（JR北海道）、[[日本貨物鐵道|日本貨物鐵道]]（JR貨物）[[根室本線|根室本線]]的[[鐵路車站|鐵路車站]]。車站編號為'''K45'''。電報略號為'''オン'''。為被廢除的夜行特急列車「[[毬藻_(列車)|毬藻]]」的車站，現在只有普通列車會在本車站停靠。

車站的名稱源自流經本地的河流[[音別川|音別川]]（o-mu-pet，意思為河口堵塞的河流）。

== 歷史 ==
* 1903年（[[明治|明治]]36年）3月1日 - 以[[北海道官設鐵道|北海道官設鐵道]]釧路線車站開始營業。為一般車站。
* 1905年（明治38年）4月1日 - 由[[日本國有鐵道|官設鐵道]]接管。
* 1984年（[[昭和|昭和]]59年）2月1日 - 廢除行李及車載貨物處理。開始貨櫃貨物處理。
* 1987年（昭和62年）4月1日 - [[國鐵分割民營化|國鐵分割民營化]]後，由JR北海道繼承。
* 2008年（[[平成|平成]]20年）8月31日 - 為特急列車「[[毬藻_(列車)|毬藻]]」的最後行駛日（以本車站計，往[[札幌站|札幌]]於8月31日、往[[釧路車站|釧路]]於9月1日停止行駛）。從此本車站唯一特別列車取消。

== 車站構造 ==
* 音別車站為2面2線、[[側式月台|側式月台]]的地面車站。月台之間設置[[平交道|平交道]]。上下行列車主要使用位於車站大樓旁的1號線，而轉換列車等時則使用2號線。
* 為配置職員車站，並設有[[綠窗口|綠窗口]]（營業時間07時20分～15時00分）。
* 在早上及晚上分別有1班及2班行駛[[釧路車站|釧路]]～本車站之間的列車。

== 貨物車站 ==
JR貨物車站位於旅客車站南面，為1面1線的貨櫃月台（全長約200米）。
