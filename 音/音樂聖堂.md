《'''音樂聖堂'''》（{{lang-en|'''Music'''}}）是美國女歌手[[瑪丹娜|瑪丹娜]]在2000年8月發行的歌曲，為她個人第九張[[錄音室專輯|錄音室專輯]]《'''[[Music_(麥當娜專輯)|音樂聖堂]]'''》（{{lang-en|'''Music'''}}）同名單曲，它在[[Billboard_Hot_100|告示牌單曲榜]]獲得四週冠軍，在英國單曲榜獲得一週冠軍<ref>[http://www.officialcharts.com/artist/_/madonna/ 英國官方排行榜瑪丹娜資料庫]</ref>。

== 歌曲簡介 ==
〈音樂聖堂〉由[[瑪丹娜|瑪丹娜]]和法國前衛派[[電子舞曲|電子舞曲]][[製作人|製作人]]{{link-en|米爾維斯·阿馬黛|Mirwais Ahmadzaï}}共同創作，兩人把舊式舞曲節奏和未來舞曲元素融合在一起，呈現出跨越不同舞曲時空的動感時尚，瑪丹娜在〈音樂聖堂〉加進了七O年代電子[[放克|放克]]與時下[[流行電音|流行電音]]。這支單曲成功的將蟄伏兩年半才發新專輯的瑪丹娜再度躍上排行榜冠軍寶座，無疑是給了2000年8月11日才又再度成為人母的瑪丹娜一個祝賀大禮。

== 銷售成績 ==
〈音樂聖堂〉在2000年8月12日進入告示牌單曲榜，首週成績穩坐第41名<ref>[http://www.billboard.com/charts/hot-100/2000-08-12 2000-08-12告示牌單曲榜]</ref>，強勁的爆發力讓這支單曲快速竄升<ref>[http://www.billboard.com/charts/hot-100/2000-09-16 2000-09-16告示牌單曲榜]</ref>。〈音樂聖堂〉在進榜第六週擊敗另一位流行天后[[珍娜·傑克森|珍娜·傑克森]]的〈[[Doesn't_Really_Matter|Doesn't Really Matter]]〉逕自登上冠軍位置，並蟬連了四週，單曲銷售更高達1白金<ref>[http://www.riaa.com/goldandplatinumdata.php?content_selector=gold-platinum-searchable-database 美國唱片業協會瑪丹娜專輯單曲銷售量一覽表]</ref>，它在榜內停留了24週<ref>[http://www.billboard.com/artist/308786/madonna/chart?page=1&f=379 美國告示牌官網瑪丹娜單曲資料庫]</ref>。這是瑪丹娜第12首美國冠軍單曲，〈音樂聖堂〉更是她睽違六年之後再度榮登單曲榜榜首的紀錄，它在2000年告示牌年終單曲榜名列第17名。

另外〈音樂聖堂〉在[[告示牌舞曲排行榜|舞曲榜]]蟬連五週冠軍，是2000年在舞曲榜表現最好的一首單曲，〈音樂聖堂〉成為2000年告示牌年度最佳「熱門舞曲榜」第一名。從〈音樂聖堂〉奪冠截至2015年為止，這16年來熱門舞曲榜再也沒出現另一首五週冠軍的舞曲，這首單曲在舞曲榜的紀錄可謂彌足珍貴。

〈音樂聖堂〉在英國單曲榜也獲得一週冠軍<ref>[http://www.officialcharts.com/artist/_/madonna/ 英國官方排行榜瑪丹娜資料庫]</ref>，成為[[瑪丹娜|瑪丹娜]]第10首英國冠軍單曲。〈音樂聖堂〉獲得2001年[[葛萊美獎|葛萊美獎]]最佳年度唱片的提名。

== 單曲收錄 ==
〈音樂聖堂〉除收錄於[[瑪丹娜|瑪丹娜]]2000年專輯《'''[[Music_(麥當娜專輯)|音樂聖堂]]'''》（{{lang-en|'''Music'''}}）中，亦收錄在瑪丹娜日後的精選專輯當中。2001年《'''[[GHV2|就是娜精選輯]]'''》（{{lang-en|'''GHV2'''}}）以及2009年《'''[[Celebration_(麥當娜專輯)|娜經典]]'''》（{{lang-en|'''Celebration'''}}）。
