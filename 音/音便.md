'''音便'''是指[[日语|日語]][[歷史|歷史]]上，為了[[發音|發音]]方便而產生的在詞中、詞末出現的[[连音|連音]]變化。在[[日本語國語史|日本語國語史]]、[[音位學|音位學]]、和[[構詞學|構詞學]]中有多種多樣的現象。

另外，本條目採用以下的標記方式進行說明。
* 在用例中，發生了音便的音使用'''粗體'''表示。 如果之後的音發生了[[濁音|濁音]]化的情況下也使用粗體。
* 現代不再使用的歷史上的例子使用 '''†''' 、[[标准语|標準語]]中沒有使用而來自[[方言|方言]]的例子使用 '''‡''' 標記。
* 對於[[动词|動詞]]等的[[变位|活用]]種類，有必要時，採用「標準現代語法中的活用種類名（古典文法中的活用種類名）」的形式表現。
* 包含音標文字的時候，歷史上[[唇音|唇音]]的ハ行音 {{IPA|ɸ}} 用h標記。

== 概要 ==
音便，是詞中1個音（1個音節）變成了別的音的現象。

變化後的音是「'''[[ウ|ウ]]'''」(u) 「'''[[イ|イ]]'''」(i) 「'''[[ン|ン]]'''」([[ん|撥音]]) 「'''[[ッ|ッ]]'''」（[[促音|促音]]）的時候分別成為'''ウ音便'''、'''イ音便'''、'''撥音便、'''和'''促音便'''<ref>{{Cite web|url=http://www.nihongokyoshi.co.jp/manbow/manbow.php?id=1161&TAB=1|title=日本語教師のページ→音便|accessdate=2016年11月29日|author=ARC Academy|date=|publisher=}}</ref>。音便不會出現在詞頭，而僅出現在'''詞中'''或者[[动词|動詞]]和[[形容詞|形容詞]]的'''活用詞尾'''。隨著音便化產生，[[音拍|音拍]]的長度時常會發生變化<ref>{{Cite journal|title=日本語の動詞における音便の問題|author=久保 さやか|url=http://www.venus.dti.ne.jp/~kaisaki/gengoken/pdf/vol4/kubo4.pdf|journal=Cercle linguistique de Waseda|issue=|doi=|others=|year=2000|volume=4|page=16|pmid=}}</ref><ref group="注">另外，也有在什麼都沒有的地方產生「ウ」音的例子（「設く　マク maku→ マウクmauku」）被稱作ウ音便的時候。</ref>。

以下為各音便的性質。
* '''ウ音便'''和'''撥音便'''關係較近。也就是說，產生ウ音便的環境也容易產生撥音便，反之亦然。同様地，'''イ音便'''和'''促音便'''關係也較近。
* 產生ウ音便、撥音便的多為'''[[イ段|イ段]]（元音是i，下同）'''、'''[[ウ段|ウ段]]（u）'''的音。 [[濁音|濁音]]型的多為「'''[[ミ|ミ]]'''」(mi) 「'''[[ビ|ビ]]'''」(bi) 「'''[[ム|ム]]'''」(mu)等。 '''ア段 (a)'''、'''[[エ段|エ段]] (e)'''、'''[[オ段|オ段]] (o)'''的例子也有少數，但多為特例。
* イ音便和促音便僅限於'''[[イ段|イ段]]'''音產生。

=== 「濁音型」的音便 ===
[[鼻音_(辅音)|鼻音]]（'''[[マ行|マ行]]'''音 ([[輔音|輔音]]為m的ma, mi, mu, me, mo，下同)、'''[[ナ行|ナ行]]'''音 (na等)）或和鼻音較親近的音（'''[[ば行|バ行]]'''音 (ba等)、'''[[が行|ガ行]]'''音 (ga等)）發生音便化的情況下，之後的音會伴隨發生強制的濁音化<ref group="注">在音便化之後原來的輔音丟失了之後，一部份原本輔音的性質也被元音以鼻化元音的形式得以保留，後續音再對此進行吸收。</ref><ref group="注">雖然較少但也有カ行等產生濁音化的例子 </ref>。

下文中這種類型的音便統稱為「'''濁音型'''」。

== 歷史 ==
歷史上，各音便都有自[[平安时代|平安時代]]初期開始音便化的例子。一種說法是，音便產生的背景中，大量流入的[[漢音|漢字音]]有著很大的影響<ref group="注">中央人多數在日常模仿中國音，所以日本語的發音體系強製性得到推廣，使音便變得可能。 
</ref>。

所有的音便並非一齊同時出現，而是根據音便的形態和音型不同而在出現時期有分前後。但是在[[院政|院政期]]時代，現在所知的多數音便已經出現了。

ウ音便、イ音便被廣泛使用，但撥音便、促音便基本是在[[文言文|漢文]]的[[漢文訓讀|訓讀]]或[[漢字詞|漢字詞]]中使用。另外，在[[和歌|和歌]]中幾乎不可見音便之例。

== 根據出現位置的分類 ==

=== 在活用詞尾出現的音便 ===

==== 動詞[[連用形|連用形]]＋「て」(te)「た」(ta)等 ====
像「読んで」(yonde)「読んだ」(yonda)「読んだら」(yondara)「読んだり」(yondari)等等一樣，[[活用_(日語)|五段動詞]]（[[四段活用|四段動詞]]、[[ラ行活用|ラ變動詞]]、[[ナ行活用|ナ變動詞]]）的[[連用形|連用形]]在詞尾接上「'''て'''」(te) 「'''た''' （< たり < て-あり）」(te) 「'''たら'''」(tara) 或列舉用的「'''たり'''」(tari) 的時候產生的變化。 ※在以下的用例中，為避免冗長僅使用「て」為例。

根據詞幹（詞尾）輔音，撥音便、イ音便、促音便都可能發生，在[[方言|方言]]中也會出現ウ音便。 

下文中為了方便，將此種類的音便稱為「動詞'''テ・タ形'''」。

==== [[形容词|形容詞]]的[[連體型|連體形]]・[[終止形|終止形]] ====
[[形容词|形容詞]]的[[連體型|連體形]]的詞尾原本是「-き」(ki)「-しき」(shiki)，但是像「高き → 高い」 「久しき → 久しい」一樣，產生了'''k''' 音的脱落然後發生イ音便化。另外，[[終止形|終止形]]原本是「-し」(shi)，但在此後由於[[语法|語法]]的單純化，也變成了和[[連體形|連體形]]沒有區別的「-い」。

==== [[形容词|形容詞]]的[[連用形|連用形]] ====
在形容詞的[[連用形|連用形]]中也有「‡高く → 高う」 「‡久しく → 久しう」這樣 '''k''' 音脫落的例子，這也被稱為ウ音便。從[[平安时代|平安時代]]中期開始在中央（[[京都|京都]]）被大量使用，現在在[[西日本|西日本]]也被廣泛使用。

[[關東地方|關東]]、[[東北地方|東北]]的方言原本不使用這種用法，但在[[東京方言|東京山手方言]]和[[标准语|標準語]]中，因為[[日語敬語|敬語]]體系受到[[近畿方言|近畿方言]]較強的影響，在「ありがとうございます」「美しゅうございます」「うれしゅう存じます」這樣後面跟著「ございます」「存じます」的情況下也會有ウ音便<ref group="注">「ありがとう」 「お早う」 是這種音便後把「ございます」 脫落的簡略形。</ref>。也有「かろうじて」 「まっとうする」等等，以固定的形態進入標準語並被廣泛使用的詞。這些之外的標準語中沒有的ウ音便，多數日本人也透過日常生活或者電視而耳濡目染。

==== 形容詞的「〜かった」 ====
形容詞的過去式「高かった」、假定表現「高かったら」、列舉表現「高かったり」等等也來自促音便。詳細請參考促音便一節。

=== 其他音便 ===
不是活用詞尾，而在[[名詞|名詞]]、[[動詞|動詞]]、[[形容詞|形容詞]]、[[副詞|副詞]]等的所謂[[詞幹|詞幹]]中發生的音便也有很多。與活用詞尾的音便不同的是，這樣的音便後面必須要在後面有跟著其他的後續音，而不會出現在詞尾。

== 音形上的分類和用例 ==

=== ウ音便 ===
變化成「ウ」音的音便被稱為'''ウ音便'''

ウ音便化的產生的[[双元音|雙元音]]經過之後的變化，多變成現代日本語的直音（一音拍）。<ref group="注">au, iu, eu, ou 分別變成 o:, yu:, yo:, o: 。 </ref>

==== 濁音型的ウ音便 ====
* 「香（か）-美（くは）し」　カ'''グハ'''シ ka gu ha shi → カ'''ウバ'''シ ka u ba shi → コーバシイ ko-bashii 「香ばしい」<ref group="注">因此「かぐわしい」 「香ばしい」 「芳しい」 是三式詞，詞源相同。</ref>
* 「日向」　ヒ'''ムカ''' hi mu ka→ ヒ'''ウガ hi u ga''' → ヒューガ hyu-ga
* 「髪-掻き」　カ'''ミカ'''キ ka mi ka ki → カ'''ウガ'''イ ka u ga i → コーガイ ko-gai「笄」<ref group="注">イ音便也同時發生。</ref>
* 「中-人」　ナカ'''ビト''' naka '''bi to'''→ ナカ'''ウド''' naka u do→ ナコード nako-do「仲人」
* 「醸し」　カ'''モシ''' kamoshi → カ'''ウジ kauji''' → コージ ko-ji 「麹」
方言中，[[中國地方|中國地方]]、[[四國|四國]]、[[九州|九州]]的一部份，或者是[[狂言|狂言]]<ref>{{Cite journal|title=バ・マ行四段動詞音便形の変遷--ウ音便から撥音便へ 狂言記を中心に|author=野田涼子|url=|journal=山梨大学国語・国文と国語教育|issue=|doi=|others=|year=2008|volume=15|page=17-28|issn=09101357|pmid=}}</ref>的台詞中，在[[ま行|マ行]]・[[ば行|バ行]]的[[五段動詞|五段動詞]]（[[四段活用|四段動詞]]）的テ・タ形中也有出現同樣的音便。
* ‡ 「読み-て」　ヨ'''ミテ yomite'''→ ヨ'''ウデ youde''' → ヨーデ yo-de
* ‡ 「呼び-て」　ヨ'''ビテ''' yobite → ヨ'''ウデ youde''' → ヨーデ yo-de

==== [[は行|ハ行]]、[[か行|カ行]]等的ウ音便 ====
零散可見的後續音不發生濁音化的例子。
* 「白-人」　シロ'''ヒ'''ト shirohito → シロ'''ウ'''トshirouto → シロートshiro-to 「素人」
* 「箒き」　ハ'''ハ'''キ hahaki → ハ'''ウ'''キ hauki「箒」<ref group="注">ア段的音便發生非常少，這個例子中實質發生的變化或可理解成「a音的脱落」（hahaki > *hawaki > *hawki > ho:ki）。</ref>
方言中西日本的多數區域，有廣泛發生[[わ行|ワ行]][[五段動詞|五段動詞]]（[[は行|ハ行]][[四段活用|四段動詞]]）的テ・タ形，以及形容詞的連用形的ウ音便。
* ‡ 「言ひ-て」　イ'''ヒ'''テ ihite → イ'''ウ'''テ iute → ユーテ yu-te 「言うて」
* ‡ 「早く」　ハヤ'''ク''' hayaku → ハヤ'''ウ''' hayau → ハヨー hayo- 「早う」
* ‡ 「久しく」　ヒサシ'''ク''' hisashiku → ヒサシ'''ウ''' hisashiu → ヒサシュー hisashu-「久しゅう」
另外和上述的「言うて」同樣現象也發生在標準語的[[わ行|ワ行]][[五段動詞|五段動詞]]的很小一部分之中。[[わ行|ワ行]][[五段動詞|五段動詞]]的テ・タ形在標準語中大半會發生促音便，但是其中的「問う」「請う」可以說基本不會發生促音便（「*問って」「*請って」）。
* 「問ひ-て」　ト'''ヒ'''テ tohite → ト'''ウ'''テ toute → トーテ to-te 「問うて」
* 「請ひ-て」　コ'''ヒ'''テ kohite → コ'''ウ'''テ koute → コーテ ko-te「請うて」
* 「厭ひ-て」　イト'''ヒ'''テ itohite → イト'''ウ'''テ itoute → イトーテ ito-te「厭うて」 
另外，像「言ふ　イ'''フ ihu''' → イ'''ウ iu'''」這樣的「フ hu→ ウ u」的變化通常不稱為ウ音便。 這是因為這個現象可以用[[ハ行轉呼|ハ行転呼]]這種更簡單的原理來說明。

=== 撥音便 ===
變成「ン」音的音便稱為'''撥音便'''。撥音便不會發生在詞尾，之後必須跟著其他音。

另外，「[[ん|ん]]」字的發明和普及是近世以後的事情，所以需注意在此之前的文獻中出現的撥音便會被寫作「[[む|む]]」(mu)甚至被無視。 例如「ひむかし（ヒンガシ）」 「ふむた（フンダ）」 「はへなり（ハベンナリ）」 「さへきなめり（サンベキナンメリ）」等等。

==== 濁音型的撥音便 ====
* 「香（か）-美（くは）し」　 カ'''グハ'''シ kaguhashi → カ'''ンバ'''シ kanbashi → カンバシイ kanbashii「芳しい」<ref group="注">「かぐわしい」 「香ばしい」 「芳しい」 是三重語。</ref>
* 「日-向か-し」　ヒ'''ムカ'''シ himukashi → ヒ'''ンガ'''シ hingashi → ヒガシ higashi「東」
* 「髪-挿し」　カ'''ミサ'''シ kamisashi → カ'''ンザ'''シ kanzashi 「簪」
* 「商人」 アキ'''ビト''' akibito → アキ'''ンド akindo'''
* 「文-板」　フミイタ humiita → フ'''ミタ''' humita → フ'''ンダ''' hunda → フダ huda「札」
* 「踏み-つける」　フ'''ミツ'''ケル humitsukeru → フ'''ンヅ'''ケル hunzukeru「踏んづける」
* 「仮-名」　カ'''リナ''' karina → カ'''ンナ kanna''' → カナ kana
テ・タ形中、マ行、バ行的[[五段動詞|五段動詞]]（[[四段活用|四段動詞]]）和[[ナ行|ナ行]][[五段動詞|五段動詞]]（[[ナ行活用|ナ行變格活用動詞]]）會產生撥音便。
* 「読み-て」　ヨ'''ミテ yomite''' → ヨ'''ンデ yonde''' 「読んで」
* 「呼び-て」　ヨ'''ビテ''' yobite → ヨ'''ンデ yonde''' 「呼んで」
* 「死に-て」　シ'''ニテ''' shinite → シ'''ンデ shinde''' 「死んで」
另外古代還有「†従ひ-て shitagahite → シタガ'''ンデ shitagande'''」這樣的[[は行|ハ行]][[四段活用|四段動詞]]的撥音便。

==== 歷史上的[[ら行|ラ行]]的撥音便 ====
歷史上[[ラ行變格活用|ラ行變格活用動詞]]「あり」「はべり」或者由「あり」變來的形容詞、[[形容動詞|形容動詞]]的[[カリ活用|カリ活用]]・[[ナリ活用|ナリ活用]]之後跟隨著助動詞的「めり」「なり」「べし」情況下，也會發生撥音便。
* 「†侍るhaberu-なりnari → ハベ'''ン'''ナリhabennari」
* 「†盛りsakari-なりnari → サカ'''ン'''ナリsakannari」
* 「†しかるshikaru-なりnari → シカ'''ン'''ナリshikannari」
* 「†さるsaru-べきbeki-なるnaru-めりmeri → サ'''ン'''ベキナ'''ン'''メリ sanbekinanmeri」等。
另外，也有 「†終はり wohari-ぬる nuru → ヲハ'''ン'''ヌル wohannuru」 「†去りsari-ぬ nu→ サ'''ン'''ヌ sannu」這種「[[四段活用|四段動詞]]＋ぬ nu」的ラ行的撥音便。

=== イ音便 ===
變成「イ」音的音便稱為'''イ音便'''。

標準語中產生於'''[[カ行|カ行]]'''、'''[[ガ行|ガ行]]'''（也就是「'''[[キ|キ]]'''」ki「'''[[ギ|ギ]]'''」gi）。 ガ行的情況下會伴隨後續音的濁音化。
* 「衝き-立て」　ツ'''キ'''タテ tsukitate → ツ'''イ'''タテ tsuitate「ついたて」
* 「月-立ち」　ツ'''キ'''タチ tukitachi → ツ'''イ'''タチ tuitachi 「朔日」
* 「埼玉」　サ'''キ'''タマ sakitama → サ'''イ'''タマ saitama
除了[[五段動詞|五段動詞]]（[[四段活用|四段動詞]]）的テ・タ形，這種音便也出現於[[形容词|形容詞]]的[[連體形|連體形]]（古語的[[連體形|連體形]]變成了現代語的[[終止形|終止形]]）。
* 「咲き-て」　サ'''キ'''テ sakite → サ'''イ'''テ saite 「咲いて」
* 「急ぎ-て」　イソ'''ギテ''' isogite → イソ'''イデ''' isoide「急いで」
* 「高き」　タカ'''キ''' takaki → タカ'''イ takai'''
* 「久しき」　ヒサシ'''キ''' hisashiki→ ヒサシ'''イ hisashii'''
方言中，中部地方以西的各地也有「‡ホカ'''シ'''テhokashite → ホカ'''イ'''テhokaite」這種、[[サ行|サ行]][[五段動詞|五段動詞]]的イ音便化的情況。另外在西日本各地，也有「‡セ'''ズテ sezute''' → セ'''ンデ''' sende → セ'''イデ seide'''」（しないで=不要）這種，作否定用的助動詞「ず」的連用形發生撥音便，再進一步發生イ音便化的情況。

=== 促音便 ===
'''[[い段|イ段]]的'''音變成「'''ッ'''」音（[[促音|促音]]）的音便稱為'''促音便'''。

促音便是產生於'''[[カ行|カ行]]、[[タ行|タ行]]、[[ラ行|ラ行]]、[[ハ行|ハ行]]'''音（也就是「'''[[キ|キ]]'''」ki 「'''[[チ|チ]]'''」chi 「'''[[リ|リ]]'''」ri 和 「'''[[ヒ|ヒ]]'''」（現代的「'''[[イ|イ]]'''」i））的現象，另外性質上它不能出現在詞尾，後面必須跟隨'''カ行'''、'''サ行'''、'''タ行'''、'''ハ行'''的其中一個音。

'''促音便可見於タ行、ラ行、ハ行'''的五段動詞（[[四段活用|四段動詞]]、[[ラ行變格活用|ラ行變格動詞]]）的テ・タ形。
* 「打ち-て」　ウ'''チ'''テ uchite → ウ'''ッ'''テ utte 「打って」
* 「言ひ-て」　イ'''ヒ'''テ ihite → イ'''ッ'''テ itte「言って」
* 「散り-て」　チ'''リ'''テ chirite → チ'''ッ'''テ chitte「散って」
* 「あり-て」　ア'''リ'''テ arite → ア'''ッ'''テ atte「あって」
另外，'''カ行'''の五段動詞（四段動詞）的テ・タ形通常都發生イ音便化，但也有「行く」這一個發生促音便的例外。
* 「行き-て」　イ'''キ'''テ ikite → イ'''ッ'''テ itte「行って」
與[[ラ行變格活用|ラ行變格動詞]]的例子「あって」(atte) 「あった」(atta)相類似地，[[形容詞|形容詞]]的[[過去式|過去式]]如「高かった」(taka-katta)的形式，其實是在來自「あり」(ari)的「[[カリ活用|カリ活用]]」之上，再發生te-ari變成tari（「タリ （< て-あり）」）然後發生促音便化，在此後[[終止形|終止形]]和[[連體形|連體形]]と合二為一，也就是終止形發生「高かったり」(taka-kattari)→「高かったる」(taka-kattaru)的變化，再進一步把詞尾的「る」(ru)脫落。 也就是說，發生了「タカカ'''リ'''タリ taka-karitari → タカカ'''ッ'''タリ taka-kattari → タカカッタル taka-kattaru →タカカッタ taka-katta」這樣的變化。
另外，「静かだった」(shizuka-datta)這樣的[[形容動詞|形容動詞]]的過去式也是類似的例子。 但是這種詞隨著時代發生了更進一步的「静かでshizukade＋あったatta」或者「静かshizuka＋だったdatta」這樣的再構成。也就是說，發生了（古語）静かなりきshizuka-nariki→静かにてありたりshizuka-nite-ari-tari→静かにてあったりshizuka-nite-attari→静かにてあったるshizuka-nite-attaru→静かにてあったshizuka-nite-atta→静かであったshizuka-deatta→静かぢゃったshizuka-jatta→静かだったshizuka-datta這樣的變化。

其他的典型例子還有[[动词|動詞]]＋[[动词|動詞]]的[[合成詞|合成詞]]。 這種類型的音便中可以看到「'''シ'''」發生促音便化的例子。多為中世以後出現。
* 「衝き-立てる」　ツ'''キ'''タテルtsukitateru → ツ'''ッ'''タテル tsuttateru「突っ立てる」
* 「掻き-攫ふ」　カ'''キ'''サラフ kakisarahu → カ'''ッ'''サラフ kassarahu → カッサラウ kassarau「かっさらう」
* 「とり-かへる」　ト'''リ'''カヘル torikaheru → ト'''ッ'''カヘル tokkaeru「とっかえる」
* 「追ひ-払ふ」　オ'''ヒ'''ハラフ ohiharahu → オ'''ッ'''パラフ opparahu → オッパラウ opparau「追っぱらう」
* 「差し-引く」　サ'''シ'''ヒク sashihiku → サ'''ッ'''ピク sappiku 「さっ引く」
