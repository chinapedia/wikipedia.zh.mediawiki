'''延安西路'''是[[中国|中国]][[上海市|上海市]]的一条交通干道，主要位于[[长宁区|长宁区]]，东西走向，东到[[华山路_(上海)|华山路]]，西到[[虹桥国际机场|虹桥国际机场]]。

== 历史 ==

与延安东路、延安中路在历史上曾为两租界界限不同，延安西路是公共租界控制的越界築路道路。1910年，[[上海公共租界工部局|上海公共租界工部局]]填柴兴浜筑长浜路，自忆定盘路（今[[江苏路|江苏路]]）迤东接静安寺南的长浜路（今[[延安中路_(上海)|延安中路]]），即今延安西路东段，同年又在公共租界以西修筑的[[上海公共租界越界筑路|越界筑路]]，1922年定名为大西路（Great Western Road），至虹桥路止。20世纪90年代，新辟虹桥路～沪青平公路段。

== 历史建筑 ==
延安西路54號 [[嘉道理爵士公馆|嘉道理爵士公馆]]（Kadoorie's House）今[[中国福利会少年宮|中国福利会少年宮]]

== 现况 ==
延安路已成为上海市内其中一条主要道路。

延安路上面则建设了双向6车道的延安高架路，连接市内高速公路。延安路高架西段由内环线高架起，西行经过虹桥及古北开发区，连接虹桥国际机场，全长6.2公里，于1995年-1996年建成，共有8个出入口，在虹桥机场亦建有专用出入口。

== 轨道交通 ==

[[上海轨道交通|上海轨道交通]][[上海轨道交通三号线|三号线]]、[[上海轨道交通四号线|四号线]]设有[[延安西路站|延安西路站]]；[[上海轨道交通二号线|二号线]][[上海轨道交通七号线|七号线]][[静安寺站|静安寺站]]在常德路延安西路口有出入口，在建[[上海轨道交通十四号线|十四号线]]站体下穿延安西(中)路。
