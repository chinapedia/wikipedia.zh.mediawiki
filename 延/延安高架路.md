'''延安高架路'''（{{lang-en|'''Yan'an Elevated Road'''}}<ref name="enName">{{Cite web |url=http://www.shgtj.gov.cn/zcfg/dmgl/201505/t20150511_653778.html |title=上海市道路名称英译导则 |accessdate=2018-09-16 |author=上海市地名办 |date=2015-04-30 |publisher=上海市规划和国土资源管理局 |archive-url=https://web.archive.org/web/20161113175158/http://www.shgtj.gov.cn/zcfg/dmgl/201505/t20150511_653778.html |archive-date=2016-11-13 |dead-url=no}}</ref>）是位于[[中华人民共和国|中国]][[上海市|上海市]]的一条东西向[[快速道路|高架快速路]]，是上海市“申”字型高架快速路网络中的“一横”。高架东西向横穿上海市中心城区，自虹桥路开始，由西向东基本沿[[延安西路_(上海)|延安西路]]、[[延安中路_(上海)|延安中路]]和[[延安东路_(上海)|延安东路]]修建。全线西起[[上海虹桥国际机场|上海虹桥国际机场]]，东到[[外滩|外滩中山东一路]]，西端与[[上海外环高速公路|外环沪青平立交]]相连，东端与[[延安东路隧道|延安东路隧道]]、[[外滩隧道|外滩隧道]]相接，全长公里约14.5公里。高架路幅宽56-71米，设双向6-8车道，设计时速60-80公里。

全线分三段建设完成，1996年率先建成的西段西起虹桥国际机场迎宾一路，东至[[内环高架路|内环高架路]]延安西路立交桥，全长5.88公里；1999年建成的中段西起内环高架路延安西路立交桥，东至[[南北高架路|南北高架路]]延安中路立交桥，全长5.56公里；1997年建成的东段西起南北高架路延安中路立交桥、东至外滩中山东一路，全长3.01公里。
== 历史 ==
=== 早期规划 ===
1948年上海市都市计划委员会编制的《上海市建成区干路系统计划》和1958年上海市城市规划勘测设计院编制的《上海市高速干道系统计划》首先提出了上海需要建设快速道路，其中1958年规划明确指出，为避免将车流导入中区、避免给市中心的建筑处理带来困难，不在延安路建设快速道路，因此80年代延安路一直未有建设快速道路的规划<ref name="planHistory">{{Cite book |url=http://www.wanfangdata.com.cn/local/toCatalog.do?gazetteers_id=FZ100798 |title=上海城市规划志 |last=《上海城市规划志》编纂委员会 |publisher=上海社会科学院出版社 |year=1999 |isbn=9787806185315 |location=上海 |pages=7480 |chapter=第六篇 城市道路交通规划}}</ref>{{Rp|333-335}}。

20世纪80年代，上海市区车流量增长迅速，1985年全市道路容纳量为机动车7.5万辆，而实有机动车15.5万辆<ref name="dissertation">{{Cite paper |title=延安高架道路——一份关于城市空间和人的活动的调查报告 |author=朱凯 |url=http://www.wanfangdata.com.cn/details/detail.do?_type=degree&id=Y697622 |accessdate=2019-01-13 |doi=10.7666/d.y697622 |year= 2005}}</ref>{{Rp|4}}。因此在1985年上海市规划院编制了《上海市中心城高架汽车专用道系统初步方案》，方案中规划建设设计时速60-80公里、宽4-6车道的快速道路，其中延安高架路作为“十字形骨架”的一部分提出，规划东西向由[[318国道|318国道]]进入市区接延安路至外滩，西段规划高架6车道，东段高架4-5车道<ref name="planHistory"/>{{Rp|335-336}}。该方案在1987年7月年获得了上海市城乡规划环境保护委员会<ref group="注">现为'''上海市规划和自然资源局'''</ref>的批复同意<ref>{{Cite web |url=http://www.shtong.gov.cn/Newsite/node2/node2245/node64620/node64624/index.html |title=上海城市规划志 大事记 |author=《上海城市规划志》编纂委员会 |accessdate=2019-01-13 |publisher=上海市地方志办公室 |archive-url=https://web.archive.org/web/20150926035305/http://www.shtong.gov.cn/Newsite/node2/node2245/node64620/node64624/index.html |archive-date=2015-09-26 |dead-url=no }}</ref>，确定上海要建设“申”字型高架网<ref>
{{Cite web |url=http://www.shsz.org.cn/view.asp?num=i-200852892233 |title=“申”字型高架路：一个大胆的创意 |author=上海市政公路行业协会 |accessdate=2019-01-13 |publisher=上海市政公路信息网 |archive-url=https://web.archive.org/web/20190105150821/http://www.shsz.org.cn/view.asp?num=i-200852892233 |archive-date=2019-01-05 |dead-url=no }}
</ref>，同时也纳入了1986版《上海市城市总体规划》<ref name="generalPlan1986">{{Cite web |url=http://www.gov.cn/zhengce/content/2012-07/06/content_6042.htm |title=国务院关于上海市城市总体规划方案的批复（国函〔1986〕145号） |accessdate=2019-01-13 |author=国务院 |date=2012-07-06 |publisher=国务院办公厅 |archive-url=https://web.archive.org/web/20170512171750/http://www.gov.cn/zhengce/content/2012-07/06/content_6042.htm |archive-date=2017-05-12 |dead-url=no }}</ref><ref name="news-generalPlan">{{Cite web |url=https://www.thepaper.cn/newsDetail_forward_2488041 |title=上海改革开放40年｜夏丽卿：上海两轮城市总体规划编制始末 |accessdate=2019-01-13 |author=谢黎萍, 郭继, 严亚南, 黄啸 |date=2018-09-29 |publisher=澎湃新闻}}</ref>。

=== 专营与收费 ===
根据1997年1月2日上海市政府发布的《上海市延安高架路专营管理办法》，自延安高架路东西两段开工起至2017年12月31日，沪港合作上海延安路高架道路发展有限公司对这两段道路实施专营，包括对延安西路与沪青平公路相交处收费口的车辆征收通行费，而延安高架路中段仍由上海市政运营<ref name="Regulation1">{{Cite web |url=http://www.shanghai.gov.cn/nw2/nw2314/nw2319/nw2407/nw26170/u26aw27322.html |title=上海市延安高架路专营管理办法 |accessdate=2018-09-16 |author=上海市人民政府 |date=1997-01-02}}</ref>。征收通行费的车辆包括社会车辆和一部分[[出租車|出租車]]<ref name="Regulation2">{{Cite web |url=http://news.makepolo.com/5084921.html|title=关于延安高架路西段道路收费标准的补充通知（沪价行[1996]第309号） |accessdate=2018-09-16 |author=上海市住建委, 上海市物价局 |date=1996-12-02}}</ref>，所征收通行费性质为城市道路贷款建设通行费，随着车流量的上升，通行费的收取影响到了延安高架路的畅通，经过研究协商<ref name="gov news">{{Cite web |url=http://www.shanghai.gov.cn/nw2/nw2314/nw9819/nw9820/u21aw95740.html |title=上海市政府新闻发布会实录（2004年11月10日） |accessdate=2018-09-16 |author=上海市政府新闻办 |date=2004-11-11 |publisher=上海市政府}}</ref>，收费最终于2005年1月31日提前取消<ref name="Regulation3">{{Cite web |url=http://www.shdrc.gov.cn/xxgk/cxxxgk/13833.htm |title=市建委、市物价局联合发文取消延安高架路西段道路收费 |accessdate=2018-09-16 |author=上海市住建委, 上海市物价局 |date=2005-02-02 |publisher=上海市发改委}}</ref>。

=== 交通管制 ===
由于交通流量巨大，为缓解道路交通拥堵，均衡高架和地面道路机动车流量，改善城市快速干道通行秩序，上海市公安局自2002年12月28日起对延安高架路实施了常态化的交通管理措施，包括在工作日的规定时间内禁止外省市号牌小客车、未载客的出租小客车及实习期驾驶员驾驶的小客车通行<ref name="Restriction1">
{{Cite web |url=http://news.sohu.com/48/63/news205716348.shtml |title=上上海高架与外环线部分道路实施交通管理 |accessdate=2018-09-16 |author=刘兴伟 倪国和 |date=2003-01-14 |publisher=文汇报 |archive-url=https://web.archive.org/web/20060216223754/http://news.sohu.com/48/63/news205716348.shtml |archive-date=2006-02-16 |dead-url=no }}
</ref>。实施常态化管制至今，上海市公安局多次调整管制方案，延长管制时间<ref name="Restriction2">{{Cite web |url=http://www.police.sh.cn/shga/wzXxfbGj/detail?pa=1b89c2c9c1e7a00e4fcbf784730c1c7f7455ef763f297bf20ab3bd6875da9b0efd02ced89da2a1c7bc0271ae75603a91f89cd8d0bb43e938 |title=上海市公安局关于调整本市部分高架道路（城市快速路）交通管理措施的通告 |accessdate=2018-09-16 |author=上海市公安局 |date=2015-04-02}}</ref>，时至今日管制时间已经延长至每工作日7时至10时、15时至20时<ref name="Restriction3">{{Cite web |url=http://www.police.sh.cn/shga/wzXxfbGj/detail?pa=110ef360e4374a4130343bc33901595301ea2fbff706971117fbb0b75602ccf6 |title=上海市公安局关于调整本市部分高架道路（城市快速路）交通管理措施的通告 |accessdate=2018-09-16 |author=上海市公安局 |date=2016-04-11}}</ref>。

=== 流量监控 ===
2003年10月20日，作为国家科技部智能交通系（ITS）示范工程的延安路高架道路交通监控系统试开通运行。监控系统通过对道路流量进行检测，并通过大中小三型可变情报板，向驾驶员提供及时的交通信息，诱导车辆在良好路况的道路上行驶。系统投用后，高架道路西向东主线的交通状况明显好转。在保持流量不变的情况下，道路服务水平有很大程度的提高，道路畅通时间明显增加<ref name="journal26">{{Cite journal |title=上海市延安路高架道路监控系统运行效果评价 |author=冯奇, 杜豫川, 孙立军, 袁文平 |url=http://www.wanfangdata.com.cn/details/detail.do?_type=perio&id=zgszgc200406024 |journal=中国市政工程 |issue=6 |doi=10.3969/j.issn.1004-4655.2004.06.024 |year=2004 |page=56-58,61}}</ref>
<ref name="journal27">{{Cite journal |title=城市交通安防应用专题——大城市ITS成功应用案例——上海延安高架道路典范工程 |author=卓长熹 |url=http://www.cqvip.com/Main/Detail.aspx?id=15770818 |journal=A&S:安防工程商 |issue=6 |year=2005 |page=101-105}}</ref>。

== 环境影响 ==
=== 上海音乐厅平移 ===
[[File:Shanghai_Concert_Hall_-_OpenStreetMap.jpg|150px]]
原上海音乐厅坐落在原延安中路龙门路口，建于1930年，是全国第一座音乐厅、上海滩的著名建筑。随着延安路高架路东段建成，西向东的西藏路匝道处成了车辆聚集地，前往上海音乐厅的观众不能在门口停车，必须绕道步行进入音乐厅。此外，高架道路往来的车辆震动也对老建筑极为不利，行车噪音也影响了音乐厅的正常演出效果。2002年，上海市政府决定出资6000万元，对上海音乐厅进行保护性迁移和功能完善性修缮。音乐厅于2002年9月1日歇业，并于12月16日开始开工，次年4月15日正式顶升<ref>
{{Cite web |url=http://www.shanghai.gov.cn/nw2/nw2314/nw2315/nw5827/u21aw16331.html |title=上海音乐厅平移工程开工 |accessdate=2018-09-16 |date=2003-04-16 |publisher=上海市政府 |archive-url=https://web.archive.org/web/20190125091953/http://www.shanghai.gov.cn/nw2/nw2314/nw2315/nw5827/u21aw16331.html |archive-date=2019-01-25 |dead-url=no }}
</ref>。最终于2003年6月17日，音乐厅向东南方向移动了66.46米，转向180度，并抬高3.38米<ref name="journal23">{{Cite journal |title=阿卡多 上海音乐厅 延安路高架 |author=姜开城 |url=http://www.wanfangdata.com.cn/details/detail.do?_type=perio&id=csdqyfh201109111 |journal=城市道桥与防洪 |issue=9 |doi=10.3969/j.issn.1009-7716.2011.09.111 |year=2011 |page=282-283}}</ref>。

=== 土壤铅沉积 ===
[[File:Crossroads_of_South-North_Elevated_Road_and_Yanan_Elevated_Road.jpg|right]]
2007年的研究表明，延安高架高峰时段平均车流量为每小时8965辆，延中立交高峰流量可达每小时16000辆，在高架所经区域内主要为商业区、住宅区和公园，无大型工业区的情况下，延安高架道路绿地土壤中[[铅|铅]]是上海市土壤背景值的3.7倍，沿线灰尘中铅是土壤铅的3-4倍。沿线绿地土壤中铅在由西到东的3个路段依次增多，并且3个路段铅的平均质量浓度均超过了上海市土壤背景值，分别为土壤背景值的2.1、3.7、4.7 倍；而同地同步采集的灰尘样品铅在分路段上也有同样的规律，3个路段灰尘铅的平均质量浓度分别是上海市土壤背景值的9.4、13.4、14.5倍。研究充分表明，延安高架路的造成了沿线土壤的铅累积，且累积量与车流量[[相关|正相关]]<ref name="journal25">{{Cite journal |title=上海市延安高架道路绿地土壤与沿线灰尘中铅的分布特征 |author=王利, 陈振楼, 许世远, 史贵涛, 李海雯 |url=http://www.wanfangdata.com.cn/details/detail.do?_type=perio&id=hjwryfz200702013 |journal=环境污染与防治 |issue=2 |doi=10.3969/j.issn.1001-3865.2007.02.013 |year=2007 |volume=29 |page=132-133,137}}</ref>。

=== 沿线绿化 ===
为从一定程度上缓解高架路带来的噪声、废气和粉尘污染，延安高架路布置了一定的桥柱绿化。延安高架桥柱绿化建设东起江西中路、西至外环沪青平公路的高架桥桥柱群，全线桥柱绿化的平均高度为7.6米。选择了[[常春藤|常春藤]]+[[五叶地锦|五叶地锦]]、[[络石|白花络石]]+五叶地锦等作为常用组合，同时选择爬山虎+常春藤、凌霄+常春藤等作为备选组合，通过各种植物组合以达到在各种不同环境下都有绿化景观的目的。此外还选择了黄金菊、花叶蔓长春、兰花三七等花卉以达到观赏目的。通过绿植的搭配种植，延安高架桥柱绿化工程实现了良好的景观效果和生态效应<ref name="journal19">{{Cite journal |title=延安路高架桥柱绿化的实践与探索 |author=王瑛, 王延洋, 陈嫣嫣, 李向茂 |url=http://www.wanfangdata.com.cn/details/detail.do?_type=perio&id=shjskj201703018 |journal=上海建设科技 |issue=3 |doi=10.3969/j.issn.1005-6637.2017.03.018 |year=2017 |page=66-58}}</ref>。

== 相关文化 ==
=== 龙柱传说 ===
[[File:Shanghai,_China_(Unsplash).jpg|right]]
延安高架路和南北高架路在延安中路立交桥的交汇处，由一根巨大的圆形柱子支撑，墩柱上包裹着白钢并有用黄铜锻造的祥龙腾云图案，该墩柱直径5米、高度32米、桩基长度62米的，是上海高架道路中最粗的一根墩柱，工程名为PM109号墩<ref name="fame1">
{{cite web|title=延安路"龙柱"只是一个传说 桩打不进去和杜月笙公馆有关|url=http://news.ifeng.com/gundong/detail_2012_02/21/12657654_0.shtml|date=2012-02-21|access-date=2018-09-20|archive-url=https://web.archive.org/web/20180331173525/http://news.ifeng.com/gundong/detail_2012_02/21/12657654_0.shtml|archive-date=2018-03-31|dead-url=no}}
</ref>。

该墩柱在建造时，打了10个月的桩，却始终未能下桩。当时有传闻，打桩动到了风水，于是请来[[玉佛寺_(上海)|玉佛寺]]时任方丈[[释真禅|真禅法师]]。真禅法师带着弟子连续念了三天经文，并主持了祭祀仪式，后来的施工果然无比顺利。真禅法师提醒说，上海的地下有一条龙，这一根桩正好打在了龙背上，所以建造完成后，必须在七天在柱子周围包覆银底金龙的图案。由于道破的天机，真禅法师在多日后无疾而圆寂。 虽然上海政府官方声明未请高僧念经祭祀，但是这个故事在上海本地人里流传甚广<ref name="fame2">
{{cite web|title=上海延安路高架“龙柱”之谜|url=http://sh.eastday.com/m/20161229/u1a12574390.html|date=2016-12-29|access-date=2018-09-20|archive-url=https://web.archive.org/web/20170101105010/http://sh.eastday.com/m/20161229/u1a12574390.html|archive-date=2017-01-01|dead-url=no}}
</ref>。

而事后通过采访上海市市政工程建设处的黄迎东、姜开城得知，PM109号墩难以下桩的原因有两个：一是墩柱所在位置原是[[杜月笙公馆|杜月笙公馆]]位置，地下先前已曾打过杨木桩，地下条件比较复杂；二是工程要求边长14.5米的八角形区域布置28根桩，布桩密度太高，在打桩过程中土体相互挤压，加大了桩体摩擦系数，增加了打桩的难度<ref name="journal22">
{{Cite journal |title=细解上海南北高架“龙柱”之迷 |author=姜开城 |url=http://www.cqvip.com/QK/97706X/201103/37266492.html |journal=城市道桥与防洪 |issue=3 |year=2011 |access-date=2019-02-25 |archive-url=https://web.archive.org/web/20121025041607/http://www.cqvip.com/QK/97706X/201103/37266492.html |archive-date=2012-10-25 |dead-url=no }}
</ref>。墩柱完工后，时任上海市市政工程建设建设处总工程师李有成认为灰暗的混凝土中心立柱与高架立交不相匹配，同时也会带来视错觉影响行车安全，于是请[[上海油画雕塑院|上海油画雕塑院]]的赵志荣先生协助<ref name="fame1"/>。最终赵志荣为墩柱设计了腾龙的图案，并将墩柱雕塑命名为《龙腾万里 日月同辉》<ref name="fame2"/>。
