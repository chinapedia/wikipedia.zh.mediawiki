'''氧化銅'''（化学式：CuO）是[[铜|铜]]的[[氧化物|氧化物]]，为黑色固体。不溶于[[水|水]]和[[乙醇|乙醇]]，溶于[[酸|酸]]、[[氯化铵|氯化铵]]及[[氰化钾|氰化钾]]溶液，[[氨|氨]]溶液中缓慢溶解。

==制备==
氧化铜可由[[氢氧化铜|氢氧化铜]]的热分解制得：<ref>北京师范大学 等. 无机化学（下册）第四版. 高等教育出版社, 2003. pp 706</ref>
: Cu(OH)<sub>2</sub> —<u><sup>353-363 K</sup></u>→ CuO + H<sub>2</sub>O

==化学性质==
氧化铜在一定温度下，可以被一些具有还原性的气体还原，如[[氢气|氢气]]、[[碳|碳]]、[[一氧化碳|一氧化碳]]、[[氨|氨]]等：
: CuO + CO → Cu + CO<sub>2</sub>
: C + 2 CuO → CO<sub>2</sub> + 2 Cu 
: CuO + H<sub>2</sub> → Cu + H<sub>2</sub>O
氧化铜和铝的混合物在灼热时会发生爆炸：
: 3 CuO + 2 Al → 2Al<sub>2</sub>O<sub>3</sub> + 3 Cu
氧化铜可溶于酸，形成铜盐，如：
: CuO + H<sub>2</sub>SO<sub>4</sub> → CuSO<sub>4</sub> + H<sub>2</sub>O
: CuO + 2 HClO<sub>4</sub> → Cu(ClO<sub>4</sub>)<sub>2</sub> + H<sub>2</sub>O
