'''氧化銣'''（'''Rb<sub>2</sub>O'''），是[[銣|銣]]的[[氧化物|氧化物]]之一，呈黃色，有很强的潮解性。

== 製備 ==

铷在空气中燃烧时，主要生成的是[[过氧化铷|过氧化铷]]，只有少量的氧化铷和[[超氧化铷|超氧化铷]]生成。当金属铷被露置于空气中时，它会很快氧化，失去金属光泽，并产生一系列有颜色的氧化产物。其中生成了铷的[[低氧化物|低氧化物]]，例如青铜色的Rb<sub>6</sub>O和红棕色的Rb<sub>9</sub>O<sub>2</sub>。铷最终的氧化产物主要是超氧化铷。

氧化銣主要由用铷还原无水[[硝酸銣|硝酸銣]]、[[氫氧化銣|氫氧化銣]]或[[超氧化銣|超氧化銣]]產生：

: 10 Rb + 2 RbNO<sub>3</sub> → 6 Rb<sub>2</sub>O + N<sub>2</sub> ↑

: 2 Rb + 2 RbOH → 2 Rb<sub>2</sub>O + H<sub>2</sub> ↑

: 3 Rb + RbO<sub>2</sub> → 2 Rb<sub>2</sub>O

== 相關化學反應 ==

氧化銣溶於水後，会剧烈反应生成[[氫氧化銣|氫氧化銣]]：

: Rb<sub>2</sub>O + H<sub>2</sub>O → 2 RbOH

將[[氫|氫氣]]通入氧化銣後，生成氫氧化銣及[[氫化銣|氫化銣]]：

: Rb<sub>2</sub>O + H<sub>2</sub> → RbOH + RbH
