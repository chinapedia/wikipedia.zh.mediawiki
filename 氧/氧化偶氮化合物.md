'''氧化偶氮化合物'''（英文：Azoxy）是[[偶氮化合物|偶氮化合物]]的[[N-氧化物|N-氧化物]]，通式写作RN=N<sup>+</sup>(-O<sup>−</sup>)R或R-N=N(→O)-R'。相应的[[官能团|官能团]]称为'''氧化偶氮基'''。

氧化偶氮基具有以下[[共振式|共振式]]，因此它既是[[1,2-偶极体|1,2-偶极体]]，也是[[1,3-偶极体|1,3-偶极体]]：

:R-N=N<sup>+</sup>(-O<sup>−</sup>)-R' ←→ R-N<sup>+</sup>-N(-O<sup>−</sup>)-R'

它们可能具有[[基因毒性|基因毒性]]。

==合成==
[[亚硝基化合物|亚硝基化合物]]和[[羟胺|羟胺]]衍生物失水[[缩合|缩合]]可以得到氧化偶氮化合物：

:R-NHOH  +  R'-N=O  →  R-N=N<sup>+</sup>(-O<sup>−</sup>)-R'  +  H<sub>2</sub>O

用[[亚砷酸钠|亚砷酸钠]]或其他还原剂还原[[硝基苯|硝基苯]]，会生成中间产物[[苯基羟胺|苯基羟胺]]和[[亚硝基苯|亚硝基苯]]。它们相互作用，也会得到[[氧化偶氮苯|氧化偶氮苯]]：<ref>Gilman, H.; Harwood, H. J. ''Organic Syntheses'', Coll. Vol. 2, p.57 (1943); Vol. 11, p.16 (1931). [http://www.orgsyn.org/orgsyn/prep.asp?prep=cv2p0057 链接]</ref>

:2 Ph-NO<sub>2</sub>  +  As<sub>2</sub>O<sub>3</sub>  +  NaOH  →   Ph-N=N<sup>+</sup>(-O<sup>−</sup>)-Ph

以氧化偶氮苯为例，它还可通过[[偶氮苯|偶氮苯]]的[[三氧化铬|三氧化铬]]氧化，[[苯胺|苯胺]]的[[过氧化氢|过氧化氢]]氧化，以及beta-苯基羟胺的[[高锰酸钾|高锰酸钾]]等途径制备。

==反应==
氧化偶氮化合物具有氧化性。用[[锌|锌]]/[[盐酸|盐酸]]作还原剂时，得到两个[[胺|胺]]；用[[三苯基膦|三苯基膦]]作还原剂时，得到偶氮化合物：
:Ar-N=N<sup>+</sup>(-O<sup>−</sup>)-Ar  +  Ph<sub>3</sub>P  →  Ar-N=N-Ar  +  [[三苯基氧膦|Ph<sub>3</sub>P=O]]

:Ar-N=N<sup>+</sup>(-O<sup>−</sup>)-Ar  +  Zn/HCl  →  2 Ar-NH<sub>2</sub>

氧化偶氮化合物的N=N-O部分是[[1,3-偶极体|1,3-偶极体]]，可以发生[[环加成反应|环加成反应]]。芳香族的氧化偶氮化合物在强酸作用下，氧原子发生重排，得到对羟基偶氮化合物：
