'''木全忠澄'''（1534年—1610年12月3日）是[[日本戰國時代|日本戰國時代]]至[[江戶時代|江戶時代]]初期的武將。父親是[[木全征詮|木全征詮]]。

== 出身 ==
本姓{{tsl|ja|紀氏}}（「[[瀧川忠征|瀧川忠征]]記念碑」（[[愛知縣|愛知縣]][[稻澤市|稻澤市]][[智元寺|智元寺]]））。先祖在[[壬申之亂|壬申之亂]]以前，從[[紀伊國|紀伊國]]移居至[[美濃國|美濃國]]，後來與家臣一同開拓[[尾張國|尾張國]]{{tsl|ja|中島郡}}的[[茜部鄉|茜部鄉]]。雖然在附近領有廣大土地，不過在[[保元之亂|保元之亂]]、[[平治之亂|平治之亂]]中，失去許多領地，更在[[建久|建久]]年間，被[[守護|守護]][[野上成經|野上成經]]奪去所有領地，於是隱居在[[木全村|木全村]]（現今愛知縣稻澤市木全、稻島），並改氏木全。在[[室町時代|室町時代]]，仕於[[室町幕府|室町幕府]]，並成為[[莊長|莊長]]，有[[木全善鎗|木全善鎗]]和[[木全一角|木全一角]]。

根據[[寶曆|寶曆]]年間編纂的『[[張州府志|張州府志]]』中記載，家祖是木全善鎗。忠征建立的[[大林寺|大林寺]]（[[名古屋市|名古屋市]][[千種區|千種區]]）中的「瀧川忠征墓誌」上，亦以紀氏為木全氏的本姓。

== 生平 ==
在[[天文_(後奈良天皇)|天文]]3年（1534年）於[[尾張國|尾張國]]出生。最初仕於[[織田氏|織田氏]]的家臣[[淺井政貞|淺井政貞]]，後來成為[[瀧川一益|瀧川一益]]的[[家老|家老]]（『[[寬政重修諸家譜|寬政重修諸家譜]]』、『[[尾張群書系圖部集|尾張群書系圖部集]]』）。智謀優秀，擅長槍術，被稱為「'''木全之槍'''」（{{lang|ja|木全の槍}}）而廣為人知。

特別是在小山的合戰中，以智謀擊破人數多於己方十多倍的敵軍。在[[織田信長|織田信長]]時期，[[美濃國|美濃國]]的國境出現[[一揆|一揆]]，一揆軍千餘人和鄉士2、3百人侵入尾張；對此，率領將1百人部下，藏於[[松樹|松樹]]茂密的小山，完全不動；見到如此狀況的人就對此責難，「請木全大人前往迎敵，到底守在山中是什麼意思」（{{lang|ja|いかに木全殿、敵に遭いて、山に篭られ候は何の意ぞ}}）「不救援己方，只有一分留心嗎」（{{lang|ja|味方を救わずして只一分の用心か}}）；此時，以冷靜的表情回應「如我等人，在此間布陣」（{{lang|ja|我ら如きものは中辺に構え候}}），並命令「此處是一揆或勝或負都必然會經過的通道。專心藏匿吧」（{{lang|ja|ここは一揆の者が勝っても負けても必ず通る道である。じっと隠れておれ}}）；戰勝的敵軍果然緩緩接近小山，讓其半數軍勢通過後，突然令自軍大叫，因為山中的反響，令人以為有許多兵力，一揆軍大驚；此時，將1百人分成3個部隊，令其中1隊部隊從松樹林中突擊，判斷木全軍只有少數兵力的敵軍停止逃走並返回迎戰，另1隊部隊立即大叫並突進，餘下的1個部隊亦用這個方式突擊，令敵軍失去戰意，多數人戰死，殘兵亦沒有見到後方並敗走。（『[[尾張志|尾張志]]』、『[[武話碎玉|武話碎玉]]』）

此後，仕於[[豐臣秀吉|豐臣秀吉]]，加入[[小姓|小姓]]組，負責謀略方面，以近侍身份出仕。在[[慶長|慶長]]15年（1610年）死去。

== 人物 ==
*在[[幕末|幕末]]編纂的『[[尾張名所圖會|尾張名所圖會]]』中記載，「木全以謀略殺死許多大意的一揆，得到世間讚賞」（{{lang|ja|木全、謀をめぐらし不意をうちて一揆を多く討ち取り、世の賞賛にあずかりしよし}}）。

== 子孫 ==
長男[[瀧川忠征|忠征]]仕於[[瀧川一益|瀧川一益]]，被賜予家名，改為{{tsl|ja|滝川氏|瀧川氏}}，之後仕於[[德川家康|德川家康]]，仕於家康的兒子兼[[尾張藩|尾張藩]]初代藩主[[德川義直|德川義直]]，成為[[家老|家老]]，領有6千石，存續至[[明治時代|明治時代]]為止。

其他兒子有[[木全河內守|木全河內守]]、[[木全兵部|木全兵部]]、[[友田新右衛門|友田新右衛門]]。根據尾張藩的『[[藩士名寄|藩士名寄]]』中記載，在[[幕末|幕末]]時期，尾張藩的{{tsl|ja|作事奉行}}和[[勘定奉行|勘定奉行]]有木全姓的[[藩士|藩士]]，明治以後，著名人物有[[陸軍|陸軍]][[少將|少將]][[木全良雄|木全良雄]]、[[銀行家|銀行家]][[篠田角太郎|篠田角太郎]]等。
