'''木次線'''（{{jpn|j=木次線|hg=きすきせん|rm=Kisuki sen|f=y}}）是一條連結[[日本|日本]][[島根縣|島根縣]][[松江市|松江市]]的{{link-ja|宍道站|宍道駅}}至[[廣島縣|廣島縣]][[庄原市|庄原市]]的{{link-ja|備後落合站|備後落合駅}}，屬於[[西日本旅客鐵道|西日本旅客鐵道]]（JR西日本）的鐵路線（[[地方交通線|地方交通線]]）。
== 概要 ==
{{link-ja|西日本旅客鐵道米子支社|西日本旅客鉄道米子支社|米子支社}}管轄宍道站，{{link-ja|西日本旅客鐵道岡山支社|西日本旅客鉄道岡山支社|岡山支社}}管轄備後落合站，其餘車站由米子支社的{{link-ja|木次鐵道部|木次鉄道部}}管轄。
== 路線資料 ==
* 管轄（事業類別）：西日本旅客鐵道（[[鐵路公司#第一種鐵道事業者|第一種鐵道事業者]]）
* 路線距離（{{link-ja|營業距離|営業キロ}}）：81.9公里
* [[軌距|軌距]]：1067毫米
* 站數：18個（包括起終點站）
** 若只限屬於木次線的車站，排除屬於山陰本線的宍道站與屬於藝備線的備後落合站<ref>『停車場変遷大事典 国鉄・JR編』[[ジェイティービー|JTB]]、1998年。ISBN 978-4-533-02980-6。</ref>則為16個<ref>[http://www.westjr.co.jp/railroad/digest/ 鉄道事業ダイジェスト] {{webarchive|url=https://web.archive.org/web/20140314194736/http://www.westjr.co.jp/railroad/digest/ |date=2014年3月14日 }} - 西日本旅客鉄道</ref>。
* 複線路段：沒有（全線[[單線鐵路|單線]]）
** 可[[列車交會|列車交會]]車站：5個（加茂中站、木次站、出雲三成站、出雲橫田站、出雲坂根站）

* 最高速度：

* {{link-ja|輸送密度|輸送密度|平均通過人員}}

== 歷史 ==
1916年（[[大正|大正]]5年），'''簸上鉄道'''宍道車站 - 木次車站間開業。1932年（[[昭和|昭和]]7年），[[日本國有鐵道|國鐵]]'''木次線''' 木次車站 - 出雲三成車站間開業。1934年（昭和9年），簸上鐵道國有化，宍道車站 - 出雲三成車站間成為木次線<ref>[http://dl.ndl.go.jp/info:ndljp/pid/2958748/3 「鉄道省告示第337号」『官報』1934年7月28日]（国立国会図書館デジタルコレクション）</ref>。1987年（昭和62年），[[國鐵分割民營化|國鐵分割民營化]]，由西日本旅客鐵道繼承。
