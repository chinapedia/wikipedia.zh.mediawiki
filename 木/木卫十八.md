'''木卫十八'''又稱為「德彌斯托」（S/1975 J 1, S/2000 J 1, Themisto），是环绕[[木星|木星]]运行的一颗[[卫星|卫星]]，它發現於1975年，之後就失蹤了，直到2000年才再度被發現。

== 發現和命名 ==
木衛十八是在1975年被[[查爾斯·科瓦爾|查爾斯·科瓦爾]]和[[伊麗莎白·羅默爾|伊麗莎白·羅默爾]]發現，予以[[臨時編號|臨時編號]]'''S/1975 J 1'''，由於對其軌道特性不清楚，因此這顆衛星隨後便失蹤了。

木衛十八曾出現在1980年代[[天文|天文]][[書籍|書籍]]的注解裡。到2000年才被[[斯科特·謝帕德|斯科特·謝帕德]]、[[大衛·朱維特|大衛·朱維特]]、[[Yanga_R._Fernández|Yanga R. Fernández]]、[[Eugene_A._Magnier|Eugene A. Magnier]]等人發現，剛開始他們以為是一個新[[天體|天體]]所以給一個'''S/2000 J 1'''的臨時編號，但之後證明跟1975年失蹤的[[衛星|衛星]]是同一個衛星。

== 軌道 ==
木衛十八的軌道很特殊，介於[[希瑪利亞群|希瑪利亞群]]和[[伽利略衛星|伽利略衛星]]之間，它既不是逆行衛星，在距離上又沒有鄰近的衛星，因此在分類上自成一組。
