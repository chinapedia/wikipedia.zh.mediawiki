'''木村毅一'''（{{bd|1904年|4月7日|1992年|7月8日||catIdx=Kimura木村}}），是[[日本|日本]]的[[高能物理學|高能物理學]]家。1930年，由於日本[[經濟不景氣|經濟不景氣]]，且當時臺灣物價相對便宜，故應[[荒勝文策|荒勝文策]]教授的邀請抵台擔任其在[[台北帝国大学|台北帝国大学]]（現在的[[国立臺湾大学|国立臺湾大学]]）[[國立臺灣大學物理文物廳|物理學講座]]的助手<ref name = "張幸真">{{cite journal zh|author=張幸真|title=台灣知識社群的轉變－以台北帝國大學物理講座到台灣大學物理系為例|url=http://ndltd.ncl.edu.tw/cgi-bin/gs32/gsweb.cgi?o=dnclcdr&s=id=%22091NTU00493013%22.&searchmode=basic|accessdate=2012-06-11|language=zh-hant|date=2003-07-31}}</ref>。之後在1934年7月25日於該處與[[荒勝文策|荒勝文策]]教授共同建造[[粒子加速器#直線加速器|直線粒子加速器]]，成功在[[臺灣|臺灣]]做出[[亞洲|亞洲]]第一次人工撞擊[[原子核|原子核]]實驗<ref name="實驗日期">{{cite journal

他在1951年[[美國|美國]]解除自[[二戰|二戰]]以來對日本[[核物理|核物理]]實驗的禁止後，接手重啟的[[京都大學|京都大學]]核子物理研究室<ref name = "鄭伯昆">{{cite journal zh|author=鄭伯昆|title=台大核子物理實驗室 (四) 有關的日本科學家|pages=574~580|id= 物理雙月刊(卅卷五期)|url=http://psroc.phys.ntu.edu.tw/bimonth/download.php?d=1&cpid=165&did=14
|accessdate=2014-11-07|language=zh-hant|date=2008年10月}}</ref><ref name="木村">{{cite web|url=http://www.rri.kyoto-u.ac.jp/JRS/dayori/D-18.pdf|title=故 木村毅一先生追悼記念特集号|publisher=原子炉實驗所だより|accessdate=2014-11-28|language=ja}}</ref>。參與成立{{link-ja|大阪府立放射線中央研究所|大阪府立放射線中央研究所}}<ref group = "注">1959年成立</ref>與{{link-ja|京都大学原子爐實驗所|京都大学原子炉実験所}}<ref group = "注">1963年成立</ref>，並對[[日本|日本]]的[[原子核|原子核]]與[[放射線|放射線]]研究領域之創立有所貢獻。

木村毅一曾任京都大学教授、京都大学原子爐實驗所所長、大阪府立放射線中央研究所所長、以及[[福井工業高等專門学校|福井工業高等專門学校]]校長。此外，他是京都大学名譽教授，並曾獲勲二等[[旭日重光章|旭日重光章]]<ref name = "atom">{{cite book|author=木村毅一|url=http://www.amazon.co.jp/%E3%82%A2%E3%83%88%E3%83%A0%E3%81%AE%E3%81%B2%E3%81%A8%E3%82%8A%E3%81%94%E3%81%A8-1982%E5%B9%B4-%E6%9C%A8%E6%9D%91-%E6%AF%85%E4%B8%80/dp/B000J4Q2D6|title=アトムのひとりごと |year=1982|publisher=丸善|location=日本|date=1982-11|language=ja}}</ref>。

== 年譜 ==
[[File:NTU_PhysicsOldBuilding.JPG|250px]]物理學講座校舍，即臺大二號館。內有[[臺大物理文物廳|臺大物理文物廳]]，即為當年木村毅一與荒勝文策建造加速器處。]]
[[File:臺大物理文物廳展覽廳.jpg|250px]]的考克饒夫－瓦耳頓型[[粒子加速器#直線加速器|直線粒子加速器]]]]
* 1904年 - 在日本[[京都府|京都府]]誕生。
* 1926年 - 自[[第三高等学校_(旧制)|第三高等学校]]畢業。
* 1929年 - 自[[京都帝国大学|京都帝国大学]]理学部物理学系畢業。
* 1930年 - 擔任[[台北帝国大学|台北帝国大学]][[國立臺灣大學物理學系|物理學講座]]助手。
* 1934年 - 7月25日，協助[[荒勝文策|荒勝文策]]在台北帝大二號館101室<ref group = "注">今[[國立臺灣大學物理文物廳|臺大物理文物廳]]</ref>建造{{le|考克饶夫-瓦尔顿加速器|Cockcroft–Walton generator}}<ref name="木村"/><ref>{{cite web|url=http://dls.ym.edu.tw/sa/dls/upload/2-3/leg/week4.pdf |title=二次大戰對科學地位的影響 |accessdate=2014-12-03 |language=zh-hant |deadurl=yes |archiveurl=https://web.archive.org/web/20150701061601/http://dls.ym.edu.tw/sa/dls/upload/2-3/leg/week4.pdf |archivedate=2015-07-01 }}</ref>，成功完成人工撞擊[[原子核|原子核]]實驗 <ref name="實驗日期" />。此為亞洲第一次，世界第二次成功實驗。
* 1936年 - 擔任京都帝国大学理学部講師。
* 1939年 - 升任京都帝国大学助教授（理学部勤務）。
* 1940年 - 獲得[[理学博士|理学博士]]學位。
* 1941年 - 補任京都帝国大学化学研究所員。
* 1945年 - 升任京都帝国大学教授（補化学研究所員）。
* 1951年 - [[美國|美國]]解禁日本[[核物理|核物理]]實驗，接手荒勝文策在京都大學的核子物理研究室<ref name = "鄭伯昆"/><ref name="木村"/>。
* 1959年 - 成為{{link-ja|大阪府立放射線中央研究所|大阪府立放射線中央研究所}}所長（非常勤）。
* 1963年 - 成為[[京都大学|京都大学]]原子爐實驗所長<ref>《木村毅一先生還暦記念 略歴・論文リスト》（非刊行物、1964年）</ref>。
* 1968年 - 自[[京都大学|京都大学]]退休、成為京都大学名譽教授。
* 1992年 - 於[[日本|日本]][[京都市|京都市]][[左京區|左京區]]去世，享壽88歲。
