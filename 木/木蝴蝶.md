'''木蝴蝶'''（[[学名|学名]]：{{lang|la|''Oroxylum indicum''}}）是[[紫葳科|紫葳科]][[木蝴蝶属|木蝴蝶属]]的植物。

== 形态 ==
大[[乔木|乔木]]；叶子对生，三至四回羽状复叶，长达1米；夏秋间开花，紫红色肉质钟状花冠，成总状花序；[[蒴果|蒴果]]长约1米，木质果瓣；种子扁平，多数，周围有膜质的白色宽翅，形状如同蝶翅。

== 分布 ==
分布在[[印度|印度]]、[[柬埔寨|柬埔寨]]、[[台湾岛|台湾岛]]、[[菲律宾|菲律宾]]、[[老挝|老挝]]、[[缅甸|缅甸]]、[[越南|越南]]、[[泰国|泰国]]、[[马来西亚|马来西亚]]、[[印度尼西亚|印度尼西亚]]以及[[中国大陆|中国大陆]]的[[贵州|贵州]]、[[广西|广西]]、[[福建|福建]]、[[广东|广东]]、[[云南|云南]]、[[四川|四川]]等地，生长于海拔500米至900米的地区，常生于热带、公路边丛林中、亚热带低丘河谷密林、以及常单株生长，目前尚未由人工引种栽培。

== 别名 ==
千张纸（植物名实图考），破故纸、毛鸦船（四川），王蝴蝶（贵州），千层纸、土黄柏（广西），兜铃（滇南本草），海船（四川，思茅，西双版纳），朝筒（思茅），“棵黄价”（广西壮族语），牛脚筒（海南）
