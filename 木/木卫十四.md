'''木卫十四'''又稱為「忒拜」（S/1979 J 2，''Thebe''），是环绕[[木星|木星]]运行的一颗[[卫星|卫星]]。

== 概述 ==

木卫十四是離木星第四近的[[衛星|衛星]]，它是由[[航海家1號|航海家1號]]于1979年3月5日发现，剛開始它只有一個叫做'''S/1979 J 2'''的[[臨時編號|臨時編號]]，直到1983年它才以[[希臘神話|希臘神話]]中的[[寧芙|寧芙]][[忒拜_(寧芙)|忒拜]]命名。木卫十四上有三四个很大的[[陨石|陨石]]坑，这些坑的直径和木卫十四的半径差不多大。

== 軌道 ==
木卫十四是屬於[[木星|木星]]的[[木星的衛星|内圈衛星群]]，但是它偏離[[木星|木星]]的[[赤道|赤道]]為1.08°，而且[[離心率|離心率]]有0.018，這對[[木星|木星]]的[[木星的衛星|内圈衛星群]]來說是非常的大，這種不尋常的狀況可以用它與[[木卫一|木卫一]]的[[軌道共振|軌道共振]]來解釋。

== 物理性質 ==
木卫十四的[[形狀|形狀]]非常不規則，形狀有點接近[[橢球|橢球]]，大小是116x98x84[[公里|公里]]，它的[[質量|質量]]、[[密度|密度]]和[[體積|體積]]並不確定，但根據估計[[密度|密度]]大約是0.86 g/cm<sup>3</sup>，而質量大慨是4.3{{e|17}}[[公斤|公斤]]。木卫十四[[同轉衛星|公轉與自轉同步]]，也就是說它的一面永遠朝向[[木星|木星]]。這跟[[木星|木星]]的[[木星的衛星|内部衛星群]]類似。木卫十四多半呈現暗紅色的，它的背對[[木星|木星]]的一面比面對[[木星|木星]]的一面的亮度多1.3倍，可能是因為面跟背對[[木星|木星]]的一面比較容易會發生[[小行星|小行星]]撞擊，而被撞擊時會噴出較亮的[[物質|物質]]（[[冰|冰]]等），因此比面對[[木星|木星]]的一面會比較亮。

目前只有一個表面特徵獲得正式命名。
