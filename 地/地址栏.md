'''地址栏'''（{{lang-en|'''address bar'''}}，又称{{lang|en|'''location bar'''}}、{{lang|en|'''URL bar'''}}）是[[网页浏览器|网页浏览器]]的一项功能，显示当前的[[统一资源定位符|网址]]，并接受键入网址，将用户导向所选定的网站。许多地址栏在地址正在输入时提供一个建议列表。这项自动完成的功能根据浏览器的历史记录提供建议。某些浏览器使用键盘快捷方式来自动完成地址。这些通常是用户设定的，以个别情况为准。自[[NCSA_Mosaic|NCSA Mosaic]]推出之後，地址栏成為网页浏览器的一大特點。

== 特点 ==
访问使用{{lang|en|[[Favicon|Favicon]]}}（代表着网站的小图标）的网站时，地址栏中或者某处附近一般会出现一个小图标。{{lang|en|Favicon}}为网站所专用，因此若网站未有指定网页图标，浏览器则显示通用图标。<ref>{{Cite web |url = http://www.photoshopsupport.com/tutorials/jennifer/favicon.html|title=Favicon - How To Create A Favicon.ico|first=Jennifer|last=Apple |publisher=Futura Studios|accessdate=2011-02-25}}</ref>在一些浏览器，地址栏也用于显示网页的安全状态。若页面已加密，又或者为表示某通信是否安全、值得信赖，地址栏也可能显示各类颜色和挂锁图标。

一些浏览器的地址栏也能用来检测出能够用来订阅页面的[[消息來源|消息來源]]。检测出消息来源时通常以[[RSS|RSS]]图标“[[File:Feed-icon.svg|16px]]”显示。如果浏览器安装了扩充套件，地址栏也可能显示各种其他图标。

网页浏览器通常包含智能书签（{{lang|en|Smart Bookmarks}}）的功能。使用这个功能时，用户设置了命令，允许促进执行网站的功能（例如搜索、编辑或发布帖子）。之后，与命令有关的关键词或术语将键入地址栏，随后输入关键词或从列表中选择命令。

一些浏览器也能执行快速搜索，方法是输入快捷方式或者搜索词，以取代网址。比方说，通过将“w”设置为[[维基百科|维基百科]]的快捷方式，可直接在地址栏输入“w 蛋糕”，将用户直接导向维基百科的[[条目|条目]][[蛋糕|蛋糕]]。这项功能可用于{{lang|en|[[Firefox|Firefox]]}}<ref>{{cite web |url = http://www.sugarrae.com/seo-sphere/creating-firefox-quick-search-bookmarks/ |title=Creating Firefox Search Bookmarks|first=Rae|last=Hoffman|date=2008-02-10|website=Sugarrae|accessdate=2014-01-05}}</ref>、{{lang|en|[[Opera|Opera]]}}和{{lang|en|[[Google_Chrome|Google Chrome]]}}。

=== 浏览器特有功能 ===
在{{lang|en|[[Opera|Opera]]}}及{{lang|en|[[Safari|Safari]]}}，地址栏可兼作[[进度条|进度条]]（{{lang|en|progress bar}}），说明页面有多少内容已被加载。

在{{lang|en|[[Google_Chrome|Google Chrome]]}}，地址栏（亦称{{lang|en|Omnibox}}“多功能框”）兼用作[[搜索插件|搜索插件]]（{{lang|en|search plugin}}）栏，允许[[渐进式搜索|渐进式搜索]]（{{lang|en|incremental search}}）。{{lang|en|Firefox}}的扩充套件也有这项功能，<ref>{{cite web|url=https://addons.mozilla.org/en-US/firefox/addon/omnibar/|title=Omnibar|author=Ajit K|website=Add-ons for Firefox|publisher=Mozilla|accessdate=2014-02-05}}</ref>而更新的几个版本中则已经内置了此功能。<ref>{{cite web|url=http://support.mozilla.org/en-US/kb/search-web-address-bar|title=Search the web from Address Bar|website=Firefox Help|publisher=Mozilla|accessdate=2014-01-05}}</ref>此“多功能框”除上列快速搜索功能外，还能够将所输入的非网址词组转为搜索，在用户所选择的[[搜索引擎|搜索引擎]]上。<ref>{{cite web|url=http://support.google.com/chrome/bin/answer.py?hl=en&answer=95440 |title=Use the Address Bar (omnibox)|website=Chrome Help|publisher=Google|accessdate=2013-05-13}}</ref>
