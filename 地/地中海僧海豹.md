'''地中海僧海豹'''（[[學名|學名]]：''{{lang|la|Monachus monachus}}''）是一種[[僧海豹|僧海豹]]。牠們現存只有350-450隻，是世上最為稀有的[[鰭足類|鰭足類]]，也是最為[[瀕危|瀕危]]的[[哺乳動物|哺乳動物]]之一。<ref name=iucn/> 牠們分佈在[[地中海|地中海]]及東[[大西洋|大西洋]]近[[北回歸線|北回歸線]]的海域。

== 特徵 ==
地中海僧海豹平均長2.3米，重約310公斤，雌海豹比公海豹較為細小。<ref name=iucn/> 牠們的上下顎各有8對[[牙齒|牙齒]]。公海豹的毛皮呈黑色，而雌海豹的呈褐色至深灰色。牠們的鼻端短而扁平，鼻孔長而向上。鰭足相對較短，有細小的爪。牠們有兩對可伸縮的[[乳頭|乳頭]]。

幼海豹出生時出生時約長80厘米，全身有黑色毛皮覆蓋，其上有一白斑，白斑的形狀足以分辨個別的[[僧海豹|僧海豹]]。牠們的壽命不詳，合計可以活到20-25歲。

== 繁殖 ==
就地中海僧海豹的[[繁殖|繁殖]]所知甚少。[[科學家|科學家]]相信牠們是[[一夫多妻制|一夫多妻制]]的。牠們全年也可[[交配|交配]]，高峰期於10月及11月。不過由於[[風暴潮|風暴潮]]等會浸入洞穴中，令幼海豹的死亡率大大提高，尤其是在[[卡沃布蘭科|卡沃布蘭科]]（Cabo Blanco）。根據[[世界自然保護聯盟|世界自然保護聯盟]]所指，幼海豹首兩個月只有少於50%能生存下來，大部份都是在出生後兩星期[[死亡|死亡]]。於9月至1月間出生的幼海豹生存率是29%，其餘時間出生的生存率則是71%。

幼海豹出生後的兩星期會首次進入水中，到了18星期大就斷奶。雌海豹平均有9小時會出外覓食。<ref name=iucn/> 們約於4歲就達至性成熟。[[妊娠期|妊娠期]]接近1年，位於卡沃布蘭科群落的妊娠期多於1年。懷孕的地中海僧海豹會躲到不能進入的海底洞穴待產。

== 食性 ==
地中海僧海豹是夜間活動的，主要吃[[魚類|魚類]]及[[軟體動物|軟體動物]]，尤其是[[八爪魚|八爪魚]]、[[烏賊|烏賊]]及[[鰻|鰻]]，每天可以吃達3公斤。牠們會到水深50-75米的地方覓食<ref name=iucn/>，但也有到深達500米的地方覓食。牠們喜歡利用高速在廣闊的地方獵食。牠們也會獵食底棲動物，曾有見到牠們會舉起石床尋找獵物。

== 棲息地 ==
地中海僧海豹的棲息地在近年不斷轉變。20世紀前牠們都會在海灘上聚集及生育。但到了近年，牠們只利用海中洞穴[[繁殖|繁殖]]，而這些洞穴往往是位於[[人類|人類]]不能到達的地方。[[科學家|科學家]]證實這是受到人口增長、觀光及工業發展造成棲息地破壞所影響的。加上牠們怕羞及不願與人接觸，逐步遠離人群，並於近年完全與人隔絕。不過這些洞穴卻造成幼海豹的高死亡率。

== 保育狀況 ==
地中海僧海豹以往分佈在[[地中海|地中海]]及[[黑海|黑海]]沿岸，包括所有地中海的離岸[[島嶼|島嶼]]，與及[[大西洋|大西洋]]及西至[[亞速爾群島|亞速爾群島]]、[[馬德拉群島|馬德拉群島]]的島嶼。南至[[岡比亞|岡比亞]]及[[佛得角|佛得角]]，北至[[葡萄牙|葡萄牙]]及[[法國|法國]]都有一些離群的地中海僧海豹出沒。<ref name=iucn/>

地中海僧海豹的數量大幅下降的原因是商業捕漁。於20世紀，因牠們破壞漁網，漁民認為牠們是[[害蟲|害蟲]]而將牠們根除。另外，城市化及[[污染|污染]]亦影響牠們的數量。<ref name=iucn/> 在[[馬爾馬拉海|馬爾馬拉海]]，牠們因污染及頻繁的海上交通而[[滅絕|滅絕]]。現時估計牠們的數量少於500隻，故處於[[濒危|濒危]]狀況。加上牠們群落分散，增加了對牠們的危害程度。
