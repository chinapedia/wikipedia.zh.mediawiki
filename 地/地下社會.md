'''地下社會'''，簡稱'''地社'''，位於[[臺北市|臺北市]][[大安區_(臺北市)|大安區]][[師大路|師大路]]，創立於1996年8月，為[[臺北市|臺北市]]歷史悠久的[[Livehouse|Livehouse]]之一，向[[臺北市商業處|臺北市商業處]]登記名稱為「地下室飲食店」，負責人陳淑楨、[[何東洪|何東洪]]為發言人暨創辦股東。樂團如[[1976_(樂團)|1976]]、[[四分衛樂團|四分衛樂團]]、[[糯米糰_(樂團)|糯米糰]]、[[瓢蟲_(樂團)|瓢蟲]]、[[旺福|旺福]]、[[白目_(樂團)|白目]]、[[宇宙人_(樂團)|宇宙人]]、[[表兒|表兒]]、[[八三夭|八三夭]]等均曾在此表演，容納人數約80－100人。

== 結束營業 ==
因消防法規的限制，地下社會將於2012年7月15日熄燈，結束其15年的營業。此舉引起知名樂團[[五月天|五月天]]、[[1976_(樂團)|1976]]、[[小應|小應]]等大批音樂人的聲援。

== 重新營業 ==
* 地下社會創辦人之一[[何東洪|何東洪]]於2012年8月11日晚上於地下社會臉書群組討論區與[http://underworld-taipei.blogspot.tw 官方部落格]發表[http://underworld-taipei.blogspot.tw/2012/08/815.html 0815復業聲明]，文中說明復業動機，以及過程經過還有未來的計畫與相關LIVE HOUSE正名行動的細節。
* 2012年8月15日，地下社會招牌重新掛在門口，恢復營業。
* 2012年8月16日，[[何東洪|何東洪]]於臉書討論區發表 [https://www.facebook.com/notes/%E4%BD%95%E6%9D%B1%E6%B4%AA/%E4%B8%8D%E8%A6%81%E9%AB%98%E8%88%88%E4%B9%9F%E4%B8%8D%E8%A6%81%E9%8C%AF%E6%84%95%E8%AB%8B%E7%90%86%E8%A7%A3/340669632688560 不要高興，也不要錯愕，請理解！]文。

== 再度結束營業 ==
2013年6月15日，地下社會再度結束營業。
官網部落格寫道：「地下社會的租約就要到期了，二零壹三年六月十五號就是我們的最後一夜。 很遺憾，政府對於 Live House 法規沒有任何作為，師大三里自救會仍然多方施壓，視我們為社會亂源。 就讓我們珍惜所有一同創造的回憶，謝謝大家！」
[http://underworld-taipei.blogspot.tw/]
