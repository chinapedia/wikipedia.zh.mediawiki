'''地下鐵赤塚站'''（{{jpn|j=地下鉄赤塚駅|hg=ちかてつあかつかえき|rm=Chikatetsu-akatsuka eki|f=y}}）是[[日本|日本]][[東京地下鐵|東京地下鐵]]的[[鐵路車站|鐵路車站]]，位於[[東京都|東京都]][[練馬區|練馬區]]{{link-ja|北町 (練馬區)|北町 (練馬区)|北町}}八丁目，並跨越至[[板橋區_(日本)|板橋區]]{{link-ja|赤塚新町|赤塚新町}}。正式的車站事務室所在地是練馬區。

此站有[[有樂町線|有樂町線]]、[[副都心線|副都心線]]停靠。包括此站的[[和光市站|和光市站]]－[[小竹向原站|小竹向原站]]路段由有樂町線與副都心線共用軌道。有樂町線的[[車站編號|車站編號]]是'''Y 03'''、副都心線是'''F 03'''。

== 歷史 ==
* 1983年（[[昭和|昭和]]58年）6月24日：[[帝都高速度交通營團|帝都高速度交通營團]]（營團地下鐵）有樂町線、營團成增站（現時[[地下鐵成增站|地下鐵成增站]]）－池袋段通車，'''營團赤塚站'''開業。
* 2003年（[[平成|平成]]15年）2月：成為業務委託站。
* 2004年（平成16年）4月1日：伴隨營團地下鐵民營化，由東京地下鐵繼承。同時站名改稱'''地下鐵赤塚站'''。
* 2008年（平成20年）6月14日：副都心線開業，此站成為有樂町線與副都心線的共用站。
* 2010年（平成22年）
** 6月19日：1號月台設置[[月台閘門|月台閘門]]<ref name="2010-25">[http://www.tokyometro.jp/news/2010/2010-25.html 有楽町線にホームドアを設置します] - 東京地下鉄ニュースリリース 2010年4月27日</ref>。
** 6月26日：2號月台設置月台閘門<ref name="2010-25"/>。
** 9月25日：啟用月台閘門<ref>[http://www.tokyometro.jp/news/2010/2010-45.html ホームドア設置計画進行中] - 東京地下鉄ニュースリリース 2010年9月6日</ref>。
* 2011年（平成23年）2月23日：引入發車音樂<ref>[http://www.switching.co.jp/news/archives/194 有楽町線の発車メロディを制作しました] {{webarchive|url=https://web.archive.org/web/20140724184255/http://www.switching.co.jp/news/archives/194 |date=2014-07-24 }} - 株式会社スイッチ 2011年4月6日</ref>。

== 車站結構 ==
[[島式月台|島式月台]]1面2線的[[地下車站|地下車站]]，位於[[川越街道|川越街道]]（[[國道254號|國道254號]]）下方。

在最初有樂町線與[[東武東上本線|東武東上線]]實施[[直通運轉|相互直通運轉]]的計畫中，是規劃接入本站北方的[[下赤塚站|下赤塚站]]。但由於用地取的困難，加上該站非東武東上線的主要車站，僅停靠普通（各站停車）班次，之後改為接入[[成增站|成增站]]，最後改在[[和光市站|和光市站]]。因此，本站不與下赤塚站共站，而是獨立設站。

=== 站內設備 ===
* 剪票口位於地下1樓，共有兩處（西側連通睦台交差點方向出入口1、2，與東側連通東武東上線下赤塚站方向出入口3、4），月台位於地下二樓。
* 地下1樓有東西剪票口連絡通道。
* 商店（{{link-ja|METRO COMMERCE|メトロコマース|METRO'S}}）位於地下1樓東側，廁所設於地下1樓兩側。
* 2009年秋季設置候車室。門為半自動式，室內有椅子與長桌。
* 電梯連通地下1樓東側與地下2樓，電扶梯連通地下1樓東側與地下2樓。此外，東側東武東上線下赤塚站方向出入口3設有電梯，出入口2則在進行電梯設置工程。
* 1號出入口屬於板橋區赤塚新町二丁目，2號出入口屬於練馬區田柄二丁目，3號出入口屬於板橋區赤塚新町一丁目，4號出入口屬於練馬區北町八丁目。

== 車站周邊 ==

== 站名由來 ==
「{{link-ja|赤塚 (板橋區)|赤塚 (板橋区)|赤塚}}」是本站北方的板橋區地名。

站名「'''地下鐵'''赤塚站」（過去為'''營團'''赤塚站）是為避免與本站北方100公尺之[[東武鐵道|東武鐵道]][[東上本線|東上本線]]的[[下赤塚站|下赤塚站]]混淆而冠上公司名（鄰站「地下鐵成增站」也是相同情形）。

在[[PASMO|PASMO]]、[[Suica|Suica]]的乘車履歷上，站名簡稱為「地下赤塚」。
