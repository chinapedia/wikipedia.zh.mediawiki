《'''地球隊長'''》（{{lang-ja|キャプテン・アース}}），是[[BONES_(動畫製作公司)|BONES]]製作的[[日本|日本]][[電視動畫|電視動畫]]作品。於2014年4月開始播放<ref name="animeanime_20131222"/>。導演為五十嵐卓哉，系列構成為榎戶洋司和[[BONES_(動畫製作公司)|BONES]]合製的動畫，製作在2010年播放的《[[STAR_DRIVER_閃亮的塔科特|STAR DRIVER 閃亮的塔科特]]》中的工作人員再度集結<ref name="animeanime_20131222"/>。漫畫版於動畫雜誌《[[Newtype_(雜誌)|Newtype]]》2013年10月號第一報開始連載。

以[[種子島|種子島]]為舞台，和持有不思議記憶的少年為中心的青春群像劇，描繪謎之巨大[[機器人|機器人]]和機械生命體的戰爭。

第一集開始時出現的英文為：

== 故事大綱 ==
高中2年級生的'''真夏大地'''，在8歲時父親因太空事故而死亡以後便離開了種子島，在東京的蘭月高中展開生活，但在暑假開始前的一晚，在電視機裡目睹了種子島上空突如其來的出現的不可思議「圓形彩虹」。於是懷著激動而緊張的心情，大地一個人前往種子島一探究竟。

造成父親死亡的事故之謎、小時候與奇怪的少女邂逅的記憶……這些複雜的回憶在大地到達島上後湧上心頭，在某個設施發出的轟鳴警報聲中，他與「Earth Engine」相遇了——為了與從天王星而來的侵略者對抗，在這滿天星辰閃耀之際，少年的戰鬥從此拉開了帷幕。<ref>{{cite web |url=http://www.dm123.cn/new/xf/2013-12-22/49661.html |date=2013-12-22 |title=2014年4月新番「Captain Earth」STAFF、PV公開 |publisher=動漫FANS |deadurl=yes |archiveurl=https://web.archive.org/web/20140426111115/http://www.dm123.cn/new/xf/2013-12-22/49661.html |archivedate=2014-04-26 }}</ref>

此外，Earth Engine及Nebula Engine還追加了大型支援兵器「{{nihongo|增力等離子連發槍|ブーステッド・プラズマグナム}}」。

== 用語 ==
=== 組織・企業・施設 ===
; {{nihongo|Globe|グローブ}}
: 超國家的組織，以[[種子島|種子島]]為基地。
: 組織內部有支持基沃斯特計劃的「方舟派」及主張對抗並迎擊KILL-T-GANG的「迎擊派」的不同派系存在，在對抗KILL-T-GANG時意見會出現分歧。
; {{nihongo|天海道|天海道}}
: 位於地球的衛星軌道，由三個人造衛星所組成的[[太空站|太空站]]的總稱。三個人造衛星分別被命名為「α衛星（ROAD GATE）」、「β衛星（MARINE GATE）」、「γ衛星（SKY GATE）」。
: 另外，天海道亦管有並保護奇沃托斯計劃成員的人工冬眠模塊（通稱：方舟模塊）。
; {{nihongo|傳承號|レガシー}}
: 久部正樹利用西頓公司製造的人造衛星。是久部為了可以在宇宙避難，放置於地球和月球正中央，位於L1點暈軌道的重要資源衛星。質量大約是１０萬噸，直徑不足１００米，儲存了實行奇沃托斯計劃之時不可缺少的物資。這個尺寸的微型天體，就算是宇宙防衛早期警戒系統都沒可能探測得到，倘若直接墜落，衝擊所產生的能量將會是幾百萬噸TNT炸藥的威力之多。動畫１４話中，傳承號被啟動了改變軌道的推動器，墜落地點是日本南部的種子島，其背後目的是藉此制裁久部，同時毀滅Globe的基地。但最後被布露美的光速兵器完全摧毀。

=== KILL-T-GANG 相關 ===
; {{nihongo|KILL-T-GANG|キルトガング}}
: 充滿謎團的機械生命體（異星人），自稱是「[[行星齒輪|行星齒輪裝置]]」。
: 透過[[遺傳工程|遺傳工程]]的方式打造出自己跟人類外貌一樣的[[化身|化身]]來。年幼時會跟一般地球人一樣地生活，但會在一定年齡後覺醒而認識到自己是KILL-T-GANG。要回到原本的身體時，則會透過Machine Goodfellow注入高濃度的情欲因子進行「情動釋放（Abreaction）」，將自己的精神抽出後跟自我區塊融合，變換成KILL-T-GANG的機械實體。
: 根據地位於[[天王星|天王星]]的衛星軌道的[[天衛四|奧伯龍]]，機體射出後會利用[[量子門|量子門]]轉移至月球附近的空間。
; {{nihongo|感應之吻|テレパシーキス}}
: 行星齒輪裝置的特性之一。在跟某個人[[接吻|接吻]]時，就能夠大致感覺到對方那一刻心中在想什麼，也就是[[心靈感應|心靈感應]]。

=== 其他的用語 ===
; {{nihongo|奇沃托斯計劃|キヴォトス計画}}
: Globe的方舟派所支持的逃離地球計劃。對外聲稱是由志願者組成的遠距離宇宙旅行，實際是將被挑選的極少數的精英們進行[[人工冬眠|人工冬眠]]，並在地球被KILL-T-GANG攻擊時乘搭宇宙船逃離地球。
; {{nihongo|馬古斯號事件|マグス号事件}}
: 事件中一艘屬於黑幫組織－麻野田組的遊艇在橫濱被引爆，發生爆炸的時候，幾乎所以麻野田組的成員都在那艘遊艇上，卻沒有人死亡。爆炸的原因是由於兇手麻野田久美子對這個圍繞著自己的世界感到非常絕望，因此把在地下室找到炸彈引爆，希望藉以終止父親人口販賣的行為。
