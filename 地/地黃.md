'''地黃'''（[[学名|学名]]：''{{lang|la|Rehmannia glutinosa}}''），又稱作'''芐'''、'''芑'''、'''生地'''，為[[玄参科|玄参科]][[地黄属|地黄属]]植物。

地黄在传统纺织业中用作[[黄色|黄色]][[染料|染料]]，故得名。地黄的[[块根|块根]]则為傳統[[中藥|中藥]]之一，最早出典於《[[神農本草經|神農本草經]]》，依照[[炮製|炮製]]方法在藥材上又分為'''鮮地黃'''、'''乾地黃'''與'''熟地黃'''。

==原植物==
[[File:Rehmannia.JPG|220px]]株高15－30厘米，初夏开花，花大数朵，淡紅紫色。

== 產地 ==
主要產地為[[中國|中國]]北方，以[[河南省|河南省]][[焦作市|焦作市]]一帶，如[[溫縣|溫縣]]、[[博愛縣|博愛縣]]、[[武陟縣|武陟縣]]等地最为著名，在[[朝鲜半岛|朝鲜半岛]]和[[日本|日本]]也有分布。焦作市在古時為[[怀庆府|怀庆府]]，因此地區出產的地黃功效最佳而頗負盛名，稱為「懷慶地黃」。

== 炮製方法 ==
鮮生地即為採集新鮮者，可榨汁用，稱'''地黃汁'''。

乾生地為採集完後晒乾者。

熟地黃則是與良酒、[[砂仁|砂仁]]拌炒，並經九次蒸、晒而得。炮製目的可使地黃的副作用（如：味苦、滋膩感、與服用後腸胃不適等情形）明顯的降低。

== 中医学用途 ==
按炮製不同，性味、用途也不同。

===鮮生地===
新鲜地黄称为'''鲜地黄'''或'''鲜生地'''。
;性味
性寒，味甘苦。
;功效
清熱涼血，養陰生津
用於因[[溫病|溫病]]或[[熱病|熱毒]]而[[吐血|吐]][[衄血|衄]]、發斑，可以清火、解熱、涼血而止血。

===乾生地===
經乾燥後的地黄稱为'''乾地黄'''或'''乾生地'''。
;性味
性涼，味甘微苦。
;功效
可以滋陰、涼血，治[[消渴|消渴]]、[[溫病|溫病]]入血分。主治阴虚内热、虚烦不眠、[[月经|月经]]过多等症。

《[[神農本草經|神農本草經]]》-{云}-：「乾地黄，味甘寒，主折跌、絶筋、傷中，逐血痺，填骨髓，長肌肉；作湯除寒熱、積聚，除痺。生者尤良，久服輕身不老。」

古方亦用在補腎藥，在《[[傷寒雜病論桂林古本|傷寒雜病論桂林古本]]》中皆以乾生地入藥，如【[[腎氣丸|腎氣丸]]】即採用乾生地、[[桂枝|桂枝]]，配[[黃酒|黃酒]]服；而不如後世相近的複方【[[桂附地黃丸|桂附地黃丸]]】(又稱【金匱腎氣丸】)是採用熟地黃、[[肉桂|肉桂皮]]。

===熟地黃===
经加工蒸制後的地黄称为'''熟地黄'''或'''熟地'''。
;性味
性微溫，味甘，歸肝、腎經。。
;功效
用於補血滋陰(尤其是腎陰)、益精填髓。主治肾虚阴亏、头晕目眩、腰酸、[[遗精|遗精]]、[[崩漏|崩漏]]等症。

[[肝經|肝經]]調血用的知名複方——【[[四物湯|四物湯]]】，尋常是以熟地黃入藥；除非病者有血熱、上火情形才改用乾生地。
