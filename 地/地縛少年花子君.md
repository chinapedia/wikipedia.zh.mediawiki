《地縛少年花子君》（{{lang-ja|地縛少年花子くん}}）是[[日本|日本]][[漫畫家|漫畫家]]あいだいろ於[[月刊GFantasy|月刊GFantasy]]上連載的漫畫作品。曾在月刊GFantasy上短期連載3話，後長期連載決定，從2015年1月號開始連載到至今。另有姊妹作（{{lang-ja|放課後少年花子くん}}）於《[[Pファンタピー|Pファンタピー]]》連載。

== 故事簡介 ==

海鷗學園的奇妙傳言之一。據說舊校舍三層女廁所的第三間裡的花子同學，會實現前來拜訪者的願望，非常喜歡巫術等超自然現象的少女八尋寧寧為了實現自己的願望，將之託付給了學園怪談。

== 登場角色 ==
=== 主要角色 ===
; {{nihongo|花子君|花子くん（はなこくん）|Hanako-Kun}}
: 本作主角，為[[七大不可思議|七大不可思議]]之七：廁所裡的花子君（トイレの花子くん）有著半透明的身體，身穿舊式制服、頭戴學生帽，監督學園裡的妖怪們，保持人類和怪異的正確關係是他的義務。總是喜歡開一些無傷大雅的玩笑或捉弄寧寧，但也時常露出溫柔的一面，並喜歡甜甜圈。武器為一把菜刀，身旁圍繞著靈魂稱之為白杖代；據源光所述，源光的奶奶曾將"廁所裡的花子同學"給封印過，似乎有著不為人知的過去......
: 底下有一個弟弟，曾透露為自己所殺害，其原因不明；花子君左臉頰上的封條為白底紅字，弟弟則反之。

;{{nihongo|柚木普|柚木 普（ゆぎ あまね）|Yugi Amane}}
: 花子君生前的名字，喜歡用「俺」來自稱。擁有一顆不起眼的石頭，並稱之為「月之石」，後來交給了土籠。
: 據土籠所述，若當時柚木普並未逝世的話，將來會成為理科老師。
: 以前在祭典時曾遇過從50多年後穿越過來的八尋寧寧，並用了寧寧給他的短冊寫下了「和寧寧姐姐、能夠再一次見面」的願望。

; {{nihongo|八尋寧寧|八尋 寧々（やしろ ねね）|Yashiro Nene}}
: 本作女主角，海鷗學園高等部一年級生，所屬於園藝部，特長是耕地，是個花癡、喜歡帥哥，但卻因為蘿蔔腿的理由而常常失戀。因聽聞校園七大不可思議的傳說中的花子可以實現願望，而到廁所尋找花子的少女，之後遇上各種奇妙的事件。起初暗戀學長源輝，但是因為不擅長表白而失敗，無意中使用了結緣石變成了金魚，後與花子結緣，效果減半恢復人類姿態，但是遇水會出現魚鱗，並且會變回魚形態。在16時的書庫中看見了花子君的過去，而感到迷茫，不知如何面對花子君，在後來與源光的對談中，解開心結並和花子君告白。是花子君的助手，現與花子君、源光等人試圖找出改變怪異傳言的真凶。

; {{nihongo|源光|源 光（みなもと こう）|Minamoto Kou}}
: 海鷗學園中等部三年級生，是稀世的驅魔師·源賴光的子孫。源家的後代，所以說能看見花子君，並且打算代替祖先驅除花子君，後被寧寧阻止，偶爾會和他們一起行動。擁有祖先留下的強力的驅魔具，前期視花子君為眼中釘並稱呼為惡靈，後期則對花子君稍有改觀，對寧寧抱有好感。
: 一年級時與三葉惣助同班，在三葉死後再度相遇，並成為朋友，是引出三葉真正願望的重要原因。
: 善長家務及料理；其兄長為源輝，底下還有一名5歲的妹妹。

; {{nihongo|源輝|源 輝（みなもと てる）|Minamoto Teru}}
: 海鷗學園高等部二年級生，擔任學生會長一職，是源光的兄長，也是天才驅魔師
: 前期為寧寧所暗戀之人，意外的挺照顧弟弟，為人兩面，一心想除掉花子君，深信世上沒有好的怪異
: 曾一度要消滅花子君，但被其弟弟源光所阻止，進而網開一面放了花子君。

; {{nihongo|柚木司|柚木 司（ゆぎ つかさ）|Yugi Tsukasa}}
: 花子君的雙胞胎弟弟，正式登場於第十七話。哥哥花子君所殺害之人，稱花子君為阿普。右臉頰上的封條為黑底紅字，性格異常扭曲，身旁圍繞著靈魂稱之為黑杖代，七峰櫻是他的助手，並與櫻及夏彥兩人為改變傳言的真凶，喜歡向七峰櫻撒嬌。
: 據本人所述，花子君所實現的是還活著的現世之人所許下的願望，自己則是實現已死去，且為陰間之人的願望。

; {{nihongo|七峰櫻|七峰 桜（ななみね さくら）|Nanamine Sakura}}
: 海鷗學園高等部三年級生，與柚木司（花子君的弟弟）結緣成為其助手，時常對於柚木司的行為感到無奈但又包容著
:初登場時被寧寧誤認為是花子君的女朋友，似乎知道些關於七大不可思議的事情，在第十七話末證實為改變傳言的真凶。

; {{nihongo|日向夏彥|日向 夏彦（ひなた なつひこ）|Hinata Natsuhiko}}
: 海鷗學園高等部二年級生，是七峰櫻的隨從，並對她抱有好感。在本作主要擔綱搞笑的角色。

; {{nihongo|三葉惣助|ミツバ|Mitsuba}}
: 原海鷗學園中等部一年級生，愛好攝影，曾與源光為同班同學，後因事故而死亡。在初中部入口意外的再次和源光相遇，並在與源光的交談以及相處下，漸漸成為朋友，但由於先前向柚木司許下"想留在大家記憶裡"的願望並不明確，進而使得柚木司改變傳言，而被改造。
: 後期於七大不可思義之三的魔鏡地獄中，被柚木司強迫吞下原境界擁有者的心臟，使得身體產生變異，成為新七大不可思義之三，並遺失某部分被改造前與源光相處的記憶，目前主要與柚木司三人組一起行動。

=== 次要角色 ===

; {{nihongo|赤根葵|赤根 葵（あかね あおい）|Akane Aoi}}
: 寧寧的同班同學兼好友並與倉井茜為青梅竹馬；很常給予寧寧一些建議，是很受歡迎的女孩子。喜歡與寧寧分享學園中的怪異傳聞，在岬的事件中，因誤入領域而失蹤，後在花子一行人成功解決事件後，回歸現實。

; {{nihongo|蒼井茜|蒼井 茜（あおい あかね）|Aoi Akane}}
: 海鷗學園高等部一年級生，與八尋寧寧、赤根葵以及山吹檸檬為同班同學，在校擔任學生會副會長一職，為人和善、學業頂尖，與赤根葵為青梅竹馬並喜歡她，據源光轉述，倉井對葵告白次數高達3624次，可每次都無疾而終，對於葵的愛甚至已經病入膏肓（？），曾因受到怪異「告白之樹」的影響而與山吹交往。
: 在第二十四話末揭露了其真實身分為七大不可思議之一，三位時鐘守衛中的第二位。

; {{nihongo|山吹檸檬||Yamabuki Lemon}}
: 海鷗學園高等部一年級生，與八尋寧寧、赤根葵以及蒼井茜為同班同學，曾陪蒼井在告白之樹下練習告白，亦受到影響而與蒼井交往。喜歡玩手機

; {{nihongo|源賴光|源 頼光（みなもと のよりみつ）|Minamoto Noyorimitsu}}
: 源家先祖，曾打敗了日本三大妖怪之一的酒吞童子。

; {{nihongo|岬老師||}}
: 彌子所心念之人，能夠看的見怪異、因為階梯的事故摔下而身亡。

=== 怪異們 ===
; 人魚
: 活了千年的人魚妖怪。吞下從同一隻人魚身上取得的鱗片的兩人，在受到人魚詛咒的同時會結下非常強力的緣分。如果未能結緣，吞下鱗片的人會變成魚；如果成功結緣，兩人的詛咒則減輕(像是半人半魚)

; {{nihongo|勿怪|勿怪（もっけ）|Mokke}}
: 很久之前就躲在人的身邊、經常做出盜取或者惡作劇的一種怪異。即羸弱又膽小，如果要攻擊的話就會聚集在一起，變身成可怕的怪物(？)；喜歡糖果。

; 木魅
:木之怪異，會對在自己樹下告白的人，強行地繁殖戀愛的萌芽之後讓他們成為戀人。

=== 七大不可思議 ===
: 七大不可思議的怪異們，棲宿在學校裡，讓生者有所畏懼是他們的任務''
* '''七大不可思議之一：三位時鐘守衛（過去、蒼井茜、未來）
:謠傳「在學校某處有著一座很大又古老的時鐘，而時鐘就是學校內所流動的時間本身，如果能轉動那個指針的話，就能自由的操控這個學校的時間；而在時鐘旁則有三位時鐘守衛，第一位的能力是將『時間倒回』、第二位的能力是將『時間停止』、第三位的能力是讓『時間前進』。」
* '''七大不可思議之二：{{nihongo|岬的階梯|ミサキ階段}} {{nihongo|彌子|ヤコ|Yako}}'''
:謠傳「黃昏的時候，如果踏上美術教室前樓梯的第四階、就會進入死者的世界，並且身體會撕裂」。
:原為供奉在神社階梯的稻荷像，因思慕著人類、隨著時間過去有著歪曲的想法，變成了怪異，外表看是狐狸、名子叫做彌子。
:※ '''''其依附物為一把剪刀'''''
* '''七大不可思議之三：魔境地獄 {{nihongo|三葉惣助|ミツバ|Mitsuba}}
:謠傳「會將闖入者害怕或不想面對的事反射出來，如果闖入者夠正面、勇敢，就很容易離開」。
:原境界擁有者形象為一頭黑色巨鷹，中途在柚木司的闖入後被其殺死，三葉則被迫吞下原境界者心臟，進而成為新七大不可思議之三。
* '''七大不可思議之四：（？？）四島芽衣
:據說喜歡著花子君（參見短篇連載）
* '''七大不可思議之五：{{nihongo|16時的書庫|16時の書庫}} {{nihongo|土籠|土籠（つちごもり）|Tsuchigomori}}'''
:謠傳「在學校的圖書館內，有個特別的書庫只能在16時的時候才能進去；書庫裡的書都是記載著每個人在校園期間的紀錄，過去以及未來全都寫在裡面，書庫共分有白、黑、紅三種書，白色記載的是還活著的人，黑色則是記載已經死亡的人，只有紅色的書，據說絕對不能看。」
:是生前柚木普的老師，依附物「月之石」則是由柚木普交給他保管的；原形為蜘蛛，喜歡抽著煙斗，目前在學校繼續擔任著老師的角色，實則為16時的書庫管理人。
:※ '''''其依附物為「月之石」'''''

==用語==
'''校園七大不可思議'''
:流傳於海鷗學園的傳說；所謂的七大不可思議就是指在自己支配的區域裡是最強的怪異。

'''傳言'''
:大部份的妖怪都無法違逆傳言，不按照傳言行動的妖怪，會慢慢的從此岸消失不見。

'''惡靈'''
:生與死之間有著一段很大的間隔，能夠跨越的只有抱負著強烈怨念的惡靈，所以惡靈能觸碰到人類。
:據說拿著兇器的幽靈就是在生前曾用那兇器，把誰的生命給奪走過。

'''神之附體／依附物'''
:能夠給予七大不可思議的特別力量，一旦破壞掉七大不可思議就會大幅度弱化
:每一位七大不可思議都有屬於自己的神之附體(依附物)。
