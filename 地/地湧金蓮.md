'''地涌金蓮'''（[[學名|學名]]：''Musella lasiocarpa''），俗稱'''千瓣蓮花'''、'''地金蓮'''、'''不倒金鋼'''，觀賞[[蕉|蕉]]的一種，為[[芭蕉科|芭蕉科]][[地湧金蓮屬|地湧金蓮屬]]屬唯一一種；多年生[[草本|草本]]植物，原產於[[中國|中國]][[雲南省|雲南省]]中部至西部。

== 形態 ==
生長匍匐莖有些為假莖，葉面大，如芭蕉葉，花碩大，從假鱗莖中取出花序，形狀如花瓣的黃色大苞片，六枚苞片為一輪，頂生或腋生，呈蓮座狀在花序上方，層層逐漸開展，由上而下，花期長久不凋謝，約250天左右。植株為直直的枝幹，苞片裡嬌小的花才是真正的花，美麗而有光澤，在苞片展開時才可看見。花序下部者為雌花，上部雄花。[[File:4660f73e440c23e9f2190.jpg|地湧金蓮的果實]]
== 習性 ==
喜溫暖，不耐寒，在[[冬天|冬天]]為粉紅色，[[夏天|夏天]]則變[[草|草]]綠光澤，會散發甜蜜的[[花|花]]香。
== 栽培 ==
[[中國|中國]]南方常用地下栽培方法，春秋分株，栽植後必需即時澆水。[[秋|秋]]末和初[[春|春]]在植株的周圍開一條溝施以腐熟[[有機|有機]]肥，並在假[[莖|莖]]根基部培以肥土，促進生長開花。[[乾旱|乾旱]]時要適當澆水，[[雨季|雨季]]時需即時排水。開花之後地上的部分假莖會逐漸枯死，要將它們砍掉，隔年才可以再發新芽。
== 引進 ==
=== 台灣 ===
於西元1996年引進[[台灣|台灣]]。

== 文化 ==
為[[雲南|雲南]]少數民族所崇敬的植物，在信仰[[佛教|佛教]]的[[傣族|傣族]]裡列為寺院必種[[五樹六花|五樹六花]]之ㄧ，尊稱他為千瓣蓮花。後來花也傳到[[泰國|泰國]]與[[寮國|寮國]]等[[佛教|佛教]][[國家|國家]]，在當地[[佛寺|佛寺]]也廣為種植，千變奇特的造型也成為建築和雕刻的設計元素。

== 得獎 ==
西元2008年12月30日，第五屆[[中國|中國]]盆栽花卉交易會，經過來自全國各地的花卉專家及花卉企業的共同評選，'''地涌金蓮'''等五種花種從1,000多個選送作品中脫穎而出，奪得[[金花獎|金花獎]]。
