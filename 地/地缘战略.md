'''地缘战略'''是[[地缘政治学|地缘政治学]]的分支学科，討論基于地理的型態、位置與國家實力對於國家[[外交政策|外交政策]][[戰略|戰略]]的影響。其中包括[[阿爾弗雷德·賽耶·馬漢|阿爾弗雷德·賽耶·馬漢]]提出的[[海權論|海權論]]來強調海洋海權重要性；[[哈爾福德·麥金德|哈爾福德·麥金德]]提出的[[世界島|世界島]]與心臟地帶理論，認為俄羅斯與中國大陸擁有歐亞大陸核心地帶的資源成為陸地強權，在沒有受到挑戰建立陸地強權的海上艦隊時，就能同時成為海上與陸上強權，而其他海權國家可能對陸上強權國家封鎖出海口相抗衡；[[尼古拉斯·斯皮克曼|尼古拉斯·斯皮克曼]]的學說以哈爾福德·麥金德為基礎，認為英國、日本、美國海上強權與俄羅斯及中國大陸等陸上強權，雙方權力交接的歐洲、中東與亞洲季風帶的沿岸[[邊緣|邊緣]]地帶權力變化，才是影響世界權力變化的重要區域。<ref name = "七○年代東亞風雲: 台灣與琉球,釣魚台,南海諸島的歸屬問題_ 2015"> {{ cite web | title = 七○年代東亞風雲: 台灣與琉球,釣魚台,南海諸島的歸屬 問題 | url = https://books.google.com.tw/books?id=w_ROjwEACAAJ&dq=%E4%B8%83%E2%97%8B%E5%B9%B4%E4%BB%A3%E6%9D%B1%E4%BA%9E%E9%A2%A8%E9%9B%B2%EF%BC%9A%E5%8F%B0%E7%81%A3%E8%88%87%E7%90%89%E7%90%83%E3%80%81%E9%87%A3%E9%AD%9A%E5%8F%B0%E3%80%81%E5%8D%97%E6%B5%B7%E8%AB%B8%E5%B3%B6%E7%9A%84%E6%AD%B8%E5%B1%AC%E5%95%8F%E9%A1%8C&hl=zh-TW&sa=X&ved=0ahUKEwi-lIjM8_rcAhUMFYgKHT_5BWwQ6AEIJTAA | publisher = 前衛出版社 台灣教授協會 | author = 許文堂 主編;王雲程;李明峻;林穎佑;陳文賢;陳俐甫;陳儀深;蔡東杰;賴怡忠 | language = zh-tw | date = 2015-09-14 }} </ref>

== 历史 ==
===古代中国===
中国春秋战国时期，地缘战略成为[[秦国|秦国]]能够最终战胜六国的重要原因。中国古代地缘战略强调固险而守。

== 台灣 ==
{{main|台灣地緣戰略}}
台灣的地緣戰略是以海洋島國為基礎，位於太平洋[[邊緣|邊緣]]，西太平洋[[第一島鏈|第一島鏈]]戰略位置，也是在[[東亞|東亞]]土地面積最小與人口最少的國家；經濟上倚靠對外航海與航空交通貿易生存，政治軍事上間受陸地與海上強權中國大陸與海上強權美國影響，軍事上台灣與美國的地緣戰略共同利益包括美國對台軍售，由美國提供台灣自我防衛武器，協防西太平洋的安全與穩定，政治上台灣目前實行與美國共同價值的自由、民主政治制度。<ref name = "七○年代東亞風雲: 台灣與琉球,釣魚台,南海諸島的歸屬問題_ 2015" /><ref name = 台灣的地緣戰略_2008> {{ cite web | title = 台灣的地緣戰略 | url = http://www.tisanet.org/quarterly/4-2-6.pdf | author = 余家哲、李政鴻 | date = 2008-08-08 }} </ref><ref name = "自由開講》怎麼看待台灣的地緣戰略位置？_2017"> {{ cite web | title = 自由開講》怎麼看待台灣的地緣戰略位置？ | url = http://talk.ltn.com.tw/article/breakingnews/1978224 | publisher = 自由時報 | language = zh-tw | date = 2017-02-17 }} </ref><ref name = "自由開講》怎麼看待台灣的地緣戰略位置？_2017"> {{ cite web | title = 自由開講》怎麼看待台灣的地緣戰略位置？ | url = http://talk.ltn.com.tw/article/breakingnews/1978224 | publisher = 自由時報 | language = zh-tw | date = 2017-02-17 }} </ref><ref name = "台灣的選擇：地緣悲情之子，還是領先東亞轉大人_2017"> {{ cite web | title = 台灣的選擇：地緣悲情之子，還是領先東亞轉大人 | url = https://opinion.udn.com/opinion/story/6067/2736919 | publisher = 聯合報 | author = 范疇 | language = zh-tw | date = 2017-10-03 }} </ref><ref name = 李世暉_2017> {{ cite web | title = 臺日關係中「國家利益」之探索：海洋國家間的互動與挑戰 | url = http://www.pf.org.tw/files/5965/E9C55902-A3CA-43F7-B7A0-344E289B984F | author = 李世暉 | date = 2017-08-21 }} </ref>
