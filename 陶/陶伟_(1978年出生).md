'''陶伟'''{{BD|1978年|3月11日|catIdx=T陶}}，[[中华人民共和国|中国]][[北京市|北京市]]人，一名退役[[足球|足球]][[运动员|运动员]]，现任[[北京国安|北京国安]]队助理教练，曾担任球队队长并入选[[中国国家男子足球队|中国国家男子足球队]]。在球场上可以担任左后卫和前腰两个位置，并且擅长主罚任意球和点球。由于曾担任北京国安队队长，故被北京球迷昵称为“'''陶队'''”。

== 职业生涯简历 ==
*1991-1995年 北京少年队

*1995年 北京威克瑞青年队

*1996-1997年 随“健力宝”青年队赴[[巴西|巴西]]留学训练<ref>{{cite web |url=http://news.sports.cn/china/chinan/news/2008-12-05/1693456.html |title='健力宝'最后大旗倒下 李玮峰一代国足悲情谢幕 |publisher=华奥星空 |date=2008-12-05 |accessdate=2009-04-06 |deadurl=yes |archiveurl=https://web.archive.org/web/20090131062422/http://news.sports.cn/china/chinan/news/2008-12-05/1693456.html |archivedate=2009-01-31 }}</ref>

*1998年开始効力[[北京国安|北京国安]] 其中于1998年、2002年、2006年和2008年获得联赛季军；于2003年获得[[中国足协杯|中国足协杯]]冠军；于2004年中超联赛第18轮北京国安面对[[上海申花|上海申花]]的比赛，陶伟在第6分钟，44分钟和89分钟分别进球，时隔83分钟是中国顶级联赛中，时隔时间最长的[[帽子戏法|帽子戏法]]。于2007年获得联赛亚军；于2009年获得[[中国足球超级联赛|中超联赛]][[冠军|冠军]]。

*[[2009年中国足球超级联赛|2009赛季]]结束后，刚帮助北京国安获得联赛冠军的陶伟曾宣布退役<ref>{{cite web |url=http://sports.nfdaily.cn/content/2009-12/07/content_6857901.ht |title=陶伟退役赴西班牙留学 王晓龙或成新中场接班人 |publisher=南方报网 |date=2009-12-07 |accessdate=2009-12-11 }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，但最终选择多踢一年，世界杯之后陶伟暂时离开北京国安教练组前往西班牙学习，他的球员生涯也随之结束。
