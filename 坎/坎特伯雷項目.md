'''坎特伯雷項目'''（{{lang-en|The Canterbury Project}}），[[自由軟體|自由軟體]]界的一個[[愚人節|愚人節]]玩笑，由[[Debian|Debian]]、[[Gentoo|Gentoo]]、[[Grml|Grml]]、[[openSUSE|openSUSE]] 以及 [[Arch_Linux|Arch Linux]]五個自由軟體社群，于2011年4月1日聯手發動<ref>{{Cite web |url=http://www.debian.org/News/2011/20110401 |title=Debian 网站上关于坎特伯雷项目的说明 |accessdate=2011-04-01}}</ref><ref>{{Cite web |url=http://blog.grml.org/archives/352-The-Canterbury-Project.html |title=Gentoo 开发日志上关于坎特伯雷项目的说明 |accessdate=2011-04-01}}（页面已删除）</ref><ref>{{Cite web |url=http://news.opensuse.org/2011/04/01/the-canterbury-distribution/ |title=Grml 网站上关于坎特伯雷项目的说明 |accessdate=2011-04-01}}</ref><ref>{{Cite web |url=http://blog.grml.org/archives/352-The-Canterbury-Project.html |title=openSUSE 网站上关于坎特伯雷项目的说明 |accessdate=2011-04-01}}</ref><ref>{{Cite web |url=http://www.archlinux.org/news/the-canterbury-project/ |title=Arch Linux 网站上关于坎特伯雷项目的说明 |accessdate=2011-04-01}}</ref>。当日上述发行版的官方网站全部改成相同的主页，宣布他們將合并为一个新的发行版，名為'''坎特伯雷'''（{{lang|en|The Canterbury Distribution}}）。他們宣稱，這個計劃的目的是展示[[自由軟體|自由軟體]][[社群|社群]]實際上是能夠為了共同目標而共同合作，而並非創造更多的差異。它的目標是建立一個像Arch一樣簡單，像Debian一樣穩定，像Gentoo一樣有高度可塑性，像Grml一樣支援Live CD框架，以及像openSUSE一樣具備開放心靈的Linux作業系統發行版。
