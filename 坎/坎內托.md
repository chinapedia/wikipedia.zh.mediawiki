'''坎內托'''（{{lang-it|'''Canneto'''}}）是一個位於[[意大利|意大利]][[墨西拿省|墨西拿省]][[卡罗尼亚|卡罗尼亚]]的一個小村莊及[[民政教區|民政教區]]，其名字在[[意大利語|意大利語]]意思為[[蘆原|蘆原]]。

==地理==
坎內托位於[[第勒尼安海|第勒尼安海]]沿岸地區，距離卡罗尼亚約10公里，離[[巴勒摩|巴勒摩]]約105公里及離[[墨西拿|墨西拿]]約125公里。

==歷史==

===自燃事件===
自2004年1月24日開始，村內開始出自無法解釋的[[自燃|自燃]]事件。先從[[保險絲|保險絲]]開始，然後到[[微波爐|微波爐]]、家庭用具、汽車等都自燃起來。後來居民要求切斷電力供應，以免再發生自燃現象，但在沒有電力供應情況下供莫名其妙地自燃。政府當局排除了電線短路引起火災，但無法解釋為何物品會自燃，居民轉向宗教尋求答案，並得到[[教宗|教宗]][[若望·保祿二世|若望·保祿二世]]的祝福，但仍無補於事<ref>{{cite web |url = http://rense.com/general96/stronglink.html |title = Strong Link Between WTC And A Small Town In Sicily |accessdate=2014-12-25 |publisher= }}</ref>。於是當地政府下令撤離村內所有居民，並派人調查事件<ref>{{cite news |url = http://tech.big5.enorth.com.cn/system/2004/04/05/000762427.shtml|title = 神秘自燃再次侵擾西西裡小鎮 居民緊急撤離 |accessdate=2014-12-25 |date=2004-04-05|publisher=北方網 }}</ref>。除此外化，村內的電腦都會自動故障，車的門鎖會自動開關，[[指南針|指南針]]失去作用，於是他們懷疑這一切都是由[[电磁场|电磁场]]引起。調查人員開始檢查村內的電磁場，發現事發時有大量電磁能脈衝，並推斷這些電磁場是人造的，並且來自海上。可是不久之後政府下令終止調查工作，使人聯想到這可能是汲及到軍事機密實驗<ref>{{cite web |url = http://www.veteranstoday.com/2013/01/13/canneto-a-double-mystery-still-unsolved/ |title = Canneto – A Double Mystery Still Unsolved |accessdate=2014-12-25 |publisher= veteranstoday.com/ }}</ref>。但一份在1932年出刊的報導已記載當地有超常的電磁場。2007年，有媒體聲稱官方報告指起因歸咎於[[外星人|外星人]]<ref>{{cite web |url = http://www.openminds.tv/mysterious-fires-officials-blamed-ufos-aliens-return-sicily/30707 |title = Mysterious fires officials blamed on UFOs and aliens return to SicilyMysterious fires officials blamed on UFOs and aliens return to Sicily |accessdate=2014-12-25 |publisher= openminds.tv }}</ref>。是次調查共動員了多次名專家及約100萬英鎊的經費<ref>{{cite news |url = http://www.telegraph.co.uk/news/worldnews/1567428/Aliens-caused-Sicily-fires-say-officials.html |title = Aliens caused Sicily fires, say officials |accessdate=2014-12-25 |date=2007-10-26|publisher=tHE tELEGRAPH}}</ref>。
