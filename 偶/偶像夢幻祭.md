{{vgname|偶像夢幻祭|ja=あんさんぶるスターズ!}}是由[[中華人民共和國|中國]][[北京|北京]]樂元素公司旗下Happy Elements K.K所製作的一款女性向偶像養成類[[手機遊戲|手機遊戲]]。遊戲圍繞“青春、偶像、高校生”等主題，玩家將扮演一名轉學到男性偶像培養學校的女生，與學院中組成十一個風格各異的組合的偶像一起打造最強偶像團體！腳本主要由[[日日日|日日日]]所擔任寫作，活動故事以及轉蛋限定故事也有由結城由乃寫作的時候。遊戲採用了live2D技術讓人物更加生動，同時遊戲聲優陣容十分華麗。

游戏和同樣由Happy Elements K.K製作日日日擔任腳本的「{{link-ja|あんさんぶるガールズ!|あんさんぶるガールズ!}}」有部分世界觀是相通的<ref>http://dengekionline.com/elem/000/001/037/1037942/</ref>。

== 故事 ==

男子偶像養成的專門學校「私立夢之咲學院」新設立的製作人科第一個轉學進入的女學生(玩家)，在個性豐富的偶像們之中進行培訓製作。

== 夢之咲學院活動 ==

=== 金星杯 ===
金星杯主要是以一年級爲主的演出。演出的人為從兩班一年級中所選出的四位代表。演出代表的條件是仍然沒有加入任何組合的一年級學生。因爲金星杯演出之後會是名組合的對決，所以演出的時候不會有太多人過來觀看，對大部分人來説，金星杯只是場熱身而已。
