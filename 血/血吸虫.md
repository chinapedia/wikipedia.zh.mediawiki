'''血吸蟲'''，又名'''裂体吸蟲'''，屬[[扁形動物門|扁形動物門]]，主要指所有歸類在[[裂體屬|裂體屬]]下19個同[[屬_(生物)|屬]]的[[物種|物種]]。其中有6[[種_(生物)|種]]可以寄生[[人類|人類]]主要者有三種，分別流行於[[中東|中東]]、[[亞洲|亞洲]]和[[南美洲|南美洲]]地區，分布範圍較廣，且是可以感染人類的主要三種血吸蟲；另外三種則侷限於[[北非|北非]]、[[馬來半島|馬來半島]]、[[湄公河|湄公河]][[流域|流域]]，由於多為[[動物株|動物株]]（zoophilic strain），對人的影響較小。血吸蟲成長的過程都必須經過在[[淡水|淡水]][[螺|螺]]類體內的寄生階段，才有能力感染其他[[宿主|宿主]]，而血吸蟲寄生後多選擇在宿主體內的[[靜脈|靜脈]]血管定居，所引起的症狀表現各有不同，皆統稱為[[血吸蟲症|血吸蟲症]]（或[[血吸蟲病|血吸蟲病]]），被[[世界衛生組織|世界衛生組織]]公佈在六大[[熱帶醫學|熱帶醫學]]疾病之一。<ref name="行政院衛生署">{{cite book |language=zh-hant |author1=范家堃 |aithor2=張淑美 |author3=黃協賢 |author4=李細祥 |year=2006年 |chapter=Ch.71 血吸蟲症 |title=《人畜共通傳染病臨床指引》 |publisher=[[行政院衛生署|行政院衛生署]][[疾病管制局|疾病管制局]]出版 }}</ref>。

==生活史==
血吸虫的生活史包括成虫、卵、毛蚴、胞蚴、尾蚴及童虫等6个阶段。

感染血吸虫的人或其它哺乳动物从粪便中排出虫卵，若粪便污染了水，虫卵被带进水中，在水里孵出毛蚴。毛蚴能在水中自由游动，并主动钻入水中的钉螺体内，发育成母胞蚴，进行无性繁殖，产生子胞蚴。子胞蚴再经一次繁殖，产生大量尾蚴，尾蚴离开钉螺在水中自由游动。人们因生产劳动、生活用水、游泳戏水等各种方式与含有尾蚴的水接触后，尾蚴便很快钻进人体皮肤，进入皮肤后即转变成童虫，经过一定时间的生长发育，最终在肝、肠附近的血管内定居寄生，并发育成熟，成为成虫。雌、雄成虫结伴合抱，交配产卵，每条雌虫每天可产卵二三千个。这样一个周期即是血吸虫的一生，即生活史。 

在血吸虫的生活史中，有两个宿主，一个是被成虫寄生的人和其它哺乳动物，称为终宿主，许多种哺乳动物都可成为血吸虫的终宿主；另一个是被幼虫寄生的钉螺，叫中间宿主，钉螺是日本血吸虫的唯一中间宿主。

== 血吸虫病 ==
{{main|血吸蟲病}}
血吸虫对人体损害导致症状称为“'''血吸虫病'''”，血吸虫病俗称“大肚子病”，属于人和[[牛|牛]]、[[羊|羊]]、[[猪|猪]]等[[哺乳动物|哺乳动物]]感染'''血吸虫'''所引起的一种[[传染病|传染病]]和[[寄生虫病|寄生虫病]]；血吸虫人对人畜危害以虫卵损害为最为严重；虫卵沉着在宿主的肝脏及肠壁等组织，形成虫卵肉芽肿，最后导致[[肝|肝]][[脾|脾]]肿大、[[肠|肠]]壁纤维化、[[肝硬化|肝硬化]]和[[肝腹水|腹水]]；儿童如果反复感染血吸虫会引起[[发育|发育]]不良、[[智力|智力]]减退、[[生殖|生殖]]机能不好，形成血吸虫性“[[侏儒症|侏儒症]]”，丧失劳动力。

中国只流行[[日本血吸虫|日本血吸虫]]，因最早发现于[[日本|日本]]而得名。日本血吸虫的[[毛蚴|毛蚴]]在水中可活2～3天，毛蚴主要寄生于[[钉螺|钉螺]]，在螺体内发育为母胞蚴、子胞蚴、尾蚴后逸出；人畜接触含有尾蚴的疫水后，尾蚴即可通过皮肤侵入；尾蚴脱去尾成为童虫，在寄主（宿主）体内移行发育，在移行发育过程中未能到达[[门静脉系统|门静脉系统]]的童虫，因不能发育为成虫而死亡。童虫一旦到达[[门静脉系统|门静脉系统]]，即定居其中发育为成虫，终生不再移动。

血吸虫传播途径：人畜传播主要有钉螺、粪便、水源。

== 流行病學 ==
血吸蟲病分布於[[北緯|北緯]]34°到[[南緯|南緯]]34°之間廣大的[[熱帶|熱帶]]和[[亞熱帶|亞熱帶]]地區，有70餘個國家和地區受到侵擾，估計約有

血吸虫病流行于[[热带|热带]]和[[亚热带|亚热带]]地区，世界上有76个[[国家|国家]]和[[地区|地区]]流行；中国血吸虫主要分布于[[长江流域|长江流域]]及[[长江|长江]]以南的[[江苏|江苏]]、[[浙江|浙江]]、[[湖南|湖南]]、[[湖北|湖北]]、[[安徽|安徽]]、[[江西|江西]]、[[四川|四川]]、[[云南|云南]]、 [[广东|广东]]、[[广西|广西]]、[[福建|福建]]和[[上海|上海]]等12个[[省|省]]（[[直辖市|市]]、[[自治区|自治区]]）409[[县|县]]（[[县级市|市]]、[[市辖区|区]]）均有发现，最南处为[[广西|广西]][[横县|横县]]，最北处为[[江苏|江苏]][[宝应|宝应]]县；主要流行区为湖南、湖北、江西、安徽、江苏、四川、云南等7省；最严重的地区湖北的[[荆州|荆州]]、湖南的[[岳阳|岳阳]]。

目前有74個國家和地區的6億人口生活在流行區，其中有2億病人。這些地區的氣候和自然環境適於本蟲中間寄主螺類的繁殖，人與疫水接觸機會亦多。中國只有日本血吸蟲病(因本病首先在日本發現，病原蟲的生活史也首先經日本學者詳細研究而闡明，故得名)，20世紀50年代以前曾流行於長江以南大部分產糧區，患者多達1000餘萬人。經大規模群眾性防治，至70年代發病人數已降至250萬，台灣僅發現動物感染日本血吸蟲。

== 中國疫情 ==
[[湖南|湖南]][[长沙|长沙]][[马王堆汉墓|马王堆汉墓]]出土的[[西汉|西汉]]古尸的肝脏、肠道均查出血吸虫[[虫卵|虫卵]]，这一发现证实至少在2100年前的[[汉代|汉代]]中国就出现血吸虫病。[[中华人民共和国|中华人民共和国]]成立以后，从1956—1957年，[[中国大陆|中国大陆]]对血吸虫病进行全面普查和防治，血吸虫流行区至此对血吸虫病进行了有效控制，并建立了如[[青浦任屯血防陈列馆|青浦任屯血防陈列馆]]等宣传场馆。

最近几年血吸虫病疫情开始变得很严峻。根据2003年[[中华人民共和国卫生部|中国卫生部]]向[[中华人民共和国全国人民代表大会常务委员会|全国人大常委会]]报告时提供的数据，[[中国大陆|大陆]]地区有427个[[县级行政区|县（市、区）]]存在[[疫情|疫情]]，受威胁人口6500万，钉螺分布[[面积|面积]]达2.22万[[公顷|公顷]]，[[洞庭湖|洞庭湖]]、[[鄱阳湖|鄱阳湖]]的江湖洲滩地带，[[四川|四川]]、[[云南|云南]]的部分山区为重疫区。据卫生部门发布的统计资料，2003年全国有血吸虫病人84万，比2000年增加了15万；其中急性感染人数明显增加，2003年发生了30多起急性血吸虫病爆发[[疫情|疫情]]，2003年流行区７省报告急性[[感染|感染]]1114人，较2002年同期上升22％。
