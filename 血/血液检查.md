'''血液检查'''（{{Lang-en|Blood test}}），是指通过[[采血|採血]]以获得受检者的[[血液|血液]]，并利用其进行临床检查以获取受检者的健康状况。主要通过[[醫事檢驗師|医检师]]进行检验。

== 血液检查的意义 ==

血液检查主要是判断自身在一定时间内的身体状况的一种重要方法。我们可以从血液中获得很多信息。特别是在一些比较轻微的疾病中，患者并未表现出明显患病体徵或仅感受轻微不适，尚能忍受，而不以为然。在疾病不发展到一定过程的时候也通常没有明显临床表现。出于这个原因，很多相关报告要求人们要经常体检，哪怕你未觉任何不适。

对我们生命的维持所不可欠缺的，属于我们身体中的一部分并且日夜不息在我们血管内流动的就是血液。通过检查血液，所读出的信息量是巨大的。在大型医院的血液检测中，往往可以检测超过2000个不同项目。在我们生活中经常看到的疾病，如糖尿病、癌症、过敏性反应（如杉树花粉）、遗传性过敏症、艾滋病、贫血、白血病等多种疾病，可从血液中获得，甚至是可能会导致动脉粥样硬化的血液状态。血液测试是发现各种疾病必不可少的过程。

== 血液检查的項目 ==

=== 血液学的检查 ===

; 血球数计算 （{{Lang-en| Blood Cells Counting}}）

: 下列项目所标识的结果，总称为全血球計算（全血算, CBC）：

:* [[红血球|红血球]]数检査...貧血或多血症时数目异常。

:* [[白血球|白血球]]数检査...感染症或白血病时增多，骨髄抑制时减少。

:* [[血小板|血小板]]数检査...配合凝血试验进行有無出血傾向的判定。

:* [[血红蛋白|血红蛋白]]量检査...用来作为貧血或多血症的指标。

; 白血球组分

: [[細菌|細菌]][[感染|感染]]的情况下[[嗜中性白血球|嗜中性白血球]]数量增加。

; 末梢血塗抹

: [[白血病|白血病]]的情况下可以观测到肿瘤细胞。
