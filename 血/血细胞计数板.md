'''血球计数板'''是一种用于计数细胞的设备。它原先是被设计用作[[全血球计数|全血球计数]]的。<div>血球计数板是由 [[Louis-Charles_Malassez|Louis-Charles Malassez]] 发明的。其由一个带有矩形的锯齿状的计数室的厚[[玻璃片|载玻片]]组成。计数室被精确地画上格线以保证每个单元格的大小相等，并且每个格子的深度也相同，故而通过血球计数板可以测量一定体积液体中的细胞数。</div>

== 规格 ==
血细胞计数板中分布 1 x 1 mm (1 mm<sup>2</sup>) 的小正方形框；这些正方形框又被进一步分为三种规格：0.25 x 0.25 mm (0.0625 mm<sup>2</sup>)、0.25 x 0.20 mm (0.05 mm<sup>2</sup>)、0.20 x 0.20 mm (0.04 mm<sup>2</sup>)；其中心部分又被进一步分为 0.05 x 0.05 mm (0.0025 mm<sup>2</sup>) 的正方形格。其各处厚度均匀，皆为 0.1 mm。<ref><cite class="citation web" contenteditable="false">[http://hemocytometer.org/2013/04/11/hemocytometer-square-size/ "Hemocytometer: square sizes"].</cite><span class="Z3988" title="ctx_ver=Z39.88-2004&rfr_id=info%3Asid%2Fen.wikipedia.org%3AHemocytometer&rft.btitle=Hemocytometer%3A+square+sizes&rft.genre=unknown&rft_id=http%3A%2F%2Fhemocytometer.org%2F2013%2F04%2F11%2Fhemocytometer-square-size%2F&rft_val_fmt=info%3Aofi%2Ffmt%3Akev%3Amtx%3Abook" contenteditable="false"> </span></ref>

== 使用 ==
欲使用血细胞计数器，首先要确保盖玻片被正确地放置于计数室上，检验标志是可看见[[牛顿环|牛顿环]]。这样的话细胞悬浮液就会因[[毛细作用|毛细作用]]被吸至计数室和盖玻片中的缝隙。而在盖玻片中每个单元格的细胞数量可以使用[[显微镜|显微镜]]直接数出，并且可以通过观察区分出不同细胞。因为体积和在此体积中的细胞数量已知，故溶液中的细胞密度可以使用以下公式得出：
<br>
<math>
\omega  = k 
    \left (
        \frac{n_0}{V}
    \right )

</math> 
<ref>{{cite web|url=http://celeromics.com/en/resources/docs/Articles/Cell-counting-Neubauer-chamber.php|title=Basic Hemocytometer usage}}</ref>

其中，<math>\omega</math> 为 细胞密度，<math>n_0</math> 为每个单元格中细胞数量的平均值，<math>V</math>为每个单元格的体积，<math>k</math>为稀释倍数

计数时，在每个单元格中，若细胞在单元格的边缘上是，通常仅计数左侧及上方的的格线及其所夹顶点的上的细胞，而忽略右侧及下方格线及另外三个顶点上的细胞，即“记上不计下，记左不计右”。<ref>{{cite book|author=Strober W|chapter=Monitoring cell growth|title=[[Current_Protocols|Current Protocols]] in Immunology|publisher=John Wiley & Sons|year=2001|location=USA|editor=Coligan JE, Bierer BE, Margulies DH, Sherach EM, Strober W|page=A.2A.1|volume=5|doi=10.1002/0471142735.ima03as21}}</ref>

== 注意事项 ==
[[File:Hemocytometer_grid_100x.jpg|right]]
* 样品在测量前必须被均匀混合，以保证其具有代表性，而非仅仅从以特定区域中取样。
* 应做适当的[[稀释|稀释]]，以保证细胞数量可测。若未进行足够的稀释，则细胞会因过于密集而难以计数；若稀释过度，则测量时的细胞数不足以支撑推断原溶液细胞密度。
* 进行测量两次测量，并比对结果。若二者差异超过正常值，则取样方法可能不可信（如溶液未均匀混合）。

== 应用 ==
* [[全血细胞计数|全血细胞计数]]：尤其适用于血细胞不正常而无法使用机器计数的病人。
* [[精液分析|精液分析]]
* 细胞培养：接种或记录[[细胞生长|细胞生长]]。
* [[酿酒|酿酒]]: 准备酵母。
* 为下游分析做准备的过程：一些实验需要有准确的细胞数量，如[[聚合酶链式反应|聚合酶链式反应]]、[[流式细胞术|流式细胞术]]，而还有一些实验需要细胞有较强的生存能力。
* 测量细胞大小：显微检查时，细胞大小可通过和照片上的已知的单元格大小比对。<ref><cite class="citation web" contenteditable="false">[http://www.microbehunter.com/2010/09/22/using-a-hemocytometer-to-calculate-cell-size/ "Measuring cell size with a hemocytometer"].</cite><span class="Z3988" title="ctx_ver=Z39.88-2004&rfr_id=info%3Asid%2Fen.wikipedia.org%3AHemocytometer&rft.btitle=Measuring+cell+size+with+a+hemocytometer&rft.genre=unknown&rft_id=http%3A%2F%2Fwww.microbehunter.com%2F2010%2F09%2F22%2Fusing-a-hemocytometer-to-calculate-cell-size%2F&rft_val_fmt=info%3Aofi%2Ffmt%3Akev%3Amtx%3Abook" contenteditable="false"> </span></ref>
