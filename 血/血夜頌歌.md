《'''血夜頌歌'''》（[[英語|英語]]：{{lang|en|''Blood Canticle''}}）是美國作家[[安·萊絲|安·萊絲]]的系列小說「[[吸血鬼紀事|吸血鬼紀事]]」第十部，於2003年10月28日出版。[[譯林出版社|譯林出版社]]於2007年12月1日發行[[簡體中文|簡體中文]]版，由嚴志軍和郭萍萍合譯。<ref>{{cite web|url=http://www.books.com.tw/products/CN10115061|title=血頌 (簡體書)|deadurl=yes|archiveurl=https://web.archive.org/web/20170322112038/http://www.books.com.tw/products/CN10115061|archivedate=2017年3月22日|df=}} [[博客來|博客來]]</ref>此作增加了來自上一部作品《[[布萊克伍德莊園|布萊克伍德莊園]]》和「梅菲爾女巫生涯」系列裡的新角色。<ref name="EW 2003-10">{{cite web |url=http://www.ew.com/ew/article/0,,526068,00.html |title= Book Capsule Review: ''Blood Canticle'' (2003) |first=Alynda |last=Wheat |date=October 31, 2003 |accessdate=November 3, 2014 |work=[[Entertainment_Weekly|Entertainment Weekly]]}}</ref>安·萊絲原本是將這部小說作為「吸血鬼紀事」系列的完結篇，讓這位著名[[吸血鬼|吸血鬼]]{{tsl|en|Lestat de Lioncourt|黎斯特·德·萊昂柯特}}的傳奇就此劃下句點。但在2014年3月，她宣佈將會出版此系列的第十一部作品《{{tsl|en|Prince Lestat|黎斯特王子}}》。<ref name="Guardian 2014-03">{{cite web|first=Alison |last=Flood |url=https://www.theguardian.com/books/2014/mar/11/anne-rice-vampire-prince-lestat-interview-with-the-vampire |title=Anne Rice revives much-loved vampire for new novel ''Prince Lestat'' |work=[[The_Guardian|The Guardian]] |date=March 11, 2014 |accessdate=June 4, 2014}}</ref><ref name="LA Times 2014-03">{{cite web |url=http://www.latimes.com/books/jacketcopy/la-et-jc-anne-rice-book-prince-lestat-20140310-story.html |title=Anne Rice brings back her vampire antihero with ''Prince Lestat'' |work=[[Los_Angeles_Times|Los Angeles Times]] |first=Carolyn |last=Kellogg |date=March 10, 2014 |accessdate=November 4, 2014}}</ref>

== 反響 ==
《血夜頌歌》在[[亞馬遜公司|亞馬遜網站]]上遭到許多讀者的「毒舌式負面評價」，萊絲隨後也在網站上寫下一篇措辭激烈的「1200字強力防禦」回復。萊絲說之所以寫下這篇回復，是因為多數評價中都包含針對她的寫作能力、身體健康和心理狀況的人身攻擊，這些攻擊式評論認為自從她的丈夫於2002年去世以後，她整個人的狀態已經每況愈下。不過萊絲也說她收到很多其他作者對她表達支持的郵件，他們認為亞馬遜上的留言「粗俗不堪且毫無價値」。<ref name="IHT">{{cite web |url=http://www.iht.com/articles/2004/10/13/features/anne.php |archiveurl=https://web.archive.org/web/20090303213033/http://iht.com/articles/2004/10/13/features/anne.php |title=Fan power takes on new meaning |date=October 14, 2004 |archivedate=2009年3月3日 |accessdate=November 3, 2014 |first=Sarah |last=Lyall |work=[[International_Herald_Tribune|International Herald Tribune]] |deadurl=yes }}</ref><ref name="Telegraph 2008-11">{{cite web |url=http://www.telegraph.co.uk/culture/donotmigrate/3562792/Anne-Rice-interview-with-the-vampire-writer.html |title=Anne Rice: interview with the vampire writer |work=[[The_Daily_Telegraph|The Telegraph]] |first=Stuart |last=Husband |date=November 2, 2008 |accessdate=August 10, 2014}}</ref>
