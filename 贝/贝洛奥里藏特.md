'''贝洛奥里藏特'''（{{lang-pt|Belo Horizonte}}），也称作'''-{zh-hans:美景市、比路贺利桑特;zh-tw:貝洛奧里藏特、比路賀利桑特;zh-hk:美景市、貝洛奧里藏特;}-'''，是[[巴西|巴西]]第四大城市，人口約210萬，面積334平方公里，[[米纳斯吉拉斯|米纳斯吉拉斯]]州政府所在地，是该州的政治、经济、文化中心。位於巴西東南部。當地有有[[冶金|冶金]]、[[宝石加工|宝石加工]]、[[汽车|汽车]]制造、[[纺织|纺织]]、[[食品|食品]]等工業較為發達。

== 历史 ==
原为库拉尔德雷农庄。[[巴西|巴西]]第一个规划建设的城市，1893年按20万人口的规模开始建设，1897年落成，原名西达德德米纳斯（“矿山城”的意思），取代东南山区的[[欧鲁普雷图|欧鲁普雷图]]成为州首府，并成为周围地区农牧产品集散地和矿业中心。二战后，随着铁路和公路的兴建，工业迅速兴起，人口由1940年的18万激增至1970年的123万，市区大大超出原来规划，成为巴西发展最快的城市之一。是[[2014年世界杯足球赛|2014年巴西世界杯]]、[[2016年夏季奥运会|2016年巴西奥运会]]的举办城市之一。


== 城市设施 ==
[[Image:Chevrolethall.JPG|thumb]]大廳]]
有[[米纳斯吉拉斯联邦大学|米纳斯吉拉斯联邦大学]]等3所高等学府和音乐、舞蹈、技术学校以及艺术宫、博物馆等设施，拥有巴西第二大足球场[[米内罗体育场|米内罗体育场]]。行政中心位于自由广场，工业区位于西郊和南郊，以人工湖为核心的旅娱乐区在潘普拉区。城市以整洁著称。

== 经济 ==
贝洛奥里藏特的工业产总值仅次于[[聖保羅_(巴西)|圣保罗]]和[[里约热内卢|里约热内卢]]，主要经济支柱为贸易、服务业和工业，主要工业部门有采矿、钢铁、汽车、石油加工、金属加工、电气、机车车辆、水泥、陶器、食品、纺织、金刚石和宝石加工、木材加工等。[[GDP|GDP]]总量位列全巴西第四，2007年人均GDP约合7000美元。

== 交通 ==
贝洛奥里藏特铁路和公路网发达，可通往内地和[[大西洋|大西洋]]沿岸各城市，[[坦奎多·尼維斯國際機場|坦奎多·尼維斯國際機場]]位于该市北郊。

== 地理 ==

位于[[巴西高原|巴西高原]]东部的埃斯皮尼亚苏山脉西南，有高速公路与铁路和南距471公里的[[里约热内卢|里约热内卢]]和西南距586公里的[[圣保罗|圣保罗]]相通。面积334.5平方千米，人口210.2万；大贝洛奥里藏特面积3670平方公里，人口2540.1万（2013）。城市坐落在四周环绕库拉尔尔代尔雷伊山的高原上，海拔830～930米。

===气候===
[[File:Igreja_de_São_Francisco_de_Assis_-_Lagoa_da_Pampulha.jpg|thumb]]
贝洛奥里藏特的[[纬度|纬度]]是南纬19°55′，属[[热带地区|热带地区]]。该地属于[[柯本气候分类法|柯本气候分类法]]下的[[热带草原气候|热带草原气候]]（高海拔热带气候，夏季温暖湿润，冬季干燥温和），接近[[副热带湿润气候|副热带湿润气候]]地带。贝洛奥里藏特距离大西洋有大约{{convert|300|km|mi|}}。

尽管较温和地方的四季的气候差异并不明显，但该地的春季与夏季，以及秋季与冬季，仍有较明显的差异。该地一年中最冷的月份是七月，有记录以来的最低温度为{{convert|2|°C|0|abbr=on}}；最热的月份通常是一月，有记录以来的最高温度为{{convert|35.4|°C|0|abbr=on}}。

贝洛奥里藏特{{convert|852|m|ft|}}的海拔高度使得它更加凉爽。贝洛奥里藏特常年的气温都很温和。每年的气温在{{convert|11|-|31|C|F}}范围内波动，年平均气温是{{convert|22|°C|0|abbr=on}}。该地冬季干燥，夏季多雨。
