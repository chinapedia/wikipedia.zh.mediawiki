{{About|一种度量放射性活度的物理单位|以“贝可勒尔”为名的物理学家|亨利·贝可勒尔}}
[[File:Portrait_of_Antoine-Henri_Becquerel.jpg|right]]
'''贝克勒尔'''（{{lang-fr|'''Becquerel'''}}，符号为：{{lang|fr|'''Bq'''}}），簡稱'''貝克'''<ref>{{cite web |language=zh-cn |url=http://www.cnctst.gov.cn/ |title=術語查詢-請輸入關鍵字" Becquerel " |publisher=全国科学技术名词审定委员会 |accessdate=2011-04-12 |deadurl=yes |archiveurl=https://web.archive.org/web/20120901061407/http://www.cnctst.gov.cn/ |archivedate=2012-09-01 }}</ref> <ref>{{cite web |language=zh-tw |url= http://terms.naer.edu.tw/search/?q=Becquerel&field=ti&op=AND&match=&group=&num=10|title=Becquerel  |publisher=國家教育研究院學術名詞資訊網 |accessdate=2013-06-17}}</ref>。是[[放射性活度|放射性活度]]的[[国际单位制导出单位|国际单位制导出单位]]，用於衡量[[放射性|放射性]]物質或'''放射源'''的計量單位{{r|新}}。贝可勒尔的[[国际单位制|国际单位制]]量纲为[s<sup>-1</sup>]，同[[频率|频率]]单位[[赫兹|赫兹]]相同，但意义完全不一样。一定量的放射性核種，若每秒有一个[[原子|原子]][[衰变|衰变]]，其放射性活度即為1貝克，例如，一克的[[镭|镭]]-226放射性活度有3.7×10<sup>10</sup>Bq。這是個相當龐大的數值，因此常用千貝克（kBq）甚至百萬貝克（MBq）來表示{{r|新}}。
放射性射線對人體具有危害，毫无防护下长期接触放射物质，會使健康受到严重损害{{r|星}}。

給予固定質量的放射性物質，其貝克數值會隨著時間的演進而改變。因此，對於短壽命同位素，必需標明其放射性衰變率與時間戳在一起，有時候，這衰變率還會調整至某特定日期（過去日期或未來日期）。例如，安全組織時常會標明出一個十日調整數目，即十日後的放射性活度。在這時間間隔內，短壽命同位素的數量可能會大幅降低。

簡而言之，貝克表現了放射源在某一時刻的放射性活度。如果放射源不變，由於自然衰變的作用放射性活度會隨時間逐漸變小。

== 命名 ==
[[File:Becquerel_plate.jpg|thumb]]发现放射性物質的底片]]
得名於法国物理学家[[亨利·贝可勒尔|亨利·贝可勒尔]]的姓，在1975年，[[國際輻射單位與度量委員會|國際輻射單位與度量委員會]]（ICRU）認為[[居里夫人|居里夫人]]與她先生[[皮埃爾·居里|皮埃爾·居里]]兩人，以及貝可勒爾共同獲得放射性物質研究的[[諾貝爾物理學獎|諾貝爾物理學獎]]，因此ICRU建議科学界以單位更小的貝克取代[[居里_(单位)|居里]]（Ci），成為活度的國際專用單位{{r|原}}。

== 前綴 ==
就像許多其它國際單位，在Bq單位的前面也能夠添加[[國際單位制詞頭|國際單位制前綴]]；通常是以[[十進制|十進制]][[數量級|數量級]]增進，如kBq（千貝克，10<sup>3</sup> Bq）、MBq（麥貝克，10<sup>6</sup> Bq）、GBq（吉貝克，10<sup>9</sup> Bq）、TBq（太貝克，10<sup>12</sup> Bq），以及PBq（拍貝克，10<sup>15</sup> Bq）。實際使用，1 Bq是很小的單位，所以，通常會使用到前綴。例如，存在於70千克质量的人體內的天然放射性核素[[鉀|钾（<sup>40</sup>K）]]會造成每秒4400次[[放射性衰变|放射性衰变]]，即4.4 kBq的[[放射性活度|放射性活度]]<ref>[http://www.fas.harvard.edu/~scdiroff/lds/QuantumRelativity/RadioactiveHumanBody/RadioactiveHumanBody.html Radioactive human body—Harvard University Natural Science Lecture Demonstrations]</ref>。在日本廣島市的原子彈爆炸估計造成了8×10<sup>24</sup>Bq（8YBq，8佑貝克）<ref>Michael J. Kennish, [http://books.google.ca/books?id=CJqcq2C792UC&printsec=frontcover#PPA74,M1 ''Pollution Impacts on Marine Biotic Communities''] , CRC Press, 1998, p. 74. ISBN 9780849384288.</ref>。

== 貝克與每秒計數之間的分別==
當使用探測器測量樣品的活度時，常會用到衡量單位[[每分鐘計數|每秒鐘計數]]（cps）或[[每分鐘計數|每分鐘計數]]（cpm）。每秒鐘計數是单位时间1秒钟内放射性检测仪器（如[[盖革计数器|盖革计数器]]）测到的[[电离辐射|电离事件]]的计数，每分鐘計數表示1分钟内的电离事件的计数。這兩種計數單位一般用於表示某种粒子的度量单位，如[[阿尔法粒子|阿尔法粒子]]、[[贝塔粒子|贝塔粒子]]。這兩種計數單位表示檢測儀器記錄的事件發生率。

表示放射源在单位时间内放射活动绝对强度的计数單位为[[每分鐘計數|每秒钟衰变数]]（{{lang|en|disintegrations per second}}，记为DPS）、[[每分鐘計數|每分钟衰变数]]（{{lang|en|disintegrations per minute}}，记为DPM）。這兩種衰變數單位表示放射源的衰變事件發生率。

1貝克等價於1每秒钟衰变数。貝克是放射性物質的放射性活度，不同於每秒鐘計數、每分鐘計數。

== 貝克與居里之間的關係 ==
[[居里_(单位)|居里]]（Ci）為早期使用的放射性單位，目前[[國際單位制導出單位|國際單位制導出單位]]已不採用，其基準相當於1[[克|克]]的[[鐳|鐳]]226[[放射性活度|放射性活度]]。

换算因子如下：
: 1 Ci = 3.7×10<sup>10</sup> Bq
: 1 Ci = 37 GBq
: 1 μCi = 37,000 Bq
: 1 Bq = 2.70×10<sup>−11</sup> Ci
: 1 Bq = 2.70×10<sup>−5</sup> μCi
: 1 GBq = 0.0270 Ci

== 應用 ==
貝克主要是用來衡量水、食物、土壤或其他放射源等物質中放射性核素釋放出放射線強度的大小{{r|新}}。
常用於食物安全檢驗輻射的量度單位，這是量化有多少輻射從物件放射出來的單位。如以此單位量度食物的輻射量時，便以每公斤放射出多少「貝克」來作標準{{r|知}}。
<br>
爲了表現單位物體的放射性活性強度，引入了貝克/升（Bq/l），貝克/千克（Bq/Kg）和貝克/平方米（Bq/m<sup>2</sup>）等單位。

== 危害標準 ==
2011年3月17日，日本[[厚生勞動省|厚生勞動省]]修訂《{{tsl|ja|食品衛生法|食品衛生法}}》，為食品暫定國家標準值。以[[碘|碘]]與[[銫|銫]]為例：半衰期很短的放射性[[碘-131|碘131]]活度上限，飲用水、牛奶及乳制品為每公斤300貝克，蔬菜類（根莖類、薯類除外）為每公斤2000貝克；至於半衰期很長的放射性[[銫-134|銫134]]及[[銫-137|銫137]]活度上限，飲用水、牛奶及乳製品為每公斤200貝克，蔬菜類、穀物類、水產品、肉、蛋等都是每公斤500貝克{{r|0317-3}}{{r|新2}}。

香港海關則採用[[國際食品法典委員會|國際食品法典委員會]]（CODEX）所定下的標準：不論任何類型食品，一律採用相同標準。當中以每公斤碘131所放射出100貝克為上限，而銫134及銫137為每公斤放射出1000貝克為上限，如果超過此輻射量即視為為超標{{r|香}}{{r|知}}。

==活度計算方法==
給定[[質量|質量]]為<math>m</math>（[[公克|公克]]）、[[原子量|原子量]]為<math>m_a</math>（[[原子質量單位|amu]]）、[[半衰期|半衰期]]為<math>t_{1/2}</math>（秒）的[[同位素|同位素]]樣品，其[[放射性活度|放射性活度]]<math>R</math>（貝克）為<ref>{{citation|last=Halliday|first=David|coauthors=Robert Resnick, Jearl Walker|title = Fundamental of Physics|publisher = John Wiley and Sons, Inc.|location = USA|edition = 7th|isbn=0-471-23231-9|page =pp. 1173-1175|year=2005}}</ref>

:<math>R=\lambda N =\frac{m}{m_a}N_A\frac{\ln(2)}{t_{1/2}}</math>；

其中，<math>\lambda=\ln(2)/t_{1/2}</math>是[[衰變常數|衰變常數]]，<math>N=m N_A/m_a</math>是放射性[[原子核|原子核]]的數目，<math>N_A=6.022 141 79(30)\times 10^{23}</math>是[[亞佛加厥常數|亞佛加厥常數]]。

例如，天然存在的[[鉀元素|鉀元素]]含有[[豐度|豐度]]為0.012%、半衰期為1.248{{e|9}}年=39{{e|15}}秒、原子量為39.96 amu的<sup>40</sup>K，所以，1公斤天然鉀的活度為32 kBq：

:<math>R = \frac{1000\times 0.012\%}{39.96}\times 6.0\  \ 10^{23}\times\frac{\ln(2)}{39 \ \ 10^{15}}=32 kBq</math>。

== 和希沃特的關係 ==
貝克是描述'''放射源'''活性強度（量綱[s<sup>-1</sup>]）的單位，而[[希沃特|希沃特]]則是描述'''被輻射物體'''所受單位輻射劑量（量綱[m<sup>2</sup>s<sup>-2</sup>]）的單位。兩者是完全不同的概念。貝克強調放射源的放射性強弱，而[[希沃特|希沃特]]則着重表示放射性被害尤其是對人體健康的危害程度。貝克只和放射源物質的元素性質和數量有關。而[[希沃特|希沃特]]由放射源活性強度，放射物性質，離放射源的距離，障礙物的有無，地形，風向等等因素決定。
