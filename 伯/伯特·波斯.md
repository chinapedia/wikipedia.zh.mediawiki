'''伯特·波斯'''（{{lang-en|'''Gijsbert (Bert) Bos'''}}，{{bd|1963年|11月10日|||}}<ref name=dis>Gijsbert Bos, [http://dissertations.ub.rug.nl/FILES/faculties/arts/1993/g.bos/titlecon.pdf Rapid user interface development with the script language Gist] {{webarchive|url=https://web.archive.org/web/20110928111941/http://dissertations.ub.rug.nl/FILES/faculties/arts/1993/g.bos/titlecon.pdf |date=2011-09-28 }}, Dissertation, Groningen University, 1993</ref>），是一位開發{{link-en|Argo (網頁瀏覽器)|Argo (web browser)|Argo}}而聞名的計算機科學家，Argo是為他的[[CSS|CSS]]提案而開發的[[網頁瀏覽器|網頁瀏覽器]]。

== 生活和工作 ==
波斯生於[[荷兰|荷兰]][[海牙|海牙]]，他曾在[[格罗宁根大学|格罗宁根大学]]学习数学，并完成了他的博士[[论文|论文]]《''Rapid user interface development with the script language Gist''》<ref name=dis/>。

1996年，他加入了[[万维网联盟|万维网联盟]]（W3C），致力於[[CSS|CSS]]的研究。他是W3C的前任主席以及现任CSS工作组的W3C Staff Contact<ref>{{Cite web |author=fantasai |url=http://www.w3.org/blog/CSS/2008/04/04/resolutions_17 |title=Resolutions 2008-03 San Diego Part I: Working Group Operations, Communication, and Charter |date=2008-04-04 |work=CSS Working Group Blog |accessdate=2008-04-16}}</ref>。他目前定居於[[法国|法国]][[索非亚安蒂波利斯|索非亚安蒂波利斯]]。

== 著作 ==
他与[[哈肯·维姆·莱|哈肯·维姆·莱]]合著了一本关於CSS的书籍。
* ''Cascading Style Sheets: Designing for the Web''，ISBN 0-201-41998-X
* ''Cascading Style Sheets: Designing for the Web''（第2版），ISBN 0-201-59625-3
*''Cascading Style Sheets: Designing for the Web''（第3版），ISBN 0321193121
