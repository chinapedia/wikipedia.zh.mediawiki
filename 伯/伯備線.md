'''伯備線'''（{{jpn|j=伯備線|hg=はくびせん|rm=Hakubi sen|f=y}}）是一條由[[日本|日本]][[岡山縣|岡山縣]][[倉敷市|倉敷市]][[倉敷站|倉敷站]]開始，途經[[新見站|新見站]]，至[[鳥取縣|鳥取縣]][[米子市|米子市]][[伯耆大山站|伯耆大山站]]的[[西日本旅客鐵道|西日本旅客鐵道]]（JR西日本）鐵道路線（[[幹線|幹線]]）。路線名取自兩縣的古代[[令制國|令制國]]——[[備中国|備中国]]及[[伯耆国|伯耆国]]，2016年春季起開始使用路線編號及顏色，代表編號為{{JR西路線記号|S|V}}<ref>{{cite web|url=http://www.westjr.co.jp/press/article/2015/09/page_7661.html|title=～お客様によりわかりやすく鉄道をご利用いただくために～岡山・福山エリアの主な路線に「路線記号」「ラインカラー」を導入します|publisher=西日本旅客鐵道|date=2015-09-17|language=ja}}</ref>。

此線越過[[中國地方|中國地方]]的山區，連結[[山陽地方|山陽地方]]的[[岡山市|岡山市]]與[[山陰地方|山陰地方]]的米子市，為「陰陽連絡線」之一。倉敷 - 新見間路線沿高梁川、生山 - 伯耆大山間路線沿著日野川。

== 概要 ==
倉敷站－備中高梁站間位於[[IC卡|IC]][[乘車卡|乘車卡]]「[[ICOCA|ICOCA]]」的岡山、廣島地域的岡山、福山地區<ref>[http://www.jr-odekake.net/icoca/area/map/okayama.html ご利用可能エリア 岡山・広島エリア｜ICOCA｜ICCOA：JRおでかけネット] {{Wayback|url=http://www.jr-odekake.net/icoca/area/map/okayama.html |date=20110404093246 }} - 西日本旅客鉄道</ref>。另外，2016年12月17日岡山、廣島、山陰、香川地域當中，作為松江、米子、伯備地區，備中高梁站－伯耆大山站間的特急停靠站（[[新見站|新見站]]、{{link-ja|生山站|生山駅}}、[[根雨站|根雨站]]）可以使用<ref name="westjr20160824">[http://www.westjr.co.jp/press/article/2016/08/page_9133.html 山陰線（出雲市～伯耆大山駅間）、伯備線（根雨駅、生山駅、新見駅）でICOCAがご利用できるようになります！] - 西日本旅客鉄道 2016年8月24日</ref>。備中高梁站以北的ICOCA使用路段只限乘車券機能，不發售ICOCA定期券。

== 運行形態 ==
岡山 - 出雲市間有特急「[[八雲號列車|八雲號]]」運行。東京 - 出雲市間的寝台特急「[[出雲號列車|Sunrise出雲]]」於伯備線行駛。

倉敷側的全列車直通運行至[[岡山站_(日本)|岡山站]]，伯耆大山側的全列車運行至[[米子站|米子站]]，還有經岡山往山陽本線[[相生車站_(兵庫縣)|相生車站]]・[[和氣車站|和氣車站]]・[[瀨戶站|瀨戶站]]、[[赤穗線|赤穗線]]方向，經米子往山陰本線至[[西出雲站|西出雲站]]的快速列車。岡山 - 備中高梁間、生山・根雨 - 米子間有區間運轉列車。

新見 - 備中神代間有[[藝備線|藝備線]]列車進入，途中的布原站只有藝備線列車停車，伯備線沒列車停車。

清音 - 總社間，是井原鐵道的第二種鐵道事業區間，與伯備線共用設施，有[[井原鐵道井原線|井原鐵道井原線]]列車運行。

== 歷史 ==
=== 路線歷史 ===
建設時分別向南北方向推進。最初開業的北側伯耆溝口 - 伯耆大山間於1919年開業。1926年到達足立車站。 

另一方、倉敷 - 宍粟（現在的豪溪）間的伯備南線於1925年開業。以後順次延伸，1927年到達備中川面車站，備中川面 - 新見 - 立間開業全線通車是1928年的事。

以地區線規格的丙線來建設，但其後作為陰陽連絡線而被重視，新幹線岡山開業時，進行米子・松江方向短縮路線整備、電氣化、部份區間改良等工程。
