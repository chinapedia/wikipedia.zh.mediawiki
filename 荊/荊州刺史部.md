'''荊州刺史部'''，是漢朝十三州刺史部之一，州治在漢壽縣（今[[湖南省|湖南省]][[常德市|常德市]]東北），東漢末移治襄陽縣（今[[湖北省|湖北省]][[襄陽市|襄陽市]]）。漢代州域範圍大致是今日的[[河南省|河南省]]西南部，[[湖北省|湖北省]]、[[湖南省|湖南省]]大部及[[貴州省|貴州省]]、[[廣東省|廣東省]]、[[廣西壯族自治區|廣西壯族自治區]]邊緣。

== 名稱由來 ==
大體指[[漢江|漢江]]至[[衡山|衡山]]以南的大片區域。春秋戰國時期，百家爭鳴，著書立說，把禹時的九州冠以稱謂，荊州即其一。{{noteTag|九州即冀州、兗州、青州、徐州、揚州、荊州、豫州、幽州、雍州。}}《尚書·禹貢》載：「荊及衡陽惟荊州。」《爾雅·釋地》：「漢南曰荊州。」《周禮·職方》：「正南曰荊州。」《釋名》云：「荊，警也，南蠻數為寇逆，州道先強，當警備之也。其地北據荊山，南及衡山之陽。」《呂氏春秋》曰：「荊，楚也。」

== 西漢荊州刺史部 ==
西漢時，領南郡、南陽郡、衡山郡、江夏郡、長沙國、桂陽郡、武陵郡、零陵郡8個郡國。

== 東漢荊州刺史部 ==
東漢時，領南陽郡、南郡、江夏郡、零陵郡、桂陽郡、武陵郡、長沙郡7個郡國，東漢末置章陵郡、南鄉郡、襄陽郡、宜都郡、新城郡、漢昌郡、臨江郡、西陵郡、固陵郡等郡。
