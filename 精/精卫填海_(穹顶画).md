'''《精卫填海》'''，是一副直径40米、高21米，中间由10根石柱围成的600平方米圆拱形穹顶油画。位于[[天津站|天津站]]进站大厅的穹顶上，1988年由中国美术协会副主席、[[天津美术学院|天津美术学院]]教授秦征和其五位学生王玉琦、吴恩海、马元、王小杰、高冬联合绘制。

== 传说 ==
[[精卫填海|精卫填海]]原是《[[山海经|山海经]]》记叙的一则[[故事|故事]]，记述的是在[[中国|中国]]上古时期[[北方|北方]]有一座[[山|山]]叫发鸠山。树上有一只[[鸟|鸟]]。它的名字叫精卫。原来精卫是[[炎帝|炎帝]]宠爱的女儿，名字叫女娃。有一天她去[[东海|东海]]玩，可是突然风暴袭来，她死了。女娃死了以后，变成了鸟，名字就叫作“精卫鸟”（也叫“冤禽”）。精卫鸟嘴白脚红，头上有花纹，经常发出“精卫”、“精卫”的叫声，好像在呼喊着自己的名字。她去西山衔来石子儿和树枝，一次又一次投到大海-{zh:里; zh-hans:里; zh-hant:裡;}-，想要把东海填平。有人认为这里反映了古代人民同自然进行抗争的思想。

== 由来 ==
[[File:Tianjin_Station_03.jpg|thumb]]
1987年4月，[[天津站|天津站]]工程兴建，原设计的[[穹顶|穹顶]]为满天星，中央是一盏鎏金大吊灯，为此还特别向国家申请了2公斤黄金。1988年春，时任[[天津市市长|天津市市长]][[李瑞环|李瑞环]]在赴[[意大利|意大利]]考察时看到[[罗马|罗马]]的[[西斯廷教堂|西斯廷教堂]]上由[[米开朗基罗|米开朗基罗]]创作的著名穹顶画《[[创世纪_(壁画)|创世纪]]》之后，当即决定修改方案，在正在兴建的天津站穹顶上创作正规的写实油画。此后，由秦征创作方案提出的《精卫填海》被选中。<ref>[http://www.022net.com/2010/3-20/43164230247915-2.html 话说著名油画家秦征：嘛叫天津人的真格的] {{webarchive|url=https://web.archive.org/web/20100826071502/http://www.022net.com/2010/3-20/43164230247915-2.html |date=2010-08-26 }}</ref>

== 绘制过程和保护措施 ==
《精卫填海》的绘制工作从1988年5月开始。秦征和他的五个学生王玉琦、吴恩海、马园、王小杰、高冬，每天在20多米高的脚手架上工作。首先，他们在穹顶本身的玻璃纤维构造上交错裱上三层粗麻布，再刷上油底子作为画布的基础。之后，他们把积木盒子摞高，将10多把帆布躺椅摆到人与画面几乎平行的穹顶最高处，然后躺在躺椅上进行创作。<ref>[http://www.jwb.com.cn/bhzb/html/2010-05/31/content_498219.htm 秦征：艺术生命在于首创] {{webarchive|url=https://web.archive.org/web/20100826095336/http://www.jwb.com.cn/bhzb/html/2010-05/31/content_498219.htm |date=2010-08-26 }}</ref> 在画面构图上，画家突破西洋油画的[[透视|透视]]原理，让海、天、云、鸟与众多女神的后代，沿着一个圆形宇宙飞旋，创造出一个虚实相间，人神合一的空灵飘渺的神话世界。<ref>[http://news.enorth.com.cn/system/2009/11/06/004265883.shtml 著名油画家秦征：心诚爱愈深，大爱生大美]</ref> 此外，为了保护画作，施工人员将冷光灯放在穹顶一周的灯槽里，还特别增加了一倍的防水层，由原来的两层防水加固到了现在的4层防水，而且天津站被定为永久无烟站也是基于对壁画的保护。<ref>[http://www.022net.com/2007/2-1/424642112338954.html 天津站改造现场吸烟做饭 著名穹顶壁画受烤验] {{webarchive|url=https://web.archive.org/web/20100826072327/http://www.022net.com/2007/2-1/424642112338954.html |date=2010-08-26 }}</ref>
