'''財團法人精神健康基金會'''，是臺灣一個旨在推廣[[腦|腦]]科學知識以及精神[[健康促進|健康促進]]之[[非營利組織|非營利組織]] <ref>{{cite web|url=http://www.brainlohas.org/ |title=財團法人精神健康基金會網站|accessdate=2010-10-15}}</ref>。由[[台大醫院|台大醫院]]精神科[[胡海國|胡海國]]教授 <ref>{{cite web|url=http://haigohwu.pixnet.net/blog |title=心腦操練 精神樂活 胡海國醫師的部落格|accessdate=2013-08-15}}</ref>。及企業家劉俊<nowiki>杰</nowiki>等人創立，以普及[[腦|腦]]科學知識、[[精神疾病|精神疾病]]早期發現<ref>{{Cite web |url=http://www.mhf.org.tw/mhball/ |title=精神健康風向球 |accessdate=2010-10-22 |work=精神健康基金會}}</ref>、精神障礙者關懷<ref>{{Cite web |url=http://web.pts.org.tw/~web01/hope/25.htm |title=走出心靈煉獄－精神障礙者需要的關懷與協助 |accessdate=2010-10-22 |work=[[公共電視文化事業基金會|公共電視文化事業基金會]]，CH13公共電視網頁，發現希望，第廿五集}}</ref>為宗旨。由於社會大眾對精神疾病常有[[偏見|偏見]]，該基金會提出「保養頭腦、免於煩惱」<ref>{{cite journal |author=胡海國 |title=保養頭腦，免除煩惱|journal=健康世界，月刊 |volume=190 |pages=95–97 |year=2001 }}</ref>口號，旨在說明腦功能障礙為各種精神疾病的主因。提倡「築心、築腦、築我」、「精神樂活」等做為拓展精神健康之道<ref>{{Cite web |url=http://www.nsc.gov.tw/scicircus/ct.asp?xItem=18095&ctNode=1415 |title=心腦操練、優築生命 |accessdate=2010-10-27 |work=2010智慧巡禮：腦與心智科學的生命教育講座（02/27～07/31，）科技大觀園，網頁 }}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，以[[心理健康促進|心理健康促進]]為導向，強調應該實踐健康生活、照顧保護頭腦，提升精神健康、預防精神疾病。該基金會活動為腦科學展覽、精神健康調查、大眾演講、精神健康志工（或稱為精神健康種子）訓練等。

==台灣精神健康日==
[[世界衛生組織|世界衛生組織]]為提高全世界對衛生及健康工作的關注，將每年的4月7日訂為[[世界衛生日|世界衛生日]]。每年於該日公布一個宣傳主題，世界各國則共同以此主題舉行相關活動，呼籲人們重視此健康議題。2001年世界衛生日即是以精神健康為主題，提出「停止歧視，勇於關懷」（Stop Exclusion，Dare to Care）<ref>{{Cite web|url= http://www.who.int/world-health-day/previous/2001/en/index.html |title= Stop Exclusion - Dare to Care |accessdate=2010-10-25 |work=WHO，World Health Day，2001，網頁}}</ref>為口號，強調精神健康的重要性及主動關懷[[精神疾病|精神疾病]]患者。

每年的10月10日則是由{{Link-en|世界心理衛生聯盟|World Federation for Mental Health}}所定之[[世界精神衛生日|世界精神衛生日]]。世界各國同樣於當日進行各種促進精神健康的宣導，呼籲民眾重視精神健康。

精神健康基金會以每年12月份的第3個星期六做為「台灣精神健康日」，於當日天舉辦「精神健康嘉年華」<ref>{{Cite web |url=http://tw.streetvoice.com/profile/home.asp?sd=454130 |title=Brain LOHAS 精神健康嘉年華 |accessdate=2010-10-25 |work=StreetVoice街聲，網頁 |deadurl=yes |archiveurl=https://web.archive.org/web/20081019000459/http://tw.streetvoice.com/profile/home.asp?sd=454130 |archivedate=2008-10-19 }}</ref>，邀請台灣民眾參與活動，提醒民眾應該每年都自我檢測精神是否健康。

==精神健康指數==
一種測量精神健康狀況的統計[[指數|指數]]，係使用精神健康基金會胡海國教授等學者所設計的問卷量表，以[[抽樣|抽樣]]調查方式施測。精神健康指數區分為「心身健康」、「個人價值」、「生活掌握」、「家庭健康 」四個層面，各層面分數以100分為最高分，四項分數總合的[[平均數|平均數]]即為精神健康指數的整體得分。該指數可以用來呈現某個[[族群|族群]]、[[社區|社區]]或[[國家|國家]]之中民眾的精神健康狀況是否良好。與{{link-en|快樂星球指數|Happy Planet Index}}、[[国民幸福总值|国民幸福总值]]的概念相同，可以做為國民生活幸福感的指標。

==思覺失調症==
某些精神疾病在初期階段，會出現思想混亂和感覺異常的情形，因症狀輕微，易被忽略，誤以為只是生活適應困難、暫時的情緒憂鬱、容易緊張，或是注意力不集中，未能就醫。2006年，由精神健康基金會及臺大醫院精神科醫療團隊提出「思覺功能障礙」一詞，<ref>{{Cite web |url=http://lifebell.blog131.fc2.com/ |title=思覺功能障礙 |accessdate=2011-03-22 |work=認識思覺障礙學習網}}</ref>以利早期偵測精神疾病和{{link-en|早期介入治療|Early intervention in psychosis}}。

於2014年中，精神健康基金會與台灣精神醫學會<ref>{{Cite web |url=http://www.sop.org.tw/ |title=台灣精神醫學會}}</ref>及中華民國康復之友聯盟<ref>{{Cite web |url=http://www.tamiroc.org.tw/ |title=中華民國康復之友聯盟}}</ref>等團體共同推動下，衛生福利部正式宣布將[[精神分裂症|精神分裂症]]更名為「思覺失調症」<ref>{{Cite web |url=https://www.facebook.com/schizophrenialohas |title=思覺失調症支持聯盟}}</ref>，國內的教科書、醫院的疾病診斷，以及社會福利使用的名稱，也都將更改為「思覺失調症」。依據精神健康基金會的調查發現，在台灣2014年聽過「思覺失調症」的人大約為25%，2016年為37.7%。

對此類的病狀，在香港所使用的名詞為[[思覺失調|思覺失調]]，意即精神病的早期徵兆。日本則是改用「統合失調症」，來取代原本的精神分裂症，以降低對精神病患的污名化。

==心腦操練==
精神健康基金會董事長胡海國教授於2006年首先提出的概念。從腦的特質及生活實際體驗為基礎，主張平時應保養頭腦、鍛練腦力。大腦有可塑性，但腦也有其限制（有限性），了解腦的特性後，於日常生活充實經驗、鍛鍊大腦迴路、強化韌性。當面對逆境時，即能承受壓力、展現{{link-en|心理復原|Psychological resilience|復原力}}。透過「心－腦」操練累積智慧，且能關懷他人、豐富人生。

這些方法使用實際生活中，即為「心腦操練」。胡海國教授歷年的論述整理集結之著作為「心腦操練－人生新主張」。

==心腦治療==
將心腦操練的方法，發展成為醫療可應用的模式，即為「心腦治療」。將腦神經科學的理論，轉化為臨床醫療可運用的法則，教導就醫患者及家屬，瞭解腦力有限、應保養頭腦（腦的有限性）」。並依患者差異給予個別建議，符合其個人生活與疾病狀況、確實可行、可持續使用（操練）的指導建議。「心腦治療」經由學習、改變（腦的可塑性）、經驗累積，促成症狀改善
