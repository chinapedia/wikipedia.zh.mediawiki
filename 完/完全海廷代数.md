在[[数学|数学]]特别是[[序理论|序理论]]中，'''完全海廷代数'''是作为[[完全性_(序理论)|完全]][[格_(数学)|格]]的[[海廷代数|海廷代数]]。完全海廷代数是三个不同[[范畴论|范畴]]的[[对象_(范畴论)|对象]]，它们是范畴'''CHey'''，locales的范畴'''Loc'''，它的[[对偶性_(范畴论)|对偶]]frames的范畴'''Frm'''。

== 定义 ==

考虑是[[完全格|完全格]]的[[偏序集合|偏序集合]]（''P'', ≤）。则''P''是完全海廷代数，如果任何下列等价条件中的一个成立：
* ''P''是海廷代数，就是说运算 ( ''x'' <math>\wedge</math> - )有一个[[伴随函子|右伴随]]（也叫做（单调）[[伽罗瓦连接|伽罗瓦连接]]的下伴随），对于每个''P''的元素''x''。
*对于所有''P''的元素''x''和所有''P''的子集''S''，下列无限[[分配律|分配律]]成立：
: <math>x \wedge \bigvee S = \bigvee \{ x \wedge s \mid s \in S \}</math>
* ''P''是分配格，就是说对于所有''P''中的''x'', ''y''和''z''，有着
: <math>x \wedge ( y \vee z ) = ( x \wedge y ) \vee ( x \wedge z )</math>
:并且''P''是交连续性的，就是说交运算 ( ''x'' <math>\wedge</math> - )对于所有''P''中的''x''是[[斯科特连续性|斯科特连续性]]的。
