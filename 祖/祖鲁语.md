'''祖鲁语'''（祖鲁语：''isiZulu''）是[[祖鲁|祖鲁]]人所使用的一種[[语言|语言]]。目前大约有九百万人使用，其中95%居住在[[南非|南非共和国]]境内。祖鲁语是24%的南非人的母語，也是南非最大的语言。大约50%的南非人可以听懂祖鲁语(Ethnologue 2005)。在[[种族隔离制度|种族隔离]]结束後的1994年，祖鲁语成为[[南非|南非]]的11种官方语言之一。

== 地理分布 ==

祖鲁语属[[班图语支|班图]]语言的东南支，亦称[[Nguni|Nguni]]组。

祖鲁语也在夸祖鲁-纳托尔省与豪登省广泛使用。[[莱索托|莱索托]]与[[斯威士兰|斯威士兰]]也有少数祖鲁语使用者。 [[津巴布韦|津巴布韦]]的[[Ndebele语言|Ndebele语言]],与[[马拉维|马拉维]]的[[Ngoni语言|Ngoni语言]]是祖鲁语的亲戚。三者都可追寻到[[十九世纪|十九世纪]]的祖鲁移民。

# 在一般情況下，普通的清塞音與清塞擦音實際上都是[[擠喉音|擠喉音]]，音標分別寫作{{IPA|[pʼ]}}、{{IPA|[tʼ]}}、{{IPA|[kʼ]}}、{{IPA|[tsʼ]}}、{{IPA|[tʃʼ]}}、{{IPA|[kxʼ]}}。
# 當前方不接鼻音時，{{IPA|/ɠ/}}和{{IPA|/k/}}、{{IPA|/kʰ/}}二者幾乎成互補分配。{{IPA|/k/}}、{{IPA|/kʰ/}}幾乎只出現在字尾，而{{IPA|/ɠ/}}只出現在字中。從其他語言借入的外來語則不受此限制，{{IPA|/k/}}和{{IPA|/kʰ/}}可能出現於其他位置，如''isekhondi'' {{IPA|/iːsekʰoːndi/}}「第二」，''ibhayisikili'' {{IPA|/iːbajisikiːli/}}「自行車」。
# [[弛聲|弛聲]]輔音皆為{{tsl|en|depressor consonant|降調輔音}}，會使該音節的音調較低。
# 輔音{{IPA|/ŋ/}}出現在某些方言中，作為輔音叢{{IPA|/nɡ/}}不在[[詞幹|詞幹]]首時的簡略發音，且{{IPA|/ŋ/}}總是弛聲音。
# 顫音{{IPA|/r/}}並非祖魯語原有的輔音。這個音只出現於從歐洲語言借入的詞彙中。

祖魯語中[[搭嘴音|搭嘴音]]的使用是這個語言的一大特色，其中搭嘴音可以依[[發音部位|發音部位]]分為三類：
* [[齒搭嘴音|齒-齒齦]] {{IPA|/ǀ/}}，相當於吸吮牙齒，發出「tsk tsk」的聲音。
* [[齒齦後搭嘴音|齒齦]] {{IPA|/!/}}，出的聲音類似瓶蓋「啵」打開的聲音。
* [[齒齦邊搭嘴音|齒齦邊]] {{IPA|/ǁ/}}，發出的聲音相當於人在溜馬時發出的聲音。

三種部位的搭嘴音又可以因送氣、鼻音、弛聲等特徵區分出五個搭嘴音，故祖魯語的搭嘴音共計有15種。

== 短语 ==
以下列出几句常用的祖鲁短语，可供参考：
* Sawubona (向一个人问'''你好''')
* Sanibonani (向一组人问'''你好''')
* Unjani? (向一个人问'''你好吗？''')
* Ninjani? (向一组人问'''你好吗？''')
* Ngiyaphila ('''我很好''')
* Ngiyabonga ('''（我）谢谢你''')
* Siyabonga ('''（我们）谢谢你''')
* Isikhathi sithini? ('''现在几点?''')
* Uhlala kuphi? ('''你在哪住?''')
* Eish！ ('''哇！''')
