'''祖沖之'''（{{bd|429年||500年|catIdx=Z}}），[[表字|字]]'''文远'''，[[范阳郡|范阳郡]][[逎县|逎县]]（今[[河北省|河北省]][[保定市|保定市]][[涞水县|涞水县]]）人，[[刘宋|刘宋]]时代[[数学家|数学家]]、[[天文学家|天文学家]]。祖冲之的主要成就在[[数学|数学]]、[[天文|天文]][[历法|历法]]和[[机械|机械]]制造三个领域。祖冲之的儿子[[祖暅之|祖暅之]]也是数学家。

== 生平 ==
祖家歷代都對天文历法素有研究，祖冲之从小就有机会接触[[天文|天文]]、[[数学|数学]]知识。祖冲之青年时，就得到博学多才的名声，[[宋孝武帝|宋孝武帝]]听说后，派他到“[[华林学省|华林学省]]”做研究工作。461年，他在[[南徐州|南徐州]]（今[[江苏|江苏]][[镇江|镇江]]）刺史府裏从事，先后任南徐州从事史、公府参军。公元464年他调至娄县（今[[江苏|江苏]][[昆山|昆山]]东北）任县令。在此期间他编制了《[[大明曆_(祖沖之)|大明曆]]》，计算了[[圆周率|圆周率]]。劉宋末年，祖冲之回到建康任[[谒者仆射|谒者仆射]]，此后直到劉宋灭亡一段时间后，他花了较大精力来研究机械制造。494年到498年之间，他在[[南齐|南齐]]朝廷担任[[长水校尉|长水校尉]]一职，受四品俸禄。

== 数学贡献 ==
在数学上，祖冲之研究过《[[九章算术|九章算术]]》和[[刘徽|刘徽]]所做的注解，给刘徽的《[[重差|重差]]》作过注解。他还著有《[[缀术|缀术]]》一书，汇集了祖冲之父子的数学研究成果。这本书内容深奥，以至“学官莫能究其深奥，故废而不理”。《缀术》在[[唐代|唐代]]被收入《[[算经十书|算经十书]]》，成为唐代国子监算学课本，当时学习《缀术》需要四年的时间，可见《缀术》的艰深。《缀术》曾经传至[[朝鲜|朝鲜]]和[[日本|日本]]，但到[[北宋|北宋]]时这部书就已轶失。人们只能通过其他文献了解祖冲之的部分工作：在《[[隋书|隋书]]·律曆志》中留有小段祖冲之关于圆周率工作的记载；唐代[[李淳风|李淳风]]在《九章算术》注文中记载了祖冲之和儿子祖暅求球体积的方法。祖冲之还研究过“开差幂”和“开差立”问题，涉及[[二次方程|二次方程]]和[[三次方程|三次方程]]的求根问题。遗留下来的祖冲之的数学贡献主要有他对[[圆周率|圆周率]]的计算结果和球体体积的计算公式。


=== 计算圆周率 ===
{{Main|密率}}
{{See also|圆周率}}

据《[[隋书|隋书]]·律曆志》<ref>https://zh.wikisource.org/wiki/%E9%9A%8B%E6%9B%B8/%E5%8D%B716</ref>记载，祖沖之以「以直徑一億為一丈，圓周盈數三丈一尺四寸一分五厘九毫二秒七忽，朒數三丈一尺四寸一分五厘九毫二秒六忽，正數在盈朒二限之間。密率，圓徑一百一十三，圓周三百五十五。約率，圓徑七，週二十二。」，以此为直径求[[圆周率|圆周率]]，求得盈数（即过剩的近似值）为<math>3.1415927</math>；肭数（即不足的近似值）为<math>3.1415926</math>，圆周率的真值介于盈肭两数之间。《隋书》没有具体说明祖冲之是用什么方法计算出盈肭两数的。一般认为，祖冲之采用的是[[割圆术_(刘徽)#祖率|刘徽割圆术]]分割到24576边形，又用[[割圆术_(刘徽)#刘徽圆周率不等式|刘徽圆周率不等式]]得祖冲之著名的圆周率不等式：
<math>3.1415926 < \pi < 3.1415927</math>。祖冲之的这一结果精确到小数点后第7位，直到一千多年后才由15世纪的[[阿拉伯|阿拉伯]]数学家[[阿尔·卡西|阿尔·卡西]]以17位有效数字打破此记录<ref>吴文俊 主编 《中国数学史大系》副卷第一卷 480页  ISBN 7-303-05292-5/O</ref>。

按照当时计算使用分数的习惯，祖冲之还采用了两个分数值的[[圆周率|圆周率]]：“约率”<math>\tfrac{22}{7}</math>（或称之为“疏率”{{NoteTag|陈仁政在《说不尽的π》一书中提到，“疏率”是华罗庚在《数学是我国人民所擅长的学科》（1951年2月10日《人民日报》第3版）一文中使用的误称，其本人已在1962年的《从祖冲之的圆周率谈起》一书中予以纠正，然此误称依然流传于众多媒体<ref>{{cite book |title = 说不尽的π |page = 第18页 |chapter = 约率‘摇身一变’成‘疏率’ |author = 陈仁政 |publisher = 科学出版社 |location = 北京 |year = 2005 |ISBN = 9787030146359}}</ref>。}}）以及“密率”<math>\tfrac{355}{113}=3.141592920354</math>。在分母<16600的所有整分数中，密率的比值最接近圆周率{{NoteFoot|<math>\frac{52163}{16604}=3.1415923874</math>才比祖率略准。}}。祖冲之可能利用[[何承天_(南朝)|何承天]]的[[调日法#圆周率约率和密率|调日法]]求得圆周率的约率和密率<ref>[[吴文俊|吴文俊]] 主编 《中国数学史大系》第四卷 123页 ISBN 7-300-0425-8/O</ref>。数学家[[华罗庚|华罗庚]]曾认为密率的求得，说明祖冲之可能已经掌握了[[连分数|连分数]]的概念。日本数学家[[三上义夫|三上义夫]]说，“约率<math> \pi=\tfrac{22}{7}</math>，无非是几百年前[[希腊|希腊]]数学家[[阿基米德|阿基米德]]已经得到的数值，但是<math> \pi=\tfrac{355}{113}</math> 这个分数，却是翻遍古希腊，古印度和阿拉伯的数学文献都找不到的分数，希腊人肯定不知道它；在[[欧洲|欧洲]]直到1586年才由[[荷兰|荷兰]]人安托尼斯宗（Adriaan Anthoniszoon）求出了<math>\tfrac{355}{113}</math>这个比值。因此，中国人掌握这个非凡的圆周率分数比欧洲早出整整一千年之久”。为纪念这位伟大的中国古代数学家，[[三上义夫|三上义夫]]要求把<math>355 \over 113</math>称为“祖率”<ref>“We are on this account strongly urged to express a desire that it should henceforth be called by the name of Tsu Ch'ong-chih's fractional value for π” Yoshio Mikami, Development of Mathematics in China and Japan p50 1913 Leipzig</ref>。

=== 计算球体体积 ===
祖冲之还和儿子祖暅之一起，用巧妙的方法解决了球体体积的计算问题。

《九章算术》中认为，球体的外切圆柱体积与球体体积之比等于正方形与其内切圆面积之比，刘徽在他为《九章算术》作的注释中指出，原书的说法是不正确的，只有“[[牟合方盖|牟合方盖]]”（垂直相交的两个圆柱体的共同部分的体积）与球体积之比，才正好等于正方形与其内切圆的面积之比。但刘徽没有给出“牟合方盖”的体积公式，所以也就得不出球体的体积公式。

祖冲之父子采用“幂势既同，则积不容异。”（即“等高处横截面积常相等的两个立体，其体积也必然相等”）这一原理，求出了“牟合方盖”的体积，而球体体积等于<math>\pi \over 4</math>乘以“牟合方盖”体积，从而最终算出球体积为<math>\pi d^3 \over 6</math>（<math>d</math>为球直径）。

祖冲之父子所采用的“幂势既同，则积不容异”这一原理，在[[欧洲|欧洲]]由[[意大利|意大利]]数学家[[卡瓦列里|卡瓦列里]]于17世纪重新发现，所以西文文献一般称该原理为[[卡瓦列里原理|卡瓦列里原理]]。为了纪念祖冲之父子发现这一原理的重大贡献，人们也称该原理为“[[祖暅原理|祖暅原理]]”。

== 天文历法贡献 ==
祖冲之在天文历法方面的成就，大都包含在他所编制的《[[大明曆_(祖沖之)|大明曆]]》及为《大明曆》所写的《驳议》中。

在祖冲之之前，人们使用的历法是天文学家[[何承天|何承天]]编制的《[[元嘉曆|元嘉曆]]》。祖冲之经过多年的观测和推算，发现《元嘉曆》存在很大的差误。于是祖冲之着手制定新的历法，[[宋孝武帝|宋孝武帝]][[大明|大明]]六年（公元462年）他编制成了《大明曆》。大明曆在祖冲之生前始终没能采用，直到[[梁武帝|梁武帝]][[天监|天监]]九年（公元510年）才正式颁布施行。《大明曆》的主要成就如下：
* 区分了[[回归年|回归年]]和[[恒星年|恒星年]]，首次把[[岁差|岁差]]<ref>https://zh.wikipedia.org/wiki/%E6%AD%B2%E5%B7%AE_(%E5%A4%A9%E6%96%87)</ref>引进历法，测得岁差为45年11月差一度（今测二分點约为70.713年沿黃道西行差一度）<ref>[[劉操南|劉操南]] 著 《古代天文曆法釋證》310頁 ISBN 978-7-308-06707-2</ref>。岁差的引入是中国历法史上的重大进步。
* 定一个回归年为365<math>{15241 \over 62769}</math>（=365.24281481日，今测为365.2421988日），直到[[南宋|南宋]][[宋宁宗|宁宗]][[庆元|庆元]]五年（公元1199年）[[杨忠辅|杨忠辅]]制[[统天曆|统天曆]]以前，它一直是最精确的数据。
* 采用391年置144闰的新[[章_(曆法)|闰周]]，比以往历法采用的19年置7闰的闰周更加精密。
* 定[[交点月|交点月]]日数为27<math>{5598 \over 26377}</math>（=27.21223日，今测为27.212225日）。交点月日数的精确测得使得准确的日月食预报成为可能，祖冲之曾用大明曆推算了从元嘉十三年（公元436年）到大明三年（公元459年），23年间发生的4次月食时间，结果与实际完全符合。
* 得出[[木星|木星]]每84年超辰一次的结论，即定[[木星|木星]]公转周期为11.858年（今测为11.862年）。
* 给出了更精确的五星会合周期，其中[[水星|水星]]和木星的会合周期也接近现代的数值。
* 提出了用[[圭表|圭表]]测量正午太阳影长以定[[冬至|冬至]]时刻的方法。

== 机械制造贡献 ==
祖冲之还曾设计制造过许多精巧的机械，在文献《[[南齐书|南齐书]]·祖冲之传》和《[[南史|南史]]·祖冲之传》中有所记载。他曾经设计制造过利用水力舂米、磨面的[[水碓磨|水碓磨]]；重新铸造了当时已经失传了的[[指南车|指南车]]，随便车子怎样转弯，车上的[[铜|铜]]人总是指着南方；制造了"千里船"，在新亭江（在今南京市西南）上试航过，一天可以航行一百多里。他还设计制造过计时仪器[[漏壶|漏壶]]和[[欹器|欹器]]。

== 紀念 ==
为纪念祖冲之：
* [[月球|月球]]背面的一座[[环形山|环形山]]被命名为「[[祖冲之环形山|祖冲之环形山]]」。
* [[小行星1888|小行星1888]]被命名为「祖冲之小行星」。
* [[上海|上海]][[浦东|浦东]][[张江高科技园区|张江高科技园区]]内有一条城市道路命名为「[[祖冲之路|祖冲之路]]」。
* [[深圳|深圳]][[龍崗區|龍崗區]][[坂田街道|坂田街道]]有一條道路命名為「沖之大道」
* 上海[[同济大学|同济大学]][[文远楼|文远楼]]（祖冲之[[表字|字]]文远）
* [[莫斯科大学|莫斯科大学]]大礼堂前的走廊壁上，用彩色大理石镶嵌着祖冲之像。
* 河北保定有一所以祖冲之命名的中学，有江泽民手书「河北[[祖冲之中学|祖冲之中学]]」石碑。

此外还有祖冲之邮票，祖冲之纪念银币等纪念品。
