'''镜子超人REFLEX'''（{{lang|ja|ミラーマンREFLEX}}，{{lang|ja|ミラーマンリフレックス}}）是在2006年发售的特撮作品<ref name = "全怪獣図鑑">{{Cite book|language=ja |author = 大石真司 |coauthors = 江口水基・島崎淳・間宮尚彦 |year = 2013 |title = 円谷プロ全怪獣図鑑 |publisher = 小学館 |page = 352 - 353 |isbn = 9784096820742 }}</ref>。全3話。

== 故事簡介 ==
鏡子所映照出的是擁有靈魂的影子，鏡子的另一邊是幽冥的世界，邪惡的咒語會將歸神的邪仙召喚而來，少女百襲與影山鑑2人將對抗邪仙的陰謀，肩負著退治邪仙的使命。

== 登場人物==
镜子超人REFLEX 
* 身長：1.9 - 40米<ref name = "全怪獣図鑑" />
* 体重：70千克 - 3万5千吨<ref name = "全怪獣図鑑" />
自古就存在的審神者，專門拔除邪仙的武神。

武器是古代剣与镜刃{{refnest|『宇宙船YEARBOOK2006』では斧<ref name = "宇宙船">{{Cite book|language=ja |year = 2006 |title = 宇宙船YEARBOOK2006 |publisher = 朝日ソノラマ |page = 42 |isbn = 4257130865 }}</ref>。、『円谷プロ全怪獣図鑑』では半円の短剣<ref name = "全怪獣図鑑" />と記載している。}}
