'''《镜花缘》'''是一部根据中国古典名著《[[镜花缘|镜花缘]]》改编的中国动画（木偶片），于1991年首播。<ref>{{cite web |title=镜花缘 |url=https://movie.douban.com/subject/1888063/ |website=豆瓣电影 |accessdate=2018-11-08}}</ref><ref>{{cite web |title=木偶片《镜花缘》即将被拍影院版 |url=http://tech.163.com/06/0927/15/2S1OT71000091QMN.html |website=网易 |accessdate=2018-11-08}}</ref>影片第4集《两面国》获1991年度[[广播电影电视部|广播电影电视部]]优秀影片奖和1993年第二届中国影视动画展播荣誉奖。

==剧情简介==
唐朝秀才[[唐敖|唐敖]]考取了[[探花|探花]]，和反对[[武则天|武则天]]称帝的[[徐敬业|徐敬业]]、[[骆宾王|骆宾王]]是把兄弟，功名被革去，无奈中随大舅子[[林之洋|林之洋]]、老船工多九公出海远洋，途中见识到了君子国、女儿国、两面国等若干光怪离奇的国家及其风土人情。
