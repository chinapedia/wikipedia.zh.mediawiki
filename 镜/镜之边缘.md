{{vgname|镜之边缘|en=Mirror's Edge|cn=鏡之邊緣|tw=靚影特務|aka=暗镜迷踪}}是由[[EA_Digital_Illusions_CE|EA DICE]]开发，并由[[艺电|艺电]]发行的[[第一人稱視角遊戲|第一人称]][[动作冒险游戏|动作冒险游戏]]，这也是史上第一款以[[跑酷|跑酷]]为题材制作的电子游戏<ref>{{cite web| date=2008-03-25| title=史上首款跑酷游戏《镜之边缘》GS前瞻| work=太平洋游戏网| url=http://pc.pcgames.com.cn/pcnews/newgame/0803/1040699.html| accessdate=2015-06-23| deadurl=no| archiveurl=https://web.archive.org/web/20150623100847/http://pc.pcgames.com.cn/pcnews/newgame/0803/1040699.html| archivedate=2015-06-23}}</ref>。该游戏公布於2007年7月10日，并於2008年11月在[[PlayStation_3|PlayStation 3]]和[[Xbox_360|Xbox 360]]的平台发行，[[Games_for_Windows|Windows]]版本则在2009年1月13日发行。游戏采用[[虚幻引擎3|虚幻引擎3]]，并加入了由[[Illuminate_Labs|Illuminate Labs]]和DICE联合开发光照新解决方案。

游戏颜色风格亮丽，不同於其他大多数第一人称游戏，其拥有更多的动作（如滑越障碍、攀墙、撞门、过独木桥、爬水管、荡绳、翻滚等），以及更大的活动自由，例如没有[[抬頭顯示器|平视显示器（HUD）]]，而且角色的腿部、手臂和躯干在屏幕上是可见的。游戏背景设定在一个言论被极权政府重度监控的社会，而包括主角绯丝（{{lang|en|Faith}}）在内的“信使”（{{lang|en|runner}}）网络则被用来逃避政府监督以传递信息。作为3D[[平台游戏|平台游戏]]，玩家需要操作主角绯丝运用源于跑酷的动作在都市中飞檐走壁、翻越障碍和通-{}-过通风井。

《镜之边缘》获得了较多的正面评价，其Windows版在[[Metacritic|Metacritic]]的汇总得分为81%。游戏的独创性和其广阔的环境均获得一致好评，而批评则集中于剧情的乏力，[[试错|试错性]]的玩法和过短的流程。本作赢得了[[年度互动成就奖|年度互动成就奖]]年度最佳冒险游戏。由瑞典歌手丽莎·密丝柯维斯基演唱的游戏主题曲《Still Alive》于2007年发售混音[[原声带|原声带]]。移动平台的同名横向卷轴《镜之边缘》于2010年4月1日登陆[[iPad|iPad]]，2010年在9月2日登陆[[iPhone|iPhone]]。其[[Windows_Phone|Windows Phone]]移植版发布于2012年7月13日，最初为[[诺基亚Lumia|诺基亚Lumia]]手机独占。官方于[[E3|E3 2013]]正式宣布其续作计划。

== 玩法 ==

[[File:Runnervision.jpg|thumb]]

在《镜之边缘》中，玩家以第一人称视角控制主角绯丝，在光鲜耀眼的城市屋顶间跳跃，墙垣上奔跑，并利用通风井进入各个建筑。这些动作技术都源自[[跑酷|跑酷]]这项运动<ref>{{cite web |author= |url=http://www.youtube.com/watch?v=8zZKRVkZvtA |title=Mirror's Edge interview by Gamereactor |publisher=YouTube |accessdate=2012-07-15 |deadurl=no |archiveurl=https://web.archive.org/web/20130727134224/http://www.youtube.com/watch?v=8zZKRVkZvtA |archivedate=2013-07-27 }}</ref>。[[游戏制作人|高级制作人]]欧文·奥布赖恩（{{lang|en|Owen O'Brien}}）表示，《镜之边缘》旨在“传达紧张感和与环境的身体接触”，并为玩家提供在第一人称游戏中史无前例的行动自由<ref name="1Upprev1">{{cite web |last=Suttner |first=Nick |publisher=1UP.com |title=''Mirror's Edge'' Preview |url=http://www.1up.com/do/previewPage?cId=3166742 |date=2008-02-29 |accessdate=2008-05-08 |deadurl=yes name="ignprev">{{cite web |last=Ahearn |first=Nate |publisher=IGN |title=Mirror's Edge First Look |url=http://xbox360.ign.com/articles/855/855848p1.html |date=2008-02-29 |accessdate=2008-05-08 }}</ref>。同样地，游戏玩家可以充分运用手足甚至躯干来完成动作，例如绯丝可以用手控制泵阀，加速跑时步幅会变大，跳远时手脚会随之晃动等等<ref name="ignprev" /><ref name="Edge">{{cite web|last=Plunkett |first=Luke |publisher=Kotaku |title=''Mirror's Edge'' Sounds [[File:Mirrors-Edge-scr1.jpg|thumb]]和[[肉搏战|肉搏战]]]]
由於运动是游戏的重點，因此在游戏中维持[[衝量|冲力]]就变得至关重要，玩家需要完成一整套行云流水般的动作<ref name="runnervision">{{cite web|last=Pellett |first=Matthew |publisher=[[Computer_and_Video_Games|CVG]] |title=Preview: Mirror's Edge |url=http://www.computerandvideogames.com/article.php?id=188249 |date=2008-05-04 |accessdate=2008-05-08 |archiveurl=https://web.archive.org/web/20080510113638/http://www.computerandvideogames.com/article.php?id=188249 |archivedate=2008-05-10 |deadurl=yes }}</ref>，倘若有所不连贯便会引起冲力的损失，这可能就会导致绯丝的失足，或者因为达不到指定的冲力而够不着某些物体<ref name="euro">{{cite web |last=Bramwell |first=Tom |publisher=[[Eurogamer|Eurogamer]] |title=Mirror's Edge Preview |url=http://www.eurogamer.net/article.php?article_id=93740 |date=2008-02-29 |accessdate=2008-05-08 |deadurl=no |archiveurl=https://www.webcitation.org/68DE3dnCm?url=http://www.eurogamer.net/articles/mirrors-edge-preview |archivedate=2012-06-06 }}</ref>。游戏的控制会因具体情况而相应简化，“向上”的按钮可以控制绯丝在穿行时跨越障碍，例如跳跃、攀爬或抓住飞索等物件，而“向下”按钮则可控制她执行其他动作，如下滑、翻滚或仅仅是下蹲<ref name="euro" /><ref name="ars">{{cite web |last=Kuchera |first=Ben |publisher=Ars Technica |title=GDC: First peek at Mirror's Edge; don't look down! |url=http://arstechnica.com/journals/thumbs.ars/2008/02/29/notes-for-embargoed-mirrors-edge |date=2008-02-29 |accessdate=2008-05-08 |deadurl=no |archiveurl=https://www.webcitation.org/68DE5LoU5?url=http://arstechnica.com/gaming/2008/02/notes-for-embargoed-mirrors-edge/ |archivedate=2012-06-06 }}</ref>。为了帮助玩家完成这一连串动作，游戏采用了“信使视角”这一系统，顾名思义，该系统可以呈现信使所能看到的环境，以便立即找到逃生路线。系统会高亮显示对游戏进行有益的环境物件，比如说，管道、斜坡和可进入的门在绯丝行进中会变为红色，尽管它显示的未必就是最优路线<ref name="euro" />。随着游戏的深入，这类视觉提示的数量会逐渐减少到只显示最终目标，当然玩家也可以选择完全关闭该提示系统<ref name="runnervision" />。系统还能被用来制造难题，让玩家想出如何利用高亮物件组合成一连串的动作，以达到目的地<ref name="euro" />。“反应时间”系统是另一个协助玩家的手段，这种类似於[[子弹时间|子弹时间]]的形式能使玩家在不失去冲力的前提下，有时间来考虑接下来怎么走，也能让跳跃和缴械的时机掌握变得更为容易<ref name="ars" />。

虽然玩家角色可以持有武器，但奥布赖恩强调，“这是一款动作冒险游戏，我们没有把它定位成射击游戏——其重点不在於枪，而在於人。”《镜之边缘》的游戏性更侧重於通过游戏环境寻找最优路线，而战斗仅仅处於次要地位。事实上，在E3展“游戏攻略”上奥布莱恩表示，有一项解锁成就正是不射杀任何敌人通关<ref name="1Upprev1" />。正因如此，游戏中枪支虽然可以通过对敌人缴械获得，但子弹用尽时就必须丢弃<ref name="runnervision" />。此外，携带武器还会拖累绯丝的行动，而且枪越沉，阻碍就越大，这便使游戏拥有了策略元素，以决定何时该牺牲灵活性换取短时的火力<ref name="1Upprev1" /><ref name="euro" />。

为了延长游戏时间，每一关都隐藏有3个包裹，玩家找到後可以解锁特典和称为“时间试炼”的[[时间竞速|时间竞速]]模式。在该模式中，玩家需要尽可能快地完成一关中的一小部分，在线排行榜和红色的“影子”可以让玩家看到他们的最快路线或排行榜上其他人的路线。

== 剧情 ==

=== 设定 ===

《镜之边缘》的故事发生在一座不知名的“乌托邦”城，生活舒适，犯罪几乎销声匿迹。但是这种和谐的城市状态是由专制[[极权|极权]]体制换来的，政府为了减少犯罪和镇压反抗，监控着所有通信，并操控媒体，司法审判和民主体制形同虚设。游戏设定的18年前，当局曾开火镇压对其统治的抗议，造成平民丧生<ref name="sickness">{{cite web |last=Totilo |first=Stephen |publisher=MTV |title=EA Discusses 'Mirror's Edge' Sickness Concerns, Lack Of Color Green |url=http://multiplayerblog.mtv.com/2008/03/07/ea-discusses-mirrors-edge-sickness-concerns-lack-of-color-green |date=2008-03-07 |accessdate=2008-05-12 |deadurl=yes |archiveurl=https://www.webcitation.org/68DE9pHsA?url=http://multiplayerblog.mtv.com/2008/03/07/ea-discusses-mirrors-edge-sickness-concerns-lack-of-color-green |archivedate=2012-06-06 }}</ref>。游戏发生时即将举行市长选举，原市长卡拉汉（{{lang|en|Callaghan}}）寻求留任以确保政府能继续控制城市，而另一位备受青睐的候选人波普（{{lang|en|Pope}}）则承诺会带来变革。

高级制作人欧文·奥布莱恩表示：“（《镜之边缘》）探讨了你为了舒适的生活愿意放弃多少个人自由，这不只是一个女孩对警察政权独裁反抗的故事，而要比这微妙得多。”<ref name="gamasutra08">{{cite web|url=http://www.gamasutra.com/view/feature/3684/living_on_the_edge_dices_owen_.php|title=Living On The Edge: DICE's Owen O'Brien Speaks|author=Christian Nutt|publisher=[[Gamasutra|Gamasutra]]|date=2008-06-06|deadurl=no|archiveurl=https://www.webcitation.org/68DEBi19a?url=http://www.gamasutra.com/view/feature/3684/living_on_the_edge_dices_owen_.php|archivedate=2012-06-06}}</ref>美国电视剧《[[萤火虫_(电视剧)|萤火虫]]》和衍生电影《[[衝出寧靜號|衝出寧靜號]]》均是奥布莱恩的灵感来源。“我们的另一主题是你不能强迫别人按照你的规则和你的社会生活，哪怕你的社会更好”，他说，“《衝出寧靜號》里探员竟然说，‘这哪是什么邪恶帝国，我们只是不理解，你们为什么不愿意加入我们这个幸福之家。’但显然他们做得太过了，这也同样发生在了我们的游戏中。”<ref>{{cite web|url=http://www.gamasutra.com/view/feature/3684/living_on_the_edge_dices_owen_.php?page=4|title=Living On The Edge: DICE's Owen O'Brien Speaks|author=Christian Nutt|page=4|publisher=[[Gamasutra|Gamasutra]]|date=2008-06-06|deadurl=no|archiveurl=https://web.archive.org/web/20080609063849/http://www.gamasutra.com/view/feature/3684/living_on_the_edge_dices_owen_.php?page=4|archivedate=2008-06-09}}</ref>

=== 梗概 ===

游戏主角是24岁的绯丝·康纳斯（Jules de Jongh配音<ref>{{cite web|url=http://www.asiaarts.ucla.edu/090102/article.asp?parentid=102616 |publisher=UCLA Asia Institute |date=2009-01-02 |accessdate=2010-02-02 |title=Best of 2008: Asian-themed Video Games |last=Hong |first=William |archiveurl=https://web.archive.org/web/20101223190848/http://www.asiaarts.ucla.edu/090102/article.asp?parentid=102616 |archivedate=2010-12-23 |deadurl=yes }}</ref><ref>{{cite web|url=http://www.pastemagazine.com/articles/2008/12/review-mirrors-edge-xbox-360.html|work=''Paste''|date=2009-12-08|title=''Mirror's Edge'' (Xbox 360)|last=Killingsworth|first=Jason|accessdate=2010-02-04|deadurl=no|archiveurl=https://www.webcitation.org/68DECoGyY?url=http://www.pastemagazine.com/articles/2008/12/review-mirrors-edge-xbox-360.html|archivedate=2012-06-06}}</ref>），其右眼周围有着独特的纹身，游戏标识也与此类似。绯丝是一位接受过跑酷训练的“信使”，负责在城市传递革命集团间的实体邮件，以躲过电话和电子邮件等渠道的监控<ref name="1Upprev1" />。绯丝对于极权政府的态度植根于她的过去。绯丝年幼时，父母都是抗议运动的活跃分子，致力于防止城市沦为专制政权。她的母亲在失控的和平示威“十一月暴乱”中丧生，16岁的绯丝从此逃出家门，流浪街头成为飞贼<ref name="Mirror's Edge">{{cite video game|title=Mirror's Edge|developer=[[EA_Digital_Illusions_CE|EA Digital Illusions CE]]|publisher=[[Electronic_Arts|Electronic Arts]]|date=2008-11-12|platform=[[PlayStation_3|PlayStation 3]]/[[Xbox_360|Xbox 360]]/[[Microsoft_Windows|Microsoft Windows]]|version=|scene=|level=|quote=}}</ref>。绯丝和另一位“信使”莎兰（{{lang|en|Celeste}}）都是被老麦（{{lang|en|Mercury}}）训练出来的，老麦在任务中给予她们无线电支持<ref name="Gametrailers">{{Citation | url = http://www.gametrailers.com/videos/iw455d/mirror-s-edge-run-story-trailer | title = Mirror's Edge Video - Run Story Trailer | work = Gametrailers | date = 2008-10-16 | accessdate = 2015-06-23 | deadurl = no | archiveurl = https://web.archive.org/web/20150924122415/http://www.gametrailers.com/videos/iw455d/mirror-s-edge-run-story-trailer | archivedate = 2015-09-24 }}</ref>。

一天，绯丝收到了身为警察的妹妹凯特在波普办公室发出的求救，随後发现波普被谋杀了，死时握有一张写着“伊卡洛斯”字样的纸条。凯特希望绯丝能查明真相，并向她的上司米勒中尉汇报。绯丝发觉一名叫做阿杰（{{lang|en|Jacknife}}）的前信使可能掌握更多情报，他向绯丝提供了波普的安全负责人、前摔跤手火绳（{{lang|en|Ropeburn}}）的线索，但却不肯再透露更多。米勒中尉获知此事後，安排会见了火绳，但火绳在绯丝试图窃听时察觉了她，并想将其杀死。最终绯丝在搏斗中占了上风，把他悬在了屋顶上，打算讯问出更多情报，不想此时火绳却被一名神秘刺客抢先杀害了。绯丝追击着刺客，发现他受雇於一家私人保安公司，这座城市都非常仰赖於它的武装力量。该公司也正训练着自己的特警，以俘虏或杀害市内的信使。绯丝在油船上重新找到了刺客的行迹，并将其迫至一隅，最终发现这名刺客居然是莎兰。莎兰承认由於“伊卡洛斯计划”，为了活命，她已转投对方来摆脱不断被安全部队追捕的生活。莎兰担心被绯丝捉住後可能会被捕，便逃之夭夭了<ref name="大软">WerlinPinksfatt．镜-{}-之边缘．大众软件．2009，4月上（291期）：117-118页。</ref>。

老麦告诉绯丝，凯特已经被控以谋杀波普之罪，并安排地点让绯丝伏击押解她入监的车队。绯丝得以成功地解救了凯特，并让她逃往老麦待命的藏身处，而绯丝则负责将警察引开。摆脱追逐之後，绯丝回到了藏身处，却发现那儿遭到了袭击，不仅凯特失踪，老麦也已奄奄一息。老麦告诉绯丝，凯特已被带往沙德——一栋作为市长卡拉汉的居所和城市的集中监控数据库的摩天大楼。米勒中尉认为绯丝是城市变革的希望，在他的帮助下，绯丝得以进入服务器机房并破坏了服务器，解开封锁到达屋顶。然而此间，米勒的无线电支持却突然被切断，并传来了一声枪响。在屋顶上，绯丝发现用枪挟持着凯特的竟是阿杰。阿杰解释说，“伊卡洛斯计划”是由卡拉汉一手策划的，旨在消灭市内所有信使，并掐断不受控制的信息流动，而他就像莎兰一样，仅仅是为了活下去而染指其中。阿杰企图乘坐待命的直升机挟凯特逃跑，但绯丝紧跟着抓住了直升机，跳进了里面并把阿杰踢了下去。阿杰坠落时開槍擊中了直升機旋翼，而駕駛也遭到隨後趕來的特警開槍擊中，直升机因而失控。绯丝帮助凯特在直升机坠毁前，安全地跳到了附近的屋顶上。二人紧紧相拥，随着镜头逐渐地拉远，两人在摩天大楼上俯瞰着整个安静的城市<ref name="大软" />。

在游戏结束播放制作人员名单时，背景声音播出了一则声明，宣布绯丝和凯特被定为在逃通缉犯，同时敦促市民在沙德服务器修复前避免使用电子通信<ref name="大软" />。

=== 译名 ===

目前网络上最为通用的译名为游戏英文名的直译——“镜-{}-之边缘”，名称中的“镜”代指游戏中干净而明亮的城市，而“镜-{}-之边缘”所描述的就是主人公绯丝等信使与墨守成规的社会格格不入，游走於城市边缘的生活状态<ref>{{cite web |url=http://jzby.uuu9.com/200902/206941.shtml |title=《镜之边缘》游戏名称 |publisher=游久网 |date=2009-02-18 |accessdate=2009-03-23 |deadurl=no |archiveurl=https://web.archive.org/web/20100718150135/http://jzby.uuu9.com/200902/206941.shtml |archivedate=2010-07-18 }}</ref>。不过这并非艺电官方指定的译名，游戏的台湾官方译名为“靓-{}-影特务”，而在[[Windows_Vista|Windows Vista]]简体中文版下安装游戏後，在“游戏”中可见“暗镜迷踪”的简体中文名称，由此可将其推断为中国大陆当时的官方译名<ref>{{cite web|url=http://game.ali213.net/viewthread.php?tid=2538859 |title=真正的官方译名 有图为证 |publisher=游侠网 |date=2009-01-15 |accessdate=2009-03-23 }}</ref>，但这一版本从未在中国大陆上市。艺电在之后的iOS版和Windows Phone版中已改采通用的“镜之边缘”作为中国大陆译名<ref>{{cite web|title=Mirror's Edge™|url=https://itunes.apple.com/cn/app/mirrors-edge/id387999538?mt=8|publisher=App Sotre|accessdate=2014-05-26|deadurl=no|archiveurl=https://web.archive.org/web/20140911223059/https://itunes.apple.com/cn/app/mirrors-edge/id387999538?mt=8|archivedate=2014-09-11}}</ref><ref>{{cite web|title=Mirror's Edge|url=http://www.windowsphone.com/zh-cn/store/app/mirror-edge/b07473fc-585f-4a3d-b4b8-5be5237c0a9b|publisher=Windows Phone应用|accessdate=2014-05-26|deadurl=no|archiveurl=https://web.archive.org/web/20140527215923/http://www.windowsphone.com/zh-cn/store/app/mirror-edge/b07473fc-585f-4a3d-b4b8-5be5237c0a9b|archivedate=2014-05-27}}</ref>。

=== 漫画 ===

《镜之边缘》推出了自己的系列漫画，剧本仍由Rhianna Pratchett设计，尽管故事的舞台以及主要人物和游戏相同，但情节与游戏并没有太多交集。漫画的画师则是Matthew Dow Smith<ref>{{cite web|url=http://www.destructoid.com/comic-con-08-ea-and-dc-team-up-for-mirror-s-edge-comic-book-96745.phtml|title=Comic-Con 08: EA and DC team up for Mirror's Edge comic book|date=2008-07-24|publisher=Destructoid|accessdate=2009-10-01|deadurl=no|archiveurl=https://web.archive.org/web/20160820170307/https://www.destructoid.com/comic-con-08-ea-and-dc-team-up-for-mirror-s-edge-comic-book-96745.phtml|archivedate=2016-08-20}}</ref>。漫画每月出一期，至2009年5月完成，一共推出了六期。

== 扩充内容和续作 ==

在艺电的支持下，独立开发商Borne Games也正在开发一款名为“镜之边缘2D”的2D横版[[网页游戏|网页游戏]]。游戏中不仅有多个关卡供选择，还支持积分榜系统——较好的成绩会自动上传到网站里，并在积分榜中公开。单关卡测试版于2008年11月11日放出，随后于2009年2月24日追加为三关卡测试版。其开发商网站表示游戏完整版本已经上线，但仍处于公测阶段<ref>{{cite web|url=http://www.bornegames.com/mirrors-edge-beta/ |title=ME 2D |date=2008-11-11 |publisher=Borne Games |accessdate=2009-03-22 |archiveurl=https://web.archive.org/web/20090815022433/http://www.bornegames.com/mirrors-edge-beta/ |archivedate=2009-08-15 |deadurl=yes }}</ref>。

2009年12月2日，艺电公布了3D[[横向卷轴游戏|横向卷轴游戏]]《镜之边缘》[[iPhone|iPhone]]和[[iPod_Touch|iPod Touch]]版，共14关，使用第三人称可变镜头视角<ref>{{cite web|author=Arn|url=http://toucharcade.com/2009/12/02/eas-mirrors-edge-coming-to-iphone/|title=EA's ''Mirror's Edge'' Coming to iPhone|work=touchArcade|date=2009-12-02|accessdate=2010-02-03|deadurl=no|archiveurl=https://web.archive.org/web/20100203031024/http://toucharcade.com/2009/12/02/eas-mirrors-edge-coming-to-iphone/|archivedate=2010-02-03}}</ref><ref name="iphonekotaku">{{cite web|url=http://kotaku.com/5416905/mirrors-edge-running-straight-to-the-iphone|last=Plunkett|first=Luke|title=''Mirror's Edge'' Running Straight to the iPhone|work=Kotaku|date=2009-12-02|accessdate=2010-01-24|deadurl=no|archiveurl=https://web.archive.org/web/20100123071348/http://kotaku.com/5416905/mirrors-edge-running-straight-to-the-iphone|archivedate=2010-01-23}}</ref>。游戏原定于2010年1月发行，但之后被推迟到4月，与iPad版同期<ref>{{cite web|url=http://kotaku.com/5485668/mirrors-edge-on-iphone-doing-the-canabalt|title=Mirror's Edge On iPhone Doing The Canabalt|date=2010-03-04|publisher=Kotaku|accessdate=2010-03-16|deadurl=no|archiveurl=https://web.archive.org/web/20100307222730/http://kotaku.com/5485668/mirrors-edge-on-iphone-doing-the-canabalt|archivedate=2010-03-07}}</ref>。

=== 续作 ===
{{main|镜之边缘 催化剂}}

《镜之边缘》发售之初，多条消息就暗示其可能存在续作。游戏发售时，高级制作人欧文·奥布莱恩透露了把游戏发展为三部曲的计划<ref>{{cite web|url=http://www.shacknews.com/onearticle.x/55312|title=EA Planning Mirror's Edge Trilogy|last=Faylor|first=Chris|date=2008-10-14|publisher=ShackNews|accessdate=2008-11-10|deadurl=no|archiveurl=https://www.webcitation.org/66nYnAxsJ?url=http://www.shacknews.com/article/55312/ea-planning-mirrors-edge-trilogy|archivedate=2012-04-09}}</ref>。2009年6月，艺电欧洲副总裁证实正在开发其续作<ref>{{cite web|url=http://www.videogamer.com/news/ea_confirms_small_team_working_on_mirror_s_edge_2.html|title=EA confirms 'small team' working on Mirror’s Edge 2|date=2009-06-12|publisher=VideoGamer|accessdate=2009-10-01|deadurl=no|archiveurl=https://www.webcitation.org/66nYpJGUV?url=http://www.videogamer.com/ps3/mirrors_edge/news/ea_confirms_small_team_working_on_mirror_s_edge_2.html|archivedate=2012-04-09}}</ref>，但DICE首席执行官Karl-Magnus Trodesson对此予以否认，表示“（并未）正式宣布正在开发”<ref>{{cite web |url=http://www.playstationlifestyle.net/2010/01/27/dice-comments-on-a-mirrors-edge-sequel/ |title=DICE Comments On A Mirror's Edge Sequel |publisher=Playstationlifestyle.net |date=2010-01-27 |accessdate=2012-07-15 |deadurl=no }}</ref>。2011年2月，Eurogamer网站称艺电已经中止了《镜之边缘2》的开发计划，原因是制作组提交的程序雏形不尽如人意<ref>{{cite web | url = http://www.eurogamer.net/articles/2011-02-14-report-ea-rejects-mirrors-edge-2-pitch | title = Report: EA rejects Mirror's Edge 2 pitch | first = Robert | last = Purchase | date = 2011-02-14 | accessdate = 2011-02-14 | publisher = [[Eurogamer|Eurogamer]] | deadurl = no | archiveurl = }}</ref>。随后，艺电又发布声明称“仍将《镜之边缘》视为一个重要的系列”，但并无下文<ref>{{cite web | url = http://www.1up.com/news/mirrors-edge-2-development-stopped | title = Mirror's Edge 2 Development "Stopped" - Report | first = Chris url = http://www.pcgamer.com/2011/06/08/ea-2011-mirrors-edge-2-to-be-powered-by-frostbite-2/ | title = EA 2011: Mirror's Edge 2 to be powered by Frostbite 2 | first = Tom | last = Senior | date = Cousins透露新作仍由DICE负责开发<ref>{{cite web | url = http://uk.ign.com/articles/2012/11/21/mirrors-edge-2-is-in-production-at-dice | title = Mirror's Edge 2 "Is in Production at DICE" | first = Andre | last = F | date = 2012-11-21 | accessdate = 2013-03-02 | publisher = IGN }}</ref>。

2013年6月，艺电在其[[E3|E3]]展前发布会正式公布了《镜之边缘》续作，新作对应[[Microsoft_Windows|Microsoft Windows]]、[[Mac_OS_X|Mac OS X]]、[[PlayStation_4|PlayStation 4]]和[[Xbox_One|Xbox One]]。游戏作为《镜之边缘》的前传，讲述绯丝的由来<ref>{{cite web |url= http://www.oxm.co.uk/55888/mirrors-edge-2-announced-an-origin-story-for-xbox-one/ |title= News: Mirror's Edge 2 announced – an origin story for Xbox One – Xbox 360 – The Official Magazine |date= 2013-06-10 |accessdate= 2013-06-11 |first= Edwin |last= Evans-Thirlwell |publisher= Official XBOX Magazine |deadurl= no |archiveurl= https://web.archive.org/web/20131029210054/http://www.oxm.co.uk/55888/mirrors-edge-2-announced-an-origin-story-for-xbox-one/ |archivedate= 2013-10-29 }}</ref>，并将使用[[寒霜3|寒霜3]]引擎<ref>{{cite web |url=http://www.engadget.com/2013/06/10/mirrors-edge-2// |title=EA announces Mirror's Edge 'reboot' for next-gen consoles (video) |publisher=engadget.com |date=2013-06-10 |accessdate=2013-06-10 |deadurl=no |archiveurl=https://web.archive.org/web/20130613042706/http://www.engadget.com/2013/06/10/mirrors-edge-2/ |archivedate=2013-06-13 }}</ref>。艺电随后确认该作为“开放世界冒险游戏”<ref>{{cite web|url=http://www.gamespot.com/e3/new-mirrors-edge-is-open-world-6409990/ |title=New Mirror's Edge is open-world |publisher=gamespot.com |date=2013-06-12 |accessdate=2013-06-12 |deadurl=yes |archiveurl=https://web.archive.org/web/20130615230555/http://www.gamespot.com/e3/new-mirrors-edge-is-open-world-6409990 |archivedate=2013-06-15 }}</ref>。2014年1月，编剧Rhianna Pratchett在[[Twitter|Twitter]]表示其个人及大部分原作团队并未参与新作开发<ref>{{cite web|url=http://gamerant.com/mirrors-edge-2-writer-rhianna-pratchett/comment-page-1/#comment-166310 |title='Mirror's Edge' Writer Not Returning For 'Mirror's Edge 2' |author=Andrew Dyce |publisher=Game Rant, LLC |year=2014 |accessdate=2015-06-23 |deadurl=yes |archiveurl=https://web.archive.org/web/20140222232348/http://gamerant.com/mirrors-edge-2-writer-rhianna-pratchett/comment-page-1/ |archivedate=2014-02-22 }}</ref>。

在2014年E3展上，镜之边缘续作的概念宣传片显示续作将主要讲述绯丝的身世，并为信使提供更多不同的玩法<ref>{{cite web|title=Prototype Footage of Mirror's Edge 2 Revealed|url=http://news.cheatcc.com/articles/394127|publisher=Cheat Code Central|accessdate=2014-07-15|deadurl=no|archiveurl=https://web.archive.org/web/20140714204421/http://news.cheatcc.com/articles/394127|archivedate=2014-07-14}}</ref>。2015年，艺电正式公布游戏续作名为“[[镜之边缘_催化剂|镜之边缘 催化剂]]”（{{lang|en|''Mirror's Edge Catalyst''}}）<ref>{{cite web|title=Mirror's Edge Catalyst|url=https://twitter.com/mirrorsedge/status/608309738330292224|website=Twitter|accessdate=2015-06-09|deadurl=no|archiveurl=https://web.archive.org/web/20160121035326/https://twitter.com/mirrorsedge/status/608309738330292224|archivedate=2016-01-21}}</ref>。在2015年的科隆游戏展上，游戏的开发人员向媒体确认《镜之边缘 催化剂》并非是《镜之边缘》的续作，而是[[重启|重启]]作品。<ref>{{cite web|title=《镜之边缘 催化剂》其实也是重启作品|url=http://www.vgtime.com/article/3261335.jhtml|publisher=游戏时光|date=2015-08-09|accessdate=2015-08-09|deadurl=no|archiveurl=https://web.archive.org/web/20160614053050/http://www.vgtime.com/article/3261335.jhtml|archivedate=2016-06-14}}</ref>。
