'''幽州刺史部'''，是漢朝十三州[[刺史部|刺史部]]之一，州治在薊縣（今[[北京市|北京市]]西南）。漢代州域範圍大致是今日的[[河北省|河北省]]北部，[[北京市|北京市]]全部，[[天津市|天津市]]、[[遼寧省|遼寧省]]、[[吉林省|吉林省]]各一部分，[[北朝鮮|北朝鮮]]大部分。

== 名稱由來 ==
「幽州」，傳說是舜在《禹貢》九州域內，析出幽、并、營3州<ref>《晉書·卷十四·志第四·地理上》“帝堯時，禹平水土，以爲九州。虞舜登庸，厥功彌劭，表提類而分區宇，判山河而考疆域，冀北創并部之名，燕齊起幽營之號，則書所謂肇十有二州，封十有二山者也。</ref>，大體指古代[[燕國|燕國]]一帶（含遼東及遼西）。《周禮·職方》載：「東北曰幽州。」《呂氏春秋》云：「北方為幽州，燕也。」

== 西漢幽州刺史部 ==
西漢時，領[[廣陽國|廣陽國]]、[[涿郡|涿郡]]、[[勃海郡|勃海郡]]、[[上谷郡|上谷郡]]、[[漁陽郡|漁陽郡]]、[[右北平郡|右北平郡]]、[[遼西郡|遼西郡]]、[[遼東郡|遼東郡]]、[[樂浪郡|樂浪郡]]、[[玄菟郡|玄菟郡]]10個郡國。

== 東漢幽州刺史部 ==
東漢時，領廣陽郡、涿郡、代郡、上谷郡、漁陽郡、右北平郡、遼西郡、遼東郡、樂浪郡、玄菟郡、遼東屬國11個郡國。
