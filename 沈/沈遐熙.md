'''沈遐熙'''（{{bd|1921年|3月|2003年|9月22日}}）[[回族|回族]]，祖籍[[江苏|江苏]][[南京|南京]]，生于[[上海|上海]]，[[中国共产党|中国共产党]]党员，[[中国伊斯兰教协会|中国伊斯兰教协会]]第五届会长<ref name=sina>{{Cite web |url =  http://news.sina.com.cn/o/2003-10-09/1035883286s.shtml|title =沈遐熙同志逝世  |author =  |date = 2003-10-09 |publisher = 新浪 |language =  |accessdate =  |quote =  }}</ref><ref name=mrgs>{{Cite web |url =http://gansu.gansudaily.com.cn/system/2009/08/27/011244481.shtml  |title = 感动甘肃人物候选人--沈遐熙 |author =  |date =2009-08-27  |publisher = 每日甘肃网  |language =  |accessdate =  |quote =  }}</ref>。

==生平==
1938年，沈遐熙从上海省立公费中学毕业后，即来到[[中国共产党|中国共产党]]领导的[[陕甘宁边区|陕甘宁边区]]投身抗日工作。1938年10月加入[[中国共产党|中国共产党]]<ref>沈遐熙革命纪念馆，党的建设2012(10):44-44</ref><ref name=sina/><ref name=mrgs/>。曾任[[镇原县|镇原县]][[三岔镇_(镇原县)|三岔镇]]回民小学校长。后来在[[张家川回族自治县|张家川]]担任[[清真寺|清真寺]]经堂教育掌学、二[[阿訇|阿訇]]。1948年，出任中共地下党清水县工委书记。1955年起，历任[[甘肃省|甘肃省]][[临夏专区|临夏专区]]专员、[[临夏回族自治州|临夏回族自治州]]州长。1965年调任[[甘肃省计划委员会|甘肃省计划委员会]]副主任、[[甘肃省地质局|甘肃省地质局]]副局长、代局长。[[文革|文革]]爆发后，沈遐熙遭受批斗<ref name=sina/><ref name=mrgs/>。

1978年，沈遐熙出任[[中央民族学院|中央民族学院]]副院长。1982年后，历任[[中国伊斯兰教协会|中国伊斯兰教协会]]副主任、代主任，第五届会长，第六、七届顾问。沈遐熙还曾经兼任中国人民争取和平与裁军协会副会长，中国非洲友好协会副会长，中国中东学会副会长，中国印尼经济文化社会合作协会副会长，中国国际交流协会理事，世界伊斯兰教联盟清真寺最高理事会理事等职务。沈遐熙是第六届全国政协委员，第七、八、九届全国政协常务委员<ref name=sina/><ref name=mrgs/>。

2003年9月22日，沈遐熙在北京病逝，享年82岁<ref name=sina/><ref name=mrgs/>。
