'''沈卫平'''（{{bd|1952年|||}}）[[北京|北京]]人<ref name=zj/>，[[中国人民解放军少将|中国人民解放军少将]]<ref name=gong>宫五一，六公主府沧桑，载 臧汝奇主编，北京朝阳门：人文历史750年，人民出版社，2010年</ref>，[[中国共产党|中国共产党]]党员，[[中华人民共和国|中华人民共和国]]作家<ref name=zj>{{Cite web |url = http://www.chinawriter.com.cn/zxhy/member/9999.shtml |title = 沈卫平 (1952～) |author =  |date =  |publisher = 中国作家网 |language =  |accessdate =  2017-01-16|quote =  }}</ref>。

==生平==
沈卫平小时候住在[[中国人民解放军总政治部|中国人民解放军总政治部]]家属院[[内务部街十一号四合院|内务部街11号]]<ref name=gong/>。1960年代末至1970年代初曾在[[中国人民解放军空军|中国人民解放军空军]]部队服役。后来自大学中文系本科毕业<ref name=zj/>。此后长期在[[中国人民解放军总政治部联络部|中国人民解放军总政治部联络部]]任职，一直任至该部副部长，并获授少将军衔<ref name=gong/>。他还曾任[[中华文化发展促进会|中华文化发展促进会]]副会长<ref name=zj/>、[[中国国际友好联络会|中国国际友好联络会]]副会长<ref>{{Cite web |url = http://news.163.com/09/0424/18/57MDGK1P000120GR.html |title = 中国国际友好联络会副会长沈卫平在开幕式上致辞 |author =  |date =  2009-04-24|publisher = 网易 |language =  |accessdate =  |quote =  }}</ref>、[[北京|北京]][[中國國際友好聯絡會|和平与发展研究中心]]研究员、[[全国台湾研究会|全国台湾研究会]]理事，兼任音像出版单位总编辑<ref name=zj/>。1985年沈卫平开始发表作品。1999年加入[[中国作家协会|中国作家协会]]。作品曾获得全军第二届新作品奖文学一等奖、《解放军文艺》奖<ref name=zj/>。
