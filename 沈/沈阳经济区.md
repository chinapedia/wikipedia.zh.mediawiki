'''沈阳经济区'''是[[辽宁省|辽宁省]]政府提出的区域发展战略。以[[沈阳|沈阳]]为中心，辐射沈阳、[[鞍山|鞍山]]、[[抚顺|抚顺]]、[[本溪|本溪]]、[[营口|营口]]、[[阜新|阜新]]、[[辽阳|辽阳]]、[[铁岭|铁岭]]八个城市，形成联系紧密的“区域经济共同体”<ref>{{cite web|url=http://www.syma.gov.cn/syjjqgk/2011-07-06/661415.htm|title=沈阳经济区概况|publisher=沈阳经济区官方网站|date=2011-07-06|accessdate=2011-09-28|language=zh-hans|archive-url=https://web.archive.org/web/20120725001503/http://www.syma.gov.cn/syjjqgk/2011-07-06/661415.htm|archive-date=2012-07-25|dead-url=yes}}</ref>。

== 概况 ==
沈阳经济区是以沈阳为中心，在半径百公里范围内，涵盖了沈阳、鞍山、抚顺、本溪、营口、阜新、辽阳、铁岭8个省辖市的[[城市群|城市群]]，是中国城市化水平最高的地区之一<ref>{{cite web|url=http://www.people.com.cn/h/2011/0927/c25408-856578538.html|title=沈阳经济区改革方案获批|publisher=人民网|date=2011-09-27|accessdate=2011-09-28|language=zh-hans}}</ref>。2010年4月6日，经国务院同意，国家发改委正式批复沈阳经济区为“[[国家新型工业化综合配套改革试验区|国家新型工业化综合配套改革试验区]]”，这标志着沈阳经济区成为继[[上海浦东新区|上海浦东新区]]、[[天津滨海新区|天津滨海新区]]、成都、重庆、武汉城市圈、长株潭城市群和深圳等七个地区后，国务院批准设立的第八个[[国家综合配套改革试验区|国家综合配套改革试验区]]<ref>{{cite web|url=http://finance.sina.com.cn/stock/hyyj/20100407/07107700242.shtml|title=沈阳经济区惹火区域板块 装备新兴产业长期受益|publisher=新浪网|date=2010-04-07|accessdate=2011-09-28|language=zh-hans}}</ref>。2011年9月16日，国务院正式批复《沈阳经济区新型工业化综合配套改革试验总体方案》，标志着沈阳经济区的改革方向的更加明确<ref>{{cite web|url=http://news.sina.com.cn/o/2011-09-27/132323224770.shtml|title=国务院批复沈阳经济区综合配套改革方案|publisher=新浪网|date=2011-09-27|accessdate=2011-09-28|language=zh-hans}}</ref>。为了推动沈阳经济区一体化发展，沈阳、抚顺、铁岭、本溪等几个省辖市共享024[[中国大陆电话区号|区号]]<ref>{{cite web |title=沈阳抚顺铁岭共用024长途电话区号今日开通 |url=http://news.sina.com.cn/c/2011-08-28/103323063863.shtml |website=新浪新闻 |publisher=东北新闻网 |accessdate=2018-07-18 |date=2011-08-28}}</ref><ref>{{cite news |title=24日起本溪共享“024”区号 |url=http://epaper.syd.com.cn/syrb/html/2014-05/17/content_997598.htm |accessdate=2018-07-18 |work=[[沈阳日报|沈阳日报]] |publisher=沈阳日报报业集团 |date=2018-05-17}}</ref>。

== 目标方向 ==
按照《总体方案》批复要求，沈阳经济区新型工业化综合配套改革，是以加快转变经济发展方式为主线，以深化改革开放为动力，坚持走中国特色新型工业化道路，切实加强体制机制创新，积极稳妥地解决老工业基地存在的突出矛盾和问题，着力构建结构优化、技术先进、清洁安全、附加值高、吸纳就业能力强的现代产业体系，推进区域一体化发展，增强'''沈阳经济区'''整体竞争力，为全面贯彻实施东北地区老工业基地振兴战略、促进中国经济长期平稳较快发展和社会和谐提供经验与示范。
