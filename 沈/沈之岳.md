'''沈之岳'''（{{bd|1913年|2月18日|1994年|2月24日||catIdx=Shen沈}}），生於[[浙江省|浙江省]][[仙居縣|仙居縣]]下閣鄉西陸村。1964年在台灣組建[[法務部調查局|調查局]]，擔任調查局首任局長，被稱為「調查局之父」{{r|ctimes-2015-260102}}。<ref name="gpwb-2015-沈"/>

==生平==
=== 早年時期 ===
沈之岳生於1913年（民國2年）2月18日生於[[浙江省|浙江省]][[仙居縣|仙居縣]]下閣鄉西陸村，家境寬裕，1933年入上海[[復旦大學|復旦大學]]就讀，在校期间因緣際會下被[[戴笠|戴笠]]收編為[[复兴社|复兴社]]特务处（军统前身）[[特務|特務員]]<ref name="gpwb-2015-沈"/>，送入浙江警官學校接受訓練<ref name=007_3>{{cite web |url=http://blog.sina.com.cn/s/blog_476745f601008mp2.html |title=蓝色007传奇 三. 被抓进的军统局 |author=萨苏 |accessdate=2008-09-11}}</ref><ref name=Chou>{{cite web |url=http://www.chinesedc.com/dcv1/Authors/kuchow/black-people/WANG%20GJ.htm |title=台北六十年代又一外交冤案 |author=周谷 |accessdate=2008-09-10}}</ref>。

=== 加入共產黨 ===
1930年代，時籍實行「[[攘外安內|安內攘外]]」政策，[[國民政府|國民政府]]全力[[剿共|剿共]]，沈之岳自學[[俄語|俄語]]、[[英語|英語]]與[[馬列主義|馬列主義]]，以「李國棟」{{r|ctimes-2015-260102}}<ref name="gpwb-2015-沈"/>之名，在[[上海|上海]]加入[[中國共產黨|中國共產黨]]外圍地下組織。據《'''[[王芳|王芳]]'''回憶錄》記述，沈曾誘殺七、八位中國共產黨黨員<ref name=Wang>{{cite web |url=http://book.people.com.cn/BIG5/69399/5093938.html |title=王芳回憶錄 |author=王芳 |publisher=浙江人民出版社 |accessdate=2008-09-10}}</ref>。

1938年，[[中國抗日戰爭|對日抗戰]]爆發，沈之岳偽裝為[[中央大學|中央大學]]學生，化名「'''沈輝'''」{{r|ctimes-2015-260102}}<ref name="gpwb-2015-沈"/>，以助手身份陪同中大教授到[[延安|延安]]考察，要求留下來「參加革命」，並通過[[共軍|共軍]]保衛部門的審查<ref name=Sa>{{cite web |url=http://www.studytalk.com/publishhtml/3/2008-07-10/20080710100053.html |title=军统骨干沈之岳成功打入延安之谜 |author=萨苏 |accessdate=2008-09-10}}</ref>。沈在延安進入[[抗日軍政大學|抗日軍政大學]]學習{{r|ctimes-2015-260102}}<ref name="gpwb-2015-沈"/>，獲[[康生|康生]]的賞識，隨後加入中國共產黨，一度進入[[中共中央|中共中央]]機要部門擔任收發工作<ref name=Wang/>。

沈之岳曾以「李国栋」的化名{{r|ctimes-2015-260102}}<ref name="gpwb-2015-沈"/>，在1939年于军统汉中训练班见过汉训班班主任程慕颐，并对训练班的工作作过指点。

1939年，沈奉命離開延安，到[[浙江|浙江]]白區工作，隨[[葉挺|葉挺]]赴[[皖|皖]]南收編八省紅軍和游擊隊，整編成立[[國民革命軍新編第四軍|國民革命軍新編第四軍]]（簡稱新四軍），奉行「向南鞏固，向東作戰，向北發展」的戰略方針，一方對日軍作戰，一方又圖擴展勢力；1940年10月初，新四軍發動[[黃橋戰役|黃橋戰役]]（今[[江蘇省|江蘇省]][[泰州市|泰州市]]內），成功佔為根據地。<ref name="gpwb-2015-沈"/>

1940年10月19日，[[蔣介石|蔣介石]]發出「[[皓電|皓電]]」令，由[[何應欽|何應欽]]、[[白崇禧|白崇禧]]發電報給[[朱德|朱德]]、[[彭德懷|彭德懷]]、葉挺和[[項英|項英]]，以調停國民政府軍與新四軍及[[第十八集團軍|第十八集團軍]]的連日衝突為由，令兩軍團接電報後1個月內全部撤到舊黃河以北，[[毛澤東|毛澤東]]授意「朱彭葉項」同意新四軍調到長江以北；新四軍未依期北上。延至1941年1月，葉挺和項英率領新四軍9000名軍士經[[涇縣|涇縣]]北上；1月5日，部隊行至茂林地區（今[[安徽省|安徽省]][[宣城市|宣城市]]內）時，遭到由[[顧祝同|顧祝同]]帶領的國軍包圍和襲擊，是謂[[皖南事變|皖南事變]]，史學家認為這可能是沈之岳提供相關情報<ref name=Sa/>。

=== 恢復身份 ===
皖南事變後，沈之岳重返戴笠系統，1941年7月被任命為[[軍統局|軍統局]]第一處科長；1943年，軍統局成立東南特偵站，沈當站長，並兼任軍事委員會[[忠義救國軍|忠義救國軍]]淞滬指揮部政治部主任，共軍至此才得知他已經為國民黨工作，從此視之為叛徒<ref name=Wang/>。<ref name="gpwb-2015-沈"/>

抗日戰爭勝利後不久，旋之進入[[國共內戰|國共內戰]]，當時沈任[[國防部|國防部]][[國民政府軍事委員會調查統計局|保密局]]第二處科長，國軍敗退[[台灣|台灣]]前任'''蘇浙情報站'''站長。

=== 退守台灣 ===
1950年，沈之岳奉派至[[蔣經國|蔣經國]]主持之[[革命實踐研究院|革命實踐研究院]][[石牌_(臺北市)|石牌]]訓練班，任副主任兼訓導組長；1951年10月調[[大陳島|大陳島]]，任江浙[[反共救國軍|反共救國軍]]總部政治部主任，兼[[中國國民黨|中國國民黨]]大陳特派員辦公室秘書長、[[中華民國|中華民國]][[浙江省政府|浙江省政府]]委員兼代省主席、大陳行政督察專員等要職。1953年大陳島總部改組為大陳防衛司令部，沈留任。

在大陳期間，沈之岳做了一件蔣介石、蔣經國父子十分感激的事：他派人潛回蔣家故鄉[[奉化|奉化]]，將[[蔣母墓|蔣母墓]]、[[豐鎬房|豐鎬房]]、[[報本堂|報本堂]]、[[雪竇寺|雪竇寺]]的諸多景物拍成照片，獻給蔣介石。

1955年，沈協助蔣經國執行「'''[[金剛計畫|金剛計畫]]'''」與「'''飛龍計畫'''」，安排[[大陳島撤退|大陳]]、[[南麂|南麂]]、[[漁山列島|漁山列島]]軍民撤退至台灣，並裁撤中華民國[[浙江省政府|浙江省政府]]。

1956年，沈任[[司法行政部調查局|司法行政部調查局]]督察室主任，後改任司法行政部調查局副局長，黨職任[[中國國民黨中央委員會|中國國民黨中央委員會]]南方工作組組長，负责整顿[[中统|中统]]；1958年調任[[國防部情報局|國防部情報局]]副局長。

=== 潛入澳門 ===
1963年，沈之岳時任情報局副局長兼中二組副主任<ref> “中二组”全称“中国国民党中央委会第二组”，成立于1952年底。主要任务是向中国大陆派遣特工并指挥潜伏特工在大陆发展国民党地下党组织，进行颠覆活动。</ref>，化名'''孫子超'''，4月20日秘密潛入[[澳門|澳門]]，部署對大陸的行動破壞，試圖刺殺當時[[中华人民共和国主席|中华人民共和国主席]][[劉少奇|劉少奇]]{{r|ctimes-2015-260102}}。沈在澳門[[新新酒店|新新酒店]]、同盟酒店各開一個房間，實際上住在「中二組」澳門特一組的駐地，沈行蹤被共產黨完全掌握，華南辦事處將資料直送北京<ref name=Wang/><ref name=007_2>{{cite web |url=http://blog.sina.com.cn/s/blog_476745f601008mod.html |title=蓝色007传奇 二.作了中共中央收发员的军统特务 |author=萨苏 |accessdate=2008-09-11}}</ref>。

當時，中共高層曾考慮將沈之岳從澳門捉拿回大陸歸案<ref name="gpwb-2015-沈"/>，給蔣介石父子以顏色看看；最後採取較緩和的做法，由[[中華人民共和國公安部|公安部]]通過[[外交|外交]]途徑通知[[澳葡|澳葡]]當局。澳葡當局逮捕了一批特務，將沈驅逐回台灣，沈因此受到撤職查辦的處分<ref name=Wang/><ref name=007_2/>。

=== 組調查局 ===
1964年，沈之岳在蔣經國的極力推薦下，負責改組[[法務部調查局|調查局]]，出任局長<ref name="gpwb-2015-沈"/>，一手建立了調查局的制度，並培育大量有別於軍統、中統特務，規定調查局幹員必須有[[大專|大專]]以上學歷、最好能通一至二種[[外語|外語]]。同時，沈之岳也一手建立了預算、決算外的調查局小財庫，在[[戒嚴|戒嚴]]時期透過兩岸地下貿易（如[[中藥材|中藥材]]）及[[炒股票|炒股票]]，替調查局開闢了預算外的工作經費來源。沈之岳上任時發生人事鬥爭，使《[[台灣新生報|台灣新生報]]》[[沈嫄璋|沈嫄璋]]等人被殺與坐冤獄<ref name="戴獨行">{{cite book |author=戴獨行|chapter=〈六、七零年代政治殺戒大觀〉|title=《白色角落》|date=1998|publisher=人間出版社|location=台灣|language=zh-tw|isbn=9578660537}}</ref>。

1972年2月22日，[[台灣獨立運動|台灣獨立運動]]人士[[辜寬敏|辜寬敏]]在沈之岳安排下，秘密從[[日本|日本]]取道[[泰國|泰國]]返回台灣，與後來接任[[行政院院長|行政院院長]]的[[蔣經國|蔣經國]]對話。<ref>{{cite book |author=戴月芳|chapter=〈水泥金融世家：鹿港辜家〉|title=《台灣大家族》|date=2012-10-20|publisher=台灣書房|location=台灣|language=zh-tw|isbn=9789866318795}}</ref><ref>{{cite news|url=http://www.my-formosa.com/DOC_82674.htm|title=精於幫倒忙的台獨生意人辜寬敏|publisher=美麗島電子報 政事觀察站|author=郭正亮|date=2015-06-22|}}</ref>

=== 晚年生活 ===
1978年1月，沈之岳從調查局卸任，調任中國國民黨中央委員會社會工作會主任，並曾任中國國民黨中央評議委員與第十、十一屆中央委員；1979年受聘為[[中華民國總統府國策顧問|中華民國總統府國策顧問]]。

1990年，沈之岳因患[[腎病|腎病]]曾到北京治療，並獲[[鄧小平|鄧小平]]接見。

1994年2月24日病逝於[[台北榮民總醫院|台北榮民總醫院]]，享壽{{age|1913|2|18|1994|2|24}}歲；同年3月11日，獲頒[[褒揚令列表|褒揚令]]<ref>{{cite web|url= http://www.president.gov.tw/Default.aspx?tabid=131&itemid=3751 |title= 總統令 |date= 1994-03-11 |accessdate= 2015-10-28|publisher= [[中華民國總統|中華民國總統]]府 | quote= 褒揚令證書文令字號，參見《[[:s:Wikisource:褒揚令|:s:Wikisource:褒揚令]]》}}</ref>。

== 私人生活 ==
沈之岳元配[[邵達鎮|邵達鎮]]，歿於1974年。育有一子-{沈}-海濱，二女-{沈}-淨崎、-{沈}-讓濱。1988年，沈之岳[[續絃|續絃]]台灣[[京劇|京劇]]名[[旦角|旦]][[徐露|徐露]]。

==身份質疑==
對於沈之岳早年往來國共之間的經歷，國民黨稱沈之岳早是戴笠手下、奉命潛入共產黨。但一說沈之岳根本就是共產黨、为共產黨在台湾至死未被发现的两大卧底之一，兩蔣雖然對共諜抱持「寧可誤殺，不可錯放」的態度，但還是沒有發現沈之岳的問題。<ref>{{cite book |author=陸鏗|title=《陸鏗回憶與懺悔錄》|date=1997-07-08|publisher=時報出版|location=台灣|language=zh-tw|ISBN=9571323349}}</ref><ref>{{cite news|title=曾永賢口述歷史掌管調查局13年沈之岳是雙面諜|publisher=中國時報|author=王銘義|date=2010-03-03|}}</ref>

==紀念==
[[宜蘭縣|宜蘭縣]][[蘇澳鎮_(臺灣)|蘇澳鎮]][[岳明新村|岳明新村]]一名由來，即是取自沈之岳的「岳」字，及其化名王明的「明」字而來。岳明新村是台灣規模最大的[[大陳島|大陳島]]移民聚落。
