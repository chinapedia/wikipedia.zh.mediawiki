'''节气'''指二十四时节和[[气候|气候]]，是[[中国|中国]]古代用来指导[[農業|农事]]之[[历法|历法]]曆注<ref>[http://www.hko.gov.hk/gts/time/24solartermsc.htm 二十四節氣]. 香港天文台. </ref>。

[[東亞文化圈|東亞傳統]][[夏曆|夏曆]]（[[農曆|農曆]]）是一种「[[阴阳合曆|阴阳合曆]]」，同時根据日、月运行制定，「陰」是以[[朔望月|朔望月]]为基准确定，「陽」是以[[地球公轉|地球自冬至繞太陽公轉]]一圈为基准确定[[歲實|歲實]]，每[[回歸年|回歸年]]約365.2422日，二十四節氣據此而劃分，其中：
*[[春分|春分]]是於[[黃經|黃經]]（太陽經度或天球經度）0°（[[春分点|春分点]]）。
*[[夏至|夏至]]是於在[[太陽|太陽]]到達黃經90°時（太陽幾乎直射[[北回歸線|北回歸線]]，北回歸線以北[[白晝|白晝]]最長）。
*[[秋分|秋分]]是於黃經180°（[[秋分點|秋分點]]）。
*[[冬至|冬至]]是於在太陽到達黃經270°時（太陽幾乎直射南回歸線，北半球白晝最短）。
一年劃分了二十四節氣，與現代曆法將[[一日|一日]]劃分為二十四[[小時|小時]]，兩者數字同為[[24|24]]。

2016年，[[联合国教科文组织|联合国教科文组织]]将“二十四节气——中国人通过观察太阳周年运动而形成的时间知识体系及其实践”列入人类[[非物质文化遗产|非物质文化遗产]]代表作名录。<ref>[http://www.thepaper.cn/newsDetail_forward_1571031 “二十四节气”正式列入联合国教科文组织非遗代表作名录]，澎湃新闻，2016-11-30。</ref>

== 起源 ==
中國古代利用[[土圭|土圭]]實測[[日晷|日晷]]，將每年日影最長定為“日至”（又稱日長至，即[[太極圖|太極圖]]中[[四象|四象]]的太陰、長至、[[冬至|冬至]]），日影最短為“日短至”（又稱短至、[[夏至|夏至]]）。在春秋兩季各有一日的晝夜時間長短相等，便定為“[[春分|春分]]”和“[[秋分|秋分]]”，人類各民族多有此四。在[[商朝|商朝]]时只有四个节气（[[四象|四象]]，即[[四季|四季]]），到了[[周朝|周朝]]时发展到了八个（[[八卦|八卦]]），到秦汉年间<ref>[[呂不韋|呂不韋]]《[[s:呂氏春秋|s:呂氏春秋]]》</ref>，二十四节气已完全确立。公元前104年，由邓平等制定的《[[太初历|太初历]]》，正式將二十四节气订于历法，明确了二十四节气的天文位置。

二十四节气名称首见于《[[淮南子|淮南子]]·天文训》，《[[史記|史記]]·太史公自序》的“論六家要旨”中也曾提到[[陰陽|陰陽]]（日月運行）、[[四時|四時]]（四季）、八位（八卦）、十二度（[[時辰|時辰]]）、二十四節气等概念。[[汉武帝|汉武帝]]时，[[落下闳|落下闳]]将节气编入《[[太初曆|太初曆]]》之中，并规定无[[中气|中气]]之月，定为上月的[[闰月|闰月]]。

二十四节气每一个分别相应于[[太阳|太阳]]在[[黄道|黄道]]上每运动15°所到达之位置。二十四节气又分为12个節氣和12个[[中气|中气]]，一一相间。二十四节气反映了[[地球公轉|太阳周年视运动]]，所以其[[西曆|西曆]]日期近乎固定，上半年的节气在6日，中气在21日，下半年的节气在8日，中气在23日，前后不差1～2日。

二十四節氣的命名反應了[[季節|季節]]和[[气候|气候]]的變化。立春、春分、立夏、夏至、立秋、秋分、立冬、冬至，又稱八位，是区分公转运动对于地球影响的八个关键节点；雨水、驚蟄、清明、穀雨、小滿、芒種、小暑、大暑、處暑、白露、寒露、霜降、小雪、大雪、小寒、大寒等十六个则反映了四季中更细微的气候变化。

== 節氣歌和民諺 ==
{{seealso|二十四節令鼓}}
由于节气反映了地球围绕太阳运动的过程，是每年季节变更的重要标志，因此对农业生产非常重要。中国农民为了更方便地根据节气来安排农事，长期以来形成了一些相關的民谚。

; 二十四节气歌 
<poem>
春雨驚春清穀天；
夏滿芒夏暑相連；
秋處露秋寒霜降；
冬雪雪冬小大寒。
每月两节不变更；
最多相差一两天；
上半年来六廿一；
下半年来八廿三。
</poem>

注意：最后两句是說上半年的節氣在6日，中氣在21日，下半年的節氣在8日，中氣在23日。日期是西历日期。

; 二十四節令歌 
<poem>
立春陽氣轉，雨水沿河邊；
驚蟄烏鴉叫，春分瀝皮乾；
清明忙種麥，穀雨種大田。
</poem>
<poem>
立夏鵝毛住，小滿雀來全；
芒種五月節，夏至不納棉；
小暑不算熱，大暑三伏天。
</poem>
<poem>
立秋忙打靛，處暑幹老虎；
白露煙上架，秋分無生田；
寒露不算冷，霜降變了天。
</poem>
<poem>
立冬交十月，小雪地封嚴；
大雪河叉上，冬至不行船；
小寒進臘月，大寒又一年。
</poem>

; 西历十二月節氣歌 
<poem>
二月 立春雨水節；
三月 驚蟄及春分；
四月 清明並穀雨；
五月 立夏小滿方；
六月 芒種及夏至；
七月 小暑大暑當；
八月 立秋還處暑；
九月 白露秋分忙；
十月 寒露又霜降；
十一月 立冬小雪張；
十二月 大雪冬至節；
一月 小寒大寒昌。
</poem>

; 各地其它諺語

*** 早立秋，冷飕飕；晚立秋，热死牛。（如果立秋时间在日出之前，之后的天气就会比较冷；如果出现在日出之后，接下来的天气就会比较闷热。）

== 国际影响 ==
节气名稱是根据中国的气候命名的，當[[緯度|緯度]]不同可能造成名稱與氣候不一致。但節氣時間的定義是根據黃道，故北半球統一。夏至日長、冬至日短。

== 農耕八節 ==
[[立春|立春]]、[[春分|春分]]、[[立夏|立夏]]、[[夏至|夏至]]、[[立秋|立秋]]、[[秋分|秋分]]、[[立冬|立冬]]、[[冬至|冬至]]
* 古人将黄道划分4季，12节，12气，72候。定义3节为一季，约30日为一节，15日为一气，5日为一候。年、季、节、气、候。
* 四季（春、夏、秋、冬）
* 十二节（立春、惊蛰、清明、立夏、芒种、小暑、立秋、白露、寒露、立冬、大雪、小寒）
* 十二气（雨水、春分、谷雨、小满、夏至、大暑、处暑、秋分、霜降、小雪、冬至、大寒）
* [[七十二候|七十二候]]（约每5日为一候）
* 一年=四季
* 一季=三节
* 一节=二气=六候
* 一候≈五日

== 七十二候 ==
{{See|物候|蝴蝶物候學}}
每氣分三候，可以見到[[黃河流域|黃河流域]]天文、地理、人事和自然界的一般景物。
# [[立春|立春]]：立春之日东风解冻，又五日蟄蟲始振，又五日魚上冰（魚陟負冰）。
# [[雨水|雨水]]：雨水之日獺祭魚，又五日鴻雁來（候雁北），又五日草木萌動。
# [[惊蛰|惊蛰]]：惊蛰之日桃始華，又五日倉庚鳴，又五日鷹化為鳩。
# [[春分|春分]]：春分之日玄鳥至，又五日雷乃發聲，又五日始電。
# [[清明|清明]]：清明之日桐始華，又五日田鼠化為鴽，又五日虹始見。
# [[谷雨|谷雨]]：谷雨之日萍始生，又五日鳴鳩拂其羽，又五日戴勝降于桑。
# [[立夏|立夏]]：立夏之日螻蟈鳴，又五日蚯蚓出，又五日王瓜生。
# [[小满|小满]]：小满之日苦菜秀，又五日靡草死，又五日小暑至（麥秋生，麦秋至）。
# [[芒种|芒种]]：芒種之日螳螂生，又五日鵙始鳴，又五日反舌無聲。
# [[夏至|夏至]]：夏至之日鹿角解，又五日蜩始鳴，又五日半夏生。
# [[小暑|小暑]]：小暑之日温风至，又五日蟋蟀居壁，又五日鹰乃学习（鹰始击）。
# [[大暑|大暑]]：大暑之日腐草为萤，又五日土潤溽暑，又五日大雨時行。
# [[立秋|立秋]]：立秋之日涼風至，又五日白露降，又五日寒蟬鳴。
# [[处暑|处暑]]：處暑之日鷹乃祭鳥，又五日天地始肅，又五日禾乃登。
# [[白露|白露]]：白露之日鴻雁來，又五日玄鳥歸，又五日群鳥養羞。
# [[秋分|秋分]]：秋分之日雷始收聲，又五日蟄蟲培坯戶，又五日水始涸。
# [[寒露|寒露]]：寒露之日鴻雁來賓，又五日雀入大水為蛤，又五日菊有黃華。
# [[霜降|霜降]]：霜降之日豺乃祭獸，又五日草木黃落，又五日蟄蟲咸俯。
# [[立冬|立冬]]：立冬之日水始冰，又五日地始凍，又五日雉入大水為蜃。
# [[小雪|小雪]]：小雪之日虹藏不見，又五日天氣上騰地氣下降，又五日閉塞而成冬。
# [[大雪|大雪]]：大雪之日鶡旦不鳴，又五日虎始交，又五日荔挺生（荔挺出）。
# [[冬至|冬至]]：冬至之日蚯蚓結，又五日麋角解，又五日水泉動。
# [[小寒|小寒]]：小寒之日雁北鄉，又五日鵲始巢，又五日雉始鴝（雉始雊）。
# [[大寒|大寒]]：大寒之日鸡始乳，又五日鷲鸟厉疾（征鸟厉疾），又五日水泽腹坚。
