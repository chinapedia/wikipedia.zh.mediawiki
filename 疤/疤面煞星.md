《'''疤面煞星'''》（{{lang-en|'''''Scarface'''''}}）是一部於1983年上映的美國[[犯罪片|犯罪片]]，由[[布萊恩·狄帕瑪|布萊恩·狄帕瑪]]執導，[[奧利佛·史東|奧利佛·史東]]編劇。本片是1932年英文同名電影《[[疤面|疤面]]》的重拍版。本片講述身無分文的古巴難民{{link-en|東尼·莫塔納|Tony Montana}}（[[艾爾·帕西諾|艾爾·帕西諾]] 飾演）到達了1980年代的邁阿密，並成為一名強大的販毒領袖的故事。其他演員還有{{link-en|瑪麗·伊莉莎白·瑪絲楚湯妮奧|Mary Elizabeth Mastrantonio}}、{{link-en|史蒂芬·鮑爾|Steven Bauer}}和[[蜜雪兒·菲佛|蜜雪兒·菲佛]]。

《疤面煞星》於1983年12月9日在美國上映，並獲得了成功的票房。但電影獲得了褒貶不一的評價，多數影評人批評其過度的暴力和藥物用法。在[[邁阿密|邁阿密]]的一些[[古巴|古巴]]外籍人士對於片中將古巴人塑造成罪犯和毒販而感到反感。但在數年後，本片獲得影評人重新看待，它被一些人認為是最佳[[黑幫電影|黑幫電影]]之一，身為編劇和導演的[[馬丁·史柯西斯|馬丁·史柯西斯]]也稱讚本片。其音樂、漫畫和遊戲也在粉絲間造成迴響，形成了一股[[邪典追捧|邪典追捧]]。

==劇情==
[[卡斯楚|卡斯楚]]執政之後，許多人逃出[[古巴|古巴]]，東尼是眾多難民之一，臉上有一道長疤痕的他，與好友曼尼來到了[[美國|美國]][[邁阿密|邁阿密]]，一無所有的他們開始為黑道老大、[[毒梟|毒梟]]法蘭克辦事，多次立下汗馬功勞，以勇猛聞名於[[江湖|江湖]]。

但功高震主，法蘭克開始猜忌東尼，東尼與產地製毒商索薩的合作被法蘭克反對，加上法蘭克的女友艾薇拉似乎與東尼產生情愫；法蘭克憤怒，派人行刺東尼，東尼負傷，東尼與曼尼也隨即報復，殺死了法蘭克自立門戶，與索薩合作販毒，很快地成為家喻戶曉的大毒梟。

東尼發跡以後，因自身的傲慢、凶悍及貪婪，導致身邊的家人或朋友都因此而遠離，甚至被東尼殺死。例如忠實的夥伴曼尼，就因為與東尼的妹妹吉娜談戀愛而被東尼[[槍決|槍決]]。

一次與索薩的合作，東尼中途改變主意，不願意連帶殺死行刺對象的老婆與孩子，還殺了索薩的一名部下，導致與索薩的合作失敗，悲憤的索薩派出一支部隊，攻入了東尼的豪宅.....。

== 製作 ==
[[File:Oliver_Stone_01.jpg|thumb]]（圖片為1987年）為《疤面煞星》寫劇本的同時還為自己對[[可卡因|可卡因]]成癮而掙扎。]]
[[阿爾·柏仙奴|阿爾·柏仙奴]]在洛杉磯的{{link-en|蒂芙尼劇院|Tiffany Theater}}觀看過1932年的英文同名電影《[[傷面人|傷面人]]》後，便開始開發《疤面煞星》。後來他打電話給他的經紀人、監製{{link-en|馬丁·布萊格曼|Martin Bregman}}，並告訴布萊格曼重拍這部電影的可能性。<ref name="Movieline">{{Cite web |url = http://movieline.com/2011/08/24/al-pacino-did-not-want-michelle-pfeiffer-for-scarface-and-8-other-revelations-about-the-gangster-cla/ |title = Al Pacino Did Not Want Michelle Pfeiffer For Scarface and 8 Other Revelations About the Gangster Classic |date = 2011-08-24 |publisher = Movieline |language = en}}</ref>柏仙奴原本想保留電影中那段歷史時期的部分，但卻意識到由於它的誇張性質，所以很難去實現。[[薛尼·盧密|薛尼·盧密]]成為導演，開發了有關在[[馬列爾事件|馬列爾事件]]期間古巴人東尼·莫塔納抵達美國的概念。<ref name="Movieline" />

布萊格曼和盧密的創作差異導致盧密退出該項目。盧密希望做出一個更政治化的故事，集中在指責現任總統的政府令可卡因大量湧入美國，而布萊格曼則不同意盧密的觀點。布萊格曼用[[白賴仁·狄龐馬|白賴仁·狄龐馬]]取代盧密，並聘請編劇[[奧利華·史東|奧利華·史東]]，後來表示只用了四通電話來確保他們的參與。史東看過原本的1932年《傷面人》，而且不喜歡這部電影，所以他最初拒絕了該提議<ref name="David Konow">{{Cite web |url = https://creativescreenwriting.com/writing-in-a-very-dark-room-oliver-stone-revisits-scarface/ |title = Writing in a Very Dark Room – Oliver Stone revisits Scarface |author = David Konow |date = 2015-01-20 |publisher = creativescreenwriting.com |language = en}}</ref>。直到他與盧密交談後，他才接受這一提議，因為他們同意將該電影從以前的時代轉變成一部當代電影<ref name="David Konow" />，說：「薛尼有一個好主意，可以把1930年代美國禁酒令的黑幫電影製作成一部現代移民黑幫電影，涉及同樣問題，而我們用禁毒代替禁酒。禁毒與禁酒創造了同樣的犯罪類型，就像黑手黨。」<ref name="David Konow" />史東在應付自己的可卡因成癮的同時研究了劇本<ref>{{Cite web |url = https://www.shortlist.com/entertainment/films/15-things-you-probably-didnt-know-about-scarface/86316#item-12 |title = 15 Things You (Probably) Didn’t Know About Scarface |publisher = ShortList |language = en}}</ref>。他和布萊格曼做了他們自己的研究，前往佛羅里達州的邁阿密，在那裡他們獲得機會看到美國檢察官辦公室和有組織犯罪局的記錄。史東搬到巴黎寫劇本，認為他在美國時無法切斷他的成癮，在2003年的一次採訪中說他當時完全脫離毒品，「因為我不認為可卡因有助於寫作，這是對腦細胞具有非常破壞性的。」

==續集==
2001年，傳出片商將計劃推出《疤面煞星》的續集，名為《東尼之子》（''Son of Tony''），並由嘻哈歌手{{link-en|古巴·林克|Cuban Link}}主演<ref name="Ozone">{{cite web |url=http://www.ozonemag.com/feb2002/cubanlink.html |title=Son of Tony |accessdate=2007-01-02|work=Ozone Magazine| archiveurl = https://web.archive.org/web/20110927133005/http://www.ozonemag.com/feb2002/cubanlink.html | archivedate = 2011-09-27| deadurl=yes}}</ref>。但計劃引起了網上正反兩極的反應，幾年後，古巴·林克表示，他可能不再參與該計劃，主要因為版權和創意控制等問題<ref name="LatinRapper">{{cite web |url=http://www.latinrapper.com/featurednews41.html|title=Cuban Link Starts His Chain Reaction|accessdate=2007-01-02|work=Latin Rapper| archiveurl = https://web.archive.org/web/20140402165713/http://latinrapper.com/featurednews41.html | archivedate = 2014-04-02| deadurl=no}}</ref>。之後，續集的計劃並未有更進一步的消息，因此被認為已作廢。

==重拍版==
2011年，環球影業宣布，他們正在發展一步新版的《疤面煞星》電影。官方表示，這部片既不是續集也不是重拍，而是一部全新的電影，但它將會採用1932年電影《[[疤面|疤面]]》的元素。而馬丁·布萊格曼則也將監製1983年版本電影的重拍<ref>{{cite news|work=[[Deadline.com|Deadline.com]]|location=United States|publisher={{link-en|潘斯奇媒體公司|Penske Media Corporation|Penske Media Corporation}}|last=Fleming, Jr.|first=Mike|url=http://www.deadline.com/2011/09/universal-pictures-readying-a-new-scarface/|title=Universal Preps New ‘Scarface’ Movie|date=2011-09-21|archivedate=2011-09-21|archiveurl=https://web.archive.org/web/20131105230303/http://www.deadline.com/2011/09/universal-pictures-readying-a-new-scarface/|accessdate=2013-11-05 }}</ref>，劇本由[[大衛·艾亞|大衛·艾亞]]負責<ref>{{cite news|work=[[Deadline.com|Deadline.com]]|location=United States|publisher={{link-en|潘斯奇媒體公司|Penske Media Corporation|Penske Media Corporation}}|last=Fleming, Jr.|first=Mike|url=http://www.deadline.com/2011/11/david-ayer-to-script-updated-scarface/|title= David Ayer To Script Updated ‘Scarface’date=|date=2011-11-29|archivedate=2011-11-29|archiveurl=https://web.archive.org/web/20140407233049/http://www.deadline.com/2011/11/david-ayer-to-script-updated-scarface/|accessdate=2014-04-07}}</ref>，而[[大衛·葉慈|大衛·葉慈]]正在會談擔任導演<ref>{{cite news|first=Nikki|last=Finke|url=http://deadline.com/2013/07/david-yates-in-final-talks-for-scarface-as-universal-very-high-on-reboot-script/|title=David Yates In Final Talks For ‘Scarface’ Helm Now Universal “Very High” On Script|work=[[Deadline.com|Deadline.com]]|location=United States|publisher={{link-en|潘斯奇媒體公司|Penske Media Corporation|Penske Media Corporation}}|archiveurl=https://web.archive.org/web/20140407233042/http://deadline.com/2013/07/david-yates-in-final-talks-for-scarface-as-universal-very-high-on-reboot-script/|date=2013-07-31|archivedate=2013-07-31|accessdate=2014-04-07}}</ref>。

2014年3月24日，[[TheWrap|TheWrap]]報導，[[帕布羅·拉瑞恩|帕布羅·拉瑞恩]]在會談執導電影，而劇本是由{{link-en|保羅·艾登納西歐|Paul Attanasio}}負責。這部電影將會是一個原創的故事，被設定於現代的[[洛杉磯|洛杉磯]]，敘述墨西哥移民為了實現自己的美國夢而在黑社會中崛起<ref>{{cite web |url=http://www.thewrap.com/scarface-remake-director-pablo-larrain-says-yes-universal/ |title=EXCLUSIVE: Spider-Man 4 Circling John Malkovich, Anne Hathaway|work=[[The_Wrap|The Wrap]]|publisher=The Wrap News Inc.|location=[[聖塔莫尼卡|Santa Monica, California]]|date=2014-03-24|accessdate=2014-03-24|archiveurl=https://web.archive.org/web/20140421185247/http://www.thewrap.com/scarface-remake-director-pablo-larrain-says-yes-universal/|archivedate=2014-04-21|deadurl=no}}</ref>。強納森·赫曼於2015年3月被聘來為劇本的初稿進行重寫<ref>{{cite news|last1=Siegel|first1=Tatian|title='Scarface' Remake Moving Forward With 'Straight Outta Compton' Writer (Exclusive)|url=http://www.hollywoodreporter.com/news/scarface-remake-moving-forward-straight-782472|accessdate=2015-03-22|work=[[The_Hollywood_Reporter|The Hollywood Reporter]]|location=United States|publisher=[[Prometheus_Global_Media|Prometheus Global Media]]|date=2015-03-18}}</ref>。
