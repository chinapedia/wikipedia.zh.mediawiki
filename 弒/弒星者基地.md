'''弒星者基地'''（{{lang-en|'''Starkiller Base'''}}），是《[[星際大戰|星際大戰]]》系列[[電影|電影]]中的虛構[[要塞|要塞]]，在[[STAR_WARS：原力覺醒|STAR WARS：原力覺醒]]中是由第一軍團所制造的行星超光速武器。<ref name="Databank">{{cite web |url=http://www.starwars.com/databank/starkiller-base |archiveurl=http://www.starwars.com/databank/starkiller-base | archivedate = 2015-12-01 |title=Starkiller base  |work=[[Star_Wars_Databank|Star Wars Databank]] |publisher=''StarWars.com'' | accessdate = 2015-12-01}}</ref>。它和以前[[銀河帝國_(星球大戰)|銀河帝國]]製造的[[死星|死星]]不一樣，不是一個人造星體，而是由一個冰層星球改造而成<ref name="Databank">{{cite web |url=http://www.starwars.com/databank/starkiller-base |archiveurl=http://www.starwars.com/databank/starkiller-base | archivedate = 2015-12-01 |title=Starkiller base  |work=[[Star_Wars_Databank|Star Wars Databank]] |publisher=''StarWars.com'' | accessdate = 2015-12-01}}</ref>，由第一軍團的指揮官---[[STAR_WARS：原力覺醒#演員|史諾克]]所指揮<ref name="Databank">{{cite web |url=http://www.starwars.com/databank/starkiller-base |archiveurl=http://www.starwars.com/databank/starkiller-base | archivedate = 2015-12-01 |title=Starkiller base  |work=[[Star_Wars_Databank|Star Wars Databank]] |publisher=''StarWars.com'' | accessdate = 2015-12-01}}</ref>。

==概述==
該行星吸收恆星的能量，并且把能量存儲在核心容器內，以進行武器充能，充能大慨要20分鐘。隨著武器充能都吸收該恆星大量的能量，加快恆星的衰變，最後該恆星會隕滅成為[[紅巨星|紅巨星]]。而弒星者基地的武器把儲存的能量以超光速發射，快到達目標時分散。運行方式與[[死星|死星]]類似。

在[[STAR_WARS：原力覺醒|STAR WARS：原力覺醒]]中，第一軍團製造了弒星者基地，類似三十多年前[[安鐸戰役|安鐸戰役]]所製造的[[死星|死星]][[要塞|要塞]]<ref name="Wookieepedia">{{cite web |url=http://starwars.wikia.com/wiki/Starkiller_Base|archiveurl=http://starwars.wikia.com/wiki/Starkiller_Base | archivedate = 2015-11-17 |title=Starkiller base  |work=[[Wookieepedia|Wookieepedia]] |publisher=''Wookieepedia'' | accessdate = 2015-11-17}}</ref>，用來消滅新共和。電影中，芬恩（Finn）為了拯救被[[凱羅·忍|凱羅·忍]]帶走的芮（Rey），而和[[韓索羅|韓索羅]]與[[秋巴卡|秋巴卡]]闖入弒星者基地，解除核心容器的防護罩，讓[[X翼戰機|X翼戰機]]得以進行轟炸。之後弒星者基地核心容器瓦解，引發爆炸，整個行星崩毀成沒有陸地的岩漿行星，疑似被[[凱羅·忍|凱羅·忍]]殺死的[[韓蘇洛|韓蘇洛]]下落不明。

==幕後==
基地名字的由來是[[佐治·魯卡斯|佐治·魯卡斯]]在經典三部曲的早期草圖中給[[路克·天行者|路克·天行者]]的名字---「安尼金·弒星者」（{{lang-en|Annikin Starkiller}}）<ref name="Wookieepedia">{{cite web |url=http://starwars.wikia.com/wiki/Starkiller_Base|archiveurl=http://starwars.wikia.com/wiki/Starkiller_Base | archivedate = 2015-11-17 |title=Starkiller base  |work=[[Wookieepedia|Wookieepedia]] |publisher=''Wookieepedia'' | accessdate = 2015-11-17}}</ref>。
