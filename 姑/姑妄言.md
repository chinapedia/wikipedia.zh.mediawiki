《'''姑妄言'''》是一部古典[[艷情小說|艷情小說]]，作者為[[清代|清朝]][[曹去晶|曹去晶]]，約有百萬字，有二十四回本與六十回本。

== 背景 ==
《姑妄言》一書完成於雍正八年（1730年）<ref>此书《自序》署“雍正庚戌中元之次日三韩曹去晶编于独醒园”，其《林钝翁总评》署“庚戌中元后一日古营州钝翁书”，知是书成于雍正八年(1730)。</ref>，《自序》署“雍正庚戌中元之次日三韩曹去晶编于独醒园”，三韓在清初是指[[辽东|辽东]]<ref>顾炎武在《日知录·外国·三韩》条云：“今人谓辽东为三韩者”</ref>，可知作者曹去晶是遼東人。

== 故事梗概 ==
故事背景是明末，當時淫乱成风，流寇赤橫，主角是一名瞎眼妓女钱贵與书生钟情的故事；全书约百万字，內容極盡淫穢之事，有大量的各种[[性交|性交]]动作，性描写约占五分之一，世家丫环仆人幾乎都是荡妇，又以[[阮大铖|阮大铖]]和[[姚华胄|姚华胄]]两家最為淫亂，絲毫不遜於《[[金瓶梅|金瓶梅]]》，“所寫者有一女多男、一男多女，及男女混交、亂倫、男女同性戀和人獸雜交，如人狗交、人驢交、人猴交等。寫采戰法則有采陰補陽，采陽補陰，因采人反被人采而致死，仙狐求人陽精反失丹。寫春宮圖冊、春藥如揭被香、金槍不倒紫金丹、如意丹等。緬鈴、白綾帶子及角先生等淫具亦時常出現。古代艷情小說中之種種套數，種種工具，均出現在此小說中。”<ref>陈庆浩《出版说明》</ref>。第六回描写七位男学生引诱和[[輪奸|輪奸]]女学生阴姑娘，令人髮指；又有人猴相姦生子之謬事，不一而足。

== 評價 ==
此書又寫盡了童家、宦家、贾家、马家、阮家、姚家、易家、铁家、钟家等九大家族，其複雜度遠過於《[[紅樓夢|紅樓夢]]》。《姑妄言》素材来源廣泛，如《[[肉蒲团|肉蒲团]]》、《[[锋剑春秋|锋剑春秋]]》、《[[西游记|西游记]]》、《[[金瓶梅|金瓶梅]]》等，也寫[[李自成|李自成]]事，並大量抄錄[[陈鼎|陈鼎]]的《滇游记》、《黔游记》，[[陸次雲|陸次雲]]的《峒溪纤志》，保留大量的[[荤笑话|荤笑话]]、[[谚语|谚语]]、[[古语|古语]]、[[俗语|俗语]]、[[歇后语|歇后语]]等[[俗文學|俗文學]]，有史料作用。小說每回皆有總評，正文之中又有夾批，評點近五万字，批語多為林鈍翁所為，一般認為曹去晶和林钝翁係同一人。<ref>《总评》记：“予与曹子去晶，虽曰异姓，实同一体。自襁褓至壮迄老，如影之随形，无呼吸之间相离。生则同生，死则同死之友也。”</ref>陈益源表示“若谓作者是位说部奇才，作品是部小说奇书，丝毫也不为过”<ref>陈益源：《〈姑妄言〉素材来源初考》</ref>

== 現狀 ==
咸丰九年（1859年）由俄罗斯天文学家[[斯卡奇洛夫|斯卡奇洛夫]]（K.I.Skachkov）將二十四回本的《姑妄言》带回[[俄国|俄国]]。1867年收藏於圣彼得堡皇家公共博物馆。但此書卻在中國絕跡，1941年，上海人[[周越然|周越然]]發現残篇，仅存三回，即第四十回、第四十一回、第四十二回。由[[上海优生学会|上海优生学会]]出版残刊本《姑妄言》，为第四十回和第四十一回（第四十二回有残缺，故不印），封面並印有“海内孤本”的字樣。[[苏联|苏联]]解体后，漢學家[[李福清|李福清]]受邀前往[[台湾|台湾]]讲学，揭露《姑妄言》仍在俄罗斯的消息。1997年，台湾大英百科股分有限公司得到列宁图书馆的授权，1月初版，收入法国学者陈庆浩、台湾学者王秋桂主编之《思无邪汇宝》。
