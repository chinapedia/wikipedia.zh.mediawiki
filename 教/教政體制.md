'''教政體制'''（即'''教會體制'''），是指[[基督教|基督教]]在其教內實行的組織原則和管理制度。各[[基督教派|基督教派]]根據其對《[[新約|新約]]》中記載有關[[基督教會|基督教會]]組織的解釋或其教會的性質和職能而製定。一般大體分三種制度：主教制、長老制和公理制，一些教派實行混合性的制度。

== 主教制 ==
''{{main|主教}}''

主教制（Episcopal polity）是基督宗教中最古老的制度，教會中最高的權威為[[主教|主教]]（bishop，或譯為監督）<ref>[http://www.cclw.net/soul/jdjxysy/htm/chapter076.html 基督教信仰实义：教政]</ref>，該詞源自希臘文「episkopos」（επίσκοπος），即「epi」（由上往下）及「skopos」（觀察者），意為監督者（overseer）。

天主教和東正教設有[[三級聖品制|三級聖品制]]的[[聖秩聖事|聖秩聖事]]（Holy orders），在[[主教|主教]]之下，設有[[司鐸|司鐸]]（priest/elder，或譯為[[长老_(基督教)|長老]]、司祭、神父）和[[執事_(基督教)|執事]]（deacon，或譯為助祭），形成以主教為首的階級制度。教會按地域劃分[[教區|教區]]，教區的主教主管其內一切行政和教務工作。

實行主教制的教會主要包括[[天主教|天主教]]、[[東正教|東正教]]及東部其他較小古老公教會、基督新教的[[聖公宗|聖公宗]]和部份[[路德宗|路德宗]]教會等。基督新教實行的主教制度源自天主教，不同的主要是基督新教的主教可以[[結婚|結婚]]。現基督新教中，完全實行主教制的新教[[宗派|宗派]]只有[[普世聖公宗|普世聖公宗]]；而[[信義宗|信義宗]]和[[衛斯理宗|衛斯理宗]]則由各區會自行修訂及使用主教制的細則。

== 長老制 ==
''{{main|長老}}''

長老制（Presbyterian polity，或稱代議制、議會制）由[[約翰·加尔文|約翰·加尔文]]所提倡，是《[[新約聖經|新約聖經]]》提及的職銜。主張教會是由[[信徒|信徒]]群眾所組成，是一個以[[議會|議會]]形式管理地區教會的制度。<ref>[http://www.pcchong.com/mydictionary1/Special/church2.htm 教會論(二) - 領袖、管治、禮儀、目的]</ref>議會內的成員由各分堂選出[[长老_(基督教)|長老]]，代表該堂出席會議。在堂會中設[[牧師|牧師]]（pastor）、傳道師（Minister）、執事和長老，共同處理教會事務。[[改革宗|改革宗]]和[[長老會|長老會]]都實行長老制，而有些實行長老制的教會所實行的長老制具體規章則有所不同。

== 公理制 ==
''{{main|會眾制}}''

公理制（Congregational polity，又稱會眾制）由[[慈運理|慈運理]]所提倡，主張堂會有獨立主權，[[基督|基督]]才是會眾元首。<ref>[http://www.cclw.net/soul/jdjxysy/htm/chapter076.html 基督教信仰实义：教政]</ref>堂會的主任[[牧師|牧師]]（Senior Pastor）為教會內最高的決策者，由信徒[[民主|民主]]選聘。教會一切內部事務由會眾決議，具體制度和禮儀也由會眾釐定。<ref>[http://www.pcchong.com/mydictionary1/Special/church2.htm 教會論(二) - 領袖、管治、禮儀、目的]</ref>很多歷史比較新的教會也採用公理制，例如[[浸信會|浸信會]]、[[公理會|公理會]]、[[神召會|神召會]]、[[貴格會|貴格會]]以及不少獨立教會。一些公理制的教會自願在地區上組成一個無領導和從屬之分的聯會，以聯繫該宗派的和分堂。公理制有一個缺點，就是常常做成教會之間的交往和合作愈來愈少，最後甚至會使一些教會脫離其宗派獨立，造成[[宗派主義|宗派主義]]以及教會分裂。
