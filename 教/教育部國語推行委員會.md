'''教育部國語推行委員會'''，俗稱'''國語推行委員會'''、'''國語會'''，為[[中華民國教育部|中華民國教育部]]內的分支功能性委員會之一。在臺灣，2000年之前的國語推行委員會多針對[[正體漢字|正體漢字]]語文研究工作、國字字典編纂、國字文字與語言統整做實際統合工作；2000年後，則強調[[兩岸|兩岸]]語文研究及[[臺灣閩南語|臺灣閩南語]]、[[臺灣客家語|臺灣客家語]]及[[臺灣原住民族語|臺灣原住民族語]]的整理與教學。名為推展國語，實際業務為總整全國語文資訊及審議語言文字之標準。

2013年隨[[行政院功能業務與組織調整|行政院功能業務與組織調整]]，縮編為'''教育部終身教育司第四科'''（閱讀及語文教育科）。

== 歷史 ==
前身為1919年4月21日成立的「'''國語統一籌備會'''」，1928年[[國民政府|國民政府]]將其改組為「'''國語統一籌備委員會'''」；1935年「'''國語統一籌備委員會'''」因沒有[[經費|經費]]而結束，經[[吳敬恆|吳敬恆]]等人努力成立「'''教育部國語推行委員會'''」；1937年「'''教育部國語推行委員會'''」因[[戰爭|戰爭]]暫停運作，1940年擴大組織恢復工作；1945年[[立法院|立法院]]通過《教育部國語推行委員會組織條例》，「'''教育部國語推行委員會'''」組織正式獲得法源依據。

[[中國抗日戰爭|中國抗日戰爭]]後，[[臺灣|臺灣]]於1946年設[[臺灣省國語推行委員會|臺灣省行政長官公署國語推行委員會]]；1948年臺灣省行政長官公署改為臺灣省政府，臺灣省行政長官公署國語推行委員會改組為[[臺灣省國語推行委員會|臺灣省政府國語推行委員會]]（[[臺灣省國語推行委員會|臺灣省國語推行委員會]]）。1949年，因[[第二次國共內戰|第二次國共內戰]]，[[中華民國政府|中華民國政府]]撤退至臺灣，教育部並未恢復國語推行委員會的建置，當時只有臺灣省國語推行委員會維持運作。此後各縣市政府陸續設置當地的國語推行委員會，例如1969年11月8日[[臺北市政府|臺北市政府]]成立[[臺北市國語推行委員會|臺北市國語推行委員會]]。

1981年，教育部為加強國語文教育，根據《教育部國語推行委員會組織條例》恢復設立「教育部國語推行委員會」。

2013年1月1日，隨行政院功能業務與組織調整，「'''教育部國語推行委員會'''」組織與業務併入'''教育部終身教育司第四科'''（閱讀及語文教育科）。<ref name="併">{{Cite web |url = http://www.edu.tw/mandr |title = 教育部所屬網頁 |accessdate = 2013-08-30 |quote = 教育部國語推行委員會業務已隨教育部組織改造納入新成立的'''終身教育司'''(電話02-77366800)}}</ref>

2013年12月31日立法院通過廢止《教育部國語推行委員會組織條例》，2014年1月15日由總統明令廢止。至此，「教育部國語推行委員會」組織及法源依據正式走入歷史。整併入教育部終身教育司後，成為第四科－－閱讀及語文教育科的一部分，語文業務能量只有半個科，語文相關業務亦漸趨式微。其後雖有成立「本國語文教育推動辦公室」，但此係任務型之編組，成員仍為原閱讀及語文教育科人員，且負責語文業務之人員持續減少，語言文字相關業務於教育部組改後明顯未獲重視。

[[臺灣戰後時期|中華民國接管臺灣後]]，國語推行委員會推行國語已具成效，鄉土語言卻式微，包括閩南語、客家語及各原住民族語<ref>{{cite book |title = 中華民國政府出版品目錄彙編 |url = http://books.google.com/books?id=nC5QAAAAIAAJ |year = 1995 |publisher=國立中央圖書館 |isbn = 978-957-678-208-4}}</ref>。臺灣[[民主化|民主化]]後，若干縣市即相繼推行本土母語教學<ref name="古國順2005">{{cite book |author = 古國順 |authorlink = 古國順| title = 臺灣客語概論 |url = http://books.google.com/books?id=rUjpzScwZKUC&pg=PP7 |year=2005 |publisher = [[五南圖書出版公司|五南圖書出版公司]]|isbn=978-957-11-4012-4 |pages = 7–}}</ref>，國語推行委員會著手整理各種本土語言的拼音、用字等資訊<ref name="藍順德">{{cite book |author= 蓝顺德 |authorlink= 藍順德|title = 教科書政策與制度 |url = http://books.google.com.tw/books?id=1BcgH3NNqkEC&printsec=frontcover&hl=zh-TW&source=gbs_ge_summary_r&cad=0#v=onepage&q&f=false |year = 2006 |publisher = 五南圖書出版公司 |isbn = 978-957-11-4108-4 |pages = 127–}}</ref>。

教育部國語推行委員會依據相關法規規定之任務職掌有：
#關於本國語言文字整理之審議事項。
#關於本國語言文字標準書籍之編訂事項。
#關於本國語言文字資料之蒐集事項。
#關於本國語言文字教學方法之實驗改進事項。
#關於統一中外譯名音讀標準之訂定事項。
#關於推行國語教育人員之訓練事項。
#關於國內不識字者及僑居國外人民語文教育之設計實施及視導事項。
#關於邊疆地方施行語言教育之設計事項。
#其他關於語文教育事項。
