{{vgname|教父|en=The Godfather: The Game}}是一款2006年的[[開放世界|開放世界]]動作冒險[[電子遊戲|電玩遊戲]]，根據1972年[[教父_(電影)|同名電影]]改編的。由[[維瑟羅遊戲|藝電紅木岸]]開發，[[美商藝電|美商藝電]]發行。

本遊戲的特色之一是聘請部份電影中的演員進行[[配音|配音]]。參與的演員包括[[詹姆斯·肯恩|詹姆斯·肯恩]]飾演{{link-en|桑尼·柯里昂|Sonny Corleone}}、[[勞勃·杜瓦|勞勃·杜瓦]]飾演{{link-en|湯姆·赫根|Tom Hagen}}，以及[[艾巴·維戈達|艾巴·維戈達]]飾演{{link-en|薩爾瓦多‧泰西歐|Salvatore Tessio}}，但[[馬龍·白蘭度|馬龍·白蘭度]]由於健康狀況不佳無法參與配音。

遊戲系統採用非線性的高自由度，與《[[俠盜獵車手|俠盜獵車手]]》系列相似，但卻擁有多重結局。時間設定為1945年到1955年間的[[紐約|紐約]]，主角Aldo Trapani加入了[[黑手黨|黑手黨]]柯里昂家族，並需打到另外四個敵對勢力，把柯里昂家族躍升為紐約最大黑幫及成為黨內教父。

2009年推出根據1974年電影《[[教父2|教父2]]》而製作的{{link-en|教父2 (遊戲)|The Godfather II (video game)|同名遊戲續集}}。但續集的銷量不佳，而且評價比第一集差，美商藝電最終放棄[[教父3|第三集電影]]改編遊戲的計劃。<ref>{{Cite web |url = http://www.gamespot.com/articles/ea-buries-godfather-franchise/1100-6211737/ |title = EA buries Godfather franchise |date = 2009-06-11 |publisher = Gamespot |language = en}}</ref>
