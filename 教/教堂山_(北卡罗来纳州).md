'''教堂山'''（{{lang-en|Chapel Hill}}）是[[美国|美国]][[北卡罗莱纳州|北卡罗莱纳州]][[奥兰治县_(北卡罗莱纳州)|奥兰治县]]（Orange County, North Carolina）的一座城镇。成立于1819年<ref>{{cite web |url=http://townofchapelhill.org/index.asp?NID=3 |title=存档副本 |accessdate=2007-04-02 |0=2007-03-11 |deadurl=yes |archiveurl=https://web.archive.org/web/20070311123251/http://townofchapelhill.org/index.asp?NID=3 |archivedate=2007-03-11 }}</ref>。著名的公立學校[[北卡罗莱纳大学教堂山分校|北卡罗莱纳大学教堂山分校]]位于该镇。美国2012年人口估测显示，教堂山共有人口5.842万<ref name="fastFact">{{cite web |url = http://www.townofchapelhill.org/index.aspx?page=43 |title = Fast Facts about Chapel Hill |language = en |deadurl = yes |archiveurl = https://web.archive.org/web/20130115061253/http://townofchapelhill.org/index.aspx?page=43 |archivedate = 2013-01-15 }}</ref>。教堂山与[[罗利_(北卡罗来纳州)|罗利]]（Raleigh）和[[德罕_(北卡羅來納州)|达勒姆]]（Durham）围成[[三角研究园|三角研究园]]（Research Triangle Park, RTP) 。
