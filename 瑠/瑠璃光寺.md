'''瑠璃光寺'''是位在[[日本|日本]][[山口縣|山口縣]][[山口市|山口市]]香山町的[[曹洞宗|曹洞宗]][[寺院|寺院]]，山號「保寧山」（{{lang|ja|ほねいさん}}），本尊[[藥師如來|藥師如來]]，建於1471年，由當時統領此地的[[大內氏|大內氏]]重臣{{Tsl|ja|陶弘房}}之夫人所建立，開山（初代住持）為[[大庵須益|大庵須益]]。

此寺廟為象徵大內氏全盛時期{{Tsl|ja|大内文化|大內文化}}的象徵，也是「西之京・山口」的代表景點，寺內區域包括{{tsl|ja|五重塔}}被合稱為「香山公園」，也是著名的賞櫻和賞梅景點。<ref name="Ishin150">{{Cite web| url=http://ishin150-yamaguchi.com/spot001/| title=瑠璃光寺| publisher=山口商工會議所| work=明治維新150年的山口市旅遊網站| accessdate=2018-05-24| language=ja}}</ref>

== 歷史 ==

; 香積寺

現在的琉璃光寺位址最早原為由[[大內義弘|大內義弘]]為祭祀家族建於14世紀末的「香積寺」，大內義弘過世後，其弟{{Tsl|ja|大内盛見|大內盛見}}為悼念大內義弘開始在此地興建五重塔，並於1442年完成。<ref name="Ishin150"/><ref name="Yam-20161208">{{Cite news| url=https://n.yam.com/Article/20151208437069| title=日本三大名塔 琉璃光寺五重塔| newspaper=太陽網| author=吳貴華| date=2016-12-08| accessdate=2018-05-24}}</ref><ref name="JTO">{{Cite web| url=https://www.welcome2japan.tw/attractions/facilities/spots/shrines_temples/6186.html| title=琉璃光寺| publisher=日本觀光局| accessdate=2018-05-24}}</ref>

; 瑠璃光寺

而瑠璃光寺最初則為大內氏的重臣{{Tsl|ja|陶弘房}}於1468年過世後，其妻為悼念陶弘房於1472年所建立，最初設於現在的山口市仁保高野地區，名為「安養寺」，1492年才更名為「瑠璃光寺」，由於位於仁保，也會被稱為「仁保瑠璃光寺」。

; 遷移

17世紀江戶時代後，由於統領起地的[[毛利氏|毛利氏]]將藩廳設於[[萩市|萩]]，因此在1604年將香積寺遷移至萩，而原本位於仁保的瑠璃光寺則在1690年遷至此地。<ref name="Ishin150"/><ref name="Yam-20161208"/><ref name="JTO"/>

== 寺內建築 ==

; 五重塔
寺內的五重塔建於1442年，高31.2公尺，現已被列為國寶，和[[京都|京都]][[醍醐寺|醍醐寺]]及[[奈良|奈良]][[法隆寺|法隆寺]]的五重塔並稱「日本三名塔」。<ref name="Ishin150"/>

; 璃光寺資料館
展示五重塔的模型及資料，並有日本其他各地的五重塔資訊。

; 香山墓所
和位於[[萩市|萩市]]的天樹院、{{tsl|ja|大照院}}、{{tsl|ja|東光寺_(萩市)|東光寺_(萩市)|東光寺}}同樣為長州藩主的墓所，此地為[[明治維新|明治維新]]時藩主13代[[毛利敬親|毛利敬親]]的墓所。<ref name="Ishin150"/>

; 枕流亭
為江戶時代末期[[長州藩|長州藩]]和[[薩摩藩|薩摩藩]]達成[[萨长同盟|萨长同盟]]之約定時，薩摩藩的[[小松帶刀|小松帶刀]]及[[西鄉隆盛|西鄉隆盛]]造訪山口時與長州藩會談的場所，經過多次遷移後，現遷移至現址。<ref name="Ishin150"/>

; 露山堂
為江戶時代末期長州藩藩主毛利敬親將藩廳自萩遷移至此後，設於藩廳內的茶室，於1891年遷移至此。
