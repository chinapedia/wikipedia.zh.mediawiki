'''豌豆荚'''（{{lang-en|'''SnapPea'''}}）是一款免费的[[Android|Android]][[智能手机|手机]]应用搜索、管理软件，由豌豆实验室于2010年8月9日发布<ref>{{cite news | author = 环球企业家 | url = http://tech.sina.com.cn/i/2010-08-09/15134524013.shtml | title = 李开复创新首秀：豌豆荚手机精灵 | publisher =新浪科技 | date = 2010-08-09 | accessdate = 2013-09-16}}</ref>，是[[李开复|李开复]][[创新工场|创新工场]]首个公布的项目。提供Android应用、音乐、视频、电子书、壁纸和主题等下载服务，并可以通过豌豆荚在[[Windows|Windows]]电脑上收发短信，管理手机中的应用、联系人、图片和视频，备份手机数据到电脑和云端。<ref>[http://wandoujia.com 豌豆荚网站的功能简介]</ref>

==组成==
豌豆荚的程序由[[PC|PC]]客户端与[[Android|Android]]手机客户端组成。其中手机客户端提供了上述的基本功能，而电脑客户端的功能则更为详细，包括浏览和下载过往应用的旧版本，收发短信、管理联系人，在注册后修改应用的说明和图片（经审核后生效）等。

===连接===
手机经豌豆荚可以通过[[USB|USB]]、[[Wi-Fi|Wi-Fi]]两种方式与电脑进行数据通讯。在通讯过程中，手机以可读写[[FTP服务器|FTP服务器]]的形式与电脑进行通讯。<ref>{{Cite web |url=http://help.wandoujia.com/entries/22186532 |title=手机与豌豆荚连接不上怎么办 |access-date=2013-08-18 |archive-url=https://web.archive.org/web/20130817093905/http://help.wandoujia.com/entries/22186532 |archive-date=2013-08-17 |dead-url=yes }}</ref>

USB方式需要启动安卓系统的"USB调试功能"<ref>{{Cite web |url=http://help.wandoujia.com/entries/21550377 |title=【官方帮助】USB 调试开关的相关问题与解答 |access-date=2013-08-18 |archive-url=https://web.archive.org/web/20130824055813/http://help.wandoujia.com/entries/21550377 |archive-date=2013-08-24 |dead-url=yes }}</ref>。USB方式下豌豆荚程序还提供了[[屏幕截图|屏幕截图]]功能。

==应用来源==
豌豆荚的应用来源包括应用的官方网站及安卓市场、安智市场等中国大陆境内二十多个应用市场。<ref>{{Cite web |url=http://tech.qq.com/a/20111214/000407.htm |title= 豌豆荚推全平台战略 称收录30万款应用和游戏 | publisher=腾讯科技 | date=2011-12-14| accessdate=2013-09-16 }}</ref>

==旗下其它产品==
开眼(Eyepetizer)：豌豆荚出品的一款精品短视频日报应用。

豌豆荚一览：一款阅读类应用，可以在一个应用中浏览多个应用的内容。

==重大事件==
2013年9月14日，豌豆荚视频搜索正式版上线<ref>{{Cite web |url= http://tech.sina.com.cn/i/2013-09-14/03408739197.shtml |title= 豌豆荚视频搜索正式版上线  |author= 古晓宇 | publisher=新浪科技| date=2013-09-14| accessdate=2013-09-16 }}</ref>，视频搜索聚合了超过58万集的正规视频资源。从视频入手，豌豆荚「重新定义搜索」，并希望成为「移动互联网的内容入口」。<ref>{{Cite web| url=http://epaper.yicai.com:81/site1/html/2013-09/13/content_189409.htm| title=王俊煜：豌豆荚要做移动互联网内容入口| coauthors=马晓芳 刘奕诗| work=第一财经日报| publisher=第一财经日报| date=2013-09-13| accessdate=2013-09-16| deadurl=yes| archiveurl=https://web.archive.org/web/20130921060131/http://epaper.yicai.com:81/site1/html/2013-09/13/content_189409.htm| archivedate=2013-09-21}}</ref>

2014年1月15日，豌豆荚4.0新版发布<ref>{{Cite web |url= http://tech.sina.com.cn/mobile/n/2014-01-15/16109099235.shtml |title= 豌豆荚4.0新版发布：确定移动内容搜索核心战略  |author= 新浪手机 | publisher=新浪科技| date=2014-01-15| accessdate=2014-04-29 }}</ref>，从应用转向内容，实现了比应用商店更多的功能和理念。

2014年3月27日，豌豆荚发布应用内搜索技术协议<ref>{{Cite web |url= http://tech2ipo.com/64099 |title= 豌豆荚向开发者开放应用内搜索API |author= 偷笑的小猪 | publisher=tech2ipo| date=2014-03-27| accessdate=2014-04-29 }}</ref>，允许豌豆荚上已经收录的140万个应用为应用内容搜索提供内容。

2016年7月，阿里巴巴斥资2亿美元全资收购豌豆荚。<ref>{{Cite web |url= http://news.mydrivers.com/1/489/489871.htm |title= 2亿美元！阿里巴巴今天下午宣布收购豌豆荚  |author= pingwest | publisher=驱动之家| date=2016-07-05| accessdate=2016-07-05 }}</ref>
