'''事代主神'''（{{lang|ja|ことしろぬし の かみ}}）是[[日本神話|日本神話]]裡出現的[[神祇|神祇]]，別名'''八重言代主神'''（{{lang|ja|やへことしろぬし の かみ}}）、'''八重事代主神'''等。根據《[[記紀|記紀]]》二書，事代主神乃是[[大國主神|大國主神]]與神屋楯比賣命（{{lang|ja|かむやたてひめ の みこと}}）所生。

== 概要 ==
=== 日本書紀之記載 ===
據《[[日本書紀|日本書紀]]》卷第二之記載，第二種說法謂天神派遣[[經津主神|經津主神]]與[[建御雷|武甕槌神]]平定[[葦原中國|葦原中國]]，大己貴神（即[[大物主神|大物主神]]）報曰：「天神勅教，不敢不從命，我將讓出葦原中國之統治權，退居幕後。」於是向前述二神推薦[[岐神|岐神]]說：「此神可以代替我，我將從此避去。」隨即躬披瑞之八坂瓊而退隱。接著經津主神以岐神為前導，平定葦原中國。抗命者即加以斬戮，歸順者仍加以褒美。是時歸順之首乃大物主神及'''事代主神'''，於天高市率八十萬國津神，表現出順服之意。

=== 古事記之記載 ===
《[[古事記|古事記]]》上卷-{云}-，平定[[葦原中國|葦原中國]]的過程中，[[天照大神|天照大神]]派遣[[建御雷|建御雷]]前往召諭大國主神，後者與[[鳥之石楠船神|天鳥船神]]下降[[出雲國|出雲國]]伊那佐之小濱，拔出[[十拳劍|十掬劍]]問大國主神是否願意交出國家。大國主神回答子'''八重言代主神'''可代其回答，但後者前往狩鳥獵魚未歸。天鳥船神召回事代主神，後者向父親大國主神說應該交還葦原中國給天神之御子，隨即傾覆其船並擊打，使船化成青柴垣且隱於其中。

== 解說 ==
神名中的「事代（{{lang|ja|ことしろ}}，漢字亦可寫成『言代』）」乃知其所言之意，故為掌管[[神諭|神諭]]之神。能寫成「事」或「言」兩個漢字，是因為日本古代對這兩個同音字沒有區別。由於事代主神乃大國主神之子，屬[[大和國|大和國]]（今[[奈良縣|奈良縣]]）在地的神祇，故同樣為葛城（今[[奈良盆地|奈良盆地]]西南部）在地神祇的[[一事主神|一事主神]]繼承了祂一部分的神格。既然事代主神在葛城王朝中佔有重要地位，祂也成為{{link|ja|八神殿|八神殿|御巫八神}}之一。

另外[[神話|神話]]中提到事代主神翻覆船隻、化成青柴垣且隱於其中的故事，在[[伊豆群島|伊豆]]的[[三宅島|三宅島]]上也出現了「三島明神」，其[[香火|香火]]並傳衍至今[[靜岡縣|靜岡縣]][[三島市|三島市]]的[[三嶋大社|三嶋大社]]。

== 信仰 ==
在[[神話|神話]]故事中提到事代主釣魚之事，故一般認為祂是釣魚迷，引申為[[海神|海神]]、商業神、[[財神|財神]]等。祂的形象讓人聯想到右手持[[釣竿|釣竿]]、左手抱鯛魚的[[七福神|七福神]][[惠比寿_(神祇)|惠比壽]]，形成有趣的插曲。

祭祀事代主神的[[神社|神社]]主要有：
* 鴨都波神社（[[奈良縣|奈良縣]][[御所市|御所市]]）
* 賀茂別雷神社<ref>俗稱上賀茂神社。</ref>（[[京都府|京都府]][[京都市|京都市]]）
* 賀茂御祖神社<ref>俗稱下鴨神社。</ref>（京都府京都市）
* [[三嶋大社|三嶋大社]]（[[靜岡縣|靜岡縣]][[三島市|三島市]]）
* 美保神社（[[島根縣|島根縣]][[松江市|松江市]]）：另奉祀[[三穗津姬|三穗津姬]]神。
* 長田神社（[[神戶市|神戶市]][[長田區|長田區]]）
* 惠美須神社（[[京都市|京都市]][[東山區_(日本)|東山區]]）
* 今宮戎神社（[[大阪市|大阪市]][[浪速區|浪速區]]）
