《'''想见你'''》（{{lang-ja|会いたかった}}），是[[AKB48|AKB48]]的第1张[[单曲|单曲]]<ref>若包含独立制作时期单曲，则为第3张。</ref>。 由[[DefSTAR_Records|DefSTAR Records]]于2006年10月25日发售。印在封面以及CD等之上的“{{lang|ja|会いたかった}}”文字是由当时Team K的成员[[小野惠令奈|小野惠令奈]]所写的。中心成員由[[前田敦子|前田敦子]]擔當。

== 概要 ==
本作为AKB48的第1张由[[日本唱片协会|日本唱片协会]]正式会员单位发行的单曲。由此，AKB48达成了主流出道（{{lang|ja|メジャーデビュー}}）的目标。这一消息是于同年8月20日，在[[AKB48剧场|AKB48剧场]]的Team A公演时由剧场支配人[[户贺崎智信|户贺崎智信]]所宣布的。当场公演结束后，运营方向观众发放了关于此事的[[号外|号外]]。

在本作发售之前，AKB已经由非协会会员单位的唱片公司（[[AKS|AKS]]）发售了《[[樱花的花瓣们|樱花的花瓣们]]》（{{lang|ja|桜の花びらたち}}）及《[[裙襬飄飄|裙襬飄飄]]》（{{lang|ja|スカート、ひらり}}）两张单曲。

自本作起，Team K的成员也开始参加单曲的录制，共同竞争选拔成员的名额。Team K的选拔成员是第一次参加单曲录制。

本作被视为是AKB48的代表作之一。AKB48以本作首次登上了[[第58回NHK红白歌合战|第58回NHK红白歌合战]]的舞台。在[[综艺节目|综艺节目]]等当中提到AKB48时，使用最多的就是本作。

== 特典 ==
首批通常盘的盘面采用了图片的形式。初回限定盘附带一张特典DVD（内容下述）。
* 发售时在部分CD贩卖店举行了促销活动，向购买者发放了限量的活动参加券（握手会参加券）。
* 为纪念主流出道，发售后举办了数场由选拔成员举办的现场销售会，同时也起到了现场握手会的作用。
* 在出货的CD当中包含了36张金卡作为“惊喜特典”。内容是由在申请时所填写的成员前往当选者的家中，赠送一份私人物品。

== MV ==
[[音乐影片|PV]]的导演继《樱花的花瓣们》后再次由[[平川雄一朗|平川雄一朗]]担任。主角为[[前田敦子|前田敦子]]。拍摄是在[[千叶县|千叶县]][[馆山市|馆山市]]的周边进行的。其中出现的火车站是JR[[内房线|内房线]]的[[那古船形站|那古船形站]]。舞蹈的情景则是在同市南部的[[州崎灯塔|州崎灯塔]]附近。本作的PV首次带有了一些剧情，而非单纯的舞蹈情景。B面曲《但是…》的歌词与A面的部分剧情相通<ref>{{Cite web|date=2006-10-25|url=http://news.livedoor.com/article/detail/2622381/?p=3|title=采访：AKB48“天然性比较多呢”|publisher=livedoor|language=日语|accessdate=2011年2月18日}}</ref>。
