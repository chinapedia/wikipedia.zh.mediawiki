'''蓝天出版社'''，位于[[北京市|北京市]][[海淀区|海淀区]][[复兴路_(北京)|复兴路]]14号，是[[中国人民解放军空军政治工作部|中国人民解放军空军政治工作部]]主办和主管的中央级出版社，是[[中国人民解放军空军|中国人民解放军空军]]惟一的图书出版单位。<ref name=lantian>{{Cite web|url =http://info.beifabook.com/BookMenu/DataList.aspx?PsHouseID=7-80158|title =蓝天出版社|author =|date =|publisher =北发图书网|language =|accessdate =2017-01-16|quote =}}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

==简介==
蓝天出版社成立于1989年3月，是中国人民解放军空军惟一的图书出版单位。自建社以来至2010年代初，该出版社已出版图书1000多种，发行总量1000多万册，先后有38种图书52次在全图、全军获得各种奖项，20多种图书被评为全国最佳销畅书，部分图书被海外出版社购买版权<ref name=lantian/>。

2016年[[深化国防和军队改革|深化国防和军队改革]]中，该出版社的上级单位中国人民解放军空军政治部改为中国人民解放军空军政治工作部<ref>{{Cite web |url = http://news.xinhuanet.com/politics/2016-06/22/c_129081785.htm?from=timeline&isappinstalled=0 |title =关于对拟表彰的全国双拥模范个人进行公示的公告  |author =  |date =2016-06-22  |publisher =新华网  |language =  |accessdate =  |quote =  }}</ref>。2016年3月底，中央军委印发《[[关于军队和武警部队全面停止有偿服务活动的通知|关于军队和武警部队全面停止有偿服务活动的通知]]》。2016年5月7日，军队和武警部队全面停止有偿服务试点任务部署会议举行，7个大单位、17个具体单位被列为试点单位，新闻出版等重点项目被纳入试点范畴。自2015年起，各军队和武警部队出版社的新书及新项目均停止审批<ref>{{Cite web |url = http://www.cbbr.com.cn/article/104077.html |title = 军队出版社将试点“全面停止”有偿服务 |author =  |date = 2016-05-10 |publisher = 中国出版传媒网 |language =  |accessdate =  |quote =  }}</ref>。
