'''蓝瓶实验'''（{{lang-en|Blue bottle}}），一种常见的化学示范实验。

==描述==
在容器中加入少量[[氢氧化钠|氢氧化钠]]和[[葡萄糖|葡萄糖]]，加水溶解。再加入少量[[亚甲基蓝|亚甲基蓝]]并[[搖晃|搖晃]]，此时溶液呈蓝色。静置一段时间后，蓝色消失，溶液变为无色。再次振荡，溶液又恢复蓝色。再次静置，溶液再变为无色，如此反复。

==原理==
该体系中的[[还原剂|还原剂]]是[[葡萄糖|葡萄糖]]，[[氧化剂|氧化剂]]则是[[空气|空气]]中的[[氧气|氧气]]。[[亞甲藍|亚甲蓝]]为[[氧化还原指示剂|氧化还原指示剂]]，[[氧化态|氧化态]]亚甲蓝呈蓝色，还原态亚甲蓝呈无色。振荡溶液时亚甲蓝被氧气氧化，溶液呈蓝色。静置时氧化态亚甲蓝逐渐被葡萄糖还原，转为无色，葡萄糖则被氧化为阿拉伯糖酸、[[甲酸|甲酸]]、[[草酸|草酸]]、赤蘚糖酸等产物。

==实验目的==
# 了解控制化学反应条件的作用
# 通过观察亚甲基蓝和还原亚甲基蓝在不同条件下的相互转化，学习观察方法，了解对比实验法

==实验用品==
* 0.1%亚甲基蓝溶液、30%NaOH溶液、葡萄糖、[[蒸馏水|蒸馏水]]
* [[錐形瓶|锥形瓶]]、[[试管|试管]]、[[滴管|滴管]]、橡胶塞、[[烧杯|烧杯]]、[[酒精灯|酒精灯]]、[[量筒|量筒]]、托盘[[天平|天平]]、[[溫度計|温度计]]、[[計時器|计时器]]

==实验步骤==
# 往锥形瓶中加50mL水，1.5克葡萄糖，逐滴滴入8～10滴0.1%亚甲基蓝，振荡至溶液呈蓝色。
# 加入2mL30%的NaOH溶液，振荡并静置锥形瓶，观察并记录现象。再振荡锥形瓶至溶液变蓝，再次静置锥形瓶，连续记录两次振荡周期（NaOH的用量不能太多）。
# 将溶液分装在两个小试管中，一号试管装满，二号试管装半管，均用塞子塞好，振荡，静置。
# 把一号试管溶液分一半到三号试管中，再在三号试管中加5滴0.1%亚甲基蓝，塞好两支试管，振荡、静置。
# 将一、三号试管置于40℃水浴中，约10分钟后振荡、静置（水浴加热至40℃，水温不能太高，否则，溶液变黄失效）。
# 实验结束时，集中回收反应溶液，留作以后使用。<ref>https://www.zybang.com/question/ef39d6d4c56f74fda437d7b908088efe.html</ref>

==实验现象==
# 一号试管始终呈无色，二号试管振荡后变蓝，然后自下而上褪色，约45秒。
# 一号试管振荡后变蓝，然后自下而上褪色约20秒，三号试管振荡后变为蓝色且较一号试管颜色深，然后自下而上褪色，约15秒。

==注意事项==
* [[氢氧化钠|NaOH]]的用量太多或[[水浴|水浴]]加热的水温太高会导致葡萄糖在强碱性条件下形成双键在不同位置的[[烯醇|烯醇]]式和碳键断裂[[分解作用|分解]]为[[醛|醛]]，醛又[[聚合|聚合]]生成树脂状物质，最终使溶液变黄失效。
** NaOH浓度太高也会使葡萄糖还原能力过强从而导致无法变蓝。
** 混合液静置较长时间后也会变成金黄色而失效。
