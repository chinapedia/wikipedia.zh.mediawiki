'''蓝宝石号'''（{{lang-de|'''''Saphir'''''}}）是曾经运行于[[比利时|比利时]][[港口|港口]][[奥斯滕德|奥斯滕德]]至[[德国|德国]][[鲁尔区|鲁尔区]]工业城市[[多特蒙德|多特蒙德]]间一班国际[[列车|列车]]所使用的名称，由[[德国联邦铁路|德国联邦铁路]]在1954年至1987年间经营。其命名源自[[蓝宝石|蓝宝石]]的德文名称，藉此凸显比利时在[[宝石|宝石加工业]]的重要地位<ref group="註">{{harvnb|UIC|1972|p=20}}.</ref>。列车自开行以来曾先后被纳入[[快速列车_(德国)#长途列车|长途列车]]、[[全欧快车|全欧快车]]和[[城际列车_(欧洲)|城际列车]]等类别，直至1987年被[[欧城列车|欧城列车]][[梅姆林号|梅姆林号]]取代而停运。在1993年至1995年期间，蓝宝石号的名称也曾被用于德国境内开行的[[城际快车|城际快车]]班次。

==历史==
===长途列车===
蓝宝石号最初是在1954年作为[[二战|二战]]前运行于[[多特蒙德|多特蒙德]]－[[奥斯滕德|奥斯滕德]]的{{link-de|普尔曼快车|Pullman-Express}}的继任者<ref group="註">{{harvnb|Goette|2008|p=87}}.</ref>，由[[德国联邦铁路|德国联邦铁路]]以[[快速列车_(德国)#长途列车|长途列车]]（{{lang|de|F}}）类别开行，并途经[[杜塞尔多夫|杜塞尔多夫]]、[[科隆|科隆]]、[[亚琛|亚琛]]、[[列日|列日]]及[[布鲁塞尔|布鲁塞尔]]等地运行<ref group="註" name="p8">{{harvnb|Cook|1954|p=8}}.</ref>。其时刻表设置可以与奥斯滕德－[[多佛尔|多佛尔]][[火车渡轮|火车渡轮]]及多佛尔－[[伦敦|伦敦]]的列车服务实现中转，从而在整体的运行图中允许乘客在上午10点离开伦敦（[[維多利亞車站_(倫敦)|维多利亚车站]]），但比原先提前三个小时抵达[[莱茵-鲁尔城市群|莱茵-鲁尔城市群]]的目的地<ref group="註">{{harvnb|Scharf|Ernst|1983|p=187}}.</ref>。

蓝宝石号使用{{link-de|德国联邦铁路VT08.5型柴联车|DB-Baureihe VT 08}}担当本务。在运营初期，蓝宝石号在[[英文|英文]][[刊物|刊物]]中按英文使用习惯被称为“{{lang|en|Sapphire}}”，但到了1963年，在英文的《库克欧陆时刻表》（{{lang|en|Cook's Continental Timetable}}）中便开始使用以德文命名的“{{lang|en|Saphir}}”<ref group="註">{{harvnb|Cook|1963|p=59}}.</ref>。

===全欧快车===
1957年6月2日，蓝宝石号成为首批被纳入[[全欧快车|全欧快车]]（{{lang|en|TEE}}）类别的列车之一，车次使用TEE74（东行）/75（西行）次<ref group="註" name="p188">{{harvnb|Mertens|Malaspina|2007|p=188}}.</ref>。VT08.5型柴联车随即在同年7月15日被替换为专为全欧快车设计的[[德國聯邦鐵路VT11.5型柴聯車|VT11.5型柴联车]]。同时，列车可在[[科隆火车总站|科隆火车总站]]与另一班全欧快车[[莱茵-美因号|莱茵-美因号]]实现中转，从而从而提供由布鲁塞尔经[[莱茵河|莱茵河谷]]至法兰克福的全欧快车接驳服务。

一年后，由于多特蒙德所在的[[鲁尔区|鲁尔区]]内已有[[巴黎-鲁尔号|巴黎-鲁尔号]]及[[帕西法尔号|帕西法尔号]]两班全欧快车提供鲁尔区经[[比利时|比利时]]南部至[[巴黎|巴黎]]的服务，为了避免重复运营，蓝宝石号在德国境内的始发及终到站遂由多特蒙德改至[[法兰克福|法兰克福]]，同时列车在法兰克福－科隆区间会与北行的莱茵-美因号连挂运行，而两班列车在南行方向则各自运行<ref group="註" name="pp189">{{harvnb|Mertens|Malaspina|2007|pp=189-191}}.</ref>。

随着蓝宝石号在1959年5月31日的运行图调整中改为经由莱茵河右岸的线路运行，并在[[威斯巴登|威斯巴登]]经停及换向，列车与莱茵-美因号的连挂运行被终止<ref group="註" name="pp189"/>。1966年，由于蓝宝石号在奥斯滕德－布鲁塞尔区间的上座率不景气，其比利时境内的始发及终到站又改至布鲁塞尔，因此列车的线路亦变为布鲁塞尔－法兰克福间运行。1971年夏天，西欧各国对列车车次进行了统一全面调整，由布鲁塞尔东行的列车使用[[奇数|奇数]]车次，由法兰克福西行则使用[[偶数|偶数]]车次，因此蓝宝石号获得了编号为TEE21/20次的车次<ref group="註">{{harvnb|Scharf|Ernst|1983|pp=347-348}}.</ref>。

[[File:TEE_Saphir_in_Intercity_network.svg|thumb]]
在1971年9月26日的运行图调整中，德国联邦铁路开始构建由四条线路组成的[[城际列车_(欧洲)|城际列车]]（{{lang|en|IC}}）网络，蓝宝石号也被定位为该网络的组成部分，与其它全欧快车及城际列车共同被纳入发车间隔为每两小时一班的运营线路。列车在此时将西行的始发站改至[[纽伦堡|纽伦堡]]，并将编组更换为由[[铁路机车|机车]]牵引的客车车厢，使用由德国联邦铁路提供的{{link-de|TEE/IC型车厢|TEE/IC-Wagen (DB)}}。而为了提高运营效率，其车厢会与运行于不同区间的德国国内列车班次轮番运用<ref group="註" name="pp191">{{harvnb|Mertens|Malaspina|2007|pp=191-192}}.</ref>。

蓝宝石在纽伦堡－科隆路段是作为城际列车2号线（[[慕尼黑|慕尼黑]]－法兰克福－科隆－[[汉诺威|汉诺威]]）的组成部分。然而，城际列车2号线原本在法兰克福至[[科布伦茨|科布伦茨]]区间是经由威斯巴登的莱茵河[[右岸|右岸]]运行，但蓝宝石号则是一个例外，它在东行沿莱茵河[[左岸|左岸]]运行，并经停[[美因茨|美因茨]]；西行才经停右岸的威斯巴登，并连续在[[法兰克福火车总站|法兰克福火车总站]]及{{link-de|威斯巴登火车总站|Wiesbaden Hauptbahnhof}}进行[[换向|换向]]<ref group="註" name="pp191"/>。此外，东行的蓝宝石号可在科隆火车总站与全欧快车莱茵-美因号（[[阿姆斯特丹|阿姆斯特丹]] → [[波恩|波恩]]）、运行于城际列车1号线的贵族号（{{lang|de|''Patrizier''}}，[[汉堡|汉堡]] → 科隆）和运行于城际列车2号线的日耳曼尼亚号（{{lang|de|''Germania''}}，汉诺威 → 科隆）实现中转。西行的蓝宝石号则可科隆火车总站与运行于城际列车1号线的施瓦本之箭号（{{lang|de|''Schwabenpfeil''}}，[[斯图加特|斯图加特]] → 汉堡）、运行于城际列车2号线的全欧快车[[钻石号|钻石号]]（布鲁塞尔 → 汉诺威）实现中转<ref group="註">{{harvnb|Mertens|Malaspina|2007|p=186}}.</ref>。

===城际列车===
自1978年5月28日起，德国联邦铁路开始在蓝宝石号所行经的城际列车2号线区间中搭载二等车厢。其中法兰克福至布鲁塞尔路段的西行列车（TEE20次）仍享受全欧快车待遇，而纽伦堡至法兰克福路段搭载二等车厢的编组则获得相同的名称、但使用IC20次的车次编号<ref group="註">{{harvnb|Scharf|Ernst|1983|p=421}}.</ref>。

1979年5月27日，蓝宝石号搭载二等车厢的运行区间被拓展至全路段，因已不再符合全欧快车的标准，它在德国境内被降级为城际列车类别<ref group="註">{{harvnb|Scharf|Ernst|1983|p=451}}.</ref>。至1980年6月，列车在全路段都被定位为国际性质的城际列车类别<ref group="註">{{harvnb|Malaspina|2006|pp=16-19}}.</ref>，同时它在德国境内的始发/终到站回迁至西部，但设于科隆而非多特蒙德<ref group="註">{{harvnb|Cook|1980|pp=96-97}}.</ref>。1981年，列车的运行线路的西端重新由布鲁塞尔延长至奥斯滕德始发及终到，提供由奥斯滕德－布鲁塞尔－科隆的服务<ref group="註">{{harvnb|Scharf|Ernst|1983|p=804}}.</ref>。自1984年6月3日起，[[比利时国家铁路|比利时国家铁路]]在相同的区间构建城际列车B线，蓝宝石号在此被纳入发车间隔为每两小时1班的服务网络<ref group="註">{{harvnb|Malaspina|2006|p=28}}.</ref>。

===后续运营===
1987年5月31日，随着[[欧城列车|欧城列车]]（{{lang|en|EC}}）类别的全新面世，蓝宝石号被运行于相同区间的欧城列车[[梅姆林号|梅姆林号]]取代而停运<ref group="註">{{harvnb|Cook|1987|p=51}}.</ref>。

在1993年5月23日的运行图调整中，梅姆林号的服务区间被延长至奥斯滕德－多特蒙德间运行<ref group="註">{{harvnb|Cook|1993|p=3}}.</ref>。此时每天共有4班全欧快车服务运营于布鲁塞尔－科隆间，梅姆林号便是其中之一<ref group="註" name="pp102">{{harvnb|Malaspina|2005|pp=102-103}}.</ref>。1997年12月14日，随着连接巴黎经布鲁塞尔至科隆之间的[[大力士高速列车|大力士高速列车]]开始投入运营，梅姆林号就此停运<ref group="註" name="pp102"/>。

另一方面，蓝宝石号的名称在1993年至1995年间也被用于德国国内的[[城际快车|城际快车]]（{{lang|de|ICE}}）服务中，但其运行区间与全欧快车时期的蓝宝石号已不尽相同<ref group="註" name="pp191"/>：

*1992年5月31日–1993年5月22日：斯图加特－汉堡
*1993年5月23日–1995年5月27日：[[卡尔斯鲁厄|卡尔斯鲁厄]]－汉堡

===现况===
自2002年以来，布鲁塞尔－法兰克福间开设有经由[[科隆-莱茵/美因高速铁路|科隆-莱茵/美因高速铁路]]运行的直通城际快车服务<ref group="註">{{harvnb|Cook|2002|p=3}}.</ref>。在2010年夏季的运行图中，每天共有3班城际快车往返于[[布鲁塞尔南站|布鲁塞尔南站]]至法兰克福火车总站之间，其单程耗时约为3小时05分至3小时15分。同时每天也有6班大力士高速列车往返于巴黎－布鲁塞尔－科隆之间，其发车间隔约为每一小时或两小时1班<ref group="註">{{harvnb|Cook|2010|p=21}}.</ref>。而由奥斯滕德始发前往德国，或由比利时始发前往鲁尔区的列车服务则已不复存在。

==使用车辆==
===柴联车===
[[File:TEE_im_DB_Museum_Nürnberg.jpg|thumb]]
在1954年的开行初期，蓝宝石号使用VT08.5型柴联车提供运营<ref group="註">{{harvnb|Scharf|Ernst|1983|p=874}}.</ref>。在列车于1957年6月2日升级为全欧快车后，本应开始替换德国联邦铁路专为全欧快车设计的VT11.5型柴联车（1969年后改称为601型<ref group="註">{{harvnb|Mertens|Malaspina|2007|pp=54-59}}.</ref>）。但因车辆交付延期，这项替换直至同年7月15日才完成<ref group="註">{{harvnb|Scharf|Ernst|1983|p=312}}.</ref>。VT11.5型柴联车采取7节固定编组（头尾2节[[动车组动车|动力车]]及中部5节[[动车组拖车|载客拖车]]）组成，其中载客拖车包括3节一等座车、1节一等座车/{{link-fr|酒吧车|Voiture-bar}}合造车和1节[[餐车|餐车]]，每日在多特蒙德或法兰克福－布鲁塞尔或奥斯滕德间执行1个往返的任务<ref group="註">{{harvnb|Mertens|Malaspina|2007|p=193}}.</ref>。

===客车车厢===
1971年9月26日起，蓝宝石号开始改用由机车牵引的“TEE/IC型车厢”<ref group="註" name="pp191"/>。这是德国联邦铁路在1962年专为[[快速列车_(德国)#长途列车|长途列车]]莱茵的黄金号所设计及稍作改进的型号，因此也被称为“莱茵的黄金型车厢”（{{lang|de|Rheingold-Wagen}}），它们均成为德国联邦铁路所担当的全欧快车或城际列车的车厢规范<ref group="註" name="pp102">{{harvnb|Mertens|Malaspina|2007|pp=102-111}}.</ref>。

客车的标准编组由5节一等车厢构成，包括1节[[隔间座车|隔间座车]]、2节[[开放座车|开放座车]]、1节隔间座车/酒吧车合造车和1节餐车<ref group="註" name="p188"/>。其编组与[[柴联车|柴联车]]时代的客车编组相同，但容量有所增加，这是因为单节客车车厢的长度大于VT11.5型柴联车的单节拖车长度<ref group="註" name="pp102"/>。这些客车车厢在当时会与其它全欧快车或城际列车班次执行一个复杂的套跑运用，例如在1971年的冬季运行图中，它们会在下列班次间执行为期3天的套跑运用<ref group="註">{{harvnb|Scharf|Ernst|1983|p=385}}.</ref>：

至1972年5月23日，隔间座车/酒吧车合造车已从编组中摘除，列车编组减为4节。此时它主要与另两班全欧快车[[范·贝多芬号|范·贝多芬号]]及伦勃朗号执行一个为期5天的套跑运用<ref group="註" name="pp191"/>。1974年，列车编组中的餐车又被隔间座车/酒吧车合造车所取代<ref group="註" name="p188"/>。

自1978年起，德国联邦铁路开始在列车运行的纽伦堡－法兰克福区间加载二等车厢，1979年后又将加载二等车厢的区间拓展至全路段。在1982年的冬季运行图中，蓝宝石号的标准编组由7节车厢构成，当中包括2节一等车厢、4节二等车厢及1节餐车<ref group="註">{{harvnb|Malaspina|2006|p=32}}.</ref>。

1987年，在列车转变为欧城列车类别并更名为梅姆林号后，开始使用由比利时国家铁路提供的客车车厢。在1993年的运行图中，梅姆林号的标准编组由7节车厢构成，当中包括2节一等车厢、4节二等车厢及1节二等车/[[自助餐车|自助餐车]]合造车<ref group="註">{{harvnb|Malaspina|2006|p=44}}.</ref>。

===牵引机车===
[[File:Raeren_abgestellte_Lok_6.jpg|thumb]]
自1971年以来，蓝宝石号在原则上以科隆为界、分别由德国联邦铁路及比利时国家铁路的[[电力机车|电力机车]]担当牵引任务<ref group="註" name="pp191"/>。其中比利时国家铁路最初使用{{link-en|15型电力机车|SNCB Class 15}}、1973年后改用[[比利时国铁18型电力机车_(阿尔斯通)|18型电力机车]]或{{link-en|16型电力机车|SNCB Class 16}}作为主要的牵引机车<ref group="註" name="pp84">{{harvnb|Mertens|Malaspina|2007|pp=84-85}}.</ref>。他们均为可适应3000[[伏特|伏]][[直流电|直流电]]（比利时）和15千伏16⅔[[赫兹|赫兹]][[交流电|交流电]]（德国）的双电压制式机车，可以直接通过两国边境的[[亚琛火车总站|亚琛火车总站]]而无需换挂。然而列车有时也会在比利时境内的部分路段使用直流电专用的{{link-en|23型电力机车|SNCB Class 23}}<ref group="註" name="pp84"/>。

列车在科隆以东的德国境内路段则主要使用[[德国铁路103型电力机车|103型电力机车]]担当本务<ref group="註" name="pp84"/>。
