'''蓝天下的单相思'''（{{lang|ja|青空片想い}}），是[[SKE48|SKE48]]的第2张单曲。分为通常盘与剧场盘两种样式，由[[w:ja:日本クラウン|日本クラウン]]于2010年3月24日发售。


== 概要 ==
* 与前作（《[[坚强之勇者|坚强之勇者]]》）发售时间相隔7个月，也是与[[w:ja:日本クラウン|日本クラウン]]签约后发售的首张单曲。
* 选拔组成员制度自本作起开始实施。
* [[音乐影片|PV]]是在位于[[东京都|东京都]]幕张的APA HOTEL之楼顶（高度距地面有190米）拍摄，当时现场温度只有-2°C<ref> SKE48 Zepp Nagoya公演MC寸剧 に,今,できることD3</ref>。

== 选拔组成员 ==
* 所有SKE48成员均参加了本作的录制。
; 团队所属情况均截止至发售时
