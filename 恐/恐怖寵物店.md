《'''恐怖寵物店'''》（Petshop of Horrors）是[[日本漫畫家|日本漫畫家]][[秋乃茉莉|秋乃茉莉]]所繪的[[恐怖漫畫|恐怖漫畫]]作品。以前由《Apple Mystery》月刊漫畫雜誌連載。1998年，在該雜誌暫時停刊以後，再次於月刊漫畫雜誌《Horror M》連載。1999年3月被動畫化，在[[TBS|TBS]]的深夜節目的「Wonderful」內播放。2004年4月，在是[[朝日|朝日]]有聲雜誌（{{lang|ja|朝日ソノラマ}}）的雙月刊漫畫雜誌《{{lang|ja|ネムキ}}》的增刊，季刊「夢幻館」中，開始新連載『新 Petshop of Horrors』，並把作品內的舞台從洛杉磯的唐人街遷移了到新宿的歌舞伎町。

==概要==
充斥各種光怪陸離人事物的華人街裡，隱藏著一間販賣各種寵物，名為「COUNT D」的寵物商店，從普通常見的小動物乃至於遊走法律邊緣，甚至只有傳說裡才會出現的生物，都會出現在這間店裡。只要顧客提出要求條件，做為現任店主的神祕人士'''D'''，將會為顧客選出最適合的寵物，幫助顧客實現願望。故事便環繞在D伯爵和各類生物及人們所發生的事件。

==登場人物==
===主要登場人物===
;D
: 人物本名不詳，人稱D伯爵（カウント・ディー），本作主人公兼身份最神秘的角色。有著異色雙瞳的短髮中國人，身穿中國服裝，常常帶著微笑。作為本來的店主的「D伯爵」的孫子擔任著寵物商店「COUNT D」的看管人，常與該地的名士和官員來往。特別喜歡甜食而且有著無論怎麼吃甜食也不胖的體質，擅長著中國武術。對動物很溫柔但對人很厭惡的性格，特別對於動物的飼養方法。自稱受封的爵位只到伯爵的祖父為止。右眼具備抹除他人記憶的能力，能與動物交談，有著大量關於動物和傳說中的動物的知識，能用血召喚生物。

; 雷歐‧歐雷克特
:美國警察，因為查緝毒品事件而來到D伯爵的店後，發現凡是跟他購買寵物的人都離奇死亡，後來就常常往他那裡跑。經常帶甜點去誘惑D伯爵想使他招供，時常與D伯爵拌嘴，但亦有聯手幫助彼此的時候。於單行本第五集第四話酒醉時能看見人型姿態的動物，隨著劇情發展開始能和動物溝通，在第一部劇情最終章時，也能在清醒狀態見到動物的人型姿態。在第二部劇情僅於番外篇登場。

:其上司也是D伯爵的顧客。

; 克利斯多福
:雷歐‧歐雷克特的弟弟，母親逝世後交由阿姨照顧，卻因為表姐莎曼珊無心的話語，受到刺激無法說話，但可以見到D伯爵店裡動物的人型姿態並與其交談，D伯爵也能聽見克利斯多福的「心聲」與他溝通。故事尾聲被朱香啟示敞開心房，返回現實，卻因此喪失了與動物交談的能力。在第二部劇情已經成年，擔任[[FBI|FBI]]的探員。

; 小Q
:常在D伯爵身邊，有著兔耳、蝙蝠翅膀的神秘生物，真實身份為D伯爵之祖父，容貌與D伯爵相仿。

;D伯爵的父親
:本名不詳，容貌和D伯爵相仿，年輕時期曾為紐約大學赴香港的留學生，但極其厭惡人類，因認定D伯爵對待人類過於仁慈，導致父子倆理念不合，但依然在私底下關心著D伯爵。最後因為主導毀滅人類的計畫，被雷歐‧歐雷克特持槍射殺，最終重生為D伯爵的後嗣。

; 吉兒
:擔任市警的女刑警，大雷歐兩年的學姐。冷靜沉著，邏輯怔推理和搜查是她的信條。對D伯爵很親切。

; 劉武飛
: 第二部劇情登場，為新宿歌舞伎町的新中國城大樓老闆，劉氏集團暨[[黑手黨|黑手黨]]首領劉翁與情婦所生之子，被D伯爵稱為「太子」。哈佛大學畢業，性格和雷歐如出一轍，經常和D伯爵拌嘴，不相信毫無科學根據的事物。有位同父異母的姊姊。對於父親經常催婚的行為感到相當不滿。

; 老陳
: 第二部劇情登場，為劉武飛的隨身管家，自劉武飛年少時便負責照顧他，對其相當忠誠。

===非人生物===
; [[饕餮|饕餮]]（王師傅、阿徹）
:中國古代漢族神話中的一種怪獸。羊身人面，目在腋下，虎齒人爪。饕餮，尤喜食人肉，古文曰饕為貪食，餮為貪財，是以饕餮也有貪婪之意思。
:伯爵在中國城中一中國餐館發現主廚王師傅是一隻修成人形的成年饕餮後，不惜以身犯險做餌，意欲將其帶回店中。卻在最後關頭被雷歐打傷饕餮，計劃失敗。
:最終王師傅被收監，最終死於自噬。
:雷歐前往寵物店告知伯爵這個消息，伯爵懷中正抱著不知從哪裡找到的阿徹。（师傅似乎与饕餮签定了契约，然而随着他逐渐的利欲熏心，身上所附的饕餮也变得暴虐了起来，后来王师傅自嗜后，加上来探监的伯爵的开导，重新恢复理智并加入宠物店。）
:人型姿態是個留著紅長髮，長有羊角，穿著袒胸上衣和長褲的衝動青年。
; [[麒麟|麒麟]]（慶麟）
:與[[龍|龍]]、[[鳳凰|鳳凰]]、[[玄武|玄武]]並列為稀有珍獸，性格慈悲愛護生命，就連花草也不忍踐踏。與其締結契約的主人，會在將來成為君主或民族領導人，但往往都得用人民或自己的血付出代價，麒是雄性的麒麟，麟則是雌性的麒麟。
:慶麟的人型姿態，是個穿戴古代華麗衣裝的小女孩。在第二部劇情登場時已經成年，說話第一人稱為「妾身」。
; [[鳳凰|鳳凰]]（翠凰）
:稀有珍獸之一，鳳是雄性，凰是雌性。只要有人與鳳凰立下寵物契約，即使主人身故並轉生到下一個時代，鳳凰也會跟隨主人在每個世代的轉生身分，直至主人完成宿願才願意離去。因為其尊貴的身份，連D伯爵店裡的多數動物們都對其恭敬有加，也不敢任意得罪鳳凰。
:翠凰的人型姿態，是穿著飄逸衣裝，宛若[[天女|天女]]的優雅女性。在本作中和從幾個世代前便戀慕著D伯爵的女孩結訂契約，後由於主人瞭解D伯爵不懂如何愛人的真實心聲後，才真正離開主人的身邊。
; 三頭紅龍（朱香 潤麗 華南）
:因為手經D伯爵、雷歐以及一名老者的孫兒之手，而孵育出的三頭紅龍。由於龍會因為培育自己的主人，擁有和主人類似的性格，甚至是孵化日期也順應著主人的祈求，因此當初紅龍蛋失蹤時，D伯爵為此擔憂不已。全名「南海龍王·赤安洪聖濟王·傲潤公主」。三顆龍頭的性格也是相差逕庭。朱香性格穩重幽謐（因為D伯爵親自培育60年）並且在平常主持大局，潤麗天真愛哭，華南性格急躁易怒（因為最後一刻經雷歐之手）。
:人型姿態是個身著古裝的三重人格女孩。根據D伯爵所言，一旦三者有了嚴重的爭執，就連整個中國城都會被破壞成碎瓦礫。目前只有D伯爵能夠在三人情緒不穩的時候使其平靜下來。克里斯多福則是能夠分辨朱香、潤麗、 華南的不同。
; [[九尾妖狐|九尾妖狐]]（阿天）
:能夠因應主人要求，在不同人眼中呈現多樣姿態，年齡超過一千歲的妖狐。
:人型姿態下是個外型叛逆的隨性少年，擅長東方的法術。在本作中一度被因為想要飼養不同寵物而起歧見的家庭收養，在事件落幕後便回到D伯爵的身邊。
; [[浣熊|浣熊]]（小胖）
:親近D伯爵的浣熊，人型姿態是個留著金鬈髮的小女孩。
;亞歷克
:外貌英俊且青春永駐的男性[[吸血鬼|吸血鬼]]，過去曾親眼見證希特勒迫害猶太民族的歷史。於第一部劇情60年前不顧D伯爵的父親的反對，和身為人類的愛人伊莉莎白結婚，但後來伊莉莎白因故逝世，令心碎不已的亞歷克決定尋找D伯爵的父親，解除自己的不死力量。初次在街上遇見雷歐時，因為遭對方砸毀了要送給D伯爵的蛋糕，憤而和對方大打出手，後來轉而與雷歐言和，吐漏自己喪偶的心聲，循線尋獲了D伯爵，最後藉著飲用D伯爵的血液解除不死之身，化為灰燼消失殆盡。
:D伯爵的店鋪裡，迄今仍保有當年亞歷克、諾瑪、D伯爵的父親的合影照片。在《新恐怖寵物店》的第一冊番外篇，則補充說明當年這張合影照片的由來。
;諾瑪
:外表冷艷的女性[[吸血鬼|吸血鬼]]，單戀著亞歷克，對於亞歷克選擇和伊莉莎白共譜戀曲一事始終無法釋懷。為了尋找亞歷克的下落，阻止他藉著D伯爵的協助解除不死之身，支身來到美國，犯下多起吸血殺人案件，並冒用已故FBI探員諾瑪·蘭格麗的身份，對外誆稱協助警方辦案，藉此掩飾自己的殺人罪行。後來諾瑪雖如願尋獲亞歷克，卻親眼見到後者解除不死之身化作灰燼的場面，傷心之餘化為吸血蝙蝠悄然離去，暫時寄宿在D伯爵的店鋪裡。
;布隆迪（[[麒麟|麒麟]]）
:《新恐怖寵物店》登場的雄性麒麟，人形狀態是個金髮藍眼的男孩，在本作中和[[希特勒|希特勒]]的情人[[伊娃·布朗|伊娃·布朗]]締結契約，滿足其「擁有孩子」的願望，並且在艾娃遭遇危機之際，現出原型將她載往希特勒的身邊。艾娃和希特勒持槍自盡後，便離開艾娃的身邊。
; 飛鐸
: 第二部劇情登場，作者秋乃茉莉另一部漫畫「[[靈感商法公司|靈感商法公司]]」的主角，表面上是另一主角常磐飼養的貓，實際上是千年前遇刺身亡的年輕法老靈魂寄宿於寵物貓飛鐸之身，為貓的祖先之一。說話第一人稱為「朕」，人型姿態是個蓄著微波浪黑髮中短髮，肌膚黝黑的男孩。

===僅登場一回的人類、非人生物===

;小P
:雄性極樂鳥，人形姿態是個穿著峇里島風格舞蹈華裝的少年，有著華麗的羽毛和美妙動人的歌聲，以新鮮的水果作為主食。是棲息在南方島嶼，連動物圖鑑也未曾記載的稀有鳥類。由於是體積較大的夜行性生物，生性敏感，需要有廣大的空間供其休息，而且只有在夜晚才會放聲高歌，也僅有和牠投緣的對象，才能耳聞牠的歌聲。相傳久遠前，某個島的皇室喜歡觀賞天堂鳥和毒蛇間的決鬥，只要天堂鳥的歌聲動人，便能成功地催眠毒蛇使其入睡，倘若天堂鳥的旋律出現一絲紊亂，將會被毒蛇吃掉喉頭和聲帶，而這些極樂鳥便經過代代演變，進而造就其後代美妙的歌喉，也僅有雄性極樂鳥才有著華麗的鳥羽和絕妙的歌聲。

:在本作中經由D伯爵的引薦，交給缺乏家庭溫暖的富家千金 安琪拉，後因為安琪拉希望見到極樂鳥繁衍後代，委託D伯爵購入雌性極樂鳥，卻因為雌性極樂鳥有著交配後食用雄性極樂鳥，補充繁殖養分的特性，最後被安琪拉購入的雌性極樂鳥殺死並吞食血肉，但仍在死後持續為安琪拉歌唱，令得悉真相的D伯爵和安琪拉為之動容。

;小P的伴侶
:雌性極樂鳥，人形姿態是個綁著辮子，身穿樸素的纏胸衣裙的少女。因為雌性極樂鳥的數目相當稀少，加上安琪拉利用一年份的金莎巧克力賄絡D伯爵，才使得D伯爵鋌而走險購得雌性極樂鳥，交送給安琪拉。後來被安琪拉送進自家溫室，成為小P的配偶，食用後者的血肉補充養分，不巧被走進探視的安琪拉撞見小P的屍體遭其掌摑，最後因為前來現場關切的D伯爵及時阻止安琪拉，說明極樂鳥的特性，才解決這場糾紛。

;安琪拉

:自認是出生德國家庭的年長婦人，但其實是經過二戰時期德國納粹屠殺猶太人期間，僥倖存活的猶太人後裔。因為帕比吃掉她童年的記憶，故自認為德國人，以德國人的身分自居度過大半輩子，臨終前因為父親與D伯爵祖父簽訂的契約解除而恢復自己的記憶。

==用語說明==

;契約

:光顧D伯爵寵物店的顧客，與D伯爵訂定的紙面契約。通常契約裡會提及顧客飼養寵物的特性，以及飼養該寵物時的注意事項。寵物交給顧客照顧後，顧客必須確實地遵守契約書上的每一條規定。一旦顧客因為違背契約書上的條例而出事，責任與後果必須由顧客自己承擔，D伯爵也不會為此負責。

:通常來到D伯爵寵物店光顧並買走寵物的顧客，D伯爵還會在與他們簽訂契約的同時，贈與顧客一盒泛著甜味的薰香。收費並沒有一定的標準，有時候D伯爵會要求顧客贈與相當份量的甜食，作為購買寵物的費用。

;寵物

:在劇情中出現的生物。通常在劇中會顯現其生物原型與人型姿態，人型狀態下的生物能和人溝通，但只有一些能看見人型姿態的對象（例如D伯爵、一些當事者、其他動物與克利斯多福等人）才能理解生物要表達的意思。有時候生物的人型姿態，其實是反映當事者內心渴望見到的人的模樣。在漫畫故事尾聲，雷歐也能短暫的看到人型姿態的動物。

;薰香

:D伯爵持有的特殊薰香，可讓聞到香味的對象看見動、植物的人形姿態外，也能讓嗅到薰香的對象踏入夢境，效用長達兩、三個小時。
