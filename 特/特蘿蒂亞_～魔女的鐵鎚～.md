'''特蘿蒂亞 ～魔女的鐵鎚～'''（原名：ドロテア 〜魔女の鉄鎚〜、作者：[[Cuvie|Cuvie]]）是2005年到2008年於[[日本|日本]][[富士見書房|富士見書房]]旗下[[月刊Dragon_Age|月刊Dragon Age]]雜誌上連載的[[漫畫作品|漫畫作品]]。發行的單行本全6卷。

== 作品摘要 ==
:Cuvie初次長期連載的一般向作品。

== 故事簡介 ==
:15世紀末期的德國，歐洲正值於魔女狩獵宗教政治迫害當中。一名[[白化症|白化症]]少女·'''特蘿蒂亞'''冒著被告發出是魔女的風險，為了拯救故鄉而戰鬥。


== 登場人物 ==
;{{nihongo|特蘿蒂亞·艾森巴赫|ドロテア・エッシェンバッハ}}
