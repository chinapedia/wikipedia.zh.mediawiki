在[[数学|数学]]分支[[拓扑学|拓扑学]]中，'''特殊化'''（或'''规范'''）'''预序'''是在[[拓扑空间|拓扑空间]]上的自然[[预序|预序]]。对在实践中考虑的大多数空间，特别是满足[[T0空间|T<sub>0</sub>]] [[分离公理|分离公理]]的那些空间，这个预序甚至是[[偏序|偏序]]（叫做'''特殊化序'''）。在另一方面，对于[[T1空间|T<sub>1</sub>空间]]这个次序成为平凡的而没有价值。

特殊化序经常在[[计算机科学|计算机科学]]应用中考虑，这里的T<sub>0</sub>空间出现在[[指称语义|指称语义]]中。特殊化序对于识别在偏序集合上合适的拓扑空间是重要的，这在[[序理论|序理论]]所要做的。

== 定义和动机 ==

考虑任何拓扑空间''X''。在''X''上的'''特殊化预序'''≤定义自设置 

:''x'' ≤ ''y'' [[当且仅当|当且仅当]]cl{''x''}是cl{''y''}的子集，

这里的cl{''x''}指示[[单元素集合|单元素集合]]{''x''}的[[闭包|闭包]]，就是说，包含{''x''}的所有[[闭集|闭集]]的[[交集|交集]]。尽管这个简短定义是方便的，注意下列陈述是等价的是有帮助的：

:''x'' ≤ ''y''当且仅当''y''包含在所有包含''x''的[[开集|开集]]中。

这解释了为什么说是“特殊化”: ''y''比''x''更特殊，因为它包含在更多开集中。这是显著直觉性的，如果你把开集看作一个点''x''可以有也可以没有的性质。更多开集包含一个点，它就有更多性质，因而它更加特殊。这种用法相容于经典逻辑概念[[属|属]](genus)和[[种|种]](species)；并相容于[[代数几何|代数几何]]的[[一般点|一般点]]的传统用法。特殊化作为想法还应用于[[求值理论|求值理论]]中。

上部元素更特殊的直觉可典型在在[[域理论|域理论]]中找到，它是在计算机科学中充分应用的序理论分支。

== 上部和下部集合 ==

设''X''是拓扑空间并设≤是在''X''上的特殊化预序。关于≤所有[[开集|开集]]都是[[上部集合|上部集合]]而所有[[闭集|闭集]]都是[[下部集合|下部集合]]。反过来一般不是真的。事实上，拓扑空间是[[Alexandrov空间|Alexandrov空间]]，当且仅当所有上部集合都是开集(或所有闭集都是下部集合)。

设''A''是''X''的子集。包含''A''的最小上部集合指示为↑''A''
而包含''A''的最小下部集合指示为↓''A''。在''A'' = {''x''}是单元素集合的情况下，我们使用符号↑''x''和↓''x''。对于''x'' ∈ ''X''我们有：

*↑''x'' = {''y'' ∈ ''X'' : ''x'' ≤ ''y''} = ∩{包含''x''的开集}。
*↓''x'' = {''y'' ∈ ''X'' : ''y'' ≤ ''x''} = ∩{包含''x''的闭集} = cl{''x''}。

下部集合↓''x''总是闭集；但是上部集合↑''x''不必须是开集或闭集。拓扑空间''X''的闭合点完全就是''X''关于≤的[[极小元|极小元]]。

== 例子 ==

* [[Sierpinski空间|Sierpinski空间]]{0,1}带有开集{∅, {1}, {0,1}}，特殊化次序是自然次序 (0 ≤ 0, 0 ≤ 1和1 ≤ 1)。
* 如果''p'', ''q''是Spec(''R'') ([[交换环|交换环]]''R''的[[交换环谱|谱]])的元素则''p'' ≤ ''q''，当且仅当''q'' ⊆ ''p'' (如[[素理想|素理想]])。所以Spec(''R'')的闭合点就是[[极大理想|极大理想]]。

== 重要性质 ==

如名字所暗示的，特殊化预序是预序，就是说它是[[自反关系|自反]]的[[传递关系|传递]]的，这实际上是容易看出来的。

由特殊化预序所确定的[[等价关系|等价关系]]就是[[拓扑不可区分性|拓扑不可区分性]]。就是说，''x''和''y''是拓扑可区分的，当且仅当  ''x'' ≤ ''y''并且  ''y'' ≤ ''x''。因此，≤的[[反对称关系|反对称]]完全就是T<sub>0</sub>分离公理：如果''x''和''y''是不可区分的，则''x'' = ''y''。在这种情况下它证实了'''特殊化序'''的说法。

在另一方面，特殊化预序的[[对称关系|对称]]等价于[[R0空间|R<sub>0</sub>]]分离公理：''x'' ≤ ''y''当且仅当''x''和''y''是拓扑不可区分性的。可得出如果底层拓扑是T<sub>1</sub>，则特殊化序是离散的，就是说''x'' ≤ ''y''当且仅当''x'' = ''y''。因此特殊化序对于T<sub>1</sub>拓扑，特别是所有[[豪斯多夫空间|豪斯多夫空间]]是没有多少价值的。

任何在两个拓扑空间之间的[[连续函数_(拓扑学)|连续函数]]都是关于这些空间的特殊化预序的[[单调函数|单调函数]]。但是反过来一般不是真的。用[[范畴论|范畴论]]的语言来说，我们在从[[拓扑空间范畴|拓扑空间范畴]]到[[预序范畴|预序集合范畴]]之间的[[函子|函子]]，它把一个拓扑空间指派到它的特殊化预序。这个函子有把[[Alexandrov拓扑|Alexandrov拓扑]]放置在预序集合上的[[伴随函子|左伴随]]。

有比T<sub>0</sub>空间更特殊的空间对于它这种次序是有价值的：[[sober空间|sober空间]]。它们与特殊化序的联系更加微妙：

对于任何sober空间''X''带有特殊化序≤，我们有
* (''X'', ≤)是[[有向完全偏序|有向完全偏序]]，就是说 (''X'', ≤)的所有[[有向集合|有向子集]]''S''都有[[上确界|上确界]] sup ''S''，
* 对于所有 (''X'', ≤)的有向子集''S''和所有的开集''O''，如果sup ''S''在''O''中，则''S''和''O''有[[非空|非空]][[交集|交集]]。

你可以把第二个性质描述为开集是“通过有向上确界不可到达的”。拓扑是关于特定次序≤是[[序相容|序相容]]的，如果它引发≤作为它的特殊化序，并且它有关于在≤中有向集合的(现存)上确界的不可到达性质。

== 在次序上的拓扑 ==

特殊化序产生了从所有拓扑获得偏序的工具。自然要问反过来也行吗：所有偏序都是作为某个拓扑的特殊化序而获得的吗?

实际上，这个问题的答案是肯定的，一般的在集合''X''上有很多拓扑，它们引发给定次序≤作为它们的特殊化序。次序≤的[[Alexandroff拓扑|Alexandroff拓扑]]扮演了特殊角色：它是引发≤的最精细的拓扑。另一个极端，引发≤的最粗糙的拓扑是[[上部拓扑|上部拓扑]]，在其中集合{''y'' in ''X'' | ''y'' ≤ ''x''}(对于某个''X''中的''x'' in )的所有补集都是开集的最小的拓扑。

在这两个极端之间还有有趣的拓扑。对于给定次序≤在上述序相容意义上最精细的拓扑是[[斯科特拓扑|斯科特拓扑]]。但是上部拓扑仍市最粗糙的序相容拓扑。事实上它的开集甚至用任何上确界都不能到达。因此任何sober空间带有特殊化序≤都精细于上部拓扑并粗糙于斯科特拓扑。然而，这种空间可能不存在。特别是斯科特拓扑不必然是sober拓扑。
