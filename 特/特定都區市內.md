'''特定都區市內'''（{{jpn|j=特定都区市内|hg=とくていとくしない|rm=Tokutei-toku-shinai|f=y}}）是[[JR|JR]]車費計算其中一個特例。

== 概要 ==
本特例根據JR的{{link-ja|旅客營業規則|旅客営業規則}}（旅規）第86條與第87條為基礎制定。目的為簡化大都市火車站的[[出閘|出閘]]業務。

本特例是因應[[日本戰後經濟奇蹟|高度經濟成長期]]的商務客、觀光客長距離移動需求所誕生。

現在於[[東京都區部|東京都區]]、[[橫濱市|橫濱]]、[[名古屋市|名古屋]]、[[京都市|京都]]、[[大阪市|大阪]]、[[神戶市|神戶]]、[[廣島市|廣島]]、[[北九州市|北九州]]、[[福岡市|福岡]]、[[仙台市|仙台]]、[[札幌市|札幌]]合計11個都市、都區內（所謂「'''特定都區市內'''」）設有各自適用的對象。

== 內容 ==
=== 旅客營業規則（旅規） ===
==== 規定 ====
* 特定都區市內車站，前往與該特定都區市內中心車站之單程營業里程超過200公里的車站時，其單程普通旅客票價以中心車站為起點計算，反之亦然。《'''旅規86條'''》
* 「東京都區內」的[[山手線|山手線]]內各站與山手線內之[[中央本線|中央本線]]、[[總武本線|總武本線]]各站（通稱「'''[[東京山手線內|東京山手線內]]'''」），來往距離東京山手線內中心車站[[東京站|東京]]之單程營業里程大於100公里小於200公里（未滿1公里無條件進位）的車站時，其單程普通旅客票價與上述情況採用相同計算方式《'''旅規87條'''》
* 上述旅規86條、87條規定，從特定都區市內所在車站出發，先離開該特定都區市內，之後再次進入該特定都區市內後抵達目標車站，或是出發後需兩度進入目標車站所屬之特定都區市內時，不適用本特例。

== 沿革 ==
* 1939年10月15日 - 制定[[日本三大都市圈|六大都市]]制度，距離東京、橫濱、名古屋、京都、大阪、神戶超過300公里的車站前往各都市內各站將以該都市中心車站作為車費計算基準
* 1942年4月1日 - 六大都市制度變更為151公里以上
* 1944年4月1日 - 廢止六大都市制度。改為東京與大阪的二大都市制度。距離變更為101公里以上。新增東京電車環狀線內制度，距離設定為51公里
* 1957年4月1日 - 二大都市制度距離變更為151km以上
* 1961年4月6日 - 二大都市制度距離變更為201km以上
* 1969年5月10日 - 變更為特定都區市內制度，追加橫濱、名古屋、京都、神戶各市內
* 1972年9月1日 - 特定都區市內制度追加札幌、仙台、廣島、北九州、福岡各市。東京電車環狀線內變更為「東京山手線內」
