'''特立尼达和多巴哥华人'''（[[英语|英语]]：''Chinese Trinidadian and Tobagonian''，有时亦作''Sino-Chinese Trinidadian and Tobagonian''）是[[中国人|华裔]]的[[特立尼达岛|特立尼达]]和[[多巴哥岛|多巴哥]]人。这一群体的人包括移民到[[特立尼达和多巴哥|特立尼达和多巴哥]]的来自[[中国|中国]]的人和[[海外华人|海外华人]]们及其后代，而这些后代也包括那些已经从特立尼达和多巴哥移民到了其他的国家的人（尤其是[[美国|美国]]、[[加拿大|加拿大]]及[[英国|英国]]，但也有到别的国家的，包括移民回中国的）。华裔混血者和纯华裔者通常都被称为“特立尼达和多巴哥华人”，然而在人口普查中，前者通常被归类于[[混血儿|混血人种]]。<ref name = Johnson>{{cite book|last=金·约翰逊|title=''龙的传人：中国人在特立尼达岛 1806─2006''|year=2006|publisher=伊恩·兰德尔出版公司|location=迈阿密金斯顿|isbn=976-637-289-6}}</ref>

早在1806年，华人便开始定居于特立尼达和多巴哥了。在1853年到1866年间，有2645名华人移民作为[[契约劳工|契约劳工]]抵达了特立尼达和多巴哥，在当地的[[甘蔗|甘蔗]]种植园与[[可可树|可可树]]种植园工作。华人移民在二十世纪前半叶达到顶峰，但是在1949年的[[第二次国共内战|中国革命]]后锐减。在1960年，特立尼达和多巴哥的（未混血的）华人人口达到了8361人的顶峰，而在2000年，则已降至了3800人。

==华人群体==
特立尼达和多巴哥华人群体是一个形形色色的混合体，其中有来自中国的第一代移民，有祖先们在特立尼达和多巴哥生活了许多代的当地人，还有原本定居在美国、加拿大及英国，之后[[流散|移居]]到了特立尼达和多巴哥的人们。如上所述，特立尼达和多巴哥华人群体包括华裔混血者和纯华裔者，而在该国的人口普查中，前者通常被归类于混血人种。大多数特立尼达和多巴哥华人来自[[广东省|广东省]]，尤其是其中的[[客家人|客家人]]。<ref name = LookLai>{{cite book|last=陆华祖|title=''西印度群岛华人：史料集，1806-1995''|year=1998|publisher={{link-en|西印度大学出版社|University of the West Indies Press}}|isbn=976-640-021-0}}</ref>

==历史==
特立尼达和多巴哥华人群体追溯其来源至1806年10月12日运载着一群在[[澳门|澳门]]、[[槟城|槟城]]及[[加尔各答|加尔各答]]招募的华人的船只“坚韧号”的抵达。<ref name = LookLai/>在1805年12月，一名[[葡萄牙人|葡萄牙人]]船长在澳门招募了141名华人，之后用船只将他们载到了槟城并在当地又招募了6人。之后他们又在加尔各答招募了53人，凑到了整整200人。八个月后，这一群人的幸存者们抵达了特立尼达。<ref name = Johnson/>金·约翰逊报告说有194人在这次航行中幸存了下来，<ref name = Johnson/>而陆华祖则报告说是192人。<ref name = LookLai/>这一群人定居在首府[[西班牙港|西班牙港]]西边的科科里特的监视庄园（Surveillance Estate）。考虑到城市附近缺乏农田，这一群人请求被允许雇出他们自己为劳工。15人被雇去当{{link-en|围网捕鱼|seine fishing}}者，1人成为了鞋匠。在特立尼达待了一年后，移民中有17人死去了。在1807年7月，他们中的61人又乘坐“坚韧号”离开了。到了1810年，他们中只有22人还留在特立尼达，而到了1834年，则只有7人了，这也是这一群体最后一次被提到。<ref name = Johnson/>

1806年“坚韧号”的抵达是华人在[[加勒比地区|加勒比地区]]的首次有组织的定居，它早于40多年后的华人契约劳工的输入。<ref name=TTReview>{{cite journal|last=陆华祖 |year=1993 |title=''来自广东的人们'' |journal=特立尼达和多巴哥评论 |volume=15 |issue=8-9 |publisher=由牙买加客家人出版 |url=http://www.chinesejamaican.com/history.html#people |deadurl=yes |archiveurl=https://web.archive.org/web/20130512200323/http://www.chinesejamaican.com/history.html |archivedate=2013-05-12 }} {{en icon}}</ref>这是在[[特立尼达和多巴哥历史|这个之后成为了一块新的英殖民地的地方]]建立一个自由劳工和农民的定居点的计划中的第一步。<ref name = Johnson/>英国皇家海军上校[[威廉·莱曼|威廉·莱曼]]建议道使用华人自由劳工来建立新的甘蔗种植园比用非洲奴隶会更便宜。与此同时，在[[海地革命|海地革命]]的余波影响下，英国官员们也建议道特立尼达和多巴哥的华人移民定居点将会在受奴役的非洲人与白人之间，提供一个缓冲作用。<ref name = LookLai/>

{{link-en|1833年废奴法案|Slavery Abolition Act 1833|英帝国的废奴之举}}导致了特立尼达的劳动力短缺。契约劳工从世界各地被输入，其中包括[[印度|印度]]和[[马德拉|马德拉]]。在1853年到1866年间，2645名华人移民乘坐八艘船抵达了特立尼达──2336男，309女，其中4儿童。这些移民构成了特立尼达的第二波华人移民。<ref name = Johnson/>第三波始于1911年的[[辛亥革命|辛亥革命]]之后并持续到了1949年的[[第二次国共内战|中国革命]]为止。这些移民大都是在之前的移民的努力下才被带到特立尼达和多巴哥的。第四波移民始于20世纪70年代晚期并继续着。<ref name=NALIS>{{cite web|url=http://library2.nalis.gov.tt/Default.aspx?tabid=249|title=''特立尼达和多巴哥华人''|accessdate=2008-11-18|publisher={{link-en|特立尼达和多巴哥国家图书馆与信息系统管理机构|National Library and Information System}}|deadurl=yes|archiveurl=https://web.archive.org/web/20081005161946/http://library2.nalis.gov.tt/Default.aspx?tabid=249|archivedate=2008-10-05}} {{en icon}}</ref>

还有一些移民最初移民到的是加勒比地区的其他地方，之后他们又定居在了特立尼达。英属[[圭亚那|圭亚那]]就是一个典型例子。在1853年到1884年间，它曾接收了13593名来自中国的契约移民。<ref name = Johnson/>
