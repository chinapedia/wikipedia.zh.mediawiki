'''碲化鈉'''是一種[[無機化合物|無機化合物]]，由[[碲|碲]]和[[鈉|鈉]]組成，屬於[[碲化氫|氫碲酸]][[鹽|鹽]]，其化學式為Na<sub>2</sub>Te。

== 制備 ==
[[碲|碲]]和[[鈉|鈉]]的化合反應可以制备碲化鈉，由于固相反应过于剧烈，因此通常需要利用[[液氨|液氨]]作為溶劑進行反應。

== 結構 ==

碲化鈉結構像許多相關化合物其化學式為M<sub>2</sub>X，例如:[[氧化鈉|氧化鈉]]、[[硫化鈉|硫化鈉]]、[[硫化鉀|硫化鉀]]等，都是反[[螢石|螢石]]結構，因此，在碲化鈉結晶每個Te<sup>2-</sup>離子周圍有八個鈉離子，每個鈉離子周圍有四個Te<sup>2-</sup>離子<ref>Wells, A.F. (1984) Structural Inorganic Chemistry, Oxford: Clarendon Press. ISBN 0-19-855370-6.</ref>

== 性質 ==
簡單[[鹽類|鹽類]]類型M<sub>2</sub>X，其中X是一個[[單原子|單原子]][[陰離子|陰離子]]，通常不溶於任何溶劑，因為它們具有很高的晶格能。此外，當其遇到水、潮濕的空氣、或醇類，碲化鈉將分解：

:Na<sub>2</sub>Te + H<sub>2</sub>O → NaHTe + NaOH

正因為如此反應，許多過程歸結為碲化鈉可能涉及[[碲氫化鈉|碲氫化鈉]]NaHTe（[[CAS号|CAS #]] 65312-92-7），這是比較容易溶解和形成的物质。

== 在有機化學的應用 ==
Na2Te發現使用有機合成，它既是一種試劑減排和作為源特在合成化合物的有机碲<ref name=ditt>”Sodium Telluride” Dittmer, D. C. in Encyclopedia of Reagents for Organic Synthesis © 2001.  DOI: 10.1002/047084289X.rs103.</ref>芳基鹵化物對二芳基取代碲化物，如下所示的合成双萘基碲：

:Na<sub>2</sub>Te  +  2 C<sub>10</sub>H<sub>7</sub>I  →  (C<sub>10</sub>H<sub>7</sub>)<sub>2</sub>Te  +  2  NaI

:Na<sub>2</sub>Te  +  RC≡C-C≡CR  +  2 H<sub>2</sub>O  →  TeC<sub>4</sub>R<sub>2</sub>H<sub>2</sub>  +  2  NaOH
