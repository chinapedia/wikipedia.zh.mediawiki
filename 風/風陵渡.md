'''风陵渡'''是古代[[黄河|黄河]]上最大的[[渡口|渡口]]，位于于黄河在[[山西省|山西]]东转处，是山西、[[陕西|陕西]]、[[河南|河南]]三省的交界点（今[[山西省|山西省]][[芮城县|芮城县]][[风陵渡镇|风陵渡镇]]）。距離東北方的芮城县約30公里。其名字是因附近的[[风后陵|风后陵]]而取的。相传[[黄帝|黄帝]]六相之一的[[风后|风-{后}-]]，与[[蚩尤|蚩尤]]作战被杀，埋葬于此，后建有风后陵。金人趙子貞《題風陵渡》有一句：「一水分南北，中原氣自全。雲山連晉壤，煙樹入秦川。」

著名[[武俠小說|武俠小說]]《[[神雕俠侶|神雕俠侶]]》第三十三回「風陵夜話」，就講述女角色[[郭襄|郭襄]]的出場，位置就是風陵渡。

== 名稱由來 ==
=== 傳說一 ===

以[[女媧|女媧]]的陵墓命名，由於女媧為[[風姓|風姓]]，故稱風陵。在風陵渡附近趙村東南方，有一座女媧墓，塚高2米，周邊30米。墓前原有明[[萬曆|萬曆]]三十八年(西元1610年)重建風后祠及碑記，惜已毀。

=== 傳說二 ===

名字是因附近的風后陵而取的。相傳[[軒轅黃帝|軒轅黃帝]]和[[蚩尤|蚩尤]]大戰于涿鹿，[[蚩尤|蚩尤]]作大霧，[[黃帝|黃帝]]部落的將士頓時東西不辨，迷失四方，不能作戰。這時候，[[黃帝|黃帝]]的臣子[[風后|風后]](也稱[[風伯|風伯]])，獻上他製作的[[指南車|指南車]]，給大軍指明方向，擺脫困境，終於戰勝蚩尤。可惜風伯在這場戰爭中被殺，埋葬在這裡，後來建有風后陵。風后關，位於趙村東南，高二米余，周圍30米。因武周[[聖曆元年|聖曆元年]]（公元689年）在此置關，又稱風陵津，是黃河南泄轉而東流之地。津即渡口，所以後來稱風陵渡。

== 地理成因 ==

[[黃河|黃河]]流過[[双河镇_(托克托县)|河口鎮]]後，為南北走向的[[呂梁山|呂梁山]]所阻，折向南流，奔騰於[[陝西|陝西]]、[[山西|山西]]兩省邊界的峽谷之間。河水從[[壺口瀑布|壺口瀑布]]瀉下後，出[[龍門|龍門]]繼續向南流到[[潼關|潼關]]附近，因受東西走向的[[秦嶺|秦嶺]]山脈其中一條支脈[[華山|華山]]所阻，因而折向東流。

== 歷史 ==

風陵渡自古以來就是[[河東|河東]]、[[河南|河南]]、[[關中|關中]]咽喉要道，為兵家必爭之地。在[[戰國時代|戰國時代]]，魏國與秦國的歷次大戰，古戰場都在這裡附近，[[東漢|東漢]]時的曹操討伐[[韓遂|韓遂]]、[[馬超|馬超]]，[[西魏|西魏]]的[[宇文泰|宇文泰]]破[[高歡|高歡]]等著名戰爭，均發生在風陵渡。

明清時在此設巡檢司和船政司，管理防守和運輸事宜。這處渡口，歷史上一直以擺船渡河來連接陝豫二省。1994年，一座長1500米的鐵路大橋，將南同蒲鐵路和隴海鐵路連接在一起，並連接黃河南北兩岸。

== 交通 ==

風陵渡鎮地處晉、秦、豫三省交界的黃河大拐彎處，地理位置獨特，交通便捷，三河交匯（[[黃河|黃河]]、[[渭河|渭河]]、[[洛河|洛河]]）、三省交界（[[山西|山西]]、[[陝西|陝西]]、[[河南|河南]]）、三路共通（鐵通、公路、水路）、三橋飛架（[[鐵路橋|鐵路橋]]、公路橋、[[高架橋|高架橋]]），是[[山西省|山西省]]的南大門，是通往中國[[華北|華北]]、西北、西南和[[中原|中原]]大地的咽喉。

== 行政建制 ==

風陵渡鎮共有59個行政村，69998人，110924畝耕地，12600畝河灘地，是一個傳統的農業大鎮。1998年被國家確定為農村小城鎮建設試點鎮。
