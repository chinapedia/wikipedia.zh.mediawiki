《'''風雲維新大將軍'''》（{{lang-ja|風雲維新ダイショーグン}}）是[[J.C.STAFF|J.C.STAFF]]和[[A.C.G.T|A.C.G.T]]共同製作的[[日本|日本]][[電視動畫|電視動畫]]作品。2014年4月至6月在[[TOKYO_MX|TOKYO MX]]獨立電視網播放。

== 概要 ==
企画、原案是『[[越戰狂想曲|越戰狂想曲]]』的[[笹原和也|笹原和也]]，導演是『[[結界女王|結界女王]]』、『[[一騎當千|一騎當千]]』、『[[秀逗魔導士_(動畫)|秀逗魔導士]]』的[[渡部高志|渡部高志]]，系列構成是『[[交響詩篇|交響詩篇]]』的[[佐藤大|佐藤大]]，角色原案是{{lang|ja|[[山下しゅんや|山下しゅんや]]}}，角色設計是[[飯島弘也|飯島弘也]]，機械設定是[[Nitro+|Nitro+]]的{{lang|ja|石渡マコト}}，動畫製作則是由[[J.C.STAFF|J.C.STAFF]]及[[A・C・G・T|A・C・G・T]]負責<ref name="animeanime_20131226"/>。

本作為描述在自古以來留傳下來的巨大[[機器人|機器人]]的[[平行世界|平行世界]]中之[[江戶時代|江戶時代]]後期，雖有[[黑船来航|黑船来航]]卻因為巨大機器人而失敗，並因此引發[[明治維新|明治維新]]的日本，由[[處男|處男]]男主角駕駛巨大機器人並與女主角們一邊[[戀愛|戀愛]]一邊展開冒險的冒險作品<ref name="animeanime_20131226"/>。

[[廣告標語|廣告標語]]是「'''正因為是處男，就要成為天下第一的男子漢！'''」<ref name="animeanime_20131226">{{Cite web|url=http://animeanime.jp/article/2013/12/26/16866.html|title=「風雲維新ダイ☆ショーグン」2014年4月放送予定 黑船襲来！巨大ロボが立ち上がり…|work=アニメ!アニメ!|publisher=イード|accessdate=2013-12-27}}</ref>。

[[網路|網路]]上認為本作前身是來自於由笹原擔任[[取締役|取締役]]的Anima公司<ref name="anima">{{Cite web|url=http://www.studioanima.co.jp/|title=anima inc|work=アニマ|publisher=アニマ|accessdate=2013-12-27}}</ref>所推出的[[跨媒體製作|跨媒體製作]]企画『艶華王道大☆將軍』<ref name="yamashita_pf"/><ref name="anima_cache_20090320"/><ref name="yamato_cache_20090806"/><ref name="akiba-souken_20131226"/>。詳情請見[[#艶華王道大☆將軍|#艶華王道大☆將軍]]。

== 故事大綱 ==
德川慶一郎是個自由自在的[[傾奇者|傾奇者]]，某日碰上自己的小弟被暗殺的事件，在追查真相的途中，遇到神秘的敵人法光院駕駛蒸氣傀儡攻擊，千鈞一髮之際，被伊賀女忍服部霧子所救，並告訴他正是將軍家的繼承人，要利用他身為處男的身體才能啟動將軍家代代相傳的鬼神‧須佐之男，在慶一郎正式駕駛須佐之男並擊退法光院後，才知道自己已經捲入一場牽動整個國家的陰謀之中。

== 登場機械 ==
=== 鬼神 === 
: 鬼神（{{lang|ja|オニガミ}}）<ref name="moca_20140305"/>。日本自古以來流傳下來的巨大機械人。

; [[須佐之男|須佐之男]]（{{lang|ja|スサノオ}}）
: 慶一郎駕駛的鬼神。會因為同機的搭檔不同而有不同的變化。
:; 須佐之男 基本型態（{{lang|ja|サナギスサノオ}}）
:: 慶一郎單獨搭乘時的型態，防護力高但缺乏武裝。
:; 須佐之男 豪天雷神
:: 慶一郎和霧子搭乘時的型態，可應付任何狀況的萬能型。
:; 須佐之男 暴龍獸神
:: 慶一郎和千春搭乘時的型態，擁有強大力量能壓倒任何眼前的敵人。
:; 須佐之男 閃光風神
:: 慶一郎和淺井兵庫搭乘時的型態，巨大雙臂變化為劍的二刀流，擁有優秀的敏捷性。

; 武御雷（タケミカヅチ）
: 重義所搭乘的鬼神

: 蒸氣傀儡中數量最多的量產機。有各種顏色的版本。

; {{lang|ja|デク}}

== 艶華王道大☆將軍 ==
2009年3月18日舉辦的「[[東京國際動畫博覽會|東京國際動畫博覽會]]2009」在Anima攤位中除了對各家媒體進行發表同時，Anima官方網站中也設立了介紹網頁，而且由[[笹原和也|笹原和也]]擔任原案及滝川廉治<ref name="takisyun">{{Cite web|url=http://takisyun.blog59.fc2.com/|title=滝川春秋|work=滝川春秋|publisher=滝川廉治|accessdate=2013-12-27}}</ref>執筆的[[小說|小說]]也開始進行連載<ref name="anima_cache_20090320"/>。

這是一部綜合「[[插畫|插畫]]+[[人形|人形]]+小説+[[漫画|漫画]]+[[廣播劇|廣播劇]]+[[動畫|動畫]]+[[3D電腦圖形|3DCG]]的新☆王道[[跨媒體製作|跨媒體製作]]企画作品」，插畫由{{lang|ja|[[山下しゅんや|山下しゅんや]]}}負責，動畫及人型則由玩具商{{lang|ja|[[やまと|やまと]]}}公司負責<ref name="hobbystock_20090318">{{Cite web|url=http://blog.hobbystock.jp/report/2009/03/etc0086.html|title=東京国際アニメフェア2009速報レポ！グッドスマイルカンパニー「ねんどろいどぷち ボカロ＃01」「figma　ゴルゴ13」、「figma シグナム」|work=シマゾウ・レポート|publisher=ホビーストック|accessdate=2013-12-27}}</ref>。特別是女主角服部霧子是一名[[巨乳|巨乳]]有才的[[女忍者|女忍者]]的設定也受到注目，{{lang|ja|やまと}}在製作人形的同時也一並以山下的霧子插畫製作出有矽膠扶手的[[滑鼠墊|滑鼠墊]]<ref name="yamato_cache_20090806"/>。之後在2010年3月25日舉辦的「東京國際動畫博覽會2010」中，於Anima攤位中更秀出了霧子全身走路的3DCG<ref name="Youtube_Veq6oYLBaSc">{{Cite web
|people= 笹原和也|date= 2010-08-24|title= 艶華王道 大☆將軍|url=http://www.youtube.com/watch?v=Veq6oYLBaSc|accessdate= 2013-12-27}}</ref><ref name="mynavi_20100326">{{Cite web|url=http://news.mynavi.jp/articles/2010/03/26/taf/|title=3D映像を筆頭に多彩な新作アニメが出揃う「東京国際アニメフェア2010」開幕|work=マイナビニュース|publisher=マイナビ|accessdate=2013-12-27}}</ref>，同年12月31日，由於笹原離開公司並跳槽到[[ILCA|ILCA]]，讓企畫因此終止，不久後Anima的官網上也刪去了介紹網頁。
