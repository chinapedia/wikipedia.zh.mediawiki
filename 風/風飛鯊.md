《'''風飛鯊'''》（{{lang-en|'''''Sharknado'''''}}）是一部2013年美國[[災難片|災難]][[電視電影|電視電影]]，由{{link-en|安東尼·費蘭提|Anthony C. Ferrante}}執導。故事敘述因一場龐大的[[海龍捲風|海龍捲風]]而讓海裡的鯊魚跑到了[[洛杉磯|洛杉磯]]市區中作亂。本片為「[[風飛鯊系列電影|風飛鯊系列電影]]」的第一部<ref>{{cite web |url = http://www.usatoday.com/story/life/tv/2013/07/11/bianco-critics-corner-july-11/2502663/ |title = TV tonight: 'Sharknado,' 'Summer Camp' |last=Bianco |first=Robert |date= 2013-07-11 |publisher = USA Today |accessdate = 2013-07-12}}</ref>。由[[伊恩·齊爾林|伊恩·齊爾林]]、[[泰拉·蕾德|泰拉·蕾德]]和[[約翰·赫德|約翰·赫德]]主演。電影於2013年7月11日在[[Syfy|Syfy]]頻道播出<ref name="Los">{{cite web |url = http://www.latimes.com/entertainment/tv/showtracker/la-et-0711-sharknado-need-we-say-more-20130711,0,1772285.story| title = 'Sharknado' keeps TV real |last = McNamara |first = Mary |date= 2013-07-11 |publisher = Los Angeles Times |accessdate = 2013-07-12}}</ref>。

{{link-en|帝王娛樂集團|Regal Entertainment Group|帝王影院}}和NCM Fathom Events合作給本片在一晚午夜場放映，在200間戲院獲得了200萬美元的票房。

==劇情==
故事敘述一場怪物級的颶風襲捲洛杉磯，導致市區大淹水，而海中的鯊魚則被吹上陸地到處吃人；就此失去酒吧的芬恩則為了避難而和朋友們一起尋找芬恩的家人，究竟他們能夠在這場災難中平安倖存嗎？
