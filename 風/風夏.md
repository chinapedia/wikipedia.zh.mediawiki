{{日本作品|風夏|ふうか}}是日本漫畫家[[瀨尾公治|瀨尾公治]]創作的[[戀愛漫畫|戀愛漫畫]]作品。2014年11號開始至2018年18號完結，連載於[[講談社|講談社]]出版雜誌《[[週刊少年Magazine|週刊少年Magazine]]》。該作品是瀨尾公治在《週刊少年Magazine》連載的第四部作品，以前作《[[涼風|涼風]]》的男女主角所生的女兒秋月風夏做女主角，以她及男主角榛名優兩人為中心所描寫的故事。改編電視動畫於2017年1月起至3月底播放，全12話。

== 劇情 ==
故事的舞台位於作者以前居住過的東京都[[板橋區_(日本)|板橋區]][[成增|成增]]<ref name="A1">『週刊少年マガジン』21・22合併号掲載 [[にしもとひでお|にしもとひでお]]・著『[[ちょっと盛りました。|ちょっと盛りました。]]』第90回「瀬尾先生と『涼風』『君町』『風夏』の聖地巡礼!!」より。</ref>。'''榛名優'''是一個孤獨的[[Twitter|Twitter]]狂熱使用者。一日，榛名優在購買晚餐的途中與一名叫'''風夏'''的女孩相撞；風夏誤以為榛名優在拍攝自己因摔倒而[[裙底風光|露出的內褲]]，便砸毀了他的手機洩憤。在他們相遇之後，開始籌組樂團。

== 登場角色 ==
=== 主要角色 ===
; {{nihongo|榛名優|榛名 優（はるな ゆう）|聲：[[小林裕介|小林裕介]]<ref name="staffcast">{{cite web|url=http://fuuka.tv/staffcast/|script-title=ja:スタッフ／キャスト|work=TV動畫「風夏」官方網站|accessdate=2017年3月20日|language=ja}}</ref>}}
: 男主角，性格內向怕生，不擅於與人面對面交流，是Twitter的使用愛好者。轉學後與秋月風夏相遇，常被其牽著鼻子走，同時也漸漸受其所吸引。因為一次提議風夏可試著朝音樂方向發展，促使風夏成立樂團，並遭她半強迫地拉為團員，成為樂團-{「}-The Fallen Moon-{」}-（後更名為Blue Wells）的貝斯手。在風夏死後長期不去上學，聽見風夏的預留音檔後重組樂團。隨故事發展，之後身兼男主唱。
; {{nihongo|秋月風夏|秋月 風夏（あきつき ふうか）|聲：[[Lynn_(日本女配音員)|Lynn]]<ref name="staffcast"/>}}
: 《涼風》的主角秋月大和與秋月涼風的長女，個性開朗活潑、行動力強。沒有使用手機，常隨身戴著耳機聽音樂。原本對自身未來感到迷惘，在和優因誤會而相識後，受他影響而建立The Fallen Moon並擔任女主唱。後來與優交往。當樂團準備競奏演唱時，被卡車撞死，導致樂團一度解散。動畫版改編為逃過死劫並與優在一起。
; {{nihongo|碧井風夏|碧井 風夏（あおい ふうか）|}}
: 來自[[伊豆|伊豆]]，是個初展才華的音樂家，比優大兩歲。在街頭表演時與優相識，之後加入「Blue Wells」擔任吉他手及女主唱。她的父親是撞死秋月風夏的卡車司機，為此風夏一度放棄音樂躲回老家，爾後受優說服而加入樂團。喜歡優，個性很男孩子氣，有著紅髮和姣好的身材。在秋月風夏過世後，優藉由許多事件感受到自己漸漸喜歡上了碧井風夏。
; {{nihongo|冰無小雪|氷無 小雪（ひなし こゆき）|聲：[[早見沙織|早見沙織]]<ref name="staffcast"/>}}
: 當紅歌手。優的青梅竹馬，小時突然搬家而分開。她和優因推特而重逢，爾後兩人的會面引發緋聞。她在向優示愛未果後引退，之後成為獨奏音樂家，另組戴兔子面具的樂團。

=== Blue Wells ===
; {{nihongo|三笠真琴|三笠 真琴（みかさ まこと）|聲：[[齊藤壯馬|齊藤壯馬]]<ref name="staffcast"/>}}
: 優與風夏的同學。擁有四分之一英裔德國血統。外貌清秀帥氣，多得女性青睞，不過本人表示自己對女性不感興趣。曾在國際鋼琴競賽少年組獲得優勝，在風夏成立的樂團中擔任鍵盤手，在漫畫163話時向友美告白。
; {{nihongo|那智一矢|那智 一矢（なち かずや）|聲：[[興津和幸|興津和幸]]<ref name="staffcast"/>}}
: 優與風夏的學長。原本想要請風夏加入田徑社，後來卻反而加入樂團當爵士鼓手。
; {{nihongo|石見沙羅|石見 沙羅（いわみ さら）|聲：[[小松未可子|小松未可子]]<ref name="staffcast"/>}}
: 優與風夏的學姐，優的網友。刺蝟樂團阿久的妹妹，在其訓練下有著高超的音樂技巧。在樂團中擔任吉他手。與優一樣不擅長和人面對面交談，不知如何應對所以總以冰冷態度對人，所以常常使人以為很冷漠。只有面對優時容易慌張及臉紅，對優特別溫柔，忌妒心強，暗戀優。
