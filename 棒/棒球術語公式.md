本篇介紹各[[棒球術語列表|棒球術語列表]]所計算的公式。

==投手==

===ERA [[自責分率|自責分率]]===
<small>英文：Earned Run Average</small><br>
<math>\ ERA = \frac{ER \times 9}{IP}</math>

===H/9 [[被安打率|被安打率]]===
<small>英文：Hits per Nine Innings</small><br>
<math>\ H/9 = \frac{H \times 9}{IP}</math>

===WHIP [[每局被上壘率|每局被上壘率]]===
<small>英文：Walks Plus Hits per Inning Pitched</small><br>
<math>\ WHIP = \frac{H + BB}{IP}</math>

===BABIP [[場內被安打率|場內被安打率]]===
<small>英文：Batting Average on Balls put Into Play</small><br>
<math>\ BABIP = \frac{H-HR}{AB-SO-HR}</math>

===BB/9 [[保送率|保送率]]===
<small>英文：Walks per Nine Innings</small><br>
<math>\ BB/9 = \frac{(BB + IBB + HBP) \times 9}{IP}</math>

===I/GS [[平均先發局數|平均先發局數]]===
<small>英文：Inning per Game Started</small><br>
<math>\ I/GS = \frac{IP}{GS}</math>

==打者==

===AVG<small>(或BA)</small> [[打擊率|打擊率]]===
<small>英文：Batting Average</small><br>
<math>\ AVG = \frac{H}{AB}</math>

===OBP [[上壘率|上壘率]]===
<small>英文：On Base Percentage</small><br>
<math>\ OBP = \frac{H + BB + HBP}{AB + BB + HBP + SF}</math>
備註: IBB 早已包含在 BB 中, 切記不要重複計算

===TB [[壘打數|壘打數]]===
<small>英文：Total Bases On Safe Hits</small><br>
<math>\ TB = 1 \times 1B + 2 \times 2B + 3 \times 3B + 4 \times HR</math>

===SLG [[長打率|長打率]]===
<small>英文：Slugging percentage</small><br>
<math>\ SLG = \frac{TB}{AB}</math>

====IsoP [[純長打率|純長打率]]====
<small>英文：Isolated Power</small><br>
<math>\ IsoP = SLG - AVG</math>

===BIP% [[將球打進場內機率|將球打進場內機率]]===
<small>英文：Balls put Into Play percentage</small><br>
<math>\ BIP\% = \frac{AB-SO-HR}{AB+BB+HBP+SF}</math>

===IsoD [[選球率|選球率]]===
<small>英文：Isolated Discipline</small><br>
<math>\ IsoD= OBP - AVG</math>

====Pure IsoD====
<small>英文：</small><br>
<math>\ IsoD= OBP - AVG - IBB</math>

===OPS [[整體攻擊指數|整體攻擊指數]]===
<small>英文：On-base percentage Plus Slugging Percentage</small><br>
<math>\ OPS = OBP + SLG</math>

====OOPS [[被攻擊指數|被攻擊指數]]====
<small>英文：Opponent On-base Plus Slugging</small><br>
<math>\ Oops = OBP + SLG</math>

====TA [[攻擊指數|攻擊指數]]====
<small>英文：Total Average</small><br>
<math>\ TA = \frac{TB + SB + BB + IBB + HBP}{AB - H + CS + GIDP}</math>

====GPA [[整體攻擊指數|整體攻擊指數]](平衡過的)====
<small>英文：Gross Production Average</small><br>
<math>\ GPA = \frac{1.8 \times OBP + SLG}{4}</math>

===SB% [[盜壘成功率|盜壘成功率]]===
<small>英文：Stolen base percentage</small><br>
<math>\ SB\% = \frac{SB}{SB + CS}</math>

===OR [[攻擊比|攻擊比]]===
<small>英文：Offense Ratio</small><br>
<math>\ OR = \frac{TB + BB}{AB - H} </math>

===RF [[球員9局製造出局數|球員9局製造出局數]]===
<small>英文：Range Factor</small><br>
<math>\ RF = \frac{PO + A \times 9}{INN}</math>

==皆適用==

===BPF 球場校正值===
<small>英文：Batting Park Factor</small><br>
<math>\ BPF = \frac{\frac{homeRS+homeRA}{homeG}}{\frac{roadRS+homeRA}{homeG}}</math>
