'''通用公共无线接口'''（{{lang-en|CPRI，Common Public Radio Interface}}），是一套技术规范，在无线[[基站|基站]]划分为无线设备控制中心（{{lang|en|REC，Radio Equipment Control}}）和无线设备（{{lang|en|RE，Radio Equipment}}）两部分的基础上，定义了两者之间的接口<ref name=CPRI_home_en>{{Cite web |url = http://www.cpri.info/index.html |title = CPRI官方网站/Home |author = |date = |publisher = |language = en |accessdate = 2015年8月4日 |quote = }}</ref><ref name=CPRI_home_cn>{{Cite web|url =http://www.cpri.info/cn/index.html|title =CPRI官方网站/Home|author =|date =|publisher =|language =中文|accessdate =2015年8月4日|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20150525234603/http://www.cpri.info/cn/index.html|archivedate =2015年5月25日}}</ref>。

CPRI是通信业界的数家厂商合作的结果。制定技术规范的参与方有[[爱立信|爱立信]]、[[华为|华为]]、[[NEC|NEC]]、[[北电网络|北电网络]]和[[西门子|西门子]]<ref name=CPRI_definitions>{{Cite web|url =http://www.cpri.info/cn/definitions.html|title =CPRI官方网站/Definitions|author =|date =|publisher =|language =中文|accessdate =2015年8月4日|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20150923210908/http://www.cpri.info/cn/definitions.html|archivedate =2015年9月23日}}</ref>。第一版的规格书在2003年9月30日发布于CPRI的官方网站<ref name=CPRI_invitation>{{Cite web|url =http://www.cpri.info/cn/invitation.html|title =CPRI官方网站/CPRI 发布会邀请函|author =|date =|publisher =|language =中文|accessdate =2015年8月4日|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20150923210910/http://www.cpri.info/cn/invitation.html|archivedate =2015年9月23日}}</ref>。CPRI支持使用[[网线|网线]]和[[光纤|光纤]]两种介质，而光纤传输能够讓REC和RE安装在相距较远的位置。
