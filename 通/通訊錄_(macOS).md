'''通訊錄'''，是指電話簿軟體應用程式（address book software application），目前美國蘋果電腦只有[[Mac_OS_X|Mac OS X]]有支援這項功能。

==特色==
*輸入與輸出完整之規格（Exports and imports cards to and from [[vCard|vCard]] format.）
*可從LDIF輸入。（Imports cards from LDIF, tab-delimited, and comma-separated files）
*與其它應用程式溝通介面（[[Application_Programming_Interface|API]] to interface with other applications.）
*[[AppleScript|AppleScript]] 支援查詢與新增、修改功能成員與群組。（support for querying, adding, modifying, and removing people and groups）
*[[iSync|iSync]] 可與手機，PDAs, iPods，及其它Macs產品同步化。
*與[[iChat|iChat]] 的整合。
*列印標籤（Prints labels）
*列印郵件列表（Print mailing lists）
*位址更動的改變（Change of address notification）
*連絡群組（Contact groups）
*智慧群組（Smart groups）
*當輸入時自動合併（Auto-merge when importing vCards）
*欄位與類型的客制化（Customize fields and categories）
*電話號碼格式的自動化（Automatic formatting of phone numbers）
*與微軟的系統同步化（Synchronizes with [[Microsoft_Exchange_Server|Microsoft Exchange Server]]）
*語音辨認搜尋（[[Speech_recognition|Speech recognition]] searching）
*整合藍芽（Integrates [[Bluetooth|Bluetooth]]-enabled phones (displaying incoming and missed calls, displaying incoming [[Short_message_service|text messages]]，lets you send text messages using the phone, etc.)）
*具查詢資料庫的功能。Capability to query an [[Lightweight_Directory_Access_Protocol|LDAP]] database containing person information
*可整合協力廠商的程式。[[Plugin|Plugin]] interface so that [[third-party|third-party]] developers can add functionality to the program
