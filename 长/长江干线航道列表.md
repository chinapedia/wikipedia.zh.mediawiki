'''长江干线航道'''是指[[长江|长江]][[干流|干流]]自[[云南|云南]][[水富|水富]]至[[上海|上海]][[长江口|长江口]]这一河段的[[航道|航道]]，全长2,838[[公里|公里]]，流经云南、[[四川|四川]]、[[重庆|重庆]]、[[湖北|湖北]]、[[湖南|湖南]]、[[江西|江西]]、[[安徽|安徽]]、[[江苏|江苏]]和上海等九个省市。长江是[[中国|中国]]内河航运最发达、运输规模最大和最为繁忙的通航河流，也是货运量位居全球内河第一的黄金水道<ref>[http://www.gov.cn/zhengce/content/2014-09/25/content_9092.htm 《国务院关于依托黄金水道推动长江经济带发展的指导意见》（国发〔2014〕39号）]. [[中华人民共和国国务院|中华人民共和国国务院]]. 2014-09-25 [2016-06-07] {{zh-cn}}.</ref>。长江干线航道是长江运输大通道的核心，也一直是中国内河水运发展与建设的重点。按照航道养护管理的特点，长江干线航道可分为：湖北[[宜昌|宜昌]]以上的上游航道，宜昌至湖北[[武汉|武汉]]的中游航道，武汉至[[长江口|长江口]]的下游航道{{efn|在按河道水文和地理特征的划分中，长江中游为宜昌至江西[[湖口县|湖口]]河段，下游为湖口以下河段。}}<ref name =概况>{{cite web | url = http://www.cjhdj.com.cn/detail/history/201401/t20140106_61923.htm | language = zh-cn | title = 长江干线航道概况 | publisher = 长江航道局 | date = 2014-01-06 | accessdate = 2016-06-07 }}</ref>。其中，水富至四川[[宜宾|宜宾]]合江门段30公里由[[四川省交通运输厅|四川省交通运输厅]]负责养护管理，宜昌庙河（上游航道里程62.5公里）至中水门（上游航道里程3.5公里）段59公里由[[长江三峡通航管理局|长江三峡通航管理局]]负责养护管理，其余河段由[[长江航道局|长江航道局]]负责养护管理{{efn|江苏[[太仓|太仓]]浏河口至上海长江入海口段120公里已于2016年5月6日由[[长江口航道管理局|长江口航道管理局]]正式转交长江航道局管理<ref>{{cite news | url = http://www.cjhdj.com.cn/dynamic/news/201605/t20160506_81578.htm | language = zh-cn | title = 长江口航道管理局和长江航道局管理关系调整 长江口120余公里深水航道划入长江航道局管理 | publisher = 长江航道局 | date = 2016-05-06 | accessdate = 2016-06-07 }}</ref>。}}<ref name =概况></ref>。为提升长江干线[[中国内河航道等级|航道等级]]，近年来中国实施了一些列重大航道整治工程，包括在下游重点实施12.5米深水航道延伸至[[南京|南京]]工程，在中游重点实施[[荆江|荆江]]河段航道整治工程，在上游重点实施宜宾－重庆段航道整治工程等<ref name =规划>{{cite web | url = http://www.cjhdj.com.cn/serve/online_notice/world_notice/201512/W020151222579167869542.pdf | language = zh-cn | title=《长江干线“十三五”航道治理建设规划环境影响报告书》 | publisher = 长江航道局 | date = 2015-12 | accessdate = 2016-06-07 }}</ref>。

== 上游航道 ==
长江干线上游航道，为[[宜宾|宜宾]]至[[宜昌|宜昌]]段航道{{efn|目前[[水富|水富]]至宜宾段30公里航道未经系统治理，枯水期仅能通航300吨级船舶<ref>{{cite web | url = http://www.cjhy.gov.cn/hangyundongtai/dianziqikan/hangyunzazhi/201308/t20130813_238340.html | language = zh-cn | title = 关于“十三五”期长江干线航道建设的思考 | publisher = 长江航运杂志 | author = 江凌 | date = 2013-08-13 | accessdate = 2016-06-07 | deadurl = yes | archiveurl = https://web.archive.org/web/20160811075549/http://www.cjhy.gov.cn/hangyundongtai/dianziqikan/hangyunzazhi/201308/t20130813_238340.html | archivedate = 2016-08-11 }}</ref>，因此该段航道暂未纳入上游航道列表。}}，又称[[川江|川江]]，属山区河流航道。其中，宜宾合江门至[[重庆|重庆]][[江津|江津]]为山区天然航道；江津至重庆[[涪陵|涪陵]]为[[三峡水库|三峡水库]]变动回水区航道；涪陵至[[三峡大坝|三峡大坝]]为常年库区航道；三峡大坝至[[葛洲坝|葛洲坝]]为两坝间航道<ref name =概况></ref>。目前，上游航道的主要问题是宜宾至重庆段存在部分急、弯、浅、险的河段，航道的有效尺度较小；三峡水库变动回水区礁石河段航道条件差，浅滩河段受泥沙淤积影响，消落期航道尺度较小；常年库区上段汛期礁石碍航明显，部分浅滩泥沙累积性淤积碍航开始显现；两坝间洪水期流速和比降大，流态紊乱，船舶航行条件复杂<ref name =规划></ref>。上游航道按[[中国内河航道等级|航道技术等级]]可分为宜宾－重庆段和重庆－宜昌段<ref name =概况></ref>。

=== 宜宾－重庆段 ===
[[宜宾|宜宾]]－[[重庆|重庆]]段为宜宾合江门至重庆羊角滩（上游航道里程660.0公里）的河段航道，全长384公里。该河段[[河床|河床]]多为卵石，间有基岩，航槽相对稳定<ref name =概况></ref>。河段有多处碍航险滩，以卵石浅滩碍航为主，航道条件较差，从2005年开始陆续对该段航道实施了整治建设工程，航道等级得到提升。目前[[中国内河航道等级|航道技术等级]]为Ⅲ级，枯水期可通航1,000吨级船舶及组成的船队，正在实施提升至Ⅱ级航道整治工程，预计2020年可基本实现全年通航2,000吨级船舶<ref>{{cite news | title = 长江干线四川航道升级 2000吨以上船舶宜宾常年通行 | url = http://www.chinanews.com/cj/2015/11-24/7638701.shtml | language = zh-cn | publisher = [[中国新闻网|中国新闻网]] | date = 2015-11-24 | accessdate = 2016-06-07 }}</ref>。2015年航道的最小维护标准尺度为水深2.7米×航宽50米×弯曲半径560米<ref name =规划></ref>。

=== 重庆－宜昌段 ===
[[重庆|重庆]]－[[宜昌|宜昌]]段为重庆羊角滩至宜昌的河段航道。该河段过去航道弯窄，滩险众多，[[三峡工程|三峡工程]]蓄水后，大部分滩险消除，航道条件显著改善<ref name =概况></ref>。[[中国内河航道等级|航道技术等级]]已达到Ⅰ级，目前可通航3,000吨级船舶<ref name =规划></ref>。2015年重庆－[[涪陵|涪陵]]段航道的最小维护标准尺度为水深3.5米×航宽100米×弯曲半径800米，涪陵－[[宜昌|宜昌]]段为水深4.5米×航宽150米×弯曲半径1,000米，[[葛洲坝|葛洲坝]]大三江航道为水深4.0米×航宽100米×弯曲半径1,000米<ref name =规划></ref>。

== 中游航道 ==
长江干线中游航道，为[[宜昌|宜昌]]至[[武汉|武汉]]段航道。该段有多个重点浅水道，历来是长江防洪的重要险段和航道建设、维护的重点与难点。[[中国内河航道等级|航道技术等级]]已达到Ⅰ级，目前可通航2,000－5,000吨级内河船舶组成的船队<ref name =规划></ref>。2015年宜昌－下临江坪（中游里程615.0公里）段航道的最小维护标准尺度为水深4.5米×航宽100米×弯曲半径750米，下临江坪－[[城陵矶|城陵矶]]（中游里程230.0公里）段为水深3.5米×航宽100米×弯曲半径750米，城陵矶－武汉段为水深3.7米×航宽150米×弯曲半径1,000米<ref name =规划></ref>。中游航道按河道特性分为宜昌－[[枝城|枝城]]（中游里程570.0公里）段、枝城－城陵矶段和城陵矶－武汉段<ref name =概况></ref>。

=== 宜昌－枝城段 ===

=== 枝城－城陵矶段 ===
[[枝城|枝城]]－[[城陵矶|城陵矶]]段，又称[[荆江|荆江]]，属平原河流航道。以[[藕池镇|藕池口]]为界分为上、下荆江：上荆江长约175公里，河段内弯道较多，弯道内有江心洲，属微弯型河段；下荆江长约165公里，属蜿蜒型河段，河道迂回曲折。目前该河段大部分严重崩岸已通过人工护岸得到控制，总体上河势已趋稳定，但仍有少数河段崩岸。<ref name =概况></ref>

== 下游航道 ==
长江干线下游航道，为[[武汉|武汉]]至[[长江口|长江口]]航道。该河段洲滩群生，航道多变。武汉至[[南京|南京]]段有[[汉江|汉江]]、[[鄱阳湖|鄱阳湖]]水系以及[[皖南|皖南]]诸支流汇入，河床宽窄相间，多形成分汊河段；南京至浏河口（下游里程25.4公里）段江面进一步展宽，特别是出[[江阴|江阴]]后江面成喇叭形逐渐展宽；江阴以下为[[潮汐|潮汐]]河段，受潮汐影响较大<ref name =概况></ref>。[[中国内河航道等级|航道技术等级]]为Ⅰ级，目前武汉至[[湖口|湖口]]段276公里可通航5,000吨级海船，湖口至[[南京|南京]]段432公里可通航5,000吨级至万吨级海船，南京至[[南通|南通]]段227公里可通航3万吨级海船；南通至浏河口段可全天候双向通航5万吨级海船<ref name =规划></ref>。2015年武汉－[[安庆|安庆]]段航道的最小维护标准尺度为水深4.5米×航宽200米×弯曲半径1,050米，安庆－[[芜湖|芜湖]]段为水深6.0米×航宽200米×弯曲半径1,050米，芜湖－南京段为水深9.0米×航宽500米×弯曲半径1,050米，南京－南通段为水深10.2米×航宽500米×弯曲半径1,050米，南通－[[长江口|长江口]]段为水深12.5米×航宽500米×弯曲半径1,050米。目前正在实施长江南京以下12.5米深水航道建设二期工程，并且12.5米深水航道已于2016年7月5日初步贯通至南京，工程计划2019年竣工验收，可实现5万吨级海船通航至南京<ref>{{cite news | title = 长江南京以下12.5米深水航道初通，5万吨级船舶来去自由 | url = http://www.thepaper.cn/newsDetail_forward_1493351 | language = zh-cn | publisher = [[澎湃新闻|澎湃新闻]] | date =2016-07-04 | accessdate=2016-07-07 }}</ref>。
