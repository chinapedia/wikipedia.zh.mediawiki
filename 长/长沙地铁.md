'''长沙地铁'''是指在[[中華人民共和國|中国]][[湖南省|湖南省]][[长沙市|长沙市]]的[[城市轨道交通系统|城市轨道交通系统]]。1号线一期工程以及2号线一期工程总投资221亿[[人民币|人民币]]<ref>[http://hn.rednet.cn/c/2009/02/10/1704983.htm 长沙地铁年内动工2015年建成 总投资221.71亿元]</ref>。长沙地铁能够和[[长沙磁浮快线|长沙磁浮快线]]、[[长株潭城际铁路|长株潭城际铁路]]、[[京广客运专线|京广客运专线]]、[[沪昆客运专线|沪昆客运专线]]实现对接换乘。2013年12月30日，长沙市轨道交通2号线一期无载客试运行通車。2014年4月29日[[长沙地铁2号线|2号线]]开通，成为中国大陆第18个开通地铁的城市。<ref name="takungpao">[http://news.takungpao.com/society/topnews/2014-04/2430924.html 长沙地铁本月开通 今日迎来首批“试乘客”]</ref>

== 简介 ==
目前，[[长沙地铁2号线|2号线]]和[[长沙地铁1号线|1号线]]都已经通车运营，在[[五一广场站|五一广场站]]实现换乘。[[长沙地铁4号线|4号线]] 已经通车试运行，预计于2019年5月正式运营<ref>{{Cite web|url=http://www.hncsmtr.com/zyxw/xwzyxw/20181229/1546078715232391_1.html|title=长沙市轨道交通集团官网-长沙地铁4号线今日通车试运行|accessdate=2019-01-09|work=www.hncsmtr.com}}</ref>。[[长沙地铁3号线|3号线]]、[[长沙地铁5号线|5号线]]一期工程所有车站均已开工，隧道盾构正在进行。[[长沙地铁6号线|6号线]]也已动工兴建。

== 标识释义 ==
长沙地铁标识由“长沙”两字的[[拼音|拼音]]首[[字母|字母]]“C”和“S”构成。地鐵公司聲稱的設計理念是：
* 图案整体为[[红|红]][[蓝|蓝]]两种色调，其中红色是长沙悠久历史文化的载体，有热情好客、积极向上之意，且在传统[[中华文化|中华文化]]中代表吉祥如意，展现了长沙地铁乘客至上、安全第一的服务宗旨；蓝色则代表科技、时尚，沉稳大气，体现了长沙地铁安全快速的特点。
* CS两个字母构成了一幅快车追逐[[太阳|太阳]]的图景，不仅体现了长沙地铁带给市民快速便捷的生活，还彰显了“心忧天下，敢为人先”的长沙精神。快车冲出圆圈，打破常规，又表现了长沙地铁开放、与世界接轨的城市理念，和奋力进取、与时俱进的时代精神。“S”下方为开口笑，体现了乘客的满意是长沙地铁的不懈追求。

==== 3号线 ====
{{main|长沙地铁3号线}}

[[长沙地铁3号线|3号线]]一期工程于2014年1月3日开工，2019年初试运营，长沙地铁3号线全长36.4公里，设25座车站，3号线一期呈西南—东北走向，从坪浦组团的莲坪大道至洋湖垸，穿越湘江，经侯家塘、东塘、长沙火车站、双拥路、长沙学院、开元路最后到达星沙龙角路，快速衔接坪浦组团、主城区和星马片区。同时建设洋湖垸车辆段、张公塘停车场和星沙大道主变电所。[[File:Chaoyangcun_Station_of_Line_3_under_construction.jpeg|thumb]]

==== 5号线 ====
{{main|长沙地铁5号线}}

[[长沙地铁5号线|5号线]]一期工程于2015年开工，2020年建成通车，线路大体沿万家丽路地下铺设，总长22.5公里，设18站。另外还将建一个车辆段。地铁5号线南下过程中有两大工程特别引人注目，一个是下穿[[浏阳河|浏阳河]]工程，另一个是下穿森林[[植物园|植物园]]工程。

==== 6号线 ====
{{main|长沙地铁6号线}}
长沙轨道交通6号线为东西向补充骨干线路，快速衔接河西副中心、城市主中心、星马片区南部、空港组团和黄花机场，加强城市“一主两次”跨江联系， 引导城市东西向拓展。6号线西起梅溪湖二期，途径东方红路→桐梓坡路→湘江→ 湘雅路→烈士公园→迎宾路→人民路，东至黄花机场，预留继续往东延伸的条件，设1座车辆段、1座停车场、1座控制中心。该线路于2016年12月28日举行开工仪式，于2017年1月10日实现首幅地下连续墙开钻，计划于2021年竣工通车运营。

== 票务 ==
长沙市轨道交通按里程计价，起步价2元可乘6公里，超过6公里采用“递远递减”的计价原则，6—16公里范围内每递增5公里加1元，16—30公里范围内每递增7公里加1元，30公里以上每递增9公里加1元。对于短期内需频繁搭乘地铁的乘客，可在客服中心购买一日票（15元/张，首次刷卡后24小时内可任意搭乘地铁）或三日票（40元/张，首次刷卡后72小时内可任意搭乘地铁）。<ref>{{cite web| url=http://www.hncsmtr.com/channels/33.html| title=长沙市地铁交通票务规则| accessdate=2017-05-14| deadurl=yes| archiveurl=https://web.archive.org/web/20170527162844/http://www.hncsmtr.com/channels/33.html| archivedate=2017-05-27}}</ref>

伤残军人凭《中华人民共和国残疾军人证》，因公致残的人民警察凭《中华人民共和国伤残人民警察证》，可免费乘坐长沙市轨道交通。残疾人凭第二代《中华人民共和国残疾人证》可免费乘坐长沙市轨道交通。65周岁及以上老年人须本人凭身份证、老年人优待证原件办理老人卡，持老人卡可免费乘坐长沙市轨道交通。具有长沙市学籍的普通中小学生、高中、职高和中专的在校学生凭身份证（或户口簿）、学生证（或学籍卡）原件办理学生卡，持学生卡可享受长沙市轨道交通票价5折优惠。每位成年人可免费带1名身高不足1.3米的儿童乘车，超过1名按超过人数购买单程票。持普通储值卡乘车可享受长沙市轨道交通票价9折优惠。

== 争议 ==
长沙地铁站出入口外观、储值卡的设计，以及长沙地铁票价在全国同类城市中偏贵等问题，引起了一定的争议。<ref name="丑？">{{cite web | url=http://hunan.sina.com.cn/zt/news/dttc/index.shtml | title=长沙地铁，真的有那么丑吗 | accessdate=2014-03-08}}</ref>

目前當局在地铁站内施行“三级安检模式”，即常态安检模式、加强安检模式和特别安检模式，{{FACT|实际操作对民众造成滋扰，亦带有法理问题}}。2017年8月，3位律师联名入禀[[长沙市|长沙市]][[雨花区|雨花区]]法院，认为全国性及长沙方面地方相关条例，未授权对乘客做出任何形式的身体搜查，长沙地铁安检已经涉及损害[[人身自由|人身自由]]、[[隐私权|隐私权]]等权利，要求法庭确认有关对乘客人身进行检查的行为违法，是中國大陸首个入禀法院质疑地铁安检违法的案件<ref>全国地铁安检第一案，湖南三律师状告地铁公司违法安检 2017-08-15 </ref>。

== 大事记 ==
*2009年2月10日，国家发改委批复了长沙市城市快速轨道交通近期建设规划。按照国务院要求，拟批轨道交通项目城市的基本条件是：GDP达到1000亿元；地方财政一般预算收入达到100亿元；城区人口300万人。批复前，长沙市已完全具备这些条件。 
*2009年4月29日，长沙市轨道交通集团有限公司授牌。 
*2009年7月，国务院批准《长沙市城市快速轨道交通近期建设规划（2008-2015年）》。
*2009年9月28日，轨道交通2号线一期工程正式开工建设。 　
*2010年12月26日，轨道交通1号线一期工程正式开工建设。
*2012年11月26日，轨道交通2号线一期工程实现“洞通”。
*2012年12月26日，轨道交通2号线一期工程实现“轨通”。
*2012年12月，国务院批准《长沙市城市轨道交通近期建设规划(2012～2018年)》。
*2013年4月4日，环保部审批通过《长沙市轨道交通3号线一期工程环境影响报告书》。
