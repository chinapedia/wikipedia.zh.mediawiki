'''長山縣'''，[[中国|中国]]旧[[县|县]]名，[[治所|治所]]在于今[[山东省|山东省]][[滨州市|滨州市]][[邹平县|邹平县]]东南，亦有部分在今[[淄博市|淄博市]]境。

== 历史 ==

汉代时称於陵，《[[元和郡县志|元和郡县志]]》卷十一《淄州》："[[淄州|淄州]]长山县，本汉於陵地。"[[南北朝|南北朝]]时，称[[武强县_(刘宋)|武强县]]。[[隋朝|隋朝]][[开皇|开皇]]十八年（598年），武强县改称长山县。[[大业_(年号)|大业]]初年，[[济南县|济南县]]并入。属[[齊郡|齐郡]]<ref>《[[隋书|隋书]]·志第二十五·地理中》齐郡旧曰齐州......长山旧曰武强，置广川郡，并东清河、平原二郡入，改曰东平原郡。开皇初郡废。又十六年置济南县，十八年改武强曰长山。大业初省济南县入焉。高苑后齐曰长乐。开皇十八年改为会城。大业初改焉。亭山</ref>。[[唐朝|唐朝]]时，属[[淄州|淄州]]、[[淄川郡|淄川郡]]<ref>《[[新唐书|新唐书]]·卷三十八·志第二十八◎地理二》淄州淄川郡......长山，上</ref>。元朝时，初属[[济南路|济南路]]，[[中統_(年號)|中统]]三年（1262年）归[[般阳府路|般阳府路]]<ref>《[[元史|元史]]·卷五十八·志第十》般阳府路......长山，中，初属济南路，中统三年来属......</ref>。[[明朝|明朝]][[洪武|洪武]]二年（1369年）七月，属[[济南府|济南府]]<ref>《[[明史|明史]]·卷四十一·志第十七◎地理二》济南府......长山，府东北。元属般阳路。洪武二年七月来属。西南有长白山。西北有小清河。南有孝妇河......</ref>。清朝时仍属济南府。清朝时，其境内[[周村|周村]]兴起，至[[咸丰_(年号)|咸丰]]年间建[[周村城墙|周村城墙]]。

1937年12月25日，长山县沦陷于日军之手，次日周村沦陷<ref name="a1">{{Cite web |url=http://www.zhcqq.gov.cn/dashiji/200408/6.html |title=大事记 1840年～1948年|accessdate=2016-05-03 |author= |date=2004-08-11|work= |publisher=周村区情网，来源：山东省淄博市周村区史志办公室|language=Zh-hans }}</ref>。1943年，为纪念长山县出生的抗日英雄[[马耀南|马耀南]]，更名为[[耀南县|耀南县]]，至1950年复名<ref>{{Cite web |url=http://epaper.qlwb.com.cn/qlwb/content/20131024/ArticelB03003FM.htm |title=消失在历史深处的老县名|accessdate=2016-05-03 |author=张永红|date=2013-10-24|work= |publisher=齐鲁晚报网，来源：齐鲁晚报|language=Zh-hans }}</ref>。1940年代，周村划出设立[[周村市|周村市]]。1949年后，属[[淄博专区|淄博专区]]。1950年3月，[[山东省人民政府|山东省人民政府]]将[[周村市|周村市]]并入长山县。11月28日，[[山东省人民政府|山东省人民政府]]复置[[周村区|周村区]]<ref>{{Cite web |url=http://www.zhcqq.gov.cn/dashiji/200408/7.html |title=大事记 1949年～1978年|accessdate=2016-05-03 |author= |date=2004-08-11|work= |publisher=周村区情网，来源：山东省淄博市周村区史志办公室|language=Zh-hans }}</ref>。1950年，长山县属[[淄博专区|淄博专区]]，1953年改属[[惠民专区|惠民专区]]，1955年废除，并入邹平县。

== 名人 ==

*[[范仲淹|范仲淹]]，生于北宋苏州，曾随母亲谢氏改嫁到长山县一朱姓人家，并改姓名叫朱说。
