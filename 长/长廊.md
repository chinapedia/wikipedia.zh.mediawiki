'''长廊'''（又称'''千柱廊'''<ref name="易明" />）是[[中国|中国]][[北京|北京]][[颐和园|颐和园]]内的一处[[廊|廊]]。它初建于18世纪中期，东起邀月门，西至石丈亭，全长728米，共273间，超过14,000幅彩绘，被[[吉尼斯世界纪录|吉尼斯世界纪录]]评为世界上最长的木廊<ref>{{cite web|title=Longest covered wooden corridor|url=http://www.guinnessworldrecords.cn/world-records/longest-covered-wooden-corridor|website= Guinness World Records |accessdate=2016-06-02}}</ref><ref name="易明" />。

== 历史 ==
长廊始建于1750年（清乾隆十五年）。[[乾隆皇帝|乾隆皇帝]]将这一地区占为[[皇家园林|皇家园林]]，以便[[孝圣宪皇后|圣母皇太后]]沿廊漫步、观雨赏雪，是[[清漪园|清漪园]]的一部分{{r|易明|page=4}}。1860年（清[[咸丰|咸丰]]十年），在[[第二次鸦片战争|第二次鸦片战争]]期间长廊和清漪园的大部分一起被[[英法联军|英法联军]]付之一炬{{r|易明|page=4}}。

1888年（清[[光绪|光绪]]十四年），[[慈禧太后|慈禧太后]]重修颐和园时重建{{r|易明|page=4}}。1902年，慈禧再次对颐和园大修{{r|易明|page=4}}。进入民国后，长廊与颐和园一起衰败荒芜{{r|易明|page=4}}。1959年，为庆祝[[中华人民共和国|中华人民共和国]]成立十周年，政府集中力一批古建工匠和彩绘艺人对长廊进行了大规模修缮{{r|易明|page=4-5}}。

1966年[[文化大革命|文化大革命]]爆发后，为了应对“[[破四旧|破四旧]]”风潮，颐和园工作人员在部分彩绘上涂了一层白粉用于保护{{r|易明|page=5-6}}。1979年，为庆祝中华人民共和国成立三十周年，对长廊再次进行修缮，并对部分损毁的彩绘进行了重绘或补绘{{r|易明|page=6}}。1990年，长廊作为世界上最长的木廊被记录在吉尼斯世界纪录{{r|易明|page=6}}。2006年，为了北京即将举办的[[2008年奥运会|2008年奥运会]]，北京市政府再对长廊进行了部分修缮{{r|易明|page=6}}。

作为颐和园的一部分，长廊于1998年12月被列入[[联合国教科文组织|联合国教科文组织]][[世界遗产|世界遗产]]。

== 布局 ==
长廊东起邀月门，沿[[昆明湖|昆明湖]]北岸向西延伸。它全程不曾离开湖岸和在它另一边的[[万寿山|万寿山]]脚之间的移行带。湖边标志性的亭子[[石舫|石舫]]位于长廊西端附近。长廊中段经过围绕万寿山靠湖一侧中心建筑群的弯曲段，该中心建筑群的主厅为[[排云殿|排云殿]]，为[[慈禧太后|慈禧太后]]庆祝生日之所。排云门为长廊中心，长廊从中贯穿。排云门本身也是一个标志，也被彩绘覆盖。

长廊全长728米(含4座亭子则为752米)，房顶下的大梁将其分为273间。沿其走向在排云门的两侧各有两座即共四座重檐八角亭，分别代表春夏秋冬四季，由东到西命名为：留佳、寄澜、秋水、清遥。排云门每一侧两亭中间都有道路，湖岸上的亭子能通过长廊的南向短延伸段到达：[[对鸥舫|对鸥舫]]在东，[[鱼藻轩|鱼藻轩]]在西。在西面还有一个和鱼藻轩反向的北向延伸段，走向一座三层八角形瞭望塔。

== 彩绘 ==
长廊的梁柱和天花板全部用[[苏式彩画|苏式彩画]]装饰。总计14,000多幅彩绘，包括640幅[[乾隆|乾隆]]时期的[[西湖|西湖]]风景古建画和数千幅山水、花鸟、博古画，特别是描述中国古典文化、民间故事、历史和传说人物的彩画最受关注<ref name="易明" />。

四座亭每座都在东西两门道处有两幅巨幅彩绘。这些彩绘的主题如下（由东到西）：

;桃花源记 : 《[[桃花源记|桃花源记]]》设定为[[东晋|东晋]][[司马曜|孝武帝]][[太元|太元]]年间事。讲述一个渔民在一处窄山洞的另一侧发现了与世隔绝的山谷“世外桃源”。谷中居民为[[秦朝|秦朝]]躲避战乱者的后人，他们从那时起在这个[[乌托邦|乌托邦]]中和平地定居，不受外界历史发展影响。渔夫回家讲了这个故事，但这个世外桃源再也找不到了。

;齐天大圣斗哪吒 : 此画描述中国[[四大名著|四大名著]]《[[西游记|西游记]]》的一个情节。此情节中，猴王[[孙悟空|孙悟空]]与[[玉帝|玉帝]]派去擒拿他的男孩神仙[[哪吒|哪吒]]战斗。在战斗中，哪吒和孙悟空都把自己变成了三头六臂的神。在画中，哪吒踩着通常用于移动的风火轮。

;张飞夜战马超 : 此画的主题是[[蜀汉|蜀国]]未来[[五虎将|五虎将]]中的二将[[张飞|张飞]]和[[马超|马超]]在[[葭萌关之战|葭萌关之战]]中激烈而僵持的战斗。取材于[[四大小说名著|四大小说名著]]之一、14世纪历史小说《[[三国演义|三国演义]]》。

;八大锤大闹朱仙镇 : 此画展现[[金朝|金朝]]和[[宋朝|宋朝]]之间的一场决战。宋将[[岳飞|岳飞]]因与入侵金军作战而显名。在[[朱仙鎮之戰_(南宋)|朱仙镇之战]]中，宋军几乎战败，但八位将军（八大锤）的勇气将几乎的败局扭转为了决定性的胜利。

;岳飞枪挑小梁王 : 和《八大锤大闹朱仙镇》相似，此画的主题也是关于宋朝英雄岳飞的故事。此画描述岳飞和想通过行贿在比武中胜出的富家贵族子弟小梁王柴桂的交战。在画中，柴桂坠马，岳飞枪刺其心杀之。

;竹林七贤 : 此画在秋水亭的西边，展现了3世纪的七位奔放的文士即[[竹林七贤|竹林七贤]]。文士[[阮籍|阮籍]]、[[嵇康|嵇康]]、[[阮咸|阮咸]]、[[山涛|山涛]]、[[向秀|向秀]]、[[王戎|王戎]]和[[刘伶|刘伶]]持有进步的政治观念，却不能实现抱负。因而他们拒绝寻求名利，转而在竹林中以作诗、饮宴、奏乐、下棋以自娱。

;赵子龙大战长坂坡 : 此画展现《三国演义》的另一幕。在[[长坂坡之战|长坂坡之战]]中，将领[[赵云|赵云]]被[[曹操|曹操]]（后其子[[曹丕|曹丕]]继业，推翻[[汉朝|汉朝]]建立[[曹魏|魏国]]）的敌军取得重大人数优势。此外赵云还要寻找未来蜀国开国者[[刘备|刘备]]的[[麋夫人|麋夫人]]和儿子[[刘禅|刘禅]]。他找到他们后，麋夫人为了不成为赵云的拖累而自杀，赵云为保护幼儿刘禅而陷入战斗。此画展示赵云着白袍，被敌兵包围，握住刘禅。最后，他成功重创敌军，救了刘禅。

;[[三英战吕布|三英战吕布]] : 此画在清遥亭的西边。主题是[[虎牢关之战|虎牢关之战]]，仍是《三国演义》小说的一场战役。此战的一方是当时还忠于养父、控制汉朝皇帝的暴虐者[[董卓|董卓]]的[[吕布|吕布]]，另一方是结义三兄弟，未来蜀国开国者刘备及其手下两位后来位列五虎将的顶级将领[[关羽|关羽]]、张飞。此画展现黑脸的张飞使枪，红脸的关羽使[[偃月刀|偃月刀]]，刘备使双刃剑。
