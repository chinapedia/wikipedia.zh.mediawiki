'''长沙伍家岭砍人事件'''是于2014年3月14日上午发生在[[湖南省|湖南省]][[长沙市|长沙市]][[开福区|开福区]][[伍家岭街道|伍家岭街道]]辖区内的一起[[无差别犯罪|无差别]]砍杀事件<ref>{{cite web |title = 长沙：今日有人当街砍杀市民 已有人员伤亡 |url = http://news.cnr.cn/native/gd/201403/t20140314_515075351.shtml |work = 新华社2014年3月14日消息 |date = 2014-03-14 |accessdate = 2014-03-14}}</ref><ref>[http://news.ifeng.com/photo/hdsociety/detail_2014_03/14/34773678_0.shtml#p=1 长沙发生街头砍杀市民事件]</ref>；事发地点位于[[伍家岭街道|伍家岭街道]]沙湖桥农贸市场附近<ref>[http://news.ifeng.com/mainland/detail_2014_03/14/34773248_0.shtml 长沙：今日有人当街砍杀市民 已有人员伤亡]</ref><ref>[http://www.ntdtv.com/xtr/b5/2014/03/14/a1081012.html 長沙一菜市場 發生砍人事件]</ref>。

海比尔·图尔迪（男，21岁）于该年2月1日来到长沙，在其同乡的饼店打工。他性格内向，脾气暴躁，与同事买买提阿卜拉·买买提尼亚孜（男，22岁）曾因琐事多次争吵，因其来长沙后不习惯当地生活，对工作、生活感到很失意，不服从店内安排，14日早上，老板表示将他辞退。10时许，海比尔·图尔迪与买买提阿卜拉·买买提尼亚孜再次发生争执，海比尔·图尔迪持刀将后者砍倒在地，猛砍致其死亡后情绪极度失控，开始袭击现场群众，砍伤刘某等4人，其中两人当场[[死亡|死亡]]，两人送[[医院|医院]]抢救无效死亡。长沙市特巡警接警后约2分半钟赶到现场，鸣枪示警。歹徒拒不配合并企图袭警，[[警察|警察]]开枪将其击毙。警方在现逮捕2人，后又逮捕3人，经查与此案无关。<ref>[http://news.163.com/14/0315/19/9NDBM8P200011229.html 长沙通报伍家岭地区致多人伤亡案件调查处置情况]</ref>

== 影响 ==
