'''长沙路'''是[[中国|中国]][[天津市|天津市]][[和平区_(天津市)|和平区]]的一条街道。修筑于1922年，原位于[[天津英租界|天津英租界]]界内。长沙路北起[[南京路_(天津)|南京路]]，南到[[重庆道|重庆道]]，长726米，宽9米。目前，长沙路地区是[[天津市历史风貌建筑|天津市历史风貌建筑]]的聚集地之一，大量的名人故居坐落于此。<ref>[http://www.tjhp.gov.cn/servlet/ContentServer?cid=1105523261039&pagename=tjhp%2FPage%2Fsub-t46&lv=2&lv2=1100820937330&lv1=1087380370033&c=Page 和平区志]{{dead link|date=十一月 2017 |bot=InternetArchiveBot }}</ref>

== 历史 ==
[[File:二十世纪五十年代的成都道与长沙路交口.jpg|thumb]]交口|left|200px]]
长沙路于1922年由[[天津英租界工部局|天津英租界工部局]]辟建，当时分别为伯斯道（Perth Road）和加的夫道（Cardiff Road），西安道以北名为伯斯道，又称49号路，即今[[南京路_(天津)|南京路]]至西安道段；西安道以南名为加的夫道，又称52号路，即今西安道至[[重庆道|重庆道]]段。1943年，日治英租界时期，这两条道路分别改称兴亚二区49号路和兴亚二区52号路。1946年，[[天津市政府_(中华民国)|天津市政府]]光复后将这两条道路接顺并以[[湖南省|湖南省]]省会[[长沙市|长沙市]]统一命名。长沙路因在[[天津英租界|天津英租界]]推广界中开发较早，故寓居长沙路的各界名流相对集中，现存名人旧宅为数不少。<ref>《天津老街道》，天津市和平区政协文史资料委员会编</ref>

== 现况与发展 ==
长沙路目前北起[[南京路_(天津)|南京路]]，南到[[重庆道|重庆道]]，从[[南京路_(天津)|南京路]]至西安道一段为南北向，从西安道至[[重庆道|重庆道]]一段为西北-东南向，均为单向道路。长沙路长726米，车行道宽9米，两侧人行道各宽2米，为沥青路面。沿途建有有[[民园大楼|民园大楼]]、[[陶茂正故居|陶茂正故居]]、关麟征旧居、刘汝明旧居、杜友樵旧居、石敬亭旧宅和米春霖旧宅等建筑。

== 交汇道路 ==
[[File:天津诚基中心.jpg|thumb]]交口的[[陶茂正故居|陶茂正故居]]|200px]]

*[[重庆道|重庆道]] 原爱丁堡道（Edinborgh Road） 
*[[成都道_(天津)|成都道]] 原伦敦道（London Road）  
*岳阳道 原福发道（Forfar Road） 
*西安道 原敦桥道（Tonbridge Road）
*潼关道 原孟买道（Bombay Road）
*[[南京路_(天津)|南京路]] 原围墙道（Elgin Road）、原小河道（Creek Road）
