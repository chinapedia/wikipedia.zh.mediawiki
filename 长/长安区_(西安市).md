'''长安区'''为[[西安市|西安市]]的一个市辖区，位于西安市主城区南部。东有西安国家民用航天产业基地、与[[蓝田县|蓝田县]]为邻，西有[[沣渭新区|沣渭新区]]、[[西安高新技术产业开发区|西安高新技术产业开发区]]、与[[鄠邑区|鄠邑区]]接壤，南接[[宁陕县|宁陕县]]、[[柞水县|柞水县]]，北靠[[雁塔区|雁塔区]]、[[灞桥区|灞桥区]]，是建设中的西安现代国际化大都市新型主城区。

== 概况 ==
长安区位于东经108°38′～109°14′，北纬33°47′～34°18′，南北跨度约55公里，东西跨度约52公里。全区总面积1590平方公里，常住人口109万，现辖25个街道办事处，区政府驻地西长安街669号。

== 历史 ==
公元前202年，[[西汉|西汉]]设立[[长安县|长安县]]。此后，县建制的[[長安縣|长安县]]存在了两千两百余年。2002年9月16日，经[[国务院|国务院]]批准撤销[[長安縣|长安县]]，设立西安市长安区。

== 民族宗教 ==
长安居民以汉族为主，属少数民族杂散居住地区，有46个少数民族在区内定居。

长安是全国宗教工作重点地区之一，中国佛教八大宗派中，[[慈恩宗|慈恩宗]]、[[净土宗|净土宗]]、[[华严宗|华严宗]]、[[律宗|律宗]]、[[三阶教|三阶教]]的祖庭寺或发祥地分别在长安的[[兴教寺|兴教寺]]、[[香积寺|香积寺]]、[[华严寺|华严寺]]、[[净业寺|净业寺]]、[[百塔寺|百塔寺]]。区内有佛教、道教、基督教三大教派，尤以“终南茅蓬修道”文化，在中国乃至世界享有崇高声望和影响。

== 交通运输 ==
[[108国道|108国道]]以及[[京昆高速|京昆高速]]、[[包茂高速|包茂高速]]、[[连霍高速|连霍高速]]穿越长安境内，[[西安绕城高速公路|西安绕城高速公路]]擦边而过，京昆高速设三星出口，包茂高速设韦曲出口。

[[西康铁路|西康铁路]]、[[西成高铁|西成高铁]]穿越长安境内，西康铁路设[[西安南站|西安南火车站]]，位于长安[[引镇街道|引镇]]，西成高铁设[[阿房宫站|阿房宫站]]，主要为[[沣东新城|沣东新城]]提供乘降，这些乘火车至[[陕南|陕南]]、[[四川|四川]]、[[重庆|重庆]]比较便利。

[[西安地铁2号线|西安地铁二号线]]和规划中西安地铁五号线连接长安区和市区，分别通至韦曲和郭杜比亚迪产业园。
