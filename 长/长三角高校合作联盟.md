'''长三角高校合作联盟'''为[[长三角地区|长三角地区]]的优秀高校联合体，简称“E8”，建立于2005年。目前拥有八所高校成员。联盟每年举办一次长三角八校友好辩论赛，具有一定影响力。

==成员==
* 上海：[[复旦大学|复旦大学]]、[[上海交通大学|上海交通大学]]、[[华东师范大学|华东师范大学]]、[[同济大学|同济大学]]
* 江苏：[[南京大学|南京大学]]、[[东南大学|东南大学]] 
* 浙江：[[浙江大学|浙江大学]]
* 安徽：[[中国科学技术大学|中国科学技术大学]]

==历史==
2005年，长三角地区的[[上海|上海]]、[[江苏|江苏]]、[[浙江|浙江]]的六所高校（[[复旦大学|复旦大学]]、[[上海交通大学|上海交通大学]]、[[浙江大学|浙江大学]]、[[浙江工业大学|浙江工业大学]]、[[浙江理工大学|浙江理工大学]]、[[东南大学|东南大学]]）组成高校合作联盟，加强校际交流，开展辩论赛、暑期实践等类活动。之后[[南京大学|南京大学]]与[[同济大学|同济大学]]加入联盟，形成八校合作联盟。2010年，[[上海交通大学|上海交通大学]]“长三角高等教育合作优秀人才培养”研讨会议决定，八校本科生共享通识课，并开展大学生创新实践。<ref>{{cite web |url=http://www.js.xinhuanet.com/xin_wen_zhong_xin/2010-03/26/content_19351407.htm |title=存档副本 |accessdate=2010-11-21 |deadurl=yes |archiveurl=https://web.archive.org/web/20160303155121/http://www.js.xinhuanet.com/xin_wen_zhong_xin/2010-03/26/content_19351407.htm |archivedate=2016-03-03 }} 优秀资源共享 长三角八高校将共享本科通识课</ref>

2011年，由[[上海|上海]]、[[江苏|江苏]]、[[浙江|浙江]]三省市教育部门主办的第三届长三角教育联动发展研讨会在上海召开，宣布长三角地区八所高校已经组成了长三角高校合作联盟（简称“E8”），八所高校互认学分。<ref>{{cite web |url=http://paper.people.com.cn/rmrb/html/2011-04/11/nw.D110000renmrb_20110411_6-04.htm |title=存档副本 |accessdate=2012-10-02 |deadurl=yes |archiveurl=https://web.archive.org/web/20160304064325/http://paper.people.com.cn/rmrb/html/2011-04/11/nw.D110000renmrb_20110411_6-04.htm |archivedate=2016-03-04 }} 沪苏浙高校将互认学分</ref>

[[华东师范大学|华东师范大学]]和[[中国科学技术大学|中国科学技术大学]]随后取代了学校实力差距较大的[[浙江工业大学|浙江工业大学]]和[[浙江理工大学|浙江理工大学]]，组建为新长三角高校合作联盟（新E8）。新E8由[[上海|上海]]的[[上海交通大学|上海交通大学]]、[[复旦大学|复旦大学]]、[[华东师范大学|华东师范大学]]、[[同济大学|同济大学]]，[[江苏|江苏]]的[[南京大学|南京大学]]、[[东南大学|东南大学]]，[[浙江|浙江]]的[[浙江大学|浙江大学]]，[[安徽|安徽]]的[[中国科学技术大学|中国科学技术大学]]八所[[985工程|985工程]]名校组成。<ref>http://ckc.zju.edu.cn/00CD022D-8886-44DC-8577-A2099F72B707.html{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }} 浙江大学 长三角八校2012年夏令营报名通知</ref>

==长三角八校友好辩论赛==
八校友好辩论赛是长三角地区高水平大学之间文化交流的重要形式，2005年在[[浙江大学|浙江大学]]举办了第一届辩论赛。<ref>[http://www.chinanews.com/edu/2014/12-07/6852659.shtml 长三角八所高校上海辩论 中国新闻网]</ref>2013年第九届辩论赛由中国科学技术大学承办，正式启用大赛专用LOGO，并进行了全程网络直播和网络投票。<ref>[http://news.ustc.edu.cn/xwbl/201412/t20141209_206483.html 我校本科生蝉联长三角八校辩论赛金奖 中国科学技术大学新闻网]</ref><ref>[http://jw.nju.edu.cn/jwoldweb/page/news668.html 南大在长三角八校(E8)2010年友好辩论赛中取得佳绩 南京大学教务处] {{webarchive|url=https://web.archive.org/web/20140808101516/http://jw.nju.edu.cn/jwoldweb/page/news668.html |date=2014-08-08 }}</ref><ref>[http://www.jwc.sjtu.edu.cn/web/sjtu/GNFXW/198411-1980000004905.htm 上海交通大学辩论队在2014年长三角八校友好辩论赛中喜获佳绩 上海交通大学教务处]</ref>
