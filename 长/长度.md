'''长度'''是[[一维空间|一维空间]]的度量，是国际单位制的七种基础度量之一<ref name="SI">[http://www.sizes.com/units/SI.htm International System of Units（SI）]</ref>。

== 概述 ==
在几何体中长度通常指最长的一维。通常在量度[[二維空間|二維空間]]中量度直線邊長時，稱呼長度數值較大的為'''長'''，不比其值大或者在「側邊」的為'''寬'''，而表達方式通常為長乘寬。所以'''寬度'''其實也是'''長度'''量度的一種，故此在[[三維空間|三維空間]]中量度「垂直長度」的'''[[高|高]]'''都是长度。

在字典中，长度被解释为“两端之间的距离”<ref name="字典">[http://xh.5156edu.com/html5/172588.html 新华字典]</ref>。这个距离既可以是直线，也可以是曲线；既可以在平面上，也可以在三维空间中。

在广义的长度中，人们也用“长度”这一更形象的度量来表示其他抽象的度量（尤其是时间度量），但一般只用到长度相关的形容词而不会用长度度量的单位。如时间的长度、寿命长度等等<ref name="长度">[http://xh.5156edu.com/html3/10629.html 长字的释义]</ref>。

== 长度的描述 ==

通常，用“长”和“短”来大致形容长度的大小，如“长木棒”/“短木棒”。在特定范围也会用特定的形容词来表述，如表示地理距离用“远”和“近”，描述人的身高用“高”和“矮”，还有纸张厚度等等。本质上这些均为长度。同样其他度量用长度来表示也可以使用对应的形容词，如时间的长短等。<ref name="长度"/>

用形容词来描述长度均有一个相对的概念，没有哪个形容长度的词语是不成对出现的。有长才有短，有高就有矮。一个人跟某些人比可以称作高，但跟另一些人比可能又变为“矮”。用古语的话说，就是“尺有所短，寸有所长”，当然此成语已不局限于字面意义<ref>[http://chengyu.soouo.com/list/chengyu_1555.htm 尺有所短，寸有所长]{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

== 长度的单位 ==
单位化的长度可以作为绝对的度量。作为日常生活中最重要的度量之一，长度的度量单位有很多种。
*[[国际单位制|国际单位制]]中将[[米_(单位)|米]]（符号m）作为标准单位<ref name="SI"/>，因而“1米”有多长需要科学而严谨的定义。
最新一米定义于1983年国际度量衡大会重新制定，此次定义时使用了自然中随处可见的[[光|光]]，并且在70年代[[光速|光速]]的测定已非常精确，所以最终定义为光在[[真空|真空]]中行进299 792 458分之1秒的距离为一标准米。故此，一旦光速得到更精准的量度，改变的数值会是米而非光速。

与米相关的单位有很多，从相当于10<sup>-24</sup>米的攸米（ym）到千米（km）乃至[[京米|京米]]（Gm）均有各自的单位和换算方式。

此外，存在不依赖于“米”的定义的长度单位如[[英里|英里]]、英尺<ref>{{Cite web |url=http://www.iselong.com/English/0008/8718.htm |title=英制尺寸的由来 |access-date=2010-12-14 |archive-url=https://web.archive.org/web/20101218161310/http://iselong.com/English/0008/8718.htm |archive-date=2010-12-18 |dead-url=yes }}</ref>与专业性较强的如[[海里|海里]]等也有较多应用。而且同样一个单位也有不同的称呼，如中文-{[[公尺|公尺]]}-就是米，[[公分|公分]]就是厘米，这些称呼在日常生活中也比较常见<ref>[http://xh.5156edu.com/html5/233878.html 公尺]</ref><ref>[http://xh.5156edu.com/html5/221571.html 公分]</ref>。

*[[光年|光年]]，指光在真空中一年時間内传播的距離。

*[[秒差距|秒差距]]，1秒差距= 3.2616 光年。

== 相对论中的长度 ==
在人们主观意识中，在任何环境下同一个物体长度应该是不变的。但是在[[狭义相对论|相对论]]理论体系中，长度与质量、时间等一样，具有相对性，取决于速度、位置等因素。
