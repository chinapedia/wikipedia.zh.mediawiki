'''长春话'''是中国[[吉林省|吉林省]]省会所使用的方言，属于[[东北官话|东北官话]]哈阜片长锦小片<ref>东北官话内部分区方式待定</ref>。由于长春城区在建国后工业、文化、商业服务业大量发展，各地移民大批融入，所以又形成了口音较四邻农村县份更轻，更接近普通话的特点。

==与普通话的区别==
松辽平原汉语方言大体都是一致的，长春话和吉林省中西部（松原，白城等），黑龙江中西部（哈尔滨，齐齐哈尔，黑河等）的特征也是相同或类似的<ref>[http://wenku.baidu.com/link?url=fcEkyvL8duaqzDoOmbYBsQ2LrGif_j97wqxtQAaPN-ez5mT7mJynD5YwFOC7dhk3n7d9OFZGXRUbmxs5n6luVajwTPBZD9Xumx2JJhZjvQi 中国汉语调值总表]</ref>

===特有词汇===
*小回 = 右转
*大回 = 左转
*完事儿 = 结束
*捎 = 带（东西）
*忽悠 = 欺骗
*琢磨 = 考虑
*寻思 = 想
*瞅 = 看
*哥们儿 = 朋友（男性）
*娘们儿 = 妇女，也可泛指女性
*爷们儿 = 成年男性
*幹仗 = 口角
*嘎 = 打赌
*嗯呢 = 表应允，四平、公主岭及松原地区亦有极近似用法
