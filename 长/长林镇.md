'''长林镇'''是一个已经撤销的行政区划<ref>{{cite news |url=http://epaper.xiancn.com/xawb/html/2011-07/05/content_50337.htm |title=汉中撤并46个乡镇 调整后再无乡建制 |accessdate=2019-03-12 |work=西安晚报 |agency=西安日报社 |date=2011-07-05}}</ref>，曾隶属于[[中华人民共和国|中国]][[陕西省|陕西省]][[汉中市|汉中市]][[勉县|勉县]]<ref>{{cite web |title=2010年统计用区划代码-勉县 |url=http://www.stats.gov.cn/tjsj/tjbz/tjyqhdmhcxhfdm/2010/61/07/610725.html |website=国家统计局 |accessdate=2019-03-12 |date=2011}}</ref>。2011年，汉中市人民政府申请撤销长林镇，辖区已经划归[[老道寺镇|老道寺镇]]<ref>{{cite web |author1=汉中市人民政府 |title=汉中市人民政府关于审定汉中市乡镇机构改革行政区划调整方案的请示 |url=http://www.hanzhong.gov.cn/xxgk/gkml/zfwj/zfjzfbwj/hzzwj/201410/t20141018_21197.html |website=汉中市人民政府网站 |accessdate=2019-03-12 |date=2011-04-25 |quote=2．撤销长林镇，将其整建制划归老道寺镇管辖。镇政府驻地老道寺。}}</ref>。

== 行政区划 ==
长林镇曾下辖以下行政区：

长寨村、范寨村、小白坡村、汤谷村、季寨村、杨寨村、南营村<ref>{{cite web |title=2010年统计用区划代码和城乡划分代码-长林镇 |url=http://www.stats.gov.cn/tjsj/tjbz/tjyqhdmhcxhfdm/2010/61/07/25/610725108.html |website=国家统计局 |accessdate=2019-03-12 |date=2011}}</ref>。
