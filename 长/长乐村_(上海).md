'''长乐村'''原名'''凡尔登花园'''，是位于[[上海市|上海市]][[黄浦区|黄浦区]][[陕西南路_(上海)|陕西南路]]39～45弄的一片[[里弄#花园里弄|花园里弄]]住宅。里弄于1925年由华懋地产有限公司投资，安利洋行设计，建成于1929年<ref name="花园里弄">[http://shtong.gov.cn/newsite/node2/node71994/node81772/node81776/node81788/userobject1ai109110.html 上海名建筑志 - 长乐村与丰子恺的日月楼] 上海地方志办公室.</ref>。长乐村被认为是传统法国式住宅，但在同时也采用了包括西班牙式平屋顶和近代立体形式在内的建筑风格<ref name="花园里弄"/><ref name="lp">{{Cite book |title = 透视上海近代建筑 |publisher = 上海古籍出版社 |pages =  | ISBN = 9787532536764 | accessdate = 2015-01-21 | url = }}</ref>。

丰子恺曾在其位于长乐村内的住所——“日月楼”居住长达21年。现其旧居二至三楼已被开辟为纪念馆<ref name="腾讯大申网">{{Cite web |title = 丰子恺旧居：上海的“凡尔登花园” |url = http://sh.qq.com/a/20130911/012938.htm#p=3 |publisher = 腾讯大申网 | date = 2013-09-11 | accessdate = 2015-01-21 }}</ref>。1994年，长乐村入选第二批[[上海市优秀历史建筑|上海市优秀历史建筑]]。

== 历史 ==
长乐村所在地块位于原[[上海法租界|上海法租界]]高档住宅区，原为德国侨民乡村俱乐部，在[[第一次世界大战|第一次世界大战]]后因德国成为战败国，法租界公董局有意就此接管俱乐部，但在遭到中国政府反对下，转而以购买的形式买下这块地皮，并命名为'''凡尔登花园'''，以作为对[[凡尔登战役|凡尔登战役]]的纪念<ref name="lp"/>。当时的凡尔登花园面积有5公顷，后来公董局在地块东部建造[[花园饭店|上海法国总会]]，西部则由华懋地产有限公司兴建新式里弄，也就是现在的长乐村<ref name="花园里弄"/>。

== 建筑特色 ==
长乐村占地2公顷，总建筑面积19148平方米<ref name="花园里弄"/>。共由7条支弄、7排建筑组成，共129个单元。每个单元都是前部2层、后部3层。长乐村排屋的屋顶格外精致，均为法国孟莎式屋顶，陡坡屋顶部分设[[老虎窗|老虎窗]]<ref name="lp"/>。外墙上部粉刷以白色沙浆，下部配以清水砖墙做出勒脚。大门处做出山墙，屋顶很尖，顶角仅60度。水平腰线也用清水砖嵌出，作装饰用<ref name="lp"/>。各户入口处门斗上均有欧式装饰。住宅内部前部是起居室；中间两侧为餐室及楼梯间、厨房；后部为次屋、平房，为仆人所住，还设有厕所及后院，有后门通向户外<ref name="lp"/>。主楼梯设在起居室内而仆人用梯则设在厨房内。

== 丰子恺旧居 ==
[[File:Fengzikai_Jiuju.JPG|缩略图]]
陕西南路39弄长乐村93号为丰子恺旧居——“日月楼”。丰子恺有为自己的居所取名的爱好，而93号寓所则因可于白天看日出日落，晚间欣赏明月，而得名“日月楼”<ref name="腾讯大申网"/><ref name="花园里弄"/>。日月楼为一西班牙式建筑，丰子恺于1954年花费6000元购入，在此居住共21年，直至去世<ref name="花园里弄"/>。在[[文化大革命|文化大革命]]期间，日月楼被占，丰子恺被迫只能在阳台上架一张小床，居住。在丰子恺过世后，丰家迁出此宅，后又在2008年重新购入此宅二至三楼以期将这里作为丰子恺旧居对外开放（一楼因条件无法谈拢而并未购得）<ref name="腾讯大申网"/>。2010年，丰子恺旧居正式对外开放。2006年1月9日，丰子恺旧居被公布为[[黄浦区文物保护单位#黄浦区登记不可移动文物|黄浦区登记不可移动文物]]。2014年10月起，一楼住户以影响其日常起居为由，实则因要求补贴且开价较高，无法达成一致，故而在丰子恺旧居门口恶意张贴有“注意：居民住宅，谢绝参观！”字样的告示，以阻挠游客前往二楼及三楼进行参观<ref>[http://www.chinanews.com/cul/2014/10-21/6702059.shtml 中国新闻网→新闻中心→文化新闻丰子恺旧居暂停开放 因一楼居民反对]</ref>。
