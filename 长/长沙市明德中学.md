'''长沙市明德中学'''，原名長沙市第三中學，位于[[长沙市|长沙市]][[天心区|天心区]]新韶西路豹子岭，是湖南省重点中学和首批省级示范性高中。由[[胡元倓|胡元倓]]在1903年创办，是[[湖南省|湖南省]]最早的近代新式教育学校。<ref>{{Cite web |url = http://mdzx.net/new/xxgk/xxqk_index.aspx |title = 长沙市明德中学概况 |publisher = 长沙市明德中学 |language = zh-cn |deadurl = yes |archiveurl = https://web.archive.org/web/20131029190038/http://mdzx.net/new/xxgk/xxqk_index.aspx |archivedate = 2013-10-29 }}</ref>

== 历史 ==
1903年，[[胡元倓|胡元倓]]在[[长沙|长沙]]创建了私立中学明德学堂，并在其他地方创建了小学、大学等私立学校，实践癸卯学制。此后由于种种原因，只保留下了中学。

1904年，[[黄兴|黄兴]]、[[陈天华|陈天华]]等人在明德学堂创立反清组织“华兴会”，召集师生30余人参加，因此被称为“辛亥革命策源地”。

[[抗日战争|抗日战争]]期间，明德中学曾在8年内迁往[[霞岭|霞岭]]、[[晓岚港|晓岚港]]、[[樟梅乡|樟梅乡]]、[[蓝田|蓝田]]，直到1946年3月才迁回长沙。<ref>{{Cite web |url = http://www.mdzx.net/new/NewConent.aspx?nid=633698814295937500 |title = 长沙市明德中学历史简介 |publisher = 长沙市明德中学 |language = zh-cn |deadurl = yes |archiveurl = https://web.archive.org/web/20131104182400/http://www.mdzx.net/new/NewConent.aspx?nid=633698814295937500 |archivedate = 2013-11-04 }}</ref>
