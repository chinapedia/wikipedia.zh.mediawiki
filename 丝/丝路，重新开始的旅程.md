《'''丝路，重新开始的旅程'''》是[[中国中央电视台|中国中央电视台]]2013年推出的一部纪录片，2013年9月2日晚在[[中国中央电视台综合频道|中国中央电视台综合频道]]首播。《丝路，重新开始的旅程》立项于2010年，最初的计划是展示[[丝绸之路|丝绸之路]]的历史，但摄制组后来决定改为呈现当代丝绸之路上普通人的奋斗故事。纪录片讲述了56个人物的故事，共分八集，每集50分钟，跨越[[中国|中国]]、[[吉尔吉斯斯坦|吉尔吉斯斯坦]]、[[巴基斯坦|巴基斯坦]]、[[印度|印度]]、[[土耳其|土耳其]]、[[阿联酋|阿联酋]]、[[卡塔尔|卡塔尔]]、[[意大利|意大利]]八个国家。

== 拍摄 ==
此纪录片在2010年获得立项。2010年时一位中共中央高层领导出访土耳其，在土耳其看到了该国拍摄的一部丝绸之路纪录片的片花。这位高层感到疑惑：为什么中国人没有拍一部丝绸之路的纪录片。中央电视台和[[日本放送协会|日本放送协会]]之前曾合拍《{{link-ja|絲綢之路 (紀錄片)|NHK特集 シルクロード|丝绸之路}}》（1980年）和《{{link-ja|新丝绸之路|新・シルクロード}}》（2006年），但中方只负责中国境内的拍摄，中国境外的拍摄均由日方完成。随后，中国中央电视台文化专题部接到任务，拍摄一部以丝绸之路为题材的纪录片。<ref name=infzm>{{cite news|title=不要嘲笑别人，也不要嘲笑自己 纪录片“丝路”中的轮回|url=http://www.infzm.com/content/94249|accessdate=2015-05-30|agency=南方周末|date=2013-09-14|deadurl=yes|archiveurl=https://web.archive.org/web/20150707213357/http://www.infzm.com/content/94249|archivedate=2015-07-07}}</ref>

摄制组原本的计划是拍一部展示丝绸之路古代文化的纪录片。为此，摄制组收集了历史、考古、宗教、民族等资料，光是电子书就有一百多本，字数达到千万量级。摄制组还向研究丝绸之路的中外学者[[林梅村|林梅村]]、[[荣新江|荣新江]]、[[吴芳思|吴芳思]]等人多次请教。摄制组初步的思路是，按照断代史的思路制作这八集纪录片。2011年7月，此片正式开机，摄制组的编导们带着机器离开北京收集素材。然而，当回到北京之后相互交流的时候，他们却发现自己对丝路上所见到的当代故事、人物更感兴趣。央视领导批准摄制组改变拍摄方向，2012年初摄制组正式转向，改为拍摄一部反映丝绸之路当代面貌的纪录片。<ref name=infzm /><ref>{{cite news|title=陈晓卿曝纪录片《丝路》花絮|url=http://365jia.cn/news/2013-09-03/AD6A2AECC47C782D.html|accessdate=2015-05-27|agency=新浪娱乐|date=2013-09-03|deadurl=yes|archiveurl=https://web.archive.org/web/20150527144706/http://365jia.cn/news/2013-09-03/AD6A2AECC47C782D.html|archivedate=2015-05-27}}</ref>导演[[陈晓卿|陈晓卿]]后来透露，做这个决定时历史部分的七百个镜头图已经完成绘制，摄制组已经做好了拍摄计划：借鉴[[美剧|美剧]]，让中央电视台主持人出演历史人物，[[王小丫|王小丫]]扮演公主，[[毕福剑|毕福剑]]出演商人，[[撒贝宁|撒贝宁]]、[[柴静|柴静]]等饰演战争中的阵亡士兵。<ref name=feng />

摄制组共用了一年多时间，在[[中国|中国]]、[[吉尔吉斯斯坦|吉尔吉斯斯坦]]、[[巴基斯坦|巴基斯坦]]、[[印度|印度]]、[[土耳其|土耳其]]、[[阿联酋|阿联酋]]、[[卡塔尔|卡塔尔]]、[[意大利|意大利]]拍摄。为拍摄[[和田玉|和田玉]]的开采，摄制组爬上海拔4,500米的高山，五天之内经历了“从夏到冬”的天气变化，一位年轻导演还因[[高原反应|高原反应]]和[[胃出血|胃出血]]住进医院。在巴基斯坦北部拍摄时，由于[[恐怖组织|恐怖组织]]刚刚在那里杀了人，当地政府派人带着[[AK47|AK47]]保护摄制人员。<ref name=feng>{{cite news|title=《丝路》讲述56个普通人的故事|url=http://news.ifeng.com/gundong/detail_2013_08/26/29033405_0.shtml|accessdate=2015-05-30|agency=都市晨报|date=2013-08-26}}</ref>三年时间里，他们一共积累了超过四万分钟的素材，最后剪辑为8集50分钟的纪录片，平均每集1,100个镜头。<ref>{{cite news|title=让“丝路”的旅程重新开始|url=http://news.163.com/13/0903/02/97QHSB4L00014AED.html|accessdate=2015-05-27|agency=北京晨报|date=2013-09-03}}</ref>与此前两部纪录片《[[絲綢之路_(紀錄片)|丝绸之路]]》、《[[新丝绸之路|新丝绸之路]]》不同，《丝路，重新开始的旅程》着眼当前，讲述丝路上普通人的故事，以“丝路上的追梦人”为中心。摄制组拍摄时采访了一百多位人物，但最后出现在纪录片中的只有56人。纪录片的摄制组共二十余人。项目由中国中央电视台投资，预算是每分钟2万元人民币。<ref name=sdz>{{cite news|title=4万分钟素材，央视重拍“丝路”纪录片|url=http://money.163.com/13/0912/05/98I49QTV00253B0H.html|accessdate=2015-05-27|agency=时代周报|date=2013-09-12}}</ref>

2013年8月21日，[[中央电视台纪录频道|中央电视台纪录频道]]在北京举办看片会。9月2日，《丝路，重新开始的旅程》在[[中央电视台综合频道|中央电视台综合频道]]晚十点四十分的《[[魅力纪录|魅力纪录]]》首播。此片的首播选在[[习近平|习近平]]出访[[中亚|中亚]]四国前夜，9月3日习近平开始了对[[土库曼斯坦|土库曼斯坦]]、[[哈萨克斯坦|哈萨克斯坦]]、[[乌兹别克斯坦|乌兹别克斯坦]]和[[吉尔吉斯斯坦|吉尔吉斯斯坦]]的访问。<ref>{{cite news|title=习近平出访中亚四国并出席两场峰会|url=http://news.sina.com.cn/c/2013-09-04/021028124442.shtml|accessdate=2015-05-30|agency=山西晚报|date=2013-09-04}}</ref>

== 内容 ==
《丝路，重新开始的旅程》讲述了现今丝绸之路上56个人的奋斗故事。除了远嫁中东的[[国际象棋|国际象棋]]选手[[诸宸|诸宸]]，绝大多数主人公都是默默无闻的普通人，例如在[[昭苏草原|昭苏草原]]学习游牧的[[重庆|重庆]]人张昱、闯荡意大利的中国模特王诗晴、在[[西安|西安]]求学的土耳其男青年阿福、在[[洛阳|洛阳]]建造寺庙的[[泰国|泰国]]工匠奈、[[喀喇昆仑公路|喀喇昆仑公路]]的客车司机王建平、立志学习“[[达瓦孜|达瓦孜]]”的[[维吾尔|维吾尔]]少年阿里木。纪录片展示了这些人温暖、光明的励志故事。<ref name=infzm />纪录片跨越中国、吉尔吉斯斯坦、巴基斯坦、印度、土耳其、阿联酋、卡塔尔、意大利八个国家。<ref>{{cite news|title=纪录片《丝路,重新开始的旅程》即将亮相央视|url=http://news.qq.com/a/20130823/014580.htm|accessdate=2015-05-30|agency=中央电视台|date=2013-08-23}}</ref>

== 评价 ==
纪录片采用同步展开、穿插叙事的方式，往往是一个故事尚未讲完便切入下一个，第二个故事没讲完的时候有切了回去。[[中央电视台纪录频道|中央电视台纪录频道]]总监[[刘文_(总监)|刘文]]称，这是国际化的叙事方式，节奏明快，充满跳跃，信息的密度大为提高。但这种叙事手段也引起了争议。中央电视台制片人[[颜占领|颜占领]]认为此片的“故事种子”很好，但是“发芽后，长势一般，收成总量虽有，颗粒却不够生动饱满，不过偶有野生的果子掺杂其间，给人瞬间的鲜亮”。制片人[[于继勇|于继勇]]认为，通过讲述故事来打动人，需要故事有一定的体量；《丝路，重新开始的旅程》的故事偏散，体量有缺陷，打动人很难。<ref name=sdz />

2013年12月4日，在搜狐网和鲁迅文化基金联合举办的第一届鲁迅文化奖颁奖盛典上，《丝路，重新开始的旅程》获年度纪录片奖。<ref>{{cite news|title=2013鲁迅文化奖年度纪录片：《丝路》|url=http://cul.sohu.com/20131205/n391283254.shtml|accessdate=2015-05-30|agency=搜狐文化|date=2013-12-05}}</ref>2014年6月13日，《丝路，重新开始的旅程》在[[上海电视节|上海电视节]]上[[国家新闻出版广电总局|国家新闻出版广电总局]]举办的“第二届优秀国产纪录片及创作人才扶持项目表彰活动”中与其他七部纪录片共同被评为优秀国产纪录片长片。<ref>{{cite news|title=《习仲勋》《苦难辉煌》等纪录片获官方表彰|url=http://news.qq.com/a/20140614/007446.htm|accessdate=2015-05-30|agency=人民网-人民日报|date=2014-06-14}}</ref>
