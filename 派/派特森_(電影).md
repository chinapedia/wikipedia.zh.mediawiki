《'''派特森'''》（{{lang-en|'''''Paterson'''''}}）是一部2016年[[美國|美國]]和[[德國|德國]]合拍的[[劇情片|劇情片]]，為[[吉姆·賈木許|吉姆·賈木許]]執導和編劇。由[[亞當·崔佛|亞當·崔佛]]與[[席芬坦·法拉哈尼|席芬坦·法拉哈尼]]主演。

《派特森》被[[第69屆坎城影展|第69屆坎城影展]]選為競爭[[金棕櫚獎|金棕櫚獎]]的競賽片之一<ref name="IW">{{cite web |url=http://www.indiewire.com/article/cannes-live-stream-2016-film-festival-lineup-list |title=2016 Cannes Film Festival Announces Lineup |accessdate=2016-04-14 |work=IndieWire}}</ref><ref name="VarCannes">{{cite web |url=http://variety.com/2016/film/festivals/cannes-film-festival-official-selection-lineup-2016-1201753269/ |title=Cannes 2016: Film Festival Unveils Official Selection Lineup |accessdate=2016-04-14 |work=Variety}}</ref>，在那贏得了{{le|金棕櫚狗狗獎|Palm Dog Award}}<ref>{{cite web |url=http://www.indiewire.com/article/palm-dog-award-2016-nellie-paterson-jim-jarmusch-cannes-20160520 |title=The 2016 Palm Dog Posthumously Awarded to Nellie, The Dog From Jim Jarmusch's 'Paterson' |date=2016-05-20 |accessdate=2016-05-20 |author=Vikram Murthi |work=Indiewire}}</ref>。電影也於[[2016年多倫多國際影展|2016年多倫多國際影展]]上的「特別放映」單元上展出。該片被{{le|亞馬遜影業|Amazon Studios}}和{{le|Bleecker Street (公司)|Bleecker Street (company)|Bleecker Street}}定於2016年12月28日在美國上映。

== 劇情 ==
故事敘述派特森是[[紐澤西州|紐澤西州]]的[[派特森_(紐澤西州)|派特森]]的一名公車司機和詩人。他每天都會依循著相同的模式生活，如下班後就會回到家與妻子蘿拉、愛犬團聚和吃飯，但派特森仍會在日復一日、一成不變的生活中利用甜美的詩句增添樂趣。

== 製作 ==
2016年1月，據透露，[[亞當·崔佛|亞當·崔佛]]和[[席芬坦·法拉哈尼|席芬坦·法拉哈尼]]將共同於[[吉姆·賈木許|吉姆·賈木許]]執導和編劇的電影中合作。製片商K5 Film的奧利佛·賽門與丹尼爾·鮑爾將作為執行製片人，而Animal Kingdom的喬書亞·阿斯特拉罕和卡特·羅根則會擔任監製一職<ref>{{cite web|url=http://deadline.com/2016/01/adam-driver-to-star-in-jim-jarmuschs-paterson-amazon-and-k5-to-co-finance-1201681250/|title=Adam Driver To Star In Jim Jarmusch’s ‘Paterson’, Amazon And K5 To Co-Finance|first=Ali|last=Jafaar|date=2016-01-13|accessdate=2016-05-05}}</ref>。

== 發行 ==
[[File:Cannes_2016_25.jpg|thumb]]、[[席芬坦·法拉哈尼|席芬坦·法拉哈尼]]和[[吉姆·賈木許|吉姆·賈木許]]出席於2016年的[[第69屆坎城影展|第69屆坎城影展]]]]
電影最先於2016年5月16日在[[第69屆坎城影展|第69屆坎城影展]]上作全球首映<ref name="IW">{{cite web |url=http://www.indiewire.com/article/cannes-live-stream-2016-film-festival-lineup-list |title=2016 Cannes Film Festival Announces Lineup |accessdate=2016-04-14 |work=IndieWire}}</ref><ref name="VarCannes">{{cite web |url=http://variety.com/2016/film/festivals/cannes-film-festival-official-selection-lineup-2016-1201753269/ |title=Cannes 2016: Film Festival Unveils Official Selection Lineup |accessdate=2016-04-14 |work=Variety}}</ref>。{{le|亞馬遜影業|Amazon Studios}}獲得了電影的美國發行權<ref>{{cite web|url=http://www.ew.com/article/2016/01/12/casting-net-adam-driver-jim-jarmusch|title=Casting Net: Adam Driver joins Jim Jarmusch comedy Paterson|publisher=[[Entertainment_Weekly|Entertainment Weekly]]|first=Will|last=Robinson|date=2016-01-12|accessdate=2016-05-05}}</ref>。後來宣布，{{le|Bleecker Street (公司)|Bleecker Street (company)|Bleecker Street}}和亞馬遜影業共同合作負責定於2016年12月28日在美國上映<ref>{{cite web|url=http://variety.com/2016/film/news/adam-driver-paterson-release-date-1201822050/|title=Adam Driver’s ‘Paterson’ Gets Awards-Season Release Date (EXCLUSIVE)|website=[[Variety_(magazine)|Variety]]|first=Brent|last=Lang|date=2016-07-25|accessdate=2016-07-25}}</ref>。

=== 評價 ===
[[爛番茄|爛番茄]]新鮮度高達97%，基於36條評論，平均分為9.1/10<ref>{{cite web|url=https://www.rottentomatoes.com/m/paterson/|title=Paterson (2016)|website=[[Rotten_Tomatoes|Rotten Tomatoes]]|accessdate=2016-09-14}}</ref>，而在[[Metacritic|Metacritic]]上得到87分，代表「普遍好評」<ref>{{cite web|url=http://www.metacritic.com/movie/paterson|title=Paterson|website=[[Metacritic|Metacritic]]|accessdate=2016-09-14}}</ref>，收穫普遍讚譽。
