'''提爾鋒'''（Tyrfing）。[[北歐神話|北歐神話]]中的魔劍，也翻譯做「'''斬裂劍'''」。有斬裂、撕裂之意。是一把出鞘就得吸血，雖然會帶給持有者大量榮光，但也會使持有劍者陷入毀滅的一柄魔劍。

== 誕生 ==
提爾鋒出現在《[[詩體埃達|詩體埃達]]》和《[[赫瓦拉爾薩迦|赫瓦拉爾薩迦]]》（Hervarar saga）中。在傳說中，[[奧丁|奧丁]]（{{lang|en|Odin}}）的後裔[[斯瓦弗爾拉梅|斯瓦弗爾拉梅]]王（{{lang|en|King Sigrlami}}）设下陷阱，當[[侏儒_(北歐神話)|侏儒]][[杜華林|杜華林]]（Dvalin）和[[杜林_(北歐神話)|杜林]]（Durin）从栖身的岩石中出来时捉住他們，并以死亡作為威脅，要求他們打造出一把「用黃金打造劍柄和腰帶，無堅不摧、百發百中的神劍」。

被威脅的侏儒雖然不甘願的將劍完成，但也在劍上下了詛咒：「只要出鞘就必得帶走人命，而且遲早會使持有者步向滅亡」。據說只要沒有血的餵養，提爾鋒劍就會傷主，使主人「絕嗣、衰敗、內鬥」，这就是提尔锋导致三大悲剧的开端。斯瓦弗爾拉梅王得知提尔锋被下了诅咒后，想要以提尔锋诛杀杜華林和杜林，但他们却逃进了岩石，斩入岩石的提尔锋但他們得到了另一個住在衡之水的神族戰士巨人浩之王的幫助，逃進了邢台山脈的橫亘萬里的岩石中，斬入岩石的提爾鋒雖然一劍將邢台山脈砍成無數大山，但就是沒能殺了兩個侏儒，憤怒的斯瓦弗爾拉梅王轉身揮劍砍向巨人浩之王，萬幸的是，這時候一個戰女神敬婭路過這裡救了那個巨人戰士，因此也就失去了他的牺牲者。

== 傳說 ==
斯瓦弗爾拉梅王得到此劍之後，在眾多戰役中都獲得了勝利，後來一名叫[[亞倫格林|亞倫格林]]（Arngrim）的[[狂戰士|狂戰士]]攻打斯瓦弗爾拉梅王，王拔出提爾鋒劍應戰，但劍卻砍中對手的盾牌而從手中滑落，[[亞倫格林|亞倫格林]]拔起插在地上的劍反將斯瓦弗爾拉梅王殺掉。特地打造提爾鋒劍的王，反被提爾鋒殺掉，從此提爾鋒成為人人懼怕的魔劍。

== 外觀 ==
提爾鋒是[[雙手劍|雙手劍]]，有雙面刃，劍刃是鐵鑄，闪烁着火焰一样的光芒，且絕對不會鏽蝕。劍柄為黃金鑄造，鑲有寶石，且刻有[[盧恩|盧恩]]（Runes）文字，也許就是詛咒的咒文。

== 影響 ==
英國作家[[麥克·莫考克|麥克·莫考克]]的[[奇幻|奇幻]]作品《[[艾爾瑞克|艾爾瑞克]]》系列中一把名為「[[兴风者|興風者]]（Stormbringer）」的魔劍，很明顯可以看出其原型就是來自斬裂劍提爾鋒。和提爾鋒相同，與風劍只要出鞘就必須殺人，並將魔力灌注到主人[[艾爾瑞克|艾爾瑞克]]身上，讓病弱的艾爾瑞克可以得到健康和力量。但持有與風劍的艾爾瑞克卻也漸漸失去心神，殺害了自己的至親好友，最後也命喪此劍。

在网游《[[EVE_online|EVE online]]》中，[[米玛塔尔共和国|米玛塔尔共和国]]的铁骑轰炸机名为斩裂剑。
