'''羊角拗'''（[[學名|學名]]：''{{lang|la|Strophanthus divaricatus}}''），別名'''羊角籐'''、'''羊角捩'''、'''羊角藕'''、'''羊角柳'''、'''羊角樹'''、'''羊角紐'''、'''羊角扭'''、'''大羊角扭蔃'''、'''菱角扭'''、'''黃葛扭'''、'''武靴籐'''、'''花拐籐'''、'''瀝口花'''、'''打破碗花'''、'''華毒毛旋花子'''、'''斷腸草'''、'''鯉魚橄欖'''、'''貓屎殼'''、'''螺心魚'''、'''金龍角'''、'''倒釣筆'''<ref>[http://www.kinmen.gov.tw/News/News.aspx?DocTag=30748 金中學生展開五虎山植物之旅]{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}金門縣政府全球資訊網</ref>等，為[[夾竹桃科|夾竹桃科]][[羊角拗屬|羊角拗屬]]<ref name = "樹木谷">[http://www.hktree.com/tree/Strophanthus%20divaricatus.htm 羊角拗]樹木谷</ref>植物，分佈於中國[[廣西|廣西]]、[[廣東|廣東]]、[[福建|福建]]、[[貴州|貴州]]、[[越南|越南]]及[[老撾|老撾]]<ref name="植物通">[http://www.botany.ac.cn/latin/Apocynaceae/Strophanthus-divaricatus-Lour-Hook-et-Arn.htm 羊角拗]{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}植物通</ref>等地，長於山坡或路旁灌木叢中。在香港，被稱之為「香港四大毒草」之一<ref name="生態日記">[http://www.hknature.net/chi/ecodiary/diary_30d.html 生態日記 2002年3月1日]{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}香港自然網</ref>，也是香港原生物種<ref name = "香港植物標本室">[http://www.hkherbarium.net/Herbarium/c_search_end.asp?id=7948 香港植物標本室]</ref>。因果實一對叉生，形如羊角，因而得名<ref name = "生態日記"/><ref>[http://nature.iacm.gov.mo/greendata/plant/plant.htm 澳門趣異植物] {{webarchive|url=https://web.archive.org/web/20081203215927/http://nature.iacm.gov.mo/greendata/plant/plant.htm |date=2008-12-03 }}澳門自然網</ref>。

==形態特徵==
[[File:Strophanthus_divaricatus_in_Shenzhen_China_1.jpg|thumb]]]]
羊角拗是一種木質[[藤本植物|藤本]]<ref name = "樹木谷"/>[[灌木|灌木]]植物，可高約10米。枝幹圓柱狀，棕褐色，有明顯白色皮孔。葉橢圓狀長圓形或卵形，頂端短漸尖，對生，紙質，葉每邊7-9條側脈，全緣，長約5-10厘米，寬約3-3.5厘米，葉柄長約5毫米。花為[[聚傘花序|聚傘花序]]腋生及頂生，花萼5枚，萼管長約5毫米，裂片急尖黃紅色，長約7毫米；花冠5枚黃色，呈漏斗狀，冠管長約2厘米，裂片延伸成長線狀，長約5-8厘米，寬約2.5毫米；喉部的副花冠10枚，冠片分離；雄蕊5枚，生於冠管上部，雌蕊長約13毫米，花期3-7月<ref name = "香港植物標本室"/>。果為[[蓇葖果|蓇葖果]]，橢圓狀，木質，叉生，形如羊角，長約15厘米，果期7-11月<ref name = "香港植物標本室"/>。種子披白色絹質毛，呈線狀披針形，長約2厘米，寬約5毫米<ref>[http://www.cvh.org.cn/difangzhi/hainan/page/3/249.pdf 海南植物誌 249頁]{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}中國數字植物標本館</ref>。全株含有白色或黃色乳液<ref>[http://www.leafvein.net/plant_page/Apocynaceae.htm 羊角拗(Goat horns)Strophanthus divaricatus]</ref><ref name = "植物通"/>。

==醫藥用途及其毒性==
全株具有毒性，毒性成份為[[羊角拗甙|羊角拗甙]]及[[毒毛旋花甙|毒毛旋花甙]]等<ref>[http://www.kepu.net.cn/gb/lives/plant/class/flhb009.html 羊角拗 Strophanthus divaricatus (Lour.) HK. et A.]{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}中國科普博覽</ref>。誤食可引致頭痛、頭暈、嘔吐、腹痛、腹瀉、瞳孔擴大、脈搏不規則、痙攣昏迷及心跳停止引致死亡等中毒症狀。但同時亦可作殺菌、消炎、止癢等<ref name =hkcww>[http://hkcww.org/hkplant/readid.php?id=2770 2770-羊角拗]柴娃娃植物網</ref>作用，主要作外用可治扭傷、皮癬及膿腫等<ref>[http://health.atnext.com/index.php?fuseaction=Article.ListArticle&sec_id=6349068&iss_id=20071029&art_id=10347585 香港四大毒草逐隻捉]{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}，壹蘋果健康網</ref>。莖部纖維可編織成繩索，種毛可作填充物<ref name =hkcww/>。
