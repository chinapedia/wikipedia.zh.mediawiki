'''羊羹'''是一種[[和菓子|日本点心]]，一般以[[紅豆|紅豆]]為材料，其後亦發展至有[[栗子|栗子]]、[[番薯|番薯]]甚至是[[綠茶|綠茶]]口味的不同款式。羊羹除了傳統的甜味之外，也另有一些咸味的羊羹。羊羹在[[日本|日本]]非常著名，為一般日本人品茶時常備的甜點之一。

依據寒天的用量可以分成「煉羊羹」及「水羊羹」兩種種類。由於煉羊羹的[[白利糖度|糖度]]很高，在[[真空包裝|真空包裝]]下妥善保存的話，大多可以在[[常溫|常溫]]下保存超過一年以上。因為以上特點，羊羹也被當作緊急時的非常食糧。

== 歷史 ==
羊羹源自[[中國|中國]]，其後再傳入日本而成為當地的傳統點心。相傳，初期羊羹的確是一種加入[[羊肉|羊肉]]煮成的[[羹|羹]]，再因其胶质而后冷卻成凍，用于佐餐，或者携帶，作为行旅的乾糧。後来羊羹傳至日本。在[[鎌倉時代|鎌倉時代]]至[[室町時代|室町時代]][[佛教|佛教]]的[[禪宗|禪宗]]傳入日本，由於[[明治維新|明治維新]]之前，多數[[日本佛教|日本佛教]][[僧侶|僧侶]]為[[素食|素食]]，[[清規|清規]]戒律不許食[[葷|葷]]，故羊羹亦慢慢演化成為一種以豆類製成的果凍形食品。此後，羊羹成為了[[茶道|茶道]]的其中一種著名茶點，而日本人亦慢慢將羊羹發展和轉化，變成今天多款不同口味的羊羹。<ref>[http://www.wagashi.or.jp/monogatari/shiru/yurai.php 和菓子の由来]{{ja}}</ref>

不過，亦另有傳說指出，羊羹原是中國的一種仿照羊的[[肝臟|肝臟]]形狀制成的食品。傳至日本後，由於「肝」與「羹」兩字讀音相近，被混淆，久而久之就被稱之為羊羹，直至今天。<ref>[http://news.xinhuanet.com/life/2009-10/26/content_12327547.htm 羊肝羹是羊肝做的嗎？] {{webarchive|url=https://web.archive.org/web/20160530063254/http://news.xinhuanet.com/life/2009-10/26/content_12327547.htm |date=2016-05-30 }}</ref>

初期的羊羹製法主要以小豆加上[[小麥|小麥]]粉混和，再加以蒸煮而成。製作方法如另一種米的粉末加入砂糖、小麥澱粉蒸煮製成的[[和菓子|和菓子]]{{link-ja|外郎|ういろう (菓子)}}相若。

[[天正|天正]]17年（1589年），[[和歌山|和歌山]]的駿河店店主[[岡本善右衛門|岡本善右衛門]]將原有的羊羹作改良，在餡料中加入[[寒天|寒天]]（一種曬製而成的乾菜，由於含有[[瓊脂|瓊脂]]，煮後會凝固成果凍狀）製成煉羊羹，並用[[竹|竹]]竿將羊羹的形狀鞏固成方形。此後，羊羹就成了現時長方形的形狀。羊羹主要依據寒天的用量分成「煉羊羹」及「水羊羹」兩種種類。

[[江户时代|江户时代]]是煉羊羹最盛行的時期，其中以江戶本鄉的藤村羊羹最為著名，而與此同時四周亦開滿了林林總總以製作羊羹為主的知名店鋪。

現時，日本除了傳統的羊羹外，亦開發了不少不同口味的品種。除了成為日本人日常的茶點外，亦成為了一種日本的特式土產。

中國北方的[[北京|北京]]和[[天津|天津]]等地亦出產羊羹，除了沿用小豆、甘栗等材料外，還有加入水果如[[山楂|山楂]]、[[桃子|桃子]]以及[[蘋果|蘋果]]等的水果口味。

羊羹在[[台灣日治時期|日治時期]]傳入[[台灣|台灣]]。[[二戰|二戰]]結束後，日本戰敗，全台各地僅剩由台灣人經營的羊羹店鋪<ref>{{cite book|url=http://210.60.91.2/project/2005/wagasi.pdf |language=ja |title=《日本の和菓子「羊羹」について》 |publisher=高苑科技大學應用外語系 |date=2017-05-02 |isbn= |pages=頁13}}</ref>。著名產地例[[花蓮縣|花蓮縣]][[玉里鎮|玉里鎮]]、[[宜蘭縣|宜蘭縣]][[蘇澳鎮_(台灣)|蘇澳鎮]]等。

== 主要材料 ==
[[瓊脂|瓊脂]]主要來自洋菜，提供了果凍狀的口感與半透明外觀。

調味料一般都是以豆為主，但由於現時已發展至有多種口味，故材料方面亦開始多樣化。
*[[紅豆|紅豆]]
*[[栗子|栗子]]
*[[山芋|山芋]]
*[[柿子|柿子]]
*[[無花菓|無花菓]]
*[[綠茶|綠茶]]或[[抹茶|抹茶]]
*[[巧克力|巧克力]]
*[[蜂蜜|蜂蜜]]
*[[鳳梨|鳳梨]]

== 著名口味 ==
*本煉：傳統的原味羊羹，以三至四小時慢火製成，無紅豆顆粒，口味較甜，適合佐以較苦澀的茶。
*小倉：以紅豆為主要材料製成，其中含有紅豆顆粒。
*栗：以[[栗子|栗子]]為主要材料製成。
*黑糖：口味較甜，以[[黑糖|黑糖]]製成。
*抹茶：綠茶口味的羊羹。
*山葉：清甜口味，顏色多為[[紫色|紫色]]。
