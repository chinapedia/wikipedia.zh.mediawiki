'''乙炔'''，俗称'''電石氣'''，是[[炔烴|炔烴]]化合物系列中體積最小的一員，主要作工業用途，特別是燒焊金屬方面。与氧气组成切割套装时，俗称'''風煤'''，風指壓縮氧气，煤指乙炔。

乙炔於1836年由英國科學家[[艾德蒙·戴维|艾德蒙·戴维]]（Edmund Davy）發現，化學式為{{化學式|C|2|H|2}}，有一個如下圖所示的直线型結構：
:H - C ≡ C - H

乙炔在室溫下是無色、極易燃的氣體。純乙炔是無臭的，但工業用乙炔由於含有[[硫化氫|硫化氫]]、[[磷化氫|磷化氫]]等雜質，而有一股大蒜的氣味。乙炔的化學能主要貯存於它的三鍵中。

在攝氏400度以上， 乙炔會聚合生成[[乙烯基乙炔|乙烯基乙炔]]（{{化學式|C|4|H|4}}）和[[苯|苯]]（{{化學式|C|6|H|6}}）。在攝氏900度以上則會形成[[炭黑|炭黑]]。

[[碳酸鈣|碳酸鈣]]（[[石灰岩|石灰岩]]）和[[煤炭|煤炭]]是生產乙炔的主要原料。首先，碳酸鈣會轉化為[[氧化鈣|氧化鈣]]，煤炭則轉化為[[焦炭|焦炭]]。然後氧化鈣和焦炭會發生反應形成[[碳化鈣|碳化鈣]]和[[一氧化碳|一氧化碳]]：
:CaO + 3C → CaC<sub>2</sub> + CO
碳化钙加水會形成乙炔和[[氫氧化鈣|氫氧化鈣]]：CaC<sub>2</sub> +2H<sub>2</sub>O → C<sub>2</sub>H<sub>2</sub>↑ + Ca(OH)<sub>2</sub>

==分子结构==
就[[价键理论|价键理论]]而言，在每个碳原子上2s [[轨道|原子轨道]] 与一个2p轨道[[杂化|轨道杂化]]形成sp杂化体。 另外两个2p轨道仍然是非杂化的。 两个sp杂化物[[重叠矩阵|轨道重叠]]的两个末端在碳之间形成强的[[σ键|σ键]]，而在另外一头的末端上，氢原子也通过σ键结合。 两个不变的2p轨道形成一对较弱的[π键]。<ref> Organic Chemistry第7版。 J.McMurry，Thomson 2008 </ref>

由于乙炔是直链状的，因此具有 D<sub>∞h</sub> [[晶体点群|点群]]结构。<ref>{{Housecroft3rd|pages=94–95}}</ref>

==用途==
每年，大約百分之80在美國生產的乙炔是用作製造其他化學品的。剩餘的則主要被用於乙炔銲接。在氧氣中燃燒乙炔可以形成攝氏3300度的火焰，每克釋放出11800焦耳的能量。

乙炔也被用於碳化物燈。以前，碳化物燈是在汽車和礦工用的燈。現在還有一些山洞探索者使用碳化物燈。碳化物燈是利用把碳化鈣加水燃燒形成乙炔時的火焰照明。

現在，乙炔是用於鐵的[[淬火|滲碳]]（硬化）過程的。在過去十年的研究發現乙炔是最適合這用途的碳氫化合物。

乙炔具[[硝化|硝化]]、[[反硝化|反硝化]]抑制作用。

==化學性质==
===生成金属盐类===
乙炔可以生成[[乙炔钠|乙炔钠]]。和[[银氨溶液|银氨溶液]]反应，生成[[乙炔银|乙炔银]]沉淀。和亚铜氨溶液反应，生成[[乙炔亚铜|乙炔亚铜]]沉淀。
===氧化===
1.可燃性：2CH≡CH+5O<sub>2</sub>→4CO<sub>2</sub>+2H<sub>2</sub>O <br />
现象：火焰明亮、带浓烟 ， 燃烧时火焰温度很高（>3000℃），用于气焊和气割。其火焰称为氧炔焰。<br />
2.被KMnO4氧化：能使紫色酸性與中性高锰酸钾溶液褪色。<br />酸性：
:C<sub>2</sub>H<sub>2</sub> + 2KMnO<sub>4</sub> + 3H<sub>2</sub>SO<sub>4</sub>→2CO<sub>2</sub>+ K<sub>2</sub>SO<sub>4</sub> + 2MnSO<sub>4</sub>+4H<sub>2</sub>O
中性：
:3C<sub>2</sub>H<sub>2</sub>+ 2KMnO<sub>4</sub> + 2H<sub>2</sub>O → CH<sub>3</sub>COOH + 2CH<sub>3</sub>COOK + 2MnO<sub>2</sub>

===加成===
与[[水|水]]、[[卤素|卤素]]单质、[[卤化氢|卤化氢]]等加成。<br />
与H<sub>2</sub>加成：CH≡CH+H<sub>2</sub> → CH<sub>2</sub>=CH<sub>2</sub><br />
与HCl的加成：CH≡CH+HCl →CH<sub>2</sub>=CHCl <br />
并可以自身加成生成[[苯|苯]]。

===加聚===
与[[乙烯|乙烯]]相比易生成低聚物，例如[[乙烯基乙炔|乙烯基乙炔]]，在特定催化剂条件下生成[[聚乙炔|聚乙炔]]。

==安全==
由於三鍵中的化學能，乙炔在壓力超過100 [[帕斯卡|kPa]]下會發生分解反應，此反應為[[放熱反應|放熱反應]]，因此可引發劇烈的[[爆炸|爆炸]]。液態或固態乙炔也會發生相同的分解反應，因此高壓乙炔必須溶解在[[丙酮|丙酮]]或[[二甲基甲醯胺|二甲基甲醯胺]]中，並置於含有多孔性材質（Agamassan）的鋼瓶中儲存。

==水果催熟==
碳化鈣俗稱電石（calcium carbide），在接觸到水或潮濕的空氣時會產生乙炔氣（acetylene），乙炔和乙烯同樣有催熟效果，只是其活性只有乙烯約三千分之一而己{{請求來源}}。電石催熟，操作簡單，但其用量無法正確的掌握，因電石在催熟48小時以後仍然無法完全溶解，使用者大都只憑經驗使用，且果實在靠近電石的地方容易有灼傷的現象發生。電石催熟的方法為將紙箱內層敷上水泥紙，後將電石以報紙包上置於紙箱之最下層，電石的用量約200公克配20公斤果實，再將硬熟的果實置於箱內後以水泥紙包紮密封，不可漏氣。紙箱之上方以棉被或布袋覆蓋，視品種之不同，經48～72小時後即可完成催熟的作業，催熟率可達90%以上。此種方法效果穩定，適用於小量果實之催熟處理。如數量多時，可以透氣之塑膠籠盛裝硬熟果，地面墊上水泥柱或木材數支，其尺寸約12*15公分．長度不拘，將電石以報紙包紮置於兩支水泥柱之中問，後將盛裝的硬熟果籠置於柱上，高度三層籠高（約120公分），依序排列，四周以棉被或布袋或催熟專用之催熟被覆蓋，必須注意覆蓋得密不通風，但切忌使用不透氣的資材如塑膠布類覆蓋，以免妨礙水分之蒸發而影響催熟之效果，如此視品種及成熟度之不同，經48～72小時之催熟後即可達到轉色、軟熟而具有商品價值。
