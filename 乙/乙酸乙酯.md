'''乙酸乙酯'''是[[乙酸|乙酸]]中的[[羥基|羥基]]被乙氧基取代而生成的[[化合物|化合物]]，结构简式为{{Chem|CH|3|COOCH|2|CH|3}}。

== 物理性质 ==
乙酸乙酯是无色易燃易挥发的液体；有特殊香味；微溶于水，易溶于[[有机溶剂|有机溶剂]]。

== 主要用途 ==
*人造水果的要素、硝酸纖維素、假漆、瓷漆、飛機翼布塗料等的溶劑、合成[[無煙火藥|無煙火藥]]、[[人造皮革|人造皮革]]、照相用[[底片|底片]]、[[電極|電極]]板、[[人造絲|人造絲]]、[[香水|香水]]、清潔[[紡織品|紡織品]]、製藥時添加香味。
*用於食品上之合成香料。
*作為氮-亞硝基雙乙醇胺的[[粹取|粹取]][[溶劑|溶劑]]。
*[[隱形眼鏡|隱形眼鏡]]的除霉。
*塗料塑膠的溶劑及其他溶劑。
*在纺织工业中可用作清洗剂
*在食品工业中可作为特殊改性酒精的香味萃取剂
*还用作制药过程和有机酸的萃取剂

== 製備 ==
乙酸乙酯可由[[乙酸|乙酸]]、[[乙酸酐|乙酸酐]]或[[乙烯酮|乙烯酮]]與[[乙醇|乙醇]]相互作用制得。乙酸與乙醇的[[酯化反應|酯化反應]]反應方程式如下：

[[File:酯化反应生成乙酸乙酯.PNG|550px]]

[[同位素|同位素]]示蹤法顯示：該反應中乙酸脫去-COOH中的-OH，成為乙氧基，取代了乙醇中-OH中的H原子。

:<math>\ CH_3COOH+C_2H_5OH \ \xrightarrow {H_2SO_4} \ CH_3COOC_2H_5+H_2O</math>

反應中[[濃硫酸|濃硫酸]]的作用為催化，吸水。該反應中的收集裝置中用到了飽和[[碳酸鈉|{{chem]]溶液，其作用有四點：

#溶解揮發出來的[[乙醇|{{chem]]
#乙酸乙酯在鹽溶液中溶解度較小
#鹽溶液密度較大，便於分層

== 水解反應 ==
在无机[[酸|酸]]或[[碱|碱]]的催化下，乙酸乙酯[[水解|水解]]为乙酸和乙醇：

:<math>\ CH_3COOC_2H_5 + H_2O \ \xrightarrow {H^+} \ CH_3COOH + C_2H_5OH </math>
:<math>\ CH_3COOC_2H_5 + OH^- \ \xrightarrow {H_2O} \ CH_3COO^- + C_2H_5OH </math>

== 醇解反应 ==
乙酸乙酯與[[甲醇|甲醇]]的[[醇解|醇解]]反應（[[酯交换反应|酯交换反应]]）方程式如下：

:<math>\ CH_3COOCH_2CH_3 + CH_3OH \ \xrightarrow\ CH_3COOCH_3 + CH_3CH_2OH </math>

== 安全性 ==
=== 健康危害 ===
乙酸乙酯大鼠[[半數致死量|LD<sub>50</sub>]]為11.3g/kg<ref name=Ullmann>Wilhelm Riemenschneider, Hermann M. Bolt "Esters, Organic" in Ullmann's Encyclopedia of Industrial Chemistry, Wiley-VCH, Weinheim, 2005. {{doi|10.1002/14356007.a09_565.pub2}}. Article Online Posting Date: April 30, 2005</ref>；对眼、鼻、咽喉有刺激作用；高浓度吸入可引进行性麻醉作用，急性肺水肿，肝、肾损害；持续大量吸入，可致呼吸麻痹；误服者可产生恶心、呕吐、腹痛、腹泻等；有致敏作用，因血管神经障碍而致牙龈出血；可致湿疹样皮炎。

慢性影响：长期接触本品有时可致角膜混浊、继发性贫血、白细胞增多等。

=== 燃爆危险 ===
本品具刺激性，具致敏性。本品易燃，其蒸气与空气可形成爆炸性混合物，遇明火、高热能引起燃烧爆炸。与氧化剂接触猛烈反应。其蒸气比空气重，能在较低处扩散到相当远的地方，遇火源会着火回燃。
