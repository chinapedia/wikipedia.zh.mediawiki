'''乙未廣州起義'''，又称'''第一次廣州起義'''或'''乙未广州之役'''，是[[清朝|清朝]][[光绪|光绪]]二十一年（1895年）農曆九月初九[[重阳节|重阳节]]由[[兴中会|兴中会]]领导人[[杨衢云|杨衢云]]、[[孙中山|孙中山]]、[[陆皓东|陆皓东]]、[[郑士良|郑士良]]等人发动的起义。兴中会计划一举攻下[[广州|广州]]，將陸皓東設計的[[青天白日旗|青天白日旗]]飄揚於[[兩廣總督|兩廣總督]]衙門，然后北上，经[[湖南|湖南]][[湖北|湖北]]进攻[[北京|北京]]。起义消息不慎泄露，陆皓东被清廷逮捕处死，成為為民主共和流血獻身第一人。孙中山仅以身免，被清廷通緝，东渡日本、然後轉到[[夏威夷|夏威夷]]、[[美國|美國]]、[[英國|英國]]。

== 背景 ==
{{main|甲午战争}}
日本自[[明治維新|明治維新]]後國力漸強，而謀進一步擴張，遂積極侵略朝鮮。1894年7月25日發生中日[[甲午戰爭|甲午戰爭]]，第一階段，從1894年7月25日到9月17日，陸戰主要是在朝鮮半島上的平壤之戰，海戰主要是黃海海戰。第二階段，從1894年9月17日到11月22日，以陸戰為主，戰場位於遼東半島。清廷對日本入侵朝鮮之意見分歧，[[光緒帝|光緒帝]]主戰，[[李鴻章|李鴻章]]主和，[[慈禧太后|慈禧太后]]則拿不定主意，認為「戰比和好」。於是，日本大軍進逼之際，清廷始倉惶備戰，故戰事一敗塗地。[[孫文|孫文]]《伦敦被难记》中自述，「近者日本命將遣師，侵入吾土，除宅居戰地之人民外，罕有知中日開釁之舉也者。」<ref name="孫中山全集">孫文自述、甘作霖譯《伦敦被难记》，[[布里斯托尔|布里斯特耳]]1897年英文初版，上海商務印書館1912年5月中文譯版，刊《孫中山全集》第一卷，北京：中華書局，1981年</ref>{{rp|52}}。

1894年初，孫在鄉間用10多天時間草擬《[[s:上李鴻章書|上李鴻章萬言書]]》，主張「富強之大經，治國之大本」在於「人能盡其才，地能盡其利，物能盡其用，貨能暢其流」。<ref name="非常言">孫中山著、葉匡政編：《孫中山非常言：論道1896－1925》，北京：人民日報出版社，2011年4月第1版</ref>{{rp|206}}6月中，棄醫，偕[[陸皓東|陸皓東]]至[[天津|天津]]上書[[李鴻章|李鴻章]]。<ref name="非常言"/>{{rp|206}}旋遊歷北京、武漢，觀察形勢。<ref name="非常言"/>{{rp|206}}10月，孫自上海經日本赴[[檀香山|檀香山]]。<ref name="非常言"/>{{rp|206}}

1894年11月24日，孫在檀香山[[歐胡島|歐胡島]]募款<ref>[[羅家倫|羅家倫]]：〈興中會成立日期之史的考訂》，刊《中山先生行誼》下冊，臺北：[[臺灣書局|臺灣書局]]，1995年10月初版</ref>，創立[[興中會|興中會]]<ref name="非常言"/>{{rp|206}}，提出「振興中華」口號和「驱除[[鞑虏|鞑虏]]，恢复中华，创立合眾政府」<ref>後來在香港，誓詞於1895年改為「驅逐韃虜，恢復中華，創立合眾政府。」</ref>。永和泰商号[[經理|經理]][[劉祥_(興中會)|劉祥]]、卑涉銀行（Bank of Bishop and Co.,Ltd.）[[華人|華人]]經理[[何寬_(興中會)|何寬]]為首任正副主席，然不久，劉祥便退出興中會。

== 筹备 ==
1894年冬，孙中山携带[[邓荫南|邓荫南]]和[[孙眉|孙眉]]等人资助的6千余元美金启程回国并筹备起义，少数[[兴中会|兴中会]]工人会员及欧美技师、将帅与其一同回国。1895年1月，孫偕[[鄧蔭南|鄧蔭南]]等[[同志|同志]]返回香港籌謀革命。<ref name="非常言"/>{{rp|206}}2月18日，孫召集舊友[[陸皓東|陸皓東]]、鄭士良、陳少白、楊鶴齡等，討論籌備“香港興中會總會”。時[[楊衢雲|楊衢雲]]、[[謝纘泰|謝纘泰]]等先以「開通民智、改造中國」為宗旨創立「[[輔仁文社|輔仁文社]]」，孫以志業相近，與其接洽同操大業，然兩派人馬在領導權上相持不下。2月21日，興中會總會於香港「乾亨行」（[[中環|中環]][[士丹頓街|士丹頓街]]13號，外懸商號招牌作掩護）。<ref name="非常言"/>{{rp|206}}各種分工活動即行展開，孫主持前方發難，楊主持後方支援。<ref name=rmw>{{cite web|url=http://history.people.com.cn/GB/198819/212502/14922104.html|title=孙中山由体制内改革转向体制外革命|publisher=人民网-文史频道|author=杨天石|date=2011-06-16|accessdate=2012-09-22}}</ref>

为了获得支持及国际势力的承认，孙中山在起义前试图与日本取得联络。1895年，在英国人[[詹姆斯·康德黎|詹姆斯·康德黎]]博士的介绍下，孙中山结识了日本人[[梅屋庄吉|梅屋庄吉]]。在与梅屋庄吉的谈话中，孙中山与其讨论了在广州发动起义的计划并提到：“欧美各国人都称中国为睡狮。如果是狮子，要醒起来才有用。”“现在的情况如果继续下去，中国就会被西欧列强殖民主义者所瓜分。不独是中国，所有亚洲各国都将成为西欧的奴隶。中日两国不幸发生战争，但我们非团结起来不可，使中国脱离殖民化的危险，是保卫亚洲的第一步。为了拯救中国，我与同志们正准备发动革命，打倒清朝。”对此，梅屋庄吉表示乐于支持并说道：“君若举兵，我以财政相助。”不久，梅屋庄吉为孙中山筹措到了资金，并派人前往[[澳门|澳门]]、[[新加坡|新加坡]]、[[厦门|厦门]]等地购买武器<ref name=rmw/>。在其联络下<ref>{{cite web |url=http://www.qstheory.cn/jj/tslj/201204/t20120405_149499.htm |title=日本财团与辛亥革命 |publisher=环球财经 |author=白益民 王宇 |date=2012-04-05 |accessdate=2012-09-22 |archive-url=https://web.archive.org/web/20121015205825/http://www.qstheory.cn/jj/tslj/201204/t20120405_149499.htm |archive-date=2012-10-15 |dead-url=yes }}</ref><ref>{{cite web |url=http://www.takung.cn/fk/content/2012-07/09/content_668115_2.htm |title=梅屋庄吉是孙中山的“幕后金主”？ |publisher=大公网 |work=《看历史》 |author= |date=2012-07-09 |accessdate=2012-09-22 }}{{Dead link|date=2018年10月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，3月1日，孙中山前往[[日本驻香港领事馆|日本驻香港领事馆]]，希望领事[[中川恒次郎|中川恒次郎]]帮助兴中会准备武器，以解决当时资金短缺、缺少武器等短板。但是中川恒次郎在仔细听取了孙中山的计划后，并不看好其前景 <ref>{{cite web |url=http://book.qq.com/s/book/0/16/16771/31.shtml |title=近代史上最混乱的30年：最后的皇权 第31节：四 小试牛刀，失败了(1) |author=李涛 |publisher=安徽教育出版社 |date= |accessdate=2012-09-22 |deadurl=yes |archiveurl=https://archive.is/20130427174308/http://book.qq.com/s/book/0/16/16771/31.shtml |archivedate=2013-04-27 }}</ref>。

3月4日，中川恒次郎在寄给[[日本外务省通商局|日本外务省通商局]]局长[[原敬|原敬]]的信中提到：“本月1日，经朋友介绍，清国人孙文（西医）来馆。该人如前日所报，正是欲颠覆现政府。他与晚（中川恒次郎的谦称）大致同龄，懂英语，可能是耶稣教徒。据称他打算于去年北洋舰队大演习后，立即举事，不慎失去机会。然而时至今日，特别是在广东省，徒党受到严重注视，不易举事。而更重要的是当前缺乏武器，而需要步枪二万五千枝，手枪一千枝，欲求为之筹措。晚答以本人职务只注意通商贸易之事，与政治之事完全无关，故甚困难。然足下等人的意图可嘉，衷心襄赞。首先想知道足下等人的目的、方法。该人答曰：其党称为兴中会，即振兴中国之会。其中有哥老会员，党员人数难以明言。原因是一有行动立刻就要被发觉，而当最后决定起事时则不能彼此来往通信。然而一旦发难，必定四方响应。然而，当问成功后谁为总统时，答曰尚未及考虑。如允诺给与前述武器，当即往各处招募党员。”<ref name=zzxs>{{cite web|url=http://www.zsnews.cn/zt/zszx2012/2005/06/07/505250.shtml |title=二、在香港谋求外交上的支持  |author=葛培林   |publisher= 中国人民政治协商会议中山市委员会 |date=2005-06-07 |accessdate=2012-09-22 }}</ref>4月17日，中川恒次郎在信中称“孙文仍时时来馆，提出务欲我国予以声援，但从来与我方并无关系，且不说内部事宜，亦不谈党员人数，又无勃兴之准备。孙文说已制定由码头附近运入武器的计划，只要我国给予一些声援，即可充分举事。尊强盗为英雄豪杰等事，皆恰有阅读《水浒传》时的感觉。总之，孙文等所说的要在两广独立成立共和国，只不过是空中楼阁而已。” <ref name=zzxs/>

興中會總會在香港成立后，于3月16日召开了首次幹部會議，決定先攻取廣州，並採用陸皓東設計之[[青天白日旗|青天白日旗]]為起義軍旗。<ref name="非常言"/>{{rp|206}}3月16日前后，孙中山与[[德国驻香港领事|德国驻香港领事]][[克纳普|克纳普]]会面 <ref name=zzxs/>。

== 经过 ==
孫中山抵廣州後，創農學會為機關，並廣徵同志，定[[重陽節|重陽節]]（10月26日）為起義之日。1895年農曆九月初九，孫率領鄭士良、陸皓東等興中會會友，準備[[乙未广州起义|乙未广州起义]]，以事機洩密，接濟未至而失敗。<ref name="非常言"/>{{rp|206}}孫於汕頭及西江沿岸募集兩軍，同時向廣州進逼。<ref name="孫中山全集"/>{{rp|53}}兩軍期於1895年10月某日，一由西南，一由東北，同時向廣州進發。<ref name="孫中山全集"/>{{rp|53}}不料會員部署略定，忽有密電馳至，謂西南、東北兩軍中途被阻。<ref name="孫中山全集"/>{{rp|54}}兩軍既不得進，則应援之勢已孤，即起事之謀已敗。<ref name="孫中山全集"/>{{rp|54}}

==结果==
陸皓東等為首的起義多數成員被清廷逮捕。11月7日，陸皓東、邱四、朱貴全被處死，孫則受到通緝。<ref name="非常言"/>{{rp|206}}當月，孫斷然改裝偕陳少白等赴[[日本|日本]]，旋組織興中會分會於[[橫濱|橫濱]]。<ref name="非常言"/>{{rp|206}}日。[[英屬香港|港府]]也在清政府壓迫下頒發放逐令，不准孫進入香港，為期五年。<ref name="孫中山全集"/>{{rp|53}}。
