'''乙先'''（{{bd|1959年|8月||}}）[[山西|山西]][[平遥|平遥]]人。[[中国国际友好联络会|中国国际友好联络会]]副会长。

==生平==
1978年至1982年，乙先在[[山西大学|山西大学]]哲学系学习，获得[[马克思主义哲学|马克思主义哲学]]学士。1982年至2006年，在[[北京市教育委员会|北京市教育委员会]]任职，其间1994年至1996年在[[中共中央党校|中共中央党校]]在职研究生学习。2006年至2015年，历任中国人民政协理论研究会研究员、中国人民政协理论研究会副主任。2015年5月，出任[[中国国际友好联络会|中国国际友好联络会]]第四届副会长<ref name=caifc>{{Cite web |url = http://www.caifc.org.cn/content.aspx?id=511 |title = 乙先 |author =  |date = 2016-05-31 |publisher = 中国国际友好联络会 |language =  |accessdate =  |quote =  }}</ref>。2016年5月，连任为中国国际友好联络会第五届副会长（同时担任该会的[[法定代表人|法定代表人]]），在四位副会长（乙先、[[邓榕|邓榕]]、[[辛旗|辛旗]]、[[程国平|程国平]]）中排名第一<ref name=caifc/><ref>{{Cite web |url = http://www.caifc.org.cn/content.aspx?id=4229 |title =友联会召开第五届会员代表大会暨理事会第一次会议 |author =  |date =2016-06-02  |publisher =中国国际友好联络会  |language =  |accessdate =  |quote =  }}</ref>。2018年1月，当选[[中国人民政治协商会议第十三届全国委员会委员名单|第十三届全国政协委员]]<ref>{{Cite web|url=http://sports.163.com/18/0126/06/D92BRKK400051C89.html|title=中国人民政治协商会议第十三届全国委员会委员名单|accessdate=2018-01-26|last=|work=网易}}</ref>。
