'''冀宝斋博物馆'''，位于[[中華人民共和國|中華人民共和國]][[河北省|河北省]][[衡水市|衡水市]][[冀州市|冀州市]]湖滨西路，是河北省一家以古陶瓷为主的大型综合性民营博物馆。其馆长是在[[二铺村|二铺村]]任职五十余年的村支书王宗泉，总顾问是原冀州镇党委副书记魏英俊<ref name=yfx/>。2012年7月，冀州冀宝斋博物馆景区被评为[[国家3A级旅游景区|国家3A级旅游景区]]。<ref>{{cite web|title=关于批准发布廊坊香河泽普生态园等12家景区为国家3A级旅游景区公告的决定|url=http://www.hebeitour.gov.cn/article.php?action=one&ai=727|work=河北旅游网|publisher=河北省旅游局|accessdate=2013-07-09|archive-url=https://web.archive.org/web/20150924025455/http://www.hebeitour.gov.cn/article.php?action=one&ai=727#|archive-date=2015-09-24|dead-url=yes}}</ref>此外，博物馆还被评为衡水市爱国主义教育基地、河北省科普基地，也是衡水市“十馆一中心一剧院”重点项目之一。<ref>{{cite news|title=河北1家博物馆藏品年代穿越 仍成省级科普基地|url=http://news.cn.yahoo.com/ypen/20130709/1787280.html|accessdate=2013-07-09|newspaper=南方都市报|date=2013-07-09|deadurl=yes|archiveurl=https://web.archive.org/web/20130808222837/http://news.cn.yahoo.com/ypen/20130709/1787280.html|archivedate=2013-08-08}}</ref>

冀宝斋博物馆的藏品真伪一直存在争议。2013年7月，网络作家[[马伯庸|马伯庸]]发表博文《少年Ma的奇幻历史漂流之旅》，揭露某博物馆展品造型荒谬，文字说明和年代明显有悖历史常识，一时成为网络热点。<ref>{{cite web| url=http://epaper.oeeee.com/A/html/2013-07/09/content_1891298.htm| title=博物馆藏品雷人 还是省级科普基地| publisher=南方都市报| accessdate=2013-07-09| deadurl=yes| archiveurl=https://web.archive.org/web/20130713075824/http://epaper.oeeee.com/A/html/2013-07/09/content_1891298.htm| archivedate=2013-07-13}}</ref>收藏家[[马未都|马未都]]曾在博客中表示冀宝斋博物馆的藏品不真。《[[北京青年报|北京青年报]]》也曾报道冀宝斋博物馆藏品的真伪争议。面对外界的质疑，馆方一直坚持称其藏品都是真品。事发之后，河北省文物局与冀州市展开调查，在7月13日公布了调查处理结果，冀宝斋博物馆被处停馆整顿，冀州市民政局吊销其注册证件。报告还要求冀州市纪委、监察局牵头设立调查组调查相关责任人。<ref>{{cite news|title=河北省文物局:无必要判断冀宝斋藏品真假|url=http://news.sina.com.cn/c/2013-07-14/232227666637.shtml|accessdate=2013-07-16|newspaper=河北青年报|date=2013-07-14}}</ref>

2013年10月28日凌晨约7时冀宝斋博物馆馆长王宗泉在家中因心脏病突发去世，享年68岁。<ref name=yfx>{{cite news|author1=易方兴|title=河北冀宝斋博物馆馆长王宗泉去世|url=http://epaper.bjnews.com.cn/html/2013-10/29/content_474132.htm|accessdate=2013-10-30|agency=新京报|issue=2013年10月29日|publisher=新京报|date=2013年10月29日|page=第A18版}}</ref>

== 历史 ==
冀宝斋博物馆创始于1980年代，产权上归属于[[河北省|河北省]][[冀州市|冀州市]]二铺村集体所有，而馆长是在二铺村任职五十余年的村支书、自称农民收藏家的[[王宗泉|王宗泉]]。<ref name=bq>{{cite news|title=河北乡村花五千万建绝世博物馆|url=http://bjyouth.ynet.com/3.1/1208/06/7351505.html|accessdate=2013-07-09|newspaper=北青网|date=2012-08-06|deadurl=yes|archiveurl=https://web.archive.org/web/20121028084734/http://bjyouth.ynet.com/3.1/1208/06/7351505.html|archivedate=2012-10-28}}</ref>2007年开始，占地60亩，主体建筑面积14000平方米，投资5400万元的新馆开始兴建。2007年11月，冀宝斋博物馆新馆奠基。2010年6月馆舍竣工并布展。2010年7月21日，河北省冀州市冀宝斋博物馆新馆落成并举行了开馆庆典，正式开馆迎客。<ref>{{cite web | url=http://news.xinhuanet.com/collection/2011-08/18/c_121878085.htm | title=冀宝斋博物馆 | publisher=新华网 | accessdate=2013-07-09}}</ref>冀宝斋博物馆陆续被评为国家3A级旅游景区，衡水市爱国主义教育基地，河北省第三批省级科普基地。

2013年7月，因作家[[马伯庸|马伯庸]]发表了一篇博文《少年Ma的奇幻历史漂流之旅》揭露了某博物馆中展示的藏品皆为年代穿越，造型不合常识的赝品，从而冀宝斋博物馆成为网络热点。媒体采访中，[[冀州市旅游局|冀州市旅游局]]表示他们只负责旅游安全和游客投诉处理，并不对博物馆的藏品真伪负责。博物馆副馆长邵宝明对记者说，马伯庸只是个作家，不是收藏家，没有资格来质疑博物馆藏品的真伪；鉴定藏品真伪时，专家可能都无法形成统一意见，“反正我们认为我们的藏品都是真的，也不需要专业机构来鉴定”。<ref name=qp>{{cite news|title=冀宝斋民间博物馆完爆故宫 藏品呆萌年代逆天|url=http://henan.163.com/13/0709/16/93BRHC2S022702D1.html|accessdate=2013-07-09|newspaper=华西都市报|date=2013-07-09}}</ref>[[国家文物局|国家文物局]]相关人员在接受人民网采访时表示，“这是一家民办博物馆，并没在相关部门进行注册认证。”同时根据“法无禁止”的原则，无法对其管理。<ref>{{cite web | url=http://news.xinhuanet.com/yzyd/culture/20130709/c_116466622.htm | title=雷人博物馆冀宝斋遭质疑 国家文物局：没经认证 | publisher=人民网 | date=2013-07-09}}</ref>

2013年7月9日，[[河北省文物局|河北省文物局]]博物馆处处长李宝才接受采访时表示，该博物馆未经任何申报程序，河北省文物局已责令当地相关部门调查。<ref>{{cite web | url=http://news.xinhuanet.com/local/2013-07/10/c_116478994.htm| title=河北省文物局责令当地调查冀宝斋博物馆奇葩藏品事件| publisher=新华网| date=2013-07-10}}</ref>7月13日，“冀州市冀宝斋博物馆问题联合调查组”公布了调查处理结果，冀宝斋博物馆被处停馆整顿，冀州市民政局吊销其注册证件。报告还要求冀州市纪委、监察局牵头设立调查组调查相关责任人，冀州市文化系统开展学习、检查活动，杜绝类似情况再次发生。<ref>{{cite news|title=河北文物局公布冀宝斋处理结果：撤销注册证 闭馆整顿|url=http://news.ifeng.com/mainland/detail_2013_07/15/27492263_0.shtml?_from_ralated|accessdate=2013-07-16|newspaper=人民网|date=2013-07-15}}</ref>

== 藏品 ==
冀宝斋博物馆以收藏元、明、清三朝官窑瓷器为主，并号称藏有大量国宝，馆长王宗泉称其馆藏足以让[[故宫博物院|故宫博物院]]无地自容。2012年8月，《[[北京青年报|北京青年报]]》报道了冀宝斋博物馆。文中提到冀宝斋博物馆藏有[[尧|尧]][[舜|舜]]时代的彩瓷、[[商朝|商朝]]青花瓶、[[秦始皇|秦始皇]]统一六国御赐款[[五彩|五彩]]瓷瓶、直径1.7米的[[元代|元代]][[釉里红|釉里红]]大盘等，“哪一件都能颠覆中国瓷器发展史”。博物馆还藏有一对[[长信宫灯|长信宫灯]]，一男一女，比[[河北省博物馆|河北省博物馆]]所藏[[满城汉墓|满城汉墓]]出土的长信宫灯还大。新闻提到，[[宣德炉|宣德炉]]在这家博物馆都算不上精品，王宗泉把三个宣德炉用作烟灰缸。王宗泉称，藏品都是老百姓“慕名送来”后博物馆收购的，每一件藏品都经过了他的鉴定，都是真品。新闻还说，有收藏界人士评论王宗泉“是当之无愧的全国十大古玩赝品收藏家，冀宝斋就是个低仿假货集中营”。而王宗泉对外界的质疑，回应称“专家没见过的东西就是假的”是不对的，“中华五千年的文明史，地大物博，你没见过的东西多了！”<ref name=bq/>此后不久，收藏家[[马未都|马未都]]也在博客上发文评论这家博物馆，认为其藏品不真。<ref name=qp/>

2013年7月，作家[[马伯庸|马伯庸]]在[[新浪博客|新浪博客]]上发表博文《少年Ma的奇幻历史漂流之旅》，讲述了他参观某博物馆的经历，并配发大量图片。马伯庸在文章中称，该博物馆展出的藏品包括大量赝品，造假手段非常离谱，包括[[汉代|汉代]]的[[五彩|五彩]]和[[青花|青花]]（实际上，五彩创烧于元代后期，青花的创烧年代有唐代和宋代两说，但现存都是残片，且与景德镇[[元青花|元青花]]没有明确的传承关系<ref name=mtc>{{cite book|author=马未都|title=马未都说陶瓷|year=2008|publisher=中华书局|location=北京|isbn=978-7-101-06201-4}}</ref>）、晋代[[斗彩|斗彩]]三英战[[赵云|赵云]]葵口盘（斗彩创烧于[[明代|明代]][[成化|成化]]年间<ref name=mtc/>）、多件[[鬼谷下山|鬼谷下山]]罐、写有[[白话文|白话文]]的[[唐代|唐代]]御赐五彩葵口盘、“[[炎帝|炎帝]]制造”青花瓷器。藏品中有一件题款“大清雍正年制”的“粉彩十二开光[[金陵十二钗|金陵十二钗]]大缸”，然而[[雍正皇帝|雍正皇帝]]逝世時[[曹雪芹|曹雪芹]]年仅21岁，因此其有违历史常识。<ref>{{cite web | url=http://opinion.hexun.com/2013-07-09/155956739.html | title=冀宝斋民间博物馆超越时代 网友称想什么就有什么 | publisher=和讯网 | accessdate=2013-07-09}}</ref>除此之外，在博物馆内还有诸多藏品的线条粗陋、笔法敷衍。同时，历史仅存孤件的著名文物如[[长信宫灯|长信宫灯]]、元青花鬼谷子下山图罐，在该馆中也有多件做工粗陋的放大版赝品，而标识均显示为真品。此外，该馆中有多套的十二生肖主题的瓷俑，不仅造型粗陋、诡异，且有性别区分。

但[[中国收藏家协会|中国收藏家协会]][[中国收藏家协会玉器收藏委员会|玉器收藏委员会]]主办的[[中藏网|中藏网]]持相反意见。中藏网认为，斋博物馆馆长王宗泉千辛万苦组织村集体资本投资了5000万，来兴建冀宝斋博物馆，完全符合党的社会主义文化大发展精神，而马伯庸是怀着不可告人目的，极尽讽刺、挖苦和无知的语言，恶意诽谤中伤冀宝斋博物馆。中藏网又称，马伯庸的诽谤引起全国收藏家的强烈愤慨，“绝不允许诋毁王宗泉等一大批中国收藏家为中华民族千秋伟业作出的卓越贡献”，同时还指责河北省文物局局长不深入调查研究，站错立场出来帮腔，诽谤诬蔑冀宝斋博物馆，应追究责任并下台。<ref>{{cite web | url=http://art.people.com.cn/n/2013/0715/c226026-22195940.html | title=中藏网指马伯庸怀不可告人目的诽谤冀宝斋 | publisher=人民网 | date=2013-07-15}}</ref>对于中藏网的说法，不少中国网民认为文章“搞笑”、“再次被雷倒”、“瞧不起老百姓智商”。<ref>{{cite web | url=http://cul.sohu.com/20130715/n381597835.shtml | title=中藏网声援山寨博物馆 网友:瞧不起老百姓智商 | publisher=搜狐文化新闻 | date=2013-07-15}}</ref>
