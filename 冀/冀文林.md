'''冀文林'''（{{bd|1966年|7月|catIdx=J}}），[[内蒙古自治区|内蒙古自治区]][[凉城县|凉城县]]人，[[中华人民共和国|中华人民共和国]]官员，官至[[副部级|副部级]]。曾在[[中国地质大学|中国地质大学]]就读，获[[地球物理|地球物理]]专业硕士学位。1989年7月正式参加工作，1994年8月加入[[中国共产党|中国共产党]]。冀文林曾长期担任[[周永康|周永康]]的[[秘书|秘书]]，被认为是周永康的心腹<ref>{{cite web |url = http://news.nanyangpost.com/2014/02/Ji-Wenlin.html |title = 周永康心腹 海南副省长冀文林被拘 |publisher = [[南洋视界|南洋视界]] |date = 2014年2月19日}}</ref>。

== 家庭 ==
冀文林于1966年7月出生于凉城县[[永兴镇|永兴镇]]西驼厂村的一个贫寒农家，有两姐姐、一个哥哥和一个弟弟，在家中排老四。

其妻是他在武汉地质学院地质系就读时认识的学妹，[[吉林市|吉林市]]人，两人育有一女。<ref >{{cite web|url=http://news.163.com/14/0704/05/A09LG3Q50001124J_2.html | language= | title=冀文林出身贫寒 两个姐姐为供其读书不敢出嫁 |publisher= 北青网-北京青年报 |date=2014-07-04 |accessdate= }}</ref>

== 履历 ==
=== 地质矿产部时期 ===
1989年7月，进入[[中华人民共和国地质矿产部|中华人民共和国地质矿产部]]，正式[[参加工作|参加工作]]；历任地矿部[[办公厅|办公厅]][[科员|科员]]、副主任科员、主任科员。

1994年8月，加入[[中国共产党|中国共产党]]。

1996年12月起，任地质矿产部部长办公室副主任。

=== 国土资源部部长周永康的秘书 ===
1998年8月起，历任[[中华人民共和国国土资源部|国土资源部]]办公厅值班室助理调研员、[[调研员|调研员]]、部长（周永康）[[秘书|秘书]]。<ref>{{cite news |title = 海南副省长冀文林被查 消息称涉中石油项目问题 |url = http://news.163.com/14/0219/02/9LDO3U6E00014AED.html |accessdate = 2014-02-20 |newspaper = 网易（转自[[新京报|新京报]]）}}</ref>

=== 四川省委书记周永康的秘书 ===
1999年12月起，历任[[中共四川省委|中共四川省委]]常委办公室副主任、[[省委办公厅|省委办公厅]]副厅级秘书；2001年9月至2003年7月，[[四川省工商管理学院|四川省工商管理学院]][[MBA|MBA]][[在职研究生|在职研究生]]学历教育班学习。

=== 国务委员兼公安部部长周永康的秘书 ===
2003年3月[[第十届全国人大第一次会议|第十届全国人大第一次会议]]上，周永康被任命为[[国务委员|国务委员]]，继续兼任[[中华人民共和国公安部|公安部]]部长。

2003年4月起，冀文林任公安部[[办公厅|办公厅]]副主任、[[国务院办公厅|国务院办公厅]]副局级秘书。

2005年9月，冀文林被授予[[二级警监|二级警监]][[警衔|警衔]]。2005年9月起，任[[中共中央办公厅|中共中央办公厅]]正局级秘书。

=== 中央政治局常委兼中央政法委书记周永康的秘书 ===
2007年10月[[中共十七届一中全会|中共十七届一中全会]]上，周永康晋升[[中共中央政治局常委|中共中央政治局常委]]、[[中央政法委|中央政法委]]书记、[[中央综治委|中央综治委]]主任，接替罗干全面负责中共政法工作。

2008年7月，冀文林兼任[[中央维护稳定工作领导小组办公室|中央维护稳定工作领导小组办公室]]副主任。

=== 国土资源部办公厅主任 ===
2008年12月起，任[[国土资源部|国土资源部]]办公厅主任；2009年1月起，为中共国土资源部党组成员<ref name="简历" />。

=== 海南时期 ===
2011年2月16日起，任[[海口市人民政府|海口市人民政府]][[海口市市长|市长]]<ref name="简历">{{cite web |url = http://www.mlr.gov.cn/zwgk/ldzc/jwl/ |language = 中文 |title = 冀文林 |publisher = [[中华人民共和国国土资源部|中华人民共和国国土资源部]] |date =  |accessdate = 2011年10月31日 |archive-url = https://web.archive.org/web/20120508222736/http://www.mlr.gov.cn/zwgk/ldzc/jwl/ |archive-date = 2012-05-08 |dead-url = yes }}</ref>。

2013年1月起，任[[海南省人民政府|海南省人民政府]]副省长。

=== 被查 ===
{{See also|周永康案}}

2014年2月18日，[[中纪委|中纪委]]披露，冀文林因涉嫌严重违纪违法，被[[中国共产党纪律检查机关|纪检]][[中华人民共和国行政监察机关|监察]]机关立案调查<ref>{{Cite web |url = http://www.ccdi.gov.cn/xwyw/201402/t20140218_18697.html |title = 海南省副省长冀文林涉嫌严重违纪违法被立案调查
|publisher = 中央纪委监察部网站 |date = 2014-02-18}}</ref><ref>{{cite web |url = http://news.163.com/14/0218/18/9LCTUHSF00014JB5.html |language = |title = 海南副省长冀文林涉嫌严重违纪违法被调查 |publisher = 网易 |work = 中央纪委监察部网站 |date = 2014-02-18 |accessdate = }}</ref><ref>[http://www.apdnews.com/news/69751.html 海南省副省长冀文林涉嫌严重违纪违法] {{webarchive|url=https://web.archive.org/web/20140220071500/http://www.apdnews.com/news/69751.html |date=2014-02-20 }}，[[亚太日报|亚太日报]]，2014年2月19日</ref>。同年7月2日被双开并移送司法机关处理，经查冀文林利用职务上的便利为他人谋取利益，收受、索取巨额贿赂；与他人通奸。<ref >{{cite web|url=http://news.xinhuanet.com/politics/2014-07/02/c_1111423913.htm | language= | title=海南省原副省长冀文林严重违纪违法被开除党籍和公职 |publisher=新华网 |date=2014年7月2日 |accessdate= }}</ref>

2016年3月30日，[[天津市第一中级人民法院|天津市第一中级人民法院]]一审宣判，冀文林犯受贿罪，判处有期徒刑12年，并处罚金人民币100万元<ref >{{cite web|url=http://news.sohu.com/20160330/n442804806.shtml | language= | title=海南省原副省长冀文林受贿案一审宣判 获刑12年 |publisher=新华网 |date=2016年3月30日 |accessdate= }}</ref>。
