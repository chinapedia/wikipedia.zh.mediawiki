'''后晋'''（936年-947年）是[[中国历史|中国历史]]上[[五代十国|五代十国]]时期的一个朝代，从后晋高祖[[石敬瑭|石敬瑭]]936年灭[[后唐|后唐]]开国到[[辽国|契丹]]947年灭后晋一共经历了两个皇帝，總計12年。为与司马氏的[[晋朝|晋朝]]相区别，又别称为'''石晋'''。

== 建国 ==

后晋的开国皇帝[[沙陀人|沙陀人]]石敬瑭是后唐开国的功臣，他曾经多次在危难中救护后唐开国皇帝[[李存勖|李存勖]]和明宗[[李嗣源|李嗣源]]。李存勖和李嗣源都十分器重他，李嗣源甚至将自己的女儿嫁给了他。后唐建立后石敬瑭任河东节度使（今[[山西|山西]]），石敬瑭成为当地军民最高指挥官。石敬瑭在河东政绩很高，而且生活清廉，很受当地人的欢迎。但李嗣源死后后唐内部互相倾轧，石敬瑭受[[李从珂|李从珂]]的猜忌，因此渐渐产生了反唐的计划。当李从珂决定将石敬瑭调离河东时，石敬瑭决定反唐。

石敬瑭在河东的兵力不足以抵挡后唐的进攻，因此石敬瑭决定求救于[[契丹|契丹]]。作为条件，他同意割让燕雲十六州（此十六个州，属今[[河北|河北]]和山西）给契丹，并对[[辽太宗|辽太宗]]耶律德光称「儿」。在这种情况下，耶律德光决定帮助石敬瑭。

契丹和石敬瑭的联军打败了后唐，攻入后唐首都[[洛阳|洛阳]]。后唐灭亡，石敬瑭称帝，国号'''晋'''，史称后晋。依據[[五行|五行]]相生的順序，[[后唐|後唐]]的「土」德之後為「金」德，因此後晉以「金」為王朝德運。<ref>{{Cite journal|title=陳元-臺北故宮藏宋元明帝王畫像與其隱喻的王朝正統性, Zhongguo wenhua 中國文化 [Chinese Culture] 44 (2016): 137-53|url=https://www.academia.edu/29963411/%E9%99%B3%E5%85%83-%E8%87%BA%E5%8C%97%E6%95%85%E5%AE%AE%E8%97%8F%E5%AE%8B%E5%85%83%E6%98%8E%E5%B8%9D%E7%8E%8B%E7%95%AB%E5%83%8F%E8%88%87%E5%85%B6%E9%9A%B1%E5%96%BB%E7%9A%84%E7%8E%8B%E6%9C%9D%E6%AD%A3%E7%B5%B1%E6%80%A7_Zhongguo_wenhua_%E4%B8%AD%E5%9C%8B%E6%96%87%E5%8C%96_Chinese_Culture_44_2016_137-53|last=Chen|first=Yuan Julian|language=en}}</ref>後晉移都[[开封|开封]]，并按约将16州让给契丹。这16州是：幽（今[[北京|北京]]市）、蓟（今[[天津|天津]][[蓟县|蓟县]]）、瀛（今河北[[河间市|河间]]）、莫（今河北[[任丘|任丘]]）、涿（今河北[[涿州|涿州]]）、檀（今北京[[密云|密云]]）、顺（今北京[[顺义|顺义]]）、新（今河北[[涿鹿|涿鹿]]）、妫（原属河北[[怀来|怀来]]，今为[[官厅水库|官厅水库]]库区）、儒（今北京[[延庆县|延庆]]）、武（今河北[[宣化区|宣化]]）、蔚（今河北[[蔚县|蔚县]]）、云（今山西[[大同市|大同]]）、应（今山西[[应县|应县]]）、寰（今山西[[朔县|朔县]]东马邑镇）、朔（今山西朔县），并向契丹称[[儿皇帝|儿皇帝]]，契丹封其为“晋帝”。

石敬瑭割让[[燕雲十六州|燕雲十六州]]为辽国和[[金国|金国]]后来对[[宋朝|宋朝]][[长江|长江]]以北地区的威胁打开了门户。

== 建国后的动乱 ==
后晋建国后一直处于动盪中，石敬瑭割地称儿的做法受到许多人的反对，包括他自己过去的亲信。石敬瑭本人到死没有改变依附契丹的政策，但国家多處发生叛乱，石敬瑭的两个儿子在这些叛乱中被杀，種種事情给他带来了极大的打击。为了对付叛乱，石敬瑭加重嚴刑峻法，同时非常猜忌自己的手下。

== 石重贵反辽亡国 ==
石敬瑭死时，立[[石重贵|石重贵]]为继承人。石重贵是他的侄子，因为在战场上立战功获得石敬瑭的赏识。但石重贵仅是一勇之夫，根本无法在國家面對困境下應付各種政治问题。石重贵登基后决定渐渐脱离对契丹的依附，他首先宣称对耶律德光称'''孙'''，但不称'''臣'''。

契丹对此当然不能坐视。944年[[契丹滅後晉之戰|契丹伐晋]]，双方在澶州（今[[河南|河南]][[濮阳|濮阳]]南）交战，互有胜负。945年契丹再次南征，石重贵亲征，再次战败契丹。947年，契丹第三次南下，后晋重臣[[杜重威|杜重威]]、[[李守貞|李守貞]]和[[張彥澤|張彥澤]]率軍向契丹投降，后晋丧失主力，契丹派張彥澤率先部入開封。石重贵被迫投降，全家被俘虏到[[契丹|契丹]]。后晋灭亡。

后晋亡后，河东[[节度使|节度使]]北平王[[刘知远|刘知远]]在[[太原|太原]]称帝，建立[[后汉|后汉]]。
