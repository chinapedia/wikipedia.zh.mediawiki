'''捷克高速鐵路'''為一項[[捷克共和國|捷克共和國]]國內當前正在計畫中的[[高速鐵路|高速鐵路]]建設案。目前捷克國內的國營鐵路公司[[捷克鐵路|捷克鐵路]]旗下已有多輛行車時速能夠到達200公里的列車[[鐵路機車|機車頭]]，然而在{{en-link|Velim 鐵路測試線|Velim railway test circuit}}中認定，捷克國內的鐵路路廊與基礎設施其行車時速無法超過160公里。

捷克國內的列車機車頭製造商[[斯柯達公司|斯柯達公司]]（Škoda）會定期生產高速鐵路用的機車頭提供給捷克鐵路公司。

== 基礎建設 ==
[[File:Alstom_AGV_Cerhenice_img_0365.jpg|thumb]]]]
捷克共和國的交通運輸部目前正在規劃其國內的高速鐵路路網，預計總長將達到660公里。<ref>{{cite web|url=http://www.mdcr.cz/NR/rdonlyres/9032A2C5-C9BC-45B9-827F-B25F97C29F83/0/a_prz.pdf |title=KOORDINAČNÍ STUDIE VRT 2003 |author=Objednatel: Ministerstvo dopravy ČR |date=2004-07-31 |language=cs |deadurl=yes |archiveurl=https://web.archive.org/web/20081010123550/http://www.mdcr.cz/NR/rdonlyres/9032A2C5-C9BC-45B9-827F-B25F97C29F83/0/a_prz.pdf |archivedate=2008-10-10 }}</ref> 其中部分認為可行的高鐵路線已完成初步的可行性評估，然而至今捷克交通運輸部尚未針對高鐵路線建設案提出具體的開發計畫。<ref>{{cite web |url=http://vrt.fd.cvut.cz/data/konference/28p.pdf |title=Studie VRT - analýza přepravních vztahů a výhledové možnosti dopravních systémů ve vybraných směrech |author=Objednatel: Ministerstvo dopravy ČR |date=2006年12月 |language=cs |access-date=2015-08-08 |archive-url=https://web.archive.org/web/20110718170646/http://vrt.fd.cvut.cz/data/konference/28p.pdf |archive-date=2011-07-18 |dead-url=yes }}</ref> 因此在2020年之前，預估將不會有任何的高鐵建設案正式進行，不過捷克的{{en-link|鐵路建設管理局|Správa železniční dopravní cesty}}，針對高鐵建設部分則有籌備進行更進一步研究的特別經費。其餘非政府組織方面也有促進推動高鐵計畫的組織如{{en-link|效率交通中心|Centrum pro Efektivní Dopravu}}<ref>{{cite web |url=http://ekonomika.idnes.cz/rychlovlaky-v-cesku-0ql-/eko-doprava.aspx?c=A130103_205120_eko-doprava_hro |title=Vysokorychlostní vlaky v Česku se vrací do hry. Na plány jde 45 milionů |author=Jan Sůra |date=2013-01-04|language=cs}}</ref>。

鐵路建設管理局當前也計畫讓部分已裝設[[ETCS|ETCS]]系統的軌道進行改善，使其自原本軌道最高能接受的時速{{convert|160|km/h|mph|0|abbr=on}}提升至時速{{convert|200|km/h|mph|0|abbr=on}}，也預定讓部分已正在規劃中原訂時速為{{convert|160|km/h|mph|0|abbr=on}}的鐵路路線修改計畫將最高時速提升至{{convert|200|km/h|mph|0|abbr=on}}。<ref>[http://technet.idnes.cz/rychlovaky-v-cesku-0al-/tec_technika.aspx?c=A130729_151023_tec_technika_rja TechNet: Česká železnice se připravuje na 200 km/h. Hrozí nám „španělská“ nehoda?]</ref>

在{{en-link|Velim试验鐵路線|Velim railway test circuit}}中，包含了傾斜式列車最高時速{{convert|230|km/h|mph|0|abbr=on}}以及普通列車需高達時速{{convert|210|km/h|mph|0|abbr=on}}長達13.3公里的大型试验線。<ref>[http://www.cdvuz.cz/zkusebni-centrum-vuz-velim/ VÚŹ: Zkušební centrum VUZ Velim]</ref>

== 營運在捷克國內的機車 ==
[[File:Brno,_hlavní_nádraží,_jednotka_680.007_(2).jpg|thumb]]
自2004年以來，捷克鐵路公司已投入7組{{en-link|捷克國鐵680型電力機車|ČD Class 680}}機車進行營運服務。這些屬於[[義大利|義大利]][[潘多利諾|潘多利諾]]車系的機車，規劃營運在[[柏林|柏林]] - [[布拉格|布拉格]] - [[維也納|維也納]]之間的鐵路路線上，並且其行車時速為230公里。<ref>[http://aktualne.centrum.cz/domaci/kauzy/clanek.phtml?id=732599 aktuálně.cz: Plán supertratě krachl, železnice přišla o 70 miliard]</ref>，而這型列車曾締造過捷克鐵路新速度紀錄是在2004年11月18日，在[[布熱茨拉夫|布熱茨拉夫]] - [[布爾諾|布爾諾]]之間的鐵道上所進行的行車測試，當時行車時速最高到達237公里<ref>{{cite web|url=http://www.pendolino.cz/clanky/document.2005-02-16.6565302389 |title=Portál Pendolino.cz - 237 km/h: Nový rychlostní rekord české železnice |accessdate=2008-02-10 |author=martin |date=2005-02-17 |deadurl=yes |archiveurl=https://web.archive.org/web/20080104054658/http://www.pendolino.cz/clanky/document.2005-02-16.6565302389 |archivedate=2008-01-04 }}</ref>。不過在現實狀態中，捷克國鐵680型電力機車從未在[[德國|德國]]營運過，僅在[[奧地利|奧地利]]與[[斯洛伐克|斯洛伐克]]進行營運載客。自從2012年以後，捷克國鐵680型電力機車只在國內的鐵路路線上營運而不駛出國外。

從2010年開始，捷克鐵路公司開始接收一批為數20組的新機車{{en-link|捷克國鐵380型電力機車|ČD Class 380}}，這批新機車的行車時速最高為200公里，並能夠拖動10輛[[鐵路客車|客車]]（班次 10-91, 21-91, 72-91 以及 88-91）。2013年，捷克鐵路公司更下訂購買了7輛 [[銳捷列車|銳捷列車]] （Railjet），其行車時速最高達到230公里。

== 由捷克製造的機車 ==
[[File:380.002-6_Škoda_109E_1.jpg|thumb]]
在1974年以及1979年中，斯柯達為[[蘇聯|蘇聯]]生產了12輛的斯柯達66Е型機車的車頭。斯柯達66Е型機車的最高時速為200公里。其在蘇聯被稱作為[[Chs200|Chs200]]（ЧС200），主要用在行駛於[[莫斯科|莫斯科]] - [[聖彼得堡|聖彼得堡]]之間的[[涅夫斯基特快號列車|涅夫斯基特快號列車]]上。在20世紀90年代，這些機車頭進行了翻新工程，並其中一輛翻新完畢的車頭在2007年地測試期間達到時速262公里的速度。<ref>{{cite web|url=http://www.uks.ru/news/loko.jpg |title=UNIVERSAL CATENARY SYSTEMS-news |author= |date= |language=ru |deadurl=bot: unknown |archiveurl=https://web.archive.org/web/20070329043631/http://www.uks.ru/news/loko.jpg |archivedate=2007-03-29 }}</ref>

自2008以後斯柯達開始生產時速200公里的機車頭斯柯達109E型機車。而批量{{en-link|捷克國鐵380型電力機車|ČD Class 380}}的速度認證就以該{{convert|200|km/h|mph|0|abbr=on}}為基準，就像是未來的德鐵102型電力機車，不過，在斯洛伐克國內一批{{en-link|斯洛伐克鐵路公司|Železničná spoločnosť Slovensko}}（ZSSK）旗下的斯洛伐克381型電力動車組其最高時速則只到達160公里。儘管接下來一批雙車組時速為200公里的機車頭德鐵102型電力機車即將交付，但其國內列車行駛速度法定上限為189公里。
