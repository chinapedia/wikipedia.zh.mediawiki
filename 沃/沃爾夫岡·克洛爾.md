'''沃爾夫岡·克洛爾'''（{{lang-de|Wolfgang Kroll}}，{{bd|1906年|3月21日|1992年|2月28日|catIdx=Kroll}}），是一名德裔臺灣物理學家。他出生於[[德國|德國]]北部的[[格賴夫斯瓦爾德|格賴夫斯瓦爾德]]。1930年，克洛爾進入[[布雷斯勞大學|布雷斯勞大學]]就讀，1930年畢業并取得博士學位。隨後幾年，克洛爾於[[萊比錫|萊比錫]]跟隨[[維爾納·海森堡|維爾納·海森堡]]進行博士後研究。

1937年，由於[[納粹德國|納粹]]掌權，克洛爾離開德國而轉往日本[[北海道|北海道]]任教，最後於1941年到達[[台灣日治時期|日治時代]]的[[臺北|臺北]]，並任教於[[臺北帝國大學|臺北帝國大學]]預科。戰後，克洛爾受邀轉任[[國立臺灣大學物理學系|國立臺灣大學物理學系]]，升任為副教授再升為教授，是戰後臺灣第一位擁有理論物理博士學位的專任教授，亦為臺大物理系第一篇在國際科學刊物上發表論文的教師。此後，他也受邀至[[東海大學_(台灣)|東海大學]]物理系等地教課。1976年8月，克洛爾自臺大物理系退休，並於隔年和[[許雲基|許雲基]]一同成為名譽教授。退休後，克洛爾亦曾在[[中國文化學院|中國文化學院]]等處兼課。1970年代後期，由於臺灣經濟開始起飛，導致物價上揚，使得克洛爾的退休金相對而言變得微薄，生活也因此陷入困境。到了1992年2月28日，克洛爾因[[肺氣腫|肺氣腫]]，病逝於[[臺大醫院|臺大醫院]]，享壽87歲。

==生平==
===早年===
[[Image:Universität_in_Breslau.jpg|left]]
沃爾夫岡·克洛爾於1906年3月21日出生於[[德意志帝國|德意志帝國]]北部的[[格賴夫斯瓦爾德|格賴夫斯瓦爾德]]<ref name = "張幸真"/>{{rp|172}}<ref name = "林夏玉"/>。克洛爾小時候曾經歷[[第一次世界大戰|第一次世界大戰]]，當時德國物資缺乏，正巧動物園的大象遭砲擊死亡，克洛爾也因此吃過大象的肉<ref name = "張秀蓉"/>。1920年代末，克洛爾進入[[布雷斯勞大學|布雷斯勞大學]]{{efn|現為波蘭的[[弗羅茨瓦夫大學|弗羅茨瓦夫大學]]。}}，在該校接受理論物理學訓練，並於1930年以研究狄拉克電子理論的論文獲得該校物理博士學位<ref name = "張幸真"/>{{rp|172}}<ref name = "古今論衡"/>{{rp|102}}<ref name = "林夏玉"/><ref name="1930paper"/>。同年，他獲得了{{le|德國科學救助組織|Notgemeinschaft der Deutschen Wissenschaft}}的獎助金，前往[[萊比錫大學|萊比錫大學]]跟隨[[維爾納·海森堡|維爾納·海森堡]]{{efn|1933年[[諾貝爾物理學獎|諾貝爾物理學獎]]得主。}}做了7年的博士後研究<ref name = "張幸真"/>{{rp|172,174}}<ref name = "古今論衡"/>{{rp|102}}<ref name = "許仲平"/>{{rp|830}}。在這段期間，他也結識了日後成為[[氫彈|氫彈]]之父的[[愛德華·泰勒|愛德華·泰勒]]<ref name = "林夏玉"/>。1930-1937年間，克洛爾發表了[[量子力學|量子力學]]、磁化頻譜、[[熱電效應|熱電效應]]等領域的相關論文，總共8篇，並奠定其日後獨立研究之基礎<ref name = "張幸真"/>{{rp|1734}}。在[[纳粹党|納粹]]取得政權後，克洛爾經常批評時政。雖然克洛爾並非[[猶太人|猶太人]]，但由於此舉受當政者所不喜，他便因此身陷危險之中。因此，克洛爾在父親與校方的協助之下，於1937年開始獨自流亡海外，前往日本<ref name = "張幸真"/>{{rp|173}}<ref name = "古今論衡"/>{{rp|103}}<ref name = "Kroll Connection"/>。克洛爾表示，他選擇前往日本的目的，是為了見識「迷人的東方」，而且他和他父親有很多朋友在日本<ref name = "張幸真"/>{{rp|173}}<ref name= "JingShin"/>{{rp|xv}}。

關於克洛爾離開德國的方式，有兩種說法。許仲平認為，克洛爾是以交換研究的名義，與日本的[[朝永振一郎|朝永振一郎]]{{efn|日本第二個諾貝爾物理學獎得主。}}交換<ref name = "許仲平"/>{{rp|830}}<ref name = "張幸真"/>{{rp|173}}<ref name= "JingShin"/>{{rp|xv}}。而亦有另一種說法指出，克洛爾是經由父親安排至日本<ref name = "張幸真"/>{{rp|173}}。克洛爾先前往英國[[倫敦|倫敦]]，並再經[[上海|上海]]轉日本，最後抵達[[北海道|北海道]]<ref name = "張幸真"/>{{rp|173}}<ref name = "古今論衡"/>{{rp|103}}。

===前往日本===
[[File:Hokkaido_Imperial_University.jpg|right]]
到了北海道之後，克洛爾前往[[北海道帝國大學|北海道帝國大學]]任職{{efn|原為1886年設立的[[札幌農學校|札幌農學校]]，1918年改為此名<ref name = "古今論衡"/>{{rp|105}}。}}，並自1937年起在該處擔任五年的講師<ref name = "林夏玉"/><ref name = "張幸真"/>{{rp|175}}<ref name = "古今論衡"/>{{rp|105}}。當時，他與甫自[[京都帝國大學|京都帝國大學]]物理科畢業的朝永振一郎在五年內共同發表了三篇論文<ref name = "張幸真"/>{{rp|175}}，同時也和朝永共同開設[[量子論|量子論]]課程<ref name = "古今論衡"/>{{rp|106}}。另外，克洛爾也於1937年在[[仁科芳雄|仁科芳雄]]的支持之下，獨自發表了有關熱傳導理論的論文<ref name="1937paper"/><ref name = "古今論衡"/>{{rp|126}}，並於1941年和梅田魁與戶谷富之一同發表研究單價金屬中[[自由電子|自由電子]]的行為以及其與比熱之關係的論文<ref name="1941paper"/><ref name = "古今論衡"/>{{rp|126}}。然而，由於北海道帝國大學的薪資並不高，克洛爾便至附近的商校兼任德文教師<ref name = "林夏玉"/>。 爾後，[[第二次世界大戰|第二次世界大戰]]爆發，克洛爾遂離開北海道帝大，轉任[[東京理化研究所|東京理化研究所]]，並在不久後離開日本本土，前往[[臺北帝國大學|臺北帝國大學]]任教<ref name = "張幸真"/>{{rp|175}}。

1941年底，克洛爾抵達臺灣，任教臺北帝國大學。此時，他的薪水也增加了六成<ref name = "林夏玉"/><ref name= "JingShin"/>{{rp|xv}}。然而，當時的講座制度之下並不允許克洛爾在物理學講座任職，而是只能在預科及[[臺北高等學校|臺北高等學校]]教授德文課程。直到戰爭結束前，才開始教授物理<ref name = "張幸真"/>{{rp|176}}。1945年4月20日起，包含克洛爾在內的全體預科師生被動員，編入第13861部隊，並於當月起在[[淡水|淡水]]至[[新莊|新莊]]間從事陣地構築，戰後方解散復學<ref name = "張幸真"/>{{rp|177}}。然而，克洛爾在戰爭期間仍不忘研究，他在1944年發表一篇數學物理論文，並登於《台北帝國大學理學部記要》<ref name = "張幸真"/>{{rp|177}}。

===戰後時期===
[[File:NTU_PhysicsOldBuilding.JPG|left]]
[[台灣戰後時期|戰後]]，[[國民政府接收臺灣|國民政府接收臺灣]]，負責接收臺北帝國大學的[[羅宗洛|羅宗洛]]亦因此成為戰後臺大的首任代理校長，並邀請克洛爾留任臺大，並由講師升任為副教授，調往[[國立臺灣大學物理學系|物理學系]]教授物理學與德語<ref name = "張幸真"/>{{rp|177-178}}。然而，在羅宗洛離任之後，由於克洛爾身為敗戰國國民，因此有了將被降職的傳聞。直到[[傅斯年|傅斯年]]就任臺灣大學校長之後，再度鼓勵其留任，同時為他加薪<ref name = "張幸真"/>{{rp|178}}。傅斯年並邀請克洛爾參加1951年5月24日大一課程設定計畫的討論，而克洛爾則是該次會議中唯一的外籍教師<ref name = "張幸真"/>{{rp|178}}。

1948年，日本的《學藝雜誌》登出了克洛爾有關[[電子|電子]]半徑研究的論文，成為臺大物理系教師第一篇在國際科學刊物上發表的論文<ref name = "張幸真"/>{{rp|179}}。1950年，克洛爾獨力完成了固體[[比熱|比熱]]方面的研究，其研究成果論文則在1952年刊登於日本京都出版的《理論物理的發展》，並於1954年以此升任教授<ref name = "張幸真"/>{{rp|179}}<ref name = "1970phys"/>{{rp|58}}。在這段期間，克洛爾也指導他當時的助教[[黃振麟|黃振麟]]，共同從事固體比熱與頻譜方面的研究，而黃振麟則在1955年完成休斯頓法的修正，利用[[插值法|插值法]]確定等頻譜線圖，避免了原本休斯頓法在高頻端的[[奇點|奇點]]問題。同時他也將該次研究成果寫成論文，發表於該年度的《物理評論》上，成為了臺灣本土學者在該刊物的第一篇論文<ref name="1970phys"/>{{rp|58,66}}<ref name="Montroll"/>{{rp|103}}。

此後，臺大物理系理論物理課程幾乎是由克洛爾負責教學，其講授課程包含了[[理論物理學|理論物理學]]、[[量子力學|量子力學]]、[[相對論|相對論]]、[[統計力學|統計力學]]、物理數學、[[電子學|電子學]]等理論課程<ref name = "張幸真"/>{{rp|178}}<ref name = "林夏玉"/>。他的理論物理課程主要是參考德國物理學家[[阿諾·索末菲|阿諾·索末菲]]所著之教科書，並將其內容整理成筆記作為教材使用<ref name = "1970phys"/>{{rp|58}}。克洛爾上課時會印講義給學生，但是在公式處留空，要求學生在上課時將其補上<ref name = "物理會訊"/>。在1970年代前，克洛爾是全臺灣唯一擁有理論物理博士學位的專任教授<ref name = "許仲平"/>{{rp|830}}<ref name = "張幸真"/>{{rp|177}}。然而，由於早期的臺大物理系以實驗物理為優先，使得理論物理相對不受重視，亦使得克洛爾在系上沒有任何研究室，甚至沒有專屬的桌子<ref name = "張幸真"/>{{rp|178}}<ref name= "JingShin"/>{{rp|xx}}。由於克洛爾認為[[中文|中文]]很難學，不懂得[[臺語|臺語]]和[[中華民國國語|中華民國國語]]，因此他便使用帶有[[德語|德語]]腔調的[[英語|英語]]授課<ref name = "林夏玉"/><ref name= "JingShin"/>{{rp|xvi}}。上課前，他會要求學生「起立敬禮」之後坐下，才開始上課<ref name= "JingShin"/>{{rp|xix}}。克洛爾上課時不苟言笑，也幾乎不提自己的研究<ref name = "許仲平"/>{{rp|831}}<ref name = "林夏玉"/><ref name= "JingShin"/>{{rp|xix}}，喜歡穿著[[獵裝|獵裝]]或歐式[[燕尾服|燕尾服]]授課<ref name = "張秀蓉"/><ref name = "古今論衡"/>{{rp|111}}，將整塊黑板抄滿公式，學生們只能拚命地抄<ref name = "許仲平"/>{{rp|831}}。然而，克洛爾與學生交談時，偏好用日語而非英語<ref name = "古今論衡"/>{{rp|119}}。由於早期物理系的學生學過日語，因此與克洛爾有較多的來往，但是到了後期的學生則較少接觸<ref name = "林夏玉"/>。

[[File:Kroll_Teaching.png|thumb]]
戰後初期，許多隨[[國民政府|國民政府]]赴臺的教授使用[[上海話|上海話]]或[[浙江話|浙江話]]教課，[[本省人|臺籍]]學生因此聽不懂，便在課餘時間以[[日語|日語]]向克洛爾請教，有時師生間也會一同[[飲酒|飲酒]]<ref name = "物理會訊"/>。克洛爾授課時，曾有一位學生視線飄至窗外，克洛爾因此將粉筆丟向他，怒斥「黑板在窗外嗎？」<ref name = "張秀蓉"/>。上課時若有學生打瞌睡或遲到早退，克洛爾也會將粉筆丟向他，如果上課時有學生私下交談則會使他漲紅著臉敲桌斥責之<ref name = "物理會訊"/><ref name = "古今論衡"/>{{rp|111}}。克洛爾為學生評分時，總是取十的整數倍（例如60分、80分），從不給諸如56分、85分之類的成績，類似於日後的等第制。這是因為他認為些微幾分的差距並沒有什麼意義，因此不應該把分數劃分得太細<ref name= "JingShin"/>{{rp|xix}}。

1957年，克洛爾受到時任[[東海大學_(台灣)|東海大學]]物理系系主任的王碩輔邀請，至該校兼課。至此之後，克洛爾便於一週內三天在東海，三天在臺大授課，奔波於[[北臺灣|北臺灣]]與[[中臺灣|中臺灣]]之間<ref name = "林夏玉"/><ref name= "JingShin"/>{{rp|xviii}}。克洛爾在東海物理系所開設的課程包含了[[光學|光學]]以及理論物理<ref name = "林夏玉"/>。到了1960年，克洛爾在[[國家長期發展科學委員會|國家長期發展科學委員會]]{{efn|現[[中華民國科技部|中華民國科技部]]。}}的支持與獎勵下，在1952年論文的基礎之上再度發表兩篇固體比熱相關研究的論文。到了隔年，他並成為長科會物理研究中心唯一的「國立研究講座教授」，並在其後持續進行比熱、磁阻、數學物理等領域之研究<ref name = "古今論衡"/>{{rp|120-121}}。在這段期間，克洛爾利用量子理論計算金屬中[[自由電子|自由電子]]比熱公式，並進一步計算束縛電子的分配函數，求得金屬之磁導率，取代了原本使用[[有效质量|有效质量]]來計算的方法<ref name = "1970phys"/>{{rp|59}}。到了1963年，克洛爾與其研究生一同對束縛電子的[[德哈斯-范阿尔芬效应|德哈斯-范阿尔芬效应]]做研究，並利用將晶格的位能以[[傅立葉級數|傅立葉級數]]展開的方式，求得磁導率的表示式<ref name = "古今論衡"/>{{rp|126}}<ref name="van Alphan"/>。他也和學生一同研究內部有皺紋的[[導波管|導波管]][[電磁波|電磁波]]方程解，並求得其導波係數，成果論文並發表於1963年至1967年的《[[中華民國物理學刊|中華民國物理學刊]]》<ref name = "許仲平"/>{{rp|832}}<ref name = "古今論衡"/>{{rp|120}}<ref name="WG1"/><ref name="WG2"/><ref name="WG3"/>。之後克洛爾也在1967年於該刊發表了有關滿足特定積分式的函數級數展開相關研究論文<ref name="1967paper"/><ref name = "古今論衡"/>{{rp|126}}。克洛爾曾經教過的學生們表示，克洛爾對於不同[[邊界條件|邊界條件]]的電磁[[微分方程|微分方程]]也很有興趣，而[[贝塞尔函数|贝塞尔函数]]對他而言就像老朋友一般<ref name = "許仲平"/>{{rp|831}}<ref name= "JingShin"/>{{rp|xvii}}。到了1972年，克洛爾則主要致力於探討[[彈性理論|彈性理論]]的相關問題<ref name = "古今論衡"/>{{rp|123}}。受到早年所受到的訓練所影響，他的研究內容都較偏向數學計算相關方面<ref name = "古今論衡"/>{{rp|123}}。

===晚年===
[[File:Kroll_lecturing.png|thumb]]
1976年8月，克洛爾自臺大物理系退休，並繼續住在臺北<ref name = "許仲平"/>{{rp|829}}<ref name = "張幸真"/>{{rp|180}}。1988年，在[[張國龍|張國龍]]、[[林清凉|林清-{凉}-]]等早年學生的建議之下，時任臺大物理系系主任黃暉理向臺大校方建議授予克洛爾為名譽教授。克洛爾並在隔年和[[許雲基|許雲基]]一同成為名譽教授<ref name = "張幸真"/>{{rp|180}}。

退休期間，克洛爾曾在[[中國文化學院|中國文化學院]]{{efn|今[[文化大學|文化大學]]。}}物理系授課，亦曾至[[臺北醫學大學|臺北醫學大學]]兼課<ref name = "林夏玉"/><ref name = "物理會訊"/>。然而，1970年代後期，由於臺灣經濟開始起飛，導致物價上揚，使得克洛爾的退休金相對而言變得微薄，生活也因此陷入困境，不足以供應生病時的醫藥費需求<ref name = "林夏玉"/>。因此，他得到了包含曾被克洛爾教導過的早年醫預科德文課學生的人們協力幫助，1991年時任臺大物理系系主任的楊信男教授亦發起募集「克洛爾教授基金」，在極短的時間內募集到超過[[新臺幣|新臺幣]]100萬元<ref name = "林夏玉"/>。負責替克洛爾看病的醫師也會頻繁地打電話給克洛爾，以關心其身體狀況，但不久後克洛爾便不耐煩地以日語回應他：「死不了！」<ref name = "張秀蓉"/>。

1991年1月，克洛爾因氣喘水腫住進了臺大醫院，並於日後康復<ref name = "物理會訊"/>。然而，到了1992年2月，克洛爾的[[肺氣腫|肺氣腫]]再度復發，並於2月28日上午11時於[[臺大醫院|臺大醫院]]病逝，享壽87歲<ref name = "張幸真"/>{{rp|180}}<ref name = "林夏玉"/><ref name = "物理會訊"/><ref name= "JingShin"/>{{rp|xxii}}，其葬禮則由爾後成為[[臺灣基督長老教會|臺灣基督長老教會]]總幹事的羅榮光牧師主持<ref name = "張秀蓉"/>。克洛爾的安息禮拜有一百多位臺日籍學生以及故舊出席，治喪委員會則由臺大物理系教師與克洛爾早期的醫預科學生們共同組成，以協助其自日本趕赴臺灣奔喪的義子處理喪事<ref name = "物理會訊"/>。克洛爾教授基金亦改變用途，用以每年在臺大物理系舉辦「克洛爾紀念演講」<ref name = "林夏玉"/>。

==影響==
克洛爾被喻為「將物理之火從德國帶到臺灣的[[普羅米修斯|普羅米修斯]]」<ref name= "JingShin"/>{{rp|v}}，而大多數早期臺灣物理學者都曾是他的學生<ref name="Spacetime27"/>。後來成為臺大物理系系主任的[[黃振麟|黃振麟]]曾經在年輕時作為克洛爾的助教。受到克洛爾的影響，黃振麟也致力於固體比熱相關領域的研究。兩人的討論並在爾後被收錄於德國出版的物理手冊（{{lang|de|Handbuch der Physik}}）中<ref name = "張幸真"/>{{rp|179}}。至於臺灣第一位獲{{le|NASA頂尖榮譽傑出成就獎|NASA Exceptional Scientific Achievement Medal}}、全球第一位命名[[类星体|类星体]]<ref name="Chiu"/>的[[天文物理|天文物理]]學家[[丘宏義|丘宏義]]則表示，克洛爾是他「最尊敬的幾位老師之一」。丘宏義說，假使他沒有被克洛爾教過量子力學、統計力學以及理論物理學的話，他「將不可能這麼快進入研究領域中」<ref name= "JingShin"/>{{rp|xxi}}。1996年，美國[[麻州大学达特茅斯分校|麻州大学达特茅斯分校]]物理學系與臺大物理系一同舉辦克洛爾紀念學術研討會，並於會後出版專題論文集<ref name= "JingShin"/>。

克洛爾生前一直住在[[新生南路|新生南路]]和[[和平東路|和平東路]]路口附近的一間日式宿舍，即便臺大校方希望能讓他換宿舍，但克洛爾堅持不換，並認為這麼做可以為臺大保留該處<ref name = "物理會訊"/>。爾後，該地區許多的日式宿舍都遭改建，而克洛爾那間約150[[坪|坪]]的宿舍則被確實保留了下來<ref name = "物理會訊"/>。克洛爾去世後，該宿舍歸還給臺大校方，而校方則和陳登壽等人簽約，以活化並更新該建築，之後他們將該建物轉型為一間歐式藝術餐廳<ref name = "Kroll Connection"/>。

另外，臺灣作家[[白先勇|白先勇]]亦在其短篇作品〈那晚的月光〉中，大幅提及了克洛爾{{efn|文中稱為「克洛教授」。}}：
{{cquote|台大物理系畢業考最後一科是理論物理。題目繁而難，延長兩小時還沒有考完。天暗了，教室裡開上了燈，李飛雲最後一個交卷。克洛教授在他面前踱來踱去，李飛雲覺得頭有點發暈，他抬起眼，發覺克洛教授正在看他，克洛教授的眼鏡反射出金光來，他感到一陣眼花，慌忙站起來，把卷子遞給克洛教授。最後一大題，他一個字也沒寫，那一題占三十分。|〈那晚的月光〉，《現代文學》第12期<ref name = "白先勇"/><ref name = "古今論衡"/>{{rp|102}}}}

==家庭==
克洛爾的父親是語言文獻學教授{{efn|亦有來源認為其為神學教授<ref name = "張幸真"/>{{rp|173}}<ref name = "許仲平"/>{{rp|830}}，甚至可能同時都是<ref name= "JingShin"/>{{rp|xiv}}。}}，任教於[[布雷斯勞大學|布雷斯勞大學]]，並曾任德國政府高官，家庭背景顯赫<ref name = "張秀蓉"/><ref name = "古今論衡"/>{{rp|102}}。克洛爾的哥哥是一位醫生，除此之外克洛爾還有一位弟弟<ref name = "古今論衡"/>{{rp|102}}。克洛爾終生未婚<ref name = "林夏玉"/>，但是他有一名義子，以及一位長年協助他處理家務的中老年女性<ref name = "張秀蓉"/>。另外，克洛爾在臺灣還有一位情人，而這位情人是克洛爾的葬禮舉辦時，在棺木上放下最後一朵花的人<ref name = "張秀蓉"/>。
