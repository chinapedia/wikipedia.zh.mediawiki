'''沃恩·弗雷德里克·兰德尔·琼斯'''（{{lang|en|Sir Vaughan Frederick Randal Jones}}，{{bd|1952年|12月31日}}）是一位新西兰美国数学家，以在[[约翰·冯·诺伊曼|冯·诺依曼]]代数和扭结多项式上的研究而闻名。1990年他被授予[[菲尔兹奖|菲尔茨奖]]，并且他因为在[[京都|京都]]举行的颁奖典礼上穿了[[紐西蘭國家橄欖球隊|新西兰国家橄榄球队]]的球衣而闻名。

== 生平 ==
沃恩·琼斯在[[新西兰|新西兰]][[吉斯伯恩|吉斯伯恩]]出生，在[[新西兰|新西兰]]剑桥长大。他在[[奧克蘭文法學校|奥克兰文法学校]]读完初高中。之后他进入新西兰[[奧克蘭大學|奥克兰大学]]深造，并在1972年和1973年相继获得本科和硕士学位。之后琼斯于1979年在[[瑞士|瑞士]][[日内瓦大学|日内瓦大学]]获得博士学位，导师为André Haefliger，博士论文题目为''Actions of finite groups on the hyperfinite II<sub>1</sub> factor。''1980年开始琼斯定居美国，1980-1981年间在[[加州大学洛杉矶分校|加州大学洛杉矶分校]]、1981-1985年间在[[宾夕法尼亚大学|宾夕法尼亚大学]]任教。之后琼斯被聘任为[[加州大學柏克萊分校|加州大学伯克利分校]]数学教授。

琼斯在扭结多项式上的研究成果现被称为琼斯多项式。但这项研究源自于一个从未有人预料到的来自冯·诺依曼代数的[[数学分析|分析学]]分支，这个分支之前被大多数人认为在[[阿蘭·科納|阿兰·科纳]]的研究下已被完善。琼斯多项式为[[扭量理论|扭结理论]]的数个经典问题以及一些低维[[拓扑学|拓扑学]]问题都提供了答案。

2011年开始琼斯在[[范德堡大学|范德堡大学]]担任数学教授。<ref>[http://as.vanderbilt.edu/math/bio/vaughan-jones Personal web page at Vanderbilt University]</ref> 此外他还在他从1985年就开始任教的[[加州大學柏克萊分校|加州大学伯克利分校]]保留着退休教授的名号。<ref>[https://math.berkeley.edu/people/faculty/vaughan-f-r-jones Personal web page at Berkeley]</ref> 琼斯还是新西兰[[奧克蘭大學|奥克兰大学]]的知名校友教授。<ref>[https://unidirectory.auckland.ac.nz/profile/v-jones Personal web page at Auckland]</ref>

琼斯还于1992年被选为世界扭结大会的终身副主席。
