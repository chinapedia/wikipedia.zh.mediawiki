'''沃加沃加'''（'''Wagga Wagga'''，[[澳大利亚英语|澳大利亚英语]]通常仅称Wagga）是位于[[澳大利亚|澳大利亚]][[新南威尔士|新南威尔士]]的一座[[城市|城市]]，紧邻[[穆林碧治河|穆林碧治河]]。它是新南威尔士州最大的内陆城市，也是全澳第五大的内陆城市和[[瑞福利納_(新南威尔士州)|瑞福利納]]地区的主要城市。沃加沃加居民现約46,735人<ref>{{cite web|url = http://getaway.ninemsn.com.au/article.aspx?id=282980|title = Getaway Fact sheets|publisher = Getaway|work = NineMSN|accessdate = 2008-10-10|deadurl = yes|archiveurl = https://web.archive.org/web/20080723121143/http://getaway.ninemsn.com.au/article.aspx?id=282980|archivedate = 2008-07-23}}</ref>。

沃加沃加同时还是[[航空|航空]]企业“[[区域快线|区域快线]]”的營運总部所在地。

== 地理 ==
沃加沃加在[[悉尼|悉尼]]西南方。斯图特[[高速公路|高速公路]]（Sturt Highway）和奥林匹克公路（Olympic Way）在此交汇。沃加沃加位于悉尼至墨尔本[[铁路系统|铁路系统]]的中段。此外城市附近还有一条支线铁路通往{{link-en|坦巴伦巴|Tumbarumba}}。

== 歷史 ==
在[[英國|英國]]人未抵達澳大利亞前，沃加沃加基本是原住民的天下。1829年來自英國的探險家抵達沃加沃加，並在此定居。至1861年，小鎮人口有627人。當時主要靠水路連接對外，至1878年，有鐵路貫通。
