'''虎爺'''，是[[中國民間信仰|中國民間信仰]]中一種以[[虎|虎]]為形象的[[神祇|神祇]]，俗稱'''虎爺公'''、'''虎爺將軍'''、'''虎將軍'''，尊稱為'''下壇將軍'''、'''山君尊神'''。俗曰：「土地神轄山中虎」，虎爺最早是[[山神|山神]]、[[土地神|土地神]]及[[城隍爺|城隍爺]]的座騎；後來更演變成[[王爺神|王爺]]、[[媽祖|媽祖]]等諸神的座騎，並有守護廟境、村莊、地區與城市之功能。
[[File:Master_Tiger_at_Li_Family_Abode_20160213.jpg|thumb]]虎爺像]]

== 來源 ==
俗曰：「土地神轄山中虎」。古人認為虎受土地之神所管，而被山神、土地神、城隍爺等神收伏的老虎具有神力，不但不會傷害人畜，且會保護人類；故人們多尊稱為虎爺，虔誠奉祀。

=== 其他 ===
許多神明的座騎亦是虎，但與此處的虎爺不同，時遭混淆。
*[[瑤池金母|瑤池金母]]乘[[白虎|白虎]]出遊。
*[[張道陵|張道陵]][[天師|天師]]騎虎入山。
*[[崔府君|崔府君]]斷虎罪，歿後乘虎。
*[[保生大帝|保生大帝]]醫治虎喉，因而伏虎。
*[[玄壇真君|玄壇真君]]、[[李鐵拐|鐵拐真人]]、[[伏虎禪師|伏虎禪師]]座騎亦為虎。

== 信仰功能 ==
*土地神等諸神的座騎。
*廟會、慶典時的前導、開路之神。
*鎮守[[廟宇|廟宇]][[城|城]][[村|村]]，驅逐邪魔精怪。
*保護[[兒童|兒童]]，許多地區會要求小孩拜虎爺為[[義父|義父]]。現代也可以讓寵物拜虎爺為義父。
*能招[[錢財|錢財]]，俗言「虎爺咬錢來」，相信虎爺具有[[財神|財神]]的能力。
*有醫療能力，能治療[[臉頰|臉頰]]腫脹的[[腮腺炎|腮腺炎]]。腮腺炎俗稱「豬頭皮」，人稱虎能食[[豬|豬]]，故可治之。一般是以[[紙錢|紙錢]]抹過虎爺塑像之口，再抹過患部。也有請[[乩童|乩童]]、[[道士|道士]]召請虎爺後，以硃砂筆在患部上書寫「虎」字等。

== 祭祀方式 ==
[[File:祀典興濟宮虎爺.JPG|thumb]]虎爺，前面可見作為供品的生雞蛋。]]
[[File:新港奉天宮虎爺殿.jpg|thumb]]
[[File:新港奉天宮金虎爺.jpg|thumb]]
一般以生[[雞蛋|雞蛋]]及簡單的生[[肉類|肉類]]加以奉祀，並將之供奉於神桌底下，但有些廟宇將虎爺奉於神桌之上，如 [[祀典興濟宮|祀典興濟宮]]、[[左營新莊仔青雲宮|左營新莊仔青雲宮]]、[[朴子大糠瑯開基祖廟|朴子大糠瑯開基祖廟]]、[[新港奉天宮|新港奉天宮]]、[[旗山天后宮|旗山天后宮]]、[[朴子配天宮|朴子配天宮]]、[[板橋慈惠宮|板橋慈惠宮]] 等。諸廟中，朴子配天宮的虎爺相傳甚為靈驗；[[新港奉天宮|新港奉天宮]]的虎爺有全台最多分靈，計有超過上千尊（包括家裡、宮廟、工廠或公司等），此外廟方特別打造「金虎爺」（目前在廟方的其他殿有多尊等信徒登記），也可稱「金虎將軍」，並在全台各地成立超過十多個「金虎爺會」（目前分會有艋舺、蘭陽、宜蘭、新北、北桃園、南桃園、新竹、苗栗、台中、彰化、南投、雲林、嘉義、朴子、台南、高雄、旗山、屏東等地，祖會為新港奉天宮金虎爺總會）。

虎爺不但會咬鬼鎮邪，也會咬錢納財，除了擺設香爐，讓民眾與虎爺對話，俗言「虎爺咬錢來」，'''虎爺'''神龕旁常設盛水小碗，水中置錢，俗稱「錢水」，據說金額至少等值，而不能以小錢換大錢，換得的錢母置於[[紅包|紅包]]袋或[[香火|香火]]袋內、暗助生財。早期兒童在供桌下穿梭是常有的事，兒童的高度最容易發現與親近虎爺，供奉於神桌底下的虎爺也因此成為兒童的守護神。

==祭典儀式：虎爺「喫炮」 ==
{{globalize|area=[[臺灣|臺灣]]}}
<ref>{{Cite web|url=http://www.google.com.tw/search?client=ms-android-htc-rev&tbm=isch&sa=1&ei=_e6aWbSOI8ja8QX60q3IDg&q=%E5%8C%97%E6%B8%AF%E8%99%8E%E7%88%BA%E5%90%83%E7%82%AE&oq=%E5%8C%97%E6%B8%AF%E8%99%8E%E7%88%BA&gs_l=mobile-gws-img.1.4.41j0l3j0i24k1.17430.17430.0.19612.1.1.0.0.0.0.556.556.5-1.1.0....0...1.1.64.mobile-gws-img..0.1.554.FlYA6X75CX8#imgrc=HG_gfNVuAEcoOM|title=北港虎爺吃炮 - Google 搜尋|accessdate=2017-08-21|work=www.google.com.tw|language=zh-TW}}</ref>「喫炮（吃炮）」為臺灣廟會中常見的炸轎儀式。在[[雲林縣|雲林縣]][[北港鎮_(台灣)|北港鎮]]每年的農曆三月十九、二十日都會舉行迎媽祖的繞境活動，在活動過程中信徒會在神轎底下燃放大量的鞭炮，以恭迎媽祖神轎的到來，並感謝媽祖的庇祐讓信徒平安順遂、生意興榮、賺大錢，民眾相信鞭炮放的越多來年的運氣就會越好，所以在這兩天的繞境出巡中，光是燃放掉鞭炮的費用就高達數千萬元之多，讓人嘆為觀止。近來，因燃放的放式不同而有：踩炮、犂炮、炮網等等，但都統稱吃炮，'''吃炮'''一詞由此而來。

[[北港朝天宮|北港朝天宮]]虎爺會根據1933年日人相良吉哉所著《台南州祠廟名鑑》記載：「'''虎爺會舊名虎爺公會，主祀神為大虎、二虎、三虎，會員五十三人，創立於1694年。'''」北港俗語：「暗街仔暗眠摸，溪仔底虎爺公」即意指傳統虎爺信仰區域在溪仔底（今[[北港鎮_(臺灣)|北港鎮]]博愛路，於民主路和中正路之間的路段），故溪仔底自古又被稱為「虎爺街」。現在的虎爺會是由陳光男、顏錦河二人重組，會員有200餘人，另有高雄分會、台北分會、中和分會、桃園分會、新北分會,彰化分會,南投分會,皆為三[[擲杯|擲杯]]求得虎爺認可而成立的分會，並有眾多信徒分靈虎爺神尊回家供奉鎮宅祈求平安，或有宮廟請回鎮守壇中，如：高雄苓雅寮保安堂,板橋冠友潁川堂,北港玉巡會,新竹廣澤行館,菓林太子宮,澳門朝天宮。

農曆六月六日為虎爺聖誕。[[北港朝天宮|北港朝天宮]]虎爺爐主之下，原有四個副東（腳仔），朱公爐，虎爺爐，同為四腳爐。另於民國101年增設副爐主兩位,並新增大虎、二虎神尊及爐,並改制副東（腳仔）為兩位,本會朱王公金身與大虎、二虎供奉於爐主家，三虎金身則供奉於會長家。昔虎爺轎過去會上下搖動，今規定不可。虎爺是北港媽祖出巡中最會「喫炮」的神轎，在濃煙漫天、轟隆作響的場景中，「虎爺轎」屹立不搖，轎夫無畏無懼的神采，讓群眾懾服。其虎爺轎的造型與日本神轎相似，步伐也特具日本味。每年北港媽祖遶境時，扛虎爺神轎者須學會「踏腳步」，因為若腳步不協調即會翻轎。神轎是由打鼓和拿扇子的人指揮，抬轎者皆為全國各地主動來北港參與之信徒，其中大部份為原籍北港的外移人口，因以前曾抬過虎爺轎，故經驗老道，每年到了農曆三月十八日就會主動回北港，並尋得虎爺轎停放的臨時行館，領取轎班衣。抬轎者大約有三分之二非會員，而會員也對抬轎者大多不相識，完全由老手帶領有興趣的新手參加。目前虎爺會抬轎人員為民國102年虎爺會有鑑於抬轎人員人數之不足,另成立虎爺會轎班團,專責虎將軍出轎時擔當抬虎爺轎及相關工作,非虎爺會轎班團成員則無法扛到虎爺神轎,虎爺轎班穿著黃色虎紋衣，頭戴黃色漁夫帽，在遶境行列中，虎虎生風的抬轎方式，在熊熊的炮火中展現威風凜凜的英姿，令人敬畏著迷。除了農曆三月十九日「北港迎媽祖」外，每年農曆正月十五[[上元|上元]]遶境，虎爺也會出轎參與伴駕，後改為虎年[[台灣元宵節|元宵節]]才會出轎。

1983年農曆三月十九日晚上，傳出朝天宮內之「八角琉璃長明燈」香火無故熄滅，緊接著就發生虎爺轎在[[北港鎮_(臺灣)|北港鎮]]民主路「玉時軒」商店前遭遇土製炸彈之難，當時虎爺轎解體，三尊虎爺均有受損，抬轎者多人送醫急救(依老會員口述當時扛轎者正在休息用餐，虎爺轎停駕在路旁，有一群少年將炮堆丟進轎底引燃，有可能好玩將土製炸彈一併丟入而導致爆炸，所以當時並無轎班人員受傷，而是那群少年送醫)，當時三虎神像還「飛」到屋頂上，於是當年虎爺會停止遶境活動。隔年（1984年）就在農曆三月二十日入廟時分，六媽亦遭土製炸彈之殃，'''虎爺臨身起乩，模樣像老虎般，走起路來虎虎生風，停止時也會捋虎鬍鬚，檢查著每一堆虎爺轎要吃的鞭炮，有時興奮之際，也會要求喫炮──由祂獨自蹲在一大堆的鞭炮上將其燃放，看過的人莫不嘆為觀止，接連稱呼「勇喔！」，該事件也被戲稱「虎爺嘛驚炸彈」'''。

==藝文作品中的虎爺==
2016年，臺灣藝術大學多媒體動畫藝術學系101級畢業製作作品「虎爺TigerGod」即以民間信仰中的虎爺作為主軸，發展出一部達七分多鐘的短篇動畫。大致敘述兒童不慎放出封印中的妖物，而在虎爺庇佑下平安無事的驚險歷程。該作品在2016巴哈姆特ACG創作大賽-動畫組中拿下人氣賞的金賞，亦於其他競賽中獲獎或入圍的成績。製作組至今仍持續發布相關動畫，並推出實體周邊。
