'''虎蹲砲'''，又写作“虎蹲炮”，是中国古代一種小型管身[[火砲|火砲]]，最早在[[宋朝|宋朝]][[曾公亮|曾公亮]]等所著的《[[武經總要|武經總要]]》的記載中为一種人力[[投石機|投石機]]。虎蹲炮的大规模研制与使用，始自中国[[明朝|明朝]][[戚继光|戚继光]][[倭寇|抗倭]]时期，由[[戚繼光|戚繼光]]在抗倭戰爭中為了反制倭寇的[[鳥銃|鳥銃]]而發明。戚继光在研究了当时的几种火炮后，为了克服发射时容易产生后座而造成自伤的缺点，在炮口安装了[[支架|支撑架]]，故虎蹲炮以射擊時外觀像猛虎蹲坐而得名。<ref name=":0">{{Cite book|title=《军事知识大百科》|last=邢涛 龚勋 等|first=|publisher=华夏出版社|year=|isbn=|location=|pages=}}</ref>

這種[[輕型炮|輕型火炮]]後來成為明軍的重要武器，連[[骑兵|騎兵隊]]都有裝備。<ref>《一看就通的中國軍事史》，藍永蔚、黃樸民、劉慶、鍾少異 著。</ref>

历史记载，1598年11月18日夜，[[明朝|中国明朝]]军队与[[朝鲜历史|朝鲜]]联军组成的水军在朝鲜[[露梁海峽|露梁海峡]]的[[露梁海戰|露梁海战]]中遭遇[[日本军队|日本军队]]百艘船的阻击，明军战船上多尊虎蹲大炮连续猛烈轰击，日军战船纷纷中弹起火，损失惨重。<ref name=":0" />

== 特点 ==
* 虎蹲炮砲身短，長約2尺，体轻，在火砲中算是輕巧，適合在山林水網中機動，也可船载作战。
* 炮筒外加装五道箍，使用时不易炸裂。
* 虎蹲炮发射时前身加铁爪钉，后身加铁绊，将其固定于地。<ref name=":0" />
* 虎蹲炮能够发射[[散彈|散弹]]，可一次射出約100顆[[鳥銃|鳥銃]]用彈丸或50顆大彈丸，火力散佈面積大，可有效殺傷密集的敵軍。

== 装填 ==
虎蹲炮的填装过程是：先装火药和土隔，再装弹丸。据明代后期[[朝鲜历史|朝鲜]]军事典籍《[[火炮式谚解|火炮式谚解]]》记载，虎蹲炮[[铸铁|铸铁]]而制，全长40厘米，口径4厘米，外径14厘米，填装一次需要[[炮捻|炮捻]]半条，[[火药|火药]]6两，[[土隔|土隔]]两寸，铅丸70颗或铁丸30颗。发射前再将一颗大铅子（重5两）放在炮口中央发射。<ref>{{Cite web|url=http://elec.it168.com/a2016/0525/2662/000002662660.shtml|title=揭秘大明王朝的火器装备|accessdate=2016-10-12|author=《中华刀剑》|date=2016-05-25|publisher=《中华刀剑》杂志社}}</ref>

== 军事运用 ==
在[[戚继光|戚继光]]所著的《[[练兵实纪|练兵实纪]]》和《[[纪效新书|纪效新书]]》中，关于虎蹲砲的具体操作记载如下：

《练兵实纪》中记载“马兵每旗总一名，队总三名，共管虎蹲炮一位。”而此马兵一旗为三队，每队12人，合计36人，也就是每36人编制有虎蹲炮一门。此炮由队长操作，列于阵前一丈（3.2米），“虎蹲炮安各旗空前，去人一丈”。

在《纪效新书》十四卷本中，也是三名队长合演虎蹲炮一门。因此，在通常状态下，虎蹲炮编制当为三队一门，即36人冷兵器队列之中，配备虎蹲炮一门，由三名队长来点放火炮。<ref>{{Cite web|url=http://m.sohu.com/n/468976096/?wscrid=95360_5|title=虎蹲炮：戚继光《纪效新书》里隐藏的秘密武器|accessdate=2016-10-12|author=《冷兵器时代》|date=2016-09-22|publisher=《冷兵器时代》杂志社}}</ref>
