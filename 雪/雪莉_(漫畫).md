《'''雪莉'''》（{{lang-ja|シャーリー}}），是日本漫畫家[[森薰|森薰]]的作品。有別於《[[艾瑪_(漫画)|艾瑪]]》的[[維多利亞時期|維多利亞時期]]背景，本作為[[愛德華時代|愛德華時期]]。<ref name=kt>[https://www.kadokawa.com.tw/p1-products_detail.php?id=34f4954N8tkyF7wRS-h6FZTGOo29gwUgqYu4-tcFkfI 台灣角川KADOKAWA 雪莉]</ref><ref name=cmx>{{Citation |title=SHIRLEY VOL. 1 |date= |url=http://www.dccomics.com/cmx/?action=on_sale&i=9597 |accessdate= |archiveurl=https://web.archive.org/web/20080616050218/http://www.dccomics.com/cmx/?action=on_sale&i=9597 |publisher=CMX MANGA |archivedate=2008-06-16 |language= |deadurl=yes }}</ref><ref  name=no2>{{lang|ja|第2巻, p.189, 「あとがき ちゃんちゃらマンガ メイド漫画で今日も元気!!」.}}</ref>

== 概要 ==
原是森薰[[同人誌|同人誌]]時期的作品《Shirley Medison》<ref name=lm>{{Citation|script-title=ja:LadyMaid -サークル活動- 今までに出した本|date=|url=http://pine.zero.ad.jp/~zad98677/ladymaid-b.htm|accessdate=|archiveurl=https://web.archive.org/web/20020613163831/http://pine.zero.ad.jp/~zad98677/ladymaid-b.htm|publisher={{lang|ja|伯爵夫人の昼食会}}|archivedate=2002-06-13|language=ja|deadurl=yes}} </ref>，2006年10月以降不定期新作《{{lang|ja|シャーリー・メディスン}}》發表<ref>[http://www.amazon.co.jp/dp/4757729987/ {{lang|ja|コミックビーム Fellows! Vol.2}}]. Amazon.</ref>，為其出道前最初真正描繪的漫画<ref name="jumpsq_mori_3">{{Citation|script-title=ja:森薫先生 直撃インタビュー 完全版 第3回 今後の展開と影響を受けた作品|date=|url=http://jumpsq.shueisha.co.jp/contents/topic-mori/index3.html|accessdate=|archiveurl=https://web.archive.org/web/20110610095637/http://jumpsq.shueisha.co.jp/contents/topic-mori/index3.html|publisher=[[集英社|集英社]]|archivedate=2011-06-10|language=ja|deadurl=yes}}</ref>。

同人誌時期全5話收錄在2003年首部單行本，從2014年的第2卷開始收錄新作。

== 故事簡介 ==
經營[[咖啡廳|咖啡廳]]的'''貝妮特·克蘭莉'''，招募女僕卻粗心忘了註明年齡限制，而收留孤苦無依的'''雪莉·麥迪遜'''，就此展開兩人之間的日常生活……。

== 登場人物 ==
; {{nihongo|'''雪莉·麥迪遜'''|シャーリー・メディスン|Shirley Madison}}
: 看到求人廣告前來應徵的13歲黑髮少女。無依無靠，有點女僕的工作經驗，年紀雖小但非常優秀，煮飯、打掃等家事表現出色，並有裁縫的才能，工作上顯露出[[完美主義|完美主義]]者的一面。仰慕主人貝妮特的樣子。
: 作者受到《[[魔女宅急便|魔女宅急便]]》主角影響的設定<ref name=ps>首部單行本附錄{{nihongo|「後記&藉口一堆鏘啷啷漫畫 世事總是難料啊！」|あとがき&いいわけちゃんちゃらマンガ 人生七転び八転び|Silly Afterword (& Excuse) Manga: The Ups and Downs of Life}}</ref>。名字由来：「雪莉」出自[[秀蘭·鄧波兒|秀蘭·鄧波兒]]、「麥迪遜」則取「{{lang|ja|メイドさん}}」[[諧音|諧音]]。{{r|lm}}
; {{nihongo|'''貝妮特·克蘭莉'''|ベネット・クランリー|Bennett Cranley<ref name=animevice>[http://www.animevice.com/shirley/16-2229/ Shirley #1 - Shirley manga] {{webarchive|url=https://web.archive.org/web/20140724233802/http://www.animevice.com/shirley/16-2229/ |date=2014-07-24 }}. Animevice.</ref>}}
: 雪莉的主人，28歲，單身的金髮美女。原是上流階級的千金小姐，因此生性樂觀。{{r|ps}}兩親死後，忙著「蒙娜麗莎」店中的生意，沒空打掃家裡，於是徵求女僕。
;  {{nihongo|'''常客老伯'''|常連の老人}}
: 貝妮特店裡的無名[[龍套|龍套]]常客。作品《妮莉與我的某日午後》，以及同貝妮特在《瑪麗·班克斯》客串出場。
