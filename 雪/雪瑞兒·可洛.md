'''雪瑞兒·蘇珊·可洛'''（{{lang-en|Sheryl Suzanne Crow}}，{{bd|1962年|2月11日}}），美國[[創作歌手|創作歌手]]、唱片製作人、音樂家和演員。她的音樂融合搖滾、民謠、嘻哈、鄉村和流行元素，已九度獲得[[葛萊美獎|葛萊美獎]]肯定。

她曾和[[滾石樂團|滾石樂團]]同台演出，也曾和眾多知名歌手合唱，包括[[米克·傑格|米克·傑格]]、[[麥可·傑克森|麥可·傑克森]]、[[艾瑞克·克萊普頓|艾瑞克·克萊普頓]]、[[盧奇亞諾·帕華洛帝|盧奇亞諾·帕華洛帝]]、[[約翰·麥倫坎普|約翰·麥倫坎普]]、[[搖滾小子|搖滾小子]]、[[密雪兒·布蘭奇|密雪兒·布蘭奇]]和[[史汀|史汀]]等。她曾為[[蒂娜·透納|蒂娜·透納]]、[[唐·亨利|唐·亨利]]和[[貝琳達·克萊兒|貝琳達·克萊兒]]演唱和聲，收錄在1991年《[[劈腿排行榜|劈腿排行榜]]》電影配樂。可洛的音樂作品包括7張錄音室專輯、2張合輯和1張現場專輯，也曾參與多張電影原聲帶。近期可洛則投入[[國家廣播公司|國家廣播公司]]的《[[超級製作人|超級製作人]]》、[[美國廣播公司|美國廣播公司]]的《[[熟女當道|熟女當道]]》、[[迪士尼頻道|迪士尼頻道]]的《[[汉娜·蒙塔娜_(第四季)|孟漢娜第四季]]》以及《[[恢復精神正常和／或恐懼大會|恢復精神正常和／或恐懼大會]]》等小螢幕的演出。

== 早年生活 ==
雪瑞兒·蘇珊·可洛在1962年2月11日出生於[[密蘇里州|密蘇里州]][[肯尼特_(密蘇里州)|肯尼特]]，律師父親溫德爾·可洛擅奏[[小號|小號]]，母親柏妮絲則是鋼琴教師<ref>{{cite news

在肯尼特中學時期，可洛擔任樂隊女領隊，同時也是全州[[田徑|田徑]]選手，曾在75公尺低欄項目拿下不少獎牌。她也參加[[活力俱樂部|活力俱樂部]]（Pep Club）、[[全國榮譽協會|全國榮譽協會]]和[[全國未來農人協會|全國未來農人協會]]。她隨後進入[[哥倫比亞市_(密蘇里)|哥倫比亞市]]的[[密蘇里大學|密蘇里大學]]就讀，獲得音樂創作、表演和教育的[[文學士|文學士]]學位。學業之餘，可洛在當地樂團「喀什米爾」演唱。她也是「ΚΑΘ」國際社會[[兄弟會與姊妹會|姊妹會]]、「ΣΑΙ」國際音樂女性互助會和「ΟΔΚ」國家領導榮譽學會的成員<ref>{{cite video |people = Ben Graham |date  = 2003-04-28 |title= Maximum Sheryl Crow |medium = Audio CD |publisher  = Chrome Dreams |location= |accessdate = 2008-03-07 |time=  |id= |isbn=  }}</ref>。可洛後獲得密蘇里大學和[[开普吉拉多|吉拉多角]]的[[東南密蘇里州大學|東南密蘇里州大學]]頒授[[榮譽學位|榮譽博士]]。

可洛曾說明她的音樂靈感並不限於特定領域，但她非常喜歡在歌曲中加入鼓聲。2008年，她在[[艾倫·狄珍妮|艾倫·狄珍妮]]的節目上表示：「如果沒有鼓點，那就算了吧！」

== 私人生活 ==
可洛自2003年起和公路自行車賽職業車手[[藍斯·阿姆斯壯|藍斯·阿姆斯壯]]交往<ref name="Softpedia" />。他們在2005年9月宣布婚訊<ref name="Softpedia">{{cite web |url=http://news.softpedia.com/news/Sheryl-Crow-Engaged-To-Lance-Armstrong-7734.shtml |title=Sheryl Crow Engaged To Lance Armstrong |work=Softpedia |date=2005-09-06 |accessdate=2011-07-14 |deadurl=yes |archiveurl=https://web.archive.org/web/20091022085120/http://news.softpedia.com/news/Sheryl-Crow-Engaged-To-Lance-Armstrong-7734.shtml |archivedate=2009-10-22 }}</ref>，但隔年2月因對婚姻和小孩的理念不同而分手<ref>{{cite web |url=http://www.foxnews.com/entertainment/2009/05/08/lance-armstrong-sheryl-crows-biological-clock-led-split/ |title=
Lance Armstrong: Sheryl Crow's 'Biological Clock' Led to Our Split |work=FOXNews |date=2009-05-08 |accessdate=2011-07-14}}</ref><ref>{{cite web |author=Alicia Dennis |url=http://www.people.com/people/article/0,,1156475,00.html |title=Lance Armstrong and Sheryl Crow Split |work=''[[人物_(雜誌)|People]]'' |date=2006-02-03 |accessdate=2011-07-14}}</ref>。在和阿姆斯壯結束交往後不久，可洛前往洛杉磯投入[[乳癌|乳癌]]治療，並在[[放射線療法|放射線療法]]後，於2006年2月進行微創手術<ref>{{cite news|url=http://news.bbc.co.uk/1/hi/entertainment/4749716.stm |title=– February 2006 – Sheryl Crow has breast cancer op |publisher=BBC News |date=2006-02-25 |accessdate=2010-10-19}}</ref><ref>{{cite web|url=http://www.dailymail.co.uk/tvshowbiz/article-378331/Sheryl-Crow-I-beat-breast-cancer.html |title=Daily Mail – February 2006 – Sheryl Crow: I will beat breast cancer |publisher=Dailymail.co.uk |date= |accessdate=2010-10-19}}</ref>。

2007年5月11日，可洛在官網上宣布她[[收養|收養]]了一名出生於2007年4月29日、2週大的男嬰，取名懷亞特·史蒂芬·可洛<ref>{{cite web |url=http://www.sherylcrow.com/blog/default.aspx?dt=05-01-2007 |title=Announcing..... |work=sherylcrow.com |date=2007-05-12 }}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。她和懷亞特定居在田納西州納什維爾外的大農場<ref>{{cite web |author=Marisa Laudadio |url=http://www.people.com/people/article/0,,20191938,00.html?xid=rss-fullcontentcnn |title=Sheryl Crow Preps for Life on the Road with Baby Wyatt |work=''[[人物_(雜誌)|People]]'' |date=2008-04-16 |accessdate=2011-07-14}}</ref><ref>{{cite web |author=Susan Mandel |url=http://www.people.com/people/article/0,,20201790,00.html?xid=rss-fullcontentcnn |title=Sheryl Crow: Baby Wyatt Is the Real 'Rock Star' |date=2008-05-22 |accessdate=2011-07-14}}</ref>。2010年6月4日，可洛發布她收養第二名男孩的消息。男孩出生於2010年4月30日，命名為李維·詹姆斯·可洛<ref>{{cite web |url=http://celebritybabies.people.com/2010/06/04/sheryl-crow-adopts-a-second-son |title=Sheryl Crow Adopts a Second Son! |date=2010-06-04}}</ref>。
