
'''雪城'''是[[西藏|西藏]][[拉萨市|拉萨市]][[布达拉宫|布达拉宫]]山脚下正面所有建筑的总称，东西长317米，南北宽170米，面积五万平方米，现已改建成'''雪城博物馆'''。<ref name="官网">{{cite web | url=http://www.potalapalace.cn/home2/zzwhdoc157.html | title=解说布宫：景点：雪城 | website=布达拉宫官方网站}}</ref>

在藏语中“'''雪'''”意为下方，专指山上城堡或宫殿正下方的村镇，这里特指布达拉宫下方的雪城。<ref name="官网"/>

== 历史 ==
[[File:Bundesarchiv_Bild_135-BB-094-10,_Tibetexpedition,_Blick_auf_die_Potalastadt.jpg|thumb]]
雪城几乎与布达拉宫属于同一时期建造，其作为政府职能机构所在地的地位直到中共占领西藏后才结束。

雪城博物馆於2007年5月31日開始正式开放，在此之前，雪城与布达拉宫一同处于文物保护状态。

== 功能 ==
在雪城建成初期其职能只是负责管辖雪城内部及拉萨近郊的18个[[宗_(行政區)|宗]]、[[谿卡|谿卡]]（领主庄园），其后逐渐演变成[[噶厦|噶厦]]政府部门所在地。

== 布局 ==
[[File:Potala_and_Sho_Doring.jpg|thumb]]
按照使用功能的不同，雪城主要可划分为办公区和生活区。

办公区又可分为[[贵族|贵族]][[领主|领主]]建筑的驻在机构，和[[噶厦|噶厦]]政府部门，生活区主要是为布达拉宫服务的平民及部分贵族所住。

现存最早的[[吐蕃|吐蕃]]碑刻[[恩蘭·達札路恭紀功碑|恩蘭·達札路恭紀功碑]]位於雪城內。該碑紀念[[吐蕃王朝|吐蕃王朝]][[赤松德贊|赤松德贊]][[贊普|贊普]]派重臣[[恩蘭·達札路恭|恩蘭·達札路恭]]、[[尚結息|尚結息]]於[[唐代宗|唐代宗]][[廣德|廣德]]元年（763年）率軍攻陷唐朝首都[[長安|長安]]。<ref name=wangyao>王尧，恩兰·达札路恭纪功碑，社会科学战线1981年04期</ref>

=== 主要机构 ===
'''[[雪巴列空|雪巴列空]]'''
[[File:Magistrates_of_Sho_in_courtroom.jpg|thumb]]
雪巴列空为雪城的核心机构，“列空”在藏语中位管理之意，雪巴列空即负责雪城管理的地方，其职能还负责拉萨及近郊的宗、谿，其职能具体有司法和税收，还负责辖区内的日常行政管理。<ref name="雪巴列空">{{cite web | url=http://info.tibet.cn:80/zt2007/07zt_xuecheng/fyscl/xblk/200706/t20070612_251882.htm | title=雪巴列空 | archiveurl=https://web.archive.org/web/20080626230009/http://info.tibet.cn/zt2007/07zt_xuecheng/fyscl/xblk/200706/t20070612_251882.htm | deadurl=yes | archivedate=2008-06-26 | website=中国西藏信息中心 | access-date=2018-07-25 }}</ref>

该机构最初名称并非雪巴列空，而是叫雪聂列空，建于[[藏历|藏历]]十一[[绕迥|绕迥]]木兔年（1675年），由第司·[[罗桑金巴|罗桑金巴]]所创建。最初职能也只是负责雪城内务的治安及保卫布达拉宫，后期权力逐渐扩充，而成为雪城的核心机构。<ref name="雪巴列空"/>

雪巴列空的行政官员称为“[[雪尼|雪尼]]”，由2民五品僧、俗官及一位[[列赞巴|列赞巴]]级世俗官负责。

雪巴的底层及二层为仓库负责存储附近所征收的粮食。

'''宝藏局造币厂'''

宝藏局创建于[[清|清]][[乾隆|乾隆]]时期，由时[[驻藏大臣|驻藏大臣]]督建，創建後成为清代西藏地区货币发行最高机构，其名称依照清廷所颁行的“中央集权，分省铸币”原则，命名为宝藏局，所铸造[[章卡|章卡]]形制为：正面“乾隆宝藏”多为银质，原型无方眼。
