'''雪莉·戴依'''（{{lang-en|Shelley Day}}）是美國的女性電子遊戲設計師、遊戲監製、以及遊戲公司[[Humongous_Entertainment|Humongous Entertainment]]、[[Hulabee_Entertainment|Hulabee Entertainment]]的成立者。

== 經歷 ==
=== 遊戲開發 ===
在製作遊戲經歷上雪莉起初於1985年加入了[[美商藝電|美商藝電]]，隨後則進入了[[LucasArts|LucasArts]]參與數款作品的開發。

之後雪莉與一同離開LucasArts的[[羅恩·吉伯特|羅恩·吉伯特]]；在1992年成立了[[Humongous_Entertainment|Humongous Entertainment]]公司，<ref>{{cite web |url=http://www.humongous.com:80/about/pr/more/corp.html |archiveurl=https://web.archive.org/web/19980210015436/http://www.humongous.com/about/pr/more/corp.html |title=Humongous Entertainment Fast Facts (Updated Aug 26, 1997) |work=Humongous Entertainment |archivedate=1998年2月10日 |accessdate=2017年4月1日 |deadurl=yes }}</ref>設計了多款適合幼齡孩童遊玩的《[[噗噗車系列|噗噗車]]》系列遊戲，並曾在1999年期間被《[[時代_(雜誌)|時代]]》雜誌列為前50位電腦網路世代菁英人士之一。<ref>{{cite web |url=http://time.com/time/digital/cyberelite/24.html |archiveurl=https://web.archive.org/web/20090512202808/http://time.com/time/digital/cyberelite/24.html |title=24 SHELLEY DAY |work=[[時代_(雜誌)|時代]] |archivedate=2009年5月12日 |accessdate=2017年4月1日}}</ref><ref>{{cite web |url=http://www.prnewswire.com/news-releases/humongous-entertainment-founders-launch-hulabee-entertainmenttm-create-next-generation-of-fun-and-learning-for-children-71988392.html |title=Humongous Entertainment Founders Launch Hulabee Entertainment(TM), Create Next Generation of Fun and Learning for Children |work=[[美通社|美通社]] |date=2001年5月21日 |accessdate=2017年4月1日}}</ref>

2001年雪莉離開成立的Humongous Entertainment，另創建名為[[Hulabee_Entertainment|Hulabee Entertainment]]的遊戲公司；主推適合孩童遊玩的線上遊戲作品。<ref>{{cite web |url=http://ritholtz.com/2005/05/the-cyber-elite-50/ |title=The Cyber Elite 50 |work=The Big Picture |author=Barry Ritholtz |date=2005年3月16日 |accessdate=2017年4月1日}}</ref>

=== 涉案 ===
後續雪莉為了想購買位於[[華盛頓州|華盛頓州]][[默瑟島_(華盛頓州)|默瑟島]]裡的豪華居所，偽造她的遊戲公司Hulabee Entertainment將會出售給[[迪士尼|迪士尼]]旗下集團的假消息；來向[[西雅圖|西雅圖]]一帶的銀行詐騙了約140萬美元貸款，之後在2005年12月2日被宣判需服刑兩年。<ref>{{cite web |url=http://www.gamelaw.org/index.php?name=News&file=article&sid=136 |archiveurl=https://web.archive.org/web/20070916211714/http://www.gamelaw.org/index.php?name=News&file=article&sid=136 |title=U.S. v. Shelly Day (2005) Well Known Developer Convicted of Fraud |work=Game Law |date=2005年12月6日 |archivedate=2007年9月16日 |accessdate=2017年4月1日 |deadurl=yes }}</ref><ref>{{cite web |url=https://www.justice.gov/archive/usao/waw/press/2005/dec/day.html |title=KIRKLAND COMPUTER GAME COMPANY EXECUTIVE SENTENCED TO PRISON FOR BANK FRAUD |work=[[美國司法部|美國司法部]] |date=2005年12月2日 |accessdate=2017年4月1日}}</ref>
