'''雪花星團'''是在[[麒麟座|麒麟座]]內，包含於[[NGC_2264|NGC 2264]]內的一個星團。

== 名稱 ==
這些[[恆星|恆星]]在輪輻上類似[[雪|雪花]]結構樣式的配置，並在直線上呈現規則的分布<ref>http://www.spitzer.caltech.edu/Media/happenings/20051222/index.shtml</ref>， 因此，[[天文學家|天文學家]]給新發現的這個星團起了個'''雪花星團'''的綽號。

== 資訊 ==
科學家相信這些是非常年輕的[[原恆星|原恆星]]，或是剛誕生的恆星，年齡大約在100,000歲。這些嬰兒還在他們誕生場所的結構中蹣跚爬行中<ref>{{cite web |url=http://www.spitzer.caltech.edu/Media/mediaimages/sig/sig05-028.shtml |title=存档副本 |accessdate=2006-12-20 |deadurl=yes |archiveurl=https://web.archive.org/web/20060907091846/http://www.spitzer.caltech.edu/Media/mediaimages/sig/sig05-028.shtml |archivedate=2006-09-07 }}</ref>，隨著時間的過去，每顆恆星自然的漂移運動將打破這些限制，雪花般的結構將不復存在。[[史匹哲太空望遠鏡|史匹哲太空望遠鏡]]的觀測顯示正如理論的預測，恆星在雲氣中形成時的溫度和密度支配了原恆星之間的空間。
