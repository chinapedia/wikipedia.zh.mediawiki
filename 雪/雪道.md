'''雪道'''（piste）是一条标记出来用于滑雪或其它冬季运动的山地斜道。

单词“piste”来源于法语「piste」（路径），在北美也称作“trail”（小径）、“slope”（斜坡）或者“groomed run”（平整雪道）。

在北美，人们也经常用piste的反义词“off piste”来表示野外滑雪，特指在[[滑雪场|滑雪场]]指定区域以外滑雪。

== 雪道维护 ==
雪道经常使用雪地履带车（snowcat）来维护。它可以压实并平整雪道，削除上面的雪包（moguls），让积雪分布更加均匀，以延长滑雪季节长度。在雪季初期自然积雪较少的时候，人们使用造雪机来增加积雪。

== 难度评级 ==
雪道的难度评级一般都由滑雪场自行进行，难度级别一般都是相对同一雪场其它雪道而言。雪道评级没有统一的标准，虽然各雪场评级方法可能相似，但是滑雪者也不应该认为相同的级别在不同雪场一定代表相同的难度。

=== 北美、新西兰和澳大利亚 ===
[[File:June_Lake_Ski_Sign.JPG|缩略图]]
北美使用彩色图形来标识雪道的相对难度。澳大利亚和新西兰也是用类似的标识系统。

雪道难度一般用斜率而非倾角来表示。斜率100%的坡道为45度角。一般来说，初级雪道（绿色圆圈）斜率在6%到25%之间，中级雪道（蓝色方块）斜率在25%到40%之间，高级雪道（黑色菱形）斜率在40%以上，然而这只是一般原则。虽然坡道斜率是难度评级的主要考量依据，其它因素也可以影响评级。如果一条雪道各处难度不一，一般它会以它最难部分来评级。

=== 欧洲 ===
[[File:Ski_slope_in_Park_Snow_Donovaly.jpg|缩略图]]
[[File:Black_piste_Agate_in_Flaine.jpg|缩略图]]
在欧洲，雪道也是通过颜色编码系统来划分难度。每个国家的颜色标识有所不同——但是所有国家都使用蓝色（简单）、红色（中等）以及黑色（专家）标记。欧洲一般不使用图案标记，有时候所有标记都是圆圈形状。
;   绿色
: （西班牙、法国、斯堪的纳维亚各国、英国、波兰） 初学者雪道。一般它们都没有特别标记的路线，而是一片一般处于山底部的巨大的、拥有平缓坡度的开放区域，或者主要雪道之间的水平通路。可能拥有绿色圆圈标志。
;   蓝色
: 较简单的雪道，类似北美的绿绿道）。一般经过平整，或者坡度极缓而无需平整。除了局部极为宽敞的路段，一般斜率不会超过25%。一般会有蓝色方块标志。<br>
;   红色
: 中等难度雪道，类似北美的蓝色方块（蓝道）。比蓝色雪道更陡或更窄，一般经过平整，除非区域太窄。雪道斜率除部分路段一般不会超过40%。一般会有红色方块标志。
;   黑色
: 专家难度雪道，等同于北美的黑道或者双黑道。一般非常陡峭，可能未经平整，也可能为雪包而平整。在奥地利、意大利和瑞士，黑道一般总是经过平整，没有平整的黑道标记为 skiroutes 或者 itineraires（见下文）；在法国，部分黑道可能经过平整，但是一般都没有。黑道是一个非常宽泛的分类，从比红道难一点到非常陡峭的雪槽（chutes），如 Courchevel 著名的 Couloirs。法国的红道和黑道标记界线一般难度较高。一般标记为黑色菱形。
; 两个或者三个黑色菱形
: （斯堪的纳维亚各国）极端高难的雪道。
; 橙色
: （奥地利、瑞士、某些其他地方）极端高难。
; 黄色、橙色方块，红色菱形
: 近年来，很多滑雪场把一些黑道标记为了黄道，这一般是指 skiroute 或者 itineraire，未平整、无巡逻的标记雪道。比较有名的例子有 Zermatt 的 Stockhorn 区域以及 Verbier 的 Tortin 雪道。在奥地利，skiroute 一般有橙色方块标记。这些雪道也经常被标记为红色菱形或者带有黑边的红色菱形（后者更难）。

=== 日本 ===
日本使用颜色系统标记难度，但是一般不用形状。一些滑雪场，尤其是那些外国游客较多的滑雪场，也是用北美或者欧洲难度标记，造成了一些混淆，一般应该以地图图例为准。标记有：
; 绿色
: 初级雪道。它们一般都在山的底部，也有设计为从山顶曲折下降的。
; 红色
: 中级雪道，在日本绝大部分的滑雪场，这些都占据了主要的部分（40%到60%的面积）。
; 黑色
: 高级雪道，滑雪区域里面最陡的部分，但是它们和其他滑雪场同样标记的雪道的相对难度一般由雪场服务的游客对象决定。
