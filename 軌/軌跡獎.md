'''轨迹奖'''（'''Locus Award'''）创立于70年代早期，颁发给《{{tsl|en|Locus_(magazine)|轨迹 (杂志)|轨迹}}》杂志年度读者投票的优胜者，给雨果奖的投票者们提供意见和建议。几十年来轨迹奖吸引到的投票者数经常超过[[雨果奖|雨果奖]]（Hugo Award）和[[星云奖|星云奖]]（Nebula Award）的总和。最近几年，轨迹奖在年度宴会上颁发。与其他奖项不同，轨迹奖还给获奖作品的出版商颁发明确的奖励证书。

== 范围 ==
科幻、奇幻和恐怖小说，短篇小说，小说集，小说选集，非小说类作品，作家，编辑，杂志，出版商。
== 时间和地点 ==
从2006年开始，颁奖仪式移至位于西雅图的[[科幻博物馆|科幻博物馆]]和[[科幻名人纪念馆|科幻名人纪念馆]]。
== 流程 ==
读者在每个奖项提名五部作品，从高到低排序；按照“卡尔”体系计票，第一名计8分，第二名计7分，以此类推。从2005年开始，每个奖项的前五名将作为“入围”作品在公开活动中公布。每年公布的投票结果包括：每部作品的得票数，总票数，第一名得票数和按照上述体系计票的得分。最终结果根据总得分进行排序。只有最终的排序显示在资料库中。
