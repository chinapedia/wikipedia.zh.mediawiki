{{地区用词2|zh-cn=体波震级|zh-tw=體波規模|equiv-hk=cn|equiv-mo=cn|as=称|before='''|after='''|start=（记作'''<math>m_b</math>'''，{{lang-en|Body wave magnitude}}），|end=，}}是一種測量[[地震|地震]][[地震度量|震級]]的方式。

==概要==
體波震級是使用[[P波|P波]]的振幅以計算地震的震級。P波是一種以5到8 km/s在整個地球中傳遞的地震波，並且在地震時會最早到達[[地震儀|地震儀]]。因此，計算體波的強度是確定遠距離地震震級最快的方式。

[[賓諾·古登堡|賓諾·古登堡]]於1945年提出以體波週期在0.5到12秒之間的體波震級計算公式：

:<math>m_\mathrm{B} = \log_{10}\left({A \over T}\right)_{\max} + Q\left(\Delta, h\right)</math> 

在這裡<math>A</math>是地表運動的幅度（單位微米），<math>T</math>是體波週期（單位秒），<math>Q\left(\Delta, h\right)</math>是距離函數的修正係數，<math>\Delta</math>是[[震央|震央]]和地震測站夾角（單位角度），<math>h</math>是[[震源|震源]]深度（單位公里）。

一般說來，體波震級是使用週期約1秒的P波，與周期4秒左右的S波以測定遠震(震央在 700 公里以外)的規模<ref name = MIT>[http://ocw.mit.edu/courses/earth-atmospheric-and-planetary-sciences/12-510-introduction-to-seismology-spring-2010/lecture-notes/lec19.pdf Introduction to Seismology], MIT OpenCourseWare</ref><ref>[http://earthquake.usgs.gov/learn/topics/measure.php Measuring the Size of an Earthquake] {{Wayback|url=http://earthquake.usgs.gov/learn/topics/measure.php |date=20150118173606 }}, Earthquake Hazards Program, USGS</ref>。類似於[[里氏地震規模|里氏地震規模]]，體波震級與地震的物理性質沒有直接的物理關連<ref name = MIT></ref>，並且在規模大於<math>m_b</math>6.2即會發生飽和，因此不適合使用於大型地震的規模測量。
