'''热力学循环'''(thermodynamical cycle)是一系列传递[[热量|热量]]并做[[功|功]]的[[热力学过程|热力学过程]]组成的集合，通过[[压强|压强]]、[[温度|温度]]等状态变量的变化，最终使[[热力学系统|热力学系统]]回到初始状态。[[状态量|状态量]]只依赖于[[热力学状态|热力学状态]]，沿热力学循环路径对此类物理量的[[路径积分|路径积分]]结果为零；而像热量和功这样的过程量与循环过程有关，路径积分不为零。[[热力学第一定律|热力学第一定律]]指出在一个循环中输入的净热量总等于输出的净功。过程可重复的特性使得系统能够被连续操作，从而热力学循环是热力学中一个很重要的概念。在实际应用中，热力学循环经常被看作是一个[[准静态过程|准静态过程]]并被当作实际[[热机|热机]]和[[热泵|热泵]]的工作模型。

[[Image:Stirling_Cycle.png|thumb]]）的P－V图]]

在[[P－V图|P－V图]]上热力学循环可表示为一个闭合曲线，P－V图的Y轴表示压强，X轴表示体积，则闭合曲线所包围的面积等于过程所做的功<math>W\,</math>：

: <math> \text{(1)} \qquad W = \oint P \ dV </math>.

这个功在数值上等于传入系统的热量<math>Q\,</math>：

: <math> \text{(2)} \qquad W = Q = Q_{in} - Q_{out} </math>.

方程（2）的表达式显示热力学循环类似于一个[[等温过程|等温过程]]，不过在循环过程中系统的[[内能|内能]]是变化的，只是当每一次循环结束时系统内能会回到初始值。

如果循环过程在P－V图上是沿顺时针方向进行的，这个循环代表着一个[[热机|热机]]，此时的输出功是正值；如果是沿逆时针方向进行的，则它代表这一个[[热泵|热泵]]，此时的输出功是负值。

== 概述 ==

两种主要的热力学循环类型是热机循环和热泵循环。热机循环将输入的部分热量转化为输出的机械功，而热泵循环通过输入的机械功将热量从低温传向高温。完全由准静态过程组成的循环能够通过控制过程的流向来作为热机或热泵循环使用。在P-V图或[[温熵图|温熵图]]上，顺时针和逆时针方向分别代表着热机和热泵循环。

=== 热机循环 ===

[[Image:Carnot_heat_engine_2.svg|200px]]

{{Main|热机}}

热机循环是热机工作的基本原理，这种循环方式为当前世界上大部分的[[发电站|发电站]]提供能量来源，也为几乎所有的[[机动车|机动车]]提供动力。热机循环按照它们所采用的热机模型可进一步分类，[[内燃机|内燃机]]中最常见的热机循环是[[奥托循环|奥托循环]]（常称做[[四冲程循环|四冲程循环]]），[[柴油机|柴油机]]中最常见的是[[迪塞尔循环|迪塞尔循环]]。[[外燃机|外燃机]]中使用的循环方式还包括采用燃气轮机方式工作的[[布雷顿循环|布雷顿循环]]，以及采用[[汽轮机|汽轮机]]方式工作的[[兰金循环|兰金循环]]。

[[Image:Stirling_Cycle.png|thumb]]

这里在P－V图上举例说明如何计算一个由四个热力学过程构成的热机循环所做的机械功：

: <math> \text{(3)} \qquad W = W_{1\to 2} + W_{2\to 3} + W_{3\to 4} + W_{4\to 1} </math>
: <math> W_{1\to 2} = \int_{V_1}^{V_2} P \, dV, \, \, </math> 系统做正功
: <math> W_{2\to 3} = \int_{V_2}^{V_3} P \, dV, \, \, </math> 当<math>V_2 = V_3\,</math>时系统不做功
: <math> W_{3\to 4} = \int_{V_3}^{V_4} P \, dV, \, \, </math> 系统做负功
: <math> W_{4\to 1} = \int_{V_4}^{V_1} P \, dV, \, \, </math> 当<math>V_4 = V_1\,</math>时系统不做功

在过程4->1以及2->3中，如果体积没有变化，则方程（3）简化为

: <math> \text{(4)} \qquad W = W_{1\to 2} + W_{3\to 4} </math>

=== 热泵循环和制冷循环 ===
{{Main|热泵}}

热泵循环和制冷循环是[[热泵|热泵]]和[[冰箱|冰箱]]的理论模型。两者的差别在于热泵的用途是保持一块区域的温度而冰箱则是使之降温。最常见的制冷循环是采用[[制冷剂|制冷剂]]的[[相变|相变]]进行的[[蒸气压缩循环|蒸气压缩循环]]。[[吸收制冷循环|吸收制冷循环]]是另一种循环方式，它不将制冷剂气化，而是将其吸收。气体制冷循环包括[[逆向布雷顿循环|逆向布雷顿循环]]和[[林德－汉普逊循环|林德－汉普逊循环]]。

== 热力学循环的类型 ==

理论上一个热力学循环由三个或多个热力学过程组成（通常为四个），这些过程可以为：

* [[等温过程|等温过程]] （温度恒定，即使伴随有吸热或放热过程）
* [[等压过程|等压过程]] （压强恒定）
* [[等容过程|等容过程]] （体积恒定）
* [[绝热过程|绝热过程]] （系统与外界无热交换）
** [[等熵过程|等熵过程]]（[[可逆绝热过程|可逆绝热过程]]） （系统与外界无热交换，同时熵保持恒定）
* [[等焓过程|等焓过程]] （焓保持恒定）

=== 卡诺循环 ===
{{main|卡诺循环}}

卡诺循环由完全[[可逆过程|可逆]]的等熵压缩和膨胀过程，以及伴随有吸热和放热的等温过程组成。卡诺循环的[[热机效率|热机效率]]只依赖于发生热传递的两个热库的[[热力学温度|热力学温度]]，对于一个循环周期，卡诺热机的效率为

: <math>\eta=1-\frac{T_L}{T_H}</math>

其中<math>{T_L}</math>是循环过程中的最低温度（低温热库的温度），<math>{T_H}</math>是最高温度（高温热库的温度）。对于卡诺制冷循环，热泵的[[性能系数|性能系数]]为

: <math>\ COP = 1+\frac{T_L}{T_H - T_L}</math>

对于一个制冷机，性能系数为

: <math>\ COP = \frac{T_L}{T_H - T_L}</math>

[[热力学第二定律|热力学第二定律]]指出，任何热力学循环设备的效率和性能系数都不可能高于卡诺循环的效率。

== 状态函数和熵 ==

如果<math>Z\,</math>是一个热力学的状态函数，则<math>Z\,</math>经过一个热力学循环后保持不变：

: <math> \oint dZ = 0 </math>.

熵作为一个状态函数，定义为

: <math> S = {Q \over T} </math>

从而有

: <math> \Delta S = {\Delta Q \over T} </math>,

可见对于任意循环过程都有

: <math> \oint dS = \oint {dQ \over T} = 0</math>

意味着经过一个循环系统的熵增为零。
