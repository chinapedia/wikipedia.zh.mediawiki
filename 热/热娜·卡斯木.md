'''热娜·卡斯木'''（{{bd|1962年|7月26日|}}），生于[[新疆维吾尔自治区|新疆维吾尔自治区]][[乌鲁木齐市|乌鲁木齐市]]<ref name="a1"/>，[[维吾尔族|维吾尔族]]，[[中华人民共和国|中华人民共和国]]医学教授、[[第十一届全国人民代表大会|第十一届全国人民代表大会]]新疆地区代表<ref>{{cite web|author=新华网|title=第十一届全国人大代表|url=http://news.xinhuanet.com/ziliao/2008-03/03/content_7708230.htm?node=36628|publisher=新华网|accessdate=2014-01-07}}</ref>。[[国务院特殊津贴|国务院特殊津贴]]专家、[[新世纪百千万人才工程|新世纪百千万人才工程]]入选者<ref name="a2">{{Cite web |url=http://www.xjmu.edu.cn/info/1058/2161.htm |title=热娜·卡斯木教授荣获全国优秀教育工作者称号|accessdate=2019-04-21 |author= |date=2014-09-15 |work= |publisher=新疆医科大学网站|language=Zh-hans }}</ref>。

1980年，考入[[新疆医学院|新疆医学院]]（今[[新疆医科大学|新疆医科大学]]）药学系。1984年，毕业留校任教。1985年，加入中国共产党<ref name="a1"/>。1994年3月，公费派往日本留学。毕业于日本[[富山医科药科大学|富山医科药科大学]]和汉药研究所药学专业。1998年7月，获得博士学位，10月回国，返校执教<ref name="a2"/>。2002年，时任[[新疆医科大学|新疆医科大学]]科研开发处副处长<ref name="a1">{{Cite web |url=http://www.ts.cn/GB/channel3/18/200204/15/17978.html |title=用知识塑造美丽人生 记新疆第一位药学女博士热娜·卡斯木|accessdate=2019-04-21 |author=通讯员：遭军 |date= |work= |publisher=天山网，来源：《新疆日报》 2002年4月15日|language=Zh-hans }}</ref>。2008年起担任全国人大代表<ref>{{cite news|title=全国人大代表信息-热娜·卡斯木|url=http://www.npc.gov.cn/delegate/viewDelegate.action?dbid=112650|accessdate=2013-04-05|newspaper=全国人大网}}</ref>。2014年，获[[中华人民共和国教育部|中国教育部]]颁发的[[全国优秀教育工作者|全国优秀教育工作者]]称号<ref name="a2"/>。
