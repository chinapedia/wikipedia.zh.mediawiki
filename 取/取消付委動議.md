'''取消付委動議'''（{{lang|en|discharge a committee}}）是一項[[議事程序|議事程序]]，旨在取消[[委員會|委員會]]調查某項[[主動議|主動議]]的權力，屬於[[重議類動議|重議類動議]]的一種。該動議必須在委員會做出結案報告前提出，委員會也可提出此動議取消次級委員會。根據《[[羅伯特議事規則|羅伯特議事規則]]》，取消付委動議需要[[附議|附議]]，可以討論但不可以[[修正動議|修正]]、重提，表決時需獲得三分之二與會者支持方為通過<ref name="RONR">{{Cite book|author=Henry M. Robert|coauthor=Daniel H. Honemann; Thomas J. Balch; William J. Evans; Sarah Corbin Robert|title=''Robert's Rules of Order: Newly Revised''|publisher=Da Capo Press|edition=10|language=en|isbn=073820384X|pages=299-300|date=2010年11月}}</ref>。

《{{Link-en|議事程序準則|The Standard Code of Parliamentary Procedure}}》中沒有類似的動議案，[[合議性團體|合議性團體]]隨時可以在通過半數支持表決後，將一件已經[[付委動議|付委]]的議案抽回自行討論或交給另一個委員會處理<ref>{{Cite book|author=Sturgis, Alice|title=''The Standard Code of Parliamentary Procedure''|publisher=McGraw-Hill|edition=4|language=en|isbn=0071365133|pages=57|date=2000年12月}}</ref>。
