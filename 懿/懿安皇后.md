'''懿安皇后'''（{{bd|8世纪||851年||catIdx=G}}）<!-- 生年无具体记载。如有发现，请提供来源。 -->，[[郭姓|郭姓]]，為[[唐宪宗|唐宪宗]][[嫡妻|嫡妻]]，[[唐穆宗|唐穆宗]]生母。郭氏一生歷經唐朝七代皇帝治世，其中五朝極盡尊貴，是所謂'''七朝五尊'''。

== 家世 ==
郭氏為[[唐朝|唐朝]]名將[[郭子儀|郭子儀]]孫女、[[郭曖|郭曖]]与[[昇平公主|昇平公主]]长女，亦是[[唐代宗|唐代宗]]的外孙女、[[唐德宗|唐德宗]]的外甥女與长孙媳、[[唐顺宗|唐顺宗]]的表妹兼媳婦，[[唐宪宗|唐宪宗]]的表姑與嫡妻。

== 早年 ==

郭氏的生年没有记载。她的家世顯赫，被[[唐德宗|唐德宗]]<ref>《[[旧唐书|旧唐书]] 卷一百二十·列传第七十》......贞元中，帝为皇孙广陵郡王纳暧女为妃。暧，贞元十六年七月卒......</ref>選為[[皇太子|皇太子]][[唐順宗|李誦]]（唐順宗）長子廣陵郡王李純的[[王妃|王妃]]<ref>《[[新唐书|新唐书]] 列传第二 后妃下》文“宪宗为广陵王，娉以为妃。”</ref>。两人成婚的时间，当是在[[貞元_(唐朝)|贞元]]九年（793年）十一月丁卯<ref>《[[资治通鉴|资治通鉴]]·唐纪五十·卷第二百三十四》</ref>。举行婚礼时，十五岁的廣陵郡王李純“亲临主家，纳迎如礼”<ref>《[[全唐文|全唐文]] 卷七百三十八》唐故银青光禄大夫检校左散骑常侍兼宫苑闲厩使驸马都尉郭公墓志铭......初宪宗居邸时，亲临主家，纳迎如礼。及今上即位，......</ref>。又因生母身分高貴（為順宗姑母），再加上父、祖皆有功於皇室，因此郭氏本身也頗得順宗異寵。貞元11年（795年），郭氏生下李纯的第三子李宥，即[[唐穆宗|唐穆宗]]。后又生一子绛王[[李悟_(绛王)|李寮]]（后改名李悟）及一女，即[[岐陽莊淑公主|岐陽莊淑公主]]，下嫁重臣[[杜佑|杜佑]]（《[[通典|通典]]》作者）之孫[[杜悰|杜悰]]。

[[貞元_(唐朝)|贞元]]二十一年（805年）初，唐顺宗即位后，三月，丈夫[[唐宪宗|李纯]]被立为太子。夏四月，儿子李宥、李寮与其他兄弟同封[[郡王|郡王]]<ref>《[[新唐书|新唐书]]·本纪第十四·顺宗 宪宗上》三月......癸巳，诏册广陵郡王淳为皇太子，改名纯......夏四月......庚戌，封太子男宁、宽、宥、察、寰、寮等六人为郡王，并食邑三千户......五月己巳......承徽王氏、赵氏可昭仪，崔氏、杨氏可充仪，王氏可昭媛，王氏可昭容，牛氏可修仪，张氏可美人......八月丁酉朔......诰立良娣王氏为太上皇后，良媛董氏为太上皇德妃......</ref>。没有史料提及郭氏获得了[[太子妃|太子妃]]的册封。当时，朝廷政局激变，唐顺宗在同年八月被逼退位。李纯的母亲[[莊憲皇后|良娣王氏]]，也没有来及册封为皇后或妃嫔。郭氏可能同样未获[[太子妃|太子妃]]的册封。

[[元和_(唐朝)|元和]]元年（806年），李純[[即位|即位]]，是為[[唐憲宗|憲宗]]。秋七月乙亥，郭氏以太子嫡妻身分卻只被冊為[[貴妃|貴妃]]，而非[[皇后|皇后]]（但為實際上的後宮之首，唐書言：上惑于內寵，不欲其立，遂藉口武霍）。她的儿子亦没有获封为[[太子|太子]]。元和四年（809年），宪宗的[[庶长子|庶长子]]邓王[[惠昭太子|李宁]]被册为太子。六年，太子李宁逝世。元和七年（812年）十月，李宥超越次兄被立为皇太子，改名恒。元和八年（813年）12月，百官群奏憲宗冊郭貴妃為[[皇后|皇后]]，但憲宗以郭貴妃出身顯族，恐郭貴妃成為皇后，將不容許憲宗有後宮之寵，因而婉拒百官之請。

== 皇太后生涯 ==
憲宗在位15年後暴崩，其子繼位，是為[[唐穆宗|穆宗]]。穆宗即位後，郭貴妃[[母憑子貴|母憑子貴]]升為[[皇太后|皇太后]]，又對郭太后親族有一番封贈。郭太后居於[[興慶宮|興慶宮]]內，穆宗每月朔望定期朝見，還曾隨侍郭太后遊幸[[驪山|驪山]]，對郭太后相當孝順。穆宗在位四年，因服用丹藥駕崩。穆宗崩後，宮中盛傳郭太后將效法[[武則天|武則天]][[臨朝稱制|臨朝稱制]]，郭太后大怒，說：「效法武氏？今太子雖幼，尚可選重德之人為輔臣，與我又何干？」於是穆宗長子即位，是為[[唐敬宗|唐敬宗]]。

== 太皇太后时期 ==
敬宗即位後，尊生母[[恭僖皇后|王氏]]為[[皇太后|皇太后]]、祖母郭太后為[[太皇太后|太皇太后]]。[[寶曆_(唐敬宗)|寶曆]]2年（826年），敬宗被宦官[[刘克明_(唐朝)|刘克明]]所殺，內外震驚，刘克明又擁立郭太皇太后的幼子絳王李悟為[[監國|監國]]，事败，李悟遇害。於是郭太皇太后下詔迎敬宗之弟即位，是為[[唐文宗|唐文宗]]。因文宗尊生母[[貞獻皇后|蕭氏]]為皇太后，加上郭太皇太后與敬宗母王太后，宮中共有三位太后，稱「三宮太后」。

郭氏為憲宗貴妃、穆宗母，又是敬宗、文宗的祖母，即使敬、文兩帝各自尊生母為皇太后，但郭氏一直受到諸帝至誠的尊養，在三宮太后中也以郭太皇太后為最尊。文宗駕崩後其弟[[唐武宗|武宗]]即位，而宮中為分辨三宮太后，稱敬宗母王氏為[[恭僖皇后|義安太后]]，文宗母為[[貞獻皇后|積慶太后]]，郭氏仍稱太皇太后，且三宮太后仍繼續受到武宗孝養。

武宗喜遊樂，又讓五坊小兒自由出入宮禁。一天，武宗朝見郭太皇太后時，向她問曰：「如何成為盛世天子？」郭太皇太后對曰：「勤於政事，聽取百官意見，不拒諫言，不納讒言，如此能為盛世天子。」於是，武宗漸漸專心為政，疏遠五坊小兒與遊樂，使唐朝國勢稍有起色。

== 宣宗即位後 ==
武宗同父親穆宗一样，因相信[[長生不老|長生不老]]之術而服食丹藥，最後中毒而死。武宗崩後，憲宗十三子光王李怡被擁立為[[皇太叔|皇太叔]]，於武宗[[梓宮|梓宮]]前即位，是為[[唐宣宗|唐宣宗]]。其兄[[唐穆宗|穆宗]]乃郭氏為太子妃時與太子[[李純|李純]]（唐憲宗）所生，故為嫡出。穆宗三子（[[唐敬宗|敬宗李湛]]、[[唐文宗|文宗李昂]]、[[唐武宗|武宗李瀍]]）亦各有子嗣，據[[宗法|宗法]]本可繼承大統。然而卻因宣宗得到宦官勢力擁戴，以庶奪嫡，登其大寶。郭太后親支皇嗣子孫亦從此喪失入承大統的機會，更被迫與當年曾侍候自己的宮女（宣宗生母鄭氏）平起平坐，難免對宣宗母子心存介蒂，因此宣宗相較於先前諸帝，對郭太皇太后的孝養較為疏薄，也引起她的不悅。

[[大中_(唐宣宗)|大中]]5年（851年）3月，郭太皇太后登勤政樓，因對宣宗的禮薄有不滿，竟欲跳樓自盡，幸被侍從阻止。宣宗聽聞郭太皇太后跳樓一事非常憤怒，當晚，郭太皇太后在興慶宮崩逝<ref>裴廷裕《东观奏记》卷一：時居興慶宮,一日,與二侍兒同升勤政樓,依衡而望,便欲殞於樓下,欲成上過。左右急持之,即聞於上,上大怒。其夕,太后暴崩,上志也。</ref>，[[諡號|諡號]]'''懿安皇后'''。郭太皇太后死後，因與宣宗生母[[孝明皇后_(唐憲宗)|鄭太后]]之間有舊怨，宣宗欲將之葬於[[景陵|景陵]]外園，且不祔憲宗廟，雖有太常官[[王皞|王皞]]力諫，認為郭氏歷經七朝，五朝皆為皇室母儀代表，應當配饗憲宗，卻被宣宗所拒。多年之後宣宗駕崩，[[唐懿宗|唐懿宗]]即位，王皞再度請命，懿安皇后郭氏這才配享憲宗。

== 去世 ==
大中二年（848年）六月初一日，郭氏与几名侍从登临勤政楼，准备跳楼自尽，左右侍从赶紧拦住。唐宣宗闻知此事很不高兴，当天晚上郭氏在兴庆宫突然去世。有关部门议定郭氏的谥号为懿安皇后，安葬在景陵外园。 
太常官王暤奏请将郭氏合葬于景陵，以其神主祔祭于唐宪宗庙室，唐宣宗不高兴，命宰相白敏中责让王暤。王暤反驳说：“郭后本是唐宪宗在东宫时的元妃，作为儿媳侍奉唐顺宗，经历五朝皆为天下母后，不应再有异议。”白敏中极为恼怒，周墀也加以指责，王暤始终不肯屈从，周墀便说：“王暤确是孤傲耿直。”不久即将王暤贬职为句容县令。 
唐懿宗咸通年间，王暤回朝再任礼官，重申坚持先前的看法，终于使朝廷下诏将郭氏神主祔祭于太庙。 

== 電視影劇 ==
在[[香港電視廣播有限公司|香港電視廣播有限公司]]所拍攝的[[古裝|古裝]][[電視劇|電視劇]]'''[[宮心計|宮心計]]'''中，由[[謝雪心|謝雪心]]飾演懿安皇后郭氏一角。劇中省略郭氏於唐朝順宗、憲宗、穆宗、敬宗、文宗年間的部分，著重在武宗、宣宗在位期間，郭氏與奸宦[[馬元贄|馬元贄]]、[[孝明皇后_(唐憲宗)|鄭太妃]]、[[唐宣宗|光王]]之間的鬥智與後宮間的生活。而在光王登基為宣宗後，郭太后多次施計對付皆不果，最後憤恨的郭太后在登勤政樓自盡不成後，被鄭太后以失心疯为由送往[[上陽宮|上陽宮]]幽閉，並被[[馬元贄|馬元贄]]失手推柱撞死；和史實中不同的是，郭氏在宣宗登基後仍為太皇太后，不同劇中降為「郭（皇）太后」、郭氏是在興慶宮逝世的，至於死因史書未提及。
