'''叶世克总督纪念碑'''（{{lang-de|Gouverneur Jaeschke Denkmal}}）是位于[[中国|中国]][[青岛市|青岛市]][[市南区|市南区]][[太平路_(青岛)|太平路]][[青岛路_(青岛)|青岛路]]路口的一栋纪念建筑，建成于德租时期的1903年。该碑历经变迁，最终于1967年完全消失<ref name=san>{{cite web|url=http://wanbao.qingdaonews.com/html/2008-04/27/content_650811.htm|title=三座消亡的殖民纪念碑|date=2008-04-27|work=王栋|publisher=青岛晚报|access-date=2017-03-26|archive-url=https://web.archive.org/web/20160304032929/http://wanbao.qingdaonews.com/html/2008-04/27/content_650811.htm|archive-date=2016-03-04|dead-url=yes}}</ref><ref Name=ming>{{cite book|author=王栋|title=《青岛影像 1898-1928：明信片中的城市记忆》|year=2017|publisher=中国海洋大学出版社|location=青岛|isbn=978-7-5670-1113-7}}</ref>。

==历史==
===叶世克其人===
[[File:Paul_Jaeschke.jpg|thumb]]
{{link-de|叶世克|Paul Jaeschke|叶世克}}（Paul Jaeschke）为第二任胶澳总督，1851年8月4日生于[[弗罗茨瓦夫|布雷斯劳]]（Breslau）的一个中产阶级家庭，1868年加入普鲁士海军，随后一路升迁，于1894年被授予上校军衔。1895年4月，叶世克被任命为{{link-de|皇帝号巡洋舰|SMS Kaiser (1874)|皇帝号巡洋舰}}的舰长，并率领该舰加入[[东亚分舰队_(德国)|东亚分舰队]]，且曾于次年访问[[胶州湾|胶州湾]]。1896年任海军最高统帅部外事局长官<ref name=jae>{{cite web |url=http://www.tsingtau.org/jaeschke-paul-1851-1901-gouverneur/|title=Jaeschke, Paul(1851-1901),Gouverneur|date=2009-03-01|work=Wilhelm Matzat}}</ref><ref name=zong>{{cite web |url=http://blog.sina.com.cn/s/blog_488832200100068k.html|title=总督叶世克|date=2006-10-16|work=李明|publisher=}}</ref>。

1898年10月10日，因胶澳总督府中国事务专员[[单威廉|单威廉]]（Wilhelm Schrameier）在克拉拉湾（Clara Bucht，今[[汇泉湾|汇泉湾]]）附近违规购地（位置在今文登路小学东侧）一事，总督[[罗绅达|罗绅达]]（Carl Rosendahl）遭到德国国内报界广泛指责，海军部国务秘书[[阿尔弗雷德·冯·提尔皮茨|阿尔弗雷德·冯·提尔皮茨]]（Alfred von Tirpitz）任命叶世克代替罗绅达为胶澳总督<ref name=san/><ref name=zong/>。1899年2月19日，叶世克到达青岛。叶世克任内曾有多项举措，如在总督府议会中增加三个平民席位、刊登政府公报等<ref name=zong/>，青岛的城市建设也在其任内蓬勃发展，而叶世克最受德国政府赏识的举措是其对山东境内平民骚乱的强硬举措。1899年至1900年的[[义和团运动|义和团运动]]期间，为保护德国在山东的传教士和教堂，同时解决[[山东铁路公司|山东铁路公司]]修筑[[胶济铁路|胶济铁路]]时与居民发生的冲突，叶世克先后多次派兵进入[[胶州|胶州]]、[[即墨|即墨]]、[[高密|高密]]、[[日照|日照]]等中国城镇，镇压义和团。他的措施有效遏制了义和团在青岛及附近地区的蔓延，因此获得了德皇和海军方面的褒奖<ref name=san/><ref name=zong/>。

1901年1月27日，叶世克突然因感染[[斑疹伤寒|斑疹伤寒]]，殁于[[青岛督署医院|督署医院]]。德国军方在追授他为海军少将的同时，对其高度评价，称他是最严峻时刻的一位坚强有力而且远见卓识的领袖人物<ref name=zong/>。为纪念叶世克，德国海军部和青岛的德国侨民决定建造一座纪念碑<ref Name=ming/>。

===建设之初===
[[File:Bundesarchiv_Bild_134-B2435,_Tsingtau,_Denkmal_Jaeschke,_Bau.jpg|thumb]]
青岛的德国侨民组织成立了一个临时委员会，专门负责为叶世克建造纪念碑的预算和施工等工作<ref name=san/>。纪念碑的位置最终被选在了威廉皇帝海岸（Kaiser-Wilhelm-Ufer，今[[太平路_(青岛)|太平路]]）南侧的防波堤上，由总督府建筑师[[彼得·贝尔纳茨|彼得·贝尔纳茨]]（Peter Bernatz）设计，并于1903年3月《[[胶澳租借条约|胶澳租借条约]]》签订六周年时竣工落成。纪念碑建设在威廉皇帝海岸与威廉街（Wilhelmstrasse，今[[青岛路_(青岛)|青岛路]]）路口南侧，前海大堤的一个半圆形凸出部分上，沿威廉街与[[胶澳总督府|胶澳总督府]]相望，是一处经典的街道对景。纪念碑是一座全花岗岩的六角形石塔，通高约为16米<ref Name=ming/>。塔身中下部的石制回廊形成了一个碑亭，塔身镶嵌有12块精心打制的铜牌，详细记叙了德国占领胶州湾的全过程和《胶澳租借条约》内容。叶世克总督的画像、生平和业绩，也被记录在其中<ref name=san/>。

===历经变迁===
[[File:青岛山海重光碑.jpg|thumb]]
1914年11月，日本[[青岛战役|占领青岛]]。叶世克纪念碑被改造为“[[大日本帝国|大日本帝国]]占领青岛纪念碑”，12块铜牌被占领军拆下后运回日本国内展览，据说后来被收藏于[[东京国立博物馆|东京国立博物馆]]<ref name=san/><ref name=ye>{{cite web |url=http://arc.dailyqd.com/2015-01/13/content_198624.htm|title=叶世克纪念碑到山海重光碑：青岛历史见证|date=2015-01-13|work=张兆新|publisher=青岛日报}}</ref><ref name=shi>{{cite web |url=http://www.taidongzhen.com/HTML/News/Qingdao/qdrw/2016/0816/60221.html|title=叶世克曾任胶澳总督 太平路建纪念碑|date=2016-08-16|work=桂纶|publisher=城市信报}}</ref>。

1923年12月，为纪念[[北洋政府|北洋政府]]接收青岛一周年，[[胶澳商埠|胶澳商埠]]督办公署将纪念塔改建为“青岛接收纪念亭”<ref name=san/>，1924年竣工<ref Name=yizuo>{{cite web|url=http://www.dailyqd.com/arc/2014-04/18/content_50196_all.htm|title=一座纪念碑的命运（上篇）|date=2014-04-18|work=刘宗伟|publisher=青岛日报 青报网}}</ref>。纪念碑碑铭被重新撰写为4字为韵、共112韵的《胶澳接收纪念碑铭》，详述了接收的前后事宜<ref Name=ming/><ref name=ye/>，作者为曾任青岛接收行政与公产两委员会主任委员、青岛行政接收副主任兼代委员长的[[王大桢|王大桢]]<ref Name=yizuo/>。

1938年1月，日本再次侵占青岛后，将原碑拆毁并改建为“建设东亚新秩序纪念塔”，1939年12月11日奠基，至1940年1月10日落成<ref Name=ming/><ref Name=yizuo/>。1945年8月[[日本投降|日本投降]]后，[[青岛市政府|青岛市政府]]在次年9月为纪念[[抗战|抗战]]胜利一周年，将此塔改为“青岛光复纪念塔”，塔身镶嵌有[[青岛市市长|青岛市市长]][[李先良|李先良]]手书的“山海重光”四字石碑，及记录抗战功绩的多个石碑<ref name=ye/><ref name=shi/>。如今仅剩“重光”二字残碑放置在[[青岛市博物馆|青岛市博物馆]]后门左侧的草地上。

1950年代，该碑改称“中苏友好纪念碑”。[[文革|文革]]时期的1967年夏季，该碑被[[中国海洋大学|山东海洋学院]]与[[青岛市第二中学|青岛二中]]的[[红卫兵|红卫兵]]捣毁<ref name=san/><ref name=ye/>。[[青岛市革委会|青岛市革委会]]曾于1967年计划在该碑原址树立[[毛泽东塑像|毛泽东塑像]]。另据艺术家徐立忠称，革委会曾计划在[[胶澳总督府|革委会办公楼]]前的广场上竖立毛泽东塑像，为方便将石料自海路沿青岛路运至塑像雕刻现场，将原纪念碑拆除。后因[[毛泽东|毛泽东]]本人要求停止设立其塑像，原计划未能实施。纪念碑原址后改设一[[毛语录|毛语录]]牌，语录牌于八十年代被拆除<ref Name=ming/>。1990年代以后，青岛市“[[两会|两会]]”等场合上曾经多次有提议在该处重建光复纪念碑或另建一纪念性建筑，均因各种原因未得实施。纪念碑基址现为一处长有树木的草地。
