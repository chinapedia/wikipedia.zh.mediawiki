'''叶剑英'''（{{bd|1897年|4月28日|1986年|10月22日|叶剑英}}），原名'''宜伟'''，字'''沧白'''，[[汉族|汉族]]，[[廣東|廣東]][[梅县|梅县]][[客家人|客家人]]，[[中国共产党|中国共产党]][[革命家|革命家]]、[[政治家|政治家]]、[[中国人民解放军军事家|军事家]]，[[中华人民共和国|中华人民共和国]]开国和第二代主要[[党和国家领导人|领导人]]之一，[[中国人民解放军|中国人民解放军]]和[[中華人民共和国|中華人民共和国]]的创建者和締造者之一，[[南昌起義|南昌起義]]的領導人之一。

叶剑英早年曾任[[黄埔军校|黄埔军校]]教授部副主任，[[国民革命军第四军|国民革命军第四军]]参谋长，参与[[北伐战争|北伐战争]]。1927年12月，组织[[广州起义_(1927年)|广州起义]]。[[第一次国共内战|第一次国共内战]]时期，任[[中华苏维埃共和国中央革命军事委员会|中革军委]]总参谋长、[[红一方面军|红一方面军]]参谋长等职务，参与历次反围剿战役及[[长征|长征]]。[[中国抗日战争|抗日战争]]时期，任[[八路军|八路军]]参谋长、中央军委参谋长。[[第二次国共内战|第二次国共内战]]时期，任中央军委副总参谋长、中共中央后委书记、华北军政大学校长兼政治委员。1949年1月，任北平聯合辦事處主任，為接管北平做了大量的工作。同年參加中共代表團，同[[中华民国政府|南京國民政府]]代表團進行[[国共和谈|和平談判]]。

[[中華人民共和國|中華人民共和國]]成立後，歷任[[中共中央華南分局|中共中央華南分局]][[第一书记_(中国)|第一書記]]、[[廣東軍區|廣東軍區]][[司令員|司令員]]兼[[政治委員|政治委員]]、[[中央人民政府委员会|中央人民政府委員]]、[[廣東省人民政府|廣東省人民政府]]主席、[[廣州市市長|廣州市市長]]，[[中央人民政府人民革命軍事委員會|中央人民政府人民革命軍事委員會]]副主席、[[中国人民解放军武装力量监察部|中国人民解放军武装力量监察部]]部长、[[军事科学院|军事科学院]]院长兼政委等职务，被授予[[中华人民共和国元帅|中华人民共和国元帅]]军衔，并在[[文化大革命|文化大革命]]期间卷入“[[二月逆流|二月逆流]]”案。[[林彪事件|林彪事件]]后，他担任[[中共中央政治局常委|中共中央政治局常委]]、[[中共中央副主席|中共中央副主席]]、[[中共中央軍委副主席|中共中央軍委副主席]]、[[中华人民共和国国防部部长|国防部部长]]。

1976年[[毛泽东逝世|毛泽东逝世]]后，他与国务院总理[[华国锋|华国锋]]、[[中共中央办公厅|中共中央办公厅]]主任[[汪东兴|汪东兴]]等人联手策划[[怀仁堂事变|怀仁堂事变]]，逮捕[[四人帮|四人帮]]成员。此后担任[[全国人大委员长|全国人大委员长]]（[[中国国家主席存废之争|廢除國家主席]]時期相當於[[中华人民共和国国家元首列表|國家元首]]）等職。1986年10月22日，叶剑英在[[北京|北京]]逝世，骨灰安葬于[[广州|广州]][[红花岗四烈士墓|红花岗]][[广州起义烈士陵园|烈士陵园]]。

== 1897-1927 出生与成长 ==
{{See also|东山中学|黄埔军校|国民革命军东征|北伐战争}}

[[清|清]][[光绪|光绪]]二十三年（1897年）3月27日，叶剑英出生于[[广东省|广东省]][[嘉应直隶州|嘉应直隶州]]（今[[梅州市|梅州市]][[梅县区|梅县区]]）雁洋堡下虎形村一个小商人家庭里{{r|A|page=1}}。1908年，11岁的叶剑英家境稍微好转，父母将他送到离[[雁洋|雁洋]]10多里路的丙村[[三堡学堂|三堡学堂]]住校读书{{r|A|page=4-5}}。受到辛亥革命影响，立志报效国家{{r|A|page=7}}。

1912年1月，叶剑英以“最优等”的成绩从三堡学堂毕业考入梅县务本中学，这年冬天梅县务本中学与其他三所中学合并受为官办。新校长是一个守旧人物，叶剑英因此联合100多名学生离开学校迁到梅城状元桥畔东山书院{{r|A|page=7}}，不久在进步人士、学生家长、乡绅和爱国华侨的支持下在县城东山脚下建起新校[[东山中学|东山中学]]<ref>{{cite book|author=广东梅县教育志编纂组编|title=梅县教育志|publisher=广东梅县教育志编纂组|year=1990|pages=9}}</ref>。叶剑英随即进入东山中学就读。在此期间，接触了一些宣传社会变革的进步书刊。连任两届学生会会长{{r|A|page=8-9}}。1915年秋，叶剑英从东山中学肄业，在与雁洋堡相邻的横山新群小学任教半年{{r|A|page=10-11}}。同年冬，随父赴南洋，次年回国{{r|A|page=11}}。

[[File:Ye_Jianying4.jpg|thumb]]

1917年夏，叶剑英入[[云南讲武堂|云南讲武堂]]{{r|A|page=12}}。毕业后，追随[[孙中山|孙中山]]投身民主革命{{r|A|page=17}}。1920年夏，参加了孙中山组织的驱逐[[舊桂系|桂系]]军阀之役{{r|A|page=18}}。翌年10月，随孙中山出巡广西<ref>{{cite book|title=将帅从这里起步|author=张植信等编著|year=1988|pages=103|location=沈阳|publisher=辽宁人民出版社|isbn=7-205-00393-8}}</ref>。1922年6月，[[陈炯明|陈炯明]]叛变时，任海军陆战队营长的叶剑英率部护卫孙中山脱险，同陈炯明军作战<ref>{{cite book|author=范硕|title=叶剑英在关键时刻|location=沈阳|publisher=辽宁人民出版社|year=2011|pages=12-13|isbn=978-7-205-06966-7}}</ref>。嗣后，前往福建任东路讨贼军第八旅参谋长，随军入粤讨伐陈炯明。言岭关大捷，名震一时{{r|A|page=22}}。

1924年初，任建国粤军第二师参谋长，受[[廖仲恺|廖仲恺]]邀请{{r|A|page=25}}，参与筹建[[黄埔军校|黄埔军校]]，任教授部副主任<ref>{{cite book|author=陈宇|title=黄埔军校年谱长编|location=北京|publisher=华文出版社|year=2014|pages=24|isbn=978-7-5075-4221-9}}</ref>。在黄埔执教期间，接触了许多共产党人，受到影响，逐步接受了[[马列主义|马列主义]]，第一次提出加入[[中国共产党|中国共产党]]的申请，未果{{r|A|page=27-28}}。7月，应[[张民达|张民达]]师长请求，返回二师，率部迎击进犯广州的[[林虎|林虎]]叛军，获胜{{r|A|page=29}}。后奉廖仲恺命，到[[香洲|香洲]]创办粤军第二师独立营，兼任该营营长。配合黄埔军校的教学，培训基层军士{{r|A|page=29}}。10月15日，与张民达指挥二师参加镇压[[广州商团叛乱|广州商团叛乱]]{{r|A|page=31}}。1925年1月，与张民达率二师参加讨伐军阀陈炯明的[[第一次东征|第一次东征]]{{r|A|page=32}}。3月，出任[[梅县|梅县]]县长<ref>{{cite book|author=梅县政府志编纂委员会|title=梅县政府志|publisher=梅县政府志编纂委员会|year=1993|pages=123-124}}</ref>。5月，任建国粤军第二师新编团团长，率新编团参加[[第二次东征|第二次东征]]{{r|A|page=41}}。

1926年1月，叶剑英任[[国民革命军第一军|国民革命军第一军]]第二十师第二团团长，旋即升任第二十师副师长{{r|A|page=42}}。1926年7月，参加[[国民革命军北伐|国民革命军北伐]]，初期任[[国民革命军第一军|国民革命军第一军]][[总预备队|总预备队]]指挥部参谋长，率部参加攻打南昌之役{{r|A|page=44}}。攻克[[南昌|南昌]]后，任[[国民革命军新编第二师|国民革命军新编第二师]]代理师长{{r|A|page=45}}。

== 1927-1937 第一次国共内战时期 ==
{{See also|南昌起义|广州起义|长征|西安事变}}

1927年4月12日蒋介石[[四一二事件|发动政变]]后，叶剑英起草并领衔签发了反蒋通电，并参与策划[[吉安暴动|吉安暴动]]{{r|A|page=47-48}}。随即赴武汉{{r|A|page=49}}，不久被任命为[[国民革命军第四军|国民革命军第四军]]参谋长<ref name="张明金">{{cite book |title = 国民党历史上的158个军 |authors = 张明金、刘立勤 |year = 2007 |publisher = 解放军出版社 |page = 39 |ISBN = 978-7-5065-5387-2}}</ref>。7月，经[[李世安|李世安]]介绍、[[周恩来|周恩来]]同意，秘密加入[[中国共产党|中国共产党]]{{r|A|page=52}}。[[南昌起义|南昌起义]]前，叶剑英得知[[汪精卫|汪精卫]]阴谋加害[[叶挺|叶挺]]、[[贺龙|贺龙]]，立即冒着风险在[[九江|九江]][[甘棠湖|甘棠湖]]一小划子上与他们商量对策，决定叶挺和贺龙指挥的部队迅速向南昌开进{{r|A|page=54-55}}。起义军撤出南昌后，叶剑英极力劝阻了[[张发奎|张发奎]]对起义军的追击{{r|A|page=56}}。8月上旬，兼任第四军军官教导团团长{{r|A|page=58}}。12月11日，叶剑英率领所部教导团参加[[广州起义_(1927年)|广州起义]]，任军事指挥部副总指挥。广州起义失败后，叶剑英潜赴[[香港|香港]]{{r|A|page=73}}。1928年冬，赴苏联[[莫斯科东方大学|莫斯科东方大学]]特别班学习{{r|A|page=75}}。

1930年下半年，叶剑英回国到达上海，与[[刘伯承|刘伯承]]等人翻译了苏联红军的步兵战斗条令和政治工作条例等资料{{r|A|page=78}}。1931年4月，到[[江西|江西]][[中央革命根据地|中央革命根据地]]{{r|A|page=79}}。此前[[第二次反围剿战争|第二次反围剿战争]]即将开始，[[毛泽东|毛泽东]]建议诱敌深入，而[[项英|项英]]反对。叶剑英对此表示支持毛泽东，随即协助指挥第二、三次反围剿战争{{r|A|page=81}}。11月，任[[中华苏维埃共和国中央革命军事委员会|中华苏维埃共和国中央革命军事委员会]]委员兼总参谋长。11月底，向中革军委提出“建设精干的统帅机关”的建议研究确定中国工农红军司令部机关的编制体制，建立健全参谋工作制度{{r|A|page=82}}。1932年10月上旬，任[[中国工农红军学校|中国工农红军学校]]校长兼政委，兼任[[瑞金|瑞金]]卫戍司令员{{r|A|page=85}}。

[[File:Ye_Jianying3.jpg|thumb]]

1933年2月，兼任[[红十九军|红十九军]]军长<ref>{{cite book |author1=林玉涵 |author2=中共福建省委党史研究室 |title = 福建工农红军发展历程 |location = 北京 |publisher = 中央文献出版社 |year=2007 |page = 153 |ISBN = 7-5073-2320-X}}</ref>，指挥东南战区第一、二、三纵队进攻[[清流县|清流县]]城{{r|A|page=90}}。5月，任[[中国工农红军|中国工农红军]]总参谋长兼[[红一方面军|红一方面军]]参谋长，12月，兼任闽赣军区及[[建宁县|建宁]]警备区司令员{{r|A|page=91}}。1934年1月，当选[[中华苏维埃共和国|中华苏维埃共和国]]第二届中央执行委员会委员<ref>{{cite web |url = http://www.rjjng.com.cn/research/2010/1411275444288.html |title = 中华苏维埃共和国中央执行委员会布告 |date = 1934-02-03 |accessdate = 2015-09-24}}</ref>。2月，任中国工农红军副总参谋长{{r|A|page=92}}。4月，任福建军区司令员{{r|A|page=93}}；其后由于[[连城战斗|连城战斗]]失利，调任中革军委第四局局长{{r|A|page=94}}。

1934年10月，随[[中央红军|中央红军]]参加[[长征|长征]]，任军委第一纵队司令员兼政治委员<ref>{{cite book |author=王树增 |title = 长征 |year=2006 |location=北京 |publisher=人民文学出版社|isbn=978-7-02-005798-6|page=76}}</ref>。部队进入广西山区时，他遭轰炸负伤{{r|A|page=98}}。1935年3月，任[[红三军团|红三军团]]参谋长。7月，红一、[[红四方面军|四方面军]]会合后，任红军前敌总指挥部参谋长{{r|A|page=103}}。1935年8月，中共中央政治局在[[毛儿盖|毛儿盖]]开会，决定部队分左、右两路军过[[松潘草地|草地]]北上甘南。率领左路军的[[张国焘|张国焘]]，拒绝执行中共中央的北上方针。叶剑英得悉后，立即将此事报告[[毛泽东|毛泽东]]。随后，中共中央在巴西召开紧急会议，决定迅速率领红一方面军主力北上<ref>{{cite book |page = 286 |author = 武国友 |title = 红军长征全史（第三卷） |url =  |year = 1996 |location =  |publisher = 东北师范大学出版社 |chapter = 红四方面军战史 |ISBN = 7-5602-1842-3 }}</ref>{{NoteTag|中共官方历史多称叶剑英截获张国焘密电，但也有研究表明密电一事缺乏事实根据<ref>{{cite web |url = http://www.21ccom.net/articles/lsjd/lccz/article_20140504105472.html |title = “草地密电”：查无实据，事出有因 |author=冯胜平 |date=2014-05-04 |accessdate=2015-09-24 |publisher = [[共识网|共识网]]|archiveurl=https://web.archive.org/web/20150925120354/http://www.21ccom.net/articles/lsjd/lccz/article_20140504105472.html|archivedate=2015-09-25|deadurl=yes}}</ref><ref>{{Cite journal |url = http://view.news.qq.com/a/20140313/005889.htm |title = 张国焘长征“密电”问题的来龙去脉 |journal = 《[[炎黄春秋|炎黄春秋]]》 |year = 2011 |issue = 1 |author = 夏宇立 |deadurl = yes |archiveurl = https://web.archive.org/web/20150925204902/http://view.news.qq.com/a/20140313/005889.htm |archivedate = 2015-09-25 }}</ref>。}}。

中央红军到达[[陕北|陕北]]后，任红一方面军和军委参谋长{{r|A|page=117}}。1936年，红军发起[[东征战役|东征战役]]，叶剑英指挥中路军作战，钳制国军主力，支援了左、右两路军的军事行动{{r|A|page=119}}。7月，被中共中央委派到[[安塞|安塞]]，进行联合[[东北军|东北军]]的工作{{r|A|page=122}}。9月，被派往西安，积极联络各方面力量<ref>{{cite book |author = 《彭雪枫传》编写组 |title = 彭雪枫传 |year = 2007 |publisher = 当代中国出版社 |ISBN = 978-7-80170-311-8 |page = 124}}</ref>。[[西安事变|西安事变]]爆发后，中共中央在会议后派出代表团，由[[博古|博古]]、[[周恩来|周恩来]]、叶剑英三人赶赴西安，三人做出分工：周恩来负责上層统战和群众工作、博古负责党内工作、叶剑英参加西北联军参谋团工作，最终促成事变和平解决<ref name="bg1">{{cite book |author1 = 吴葆朴 |author2 = 李志英 |title = 秦邦宪（博古）传 |year=2007 |location=北京 |publisher=中共党史出版社 |ISBN = 978-7-80199-6855 |page = 211}}</ref><ref name="Leung2002">{{cite book |author = Pak-Wah Leung |title = Political Leaders of Modern China A Biographical Dictionary |url = http://books.google.com/books?id=lpR8yKWjHRQC&pg=PA3 |date = 2002-01-01 |publisher = Greenwood Publishing Group |ISBN = 978-0-313-30216-9 |page = 4}}</ref>。1937年3月8日，叶剑英与周恩来同国民政府谈判代表[[顾祝同|顾祝同]]、[[贺衷寒|贺衷寒]]、[[张冲_(1900年)|张冲]]会谈。双方就中共承认服从国民党的领导地位、取消苏维埃政府、红军改编为国民革命军以及容许中共在适当时期内公开、红军的编制人员、给养及补充与国军同等待遇等问题达成《“三八”协议》{{r|A|page=137}}。

== 1937-1949 抗日战争与第二次國共內戰时期 ==
{{See also|八路军|中共中央南方局|政治协商会议|军事调处执行部|北平和平解放|北平和谈}}
[[File:1939eightroadarmy.jpg|thumb]]、叶剑英、[[李振亚|李振亚]]、[[李涛_(解放军)|李涛]]、[[边章五|边章五]]、[[薛子正|薛子正]]。]]
1937年8月，叶剑英与周恩来、朱德一起到南京参加蒋介石召开的[[最高国防会议|最高国防会议]]{{r|A|page=142}}。其后红军主力改编为[[国民革命军第八路军|国民革命军第八路军]]，叶剑英任参谋长，被授予国民革命军中将军衔<ref>{{cite book | author = 中国人民解放军历史资料丛书编审委员会 | coauthors =  | title = 八路军·文献 | url =  | date = 1994 | location =  | publisher = 解放军出版社 | id =  | isbn = 7-5065-2325-6 | pages = 19-20 }}</ref><ref>{{cite web|url =http://www.sn.xinhuanet.com/2008-07/30/content_13975439.htm|title =红军改编为八路军初期部队作战序列|date =2008-07-30|accessdate =2015-08-21|publisher =新华网|deadurl =yes|archiveurl =https://web.archive.org/web/20150924103019/http://www.sn.xinhuanet.com/2008-07/30/content_13975439.htm|archivedate =2015-09-24}}</ref>。10月，任八路军驻南京代表{{r|A|page=145}}。12月23日，任[[中共中央长江局|中共中央长江局]]委员，负责军事工作{{r|A|page=149}}。1939年1月，任[[中共中央南方局|中共中央南方局]]常委，协同周恩来在[[国统区|国统区]]做统战工作，宣传中共抗日主张，广泛联络国民政府上层人士，并多次参与同国民政府谈判{{r|A|page=156}}。1939年2月，叶剑英参与创办[[南岳游击干部训练班|南岳游击干部训练班]]，任副教育长，讲授抗日游击战战略战术，宣传持久战思想<ref>{{cite book|author=衡山县政协文史工作委员会|title=衡山文史资料 第2辑|pages=63}}</ref><ref>{{cite book|author1=萧泓|author2=龚格|title=抗日战争中的元帅将军|location=北京|publisher=中央文献出版社|year=2005|pages=116-126|isbn=7-5073-1902-4}}</ref>。1940年3月，叶剑英出席[[蒋介石|蒋介石]]在重庆召开的全国参谋长会议，作了《作战与磨擦问题》的长篇发言，宣讲八路军战绩，驳斥国民政府方面对八路军的攻击{{r|A|page=169-173}}。这一年，出版了《叶剑英抗战言论集》{{r|A|page=180}}。

1941年2月，由于[[皖南事变|皖南事变]]，叶剑英返回[[延安|延安]]{{r|A|page=181}}，任中央军委参谋长兼十八集团军参谋长，协助[[毛泽东|毛泽东]]、[[朱德|朱德]]指挥中共军队作战，并领导制定了一系列加强参谋工作的制度和措施{{r|A|page=182}}。11月，叶剑英兼任中央教育委员会委员、军事学院副院长。1943年6月，国共摩擦又起，叶剑英向中共中央提出以智取胜的政治作战方案，大力开展宣传战{{r|A|page=187-188}}。1944年6、7月间，叶剑英先后向在延安的[[中外记者参观团|中外记者参观团]]和[[美军观察组|美军观察组]]介绍中共军队在敌后各抗日根据地的作战情况和战绩<ref>{{Cite news|url=http://history.sina.com.cn/bk/kzs/2014-09-04/100199396.shtml| title=叶剑英：八路军七年来在华北抗战的概况| publisher=凤凰网|date=2014-09-04}}</ref>{{r|A|page=189}}。1945年，在[[中国共产党第七次全国代表大会|中国共产党第七次全国代表大会]]上被选为中央委员<ref>{{cite book|author=中共中央党史研究室第一研究部编|title=中国共产党第七次全国代表大会代表名录 下|location=上海|publisher=上海人民出版社|year=2005|pages=726|isbn=7-208-05733-8}}</ref>。1945年，中国抗日战争结束后，成为首批被[[中华民国国民政府|中华民国国民政府]]授予[[抗战胜利勋章|抗战胜利勋章]]的军事将领之一<ref>{{cite web|url=http://collection.sina.com.cn/jczs/2015-08-14/doc-ifxfxzzn7475344.shtml|title=国民政府颁发的抗战胜利勋章|date=2015-08-14|accessdate=2015-09-24|publisher=新浪}}</ref>。

[[File:Maozedong_and_yejianying.jpg|thumb]]
1945年12月，叶剑英参加以周恩来为首的代表团，到重庆进行停战谈判，出席[[政治协商会议|政治协商会议]]{{r|A|page=198}}。1946年1月，叶剑英赴北平任[[军事调处执行部|军事调处执行部]]中共代表，与国民党代表、美国代表一起，调处国共军事冲突和监督双方执行停战协议<ref>{{cite book|author=军事科学院《叶剑英传》编写组|title=叶剑英传略|location=北京|publisher=军事科学出版社|year=1987|pages=142}}</ref>。1947年2月，他返回延安{{r|A|page=219}}。3月，国军攻占延安后，中共中央机关一分为三，叶剑英和[[杨尚昆|杨尚昆]]率一部分中央机关赴晋西北{{r|A|page=220}}；随后，叶剑英任中共中央后方委员会书记<ref>{{cite book|author=张静如，梁志祥主编|title=中国共产党通志 第2卷|location=北京|publisher=中央文献出版社|year=2001|pages=153|isbn=978-7-5073-1050-4}}</ref>。7月，出席[[全国土地会议|全国土地会议]]，在会上作了军事问题的报告{{r|A|page=224}}。12月，出席中央在[[米脂|米脂]]杨家沟召开的工作会议，同[[任弼时|任弼时]]一起，主持了土地问题的讨论{{r|A|page=226}}。

1948年5月，叶剑英任[[华北军政大学|华北军政大学]]校长兼政治委员{{r|A|page=229}}。他主持制定正确的教育方针，培养和建立教员队伍，发扬教学民主，输送了大批干部<ref>{{cite book|author=中国中共党史人物研究会编|title=中共党史人物传 精选本 5 军事卷 上|location=北京|publisher=中共党史出版社|year=2010|pages=211-213|isbn=7-5098-0257-1}}</ref>。1949年初，任[[北平市|北平市]]军事管制委员会主任兼市长、北平联合办事处主任<ref>{{cite book|author=范硕主编|title=新北平首任市长叶剑英|location=北京|publisher=中央文献出版社|year=1999|pages=3|isbn=7-5073-0717-4}}</ref>的叶剑英和[[聂荣臻|聂荣臻]]、[[彭真|彭真]]一起，促成了[[北平和平解放|北平和平解放]]{{r|A|page=239}}。期间他组织了一系列工作，包括对傅作义军队的改编和对旧北平市政府、学校、厂矿等各方面的接管工作{{r|A|page=241}}，致力于北平的市政建设，维护社会安定，恢复发展生产，改善文化教育等{{r|A|page=247}}。4月，他参加以周恩来为首的中共代表团，同以[[张治中|张治中]]为首的南京国民政府代表团的和平谈判{{r|A|page=249-251}}。8月，任[[中共中央华南分局|中共中央华南分局]]第一书记、广东军区司令员兼政治委员<ref>{{cite book|author1=广东省地方史志编纂委员会编|title=广东省志 军事志|date=1999年11月|location=广州|publisher=广东人民出版社|isbn=7-218-03180-3|pages=77|accessdate=2015-08-23}}</ref>。

== 1949-1966 中华人民共和国初期 ==
{{See also|广东战役|中国人民解放军军事科学院}}
[[File:Ye_Jianying_and_Liu_Yalou_in_USSR,1957.jpg|thumb]]在苏联。]]
1949年10月初，叶剑英和陈赓指挥[[广东战役|广东战役]]，14日攻占广州{{r|A|page=257}}。随后，任[[广东省人民政府|广东省人民政府]]主席兼[[广州市|广州市]]市长<ref>{{cite book|author=黄勋拔主编|title=广东省志 政治纪要|location=广州|publisher=广东人民出版社|year=2004|pages=138|isbn=7-218-04393-3}}</ref>。1949年底至1953年，他领导了[[华南地区|华南地区]]的[[剿匪|剿匪]]斗争{{r|A|page=260}}、经济建设以及广州的市政建设等各项重大工作，取得了显著成效{{r|A|page=267-270}}。1950年1月，任广州[[南方大学_(广州)|南方大学]]校长，为[[南方|南方]][[高等教育|高等教育]]作出重要贡献<ref>{{cite book|author=罗明|title=罗明回忆录|pages=218-230|publisher=广东人民出版社|year=1990|isbn=7-211-01509-8}}</ref>。

1950年9月，兼任广东省人民政府财经委员会主任。1950年春开始，他在领导广东省的[[土地改革|土地改革]]中，注意保护华侨和民族工商业者的利益。他还曾兼任华南垦殖局局长，亲自领导开拓了[[橡胶|橡胶]]和热带作物的生产事业{{r|A|page=281}}。1951年5月，任华南军区司令员<ref>{{cite book|author=姜思毅主编|title=中国人民解放军大事典 上|location=天津|publisher=天津人民出版社|year=1992|pages=1131|isbn=7-201-01046-8}}</ref>。1952年6月，任中南军区代司令员，出席华南分局扩大会议。会议中心内容是批评华南分局领导工作中在农民问题上的“右倾”错误和“地方主义”错误，叶剑英在会上违心地作了检讨发言{{r|A|page=287-289}}。1953年5月，任[[中共中央中南局|中共中央中南局]]代第一书记、[[中南行政委员会|中南行政委员会]]代主席{{r|A|page=291}}。

[[File:Peng_Dehuai,_Ye_Jianying,_Nikita_Khrushchev,_Nikolai_Bulganin.jpg|thumb]]、[[尼古拉·布尔加宁|尼古拉·布尔加宁]]和[[赫鲁晓夫|赫鲁晓夫]]。|210px]]
1954年6月，叶剑英任[[中央人民政府人民革命军事委员会|中央人民政府人民革命军事委员会]]副主席；9月，当选[[中华人民共和国国防委员会|中华人民共和国国防委员会]]副主席，并担任[[中国人民解放军武装力量监察部|中国人民解放军武装力量监察部]]部长<ref>{{cite book|author=《中国人民解放军军史》编写组|title=中国人民解放军军史：第五卷|year=2011|publisher=军事科学出版社|pages=17|isbn=978-7-80237-427-0}}</ref>。1955年4月，任[[训练总监部|训练总监部]]代部长，主持全军的军事训练工作{{r|A|page=293}}。他强调从实战需要出发，进行现代条件下的军事训练，把解放军建设成优良的正规化、现代化军队{{r|A|page=294}}。9月27日，被授予[[中华人民共和国元帅|中华人民共和国元帅]]军衔，获一级[[八一勋章_(1955年)|八一勋章]]、一级[[独立自由勋章|独立自由勋章]]、一级[[解放勋章|解放勋章]]。11月，组织并主持了辽东半岛方面军抗登陆战役中集团军海岸防御的军事演习{{r|A|page=295}}。1956年6月，他主持召开全军院校会议，强调办好院校对部队现代化建设的重大意义。9月，出席[[中国共产党第八次全国代表大会|中国共产党第八次全国代表大会]]，再次被选为中央委员<ref>{{cite book|author=王琦|title=历届中共中央委员人名词典 1921-1987|year=1992|publisher=中共党史出版社|page=61|isbn=7-80023-494-0}}</ref>。1956年12月，叶剑英率中国军事代表团访问[[缅甸|缅甸]]。之后，他还率军事代表团访问了[[苏联|苏联]]、[[印度|印度]]、[[波兰|波兰]]等国家{{r|A|page=302-303}}。

1958年3月，叶剑英任[[中国人民解放军军事科学院|中国人民解放军军事科学院]]第一任院长兼政委{{r|A|page=306}}。主持制定了军事科学研究的方针、原则和方法，建设了一支科研干部队伍，培养了一批科研人才{{r|A|page=307-309}}。同年，兼任[[中国人民解放军高等军事学院|高等军事学院]]院长<ref>{{cite book|title=中华人民共和国军事史要|author=军事科学院军事历史研究所|pages=244|location=北京|publisher=军事科学出版社|year=2005|isbn=7-80137-800-8}}</ref>。1959年9月，叶剑英任[[中共中央军委|中共中央军委]]常委{{r|A|page=320}}。 1960年，任军委军事训练和军事学术研究委员会主任，参与领导研究国家防御作战问题{{r|A|page=320}}；主张军事训练和军事科学研究相结合，以总结解放军军的经验为主，探讨在现代条件下的战争指导规律；主持制定解放军一系列条令条例{{r|A|page=325}}。1963年12月，叶剑英建议在全军推广[[郭兴福|郭兴福]]教学法，得到毛泽东和军委的赞许{{r|A|page=326}}。1965年1月，当选为第四届[[全国政协副主席|全国政协副主席]]。1966年1月，任[[中共中央军委副主席|中共中央军委副主席]]。5月15日任[[中共中央书记处|中共中央书记处]]书记、中央军委秘书长，主持军委日常工作{{r|A|page=455}}。8月，在[[中共八届十一中全会|中共八届十一中全会]]上当选为[[中共中央政治局|中共中央政治局]]委员<ref>{{cite book|author=麦克法夸尔;弗正清|title=剑桥中华人民共和国史 下卷|pages=141|publisher=中国社会科学出版社|year=1992|isbn=978-7-5004-1185-7}}</ref>。

== 1966-1976 文化大革命时期 ==
{{See also|二月逆流|九一三事件|西沙海战|反击右倾翻案风}}
[[File:1966-11_1966年国庆典礼毛泽东叶剑英.jpg|thumb]]
1966年，[[文化大革命|文化大革命]]爆发.{{r|A|page=334}}。1966年10月，叶剑英在中央工作会议上多次发言，坚决反对搞乱军队{{r|A|page=336}}。11月13日，他在北京工人体育场召开的军队院校和文体单位十万人大会上讲话，对当时的错误做法提出批评{{r|A|page=337}}。11月29日，他在北京工人体育场召开的军队院校师生大会上再次讲话，严厉谴责打、砸、抢行为{{r|A|page=338}}。1967年1月中旬，毛泽东指使[[陈伯达|陈伯达]]教唆红卫兵开始批斗[[总政治部|总政治部]]主任[[萧华_(上将)|萧华]]，之后周恩来出面辟谣{{r|zj|page=321}}{{r|bio|page=822}}。1月19日，在[[京西宾馆|京西宾馆]]召开军委碰头会，围绕军队内部是否展开文革运动，叶剑英、[[徐向前|徐向前]]、[[聂荣臻|聂荣臻]]三位元帅与江青、[[陈伯达|陈伯达]]、[[康生|康生]]、[[姚文元|姚文元]]争执起来{{r|bio|page=823}}。此时[[叶群|叶群]]突然拿出发言稿批判萧华，当晚[[红卫兵|红卫兵]]抄了萧华的家{{r|bio|page=824}}。次日上午，京西宾馆继续开会，[[萧华_(上将)|萧华]]讲述被抄家经过，作为全军文革小组组长的徐向前出于责任，和陈伯达一起批评萧华<ref>{{cite book|author=Jin Qiu|title=The Culture of Power: The Lin Biao Incident in the Cultural Revolution|publisher=Stanford University Press|year=1999|pages=95|isbn=0804735298}}</ref>。徐向前与叶剑英气愤之下敲桌子，而葉劍英亦因此拍伤手骨{{r|A|page=344}}。该事情被[[四人帮|四人帮]]诬陷为“京西宾馆事件”，为“[[二月逆流|二月逆流案]]”之始{{r|zj|page=322}}。

京西宾馆会议刚结束，红卫兵开始批斗[[杨勇_(上将)|杨勇]]，中央军委工作开始陷入混乱{{r|bio|page=827}}。情急之下，徐向前于1月24日晚，独闯[[林彪|林彪]]住处[[毛家湾|毛家湾]]，并开门见山要求军队必须保持稳定，并要求制定规则{{r|zj|page=323}}。随后[[林彪|林彪]]表示同意，并请叶剑英、聂荣臻赶到毛家湾，共同商议出七条意见<ref>{{cite book|author=王年一|title=大动乱的年代|year=2009|location=北京|publisher=人民出版社|page=142|isbn=978-7-01-007863-2}}</ref>{{r|bio|page=828}}。之后叶、聂、徐三位元帅赶赴[[钓鱼台国宾馆|钓鱼台国宾馆]]，与[[中央文革小组|中央文革小组]]讨论，确定七条意见{{r|bio|page=828}}。后经各军区领导讨论，改成八条意见{{r|bio|page=829}}。1月28日，林彪与徐向前到[[中南海|中南海]][[毛泽东|毛泽东]]处审批该八条命令{{r|bio|page=829-830}}。此后该命令得到贯彻，各大军区秩序得以恢复{{r|bio|page=831}}。

同年2月8日，国务院总理[[周恩来|周恩来]]在[[怀仁堂|怀仁堂]]召开中央政治局碰头会{{r|zj|page=324}}。会上，徐向前同陈伯达就[[刘志坚|刘志坚]]问题发生严重争执{{r|zj|page=325}}。2月11日，叶剑英和徐向前等与陈伯达、[[康生|康生]]、[[张春桥|张春桥]]就军队稳定问题发生争执<ref name="wenge1">{{cite book | author = 唐少傑 | title = 一葉知秋: 清華大學1968年「百日大武鬥」 | url =  | date = 2003年 | publisher = Chinese University Press | id =  | isbn = 978-9-6299-6126-8 |pages=55}}</ref>{{r|bio|page=832}}。2月16日，军方与文革小组的冲突达到顶峰，[[谭震林|谭震林]]、[[陈毅|陈毅]]、[[李先念|李先念]]纷纷发言怒斥文革小组{{r|bio|page=832}}。会后中共文革小组向毛泽东汇报，经毛泽东表态，文革小组开始制造“[[二月逆流|二月逆流案]]”{{r|bio|page=834}}。2月25日至3月18日，中共中央多次开会，叶剑英同谭震林、陈毅、[[李富春|李富春]]、李先念、徐向前、聂荣臻等被冠以“二月逆流”的罪名，遭江青、康生、陈伯达、谢富治等人围攻批斗{{r|A|page=346}}。1968年3月，发生[[杨余傅事件|杨余傅事件]]，叶剑英、聂荣臻等元帅被诬陷为“黑后台”{{r|A|page=349}}。1968年10月，[[中共八届十二中全会|中共八届十二中全会]]召开，[[朱德|朱德]]、[[陈云|陈云]]、叶剑英、徐向前、陈毅、[[聂荣臻|聂荣臻]]等人被分为不同小组，进行批判{{r|bio|page=841-842}}。1969年1月3日，毛泽东表示为二月逆流案平反{{r|zj|page=329}}。然而在同年4月[[中国共产党第九次全国代表大会|中国共产党第九次全国代表大会]]上，朱德、陈毅、叶剑英等元帅继续被分组批判{{r|bio|page=843-844}}。

1969年1月，叶剑英被下放到新华印刷厂蹲点{{r|A|page=352}}。6至10月，叶剑英与陈毅、徐向前、聂荣臻受毛泽东、周恩来委托，全面深入地分析国际形势，为中国外交工作提出意见和建议{{r|A|page=353}}。同年10月，林彪发布《[[林副主席指示第一号令|林副主席指示第一号令]]》，全军进入紧急战备状态，各军队将领被迫疏散离京{{r|bio|page=848-849}}。其中朱德去广州、叶剑英到长沙、刘伯承去武汉、陈毅在石家庄、聂荣臻去[[邯郸|邯郸]]<ref>{{cite book|author=《刘伯承传》编写组|title=刘伯承传|pages=414|year=2007|location=北京|publisher=当代中国出版社|isbn=978-7-8009-2101-8}}</ref>，徐向前在[[开封|开封]]，[[彭德怀|彭德怀]]在押，此时十大元帅中除[[罗荣桓|罗荣桓]]和[[贺龙|贺龙]]去世外，只有林彪在北京{{r|bio|page=849}}。1970年8月，叶剑英参加[[中共九届二中全会|中共九届二中全会]]{{r|A|page=355}}。会后，他按照毛泽东、周恩来的指示，带领调查组到福建、广东、广西、湖南、湖北等地，调查[[陈伯达|陈伯达]]历史{{r|A|page=356}}。

1971年7月，叶剑英受毛泽东、周恩来委托，主持接待秘密来华访问的美国总统国家安全事务助理[[基辛格|基辛格]]{{r|A|page=366}}。之后林彪[[九一三事件|九一三事件]]爆发，叶剑英重新以军委副主席身份主持军委日常工作{{r|A|page=357}}。同时，中共中央撤消军委办事组，成立军委办公会议，由叶剑英主持，并由[[谢富治|谢富治]]、[[张春桥|张春桥]]、[[李先念|李先念]]、[[李德生|李德生]]、[[纪登奎|纪登奎]]、[[汪东兴|汪东兴]]、[[陈士榘|陈士榘]]、[[张才千|张才千]]、[[刘贤权|刘贤权]]组成，办公会议制度维持至1975年2月<ref>{{cite book|author=《中国人民解放军军史》编写组|title=中国人民解放军军史：第六卷|year=2011|publisher=军事科学出版社|pages=165|isbn=978-7-80237-427-0}}</ref>。1972年，参加接待先后来华访问的美国总统[[尼克松|尼克松]]和日本首相[[田中角荣|田中角荣]]。1973年，又协助周恩来接待第二次来访的基辛格{{r|A|page=366}}。1973年8月，在[[中共十届一中全会|中共十届一中全会]]上，他当选为[[中共中央政治局常委|中共中央政治局常委]]、[[中共中央副主席|中共中央副主席]]。1974年1月，叶剑英受毛泽东、周恩来委托，同邓小平一起指挥[[西沙自卫反击战|西沙之战]]，占领[[西沙群岛|西沙群岛]]{{r|A|page=363-364}}。

1975年1月，在[[第四届全国人民代表大会|第四届全国人民代表大会]]会议上，他被任命为[[中华人民共和国国防部部长|国防部长]]{{r|A|page=370}}。6、7月间，叶剑英和[[邓小平|邓小平]]一起，主持召开了中央军委扩大会议。他在会上就国际形势、压缩军队定额、调整编制体制、安排超编干部等问题作了讲话{{r|A|page=371-372}}。在他主持下，对全军二十几个大单位的领导班子进行了调整配备{{r|A|page=373}}。1976年2月，在[[反击右倾翻案风|反击右倾翻案风]]运动中，叶剑英被毛泽东下令“养病休息”，由[[陈锡联|陈锡联]]负责主持中央军委日常工作{{r|A|page=376}}。

== 1976-1986 改革开放时期 ==
{{See also|怀仁堂事变|改革开放|叶九条|八二宪法|华国锋|邓小平}}
=== 抓捕四人帮 ===
1976年9月9日[[毛泽东之死|毛泽东逝世]]，在治丧初期，时任中共中央第一副主席和国务院总理的[[华国锋|华国锋]]就计划解决“[[四人帮|四人帮]]”问题{{r|中华人民共和国史V8|page=672}}。9月11日，华国锋向国务院副总理[[李先念|李先念]]交底，通过李居中联系，与叶剑英取得一致意见。华国锋积极行动，又得到[[中共中央办公厅|中共中央办公厅]]主任[[汪东兴|汪东兴]]、中央军委常委[[陈锡联|陈锡联]]、[[中共北京市委|中共北京市委]]和[[北京卫戍区|北京卫戍区]]负责人[[吴德|吴德]]等人的支持，在力量对比上完全压倒了“四人帮”。华国锋考虑过开会解决和武力解决两种方式，9月21日与叶剑英密谈后初步确定以“隔离审查”的方式强硬解决{{r|A|page=383}}。9月26日晚，华征求李、吴的意见后一致认为，在政治局会议获得多数票有把握，在中央全会则无把握，通过党内斗争的正常程序已无法顺利拿下“四人帮”，遂决定“先斩后奏”，待果断处置“四人帮”之后再召开政治局会议追认。叶剑英获悉商议结果后表示完全同意。{{r|中华人民共和国史V8|page=695}}<ref name="刘晓2000">{{cite book|title=激荡岁月：1976年的中国|author=刘晓|publisher=中国工人出版社|year=2000|page=219-225}}</ref>

这种“半合法”的方式是特殊情况下不得不采取的特殊手段，得到[[陈云|陈云]]“不可避免”、“下不为例”的表态支持<ref>{{cite book|author=尹家民|title=红墙见证录（下）：共和国风云人物留给后世的真相|publisher=当代中国出版社|year=2004|ISBN=7-80170-272-7|pages=1349}}</ref>。汪东兴率[[8341部队|8341部队]]负责抓捕行动，10月4日确定了具体行动计划{{r|中华人民共和国史V8|page=697-698}}。10月6日，抓捕行动按计划展开，华国锋以讨论《[[毛泽东选集第五卷|毛泽东选集第五卷]]》的名义，召集[[王洪文|王洪文]]、[[张春桥|张春桥]]、[[姚文元|姚文元]]在怀仁堂开会，晚8时左右，三人先后抵达，即由华、叶、汪宣布对其“隔离审查”；几乎同时，[[江青|江青]]、[[毛远新|毛远新]]分别在住处被宣布“隔离审查”和“就地监护”，[[谢静宜|谢静宜]]、[[迟群|迟群]]等“四人帮”党羽在北京市委被控制{{r|A|page=387}}。10月6日晚10时许，华国锋在北京西郊玉泉山9号楼（叶剑英住处）主持召开中央政治局紧急会议。在会上宣布粉碎“四人帮”是毛泽东的遗愿、“四人帮”被抓是文化大革命的胜利{{r|ye|page=146}}。会议决定[[华国锋|华国锋]]为[[中国共产党中央委员会主席|中国共产党中央委员会主席]]{{r|A|page=388}}。10月14日，中央公布了粉碎四人帮的消息，全国沸腾，[[毛泽东逝世|毛泽东逝世]]之后的沉闷局面一扫而空<ref name="苏继红2013">{{cite book|author=邓书杰 李 梅 吴晓莉 苏继红|title=转机时刻（1970－1979）（中国历史大事详解）|url=http://books.google.com/books?id=-ghcAgAAQBAJ&pg=PT367|date=18 December 2013|publisher=青苹果数据中心|page=367|id=GGKEY:195P4P59P2F}}</ref>。1977年3月，中共中央发出九号文件，通知全党：中共中央军委的日常工作仍由叶剑英负责主持。5月14日，在住所过八十寿辰，徐向前、聂荣臻、王震、粟裕、余秋里、杨成武、邓小平前来祝寿，叶剑英当着众人的面对邓小平说：“你也是老帅嘛，你是我们老帅中的领班。”<ref>[https://history.ifeng.com/c/7ijNpo8ZCBm 邓小平能第三次复出 这两位共和国元帅出力最大]</ref>7月，[[中共十届三中全会|中共十届三中全会]]上，华国锋和[[邓小平|邓小平]]分别被选为党中央的正副主席，叶剑英继续担任排名第一的副主席。8月批示“上报军委批办的文件，可先送小平同志”，随即传达军委。

=== 国家元首 ===
[[文化大革命|文化大革命]]结束后，作为党内唯一副主席的叶剑英成为仅次于中共中央主席[[华国锋|华国锋]]的第二号人物。叶剑英对[[邓小平|邓小平]]在文化大革命后重新出山起主要作用{{r|A|page=392}}。1978年3月，叶剑英当选第五届[[全国人大常委会委员长|全国人大常委会委员长]]（[[中国国家主席存废之争|废除国家主席]]时期作为[[中华人民共和国国家元首列表|国家元首]]）{{r|A|page=401}}。12月24日，在人民大会堂主持中共中央为彭德怀、陶铸举行的追悼大会。邓小平、陈云分别致悼词。1979年1月1日，叶剑英主持[[全国人大常委会|全国人大常委会]]通过发表了《[[告台湾同胞书|告台湾同胞书]]》，指出“实现中国的统一，是人心所向，大势所趋”，并提出在海峡两岸“发展贸易，互通有无，进行经济交流”和“双方尽快实现通航通邮”的方针<ref>{{cite book|author1=刘峰搏|author2=杜广强|title=两岸三通简史|location=北京|publisher=台海出版社|year=2007|pages=29-35|isbn=7-80141-579-5}}</ref>。1979年10月1日，為了慶祝中華人民共和國成立三十周年，[[中共中央|中共中央]]決定由葉劍英出面講話<ref>刑小群：〈鄭惠口述史（三之三）〉，[[香港|香港]]：《[[明報月刊|明報月刊]]》，2003年9月，第103頁</ref>。叶剑英在讲话中，论述中华人民共和国成立三十年来的成就和失误，总结文化大革命的教训，批判[[两个凡是|两个凡是]]，明确提出在建设高度物质文明的同时，要建设高度的社会主义精神文明，号召全国人民团结一致，向着[[四个现代化|四个现代化]]的目标前进<ref>{{cite web|url=http://news.xinhuanet.com/ziliao/2005-02/05/content_2549765.htm|title=在庆祝中华人民共和国成立三十周年大会上的讲话|author=叶剑英|date=1979-09-29|accessdate=2015-09-28}}</ref>。

1980年9月，叶剑英担任宪法修改委员会主任委员，领导制定了《[[中华人民共和国刑法|刑法]]》、《[[中华人民共和国刑事诉讼法|刑事诉讼法]]》等二十二个法律{{r|A|page=402}}。1981年9月30日，叶剑英就[[台湾问题|台湾问题]]提出九条方针政策，建议举行国共两党对等谈判，实现第三次国共合作，即“[[叶九条|叶九条]]”<ref>{{Cite book|author=吴石坚|title=叶剑英与中国特色社会主义|location=上海|publisher=上海社会科学院出版社|year=2011|pages=60-66|isbn=978-7-80745-844-9}}</ref>。1982年12月，第五届全国人大第五次会议通过了叶剑英主持制定的“[[八二宪法|八二宪法]]”{{r|A|page=403}}。

=== 晚年 ===
文化大革命结束后，在叶剑英的支持下[[邓小平|邓小平]]回到中共最高决策层，担任中共中央副主席、总参谋部长、国务院副总理开始拨乱反正，与主张“[[两个凡是|两个凡是]]”的[[凡是派|凡是派]]展开论战<ref name="F2001">{{cite book|author=Joseph Fewsmith|title=Elite Politics in Contemporary China|url=http://books.google.com/books?id=LOSwVE6ZQasC&pg=PA45|date=1 January 2001|publisher=M.E. Sharpe|isbn=978-0-7656-0686-0|pages=45}}</ref>，[[求是派|求是派]]渐渐在舆论和中共高层权力中获胜，而毛泽东式的领袖崇拜逐渐失去了光环<ref name="45event">{{cite book |author =  杨晋川、刘庸安、李从国、程国林编 |title = 1976-1992中国政坛风云录 | year = 1993 | publisher = 改革出版社  |isbn = 978-7-8007-2467-1|pages=51}}</ref><ref name="Guo2012">{{cite book|author=Xuezhi Guo|title=China's Security State Philosophy, Evolution, and Politics|url=http://books.google.com/books?id=-7_lfNx76GcC&pg=PA362|date=29 August 2012|publisher=Cambridge University Press|isbn=978-1-107-02323-9|page=362}}</ref>。党内的众元老们指责华国锋<ref>{{cite book | author = Michael E. Marti | coauthors = | title = China and the Legacy of Deng Xiaoping: From Communist Revolution to Capitalist Evolution | url = | date = 2002 | location = | publisher = Brassey's | id = | isbn = 978-1-57488-540-8 | page = 10 }}</ref><ref>{{cite book | author = 邓小平 | coauthors = | title = 邓小平文选 | url = | date = 1994年 | location = 北京 | publisher = 人民出版社 | id = 第二卷 | isbn = 978-7-5065-2523-7 | page= 42-47页、38-39页 }}</ref>是“极左路线错误”，华国锋一派的领导地位逐渐被取代<ref>{{cite book|title=世纪脉络 从 "一大" 到 "十五大"|url=http://books.google.com/books?id=gNxTAAAAIAAJ|year=1997|publisher=华夏出版社|isbn=978-7-5080-1415-9|page=229}}</ref><ref name="Chen1995">{{cite book|author=Feng Chen|title=Economic Transition and Political Legitimacy in Post-Mao China Ideology and Reform|url=http://books.google.com/books?id=hCe50-WRUrEC&pg=PA36|year=1995|publisher=SUNY Press|isbn=978-0-7914-2657-9|pages=36–38}}</ref>。华国锋的权力也在[[1978年中央工作会议|1978年中央工作会议]]、[[中国共产党第十一届中央委员会第三次全体会议|中共十一届三中全会]]后不断削弱{{r|Li2012}}。

1980年2月，在[[中共十一届五中全会|中共十一届五中全会]]上，中共中央决定恢复[[中共中央书记处|中共中央书记处]]，[[中共中央秘书长|中共中央秘书长]][[胡耀邦|胡耀邦]]任[[中国共产党中央书记处总书记|总书记]]，开始改变权力高度集中于党主席[[华国锋|华国锋]]的格局。基于党政分开的原则，在9月的五届全国人大第三次会议上，华辞去了兼任的[[国务院总理|国务院总理]]一职，由[[赵紫阳|赵紫阳]]接任。{{r|郑刚1998}}。8月重要政治局扩大会议以后，很多人提出华不宜担任[[中共中央主席|中共中央主席]]和[[中共中央军委主席|中共中央军委主席]]{{r|蕭冬連|page=386-387}}。11月10日至12月5日，[[中共中央政治局|中共中央政治局]]连续召开9次会议，讨论华国锋过去四年的执政问题{{r|ye|page=455}}。陈云、李先念、胡耀邦等人在会上向华提出严肃批评，华作了许多解释和自我辩护，不肯承认错误，会议一再延长。叶剑英在会上检讨自己有“[[周公旦|周公]]辅[[周成王|成王]]”的“封建”思想<ref name="张江明1997">{{cite book|author=张江明|title=历史转折关头的叶剑英|url=http://books.google.com/books?id=3fFoAAAAIAAJ|year=1997|publisher=中共党史出版社|isbn=978-7-80136-091-5|page=159}}</ref>{{r|蕭冬連|page=391}}，自我批评在宣传华时说了过誉的话，还说：“如果国锋同志不愿意承担责任，那就由我承担好了。”叶剑英此言既出，华不再辩解，表示接受批评并提出辞职<ref>{{cite journal|journal=党史纵览|issue=11|author=袁小伦|title=由远及近到渐行渐远：叶剑英与华国锋的政治交往|url=http://news.ifeng.com/history/1/renwu/200808/0820_2665_735841.shtml|year=2005|}}</ref>{{r|ye|page=458}}。在1981年6月召开的十一届六中全会上，华国锋的辞呈得到批准，仅象征性担任位列最后的政治局常委和中央委员会副主席。

1982年9月，叶剑英出席[[中国共产党第十二次全国代表大会|中国共产党第十二次全国代表大会]]，再次当选[[中共中央政治局常委|中共中央政治局常委]]、[[中国共产党中央军事委员会副主席|中国共产党中央军事委员会副主席]]，党内位列第二位，仅次于党总书记[[胡耀邦|胡耀邦]](實際上地位高過胡耀邦，僅次於鄧小平，與陳雲同級)之後，在军委主席[[邓小平|邓小平]]之前{{r|A|page=420}}。1983年4月20日，致中央政治局常委关于全国人大和全国政协人事安排问题的信。6月20日，第六届全国人大第一次会议决定，成立[[中华人民共和国中央军事委员会|中华人民共和国中央军事委员会]]，出任副主席。1985年9月，叶剑英在[[1985年中国共产党全国代表会议|1985年中国共产党全国代表会议]]前的中共十二屆四中全會上，和六十三位老同志一起提出，不再担任中央委员、候补中央委员{{r|A|page=420}}。1986年10月22日凌晨1点16分，叶剑英病逝于[[北京|北京]]，享年89歲{{r|A|page=423}}。中共中央、全國人大常委會、國務院、中央軍委發布訃告，評價葉劍英是「久經考驗的共產主義忠誠戰士，堅定的馬克思主義者，偉大的無產階級革命家、政治家、軍事家，中國人民解放軍的締造者之一，中華人民共和國的開國元勛，長期擔任黨、國家和軍隊重要領導職務的卓越領導人」。1987年10月22日，叶剑英的骨灰安葬于[[广州|广州]][[红花岗烈士陵园|红花岗]][[广州起义烈士陵园|烈士陵园]]{{r|A|page=426}}。

== 纪念 ==
[[File:梅州剑英图书馆正门.jpg|thumb]]
=== 故居和纪念馆 ===
叶剑英元帅的故居位于[[梅州市|梅州市]][[梅县区|梅县区]][[雁洋镇|雁洋镇]]雁上村，纪念馆建于1987年7月，馆名由原国家主席[[杨尚昆|杨尚昆]]题写，纪念馆陈列展览有大量珍贵的照片、手稿、题词、文献、文物。2005年5月5日在故居和纪念馆的基础上，整合扩建成为[[叶剑英纪念园|叶剑英纪念园]]，叶剑英纪念园是全国100个红色旅游经典景区和广东3个红色旅游经典景区之一，占地面积482亩，总投资1.6亿元，分为人文秀区、纪念景区、生态林区和旅游休闲服务区四个区。纪念园门前有一座叶剑英元帅的全身铜像。<ref>{{Cite web|url = http://www.mzta.gov.cn/info/you/jq/2011/1113/2041.html|title = 叶剑英纪念园|author = |date = 2015-06-25|accessdate=2015-09-26|publisher = 梅州市旅游网}}</ref>

1962年9月24日，[[毛泽东|毛泽东]]在[[中共八届十中全会|中共八届十中全会]]上说：“叶剑英同志搞了一篇文章，很尖锐，大关节是不糊涂的。我送你两句话‘诸葛一生唯谨慎，吕端大事不糊涂’。诸葛，大家都知道，是[[诸葛亮|诸葛亮]]。[[吕端|吕端]]是宋朝的一个宰相，说这个人大事不糊涂。”<ref>{{cite journal|url=http://history.people.com.cn/GB/205396/14001302.html|title=毛泽东最后一次长篇讲话：认为“四人帮”的问题不大|author=赵一楠|journal=湘潮|year=2011|issue=1}}</ref>1967年夏，毛泽东对[[杨成武|杨成武]]说：“剑英同志在关键时刻是立了大功的。他救了党，救了红军，救了我们这些人。”1971年8月28日，毛泽东在长沙同[[刘兴元|刘兴元]]、[[丁盛|丁盛]]、[[韦国清|韦国清]]、[[汪东兴|汪东兴]]谈话，提及1935年红军分裂时说：“叶剑英同志在这个关键时刻是有功劳的，所以你们应当尊重他。那时我们的路线是正确的，那时军队如果不到西北，那里还有点根据地，那怎么能到华北地区、东北地区呢？怎么能在抗日战争时搞那么多根据地呢？”{{r|A|page=111}}

1986年10月22日，[[中共中央|中共中央]]、[[全国人大常委会|全国人大常委会]]、[[中华人民共和国国务院|国务院]]、[[中央军委|中央军委]]发布讣告，评价叶剑英是“久经考验的共产主义忠诚战士，坚定的马克思主义者，伟大的无产阶级革命家、政治家、军事家，中国人民解放军的缔造者之一，中华人民共和国的开国元勋，长期担任党、国家和军队重要领导职务的卓越领导人”{{r|A|page=423}}。10月29日，在人民大会堂举行叶剑英追悼会，[[中顾委|中顾委]]主任[[邓小平|邓小平]]主持大会，总书记[[胡耀邦|胡耀邦]]致悼词。悼词特别肯定了叶剑英在中共关键时刻所做的贡献{{r|A|page=425}}。1989年11月，叶剑英被中共中央军委确定为[[中国人民解放军军事家|中国人民解放军军事家]]之一<ref>{{cite web|date=2007-03-05|accessdate=2015-09-03|title=中国当代军事家|publisher=中国网|url=http://www.china.com.cn/military/txt/2007-03/05/content_7906177.htm}}</ref>。

叶剑英在去世后仍然受到官方和学术界的关注和讨论。1997年4月28日，中共中央召开“纪念叶剑英同志诞辰一百周年座谈会”。中共中央总书记[[江泽民|江泽民]]在会上发言，称赞他“党性坚强，信念坚定。他对社会主义、共产主义事业矢志不渝，对党和人民无比忠诚。他无私无畏，有胆有识。在各种严峻的考验面前，他总是把党和人民的利益放在首位，坚决同一切危害党和人民利益的行为作斗争。在重大和紧要的历史关头，他总是挺身而出，力挽狂澜，表现出伟大的革命气魄和高超的斗争艺术。”<ref>{{cite web|author=江泽民|url=http://cpc.people.com.cn/GB/69112/118416/118422/7102813.html|title=在纪念叶剑英同志诞辰一百周年座谈会上的讲话|publisher=人民日报|date=1997-04-29|accessdate=2015-09-28}}</ref>

=== 负面 ===
一般认为，叶剑英应对原广东军区副司令员[[陈光_(军事家)|陈光]]之死负主要责任。葉劍英1950年任廣東軍區司令員兼政委時，与陈光发生冲突；随后陈光被開除黨籍，遭到軟禁。陈光1954年6月自焚<ref>{{Cite news| url=http://history.dwnews.com/photo/2013-12-23/59360654.html| title=蒙冤自焚的中共開國名將[圖集]| author=司馬清揚| publisher=多維歷史| date=2013-12-23| accessdate=2014-12-04| deadurl=yes| archiveurl=https://web.archive.org/web/20140501154153/http://history.dwnews.com/photo/2013-12-23/59360654.html| archivedate=2014年5月1日| df=}}</ref><ref>{{Cite web | url=http://www.21ccom.net/articles/lsjd/lccz/article_2010100921352.html | publisher=共識網 ||author=李作鵬| title=我為什麼沒有阻攔林彪起飛|date=2010-10-09|accessdate=2014-12-04|archiveurl=https://web.archive.org/web/20160306072428/http://21ccom.net/articles/lsjd/lccz/article_2010100921352.html|archivedate=2016-03-06|deadurl=yes}}</ref>。

==家族及私人生活==
叶剑英的祖先宋朝时由中原南迁到梅州<ref>{{cite book|author1=刘继贤主编|author2=中国人民解放军军事科学院编|title=叶剑英年谱 上|location=北京|publisher=中央文献出版社|year=2007|pages=1|isbn=7-5073-2282-3}}</ref>。叶剑英祖父[[叶福智|叶福智]]，早年以屠宰为业，育有五子，第五子为叶剑英父亲[[叶钻祥|叶钻祥]]（1871-1921）。叶钻祥字裕芬，自幼随二哥[[叶钦祥|叶钦祥]]习武，曾考取梅州武秀才{{r|A|page=1}}。叶剑英母亲[[陈秀云|陈秀云]]（1872-1944），共生有4男4女，存活2男2女。即儿子叶剑英、[[叶道英|叶宜导]]（1906-1989），女儿[[叶才英|叶才英]]、[[叶顺英|叶顺英]]。

叶剑英早年在家乡有一门[[包办婚姻|包办婚姻]]，无子女。1924年初，在广州与医务工作者[[冯华|冯华]]结婚{{r|A|page=23}}。当年11月，生下长子[[叶选平|叶选平]]，后又生了长女[[叶楚梅|叶楚梅]]。1927年，在广州与[[曾宪植|曾宪植]]结婚{{r|A|page=58}}。1937年，在延安与长征女干部[[危拱之|危拱之]]结婚，二人无子女。1940年，在重庆与[[吴博|吴博]]结婚{{r|A|page=181}}。1948年末，在北京[[良乡|良乡]]与[[华北军政大学|华北军政大学]]学员[[李剛_(葉劍英妻)|李刚]]结婚，生子[[叶选廉|叶选廉]]，女儿[[叶文珊|叶文珊]]。1955年二人离异{{r|A|page=239}}。1955年后，叶剑英没再正式结过婚。身边还先后有过三位女人，主要是照顾叶帅的生活，而没有正式名份，是否有子女亦不详<ref>{{cite book|author=[[孔庆东|孔庆东]]|title=脍炙英雄|location=北京|publisher=中国文联出版社|year=2012|pages=251-257|isbn=978-7-5059-7340-4}}</ref>。[[罗瑞卿|罗瑞卿]]之子[[罗宇|罗宇]]回忆说，1956年在北戴河，由于叶与其他护士有染，叶的另一位“小夫人”全身赤裸，欲投海自尽，被警卫所阻，但闹的北戴河“沸沸扬扬”<ref>{{cite book|author=[[羅宇|羅宇]]|title=告别总参谋部——罗宇回忆录|location=纽约|publisher=开放出版社|year=2015|pages=284|isbn=962-7934-49-6}}</ref>。叶因为妻子、红颜众多，使之在民间也有“花帅”之称。
