'''叶梓萱'''（英文：'''Yoyiki.Una'''，{{bd|1992年|11月14日|}}），上海人，中国大陆女演员，模特<ref>{{cite web |url = http://www.mingxing.com/neidi/yezixuan/ziliao.html|title = 叶梓萱的个人资料|publisher = 明星网|accessdate = 2014-11-26|language = zh-cn}}</ref>。因[[ChinaJoy|ChinaJoy]]“强吻门”事件走红，因其身材火辣，被网友冠以“[[宅男女神|宅男女神]]”的称号。

== 相关事件 ==

=== 强吻门 ===
2010年，在ChinaJoy[[三国杀|三国杀]]展台，叶梓萱在接受玩家合影时突然被一男子强吻，之后该男子迅速从后台溜走。此事引发网友热议，叶梓萱也因此走红。<ref>{{Cite web|url = http://star.pclady.com.cn/data/1112/773258.html|title = 叶梓萱强吻门之猥琐男直接强吻|accessdate = 2015-07-22|author = |date = 2011-12-14|publisher = |language = zh-cn}}</ref>

=== 日本宣传照 ===
2011年，叶梓萱应邀为日本某单机游戏厂商拍摄宣传照，期间日方执意要求叶梓萱半裸拍摄，遭到叶拒绝后双方产生争执，最终合作被取消。<ref>{{Cite web|url = http://news.178.com/201101/89578770140.html|title = 某日本游戏商要求叶梓萱半裸拍摄宣传照 遭严正拒绝|accessdate = 2015-07-22|author = |date = 2011-01-21|publisher = |language = zh-cn}}</ref>
