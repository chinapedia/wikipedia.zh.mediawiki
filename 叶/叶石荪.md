'''叶石荪'''（或'''叶麐'''，{{bd|1893年||1977年}}）[[中國|中國]][[心理学家|心理学家]]，教育学家，[[诗人|诗人]]。 四川兴文人。曾任[[清华大学|清华大学]]、[[北京大学|北京大学]]、[[山东大学|山东大学]]、[[武汉大学|武汉大学]]、[[四川大学|四川大学]]、[[西南師範大學|西南師範大學]]、[[黄埔军校|中央陆军学校]]教授，[[中国心理学会|中国心理学会]]常务理事等职。

== [[辛亥革命|辛亥革命]]至[[二次革命|二次革命]]時期 ==

早年入永宁中学就读，受校长[[杨苍白|杨苍白]]（后曾任孙中山总统府秘书长， 四川省长等职），教师[[向楚|向楚]]（重庆辛亥革命的主要宣传者、组织者和领导者之一， 曾任四川省政务厅长、代省长）的影响， 倾向于建立共和体制。 中学就读两三年后， 考入[[陆军小学堂|四川陆军小学堂]]，为武装革命作准备。 

[[四川陆军小学堂|四川陆军小学堂]]毕业后，参加保路武装革命，任督队官。在随之而来的[[辛亥革命|辛亥革命]]中，担任[[大汉蜀军政府|大汉蜀军政府]]重庆城防副官，并且与[[熊克武|熊克武]]等人协商组织学生军北伐对[[满清|满清]]作战。

[[辛亥革命|辛亥革命]]胜利后，放弃军政职务，入[[孙中山|孙中山]]创办的[[苏州蒙藏垦殖学堂|苏州蒙藏垦殖学堂]]，以及在上海的[[南洋公学|南洋公学]]求学。适逢[[二次革命|二次革命]]，又参与了[[陈英士|陈英士]]的进攻[[江南机器制造总局|江南机器制造总局]]的战斗。因学业优异， 得以免费[[肄业|肄业]]于[[南洋公学|南洋公学]]。旋离开[[上海|上海]]，入[[河海工程学校|河海工程学校]]，毕业后任[[济南|济南]]分校教师。又入[[北京大学|北京大学]][[哲学系|哲学系]]。在[[北京大學|北大]]進修期间，加入[[陈独秀|陈独秀]]领导的[[新青年会|新青年会]]。

在[[北大|北大]]学习期间，在晨报上撰文支持[[朝鲜|朝鲜]]的[[抗日|抗日]]独立运动。1952年[[朝鲜人民代表团|朝鲜人民代表团]]访问[[成都|成都]]期间, 为此向叶麐致谢.

[[北京大学|北京大学]]毕业后，至[[法国|法国]]学习[[心理学|心理学]]，获[[里昂大学|里昂大学]][[博士学位|博士学位]]。继而在[[巴黎大学|巴黎大学]][[心理學|心理]]研究所工作二年。而后回国，任[[清华大学|清华大学]]教授。期间亦任[[北京大学|北京大学]]教授。

== [[抗日戰爭|抗日戰爭]]至晚年 ==

在[[清华大学|清华大学]]工作五年后的安息年，到[[山东大学|山东大学]]任教（见[[徐中玉|徐中玉]]的回忆）。而后[[卢沟桥事变|卢沟桥事变]]爆发，离开[[山东|山东]]，应[[武汉大学|武汉大学]]教务主任[[朱光潜|朱光潜]]的邀请， 到[[武汉大学|武汉大学]]任教。而后应[[四川大学|四川大学]][[校长|校长]][[黄季陆|黄季陆]]的邀请，任[[四川大学|四川大学]]教务长、副校长、代理校长等职务， 亦任[[中央大学|中央大学]]医学院教授。1949年以后，担任过[[川西行署委员|川西行署委员]]等职务。1952年院系调整，到[[西南師範大學|西南師範大學]]教育系任职。在对外联络方面，担任过[[中國|中]][[法國|法]]、[[中國|中]][[比利時|比]]、[[中國|中]][[瑞典|瑞]]、[[中國|中]][[蘇聯|苏]]友好协会的理事。

[[抗日战争|抗日战争]]时期，也在[[黃埔軍校|中央陆军学校]]（原黄埔军校）兼任军事心理学教授。

== 著作 ==
著作側重於[[文艺心理学|文艺心理学]]、[[教育心理学|教育心理学]]方面，部分诗词收藏在《[[雍园词钞|雍园词钞]]》、《[[吴宓诗集|吴宓诗集]]》中。
