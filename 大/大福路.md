'''大福路'''即'''[[佛山|佛山]]大福路'''，同[[福禄路|福禄路]]、[[福賢路|福賢路]]、[[福宁路|福宁路]]、[[筷子路|筷子路]]一樣是帶有濃郁[[佛山|佛山]]地方特色的路名。後來[[佛山市人民政府|佛山市人民政府]]宣佈將大福路易名成嶺南大道。

== 歷史 ==

據稱大福路這一路名已有70餘年的歷史.<ref>中国行政区划网,大福路改名触动佛山人情感,2006-10-19 http://xzqh.org.cn/ONEWS_zq.asp?id=1296{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

== 更改路名 ==

[[2006年|2006年]]9月30日，佛山[[民政局|民政局]]通报市內7條主幹路改名方案<ref>佛山市民政局,中心组团七条主干道路命名更名敲定,2006-10-09 http://www.fsmz.gov.cn/mzdt/mzxw/200610/t20061009_175982.htm</ref>。雖然這個過程中[[市民|市民]]對大福路易名的反對聲音甚大<ref>广东地市新闻·南方网,佛山大福路擬改名為嶺南大道引發網友熱論,2006-10-17,14:07:57 {{cite web |url=http://www.southcn.com/news/dishi/foshan/shehui/200610170623.htm |title=存档副本 |accessdate=2008-08-01 |deadurl=yes |archiveurl=https://web.archive.org/web/20160304141736/http://www.southcn.com/news/dishi/foshan/shehui/200610170623.htm |archivedate=2016-03-04 }}  繁體字</ref><ref>C2000.cn,博客,我佛山人,佛日有胆拿“福娃”说事，乜冇胆拿“大福路”说事？,2006-10-18 {{cite web |url=http://blog.c2000.cn/user1/104/10008.html |title=存档副本 |accessdate=2008-08-01 |deadurl=yes |archiveurl=https://web.archive.org/web/20100605211551/http://blog.c2000.cn/user1/104/10008.html |archivedate=2010-06-05 }}</ref>，甚至有網友批評:大福路改名突顯政府急功近利的浮躁心態。<ref>广东地市新闻·南方网,佛山大福路拟改名为岭南大道引发网友热论,2006-10-17,14:07:57 {{cite web |url=http://www.southcn.com/news/dishi/foshan/shehui/200610170623.htm |title=存档副本 |accessdate=2008-08-01 |deadurl=yes |archiveurl=https://web.archive.org/web/20160304141736/http://www.southcn.com/news/dishi/foshan/shehui/200610170623.htm |archivedate=2016-03-04 }}  簡體字</ref>但是佛山民政局方面聲稱:"更名方案经征询[[市民|市民]]意见和[[地名专家|地名专家]]论证"<ref>佛山市民政局,中心组团七条主干道路命名更名敲定,2006-10-09 http://www.fsmz.gov.cn/mzdt/mzxw/200610/t20061009_175984.htm</ref>，同時佛山政府並沒有提供給[[市民|市民]]反應相關意見的途徑，亦無公佈地名[[專家|專家]]名單。最終大福路亦逃不過被改名的命運。
