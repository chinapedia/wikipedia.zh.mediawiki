'''大和國'''（{{jpn|j = 大和国|hg = やまと の くに|rm = Yamato no kuni|kyu = 大和國|hgt = |f = y}}），[[日本|日本]]古代的[[令制国|令制国]]之一，属[[畿内|畿内]]五国之一，又称“和州”<ref name=Yamato-Ktbk>{{cite web|url=https://kotobank.jp/word/%E5%A4%A7%E5%92%8C%E5%9B%BD-144220|script-title=ja:大和国|publisher=朝日新聞社Kotobank|accessdate=2018-09-07|archive-url=https://web.archive.org/web/20180908164848/https://kotobank.jp/word/%E5%A4%A7%E5%92%8C%E5%9B%BD-144220|archive-date=2018-09-08|dead-url=no}}</ref>，古称「[[倭|倭]]」、「大倭」、「大養徳」<ref name=Tokugawa1909287>{{cite book |author1=德川光圀 |author2=etal |title=大日本史 第140-141 卷287-289 志 |date=1907 |publisher=徳川总子|doi=10.11501/769983|url=http://dl.ndl.go.jp/info:ndljp/pid/769983}}</ref>，在《[[延喜式|延喜式]]》中国力属于大国，领域大约相当于现代的[[奈良縣|奈良縣]]<ref name="Frédéric2005">{{cite book|author=Louis-Frédéric|script-title=ja:Japan Encyclopedia|url=https://books.google.com/books?id=p2QnPijAEmEC&pg=PA1046|year=2005|publisher=Harvard University Press|isbn=978-0-674-01753-5|page=1046|access-date=2018-09-08|archive-url=https://web.archive.org/web/20161219184031/https://books.google.com/books?id=p2QnPijAEmEC&pg=PA1046|archive-date=2016-12-19|dead-url=no}}</ref>。

==名称==
大和國古称「倭」、「大倭」、「大養德」，发音都是相同的“Yamato”（{{lang|ja|ヤマト}}）。根据《[[续日本纪|续日本纪]]》的记载，[[奈良時代|奈良時代]]中的[[天平_(聖武天皇)|天平]]九年十二月廿七日（[[公元|公元]]738年1月21日），朝廷将「大倭國」改称「大養德國」<ref>{{cite book|author=菅野真道|script-title=ja:[[續日本紀|續日本紀]] 卷第十二|quote=丙寅。改大倭國。爲大養徳國。}}</ref>。[[天平宝字|天平宝字]]元年（公元757年），「大養德國」改称「大和國」<ref name=Yamato-Ktbk/>。

根据《[[古事類苑|古事類苑]]》、《[[釋日本紀|釋日本紀]]》的说法，传说[[開天闢地_(日本神話)|天地開闢]]而後泥土未乾故人民居住於山，因人跡多見而称此國為「山跡」（{{lang|ja|やまと}}），後改為同音的「大和」<ref>{{Cite book|editor=神宮司庁古事類苑出版事務所|script-title=ja:古事類苑 地部2|section=大和國|publisher=神宮司庁|year=1896-1914|id={{NCID|BN07926698}}|url=http://dl.ndl.go.jp/info:ndljp/pid/897580/35?tocOpened=1}}</ref><ref>{{Cite book|author = 卜部兼方|script-title=ja:釈日本紀（《国史大系. 第7巻》所収）|location = [[東京|東京]]|publisher = {{link-ja|経済雑誌社}}|date = 1898年8月6日|accessdate = 2018年12月12日|url = http://dl.ndl.go.jp/info:ndljp/pid/991097/274|id={{NCID|BN01668840}}}}</ref>。

==历史==
大和国是大和朝廷的发源地，历史上境内曾有{{link-ja|鬥雞國造|闘鶏国造}}、{{link-ja|大倭国造|倭国造}}、{{link-ja|葛城国造}}，以及菟田、春日、層富、山边、十市、高市、磯城等由县主管理的县存在。此外，大和国境内也曾存在过吉野国（后来成为大和国的一个郡[[吉野郡|吉野郡]]）和[[芳野监|芳野监]]两个行政建制。[[天平宝字|天平宝字]]元年（公元757年），此地的名称由“大养德国”改为“大和国”。{{link-ja|飛鳥京}}、[[藤原京|藤原京]]、[[平城京|平城京]]都在大和国境内。奈良时代前后，朝廷曾在大和国境内修建许多[[佛教|佛教]][[寺庙|寺庙]]，如[[飛鳥寺|飛鳥寺]]、[[法隆寺|法隆寺]]。784年，朝廷将都城迁到[[山城国|山城国]]境内的[[长冈京|长冈京]]，很快又再度迁入[[平安京|平安京]]（即现代的[[京都|京都]]）<ref name=Yamato-Ktbk/><ref name=Tokugawa1909287/><ref name="池田末則2004">{{cite book|author=池田末則|script-title=ja:地名伝承学論|url=https://books.google.com/books?hl=zh-TW&id=1-s4AAAAMAAJ&focus=searchwithinvolume&q=%E8%8A%B3%E9%87%8E|year=2004|publisher=クレス出版|page=450|ISBN=978-4877332297}}</ref><ref name="大岡信日本うたことば表現辞典刊行会2008">{{cite book|author1=大岡信|author2=日本うたことば表現辞典刊行会|script-title=ja:日本うたことば表現辞典: 歌枕編|url=https://books.google.com/books?id=4-gZAQAAMAAJ&q=%E9%A3%9B%E9%B3%A5%E4%BA%AC+%E8%97%A4%E5%8E%9F%E4%BA%AC+%E5%B9%B3%E5%9F%8E%E4%BA%AC+%22%E5%A4%A7%E5%92%8C%E5%9B%BD%22&dq=%E9%A3%9B%E9%B3%A5%E4%BA%AC+%E8%97%A4%E5%8E%9F%E4%BA%AC+%E5%B9%B3%E5%9F%8E%E4%BA%AC+%22%E5%A4%A7%E5%92%8C%E5%9B%BD%22&hl=zh-TW&sa=X&ved=0ahUKEwjctryItqzgAhV3wcQBHTSsAtYQ6AEITzAG|year=2008|publisher=遊子館|ISBN=978-4946525926}}</ref>。

[[平安时代|平安时代]]后期，大和国成为以[[源满仲|源满仲]]的次子{{link-ja|源赖亲|源頼親}}为家祖的{{link-ja|大和源氏}}根据地<ref>{{cite web|url=https://kotobank.jp/word/大和源氏-1430114|script-title=ja:大和源氏|publisher=朝日新聞社Kotobank|accessdate=2019-02-08}}</ref>。[[日本南北朝时代|日本南北朝时代]]的[[吉野朝廷|吉野朝廷]]（南朝）的根据地[[吉野|吉野]]也位于大和国境内<ref>{{cite web|url=https://kotobank.jp/word/南朝-108871|script-title=ja:南朝|publisher=朝日新聞社Kotobank|accessdate=2019-02-08|archive-url=https://web.archive.org/web/20160403143142/https://kotobank.jp/word/%E5%8D%97%E6%9C%9D-108871|archive-date=2016-04-03|dead-url=no}}</ref>。

从[[镰仓时代|镰仓时代]]到[[室町时代|室町时代]]，都没有设置大和国的[[守护_(职位)|守护]]，大和国实际上由[[興福寺_(奈良市)|興福寺]]统治<ref name=Yamato-Ktbk/>。[[安土桃山时代|安土桃山时代]]时，大和國境内的土地均大多由[[寺社|寺社]]控制。这些寺社具有一定军事实力，在当地有很高发言权，因此当时大和国又称为“神国”。[[丰臣政权|丰臣政权]]时代，[[丰臣秀吉|丰臣秀吉]]的弟弟[[丰臣秀长|丰臣秀长]]在大和国境内通过[[检地|检地]]、[[刀狩|刀狩]]等手段弱化了当地寺社的势力<ref>{{cite web|url=https://www.sankei.com/west/news/140102/wst1401020051-n2.html|script-title=ja:【秀吉と大和の城（２）】大阪の“副都”担った「郡山城」…秀吉の命受け弟・秀長が統治、強力な寺社勢力抑え１００万石規模に（2/2ページ）|website=産経WEST|date=2014年1月2日|accessdate=2019-02-08|archive-url=https://web.archive.org/web/20190114210309/https://www.sankei.com/west/news/140102/wst1401020051-n2.html|archive-date=2019-01-14|dead-url=no}}</ref>。

进入[[江户时代|江户时代]]以后，[[德川幕府|德川幕府]]曾在大和国设置奈良奉行。大和国的部分领地是幕府的直辖领地，其馀土地分属于7个[[藩_(日本)|藩]]。1871年[[废藩置县|废藩置县]]时，大和国改为[[奈良县|奈良县]]。1876年，奈良县併入[[堺县|堺县]]，随后又在1881年併入[[大阪府|大阪府]]。1887年，在当地的要求下，奈良县再度独立并延续至今<ref name=Yamato-Ktbk/>。

==行政区划==
===国厅、国分寺与一宫===
大和国[[国府|国府]]的国厅最初位于今[[奈良县|奈良县]][[高市郡|高市郡]][[高取町|高取町]]一带，后来迁到今奈良县[[大和郡山市|大和郡山市]]今国府町一带。大和国的[[国分寺|国分寺]]是今属奈良县[[奈良市|奈良市]][[东大寺|东大寺]]，也是古代日本全国的总国分寺。大和国的国分尼寺是今属奈良县奈良市的[[法華寺_(奈良市)|法華寺]]。东大寺和法華寺全境均为日本的国之史迹。东大寺的金堂（大佛殿）、南大门、[[卢舍那佛|卢舍那佛]]（大佛）等已指定为[[日本国宝|日本国宝]]，中门、石造狮子等是[[重要文化财|重要文化财]]。法华寺中的木造十一面观音立像已指定为日本国宝、本堂以及木心乾漆维摩居士坐像等是重要文化财<ref>{{cite web|url=https://kotobank.jp/word/法華寺-133479|script-title=ja:法華寺|publisher=朝日新聞社Kotobank|accessdate=2019-02-08}}</ref><ref>{{cite web|url=https://kotobank.jp/word/東大寺-103867|script-title=ja:東大寺|publisher=朝日新聞社Kotobank|accessdate=2019-02-08|archive-url=https://web.archive.org/web/20160329163932/https://kotobank.jp/word/%E6%9D%B1%E5%A4%A7%E5%AF%BA-103867|archive-date=2016-03-29|dead-url=no}}</ref>。

大和国的[[一宫|一宫]]是今属奈良县[[樱井市|樱井市]]的[[大神神社|大神神社]]。大神神社是日本的国之史迹，攝社大直禰子神社社殿、拜殿、三鸟居、朱漆金銅装盾、《[[周書|周書]]》 第十九卷等是重要文化财<ref>{{cite web|url=https://kotobank.jp/word/大神神社-39601|script-title=ja:大神神社|publisher=朝日新聞社Kotobank|accessdate=2019-02-08}}</ref>。
