'''大肚溪北號誌站'''位於[[台灣|台灣]][[臺中市|臺中市]][[大肚區|大肚區]]，曾為[[臺灣鐵路管理局|臺灣鐵路管理局]][[縱貫線_(南段)|縱貫線]]、[[臺中線|臺中線]]、[[海岸線_(臺鐵)|海岸線]]（現已廢止）的鐵路號誌站。

== 車站構造 ==
*已消失。
*早期海線與山線鐵路由本站分歧。

== 利用狀況 ==
*已廢止。
*山海線南邊轉運車站（南王田）。
*供山線列車利用流量較小的海線路軌（大肚溪）。
*為免山線大肚溪橋被大水沖毀而改建，同時將山線列車切換至海線大肚溪橋（大肚溪北）。

== 車站週邊 ==
*中山高速公路王田交流道

== 歷史 ==
*1929年7月10日：設立南王田站。
*1935年10月1日：廢站。
*1946年8月12日：設立大肚溪號誌場。<ref name="XN">《慶祝中部鐵路通車100週年 彰化扇形車庫修復落成 臺灣鐵路創建118週年紀念專輯》第32頁 2005年 李慶金編撰 彰化縣文化局發行</ref>
*1949年6月6日：廢站。
*1963年7月12日：設立大肚溪北號誌站。<ref name="XN" />
*1987年4月20日：大肚溪新橋建成後廢站。<ref name="XN" /><ref>另《交通年鑑》記載大肚溪新橋於1988年2月4日完工通車。交通部，[[中華民國|中華民國]]78年《交通年鑑》，[[臺北市|臺北市]]，1990年10月10日，頁383。</ref>
