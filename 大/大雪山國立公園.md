'''大雪山國立公園'''（{{jpn|j=大雪山国立公園|hg=だいせつざんこくりつこうえん}}）為[[日本|日本]][[北海道|北海道]]中部的[[日本国立公园|國立公園]]，成立於1934年12月4日。面積2,267.64平方千米，比[[東京都|東京都]]的面積還大，為全日本最大的國立公園<ref>{{Cite web |url=http://www.env.go.jp/park/doc/data/national/np_3.pdf |title=国立公園土地所有別面積 |format=PDF |publisher=環境省 |accessdate=2016-08-20}}</ref>，其中98％為國有地，加上北海道道有地則達到99％。

公園以[[旭岳|旭岳]]、北鎮岳、白雲岳、[[黑岳|黑岳]]、赤岳，以及[[大雪山_(北海道)|大雪山系]]的[[十勝岳|十勝岳]]、[[富村牛山|富村牛山]]、{{lang|ja|ニペソツ山}}、石狩岳等高度約2000公尺左右的群山所組成。區域內有多種動物和[[高山植物|高山植物]]。

== 概要 ==
大雪山國立公園於1934年12月4日與[[阿寒摩周國立公園|阿寒摩周國立公園]]<ref>1934年（昭和9年）12月4日内務省告示第567号「阿寒國立公園指定」</ref>、[[日光国立公園|日光国立公園]]<ref>1934年（昭和9年）12月4日内務省告示第569号「日光國立公園指定」</ref>、[[中部山嶽國立公園|中部山嶽國立公園]]<ref>1934年（昭和9年）12月4日内務省告示第570号「中部山岳國立公園指定」</ref>、阿蘇國立公園（現在[[阿蘇九重國立公園|阿蘇九重國立公園]]）<ref>1934年（昭和9年）12月4日内務省告示第571号「阿蘇國立公園指定」</ref>同時成為國立公園<ref>1934年（昭和9年）12月4日内務省告示第568号「大雪山國立公園指定」</ref>，是北海道最為悠久的國立公園之一。園內有大雪火山群、十勝岳連峰、石狩岳連峰等山岳，加上地處高緯度，形成{{link-ja|雪溪|雪渓}}與[[永凍土|永凍土]]層等特殊景觀{{Sfn|大雪山国立公園管理計画書|2007|p=1}}。園內的天然紀念物有在1971年（昭和46年）成為國家[[天然紀念物|天然紀念物]]的[[大雪山|大雪山]]<ref>[http://bunka.nii.ac.jp/heritages/detail/171265 大雪山]</ref>，1977年（昭和52年）改為特別天然紀念物{{Sfn|大雪山国立公園管理計画書|2007|p=7}}，以及同樣為天然紀念物的{{link-ja|薄羽黃蝶|ウスバキチョウ}}與{{link-ja|日本酒眼蝶|ダイセツタカネヒカゲ}}等高山蝶，和[[黑啄木鳥|黑啄木鳥]]、[[金雕|金雕]]、{{link-ja|島梟|シマフクロウ}}等鳥類。1951年（昭和26年），{{link-ja|羽衣瀑布|羽衣の滝}}成為北海道名勝，之後{{link-ja|然別湖|然別湖}}的「{{link-ja|宮剖紅點鮭|ミヤベイワナ}}」在1968年（昭和43年）成為北海道天然紀念物{{Sfn|大雪山国立公園管理計画書|2007|p=7}}<ref>{{Cite web |url=http://www.dokyoi.pref.hokkaido.lg.jp/hk/bnh/grp/01/21shitei-bunkazai-do-tennen.pdf |title=道指定文化財概要一覧（天然記念物） |format=PDF |work=北海道教育委員会 |accessdate=2016-08-24}}</ref>。

== 歴史 ==
* 1934年（[[昭和|昭和]]{{0}}9年）：指定為國立公園。
* 1977年（昭和52年）：十勝川源流地區解除指定（改為{{link-ja|原生自然環境保全地域|原生自然環境保全地域}}）。
* 2000年（平成12年）：層雲峽遊客中心開幕<ref>{{Cite web |url=http://www.town.hokkaido-kamikawa.lg.jp/section/kikakusoumu/chs8120000000hy5.html |title=まちづくりのあゆみ |publisher=北海道上川町 |accessdate=2016-08-20}}</ref>。
* 2013年（平成25年）：東大雪自然館開幕<ref>{{Cite web |url=http://www.kamishihoro.jp/entry/00001408 |title=ひがし大雪自然館がオープンしました |date=2013-05-01 |publisher=北海道上士幌町 |accessdate=2016-08-20}}</ref>。
== 自然 ==
大雪山國立公園內可見250種以上的[[高山植物|高山植物]]<ref>{{Cite book|author= |year=2007 |month=05 |title=花の百名山地図帳 |publisher=山と溪谷社 |pages=20-25 |isbn=9784635922463}}</ref><ref name="高山植物">{{Cite web |url=http://www.daisetsuzan.or.jp/nature/ |title=大雪山の自然〜高山植物 |publisher=大雪山国立公園連絡協議会 |accessdate=2016-08-19}}</ref>。另外，還有290種以上的指定植物與60種以上的保育紅皮書物種（僅計高等植物）<ref>{{Cite web |url=https://www.env.go.jp/nature/ari_kata/shiryou/031208-4-10.pdf |title=各国立公園及び特定地域におけるRDB種及び指定植物種 |format=PDF |publisher=環境省 |accessdate=2016-08-20}}</ref><ref>{{Cite web |url=http://www.env.go.jp/nature/np/plant_prot/mat02_2.pdf |title=指定植物リスト |date=2016-02-08 |format=PDF |publisher=環境省 |accessdate=2016-08-20}}</ref>。[[鳥類|鳥類]]目前已發現140種以上<ref name="鳥たち">{{Cite web |url=http://www.daisetsuzan.or.jp/nature/bird/ |title=大雪山の鳥たち |publisher=大雪山国立公園連絡協議会 |accessdate=2016-08-19}}</ref>，包括日本國內首個繁殖紀錄的[[松雀|松雀]]與稀少種[[三趾啄木鳥|三趾啄木鳥]]、[[黑啄木鳥|黑啄木鳥]]、{{link-ja|シマフクロウ|毛腿魚鴞}}、[[鬼鴞|鬼鴞]]、數量減少的[[花尾榛雞|花尾榛雞]]等<ref name="鳥たち"/>。另外還可見到北海道所有的五種高山蝶（{{link-ja|艾雯絹蝶|ウスバキチョウ}}、牯灰蝶 、{{link-ja|佛珍蛺蝶|アサヒヒョウモン}}、淡酒眼蝶、波翅紅眼蝶）<ref>{{Cite web |url=http://www.daisetsuzan.or.jp/nature/butterfly/ |title=大雪山の高山蝶 |publisher=大雪山国立公園連絡協議会 |accessdate=2016-08-19}}</ref>以及3,000種以上的[[昆蟲|昆蟲]]<ref>{{Cite web |url=http://www.daisetsuzan.or.jp/nature/insect/ |title=大雪山の昆虫 |publisher=大雪山国立公園連絡協議会 |accessdate=2016-08-19}}</ref>。[[動物|動物]]では[[棕熊|棕熊]]、[[北海道赤狐|北海道赤狐]]、[[貉|貉]]、{{link-ja|北海道鼬|エゾイタチ}}、[[紫貂|紫貂]]、{{link-ja|北海道白鼬|エゾオコジョ}}、{{link-ja|北海道雪兔|エゾユキウサギ}}、{{link-ja|北海道鼠兔|エゾナキウサギ}}、{{link-ja|エゾシマリス|北海道花栗鼠}}、{{link-ja|北海道小飛鼠|エゾモモンガ}}、{{link-ja|北海道紅松鼠|エゾリス}}、{{link-ja|北海道梅花鹿|エゾシカ}}、[[嚙齒目|嚙齒目]]類、[[鼩鼱科|鼩鼱科]]類等<ref>{{Cite web |url=http://www.daisetsuzan.or.jp/nature/animal/ |title=大雪山の動物 |publisher=大雪山国立公園連絡協議会 |accessdate=2016-08-19}}</ref>。

=== 山岳 ===
大雪山國立公園可分為北大雪、表大雪、東大雪、十勝連峰四個山域<ref name="山々">{{Cite web |url=http://www.daisetsuzan.or.jp/nature/mount/ |title=大雪山の山々 |publisher=大雪山国立公園連絡協議会 |accessdate=2016-08-19}}</ref>。由於氣象條件與緯度影響，[[林線|林線]]低至1,500公尺左右，因此高山地帶廣大<ref name="山々"/>。登山季僅限於6月下旬至10月初旬。通常7月上旬仍有殘雪，而第一場降雪大約會在9月中旬<ref name="山々"/>。
[[大雪山|大雪山]]、沼之平、[[十勝岳|十勝岳]]、{{link-ja|富良野岳|富良野岳}}為{{link-ja|花之百名山|花の百名山}}之一<ref>{{Cite book|author=田中澄江 |year=1997 |month=6 |title=花の百名山（愛蔵版） |publisher=[[文藝春秋|文藝春秋]] |pages=161-178 |isbn=4-16-352790-7}}</ref>，黑岳、小泉岳、白雲山則入選{{link-ja|新花之百名山|新・花の百名山}}<ref>{{Cite book|author=田中澄江 |year=1995 |month=06 |title=新・花の百名山 |publisher=文春文庫 |pages=28-32 |isbn=4167313049}}</ref>。
