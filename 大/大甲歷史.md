==地名起源<ref name="大甲道卡斯"> {{Cite book| author =李安娜等| title =《大甲道卡斯 : 鐵砧山風景特定區導覽》| location =臺灣臺中| publisher =臺中縣大甲鎮公所| date =2004-03-23| pages =11| ISBN =| accessdate =2016-10-02| url =http://old.dajia.taichung.gov.tw/book10/EB00K11-A02.htm#1| language =zh-tw| quote =| deadurl =yes| archiveurl =https://web.archive.org/web/20160824002933/http://old.dajia.taichung.gov.tw/book10/EB00K11-A02.htm#1| archivedate =2016-08-24}} </ref>==
關於大甲之名的起源，存在兩種說法：
*[[河洛話|河洛話]]的大甲類似道卡斯(TAOKAS)的譯音，源於當時外來的漢人對以當地[[道卡斯族|道卡斯族]]稱呼此地地名，而由此推斷部分學者認為大甲為[[道卡斯族|道卡斯族]]的大本營。
*源於易經拆字學所帶來的不同說法，「大」- 「大字頭出天，兩腳八字開」；「甲」- 「甲字成田，帶有尾」。

==歷史紀實==
===史前時期===
大甲地區為[[平埔族|平埔族]][[道卡斯族|道卡斯族]]主要活動範圍，屬於蓬山八社聚落中的[[大甲西社|大甲西社]]、[[大甲東社|大甲東社]]、[[日南社|日南社]]與[[雙寮社|雙寮社]]<ref name="大甲道卡斯"/> 。根據考古所發現的史前遺址，大甲地區最早史前文化為距今4000~3000年前，屬於[[新石器時代|新石器時代]][[牛罵頭文化|牛罵頭文化]]的麻路頭遺址與水源地遺址；接著為距今3500~2000年前屬於新石器時代後期的[[營埔文化|營埔文化]]之麻路頭遺址、水源地遺址與[[鐵砧山|鐵砧山]]平頂遺址；最後則是距今2000~400年鐵器時代[[番仔園文化|番仔園文化]]的[[番仔園遺址|番仔園遺址]]<ref>{{Cite web |url = http://digimuse.nmns.edu.tw/da/collections/ha/a0/intro/ |title = 臺灣史前文化 臺灣中部考古學概述 |author = [[國立自然科學博物館|國立自然科學博物館]]、數位典藏國家型科技計畫 |date =   |publisher = [[國立自然科學博物館|國立自然科學博物館]]、數位典藏國家型科技計畫  |language = zh-tw |accessdate = 2016-10-02 |quote =  }}</ref>。

===明鄭時期===
漢族移墾大甲，據淡水廳志記載，明[[永曆|永曆]]19年（1665年），諮議參軍[[陳永華_(明朝將領)|陳永華]]請設[[屯田制|屯田制]]，於是[[鄭經|鄭經]]屯軍大甲溪，北上開拓大甲之地，是年於大甲社置通事，負責與原住民接觸。永曆24年（1670年）鄭經命右武衛[[劉國軒|劉國軒]]出鎮[[半線|半線]]，劉國軒命副將駐大甲[[鐵砧山|鐵砧山]]，經略蓬山八社。大甲地區之平埔族與漢人較大規模的正式接觸始於此，不過此時閩粵移民尚未在此長期定居，到大甲的漢人只是通事或社商，一直到滿清入臺，人口相當稀少。

===清朝階段===
[[康熙|康熙]]40年（1701年）有大批漢人一自[[鹿港鎮|鹿港]]北進，一從[[大安港|大安港]]上岸移入本區求生，其中閩籍有林、張二姓，粵籍有邱姓（邱太三即其後人）率眾開墾九張犁、[[日南|日南]]、大安及鐵砧山腳等荒埔，至此大甲地區因開墾而日漸發展，到康熙45年大甲已形成一小型市街。[[雍正|雍正]]12年（1734年）[[林成祖|林成祖]]自中國大陸來臺定居大甲拓地，歲入穀萬石，對本地區開發貢獻頗大。

''大甲西社抗清事件''

'''大甲西社抗清事件'''為台灣原住民[[平埔族|平埔族]]歷史上最大的民眾起義事件，以[[大甲西社|大甲西社]]的[[林武力|林武力]]為首。發生於[[台灣清領時期|台灣清領時期]][[雍正|雍正]]年間的該事件，持續長達一年。事件過後不僅[[漢人|漢人]]與平埔各族群間傷亡慘重，也讓[[清廷|清廷]]於平埔各族境內加強駐兵，加強控制平埔族群。此「番亂」名稱，來自[[福建|福建]][[提督|提督]][[許良彬|許良彬]]奏摺所稱。

此外，事件爆發後，[[鳳山|鳳山]]總兵奉令率軍前往大甲協助平定，導致鳳山守備鬆散，曾參與[[朱一貴事件|朱一貴事件]]的舊部便趁機起事，造成[[鳳山|鳳山]]的[[吳福生事件|吳福生事件]]<ref>[[吳福生事件|吳福生事件]]</ref>。

== 事件經過 ==
1731年（清[[雍正|雍正]]九年）年末，原住民族[[道卡斯族|道卡斯族]]位於[[大甲區|大甲]]一帶的[[大甲西社|大甲西社]]，因清廷官吏指派勞役過多而群起反抗，發動武裝抗官行動；燒毀[[臺灣府淡水撫民同知|臺灣府淡水撫民同知]]衙門，殺傷衙役兵丁，同知[[張弘章|張弘章]]逃逸。事發之初，時任[[臺灣鎮總兵|臺灣鎮總兵]][[呂瑞麟|呂瑞麟]]正好北巡到北台灣，聞變趕回貓盂（在今[[苗栗縣|苗栗縣]][[苑里鎮|苑里鎮]]），被原住民起事者圍困。隨後，呂某突圍入彰化縣治，並向[[臺灣府|臺灣府]]徵兵合攻，仍未平復。

1732年八月，負責征討大甲西社亂事的[[福建分巡臺灣道|福建分巡臺灣道]][[倪象愷|倪象愷]]的劉姓表親為求立功，竟將[[大肚社|大肚社]]（在今臺中市[[大肚區|大肚區]]）五名前來幫助官府運糧的「良番」（歸化的[[原住民|原住民]]）斬首，謊稱這五人是大甲西社的「作亂生番」。此事引起已歸化原住民的不滿，群湧[[彰化縣_(清朝)|彰化縣]]城理論抗議；知縣隨意敷衍，引起大肚社原住民不滿，於是聯合原[[大肚王國|大肚王國]]成員之部落，即[[南大肚社|南大肚社]]、[[沙轆社|沙轆社]]（在今臺中市[[沙鹿區|沙鹿區]]）、[[牛罵頭社|牛罵頭社]]（在今臺中市[[清水區_(臺中市)|清水區]]）、[[樸子籬社|樸子籬社]]（今臺中市[[東勢區|東勢區]]），[[吞霄社|吞霄社]]（今苗栗縣[[通霄鎮|通霄鎮]]）、[[阿里史社|阿里史社]]（今臺中市[[潭子區|潭子區]]）等十餘社的[[平埔族|平埔族]]群原住民約兩千餘人圍攻彰化縣城，焚燒附近數十里民房，漢人百姓逃散。

事件擴大後，其他各社如[[蓬山社|蓬山社]]（今大甲）、[[貓羅社|貓羅社]]（今彰化縣[[芬園鄉|芬園鄉]]）、[[岸裡社|岸裡社]]（今臺中市[[神岡區|神岡區]]）、水裡（今臺中[[龍井區|龍井區]]）、[[阿束社|阿束社]]（今彰化市茄冬腳、彰化縣和美鎮）等原住民番社也起而響應。至此[[大安溪|大安溪]]到[[大甲溪|大甲溪]]之間，原屬於[[大肚王國|大肚王國]]領域內的山谷平原都陷入動盪之中。這是清領時期平埔族群武力反抗事件中規模最大的一次，當時彰化縣（大約為今臺中市、彰化縣之平原地區）境內平埔族各社幾乎全部參加了此行動。

== 派兵鎮壓 ==
當時署理[[閩浙總督|福建總督]][[郝玉麟|郝玉麟]]派[[臺灣總兵|臺灣總兵]][[王郡|王郡]]親往督征，並先後從[[中國大陸|中國大陸]]徵調官兵六千多名，以及[[銀錢|銀錢]]、[[白米|米糧]]、[[軍火|軍火]]、[[冷兵器|器械]]，船載一百多艘來臺征討。九月初，清軍採「以番制番」之策，使各敵對之部落互相攻擊，並得[[岸裡社|岸裡社]]部分原住民的協助，襲破阿束社。十月中旬清軍兵分七路，攻破水裡社、牛罵頭社及沙轆社。十一月初清兵攻進小坪山反抗軍據點，獲[[牛|牛]]千餘頭、[[馬|馬]]八匹、[[車|車]]數百輛，焚毀[[糧食|糧食]]四百堆。最後[[吞霄社|吞霄社]]、[[大甲西社|大甲西社]]、[[貓盂社|貓盂社]]、[[雙嶚社|雙嶚社]]、[[苑裡社|苑裡社]]、[[房裡社|房裡社]]等抵抗失敗，各[[部落|部落]]先後率眾投降，事件才告結束。事件歷經七個月，清軍「計擒獲男、婦一千餘名，陣斬首級四十一名，傷死二十一名，軍前梟首一十八名」<ref>[[周璽|周璽]]《彰化縣志》卷六·田賦志·番丁</ref>，林武力等十三名首領依[[清律|清律]]判決斬決梟示，沙轆土官被判成立唆謀罪名而杖斃<ref>[[國立故宮博物院|國立故宮博物院]]《宮中檔雍正朝奏摺》二十輯</ref>。

事件後，清廷改大甲西社為「德化社」、牛罵頭社為「感恩社」、沙轆社為「遷善社」、貓盂社為「興隆社」，並且建了一個「鎮番亭」於彰化東郊的瞭望山（即[[八卦山|八卦山]]），並改山名為「定軍山」。

== 結果 ==
亂平之後，參與的番社勢力大幅衰退。直到乾隆年間，生計仍然困苦，官方屢次救濟。[[福建分巡臺灣道|福建分巡臺灣道]][[劉良璧|劉良璧]]作〈沙轆行〉形容：「皇恩許遷善，生者還其鄉；番婦半寡居，番同少雁行。嗟乎沙轆番，盛衰物之常。祇今防廳廨，荒烟蔓道旁。」

其後清朝放寬海禁，岸裡社接收大肚王國舊地、鹿場等，領域遍及台中盆地。

====林爽文之亂====
[[乾隆|乾隆]]五十二年（1787年）6月8日[[台灣府淡水撫民同知|淡水同知]][[徐夢麟|徐夢麟]]棄[[彰化市|彰化]]而守大甲，參贊[[藍元枚|藍元枚]]命守備[[潘國材|潘國材]]率兵六百人前來大甲增援。徐鼎士領兵一千八百人來援，結營於大甲溪北，並鑄砲二百餘座，使大甲成為[[中臺灣|中臺灣]]清兵攻剿[[林爽文之亂|林爽文之亂]]的大本營。

==== 大甲城的興建 ====
道光七年（1827年），大甲人口驟增，車馬輻輳，紳民發起捐建[[大甲城|大甲城]]，周圍510丈設門。同年將竹塹營守備移駐大甲，大甲設有守備，千總，把總衙門各一座，[[竹塹|竹塹]]以南，彰化以北全歸[[大甲守備|大甲守備]]節制。

==== 鴉片戰爭 ====
道光二十年（1840年）鴉片戰爭爆發，[[臺灣兵備道|臺灣兵備道]][[姚瑩|姚瑩]]、[[台灣府淡水撫民同知|台灣府淡水撫民同知]][[曹謹|曹謹]]等人鑒於英船出沒臺海，於是請求調撥經費以籌佈署。其時，協防臺灣有淡水同知曹謹、署鹿港[[同知|同知]][[魏瀛|魏瀛]]、澎湖[[通判|通判]][[范學恒|范學恒]]，彰化[[知縣|知縣]][[黃開基|黃開基]]，[[台灣北路協|台灣北路協]][[副將|副將]][[關桂|關桂]]，[[游擊|游擊]][[安定邦|安定邦]]等人一致認為大甲地位重要，須增強力量。

大甲街兵力配置有[[大甲中軍守備|大甲中軍守備]][[何必捷|何必捷]]，[[千總|千總]][[何建忠|何建忠]]，[[把總|把總]][[翁標桂|翁標桂]]，並駐兵丁二百人，作為[[大安港|大安港]]的後盾。大安港兵力重在大砲、鳥銃、長槍，籐牌短刀均缺乏，[[大甲巡檢|大甲巡檢]][[劉其鐘|劉其鐘]]帶鄉勇一百名協助。1842年初英船數度窺伺大安港，有強行登陸之勢，竹塹名士[[鄭用錫|鄭用錫]]知聞大甲沿海告急，自募兵百餘人前往大甲，參加大安港防禦工作。2月英軍準備登陸大安港，被守軍擊沉兵船一隻，英軍遂退去，是為[[大安之役|大安之役]]。

'''''大安之役'''''   

清方記載編輯

道光廿二年正月二十四日英船「阿吶」(Brig Ann，又譯作「阿恩」號)至梧棲港（今[[台中港|台中港]]）探訪後，在外洋滯留。因[[水師|水師]]實力不如英方，[[台灣鎮|台灣鎮]][[達洪阿|達洪阿]]通知[[淡水區|淡水]]與[[鹿港鎮|鹿港]]等地守兵嚴加戒備，令其不得與英國進行海戰，並決定誘敵入台灣內陸河川擱淺。

正月三十日，阿吶號想從位於台灣中部[[大安溪|大安溪]]的大安港上岸，清軍於是派數艘漁船誘導其入河內多礁處。英軍因不諳地勢，隨後於大安溪中段處擱淺。達洪阿即派兵前往伏擊，最後英船被擊沉，數十名英國人被殺。另外尚有十八名英國軍官，一名紅人（印地安人），三十名印度人，五名[[廣東|廣東]]、[[九龍|九龍]]人遭俘虜，清軍並在船上擄獲11門大砲等兵器，及該英艦先前於中國長江海戰獲得的少數中國戰利品。

=== 英方記載編輯 ===
臺灣學者章瑄文『綜合「阿吶」號當事人與戰時其他英國軍士於事後出版之記錄，則該船於1842年3月8日（道光廿二年正月廿七日）自舟山攜帶財貨起錨南駛，目的地為澳門。 途中船長顛林見風勢順暢，甚至計畫於泉州、廈門稍事停留進行補給。然自9日 傍晚起風勢轉強，大雨驟起，濃霧中海面能見度極低，「阿吶」號遂逐漸偏離原訂航線，終至誤觸暗礁。其後「阿吶」號經歷一整日風雨飄搖，乃於3月11日（正月卅日）清晨擱淺於臺灣洋面，當時船員雖試圖搶救武器財貨，然因船身已遭嚴重損毀，船尾沖失、桅杆斷折，尤其火礮彈藥幾乎已全數不堪使用；眼看風浪日強，情勢危急，船長乃決定趁退潮之際棄船登岸尋找接駁船隻，俾能早日離開此地前往泉州，稍早船員中尚有人認為船隻係擱淺於烏坵嶼 (OcKseu)，直至天色大明，才自岸邊景況判斷自己身處福爾摩沙島北方淡水與雞籠之間海岸。此時船上成員中已有一名印度水手與一名中國看銀師先是遭海潮沖走，繼而 為臺灣官方俘虜。其餘船員遂於11日早晨七時左右，攜帶少許僅存武器、羅盤、望遠鏡與糧食衣物，棄船往東北行進。一行人沿水深及膝淺灘上行未久，越過沙洲，便發現一淺水小灣，當時灣中有四艘滿載米穀之舢舨，船上漁民友善招呼英人前往，「阿吶」號船員乃請中國籍木匠與看銀師擔任翻譯，登船與船夫溝通，表示願以三千元酬勞重金雇用其中兩艘船將英人帶往泉州。船夫起初表示，若風向配合，則願意考慮，並友善招待英人享用餐食；英人特別指出，此時雙方並未發生任何衝突。此時天又下雨，英人全身濕透，狼狽不堪。肐哩乃感嘆自身處境：「身處敵人的土地上，卻沒有任何武裝可供自保」。隨即英人便遭小灣北方岸上聞訊集結而來之全副武裝官民成群包圍並投擲石塊攻擊，人數有數百至數千不等，其後更有一名乘轎官員帶領一隊兵士抵達堤岸加入攻擊行動。雙方僵持數小時後，顛林船長見勢不可為，遂令「阿吶」號船員主動棄械投降。孰料隨後數百官民一擁而上，對毫無還擊能力之船員施以毒打淩虐，並威脅將之殺害。水手們非但於寒風中被強迫脫去身上衣物，而後更遭重枷繫身綁赴淡水廳治，而「阿吶」號船上所裝載貨物與船員個人財物亦遭一干兵民掠奪殆盡』。

====戴潮春事件====
同治元年（1862年），[[戴潮春|戴潮春]]命將攻陷大甲，臺灣南北交通因此中斷。清軍急調數路援軍，[[竹塹|竹塹]]的[[林占梅|林占梅]]，翁仔社的[[羅冠英|羅冠英]]，[[噶瑪蘭|噶瑪蘭]]的屯勇。是年五月大甲城被清軍奪回，為戴潮春所占的[[彰化市|彰化]]城益形孤立，於是戴潮春急調大軍圍攻大甲，於大甲激戰數日。1863年正月，戴潮春主力在大甲慘敗後，其根據地亦無法防守，最後失敗。

===日治時期===
1895年6月24日，日本吉野艦砲轟[[大安港|大安港]]及大甲郊外三里處，鎮民震驚。7月4日，凌晨三時許，日大隊再攻大甲，[[湯人貴|湯人貴]]領軍相峙，[[徐驤|徐驤]]由側突擊，日軍稍退。時[[李惟義|李惟義]]軍撤至大甲街，築壘備戰。臺民偽裝敵兵突入大甲，惟義再撤退。鄉民具酒食迎日軍。黃海、吳朝宗、謝漢秋、朱麗等鄉紳前往大安港迎日軍進北門，大甲街遂淪於日軍之手。

1896年，[[大甲鎮瀾宮|大甲鎮瀾宮]]充作日軍衛戍醫院。日商人成井元齡以日本草帽教[[苑裡鎮|苑裡]]西勢庄女子[[洪鴦|洪鴦]]用藺草模仿編織，名「[[大甲草帽|大甲草帽]]」，鄰近因相仿效編織。1897年，黃海任大甲街長，朱麗任副街長。

===中華民國時期===
1945年8月15日，日本宣佈投降，中華民國接管[[臺灣|臺灣]]，臺人聞之欣喜，在大甲民間知悉時，最大感覺即「空襲、掃射」結束，天空歸於平靜，但經常有臺人毆打日人的事件。日本宣佈投降後，日本人被管制集結，準備被遣送回日，許多百姓明目張膽侵入日籍教師寓所，竊取財物；大甲的日人到[[臺中市|臺中市]]集合，坐火車到[[基隆|基隆]]。

1946年1月，公布鄉鎮組織規程草案，改「庄、街」為「鄉、鎮」，並定里名。3月大甲家政女學校改稱為大甲女子家政學校，[[郭朝輝|郭朝輝]]接收並任校長，4月18日又改為[[臺中縣立大甲初級中學|臺中縣立大甲初級中學]]（今[[國立大甲高級中學|國立大甲高級中學]]），11月[[陳炘|陳炘]]籌組的臺灣株式會社改名為臺灣信託公司。2010年12月25日，因臺中縣市合併升格為直轄市，[[大甲區|大甲鎮]]改為大甲區。

=== 林爽文之亂 ===
[[乾隆|乾隆]]五十二年（1787年）6月8日[[台灣府淡水撫民同知|淡水同知]][[徐夢麟|徐夢麟]]棄[[彰化市|彰化]]而守大甲，參贊[[藍元枚|藍元枚]]命守備[[潘國材|潘國材]]率兵六百人前來大甲增援。徐鼎士領兵一千八百人來援，結營於大甲溪北，並鑄砲二百餘座，使大甲成為[[中臺灣|中臺灣]]清兵攻剿[[林爽文之亂|林爽文之亂]]的大本營。

=== 大甲城的興建 ===
道光七年（1827年），大甲人口驟增，車馬輻輳，紳民發起捐建[[大甲城|大甲城]]，周圍510丈設門。同年將竹塹營守備移駐大甲，大甲設有守備，千總，把總衙門各一座，[[竹塹|竹塹]]以南，彰化以北全歸[[大甲守備|大甲守備]]節制。

=== 鴉片戰爭 ===
道光二十年（1840年）鴉片戰爭爆發，[[臺灣兵備道|臺灣兵備道]][[姚瑩|姚瑩]]、[[台灣府淡水撫民同知|台灣府淡水撫民同知]][[曹謹|曹謹]]等人鑒於英船出沒臺海，於是請求調撥經費以籌佈署。其時，協防臺灣有淡水同知曹謹、署鹿港[[同知|同知]][[魏瀛|魏瀛]]、澎湖[[通判|通判]][[范學恒|范學恒]]，彰化[[知縣|知縣]][[黃開基|黃開基]]，[[台灣北路協|台灣北路協]][[副將|副將]][[關桂|關桂]]，[[游擊|游擊]][[安定邦|安定邦]]等人一致認為大甲地位重要，須增強力量。

大甲街兵力配置有[[大甲中軍守備|大甲中軍守備]][[何必捷|何必捷]]，[[千總|千總]][[何建忠|何建忠]]，[[把總|把總]][[翁標桂|翁標桂]]，並駐兵丁二百人，作為[[大安港|大安港]]的後盾。大安港兵力重在大砲、鳥銃、長槍，籐牌短刀均缺乏，[[大甲巡檢|大甲巡檢]][[劉其鐘|劉其鐘]]帶鄉勇一百名協助。1842年初英船數度窺伺大安港，有強行登陸之勢，竹塹名士[[鄭用錫|鄭用錫]]知聞大甲沿海告急，自募兵百餘人前往大甲，參加大安港防禦工作。2月英軍準備登陸大安港，被守軍擊沉兵船一隻，英軍遂退去，是為[[大安之役|大安之役]]。
