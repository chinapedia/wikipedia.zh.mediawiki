'''大蛇站'''（{{jpn|j=大蛇駅|hg=おおじゃえき|rm=Ōja eki|f=y}}）是[[東日本旅客鐵道|東日本旅客鐵道]]（JR東日本）[[八戶線|八戶線]]沿線的[[鐵路車站|鐵路車站]]，位於[[青森縣|青森縣]][[三戶郡|三戶郡]][[階上町|階上町]]大字道佛字大蛇。

== 車站結構 ==
[[側式月台|側式月台]]1面1線的[[地面車站|地面車站]]。[[八戶站|八戶站]]管理的[[無人站|無人站]]。

== 車站周邊 ==
小村落。赤保内的日本[[七叶树|七叶树]]是县自然保护植物。

== 历史 ==
* 1956年（[[昭和|昭和]]31年）12月10日－本站開業。
* 1987年（昭和62年）4月1日－[[國鐵分割民營化|國鐵分割民營化]]，本站由JR東日本接手管理。
