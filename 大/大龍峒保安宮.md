'''大龍峒保安宮'''位於[[臺灣|臺灣]][[臺北市|臺北市]][[大同區_(臺灣)|大同區]][[哈密街|哈密街]]，為祭祀[[保生大帝|保生大帝]]的[[臺灣民間信仰|臺灣民間信仰]]廟宇，因為創廟者為來自中國的[[福建|福建]][[同安|同安]]人，因此保安之名有「保佑同安」的意思。保安宮和[[艋舺龍山寺|艋舺龍山寺]]、[[艋舺清水巖|艋舺清水巖]]合稱「臺北三大廟門」<ref>[http://kids.tpml.edu.tw/sp.asp?xdurl=DR/library03_1.asp&id=22641&mp=100 請問臺北三大廟門是指那三座廟？]</ref>。

保安宮初創於1742年，本為簡單木造小[[庵|庵]]，1804年正式改庵建廟。19世紀後又經幾度改建，逐漸發展成3000[[坪|坪]]的三殿三進式大廟。1985年，[[中華民國內政部|中華民國內政部]]認定該廟宇為國家二級古蹟（今[[直轄市定古蹟|直轄市定古蹟]]），2018年11月12日文化部公告升格為國定古蹟  。1995年，因應緊鄰街道擴建，保安宮自費整修，以傳統工法結合現代科技達到「[[平衡工法|平衡工法]]」、「[[整舊如舊|整舊如舊]]」的盡復舊觀效果，成為全臺灣各級古蹟修復的最高典範，並獲得2003年的[[聯合國教科文組織亞太區文物古蹟保護獎|聯合國教科文組織亞太區文物古蹟保護獎]]榮譽獎。

[[日治時代|日治時代]]有兩座廟宇從大龍峒保安宮分靈，一為明治三十年（1897年）[[板橋|板橋]][[江翠|江翠]][[板橋保安宮潮和宮|保安宮潮和宮]]，一為大正五年（1916年）的[[永和保福宮|永和保福宮]]。<ref>[http://au1259.epage.au.edu.tw/ezfiles/54/1054/attach/90/pta_62678_5549214_80359.pdf  張家麟《誰為保生大帝立廟？-以中臺灣以北八縣市宮廟為範圍》真理大學宗教文化與組織管理學系]</ref>

== 興廟沿革 ==

[[File:Dai_fu_chong_jun.JPG|220px]]
[[File:dalong2.JPG|220px]]

大龍峒保安宮主要祭祀的[[保生大帝|保生大帝]]，為[[中國|中國]][[閩南|閩南]]地區所信奉的醫神，俗稱「大道公」、「吳真人」或「花橋公」。

1742年，移居大浪泵漢人因普生[[瘴癘|瘴癘]]，為了祈求醫神力量壓制瘴癘，本地居民便由福建泉州同安[[白礁慈濟宮|白礁慈濟宮]][[分靈|乞靈分火]]來台，於大浪泵落成，取名保安宮。保安宮初建時為一小廟，但迅速成為當地同安人的信仰中心。其後經過兩次小幅度整修，然而在18世紀末，仍只是一座小型木造廟宇。

1804年，四十四坎所在商人集資，將其木造小廟擴建成三殿三進，左右各五開間的大廟宇。經過多年施工後，於1830年完工。1835年後與台灣許多廟宇相同，都是揉合儒（鸞堂信仰）、釋（佛）的道教廟宇。

完工後的這座大廟正式被命名為保安宮，廟名意思有著「保佑同安」的涵義，除此之外，該廟也俗稱大浪泵大道公廟或大浪泵宮。

== 修復與改建 ==
[[File:大龍峒保安宮內殿.jpg|右]]
[[File:大龍峒保安宮36官將1.jpg|缩略图]][[File:大龍峒保安宮12.jpg|缩略图]]

因該廟多為木造雕樑建物，落成後經過1855年、1868年、1917年、1936年、1967年的五次大幅度重修。其中，1917年的整修規模最大，當時[[陳應彬|陳應彬]]與郭塔兩位廟宇師傅，對陣興修，造成現今前殿左右兩邊簷椽雕刻相異，成為台灣少見現存的[[對場作|對場作]]寺廟。該次整修耗時三年，連同鐘樓等建築約等同現今建築面貌。

因為該廟及其用地於[[日治時期|日治時期]]與1950年-1966年，分別被[[台灣總督府|台灣總督府]]與[[國民政府|國民政府]]充作學校與軍眷[[眷村|眷村]]，廟宇增建許多累贅建物。致使1970年代末期，該廟連同廟埕達到最大建物面積，但同時卻略為失去原有建築風貌。

1980年代中期，台灣經濟快速發展，大龍峒地區住宅迅速陸續擴張，該廟部分增建被視為違章建築。另一方面，大廟廟埕部分也被劃分道路用地。為因應此項不利因素發展，大龍峒保安宮副董事長廖武治於1994年展開長達七年廟宇整修過程。
[[File:大龍峒保安宮0.jpg|缩略图]]

該整修過程採用台灣首先平衡工法、整舊如舊的古蹟整建原則，將所有寺廟建物文物做不損壞原貌的整修。另外，最重要的是該廟將日治時期與二戰戰後所有非廟宇之增建，如鋼柱臺基，鋼架塑膠浪板拜亭及多數鋼筋混凝土建物等等累贅建物拆除。
2003年，保安宮在[[聯合國教科文組織|聯合國教科文組織]]亞太地區文化資產保存競賽中9個國家，22個參賽專案中脫穎而出，並獲得亞太文化資產保存獎。對於保安宮的修復工作，大會評委會給予高度評價：「保存興建於1805年的大龍峒保安宮，主要表現於其技術成就，及其為文化資產保存工作之雄心範圍之廣與用心之深。該專案採平衡方法，在使用現代科學保護方法和傳統建築及裝飾工藝之間取得了平衡，以盡其所能地恢復此一歷史建物之原貌。其對修復細節之審慎注意及對此專案之慷慨贊助，在保護此建物之宗教和建築上具有重大意義。」

== 建築特色 ==
大龍峒保安宮坐北朝南，因三殿間隔寬敞，第一進又為五開間的前殿加上左右各三開間的山門，因此從寬達十一開間的第一進正面看去，該廟宛若三座橫向並排的三殿宇。前殿又稱三川殿，牆面為石雕，為19世紀初作品。中門為1804年所建[[龍|蟠龍]]八角檐柱一對，是保安宮現存最早的石雕作品。

重檐歇山屋頂的正殿，四周環以方柱與八角柱相間的檐廊，正面安蟠龍柱兩對，單蟠龍外柱為1805年所建作品。該殿祭祀保生大帝的神龕兩頭，分別掛[[陳維英_(台灣)|陳維英]]所提[[楹聯|楹聯]]一對，殿外側背三面彩繪巨幅壁畫，是1960年代，[[臺南|臺南]]春源畫師[[潘麗水|潘麗水]]（男，1914年-1995年）的畫作。

後殿面寬十一開間，中奉[[神農大帝|神農大帝]]，左右兩側則附祀[[至聖先師|至聖先師]][[孔子|孔子]]與[[關聖帝君|關聖帝君]]。殿右為保恩堂，供奉歷代先賢神位。

在後殿之後，附設圖書館共設閱覽室、期刊閱報室以及二間館藏室，並設有[[凌霄寶殿|凌霄寶殿]]、[[大雄寶殿|大雄寶殿]]，供奉[[佛|佛]]家[[三世佛|三寶佛]]。

2011年4月18日保生大帝聖誕，也有把四[[釋奠佾舞|佾舞]]溶入祭典中，與祭孔的六[[釋奠佾舞|佾舞]]有些不同。

===剪黏===
保安宮的剪黏作品獨樹一幟不同他廟。剪黏材料依時間分為四個時期：陶瓷碗時期（清朝）、日本茶碗時期（日治至終戰初期）、彩色玻璃及壓克力時期（終戰初期）、淋湯時期（1970年代後）。自1996年保安宮以傳統的工法，用彩色的磁碗，用專門剪碗的剪刀來剪磁碗片，並且用糯米為黏著劑。傳統工法製作的剪黏，色澤較為沉穩，磁碗片較不易脫落。<ref>{{cite web|url=http://www.baoan.org.tw/HTML/Historic/20080813-3.htm |title=財團法人台北保安宮《大龍峒保安宮古蹟保存經驗》}}</ref>

==保生文化祭==
保生文化祭為保安宮每年舉行、為期一個多月的活動，提升民俗活動素質，將家姓戲、過火、放火獅、遶境踩街等藝術化及精緻化，同時加入廟埕文化新元素：如文物展、宗教導覽、中西醫義診、兒童寫生、攝影比賽等活動，保安宮逐漸蛻變為集宗教、教育、藝術、戲劇於一身的文化機構。

===傳統戲曲與陣頭表演===
「家姓戲」演出，早已轉型為現代社會中的民俗文化新風貌。其劇種比起過去更豐富，包括布袋戲、高甲戲、歌仔戲、北管戲及南北管演奏，演出團體涵蓋有職業劇團、保安宮社團、學校藝術科系等類。

=== 大道公遶境踩街放火獅 ===
在[[保生大帝|保生大帝]]聖誕([[農曆|農曆]]3月15日)當天與前一天所舉行的大型祭典，和大稻埕霞海城隍祭、艋舺青山王祭典合稱[[台北三大廟會|台北三大廟會]]，除了神明繞境，於15日當天下午一時舉行過火儀式，於莊嚴隆重的古禮祭祀儀式之後，依照神明所指示的起火龍方向及過火時間準備過火，此活動為祭典的重頭戲。

繞境大抵陣容為保安宮各個力士會(中壇元帥、五穀先帝、保安祖、平安祖、白礁祖、六祖、五祖、四祖、三祖、二祖、老祖)以及其所屬軒社德樂軒之外，另外每年會有不固定參與繞境的廟宇或是北管軒社或獅陣。

2013年12月13日凌晨零點左右，存放部分力士會神轎車社子的附近工場倉庫起火，保管庫受到波及導致神轎車毀損，因此保安宮決議停止繞境三年。

===其他文化及公益活動===
[[File:大龍峒保安宮9.jpg|右]]
[[File:大龍峒保安宮5.jpg|右]]
[[File:大龍峒保安宮8.jpg|缩略图]]
保生文化祭期間也會舉行文化與公益性質活動，如：
*古蹟導覽：聘請研究專家如歷史學家[[李乾朗|李乾朗]]教授、保安宮文史工作會林志峰會長介紹保安宮的細節。
*音樂會：2000年舉行過兩場祝聖音樂會，邀請台北弦樂四重奏和台北長號四重奏，演奏多首雅俗共享的中西樂曲。
*學術研討會：曾舉行海峽兩岸吳真人文化學術研討會、基督徒（教廷等代表）與道教對話國際學術研討會等。
*義診：因為保生大帝為醫神，因此也有中醫義診、中醫講座、中醫文物藥材展等活動。展出大量活體醫藥材料盆栽而非藥行乾燥商品。


== 大龍峒五穀先帝出巡 ==
保安宮內，除保生大帝外，亦奉祀另外一位[[醫神|醫神]][[五穀先帝|五穀先帝]]（即神農大帝），固定每年農曆四月廿六日出巡遶境，基本參與繞境陣容和路線與三月十四日保生大帝夜巡時相同，唯一差別在於，保生大帝夜巡由五穀先帝神轎排在尾陣。

== 重要護持人物 ==
;陳維菁
:陳維菁(1803-1876)：同治七年(1868)保安宮進行重修時，陳維菁擔任修復的領導人之一。
;陳培根
:陳培根，大龍峒地區著名富紳文士，熱心公益。大正六年(1917)保安宮重修時，擔任一堡管理人。
;林拱辰
:林拱辰1957年起管理期間，廟務逐漸步入軌道，加上宗教熱誠與犧牲奉獻、富於創意，使得保安宮建立宗教聲望，得以振衰起敝，是重要功臣。
;廖武治
:1943年生於臺北市，國立藝專（現臺藝大）美術科肄業，奠基培養藝術修為。拜在農禪寺聖嚴法師門下習佛，至今尚勤做早晚課，1979年起 擔任大龍峒保安宮董事。次年出版《大龍峒保安宮宗教建築藝術導覽》臺北保安宮。2002年出版《台北大龍峒保安宮古蹟保經驗》臺北保安宮。次年古蹟修復主持人副董事長廖武治在文化資產保存卓越成就，獲聯合國教科文組織頒發「2003年亞太文化資產保存獎」。 2004年當選董事長。2010年臺灣保生大帝信仰總會連任第二屆理事長。

2014年羅馬梵諦岡教廷駐華大使館副館長桑愛文神父、天主教會臺灣地區主教團宗教交談委員會執行秘書鮑霖神父、助理祕書吳聖年一行，於6月前來拜訪廖董事長，希望能有長久的交談合作，如合辦關懷社會的議題研討會等。2016年10月15-16日舉辦基督徒(教廷等代表)與道教對話國際學術研討會。同年出版《真人所居》臺北保安宮。

2018年3月14日上午8時45分，廖武治董事長，率領保安宮團員到教廷參與宗教交流對話，進入梵蒂岡晉見教宗包括保安宮常務董事謝趙欽桂、董事蘇勝中、教授胡其德等。雙方會面時間近半個小時，並致贈教宗他親繪的油畫，繪畫主題即為國定古蹟保安宮建築。<ref>{{cite news|url = https://www.taiwanembassy.org/va/post/3191.html|title = 跨信仰對話 台灣保安宮董事長面見教宗 | author = 黃雅詩 | date = 2018-03-14|work = 中央社}}</ref>
