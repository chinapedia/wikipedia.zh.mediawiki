'''大江乡'''，是{{PRC admin/fullname|43/11/21/207/000|3}}曾下辖的一个[[乡镇|乡镇]]级行政单位<ref>{{PRC admin/ref sgc|{{PRC admin/parent|43/11/21/207/000}}}}</ref>。

2013年3月，[[湖南省人民政府|湖南省人民政府]]批准以[[大江林场|大江林场]]辖区，设立大江乡，乡政府驻地在大江<ref name="a1">{{Cite web |url=http://www.yongzhou.gov.cn/2013/0329/106905.html |title=省政府批准永州新设三个乡 祁阳2个东安1个 |accessdate=2018-06-21 |author= 记者：欧春涛，通讯员：王治国|date=2013-03-29 |work= |publisher=永州新闻网，来源：永州日报|language=Zh-hans }}</ref>。2015年，祁阳县撤并小微乡镇，大江乡并入[[肖家村镇|肖家村镇]]。原因是大江乡人口过少，仅六千余人<ref>{{Cite web |url=http://www.qy.gov.cn/article-2-13867.aspx |title=宋振平就乡镇区划调整深入大江乡召开恳谈会 |accessdate=2018-06-21 |author=刘志军|date=2015-11-03 |work=责任编辑：刘敏 |publisher=祁阳县往人民政府网站，来源：祁阳新闻网|language=Zh-hans }}</ref>。获得[[湖南省人民政府|湖南省人民政府]]的批准后，11月24日，永州市人民政府下发永政函〔2015〕216号文件。将大江乡的十个[[建制村|建制村]]并入[[肖家村镇|肖家村镇]]，设立[[肖家镇_(祁阳县)|肖家镇]]。两个建制村并入[[大忠桥镇|大忠桥镇]]<ref>{{Cite web |url=http://www.yzcity.gov.cn/cnyz/szfwj/201705/401c15abb12346a09891fd95dcf3f13d.shtml |title=永州市人民政府关于祁阳县乡镇区划调整的通知 |accessdate=2018-06-21 |author=|date=2015-12-03 |work= |publisher=永州市人民政府网站，永政函〔2015〕216号文件|language=Zh-hans }}</ref>。

== 行政区划 ==

2013年时，大江乡下辖原[[大江林场|大江林场]]管辖的大江边、郑家冲、绘家冲、长冲、黄沙等5个国有工区及代管的12个[[建制村|建制村]]<ref name="a1"/>

2015年时，下辖12个[[行政村|行政村]]：九渡村、瓦屋村、大院村、塘岭村、彭家湾村、唐家冲村、赵家棚村、百岭村、黄沙村、白竹村、青石村、龙凼村<ref>{{Cite web |url=http://www.stats.gov.cn/tjsj/tjbz/tjyqhdmhcxhfdm/2015/43/11/21/431121209.html |title=2015年统计用区划代码和城乡划分代码(截止2015年09月30日)》......》大江乡 |accessdate=2018-06-21 |author=|date= |work= |publisher=国家统计局网站|language=Zh-hans }}</ref>。
