《'''大嘴鳥'''》是1999年7月1日到2001年3月29日在[[日本|日本]][[東京電視台|東京電視台]]系列（[[東京電視網|TXN]]）播放，以[[森永製菓|森永製菓]]的[[巧克力球|巧克力球]][[吉祥物|吉祥物]]「[[大嘴鳥|大嘴鳥]]」為主角的[[日本電視動畫|日本電視動畫]]。全91話。

== 概要 ==
以大嘴鳥作為[[主角|主角]]，不僅有搞笑的內容，也會在故事上加上某種寓意，是一部有著多元內容的作品。

大嘴鳥是森永巧克力球的吉祥物，當然，巧克力球也有在這部動畫中登場。

電視動畫方面，首次於1999年7月1日至2001年3月29日在[[東京電視台|東京電視台]]系列（[[東京電視網|TXN]]）播出。之後分別在2007年5月26日於[[衛星電視|CS放送]]所屬的[[Kids_Station|Kids Station]]和2009年7月於[[全國獨立放送協議會|獨立UHF放送局]]所屬的[[東京都會電視台|TOKYO MX]]重播。更早以前，由森永製菓提供在[[卡通頻道|卡通頻道]]重播。在[[台灣|台灣]]，曾由[[東森幼幼台|YOYOTV]]和[[MOMO親子台|MOMO親子台]]播出。

== 登場角色 ==
在大嘴鳥中所出現的角色，除大嘴鳥外，全都是原創角色。而本作世界觀為了能符合大嘴鳥此一主題，所以住在天使島的居民除了某些角色外，全都使用鳥的造型；另外，這些角色的名字都取自於牠們的外表或表現所得來（例如「大嘴鳥」是以牠的大嘴來命名，「大眼鳥」則是以牠的粗眉毛和大眼而得名）。

=== 主角 ===
;大嘴鳥（キョロちゃん　[[声優|声]]：[[伊東みやこ|伊東みやこ]] / [[傅其慧|傅其慧]](台灣)）
: 首次登場：第1集
: 設定年齡為八歳，有「咖啡色蛋蛋勇者」之稱。是在天使島上的神殿中誕生，從蛋裡出來的。小時候和萬事通博士一起到世界各處探險，在經過數次冒險後，和博士分開並回到天使島。
: 主要特徵是有著一副巨大的[[喙|喙]]，比鑽石還硬，能夠拿來砍斷樹且做成[[獨木舟|獨木舟]]，還能將巨大岩石一分為二，或變成強力鑽頭，並成為[[摩氏硬度|摩氏硬度]]第十一級的素材。
: 性格明亮且有顆率直的心，也有點[[天然呆|天然呆]]，有時一言一行會造成周圍的混亂（大部份自己都沒有惡意）。
: 有優異的戰鬥和運動能力。在困難時會試著使用頭腦，但連乘法都不懂，腦袋似乎不好，但當牠的左腦運行時，能夠變成超級天才大嘴鳥，既可解決關於智力犯的事件，還因為所寫的科學[[論文|論文]]還差點要被保送到大學去做研究。
: 和怪盜蒙奇有特殊關係，而且會模仿牠而變成「怪盗大嘴鳥」。平常是個好孩子，但也會因為腦中的「壞壞大嘴鳥」而變成到處作惡的不良少年。
:牠什麼食物都愛吃，只有[[青椒|青椒]]是牠最討厭的。

=== 和大嘴鳥有特別關係的角色===
;泡芙（パフー）
: 故事後半出現的角色，在天使島上並不存在這種生物，是大嘴鳥的寵物。生物名是彆扭毛蟲。而性格也是蟲如其名，有點畏畏縮縮。身體雖小，但食量卻很大。因為沒有受過訓練，所以老是會出錯，讓大嘴鳥感到相當困擾。對話會以「泡芙」來回答，大嘴鳥能夠聽懂牠在說什麼。另外，名字則是從牠的聲音得來的。

;萬事通博士（マツゲール博士　声：[[納谷六朗|納谷六朗]] / [[魏伯勤|魏伯勤]](台灣)）
: 首次登場：第1集
: 唯一出現的人類角色，也是養育大嘴鳥的人，角色原型是[[凡爾納|凡爾納]]的小說《[[環遊世界八十天|環遊世界八十天]]》中名為「萬事通」的助手。有著[[學者|學者]]和[[冒險家|冒險家]]身份，但在什麼地方做什麼研究則不明。
: 為了再度探尋天使島的秘密，和大嘴鳥一起到天使島上，然後又暫時分開而再會。之後，為了在世界各處冒險而又離開大嘴鳥。
: 每集的最後都有大嘴鳥寫給博士的信，內容則由博士朗讀。另外也會有大嘴鳥的朋友們在寫給博士的信上寫信的例子。
: 由於到了最後才又出場，所以其他角色和他見面的次數相當稀少。

=== 大嘴鳥的朋友 ===
;大眼鳥（パチクリ　声：[[本田貴子|本田貴子]] / [[雷碧文|雷碧文]](台灣)）
: 首次登場：第1集
: 男孩，身體是藍色。大嘴鳥最初的朋友，所以也是大嘴鳥第一好的朋友，設定年齡為8歲。
: 是名有點調皮好動的小孩，思考直來直往，而且行動力相當高。在家都不做家事，和自稱詩人的叔叔住在一起。喜歡武術和[[特攝片|特攝片]]，所以常看函授[[錄影帶|錄影帶]]，但老是被大嘴鳥擊敗。

;小紅莓（クリン　声：[[上村貴子|上村貴子]] / [[詹雅菁|詹雅菁]](台灣)）
: 首次登場：第2集
: 女孩，身體是粉紅色，家裡是麵包店。設定年齡為8歲，個性早熟且長相可愛，雖然有女孩子的一面，但也會顯現出其反面，就是有些任性和[[歇斯底里|歇斯底里]]。老是擔心自己的姐姐小蜜桃無法結婚。
:對大嘴鳥有如弟弟般的照顧。
: 另外也是個路痴，所以送貨常失敗。

;聰明鳥（ミッケン　声：[[間宮くるみ|間宮くるみ]] / [[林美秀_(配音員)|林美秀]](台灣)）
: 首次登場：第2集
: 經常拿著書（通常是[[百科全書|百科全書]]）並且穿著像是蛋殼的[[吊帶褲|吊帶褲]]的男孩，身體是黃色，興趣是讀書和拼[[拼圖|拼圖]]，比大嘴鳥牠們小一歲。
: 對人會使用[[敬語|敬語]]，態度端正和有著酷酷的一面。但在陷入危機時，會失去冷靜而哭出來。
: 擁有高等的知識和使用大人的語氣等博學的一面，卻完全比不上變成超天才狀態的大嘴鳥。
: 常被搞錯成嬰兒是牠最大的困擾，有時也不喜歡自己的長相。
: 自己或和大嘴鳥在一起都常會遇上一些很奇怪的情況，像是遇到或變成嘻巴嘻巴，還有進入大嘴鳥的腦裡。

=== 大嘴鳥朋友的家人 ===
;梅可樂（メグロ　声：[[水內清光|水內清光]] / [[林谷珍|林谷珍]](台灣)）
: 首次登場：第3集
: 大眼鳥的叔叔，[[自由業|自由業]]，喜歡創作[[川柳_(日本文學)|川柳]]。雖然和大眼鳥住一起，但家事全都託負給大眼鳥。雖然擅於煮菜，但也不常做。
: 由於一直過著自由奔放的生活，所以手頭經常很緊。在生活中會以[[哲學|哲學]]思考作為基礎，所以讓牠看起來像是掌握這個世界的意義，因此在牠的繪本中經常會有獨特的世界観。
: 曾引起模仿梅可樂的熱潮過。

;小蜜桃（マスカーラ　声：[[芳野日向子|金丸日向子]] / [[傅其慧|傅其慧]](台灣)）
: 島內第一好吃的麵包店之店長，也是小紅莓的姐姐。因為雙親不在，以[[監護人|監護人]]的身分和妹妹住在一起。
: 在天使島中被說是絕無僅有的美女，所以也被當成是像偶像般的存在。因此，她也演出介紹天使島的廣告。基本上個性相當溫柔，但會有美中不足的地方。有潔癖，只要在髒亂的房間五分鐘以上，就會變成掃除戰士。
: 是有[[雙重人格|雙重人格]]的人，在生氣到超過[[臨界點|臨界點]]時會變成另外一個人「超憤怒戰士小蜜桃」，但在回復後經常不知道自己變身過。
: 台灣藝人[[朱安禹|朱安禹]]（水蜜桃姐姐）的藝名即是以此命名。
;曼拉吉利（マナジリ　声：[[鹽屋浩三|鹽屋浩三]] / [[魏伯勤|魏伯勤]](台灣)）
: 聰明鳥的爸爸，蘋果奶奶是他的母親。在吉羅利家的玩具工場擔任新型玩具[[設計師|設計師]]。相當熱愛玩具和孩子，但因為自己的兒子過於優秀，所以也會有想要讓他覺得自己很棒的一面。
: 會依噴嚏大小和感到危機時發出的聲音而產生巨大的聲音。
: 由於公司收入減少，所以曾數次被放到裁員名單。但每當被放到名單上時，卻能多次意外地開發出熱門商品。
: 因為家裡老舊的關係，因此希望能夠加以重建。

;蘋果奶奶（リンクル　声：[[定岡小百合|定岡小百合]] / [[雷碧文|雷碧文]](台灣)）
: 首次登場：第2集
: 聰明鳥的奶奶，曼拉吉利的媽媽。似乎有點老人痴呆症，卻擁有在奇怪的事情發生時，能很快明白原因等超越一般人的能力。是個平常時表現平凡，但在必要時會有驚人表現的老婆婆。
: 總是說著關於食物的發言，另外對於芋頭爺爺的問題常答非所問，並經常有耍呆的發言。
: 年輕時，是位相當美麗的女性。另外還有以獎學金進入過都市的料理学校的經歷。那時也與聰明鳥的爺爺結婚。在糖果公司的[[廣告曲|廣告曲]]中，自己還擔任演唱和演出的角色。

=== 天使之島的居民 ===
;團團轉刑警（グリグリ警部　声：[[森田順平|森田順平]] / [[魏伯勤|魏伯勤]](台灣)）
: 首次登場：第3集
: 警察中的刑警，有相當強的能力，只要有問題出現，就會立刻拜託牠解決、不過也有常失敗的情形。有次也是穿著制服出場，另外把警車改成[[越野車|越野車]]，除了私用和公用外，也有拿它來參加賽車過。
: 因為窮又沒志氣，常常無法下定決心向喜歡的小蜜桃告白，也總是會失去主導事情的主導權。
: 另一個夢想是成為怪盜，所以也有著怪盜蒙奇的身分。

;芋頭爺爺（マクモー　声：[[大竹宏|大竹宏]] / [[林谷珍|林谷珍]](台灣)）
: 首次登場：第2集
: 一個人住，總是和蘋果奶奶一起散步和曬太陽。個性和善且知識淵博，有汽車和機車駕照。
: 在小時候就已喜歡上蘋果奶奶，但由於牠之後要去都市的料理學校上課，結果無疾而終。
: 很敬愛母親，在牠生日時一定會用母親喜歡的花來裝飾牠的遺照。

;吉羅利（ドン．ジロリ　声：[[梅津秀行|梅津秀行]] / [[林谷珍|林谷珍]](台灣)）
: 首次登場：第3集
: 天使島上的大富豪，是足以影響島上經濟的人物，個性也相當暴躁，並很愛錢。祖先是海盜，過去也曾是豪門。擁有能夠舉辦越野賽的巨大土地，不僅有豪宅，而且也有寶物會藏在屋內。
: 是家玩具公司的老闆，但實際上除了玩具，也會做各式各樣的產品，但在發展這些商務時，因為總使用冷酷邪惡的手段，而遭到大家的非難。
: 由於常被妻子指責和爬到牠頭上，為反抗，所以也會上高級俱樂部。
: 學生時代參加相撲社，擁有學生橫綱的輝煌經歷。
: 開會時，如果手下的主意太差，會把牠們送到訓練室加以強化訓練。
: 有一度變成像梅可樂那麼悠閒，讓大家因此相當困擾。但後來看到大嘴鳥引起吉羅利風潮後，而恢復。

;金母雞（メメリッチョ　声：[[真山亞子|水原琳]] / [[傅其慧|傅其慧]](台灣)）
: 首次登場：第3集
: 吉羅利的妻子。身材相當豐滿，所以常努力減肥，年輕時的身材纖瘦許多。喜歡奢侈。
: 深愛著吉羅利，經常支持吉羅利的話，也很溺愛飛機鳥。
: 在老公變成像梅可樂一樣悠哉時，還因為無法接受，而要求大嘴鳥讓他恢復原貌。

;飛機鳥（チロリ　声：[[志賀克也|志賀克也]] / [[詹雅菁|詹雅菁]](台灣)）
: 首次登場：第3集
: 吉羅利的兒子，常以吉羅利兒子之名到處欺負別人，時常會帶著手下怪頭和歪頭。有著有錢人少爺的個性和缺點。
: 相當喜歡小蜜桃，但在她的面前都會變得很害羞。

;怪頭（コント　声：[[喜田亞由美|喜田亞由美]] / [[雷碧文|雷碧文]](台灣)）
: 首次登場：第3集
: 飛機鳥的手下，身體是綠色。
: 和小紅梅曾是筆友，不過兩人到最後還是沒以筆友身份見到面。

;歪頭（タクト　声：[[大平泉|大平泉]] / [[林美秀_(配音員)|林美秀]](台灣)）
: 首次登場：第3集
: 飛機鳥的手下。
: 曾因為想過當老大的癮，而收大嘴鳥當手下，但最後還是失敗。

;八卦嬸（メトリ　声：[[橘U子|橘U子]] / [[詹雅菁|詹雅菁]](台灣)）
: 首次登場：第3集
: 是名擅於收集情報、喜歡流言＆流行八卦的阿姨。有著原來在宇宙中以正義一方的角色活躍著的異色經歷。興趣是[[Cosplay|Cosplay]]，像是變身成政治人物和賽車皇后等。
: 基本上在她洩露情報的時候，就能立刻傳遍整個天使島，不過情報常被嚴重誇張化、也近乎於[[捏造|捏造]]。
: 最喜歡醜聞，時常都在找尋有趣的消息。座右銘是「沒火時就要讓它起煙、起火的時候就要讓它火上加油」。作為大嘴鳥報的傑出[[總編輯|總編輯]]，還因此挖出政治人物[[收賄|收賄]]的爆炸性新聞。
: 經常找便宜貨和四處殺價，所以在特賣會中也是毫無敵手。也曾在超市中負責煮[[香腸|香腸]]來給客人試吃。

;迪曼頓（デメントン　声：[[大川透|大川透]] / [[魏伯勤|魏伯勤]](台灣)）
: 首次登場：第2集
: 天使島上現任的總統。雖然常有醜聞，但支持率很高。喜歡金錢和權力，數次因為收賄而被逮捕。和前任的羅肯一直在互相較勁，有著不服輸的性格。興趣是吹奏尺八。
: 在小時候似乎個性就很不好。

;羅肯（ローガン　声：[[鹽屋浩三|鹽屋浩三]] / [[林谷珍|林谷珍]](台灣)）
: 首次登場：第2集
: 天使島上前任總統，現今目標放在下任總統的位置。外表衰老且看起來相當溫和，但實際上是個老奸巨滑的人。和迪曼頓一樣喜歡金錢和權力。有個[[遊樂園|遊樂園]]，所以也想當天使島的經營者。

== 天使島資訊 ==

; 地理
* 大嘴鳥所住，一個座落於地球上的杜撰的島，特徵是島中央有一座像是[[扁桃|扁桃]]狀的火山，被叫作天使山，[[標高|標高]]3333公尺，也是島上的觀光景點，被認為以[[富士山|富士山]]原型。
* 故事中心背景的天使島市街地，是沿著街道而形成村落。
* 另外這個島上因為過去火山活動活躍，所以有大規模的地殻變動、因此常有地震。
* 有個芙拉米康市的島、和天使島有來往，到那邊的船要34日加6小時45分。

; 政治・司法制度
* 在天使島上，孩子也有[[選舉權|選舉權]]，但實際上除了聰明鳥外，其他人似乎都不知道[[選舉|選舉]]的真正意義。
* 是總統制，能夠加以直選選出來。
* 最近由迪曼頓和羅肯互相當總統和副總統，導致政治停滯，而不能實現[[民主主義|民主主義]]的原來目的。
* 另外天使島上總統也擔任[[法官|法官]]，似乎沒有規定[[三權分立|三權分立]]，也沒讓[[司法權|司法權]]獨立出來。然後[[檢察官|檢察官]]是副總統，[[律師|律師]]則是由[[警官|警官]]（團團轉刑警）兼任，而判決等採用居民組成[[陪審團|陪審團]]。
* 法院體制採用[[陪審制|陪審制]]。

; 經濟
* 天使島的經濟是來自於吉羅利家的資本而成立。
* 有自己的貨幣單位叫做「達(D)」，與現實世界中的日圓匯率是1:1。
* 有披薩店且能外送。

; 其他
* 天使島上沒有考試戰爭，但有對自己考試的制度存在。
* 動畫中，除了船和公車外，無其他可離開島的交通工具，但根據蘋果奶奶所說「想搭噴射機」來看，天使島似乎有飛機。
* 由於天使島沒有機場，郵寄物都要用船來運送到附近的島上，在那裡可能有機場能運送到全世界。
* 很流行吹直笛，一年有一次「直笛會」。
* 天使島上有高級俱樂部。
* 有一間叫米卡耶魯艾島的大學，但不能確認天使島上有大學存在。
