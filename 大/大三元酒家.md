'''大三元酒家'''（{{lang-en|Tai Sam Yuen Restaurant}}）是[[廣州|廣州]]的一間原老牌酒樓，1920年代中與西園、南園、文園合稱'''[[廣州四大酒家|廣州四大酒家]]'''，在港澳及東南亞一帶享有盛名。又為'''长堤三大'''(另二為[[大同酒家|大同酒家]]、[[大公餐厅|大公餐厅]])之一。名稱源於舊日[[科舉制|科舉制]]，以連中鄉試、會試、殿試榜首的解元、會元、狀元為三元及第，取名寓意'''酒家榜首，食肆班頭'''<ref name=":0">廣州觀光大全·酒家饭店 程慧</ref>。

== 歷史 ==
酒家於國民8年（1919）由溫心田創辦。首家設置電梯載客上落。當時的大三元酒家以筵席宴會為主，菜肴高檔，擅製鮑參翅肚等，尤以[[紅燒大裙翅|紅燒大裙翅]]著稱，時每款售價合白銀60元，時達官顯貴均以能嘗此菜為榮。海外華僑歸來也喜到此宴客。因而開業不久，大三元便聲名鵲起，成為廣州飲食業的“最高食府”之一。

1953年由於資金缺乏歇業。1959年由政府投資裝修復業，成為國營企業。[[文革|文革]]期間，大三元易名'''今勝昔飯店'''，難以被歸僑認可，生意停滯。

1982年10月，廣州市飲食服務公司（把持酒家單位）與香港龍星投資有限公司合作經營大三元，吸收回香港發展的先進烹飪技藝，推出一批創新菜點。此外還派出廚點（心）師到南京和北京的大三元酒家主理廚政。

1999年，大三元改營速食，2000年4月由於鋪面欠租被迫停業。

2005年，大三元酒家被清拆，後曾用作停車場。

2011年，越秀區政府部門公開徵集大三元酒家重建設計機構，擬建地下一層、地上五層。

2012年，原址變為廣州民間金融街工程部，傳言在地皮建設金融街綜合大樓<ref name=":1">[http://news.gd.sina.com.cn/news/20120412/1274758.html 大三元酒家原址重建？难！]  信息时报 2012-04-12 </ref>。

2013年2月，越秀區政府部門放風，向社會各界徵集大三元酒家1930年代外立面圖文資料<ref>[http://news.163.com/16/1203/09/C7BNJBOH00014AEE.html 大同蛋挞、大公西饼、大三元“红烧大裙翅”成绝唱？] 金羊网(广州) 2016-12-03 </ref>。

2014年，恒大酒店集團將北京大三元品牌納入旗下<ref>[http://www.prnasia.com/story/189549-1.shtml 恒大酒店集团携手“大三元”引领金秋月饼风尚] 美通社 2017-09-27</ref>。

== 出品 ==
大三元酒家素以酒席聞名，而酒席又以[[紅燒大裙翅|紅燒大裙翅]]馳名，1930年代聘得有'''翅王'''之稱的名廚吳鑾主理廚政後，紅燒大裙翅更是蜚聲中外<ref>廣州市志·商業·飲食志</ref>。該菜選用老虎鯊、鱘頭鯊等上等裙翅作原料，經浸發、滾煨翅、烤翅、上翅幾道工序，在火候和時間上都極為講究，烹製出來的翅，拿筷子夾任何一根，兩端都會自然下垂並合攏成橢圓形。菜式特點為韌中帶脆，湯清味鮮，濃郁不膩，為粵菜中之上品。

其他名菜有<ref name=":0" />：鳳爪燉海狗、茶香雞、片皮乳豬、生炒生魚絲、魚燴三蛇、蟹肉瓊山豆腐、香麻涼拌雞絲、雀巢鴛珠圓、富貴石榴蝦、汁金錢鱔等。

點心有：蜂巢荔芋角、奶皮豬油飽、冰肉焗蛋撻、海南椰絲撻、 OK布冧雞蛋捲、軟皮雙色塊等。

== 粵語俗語 ==
在传统廣州人心目中，大三元酒家的名聲如雷貫耳，亦就流傳些俗語印證記憶。口語中街坊常會講笑“聽晚我請你大三元'''一席酒'''（粵語諧音‘'''一直走'''’）”，對方亦會笑答“今晚我請你大三元'''四圍酒'''（粵語諧音‘'''四圍走'''’)”<ref name=":1" />。
