《'''大亨小傳'''》（{{lang-en|'''''The Great Gatsby'''''}}）是一部1926年美國[[無聲電影|無聲電影]]，由{{link-en|赫伯·佈雷農|Herbert Brenon}}執導。<ref>{{Cite web |url = http://memory.loc.gov/diglib/ihas/loc.mbrs.sfdb.5791/default.html |title = The Great Gatsby |publisher = The Library of Congress |language = en}}</ref>這是第一部改編自[[法蘭西斯·史考特·費茲傑羅|法蘭西斯·史考特·費茲傑羅]]的1925年[[大亨小傳|同名小說]]的電影。[[沃納·巴克斯特|沃納·巴克斯特]]飾演{{link-en|傑·蓋茲比|Jay Gatsby}}，{{link-en|洛伊絲·威森|Lois Wilson (actress)}}飾演Daisy Buchanan。<ref>{{cite book|editor=Tredell, Nicolas|title=''Fitzgerald's The Great Gatsby: A Reader's Guide''|year=2007|publisher=A & C Black|isbn=0-826-49011-5|page=96}}</ref>

本片由{{link-en|Famous Players-Lasky|Famous Players-Lasky}}公司製作，[[派拉蒙電影|派拉蒙電影]]發行。《大亨小傳》現在被認為是[[散失電影|散失電影]]了。<ref>{{Cite web |url = http://www.silentera.com/PSFL/data/G/GreatGatsby1926.html |title = The Great Gatsby |publisher = silentera.com |language = en}}</ref>
