'''大非川之战'''是唐朝为了占据[[青海|青海]]西部并帮助[[吐谷浑|吐谷浑]]复国而与吐蕃军队进行的战役，但是最终唐军遭到了失败，损失数万军队。并因此丧失了青海西部。

== 战争起因 ==
战前，在[[吐谷浑|吐谷浑]]叛臣[[素和贵|素和贵]]的帮助下，[[吐蕃|吐蕃]]军入侵并占领了吐谷浑，吐谷浑王带妻子[[弘化公主|弘化公主]]迁到[[浩门河|浩门河]]流域，并向唐朝请求援助。

[[总章|总章]]二年（公元669年），吐蕃以[[贊悉若|贊悉若]]为大相，由其弟[[論欽陵|論欽陵]]带领军队驻扎在被其强行吞并的吐谷浑。总章三年(公元670年)，論欽陵率领40万吐蕃军队入侵并攻占了[[安西四镇|安西四镇]]。四月初九，唐朝以[[薛仁贵|薛仁贵]]为逻娑道行军大总管，[[阿史那道真|阿史那道真]]、[[郭待封|郭待封]]为副，领军5万人反击吐蕃军，以占据青海西部，并帮助“护吐谷浑还国”(《新唐书·吐蕃传》)。

== 战役进程 ==

薛仁贵率军经鄯州（今青海省乐都县）至[[青海湖|青海湖]]南之大非川。薛仁贵留郭待封率2万人守护辎重于大非川（今青海省共和县西南的切吉旷原）。随后，薛仁贵率军轻装奔袭。在河口（今青海省玛多县）大败吐蕃军队。薛仁贵乘胜进占乌海城。

但郭待封不服军令，擅率后队继进。吐蕃军以20余万军队发起攻击，击败郭部，唐军辎重尽丧。薛仁贵被迫退保大非川。

吐蕃军与唐军对峙到八月，主力齐集后，钦陵指挥40余万大军进攻唐军。薛仁贵在无险无粮的情况下，顽强抵抗，最终唐軍全軍潰敗，薛仁贵被迫“与钦陵约和而还”<ref>{{wikisource-inline |資治通鑑/卷201|《資治通鑑》卷201}}</ref>

670年，吐蕃军占据了青海西部、安西四镇。唐朝被迫将安西都护府迁至西州（今新疆吐鲁番），吐谷浑亦被吐蕃彻底吞并。但是675年，唐朝再次控制安西四镇。687年唐朝废安西四镇，692年唐军大破吐蕃军，唐朝再次控制安西四镇，重置安西都护府于龟兹。此后唐朝长期控制安西。

== 唐军失败原因 ==
唐军失败的主要原因是地理条件和抗命事件。

大非川海拔3250米，乌海海拔是4000米，河口海拔是4240米，地理条件惡劣，吐蕃軍隊適應當地環境，而薛仁贵率领的唐朝军队不适应高原的地理条件，使得行军和作战都受到了影响。<ref name="Graff2002">{{cite book|author=David Andrew Graff|title=Medieval Chinese Warfare, 300-900|url=https://books.google.com/books?id=QDsfWBimDEkC&pg=PA206| year=2002|publisher=Psychology Press| isbn=978-0-415-23955-4|page=206}}</ref>

同时郭侍封的抗命造成了唐军辎重的丧失，成为失败的最主要原因。
