'''-{zh-cn:戴维·克罗克特;zh-hans:大卫·克洛科特; zh-hant:大衛·克拉克;}-'''（'''Davy Crockett'''，本名'''David de Crocketagne'''，{{bd|1786年|3月17日|1836年|3月6日|catIdx=C}}），[[美国|美国]][[政治家|政治家]]和战斗[[英雄|英雄]]。他曾当选代表[[田纳西州|田纳西州]]西部的[[众议员|众议员]]，因参与[[得克薩斯革命|德克萨斯独立运动]]中的[[阿拉莫戰役|阿拉莫戰役]]而战死。

== 早年生涯 ==
克洛科特生于[[田纳西州|田纳西州]][[格林维尔_(田纳西州)|格林维尔]]。他的祖先是来自[[乌尔斯特|乌尔斯特]]（北爱尔兰）的苏格兰裔和法国裔[[归正宗|归正宗]]教徒。他是九个孩子中的第五个，几乎没有受过什么教育。以19世纪早期的标准来说，克洛科特称得上是一个身材魁梧的大汉：他身高1.78米左右，体重86公斤。
1813年9月24日他加入了田纳西骑兵志愿军第二团，然而他只在那里服役了90天。他的长官是很多年后当上美国[[总统|总统]]的[[安德鲁·杰克逊|安德鲁·杰克逊]]。在服役期间，他参加了对[[克里克印第安人|克里克印第安人]]的战争，并一举成名。

== 政治生涯 ==
退役后，克洛科特在他的故乡开始他的政治生涯。最先他在市政府内任职，消灭了当地的贪污腐败现象。1826年和1828年他被选为代表田纳西州的众议员而进入了[[美国国会|美国国会]]。在华盛顿，克洛科特又遇到了他当年的老上司，时为总统的[[安德鲁·杰克逊|安德鲁·杰克逊]]。但他们却因为一项旨在驱逐印第安人的法案而产生了激烈的争吵。由于杰克逊素来厌恶印第安人，所以极力推动法案实行，而亲身经历过印第安人战争的克洛科特则倾向和印第安人和平共处。由于新法案受到大部分白人移民的拥护，克洛科特甚至因此而在1830年输掉了他的众议员席位。不过1832年他又再一次夺回。
 
1834年大卫·克洛科特出版了他的回忆录。由于他为了此书投入了过多的时间，他再一次输掉了选举。在此之后，他决心去德克萨斯支持并参与当地反抗墨西哥统治的革命。1836年1月14日，克洛科特和其他65人一同投奔德克萨斯独立政府，每个人都被许诺将在战争胜利后得到19平方公里的土地。

== 德克萨斯独立战争 ==
1836年2月6日，克洛科特领着12人来到了[[圣安东尼奥_(得克萨斯州)|圣安东尼奥]]的阿拉莫以增援守军。阿拉莫之战最终在2月23日爆发。克洛科特的任务是负责在教堂前栅栏的防守。阿拉莫守军共有约187人。对于他们来说，几乎没有任何获胜希望：援军始终未来，而包围他们的墨西哥军队则有5000人。13天后，德克萨斯军被击败，尽管他们最终赢得了战争的胜利。传说中克洛科特死于战斗中，不过依据最新的证据显示，克洛科特和其他起义领导人在战斗结束后被逮捕，并在第二天在[[安东尼奥·洛佩斯·德·桑塔·安纳|桑塔·安那]]的命令下被当众枪决。
