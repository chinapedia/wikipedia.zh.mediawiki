'''大湳'''是[[台灣|台灣]][[桃園市|桃園市]][[八德區|八德區]]的一個地名。

== 位置 ==
大湳位在八德區中心偏北之地區，於[[茄苳溪|茄苳溪]]以東。大湳的地理位置較接近於[[桃園區|桃園區]]商區，加上生活機能密切與[[台鐵桃園車站|台鐵桃園車站]]周邊地區結合，因此現今的大湳已是八德最繁華的市區，但大湳範圍界定並不明確。

廣義來說可指[[八德區|八德區]]建國路、和平路以北，[[國道二號|國道二號]]以南，福國街、桃德路以東，東勇街以西，及[[桃園區|桃園區]]國際路桃園大圳以南，這些範圍都可稱大湳。狹義來說可指建國路以北、大智路以南、福國街以東、忠勇街以西。

== 由來 ==
其名因土地多湳田，地勢較低加上該區較多雨，使地表易積水，地面多水堀和爛泥 ；「湳」為[[閩南語|閩南語]]發音，指易積水，地為鬆軟或泥濘，行人或牛隻會陷入泥濘的難走地區，便是此意得名。

== 水文 ==
