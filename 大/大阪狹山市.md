'''大阪狹山市'''（{{jpn|j=大阪狭山市|hg=おおさかさやまし|rm=Ōsakasayama shi|f=y}}）是位於日本[[大阪府|大阪府]]東南部的城市，為[[大阪市|大阪市]]和[[堺市|堺市]]的[[臥城|臥城]]，在轄內南部為自1960年代開始開發的{{tsl|ja|狭山ニュータウン|狹山新市鎮}}。

== 歷史 ==

本地在令制國時代屬於[[河內國|河內國]][[丹比郡|丹比郡]]，過去被稱為狹山鄉，「狹山」之名源於本地位於羽曳野丘陵和河泉丘陵之間狹窄的地形。

境內的{{tsl|la|狭山池 (大阪府)|狹山池 (大阪府)|狹山池}}在8世紀完成的史書[[古事記|古事記]]及[[日本書紀|日本書紀]]皆有記載，據考證約完成於[[飛鳥時代|飛鳥時代]]的616年，為日本最古老的水庫形式的人工蓄水池。<ref>{{Cite web| url=http://www.city.osakasayama.osaka.jp/gyosei/shinogaiyo/sayamaike/1442542558499.html| title=狹山池的歷史| publisher=狹山市政府| accessdate=2018-04-24| language=ja}}</ref><ref>{{Cite web| url=http://www.sayamaikehaku.osakasayama.osaka.jp/_opsm/p-exhibition/2.html| title=第2ゾーン 狭山池の誕生| publisher=大阪府立狹山池博物館| language=ja| accessdate=2018-04-24}}</ref><ref name="History">{{Cite web|url=http://www.city.osakasayama.osaka.jp/gyosei/shinogaiyo/purofuiru/1410315788622.html| title=大阪狭山の歴史| publisher=大阪狹山市政府| accessdate=2018-04-24| language=ja}}</ref>

1591年，本地被[[豐臣秀吉|豐臣秀吉]]封給[[後北條氏|後北條氏]]的[[北條氏規|北條氏規]]，其子[[北條氏盛|北條氏盛]]此後先後繼承了此地及其伯父[[北條氏直|北條氏直]]的領地，成為領有1萬1千石的[[大名_(稱謂)|大名]]，由於藩廳設於本地，因此稱為[[狹山藩|狹山藩]]。但現在的大阪狹山市轄區除了屬於狹山藩外，也有部分區域屬於[[小田原藩|小田原藩]]、[[館林藩|館林藩]]、{{tsl|ja|丹南藩}}、[[旗本|旗本]]領，在江戶時代結束後，這些區域先後被併入[[堺縣|堺縣]]，並在1889年實施[[町村制|町村制]]後分屬'''狹山村'''和{{tsl|ja|三都村 (大阪府)|三都村 (大阪府)|三都村}}兩個行政區劃，這兩個村先在1931年合併，1951年升格為'''狹山町'''，1987年再升格為市，但由於當時在[[埼玉縣|埼玉縣]]已有[[狹山市|狹山市]]，因此本市在升格為市的同時更名為'''大阪狹山市'''。<ref name="History"/>

== 觀光資源 ==

[[File:Sayamaike-Museum01.JPG|thumb]]

轄內的{{Tsl|ja|狭山池 (大阪府)|狹山池 (大阪府)|狹山池}}由於是日本最古老的水庫形式的人工蓄水池，現已被列為國家史跡，在池的北側建有大阪府立狹山池博物館，博物館建築由[[安藤忠雄|安藤忠雄]]所設計，館內除介紹狹山池的歷史，也介紹狹山本地的歷史文物資料。

過去轄內在{{tsl|ja|狭山池_(大阪府)|狹山池_(大阪府)|狹山池}}東側曾有成立於1938年的{{tsl|ja|さやま遊園|狹山遊樂園}}，此地過去原為[[狹山藩|狹山藩]]政治中心{{tsl|ja|狭山陣屋|狹山陣屋}}所在地，狹山藩藩主北條氏後代為活化地區，將土地贈出，由[[南海電氣鐵道|南海電氣鐵道]]經營遊樂園，過去極盛時期一年入園旅客數量曾達到54萬人次，也因此相鄰此地的鐵路車站[[大阪狹山市車站|大阪狹山市車站]]在當時也曾一度被命名為「狹山遊園前車站」；但之後由於旅遊的多樣化，造成來園人數逐漸降低而虧損，最終在2000年停止營業，此地現留有部分地區做為「狹山公園」對外開放。

== 教育 ==
轄內的高等教育學校都是配合{{tsl|ja|狭山ニュータウン|狹山新市鎮}}開發所設立的大學，包括由{{tsl|ja|学校法人帝塚山学院|學校法人帝塚山學院|帝塚山學院}}成立的{{tsl|ja|帝塚山学院大学|帝塚山學院大學}}，和[[近畿大學|近畿大學]]醫學部所在的大阪狹山校區。
