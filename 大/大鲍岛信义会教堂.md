'''大鲍岛信义会教堂'''是[[胶州湾租借地|德属胶澳保护领]]的一座教堂，位于德租时期的[[大鲍岛|大鲍岛]]III号街区东南角，即今天的[[胶州路_(青岛)|胶州路]][[易州路|易州路]]路口西北角处。教堂由柏林信义会建立，传教士昆祚（Adolf Kunze）设计，于1899年9月2日开工，1901年1月2日投入使用并举行柏林信义会在青岛的第一次洗礼。这座教堂是柏林信义会在青岛建造的第一座教堂，设有500个座位，外形极其简单，仅由矩形的礼拜堂和正立面上方的一个带有十字架的小塔楼组成，具有明显的应急特点。1925年，该教堂由{{le|美国信义会|United Lutheran Church in America}}接手。1931年，教会为教堂增建了一个哥特复兴风格的塔楼。1936年，教堂所在的土地被一华人买下，该教堂遂被拆除，以建设一座新式里院<ref>{{cite book |author=青岛城市建设档案馆编 |title=《大鲍岛: 一个青岛本土社区的成长记录》|year=2013 |publisher=山东画报出版社 |location=济南 |isbn=9787547409138}}</ref><ref>{{cite web |url=http://www.tsingtau.org/kunze-adolf-1862-1922/|title=Kunze, Adolf (1862 – 1922), Missionar}}</ref><ref>{{cite web |url=http://lib.sdsqw.cn/bin/mse.exe?seachword=&K=b2&A=62&rec=84&run=13|title=青岛市志·民族宗教志·信义会差会(德国柏林教会、美国鲁东信义会)}}</ref><ref>{{cite web |url=http://blog.sina.com.cn/s/blog_48883220010002eq.html|title=昆泽和柏林传教会住宅楼}}</ref>。
