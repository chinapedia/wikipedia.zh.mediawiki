在[[计算机|计算机]][[编程语言|编程语言]]里，'''大小写敏感性'''（{{lang-en|'''case sensitivity'''}}）是指使用[[大写字母|大写字母]]、[[小写字母|小写字母]]造成不同效果的情况。可能受到大小写影响的情况有用户名称、[[档案名称|档案名称]]、[[密码|密码]]、[[标签_(元数据)|标签]]、[[变量_(程序设计)|变量]]名称以及在指定的文本中搜索关键词等。

一些[[编程语言|编程语言]]是大小写敏感的，例如[[Java|Java]]、[[C++|C++]]、[[C♯|C♯]], [[C语言|C语言]],<ref name="k&r1e">{{cite book | last = Kernighan | first = Brian W. | authorlink = Brian Kernighan | coauthors = [[Dennis_M._Ritchie|Dennis M. Ritchie]] | chapter=Chapter 2: Types, Operators and Expressions |pages=33 | title = [[The_C_Programming_Language_(book)|The C Programming Language]] | edition = 1st | publisher = [[Prentice_Hall|Prentice Hall]] | location = [[Englewood_Cliffs,_NJ|Englewood Cliffs, NJ]] | isbn = 0-13-110163-3 |date=February 1978}}</ref> [[Ruby|Ruby]]<ref>{{cite book | last = Matsumoto| first = Yukihiro| authorlink = Yukihiro Matsumoto | chapter=Chapter 2: Language Basics |pages=9 | title = Ruby in a nutshell | edition = 1st | publisher = [[O'Reilly_Media|O'Reilly Media]] | isbn = 0-596-00214-9 |date=January 2002}}</ref>以及[[XML|XML]])。其他的编程语言则并不是大小写敏感的，例如大多数[[BASIC|BASIC]]（除了[[BBC_BASIC|BBC BASIC]]）、[[Fortran|Fortran]]、[[SQL|SQL]]<ref>Although one can explicitly set a single database or column [[collation|collation]] to be case-sensitive</ref>和[[Pascal_(程式語言)|Pascal]]。同时，在[[Haskell|Haskell]]、[[Prolog|Prolog]]和[[Go|Go]]这些语言，大写字母的标识符可以进行[[形式语义学|形式语义学]]的编码。
