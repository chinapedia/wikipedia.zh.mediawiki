'''大乘本生心地觀經'''，收於[[大正藏|大正藏]]本緣部<ref>印順法師認為本經編在「本緣部」不大適合，由於本經說發菩提心，依月輪觀修三種秘密法門，速證菩提，應編入「密教部」</ref>。該經中敘述佛陀在王舍城[[耆闍崛山|耆闍崛山]]中，為文殊、彌勒等諸大菩薩開示出家住[[阿蘭若|阿蘭若]]者，如何觀心地、滅妄想，而成佛道。自古以來，以經中關於報四恩之思想而馳名<ref name=ma2/><ref name=ma>{{cite web|title=大乘本生心地觀經|url=http://cidian.foyuan.net/word/319194/}}</ref>。

因經名帶有「本生」二字，而被歸入本緣部，但僅在序品略為提及，非主談本生故事，其思想主要是以心為主導。主張一切諸法皆從心地而生，皆從心地而起<ref name=ma3>{{cite web|author1=佛書解題（香光版）|title=大乘本生心地觀經|url=http://www.gaya.org.tw/library/readers/guide-xdgj.htm}}</ref>。

== 題解 ==
[[本生_(佛教)|本生]]，記錄佛陀還未成佛的前生故事。

心地，心為萬法之本，能生一切諸法，故曰「心地」。又，修行者依心而修行，長養諸善根功德，故曰「心地」<ref>{{cite web|title=心地|url=http://www.budaedu.org/doctrin/d42.php3}}</ref><ref>{{cite web|title=心地|url=https://www.buddhistdoor.org/tc/dictionary/details/%E5%BF%83%E5%9C%B0}}</ref>。一切眾生之本覺真性，即為心地、[[佛性|佛性]]<ref>[[禪源諸詮集都序|禪源諸詮集都序]]：-{「禪是天竺之語，具云禪那。中華翻為思惟修，亦名靜慮，皆定慧之通稱也。源者是一切眾生本覺真性，亦名佛性，亦名心地，悟之名慧，修之名定。定慧通稱為禪那。此性是禪之本源，故云禪源」}-</ref>。

== 傳譯 ==
據經序所說，本經梵本為[[唐高宗|唐高宗]]時[[師子國|師子國]]所進獻，由[[迦濕彌羅國|罽賓]]三藏般若等八人，於唐朝[[貞元_(唐朝)|貞元]]六年譯出<ref name=ma/>。

== 內容 ==
《大乘本生心地觀經》共計有十三品：序品、報恩品、厭捨品、無垢性品、阿蘭若品、離世間品、厭身品、波羅密多品、功德莊嚴品、觀心品、發菩提心品、成佛品、囑累品。

本經的成立係以《般若》、《維摩》、《法華》、《華嚴》、《涅槃》等大乘佛教思想為基礎，再加上[[攝論宗|攝論宗]]的唯心說、[[唯識宗|唯識宗]]的唯識說而建立三界唯心唯識說。在實踐方面，則主張依從[[彌勒信仰|彌勒信仰]]往生[[兜率天|兜率天]]，教人持守[[菩薩戒|菩薩戒]]，行十[[波羅密|波羅蜜]]及[[禪定|禪定]]觀心，並勸修《諸佛境界攝真實經》等所說的三密修行；對在家眾則說報恩行<ref name=ma2/>。

== 考證 ==
本經的成立年代，從經中三界唯心[[唯識|唯識]]說來看，應當是在[[世親|世親]]之後，當在西元五、六百年左右<ref name=ma2>{{cite web|title=中華佛教百科全書——【大乘本生心地觀經】|url=http://fofa.foxue.org/2006/429_0321/73257.html}}</ref><ref name=ma3/>。

本經含有《般若》、《維摩》、《法華》、《華嚴》、《[[大般涅槃經_(大乘)|涅槃]]》、《賢劫》、《[[觀普賢經|觀普賢經]]》、《諸佛境界攝真實》、《[[梵網經_(大乘佛教)|梵網]]》、《[[首楞嚴三昧經|首楞嚴三昧經]]》、《[[彌勒上生經|彌勒上生經]]》等諸經的思想觀點，並以《瑜伽》、《攝大乘》、《唯識》諸論為其基礎<ref name=ma2/>。

此外，本生譚中也含有不少的經典。例如，〈流水長者的本生〉系依據《[[金光明經|金光明經]]》，〈九劫超越〉依據《[[大毗婆沙論|大毗婆沙論]]》，〈屍毗王〉、〈薩埵王子〉、〈金色大鹿王〉、〈慈力王〉等依據《六度集經》、《菩薩本生鬘論》等，可見本經成立於印度大乘佛教末期，是以一切佛教為背景<ref name=ma2/>。
