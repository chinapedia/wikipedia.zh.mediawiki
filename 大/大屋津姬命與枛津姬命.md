'''大屋津姬命'''（{{lang|ja|おほやつひめ の みこと}}）和'''枛津姬命'''（{{lang|ja|つまつひめ の みこと}}）為《[[日本書紀|日本書紀]]》表記之名，祂們是[[日本神話|日本神話]]裡掌管[[树|樹木]]的[[女神|女神]]。姊姊大屋津姬命的別名為'''大屋都比賣神'''（{{lang|ja|おほやつひめ の かみ}}）、'''大屋都媛命'''等，妹妹枛津姬命的別名則有'''抓津姬神'''（{{lang|ja|つまつひめ の かみ}}）、'''抓津姬命'''、'''都麻都比賣命'''、'''爪津姬神'''等。

== 概要 ==
=== 日本書紀之記載 ===
根據《日本書紀》卷第一的記載，[[素戔嗚尊|素戔嗚尊]]與[[五十猛神|五十猛神]]下降至[[新羅|新羅]]國曾尸茂梨（{{lang|ja|そしもり}}，今[[韓國|韓國]][[江原道|江原道]][[春川市|春川市]]牛頭山）<ref>{{cite web|url=http://kamnavi.jp/it/sosimori.htm|title=曽尸茂梨(ソシモリ)考|date=|accessdate=2012-03-14|language=ja}}</ref>，但是五十猛神不想居住在此，遂以埴土作舟並乘之東渡，抵達[[出雲國|出雲國]]簸川上鳥上峰（今[[島根縣|島根縣]]斐伊川一帶）。祂下降時帶來許多樹種，播種在[[九州島|筑紫]]、[[八島_(消歧義)|大八洲國]]內，成為[[紀伊國|紀伊國]]之有功天神。

而次段則記載[[素戔嗚尊|素戔嗚尊]]拔[[鬍鬚|鬍髯]]化作[[杉木|杉]]、胸毛化作[[桧树|檜]]、尻毛成柀、眉毛化作[[樟树|櫲樟]]。五十猛神乃[[素戔嗚尊|素戔嗚尊]]之子，其胞妹為'''大屋津姬命'''、'''枛津姬命'''，此三神皆奉祀於紀伊國。

=== 先代舊事本紀之記載 ===
據《[[先代舊事本紀|先代舊事本紀]]》卷第四地祇本紀，[[素戔烏尊|素戔烏尊]]和[[五十猛神|五十猛神]]降於新羅曾尸茂梨，但後者不欲居於此，故以埴土作船、乘之東渡，抵[[出雲國|出雲國]]簸之河上與[[安藝國|安藝國]]可愛之河上鳥上峰。而五十猛神將八十樹種遍植於筑紫、大八洲內，與其胞妹'''大屋姬命'''、'''抓津姬命'''成為紀伊國奉祀之神。

== 解說 ==
依照《[[日本書紀|日本書紀]]》和《[[先代舊事本紀|先代舊事本紀]]》的紀錄，自古以來大屋津姬命和枛津姬命便被奉為[[林業|林業]]之神。尤其素有「木之國」之稱的紀伊國（今[[和歌山縣|和歌山縣]]全境及[[三重縣|三重縣]]南部）林業興盛，這二位神明在當地香火鼎盛。姊姊神名中的「屋（{{lang|ja|ヤ}}）」即為[[住宅|房屋]]之意，縱然樹木乃其管轄範圍，但房屋由木材構築而成。至於妹妹神名的「枛（{{lang|ja|ツマ}}）」，則是一種可作為建築材料的木材；也因此這兩位姊妹神祇成為[[建築業|建築業]]之神。

此外，[[島根縣|島根縣]][[大田市|大田市]]「大屋町」的地名即取自大屋津姬命。

== 信仰 ==
在日本主祭此二位女神的[[神社|神社]]計有下述：
* 大屋都姬神社（[[和歌山縣|和歌山縣]][[和歌山市|和歌山市]]）
* 都麻津姬神社（和歌山縣和歌山市吉禮）
* 伊太祁曾神社（和歌山縣和歌山市）
* 賣布神社（[[島根縣|島根縣]][[松江市|松江市]]）
* 妻山神社（[[佐賀縣|佐賀縣]][[杵島郡|杵島郡]][[白石町|白石町]]）
