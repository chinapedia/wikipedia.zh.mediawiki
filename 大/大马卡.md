'''大马卡'''（{{lang-ms|'''MyKad'''或'''Kad Pengenalan Malaysia'''}}、{{lang-en|'''Malaysian Identity Card'''}}、简称：'''IC'''）是[[马来西亚公民|马来西亚公民]]的身份证明。[[马来西亚政府|马来西亚政府]]于2001年9月开始发行「政府多用途智能卡」（大马卡），取代自1948年起开始发行的马来西亚身份证。大马卡是依据《1959年马来西亚国民注册法令》（Laws of Malaysia Act 78 - National Registration Act 1959）和《马来西亚国籍法》（Malaysian Nationality Law）执行。

[[马来西亚|马来西亚]]于2001年发行大马卡，是全球第一个使用「多用途智能卡」取代[[身份证|身份证]]的国家，同时也是第一个在身份证上使用[[微芯片|微芯片]]技术的国家。马来西亚政府于1998年通过拨款予国内的IRIS Corporation公司研发微芯片技术，因此也是全球第一个研发微芯片技术的国家。通过大马卡内置的微芯片，持有大马卡的马来西亚公民可使用大马卡缴付公共交通费、[[过路费|过路费]]、停车费、进行[[网络|网络]][[交易|交易]]、到各大[[银行|银行]]提款或[[存款|存款]]、上网更新国民资料、[[电子钱包|电子钱包]]、记录基本[[健康|健康]]资料（例如[[血型|血型]]、[[药物敏感|药物敏感]]资料等，以便加快[[医疗|医疗]]方面的诊断程序）、到商店购买并支付商品款项等等。此外大马卡里包含[[马来西亚护照|马来西亚护照]]资料，方便移民局处理出入国事务和申办护照之手续。未来马来西亚政府也打算以大马卡取代驾驶执照。

== 名称由来 ==
MyKad里的“My”是马来西亚的国家[[域名|域名]]，显示了MyKad的区域性。另外，“My”也可被解释为[[英文|英文]]中“我的”之意，表示持卡人的身份。而“Kad”则是[[马来文|马来文]]对英文“Card”（卡）的[[音译|音译]]，同时也是马来文“'''K'''ad '''A'''kuan '''D'''iri”（个人身份卡）的[[缩写|缩写]]。

== 种类 ==
大马卡共分为两大类共五种，即为：
=== 马来西亚公民 ===
* 大马卡（MyKad）：仅签发予年满12岁的马来西亚公民。
* 大马儿童卡（MyKid）：仅签发予未满12岁的马来西亚公民。
* 大马军人卡（MyTentera）：仅签发予服役于[[马来西亚武装部队|马来西亚武装部队]]的马来西亚公民。
* 大马警察卡（MyPolis）：仅签发予服役于[[马来西亚皇家警察|马来西亚皇家警察]]的马来西亚公民。
=== 非马来西亚公民 ===
* 大马永久居民卡（MyPR）：仅签发予持有永久居留马来西亚权的外国人士。

== 用途 ==
MyKad最常被当作马来西亚公民的身份证使用，但是也兼作各种用途，甚至能代替其他文件（[[驾驶执照|驾驶执照]]）。以下是官方资料列出的MyKad用途：

*[[身份证|身份证]]：MyKad自2001推行以来，渐渐取代了马来西亚在1949年后使用的旧身份证系统。
*[[驾驶执照|驾驶执照]]：MyKad也能被当作驾驶执照使用，政府也计划以MyKad取代现用的驾照。
*[[护照|护照]]资料：MyKad里包含马来西亚护照资料，以方便移民局处理出入国事务。然而，MyKad将不会替代现有的护照，公民出国时还是必须携带护照文件。
*健康资料：基本健康资料，例如血型，也能由MyKad中读取，这能加快医疗方面的诊断程序。
*[[电子现金|电子现金]]：MyKad能用来支付商品或服务的款项，尤其是那些涉及款项较小的交易。
*[[一觸即通卡|一觸即通卡]]（Touch 'n Go）：MyKad也能被用在交通服务上，例如缴付过路费、公车服务、[[轻快铁|轻快铁]]、[[捷运|捷运]]、停车费等。
*[[提款卡|提款卡]]：MyKad的晶片技术提高了资料的隐秘性，使得持有人能用MyKad到马来西亚各大银行进行提款、存款、查询结余、更改密码等服务。
*[[数位签章|数位签章]]：MyKad持有人也能使用[[电子证书|电子证书]]进行网络交易。

== 资料 ==
MyKad左上角的“Kad Pengenalan Malaysia”以及“Malaysian Identity Card”意为“马来西亚身份证”，旁边分别是MyKad标志以及[[马来西亚国旗|马来西亚国旗]]，接着下面为身份证号码，而晶片下显示的是持卡人的[[名字|名字]]和地址。持卡人照片位于MyKad右边，照片下面则表明持卡人的[[性别|性别]]；背面则是登记局局长的签名。

== 身份证号码（NRIC）的结构 ==
现有的马来西亚身份证号码格式于1990年推出，有12个数字，被"-"符号分为3个部分，如下：

:'''年年月月日日-生地-###性'''

前六个号码为出生日期。

生地为出生地点。

<nowiki>###</nowiki>是由马来西亚各州注册局的电脑系统生成的特别号码。

性代表性别，单数为男性，双数为女性。

在身份证的背面，在这12个号码时候还有2个号码，表示一个人曾拥有的大马卡的数量。

== 琐事 ==
*有公众投诉，MyKad晶片里显示的个人资料并不准确，尤其是宗教信仰和性别。
*政府规定，遗失MyKad者在重新申请时，必须依据遗失的次数缴交申请费，引来民众的不满。
*MyKad里的My是指马来西亚的域名。
== 遗失罚款 ==
至于国民遗失大马卡所面对的罚款，国民登记局会对第一次遗失大马卡的人士，罚款100马币，第二次丢失罚款300马币，第三个次或之后罚款1000马币。
