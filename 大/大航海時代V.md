{{Vgname|大航海時代V|ja=大航海時代V}}是[[光榮公司|光荣]]开发的一款航海探险主题游戏。它是《[[大航海时代IV_PORTO_ESTADO|大航海时代IV]]》的续作。这部作品一开始是以网页游戏的形式登陆，开发者竹田智一。<ref>{{Cite web |author=コーエーテクモゲームス|date=2013-12-21 |url=http://www.gamecity.ne.jp/d5/|title=大航海時代V|publisher=コーエーテクモゲームス|accessdate=2014-04-03}}</ref><ref>{{Cite web |author=箭本進一 |date=2013-12-21 |url=http://www.4gamer.net/games/242/G024279/20131220043/ |title=シリーズ最新作「大航海時代V」は，シリーズの遺伝子をしっかりと受け継いだシングルプレイ重視の作品に。ブラウザ展開の理由も聞いたプロデューサーインタビュー |work=4Gamer |publisher=Aetas |accessdate=2014-04-03}}</ref>2014年3月18日日服正式上线。游戏中的城市设施有酒场、造船所和交易所等。玩家可在酒场中治疗、搜集情报，并与老板娘交往，造船所用来改装、升级船只，而交易所的投资能够促成城市发展。游戏中出场的真实历史人物包括[[达迦马|达迦马]]、[[雷斯·欧玛蕾|雷斯·欧玛蕾]]、[[玛丽·里德|玛丽·里德]]、[[唐·梵·奥斯托利亚|唐·梵·奥斯托利亚]]、[[查尔斯|查尔斯]]、[[夏洛特·德·贝莉|夏洛特·德·贝莉]]、[[巴巴羅薩·海雷丁帕夏|巴巴罗斯·哈依勒丁]]、[[墨卡托|墨卡托]]等。<ref>{{cite web |url=http://pc.tgbus.com/info_57/27959/ |title=《大航海时代5》3月18日起航 最新情报及截图公开_游戏资讯_巴士单机游戏 PC.TGBUS.COM |format= |work= |accessdate=}}</ref>航海士系统是该游戏的一个特色，可以通过钻石抽取、奖券抽取、NP抽取、随机事件加入、活动赠送的方式获得新的航海士。<ref>{{cite web |url=http://www.265g.com/dhhsd5/gl/514368.html |title=大航海时代5航海士怎么获得 航海士获得攻略 - 265G大航海时代5网页游戏官网 |format= |work= |accessdate=}}</ref>

該作大陸服由中國手游代理，台服是紅心辣椒代理。
