'''大尔朱氏'''（{{bd|6世纪|?|556年|catIdx=Er,爾}}），北秀容（今山西省忻州市忻府区）人，[[尔朱荣|尔朱荣]]与[[北鄉公主|北鄉公主]]的女儿，先后成为魏孝明帝[[元诩|元诩]][[嫔|嫔]]、[[魏孝莊帝|魏孝莊帝]]元子攸的皇后及追尊齐神武帝[[高欢|高欢]]的妾。

== 北魏 ==

大尔朱氏原为北魏孝明帝[[元诩|元诩]]的[[嫔|嫔]]，[[武泰|武泰]]元年（528年），孝明帝被自己的母亲[[宣武灵皇后|胡太后]]害死，胡太后立临洮王[[元寶暉|元寶暉]]三岁的世子[[元钊|元钊]]为帝，大尔朱氏等嬪妃全被勒令做尼姑。

尔朱荣不满胡太后的作为，便拥立彭城王[[元勰|元勰]]第三子长乐王[[元子攸|元子攸]]为帝，是为孝莊帝。四月，尔朱荣制造了“[[河阴之变|河阴之变]]”，杀死胡太后和幼主元钊。之后尔朱荣官至大丞相，控制了北魏的朝廷。又把女儿大尔朱氏改嫁为孝莊帝皇后。大尔朱氏脾氣暴躁，十分轻视孝莊帝。

[[永安_(北魏)|永安]]三年（530年），孝莊帝不甘做[[傀儡|傀儡]]皇帝，以大尔朱氏懷孕臨産為契機，亲手殺死了來探望外孫的尔朱荣。尔朱荣的长子[[尔朱菩提|尔朱菩提]]等30人随尔朱荣入宫，也都被伏兵所杀。同年十月，尔朱荣的侄子[[汾阳|汾阳]][[刺史|刺史]][[尔朱兆|尔朱兆]]和[[尔朱世隆|尔朱世隆]]共谋，推戴[[太原郡|太原]][[太守|太守]]兼[[并州|并州]]事、长广王[[元晔|元晔]]即位。攻克洛阳后，孝莊帝被送晋阳拘禁，不久被缢杀。大尔朱氏新生的太子也被她的家人奪去活活摔死。

== 东魏 ==

大尔朱氏做寡妇不久，又被[[高欢|高欢]]纳为[[别室|别室]]，并生下一子叫[[高浟|高浟]]，高欢对她的敬重超过正妻[[娄昭君|娄昭君]]，见必束带，自称下官。高欢迎[[蠕蠕公主|蠕蠕公主]]还，大尔朱氏迎于木井北，与蠕蠕公主前后别行，不相见。蠕蠕公主引角弓仰射翔鸱，应弦而落；大尔朱氏引长弓斜射飞乌，亦一发而中。高欢喜曰：“我此二妇，并堪击贼。”后大尔朱氏出家为尼，高欢为她兴建起佛寺。

== 北齐 ==

高欢死去后，他的儿子[[高洋|高洋]]取代[[东魏|东魏]]，建立[[北齐|北齐]]，高浟封彭城王，大尔朱氏也被封为'''彭城太妃'''。北齐[[天保_(北齐)|天保]]七年（556年），高洋竟对大尔朱氏无礼，大尔朱氏怒而拒之，被高洋杀死。
