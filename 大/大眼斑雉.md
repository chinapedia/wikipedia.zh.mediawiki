'''大眼斑雉'''（[[学名|学名]]：''{{lang|la|Argusianus argus}}''），又名'''百眼雉雞'''，是一種分佈在[[東南亞|東南亞]][[婆羅洲|婆羅洲]]、[[蘇門答臘|蘇門答臘]]及[[馬來半島|馬來半島]]密林的[[雉|雉]]。在[[亞洲|亞洲]]一些[[汉字文化圈|汉字文化圈]]的地區被認為就是[[鸾鸟|鸾鸟]]的现实化身。

== 特徵 ==
大眼斑雉是褐色的，頭及頸都是藍色的，上胸赤紅色，冠及頸背的[[羽毛|羽毛]]是黑色的，腳呈紅色。雄雞是[[雉科|雉科]]所有種中最大的，長達2米。牠們的尾羽很長。雄雞最特別的地方是有很大、很闊及很長的次級飛羽，飛羽上有大眼裝飾。雌雞較細小及沉色，尾羽較短，眼狀裝飾亦較細小。雄性小雞3歲就會有成年的羽毛。

== 行為 ==
大眼斑雉會於早上及晚上在底層覓食。牠們並沒有[[雞形目|雞形目]]的油脂腺，而雌雞每次只會生2隻蛋。

大眼斑雉並不像其他[[雉科|雉科]]般色彩耀目，但其求愛方式卻十分獨特。雄雞會在[[森林|森林]]中清理場地，準備求愛之用。雄雞會大聲啼叫來吸引雌雞，並在雌雞面前張開雙翼成扇狀，展示雙翼上的「眼睛」，真正的眼睛則藏在當中，注視著雌雞。故此，[[卡爾·林奈|卡爾·林奈]]（Carolus Linnaeus）為牠們取了[[希臘神話|希臘神話]]中百眼巨人[[阿耳戈斯_(百眼巨人)|阿耳戈斯]]為其[[學名|學名]]。

大眼斑雉的求愛方式很像一些[[一夫多妻制|一夫多妻制]]的[[鳥類|鳥類]]，且很多人都相信牠們在野外是一夫多妻制的，不過牠們其實是[[一夫一妻制|一夫一妻制]]的。<ref>{{Cite web|author=Marquita Hafford|url=http://www.whozoo.org/Intro2002/MarquitaHaf/MYH_arguspheasant.htm|title=Argus Pheasant|publisher=WhoZoo|accessdate=2009-04-30}}</ref>

== 保育 ==
由於不斷失去棲息地及被獵殺，大眼斑雉被[[世界自然保護聯盟|世界自然保護聯盟]]列為[[近危|近危]]<ref name=IUCN/>，且受到《[[瀕危野生動植物種國際貿易公約|瀕危野生動植物種國際貿易公約]]》附錄二的保護。
