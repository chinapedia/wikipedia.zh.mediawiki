'''大韩民国国宝'''（{{lang|ko|대한민국의 국보}}）是[[韩国政府|韩国政府]]指定的具有特殊文化、历史和艺术价值的历史文物、遗迹和建筑<ref name="EncyKorea">{{cite web |title = National Treasure (국보 國寶) |url = http://100.empas.com/dicsearch/pentry.html?s=K&i=286573&v=45 |publisher = [[Empas|Empas]]/[[EncyKorea|EncyKorea]] |language = ko |accessdate = 2008-04-14}}</ref>。

1962年，韓國政府制定了《文化財保護法》，將具有悠久歷史的國家文物經過鑑定程序後，認定為有形文物財，並給予編號，有形文物財編號第一號是「[[崇禮門|崇禮門]]」（俗稱南大門）。 

大韩民国国宝列表包括许多旅游景点如[[宗廟_(首爾)|宗庙]]、[[海印寺|海印寺]]、[[佛国寺|佛国寺]]、[[石窟庵|石窟庵]]、[[高丽大藏经|高丽大藏经]]等。目前已有310物品已被列入大韩民国国宝列表<ref>{{cite web |website = Cultural Heritage Administration website |title = Cultural properties search results |url = http://www.cha.go.kr/english/search_plaza/EDetail_Result.jsp?maxDocs=10000&docStart=1&docPage=365&requery=0&region=&targetzone=&queryText=V_EKDCD%3D%2711%27&Province=ALL&Asset=11&Reign=ALL&query=&region=ztitle&asset=11&startNum=&endNum=&startYear=&endYear=&province=00&reign=99 |accessdate = 2006-01-10 |deadurl = yes |archiveurl = https://web.archive.org/web/20070930220541/http://www.cha.go.kr/english/search_plaza/EDetail_Result.jsp?maxDocs=10000&docStart=1&docPage=365&requery=0&region=&targetzone=&queryText=V_EKDCD='11'&Province=ALL&Asset=11&Reign=ALL&query=&region=ztitle&asset=11&startNum=&endNum=&startYear=&endYear=&province=00&reign=99 |archivedate = 2007-09-30 }}</ref>。
