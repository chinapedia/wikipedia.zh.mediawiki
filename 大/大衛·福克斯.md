'''大衛·福克斯'''（{{lang-en|David Fox}}，{{bd|1950年|12月30日||}}）是[[美國|美國]]的男性電子遊戲設計師、遊戲總監。除有開發[[LucasArts|LucasArts]]公司所推出遊戲經歷外，也曾在1977年於[[加州|加州]][[馬林縣_(加利福尼亞州)|馬林縣]]一帶成立世上第一棟可供大眾使用電腦設備的公共場所。<ref name="EE">{{cite web |url=http://www.electriceggplant.com/about.shtml |title=About Us |work=Electric Eggplant |accessdate=2016年4月20日}}</ref>

==生平==
=== 早年 ===
在十一歲時期，福克斯曾在垃圾桶中找尋到[[漢納巴伯拉動畫|漢納巴伯拉動畫]]公司所扔棄的《[[摩登原始人|摩登原始人]]》塞璐璐畫稿；之後將此製作了一部簡易的8釐米卡通片。<ref name="EE" />

而從[[索諾馬州立大學|索諾馬州立大學]]結束學業後，1977年福克斯與他的妻子安妮·福克斯（Anne Fox）於[[馬林縣_(加利福尼亞州)|馬林縣]]成立了一座名爲馬林電腦中心（Marin Computer Center）；可供大眾自由使用電腦的場所。<ref>{{cite web |url=http://www.digibarn.com/collections/newsletters/peoples-computer/peoples-1978-nov-dec/marin-computer-center.html |title=Marin Computer Center in People's Computers Nov-Dec 1978 |work=DigiBarn |accessdate=2016年4月20日}}</ref>另也有出版數本有關電腦動畫、及[[BASIC|BASIC]]程式語言內容書籍。

=== LucasArts時期 ===

在進入[[盧卡斯影業|盧卡斯影業]]旗下的電子遊戲部門[[LucasArts|LucasArts]]（當時稱為 Lucasfilm Games）後，福克斯除擔任遊戲設計師外，也在1986年擔任遊戲總監發行一款改編自奇幻電影《[[魔王迷宮|魔王迷宮]]》的[[魔王迷宮_(1986年遊戲)|同名作品]]；此亦爲LucasArts首部推出的[[冒險遊戲|冒險]]類型遊戲。<ref>{{Cite web |url=http://www.ign.com/articles/2014/03/17/maniac-mansion-revisiting-a-lucasarts-point-and-click-classic |archiveurl=https://web.archive.org/web/20140317091650/http://www.ign.com/articles/2014/03/17/maniac-mansion-revisiting-a-lucasarts-point-and-click-classic |title=Maniac Mansion: Revisiting a LucasArts Point and Click Classic |work=[[IGN|IGN]] |author=Kosta Andreadis |date=2014年3月16日 |archivedate=2014年3月17日 |accessdate=2016年4月20日}}</ref>

後續福克斯除與[[羅恩·吉伯特|羅恩·吉伯特]]等人開發《[[瘋狂大樓|瘋狂大樓]]》、《[[異形大進擊|異形大進擊]]》等遊戲外，也曾擔任LucasArts和[[休斯飛機公司|休斯飛機公司]]合力開發之多人電子娛樂設備「Mirage」的相關娛樂產品經理。<ref>{{cite web |url=http://variety.com/1992/film/news/lucas-hughes-create-space-mirage-101578/ |title=Lucas, Hughes create space Mirage |work=[[綜藝_(雜誌)|綜藝]] |author=Matt Rothman |date=1992年11月24日 |accessdate=2016年4月20日}}</ref><ref name="EE" />

=== 離任LucasArts ===

從LucasArts離職後，福克斯有前往Rocket Science Games公司繼續擔任遊戲設計師外。之後也有在網路公司Talk City, Inc.、或科技商New Trust等其它新興科技公司就職。<ref name="EE" />

在2014年曾在LucasArts和福克斯共事的羅恩·吉伯特與[[蓋瑞·溫尼克|蓋瑞·溫尼克]]開始構思一款模仿過往年代風格的冒險遊戲《[[銀蓮公園|銀蓮公園]]》，隨後福克斯也加入他們的開發團隊擔任程式設計。<ref>{{cite web |url=http://mixnmojo.com/news/David-Fox-and-Ken-Macklin-Join-the-Thimbleweed-Park-Team |title=David Fox and Ken Macklin Join the Thimbleweed Park Team! |work=MixMojo |author=Jennifer |date=2014年12月10日 |accessdate=2016年4月20日}}</ref>
