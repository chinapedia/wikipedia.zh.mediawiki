《'''降B大調弦樂四重奏大賦格'''》（[[德文|德文]]：'''{{lang|de|Große Fuge}}'''），'''[[作品号|作品]]133'''，是[[路德维希·范·貝多芬|路德维希·范·貝多芬]]于1825年至1826年间创作的單樂章[[弦樂四重奏|弦樂四重奏]]，原本打算用作[[第13弦乐四重奏_(贝多芬)|Op.130]]的終曲樂章。樂章以技術上難度及其內省性格（即使是貝多芬晚期作品之中）出名。乐曲创作时貝多芬已全聾。

貝多芬起初作此曲作為[[第13弦樂四重奏_(貝多芬)|第十三弦樂四重奏]]的結尾，但由於樂章对於當時樂手的演奏技术要求太高，聽眾亦不太能接受，因而出版者請他另寫一段新的終曲。一向不理會觀眾意見及口味的貝多芬卻同意了，於是以作品編號133出版此賦格。
 
大多數19世紀樂評都不看好此大賦格。[[丹尼爾·梅森|丹尼爾·梅森]]批評此曲使人很「反感」（{{lang|en|repellent}}），而[[路易斯·史波|路易斯·史波]]就稱它及其它貝多芬晚期作品為「無法解释和無法修改的恐怖」。但自從20世紀初，人們對其評價逐渐好轉，現在大賦格被認為是貝多芬最出色的作品之一。[[斯特拉文斯基|斯特拉文斯基]]說：
{{cquote2|這絶對是首當代音樂，而且永遠都會是當代。}}

== 分析 ==

全曲时长约18至20分钟。

樂章開始時是24音節「序幕」（''Overtura''），介紹[[賦格|賦格]]的兩主題之一：來自貝多芬十五號弦樂四重奏（op.132）第一樂章的旋律。接着作品突然跳進強烈的[[雙賦格|雙賦格]]，其中第二主題有大幅度的[[跳音|跳音]]，四支琴發出三連音、斷音及交叉旋律。

接着的賦格是一連串調、節奏、緩急互相對比的樂段。有時一節會忽然斷掉。將近尾時，音樂慢落來，時有長的停頓，接着序幕重現，帶入急速的終結。

像貝多芬其它晚期的終曲（例如第九交響曲的[[歡樂頌|歡樂頌]]）一樣，此賦格可以當作由多段子樂章組成的大樂章。每一小節都建立於原初主題的一次[[變奏|變奏]]上。作曲的手法亦為同期之代表：夾雜着[[變奏曲式|變奏曲式]]、[[奏鳴曲式|奏鳴曲式]]和[[賦格曲|賦格曲]]。

== 聯彈版大賦格及其手稿 ==

1826年初，出版商阿爾塔利亞在提出將大賦格從原本的弦樂四重奏作品中分拆之前，已向貝多芬指有「不少意見」希望此曲有一個鋼琴四手的改編版本（由於此曲當時並不受歡迎，因此這有可能僅是出版商的託辭），<ref>{{cite book |last=Solomon |first = Maynard |authorlink = Maynard Solomon |title = Beethoven |year = 1978 |publisher = Granada Publishing |isbn = 0-586-05189-9}}</ref>但當時貝多芬對此並無興趣。因此阿爾塔利亞轉而邀請奧地利作曲家{{link-de|安東|Anton Halm}}改編。然而貝多芬對該改編不甚滿意，於是自行譜寫一更忠於原作的版本。此版本於其第十四弦樂四重奏（作品131）後完成，阿爾塔利亞以作品134出版。<ref>{{cite book |last=Cooper |first=Barry |title=Beethoven |url=https://books.google.com/books?id=Tq5ci-9OMLEC&printsec=frontcover&dq=cooper+beethoven#v=onepage&q=cooper%20beethoven&f=false |accessdate=3 July 2013 |year=2008 |publisher=Oxford University Press |isbn=978-0195313314}}</ref>

2005年10月13日，有報導称[[賓夕凡尼亞州|賓夕凡尼亞州]]{{link-en|溫尼伍德|Wynnewood, Pennsylvania}}的{{link-en|柏爾馬神學院|Palmer Theological Seminary}}，一名圖書館管理員找到鑑證過的貝多芬1826年手稿，標題為「Grosse Fuge (a piano four-hands version of the op. 133 [[弦樂四重奏|string quartet]] finale)」<ref>http://www.nytimes.com/2005/10/13/arts/music/13beet.html</ref><ref>http://www.cbc.ca/arts/story/2005/10/13/beethoven_051013.html</ref>。此篇樂章即是貝多芬的鋼琴四手聯彈改編，作品134，已經失蹤了115年。2005年12月1日，[[蘇富比|蘇富比]]拍賣此手稿，以112万[[ISO_4217|英鎊]]（合195万美元）賣了給一名不知名的人。後來此人於2006年2月将此手稿及其它139份罕本原稿送給了紐約朱利亞音樂學院，大家這才知道他是亿万富翁{{link-en|布魯斯·科夫勒|Bruce Kovner}}。

此手稿曾出现在1890年[[柏林|柏林]]的一次拍卖会上，拍賣給了俄亥俄[[辛辛那提|辛辛那提]]的一個企业家。1952年，其女兒将該手稿及其他音乐作品（包括莫扎特的幻想曲）送給了賓州[[費城|費城]]的一所教堂。其后那所圖書館不知道用什么方式得到了此份手稿。

==大賦格與大眾文化==

值得注意的是，这篇作品在大眾文化中得到了回響。[[P·D·Q·巴哈|P·D·Q·巴哈]]寫了篇「最大賦格」（''Grossest Fugue''）。[[金·史丹利·羅賓森|金·史丹利·羅賓森]]寫了一本小說《Fifty Degrees Below》（New York: Bantam Books, 2005）說全球暖化，其中一幕，有一個人一邊在發了瘋似的吸塵，一邊将不同的房间中的麥克風聲響開至震耳欲聋，播放大賦格及「[[第29钢琴奏鸣曲_(贝多芬)|槌子键琴奏鳴曲]]」（''Hammerklavier''）的賦格樂章。
