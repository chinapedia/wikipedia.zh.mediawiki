'''大窑龙泉窑古瓷窑址'''位于中国[[浙江省|浙江省]][[丽水市|丽水市]][[龙泉市|龙泉市]][[小梅镇|小梅镇]]大窑村及周边，1988年被列入[[第三批全国重点文物保护单位|第三批全国重点文物保护单位]]。2013年大窑村南侧一山之隔的[[上垟窑址|上垟窑址]]（属[[庆元县|庆元县]]）做为[[第七批全国重点文物保护单位|第七批全国重点文物保护单位]]合并项目并入。

== 大窑村窑址 ==
大窑村下属的四个自然村是[[龙泉青瓷|龙泉青瓷]]的发源地和中心产区，大窑村窑址群确定的窑址超过80处<ref>{{Cite web |url=http://zjnews.zjol.com.cn/zjnews/201608/t20160818_1855065.shtml |title=龙泉大窑枫洞岩遗址日前发布10年来最新考古成果|accessdate=2017-03-13 |author=记者：马黎 |date=2016-08-18|work= |publisher=浙江在线，来源：钱江晚报|language=Zh-hans }}</ref>。

== 上垟窑址 ==
'''上垟窑址'''位于[[浙江省|浙江省]][[丽水市|丽水市]][[庆元县|庆元县]]竹口镇上洋村上洋溪沿线两岸山坡，属[[龙泉窑|龙泉窑系]]窑址。1997年8月29日列入[[浙江省文物保护单位|浙江省文物保护单位]]，2013年5月归入[[第三批全国重点文物保护单位|第三批全国重点文物保护单位]][[大窑龙泉窑遗址|大窑龙泉窑遗址]]。

===分布===
共有窑址8处，每处窑址又拥有多条窑床。分布范围约10万平方米。

===特征===
器形有碗、盘、洗、壶、瓶、罐、盏托、烛台、粉盒等。[[釉色|釉色]]为[[青釉|青釉]]，有淡青、青黄、豆青、粉青、青灰、水青等。采用刻、划、印三种装饰手法，多为折扇、莲瓣、荷花、蕉叶、水波、缠枝花等题材。烧制工艺水平较高，以北宋早期的淡青釉瓷最具代表性<ref>{{cite web|url=http://www.zjww.gov.cn/unit/2006-03-01/56569422.shtml | language= | title=上垟窑址 |publisher=浙江省文物局 |date= |accessdate= }}</ref>。
