'''大陆漂移学说'''是地球大陆相对于彼此的运动，因此似乎在海床上“漂流”<ref name=usghistory>[http://pubs.usgs.gov/gip/dynamic/historical.html USGS: Historical perspective on plate tectonics, last updated 19 September 2011]</ref>。最初由[[亞伯拉罕·奧特柳斯|亞伯拉罕·奧特柳斯]]在1596年提出，後來德国科学家[[阿尔弗雷德·魏格纳|阿尔弗雷德·魏格纳]]在1912年加以闡述，中文中“'''大陆漂移说'''”、“'''大陆漂移假说'''”均指同一概念。這個大膽的學說一直被學界忽視，直至1960年代[[海洋擴張說|海洋擴張說]]出現，令大陸飄移說得以發展，後來更闡述為[[板塊構造理論|板塊構造理論]]。大陆漂移的想法已经被[[板块构造理论|板块构造理论]]纳入，解释大陆如何移动<ref>{{Harvnb|Oreskes|2002|p=324}}</ref>。

主要内容为远古时代的[[地球|地球]]只有一块“[[泛古陆|泛古陆]]”或稱[[盤古大陸|盤古大陸]]的庞大陆地，被称为“[[泛大洋|泛大洋]]”的水域包围，大约于2亿年以前“泛大陆”开始破裂，到距今约二、三百万年以前，漂移的[[大陆|大陆]]形成现在的[[洲|七大洲]]和[[洋|五大洋]]的基本地貌。

值得一提的是大陸漂移學說與板塊構造學說有些根本的不同，前者假設推動力是[[潮汐|潮汐]]，後者假想推動力是由於[[地幔|地幔]]出現對流，拖動板塊。但在魏格纳1929年第四版的《大陆和海洋的形成》中，他已经接受了地幔对流提供动力的思想。

== 由來 ==
[[File:Alfred_Wegener_1910.jpg|thumb]]

[[亞伯拉罕·奧特柳斯|亞伯拉罕·奧特柳斯]]在他的地理[[百科全書|百科全書]]中提及[[美洲|美洲]]是「因[[地震|地震]]與[[潮汐|潮汐]]而從[[歐洲|歐洲]]及[[非洲|非洲]]分裂開去」及「如果有人拿出[[世界地圖|世界地圖]]，然後仔細觀測三[[大洲|大洲]]的[[海岸線|海岸線]]，就會發現（大陸）分裂的痕跡」。

== 爭論紛紛 ==
來自南非的地质學家[[杜托伊特|杜托伊特]]（Alexander du Toit）和英國的[[阿瑟·霍尔姆斯|阿瑟·霍尔姆斯]]（Arthur Holmes）都支持[[阿尔弗雷德·魏格纳|阿尔弗雷德·魏格纳]]的假說。到了1950年代，大陸飄移學說仍然得不到普遍接受。在1960年代前，終於出現轉機。[[罗伯特·迪茨|罗伯特·迪茨]]（Robert S. Dietz）、[[布鲁斯·希曾|布鲁斯·希曾]]（Bruce Heezen）和[[哈利·哈蒙德·赫斯|哈利·哈蒙德·赫斯]]（Harry Hess）得出一份地質研究報告，這就是[[海底扩张学说|海底扩张学说]]，得出實在的地質證據，解釋了三大洲為何分裂，令大陸漂移學說更具說服力。

== 理論 ==

首先，假設地球內部是[[玄武岩|玄武岩]]質，而地表則是[[花崗岩|花崗岩]]質，而大陸就像冰山浮在海面一樣，浮在融熔狀的玄武岩上。大陸因為潮汐的推動而移動分離。

== 证据 ==

* [[地质|地质]]构造方面的证据：[[阿巴拉契亚山脉|阿巴拉契亚山脉]]（和非洲[[阿特拉斯山脈|阿特拉斯山脈]]是同一條）是東北－西南走向，臨至大西洋西岸就中斷，而地質研究證明[[斯堪的納維亞山脈|斯堪的納維亞山脈]]與蘇格蘭、愛爾蘭的山脈是與阿巴拉契亚山脉同源。另外，有證據證明南非的開普山和布宜諾斯艾利斯山是同出一徹。可見曾有段時間，美洲、非洲和歐洲是相連。
* 大陸邊緣的吻合：將[[大西洋|大西洋]]兩岸的[[非洲|非洲]]和[[南美洲|南美洲]]拼在一起時，兩岸的大陸邊緣能十分吻合且完美的貼合。且經由兩岸岩層的研究，發現在非洲某處海岸的岩層，恰與拼合後的南美洲海岸的岩層相同，再度印證兩塊大陸曾經是相連的。
* 古生物[[化石|化石]]方面的证据：活在約2億年前的[[中龍|中龍]]是一種住在陸上淡水沼澤的[[爬蟲類|爬蟲類]]，無法越過大洋。地質學家在大西洋兩側的南美洲與[[南非|南非]]發現了中龍化石，即可證明南美洲與非洲過去是相連的。另外，二至三億年前的[[舌羊齒|舌羊齒]]植物，因種子很大無法藉風力飄洋過海，但此種化石卻出現在非洲、[[澳洲|澳洲]]、[[印度|印度]]、南美洲及[[南極洲|南極洲]]，由此可見，過去這些大陸是彼此連接在一起的。还有活证据——在两个大陆上发现同样的动物。 例如，在南美洲和非洲，发现了一些[[蚯蚓|蚯蚓]]家族（例如Ocnerodrilidae、Acanthodrilidae、Octochaetidae）。
* [[氣候|氣候]]的證據：在印度南部有冰川作用的痕跡，而印度南部遠離喜馬拉雅山，北部的冰川不可能在溶化前來到南部，何況印度南部是低緯度地區，年溫度高，不可能出現冰川。證明印度曾經是中高緯度地區。另外，地理學家在南極洲發現豐富的煤礦，[[煤|煤]]是由遠古植物遺骸變化而成，若南極洲一直都在南極圈內，嚴寒的天氣根本不容許南極洲有茂密的森林，休說豐富的煤礦。從而反證南極洲曾在低緯度地區。
* 古[[磁場|磁場]]的證據：把非洲、南美洲、澳洲、印度和南極洲各大陸連接在一起時，各大陸測得的古生代磁極邊緣軌跡大致重合在一起，證明它們在古生代期間確實屬於同一大陸地塊。
* 海底擴張學說和板塊構造學說的創立：60年代在洋底發現的條帶狀海底磁異常、轉換斷層及DSDP/ODP的鑽探成果，是海底擴張學說和板塊構造學說的基礎和主要內容。

== 發展 ==
此學說在魏格納因北極考察中，遭遇暴風雪而喪生之後就沉寂一時，到了六十年代，[[海底扩张学说|海底扩张学说]]中的地質報告證明了大西洋正在擴張，而三大洲就漸漸分離，令學界重新研究大陸漂移學說，後來更有人提出[[板块构造学说|板块构造学说]]，板块构造学说亦是現今[[地質學|地質學]]根基。
