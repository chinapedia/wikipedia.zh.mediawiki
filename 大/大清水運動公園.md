'''大清水運動公園'''為[[新北市|新北市]][[土城區|土城區]]內的一座小型山區[[公園|公園]]，位於明德路一段及永豐路之間。
[[File:Ooshimizu1.jpg|thumb]]

== 公園概要 ==
'''大清水運動公園'''的前身為「土城市第五[[公墓|公墓]]」，在都市發展計劃下規劃為公園預定地，其佔地約六千五百多坪，為土城區三處公墓改建的公園中面積最大的一座。改建過程中，於[[民國95年|民國95年]]8月完成公墓遷葬，[[民國96年|民國96年]]3月27日起動土改建，並於[[民國97年|民國97年]]2月27日竣工正式啟用。'''大清水運動公園'''總共有五個出入口，分別位於：其一、77148號路燈前（[[新北市立清水高級中學|清水高中]]斜對面）；其二、明德路一段137-3號前；其三、永豐路240號前；其四、永豐路242巷內（一邊通往國際路）；其五、通往萬善祠有應公經廟進出。

== 公園內的設施 ==
公園內設有休閒石椅、槌球場、景觀平台、[[涼亭|涼亭]]、木棧步道及兒童遊戲區等等，其中兒童遊戲區可見的設計有[[長桿|長桿]]、[[溜滑梯|溜滑梯]]、[[單槓|單槓]]等遊樂設施。

== 公園週邊設施 ==
[[新北市立清水高級中學|新北市立清水高級中學]]毗鄰明德路一段的兩個出入口。

== 公園相關事件 ==
曾於[[民國99年|民國99年]]10月22日下午時在公園內發現一具左手腕及腹部有多處刀傷的老人屍體。於警方的調查中，初步排除他殺，疑為久病厭世自盡。

== 交通路線 ==
*搭乘[[藍17|藍17]]、[[657|657]]路公車於「手信坊文化館」下車，步行約一分鐘即可挨近明德路一段137-3號或77148號路燈前出入口。
*搭乘[[藍40|藍40]]路公車於「應天府」下車，步行約二至三分鐘即可挨近永豐路240號前或萬善祠有應公的出入口。

於某日國際公園城站牌已更改為「手信坊文化館」
