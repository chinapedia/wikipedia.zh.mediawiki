'''大澤基宿'''（1567年—1642年2月24日）是[[日本戰國時代|日本戰國時代]]至[[江戶時代|江戶時代]]初期的武將、[[旗本|旗本]]。父親是[[大澤基胤|大澤基胤]]。母親是{{tsl|ja|木寺宮}}。別名'''基宥'''。通稱'''兵部'''、'''真松齋'''。

== 生平 ==
在[[永祿|永祿]]10年（1567年）出生，家中長男。永祿12年（1569年），父親{{tsl|ja|左衛門佐}}[[大澤基胤|基胤]]的居城{{tsl|ja|堀江城}}被[[德川家康|德川家康]]攻擊，在頑強抵抗後，接受勸告而臣服。

在[[小牧長久手之戰|小牧長久手之戰]]等戰事中從軍。[[天正_(日本)|天正]]16年（1588年）3月，敘任[[從五位下|從五位下]][[侍從|侍從]]、[[兵部大輔|兵部大輔]]。慶長5年，在[[關原之戰|關原之戰]]後，被賜予[[遠江國|遠江國]]{{tsl|ja|敷知郡}}{{tsl|ja|堀江村}}等6村，共1千5百5十石。[[慶長|慶長]]6年（1601年），昇進[[從四位下|從四位下]]。慶長8年（1603年）2月12日，在家康的{{tsl|ja|将軍宣下|將軍宣下}}時，與[[公家|公家]][[二條康道|二條康道]]相談儀式的事。此後，在[[李氏朝鮮|朝鮮]]和[[琉球|琉球]]的使者謁見時，負責展示文書等。

慶長13年（1608年）12月24日，與[[吉良義彌|吉良義彌]]一同就任高家職。慶長14年（1609年）9月23日，昇進{{tsl|ja|右近衛少将|右近衛少將}}。此後，昇進[[正四位|正四位下]]{{tsl|ja|左近衛中将|左近衛中將}}。慶長19年（1614年）6月28日，父親基胤死去。[[寬永|寬永]]9年（1632年）9月12日，把[[家督|家督]]讓予長男[[大澤基重|基重]]並隱居。隱居後，自號'''真松齋'''。在寬永19年（1642年）1月25日於堀江死去。享年76歲。

== 子孫 ==
有4男2女。另外有1個養子。長男[[大澤基重|基重]]繼承[[大澤氏|大澤家]]。次男[[持明院基定|基定]]成為[[公家|公家]][[持明院基久|持明院基久]]的養子。三男[[大澤基成|基成]]成為分家和[[旗本|旗本]]。四男[[大澤基近|基近]]成為分家和[[德川賴宣|德川賴宣]]的家臣。其中1個女兒是[[山名豐政|山名豐政]]室。

== 人物 ==
*被認為以[[高家|高家]]身份活動，是在職務上的「高家」開始。而[[江戶幕府|江戶幕府]]開始使用「高家」的職名，是在[[元和_(後水尾天皇)|元和]]、[[寬永|寬永]]期間。而擔任高家工作的原因，被認為是因為生母的家格（{{tsl|ja|木寺宮}}是以[[後醍醐天皇|後醍醐天皇]]的哥哥[[後二條天皇|後二條天皇]]的兒子[[邦良親王|邦良親王]]為家祖的皇族）。
