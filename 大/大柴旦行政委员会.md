'''大柴旦行政委员会'''，[[蒙古语|蒙古语]]称'''伊克柴达木'''，意为“大[[鹹水湖|盐湖]]”，是[[中国|中国]][[青海省|青海省]][[海西蒙古族藏族自治州|海西蒙古族藏族自治州]]下辖的一个[[县级行政区|县级行政区]]，位于[[柴达木盆地|柴达木盆地]]北部，是中国重要的矿业基地之一。大柴旦东与[[德令哈市|德令哈市]]毗邻，西与[[茫崖市|茫崖市]]相连，南隔[[察尔汗盐湖|察尔汗盐湖]]与[[格尔木市|格尔木市]]相望，北与[[甘肃省|甘肃省]]接壤，面积2.1万平方公里，下辖[[柴旦镇|柴旦镇]]和[[锡铁山镇|锡铁山镇]]，总人口1.5万人。大柴旦行政区的政府机关称为大柴旦行政委员会，是海西蒙古族藏族自治州人民政府的派出机构<ref>{{Cite web|url=http://www.hxdcd.gov.cn/structure/zjcd/cdnjxx_93063_1.htm|title=大柴旦行政区|accessdate=2018-05-09|date=2008-05-13|publisher=大柴旦行政委员会|archiveurl=https://web.archive.org/web/20091216104847/http://www.hxdcd.gov.cn/structure/zjcd/cdnjxx_93063_1.htm|archivedate=2009-12-16|deadurl=yes}}</ref>，与中共大柴旦工作委员会合署办公。

== 自然地理 ==
大柴旦地区海拔较高，行政委员会驻地柴旦镇海拔3173米，气候终年多风少雨，年平均降水量为83.5毫米，无霜期为108天，年平均[[气温|气温]]1.4[[摄氏度|摄氏度]]，属高原[[沙漠|沙漠]]气候。大柴旦地区的河流均为[[内流河|内流河]]，湖泊多为[[鹹水湖|盐湖]]，水源以周边山地冰雪融水和[[地下水|地下水]]为主要补给来源。区内多[[雅丹地貌|雅丹地貌]]、[[温泉|温泉]]。

大柴旦地区矿产丰富，[[铅|铅]]、[[锌|锌]]、[[金|金]]、[[银|银]]、[[铬|铬]]、[[重晶石|重晶石]]和[[煤矿|煤矿]]蕴藏量大，另外在其盐湖中还富含[[硼|硼]]、[[锂|锂]]、[[芒硝|芒硝]]、[[溴|溴]]、[[食盐|食盐]]、[[钾|钾]]和[[镁|镁]]。其矿产保有储量潜在总价值估计为57560亿元[[人民币|人民币]]，约占青海省的三分之一。

== 历史沿革 ==
*隋唐时期，为吐谷浑、吐蕃辖区
*元明清时期，为蒙古族左翼和硕特部北左旗驻地
*青海解放后，隶属青海省[[都兰县|都兰县]]
*1956年，设柴达木工作委员会、行政委员会，驻大柴旦
*1958年，[[海西蒙古族藏族自治州|海西]][[中国共产党海西蒙古族藏族自治州委员会|州委]]、[[海西蒙古族藏族自治州人民政府|州人民政府]]迁至大柴旦
*1959年，设市，成为当时柴达木盆地政治、经济、文化和交通中心
*1964年，撤市设镇
*1992年，成立中共大柴旦工作委员会、大柴旦行政委员会，为海西州委、州政府的县级派出机构。<ref name=":0">{{cite web|url=http://www.hxdcd.gov.cn/zjcd/cdgk.htm|title=柴旦概况|accessdate=2018-01-10|publisher=大柴旦行政委员会}}</ref>

==人口==
2010年六普数据显示：全区[[常住人口|常住人口]]为13671人，同2000年第五次全国人口普查相比，十年共增加4726人，增长52.83%，年平均增长率为4.33%。平均每个家庭户的人口为2人，比2000年第五次全国人口普查的2.4人减少了0.4人。常住人口[[性别比|性别比]]（以女性为100，男性对女性的比例）由2000年第五次全国人口普查的135.64上升为229.58，上升了93.94个百分点。全区常住人口中，0—14岁占8.54%；15—64岁占90.54%；65岁及以上的占0.92%。同2000年第五次全国人口普查相比，0—14岁人口的比重下降了9.49个百分点，15—64岁人口的比重上升了9.72个百分点，65岁及以上人口的比重下降了0.23个百分点。
 
全区常住人口中，汉族人口为10893人，占常住人口的79.68%；各少数民族人口为2778人，占常住人口的20.32%。其中：回族924人，占6.76%；哈萨克族457人，占3.34%；蒙古族419人，占3.06%；藏族395人，占2.89%；土族203人，占1.49%；撒拉族150人，占1.1%；其他民族230人，占1.68%。同2000年第五次全国人口普查相比，汉族人口增加了3216人，增长了41.89%；各少数民族人口增加了1510人，增长了119.08%。少数民族人口比重上升了6.14个百分点。

全区常住人口中，文盲人口（15岁及以上不识字的人）为514人，文盲率为4.11%。同2000年第五次全国人口普查相比，文盲人口持平，文盲率下降1.64个百分点。

全区常住人口分布：[[柴旦镇|柴旦镇]]9750人，[[锡铁山镇|锡铁山镇]]3921人。柴旦镇常住人口中，镇区人口4755人，农牧区人口893人（其中马海公司147人，柴旦村及马海村746人），周边矿山企业及公路施工人数4102人。 锡铁山镇常住人口中，镇区人口3405人，周边地区人数516人。
== 经济 ==
该地原为[[蒙古族|蒙古族]]和[[哈萨克族|哈萨克族]][[牧场|牧场]]，后随着[[采矿业|采矿业]]的发展，[[汉族|汉族]]成为当地的优势人口。当地经济以矿业为支柱，铅、锌、岩金、重晶石、伴生银、伴生铬等7种矿产位居全省之首；湖盐、芒硝、溴、锂、铬等多种矿产也位居全省前列。优势矿种为铅、锌、岩金、硼、锂、煤。潜在优势矿种为钾盐、镁盐、重晶石、芒硝、溴等。这些矿产探明资源量保有储量潜在经济价值达57560亿元，分别占省、州矿产潜在经济价值总量的26%和38.37%。主要矿山均控制在[[西部矿业|西部矿业]]手中。2007年人均GDP达100419元人民币（约合13740[[美元|美元]]）。近年来发展迅速，2016年完成地区生产总值29.9亿元，增长11.4%；完成规上工业增加值19.7亿元，增长11.8%；完成固定资产投资53.3亿元，增长17.1%；完成全区公共财政预算收入8.8亿元，其中地方公共财政预算收入2.8亿元，增长7.4%；城镇常住居民人均可支配收入26862元，增长9%；农村常住居民人均可支配收入11465元，增长9.2%；实现社会消费品零售总额1.54亿元，增长10.2%。<ref name=":0" />
