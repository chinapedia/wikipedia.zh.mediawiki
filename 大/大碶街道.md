'''大碶街道'''，是[[中国|中国]][[浙江省|浙江省]][[宁波市|宁波市]][[北仑区|北仑区]]的一个[[街道_(行政区划)|街道]]，位于北仑区西部，常住人口5.2万人，街道办事处位于镇兴路26号。<ref name="xzqh">[http://www.xzqh.org/html/show.php?contentid=6385 行政区划网 浙江省 > 宁波市 > 北仑区 > 大碶街道]</ref>

== 地理 ==
大碶街道位于北仑区西部，东连[[柴桥街道|柴桥街道]]和[[霞浦街道|霞浦街道]]，西接[[鄞州区|鄞州区]][[东吴镇|东吴镇]]、[[五乡镇|五乡镇]]，辖区面积105.5平方公里，是浙江省面积最大的街道。辖区西面为灵峰山，南部太白山为[[天台山_(浙江)|天台山]]余脉，主要河流为[[岩河|岩河]]。<ref name="xzqh" />

== 下辖 ==
大碶街道下辖6个社区，41个行政村：<ref name="xzqh" />
* 社区：坝头、太白、灵岩、学苑、金泉、高田王

== 经济 ==
大碶街道经济发达，镇域经济位列宁波市前20。工业以模具、电子电器、机械五金为主，有“中国模具之乡”的称号。<ref name="official">[http://www.dq.gov.cn/z1_01.asp?cid=82 大碶概况] {{webarchive|url=https://web.archive.org/web/20110314052517/http://www.dq.gov.cn/z1_01.asp?cid=82 |date=2011-03-14 }}</ref>

== 旅游 ==
九峰山、古育王遗址、灵峰寺和乐氏宗祠是大碶街道的重要旅游目的地。<ref name="official" />

== 交通 ==
{{中国国道名|329}}和{{中国高速公路名|S1|[[甬台温高速公路|甬台温高速公路]]宁波段||Zhejiang}}通过大碶街道，其中后者设立北仑出口并延伸至[[大碶疏港高速公路|大碶疏港高速公路]]。铁路[[北仑支线|北仑支线]]通过大碶街道，设立[[大碶站_(铁路)|大碶站]]，未来还将在邬隘设立[[宁波铁路集装箱中心站|宁波铁路集装箱中心站]]。[[宁波轨道交通1号线|宁波轨道交通1号线]]通过大碶街道，设立[[邬隘站_(地铁)|邬隘站]]和[[大碶站_(地铁)|大碶站]]。
