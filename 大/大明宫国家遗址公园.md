'''大明宫国家遗址公园'''是一个位于[[西安|西安]]的[[国家考古遗址公园|国家考古遗址公园]]。其范围位于[[唐朝|唐朝]][[大明宫|大明宫]]内，於2010年10月1日正式开园。从拆迁到建设，大明宫遗址公园耗资120亿元人民幣。<ref name=cnwest20100927>{{Cite web |url=http://news.cnwest.com/content/2010-09/27/content_3541343.htm |title=大明宫国家遗址公园“十一”开园 成人票价为60元 |date=2010-09-27}}</ref>

遺址公園有三分之二區域免費開放，遊客進入遺址密集區需要購票。

==历史==
1938年[[花园口决堤事件|花园口决堤事件]]后，大量河南省难民沿火车道走到西安，定居道北。1980年代初，道北地区作为唐大明宫遗址所在地，因为文物保护法的规定被要求不能进行各种大型建设。<ref name=nfzm>{{Cite web |url= http://www.infzm.com/content/109641 |title= 更多遗址在凋零中 隋唐长安城怎么保护？ |author= 朱晓佳 |date= 2015年5月21日 |publisher= 南方周末 |language= Zh-hans |accessdate= 2015年5月25日 |quote=  |deadurl= yes |archiveurl= https://web.archive.org/web/20150525005319/http://www.infzm.com/content/109641 |archivedate= 2015年5月25日 }}</ref>

2008年遗址公园开始建设，2010年9月30日舉行開園儀式，<ref name=xh20100930>{{Cite web |url=http://news.xinhuanet.com/society/2010-09/30/c_12623919.htm |title=西安120亿“重建”盛唐皇宫成为市民公园 比故宫大四倍 |website=新華網 |date=2010-09-30}}</ref>10月1日正式开园。

==布局==
===大明宫遗址博物馆===
大明宫遗址公园的地下文物博物馆，依托于保护前村民们私自挖凿的垃圾坑建设。<ref name=nfzm/>

==争议==
大明宫开园后刚建好的大量现代仿古建筑被拆除，这是被认为要为申请[[世界遗产|世界遗产]]做工作。<ref>[http://news.ifeng.com/mainland/detail_2010_08/28/2347817_0.shtml 西安大明宫遗址公园为申遗拆除已建成现代式建筑]</ref>

==火灾==
2015年8月19日凌晨1时20分，遗址公园内的仿古建筑模型“栖凤阁”发生火灾，建筑上层木质部分被烧毁，钢构的下半部分也出现毁坏。2015年9月7日，嫌疑人雷某某被西安市公安局新城分局抓获。<ref>{{Cite web |url= http://politics.people.com.cn/n/2015/0908/c1001-27554676.html |title= 西安大明宫遗址公园纵火嫌疑人已被刑事拘留 |author=  |date= 2015年9月8日 |publisher=  人民网|language= Zh-hans |accessdate= 2015年9月8日 |quote=  }}</ref>
