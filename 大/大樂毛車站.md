'''大樂毛車站'''（{{jpn|j=大楽毛駅|hg=おたのしけえき|rm=Otanoshike eki|f=y}}）是一位於[[北海道|北海道]][[釧路市|釧路市]]大樂毛5丁目，[[北海道旅客鐵道|北海道旅客鐵道]]（JR北海道）[[根室本線|根室本線]]沿線的[[鐵路車站|鐵路車站]]。車站編號為'''K50'''。電報略號為'''{{lang|ja|シケ}}'''。

車站名稱來自[[阿伊努語|阿伊努語]]的「ota-noski/{{lang|ja|オタ・ノシキ}}」（砂灘的中央）<ref>{{Cite web|url=http://www.pref.hokkaido.lg.jp/ks/ass/grp/2130P.pdf|title=アイヌ語地名リスト|accessdate=2017-08-16|author=北海道庁|date=|publisher=北海道庁}}</ref>。

== 歷史 ==
* 1901年（[[明治|明治]]34年）7月20日 - 以[[北海道官設鐵道|北海道官設鐵道]]車站開始營業。一般車站。
* 1905年（明治38年）4月1日 - 由[[日本國有鐵道|官設鐵道]]接管。
* 1979年（[[昭和|昭和]]54年）7月15日 - 除專用線外，廢除其他[[載貨汽車|載貨汽車]]的貨物處理。
* 1984年（昭和59年）2月1日 - 廢除所有載貨汽車貨物及行李處理。
* 1986年（昭和61年）11月1日 - 車站簡易委託化。
* 1987年（昭和62年）4月1日 - [[國鐵分割民營化|國鐵分割民營化]]後，由JR北海道繼承。
* 1989年（平成元年）10月2日 - 改建為現在的車站大樓。與釧路市政廳大樂毛分所共同興建。
* 2001年（[[平成|平成]]13年） - 廢除簡易委託，車站無人化。

== 車站構造 ==
大樂毛車站為1面2線、[[島式月台|島式月台]]的地面車站。基本上所有上下行列車都使用1號線停靠。而兩個月台用[[架空道路|架空道路]]來連接。而之前車站大樓對出曾存在著1號線，但被廢除後將原本島式月台的2號及3號線改為現在的1號及2號線。

車站大樓主要位於車站北面。現在為[[釧路車站|釧路車站]]管理的無人車站。車站大樓內有已關閉的辦公窗口外，亦設置了[[自動售票機|自動售票機]]（操作時間07時00分～18時00分）。早上及晚上分別有1列以本車站為起點及終點站、往釧路方向行駛的列車，不過星期日將會暫停。

車站在戰後曾經發展為[[軍馬|軍馬]]（釧路種）的大型交易中心。而本州製紙（現在的[[王子製紙|王子製紙]]）亦曾在附近開設釧路工廠，亦開始運輸處理木塊等原材料，成為北海道少數的貨物處理車站。但在昭和50年代開始運輸工作由[[載貨汽車|載貨汽車]]代替，因此廢除貨物處理及專用路軌。

== 車站周邊 ==
在之前為有人車站時，車站大樓為古老的木造建築。現在作為釧路市的郊區住宅區，住宅的數量不斷增加，而主要乘客是前往釧路市中心地區的上班及上學人士。
