'''大眾媒體'''，又稱'''大眾傳媒'''，是指向[[大眾傳播|大量受眾廣播]]之[[傳播媒體|傳播媒體]]，同時是[[第三產業|第三產業]]、[[知識產業|知識產業]]和[[訊息產業|訊息產業]]。實現交流的技術手段各異：[[電台廣播|電台廣播]]、錄制[[音樂|音樂]]、[[電影|電影]]和[[電視|電視]]等廣播媒體通過電子途徑傳輸信息；[[報紙|報紙]]、[[書籍|書籍]]、宣傳冊和[[漫畫|漫畫]]等紙質[[媒體|媒體]]採用實物傳播信息；<ref>Riesman ''et al.'' (1950) ch.2 p.50</ref>[[公告牌|公告牌]]、標誌和招貼畫等戶外媒體則主要放置在商業建築、體育場館、商店和公交車上。<ref name="Mass Media">{{cite web | url=http://www.enotes.com/mass-media-reference/mass-media | title=Mass Media | accessdate=November 28, 2011}}</ref>[[公共演說|公共演說]]和事件組織也可以被看作是一種形式的大眾媒體。<ref name="buzzle">{{cite web | url=http://www.buzzle.com/articles/different-types-of-mass-media.html | title=Different Types of Mass Media | publisher=Buzzle.com | accessdate=November 26, 2011 | author=Manohar, Uttara}}</ref>[[數碼媒體|數碼媒體]]則包括[[互聯網|互聯網]]和[[流動電話網絡|流動電話網絡]]。

控制這些技術的組織，例如[[電視台|電視台]]或[[出版社|出版社]]，通常也被稱作是大眾媒體。<ref name="dictionary">"Mass media", Oxford English Dictionary, online version November 2010 {{page needed|date=November 2011}}</ref><ref name="Arguing for a general framework for mass media scholarship">{{cite book|author=Potter, W. James|title=Arguing for a general framework for mass media scholarship|publisher=SAGE|year=2008|isbn=978-1-4129-6471-5|page=32|url=http://books.google.com/books?id=H9u9E2wsVjAC&pg=PA32}}</ref>

== 定义 ==

{{Fact|20世纪末，大众媒体被分为8大产业：书籍、报纸、杂志、录音制品、电台广播、电影、电视和互联网|time=2013-08-21T04:42:34+00:00}}。随着数字交流科技在20世纪末21世纪初的快速发展，对于何种形式的媒体可以被归类为大众媒体的问题开始变得突出。例如，对于移动电话、视频游戏和电脑游戏（例如网络游戏）列入其中。2000年代，一种被称作“{{link-en|七大大众媒体|Seven mass media}}”的分类开始流行，根据发明时间排序，它们分别是：
# [[印刷|印刷]]（包括[[圖書|圖書]]、小冊、[[報紙|報紙]]、[[雜誌|雜誌]]等）：自十五世紀後期開始。
# [[錄音|錄音]]（包括[[黑膠唱片|黑膠唱片]]、[[磁帶|磁帶]]、[[卡式錄音帶|卡式錄音帶]]、[[CD|CD]]、[[DVD|DVD]]等）：自十九世紀後期開始。
# [[電影|電影]]約從1900年開始。
# [[電台廣播|電台廣播]]約從1910年開始。
# [[電視|電視]]約從1950年開始。
# [[網際網路|網際網路]]約從1990年開始。
# [[行動電話|行動電話]]約從2000年開始。

=== 特征 ===

根据[[剑桥大学|剑桥大学]]约翰·汤普森的研究，大众媒体共有5大特征：<ref name="thompson">Thompson: [http://books.google.com/books?id=iXHzjIwQae4C ''The Media and Modernity''] pp.26-8, 74</ref>

* “包括生产和传播的技术及制度方法”，这在媒体发展史上反应的很明显，它们都有自己的商业用途。
* 包含“符号形式的商品化”，材料的生产依赖于其制造和销售大量作品的能力。例如电台依赖其出售给广告商的播出时间，而报纸则依赖于其出售的版面。
* “隔离开信息的生产制造和接收”。
* 同制造者相比，它“在时间和空间上的广泛传播” 。
* “信息分发”——是一种“一对多”的交流模式，产品大量生产并分发给大量受众。

== 形式 ==

=== 广播媒体 ===

广播媒体的内容播出顺序被称为档期。[[电视|电视]]和[[电台广播|电台广播]]节目都是通过广播频道传播的。

[[有线电视|有线电视]]节目通常会和电视、电台同时播出，但是受众更窄。

某个播出组织会在同一时间通过不同频道播出多套节目。而同时也会有两个或以上的组织共享一个频道，他们各自在一天内固定的时段使用该频道。

=== 电影 ===
{{main|电影}}
电影既可指某一单一动态影像作品，也可以指整个领域。电影是由录制人员创作，使用[[摄像机|摄像机]]记录的，或者是使用[[动画|动画]]技巧或者[[特殊效果|特殊效果]]创作出的。其包含一系列单一的画面，但是当这些画面连续快速播放时，就给观众一种动态影像的感觉。由于[[視覺暫留|視覺暫留]]效果，画面之间的闪烁并不会被察觉。另外造成人们感觉画面是动态的心理效果是[[贝塔运动|贝塔运动]].

很多人都将电影视作一种重要的[[艺术|艺术]]形式：电影可以娱乐、教育、开化并启发观众。通过[[配音|配音]]和[[字幕|字幕]]翻译电影传达的信息，任何电影都可以获得全世界的关注。电影也是由某一特定文化创造出的手工品，可以反映并反过来影响其文化。

=== 电子游戏 ===
{{main|电子游戏}}
电子游戏是指人通过电子设备（如电脑、游戏机及手机等）进行的一种娱乐方式。在日常使用中，“[[电脑游戏|电脑游戏]]”或者“PC游戏”通常指在[[个人电脑|个人电脑]]上进行的游戏；[[电视游戏|电视游戏]]则是在专门设计的游戏平台上玩的游戏，通过和标准的[[电视|电视]]机互动完成；[[街机|街机游戏]]则是指专门为设计给街头路人每次付费有偿玩的游戏。电子游戏还包括：移动电话、个人数字助理、高级计算器等上的游戏。

==== 博客 ====
{{main|博客}}
随着互联网的快速发展，博客也变得无处不在。博客是一般由个人维护的网站，其中内容主要是评论、时间描述或者一些如图片、视频等互动媒体。博客中的文章往往按照时间先后逆向排序，最新撰写的博文会出现在最上面。许多博客提供有关某一特定话题的评论或欣慰；其他功能主要是个人在线日记。一个典型的博客包括有文字、图片、图标、链往其他博客、网页或相关媒体的链接。对于许多博客而言，读者可以在留言区同博主交流是很重要的一部分。博客主要以文字为主，但是其他形式的博客也开始兴起。[[微博客|微博客]]是另一种类型的博客，其中的博文长度都很短小。

==== RSS订阅 ====
[[RSS|RSS]]是一種聚合新聞和類新聞網站的格式，類新聞網站包括像是[[连线|Wired]]之類主要的新聞網站、像[[Slashdot|Slashdot]]社群網站、以及個人的部落格。RSS檔案包含了全文或是節錄的文字，再加上發用者所訂閱之網摘佈資料和授權的元數據。讀者可以依需求訂閱不同網站的RSS，當網站有更新時，網站會送出RSS，讀者可以配合[[瀏覽器|瀏覽器]]或是其他軟體更新的部份內容或全部內容。RSS可以使讀者用自動的方式和網站互動，不需固定進入一網站也可以知道網站更新的內容，而且訂閱內容時不需提供[[電子信箱|電子信箱]]等個人資訊<ref>{{Cite web

}}</ref>。

==== 播客 ====
{{Main|播客}}
[[播客|播客]]是一系列电子媒体文件，通过互联网传播，在便携式媒体播放器和电脑上播放。播客可以指上述用互联网传播媒文件的形式、电子媒体文件本身，也可指製作上述電子媒體文件的人。

==== 图书 ====
{{Main|图书}}
[[File:Brockhaus_Lexikon.jpg|right]][[布罗克豪斯社交词典|社交词典]]，1902年]]

图书是一系列印有文字的纸张，一侧装订在一起，通过封面包裹起来。电子格式发布的书籍被称作[[电子书|电子书]]。

==== 杂志 ====
杂志是一个周期性的出版物，其中包含多样的文章，主要通过广告和/或读者购买来获得资金。

杂志一般分为消费者杂志和商业杂志两类。

==== 报纸 ====
报纸是包含新闻、信息和广告的出版物，通常印刷在成本较低的新闻纸上。报纸可能有或没有特定兴趣指向，通常按日或周发行。第一张印刷报纸出版于1605年，在电台和电视的竞争之下，此类媒体依然繁荣发展。但是近来互联网的兴起对于报纸商业模式有着重大打击。

== 目的 ==
[[File:911-Panel.JPG|thumb]]新闻博物馆中陈列的911次日全美及世界各地区的报纸头条。]]

大众媒体不仅仅局限于新闻，尽管人们往往会误解这一点。它们可以用作如下目的：
* [[倡導|倡導]]，可以为商业或社会服务。包括有[[广告|广告]]、[[市场营销|市场营销]]、[[政治宣传|政治宣传]]、[[公共关系|公共关系]]以及[[政治|政治]]沟通。
* [[娱乐|娱乐]]，传统的方式是通过表演、音乐、体育、电视节目以及阅读；自从20世纪后期，电子游戏开始兴起。
* 公共服务公告以及预警。<ref name="wisegeek">{{cite web | url=http://www.wisegeek.com/what-is-mass-media.htm | title=What is Mass Media? | publisher=Conjecture Corporation | date=4 October 2011 | accessdate=November 26, 2011 | author=Smith, S.E.}}</ref>

== 道德问题和批评 ==
缺乏对本地或特定话题的关注是对大众媒体的常见批评。大众新闻媒体为了满足更广大读者群的需求，不得不选择播报国家或者国际新闻。这样它们常常会忽视一些很有趣或者很重要的本地新闻，因为对于大多数读者来说他们并没有兴趣。<ref name="wisegeek" />

但有時大眾媒體會出現過於重視社會新聞，忽略其他議題的情形：大眾媒體（尤其是電視）很注重[[收視率|收視率]]，而當一些社會新聞出現時，相關報導的收視率都會上昇，因此新聞會集中在特定社會新聞的相關報導，忽略大眾不一定有興趣的事件<ref>{{cite web
 |title= 【公民看國會第166集】文字大綱：大眾媒體怎麼看？新聞清流何處在？
 |publisher= 公民監督國會聯盟
 |url= http://www.ccw.org.tw/p/18105
 |accessdate= 2013-08-21}}</ref>。

“大众”一词就意味着媒体产品的受众是由大量被动接受、无差别的个体组成。这种印象连同早年对于大众文化和世俗社会的批评，让人们认为大众媒体对于现代社会生活有着负面影响，创造一种模糊而同质的文化，取悦个体而不会让他们有挑战的机会。但是，交互式的数字媒体的读者已经不再只是被动接受。<ref name="thompson" />

20世纪50年代开始，在一些达到更高工业化程度的国家里，影院、电台、电视等大众媒体在政治力量中扮演关键角色。<ref name="Eco67">Eco, U. (1967) quote:{{quotation|Not long ago, if you wanted to seize political power in a country, you had merely to control the army and the police. Today it is only in the most backward countries that fascist generals, in carrying out a coup d'etat, still use tanks. If a country has reached a high level of industrialization the whole scene changes. The day after the fall of Khrushchev, the editors of Pravda, Izvestiia, the heads of the radio and television were replaced; the army wasn't called out. Today a country belongs to the person who controls communications.}} {{page needed|date=November 2011}}</ref>近期的研究显示，媒体所有权有着不断集中化的趋势，许多媒体产业已经高度集中，由少数公司掌控。<ref name="The SAGE Handbook of Media Studies">{{cite book|editor=Downing, John|title=The SAGE Handbook of Media Studies|publisher=SAGE|year=2004|isbn=978-0-7619-2169-1|page=296|url=http://books.google.com/books?id=isFAwqhITCUC&pg=PA296}}</ref>
