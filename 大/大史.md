'''《大史》'''（{{lang-pi|Mahāvaṃsa}}，[[英文|英文]]譯名：Mahavamsa/Mahawansha，在巴利三藏英譯目錄或辭典中常縮寫爲Mhvs.或Mhv.），[[斯里蘭卡|斯里蘭卡]]古代[[巴利文|巴利文]]歷史文獻，採用[[編年|編年]][[史詩|史詩體]]，作者是[[摩訶那摩|摩訶那摩]]（[[英文|英文]]譯名：Mahanama）等。該書涉及[[斯里蘭卡|斯里蘭卡]]、[[印度|印度]]等[[南亞|南亞]]等地區，以及[[上座部佛教|南傳上座部佛教]]的歷史。此外，由於該書辭藻華麗，因而亦被奉為[[斯里蘭卡|斯里蘭卡]]古典文學的瑰寶。<ref name="《大史》譯者序">[http://www.hkbuddhist.org/magazine/484/484_16.html 香港佛教聯合會：韓廷傑《大史》中譯本譯者《序言》] {{webarchive|url=https://web.archive.org/web/20071108045126/http://www.hkbuddhist.org/magazine/484/484_16.html |date=2007-11-08 }}</ref>

== 關於篇幅問題 ==
關於《大史》的篇幅，是有所爭議。其爭論焦點在於，《大史》應包括現今流傳的全部101章，抑或只包括由[[摩訶那摩|摩訶那摩]]（Mahanama）所寫的第1章至第37章第50頌。

*主張僅包括[[摩訶那摩|摩訶那摩]]所撰的第1章至第37章第50頌：最先提出的是[[德國|德國]]學者[[威廉·蓋格爾|威廉·蓋格爾]]（Wilhelm Geiger），他的主要理由是在第37章第51頌前，出現了一句「向[[薄伽梵|薄伽梵]]、[[阿羅漢|阿羅漢]]、[[三藐三佛陀|三藐三佛陀]]致敬」。而這句句子，是一般[[巴利文|巴利文]]書籍中的開首語，因而從第37章第51頌開始，便應被視為另一部著作。[[威廉·蓋格爾|威廉·蓋格爾]]將第37章第51頌以後的部份列為《小史》。<ref name="《大史》譯者序" />[[斯里蘭卡|斯里蘭卡]]學者[[E·F·C·盧多維克|E·F·C·盧多維克]]更認為「《小史》，係《大史》的近代部份。」<ref>《[[錫蘭現代史|錫蘭現代史]]》，[[E·F·C·盧多維克|E·F·C·盧多維克]]著，[[四川大學|四川大學]]外語系翻譯組譯，[[四川|四川]][[人民出版社|人民出版社]]1980年版，第498頁。</ref>

*主張全部101章：[[斯里蘭卡|斯里蘭卡]]學者[[羅睺羅_(現代)|羅睺羅]]認為，從第1章至第101章，每章的末尾都注明《大史》，因而只能說這101章合起來是一部大書。另外，第37章第51頌之前的「向[[薄伽梵|薄伽梵]]、[[阿羅漢|阿羅漢]]、[[三藐三佛陀|三藐三佛陀]]致敬」這一句，只不過是補寫者所添加罷了，不能因此而將全書分為《大史》及《小史》。（據學者[[韓廷傑|韓廷傑]]所說，[[威廉·蓋格爾|威廉·蓋格爾]]後來亦承認自己「在《大史》的問題上犯了一些錯誤。」）<ref name="《大史》譯者序" />

== 編撰過程與內容 ==
《大史》從大約公元6世紀開始，到18世紀，由不同時期的作者續寫而成。據[[韓廷傑|韓廷傑]][https://web.archive.org/web/20071108045126/http://www.hkbuddhist.org/magazine/484/484_16.html 中譯本序言]裡的分析，全書101章，是分成五個階段寫成的，而內容亦大致可分成五個部份：

*第一部份：從第1章至第37章第50頌，作者是生活在大約公元5世紀的宗教長老[[摩訶那摩|摩訶那摩]] （Mahanama），根據《[[島史|島史]]》（Dipavamsa／Deepavamsa）等材料寫成，內容為公元前5世紀至公元4世紀的史事。
*第二部份：從第37章第51頌至第79章第84頌，作者是高僧[[達摩祇底|達摩祇底]]（Dhammakitti），內容是4世紀至12世紀的史事。
*第三部份：從第80章至第90章，作者不詳，內容是12世紀至14世紀的史事。
*第四部份：從第91章至第100章，作者是[[T·蘇曼伽羅|T·蘇曼伽羅]]（Tibbotuvare Sumangala）。內容是14世紀至1758年的史事。
*第五部份：第101章，作者是[[H·S·蘇曼伽羅|H·S·蘇曼伽羅]]（Hikkaduve Siri Sumangala），內容是[[英國|英國]]入侵[[斯里蘭卡|斯里蘭卡]]時的情況。<ref name="《大史》譯者序" />

== 史料價值 ==
*構建了較完整的[[斯里蘭卡歷史|斯里蘭卡歷史]]及為[[南亞|南亞]]史提供了資料：在[[南亞|南亞]]地區的多個國家中，古代的修史事業並不蓬勃，而由《大史》所帶起的續寫風氣，在一千多年間蔚為[[斯里蘭卡|斯里蘭卡]]的修史傳統。學者[[韓廷傑|韓廷傑]]更指出，「在[[斯里蘭卡|斯里蘭卡]]、[[泰國|泰國]]、[[緬甸|緬甸]]、[[老撾|老撾]]、[[柬埔寨|柬埔寨]]等一些信仰[[上座部佛教|上座部佛教]]的國家中，只有[[斯里蘭卡|斯里蘭卡]]有這樣一部完整的歷史」；以及它「涉及到其他的[[上座部佛教|上座部佛教]]國家」<ref name="《大史》譯者序" />，這就突顯了《大史》在[[南亞|南亞]]地區的史學地位。到了20世紀，[[斯里蘭卡|斯里蘭卡]]學者[[E·F·C·盧多維克|E·F·C·盧多維克]]編修自18世紀以來的《錫蘭現代史》時，亦參考了這部文獻。
*對[[上座部佛教|上座部佛教]]的研究提供重要史料：由於《大史》是以[[佛教|佛教]]歷史發展為基礎而寫成，因而它亦是重要的「研究[[上座部佛教|上座部佛教]]最主要的參考書」。<ref name="《大史》譯者序" />

== [[中文|中文]][[翻譯|翻譯]]版本的情況 ==
[[File:《大史》中譯本.jpg|thumb]]
《大史》一書，在20世紀上半期已被翻譯成[[德文|德文]]、[[英文|英文]]、[[俄文|俄文]]、[[日文|日文]]等版本。而[[中文|中文]]譯本方面，則有[[台北|台北]][[佛光文化事業有限公司|佛光文化事業有限公司]]出版的[[韓廷傑|韓廷傑]]譯本，這一版本包括了從6世紀至18世紀寫成的101章。此外，[[北京|北京]][[商務印書館|商務印書館]]所翻譯出版的《古印度[[吠陀時代|吠陀時代]]和列國時代史料選輯》，亦收錄了一部份《大史》的內容。
