'''大溪里'''，位於[[嘉義市|嘉義市]]西郊，隸屬於[[嘉義市|嘉義市]][[西區_(嘉義市)|西區]]行政區劃。北以北港路和下埤里、竹村里為界，東鄰北湖里、福安里、港坪里，南和頭港里為鄰，西接[[嘉義縣|嘉義縣]][[太保市|太保市]]埤腳村。<ref>[http://www.chiayi.gov.tw/web/west/new/left_03_04_06_07.asp 嘉義市西區區公所－大溪里] {{webarchive|url=https://web.archive.org/web/20160304200821/http://www.chiayi.gov.tw/web/west/new/left_03_04_06_07.asp |date=2016-03-04 }} </ref>

== 地理概況 ==
大溪里，俗稱大溪厝，位於現在的[[嘉義市|嘉義市]][[世賢路|世賢路]]西面、[[北港路|北港路]]南面區域。大溪厝為[[嘉義市|嘉義市]]的主要農業區，農作物以水稻為主。灌溉用水取自[[嘉義市|嘉義市]]河溝水（源自現在的嘉義特教學校北側水溝），經由大圳引流至各區農田。由於大溪厝地區未實施農地重劃，早期對外聯絡道路僅連接[[北港路|北港路]]一條，其餘為產業道路。公車及客運等大眾運輸工具未進入村內，形成較為封閉的聚落，亦為今日[[嘉義市|嘉義市]]保有較多自然風貌的地區。

== 鐵公路交通 ==
* [[糖鐵|糖鐵]]：1970年以前，可乘坐[[臺灣糖業鐵路|臺灣糖業鐵路]][[朴子線|朴子線]]五分車（火車站址約於今省道[[台18線|台18線]][[高鐵|高鐵]]大道[[嘉義公車捷運|嘉義BRT]]「[[大溪厝|大溪厝]]站」現址）。
* 公路：早期大溪厝的公車及客運招呼站皆設在北港路，距離庄內甚遠。旅外民眾返鄉，須在[[嘉義火車站|嘉義火車站]]搭乘[[計程車|計程車]]，交通極為不便。
* [[嘉義BRT|嘉義BRT]]：自[[省道|省道]][[台18線|台18線]][[高鐵|高鐵]]大道開通後，[[嘉義BRT|嘉義BRT]]在大溪厝設站，增進公路交通的便利，亦促進大溪厝的發展。（BRT採用全線[[低地板公車|低地板無障礙公車]][[File:標誌(小)mark(S).jpg|25px]]行駛）
* [[國光客運|國光客運]]:[[國光客運|國光客運]]在此地區設置大溪里站([[嘉義|嘉義]]端下客站)，其行經[[嘉義市|嘉義市]]的4條[[國道|國道]][[客運|客運]]路線皆有停靠供旅客下客，停靠路線如下:

== 文化景觀 ==
*生態景觀：大溪厝附近的田園水域為多種候鳥的棲息地，至今成為良好的賞鳥地點。
*[[古井|古井]]：大溪厝保有古井7口，建造年代超過300年，昔日為居民的生活水源。
*大圳池塘：大溪厝的農田皆為水田，各處溝渠水池較多。入庄首先經過大圳，民眾稱此地為「大橋頭」。「大橋頭」外面，就是庄外。昔日靠路邊的池塘有多處，庄頭古井邊一處，集會所彎道前小廟邊一處及庄尾一處，現在皆已填平蓋屋。
*[[賴姓|賴氏]][[宗祠|宗祠]]：頂公祠，位於大安街161號（原址為大溪厝94號），約建於1910年；下公祠，位於大溪路382號（集會所前，原址為大溪厝161號）。
*紫微宮：主祀[[玄天上帝|玄天上帝]]，配祀[[神農大帝|神農大帝]]、[[福德正神|福德正神]]，為里民的宗教信仰中心。創建於清道光年間，重建於2006年。<ref>鄧豐洲，台灣寺廟圖誌第1集，臺南市，2014年。</ref>

== 著名人物 == 
* [[鄧豐洲|鄧豐洲]]：為[[漢詩|漢詩]][[詩人|詩人]]、文史作家、[[仙道人士著作列表|仙道人士]]、環保人士、「善陽玄苑」創立者。[[鄧豐洲|鄧豐洲]]的「故鄉」詩，描述大溪里的農村風光與景觀演變：
圳水潺潺繞厝莊，農田廣闊大溪長；清風碧浪添春意，烈日泥場曬稻忙。<br>
古代房樓除舊貌，兒時景象換新裝；塵封往事重追憶，歲月悠悠已遠颺。<ref>鄧豐洲，鄧豐洲古典詩集，臺南縣，2010年。</ref><br>
