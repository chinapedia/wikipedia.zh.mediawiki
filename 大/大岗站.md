'''大岗站'''位于[[中華人民共和國|中國]][[江西省|江西省]][[抚州市|抚州市]][[臨川區|臨川區]][[大岗镇_(抚州市)|大岗鎮]]，是[[昌福线|昌福线]]上的一座[[五等站|五等站]]，由[[南昌铁路局|南昌铁路局]][[南昌车务段|南昌车务段]]管辖<ref>{{Cite web|url=http://www.chnrailway.com/news/20130205/0205435072.html|title=南昌车务段“两节”送温暖 慰问显真情|accessdate=|author=|date=2013-02-05|publisher=中华铁道网}}{{Dead link|date=2018年10月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref>{{Cite web|url=http://www.chnrailway.com/news2/gntl/2013/08/258941.shtml|title=南昌车务段全力以赴确保向莆线按时开通|accessdate=|author=|date=2013-08-16|publisher=中华铁道网}}</ref>。

== 历史 ==
1961年8月[[向乐线|向乐线]]开始建设，1963年9月[[江家站|江家]]到[[抚州北站|抚州北]]开始施工，年底铺轨至抚州北<ref name="江西省铁路志7-5035-3">{{cite book |title=《江西省铁路志》 |author=傅金土 |year=1994年12月 |edition=1994年12月第1次印刷 |publisher=中共中央党校出版社 |isbn=7-5035-3 |ignore-isbn-error=true |oclc=36212671}}</ref>{{rp|53}}。

1964年3月开始工程临时运营，1965年10月16日到26日[[南昌铁路局|南昌铁路局]]进行验收，11月全线办理临时运营，12月8日到24日办理交接，1966年1月1日正式交付运营<ref name="江西省铁路志7-5035-3"/>{{rp|53}}。

1995年为向乐线上的四、五等站{{efn|参考资料中将四等站和五等站放在一起标识}}，由[[上海铁路局|上海铁路局]][[南昌铁路分局|南昌铁路分局]]管辖<ref>{{cite book |title=《中国铁路交通地图集》 |author= |year=1995年1月 |edition=1995年1月河北第1次印刷 |publisher=中国铁道出版社 中国地图出版社 |isbn=7-113-01810-6 |oclc=48531409}}</ref>{{rp|100}}。

2003年为向乐线上的客、货运四等站，由南昌铁路局管辖，离[[向塘西站|向塘西]]37公里，距[[江边村站|江边村]]85公里<ref>{{cite book |title=《中国铁路站名词典》|author=孙本祥 编 |year=2003年4月 |edition=2003年4月第1次印刷 |publisher=中国铁道出版社 |isbn=7-113-03941-3 |oclc=53211800}}</ref>{{rp|382}}。

2004年为向乐线上的四、五等站，由南昌铁路局管辖<ref>{{cite book |title=《中华人民共和国铁路地图集》 |author= |year=2004年6月 |edition=2004年6月第2次印刷 |publisher=中国铁道出版社 |oclc=62999371 |id=统一书号17113·1219}}</ref>{{rp|77}}；2006年同<ref>{{cite book |title=《中华人民共和国铁路地图集》 |author= |year=2006年6月 |edition=2006年6月第3次印刷 |publisher=中国铁道出版社 |oclc=879963547 |id=统一书号17113·1214}}</ref>{{rp|40}}。

2007年11月24日[[向莆铁路|向莆铁路]]江西段开工，确认沿线将设大岗站<ref>{{Cite web|url=http://jiangxi.jxnews.com.cn/system/2007/11/23/002617848.shtml|title=向莆铁路（江西段）今日开工（组图）|accessdate=2016-09-24|author=|date=2007-11-24|work=jiangxi.jxnews.com.cn|publisher=大江网|archiveurl=https://web.archive.org/web/20150509120906/http://jiangxi.jxnews.com.cn/system/2007/11/23/002617848.shtml|archivedate=2015-05-09}}</ref>。

2008年为向乐线上的客运四、五等站<ref>{{cite book |title=《中华人民共和国铁路地图集》 |author= |year=2008年12月 |edition=2008年12月第4次印刷 |publisher=中国铁道出版社 |oclc=502099863 |id=统一书号17113·1308}}</ref>{{rp|40}}；2012年同<ref>{{cite book |title=《中国铁路地图集》 |author= |year=2012年2月1日 |edition=2012年7月第3次印刷 |publisher=中国铁道出版社 |isbn=978-7-113-13840-0 |oclc=818363258}}</ref>{{rp|40}}。

2010年8月为配合向莆铁路建设大岗站进行了闭站信号改造<ref>{{Cite web|url=http://www.peoplerail.com/rail/show-478-80047-1.html|title=向莆铁路大岗站闭站完成|accessdate=2016-10-01|author=|date=2010-08-09|work=www.peoplerail.com|publisher=}}</ref>。

2012年7月大岗站改造完成<ref>{{Cite web|url=http://www.cnenbd.com/Article/news_qyyw/Article_12333.html|title=向乐线将恢复营运并启用新抚州北站|accessdate=2016-10-01|author=|date=2012-07-27|work=www.cnenbd.com|publisher=|deadurl=yes|archiveurl=https://web.archive.org/web/20161001214201/http://www.cnenbd.com/Article/news_qyyw/Article_12333.html|archivedate=2016-10-01}}</ref>。

2013年5月6日向乐线为保证向莆铁路联调联试停运<ref>{{Cite web|url=http://jxfz.jxnews.com.cn/system/2013/04/27/012393307.shtml|title=5月5日南昌至江边村7205次列车停驶 5月6日向乐线停运|accessdate=2016-09-30|author=|date=2013-04-27|work=jxfz.jxnews.com.cn|publisher=}}</ref>，之后确认向乐线在向莆铁路通车后不会恢复客运服务因此大岗站也将停止客运服务<ref>{{Cite web|url=http://jiangxi.jxnews.com.cn/system/2013/08/30/012609406_02.shtml|title=鹿心社与网民在线交流 37个问题落实情况公布-鹿心社-江西新闻网|accessdate=2016-09-30|author=|date=2013-08-30|work=jiangxi.jxnews.com.cn|publisher=}}</ref>。

2015年为[[昌福线|昌福线]]（建设时称为向莆铁路的大部分路段）上的车站{{efn|参考资料中昌福线沿线车站均标志四、五等车站}}，由南昌铁路局管辖<ref>{{cite book |title=《中国铁路地图集》 |author= |year=2015年2月28日 |edition=2015年3月第4次印刷 |publisher=中国铁道出版社 |isbn=978-7-113-13840-0}}</ref>{{rp|40}}。

2016年南昌铁路局重新公布了管内的车站等级，本站为五等站<ref>{{cite journal |author=南昌铁路局办公室 |journal = 南昌铁路局文件 |date=2016-06-20 |issue= |title = 《南昌铁路局关于重新公布车站等级等事项的通知》 |id = 南铁劳卫〔2016〕226号 |url=http://www.aiweibang.com/yuedu/126493427.html |page = }}</ref>。

== 车站结构 ==
改造后的车站有四条股道其中两条为昌福线正线，线位东北方向设有站房。
