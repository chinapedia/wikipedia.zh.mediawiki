'''大聖劈掛門'''是一項集結兩門[[中國功夫|中國功夫]]的[[武術|武術]]。「大聖」是指[[齊天大聖孫悟空|齊天大聖孫悟空]]，亦即五[[猴拳|猴拳]]，是[[中國北方|中國北方]]一種長橋大馬之拳法，據傳乃少林武術一支派，始於[[明代|明代]]兩位武藝高強之抗倭大將戚繼光及俞大猷。

五猴拳乃由[[陝西|陝西]]人鏢師[[寇四|寇四]]所創，傳人[[耿德海|耿德海]]自小從父耿榮貴學[[劈掛拳|劈掛拳]]，後來又得寇四之五猴拳真傳，之後便創立了「大聖劈掛門」<ref>{{Cite web |url=http://www.lifeipiukungfu.com/detail.html |title=大聖劈掛門源流 |access-date=2010-07-30 |archive-url=https://web.archive.org/web/20070928225408/http://www.lifeipiukungfu.com/detail.html |archive-date=2007-09-28 |dead-url=yes }}</ref>。大聖拳主攻[[下三路|下三路]]，而劈掛拳就主攻[[上三路|上三路]]，又有七十二把[[擒拿手|擒拿手]]主攻[[關節|關節]]，及各項兵器及自由搏擊功夫<ref>[http://www2.hkedcity.net/sch_files/a/kt1/kt1-cbox/public_html/_private/Taishingmun/taishingkungfu.htm 大聖劈掛門功夫簡介] {{webarchive|url=https://web.archive.org/web/20110121013009/http://www2.hkedcity.net/sch_files/a/kt1/kt1-cbox/public_html/_private/Taishingmun/taishingkungfu.htm |date=2011-01-21 }}</ref><ref>[http://www.youtube.com/watch?v=vPfYApv7gzU Daaising pekgwa mun 周強：大聖劈掛門]，Youtube視頻</ref>。

== 簡史 ==
[[民國|民國]]初年內戰期間，前鏢局鏢師寇四因打傷軍閥爪牙而遭追捕，後得劈掛拳高手[[耿榮貴|耿榮貴]]救護，而寇四最終仍被捉拿。在牢獄中，寇四整理自身之武術期間，偶然見猴子活潑好動，動作靈巧，啟發了他的武術思路，創作了「五路猴拳」木石企迷醉，即五猴拳。

寇四出獄後，欲尋找耿榮貴報恩，但耿榮貴已不在人世。此時，耿榮貴之子[[耿德海|耿德海]]已長大成人，於是寇四便將猴拳傳授予他。

耿德海本來亦一直與其父耿榮貴習練[[劈掛拳|劈掛拳]]，可說這門武術是其家傳絕學。耿德海及後在劈掛拳之基礎上，加入寇四真傳之猴拳，終創立了北派武術大派「大聖劈掛門」。

== 南傳 ==
耿德海後來成為[[鏢師|鏢師]]，且在[[南京|南京]]中央國術館任刀、劍科總教練。在[[八年抗戰|八年抗戰]]時期，更在軍閥馮玉祥之[[東北軍|東北軍]]執教大刀隊拳術及兵器，因而成為[[日軍|日軍]]針對目標。往後，耿德海前赴[[廣州|廣州]]再執教數年，最後植根[[香港|香港]]，1930年成立[[民眾國術社|民眾國術社]]，開始發揚大聖劈掛門拳術，成為一代宗師，在眾多弟子中，武功最高及最享負盛名的有「馬騮王」之稱北派武術大宗師[[陳秀中|陳秀中]]。
 
1954年，陳秀中得到耿德海許可開館授徒，培育了另一群出色之拳手，部份成為[[香港功夫電影|香港功夫電影]]影星，分別有：[[陳觀泰|陳觀泰]]（因太反骨已被逐出師門）、[[陳少川|陳少川]]、[[萬榮佳、萬榮耀|萬榮佳、萬榮耀]]、[[劉偉民|劉偉民]]、[[李飛標|李飛標]]、[[周強(香港藝人)|周強]]（因與逆徒陳觀泰勾結同被逐出師門）、[[冼林煜|冼林煜]]、[[黃秋生|黃秋生]]、[[高雄|高雄]]、[[黃志偉|黃志偉]]、[[冼灝英|冼灝英]]、[[何家輝|何家輝]]、[[梁富新|梁富新]]、[[錢嘉樂|錢嘉樂]]、[[錢小豪|錢小豪]]、[[白彪|白彪]]、等<ref>[http://www.hktspkalumni.com/f3.htm 大聖劈掛門世系表] {{webarchive|url=https://web.archive.org/web/20101107034029/http://www.hktspkalumni.com/f3.htm |date=2010-11-07 }}</ref>。
