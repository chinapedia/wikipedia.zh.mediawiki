《'''大姊當家'''》（[[日語|日語]]：{{lang|ja|とと姉ちゃん}}），為[[NHK|NHK]]於2016年度上半年推出的第94部[[晨間小說連續劇|晨間小說連續劇]]。由[[高畑充希|高畑充希]]主演<ref name="nikkan">[http://www.nikkansports.com/entertainment/news/1527494.html 高畑充希が朝ドラヒロイン　来春「とと姉ちゃん」]</ref>。

== 企畫製作 ==
以出生於[[靜岡縣|靜岡縣]][[遠江|遠州]]的小橋常子為女主角，敘述常子在父親過世後，以女兒的身分代父職，守護著母親及妹妹，在[[第二次世界大戰|第二次世界大戰]]前後動盪的昭和時代，為了生存而努力不懈的三姊妹之家族紀事，並描述其後在東京出版女性綜合雜誌的出版社中，與天才編輯花山伊佐次相遇，進而合作出版雜誌《你的生活》（あなたの暮し）的經歷<ref name="topics2015626">{{Cite web |date=2015-06-26 |url=http://www.nhk.or.jp/dramatopics-blog/1000/221702.html |title=平成28年度前期 連続テレビ小説「とと姉ちゃん」制作のお知らせ |work=NHKドラマトピックス |publisher=日本放送協会 |accessdate=2015-06-27}}</ref><ref name="sponichi2015626">{{Cite news |title=来春朝ドラは「とと姉ちゃん」 「暮しの手帖」創業者モチーフ |newspaper=[[體育日本|體育日本]] |date=2015-06-27 |url=http://www.sponichi.co.jp/entertainment/news/2015/06/27/kiji/K20150627010616640.html |accessdate=2015-06-29}}</ref>。故事主要以靜岡縣及東京都江東區深川為主要場景<ref name="sanspo2015626">{{Cite web |date=2015-06-26 |url=http://www.sanspo.com/geino/news/20150626/geo15062617120029-n1.html |title=来年4月スタートNHK連続テレビ小説は「とと姉ちゃん」 |publisher=sanspo.com |accessdate=2015-06-27}}</ref>。

本劇改編自出版生活綜合雜誌《[[生活手帖|生活手帖]]》的出版社「生活手帖社」（暮しの手帖社）<ref group="注">1946年花森安治與妹妹芳子共同成立，最初的公司名稱為「衣裝研究所」，1948年與大橋鎭子創辦「美麗的生活手帖」雜誌。1951年公司改名為「生活手帖社」、1953年雜誌改名為「生活手帖」，皆沿用至今。</ref>創辦人大橋鎭子（1920年3月10日－2013年3月23日）與花森安治（1911年10月25日－1978年1月14日）的真人真事，以「活在當下是最重要的事」（当たり前の暮らしを大切にすること）與「戰後女性所期望的正常生活」（戦後の女性の暮らしの復興）為故事主軸，是一部以正面態度看待生活上遇到的困境，並對未來充滿希望的人生故事<ref name="oricon2015626">{{Cite web |date=2015-06-26 |url=http://www.oricon.co.jp/news/2054968/full/ |title=NHK来春朝ドラ『とと姉ちゃん』に決定 “中身はオヤジ”のヒロイン |work=ORICON STYLE |publisher=[[オリコン|オリコン]] |accessdate=2015-06-27}}</ref>。由西田征史擔任編劇<ref name="oricon2015626" />。開拍時間預定為2015年秋季<ref name="topics2015626" />。7月舉行女主角徵選<ref name="sanspo2015626" />，8月24日公布人選，由23歲的[[高畑充希|高畑充希]]擔任女主角<ref name="nikkan"/>。8月31日，NHK召開記者會，正式公布由高畑充希主演。開拍時間為2015年秋季<ref name="topics2015626" />。11月6日召開記者會，正式介紹參與演出的其他演員<ref>{{Cite news|url= http://www.nikkansports.com/entertainment/news/1562580.html |title= 川栄李奈、朝ドラに初出演「ファブリーズの子供に」 |newspaper= 日刊スポーツ |publisher= 日刊スポーツ新聞社 |date= 2015-11-06 |accessdate= 2015-11-06 }}</ref>。2016年1月20日公布主題曲，由[[宇多田光|宇多田光]]演唱，歌曲名為「[[將花束獻給你|將花束獻給你]]」（花束を君に）。本劇時代設定為1930年（昭和5年）至1988年（昭和63年）。

== 劇情概要 == 
時間是1958年的東京。戰後由三姐妹共同創辦的生活綜合雜誌「你的生活」編輯部充滿了活力，編輯人員忙進忙出，為了讓新一集的雜誌內容更加充實，所有人無不費盡心思。此時一位編輯人員告訴總編輯'''小橋常子'''（[[高畑充希|高畑充希]] 飾），某份稿子發稿時弄錯了題目，必須請寫稿的作家重新寫過，但這位作家素以難搞出了名，大家都傷透了腦筋。為了不讓新出刊的雜誌開天窗，常子決定親自拜訪這位作家，說著就飛也似的離開了出版社。這是幼年時即失去父親（[[西島秀俊|西島秀俊]] 飾），自身兼代父職守護母親（[[木村多江|木村多江]] 飾）及兩個妹妹（[[相樂樹|相樂樹]] 、[[杉咲花|杉咲花]] 飾）的樂觀女性的奮鬥故事…
