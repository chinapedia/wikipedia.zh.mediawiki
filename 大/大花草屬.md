'''大花草屬'''（[[学名|学名]]：''{{lang|la|Rafflesia}}''）又名'''大王花属'''、'''莱佛士花属'''、'''屍花属'''，是世界上最大的一種[[花|花]]。生長于[[马来半岛|马来半岛]]及[[婆羅洲|婆羅洲]]、[[蘇門答臘|蘇門答臘]]等[[島嶼|島嶼]]。大花草無[[根|根]]無[[莖|莖]]，是一種[[寄生植物|寄生植物]]，靠吸收[[葡萄科|葡萄科]]植物的養分為生。其直徑可達1.4[[米|米]]之長。大花草開花時奇臭無比，發出腐肉味的[[臭氣|臭氣]]，靠吸引廁蠅與[[甲蟲|甲蟲]]為其[[傳粉|傳粉]]。

其中最大的一种大花草为[[阿諾爾特大花草|阿諾爾特大花草]]，其花直径能够达到3尺之長。

== 发现 ==
1818年，[[自然历史#博物学家|自然学家]][[约瑟夫·阿诺德|约瑟夫·阿诺德]]（{{lang|en|Joseph Arnold}}）和[[英国|英国]]在[[新加坡|新加坡]]殖民地的创建者[[萊佛士|托马斯·斯坦福德·莱佛士]]（{{lang|en|Sir Thomas Stamford Bingley Raffles}}）率领的科学探险队在[[印尼|印尼]][[苏门答腊|苏门答腊]]发现了这种植物，并以莱佛士的名字命名。其带有斑点的血红色花朵有7公斤重，直径一般1米以上，散发着一种腐烂尸体般的恶臭；{{fact|这种花朵竟然会散发一定的热量并借以伪装成带有余温的动物尸体}}，吸引[[苍蝇|苍蝇]]和几种[[食腐蜂|食腐蜂]]类来传播花粉。

== 生态 ==
作为植物，大花草并不能进行[[光合作用|光合作用]]，而是靠着寄生在[[葡萄科|葡萄科]][[崖爬藤属|崖爬藤属]]一类的热带[[藤本植物|藤类]]上吸收养分存活。大花草没有根、茎、叶等一般植物的部分，只有几片花瓣；从长出花苞到开花需9个月的时间，花期为3天到1周左右。

刚开花时大花草会发出一种淡淡的香味，而随后会变成一种腐肉般的恶臭，并藉由这种味道吸引苍蝇等食腐昆虫替自己传播花粉。

大王花開花期過後，會崩塌成一團類似燒焦的黑色黏稠物質。

== 起源 ==
生物学家戴维斯说，大花草属植物的起源要追溯到约1亿年前的白垩纪，即恐龙时代末期。一般认为，開花植物从那时候才开始在地球上出现。研究人员推断，大花草的花朵在4600万年间增大了约79倍，然后才进入缓慢的进化阶段。
