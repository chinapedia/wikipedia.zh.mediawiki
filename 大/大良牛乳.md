'''大良牛乳'''，又叫'''金榜牛乳'''，是一种原产于[[顺德区|顺德]][[大良街道|大良]]金榜村（现金榜居委会）的白色圆状薄片[[牛奶|牛奶]]制成品，[[顺德美食|顺德美食]]之一，有“'''东方-{芝士}-'''”称号<ref name="sdlib">{{cite web|url=http://www.sdlib.com.cn/news_view.asp?newsid=3645|title=走进顺德·牛乳饼·双皮奶·姜撞奶|work=《顺德文丛·美味顺德》|date=2011-02-16|accessdate=2011-04-08}}{{dead link}}</ref><ref name="chenyizhong">{{cite web|url=http://epaper.sc168.com.cn:888/disnews.asp?Fid=9275&FlayoutId=2229|title=像炮制凤城美食那样演绎美食传奇|author=陈毅忠|publisher=珠江商报|date=2010-12-20|accessdate=2011-04-08}}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。因其圆片形状，故也被称为“'''金榜牛乳饼'''”<ref name="sdlib" />。大良牛乳始于明代，味道略[[咸|咸]]而甘甜可口，可佐[[粥|粥]]下[[饭|饭]]。除大良外，东南亚多个华人聚居地也有生产<ref>{{cite web|url=http://www.southcn.com/travel/guangdong/shunde/zhinan/200211040774.htm|title=顺德美食|author=|work=南方网旅游频道|date=2002-11-04 |accessdate=2011-04-08}}</ref>。

== 历史 ==
根据顺德作家协会资深作家关耀权的一篇文章《东方芝士牛乳饼》，大良牛乳起源于[[明代|明代]]<ref name="chenyizhong" />：
{{Quotation|在明朝年间，有一名来自[[意大利|意大利]]的[[天主教|天主教]][[传教士|传教士]]经[[澳门|澳门]]路过大良，其间救治了一名金榜人，传教士获邀在金榜小住，他发现当地人往往把卖不完的牛奶倒掉，觉得这样很可惜，于是就用他们卖剩的牛奶做成自己喜欢吃的[[乾酪|-{芝士}-]]，金榜人觉得这样很好，可惜[[乾酪|-{芝士}-]]不合本地人的口味和习惯，于是加以改良，制成更富营养价值，易于入口和消化的大良金榜牛乳。}}

== 制作方法 ==
當地人因耕地不足而养水牛获取[[水牛奶|水牛奶]]。但因水牛奶有时供过于求而无法存放，于是村民将水牛奶混入食[[醋|醋]]和食鹽，使牛奶[[蛋白|蛋白]]凝固，後再用模壓製成比較脆的圓片狀，再與鹽水一起入樽出售。

原料为新鲜[[水牛奶|水牛奶]]、[[白醋|白醋]]、没有经过加工的[[海盐|海盐]]制作的高浓度盐水。工具为[[茶杯|茶杯]]、小勺以及成型模具。将水牛奶加热至微温，白醋加热至微热。在茶杯里倒入1/3杯白醋，用勺子将一勺牛奶倒入杯中，然后将凝固成团状的牛奶倒入模具成型并将牛奶里的醋压挤出来。成型的牛奶片放入盐水中，泡四个小时即成。<ref>{{cite web|url=http://gb.chinareviewnews.com/crn-webapp/cbspub/secDetail.jsp?bookid=16556&secid=30790|title=大良金榜牛乳的制作方法|work=《顺德简史读本》（第二节 明清以来的经济）|author=梁利龙|accessdate=2011-04-08|archive-url=https://web.archive.org/web/20181209213053/http://gb.chinareviewnews.com/crn-webapp/cbspub/secDetail.jsp?bookid=16556&secid=30790|archive-date=2018-12-09|dead-url=yes}}</ref>

制作金榜牛乳关键是掌握牛奶和醋的温度，以及使用海盐。

== 食用方法 ==
大良牛乳的味道鹹中帶微甜味，味道类似[[奶酪|奶酪]]但有所区别。大多数顺德人通常将其與[[廣東|廣東]][[粥|粥]]混合一起吃，也有人将其冲热水喝。还有人用其炒鸡蛋。
