'''大阿爾克那'''（'''Major Arcana'''）是指[[塔羅牌|塔羅牌]]的組合，在[[塔羅牌|塔羅牌]]中一般俗稱為"大秘儀"、"大牌"、"王牌"(trump)，共包含22張牌，編號由0 - 21。
阿爾克那（Arcana）為[[中古|中古]][[世紀|世紀]][[煉金術師|煉金術師]]所追尋的奧秘，天地萬物運行的道理。0為愚者超然於其它21張塔羅牌之外，部分[[占卜|占卜師]]認為愚者早已具備空白牌的功能，此派認為再加入空白牌的行為是不智的、多餘的、破壞平衡的行為。

[[塔羅牌|馬賽塔羅牌]]一般會以VIII為[[正義|正義]]，XI為[[力量|力量]]。只是[[塔羅牌|偉特塔羅牌]]基於西方[[神秘學|神秘學]]中的[[占星術|占星術]]原則把兩張牌掉轉，因為[[正義|正義]]所代表的是[[天秤座|天秤座]]而[[力量|力量]]所代表的是[[獅子座|獅子座]]，明顯地獅子座應放在天秤座前的，在更改順序後的塔羅牌被認為更符合星相週期發展，及後很多新出現的[[塔羅牌|塔羅牌]]亦跟隨這種排列。但馬賽牌或托特牌則認為塔羅牌本身也沒有順著這種星座週期，如排列於兩牌之間的[[命運之輪|命運之輪]]所代表的就是[[木星|木星]]，所以即使不是跟從星座順序也不是什麼問題。與[[小阿爾克那|小阿爾克那]]的應用及相對關係，可對應世界一切的現象。

== 編號與意義 ==
編號和名稱各具代表不同的意義，最通用的塔羅牌'''偉特塔羅牌'''中的大牌次序如下：

　 0. Ο（最初的虛無）- [[愚者_(塔羅牌)|愚者]]（Le Fou，狡猾之者）- 天王星
# I（萬物的開端）- [[魔術師_(塔羅牌)|魔術師]]（Le Bateleur，出發、發現、創造）- 水星
# II（陰陽）- [[女祭司_(塔羅牌)|女祭司]]（La Papesse，女性的神秘）- 月球
# III（愛、生命、和平）- [[皇后_(塔羅牌)|皇后]]（L'Impératrice，包容、豐收、豐腴）- 金星
# IV（[[四元素說|四大元素]]）- [[皇帝_(塔羅牌)|皇帝]]（L'Empereur，自我的權利）- 白羊座
# V（平均的力量）- [[教皇_(塔羅牌)|教皇]]（Le Pape，知識與道德）- 金牛座
# VI（天地的調和）- [[戀人_(塔羅牌)|戀人]]（L'Amant，上天的祝福）- 雙子座
# VII - [[戰車_(塔羅牌)|戰車]]（Le Chariot，熱情）- 巨蟹座
# VIII（力量的穩定）- [[力量_(塔羅牌)|力量]]（La Force，支配）- 獅子座
# IX（調和大三角）- [[隱者_(塔羅牌)|隱者]]（L'Ermite，知識）- 處女座
# X（宇宙的基礎）- [[命運之輪|命運之輪]]（La Roue de la fortune，天使與魔鬼）- 木星
# XI（不完整）- [[正義_(塔羅牌)|正義]]（La Justice，公平的真理）- 天秤座
# XII（終點）- [[倒吊人_(塔羅牌)|吊人]]（Le Pendu，自我犧牲）- 海王星
# XIII（Le nombre de la mort）- [[死神_(塔羅牌)|死神]]（La Mort，循環）- 天蠍座
# XIV（適當）- [[節制_(塔羅牌)|節制]]（Tempérance，自我控制）- 射手座
# XV（野性）- [[惡魔_(塔羅牌)|惡魔]]（Le Diable，慾望）- 魔羯座
# XVI（惡意的基盤）- [[塔_(塔羅牌)|塔]]（La Maison Dieu，驕傲者必亡）- 火星
# XVII（治癒）- [[星_(塔羅牌)|星星]]（L'Étoile，魂魄飛向宇宙）- 水瓶座
# XVIII（不安、淨化）- [[月_(塔羅牌)|月亮]]（La Lune，黑暗的降臨）- 雙魚座
# XIX（新生命誕生）- [[太陽_(塔羅牌)|太陽]]（Le Soleil，崇拜）- 太陽
# XX（極限）- [[審判_(塔羅牌)|審判]]（Le Juge，復活）- 冥王星
# XXI（最終的虛無）- [[世界_(塔羅牌)|世界]]（Le Monde，完美無缺）- 土星
