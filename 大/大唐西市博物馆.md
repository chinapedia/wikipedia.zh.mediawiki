'''大唐西市博物馆'''，位于[[中国|中国]][[陕西省|陕西省]][[西安市|西安市]][[莲湖区|莲湖区]]劳动南路118号，是由陕西佳鑫集团投资设立的民营博物馆，是西安“[[唐皇城复兴计划|唐皇城复兴计划]]”{{NoteTag|即2005年以来西安市以复兴唐风建筑、重现盛唐风貌为目的的城市建设活动。}}[[大唐西市|大唐西市]]文化产业项目的组成部分<ref>{{Cite web |url= http://district.ce.cn/zg/201003/09/t20100309_21085216.shtml |title= 全国首家遗址类民办大唐西市博物馆即将开馆 |authors = 夏先清、庞博 |date= 2010年3月9日 |publisher= 中国经济网 |language= zh-hans |accessdate= 2013年4月26日 |quote=  }}</ref>。

==历史==
2001年，中共西安市委、西安市人民政府决定由民营企业西安大唐西市文化产业投资有限公司独立开发建设“西市遗址恢复改造项目”。在改造项目实施中，发现了唐长安城西市东北“十字街”遗址，西安大唐西市文化产业投资有限公司随后主动变更了规划，邀请[[中国社会科学院考古研究所|中国社会科学院考古研究所]]的[[安家瑶|安家瑶]]带队开展西市遗址考古发掘。三个多月后，发掘出土唐长安城西市东北“十字街”遗迹以及唐井、石函、作坊、车辙等遗迹。此后，西安大唐西市文化产业投资有限公司按“原样封存、原地保护、原物展示”的方针，投资3.2亿元人民币，建设了建筑面积3.5万平方米具备遗址保护和展示功能的大唐西市博物馆<ref name=sikao/>。

2010年4月7日建成对外开放<ref name=pp/>。

2010年10月，以大唐西市博物馆为代表的“民营资本投资模式”成为陕西省大遗址保护五大实践模式之一。

2010年12月，大唐西市被[[中华人民共和国文化部|中华人民共和国文化部]]命名为第四批国家文化产业示范基地<ref name=sikao>{{Cite web |url =http://www.zgjrzk.com/news/201510/345833.html  |title = 大唐西市董事长吕建中对文化产业和谐发展的思考 |author =  |date =2015-10-15  |publisher = 中国金融智库 |language =  |accessdate =  |quote =  }}</ref>。

2017年入选中国第三批国家一级博物馆，成为中国第一家被授予“国家一级博物馆”的民办博物馆。2017年5月19日晚，大唐西市博物馆举行“国家一级博物馆”挂牌仪式<ref name=pp>{{Cite web |url =http://www.thepaper.cn/newsDetail_forward_1690229  |title =陕西“大唐西市”晋升一级博物馆，系首家获此称号的民办馆  |author =  |date = 2017-05-21 |publisher =  澎湃新闻|language =  |accessdate =  |quote =  }}</ref>。

==建筑==
该馆建于[[唐长安城|唐长安城]]西市遗址之上，由西安建筑科技大学教授刘克成设计<ref>{{Cite web|url=http://www.sn.xinhuanet.com/2007-08/24/content_10956221.htm|title=争议中的“大唐西市”|author=丁静、都红刚|date=2007-08-24|publisher=新华网陕西频道|language=zh-hans|accessdate=2014年7月26日|quote=|deadurl=yes|archiveurl=https://web.archive.org/web/20140728055753/http://www.sn.xinhuanet.com/2007-08/24/content_10956221.htm|archivedate=2014年7月28日}}</ref>，占地面积20亩（含2500平方米的遗址保护面积），建筑面积3.5万平方米，地上四层、地下两层，采用框架剪力墙结构和钢结构，展陈面积1.1万平方米，以[[丝绸之路|丝绸之路]]文化和商业文化为展示主题，分基本陈列、专题展览、临时展览、特别展览和艺术空间五个部分。馆藏文物两万余件，以西市遗址出土文物和博物馆创办者吕建中所收藏的文物为主。<ref>{{Cite web|url=http://www.dtxsmuseum.com/gaikuang/about.aspx?bigid=1&classid=3|title=西安大唐西市博物馆简介|author=|date=|publisher=西安大唐西市博物馆|language=zh-hans|accessdate=2014年7月26日|quote=|deadurl=yes|archiveurl=https://web.archive.org/web/20140726185846/http://www.dtxsmuseum.com/gaikuang/about.aspx?bigid=1&classid=3|archivedate=2014年7月26日}}</ref>

== 陈列 ==
一楼：西市遗址展示区，包括东北十字街遗址{{NoteTag|由于唐长安城西市遗址距地表较浅，基本上已遭到破坏，东北十字街遗址及南大街沟渠遗址为其中保存最好的部分。}}、道路车辙遗址、石板桥遗址、店铺房基遗址以及不同时期的排水渠遗址等<ref>{{Cite web |url= http://news.163.com/10/0428/10/65BMUL83000146BD.html |title= 大唐西市博物馆，残迹中看长安盛景 |author= 徐小青 |date= 2010-04-28 |publisher= 网易 |language= zh-hans |  accessdate= 2013年4月26日 |quote=  }}</ref>。另设《江文湛捐赠作品陈列》和尾厅展览《今日西市》。

二楼：基本陈列《丝路起点 盛世商魂》，从西市遗址、西市概况、西市行肆、文化交融、商业理念五个方面展示了大唐西市作为公元[[7世纪|7世纪]]国际贸易中心的盛况。<ref name="xinhua">{{Cite web |url = http://news.xinhuanet.com/society/2010-03/10/content_13141027.htm |title= 西安大唐西市博物馆开馆 |author= 冯国  |date= 2010年3月10日 |publisher= 新华网 |language= zh-hans |accessdate= 2013年4月26日 |quote=  }}</ref>

三楼：专题陈列《百工体验》，以店铺为基本形式，通过10种传统手工艺的示范对丝路文化进行了演绎和补充<ref name="xinhua" />。另设临时展览和多媒体影视厅。

四楼：特别展览《精品选粹》，大部分为博物馆创办者吕建中的个人收藏，分为青铜器、金银玉器和钱币陶瓷三个部分。
