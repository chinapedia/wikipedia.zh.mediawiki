《'''大逃殺'''》（{{lang-ja|'''バトル・ロワイアル'''}}，{{lang-en|'''Battle Royale'''}}；簡稱'''BR'''）是[[日本|日本]][[小說家|小說家]][[高見廣春|高見廣春]]所撰寫的[[恐怖小說|恐怖小說]]作品。本作品亦曾陸續被改編為同名[[大逃殺_(電影)|電影]]、[[漫畫|漫畫]]、[[網頁遊戲|網頁遊戲]]及[[紙牌遊戲|紙牌遊戲]]、[[週邊商品|週邊商品]]等等。

== 作品概要 ==
書名「Battle Royale」指的是在美國[[摔角|摔角]]界中的[[世界摔角娛樂|WWE（世界職業摔角）]]多人對戰，在這種比賽模式中，將會有不定數的選手參戰，直到最後一名沒被擊倒的選手，即為比賽中的冠軍。作者藉此暗示書中的42名學生為了生存，必須戰鬥到只剩自己一人的心情。（小說故事中，從頭到尾都沒有出現「BR」這個詞彙，只有這項遊戲的名稱「計畫」。）

作者於1996年開始寫作本書，並於1999年參加[[角川書店|角川書店]]所舉辦的恐怖小說比賽。由於所述寫的題材過於驚悚，也太不合乎道德標準，引起了評審們的批判、認為「讀了以後很不愉快」，所以沒有給予這本小說首獎。本作在1999年4月由[[太田出版|太田出版]]在日本出版。

2000年，本作同時改編為電影和漫畫，由日本導演[[深作欣二|深作欣二]]改編成電影《[[大逃殺_(電影)|大逃殺]]》、由[[日本漫畫家|日本漫畫家]]{{link-ja|田口雅之}}改編成漫畫《生存遊戲》<ref>2003年，[[長鴻出版社|長鴻出版社]]發行的《生存遊戲》中文漫畫版在[[臺北市議會|臺北市議會]]的壓迫下，被提出查禁，也正式從[[臺灣|臺灣]]市場內退出，總共出版至第10集。之後到第15集結束，僅能從其他非正式發行管道中找尋。</ref>。另外，電影《大逃殺》也因主題被視為怪誕，使得本片劇組成員被要求至[[日本國會|日本國會]]作證說明，這也開了演藝人員需到國會作證的先例，身為劇中一角的[[北野武|北野武]]也對[[日本政府|日本政府]]要求不得禁演。

2005年[[台灣|台灣]][[繁體中文|繁體中文]]版由[[木馬文化|木馬文化]]出版。

== 故事簡介 ==
從全國共4萬3千班中學三年級學生當中，隨機抽選1班，被抽選中的班級參加1場互相殘殺的遊戲，直到只剩下1人生還為止；否則，全部格殺勿論…被抽選中的班級，會被送到隨機挑選出來的地點進行遊戲。

*每人獲發[[糧食|糧食]]、[[水|水]]、[[地圖|地圖]]、[[指南針|指南針]]、[[手電筒|手電筒]]以及一款防身武器
*每人強制佩帶特殊設計頸環，防水防震防火、可作探測行蹤和[[竊聽|竊聽]]之用，強行破壞頸環會爆炸
*遊戲場所當中某些地區由電腦[[隨機|隨機]]挑選列為[[禁區|禁區]]，闖入者會被啟動頸環爆炸致命
*禁區會隨時間（每1小時或2小時）逐漸增加
*每6小時會有[[廣播|廣播]]，分別播報已死亡人員名單及公佈接下來新增的禁區位置
*遊戲場所會被[[士兵|士兵]]嚴密包圍，沒有逃走機會
*若因生存者抵抗互相不殘殺導致24小時之內無人死亡或者三日後生還者超過一人，剩餘生還者頸環會爆炸

== 遊戲規則 ==
'''大東亞共和國專守防衛陸軍戰鬥實驗第六十八項計畫大綱：'''

* 計畫簡介：
:: 此實驗為六十八項計畫。每年任意自全國中學三年級生隨機挑選對象班級實施。
:: 對象班級的學生彼此戰鬥、械鬥，其中殺人罪、傷害罪、威脅恐嚇等罪均不會予以判刑。
:: 若實驗進行順利，並且剩下最後一人生還，將統計過程中的各項數據，調查其所花費的時間等資料，做為政府官方參考備檔。

* 實施地點：
:: 均在抽中班級所在的城市附近。多數方圓六公里的孤島，但也曾經在用鐵網圍起來的山區或是都市內的一小塊區域

* 基本配備：
:: 高科技定位項圈，防水耐撞，遭破壞立即「爆炸」
:: 隨機發放高檔武器
:: 基本三天份飲水、三天份食物、遊戲地圖、指南針、背包

* 實驗規則：
:: 不限手段、規則殺人
:: 僅能有一位生存者
:: 活動範圍逐日減少
:: 若連續二十四小時無人死亡，則處死所有生存者（引爆所有學生的項圈）

*優勝者
:: 將獲得生存權，並護送回國。
:: 總統的親筆簽名。
