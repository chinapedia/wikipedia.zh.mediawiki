{{nihongo|《'''大提琴手高修'''》|セロ弾きのゴーシュ}}為日本動畫導演[[高畑勳|高畑勳]]改編自作家[[宮澤賢治|宮澤賢治]]創作的同名童話《[[大提琴手高修|大提琴手高修]]》，於1982年發表的劇院動畫。

== 劇情簡介 ==
動畫版劇情基於原作、同樣描寫著一名演奏方法不靈光的大提琴手──高修，在被樂團長指責問題後、每晚回家練習時被各種不同小動物闖入家中打擾。

而高修也在與小動物的互動中，慢慢地影響了自己演奏的能力。

== 主要角色 ==
{{nihongo|'''高修'''|ゴーシュ}}
:隸屬金星音樂團的大提琴手，有著不夠純熟的演奏技巧。
:在被樂團長斥責後而每夜通宵達旦地苦練，但常在練習中被其牠小動物進門打擾。

{{nihongo|'''貓咪'''|ねこ}}
:最先造訪高修家的動物，為一頭[[三毛貓|三毛貓]]。
:擅自將高修田裡種的蕃茄採下、厚臉皮地當成是送給高修的禮物，希望對方能因此為牠演奏[[舒曼|舒曼]]的曲子。

{{nihongo|'''布穀鳥'''|かっこう}}
:一隻了解音階高低不同音樂知識的鳥，到高修家想與對方互相切磋音樂。

{{nihongo|'''小狸子'''|子だぬき}}
:攜帶著樂譜「快樂馬車伕」、來與高修合奏的小狸。

{{nihongo|'''野鼠母子'''|野鼠の親子}}
:帶著生病中的兒子一同前來高修家的母野鼠，希望高修能藉由演奏樂曲來醫治自己兒子的病情。

== 製作 ==
=== 概要 ===
《大提琴手高修》的動畫改編提案最初為動畫工作室OH!Production裡候補的企劃之一，之後由高畑勳接手耗費5年多的時間才完成。<ref>{{cite book |title={{lang|ja|映画を作りながら考えたこと　1955～1991}} |year=1991 |publisher=[[德間書店|德間書店]] |author=[[高畑勳|高畑勳]] |isbn=978-4-19-554639-0|page=183頁}}</ref>因作品中牽扯樂手在演奏樂器的橋段，擔任角色設計的才田俊次也為了能讓動畫人物能有精準的表現而購買了大提琴的教學手冊作為參考。<ref>{{cite book |title={{lang|ja|映画を作りながら考えたこと　1955～1991}} |year=1991 |publisher=德間書店 |author=高畑勳 |isbn=978-4-19-554639-0|page=188頁}}</ref>

原作中金星音樂團演奏的樂曲雖有提到其名稱為「第六交響曲」，但並未特別指明是出於何人的創作。之後高畑勳在觀察原作中人與自然動物之間互動的風格、以及作者宮澤賢治的個人喜好後，推定此第六交響曲必定為[[貝多芬|貝多芬]]的《[[第6號交響曲_(貝多芬)|田園]]》，進而將此樂曲套入在動畫中。<ref>{{cite book |title={{lang|ja|映画を作りながら考えたこと　1955～1991}} |year=1991 |publisher=德間書店 |author=高畑勳 |isbn=978-4-19-554639-0|page=185頁}}</ref>而原作中另提到的兩首樂章{{nihongo|「印度獵虎」|インドの虎狩}}及{{nihongo|「快樂的馬車伕」|愉快な馬車屋}}在動畫版裡表現出來的曲調，則由動畫的配樂負責人[[間宮芳生|間宮芳生]]所創作。<ref>{{cite book |title={{lang|ja|映画を作りながら考えたこと　1955～1991}} |year=1991 |publisher=德間書店 |author=高畑勳 |isbn=978-4-19-554639-0|page=206頁}}</ref>
