'''大專畢業生至企業職場實習方案'''，俗稱'''22K方案'''或'''22K政策'''，是[[中華民國|中華民國]][[中華民國教育部|教育部]]於2009年4月至2011年9月間，分兩階段實施的就業補助方案。由教育部（時任教育部長[[鄭瑞城|鄭瑞城]]）、大專院校與[[行政院勞工委員會|行政院勞工委員會]]（時任主任委員 [[王如玄|王如玄]]）的合作平台，來協助近三年大專畢業生進入職場實習，由政府提供每個畢業生每月新台幣2萬2千元薪資（俗稱22k）並外加[[勞保|勞]][[健保|健保]]，輔導他們進入和學校建教合作之企業職場，實習工作為期一年，總共提供3萬餘個名額。此方案實施及結束後，卻帶動企業以22K作為大專畢業生起薪 ，連帶使青年薪資無法提升，造成青年貧窮化。不過相較2001年政府所推動的17K擴大就業方案，補助幅度已大幅增加增加54%。

== 方案實施經過 ==
受到[[金融海嘯|金融海嘯]]影響，企業提供職缺減少，失業率飆高，因此在2009年4月-2010年9月間，從[[行政院經濟建設委員會|行政院經濟建設委員會]]《[http://excellence.ttu.edu.tw/ezfiles/85/1085/img/918/906371448.pdf 振興經濟擴大公共建設特別條例]》特別預算支出108億元，實施「95學年度至97學年度大專畢業生至企業職場實習方案」，<ref>{{cite web|title=培育優質人力促進就業計畫 方案一之一大專畢業生至企業職場實習方案實施要點|url=http://www.ejob.gov.tw/2009_togenjob/pdf.html|publisher=全國就業e網|date=2010-08-11|accessdate=2014-08-11|deadurl=yes|archiveurl=https://web.archive.org/web/20140521030756/http://www.ejob.gov.tw/2009_ToGenJob/pdf.html|archivedate=2014-05-21}}</ref>並由教育部主辦。以實習名義，提供近3年大專畢業生1年期工作機會，名額共41470人，由政府補助企業每人每月本薪2萬2000元及勞、健保費用4190元共26190元，相較2001年政府所推動的17K擴大就業方案， 補助幅度增加54%。另亦補助年終獎金，給薪優於基準者由企業自行負擔。

由於第一階段造成的多項爭議，2010年9月-2011年9月實施第二階段「96學年度至99學年度大專畢業生至企業職場實習方案」時，補助金額改為1萬元，期限亦降為半年，實際共補助22414人，支出18億元。<ref>{{cite web| url =http://plan5.erp.moe.gov.tw/plan_detail.php?sn=1| title =九十六學年度至九十八學年度大專畢業生至企業職場實習方案實施要點| work =中華民國教育部| date =2010-08-11| accessdate =2014-08-11}}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

2013年6月3日，大專畢業生至企業職場實習方案辦法廢止。<ref>{{cite web | url =http://edu.law.moe.gov.tw/LawContent.aspx?id=GL000446 |work =中華民國教育部 |title=廢/停九十六學年度至九十九學年度大專畢業生至企業職場實習方案實施要點 |accessdate=2014-0811}}</ref>

== 方案爭議 ==
由於第一階段政府補助額高達26190元，又沒有要求企業額外增加薪資，因此許多企業都直接以政府補助額實施而未額外增加薪資，26190元當中包含原本應由企業負擔的勞、健保費用，實習生實際可領取的薪資為22000元，此金額雖高出當時[[台灣基本工資|台灣基本工資]]17280元，卻低於當時大專畢業生平均薪資。而在方案實施後，出現許多企業在聘用大專畢業生時將薪資訂在22000元，而[[行政院主計總處|主計處]]的「受僱就業者每月主要工作之經常性收入」統計，在20至24歲，相當於一般大專畢業生的年齡層，2008年5月方案實施前為24313元，2009年方案實施後則只剩下21685元，似乎也顯示出方案實施及金融海嘯造成的薪資影響<ref>{{cite news|title=實施大專畢業生實習方案後　青年起薪回不去卡22Ｋ|url=http://www.ettoday.net/news/20120822/90931.htm|accessdate=2014-08-11|newspaper=ettoday|date=2012-08-22}}</ref>。另外同樣由主計處公佈的「名目及實質平均薪資」，2009年創下13年新低，也被認為與此方案有關<ref>{{cite journal|author=朱學恆|title=名目及實質平均薪資|journal=非凡新聞週刊|issue=182|url=http://www.ustv.com.tw/UstvMedia/magdetail/index/mag-emag-bigmark_id-1785-page-1.html|date=2009年10月|date=11}}</ref>。

此外，政府支出高額補助，雖然對於當時的失業率有所改善，但根據[[中華民國審計部|審計部]]提出的「一百年度中央政府振興經濟擴大公共建設特別決算審核報告」，該方案2階段之聘期屆滿後，平均留用只有12933人，占平均總媒合人數的40.49%，並不算高<ref>{{cite news|author=楊毅|title=大專畢業生至企業實習 22K成效不佳 留用率僅4成|url=https://tw.news.yahoo.com/%E5%A4%A7%E5%B0%88%E7%95%A2%E6%A5%AD%E7%94%9F%E8%87%B3%E4%BC%81%E6%A5%AD%E5%AF%A6%E7%BF%92-22k%E6%88%90%E6%95%88%E4%B8%8D%E4%BD%B3-%E7%95%99%E7%94%A8%E7%8E%87%E5%83%854%E6%88%90-213000983.html|accessdate=2014-08-11|newspaper=yahoo新聞轉述中國時報報導|date=2013-04-09}}</ref>，對長期失業率助益有限。而在工作內容上，大多企業在此方案所聘的人員是屬於業務助理工作，對於原始方案目標的「縮短產學落差」，也幾乎沒有助益<ref>{{cite journal|author=史書華|title=大專生企業實習，誰準備好了？|journal=Cheers雜誌|issue=105|url=http://www.cheers.com.tw/article/article.action?id=5022225&page=1|date=2009年6月}}</ref>。

而在方案實施期間，雖有先設定基本門檻，如該企業不得有大量解雇勞工紀錄等，但仍出現許多企業違反規定之情況，嚴重者如在錄用實習生後解雇正職勞工<ref>{{cite news|author=薛荷玉|title=22K方案結束 留用率達6成5|url=http://mag.udn.com/mag/edu/storypage.jsp?f_ART_ID=278450|accessdate=2014-08-11|newspaper=聯合報|date=2010-10-21}}</ref>，輕微者如將勞健保寄到別家廠商投保，此類狀況不一的違規情況，第一階段共有108家，第二階段則有171家。

2010年4月，當時的[[勞委會|勞委會]]主委[[王如玄|王如玄]]，接受[[天下雜誌|天下雜誌]]專訪時，對於該方案，說出了「嫌兩萬二少？」「要是沒有這個方案，這些人一毛都沒有」等評語，並強調政府無法干預求職市場，供過於求造成價格下降<ref>{{cite journal|author=陳一姍、丁筱晶|title=王如玄：金融海嘯也許是個轉機|journal=天下雜誌|date=4|year=2011|month=13|issue=444|url=http://m.cw.com.tw/news/article.action?id=5000466&idRssItem=1447235}}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，引發軒然大波<ref>{{cite news|author=洪素卿、胡清暉|title=大專生月領2萬2 王如玄：不少了 要是沒有22k方案 「這些人一毛都沒有」|url=http://news.ltn.com.tw/news/focus/paper/385575|accessdate=2014-08-11|newspaper=自由時報|date=2010-04-07}}</ref>。

2012年12月，人力銀行業者公布「新鮮人就業追蹤調查」，統計社會新鮮人實領平均月薪約24982元，其中約有29%受訪者月領不到22K<ref>{{cite news|author=陳如嬌、楊惠琪|title=2成9新鮮人 起薪不到2萬|url=http://www.appledaily.com.tw/appledaily/article/headline/20121206/34688562/|accessdate=2014-08-11|newspaper=蘋果日報|date=2012-12-06}}</ref>。

== 22K後續爭議 ==
2013年9月7日，[[國立臺灣大學|台灣大學]]校長[[楊泮池|楊泮池]]說出「學生應培養基礎能力，假如沒有能力，22K仍是太高」引發爭議<ref>{{cite news|author=許敏溶|title=台大校長：若沒能力 領22K太高|url=http://www.appledaily.com.tw/appledaily/article/headline/20130908/35279844/|accessdate=2014-08-11|newspaper=蘋果日報|date=2013-09-08}}</ref>。

2014年5月8日，[[鴻海集團|鴻海集團]]總裁[[郭台銘|郭台銘]]表示，他認為台灣年輕人大有可為，比鄰近國家更有希望，還說到底哪家公司月薪只給22K，要是知道了要把他買下來然後調薪<ref>{{cite news|title=郭董：哪家公司只付薪22K　我買下再調薪|url=http://www.appledaily.com.tw/realtimenews/article/new/20140508/394207/|accessdate=2014-08-11|newspaper=蘋果日報|date=2014-05-08}}</ref>。
