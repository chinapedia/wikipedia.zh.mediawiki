《'''大南實錄'''》（{{vie|v=Đại Nam thực lục|hn=大南實錄{{`}}大南寔錄|f=h}}），是[[越南|越南]][[阮朝|阮朝]]的官修[[編年體|編年體]]史書，由阮朝國史館編修，584卷，採用[[漢文|漢文]][[文言文|文言文]]寫成。

== 名称 ==
其本来的书名与中国的《[[明實錄|明實錄]]》、《[[清實錄|清實錄]]》等类似，使用“'''實錄'''”作为名称，而後由于「'''實'''」是第二代皇帝[[明命帝|明命帝]]的[[皇后|皇后]][[胡氏華|胡氏]]之諱，因此後来改用同音同義的「'''寔'''」字{{NoteTag|「實」为嘉隆帝所给的賜名<ref>陳, 1982, pp. 570-571.</ref>。}}。

== 編修過程 ==
由於《[[大南|大南]]實錄》是緊接著[[阮朝|阮朝]]的歷史發展而進行編修，因此整個編修過程可說是與[[阮朝|阮朝]]相始終，時間相當漫長。

《大南實錄》編纂的開始，是[[嘉隆|嘉隆]]十年（1811年），[[阮福映|阮世祖]]下令由侍中大學士[[范貴適|范貴適]]負責撰修「國朝實錄」。其後，在[[明命|明命]]年間，編修實錄的事業得到充份的支持，這是由於[[阮福晈|阮聖祖]]即位之初，即已考慮到編寫書籍之事，他設置了國史館，專司編修國史。<ref>《[[越南史略|越南史略]]·第五卷·第五章》（或譯作《越南通史》），【越】[[陳仲金|陳仲金]]撰，北京[[商務印書館|商務印書館]]1992年版，第321-322頁。</ref>由[[阮文仁|阮文仁]]、[[鄭懷德|鄭懷德]]等多位官員負責編撰工作。其後經過歷代史官的續編，至20世紀時，已編成至[[啟定|啟定帝]]（1916年-1925年）時期歷代阮朝帝王的實錄。<ref name="東南亞歷史詞典16">《東南亞歷史詞典‧「大南實錄」條》，16頁。</ref>

== 內容 ==
《大南實錄》的內容，主要是記載歷代皇帝事跡的「實錄」，另外還有記錄各階層人物的「列傳」。
* 大南實錄前編（十二卷），載自[[阮潢|阮潢]]至[[阮福淳|阮福淳]]約二百多年間歷代[[阮主|阮主]]在[[越南|越南]]中、南部經營管治的歷史。<ref name="陳荊和3">陳荊和《「國史遺編」的編者與內容》，收錄於《國史遺編》，3頁。</ref><ref name="東南亞歷史詞典16" />阮憲祖绍治四年（清道光二十四年，1844年）刊。日本庆应大学1961年（日本昭和36年）影印出版。
* 大南列傳前編（六卷），載阮主時期的后妃、公子及諸臣的列傳。<ref name="陳荊和3" /><ref name="東南亞歷史詞典16" />阮翼宗嗣德五年（清咸丰二年，1852年）刊。日本庆应大学1961年（日本昭和36年）影印出版。
* 第一紀[[阮福映|嘉隆朝]]實錄（六十卷），當中的卷一至卷十六是1778年至1802年[[西山朝|新]][[舊阮|舊阮]]之間的戰事，其餘則為1802年至1819年間嘉隆朝的歷史。<ref name="陳荊和3" /><ref name="東南亞歷史詞典16" />翼宗嗣德元年（清道光二十八年，1848年）刊。日本庆应大学1963年（日本昭和38年）影印出版。
* 第二紀[[阮福晈|明命朝]]實錄（二百二十卷），載1820年至1840年間明命朝的歷史。<ref name="陳荊和3" /><ref name="東南亞歷史詞典16" />翼宗嗣德十四年（清咸丰十一年，1861年）刊。日本庆应大学1971年（日本昭和46年）影印出版。
* 第三紀[[阮福暶|紹治朝]]實錄（七十二卷），載1841年至1847年間紹治朝的歷史。<ref name="陳荊和3" /><ref name="東南亞歷史詞典16" />翼宗嗣德三十年（清光绪三年，1877年）刊。日本庆应大学1977年（日本昭和52年）影印出版。
* 第四紀，[[嗣德帝|嗣德朝]]實錄（七十卷），載1847年至1883年間嗣德、[[協和帝|協和]]等朝的歷史。<ref name="陳荊和4">陳荊和《「國史遺編」的編者與內容》，收錄於《國史遺編》，4頁。</ref><ref name="東南亞歷史詞典16" />[[成泰帝|成泰帝]]成泰六年（清光绪二十年，1894年）刊。日本庆应大学1979年（日本昭和54年）影印出版。
* 第五紀，[[建福帝|建福朝]]、[[咸宜帝|咸宜帝]]實錄（八卷），載1883年至1885年建福朝、咸宜朝的歷史。<ref name="陳荊和4" /><ref name="東南亞歷史詞典16" />[[成泰帝|成泰帝]]成泰十二年（清光绪二十六年，1900年）刊。日本庆应大学1980年（日本昭和55年）影印出版。
* 第六紀，[[同慶帝|同慶朝]]實錄（十一卷），載1885年至1888年主要记载同慶朝的歷史。<ref name="陳荊和4" /><ref name="東南亞歷史詞典16" />[[维新帝|维新帝]]维新三年（清宣统元年，1909年）刊。日本庆应大学1980年（日本昭和55年）影印出版。
* 第六紀附編，分為[[成泰帝|成泰帝]]附編和[[維新帝|維新帝]]附編（二十九卷）。<ref name="東南亞歷史詞典16" />弘宗启定七年（1922年）開始編修，保大年間編修完成。
* 第七紀，[[啟定|启定朝]]實錄（十卷）。<ref name="東南亞歷史詞典16" />[[保大帝|保大帝]]保大十四年（1939年）編修完成。
* 大南實錄正编列傳初集（三十三卷），是[[阮福映|嘉隆朝]]的后妃、皇子、公主、貴戚、諸臣、[[西山朝|偽西]]、外國（[[柬埔寨|高蠻]]、[[暹羅|暹羅]]等）的列傳。<ref name="陳荊和4" /><ref name="東南亞歷史詞典16" />[[成泰帝|成泰帝]]成泰元年（清光绪十五年，1889年）刊。日本庆应大学1962年（日本昭和37年）影印出版。
* 大南实录正編列傳二集（四十六卷），是[[阮福晈|明命]]、[[阮福暶|紹治]]、[[嗣德|嗣德]]三朝的后妃、皇子、公主、貴戚、諸臣、忠義、行義、烈女、隱逸、逆臣列傳。<ref name="東南亞歷史詞典16" />[[成泰帝|成泰帝]]成泰七年（清光绪二十一年，1895年）刊。日本庆应大学1981年（日本昭和56年）影印出版。

== 史料價值 ==
《大南實錄》記錄了阮朝歷代后妃、皇子、公主、宗室、貴戚、諸臣、忠義、行列女、高僧、逆臣、奸臣等等不同類型人物的事跡，並且有[[西山朝|西山朝]]及鄰近國家的情況，包括了近代[[越南|越南]]的政治、經濟、軍事、文教、外交及戰爭等等多個方面的史料。另外，在它的涉及[[中國|中國]][[明朝|明]][[清朝|清]]時期的部份史料，亦可供[[中國歷史|中國歷史]]研究者參考。<ref name="東南亞歷史詞典16" />

== 流傳情況 ==
據學者[[陳荊和|陳荊和]]所說，《大南實錄》及各篇列傳原是秘藏在阮朝宮廷的典籍，素不公開，世上的大學、圖書館及研究所亦收藏不多。1933年，[[日本|日本]][[慶應大學|慶應大學]][[松本信廣|松本信廣]]教授赴越旅行，曾找來六套「實錄」及「列傳」以分贈日本各著名大學。其後，慶應大學將之刻印出版。在越南方面，1960年代也陸續出版了[[越南語|越文]]版。<ref>陳荊和《「國史遺編」的編者與內容》，收錄於《國史遺編》，6-7頁。</ref><ref name="東南亞歷史詞典16" />

从2001年至2007年，越南[[教育出版社|教育出版社]]陆续出版越南文《大南實錄》共十冊，第一冊為寔錄前編和正編第一紀，第二、三、四、五冊為正編第二紀，第六冊為正編第三紀，第七、八冊為正編第四紀，第九冊為正編第五紀和第六紀，第十冊為索引。2005年，越南[[順化出版社|順化出版社]]出版越南文《大南列傳》四冊，第一冊為列傳前編，第二冊為列傳正編初集，第三、四冊為列傳正編二集。2012年越南[[文化藝術出版社|文化藝術出版社]]出版越南文《大南實錄正編第六紀附編》和《大南實錄正編第七紀》。
