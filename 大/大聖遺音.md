'''大聖遺音'''是[[唐朝|唐朝]][[唐肅宗|肅宗]]元年（756年）時由[[四川|四川]]斫琴世家雷氏為皇室斫成的[[古琴|古琴]]，現存兩床，其中一床原為[[王世襄|王世襄]]所藏，2003年以891萬人民幣成交，是當時拍賣價最高的古琴，後在2011年以1.15億人民幣的價格再次拍賣，這個價格僅次於1.3664億的宋琴[[松石間意|松石間意]]；另一床藏於[[北京故宮博物院|北京故宮博物院]]。<ref name="中評網">{{Cite web |url= http://hk.crntt.com/doc/1017/0/2/9/101702968.html|title=中國唐代古琴拍出1.15億元|accessdate=2015-02-19|work= |publisher=中國評論新聞網|date=2011-05-23}}</ref>其名取自[[王符|王符]]《[[潜夫論|潜夫論]]·潜嘆》中的“昔唐[[堯|堯]]之大聖也，聪明宣昭。虞[[舜|舜]]之大聖也，德音發聞。”<ref name="王風"/>

==伏羲式==
此大聖遺音屬[[古琴琴式|伏羲式]]、由桐木斫製，尺寸不詳，以[[鹿角霜|鹿角霜]]為灰胎，髤紫漆，金徽玉軫，龍池為圓形，鳳沼為橢圓形。通體蛇腹斷，琴額有冰紋斷。秦腹龍池上方刻草書“大聖遺音”，內刻隸書年款“至德丙申”，兩側刻隸書“嶧陽之桐，空桑之材，鳳鳴秋月，鶴舞瑤臺”，龍池下方是方印“困學”和“玉振”，其中“困學”印還見於其他五床現存的唐琴，只有“玉振”是僅見於此。所有字、印除去年款以外皆髹金。琴軫和琴足製於明，質地為青玉。<ref name="文化中國"/>

該床琴自唐朝以來沒有經過大修，傳世琴中十分難見。[[管平湖|管平湖]]認為這床琴是現存唯一具備《[[琴操|琴操]]》中琴應該有的“九德”，即“奇、古、透、潤、靜、圓、勻、清、芳”者。<ref name="王風"/><ref>{{Cite web|url=http://www.guqin.net/newweb/tq1.htm|title=大聖遺音琴·伏羲式|accessdate=2015-02-19|work=|publisher=中國古琴網|deadurl=yes|archiveurl=https://web.archive.org/web/20120126055529/http://www.guqin.net/newweb/tq1.htm|archivedate=2012-01-26}}</ref>

===歷史===
[[楊時百|楊時百]]的《[[琴學叢書|琴學叢書]]·藏琴錄·籠門寒玉》曾記載：“西園主人（[[溥侗|溥侗]]）因‘大聖遺音’‘玉振’印上有印方寸‘困學’二字，定為[[鲜于伯機|鲜于伯機]]印，或‘玉振’亦鲜于氏印也。”但[[鄭珉中|鄭珉中]]認為此說過於武斷，他認為“玉振”一定不是鮮于氏印，其它刻有此印的琴也看不出和鲜于氏有什麼關係。<ref name="王風">王風，<古琴器考辨示例>，《琴学荟萃:第3届古琴国际学术研讨会论文集》，ISBN 9787533326760</ref>

近代有跡可循的收藏家最早為琴人錫寶臣，之後可能是在1946年至1948年間[[王世襄|王世襄]]、[[袁荃猷|袁荃猷]]夫婦以“飾物三件及日本版《唐宋元明名畫大觀》，再加翠戒指三枚”從其孫章則川處換得。<ref name="文化中國"/><ref>{{Cite web |url= http://news.pchome.com.tw/magazine/report/cl/artouch_antique/1673/126270720019661018006_2.htm|title=玩物因緣95年：追憶王世襄|accessdate=2015-02-19|work= |publisher=PCome新聞|date= 2010-01-06}}</ref> 當時琴的情況很差，由管平湖替王世襄聘請銅器修復家高英製作銅套，又請[[金禹民|金禹民]]刻：“世襄、荃猷，鬻書典釵，易此枯桐”，由管平湖安裝、修復。<ref name="文化中國">{{Cite web |url= http://cul.china.com.cn/zhenbao/2011-07/07/content_4317141.htm|title=唐代古琴 大聖遺音|accessdate=2015-02-19|work= |publisher=文化中國|date=2011-05-31}}</ref> 2003年面世，同年11月以891萬人民幣的價格刷新當年[[九霄環佩|九霄環佩]]創下的古琴拍賣紀錄，2011年5月22日在[[中國嘉德國際拍賣有限公司|嘉德]]春拍中再次成交，價格達1.15億，僅次於1.3664億的宋琴[[松石間意|松石間意]]。<ref name="中評網2">{{Cite web |url= http://hk.crntt.com/doc/1017/1/6/5/101716529.html?coluid=7&kindid=0&docid=101716529&mdate=0601093336|title=市場上唯一唐琴：“大聖遺音”的億元傳說|accessdate=2015-02-19|work= |publisher=環渤海新聞網|date=2011-06-01}}</ref><ref>[https://www.youtube.com/watch?v=KoqIwX_SCzk 千年唐代古琴大剧院再次奏响 - 大圣遗音]</ref>

==神農式==
神農式的大聖遺音現藏於故宮博物院。<ref name="文化中國"/>琴長{{convert|120|cm}}，肩寬{{convert|20.5|cm}}，尾寬{{convert|13.4|cm}}。<ref name="中華博物">{{Cite web |url= http://www.gg-art.com/profesional/index_b.php?detailid=609|title=大聖遺音琴|accessdate=2015-02-19|work= |publisher=中華博物}}</ref><ref name="鑒定"/>桐木製，髹栗殼色漆、其上罩有黑漆，還有朱漆補綴期間。斷紋為蛇腹間牛毛斷紋。龍池兩側刻“巨壑迎秋，寒江印月。萬籁悠悠，孤桐颯裂”。其他形制均與前者相同。<ref name="王風"/>

清朝時藏於宮中，但具体的历史已經無法查證。在清宮南庫中因久無人打理而積滿水鏽，清宮善後委員會清點時標記為“破琴一張”，編號崑字一〇七號。幾十年後在1947年由於故宮博物館任職的王世襄發現，并轉入珍寶庫。1949年他得到館長[[馬衡|馬衡]]批准，將此琴交由管平湖修復。<ref name="鑒定">鄭珉中，《蠡测偶录集-古琴鉴定及其他》，紫禁城出版社，2010，ISBN 9787800477768</ref>
