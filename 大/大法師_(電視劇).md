《'''大法師'''》（{{lang-en|''The Exorcist''}}）是一部改編自{{link-en|威廉·彼得·布拉蒂|William Peter Blatty}}的{{link-en|大法師 (小說)|The Exorcist (novel)|同名小說}}，由{{link-en|傑若米·史雷特|Jeremy Slater}}開創，並於2016年9月23日在[[福斯廣播公司|FOX]]開播的[[美國|美國]]電視劇。<ref name="Series Order">{{cite web|url=http://deadline.com/2016/05/the-exorcist-lethal-weapon-making-history-pitch-the-mick-apb-fox-series-orders-1201752948/|title=‘The Exorcist’, ‘Lethal Weapon’, ‘Making History’, ‘Pitch’, ‘The Mick’ & ‘APB’ Get Fox Series Orders|work=[[Deadline.com|Deadline.com]]|accessdate=2016-05-11|date=2016-05-10|first=Nellie|last=Andreeva|publisher={{link-en|潘斯奇媒體公司|Penske Media Corporation}}}}</ref>由{{le|阿方索·埃雷拉|Alfonso Herrera}}與{{link-en|班·丹尼爾斯|Ben Daniels}}領銜主演，其他主要演員還包括[[吉娜·戴維斯|吉娜·戴維斯]]、[[約翰·趙|約翰·趙]]、[[阿蘭·拉克|亞倫·拉克]]與{{le|布莉安娜·海德布蘭德|Brianna Hildebrand}}等。

2017年5月12日，本劇獲FOX第2季續訂，<ref name="Season 2 Renewed"/>但在播出2季後，電視網於2018年5月11日宣布取消劇集。<ref name="Canceled"/>

== 故事 ==
以循序漸進、系列化心理驚悚的方式，講述行事作風迥異的兩個神父，攜手處理一個遭受惡靈控制的家庭，並與真正的惡魔正面對決。<ref name="Laura"/>

=== 主要角色 ===
*; {{link-en|阿方索·埃雷拉|Alfonso Herrera}} 飾演 湯瑪斯·歐提嘉（{{lang|en|Tomas Ortega}}）
: 神父，[[芝加哥|芝加哥]]冉冉升起的神職新星。
*; {{link-en|班·丹尼爾斯|Ben Daniels}} 飾演 馬可斯·基恩（{{lang|en|Marcus Keane}}）
: 出身自英格蘭中部，擁有許多驅魔經歷的神父，但礙於手法殘酷而不被教會認可，為一名[[雙性戀|雙性戀者]]。
*; [[吉娜·戴維斯|吉娜·戴維斯]] 飾演 安潔拉·蘭斯（{{lang|en|Angela Rance}}）
: 本名：{{link-en|芮根·麥尼爾|Regan MacNeil}}（{{lang|en|Regan MacNeil}}）<br/>繁忙的飯店經理人，亨利的妻子。盡她所能地保持積極，並藉宗教力量尋求慰藉與幫助。幼時曾被惡靈纏身，如今惡靈卻再度上門，威脅著她過去的秘密與家人的性命。
*; 漢娜·卡索卡（{{lang|en|Hannah Kasulka}}） 飾演 凱茜·蘭斯（{{lang|en|Casey Rance}}）
: 安潔拉與亨利的小女兒，因為活在姐姐的陰影下而將自我獻給了惡魔。
*; 布萊恩妮·豪伊（{{lang|en|Brianne Howey}}） 飾演 凱瑟琳·蘭斯（{{lang|en|Katherine Rance}}）
: 安潔拉與亨利的大女兒，前芭蕾舞舞者，因一場車禍意外而開始封閉自己。
*; 庫特·席爾瓦（{{lang|en|Kurt Egyiawan}}） 飾演 戴文·班奈特（{{lang|en|Devon Bennett}}）
: 教會神職人員，[[梵蒂岡|梵蒂岡]]教皇的聯繫人員。
*; [[阿蘭·拉克|亞倫·拉克]] 飾演 亨利·蘭斯（{{lang|en|Henry Rance}}）
: 安潔拉的丈夫，因意外而傷及腦袋。
*; [[約翰·趙|約翰·趙]] 飾演 安德魯·“安迪”·金（{{lang|en|Andrew "Andy" Kim}}）
: 前兒童心理醫師，居住於[[華盛頓州|華盛頓州]]的幽僻私人小島納奇本島上，與五名寄養孩童組建家庭。
*; {{le|布莉安娜·海德布蘭德|Brianna Hildebrand}} 飾演 菲芮媞（{{lang|en|Verity}}）
: 即將18歲，安德魯寄養家庭的孩子。表面看似不在乎，事實上卻願為寄養家庭的家人們赴湯蹈火。
*; {{le|祖蕾卡·羅賓遜|Zuleikha Robinson}} 飾演 鼠兒（{{lang|en|Mouse}}）
: 忠心的教徒，-{欲}-推翻天主父權制。曾與馬可斯關係親近，因好奇而遭惡靈附身，並對馬可斯驅魔未成功的離開感到不諒解。
*; {{le|李麗君|Li Jun Li}} 飾演 蘿絲·庫柏（{{lang|en|Rose Cooper}}）
: 被分配觀察安德魯一家的社工，與安德魯有過一段情。

=== 常設角色 ===
*; {{link-en|雪倫·葛雷斯|Sharon Gless}} 飾演 [[大法師|克莉絲·麥尼爾]]（{{lang|en|Chris MacNeil}}）
: 安潔拉的母親，著名女演員，與女兒關係不睦，最後遭附身的安潔拉殺害。
*; {{le|卡蜜兒·格瓦提|Camille Guaty}} 飾演 奧莉薇雅（{{lang|en|Olivia}}）
: 湯瑪斯的姊姊，育有一子，也非常關心湯瑪斯。
*; 羅伯特·艾密特·朗尼（{{lang|en|Robert Emmet Lunney}}） 飾演 [[帕祖祖|帕祖祖]]（{{lang|en|Pazuzu}}）
: 曾經附身在芮根體內，多年後纏上凱茜，而安潔拉為拯救凱茜，選擇再度與它結合。
*; {{link-en|蒂安娜·達納根|Deanna Dunagan}} 飾演 伯納戴修女（{{lang|en|Mother Bernadette}}）
: 修女院院長，擅長以柔性的方式驅魔，後遭帕祖祖復仇而身亡。
*; 茉札姆·麥克（{{lang|en|Mouzam Makkar}}） 飾演 潔西卡（{{lang|en|Jessica}}）
: 湯瑪斯的舊情人，婚姻不美滿的有夫之婦，與湯瑪斯展開地下戀情。
*; {{link-en|法蘭西斯·貴南|Francis Guinan}} 飾演 賽門弟兄（{{lang|en|Brother Simon}}）
: 募捐會主席，為惡魔儀式的頭目。在準備謀害教宗時，及時被馬可斯持十字架刺頸身亡。
*; 克絲汀·費茲傑拉德（{{lang|en|Kirsten Fitzgerald}}） 飾演 瑪麗亞·沃特斯（{{lang|en|Maria Walters}}）
: 一名患有癌症的富有女商人，是個虔誠的信徒，非常渴望能獲得惡靈的青睞。在與惡靈結合後，被病痛纏身，終被鼠兒了結。
*; 琪拉·納頓（{{lang|en|Keira Naughton}}） 飾演 雀莉·瑞戈（{{lang|en|Cherry Rego}}）
: 雷斯特的妻子，經營觀光巴士生意，對宗教頗有研究，在幫助馬可斯的同時遭遇不測。
*; 肯·馬克斯（{{lang|en|Ken Marks}}） 飾演 雷斯特·瑞戈（{{lang|en|Lester Rego}}）
: 雀莉的丈夫，經營觀光巴士生意，對宗教頗有研究，在幫助馬可斯的同時遭遇不測。
*; {{link-en|蒂姆·霍柏|Tim Hopper}} 飾演 傑費（{{lang|en|Jaffey}}）
: 警司，與瑪麗亞等人進行惡魔儀式。
*; {{le|艾莉西亞·維特|Alicia Witt}} 飾演 妮可·“妮姬”·金（{{lang|en|Nicole "Nikki" Kim}}）
: 安德魯的妻子，因[[抑鬱性障礙|憂鬱症]]而[[自殺|自殺]]身亡，後來被惡靈作為蠱惑安德魯的化身。
*; {{le|賽勒斯·阿諾德|Cyrus Arnold}} 飾演 大衛·強森三世（{{lang|en|David Johnson III}}）
: 綽號卡車（{{lang-en|Truck}}），寄養家庭的孩子，與同齡孩子相比身形高大，但心智仍然是個孩子。遭到母親遺棄在路邊，患有[[夢遊症|夢遊症]]。
*; {{le|卡琳·科諾瓦|Karin Konoval}} 飾演 德蘿芮絲·納瓦洛（{{lang|en|Delores Navarro}}）
: 失蹤許久的驅魔人，實際上遭到惡靈附身，並因宿主不復存在終而殞落。
*; {{le|克里斯托弗·寇辛斯|Christopher Cousins}} 飾演 彼得·奧斯本（{{lang|en|Peter Osborne}}）
: 美國魚類與野生動物調查官，安德魯一家的親友。
*; 艾力克斯·巴里馬（{{lang|en|Alex Barima}}） 飾演 謝爾比（{{lang|en|Shelby}}）
: 寄養家庭的孩子，是個寬容的人。自我與信仰拉扯著，在發現怪異事件發生後，相信唯有上帝能拯救他的家庭。
*; 杭特·狄隆（{{lang|en|Hunter Dillon}}） 飾演 凱勒布（{{lang|en|Caleb}}）

== 評價 ==
《大法師》獲得了普遍的好評。於[[爛番茄|爛番茄]]整合的評論中，第1季獲得78%新鮮度、6.12/10分（50位評論家）。評論家普遍共識：「《大法師》並未趨近原作，但合理的恐怖與可信的特效仍然擁有緊張的敘述。」<ref>{{cite web|title=The Exorcist: Season 1|url=https://www.rottentomatoes.com/tv/the_exorcist/s01/|website=[[爛番茄|爛番茄]]|accessdate=2016-12-18}}</ref>於[[Metacritic|Metacritic]]整合的評論中，第1季獲得62分（滿分100分；28位評論家），屬普遍的好評。<ref>{{cite web|title=The Exorcist (2016) - Season 1 Reviews|url=http://www.metacritic.com/tv/the-exorcist-2016|website=[[Metacritic|Metacritic]]|accessdate=2016-09-29}}</ref>

第2季依然獲得不錯的評價，於爛番茄整合的評論中，獲得100%新鮮度、7.67/10分（5位評論家）<ref>{{cite web|title=The Exorcist: Season 2|url=https://www.rottentomatoes.com/tv/the_exorcist/s02/|website=爛番茄|accessdate=2017-10-22}}</ref>
