《'''大魔法峠'''》（{{lang-ja|大魔法峠}}），是[[日本漫畫家|日本漫畫家]][[大和田秀樹|大和田秀樹]]在《[[月刊少年Ace|月刊少年ACE]]》與《[[:ja:エース桃組|ACE桃組]]》（皆由[[角川書店|角川書店]]發行）上連載的[[日本漫畫|日本漫畫]]作品，故事內容是魔法王國[[公主|公主]]田中蒲妮惠到[[日本|日本]]修業的故事。標題源自-{[[中里介山|中里介山]]}-原作的日本小說《[[大菩薩嶺_(小說)|大菩薩嶺]]》（原文「{{lang|ja|大菩薩峠}}」）<ref>「峠」是[[和製漢字|和製漢字]]，同「嶺」。</ref>。

== 故事簡介 ==
來自位於天空的「聖魔法王國」的公主田中蒲妮惠，為了要成為下任[[女王|女王]]而來到地上修業。雖然外表看起來只是可愛柔弱的[[魔法少女|魔法少女]]，但真面目是為了登上大位不惜排除一切障礙的[[君主论|馬基維利主義]]者；除了魔法外，還擅長各式讓對手肉體跟[[精神|精神]]遭完全破壞的[[關節技|關節技]]（[[:en:Submission_(combat_sport_term)|Submission]]）。在人間的修業中，到底有多少人會被這「人肉推土機」鏟平呢？

== 人物介紹 ==
; {{nihongo|田中蒲妮惠|田中ぷにえ||[[声優|声優]]：[[佐藤利奈|佐藤利奈]]}}
:魔法王國的公主，表面上看起來是甜美的魔法美少女，但實際上是城府極深、為求勝利不擇手段的馬基維利主義者。擅長關節技，沒有她-{折}-不斷的關節。會以無情的關節技制裁來犯的對手。在她的眼中，打撃-{系}-的招數只是「花拳繡腿」，關節技才是「王者之技」。為了登上大位，就算對手是手足，也會毫無顧忌地痛下毒手。
:其姓名源自日本男演員[[田中邦衛|田中邦衛]]。

; {{nihongo|大姊頭|姉御||声優：[[川澄綾子|川澄綾子]]}}
:漫畫中常見會找主角麻煩的不良少女，身邊跟著一群兇惡的跟班，但是找了蒲妮惠幾次麻煩之後都遭到痛擊慘敗，之後懂得敬鬼神而遠之。
:以普通人的強度而言，大姊頭具有相當的實力，但是周遭來自魔法王國的人都太強，所以常常吃敗仗。此外，大姊頭也是一個在劇中負責[[吐槽|吐槽]]的角色。
:雖然是個不良少女，但有著寫詩與撰寫[[手機小說|手機小說]]的少女興趣。

; {{nihongo|國鐵子|国鉄子||声優：[[下屋則子|下屋則子]]}}
:[[鐵道迷|喜歡鐵道]]的[[內向|內向]]資優[[眼鏡娘|眼鏡娘]]。
:熟悉失傳已久的[[新幹線|新幹線]]餐車[[咖哩|咖哩]]口味，在意外當上[[學生會|學生會]]會長後曾推廣[[鐵道模型|鐵道模型]]製作。

; {{nihongo|艾莉絲·馮·巴魯巴洛克|エリーゼ＝フォン＝バルバロック|声優：[[能登麻美子|能登麻美子]]}}
:前魔法王國公主，因為田中一家奪權的關係而跟父親一起遭[[放逐|放逐]]，意圖向蒲妮惠復仇。

; {{nihongo|啪呀蛋|パヤたん||声優：[[齋藤千和|齋藤千和]]/[[中田讓治|中田讓治]]}}
:全名{{nihongo|'''啪呀·-{里}-賓克斯頓'''|パヤ＝リビングストン}}，軍人出身，階級為[[上校|上校]]。
:為龍蛇混雜的瓦庫吉祥物村（{{lang|ja|わくわくマスコット村}}）中之最強[[吉祥物|吉祥物]]。在與蒲妮惠的對決中敗北之後，以隨時都會找機會暗殺她的情況成為公主的家臣。
:當初為了得到聖魔法王國的永久居住權而加入了聖魔法王國吉祥物[[海軍陸戰隊|海軍陸戰隊]]。過去曾參與[[斯大林格勒战役|斯大林格勒战役]]跟[[越南战争|越戰]]，因為長時間在戰場上目睹了地獄般的景像而受到了深刻的心理創傷。即使退伍後仍長期受[[創傷後心理壓力緊張症候群|創傷後心理壓力緊張症候群]]所苦，時常會回想起戰場上的殘酷影像。
:腹部的「パ」字樣是在越戰時被俘，受敵人的嚴刑拷問時所[[烙印|烙印]]上的。

; {{nihongo|田中艾斯梅蘭達|田中エスメラルダ||声優：[[小山茉美|小山茉美]]}}
:蒲妮惠的母親。君臨聖魔法王國的女王，擁有比蒲妮惠更強的關節技與更加冷酷的心，同樣信奉著「實力即正義」的信條。有著強烈的權力欲，透過簒奪王權之力而爬上了女王之位。在權力鬥爭的年代，使用了奸計打倒了王國中實力比她更加強大的魔法使們，接下來打算征服[[太陽系|太陽系]]。由於實施高壓統治，所以國內外樹敵眾多，時常受到來自先王家及眾多政敵們的狙擊。

; {{nihongo|田中公彥|田中キミヒコ||声優：[[飛田展男|飛田展男]]}}
:艾斯梅蘭達的丈夫，蒲妮惠的父親。有著巨大金髮的摩希根風髮型，額頭上有著「Z」文字，僅穿著一件[[內褲|內褲]]外加一件披風。非常擔心女兒們的狀況，經常偷偷跟蹤及出手干涉。真正的[[受虐癖|受虐狂]]，被艾斯梅蘭達虐待時會得到快感，有習慣性偷腥的毛病。

; {{nihongo|蒲妮惠的祖母|ぷにえの祖母||}}
:公彥的母親，艾斯梅蘭達的婆婆。讓[[白雪公主|白雪公主]]吃下毒蘋果並使得[[睡美人|睡美人]]沉睡的元兇。《[[浮士德|浮士德]]》也記錄了她的存在。為了見孫女們而來到了三百年來未曾到訪的人界。[[巢鴨|巢鴨]]的偶像{{nihongo|'''屁川清志'''|屁川きよし}}的大粉絲。

; {{nihongo|田中純|田中ぴゅん||声優：[[齋藤彩夏|齋藤彩夏]]}}＆{{nihongo|田中螢|田中ぽたる||声優：[[花澤香菜|花澤香菜]]}}
:蒲妮惠的[[雙胞胎|雙胞胎]]妹妹們，八歲。與蒲妮惠一樣都是田中女王弱肉強食以力量決勝負的家訓下教育出的產物。為了王位，隨時都會暗算蒲妮惠。使用祕法成長之後，連攜的摔角合體技讓蒲妮惠陷入苦戰。

; {{nihongo|山田保健委員|山田保健委員||声優：{{link-ja|中野慎太郎|中野慎太郎|中野慎太郎}}}}
:蒲妮惠班上的保健委員，大姊頭對他一見鍾情。

; {{nihongo|雪風夕菜|雪風夕菜||}}
:漫畫研究社社長，本名'''毒島花子'''。

; {{nihongo|胃院長太郎|胃院長太郎||}}
:蒲妮惠班上的班長，對蒲妮惠一見鍾情。

; {{nihongo|青木副署長|青木副署長||声優：飛田展男}}
:蒲妮惠擔任「一日署長」的警察署副署長。

; {{nihongo|警察署長|警察署長||声優：[[乃村健次|乃村健次]]}}
:蒲妮惠擔任「一日署長」的警察署署長。

; {{nihongo|王島奈保奈|王島奈保奈||}}
:相貌威嚴、心思細膩的、熱愛棒球的「少女」。

; {{nihongo|國鐵郎|国鉄郎||}}
:國鐵子的父親，身高9呎，家裡經營壽司店，與國鐵子同樣熱愛鐵道。

; {{nihongo|國目照|国目照||}}
:國鐵子的母親，無法理解為何國鐵郎與國鐵子同樣熱愛鐵道。

; {{nihongo|鐵道之神|鉄道の神様||}}

; {{nihongo|[[麻生太郎|麻生太郎]]|麻生タロー||}}
