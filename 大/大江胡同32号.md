'''大江胡同32号'''，位于[[北京市|北京市]][[东城区_(北京市)|东城区]][[大江胡同|大江胡同]]32号，是[[北京市东城区普查登记文物|北京市东城区普查登记文物]]。

==简介==
大江胡同32号占地面积大约130平方米，曾是一家商铺，后转为[[公私合营|公私合营]]院，后来成为民宅。2006年，该院住户全部迁走后，负责前门东侧拆除腾退工作的大前门公司对该院进行封堵管理，其间曾发现文物构件丢失，因此在2011年将院门口“乾义成”牌子及砖雕收回保管。2012年，大江胡同32号被定为北京市东城区普查登记文物，但未挂牌。2015年，在[[前门东路|前门东路]]东侧的拆迁中，北京市万泽星房屋拆除工程有限责任公司擅自拆除了大江胡同32号四合院，整个院落几乎被夷平。2015年10月22日，东城区文化委员会执法队在巡查时发现大江胡同32号文物建筑被违法拆除，乃到现场调查取证，责令停止侵犯文物的行为。2015年11月，中国古迹遗址保护协会会员[[曾一智|曾一智]]向东城区文化委员会举报了大江胡同32号被违法拆除。同月，东城区文化委员会和大前门公司就大江胡同32号、[[冰窖斜街15号|冰窖斜街15号]]文保院落被拆除一事进行了说明，大前门公司表示大江胡同32号将根据原貌恢复，并将原保存的“乾义成”牌匾及砖雕、门墩等安回原处。2016年6月7日，东城区文化委员会作出“东文执罚（2016）第17号”决定，对擅自拆除不可移文物大江胡同32号的北京市万泽星房屋拆除工程有限责任公司处以罚款<ref name=tengxun>{{Cite web |url = http://view.inews.qq.com/a/BJC2015112101050803?clickfrom=weixinhot&refer=wx_hotnews |title =东城区严查私拆文保院落行为  |author =  |date =  2015-11-21|publisher =腾讯  |language =  |accessdate =  |quote =  }}</ref><ref name=zgwwxx>{{Cite web |url = http://www.ccrnews.com.cn/index.php/Xinwenzixun/content/id/62986.html |title =  2016年度全国文物行政处罚案卷评查优秀案卷|author =  |date = 2017-01-03  |publisher =中国文物信息网 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web|url = http://whw.bjdch.gov.cn/n1523901/n4144719/n4197661/c4281912/content.html|title = 东城区文委行政处罚案件信息公开表（2016年5月27日至8月8日）|author = |date = |publisher = 东城区文化委员会|language = |accessdate = 2017-02-27|quote = }}{{Dead link|date=2018年9月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。
