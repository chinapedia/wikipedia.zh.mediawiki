'''大豐銀行'''（[[葡文|葡文]]名稱：{{lang|pt|'''Banco Tai Fung'''}}；[[英文|英文]]名稱：{{lang|en|'''Tai Fung Bank'''}}）是[[澳門|澳門]]主要[[銀行|銀行]]之一。前身乃'''大豐銀號'''，於1942年由[[何賢|何賢]]等人創立。

== 歷史 ==
大豐銀行前身是大豐銀號，大豐銀號於1942年，由本來於[[香港|香港]]經營[[恆生銀行|恆生銀行]]的[[何添|何添]]、[[何善衡|何善衡]]、[[林炳炎|林炳炎]]，以及澳門聞人[[傅老榕|傅老榕]]等人合資成立，並聘何添的胞弟[[何賢|何賢]]為總經理，及後[[香港重光|香港重光]]，主要股東均回港重辦恆生銀行，何賢繼續留守，及後成為大股東。某程度上而言，大豐銀行與恆生銀行可算是姊妹銀行。

1972年，[[澳葡政府|澳葡政府]]頒佈《[[澳門銀行法|澳門銀行法]]》規定所有銀號要註冊為銀行，大豐銀號於同年1月20日正式獲有關當局批准註冊為「大豐銀行有限公司」，成為澳門首家以本地資金註冊的銀行，當時註冊資本為500萬元[[澳門元|澳門元]]。

1984年，由於[[擠提|擠提]]而導致銀行出現資金短缺問題，而銀行也面臨倒閉危機，同年9月13日，[[中國銀行|中國銀行]]總行收購大豐銀行50%股權，資本額增至1億6,000萬澳門元。
1997年，位於[[新口岸新填海區|新口岸新填海區]]的[[大豐銀行#總部|大豐銀行總行大廈]]落成啟用，而位於[[新馬路|新馬路]]的大豐銀行總行大廈重建。1999年3月，大豐銀行董事長[[何厚鏵|何厚鏵]]當選[[澳門特別行政區行政長官|第一任中華人民共和國澳門特別行區行政長官]]，因而把有關資產轉至其母何陳瓊；同年6月，[[澳門特別行政區行政長官辦公室|中華人民共和國澳門特別行政區行政長官臨時辦公室]]遷入同位於大豐銀行總行大廈的21樓辦公。
2017年，大豐銀行成立75周年暨開設上海分行。

大豐銀行已與分佈世界各地的300多間金融機構建立了代理行關係。

== 總部 ==
[[File:Edifício_do_Banco_Tai_Fung_e_Edifício_CNAC.jpg|right]]
大豐銀行總部現設於[[新口岸新填海區|新口岸新填海區]]（[[皇朝區|皇朝區]]）[[宋玉生廣場|宋玉生廣場]]418號，名為「'''大豐銀行總行大廈'''」；大豐銀行總行大廈是該區首批建成的商業大廈。大廈高24層，於1997年12月投入使用，由[[巴馬丹拿|巴馬丹拿]]作建築設計<ref>{{Cite web |url=https://www.p-t-group.com/gb/allhistory.html |title=巴马丹拿集团历史 |accessdate=2018-08-21}}</ref>。大廈外牆利用了花崗石和銀、灰色玻璃幕牆鋪設，大廈正面的玻璃幕牆有序地排列成中國古代錢幣的形狀。

而「'''大豐銀行大廈'''」則位於[[新馬路|新馬路]]296號，於1974年落成啟用，是新總部落成前的總部，現在是大豐銀行的新馬路分行。

== 分行 ==
目前大豐銀行除了總行之外，在澳門設有25間分行，其中20間位於[[澳門半島|澳門半島]]，4間位於[[氹仔|氹仔]]，1間位於[[路環|路環]]。

2017年更首次在中國內地成立第一間上海市分行，主要從事在澳門特別行政區獲許可經營之業務。

== 汽車櫃員機 ==
大豐銀行新總行大廈內設有兩部汽車[[櫃員機|櫃員機]]，是[[港澳地區|港澳地區]]首創。汽車櫃員機服務於1997年11月開始啟用，位於新總行大廈內的汽車通道，其位置特別設計，配合駕駛者可以安坐車中，亦可使用自動櫃員機。

== 大豐錶行 ==
大豐錶行是大豐銀行其下的附屬公司，業務是銷售高檔次的手錶和時鐘，地址是大豐銀行新馬路分行地下。
