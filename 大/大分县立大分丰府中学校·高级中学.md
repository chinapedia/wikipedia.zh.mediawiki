== 历史 ==
* 1985年4月1日 - 大分县教育厅内设置，大分新设高级中学成立准备室。
* 1985年10月5日 - 决定校名。
* 1986年4月1日 - 建校。
* 1995年9月30日 - 纪念仪式举行创立10周年。
* 1997年4月8日 - 引进65分授课，2期制。
* 2000年11月21日 - 纪念仪式举行创立15周年。
* 2005年9月29日 -　纪念仪式举行创立20周年。
* 2007年4月11日 - 大分县立大分丰府中学校建校，综合初高中。

== 初中的教育课程 ==
其他的公立初中的课程没有一样的教育课程存在。
* 授课前早上，进行「英语会话练习」「计算练习」「词汇练习」等等。这个叫「丰府时间」。

=== 指导的重点 ===
* 通过推进由于初中生和高中生的异年龄集体的交流活动的事，丰富的哺育人性和社会性。
* 培育尊重生命和人权的心的同时，承认每个人好致力于支撑互相的集体制作。
* 奖励兴趣小组活动，通过身心的锻炼，培养丰富的感性和健壮地活的精力和体力。
* 推进一边安排跟地域和监护人的联合，一边被打开了的学校制作。

=== 学生支持（StudySaport） ===
有被招呼稳定支持(简略sutasapo)，高一给初中学生告诉学习。被成为告诉的立场的高中生，级名次的上位5名选拔。

== 学校仪式 ==
* [[文化节|文化节]]，合起[[体育节|体育节]]招呼「丰饶节」。不过，7月做为「文化的部类」，把做为9月「体育的部类」。
* 3年级，入学考试前的11月去到，[[城岛后乐园|城岛后乐园]]。
* 2年级　修学旅行去[[新西兰|新西兰]]。
* 1年级，2年级　参加[[马拉松|马拉松]]比赛。每年2月被召开。
* 夏季「丰饶节体育的部类(体育节)」和「丰饶祭文化的部类(文化节)」。
* 以初中三年级，有体验在日本第一的[[修学旅行|修学旅行]]活动。大分丰府中学校第一次的修学旅行，第一天和第二天去[[富士山|富士山]]登山。　第三天采访等向在东京都内自主进修(大分县出生的企业被活跃)。　第四天是按路线的是不是进修([[东京迪士尼乐园|东京迪士尼乐园]]，[[上野|上野]]方面)。

== 特色 ==
* 午饭是由于盒饭自备。只牛奶是提供供给饮食,不过，高中内购买部和学校食堂，也能利用[[自动贩卖机|自动贩卖机]]。
* 也有在大分县内新奇的中间高联合的俱乐部活动。是茶道小组，乐队，报纸部，戏剧小组(文化部多)那个一个例子。

== 兴趣小组活动 ==
* 体育部特别[[击剑|击剑]]有名，全国称霸也完成着。

== 设施 ==
=== 学生食堂 ===
在县内的普通科的学校新奇地有[[学生食堂|学生食堂]]。营业时间以只午休，下课后没营业。也有从那个宽度时常会议室替代被利用的事。

=== 购买部 ===
与学生食堂另外购买部被设置。营业时间到下午2点短。购买[[文具|文具]]和[[面包|面包]]，能利用[[复印机|复印机]]。

== 学校生活 ==
禁止[[手机|手机]]。只是，因为到在初中普通科高中学区成为全县一区之前的旧第3封学区（[[大分市|大分市]]·[[由布市|由布市]]，該校也这个区域内）的人也准许，最近从学生意见被发出了「应该准许」。
