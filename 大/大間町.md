'''大間町'''（{{jpn|j=大間町|hg=おおままち|rm=Ōma machi|f=y}}）位於[[日本|日本]][[青森縣|青森縣]]的[[下北半島|下北半島]]北端，是[[本州|本州]]最北端的[[町|町]]。面向[[津輕海峽|津輕海峽]]，在天氣良好時可看見對岸的[[北海道|北海道]]；此處漁民擅長捕釣[[黑鮪魚|黑鮪魚]]，皆以[[聲納|聲納]]追埔魚群動向，並被[[日本電視台|日本電視台]]拍成影集；年齡最大的釣鮪魚民是出生於1927年的七島啟一先生，到78歲仍出海釣200公斤的黑[[鮪魚|鮪魚]]。<br>

此外，同樣的町名出現在[[群馬縣|群馬縣]]大間間町（おおまままち）。不過，在2006年（平成18年）3月27日因合併為[[綠市|綠市]]而消失。

[[大間核能發電廠|大間核能發電廠]]位於此町，正在建設中。

== 歷史 ==
* 1889年（明治22年）4月：施行町村制。大間村、奥戶村合併為'''大奧村'''。
* 1942年（昭和17年）11月3日：町制施行、改稱為'''大間町'''。

== 地理 ==
* 位於下北半島的西北面，亦是本州最北面的町。北面的大間崎面向津輕海峽，因此在該地設置了大間埼燈塔。與津輕海峽對岸的北海道汐首岬（[[函館市|函館市]]）相距只有約20 km。
* 由於地理上原因，能夠從太平洋看到日出，日本海看到日落。

== 行政 ==
町長：金澤滿春 

== 產業 ==
漁業及觀光為主要產業。2015年電源開發大間核能發電所預定開始經營。<br>
由於近幾年的「大金槍魚」的新聞提升了本町知名度，成為受注目的旅遊景點。
