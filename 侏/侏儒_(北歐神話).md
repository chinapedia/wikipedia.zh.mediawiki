'''侏儒'''（北歐地區寫法：Dvergr<sup>單數</sup>／Dvergar<sup>眾數</sup><small>(儒威爾格)</small>、Svartálfar<small>(黑精靈)</small>、Døkkálfar<small>(暗精靈)</small>）<ref>英語︰Dwarf、古德語︰Zwerc。</ref>是許多[[奇幻|奇幻]]作品中虛構生物「[[矮人|矮人]]」（{{lang|en|Dwarf}}）的原型。和另一種虛構生物「[[侏儒_(龍與地下城)|侏儒]] / [[地侏|地侏]] / [[地精|地精]]」（{{lang|en|Gnome}}）則不盡相同<ref>「矮人」、「侏儒」或「黑暗精靈」等譯名和很多神話或奇幻作品重疊，但實際上卻又各有不同，所以閱讀使用時需配合不同作品來做分別。</ref>。亦有翻作「小矮人」、「地靈」、「黑妖精」。在《[[龍與地下城|龍與地下城]]》的設定中，則翻譯成「灰矮人」或「[[杜爾加矮人|杜爾加矮人]]」。

北歐'''侏儒'''是[[北歐神話|北歐神話]]（[[日耳曼|日耳曼]][[神話|神話]]）中的重要的特殊創造物，他們習慣住在山裡、地底或礦井，喜歡[[石頭|石頭]]、[[冶金|冶金]]，具有高超的[[工藝|工藝]]技巧，擅長[[魔法|魔法]]、[[盧恩|盧恩]]文字（Runes），是許多神器的製造者。另外，北歐'''侏儒'''（Dwarf）在13世紀以前的作品中，皆沒有被描述為矮小的樣子，想像中他們應該和人類約同等大小。後來在[[民間傳說|民間傳說]]和[[文學|文學]]中，漸漸因為趣味或[[幽默|幽默]]的理由而逐漸矮化。

== 特徵 ==
北歐神話中描述他們的外表：皮膚為蒼白如屍體，頭髮及鬍鬚為黑色。早期神話中，他們的身高和一般人差不多，甚至也可以高大的。他們還害怕[[陽光|陽光]]，被照到時會變成[[石頭|石頭]]。一個故事中提到[[索爾|索爾]]（{{lang|en|Thor}}）為了阻止侏儒[[阿爾維斯|阿爾維斯]]（Alvis）和他[[女兒|女兒]]結婚，以測試阿爾維斯的[[智慧|智慧]]為藉口問了一晚上的問題。當[[太陽|太陽]]升起時，這個侏儒就因為照到陽光而化為[[石頭|石頭]]。

後來的[[歐洲|歐洲]][[民間傳說|民間傳說]]中形容他們為住在地底的醜陋小矮人，很容易和[[地精|地精]]（{{lang|en|Gnome}}）混淆，區分的方法是侏儒身體比例不正常，而[[地精|地精]]的肢體是正常的。侏儒身形扭曲，[[腿|腿]]短而[[手|手]]長，長著大腦袋和一張飽經風霜的[[臉孔|臉孔]]。他們雖然住在地底下，但有時還會出來舉行慶典。他們對[[人類|人類]]很有戒心，偶而會佔住人類的房子，如果屋主拒絕他們，侏儒們就會給人們帶來厄運。

== 誕生和住所 ==
在[[北歐神話|北歐神話]]中，提到侏儒的出現，共有兩種完全不同的說法︰

《[[詩體埃達|詩體埃達]]》中描述侏儒是[[奧丁|奧丁]]和他的兄弟[[威利|威利]]、[[菲|菲]]利用巨人[[布里米爾|布里米爾]]（Brimir）的血液和[[布萊恩|布萊恩]]（Blain）的四肢所做的，第一個侏儒為[[摩索尼爾|摩索尼爾]]（Motsognir），第二個是[[杜林_(北歐神話)|杜林]]（Durin）。

在《[[散文埃達|散文埃達]]》（{{lang|en|Prose Edda}}）中提到，侏儒是由[[尤彌爾|尤彌爾]]（Ymir）的屍體上的[[蛆|蛆]]變化而成的。當[[奧丁|奧丁]]（{{lang|en|Odin}}）和他的兄弟殺死[[尤彌爾|尤彌爾]]後，他的屍體上長出兩種不同的[[蛆|蛆]]，一種白皙且溫和，另一種則色黑且狡詐，[[白色|白色]]的蛆變成了精靈，而[[黑色|黑色]]的蛆就變成了侏儒。

奧丁讓他們住在[[斯瓦塔爾法海姆|斯瓦塔爾法海姆]]（Svartalfaheim，侏儒之國），另外據說還有一居住地叫[[尼德威阿爾|尼德威阿爾]]（Nidavellir，矮人之鄉）。因為記載混亂是北歐神話常有的特色，目前我們不能確定這兩個地名是否其實是同一個地區。

== 工藝 ==
侏儒都是優秀的[[工藝|工藝]]師、[[冶金|冶金]]匠、[[寶藏|寶藏]]的守護者。他們通曉[[預言|預言]]、[[魔法|魔法]]、[[盧恩|盧恩]]文字（Runes）的能力，使他們做出來的任何[[飾品|飾品]]或[[武器|武器]]都帶有很高的[[魔力|魔力]]。

重要的代表有：
*[[安德華拉諾特|安德華拉諾特]]（Andvarinaut）。
*[[布里希嘉曼|布里希嘉曼]]（Brisingamen）：[[弗蕾亞|弗蕾亞]]（Freya）的[[項鏈|項鏈]]。
*[[德羅普尼爾|德羅普尼爾]]（Droupnir）：[[奧丁|奧丁]]（{{lang|en|Odin}}）的聚金[[指環|指環]]。
*[[格萊普尼爾|格萊普尼爾]](Gleipnir)。
*[[永恆之槍|永恆之槍]]（Gungnir）：[[奧丁|奧丁]]（{{lang|en|Odin}}）的[[武器|武器]]。
*[[雷神之鎚|雷神之鎚]]（Mjolnir）：[[索爾|索爾]]（{{lang|en|Thor}}）的[[武器|武器]]。
*[[斯基德普拉特尼|斯基德普拉特尼]]（Skidbladnir）：[[弗雷|弗雷]]（{{lang|en|Freyr}}）的船。
*[[提爾鋒|提爾鋒]]（Tyrfing）︰又譯「斬裂劍」，是把出鞘就要殺人的魔劍。
等等。
