'''清明上河园'''是一处位于[[河南省|河南省]][[开封市|开封市]][[龙亭湖|龙亭湖]]西岸的[[宋代|宋代文化]][[主题公园|主题公园]]，同时也是[[中国国家5A级旅游景区|中国国家5A级旅游景区]]和中国非物质文化遗产展演基地。它是以画家[[张择端|张择端]]的写实画作《[[清明上河图|清明上河图]]》为蓝本，按照《[[营造法式|营造法式]]》为建设标准，以宋朝市井文化、民俗风情、皇家园林和古代娱乐为题材，以游客参与体验为特点的文化主题公园。<ref name="xlwqm">{{cite web|url=http://finance.sina.com.cn/roll/20110119/20509280586.shtml|title=清明上河园荣膺国家5A级旅游景区|publisher=中国产经新闻报|date=2011年1月19日| language =zh-hans }}</ref>2009年，清明上河园成为中国世界纪录协会中国第一座以绘画作品为原型的仿古主题公园。<ref name="khn">{{cite web|url=http://news.gxnews.com.cn/staticpages/20100526/newgx4bfcf1df-2976932.shtml|title=走访清明上河园 感受开封古都文化|publisher=广西新闻网|date=2010年5月26日| language =zh-hans }}</ref> 

== 景区 ==
清明上河园占地600余亩，其中水域面积约120亩，建筑面积1万平方米。双亭桥将景区分为南苑和北苑。<ref name="jqjd">{{cite web|url=http://travel.dili360.com/ztx/2011/060219683.shtml|title=清明上河园：匠心独具精雕细刻|publisher=中国旅游报|date=2011-06-02|language=zh-hans}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>整个景区仿照《清明上河图》，设有驿站、民俗风情、特色食街、宋文化展示、花鸟鱼虫、繁华京城、休闲购物和综合服务等八个功能区，并设有校场、虹桥、民俗、宋都等四个文化区。<ref>{{cite web|url=http://www.china.com.cn/city/zhuanti/9henan/2010-05/26/content_20124824.htm|title=清明上河园采风，重温汴京之繁荣|publisher=中国网|date=2010-05-26| language =zh-hans }}</ref>

=== 南苑景区 ===
[[File:Kaifeng1.jpg|right]]
南苑景区主要表现北宋的[[民俗|民俗]]风情和市井文化。南苑的设计源于宋代《清明上河图》，不同之处在于把园林设计成“如意结”。[[汴河|汴河]]蜿蜒如带，形似一个硕大的如意结，串连起公园内的各个景点，使各个景区既相互独立，又血脉相连。而中国十大名桥之一的虹桥是如意结的中点，亦是清明上河园的核心部位。虹桥高5米，宽10米，跨径25米，这样的设计规模在宋代比较少见。<ref name="jqjd"/>

虹桥以东是迎宾主题区，有张择端雕像矗立。虹桥以东是住宿休息区，有醉杏楼、文绣院、仿古驿站等建筑。虹桥以西是民风民俗区，有工艺和小吃一条街。虹桥以北是游艺活动区，在此可以观赏宋朝精彩表演及各种绝活。

=== 北苑景区 ===
北苑呈现的则是[[皇家园林|皇家园林]]和古代娱乐主题。按照[[清代|清代]]清院版的《清明上河图》进行构造，其主要景观全部围绕着景龙湖为中心。景区包括临水大殿、茗春坊、九龙桥、拂云阁等景点，其中临水大殿由宣和殿、宣德殿两殿连接而成；水心榭四面临水，一桥相连；丹东宫由造纸馆，织锦院，司南坊，大宋官窑，火药馆等组成。<ref name="jqjd"/>

== 大宋.东京梦华 ==
《大宋·东京梦华》是在清明上河园皇家园林区的景龙湖上举行的大型水上实景演出项目。该项目由[[梅帅元|梅帅元]]担任导演，全剧共70分钟，有700多名演员参与演出。<ref name="khn"/>演出把张择端的《清明上河图》长卷和[[孟元老|孟元老]]的《[[东京梦华录|东京梦华录]]》中的相关记载作为切入点，以八阕经典[[宋词|宋词]]蕴含的意境为主线，以亭台楼榭、水系桥廊为背景，生动再现了宋代都城的繁华盛景和宋朝的辉煌。<ref>{{cite web|url=http://culture.people.com.cn/GB/87423/9156140.html|title=品《大宋·东京梦华》|publisher=人民网|date=2009年4月20日| language =zh-hans }}</ref>版本可分为[[日间|日间]]现实版和[[夜间|夜间]]梦幻版。<ref>{{cite web|url=http://www.people.com.cn/h/2011/0920/c25408-2411345078.html|title=5A清园 “文化”赢世界|publisher=人民网|date=2011年9月20日| language =zh-hans }}</ref>，夜间版演出通过运用亮彩技术，映射出流光溢彩的“柳树”和随波荡漾的“水中明月”，运用机械技术设计的可自由拼合的水上栈桥，可“绽放”的菊花舞台等，制造出梦幻般的场景效果。<ref>{{cite web|url=http://www.henan.gov.cn/bsfw/system/2008/05/09/010071765.shtml|title=开封实景演出《大宋东京梦华》塑休闲之都|publisher=河南省政府网|date=2008年5月9日| language =zh-hans }}</ref>
