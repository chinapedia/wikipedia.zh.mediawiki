'''清水崇'''（日語名：清水崇，1972年7月27日-）是一名[[電影|電影]][[導演|導演]]。他出生於[[日本|日本]][[群馬縣|群馬縣]][[前橋市|前橋市]]。拍攝作品以[[恐怖片|恐怖片]]類型居多，被譽爲「日本恐怖大師」。

== 生活 ==
孩童時期清水崇因觀賞[[史蒂芬·史匹柏|史蒂芬·史匹柏]]的科幻影片《[[E.T.外星人|E.T.外星人]]》所影響，而興起拍攝電影的念頭，<ref>{{Cite web |url=http://tw.movie.yahoo.com/movieheadline/d/a/100316/3/j8y.html |title=恐怖大師清水崇從小愛嚇人 3D鬼片再創新世紀恐怖經典 |publisher=[[Yahoo!奇摩|Yahoo!奇摩]] |date=2010年3月16日 |accessdate=2014年1月10日 |deadurl=yes |archiveurl=https://web.archive.org/web/20140110182337/http://tw.movie.yahoo.com/movieheadline/d/a/100316/3/j8y.html |archivedate=2014年1月10日 }}</ref>之後在學生時代的[[青春期|青春期]]裡，清水崇便嘗試製作恐怖影片。當時清水崇曾構思一部長度數分鐘、名叫《家庭訪問》的恐怖短篇，該短片後續則成為他在拍攝電影《[[咒怨|咒怨]]》的[[靈感|靈感]]來源。<ref>{{Cite web |url=http://www.geneon-ent.co.jp/movie/column/article/vol27_200307.html#03 |script-title=ja:呪怨 劇場版 デラックス版 |publisher=Geneon Entertainment INC. |accessdate=2014年1月10日 |deadurl=yes |archiveurl=https://web.archive.org/web/20150924021357/http://www.geneon-ent.co.jp/movie/column/article/vol27_200307.html#03 |archivedate=2015年9月24日 }}</ref>

在未讀完[[大學|大學]]而決定輟學的清水崇，於導演[[黑澤清|黑澤清]]與[[高橋洋|高橋洋]]的推薦幫助下，為恐怖電視影集{{nihongo|《學校怪談G》|学校の怪談G}}拍攝{{nihongo|《角落》|片隅}}、《44444444444》兩部短片。<ref>{{Cite web |url=http://ent2.excite.co.jp/cinema/interview/thejuon/interview01.html |script-title=ja:『THE JUON/呪怨』清水崇監督インタビュー |publisher=Excite Japan |accessdate=2014年1月10日}}</ref>之後在1999年則製作了《咒怨》、《咒怨2》兩部[[錄影帶首映|錄影帶首映]]原創影集。

2001年清水崇發表個人首部執導的電影作品《[[富江3怨念再生|富江3怨念再生]]》，該影片系列改自於恐怖漫畫家[[伊藤潤二|伊藤潤二]]筆下的詭異女子角色「[[富江|富江]]」。<ref>{{Cite web |url=http://pff.jp/26th/tokyo/over_the_sea.html |script-title=ja:国境を越えた日本映画～映画監督・清水崇 |publisher={{lang|ja|26回 ぴあフィルムフェスティバル}} |accessdate=2014年1月10日 }}</ref>次年清水崇將先前拍攝的原創恐怖影集《咒怨》改拍成電影上映，隨後電影版《咒怨》在票房市場上獲得好評接續推出續集《咒怨2》；清水崇並獲得西方[[好萊塢|好萊塢]]影藝界的邀請，親自前往國外拍攝西洋版的《咒怨》系列作品─《[[不死咒怨|不死咒怨]]》及《[[鬼謎藏|鬼謎藏]]》；<ref>{{Cite web |url=http://www.cinematopics.com/cinema/works/output2.php?oid=4669 |script-title=ja:史上初！日本人監督　全米NO.1！！恐怖は、世界に増殖している―。 |publisher=Cinema Topic Online |accessdate=2014年1月10日 }}</ref><ref>{{Cite web |url=http://www.cinematoday.jp/page/N0033292 |script-title=ja:『呪怨』清水崇監督、次回作はハリウッドで！CBSフィルムスが正式発表 |publisher=Cinema Today |date=2011年6月24日 |accessdate=2014年1月10日}}</ref>並分別於2004及2006年推出。其中《不死咒怨》曾創下連續兩週[[北美|北美]]票房第一的成績。<ref>{{Cite web |url=http://movies.yahoo.co.jp/interview/200502/interview_20050221001.html |script-title=ja:『THE JUON　呪怨』清水崇監督独占インタビュー |publisher=[[Yahoo|Yahoo]] Japan |accessdate=2014年1月10日 |deadurl=yes |archiveurl=https://web.archive.org/web/20140110170131/http://movies.yahoo.co.jp/interview/200502/interview_20050221001.html |archivedate=2014年1月10日 }}</ref>

2007年清水崇執導日本首部具備3D立體視覺效果的恐怖影片《[[戰慄迷宮3D|戰慄迷宮3D]]》，<ref>{{Cite web |url=http://www.creators-station.jp/seehim/806 |script-title=ja:日本初の実写長編デジタル３D映画『戦慄迷宮　３D』、とうとう完成！ |publisher={{lang|ja|クリエイターズステーション}} |date=2009年8月21日 |accessdate=2014年1月10日 }}</ref>並也至海外在[[香港|香港]]影星[[麥浚龍|麥浚龍]]於2013年推出的《[[殭屍_(電影)|殭屍]]》影片中擔任監製一職。<ref>{{Cite web |url=http://www.appledaily.com.tw/realtimenews/article/new/20131108/289064/ |title=麥浚龍《殭屍》　勾三池崇史合作 |publisher=[[蘋果日報_(台灣)|蘋果日報]] |author=張哲鳴 |date=2013年11月8日 |accessdate=2014年1月10日}}</ref>除了拍攝恐怖影片為主外，清水崇也擔任日本兒童文學作家[[角野榮子|角野榮子]]的奇幻小說《[[魔女宅急便_(小說)|魔女宅急便]]》改編的[[魔女宅急便_(2014年電影)|同名真人電影版]]導演。<ref>{{Cite web |url=http://www.oricon.co.jp/news/2025427/ |script-title=ja:実写『魔女の宅急便』おソノ役は尾野真千子　清水崇監督はオリジナル世界観に自信  |publisher=[[Oricon|Oricon]] |date=2013年6月10日 |accessdate=2014年1月10日}}</ref>
