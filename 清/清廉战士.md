《'''清廉战士'''》是[[中华人民共和国|中华人民共和国]]首部以廉政文化为主题的大型[[2D|2D]][[网络游戏|网络游戏]]，由[[浙江省|浙江省]][[宁波市|宁波市]][[海曙区|海曙区]]纪委和海曙区街道党工委等单位负责开发。游戏于2007年7月25日公开测试，2007年8月14日宣布关闭，公开运营不到20天。

== 游戏内容 ==
玩家扮演一位“清廉战士”，不断接到任务斩杀[[贪官|贪官]]，同时玩家还会遭到贪官卫兵、[[情妇|情妇]]、[[子女|子女]]等的攻击报复。玩家以此积累经验值，最后使社会变成“清廉仙境”。

游戏中的165个人物原型均来自[[历史|历史]]记载的人物，既有斩妖除魔的忠臣青天，也有倒行逆施的贪官污吏。贪官奸佞按照贪欲指数设定贪官的着装和难度攻克级别。

游戏大部分游戏地图以宁波建筑为原型，游戏中人物使用的道具和药品也是宁波风味。<ref name=sina></ref>
==开发==
2005年年初，中国各地掀起廉政文化的建设热潮。海曙区纪委和海曙区西门街道党工委领导提出能否研发一款以“反腐倡廉”为主题的网络游戏，使得廉正文化建设工程不仅可以全民参与，而且可以影响和教育青少年。<ref name=qq>{{Cite web|url =http://news.qq.com/a/20070907/002778.htm|title=网游《清廉战士》20天生存身前身后事|publisher = 腾讯新闻|date = 2007-09-07 |accessdate = 2017-04-20}}</ref>经过讨论后，决定启动一项由海曙区纪委出资、海曙区西门街道党工委出人、团海曙区委把关的大型网络游戏研发工程。游戏最终命名为《清廉战士》。游戏爱好者华彤初任研发小组主要成员。游戏开发花费十万元[[人民币|人民币]]。<ref name=qq></ref><ref name=sohu>{{Cite web|url =http://news.sohu.com/20070913/n252125703.shtml|title=网游《清廉战士》猝死之谜|publisher = 搜狐新闻|date = 2007-09-13 |accessdate = 2017-04-20}}</ref>

6名电脑游戏爱好者经过2年多的摸索和制作，《清廉战士》于2007年6月完成。经过修改、调试后，最终决定7月25日正式对外开通。<ref name=qq></ref>

==运营==
游戏方声明，《清廉战士》是纯公益游戏，永不收费，且不出售道具。为了防止游戏沉迷，游戏进行两小时后自动断网。<ref name=qq></ref>

2007年7月25日，游戏公开测试，有300多名玩家在线注册，之后又陆续有7000多人注册，原设计供五六百人同时在线的服务器无法承受。<ref name=sina>{{Cite web|url =http://news.sina.com.cn/c/2007-08-30/111913778418.shtml|title=网游《清廉战士》猝死之谜|publisher = 新浪新闻|date = 2007-08-30 |accessdate = 2017-04-20}}</ref>7月31日，由于服务器不堪重负，游戏方宣布停运升级。<ref name=qq></ref>8月10日，该游戏的发布网站“中国廉政文化游戏网”通知，进行软件、硬件升级。<ref name=techsina>{{Cite web|url =http://tech.sina.com.cn/h/2007-08-21/1324397430.shtml|title=每日一贫：夭折的清廉战士，钟为谁鸣？|publisher = 新浪科技|date = 2007-08-21 |accessdate = 2017-04-20}}</ref>

2007年8月14日，游戏停止运营。“中国廉政文化游戏网”对外宣布：“接有关部门通知，本网站将关闭。”<ref name=sina></ref>

在游戏停运之前，在线人数最多时过千，注册人数过万。<ref name=techsina></ref><ref name=sina></ref>

== 反响 ==
由于游戏的制作团队和技术等限制，游戏的制作相当粗糙，游戏中的人物头像、贴图、音乐大多复制自其他游戏。<ref>{{Cite web|url =http://news.qq.com/a/20070816/000589.htm|title=《反腐败网络游戏《清廉战士》网站开张半月后关闭|publisher = 腾讯新闻|date = 2007-08-16 |accessdate = 2017-04-20}}</ref>

游戏本身存在许多[[程序错误|程序错误]]，在游戏中不能看到自己和其他玩家的形象，游戏界面只是一大片无人的建筑。<ref name=sohu></ref>

[[游戏|游戏]]还存在着不少知识性错误，如在[[康熙皇帝|康熙皇帝]]“要求獲得20個[[李林甫|李林甫]]人頭”的小任務中，就有四個[[錯別字|錯別字]]與一個歷史差錯（康熙皇帝是[[清朝|清朝]]人，李林甫是[[唐朝|唐朝]]人）。<ref name=sohu></ref>

游戏中通过杀死贪官、情妇、儿女的方式进行有悖于现代法律的血腥执法也引发了争论。<ref>{{Cite web|url =http://news.sina.com.cn/o/2007-08-16/024412391669s.shtml|title=《清廉战士》：网游界的芙蓉姐姐|publisher = 新浪新闻|date = 2007-08-16 |accessdate = 2017-04-20}}</ref>

==相关游戏==
《清廉战士》设计者华彤（时任海曙区西门街道党工委下属龙柏社区党支部书记）2007年接受《[[南方都市报|南方都市报]]》采访时说，在开发该网游前，他们就已经成功推出了一系列廉政单机游戏，此次趁《清廉战士》上线，也还推出其他游戏供网友免费下载。这些游戏包括《官场人生》、《廉侠除贪记》、《打倒贪官》、《廉政象棋》、《清官连连看》、《清官道》等，其中的《廉侠除贪记》可说是《清廉战士》的前身。<ref name=n070803 /><ref>{{cite web|author1=IT之家|title=早在《人民的名义》十年前，网游就被用来反腐|url=https://www.sohu.com/a/134275671_114760|website=搜狐|accessdate=2017-10-30|date=2017-04-16|deadurl=yes|archiveurl=https://web.archive.org/web/20171107013945/https://www.sohu.com/a/134275671_114760|archivedate=2017年11月7日|df=}}</ref><ref>{{cite news|title=小游戏里的大用心|url=http://epaper.oeeee.com/epaper/A/html/2014-01/12/content_2537886.htm|accessdate=2017-10-30|agency=南方都市报|publisher=南都网|date=2014-01-12}}</ref>而养成类游戏《官场人生》也在一时间备受网友关注，游戏内容是大学刚毕业的主人公“姚清廉”在任职公务员的过程中保持清廉、升官晋级的故事。<ref name=n070803>{{cite web|author1=南方都市报|title=廉政游戏《官场人生》学习文件升官晋级|url=https://news.sina.com.cn/o/2007-08-03/095912321167s.shtml|website=新浪|accessdate=2017-10-30|date=2007-08-03}}</ref>
