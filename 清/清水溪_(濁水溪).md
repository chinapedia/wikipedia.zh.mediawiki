'''清水溪'''，原名'''阿拔泉溪'''<ref name="竹山鎮公所"/><ref name="王豐仁"/>，位於[[台灣|台灣]]中部，為[[濁水溪|濁水溪]]的支流，河長51公里，流域分佈於[[雲林縣|雲林縣]]、[[南投縣|南投縣]]、[[嘉義縣|嘉義縣]]。由於清水溪在觸口山與濁水溪匯合時，清水、濁水同流，因為濁水的水量比較大，顏色又濃，清水混合濁水，一清一濁，故得名<ref name="走讀台灣"/>。

清水溪的上游為[[阿里山溪_(嘉義縣)|阿里山溪]]，發源於[[阿里山脈|阿里山脈]]標高2,663公尺的[[大塔山|大塔山]]<ref>[http://www.alishan.gov.tw/content/index.asp?m=1&m1=4&m2=86 阿里山鄉公所：地理與環境] {{webarchive|url=https://web.archive.org/web/20081007144753/http://www.alishan.gov.tw/content/index.asp?m=1&m1=4&m2=86 |date=2008-10-07 }}</ref>，其主流至[[豐山|豐山]]附近與[[石鼓盤溪|石鼓盤溪]]會合後，才始稱「清水溪」。清水溪的主流轉向西，在流經[[草嶺|草嶺]]、倒交山後，再轉向北，流經[[樟湖|樟湖]]、檳榔宅後，至[[桶頭|桶頭]]開始，河道兩度方向改變，因此出現[[河階|河階]]與[[攻擊坡|攻擊坡]]，繼續流經[[林頂|林頂]]、[[瑞竹|瑞竹]]、[[過溪|過溪]]，最後於[[觸口|觸口]]注入濁水溪主流。

== 地理 ==
清水溪流域的地形東南側較高，向西北側降低，地表岩層分布以[[中新世|中新世]]桂竹林層的[[砂岩|砂岩]]為主<ref name="王豐仁"/>。土壤為沖積土，由砂礫及粘土混合而成，土壤為暗灰色至橄欖灰色、褐黑色，黏性高，無污染，含豐富有機物質氮、磷、鉀、矽酸鎂，適宜農耕<ref name="走讀台灣"/>。

== 歷史 ==
位處清水溪上游的[[雲林縣|雲林縣]][[古坑鄉|古坑鄉]][[草嶺|草嶺]]，當地的堀坔山由於地形、岩性、地質等因素，加上[[地震|地震]]、豪雨誘發，自1862年至今已有四次[[山崩|山崩]]造成[[堰塞湖|堰塞湖]]的紀錄<ref name="濁水溪支流治理"/>。1898年（明治31年）發生[[戊戌大水災|戊戌大水災]]，[[草嶺潭|草嶺潭]]第一次潰決，造成下游的濁水溪流路北移，洪水回歸[[舊濁水溪|舊濁水溪]]故道，使舊濁水溪成為濁水溪下游的主流，洪水亦造成[[二水|二水]]、[[溪州|溪州]]、[[北斗鎮_(台灣)|北斗]]、[[田尾|田尾]]、[[埤頭|埤頭]]到[[二林|二林]]都有嚴重水患，洪水沖破河岸，北斗街全市浸水，四塊厝、沙仔崙、曾厝崙、北勢寮土地大量流失，埤頭庄、田尾鄉也災情慘重<ref name="張素玢"/>。

1941年至1942年第二次形成堰塞湖，1951年（民國40年）[[中華民國國軍|中華民國國軍]]第75軍16師工兵營奉命開鑿草嶺間的公路，以利草嶺潭開發，並協助整修草嶺潭天然土壩的坍方，於該年5月18日清晨，由於連日豪雨造成山洪暴發，土壩潰決，奪走了74名官兵的性命<ref name="濁水溪支流治理"/>。1979年（民國68年）8月15日，草嶺潭又因豪雨而形成，但很快在8月24日就因豪雨而潰決，並未造成災害<ref name="濁水溪支流治理"/>。

1999年（民國88年）發生[[921大地震|921大地震]]後，草嶺再度形成一個蓄水量4,600萬立方公尺的草嶺潭，稱為「[[新草嶺潭|新草嶺潭]]」，崩坍土方量高達1億2千萬立方公尺，土壩高約有40～50公尺，阻塞河道長達約4,815公尺，2004年（民國93年）[[敏督利颱風|敏督利颱風]]帶來大量砂石，再度把新草嶺潭填滿<ref name="濁水溪支流治理"/>。

== 交通與產業 ==
清水溪流域交通路線以[[縣道149號|縣道149號]]（鯉南路）南北貫穿其中，北起[[南投縣|南投縣]][[竹山鎮|竹山鎮]]，往西南至[[雲林縣|雲林縣]][[古坑鄉|古坑鄉]]，自竹山市街以南大致沿著清水溪河岸往上游方向前進<ref name="王豐仁"/>。此外縣道149線向東轉接投49線可鄰近[[鹿谷鄉|鹿谷鄉]]<ref name="王豐仁"/>。

本區產業以農業為主，多種植[[茶葉|茶葉]]、[[竹筍|竹筍]]、[[柳丁|柳丁]]、[[咖啡|咖啡]]等為當地特產，沿途景象多以茶園、果園或竹林為主<ref name="王豐仁"/>。
