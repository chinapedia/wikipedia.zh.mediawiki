'''清瀨市'''（{{jpn|j=清瀬市|hg=きよせし|rm=Kiyose shi|f=y}}）為位于[[東京都|東京都]][[多摩地區|多摩地區]]東北部的城市。

== 地理 ==
位於東京都伸入[[埼玉縣|埼玉縣]][[新座市|新座市]]與[[所澤市|所澤市]]之間的突出地。西邊與所澤市以{{link-ja|柳瀨川|柳瀬川}}為界，南邊與[[東久留米市|東久留米市]]以{{link-ja|野火止用水|野火止用水}}為界。市內綠地面積佔全體市域約46%を。

== 地形 ==
本市位於[[武藏野台地|武藏野台地]]東北端前方約15公里的平坦地。全市呈現約6.5km×2km的狹長形狀，其長軸與[[台地|台地]]傾斜方向相同，為西高東低的地形。西方的竹丘（たけおか）地區標高約65公尺，東北的下宿（したじゅく）地區約20公尺。市域北部為柳瀨川的沖積低地，其他則屬洪積台地。柳瀨川附近的中里（なかざと）、下宿地區比柳瀨川低地高約5〜10公尺，與清瀨旭丘團地附近的台地低約10〜15公尺，下清戶（しもきよと）、中清戶（なかきよと）、上清戶（かみきよと）地區則是與[[西武鐵道|西武鐵道]][[西武池袋線|池袋線]]周邊連成一氣的台地。從團地北側的崖面觀察地層，可見深約6公尺的關東壤土層，其下方是武藏野礫層。

== 歷史 ==
=== 原始〜江戶時代 ===
人類活動的紀錄可追溯至約兩萬年前的[[末次冰期|末次冰期]]，主要遺跡有下宿強清水（したじゅくこわしみず）遺跡與野鹽西原遺跡。[[繩文時代|繩文時代]]，人口在中期達到頂點。[[平安時代|平安時代]]後期，人口持續增加，當時人類居住地區在柳瀨川附近的野鹽、中里、下宿等地。

之後，此地成為{{link-ja|武藏七黨|武蔵七党}}{{link-ja|村山黨|村山党|村山氏}}的勢力範圍。

[[足利尊氏|足利尊氏]]開設幕府後，以[[信濃國|信濃國]]{{link-ja|佐久郡|佐久郡}}大石鄉為據點的{{link-ja|大石氏|大石氏}}將勢力伸進此地。

5世紀後半，大石氏於柳瀨川對岸、現在的所澤市設置{{link-ja|瀧之城|滝の城}}。[[戰國時代_(日本)|戰國時代]]，此地成為[[後北條氏|後北條氏]]領地。瀧之城之後在[[豐臣秀吉|豐臣秀吉]][[小田原征伐|小田原征伐]]中陷落，現僅留下「瀧之城跡」遺址。

=== 明治以後 ===
[[明治維新|明治維新]]後，清瀨由{{link-ja|武藏知縣事|武蔵知県事}}管轄。1869年，設立{{link-ja|品川縣|品川県}}。1871年，清瀨市域編入[[東京府|東京府]]與{{link-ja|入間縣|入間県}}。同年，[[多摩地域|東多摩郡除外的多摩郡]]編入[[神奈川縣|神奈川縣]]。

1889年，[[町村制|町村制]]施行，上清戶村、中清戶村、下清戶村、清戶下宿村、中里村、野鹽村與[[埼玉縣|埼玉縣]]{{link-ja|新座郡|新座郡}}野火止村一部分合併成立[[北多摩郡|北多摩郡]]'''清瀨村'''。1893年，三多摩從神奈川縣劃入東京府。

* 1915年 - 武藏野鐵道（現[[西武鐵道|西武]][[西武池袋線|池袋線]]）開通。當時此地未設站。1924年6月24日，開設[[清瀨站|清瀨站]]。
* 1939年 - 傷痍軍人東京療養所（現東京醫院）開院（日本正式開始肺結核療養）。
* 1954年4月5日 - 昇格為'''清瀨町'''。
* 1970年10月1日 - 昇格為'''清瀨市'''。

== 市名由來 ==
[[File:Kiyosesuitengu.JPG|thumb]]
市名由來有各種說法。
*由市內地名「'''清'''戶」與柳'''瀨'''川各取一字而來。
**[[日本武尊|日本武尊]]來到此地於樹下休息時，見樹下的土說出「清き土なり」，因此稱為「清土」，後改為同音的「清戶」。
**現在市內有「上清戶」、「中清戶」、「下清戶」等地名。
*市內的柳瀨川「'''清'''らかなせせらぎ（'''瀬'''ぎ）」（清澈潺潺）而得名。

== 人口 ==
{{日本人口統計|code=13221|name=清瀬市|location=Kiyose, Tokyo}}
=== 日夜間人口 ===
2005年夜間人口（居住者）為73,506人，日間活動人口總計為60,733人，是夜間人口的0.826倍。此外，依據平成22年國勢調査，往東京都[[東京都區部|特別區部]]的通勤率達33.3%。

市內往市外的通勤者有22,765人，市外往市內的通勤者僅9,915人。市內往市外的通學生有3,234人，市外往市內的通學生有3,311人，兩者相當<ref>東京都編集『東京都の昼間人口2005』平成20年発行152,153ページ</ref>。

== 產業 ==
=== 農業 ===
農地占整體市域四成。2002年（平成14年），市內有87.0%的農地被指定為生產綠地。作物以[[胡蘿蔔|胡蘿蔔]]與[[菠菜|菠菜]]等蔬菜為主。其中胡蘿蔔出貨額達都內產值的一半。因此本市有許多使用胡蘿蔔作為原料的食品。此外酪農業也十分興盛，主要集中在中清戶清瀨市役所周邊。

=== 醫療 ===
[[File:Symbol_of_Kiyose.jpg|thumb]]
此地在戰前是[[結核|結核]]患者的療養地，市內西南部有{{link-ja|結核預防會|結核予防会|財團法人結核預防會}}複十字醫院、{{link-ja|國立醫院機構東京醫院|国立病院機構東京病院}}等大型醫院。此外，市內南部也聚集許多研究設施。
