'''清华大学玉泉医院'''（清华大学第二附属医院），位于[[北京市|北京市]][[石景山区|石景山区]][[石景山路|石景山路]]5号，是[[清华大学|清华大学]]的附属医院，为二级甲等医院。

==简介==
1983年12月，[[中华人民共和国电子工业部|电子工业部]]四〇二医院（后更名为“北京玉泉医院”）正式开院，是一所向社会开放的综合性医院。该医院起初隶属[[中华人民共和国电子工业部|中华人民共和国电子工业部]]，后隶属[[中华人民共和国信息产业部|中华人民共和国信息产业部]]。2003年4月10日，北京玉泉医院和[[北京华信医院|北京酒仙桥医院]]都划归清华大学。2003年并入清华大学之后，适逢[[非典|非典]]疫情，两所医院相继被指定为北京市收治非典的定点医院。2004年4月23日，北京玉泉医院更名为清华大学玉泉医院（清华大学第二附属医院）并挂牌。<ref>{{Cite web |url = http://www.china.com.cn/info/2009-07/29/content_18227818.htm |title = 清华大学玉泉医院 |author =  |date = 2009-07-29 |publisher = 中国网 |language =  |accessdate =  |quote =  }}</ref><ref name=rmw>{{Cite web |url = http://haodf.people.com.cn/yiyuan/DE4roiYGYZwYwWNBM4FM2b44i.htm |title =  清华大学玉泉医院|author =  |date =  |publisher = 人民网 |language =  |accessdate = 2017-01-08 |quote =  }}</ref><ref>{{Cite web|url =http://www.tsinghua.edu.cn/publish/news/6664/2011/20110225231435359869515/20110225231435359869515_.html|title =清华大学成立两所附属医院|author =|date =2004-04-27|publisher =清华大学新闻网|language =|accessdate =|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20170109021849/http://www.tsinghua.edu.cn/publish/news/6664/2011/20110225231435359869515/20110225231435359869515_.html|archivedate =2017-01-09}}</ref>

该医院设有“清华大学临床神经科学研究院”和“脑神经疾病研究所”两个神经科学研究机构，主办国家级杂志《Translational Neuroscience and Clinics（英文）》。<ref name=jj>{{Cite web |url = http://www.yuquanhosp.com/news.asp?id=31 |title = 医院简介 |author =  |date =  |publisher =  清华大学玉泉医院|language =  |accessdate = 2017-01-08 |quote =  }}</ref>

该医院的重点学科是脑科中心，由神经外科、神经内科、精神卫生科组成，是[[清华大学医学院|清华大学医学院]]博士后流动站及博士学位授予点，也是[[清华大学生命科学与医学研究院|清华大学生命科学与医学研究院]]脑科学与神经疾病研究所的临床治疗中心。<ref name=rmw/><ref name=jj/>
