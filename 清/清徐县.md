'''清徐县'''在[[中国|中国]][[山西省|山西省]][[太原市|太原市]]南部，[[汾河|汾河]]纵贯，是太原市所辖的一个[[县|县]]，为中国醋都。

有民谣称：“自古酿[[醋|醋]]数山西，追源溯流在清徐”。清徐“福源昌”生产的“[[山西老陈醋|山西老陈醋]]”曾获1924年[[巴拿马|巴拿马]][[国际博览会|国际博览会]]优质商品一等奖。

== 历史 ==
1952年[[清源县_(隋朝)|清源县]]与[[徐沟县|徐沟县]]合并而成。

== 地理 ==
地势西北高，东南低。西北山地属[[吕梁山|吕梁山]]余脉，东南为平原，属[[太原盆地|太原盆地]]。

== 人口 ==
2010年第六次全国人口普查清徐县常住人口共343861人。<ref>[http://www.tyxc.gov.cn/content/2011-05/15/content_99530.htm 太原市2010年第六次全国人口普查主要数据公报]</ref>

== 方言 ==
清徐县境内方言属[[晋语|晋语]][[并州片|并州片]]方言，有五个声调，其中两个入声调，保留有喉塞音的入声韵尾。

== 经济 ==
2011年全县实现地区生产总值（GDP）117.26亿元，人均地区生产总值33989元。三次产业结构比为12.4：65.4：22.2。<ref>[http://www.qxtj.gov.cn/show.asp?id=607 清徐县2011年国民经济和社会发展统计公报]{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>
