'''清水佐紀'''（1991年11月22日—）是[[日本|日本]]的女性[[偶像|偶像]]、[[歌手|歌手]]及[[藝人|藝人]]。[[神奈川縣|神奈川縣]][[座間市|座間市]]出身，是「[[Hello!_Project_Kids|Hello! Project Kids]]」的成員之一。屬於[[UP-FRONT|UP-FRONT]]經紀公司。身高為152.9公分。愛稱為「{{lang|ja|キャプテン}}」、「{{lang|ja|さきちゃん}}」、「{{lang|ja|しみハム}}」等。

== 特徵 ==
# 清水佐紀是Berryz工房中年紀最大的。由於很會記憶舞步，因此當上隊長（Captain）。其他的成員也都稱她為「Captain」。
# 在某一時期的MC自我介紹「最嬌小的清水佐紀」，不過她後來超過149cm的嗣永桃子成為150cm(現在身高為152.5 cm)。
# 是在東京都出生，不過現在是在神奈川縣座間市居住。因為是在東京都內的醫院出生，但在座間市成長，實質上的出生地於是被歸為座間市。
# 2005年秋天的旅行「スイッチＯＮ！」衣裳被安排號碼43(清水(しみず)的「しみ」諧音)。2006年新年召開的「Hello! Project 2006Winter ～ワンダフルハーツ～」與同年春天旅行「にょきにょきチャンピオン！」也就繼承這個號碼。
