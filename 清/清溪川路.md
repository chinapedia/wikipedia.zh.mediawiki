'''清溪川路'''位於[[首爾特別市|首爾特別市]][[鍾路區|鍾路區]]，從[[東華免稅店十字路|東華免稅店十字路]]（[[太平路1街|太平路1街]]<ref>{{cite web |url=http://big5chinese.visitkorea.or.kr/cht/SI/SI_CH_2_1_2_1.jsp?cid=332882 |title=清溪川路 (청계천길) |publisher=韓國旅遊官方網站 |accessdate=2011-02-04 |archive-url=https://web.archive.org/web/20090523192320/http://big5chinese.visitkorea.or.kr/cht/SI/SI_CH_2_1_2_1.jsp?cid=332882 |archive-date=2009-05-23 |dead-url=yes }}</ref>）到位於[[東大門區|東大門區]][[踏十里洞|踏十里洞]]的[[新踏鐵橋|新踏鐵橋]]，全程雙程六線行車，全長5.9公里。以[[黃鶴橋|黃鶴橋]]為基準，西北側是[[鍾路區|鍾路區]]，西南側是[[中區_(首爾)|中區]]，東北側是[[東大門區|東大門區]]，而東南側是[[城東區|城東區]]。

== 歷史 ==
*1966年11月：從[[東華免稅店十字路|東華免稅店十字路]]到[[興仁之門|興仁之門交叉路]]之間興建長約2.85公里的覆蓋道路。
*1970年11月：到今日的[[古山子橋|古山子橋]]<ref>{{cite web |url=http://cafe251.daum.net/_c21_/bbs_search_read?grpid=19U7o&fldid=M3rM&contentval=0000Jzzzzzzzzzzzzzzzzzzzzzzzzz&nenc=mEGi1nuAncpsxWNA-drMAQ00&fenc=cy4yNaH.v4o0&q=&nil_profile=cafetop&nil_menu=sch_updw |publisher=Daum 카페: 홍순원카페 |title=청계천 이야기 |trans_title=清溪川的故事 |accessdate=2011-02-04 |date=2007-06-07 }}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>部份的覆蓋道路完工。
*1984年11月：道路名稱改為清溪川路。
