《'''清明上河圖'''》，目前已知最早的版本爲[[北宋|北宋]]畫家[[張擇端|張擇端]]所作，原畫長約5米，高24.8厘米，現藏[[北京|北京]][[故宫博物院|故宫博物院]]。《清明上河圖》描绘<ref> 楊新：〈[http://www.dpm.org.cn/shtml/400/@/97260.html 《清明上河圖》贊] {{Wayback|url=http://www.dpm.org.cn/shtml/400/@/97260.html |date=20091128093230 }}〉，載2004年12月20日，[[故宮博物院|故宮博物院]]網站。</ref>北宋京城[[汴梁|汴梁]]（今[[河南省|河南省]][[开封市|开封市]]）及[[汴河|汴河]]两岸的繁华和热闹的景象和优美的自然风光。作品以长卷形式，采用[[透视|散点透视]]的[[构图法|构图法]]，将繁杂的景物纳入统一而富于变化的画卷中，畫中主要分開兩部份，一部份是[[農村|農村]]，另一部是市集。畫中約莫有814人，[[牲畜|牲畜]]60多匹，[[船|船]]隻28艘，[[房屋|房屋]][[樓宇|樓宇]]30多棟，車20輛，[[轎|轎]]8頂，[[樹木|樹木]]170多棵，往來衣着不同，神情各异，栩栩如生，其间还穿插各种活动，注重情节，构图疏密有致，富有节奏感和韵律的变化，笔墨章法都很巧妙，颇见功底。這幅畫作對於各種形態的幾乎正確描繪性使其負有盛名。《清明上河圖》是中國十大傳世名畫之一{{notetag|中國十大傳世名畫為：

== 原作 ==
《清明上河圖》的作者是[[宋徽宗|宋徽宗]]朝任朝[[翰林|翰林]]畫院畫史[[張擇端|張擇端]]，也有一說作者不止[[張擇端|張擇端]]一人；[[董其昌|董其昌]]《容臺集》說：「乃[[南宋|南宋]]人追憶故京之盛，而寓[[清明|清明]]繁盛之景，傳世者不一，以張擇端所作為佳。」「[[绍兴_(南宋)|紹興]]初，故老閒坐，必談[[京師|京師]]風物，聽之感慨有流涕者，故其時西北耆舊，談宣政故事者，為人所重。」甚至有人提出观点说是[[南宋|南宋]]人懷念昔日強盛時期而作。[[宋徽宗|宋徽宗]]酷愛此畫，用“[[瘦金書|瘦金體]]”在圖上題寫“清明上河圖”五字。

《清明上河圖》歷十年畫成，最早由[[北宋|北宋]]宮廷收藏，后为[[宋神宗|宋神宗]][[钦圣皇后|向皇后]]族人所藏。[[靖康之變|靖康之禍]]後流入金国。1186 年，金朝張著、張公藥、郦權、王磵、張世積等人先後得此畫，識畫睹畫思故國，分別題跋于圖後。後歷經輾轉，為南宋[[賈似道|賈似道]]所得。[[元朝|元朝]]時期再度進宮，被收入秘府。爲官匠裝池者以赝本偷換出宮。售予某貴官，中途又爲保管人偷售給杭州陳彥廉。[[至正|至正]]年間又被調包，流落民間。

[[明朝|明朝]]初期，由[[大理寺|大理寺]]卿[[朱文徽|朱文徽]]、[[大學士|大學士]][[徐溥|徐溥]]收藏。1451 年，[[李東陽|李東陽]]在圖後兩次題寫長跋，詳記畫面內容和在明代中朝流傳始末；弘治以後，固歸華蓋殿大學土徐溥所有。[[徐溥|徐溥]]臨終前贈予[[李東陽|李東陽]]，[[嘉靖|嘉靖]]三年（1524年）圖歸兵部尚書[[陸完|陸完]]，[[陸完|陸完]]死後，其夫人將《清明上河圖》縫入枕中，後被娘家[[外甥|外甥]]王某曾臨摩此畫<ref>[[李日華|李日華]]：《味水軒日記》</ref>。陸完之子將《清明上河圖》賣至昆山[[顧鼎臣|顧鼎臣]]家。後來落到[[宰相|宰相]][[嚴嵩|嚴嵩]]、[[嚴世蕃|嚴世蕃]]父子手上，明人[[田藝蘅|田藝蘅]]《[[留青日札|留青日札]]》載嚴嵩為得《清明上河圖》，以1,200百金從[[蘇州|蘇州]]陸氏處購得，“饞得其贗本，卒破數十家”<ref>[[顧公燮|顧公燮]]《消夏閒記摘抄》載：太倉[[王忬|王忬]]收藏有《清明上河圖》，[[嚴世蕃|嚴世蕃]]知道後強行索要，王忬便將摹本獻給[[嚴嵩|嚴嵩]]。裝裱匠湯臣認出畫是假貨，指證說：只看屋角雀是否一腳踏二瓦便可證實。[[嚴嵩|嚴嵩]]大恨，便尋機將[[王忬|王忬]]害死。</ref>。[[嚴嵩|嚴嵩]]倒臺，圖被沒收，第三次納入宮廷。經[[明代|明代]]皇室收藏，後來太監[[馮保|馮保]]偷出，在畫上加了題[[跋|跋]]，之後真本又不知去向。

[[清朝|清朝]]時由[[陸費墀|陸費墀]]保存，在上面矜印題跋。後由[[湖廣總督|湖廣總督]][[畢沅|畢沅]]收藏<ref>徐耿华《学者督抚毕沅》：“《清明上河图》到清朝后，先由陆费墀（安徽相乡人）收藏。陆费墀是乾隆时进士，他得图后也在上面矜印题跋。后被毕沅购得……他得《清明上河图》以后，与其弟毕泷同赏，现今画上有二人印记。”</ref>，畢沅死後，《清明上河圖》第四次進宮，深藏[[紫禁城|紫禁城]]迎春閣內。嘉庆帝命人将其收录於《[[石渠宝笈|石渠宝笈]]·三编》一书之中。此後，《清明上河圖》一直在清宮珍藏。雖經歷1860年[[英法聯軍|英法聯軍]]入侵和1900年[[八國聯軍|八國聯軍]]入侵[[北京|北京]]二難，但均未受損。

1924年曾被[[溥儀|溥儀]]和兩幅仿作一起帶至東北，但[[溥儀|溥儀]]也不知何者為真品。1932年，溥儀在日本人扶植下，建立[[僞滿洲國|僞滿洲國]]，于是這幅名畫又被帶到[[長春|長春]]，存在僞皇宮東院圖書樓中。1945年被收入東北博物館（今[[遼寧省博物館|遼寧省博物館]]）當成贗品處理，直至1950年[[冬天|冬天]]才由[[楊仁愷|楊仁愷]]等人從庫房的贗品堆裡鑑定出真跡，後藏于北京[[故宫|故宫]]。[[文化大革命|文化大革命]]中，[[李作鵬|李作鵬]]曾利用權勢將《清明上河圖》強行從北京故宮“借”出，據爲己有。[[林彪|林彪]]倒台後，《清明上河圖》才又重見天日，如今依舊收藏於北京故宮。<ref>流泉．《尘埃历尽-中国珍贵文物蒙难纪实》：山东画报出版社，1999年．</ref>

== 摹本 ==
張擇端的《清明上河圖》以清明時節作為引子，展開對[[汴京|汴京]]的精描細繪，是寫實風俗畫的杰作，受到歷代畫家的喜愛因而有許多仿本出現。其中“[[明四家|明四家]]”之一[[仇英|仇英]]仿作的《清明上河圖》最有影響，蘇州一帶仿間大都以“仇本”為底本。明朝后期，大量蘇州仿制的《清明上河圖》散落民間，后來紛紛進入清內府，一時間魚龍莫辯，清宮不得不組織畫工，另起爐灶再畫了一張「加長版」《清明上河圖》，是為歷代版本中內容最為富麗的一幅，幅度也較張本及仇本長，今人稱為“清院本”。

《清明上河圖》是一幅舉世聞名的現實主義風俗畫卷。千余年來，此畫聲名顯赫，廣受青睐，仿摹者众多。各地公私藏家手中還有許多摹本和僞造本。資料載，僅清代皇宮所藏的摹本就有12幅。有人統計，現存《清明上河圖》有30多本，其中中國大陸藏10余幅，[[台灣|台灣]]藏9幅，[[美國|美國]]藏5幅，[[法國|法國]]藏4幅，英國和日本各藏一幅，光是[[台北|台北]][[國立故宮博物院|國立故宮博物院]]就藏有8幅。<ref>[http://theme.npm.edu.tw/exh105/RiverQingming/ch/index.html 清明上河圖特展]，[[台北|台北]][[國立故宮博物院|國立故宮博物院]]</ref>至于流傳到社會上還有多少幅，沒有人能弄清楚。

===清院本===
清院本《清明上河圖》是由清宮畫院的五位畫家——[[陳枚|陳枚]]、[[孫祜|孫祜]]、[[金昆|金昆]]、[[戴洪|戴洪]]、[[程志道|程志道]]在[[乾隆|乾隆]]元年（公元1736年）協作畫成，是參照各朝的仿本、集各家所長之作品。其中增加了明清時代的特殊風俗，如踏青、表演等等娛樂活動，亦增添了許多豐富的情節，如戲劇、猴戲、特技、擂臺等等，使畫中人物增加到超過4,000人，畫作尺寸擴大至長1152.8公分及寬35.6公分；同時因受到西洋畫風的影響，街道房舍均以[[透視|透視]]原理作畫，并有西式建筑行置其中。此卷用色鮮麗明亮，用筆達圓熟細致，所畫之橋梁、屋宇、人物皆細膩嚴謹，是院畫中的精品之作。現藏於台北國立故宮博物院。

===仇本===
又稱“明本”、“仇英本”，是“吳門四家”之一，明代著名畫家[[仇英|仇英]]，根據“清明上河”這一題材，參照“張本”的構圖結構，以明代[[蘇州|蘇州]]城爲背景，采用青綠重設色方式，重新創作了一幅全新畫卷，風格與宋本迥異。“仇本”也是後世衆仿作鼻祖，明人筆記載當時各種以此爲藍本的仿本層出不窮，一時間成爲達官豪門相互饋贈的高檔禮物。被稱爲仇英仿本。現藏台北國立故宮博物院。

== 畫面特寫 ==
清明上河圖的畫軸大致分成左、右兩部份，右邊主要描述鄉間的從容步調，主要是一些[[農人|農人]]、[[牧羊者|牧羊者]]、[[養豬|養豬]]的人等；從一條鄉間小徑逐漸拓寬、並連接到市鎮中的馬路。畫面左端則是描繪城市生活，出現有許多的經濟活動，比如人們正在將貨物上載到船上、商家、及稅捐辦公室等，都可以在畫軸左端瞧見。各式各樣行業的人都有，包含[[販夫走卒|販夫走卒]]、[[賣弄戲法|賣弄戲法]]的、[[演員|演員]]、[[乞丐|乞丐]]、化緣的僧侶、[[算命|算命]]仙、[[醫生|醫生]]、[[客棧|客棧]]老闆、[[老師|老師]]、磨坊主人、鐵工、木匠、石匠、讀書人等都有。畫中流貫汴京的汴河是當時南北交通樞紐，對北宋京城十分重要，因為城中的民生所需、奇珍百物，都全靠汴河供應。畫中隨處可見穿梭的[[船舶|船舶]]，搬運貨物的苦力，顯示汴河交通很繁忙。

在畫面左端進入城門後，各式各樣的商業活動包含賣[[酒|酒]]的、[[穀物|穀物]]市場、[[二手商品店|二手商品店]]、[[廚具|廚具]]店、[[弓|弓]][[箭|箭]]店、[[燈籠|燈籠]]店、[[樂器|樂器]]行、[[金飾|金飾]]行、布莊、畫廊、藥店、餐廳等應有盡有。在虹橋上，小販更是琳瑯滿目。

畫面中的[[虹橋|虹橋]]，是整幅畫軸的焦點。畫家在橋面上描繪出非常熱鬧的人群。一艘[[船|船]]用有點怪的角度要從橋下穿越，由於[[桅杆|桅杆]]不夠低，有點威脅到橋身的安全。橋上的人和河邊的人正在賣力地朝向船上的人呼喊、並以手勢表達。

繁忙的河道，造就汴河兩岸蓬勃的工商百業。汴京城的街道商店林立，有邸店（旅館）、醫藥舖、各式攤販等。宋代的人喜愛飲酒，因此汴京城內酒肆林立，資本雄厚的大酒商開設的酒店，稱為「正店」，中、小型的酒店，則稱為「腳店」，還有一些賣下價酒的「小店」，酒業可算是汴京的一大行業。由於酒客很多，因此亦吸引了許多小販在店前擺賣。除了商店和小吃店以外，畫面中還包含許多[[寺廟|寺廟]]、[[私人住宅|私人住宅]]、[[官邸|官邸]]，各種[[階級|階級]]和型態的房舍都有，有些還有前院和後院。畫面中的人們用各種不同型式的交通工具，有馬車、驢子拉車、轎子等。河川中央滿是漁船、和載人的遊艇，河邊則有一些苦力工人在拉著大船使其靠岸固定。還有小學童翹課在屋外尿尿景象。當中婦女形象不多，大戶人家的婦女上街都得坐[[轎|轎]]，坐在轎中半遮面、窺看這個城市的熱鬧景，還有[[婢女|婢女]]侍側，主要都是出外購物的女人。酒樓、茶肆更不是婦女人家流連之地，由此可見宋代社會對婦女「拋頭露面」的限制。新娘嫁娶內容平易近人，反應當時生活樣態。

總之，從《清明上河圖》中可以看到以下幾個非常鮮明的特征：
*內容豐富，描繪東西繁多。
*結構嚴謹，繁而不亂，長而不冗，段落分明。
*在技法上，大手筆與精細的手筆相結合。

==後世評價 ==
《清明上河圖》不僅僅是一件偉大的現實主義繪畫藝術珍品，同時也爲我們提供了北宋大都市的商業、手工業、民俗、建築、交通工具等詳實形象的第一手資料，具有重要歷史文獻價值。其豐富的思想內涵、獨特的審美視角、現實主義的表現手法，都使其在中國乃至世界繪畫史上被奉爲經典之作。

白壽彜任顧問的《中國通史（彩圖本）》對《清明上河圖》的評價：全卷所繪人物五百余位，牲畜五十多只，各種車船二十余輛艘，房屋众多，道具無數，場面巨大，段落分明，結構嚴密，有條不紊。技法閑熟，用筆細致，線條遒勁，凝重老練。反映了高度精純的繪畫功力和出色的藝術成就。同時，因爲畫中所繪爲當時社會實錄，爲後世了解研究宋朝城市社會生活提供了重要的歷史資料。

《簡明不列顛百科全書》在“[[張擇端|張擇端]]”詞條內對《清明上河圖》的評價：是一幅具有重要歷史價值的風俗長卷，畫家成功地描繪出[[汴京|汴京]]城內及近郊在[[清明|清明]]時節社會上各階層的生活景象。主要表現的是勞動者和小市民。對人物、建築物、交通工具、樹木、水流之間的相互關係的處理，非常巧妙，整體感很強，具有極大的考史價值。此後歷代繪制的都市風俗畫，無不受其影響。

==畫作之謎 ==
歷史上對張擇端創作《清明上河圖》的年代，以及“上河”一詞曾有過一些爭論；對畫中描繪的是[[清明|清明]]時節，從金代以來似無異議。明代的《味水軒日記》中記載，這幅畫不但有[[宋徽宗|宋徽宗]]的瘦金體題簽、雙龍小印，並且還有宋徽宗的題詩；詩中有“水在上河春”一句。這樣一來，這畫卷描繪的是春天景色就更不用懷疑了，中國近代及當代美術史家[[鄭振铎|鄭振铎]]、徐邦達、張安治等均主“春景”之說。但是，也有人對此提出異議。

20世紀80年代中期，鄒身城先生在中國宋史研究會上提出論文《宋代形象史料〈清明上河圖〉的社會意義》，認爲“清明”既非節令，亦非地名。這裏“清明”一詞，本是畫家張擇端進獻此畫時所作的頌辭。故有人認爲，這裏的“清明”要從廣義上去理解。《[[後漢書|後漢書]]》有例證，是出云：“固幸得生‘清明之世’……”從語氣看這個“清明”係指政治開明。畫中題款“清明”語，本是張擇端進獻此畫，請帝王們賞識所作的頌辭。金人在畫面上留下的跋文說：“當日翰林呈畫本，承産風物正堪傳。”點明此畫主題在于表現承平風物。考張擇端行年，他於徽宗朝在翰林書畫院供職，此畫的第一位收藏人便是宋徽宗，證畫家意在稱頌盛世，討最高統治者歡心。知道了這個背景，顯然“清明”一詞不是指節令。

還有人認爲清明之意是指“清明坊”。根據是當時東京城劃分一百三十六坊，外城東郊區共劃分三坊，第一坊就是“清明坊”。

同時長期以來一些專家學者對“上河”二字的含義也有諸多解釋。關於“上河”的含義主要有幾種觀點：有專家學者認爲“上河”是指“河的上遊”；有專家學者認爲“上河”是“逆水行舟”之意；也有專家學者認爲“上河”即“上墳之意”；還有專家學者認爲“上河”即“趕集上街之意”。

==大事記 ==
2011年中華民國[[國立故宮博物院|國立故宮博物院]]、[[公視|公視]]與歐洲藝術媒體[[Arte|Arte]]頻道[[法國公視|法國公視]]聯手製作的紀錄片'''《話·畫——清明上河圖》'''，其拍攝的是故宮典藏的清院本《清明上河圖》真跡；帶領觀眾從新媒體的角度與名人口白，重現當年清明佳節古城熱鬧繁盛的景象。該片於2011年7月7日首播。
