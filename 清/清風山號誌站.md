'''清風山號誌站'''（{{jpn|j=清風山信号場|hg=せいふうざんしんごうじょう|rm=Seifūzan shingōjō|f=y}}）位於[[日本|日本]][[北海道|北海道]][[勇拂郡|勇拂郡]][[占冠村|占冠村]]字仁仁宇（{{lang|ja|ニニウ}}）2953，是[[北海道旅客鐵道|北海道旅客鐵道]]（JR北海道）[[石勝線|石勝線]]上的[[號誌站|號誌站]]。

== 簡介 ==
清風山號誌站在實際設置之前，原本預計根據附近的聚落，設置'''仁仁宇車站'''（{{lang|ja|ニニウ駅}}，「ni-ni-us/{{lang|ja|ニニウ}}」在[[愛奴語|愛奴語]]中意指「多樹木的地方」<ref>[http://www.onitoge.org/ninew/hakubutsukan/1tengakkou.htm 關於「{{lang|ja|ニニウ}}」這地名]</ref>）。仁仁宇自[[明治時代|明治時代]][[拓殖|殖民]]計畫開始實行以來，就一直是個交通極為不方便、猶如陸中孤島的社區，因此長期以來當地居民一直期盼能有鐵路經過並設置車站。然而，由於人口逐漸稀疏，在十勝線實際通車之前，全村的人口皆已遷出、廢村，成為無人地帶，因此原本預計要設置的仁仁宇車站也降級成為號誌站。另外，由於此區域是[[熊|熊]]經常出沒的地帶，在石勝線進行高速化工程時，甚至必須有[[獵人|獵人]]同行以保安全。

現在附近正在建設[[道東自動車道|道東自動車道]]。

== 車站構造 ==
擁有2線的號誌站。號誌站兩端設有[[防雪廊|防雪廊]]（{{lang|ja|スノーシェルター}}）以避免[[轉轍器|轉轍器]]凍結。

== 車站周邊 ==
由占冠村的林道往山中深處前進即可到達，附近為廣闊的原生森林地帶。號誌站雖沒有柵欄圍住，但沒有允許是禁止進入鐵路線。

== 历史 ==
*1981年（[[昭和|昭和]]56年）10月1日 - [[日本國有鐵道|日本國有鐵道]]石勝線清風山號誌站設立。
*1987年（昭和62年）4月1日 - [[國鐵分割民營化|國鐵分割民營化]]後由JR北海道接收管理。
*2011年（[[平成|平成]]23年）5月27日 - [[超級大空號列車|超級大空號]]於本號誌站附近脫軌，並在第1仁仁宇隧道內停車後發生火災。<ref>{{ja}}{{PDFlink|[http://www.jrhokkaido.co.jp/press/2011/110528-1.pdf 石勝線清風山號誌站內列車脫軌意外した列車脱線事故]}} - 北海道旅客鐵道新聞稿 2011年5月28日</ref><ref>{{PDFlink|[http://www2.jrhokkaido.co.jp/global/pdf/apology.pdf 石勝線事故聲明]}}</ref>
