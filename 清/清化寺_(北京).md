'''清化寺'''（后讹作清华寺），位于[[北京市|北京市]][[东城区_(北京市)|东城区]][[清华街|清华街]]7号，是[[汉传佛教|汉传佛教]]寺院。清华街原名清化寺街，即因该寺而得名。2013年1月被东城区文化委员会以“敕建清华禅林”之名列为[[北京市东城区普查登记文物|北京市东城区普查登记文物]]挂牌保护。

该寺始建于[[明朝|明朝]]，[[清朝|清朝]]重修。大雄宝殿为庑殿顶，是北京南城唯一一座[[庑殿顶|庑殿顶]]庙宇建筑。该寺坐北朝南，原来有山门三间，门额石匾上书“敕建清化禅林”。整个寺院分为三进院落。后殿在[[两广路|两广路]]建商业街时被拆除。各配殿被居民拆除改建。2014年修缮前，早已年久失修，仅存前殿和大雄宝殿，并有诸多险情，前殿和大雄宝殿建筑面积112.11平方米，当时前殿和大雄宝殿内有现状居民6户，企事业单位1家。2014年修缮时，大雄宝殿及周边平房为东城区房屋土地经营管理二中心直管公房。为抢救保护文物，经东城区名城办审核、东城区人民政府批准，前殿和大雄宝殿的腾退修缮工程被列入名城保护专项资金支持项目。2014年，东城区房屋土地经营管理二中心完成了6户居民的搬迁安置，同年东城区文化委员会组织实施了大雄宝殿的修缮工作。但前殿因被那家企事业单位占用而未能腾退修缮。<ref>{{Cite web|url =http://www.bj.xinhuanet.com/jyyf/2013-01/10/c_114319381.htm|title =北京东城将腾退修缮清华寺|author =|date =2013-01-10|publisher =新华网|language =|accessdate =|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20130114162919/http://www.bj.xinhuanet.com/jyyf/2013-01/10/c_114319381.htm|archivedate =2013-01-14}}</ref><ref>{{Cite web |url = http://house.hexun.com/2015-01-20/172570209.html |title =  清华寺文物腾退修缮工程基本完成|author =  |date = 2015-01-20 |publisher = 和讯网 |language =  |accessdate =  |quote =  }}</ref><ref name=xieshang>{{Cite web |url = http://mt.sohu.com/20161020/n470794898.shtml |title = 明代清华寺被违建包围 北京东城文委：将与产权单位协商 |author =  |date =  |publisher = 搜狐 |language =  |accessdate = 2016-10-20 |quote =  }}</ref>2016年，东城区推动该寺二期腾退修缮工作。<ref>{{Cite web |url = http://mt.sohu.com/20160617/n454985034.shtml |title = 太狠了！完成时间都敢公开！北京各区谁敢不落实？！ |author =  |date = 2016-06-17 |publisher = 搜狐 |language =  |accessdate =  |quote =  }}</ref><ref name=xieshang/>
