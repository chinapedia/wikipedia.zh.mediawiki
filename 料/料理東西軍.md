《'''料理東西軍'''》（'''{{lang|ja|どっちの料理ショー}}'''，直譯：'''哪道料理秀'''）是[[日本|日本]][[讀賣電視台|讀賣電視台]]於1997年4月17日到2005年3月17日所製作的美食[[綜藝節目|綜藝節目]]。[[台灣|台灣]][[緯來日本台|緯來日本台]]於1999年開始引進，以[[日語|日語]]原音、[[中文|中文]][[字幕|字幕]]播映。

== 內容 ==
兩位[[主持人|主持人]][[關口宏|關口宏]]和[[三宅裕司|三宅裕司]]，每集分別帶領兩個[[廚師|廚師]]團隊互相比較廚藝。每週都各選出一項特選素材，並作一道菜來進行比賽(有時會作套餐)。

每集來賓人數為7位，但《新料理東西軍》時期有時會出現9位來賓。來賓共有3次選擇機會：第1次為開始烹飪前；第2次為試吃後（但有時只會有1人透過轉盤獲得試吃機會）；第3次為最後選擇，全體來賓同時舉牌[[多數決|多數決]]，選贏的一方就可以吃到這次的料理，輸者則只能乾瞪眼。選輸的一方，廚師必須當場把自己做的菜撤出餐桌，並在錄影結束之後親自吃完自己做的菜，稱為“敗戰主廚的晚餐”（{{lang|ja|負けシェフの晩餐}}）。

== 影響 ==
節目中對美食的介紹與兩隊競爭時的刺激，均培養了不少忠實[[觀眾|觀眾]]，在日本及台灣兩地都獲得了極高[[收視率|收視率]]，本節目與《[[電視冠軍|電視冠軍]]》及《[[火焰挑戰者|火焰挑戰者]]》在台灣市場中均被視為日本優質[[電視節目|電視節目]]的代表。每次預算平均都有三千萬日圓以上，甚至達到數億日圓。

== 停播 ==
經過近八年的播放，《料理東西軍》的題材進入疲乏期。於是《料理東西軍》於2005年4月改版成為《新料理東西軍》（{{lang|ja|新どっちの料理ショー}}），將部份節目規則改變，一般觀眾也可參加錄影；但反應不如預期，收視表現也不如以前。加上節目助理主持人之一的[[日本電視台|日本電視台]][[播報員|播報員]][[炭谷宗佑|炭谷宗佑]]在2006年2月20日傍晚在[[橫濱車站|橫濱車站]]以[[照相手機|照相手機]][[偷拍|偷拍]]1名16歲高中女生的[[裙底春光|裙底春光]][[被捕|被捕]]，導致節目聲勢更一落千丈，也使得不少女性觀眾發起拒看行動。為此，讀賣電視台宣佈於2006年9月停播《新料理東西軍》，改播旅遊節目《[[東西軍旅行團|東西軍旅行團]]》（{{lang|ja|ニッポン旅×旅ショー}}，直譯：'''日本旅遊×旅秀'''）。《東西軍旅行團》主持人一樣由關口宏與三宅裕司合作，並加入新血[[藤井隆|藤井隆]]。

== 特別節目 ==
2007年新春播出特別版 －適合配飯的料理：『和風對中式』
