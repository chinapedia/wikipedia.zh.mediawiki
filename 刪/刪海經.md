{{engname|《刪海經》|'''The Lost Sea'''}}由導演[[洪淳修|洪淳修]]所執導的一部紀錄片<ref>{{Cite web |url = http://www.tidf.org.tw/zh-hant/films/243|title =刪海經 台灣國際紀錄片影展}}</ref>，歷時五年完成，拍攝地為[[金門縣|金門縣]][[後豐港|後豐港]]。片名發想自記載怪物故事的中國古籍《[[山海經|山海經]]》，取其[[諧音|諧音]]為「刪」海經，來隱喻海中一種名為[[鱟|鱟]]的古生物，因[[水頭商港|水頭商港]]的[[填海造陸|填海造陸]]工程，導致海洋整個被填平，最後使鱟與漁人失去海洋的故事。該片觸及的不只是生態環境議題，更以政治經濟的角度來看待鱟，思考[[台海兩岸|台海兩岸]]之間在政治上的矛盾關係，是拍攝鱟相關紀錄片中少見的影片觀點。影片發表後獲多項國內外獎項肯定。

==影片簡介==
在金門，有一群人，他們的祖先因戰爭落腳於此，三百年來烽火連天的歲月，他們仰賴家門前的海維持了生計；在金門，海裡有一種活化石，當大部分棲地在人為開發下消失殆盡，牠們卻在充滿軍事陷阱的沙灘上生生不息。和平來了，他們和牠們的生存危機才正要開始…。<ref>{{Cite web |url = http://thelostseatw.blogspot.tw/2014/04/test.html#more |title =刪海經 The Lost Sea  }}</ref>
