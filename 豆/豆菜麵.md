'''豆菜麵'''是一種[[臺灣|臺灣]][[麵食|麵食]]，盛行於[[臺南市|臺南市]]與[[嘉義縣|嘉義縣]]交界，包括[[大內區|大內]]、[[新營區|新營]]、[[六甲區|六甲]]、[[下營區|下營]]（以上屬臺南市）、[[義竹鄉|義竹]]（屬嘉義縣）等地的地方小吃，分佈範圍大致位於[[八掌溪|八掌溪]]與[[曾文溪|曾文溪]]之間的嘉南平原中心地帶。

除上述區域外，在臨近地區東山、後壁、麻豆、官田、善化等地也有，但重要性不及前述五地。近年在經濟發展之後全臺各地擴散，多來自於白河、新營、六甲或下營等。

==歷史==
豆菜麵的起源，在歷史上已難以考據，不同於府城台南的擔仔麵有肉臊、蝦子、貢丸（以發展當時而言）等較高檔配料，豆菜麵相較顯得簡單而清爽，主要組成的澱粉顯然是裹腹為主，能吃飽是主要考量。

據台南大學台灣文化研究所黃世祝教授推測，應該是早期食材取得不易、可久置、易料理、價格低等因素，可能是造就先民以簡單的麵條放涼、加上煮熟放冷的豆菜、淋上醬油組成豆菜麵的形成背景原因，由此而在當時屬於"草地所在"的區域風行。

== 起源 ==
豆菜麵的起源現今存在以下幾種說法：
# 白河豆菜麵現今主要經營者已經輾轉歷三代以上，而且前後有過好幾個不同的店家，根據1920年代出生的耆老表示，幼時店仔口街上即有人販售豆菜麵，以此推測白河豆菜麵的經營可能始自清末或至遲日治時期已有，目前在市場內經營的吳家口碑深受肯定，早上的山產市場轉角天主教堂對面也有一家風味甚佳。
# 新營豆菜麵1951年，蔡菊花女士為了養家餬口，以豆菜麵做為麵擔生意，在[[新營區|新營區]]民治路上沿街叫賣，後來接受顧客建議，於是改成定點販賣<ref>註：販賣地點位在現今的[[臺南市政府|臺南市政府]]民治市政中心旁，當時並沒有店鋪，因此沒有店名或招牌。</ref>，在日積月累下，憑著豆菜麵而在街坊傳開這項口碑，因此奠定了豆菜麵創始人「蔡菊花」創始人<ref>註：1984年，蔡菊花傳給第二代張正忠經營。</ref>，定名為阿忠豆菜麵。
# 上述說法並不為市場內的豆菜麵業者認同，認為是為了商業而作的宣傳，新營人普通認為延平市場內的二家豆菜麵才是新營正宗道地豆菜麵。
# 六甲豆菜麵創始人源自[[六甲區|六甲區]]「顏寅木」先生<ref>[http://www.comnews.gio.gov.tw/bigthing/south.aspx?articale_id=3387 簡單俐落的南台灣涼麵—豆菜麵]{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。
# [[下營區|下營區]]：阿太豆菜麵是祖傳四代了，現已經八十多年的歷史。
# 義竹市場內的老牌製麵店兼營豆菜麵，口感風味也備受肯定。

==製程==
豆菜麵的主要材料為麵條、豆芽菜、蒜味醬油。麵條類似油麵但較扁薄，口感Q彈柔軟，與一般圓粗且偏硬的油麵不同，具有其獨特的風味，通常是製成水煮過後放冷，再加上煮熟的豆菜，並佐以蒜味醬油<ref>註：蒜味醬油指加入[[蒜頭|蒜頭]]熬煮製成的醬油。</ref>，便完成一道「豆菜麵」。

==食用方式==
業者將黃色條狀麵條煮熟、拌上[[豆芽菜|豆芽菜]]、淋上醬油為主要料理方式。豆菜麵攤通常不賣小菜，但會販售配湯，通常以肉羹及味噌湯為主。清明時節因包潤餅需求，麵條消耗量大，業者甚至必須準備上千斤的麵條才夠應付市場所需。地方上有廟會等重大活動時也常以豆菜麵為主食。

食用時主要都是乾麵，而且是將麵條以常溫冷卻方式，待冷卻後即可食用。白河、新營、義竹的豆菜配合肉羹湯販售，也有的直接將麵與羹加在一起，即成麵羹。六甲販賣豆菜麵的店家搭配的是豆醬湯，豆醬湯風味不同於[[味噌湯|味噌湯]]，可搭配豆菜麵一起食用，建議拌勻已淋上大蒜醬油的豆菜麵，可突顯香氣與打開[[味蕾|味蕾]]。另外也發展出配有荷包蛋的豆菜麵，以及味增湯。下營鄉:阿太豆菜麵的手工豆菜麵是搭配西螺蔭油(比較甘甜)再搭配著特調味噌湯。
