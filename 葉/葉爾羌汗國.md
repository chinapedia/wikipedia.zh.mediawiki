'''葉爾羌汗國'''（يەركەن سەئىدىيە  خانلىقى ）是由[[东察合台汗国|东察合台汗国]]汗[[速檀阿黑麻|速檀阿黑麻]]之子[[苏丹_(称谓)|蘇丹]][[赛德|赛德]]在1514年於原[[察合台汗國|察合台汗國]]的舊地上創立的一個國家，部分史学家认为是[[东察合台汗国|东察合台汗国]]的延续。至1680年成為[[準噶爾汗国|準噶爾汗国]]的附属国，在17世纪末[[策妄阿拉布坦|策妄阿拉布坦]]时期消亡。早期的首都在喀什噶尔（今[[喀什市|喀什市]]），后迁往叶尔坎[[葉爾羌城|葉爾羌城]]（今[[莎車县|莎車县]]）。

疆域最大时包括[[吐魯番|吐魯番]]，[[哈密|哈密]]，[[塔里木盆地|塔里木盆地]]。汗國版圖的東达[[敦煌|敦煌]]、南达吐蕃[[烏思藏|烏思藏]]、西南达[[博羅爾|博羅爾]]、[[巴爾蒂斯坦|巴爾蒂斯坦]]，與[[烏茲別克|烏茲別克]]以[[費爾干納谷地|費爾干納谷地]]為界，北方以[[天山|天山]]為界，與[[準噶爾|準噶爾]]相鄰。

== 历史 ==

14世纪，今新疆南部的[[突厥化蒙古人|突厥化蒙古人]]大部分皈依伊斯兰教，融入到当地的突厥语系民族，成为现如今[[维吾尔族|维吾尔族]]的四大族源{{NoteTag|[[维吾尔族|维吾尔族]]族源，西域[[高加索人种|高加索人种]]土著（如[[吐火羅人|吐火羅人]]和[[塞人|塞人]])，[[西突厥|西突厥]]旧部，回鹘西迁部，[[突厥化|突厥化]]/[[伊斯兰化|伊斯兰化]]东阿尔泰语系各部落。}}之一。

叶尔羌汗國信奉[[伊斯蘭教|伊斯蘭教]]，蘇非派的勢力很大。速檀赛德曾經想成為一位[[德爾維希|德爾維希]]（即托缽僧），投身和卓[[馬黑麻·尤素福|馬黑麻·尤素福]]的門下，後來及時受勸阻。由此時起，和卓家族興起。[[黑山派|黑山派]]和[[白山派|白山派]]兩派互鬥不已，直到亡國。白山派的据点是喀什噶爾，黑山派的据点是葉爾羌城。

1680年準噶爾部首領[[噶爾丹|噶爾丹]]攻佔葉爾羌後，扶植葉爾羌汗後裔[[阿卜都里什特|阿卜都里什特]]作為傀儡政權進行統治。到了[[策妄阿拉布坦|策妄阿拉布坦]]時期，約在1700年準噶爾出兵重新控制了[[天山南路|天山南路]]，不再立葉爾羌汗，葉爾羌汗國徹底滅亡。

== 政治 ==

汗國大部制度和[[成吉思汗|成吉思汗]]時代無異，只是信奉[[伊斯蘭教|伊斯蘭教]]。汗國的官員分為軍事官員、宮廷官員、中央官員、宗教人士官員。汗由[[庫里台大會|庫里台大會]]推舉。法官由宗教官員與世俗官員出任。由案情決定，官教案件由哈的負責，習慣法由[[伯克_(官职)|伯克]]負責。

== 经济 ==

經濟為園地業、商業、畜牧業。以農業為主，土地分別為國有土地、私人物土地、宗教莊園土地。官地由佃農種植，稱為[[塔蘭奇|塔蘭奇]]，是[[農奴|農奴]]。收成五成交給汗國，私人土地則是一成，這是經教確定的。[[黑汗王朝|黑汗王朝]]已有此制度。
