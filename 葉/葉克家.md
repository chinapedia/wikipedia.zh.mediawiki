'''葉克家'''（Keh Chia Yeh，{{bd|1956年|12月20日|||}}）台灣水利工程學者，現擔任[[國立交通大學|國立交通大學]]土木工程學系教授，曾任該系的系主任。

== 現職 ==
* 國立交通大學 土木工程學系 教授<ref>[http://www.cv.nctu.edu.tw/chinese/teacher/teacher03.shtml 國立交通大學土木工程學系]</ref>
* 國家災害防救科技中心(NCDR) 洪旱災害防治組 召集人<ref>[http://www.ncdr.nat.gov.tw/CusPage.aspx?WebSiteID=5853983c-7a45-4c1c-9093-f62cb7458282&id=2&subid=35&PageID=26 國家災害防救科技中心洪旱災害防治組]</ref>
* 國立交通大學 防災與水環境研究中心 教師<ref>[http://dpwe.nctu.edu.tw/member/teachers/3 國立交通大學防災與水環境研究中心]</ref>

== 學歷 ==
* [[美國|美國]][[愛荷華大學|愛荷華大學]] 土木及環境工程 博士 (1990)
* [[國立台灣大學|國立台灣大學]] 土木工程水利組 碩士 (1980)
* [[國立成功大學|國立成功大學]] 水利工程 學士 (1978)
== 經歷 ==
* 國立交通大學 教授（1997 至今）
* 國立交通大學 土木工程學系 系主任（2008 - 2010）
* 國立交通大學 副教授（1991 - 1997）
* 財團法人中興工程顧問社 水工部 工程師 （1982 - 1987）
== 領域 ==
* 水利工程
* 輸砂力學
* 河川水力學
* 水工試驗
