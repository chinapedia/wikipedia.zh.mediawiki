'''葉王'''（1826年－1887年），字'''麟趾'''、號'''和雲'''，外號「獅」，時人以「王獅」相稱，後人改為「王師」以示尊稱。[[台灣|台灣]][[嘉義縣|嘉義]]人，為第一位台灣出身的[[交趾陶|交趾陶]]藝師。在[[台南縣|台南縣]]、[[嘉義縣|嘉義縣]]一帶廟宇中皆可見其作品。其[[寶石釉|寶石釉]]製作技術為當今台灣交趾陶二大支派之一，與[[洪坤福|洪坤福]]的水彩釉齊名，作品曾遍及全台，有「台灣絕技」、「台灣交趾第一人」之譽。

== 生平 ==
葉王的父親葉清嶽為[[福建省|福建省]][[漳州市|漳州]][[平和縣|平和縣]]人，13歲移居台灣嘉義縣[[民雄鄉|民雄鄉]]，<ref>江紹瑩，〈臺灣交趾第一人王師葉麟趾的研究考略〉</ref>葉王為其次子，另有長兄精英。葉清嶽本為陶工，其作品在日治時期即被日人攜出。<ref name="嘉縣志243">李淑卿纂修，《嘉義縣志．卷十一．藝術志》，嘉義：嘉義縣政府。2009年12月。ISBN 9789860208115 頁243</ref>葉王受家學浸濡，幼時即以泥塑為樂。當時[[廣東省|廣東]]交趾陶師劉詩構應聘前往台南建築兩廣會館（或說為[[三山國王廟|三山國王廟]]）<ref>兩廣會館說出於張李德和，但施翠峰提出會館修建時間與葉王年代不符，三山國王廟之說則出於王福源。參見王福源，〈從歷代彩陶釉看台灣交趾陶－嘉義陶淵源與研究〉，《嘉義交趾陶文化創意產業之活化與發展》，嘉義：嘉義市政府文化局。2005年。頁8-9</ref><ref name="李鑄恆">李鑄恆，〈葉王交趾陶胭脂紅釉彩研究〉，雲林：國立雲林科技大學視覺傳達設計系碩士班碩士論文。2007年2月。</ref>，行經嘉義時偶見葉王泥塑的技術精巧，於是收他為徒，帶往台南傳授交趾陶技術。<ref name="嘉縣志244">《嘉義縣志．卷十一．藝術志》，頁244</ref>

葉王習得技藝後，所製作品以今[[嘉義市|嘉義市]]一帶為主。清咸豐3年（1853年）嘉南地區發生大地震，包括[[佳里金唐殿|佳里金唐殿]]、學甲慈濟宮的修繕，其裝飾工程皆由葉王主持。[[咸豐|咸豐]]、[[同治|同治]]年間，因翻修廟宇盛行，也成為葉王最活躍的創作時期。中南部寺廟與豪宅皆可見其作品，但至日治時期其價值才漸受重視。<ref name="嘉縣志244" />

== 傳承 ==
葉王沒有嫡傳弟子，但曾將製釉技術傳予黃得意、許子瀾二人<ref name="縣志2445">《嘉義縣志．卷十一．藝術志》，頁244-245</ref>（一說為傳予三人：許子瀾與二佚名者，其中一人姓簡<ref name="市府網">嘉義市政府文化局[http://www.cabcy.gov.tw/Koji/person-1.asp 交趾陶館網頁介紹]{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}，2010年11月5日查閱</ref>。），其中許氏為清代秀才，習藝僅為玩票而未成大業，黃氏則得葉王的上釉技術<ref name="縣志2445" />，並傳予[[林添木|林添木]]。林添木融合葉王之釉彩技術與習自蔡文董的[[剪黏|剪黏]]技藝，曾一度以仿葉王手法的方式創作<ref>《嘉義縣志．卷十一．藝術志》，頁252</ref>，並授藝予高枝明、蘇俊夫、葉星佑、吳志榮、羅木枝、林智信、林俊亨、葉貞吉、劉奇、[[林洸沂|林洸沂]]等藝師，<ref>國立歷史博物館編輯委員會，《彩塑人間：臺灣交趾陶藝術展》，台北：國立歷史博物館。1999年。ISBN 9570251271 頁142</ref><ref>葉王傳承譜系說法相當紛亂，如《嘉義縣志．卷十一．藝術志》於「葉王」小節所述為葉王傳黃得意、黃得意傳蔡文董、蔡文董傳林添木。於「林添木」小節之譜系則如文中所述。</ref>至此，葉王一派方得發揚光大。

== 作品特色 ==
葉王交趾陶作品多用於廟宇裝飾，題材廣泛，涵蓋古今歷史、小說、掌故，以人物為最大宗。<ref name="縣志245">《嘉義縣志．卷十一．藝術志》，頁245</ref>其所塑人像多半形體不大，胎質薄、重量輕。以刮平線刻手法，細緻的表現人物的面孔、衣紋，使人物栩栩如生、各具性格。<ref>[http://www.epochtimes.com/b5/4/3/1/n476238.htm 〈葉王交趾陶完璧歸趙〉]，大紀元時報，2004年3月1日</ref>在[[法國|法國]][[巴黎|巴黎]][[世界博覽會|世界博覽會]]展出時，被評為「台灣絕技」<ref name="謝譯增">謝譯增，〈以台灣交趾陶文化應用於產品設計之研究〉，大同大學工業設計研究所碩士論文。2009年9月。</ref>。

用色部分，葉王以寶石釉為長，與嘉義稍晚出的洪坤福水彩釉不同。<ref>《嘉義縣志．卷十一．藝術志》，頁242</ref>他結合了低溫軟陶的鉛釉與高溫的瓷器釉<ref name="謝譯增" />，釉彩調合典雅、色澤飽和，著名的有胭脂紅、翡翠綠、古黃諸色。據當今科學分析，胭脂紅釉質中包括[[鉛|鉛]]、[[鉀|鉀]]、[[矽酸|矽酸]]的[[氧化物|氧化物]]，翡翠綠含鉛、[[鈉|鈉]]、[[鈣|鈣]]、[[硼|硼]]、矽酸之氧化物，古黃則是以鉛、[[鋁|鋁]]、矽酸氧化物為主要成份，以攝氏870～920度的溫度燒製而成。<ref>曾永寬、徐必彥、李尚哲、裴晉哲，〈回溯葉王交趾陶古黃釉彩配方〉，《文化資產保存學刊》第六期，台北：行政院文化建設委員會文化資產總管理處籌備處。2008年12月。頁46-61。</ref>其中「胭脂紅」為葉王獨創的新釉色，被認為是葉王派最重要的特徵。<ref name="謝譯增" />

== 作品分布 ==
葉王作品主要分佈於嘉南地區，日人亦有遠至台北、淡水、新竹等地的說法。<ref name="李鑄恆" />目前可考的創作可參見後表。這些作品因翻修、盜取、損壞等因素，多已不傳，今中華民國三級古蹟－[[新竹鄭氏家廟|新竹鄭氏家廟]]中尚有留存。另外，[[臺南市|臺南市]][[學甲慈濟宮|學甲慈濟宮]]有葉王作品百餘件，其中56件於1980年遭竊，慈濟宮遂將大部分葉王作品卸下，收藏於宮內附屬的「葉王交趾陶文化館」中，目前殿中作品則為葉王派後人林洸沂之仿作。2003年，[[震旦文教基金會|震旦文教基金會]]在偶然機會下，至海外買回36件失竊作品，並無償歸還慈濟宮，目前亦已納入館藏。<ref>文建會[http://superspace.cca.gov.tw/ct.asp?xItem=12925&ctNode=11195&mp=28 學甲慈濟宮葉王交趾陶文化館介紹網頁]{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}，2010年11月5日查閱</ref><ref>[http://www.tcgs.org.tw/treasure_koji.html 慈濟宮葉王交趾陶文化館頁面]，2010年11月5日查閱</ref>
