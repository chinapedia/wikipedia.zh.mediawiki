'''葡式碎石路'''（[[葡萄牙語|葡萄牙語]]：'''{{lang|pt|Calçada Portuguesa}}'''）是在[[葡萄牙|葡萄牙]]地區及前[[葡萄牙帝國|葡萄牙帝國]]殖民地，如[[巴西|巴西]]及[[澳門|澳門]]，最常用來鋪砌行人路路面的傳統方式。這鋪砌方法亦常見於[[廣場|廣場]]及[[行人專用區|行人專用區]]。 

其中一個最突出地運用這鋪砌技巧的，是在[[科英布拉|科英布拉]]市的葡萄牙皇后[[聖伊利莎白|聖伊利莎白]]圖像，以黑白色的[[玄武岩|玄武岩]]及[[石灰岩|石灰岩]]設計。

== 歷史起源 ==
路面鋪砌作為手工藝相信源自於[[美索不達米亞|美索不達米亞]]，當地以石料作為建築物內部及外部的建材，後來把技術傳到[[古希臘|古希臘]]及[[古羅馬|古羅馬]]。

羅馬人以於附近找到的材料鋪砌[[羅馬街道|羅馬街道]]接通帝國，部份技術後來應用到碎石路上，尤其用於路基及鋪面上。

在[[伊比利半島|伊比利半島]]的[[阿拉伯人|阿拉伯人]]亦遺下鋪砌藝術的痕跡。

== 碎石的鋪砌 ==

== 澳門的葡式碎石路 ==

在1990年代[[澳葡政府|澳葡政府]]重金在葡萄牙請來技師，在[[議事亭前地|議事亭前地]]至[[板樟堂前地|板樟堂前地]]鋪砌黑白相間的波浪形路面，使之成為[[行人專用區|行人專用區]]，為澳門首次採用葡式碎石路。後來範圍不斷擴大，包括[[媽閣廟前地|媽閣廟前地]]、[[崗頂前地|崗頂前地]]、[[澳督府|澳督府]]前地及部份公園等都鋪上碎石。惟後期特別是[[澳門回歸|回歸]]後鋪設的碎石造工不佳，大雨過後碎石脫落，且易絆倒行人，且{{Fact|圖案造型粗糙，空隙也較為明顯，碎石排列不均，碎石中的塡充物料更覺礙眼|time=2009-11-13T03:24:00+00:00}}。另外市政當局在舊城區也舖設[[石板街|石板街]]。

== 不確定的將來 ==

很少技師（''calceteiros''）享受這艱鉅的勞力工作，因為要花很長時間煞費苦心地把碎石安放在特定的位置。工資水平低亦難以吸引學徒。

鋪好的路面亦會對行人構成危險，以及對傷殘人士造成障礙。這路面在潮濕時特別危險，因為光滑的表面導致行人容易滑倒及失足。而且碎石路亦容易鬆脫，危害行人安全。

以這種方法來鋪砌路面相對於[[水泥|水泥]]路或[[瀝青|瀝青]]路有較高的成本及較短的壽命，但亦較容易開挖，去處理地下設施。

這工藝曾經流行於葡萄牙的各城市及村莊，但現在逐漸只用於歷史或重要的建築物上。材料的不足、懂得這工藝的技師越來越少、以及認為鋪砌得過濫的批評，逼使[[市政機構|市政機構]]考慮其他鋪砌路面的方式。

[[巴西|巴西]]城市[[聖保羅_(巴西)|聖保羅市]]最近重整[[Paulista_Avenue|Paulista Avenue]]的行人道，它的葡式碎石鋪面就被換成便宜及普通的路面。

澳門的[[崗頂前地|崗頂前地]]行車路面亦曾經為葡式碎石鋪面，下雨天經常有[[電單車|電單車]]滑倒，備受批評，後來當局把行車路面改鋪成[[麻石|麻石]]路面。
