'''葡萄牙護照'''（{{lang-pt|'''Passaporte português'''}}），是由[[葡萄牙共和國|葡萄牙共和國]]政府向其公民發出的[[旅行證件|旅行證件]]。不過，葡萄牙公民在[[歐盟|歐盟]]國家、[[冰島|冰島]]、[[列支敦士登|列支敦士登]]、[[挪威|挪威]]及[[瑞士|瑞士]]旅行时，也可以使用葡萄牙居民[[身份證|身份證]]作为有效的身份证件。

==新版電子護照==
2006年8月28日，葡萄牙政府開始簽發'''葡萄牙電子護照'''（{{lang-pt|Passaporte Electrónico Português}}），葡萄牙護照加上了[[生物特徵護照|生物防偽特徵]]和一塊微型晶片，晶片上記錄了護照上所有的資料包括持有人的相片和[[指紋|指紋]]資料。而歐盟也規定歐盟成員國發出的護照必須在2008年2月28日前全部加上護照持有人的指紋資料。

自2017年7月起，葡萄牙政府簽發新版的電子護照，該電子護照採用了創新的主題設計與安全技術。護照的內頁印有在2015年12月或之前，被聯合國教科文組織認定的葡萄牙的物質與非物質文化遺產。

葡萄牙電子護照除了達到[[國際民航組織|國際民航組織]]對電子護照的要求外，並符合美國的[[旅遊許可電子系統|旅遊許可電子系統]]ESTA（Electronic System for Travel Authorization）豁免簽證入境要求。一般來說葡萄牙護照有效期為2年（小童版護照），有效期為5年（成人版護照），且不能續期，申請者需要重新申請過。而舊版本的護照是仍然有效的。

===封面===
護照使用深紅底色和金字，文字方面只使用[[葡文|葡文]]。根據歐盟標準，護照印上「{{lang|pt|União Europeia}}」即歐盟的葡文，下方印上葡萄牙的葡文名稱，最後則是護照的葡文「{{lang|pt|Passaporte}}」。而2006年版的封面均印有[[葡萄牙國徽|葡萄牙國徽]]喻意[[歐盟|歐盟]]的「十二顆五角金星環成一圓」及[[電子護照|電子護照]]的圖案，而封面及及封底在[[紫外光|紫外光]]下更可看到葡萄牙共和國葡文名稱（{{lang|pt|República Portuguesa}}）的首個葡文字母「{{lang|pt|RP}}」、喻意[[歐盟|歐盟]]的「十二顆五角金星環成一圓」及[[電子護照|電子護照]]的圖案。另外，由於該護照加入了[[生物特徵護照|生物防偽特徵]]，故在封面下方加上了生物特徵護照的標誌。另一方面，封面上也有防偽特徵的，就是[[隱形墨水|隱形墨水]]的印刷，該印刷只有在紫外光燈的照射下才可以看到。

===用戶資料頁===

用戶資料位於整本護照的第一頁，它記錄以下的資料：
* 護照類型
* 簽發國家代碼（PRT）
* 護照號碼
* 姓名
* 國籍（''Portuguesa''）
* 身高（米）
* 出生日期（日.月.年）
* 身份證號碼（和認辨證的一樣）
* 性別（男）或（女）	
* 出生地點（城市，國家）
* 發出日期（日.月.年）
* 發出地點
* 有效日期（日.月.年）
* 電子簽名

===簽證頁面===
整本護照雖然共有32頁（不包括用戶頁和資料頁），但只有28頁是可以用來做[[簽證|簽證]]的用途。其餘4頁用來印上了用戶頁資料對應歐盟所有國家語言的翻譯。而所有的簽證頁面都印上了葡萄牙國徽的[[水印|水印]]。

===語言===
跟從歐盟的標準，護照內包含了所有歐盟國家的[[歐盟語言|官方語言]]，不過封面上的文字只有葡文，但其翻譯則在第一頁上。而用戶資料頁就只使用了[[葡文|葡文]]、[[英文|英文]]和[[法文|法文]]，其他語言在護照最後的三頁上。

==類型==
葡萄牙電子護照共分為以下四種類型：
* 護照（{{lang|pt|Passaporte}}）
* 特殊（{{lang|pt|Especial}}）
* 外交（{{lang|pt|Diplomático}}）
* 臨時（{{lang|pt|Temporário}}）

==申請==
所有居住在葡萄牙及海外的葡萄牙公民也合資格申請葡萄牙護照及[[葡萄牙國民身份證|葡萄牙國民身份證]]。在葡萄牙，申請者需要親身到市民中心（葡文：{{lang|pt|Lojas do Cidadão}}）辦理申請護照及國民身份證。而居住在海外的葡萄牙公民也同樣要親身前往葡萄牙駐外國當地大使館或領事館申請護照及國民身份證。
==葡萄牙公民簽證要求==

==擁有葡萄牙國籍的澳門人==
[[葡萄牙|葡萄牙]]管理[[澳門|澳門]]四百多年以來，[[澳葡政府|澳葡政府]]簽發給澳門人的葡萄牙身份證明文件：葡萄牙護照及葡萄牙外國人士護照（{{lang|pt|Passaporte para Estrangeiros}}）。
===在澳門回歸中國之前，澳門人擁有葡萄牙國籍的3種方法===
#根據《[[葡萄牙國籍法|葡萄牙國籍法]]》規定，在1981年11月20日之前在葡萄牙及澳門出生的人士被自動推定為葡萄牙公民。因此不論此人其血統，種族，宗教信仰及父母的國籍等各種因素，均可以自動取得葡萄牙國籍。<ref>https://ukmacau.com/2016/02/24/%e8%91%a1%e7%b1%8d%e5%95%8f%e9%a1%8c%e6%87%b6%e4%ba%ba%e5%8c%85/</ref>
#父親或母親以血統方式過繼葡萄牙國籍給予所生子女。其父親或母親擁有葡萄牙國籍，便可以憑葡萄牙公民在海外所生的子女條件來把葡萄牙國籍過繼給予所生子女。
#以夫妻婚姻關係方式過繼葡萄牙國籍給予外國人配偶。根據葡萄牙國籍法，外國人配偶與葡萄牙公民結婚滿三年，沒有觸犯任何[[刑事罪行|刑事罪行]]及能操最低標準要求的[[葡萄牙語|葡萄牙語]]，便可以申請歸化取得葡萄牙國籍。<ref>https://www.cgportugal.org/cn/registocivil</ref>

[[澳門|澳門]]在回歸中國之前為葡萄牙屬地（1557年至1999年12月19日），昔日[[澳葡政府|澳葡政府]]簽發給合資格澳門人的葡萄牙身份證明文件。據《[[中葡聯合聲明|中葡聯合聲明]]》葡萄牙備忘錄表示，凡擁有葡萄牙國籍的澳門人，在[[澳門回歸中國|澳門回歸中國]]之後可以繼續使用並重新續領使用葡萄牙護照，以及可以把葡萄牙國籍過繼給予所生子女。之後任何人士不得再憑澳門的任何關係，從而取得葡萄牙國籍。<ref>[http://www.imprensa.macau.gov.mo/bo/i/88/23/dc/cn/ 《中華人民共和國政府和葡萄牙共和國政府關於澳門問題的聯合聲明》] {{webarchive|url=https://web.archive.org/web/20070701080046/http://www.imprensa.macau.gov.mo/bo/i/88/23/dc/cn/ |date=2007-07-01 }}</ref>另外，除持有葡萄牙護照的葡萄牙國籍澳門人，對於非葡萄牙國籍的澳門人，澳葡政府便發出葡萄牙外國人士護照（{{lang|pt|Passaporte para Estrangeiros}}）。在澳門回歸中國之後，理論上葡萄牙外國人士護照（{{lang|pt|Passaporte para Estrangeiros}}）是會即時失效，但考慮到擁有葡萄牙外國人護照的非葡萄牙國籍澳門人未必能夠在葡萄牙外國人士護照失效之前申請[[澳門特別行政區護照|澳門特別行政區護照]]，故葡萄牙與中國雙方討論後，同意把葡萄牙外國人士護照的有效日期延長至2001年12月19日<ref>[http://www.chinanews.com.cn/zhuanti/aomen/lawabc/p4-52.html 澳門特別行政區基本法ABC] {{zh-cn}}</ref>。
: 2015年，葡萄牙葡僑事務國務秘書西沙里奧（{{lang|pt|José de Almeida Cesário}}）所作出統計，全澳門64萬以上人口當中，有16萬8千名[[澳門人|澳門華人]]，中葡混血人士和[[澳門土生葡人|土生葡人]]擁有葡萄牙國籍。<ref>http://www.bbc.com/zhongwen/trad/china/2015/10/151004_portugal_election_macau</ref>
