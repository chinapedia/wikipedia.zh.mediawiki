{{人物信息框|政治運動 = [[緬甸國教法抗爭|緬甸國教法抗爭]]|姓名 = Aung Phyo Phyo|國籍 = {{RUM}}|組織 = [[全緬甸學生會聯盟|全緬甸學生會聯盟]]|宗教信仰 = [[上座部佛教|上座部佛教]]}}{{Burmese characters}}'''翁斐斐'''（[[缅甸语|緬甸語]]：ၿဖိဳးၿဖိဳးေအာင္ ，羅馬化拼音：Phyo Phyo Aung；{{bd|1988年}}）<ref>{{Cite web|url = http://burmawatchinternational1989.blogspot.tw/2011/06/phyo-phyo-aung-hnin-pwint-wai.html|title = Phyo Phyo Aung @ Hnin Pwint Wai|accessdate = 2015-12-22|author = |date = |publisher = }}</ref>，是一位緬甸學生運動的行動者。

1989年，在翁斐斐一歲大時，他的父親已經遭逮入獄二十年了。

目前翁斐斐擔任{{Translink|en|All Burma Federation of Student Unions|全緬甸學生會聯盟|全緬甸學生會聯盟}}（{{link-en|ABFSU|All Burma Federation of Student Unions}}）的秘書長，於2015年3月10日的一場學生和平示威活動中，與其他數十名學生一同遭到拘捕，學生們也反映他們不能與辯護律師聯絡。根據[[國際特赦組織|國際特赦組織]]香港分會的問張敘述，翁斐斐可能須面臨高達9年以上監禁。

== 親屬 ==
他的父親{{Translink|en|Nay Win Maung}}，同樣是一位行動者，且屢次遭到逮補、監禁，在軍政府統治下，在1969年被關進監獄，到1989年還沒被釋放出來<ref>{{Cite book|title = Burma's Forgotten Prisoners|last = |first = |publisher = |year = |isbn = 1564325172|location = |pages = }}</ref>。
