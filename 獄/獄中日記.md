《'''獄中日記'''》（[[越南語|越南語]]：'''''Nhật ký trong tù'''''／'''''Ngục trung nhật ký'''''），是[[越南共產黨|越南共產黨]]領導人物[[胡志明|胡志明]]的漢文詩集。1942年8月，胡志明在[[中國|中國]][[廣西|廣西]]被[[國民政府|重慶國民政府]]所捕，先後被關押在[[南寧|南寧]]、[[柳州|柳州]]、[[桂林|桂林]]等地的監獄達一年餘，至1943年9月獲釋，其間寫下百多首漢文詩，反映他的獄中生活及愛國、樂觀精神。在後世，《獄中日記》被視為研究胡志明思想、[[中越關係|中越關係]]的參考材料。

== 寫作背景 ==
{{see also|胡志明}}
[[File:Giap-Ho.jpg|thumb]]（右）]]
《'''獄中日記'''》是1942年8月至1943年9月期間，[[印度支那共產黨|印度支那共產黨]]（即[[越南共產黨|越南共產黨]]）領袖[[胡志明|胡志明]]在[[中國|中國]][[廣西|廣西]]被[[國民政府|國民政府]]關押獄中時創作而成。1942年8月，因[[越南|越南]]已受[[大日本帝國|日本]]支配，胡志明為進行抗日行動，前往中國，欲聯繫越南抗日人員，但進入邊境時被重慶政府拘捕。此後，胡志明被輾轉押送到廣西十三個縣三十個監獄，從詩中可知包括了[[靖西縣|靖西]]、[[德保縣|天保]]、[[隆安縣|隆安]]、[[同正縣|同正]]、[[南寧|南寧]]、[[來賓|來賓]]、[[柳州|柳州]]、[[桂林|桂林]]等地<ref>散見胡志明《獄中日記》，收錄於黃錚編注《胡志明漢文詩抄·注釋·書法》，廣西師範大學出版社。</ref>，因監獄條件惡劣，衣食不足，使他牙齒脫落，頭髮變白，形容憔悴。但雖如此，胡志明仍對越南獨立運動抱樂觀態度，深信自己「身体在獄中，精神在獄外，欲成大事業，精神更要大」<ref>胡志明《獄中日記·卷頭》，收錄於黃錚編注《胡志明漢文詩抄·注釋·書法》，廣西師範大學出版社，24頁。</ref>，於是在「囚中無所為」的時候「聊借吟詩消永日」<ref>胡志明《獄中日記·開卷》，收錄於黃錚編注《胡志明漢文詩抄·注釋·書法》，廣西師範大學出版社，25頁。</ref>，創作出漢文詩百多首<ref group="注">學者黃錚指出，胡志明在獄中寫的詩有133首，出獄後再寫一首而成134首（見黃錚《胡志明漢文詩抄·注釋·書法》「前言」，廣西師範大學出版社，2頁）。[[漢喃研究院|漢喃研究院]]網址[http://www.hannom.org.vn/detail.asp?param=1023&Catid=493 《獄中日記》]列出的詩有135首。</ref>，以反映他的經歷及心理狀況。<ref>越共中央直屬黨史研究院《胡志明主席──生平和業績簡略》，收錄於李家忠編譯《越南國父胡志明》，世界知識出版社，39─40頁。</ref>

胡志明的獄中詩，抄錄在一本淺綠色封面的日記本上，封面畫有一雙戴著手銬的手，並寫有「獄中日記」四個大字，是為留存後世的《獄中日記》。<ref>黃錚編注《胡志明漢文詩抄·注釋·書法》「前言」，廣西師範大學出版社，2頁。</ref>

== 詩作風格 ==
據[[中國大陸|中國大陸]]學者[[王曉平|王曉平]]的分析，《獄中日記》的詩，無視[[古體詩|古體詩]]與[[近體詩|近體詩]]的格律體制要求，也不完全按照[[現代詩|現代詩]]的寫法，而是胡志明以自己認為能夠寫出自身感受的方式，意到言止，在語言上接近現代[[白話詩|白話詩]]，某些句子還帶有中國[[五四時期|五四時期]]的新詩味道。<ref name="亞洲漢文學353">王曉平《亞洲漢文學·送別夕陽》，天津人民出版社，353頁。</ref>

此外，胡志明在獄中生活卻能寫出漢詩百多首，與他的詩作靈感及取材有閞。據王曉平分析有如下幾點。

一是它多用白話口話，隨時以幽默的語言寫下坐牢的感受。<ref name="亞洲漢文學352">王曉平《亞洲漢文學·送別夕陽》，天津人民出版社，352頁。</ref>例子如《世路難》裡稱：

{{cquote2|

余原代表越南民，

擬到中華見要人。

無奈風波平地起，

送余入獄作嘉賓。<ref>胡志明《獄中日記·世路難（三首）》，收錄於黃錚編注《胡志明漢文詩抄·注釋·書法》，廣西師範大學出版社，27─28頁。</ref>}}

二是胡志明從每天發生在監獄裡的事件提取詩材，將之與外面現實聯繫起來，寫出整個社會的黑暗與不合理，復以平淡樸素的語言加描畫。<ref name="亞洲漢文學352"/>例子有《賭》裡寫道：

{{cquote2|

民間賭博被官拉，

獄裡賭博可公開；

被拉賭犯常嗟悔，

何不先到這裡來。<ref>胡志明《獄中日記·賭》，收錄於黃錚編注《胡志明漢文詩抄·注釋·書法》，廣西師範大學出版社，34─35頁。</ref>}}

三是胡志明從枯燥無聊的囚徒生活中尋找詩情，寄望勝利和未來，顯示其鬥志不衰，豪情不減。<ref name="亞洲漢文學353"/>例子有《早》裡的詩句：

{{cquote2|

早起人人爭獵虱，

八鐘響了早餐開；

勸君且吃一個飽，

否極之時必泰來。<ref>胡志明《獄中日記·早（二首）》，收錄於黃錚編注《胡志明漢文詩抄·注釋·書法》，廣西師範大學出版社，28頁。</ref>}}

== 後世評價 ==
在後世學界，都視《獄中日記》為研究[[胡志明|胡志明]]思想理念的重要材料。越共學者指出本書「飽含著一位偉大革命戰士崇高的思想感情中的『鋼鐵』內涵」。<ref>越共中央直屬黨史研究院《胡志明主席──生平和業績簡略》，收錄於李家忠編譯《越南國父胡志明》，世界知識出版社，41頁。</ref>在[[中國大陸|中國大陸]]，史学界認為它是越南、中國現代關係史的珍貴史料和[[國際主義|國際主義]]、[[愛國主義|愛國主義]]詩篇<ref>《東南亞歷史詞典·「獄中日記」條》，上海辭書出版社，324頁。</ref>，可從中了解胡志明的革命信念、世界觀和方法論。<ref>黃錚編注《胡志明漢文詩抄·注釋·書法》「前言」，廣西師範大學出版社，3頁。</ref>文學界學者[[王曉平|王曉平]]形容包括《獄中日記》在內的胡志明、[[潘佩珠|潘佩珠]]漢文詩，都是「越南漢文學落幕前的壯歌」。<ref>王曉平《亞洲漢文學·送別夕陽》，天津人民出版社，345頁。</ref>[[臺灣|臺灣]]作家[[楊碧川|楊碧川]]認為，它能令讀者「從中讀到一位革命家的偉大心靈，並了解當時中國獄政的黑暗面」<ref>楊碧川《胡志明與越南獨立》，一橋出版社，191。</ref>。

== 流傳及出版情況 ==
[[胡志明|胡志明]]《獄中日記》詩稿的原件，被收藏在[[越南革命博物館|越南革命博物館]]保存，從1959年起，[[越南國家文學院|越南國家文學院]]開始進行整理工作，1960年翻譯《獄中日記》成[[越南語|越南語]]，同年有[[越南外文出版社|越南外文出版社]]（中文本）和[[越南文學出版社|越南文學出版社]]（中越文對照本）分別出版，這兩個版本均收錄了100首詩。其後，越南國家文學院又作補充翻譯的工作，重新考訂和公開之前未發表的獄中詩。1983年越南文學出版社推出新版本的《獄中日記》，收錄了113首。1990年，越南國家文學院繼續補充翻譯，由越南文學出版社出版134首詩版本的《獄中日記》。1995年，越南文學出版社出版的《胡志明文學選集》，也收錄了《獄中日記》及其他36首漢文詩。<ref>黃錚編注《胡志明漢文詩抄·注釋·書法》「前言」，廣西師範大學出版社，2─3頁。</ref>[[漢喃研究院|漢喃研究院]]的網站內，可閱讀全部的《獄中日記》。<ref>[http://www.hannom.org.vn/detail.asp?param=1023&Catid=493 漢喃研究院──《獄中日記》]。</ref>

在[[中國大陸|中國大陸]]，早在1960年，[[北京|北京]][[人民文學出版社|人民文學出版社]]出版了《“獄中日記”詩抄》，當中收錄胡志明獄中詩100首，其後曾有再版。<ref>黃錚編注《胡志明漢文詩抄·注釋·書法》「前言」，廣西師範大學出版社，3頁；胡志明《“獄中日記”詩抄》「出版說明」，北京人民文學出版社。</ref>2003至2004年間，學者黃錚根據越南方面的研究成果，編注《胡志明漢文詩抄·注釋·書法》，當中收錄133首獄中詩<ref group="注">黃錚不把《新出獄，學登山》列入獄中詩。</ref>和其他36首漢詩，由[[廣西師範大學出版社|廣西師範大學出版社]]出版。<ref>黃錚編注《胡志明漢文詩抄·注釋·書法》「前言」，廣西師範大學出版社，4頁。</ref>在[[臺灣|臺灣]]地區，作家[[楊碧川|楊碧川]]著作《胡志明與越南獨立》裡，附有一部份《獄中日記》的詩作。<ref>楊碧川《胡志明與越南獨立》，一橋出版社，191─227。</ref>
