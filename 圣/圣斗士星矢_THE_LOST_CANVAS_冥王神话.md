《'''聖鬥士星矢 THE LOST CANVAS 冥王神話'''》是由[[日本漫畫家|日本漫畫家]][[手代木史織|手代木史織]]描繪的[[日本漫畫|日本漫畫]]作品。2006年8月於《周刊少年Champion》以每周連載形式開始連載自己的作品《聖鬥士星矢 THE LOST CANVAS 冥王神話》。故事講述距離《[[聖鬥士星矢|聖鬥士星矢]]》本篇所處年代的243年之前的聖戰的故事。受到天秤座的黃金聖鬥士童虎的引薦，天馬前往希臘聖域接受聖鬥士的訓練。在聖域，天馬遇到了兒時的玩伴薩莎。薩莎，原來竟然是女神雅典娜的轉世……

由於此作作者為手代木史織，與車田正美所作的漫畫《ND冥王神話》可以看作完全自我獨立的漫畫，並無關聯之處。雖然故事時間點都差不多是18世紀，並有相同人物。但本作可以當作手代木根據車田給出的聖鬥士資料，融合自己的想法與設定而構建出的全新聖鬥士世界。

== 劇情簡介 ==
故事以18世紀的[[歐洲|歐洲]]為舞台，在當時的[[意大利|意大利]]某小鎮上，有著[[日本|日本]]血統的孤兒'''天馬'''和喜愛繪畫的'''亞倫'''是一對摯友。某日'''亞倫'''受到大聖堂的神父的指引，去到一處被稱為'''「地面上的極樂淨土」'''的地方尋找'''「真實之紅」'''。在那里'''亞倫'''遇到了一個叫'''[[潘多拉|潘多拉]]'''的神秘少女，自此'''亞倫'''的身體開始發生了變化。

另一方面，受到'''黄金圣斗士 [[天秤座|天秤座]] 童虎'''的引荐，'''天马'''决定前往[[希腊|希腊]]圣域接受圣斗士的训练。在圣域，'''天马'''遇到了儿时的玩伴，同时也是'''亚伦'''的妹妹'''紗莎'''。几年前被人收养的'''紗莎'''，原来竟然是'''女神 [[雅典娜|雅典娜]]'''的转世。
2年之後，'''天馬'''終於成為天馬星座的聖鬥士。另一方面，受到'''潘多拉'''引導的'''亞倫'''，體內的'''冥王 哈帝斯'''的靈魂覺醒，並向聖域發動全面進攻……

== OVA ==
在[[OVA|OVA]]《[[圣斗士星矢|圣斗士星矢 冥王哈迪斯 极乐净土篇]]》于2008年8月发售完最后一卷，终于结束了这部延续多年的大作之后没几天，一个名为《[[圣斗士星矢_冥王神话|圣斗士星矢 冥王神话]]》的新作又展现在广大圣斗士FANS的面前。作为[[漫画|漫画]]《[[圣斗士星矢|圣斗士星矢]]》的[[前传|前传]]，《冥王神话》主要分别由原作者[[车田正美|车田正美]]本人执笔的《[[圣斗士星矢_NEXT_DIMENSION_冥王神话|NEXT DIMENSION]]》，以及由漫画家[[手代木史织|手代木史织]]编绘的《[[圣斗士星矢_THE_LOST_CANVAS_冥王神话|THE LOST CANVAS]]》。这两部漫画描述的都是发生在《圣斗士星矢》243年前的[[圣战|圣战]]，只不过在设定上各自有些不同。作品的主人公包括当时天马圣斗士天马、天马的挚友亚伦、[[雅典娜|雅典娜]]的转生薩莎，以及在《[[圣斗士星矢|冥王哈迪斯 十二宫篇]]》中为大家所熟知的[[黄金圣斗士|黄金圣斗士]]童虎和史昂。

而此次即将动画化的，正是由手代木史织所绘画的《THE LOST CANVAS》。由于作品的主人公已经与《圣斗士星矢》中的不同，所以这次的[[声优|声优]]大换血也早早成为了FANS们关注的问题。根据目前已经公布的声优名单，四大主角将由[[柿原彻也|柿原彻也]]、[[下野纮|下野纮]]、[[平野绫|平野绫]]以及[[水树奈奈|水树奈奈]]分别担任配音。

動畫《聖鬥士星矢 THE LOST CANVAS 冥王神話》由動畫公司[[TMS_Entertainment|TMS Entertainment]]制作，從2009年春天開始以[[OVA|OVA]]的形式發行。DVD第一章共6卷13話，第1卷於6月24日發售。六卷發售完畢後，全章於日本的有線電視頻道Kids Station等播放。第2章於2011年2月起每月發售<ref>[http://loscan.blog.so-net.ne.jp/2010-12-09-1 官方網誌]</ref>，全六卷。
