'''圣奥斯定堂'''、'''聖奧古斯丁教堂'''（{{lang-en|San Agustin Church}}）位于[[马尼拉|马尼拉]][[王城区|王城区]]的古城墙内，是1607年落成的<ref name=al1>Aluit, p. 41</ref>[[罗马天主教|罗马天主教]][[奥斯定会|奥斯定会]]教堂，教堂建築本身是[[菲律宾|菲律宾]]最古老的建筑物<ref name=torres2/>。

1976年，圣奥斯定堂被菲律宾政府命名为国家历史名勝<ref name=lay4/>。1993年，圣奥斯定堂等西班牙殖民时期修建的四座菲律宾教堂被列为[[世界遗产|世界遗产]]，称为“[[菲律宾的巴洛克教堂|菲律宾的巴洛克教堂]]”。<ref name = 吴杰伟_2012> {{ cite web | title = 东南亚教堂艺术的表现形式与本土化特点 | url = http://sea.cssn.cn/webpic/web/sea/upload/2012/07/d20120725135249418.pdf | author = 吴杰伟 | date = 2012-04-25 }} </ref>

==历史==
目前的教堂已经是第三次修建的圣奥斯定堂<ref name=lay3>Layug, p. 83</ref>。圣奥斯定堂是[[西班牙人|西班牙人]]在[[吕宋岛|吕宋岛]]修建的第一座宗教建筑<ref name=ph>{{cite web |title=San Agustin Church (Intramuros, Manila) |url=http://heritageconservation.wordpress.com/2006/07/29/san-agustin-church-intramuros-manila |author=Heritage Conservation Society |accessdate=2008-03-24 }}</ref>，第一次修建聖奧古斯丁教堂是1571年由西班牙人用[[竹子|竹子]]和芦苇建成，但是在1574年12月中國海盜[[林鳳_(潮州饒平)|林鳳]]逃亡到菲律賓時，进攻马尼拉並以大火毁壞聖奧古斯丁教堂<ref name=torres2>Torres, p. 62</ref><ref name=al0>Aluit, p. 40</ref><ref name = 吴杰伟_2012 />。第二次修建教堂用木头建造<ref name=al0/>，于1583年2月被毁，起火原因是在[[菲律宾总督列表|西班牙总督]]Gonzalo Ronquillo de Peñalosa的葬礼上，蜡烛点燃了棺材布<ref name=torres2/>。

第三次修建奧古斯丁教堂時，受奥斯定会位於當時同樣被西班牙殖民的[[墨西哥|墨西哥]]{{tsl|en|Puebla Cathedral|普埃布拉大教堂}}建築設計影響，胡安·馬西亞斯（Juan Macias）决定用石料重建教堂以及毗邻的[[修道院|修道院]]。工程开始于1586年，<ref name=ph/><ref name=al0/>由于缺乏资金、材料以及石匠，工程进度缓慢，<ref name=al0/> 到1604年修道院完成<ref name = 吴杰伟_2012 />，1607年1月19日聖奧古斯丁教堂正式宣布完成，命名为'''马尼拉圣保禄堂'''。<ref name=al0/> 胡安·馬西亞斯（Juan Macias）在教堂完成之前去世<ref name=al1>Aluit, p. 41</ref>。

1762年，在[[七年战争|七年战争]]期间，占领马尼拉的[[英国|英国]]军队洗劫了圣奥斯定堂<ref name=torres3>Torres, p. 63</ref>。1854年，由建筑师 Luciano Oliver修复了教堂<ref name=ph/>。九年后，1863年6月3日，一场大地震毁灭了马尼拉，圣奥斯定堂为该市唯一幸存的公共建筑<ref>Fernandez, p. 216</ref>。1880年马尼拉再次遭受一系列大地震的袭击 - 从7月18日到20日。这一次，地震使教堂的钟楼出现裂纹<ref>Hannaford, p. 21</ref>，后经修复<ref>Laya and Gatbonton, p.102.</ref>。此前该教堂还经受住了其他地震的袭击：1645年，1699年，1754年，1796年，1825年和1852年。

1898年8月18日，[[美西战争|美西战争]]后，西班牙总督Fermin Jaudenes 在这座教堂准备马尼拉向美国投降的条款<ref name=lay3/><ref name=torres3/>。在[[第二次世界大战|第二次世界大战]]期间，日本占领了菲律宾，圣奥斯定堂被改为关押囚犯的集中营<ref name=lay3/>。在[[马尼拉战役_(1945年)|马尼拉战役]]的后期，数百名王城区居民和神职人员作为人质被[[日本军队|日本军队]]关押扎起这座教堂，其中许多人在长达三周的战役中遭到杀害<ref name=lay3/>。这座教堂本身在美国和菲律宾军队的轰击中幸存了下来，只有屋顶受损，成为城墙内七座教堂中唯一幸存的一座<ref name=lay3/>，但是毗邻的修道院被完全摧毁，于20世纪70年代改建为博物馆<ref name=lay4>Layug, p. 84</ref><ref name=torres3/>。

==特点==
圣奥斯定堂长67.15米，宽24.93米<ref name=lay4/>。它的椭圆形的基础使它能够承受那些摧毁马尼拉其他许多教堂的众多地震。<ref name=lay4/>有人说，设计是源自于[[墨西哥|墨西哥]]的奥斯定会教堂<ref name=ph/>。其立面其貌不扬，甚至被批评为“缺乏风度和魅力”，但是拥有华丽的木门雕刻<ref name=ph/>。教堂的庭院有几只花岗岩雕刻的狮子，是加入天主教的华人赠送<ref name=lay3/>。

教堂平面为拉丁十字<ref name=lay4/>，内部有14个小堂，精美的天花板画是1875年[[意大利|意大利]]艺术家 Cesare Alberoni 和 Giovanni Dibella的作品<ref name=lay5>Layug, p. 85</ref>。

1965年教堂為了紀念天主教在菲律賓傳教400年，展出16世紀到19世紀期間百座天主教堂照片後，開始走向收藏菲律賓與西班牙珍貴歷史與藝術精品的博物館，也因此在1976年被政府定為國家歷史名勝（{{lang-en|National Historical Landmark}}）。<ref name = 吴杰伟_2012 />
