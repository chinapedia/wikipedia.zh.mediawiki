'''圣若望主教座堂''' ({{lang-mt|Kon-Katidral ta’ San Ġwann}})是[[罗马天主教|罗马天主教]]在[[马耳他|马耳他]]的[[主教座堂|主教座堂]]之一，也是首都[[瓦莱塔|瓦莱塔]]的地标建筑之一。它是由[[医院骑士团|马耳他骑士团]]兴建于1573年到1578年，为骑士团的修道院教堂，后来随着时间的推移，取得了与[[姆迪纳|姆迪纳]]总主教座堂平等的地位。在19世纪20年代，马耳他主教获准在姆迪纳之外，可使用圣若望堂作为另一个主教座堂。教堂的设计者是[[马耳他|马耳他]]军事建筑师卡萨尔，他也设计了瓦莱塔其他一些重要建筑。

== 立面 ==
圣若望主教座堂的立面很简朴，是在1565年围城结束之后立即建造。

== 内部 ==
其内部与立面形成强烈对比，为装饰华丽的巴洛克风格 <ref>[http://www.trekearth.com/gallery/Europe/Malta/photo21188.htm St. Johns Co-Cathedral ]</ref>，主要出自卡拉布里亚艺术家和骑士马蒂亚普雷蒂之手。普雷蒂设计了雕刻错综复杂的石头墙，拱形的天花板和描绘圣若望生活场景的侧祭台。

正门附近有大教長[[锡耶纳|锡耶纳]]的Marc'Antonio Zondadari的纪念碑，他是教宗[[亚历山大七世|亚历山大七世]]的侄子。<gallery perrow=6>

== 小堂 ==
[[File:Malta-Valletta-StJohnChapel.jpg|thumb]]
圣若望主教座堂内有8个小堂，分别供奉骑士团8个部分的主保圣人。在座堂左侧有以下的小堂：

盎格鲁-巴伐利亚小堂原名圣物小堂，保存几个世纪以来骑士团获得的圣物。

[[普罗旺斯|普罗旺斯]]小堂供奉[[圣弥额尔|圣弥额尔]]。

[[法国|法国]]小堂供奉[[圣保禄|圣保禄]]。

[[意大利|意大利]]小堂供奉[[亚历山大的加大肋纳|亚历山大的加大肋纳]]。

[[德国|德国]]小堂供奉[[主顯節|基督显容]]。

在座堂右侧有以下的小堂：

圣体小堂原来供奉圣母。

[[奥弗涅|奥弗涅]]小堂供奉[[圣巴斯弟盎|圣巴斯弟盎]]。

[[阿拉贡|阿拉贡]]小堂供奉[[圣乔治|圣乔治]]。

[[卡斯蒂利亚王国|卡斯蒂利亚]]、[[莱昂王国|莱昂]]和[[葡萄牙|葡萄牙]]小堂供奉[[圣雅各伯|圣雅各伯]]。

== 艺术品 ==
[[File:Michelangelo_Caravaggio_021.jpg|thumb]]
该堂最著名的艺术品是[[卡拉瓦乔|卡拉瓦乔]]的杰作《[[被斬首的聖施洗者約翰_(卡拉瓦喬)|被斩首的圣若翰洗者]]》（1608年）。这是由画家署名的唯一一件画作，应用了明暗对比风格。
