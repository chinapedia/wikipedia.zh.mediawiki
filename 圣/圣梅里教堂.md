'''圣梅里教堂''' ('''''Église Saint-Merri''''') 是[[巴黎|巴黎]]的一座[[教堂|教堂]]，位于[[右岸|右岸]]繁忙的圣马丁街<ref>{{cite web |url=http://www.saintmerri.org/ |title=存档副本 |accessdate=2010-08-19 |deadurl=yes |archiveurl=https://web.archive.org/web/20100819004331/http://www.saintmerri.org/ |archivedate=2010-08-19 }}</ref>。

这座教堂供奉的是8世纪的[[欧坦|欧坦]]修道院院长 Medericus，他前往巴黎朝圣，后来死在那里。884年，Medericus又名梅里，被称为右岸的[[主保圣人|主保圣人]]<ref>{{cite web |url=http://www.saintmerri.org/ |title=存档副本 |accessdate=2010-08-19 |deadurl=yes |archiveurl=https://web.archive.org/web/20100819004331/http://www.saintmerri.org/ |archivedate=2010-08-19 }}</ref>。

目前教堂建于1500年到1550年，16世纪[[哥特式建筑|哥特式]]风格，受英国垂直风格影响。中殿的窗户是16世纪初的作品，讲道坛是 Slodtz的作品，于1753年制造<ref>{{cite web |url=http://www.saintmerri.org/ |title=存档副本 |accessdate=2010-08-19 |deadurl=yes |archiveurl=https://web.archive.org/web/20100819004331/http://www.saintmerri.org/ |archivedate=2010-08-19 }}</ref>。

管风琴在1781年由著名的管风琴制造者Cliquot重制。由[[卡米爾·聖-桑|卡米爾·聖-桑]]演奏。1832年，在反对[[七月王朝|七月王朝]]的共和党起义期间，这座教堂成为街垒战的场地<ref>{{cite web |url=http://www.saintmerri.org/ |title=存档副本 |accessdate=2010-08-19 |deadurl=yes |archiveurl=https://web.archive.org/web/20100819004331/http://www.saintmerri.org/ |archivedate=2010-08-19 }}</ref>。

钟楼内有巴黎最古老的[[钟|钟]]，1331年铸造，在[[法国革命|法国革命]]中得以幸存下来<ref>{{cite web |url=http://www.saintmerri.org/ |title=存档副本 |accessdate=2010-08-19 |deadurl=yes |archiveurl=https://web.archive.org/web/20100819004331/http://www.saintmerri.org/ |archivedate=2010-08-19 }}</ref>。

今天该教堂仍是一个举行礼拜的地点，并且设有雷阿尔-波布牧民中心<ref>{{cite web |url=http://www.saintmerri.org/ |title=存档副本 |accessdate=2010-08-19 |deadurl=yes |archiveurl=https://web.archive.org/web/20100819004331/http://www.saintmerri.org/ |archivedate=2010-08-19 }}</ref>。
