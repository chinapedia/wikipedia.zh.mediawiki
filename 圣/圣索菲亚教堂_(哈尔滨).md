'''圣索菲亚教堂'''（{{lang-rus|Софийский собор}}）位于[[中国|中国]][[黑龙江省|黑龙江省]][[哈尔滨市|哈尔滨市]][[道里区|道里区]]，近[[中央大街|中央大街]]，是[[远东|远东]]地区最大的[[东正教|东正教]][[教堂|教堂]]。教堂现已辟为'''哈尔滨建筑艺术馆'''，不再行使宗教功能。<ref name=ency>{{cite encyclopedia |year=2009|title =哈尔滨东正教堂|encyclopedia=[[中国大百科全书|中国大百科全书]]|publisher=中国大百科全书出版社|location=北京|edition=第二版|ISBN=978-7-5000-7958-3}}</ref>“圣索菲亚”词源为“神圣的智慧”。<ref name=wisdom/>

==历史==
圣索菲亚教堂于1907年3月17日奠基修建，最初是[[俄罗斯帝国|俄罗斯帝国]]东西伯利亚第四步兵师修建的随军教堂；同年，俄国[[犹太|犹太]]裔茶叶商人伊·费·奇斯佳科夫<ref>{{Cite web |url = http://zarubezhje.narod.ru/tya/ch_035.htm |script-title = ru:Чистяков, Илья Федорович |language = ru |deadurl = no |archiveurl = https://web.archive.org/web/20160304203806/http://zarubezhje.narod.ru/tya/ch_035.htm |archivedate = 2016-03-04 }}</ref>（{{lang|ru|И. Ф. Чистяков}}）出资，在原基础上修建全木结构教堂，于8月2日竣工。<ref name=ency/><ref name=DSJ>{{Citation |author=徐苏宁 编|chapter=哈尔滨城市建筑大事记|title=中国近代建筑总览·哈尔滨篇|pages=120-140|publisher=中国建筑工业出版社|place=北京|publication-date=1992|ISBN=7112014824}}</ref>

1912年11月4日，教堂重建竣工，在原木墙外部包砌砖墙、形成砖木结构式教堂；[[海参崴|海参崴]]和[[堪察加|堪察加]]教区[[大主教|大主教]]{{Link-ru|叶夫谢维·尼科利斯基|Евсевий (Никольский)}}（{{lang|ru|Евсевий Никольский}}）与[[圣尼古拉教堂_(哈尔滨)|圣尼古拉教堂]]的[[神父|神父]]佩卡尔斯基<ref>{{Cite web|url=http://zarubezhje.narod.ru/mp/p_128.htm|script-title=ru:Протоиерей Леонтий Пекарский|language=俄语|deadurl=no|archiveurl=https://web.archive.org/web/20160304202034/http://zarubezhje.narod.ru/mp/p_128.htm|archivedate=2016-03-04}}</ref>（{{lang|ru|Пекарский}}）等为之祝圣。<ref name=DSJ/><ref name=CHS>{{cite journal zh|author=常怀生|year=1998|title=圣·索菲亚教堂：哈尔滨建筑艺术馆|journal=建筑学报|issue=3|pages=37-39}}</ref><ref name=ZJZ>{{Citation |chapter=宗教·东正教·东正教堂·圣索菲亚教堂|title=哈尔滨市志|publisher=黑龙江人民出版社|place=哈尔滨|publication-date=1999|ISBN=9787207043474}}</ref>

1923年9月27日，教堂再次重建。在哈的俄国建筑师米哈伊尔·马·奥斯科尔科夫（{{lang|ru|Михаил Матвеевич Осколков}}）设计了教堂，外形仿造{{Link-ru|瓦西里·科夏科夫|Косяков, Василий Антонович}}（{{lang|ru|Василий Косяков}}）的[[圣彼得堡|圣彼得堡]]{{Link-ru|主显教堂|Богоявленская церковь (Санкт-Петербург)}}（{{lang|ru|Богоявленская церковь}}）。
历时9年时间，最终于1932年11月25日建成。<ref name="DSJ" /><ref>{{Cite web|url = http://www.archjournal.ru/rus/04%2049%202007/vklad.htm|title = 俄罗斯建筑师对哈尔滨建筑的贡献|language = ru|deadurl = no|archiveurl = https://web.archive.org/web/20160304194700/http://www.archjournal.ru/rus/04%2049%202007/vklad.htm|archivedate = 2016-03-04}}</ref>

1934年8月29日，圣索菲亚教堂成为培养神职人员的哈尔滨东正教会神学院。<ref name="ZJZ" />

1958年，教堂被关闭。<ref>{{cite journal zh |author=王志军、姜玉洪|year=2010|title=论20世纪早期东正教对哈尔滨城市文化品格的影响|journal=西伯利亚研究|issue=1|pages=36-41}}</ref>“[[文化大革命|文化大革命]]”期间，教堂深受破坏，圣器、文物全部丢失，并先后被用作[[哈尔滨话剧院|哈尔滨话剧院]]练功房和[[哈尔滨市第一百货商店|哈尔滨市第一百货商店]]仓库。<ref name=LSYG>[http://www.sofia.com.cn/Article_Show.asp?ArticleID=17 哈尔滨市建筑艺术馆官网：索菲亚教堂历史沿革]{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>且被四周居民楼遮蔽，长期不为人所知。<ref name=ency/>

1986年，索菲亚教堂被列为[[哈尔滨市保护建筑列表|哈尔滨市I类保护建筑]]。<ref name=LSYG/>1996年，被列入第四批[[全国重点文物保护单位|全国重点文物保护单位]]。<ref>[http://www.sach.gov.cn/col/col1647/index.html 国家文物局：国务院关于公布第四批全国重点文物保护单位的通知] {{webarchive|url=https://web.archive.org/web/20141008064915/http://www.sach.gov.cn/col/col1647/index.html |date=2014-10-08 }}</ref>1997年，拆除周边具有火灾隐患的相邻建筑1.4万平方米，辟建广场7000平方米，并将教堂改作哈尔滨市建筑艺术馆之用。<ref name=ency/>2007年，索菲亚教堂景区获评[[国家4A级旅游景区|国家4A级旅游景区]]。<ref>[http://www.cnta.gov.cn/html/2008-7/2008-7-2-10-28-07283.html 国家旅游局：关于批准发布北京平谷青龙山旅游区等87家景区为国家4A级旅游景区公告的决定] {{webarchive|url=https://web.archive.org/web/20130201084846/http://www.cnta.gov.cn/html/2008-7/2008-7-2-10-28-07283.html |date=2013-02-01 }}</ref>

2006年教堂广场再次进行了扩建改造，喷泉升级后冬季可以喷雪，广场的面积也扩大了许多。<ref>{{Cite web|url=http://news.xinhuanet.com/photo/2006-12/11/content_5467873.htm|title=哈尔滨建筑艺术广场扩建改造完工 洋溢欧陆风情|accessdate=2016-08-23|work=news.xinhuanet.com|deadurl=no|archiveurl=https://web.archive.org/web/20160914215235/http://news.xinhuanet.com/photo/2006-12/11/content_5467873.htm|archivedate=2016-09-14}}</ref>

2018年7月3日起，教堂开始97年后的第一次修缮工作，工期一年。<ref>{{Cite web|url=http://www.sohu.com/a/237526171_349336|title=索菲亚教堂被今起“保护性修缮”｜为啥旅游旺季开修？还能拍出美美哒照片吗?|accessdate=2018-08-30|date=2018-06-24|work=www.sohu.com}}</ref>

==建筑特点==
圣索菲亚教堂为[[清水红砖|清水红砖]]砌成的砖石结构，屋顶为铁皮屋面刷绿色[[油漆|油漆]]。<ref name=CHS/>

平面呈[[拉丁十字|拉丁十字]]形，南北对称，东端短、西端长；南北宽约28米，东西长约42米。西端设朝西的正门，东端设与正门相背的后门，南北两翼均有朝西开的耳门。拉丁十字的中轴线上为主礼拜厅，东端为主祭台；南北两翼各设一个小礼拜厅。二层设唱诗台。总面积721平方米，可容纳1500人从事宗教活动。<ref name=CHS/>

十字相交处形成四个砖垛，上置外径10米的圆形鼓座，戴以巨型[[穹顶|穹顶]]，外部罩以[[洋葱形圆顶|“洋葱头”]]式窟窿顶罩，罩上有十字架，至十字架顶端通高53.35米。正门顶端则设“{{Link-en|帐篷顶|Tented roof}}”式尖塔，用作钟楼。两翼和后门亦设“帐篷顶”尖塔。中央穹顶最高，钟塔略低，其余三座尖塔最低。<ref name=CHS/>

另有地下室，设库房和锅炉房等。<ref name=CHS/>

圣索菲亚教堂结合了[[拜占庭式建筑|拜占庭式建筑]]风格（平面呈十字架形；方形房间的墩柱上设穹顶；中央大穹顶和四面4个小穹顶）和[[俄罗斯|俄罗斯]]民间木建筑风格（“洋葱头”和“帐篷顶”），是一座型制严整、风貌地道的俄罗斯式东正教堂。<ref>{{cite book |author=陈志华 |year=2010 |title=外国建筑史|edition=第4版|publisher=中国建筑工业出版社|location=北京|ISBN=978-7-112-11293-7}}</ref>{{rp|97-99、239}}<ref>{{cite book |author =［英］大卫·沃特金 |others = 傅景川 等 译 |year=2004 |title=西方建筑史|publisher=吉林人民出版社|location=长春|ISBN=7-206-04386-0}}</ref>{{rp|80-82}}<ref>{{Citation |author=侯幼斌、刘松茯|chapter=哈尔滨近代建筑概说|title=中国近代建筑总览·哈尔滨篇 |pages=1、12 |publisher=中国建筑工业出版社|place=北京|publication-date=1992 |ISBN=7112014824}}</ref>[[日本|日本]][[建筑史|建筑史]]学家{{Link-ja|西泽泰彦|西澤泰彦}}认为，圣索菲亚教堂在规模之大、构思之完整方面，足可以与[[莫斯科|莫斯科]][[圣瓦西里大教堂|圣瓦西里大教堂]]相比。<ref>{{Citation |author=西泽泰彦 |others = 张复合 译 |chapter=哈尔滨近代建筑的特色|title=中国近代建筑总览·哈尔滨篇|pages=13-18|publisher=中国建筑工业出版社|place=北京|publication-date=1992|ISBN=7112014824}}</ref>
