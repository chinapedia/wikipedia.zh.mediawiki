'''圣保罗人大道'''（{{lang-pt|Avenida Paulista}}），或音譯為'''保利斯塔大道'''，是[[巴西|巴西]]最大都市[[圣保罗_(巴西)|圣保罗]]的一條繁華街道、也是該城最重要的經濟與金融中樞、觀光景點<ref>{{cite web

|publisher=Veja São Paulo
|accessdate=
|last =
|first =
}}</ref>和娛樂中心之一，长2.8公里，呈西北－東南走向，于1891年正式开通<ref name="中國拉丁美洲研究網">{{Cite web |url=http://ilas.cass.cn/cn/lmgl/content.asp?InfoID=10229 |title=圣保罗的金融街——保利斯塔（PAULISTA）大街 |publisher=中國社會科學院拉丁美洲研究所 |date=2009-06-12 |accessdate=2014-04-01 |deadurl=yes |archiveurl=https://web.archive.org/web/20110909082231/http://ilas.cass.cn/cn/lmgl/content.asp?InfoID=10229 |archivedate=2011-09-09 }}</ref>。大道周邊的商辦、企業總部、[[摩天大樓|摩天大廈]]與文化設施雲集，沿線地段還分布有全[[南美洲|南美洲]]最昂貴的幾處[[房地產|房地產]]<ref name="劇變">{{Cite book |title=地球相簿：劇變的一百年|author=皮爾斯（{{lang|en|Fred Pearce}}）|others=黃中憲譯|publisher=貓頭鷹出版|year=2008|isbn=978-986-7001-70-2}}</ref>{{rp|p99}}<ref name="brazilmycountry">{{Cite web |url=http://www.brazilmycountry.com/sao-paulo-attractions/paulista-avenue/ |title=Paulista Avenue – Avenida Paulista |publisher=BrazilMyCountry.com |accessdate=2014-04-01}}</ref>。[[聖保羅地鐵|聖保羅地鐵]]的[[聖保羅地鐵2號線|2號線]]沿著該路鋪設，設有3座車站。

==歷史==
[[File:Jules_Victor_André_Martin_-_Avenida_Paulista_no_Dia_de_Sua_Inauguração,_1891.jpg|thumb]]
[[File:Avenida_Paulista_1902.jpg|thumb]]
聖保羅人大道是由[[烏拉圭|烏拉圭]]移民巴西的[[工程師|工程師]]{{Link-pt|艾吾傑紐·利瑪|Joaquim Eugênio de Lima}}與友人合夥投資修建的<ref name="中國拉丁美洲研究網"></ref>，於1891年12月8日竣工，以使渴望搬離擁擠城區、尋覓新住所的市民們能夠遷到郊外，當時也是聖保羅快速擴張的年代，[[咖啡|咖啡]]產業和[[工業化|工業化]]帶動了該市的發展<ref>{{Cite book |title=Planning Latin America's Capital Cities 1850-1950 |author=Almandoz, Arturo |chapter=Rio de Janeiro and São Paulo: Times of Disruption (1870-1890)}}</ref>，包括該大道在內的街路、樓房與中央公園等新建設施如雨後春筍般冒出<ref name="GoHouse">{{Cite web |url=http://gohouse.com/attractions/avenidapaulista-saopaulo-saopaulo-touristattraction |title=Avenida Paulista in São Paulo, São Paulo |publisher=GoHouse.com |accessdate=2014-04-01 |deadurl=yes |archiveurl=https://web.archive.org/web/20140407063230/http://gohouse.com/attractions/avenidapaulista-saopaulo-saopaulo-touristattraction |archivedate=2014-04-07 }}</ref>。

大道通車後，街名定為「聖保羅人」以向該市的歷代居民致敬。啟用初期的路面分流為三種車道：[[路面電車|電車路軌]]、[[馬車|馬車]]道和[[人行道|人行道]]<ref name="GoHouse"></ref>，並且鋪設成林蔭大道，周圍地帶視野開闊、氛圍寧靜，許多咖啡業巨亨或園主在路旁購置豪宅<ref name="劇變"></ref>{{rp|p98}}<ref>{{Cite book |title=南美 |author=戴月芳|publisher=錦繡出版事業股份有限公司|year=1991 |isbn=957-9425-17-5}}</ref>{{rp|p38}}。該街後來的交通地位越來越重要，因此在1909年時成了聖保羅第一條有完整鋪面的市區道路<ref name="GoHouse"></ref>。

到了1950年代以後，隨著聖保羅人大道成了大型銀行和公司青睞的辦公室選址<ref name="Berkeley">{{Cite web |url=http://www.ced.berkeley.edu/courses/sp08/arch201/UrbanFoodStudioProgramApril07.pdf |title=Urban Food Studio |accessdate=2014-04-01 |date=2008 |format=[[PDF|PDF]] |publisher=[[柏克萊加州大學|U.C. Berkeley]] |author=Prof. R. Davids |archive-url=https://web.archive.org/web/20150923201436/http://www.ced.berkeley.edu/courses/sp08/arch201/UrbanFoodStudioProgramApril07.pdf |archive-date=2015-09-23 |dead-url=yes }}</ref>，大道兩側的住宅漸漸有新大樓包圍<ref name="GoHouse"></ref>，該市的[[中央商務區|中央商務區]]也開始了新一個階段的擴張，聖保羅都心以西、以南的區域也開始出現高密度的商業區<ref name="Berkeley"></ref>。過了1960年代至1970年代之後，聖保羅人大道沿路已有數棟高於30層樓以上的大廈<ref>{{Cite web |url=http://www.jardinslifestyle.com.br/detalhe_conteudo.asp?cod=53 |title=Avenida Paulista |publisher=Jardins life style |accessdate=2014-04-01 |deadurl=yes |archiveurl=https://web.archive.org/web/20140407062102/http://www.jardinslifestyle.com.br/detalhe_conteudo.asp?cod=53 |archivedate=2014-04-07 }}</ref>。

==周邊環境==
聖保羅人大道及周圍街區除了公司行號、企業總部、金融和銀行辦公室外，還分布著數家購物商場<ref name="中國拉丁美洲研究網"></ref>、[[醫院|醫院]]、{{Link-en|駐巴西外交機構列表|List of diplomatic missions in Brazil|外國使領館}}、學校等設施，使得一天之內有數以萬計的人潮進出此地<ref name="brazilmycountry"></ref>。

居住在聖保羅人大道周邊樓房裡的居民共有20萬人，若是視之為一座獨立城鎮，將可列入巴西前150大的市鎮當中，其數字規模最接近於[[大普拉亞|大普拉亞]]的24萬人<ref>{{Cite web |url=http://sptv.globo.com/Sptv/0,19125,LPO0-6146-20080723-326152,00.html |publisher=SPTV |title=Avenida Paulista tem gerente para resolver problemas |date=2008-07-23 |accessdate=2009-01-10}}</ref>。
