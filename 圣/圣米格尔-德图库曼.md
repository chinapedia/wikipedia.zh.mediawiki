'''聖米格爾-德-圖庫曼'''（{{lang-es|San Miguel de Tucumán}}）是[[阿根廷|阿根廷]][[城市|城市]]，為該國最小[[阿根廷省份|省份]]——[[图库曼省|图库曼省]]之[[首府|首府]]。

== 名称由来 ==
关于图库曼这个名称的来源有争议。说这个名字来自于[[克丘亞語|克丘亞語]]中{{lang|qu|Yucuman}}（“河流发源的地方”）的理论被证明是错误的。另一个理论是这个名字来自更早的时期，在[[印加帝國|印加帝國]]入侵之前这里的居民是[[迪亚基塔人|迪亚基塔人]]，而图库曼这个名称来自于他们的酋长的称呼{{lang|qu|Tukma}}。第三个理论是当地的印第安人将这个地方以一种被称为{{lang|qu|Tuku-Tuku}}的甲虫来命名。{{lang|qu|Tuku-tuku-manita}}是“{{lang|qu|Tuku-tuku}}甲虫多的地方”的意思。

「聖米格爾」乃[[西班牙語|西班牙語]]中[[天使長|天使長]][[聖米迦勒|聖米迦勒]]的名字。

== 历史 ==
[[File:Tucumán-Casa-Gobierno-1.jpg|thumb]]
在[[前哥倫布時期|前哥倫布時期]]今天图库曼省地区的居民受印加文化的影响很深。当地的迪亚基塔人主要居住在图库曼东部的山区里。

1564年图库曼省被设立，省内的第一座城市是11年前建立的[[圣地亚哥-德尔埃斯特罗|圣地亚哥-德尔埃斯特罗]]。
[[File:Cathedral_tucuman2.JPG|thumb]]

1565年5月31日圣米格尔-德图库曼建城。由于当地的印第安人的反抗，以及原来城市的水质差，1685年9月27日图库曼省的省长将城市迁走并重新建立。1776年圣米格尔-德图库曼被划入[[拉普拉塔总督区|拉普拉塔总督区]]。

1810年5月25日在布宜諾斯艾利斯爆发了市民反抗总督的起义。1812年[[曼努尔·贝尔格拉诺|曼努尔·贝尔格拉诺]]来到图库曼。同年9月24日他在图库曼战役中击败了保王军。1816年7月9日在图库曼会议上，在今天被称为“独立大厦”（{{lang|es|Casa Independencia}}）的建筑物内阿根廷宣布从西班牙独立。

1969年阿根廷的游击战愈盛，在图库曼发生多起谋杀事件。

1970年当地省长试图调停各政治派别和工会的争端的努力失败，此后[[阿根廷人民革命军|阿根廷人民革命军]]等游击队开始其武装暴力行动。

== 居民 ==
图库曼省55%的居民住在圣米格尔及其附近地区，约90%的居民住在一个约80千米长的、南北走向的低地。
[[File:LaMerced_Tucuman.JPG|thumb]]

== 经济 ==
在圣米格尔-德图库曼附近集中着图库曼省的制糖工业。最重要的外国企业是圣米格尔-德图库曼附近的[[斯堪尼亚汽车|斯堪尼亚汽车]]的工厂。

== 教育 ==
圣米格尔-德图库曼有两座国立大学（1914年成立的图库曼国家大学和国家技术大学-图库曼地区系），此外市内还有一座拥有跨地区影响的天主教大学。

== 交通 ==
=== 铁路 ===
[[File:Argentine_train_-_Ferrocentral_-_Buenos_Aires_-_Tucuman.jpg|thumb]]
过去图库曼是阿根廷重要的铁路枢纽，今天只有去布宜諾斯艾利斯的客运服务了。

=== 市内交通 ===
市内交通以及与周边村镇的交通均是通过公共汽车完成的。

=== 远距交通 ===
[[圣米格尔-德图库曼机场|圣米格尔-德图库曼机场]]位于城市的东部。从这里有班机去往布宜諾斯艾利斯以及[[玻利维亚|玻利维亚]]的[[圣克鲁斯|圣克鲁斯]]。

长途汽车站位于市中心以东2千米处。这里是所有长途汽车的出发点。

=== 公路 ===
圣米格尔-德图库曼周围有一条环城高速公路。圣米格尔-德图库曼与阿根廷西北、东北以及[[潘帕|潘帕]]地区许多城市有国家公路连接。其中最重要的是9号公路，该公路从布宜諾斯艾利斯通过[[罗萨里奥|罗萨里奥]]、[[科尔多瓦_(阿根廷)|科尔多瓦]]、圣地亚哥-德尔埃斯特罗、圣米格尔-德图库曼、[[萨尔塔|萨尔塔]]、[[圣萨尔瓦多-德胡胡伊|圣萨尔瓦多-德胡胡伊]]直到玻利维亚边境城市[[拉奇阿卡|拉奇阿卡]]。
