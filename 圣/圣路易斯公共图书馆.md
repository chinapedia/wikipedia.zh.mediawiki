'''圣路易斯公共图书馆'''（St. Louis Public Library）是美国密苏里州[[圣路易斯|圣路易斯]]的市立[[公共图书馆|公共图书馆]]系统，包括中央图书馆和16个分馆。

圣路易斯公共图书馆由Ira Divoll创立于1865年<ref>{{cite web |url=http://central.slpl.org/before_central.html |title=The Library Before Central |author= |date= |work=St. Louis Public Library |accessdate=2016-04-23}}</ref>。1901年，圣路易斯公共图书馆获得安德鲁·卡内基的巨额捐赠，得以兴建目前的中央图书馆。馆藏书籍也由1865年的1500册增长到90,000册。到2014年，馆藏书籍达到460万册<ref>{{cite web |url=http://www.slpl.org/slpl/library/article240165754.asp |title=History and mission |author= |date= |work=St. Louis Public Library |accessdate=2011-09-27 |deadurl=yes |archiveurl=https://web.archive.org/web/20111001232852/http://www.slpl.org/slpl/library/article240165754.asp |archivedate=2011-10-01 }}</ref>。
