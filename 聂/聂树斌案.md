据新华社事后查证，'''聂树斌案'''是指1994年8月在[[河北省|河北省]][[石家庄市|石家庄市]]西郊发生的一起强奸杀人案。此案定罪于1995年“[[中华人民共和国严打行动|严打]]”期间，是[[中华人民共和国|中华人民共和国]]历史上最著名的冤案之一，被认为是中国“严打”这一特殊时期司法不公正的代表性案件<ref>{{cite web|url=http://cn.nytimes.com/china/20161205/china-court-exonerates-innocent-executed|title=聂树斌案背后的中国司法困境|publisher=纽约时报中文网|accessdate=2016-12-05}}</ref>。

==聂树斌之死==
1994年8月8日早上，石家庄液压件厂一名年龄34-35岁左右的女工[[康菊花|康菊花]]被报失踪，次日中午，在工厂后方的玉米地里发现了已高度腐烂的受害者尸体，旁边还有一辆自行车。死者生前曾遭强奸。

聂树斌为[[石家庄市|石家庄市]][[鹿泉区|鹿泉区]]综合职业技术学校[[校办工厂|校办工厂]]工人，1994年9月23日，聂树斌被案件侦查单位石家庄市公安局桥西分局传唤，9月24日被监视居住，10月1日被石家庄市公安局桥西分局作为案件[[犯罪嫌疑人|犯罪嫌疑人]][[刑事拘留|刑事拘留]]，10月9号被正式[[逮捕|逮捕]]。<ref name="sm">{{cite web|url=http://news.xinhuanet.com/legal/2014-12/19/c_1113701899.htm|title=聂树斌案从“铁案”到疑案的始末|accessdate=2015-04-28|publisher=新华网|archive-url=https://web.archive.org/web/20141225173211/http://news.xinhuanet.com/legal/2014-12/19/c_1113701899.htm|archive-date=2014-12-25|deadurl=yes}}</ref>

石家庄市人民检察院于1995年3月3日以聂树斌犯[[故意杀人罪|故意杀人罪]]、[[强奸罪|强奸罪]]，向石家庄市中级人民法院提起[[公诉|公诉]]。石家庄市中级人民法院不公开开庭审理了此案，并于1995年3月15日作出（1995）石刑初字第53号死刑判决。聂树斌不服，向河北省高级人民法院提出上诉。旋月余，河北省高级人民法院于1995年4月25日作出（1995）冀刑一终字第129号死刑判决<ref>1995年4月25日河北省高级人民法院（1995）冀刑一终字第129号《刑事附带民事判决书》</ref>。

根据案卷记载，未满21岁的聂树斌于1995年4月27日被执行[[枪决|枪决]]。翌日，即4月28日，聂树斌父亲去[[看守所|看守所]]给孩子送点吃的、穿的，被狱警告知，聂树斌已于头一天被执行了死刑。若以1995年4月27日死刑计算，聂树斌时年20岁零5个月。聂树斌的母亲在接受采访时回忆，聂树斌胆小内向，天生就有口吃，从来不敢跟人吵嘴和打架，被抓前一年让他杀一只家中放养的瘸腿的母鸡都不敢<ref>{{cite web|url=http://news.163.com/16/1210/18/C7UQ1DSU0001875N.html|title=22年！央视首次披露“聂树斌案”诸多细节|accessdate=2016-12-10|publisher=网易}}</ref>。

== 真凶上诉直聂树斌之冤 ==
2005年1月18日，涉多起奸杀命案的[[王书金|王书金]]在河南[[荥阳|荥阳]]被捕。当时，[[荥阳|荥阳]]索河路派出所的民警发现在当地砖厂干活、绰号“大王”的[[王书金|王书金]]行为反常，看见警察和警车就躲，多年来即使春节也不回家，沉默寡言，从不跟人提起他的真实姓名和河北老家的情况。1月17日，派出所民警到其居住的工棚查验身份，发现其没有身份证也没有暂住证，于是将其带到派出所进一步调查。1月18日凌晨，[[王书金|王书金]]供认了1993-1995年间犯下的四起强奸杀人案和两起强奸案。1月19日，[[王书金|王书金]]被移交给[[广平县|广平县]]警方。邯郸警方针对[[王书金|王书金]]交代的这六起案件中的其中一起，即1994年8月5日在石家庄西郊强奸杀害一名妇女的案件到石家庄调查的时候，却得知这起案件早已告偵破，当地警方在案发一个月后就抓到了“凶手”，案发八个月后就判处并执行了死刑。河南媒体以《河北“摧花狂魔”荥阳落网》进行了报道。<ref>{{cite web|url=http://www.infzm.com/content/91762|title=“聂树斌冤杀案”悬而未决，防“勾兑”公众吁异地调查（2005年3月24日）|accessdate=2015-04-28|publisher=南方周末|archive-url=https://web.archive.org/web/20140724191736/http://www.infzm.com/content/91762|archive-date=2014-07-24}}</ref>

鉴于案件前后事实矛盾，[[河北省高级人民法院|河北省高级人民法院]]于3月表态对聂树斌案进行复查。2007年3月12日，[[邯郸市中级人民法院|邯郸市中级人民法院]]一审判决王书金死刑，否认聂树斌案是其所为，因此不认定王书金有重大立功表现。2007年4月，王书金以未起诉他在石家庄西郊玉米地的奸杀案为理由之一，向[[河北省高级人民法院|河北省高级人民法院]]提出上诉。2007年7月，河北省高级人民法院二审不公开开庭审理了[[王书金案|王书金案]]。王书金继续对石家庄玉米地案供认不讳。然而2013年9月，河北省高级人民法院依然判决裁定王书金非聂树斌案真凶<ref>{{cite web|url=http://news.xinhuanet.com/legal/2013-09/28/c_125453494.htm|title=王书金案二审维持原判 将报最高人民法院核准死刑（2013年09月28日）|publisher=新华网|archive-url=https://web.archive.org/web/20131004104713/http://news.xinhuanet.com/legal/2013-09/28/c_125453494.htm|archive-date=2013-10-04|accessdate=2016年12月5日|deadurl=yes}}</ref>。

== 最高法院介入与案件异地复查 ==

=== 重審阻力 ===
自2005年3月11日(因應前不久，河北王书金自供-為聂树斌案“凶手”)起，至2016年12月2日止，最高人民法院第二巡回法庭宣告撤销原判，改判无罪。此申訴案已拖延近11年，主要為河北公檢法，特別是[[河北省高級人民法院|河北高院]]，一直推延此案進行案卷復查申請。

有消息稱此案最終可繼續進行案卷復查至可重新翻案，得益於前河北省常委、政法委[[書記|書記]][[張越|張越]]的撤職，而其遲遲不配合重審是受前國安部長[[許永躍|許永躍]]的默許而行<ref>[https://www.hk01.com/%E4%B8%AD%E5%9C%8B/57600/%E8%81%B6%E6%A8%B9%E6%96%8C%E5%A7%A6%E6%AE%BA%E5%86%A4%E6%A1%88%E6%AD%BB%E5%BE%8C21%E5%B9%B4%E7%B5%82%E7%84%A1%E7%BD%AA%E6%98%AD%E9%9B%AA-%E7%9C%81%E9%A0%98%E5%B0%8E%E6%9B%BE%E6%89%B9%E7%A4%BA-%E5%BF%AB%E6%AE%BA 聶樹斌姦殺冤案死後21年終無罪昭雪　省領導曾批示：「快殺」]</ref>。

=== 冤案复查 ===
至2014年12月12日，[[最高人民法院|最高人民法院]]指令[[山东省高级人民法院|山东省高级人民法院]]复查聂树斌强奸杀人案<ref name="sm" />。不久有说法在网上流传，称聂树斌被枪决，是因为当时为了给身患[[尿毒症|尿毒症]]的[[章含之|章含之]]寻找可移植的肾源，且聂树斌的肾脏可以匹配<ref>{{cite news|url=http://hk.apple.nextmedia.com/international/art/20141217/18971956|title=河北青年姦殺案翻案或有內情 中共名媛換腎　疑來自含冤死囚|date=2014-12-17|publisher=[[新浪微博|新浪微博]]/《[[蘋果日報_(香港)|蘋果日報]]》|accessdate=2014-12-17|language=zh-hant}}</ref>。章含之的女儿[[洪晃|洪晃]]否认此说法，理由是聂树斌的死亡日期与章含之的移植日期相差数月<ref>[http://news.qq.com/a/20141231/045780.htm 洪晃：母亲肾移植与聂树斌冤案无关]</ref>。然而由于聂树斌的死亡日期本身存在严重的疑问，仍然有观点认为两者很可能相关<ref name="sj">{{cite web|url=http://www.china50plus.com/聂树斌死亡时间与章含之换肾时间考|title=聂树斌死亡时间与章含之换肾时间考|accessdate=2016-12-15}}</ref>。2015年3月16日，山东省高级人民法院通知聂树斌案两名申诉代理律师[[李树亭|李树亭]]、[[陈光武|陈光武]]阅卷。

==== 侦查笔录遗失与卷宗造假 ====
聂树斌案申诉代理律师[[李树亭|李树亭]]、[[陈光武|陈光武]]阅卷发现聂案一二审卷宗中至少6处签字涉嫌造假<ref>[http://news.ifeng.com/a/20150318/43366748_0.shtml 聂树斌案律师：案卷存严重作伪痕迹]</ref>。聂树斌卷宗中前5天询问笔录缺失，而9月28日的询问笔录中却突然出现聂树斌认罪的叙述；此外，虽然原办案人员在案发后拿走了聂树斌所在车间的案发当月考勤表，却也未放入卷宗，且未能做出合理解释，导致聂树斌是否有作案时间缺少原始书证。

==== 辩护律师资质 ====
据[[新华社|新华社]]查证，二审（即终审）裁决中，法院指派某司法局的工作人员张景和为聂树斌辩护律师。从严格的意义上说，张景和没有律师资格，这一点，后来张景和也承认。聂家人见到了张表达了对案件的疑问。张景和对着聂家人愤怒地说：“你们怎么能信记者的话？记者的话能信吗？”聂母回问：“那我们该信谁的？张景和大声地说：“你们应该信政府。”<ref name="sm" />

==== 死刑时间与家属知情权 ====
聂树斌案申诉代理律师[[李树亭|李树亭]]在2015年3月21日透露，之前他在案卷中发现一份聂树斌亲笔书写的“刑事上诉状”，落款时间为“一九九五年五月十三日”——这一时间是在1995年4月27日聂树斌被执行死刑16天后。[[李树亭|李树亭]]据此认为，聂树斌被执行死刑的时间存在疑问。另有一种可能是聂树斌自己写错了日期，不过，[[李树亭|李树亭]]认为，根据聂树斌在“上诉状”中“所表现出的求生欲望，是不可能将这么重要材料上的日期写错的”。<ref>{{cite web|url=http://news.163.com/15/0322/07/AL9VELDP0001124J.html|title=律师发现聂树斌被处死16天后亲笔所写上诉状|accessdate=2015-03-22|publisher=网易}}</ref>

文件显示，聂树斌于1995年4月27日被执行死刑，然而这个日期因为存在诸多疑点而被广泛质疑，例如落款为“五月十三日”的“刑事上诉状”，以及枪决现场照片显示季节似乎为冬季等<ref name="sj" />。聂树斌从被判处死刑直至被枪决，他的家人从未收到过一审和二审判决书，其父是在到看守所为聂树斌送生活用品时才知道他已在前一天被执行了<ref>[http://news.sohu.com/20050402/n224988644.shtml 枪决10年陡起疑团]</ref>。

==== 真凶自称被逼翻供 ====
据媒体报道，王案二审二次开庭前，关押在[[河北|河北]][[磁县|磁县]]看守所的王书金对其辩护律师[[彭思源|彭思源]]透露：河北方面曾派工作组进驻看守所“做工作”，逼他翻供，要他否认是石家庄西郊玉米地强奸杀人案的凶手。[[王书金|王书金]]在山东省高院的法官提讯时说：“虽然我现在脑袋不清醒，但当初回答警察的提问是清醒的！原来讲的是真话，只是在工作组面前说了假话。”[[彭思源|彭思源]]对此的解释是：“因为工作组的人打了他，他忍受不了才说了假话。因此他向山东高院法官声明：他在工作组面前讲的一切都作废！”<ref>{{cite web|url=http://news.163.com/15/0316/02/AKQ10TIC00014AED.html|title=聂树斌案嫌犯:河北工作组打我 逼我否认强奸杀人|accessdate=2015-03-16|publisher=网易}}</ref>

== 最高法院再审 ==
2015年4月28日，山东省高级人民法院召开听证会，听取申述人及其代理律师、原办案单位代表意见。该听证会邀请听证人员但不接受其他人旁听。<ref>{{cite web|url=http://news.xinhuanet.com/2015-04/29/c_1115122526.htm|title=山东省高院召开聂树斌案听证会|publisher=新华网|date=2015-04-29}}</ref>同日傍晚，山东高院发布了听证会摘要。在先后四次向最高人民法院申请延长复查期限之后，山东省高级人民法院最终认定河北两级法院的判决「缺少能够锁定聂树斌作案的客观证据，在被告人作案时间、作案工具、被害人死因等方面存在重大疑问，不能排除他人作案的可能性，原审认定聂树斌犯故意杀人罪、强奸妇女罪的证据不确实、不充分。」<ref>{{cite web |url = http://www.caixin.com/2016-06-08/100952656.html|title = 山东高院负责人就聂树斌案复查情况答记者问 |publisher = 财新 |accessdate = 2016-06-08}}</ref>2016年6月6日，最高人民法院同意山东高院的意见，决定提审此案。20日，决定由[[中华人民共和国最高人民法院第二巡回法庭|第二巡回法庭]]审理该案。7月4日，第二巡回法庭组成合议庭如下：<ref>{{cite web |url = http://news.sina.com.cn/c/nd/2016-12-02/doc-ifxyiayq2039388.shtml|title = 聂树斌案再审改判无罪 |publisher = 新华社 |accessdate = 2016-12-02}}</ref>

== 国家赔偿与后续追责 ==
聂树斌无罪平反后，国家赔偿及追责程序启动<ref>{{cite web|url=http://www.chinanews.com/gn/2016/12-02/8081475.shtml|title=最高人民法院再审改判聂树斌无罪|accessdate=2016-12-02|publisher=中国新闻网}}</ref>。2016年12月14日下午，聂树斌的母亲张焕枝在四川泽仁律师事务所律师辜光伟、北京京师（天津） 律师事务所律师王殿学的陪同下，向河北省高级人民法院申请国家赔偿，提出含1200万元精神损害抚慰金在内共计7项赔偿请求，总金额达1391余万元。河北省高级人民法院随即决定立案受理。<ref>{{cite web|url=http://finance.ifeng.com/a/20161214/15077475_0.shtml|title=聂树斌家属申请国家赔偿：索赔1千391万余元 河北高院受理|accessdate=2016-12-14|publisher=中国新闻网}}</ref>

河北省高级人民法院2017年3月30日发布[[国家赔偿|国家赔偿]]决定书，赔偿聂树斌家属人民币268万余元（聂树斌死亡赔偿金与丧葬费：126万元；聂树斌被侵犯自由217天赔偿费用：5万2千余元；聂树斌父母精神损害抚慰金：130万元；聂树斌母亲生活费：6万4千元），且指出在赔偿前已在新闻媒体上道歉。聂树斌母亲张焕枝对媒体表示將不再继续申诉。<ref>{{Cite web|url=http://www.bjnews.com.cn/inside/2017/03/31/438523.html|title=聂树斌案获赔268万 精神赔偿“创纪录” - 深度 - 新京报网|accessdate=2019-01-13|author=|date=|work=新京网|publisher=}}</ref>

尽管河北省高级人民法院在最高法院为聂树斌案平反当日，就在中国大陆社交媒体微博上回应最高人民法院称会及时跟进调查追责，然而至2018年底却惟无实质行动。2017年8月，聂树斌母亲张焕枝对媒体称不放弃追责，并于同月30日向最高人民检察院和最高人民法院分别写信，敦促启动聂树斌冤案追责程序。媒体指同为冤假错案的呼格吉勒图案至2016年1月已经追责27人。<ref>{{Cite web|url=http://m.sohu.com/a/286711247_260616|title=聂树斌案平反两年未追责|accessdate=2019-01-13|author=|date=|work=手机搜狐网|publisher=}}</ref>
