'''聂傲双'''{{BD|1995年|1月16日|||N}}，是一名中国[[足球|足球]]运动员，司职[[中场|中场]]。目前效力于[[中甲|中甲]]球队[[武汉卓尔|武汉卓尔]]。

==俱乐部生涯==
聂傲双出道于[[武汉卓尔|武汉卓尔]]青训系统。[[中华人民共和国第十二届运动会|2013年全运会]]乙组比赛代表湖北队出场，表现受到好评。2014年1月与队友[[明天_(足球运动员)|明天]]一起被选派至法国[[索肖足球俱乐部|索肖足球俱乐部]]进行为期三个月的学习训练。<ref>{{cite web |url=http://sports.sina.com.cn/j/2013-11-28/13436907876.shtml |title=卓尔牵手法甲索肖俱乐部 两希望之星将赴法“镀金” |author= |date=2013年11月28日 |publisher=新浪体育  |accessdate=2017年1月18日}}</ref>2014年进入武汉卓尔一线队名单但并未在联赛中出场。2015年首次分别在对[[武汉宏兴|武汉宏兴]]的[[足协杯|足协杯]]比赛和对[[江西联盛|江西联盛]]的[[中甲|中甲]]联赛中出场。2016年7月武汉卓尔俱乐部官方宣布与聂傲双签订了新的工作合同。<ref>{{cite web |url=http://sports.sina.com.cn/china/b/2016-07-01/doc-ifxtsatm1170225.shtml |title=卓尔官方宣布续约7将 称将尽全力助年轻球员发展 |author= |date=2016年7月1日 |publisher=新浪体育  |accessdate=2017年1月19日}}</ref>

2018年4月4日中甲联赛第四轮武汉卓尔主场对阵[[辽宁宏运|辽宁宏运]]的比赛，聂傲双首发出场，并在下半场21分钟时禁区内接[[让·埃夫拉·夸西|-{zh-hans:埃弗拉;zh-hk:高亞斯}-]]的传球兜射远角将球射入，打入个人首例联赛入球。<ref>{{cite web|title=中甲-埃弗拉传射拉斐尔建功 卓尔多点开花4-0辽足|url=http://sports.sina.com.cn/china/b/2018-04-04/doc-ifysuuya6243978.shtml|website=新浪体育|accessdate=2018-04-05}}</ref>
