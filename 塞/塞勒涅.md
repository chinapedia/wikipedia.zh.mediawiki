'''塞勒涅'''（{{lang-el|Σελήνη}}，月亮）[[希腊神话|希腊神话]]中的[[月亮女神|月亮女神]]。在[[罗马神话|罗马神话]]中，她的对应者是'''[[卢娜|卢娜]]'''。

== 简要介绍 ==
按照[[奥林波斯教|奥林波斯教]]的谱系，塞勒涅是一个[[女提坦|女提坦]]。关于她的出身有多种说法。[[赫西俄德|赫西俄德]]说塞勒涅和[[太阳神|太阳神]][[赫利俄斯|赫利俄斯]]是雙胞胎，都是第一代提坦神[[许珀里翁|许珀里翁]]和[[忒亚|忒亚]]的孩子<ref>赫西俄德，《[[神谱|神谱]]》</ref>。[[欧里庇得斯|欧里庇得斯]]则认为塞勒涅是赫利俄斯的女儿<ref>Εὐριπίδης,"Φοίνισσαι",175-176ff ',ὦ λιπαροζώνου θύγατερ Ἀελίου  Σελαναία'</ref>。还有些神话记载说塞勒涅与赫利俄斯是夫妻关系。由于后来被与[[阿耳忒弥斯|阿耳忒弥斯]]相混的结果，一些古典作家开始称塞勒涅为[[宙斯|宙斯]]的女儿。此外，还有说法认为她是提坦神[[帕拉斯|帕拉斯]]的女儿。<ref>[[荷马|荷马]]，《赫耳墨斯颂歌》</ref>

塞勒涅在神话中的传统形象是：长有双[[翅膀|翼]]，头顶金色光环，乘坐一辆由两匹神[[马|马]]（一说是[[牛|牛]]，取以牛角象征[[新月|新月]]之意）拉动的马车在夜空中飞驰，当[[白昼|白昼]]来临时消失于[[大洋河|大洋河]]中。[[文艺复兴|文艺复兴]]以后的艺术作品也常把她表现成皮肤白皙、驾驶着马车（或骑马）的女神，有时手持[[火炬|火炬]]（古希腊女诗人[[萨福|萨福]]在其作品中就提到过这种形象）。

在古典神话中，塞勒涅与其兄弟赫利俄斯每天轮流驾车从天空经过，形成昼夜交替；而在这之间，是他们的姐姐、[[黎明|黎明]]女神[[厄俄斯|厄俄斯]]主导的时期。

== 与塞勒涅有关的神话 ==
[[File:Pergamonmuseum_-_Antikensammlung_-_Pergamonaltar_31.JPG|300px]]
塞勒涅与[[恩底弥翁|恩底弥翁]]的爱情故事在希腊神话里非常有名。恩底弥翁是一个受塞勒涅爱慕的美少年，被众神赐予了永恒的[[青春|青春]]，但代价是在[[卡里亚|卡里亚]]的[[拉特穆斯山|拉特穆斯山]]的山洞里永远长眠。[[厄勒亚|厄勒亚]]（[[古希腊|古希腊]]一地区，大致相当于今[[伊利亚州|伊利亚州]]）地区的神话说，塞勒涅给恩底弥翁生育了50个女儿，这是对古希腊[[历法|历法]]中一年有50个星期的反映。

[[阿波罗尼乌斯|阿波罗尼乌斯]]较为详细地记载了这个神话。恩底弥翁是一个凡人猎手或牧羊人（按照[[保萨尼阿斯|保萨尼阿斯]]的说法，恩底弥翁则是[[厄勒亚|厄勒亚]]的国王）。他的外貌俊美无比，被这种惊人的美貌迷住的塞勒涅请求宙斯让恩底弥翁永葆青春。但塞勒涅从厄俄斯处得知，请求神赐予凡人永生是不可能的，于是这个请求变成了让恩底弥翁永远沉睡，这样他就既不会死去也不会离开她。<ref>阿波罗尼乌斯，《阿耳戈英雄记》，IV</ref>[[西塞罗|西塞罗]]也说在这个事件中是塞勒涅主动提出了让恩底弥翁沉睡的请求。但在其他版本中，长眠不醒是恩底弥翁自己做出的决定。恩底弥翁所在的拉特穆斯山离[[米利都|米利都]]很近。每天夜里塞勒涅从天空经过时都会面带愁容地探望沉睡中的恩底弥翁。<ref>保萨尼阿斯，Vol 1.5</ref>塞勒涅与恩底弥翁的女儿们被称为[[墨娜厄|墨娜厄]]（[[墨娜|墨娜]]是塞勒涅的一个别名）。

在神话中，塞勒涅与牧神[[潘_(希腊神话)|潘]]也有过一段风流韵事。潘为了向塞勒涅献殷勤，送给她一群小[[羊|羊]]；<ref>[[维吉尔|维吉尔]]，[[农事诗|农事诗]]，III</ref>也有说法认为潘把自己装扮成小羊，企图诱奸塞勒涅。<ref>[[凯雷尼·卡罗伊|凯雷尼·卡罗伊]]，《希腊人的众神》</ref>塞勒涅的牛车（或马车）上的[[轭|轭]]也是潘送的。

此外，塞勒涅还曾与宙斯结合，并生育了女儿[[潘狄亚|潘狄亚]]。<ref>[[荷马颂歌|荷马颂歌]]，XXXII</ref>某些资料认为塞勒涅也是[[涅墨亚狮子|涅墨亚狮子]]的母亲（而一般认为其母为[[厄客德娜|厄客德娜]]）。

== 現代学者猜想的来源和性质 ==
现代[[神话学|神话学者]]一般认为，塞勒涅是一个从[[东方|东方]]传入[[希腊|希腊]]的外来神。希腊人本身并不崇拜[[月球|月球]]（有研究者认为[[太阳|太阳]]崇拜对于希腊人来说也是舶来品，或是[[米诺斯文明|前希腊时期]]的遗留物）。她的名字难以用希腊语来解释。如果一定要与希腊语建立联系的话，那么“塞勒涅”一词可能与希腊语中表达“[[光|光]]”的词汇{{lang|el|Σελας}}有关。<ref>凯雷尼·卡罗伊，《希腊人的众神》</ref>另一方面，塞勒涅的历史非常悠久。最早提到塞勒涅的文献是[[荷马|荷马]]的《塞勒涅颂歌》，那里面描述这位女神为“展翅高飞的塞勒涅”。<ref>荷马颂歌，XXXII</ref>从很早的时候起塞勒涅就在神话里占有重要地位，这说明她可能在前希腊时期就已经被引入了（但尚无确切依据证明[[克里特文明|克里特文明]]曾有过对月神的崇拜）。

塞勒涅是一个古老的神祇；在[[古典时期|古典时期]]，她作为月神的大部分职能都被[[阿耳忒弥斯|阿耳忒弥斯]]取代（部分原因是希腊人长期视月球崇拜为野蛮人的习俗，于是用狩猎女神阿耳忒弥斯兼领其职务，尽管后者也是从东方引入或深受东方影响的神祇<ref>《神话辞典》，11页，266页</ref>）。在这一时期，塞勒涅的形象经常被与其他女神，如阿耳忒弥斯、[[赫卡忒|赫卡忒]]、[[伊俄|伊俄]]甚至[[赫拉|赫拉]]相混同。此外，现代学者一般认为[[克里特岛|克里特岛]]的[[帕西法厄|帕西法厄]]（牛头怪[[弥诺陶罗斯|弥诺陶罗斯]]之母）是将塞勒涅凡人化的产物。

塞勒涅在古希腊的[[巫术|巫术]]迷信中占有重要地位。古代人相信，月亮的圆缺会对人类的生活造成影响：[[满月|满月]]使所有事物蓬勃壮大，而亏月则阻碍一切事物发展，所以亏月有利于治病或驱邪。[[色萨利|色萨利]]地区的居民还相信，女巫会在[[月食|月食]]时施行法术。直到[[希腊化时代|希腊化时代]]，人们仍然认为塞勒涅是[[灵魂|灵魂]]的载体。

在古希腊，纪念塞勒涅的节日是[[春分|春分]]日。

== 其他 ==
[[日本|日本]]于2007年发射的[[月亮女神_(航天器)|月亮女神号]][[月球|月球]]探测器的[[英文|英文]]名称（Selenological and Engineering Explorer）的[[缩写|缩写]]是“塞勒涅”（SELENE）。

[[化学|化学]][[元素|元素]][[硒|硒]]（selenium）的名字来自塞勒涅。
