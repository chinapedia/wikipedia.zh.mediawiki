'''塞爾維亞21歲以下國家足球隊'''（{{lang-sr|Mlada reprezentacija Srbije}}，簡稱'''塞爾維亞U21'''）代表[[塞爾維亞|塞爾維亞]]參賽所屬年齡組別的[[英式足球|足球]]競賽，主要為兩年一度的[[歐洲U-21足球錦標賽|歐洲U-21足球錦標賽]]，由[[塞尔维亚足球协会|塞尔维亚足球协会]]負責監控。

塞爾維亞U21被視為「塞爾維亞和黑山U21」的後繼球隊，而後者則承接前[[南斯拉夫21歲以下國家足球隊|南斯拉夫21歲以下國家足球隊]]的地位。

== 歷史 ==
在前[[南斯拉夫社会主义联邦共和国|南斯拉夫]]於1992年解體前由[[南斯拉夫21歲以下國家足球隊|南斯拉夫U21]]代表出賽，曾贏得首屆[[歐洲U-21足球錦標賽|歐洲U-21足球錦標賽]]的冠軍；分裂後的1996年歐洲U-21足球錦標賽由代表[[南斯拉夫联盟共和国|南斯拉夫联盟共和国]]（Federal Republic of Yugoslavia，簡稱「南聯盟」）的U21球隊參賽，「南聯盟」其後於2003年更名為[[塞爾維亞與蒙特內哥羅|塞爾維亞和黑山]]（Serbia and Montenegro），球隊名稱亦隨之更改。

2006年[[蒙特內哥羅|黑山]]脫離塞爾維亞和黑山[[獨立|獨立]]，原「塞爾維亞和黑山足球總會」改組成為「[[塞爾維亞足球總會|塞爾維亞足球總會]]」，而其轄下的「塞爾維亞U21」取代「塞爾維亞和黑山U21」的席位參加[[2007年歐洲U-21足球錦標賽|2007年歐洲U-21足球錦標賽]]外圍賽，最後更獲得亞軍的榮譽；直到[[蒙特內哥羅足球協會|黑山足球協會]]成立並獲得正式承認時，外圍賽經已展開，因此「[[黑山21歲以下國家足球隊|黑山U21]]」未能趕及參賽，要到[[2009年歐洲U-21足球錦標賽|2009年]]的外圍賽可進行正式競爭性的比賽。

== 競賽歷史 ==
;南斯拉夫聯盟共和國/塞爾維亞和黑山
[[塞爾維亞國家足球隊|南聯盟大國腳]]早於1994年12月已進行首場比賽，但U21隊因[[政治|政治]]問題而被禁止參加1996年[[歐洲U-21足球錦標賽|歐洲U-21足球錦標賽]]，要等到1998年外圍賽才踢首場比賽，主場1-0擊敗[[馬爾他21歲以下國家足球隊|馬爾他U21]]。
 
南聯盟U21未能晉級其參加的首三屆歐洲U-21足球錦標賽決賽週，最接近是2000年，分組賽8戰5勝2和1負以較佳成績的次名身份出線次輪附加賽，在中立場[[巴塞隆拿|巴塞隆拿]]對[[英格蘭21歲以下足球代表隊|英格蘭]]，以0-3落敗而在外圍賽止步。

}}</ref></small>，對手為[[挪威21歲以下國家足球隊|挪威]]，首回合主場大勝5-1<small><ref>{{cite web |language=en |url=http://www.uefa.com/competitions/under21/history/season=2004/round=1586/match=73597/index.html |url=http://www.uefa.com/competitions/under21/history/season=2004/round=1586/match=73598/index.html |title=Norway go down fighting |publisher=[[UEFA|''uefa.com'']] |accessdate=2009-09-17 }}</ref></small>。這支由[[弗拉基米尔·彼得洛维奇|福拉多]]（Vladimir Petrović）領導的球隊陣中有[[布拉尼斯拉夫·伊雲奴域|伊雲奴域]]（Branislav Ivanović）及其後轉投黑山的[[西蒙·武克切維奇|武克切維奇]]（Simon Vukčević），分組賽首場即以3-2擊敗[[克羅地亞21歲以下國家足球隊|克羅地亞]]<small><ref>{{cite web |language=en |url=http://www.uefa.com/competitions/under21/history/season=2004/round=1608/match=74208/index.html |title=Ivanovic culls Croatian comeback |publisher=[[UEFA|''uefa.com'']] |accessdate=2009-09-17 }}</ref></small>，最終與義大利同為3戰2勝1負得6分，只因對賽時以1-2見負而以次名出線四強淘汰賽<small><ref>{{cite web |language=en |url=http://www.uefa.com/competitions/under21/history/season=2004/round=1608/match=74209/index.html |title=Sculli keeps Azzurrini alive |publisher=[[UEFA|''uefa.com'']] |accessdate=2009-09-17 }}</ref></small>。準決賽對[[瑞典21歲以下國家足球隊|瑞典]]，上半場已落後0-1，直到完場補時才扳平1-1，[[加時賽|加時]]雙各有一次中柱中楣後再無紀錄，於[[互射十二碼|互射十二碼]]以6-5僅勝<small><ref>{{cite web |language=en }}</ref></small>；決賽對手是四屆冠軍義大利，是雙方本屆賽事繼外圍賽及決賽週分組賽同組對賽後第四次碰頭，之前3戰1勝2負稍佔下風，更不幸是[[尼克拉·米亚洛维奇|米查洛域]]（Nikola Mijailović）在上半場僅34分鐘落後一球時已因兩[[黃牌|黃]]一[[紅牌|紅]]被逐離場，大部分時間10人應戰下，到下半場83至85分鐘再連失兩球，而伊雲奴域亦在補時階段被逐，只餘9人直到終場以0-3落敗屈居亞軍<small><ref>{{cite web |language=en 兩年後伊雲奴域仍然在陣，加上[[杜什科·托希奇|托錫]]（Duško Tošić）、[[Nenad_Milijaš|米利查斯]]（Nenad Milijaš）、及其後轉隸黑山的[[米爾科·武齊尼奇|胡仙歷]]（Mirko Vučinić）再次參賽，外圍賽10戰7勝1和2負，壓倒兩屆冠軍[[西班牙21歲以下國家足球隊|西班牙]]緊隨[[比利時21歲以下國家足球隊|比利時]]以次名出線附加賽<small><ref>{{cite web |language=en |url=http://www.uefa.com/competitions/under21/history/season=2006/round=1910/group=1911.html |title=Season 2004-2006 Qualifying round Group 7 |publisher=[[UEFA|''uefa.com'']] |accessdate=2009-09-17 }}</ref></small>，巧遇[[克羅地亞21歲以下國家足球隊|克羅地亞]]，首回合主場3-1獲勝<small><ref>{{cite web |language=en |url=http://www.uefa.com/competitions/under21/history/season=2006/round=2155/match=82016/index.html |title=Vučinić curtails Croatian hopes |publisher=[[UEFA|''uefa.com'']] |accessdate=2009-09-17 }}</ref></small>，次回合作客再勝2-1，累計5-2連續兩年晉身決賽週<small><ref>{{cite web |language=en |url=http://www.uefa.com/competitions/under21/history/season=2006/round=2155/match=82017/index.html |title=Job done for Serbia and Montenegro |publisher=[[UEFA|''uefa.com'']] |accessdate=2009-09-17 }}</ref></small>。分組賽最後一戰以0-2負於[[法國21歲以下國家足球隊|法國]]，塞黑正處於被淘汰邊沿，幸好同時舉行另一場比賽，主辨國[[葡萄牙21歲以下國家足球隊|葡萄牙]]於完場補時4分鐘才射入，1-0擊敗[[德國21歲以下國家足球隊|德國]]，3隊同為1勝2負得3分，塞黑以較佳[[得失球差|得失球差]]取得次名出線資格。準決賽對[[烏克蘭21歲以下國家足球隊|烏克蘭]]，戰至[[加時賽|加時]]後仍然兩無紀錄，[[互射十二碼|互射十二碼]]雙方首球均告射失，其餘4球全部射入，進入[[突然死亡|突然死亡]]，先射的烏克蘭射入，但緊接的[[米兰·普罗维奇|米兰·普罗维奇]]（Milan Purović）射失，4-5不敵無緣再次晉身決賽<small><ref>{{cite web |language=en |url=http://www.uefa.com/competitions/under21/history/season=2006/round=1913/match=84132/index.html |title=Penalty drama goes Ukraine's way |publisher=[[UEFA|''uefa.com'']] |accessdate=2009-09-17 }}</ref></small>。
