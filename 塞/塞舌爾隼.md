'''塞舌爾隼'''（''Falco araea''），又名'''塞舌島隼'''，是[[塞舌爾|塞舌爾]]特有的一種[[猛禽|猛禽]]。

== 特徵 ==
塞舌爾隼是最細小的[[隼|隼]]，只長18-23厘米，翼展40-45厘米。牠們的翼較短及圓。雄隼的上身呈紅褐色，有黑色斑點，下身呈淡黃色，沒有斑點。[[頭部|頭部]]及[[臀部|臀部]]呈深藍灰色。尾巴是藍灰色的，有黑色斑紋。喙深色，腳及[[蠟膜|蠟膜]]都是黃色的。雌隼外表像雄隼，稍為大型及淡色。雛鳥是褐色的，頭部有斑紋，[[胸部|胸部]]有斑點，尾巴端呈淡黃色。

== 生態 ==
塞舌爾隼棲息在[[森林|森林]]、叢林及農地。牠們很少會在天上飛翔，會坐在一旁待獵物出現，俯衝而下捕捉獵物。牠們主要吃[[蜥蜴|蜥蜴]]，佔了其食性的92%，並且會吃細小的[[鳥類|鳥類]]、[[青蛙|青蛙]]、[[大家鼠|大家鼠]]及[[昆蟲|昆蟲]]。

塞舌爾隼的地盤只有40公頃大，是[[猛禽|猛禽]]中最細小的。牠們於8月至10月間[[繁殖|繁殖]]。牠們會在山崖、樹上或建築物中築巢。牠們築巢只是在掘坑，並不會用任何物料。每次會生2-3隻蛋，蛋呈白色，有褐色的斑紋，孵化期為28-31日。雛鳥出生後35-42日就會換羽，並與父母同住往後的14星期。

== 保育狀況 ==
塞舌爾隼的數量只有約800隻，故被列為[[易危|易危]]。<ref name=IUCN/>在低地築巢的失敗率達70-80%。牠們以往可能在[[塞舌爾|塞舌爾]]最大的島嶼上[[繁殖|繁殖]]，但現時已知只在[[馬埃島|馬埃島]]、[[錫盧埃特島|錫盧埃特島]]、[[北島_(塞舌爾)|北島]]、[[普拉蘭島|普拉蘭島]]及其他周邊小島繁殖。牠們於1977年被重新引入到普拉蘭島。

對塞舌爾隼的威脅包括有失去棲息地、被掠食及與[[入侵物種|入侵物種]]的競爭。[[大家鼠|大家鼠]]、[[貓|貓]]及[[倉鴞|倉鴞]]都令[[蜥蜴|蜥蜴]]的數量大幅減少，且會吃塞舌爾隼的蛋及雛隼。倉鴞及[[家八哥|家八哥]]也佔據了很多適合築巢的位點。

以往因誤會塞舌爾隼會獵食[[家禽|家禽]]及迷信牠們是[[死亡|死亡]]的先兆而遭[[人類|人類]]獵殺，但這種情況已經很少出現。
