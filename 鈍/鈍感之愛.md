《'''鈍感之愛'''》（{{lang-en|'''''Anesthesia'''''}}）是一部2015年美國[[獨立電影|獨立]][[劇情片|劇情電影]]，由[[提姆·布雷克·尼爾森|提姆·布雷克·尼爾森]]編劇、導演、演出及監製。提姆·布雷克·尼爾森與[[山姆·華特斯頓|森·禾達史東]]、[[格蓮·高絲|格蓮·高絲]]、[[姬絲汀·史超域|姬絲汀·史超域]]、[[葛雷琴·莫爾|葛雷琴·莫爾]]及[[寇瑞·斯托尔|哥里·史托]]演出。電影於2015年4月22日舉行的[[翠貝卡電影節|翠貝卡電影節]]上首映<ref name="Tribeca">{{cite web |url=http://variety.com/2015/film/news/tribeca-film-festival-2015-schwarzenegger-zombie-1201446961/|title=Tribeca Festival Adds Mary J. Blige, Cannibal Cop and a Schwarzenegger Zombie Movie |accessdate=2015-03-13|work=[[Variety_(magazine)|Variety]]}}</ref>，並於2016年1月8日[[有限上映|有限上映]]並透過IFC電影[[自選影像|自選影像]]上映<ref>{{cite web|url=http://www.ifcfilms.com/films/anesthesia/|title=Anesthesia|publisher=IFCFilms.com|accessdate=2015-11-11}}</ref>。

==劇情==
一個大學裡的哲學教授沃爾特·薩爾在一次搶劫時受傷，他為了逃脫，隨意地按響蜂鳴器，讓一個有孩子的中年父親阿森醒來，阿森勉強地回應沃爾特的請求，沃爾特在他懷裡失去意識。透過探索這些人與劫匪，以及癮君子阿喬之間的關係如何連結在一起，我們一起探索這個紐約市。沃爾特·薩爾、阿森、阿喬與行凶者經歷的波紋出現出來，當中包括一個[[酗酒|酗酒]]的主婦、一個不顧一切地失去了童貞的少年、一個才華橫溢但失敗的作家、一對面對末期疾病的父母，以及一個傑出的研究生蘇菲以[[自殘|自殘]]來讓自己覺得活著……

==評價==
根據[[Rotten_Tomatoes|爛番茄]]網站，《鈍感之愛》評分為26%，基於43個評論，平均分為4.9/10<ref>{{cite web |url=http://www.rottentomatoes.com/m/anesthesia/ |title=Anesthesia (2016) |work=Rotten Tomatoes |accessdate=2016-01-04}}</ref>。[[Metacritic|Metacritic]]給予58/100，從14個評論中抽樣取得，表示為「混合或平均評論」<ref>{{cite web |url=http://www.metacritic.com/movie/anesthesia |title=Anesthesia |work=Metacritic |accessdate=2016-01-04}}</ref>。

《鈍感之愛》於2015年[[翠貝卡電影節|翠貝卡電影節]]上首映，獲得普遍積極的評論。《The Wrap》的丹·卡拉漢（Dan Callahan）給這部電影一個積極的評論，並稱讚表現指出，導演兼編劇[[提姆·布雷克·尼爾森|提姆·布雷克·尼爾森]]避免了在一場充滿憤怒和活力的當代戲劇中的多愁善感，他還指透過電影的憤怒，賦予了它的味道、它的心情以及它的重力。這部電影需要非常認真對待，它贏得這項權利。這愁眉苦臉的絕望讓尼爾森的臉孔在屏幕上永遠存在，也充分證明了他在這裡的創作，以及他給[[姬絲汀·史超域|姬絲汀·史超域]]的方向，他加入的力量是非常充滿活力的<ref>http://www.thewrap.com/anesthesia-tribeca-review-kristen-stewart-glenn-close-tim-blake-nelson/</ref>。

《[[The_Hollywood_Reporter|荷李活報道]]》的約翰·德福（John DeFore）寫道：「任何觀眾進入電影，他們當中沒有一個會在影片的尾聲不想擁抱[[山姆·華特斯頓|森·禾達史東]]，演員完美地體現了一種學習的人文主義的味道，讓我們通過幾個更抽象的焦點討論社會的興衰。<ref>http://www.hollywoodreporter.com/review/anesthesia-tribeca-review-790972</ref>」
