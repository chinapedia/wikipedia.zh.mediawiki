'''腾越起义'''，是1911年10月27日繼[[武昌起義|武昌起義]]後，發生在[[雲南省|雲南省]][[腾越厅|腾越厅]]（今腾冲县），由[[同盟會|同盟會]]發起的反抗[[清|清]]政府武裝行動。

== 背景 ==
清光绪三十二年（1906年）三月，中国同盟会[[云南|云南]]分会骨干[[杨振鸿|杨振鸿]]（云南昆明人，日本振武学校毕业，后又进日本陆军测量学校学习）离开日本，取道[[越南|越南]]回国。六月云南总督[[丁振铎|丁振铎]]将杨调往腾越厅任西防营管带（营长）随他前往的同盟会员[[黄毓英|黄毓英]]（子和）、董鸿勛（千城）、张定臣、杜仲琦等人。

杨振鸿到腾越后，一面整训部属，学习日本的新式操典，同时发展同盟会组织。1907年，先后介绍了腾越人张文光（绍三）、刘辅国（弼臣）以及青年教师康占斗、周从锡、方百川、杨贡臣等人加人同盟会，先后发展了三百多人。

1908年十月，[[光绪皇帝|光绪皇帝]]和[[慈禧太后|慈禧太后]]相继死亡，杨振鸿认为是起义的大好机时，立即从[[仰光|仰光]]起身，经[[腊戍|腊戍]]、[[昔董|昔董]]、[[干崖|干崖]]、[[蛮允|蛮允]]到[[保山|保山]][[蒲缥|蒲缥]]，与何畏、吴品芳、何兴、杨毓铣、彭縸等人一起于12月23日发动了永昌起义。但由于起义队伍大多是农民，缺乏严密的组织和训练，起义失败。杨振鸿恶性疟疾发作在蒲缥病逝，张文光成为了后来腾越起义的主要领导人。

1909年张文光秘密成立了自治会，发展农民、会党群众数千人入会。以张鉴安、李治、刘辅国、马登云、黄安和、周维美、徐占标、周开勛、明增耀、明增广、杨贡臣等50多人为核心秘密召开会议，宣布革命宗旨，誓死为革命奋斗。他们在腾越、[[永昌|永昌]]、[[龙陵|龙陵]]等地宣传革命，同时争取驻边地的清军下级军官参加自治会。

宣统二年（1910年）黄毓英与同盟会革命党人吴品芳、马幼伯、杜韩甫等到腾，与张共商起义事宜，两人饮血为誓：“光（张文光）在腾举事，君（黄毓英）应我于滇垣（省会昆明），君在滇举事，我应君于腾越。”

1911年3月，黄兴组织二次广州起义，张文光与同仁商议准备起义响应，不料事机泄露，遭腾越官府通缉，只好到[[干崖土司|干崖土司]][[刀安仁|刀安仁]]家躲避，刀安仁在日本留学时认识孙中山并参加了同盟会，这次与张文光相识，一见如故，刀安仁讲述了孙中山在日本对在云南发动起义的鼓励和期待，两人一起详细讨论了起义的有关事项和日期。

== 過程 ==
宣统三年（1911年）10月10日武昌起义爆发，张文光回到腾越，议定10月27日（农历九月初六）举行起义响应，10月14日致电昆明黄毓英起义日期。24日前往干崖刀安仁处取印信和刀安仁从日本带回的同盟会革命方略，27日下午返腾，腾越所有同盟会、自治会干部都已集中。张文光宣读了誓词，分配任务，同时派人联络西防第四营、西防第五营的拥护起义的官兵，约定晚9时起义。规定“1，以‘独立’为夜间联络口令，2，用黑纱帕包在头上做标志，3，起义时由陈定洲负责通知英国领事馆、税务司、基督教堂，叫外国人闭门勿惊。

晚9时整，起义军分四路行动：一路到萧公祠营门协同起义官兵杀死管带张桐，一路至东营西防营配合起义官兵，劝说管带曹福祥归附被拒，起义军杀死曹福祥。另一路起义军至西防五营，清军管带谭正兴闻兵变越墙逃走，起义官兵转攻城西腾越厅署。张文光同时率部直攻腾越总兵公署。

经过三小时猛攻，清政府腾越各署、局相继被起义军占领，腾越道尹耿葆奎、腾越厅臣温良彝、盐务总办彭友兰及永昌知府陈书田连夜逃走，总兵张嘉鈺自杀，腾越全城光复。起义军沿街告示:“张文光纠合义军，除满奴，复汉土，文明举动，勿与百姓仇，百姓各家勿恐。”“及至事定，即派员巡视街巷，各民安眠如恒，义军未敢扰民分厘。”

腾越起义时，英国领事到[[密支那|密支那]]，未在腾，腾越海关税务司英人好威罗，请求出[[缅甸|缅甸]]，张文光派军哨八名礼送出，不久，英国人见腾越起义后秩序井然，全部返腾越。

== 影響 ==
腾越起义的成功，成为云南辛亥革命的先声，推动了昆明起义。三天后，云南昆明“重九”起义爆发。
