'''{{ruby-py|蚌埠|-{zh-cn:Bèngbù ;zh-tw:ㄅㄥˋㄅㄨˋ ;zh-hk:pong<sup>5</sup>fau<sup>6</sup>}-}}市'''，简称'''蚌'''，舊作'''-{蜯埠}-'''、'''-{蜯埔}-'''，别名'''珠城'''，是[[中华人民共和国|中华人民共和国]][[安徽省|安徽省]]下辖的[[地级市|地级市]]、安徽省重要枢纽城市、[[皖北|皖北]]地区的商贸中心与加工制造业中心和邮电通讯指挥调度中心。蚌埠位于[[华北平原|华北平原]]南端[[京沪铁路|京沪铁路]]和[[淮南铁路|淮南铁路]]的交汇点，[[淮河|淮河]]穿城而过，临近[[凤阳|凤阳]]、[[宿州|宿州]]。由于史载蚌埠地区“古乃采珠之地”，素有禹会诸侯地，淮上明珠城之称，从而得名“[[珠城|珠城]]”。1947年初蚌埠设市，直属[[安徽省|安徽省]]，是安徽省第一个设市的[[省辖市|省辖市]]（公元1947年）。蚌埠全市面积5952平方公里，其中市区面积601.5平方公里，截止2016年末，蚌埠市含三县总户籍人口379.52万，常住人口333.20万人，其中城镇常住人口160万。蚌埠市下辖[[龙子湖区|龙子湖区]]、[[蚌山区|蚌山区]]、[[禹会区|禹会区]]、[[淮上区|淮上区]]四个市辖区，管辖[[怀远县|怀远县]]、[[五河县|五河县]]和[[固镇县|固镇县]]三个县。

蚌埠是中国三线城市<ref>{{Cite web |url=http://news.ifeng.com/a/20170526/51165315_0.shtml|title=2017年中国城市分级完整名单·2017城市商业魅力排行榜|author=新一线城市研究室 |date= 2017-05-26|publisher=  [[凤凰资讯|凤凰资讯]] |language=  |accessdate=  |quote=  }}</ref>、[[国家区域中心城市|国家区域中心城市]]<ref>{{Cite web |url=http://mt.sohu.com/20160817/n464702266.shtml|title= 中国10个国家中心城市和100个区域中心城市曝光|author= [[搜狐网|搜狐网]] |date= 2016-07-31|publisher=|language=  |accessdate= |quote=  }}</ref>、[[全国性综合交通枢纽|全国性综合交通枢纽]]<ref>{{Cite web |url=http://www.gov.cn/zhengce/content/2017-02/28/content_5171345.htm|title= 国务院关于印发“十三五”现代综合交通运输体系发展规划的通知|author=中华人民共和国中央人民政府 |date= 2017|publisher=   |language=  |accessdate= |quote=  }}</ref>、[[淮河流域|淮河流域]]中心城市、[[皖北|皖北]]地区中心城市和[[安徽省|安徽省]]旅游中心城市<ref>{{Cite web |url=http://www.ahwang.cn/anhui/20170407/1622677.shtml|title=安徽将建立合肥 芜湖 黄山 蚌埠 四大旅游中心城市|author= |date= 2017-04-07|publisher=安徽网|language=  |accessdate= |quote=  }}</ref>。在[[全国文明城市|全国文明城市]]2016年度测评中，蚌埠居全国第五全省第一。<ref>{{cite web|author=蚌埠市人民政府|title=全国文明城市2016年度测评 蚌埠居全国第五全省第一|url=http://www.bengbu.gov.cn/bbxw/article.jsp?articleId=1064978551|publisher=蚌埠市人民政府办公室|accessdate=2017-04-14}}</ref>2017年11月，第五届[[全国文明城市|全国文明城市]]名单公布，蚌埠市荣获全国文明城市称号。<ref>{{Cite web |url=http://mp.weixin.qq.com/s?__biz=MzA5ODA0NDIzOA==&mid=2652304597&idx=1&sn=fbb942bb0efae4c2ebd0f536386af930&chksm=8b75c4f4bc024de20fe59d0532b63a7bcd33cdb47964a8d16768bf6c7c368aa1a8509470b517&mpshare=1&scene=23&srcid=1114AActv0P68QaH3gLdIamd#rd|title= 第五届全国文明城市名单和复查确认继续保留荣誉称号的往届全国文明城市名单|author= |date= 2017-11-14|publisher=中国文明网|language=  |accessdate= |quote=  }}</ref>当前，随着国家加快实施中部崛起战略，[[安徽省|安徽省]]参与泛[[长三角|长三角]]区域分工合作和建设[[合芜蚌自主创新综合配套改革试验区|合芜蚌自主创新综合配套改革试验区]]<ref>{{Cite web |url=http://money.163.com/16/0608/18/BP2EB0A000253B0H.html|title= 国务院决定建设福厦泉与合芜蚌国家自主创新示范区|author= |date= 2016-06-08|publisher=网易|language=  |accessdate= |quote=  }}</ref>，蚌埠自身的资源优势和交通区位、产业、科技优势将进一步显现。{{OR}}

==地名由来==
[[夏朝|夏朝]]时，现蚌埠西和[[怀远县|怀远县]]属[[涂山氏国|涂山氏国]]，市区东部属[[钟离国|钟离国]]，钟离一名则保持到明洪武年間。

古代，[[淮河|淮河]]出怀远[[荆山|荆山]]口至蚌埠地域，水流趋缓，水质清澈，水草繁盛，适宜河蚌繁殖，且蚌中出珠<ref name="蚌埠市志">《蚌埠市志》，方志出版社，1995年10月，ISBN 978-7-80122-025-7</ref>。清《凤阳府志》记载：“'''蚌步集'''{{NoteTag|蚌埠古称“蚌埠（步）集”。}}在灵璧西南160里，乃古采珠之地。”

明洪武年間，钟离改為中立縣、臨淮縣，增設鳳陽縣；今蚌埠市區西屬懷遠縣，東屬鳳陽縣；今固鎮縣分屬宿州及懷遠、靈璧、五河、鳳陽縣。《[[尚书|尚书]]·[[禹贡|禹贡]]》中记载：“泗滨浮磬，淮夷蠙珠暨鱼。”意即泗水附近产一种青石，可以雕琢磬，蠙就是蚌，此蚌出珠。采珠船舶在此停留，该地设渡成埠，史称蚌埠渡，继而有蚌埠村、蚌埠集等称谓。因此，蚌埠又称珠城。<ref>{{cite book |author=中共蚌埠市委宣传部 |title=蚌埠市情手册 }}</ref>

== 历史 ==
{{Main|蚌埠历史}}
=== 古代 ===
==== 早期文明 ====
距今7000多年前，蚌埠地域有人类活动踪迹。[[夏朝|夏朝]]时，现蚌埠西和[[怀远县|怀远县]]属[[涂山氏国|涂山氏国]]，市区东部属[[钟离国|钟离国]]。<ref name="蚌埠市志"/>

今蚌埠地域曾为淮夷族聚居区。传说[[大禹治水|大禹治水]]南下淮泗，在今怀远县境涂山娶涂山氏女为妻，并生[[启|启]]。<ref>{{cite web |url=http://www.guilin.net/tour/tool/map/googleearth/anhui/bengbu.asp |title=蚌埠市历史沿革 |publisher=安徽蚌埠市三维立体全景卫星电子地图 |date= |accessdate=2009年11月7日 }}{{Dead link|date=2018年8月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

==== 封建時期 ====
[[春秋戰國|春秋戰國]]時期，今一市三縣地域曾分別為[[徐方|徐方]]、[[魯|魯]]、[[宋|宋]]、[[吳|吳]]、[[越|越]]、[[楚|楚]]等國的邑地。

[[秦朝|秦朝]]建立後統一實行郡縣制，今淮河南岸懷遠、蚌埠地域屬九江郡曲陽、鍾離等縣分領，今淮河北岸懷遠、蚌埠、固鎮、五河地域為泗水郡蘄、徐等縣分領。後迭經變更，歷經[[西漢|西漢]]、[[東漢|東漢]]、[[兩晉|兩晉]]南北朝、[[隋|隋]]、[[唐|唐]]、[[五代十國|五代十國]]，至[[南宋|南宋]][[寶祐|寶祐]]五年（1257年），始置懷遠軍及荊山縣，轄今懷遠縣及蚌埠市區西部、固鎮縣一部，今蚌埠市區東部屬濠州鍾離縣；南宋[[咸淳|咸淳]]七年（1271年）始置淮安軍及五河縣。南宋、[[金|金]]對峙時期，今固鎮縣先為宿州蘄縣、靈璧縣及泗州虹縣分領，後為[[宿州|宿州]]、靈璧、荊山、五河等縣分領。  <ref>{{cite web |url=http://www.hefei.gov.cn/n1070/n304695/n480699/4437594.html |title=蚌埠简介 |publisher=中国·合肥门户网站 |date=2008年11月1日 |accessdate=2010年1月22日 }}{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>

元至元二十八年（1291年）撤銷懷遠軍，改荊山縣為懷遠縣，今蚌埠市區西屬懷遠縣，東屬鍾離縣；今固鎮縣為宿州及靈璧、五河、懷遠縣分領，五河縣屬泗州。

明洪武年間，鍾離縣先後改為中立縣、臨淮縣，增設鳳陽縣；今蚌埠市區西屬懷遠縣，東屬鳳陽縣；今固鎮縣分屬宿州及懷遠、靈璧、五河、鳳陽縣。

清同治二年(1863年)，劃鳳陽縣馬村溝以西、懷遠縣席家溝以東、靈璧縣後樓各一片地域為蚌埠轄地，在淮河北小蚌埠正街原鳳陽縣主簿衙署設立三縣司（後遷至淮河南老大街），為蚌埠最早的地方[[行政機構|行政機構]]。三縣司脫離三縣所轄,直屬鳳陽府衙。同時，三縣司由三縣會委一名佐雜，帶10多名兵丁常駐蚌，專管蚌埠一方行政治安之事。最早的三縣司行政長官，由鳳陽人王福堂充任。三縣司機構主要負責處理民事糾紛,兼有司法、行政兩權。但縣司並不勤於政務。<ref name="解放前的蚌埠地区政府历史沿革 ">{{cite web |url=http://www.ahage.net/bengbu/bengbuzhengfu.html |title=解放前的蚌埠地区政府历史沿革 |publisher=安徽文化网 |date=2008年7月12日 |accessdate=2010年1月22日 |archiveurl=https://web.archive.org/web/20160305024117/http://www.ahage.net/bengbu/bengbuzhengfu.html |archivedate=2016年3月5日 |deadurl=yes }}</ref>

1909年，[[津浦铁路|津浦铁路]][[蚌埠淮河铁路老桥|蚌埠淮河桥]]开工建设。1911年淮河桥和[[蚌埠火车站|蚌埠火车站]]建成后，逐渐形成较为繁华的地区。<ref name="蚌埠市志"/><ref>{{cite web |url=http://www.bbe8.com/bengbu/bengbu_city.htm |title=蚌埠市历史 |publisher=蚌埠易发网 |date= |accessdate=2010年1月22日 |deadurl=yes |archiveurl=https://web.archive.org/web/20080514194448/http://www.bbe8.com/bengbu/bengbu_city.htm |archivedate=2008年5月14日 |df= }}</ref>

=== 近现代 ===
==== 中華民國 ====
1912年[[中華民國|中華民國]]成立，首都[[南京|南京]]，蚌埠的地理位置，决定了其保衛首都的重要軍事地位。民國元年，蚌埠為[[鳳陽|鳳陽]]縣轄區，三縣司廢。在蚌埠設置二區保衛團團防局，局長由蔣雲卿擔任，民國2年底，軍閥[[倪嗣沖|倪嗣沖]]在蚌督皖，靠警察機構代行地方政務，區團防局作為行政機構只徒具虛名。民國14年10月[[直系|直系]]軍及[[奉系|奉系]]軍在蚌埠爆發[[蚌埠戰鬥|蚌埠戰鬥]]，民國16年[[北伐|北伐]]勝利後，團防局撤銷。[[鳳陽|鳳陽]]縣在蚌埠設區公所。區長有年更一次者，有數月更一次者。1929年1月，蚌埠曾籌備[[建市|建市]]，並成立市政籌備處，由裴益祥主持。但至翌年春，該籌備處即撤銷。<ref name="解放前的蚌埠地区政府历史沿革 "/> 民國21年，鳳陽縣改為自治區，將原17個區並為10個區，蚌埠為第七區，仍設區公所。民國24年，鳳陽縣行政區劃調並為4個區,蚌埠為其第二區，設區署。

1937年[[中日戰爭|中日戰爭]]全面爆發，当时國民政府在蚌埠驻有重兵，[[桂系|桂系]]廖磊的第七军，东北[[俞学忠|俞学忠]]的第五十一军，均驻扎此地，统归第五战区司令官[[李宗仁|李宗仁]]指挥。日軍为侵占这一重要据点，先以大批飞机侦察轰炸，接着以排炮(即多门大炮同时发射)为掩护，并以铁甲车为前锋，沿铁路线两侧，乘隙攻击进犯。终于在1938年2月21日，日军分由凤阳刘府、临淮关、长淮卫及沿着[[淮河|淮河]]南岸分数路合笼。在沿淮河进犯时，用小汽艇散放烟幕掩护，进入蚌埠市区。国军不敵，向[[徐州|徐州]]方向撤退。在抗战時期，蚌埠為日本所扶持的安徽省維新政府和安徽省政府所在地。<ref>{{cite web |url=http://www.ahage.net/bengbu/1317.html |title=蚌埠历史风云——残暴日寇侵陷蚌埠的暴行 |publisher=安徽文化网 |date=2007年11月13日 |accessdate=2010年1月22日 |archiveurl=https://web.archive.org/web/20090615142937/http://www.ahage.net/bengbu/1317.html |archivedate=2009年6月15日 |deadurl=yes }}</ref><ref name="日寇侵陷蚌埠的暴行">{{cite web |url=http://www.bbzx.gov.cn/detail/news.asp?id=371 |title=日寇侵陷蚌埠的暴行 |publisher=蚌埠政协网 |date=2005年12月8日 |accessdate=2010年1月23日 }}{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>

==== 抗战時期 ====
民國27年暨1938年，汪政府安徽省政府成立，省府设在蚌埠，<ref>{{cite web |url=http://www.bbzx.gov.cn/detail/news.asp?id=376 |title=日伪时期蚌埠傀儡政权更迭情况 |publisher=蚌埠政协网 |date=2005年12月8日 |accessdate=2010年1月22日 |deadurl=yes |archiveurl=https://web.archive.org/web/20150923182824/http://www.bbzx.gov.cn/detail/news.asp?id=376 |archivedate=2015年9月23日 |df= }}</ref> 至此，蚌埠便成为抗战時期安徽的[[政治|政治]]、[[经济|经济]]、[[文化|文化]]和[[交通|交通]]中心，也成为迁往重庆的国民政府對日斗争的主要地区。

同年，[[日本|日本]]軍攻占蚌埠，在日本統治時期，日本扶持建立了投日的地方政府并與[[漢奸|漢奸]]相勾結，在幾條狹窄的馬路邊開設[[洋行|洋行]]和[[俱樂部|俱樂部]]。 日本在蚌埠設的洋行眾多，較大的有[[三井|三井]]洋行、[[東棉|東棉]]洋行、[[三菱|三菱]]洋行、揚子蛋品公司、[[東芝|東芝]]洋行、[[三洋|三洋]]株式會社、[[江南株式會社|江南株式會社]]等等，這些洋行由日本的大小資本家掌管。<ref>{{cite web |url=http://www.ahage.net/bengbu/8184.html |title=蚌埠沦陷时期的洋行、俱乐部和戒吸所 |publisher=安徽文化网 |date=2008年12月9日 |accessdate=2010年1月22日 |archiveurl=https://web.archive.org/web/20140426232710/http://www.ahage.net/bengbu/8184.html |archivedate=2014年4月26日 |deadurl=yes }}</ref> 日本設立各色洋行的目的，是把占領區變成原料補給基地和市場。此外日本在三號碼頭建立了上海內河輪船公司蚌埠分公司，出售客票，日本人還經營控制著內河水上所有的貨物運輸。資料顯示，抗战時期的蚌埠水陸運輸繁忙，且目的地幾乎涉及所有日占區。日軍利用沈席儒，在蚌埠開設俱樂部，實際是公開的[[賭場|賭場]]。<ref name="蚌埠市志"/><ref name="日寇侵陷蚌埠的暴行"/>

==== 國共內戰 ====
中國[[中国抗日战争|抗日戰爭]]勝利後，懷遠縣改屬安徽省第十專區，鳳陽縣先後改屬安徽省第九專區、第五專區，五河縣屬安徽省第四專區。民國34年11月25日，經[[安徽省|安徽省]]主席[[李品仙|李品仙]]頒令，報準[[國民政府|國民政府]][[行政院|行政院]]核備，蚌埠重新成立市政籌備處，於民國35年1月起接管蚌埠行政事務。1946年11月，蚌埠成立市政籌備處，脫離鳳陽縣，轄原屬鳳陽縣的淮河以南部分地區和淮河以北小蚌埠地區。<ref name="蚌埠市志"/>

1947年1月1日，經[[國民政府|國民政府]][[中華民國內政部|內政部]]批准，蚌埠正式設市，直屬安徽省，為安徽省第一個設市的城市。市政府即日成立，李品和為蚌埠市首任市長。市政府建立後，正值國內全面[[国共内战|內戰]]。市政府為維護國民政府統治，很快建立起保甲組織，按照“每甲10-30戶，每保10-30甲，每區10-30保”的原則，全市共設5區98保1887甲。

国军在辽沈战役中节节失利，令总统蒋中正预感到东北方面败局已定。考虑到下一步解放军中野有进攻郑州与华野在江淮地区会合的可能，为加强对防守中原和华东地区，扭转不利态势，巩固江淮、屏障南京，国军集结了五个兵团和三个绥靖区的部队共80万人，以[[徐州|徐州]]为中心，利用津浦、[[陇海铁路|陇海]]两条铁路，组成东起海州、西迄商丘、北起临城、南达蚌埠，“一点两线”的防御阵线，采取战略守势，准备抗击共产党的进攻。1948年10月下旬，国军调整了战略部署，按照白崇禧“守江必守淮”的指导思想，将徐州剿总的兵力向蚌埠方向转进，重点守备淮河一线。1948年11月6日夜，解放军华东野战军发起[[淮海战役|淮海战役]]。本战役对全中国乃至世界局势都有深远影响。<ref>[[杜聿明|杜聿明]]，《淮海战役始末》，中国政协文史资料委员会编《中华文史资料文库》（中国文史出版社，1996年），第7册第1-25页。见 [http://www.hoplite.cn/templates/hpzc2nz0015.html 转载]。</ref><ref>張福群,1995,蔣介石功過錄 p122</ref>

1949年1月20日，蚌埠被攻克。1949年10月1日[[中華人民共和國中央人民政府_(1949–1954)|中華人民共和國中央人民政府]]成立，蚌埠市、懷遠縣、五河縣屬皖北行署，今固鎮縣分屬宿縣、五河、靈璧、懷遠4個縣。

=== 当代 ===
1952年4月，蚌埠市直屬安徽省；[[宿縣|宿縣]]、[[五河|五河]]、[[靈璧|靈璧]]、[[懷遠|懷遠]]4縣改屬安徽省[[宿縣專區|宿縣專區]]，1956年1月改屬[[蚌埠專區|蚌埠專區]]，1961年4月仍屬宿縣專區。 1964年10月，析宿縣、懷遠縣、五河縣、靈璧縣各一部分地區置固鎮縣，屬宿縣專區。 1971年宿縣專區改為地區。1983年7月，懷遠縣、五河縣、固鎮縣由宿縣地區改劃屬蚌埠市至今。

中華人民共和國建國后，蚌埠是國內為數不多的重要[[工業|工業]]城市，有著良好的[[轻工业|輕]]、[[重工業|重工業]]基礎。 “[[一五|一五]]”期間的100個重點項目，蚌埠一地即佔3項 。这时期，蚌埠市经济实力位居全省第二，之后由于交通的优势地位受到削弱、外部区域环境的变化，经济发展活力渐渐不足，城市地位下降，但仍是安徽北部的重要城市和交通枢纽。随着安徽省区域经济发展战略思路的调整，蚌埠成为安徽省重点培育的中心城市之一。<ref>{{cite web|author=中国城市规划行业信息网|title=蚌埠市城市总体规划(2005年-2020年)|url=http://www.cityup.org/case/general/20070628/32270.shtml|accessdate=2010-08-23|date=2007-06-29}}</ref>

== 地理 ==
[[File:Large_oJBa_238i200066.jpg|thumb]]西岸的中国南北分界雕塑]]
=== 位置范围 ===
蚌埠市位于东经116°45′-118°04′，北纬32°43′至33°30′之间，其中市区位于东经117°31′-117°11′、北纬33°01′-32°49′之间，东至五河县浮山峡山口，与[[明光市|明光市]]和[[江苏省|江苏省]][[泗洪县|泗洪县]]毗邻；西至怀远县万福镇关圩村颜李庄，与[[蒙城县|蒙城县]]、[[凤台县|凤台县]]搭界；北至固镇县大韩家西北的沱河中心线，与[[濉溪县|濉溪县]]、[[宿州市|宿州市]]、[[灵璧县|灵璧县]]、[[泗县|泗县]]接壤；南至怀远县常坟镇新桥村新庄南，与[[淮南市|淮南市]]、凤阳县相连。全市东西长135km，南北宽86.5km。<ref>{{cite web|title=地理位置|url=http://www.bengbu.gov.cn/dt2111111203.asp?DocID=2111114040|accessdate=2010-08-24|deadurl=yes|archiveurl=https://web.archive.org/web/20100405055839/http://www.bengbu.gov.cn/dt2111111203.asp?DocID=2111114040|archivedate=2010年4月5日|df=}}</ref>

蚌埠市地处安徽省北部淮河流域，京沪铁路与淮南铁路相连，淮河以北处于淮北平原南端，以南处于江淮丘陵区。
=== 地形 ===


蚌埠市位于安徽省东北部，[[淮河|淮河]]中游，地处安徽省淮北平原和[[江淮丘陵|江淮丘陵]]的交接地带，地形以[[平原|平原]]为主。地势西北高，东南低，自然坡降约为万分之一。南部地区分布有少量的[[丘陵|丘陵]]和[[台地|台地]]。

淮河穿行在怀远县荆山、涂山之间和五河县铁锁岭、浮山处，形成峡口荆山峡和浮山峡{{NoteTag|淮河中游有淮河小三峡，分别为：峡三峡、荆山峡和浮山峡。}}。

=== 地质 ===
蚌埠地区的地质构造较为复杂，东西向和北东向呈断裂发育。其中，东西向以太古代变质岩所构成的复背斜为基础，[[背斜|背斜]]的北翼是[[新生代|新生代]]断陷盆地。全市位于[[中朝准地台|中朝准地台]]南缘的淮河台坳中部，次级构造单元为蚌埠台拱。

蚌埠所在的[[地层|地层]]发育不全，基岩露头零星，全市位于华北地层区的淮河地层分区。蚌埠的地表以大面积沉积物覆盖。岩石地层主要是[[侵入岩|侵入岩]]和[[变质岩|变质岩]]，还有少量[[花岗岩|花岗岩]]、[[云母|云母]]、[[钾长石|钾长石]]、[[蛇纹岩|蛇纹岩]]、[[重晶石|重晶石]]、石棉[[第四纪|第四纪]]松散沉积物等。<ref>{{cite web|title=安徽省蚌埠市地理概况|url=http://www.qlxxw.cn/cn/120744.html|accessdate=2010-08-23}}</ref> 已发现矿床（点）38处，矿种约22种。<ref>{{cite web|author=中国非金属矿网|title=蚌埠的矿产资源|url=http://www.nmm365.cn/mr/nmmmr0139.htm|accessdate=2010-08-24}}</ref>

===水文===
蚌埠市的水资源较为丰富，最大的自然地表水是淮河，共有水面409km<sup>2</sup>，年平均径流深244mm，相应流量为14.34亿m<sup>3</sup><ref name="安徽省情概览">{{cite book|title=安徽省情概览|author=庄祖武、孙永珊|year=1995年|publisher=安徽人民出版杜|isbn=7-212-01211-4|accessdate=2010-09-04}}</ref>。

淮河以北河流湖泊较多，有江南水乡的特点，较大的地表水和湖泊有沱河、浍河、北淝河、澥河、涡河、芡河、茨淮新河、怀洪新河、香涧湖、沱湖、天井湖、四方湖等。淮河以南的地形以小的丘陵为主，较大的地表水仅有天河、龙子河等。

地下水资源主要分布在蚌埠市所辖三县，有埋藏浅、水质好、储量大的特点，地下水资源较为丰富<ref>{{cite web|title=安徽省蚌埠市地理概况|url=http://www.qlxxw.cn/cn/120744.html|publisher=齐鲁信息网|accessdate=2010-09-05}}</ref>。

== 经济 ==
[[File:蚌埠市淮河路沿线.jpg|thumb]][[File:蚌埠市东海大道.png|thumb]][[File:蚌埠市万达广场-南翔城市广场.jpg|thumb]]
清末时期，蚌埠地区的农民多以家庭作坊方式，从事草鞋编制、麻油压榨等活动，并在集市上出售。民国开埠后，[[手工业|手工业]]迅速发展，到1937年发展到43个行当，作坊、工场近700户。中华人民共和国成立后，皖北行署提出“确定以蚌埠为经营重点”的发展方针<ref>{{cite book |author=侯永，欧远方 |title= 当代安徽纪年 |publisher= 当代中国出版社年版|year= 1992 |month= |isbn= }}</ref>。由于基础相对较好且有政策倾斜，蚌埠市逐渐形成较为完备的工业体系，[[改革开放|改革开放]]以前，蚌埠一直是安徽省最重要的工业城市之一。改革开放以后，蚌埠市的经济地位有所下降，但仍有较强的工业实力。2006年，[[安徽省|安徽省]]提出两淮一蚌战略。“两淮”为[[淮南|淮南]]、[[淮北|淮北]]，“一蚌”为蚌埠市<ref>{{cite web | language =zh  | publisher =安徽省发展和改革委员会 | title =构建“两淮一蚌”城市经济圈 促进皖北地区跨越式发展 | url =http://www.ahpc.gov.cn/info.jsp?xxnr_id=10006502  | author =安徽省发展和改革委员会 经研所《安徽经济》2006第 1－2期  | date =2006-06-27  | accessdate =2009-08-27  }}</ref>，希望促进老工业城市重振雄风，但收效甚微，目前已不再提及该战略。
2008年底，安徽省提出建设[[合芜蚌自主创新试验区|合芜蚌自主创新试验区]]的构想，2009年12月，国家正式批复该综合配套改革试验区<ref>{{cite web | language =  | publisher =中安在线  | title =合芜蚌自主创新试验区破茧而出 | url =http://ah.anhuinews.com/system/2008/10/17/002137745.shtml  | author =中安在线-[[安徽日报|安徽日报]]   | date =2008-10-17  | accessdate =2009-08-27  }}</ref><ref>{{cite web | language =zh  | publisher =中安在线  | title =“合芜蚌试验区”上升到国家层面 | url =http://news.xinmin.cn/rollnews/2010/01/01/3228641.html  | author =新民网 |  date =2010-01-01  | accessdate =2010-01-02 }}</ref>。随后几年，蚌埠市又相继提出城市大建设、重返全省第一方阵等措施，蚌埠的经济实力有显著提升。2016年，合芜蚌自主创新试验区跻身为[[国家自主创新示范区|国家自主创新示范区]]<ref name="福建、安徽">{{cite web | url=http://money.163.com/16/0608/18/BP2EB0A000253B0H.html | title= 国务院：建设福厦泉与合芜蚌国家自主创新示范区  | publisher=网易财经 | accessdate=2016-07-07}}</ref>。

=== 农业 ===
蚌埠所辖三县均为全国商品粮大县，农副产品资源丰富，大部分自然植被早已被连片的农业植被所取代。农作物熟制为一年两熟，冬季多种植[[小麦|小麦]]，同时也有部分区域种植两季稻。由于蚌埠市地跨淮河两岸，淮河以南夏季粮食作物多[[水稻|水稻]]，淮河以北[[玉米|玉米]]的种植率逐渐增高<ref>{{Cite thesis |chapter= |title=当代安徽淮河流域工业发展研究——以年鉴、方志等史料为中心 |url= |degree=Ph.D.|author=王成 |last= |first= |year=2014 |publisher=安徽大学 |accessdate=2016-07-07 |docket= |oclc= }}</ref>。从耕作面积来看，小麦所占面积居绝对优势，稻谷和玉米耕作面积大致相当，稻谷耕作面积稍多。[[经济作物|经济作物]]方面，本区域油料、果蔬、肉类和水产品等均较为丰富，比较有名的有固镇花生、怀远石榴、五河沱湖螃蟹等。

===工业===
[[中华人民共和国|中华人民共和国]]成立后到20世纪末的50年间，蚌埠形成的工业体系主要有机械、热电、化工、生化、建材、卷烟、酿造、食品、纺织、医药、电子等。能生产400多大类、上万种产品。工业结构中，轻纺工业一直居于主要地位。

进入21世纪，蚌埠市作为安徽省重要的工业城市，经过多年建设和发展，经济实力有了较大增强。现有工业企业1258家，其中大中型企业66家。近年来，蚌埠市全面推进[[国有企业|国有企业]]改革，加快资产重组步伐，着力实施“创牌造舰”战略，工业结构出现了新的变化，新的优势产业逐步形成，涌现出一批骨干企业和优势产品。蚌埠现以精细化工、玻璃深加工及汽车零部件为主导产业。此外，蚌埠的仿古玉器制造行业规模较大，是国内三大[[古玩|古玩]][[玉器|玉器]]生产加工地。

蚌埠的大型企业主要有[[丰原集团|丰原集团]]、[[蚌埠卷烟厂|蚌埠卷烟厂]]、[[蚌埠玻璃工业设计研究院|蚌埠玻璃工业设计研究院]]等，其中丰原集团为亚洲第一、世界第三的[[柠檬酸|柠檬酸]]生产企业，现有上市公司五家；皖酒集团具有六十年酿造白酒的历史，是安徽省白酒行业支柱企业；玻璃工业设计研究院自主开发了数百项专有技术成果，为中国玻璃工业现代化和发展循环经济提供了强有力的技术支撑。

===金融商贸业===
[[津浦铁路|津浦铁路]]通车后，蚌埠吸引皖北、皖西、豫东、苏北西部等地产品运到蚌埠后改用火车装运，转向津沪等地，成为区域性水陆交通枢纽。随着产品物资和货币流量的提高，金融业出现并发展起来，中国银行、[[上海商业储蓄银行|上海商业储蓄银行]]等相继在蚌埠设立分支机构。金融机构一般选址在水陆交通便捷的二马路、新船塘等区域<ref>{{cite journal |last=马兰花  |first=|authorlink= |coauthors= |year=2015 |month= |title=铁路交通对近代蚌埠金融业的影响 |journal= 安徽师范大学学报 |volume= |issue= |pages= |id= |url= |accessdate= 2016-07-07 |quote= }}</ref>。

当前，蚌埠是皖北重要的商贸中心城市，光彩大市场、义务商贸城等批发零售市场辐射周边。蚌埠保税物流中心、蚌埠铁路无水港等配套设施为商贸业发展提供支持。金融商贸业最发达的地区位于淮河路沿线和东海大道、朝阳路附近。吸引了万达广场、新地城市广场、银泰城、宝龙城市广场、海亮城市广场、国购广场等诸多商业综合体入驻。2015年，全市社会消费品零售总额570.6亿元<ref>[http://data.ahtjj.gov.cn/ 安徽数据 安徽省统计局]</ref>。

== 政治 ==
===党派===
国民党蚌埠市党部建立于1926年，党部主要从事宣传训练、筹集资金、控制社团和监视等活动。中华人民共和国成立前夕，时任书记长邵恩逃离。

1926年，苏美一等人建立蚌埠地区最早的党组织中共蚌埠特别支部，后又相继建立了小蚌埠支部、工作委员会、工委等。1949年正式建立中共蚌埠市委员会。1952年安徽省委成立，市委接受省委领导。目前，蚌埠市委与市政府驻地同为东海大道政府办公大楼。此外，1949年之前蚌埠市还有[[中国民主社会党|中国民主社会党]]、[[中国青年党|中国青年党]]等，但势力较小，1949年前后相继解散。

中华人民共和国成立后，部分[[民主党派|民主党派]]组织建立。其中，[[中国国民党革命委员会|中国国民党革命委员会]]市委员会成立于1950年，定期开展党员大会；[[中国民主同盟|中国民主同盟]]市委员会和[[中国民主建国会|中国民主建国会]]市委员会成立于1953年；[[中国农工民主党|中国农工民主党]]市委员会成立于1956年；[[九三学社|九三学社]]市委员会成立于1958年；[[中国民主促进会|中国民主促进会]]市委员会成立于1982年<ref name="蚌埠市志"/>。

===政府===
[http://www.bengbu.gov.cn/ 蚌埠市人民政府]是中华人民共和国安徽省蚌埠市的一级国家行政机关，是蚌埠市人民代表大会的执行机关。蚌埠市人民政府对蚌埠市人民代表大会负责并报告工作。蚌埠市人民政府在蚌埠市人民代表大会闭会期间，对蚌埠市人民代表大会常务委员会负责并报告工作。

=== 行政区划 ===
{{main|蚌埠市行政区划}}
中华人民共和国建国初期，蚌埠市由[[皖北行署区|皖北行署区]]直辖，1952年安徽省成立后，蚌埠市由安徽省直辖<ref>{{cite web|title=安徽行政区划调整大构想|url=http://3y.uu456.com/bp_40tok5ge6q20sz53255g_1.html|accessdate=2016-07-06}}{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>。

自1956年起，蚌埠市下辖东市区、西市区、中市区和郊区共四区，后逐渐吞并临近县的部分地域。1983年起，[[怀远县|怀远县]]、[[五河县|五河县]]、[[固镇县|固镇县]]三县归蚌埠市管辖，自此，市辖地域与目前大体相当。2004年，东、西、中、郊四区分别更名为龙子湖、禹会、蚌山、淮上区，并并入怀远县和固镇县的部分乡镇。2013年和2014年，禹会区和淮上区又分别并入部分县辖乡镇<ref>{{cite web|title=蚌埠市新一轮行政区划调整 沫河口划归淮上区|url=http://news.focus.cn/bb/2014-04-09/4911272.html|publisher=搜狐焦点|accessdate=2016-07-06}}</ref>。

蚌埠市现辖[[龙子湖区|龙子湖区]]、[[蚌山区|蚌山区]]、[[禹会区|禹会区]]、[[淮上区|淮上区]]4个[[市辖区|市辖区]]，以及怀远县、五河县、固镇县3个[[县_(中华人民共和国)|县]]。蚌埠市还设立两个经济管理区，分别是[[国家级高新技术产业开发区|国家级]]高新区[[蚌埠高新技术产业开发区|蚌埠高新技术产业开发区]]和由原新城区改建而来的[[蚌埠经济开发区|蚌埠经济开发区]]。

截止2008年末，蚌埠户籍总户数100.99万户，其中，[[市辖区|市辖区]] 28.79万户。年末总人口358.31万人，比年初增加3.04万人。总人口中，农业人口258.05万人，非农人口100.26万人。市辖区年末总人口92.51万人，其中非农人口65.50万人。全市人口中，男性人口185.57万人，占总人口的51.8%；女性人口172.74万人，占总人口的48.2%。<ref>{{cite web |url=http://www.bengbu.gov.cn/dt2111111203.asp?DocID=2111114038 |title=人口 |publisher=蚌埠市人民政府 |date=2009年8月10日 |accessdate=2009年11月8日 }}{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}
</ref> 2006年-2010年，全市死亡人口呈逐年上升趋势，从期初的2.22万左右上升到2010年的2.43万，死亡人口的增加与老年人口在整个人口比重上升有关，并随着人口老龄化日益严重，死亡人口逐年上升。<ref>{{cite web|title=蚌埠市“十一五”人口规划|url=http://www.bengbu.gov.cn/gh/yjkt/8.htm|accessdate=2010-08-26|deadurl=yes|archiveurl=https://web.archive.org/web/20051122235103/http://www.bengbu.gov.cn/gh/yjkt/8.htm|archivedate=2005年11月22日|df=}}</ref>

根据2010年[[第六次全国人口普查|第六次全国人口普查]]，全市[[常住人口|常住人口]]为3164467人<ref>{{cite book |author=蚌埠市统计局|title=《蚌埠市2010年第六次全国人口普查主要数据公报》}}</ref>，同[[第五次全国人口普查|第五次全国人口普查]]相比，十年共减少123862人，下降3.77%，年平均下降0.38%。其中，男性人口为1612771人，占50.97%；女性人口为1551696人，占49.03%。总人口性别比（以女性为100）为103.94。0－14岁人口为585261人，占18.50%；15－64岁人口为2243337人，占70.89%；65岁及以上人口为335869人，占10.61%。居住在城镇的人口为1373554人，占43.41%，居住在乡村的人口为1790913人，占56.59%。

全市以[[汉族|汉族]]居多，此外还有[[回族|回族]]、[[蒙古族|蒙古族]]、[[朝鲜族|朝鲜族]]、[[满族|满族]]、[[藏族|藏族]]、[[苗族|苗族]]、[[壮族|壮族]]等36个少数民族<ref>{{cite web|url=http://dpc.bb.ah.cn/detail/news.asp?id=34|title=蚌埠概况|accessdate=2010-08-26|deadurl=yes|archiveurl=https://web.archive.org/web/20110815054634/http://dpc.bb.ah.cn/detail/news.asp?id=34|archivedate=2011年8月15日|df=}}</ref>，各民族分散杂居。少数民族约占全市总人口的1.1%，其中回族人数最多。

=== 方言 ===
蚌埠话属于[[中原官话|中原官话]][[信蚌片|信蚌片]]，与中原官话其他各片相同，具有阴平、阳平、上声、去声四个声调，但在语音上与其他各片相差较大<ref>贺巍. 中原官话分区(稿)[J]. 方言,2005,02:136-140.</ref>。信蚌片在蚌埠地区又可细分为蚌固型和五凤型，两者语音特征不同。例如，在处理古入声字时，蚌固型将清入、次浊入多数归阴平，将全浊入归阳平，而五凤型将绝大多字归去声<ref>贡贵训. 皖北中原官话信蚌片的语音差异[J]. 宁夏大学学报(人文社会科学版),2011,06:26-30.</ref>。怀远县西南部分地区与蚌埠市其他地区略有不同，其语音上与淮南话、江淮官话洪巢片更为相似。

=== 宗教 ===
五河县是[[天主教|天主教]]最早传入安徽省的地方，清顺治六年（1649年）汤若望即来此传教。雍正以后查禁天主教，教堂被毁，但仍有少量地下天主教徒。晚清时期，法国籍耶稣会神父又以五河县为基地，向安徽省北部淮河沿岸地区传教。天主教[[蚌埠教区|蚌埠教区]]主教座堂建于1921-1922年间，位于蚌埠市龙子湖区与蚌山区交界处的淮河路天桥东南，购地4000多平方米，建教堂、楼平房计100多间，目前仅存一塔。

=== 体育 ===
清末民间武术和自娱性的健身活动的兴起标志蚌埠体育发展的开始。20世纪20年代，[[篮球|篮球]]、[[足球|足球]]、[[乒乓球|乒乓球]]、[[田径|田径]]等现代体育运动项目相继兴起。1973年-2004年，蚌埠的体育健儿在各项比赛中共有4人获世界冠军，113人次获全国冠军，并先后破[[世界纪录|世界纪录]]2人次、破世界青少年纪录1人次、破全国纪录1人次、破省纪录81人次。蚌埠市于1981年成为全国足球重点城市，有“足球城”的誉称，这是[[国家体育运动委员会|国家体育运动委员会]]指定的全国20个城市之一。首届中超冠军、1992年7月芬兰赫尔辛基杯冠军、亚冠联赛四强获得者[[李毅_(足球运动员)|李毅]]来自蚌埠，他发明的一个护球技巧便命名为“蚌埠回旋”<ref>{{cite web|title=体育概况|url=http://www.bengbu.gov.cn/dt2111111206.asp?DocID=2111114067|accessdate=2010-09-04|deadurl=yes|archiveurl=https://web.archive.org/web/20100608040439/http://www.bengbu.gov.cn/dt2111111206.asp?docid=2111114067|archivedate=2010年6月8日|df=}}</ref>。

中华人民共和国成立初期，蚌埠市的体育设施极差，仅在小南山上设置了2个[[篮球|篮球]]场。之后，体育场所的总面积逐渐增加，其中主要有位于市区繁华地段，占地面积达16.8万平方米的蚌埠市体育场和内部方型结构类似“[[水立方|水立方]]”、构架类似“[[国家体育场_(北京)|鸟巢]]”的[[蚌埠龙湖体育馆|蚌埠龙湖体育馆]] <ref>{{cite web|author=王朝网络|title=蚌埠龙湖体育馆酷似“鸟巢” 由“水立方”设计单位设计|url=http://www.wangchao.net.cn/lvyou/detail_62724.html|accessdate=2010-09-04}}</ref>。以承办[[安徽省第十四届运动会|安徽省第十四届运动会]]为契机，蚌埠市建设了一批新的体育运动场所，包括[[蚌埠体育中心|蚌埠体育中心]]（奥体中心）、[[龙子湖区全民健身体育中心|龙子湖区全民健身体育中心]]、[[禹会区全民健身体育中心|禹会区全民健身体育中心]]和[[龙子湖体育公园|龙子湖体育公园]]等。同时对蚌埠市体育场进行升级改造<ref>{{cite web|author=蚌埠新闻网|title=五个体育场馆今年陆续开建|url=http://www.bbnews.cn/xw/jrtj/2016/03/520187.shtml|accessdate=2018-07-24}}</ref>。

=== 教育 ===
{{main|蚌埠教育}}
民国时期，县学和[[书院|书院]]相继消失，新式教育与[[私塾|私塾]]长期并存。直至中华人民共和国成立前夕，全市仍保留私塾30余所，学童400多人。自津浦铁路通车到1938年，全市共创立小学20余所，其中多为学制四年，开设二至四个班的初级小学，还有少数学制六年，开设六个班的高级小学，主要小学有正街小学、船民小学、回民成德小学、私立勤民小学、含美小学等。二十年代中期开始出现中学，位于华丰街厚德里创设的皖北中学是蚌埠最早的中学，而私立江淮中学发展较好，当时成为皖北地区颇有影响的中学，是今蚌埠二中的前身。<ref>王洪刚. 蚌埠早期城市现代化研究（1908-1947）[D].扬州大学,2017.</ref>

蚌埠目前拥有[[安徽财经大学|安徽财经大学]]、[[蚌埠医学院|蚌埠医学院]]、[[蚌埠学院|蚌埠学院]]、[[安徽科技学院|安徽科技学院]]、[[安徽财经大学商学院|安徽财经大学商学院]]、[[安徽电子信息职业技术学院|安徽电子信息职业技术学院]]等本专科院校，[[中国人民解放军陆军装甲兵学院|中国人民解放军陆军装甲兵学院]]、[[中国人民解放军空军航空大学|中国人民解放军空军航空大学]]、[[中国人民解放军陆军军事交通学院汽车士官学校|中国人民解放军陆军军事交通学院汽车士官学校]]、[[中国人民解放军海军士官学校|中国人民解放军海军士官学校]]等军事院校。在基础教育方面，有[[蚌埠一中|蚌埠一中]]、[[蚌埠二中|蚌埠二中]]、[[蚌埠三中|蚌埠三中]]、蚌埠四中、[[蚌埠鐵路中學|蚌埠鐵路中學]]、[[懷遠一中|懷遠一中]]、[[蚌埠四中|蚌埠四中]]、[[懷遠二中|懷遠二中]]、[[懷遠三中|懷遠三中]]等八所省級示範高中。

=== 医疗 ===

====普速铁路====
[[蚌埠站|蚌埠站]]是[[华东地区|华东地区]]重要的铁路枢纽，为一等站。在修建原[[津浦铁路|津浦铁路]]时，因蚌埠集附近淮河水域平缓、建桥方便，因此原津浦铁路工程师德纪决定放弃原先的从[[临淮镇_(凤阳县)|临淮镇]]过江方案，改为[[蚌埠淮河铁路老桥|在蚌埠架桥]]过河并在此设立车站<ref>{{cite web|url=http://www.ah.xinhuanet.com/news2005/2006-10/25/content_8387311.htm|title=借款修建、见证历史的车站--蚌埠火车站|publisher=新华网|accessdate=2006-10-25|deadurl=yes|archiveurl=https://web.archive.org/web/20160304203757/http://www.ah.xinhuanet.com/news2005/2006-10/25/content_8387311.htm|archivedate=2016-03-04}}</ref>。[[上海铁路局|上海铁路局]]铁路办事处曾驻蚌埠并管辖[[安徽省|安徽省]]铁路，现办事处已迁至[[合肥|合肥]]。蚌埠站铁路日接发能力320列，从蚌埠乘火车可直达国内多数重要城市。[[安徽省|安徽省]]通行列车的一百多年来，[[蚌埠站|蚌埠站]]'''一直是省内列车通达度最高的车站'''。<ref>{{cite web|url=https://news.uc.cn/a_10993830772633863724/ |title=安徽16个城市铁路通达度排行|publisher= UC头条 |accessdate= 2017-06-27 }}</ref>蚌埠是[[京沪铁路|京沪铁路]]、[[水蚌铁路|水蚌铁路]]铁路枢纽，南下[[水蚌铁路|水蚌铁路]]列車在此变更车次，替换车头；蚌埠站于2007年6月1日通行[[动车组|动车组]]，为安徽省首个动车停靠站，使蚌埠巩固了与长三角的紧密联系。

[[蚌埠东站|蚌埠東站]]史称长淮卫站，初建于1909年，于1975年6月1日重建，拥有中国华东地区一流的大型货运编组站及最大的货场，现为国家一等站，並拓展了铁海联运业务。蚌埠东站以车辆解编作业为主要工作，并和蚌埠站携手提高运输编组效率。<ref>{{cite web|author=陈凯|title=蚌埠站和蚌埠东站携手提高运输编组效率|url=http://news.chineserailways.com/Html/13/201004/20100403164926.html|publisher=人民铁道报|accessdate=2010-09-05|deadurl=yes|archiveurl=https://web.archive.org/web/20100621234201/http://news.chineserailways.com/HTML/13/201004/20100403164926.html|archivedate=2010年6月21日|df=}}</ref>

====2000年前====
蚌埠机场历史悠久，民国时期为重要的军用空港。20世纪时，蚌埠机场位于现龙子湖区淮河文化广场南侧，当时开行民航班机、军民共用，但客流十分惨淡。1999年，因机场严重制约城市发展被整体拆除。后为迎接新世纪的到来，蚌埠市在原老机场地块修建了蚌埠市地标建筑之一的淮河文化广场、改建空军第十三飞行学院，修筑或改造涂山路、解放路、延安路等主干道路。{{来源请求|time=2018-3-4}}

====2000年后====
因蚌埠市重要的军事地位和区域战略位置，主城区西南重新修建了一座4C级标准军用机场，位于蚌埠市禹会区燕山乡仁和集，机场代号为BFU和ZSBB，但至今未开行民用航空。蚌埠腾湖机场选址于怀远县淝河乡腾湖，设计标准为4C，目前正处筹备阶段。<ref>{{cite web|author=蚌埠论坛|title=蚌埠民用机场选址确定 位于怀远县淝河乡腾湖|url=http://365jia.cn/news/2016-02-15/2A10A54D1413F9F4.html|publisher=万家资讯|accessdate=2016-03-11}}</ref>

=== 水运 ===
蚌埠港為[[淮河|淮河]]沿岸最重要的港口，“千里淮河第一大港”的蚌埠港是全国28个内河主要[[港口|港口]]之一，现已拥有蚌埠新港及怀远、五河、固镇等四个港区，码头39座，生产用泊位60个，码头岸线总长2324米，综合通过能力450万吨，最大靠泊船舶吨级1000吨。依托蚌埠市大中型企业和淮河中游地区腹地经济发展，蚌埠港近年来港口吞吐量、出口量均保持10%左右的增势。去年货物吞吐量324.22万吨，其中出口货物223.45万吨。今年1-6月，港口吞吐量已达230万吨。全市共有水路运输企业47家，拥有各类运输船舶3254艘、151万载重吨，其中千吨级以上船舶262艘。辖区水路运输主要货种为矿建材料、煤炭、粮食、化工原材料，出口货物大多发往苏、浙、沪等长三角地区。

[[蚌埠閘|蚌埠閘]]位于安徽省蚌埠市西郊许庄淮河干流上，為淮河中游大型水利枢纽工程，千里淮河第一闸。为发展灌溉、航运，结合公路交通、水力发电、城市供水等综合利用，于1962年建成，[[毛澤東|毛澤東]]曾題詞“一定要把淮河修好”。蚌埠閘由拦河节制闸、水力发电站厂房、船闸及附设的公路桥等组成。闸上利用河槽、湖泊等蓄水调节淮河水量。设计灌溉面积60万公顷。节制闸、溢洪道的设计泄洪流量各为1.014万与0.286万立方米／秒，校核泄洪流量各为1.14万与0.36万立方米／秒。 船闸可通过2×1000吨运输船队。

=== 公路运输 ===
蚌埠市的公路交通始于1922年，是南北主要咽喉之一，与铁路、航运交织成网，同时是南北通达的交通中心。如今公路四通八达，高标准的城市出入口道路已与[[国道|国道]]、[[省道|省道]]相联结<ref name="安徽省情概览"/>。蚌埠于1954年建立了长途汽车站，现拥有怀远县汽车运输公司、蚌埠长途汽车中心站、客运南站、客运北站等长途汽车站<ref>{{cite web|title=下属单位|url=http://www.bbqygs.com/bbqygs/index.htm|accessdate=2010-09-05|deadurl=yes|archiveurl=https://web.archive.org/web/20101114002318/http://www.bbqygs.com/bbqygs/index.htm|archivedate=2010年11月14日|df=}}</ref>，开行发往全国各地的长途班车。其中，开往[[淮南市|淮南市]]、[[宿州市|宿州市]]、[[合肥市|合肥市]]、[[滁州市|滁州市]]等方向的班车白天均保持30分钟发车间隔，开往[[怀远县|怀远县]]、[[固镇县|固镇县]]、[[五河县|五河县]]、[[凤阳县|凤阳县]]方向的班车白天实现公交化运行。

====[[共享单车|共享单车]]====
2017年7月，蚌埠首家共享单车企业——牛拜单车进驻，初期投放约3000辆。同月，该家企业因未向有关部门报批便擅自投放共享单车，导致数千辆单车被城管等相关行政执法人员暂扣。<ref>{{cite web|author=蚌埠广播电视网|title=牛拜单车“来去匆匆” 到底发生了什么？|url=http://www.ahbbtv.com/bencandy.php?fid=218&id=32724|accessdate=2017-07-18}}</ref>2017年12月，因企业经营不善，并曝出用户押金迟迟无法退还的丑闻，本地各家新闻媒体及政府机关曾多次就押金问题与牛拜单车蚌埠地区负责人约谈，但是并没有取得任何实质性进展，负责人仅表示押金可退换购物券。目前牛拜单车在蚌埠的处境十分窘迫。<ref>{{cite web|author=搜狐新闻|title=蚌埠牛拜单车押金退还难，咋办？报小美帮你问到了！|url=http://www.sohu.com/a/212149573_440737|accessdate=2017-12-22}}</ref><ref>{{cite web|author=安青网|title=蚌埠：市民反映牛拜单车退费难|url=http://www.ahyouth.com/news/20171225/1321748.shtml|accessdate=2017-12-25}}</ref>

2018年1月2日，可使用[[支付宝|支付宝]]直接扫码骑行的哈罗单车（俗称小蓝车）进驻蚌埠，投放数量较牛拜单车相比翻了两番，目前仍在加紧投放中。运行初期哈罗单车的运营范围仅覆盖主要建成区，至2018年3月运营范围已扩大至市区中环线内全部地区。<ref>{{cite web|author=[[新浪微博|新浪微博]]|title=哈罗单车蚌埠的微博|url=https://weibo.com/u/5136745060?refer_flag=1005055014_&is_hot=1|accessdate=2018-03-03}}</ref>目前，因小蓝车使用快捷便利、押金低、月卡用车价格低以及对青少年、中老年等特殊人群的优惠或免费政策，并且有效弥补蚌埠市长期无有效公共自行车供使用的空白，该企业的共享单车在蚌埠市运营情况良好，已成为多数市民短途出行的首选交通工具。<ref>{{cite web|author=百度|title=蚌埠街头出现小蓝车，又一家共享单车进驻蚌埠！|url=https://baijiahao.baidu.com/s?id=1588546945968983549&wfr=spider&for=pc|accessdate=2018-01-03}}</ref>

蚌埠市作为[[国家园林城市|国家园林城市]]，且是[[安徽省|安徽省]]的四大旅游中心城市之一，<ref>{{cite web|author=中华人民共和国住房和城乡建设部|title=住房城乡建设部关于2015年国家生态园林城市、园林城市、县城和城镇的通报|url=http://www.mohurd.gov.cn/wjfb/201601/t20160129_226498.html|publisher=住房和城乡建设部信息中心|accessdate=2016-01-15}}</ref><ref>{{Cite web |url=http://www.ahwang.cn/anhui/20170407/1622677.shtml|title=安徽将建立合肥 芜湖 黄山 蚌埠 四大旅游中心城市|author= |date= 2017-04-07|publisher=安徽网|language=  |accessdate= |quote=  }}</ref>截至2015年共有A级旅游景区28个，其中4A级旅游景区5个，3A级旅游景区7个，2A级旅游景区16个；国家级重点文物保护单位1个，省级文物保护单位11个，省级风景名胜区2个，省级自然保护区1个。2015年，蚌埠全市旅游总收入159.8亿元，比上年增长35%。接待国内游客2628.9万人次，接待入境游客5.1万人次。<ref>{{cite web |url=http://www.ahtjj.gov.cn/tjj/tjnj/2016/2016.html?ver=2 |title=《安徽统计年鉴—2016》 |publisher=安徽省统计局 |date=2016年 |accessdate=2017年2月1日 |deadurl=yes |archiveurl=https://web.archive.org/web/20170711121905/http://www.ahtjj.gov.cn/tjj/tjnj/2016/2016.html?ver=2 |archivedate=2017年7月11日 }}</ref><ref>{{cite web |url=http://my12340.net/article.aspx?ID=3390 |title=蚌埠市2013年国民经济和社会发展统计公报（安徽） |publisher= |date=2014年 |accessdate=2014年7月1日 }}{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>

===著名景观===
蚌埠拥有“蚌埠十景”一说，分别为十面埋伏、霸王别姬等典故的出处垓下古战场的垓下凭吊；包含[[白乳泉|白乳泉]]，[[卞和洞|卞和洞]]、[[启母石|启母石]]、[[禹王庙|禹王庙]]的荆涂揽胜；淮河文化广场以及音乐[[喷泉|喷泉]]的新城夜色；涵盖[[龙子湖|龙子湖]]和[[锥子山|锥子山]]风景区，包括[[汤和墓|汤和墓]]、珍珠桥、龙子湖大桥、烈士陵园、[[湖上升明月|湖上升明月]]在内的龙湖风光；城中河淮河两岸的风光淮畔观柳；市中心的[[津浦大塘公园|津浦大塘公园]]晨光；相传[[大禹|大禹]]会诸侯“禹会诸侯于涂山，执玉帛者万国”的禹会怀古；包括望淮塔、张公湖的[[张公山|张公山]]麓；[[沱湖|沱湖]]品蟹和蚌山夕照。<ref>{{cite web|title=蚌埠十景|url=http://ah.people.com.cn/GB/channel1608/1609/201008/02/202915.html|publisher=人民网安徽频道|accessdate=2010-09-04}}{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>

===城市公园和广场===
预计到2020年，蚌埠城区绿地率将达到32.9%，绿化覆盖率达到41.7%。<ref>{{cite web|title=2020年蚌埠绿化覆盖率达41.7% |url=http://ah.anhuinews.com/system/2010/09/02/003347129.shtml|publisher=中安在线 |accessdate=2010-09-04}}</ref> 城市主要公园有[[龙湖桥头公园|龙湖桥头公园]]、淮滨公园、[[张公山公园|张公山公园]]、津浦大塘公园、南山儿童公园等。主要广场有大禹文化广场、淮河文化广场等。
