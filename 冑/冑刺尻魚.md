'''冑刺尻魚'''，俗名'''火焰神仙'''，為[[輻鰭魚綱|輻鰭魚綱]][[鱸形目|鱸形目]][[鱸亞目|鱸亞目]][[蓋刺魚科|蓋刺魚科]]的一個[[種|種]]。

== 分布 ==
本魚分布於[[太平洋|太平洋]]區，包括[[菲律賓|菲律賓]]、[[澳洲|澳洲]]、[[所羅門群島|所羅門群島]]、[[馬紹爾群島|馬紹爾群島]]、[[密克羅尼西亞|密克羅尼西亞]]、[[新喀里多尼亞|新喀里多尼亞]]、[[庫克群島|庫克群島]]、[[斐濟|斐濟]]、[[東加|東加]]、[[萬納杜|萬納杜]]、[[法屬波里尼西亞|法屬波里尼西亞]]、[[關島|關島]]、[[夏威夷群島|夏威夷群島]]、[[吉里巴斯|吉里巴斯]]、[[強斯頓環礁|強斯頓環礁]]、[[薩摩亞群島|薩摩亞群島]]等海域。

== 深度 ==
分布在15至60公尺的水深。

== 特徵 ==
本魚魚體及背鰭、臀鰭主要為橘紅色，從鰓蓋後面至尾柄以及尾鰭為金黃色，上面有三至四條的垂直條紋。背鰭及臀鰭有紫色邊緣，並自逐漸變寬與鰭後部的黑色條紋相結合，胸鰭和腹鰭及鰓蓋為橘紅色。背鰭硬棘14枚、背鰭軟條16-18枚;臀鰭硬棘3枚、臀鰭軟條17-18枚。體長可達15公分。

== 生態 ==
本魚棲息於[[潟湖|潟湖]]或臨海的礁石區，以[[藻類|藻類]]為食，通常以一條雄魚配數條雌魚為一群體。

== 經濟利用 ==
為色彩鮮豔具高經濟價值的觀賞魚，容易飼養，性情溫和。
