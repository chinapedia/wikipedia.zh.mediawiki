'''歪腰郵筒'''，是[[臺灣|臺灣]][[臺北市|臺北市]][[中山區_(臺北市)|中山區]]兩個遭到[[颱風蘇迪羅_(2015年)|颱風蘇迪勒]]侵襲而掉落的招牌砸歪的[[郵箱|郵箱]]，由於其造型可愛而獲得民眾的高度關注。之後[[中華郵政|中華郵政]]響應民眾的要求而同意考慮將歪斜的郵箱予以保留，並且進一步發展成為臺北市另類景點<ref>{{Cite web|url=http://udn.com/news/story/9/1109842 |title=蘇迪勒發威 台北市郵筒變「阿帕契郵筒」 |author=黃義書 |date=2015年8月9日 |publisher=[[聯合新聞網|聯合新聞網]] |language=zh-hant |accessdate=2015年8月12日 |archiveurl=https://web.archive.org/web/20150819091803/http://udn.com/news/story/9/1109842 |archivedate=2015-08-19 |deadurl=no }}</ref>。在考量週遭民眾安全和交通狀況，決定將歪腰郵筒遷移至[[臺北郵局|臺北郵局]]。但是計畫遷移的舉動則受到許多民間人士反對，前[[中華民國文化部|中華民國文化部]]部長[[龍應台|龍應台]]更批評中華郵政不懂其價值。之後中華郵政則順應社會大眾意見而暫緩遷移工作，並且計畫之後與專家學者共同討論<ref>{{Cite web|url=http://udn.com/news/story/9/1116697 |title=龍應台批太蠢 歪腰郵筒不搬了 |author=徐子晴 |date=2015年8月12日 |publisher=[[聯合新聞網|聯合新聞網]] |language=zh-hant |accessdate=2015年8月12日 |archiveurl=https://web.archive.org/web/20150819091958/http://udn.com/news/story/9/1116697 |archivedate=2015-08-19 |deadurl=no }}</ref>。

== 經過 ==
=== 引起風潮 ===
[[File:歪腰郵筒_(2).jpg|thumb]]

2015年8月7日，受[[颱風蘇迪羅_(2015年)|颱風蘇迪勒]]侵襲[[臺灣|臺灣]]並引起強風影響，[[臺北市|臺北市]]市區許多路樹和招牌都因而傾倒或掉落。而在[[中山區_(臺北市)|中山區]]鄰近[[中華航空|中華航空]]舊總部「華航大樓」旁、南京龍江路口的兩座紅色與綠色郵箱，也被掉落的依定美時尚診所廣告招牌砸中、以及強風吹襲而使得郵箱腰桿朝同一方向彎曲；該診所廣告招牌在隔天便被吊車載走<ref>{{Cite web|url=http://www.cna.com.tw/news/ahel/201508105007-1.aspx |title=郵筒歪腰 台北郵局：招牌砸到加風吹 |author= |date=2015年8月10日 |publisher=[[中央通訊社|中央通訊社]] |language=zh-hant |accessdate=2015年8月12日 |archiveurl=https://web.archive.org/web/20150817140509/http://www2.cna.com.tw/news/ahel/201508105007-1.aspx |archivedate=2015-08-17 |deadurl=no }}</ref>。然而由於兩座被砸歪的郵箱造型可愛，許多路過民眾紛紛擺出俏皮姿勢與郵箱合照；而這些郵箱以及合影相片隨即在網際網路上廣泛流傳，進一步吸引更多[[批踢踢|批踢踢]]等網站網友的注意<ref>{{Cite web|url=http://www.cna.com.tw/news/aloc/201508080259-1.aspx |title=颱風蘇迪勒吹歪郵筒 模樣萌翻 |author=黃麗芸 |date=2015年8月8日 |publisher=[[中央通訊社|中央通訊社]] |language=zh-hant |accessdate=2015年8月12日 |archiveurl=https://web.archive.org/web/20150816144205/http://www.cna.com.tw/news/aloc/201508080259-1.aspx |archivedate=2015-08-16 |deadurl=no }}</ref>。很快地歪腰郵筒吸引越來越多路過民眾前往郵箱所在地圍觀，並且擺出不同動作拍照留念<ref>{{Cite web|url=http://udn.com/news/story/3/1108942 |title=郵筒被招牌砸歪 風災後北市新景點 |author=林俊良 |date=2015年8月8日 |publisher=[[聯合新聞網|聯合新聞網]] |language=zh-hant |accessdate=2015年8月12日 |archiveurl=https://web.archive.org/web/20150819092232/http://udn.com/news/story/3/1108942 |archivedate=2015-08-19 |deadurl=no }}</ref>。

到了8月9日下午，由於聚集的民眾不斷增加而開始有民眾排隊等候拍照，而後方的便利商店（7-11龍京門市）也因而獲利<ref>{{Cite web|url=http://udn.com/news/story/3/1109694 |title=郵筒「歪腰」夯到半夜也要照 民眾求情別撤 |author=陳皓嬿點和侯俐安 |date=2015年8月9日 |publisher=[[聯合新聞網|聯合新聞網]] |language=zh-hant |accessdate=2015年8月12日 |archiveurl=https://web.archive.org/web/20150819092409/http://udn.com/news/story/3/1109694 |archivedate=2015-08-19 |deadurl=no }}</ref>。很快地歪腰郵筒成為臺北市另類的熱門拍照與打卡新景點<ref name = "颱風蘇迪勒的紀念品">{{Cite web |url = http://www.cna.com.tw/news/firstnews/201508095010-1.aspx |title = 颱風蘇迪勒的紀念品 歪腰郵筒將保留 |author = 陳葦庭 |date = 2015年8月9日 |publisher = [[中央通訊社|中央通訊社]] |language = zh-hant |accessdate = 2015年8月12日 |quote =  }}</ref>，並且將大量遊客也將南京龍江路口道路嚴重堵塞而造成周邊交通混亂<ref name="歪腰郵筒13日搬家">{{Cite web |url = http://www.cna.com.tw/search/hydetailws.aspx?qid=201508110149&q=%E6%AD%AA%E8%85%B0%E9%83%B5%E7%AD%92 |title = 朝聖要快 歪腰郵筒13日搬家 |author = 陳葦庭 |date = 2015年8月11日 |publisher = [[中央通訊社|中央通訊社]] |language = zh-hant |accessdate = 2015年8月12日 |quote =  |deadurl = yes |archiveurl = https://web.archive.org/web/20160304124052/http://www.cna.com.tw/search/hydetailws.aspx?qid=201508110149&q=%E6%AD%AA%E8%85%B0%E9%83%B5%E7%AD%92 |archivedate = 2016年3月4日 }}</ref>。對此中華郵政則表示由於獲得許多民眾反映希望可以維持歪腰郵筒的樣貌，因此決定傾向順應民意而予以保留。但是中華郵政也提到在這之前將測試以確認雨水不會滲進郵箱，同時也會評估是否會影響行人安全，在認定沒有問題後就會讓這兩個郵箱保留現在的樣子<ref name = "颱風蘇迪勒的紀念品"/>。而在當天，郵局人員便前往現場投遞信件以測試傾斜的郵筒會不會有雨水滲漏的問題。

對此中華郵政郵務處處長[[李甘祥|李甘祥]]表示過去不曾發生過郵箱遭到砸歪的情況，而這次強烈颱風蘇迪勒在[[新北市|新北市]][[永和區|永和區]]也造成一支郵箱損壞，而已經被郵局人員收走。由於歪腰郵筒之後每天都吸引大批人潮前往拍照，[[中華郵政|中華郵政]]則在第一時間派遣兩位郵差查勘並在現場輪值以維持秩序；現場的輪值郵差也配合民眾拍照需求而擺弄姿勢，進而拍下一系列趣味照片<ref name = "歪腰郵筒13日搬家"/>。到了8月11日時，在龍江路上持續有等待與歪腰郵筒拍照的排隊人潮。同時中華郵政也要求站崗的郵差不能和現場民眾合照，以專心維持現場秩序<ref name = "小紅小綠要搬家了">{{Cite web |url = http://www.cna.com.tw/news/firstnews/201508110114-1.aspx |title = 歪腰郵筒人潮擠爆 小紅小綠要搬家了 |author = 陳葦庭 |date = 2015年8月11日 |publisher = [[中央通訊社|中央通訊社]] |language = zh-hant |accessdate = 2015年8月12日 |quote =  }}</ref>。同時也有公關公司安排[[OPEN小將|OPEN小將]]和LOCK小醬前往現場與民眾共同拍照，並且趁機宣傳電影《[[OPEN！_OPEN！|OPEN！ OPEN！]]》<ref name = "外國網友讚台灣人苦中作樂">{{Cite web |url = http://www.cna.com.tw/news/firstnews/201508110132-1.aspx |title = 歪腰郵筒爆紅 外國網友讚台灣人苦中作樂 |author =  |date = 2015年8月11日 |publisher = [[中央通訊社|中央通訊社]] |language = zh-hant |accessdate = 2015年8月12日 |quote =  }}</ref>。之後中華郵政則表示將會順勢推出文創商品，包括有歪腰郵筒公仔、個人化郵票等<ref>{{Cite web |url = http://udn.com/news/story/8471/1115959 |title = 台北「歪腰」郵筒紀念郵戳 今明限定 |author = 徐子晴 |date = 2015年8月12日 |publisher = [[聯合新聞網|聯合新聞網]] |language = zh-hant |accessdate = 2015年8月12日 |quote =  }}</ref>。

=== 計畫遷移 ===
[[File:Taipei_Beimen_Post_Office_20171018.jpg|thumb]]。]]

8月11日，中華郵政總經理[[陳憲著|陳憲著]]表示，由於歪腰郵筒所在的南京龍江路口的人行道過於狹小，大量遊客聚集已經造成附近交通壅塞。為避免影響交通與附近居民生活作息，也為了保護用路人安全，決定在8月13日將郵箱移至[[博愛路_(台北市)|博愛路]]上的[[臺北郵局|臺北郵局]]。對此，中華郵政認為，該處寬敞的人行道將可容納更多遊客，同時由於場地較為寬敞而不會派遣郵差輪值站崗。<ref name = "小紅小綠要搬家了"/>之後，中華郵政董事長[[翁文祺|翁文祺]]表示，中華郵政計畫在8月13日下午5時拆除歪腰郵筒、移往[[臺北郵局|臺北郵局]]入口處左側作為展示用途，並且計畫擺放立牌說明歪腰郵筒，而原地址則會放置新郵箱供民眾使用<ref>{{Cite web |url = http://www.cna.com.tw/topic/popular/5172-1/201508110297-1.aspx |title = 歪腰郵筒除役 中華郵政公布紀念戳章 |author = 陳葦庭 |date = 2015年8月11日 |publisher = [[中央通訊社|中央通訊社]] |language = zh-hant |accessdate = 2015年8月12日 |quote =  }}</ref>。

對此中華郵政考量到[[郵政博物館|郵政博物館]]在臺北郵局建築內部設有親子互動館，並且擁有更寬敞的人行道以提供遊客拍攝；而同時由於臺北郵局本身是古蹟，因此除了能夠使得遷移至該處的歪腰郵筒具有寓教於樂的體驗外，並且進而理解建築物的歷史與文化<ref>{{Cite web |url = http://udn.com/news/story/9/1116679 |title = 順應民意 歪腰郵筒不搬家了！ |author = 邱瓊平 |date = 2015年8月12日 |publisher = [[聯合新聞網|聯合新聞網]] |language = zh-hant |accessdate = 2015年8月12日 |quote =  }}</ref>。同時中華郵政也專案製作「微笑萌郵筒」紀念戳，並且讓8月12日下午5時到8月13日下午5時這段時間投入歪腰郵筒的郵件得以蓋上紀念戳章，藉此讓民眾得以共同紀念這段歷史<ref>{{Cite web |url = http://udn.com/news/story/6/1114341 |title = 微笑萌郵筒 8月13日除役遷址 |author = 楊文琪 |date = 2015年8月11日 |publisher = [[聯合新聞網|聯合新聞網]] |language = zh-hant |accessdate = 2015年8月12日 |quote =  }}</ref>。

8月12日上午，中華郵政主任秘書[[王淑敏|王淑敏]]表示，歪腰郵筒已經造成居民不方便，同時大批人潮在該處照相過於危險，因此將按照原定時間移到臺北郵局，並且表示移至臺北郵局後將配合該環境而有不同感受<ref>{{Cite web |url = http://www.cna.com.tw/news/ahel/201508125009-1.aspx |title = 歪腰郵筒 中華郵政：明天照搬 |author =  |date = 2015年8月12日 |publisher = [[中央通訊社|中央通訊社]] |language = zh-hant |accessdate = 2015年8月12日 |quote =  }}</ref>。翁文祺則表示，由於龍江路附近車輛眾多，中華郵政不可能每天派人前往現場維持秩序；在未能解決遊客影響安全和交通的情況下，只能夠按照計畫將歪腰郵筒搬到臺北郵局<ref>{{Cite web |url = http://udn.com/news/story/9/1116417 |title = 網友盼郵筒不要搬 中華郵政為難 |author = 邱瓊平 |date = 2015年8月12日 |publisher = [[聯合新聞網|聯合新聞網]] |language = zh-hant |accessdate = 2015年8月12日 |quote =  }}</ref><ref>{{Cite web |url = http://udn.com/news/story/8471/1116602 |title = 歪腰郵筒搬家 龍應台批愚蠢 |author = 邱瓊平 |date = 2015年8月12日 |publisher = [[聯合新聞網|聯合新聞網]] |language = zh-hant |accessdate = 2015年8月12日 |quote =  }}</ref>。

=== 取消遷移 ===
不過面對民間許多反對遷移之意見，中華郵政董事長翁文祺在考量該項決定已經引起社會許多討論後，在8月12日下午親自召開臨時會議。最後於臨時會議上，中華郵政決定順應民間意見而將爆紅的歪腰郵筒暫定原地保留以維護場域文物。中華郵政對此計畫先行保留2個禮拜，近期內則是聽取專家學者和各處意見對於遷移決定的看法，以便審慎評估在熱潮過後的遷移計畫<ref>{{Cite web |url = http://www.appledaily.com.tw/realtimenews/article/life/20150812/668281/ |title = 【更新】歪腰郵筒命運　中華郵政：明不搬了！ |author = 李姿慧 |date = 2015年8月12日 |publisher = [[蘋果日報_(台灣)|蘋果日報]] |language = zh-hant |accessdate = 2015年8月12日 |quote =  }}</ref>。但是翁文祺也公開表示由於歪腰郵筒已經成為臺北市的觀光資產，因此希望[[臺北市政府|臺北市政府]]能夠共同協助維護周邊的交通秩序與安全，而不能只藉由中華郵政的郵差來執行配合民眾拍照以及指揮交通的任務<ref>{{Cite web |url = http://www.cna.com.tw/news/firstnews/201508125011-1.aspx |title = 歪腰郵筒太熱門 順民意傾向不搬家 |author = 汪淑芬 |date = 2015年8月12日 |publisher = [[中央通訊社|中央通訊社]] |language = zh-hant |accessdate = 2015年8月12日 |quote =  }}</ref><ref>{{Cite web |url = http://www.cna.com.tw/news/ahel/201508120255-1.aspx |title = 順應民意 歪腰郵筒暫原地保留 |author = 汪淑芬 |date = 2015年8月12日 |publisher = [[中央通訊社|中央通訊社]] |language = zh-hant |accessdate = 2015年8月12日 |quote =  }}</ref>。

中華郵政同時還在記者會上表示由於歪腰郵筒更加傾斜，因此除了將會就地補強外，同時呼籲現場民眾不要推擠，以確保自身安全並且共同維護郵筒現狀。另外一方面，由臺北郵局專案製作的「微笑萌郵筒」宣傳戳活動仍然會對8月12日下午5時前投入郵筒的郵件蓋上印記<ref>{{Cite web |url = http://udn.com/news/story/9/1116978 |title = 中華郵政：歪歪郵筒明天不搬了 |author = 徐子晴 |date = 2015年8月12日 |publisher = [[聯合新聞網|聯合新聞網]] |language = zh-hant |accessdate = 2015年8月12日 |quote =  }}</ref>。

== 影響 ==
=== 廣受歡迎 ===
對此網友很快便將歪腰郵筒的姿勢和先前[[阿帕契打卡案|阿帕契打卡案]]做比較，並且戲稱這是「阿帕契郵筒」<ref>{{Cite web |url = http://udn.com/news/story/3/1108830 |title = 蘇迪勒強大 郵筒「歪腰」萌翻 |author = 陳柏亨 |date = 2015年8月8日 |publisher = [[聯合新聞網|聯合新聞網]] |language = zh-hant |accessdate = 2015年8月12日 |quote =  }}</ref>。而[[勞動部勞動力發展署中彰投分署|勞動部勞動力發展署中彰投分署]]青年職涯發展中心在新成立的創作工坊中，服務人員也使用[[Yahoo!奇摩|Yahoo!奇摩]]首頁的新聞相片、並花費2個小時而以[[3D列印|3D列印]]製作歪腰郵筒的模型，藉此宣傳自身3D建模設計和製作課程以及提供青年創業發想、產品製造設備與合作媒合平台的創客基地<ref>{{Cite web |url = http://www.cna.com.tw/news/aloc/201508120138-1.aspx |title = 歪腰郵筒爆紅 職訓班3D列印動手作 |author = 郝雪卿 |date = 2015年8月12日 |publisher = [[中央通訊社|中央通訊社]] |language = zh-hant |accessdate = 2015年8月12日 |quote =  }}</ref>。[[衛生福利部疾病管制署|衛生福利部疾病管制署]]在其宣導[[登革熱|登革熱]]疫情防治的海報上，也立刻添加了歪腰郵筒的設計<ref>{{Cite web |url = http://udn.com/news/story/9/1115589 |title = 跟流行 疾管署竟然也「歪腰」… |author = 魏忻忻 |date = 2015年8月11日 |publisher = [[聯合新聞網|聯合新聞網]] |language = zh-hant |accessdate = 2015年8月12日 |quote =  }}</ref>。同時也有網友分享「歪腰郵筒產生器」的網址，只要上傳圖片後便可以經過簡單步驟獲得與歪腰郵筒的合照<ref name = "歪腰郵筒產生器">{{Cite web |url = http://www.cna.com.tw/news/afe/201508120259-1.aspx |title = 歪腰郵筒產生器 想合照免排隊 |author = 江明晏 |date = 2015年8月12日 |publisher = [[中央通訊社|中央通訊社]] |language = zh-hant |accessdate = 2015年8月12日 |quote =  }}</ref>。

另外陸陸續續有商人開始生產相關的文具商品，並且在[[露天拍賣|露天拍賣]]上販售歪腰郵筒的紀念磁鐵、鑰匙圈、名片座等，另外也有插畫家跟進創作[[行動應用程式|行動應用程式]]的貼圖<ref name = "歪腰郵筒產生器"/>。臺北市[[六福皇宮|六福皇宮]]的[[一禮烘培|一禮烘培]]很快便推出歪腰郵筒造型杯子蛋糕提供給住房客戶以及民眾購買，分別使用法國進口的巧克力與大量核桃製成內餡，並且添加[[開心果|開心果]]以增加口感<ref>{{Cite web |url = http://udn.com/news/story/9/1114973 |title = 歪腰郵筒超夯！六福皇宮打造杯子蛋糕 |author = 史榮恩 |date = 2015年8月11日 |publisher = [[聯合新聞網|聯合新聞網]] |language = zh-hant |accessdate = 2015年8月12日 |quote =  }}</ref>。[[臺中市|臺中市]][[中友百貨公司|中友百貨公司]]邀請[[樂高|樂高]]達人[[黃彥智|黃彥智]]率領[[台灣創意積木發展協會|台灣創意積木發展協會]]舉辦積木玩具大展，在展出的79件作品中便包含積木版歪腰郵筒，並且隨即吸引許多民眾與之合照<ref>{{Cite web |url = http://www.cna.com.tw/news/aloc/201508120271-1.aspx |title = 積木玩具大展中市登場 歪腰郵筒正夯 |author = 郝雪卿 |date = 2015年8月12日 |publisher = [[中央通訊社|中央通訊社]] |language = zh-hant |accessdate = 2015年8月12日 |quote =  }}</ref><ref>{{Cite web |url = http://udn.com/news/story/6/1117124 |title = 中友百貨積木玩具展登場 迷你郵筒搶鏡 |author = 宋健生 |date = 2015年8月12日 |publisher = [[聯合新聞網|聯合新聞網]] |language = zh-hant |accessdate = 2015年8月12日 |quote =  }}</ref>。
就連LINE的貼圖也出現了歪腰郵筒<ref>{{Cite web |url = http://line.me/S/sticker/1179307 |title = 歪腰郵筒LINE貼圖 |date = 2015年8月20日 |publisher = [[聯合新聞網|聯合新聞網]] |language = zh-hant |accessdate = 2015年8月12日 |quote =  }}</ref>
許多民眾也紛紛在為歪腰郵筒設置的[[Facebook|Facebook]]專頁上分享照片，而這些照片之後則中國大陸廣泛流傳並且獲得數千名社群網路服務使用者熱烈討論。而中國女演員[[范冰冰|范冰冰]]和其男友[[李晨_(演員)|李晨]]前往臺灣進行雷射去斑和淨膚療程時，後者也趁機會在[[新浪微博|新浪微博]]上分享與歪腰郵筒的合照<ref>{{Cite web |url = http://www.cna.com.tw/news/firstnews/201508125003-1.aspx |title = 與范冰冰來台 李晨和歪腰郵筒合照 |author =  |date = 2015年8月12日 |publisher = [[中央通訊社|中央通訊社]] |language = zh-hant |accessdate = 2015年8月12日 |quote =  }}</ref><ref>{{Cite web |url = http://udn.com/news/story/8/1116108 |title = 范冰冰來台度假 李晨黏緊緊 |author = 陳昭妤 |date = 2015年8月12日 |publisher = [[聯合新聞網|聯合新聞網]] |language = zh-hant |accessdate = 2015年8月12日 |quote =  }}</ref><ref>{{Cite web |url = http://udn.com/news/story/8/1116547 |title = 范冰冰、李晨愛相隨 來台嘗美食求美肌 |author = 闕志儒 |date = 2015年8月12日 |publisher = [[聯合新聞網|聯合新聞網]] |language = zh-hant |accessdate = 2015年8月12日 |quote =  }}</ref>。另外經由外國新聞媒體報導歪腰郵筒一事後，許多海外網友也讚賞臺灣面對困境時展現的幽默感，[[英國廣播公司|英國廣播公司]]、《每日郵報》等新聞媒體都對於此事進行報導<ref name = "外國網友讚台灣人苦中作樂"/>。而美國《[[紐約郵報|紐約郵報]]》也在網站上報導說臺灣的一場風災反而讓被廣告牌壓歪的郵箱成為最熱門的旅遊景點，許多民眾因而排隊等待與歪腰郵筒共同照相，同時報導中還引用[[歐洲新聞圖片社|歐洲新聞圖片社]]6張各種民眾與郵差擺出逗趣模樣的照片<ref>{{Cite web |url = http://www.cna.com.tw/news/aopl/201508110189-1.aspx |title = 歪郵筒莫名爆紅 美媒：旅遊新景點 |author = 黃兆平 |date = 2015年8月11日 |publisher = [[中央通訊社|中央通訊社]] |language = zh-hant |accessdate = 2015年8月12日 |quote =  }}</ref>。

2016年6月23日，紅色郵筒被一名醉漢給踹直，隔日中華郵政派人將紅色郵筒恢復為歪腰的原樣<ref>{{Cite web |url = http://news.ltn.com.tw/index.php/news/life/breakingnews/1740786 |title = 沒想到吧！歪腰郵筒被踹直 郵局「整形」又歪了 |author = |date = 2016年6月24日 |publisher = [[自由時報|自由時報]] |language = zh-hant |accessdate = 2017年6月26日 |quote =  }}</ref>。

=== 搬遷問題 ===
對於歪腰郵筒移動位置一事，許多網友則紛紛持反對意見而傾向不予以搬遷<ref>{{Cite web |url = http://www.cna.com.tw/news/ahel/201508120218-1.aspx |title = 歪腰郵筒太萌了 順民意傾向不搬家 |author = 汪淑芬 |date = 2015年8月12日 |publisher = [[中央通訊社|中央通訊社]] |language = zh-hant |accessdate = 2015年8月12日 |quote =  }}</ref>。主要意見表示這將會導致郵箱沒人理會並且失去紀念價值，而希望中華郵政可以原地保留以彰顯紀念價值。不過亦有評論則戲稱臺灣民眾的盲從現象，進而表示舉辦全臺巡迴展以療癒各地民眾，而門票收入還能為中華郵政賺進觀光財<ref>{{Cite web |url = http://www.cna.com.tw/news/firstnews/201508110238-1.aspx |title = 歪腰郵筒 網友：搬走就失去味道了 |author = 陳葦庭 |date = 2015年8月11日 |publisher = [[中央通訊社|中央通訊社]] |language = zh-hant |accessdate = 2015年8月12日 |quote =  }}</ref>。對此前[[中華民國文化部|中華民國文化部]]部長[[龍應台|龍應台]]在Facebook上發表題為「不只是兩個鐵罐頭」的文章，內容中表示有關於搬遷的問題是重要的議題，然而中華郵政並完全沒有理解到歪腰郵筒的意義<ref>{{Cite web |url = http://udn.com/news/story/9/1116362 |title = 歪腰郵筒搬家 龍應台：真正愚蠢 |author = 邱瓊平 |date = 2015年8月12日 |publisher = [[聯合新聞網|聯合新聞網]] |language = zh-hant |accessdate = 2015年8月12日 |quote =  }}</ref>，並且直接批評說：「把小紅小綠搬到別處去，才是真正愚蠢。<ref>{{Cite web |url = http://news.cts.com.tw/cts/life/201508/201508121647654.html#.VcwjG4iqpBc |title = 龍應台批愚蠢! 歪腰郵筒暫不搬了! |author = 黃益源 |date = 2015年8月12日 |publisher = [[中華電視公司|中華電視公司]] |language = zh-hant |accessdate = 2015年8月12日 |quote =  }}</ref>」

對於龍應台希望中華郵政能夠讓歪腰郵筒留在原地的說法，臺北市文化資產審議委員會委員[[李乾朗|李乾朗]]表示由於歪腰郵筒並不具備文化資產的身分，因此可以對於郵箱進行遷移。而關於為了保障歪腰郵筒得以繼續留在原地而爭取文化資產法定身分的作法，李乾朗表示其前提是必須有人提報此案並且啟動文化資產評鑑，否則[[臺北市政府文化局|臺北市政府文化局]]和[[中華民國文化部|中華民國文化部]]都不便發表意見。不過對於歪腰郵筒是否具有列為文化資產的價值，李乾朗則表示就年代、唯一性、特殊性等客觀指標衡量而認為歪腰郵筒目前機會很低<ref>{{Cite web |url = http://udn.com/news/story/9/1116856 |title = 歪腰郵筒不具文資身分 學者：可搬遷 |author = 周美惠 |date = 2015年8月12日 |publisher = [[聯合新聞網|聯合新聞網]] |language = zh-hant |accessdate = 2015年8月12日 |quote =  }}</ref>。

=== 藝術性 ===
前中華民國文化部部長龍應台在《不只是兩個鐵罐頭》一文中，認為兩個歪腰郵筒是集體的「會心一笑」，並且包含著許多情感記憶在這之中。對此她提到：

{{Cquote 3|兩個歪腰郵筒之所以迷倒了我們，是因為它意外地凝聚了我們的集體記憶：所有臺灣人對於颱風的感受、那一夜的狂風暴雨的每一個腦海中的鏡頭、南京和龍江路口的街景和氛圍、清早發現郵筒歪頭時的驚訝片刻和會心的幽默，甚至包括了其後群眾的不可思議的傻像……把小紅小綠搬到別處去，才是真正愚蠢。它使一個活生生的記憶，一個美好的難得的會心一笑變成兩個壞掉的郵筒、沒意義的鐵罐頭。|[[龍應台|龍應台]]<ref>{{Cite web |url = http://www.cna.com.tw/news/firstnews/201508125008-1.aspx |title = 歪腰郵筒要搬家 龍應台：才是真正愚蠢 |author =  |date = 2015年8月12日 |publisher = [[中央通訊社|中央通訊社]] |language = zh-hant |accessdate = 2015年8月12日 |quote =  }}</ref>}}

=== 其他郵箱 ===
在臺北市因為受到颱風影響的歪腰郵筒廣泛受到注目後，臺灣各地也陸陸續續也其他特殊的郵箱獲得民眾關注。其中在颱風襲擊臺灣前，[[臺南市|臺南市]][[永康區|永康區]]蔦松一街與蔦松二街路口便有一個便被貨車倒車時撞歪的綠色郵箱，很快該座設立近50年的單座郵箱照片也在網際網路上傳開，並且吸引許多民眾和記者前往拍攝<ref>{{Cite web |url = http://www.cna.com.tw/news/aloc/201508100155-1.aspx |title = 歪腰郵筒爆紅 台南也有一個 |author = 楊思瑞 |date = 2015年8月12日 |publisher = [[中央通訊社|中央通訊社]] |language = zh-hant |accessdate = 2015年8月12日 |quote =  }}</ref>。不過之由於這支郵箱每個月收件數平均11件左右而長年效益不高，又經常在路口被車輛撞到，因此在被颱風傾倒的樹木壓到後主管單位永康郵局便針對安全性進行內部檢討。在評估該處距離永康郵局和鹽行郵局不遠後，永康郵局認定不會造成民眾不便而決定將該郵箱移除<ref>{{Cite web |url = http://www.cna.com.tw/news/aloc/201508110193-1.aspx |title = 安全性考量 台南歪腰郵筒拆了 |author = 楊思瑞 |date = 2015年8月11日 |publisher = [[中央通訊社|中央通訊社]] |language = zh-hant |accessdate = 2015年8月12日 |quote =  }}</ref>。

另外[[花蓮縣|花蓮縣]][[玉里鎮|玉里鎮]]也有一座據傳10年前就在[[玉里協天宮|玉里協天宮]]附近、位於中華路路邊傾斜的郵箱，該郵箱的照片也在網際網路上傳開而引發關注<ref>{{Cite web |url = http://www.cna.com.tw/news/asoc/201508100214-1.aspx |title = 歪腰郵筒非台北專利 花蓮也有一個 |author = 李先鳳 |date = 2015年8月11日 |publisher = [[中央通訊社|中央通訊社]] |language = zh-hant |accessdate = 2015年8月12日 |quote =  }}</ref>。不過花蓮郵局在8月11日時便雇請工人以地基整修方式將郵箱弄正，對此花蓮郵局局長[[江進達|江進達]]澄清該郵箱歪腰的時間僅有2年，原因是民眾開車倒車時不小心撞到郵箱而成<ref>{{Cite web |url = http://news.ltn.com.tw/news/life/breakingnews/1409051 |title = 玉里歪腰郵筒「扶正」 遊客拍照撲空！ |author = 花孟璟 |date = 2015年8月11日 |publisher = 《[[自由時報|自由時報]]》 |language = zh-hant |accessdate = 2015年8月12日 |quote =  }}</ref>。另外一方面，[[花蓮港|花蓮港]]海灣處一座重達12噸的蛇紋石郵筒則是由於經過多次颱風襲擊而未受到影響，同樣獲得新聞媒體的注目<ref>{{Cite web |url = http://www.cna.com.tw/news/ahel/201508120169-1.aspx |title = 花蓮港蛇紋石郵筒 傳遞旅人足跡與故事 |author = 李先鳳 |date = 2015年8月12日 |publisher = [[中央通訊社|中央通訊社]] |language = zh-hant |accessdate = 2015年8月12日 |quote =  }}</ref>。
