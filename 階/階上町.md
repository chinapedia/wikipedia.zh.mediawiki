'''階上町'''（{{jpn|j=階上町|hg=はしかみちょう|rm=Hashikami chō|f=y}}）是青森縣最東南端，與岩手縣接壤的[[町|町]]。

== 概要 ==
階上町是位於青森縣最東南邊的自治體。西面及北面鄰近西八戶市，南面鄰近岩手縣洋野町，東面則面向太平洋。近幾年，由於八戶市積極開發作為[[臥城|臥城]]的地區，人口正在增加。有著良好的天然資源，沿岸的三陸海岸地區提供豐富的漁獲，而內陸地匿則是繁盛的奶酪畜牧業。該城市有著能看到太平洋及縣南地區的階上岳（740 m），為周邊地區提供一處休憩景點。特產包括豐富的海產、草莓煮、鹽海膽，海藻麵條等。

== 歷史 ==
* 724年～728年（神龜元年至神龜5年） - 位於現在的階上町寺下地區興建應物寺。詳細情況不明，但以現在殘留的五重塔（於大正時代倒塌）推斷，估計收納[[行基|行基]]的觀音像。
* 1444年（文安元年） - 按琳阿孝寬大法師的意願，於現在的階上町道佛地區興建西光寺。而這亦成現在階上町核心地區的基礎。
* 1664年（寬文4年） - 成為八戶藩的領地。
* 1807年（文化4年） - 為了保衛太平洋沿岸，八戶藩為階上町小舟渡海岸進行海灣鞏固設置。
* 1871年（明治4年） - 廢藩置縣而編入八戶縣。該年9月改稱為青森縣。
* 1873年（明治6年） - 大小區制實施，屬於第9大區4小區。
* 1878年（明治11年） - 大小區制廢止，施行郡制而編入三戶郡。
* 1889年（明治22年）4月1日 - 田代村、晴山澤村、平內村、鳥屋部村、金山澤村、赤保內村、道佛村、角柄折村合併為'''階上村'''。
* 1980年（昭和55年）5月1日 - 施行町制，改為階上町。

== 地理 ==
* 階上町的大部分地區為平地。內陸的南部地區聳立著作為城市符號的階上岳。城市的東側連著太平洋，擁有國內少見的[[溺灣|溺灣]]式海岸線。

== 氣候 ==
* 階上町的平均氣溫為9.7度、年降雨量為861 mm。
* 因為階上町沒有設置氣象廳的氣象觀測所，因此要參考八戶市的數據。本町的氣候為太平洋方面氣候，不過，夏天會受到偏東風影響而變得寒冷，也有從海吹來的冷風令到夏天有較寒冷的時候。雖然位於東北，但是降雪量很少及乾燥，冬天時的晴天及日照時間也較多。

== 名勝、古蹟、觀光景點、祭典、活動 ==
=== 古蹟 ===
* 寺下觀音堂 - 於724年行基創建。成為奧州糠部三十三觀音之一。
* 道佛館遺跡
* 茨島的七葉樹（赤保內字茨島） - 本縣的自然保護植物、樹齡約600～800年。
* 水晶花（保內字赤保內） - 本縣的自然保護植物、樹齡不詳。不過，在日本中沒有記錄有水晶花的大樹。
=== 觀光景點 ===
* 階上岳 - 杜鵑的簇生地，每年自5月下旬至6月上旬期間盛開。
* 八戶郊區俱樂部
* 小舟渡海岸 - 青森縣最早看到日出的地方（山間部除外）。
