'''亦庄天主堂'''，全名'''天主教北京教区亦庄圣若望堂'''，位于[[北京市|北京市]][[大兴区|大兴区]][[亦庄地区|亦庄地区]]荣华中路7号国荣国际B座1号楼4层，隶属[[天主教北京教区|天主教北京教区]]。

==简介==
2010年，郭文武神父从[[深圳|深圳]]调回北京，在北京城南建立了三个祈祷所西红门、金星、亦庄，统称石景山堂区。<ref name=hushui/>2010年9月11日，亦庄祈祷所第一台感恩弥撒圣祭在亦庄贵园东里小区一民居内举行，这标志着亦庄祈祷所成立。2011年8月，亦庄祈祷所迁至博客雅居写字楼。因教友人数增多，2014年7月亦庄祈祷所迁至荣华中路7号国荣国际B座1号楼4层（马赛克楼）。2014年7月5日正式启用，举办第一台弥撒。经郭文武神父倡导，北京城南的石景山、西红门、金星、亦庄四所祈祷所同步开展为期一年（2014年7月1日至2015年6月30日）的“[[圣经|圣经]]家庭传递”活动。2014年12月27日，北京教区[[李山|李山]]主教主祭、教区神父共祭，为亦庄天主堂举办开堂弥撒，并祝圣了新祭台、新读经台、若望像，位于国荣国际的新教堂奉圣若望宗徒为主保，取名天主教北京教区亦庄圣若望堂。<ref>{{Cite web |url =  http://www.tianguangbao.org/2014.7.15/18.htm|title =亦庄祈祷所喜迁新居  |author =  |date =2014-07-15  |publisher = 天光 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://www.chinacatholic.org/News/index/id/28676.html |title = 北京亦庄祈祷所喜迁新居新动向 |author =  杨若瑟|date = 2014-07-09 |publisher = 信德网 |language =  |accessdate =  |quote =  }}</ref>2016年，亦庄脱离石景山堂区成为独立的堂区，北京教区房产办公室副主任卞寅神父兼任亦庄堂区主任司铎。<ref name=jian/>亦庄祈祷所自成立后，注重教理学习，并积极开展社会公益活动，组织慰问老人和病患。<ref>{{Cite web |url =  http://mt.sohu.com/20160919/n468719601.shtml|title =  北京：北京教区亦庄教友中秋佳节给老人和病人送去|author =  |date = 2016-09-19 |publisher =搜狐  |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://www.chinacatholic.org/news/23832 |title = 北京：亦庄祈祷所教友以避静、学习“开年” |author = 张玛利亚 |date =2013-03-01  |publisher =信德网  |language =  |accessdate =  |quote =  }}</ref>2016年时，亦庄堂区约有300多位教友。<ref>{{Cite web |url = http://www.chinacatholic.org/News/show/id/35176.html |title = 北京：亦庄圣若望天主堂隆重庆祝圣神降临瞻礼 |author =  |date = 2016-05-17 |publisher = 信德网 |language =  |accessdate =  |quote =  }}</ref>
