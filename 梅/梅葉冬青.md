'''梅葉冬青''' （[[學名|學名]]：''{{lang|la|Ilex asprella}} ''）<ref>[http://www.hkflora.com/qef/site_peak/plant/show_plant.php?plantnum=12341 香港植物及植被]香港中文大學</ref>，別稱崗梅（廣東埔）、苦梅根（廣東埔）、燈稱花、燈花樹（台灣植物志）、點稱星、點秤星、假秤星（[[廣東|廣東]][[東莞|東莞]]）、秤星樹、燈秤仔、燈秤花、萬點金、烏雞骨、燈絲仔、山梅根、假青梅（中國高等植物圖鑑）、秤星樹、秤星木、天星木、汀秤仔（[[香港|香港]]）、相星根（[[廣西|廣西]][[梧州|梧州]]）、百解茶及檀樓星等<ref name = " cpami ">[http://w3.cpami.gov.tw/khmp/plant-handbook2/html/page103.htm 燈稱花]臺灣內政部營運署</ref><ref name = " 陽明山 ">[http://www.ymsnp.gov.tw/web/webpage.aspx?f=data_file/plant93/plant93_c3-122.htm 陽明山的代表性植物]陽明山國家公園</ref><ref name=" 國立台灣師範大學 ">[http://biogeo.geo.ntnu.edu.tw/service/biogeodb/plants/tempplq.asp?status=hplant&cluster=num&numland=TW_N014&key_words=162-001 162-001 生物地理與自然保育研究室] {{webarchive|url=https://web.archive.org/web/20040928100806/http://biogeo.geo.ntnu.edu.tw/service/biogeodb/plants/tempplq.asp?status=hplant&cluster=num&numland=TW_N014&key_words=162-001 |date=2004-09-28 }}國立台灣師範大學地理學系</ref><ref name = " 崗梅 ">[http://www.zysj.com.cn/zhongyaocai/yaocai_g/gangmei.html 崗梅]中醫世家</ref>，為[[冬青科|冬青科]][[冬青屬|冬青屬]]植物<ref name = " 樹木谷 ">[http://www.hktree.com/tree/Ilex%20asprella.htm 梅葉冬青]樹木谷</ref>。[[種加詞|種加詞]] ''asprella'' 意為「具粗糙鱗片的」<ref name = " 樹木谷 "/>。本種為冬青科內少數會落葉的品種之一<ref name = " 陽明山 "/><ref name=" 十二生肖藥用植物區 ">[http://www.herb.nat.gov.tw/b2b_cpinfo01.asp?id=966 十二生肖藥用植物區] {{webarchive|url=https://archive.is/20121223014409/http://www.herb.nat.gov.tw/b2b_cpinfo01.asp?id=966 |date=2012-12-23 }}台北市內雙溪森林藥用植物園</ref>
花期1－3月，果期3－6月<ref name=" 台北植物園 ">[http://tpbg.tfri.gov.tw/Plants/plants_info.asp?rid=837 植物資料庫]{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}台北植物園</ref>。

「崗梅」名稱的由來源於梅葉冬青生長在山邊但習性像梅花<ref name = " 樹木谷 "/>；「燈秤花」的名稱由來源於本種植物小枝光滑呈褐色貌似秤桿，皮孔像似秤點而得名<ref>[http://www.envi.org.tw/twtrees/book4/102.htm 燈秤花] {{webarchive|url=https://archive.is/20120803181919/http://www.envi.org.tw/twtrees/book4/102.htm |date=2012-08-03 }}台灣樹木解說</ref>。

==分佈及生境==
分佈於[[中國|中國]]東南部、[[菲律賓|菲律賓]][[呂宋|呂宋]]、[[琉球|琉球]]等地。多生長於山坡草叢、路旁及次生林綠野徑旁，[[海拔|海拔]]400[[米|米]]至1,000米的地區<ref>[http://e-info.org.tw/topic/plant/2004/pl04020401.htm 植物簡介-我的賞花日記]環境資源中心</ref><ref name=" 浙江植物誌 ">[http://www.cvh.org.cn/difangzhi/zhejiang/page/4/006.pdf  浙江植物誌]{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}第6頁</ref>。性喜高溫，全日照或半日照均可，以[[腐植質|腐植質]][[土壤|土壤]]生長最佳，多以[[種子|種子]]繁殖<ref>[http://goodall.a-team.com.tw/product_info.php?products_id=50&Twesid=1f575d439a0c 燈稱花]{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}綠姆指</ref>。<ref name=" 國立臺灣博物館 ">[http://www.ntm.gov.tw/tw/reseach/4aa1.aspx?id=21 學術與典藏] {{Wayback|url=http://www.ntm.gov.tw/tw/reseach/4aa1.aspx?id=21 |date=20081005063657 }}國立臺灣博物館</ref><ref name = " 香港野花網 ">[http://www.producegreen.org.hk/hkwildflower/100flower/flower_42.htm  梅葉冬青]香港野花網</ref><ref name = " 國立台灣師範大學 "/>

==形態特徵==
梅葉冬青是一種落葉灌木，可高達3[[米|米]]。有長枝及短枝，長枝纖細，小枝光滑無毛，[[綠色|綠色]]具明顯的[[白色|白色]]皮孔，干後[[褐色|褐色]]。[[葉|葉]]卵狀橢圓形或卵形，互生，膜質或紙質，表面綠、深綠至黃綠色，背面淺綠色，背部有細腺點，基部寬楔形，先端漸尖成尾狀，具細鋸齒狀葉緣，網脈不明顯，羽狀側脈6－8對，中脈上部稍凹下，表面脈上披直毛，長約3－7厘米，寬約1.5－3厘米；葉柄長約3－8毫米。[[花|花]]為繖形花序，[[白色|白色]]，[雄花2－3朵，簇生或單生葉腋，4或5數，花萼圓形或裂片闊[[三角形|三角形]]，無毛，直徑約2.5－3毫米，[[花冠|花冠]]直徑約6毫米，[[雄蕊|雄蕊]]著生於[[花冠|花冠]]上，與[[花瓣|花瓣]]互生，花瓣近[[圓形|圓形]]，基部結合；雌花單生葉腋，無毛，4或6數，[[花萼|花萼]]4－5裂，直徑約2.5－3毫米，花瓣4－5枚，基部結合，花梗長約2－2.5厘米，結果時可長達3厘米，[[子房|子房]]球狀卵形，[[直徑|直徑]]約1.5毫米，花柱明顯，柱頭呈盤狀。果為[[核果|核果]]，球形或橢圓形，外有縱溝，成熟時[[黑色|黑色]]，內果皮石質，基部具宿存之萼，分核4－6粒，直徑約5－6毫米，果梗長約2－3厘米。<ref name = " 香港野花網 "/><ref name = " 國立台灣師範大學 "/><ref name = " 台北植物園 "/><ref name = " 十二生肖藥用植物區 "/><ref name = " 浙江植物誌 "/>

==醫藥用途==
梅葉冬青[[根|根]]、[[葉|葉]]入[[藥|藥]]；根部性味苦、甘，性寒<ref>[http://libproject.hkbu.edu.hk/was40/detail?channelid=35734&searchword=herb_id=D00345 崗梅根]香港浸會大學</ref>，歸肺、肝、大腸經，無毒，能清熱解毒，活血，生津，可治感冒、眩暈 、頭痛、咽喉腫痛 、百日咳、淋病及跌打損傷等。葉性味苦、甘，性寒<ref>[http://libproject.hkbu.edu.hk/was40/detail?channelid=35734&searchword=herb_id=D00337 崗梅葉]香港浸會大學</ref>，歸心、肺、肝經，清熱解毒，可治感冒及跌打損傷等。葉含熊果酸，對心絞痛及冠心病有治療作用<ref name = " 浙江植物誌 "/>。為涼茶廿四味之原料之一。<ref name = " 香港野花網 "/><ref name = " 十二生肖藥用植物區 "/><ref name = " 崗梅 "/><ref>[http://www.zysj.com.cn/zhongyaocai/yaocai_g/gangmeigen.html 崗梅根]中醫世家</ref><ref>[http://www.zysj.com.cn/zhongyaocai/yaocai_g/gangmeiye.html 崗梅葉]中醫世家</ref><ref>[http://www.cn939.com/tcm-article-read-35629.html 根類—崗梅]中國中醫網</ref><ref>[http://www.zhong-yao.net/zycx/G/200802/2264.html 崗梅根] {{webarchive|url=https://web.archive.org/web/20101206183351/http://www.zhong-yao.net/zycx/G/200802/2264.html |date=2010-12-06 }}中醫中藥網</ref><ref>[http://www.tcmlib.com/zy/html24/showdetail-323331362ce5b297e6a285e6a0b92c7a79.html 中醫在線知識庫] {{webarchive|url=https://archive.is/20130428045018/http://www.tcmlib.com/zy/html24/showdetail-323331362ce5b297e6a285e6a0b92c7a79.html |date=2013-04-28 }}意古中醫</ref>
