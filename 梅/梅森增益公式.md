'''梅森增益公式'''（MGF）是寻找线性[[信号流图|信号流图]]（SFG）[[传递函数|传递函数]]的方法。该公式是{{le|塞缪尔·杰斐逊·梅森|Samuel Jefferson Mason}}推导出的，<ref>{{cite journal|last=Mason|first=Samuel J. |date=July 1956|title=Feedback Theory - Further Properties of Signal Flow Graphs|journal=Proceedings of the IRE|pages=920–926|doi=10.1109/jrproc.1956.275147|volume=44}}</ref> 也是用他的名字命名的。MGF是用代数方法标记每个信号，将信号依赖于其他信号的方式写成方程，然后求解多元方程组得出输出信号与输入信号的关系，以求传递函数的方法。MGF提供了由信号流图一步一步获得传递函数的方法。通常，MGF可以通过检查信号流图确定。该方法可以很容易地处理多变量、多回路包括内循环回路的信号流图。MGF经常出现在{{le|控制系统|control system}}和[[数字滤波器|数字滤波器]]的内容中，因为控制系统和数字滤波器常會用信号流图表示。

== 公式 ==
增益公式如下：

: <math>G = \frac{y_\text{out}}{y_\text{in}} = \frac{ \sum_{k=1}^N  {G_k \Delta _k} }{ \Delta\ }  </math>

<!-- extra blank line between two lines of "displayed" [[TeX|TeX]] for easier legibility -->

: <math>\Delta = 1 - \sum L_i + \sum L_iL_j- \sum L_iL_jL_k + \cdots + (-1)^m \sum \cdots +\cdots </math>

其中：
*Δ = 图的特征式
*''y''<sub>in</sub> = 输入节点变量
*''y''<sub>out</sub> = 输出节点变量
*''G'' = ''y''<sub>in</sub> 和 ''y''<sub>out</sub> 间的总增益
*''N'' = ''y''<sub>in</sub> 与 ''y''<sub>out</sub> 间的前向通道的总数
*''G''<sub>''k''</sub> = ''y''<sub>in</sub> 与 ''y''<sub>out</sub> 间的第 ''k'' 个前向通道的路径增益
*''L''<sub>''i''</sub> = 系统中每个闭环回路增益
*''L''<sub>''i''</sub>''L''<sub>''j''</sub> = 任意2个互不接触回路（没有公共节点）的回路增益的乘积
*''L''<sub>''i''</sub>''L''<sub>''j''</sub>''L''<sub>''k''</sub> = 任何三个互不接触回路的回路增益的乘积
*Δ<sub>''k''</sub> = Δ 对于第 ''k'' 个前向通道的余子式，即将接触第 ''k'' 个前向通道的回路去除。

===定义<ref name="Kuo, 2nd ed, p 59">{{cite book|last=Kuo|first= Benjamin C. |year= 1967 |title=Automatic Control Systems |edition=2nd |publisher= Prentice-Hall |isbn= |doi= |pp=59-60}}</ref>===
*通道：一组沿标识方向的连接输入输出的连续分支
*前向通道：从一个输入节点到一个输出节点的通道，每个节点接触不超过一次。
*回路：起始和终止在相同节点，并且没有节点接触两次的通道。
*路径增益：该通道的所有分支的增益之积。
*回路增益：回路中所有分支的增益之积。

=== 步骤 ===
要使用这种方法，
# 列出前向通道和它们的增益，将他们记为 ''G''<sub>''k''</sub>。
# 列出所有回路和它们的增益，将他们记为 ''L''<sub>''i''</sub>（对第 ''i'' 个回路）。列出所有互不接触的回路和他们的增益（''L''<sub>''i''</sub>''L''<sub>''j''</sub>）。列出所有三个互不接触的回路（''L''<sub>''i''</sub>''L''<sub>''j''</sub>''L''<sub>''k''</sub>），然后四个，以至更多，直到找不出更多的互不接触的组合。
# 计算特征式 Δ 和余子式 Δ<sub>''k''</sub>。
# 运用公式。

==例子==

===二端口电路===

[[Image:Circuit_with_two_port_and_equivalent_signal_flow_graph.png|left]]

需要求从 '''V<sub>in</sub>''' 到 '''V<sub>2</sub>''' 的传递函数。

只有一个前向通路：
:* '''V<sub>in</sub>''' 到 '''V<sub>1</sub>''' 到 '''I<sub>2</sub>'''  到  '''V<sub>2</sub>''' 增益为<math> G_1 = -y_{21} R_L \, </math>

有三个回路：
:* '''V<sub>1</sub>'''  到 '''I<sub>1</sub>''' 到 '''V<sub>1</sub>''' 增益为 <math> L_1 = -R_{in} y_{11} \, </math>
:* '''V<sub>2</sub>'''  到 '''I<sub>2</sub>''' 到 '''V<sub>2</sub>''' 增益为 <math> L_2 = -R_L y_{22} \, </math>
:* '''V<sub>1</sub>'''  到 '''I<sub>2</sub>''' 到 '''V<sub>2</sub>'''  到 '''I<sub>1</sub>'''  到 '''V<sub>1</sub>''' 增益为 <math> L_3 = y_{21} R_L y_{12} R_{in} \, </math>

:<math> \Delta = 1 - ( L_1 + L_2 + L_3 ) + ( L_1 L_2 ) \, </math>  注意：'''L<sub>1</sub>''' 和 '''L<sub>2</sub>''' 相互不接触，而 '''L<sub>3</sub>''' 接触其他两个回路。

:<math> \Delta_1 = 1 \, </math>  注意：前向通路接触了所有回路，所以剩下的就是 '''1'''。

:<math> G = \frac { G_1 \Delta_1 } { \Delta }   =  \frac { -y_{21} R_L } {1 + R_{in}y_{11} + R_L y_{22} - y_{21} R_L y_{12} R_{in} + R_{in}y_{11} R_L y_{22} }    \, </math>

===数字IIR双二阶滤波器===

[[Image:Signal_flow_graph_for_a_digital_IIR_Biquad.png|right]]

经常将[[数字滤波器|数字滤波器]]化成信号流图。

: 有两个回路
:*<math> L_1 = -a_1 Z^{-1} \, </math>
:*<math> L_2 = -a_2 Z^{-2} \, </math>

:<math> \Delta = 1 - ( L_1 + L_2 ) \, </math> 注意，这两个回路接触，所以没有他们的乘积项。

:有三个前向通道
:*<math> G_0 = b_0  \, </math>
:*<math> G_1 = b_1 Z^{-1} \, </math>
:*<math> G_2 = b_2 Z^{-2} \, </math>

: 所有的前向通道都接触，所以 <math> \Delta_0 = \Delta_1 = \Delta_2 = 1  \, </math>

: <math> G = \frac { G_0 \Delta_0 +G_1 \Delta_1  + G_2 \Delta_2  } {\Delta} \, </math>

: <math> G = \frac { b_0 + b_1 Z^{-1} + b_2 Z^{-2} } {1 +a_1 Z^{-1} + a_2 Z^{-2} } \, </math>

此信號流中有六個迴路，分別是：
:*<math> L_0 = -  \frac {\beta} {sM} \, </math>

:*<math> L_1 = \frac{-( R_M + R_S)} {sL_M} \, </math>

:*<math> L_2 = \, \frac {-G_M K_M} {s^2 L_M M} </math>

:*<math> L_3 = \frac {-K_C R_S} {sL_M} \, </math>

:*<math> L_4 = \frac {-K_V K_C K_M G_T} {s^2 L_M M} \, </math>

:*<math> L_5 = \frac {-K_P K_V K_C K_M } {s^3 L_M M} \, </math>

:*<math> g_0 = \frac {-K_P K_V K_C K_M } {s^3 L_M M} \, </math>

== 等效矩陣-{zh-cn:形式;zh-tw:型式}- ==
梅森增益公式可以用簡單的矩陣公式表示。假設<math>\mathbf{T}</math>是信號流圖的轉換矩陣，其中<math>t_{nm} = \left[\mathbf{T}\right]_{nm}</math>是由節點''m''到'n''所有分支transmittance的和，則節點''m''到'n''的增益為<math>u_{nm} = \left[\mathbf{U}\right]_{nm}</math>，其中

: <math> \mathbf{U} = \left ( \mathbf{I} - \mathbf{T} \right ) ^ {-1} </math>,

而<math>\mathbf{I}</math>為單位矩陣。

梅森增益公式也可以用在推導有巢狀網路（其內迴路在外迴路以內）的離散網路z域传递函数。若是離散網路可以以信號流圖表示，則梅森增益公式可以計算網路的z域传递函数H(z)。

==複雜度及計算應用==
由於有向圖的路徑數會隨節點會以階乘的形式增加，因此梅森增益公式的複雜度會以階乘的形式往上成長。先考慮完整的有 <math>n</math>個頂點的有向圖，每二個頂點都有一個邊，從節點<math>y_{in}</math>到節點<math>y_{out}</math>的路徑會是其他<math>(n-2)!</math>個節點的<math>(n-2)!</math>個排列，因此在一般的情形下，[[高斯消去法|高斯消去法]]會比此方式有效率很多。

不過梅森增益公式將互相連接的系統的轉換函數用代數及組合的方處理，允許一些代数系统理论中較廣義的定義及計算方式。高斯消去法中會有許多的倒數計算，而梅森增益公式將其整理為單一的[[逆元素|逆元素]]，其通式為
::<math>\frac{p}{1-q},</math>

如上所述，<math>q</math>是迴路乘積的總和，每一個都是环论中的[[理想_(环论)|理想]]<!--(for example, the strictly causal operators).-->。上式的分數形成[[有理函數|有理函數]]的[[子環|子環]] <math>R(1+\langle L_i\rangle)^{-1}</math>。上述的觀察可以延伸到非交換性的情形<ref>{{cite journal|authors=Pliam, J.O. and Lee,E.B.|year=1995|title=On the global properties of interconnected systems|journal=IEEE Trans. Circuits and Syst. I|pages=1013–1017|doi=10.1109/81.481196|volume=42}}</ref>，不過梅森增益公式會被[[里格尔公式|里格尔公式]]（Riegle's rule）所取代。
