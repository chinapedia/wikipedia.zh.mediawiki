'''梅山街道'''是[[中华人民共和国|中国]][[浙江省|浙江省]][[宁波市|宁波市]][[北仑区|北仑区]]下辖的一个[[街道_(行政区划)|街道]]。位于北仑区东南部的[[梅山岛|梅山岛]]，是一个[[浙江省海岛行政区列表|海岛街道]]，面积32.8平方千米。2004年人口为1.51万人，下辖13个[[村|村]]。

== 地理位置 ==
梅山街道是一个海岛街道，北隔梅山港与[[春晓街道|春晓街道]]相望，南隔[[佛渡水道|佛渡水道]]与[[舟山市|舟山市]][[佛渡岛|佛渡岛]]、[[六横岛|六横岛]]相望。

== 下辖 ==
梅山街道下辖梅东、碑塔、梅中、梅港、梅西5个村。

== 交通 ==
2010年通车的[[梅山大桥|梅山大桥]]是连接梅山岛与大陆的陆路通道。连接[[六横岛|六横岛]]的[[六横疏港公路|六横疏港公路]]已纳入规划。

== 保税港 ==
2008年2月24日，经[[中华人民共和国国务院|国务院]]批准，[[宁波梅山保税港区|宁波梅山保税港区]]正式成立。这是中国大陆第五个保税港区。
