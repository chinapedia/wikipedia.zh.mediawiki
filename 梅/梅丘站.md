'''梅丘站'''（{{jpn|j=梅ヶ丘駅|hg=うめがおかえき|rm=Umegaoka eki|f=y}}）是一個位於[[東京都|東京都]][[世田谷區|世田谷區]][[梅丘|梅丘]]一丁目，屬於[[小田急電鐵|小田急電鐵]][[小田原線|小田原線]]的[[鐵路車站|鐵路車站]]。[[車站編號|車站編號]]是'''OH 09'''。

== 歷史 ==
* 1934年（[[昭和|昭和]]9年）4月1日：車站啟用，最初是定位為{{Link-ja|東京山手急行電鐵|東京山手急行電鉄}}的接駁站，上行線和下行線均使用同一月台<ref name="1seiki">小田急線 沿線の1世紀（世界文化社 2009年5月）p.36</ref>，即[[島式月台|島式月台]]<ref name="1seiki"/>。
* 1962年（昭和37年）：由於東京山手急行電鐵最終未能建成，車站改建為[[側式月台|側式月台]]<ref name="1seiki"/>。
* 1998年（[[平成|平成]]10年）4月16日：改建為高架車站以及複線化<ref>小田急線、新駅舎のデザイン決定　東京・世田谷の豪徳寺など5駅 読売新聞 1998年4月17日 東京朝刊29頁</ref>。
* 2004年（平成18年）12月11日：新設的區間準急開始停靠此站。
* 2016年（平成28年）3月26日：區間準急廢除，本站僅停靠各站停車。
* 2018年（平成30年）3月17日：月台有效長度可停靠10輛編組列車，成為[[東京地下鐵千代田線|東京地下鐵千代田線]]直通列車（各站停車）停靠站。

== 車站結構 ==
本站是[[側式月台|側式月台]]2面4線[[高架車站|高架車站]]。2018年3月3日，[[代代木上原站|代代木上原站]]至[[登戶站|登戶站]]區間完成[[複複線|複複線]]化。

2018年3月3日，東北澤站至本站複複線化<ref>{{Cite web |url=http://www.odakyu.jp/program/info/data.info/8732_1784221_.pdf |title=存档副本 |access-date=2018-11-05 |archive-url=https://web.archive.org/web/20180130144117/http://www.odakyu.jp/program/info/data.info/8732_1784221_.pdf |archive-date=2018-01-30 |dead-url=yes }}</ref>。本站月台在本站至[[喜多見站|喜多見站]]區間複複線化完成時，同步改建成可停靠10輛編組列車。3月2日以前，[[世田谷代田站|世田谷代田站]]方向最後兩輛車廂受到急行線與緩行線轉轍器與信號機的影響無法停靠。因此本站改在[[豪德寺站|豪德寺站]]側設置狹窄的臨時月台，實質月台有效長度為8輛編組。轉轍器與臨時月台在3月3日至3月17日之間撤除。

2018年3月2日以前，本站是複複線區間的東端，2018年3月3日複複線區間延長至代代木上原，待避站從本站改成[[東北澤站|東北澤站]]。

== 車站周邊 ==
{{See also|梅丘|代田 (世田谷區)|松原 (世田谷區)|赤堤}}
本站是{{link-ja|羽根木公園|羽根木公園}}的最近站，梅花花期有許多遊客前來賞花。另外也是[[國士館大學|國士館大學]]世田谷校區與{{link-ja|國士館中學校、高等學校|国士舘中学校・高等学校}}的小田急最近站，有許多學生使用<ref>{{cite web|url=http://itot.jp/13112/721 |title=梅ヶ丘駅 |publisher=あいとっと |accessdate=2014-03-02 }}</ref>。

多部電視劇曾在北口取景。北口知名的[[欅樹|欅樹]]因站前圓環整備工程，2005年2月28日有部分遭到砍除<ref>{{cite web|url=http://www.ne.jp/asahi/setagaya-kugikai/mutouha-shimin/2007.html |title=梅ヶ丘駅前けやき伐採問題ーー景観破壊と駅周辺事業へのアクション |author=木下泰之 |accessdate=2014-03-02 }}</ref>。
