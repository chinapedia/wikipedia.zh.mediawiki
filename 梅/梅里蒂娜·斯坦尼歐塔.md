'''梅里蒂娜·迪米特里耶芙娜·斯坦尼歐塔'''（{{lang-be|'''Меліціна Дзмітрыеўна Станюта'''}}；{{lang-ru|Мелитина Станюта}}；{{lang-en|Melitina Staniouta}}，{{bd|1993年|11月15日|||Melitina Staniouta}}) 是著名[[白俄羅斯|白俄羅斯]][[韻律體操|韻律體操]]運動員。 她是三次(2010年，2013年和2015年)[[世界韻律體操錦標賽|世界韻律體操錦標賽]]個人全能季軍，[[2015年歐洲運動會體操比賽|2015年歐洲運動會]]韻律體操個人全能季軍以及2014年歐洲錦標賽個人全能亞軍。

==生平==
===早年===

斯坦尼歐塔出生於[[白俄羅斯|白俄羅斯]][[明斯克|明斯克]]，她在三歲的時候被啟蒙教練Svetlana Burdzevitskaya在街上發掘並邀請,於是斯坦尼歐塔開始了她的[[韻律體操|韻律體操]]生涯，Burdzevitskaya稱讚斯坦尼歐塔是個極有才華的女孩，說她總能立刻明白教練的要求，並且有優秀的身體條件。她於十一歲的時候開始跟隨前白俄羅斯韻律體操國家隊成員{{link-en|Larissa Loukianenko | Larissa Loukianenko }}訓練，並於其後進入了國家代表隊。<ref>{{Cite web |url = http://sportpanorama.by/online/other-online/15304/ |title = Melitina Staniouta: to live in a world of competition and struggle |date = 2012-10-27 |publisher = sportpanorama.by |deadurl = yes |archiveurl = https://web.archive.org/web/20160304085311/http://sportpanorama.by/online/other-online/15304/ |archivedate = 2016-03-04 }}</ref>

在2008年，她開始贏得一些少年組級別比賽的獎牌，當中最重要的包括在[[都靈|都靈]]歐洲錦標賽獲得的繩、帶、和團體銀牌。

===2009-2012年===
即使是在成年組的第一年，但作為備受矚目的新秀，斯坦尼歐塔已參與了不少世界盃跟大獎賽分站賽，她也不負期待獲得了很不錯的成績。在五月的歐洲錦標賽她進入了球、繩兩個單項的決賽，她還在之后於[[台灣|台灣]][[高雄|高雄]]舉行的[[世界運動會|世界運動會]]進入了全部單項的決賽，並贏得了球操的銅牌。在九月，斯坦尼歐塔前往[[日本|日本]][[三重縣|三重縣]]參與[[世界韻律體操錦標賽|世界韻律體操錦標賽]]，在那裹她贏得了團體銀牌和圈操銅牌。她在其後的世界盃決賽也獲得了一銀(繩)四銅(個人全能、圈、球、帶)的好成績。

在2010年，斯坦尼歐塔的表現依然亮眼。她在歐洲錦標賽時因出現了失誤所以個人全能僅排名第六，但她在世界盃決賽贏得了一銀(球)三銅(個人全能、繩、帶)。其後她更在[[莫斯科|莫斯科]][[世界韻律體操錦標賽|世界錦標賽中]]贏得了團體銀牌，個人全能季軍和繩操銅牌，然而當時鮮少人知道的是，她的腳在比賽第四天就已經受傷了<ref name=" kpru ">{{Cite web |url = http://www.kp.ru/daily/25786.5/2769916/ |title = Melitina Stanniouta"Getting the Bronze medal in World Championship(Moscow 2010), I took a broken bone ..." |date = 2011-11-14|publisher = kp.ru }}</ref>。之后斯坦尼歐塔帶傷赴日，和隊友[[莉寶·查卡辛娜|莉寶·查卡辛娜]]及Elena Bolotina代表明斯克迪納摩(Dinamo)參加世界藝術體操俱樂部錦標賽(Aeon Cup)，贏得了團體銀牌和個人全能銅牌。最后她在十二月接受了手術，這年間除了腳傷外，她的手指，腿部和鎖骨也骨折過。

在手術後斯坦尼歐塔花了好些時間來休養，終于直到3月才能正式開始訓練，並在四月下旬才開始比賽，展開她遲來的2011年賽季。<ref>{{Cite web |url = http://by.tribuna.com/others/gymnastics/156339229.html |title = Melitina Staniouta: "At first no one knew that I broke my leg" |date = 2011-10-14 |publisher = by.tribuna.com }}</ref>
一個月後，在[[明斯克|明斯克]]舉行的歐洲錦標賽中斯坦尼歐塔仍未能回復狀態，所以除了團體銀牌外她未能收獲其他個人獎牌。
九月，在[[蒙彼利埃|蒙彼利埃]]舉行的[[世界韻律體操錦標賽|世界韻律體操錦標賽]]是本賽季最高榮譽的比賽，尤其這次世錦賽是[[2012年夏季奧林匹克運動會|2012年夏季奧林匹克運動會]]的初次資格賽，更顯重要。在這次比賽中，斯坦尼歐塔雖只能和隊友[[莉寶·查卡辛娜|莉寶·查卡辛娜]]、{{link-en|Aliaksandra Narkevich |Aliaksandra Narkevich}}、Hanna Rabtsava獲得了團體銀牌，但仍以110.250總分在個人全能排名第六，順利和[[莉寶·查卡辛娜|莉寶·查卡辛娜]]一起替白俄羅斯拿下兩個奧運名額。

斯坦尼歐塔在經歷了上賽季的低迷後，在2012年賽季的狀態大好。她在於[[下諾夫哥羅德|下諾夫哥羅德]]舉行的2012年歐洲錦標賽中個人全能僅排名第六。在世界盃決賽中她個人全能排名第五，在單項決賽中雖然球僅排第七，但圈、棒、帶均獲得銅牌。
到了[[2012年倫敦奧運會|2012年倫敦奧運會]]，斯坦尼歐塔雖是首次登場，但作為一名富有大賽經驗並擁有高強實力的選手，她仍被視為有力的獎牌競爭者<ref>{{Cite web |url = http://www.ueg.org/en/The-London-Olympic-Games--Rhythmic-Gymnastics-2012-08-10-1050|title = The London Olympic Games - Rhythmic Gymnastics|date = 2012-08-10|publisher = ueg.org }}</ref>。最終她以108.675的總分排名第十二位，無緣決賽。

===2013年===
倫敦奧運過後，新賽季開始，剛開始實施的新二十分制無損斯坦尼歐塔的競爭力。她首先在年初時參加LA Lights比賽並贏得了全能銅牌，其後在[[塔爾圖|塔爾圖]]世界盃贏得個人全能和圈操銀牌，以及球、棒、帶三個單項的金牌，取得了個很好的開始。在之後三月的[[霍隆|霍隆]]大獎賽贏得圈的金牌以及棒和帶的銀牌。在[[蒂艾|蒂艾]]大獎賽斯坦尼歐塔獲得了圈、棒、帶的銅牌。除此之外她還在[[里斯本|里斯本]]世界盃收獲了帶操銀牌和棒操銅牌。

四月時，斯坦尼歐塔在[[布加勒斯特|布加勒斯特]]世界盃取得了突破──贏得首面的世界盃個人全能金牌，還贏得球、棒、帶三個單項的金牌<ref>{{cite web|url=http://www.fig-gymnastics.com/site/figNews/view?id=797|publisher= FIG|title=Staniouta triumphs in Bucharest|date = 2013-04-23}}</ref>。接著，她於[[佩薩羅|佩薩羅]]世界盃拿到第二面的世界盃個人全能金牌，並且獲得了球、棒、帶的金牌和圈操銀牌<ref>{{cite web|url=http://www.fig-gymnastics.com/site/figNews/view?id=850|publisher= FIG|title=Staniouta and Italy prevail in Pesaro|date = 2013-04-29}}</ref>，成為整場賽事的大贏家。在[[科爾貝埃索納|科爾貝埃索納]]世界盃中，取得兩銀(全能、球) 一銅(棒)的好成績。一星期後，她參加了在家鄉[[明斯克|明斯克]]舉辦的世界盃賽事，拿到全能銅牌和棒、帶兩個單項的金牌。在六月初於[[維也納|維也納]]舉行的歐洲錦標賽中，和隊友{{link-en|Arina Charopa | Arina Charopa }}及{{link-en|Katsiaryna Halkina | Katsiaryna Halkina }}拿到了團體銅牌，其後再獲得圈、棒共三面銅牌。七月中，她前往[[俄羅斯|俄羅斯]][[喀山|喀山]]參加[[世界大學運動會|世界大學運動會]]，拿到了一面銀牌。在[[卡利|卡利]][[世界運動會|世界運動會]]中她贏得球、棒的金牌和圈的銀牌。因為世運會不設個人全能項目，而帶操單項因場地問題而取消，在三個項目中贏得了兩金一銀的斯坦尼歐塔被gymmedia稱為「是次賽事中最成功的體操運動員」<ref>{{Cite web |url = http://www.gymmedia.com/rhythmic-gymnastics/World-Games-Melitina-Staniouta-most-successfully-Gymnast-Cali |title = Melitina Staniouta: most-successfully Gymnast Cali |date = 2013-07-28 |publisher = gymmedia }}</ref>。

八月中旬，她於在[[聖彼得堡|聖彼得堡]]舉行的世界盃決賽贏得全能銀牌和圈、棒的銅牌。在於[[基輔|基輔]]舉行的[[世界韻律體操錦標賽|世界錦標賽中]]中，斯坦尼歐塔首先成功晉級所有單項的決賽並獲得了圈和帶的銅牌(這次世錦賽中不設團體·team的賽事)，最後她僅次於俄羅斯的[[雅娜·庫德亞特賽娃|雅娜·庫德亞特賽娃]]和烏克蘭運動員{{link-en|Ganna Rizatdinova|Ganna Rizatdinova}}，以72.166的總分排名第三，拿到她第二面的世錦賽個人全能銅牌。

世錦賽過後她前往[[布爾諾|布爾諾]]參加大獎賽，在那兒她獲得了個人全能、圈、球、帶的銅牌和棒的銀牌。然後她和Katsiaryna Halkina及少年組代表{{link-en|Hanna Bazhko|Hanna Bazhko}}一起前往[[日本|日本]][[東京|東京]]，代表Dinamo參加世界藝術體操俱樂部錦標賽(Aeon Cup)，她最後贏得了團體和全能銀牌。

===2014年===
斯坦尼歐塔在14年初先前往[[紐約|紐約]]參加LA Lights比賽並贏得了全能金牌。在接下來的[[莫斯科|莫斯科]]大獎賽上，她全能排名第七和獲得球的銅牌，也在[[蒂艾|蒂艾]]大獎賽全能排名第六和贏得帶的金牌。三月下旬 ，她前往[[德國|德國]]參加了[[斯圖加特|斯圖加特]]世界盃比賽，最後她全能排名第四，贏得了球的銀牌和圈、棒、帶的銅牌。她亦在[[里斯本|里斯本]]世界盃贏得了全能銀牌、圈的金牌和球的銀牌，並於[[佩薩羅|佩薩羅]]世界盃上獲得棒的銅牌。其後她參加了於[[希臘|希臘]]舉行的Kalamata Cup贏得了全能金牌。而在5月22至24日，她參加了[[塔什干|塔什干]]世界盃並在全能項目排名第五以及贏得圈、棒、帶共3枚銀牌和球的銅牌。在稍後舉行的[[明斯克|明斯克]]世界盃上，斯坦尼歐塔成功贏得全能銀牌帶的金牌和球、棒的銅牌。

在6月10至15日，她在[[巴庫|巴庫]]歐洲錦標賽奪得了個人全能銀牌。距離上次有白俄羅斯選手在歐錦賽上拿到個人全能獎牌已經是2000年的事了(當時是{{link-en|Yulia Raskina| Yulia Raskina}}贏得全能銀牌)，所以斯坦尼歐塔是唯一一位在這14年間重回歐洲錦標賽個人全能頒獎台的白俄羅斯運動員。8月8至10日,她在[[索菲亞|索菲亞]]世界盃全能排名第四，並拿到圈、帶的銀牌。在9月5至7日，她前往[[喀山|喀山]]參與世界盃總決賽，她以高分奪得了全能季軍，棒、帶的銀牌，球的銅牌並在圈操單項決賽的中排名第五。9月22至28日，她和隊友{{link-en|Arina Charopa | Arina Charopa }}、{{link-en|Katsiaryna Halkina | Katsiaryna Halkina }}代表[[白俄羅斯|白俄羅斯]]，前往[[伊茲密爾|伊茲密爾]]參加2014年世界藝術體操錦標賽。她們首先以合共136.073分贏得團體賽銀牌，之後斯坦尼歐塔成功晉級球、棒、帶單項決賽，並奪得了球的銅牌，可惜之後她因失誤導致個人全能僅排名第七<ref>{{Cite web|url = http://www.rhythmicgymnasticsresults.com/worlds/2014/izmir.html|title = Izmir (TUR)2014 Sep 22-28 Results|publisher = rhythmicgymnasticsresults.com|deadurl = yes|archiveurl = https://web.archive.org/web/20140918140325/http://www.rhythmicgymnasticsresults.com/worlds/2014/izmir.html|archivedate = 2014-09-18}}</ref>。世錦賽完結後她與Katsiaryna Halkina和少年組代表{{link-en|Mariya Trubach|Mariya Trubach}}代表Dinamo到[[東京|東京]]參加世界藝術體操俱樂部錦標賽(Aeon Cup)，最後她和隊友贏得了團體銀牌和全能銅牌<ref>{{Cite web|url = http://www.rhythmicgymnasticsresults.com/2014/as/jpn/aeoncup.html|title = Tokyo (JPN) 2014 Oct 17-18|publisher = rhythmicgymnasticsresults.com|deadurl = yes|archiveurl = https://web.archive.org/web/20150924090923/http://www.rhythmicgymnasticsresults.com/2014/as/jpn/aeoncup.html|archivedate = 2015-09-24}}</ref>。

===2015年===
斯坦尼歐塔先在[[紐約|紐約]]的LA Lights比賽贏得了全能銀牌。二月下旬時參加了[[莫斯科|莫斯科]]大獎賽，在那兒她收獲了一面個人全能銅牌、棒的金牌，圈、球的銀牌並在帶的單項決賽上排名第四。之後她前往[[拉脫維亞|拉脫維亞]][[里加|里加]]參與Baltic Hoop比賽，成功贏得了全能金牌、圈操金牌、棒操銀牌和帶的銅牌。在4月3至5日時，在[[布加勒斯特|布加勒斯特]]世界盃奪得個人全能、圈、球、帶共四面銅牌。在接下來的[[佩薩羅|佩薩羅]]世界盃她全能排名第五，拿到一銀(球)二銅(圈、帶)。她還去了[[波蘭|波蘭]][[斯塞新|斯塞新]]參與國際賽Spring Competition，成功以72.2高分奪得全能金牌。在5月1至3日時於[[明斯克|明斯克]]舉行的歐洲錦標賽中，她先和{{link-en|Katsiaryna Halkina | Katsiaryna Halkina }}一起拿下團體銀牌，其後還拿到了圈、帶的銀牌和球、棒的銅牌。在[[柏林|柏林]]大獎賽中她拿下全能銀牌、帶操金牌和圈、球、棒的銅牌。

6月15至21日時她前往[[巴庫|巴庫]]參加[[2015年歐洲運動會|第一屆歐洲運動會]]，成功奪得了個人全能銅牌、圈操銀牌，球、棒的銅牌和在帶操決賽上排名第四。斯坦尼歐塔還參加了[[光州|光州]][[2015年夏季世界大學運動會|世界大學運動會]]，在這場賽事中她拿下全能銅牌、帶操金牌、棒操銀牌、圈操銅牌和在球操決賽上排名第四。她在八月上旬舉行的[[布達佩斯|布達佩斯]]世界盃上獲得個人全能、圈和球操的銅牌，並在帶的單項決賽上排名第六。在[[索菲亞|索菲亞]]世界盃中全能排名第四，拿到球的銀牌、圈的銅牌，以及在帶操和棒操單項決賽上分別排名第四和第五。稍後她於在[[喀山|喀山]]舉行的世界盃總決賽全能排名第四，拿到球和帶操的銅牌，以及在圈操和棒操單項決賽上分別排名第五和第八。在9日9至13日時，斯坦尼歐塔於[[斯圖加特|斯圖加特]]參加世界錦標賽, 她首先成功和隊友Katsiaryna Halkina、{{link-en|Arina Charopa | Arina Charopa}}、{{link-en|Hanna Bazhko|Hanna Bazhko}}奪得團體賽銀牌，以個人全能預賽第六的成績晉級個人全能決賽和所有單項決賽。她先是獲得了球的銅牌並在其他三個單項中排名第四，最後更在全能決賽中以72.132總分贏得了個人全能季軍。在10月2至4日時，她與Arina Charopa和少年組代表Yulia Isachanka代表Dinamo到[[東京|東京]]參加世界藝術體操俱樂部錦標賽(Aeon Cup)，最後她全能排名第四並和隊友贏得了團體銅牌。

== 個人生活 ==
斯坦尼歐塔的曾祖母是曾被授予蘇聯人民藝術家稱號的白俄羅斯演員，Stefania Staniouta(Стефания Станюта) <ref name=" kpru "/>。
她的祖父是白俄羅斯作家，Alexander Staniouta(Александр Станюта) <ref>{{Cite web |url = http://citydog.by/post/melita-stanyuta/ |title = Мелита Станюта: «Если чего-то добьюсь в спорте – меня обязательно снимут "в Бонакве"» |date = 2013-09-25 |publisher = city.dog }}</ref>。

她的父母之所以替她取了「梅里蒂娜」(Melitina) 這個名字是因為他們看了一部叫«Любовные письма с подтекстом»的電影，而其女主角的名字就是Melitina。這個希臘名字的意思是"親愛的"。<ref>{{Cite web |url = http://www.kp.ru/daily/24361/546851/ |title = Мелитина СТАНЮТА: «Звездой себя не считаю и нос не задираю» |date = 2009-09-16 |publisher = kp.ru }}</ref>

在不用訓練的時候她喜歡去劇院和影院，也喜歡看書<ref name= minpraud.by>{{Cite web |url = http://minpraud.by/articles.php?id=4995&current_date=2010-10-23 |title = Мелитина Станюта: Художественная гимнастика — моя первая любовь. И вторая тоже |date = 2010-10-13 |publisher = minpraud.by }}</ref>，最喜愛的書是[[亂世佳人|亂世佳人]]。她喜歡吃海鮮，喜歡日本酒<ref name=" zn.sb.by "/>，也像普通女生一樣熱衷購物 <ref name=" kpru "/>。

斯坦尼歐塔會英語和法語，也略懂意大利語<ref>{{Cite web |url = http://www.ctv.by/новости/белорусская-гимнастка-17-летняя-мелита-станюта-карьера-важнее-семьи |title = Белорусская гимнастка 17-летняя Мелита Станюта: Карьера важнее семьи |date = 2011-02-03 |publisher = ctv.by }}</ref>。

她現就讀於[[白俄羅斯國立大學|白俄羅斯國立大學]]<ref name=" zn.sb.by ">{{Cite web |url = http://zn.sb.by/persona-3/article/melita-stanyuta-poka-ne-zakonchu-so-sportom-zamuzh-ne-poydu-.html |title = Melita Stanyuta: "Still not finished with the sport, not going to get married" » |date = 2012-03-27 |publisher = zn.sb.by }}</ref>。

她曾出演Bonaqua的礦泉水廣告。
