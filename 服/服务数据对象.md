'''服务数据对象'''（SDO）是一种以统一的方式访问异构数据的技术。2004年，服务数据对象的技术规范最初由[[BEA|BEA]]和[[IBM|IBM]]合作开发，并通过了Java社群过程的批准。规范的第二版作为服务组件体系结构的关键部分在2005年11月份推出。

服务数据对象规范现在由下面公司开发：
* [[BEA|BEA]]
* [[IBM|IBM]]
* [[Rogue_Wave_(company)|Rogue Wave (company)]]
* [[Oracle_Corporation|Oracle]]
* [[SAP公司|SAP]]
* [[Siebel|Siebel]]
* [[Sybase|Sybase]]
* [[XCalia|XCalia]]
* [[Software_AG|Software AG]]
* [[Sun_Microsystems|Sun Microsystems]]

现在有以下的SDO产品
* Rogue Wave Software (HydraSDO)
* CodeFutures Software (FireStorm/SDO)
* Xcalia Software (XIC)
* BEA (ADSP)
* IBM (Virtual XML Garden)
* IBM (WebSphere 6.1 Feature Pack)
