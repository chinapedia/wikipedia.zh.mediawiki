'''吴化文'''（{{bd|1904年||1962年|4月3日|catIdx=Wu,吴}}），字'''绍周'''，[[山东省|山东省]][[莱州府|莱州府]][[掖县|掖县]]（在今[[莱州市|莱州市]]境）人，[[中华民国|中华民国]]军事将领、[[中华人民共和国|中华人民共和国]]政治人物<ref name="fan">{{Cite book|author =范銀飛 |title = 民国高級将領列传 5 「吴化文」|year = 1999|publisher = 解放軍出版社|isbn = 7-5065-1494-X}}</ref><ref name=xu>{{Cite book|author = 徐友春主編|title = 民国人物大辞典 増訂版|year = 2007|publisher = 河北人民出版社|isbn = 978-7-202-03014-1}}</ref>。

== 生平 ==
=== 国民軍、国民革命軍的活動 ===
[[File:Wu_Huawen2.jpg|thumb]]
1920年（[[民国紀元|民国]]9年），吴化文加入[[馮玉祥|馮玉祥]]的军队，以後逐渐昇职。1923年（民国12年），经馮玉祥推薦，吴化文加入北京高級教導团。1925年（民国14年），入北京陸軍大学正則班第7期，1927年（民国16年）7月毕业。此后担任[[西北軍|西北軍]][[洛陽市|洛陽]]初級軍校教育長兼特務团团長，後来隶属[[韓復榘|韓復榘]]率领的[[国民革命軍|国民革命軍]]第20師。[[韓復榘|韓復榘]]脱离馮玉祥、投靠[[蒋介石|蒋介石]]後，吴化文继续跟随[[韓復榘|韓復榘]]。后来吴化文担任[[國民革命軍第三路軍|國民革命軍第三路軍]]手枪旅第二团团长。<ref name=fan/><ref name=xu/><ref name=xingtai>[http://news.ifeng.com/history/zhongguojindaishi/detail_2011_04/22/5913919_0.shtml 乔春景口述、乔季芬 马棋琪整理，邢台文史资料第三辑，1987年]</ref>

1930年（民国19年）[[中原大战|中原大战]]後，韓復榘軍扩充为第3路軍。1934年，吴化文接替[[雷台平|雷台平]]担任手槍旅旅長，并且兼任[[济南市|济南]]警備司令。[[抗日战争|抗日战争]]爆发後的1938年（民国27年）1月，韓復榘被蒋介石以敵前逃亡之罪处决，手槍旅改编为第28旅，吴化文继续担任该旅旅長。接替韓復榘担任[[山东省政府主席|山东省政府主席]]的是[[沈鸿烈|沈鸿烈]]，吴化文成为沈鸿烈的手下。1939年（民国27年）1月，第28旅扩编为新編第4師，吴化文任師長。翌年8月，吴化文任[[山東省政府|山東省政府]]委員兼山東保安第1師師長。在同日軍作战中，吴化文起初与[[八路軍|八路軍]]关系良好，後因国共不和而不得不结束合作。<ref name=fan/><ref name=xu/><ref name=xingtai/>

=== 从曲线救国到参加人民解放軍 ===
[[File:Wu_Huawen3.jpg|thumb]]后，到[[南京|南京]]拜会[[汪精卫|汪精卫]]。左为吴化文，右为汪精卫]]
[[File:Wu_Huawen4.jpg|thumb]]后，到[[南京|南京]]，受到汪精卫政权要员[[褚民谊|褚民谊]]（右二）、[[刘郁芬|刘郁芬]]（右一）、[[林柏生|林柏生]]（右三）等人的欢迎。]]
1943年（民国32年）1月18日，吴化文表示参加[[汪精卫政权|汪精卫政权]]。吴化文的投降是奉[[国民政府|国民政府]]農林部部長[[沈鴻烈|沈鴻烈]]（吴化文的老上级）的指示，并获[[蒋介石|蒋介石]]默认（名曰「[[曲线救国|曲线救国]]」）<ref name="fan"/>{{rp|286}}。吴化文部改组为山東方面軍，吴化文任总司令。7月，山東方面軍改组为和平建国軍第3方面軍，吴化文任上将总司令。1945年（民国34年）1月，吴化文任[[汪精卫政权|汪精卫政权]]的国民政府軍事委員会委員。<ref name=fan/><ref name=xu/><ref name=xingtai/><ref name=liu>{{Cite book|author = 劉寿林等編|title = 民国職官年表|year = 1995|publisher = [[中華書局|中華書局]]|isbn = 7-101-01320-1}}</ref>

[[抗日战争|抗日战争]]结束後，1945年，吴化文回到蒋介石手下，和平建国軍第3方面軍改编为国民革命軍第五路軍，吴化文继续担任总司令，并兼任[[津浦铁路|津浦铁路]]南段警備司令。当时，第五路军下属两个军，一个独立师，其中第一军军长[[于林安|于林安]]，第二军军长[[杨友柏|杨友柏]]，独立师师长[[徐日政|徐日政]]，全军三万多人，驻扎[[安徽|安徽]][[蚌埠|蚌埠]]。改编后不久，第五路军奉[[蒋介石|蒋介石]]的命令开赴[[山东|山东]][[兖州|兖州]]，途中与[[八路军|八路军]]遭遇，第一军被歼灭，军长[[于怀安|于怀安]]被俘。1946年（民国34年），吴化文任山東保安第2纵队司令。当时，被俘后的[[于怀安|于怀安]]不断与吴化文秘密通信，劝其起义。1946年7月上旬，蒋介石电令吴化文到[[南京|南京]]面谈，在南京数日，吴化文还与[[王一民|王一民]]到[[冯玉祥|冯玉祥]]家看望冯玉祥，并经冯玉祥介绍，会见了[[李济深|李济深]]及[[章伯钧|章伯钧]]等人，通过[[章伯钧|章伯钧]]与[[中共|中共]]方面的[[周恩来|周恩来]]取得联系。吴化文回到[[兖州|兖州]]后，[[山东野战军|山东野战军]]司令员[[陈毅|陈毅]]即放回被俘的[[于怀安|于怀安]]。1946年11月，陈毅与吴化文双方派代表联系，取得默契。<ref name=xingtai/><ref name=bofu>{{cite web|url=http://paper.dzwww.com/qlwb/data/20080924/html/35/content_1.html?jdfwkey=abz113|title=我的伯父吴化文将军与济南战役|publisher=齐鲁晚报|author=吴欣民|date=2008-09-24|accessdate=|language=Zh-hans}}</ref>

1947年2月，山東保安第2纵队改组为整編第84師，吴化文任師長，该师归[[邱清泉|邱清泉]]指挥。1948年（民国37年），整編第84師调往[[济南|济南]]。同年8月，整编第2师（师长[[阎子丰|阎子丰]]）被拨给吴化文，成立整編第96軍，吴化文任整編第96軍軍長兼济南西守備区指揮官，整編第96軍下辖整編第84師（师长吴化文兼）、整编第2师，归[[王耀武|王耀武]]指挥。[[军统|军统局]]还委任吴化文为山东特派员。<ref name=fan/><ref name=xu/><ref name=xingtai/><ref name=bofu/>
[[File:Wu_Huawen1.jpg|thumb]]
1948年9月19日，经过[[中国共产党|中国共产党]]方面的推动，在[[济南战役|济南战役]]中，吴化文率3個旅2万人在[[济南|济南]]阵前投向[[中国共产党|中国共产党]]方面，并发表起义宣言。起义后，吴化文部被改编为中国人民解放军暂编第一军，吴化文任军长，下辖一、二两师，归[[华东军区|华东军区]]管辖。<ref name=fan/><ref name=xingtai/><ref>[http://www.xiexingcun.com/zhouenlai/1-zeljswx450.htm 吴化文部起义宣言及编制（一九四八年九月二十日），载 周恩来军事文选 第三卷，人民出版社，1997年]</ref>同年10月，该部又改为[[中国人民解放军第35军|中国人民解放軍第三十五軍]]，吴化文继续任軍長。同年12月，吴化文率第三十五军参加[[淮海战役|淮海战役]]。<ref name=fan/><ref name=xu/><ref name=xingtai/><ref name=bofu/>

1949年1月29日，[[华东野战军|华东野战军]]前委在给[[中共中央华东局|中共中央华东局]]和淮海战役总前委的电报中称：“为了使吴化文、[[何基沣|何基沣]]、[[张克侠|张克侠]]各部起义军队得到迅速改造，并以便解决干部问题，经我们讨论以[[鲁中南纵队|鲁中南纵队]]与吴化文部合编为三十五军，以[[渤海纵队|渤海纵队]]与张克侠部合编为三十三军，以江淮两个旅与何基沣部合编为三十四军。当[[陈毅|陈毅]]同志从华东局回来前，本拟以十二纵与何基沣部合编，但因十二纵已组成三个师，且将来主力南渡，尚须留两个军于江北，故仍以将江淮之两个旅也上升与何基沣部合编为宜。……”[[中共中央华东局|中共中央华东局]]和淮海战役总前委采纳了[[华东野战军|华东野战军]]前委意见，并获[[中央军委|中央军委]]批准。[[鲁中南纵队|鲁中南纵队]]乃与吴化文部原第三十五军合编为第三十五军，吴化文仍任军长。<ref>[http://news.xinhuanet.com/mil/2013-01/17/c_124244680_4.htm 解秘我陆军集团军番号：8个军团拱卫首都，新华网，2013年01月17日]</ref>
[[File:Wu_Huawen5.jpg|thumb]]
1949年，吴化文率第三十五军参加[[渡江战役|渡江战役]]，于1949年4月参加占领[[南京市|南京]]的行动，该军104师312团占领了[[南京总统府|南京总统府]]，把[[中国人民解放军|中国人民解放军]]的[[中国人民解放军军旗|战旗]]插上了总统府的大门。后来，第三十五军打到[[浙江|浙江]]，吴化文兼任杭州警备司令。[[中華人民共和国|中華人民共和国]]成立後，吴化文以生病为理由退役，转业到地方。1951年，吴化文出任[[浙江省人民政府|浙江省人民政府]]委員兼交通厅厅長。后来，历任[[中国人民政治協商会議全国委員会|中国人民政治協商会議全国委員会]]委員、浙江省中国人民政治协商会议委员会副主席。因中华人民共和国成立前的軍功，吴化文获得一級[[解放勋章|解放勋章]]。<ref name=fan/><ref name=xu/><ref name=xingtai/><ref name=bofu/>

1962年初，吴化文因[[胰腺癌|胰腺癌]]入住[[上海华东医院|上海华东医院]]，并接受了手术。1962年4月3日，吴化文因[[胰腺癌|胰腺癌]]病逝，享年59岁。吴化文逝世后，1962年4月7日在[[杭州市|杭州市]]凤山门殡仪馆举行了追悼会，[[周恩来|周恩来]]、[[陳毅|陳毅]]及八大軍区均献花圈致哀。<ref name=fan/><ref name=xu/><ref name=bofu/>

== 家庭 ==
[[File:Wu_Huawen6.jpg|thumb]]
[[File:Wu_Huawen7.jpg|thumb]]
吴化文兄弟三人，姊妹四人。吴化文在兄弟中排行第二。<ref name=jntsg>[http://www.jnlib.net.cn/jnkz/xwzl4.html 吴化文济南起义，济南市图书馆，于2013-05-02查阅] {{webarchive|url=https://web.archive.org/web/20101206163537/http://www.jnlib.net.cn/jnkz/xwzl4.html |date=2010-12-06 }}</ref>
*长兄：吴化善，早逝。<ref name=jntsg/>
*三弟：吴化纲，原任吴化文部电台台长，随吴化文起义后，转至[[山东省民政厅|山东省民政厅]]任职，退休后定居[[济南|济南]]。<ref name=jntsg/>

吴化文前后共有四位妻子：<ref name=jntsg/>
*元配：马玉珍，早逝。<ref name=jntsg/>
*继室：赵华珍，原为北京[[协和医院|协和医院]]护士，1920年代与吴化文结婚，1960年代在[[南京|南京]]去世。生有两子：<ref name=jntsg/>
**长子：吴新民，1960年代在[[南京|南京]]去世。<ref name=jntsg/>
***孙：吴挥，吴新民之子，后在[[中国科学院南京土壤研究所|中国科学院南京土壤研究所]]任职。<ref name=jntsg/>
**次子：吴哲民，1940年代毕业于[[中央陆军军官学校|中央军官学校]]，毕业后在吴化文部历任连长、参谋，1948年随吴化文起义，后加入[[中国共产党|中国共产党]]，曾任江苏省镇江市磷肥厂厂长、镇江市人大常委。<ref name=jntsg/>
*第三个妻子：[[林世英|林世英]]，山东掖县人，1930年代吴化文任韩复榘的手枪旅旅长时与吴化文结婚。吴化文起义后，林世英随吴化文到浙江，后任浙江省政协常委。生有两名子女，后均在大学任教，其一后赴[[美国|美国]]。<ref name=jntsg/>
**内弟：林世昌，林世英的四弟。原为北京某大学学生，中共党员。[[济南战役|济南战役]]前，林世昌数次赴吴化文处探亲，林世英受林世昌影响，姐弟二人对吴化文进行劝说，对吴化文最终决定起义起到很大作用。<ref name=jntsg/>
*第四个妻子：宁宜文，1940年代与吴化文结婚。中华人民共和国成立后寓居[[浙江省|浙江省]][[萧山县|萧山县]]。生有四名子女。<ref name=jntsg/>
