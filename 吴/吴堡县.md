'''吴{{ruby-py|堡|-{zh-cn:bǔ;zh-tw:ㄅㄨˇ}-}}县'''是[[中国|中国]][[陕西省|陕西省]][[榆林市|榆林市]]下辖的一个[[县|县]]，位于榆林东南角，距榆林市176公里，距[[延安市|延安市]]260公里，距[[西安市|西安市]]628公里。

== 地理 ==
吴堡县位于[[黄河|黄河]]西岸，与[[山西省|山西省]][[柳林县|柳林县]]相对；与[[佳县|佳县]]、[[绥德|绥德]]相邻。吴堡境内典型[[黄土|黄土]][[黄土高原|高原]]沟壑纵横的地形，气候干旱缺水，水土流失严重。

== 经济 == 
吴堡是贫困县。2005年发现有大型优质煤田，尚未开采。还有煤成气、浊氟石、磷矿等。[[307国道|307国道]]和[[太中银铁路|太中银铁路]]过境。 特产红枣、蚕桑等。

== 方言 == 
吳堡話屬于[[晉語|晉語]]吕梁片，老派可以區分[[尖團|尖團]]，是寥寥無幾可分尖團的晉語。
吳堡縣面積雖然不大，但內部差異却不小，可以略分為上吳堡話語下吳堡話。自東往西，橫溝、槐樹港、前胡家山、東王家山、車家塬、深砭墕、斜側、高家莊一線以北說上吳堡話，以南說下吳堡話。縣城所在地宋家川鎮的口音屬于下吳堡話。兩種口音的主要區別在于：①部分果攝合口字上吳堡讀[ɤu(uɤu)]，下吳堡讀[u];②蟹攝開口四等透母、定母(平聲)字，如「梯體剃提」，上吳堡念[tʰɛe]，下吳堡念[tɕʰi]。③有些詞語上下吳堡說法不同。请比较：

== 历史文化 ==
相传这里曾经被[[赫连勃勃|赫连勃勃]]用来安置南朝[[刘裕|刘裕]]（宋）的俘虏，因此得名吴儿堡。

[[北汉|北汉]]时吴堡建有石头古城，目前仍有部分明清建筑。由于山上取水困难，居民大都迁到山下新城。

作家[[柳青_(作家)|柳青]]、经济学家[[张维迎|张维迎]]是吴堡人。

吴堡曾是[[陕甘宁边区|陕甘宁边区]]的组成部分。
