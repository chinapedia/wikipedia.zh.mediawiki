'''吴继德'''{{BD|1955年|9月|||catIdx=W}}，出生于[[甘肃省|甘肃省]][[通渭县|通渭县]]，[[中国共产党|中国共产党]]党员，毕业于[[中共中央党校|中共中央党校]]，最近担任中共甘肃省委统战部副部长，甘肃省工商联党组书记、副会长，2014年11月在[[习近平反腐|习近平反腐]]中落马。<ref>{{cite web |url= http://news.163.com/14/1105/16/AAA3U2BG0001124J.html|title=甘肃省委统战部原副部长吴继德被立案侦查 |accessdate=2014 |publisher=网易 |work= |language=zh-hans}}</ref><ref>{{cite web |url=http://finance.ifeng.com/a/20141105/13250899_0.shtml |title= 甘肃省委统战部原副部长吴继德涉受贿被立案侦查|accessdate=2014 |publisher=凤凰网 |work= |language=zh-hans}}</ref>

==生平==

1976年9月到1978年9月，吴继德就读于定西农学院林果专业。

1978年9月，吴继德参加工作，1983年12月，吴继德加入[[中国共产党|中国共产党]]。

1978年9月到1984年1月，吴继德在甘肃省[[通渭县|通渭县]]工作。

1984年1月到1989年8月，吴继德在[[定西市|定西市]]工作，曾担任共青团定西地委宣传部副部长、部长、共青团定西地委副书记、中共定西地委政研室副主任。

1989年8月到1991年6月，吴继德被擢升为中共[[漳县|漳县]]县委副书记。

1991年6月到1994年12月，吴继德在中共[[兰州市|兰州市]]纪委党风检查教育室工作。

1994年12月到1997年9月，吴继德担任中共兰州市西固区委副书记。

1997年9月到2002年3月，吴继德在[[榆中县|榆中县]]工作，曾担任中共榆中县委副书记、榆中县人民政府代县长、中共榆中县委书记（1999年升正职）。

2002年3月到2010年9月，吴继德在甘肃省省府兰州市工作，曾担任兰州市人民政府副市长、党组成员、中共兰州市委常委，兰州市人民政府常务副市长、党组副书记。

2010年9月，吴继德升职为中共甘肃省委统战部副部长，甘肃省工商联党组书记、副会长。

2014年10月22日，吴继德接受[[中纪委|中纪委]]调查，11月，吴继德因为受贿被甘肃省人民检察院立案侦查。<ref>{{cite web |url=http://www.ccdi.gov.cn/ajcc/201410/t20141022_29181.html |title=甘肃省委统战部副部长吴继德接受组织调查 |accessdate=2014 |publisher=中纪委 |work= |language=zh-hans}}</ref>
