'''吴承恩'''（{{bd|1506年||1582年||catIdx=W吴}}），字'''汝忠'''，号'''射阳山人'''或'''射阳居士'''，[[淮安府|淮安府]][[山陽縣|山陽縣]]（今[[江蘇省|江蘇省]][[淮安市|淮安市]][[淮安區|淮安區]]）人，吴承恩父亲吴国迁由[[安庆府|安庆府]]桐城县高甸至[[淮安府|淮安府]][[山陽縣|山陽縣]]。<ref name="book200 " >{{cite book |language =zh-hans|author=刘庭乾、李桂奎、喻斌、张辉忠|title=《新编中国古代文学史》|year=1995年|publisher=天津教育出版社|location=天津|pages=第211-219页|isbn=9787530922286}}</ref>其代表作為《[[西游记|西游记]]》。

== 生平 ==
[[File:Xyj-sunwukong.jpg|right]]
父[[吴锐|吴锐]]，字廷器，卖“彩缕文羯”，“又好谭时政，竟有所不平，辄抚几愤惋，意气郁郁”。吴锐入赘徐家，“遂袭徐氏业，坐肆中”，棄儒成為商人。而后娶张氏为侧，吴承恩为侧室张氏所生。<ref>有專家统计《西遊記》自第十三回到第九十八回到达西天，前后共十个故事涉及招女婿。</ref>吴承恩自幼聰慧，喜讀野言稗史、志怪小說，“尝爱唐人如牛奇章、段柯古辈所著传记，善模写物情，每欲作一书对之”，“髫龄，即以文鸣于淮”<ref>吳國榮《射陽先生存稿跋》</ref>，頗得官府、名流和鄉紳的賞識。[[嘉靖|嘉靖]]八年（1529年），吴承恩到淮安知府[[葛木|葛木]]所创办的龙溪书院读书，得到葛木的赏识。[[朱应登|朱应登]]认为他“可尽读天下书”，而“以家所藏图史分其半与之”，嘉靖二十九年（1550年）大约50岁才补得一个岁贡生<ref>同治《山阳县志》誉之“工书，嘉靖中岁贡生，官长兴县丞，英敏博洽，为世所推，一时金石之文，多出其手”。</ref>，到[[北京|北京]]等待分配官职，没有入选。六年後，由于母老家贫，得到大臣[[李春芳|李春芳]]的帮助做了浙江[[长兴|长兴]][[县丞|县丞]]，常与友人[[朱曰藩|朱曰藩]]豪飲，寄趣于诗酒之间，和[[嘉靖|嘉靖]]状元[[沈坤|沈坤]]，诗人[[徐中行_(明朝)|徐中行]]有往來。终因受人诬告，两年后“拂袖而归”<ref>天启《淮安府志》</ref>，晚年以卖文为生，约六十七岁时到过杭州，大约活了将近80岁，晚景凄凉。<ref name="book200 " />

== 亲属 ==
吴承恩一生没有子嗣。吴承恩早年师从[[胡琏_(弘治进士)|胡琏]]<ref>吴承恩为胡琏之孙胡应恩的妻子张氏贺寿所作的《寿胡内子张孺人六袠序》中写道：“自孺子之归西畹也，我师南津翁，方在九列”。</ref>，平生和[[沈坤|沈坤]]、[[朱曰藩|朱曰藩]]、[[李春芳|李春芳]]为莫逆之交，和吴中名士[[文徵明|文徵明]]、[[王宠|王宠]]书画相和，诗词唱和，有明朝中晚期士子疏狂自傲，不合时流的时代精神风骨。吴承恩在[[长兴|长兴]]为官时，和后七子之一徐中行交往密切。晚年和[[陈耀文|陈耀文]]、[[陈文烛|陈文烛]]、[[邵元哲|邵元哲]]结交为翰墨之友。<ref name="book100 " />

== 作品 ==
吴承恩一生创作什多，但是由于家贫，又没有子女，作品多散失。据记载有志怪小说集《禹鼎记》已失传。吳承恩的甥外孙[[丘度|丘度]]搜集其残存之稿，仅“存十一于千百”，編《射阳先生存稿》四卷，包括诗一卷、散文三卷。

=== 风格 ===
吴承恩的诗歌长篇歌行激越豪放，近乎[[李白|李白]]。词曲出入《[[花间集|花间集]]》，近乎[[秦观|秦观]]。散文学习[[歐陽脩|歐陽脩]]、[[曾巩|曾巩]]，与唐宋古文家风格接近。<ref name="book100 " >{{cite book |language =zh-hans|author=陶雯|title=《中国古代文化宝典》|year=1997年|publisher=光明日报出版社|location=北京|pages=第112-116页|isbn=7-80091-986-2}}</ref>

1936年[[赵景深|赵景深]]首次撰成《西游记作者吴承恩年谱》。

=== 小说 ===
一般公认吴承恩是《[[西遊記|西遊記]]》的最后定稿作者。[[胡适|胡适]]与[[鲁迅|鲁迅]]根据清代[[吴玉搢|吴玉搢]]的《山阳志遗》、[[阮葵生|阮葵生]]的《茶余客话》和[[丁晏|丁晏]]的《石亭记事续篇》等书的考证，得出了《西游记》的作者是[[淮安|淮安]]嘉靖中岁[[贡生|贡生]]吴承恩的结论。{{NoteTag|最早对吴承恩为小说《西游记》作者提出怀疑的是[[俞平伯|俞平伯]]。1933年，俞平伯在《驳〈跋销释真空寶卷〉》中明确提出：“吳氏作《西游记》，根据《淮安府志》，志书上所谓《西游记》是不是这个《西遊记》呢？也难定。”认定吴承恩是小说《西游记》作者的主要依据有三，一是天启《淮安府志》“淮贤文目”在吴承恩名下记有《西游记》；二是吴承恩的《禹鼎志》、《二郎搜山图歌》的创作可以看出吴承恩有创作《西游记》的可能；三是《西游记》所用方言为吴承恩家乡淮安的方言。1983年，[[章培恒|章培恒]]在《社会科学战线》第4期上发表《百回本（西游记）是否吴承恩作》一文详细地论证了百回本小说《西游记》非吴承恩作。}}

== 评价 ==
天启《[[淮安府志|淮安府志]]》评价他“性敏而多慧，博极群书，为诗文下笔立成，清雅流丽，有[[秦观|秦少游]]之风。复善谐谑，所著杂记几种，名震一时”。

淮安知府[[陳文燭|陳文燭]]称为“李太白、辛幼安之遗也”。

《长兴县志》称他：“性耽风雅，作为诗，缘情体物，习气悉除。其旨博而深，其辞微而显，张文潜后殆无其伦。”

[[朱彝尊|朱彝尊]]《[[明诗综|明诗综]]》卷四十八谓其诗“习气息除，一时殆鲜其匹。”

== 其他 ==
2016年，是《西遊記》作者吳承恩的510週年誕辰。2016年10月28號，江蘇省淮安市政府與《人民文學》雜誌社共同設立了「吳承恩長篇小說獎」，並致力使它成為具廣泛影響力的國際級文學獎。
