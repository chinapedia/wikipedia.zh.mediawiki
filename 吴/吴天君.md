'''吴天君'''（{{bd|1957年|2月|}}），[[河南|河南]][[濮阳|濮阳]]人。1979年12月参加工作，1982年12月加入中国共产党。[[安阳农业大学|安阳农业大学]]农学系农学专业毕业，[[中国地质大学|中国地质大学]]资源产业经济专业博士研究生。

先后担任中共[[内黄县|内黄县]]县委书记，中共[[新乡市|新乡市]]委副书记，中共新乡市委副书记、新乡市人民政府市长，中共[[新乡市|新乡市]]委书记等职。2011年5月，任中共郑州市委副书记、代市长<ref>{{cite web|url=http://www.chinanews.com/gn/2011/06-01/3082210.shtml|title=赵建才辞去郑州市市长职务 吴天君任代市长|publisher=中国新闻网|accessdate=2011-06-01}}</ref>；同年10月当选为[[郑州市|郑州市]]市长。2011年12月，任[[中共河南省委|中共河南省委]]常委、政法委书记。2012年2月，改兼中共郑州市委书记<ref>{{cite web|url=http://news.xinhuanet.com/politics/2012-02/07/c_122665273.htm|title=吴天君任郑州市委书记 连维良不再担任另有任用|publisher=新华网|accessdate=2012-02-07}}</ref>。2016年5月，不再担任中共郑州市委书记职务<ref>{{cite web|url=http://www.thepaper.cn/newsDetail_forward_1473852| title=马懿任河南郑州市委书记，程志明任郑州市委副书记|publisher= 澎湃新闻|accessdate=2016-05-24}}</ref>，再度兼任中共河南省委政法委书记<ref>{{cite web|url=http://www.thepaper.cn/newsDetail_forward_1474952| title=郑州市委原书记吴天君再度兼任河南省委政法委书记|publisher= 澎湃新闻|accessdate=2016-05-26}}</ref>。2016年11月，不再担任中共河南省委常委<ref>{{cite news|title=李亚、许甘露、穆为民新任河南省委常委 尹晋华等3人不再担任|url=http://district.ce.cn/newarea/sddy/201611/04/t20161104_17533554.shtml|accessdate=2016-11-04|newspaper=中国经济网}}</ref>。

== 落马 ==
2016年11月11日，[[中央纪委监察部网站|中纪委监察部网站]]发布消息：河南省委原常委，省委政法委书记吴天君涉嫌严重违纪，目前正接受组织调查<ref>{{Cite web|url=http://www.ccdi.gov.cn/xwtt/201611/t20161111_89448.html|title=河南省委原常委、政法委书记吴天君涉嫌严重违纪接受组织调查|accessdate=2016-11-11|author=|date=2016-11-11|publisher=中央纪委监察部网站}}</ref>。吴是中共十八大以来河南省第三位落马的省部级官员。民间对吴天君的称呼，有“一指霉”、“吴一指”、“[[拆迁|拆迁]]大佐”、“[[强制拆迁|强拆]]书记”等。

2017年1月23日，中纪委通报吴天君被双开，移送司法机关<ref>{{cite web |url = http://www.ccdi.gov.cn/xwtt/201701/t20170123_93321.html |title = 河南省委原常委、政法委书记吴天君严重违纪被开除党籍和公职 |accessdate = 2017年1月23日 |publisher = 中央纪委监察部网站}}</ref>。2月6日，最高人民检察院宣布依法对吴天君以涉嫌受贿罪立案侦查并采取强制措施<ref>{{Cite web|url=http://news.163.com/17/0206/15/CCJQI5P3000187VE.html|title=河南省委原常委、政法委书记吴天君涉嫌受贿罪被立案侦查|accessdate=2017-02-06|author=|date=2017-02-06|publisher=网易}}</ref>。

2017年8月4日，湖北省襄阳市中级人民法院公开宣判吴天君受贿案，对被告人吴天君以受贿罪判处有期徒刑11年，并处没收个人财产人民币100万元。对吴天君受贿所得财物及其孳息予以追缴，上缴国库。吴天君当庭表示服从判决，不上诉<ref>{{cite web |url = http://www.ccdi.gov.cn/yw/201708/t20170805_104244.html |title = 河南省委原常委、政法委原书记吴天君受贿案一审宣判 |accessdate = 2017年8月5日 |publisher = 中央纪委监察部网站}}</ref>。
