'''吴王'''，中国古代封爵之一。

== 周朝 ==
诸侯国[[吴国|吳国]]的国君。
* [[阖闾|阖闾]]
* [[夫差|夫差]]

== 汉朝 ==
* [[刘濞|刘濞]]，[[汉高祖|汉高祖]]兄[[刘喜_(代王)|刘喜]]之子，前195年-前153年在位。

== 三国 ==
* [[孙权|孙权]]，220年受[[曹魏|曹魏]]封为吴王，222年脱离曹魏自建年号，229年正式登基。

== 晋朝 ==
* [[司马晏|司马晏]]，[[晋武帝|晋武帝]]之子。
* [[司马岳|司马岳]]，[[晋明帝|晋明帝]]之子，后改封琅邪王，登基为[[晋康帝|晋康帝]]。

== 汉赵 ==
* [[刘逞|刘逞]]，匈奴汉国皇帝[[刘聪|刘聪]]之子。

== 前燕 ==
* [[慕容垂|慕容垂]]，前燕太祖[[慕容皝|慕容皝]]之子，后投奔[[前秦|前秦]]，前燕灭亡后叛前秦，复辟[[后燕|后燕]]为其开国皇帝。

== 北魏 ==
* [[拓跋余|拓跋余]]，[[北魏太武帝|北魏太武帝]]之子，后改封南安王，短暂登基为帝。
* [[长孙陈|长孙陈]]，又名[[长孙陵|长孙陵]]，[[长孙肥|长孙肥]]之子，吴郡公，追赠吴郡王。

== 东魏 ==
* [[萧正表|萧正表]]，[[梁武帝|梁武帝]][[萧衍|萧衍]]弟临川王[[萧宏|萧宏]]之子，吴郡王，后改兰陵郡王。

== 西魏 ==
* [[元公|元公]]，西魏文帝[[元宝炬|元宝炬]]之子。

== 南朝齐 ==
* [[萧宝卷|萧宝卷]]，[[齐明帝|齐明帝]]之子，登基为帝，曾被起兵作乱的[[崔慧景|崔慧景]]矫诏废为吴王。

== 西梁 ==
* [[萧岑|萧岑]]，西梁中宗[[萧詧|萧詧]]之子，吴郡王。

== 南朝陈 ==

* [[李恪_(吴王)|李恪]]，[[唐太宗|唐太宗]]之子。

** [[杨隆演|杨隆演]]，杨渥弟和继承人，唐朝灭亡后仍称唐朝吴王，后称吴国王并建年号。
** [[杨溥_(吴)|杨溥]]，杨隆演弟和继承人，后称帝。
* [[钱镠|钱镠]]，唐末群雄之一，[[后梁|后梁]]年间改吴越国王，[[吴越国|吴越国]]开国之君。

== 元朝 ==
封地在今江苏省境内，为金印兽纽王。
* [[朵列纳|朵列纳]]，[[哈赤温|哈赤温]]之后，[[皇庆|皇庆]]元年（1312年）由[[济王|济王]]改封。
* [[泼皮|泼皮]]，[[朵列纳|朵列纳]]之子，[[天历|天历]]三年（1330年）徙封[[济阳王|济阳王]]。 
* [[木南子|木南子]]，[[哈赤温|哈赤温]]之孙，[[忽剌忽儿|忽剌忽儿]]之子，[[天历|天历]]三年（1330年）由[[济阳|济阳]]徙封。
* [[搠思坚|搠思坚]]，[[木南子|木南子]]之子。
* [[朵尔赤|朵尔赤]]，[[搠思坚|搠思坚]]之子，即《明史》中的朵耳只。

== 元末 ==
* [[張士誠|張士誠]]，[[至正|至正]]二十三年（1363年）九月，自称吴王。
* [[朱元璋|朱元璋]]，[[龙凤_(韩林儿)|龙凤]]十年（1364年），自称吴王。[[至正|至正]]二十八年（1368年）称帝。

== 明朝 ==
* [[朱橚|朱橚]]，明太祖朱元璋第五子。洪武三年（1370年）封为吴王，洪武十一年（1378年）改封周王。
* [[朱允熥|朱允熥]]，明兴宗朱标的嫡三子，母孝康皇后常氏。洪武十一年（1378年）出生，建文元年（1399年）受封吴王，封国杭州，未就藩；建文四年（1402年），成祖即位，贬为广泽王，居漳州。同年九月召回，废为庶人，禁锢于凤阳。永樂十五年（1417年）去世。南明朝廷追复他為吴王，谥曰悼或哀。
