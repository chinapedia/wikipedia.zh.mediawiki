'''吴克烈'''（{{lang|ko|오극렬}}，{{bd|1931年|catIdx=W}}）[[朝鲜劳动党|朝鲜劳动党]][[朝鲜劳动党中央委员会政治局|中央政治局]]候补委员、朝鲜民主主义人民共和国[[最高人民会议|最高人民会议]]议员。[[朝鲜次帅列表|大将]]军衔。

吴克烈出生于[[中国|中国]][[吉林省|吉林省]]，父亲是[[金日成|金日成]]的战友[[吴仲洽|吴仲洽]]（1910－1939），在战争中牺牲。少年时期吴克烈是[[金正日|金正日]]的玩伴，与金正日和前外交相[[许錟|许錟]]結拜為義兄弟，<ref name = "touitsu11921">[http://news.onekoreanews.net/detail.php?number=63931 金正日にも義兄弟がいる] 統一日報 2011年9月21日</ref>。毕业于万景台军事学院和[[蘇聯空軍大學|蘇聯空軍大學]]。1964年11月任金策空軍大学学部長，升[[少将|少将]]。1967年10月任空军司令员，晋升[[中将|中将]]。同年11月当選第4届[[最高人民会议|最高人民会议]]代議員。1969年任人民军总参谋部作战局局长。

1970年11月、当選中央委員会委員。1972年12月、当選第5届最高人民会議代議員。

1977年10月、就任[[朝鲜人民军|朝鲜人民军]]副参謀長。同年11月、当選第6届最高人民会議代議員。

1979年9月、当选劳动党中央政治局候补委员。同年9月任人民军总参谋长<ref>http://d-arch.ide.go.jp/browse/pdf/1979/102/1979102TPC.pdf</ref>。

1980年9月晋升为[[上将|上将]]。同年10月10日在第6届全党代表大会上入选为中央政治局委员<ref>http://d-arch.ide.go.jp/browse/pdf/1980/102/1980102REF.pdf</ref>

1985年4月13日，劳动党中央人民委員会授与他[[朝鲜次帅列表|人民軍大将]]称号<ref>「[http://d-arch.ide.go.jp/browse/pdf/1985/102/1985102DIA.pdf 重要日誌]」『アジア動向年報』1985年版、69ページ</ref>。

1988年2月、总参謀長退任。金日成考虑到吴克烈的地位和权利太高造成对金正日的威胁，将其解任<ref name = "touitsu11921"/>。同年9月，就任劳动党民間防衛部部長。1989年担任劳动党作战部长一职<ref name = "touitsu11921"/>，主要负责对[[韓國|韓國]]输出间谍执行军事侦察和破坏、暗杀任务，1996年[[江陵潜艇渗透事件|江陵潜艇渗透事件]]正是由劳动党作战部一手策划的作战行动，暗杀金正日前妻成惠琳（已故）的侄子[[李韩永|李韩永]]事件也是由劳动党作战部负责实施的。

2009年2月20日，据[[朝鲜中央通讯社|朝鲜中央通讯社]]报导，吴克烈获任命为[[朝鲜民主主义人民共和国国防委员会|国防委员会]]副委员长。2011年12月金正日死后[[金正恩|金正恩]]体制组成，2012年4月13日，吴克烈在[[朝鲜劳动党第四次代表会议|朝鲜劳动党第四次代表会议]]上重新当选中央政治局候补委员<ref>{{Cite news
