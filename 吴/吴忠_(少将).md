'''吴忠'''（{{bd|1921年||1990年||catIdx=W}}），原名吴光珠，[[表字|字]]宝山。[[四川省|四川省]][[苍溪县|苍溪县]]人，[[中国人民解放军开国少将|中国人民解放军开国少将]]，中国共产党第九、十、十一届中央候补委员。

== 早年經歷 ==
1933年参加[[中国工农红军|中国工农红军]][[红三十军|红三十军]]{{NoteTag|红三十军政委是[[李先念|李先念]]。1976年抓捕四人帮案件中，李先念是主要策划者之一。}}，同年加入[[中国共产主义青年团|中国共产主义青年团]]，1935年加入[[中国共产党|中国共产党]]，经历了[[长征|长征]]、[[抗日战争|抗日战争]]、[[淮海战役|淮海战役]]、[[渡江战役|渡江战役]]、[[上海战役|上海战役]]和[[西南战役|西南战役]]<ref>{{cite web |title=吴忠--党史频道--人民网 |url=http://dangshi.people.com.cn/GB/165617/165618/166491/167909/12092284.html |publisher=人民网 |accessdate=2018-07-02}}</ref>。

== 中華人民共和建國後至文革 ==
1955年授予少将军衔，荣获三级[[八一勋章_(1955年)|八一勋章]]，二级[[独立自由勋章|独立自由勋章]]，二级[[解放勋章|解放勋章]]，一级[[红星功勋荣誉章|红星功勋荣誉章]]。同时他是至今为止中国人民解放军中最年轻的获将衔者。

1970年任[[北京卫戍区|北京卫戍区]]司令员。在1976年[[懷仁堂事變|懷仁堂事變]]中扮演要角，10月2日[[華國鋒|華國鋒]]秘密召見了吳忠并獲得了支持。10月4日下午，華國鋒等三人最終敲定了事變方案，10月6日懷仁堂事變成功扳倒[[四人幫|四人幫]]，[[文革|文革]]結束，[[改革開放|改革開放]]開始。

== 參與越南戰爭與被隔離審查<ref>[http://dangshi.people.com.cn/BIG5/85038/15447529.html 33歲被授銜 誰是我軍歷史上最年輕的開國將軍？ (6)--黨史頻道-人民網]</ref> ==
于1977年9月至1980年1月任[[中国人民解放军广州军区|广州军区]]副司令员，上任後不久，邊境地區形勢日趨緊張，根據軍區[[黨委|黨委]]的決定，吳忠於1978年夏天前往[[廣西|廣西]]，視察邊境地區的戰備工作。這年底，[[1979年中越戰爭|對越南的戰爭]]即將開打，吳忠主動向軍區黨委請戰並獲得批准，負責統一指揮廣州軍區南集團部隊，擔負主要戰役突擊任務。然而吳忠隨後卻因在任北京衛戍區司令員期間，因受命對聚集在天安門廣場吊唁周總理的群眾實行[[四五運動|清場]]這一錯誤，被列為「揭批查」的對象（即清查與[[林彪|林彪]]、四人幫兩個[[反革命|反革命]]集團有牽連的對象的運動）。[[中央軍委|中央軍委]]決定免去吳忠的廣州軍區副司令員職務，開始對他在「[[文化大革命|文化大革命]]」中的表現進行清查，但吳忠未服從命令去北京，而是繼續參與越南戰事。

戰役結束後，吳忠面對長達8年的審查。直到1987年6月18日，經中央軍委批准，北京軍區黨委對吳忠作出最后結論：「經審查，吳忠同志1971年3月至1977年9月，在任北京衛戍區司令員、[[北京|北京]][[市委書記|市委書記]]，分管政法期間，沒有參與林彪、四人幫篡黨奪權的陰謀活動……吳忠同志在天安門事件中的錯誤，是執行問題。鑒於當時的歷史條件，經中央軍委批准，予以結論。」

== 晚年至逝世 ==
1988年7月，吳忠被中央軍委授予中國人民解放軍一級紅星功勛榮譽章。兩個月后，鑒於吳忠的身體狀況，中央軍委批准他離職休養。離休后，吳忠積極搜集資料，醞釀著寫回憶錄，後來撰寫《中原主帥[[劉伯承|劉伯承]]北渡[[淮河|淮河]]記》、《諄諄教誨寓深情--關於劉帥一封重要信件的回憶》等文章，成為研究劉伯承的重要史料。

1990年2月26日，吳忠因车祸在海南逝世，终年71岁。<ref>{{cite web |title=新中国死于意外的六位将军：吴忠、皮定均等_ 读史忆人 _文史观澜 |url=http://history.rmlt.com.cn/2013/0924/155167.shtml |publisher=人民网 |accessdate=2018-07-02}}</ref>
