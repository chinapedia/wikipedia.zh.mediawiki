'''吴仁华'''（{{bd|1956年|9月12日|||}}），[[文獻學|历史文献]]学者，[[八九民运|八九民运]]的参与者和见证人，出生于[[中国|中国]][[浙江省|浙江省]][[温州市|温州市]][[苍南县|苍南县]]，长期参与[[中華人民共和國民主運動|中國民主运动]]，當中包括[[八九民運|八九民運]]的研究工作<ref name ="initium01">[https://theinitium.com/article/20170602-mainland-tiananmensquare/ 羅四鴒 把手指放在傷口上：吳仁華記錄六四的28年], [[端傳媒|端傳媒]] ,2017-05-30</ref>。

吳仁華專注於搜集六四事件的資料，搜尋和追查事件中的被捕人士、死難者和施害者名單<ref name = "ming01"/>。著有《六四天安門血腥清場內幕》、《六四屠殺內幕解密：六四事件中的戒嚴部隊》和《天安門事件逐日記錄》<ref name="吴仁华2009">{{cite book|author=吴仁华|title=六四事件中的戒严部队|url=http://books.google.com/books?id=xcksnQEACAAJ|year=2009年|location=加州 | publisher=真相出版社 | isbn=978-0-9823203-8-9 | oclc=427606994}}</ref><ref>{{cite news| url=http://www.rfa.org/mandarin/yataibaodao/kejiaowen/th-08142013101249.html| author=萧融| title=吳仁華在洛完成第三本專著 《天安門事件逐日記錄》明年六四面世| date=2013年8月14日| publisher=自由亞洲電台| deadurl=no| archiveurl=https://web.archive.org/web/20141106183901/http://www.rfa.org/mandarin/yataibaodao/kejiaowen/th-08142013101249.html| archivedate=2014年11月6日}}</ref>。他現居[[台灣|台灣]]，在[[東吳大學|東吳大學]]和[[中正大學|中正大學]]開設「六四事件真相研究」課程<ref name="DW03">{{cite news |title=專訪﹕研究六四29載讓我很痛苦 |url=https://www.dw.com/zh/%E4%B8%93%E8%AE%BF%E7%A0%94%E7%A9%B6%E5%85%AD%E5%9B%9B29%E8%BD%BD%E8%AE%A9%E6%88%91%E5%BE%88%E7%97%9B%E8%8B%A6/a-44041018?&zhongwen=trad |agency=德國之聲 |date=2018-06-04}}</ref>。

==生平==
===早年經歷===
1974年，吳仁華[[高中|高中]]畢業後便隨即「[[上山下乡运动|下鄉插隊]]」，擔任[[小學|小學]]和[[初中|初中]][[民辦教師|民辦教師]]。在1976年11月至1978年2月期間任職於[[公安邊防部隊|公安邊防部隊]][[平陽縣|平陽縣]]大隊<ref>陳子華《浴火重生：「天安門黑手」備忘錄》，明鏡出版社，2004年，第564頁</ref>。1978年2月至1982年2月，吳仁華就讀於[[北京大學|北京大學]]，為中文系古典文献专业，並且获得[[文学学士|文学学士]]学位<ref name="st">{{cite news | url=http://news.singtao.ca/toronto/2013-01-13/china1358061874d4294580.html | title=「六四」流亡22年入 境陪伴母親40多天 吳仁華返鄉探母 | date=2013年1月13日 | publisher=星島日報 | deadurl=no | archiveurl=https://web.archive.org/web/20150518025600/http://news.singtao.ca/toronto/2013-01-13/china1358061874d4294580.html | archivedate=2015年5月18日 }}</ref>。在身為北京大学中文系古典文献专业硕士研究生期間，他曾在[[中華書局|中華書局]]編輯部哲學編輯室任職[[編輯|編輯]]。1989年，已取得[[文學碩士|文學碩士]]學位的吳仁華在[[中國政法大學|中國政法大學]]法律古籍整理研究所任助理[[研究員|研究員]]<ref name ="initium01" />。

===參與八九民運===
1989年八九民運，他與學生們一起在[[天安門廣場|天安門廣場]]留守，期間擔任糾察隊。6月4日，他親歷同伴被[[坦克|坦克]]壓死的場景。1990年2月，吳仁華逃到[[珠海|珠海]]，游泳到[[澳門|澳門]]，在[[黃雀行動|黃雀行動]]救援人員安排下坐漁船[[偷渡|偷渡]][[香港|香港]]，7月5日，吳仁華流亡到[[美國|美國]]，現定居[[洛杉磯|洛杉磯]]。多年來利用考據學、目錄學的專業知識，在[[中國政府|中國政府]]流出的僅有資料中追查涉事官兵、死傷者、被捕入獄者的名單<ref name="ming01">[https://news.mingpao.com/pns/dailynews/web_tc/article/20170604/s00005/1496512580065  什麼人訪問什麼人﹕六四歷史人吳仁華 半生故紙堆 孤獨尋 一瞬現實] {{webarchive|url=https://web.archive.org/web/20170623223329/https://news.mingpao.com/pns/dailynews/web_tc/article/20170604/s00005/1496512580065 |date=2017-06-23 }}, 黃熙麗，[[明報|明報]], 2017-06-04</ref>。

===後期經歷===
1991年5月至2005年7月，吳仁華擔任由旅居海外的中國新聞工作者創辦的中文報紙《[[新聞自由導報|新聞自由導報]]》總編輯。2012年11月，吳仁華使用[[美國護照|美國護照]]獲得[[中華人民共和國|中華人民共和國]]的入境許可，看望闊別已久的親人。其後[[国家安全机关|中國國安部門]]的人員告知他「以後不會再有入境機會」<ref name="st" />。

吳仁華曾任[[中国民主团结联盟|中国民主团结联盟]]（中国民联）联委会委员、[[中国民主联合阵线|中国民主联合阵线]]（民联阵）总部理事、《[[中国之春|中国之春]]》[[董事|董事]]、中国战略研究所理事。现任[[中国宪政协进会|中国宪政协进会]][[秘书长|秘书长]]、[[中国民主党|中国民主党]]全国委员会[[顾问|顾问]]<ref name="吳仁華2007">{{cite book|author=吳仁華|title=天安門血腥清場内幕: 1989,6,4| url=http://books.google.com/books?id=zMSBNwAACAAJ| year=2007年|location=加州 |publisher=真相出版社|isbn=978-0-9795273-6-4 | oclc=769008579}}</ref>。

==六四事件研究==
吳仁華以八九民運的參與者和六四事件現場目擊者的身份，開展一系列搜尋，搜尋和追查事件中的被捕人士、死難者和施害者名單，包括搜尋死難者和傷殘者，搜尋死難者的家人，搜尋搶救受難者的[[醫生|醫生]]和[[護士|護士]]等等。他說：「我們可以做一個尋找系列，我已經找到了兩千多官兵的名單，我可以拋磚引玉，把他們陸續公佈到網上，建立一個專題。我提示大家，搜尋這些軍人的下落，然後你可以給他施加壓力，讓他們說出當年自己的見證。」<ref name = "rfi01" />

吳仁華在《天安門血腥清場內幕》一書中說，祇要軍隊聽從民眾的勸阻，停止前進，民眾就對他們非常友善。例如在[[天壇|天壇]]東大門墻外，有6、7萬民眾包圍300多名官兵，但因為那些官兵沒有開槍也不再前進，就沒有一個受到傷害，民眾還給他們送食物和飲料，並主動把途中受傷的11名官兵送往[[醫院|醫院]]急救。他也說，在六四事件中，軍警死了15人，沒有一個早於1989年6月3日晚上11點，可見軍隊鎮壓在先<ref name = "su01"/>。

===建立名單===
吳仁華認為，牽涉到六四屠殺如此一個重大的侵犯人權的災難性事件如果沒有加害者和害者的基本記錄，那麼這個歷史紀錄就完全是不合格。他在接受[[德國之聲|德國之聲]]訪問時指出，研究工作中最困難的地方是工作難度會隨著年代愈來愈久遠而愈來愈大，如果現時不做相關研究，六四事件很可能像[[文化大革命|文化大革命]]、[[反右派運動|反右派運動]]、[[土改運動|土改運動]]、[[鎮壓反革命運動|鎮壓反革命運動]]等事件一樣沒有留下加害者和受害者的歷史紀錄。他說工作的重要性在於還原真相和留下歷史紀錄，他指中國政府一直利用掌控媒體顛倒黑白，「利用錄像、媒體把軍隊開槍跟部分民眾以暴制暴，把時間先後、關係因果顛倒」，為暴力鎮壓民眾提供「合理依據」<ref name = "DW03"/>。

====部分鎮壓者====
吳仁華調查和考證出執行北京戒嚴任務的包括14個陸軍集團軍、一個直屬[[中央軍委|中央軍委]]、是中國當時最精銳的部隊的[[空降兵|空降兵]]第15軍、[[北京軍區|北京軍區]]的砲兵第14師、天津警備區的坦克第一師、北京衛戍區的警衛第一師、警衛第三師武警部隊北京總隊，進京官兵人數在20萬到25萬之間。他指出，軍隊的任務是鎮壓民運，但同時也是為了防止中共黨內可能發生的政變跟部隊的兵變<ref name = "su01">舒崇《把刽子手钉在历史的耻辱柱上——读吴仁华新着〈六四事件中的戒严部队〉》，[[北京之春|北京之春]]，2009-06-30</ref><ref name ="initium01"/><ref>[https://www.voachinese.com/a/wurenhua-20160603/3360706.html 曹雅学: 天安门运动与六四屠杀的全景记录者 - 吴仁华采访录] {{webarchive|url=https://web.archive.org/web/20170704071416/https://www.voachinese.com/a/wurenhua-20160603/3360706.html |date=2017-07-04 }}，[[美國之音|美國之音]]，2016-06-04</ref>：
*陸軍第38軍和空降兵第15軍，被稱為「殺人最多」的兩支部隊。
*戒嚴部隊中的坦克第一師：故意在[[自行車|自行車]]道高速行駛追殺學生，造成11人死亡多人受傷，指揮官是[[團長|團長]]羅剛和副團長賈振祿，軋人的[[坦克|坦克]]車編號是106。吳仁華說：「……就是這個106號坦克，衝進學生隊伍，當場壓死那麼多學生，所以當年我跟很多學生一路從六部口哭喊著記住這個坦克的編號106，一直哭喊著回到校園！」
*[[吴彦輝|吴彦輝]]：天津警備區第一師第一團第一連第一排106坦克二炮手。
*閆紅計：38軍坦克第一師的師參謀長，他堅決地執行命令，在事件當中帶領該坦克部隊的先頭部隊最先抵達天安門廣場，包括製造了六部口慘案的那三輛坦克。他的行動被軍方宣傳為「英雄事蹟」，他亦曾在[[北京廣播學院|北京廣播學院]]門口殺死了一個試圖阻攔的學生。

====部分抗命者====
他也指出有不少軍人因不願執行上司的命令，拒絕向人民開槍而遭受迫害和處罰<ref name ="initium01"/><ref name="rfi01">[http://cn.rfi.fr/中国/20100605-展开网络搜寻，揭示六四真相 展开网络搜寻，揭示六四真相] {{webarchive|url=https://web.archive.org/web/20180408133317/http://cn.rfi.fr/%E4%B8%AD%E5%9B%BD/20100605-%E5%B1%95%E5%BC%80%E7%BD%91%E7%BB%9C%E6%90%9C%E5%AF%BB%EF%BC%8C%E6%8F%AD%E7%A4%BA%E5%85%AD%E5%9B%9B%E7%9C%9F%E7%9B%B8 |date=2018-04-08 }}, 王山，法新社, 2010-06-05</ref><ref name = "su01"/>：
*38軍軍長[[徐勤先|徐勤先]]：因抗命被解除職務，後來判刑5年。
*第28軍軍長[[何燕然|何燕然]]及政委[[張明春|張明春]]：因抗命被降級調職。
*第38軍步兵116師師長[[許峰|許峰]]：因抗命被解除軍職。
*另有多名下級軍人在當時被官方聲稱失踪，但其實是離開現場，受到的處罰更為嚴厲。

===開設課程===
2018年，吳仁華在東吳大學和中正大學開設「六四事件真相研究」課程，當中有對六四事件不瞭解的中國大陸學生參與。吳仁華說，陸生即使在台灣，也還是有心理恐懼，儘管事先說好不要錄聽課者和提問者，但陸生還是不敢提問，亦有陸生指有所謂「職業學生」負責監控或打小報告<ref name = "DW03" />。
