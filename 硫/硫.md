'''硫'''是一种[[化学元素|化学元素]]，在[[元素周期表|元素周期表]]中它的[[化学符号|化学符号]]是'''S'''，[[原子序数|原子序数]]是16。硫是一种非常常见的无味无臭的非金属，纯的硫是黄色的晶体，又稱'''硫黄'''、'''硫磺'''。硫有许多不同的化合价，常見的有-2, 0, +4, +6等。在自然界中常以[[硫化物|硫化物]]或[[硫酸盐|硫酸盐]]的形式出现，尤其在[[火山|火山]]地区纯的硫也在自然界出现。'''硫单质'''难溶于水，微溶于[[乙醇|乙醇]]，易溶于[[二硫化碳|二硫化碳]]。对所有的生物来说，硫都是一种重要的必不可少的元素，它是多种[[氨基酸|氨基酸]]的组成部分，尤其是大多数[[蛋白质|蛋白质]]的组成部分。它主要被用在[[肥料|肥料]]中，也廣泛地被用在[[火药|火药]]、[[潤滑劑|潤滑劑]]、[[殺蟲劑|殺蟲劑]]和[[抗真菌剂|抗真菌剂]]中。

== 主要特征 ==
纯的硫呈浅黄色，质地柔软，轻。与[[氢|氢]]结成有毒化合物[[硫化氢|硫化氢]]后有一股臭味（臭鸡蛋味）<ref>一般所謂硫的臭味其實是[[硫化氢|硫化氢]]或是[[有机硫化学|有机硫化合物]]的臭味</ref>。硫燃烧时的火焰是蓝色的，并散發出一种特别的硫磺味（[[二氧化硫|二氧化硫]]的气味）。硫不溶于水但溶于[[二硫化碳|二硫化碳]]。硫最常见的化学价是-2、+2、+4和+6。在所有的物态中（固态、液态和气态），硫都有不同的[[同素异形体|同素异形体]]，这些同素异形体的相互关系还没有被完全理解。晶体的硫可以组成一个由八个原子组成的环：S<sub>8</sub>。

硫有两种晶体形式：斜方晶八面体和单斜棱晶体，前者在室温下比较稳定。

== 用途 ==
硫在工业中很重要，比如作为[[电池|电池]]中或溶液中的[[硫酸|硫酸]]。硫被用来制造[[黑火药|黑火药]]。在[[橡胶|橡胶]]工业中做[[硫化剂|硫化剂]]。硫还被用来杀[[真菌|真菌]]，用做化肥。硫化物在[[造纸业|造纸业]]中用来漂白。硫酸盐在[[烟火|烟火]]中也有用途。硫代硫酸钠和硫代硫酸銨在照相中做定影剂。[[硫酸镁|硫酸镁]]可用做润滑剂，被加在肥皂中和轻柔磨砂膏中，也可以用做肥料。

== 生理作用 ==
[[半胱氨酸|半胱氨酸]]、[[蛋氨酸|蛋氨酸]]、[[同型半胱氨酸|同型半胱氨酸]]和[[牛磺酸|牛磺酸]]等氨基酸和一些常见的[[酶|酶]]含硫，因此硫是所有[[细胞|细胞]]中必不可少的一种元素。在蛋白质中，[[多肽|多肽]]之间的[[二硫键|二硫键]]是蛋白质构造中的重要组成部分。有些[[细菌|细菌]]在一些类似[[光合作用|光合作用]]的过程中使用[[硫化氢|硫化氢]]作为[[电子|电子]]提供物（一般植物使用水来做这个作用）。[[植物|植物]]以硫酸盐的形式吸收硫。无机的硫是[[铁硫蛋白|铁硫蛋白]]的一个组成部分。在[[细胞色素氧化酶|细胞色素氧化酶]]中，硫是一个关键的组成部分。

工业和发电厂燃烧[[煤|煤]]释放出来的大量二氧化硫在空气中与水和氧结合形成硫酸，它造成[[酸雨|酸雨]]降低水和土壤的[[pH值|pH值]]，对许多地区的自然环境造成巨大破坏。

== 历史 ==
在古代人类就已经认识硫了。中国人发明的[[火药|火药]]是[[硝酸钾|硝酸钾]]、[[碳|碳]]和硫的混合物。1770年代[[安托万·拉瓦锡|安托万·拉瓦锡]]证明硫是一种元素。

== 来源 ==
[[File:Fumarole_with_sulfur.jpg|thumb]]地熱氣井與硫磺]]
[[File:北投硫磺礦.jpg|thumb]]氣井的硫磺]]
在自然界中硫主要以硫化物（如[[黄铁矿|黄铁矿]]）和硫酸盐（如[[石膏|石膏]]）的形式出现，在[[热泉|热泉]]和火山地区也有纯的硫存在。除此以外一些矿物如[[辰砂|辰砂]]、[[方铅矿|方铅矿]]、[[闪锌矿|闪锌矿]]和[[辉锑矿|辉锑矿]]等也都是硫化物的矿物。煤和[[石油|石油]]中也含少量硫，这是为什么在燃煤和石油时有[[二氧化硫|二氧化硫]]被释放出来（酸雨）。今天许多国家要求燃烧煤和石油时被释放的二氧化硫要进行回收。这些被回收的硫是今天工业中使用的硫的一个重要来源。此过程通常通过一个叫做“克劳斯工艺”的过程来实现。另一个重要来源是硫矿。

在工业中，最重要的硫的化合物是硫酸。硫酸是所有工业过程中必不可少的一个原材料，因此硫酸的消耗量被看做是一个国家工业化程度的一个指标。在[[美国|美国]]硫酸是所有生产得最多的化合物。

[[木卫一|木卫一]]表面的黄色主要是它的火山释放的硫造成的。[[月球|月球]]上[[阿利斯塔克环形山|阿利斯塔克环形山]]中比较暗的地区可能是硫形成的。在许多[[陨石|陨石]]中有硫。基於人類對外星資源的興趣強烈提升，木衛一的硫磺也曾經是研究考慮開採的對象；然而成本驚人，除了構想以外，並未付諸實行。

== 化合物 ==
许多[[有机物|有机物]]难闻的味道来自于它们分解产生的[[硫化氢|硫化氢]]之类的化合物。这些化合物有一股特别的臭鸡蛋味道。硫化氢的溶液是[[酸性|酸性]]的，与[[金属|金属]]反应形成金属的硫化物。铁的硫化物在大自然中很常见，如[[黄铁矿|黄铁矿]]。黄铁矿有[[半导体|半导体]]的特性。[[方铅矿|方铅矿]]即[[硫化铅|硫化铅]]则是第一种被发现的[[半导体|半导体]]。

聚合的[[氮化硫|氮化硫]]有金属特性，尽管它不含任何金属，这个复合物还显示特别的电学和光学特性。让熔化的硫速凝可以获得无晶态的硫，[[伦琴衍射|伦琴衍射]]显示其中含有由八个硫[[原子|原子]]组成的环。这种硫在室温下不十分稳定，它渐渐恢复为晶体状态。

硫有多种[[氧化物|氧化物]]，除了<chem>SO2</chem>和<chem>SO3</chem>之外，还有<chem>S_{x}O</chem>（x=1,2,5～8）等化合物。

其它重要的硫的化合物有：
* [[连二亚硫酸钠|连二亚硫酸钠]]：<chem>Na2S2O4</chem>，是一种强有力的还原剂
* [[亚硫酸|亚硫酸]]：<chem>H2SO3</chem>，是二氧化硫在水中的溶液。亚硫酸和亚硫酸盐是有力的还原剂。二氧化硫的其它产物包括[[焦亚硫酸|焦亚硫酸]]离子（<chem>{S2O5}^{2-}</chem>）
* [[硫代硫酸|硫代硫酸]]盐：<chem>{S2O3}^{2-}</chem>，是氧化剂。硫代硫酸氨有可能可以代替氰化物来洗金
* [[连二硫酸|连二硫酸]]：<chem>H2S2O6</chem>，及其盐
* [[连多硫酸|连多硫酸]]：<chem>H2S_{n}O6</chem>，''n''可以从3一直到80
* [[硫酸盐|硫酸盐]]：是硫酸的盐
* [[硫化物|硫化物]]：是硫与其它元素的化合物
* [[过一硫酸|过一硫酸]]是三氧化硫与浓的[[过氧化氢|过氧化氢]]的反应物
* [[硫氰酸盐|硫氰酸盐]]是硫氰离子（<chem>SCN-</chem>）的化合物
* [[硫氰|硫氰]]：<chem>(SCN)2</chem>
* [[二硫化碳|二硫化碳]]：一种良好的有机溶剂，通常用于纺织工业，工业上通过甲烷与硫磺反应来生产

== 同位素 ==
硫有18种同位素，其中四种是稳定的：S-32（95.02%）、S-33（0.75%）、S-34（4.21%）和S-36（0.02%），除<chem>^35S</chem>外，其它放射性同位素的半衰期都很短。硫-35由[[宇宙射线|宇宙射线]]射击空气中的[[氩|氩]]-40而导致，其[[半衰期|半衰期]]为87天。

硫化物沉淀时根据温度的不同S-34的含量少许不同。假如在一个矿物中硫化物和[[碳酸盐|碳酸盐]]同时存在的话，那么根据[[碳|碳]]-13和硫-34的含量可以推算出矿物形成时矿水的[[pH值|pH值]]和[[氧|氧]]的[[逸度|逸度]]。

在[[森林|森林]]生态系统中，硫酸盐主要来自空气，少量来自矿物的风化。其中硫的同位素的不同含量可用来确定它们的来历。

== 注意 ==
在接触二硫化碳、硫化氢和二氧化硫时要非常小心。二氧化硫可以在[[肺|肺]]中与水结合成[[亚硫酸|亚硫酸]]，亚硫酸可以导致[[肺出血|肺出血]]和[[窒息|窒息]]。[[硫化氢|硫化氢]]毒性非常高。虽然硫化氢的味道一开始非常强烈，但人的嗅觉很快就被它压抑了。因此受害人有可能未察觉它的存在。
