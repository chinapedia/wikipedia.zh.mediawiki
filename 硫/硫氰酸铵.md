'''硫氰酸銨'''是一種由铵离子和硫氰酸根组成的无机铵盐，分子式為NH<sub>4</sub>SCN，在空气中易[[潮解|潮解]]，溶于水时呈吸热反应，170℃时分解为[[氨|氨]]、[[二硫化碳|二硫化碳]]和[[硫化氢|硫化氢]]，可用於[[除草剂|除草剂]]。

== 用途 ==
硫氰酸銨可用于制造[[除草剂|除草剂]]、[[硫脲|硫脲]]和透明的人造[[树脂|树脂]]。硫氰酸銨也可用于比色法测定软饮料中的铁含量。

== 合成方法 ==
硫氰酸銨是由二硫化碳和氨水反应得到，这个反应会生成中间体[[二硫代氨基甲酸铵|二硫代氨基甲酸铵]]，经加热后分解成硫氰酸銨和[[硫化氢|硫化氢]]：
: CS<sub>2</sub> + 2 NH<sub>3</sub>(aq) → NH<sub>2</sub>C(=S)SNH<sub>4</sub> → NH<sub>4</sub>SCN + H<sub>2</sub>S
