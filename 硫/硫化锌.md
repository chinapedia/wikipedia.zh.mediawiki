'''硫化锌'''（[[化学式|化学式]]：[[锌|Zn]][[硫|S]]）是[[锌|锌]]的[[硫化物|硫化物]]，为白色至黄色粉末或晶体，难溶于水，主要以[[闪锌矿|闪锌矿]]和[[纤锌矿|纤锌矿]]的形式存在。<ref name = "Wells"> Wells A.F. (1984) ''Structural Inorganic Chemistry'' 5th edition Oxford Science Publications ISBN 0-19-855370-6 </ref>这两种结构都为宽[[禁带|禁带]][[半导体|半导体]]材料，在光电子器件中有广泛应用。闪锌矿结构为[[立方晶系|立方晶系]]，在300K时的禁带宽度为3.54[[电子伏特|eV]]；纤锌矿结构为[[六方晶系|六方晶系]]，禁带宽度为3.91eV。纯的闪锌矿型会在1020°C时转变为纤锌矿型，但杂质的存在会使温度降低。

硫原子在闪锌矿中为[[立方紧密堆积|立方紧密堆积]]，在纤锌矿中为[[六方紧密堆积|六方紧密堆积]]；两种情况下，锌原子都占一半的四面体空隙。

==制备==
硫化锌可通过向含Zn<sup>2+</sup>的溶液中通入[[硫化氢|硫化氢]]，并控制pH为3得到。
: Zn<sup>2+</sup> + H<sub>2</sub>S → ZnS↓ + 2 H<sup>+</sup>

单质化合也会产生硫化锌。
: Zn + S → ZnS

黄原酸锌在氨性条件下的热分解也能得到硫化锌。<ref>廉会良. 重金属黄原酸盐的热分解机理研究[D]. 济南大学, 2013. pp. 31</ref>

==化学性质==
硫化锌和卤素反应，生成卤化锌和硫或卤化硫。

==用途==
硫化锌主要用作颜料和填料，硫化锌和[[硫酸钡|硫酸钡]]的混合物俗称[[锌钡白|锌钡白]]或[[立德粉|立德粉]]，是一种很常用的白色颜料，不过已被[[钛白粉|钛白粉]]所大量取代。硫化锌也可用作光学材料和[[荧光粉|荧光粉]]的发光成分。
