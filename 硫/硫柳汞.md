'''硫柳汞'''或'''邻乙汞硫基苯酸钠'''（[[INN|INN]]：Thiomersal），是一種[[有機|有機]][[汞化合物|汞化合物]]，用途為[[抗菌|抗菌]]和抗[[真菌|真菌]]劑，被用于疫苗、皮肤测试剂、纹身药水等液体的防腐。

其用于疫苗防腐引发争议，有人声称这会导致儿童汞中毒和自闭症，引发部分公众恐慌。但官方宣称并没有合理的证据证明疫苗中微量的硫柳汞对人体有任何害处。目前美国、加拿大及一些欧洲国家已经不再将其用作疫苗防腐。<ref>{{cite web |title = Thimerosal in Vaccines and Autism加拿大官方解释 |url = http://www.phac-aspc.gc.ca/im/q_a_thimerosal-eng.php |accessdate = 2014-06-05 |deadurl = yes |archiveurl = https://web.archive.org/web/20140709223525/http://www.phac-aspc.gc.ca/im/q_a_thimerosal-eng.php |archivedate = 2014-07-09 }}</ref>

{{link-en|小罗伯特·肯尼迪|Robert F. Kennedy, Jr.}}發表硫柳汞（一種含有有機[[水銀|水銀]]的[[疫苗|疫苗]][[防腐劑|防腐劑]]）可能和[[自閉症|自閉症]]有關<ref>{{cite web |title = 疫苗：致命的免疫（Vaccinations: Deadly Immunity） |url = http://www.globalresearch.ca/vaccinations-deadly-immunity/14510 |accessdate = 2013-05-06 }}</ref>。他分析了水銀中毒和自閉症的病徵相似之處，也表示[[安曼教派|安曼教派]]兒童沒有接受防疫注射，自閉症發病率很低。此研究沒有任何流行病學數據支持。可是，[[美國疾病控制與預防中心|美國疾病控制與預防中心]]回應此研究，指硫柳汞不似是自閉症的原因。但基於公眾憂慮，疾病管制中心、食物及藥物管理局，及國家健康署共同於一九九九年發表聲明，要求藥廠停止使用硫柳汞作為儿童疫苗防腐之用，但在成人疫苗中还有使用。

Geier等發表了十一份研究，指出自閉症和兒童接種疫苗有關。他根據美國Vaccine Adverse Event Reporting System (VAERS)數據進行分析，指出美國防疫計劃使用沒有含有硫柳汞的疫苗後，美國兒童的自閉症發病率有所下降。美國兒科協會嚴重指摘此研究，表示VAERS的數據有偏頗，不能用於流行病學研究。Madsen的研究也發現，就算丹麥於1992年停用硫柳汞，自閉症的發病率也不跌反升。
