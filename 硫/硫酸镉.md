'''硫酸镉'''，化学式CdSO<sub>4</sub>，常为水合物 3CdSO<sub>4</sub>·8H<sub>2</sub>O 的形式。

== 性质 ==
无色无气味[[单斜|单斜]]结晶，溶于水，不溶于[[乙醇|乙醇]]、[[乙醚|乙醚]]和[[乙酸|乙酸]]。易[[风化|风化]]。剧毒！

== 制备 ==
1、金属[[镉|镉]]、蒸馏水和[[硫酸|硫酸]]在反应器中，30％[[过氧化氢|过氧化氢]]存在下进行反应，然后再经过滤、浓缩结晶、离心分离和干燥，制得硫酸镉成品。
:<math>\rm \ Cd+H_2SO_4 \ \xrightarrow {H_2O_2} \ CdSO_4+2H_2O</math>

2、[[氧化镉|氧化镉]]溶于硫酸，除杂质，经蒸发结晶、离心分离、50 °C以下干燥，制得硫酸镉。
:<math>\rm \ CdO+H_2SO_4\rightarrow CdSO_4+H_2O</math>

== 用途 ==
塑料工业中用作[[聚氯乙烯|聚氯乙烯]]的防老剂。电池工业中用作[[镉电池|镉电池]]、[[韦斯顿电池|韦斯顿电池]]和其他标准电池中的电解质。医药工业中用作[[角膜炎|角膜炎]]等洗眼水中的防腐剂和收敛剂。化学分析中，用作[[马氏试砷法|马氏试砷法]]中的催化剂，以用于检测[[硫化氢|硫化氢]]和[[反丁烯二酸|反丁烯二酸]]；还用于标准镉元素和其他镉盐的制造。也用于镉肥生产。

== 毒性 ==
硫酸镉为剧毒物，其粉尘或烟尘可引起中毒，影响呼吸系统、消化系统和神经系统。工作人员操作时，需带强制送空气的软管式防毒面具、防护眼镜，穿着光滑织品制成的无缝长工作服或连衫裤，戴橡皮手套和围裙。操作期间和操作后漱口。
