'''硫酸铝'''（[[化学式|化学式]]：Al<sub>2</sub>(SO<sub>4</sub>)<sub>3</sub>）是一个被广泛运用的工业试剂，通常会与[[明矾|明矾]]混淆。硫酸铝通常被作为[[絮凝|絮凝]]剂，用于提纯[[饮用水|饮用水]]及[[污水处理|污水处理]]设备当中，也用于[[造纸|造纸]]工业。 

自然状况下，硫酸铝几乎不以无水盐形式存在。它会形成一系列的水合物，其中十六水硫酸铝是最常见的。

硫酸铝也是一种很有效的[[软体动物|软体动物]][[杀虫剂|杀虫剂]]，能杀灭西班牙[[鼻涕虫|鼻涕虫]]（Spanish slugs）。

== 实验室制备 ==

硫酸铝可以通过将[[氢氧化铝|氢氧化铝]]（Al(OH)<sub>3</sub>）投入[[硫酸|硫酸]]（H<sub>2</sub>SO<sub>4</sub>）中制备。

:2Al(OH)<sub>3</sub> + 3H<sub>2</sub>SO<sub>4</sub> + 10H<sub>2</sub>O → Al<sub>2</sub>(SO<sub>4</sub>)<sub>3</sub>·16H<sub>2</sub>O

== 用途 ==

*在[[纺织品|纺织品]]的印染中作为媒介。由於Al<sup>3+</sup>會和SO<sub>4</sub><sup>2-</sup>水解出[[氫氧化鋁|氫氧化鋁]][[膠體|膠體]]，因此硫酸铝常用于[[净水|净水]]，在水的净化中，它使杂质凝结，更容易[[沉淀|沉淀]]和[[过滤|过滤]]。
* 当溶解于大量[[中性|中性]]或微[[碱性|碱性]]的水中时，产生[[胶体|胶体]]沉淀[[氢氧化铝|氢氧化铝]]，Al(OH)<sub>3</sub>。在印染布料时，氢氧化铝胶体使得染料更容易附着于[[植物|植物]][[纤维|纤维]]之上。
* 硫酸铝也被用来调节[[土壤|土壤]][[pH|pH]]值，因为它[[水解|水解]]生成氢氧化铝的同时产生少量的[[硫酸|硫酸]]稀溶液。
