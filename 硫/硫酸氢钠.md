'''硫酸氢钠'''（[[化学式|化学式]]：{{化學式|鈉|氫|硫|氧|4}}），也称'''酸式硫酸钠'''，硝石餅（niter cake） 。它的无水物有吸湿性。硫酸氢钠在达到其熔点前即分解为[[焦硫酸钠|焦硫酸钠]]，故其熔点很难测定。

硫酸氢钠的水溶液显[[酸性|酸性]]，1mol/L溶液的[[pH|pH]]大约为1.4。在某些情况下，硫酸氢钠的溶液可以代替39.0-42.0%的[[硫酸|硫酸]]溶液使用<ref name="Acid">{{cite web|url=http://www.sciencelab.com/page/S/PVAR/10426/SLS4258|title=Sodium Hydrogen Sulfate|accessdate=2008-10-19}}</ref>。比如，蒸馏硫酸氢钠和[[乙酸钠|乙酸钠]]的混合溶液可以得到[[乙酸|乙酸]]，硫酸氢钠溶液也可以与大部分的[[碳酸盐|碳酸盐]]反应并生成[[二氧化碳|二氧化碳]]。

在一定程度上，硫酸氢钠表现出硫酸和[[硫酸钠|硫酸钠]]混合物的性质。这一点在硫酸氢钠溶于[[乙醇|乙醇]]时就能体现出来（它会分解成溶于乙醇的硫酸和不溶于乙醇的硫酸钠）。<ref name="merck">{{cite book|title=Merck Index of Chemicals and Drugs|edition=9th ed}}</ref>

: 2NaHSO<sub>4</sub> → Na<sub>2</sub>SO<sub>4</sub> + H<sub>2</sub>SO<sub>4</sub>

== 生产 ==

硫酸氢钠可通过两种方法获得。

* 混合等[[物质的量|物质的量]]的[[氢氧化钠|氢氧化钠]]和硫酸，可以得到硫酸氢钠和[[水|水]]。

: NaOH + H<sub>2</sub>SO<sub>4</sub> → NaHSO<sub>4</sub> + H<sub>2</sub>O

* [[氯化钠|氯化钠]]（食盐）和硫酸可在高温下反应，生成硫酸氢钠和[[氯化氢|氯化氢]]气体。

: NaCl + H<sub>2</sub>SO<sub>4</sub> → NaHSO<sub>4</sub> + HCl

液态的硫酸氢钠会被喷洒到空中，在空中它冷却并凝固成固态的硫酸氢钠颗粒。副产物氯化氢可以溶解在水中而形成[[盐酸|盐酸]]。

== 用途 ==

* 家用清洁剂（45%溶液）；
* 金属[[银|银]]的提取；
* 降低游泳池水的[[碱度|碱度]]；
* 宠物食品；<ref name="pet">{{cite web|url=http://www.chickensoupforthepetloverssoul.com/products/cats/dry_food/adult_cat_formula/|title=Adult Cat Formula|accessdate=2008-10-19|deadurl=yes|archiveurl=https://web.archive.org/web/20081008142244/http://www.chickensoupforthepetloverssoul.com/products/cats/dry_food/adult_cat_formula/|archivedate=2008-10-08}}</ref>
* 在实验室分析土壤和水的样本时作防腐剂；
* 在实验室中用于制取硫酸。
* 中和鹼性廢料
