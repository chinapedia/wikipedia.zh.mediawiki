'''硫化氫'''是[[无机化合物|无机化合物]]，[[化學式|化學式]]為H<sub>2</sub>S。正常是無色、[[易燃|易燃]]的[[酸性|酸性]][[氣體|氣體]]，氢又从水里逸出。硫化氢是[[毒性|急性劇毒]]，吸入少量高濃度硫化氫可於短時間內致命。低濃度的硫化氫對[[眼|眼]]、[[呼吸系統|呼吸系統]]及[[中樞神經|中樞神經]]都有影響。

== 分佈 ==
硫化氫自然存在於[[原油|原油]]、[[天然氣|天然氣]]、[[火山|火山]]氣體和[[溫泉|溫泉]]之中。少量的硫化氫在原油中、但天然氣可以包含高達90％硫化氫；也可以在[[細菌|細菌]]於缺氧狀態下分解[[有機物|有機物]]的過程中產生，或者是一部分的腐敗海藻中常見。人體釋放出的[[屁|屁]]含有極小量（少於屁成分的1%）的硫化氢。

== 制备 ==
实验室一般使用
: <math>\mathrm{FeS+2HCl \longrightarrow FeCl_2+H_2S}</math>
来制取硫化氢。<br/>
也可以用
: <math>\mathrm{FeS+H_2SO_4 \longrightarrow FeSO_4+H_2S}</math>
但还有一个更加简单的方法，[[硫化铝|硫化铝]]（白色）的水解<ref>《无机化学》丛书.第二卷.§2.5含Al-S、Al-Se键的化合物及碲化物.P461</ref>：
: <math>\mathrm{6H_2O+Al_2S_3=3H_2S+2Al(OH)_3}</math>
此方法是利用了活泼金属（[[碱金属|碱金属]]除外）硫化物的[[水解|水解]]。[[硒化铝|硒化铝]]（灰色）和[[碲化铝|碲化铝]]（暗灰色）也能和水迅速反应，生成[[硒化氢|硒化氢]]和[[碲化氢|碲化氢]]。

== 性质 ==
=== 物理性質 ===
2015年，物理學者發現，硫化氫在溫度203K （-70 °C）、極度高壓的環境下(至少150GPa，也就是約150萬標準大氣壓）會發生[[超導|超導]][[相變|相變]]，是目前已知最高溫度的[[超導體|超導體]]<ref name=cart15>{{cite web|last1=Cartlidge|first1=Edwin|title=Superconductivity record sparks wave of follow-up physics|url=http://www.nature.com/news/superconductivity-record-sparks-wave-of-follow-up-physics-1.18191|website=Nature News|accessdate=18 August 2015|date=18 August 2015}}</ref>。

=== 化學性質 ===
硫化氢比同族的水不稳定，加热高于700K时即发生分解。

硫化氫比[[空氣|空氣]]稍重，硫化氫和空氣的混合物极易爆炸。硫化氫和氧氣燃燒會產生藍色火焰，形成[[二氧化硫|二氧化硫]]和[[水|水]]。在一般化学反应里，硫化氫是一种[[還原劑|還原劑]]，如硫化氫可與二氧化硫反應形成單質硫和水。
: <math>\rm SO_2 + 2H_2S {=\!=\!=\!=} 3S + 2H_2O</math>
{{citation needed|工业上可以用此反应制造高纯度[[硫磺|硫磺]]；也可以與硫酸反應：}}
{{citation needed|:<math>\rm H_2SO_4 + H_2S {=\!=\!=\!=} SO_2\uparrow + 2H_2O + S\downarrow</math>}}

硫化氫可微溶於水，形成弱酸，称为“'''氢硫酸'''”。其水溶液包含了氢硫酸根HS<sup>-</sup>（在摄氏18度、浓度为0.01-0.1摩/公升的溶液里，[[酸度系数|p''K<sub>a</sub>'']] = 6.9）和离子硫S<sup>2-</sup>（p''K<sub>a</sub>'' = 11.96）。一开始清澈的氢硫酸置放一段时间后会变得混浊，这是因为氢硫酸会和溶解在水中的氧起缓慢的反应，产生不溶于水的单质硫。

硫化氢和金属离子接触会形成硫化金屬，硫化金屬往往是深暗色的。用於檢測硫化氫的[[醋酸鉛|醋酸鉛]]紙和硫化氢的气体接触时会產生灰色的[[硫化鉛|硫化鉛]]（II）。硫化金屬與強酸反應时则会释放出硫化氫。

此外，{{citation needed|氣態硫化氫和濃[[硝酸|硝酸]]接觸时會发生爆炸}}。與[[醇|醇]]反應则形成[[硫醇|硫醇]]。
: <math>\rm 6HNO_3 + H_2S {=\!=\!=\!=} SO_2\uparrow + 6NO_2\uparrow + 4H_2O</math>

硫化氫是[[酸性|酸性]]的，它與[[鹼|鹼]]及一些[[金屬|金屬]]（如[[銀|銀]]）有化學反應。例如：硫化氫和[[銀|銀]]接觸後，會產生黑褐色的[[硫化銀|硫化銀]]，如果[[氧|氧]]气存在反应趋势更大：
: <math>\rm H_2S + 2Ag {=\!=\!=\!=} Ag_2S + H_2 \uparrow</math>
该反应能够进行的原因是硫化银的[[生成自由能|生成自由能]]负值很大，增大了正方向反应的趋势。
