'''硫醇'''或'''巰'''('''Thiol'''）是包含[[巯基|巯基]][[官能团|官能团]]（-SH）的一类非[[芳香化合物|芳香化合物]]，是一类[[有机硫化合物|有机硫化合物]]。可以看成[[醇|醇]]中的[[氧|氧]]原子被[[硫|硫]]原子替换。

==结构和成键==
硫醇中，硫原子为不等性sp<sup>3</sup>杂化态，两个单电子占据的sp<sup>3</sup>[[杂化轨道|杂化轨道]]分别与[[烃基|烃基]]碳和氢形成[[σ键|σ键]]，还有两对孤对电子占据另外的两个sp<sup>3</sup>杂化轨道。由于硫的3s和3p轨道形成的杂化轨道比氧的2s和2p轨道形成的杂化轨道大，故C-S和S-H键分别比C-O和O-H键长。在[[甲硫醇|甲硫醇]]中C-S和S-H键键长分别为0.182 nm和0.134 nm，都比[[甲醇|甲醇]]中的C-O和O-H键长大。∠CSH则为96°，小于∠COH。

硫的[[电负性|电负性]]比氧小，所以硫醇的[[偶极矩|偶极矩]]也比相应的醇小。

==物理性质==
除[[甲硫醇|甲硫醇]]在室温下为气体外，其他硫醇均为液体或固体。硫醇分子间有偶极吸引力，但小于醇分子间的偶极吸引力，且硫醇分子间无明显的[[氢键|氢键]]作用，也无明显的缔合作用。因此，硫醇的沸点比分子量相近的[[烷烃|烷烃]]高，比分子量相近的醇低，与分子量相近的[[硫醚|硫醚]]相似。

硫醇与水间不能很好地形成氢键，所以硫醇在水中的溶解度比相应的醇小得多。常温下，[[乙硫醇|乙硫醇]]在水中的溶解度仅为1.5g/100mL。

低级的硫醇有强烈且令人厌恶的气味，乙硫醇的臭味尤其明显，所以常用乙硫醇作为[[天然气|天然气]]中的警觉剂，用以警示天然气泄漏。不过随着分子量的增加，硫醇的臭味渐弱，九碳以上的硫醇则有令人愉快的气味。

==化学性质==
巯基是硫醇化学性质的主要体现。其中S-H键涉及硫较大的3s/3p组成的杂化轨道与氢较小的1s轨道成键，所以S-H键较弱，硫醇具有酸性。硫上还有孤对电子，所以巯基也可被氧化。

硫醇的酸性比相应的醇强，可溶于氢氧化钠的乙醇溶液中生成比较稳定的盐，通入二氧化碳又变回硫醇。硫醇可与一些重金属盐生成不溶于水的硫醇盐，两者[[软硬酸碱理论|软软相吸]]。许多重金属离子在体内的毒性即是因为其可与生物分子的巯基结合。另一方面，也可利用硫醇（如[[二巯基丙酸|二巯基丙酸]]）通过形成不溶沉淀的方法将重金属离子从尿液排出，起到解毒作用。

硫醇很容易被氧化。弱氧化剂（如空气、碘、氧化铁、二氧化锰等）即可将硫醇氧化为[[二硫化物|二硫化物]]。硫醇与二硫化物形成的氧还共轭对是生物体内的常见机制，如[[半胱氨酸|半胱氨酸]]-[[胱氨酸|胱氨酸]]还氧对。生成的二硫化物中的[[二硫键|二硫键]]在维持[[蛋白质|蛋白质]]空间结构方面有重要作用。
:<math>\mathrm{\ 2RSH \ \xrightarrow{[O]} \ RSSR+H_2O}</math>

硫醇用强氧化剂（如[[高锰酸钾|高锰酸钾]]、[[硝酸|硝酸]]、[[高碘酸|高碘酸]]）氧化，经过中间物[[次磺酸|次磺酸]]、[[亚磺酸|亚磺酸]]，最终生成[[磺酸|磺酸]]。此法可用于脂肪磺酸的制备。
:<math>\mathrm{\ RSH \to [ RSOH \to RS(O)OH ] \to RS(O)_2OH}</math>

对硫醇[[催化加氢|催化加氢]]，可实现脱硫，产生相应的[[烃|烃]]。石油炼制中的加氢[[脱硫|脱硫]]即是基于此反应。石油中有少量硫醇，硫醇的存在不仅会使汽油具有令人讨厌的气味，还会在燃烧时转变为有毒、腐蚀性的[[二氧化硫|二氧化硫]]和[[三氧化硫|三氧化硫]]。
:<math>\mathrm{\ RSH+H_2 \ \xrightarrow[\Delta]{MoS_2} \ RH+H_2S}</math>

此外，硫醇还可发生一些与醇相似的反应，例如与[[羧酸|羧酸]]生成[[硫醇酯|硫醇酯]]，与[[醛|醛]]、[[酮|酮]]生成[[缩硫醛|缩硫醛]][[缩硫酮|酮]]。后一反应用于在有机合成中保护羰基或除去羰基，或实现羰基的[[极性转换|极性转换]]。

==制取==
硫醇可通过[[卤代烃|卤代烃]]与[[硫氢化钠|硫氢化钠]]（或其他碱金属硫氢盐）起[[SN2|S<sub>N</sub>2反应]]得到：
:<math>\mathrm{\ RX + NaSH \ \xrightarrow{C_2H_5OH} \ RSH + NaX}</math>

生成的硫醇与硫氢盐进一步反应，产生硫醇盐：
:<math>\mathrm{\ RSH+HS^-Na^+ \rightleftharpoons RS^-Na^+ + H_2S}</math>

不过硫醇盐中的烃硫负离子是很强的[[亲核试剂|亲核试剂]]，易与卤代烷反应生成副产物[[硫醚|硫醚]]：
:<math>\mathrm{\ RS^-+RX\to RSR+X^-}</math>

所以必须使用过量的硫氢盐以避免副反应的发生。

硫醇也可通过[[二硫化物|二硫化物]]（R-S-S-R'）还原得到。还原剂可以是[[氢化铝锂|氢化铝锂]]、[[液氨|液氨]]中的金属锂，或锌加酸（如[[乙酸|乙酸]]）。
:<math>\mathrm{\ R-S-S-R' \to RSH + HSR'}</math>

另外，用[[硫脲|硫脲]]与卤代烃反应，生成S-烷基异硫脲盐，后者易水解生成[[尿素|尿素]]和硫醇，也是实验室制备硫醇的常用的方法，反应产率高。
:<math>\mathrm{\ RX+H_2NC(=S)NH_2\to RSC(=NH_2^+X^-)NH_2 \ \xrightarrow{H_2O/NaOH} \ RSH+H_2NCONH_2}</math>
