'''硫酸铅'''，[[分子式|分子式]]為{{化學式|铅|硫|氧|4}}，白色[[单斜|单斜]]或[[正交晶系|正交]]晶体，难溶于水，有毒。是[[铅矾|铅矾]]或硫酸铅矿的主要成分，熔点1443K，K<sub>sp</sub>=1.06×10<sup>−8</sup>，主要用于白色油漆颜料、[[铅蓄电池|铅蓄电池]]等。

==性质==
可溶于[[硝酸|硝酸]]、[[浓硫酸|浓硫酸]]、饱和 [[乙酸铵|NH<sub>4</sub>Ac]] 及[[苛性碱|苛性碱]]：

:<math>\rm \ PbSO_4+HNO_3\rightarrow HSO_4^-+Pb(NO_3)^+</math>
:<math>\rm \ PbSO_4+H_2SO_4 \rightarrow Pb(HSO_4)_2</math>
:<math>\rm \ PbSO_4+3Ac^-\rightarrow PbAc_3^-+SO_4^{2-}</math>
:<math>\rm \ PbSO_4+3OH^- \rightarrow [Pb(OH)_3]^-+SO_4^{2-}</math>

加热时可被[[氢气|氢气]]、[[碳|碳]]、[[一氧化碳|一氧化碳]]和[[碱金属|碱金属]]等[[还原剂|还原剂]]还原为铅单质或[[硫化铅|硫化铅]]。

[[碳酸钠|碳酸钠]]（[[碳酸钾|钾]]）水溶液与可溶性[[铬酸盐|铬酸盐]]水溶液与固态硫酸铅的反应，可以很快达到平衡，常温下较短时间内硫酸铅就能被完全转化为[[碳酸铅|碳酸铅]]沉淀或[[铬酸铅|铬酸铅]]沉淀。

==制备方法==
#由[[硝酸铅|硝酸铅]]和[[硫酸盐|硫酸盐]]反应：
#:<math>\rm Pb(NO_3)_2+Na_2SO_4 \rightarrow PbSO_4 \downarrow +2NaNO_3</math>
#由 [[一氧化铅|PbO]] 和 [[硫酸|H<sub>2</sub>SO<sub>4</sub>]] 反应：
#:<math>\rm \ PbO+H_2SO_4 \rightarrow PbSO_4 \downarrow +H_2O</math>
#铅溶于较浓的硫酸后用水稀释：
#:<math>\rm \ Pb+H_2SO_4 \rightarrow PbSO_4 \downarrow +H_2\uparrow</math>
#[[过氧化氢|过氧化氢]]氧化[[硫化铅|硫化铅]]：
#:<math>\rm \ PbS+4H_2O_2 \rightarrow PbSO_4 \downarrow +4H_2O</math>
#电解二水合 [[硫酸氢铅|Pb(HSO<sub>4</sub>)<sub>2</sub>]] 硫酸溶液，用铅作阳极，浓缩阳极液制得。

==用途==
[[铅酸蓄电池|铅酸蓄电池]]：<math>(-)\rm \ Pb, PbSO_4\mid H_2SO_4\mid PbSO_4, PbO_2(+)</math>

:<math>\rm \ (+) \ PbO_2+HSO_4^-+3H^++2e^-\rightarrow PbSO_4+2H_2O</math>
:<math>\rm \ (-) \ Pb+HSO_4^-\rightarrow PbSO_4+H^++2e^-</math>

总反应为：<math>\rm \ Pb+PbO_2+2H_2SO_4 \rightarrow 2PbSO_4+2H_2O</math>（反方向为充电时反应）
