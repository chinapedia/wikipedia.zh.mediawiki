'''床垫'''（又称'''床褥'''）是一个和床大小差不多的垫子，放在床上，在睡眠时支撑人的背部，使人[[睡眠|睡眠]]时躺卧舒适。有时床垫也可单独做床来使用。床垫可以用毛、[[稻草|稻草]]、[[棉花|棉花]]、[[化纤|化纤]]、[[泡沫橡胶|泡沫橡胶]]、[[羽毛|羽毛]]、[[弹簧|弹簧]]等填充；充气床垫用空气填充；[[水床|水床]]用水来填充。欧洲人的床垫是[[十字军东征|十字军东征]]的时候从[[阿拉伯人|阿拉伯人]]那里学到的，而阿拉伯文中的“matrah”演变成了现在英文中的“Mattress”，法文中的“Matelas”。

床垫一般放在[[床架|床架]]上，这样可以减少床垫与地面的摩擦，延长床垫寿命。早期床垫用自然材料填充，如羽毛、马毛、稻草、棉花。在亚洲人们用竹子做床垫，在东南亚人们用木棉来做床垫。20世纪发达国家的床垫逐渐开始用弹簧、棉絮和合成纤维来填充。有了密闭材料，水床和充气床也相继出现。目前北美的床垫多用弹簧、合成纤维和合成橡胶来使床垫柔软有弹性。而在欧洲聚氨酯发泡塑料和橡胶填充也很流行。
==历史==
*1881年，美国德克萨斯州休斯敦郊外一个叫“丝涟”的小镇，一位名叫Daniel Haynes的英国裔扎棉机工匠开始了一种棉塞床垫的生产。

*1899年，英國Vi-spring詹姆斯‧马歇尔（James Marshall）设计了一套手工制造的承托系统：獨立筒袋裝彈簧的原型，螺旋弹簧以未经漂白的棉布包裹、以马尾毛绗缝以及配以床垫外拉扣。

*1900年，美.西蒙斯Simmons，註冊袋装獨立筒弹簧的專利；1901年發明了量產獨立筒的機器。

*二十世纪初，邓禄普Dunlopilla，橡胶发泡软垫；

*1932年，意大利，马里奥·贝里纪，以聚氨基甲酸泡沫用于垫子；

*第二次世界大战时，德国，路易斯·考拉尼，还设计了一种椭圆形床垫，并按人体躺在床垫上不同部位的体压，分成软、中、硬的三个部分，使人体骨骼保持最佳状态。

*70～80年代的美国开始流行注水床垫，但是在90年代中叶，因为其过渡的可塑性造成年轻人脊椎问题频发，被慢慢退出了主流市场。

*90年代后，随着航天科技的民用化趋势，原先用作高级汽车座椅和航天材料的Visco Mind（记忆绵）材料慢慢成为欧美市场的主流。

==类型==
市面上比较常见的床垫按照内芯材料分类主要包括：

<big>椰棕床垫（中式）——Coconut Mattress''</big>
是迄今为止历史最悠久的床垫，历史可以追溯到南宋时期。
就是用椰子树和棕榈树皮编织成的板状支撑网制成的床垫。市面上较多的是棕榈树和椰树的材料，也有用树藤编制的。

<big>网式弹簧床垫（英式）——Bonnel Mattress</big>
1881年由旅美英国人在美国一个叫Sealy的小镇发明，又名板式弹簧床垫，即利用钢丝编织成的箱装结构支撑面料。

<big>袋装弹簧床垫（美式）——Pocket Spring Mattress</big>
發明於1900年，将弹簧装入袋子，再通过排列的固定的方式制成床垫内芯，英国VI-Spring和美国Simmons均聲稱為自己发明首創。

<big>泡绵床垫——Foam Mattress</big>
19世纪初叶由意大利人发明了发泡聚氨酯材料（泡绵）床垫。利用泡绵的独特弹性和柔软度和舒适感，加上意大利的手工传统，在二战后的30年里在欧洲十分流行

<big>水床垫和充气床垫</big>
19世纪80年代发明于美国，利用高强度尼龙材料包裹塑胶内胆，将水或空气注入其中。具有较高的便携性。

<big>乳胶床垫——Latex Mattress</big>
19世纪80年代，东南亚经济的蓬勃发展和乳胶发泡技术的成熟后的产物，类似于海绵床垫，但是其天然的性质，和丰富的弹性，在欧洲非常受欢迎

<big>记忆棉绵床垫——Memory Mattress</big>
在美国航空航天局开发了一种叫做“Memory Foam”的聚氨酯泡沫，用于航天飞机的座椅和卧具之后，其独特的触感和舒适度，让之前的诸多材料望尘莫及。结合了镭射切割、高速线切割工艺和越来越多的人体工程学技术的加入，具有极高可塑性的记忆棉床垫如今占领了欧洲的高端床垫市场<ref>[https://page00.com/mattress/-mattress/ quality mattress manufacturers] {{Wayback|url=https://page00.com/mattress/-mattress/ |date=20161111200806 }}, 中高端床垫品牌 </ref>。

==保养==

如果正確使用床垫，基本上使用十来年沒問題；但若不好好保養，会大大减少使用寿命。

想延长床垫使用寿命，下面幾點要特別注意:<ref>{{Cite web|url=http://www.panbor.com.tw/page_show/5|title=如何保養床墊}}</ref>
# 避免亂坐亂跳：避免長期的坐在床墊邊緣，或是在床上跳來跳去，免得彈簧彈性疲乏。 
# 保持床墊整潔：平時應套上床單或使用保潔墊，避免髒污滲透到床墊內。 
# 臥室避免過於潮濕：台灣一向濕氣重，為避免床墊受潮，除濕機是最佳選擇。
# 不要在床上抽煙：除了避免引發火災之外，另一個原因是床墊內的填充物很容易吸收味道或煙塵，在床上抽煙會加速床墊內的填充物因為吸收到煙塵而加速骯髒。 
# 常換面或換方向：每一段固定的時間，如換季的時候，可以將床墊翻面或換個方向，這樣可避免重量老是分佈在局部的彈簧而加速彈性疲乏。 
平常時候，依照下面方式做基本保養就差不多了:
# 頭六個月內，每個月將床墊翻轉一次，以後每三個月翻轉一次。 這樣可 使床墊受力平均，而不會使某些彈簧因使用頻率過高，而提早出現彈性疲乏。
# 每年用吸塵器清理床墊數次，以保持床墊清潔。需經常將床墊晾曬，以保持睡眠清爽舒適。
