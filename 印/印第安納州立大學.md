'''印第安納州立大學'''（'''Indiana State University'''，縮寫：'''ISU'''）是一所位于美國[[印第安納州|印第安納州]][[特雷霍特|特雷霍特]]的[[公立大學|公立大學]]，成立于1865年<ref>{{cite web|url=http://www.indstate.edu/majors/ |title=Academics | Indiana State University |publisher=Indstate.edu |accessdate=2015-05-21}}</ref>。它是美國中西部的較好的大學之一<ref name=Taylor2014>{{cite news|url=http://www.indstate.edu/news/news.php?newsid=4030|title=Forbes, Princeton Review again list Indiana State among top colleges|last=Taylor|first=Dave|date=2014-08-06|publisher=Indiana State University|accessdate=2014-09-01}}</ref>，原本是一所師範學校，1965年升格為大學。
