'''印度斯坦语'''（'''हिन्दुस्तानी'''，'''{{Nastaliq|2=ur|ہندوستانی}}'''）又称'''印地—乌尔都语'''，是语言学家对[[印度斯坦|印度斯坦]]（即[[南亞|南亞]]北部）的[[印地语|印地语]]、[[乌尔都语|乌尔都语]]及其相應[[方言|方言]]的统称。这些[[語種|語種]]的[[语法|语法]]基本相同，都有共同的基本词汇。

印地语和乌尔都语雖同屬於印度斯坦语，但兩者使用的文字不同，前者使用[[天城文|天城文]]，後者使用[[烏爾都字母|波斯—阿拉伯字母]]<ref>{{cite web |url= http://www.yinduabc.com/book/1664.htm |title=古印度的梵语与当今的印地语有什么联系与区别？|language=zh-cn|publisher=印度之窗|accessdate=2013年10月27日}}</ref>。

== 辞源 ==
'''印度斯坦语'''一词源自“[[印度斯坦|印度斯坦]]”，意即“[[印度教|兴都教徒]]之地”。[[突厥|突厥]]-[[波斯|波斯]][[莫卧儿帝国|莫卧儿]]在入主[[南亚|南亚]]、征服[[印度|印度]]以后，印度斯坦语一度处于[[伊斯兰文化|伊斯兰文化]]的影响之下，并吸收了大量的[[波斯语|波斯]]、[[阿拉伯语|阿拉伯]]和[[突厥语|突厥语]]词汇。随后[[莫卧儿帝国|莫卧儿帝国]]进一步扩张，將该语言作为[[通用语|通用语]]传播到整个[[北印度|北印度]]，并在随后几个世纪中作为主要通用语言而保留下来。这种语言以北印度当地语言为主，参合了很多新的词汇，渐渐成为一种文学语言，随同[[波斯人|波斯人]]一起带入穆斯林宫庭；最后被称为“乌尔都语”（{{IAST|zabān-e urdū}} {{Nastaliq|2=ur|زبان اردو}}  ज़बान-ऐ उर्दू）——即突厥军中语言之意，同时成为一种混杂、高度波斯化的宫庭语言。

19世纪后半叶[[大英帝国|大英帝国]]征服印度以后，印度斯坦语和乌尔都语二个词汇交替使用，并将其定为[[英属印度|英属印度]]官方的行政事务语言，这进一步促成了之后成为印度和巴基斯坦官方语言的地位。1947年的[[印度独立|印度独立]]和随后的[[印巴分治|印巴分治]]，印度共和国将这种语言[[梵语|梵语化]]（以[[天城文|天城文]]書寫）、巴基斯坦将其[[伊斯兰化|伊斯兰化]]（以[[阿拉伯字母|阿拉伯字母]]書寫），分别命名为“'''印地语'''”和“'''乌尔都语'''”，并分别定为国语。至此以后，尽管不规范的印度斯坦方言也就是乌尔都语方言，但乌尔都语不再是印度斯坦语的同义词，它仅指巴基斯坦官方国语的含义。

对于居于[[印度次大陆|印度次大陆]]特别是印、巴二国的民族来说，“印度斯坦语”一词饱含着复杂情感，不同国家的民族有不同感受。但现代社会，对于[[宝莱坞|宝莱坞]]电影电影制片商来说，使用“印度斯坦语”替代印、巴二国各自称之为“印地语”或“乌尔都语”的官方语言，是很中性和均可接受的对二国官方语言的称呼。

=== 印地语 ===
{{main|印地语}}
印度斯坦语诸方言通行于全印度，在现代印度，说到“印度斯坦语”，人们不再认为是通用语的代名词了，一般认为印地语就是印度斯坦语，或者在说到印度斯坦语马上联想到北部的{{link-en|印度古典音乐|Hindustani classical music}}。

在印度独立以后政府通过与[[梵语|梵语]]结合推出“标准印地语”，标准印地语以{{link-en|卡里波利语|Khariboli}}，即[[德里|德里]][[方言|方言]]）为基础，采用源于印度本土的[[天城文|天城文]]，书写上没有使用波斯語字母的乌尔都语流畅。在{{link-en|普鲁沙塔姆·丹顿|Purushottam Das Tandon}}倡导下，标准印地语广泛用于社交和官方场合，并渐渐推广到全印学校教学中。

在[[印度教|印度教]][[圣地|圣地]][[瓦拉那西|瓦拉那西]]，学者们致力于将本地方言（属于印度斯坦语方言即印地语东部方言）梵语化。{{Fact|标准印地语在影视领域不受重视|time=2010-02-24T07:15:41+00:00}}，在新闻传播媒体越来越多被边缘化。

=== 乌尔都语 ===
{{main|乌尔都语}}
乌尔都语雖然是[[巴基斯坦|巴基斯坦]]最少人當成母語的語言，可是是巴基斯坦的[[官方语言|官方语言]]<ref name=xznu>
{{cite web|author=满在江、谢妍、艾佳|date=2011-07-04|url=http://prc.xznu.edu.cn/s/414/t/1943/83/1b/info99099.htm|title=巴基斯坦的语言与民族关系探析|publisher=中國-巴基斯坦教育文化研究中心|accessdate=2013-10-27|deadurl=yes|archiveurl=https://web.archive.org/web/20131029194926/http://prc.xznu.edu.cn/s/414/t/1943/83/1b/info99099.htm|archivedate=2013-10-29}}</ref>，[[南亚|南亚]]主要地区性语言，也是[[印度|印度]][[安得拉邦|安得拉邦]]、[[北方邦|北方邦]]、[[德里|德里首都中央直辖区]]（National Capital Territory of Delhi）、[[查谟和克什米尔邦|查谟和克什米尔邦]]的官方语言。“乌尔都”源自很正式的波斯社交词汇 {{IAST|zabān-e urdū-e mo'alla}}——营地语言，当时士兵侍服莫卧儿主人以后，这种语言出现共同的语音，之后渐渐演变为莫卧儿贵族统治者的宫廷语言，并成为首都德里上流社会的语言，其发展主要受[[德里|德里]]周围的都市如[[勒克瑙|勒克瑙]]、[[拉合尔|拉合尔]]、[[阿格拉|阿格拉]]等莫卧儿王朝乌尔都诗人影响，其书写方式采用了改进的[[阿拉伯字母|阿拉伯字母]]；印巴分治以前，乌尔都语和印度斯坦语的含义完全相同。

=== 市井方言 ===
印度斯坦诸方言在当地称作“巴扎尔语”（{{IAST|Bāzār Hindustānī}}，巴扎尔印度斯坦语），西方人对巴扎尔语的定义为通行于北印度、巴基斯坦各处商贸活动中的印度斯坦语诸方言。这类语言作为[[通用语|通用语言]]和标准的印地语、乌尔都语相比，存在[[语调|语调]]变化、地域差异，由于夹杂各地方言，很像[[皮钦语|皮钦语]]，对应于正式场合使用的文雅标准的印地语、乌尔都语和梵语，巴扎尔语应只是为“市场语言、市井语言”。

== 一种语言的二种方言 ==
印地语和乌尔都语在[[口语|口语]]表达上不过是一种语言的二种[[方言|方言]]而已，但在书写和规范词汇差异巨大，乌尔都语的书写波斯和阿拉伯味浓重，印地语的书写更像梵文，甚至和古印度语相似<ref name=xznu/>。然而这二种语言语法、基本词汇一样，甚至连发音、动词和语序都相同，而且都是从[[古印度语|古印度语]]发展而来。二种语言及其方言都被称作“印度斯坦语”，或者说“印度斯坦语”就是[[印度次大陆|印度次大陆]]的[[通用语|通用语]]，这种语言特别是城市完全可以互通。由于政治的原因无论乌尔都语、梵文化的印地语、地区化印地语带上了国家的烙印，但印度斯坦语作为通用语言将变得更加适用，因为它既没有梵文化，也没被阿拉伯化，更无国家烙印。

把印度斯坦语作为一种语言看待完全可以从[[宝莱坞|宝莱坞]]流行文化获得启示，巴基斯坦人和印度人就是这种语言的使用者，在政治对立的情形下这一地区出现罕见的现象——印度斯坦语和印、巴二国的国语差异变得越来越小。若把印度斯坦语和漢语都分别作为一种语言看待，那么它就是仅次于[[汉语|汉语]]的世界第二大语言，使用人口超過五億人<ref>
{{cite web|url=http://www.india.org.tw/serviceContent.aspx?id=49|title=學習印度語|publisher=印度－台北協會|accessdate=2013-10-27|deadurl=yes|archiveurl=https://web.archive.org/web/20131029190428/http://www.india.org.tw/serviceContent.aspx?id=49|archivedate=2013年10月29日|df=}}</ref>，但也有研究認為印度斯坦语只能排到第四大語言<ref name=xznu/>。

== 书写系统 ==
当代的印度斯坦语使用[[天城文|梵文字母]]或波斯-阿拉伯语字母作为书写体，但历史上曾使用[[凯提文|凯提文]]。现在作为印度斯坦语方言之一的印地语使用[[天城文|梵文字母]]书写，其另外一种主要方言乌尔都语则使用波斯-阿拉伯文的书写格式，其中波斯體[[烏爾都字母|烏爾都字母]]成为乌尔都语规范书写方式。

由于英语全球化以及[[拉丁字母|罗马字母]]的普及，印度斯坦语有时也用罗马字母书写。这种改写体也被称为乌尔都罗马字母。无论[[天城文|梵文体]]和波斯-阿拉伯体的书写爱好者有多么反对，拉丁化的印度斯坦语很流行，特别在网络和年轻的电脑爱好者中渐成时尚。
