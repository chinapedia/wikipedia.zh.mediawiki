'''印洲塘海岸公園'''（{{lang-en|'''Yan Chau Tong Marine Park'''}}）位於[[香港|香港]][[船灣郊野公園|船灣郊野公園]]的東北面<ref name="afcd">{{cite web |url = http://www.afcd.gov.hk/tc_chi/country/cou_vis/cou_vis_mar/cou_vis_mar_des/cou_vis_mar_des_yan.html |title = 指定的海岸公園 |location = 香港 |publisher = [[自然漁農護理署|自然漁農護理署]] |accessdate=2014-04-06}}</ref><ref name="hk marine">{{cite book |title = 香港海洋情報 |publisher=天地圖書出版社 | pages=86-89}}</ref><ref name="geopark">{{cite web |url = http://www.geopark.gov.hk/b5_s4f9.htm |title=幽靜的海岸公園-印洲塘 | accessdate=2014-04-06}}</ref>，總面積約680[[公頃|公頃]]。印洲塘海岸公園由兩個[[海灣|海灣]]組成，比較少的位於[[荔枝窩|荔枝窩]]，用連接[[涌灣咀|涌灣咀]]和[[九蘆頭|九蘆頭]]北部的直線做界限；另外一個海灣的兩條海上界線就分別連接著[[西流江|西流江]]跟[[石岩頭|石岩頭]]、[[往灣洲|往灣洲]]的[[老沙田|老沙田]]與[[蕩排頭|蕩排頭]]。<ref name="hk marine" />

== 地理 ==
印洲塘海岸公園的[[吉澳海|吉澳海]]、[[印洲塘|印洲塘]]和小島的[[海岸線|海岸線]]極為參差不齊，形成了特別的地貌，包括[[海灣|海灣]]、[[岬角|岬角]]、[[半島|半島]]、[[陡崖|陡崖]]和[[沙坑|沙坑]]。在每個沙灘之間，有很多[[岩石|岩石]]和[[島|島]]散佈。有一塊近[[西流江|西流江]]的稱為[[印洲|印洲]]，其形狀尤如[[印|印]]般，此可能為印洲的名師來源。<ref name="hk marine" />

== 物種發現 ==
早期的研究指出，荔枝窩與三椏涌曾經錄到[[矮大葉藻|矮大葉藻]]、[[喜鹽草|喜鹽草]]和[[小喜鹽草|小喜鹽草]]這三種[[海草|海草]]。這些海草床可以作為[[哺育場|哺育場]]，提供一個藏身之所予[[軟體動物|軟體動物]]跟[[魚類|魚類]]的[[幼體|幼體]]。除了作藏身之所外，海草亦都可以作為[[海膽|海膽]]和[[海龜|海龜]]等等[[動物|動物]]的[[食物|食物]]。<ref name="hk marine" />

除了海草之外，[[紅樹|紅樹]]在這個[[海岸公園|海岸公園]]物種比較繁多的地方生長。由於[[紅樹林|紅樹林]]生長需要一些特定的環境和條件，如果[[鹽|鹽]]濃度跜跟[[潮水|潮水]]高低不一，它們就會[[死亡|死亡]]。所以這些[[動植物|動植物]]都要對嚴峻環境具備高度適應力。兩個有代表性的紅樹林分別位於荔枝窩和三椏村，兩個樹林[[面積|面積]]各大約3[[公頃|公頃]]。在荔枝窩可以找到[[香港|香港]]的記錄入面全數八種的真紅樹，而在三椏村亦可找到5種。<ref name="hk marine" />
