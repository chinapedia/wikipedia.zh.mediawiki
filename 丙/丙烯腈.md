'''丙烯腈'''（CH<sub>2</sub>=CH-C≡N）是有刺激性氣味，極易燃的[[有機化學|有機]]液體。[[IUPAC|IUPAC]]名稱是2-[[丙烯|丙烯]][[腈|腈]]（英文：2-propenenitrile）。

== 性质 ==
丙烯腈是无色有刺激性气味的易燃液体。有毒。略溶于水，易溶于一般[[有机溶剂|有机溶剂]]。是一种[[共轭结构|共轭]][[不饱和|不饱和]][[腈|腈]]，分子中的碳－碳[[双键|双键]]和氰基基团可以参与各种反应，如水合为[[丙烯酰胺|丙烯酰胺]]、[[丙烯酸|丙烯酸]]，与双烯体发生[[双烯合成|双烯合成]]，氢化为[[丙腈|丙腈]]、[[丙胺|丙胺]]，与丙烯酰胺发生[[共聚|共聚]]，电解偶联为[[己二腈|己二腈]]，以及用作醇、胺的[[氰乙基化|氰乙基化]]试剂。

== 用途 ==
丙烯腈的最主要的用途是作为单体来生产[[聚合物|聚合物]]，如[[聚丙烯腈|聚丙烯腈]]，[[腈纶纤维|腈纶纤维]]和[[丁腈橡胶|丁腈橡胶]]。有一小部分丙烯腈被用作[[熏蒸剂|熏蒸剂]]。丙烯腈和其衍生物入2-氯-丙烯腈可以发生[[狄尔斯-阿尔德反应|狄尔斯-阿尔德反应]]。

== 生产 ==
在工业上，丙烯腈一般通过[[丙烯|丙烯]]的[[氨氧化|氨氧化]]制取：
:CH<sub>3</sub>-CH=CH<sub>2</sub> + [[氨|NH<sub>3</sub>]] + 1.5 [[氧|O<sub>2</sub>]] → CH<sub>2</sub>=CH-C≡N + 3 [[水分子|H<sub>2</sub>O]]

2001年的世界丙烯腈产量是4百万吨，其中的一半产自美国。

== 安全性 ==
丙烯腈是高度易燃的，很容易被火花點燃，或和某些[[氧化劑|氧化劑]]反应而[[自燃|自燃]]。燃燒中的丙烯腈會釋出[[氮氧化物|氮氧化物]]和[[氫氰酸|氫氰酸]]。在不加入[[阻聚剂|阻聚剂]]的情况下，遇到加熱、[[碱金属|碱金属]]或是暴露於[[紫外光|紫外光]]时，丙烯腈会发生十分剧烈的聚合反应，乃至爆炸。所以丙烯腈需储存在远离明火和会反应的化学物质，具有良好通风条件的柜中。

丙烯腈同時[[有毒|有毒]]。接觸皮膚或眼睛會引發劇烈的刺激. 吸入或長時期的皮膚接觸可能導致思维混亂，不省人事，或死亡。丙烯腈被分類作為一種可能致癌的物質（IARC第2B类），在處理期間應佩帶認可的安全鏡、手套和呼吸器。
