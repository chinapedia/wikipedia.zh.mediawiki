《'''画江湖之不良人'''》是一部中国大陆三维武侠动画，由[[北京若森数字科技有限公司|北京若森数字科技有限公司]]在2014年7月推出，采取每周更新模式，第一部于2015年8月完结，并在2016年7月推出第二部，又與2018年10月推出第三部，目前每周更新一集，尚未完结。主要讲述了在唐朝末年藩镇割据的背景下，神秘少年李星云与其朋友在险恶的江湖之中闯荡的故事。官方称其为中国首部成人三维武侠连续剧。由该剧改编的同名电视剧也将在2016年9月推出。同时亦有改编同名小说与相应手游。

== 概要 ==
=== 设定 ===
动画背景设定在[[唐朝|唐朝]]末年、[[五代十国|五代十国]]时期，天下大乱，藩镇割据，群雄并起。历史中的不良人为主管侦查逮捕的小吏，由不良帅统领{{efn|梁章钜《称谓录·隶》引《说铃续》：“缉事番役，在唐称为不良人，有不良帅主之，即汉之大谁何也。”}}。在动画中，不良帅为唐初著名相师[[袁天罡|袁天罡]]，在服用长生丹后虽达到长生效果，但容貌尽毁。唐亡后，其将不良人解散，安排在各个地区，把名剑[[龍泉劍|龙泉]]交于不良人陆佑劫，等待时机以求东山再起。因国库宝藏在[[黄巢|黄巢]]攻下[[长安|长安]]后消失，所以民间传说在龙泉剑上隐藏着巨大的秘密。此时江湖上还有[[朱友珪|朱友珪]]的玄冥教，[[李嗣源|李嗣源]]的通文馆，[[李茂贞|李茂贞]]的幻音坊三大势力。整部动画中穿插着黄巢起义、朱温篡位、李氏族诛、友珪弑父、沙陀争霸等著名历史事件。[[朱温|朱温]]、[[李克用|李克用]]、李茂贞、李嗣源等历史人物也将纷纷登场<ref>{{Cite web|url = http://cartoon.youth.cn/dmyc/201505/t20150507_6620888.htm|title = 三维成人武侠动画连续剧《画江湖之不良人》|accessdate = 2015-10-06|author = |date = 2015-05-07|publisher = 中国青年网|language = zh-cn|archive-url = https://web.archive.org/web/20150907033714/http://cartoon.youth.cn/dmyc/201505/t20150507_6620888.htm|archive-date = 2015-09-07|dead-url = yes}}</ref>。

== 制作 ==
《画江湖之不良人》的编剧团队继承自《[[侠岚|侠岚]]》，在进行故事创作时参考了《[[旧唐书|旧唐书]]》、《[[新唐书|新唐书]]》以及《[[殘唐五代史演義|残唐五代史演义]]》中的掌故。主角李星云是一个基于史实的基础上虚构的人物，因为[[唐昭宗|唐昭宗]]李晔的后代除了后来的[[唐哀帝|唐哀帝]]李祚外历史并无过多介绍，所以团队虚构出了唐昭宗的第十子，李祚的胞弟——李星云。角色黑白无常的灵感则来自于中国传统文化中的[[黑白无常|黑白无常]]，制作初期设计为双胞胎兄弟，后改为兄妹（非亲兄妹）。配音演员大部分来自[[北京电影学院|北京电影学院]]2003级表演系配音班的毕业生。<ref>{{Cite web|url = http://www.08kan.com/gwk/MjM5NDQ5MDQyNw/202337495/1/da2e7a7a2fb9452018c124a6a4907755.html|title = 布丁娘独家访谈—《画江湖之不良人》制作组|accessdate = 2015-10-06|author = |date = 2014-12-24|publisher = 布丁动画|language = zh-cn}}</ref>
