'''五一九事件'''（[[中国大陆|中国大陆]]亦称為'''五一九惨案'''<ref>{{cite web |url= http://news.xinhuanet.com/sports/2007-12/03/content_7191802.htm |title=资料：中国足球“519惨案”|website=新华网|date=2007-12-03|accessdate=2011-08-25}}</ref>，香港称为'''五一九之役'''），是於1985年5月19日在[[中華人民共和國|中華人民共和國]][[北京工人体育场|北京工人体育场]]进行的[[1986年世界盃亞洲区外圍賽|1986年世界盃亞洲区外圍賽]]，中国国家足球队在最後一輪只需在主場打和香港足球代表队即可晉身[[1986年國際足協世界盃|世界盃決賽周]]亚洲区预选赛下一轮，但[[香港足球代表队|香港足球代表队]]在比赛中以2：1的比分击败[[中国国家足球队|中国国家足球队]]，将中国淘汰出局后，北京發生大型[[球迷骚乱|球迷骚乱]]，為[[中国历史|中国历史]]上首宗球迷骚乱事件<ref>{{cite web |url= http://sports.sohu.com/20110524/n280655272_1.shtml |title=中国球场暴力史:519骚乱难忘 足协无为推波助澜|website=搜狐体育|accessdate=2011-08-25}}</ref>。

==背景==
;1986年世界杯外圍賽前
[[1986年世界杯|1986年世界杯]]亚洲区外圍賽是[[中国国家足球队|中国国家男子足球队]]第3次冲击[[世界杯足球赛|世界杯]]资格，亦是[[香港足球代表队|香港足球代表队]]第4次冲击世界杯参赛资格。此前双方都未曾成功打入世界杯决赛圈。

[[1986年世界盃外圍賽|1986年世界盃外圍賽]]将[[大洋洲|大洋洲]]賽區與亞洲分開，所有亞洲参赛球队分为东西亚两个区，而各区最终将各产生一支总共两支出线队伍代表亚洲参加1986年世界杯。故是次中國隊在外圍賽不須面對來自西亞及大洋洲的對手。中国队与香港队、[[文莱国家足球队|文莱队]]和[[澳门足球代表队|澳门队]]分入了东亚区的第三小组。

中国队在[[1982年世界盃外圍賽|1982年世界盃外圍賽]]是唯一打入最後階段比賽的東亞球隊，最後在附加赛中1 : 2憾负于[[新西兰国家足球队|新西兰队]]，离[[1982年世界杯足球赛|决赛圈]]仅差一步之遥。在[[1984年亞洲盃足球賽|1984年亚洲杯]]上中国队获得亚军，是眾多東亞球隊中成績最好。当时東亞地區的[[日本国家足球队|日本队]]仍是亞洲二流球隊，[[韩国国家足球队|韩国队]]在1984年亚洲杯表现不佳，所以在沒有西亞及大洋洲球隊競爭下，1986年世界杯外圍賽被认为是中国队冲击世界杯的最佳机会。

;外圍賽過程
1985年2月17日，第三小组比赛开始进行，在双方于香港[[政府大球场|政府大球场]]的首回合交锋中，两队相持90分钟最终0：0互交白卷。

==比赛日==
;赛前布置
赛前，中國國內无论媒体还是球迷，或是足球界人士都认为中国队拿下此役不在话下。为此，中国主帅曾雪麟还派出了一个以攻击为主的阵型，场上队员同5月12日在工体6 : 0大胜澳门队的配置完全相同，并且在同澳门队的比赛中场上的11名球员都打满全场。就在赛前准备会上，[[曾雪麟|曾雪麟]]要求队员在前15分钟内必须进球<ref>{{cite web |url= http://sports.163.com/07/0319/03/39TS8I2100051C88.html |title=中国足球大事记之——“五一九事件”|website=网易体育|accessdate=2011-08-25}}</ref>。

香港队主教练[[郭家明|郭家明]]在赛前放话称中国队赢下香港没有问题，不过在训练中却加紧了定位球的训练，赛前指导中也着重要求球员在比赛中要尽量把节奏放慢。

;比赛过程
开球后，中国队便向香港队球门发起猛攻，但是并未获得太大收获。第18分钟，香港获得一个前场任意球，胡國雄撥交张志德命中，香港1 : 0暂时领先。

第31分钟，中国队杨朝晖20米开外一脚远射被香港门将陈云岳扑出，李辉门前凌空补射扳平1:1，雙方亦以此比數完成上半場，形勢對賽和亦可出線的中國隊有利。

但下半场开球后，中国队依然全力向香港队球门发难，却在第60分钟时被香港队打反击，刘荣业在禁区内造成混乱，中国队後衛铲球未果，后排插上的顾锦辉劲射打入一球，香港2 : 1再次在客场领先中国队。

在再次取得领先后，香港队马上进行密集防守并有意消磨时间，以圖保持勝果至完場。中国队在24 : 5的全场射门比率面前却无法攻破香港大门，11次角球也未能建功，最终在补时5分钟后，裁判吹响了终场哨，香港队在客场战胜了中国队，获得了第三小组的小组第一并出线，而中国队只能接受被淘汰出局的命运。

;北京球迷骚乱
赛后，失望的中国球迷开始失去理智，先是從看台上向球場投擲雜物<ref>[http://beyondnewsnet.com/20150422-1986-world-cup-asian-zone-qualifying-round/ 余錦基 談五一九話當年]，《超越新聞網》</ref>，之後一些球迷开始在球場外搗亂，[[足球流氓|足球流氓]]掀翻了球队大巴，並且沿街破坏物品。一些球迷甚至来到了运动员楼下要求曾雪麟出来和他们对话，而后开始要求时任[[中國足协|中國足协]]主席的[[李凤楼|李凤楼]]同他们对话<ref>{{cite web |url= http://news.ifeng.com/sports/gnzq/detail_2010_10/13/2772754_9.shtml |title=中国十大球迷冲突:京豫球迷暴力狂? 5·19震惊世界|website=凤凰网|date=2010-10-13|accessdate==2011-08-25}}</ref>。

第二日中央报刊人民日报消息称事件中共有127人遭到警方逮捕，当时新闻联播也对这场足球骚乱进行了报道<ref>{{cite web |url= http://sports.sina.com.cn/n/2005-05-19/ba1568730.shtml |title=《足球之夜》特别节目：5.19 一个时代的缩影(9)|website=新浪网|date=2005-05-19|accessdate=2011-08-25}}</ref>。

;曾雪麟辭職
[[曾雪麟|曾雪麟]]因为在小组赛最后一轮负于香港而未能小组出线，于当年5月30日引咎辞职。

;30周年紀念
香港於2015年慶祝五一九30周年，先於[[賀歲盃足球賽|賀歲盃足球賽]]舉行一場紀念賽，再於5月19日當晚於[[九龍塘|九龍塘]]舉行30周年晚宴<ref>[http://hk.apple.nextmedia.com/sports/art/20150520/19153622 30周年晚宴戰將聚頭 五一九薪火相傳]，《[[蘋果日報_(香港)|蘋果日報]]》</ref>。主辦晚宴的當年香港隊領隊[[余錦基|余錦基]]並計劃邀請著名國際足球名宿來港，舉行一場元老球星經典賽<ref>[http://beyondnewsnet.com/20150521-519/ 五一九30周年 組港腳會辦經典賽]，《超越新聞網》</ref>。
