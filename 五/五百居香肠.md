'''五百居香肠'''又称'''大名五百居香肠'''、'''五百居'''，是河北[[邯郸|邯郸]]地区汉族的风味传统名菜，邯郸十大名小吃之一，属于[[冀菜系|冀菜系]]。

== 起源 ==
五百居香肠起源于[[清朝|清]][[道光|道光]]元年，创始人[[王湘云|王湘云]]原籍山东[[济南府|济南府]]，后来[[大名府|大名]]谋生。后来他发现大名地区交通便利，物产丰富，商业和饮食业发达。于是开始制作香肠及熟肉制品，因为大名府距济南府约五百里，故取名为“五百居”<ref>{{Cite web |url=http://travel.hebnews.cn/zt/2012/2012-09/29/content_2868168.htm |title=闻名遐迩 邯郸最有名的十大地道小吃 |access-date=2014-10-05 |archive-url=https://web.archive.org/web/20130221210750/http://travel.hebnews.cn/zt/2012/2012-09/29/content_2868168.htm |archive-date=2013-02-21 |dead-url=yes }}</ref>。

五百居香肠香味醇厚浓郁，味道鲜美，色泽纯正。而且还经久耐放，夏天时候也不会被腐蛀，远销海外<ref>[http://www.he.xinhuanet.com/news/2014-03/19/c_119829183.htm 邯郸：五百居香肠远销港澳] {{webarchive|url=https://web.archive.org/web/20141006113202/http://www.he.xinhuanet.com/news/2014-03/19/c_119829183.htm |date=2014-10-06 }}新华网</ref><ref>[http://www.handannews.com.cn/epaper/handrb/html/2013-10/15/content_127253.htm?div=-1 餐饮业路在何方？] {{Wayback|url=http://www.handannews.com.cn/epaper/handrb/html/2013-10/15/content_127253.htm?div=-1# |date=20141006091703 }}邯郸日报社</ref>。

== 評價 ==
五百居香肠为“八大地方风味美食”大名“[[二五八|二五八]]”{{注|“二五八”包括[[二毛烧鸡|二毛烧鸡]]和五百居香肠，[[郭八火烧|郭八火烧]]。}}之一。
