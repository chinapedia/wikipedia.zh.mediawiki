'''五毛蛋争议'''是[[中华人民共和国总理|中华人民共和国总理]][[温家宝|温家宝]]在2011年10月1日到[[北京航空航天大学|北京航空航天大学]]考察期间，在询问饭菜价格时，北航2010级研究生范祺锋告诉温家宝[[鸡蛋|鸡蛋]]是五毛钱一个<ref>{{cite web |url = http://politics.people.com.cn/GB/15805011.html |title = 北航学生讲述吃早饭意外遇温家宝，听温家宝讲菜价 |publisher = 人民网 |date = 2011-09-28 |language = zh-hans}}</ref>，温家宝听到范祺锋的话后称物价已经稳定下来了，进而引发的巨大争议。2011年以来，中国[[消费者价格指数|消费者价格指数]]居高不下，尤其当年9月份的统计，农产品连涨6周<ref>{{cite web |url = http://finance.sina.com.cn/g/20110928/094210554258.shtml |title = 农产品连涨六周 CPI居高不下 |publisher = 财经网 |date = 2011-10-04 |language = zh-hans}}</ref>，在这种情况下，温家宝的这一番谈话<ref>{{cite web |url = http://www.qingdaonews.com/content/2011-10/04/content_8961526.htm |title = 温家宝和大学生聊菜价，五毛钱鸡蛋引网友热议 |publisher = 青岛新闻网 |date = 2011-10-04 |language = zh-hans}}</ref>成为中国国庆期间网民热议的话题，有网民对北航饭堂的价格的真实性提出质疑<ref>{{cite web |url = http://www.actimes.com.au/actimes/plus/view.php?aid=226537 |title = 香港明报：“五毛钱”鸡蛋激发网民争议 |publisher = 澳大利亚时报 |date = 2011-10-05 |language=zh-hans}}</ref>，“[[美国之音|美国之音]]”称，有许多中国网民对温家宝进行调侃，在此之前，已经有许多中国网民抱怨中国总理温家宝只是一个善于装模做样演戏的“[[中国影帝温家宝|影帝]]”，这次网民“演[[双簧|双簧]]”<ref name="mgyy">{{cite web |url = http://www.voanews.com/chinese/news/20111003-Sound-and-Fury-over-an-50-cent-Egg-131072553.html |title = “五毛蛋”见中国 | publisher = 美国之音 |date = 2011年10月4日 |language = zh-hans}}</ref>。之后，五毛蛋也成为了[[新浪微博|新浪微博]]不能搜索的[[敏感词|敏感词]]<ref name="dwyy">{{cite web |url = http://global.dwnews.com/news/2011-10-06/58187511.html |title = “五毛蛋”成中国最新敏感词 |publisher = 多维网 |date = 2011-10-06 |language = zh-hans |access-date = 2011-11-02 |archive-url = https://web.archive.org/web/20111009091639/http://global.dwnews.com/news/2011-10-06/58187511.html |archive-date = 2011-10-09 |dead-url = yes }}</ref>。有网民反映，很多北京的高校的鸡蛋价格是五毛或者更低，大学的食品价格是享受了国家和学校的双重补贴后的价格，并不一定能反映当前市场的价格<ref name="zgsb"/>。事件发生后，许多非大学师生到北航大学食堂购餐，导致北航等高校出现针对鸡蛋等基本食品实行限购措施<ref name="jhsb"/>。

==缘起==
“五毛蛋”争议缘起于2011年10月2日中国官方媒体发布的一条简讯<ref name="mgyy" />：
{{Quote|
<center>'''国庆日，北航学生向温家宝报物价'''</center>
“鸡蛋多少钱一个？”温家宝来到北航学生食堂，向正在吃饭的范祺锋同学问。“一个鸡蛋、一个鸡蛋饼、一块腐乳共2元，鸡蛋好像5毛钱一个。”范祺锋答道。总理又关切地问：“这顿饭一共花了多少钱？”“再加上5个包子，一碗粥，一共花了4元钱。”}}

此新闻一经媒体报道，「五毛蛋」成为中国大陆[[通货膨胀|通货膨胀]]导致高物价的一颗引爆弹，大量网民质疑北航学生所报的物价，很多人坚决不相信北京还有五毛蛋，认为五毛蛋是20世纪的价格。<ref name="mgyy" />同时对温家宝所说的“物价已经稳定下来了”的说法也大加否定<ref>{{cite web |url = http://www.dw-world.de/dw/article/0,,15441507,00.html |title = “五毛蛋”让温家宝“影帝”桂冠失色 |publisher = 德国之声 |date = 2011-10-06 |language = zh-hans}}</ref>。[[新浪微博|新浪微博]]随即将“五毛蛋”设置为不能搜索的[[敏感词|敏感词]]<ref name="mgyy" />。

==网民观点==
网民的观点大致可划分为两大类。反对的人认为，“五毛蛋”是虚无的谎言鸡蛋，据[[多维新闻网|多维新闻网]]归纳，这些网民的观点又可归纳如下<ref name="dwyy"/>：
* （一）第一小类人认为这纯粹是胡扯；
* （二）在食品掺假、造假在中国盛行的背景下，第二小类人认为五毛蛋是合成蛋；
* （三）第三小类人认为是“五毛”蛋，即中国官方雇用的[[五毛党|五毛党]]谎报的鸡蛋价格。 
* （四）甚至有人认为，“五毛蛋”是10月1日9时温家宝在场的时候，中国官方及其官方媒体拿出来让中国公众相信物价稳定的幌子，或者是个鹌鹑蛋。

但也有网民反映，不只北航，很多北京的高校的鸡蛋价格是五毛或者更低，但是大学的食品价格是享受了国家和学校的双重补贴后的价格，不一定具有市场的代表性<ref name="zgsb">{{cite web |url = http://news.chinatimes.com/mainland/17180501/172011100500204.html |title = 溫家寶訪北航 5毛蛋惹爭議 |publisher = 中时电子报 |date = 2011年10月5日 |language = zh-tw |deadurl = yes |archiveurl = https://web.archive.org/web/20111009034327/http://news.chinatimes.com/mainland/17180501/172011100500204.html |archivedate = 2011年10月9日 }}</ref>。

==后续==
在一片质疑声中，还有传言称北京航空航天大学的校长批评学生范祺锋“觉悟不高”，这一说法遭到当事人的否认，而北京航空航天大学宣传部的部长蔡劲松称范祺锋能遇到中国的总理是一件高兴的事，学校对他进行批评是不可能的，网上的言论纯属无稽之谈。同时，蔡劲松解释到事后他问过食堂，北航的鸡蛋确实是5毛一个，之所以这么便宜，是因为大学生的食堂一般都有补助，除了中央政府给予补助外，学校自身也给予一定补助。因此，学校的饭菜相对于外界比较便宜，所以对外一般也不办理饭卡<ref>{{cite web |url = http://news.xinhuanet.com/politics/2011-10/04/c_122118835.htm |title = 北航确认鸡蛋5毛一个，当事人否认遭到校长批评 |publisher=新华网 | date=2011年10月4日 | language=zh-hans}}</ref>，但每人一次只能限购两个<ref name="jhsb">{{cite web |url = http://www.chinadaily.com.cn/dfpd/shehui/2011-10/13/content_13882850.htm |title = 北航限购“五毛鸡蛋” 一次最多购买两个 |publisher = 京华时报 |date = 2011年10月13日 |language = zh-hans}}</ref>，后来很多大学的学生都在网络上比较各自大学的鸡蛋价格；如[[北京交通大学|北京交通大学]]的鸡蛋更便宜，只有四毛五，[[清华大学|清华大学]]也是五毛<ref name="zgsb"/>。
