'''五苯基铋'''是一种[[金属有机化合物|金属有机化合物]]，[[化学式|化学式]]为Bi(C<sub>6</sub>H<sub>5</sub>)<sub>5</sub>，或简写为BiPh<sub>5</sub>。五苯基铋不稳定，室温下数日即完全分解。
==制备==
在-75℃，用[[苯基锂|苯基锂]]和[BiPh<sub>3</sub>]Cl<sub>2</sub>或[BiPh<sub>4</sub>]Cl反应可以得到五苯基铋：
: Li(C<sub>6</sub>H<sub>5</sub>) + [Bi(C<sub>6</sub>H<sub>5</sub>)<sub>4</sub>]Cl → Bi(C<sub>6</sub>H<sub>5</sub>)<sub>5</sub> + LiCl
==化学性质==
*五苯基铋可以和[[卤化氢|卤化氢]]、[[卤素|卤素]]反应，产生[[苯|苯]]或卤代苯，并形成季铋化合物<ref name=w />：
:: Bi(C<sub>6</sub>H<sub>5</sub>)<sub>5</sub> + HX → [Bi(C<sub>6</sub>H<sub>5</sub>)<sub>4</sub>]X + C<sub>6</sub>H<sub>6</sub>
*和[[三苯基硼|三苯基硼]]反应，产生铋鎓盐：
:: Bi(C<sub>6</sub>H<sub>5</sub>)<sub>5</sub> + B(C<sub>6</sub>H<sub>5</sub>)<sub>5</sub> → [Bi(C<sub>6</sub>H<sub>5</sub>)<sub>4</sub>][B(C<sub>6</sub>H<sub>5</sub>)<sub>4</sub>]
