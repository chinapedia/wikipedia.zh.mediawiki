'''五丈原之戰'''是[[中國|中國]]的[[三國|三國]]時期發生的一場戰役，主要戰場五丈原，交戰雙方為[[曹魏|曹魏]]及[[蜀漢|蜀漢]]，時為[[蜀汉|蜀汉]]后主[[劉禪|劉禪]][[建兴_(刘禅)|建兴]]十二年（234年，[[魏明帝|魏明帝]][[青龍_(曹魏)|青龍]]二年）。當時，蜀漢丞相[[諸葛亮|諸葛亮]]率軍[[诸葛亮北伐#第五次北伐|第五次北伐]]，由[[漢中|漢中]]出發，取道[[斜谷|斜谷]]，穿越秦嶺，進駐五丈原。在此，蜀军与魏军对峙了100余天。最后，诸葛亮病逝于此。

== 背景 ==
231年的第四次北伐，蜀军粮尽退军。撤退時，魏方[[車騎將軍|車騎將軍]][[張郃|張郃]]被射傷，其後傷重病死。從蜀方第一次北伐開始，由於連年派出數萬規模的軍隊出征，而每次均為兵糧不足而要退兵，所以諸葛亮在其後的三年裡，勸農講武，作[[木牛流马|木牛]]、[[木牛流马|流馬]]，運米糧到斜谷口集積，又住在斜谷邸閣，與民休息，準備再次北伐<ref>《[[晋書|晋書]]·宣帝紀》</ref>。

== 經過 ==
234年春[[二月|二月]]，諸葛亮再度出征曹魏，悉大眾軍由[[斜谷|斜谷]]出，以流馬運輸，意取長安；同時派使臣到東吳，希望孫權能同時攻魏，孫權答應進攻合肥。根據《晉書·宣帝紀》，兩軍合共十餘萬。[[四月|四月]]，諸葛亮到達[[郿縣|郿縣]]，據渭河南[[武功|武功]][[五丈原|五丈原]]，司馬懿立即从渭河北渡河，背水防守，形成诸葛亮與魏[[大都督|大都督]][[司馬懿|司馬懿]]在[[渭南|渭南]]對峙局面。

曹將[[郭淮|郭淮]]認為蜀軍必會渡渭河爭奪北原，司馬懿認同，便派郭淮先前往防備，又把人口集中的[[渭水|渭水]]以南築砦加固防備。果然，蜀軍到來，但被郭淮擊退。渭水呈东西走向，有一条呈南北走向并向北流入渭水的武功水成为司马懿和诸葛亮争夺的焦点。[[五丈原|五丈原]]位于武功水之西，诸葛亮派遣孟琰占据武功水东岸。司马懿曾于武功水涨水之时试图偷袭武功水东的孟琰所部，但遭诸葛亮搭浮桥以弩兵击退<ref>《水经注》卷18：诸葛亮《与步骘书》曰「仆前军在五丈原，原在武功西十里馀。」水出武功县，故亦谓之武功水也。是以诸葛亮《表》云「臣遣虎步监孟琰，据武功水东。司马懿因水长，攻琰营，臣作竹桥，越水射之。桥成驰去。」</ref>。諸葛亮每每患糧不繼，使己志不申，是以分兵屯田，為久駐之基。耕者雜於渭南河濱居民之間，而百姓相安，軍無私事。而[[孫權|孫權]]也曾率十萬大軍北上響應蜀漢，但被[[满宠|满宠]]领数十勇士放火烧船打败。（詳見'''[[合肥之戰#第三、四次戰役（合肥新城之战）|合肥之戰]]'''）。

諸葛亮自至，數次派人挑戰。司马懿吸取在诸葛亮第四次北伐时魏军与蜀汉军队正面作战惨败（详见[[諸葛亮北伐#第四次北伐|诸葛亮北伐]]）的教训，始終堅守不出。其後諸葛亮故意讓人帶一套女人的衣服、頭巾送給司馬懿，表示司馬懿就像女人一樣。魏軍將領見此情形都火冒三丈，紛紛要求出戰，為了搪塞將領們的要求，司馬懿亦上表向魏明帝固請出請戰。曹叡便使[[衛尉|衛尉]][[辛毗|辛毗]]持節以節制司馬懿。[[姜維|姜維]]對諸葛亮說：「辛佐治仗節而到，賊不復出矣。」諸葛亮曰：「彼本無戰情，所以固請戰者，以示武於其眾耳。將在軍，君命有所不受，苟能制吾，豈千里而請戰邪！」這只是做樣而已。

司馬懿讓2000餘人在軍營東南角高呼萬歲。諸葛亮使者就問怎麼回事。司馬懿說是孫吳有使者來請降。諸葛亮得知后說：“孫吳一定不會投降。你司馬懿都是快60歲的老頭了，何必要耍這種鬼把戏呢！”<ref>《通典》卷150：宣王使二千馀人，就军营东南角，大声称万岁。亮使问之，答曰："吴朝有使至，请降。"亮谓曰："计吴朝必无降法。卿是六十老翁，何烦诡诳如此。"</ref>

後諸葛亮又派下戰使到司馬懿處，司馬懿問諸葛亮寢、食和其事情之勞累，不問軍事。使對司馬懿說：「諸葛公夙興夜寐，罰二十以上，皆親攬焉；所啖食不至數升。」司馬懿說：「亮將死矣。」魏、蜀兩軍相峙百餘日。是年[[八月|八月]]，諸葛亮積勞成疾而病倒，病情日益惡化。消息傳到[[成都|成都]]，劉禪派[[李福|李福]]去探望諸葛亮，並詢問以後國家大計和繼承人問題。諸葛亮也對各將領交代後事，要[[楊儀|楊儀]]和[[費禕|費禕]]統領各軍撤退，由魏延、-{姜維}-負責斷後。

不久，諸葛亮逝世，享年五十四歲。楊儀、姜維等秘不發喪，整頓軍馬而退出。百姓奔走相告司馬懿，司馬懿追出。姜維令楊儀反旗鳴鼓，若將回軍向司馬懿進擊，司馬懿乃撤退，不敢追偪。於是楊儀結陣而去，入斜谷後才發喪。司馬懿之退，百姓為之笑話，稱「{{Link-ja|死諸葛嚇走生仲達|死せる孔明生ける仲達を走らす}}」。或以告司馬懿，司馬懿說：「吾能料生，不便料死也。」司马懿仔细查看了诸葛亮生前安营下寨之处，评价道：“天下奇才也！”对诸葛亮的军事才能给予了高度评价。

== 戰後 ==
《三國志》載撤退後的蜀軍，[[軍師|軍師]][[魏延|魏延]]、[[長史|長史]][[楊儀|楊儀]]為了是否撤兵兩派人馬發生鬥爭，魏延遭楊儀派遣的馬岱所殺。《魏略》則載楊儀放出魏延投敵之謠言，起兵追而殺之。

之後楊儀在軍事和政事遭到蒋琬等人刻意架空，楊儀雖任中軍師，卻無所統括、如同虛職。

後來[[蒋琬|蒋琬]]和[[費禕|費禕]]掌握了蜀軍的實權軍事和政事。楊儀為人心胸狹隘，不爽无法执掌大权，出言反逆，最後被貶為平民後因毀謗國政入獄，心有不甘而自殺。

蔣琬曾計劃繼續北伐，但為其他重臣反對而未有實行。

在抵抗過蜀軍的侵攻後，司馬懿於238年討伐[[遼東|遼東]]的[[公孫淵|公孫淵]]，滅公孫淵。自此，曹魏東西外患除去，司馬懿亦因為取得大功，確立其在曹魏朝廷內的地位。239年，年輕的曹叡病故，其養子[[曹芳|曹芳]]尚年幼，令司馬懿有機會把持政局，凌駕曹氏後人的地位，間接種下後來三國歸晉的基礎。
