《'''五月的鲜花'''》，[[中国抗日战争|中国抗战]]歌曲，[[张光年|光未然]]词，[[阎述诗|阎述诗]]曲。<ref name="Bunny_xinhuanet.com">{{Cite web |url= http://news.xinhuanet.com/politics/2009-05/25/content_11434058.htm |title= 100首爱国歌曲名单公布 |author=  |date=2009-05-25 |publisher= 新华网 |language= zh-hans |accessdate= 2011-05-29 导入 |quote= }}</ref>

该曲歌词创作于1935年，配曲于1936年。1935年，光未然在[[武汉|武汉]]所作[[独幕话剧|独幕话剧]]《[[阿银姑娘|阿银姑娘]]》，该曲为这一话剧的[[序曲|序曲]]。1936年夏，[[东北大学_(中国)|东北大学]]排演《阿银姑娘》时，因剧本上只有歌词，没有曲谱，该校教师[[阎述诗|阎述诗]]为其创作了乐曲。<ref name="Bunny_people.com.cn"/>1959年，该曲被选作电影《[[青春之歌|青春之歌]]》的插曲，并随电影的热播红遍全国。<ref name="Bunny_people.com.cn">{{Cite web |url= http://cpc.people.com.cn/GB/64093/64387/7557798.html |title= 《五月的鲜花》五月鲜花红大地  |author= 吴秀萍 |date=2008-07-24 |publisher= 中国共产党新闻网 |language= zh-hans |accessdate=2011-05-29 导入 |quote= }}</ref>

== 词曲版权 ==
*词作者光未然于2002年去世。
*曲作者阎述诗于1963年去世。
