'''浙江五芳斋实业股份有限公司'''是一家以食品的生产、加工和销售为主的[[中华老字号|中华老字号]]，著名产品为五芳斋[[粽子|粽子]]。为中国最大的粽子产销企业<ref name=abw>{{Cite web |url=http://www.chinaupb.com/article/2008/1030/article_1469.html# |title=中国最大粽子产销商-五芳斋-阿布万品牌网 |access-date=2009-08-18 |archive-url=https://web.archive.org/web/20090103001352/http://www.chinaupb.com/article/2008/1030/article_1469.html# |archive-date=2009-01-03 |dead-url=yes }}</ref>。
== 历史 ==
由于盛产稻米而商贸发达，[[浙江|浙江]][[嘉兴市|嘉兴市]]居民历来讲究饮食之道。[[明朝|明朝]]时期“嘉湖细点”已驰名中华大地。据[[清|清]][[乾隆|乾隆]]年间《古禾杂谈》一书记载：嘉兴粽子，昔日有白水粽、红枣粽、赤豆粽、豆板粽和碱水粽，俗称“五芳”。
然而如今五芳斋企业的创始则是以弹棉花为业的浙江[[兰溪|兰溪]][[手工艺人|手工艺人]]张锦泉，在1921年春夏季生意冷淡之时经营的粽子摊。因外形别致，选料、制作考究，风味独特而招徕了很多顾客。十多年后，张先生召集了几名老乡，在嘉兴的闹市张家弄口，租间门面开了首家“五芳斋”粽子店，经营火腿鸡肉粽、重油夹沙粽。事隔数年后，嘉兴人冯昌年、朱庆堂两人在首家店对面和隔壁也开设了“合记”和“庆记”两家五芳斋。三家呈品字形分布的五芳斋粽子店，一时成了嘉兴这座江南古城的独特风景。因为三位老板之间的激烈竞争，促使粽子选料考究，配制方案完善，口味精美，嘉兴张家弄“五芳斋”由此声誉鹊起，驰名[[江南|江南]]。1949年后的[[城市改造|城市改造]]使得张家弄不复存在，而三家粽子店也在1956年的[[公私合营|公私合营]]中联手成立了新“五芳斋”<ref>[http://nhnews.zjol.com.cn/nhnews/system/2009/05/26/011142175.shtml 嘉兴粽子与五芳斋--南湖新闻网]</ref>。
== 发展 ==
1998年，五芳斋被一家[[房地产|房地产]]企业收购，成为民营企业。由五家分公司构成其食品生产原料供应链。五芳斋以浙江为基地，在全国各地开设连锁店，并有意打造为著名的中式连锁快餐品牌<ref>[http://www.linkshop.com.cn/Web/Article_News.aspx?ArticleId=100314 浙江五芳斋粽子的不老传奇_联商网]</ref>。
== 产品 ==
五芳斋以粽子而闻名华夏大地，花色品种近百个。另外公司借助品牌优势也开发中式点心以及卤味食品，如[[汤圆|汤圆]]、[[月饼|月饼]]、[[咸蛋|咸蛋]]、[[酱鸭|酱鸭]]等。
== 五芳斋集团 ==
如今，五芳斋已经形成一家以食品产业、[[农业|农业]]为核心，兼具[[房地产|房地产]]、[[金融|金融]]和[[贸易|贸易]]等业务的大型民营企业集团。集团总部所在地为浙江省嘉兴市。集团热心支持社会公益事业，分别在[[浙江工商大学|浙江工商大学]]和嘉兴一中设立了“五芳斋”奖学金，并积极捐助社会慈善机构<ref name=abw />。
