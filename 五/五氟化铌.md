'''五氟化铌'''，也称'''氟化铌(V)'''，化学式为NbF<sub>5</sub>。它是一种无色的[[单斜|单斜]]晶体，通常用作[[铌|铌]]化学的起始原料。

== 制备 ==
五氟化铌是从铌矿石中提炼金属铌的中间产物。它也可由金属铌在250至300°C发生[[氟化|氟化]]反应制得，氟化剂可使用氟气、无水[[氟化氢|氟化氢]]或[[二氟化锡|二氟化锡]]。将五氟化铌蒸汽在[[派热克斯玻璃|派热克斯玻璃]]或[[石英玻璃|石英玻璃]]管中浓缩，然后在120°C下真空[[升华|升华]]即可收集到无色晶体。如果与无水氟化氢反应，则会生成五氟化铌与三氟化铌的混合物。

:<math>\rm \ 2Nb + 5F_2 \xrightarrow{\Delta} 2NbF_5</math>
:<math>\rm \ 2Nb + 5SnF_2 \xrightarrow{\Delta} 2NbF_5 + 5Sn</math>

如果将金属铌溶于浓[[硝酸|硝酸]]和[[氢氟酸|氢氟酸]]的混合酸（或浓氢氟酸），蒸干溶液并灼烧残渣可得到[[一氟二氧化铌|一氟二氧化铌]]（NbO<sub>2</sub>F），因此不常用该法制备五氟化铌。

五氟化铌也可由氟气、[[氟化锌|氟化锌]]、无水氟化氢与[[五氯化铌|五氯化铌]]反应制得：
:<math>\rm \ 2NbCl_5 + 5F_2 \xrightarrow{\Delta} 2NbF_5 + 5Cl_2</math>

氟配合物热分解法也经常被使用：
:<math>\rm \ (BrF_2)NbF_6 \xrightarrow{\Delta} BrF_3 + NbF_5</math>
:<math>\rm \ K_2NbF_7 + 2AlF_3 \xrightarrow{1073K} KAlF_4 + NbF_5</math>

== 性质 ==
五氟化铌吸水性很强，在空气中就会[[潮解|潮解]]，但速率较慢不冒烟。因此不可能在水溶液中制得五氟化铌。五氟化铌气态时为[[三角双锥|三角双锥]]形分子，液态时为双聚体，固态时为四聚体。固态五氟化铌是由四个<nowiki>[NbF</nowiki><sub>6</sub><nowiki>]</nowiki>八面体组成的，铌不仅与五个氟原子成键，还互相通过[[金属—金属键|金属—金属键]]形成四边形的骨架。液态五氟化铌还会发生[[自耦电离|自耦电离]]：
:<math>\rm \ 2NbF_5 \rightleftharpoons NbF_4^+ + NbF_6^-</math>（小于1%）

五氟化铌是强[[路易斯酸|路易斯酸]]，可溶于水和酸。溶解的过程还伴随着[[水解|水解]]，产生[[氟氧化铌|氟氧化铌]]和[[氢氟酸|氢氟酸]]。与碱溶液反应产生组成不定的水合[[五氧化二铌|五氧化二铌]]沉淀。<ref>{{cite book|title=《无机化学丛书》第八卷：钛分族、钒分族、铬分族|pages=P328-329|author=张青莲|isbn=7-03-002238-6|location=北京|publisher=科学出版社}}</ref>
