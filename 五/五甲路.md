'''五甲路（Wujia Rd.）'''為[[高雄市|高雄市]][[鳳山區|鳳山區]]的南北向重要幹道，本道路共分成三個部分，編號都是[[市道183號|市道183號]]。起端於[[中山路_(鳳山區)|中山路]]口接[[維新路_(鳳山區)|維新路]]，末端接[[五甲媽祖港橋|五甲媽祖港橋]]與[[前鎮區|前鎮區]][[中山路_(高雄市)|中山四路]]交會，同時是[[市道183號|市道183號]]終點，續行[[鎮中路|鎮中路]]往草衙。是[[鳳山區|鳳山市區]]往[[五甲|五甲]]、[[中山高速公路|國道一號]][[五甲交流道|五甲交流道]]及[[前鎮區|前鎮區]]的重要道路。

== 行經行政區域 ==
（由北至南）
* [[鳳山區|鳳山區]]
* [[前鎮區|前鎮區]]

== 分段 ==
五甲路共分成三個部份，屬[[File:TW_CHW183.svg|23px]][[市道183號|市道183號]]：
* 五甲一路：北起於[[中山路_(鳳山區)|中山路]]口接[[維新路_(鳳山區)|維新路]]，南於[[臨海路_(鳳山區)|臨海路]]口右行接五甲二路（直行接[[File:TWHW1.svg|23px]][[五甲交流道|五甲交流道]]）。
* 五甲二路：北於臨海路口接五甲一路，南於[[自強路_(鳳山區)|自強路]]口接五甲三路。
* 五甲三路：北於自強路口接五甲二路，南於[[中山路_(高雄市)|中山四路]]口接[[前鎮區|前鎮區]][[鎮中路|鎮中路]]。
