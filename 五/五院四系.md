'''五院四系'''是指[[中华人民共和国|中华人民共和国]]国内部分[[法学|法学]]专业的学院及院系。

“五院”分別为：[[中國政法大學|北京政法學院]]、[[西南政法大學|西南政法學院]]、[[華東政法大學|華東政法學院]]、[[西北政法大學|西北政法學院]]、[[中南財經政法大學|中南政法學院]]

“四系”分别为：[[北京大學|北京大學]][[北京大學法學院|法律系]]、[[中國人民大學|中國人民大學]][[中國人民大學法學院|法律系]]、[[武漢大學|武漢大學]][[武漢大學法學院|法律系]]、[[吉林大學|吉林大學]][[吉林大學法學院|法律系]]。

然而，現時「五院」均已成為政法大學，「四系」亦已成為法學院。<ref name="“五院四系”——直击法学豪门">{{cite web | url=http://yz.chsi.com.cn/kyzx/zyk/201204/20120419/304225490.html | title=“五院四系”——直击法学豪门 | accessdate=2013-06-21}}</ref>

==地位==
中华人民共和国成立初期，由中央人民政府统一部署规划，在全国高等学校实行院系调整，使全国各大区建立了[[北京政法学院|北京政法学院]]、[[华东政法学院|华东政法学院]]、[[中南政法学院|中南政法学院]]、[[西南政法学院|西南政法学院]]、[[西北政法学院|西北政法学院]]。此外在[[北京大学|北京大学]]、[[中国人民大学|中国人民大学]]、[[吉林大学|吉林大学]]、[[武汉大学|武汉大学]]等综合性大学相继设立和恢复了法律系，使中国的法学教育初具规模。改革开放以后，中国的法学教育和法学研究进入了一个快速发展时期。为实现高等法学教育领域综合改革新突破，[[中共中央政法委员会|中央政法委]]、[[中华人民共和国教育部|中华人民共和国教育部]]联合启动的培养工程“卓越法律人才教育培养计划”<ref name="教育和公检法司等部门共建百余培养基地 卓越法律人才教育
培养计划启动">{{cite web | url=http://www.moe.edu.cn/s78/A08/gjs_left/moe_739/s6549/s7719/201206/t20120614_159429.html | title=教育和公检法司等部门共建百余培养基地 卓越法律人才教育培养计划启动}}</ref>。除高校法学院外，成立于1978年的[[中国社会科学院研究生院|中国社会科学院研究生院法学系]]，作为国家级法学研究机构设立的法学系，在中国高等法学教育界也占有重要一席。

==沿革==
“五院”以及其更名时间如下：
* 北京政法学院 → [[中国政法大学|中国政法大学]]（更名时间：1983）
* 西南政法学院 → [[西南政法大学|西南政法大学]]（更名时间：1995）
* 中南政法学院、中南财经大学合并 → [[中南财经政法大学|中南财经政法大学]]（更名时间：2000.5.26）
* 西北政法学院 → [[西北政法大学|西北政法大学]]（更名时间：2006.11）
* 华东政法学院 → [[华东政法大学|华东政法大学]]（更名时间：2007.03）

“四系”以及其改制时间如下：
* 北京大学法律学系 → [[北京大学法学院|北京大学法学院]]（改制时间：1999.6.26）
* 中国人民大学法律系 → [[中国人民大学法学院|中国人民大学法学院]]（改制时间：1988）
* 武汉大学法律系 → [[武汉大学法学院|武汉大学法学院]]（改制时间：1986）
* 吉林大学法律系 → [[吉林大学法学院|吉林大学法学院]]（改制时间：1988）
