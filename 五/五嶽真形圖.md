'''五嶽真形圖'''，題為[[東方朔|東方朔]]編，實際應出於漢末魏晉之際，為中國古代[[道士|道士]]繪製的一種特殊山嶽圖，在[[道教|道教]]中用以「辟兵凶逆」，為[[符籙|符籙]]之最古者<ref name=five>{{cite journal|author1=辛德勇|title=記東方朔《五嶽真形圖序》存世 最早的寫本|journal=九州|url=http://www.zggds.pku.edu.cn/004/001/260.pdf|author=|access-date=2016-06-25|archive-url=https://web.archive.org/web/20160816114128/http://www.zggds.pku.edu.cn/004/001/260.pdf|archive-date=2016-08-16|dead-url=yes}}</ref><ref name=five2>{{cite book|editor=胡孚琛|title=中華道教大辭典|publisher=中國社會科學出版社|pages=275}}</ref>。後世此書傳本甚多，今《[[正統道藏|正統道藏]]》本一卷，收入洞玄部靈圖類。另有《[[雲笈七籤|雲笈七籤]]》本，收入該書卷七九。今[[河南|河南]]登封縣嵩山[[中嶽廟|中嶽廟]]內存此圖碑刻。

== 題解 ==
[[五嶽|五嶽]]，指東嶽[[泰山|泰山]]、西嶽[[華山|華山]]、中嶽[[嵩山|嵩山]]、北嶽[[恆山|恆山]]、南嶽[[衡山|衡山]]。乃源於古代中國對山川的崇拜祭祀，依附[[五行|五行]]思想而成形<ref>{{cite web|title=修道之人保命護身的密碼，《五嶽真形圖》！|url=https://read01.com/MJzO4d.html}}</ref>。

真形，本來的形象；真實的形體或形象<ref>{{cite web|title=真形|url=http://dic.academic.ru/dic.nsf/chi_big_enc/300072/%E7%9C%9F%E5%BD%A2}}</ref>。五嶽真形，即五嶽的山水之象，其形勢「盤曲迴轉，高下參差，長短卷舒，波流似於奮筆，鋒鋩暢乎嶺崿」<ref>{{cite web|title=洞玄靈寶五嶽古本真形圖并序|url=http://www.ctcwri.idv.tw/CTCWRI-MTS/CMT02%E6%B4%9E%E7%8E%84%E9%83%A8/CMT0204%E9%9D%88%E5%9C%96%E9%A1%9E/CH0204XX/CH020413%E6%B4%9E%E7%8E%84%E9%9D%88%E5%AF%B6%E4%BA%94%E5%B6%BD%E5%8F%A4%E6%9C%AC%E7%9C%9F%E5%BD%A2%E5%9C%96.htm|quote=五嶽真形者，山水之象也。盤曲迴轉陵阜，形勢高下參差，長短卷舒，波流似於奮筆，鋒鋩暢乎嶺崿。雲林玄黃，有如書字之狀。是以天真道君，下觀規矩，擬縱趨向，因如字之韻，而隨形而名山焉。|deadurl=yes|archiveurl=https://archive.is/20160625002727/http://www.ctcwri.idv.tw/CTCWRI-MTS/CMT02%E6%B4%9E%E7%8E%84%E9%83%A8/CMT0204%E9%9D%88%E5%9C%96%E9%A1%9E/CH0204XX/CH020413%E6%B4%9E%E7%8E%84%E9%9D%88%E5%AF%B6%E4%BA%94%E5%B6%BD%E5%8F%A4%E6%9C%AC%E7%9C%9F%E5%BD%A2%E5%9C%96.htm|archivedate=2016-06-25}}</ref>。

== 內容 ==
其一為〈五嶽真形圖序〉，概述五嶽四山（即[[泰山|泰山]]、[[衡山|衡山]]、[[嵩山|嵩山]]、[[華山|華山]]、[[恆山|恆山]]、[[青城山|青城山]]、[[廬山|廬山]]、[[天柱山|霍山]]、[[潛山|潛山]]）之真形與靈跡。內稱五嶽四山之「真形圖」乃取象山水曲折參差之狀圖寫而成，出自[[神農|神農]]、[[黃帝|黃帝]]之世。凡人有此真形圖，人山不迷，可避兵災毒害。又稱五嶽四山皆有神靈，並逐一列舉每山之山君及其部領群仙玉女之數<ref name=five2/>。

其二為〈靈寶五嶽古本真形圖〉，首載「五嶽山真形文」一幅、「三天太上大道君長生符」六枚。其次列舉五嶽四山真形圖九幅，每圖前各有符一枚。各符圖後有文字，略敘書寫佩帶符圖之法<ref name=five2/>。

其三〈洞元靈寶五嶽真形圖〉，此篇符圖文字與古本略同，惟符文筆畫較細，並且各真形圖內外有小字注文。疑此篇乃宋代傳世之《五嶽真形圖》別本<ref name=five2/>。

== 考據 ==
據推斷《五嶽真形圖》應該源自一種由高山絕頂向下俯視的[[鳥瞰圖|鳥瞰圖]]，或者說是山嶽平面圖。後代通行的《五嶽真形圖》，往往完全脫離實際地理形勢，只是標繪爲一種圖形、符籙符號，含有護符成份在內，已無地形圖之作用<ref name=five/>。
