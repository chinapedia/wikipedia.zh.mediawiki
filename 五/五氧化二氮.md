'''五氧化二氮'''（化学式：[[氮|N]]<sub>2</sub>[[氧|O]]<sub>5</sub>），又称'''硝酐'''，是[[硝酸|硝酸]]的[[酸酐|酸酐]]。通常状态下呈无色柱状结晶体，易溶于水，水溶液呈酸性。可以用P<sub>2</sub>O<sub>5</sub>将浓HNO<sub>3</sub>脱水得到。

==性质==
五氧化二氮很容易[[潮解|潮解]]，而且在10℃以上能分解生成毒气[[二氧化氮|二氧化氮]]及[[氧气|氧气]]，但在-10℃以下时较稳定。室温下易[[挥发|挥发]]、对光和热敏感。溶于热水时生成[[硝酸|硝酸]]。遇高温及易燃物品，会引起燃烧爆炸。由五氧化二氮引起的火灾，可使用[[水|水]]或[[泡沫|泡沫]]进行扑灭。

==结构==
研究表明，固体N<sub>2</sub>O<sub>5</sub>是由直线型的NO<sub>2</sub><sup>+</sup>(N-O键长115.4pm)和平面正三角形的NO<sub>3</sub><sup>-</sup>(N-O键长124.0pm)。<ref>刘新锦等.无机元素化学 氮族元素.北京:科学出版社.2010.01  ISBN 978-7-03-026399-5</ref>

==制备==
通过[[五氧化二磷|五氧化二磷]]使[[浓硝酸|浓硝酸]]在低温下(263K)的[[脱水|脱水]]反应或通过[[臭氧|臭氧]]氧化[[二氧化氮|二氧化氮]]的反应来制备。
:4HNO<sub>3</sub> + P<sub>4</sub>O<sub>10</sub> → 2N<sub>2</sub>O<sub>5</sub> + 4HPO<sub>3</sub>

:2NO<sub>2</sub> + O<sub>3</sub> →  N<sub>2</sub>O<sub>5</sub> + O<sub>2</sub>

==反应==
N<sub>2</sub>O<sub>5</sub>具有强[[氧化性|氧化性]]，能将[[钠|Na]]、[[碘|I<sub>2</sub>]]等转变为Na<sup>+</sup>、[[五氧化二碘|I<sub>2</sub>O<sub>5</sub>]]等。N<sub>2</sub>O<sub>5</sub>与无水[[硫酸|硫酸]]、[[高氯酸|高氯酸]]等反应能产生硝[[鎓|鎓]]离子(NO<sub>2</sub><sup>+</sup>)，是制取硝鎓盐的一条途径。

:N<sub>2</sub>O<sub>5</sub> + 3H<sub>2</sub>SO<sub>4</sub> → 2NO<sub>2</sub><sup>+</sup> + 3HSO<sub>4</sub><sup>-</sup> + H<sub>3</sub>O<sup>+</sup>
