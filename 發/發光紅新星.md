'''發光紅新星'''（縮寫為''LRNe''）被認為是兩顆恆星合併所造成的爆炸現象。它們的特徵是有明顯的紅色，和[[光度曲線|光度曲線]]在[[紅外線|紅外線]]区反覆的回到原來的光度逗留和徘徊。不要將發光紅新星與標準的[[新星|新星]]——以[[白矮星|白矮星]]為主角，在表面發生爆炸——混淆了。

== 發現 ==
在過去的30年，只有很少數量的天體被觀察到有發光紅新星的特徵。在[[仙女座大星系|仙女座大星系]]M31，於1988年閃爍的{{link-en|M31 RV|M31-RV}}可能是一顆發光紅新星；在[[銀河系|銀河系]]，
1994年的[[人馬座V4332|人馬座V4332]]有著相同的閃爍，接著還有2002年的[[麒麟座V838|麒麟座V838]]，曾被嚴謹的觀察。

第一顆被確認的發光紅新星是在[[M85|M85]]星系的[[M85_OT2006-1|M85 OT2006-1]]。它是在[[立克天文台|立克天文台]]進行[[超新星|超新星]]搜尋時首度被發現，然後由[[加州理工|加州理工]]和[[柏克萊|柏克萊]]的一群[[天文學家|天文學家]]繼續研究。這個由[[Shrinivas_Kulkarni|Shrinivas Kulkarni]]領導的小組證實了他和已知的由[[漸近巨星分支#AGB階段|熱脈衝]]造成爆炸的[[新星|新星]]不同，並且在2007年5月23日發表在[[自然雜誌|自然]]的論文中宣稱發光紅新星是一種恆星爆炸新星的新類型。

== 特徵 ==

發光紅新星在觀測上有下列的特徵：

* 爆炸時的[[亮度#天文學|亮度]]在[[超新星|超新星]]（較亮）和[[新星|新星]]（較暗）之間。

* 可見光持續數星期或數月，有著獨特的紅色，光度隨著時間變暗而顯得更紅。隨著可見光的暗淡，紅外線會增強並且持續反覆一段時間，通常會變暗和再增亮數次。

* 對M85 OT2006-1的紅外線觀測顯示是溫度非常低（略低於1,000[[熱力學溫標|K]]）的恆星。這可能是也可能不是其他發光紅新星的特徵。I

== 演化 ==
研究M85 OT2006-1的團隊相信它是兩顆[[主序星|主序星]]合併時發生的現象。想知道更多的訊息和詳情可以參考維基媒體的[[麒麟座V838|麒麟座V838]]和''mergebursts''。

當''mergebursts''發生之際，LNRe的膨脹看似極端的迅速，在幾個月內半徑就達到[[太陽|太陽]]的數千倍甚至上萬倍。因此溫度能降至如此的低，同時也解釋了令人迷惑的亮度閃爍與低溫並存的天體。
