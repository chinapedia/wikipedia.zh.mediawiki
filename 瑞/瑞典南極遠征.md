'''瑞典南極遠征'''（{{lang-sv|Första svenska Antarktisexpeditionen}}）是於1901年至1904年由[[奧托·努登舍爾德|奧托·努登舍爾德]]和[[卡爾·安東·拉森|卡爾·安東·拉森]]所率領的瑞典遠征行動。
[[Image:Snow.Hill.Is.Jan99.jpeg|right]]

== 背景 ==
瑞典地質學家暨地理學家奧托·努登舍爾德組織並領導這次南極半島遠征，卡爾·安東·拉森為[[南極號|南極號]]船長，且有7名科學家及16名船員同行。1901年10月16日，南極號離開[[哥特堡|哥特堡]]，前往南極。

儘管航海過程艱辛且結局並不理想，但仍被視為是一次成功的科學行動。他們探索了許多地方，像是：[[葛拉漢地|葛拉漢地]]東部、[[渴望角|渴望角]]、[[詹姆斯羅斯島|詹姆斯羅斯島]]、[[茹安維爾島|茹安維爾島]]和[[帕默群島|帕默群島]]等。此次遠征也收集了珍貴的地理和海洋生物樣本，使奧托·努登舍爾德獲得了崇高的名聲，但也造成他負債累累。

兩座島嶼分別和此次遠征有密切關係。一是[[斯諾希爾島|斯諾希爾島]]，諾登斯基爾德和五名夥伴在島上度過了兩年冬天。另一座是[[保萊特島|保萊特島]]，1903年2月至1903年11月，因南極號沉沒而受困之處。

== 斯諾希爾島 ==
1901年，在前往斯諾希爾島的途中，諾登斯基爾德先經過[[阿根廷|阿根廷]][[布宜諾斯艾利斯|布宜諾斯艾利斯]]，獲得阿根廷政府資助補給品和其他援助，包括年輕的阿根廷中尉[[何塞·瑪麗亞·索布拉爾|何塞·瑪麗亞·索布拉爾]]。索布拉爾和藝術家[[法蘭克·威爾伯特·斯托克斯|法蘭克·威爾伯特·斯托克斯]]與諾登斯基爾德在斯諾希爾島度過了兩年時光，索布拉爾成為第一位在南極洲過冬的人。也許是因為這位阿根廷中尉的緣故，阿根廷政府後來於1903年派出[[烏拉圭號|烏拉圭號]]援救所有落難的遠征隊員。

[[Image:Paulet.Is.Dec.04.jpg|right]]

== 保萊特島 ==
在船撞上浮冰漂流25英里沉沒後，20名隊員乘救生艇登上此島，並搭建了雙層厚的石棚屋避難，該棚屋至今仍佇立於該地。除了僅存的船上補給品外，它們也獵殺企鵝與獲取鳥蛋維生。
