'''瑞典國旗'''即[[瑞典|瑞典王國]]的國旗，於1906年6月22日正式制定。在1157年，瑞典國王[[埃里克九世|埃里克九世]]於遠征[[芬蘭|芬蘭]]前向神禱告，突然看到如同金色十字架的光芒橫越青空。根據這個傳說，藍底金十字與藍底金王冠的圖案成為了瑞典的國家象徵。也有指金十字是來自瑞典[[福爾孔王朝|福爾孔王朝]]（House of Folkung）中的金色十字架，並以藍底為本而設計的。

</gallery>

== 设计 ==
*根據瑞典法例1982:269，旗幟的比例為10/16（高/闊），左邊上下兩個相等的藍色長方形的邊長比例各為4/5，右邊的藍色長方形比例各為4/9（高/闊）。黃帶的闊度為旗幟高度（短边）的1/5。
*根據瑞典法例1983:826，國旗使用的顏色為{{link-en|NCS顏色系統|Natural Color System}}：黃色 - NCS 0580-Y10R、藍色 - NCS 4055-R95B。

== 历史 ==
瑞典國旗的歷史可追溯至16世紀，相信源自瑞典的國家[[紋章|紋章]]。紋章的底色為藍色，上繪有三個黃色皇冠。1569年約翰三世採用此國旗。在此之前此旗曾在約翰的封地（現[[芬蘭|芬蘭]]西南部）中使用。

== 在瑞典以外的使用 ==
原屬瑞典在美殖民地「[[新瑞典|新瑞典]]」的兩座[[美國|美國]]城市——[[費城|費城]]和[[威尔明顿_(特拉华州)|威爾明頓]]的市旗以瑞典國旗的用色為底，再鑲入各自的市徽，彰顯其[[瑞典－美國關係|與瑞典的淵源]]。
<gallery>
|威爾明頓市旗
File:Flag of Philadelphia, Pennsylvania.svg|費城市旗
</gallery>

此外，[[阿根廷|阿根廷]]足球俱樂部[[小保加體育會|博卡青年]]的球衣色彩採用與瑞典國旗一致的藍黃兩色，這一舉動源於1906年，當時博卡青年隊將球衣顏色從之前的黑白改為第一艘駛入博卡區港口的輪船所懸掛的旗幟色彩，而駛入博卡港口的第一艘輪船正是從哥本哈根駛來的瑞典貨輪。<ref>{{cite web |author=Georgina Turner and James Dart |url = http://football.guardian.co.uk/theknowledge/story/0,13854,1643916,00.html |title=Turner, Georgina & Dart, James.
