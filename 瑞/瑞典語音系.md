[[瑞典語|瑞典語]]發音有如下特徵：[[元音|元音]]衆多，包括17個元音[[音位|音位]]，均爲[[單元音|單元音]]；有一個獨特的清擦音“sje音”/[[ɧ|{{IPA]]/；有區別詞義的[[聲調|聲調]]。

和[[丹麥語|丹麥語]]及[[芬蘭語|芬蘭語]]不同，瑞典語並沒有一個標準發音。在不同的方言中，元音和一部分輔音（尤其是部分清擦音和字母r有關的發音）在不同方言中有相當區別。三個主要的優勢方言爲：中部標準瑞典語、芬蘭瑞典語、南部標準瑞典語。本文主要描述[[斯德哥爾摩|斯德哥爾摩]]地區使用的中部瑞典標準語的發音。

=== 元音 ===
瑞典語的元音數量很多且複雜。和多數其它[[日耳曼語|日耳曼語]]一樣，瑞典語區分長短元音。元音的長短一般也影響到其音質。非重讀的元音總是短元音。非重讀的{{IPA|/ɛ/}}有時弱化爲{{IPA|[ə]}}，而{{IPA|/r/}}之前的元音有時開口度較大或有一定的[[卷舌|卷舌]]。儘管一般認爲瑞典語的元音均爲單元音，但一些長元音在一些方言，包括中部標準方言中有一定程度的[[複元音|複元音]]趨勢。

值得注意的是，瑞典語的部分長元音和[[德語|德語]]或國際音標符號的標準讀音有較大差距。高長元音有在結尾加[[半元音|半元音]]而成爲[[前響雙元音|前響雙元音]]的趨勢，如{{IPA|/iː/}} -> {{IPA|[iːj]}}; {{IPA|/ʉː/}} -> {{IPA|[ʉːw]}}; {{IPA|/uː/}} -> {{IPA|[uːw]}}。{{IPA|/ʉː/}}比國際音標標準讀音更偏前，接近{{IPA|[yː]}}，{{IPA|/iː/}}也有時候化成爲{{IPA|[ɨ̘ː]}}（即與/ʉː/相對應的不圓脣元音），而{{IPA|/yː/}}音的脣皺得更緊，以保持與{{IPA|/ʉː/}}、{{IPA|/øː/}}的區別。半高的長元音{{IPA|/eː/}}、{{IPA|/øː/}}和{{IPA|/oː/}}有成爲[[後響雙元音|後響雙元音]]的趨勢，表現爲前加半元音或者後加弱而短的{{IPA|[ə]}}音。

尤其在{{IPA|/r/}}音前，{{IPA|/ɛː/}}和{{IPA|/øː/}}分別低化成爲{{IPA|[æː]}}和{{IPA|[œ̞ː]}}，如：

# 瑞典語的[[塞音|塞音]]分爲[[清濁音|清濁]]相對的三對：{{IPA|/p/}}、{{IPA|/b/}}；{{IPA|/t/}}、{{IPA|/d/}}；{{IPA|/k/}}、{{IPA|/g/}}。其中清塞音和[[英語|英語]]類似，在詞首送氣，而在{{IPA|/s/}}之後不送氣，在詞中也不送氣（類似[[日語|日語]]）。如：ko {{IPA|[kʰuː]}}；sko {{IPA|[skuː]}}（對比英語cool {{IPA|[kʰuːɫ]}}和school {{IPA|[skuːɫ]}}）。
# 瑞典語的“tje音”，即{{IPA|/ɕ/}}，實際發音因人而略有差異，一般爲{{IPA|[ɕ]}}（漢語拼音x），或者接近{{IPA|[ʃ]}}（英語sh）。在芬蘭瑞典語裏，此音會被發爲[[塞擦音|塞擦音]]{{IPA|[t͡ɕ]}}（漢語拼音q）或{{IPA|[t͡ʃ]}}（英語ch）。
# 瑞典語的“sje音”，即{{IPA|/ɧ/}}音，按國際音標應爲舌尖和舌根同時分別接近[[硬顎|硬顎]]和[[軟顎|軟顎]]發出的清擦音，或者{{IPA|/ʂ/}}（漢語拼音sh）和{{IPA|/x/}}（漢語拼音h）音同時發出，爲瑞典語所特有。實際發音根據發音人地區、階層、年齡、性別等會有較大差異。主要以舌前部擦出的音接近{{IPA|[ʂ]}}或者{{IPA|[ʃ]}}，稱作“亮的sje音”，常見于北部和芬蘭；主要以舌根擦出的音接近{{IPA|/x/}}，稱作“暗的sje音”，常見于南部。
# {{IPA|/r/}}音變化也相當大。實際發音中，通常只有在著重強調的時候才用舌尖顫音{{IPA|[r]}}。南部通常成爲小舌顫音{{IPA|[ʀ]}}。在中部標準瑞典語中，{{IPA|/r/}}音經常成爲濁擦音{{IPA|[ʐ]}}（漢語拼音聲母r重讀時）、[[近音|近音]]{{IPA|[ɹ]}}（類似英語red中的r）或[[閃音|閃音]]{{IPA|[ɾ]}}。在一般的{{IPA|/r/}}不讀小舌音的瑞典方言（包括中部標準瑞典語）中，{{IPA|/r/}}除造成之前的元音{{IPA|/ɛ/}}和{{IPA|/œ/}}等低化外，也可和後面緊隨的舌尖輔音{{IPA|/t, d, n, l, s/}}相融合形成相應的卷舌輔音{{IPA|[ʈ, ɖ, ɳ, ɭ, ʂ]}}而不發出單獨的顫音，如karta {{IPA|[kʰɑːʈa]}}、fersk {{IPA|[fæʂk]}}等。在口語中，詞尾的{{IPA|/r/}}也可以和後面詞首的字母融合。
# 由{{IPA|/s/}}卷舌化或來源於一些外來詞的{{IPA|/ʂ/}}發音可能和“亮的sje音”難以區分，但不會發成“暗的sje音”。
# 和絕大多數語言一樣，瑞典語的{{IPA|/m, n/}}接在[[脣齒音|脣齒音]]{{IPA|/f, v/}}之前是會同化爲脣齒鼻音{{IPA|[ɱ]}}。
# 瑞典語的{{IPA|/ŋ/}}與漢語、英語或德語的舌根鼻音{{IPA|/ŋ/}}有較大不同，部位靠前而接近舌面中-硬顎鼻音{{IPA|[ɲ]}}。
# 瑞典語的清塞音雙寫時（如pp, tt, ck）會在除阻前有一個較長的屏氣停頓時間，類似[[意大利語|意大利語]]的雙寫輔音或[[日語|日語]]的[[促音|促音]]，如 bott {{IPA|[bʊtt]}} {{Audio|sv-bott.ogg|錄音}}。此外卷舌的rt字母組合發音也類似雙寫輔音，如ärt {{IPA|[æʈʈ]}} {{Audio|sv-ärt.ogg|錄音}}。

=== 重音和聲調 ===
瑞典語的韻律在集中優勢方言之間區別相當大。和大多數語言一樣，瑞典語的重音除能夠用來強調句中的某個詞以外，某些句調也可表達疑問等含義。和英語類似，瑞典語的重音會引起時長的變化，且詞的重音位置可以用於區別詞義。如
:formel {{IPA|['fɔrmɛl]}} （公式）
:formell {{IPA|[fɔr'mɛl]}}  （形式的）

在多數方言中，在雙音節和多音節詞中，重音分爲兩種，即“銳聲”和“重聲”，或稱“1調”和“2調”，一些學者由此認爲瑞典語也屬於[[有聲調語言|有聲調語言]]。銳聲的重音音節較高，其後隨的弱音節降低，類似于英語的重音。重聲的重音音節較低沉，而其後隨的弱讀音節較高。用[[五度標調法|五度標調法]]來表示，對於重音在前的雙音節詞，銳聲接近{{IPA|[˥˥⁵⁵ ˧˨³²]}}，而重聲接近{{IPA|[˨˩²¹ ˦˨⁴²]}}。在不同教科書和字典中，兩種重音的標法並不統一。本文中用音節之前的'和ˇ分別標出銳聲和重聲。

值得注意的是，瑞典語中有300多對雙音節的詞，僅能凴兩種重音類型相區別。大致規則如下：如果詞典中的原形是雙音節的（含複合詞），則爲重聲。如果原形爲單音節，則爲銳聲。各種詞形變化及衍生形成的雙音節通常亦爲重聲，但由單音節加定冠詞尾仍爲銳聲。比較：
:anden {{IPA|['andɛn]}} （銳聲，原形and，“鴨子”）和
:anden {{IPA|[ˇandɛn]}} （重聲，原形ande，“靈魂”）
{{Audio|Sv-anden_anden.ogg|錄音}}

這種聲調區別早在[[老挪威語|老挪威語]]中就已出現。而進年來新借入的詞彙通常爲銳聲。

== 瑞典文拼寫 ==
瑞典文拼寫和發音的關係例外較多，和英語類似，有拼讀不規則、詞尾不發音或外來借詞等情況，經常要借助字典才能確知一個單詞的讀音。另外由於瑞典語無標準方言，講母語的各人拼讀也會有差別。以下規則僅爲大致參考。

=== 元音 ===
