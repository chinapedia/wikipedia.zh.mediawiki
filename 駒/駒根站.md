'''駒根站'''（{{jpn|j=駒ケ根駅|hg=こまがねえき|rm=Komagane eki|f=y}}）是位於[[日本|日本]][[長野縣|長野縣]][[駒根市|駒根市]]東町的[[東海旅客鐵道|東海旅客鐵道]]（JR東海）[[飯田線|飯田線]]車站。

本站有一班快速列車是由此前往[[岡谷站|岡谷站]]，此列車並不是[[美鈴_(列車)|美鈴列車]]。

== 車站構造 ==
*有單式月台1面1線和島式月台1面2線，可以[[列車交會|列車交會]]
*[[地上車站|地上車站]]。
*由[[伊那市站|伊那市站]]管理的晚上與早上[[無人站|無人站]]。
*1號月台有站舍。1號和2號月台可使用[[平交道|平交道]]橫過路軌。
*多數列車會使用1號月台。
*3號月台旁有[[側線|側線]]，通常會讓列車過夜停泊。
*設有[[綠窗口|綠窗口]]。

== 历史 ==
* 1914年（[[大正|大正]]3年）10月31日 - 伊那電車軌道（1919年改名為[[伊那電氣鐵道|伊那電氣鐵道]]）宮田站至赤穗站延伸段啟用，新設'''赤穗站'''，為終點站。
* 1914年（大正3年）12月26日 - 伊那電車軌道線赤穗站至伊那福岡站延伸段啟用，車站變為中途站。
* 1943年（[[昭和|昭和]]18年）8月1日 - 伊那電氣鐵道線國有化，成為[[日本國有鐵道|國鐵]]飯田線車站。
* 1959年（昭和34年）10月1日 - 車站改名為'''駒根站'''。
* 1980年（昭和55年）7月4日 - 新的站舎建成。
* 1982年（昭和57年）11月1日 - 貨物起卸服裝停止，只有旅客服務。
* 1985年（昭和60年）3月14日 - 行李起卸服裝停止。
* 1971年（昭和46年）4月1日 - 旅客上落車站的制限廢除。
* 1987年（昭和62年）4月1日 - 因[[國鐵分割民營化|國鐵分割民營化]]，本站成為東海旅客鐵道的車站。
* 2002年（[[平成|平成]]14年）7月1日 - 晚上的窗口服務停止。
