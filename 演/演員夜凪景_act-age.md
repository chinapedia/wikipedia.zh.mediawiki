《'''act-age 新世代演員'''》（{{lang-ja|アクタージュ act-age}}），是由[[マツキタツヤ|マツキタツヤ]]負責故事原作、[[宇佐崎しろ|宇佐崎しろ]]負責作畫的漫畫作品。於《[[週刊少年Jump|週刊少年Jump]]》2018年8號開始連載<ref>{{Cite web |url = https://natalie.mu/comic/news/266139 |title = 天才的な演技力を持つ少女が女優への道を歩む、ジャンプ新連載「アクタージュ」 |website = natalie comic |date = 2018-01-22 |language = ja|}}</ref>。

故事原型為同一創作組合在《週刊少年Jump》2017年9號刊載的單篇作品《歡迎來到阿佐谷藝術高中影像科》<ref>{{cite web |url=https://ddnavi.com/news/350225/a/ |title= ジャンプ最新号に掲載された読み切りが話題に『阿佐ヶ谷芸術高校映像科へようこそ』|date= 2017-01-31|website= ダ・ヴィンチニュース|language=ja}}</ref>，講述擁有如天才般演技資質的少女，作為一名演員不斷成長的故事。

== 故事簡介 ==
想成為演員支持家庭收入的女高中生「夜凪景」，參加了「STARS」藝能事務所的演員選拔，雖然最終落選，擔任評審的電影導演「黑山墨字」卻看中她的才能，想讓她在自己籌拍的電影中出演主角。接受導演的邀請後，夜凪景加入「大黑天」工作室，為了讓她尚未顯現的才能更加成熟，黑山安排各種不同的工作讓夜凪磨練演技，目標是將她打造成一名出色的演員。

; {{Nihongo|巖裕次郎|いわお　ゆうじろう|Iwao Yuujirou}}
: 劇團「天球」的劇場總監，平時有吸菸習慣。會用「味道」來形容演員和感情表現，並表示自己只會起用「有味道」的演員。
: 被評為嚴厲的劇場總監，謠傳指導過程中會動用木刀，表現不好的演員會有生命危險。
: 接受黑山的推薦，看了夜凪在《Death Island》的表現後，同意讓她演出「銀河鐵道之夜」的舞台劇。
: 在舞台劇第一天，因胰臟癌而過世未能看見劇團的演出。

; {{Nihongo|明神阿良也|みょうじん　あらや|Myoujin Araya}}
: 劇團「天球」所屬的男性演員。21 歲，生日 10 月 10 日，血型 B 型，身高 178 cm。<ref name="vol4">漫畫單行本第四卷</ref>
:* 喜歡的東西：凡事容易感到厭倦，所以總是沒有固定，最近是蘋果和夜凪咖哩。
:* 討厭的東西：香水、說謊。
:* 興趣：跟蹤、搭訕 (不論男女老幼，幾乎是無意識之下就這麼做了)
:* 喜歡的電影：《[[音乐之声_(电影)|真善美]]》、《[[在黑暗中漫舞|在黑暗中漫舞]]》、《[[艺术家_(电影)|大藝術家]]》 (喜歡看音樂劇，但也知道自己是音痴)
: 在舞台劇「[[銀河鐵道之夜|銀河鐵道之夜]]」飾演喬凡尼。
: 巖裕次郎親自發掘的演員，擁有獨特的演技和語調，和巖裕次郎同樣用「味道」來形容其他演員。為了演出獵人的角色，取得狩獵許可證後獨自前往深山獵熊，也是他塑造角色的必經過程。他會將周圍的所有事物融入到角色塑造之中，並將過程形容為「吞食」，被稱為「舞台劇界的附身型變色龍」，屬於實力派的年輕演員。
: 不太會喝酒。

; {{Nihongo|青田龜太郎|青田　亀太郎（あおた　かめたろう）|Aota Kametarou}}
: 劇團「天球」所屬的男性演員。22 歲，生日 11 月 8 日，血型 O 型，身高 176 cm。<ref name="vol4"/>
:* 喜歡的東西：眼鏡、若隱若現且引人情慾的畫面。
:* 討厭的東西：隱形眼鏡。
:* 興趣：漫畫。
:* 喜歡的電影：《[[萬花嬉春|萬花嬉春]]》、《[[摩登大聖|摩登大聖]]》、《[[鬼靈精|鬼靈精]]》
: 暱稱是「龜」。在舞台劇「銀河鐵道之夜」飾演札內利。
: 個性輕浮，擅長浮誇或是俗氣的演技，是支撐劇團的幹部級人物。

; {{Nihongo|三坂七生|さんざか　ななお|Sanzaka Nanao}}
: 劇團「天球」所屬的女性演員。23歳，生日 3 月 7 日，血型 A 型，身高 154 cm。<ref name="vol4"/>
:* 喜歡的東西：所有種類的狗，特別是柴犬。
:* 討厭的東西：有貓的感覺的女人。
:* 興趣：夾娃娃機。(喜歡布偶，但操作夾娃娃機的技術很糟糕)
:* 喜歡的電影：《[[小飛象|小飛象]]》、《[[101忠狗|101忠狗]]》、《[[美女與野獸_(1991_電影)|美女與野獸]]》 (很喜歡[[迪士尼動畫工作室|迪士尼]]的動畫電影)
: 中學時看了劇團「天球」的演出之後立志成為演員。
: 個性冷淡且言詞辛辣，外表乍看相當平凡，但是摘下眼鏡和髮圈後，能夠讓人瞬間改變印象。青田稱其為「詐騙」一般的演技。
: 為了測試夜凪的演技，以「行駛中的火車」為題和她演了一場小短劇。

=== 杉並北高中電影研究社 ===
; {{Nihongo|朝陽日奈|朝陽　ひな（あさひ　ひな）|Asahi Hina}}
: 16歲。夜凪景的同班同學，電影研究社的成員之一，但加入的目的只是為了湊人數。似乎喜歡花井遼馬。是夜凪第一個嘗試交朋友的對象，但她認為夜凪是個故作清高的人而覺得反感。

; {{Nihongo|花井遼馬|はない　りょうま|Hanai Ryouma}}
: 16歲。電影研究社的成員之一，但加入的目的只是為了湊人數。在學校被周圍視為問題人物。沒有察覺朝陽日奈的感情，但聽她提起夜凪景的話題之後卻表現出濃厚興趣。

; {{Nihongo|吉岡新太|よしおか　あらた|Yoshioka Arata}}
: 17歲。夜凪景的同班同學，電影研究社的成員之一，是個電影狂熱愛好者。某一次分配到夜凪旁邊的座位，自從看到她的側臉之後，便開始構思以她為女主角的微電影。在夜凪為如何交朋友苦惱時，主動向她搭話，並邀請她加入電影研究社。

=== 其他人物 ===
; {{Nihongo|高田高二郎|たかだ こうじろう|Takada Koujirou}}
: 隸屬於三船俱樂部的男性演員。40歲。
: 在某齣時代劇中扮演[[大名|大名]]的隨從，其中有一幕是他要斬殺阻礙大名隊列前進的少女，然而卻被入戲太深的夜凪從旁飛踹了一腳。

; {{Nihongo|天知心一|あまち しんいち|Amachi Shinichi}}
: 只為錢而行動的演藝事業製作人，同時也是個守財奴。夜凪景在舞台劇「銀河鐵道之夜」嶄露頭角後，以金錢利誘對她進行挖角，在此之前杜撰了一篇加油添醋的報導，試圖將夜凪塑造成「悲劇的女主角」以博取鎂光燈。
