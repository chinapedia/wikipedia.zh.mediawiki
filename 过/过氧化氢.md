'''过氧化氢'''，[[分子式|分子式]]H<sub>2</sub>O<sub>2</sub>，是除[[水|水]]外的另一种[[氢|氢]]的[[氧化物|氧化物]]。粘性比水稍微高，化学性质不稳定，一般以30%或60%的[[水溶液|水溶液]]形式存放，其水溶液俗称'''双氧水'''。过氧化氢有很强的[[氧化性|氧化性]]，且具弱[[酸性|酸性]]。

== 性质 ==

纯过氧化氢是淡蓝色的粘稠液体，熔点{{convert|-0.43|°C}}，沸点{{convert|150.2|°C}}。凝固点时[[固体|固体]][[密度|密度]]为1.71g/cm<sup>3</sup>，密度随[[温度|温度]]升高而减小。它的[[缔合度|缔合程度]]比[[水|H<sub>2</sub>O]]大，所以它的[[介電係數|介電係數]]和[[沸点|沸点]]比[[水|水]]高。纯过氧化氢比较稳定，若加热到{{convert|153|°C}}便猛烈的分解为[[水|水]]和[[氧气|氧气]]。

过氧化氢可与水以任意比互溶，因其可以发生微弱[[电离|电离]]，所以溶液呈弱[[酸性|酸性]]。
:H<sub>2</sub>O<sub>2</sub> ↔ H<sup>+</sup> + HO<sub>2</sub><sup>-</sup>   K<sub>a1</sub>=2.4×10<sup>-12</sup>

:HO<sub>2</sub><sup>-</sup> ↔ H<sup>+</sup> + O<sub>2</sub><sup>2-</sup>    K<sub>a2</sub>≈10<sup>-25</sup>

过氧化氢与过量[[氢氧化钠|氢氧化钠]]反应的产物是Na<sub>2</sub>O<sub>2</sub>和H<sub>2</sub>O。而与[[氢氧化钡|氢氧化钡]]反应时产物为[[过氧化钡|过氧化钡]]。

过氧化氢分子为椅型结构，见下图。左图为气态时的结构，右图为固态[[晶体|晶体]]时的结构：

[[File:H2O2_gas_structure.svg|300px]][[File:H2O2_solid_structure.svg|300px]]

过氧化氢可溶于[[乙醇|乙醇]]、[[乙醚|乙醚]]，不溶于[[苯|苯]]。对有机物有很强的[[氧化|氧化]]作用，一般作为[[氧化剂|氧化剂]]使用。

== 催化劑 ==

体内的[[過氧化氫酶|過氧化氫酶]]（Catalase）可以催化雙氧水的分解反应，使其釋放出氧氣，转化为对有机体无毒的水：

: <math>\rm 2H_2O_2 (l) \longrightarrow 2H_2O (l) + O_2 (g)</math>
<!-- :2H<sub>2</sub>O<sub>2(l)</sub> → 2H<sub>2</sub>O<sub>(l)</sub> + O<sub>2(g)</sub> -->

== 发现 ==

1818年，L.J.Thenard发现水系无机物、[[有机物|有机物]]在自动氧化时，或者在[[生物|生物]]体内呼吸[[氧气|氧气]]时，在生成水之前会生成过氧化氢。

== 制备 ==

1950年代以前采用'''电解法'''制备过氧化氢——电解硫酸氢盐溶液（如[[硫酸氢铵|硫酸氢铵]]、[[硫酸氢钾|硫酸氢钾]]）得到过二硫酸盐<ref>电解：KHSO<sub>4</sub> → K<sub>2</sub>S<sub>2</sub>O<sub>8</sub>+ H<sub>2</sub>↑</ref>，再加入硫酸<ref>K<sub>2</sub>S<sub>2</sub>O<sub>8</sub> + H<sub>2</sub>SO<sub>4</sub> = K<sub>2</sub>SO<sub>4</sub> + H<sub>2</sub>S<sub>2</sub>O<sub>8</sub>使过二硫酸水解，水解过程分为两步：H<sub>2</sub>S<sub>2</sub>O<sub>8</sub> + H<sub>2</sub>O = H<sub>2</sub>SO<sub>5</sub> + H<sub>2</sub>SO<sub>4</sub>；
H<sub>2</sub>SO<sub>5</sub> + H<sub>2</sub>O = H<sub>2</sub>SO<sub>4</sub> + H<sub>2</sub>O<sub>2</sub></ref>得到过氧化氢。

也有[[水解|水解]][[过硫酸铵|过硫酸铵]]或[[过氧化钠|过氧化钠]]制取的：
:<Math>\rm (NH_4)_2S_2O_8 + 2H_2O {=\!=\!=\!=} H_2O_2 + 2NH_4HSO_4 </Math>

:<Math>\rm Na_2O_2 + 2H_2O {=\!=\!=\!=} H_2O_2 + 2NaOH </Math>

1953年，[[杜邦|杜邦]]公司采用[[蒽醌法|蒽醌法]]制备，以[[烷基|烷基]][[蒽醌|蒽醌]]如[[2-乙基蒽醌|2-乙基蒽醌]]为媒介物，循环氧化还原製得。现在世界各国基本上都是用这一技术。

过氧化氢是非常强的氧化剂，它和其他氧化剂的[[标准电极电势|标准电极电势]]值列在右面，值越高代表氧化性越强：

=== 分解反应 ===

过氧化氢可自发分解[[歧化|歧化]]生成[[水|水]]和[[氧气|氧气]]：

: <math>\rm 2H_2O_2  \longrightarrow 2H_2O  + O_2 </math>
<!-- :2 H<sub>2</sub>O<sub>2</sub> →  2 H<sub>2</sub>O + O<sub>2</sub> -->
该反应在[[热力学|热力学]]上自发进行：[[标准摩尔生成焓|Δ''H''<sup><s>o</s></sup>]]为−98.2 [[千焦|kJ]]·[[摩尔_(化学)|mol]]<sup>−1</sup>，[[吉布斯能|Δ''G''<sup><s>o</s></sup>]]为−119.2 kJ·mol<sup>−1</sup>，ΔS为70.5 J·mol<sup>−1</sup>·K<sup>−1</sup>。[[重金属|重金属]][[离子|离子]][[铁|Fe]]<sup>2+</sup>、[[锰|Mn]]<sup>2+</sup>、[[铜|Cu]]<sup>2+</sup>等对过氧化氢的分解有[[催化|催化]]作用。它们在酸性溶液中的[[电势|电势]]介于过氧化氢的电势（0.694~1.76V）之间。例如Fe<sup>3+</sup>，认为过氧化氢把Fe<sup>3+</sup>[[还原|还原]]为Fe<sup>2+</sup>,而本身被[[氧化|氧化]]成氧气，产生的Fe<sup>2+</sup>又被过氧化氢氧化为Fe<sup>3+</sup>，过氧化氢被还原成水。过氧化氢在[[酸|酸]]性和中性介质中较稳定，在[[碱|碱]]性介质中易分解。用[[波长|波长]]为320～380[[纳米|nm]]的光照射会使过氧化氢分解速度加快，故过氧化氢应盛于棕色瓶中并放在阴凉处。在处理无水或浓缩过氧化氢时，必须在无尘、无金属杂质等条件下进行，以防止发生[[爆炸|爆炸]]。
H<sub>2</sub>O<sub>2</sub>与Fe<sup>2+</sup>的混合溶液称为[[芬顿试剂|芬顿试剂]]（Fenton）。某些离子如Fe<sup>2+</sup>、Ti<sup>3+</sup>催化下，过氧化氢分解反应会生成[[自由基|自由基]]中间体HO·（[[羟基自由基|羟基自由基]]）和HOO·。
一般使用的双氧水中都会含有一定量的稳定剂，以减少过氧化氢的分解。常用的稳定剂包括：[[锡酸钠|锡酸钠]]、[[焦磷酸钠|焦磷酸钠]]、[[8-羟基喹啉|8-羟基喹啉]]和有机[[亚磷酸酯|亚磷酸酯]]。<ref>[http://www.h2o2.com/intro/faq.html#4 过氧化氢介绍]</ref>

=== 氧化还原反应 ===

过氧化氢可在水溶液中氧化或还原很多无机离子。用作[[还原剂|还原剂]]时产物为[[氧气|氧气]]；用作[[氧化剂|氧化剂]]时产物为[[水|水]]，其优点是氧化性强，还原产物为水，不引入杂质且不污染环境，因此过氧化氢是一种用途十分广泛的氧化剂。例如[[酸性|酸性]]溶液中，过氧化氢可将Fe<sup>2+</sup>氧化为Fe<sup>3+</sup>：

: <math>\rm 2 Fe^{2+}(aq) + H_2O_2 + 2 H^+(aq) \longrightarrow 2 Fe^{3+}(aq) + 2H_2O(l)</math>
<!-- :2 Fe<sup>2+</sup>(aq)  +  H<sub>2</sub>O<sub>2</sub>  + 2 [[水合氢离子|H<sup>+</sup>]](aq)  → 2 Fe<sup>3+</sup>(aq)  +  2H<sub>2</sub>O(l) -->
与过氧化氢作用，[[亚硫酸根|亚硫酸根]]（{{chem|SO|3|2-}}）可被氧化为[[硫酸根|硫酸根]]（{{chem|SO|4|2-}}），[[高锰酸钾|高锰酸钾]]在酸性溶液中会被还原为Mn<sup>2+</sup>。由于[[标准电极电势|标准电极电势]]的缘故，反应在不同[[pH|pH]]环境下进行的方向可能不同，如碱性溶液中，过氧化氢会将Mn<sup>2+</sup>氧化为Mn<sup>IV</sup>，以[[二氧化锰|MnO<sub>2</sub>]]形式生成。

过氧化氢还原[[次氯酸钠|次氯酸钠]]的反应可用于在实验室中制备氧气：

: <chem>{NaClO} + H2O2 -> {O2} + {NaCl} + H2O</chem>
<!-- :NaOCl + H<sub>2</sub>O<sub>2</sub> = O<sub>2</sub> + NaCl + H<sub>2</sub>O -->
有机化学中，过氧化氢常用作[[氧化剂|氧化剂]]，可将[[硫醚|硫醚]]氧化为[[亚砜|亚砜]]。[[甲基苯基硫醚|甲基苯基硫醚]]与其反应时，会被氧化为[[甲基苯基亚砜|甲基苯基亚砜]]，以甲醇作溶剂或[[三氯化钛|三氯化钛]]催化，产率为99%：

: <chem>{PhSCH3} + {H2O2} -> {PhS(O)CH3} + H2O</chem>
<!-- :Ph-S-CH<sub>3</sub>  +  H<sub>2</sub>O<sub>2</sub>  =  Ph-S(O)-CH<sub>3</sub>  +  H<sub>2</sub>O -->
过氧化氢的碱性溶液可用于富电子[[烯烃|烯烃]]（如[[丙烯酸|丙烯酸]]）的[[环氧化反应|环氧化反应]]，以及在[[硼氢化-氧化反应|硼氢化-氧化反应]]第二步中氧化[[烷基硼|烷基硼]]至[[醇|醇]]。

=== 生成过氧化物 ===

过氧化氢与很多无机或有机化合物反应时，过氧链保留并转移到另一分子上，生成新的[[过氧化物|过氧化物]]：

* 过氧化氢在低温下与[[铬酸|铬酸]]或[[重铬酸盐|重铬酸盐]]酸性溶液反应时，会生成不稳定的蓝色[[过氧化铬|过氧化铬]]CrO(O<sub>2</sub>)<sub>2</sub>，可用[[乙醚|乙醚]]或[[戊醇|戊醇]]萃取。这个反应可以用来检验过氧化氢和[[铬酸根|铬酸根]]或[[重铬酸根|重铬酸根]]。而在水溶液中过氧化铬会与过氧化氢进一步反应，蓝色迅速消失，得到氧气和铬离子。

:4H<sub>2</sub>O<sub>2</sub> + Cr<sub>2</sub>O<sub>7</sub><sup>2-</sup> + 2H<sup>+</sup> → 2CrO<sub>5</sub> + 5H<sub>2</sub>O

:7H<sub>2</sub>O<sub>2</sub> + 2CrO<sub>5</sub> + 6H<sup>+</sup> → 7O<sub>2</sub>↑ + 2Cr<sup>3+</sup> + 10H<sub>2</sub>O

* 过氧化氢与[[硼砂|硼砂]]反应会生成[[过硼酸钠|过硼酸钠]]，可用作消毒剂：
:: <math>\rm Na_2B_4O_7 + 4 H_2O_2 + 2 NaOH \longrightarrow 2 Na_2B_2O_4(OH)_4 + H_2O</math>

<!-- ::Na<sub>2</sub>B<sub>4</sub>O<sub>7</sub>  +  4 H<sub>2</sub>O<sub>2</sub>  +  2 NaOH  →  2 Na<sub>2</sub>B<sub>2</sub>O<sub>4</sub>(OH)<sub>4</sub>  +  H<sub>2</sub>O -->

* 过氧化氢可生成很多含有O<sub>2</sub><sup>2−</sup>过氧根离子的无机盐类，其中比较重要的如[[过氧化钙|过氧化钙]]、[[过氧化钠|过氧化钠]]和[[过氧化镁|过氧化镁]]。

* 与[[羧酸|羧酸]]（RC<sup>=O</sup><sub>-OH</sub>）反应，生成具氧化性的[[过氧酸|过氧酸]]（RC<sup>=O</sup><sub>-O-OH</sub>），可用于[[烯烃环氧化反应|烯烃环氧化反应]]等用途。与[[丙酮|丙酮]]反应生成炸药[[三过氧化三丙酮|三过氧化三丙酮]]（TATP），与[[臭氧|臭氧]]反应生成[[三氧化二氢|三氧化二氢]]，与[[尿素|尿素]]反应生成[[过氧化尿素|过氧化尿素]]。

* 过氧化氢与[[三苯基氧化膦|三苯基氧化膦]]生成酸碱[[加合物|加合物]]，有些反应中用作过氧化氢的等同试剂。

=== 碱性 ===

与水相比，过氧化氢的碱性要弱得多，只有与[[超强酸|很强的酸]]反应才会生成加合物。[[超强酸|超强酸]][[氟锑酸|HF/SbF<sub>5</sub>]]可将过氧化氢质子化，生成含[H<sub>3</sub>O<sub>2</sub>]<sup>+</sup>离子的产物。

== 应用 ==

一般低濃度（如3%）的過氧化氫，主要用於殺菌及外用的醫療用途，例如作為傷口消毒。檢驗血液的最佳比率:6%的雙氧水加上0.005%的[[亞甲藍|亞甲藍]]。至於較高濃度者（大於10%），則用於[[紡織品|紡織品]]、[[皮革|皮革]]、紙張、木材製造工業，作為[[漂白|漂白]]及去味劑。過氧化氫也是[[染髮劑|染髮劑]]的成份之一，还用作合成有机原料（[[邻苯二酚|邻苯二酚]]）的材料，医药、[[金属|金属]]表面处理剂，[[聚合引发剂|聚合引发剂]]等。还可用作[[火箭推进剂|火箭推进剂]]。

很多[[颜料|颜料]]、[[涂料|涂料]]中含有[[铅白|铅白]]。使用铅白的[[油画|油画]]、[[壁画|壁画]]等艺术品长时间暴露在空气中，与[[硫化氢|硫化氢]]作用生成[[硫化铅|硫化铅]]而变暗发黑，用过氧化氢涂刷后，会生成白色的[[硫酸铅|硫酸铅]]从而使其复原。

:PbS + 4H<sub>2</sub>O<sub>2</sub> → PbSO<sub>4</sub> + 4H<sub>2</sub>O

===危險行為===

# 萬一誤飲或以雙氧水[[灌肠_(医学)|灌肠]]，可能導致口腔或消化道的黏膜受侵蝕而導致發炎，嚴重時可能穿孔或出血。
# 注射雙氧水，將導致[[血栓|血栓]]，容易導致器官壞死，嚴重者可能會有生命危險。
# 一般的食品加工禁止使用雙氧水。

== 历史事件 ==
* 1934年7月16日，[[德国|德国]]使用火箭發動機的過氧化氫爆炸，炸死三人。
* 1939~1945年，[[第二次世界大戰|第二次世界大戰]]中，[[納粹德國|納粹德國]]在[[集中營|集中營]]中使用過氧化氫作為對人的致死試驗品。
* 2000年8月12日，[[库尔斯克号核潜艇|库尔斯克号核潜艇]]鱼雷舱中的鱼雷在发射时由于内部过氧化氢燃料与[[催化剂|催化剂]]接触，剧烈反应，导致爆炸，继而引起连锁反应，引爆了鱼雷舱中的所有鱼雷，彻底炸毁潜艇的前部，令潜艇最终沉没。全艇118人无人生还。
* 2005年7月7日08:50-9:47的[[伦敦地铁爆炸案|伦敦地铁爆炸案]]中，恐怖分子使用的[[炸药|炸药]]是"过氧化氢炸药"，[[三過氧化三丙酮|三過氧化三丙酮]]，俗称"TATP，其原理是它在爆炸时并不会产生任何火焰。因为只需很少的能量就可引发炸药爆炸。且这个过程并非氧化反应而是一个分解过程。在这个过程中，TATP分子释放出丙酮，使联在一起的氧原子散开，形成氧气和臭氧。这个过程释放出的能量足可使另一个分子发生化学反应，维持了反应的连续发生。一个TATP分子可以生成四个气体分子，这就是TATP会发生爆炸的原因。在不到一秒钟内，仅几百克的TATP就可产生成百上千升气体而引起爆炸，造成52人死亡。
