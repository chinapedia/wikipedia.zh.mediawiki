'''过氧酸'''（peroxy acid），简称'''过酸'''，是分子中含有[[过氧基|过氧基]] －O－O－ 的[[酸|酸]]类。可以分为[[无机|无机]]过氧酸和[[有机|有机]]过氧酸两类。

==无机过氧酸==
过氧酸（盐）主要由[[碳族元素|碳族]]、[[氮族元素|氮族]]和[[氧族元素|氧族元素]]的[[含氧酸|含氧酸]]（盐）所衍生出。但周期表中，[[3族元素|第三]]至[[7族元素|第七]]族的许多元素皆能生成过氧酸或过氧酸盐，包括[[硼|硼]]、[[铀|铀]]；[[碳|碳]]、[[锗|锗]]、[[锡|锡]]、[[钛|钛]]、[[锆|锆]]、[[铪|铪]]；[[氮|氮]]、[[磷|磷]]、[[砷|砷]]、[[钒|钒]]、[[铌|铌]]、[[钽|钽]]；[[硫|硫]]、[[硒|硒]]、[[碲|碲]]、[[铬|铬]]、[[钼|钼]]、[[钨|钨]]；[[锰|锰]]等。

无机过氧酸（盐）的一些例子有：
*[[过一硫酸|过一硫酸]]（H<sub>2</sub>SO<sub>5</sub>）
*[[过二硫酸|过二硫酸]]（H<sub>2</sub>S<sub>2</sub>O<sub>8</sub>）及其盐类
*[[过硝酸|过硝酸]]（HNO<sub>4</sub>）
*[[过一磷酸|过一磷酸]]（H<sub>3</sub>PO<sub>5</sub>）
*[[过二磷酸|过二磷酸]]（H<sub>4</sub>P<sub>2</sub>O<sub>8</sub>）
*[[过碳酸钠|过碳酸钠]]
*[[过硼酸钠|过硼酸钠]]

除过二硫酸外，大多数的过氧酸皆可由[[过氧化氢|过氧化氢]]与相应的[[正酸|正酸]]（或[[酸酐|酸酐]]或其[[盐|盐]]）作用而制得。所有过氧酸或过氧酸盐用[[稀硫酸|稀硫酸]]进行酸化时都能放出过氧化氢。

一个检验过氧酸及其盐的方法是 Riesenfeld 反应，即用[[碳酸氢钠|碳酸氢钠]]把30％[[碘化钾|碘化钾]]溶液的 [[pH|pH]] 值调到 7.5～8，然后加入待测试液。若试液含有过氧酸或过氧酸盐，则在条件下即将碘化钾氧化为[[碘|碘]]单质。

==有机过氧酸==
有机过氧酸一般是[[有机酸|有机酸]]中的 －OH 被 －OOH 替换后形成的化合物，即过氧化氢中的一个氢原子被[[酰基|酰基]]所取代后形成的化合物。最常见的有机过氧酸是过氧[[羧酸|羧酸]]，通式为 R-C(O)-O-OH。它又可大致分为[[脂肪族|脂肪族]]有机过氧酸以及[[芳香族|芳香族]]和[[环状化合物|环状]]有机过氧酸两类。

有机过氧酸的一些例子有：
*[[过氧甲酸|过氧甲酸]]
*[[过氧乙酸|过氧乙酸]]
*[[过氧三氟乙酸|过氧三氟乙酸]]
*[[过氧丙酸|过氧丙酸]]
*[[过氧丁酸|过氧丁酸]]
*[[过氧异戊酸|过氧异戊酸]]
*[[长链过氧脂肪酸|长链过氧脂肪酸]]
*[[过氧苯甲酸|过氧苯甲酸]]
*[[间氯过氧苯甲酸|间氯过氧苯甲酸]]
*[[硝基过氧苯甲酸|硝基过氧苯甲酸]]
*[[单过氧邻苯二甲酸|单过氧邻苯二甲酸]]

制备方法：

1、脂肪族过氧酸最普遍和成熟的制备方法，是用30～98％浓度的[[过氧化氢|过氧化氢]]与羧酸在酸催化下反应制取。这个方法最早是由 D'Ans 等人在20世纪初为制备短链水溶性过氧酸而研制开发的。水溶性羧酸在酸催化剂存在下，与过氧化氢充分混合即可得到相应的过氧酸。酸性催化剂最好是[[硫酸|硫酸]]或[[磺酸型离子交换树脂|磺酸型离子交换树脂]]，但酸性较强的羧酸如甲酸在不存在酸催化剂的条件下，也会很快与过氧化氢发生反应。这是一个[[可逆反应|可逆反应]]，工程上一般用[[真空恒沸气提|真空恒沸气提]]方法将反应产生的水带走，使产率增高。有时也加入金属失活剂或稳定剂以减少过氧酸的分解。

对于芳香族过氧羧酸来说，芳香酸一般与过氧化氢相互不溶，故难以发生反应。而且若向其中加入硫酸，常会引起剧烈的芳环[[氧化反应|氧化反应]]而易发生事故。1970年代时诞生了一种新的合成工艺，即先将反应物用水稀释，并加入[[甲磺酸|甲磺酸]]进行反应，接着用[[苯|苯]][[萃取|萃取]]和水洗，以除去过量的甲磺酸和过氧化氢，这样就可以直接得到芳香过氧酸溶液。这种方法也适用于一些不溶于水的芳香过氧酸的制取。

:<math>\rm \ RCO_2H+H_2O_2\rightleftharpoons RCO_3H+H_2O</math>

2、一些芳香过氧酸也可以通过液相中芳香[[醛|醛]]的[[自由基|自由基]]引发的[[自氧化反应|自氧化反应]]来制备。
:<math>\rm \ RCHO + O_2 \rightarrow RCO_3H</math>

3、通过[[有机过氧化物|有机过氧化物]]在强碱存在下水解来制备相应的过氧酸。这是一种较为古老的方法，但目前仅用于一些芳香过氧酸的制备。
:<math>\rm \ RC(O)OOC(O)R + H_2O \rightarrow RCO_3H + RCO_2H</math>

4、[[酸酐|酸酐]]与30～98.5％浓度的过氧化氢反应生成过氧酸。此法用于制取单过氧邻苯二甲酸。
:<math>\rm \ RC(O)OC(O)R + H_2O_2 \rightarrow RCO_3H + RCO_2H</math>

5、[[酰卤|酰卤]]、[[过氧化钠|过氧化钠]]或过氧化氢与碱反应生成过氧酸。
:<math>\rm \ RC(O)Cl + H_2O_2 \rightarrow RCO_3H + HCl</math>

6、[[硼酸|硼酸]]与羧酸酐与过氧化氢作用，得到很高产率的无水过氧酸。这种方法可用于制备过氧乙酸、过氧丙酸、过氧丁酸、[[过氧硬脂酸|过氧硬脂酸]]和[[过氧棕榈酸|过氧棕榈酸]]等。
:<math>\rm \ B(OCOR)_3 + 3H_2O_2 \rightarrow 3RCO_3H + H_3BO_3</math>

7、过硼酸钠与酸酐（或二乙酸乙酰过氧化物）作用生成过氧乙酸和单过氧苯二酸。
:<math>\rm \ RC(O)OC(O)R + NaBO_3 \rightarrow RCO_3H + RCO_2H</math>

8、[[乙烯酮|乙烯酮]]与过氧化氢作用得到过氧化物。工业上用这种方法制取无水过氧乙酸已获得成功。
:<math>\rm \ H_2CCO + H_2O_2 \rightarrow CH_3CO_3H</math>

9、[[臭氧|臭氧]]催化下，经过[[紫外线|紫外线]]引发，通过羧酸与氧气反应直接合成过氧酸。
:<math>\rm \ RCO_2H + \tfrac{1}{2} O_2 \rightarrow RCO_3H</math>

10、[[臭氧化物|臭氧化物]]与脂肪族羧酸作用并经溶剂分解得到过氧酸。

用途：有机过氧酸是[[烯烃环氧化反应|烯烃环氧化反应]]、[[Baeyer-Villiger氧化重排反应|Baeyer-Villiger氧化重排反应]]、[[硫醚|硫醚]]氧化等其他氧化反应中的试剂。
