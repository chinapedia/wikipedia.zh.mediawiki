'''六戶町'''（{{jpn|j=六戸町|hg=ろくのへまち|rm=Rokunohe machi|f=y}}）是[[青森縣|青森縣]][[上北郡|上北郡]]東南部的[[町|町]]。本町擁有平坦的地形，沒有明顯的高山或湖沼，有以[[十和田湖|十和田湖]]為源頭的奧入瀨川在町內流動，其流域發展出廣闊的水田。積雪不多，氣候十分穩定。

== 歷史 ==
古時，「六戶」的地名，並不單止是單獨的村名，也包含了現在六戶町以外的奧入瀨町和三澤市南部，指奧入瀨川（舊稱：相坂川）流域一帶的名稱。
* 1889年（明治22年）4月1日 - 折茂村、柳町村、小平村、鶴喰村、犬落瀨村、上吉田村、下吉田村合併為'''六戶村'''。
:（各村的名字以[[大字|大字]]繼續存在。）
* 1948年（昭和23年）2月11日 - 三戶村部份地區併入六戶村。
* 1957年（昭和32年）10月1日 - 施行町制，改為'''六戶町'''。
* 2007年（平成19年） - 迎接町制施行50周年。

== 地理 ==
位於[[上北郡|上北郡]]東南部。町的南部有以[[十和田湖|十和田湖]]為源頭的奧入瀨川。

中北部是被稱作為三本木原的台地，有從奧入瀨川以十和田市法量分支出來的人工河川：稻生川（三本木原用水路）。

北部鄰近[[三澤市|三澤市]]的邊界，亦有姉沼川的支流。

== 行政 ==
* 町長：吉田豐（1996年（平成8年）1月28日就任、五任）
* 町議會：12人
在平成的大合併中，原本與下田町及百石町（現[[奧入瀨町|奧入瀨町]]）合併作目的而設立了合併協會，不過後期因為特例債的運用方針而對立並且脫離。受到這個的緣故，町內的合併贊成派和反對派因而對立，並引發起罷免町長及進行了町長選舉，結果反對派勝利，因此繼續以獨立的町而存在。
反對派稱，雖然前述的2町在地理及文化上有很強的關聯，不過在電話（市外局號）、警察、消防、保健醫療及基礎設施等方面在管轄上有很多不同的地方，因此沒有充分時間作出調整為反對合併的理由之一。
