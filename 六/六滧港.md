'''六滧港'''（原名六滧竖港，滧：yáo）是[[上海市|上海市]][[崇明县|崇明县]]东部的一条河流，该河自[[长江口|长江口]]南水道南六滧港水闸起，经[[向化镇|向化镇]]，在北六滧港水闸流入长江口北水道，全长约12.6公里。该河枯水期时航道水深1到2米，可供50至80吨级的船舶航行。<ref>{{cite web |url=http://www.shtong.gov.cn/newsite/node2/node2245/node70862/node70867/node70901/node70905/userobject1ai73632.html |language=zh-hans |title=专业志 >> 上海地名志 >> 第一篇自然地理实体地名 >> 第一章海、河、湖 >> 第二节　河流 |publisher=上海市地方志办公室 |accessdate=2011-08-13 |deadurl=yes |archiveurl=https://web.archive.org/web/20150203074842/http://www.shtong.gov.cn/newsite/node2/node2245/node70862/node70867/node70901/node70905/userobject1ai73632.html |archivedate=2015-02-03 }}</ref>六滧港原位于大港东面1.5公里处，北止于赵公堤，1933年民国政府曾疏浚此河。<ref>{{cite web|url=http://www.shtong.gov.cn/newsite/node2/node4/node2250/node4426/node16056/node16299/node16322/userobject1ai5512.html | language=zh-hans | title=区县志 >> 县志 >> 崇明县志 >> 卷十二水利 >> 三、干河疏浚治理 >> (一)疏浚整治工程 |publisher=上海市地方志办公室 |accessdate=2011-08-13 }}</ref>1958年在该港南口修建了水闸，并在原港新开2.5公里至向化的河段，再向沿原港疏浚拓宽，交[[北横引河|北横引河]]于二堡镇后折向西接[[茅家港|茅家港]]，工程全长12.6公里，河道底部最宽处10米，最窄处5米，海拔在1.5米上下。1978年又疏浚拓宽[[南横引河|南横引河]]至向化五大队长4公里的河段，河底拓宽至15米，坡比1∶3。次年据崇明县规划，新开六滧港南段，并直线向北挖掘，共新开河道5.77公里。北六滧港水闸在1980年建成后，新建的2.18公里河道与原河道相接，贯穿崇明岛南北。<ref>{{cite web|url=http://www.shtong.gov.cn/newsite/node2/node4/node2250/node4426/node16056/node16299/node16322/userobject1ai5518.html | language=zh-hans | title=区县志 >> 县志 >> 崇明县志 >> 卷十二水利 >> 三、干河疏浚治理 >> (七)六滧港 |publisher=上海市地方志办公室 |accessdate=2011-08-13 }}</ref>
