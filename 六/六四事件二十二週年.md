2011年是[[辛亥革命|辛亥革命]]100週年及'''[[六四事件|六四事件]]二十二週年'''。從1989年的六四事件發生起至今，[[香港市民支援愛國民主運動聯合會|香港市民支援愛國民主運動聯合會]]（支聯會）一直舉辦[[維園六四燭光晚會|維園六四燭光晚會]]。而每年堅持參加燭光晚會的支聯會主席[[司徒華|司徒華]]先生於2011年初病故，外界觀望香港[[民主|民主]]動員力量是否會消減。此次維園六四燭光晚會的主題「平反六四，革命尚未成功；建設民主，同志仍需努力」是由司徒華生前親自勾勒。

== 中國大陆 ==
六月四日臨近前[[中國|中國]]一批支持民主人士被[[國保|國保]]約談，禁止他們參加任何形式的悼念活動，甚至是連白色衣服也不得穿著、不許接受媒體採訪、不得外出聚會。還有許多中國[[異議人士|異議人士]]遭到了[[中國政府|中國政府]]的監視與警告，[[北京|北京]]的大街小巷上警察與便衣開始增多，連不起眼的小地方有時都會出現。由於2011年初發生的中東革命與“[[中國茉莉花革命|中國茉莉花革命]]”後中共大為緊張，於是限制學生的自由活動，甚至組織活動，北京高校附近便衣、警車暴增<ref>{{cite web|url=http://ap.ntdtv.com/b5/20110528/video/63304.html|title=【禁聞】六四敏感日 多人被約談 禁穿白衣|accessdate=2011-05-28 |publisher=新唐人 |language=zh}}</ref>。

特別的是有一位廖姓香港人前去[[北京大學|北京大學]]購時書在經過一條北大學生宿舍通往飯堂必經之路時發現水泥路面上有刻著「平反六四」四個字（根據[[混凝土|混凝土]]凝固程度應為5月初左右所刻），廖先生拍照時周邊學生只是看了一下並沒有圍觀，很多學生保持低調未向校方舉報。

公開呼籲平反六四與簽署[[零八憲章|零八憲章]]的北大教授[[夏業良|夏業良]]被有關部門約談，零八憲章首批簽署者和北京自由撰稿人[[莫之許|莫之許]]也被警方警告。

== 香港 ==
[[File:VIIV_label_in_front_of_Times_Square.JPG|thumb]]附近]]
本次[[香港|香港]]的主體紀念活動仍與往年一樣為[[維園六四燭光晚會|維園六四燭光晚會]]，主題是「平反六四，革命尚未成功；建設民主，同志仍需努力」，有15萬人參加，平了歷年悼念人數的紀錄。新增的是[[教協|教協]]和[[支聯會|支聯會]]在6月3日晚至6月4日舉行主要由學生參與的22小時體驗營活動，目的是讓年輕人更了解六四。

5月18日[[泛民主派|泛民主派]]議員、[[支聯會|支聯會]]主席[[李卓人|李卓人]]於立法會上再次提出[[平反六四議案|“毋忘六四事件，平反八九民運”議案]]，與往12年同樣遭[[建制派|建制派]]抵制未能通過。5月22日支聯會舉行了主旨為要求平反六四與釋放[[艾未未|艾未未]]的放飛“民主風箏行動”，民主風箏的緣由是六四學運時北京學生放風箏表達對民主的嚮往。支聯會還於5月28日在[[銅鑼灣|銅鑼灣]][[時代廣場_(香港)|時代廣場]]豎立了[[民主女神像|民主女神像]]來宣傳六四22周年紀念活動，今次沒有與2010年一樣遭到[[香港警察|香港警方]]與[[食物環境衛生署|食環署]]干預，5月29日舉辦了愛國民主大遊行。

== 臺灣 ==
臺灣有由六四學生領袖[[王丹|王丹]]，和其他如[[黃俊傑|黃俊傑]]、[[金昱|金昱]]、[[林家興|林家興]]、[[凃京威|凃京威]]、[[吳君薇|吳君薇]]、[[周慶昌|周慶昌]]等學生組成的「臺灣學生促進中國民主化聯合工作小組」召集的於[[中正紀念堂|中正紀念堂]]舉辦的「六四」二十二周年臺灣悼念晚會，約有400人參加<ref>[http://news.ltn.com.tw/news/life/breakingnews/504046 台跨校串聯悼六四　籲中共認錯]</ref><ref>[http://www.appledaily.com.tw/appledaily/article/international/20110605/33437134/ 自由廣場 空椅挺艾未未]</ref>。

== 日本 ==
六四紀錄片《[[長城外|長城外]]》在[[東京|東京]]於5月21日首放，在六四此天公映。此片由在日的作家翰光用了3年時間製作，內容為20位海外流亡的名人講述從文革、[[西單民主牆|西單民主牆]]到六四的這段歷史，受訪者包括[[楊建利|楊建利]]、[[王丹|王丹]]、[[高行健|高行健]]、[[鄭義|鄭義]]、[[黃翔|黃翔]]等人<ref>{{cite web|url=http://big5.soundofhope.org/programs/162/189776-1.asp|title=六四紀錄片《長城外》東京首映|accessdate=2011-05-28|publisher=希望之聲|language=zh}}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。
