'''六義園'''（{{jpn|j=六義園|hg=りくぎえん}}）是[[日本|日本]][[東京都|東京都]][[文京區|文京區]][[本駒込|本駒込]]六丁目的[[東京都立公園|都立庭園]]。

== 沿革 ==
六義園是[[德川將軍家|德川]]五代[[征夷大將軍|將軍]][[德川綱吉|德川綱吉]][[側用人|側用人]][[柳澤吉保|柳澤吉保]]在下屋敷打造的{{link-ja|大名庭園|大名庭園}}。1695年（元祿8年），柳澤拜領約2萬7千坪的[[加賀藩|加賀藩]]舊下屋敷，之後花費七年時間造山、引入{{link-ja|千川上水|千川上水}}水源造池，完成此座回遊式築山泉水庭園。

「六義園」的名稱是來自[[紀貫之|紀貫之]]《[[古今和歌集|古今和歌集]]》序文的和歌六體「六義」（むくさ）<ref>「六義」源自於《[[詩經|詩經]]》體裁「風」、「雅」、「頌」與表現手法「賦」、「比」、「興」。[[紀貫之|紀貫之]]引用來表示和歌六體。</ref>。

1702年（元祿15年），庭園與下屋敷一同完成。之後將軍綱吉曾多次到訪，光是有紀錄的就高達58次，可見吉保的寵臣地位與當時庭園的高評價<ref>依據紀錄，當時將軍世子、之後的六代將軍[[德川家宣|德川家宣]]，與[[紀州藩|紀州藩]]主、之後的八代將軍[[德川吉宗|德川吉宗]]也曾多次到訪六義園。</ref>。

柳澤家下任當主吉里由[[甲府藩|甲府]]轉封{{link-ja|大和郡山藩|大和郡山藩|大和郡山}}，六義園逐漸荒廢。

明治初期，[[三菱財閥|三菱財閥]]創業者[[岩崎彌太郎|岩崎彌太郎]]收購六義園，整頓自維新後荒廢的庭園，並在周邊建造紅磚牆。在經歷[[關東大地震|關東大地震]]後，1938年（昭和13年）贈與[[東京市|東京市]]。1953年（昭和28年）被指定為特別名勝。

== 特色 ==
[[File:Togetsukyo_(Crossing-Moon_Bridge).jpg|thumb]]
六義園以[[杜鵑花|杜鵑花]]聞名，「說到駒込就是杜鵑綻開之街」（駒込と言えばツツジの花の咲く街），是此地象徵性的存在。庭園入口附近的[[枝垂櫻|枝垂櫻]]也是著名賞櫻景點。草皮維護也十分完備，是都內頗具代表性的日本庭園，海外遊客眾多。

== 庭園資訊 ==
* 所在地：東京都文京區本駒込6丁目
* [[東日本旅客鐵道|JR]][[山手線|山手線]]或[[東京地下鐵|東京地下鐵]][[東京地下鐵南北線|南北線]][[駒込站|駒込站]]步行7分。
*旺季<ref> 3月下旬的枝垂櫻、黃金周的杜鵑花季、11月下旬～12月上旬的紅葉期等。</ref>開放車站正面的染井門。
* [[都營地下鐵|都營地下鐵]][[都營地下鐵三田線|三田線]][[千石站|千石站]]步行10分。
* 開園時間：9:00～17:00（入園截止至16:30）。
* 停車場：無。

== 活動 ==
櫻花與紅葉時期有夜間點燈。
* 3月下旬：正門附近的枝垂櫻
* 11月下旬～12月上旬：紅葉
