'''六月节'''也被称作是'''六月独立日'''，'''自由日'''或'''解放日'''，是一个[[美国|美国]]的节日，为了纪念美国[[德克萨斯|德克萨斯]][[美国州份|州]]在1865年6月宣布[[解放奴隸宣言|废除]][[奴隶制|奴隶制]]的事件，更一般地，是为了纪念[[非裔美国人|美国南部黑人]]奴隶的解放。这个节日于6月19日举行庆祝活动，六月节这个词是''六月''和''第十九''的[[混成詞|混成词]]，<ref>{{Template:Cite news|url = http://www.blackvoicenews.com/news/46366-juneteenth-celebrated-in-coachella.html|title = Juneteenth Celebrated in Coachella|date = June 22, 2011|work = Black Voice News}}</ref><ref>{{Template:Cite encyclopedia|url = http://www.merriam-webster.com/dictionary/juneteenth|title = Juneteenth|work = Merriam Webster Dictionary|accessdate = 2013-05-30}}</ref> 在大多数州中，六月节被当作一个国家的节日或特殊的日子来进行庆祝。

这个节日主要是在当地进行庆祝。庆祝的传统主要包括一起阅读《[[解放黑人奴隶宣言|解放黑人奴隶宣言]]》，吟唱传统歌曲“Swing Low, Sweet Chariot”以及“扬声高唱”，阅读著名非裔美国作家[[拉尔夫·艾里森|拉尔夫·艾里森]]和[[马娅·安杰卢|马娅·安杰卢]]的作品。<ref>Taylor, 2002. pp. 28–29.</ref> 庆典活动包括游行、马术竞赛、街头集会、野炊郊游、家庭聚会、公园聚会、重现历史以及[[選美活動|六月节选美比赛]]等。<ref name="celeb">{{Template:Cite web|url = http://www.juneteenth.com/howtocelebrate.htm|title = How to Celebrate|website = Juneteenth.com|accessdate = June 19, 2014}}</ref>

== 历史 ==
[[File:Ashton_Villa_Galveston_Texas.jpg|thumb]]
在[[南北战争|美国内战]]期间，总统[[亚伯拉罕·林肯|亚伯拉罕·林肯]]在1862年9月22日颁布了《[[解放黑人奴隶宣言|解放黑人奴隶宣言]]》，并于1863年1月1日开始生效。该宣言宣布释放在叛乱的[[美利坚联盟国|美利坚联盟国]]内的所有奴隶或者不属于美利坚联盟的奴隶（这就不包括被占领的田纳西州，弗吉尼亚州和较小的路易斯安那州）。它同时宣布联盟可招募之前的奴隶和自由的黑人加入军队，此次招募开始于1963年的春天。奴隶们常常逃到联盟战线上去寻求保护，而且许多人都开始在军队服役。在一些区域，违禁品营地被改造成房子，供自由人暂时居住，也有部分被改造为学校，同时还为成年人提供工作。林肯敦促留在联盟内的边境州，根据逐步废除奴隶制度和补偿制度的条款，释放奴隶，但是没有人这样做。这些奴隶直到战争结束才被解放。<span class="cx-segment" data-segmentid="103"></span>

即使奴隶获得了自由，这也是一个艰难的时代。违禁品营地十分拥挤，卫生条件很差，大多数的军事营也是如此。因此双方很多士兵都死于疾病而非战争所受的伤害，由于战争导致的社会混乱和普遍的恶劣条件，从1862到1870年，很多之前的奴隶都死于疾病，包括天花传染病。<ref>{{Template:Cite book|first = Jim|last = Downs|title = Sick from Freedom: African-American Illness and Suffering During the Civil War and Reconstruction|url = {{google books|h8u7q5CB9yYC|plainurl=yes}}|edition = First|location = New York|publisher = Oxford University Press|date = 2012}}</ref>

由于在地理位置上的独立性，德州没有成为战场。当地的奴隶除了逃跑外，都没有受到奴隶解放宣言的影响。<ref name="cruz">{{Template:Cite news|url = http://www.time.com/time/nation/article/0,8599,1815936,00.html|title = A Brief History of Juneteenth|date = 2008-06-18|first = Gilbert|last = Cruz|work = Time magazine|accessdate = 2013-05-30}}</ref> 种植园主和其他奴隶主从东部各州移民到德州来逃避战争，许多人带着他们的奴隶财产一起逃走，到美国内战结束时德州的奴隶数量剧增。<ref name="gates">{{Template:Cite web|url = http://www.pbs.org/wnet/african-americans-many-rivers-to-cross/history/what-is-juneteenth/|first = Henry Louis, Jr.|last = Gates|title = What Is Juneteenth?|agency = Originally posted on ''The Root''|website = The African Americans: Many Rivers to Cross|publisher = PBS|date = |access-date = 30 September 2014}}</ref>

到1865年，德州约有25万奴隶。<ref name="gates">{{Template:Cite web|url = http://www.pbs.org/wnet/african-americans-many-rivers-to-cross/history/what-is-juneteenth/|first = Henry Louis, Jr.|last = Gates|title = What Is Juneteenth?|agency = Originally posted on ''The Root''|website = The African Americans: Many Rivers to Cross|publisher = PBS|date = |access-date = 30 September 2014}}</ref> 由于战争结束的消息传播缓慢，直到1865年6月才传到德州，密西西比的军队直到6月2号才投降。1865年6月18日，[[北軍|联盟]]将军戈登·格兰杰抵达[[加尔维斯顿岛|加尔维斯顿岛]]，2000联邦军代表联邦政府占领了德州。<ref name="cruz">{{Template:Cite news|url = http://www.time.com/time/nation/article/0,8599,1815936,00.html|title = A Brief History of Juneteenth|date = 2008-06-18|first = Gilbert|last = Cruz|work = Time magazine|accessdate = 2013-05-30}}</ref> 6月19日，站在加尔维斯顿的阿什顿维拉家的阳台上，格兰杰大声朗读了“第三号将军令”，宣布了奴隶的完全[[解放|解放]]。<span class="cx-segment" data-segmentid="119"></span>
[[File:Emancipation_Day_in_Richmond,_Virginia,_1905.jpg|thumb]]
加尔维斯顿之前的奴隶们在听到公告之后跑到大街上欢庆，尽管之后的很多年，他们还要通过改变而努力工作，以对抗白人的阻挠。但是在第二年，获得自由的人在德州组织了第一次年度六月节庆典。<ref name="TXJ19">{{Template:Cite web|url = http://www.tsl.state.tx.us/ref/abouttx/juneteenth.html|title = Juneteenth|website = Texas State Library and Archives Commission|accessdate = 2006-07-06}}</ref> 由于国家资助建立隔离设施，一些城市禁止黑人使用公共公园。获得自由的人筹集资金去购置土地来举办他们的庆祝活动，例如休斯顿的解放公园，梅克西亚的布克·华盛顿公园，奥斯汀的解放公园等。<ref name="gates">{{Template:Cite web|url = http://www.pbs.org/wnet/african-americans-many-rivers-to-cross/history/what-is-juneteenth/|first = Henry Louis, Jr.|last = Gates|title = What Is Juneteenth?|agency = Originally posted on ''The Root''|website = The African Americans: Many Rivers to Cross|publisher = PBS|date = |access-date = 30 September 2014}}</ref><ref name="TXJ19">{{Template:Cite web|url = http://www.tsl.state.tx.us/ref/abouttx/juneteenth.html|title = Juneteenth|website = Texas State Library and Archives Commission|accessdate = 2006-07-06}}</ref>

在20世纪初，经济和政治力量导致六月节的庆祝活动减少。从1890年到1908年，德州和其他原来的邦联州都通过了新的宪法和修正案，实际上剥夺了黑人的公民权，将他们排除在政治进程之外。白人民主党占统治地位的州通过了吉姆克劳法，以加强黑人的二等地位。经济大萧条迫使许多黑人离开农场去城市找工作。在城市环境中，这些非裔美国人很难请一天假来庆祝这个节日。从1940年到1970年的第二波大迁徙中，500多万黑人离开德州，路易斯安那州和南部的其他地区来到北部和西部海岸，在那里他们可以为第二次世界大战的国防工业工作。正如历史学家威尔克森•伊莎贝尔写到：“来自德州的人们将六月节带到了洛杉矶、奥克兰、西雅图和其他他们去到的地方。”<ref>{{Template:Cite book|first = Isabel|last = Wilkerson|title = The Warmth of Other Suns: The Epic Story of America's Great Migration|url = {{google books|Y03WKII5m7QC|plainurl=yes}}|location = New York|publisher = Random House|date = 2010|access-date = June 19, 2015}}</ref>

到20世纪五六十年代，民权运动将非裔美国青年的集中于努力争取种族平等，改变他们的未来。但是很多人将他们的努力与历史上他们祖先的努力联系到了一起。在接下来的1968年，拉尔夫·阿伯内西在华盛顿特区发起穷人运动，很多参加者回到家里，在那些之前没有庆祝过六月节的地方开始了六月节的庆祝活动。

自20年代80年代到90年代，六月节在非裔美国人群体中被更为广泛地庆祝。1994年，一群社团领导人聚集到路易斯安那州的[[新奥尔良|新奥尔良]]基督教团结浸信会教堂，一起商讨全国性的六月节庆祝活动。<ref>{{Template:Cite news|url = http://newsone.com/2021601/juneteenth-history|title = Juneteenth: Celebrating The Early Moments Of Freedom Today|publisher = News One|date = June 19, 2012|accessdate = June 19, 2014|last = Chandler|first = D.L.}}</ref> 1994年保罗在密歇根州的弗林特开始了六月节的社区庆祝活动，正如他所说，“……这是我们快乐的一天。”<ref name="moskin2004">{{Template:Cite news|url = http://www.nytimes.com/2004/06/18/national/18june.html|title = An Obscure Texas Celebration Makes Its Way Across the U.S.|last = Moskin|first = Julie|work = The New York Times|date = 2004-06-18|accessdate = 2011-04-28}}</ref> 六月节的非正式纪念活动已经传播到其他的州，包括缅因州的波特兰，其中一部分是由得克萨斯州人传播的。<ref name="moskin2004">{{Template:Cite news|url = http://www.nytimes.com/2004/06/18/national/18june.html|title = An Obscure Texas Celebration Makes Its Way Across the U.S.|last = Moskin|first = Julie|work = The New York Times|date = 2004-06-18|accessdate = 2011-04-28}}</ref> 海外侨民也在其他城市庆祝，比如巴黎。除了那些个人团体以外，一些驻扎在其他国家的美国士兵也会庆祝这个节日。<ref name="moskin2004">{{Template:Cite news|url = http://www.nytimes.com/2004/06/18/national/18june.html|title = An Obscure Texas Celebration Makes Its Way Across the U.S.|last = Moskin|first = Julie|work = The New York Times|date = 2004-06-18|accessdate = 2011-04-28}}</ref><ref name="jtcom02">{{Template:Cite web|url = http://www.juneteenth.com/international.htm|title = The World Celebrates Freedom|website = Juneteenth.com|accessdate = 2006-06-19}}</ref>

国家六月节纪念基金会等组织正在努力争取国会批准六月节为一个国家纪念日，<ref name="gates">{{Template:Cite web|url = http://www.pbs.org/wnet/african-americans-many-rivers-to-cross/history/what-is-juneteenth/|first = Henry Louis, Jr.|last = Gates|title = What Is Juneteenth?|agency = Originally posted on ''The Root''|website = The African Americans: Many Rivers to Cross|publisher = PBS|date = |access-date = 30 September 2014}}</ref> 其他人正在努力准备世界范围内的150周年纪念日庆祝活动。

== 官方地位 ==
在1980年，德州第一个将六月节作为州立节日，由民主党州众议员爱德华提出然后立法确定。该项立法遭到了达拉斯郡的非裔美国人代表克雷的反对，他宣称把六月节设为假期 是“不正当的”，并贬低了庆祝活动，认为该活动是、“就像是在国会的场地隆重地傻笑并爆裂西瓜一样”。<ref>{{Template:Cite web|url = http://www.texasmonthly.com/story/independence-day/page/0/1|first = Anne|last = Dingus|title = Once a Texas-only holiday marking the end of slavery, Juneteenth is now celebrated nationwide with high spirits and hot barbecue|date = June 2001|work = [[Texas_Monthly|Texas Monthly]]|accessdate = 11 October 2013}}</ref> 六月节是一个“部分员工的假期”，州办公部门不应该放假，只有部分职员以“浮休”的方式放假。公立学校并没有受到影响，因为6月19号已经进入暑假。<span class="cx-segment" data-segmentid="188"></span>

到2008年，美国接近一半的州将六月节视为一个正式的节日来庆祝。<ref name="cruz">{{Template:Cite news|url = http://www.time.com/time/nation/article/0,8599,1815936,00.html|title = A Brief History of Juneteenth|date = 2008-06-18|first = Gilbert|last = Cruz|work = Time magazine|accessdate = 2013-05-30}}</ref> 2014年5月，经马里兰州议会批准，正式承认该节日，至此，美国50个州中有43个州以及哥伦比亚特区已经认可了六月节作为一个州立节日或正式节假日，当天会休假庆祝。还没有认可的州包括亚利桑那州，夏威夷，新罕布夏州，北达科他州和犹他州。<ref>{{Template:Cite news|url = http://baltimoretimes-online.com/news/2014/may/30/juneteenth-officially-recognized-maryland/|first = Stacy M.|last = Brown|title = Juneteenth officially recognized in Maryland|work = Baltimore Times|date = 30 May 2014|access-date = 1 October 2014}}</ref>

1996年，由[[芭芭拉-罗斯·科斯林|芭芭拉-罗斯·科斯林]]（D-MI）发起的美国众议院的众议院第195号决议，介绍了认可“六月独立日”的第一部立法。1997年，国会通过参议院联合决议第11号文件和众议院联合第56号决议文件，确认通过了这个节日。2013年，美国参议院通过了参议院第175号决议承认了卢拉•布里格斯•洛维（前六月节国际协会主席）对“成功地使六月节得到国家认可”所做的贡献，并可继续担任国家六月节纪念基金会的领导工作。<ref name="sr175">{{Template:Cite web|url = https://www.congress.gov/bill/113th-congress/senate-resolution/175/text|title = S.Res.175 - A resolution observing Juneteenth Independence Day, June 19, 1865, the day on which slavery finally came to an end in the United States|website = United States Congress|date = June 19, 2013|accessdate = June 19, 2015}}</ref>
