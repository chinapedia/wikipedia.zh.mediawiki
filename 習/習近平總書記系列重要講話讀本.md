《'''習近平總書記系列重要講話讀本'''》，或稱「習語錄」，是一本記載[[中國共產黨中央委員會總書記|中共中央總書記]][[習近平|習近平]][[習近平總書記重要講話精神|講話內容]]的書籍。2014年版分為12個專題，2016年版分為16個專題，由中共官方的[[人民出版社|人民出版社]]和學習出版社联合出版。

== 歷史 ==
此書按国家[[通讯社|通讯社]][[新華社|新華社]]的解釋，內容爲「阐述了习近平系列重要讲话的意义、内涵、精神实质和实践要求」。<ref name="SHH">{{Cite web |url = http://news.xinhuanet.com/politics/2014-06/23/c_1111272000.htm |title=《习近平总书记系列重要讲话读本》|publisher=新華社}}</ref>

目前，[[中共中央宣傳部|中共中央宣傳部]]和[[中共中央組織部|中央組織部]]已下達通知，要求在[[中國共產黨|中國共產黨]]內部學習該讀本。同時，還要求各[[高校|高校]]將其作爲「師生理論學習教材」。<ref name="BBC">{{Cite web |url = http://www.bbc.co.uk/zhongwen/trad/china/2014/06/140624_china_xijinping_uni.shtm |title = 中宣部要求把習近平講話納入高校教材 |publisher=BBC中文網}}</ref>

2014年版32開定价13元人民幣，2016年版32開定價16元人民幣，书底另有「出版物版权追溯保护验证码」。

中華人民共和國官方媒體稱「該書十分[[畅销书|暢銷]]」<ref>{{cite news|title=亚马逊发布2016年中图书排行榜 《岛上书店》雄霸榜首-新华网|url=http://news.xinhuanet.com/book/2016-06/30/c_129104658.htm|accessdate=2017-07-26|agency=新华网|date=2016-06-30}}</ref>，但據報道不少大陸網民對此進行了調侃<ref>{{cite web |url = http://www.bbc.co.uk/zhongwen/trad/china/2014/08/140817_xi_jinping_speeches.shtml |title=中國官媒報道習近平著作暢銷遭網友調侃 |publisher=BBC中文網}}</ref>。

== 評價 ==
中國[[新華社|新華社]]稱：「《读本》为广大党员、干部、群众学习讲话精神提供了重要辅助材料。」<ref name="SHH" />

《[[自由時報|自由時報]]》報道，中宣部要求把本书納入高校教材，有網友對此發表評論稱「孩子的[[洗腦|洗腦]]不能等！」<ref>{{cite web |url = http://news.ltn.com.tw/news/world/breakingnews/1038980|title=習近平講話編讀本納教材 網友諷：孩子的洗腦不能等}}</ref>

[[英國廣播公司|英國廣播公司]]在一篇有關該書的報道中，在一個章節用了「『[[神化|造神]]』運動？」這一小標題以示評價。<ref name="BBC" />
