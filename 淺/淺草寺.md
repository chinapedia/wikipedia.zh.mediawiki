'''淺草寺'''（{{jpn|j=浅草寺|hg=せんそうじ|rm=Sensō ji}}），又名'''金龍山淺草寺'''，位於[[日本|日本]][[東京都|東京都]][[台東區|台東區]][[淺草|淺草]]二丁目，是東京都内历史最悠久的[[寺院|寺院]]。山號為金龍山。供奉的本尊是[[聖觀音|聖觀音]]。原屬[[天台宗|天台宗]]，於[[第二次世界大戰|第二次世界大戰]]後獨立，成為聖觀音宗的[[總本山|總本山]]。觀音菩薩本尊通稱為「淺草觀音」。

== 起源與历史 ==
[[File:Toto_Kinryuzan_Senso-ji_zu,_1820.jpg|thumb]]
[[File:Nakamise1411.jpg|thumb]]
[[File:Sensoji_at_night.JPG|thumb]]
根據寺傳的資料，[[推古天皇|推古天皇]]36年（628年），於宮戶川（現・[[隅田川|隅田川]]）捕魚的檜前浜成・竹成（{{lang|ja|ひのくまのはまなり・たけなり}}）兄弟於漁網發現了一尊佛像。就是淺草寺本尊的聖觀音像。兄弟的主人・土師中知（{{lang|ja|はじのなかとも}}）拜了此佛像之後出家，把房屋改成寺廟來供奉。這是淺草寺之始。觀音像，相傳為高一寸八分（約5.5cm）的金色像，由於是非公開的秘藏佛像，故實體不明。其後，[[大化_(孝德天皇)|大化]]元年（645年），據說勝海上人為寺院進行整備、經觀音報夢告知而把本尊定為秘藏佛像。至平安時代初期[[天安_(文德天皇)|天安]]元年（857年）（[[天長_(淳和天皇)|天長]]5年－828年），[[延暦寺|延暦寺]]僧人・[[円仁|円仁]]（慈覺大師）來寺製造了「{{lang|ja|お前立ち}}」（代替秘藏佛像供人參拜）觀音像。故稱淺草寺由勝海開基（創立者），円仁中興開山。雷門與仁王門於[[天慶_(朱雀天皇)|天慶]]5年（942年），安房守[[平公雅|平公雅]]任武藏守的時候創建。

淺草寺於文獻首次出現是鎌倉時代的『[[吾妻鏡|吾妻鏡]]』。近世為徳川家的祈願寺，關東有數的觀音靈場，参拜者眾。

[[江戶時代|江戶時代]]後半，境内「[[仲見世|仲見世]]」地區開始出現商店與芝居小屋，亦有賣藝人集中，成為當時庶民的娛樂中心。直至近代，淺草依然是平民的繁華街，娛樂場所發達，淺草寺作為此處地標。明治初期被指定為公園地、1885年（明治18年）於表参道兩側的「仲見世」漸漸變成近代的磚造建築物。1917年（大正6年）開始上演日本語的喜[[歌劇|歌劇]]「淺草OPERA（{{lang|ja|淺草オペラ}}）」，於電影普及之前，大眾演劇盛極一時。第二次世界大戰後的淺草，由於東京都内其他娛樂場所的多樣化發展，而經歷一時衰退。但是作為東京代表性的觀光地，加上羽子板市、燈籠花市等活動，觀光客與善信不絕。

== 寺内 ==
[[File:People_at_the_Asakusa_Sensoji_Kaminarimon_entrance.jpg|thumb]]
[[File:Sensoji_Dragon_Fountain.JPG|thumb]]
'''雷門'''表参道入口之門。[[切妻造|切妻造]]八脚門，左面的是風神像，右面的是雷神像，正式名稱為「風雷神門」，通稱為「雷門」。[[慶應|慶應]]元年（1865年）被燒毀、1960年（昭和35年）、約1世紀之後以鋼筋混凝土再重建。門内置有實業家[[松下幸之助|松下幸之助]]於淺草觀音祈願之後病癒，作為報答而寄贈的大燈籠。

[[File:Hozomon_012006.JPG|thumb]]
'''宝藏門'''鑽過雷門、通過仲見世商店街之後就是寶藏門。[[入母屋造|入母屋造]]的二重門（2層建築、外觀上為上下二重門）。現在之門於1964年（昭和39年）再重建時以鋼筋混凝土建造、由實業家[[大谷米太郎|大谷米太郎]]夫妻捐款興建。門的左右置有仁王（[[金剛力士|金剛力士]]）像、之前名叫「仁王門」、昭和再重建後才稱為宝藏門。其名來自門上層的文化財產収藏庫。

[[File:Senso-ji_main_hall_012006.JPG|200px]]
[[File:Asakusa_-_Senso-ji_35_(15576702197).jpg|thumb]]
[[File:Sensō-ji_-_Ceiling_-_August_2013_-_Sarah_Stierch_02.JPG|thumb]]
'''本堂'''（重要文化財產）本尊觀音像供奉之地，故又稱觀音堂。舊堂於[[慶安|慶安]]2年（1649年）重建，為近世大型寺院本堂的代表作與國宝（當時），1945年（昭和20年）於[[東京大空襲|東京大空襲]]被燒燬。現在之堂於1958年（昭和33年）再重建時以鋼筋混凝土建造。外陣有[[川端龍子|川端龍子]]（{{lang|ja|りゅうし}}）作畫的「{{lang|ja|龍の圖}}」、[[堂本印象|堂本印象]]作畫的「{{lang|ja|天人散華の圖}}」等天井畫。

内陣中央為本尊安置間，高6米的宮殿。宮殿内部分為前之間與奥之間，奥之間有秘藏本尊，前之間為「{{lang|ja|お前立ち}}」觀音像。於毎年12月13日會舉行的開扉法事，與有特別儀式需要開扉的時候，信徒可以到此參拜「{{lang|ja|お前立ち}}」像，但秘藏本尊像不作公開。宮殿的左右有脇侍[[梵天|梵天]]・[[帝釋天|帝釋天]]像，堂内後方左右有[[不動明王|不動明王]]像與[[愛染明王|愛染明王]]像。

[[File:Senso-ji_pagoda_phone_booths.jpg|thumb]]
'''五重塔'''重建前之塔於慶安元年（1648年）建成、與本堂同樣於[[關東大地震|關東大地震]]時倒塌、1945年時東京大空襲被燒毀。現在之塔於1973年（昭和48年）重建、鋼筋混凝土建造、鋁合金瓦頂、基壇高約5米、塔自體高約48米。基壇内部有收納供養靈位牌的靈牌殿、塔的最上層置有[[斯里蘭卡|斯里蘭卡]]傳來的[[舍利子|舍利子]]。現在塔之本堂位於西側、重建以前本位於東側。

'''二天門'''（重要文化財產）－位於本堂東側，向東興建，切妻造八脚門，木材建造。[[元和|元和]]4年（1618年）時建築，沒被第二次世界大戰破壞的貴重建築物。當時的將軍[[德川秀忠|德川秀忠]]為供奉父親[[德川家康|德川家康]]而於日光與淺草建東照宮，這門是作為淺草東照宮的隨身門而被興建（東照宮於1642年燒毀、沒有再重建）。現在，門的左右安置有二天（持國天、増長天）。

[[File:Asakusa_shrine_2012.JPG|thumb]]
'''淺草神社'''（重要文化財產）－位於本堂東側。此神社供奉的是淺草寺草創相關的三人。明治神佛分離之後與淺草寺屬不同法人管理。詳細參照[[淺草神社|淺草神社]]。

'''傳法院'''位於宝藏門前西側、淺草寺和尚生活的地方。有由[[小堀遠州|小堀遠州]]製作的回遊式庭園。一般不公開開放。

== 重要節慶與活動 ==
* 針供養（2月8日）
* 金龍之舞－3月18日的觀音示現會（{{lang|ja|じげんえ}}）、10月18日的菊供養之際、有舞金龍活動。
* 三社祭（5月17・18日頃）淺草神社的祭禮、著名的江戶三大祭之一。
* 四万六千日（{{lang|ja|しまんろくせんにち}}）（7月9日－10日）於此日參拜被認為能得到等同于参拜了4万6千日的功德。設'''灯笼花市'''。
* 万靈燈籠供養會（{{lang|ja|8月15日}}）祭先人的盂蘭盆會。
* 歳市（12月17日－19日）毎月18日為觀音菩薩的縁日、特別於年末的緣日設歳市。設'''羽子板市'''。

== 文化財產 ==
* 二天門（重要文化財產）
* 法華經（國宝）－平安時代11世紀頃的作品。以金銀泥作裝飾的「裝飾經」的代表作。寄存於[[東京國立博物館|東京國立博物館]]。
* 元版一切經（重要文化財產）－中國・元朝（明治初期、由鶴岡八幡宮傳來）
* 六角堂（[[東京都指定文化財產一覽#建造物|東京都指定建造物]]）
* 緣起（[[東京都指定文化財產一覽#繪畫・典籍|東京都指定繪畫・典籍]]）
其他還有、江戶時代的繪馬、當中包括有[[歌川國芳|歌川國芳]]等著名繪師的作品。

==參拜人數==

根據東京觀光財團的資料，把觀光者的人數也算在內的話，一年人次有3000-4000萬人。按照寺院、神社的參拜、觀光人數排行，這裡是日本第一。主參道上遊人終日絡繹不絕，元旦三天的參拜者達到百萬人以上。每月一八日的吉日或「茶湯月參講」的功德日之時，大殿內信者雲集，祈禱聲充耳不絕。
