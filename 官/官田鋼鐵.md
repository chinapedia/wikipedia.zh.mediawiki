'''官田鋼鐵股份有限公司'''，簡稱'''官田鋼鐵'''或'''官田鋼'''；創立於1973年7月28日，是生產[[鋼鐵|鋼鐵]]產品的公司，總部位於[[臺南市|臺南市]]。

==歷史==
早期製造銷售鍍鋅鐵線( 永康廠 ) 。

1989年設立麻豆廠，生產球化退火線材。

1996年興建年產量 40 萬噸盤元之熱軋鋼廠 ( 官田廠 ) ，生產各類線材盤元。
