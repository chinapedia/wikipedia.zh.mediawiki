'''角之濱站'''（{{jpn|j=角の浜駅|hg=かどのはまえき|rm=Kadonohama eki|f=y}}）是[[東日本旅客鐵道|東日本旅客鐵道]]（JR東日本）[[八戶線|八戶線]]沿線的[[鐵路車站|鐵路車站]]，位於[[岩手縣|岩手縣]][[九戶郡|九戶郡]][[洋野町|洋野町]]種市。

== 車站結構 ==
角之濱站是[[無人站|無人站]]，月台採單線[[側式月台|側式月台]]式設計。

== 附近環境 ==
角之濱站鄰近漁港，北側不遠處與[[青森縣|青森縣]][[九戶郡|九戶郡]][[階上町|階上町]]接壤，為岩手縣與青森縣東邊的邊界。
* 角濱簡易郵局
* [[縣道247號角之濱玉川線|縣道247號角之濱玉川線]]
* [[國道45號|國道45號]]（種市繞道）
* 洋野町立角濱小學
* 洋野町立角濱中學

== 历史 ==
* 1954年（昭和29年）8月5日－本站開業。
* 1987年（昭和62年）4月1日－[[國鐵分割民營化|國鐵分割民營化]]，本站由JR東日本接手管理。
