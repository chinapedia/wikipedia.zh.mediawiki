'''角野榮子'''（{{lang-ja|角野栄子}}，{{bd|1935年|1月1日||}}）是[[日本|日本]][[東京|東京]]出生的女性兒童文學、繪本、散文作家。代表性作品有繪本《大盜布拉布拉》、《褲子船長的故事》，以及小說《[[魔女宅急便_(小說)|魔女宅急便]]》等。

==生平==
自[[早稻田大學|早稻田大學]]畢業後，角野榮子曾在出版社裡就職，在25歲時移民到[[巴西|巴西]]居住兩年。之後角野以巴西的生活經歷；寫下個人文學處女作{{nihongo|《少年路易、來到巴西》|ルイジンニョ少年、ブラジルをたずねて}}，但作品延在1970年才正式發表。<ref>{{Cite web |url=http://kiki-jiji.com/hoge/kadono/index.html |script-title=ja:角野栄子のポケット |work=kiki-jiji.com |accessdate=2013年5月20日 }}</ref>

返回日本後，角野開始主攻創作兒童文學，<ref>{{Cite book |script-title=ja:魔女の宅急便 ロマンアルバム |publisher=[[德間書店|德間書店]] | isbn=978-4-19-720158-7 |pages=79頁 |date=2008年9月25日 |accessdate=2013年5月20日 }}</ref>以{{nihongo|《大盜布拉布拉》|大どろぼうブラブラ氏}}一書獲得1982年第29屆的[[產經兒童出版文化獎|產經兒童出版文化獎]]的大獎，<ref>{{Cite web |url=http://www.eventsankei.jp/child_award/old_prize.html#29  |script-title=ja:「産経児童出版文版文化賞」過去の受賞作品をご紹介致します 第29回＜昭和57年／1982年＞ |publisher=產經新聞 |accessdate=2013年5月20日 }}</ref>，並以1985年發表的小說《[[魔女宅急便_(小說)|魔女宅急便]]》得到當年的野間兒童文藝獎。<ref>{{Cite web |url=http://corp.kodansha.co.jp/award/archive/noma-jido-bungei.html |script-title=ja:野間児童文芸賞 (過去の受賞者一覧) |publisher=[[講談社|講談社]] |accessdate=2013年5月20日 |deadurl=yes |archiveurl=https://web.archive.org/web/20130809170738/http://corp.kodansha.co.jp/award/archive/noma-jido-bungei.html |archivedate=2013年8月9日 }}</ref>《魔女宅急便》在發表數年後曾讓[[吉卜力工作室|吉卜力工作室]]改編為[[魔女宅急便|同名動畫]]，於1989年在戲院上映。<ref>{{Cite book |script-title=ja:スタジオジブリの軌跡 1984-2011 |publisher=[[德間書店|德間書店]] |isbn=978-4-19-720326-0 |pages=53頁 |date=2011年4月1日 |accessdate=2013年5月20日 }}</ref>角野在2000年獲得日本政府肯定她的文藝成就，得授[[紫綬褒章|紫綬褒章]]的榮譽。<ref>{{Cite web |url=http://www.waseda.jp/student/weekly/contents/2010a/1223/223e.html  |script-title=ja:絵本・童話作家　角野 栄子さん |work=[[早稻田大學|早稻田大學]] |date=2010年7月8日 |accessdate=2013年5月20日 }}</ref>2018年，獲頒授該年度[[國際安徒生文學獎|國際安徒生文學獎作家獎]]。<ref>{{Cite web |url=http://asahichinese-f.com/cool_japan/style_culture/11446299  |title=《魔女宅急便》原作角野榮子 獲頒國際安徒生作家獎 |publisher=朝日新聞中文網 |date=2018年3月28日 |accessdate=2018年3月30日 }}</ref>

除從事文學創作外，角野榮子也有從事翻譯過日本國外的兒童繪本、或是把一些如由[[安徒生|安徒生]]等作家創作的童話給重新詮釋編寫的作品。
