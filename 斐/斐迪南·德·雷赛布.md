'''斐迪南·玛利·维孔特·德·雷赛布'''（'''Ferdinand Marie Vicomte de Lesseps''' {{bd|1805年|11月9日|1894年|12月7日}}）是[[法国|法国]][[外交官|外交官]]、[[实业家|实业家]]。著名的[[苏伊士运河|-{zh-hans:苏伊士运河;zh-hk:蘇彝士運河;zh-tw:蘇伊士運河;}-]]即由他主持开凿。

== 早年和外交官生涯 ==
雷赛布生于[[马赛|马赛]]，其母卡特琳娜是[[西班牙|西班牙]]人。他的青少年时期是在[[意大利|意大利]]度过的。1825年起年轻的雷赛布在[[里斯本|里斯本]]从事副[[领事|领事]]，从而进入外交部门。1828年雷赛布又专任[[突尼斯|突尼斯]]的副领事。1832年又成为驻[[亚历山大港|亚历山大港]]的副领事，由此雷赛布与[[埃及|埃及]]结缘。

1833年雷赛布来到[[开罗|开罗]]任职，期间他担任过[[穆罕默德·阿里_(埃及)|穆罕默德·阿里]]之子，后来的[[穆罕默德·阿里王朝君主列表|埃及总督]][[赛义德·巴夏|赛义德·巴夏]]的家庭教师。赛义德·巴夏深受雷赛布的影响，而雷赛布开凿苏伊士运河的想法也就在这一期间内萌发。

1837年雷赛布回[[法国|法国]]完婚，此后他又先后在[[鹿特丹|鹿特丹]]、[[马拉加|马拉加]]、[[巴塞罗那|巴塞罗那]]任职。1846年至1849年任驻[[马德里|马德里]][[大使|大使]]。1849年他曾参与斡旋[[教皇|教皇]][[庇护九世|庇护九世]]安全返回[[梵蒂冈|梵蒂冈]]的工作。不久他辞去了外交官职务，而第一段婚姻也宣告结束。

== 苏伊士运河的成功 ==
1854年应巴夏之邀，雷赛布重回埃及。巴夏同意由雷赛布来主持[[苏伊士运河|-{zh-hans:苏伊士运河;zh-hk:蘇彝士運河;zh-tw:蘇伊士運河;}-]]开凿的工程。但[[英国|英国]]出于自己利益的考虑，设法对工程进行阻挠。在[[欧仁妮皇后|欧仁妮皇后]]（雷賽布之表妹）及[[拿破仑三世|拿破仑三世]]的支持下，雷赛布于1858年底成立了国際苏伊士运河股份有限公司以获得资金，并终于在1869年完成工程。雷赛布由此名利双收。1884年他被选为[[法兰西学院_(机构)|法兰西学院]]院士。

== 饮恨巴拿马运河 ==
雷赛布自扬名-{zh-hans:苏伊士运河;zh-hk:蘇彝士運河;zh-tw:蘇伊士運河;}-之后，曾有考虑进行其它工程。1879年一次地理学会聚会上，包括他在内的135位学会成员选取开凿[[巴拿马运河|巴拿马运河]]为目标。雷赛布年过古稀仍被任命为巴拿马运河公司总经理，不幸这也注定了雷赛布无法保住晚节。

工程从一开始就困难重重：经营不善，资金周转不灵；[[巴拿马|巴拿马]]地区潮湿的气候带来的[[疟疾|疟疾]]、[[黄热病|黄热病]]肆虐造成大量工人死亡。最终1889年公司破产，為此法國付出20億法朗的代價，許多法國投資人血本無歸，引发的[[巴拿马运河丑闻|巴拿马运河丑闻]]使雷赛布面临5年的牢狱之灾。虽然1893年最高法院因「老邁」裁决他无罪，但雷赛布已深受打击，不久去世。

== 家庭 ==
雷赛布与第一任妻子共有5子。1869年他有了第二次婚姻，共有12名子女。
