'''杜襲'''（{{bd|？||？|catIdx=D}}），[[表字|字]]'''子绪'''，[[颍川郡|颍川郡]][[定陵县|定陵县]]（今[[河南省|河南省]][[叶县|叶县]]）人。[[東漢|東漢]]末及[[三國|三國]]時[[曹魏|曹魏]]官員。曾長期在[[關中|關中]]地區戍守。

== 生平 ==
===早期===
[[東漢|東漢]]末年與同郡人[[繁欽|繁欽]]和[[趙儼|趙儼]]一起到[[荊州|荊州]]避亂，獲荊州[[州牧|牧]][[劉表|劉表]]以賓禮接待。當時[[繁欽|繁欽]]的才能多次讓劉表驚異，甚受賞識，但杜襲稱自己來荊州是等待輔助明主，一建功勳的一天，認為劉表不是撥亂之主，不應委身輔助。於是再走到[[長沙郡|長沙郡]]。

===建安時期===
[[建安_(漢獻帝)|建安]]元年（196年），[[曹操|曹操]]迎[[漢獻帝|漢獻帝]]遷都[[許昌|許昌]]，杜襲於是回到潁川，獲曹操任命為[[西鄂|西鄂]][[縣長|縣長]]。因為西鄂當地多盜賊，官員都徵集人民以保衛城池，不能務農，於是令土地荒蕪而人民生活困苦。杜襲上任後留下丁男保衛城池，容讓老弱者務農，令吏民都十分高興。

建安六年（202年），劉表以步騎一萬人攻西鄂，杜襲則召集五十多人，約誓抵抗，令到一些原本想自保的人都感動出戰；其他吏民都因為感謝杜襲的恩德而聽從杜襲的指揮。最終在戰場斬殺了離軍數百人，但始終寡不敵眾，城池還是被攻破，杜襲更帶領傷兵和吏民突圍出城，收集餘眾到摩陂的軍營，期間吏民都沒有反叛。

後來[[司隸校尉|司隸校尉]][[鍾繇|鍾繇]][[表|表]]他任議郎參軍事。後又獲[[荀彧|荀彧]]推薦，被任為[[丞相|丞相]]軍祭酒。建安十八年（213年），曹操封魏公，建魏國，杜襲與[[王粲|王粲]]、[[和洽|和洽]]等同被任命為[[侍中|侍中]]。建安二十年（215年），杜襲改領丞相[[長史|長史]]，隨同曹操征伐盤據[[漢中|漢中]]的[[張魯|張魯]]。

次年（216年）曹操回[[鄴城|鄴城]]，任命杜襲為[[駙馬都尉|駙馬都尉]]，留督漢中軍事。期間成功勸導八萬多名漢中地區的人移居[[洛陽|洛陽]]、鄴城地區。建安二十四年（219年），主將[[夏侯淵|夏侯淵]]戰死，杜襲與[[郭淮|郭淮]]為安定軍心，於是決定以[[張郃|張郃]]替代夏侯淵的位置，成功安定軍心。曹操撤出漢中後任命杜襲為留府長史，駐守[[關中|關中]]。

===曹魏時期===
[[延康_(东汉)|延康]]元年（220年），[[曹丕|曹丕]]繼位魏王，賜杜襲爵'''關內侯'''。同年稱帝，又任命杜襲為督軍糧御史，封武平亭侯。後又先後任督軍糧執法和入朝任[[尚書|尚書]]。[[太和_(曹叡)|太和]]元年（227年），[[曹叡|曹叡]]繼位，進封'''平陽鄉侯'''。

次年（228年）[[蜀漢|蜀漢]]丞相[[諸葛亮|諸葛亮]]北伐，[[大将军_(东亚)|大將軍]][[曹真|曹真]]領兵抵抗，並遷杜襲為大將軍[[軍師|軍師]]。[[太和_(曹睿)|太和]]五年（231年），曹真逝世，由[[司馬懿|司馬懿]]接替，杜襲亦轉任其軍師；諸葛亮寫信給司馬懿，求杜襲[[孟建|孟建]]近況<ref>《三國志·[[溫恢|溫恢]]傳》注引《魏略》：亮后出祁山，答司马宣王书，使'''杜子绪'''宣意於公威也。</ref>。後來稱病回朝，改任[[太中大夫|太中大夫]]。及後逝世，追贈少府，諡'''定侯'''。

== 性格特徵 ==
*杜襲勸諫時都婉轉而不是犯顏直諫，如當時许攸（與[[袁紹|袁紹]]降將[[許攸|許攸]]不是同一人）坐擁[[部曲|部曲]]，不歸附曹操而且言辭傲慢。曹操知道後大怒，決意要討伐，群臣諫止但都不聽，杜襲於是不直接諫止，先問曹操許攸的能力是甚麼人，曹操說是凡人；杜襲於是說凡人是不會明白像曹操這種非凡的人的想法如何，如果攻打許攸，只會被認為是避強攻弱。而且許攸能力有限，亦不需曹操親征。曹操最後聽從，派人安撫許攸。許攸立刻降服。

== 家庭 ==
'''曾祖父'''
*杜安，年少被稱為[[神童|神童]]，東漢官至[[巴郡|巴郡]][[太守|太守]]，當時著名。

'''祖父'''
*[[杜根|杜根]]，東漢官至濟陰太守，當時著名。

'''兄弟'''
*[[杜基|杜基]]，太和年間分杜襲[[食邑|食邑]]而封關內侯。

'''兒'''
*[[杜會|杜會]]，嗣子。

'''孫'''
*[[杜毓|杜毓]]<ref>《世說新語注·品藻·第8》：晉諸公贊曰：“杜育字方叔，襄城鄧陵人，'''杜襲'''孫也。”</ref>

== 评价 ==
*[[曹操|曹操]]：“释骐冀而不乘。焉皇皇而更索？”（《三国志·魏书·和常杨杜赵裴传第二十三》）
*[[陈寿|陈寿]]：“和洽清和干理，[[常林|常林]]素业纯固，杨俊人伦行义，'''杜袭温粹识统'''，赵俨刚毅有度，裴潜平恒贞干，皆一世之美士也。”“初，（趙）俨与同郡[[辛毗|辛毗]]、[[陈群_(三國)|陳群]]、杜袭并知名，号曰辛、陈、杜、赵云。”（《三国志·魏书·和常杨杜赵裴传第二十三》）
*[[刘咸炘|刘咸炘]]：“此诸人皆扬历中外，而和、杜、赵、裴又皆自荆州归曹氏。”（《三国志集解》）
