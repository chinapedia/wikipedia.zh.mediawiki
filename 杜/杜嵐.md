'''杜嵐'''{{OME}}（{{lang-en|'''Du Nam'''}}，{{bd|1914年||2013年|3月29日|catIdx=Du杜岚}}），原名'''杜芳銘'''，曾用名'''杜曉霞'''，[[陝西省|陝西省]][[米脂縣|米脂縣]]紅崖村人，1914年出生於一個農民家庭。曾任澳門[[濠江中學|濠江中學]]校長，但因年紀老邁，后转任澳門濠江中學名譽校長。1936年，年僅22歲的杜嵐隻身到濠江中學任教，1947年起肩負起濠江中學校長的重任，直至2000年轉任名譽校長，2013年3月29日病逝。

少女時代已活躍於反帝反[[封建|封建]]行列的杜嵐，積極參加[[抗戰|抗戰]]團體，在戰爭的烽火中鍛煉成長，被稱為「陝北最勇敢的女孩」<ref>{{cite news |url = http://paper.wenweipo.com/2006/08/28/zt0608280030.htm |title = 杜嵐校長，傳奇人生 |location = 香港 |publisher = 文匯報}}</ref>。

== 生平 ==
杜嵐從小受祖母性格的影響，衝破封建束縛，撕掉裹腳布，進入鄰村桃鎮小學，再轉入縣城女子學校繼續學業；1928年以優異成績跳級考入米脂縣三民二中（今米脂中學），受進步思想影響，參加反帝反封建宣傳。在愛國運動中，仍為活躍的骨幹分子，中學畢業後，易名“晚霞”，先後進[[北京平民大學|北京平民大學]]附屬中學、[[中國大學|中國大學]]教育哲學系就讀，同時積極參加「反帝大同盟」和「互濟會」的活動。30年代她和其他八十多人被押送南京憲兵司令部，取保釋放後，到了[[上海|上海]]，參加「全國各界抗日救國聯合會」的活動。1936年後到[[香港|香港]]，進入新聞學院；往後到澳門立足，於1936年與[[黃健|黃健]]結為伉儷，次年在澳門濠江中學執教，開始教育生涯。

== 澳門第一面五星國旗 ==
杜嵐女士其中一項最為人熟悉的事蹟是於1949年10月1日[[中华人民共和国|新中國]]成立當中，杜嵐帶領濠江中學一眾師生在學校升起澳門第一面[[五星紅旗|五星紅旗]]，從此學校的教育事業隨著祖國的強大而不斷發展，而學校亦全面實行愛國教育。在杜嵐擔任校長的50餘年間，濠江中學堅持[[愛國愛澳|愛國愛澳]]的辦學宗旨，有教無類，為國家、為澳門培養了無數優秀人才。

== 榮譽 ==
杜嵐的名字在澳門家喻戶曉。她曾任中華教育會理事、理事長，中華教育會名譽顧問，中山同鄉會監事，廣東省政協委員、人民代表，並被澳督選聘為教育委員會委員。而前國家主席[[江澤民|江澤民]]題詞「濠江中學，桃李芬芳」，是給予濠江中學的最大鼓勵，也是對杜嵐校長為教育事業所作傑出貢獻的肯定。而濠江中學在杜嵐的帶領下，成為澳門學生人數最多、教職人員數最多，規模最大的學校。

=== 獎勵勳章 ===
*1985年獲澳督頒發「教育勞績勛章」、「勞績勳章」
*1995年澳門總督頒授予濠江中學「文化功績勳章」
*[[澳門回歸|澳門回歸]]後，又獲得[[澳門特區政府|澳門特區政府]]嘉獎，頒授「特區政府勞績勛章」
*1993年杜嵐校長以卓著的治學成就榮載世界[[名人錄|名人錄]]，被英國及美國婦女名人協會吸收為會員，獲國際名人證書兩份
