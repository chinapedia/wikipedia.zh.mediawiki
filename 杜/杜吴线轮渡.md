'''杜吴线轮渡'''，为[[中华人民共和国|中国]][[上海市|上海市]][[黄浦江|黄浦江]]上的一条对江轮渡航线，航线西起[[闵行区|闵行区]][[贡山路|贡山路]]吴泾轮渡站，东至[[闵行区|闵行区]][[沈杜公路|沈杜公路]]杜行轮渡站。<ref>[http://www.shanghaiferry.com/1-%E7%BD%91%E9%A1%B5ppt%E8%BD%AC%E5%9B%BE%E7%89%87%E8%B5%84%E6%96%99/%E8%88%AA%E7%BA%BF%E8%88%AA%E7%8F%AD%EF%BC%88%E5%90%84%E6%B8%A1%E5%8F%A3%E7%AE%80%E4%BB%8B%EF%BC%89/1-%E6%9D%9C%E5%90%B4%E8%88%AA%E7%BA%BF%E7%AE%80%E4%BB%8B.png 上海轮渡杜吴线航线简介]</ref>
<br>杜吴线轮渡始于民国17年。1958年9月30日，由上海市轮渡公司在该地新建对江轮渡，航线定名杜吴线轮渡。1959年11月，航线划归上海县交通运输局运输公司客运站经营。1964年1月，航线交还由上海市轮渡公司经营。1973年，市轮渡对该航线轮渡设施进行改建。1980年，航线开辟汽车过江业务，实行车、客混装运营。<ref name="渡运">[http://www.shtong.gov.cn/newsite/node2/node2245/node4501/node56018/node56020/userobject1ai43385.html 上海市地方志 专业志 >> 上海内河航运志 >> 第六章渡运 >> 节] {{webarchive|url=https://web.archive.org/web/20140101093158/http://www.shtong.gov.cn/newsite/node2/node2245/node4501/node56018/node56020/userobject1ai43385.html |date=2014-01-01 }}</ref><ref name="交通">[http://www.shtong.gov.cn/newsite/node2/node4/node2249/node2251/node2312/node2314/node2316/userobject1ai123.html 上海市地方志 区县志 >> 区志 >> 闵行区志 >> 第五篇交通·邮电 >> 第一章交通 >> 节]</ref><ref name="沿革">[http://www.shanghaiferry.com/1-%E7%BD%91%E9%A1%B5ppt%E8%BD%AC%E5%9B%BE%E7%89%87%E8%B5%84%E6%96%99/%E7%99%BE%E5%B9%B4%E4%BC%A0%E6%89%BF/%E8%BD%AE%E6%B8%A1%E7%AE%80%E5%8F%B2/10-%E5%90%84%E6%B8%A1%E5%8F%A3%E5%8E%86%E5%8F%B2%E6%B2%BF%E9%9D%A9.html 上海轮渡各渡口历史沿革] {{webarchive|url=https://web.archive.org/web/20140202141841/http://www.shanghaiferry.com/1-%E7%BD%91%E9%A1%B5ppt%E8%BD%AC%E5%9B%BE%E7%89%87%E8%B5%84%E6%96%99/%E7%99%BE%E5%B9%B4%E4%BC%A0%E6%89%BF/%E8%BD%AE%E6%B8%A1%E7%AE%80%E5%8F%B2/10-%E5%90%84%E6%B8%A1%E5%8F%A3%E5%8E%86%E5%8F%B2%E6%B2%BF%E9%9D%A9.html |date=2014-02-02 }}</ref>
<br>目前，航线使用客渡渡轮运营。在[[闵浦大桥|闵浦大桥]]建成前，该航线设有装载小型汽车过江业务。但在大桥建成后，汽车均改走大桥过江。现在航线客流以行人、非机动车、摩托车为主。
<br>杜吴线轮渡航线从[[闵浦大桥|闵浦大桥]]下斜穿而过，为上海仅有的两条从黄浦江大桥下穿行而过的轮渡航线之一，另一条为[[南陆线轮渡|南陆线轮渡]]。

== 历史 ==
[[File:沪航客79.JPG|thumb]]（2009年）]]
杜吴线轮渡开辟于民国17年，初为手摇木船摆渡。上海战役后，1958年9月30日，由上海市轮渡公司在该地新建对江轮渡，投入一艘40客位的市轮渡205号渡轮运营，以替代原手摇木船摆渡，航线定名杜吴线轮渡。1959年11月，上海县人民政府与上海市轮渡公司进行协商，将航线划归上海县交通运输局运输公司客运站经营。后因客运站对航线经营不善，渡轮保养不力，时常抛锚，渡轮常不能正常运营，加之过渡客流过少，航线严重亏损。1964年1月，由上海县人民政府批准，将航线及轮渡设备交还上海市轮渡公司经营。1973年，上海市轮渡公司投资13万元人民币，对该航线轮渡站、码头设施进行改建。1980年，上海市内河航运局第二管理区将车辆渡3号渡轮调入杜吴线轮渡运营，开始装载汽车过江业务，实行车、客混装营运。为应付逐年上升的客流，1986年4月1日起，上海市轮渡公司将500客位的沪航客14号渡轮调入杜吴线轮渡。<ref name="渡运"/><ref name="交通"/><ref name="沿革"/>
<br>杜吴线轮渡旁即为闵浦大桥，在大桥建成前，该航线使用的1000客位客渡渡轮同时承担装载小型汽车过江业务，渡轮额定载车6辆，满载汽车时，渡轮装载客位为500客位。随着2010年4月20日[[闵浦大桥|闵浦大桥]]下层的普通公路层通车，汽车均改走大桥过江。航线目前以渡运行人、非机动车、摩托车为主。
