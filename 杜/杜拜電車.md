'''杜拜電車'''（{{lang-ar|ترام دبي}}，前稱蘇否電車）是一座位於[[阿拉伯联合酋长国|阿拉伯联合酋长国]]（阿聯酋）[[杜拜|杜拜]]蘇否的[[有軌電車|有軌電車]]系統，全長{{convert|14.5|km|mi|sp=us}}；其中杜拜電車第一期路線全長{{convert|10.6|km|mi|sp=us}}，設站11座，於2014年11月12日開始載客。電車線沿[[D94路|蘇否路]]興建，起於[[杜拜碼頭|杜拜碼頭]]，終於[[阿拉伯塔|阿拉伯塔]]和[[阿聯酋購物中心|阿聯酋購物中心]]。電車線在朱美拉湖塔樓站和達馬克地產站與[[杜拜地鐵|杜拜地鐵]][[杜拜地鐵紅線|紅線]]交匯，並於[[朱美拉棕櫚島|朱美拉棕櫚島]]位於蘇否路的入口和[[朱美拉棕櫚島單軌鐵路|朱美拉棕櫚島單軌鐵路]]交匯<ref name="gulfnews.com"/>。杜拜電車也是歐洲以外第一座以[[軌道供電#APS系統|地面供電]]的方式驅動列車的地面電車系統<ref name=khaleej>{{cite web |author=Staff Report |url=http://www.khaleejtimes.com/kt-article-display-1.asp?xfile=data/transport/2014/April/transport_April19.xml&section=transport |title=10 facts you should know about the Dubai tram |publisher=Khaleej Times |date= |accessdate=2016-01-18 |archive-url=https://web.archive.org/web/20150703035752/http://www.khaleejtimes.com/kt-article-display-1.asp?xfile=data%2Ftransport%2F2014%2FApril%2Ftransport_April19.xml&section=transport |archive-date=2015-07-03 |dead-url=no }}</ref>。

== 路線及車站 ==
杜拜電車第一期路線全長10.6公里，設有11座車站，是全球首條完全以[[軌道供電#APS系統|地面供電]]的方式驅動列車的電車系統<ref name=irj2/>，也是歐洲以外首座採用這種方式驅動列車的電車系統<ref name=khaleej/>。這條電車線的起點是位於[[杜拜地鐵|杜拜地鐵]][[杜拜地鐵綠線|綠線]][[沙拉夫DG站|沙拉夫DG站]]附近的車廠，電車離開車廠後會駛入沿[[D94路|蘇否路]]興建的地面線，途經知識村、媒體城等各站，之後電車還會駛入在[[杜拜碼頭|杜拜碼頭]]區興建的高架路段。杜拜電車設有三個換乘站：乘客可以在朱美拉湖塔樓站和杜拜碼頭站轉乘杜拜地鐵[[杜拜地鐵紅線|紅線]]，並於朱美拉棕櫚島站轉乘[[朱美拉棕櫚島單軌鐵路|朱美拉棕櫚島單軌鐵路]]<ref name=irj2>{{cite news|url=http://www.railwaygazette.com/news/news/middle-east/single-view/view/dubai-tramway-opens.html|title=Dubai tramway opens|date=2014-11-12|work=Railway Gazette|accessdate=2016-10-02|location=UAE|archive-url=https://web.archive.org/web/20160728172028/http://www.railwaygazette.com/news/news/middle-east/single-view/view/dubai-tramway-opens.html|archive-date=2016-07-28|dead-url=no}}</ref>。

== 列車 ==
[[File:Dubai_Alstom_Citadis_402.jpg|thumb]]
道路交通管理局向法國鐵路車輛製造商[[阿爾斯通|阿爾斯通]]採購了11列[[阿爾斯通Citadis|阿爾斯通Citadis]]402型電車，並於2013年6月14日正式接收這一批電車<ref name=rgi>{{cite news|url=http://www.railwaygazette.com/news/single-view/view/alstom-reveals-dubais-first-al-sufouh-tram.html|title=Alstom reveals Dubai's first Al Sufouh tram|date=2013-06-17|work=Railway Gazette|accessdate=2016-09-20|location=UAE|archive-url=https://web.archive.org/web/20150924141157/http://www.railwaygazette.com/news/single-view/view/alstom-reveals-dubais-first-al-sufouh-tram.html|archive-date=2015-09-24|dead-url=no}}</ref>。通車初期，杜拜電車系統共有11列列車，一列電車全長44公尺，共設有7節車廂（包括頭等黃金級車廂、普通等銀級車廂，以及婦孺專用車廂），可以搭載408名乘客。<ref name=rgi/><ref name=wam>{{cite news|url=http://www.emirates247.com/news/government/mohammed-attends-dubai-tram-test-drive-2014-08-12-1.559190|title=Mohammed attends Dubai Tram test drive|author=Wam|date=2014-08-12|work=Emirate 24/7 News|publisher=Dubai Media news|url=http://gulfnews.com/guides/life/transport/guide-to-dubai-tram-network-1.1373821|title=Guide to Dubai tram network|last=Fullard|first=Martin|date=2014-08-19|work=Gulf News|publisher=Al Nisr news|url=http://gulfnews.com/news/uae/transport/how-dubai-tram-ensures-top-safety-standards-1.1512169|title=How Dubai Tram ensures top safety standards|last=Shafaat LLC|accessdate=2016-09-20|location=Dubai|archive-url=https://web.archive.org/web/20161004130253/http://gulfnews.com/news/uae/transport/how-dubai-tram-ensures-top-safety-standards-1.1512169|archive-date=2016-10-04|dead-url=no}}</ref>杜拜電車採用APS二型地面供電系統，電車會從軌道上收集電力。電車能夠適應杜拜高溫、潮濕、多沙和容易引致腐蝕的環境，而且還設有顯示實時乘客資訊的裝置。<ref name=rgi/>道路交通管理局計劃在電車第二期路線通車時採購另外14列電車，令電車車隊的規模增加到25列電車<ref name=wam/>。

== 營運資料 ==
信佳集團在2013年9月和道路交通管理局簽訂為期75個月的杜拜電車服務經營合約，這家公司也是杜拜地鐵的運營商<ref>{{cite news|url=http://www.emirates247.com/news/emirates/dubai-tram-to-connect-jbr-jlt-marina-and-al-sofouh-palm-jumeirah-by-next-year-2013-09-15-1.521105|title=Dubai Tram to connect JBR, JLT-Marina and Al Sofouh-Palm Jumeirah by next year|date=2013-09-15|work=Emirates 24/7 News|publisher=Dubai Media Incorporated|accessdate=2016-10-14|archive-url=https://web.archive.org/web/20161019051147/http://www.emirates247.com/news/emirates/dubai-tram-to-connect-jbr-jlt-marina-and-al-sofouh-palm-jumeirah-by-next-year-2013-09-15-1.521105|archive-date=2016-10-19|dead-url=no}}</ref>。

杜拜電車每天的首班車在當地時間6時30分開出（星期五的首班車在9時開出），末班車在翌日1時00分開出。杜拜電車每天的營運時數是19小時，列車走完全程需要48分鐘。營運方會在特別的場合延長電車的服務時間。杜拜電車每班車的間隔時間為8分鐘<ref name=rta>{{cite web|url=http://www.rta.ae/wpsv5/links/handy-guide/tram-handy-guide-en.pdf|title=The Handy Guide for Using the Dubai Tram|date=2016-01|publisher=Roads and Transport Authority|accessdate=2016-09-29|location=DubaI}}</ref>。

乘客可以使用[[Nol卡|Nol卡]]支付杜拜電車的車費。Nol卡分為為公共交通工具常客而設的銀卡和藍卡、為頭等車廂乘客而設的金卡和個人金卡，以及為旅客而設的紅卡；當中藍卡和個人金卡的加密功能比銀卡和金卡更強。杜拜電車第一期路線全線都屬於同一個收費區<ref name=rta/>。
