《'''笑傲六十年：有话说李敖'''》是[[凤凰卫视中文台|凤凰卫视中文台]]播出关于著名文化人[[李敖|李敖]]的[[纪录片|纪录片]]，讲述了李敖从[[出生|出生]]到定居[[台湾|台湾]]、從事政治批判直到出任[[立法委員|立法委員]]的生平故事。<ref name="fish" >{{cite web |url=http://phtv.ifeng.com/program/fhdsy/200908/0803_1720_1282526.shtml |title=〔预告〕有话说李敖：惨绿少年 初抵台湾 |accessdate=2009 |publisher=凤凰网 |work= |language=zh}}</ref><ref name="bee" >{{cite web |url= http://video.sina.com.cn/v/b/23607467-1097479661.html |title=笑傲六十年：有话说李敖（二） |accessdate= |publisher=新浪网视频 |work= |language=zh}}</ref>

== 剧情 ==
全剧分5集，分别是：
*第一集：惨绿少年，初抵台湾
*第二集：狂妄青年，崭露头角
*第三集：[[文星|文星]]之火，火首燎原
*第四集：隐居谈情，东山再起
*第五集：以笔载史，叱吒风云

== 制作 ==
*采访：蔡婉颖、苏静芬、许真佩、陈冠华
*校稿：蔡婉颖、杜晓惠
*片头：刘伟长
