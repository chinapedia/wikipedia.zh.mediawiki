《'''笑傲江湖Online'''》（Swordsman）是一款2013年發行的[[大型多人線上角色扮演遊戲|多人線上角色扮演遊戲]]，改編自小說《[[笑傲江湖|笑傲江湖]]》，由[[完美世界|完美世界]]研發，在臺灣由[[艾玩天地|艾玩天地]]發行，由[[陳喬恩|陳喬恩]]、[[蕭敬騰|蕭敬騰]]代言，本遊戲的主題曲是蕭敬騰改編並主唱的《[[滄海一聲笑|滄海一聲笑]]》<ref>http://www.youtube.com/watch?v=nFDOYfuASNo</ref>，於11月20日開始封測<ref>http://sssonline.iwplay.com.tw/NewsList/newsevent/20131112/5518.html</ref>。

== 操作 ==
遊戲中有三種操作模式：「笑傲經典」、「傳統3D」、「動作競技」在游戏中可以自由切换这三种模式。<ref>http://sssonline.iwplay.com.tw/gameinfo/gameinfo4.html</ref>

== 內容 ==
遊戲中可從十大門派之中擇一加入，分別是：「[[華山|華山]]」、「[[唐門|唐門]]」、「[[衡山|衡山]]」、「[[五仙|五仙]]」、「[[日月|日月]]」、「[[少林|少林]]」、「[[峨嵋|峨嵋]]」、「[[武當|武當]]」、「[[恆山|恆山]]」、「[[逍遙|逍遙]]」。选了其中的一个门派之后不可再变<ref>http://sssonline.iwplay.com.tw/sects/index.html?star=be00</ref>

== 競賽 ==
於2015年曾舉辦笑傲江湖Online 蓋世盃世界冠軍聯賽。

== 微電影 ==
在2013年推出“笑傲江湖 東方不敗外傳”微電影，[[東方不敗|東方不敗]]由[[陳喬恩|陳喬恩]]飾演。此微電影共有三種不同的結局，分別為「女人心」<ref>https://www.youtube.com/watch?v=2w8lVScu6ds</ref>、「人間悲劇」<ref>https://www.youtube.com/watch?v=KxRtGApODug</ref>、「無情的男人」<ref>https://www.youtube.com/watch?v=L2aa3pz7gBA</ref>。一結局共15分鐘。
