'''房玄龄'''（{{bd|579年||648年|8月18日|catIdx=F房玄龄}}），名'''乔'''，字'''玄龄'''，[[以字行|以字行]]，其神道碑则作名玄龄，[[齐州_(北魏)|齐州]][[临淄|临淄]]（今[[山东省|山东省]][[淄博市|淄博市]]临淄区北）人，[[房彦謙|房彦謙]]之子。[[唐朝|唐朝]]初年名相、[[凌煙閣二十四功臣|凌煙閣二十四功臣]]之一。

== 生平 ==
[[File:Fang_Xuanling.jpg|thumb]]
房玄龄11岁时本州举[[进士|进士]]，授[[羽骑尉|羽骑尉]]。隋末天下大乱，房玄龄在渭北投秦王[[李世民|李世民]]后，为秦王参谋划策，典管书记，是秦王得力的谋士之一。唐朝[[武德|武德]]九年（626年），他参与[[玄武门之变|玄武门之变]]，与[[杜如晦|杜如晦]]、[[长孙无忌|长孙无忌]]、[[尉迟敬德|尉迟敬德]]、[[侯君集|侯君集]]五人并功第一。唐太宗李世民即位后，房玄龄为[[中书令|中书令]]；[[貞觀_(唐朝)|贞观]]三年（629年）二月为[[尚书左仆射|尚书左仆射]]；贞观十一年（637年）封'''梁国公'''；贞观十六年（642年）七月进位[[司空|司空]]，仍综理朝政。贞观二十二年[[七月廿四|七月廿四]][[癸卯|癸卯]]日（648年8月18日），房玄龄病逝，谥'''文昭'''，追贈[[太尉|太尉]]。

唐高宗永徽三年，玄齡次子[[房遺愛|房遺愛]]與其妻[[高陽公主|高陽公主]]被指謀反，遺愛被處死，公主被[[賜自盡|賜自盡]]，諸子被發配流放到嶺表。<ref>《舊唐書》列傳第十六</ref>玄齡嗣子遺直也被連累，被貶為銅陵尉。<ref>《新唐書》列傳第二十一</ref>房玄龄配享太庙的待遇也因而被停止。

因房玄龄善谋，而杜如晦处事果断，因此人称“[[房谋杜断|房谋杜断]]”。后世以他和杜如晦为良相的典范，合称“房、杜”。《新唐书》本传对房的评价是“玄龄当国，夙夜勤强，任公竭节，不欲一物失所。无媢忌，闻人善，若己有之。明达吏治，而缘饰以文雅，议法处令，务为宽平。不以己长望人，取人不求备，虽卑贱皆得尽所能。或以事被让，必稽颡请罪，畏惕，视若无所容”。

== 評價 ==
唐朝史官[[柳芳|柳芳]]評：「玄齡佐太宗定天下，及終相位，凡三十二年，天下號為賢相；然無跡可尋，德亦至矣。故太宗定禍亂而房、[[杜如晦|杜]]不言功，[[王珪_(唐朝)|王]]、[[魏徵|魏]]善諫諍而房、杜讓其賢，英、衛善將兵而房、杜行其道，理致太平，善歸人主。為唐宗臣，宜哉！」<ref>{{Cite web|url=https://zh.wikisource.org/wiki/%E8%B3%87%E6%B2%BB%E9%80%9A%E9%91%91/%E5%8D%B7199|title=資治通鑑/卷199 - 维基文库，自由的图书馆|accessdate=2018-02-05|work=zh.wikisource.org|language=zh}}</ref>

== 故乡 ==

唐人墓志所称某地人，多指[[郡望|郡望]]所在。故房玄龄墓志依[[清河房氏|清河房氏]]郡望，称其“清河郡县人也”。关于房玄龄[[故乡|故乡]]、[[籍贯|籍贯]]所在，多有争议。{{or|有今济南、淄博两说。清河房氏，认为房玄龄是今天[[临淄|临淄]]人，根据是《[[唐书|唐书]]》中房玄龄是“齐州临淄人”的记载。如，第六版《辞海》“房”字“房玄龄”条就认为是其为今淄博市临淄区北人。（《辞海》583页） 
认为房玄龄是济南人，根据是《[[新唐书|新唐书]]·宰相世系表》记载，房玄龄的祖先“随慕容德南迁，因居济南。” 理由是隋唐的“齐州”并非[[漢代|漢]][[曹魏|魏]]时的[[齊郡|齊郡]]，而是[[济南郡|济南郡]]所在地。隋朝末期，济南郡一帶曾一度改称[[临淄郡|临淄郡]]，所以《唐书》说他是“齐州临淄人”。}}

==著作==
房玄龄監修了[[二十四史|二十四史]]之一的《[[晋书|晋书]]》。

==軼事==
相傳唐太宗曾賜了宰相房玄齡兩名美女，房玄齡夫人卻拒絕讓這兩名美女進門，唐太宗得知後宣房夫人進宮，怒道：「你敢違朕的旨意，這裡有兩條路任你選擇，一條是領回二位美女，和和氣氣過日子，另一條是吃了這罈『毒酒』省得妒嫉旁人了。」誰知房夫人性烈，拿起「毒酒」一飲盡，飲完才知竟是醋。唐太宗歎了口氣道：「房夫人，莫怨朕用這法子逼你，你妒心也太大了。不過念你寧死也戀著丈夫，朕收回成命。」從此，「吃醋」這個詞便成了女人間妒忌的代名詞。

[[隋唐嘉話|隋唐嘉話]]《中》： 梁公夫人至妬，太宗將賜公美人，屢辭不受。帝乃令皇后召夫人，告以媵妾之流，今有常制，且司空年暮，帝欲有所優詔之意。夫人執心不迴。帝乃令謂之曰：「若寧不妬而生，寧妬而死？」曰：「妾寧妬而死。」乃遣酌卮酒與之，曰：「若然，可飲此酖。」一舉便盡，無所留難。帝曰：「我尚畏見，何況於玄齡！」

==房玄龄碑==
全称《大唐故尚书左仆射司空太子太傅上柱国赠太尉并州都督梁文昭公之碑文》，现位于陕西[[醴泉县|醴泉县]][[唐昭陵|昭陵]][[博物馆|博物馆]]。宋[[赵明诚|赵明诚]]《金石录》云：《房玄龄碑》“其后题‘修国史河南公’而名姓残缺者，[[褚遂良|褚遂良]]也”。[[褚遂良|褚遂良]]在[[永徽|永徽]]三年（652年）征拜吏部尚书、同中书门下三品，监修国史，则可推知房玄龄碑立于永徽三年之内。
