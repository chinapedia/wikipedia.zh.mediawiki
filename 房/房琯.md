'''房琯'''（{{bd|696年||763年|catIdx=F房}}），字'''次律'''，[[河南|河南]]緱氏（今河南[[偃師|偃師]]緱氏鎮）人，[[唐代|唐代]][[开元|开元]][[天宝|天宝]]年间官员。

==生平==
父[[房融|房融]]於神龍元年，貶死高州。房琯少好學，以廕補[[弘文生|弘文生]]。博學多聞，與[[呂向|呂向]]隱居陸渾山讀書。開元十二年（724年）獻《封禪書》，官至[[給事中|給事中]]。天寶五年（746年）謫守宜春，“首立學宮”<ref>[[嚴嵩|嚴嵩]]：《正德袁州府志》</ref>，“所在為政，多興利除害，繕修廟宇，頗著能名”。[[安史之乱|安史之乱]]前夕拜[[左庶子|左庶子]]，迁[[刑部侍郎|刑部侍郎]]。隨玄宗奔蜀，往肅宗處送交傳國玉璽。肅宗任命他為宰相，房琯打算用車戰法收復長安，但泥於古法，以二千輛牛車打頭陣，在步騎兵左右掩護下出擊，監軍邢廷恩又急於促戰，<ref>[[潘德舆|潘德舆]]《养一斋诗话》卷三：“琯之衄师非无罪，而杜之救琯为徇私矣。然琯之败也，由於恢复两京之遽，固志大而虑疏；而琯之初意，亦欲持重俟时，中人邢廷恩等促战，仓皇遂及於败，则亦不当专为琯罪也。”</ref>在陳濤斜（又稱咸陽斜，今[[陝西|陝西]][[咸陽|咸陽]]東。杜甫詩作陳陶斜。因其路斜出。故曰斜。<ref> {{ Cite web |url = http://shuiren.org/kodaichimei/placename/24014.html |title = 中國古今地名大辭典 |quote = 在陝西咸陽縣東。亦謂之咸陽斜。唐房琯與安祿山將戰於陳濤斜。琯效古法。用車戰。大敗。杜甫詩作陳陶斜。〔通鑑注〕其路斜出。故曰斜。}} </ref>）被安祿山軍打敗，四萬唐軍全軍覆沒，<ref>[[杜甫|杜甫]]《悲陈陶》-{云}-：“盂冬十郡良家子，血作陈陶泽中水。野旷天清无战声，四万义军同日死。”</ref>史稱[[陳濤斜之戰|陳濤斜之戰]]。唐军收复长安后，房琯随肃宗回京，被授[[金紫光禄大夫|金紫光禄大夫]]，封'''清河郡公'''。晚年喜聽門客[[董廷蘭|董廷蘭]]彈琴，董廷蘭依仗房琯的权势，招纳贿赂，被御史彈劾，[[至德|至德]]二年五月貶房琯為[[太子少師|太子少師]]。好友左拾遗[[杜甫|杜甫]]曾為他打抱不平，上書說“罪细，不宜免大臣。”<ref>《[[新唐书|新唐书]]》卷二百一杜甫本傳：“琯时败陈涛斜，又以客董庭兰罢宰相，甫上疏言‘罪细不宜免大臣’。'帝怒，诏三司杂问。宰相张镐曰：‘甫若抵罪，绝言者路。’帝乃解。”两《唐书》和《藏书》称：房琯与杜甫为布衣之交。杜甫由长安逃至行在凤翔，拜[[右拾遗|右拾遗]]，是由房琯所荐。清人[[熊寶泰|熊寶泰]]《杜甫》诗-{云}-：“千秋诗史有谁知？房杜交深患难余。”</ref>[[乾元|乾元]]元年六月被贬为[[邠州|邠州]]（今[[陝西|陝西]][[彬縣|彬縣]]）刺史，颇著政声。<ref>《[[全唐文|全唐文]]》卷七六一[[郑处诲|郑处诲]]《邠州节度使厅记》：“邠实为近郡，申王薛王以亲贤之责居之，太尉房公以盛德之重居之。”</ref>乾元二年六月拜[[太子賓客|太子賓客]]、[[上元|上元]]元年（760年）四月，改[[禮部尚書|禮部尚書]]。不久，出京為[[晉州|晉州]]（今[[山西|山西]][[臨汾市|臨汾市]]）刺史，八月，改為[[漢州|漢州]]（今[[四川|四川]][[廣漢市|廣漢市]]）刺史。代宗即位，拜為特進、[[刑部尚書|刑部尚書]]。[[廣德|廣德]]元年（[[寶應|寶應]]二年七月改元廣德）八月四日於赴任途中卒於[[阆州|阆州]]（今[[四川|四川]][[阆中|阆中]]）僧舍，<ref>[[段成式|段成式]]《[[酉阳杂俎|酉阳杂俎]]·壶史》：“房琯太尉祈邢算终身之事。邢言：‘若来由东南，止西北，禄命卒矣。降魄之处，非馆非寺，非途非署。病起于鱼飧，休于龟兹板。’後房自袁州除汉州，及罢归，至阆州，舍紫极宫。适雇工治木，房怪其木理成形，问之，道士称：‘数月前，有贾客施数段龟兹板，今治为屠苏也。’房始忆邢之言。”《[[太平廣記|太平廣記]]》卷第二百十五亦載此事。</ref>葬于阆州城外，年六十七，追封为[[太尉|太尉]]。

房琯有三子：[[房宗偃|房宗偃]]，[[御史中丞|御史中丞]]；[[房乘|房乘]]，[[祕書郎|祕書郎]]；[[房孺復|房孺復]]，[[容州|容州]]刺史。房乘子[[房啟|房啟]]，容管經略使；房启子[[房越|房越]]。
