{{日本作品|房東妹子青春期|大家さんは思春期!}}，是[[水瀨瑠瑠羽|水瀨瑠瑠羽]]的[[四格漫畫|四格漫畫]]作品，於[[芳文社|芳文社]]的雜誌《{{tsl|ja|まんがタイム|Manga Time}}》和《{{tsl|ja|まんがタイムファミリー|Manga Time Family}}》連載。[[電視動畫|電視動畫]]於2016年1月10日开始播放。

== 概要 ==
作品描述因為就職的關係而開始獨居生活的前田，在居住的公寓和中學生房東與房客所度過的日常生活。

== 登場角色 ==
=== 主要登場角色 ===
; {{nihongo|里中千惠|里中 チエ（さとなか チエ）|聲：[[久保百合花|久保百合花]]}}
: 住在101號房，公寓的房東，中學一年級生，不論早上到晚上，週日或周末都穿著校服，本人解釋這令她感到安心和專心。雖然只是舊校服，但本人十分珍惜。
: 多次提出並進入前田的房間，理由包括教他煮菜和保護她，但本人並未發現前田的不知所措和白井麗子對二人獨處的擔心。
: 雖然平日十分和順，但對著同級生時也可以變得強硬。
; {{nihongo|前田|前田（まえだ）|聲：[[近藤隆|近藤隆]]<ref name="niconicochannel">{{cite web |script-title=ja:大家さんは思春期！ |url=http://ch.nicovideo.jp/ooyasan-anime |publisher={{tsl|ja|ニコニコチャンネル|Niconico频道}}|accessdate=2015-12-25 |language=ja }}</ref>}}
: 住在203號房，上班族，全名不詳。23歲。由於就職的因素搬進千惠的公寓。
: 千惠是第一個進入他房間的女生。對「可愛的房東」提出進入他的房間的行為，感到不知所措。雖然曾有非分之想，但在她純潔的笑容前感到非常內疚。
: 曾因千惠無惡意地説他是爸爸般的人，即自己是大叔，而受到打擊。
: 早上與千惠跟麗子打招呼而招致路上的學生和上班族投以嫉妒的眼光。
; {{nihongo|白井麗子|白井 麗子|聲：[[生天目仁美|生天目仁美]]}}
: 住在204號房，很疼愛千惠的女性，職業是酒吧的女招待。
: 曾表示自己喜歡年紀大的有錢人。
; {{nihongo|熊川|熊川|}}
: 住在104號房，少女向的輕小說作家，在少女小說的世界很有名氣。
; {{nihongo|千代|チヨ|}}
: 前代房東，千惠的祖母，總是穿著和服。
: 擁有「傳說的廚師」、「指尖的魔術師」等稱呼。

=== 千惠的朋友 ===
; {{nihongo|植野真由|植野 真由（うえの まゆ）|聲：[[木戶衣吹|木戶衣吹]]}}
: 跟千惠一起吃便當後，跟千惠交情變好的好友。獨生女，體育委員。
: 沒有參加社團，但是有加入料理同好會。雖未參加社團，但有學習插花、茶道、書法與舞蹈，並擅長唱歌。
; {{nihongo|宮村雪|宮村 雪（みやむら ゆき）|聲：[[山崎惠理|山崎惠理]]}}
: 最初邀請總是獨自行動的千惠一起吃便當的同學。
: 網球社、料理同好會成員。小學生開始就為了胸部發育而煩惱。愛操心並喜歡可愛的東西。
: 有兩個弟弟。
; {{nihongo|森明日香|森 明日香（もり あすか）|聲：{{tsl|ja|中島由貴 (声優)|中島由貴 (声優)|中島由貴}}}}
: 田徑社、料理同好會成員。運動神經超群且喜歡戀愛話題。
: 普遍戴著眼鏡，但當進行社團活動摘下眼鏡時的長相會讓人完全認不出來。
; {{nihongo|金本櫻|金本 さくら（かねもと さくら）|聲：和田京子}}
: 男子排球社的經紀人，並加入料理同好會。擅於繪畫。喜歡漫畫、照顧人。體形豐腴。

=== 千惠的同學 ===
; {{nihongo|佐佐木健吾|佐々木 健吾|聲：[[小林沙苗|小林沙苗]]}}
: 喜歡千惠的男學生，曾在公寓附近徘徊而被當成可疑人物。因為跟蹤千惠到家門前，所以表白到一半就被狠狠拒絕。
; {{nihongo|瀧澤瑠衣|滝沢 瑠衣|}}
: 對自己的美貌很有自信的女學生，把在男女間人氣都很高的千惠當成勁敵。

== 電視動畫 ==
2015年9月發表改編動畫的消息<ref>{{Cite web|url=http://natalie.mu/comic/news/160325|script-title=ja:安アパートの大家をする女子中学生4コマ「大家さんは思春期!」がアニメ化|publisher=コミックナタリー（ナターシャ）|date=2015年9月17日|accessdate=2015年11月24日|language=ja}}</ref>，之後在《Manga Time》2016年1月號公布[[電視動畫|電視動畫]]于2016年1月10日開始播放<ref>{{Cite web|url=http://www.e-hon.ne.jp/bec/SA/DetailZasshi?refShinCode=0900000004910086450162&Action_id=101&Sza_id=C0|script-title=ja:まんがタイム 2016年1月号|publisher=[[東販|東販]]|accessdate=2015年12月26日|language=ja}}</ref>。2016年1月10日由[[TOKYO_MX|TOKYO MX]]、[[SUN電視台|SUN電視台]]首播。<ref>{{Cite web|url=http://ooyasan-anime.com/onair/|script-title=ja:TVアニメ「大家さんは思春期！」放送情報|accessdate=2016年1月15日|language=ja}}</ref>

房東里中千惠聲優是由擔任《[[LoveLive!|LoveLive!]]》角色小泉花陽聲優的[[久保百合花|久保百合花]]擔任。動畫由Seven Arcs Pictures製作。監督是小川優樹，角色設計、總作畫監督由清水厚貴擔任。主題曲《Shining Sky》由高中生木戶衣吹及山崎惠理組成的「every♥ing!」演唱。<ref>{{Cite web|url=http://natalie.mu/comic/news/166429|script-title=ja:「大家さんは思春期！」チエちゃんの声は久保ユリカ、キービジュも公開|publisher=[[Natalie|Natalie]]|date=2015年11月18日|accessdate=2016年1月15日|language=ja}}</ref><ref>{{Cite web|url=http://ooyasan-anime.com/staffcast/|script-title=ja:スタッフ&キャスト|accessdate=2016年1月16日|language=ja}}</ref>
