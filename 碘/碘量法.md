
'''碘量法'''或称'''碘量滴定法'''是一种[[分析化学|分析化学]]中常用到的[[滴定|滴定]]方法，同时也是一种[[氧化还原滴定|氧化还原滴定]]的方法，碘量法中，通过[[碘|碘单质]]的生成或者反应完全作为滴定终点的现象。
碘量法分为间接碘量法和直接碘量法，间接碘量法通过与被分析物反应释放出的碘滴定，而直接碘量法使用碘单质滴定

==基本原理==
[[File:Potassium_iodide_starch.jpg|thumb]]

碘量法经常被用来测定水溶液中[[氧化剂|氧化剂]]的[[浓度|浓度]]，比如[[生态学|生态研究]]中的测定[[溶解氧|溶解氧]]含量或者例如游泳池中活性氯的浓度。
首先需要知道样品溶液的体积，再加入过量且量已知的[[碘化物|碘化物]]溶液，样品中的氧化剂将碘离子氧化为碘单质，碘单质溶于含碘化物的溶液生成{{Chem|I|-|3}}，溶液最终变为深褐色。

之后，含{{Chem|I|-|3}}的溶液用标准[[硫代硫酸钠|硫代硫酸钠]]溶液滴定，并且以[[淀粉|淀粉]]为指示剂，以此计算出之前生成的{{Chem|I|3|-}}的量，间接地计算出之前的氧化剂的量。

{{Chem|I|-|3}}具有一定的氧化性：
: I<sub>3</sub><sup>-</sup> + 2 e<sup>-</sup> {{unicode|⇌}} 3 I<sup>-</sup> (''E''<sup>o</sup> = + 0.5355 V)

同时[[硫代硫酸钠|硫代硫酸钠]]具有较低的[[电极电势|电极电势]]：
: S<sub>4</sub>O<sub>6</sub><sup>2-</sup> + 2 e<sup>-</sup> {{unicode|⇌}} 2 S<sub>2</sub>O<sub>3</sub><sup>2-</sup> (''E''<sup>o</sup> = + 0.08 V)<ref>{{RubberBible87th}}</ref>

并组成了最终的总反应：
: I<sub>3</sub><sup>-</sup> + 2 S<sub>2</sub>O<sub>3</sub><sup>2-</sup> → S<sub>4</sub>O<sub>6</sub><sup>2-</sup> + 3 I<sup>-</sup> (''E''<sup>o</sup> = + 0.4555 V)

为了方便起见，在水溶液中，一般写作{{Chem|I|2}}而不是{{Chem|I|3|-}}。

当溶液深蓝色消失时，意味着碘单质已被完全反应， 即到达了滴定终点。

当然，用来滴定碘含量的还原剂并不一定选用[[硫代硫酸钠|硫代硫酸钠]]，我们也可以选择[[氯化亚锡|氯化亚锡]]、砷(III)、锑(III)。<ref name="Vogel">{{VogelQuantitative6th}}</ref>
值得注意的是，在pH值较低时，会发生如下的副反应：
:S<sub>2</sub>O<sub>3</sub><sup>2−</sup> + 2 H<sup>+</sup> → SO<sub>2</sub> + S + H<sub>2</sub>O

有些反应在特定的pH值范围内是可逆的，因此在滴定之前应当调整样品溶液至合适的pH值。例如，如下反应，在pH<4时为可逆反应：
: H<sub>3</sub>AsO<sub>3</sub> + I<sub>2</sub> + H<sub>2</sub>O → H<sub>3</sub>AsO<sub>4</sub> + 2 H<sup>+</sup> + 2 I<sup>-</sup>

碘单质的易挥发性容易引起滴定的误差，这可以通过确保加入的碘化物是过量来有效避免，同时，降低溶液的温度也是减小挥发造成的误差的有效方法。强光、[[亚硝酸盐|亚硝酸盐]]、[[铜|铜离子]]会[[催化剂|催化]]碘化物氧化为碘单质，所以在加入碘化物之前，应当除去样品中的亚硝酸盐和铜离子。

==应用==
碘量法在容量分析法中应用十分广泛，例如测定铜离子、[[氯酸盐|氯酸盐]]、[[过氧化氢|过氧化氢]]和水中溶解氧的含量：
: 2 Cu<sup>2+</sup> + 4I<sup>-</sup> → 4 CuI + I<sub>2</sub>
: 6 H<sup>+</sup> + ClO<sub>3</sub><sup>-</sup> + 6 I<sup>-</sup> → 3 I<sub>2</sub> + Cl<sup>-</sup> + 3 H<sub>2</sub>O
: 2 H<sup>+</sup> + H<sub>2</sub>O<sub>2</sub> + 2 I<sup>-</sup> → I<sub>2</sub> + 2 H<sub>2</sub>O
: 2 H<sub>2</sub>O + 4 Mn(OH)<sub>2</sub> + O<sub>2</sub> → 4 Mn(OH)<sub>3</sub>
: 2 Mn<sup>3+</sup> + 2 I<sup>-</sup> → I<sub>2</sub> + 2 Mn<sup>2+</sup>

===测定[[亚硫酸盐|亚硫酸盐]]及[[亚硫酸氢盐|亚硫酸氢盐]]===
在酸性介质中，亚硫酸根或亚硫酸氢根能将碘单质还原为碘离子。因此，将过量的碘标准溶液加入已知体积的样品中，亚硫酸根或亚硫酸氢根会与碘离子发生定量反应：

: SO<sub>3</sub><sup>2-</sup> + I<sub>2</sub> + H<sub>2</sub>O → SO<sub>4</sub><sup>2-</sup> + 2 H<sup>+</sup> + 2 I<sup>-</sup>

: HSO<sub>3</sub><sup>-</sup> + I<sub>2</sub> + H<sub>2</sub>O → SO<sub>4</sub><sup>2-</sup> + 3 H<sup>+</sup> + 2 I<sup>-</sup>


===测定铁氰化钾===
当碘离子被加入铁氰化钾溶液时，溶液中存在以下反应平衡：

: 2 [Fe(CN)<sub>6</sub>]<sup>3-</sup> + 2 I<sup>-</sup> {{unicode|⇌}} 2 [Fe(CN)<sub>6</sub>]<sup>4-</sup> + I<sub>2</sub>

在强酸性条件下，该反应会向正方向进行，但在中性条件下会变为可逆反应，但在强酸性条件下，硫代硫酸钠会分解。因此，为促进反应的进行，在溶液中加入过量的锌盐，锌离子会与亚铁氰根发生定量的反应并生成沉淀，促进反应的进行：

: 2 [Fe(CN)<sub>6</sub>]<sup>3-</sup> + 2 I<sup>-</sup> + 2 K<sup>+</sup> + 2 Zn<sup>2+</sup> → 2 KZn[Fe(CN)<sub>6</sub>] + I<sub>2</sub>

这样可以避免使用强酸性环境，有效的避开了硫代硫酸盐在强酸性下分解的问题，也使得铁氰化钾可以用碘量法进行测量。
