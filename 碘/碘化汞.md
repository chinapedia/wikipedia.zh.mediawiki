'''碘化汞'''（化學式：[[汞|Hg]][[碘|I<sub>2</sub>]]）是橘紅色的[[化合物|化合物]]。碘化汞极難溶於水（<100 ppm），和[[氯化汞|氯化汞]]不同。

碘化汞是无机汞化合物中，挥发性最大的。<ref name=env>《环境化学》. 戴树桂 主编. 高等教育出版社. 第六章 典型污染物在环境各圈层中的转归与效应. P<sub>391</sub></ref>
== 性質 ==

有红色和黄色两种变体。几乎不溶于水，溶于[[甲醇|甲醇]](2.6)、[[乙醇|乙醇]](1.8)、[[氯仿|氯仿]]、[[乙醚|乙醚]](0.5)、[[丙酮|丙酮]](3)、[[苯|苯]](0.2)、[[甘油|甘油]]、[[二硫化碳|二硫化碳]]等[[有机溶剂|有机溶剂]]（括号内为[[溶解度|溶解度]]，g/100g 有机溶剂，25 °C）。极毒，有强烈刺激性。

储存时，远离火种、热源，避免光照。包装密封。避免接触[[溴化物|溴化物]]、[[氯化物|氯化物]]、[[氨|氨]]、[[碱|碱]]、[[氰化物|氰化物]]、[[铜|铜]]盐、[[铅|铅]]盐、[[碘仿|碘仿]]和[[过氧化氢|过氧化氢]]。

碘化汞在自然界中以极其罕见的[[碘汞矿|碘汞矿]]（Coccinite）的形式存在。

== 结构 ==

碘化汞是有[[熱色性|熱色性]]的固体。加熱到126 °C以上时，碘化汞會從红色的α-型变体[[相變|相變]]為淺黃色的β-型晶形，當冷卻時，碘化汞的會慢慢的恢復為原來的顏色。碘化汞常用來做為[[熱變色|熱變色]]的實例。<ref>[http://jchemed.chem.wisc.edu/JCESoft/CCA/CCA2/MAIN/THERMHG/CD2R1.HTM Thermochromism: Mercury(II) Iodide]</ref>

红色的α-型变体在室温下稳定，具有以 HgI<sub>4</sub> 四面体顶点相连接的层状结构，碘原子处于畸变的立方密堆积体中，汞原子占四面体空穴的四分之一。超过129摄氏度时，黄色的β-型变体是稳定的，它具有类似[[溴化汞|溴化汞]]的[[正交晶系|正交晶系]]结构。此外，迅速蒸发红色碘化汞的温热溶液可以得到另外一种橙色的晶体，但看来其类似于α-型。<ref>萧文锦，钟兴厚，袁启华．《无机化学丛书》第六卷 卤素 铜分族 锌分族．北京：科学出版社，1998年10月．页833．书号 03-003647-6．</ref>

== 製備 ==

红色碘化汞由[[碘化钾|碘化钾]]溶液加入[[氯化汞|氯化汞]]溶液而得（见下式）。黄色碘化汞由将红色碘化汞溶于[[乙醇|乙醇]]后再倾入冷水中而制得，在室温下不稳定，经过几小时就转变为红色碘化汞。

: HgCl<sub>2</sub> + 2 KI → HgI<sub>2</sub> + 2 KCl

碘化汞也可由单质[[汞|汞]]和[[碘|碘]]经化合反應而成：

: Hg + I<sub>2</sub> → HgI<sub>2</sub>

== 應用 ==

碘化汞常用來製備[[奈斯勒试剂|奈斯勒试剂]]，可用來鑒定試樣中[[氨|氨]]的成份。碘化汞也是[[半导体材料|半导体材料]]，用在室溫下運作的[[伽馬射線|伽馬射線]]或[[X射線|X射線]]檢測或顯像設備。

在醫藥上，碘化汞用來治療[[梅毒|梅毒]]。在[[獸醫學|獸醫學]]中，在治療[[骨刺|骨刺]]、[[粘液囊|粘液囊]]腫大的軟膏中含有碘化汞。常用的外傷消毒用藥[[紅藥水|紅藥水]]不能與[[碘酒|碘酒]]一起使用，是因為兩者會起化學反應產生劇毒的碘化汞。
