'''碘钟反应'''（Iodine clock reaction）是一种[[化学振荡|化学振荡]]反应，其体现了[[化学动力学|化学动力学]]的原理。它于1886年被[[瑞士|瑞士]][[化学家|化学家]]Hans Heinrich Landolt发现。在碘钟反应中，两种（或三种）无色的液体被混合在一起，并在几秒钟后变成碘蓝色。碘钟反应可以通过不同的途径实现。

==过氧化氢型碘钟==
向[[硫酸|硫酸]]酸化的[[过氧化氢|过氧化氢]]溶液中加入[[碘酸钾|碘酸钾]]、[[硫代硫酸钠|硫代硫酸钠]]和[[淀粉|淀粉]]的混合溶液。此时在体系中存在两个主要反应，[[离子方程式|离子方程式]]为：

H<sub>2</sub>O<sub>2</sub>([[溶液|aq]])+3[[碘离子|I<sup>−</sup>]](aq)+2[[氢负离子|H<sup>+</sup>]]→ I<sub>3</sub><sup>−</sup>+2H<sub>2</sub>O

I<sub>3</sub><sup>−</sup>(aq)+2[[硫代硫酸根|S<sub>2</sub>O<sub>3</sub><sup>2−</sup>]](aq) → 3I<sup>−</sup>(aq)+[[连四硫酸根离子|S<sub>4</sub>O<sub>6</sub><sup>2−</sup>]](aq)

==碘酸盐型碘钟==
向用硫酸酸化的[[碘酸盐|碘酸盐]]中加入[[亚硫酸氢钠|亚硫酸氢钠]]（以及少量淀粉溶液），此时体系中出现如下反应：

[[碘酸根离子|IO<sub>3</sub><sup>−</sup>]] (aq) + 3[[亚硫酸氢根离子|HSO<sub>3</sub><sup>−</sup>]] (aq) →  I<sup>−</sup> (aq) + 3[[硫酸氢根离子|HSO<sub>4</sub><sup>−</sup>]](aq)

然后过量的碘酸根离子与碘离子发生[[归中反应|归中反应]]：

IO<sub>3</sub><sup>−</sup> (aq) + 5I<sup>−</sup> (aq) + 6H<sup>+</sup> (aq) →  3I<sub>2</sub> + 3H<sub>2</sub>O (l)

接着亚硫酸氢钠将生成的碘还原：

I<sub>2</sub> (aq) + HSO<sub>3</sub><sup>−</sup> (aq) + H<sub>2</sub>O (l) →  2I<sup>−</sup> (aq) + HSO<sub>4</sub><sup>−</sup>(aq) + 2H<sup>+</sup> (aq)

==过硫酸盐型碘钟==
通过[[过硫酸钾|过硫酸钾]]、[[过硫酸钠|过硫酸钠]]或[[过硫酸铵|过硫酸铵]]将碘离子氧化成碘单质。加入硫代硫酸钠可以将碘单质还原回碘离子。离子方程式如下：

2I<sup>−</sup>(aq) + [[过硫酸根离子|S<sub>2</sub>O<sub>8</sub><sup>2−</sup>]](aq) → I<sub>2 </sub>(aq)+ 2[[硫酸根离子|SO<sub>4</sub><sup>2−</sup>]](aq)

I<sub>2</sub> (aq) + 2S<sub>2</sub>O<sub>3</sub><sup>2−</sup>(aq) → 2I<sup>−</sup>(aq) + S<sub>4</sub>O<sub>6</sub><sup>2−</sup>(aq)

==氯酸盐型碘钟==
将[[卢戈氏碘液|卢戈氏碘液]]、[[氯酸钠|氯酸钠]]和[[高氯酸|高氯酸]]混合，离子方程式如下：

I<sub>3</sub><sup>−</sup>  → I<sup>−</sup> + I<sub>2</sub>

[[氯酸根离子|ClO<sub>3</sub><sup>−</sup>]] + I<sup>−</sup> + 2H<sup>+</sup> → [[次碘酸|HIO]] +[[亚氯酸|HClO<sub>2</sub>]]

ClO<sub>3</sub><sup>−</sup> + HIO + H<sup>+</sup> → [[亚碘酸|HIO<sub>2</sub>]] + HClO<sub>2</sub>

ClO<sub>3</sub><sup>−</sup> + HIO<sub>2</sub> → IO<sub>3</sub><sup>−</sup> + HClO<sub>2</sub>

==一个简单的碘钟实验==
下面介绍一个简单的碘钟实验<ref>{{cite book|title=《中学化学实验大全》|pages=431~432|ISBN=7-5320-3471-2|language=zh-hans}}</ref>，又被稱為B-R[[震盪反應|震盪反應]]
===实验药品===
29%过氧化氢溶液、[[丙二酸|丙二酸]]、[[硫酸锰|硫酸锰]]、可溶性淀粉、碘酸钾、2mol/L硫酸

===实验步骤===
#配置甲溶液：量取97mL29%的过氧化氢溶液，转移入250mL[[容量瓶|容量瓶]]里，用蒸馏水稀释到刻度，得3.6mol/L过氧化氢溶液。
#配置乙溶液：分别称取3.9g丙二酸和0.76g硫酸锰，分别溶于适量水中。另称取0.075g可溶性淀粉，溶于50mL左右沸水中。把三者转移入250mL容量瓶里，稀释到刻度，得到含0.15mol/L丙二酸、0.02mol/L硫酸锰、和0.03%淀粉的混合溶液。
#配置丙溶液：称取10.75g碘酸钾溶于适量热水中，再加入20mL2mol/L硫酸溶液酸化。转移入250mL容量瓶里，稀释到刻度，得到0.2mol/L碘酸钾和0.08mol/L硫酸的混合溶液。
#将甲、乙、丙三组溶液以等体积混合在[[锥形瓶|锥形瓶]]中，这混合溶液分别含过氧化氢1.2mol/L、丙二酸0.05mol/L、硫酸锰0.0067mol/L、碘酸钾0.067mol/L、淀粉0.01%。混合后，反应液由无色变为蓝紫色，几秒后褪为无色，接着又称琥珀色变逐渐加深，蓝紫色又反复出现，几秒后又消失，这样周而复始地呈周期性变化。此為B-R振荡反应，又叫“碘钟反应”。振荡周期约为8秒，反复振荡能持续10多分钟。
===注意事项===
#碘钟反应速率与温度有关，但溫度不能太高，否則碘-澱粉錯合物不安定，藍色會不明顯
#丙溶液会随室温降低，碘酸钾以晶体析出，微热又溶解。甲溶液不宜放置太久，否则过氧化氢分解失效而导致实验失败。
#不可随意更改药品用量
