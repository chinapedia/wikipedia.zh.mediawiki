'''碘酸钠'''（[[钠|Na]][[碘|I]][[氧|O<sub>3</sub>]]）是[[碘酸|碘酸]]的[[钠|钠]][[盐|盐]]。碘酸钠是一种[[氧化剂|氧化剂]]，但比[[氯酸钠|氯酸钠]]和[[溴酸钠|溴酸钠]]稳定得多。它与可燃物或[[还原剂|还原剂]]接触可能导致火灾。

== 制备 ==
它可以由[[氢氧化钠|氢氧化钠]]与[[碘酸|碘酸]]的[[中和反应|中和反应]]制备：

:[[碘酸|HIO<sub>3</sub>]] + [[氢氧化钠|NaOH]] → NaIO<sub>3</sub> + [[水|H<sub>2</sub>O]]

还可以向热的氢氧化钠或[[碳酸钠|碳酸钠]]溶液加入碘单质，使其发生[[歧化反应|歧化反应]]生成碘酸钠：

:3 [[碘|I<sub>2</sub>]] + 6 [[氢氧化钠|NaOH]] → NaIO<sub>3</sub> + 5 [[碘化钠|NaI]] + 3 [[水|H<sub>2</sub>O]]

此外，用[[高锰酸盐|高锰酸盐]]、[[铬酸盐|铬酸盐]]氧化碘单质，[[氯酸钾|氯酸钾]]氧化[[三氯化碘|三氯化碘]]都可以制备碘酸钠。
:KClO<sub>3</sub> + ICl<sub>3</sub> → KIO<sub>3</sub> + 2 Cl<sub>2</sub>

== 反应 ==
碘酸钠在水溶液中能被[[次氯酸盐|次氯酸盐]]等强氧化剂氧化成[[高碘酸钠|高碘酸钠]]：

:NaIO<sub>3</sub> + [[次氯酸钠|NaClO]] → [[高碘酸钠|NaIO<sub>4</sub>]] + [[氯化钠|NaCl]]

碘酸钠受热则分解为[[碘化钠|碘化钠]]和[[氧|氧]]气：
:2 NaIO<sub>3</sub> → 2 NaI + 3 O<sub>2</sub>

== 安全 ==
保存需避免受热、震动、摩擦，不可与可燃材料、[[铝|铝]]、[[有机化合物|有机化合物]]、[[碳|碳]]、[[过氧化氢|过氧化氢]]、[[硫化物|硫化物]]等接触。

== 应用 ==
医学上用于防腐消毒。
