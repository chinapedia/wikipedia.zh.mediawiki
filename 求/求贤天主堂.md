'''求贤天主堂'''，位于[[北京市|北京市]][[大兴区|大兴区]][[榆垡镇|榆垡镇]]求贤村，隶属[[天主教北京教区|天主教北京教区]]。

==简介==
求贤天主堂始建于约1901年。老教堂已毁。现在的教堂是[[改革开放|改革开放]]后重建的。天主堂的建筑形式一般，但有独立的院子。求贤天主堂是北京教区内规模较小的一座教堂。<ref>{{Cite web|url=http://www.mzb.com.cn/html/report/298255-1.htm |title=北京的天主教建筑 |author= |date=2012-04-28 |publisher=中国民族宗教网 |language= |accessdate= |quote= |deadurl=yes |archiveurl=https://web.archive.org/web/20161225151542/http://www.mzb.com.cn/html/report/298255-1.htm |archivedate=2016-12-25 }}</ref><ref>{{Cite web |url = http://dxmw.bjdx.gov.cn/web/mw/xwdt/tpxw/265241.htm |title = 赵宏生委员带队到大兴检查和谐寺堂建设工作 |author =  |date =2010-06-07  |publisher =北京市大兴区民族事务委员会  |language =  |accessdate =  |quote =  }}</ref>2010年代初，该堂平时每天进堂的教友多为老年人，本堂马曌神父为吸引白天上班的年轻人，每晚在堂里举办关于年轻人的灵修祈祷。求贤堂还有个多年形成的传统，每年[[清明节|清明节]]和追思已亡瞻礼，神父要随教友到坟地拜安所，特别为炼灵祈祷。<ref name=ma/>2013年，在本堂马曌神父的指导下，求贤堂乐队成立。求贤堂乐队由该堂区的教友组成，除了为本堂区服务外，还为北京教区下辖的各堂区提供服务。<ref>{{Cite web |url =http://www.tianguangbao.org/2015.2.1/24.htm |title = 求贤堂乐队成立两周年 |author =  |date = 2015-02-01 |publisher = 天光 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url =http://www.tianguangbao.org/2013.10.15/15.htm  |title = 求贤堂号队崭新亮相 |author = 康玛利亚 |date =  2013-10-15|publisher = 天光 |language =  |accessdate =  |quote =  }}</ref>
