'''货币符号'''是一种常被用来作为[[货币|货币]]名称的图像速记[[符号|符号]]。货币符号主要在各自的国家内使用，而国际上使用[[ISO_4217|ISO 4217]]作为货币标识。世界上大多数货币并没有特定的符号。

== 擺放的位置 ==
不同的货币，其符号与金额的位置也不同。比如很多拉丁美洲和英语国家，把货币符号放在金额前面（比如「£50.00」）；另外也有很多国家把符号放在金额后面（比如「50.00 S₣」）；葡萄牙士姑度和法国法郎的符号在弃用之前放在进制符位置（比如「50$00」或「12₣34」）。在欧洲，如果国内没有传统标准，那么-{zh-hans:默认;zh-hant:預設}-地把(€)放在金额前面。当然，很多[[欧元区|欧元区]]国家仍然保持自己的传统。

== 小数点的用法 ==
有的国家小数点的用法也有自己的传统。比如英国经常用[[间隔号|间隔号]]当作小数点用在商品标价中（比如「£5·52」)，但是在打印体中不这么用。也有一些国家用逗号（比如「5,00 €」）。

== 历史上使用过的货币符号 ==
* ₳ [[阿根廷奥斯特拉尔|阿根廷奥斯特拉尔]]
* ₢ [[巴西克鲁塞罗|巴西克鲁塞罗]]
* ℳ 1945年前德国[[國家馬克|国家马克]]的符号
* ₰ 1945年前德国[[國家馬克|国家马克]] [[芬尼|芬尼]]的符号
* ₯ [[希腊德拉克马|希腊德拉克马]]
* ₠ [[欧洲货币单位|欧洲货币单位]]，没有被广泛使用，现在被[[欧元|欧元]]替代。
* ₣ [[法郎|法郎]]以前在[[法国|法国]]使用。大多数人喜欢写为"FF"，现在作为[[非洲法郎|非洲法郎]]和[[太平洋法郎|太平洋法郎]]的符号。
* ₤ [[里拉|里拉]]，曾在[[意大利|意大利]]，[[圣马力诺|圣马力诺]]和[[梵蒂冈城|梵蒂冈城]]使用过，现在在[[马耳他|马耳他]]使用，也做GBP的手写符号。
* Kčs [[捷克斯洛伐克克郎|捷克斯洛伐克克郎]]
* ₧ [[比塞塔|比塞塔]]，曾在西班牙和安道尔使用。大多数人喜欢写为“pts”
* Rs. [[印度卢比|印度卢比]]
* ''ƒ'' [[荷兰盾|荷兰盾]]，现在在荷属安的列斯群岛和阿鲁巴使用
