'''菊池正士'''（{{bd|1902年|8月25日|1974年|11月12日||catIdx=Kikuchi Seishi}}），[[日本|日本]]核物理学家，他最早发现了[[电子显微学|电子显微学]]中出现的{{le|菊池線 (物理)|Kikuchi line}}，并给出了正确的理论解释<ref name = "岩波理化学">{{cite book|title=《岩波理化学辞典》第5版 |year=1998 |publisher=[[岩波书店|岩波书店]] |location=[[东京都|东京都]] |isbn=4-00-080090-6}}</ref>。

==简历==
1902年生于[[东京|东京]]。1926年毕业于[[东京帝国大学|东京帝国大学]]理学部[[物理学|物理学]]专业。1928年进入[[理化学研究所|理化学研究所]]开始[[电子|电子]]束[[衍射|衍射]]方面的研究，并在世界上首次观察到[[云母|云母]]薄膜的完整电子衍射花样，首次提出并解释了'''菊池线'''现象。1929年赴[[德国|德国]]留学。1934年起任[[大阪帝国大学|大阪帝国大学]]教授，主持建造了日本第二个考克饒夫－瓦耳頓型（{{lang-en|Cockcroft - Walton}}）直流高压加速器（次於[[臺北帝國大學|臺北帝國大學]]物理學講座的[[荒勝文策|荒勝文策]]教授）<ref>{{cite web|url=https://watanaby.files.wordpress.com/2013/02/inoue-1.pdf |language=ja |title=破壊されたサイクロトロンこぼれ書き |author=井上 信 | date=2008-09-30 |accessdate=2015-01-26 }}</ref><ref>{{cite web |url=http://archives.kek.jp/exhibit/jyohohiroba_pamphlet.pdf |language=ja |title=加速器科学の 過去 黎明期 |publisher=[[KEK|KEK]] |accessdate=2015-01-26 |deadurl=yes |archiveurl=https://web.archive.org/web/20160314011740/http://archives.kek.jp/exhibit/jyohohiroba_pamphlet.pdf |archivedate=2016-03-14 }}</ref>。[[二战|二战]]结束后，于1955年起任[[东京大学|东京大学]]原子核研究所首任所长，并成功主持完成了可变能量[[回旋加速器|回旋加速器]]的建造。1959年～1964年任[[日本原子力研究所|日本原子力研究所]]理事长。1974年逝世<ref name = "岩波理化学"/>。
