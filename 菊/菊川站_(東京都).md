'''菊川車站'''（{{jpn|j=菊川駅|hg=きくかわえき|rm=Kikukawa eki|f=y}}）是位於[[日本|日本]][[東京都|東京都]][[墨田區|墨田區]][[菊川_(墨田區)|菊川]]三丁目<ref name="ekizyouhou">[http://www.kotsu.metro.tokyo.jp/subway/stations/kikukawa/s12.html 菊川｜東京都交通局] {{webarchive|url=https://www.webcitation.org/69ZvwAnZF?url=http://www.kotsu.metro.tokyo.jp/subway/stations/kikukawa/s12.html |date=2012-07-31 }}、東京都交通局公式ホームページ、各駅情報（菊川駅）、2010年11月14日閲覧</ref>，屬於[[東京都交通局|東京都交通局]]（[[都營地下鐵|都營地下鐵]]）[[新宿線_(都營地下鐵)|新宿線]]的[[鐵路車站|鐵路車站]]。

[[車站編號|車站編號]]是'''S 12'''<ref name="ekizyouhou"></ref>。

== 歷史 ==
* 1978年（[[昭和|昭和]]53年）12月21日：開業。
* 2018年（[[平成|平成]]30年）11月3日：啟用[[月台閘門|月台閘門]]<ref>{{Cite press release|url=https://www.kotsu.metro.tokyo.jp/pickup_information/news/pdf/2018/sub_p_20180226_h_01.pdf|format=PDF|language=ja|title=都営新宿線におけるホームドアの設置、運用開始について|publisher=東京都交通局|date=2018-02-26|accessdate=2018-12-15}}</ref>。

== 车站周辺 ==
車站位於{{link-ja|東京都道、千葉縣道50號東京市川線|東京都道・千葉県道50号東京市川線|新大橋通}}與{{link-ja|東京都道319号環状三号線|東京都道319号環状三号線|三目通}}交差點附近。周邊是[[菊川_(墨田區)|菊川]]（一丁目～三丁目）、[[森下_(江東區)|森下]]（五丁目）、[[立川_(墨田區)|立川]]（二丁目～四丁目）<ref name="deiriguchi">[http://www.kotsu.metro.tokyo.jp/subway/stations/kikukawa/gateway.html 駅出入口情報] {{webarchive|url=https://web.archive.org/web/20100203145206/http://www.kotsu.metro.tokyo.jp/subway/stations/kikukawa/gateway.html |date=2010-02-03 }}、東京都交通局公式ホームページ、駅出入口情報（菊川駅）、2010年11月14日閲覧</ref>。

車站出入口有A1～A4共4處。

==可供轉乘的巴士路線==
最近的巴士站是'''菊川站前'''。以下路線由[[東京都交通局|東京都交通局]]（[[都營巴士|都營巴士]]）運行。
; 1號乘車處（新大橋通上、A1出口附近）
* 錦11：經[[江東橋|江東橋]]往[[錦糸町站|錦糸町站]]、[[龜戶站|龜戶站]]

; 2號乘車處（新大橋通上、A2出口附近）
* 錦11：經[[水天宮前站|水天宮前]]、[[茅場町站|茅場町]]、[[新富町站|新富町]]往[[築地站|築地站]]

; 3號乘車處（三目通上、{{link-ja|丸悅|マルエツ}}前）
* 東20：經江東橋往錦糸町站

; 4號乘車處（三目通上、丸悅對面）
* 東20：經[[木場站|木場站]]、[[門前仲町站|門前仲町]]、[[日本橋站_(東京都)|日本橋站]]往[[東京站|東京站丸之內北口]]
* 業10：經木場站、[[豐洲站|豐洲站]]、[[銀座站|銀座四丁目]]往[[新橋站|新橋]]行＼經木場站、豐洲站、[[東雲_(江東區)|東雲]]都橋往{{link-ja|都營巴士深川營業所|都営バス深川営業所|深川車庫}}

; 5號乘車處（三目通上、Jonathan's前）
* 業10：經[[綠_(墨田區)|綠]]三丁目、[[石原_(墨田區)|石原]]三丁目、[[本所吾妻橋站|本所吾妻橋]]往[[東京晴空塔站|東京晴空塔站]]
