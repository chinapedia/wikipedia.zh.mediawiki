'''菊地 最愛'''（きくち もあ、{{bd|1999年|7月4日||}}）是[[日本偶像|日本偶像]]、[[歌手|歌手]]、[[藝人|藝人]]。经纪公司为[[AMUSE|AMUSE]]。於2010年起，她以'''MOAMETAL'''名義擔任金屬偶像舞蹈組合[[BABYMETAL|BABYMETAL]]的SCREAM及舞蹈{{Refnest|BABYMETALにおけるスクリームは本来の「[[叫び|叫び]]」や「[[悲鳴|悲鳴]]」ではなく、「合いの手」や「掛け声」などを意味する<ref>{{Cite web|url=http://news.mynavi.jp/news/2014/08/15/132/|title=今さら聞けない！？話題のアイドル「BABYMETAL」の注目すべき3つのポイントを紹介！|accessdate=2015-08-10|work=[[マイナビニュース|マイナビニュース]]|publisher=[[マイナビ|株式会社 マイナビ]]|date=2014-08-15|deadurl=yes|archiveurl=https://web.archive.org/web/20160305222914/http://news.mynavi.jp/news/2014/08/15/132/|archivedate=2016-03-05}}</ref>。|group="注"}}。為女子偶像組合[[櫻花學院|櫻花學院]]及衍生組合Twinklestars、Miniparti的元成員。[[愛知縣|愛知縣]][[名古屋市|名古屋市]]出身<ref name="ameblo">{{Cite web|url=http://ameblo.jp/sakuragakuin/entry-11772092273.html|title=待ってる時間に読んでください|accessdate=2015-08-19|date=2014-02-15|work=[[アメーバブログ|アメーバブログ]]|publisher=[[さくら学院|さくら学院]] 菊地最愛}}</ref>。

== 經歷 ==
菊地自少開始雜誌模特的工作。2007年，她獲得「[[CIAO|CIAO]] GIRL選拔2007」準優勝賞，然後加入[[Amuse|AMUSE]]。2008年至2010年間拍攝廣告及出席活動。2010年8月、以小學部「轉校生」的身份加入櫻花學院{{Refnest|[[テレ朝動画|テレ朝動画]]・LoGiRL『さくら学院の放課後!〜学んdeマンデー!〜』第1回放送や{{出典無効|date=2015年8月|title=検証可能性を満たしません。}}、さくら学院ブログの[[田口華|田口華]]の発言によると、菊地と[[水野由結|水野由結]]はギミックとして「転入生」であるが事実上のオリジナルメンバーであるとされている<ref>{{Cite web |url=http://ameblo.jp/sakuragakuin/entry-11837857048.html|title=今年も4月は大変でした|accessdate=2015-08-19|work=[[アメーバブログ|アメーバブログ]]|publisher=[[さくら学院|さくら学院]] [[田口華|田口華]]|date=2014-04-30}}</ref>。|group="注"}}<ref>{{Cite news|url=http://www.barks.jp/news/?id=1000065632|title=あの可憐Girl'sのメンバーが“さくら学院”として再びお目見え|date=2010-11-11|newspaper=BARKS|accessdate=2014-08-03}}</ref>。同年10月，參於衍生組合舞棒部「{{link-ja|Twinklestars|Twinklestars}}」、11月以'''MOAMETAL'''名義參與重音部「[[BABYMETAL|BABYMETAL]]」。

2011年5月2日至2013年3月25日，菊地與[[水野由結|水野由結]]以報榜女孩的身份，一同出演電視節目《{{link-ja|saku saku|Saku saku}}》中的環節「速報![[神奈川|神奈川縣]]最暢銷的CD單曲周榜」。同年7月，以第二代的成員身份參與烹飪部的「[[Mini-pati|Mini-pati]]」。

2013年4月，[[中元鈴香|中元鈴香]]（SU-METAL）於櫻花學院畢業<ref group="注" name="graduation">成員於初中畢業的同時也要從櫻花學院畢業（離隊）。</ref>，BABYMETAL亦隨之脫離櫻花學院獨立，以「課外活動」為名繼續活動<ref>{{Cite web|url=http://www.sakuragakuin.jp/news/others.php?page=5|title=さくら学院 職員室よりお知らせ|accessdate=2015-08-18|date=2013-02-01|publisher=[[さくら学院|さくら学院]]公式サイト 株式会社[[アミューズ|アミューズ]]|archiveurl=https://web.archive.org/web/20130613161823/http://www.sakuragakuin.jp/news/others.php?page=5|archivedate=2013-06-13|deadurl=yes}}</ref>。2014年5月，菊地出任櫻花學院的學生會長（隊長）<ref>{{Cite news|url=http://towershibuya.jp/news/2014/05/05/9125|title=【速報！】さくら学院、転入生＆生徒会長発表！|date=2014-05-05|newspaper=[[タワーレコード|タワーレコード]]渋谷|accessdate=2014-08-03|deadurl=yes|archiveurl=https://web.archive.org/web/20140808054743/http://towershibuya.jp/news/2014/05/05/9125|archivedate=2014-08-08}}</ref>。2015年3月29日，於櫻花學院畢業<ref group="注" name="graduation">成員於初中畢業的同時也要從櫻花學院畢業（離隊）。</ref><ref>{{Cite news|url=http://www.oricon.co.jp/news/2050754/full/|title=さくら学院、菊地最愛&水野由結ら涙の卒業式|date=2015-03-29|newspaper=[[オリコン|ORICON STYLE]]|accessdate=2015-03-30}}</ref>。她畢業後繼續以BABYMETAL成員的身份活動<ref name="oricon1">{{Cite web|url=http://www.oricon.co.jp/news/2050736/full/|title=さくら学院の卒業メンバーが学院生活と今後について語る|accessdate=2015-08-14|date=2015-03-29|work=ORICON STYLE|publisher=[[オリコン|オリコン]]株式会社}}</ref>。
== 個人 ==

菊地自少便學習[[芭蕾舞|芭蕾舞]]、[[网球|網球]]、[[体操|體操]]、[[游泳|遊泳]]<ref name="entamenext">{{Cite web|url=http://www.entamenext.com/news/detail/id=718|title=さくら学院 菊地最愛への100の質問【完全版】|accessdate=2015-06-30|date=2014年7月1日|publisher=月刊エンタメ [[徳間書店|徳間書店]]}}</ref>。[[日式咖喱飯|咖哩飯]]、[[梳士巴利漢堡扒|漢堡包]]、[[馬鈴薯沙拉|馬鈴薯沙拉]]是她擅長的菜式<ref name="entamenext">{{Cite web|url=http://www.entamenext.com/news/detail/id=718|title=さくら学院 菊地最愛への100の質問【完全版】|accessdate=2015-06-30|date=2014年7月1日|publisher=月刊エンタメ [[徳間書店|徳間書店]]}}</ref>。菊地的公開資料是[[ABO血型系统|A型]]血，但她在博客中曾表示「其實我不知道。」<ref name="ameblo">{{Cite web|url=http://ameblo.jp/sakuragakuin/entry-11772092273.html|title=待ってる時間に読んでください|accessdate=2015-08-19|date=2014-02-15|work=[[アメーバブログ|アメーバブログ]]|publisher=[[さくら学院|さくら学院]] 菊地最愛}}</ref>。她曾以為松子DELUXE是正在懷孕的女性，向友人道：「她根本都不會產寶寶呢（ぜんぜん赤ちゃんが産まれないね）。」而被友人訕笑，為她在15歲時一生最難為情的事<ref name="entamenext">{{Cite web|url=http://www.entamenext.com/news/detail/id=718|title=さくら学院 菊地最愛への100の質問【完全版】|accessdate=2015-06-30|date=2014年7月1日|publisher=月刊エンタメ [[徳間書店|徳間書店]]}}</ref>。

菊地喜歡的偶像為[[℃-ute|℃-ute]]的[[鈴木愛理|鈴木愛理]]，第一次買的唱片是月島きらり starring 久住小春 (モーニング娘。)的《チャンス!》<ref name="entamenext">{{Cite web|url=http://www.entamenext.com/news/detail/id=718|title=さくら学院 菊地最愛への100の質問【完全版】|accessdate=2015-06-30|date=2014年7月1日|publisher=月刊エンタメ [[徳間書店|徳間書店]]}}</ref>。菊地加入櫻花學院之前就有「想當super moa醬」這想法<ref name="ameblo">{{Cite web|url=http://ameblo.jp/sakuragakuin/entry-11772092273.html|title=待ってる時間に読んでください|accessdate=2015-08-19|date=2014-02-15|work=[[アメーバブログ|アメーバブログ]]|publisher=[[さくら学院|さくら学院]] 菊地最愛}}</ref>、「super moa醬」的意思是像動畫《[[偶像宣言|偶像宣言]]》的主角月島星璃那樣的女孩<ref name="oricon1">{{Cite web|url=http://www.oricon.co.jp/news/2050736/full/|title=さくら学院の卒業メンバーが学院生活と今後について語る|accessdate=2015-08-14|date=2015-03-29|work=ORICON STYLE|publisher=[[オリコン|オリコン]]株式会社}}</ref>。另外，她亦提及憧憬的女性名人為[[柴咲幸|柴咲幸]]<ref name="entamenext">{{Cite web|url=http://www.entamenext.com/news/detail/id=718|title=さくら学院 菊地最愛への100の質問【完全版】|accessdate=2015-06-30|date=2014年7月1日|publisher=月刊エンタメ [[徳間書店|徳間書店]]}}</ref>。

菊地喜歡的動畫為《[[LoveLive!|LoveLive!]]》、《[[冰雪奇缘|冰雪奇緣]]》、《[[风之谷|風之谷]]》<ref name="entamenext">{{Cite web|url=http://www.entamenext.com/news/detail/id=718|title=さくら学院 菊地最愛への100の質問【完全版】|accessdate=2015-06-30|date=2014年7月1日|publisher=月刊エンタメ [[徳間書店|徳間書店]]}}</ref>。尊敬的人為自己的母親；《[[LoveLive!|LoveLive!]]》中的高坂穂乃果、東條希；《[[絕園的暴風雨|絕園的暴風雨]]》中的不破愛花<ref name="entamenext">{{Cite web|url=http://www.entamenext.com/news/detail/id=718|title=さくら学院 菊地最愛への100の質問【完全版】|accessdate=2015-06-30|date=2014年7月1日|publisher=月刊エンタメ [[徳間書店|徳間書店]]}}</ref>。喜歡的電影有《[[劇場版_魔法少女小圓|劇場版 魔法少女小圓]]》、《[[查理和巧克力工厂|查理和巧克力工廠]]》<ref name="entamenext">{{Cite web|url=http://www.entamenext.com/news/detail/id=718|title=さくら学院 菊地最愛への100の質問【完全版】|accessdate=2015-06-30|date=2014年7月1日|publisher=月刊エンタメ [[徳間書店|徳間書店]]}}</ref>。

在櫻花學院中喜歡的歌曲為《[[朝著夢想_/_Hello_!_IVY|朝著夢想]]》《マシュマロ色の君と》《未完成シルエット》<ref name="entamenext">{{Cite web|url=http://www.entamenext.com/news/detail/id=718|title=さくら学院 菊地最愛への100の質問【完全版】|accessdate=2015-06-30|date=2014年7月1日|publisher=月刊エンタメ [[徳間書店|徳間書店]]}}</ref>。
