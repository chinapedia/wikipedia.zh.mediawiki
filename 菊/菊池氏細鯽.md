'''菊池氏細鯽'''，又稱'''台灣細鯽'''，俗名馬達卡、田溝魚、瘦魚，為[[輻鰭魚綱|輻鰭魚綱]][[鯉形目|鯉形目]][[鯉科|鯉科]][[細鯽屬|細鯽屬]]的其中一個[[種|種]]。

== 分布 ==
本魚僅分布於[[台灣|台灣]]北部及東部的溪流，為特有種。

==深度==
水深0－5公尺。

== 特徵 ==
本魚體延長而側扁，下頷略突出，無鬚，口裂大。體色土黃色，背部具微黃綠色金屬光澤，體側具一條暗藍色縱帶，特徵是側線不完全且極短。鱗片大；背鰭、臀鰭及尾鰭呈淡黃色，尾鰭叉形，體長可達8公分。

== 生態 ==
本魚為初級淡水魚，性活潑且善跳躍，喜棲息於在較緩流且水生[[植物|植物]]茂盛的河溝、池沼或水渠中。屬雜食性，以掉落水面之[[昆蟲|昆蟲]]和[[藻類|藻類]]為食。

== 經濟利用 ==
可食用，適合油炸，另可做為觀賞魚。
