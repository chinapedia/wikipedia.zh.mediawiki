'''維丁''' （[[保加利亞語|保加利亞語]]：'''Видин'''）是位於[[保加利亞|保加利亞]]西北部的一個城市。處[[多瑙河|多瑙河]]南岸。靠近[[塞爾維亞|塞爾維亞]]與[[羅馬尼亞|羅馬尼亞]]兩國邊境。維丁是[[維丁州|維丁州]]的首府所在地。2011年2月，維丁有人口47,138人<ref name="cities census 2011">{{bg icon}}[http://www.nsi.bg/census2011/pagebg2.php?p2=36&sp2=37&SSPP2=45 National Statistical Institute - Main Towns Census 2011] {{webarchive|url=https://web.archive.org/web/20110408064046/http://www.nsi.bg/census2011/pagebg2.php?p2=36&sp2=37&SSPP2=45 |date=2011-04-08 }}</ref>。維丁是一個農業和貿易中心，並且以葡萄酒著稱。[[南極|南極]][[南設德蘭群島|南設德蘭群島]]的[[利文斯頓島|利文斯頓島]]上的[[維丁高地|維丁高地]]就得名自維丁。

== 地理和人口 ==
維丁位於保加利亞的西北部，是保加利亞在[[多瑙河|多瑙河]]沿岸的最重要的港口。其對岸是羅馬尼亞城市[[卡拉法特|卡拉法特]]，兩者之間有渡船連接。兩都市之間的距離約有2公里。

維丁及其以北居住有[[阿羅馬尼亞人|弗拉赫人]]。維丁在古羅馬尼亞語中的名稱是Diiu。

== 歷史 ==
在維丁近郊有名為'''杜諾尼亞'''（Dunonia）的古[[凱爾特人|凱爾特人]]的集落。後古[[羅馬人|羅馬人]]在這裡建造了要塞化都市'''波諾尼亞'''（Bononia）。城市位於保加利亞西北部和塞爾維亞東部之間，直至46年為止，都是[[默西亞_(羅馬行省)|默西亞]]的重要據點。

在[[斯拉夫人|斯拉夫人]]開始在這一地區居住的時期，斯拉夫人將這裡稱為'''比丁'''（Badin）或者'''布丁'''（Bdin）。現在的城市名也是由來於此。

維丁主要的文化遺產是建於10世紀至14世紀期間的[[巴巴維達|巴巴維達]]要塞。在[[中世紀|中世紀]]，維丁是保加利亞的重要都市，這裡設有主教座，是地區的中心都市。在971年至976年期間，維丁是[[保加利亞第一帝國|保加利亞第一帝國]]皇帝[[薩穆伊爾_(保加利亞皇帝)|薩穆伊爾]]所領的中心城市（薩穆伊爾的兄弟統治南部地區）。1003年，在經過了8個月的包圍之後，維丁被[[東羅馬帝國|拜占庭帝國]]的[[巴西爾二世|巴西爾二世]]征服。維丁在[[保加利亞第二帝國|保加利亞第二帝國]]的勃興期再次成為重要城市。城市的統治者（{{Lang|bg|Деспот}} / ''Despot''）是帝国的重要人物，有多位皇帝都出身於此。13世紀中期，城市由希希曼家（{{lang|bg|Шишмановци}} / ''Shishmanovtsi''）統治。

奧斯曼帝國的支配區域擴大至[[多布羅加|多布羅加]]、[[普里萊普|普里萊普]]、[[丘斯滕迪爾|維爾巴扎德]]（丘斯滕迪爾），維丁的自治獨立並未持續太長時間。在1396年的[[尼科波利斯戰役|尼科波利斯戰役]]中，[[伊萬·斯拉齊米爾|伊萬·斯拉齊米爾]]（{{lang|bg|[[:bg:Иван_Срацимир|Иван Срацимир]]}} / ''Ivan Sratsimir''）集結兵力幫助斯拉夫人。然而在[[尼科波利斯|尼科波利斯]]敗于奧斯曼軍隊，維丁完全被[[巴耶濟德一世|巴耶濟德一世]]率領的奧斯曼帝國統治。

在奧斯曼帝国統治時代的晚期，維丁是反抗奧斯曼帝國的土耳其人[[奧斯曼·帕茲萬特奧盧|奧斯曼·帕茲萬特奧盧]]（{{Lang|tr|[[:en:Osman_Pazvantoğlu|Osman Pazvantoğlu]]}}）所創建的國家的中心都市。

1850年，因農民對地主階層的壓迫日益不滿，加上歐洲「[[1848年革命|1848年革命]]」的影響，維丁爆發了大規模的叛亂。

1885年的[[塞爾維亞-保加利亞戰爭|塞爾維亞-保加利亞戰爭]]期間，維丁曾被塞爾維亞軍隊支配。塞爾維亞軍隊雖然有數量優勢，保加利亞軍隊仍在後來戰勝了塞爾維亞軍隊。

== 觀光 ==
維丁有兩處保存狀態良好，建於中世紀時期的要塞，分別是[[巴巴維達|巴巴維達]]和加勒托，此外還有眾多頗具歷史的[[東正教|東正教]]的教堂。如[[聖龐塔萊翁|聖龐塔萊翁]]教堂、佩托卡教堂（均建于17世紀）、聖德米特里教堂（19世紀）。還有[[猶太教|猶太教]]的[[猶太會堂|猶太會堂]]（1894年）、[[清真寺|清真寺]]、18世紀後期土耳其人統治者奧斯曼·帕茲萬特奧盧的書庫、1798年的十字型兵舍等古建築物。許多古跡都得到了修復。

== 維丁廣播電台 ==
在1973年，維丁郊區建設了一處中波廣播電台。其電波可覆蓋[[歐洲|歐洲]]全境。頻率是576千赫及1224千赫。出力是500千瓦。為送出576千赫的電波，建設了高259米的帶支架的信號塔。下部有筐型的天線。而送出1224千赫的信號塔則帶有4個支架，並不直接接觸地面，支架上設有筐型天線。
