'''維壹'''（{{lang|en|'''Harbour One'''}}），是位於[[香港|香港]][[香港島|香港島]][[石塘咀|石塘咀]][[德輔道西|德輔道西]]的單幢住宅，樓高共43層，由[[英皇集團（國際）|英皇地產]]發展，於2012年第三季入伙。

== 歷史 ==
維壹前身為一於1957年建成的七層高唐樓——美景大廈及德富工業大廈，先於2006年，[[田生集團|田生集團]]以2.34億收購美景大廈<ref>[http://www.richfieldgroup.hk/redevelopment.php 田生集團網頁] {{webarchive|url=https://web.archive.org/web/20100125161041/http://www.richfieldgroup.hk/redevelopment.php |date=2010-01-25 }}</ref>，及後[[田生集團|田生集團]]再以1.97億元收購鄰近的德富工業大廈，後[[英皇集團（國際）|英皇地產]]購入地盤，而其曾考慮把其發展成[[酒店|酒店]]，其後改成發展37層高住宅項目。<ref>[http://www.capital-weekly.com/common_emag/view_content.php?issue_no=234&pub_id=26&top_id=46&art_id=106590 西環樓重建發展商賺到笑]</ref>

== 簡介 ==
住宅由於望向[[維多利亞港|維多利亞港]]及位於名校網，故發展商把其包裝成[[豪宅|豪宅]]，開售呎價由$15,000起。樓宇共高37層，其中1至6樓為[[停車場|停車場]]、7樓及32樓則為[[空中花園|空中花園]]、而8樓為[[會所|會所]]，其設施包括室內泳池、健身室、宴會廳等。

另外，41及42樓為特色單位，只為1梯1伙，其中41樓設有平台，而42樓則包括[[天台|天台]]。

== 事件 ==
2010年9月18日下午5時，維壹地盤發生[[工業意外|工業意外]]，一名56歲[[紮鐵|紮鐵]][[工人|工人]]由高處下墮，證實死亡。[http://www.881903.com/page/zh-tw/newsdetail.aspx?source=tbar&itemid=281563&csid=261_1780]
