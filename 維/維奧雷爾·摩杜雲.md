'''維奧雷爾·迪努·摩杜雲'''（'''{{lang-ro|Viorel Dinu Moldovan}}'''，{{bd|1972年|8月8日|catIdx=Moldovan ,Viorel}}<ref>{{en}}[http://www.transfermarkt.co.uk/en/trainer/8724/viorel-moldovan/aufeinenblick.html transfermarkt profile] {{webarchive|url=https://web.archive.org/web/20090611000129/http://www.transfermarkt.co.uk/en/trainer/8724/viorel-moldovan/aufeinenblick.html |date=2009-06-11 }}</ref>），簡稱'''維奧雷爾·摩杜雲'''（'''{{lang-ro|Viorel Moldovan}}'''），出生在[[比斯特里察|比斯特里察]]，是一名[[羅馬尼亞|羅馬尼亞]]足球領隊及前足球員，司職[[足球前鋒|前鋒]]。他是1990年代[[羅馬尼亞國家足球隊|羅馬尼亞]]的關鍵球員之一。 

== 生平 ==
摩杜雲先後效力過[[比斯特里察哥利亞|比斯特里察哥利亞]]、[[布加勒斯特戴拿模足球會|布加勒斯特戴拿模]]、[[納沙泰爾足球會|納沙泰爾]]、[[蘇黎世草蜢足球會|草蜢]]、[[高雲地利足球會|高雲地利]]、[[費倫巴治體育會|費倫巴治]]、[[南特足球會|南特]]、[[艾華達足球會_(阿布札比)|艾華達]]、[[塞維特足球會|塞維特]]、[[添美索亞拿足球會|添美索亞拿]]和[[布格勒斯特迅速足球會|布格勒斯特迅速]]。

他足球生涯最成功的莫過於在1996年至1998年在[[瑞士|瑞士]]效力球會[[納沙泰爾足球會|納沙泰爾]]和[[蘇黎世草蜢足球會|草蜢]]，他在1996和1997年連續兩年成為[[瑞士足球超級聯賽|瑞超]]神射手。他在1998年至2000年效力[[費倫巴治體育會|費倫巴治]]和在2000年至2004年間效力[[南特足球會|南特]]時亦使人難忘。2001年，他作為一名決定性球員，協助南特奪得[[法國足球甲級聯賽|法甲]]冠軍。他被外借至[[英格蘭|英格蘭]]球會[[高雲地利足球會|高雲地利]]期間，他在[[英格蘭足總盃|足總盃]]對[[阿士東維拉足球會|阿士東維拉]]的賽事中射入1球<ref>{{en}}{{cite web

== 國際賽 ==
摩杜雲代表[[羅馬尼亞國家足球隊|羅馬尼亞]]上陣70場賽事並攻入25球。他代表羅馬尼亞隊參與了[[1994年世界盃|1994年世界盃]]、[[1996年歐洲國家盃|1996年歐洲國家盃]]、在[[1998年世界盃|1998年世界盃]]首回合分別對[[英格蘭足球代表隊|英格蘭]]和[[突尼西亞國家足球隊|突尼西亞]]的賽事中取得入球和[[2000年歐洲國家盃|2000年歐洲國家盃]]。

== 領隊 ==
摩杜雲曾擔任[[烏尼雷亞足球會|烏尼雷亞]]的運動總監及[[瓦斯路易足球會|瓦斯路易]]的領隊。2009年5月26日，他由於未能帶領球隊取得歐洲賽資格而被解僱，並由助教{{Link-en|基斯坦·杜爾卡|Cristian Dulca}}暫代職務直至新領隊上任<ref>{{ro icon}}[http://www.gsp.ro/index.php?section=articol&screen=index&id=139989&rss=1 Porumboiu: "Despărţirea de Viorel Moldovan s-a făcut pe cale amiabilă"]</ref>。7月28日，意大利領隊{{Link-en|尼科洛·拿玻里|Nicolò Napoli}}離開[[巴索夫足球會|巴索夫]]，他的職位由摩杜雲所取代，他與會方簽訂兩年合約<ref>{{ro icon}}[http://www.cotidianul.ro/fc_brasov_napoli_inlocuit_de_viorel_moldovan_in_scaunul_de_antrenor-93430.html FC Braşov: Napoli înlocuit de Viorel Moldovan în scaunul de antrenor] {{webarchive|url=https://web.archive.org/web/20090730010322/http://www.cotidianul.ro/fc_brasov_napoli_inlocuit_de_viorel_moldovan_in_scaunul_de_antrenor-93430.html |date=2009-07-30 }}</ref>。
