'''海牙和平会议'''，亦称'''海牙会议'''，中文中也称'''世界保和大会'''或'''万国和平会议'''。第一次会议于1899年5月18日到6月29日举行，参加的有[[清朝|中]]、[[俄羅斯帝國|俄]]、[[英国|英]]、[[法蘭西第三共和國|法]]、[[美国|美]]、[[大日本帝國|日]]等二十六国。第二次会议于1907年6月15日到10月18日举行，参加的有四十四国，包括第一次会议的全体参加国。两次会议通过的十三个[[公约|公约]]和三个[[宣言|宣言]]，合称“[[海牙公约|海牙公约]]”。

==第一次海牙和平会议==
第一次海牙和平會議由俄國沙皇[[尼古拉二世|尼古拉二世]]於1898年8月29日提出，於1899年5月18日在荷兰海牙的[[豪斯登堡宫|豪斯登堡宫]]正式舉行，並於同年7月29日簽署，1900年9月4日生效。 1899年“海牙公約”由四個主要部分和三個附加聲明（最終的主要部分是由於某種原因，相同的第一個附加聲明）組成：

第一，和平解決國際爭端
第二，陸戰法規和海關
第三，採用1864年日內瓦公約內的海戰原則
第四，禁止使用氣球彈和炸藥

“公約”的主要作用是禁止現代技術在戰爭中使用。另外，“公約”還設立了常設仲裁法院。

==第二次海牙和平会议==
在1907年舉行的第二次會議，因為幾個重大決策導致失敗。然而，參與會議的主要列強也嘗試了在20世紀成功的國際合作。

第二次會議由西奧多羅斯福總統在1904年提出，但因為日俄戰爭而推遲。第二次和平會議在1907年6月15日至1907年10月18日舉行，它擴大原有的海牙公約，修改了一部分條約，並加入了新成員，對海戰更加注重。英國試圖以安全為由限制軍備，但以德國為首的其他列強，擔心英國試圖阻止德國艦隊的增長。德國也拒絕強制仲裁的建議。但是，會議卻擴大了自願仲裁機制，訂立公約規範收集債務，戰爭規則，以及中立國的權利和義務。

最終，協議於1907年10月18日簽署，並在1910年1月26日生效。它由13個部分，其中12人是批准和生效：

# 和平解決國際爭端
# 合同債務的限制
# 展開敵對行動
# 陸戰法規和海關
# 在中立國戰爭時的權利與義務
# 敵人的商船在敵對行動的爆發狀態
# 商船轉換成戰爭船
# 自動潛艇聯繫地雷的鋪設
# 在戰時海軍轟擊
# 採用“日內瓦公約”的海戰原則
# 與捕獲權的行使方面的某些限制，在海軍戰爭
# 創造一個國際獎法院尚未批准
# 在海軍戰爭中立國的權利和職責
