'''海啸'''是一种具有强大破坏力的[[海浪|海浪]]。当[[地震|地震]]发生於海底，因震波的动力而引起海水剧烈的起伏，形成强大的波浪，向前推进，将[[沿海|沿海]]地带一一淹没的自然現象，称之为海啸。

== 詞源 ==
[[File:_Tsunami-kueste.01.vm.jpg|thumb]]
[[File:_2004_Indian_Ocean_earthquake_Maldives_tsunami_wave.jpg|300px]]]]

'''海啸'''在許多西方語言中稱為“'''tsunami'''”，詞源自[[日語|日語]]“津波”，即「港邊的波浪」（「津」即「[[港口|港]]」）。这也显示出了日本是一个经常遭受海啸袭击的国家。汉字又称'''海溢'''。“'''tsunami'''”一詞，在1963年的國際科學會議上正式列入國際術語。

中國最早在漢朝就已有對海嘯的紀錄，在中國古書記載的'''海溢'''、'''海潮溢'''、'''海吼'''、'''海唑'''、'''海沸'''都是指海嘯<ref>[http://www.agri-history.net/news/0501211.htm 中国古代的海啸灾害]</ref>。

== 特性 ==
這是一種波長極長的[[重力波_(流體力學)|重力波]]。若是發生於在近岸淺水海域，波浪因深度漸淺而波高驟增，可造成嚴重災害。此波於大洋中傳遞極快，每秒速度可超過二百公尺。在地震發生處的波動可能只有0.1公尺到5公尺的高度，波長卻可能是1500公里。隨著水波傳到岸邊波長會縮短到5公里，高度卻可達到50公尺。

=== 原因 ===
[[File:Propagation_du_tsunami_en_profondeur_variable.gif|thumb]]
海啸通常由震源在海底下500公里以内、规模6.0以上的[[海底地震|海底地震]]引起。海啸[[波长|波长]]比[[海洋|海洋]]的最大深度还要大，在海底附近傳播也没受多大阻滞，不管海洋深度如何，[[波|波]]都可以传播过去，海嘯在海洋的傳播速度大約是每小時五百到一千公里，而相鄰兩個浪頭的距離也可能遠達500到650[[公里|公里]]，當海嘯波進入陸坡後，由於深度變淺，波高突然增大，它的这种波浪运动所卷起的海涛，波高可达数十米，并形成“水墙”。由地震引起的波動與海面上的海浪不同，一般海浪只在一定深度的水層波動，而地震所引起的水體波動是從海面到海底整個水層的起伏。

此外，海底火山爆发，土崩及人為的'''水底核爆'''，或者是[[隕石|隕石]]撞擊都會造成海嘯，“水墙”可達百尺。而且隕石造成的海嘯在任何水域也有機會發生，不一定在[[地震帶|地震帶]]。不過隕石造成的海嘯可能千年才會發生一次。海啸等自然灾害都会产生[[次声波|次声波]]，[[大象|大象]]可以听到次声波，對遠處發生的火山地震都會有反應，{{Fact|例如[[2004年印度洋大地震|2004年印度洋大地震]]产生的海啸，由于大象听到海啸产生的次声波，不听主人指挥，快速离开现场，乘坐大象的[[游客|游客]]才得以生还。|time=2011-03-12T06:01:37+00:00}}

近地海嘯所引起的海潮暴漲通常伴隨著地震，遠洋地震所引起的海嘯則不會感受到地震動。史籍上若提到大地震，之後海潮暴漲，則海嘯的可能性極大。然而多數記載只提到海潮暴漲，並無其他說明，是否是地震海嘯就有待進一步探討。因此將史籍上所述之海嘯或疑海嘯，以海嘯信度表示其發生之可能性。
[[File:2004-tsunami.jpg|thumb]]]]
當海嘯從較寬廣較深的海域傳到海岸時，則會變形。當海水傳到岸邊時，因為水深變淺，所以波浪的傳遞速度變慢。當前一波海浪的速度變慢後，後一波因為速度未降追了上來，所以變成波高變高。所以即便在深水區不高的波浪，到了岸邊波高卻會增加許多。

當海嘯到達海岸時，看來很像加速版的潮汐的起落。如果波高太高時波浪則會碎掉，或是可以看到很高的水牆。不過海嘯一般很少在岸邊成為如塔般很高的水牆，或是看到明顯的破浪，因為有時波浪是在離岸較遠處就已破碎了。另外海嘯來襲時波浪若進入淺水海灣或河流出海口，也可能看到類似階梯狀波浪的湧潮（bore）出現。這些都會造成海岸邊的海水高度升高。若震央較近海岸時，甚至有觀測到升高相當於十層樓高的。

由於熱帶氣旋吹襲沿岸地區時，風暴中心氣壓低把海水吸起，亦會引發海水暴漲、巨浪隨強風湧上陸地，造成嚴重傷亡及廣泛破壞，因此亦有不少人以海嘯作形容，甚或與海嘯混淆。但事實上由熱帶氣旋引發的海水上漲稱為[[風暴潮|風暴潮]]。

==强度或大小的等级==
与地震一样，有过多次尝试建立衡量海啸强度或大小的等级，使得不同事件之间可以比较。<ref name="Gusiakov">{{cite web |author = Gusiakov V. |title = Tsunami Quantification: how we measure the overall size of tsunami (Review of tsunami intensity and magnitude scales) |url = http://www.ngdc.noaa.gov/hazard/data/presentations/jtc/gusiakov.pdf |year= |accessdate=2009-10-18}}</ref>

===强度等级===
最早的被日常的用来衡量海啸强度等级方案是用于[[地中海|地中海]]的''西伯格－安布拉塞斯等级（Sieberg-Ambraseys scale）''<ref name="iem">{{cite web |title = 烈度等级表 |url = http://www.tsunami.net.cn/hxdengji-liedu.htm |publisher=中国地震局工程力学研究所 |accessdate=2013-07-27}}</ref>和用于太平洋的“今村－饭田强度分级（Imamura-Iida intensity scale）”<ref name=ifeng2010>{{cite web |title = 名词解释：海啸——海啸的规模 |url = http://news.ifeng.com/world/special/zhilidizhen/ziliao/detail_2010_02/27/1193310_1.shtml |publisher=凤凰网 |date=2010-02-27 |accessdate=2013-07-27}}</ref>。索洛维耶夫又修改了今村－饭田分级，根据如下公式计算了海啸强度''I''级：

:<math>\,\mathit{I} = \frac{1}{2} + \log_{2} \mathit{H}_{av}</math>

这里<math>\mathit{H}_{av}</math>是沿最近的海岸的平均海浪高度。这个分级方案也被称为“索洛维耶夫-今村海啸强度等级（Soloviev-Imamura tsunami intensity scale）”，并被用于NGDC/NOAA所编撰的全球海啸目录中。<ref>[http://ngdc.noaa.gov/hazard/tsu_db.shtml National Geophysical Data Center / (NGDC/WDS) Global Historical Tsunami Database]</ref>新西伯利亚海啸实验室（Novosibirsk Tsunami Laboratory）也使用这个分级作为海啸大小的主要参数。

===大小等级===
第一个真正的计算给定地点的海啸大小而不是强度的等级方案是由穆蒂和卢米斯（Murty & Loomis）提出的ML分级（ML scale）。这个分级是基于潜在能量的。<ref name=Gusiakov/>但因为计算海啸的潜在能量很困难，所以该分级很少被使用。阿部引入了“海啸大小等级<math>\mathit{M}_{t}</math>”，其计算公式为：

:<math>\,\mathit{M}_{t} = {a} \log h + {b} \log R = \mathit{D}</math>

这里''h''是最大海啸波幅（单位：米，m），由距离震中距离 为''R''的验潮仪测得；''a''，''b''和''D''是用于使the M<sub>t</sub>等级尽可能匹配矩震级（moment magnitude scale）的常数。<ref>{{Cite book|author=Abe K.|year =1995|title=Estimate of Tsunami Run-up Heights from Earthquake Magnitudes|work=Tsunami: progress in prediction, disaster prevention, and warning|accessdate=2009-10-18|url=http://books.google.com/?id=5YjaGdQOJIwC&pg=PA21&dq=abe+magnitude+scale+tsunami+1981&q=abe%20magnitude%20scale%20tsunami%201981|isbn=978-0-7923-3483-5}}</ref>

===  美国西海岸·阿拉斯加海啸警报中心 ===
美国西海岸·阿拉斯加海啸警报中心（WCATWC；暂译）中，将美国及其周边地区依照海岸地形等要素划分为11个区域。各区域分别有2种（或1种乃至没有）TIS（Tsunami Information Statement，海啸情报）、3种（或1种）Warning（警报），共1~5级警报。

=== 日本 ===
<!--[[ファイル:Numazu_port_gate.jpg|thumb]]）]]-->

[[氣象廳_(日本)|日本气象厅]]将24小时监视造成海啸的地震活动，并于地震发生后最快2分钟内发布海啸预报·警报（[[海啸预报|海啸预报]]·[[海啸注意报|海啸注意报]]、[[海啸警报|海啸警报]]、[[海啸警报|大海啸警报]]）。即使是震源较远但规模较大的地震或震度较小的地震也可能会发布海啸警报。为了缩减发布警报所需要的时间，也进行了将[[地震计|地震计]]更换为性能更高的仪器或者装设海底地震计的工作。

原本发布海啸警报需要3分钟左右的时间，由于2007年10月开始，对日本近海的地震启用[[紧急地震速报|紧急地震速报]]系统，因此发布警报需要的时间得以缩减，最快可在2分内发布海啸警报（如2007年3月的[[能登半岛地震|能登半岛地震]]等）。

此外，发表海啸警报的时候，电视台、广播电台等会进行[[紧急警报广播|紧急警报广播]]。

根据震源的位置、震级、断层数据，可计算预测海啸是否发生及高度等。气象厅拥有一个数据库，可以根据震源要素进行检索，并决定发布何种警报信息。

=== 香港 ===
[[香港天文台|香港天文台]]制定了海嘯警報，如預料南海或太平洋發生的強烈地震會引發海嘯，導致香港受顯著海嘯（即海嘯高度比正常水位高出0.5米以上）影響，而預計海嘯會在三小時內抵達香港，天文台會發出海嘯警告；對於有可能影響香港但在三小時後才抵達香港的顯著海嘯，天文台會首先發出海嘯報告通知市民。此外，如香港可能受海嘯影響但預料香港的海嘯不顯著，天文台亦會發出海嘯報告通知市民。<ref>{{Cite web|url=http://www.weather.gov.hk/gts/equake/tsunami_mon_c.htm|title=香港的海嘯監測|accessdate=2017-11-22|author=香港天文台|date=2014-02-24}}</ref>

=== 中国大陆 ===
中国大陆于2004年开始设立海啸预警，预警分为蓝、黄、橙、红四个等级（或用罗马数字排列IV、III、II、I），具体标准如下<ref>{{cite web |url=http://www.nmefc.cn/nr/cont.aspx?itemid=4&id=1684 |title=海啸灾害概述|website=国家海洋环境预报中心 }}</ref>
*海啸Ⅳ级警报（蓝色） ：受海啸影响，预计沿岸验潮站出现小于1米（正常潮位以上，下同）海啸波高、受灾地区轻微损失时，发布Ⅳ级海啸警报。
*海啸Ⅲ级警报（黄色）： 受海啸影响，预计沿岸验潮站出现1-2米海啸波高、受灾地区发生房屋、船只等受损时，发布Ⅲ级海啸警报。 
*海啸Ⅱ级警报（橙色）： 受海啸影响，预计沿岸验潮站出现2-3米海啸波高、局部岸段严重受损、危及生命财产时，发布Ⅱ级海啸警报。
*海啸Ⅰ级警报（红色）：受海啸影响，预计沿岸验潮站出现3米以上海啸波高、300公里以上岸段严重受损、危及生命财产时，发布Ⅰ级海啸警报。 

[[2011年日本东北地方太平洋近海地震|2011年日本东北地方太平洋近海地震]]发生后，国家海洋预报台一度对东南地区发布海啸蓝色警报，这也是中国大陆目前唯一的一次发布海啸警报。<ref>{{cite news |url=https://news.qq.com/a/20110311/002415.htm |title=中国国家海洋预报台发布海啸蓝色警报 |website=腾讯新闻 |date=2011-3-11}}</ref>

2015年，修改后的《风暴潮、海浪、海啸和海冰灾害应急预案》发布，对海啸警报标准进行调整：
*海啸信息（海啸警报消息）：受地震或其他因素影响，预计海啸波将会在中国沿岸产生0.3米以下的海啸波幅，或者没有海啸。
*海啸黄色警报：受地震或其他因素影响，预计海啸波将会在中国沿岸产生0.3（含）－1米的海啸波幅时。
*海啸橙色警报：受地震或其他因素影响，预计海啸将会在中国沿岸产生1（含）－3米的海啸波幅时。
*海啸红色警报：受地震或其他因素影响，预计海啸波将在中国沿岸产生3（含）米以上的海啸波幅时。

== 海嘯對[[臺灣|臺灣]]影響 ==

雖然臺灣亦處[[環太平洋火山帶|環太平洋火山帶]]，不過和[[日本|日本]]不同，臺灣很少有海嘯侵襲。主要原因是海嘯多來自太平洋的海底地震，會從臺灣東部靠近，而在臺灣東部的海底，[[菲律賓海板塊|菲律賓海板塊]]和[[歐亞板塊|歐亞板塊]]交界處，海底地形非常陡峭，容易使波浪受到折射而遠離，不利海嘯成形<ref>[http://sa.ylib.com/news/newsshow.asp?FDocNo=635&CL=32 臺灣會不會發生大海嘯？]《[[科學人|科學人]]》2005年2月號</ref>。2011年日本東北大地震（海底地震）臺灣僅觀測到10公分潮差，而1960年智利大地震所引發的海嘯對於臺灣也沒有造成重大災害。

一般認為臺灣最可能發生海嘯的地方是[[基隆市|基隆]]到[[宜蘭縣|宜蘭]]一帶沿海，在過去便曾有發生之實例。1867年12月18日，[[雞籠|雞籠]]（今基隆）附近發生有感地震，震央約在雞籠嶼（今[[基隆嶼|基隆嶼]]）東方500公尺左右的海底，造成雞籠港（今[[基隆港|基隆港]]）港內的海水急速往外海退去，接著海嘯撲向[[瑞芳區|瑞芳]]、[[萬里區|萬里]]、金包里（今[[金山區_(新北市)|金山]]）一帶沿海發生災情，史稱「[[1867年基隆海嘯|雞籠海嘯]]」<ref>{{cite book|author=徐泓|title=清代台灣天然災害史料彙編 |year=1983年 |publisher=行政院國家防災科技研究報告 |location=臺北 |isbn= }}</ref>。

雖然南中國海發生大規模地震機率非常低，萬一[[中国南海|中国南海]]一帶的地方出現大地震，勢必會對臺灣西南沿海、中國大陸東南沿海、[[中南半島|中南半島]]、[[菲律賓|菲律賓]]等地造成海嘯威脅。不過，臺灣仍有學者提出[[馬尼拉海溝|馬尼拉海溝]]恐有潛在危機<ref>[http://www.ncu.edu.tw/~ncu7009/news/clip/read/2516 中大研究 馬尼拉海溝若地震　大海嘯直衝臺灣西南] {{webarchive|url=https://web.archive.org/web/20110316213914/http://www.ncu.edu.tw/~ncu7009/news/clip/read/2516 |date=2011-03-16 }}</ref>。

== 紀錄 ==
全球的海嘯發生區大致與地震帶一致。全球有記載的破壞性海嘯大約有260次左右，平均大約6、7年發生一次。發生在[[環太平洋地區|環太平洋地區]]的地震海嘯就佔了約80%。而[[日本|日本]]列島及附近海域的地震又佔太平洋地震海嘯的60%左右，日本是全球發生地震海嘯且受害最深的國家。

近年來較大規模的海嘯有：
* 2011年3月11日於[[日本|日本]][[宮城縣|宮城縣]][[仙台市|仙台市]]以東的[[太平洋|太平洋]]海域發生[[矩震级|M<sub>w</sub>]]9.1级[[地震|地震]]，引發海嘯。參見[[2011年東北地方太平洋近海地震|2011年東北地方太平洋近海地震]]
* 2010年2月27日於[[智利|智利]][[比奧比奧|比奧比奧]]沿岸附近陸地發生[[矩震级|M<sub>w</sub>]]8.8级[[地震|地震]]，隨後引發海嘯襲擊環太平洋沿岸國家。參見[[2010年智利大地震|2010年智利大地震]]
* 2004年12月26日於[[印尼|印尼]]的[[蘇門達臘|蘇門達臘]]外海發生[[矩震级|M<sub>w</sub>]]9.3级海底[[地震|地震]]。海嘯襲擊[[斯里蘭卡|斯里蘭卡]]、[[印度|印度]]、[[泰國|泰國]]、[[印尼|印尼]]、[[馬來西亞|馬來西亞]]、[[孟加拉|孟加拉]]、[[马尔代夫|馬爾代夫]]、[[缅甸|缅甸]]和[[非洲|非洲]]東岸等[[國家|國家]]，造成30餘萬人喪生。准确[[死亡|死亡]]数字已无法统计。參見[[2004年印度洋大地震|2004年印度洋大地震]]。
* 1998年7月兩個规模为7.0级的海底[[地震|地震]]，造成[[巴布亞紐幾內亞|巴布亞紐幾內亞]]約2100人喪生。
* 1992年9月[[尼加拉瓜|尼加拉瓜]]發生海嘯。
* 1883年8月27日[[荷属东印度群岛|荷属东印度群岛]]上[[喀拉喀托火山|喀拉喀托火山]]爆發，引起的海嘯使約36,000人[[死亡|死亡]]。
