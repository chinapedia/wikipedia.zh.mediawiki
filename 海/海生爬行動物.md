'''海生爬行動物'''是群再次適應水生環境或半水生環境的[[爬行動物|爬行動物]]。

最早的海生爬行動物是出現在[[古生代|古生代]][[二疊紀|二疊紀]]的[[中龍類|中龍類]]。在[[中生代|中生代]]期間，許多爬蟲類生物群適應海中生活，包括許多相似的演化支：[[魚龍類|魚龍類]]、[[蛇頸龍類|蛇頸龍類]]、[[滄龍類|滄龍類]]、[[幻龍類|幻龍類]]、[[海龍類|海龍類]]、[[海鱷類|海鱷類]]、[[楯齒龍類|楯齒龍類]]、史前海龜。魚龍類、蛇頸龍類曾一度被歸類於「Enaliosauria」，這個分類目前已被遺棄不用<ref>Williston SW (1914) [http://books.google.dk/books?id=uB7YkzwYaswC&pg=PA75&lpg=PA75&dq=enaliosauria+ichthyosauria&source=bl&ots=Yeq9poqoWo&sig=S0liTPucK2TUueoR1_q7GX7DZ10&hl=da&ei=xmXkTPTnFozoObKQyJIB&sa=X&oi=book_result&ct=result&resnum=9&ved=0CFQQ6AEwCA#v=onepage&q=enaliosauria%20ichthyosauria&f=false ''Water Reptiles of the Past and Present''] University of Chicago Press (reprint 2002). ISBN 1-4021-4677-9</ref>。
==历史==
2.35亿年前，爬行动物于三叠纪中期进入水中，并统治海洋达1.7亿年。最早的大型海洋爬行动物是幻龍類；同一时期还有[[楯齒龍類|楯齒龍類]]。到了1亿年前，出现了蛇頸龍。其中[[上龙类|上龙类]]是蛇頸龍中体型最大的，长达12米。鱼龙也在这一时期出现，体长达15米。它们在9000万年前灭绝。沧龙则存活到6500万年前恐龙时代结束。<ref>{{cite book|author1=李基科|title=發現第二個地球《385天的地球2.0》|date=2015|publisher=丹陽文化|page=31}}</ref>

在[[白堊紀|白堊紀]]末的[[白堊紀-第三紀滅絕事件|大規模滅絕事件]]後，海生爬行動物數量減少。現存海生爬行動物包括：大型[[海龜|海龜]]、[[海蛇|海蛇]]、[[海鬣蜥|海鬣蜥]]、以及某些[[鱷魚|鱷魚]]物種。

== 現存物種 ==
* '''[[海龜|海龜]]'''：現存共有 7 種海龜，多半棲息於[[中美洲|中美洲]]、[[南美洲|南美洲]]與[[加勒比|加勒比]]地區的[[熱帶|熱帶]]沿岸海域。不過海龜遷徙的範圍路徑很廣，最北曾經在[[斯堪地那維亞|斯堪地那維亞]]發現。海龜為獨居動物，但在築巢期時也會群聚在一起。雖然僅有 7 種海龜為完全海生，許多種類的[[烏龜|烏龜]]棲息於[[汽水_(水域)|汽水]]水域中<ref name=":0">{{Cite journal|title = Marine Reptiles|url = https://dx.doi.org/10.1371/journal.pone.0027373|journal = PLoS ONE|date = 2011-11-08|pmc = 3210815|pmid = 22087300|pages = e27373|volume = 6|issue = 11|doi = 10.1371/journal.pone.0027373|first = Arne Redsted|last = Rasmussen|first2 = John C.|last2 = Murphy|first3 = Medy|last3 = Ompi|first4 = J. Whitfield|last4 = Gibbons|first5 = Peter|last5 = Uetz}}</ref><ref name=":1">{{Cite web|url = http://www.britannica.com/animal/sea-turtle|title = Sea Turtle|date = |accessdate = December 8, 2015|website = Encyclopædia Britannica Online|publisher = |last = Zug|first = George R.}}</ref>。
* '''[[海蛇|海蛇]]'''：現存共有超過 60 種的海蛇，棲息於[[印度洋|印度洋]]和[[太平洋|太平洋]]的[[熱帶|熱帶]]及[[亞熱帶|亞熱帶]]海域，有部分的目擊報告指出海蛇有可能也出沒於[[大西洋|大西洋]]。海蛇具有[[毒液|毒液]]，對人類可能致命，儘管大多時間牠們在反擊時僅會注入非致命的極少量毒液。海蛇和陸生[[蛇|蛇]]最大的差異在於牠們具有扁平的尾部<ref name=":0" /><ref>{{Cite web|url = http://www.britannica.com/animal/sea-snake|title = Sea Snake|date = |accessdate = December 8, 2015|website = Encyclopædia Britannica Online|publisher = |last = |first = }}</ref>。
* '''[[海鬣蜥|海鬣蜥]]'''：海鬣蜥僅棲息於[[加拉巴哥群島|加拉巴哥群島]]而且尚未完全適應海生生活。儘管海鬣蜥幾乎完全生活於海中並以{{link-en|海洋植物|Marine botany}}為食，牠們必須回到陸地上築巢以及曬太陽來維持體溫，此時的海鬣蜥十分容易受到陸地[[掠食者|掠食者]]的攻擊<ref name=":0" /><ref>{{Cite web|url = http://animals.nationalgeographic.com/animals/reptiles/marine-iguana/?source=A-to-Z|title = Marine Iguanas|date = |accessdate = December 8, 2015|website = National Geographic|publisher = |last = |first = }}</ref>。
* '''[[灣鱷|灣鱷]]與[[美洲鱷|美洲鱷]]'''：所有 23 種現存的鱷魚均非完全海生，不過[[灣鱷|灣鱷]]可以透過[[鹽腺|鹽腺]]來排出體內多餘的[[鹽分|鹽分]]，因此得以棲息在[[東南亞|東南亞]]及[[澳洲|澳洲]]的汽水水域中。灣鱷為現存最大型的鱷魚，同時也是現存最大型的爬蟲類，體長可達 {{convert|6|m|ftin|0|abbr=on}}<ref name=":0" /><ref name=":2">{{Cite web|url = http://animals.nationalgeographic.com/animals/reptiles/saltwater-crocodile/|title = Saltwater crocodile|date = |accessdate = December 8, 2015|website = National Geographic|publisher = |last = |first = }}</ref>。美洲鱷同樣也偏好棲息於汽水水域而非淡水之中<ref name="Ellis1981">{{cite journal|last1= Ellis|first1=T. M.|title= Tolerance of Sea Water by the American Crocodile, ''Crocodylus acutus''|journal= Journal of Herpetology|volume= 15|issue= 2|year= 1981|pages= 187|doi= 10.2307/1563379|jstor=1 563379}}</ref>。

==习性==
有些海生爬行動物，例如魚龍類與滄龍類，極少回到陸地上，出生在水裡。其他如海龜與[[鹹水鱷|鹹水鱷]]，仍然會到岸邊產卵。
