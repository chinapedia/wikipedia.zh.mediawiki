'''海禁'''（又稱'''洋禁'''），是一种[[鎖國政策|鎖國政策]]，旨在禁止民间私自出海，有謂「尺板不得出海」<ref>[[#山本|山本2002]] 第135頁、[[#檀上2005|檀上2005]] 第145頁</ref>，也限制外國商人前往本國通商。具体实施隨著时間變遷而有张有弛，即「严禁」及「弛禁」之分。{{Fact|海禁目的是為消滅反叛勢力、打擊[[海盜|海盜]]及[[走私|走私]]，保障社會穩定。}}然而，實際上真正受到打擊的是本國對外的貿易及[[漁業|漁業]]。[[海盜|海盜]]和[[走私|走私]]商人的活動亦反而因為海禁而更加猖獗，對本國及其周邊國家的社會和經濟發展產生了消極影響。

[[中国历史|中国历史]]上，海禁最早實行于14世紀末，橫跨[[元朝|元]]、[[明朝|明]]、[[清朝|清]]三個朝代。元朝曾經4次實行海禁，每次持续时间都比較短，共約11年，对海外贸易並未产生全局性的負面影响。而明清两朝的海禁，不仅在政策上有所强化，而且持续時間長達多年。中國的海禁也成為[[東南亞|東南亞]]陶瓷業發展的契機。另一方面，[[藩屬國|藩屬國]]例如[[琉球|琉球]]等國家，亦因為海禁的關係，利用獨佔與中國貿易的契機而獲取大量利益。此外，[[朝鲜王朝|朝鲜王朝]]及[[日本|日本]][[德川幕府|德川幕府]]皆執行海禁逾200年，規定指定國籍和商人外，官民禁止与別國商人貿易。

===海禁的實施===
[[File:YuanJunk(14thcentury).jpg|left]]
「海禁」是[[明朝|明朝]]的一项鎖國性質的基本國策，從洪武年間到隆慶年間，實行了近200年，縱貫大半個明朝。形成了成熟的海禁政策與概念。明朝海禁嚴格禁止人民對外通商貿易，限制外国人到中国进行贸易<ref>[[#佐久間|佐久間1992]] 197-199頁、[[#熊|熊1997]] 90頁、[[#檀上2005|檀上2005]] 147,162頁、[[#上田|上田2005]] 95頁</ref>。明朝唯恐禁海不力，有关海禁的诏令每过几年就會重申一遍。

明朝海禁时期规定人民不但“片板不许入海”<ref>《明史》卷二百五《朱纨传》</ref>，甚至民船造得稍大一些就是违法。《大明律》规定：“若奸豪势要及军民人等，擅造三桅以上违式大船，将带违禁货物下海，前往番国买卖，潜通海贼，同谋结聚，及为向导劫掠良民者，正犯比照已行律处斩，仍枭首示众，全家发边卫充军。其打造前项海船，卖与夷人图利者，比照将应禁军器下海者，因而走泄军情律，为首者处斩，为从者发边充军。”

[[蒙元民變|元末起義軍]]領袖之一的[[朱元璋|朱元璋]]擊敗其他義軍，於公元1368年建立[[明朝|明朝]]。一方面，明朝前期[[倭寇|倭寇]]猖獗。據《[[明史|明史]]》及《[[明實錄|明實錄]]》記載，仅1368年至1374年七年間，倭寇對中國沿岸的侵擾次數就多達二十三次<ref>[[#熊|熊1997]] 90頁</ref>。

此時正值日本的[[南北朝时代_(日本)|南北朝內戰]]，流離失所的日本平民為逃避戰亂，紛紛流亡海上。1392年（明洪武25年），日本北朝的[[足利氏|足利氏]]吞併南朝，不少南朝遺臣相繼逃亡出海，與日本浪人組成武裝，形成倭寇，侵擾中國沿岸地區<ref name="ming start japanese rob">{{cite book |title = 析論中國歷史《治亂因果篇》（上冊） |publisher = 永柏出版有限公司 |author = 呂振基、蘇榮 編著 |year = 2005年4月初版、2005年8月第二版 |pages = 第266頁 |isbn = 988-98403-1-6}}</ref>。另一方面，曾割據[[江南|江南]]沿海州縣的[[張士誠|張士誠]]和[[方國珍|方國珍]]，自被朱元璋擊敗后，其部下亡命海上，勾結日本[[浪人|浪人]]，侵擾[[山東|山東]]、[[浙江|浙江]]、[[福建|福建]]等沿海地區。有鑒於[[蘇州|蘇]]、[[松江府|松]]、[[常州府|常]]、[[嘉兴府|嘉]]、[[湖州府|湖]]等州是張士誠的舊地，朱元璋因而對以上地区採取歧視性政策，所征[[田賦|田賦]]較其他地區高出十倍以上<ref name="ming start japanese rob" />，當地百姓不堪其苦，流亡他鄉，或作為倭寇嚮導。此外，閩浙大族亦為了利益私通倭寇。

為了防止沿海反叛势力私通倭寇，洪武三年（1370年），明廷“罢太仓黄渡市舶司”<ref>《明太祖实录》卷四十九</ref>；洪武四年（1371年）十二月頒布「禁海令」，規定：“……仍禁濒海民不得私出海”；洪武七年（1374年），[[明朝|明朝]]撤销了自[[唐朝|唐朝]]起就存在的负责海外贸易的福建泉州、浙江明州（今宁波）、广东广州三處[[市舶司|市舶司]]。

[[永乐|永乐]]年间虽有[[郑和下西洋|郑和下西洋]]，但海禁政策较洪武年间更为严苛。永乐二年(1404年)诏令“原有海船者，悉改为平头船，所在有司，防其出入”<ref>《明太宗实录》卷二十七，永乐二年正月辛酉</ref>。永乐五年（1407年）甚至在[[越南|安南]]也实行海禁：“亦不许军民人等私通外境，私自下海贩鬻番货”<ref>《明太宗实录》卷六十八，永乐五年六月癸未</ref>。

===倭寇侵襲===
[[File:WakouAttack.jpg|left]]
明朝洪武、永乐年间，因当时明朝国力较强，没有造成重大影响。同時，洪武、永乐年间大力打擊海上的商業活動，阻礙海上貿易發展。因为明朝政治腐败，海防松弛，并实施闭关锁国措施，加上日本当时战乱频发，一些武士和海盗与部分受到闭关锁国负面影响的明朝商人联合，入侵中国，以至倭寇横行，在大片沿海城市展开掠夺。

=== 隆慶開港 ===
直到16世紀中葉，明廷為了重建海上秩序，對海禁存廢產生嚴重分歧。隆庆元年（1567年），福建巡抚[[涂泽民|涂泽民]]请开海禁，获准。但是，明朝僅開放了福建[[漳州|漳州]]的[[月港|月港]]一處口岸，而且只允許[[泉州|泉州]]和[[漳州|漳州]]的商人對外貿易<ref>{{cite web|title=月港：明朝外贸的唯一亮点|url=http://finance.qq.com/a/20120607/006466.htm|publisher=騰訊网|accessdate=2014-07-21}}</ref>，並且禁止對日貿易，史稱「[[隆慶開關|隆慶開關]]」。由于政府部分开启海禁，得以缓解。

===禁海與遷界===
{{see also|遷界令|明郑}}
顺治三年(1646年)编制的《大清律》保留了《大明律》中有关“私出外境及违禁下海”的条文。顺治四年(1647年)因广东平定颁发的“恩诏”中说：“广东近海，凡系漂洋私船，照旧严禁”。顺治十年的一份户部题本说：“自我朝鼎革以来，沿海一带，俱有严禁”。不过，当时的海禁政策并未真正执行，清人根本不知有海禁。甚至康熙十五年(1676年)时任[[江苏|江苏]]巡抚的[[慕天颜|慕天颜]]在《请开海禁疏》也说：“记顺治六七年间，彼时禁令未设”。

为了削弱、消灭[[郑成功|郑成功]]等抗清力量，早在顺治十二年（公元1655年），浙闽总督屯泰奏请“沿海省份，应立严禁，无许片帆入海，违者置重典。”顺治十三年（1656年）六月，清廷正式颁布“禁海令”，敕谕浙江、福建、广东、江南、山东、天津各省督抚提镇曰：“严禁商民船只私自出海，有将一切粮食、货物等项与逆贼贸易者，……不论官民，俱行奏闻正法，货物入官，本犯家产尽给告发之人。该管地方文武各官不行盘诘擒辑，皆革职，从重治罪；地方保甲通同容隐，不行举首，皆论死。”然而，海禁实行五年未达到预期效果。

順治十六年（公元1659年），鄭成功率領南明軍隊北伐失利後，退守廈門，清廷趁明軍新敗之際，派遣安南將軍[[達素|達素]]率領清軍進入福建，並將浙江[[明安達禮|明安達禮]]屬下八旗軍以及沿海各省綠營水軍，都歸達素親自指揮，企圖消滅鄭成功麾下明軍，史稱「[[厦门战役_(1660年)|廈門戰役]]」。在該戰役中，鄭成功麾下明軍大勝，清廷沿海各省水師在此戰中幾乎全軍覆滅，其海上力量受到毀滅性打擊，故此在顺治十八年（1661年），清廷采纳了[[黄梧|黄梧]]的“[[遷界令|遷界令]]”，将東南沿海居民內遷三十至五十里，村社田宅悉皆焚弃<ref>萧一山《清代通史》第372页，第370页，清廷对于沿海之政策及附录。中华书局版</ref>。康熙年间早期也曾多次重申海禁政策。

=== 展界與弛禁 ===
康熙亲政后，一些地方逐渐“开边展界”，人民回乡复业。康熙七年(1668年)，郑氏已退守臺湾，广东先行松弛海禁。康熙八年（1669年）允许沿海居民复界。平定[[三藩之亂|三藩之亂]]的战争即将结束之际，福建总督范承谟、姚启圣，江苏巡抚慕天颜，福建巡抚吴兴祚，广东巡抚李士祯等先后上疏，要求废除海禁，准民出海贸易。其中以康熙十五年(1676年)[[慕天颜|慕天颜]]的《请开海禁疏》最具代表性，由其中所述“今则盛京、直隶、山东之海船固听其行矣，海洲云臺之弃地亦许复业矣，[[香山|香山]]、[[澳门|澳门]]之陆路再准贸贩矣”可知此时北方的沿海省份已解除海禁，唯江浙闽粤四省尚在禁列之中，但江苏和广东的部分地区已弛禁。

康熙二十二年（1683年），清朝平定[[台灣|臺灣]]，正式展界。所谓“展界”，就是安排在顺治十八年前后被迁界令强迁的沿海居民复归故土。十一月，[[康熙帝|康熙帝]]将吏部侍郎杜臻、内阁学士席柱差往福建、广东主持沿海展界事宜，行前谕曰：“迁移百姓事关紧要，当查明原产给还原主，尔等会同总督巡抚安插，务使兵民得所。”<ref>《清圣祖实录》卷一一三，第7页</ref>

康熙二十三年（1684年），清朝正式开海，准许百姓对外贸易<ref>《清圣祖实录》卷一一七，第10页</ref>，并在“粤东之澳门（一說廣州）、福建之漳州府、浙江之宁波府、江南之云臺山”分别设立粤海关、闽海关、浙海关、江海关作为管理对外贸易和征收关税的机构。江浙闽粤四大海关总领各自所在省的所有海关口岸，通常下辖十几至几十个海关口岸<ref name="劉軍《财经问题研究》2012年11期 第21-30页">劉軍《财经问题研究》2012年11期 第21-30页</ref>。

=== 定點通商與縮禁貿易 ===
康熙五十六年(1717年)，清廷禁止南洋贸易，但并未禁止东洋贸易和西洋贸易<ref>《康熙起居注》（康熙五十五年十月）：“凡商船，照旧东洋贸易外，其南洋吕宋、噶罗吧等处不许商船前往贸易，于南澳等地方截住”“内地商船，东洋行走犹可。……至于外国商船，听其自来”</ref>。雍正五年(1727年)，时隔十年后清廷重新开放南洋贸易。乾隆二十二年（1757年），[[乾隆帝|乾隆帝]]以海防重地規範外商活動為理由，谕令[[西洋|西洋]]商人只可以在[[广东|广东]]通商{{notetag|1=乾隆谕令“本年来船虽已照上年则例办理，而明岁赴浙之船，必当严行禁绝。……此地向非洋船聚集之所，将来只许在广东收泊交易，不得再赴宁波。如或再来，必令原船返棹至广，不准入浙江海口。豫令粤关，传谕该商等知悉。……令行文该国番商，遍谕番商。嗣后口岸定于广东，不得再赴浙省”这一上谕只是让“外洋红毛等国番船”、“番商”只在广东通商，不得再赴浙江等地，而不是一些资料中所说的关闭江、浙、闽三海关，更不是“广州一口通商”。}}。但是实际上，当时在南洋的一些西方殖民者仍被允许到闽、浙、江海关贸易，特别是闽海关。例如，乾隆四十六年(1781年)、四十八年(1783年)、五十一年(1786年)，嘉庆十二年(1807年)、十四年(1809年)，均有[[西班牙|西班牙]]商人从南洋[[吕宋|吕宋]]到厦门贸易<ref>[清]周凯撰辑.道光厦门志[M].厦门：鹭江出版社，1996.</ref>。

===被迫开海===
[[第一次鸦片战争|第一次鸦片战争]]后，由于清朝战败，为了保全国家，除了被迫割地赔款，随之也被迫全面开启海禁，自此，清朝長達200多年的海禁政策結束。

== 明清海上貿易 ==
明末隆慶開關後，尽管明朝对海上貿易秩序的控制薄弱，但海上贸易仍有所发展，郑氏集团的规模为此提供了充分证明。东南亚欧洲殖民当局的一些文献从另一方面提供了明末海上贸易有所发展的证据。据统计，[[萬曆|萬曆]]十六年以前，從[[廣州|廣州]]出口到[[馬尼拉|馬尼拉]]的貨物總值為22萬西元，其中絲貨量值為19萬西元；萬曆二十一年的絲貨值達到25萬西元。崇禎九年以前，每艘開往[[墨西哥|墨西哥]]的“大帆船”，登記運載中國絲貨為300—400箱至500箱<ref>{{cite web |author1=刘军 |title=明清时期“闭关锁国”问题赘述 |url=http://www.xinfajia.net/12406.html |accessdate=2018-08-25}}</ref>。萬曆八年至十八年，每年運往果阿的絲貨為3000石，價值銀為24萬兩，利潤達36萬兩；崇禎九年，絲貨達6000石，贏利72萬兩。當時，不僅日本、[[葡萄牙|葡萄牙]]和[[西班牙|西班牙]]人穿上了中國絲綢，而且東南亞土著、墨西哥[[印第安人|印第安人]]和黑人也都普遍穿著中國的[[絲綢|絲綢]]與[[棉布|棉布]]服裝<ref name="劉軍《财经问题研究》2012年11期 第21-30页"/>。

清初康熙開海前，中國的海上貿易仍維持相當大的規模。康熙開海後，海上貿易量持續增長，貿易港口擴大，貿易國家、商船數量、進出口商品數量和貿易商品流通量值均不斷增加，對海上貿易的監管也較明代有效得多<ref name="劉軍《财经问题研究》2012年11期 第21-30页"/>。
