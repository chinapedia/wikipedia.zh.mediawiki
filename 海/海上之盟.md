'''海上之盟'''乃[[北宋|北宋]]及[[金國|金國]]為了聯合力量夾攻[[遼國|遼國]]而簽訂的軍事合作盟约 。宣和二年（1120年）[[宋朝|宋]]、[[金朝|金]]双方商定：[[金朝|金]]取[[辽朝|辽]]中京大定府，[[宋朝|宋]]取[[辽朝|辽]]南京析津府，[[辽朝|辽]]亡后，[[宋朝|宋]]将原给辽的岁币转纳于[[金国|金国]]，金同意将[[燕云十六州|燕云十六州]]之地归宋朝。由於双方接觸都因地理上受遼國阻隔，而需要海上經[[渤海|渤海]]往來，故称「海上之盟」<ref>{{cite web|title=宋金海上之盟|url=http://www.china.com.cn/aboutchina/zhuanti/lddw/2007-12/12/content_9375976.htm|publisher=中国网|accessdate=2014-01-17|language=中文|date=2007-12-12}}</ref> 。

== 背景 ==
自北宋立國於[[中原|中原]]以後，強大的遼國一直位於宋的北方，且時有南侵攻宋的野心，形成宋遼南北敵意對峙的局面。公元1004年，[[宋真宗|宋真宗]]與遼簽署了[[澶淵之盟|澶淵之盟]]後，宋遼之間維持一段達約一百年的和平，雙方之間都沒有完全消滅對方的能力<ref>{{cite journal|author=任崇岳|title=《论“澶渊之盟”后的宋辽关系》|journal=《历史教学》|date=01期|year=1984年|language=中文}}</ref> 。宋徽宗[[政和|政和]]五年（1115年），位於遼國東北勢力範圍內的[[女真族|女真族]]在[[完顏阿骨打|完顏阿骨打]]的領導下反抗遼國的統治，屢敗遼軍，並建立[[金國|金國]]，遼國國勢迅速下滑。此時北宋君主[[宋徽宗|宋徽宗]]與大臣[[蔡京|蔡京]]、[[童貫|童貫]]認為遼國亡國在即，金國會取而代之，決定聯金攻遼，不但可向金以示和好，而且以圖收復失去已達二百年的[[燕雲十六州|燕雲十六州]]之地。

== 內容 ==
[[重和|重和]]元年（1118年），徽宗派武義大夫馬政自[[山東|山東]]登州（今山東[[蓬萊|蓬萊]]）乘船渡海，以買馬為幌子，與金談判攻遼。此後宋金使者頻繁接觸。[[宣和_(北宋)|宣和]]二年（1120年），雙方商定以下的內容：
* 宋金各自進軍攻遼，其中金軍攻取遼上京（今内蒙古自治区[[巴林左旗|巴林左旗]]林东镇南）與中京大定府（今[[遼寧|遼寧]]昭烏達盟[[宁城县|宁城县]]天义镇大明乡），宋軍攻取遼的西京大同府（今[[山西|山西]][[大同市|大同]]）和南京析津府（今[[北京|北京]]）。
* 宋答應滅遼後，將原來於[[澶淵之盟|澶淵之盟]]輸給遼的歲幣轉輸給金。
* 金則答應將[[燕雲十六州|燕雲十六州]]還给宋。

結果，宋虽然收降易州守将[[高凤|高凤]]、涿州守将[[郭药师|郭药师]]，却最终攻遼失敗，而金軍順利攻下遼上京、遼中京及遼南京。金方事後指責宋未能兌現承諾「攻陷遼南京」，而拒絕還燕雲。金宋雙方經交涉後，北宋允以二十萬兩銀、三十萬匹絹給金，並納燕京代租錢一百萬貫，金才交還燕雲六州（景、檀、易、涿、薊、順，其中景、易二州不在十六州之内）及遼南京城。金軍得到六州中奚、契丹、渤海等族的人口财产，仅将汉人的财产留给宋朝，还要挟宋朝交出逃到宋朝的辽臣[[赵温讯|赵温讯]]、[[李处能|李处能]]等，并以追杀辽朝余部为名向宋朝索得军粮20万石。金朝还向宋朝索要投降宋朝的郭药师部“怨军”八千人，宋朝没有答应，作为交换，将幽州城辖区内150贯以上家产的3万余户人家交割给金朝。金军撤出城前還將遼南京城內財物和人口搜刮一空，宋接收的只是一座「城市丘墟，狐狸穴處」的空城。宋改遼[[遼南京|南京析津府]]為[[燕山府|燕山府]]。

此后，宋朝还要每年交给金朝140万两白银，幽州每年100万两的租税归金朝，而且要用实物交割。宋朝提出以辽朝西京大同府为交换，但金朝却提出要迁走其全部居民，宋朝只得又给了金朝20万两，但金朝却没有将大同府让给宋朝。

宋朝以收复燕云之功论功行赏，封[[童贯|童贯]]为豫国公再进封广阳郡王，升[[蔡攸|蔡攸]]为少师，宰相[[王黼|王黼]]由少师进位太傅，赏玉带，[[赵良嗣|赵良嗣]]为延康殿学士，命[[王安中|王安中]]作《复燕云碑》。

== 影響 ==
海上之盟簽訂後，金宋果然合力滅了[[遼國|遼國]]。但是遼國滅亡以後，宋朝變相失去遼國作為它的屏障，以阻擋金兵南下。此後金宋邊境正式接壤，双方仍然因为燕云十六州的归属等问题发生冲突，最终金兵果然於遼亡後南侵宋。1126年，北宋便在朝政敗壞，國力和軍力不振的情況下，遭強大的金兵攻克其[[首都|首都]][[汴京|汴京]]及中原一帶的領土，釀成[[靖康之變|靖康之變]]，立國167年的北宋滅亡。
