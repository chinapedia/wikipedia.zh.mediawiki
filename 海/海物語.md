《'''海物語'''》是2009年夏季于[[中部日本放送|CBC]]制作・[[日本新聞網_(TBS)|日本新聞網]]播放的[[日本電視動畫|日本電視動畫]]作品。

== 登場人物 ==
=== 海之世界 ===
; {{nihongo|[[瑪琳小姐|瑪琳]]|マリン||聲優：[[阿澄佳奈|阿澄佳奈]]}}
:来自大海的少女，有著天真烂漫，好奇心旺盛的性格。因為撿到宮守夏音的戒指，所以到空之世界尋找空之世界的主人。口頭是『愛死妳啦』。由於本身太過於純潔，反而對黑暗沒甚麼抵抗力，於月蝕之日時又到黑暗籠罩的影響，不受控制地被邪惡力量感染而對烏琳說出傷人的話，其後亦有多次因為情感上的干擾而無法變身。
; {{nihongo|烏琳|ウリン||聲優：[[堀江由衣|堀江由衣]]}}
: 瑪琳的妹妹，因為不放心姊姊瑪琳到空之世界，所以陪同瑪林一起前往。雖然自身有海之巫女的血統，但因為年紀尚幼小故其能力尚未覺醒。非常喜歡姐姐瑪琳，但卻因為自身沒有力量而感到自卑，再加上瑪琳因為夏音而往往忽略的烏琳，最後於月蝕之日受到黑暗之引誘而被瑟多拉附身並在海底封印瑟多拉的神殿裡羽化成暗之巫女。
; {{nihongo|薩姆|サム||聲優：[[間島淳司|間島淳司]]、[[儀武祐子|儀武祐子]]（幼少時）}}
:瑪莉的朋友，喜欢海葡萄。會不自覺的對異性拋媚眼，但反應往往不如預料中的好。
; {{nihongo|佤琳|ワリン||聲優：[[福井裕佳梨|福井裕佳梨]]}}
:瑪莉的朋友。
; {{nihongo|松本|まつもと||聲優：[[納谷六朗|納谷六朗]]}}
:辅助海巫女、空巫女的圣龟，海世界的长老，因為特殊血緣而使得壽命可以很長。
; {{nihongo|市川|いちかわ||聲優：[[明坂聰美|明坂聰美]]}}
:松本的女兒，對醃製品有特別喜愛。擁有聖龜的血統，故初期可以不受瑟多拉的邪惡之力影響。

=== 空之世界（人類世界） ===
; {{nihongo|宮守 夏音|みやもり かのん||聲優：[[壽美菜子|壽美菜子]]}}
:居住在天神子島的女高中生，經常露出一副消极的性格，常常被别人说散发出邪气而不接近她。口頭蟬是『好想去死』。因為自己的不安而與男友小島產生誤會，至此把小島送的戒指扔到海裡。
; {{nihongo|宮守 都|みやもり みやこ||聲優：[[永井幸子|永井幸子]]}}
:夏音的母親，继承家业的占卜师。
; {{nihongo|鈴木|すずき||聲優：[[儀武祐子|儀武祐子]]}}
:夏音的青梅竹馬，喜歡瑜伽。
; {{nihongo|小島|こじま||聲優：[[澤城美雪|澤城美雪]]}}
:夏音的前男友。因为误会被甩，但是还想和夏音继续交往。
; {{nihongo|大島|おおしま||聲優：[[豐崎愛生|豐崎愛生]]}}
:喜欢小岛，把夏音当敌人看的女孩。
; {{nihongo|歌者|うたしゃ||聲優：[[宮原永海|宮原永海]]}}
:在天神子岛上唱着流传下来的岛歌。

== 用語 ==
; {{nihongo|海之巫女|海の巫女}}
; {{nihongo|空之巫女|空の巫女}}
:海和空之巫女是心作為力量的起源有，隱藏對抗瑟多拉得到的力量。 
; 海人
:居住在海底的種族，外表和普通的人相同，但是到了地上的話，就無法作呼吸，戴上麥卡托之環後就能呼吸。與空人不同的地方是，海人是由貝殼所孕育出來的，故沒有空人所謂的親族之關係。
; 空人
:空人是海人稱呼人類的名字。
; {{nihongo|麦卡托之环|メルカトルリング}}
:海人所使用手鐲，戴上麦卡托之环後，即使在地上也變得能呼吸。相反地戴上麥卡托之環後，就變得無法在海裏呼吸。 
; {{nihongo|奥特里斯之环|オルテリウスリング}}
:空人進入海所以使用的手鐲，戴上奥特里斯之环後，即使在海裏也變得能呼吸，所以奥特里斯之环是和麦卡托之环完全相反的手鐲。平時是放在米糠醃的桶裏的拌鹽米糠的鐵釘用。 
; {{nihongo|瑟多拉|セドナ|||聲優：[[桑島法子|桑島法子]]}}
:傳說記載,瑟多拉很久以前從海裹來到空中，用強大的黑暗力量使得空墜落,讓黑暗籠罩世界,之後被空之巫女、海之巫女兩人合力封印在地上與海中。而由於烏琳解除了棺材的封印，導致瑟多拉在地上的封印解除,之後又將海洋生物邪惡化，奉命他們去收集邪惡的靈魂。實際上,瑟多拉並沒有所謂的實體存在,在歷史中出現多次,但每次都被空、海巫女封印,而這此得到烏琳這個實體是頭一遭,於第十一話時瑪琳與夏音才得知瑟多拉原來是島民流放到打海的各種悲傷集合體。
