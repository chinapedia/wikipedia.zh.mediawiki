'''海盜青年 (瑞典)''' ({{lang-sv|Ung Pirat}}) 是一個瑞典[[海盜黨|海盜黨]]的青年組織。他開始於2006年12月並且在 2009年4月到2010年春季，擁有超過22000會員，並且為瑞典最大會員數最多的青年組織。<ref>http://www.dn.se/fordjupning/europa2009/gratisgenerationen-ger-sig-in-i-politiken-1.886073</ref><ref>{{cite web |url=http://www2.unt.se/printartikel/1,3110,MC=70-AV_ID=588516-SC_ID=65,00.html |title=存档副本 |accessdate=2013-12-30 |deadurl=yes |archiveurl=https://web.archive.org/web/20120728203404/http://www2.unt.se/printartikel/1%2C3110%2CMC%3D70-AV_ID%3D588516-SC_ID%3D65%2C00.html |archivedate=2012-07-28 }}</ref><ref>{{cite web |url=http://svt.se/svt/jsp/Crosslink.jsp?d=22577&a=1396668 |title=存档副本 |accessdate=2009-11-21 |deadurl=yes |archiveurl=https://web.archive.org/web/20110612235759/http://svt.se/svt/jsp/Crosslink.jsp?d=22577&a=1396668 |archivedate=2011-06-12 }}</ref><ref>http://www.sr.se/cgi-bin/halland/nyheter/artikel.asp?Artikel=2730705{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref>{{cite web |url=http://www.piratpartiet.se/storlek |title=存档副本 |accessdate=2017-11-23 |deadurl=yes |archiveurl=https://web.archive.org/web/20120919080451/http://www2.piratpartiet.se/storlek |archivedate=2012-09-19 }}</ref>
