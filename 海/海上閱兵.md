'''海上閱兵'''（{{lang-en|Fleet review}}）又称'''阅舰式'''或'''观舰式'''（源自日文中的“{{lang|ja|観艦式}}”），是[[海軍|海軍]]的閱兵儀式。此項儀式由[[英國|英國]]開始，近年來，成為國際上軍事演習或慶祝建國、建軍周年等慶典時展示軍力的儀式。

海上阅兵通常分为'''阅兵式'''和'''分列式'''。举行阅兵式时，受检阅的舰船都有固定的泊位，整齐地排列。检阅舰则搭载本国[[国家元首|元首]]、[[政府首脑|政府首脑]]和海军将领，以及受邀的别国宾客依次检阅列队战舰。分列式又称为码头校阅式，举行分列式时，受阅舰艇编队以一定顺序从码头的检阅台前通过。<ref>{{Cite web |url = http://news.163.com/09/0401/07/55Q0IP9U000120GR.html |title = 海上大阅兵 中外舰艇奔赴青岛(图) |author =  |date = 2009-04-01 |publisher = 网易 |language = zh-cn |accessdate = 2018-04-15 }}</ref>

==[[中国人民解放军海军|中国人民解放军海军]]海上閱兵==
*1957年8月4日，周总理受毛主席委托，检阅了北海舰队前身青岛基地官兵。
*1995年10月19日，人民海军北海舰队在黄海某海域，举行了新中国成立以来规模最大的海上联合军事演习和盛大的海上阅兵式。
*2005年8月18日，中俄两军首次在青岛、山东半岛东南海域等地区举行“和平使命-2005”中俄联合军事演习。2005年8月23日下午，海上分列式在山东半岛东南海域开始。
*2009年4月23日下午，为纪念中国人民解放军海军成立60周年而举行的海上阅兵在[[青岛|青岛]]附近[[黄海|黄海]]海域举行。海上检阅式分两个阶段，第一阶段为海上分列式，检阅中国海军的舰艇和飞机；第二阶段为海上阅兵式，检阅来自14个国家的21艘舰艇。<ref>{{cite web |title=组图：庆祝人民海军成立60周年海上阅兵活动举行 |url=http://military.people.com.cn/GB/43331/9185727.html |website=人民网}}</ref>
*2018年4月12日上午，中央军委在[[南海|南海]]海域隆重举行海上阅兵，展示人民海军崭新面貌，激发强国强军坚定信念。<ref>{{cite web |title=中央军委在南海海域隆重举行海上阅兵 |url=http://www.mod.gov.cn/shouye/2018-04/12/content_4809295.htm |website=中华人民共和国国防部}}</ref>

==[[加拿大海軍|加拿大海軍]]海上閱兵==
在[[加拿大|加拿大]]，海上閱兵可能需要在[[大西洋|大西洋]]或[[太平洋|太平洋]]海岸的地方，通常在[[哈利法克斯|哈利法克斯]][[哈利法克斯港|港]]和[[維多利亞_(英屬哥倫比亞)|維多利亞]][[維多利亞港_(英屬哥倫比亞)|港]]。

*1958年7月：為了紀念[[不列顛哥倫比亞|不列顛哥倫比亞]]加入[[加拿大聯邦|加拿大聯邦]]100週年，由王妹[[瑪格麗特公主_(斯諾登伯爵夫人)|瑪格麗特公主]]主持加拿大皇家海軍的海上閱兵。

*1959年6月：為了慶祝[[聖羅倫斯航道|聖羅倫斯航道]]（Saint Lawrence Seaway）開幕，[[英國女王|英國女王]][[伊麗莎白二世|伊麗莎白二世]]出席在[[蒙特利爾|蒙特利爾]]舉行由加拿大皇家海軍和[[美國海軍|美國海軍]]艦艇參加的海上閱兵。

*1959年7月：英國女王伊麗莎白二世主持在[[哈利法克斯港|哈利法克斯港]]舉行的加拿大皇家海軍的海上閱兵。

*2010年6月12日：為紀念加拿大海軍成立100週年，[[加拿大總督|加拿大總督]][[米夏埃爾·讓|米夏埃爾·讓]]將軍主持在[[Esquimalt港|Esquimalt港]]舉行由加拿大海軍、[[澳大利亞皇家海軍|澳大利亞皇家海軍]]、[[法國海軍|法國海軍]]、[[日本海上自衛隊|日本海上自衛隊]]、[[紐西蘭皇家海軍|紐西蘭皇家海軍]]和美國海軍艦艇參加的國際海上閱兵。

*2010年6月29日：為紀念加拿大海軍成立100週年，英國女王伊麗莎白二世主持在[[哈利法克斯港|哈利法克斯港]][[貝德福德盆地|貝德福德盆地]]舉行由[[英國皇家海軍|英國皇家海軍]]、加拿大海軍、美國海軍、[[巴西海軍|巴西海軍]]、[[丹麥皇家海軍|丹麥皇家海軍]]、法國海軍、[[德國海軍|德國海軍]]和[[荷蘭皇家海軍|荷蘭皇家海軍]]艦艇參加的國際海上閱兵。
