'''海峽兩岸服務貿易協議存查爭議'''是指在2014年3月17日時，[[中國國民黨|中國國民黨]]籍[[立法委員|立法委員]][[張慶忠|張慶忠]]在被[[民主進步黨|民主進步黨]]籍立法委員阻擋登上主席臺的情況下，於會議室後方宣布《[[海峽兩岸服務貿易協議|海峽兩岸服務貿易協議]]》依法視為已審查並且交由院會存查，進而引起的關於違背朝野協商內容以及法律引用問題所帶來的爭議事件。其中中國國民黨主張《海峽兩岸服務貿易協議》已經超過3個月的期限，因此依照法律所提到的行政命令審查之規定視為已審查，並且轉交立法院院會存查。但是民主進步黨則主張《海峽兩岸服務貿易協議》並非行政命令，因此必須依照先前朝野協商的內容在審查委員會內進行逐條逐項的實質審查，之後才能夠提交給院會表決。而在這次事件發生過後，包括民主進步黨在內的反對團體紛紛發起行動要求恢復逐條審查，並且進一步引起之後由大學生發起的[[318青年佔領立法院行動|佔領立法院行動]]。

== 背景 ==
=== 協議簽署 ===
{{See Also|兩岸服務貿易協議}}

2010年6月29日時，為了推動兩岸經貿正常化而使得兩岸兩會（海協會與海基會）共同簽署了《[[海峽兩岸經濟合作架構協議|海峽兩岸經濟合作架構協議]]》。之後雙方依照《海峽兩岸經濟合作架構協議》第四條「服務貿易」規定，兩岸同意在服務貿易早期基礎上在協議生效後6個月內就雙方的服務貿易協議展開協商。其中《[[海峽兩岸服務貿易協議|海峽兩岸服務貿易協議]]》討論的內容要求進一步制度化規範和保障兩岸雙方服務提供者的權益，擴大業界交流合作和市場規模並且減少限制性措施。而在2013年6月21日雙方正式簽署之後，[[馬英九政府|馬英九政府]]多次表示此案攸關臺灣經濟發展並且呼籲以[[中國國民黨|中國國民黨]]立法委員佔大多數的[[立法院|立法院]]能夠盡早通過<ref>{{Cite web |url = http://www.bbc.com/news/world-asia-26641525 |title = Protesters occupy Taiwan parliament over China trade deal |author =  |date = 2014年3月19日 |publisher = [[BBC新聞|BBC新聞]] |language = en |accessdate = 2014年3月19日 |quote =  }}</ref>。其中​​《海峽兩岸服務貿易協議》獲得通過後將允許臺灣和中國大陸的服務型企業得以互相往來，並且在對方領土設立分支機構或者是連鎖店鋪。其中[[中華人民共和國|中華人民共和國]]將允許臺灣企業提供線上購物、醫療、交通、旅遊等服務，而[[中華民國|中華民國]]則將允許中國大陸企業得以經營出版、影視、醫療照顧等工作。其中預期經濟政策開放的市場以及就業效應，政府認為藉由《海峽兩岸服務貿易協議》的施行能夠對於臺灣的電子商務、資訊業、線上遊戲業、金融業、環保業、物流以及運輸業等產業有正向影響<ref>{{Cite web |url = http://www.npf.org.tw/post/2/13204 |title = 2013年國內十大財經大事剖析 |author = 謝明瑞 |date = 2014年2月7日 |publisher = [[財團法人國家政策研究基金會|財團法人國家政策研究基金會]] |language = zh-hant |accessdate = 2014年4月6日 |quote =  }}</ref>。

而[[中華民國經濟部|中華民國經濟部]]在公告準備進行有關《海峽兩岸服務貿易協議》的談判後，分別在2013年4月25日<ref>{{Cite web |url = http://lci.ly.gov.tw/LyLCEW/communique1/final/pdf/102/28/LCIDC01_1022801.pdf |title = 立法院公報 第102卷第28期 |author =  |date = 2013年5月16日 |publisher = [[立法院|立法院]] |language = zh-hant |accessdate = 2014年4月6日 |quote =  }}</ref>、5月2日以及5月30日召開3次立法院專案報告<ref>{{Cite web |url = http://lci.ly.gov.tw/LyLCEW/communique1/final/pdf/102/31/LCIDC01_1023101.pdf |title = 立法院公報 第102卷第31期 |author =  |date = 2013年5月24日 |publisher = [[立法院|立法院]] |language = zh-hant |accessdate = 2014年4月6日 |quote =  }}</ref><ref>{{Cite web |url = http://lci.ly.gov.tw/LyLCEW/communique1/final/pdf/102/44/LCIDC01_1024401.pdf |title = 立法院公報 第102卷第44期 |author =  |date = 2013年6月19日 |publisher = [[立法院|立法院]] |language = zh-hant |accessdate = 2014年4月6日 |quote =  }}</ref>。而為了基於談判期間保密之理由，各個主管機關也以小型非公開的方式向46種服務業公會代表進行110次諮商會議或者小型座談<ref>{{Cite web |url = http://www.taiwannews.com.tw/etn/news_content.php?id=2447342 |title = 說明服貿 經部卯力跑攤 |author = 黃巧雯 |date = 2014年3月29日 |publisher = [[Taiwan_News|Taiwan News]] |language = zh-hant |accessdate = 2014年4月6日 |quote =  }}</ref>。然而雖然今日中國大陸已經成為臺灣最大的貿易夥伴、同時雙方之間的交流關係也逐漸獲得改善，但是在野黨、民間學術界、社會團體等仍然擔憂牽涉臺灣服務產業貿易和投資的開放條件的《海峽兩岸服務貿易協議》對於臺灣中小型企業的負面影響，認為不完善的經濟制度將會促使得臺灣更加仰賴市場規模較為龐大的中國大陸，甚至可能會嚴重影響中華民國自身的政治與經濟環境<ref>{{Cite web |url = http://www.jiji.com/jc/c?g=int_30&k=2014031900405 |script-title = ja:学生が国会議場占拠＝中国との協定に反対－台湾 |author =  |date = 2014年3月19日 |publisher = [[時事通信社|時事通信社]] |language = ja |accessdate = 2014年3月19日 |quote =  }}</ref><ref>{{Cite web |url = http://www.washingtonpost.com/business/china-pact-opponents-occupy-taiwan-legislature/2014/03/18/3f049e50-af02-11e3-b8b3-44b1d1cd4c1f_story.html |title = China trade pact foes occupy Taiwanese legislature |author = [[美聯社|美聯社]] |date = 2009年3月19日 |publisher = 《[[華盛頓郵報|華盛頓郵報]]》 |language = en |accessdate = 2014年3月19日 |quote =  }}</ref>。與此同時社會運動團體還批評雙方政府談判缺乏透明度，特別是沒有公開在2013年6月在[[上海市|上海市]]召開的討論內容。這些種種的措施使得立法院朝野兩黨開始不斷發生衝突，

=== 朝野協商 ===
在2013年6月25日時，中國國民黨與民主進步黨在經過朝野協商後同意強制逐條審查並且表決通過，並且未經實質審查通過前協議不得啟動生效。其中朝野協商的簽署者除了立法院院長[[王金平|王金平]]和立法院副院長[[洪秀柱|洪秀柱]]，還包括中國國民黨籍立法委員林鴻池、[[賴士葆|賴士葆]]和[[林德福|林德福]]、民主進步黨籍立法委員[[柯建銘|柯建銘]]、[[邱議瑩|邱議瑩]]和[[潘孟安|潘孟安]]、台灣團結聯盟立法委員[[許忠信|許忠信]]和[[黃文玲|黃文玲]]以及親民黨籍立法委員[[李桐豪|李桐豪]]。之後[[中華民國經濟部|中華民國經濟部]]在7月時提出《兩岸洽簽服務貿易協議對我總體經濟及產業之影響評估》，但是對於外界所質疑的負面影響並無回應。而到了同年8月召開第二次臨時會期間，王金平再次表示的《海峽兩岸服務貿易協議》將採取逐條審查的方式進行，不可能自動生效<ref>{{Cite web |url = http://iservice.libertytimes.com.tw/2013/specials/stp/news.php?rno=4&no=763359&type=a |title = 王金平重批：協調好的 都沒遵守 |author = 曾韋禎 |date = 2014年3月19日 |publisher = 《[[自由時報|自由時報]]》 |language = zh-hant |accessdate = 2014年3月26日 |quote =  }}</ref>。另外一方面在朝野協商中還同意在協議實質審查前分別由中國國民黨以及民主進步黨舉辦16場的公聽會，然而包括[[反黑箱服貿行動聯盟|反黑箱服貿行動聯盟]]、[[黑色島國青年陣線|黑色島國青年陣線]]、律師[[賴中強|賴中強]]和學者[[鄭秀玲|鄭秀玲]]等人則批評公聽會流於形式，同時政府對於公民團體與反對學者提出的意見並沒有回應或者有所改變<ref>{{Cite web |url = http://anntw.com/articles/20140320-bTCx |title = 服貿16場公聽會？ 民團：流於形式！ |author = 方家敏 |date = 2014年3月20日 |publisher = 《[[台灣醒報|台灣醒報]]》 |language = zh-hant |accessdate = 2014年3月26日 |quote =  }}</ref>。

不過中華民國經濟部部長[[張家祝|張家祝]]則表示雖然依照第8條的緊急磋商條款，在協議施行後只要認定對產業造成實質性損害便能夠暫停進行，但是按照國際慣例簽署後的條約在生效前都不應該有所更動<ref>{{Cite web |url = http://www.udn.com/2014/3/18/NEWS/NATIONAL/NATS6/8555046.shtml |title = 張家祝：服貿協議生效前不能修改 |author = 盧沛樺 |date = 2014年3月18日 |publisher = [[聯合新聞網|聯合新聞網]] |language = zh-hant |accessdate = 2014年3月22日 |quote =  }}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref>{{Cite web |url = http://www.cna.com.tw/news/aipl/201403060327-1.aspx |title = 張家祝：服貿不可能重啟談判 |author = 王靖怡、戴雅真和溫貴香 |date = 2014年3月6日 |publisher = [[中央通訊社|中央通訊社]] |language = zh-hant |accessdate = 2014年3月22日 |quote =  }}</ref>。由於執政的中國國民黨在立法院113個席次中佔有65席，因此作為主要反對黨的[[民主進步黨|民主進步黨]]則多次嘗試拖延協議審查。其中民主進步黨依照協議內容所舉辦的8場公聽會拖延了4個月，而民主進步黨黨團總召[[柯建銘|柯建銘]]更公開表示不會讓《海峽兩岸服務貿易協議》於這會期通過<ref>{{Cite web |url = http://www.chinatimes.com/newspapers/20140320000941-260301 |title = 馬英九鐵了心 本會期過服貿 |author = 陳筑君 |date = 2014年3月20日 |publisher = [[中時電子報|中時電子報]] |language = zh-hant |accessdate = 2014年3月25日 }}</ref>。另外一方面，自2013年6月20日開始託付委員會審查的《海峽兩岸服務貿易協議》到同年12月底時便已經逾期3個月，對此前行政院經濟建設委員會主任委員[[尹啟銘|尹啟銘]]則表示將原先的點出《海峽兩岸服務貿易協議》從備查案改為審查案、甚至在朝野協商中要求逐條審查與表決，已經違反了《[[臺灣地區與大陸地區人民關係條例|臺灣地區與大陸地區人民關係條例]]》及《[[地方制度法|地方制度法]]》的規範<ref>{{Cite web |url = http://www.chinatimes.com/newspapers/20140312000447-260109 |title = 尹啟銘專欄－審查服貿協議 立法院知法違法 |author = [[尹啟銘|尹啟銘]] |date = 2014年3月12日 |publisher = [[中時電子報|中時電子報]] |language = zh-hant |accessdate = 2014年3月22日 |quote =  }}</ref>。

== 經過 ==
=== 計畫通過 ===
早在2013年12月26日時，立法院內政委員會召集委員[[張慶忠|張慶忠]]便刻意安排此次《海峽兩岸服務貿易協議》審查會議，並且藉由過往立法院「一案一召委」的慣例以方便搶佔主導權。不過當時部份藍營人士認為依照先前朝野協商結果應該在公聽會結束後才能進入委員會審查，並且藍營內部本身因為未能就如何通過《海峽兩岸服務貿易協議》達成共識而一再向綠營妥協<ref>{{Cite web |url = http://www.chinatimes.com/newspapers/20140318001130-260202 |title = 服貿再拖 只會掐死經濟 |author = 呂雪彗 |date = 2014年3月18日 |publisher = [[中時電子報|中時電子報]] |language = zh-hant |accessdate = 2014年3月22日 |quote =  }}</ref>。3月10日至3月14日期間由民主進步黨立法委員[[陳其邁|陳其邁]]擔任立法院內政委員會在主席時，雖然依照協商內容兩度準備針對協議實質內容逐條審查<ref>{{Cite web |url = http://www.faz.net/aktuell/politik/ausland/studenten-stuermen-taiwans-parlament-12853335.html |title = Studenten besetzen Taiwans Parlament |author = Klaus Bardenhagen |date = 2014年3月19日 |publisher = 《[[法蘭克福彙報|法蘭克福彙報]]》 |language = de |accessdate = 2014年3月19日 |quote =  }}</ref>。但是執政黨以及在野黨立法委員仍為了爭奪主導權而多次在立法院內爆發肢體衝突，其中中國國民黨黨團以民主進步黨無權排審為由杯葛議事。其中柯建銘指控中國國民黨破壞立法院議事規則與慣例，並且認為在下一周中國國民黨立法委員張慶忠不應該排案進行審理<ref>{{Cite web |url = http://www.udn.com/2014/3/17/NEWS/NATIONAL/NAT1/8551953.shtml |title = 藍搶服貿主導權 綠嗆不讓會開成 |author = 林政忠 |date = 2014年3月17日 |publisher = [[聯合新聞網|聯合新聞網]] |language = zh-hant |accessdate = 2014年3月22日 |quote =  }}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。而包括林鴻池等中國國民黨黨團幹部曾數度找立法院院長[[王金平|王金平]]請求協商化解僵局，但是卻一直未獲得正面回應。而王金平在3月16日接受採訪時表示《海峽兩岸服務貿易協議》已經交由委員會審查，在立法院院會無權干預的情況下只能視情形協調處理<ref>{{Cite web |url = http://www.udn.com/2014/3/17/NEWS/NATIONAL/NAT1/8551952.shtml |title = 王金平黨籍案牽動服貿 立院戰局添亂 |author = 陳乃綾、林河名和李昭安 |date = 2014年3月17日 |publisher = [[聯合新聞網|聯合新聞網]] |language = zh-hant |accessdate = 2014年3月22日 |quote =  }}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

不過藍營在認定綠營在聯席委員會上並沒有意願按照協商逐條審理，並且認為在反對人士泛政治化操作下要理性進入逐條實質審查已經十分困難，因此決定改採取強勢作法以通過協議。對此中國國民黨黨團核心幹部於3月15日和3月16日密集開會討論，其中包括審慎評估何時展開行動以及有關王金平黨籍案所可能帶來的效應等，最後要求所有人高度保密並且控制知情人士不超過5人。其中中國國民黨一開始設定的目標為設法將《海峽兩岸服務貿易協議》送出審查委員會，藉此脫離朝野政黨意見僵持不下的困局，之後在透過4月初召開的立法院院會針對《海峽兩岸服務貿易協議》進行表決。在會議中便有立法委員建議仿照過去民主進步黨立法委員佔領主席臺時，由中國國民黨籍召集委員直接宣佈開會並且讓迅速送出委員會的方式處理之。而為了解決之後綠營提出的確認議事錄等抗議手段，在這之後黨團則決定藉由改變討論議程並且不再召開聯席委員會的方式應對<ref>{{Cite web |url = http://www.chinatimes.com/newspapers/20140318001432-260102 |title = 服貿闖關 藍迫王金平表態 |author = 楊毅 |date = 2014年3月18日 |publisher = [[中時電子報|中時電子報]] |language = zh-hant |accessdate = 2014年3月22日 |quote =  }}</ref>。
