'''海湾阿拉伯国家合作委员会'''（[[阿拉伯语|阿拉伯语]]：'''{{rtl-lang|ar|مجلس التعاون لدول الخليج الفارسي}}'''），原名'''海湾合作委员会'''（阿拉伯语：'''{{rtl-lang|ar|مجلس التعاون الخليج الفارسي}}'''），[[中文|中文]]简称'''海合会'''，是一个包括[[阿拉伯|阿拉伯]][[波斯湾地区|波斯湾地区]]的6个[[国家|国家]]（即'''波斯湾六国'''）在内的政府[[国际组织|国际组织]]和[[贸易集团|贸易集团]]，其目标主要针对[[经济|经济]]和[[社会|社会]]方面。

== 概况 ==
海合会于1981年5月25日在阿联酋[[首都|首都]][[阿布扎比|阿布扎比]]宣告成立，成员国包括[[巴林|巴林]]、[[科威特|科威特]]、[[阿曼|阿曼]]、[[卡塔尔|卡塔尔]]、[[沙特阿拉伯|沙特阿拉伯]]和[[阿拉伯联合酋长国|阿拉伯联合酋长国]]；成员国之间的经济一体化协议于同年11月11日在沙特阿拉伯首都[[利雅得|利雅得]]签署。这六个国家通常被称为'''海合国'''<ref>{{cite web|url=http://www.tourism.australia.com/NewsCentre.asp?sub=0315&al=1911|title=澳大利亚旅游局加大对波斯湾国家的投资|work=[[澳大利亚旅游局|澳大利亚旅游局]]|date=2005年10月6日|accessdate=2007年4月7日|deadurl=yes|archiveurl=https://web.archive.org/web/20070904120023/http://www.tourism.australia.com/NewsCentre.asp?sub=0315&al=1911|archivedate=2007年9月4日}}</ref> 或'''波斯湾六国'''。

不是每一个[[波斯湾|波斯湾]]（[[阿拉伯人|阿拉伯人]]称[[阿拉伯湾|阿拉伯湾]]）沿岸的国家都是海合会的成员国。[[伊朗|伊朗]]被故意排除在外，因为它不是[[阿拉伯国家|阿拉伯国家]]<ref>{{cite web |author=西蒙·亨德森|url=http://www.washingtoninstitute.org/templateC05.php?CID=2424 |title=波斯湾风云：阿拉伯国家和伊朗核问题|work=[[华盛顿近东政治研究所|华盛顿近东政治研究所]] |date=2005年12月21日|accessdate=2007年4月7日}}</ref>；而“直到2003年，[[伊拉克|伊拉克]]一直都扮演着阿拉伯波斯湾国家抵制伊朗的角色。”<ref>{{cite web |url=http://www.upi-fiia.fi/fin/tilaisuudet/menneet_tilaisuudet_2007/iran_iraq_and_the_gulf_region/ |title=伊朗、伊拉克和波斯湾地区|author=[[穆斯塔法·阿拉尼|穆斯塔法·阿拉尼]]博士，[[波斯湾研究中心|波斯湾研究中心]]|work=芬兰国际事务研究所|date=2007年2月27日|accessdate=2007年4月7日}}</ref> 虽然伊拉克是沿波斯湾的国家，但也未加入海合会。[[也门|也门]]虽然不在[[波斯湾地区|波斯湾地区]]，但它正在与波斯湾六国谈判以取得海合会成员国资格。<ref>{{cite web|url=http://yementimes.com/article.shtml?i=905&p=local&a=7 |title=也门有望在2016年加入海合会 |work=[[也门时报|也门时报]] |date=2006年12月19日 |accessdate=2007年4月7日 |deadurl=yes |archiveurl=https://web.archive.org/web/20070922034941/http://yementimes.com/article.shtml?i=905&p=local&a=7 |archivedate=2007年9月22日 }}</ref>。另外，在2001年12月召开的海合会第22届首脑会议，同意也门加入其[[卫生|卫生]]、[[教育|教育]]、劳工和社会事务部长理事会等机构。

== 宗旨 ==
* 制定在诸如[[经济|经济]]、[[金融|金融]]、[[贸易|贸易]]、[[海关|海关]]、[[旅游|旅游]]、[[立法|立法]]和[[行政|行政]]等方面的类似规则；
* 促进[[工业|工业]]、[[矿业|矿业]]、[[农业|农业]]、[[水力|水力]]和[[畜力|畜力]]的科学技术进步；
* 建立科学研究中心；
* 设立共同[[投资|投资]]；
* 鼓励私营部门合作；
* 加强和密切成员国人民之间的联系、交往与合作；

== 组织机构 ==
* 最高理事会：海合会的最高权力机构，由各成员国[[国家元首|国家元首]]组成。主席由各成员国国家元首按国名字母顺序轮流担任，任期为一年。
* 部长理事会：由各成员国[[外交大臣|外交大臣]]（外交部长）<ref>波斯湾六国中除[[阿联酋|阿联酋]]外现均为[[君主制|君主制]]国家，所以不称外交部长而称外交大臣；组成阿联酋的七个酋长国亦为君主国，但整个阿联酋为[[联邦制|联邦制]]国家，其外交长官称外交部长。</ref> 或代表他们的其他大臣（部长）组成。主席由各成员国按字母顺序轮流担任，任期为一年。
* 总秘书处：位于利雅得，由秘书长和负责政治、财经及军事事务的三名助理秘书长主持工作。秘书长按国名[[阿拉伯字母|阿拉伯字母]]顺序轮流担任，并由最高理事会任命。
