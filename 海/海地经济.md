[[海地|海地]]是西半球最穷的国家，亦是世界上最不发达的国家之一。[[联合国开发计划署|联合国开发计划署]]的[[人类发展指数|人类发展指数]]评价中，海地在177个国家中名列第153位。2004年，海地的GDP（按购买力计算）为120.5亿美元，排名第145位；人均GDP为1500美元，排名第159位。2004年海地通货膨胀率为22%，外债为14亿美元。

海地货币为[[古尔德|古尔德]]（Gourde），缩写为HTG，一古尔德合100生丁。

==經濟史==
海地從法國政府在1804年獨立前，是世界上最富有和最有生產力的殖民地。獨立後海地在國際舞台上遭受孤立，[[让-皮埃尔·布瓦耶|让-皮埃尔·布瓦耶]]時期法国对海地提出了索赔要求，要求其赔偿独立战争期间没收的白人种植园。这一要求在1825年遭到拒绝，法国于是派遣了一支舰队来封锁伊斯帕尼奥拉岛，强索3000万美元的赔偿。布瓦耶最后同意支付1800万美元，但是海地无力支付，于是同法国展开谈判。谈判的最终结果是把赔款减为1200万美元，在30年内支付，每年支付40万美元。为了应付赔款，布瓦耶减少了政府开支，并在农村采取军事管理政策，以期维持农业生产。1883年，最終还完賠款。

== 農業 ==
[[File:Papslam.png|thumb]]
1987年之前，海地國內生產的稻米仍能供應國內需求，海地人以稻米為主食，美國前總統[[柯林頓|柯林頓]]任內（1992～2000），柯林頓政府施壓海地政府調降美國稻米進口關稅從35％到3％，美國農業部補助下的美國稻米，竟佔海地民眾稻米年消費總量40萬公噸的4分之3。農業生產基礎完全被摧毀摧毀，柯林頓在聯合國駐海地特別大使任內因此道歉。
海地的经济以农业为主。大部分农民在自己拥有的小块土地上耕作，提供基本的口粮。不获利的小农场占全国耕地总数的70%。造成这种现象的原因是海地独立后将面积广大的甘蔗种植园拆分成小地产分配给黑人所致（1791年海地的糖出口量为74000吨，1825年跌至2吨，1960年为15000吨）。在1791-1802年海地黑人起义期间，北部干燥平原上的灌溉系统被黑人奴隶破坏，至19世纪初被彻底废弃，导致占海地人口50%的北海地人只能在占全国耕地总面积17%的耕地上生活。1915年-1934年美国占领期间，在海地北部建立了一些大农场，其面积占全国耕地的7%，但提供了海地全国30%的农业产量。

海地目前的主要粮食作物为[[玉米|玉米]]（种植面积约30万[[公顷|公顷]]）和[[高粱|高粱]]。北方有少量[[旱稻|旱稻]]，阿蒂博尼特区有7万公顷的[[水稻|水稻]]田。木薯、甘薯、[[山药|山药]]等块茎作物和香蕉仅供海地本国居民食用，并不出口。海地的主要经济作物是[[咖啡|咖啡]]，其种植面积约14万公顷，其出口量在2.2万吨至4万吨之间浮动。次要的经济作物包括甘蔗和剑麻。2004年，海地的出口总额为3.38亿美元，最大的出口对象是美国（占海地出口总额的81.8%），其次是[[多米尼加|多米尼加]]和[[加拿大|加拿大]]。粮食是海地最主要的进口商品，其次是[[纺织|纺织]]品和其他消费品。2004年海地的进口额为10.85亿美元，最大的进口国是[[美国|美国]]（占海地进口总额的52.9%），其次是多米尼加、[[日本|日本]]和[[德国|德国]]。

== 工業 ==
[[File:Bas-Ravine.jpg|thumb]]
由于能源不足，海地的工业非常不发达。1960年代在阿蒂博尼特河谷建立了[[水力发电|水力发电]]厂，首都[[太子港|太子港]]有[[水泥|水泥]]、麵粉、制鞋、纺织、肥皂、制药、饮料、[[植物油|植物油]]、塑料、[[家具|家具]]、酿酒等小型消费工业，北部平原有剑麻加工厂，累凯平原的[[甘蔗|甘蔗]]种植区有糖厂。海地的[[失业率|失业率]]极高通常在60%上下，三分之二的工人没有固定的工作。
== 觀光業 ==
[[File:Tap_tap_public_transportation_Haiti.jpg|thumb]]
海地政府曾致力于发展[[旅游|旅游]]业，并曾经设立了专门的旅游部。到1956年，海地的外国游客已经达6.6万人，主要是来自北美的游客，他们为海地政府带来了除[[咖啡|咖啡]]出口外的第二大[[外汇|外汇]]收入（1961年为1000万[[美元|美元]]）。1962年后，由于政治原因和[[愛滋病|愛滋病]]，来自外国的游客数量大大下降。

== 救濟 ==
美国是海地最大的援助国，在1995财政年度至1999财政年度之间，美国向海地提供了8.84亿美元的援助，用于购买食品、农业发展、[[教师|教师]]培训、计划生育和健康保障计划。美国和平队和其他民间慈善组织也向海地派出援助人员。[[国际货币基金组织|国际货币基金组织]]和[[国际开发银行|国际开发银行]]也向海地提供了财政援助。
