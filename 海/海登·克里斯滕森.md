'''海登·克里斯滕森'''（{{lang|en|Hayden Christensen}}，{{bd|1981年|4月19日||}}）是一位获[[金球奖_(影视奖项)|金球奖]]提名的[[加拿大|加拿大]]籍[[电视|电视]]和[[电影|电影]][[演员|演员]]。他年少时就出现在加拿大的电视节目中，并于1990年代转战[[美国|美国]]。在《[[:en:Life_as_a_House|生命如屋]]》的山姆·门罗之前，他扮演的都是次要[[角色|角色]]。他后来因饰演《[[星球大战|星球大战]]》前传系列电影的男主角[[安纳金·天行者|安纳金·天行者]]（[[達斯·維達|達斯·維達]]）和《[[时空骇客|时空骇客]]》男主角戴维·莱斯而获得国际声誉。

== 生平 ==
=== 早年 ===
海登·克里斯滕森于1981年4月19日出生于[[加拿大|加拿大]][[不列颠哥伦比亚省|不列颠哥伦比亚省]]最大[[城市|城市]][[温哥华|温哥华]]，父亲戴维·克里斯滕森是一位拥有[[丹麦人|丹麦]]血统的[[软件|软件]][[程序设计|程序设计]]员和企业[[公关|公关]]部主管，母亲阿丽娅则拥有[[瑞典人|瑞典]]和[[意大利人|意大利]]血统，专门为一些大公司老板写演讲稿<ref>{{cite web |url=http://www.telegraph.co.uk/arts/main.jhtml?xml=/arts/2002/03/13/bfhayd.xml |title=A force to be reckoned with |accessdate=2008-07-01 |date=2002-03-13 |work=[[The_Daily_Telegraph|Telegraph]]}}</ref>。海登有一个哥哥（托夫）、一个姐姐（海吉莎）和一个妹妹（凯伦）<ref name="people">{{cite web |url=http://www.people.com/people/archive/article/0,,20136997,00.html |title=Hayden Christensen: Actor |accessdate=2008-07-01 |date=2002-05-13 |work=People Magazine}}</ref><ref name="telegraph">{{cite web |first=John |last=Hiscock |url=http://www.telegraph.co.uk/arts/main.jhtml?xml=/arts/2005/05/13/,/bfhay13.xml |title= Skywalker comes down to earth |accessdate=2008-07-01 |date=2005-05-13 |work=The Daily Telegraph}}</ref><ref>{{cite web|url=http://movies.yahoo.com/movie/contributor/1800421140/bio |title=Hayden Christensen Biography |accessdate=2008-07-01 |publisher=[[Yahoo!|Yahoo!]] |deadurl=yes |archiveurl=https://web.archive.org/web/20060101122719/http://movies.yahoo.com/movie/contributor/1800421140/bio |archivedate=2006-01-01 }}</ref>。
