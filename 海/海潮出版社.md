'''海潮出版社'''，位于[[北京市|北京市]][[海淀区|海淀区]][[北京三环路|西三环中路]]19号，是[[中国人民解放军海军|中国人民解放军海军]]出版机构。

==简介==
海潮出版社成立于1986年，是中国人民解放军海军唯一的综合性出版机构<ref name=rmw>{{Cite web |url =http://military.people.com.cn/GB/52578/52579/225988/index.html  |title = 海潮出版社 |author =  |date = 2011-07-06 |publisher = 人民网 |language =  |accessdate =  |quote =  }}</ref>。起初“海潮出版社”和“海军出版社”是同一出版社的两个社牌，后来“海军出版社”社牌约在2006年左右停用<ref>李星 主编，大海的旋律——前进中的海军军乐团，海军出版社，2006年（内部出版）</ref>。海潮出版社和位于天津的[[中国人民解放军海军参谋部航海保证局|中国航海图书出版社]]共同构成“中国人民解放军海军出版社”（中国人民解放军海军出版社不是出版单位，只是冠以“出版社”名称的军内单位。海潮出版社、中国航海图书出版社是出版单位）<ref name=xiaoli>晓黎，以图书出版展海军雄风，中国图书商报2007年9月11日</ref>。截至2011年，海潮出版社出版图书6000多种，军事图书占70%以上<ref name=rmw/>。

2016年[[深化国防和军队改革|深化国防和军队改革]]前，海潮出版社由海军政治部主管，海军司令部主办<ref>{{Cite web |url =http://www.gapp.gov.cn/zongshu/serviceSearchResultbve.shtml?ID=13929  |title = 海潮出版社 |author =  |date =  |publisher =  中华人民共和国国家新闻出版广电总局|language =  |accessdate =2017-02-24  |quote =  }}</ref>。2016年3月底，中央军委印发《[[关于军队和武警部队全面停止有偿服务活动的通知|关于军队和武警部队全面停止有偿服务活动的通知]]》。2016年5月7日，军队和武警部队全面停止有偿服务试点任务部署会议举行，7个大单位、17个具体单位被列为试点单位，新闻出版等重点项目被纳入试点范畴。自2015年起，各军队和武警部队出版社的新书及新项目均停止审批<ref>{{Cite web |url = http://www.cbbr.com.cn/article/104077.html |title = 军队出版社将试点“全面停止”有偿服务 |author =  |date = 2016-05-10 |publisher = 中国出版传媒网 |language =  |accessdate =  |quote =  }}</ref>。

==报刊==
*《海军杂志》：1950年代初创刊，为月刊。和《[[人民海军_(报纸)|人民海军]]》报号称中国人民解放军海军的“一报一刊”<ref name=xiaoli/>。《海军杂志》是海军唯一的以军事工作为主的综合性刊物。自1979年10月复刊后，到1996年已出满200期，1996年[[中共中央政治局常委|中共中央政治局常委]]、中央军委副主席[[刘华清|刘华清]]为《海军杂志》题词：“贴近部队，坚持特色，把海军杂志办得更好。”<ref>《海军杂志》庆祝复刊200期，现代军事1996年08期</ref>2016年[[深化国防和军队改革|深化国防和军队改革]]前，《海军杂志》由海军政治部主管，海军司令部主办。海潮出版社社长兼《海军杂志》主编（例如海潮出版社社长陈立明就兼任《海军杂志》主编<ref name=xiaoli/>）。
