'''海地文化'''相當多元，主要偏向[[非洲|非洲]]的[[伏都教|伏都教]]（[[巫毒|巫毒]]）文化<ref>[http://www.onewitch.url.tw/voodoo.htm 巫毒教的起源與介紹] {{webarchive|url=https://web.archive.org/web/20090323063158/http://www.onewitch.url.tw/voodoo.htm |date=2009-03-23 }}</ref>，在[[法國|法國]][[海地歷史|殖民海地時期]]，當地曾經被大量傳入[[天主教|天主教]]的信仰習慣。

== 宗教與神話 ==
[[海地|海地]]的宗教信仰相當多元，其大多數是在[[法國|法國人]]的[[殖民|殖民]]海地時期所引進。而這些宗教也孕育出許多豐富的[[神話|神話]]故事，但較少為人知。現今海地的[[國教|國教]]是[[天主教|天主教]]。

|}經過調查，约80%的[[海地|海地人口]]宣称信仰[[天主教|天主教]]，20%的海地人口則是信仰[[基督教|基督教]]绝大多数目前从属于罗马天主教。与其它[[拉丁美洲|拉丁美洲]]国家类似，海地也曾是欧洲天主教国家的殖民地（[[法國|法國]]），因此天主教的信仰也就因此進入了海地。由於天主教的[[信仰|信仰]]觀念已在當地大多數人民的思想中深耕蒂固，所以在[[海地|海地]]正式獨立後，[[天主教|天主教]]很快的在[[海地政治|海地宪法]]中被[[海地國民議會|定为]]正式国教。至今约有80%至85的海地人为天主教徒。根据海地天主教会的统计，目前在海地的两个主教区内共有251个教区和1500个基督徒乡村社区。

另有相当数量海地人信奉[[伏都教|伏都教]]（又称巫毒教（Voodoo））这种起源自非洲的传统多神崇拜， 因法国人由非洲掳走大量黑人到法国的殖民地做奴隶<ref>[http://www.onewitch.url.tw/voodoo.htm 信仰巫毒教的人口] {{webarchive|url=https://web.archive.org/web/20090323063158/http://www.onewitch.url.tw/voodoo.htm |date=2009-03-23 }}</ref>，因而传入海地。但目前没有关于海地伏都教信徒的权威统计数字，具有不少爭議（海地人有句俗语，海地有80%天主教徒但有100%巫毒教徒）。

雖然[[巫毒|巫毒教]]在[[海地|海地]]之信仰人口還不確定，但是對於[[海地音樂|音樂]]與[[節慶|節慶]]部分影響程度比[[天主教|天主教]]更為深入和廣泛，甚至是兩者混合。<ref>《各國宗教與文化》p168</ref>。

=== 神話 ===
{{main|海地神話}}

'''海地神話'''（又稱[[美國|美國]][[巫毒|巫毒]]神話）是一種發源於[[海地|海地]]的[[神話|神話]]。海地神話起源於[[法國|法國]][[海地歷史|殖民海地時期]]，融合[[巫毒|巫毒教]]與[[基督教|基督教]]文化所發展出來，與[[達荷美神話|達荷美神話]]有幾分類似。在神話中的一些傳說也有受到[[印地安人|印地安人]]文化之影響。<ref>[http://www.webster.edu/~corbetre/haiti/voodoo/biglist.htm 海地、巫毒-神話故事] {{webarchive|url=https://web.archive.org/web/20060516165730/http://www.webster.edu/~corbetre/haiti/voodoo/biglist.htm |date=2006-05-16 }}</ref>

== 音乐 ==
{{main|海地音乐}}
海地人大多信仰祖先从非洲带到新大陆的多神教-伏都教（Voodoo），该教认为音乐是现实世界和理想世界间的桥梁和中介，这对海地音乐影响极为深远，海地流行的各种音乐体裁，差不多都与伏都教的宗教仪式以及社会礼仪有关。

在非洲音乐中，鼓具有特殊重要的地位。海地音乐最重要的乐器也是鼓，伏都教仪式用鼓乐伴奏。海地鼓乐队一般由三种大小不同的鼓和其他打击乐器组成，许多打击乐器就地取材、经过少许加工而制成的，如称为“擦擦”的摇晃器，便是装满干种子的葫芦，而“阿桑”则是挂着珠子的葫芦。<ref>*[http://gov.ht/ 認識海地音樂]{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}
</ref>

== 藝術 ==

== 烹饪 ==
[[File:Joumou01.jpg|thumb]]
{{main|海地烹饪}}
{{see also|法國烹飪|非洲烹飪}}

'''海地烹飪'''受到[[非洲菜|非洲菜]]與[[法國烹飪|法式烹飪]]影響極深，是由信仰[[伏都教|伏都教]]的非洲[[奴隸|奴隸]]和殖民者－[[法國|法國]]引進來。時至今日，這兩種飲食文化在[[海地|海地]]已經混合，演變成另一種獨特的[[飲食|飲食]]文化。海地的烹飪習慣也與一些[[加勒比海|加勒比海]]國家的烹飪文化有所類似。<ref>[http://www.everyculture.com/Ge-It/Haiti.html 海地的飲食]</ref>

大米和豆子是現在海地平民的主要[[食物|食物]]，其他常见食物和[[调味料|调味料]]包括[[豬|猪肉]]、[[羊|羊肉]]、鸡、[[鱼|鱼]]、[[玉米粉|玉米粉]]、[[秋葵|秋葵]]、[[茄子|茄子]]、[[番茄|番茄]]、[[辣椒|红辣椒]]、[[辣椒|绿辣椒]]、[[香蕉|香蕉]]（[[大蕉|大蕉]]）、[[芹菜|芹菜]]、[[洋葱|洋葱]]、芫荽、[[大蒜|大蒜]]。

=== 飲料 ===
海地也擁有相當豐富的飲料[[文化|文化]]，是在各個節慶中缺一不可的重要角色。目前主要的[[飲料|飲料]]種類有兩種，一種是[[啤酒|啤酒]]，另一種是非[[酒精|酒精]]的熱帶[[水果|水果]]飲品。
