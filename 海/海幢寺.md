'''海幢寺'''（幢，[[香港語言學學會粵語拼音方案|粵音]]：tong4，[[汉语拼音|拼音]]：chuáng），位于[[广州市|广州市]][[海珠区|海珠区]][[同福路|同福中路]]和[[南华路|南华中路]]之间，占地面积1.97万平方米。是广州四大名寺（[[光孝寺_(广州)|光孝寺]]、[[华林寺_(广州)|华林寺]]、[[六榕寺|六榕寺]]和'''海幢寺'''）之一，建于[[清朝|清]]初，距今300多年（曾毁于桂系军阀[[岑春煊|岑春煊]]之手）。该寺以保存有大量珍贵历史文物而闻名于世。寺里保存有[[澹为|澹为]][[和尚|和尚]]的墨迹与诗稿。澹为本是[[崇祯|崇祯]][[进士|进士]]，后削發为僧，在该寺长住。他的诗稿有反清情绪，当时无法刊刻，原稿完整无缺，至今仍成为该寺的法宝之一。

== 历史 ==
[[File:The_Sea-screen_Temple_at_Honam_Canton.png|thumb]]
[[File:Haizhuang_Si.jpg|thumb]]
海幢寺地处万松岭，该岭在历史上也建造过不少名寺，但已先后颓毁，其原址南汉时称为“千秋寺”，後廢為民居。（有說是廢為墓葬區）明代成為郭氏花園「蘭園」，至明末，光牟、池月兩位僧人向園主郭龍岳募緣得地建佛堂，依佛經「'''海幢比丘'''（[[梵語|梵語]]：{{IAST|Sa^gara-dhvaja}}）潛心修習《[[摩訶般若波羅蜜多心經|摩訶般若波羅蜜多心經]]》成佛」之意，將佛堂取名為海幢寺。<ref>{{cite book |author1=祝鵬 |title=廣東省廣州市佛山地區韶關地區沿革地理 |date=1984年10月 |publisher=學林出版社}}</ref><ref>{{cite book |author1=鄧端本 |title=廣州說古 |date=1987年10月 | publisher=廣東旅遊出版社}}</ref>

清[[順治|順治]]十二年（1655年），曹洞宗第三十三世空隱和尚在寺住持。此後，天然和尚、阿字和尚、古云和尚等名僧先後在此住持傳法。[[康熙|康熙]]五年（1666年）至康熙十八年的10多年間，阿字和尚在平南王[[尚可喜|尚可喜]]的大力支持下，購置寺旁田地，修建了大雄寶殿、天王殿等，院閣23座。乾隆三十一年（1766年），增建毘盧閣。[[道光|道光]]年間，南海人伍右肅施銀修建韋馱、伽藍兩殿。[[光緒|光緒]]年間，寺內有僧人300多，一時高僧云集，盛極一方。寺內的四大天王、十六尊者和慈航普渡，與眾不同的男身觀音極具特色。海幢寺既成為弘揚佛法之所，又是[[廣州人|廣州人]]旅遊勝地之一，寺內曾有「古寺參雲、珠江夜月、飛泉卓石、海日吹霞、江城夜雨、石磴叢蘭、竹韻幽鐘、花田春曉」八大景點。

[[File:Haichuang_Si_2.jpg|thumb]]
[[File:HONAM_TEMPLE,_CANTON.jpg|thumb]]
清[[巡撫|巡撫]]劉某捐資為之正式建成山門，在今南華中路處，以前該處是「海皮」（[[珠江|珠江]]邊）。18世紀是海幢寺的鼎盛時期，寺院規模超過現址的三倍。前臨珠江，後倚萬松嶺（今金粟園至龍福西二巷），寺院後東界到達現今的同福上街。最初落成的大雄寶毆闊七楹（約35米），高三尋（合丈四尺多，約7、5米）。後來又建成規模更加宏偉的藏經閣以及天王毆、韋馱殿、伽藍毆等一系列殿堂及配套設施。清《康熙鼎建碑》稱此閣「碧瓦朱甍，侵霄爍漢」。此後還相繼建成叢觀、西彈、鏡空、松雪、悟閒，畫禪等堂宇，地藏、諸天、聞清鐘等樓閣，惜陰、就樹等廊軒，幢隱盧、空緣彈等館舍，還有普同塔、瘞鹿塚等僧侶墓園，寺後有鬆園、寧福莊、瘞鹿亭等。規模之大，為廣州寺廟之冠，也是“四大丛林”之一。

[[中華民國|中華民國]]建國初期因開闢[[同福路|同福路]]把寺園腰斬為南北兩部分。民國十七年（1928年），[[廣州市政府|廣州市政府]]把海幢寺部分房地劃出建「河南公園」，民國廿二年（1933年）9月改稱「海幢公園」。至[[中華民國政府|中華民國政府]]遷往[[台灣|台灣]]前夕，海幢寺的許多殿房因開馬路而被清拆，只保留大雄寶殿、天王殿、舍利塔殿等部分建築。

[[File:Entrance_to_Hoi_Tong_Monastery,_1903.JPG|thumb]]
[[File:Monks_at_Hoi_Tong_monastery,_Ho_Nam,_China_(1903).jpg|thumb]]。]]
1949年[[中华人民共和国|中华人民共和国]]成立后，对公园进行过多次修葺，寺內仍有乾慧、素仁兩位僧人居住管理。

1966年[[文化大革命|文化大革命]]期间，海幢寺惨遭浩劫，四大天王殿被清拆，寺內所有佛像被砸爛，文物也大多流失損毀。1980年代末，大雄寶殿成為音樂茶座和舞廳。

1992年，廣州市佛教協會收回被佔用的殿堂。1993年1月，成立海幢寺重修委員會，同年3月7日，经[[广州市人民政府|广州市人民政府]]批准，海幢寺舉行灑淨儀式，恢復正常的宗教活動，由市佛教協會副會長新成法師任方丈，同時負責海幢寺的重修復建工作。重修項目共投資1000多萬元[[人民幣|人民幣]]，修復了大雄寶殿，重建了天王殿、塔殿、放生池和7层僧舍、藏经阁综台楼等。1997年又在大雄宝殿前空地东侧新建一座名为宝严塔的石塔。

[[File:Great_Temple_at_Honan,_Canton.png|thumb]]
現在寺內有30多位僧人，大雄寶殿供奉的3座銅佛，高10多米，稱三世佛，即過去、現在、未來佛。過去佛為[[燃燈佛|燃燈佛]]，現在佛為[[釋迦牟尼|釋迦牟尼]]，未來佛為[[彌勒菩薩|彌勒菩薩]]。重建後的天王殿內四大天王，其高度、體積是廣東省境內寺院之最。 1993年8月，海幢寺被定為廣州市文物保護單位。

== 建筑及特色 ==
该寺的大雄宝殿及塔殿，均坐南朝北，面对[[珠江|珠江]]。寺内的天王殿，原有四大[[金刚|金刚]]和三玄佛像等。四大金刚，以高大威猛，元神毕露而闻名，但在文化大革命期間被毁，神像為後來重塑。该寺的石造舍利塔构造别致。寺里有300多年的鹰爪兰一株，据说是明末郭岳龙“兰园”的遗物，年代比海幢寺还要久远，因而有“未有海幢，先有鹰爪”之说，此花现被铁栏杆保护。原寺里还有金鱼池、巨石等陈设。该巨石原为清代富商伍某购赠，石状如“猛虎回头”。抗战期间，日军已准备将其运往[[日本|日本]]，已经搬动，但仍未成事時日本就宣布投降。1951年，该石归回海幢寺。

== 现状 ==
[[File:Haizhuangsi_tonggao.jpg|thumb]]
由于历史缘故，十年浩劫文革期間，海幢寺受到毁滅性破壞，被辟为公园，而当时海珠区人民政府与寺院议定以绿篱为界，划分海幢寺和公园，寺廟位於公园的西侧。此外，当时公园内还建有儿童游乐园、酒家等。然而寺内却没有鼓楼、山门、牌坊等建筑，形成了“寺不像寺，园不像园”的尴尬局面。<ref name="name1">[http://www.folou.com/thread-96394-1-1.html 广州一公园改为寺庙（海幢寺），不需再购门票] {{webarchive|url=https://web.archive.org/web/20100217043159/http://www.folou.com/thread-96394-1-1.html |date=2010-02-17 }}，信息时报</ref>2006年海幢寺得到海珠区政府的批准将寺交回海幢寺恢复寺院管理，并在2008年11月中旬起开始施工，重建寺门、牌坊和钟楼等建筑，并复名海幢寺，不再使用海幢公园这一地名。<ref name="name2">[http://news.sina.com.cn/o/2008-11-26/133914789017s.shtml 海幢公园更名“海幢寺”]，新浪转载羊城晚报</ref>但同时《羊城晚报》、《信息时报》等媒体报道海幢寺因此而免门票<ref name="name1"/><ref name="name2"/>，后海幢寺澄清并非免门票，而是记者误解海幢寺免门票只是针对持有合法证件、真正皈依佛门的人士<ref>[http://www.ycwb.com/ePaper/ycwb/page/1/2009-02-11/A10/5721234325518698.pdf 海幢寺澄清“免收门票”一说 称之前媒体报道有误！ 普通人烧香拜佛，2元门票不能免！] {{webarchive|url=https://web.archive.org/web/20090220090835/http://ycwb.com/ePaper/ycwb/page/1/2009-02-11/A10/5721234325518698.pdf |date=2009-02-20 }}羊城晚报报道</ref>。

目前寺廟已實施免費入場。

== 英文名稱及「幢」字發音 ==
{{further|广州地名}}
海幢寺的英文名稱在1949年前使用「{{lang|en|Honam Temple}}」（即「河南寺」）或傳統的[[粵語羅馬化方案|粤式]][[郵政式拼音|郵政式拼音]]「{{lang|en|Hoi Tong Monastery}}」，現時翻译作「{{lang|en|Hoi Tong Monastery}}」。按《[[新華字典|新華字典]]》，「幢」在普通話是多音字，發「{{lang|en|chuáng}}」音有「一種刻有佛號或經咒的石柱」的意思，符合詞源「海幢比丘」的讀音。而發「{{lang|en|zhuàng}}」是量詞和形容詞，前者指房屋的棟數，譬如「一幢樓」，後者指愚昧無知。廣州經常有不少巴士公司在報「海幢公園」站時國粵雙語皆讀錯音，粵語錯讀成「童」，正確為「唐」音，而普通話錯讀成「撞」，正確應為「床」音。
