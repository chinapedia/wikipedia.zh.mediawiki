'''海底'''是指[[麻将|麻将]]（包括日本牌、香港牌等）中一局中的最後一次摸牌和打牌。在海底自摸到的牌称'''海底撈月'''，接着打出的牌是'''河底撈魚'''。在日本麻雀，海底牌是除了[[牆牌#王牌|王牌]]之外，[[牆牌|牆牌]]（山）的最後1枚。摸到海底牌的玩家或者宣言自摸，或者打出1张牌（河底牌）。如果其余的玩家没有人对这张牌宣言糊牌，这局就会结束（[[流局|流局]]）。

== 限制 ==
;海底不能槓
:不管是[[槓#暗槓|暗槓]]还是[[槓#加槓|加槓]]，都不能用海底牌进行。这是因为如果用海底牌开杠而取得1枚[[嶺上牌|嶺上牌]]，本来应该有14枚的王牌{{注|此處王牌指日本麻雀中開門位置右側七墩共14張牌。其中第三墩的上位牌會翻開作為懸賞指示牌。}}就会变成只有13枚。
;河底不能吃碰
:不能对河底牌进行吃、碰、明杠。这是因为如果对海底牌进行吃、碰、明杠时，会让吃、碰、明杠者打出的牌重新成为河底，从而无法确定河底牌是哪张。
;在很接近海底时不能立直
:現在一般规则中，要求牌山中最少有4枚牌才能宣言[[立直|立直]]（而在[[三人麻将|三人麻将]]当中至少要3枚）。也就是说，要宣言立直，需要自己还有最少一次自摸的机会。

== 海底摸月 ==
'''海底摸月'''是指自摸海底牌和了，1翻，略称'''海底'''。如果在[[門前清|門前清]]的状态下达成，还能得到[[門前清自摸和|門前清自摸和]]的1翻。另外，即使是門前清不成立，也没有役的[[聴牌#形式聴牌|形式听牌]]，也能通过海底摸月和了。

自摸在海底牌之前的那张牌并开杠，使用嶺上牌和了的时候，岭上牌就成为那一局中摸到的最后的一张牌。不过，只有在直接自摸海底牌和了的情况下海底摸月才成立，使用岭上牌和牌的时候海底捞月不成立。换句话说，海底摸月和[[嶺上開花|嶺上開花]]不能复合。

亦有地方役為「{{lang|ja|石の上にも三年}}（置於石上三年，意譯為鐵杵磨成針）」，海底摸月若是與[[双立直|雙立直]]複合，則計為役滿，河底撈魚則不行。

== 河底捞鱼 ==
'''河底撈魚'''在荣和河底牌时成立，1翻。虽然应该略称'''河底'''，不过经常和海底摸月一样略称「海底」或是稱呼為「海底和」。
若此時和牌得點很少，例如閒家1翻30符只有1000點，則可考慮放棄'''河底撈魚'''荣和，待'''流局'''有多人不聽牌的話可獲得更多點數（1500/3000點）。
