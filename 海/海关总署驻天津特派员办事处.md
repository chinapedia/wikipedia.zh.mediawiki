'''海关总署驻天津特派员办事处'''成立于2002年7月10日，是[[中华人民共和国海关总署|中华人民共和国海关总署]]设立在[[天津市|天津市]]的派出机构，该办事处直属海关总署领导。

== 主要职责 ==
海关总署驻天津特派员办事处的主要职责为负责管辖区域内直属海关的执法督查、干部考察、廉政督察和对署管干部进行诫勉谈话等工作。<ref>[http://www.people.com.cn/GB/shizheng/3586/20020710/773165.html 海关管理体制的重大改革 海关总署驻天津特派员办事处成立，人民网，2002年7月10日]</ref>

== 管辖范围 ==
海关总署驻天津特派员办事处的管辖范围包括[[北京海关|北京海关]]、[[天津海关|天津海关]]、[[石家庄海关|石家庄海关]]、[[太原海关|太原海关]]、[[呼和浩特海关|呼和浩特海关]]、[[满洲里海关|满洲里海关]]、[[大连海关|大连海关]]、[[沈阳海关|沈阳海关]]、[[长春海关|长春海关]]、[[郑州海关|郑州海关]]、[[哈尔滨海关|哈尔滨海关]]、[[兰州海关|兰州海关]]、[[银川海关|银川海关]]、[[乌鲁木齐海关|乌鲁木齐海关]]、[[海关总署秦皇岛培训学校|海关总署秦皇岛培训学校]]。<ref>{{Cite web |url=http://tianjin_sup.customs.gov.cn/publish/portal92/tab4665/ |title=海关总署驻天津特派员办事处简介，海关总署驻天津特派员办事处，于2011-7-7查阅 |access-date=2011-07-11 |archive-url=https://web.archive.org/web/20131203052117/http://tianjin_sup.customs.gov.cn/publish/portal92/tab4665/ |archive-date=2013-12-03 |dead-url=yes }}</ref>
