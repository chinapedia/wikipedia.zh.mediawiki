'''海外兵团'''或称'''海外军团'''，在中文通常是指[[归化|归化]]它国国籍的原[[中华人民共和国国籍|中华人民共和国国籍]]运动员，近年亦指在海外任职的教练<ref>{{Cite web |url=http://sports.sohu.com/20140506/n399168389.shtml |title=“海外兵团”成过去式？ 国乒输出教练变升级版 |accessdate=2015-09-21 |author=作者：郭健 |date=2014-05-06 |work= |publisher=搜狐网 |language=Zh-hans }}</ref>。在其它国家亦存在类似现象。

== 中國 ==
自中华人民共和国建国起，对[[中华人民共和国体育|运动员]]培训制度采用[[举国体制|举国体制]]。随后，中国运动员在[[乒乓球|乒乓球]]、[[举重|举重]]、[[跳水|跳水]]等运动项目上取得了全球领先的优势。1978年，[[中国大陆|中国大陆]]开始[[改革开放|改革开放]]，[[中华人民共和国公民|中国公民]][[移民|移民]]逐渐宽松。从1980年代起，有众多运动员移民它国，并改换国籍，代表别国参加比赛。同时，[[中华人民共和国国籍法|中华人民共和国国籍法]]不承认[[双重国籍|双重国籍]]，入籍它国，即不能保留中华人民共和国国籍。1990年代，代表日本参赛的前中国籍乒乓球运动员[[何智丽|何智丽]]曾在中国引起巨大的反响。对海外兵团亦有相反的评价。

[[中国乒乓球|乒乓球]]运动在中国有[[国球|国球]]之称。中国乒乓球队在各类国际比赛中，占有绝对优势。2012年的报道，前中国籍乒乓球员[[陈卫星|陈卫星]]表示，海外兵团在推动一个运动项目的国际交流发挥的积极作用。时任中国乒乓球男队主教练的[[劉國梁|劉國梁]]表示，没有海外兵团，全球乒乓球可能会变成一潭死水<ref>{{Cite web |url=http://zqb.cyol.com/html/2012-08/05/nw.D110000zgqnb_20120805_1-03.htm |title=从海外兵团到海外使者 |accessdate=2015-09-21 |author=特派记者：慈鑫 |date=2012-08-05 |work= |publisher=中青在线，来源：中国青年报|language=Zh-hans }}</ref>。

== 归化运动员 ==

{{main|归化运动员}}

這種情形並非只有中國才有，在一個國家對獎牌的渴望，選手對金錢的需求的情況下，海外兵团也會在其他國家發生。另外各項對同一個國家的人數限制也是原因之一。<ref>{{cite news|title= 運動／國際田總禁止轉國籍　日本批最該管的是中國桌球|url=http://www.setn.com/News.aspx?NewsID=222848 |accessdate=2017-02-08 |agency= |date=2017-02-08 }}</ref>

例如有些[[肯亞|肯亞]]出生的運動員在奧運會上會代表其他國家出賽而非自己的國家出賽，尤其大部分以代表[[美國|美國]]和[[巴林|巴林]]出賽最多。在2016年奧運會，共有約20名運動員包含多個獎牌得主。<ref>{{cite news|title=杜絕轉籍撈金　IAAF將限縮轉籍規定 |url=http://www.appledaily.com.tw/realtimenews/article/new/20170208/1051256/ |accessdate=2017-02-08 |agency= |date=2017-02-08 }}</ref>

巴西足球員轉換國籍者多達5000人以上；世界桌球錦標賽，超過150名中國選手代表其他國家參賽。大批肯亞長跑選手轉籍至中東國家，連名字都改成伊斯兰教名字。

部分國家也是透過這種動作，在多個項目達到理想的成績。
=== 其他關係 ===
* 當一個國家全部或部分項目受到不定期限的禁賽，該國就有可能有運動員大量的轉籍求出路。造成人才外流，該國體育也隨之下滑。如1992年的南斯拉夫，2016年的俄羅斯田徑及傷健運動員及部分項目。<ref>俄羅斯興奮劑風波，醜聞還是陰謀？</ref>而那些轉籍的運動員當轉到其他國家時，可能使比賽品質下降。

=== 限制 ===
有些項目限制這種情形，如足球和冰球一經代表國家出賽就難以轉籍，桌球21歲以後禁止轉籍，部分其他項目在轉籍3年後才能代表新國家參賽。

=== 利弊 ===
====正面效果====
* 可以改善部分國家體育人才不足的問題。
====負面效果====
* 由於這樣會使個人項目有如足球的[[轉會_(足球)|轉會市場]]、洲際賽事（如：[[亞洲運動會|亞洲運動會]]）品質不純等問題，而產生大規模的議論。
* 成為部分較富裕的國家提振體育成績的工具。
* 當這些運動員的原籍國家被整隊或部分禁賽時，可能激起他們的對「祖國」的同情導致比賽混亂，如持被整隊禁賽國家的國旗、攻擊與其敵對國家選手的行為等比賽單位不允許的行為。

=== 誤解 ===
這些情形通常不會被視為海外兵團的一部份：
* 前國家解體，由新國家代表繼續參加，如[[蘇聯|蘇聯]]、[[南斯拉夫|南斯拉夫]]
* 在參賽年齡前移民到代表國家的國家，[[馬塞爾·德塞利|馬塞爾·德塞利]]、[[帕特里克·维埃拉|帕特里克·维埃拉]]。
* 可能出生地、親戚、居住地而代表國家，[[凯文·库兰伊|凯文·库兰伊]]、[[欧文·哈格里夫斯|欧文·哈格里夫斯]]，通常代表實力較差的國家參賽。

當然也有拒絕成為海外兵團的例子。如[[利昂内尔·梅西|-{zh-hans:梅西; zh-hk:美斯; zh-tw:梅西;}-]]雖然居於[[巴塞隆拿|巴塞隆拿]]，并于2005年9月26日获得西班牙国籍，但他並沒有代表[[西班牙國家足球队|西班牙國家足球队]]出賽，而選擇為出生國[[阿根廷國家足球隊|阿根廷]]披甲。
