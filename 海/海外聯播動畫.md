'''海外聯播動畫'''，是指在海外地區進行「'''同步'''」、「'''同週'''」或「'''隔週'''」聯合首播的日本動畫作品。

== 概要 ==
*日本於2008年首次與美國代理商[[Funimation|Funimation]]合作，在[[Crunchyroll|Crunchyroll]]同步播出《[[迷宮塔_(動畫)|迷宮塔 ～烏魯克之盾～]]》。2009年推出的《[[花冠之淚|花冠之淚]]》則是第一部由原播出地與海外電視台同步首播的動畫；之後日本版權業者也與海外方面合作（或授權），陸續推出聯合首播動畫，除了能使海外地區的觀眾能早日收看最新播出進度之外，也能減少非法盜錄、非法散布帶來的問題。[[TBS電視台|TBS電視台]]稱此作法為「預先銷售」<ref>[http://www.tbs.co.jp/eng/chinese/ourbusiness/ob_4.html TBS电视台海外节目营销的举措]</ref>或「Near-Simulcast of Anime Overseas」（海外準同步動畫）<ref>[http://www.tbs.co.jp/eng/ourbusiness/ob_4.html International Program Sales - TBS]</ref><ref>TBS稱「[http://www.tbs.co.jp/eng/programsales/news/en2010040811.html Near-simulcast ]」為「[http://www.tbs.co.jp/eng/programsales/news/jp2010040811.html 同日放送]」。</ref>。

*在中國大陸由於[[視訊|視訊]]非一般電影、電視節目及出版品，可避開官方事前審查機制，2011年底[[土豆網|土豆網]]與[[東京電視台|東京電視台]]合作首例一開，各大視訊網站紛紛自購或跟台港代理商聯合首播動畫。

== 播出型態 ==
# 此一類型動畫經由海外的電視台或視訊網站合作播放。
# 由於與海外聯合播出的關係，仍免不了內容牴觸海外當局訂定的尺度規範而遭刪剪的情形<ref>例如《鋼之鍊金術師 BROTBERHOOD》在海外聯播時，台灣[[Animax|Animax]]版本刪除入浴畫面，香港[[TVB|TVB]]也剪掉抽菸畫面。</ref>。
# 因為與日同步首播（大多都是日方首播後七天內），一般為原音配音版本。一般而言，在海外電視台播出後，通常會在事後重播當地配音的版本<ref>香港[[J2|J2]]的聯合首播動畫比日本晚大約兩到三週播出，令TVB配音組有時間配音，在播出時以粵日雙語播放。</ref>。
# 若遇到日方播放計劃延後的情形，聯播單位也會同步延後。

== 合作方式 ==
日本版權業者運作模式如下：
#直接與海外電視台合作──如《[[花冠之淚|花冠之淚]]》、《[[犬夜叉#犬夜叉_完結篇|犬夜叉 完結篇]]》及《[[學生會長是女僕|學生會長是女僕]]》。
#經海外代理商合作、由電視台配合播出──如《[[鋼之鍊金術師_FULLMETAL_ALCHEMIST|鋼之鍊金術師 BROTBERHOOD]]》、《[[黑執事|黑執事II]]》。
#經海外代理商合作、以[[網路|網路]]播出──如《鋼之鍊金術師 BROTBERHOOD》、《[[學園默示錄|學園默示錄]]》、《[[祝福的钟声|祝福的钟声]]》與《[[傳說的勇者的傳說|傳說的勇者的傳說]]》。
#直接授權海外視訊網站播出──如《[[天降之物|天降之物]]》、《[[玩伴貓耳娘|玩伴貓耳娘]]》、《[[HEROMAN|HEROMAN]]》和《[[我的妹妹哪有這麼可愛！|我的妹妹哪有這麼可愛！]]》、《[[咎狗之血|咎狗之血]]》。
