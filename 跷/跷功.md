'''跷功'''是中国[[戏曲|戏曲]]中特有的表演技能，演员穿上特制的跷鞋模仿[[裹脚|小脚]]女性，仅用[[脚趾|脚趾]]承重<ref name="a2"/>。

[[清朝|清朝]][[康熙|康熙]]十年（1671年），[[清朝政府|政府]]开始禁止女性登台表演<ref>{{Cite web |url=http://www.xijucn.com/html/bangzi/20091110/12214.html |title=戏剧里的女演员 |accessdate=2014-11-23 |author=甄光俊 |date= |work= |publisher=中国戏剧网|language=Zh-hans }}</ref>。女性角色则由男性[[反串|反串]]。男性演员，尤其是[[京剧|京剧]][[男旦|男旦]]，需要通过练习跷功来模仿女性的步态，虽然角色本身可能并非缠足时代的女性。在某些剧目中，跷功被用以表现女性的[[性特征|性特征]]。譬如《[[战宛城|战宛城]]》中，饰演邹氏的演员需要向观众露出跷脚，用以表现[[寡居|寡居]]的邹氏[[思春|思春]]<ref name="a1">{{Cite web |url=http://ent.sina.com.cn/j/2010-10-22/15233122367_7.shtml |title=《新舞台》创刊号：京剧男旦的“单传”时代(7) |accessdate=2014-11-23 |author=闫平 |date=2010年10月22日|work= |publisher=|language=Zh-hans }}</ref>，即使邹氏这一角色是一位[[东汉|东汉]]女性<ref>邹氏，即[[張濟_(武威)|张济]]妻，後與[[曹操|曹操]]發生性關係。</ref>。跷功的练习过程几乎如同缠足一样痛苦<ref>{{Cite web |url=http://www.historychina.net/qsyj/ztyj/sxwh/2005-08-29/24992.shtml |title=从科学话语到国家控制——“缠足”美丑与身体政治（上） |accessdate=2014-11-23 |author=杨念群 |date= |work= |publisher=中华文史网，文章来源：汪民安编《身体的文化政治学》，河南大学出版社，2004年|language=Zh-hans }}</ref>，同时，与缠足一样，练习跷功会导致脚部变形<ref name="a1"/>。

[[晚清|晚清]]时，女演员开始公开登台表演，到民国初年，中国女性已开始[[放足|放足]]，演员仍需从幼时开始练习跷功。中华人民共和国建国后，跷功表演已经绝迹。与传统戏曲相反，做为更受欢迎的娱乐形式，宋朝以后题材的[[电影|电影]]和[[电视剧|电视剧]]中，鲜见演员表现女性的小脚。近年来，有些女性旦角员开始重新练习跷功。2004年，有剧团在表演清朝题材的剧目时恢复这种表演，全场女性角色穿跷。进行这种表演同样导致演员脚部受伤、变形<ref name="a2">{{Cite web |url=http://ent.sina.com.cn/2004-02-06/1051293582.html |title=新缠天足再现跷功奇技(图) |accessdate=2014-11-23 |author= |date=2004-02-06|work= |publisher=新浪网，来源：长江日报|language=Zh-hans }}</ref>。
