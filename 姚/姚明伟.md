'''姚明伟'''（{{bd|1947年2月||2001年|10月30日}}）祖籍[[安徽省|安徽省]][[贵池县|贵池县]]，生于[[河北|河北]][[阜平|阜平]]。原机械工业部副部长、党组成员<ref name=zhuidao>{{Cite web |url =  http://www.people.com.cn/GB/paper1668/4745/519328.html|title =姚明伟同志追悼会在京举行  |author =  |date =  |publisher =中国汽车报2001年11月14日，第1版  |language =  |accessdate =  |quote =  }}</ref>，前中共中央政治局常委、国务院副总理[[姚依林|姚依林]]唯一的儿子。

==生平==
姚明伟早年在[[北京市第三十五中学|北京市第三十五中学]]学习，在校期间曾任[[任志强|任志强]]所在初一班级的辅导员。姚明伟上完高三后去[[越南|越南]]学习。任志强所在班级的辅导员此后先后为蒋小泉、[[王岐山|王岐山]]<ref name=sina>{{Cite web |url = http://finance.sina.com.cn/360desktop/zl/lifestyle/20131028/101717135339.shtml |title =任志强：初中辅导员是王岐山  |author =  |date =2013-10-28  |publisher = 新浪 |language =  |accessdate =  |quote =  }}</ref>。

姚明伟曾在[[中华人民共和国第一机械工业部|中华人民共和国第一机械工业部]]工作。后历任[[中国机械设备进出口总公司|中国机械设备进出口总公司]]副总经理、总经理，[[中华人民共和国机械电子工业部|中华人民共和国机械电子工业部]]国际合作司司长，深圳中机实业开发中心主任，中国机械工业国际合作咨询服务中心主任，[[中华人民共和国机械工业部|中华人民共和国机械工业部]]副部长、党组成员，[[国家机械工业局|国家机械工业局]]副局长、党组成员，[[中国机械工业联合会|中国机械工业联合会]]党委书记兼会长<ref name=zhuidao/>。

2001年10月30日，姚明伟在北京病逝，享年54岁。2001年11月8日，姚明伟的追悼会在北京举行<ref name=zhuidao/>。
