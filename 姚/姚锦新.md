'''姚锦新'''（{{bd|1911年|10月25日|1992年|6月11日}}）祖籍[[安徽省|安徽省]][[贵池县|贵池县]]，生于[[上海|上海]]，[[中华人民共和国|中华人民共和国]]音乐教育家、音乐理论家<ref name=chentong>沉痛悼念姚锦新教授，中央音乐学院学报1992年第03期</ref><ref name=guichi>{{Cite web |url = http://www.ahgc.gov.cn/402881e5463c9c1e01463cb4a2df0064/201409/01223411jkt4.html |title = 姚锦新 |author =  |date = 2012-07-12 |publisher = 中国贵池 |language =  |accessdate =  |quote =  }}</ref>。

==生平==
1917年，姚锦新六岁丧父<ref name=chentong/>。随母亲蔡亦民和刚3个月大的弟弟[[姚依林|姚依林]]投奔时任[[江苏省|江苏省]][[武进县|武进县]]知事的二伯父[[姚绍枝|姚绍枝]]<ref>{{Cite web |url =  http://dangshi.people.com.cn/GB/85038/10313886.html|title = 姚依林的早年岁月和他的大家族 |author =水新营、支锐  |date =  |publisher = 党史纵览2009年第8期 |language =  |accessdate =  |quote =  }}</ref><ref name=guichi/>。后举家迁回上海，靠母亲当家庭教师和父亲的遗产生活<ref name=guichi/>。1927年，进入上海[[中西女塾|中西女塾]]学习<ref name=chentong/>。1928年11月，上海举办纪念[[舒伯特|舒伯特]]逝世一百周年的音乐节，其间举办了音乐比赛，中西女塾派姚锦新参加钢琴比赛，获得三等奖。1929年底，[[英国|英国]][[三一音樂學院|圣三一学院]]音乐考级委员会在上海设钢琴等级考试，中西女塾学生姚锦新和萧嘉珍参加了钢琴高级考试。姚锦新的分数是[[亚洲|亚洲]]地区第一名，获得了在[[伦敦|伦敦]]圣三一学院学习一年的奖学金（但她此后未去圣三一学院），这也是由中国教师（中西女塾钢琴教师[[邱贞蔼|邱贞蔼]]）教出的学生首次获殊荣。1930年1月18日，姚锦新举办了毕业钢琴音乐会，结束了在中西女塾琴科的学习<ref name=zxnsh/>。

1930年，姚锦新考入[[北平|北平]][[清华大学|清华大学]]主修外语。1932年，留学[[德国|德国]]，曾在[[柏林|柏林]]国立音乐院主修[[钢琴|钢琴]]，兼修作曲理论。1939年学成归国，因正值[[抗日战争|抗日战争]]全面爆发初期，被迫滞留[[香港|香港]]九个月后转赴[[美国|美国]]<ref name=chentong/>。在香港期间，在[[半岛酒店|半岛酒店]]举办钢琴独奏会<ref name=guichi/>。到美国后，起初在[[耶鲁大学|耶鲁大学]]音乐院主修理论作曲，后转学到[[加利福尼亚大学|加利福尼亚大学]]音乐院继续作曲理论的学习。其间，1943年秋至1946年冬，还在美国[[第二次世界大战|第二次世界大战]]期间在[[旧金山|旧金山]]设的国家广播电台从事音乐节目编制和华语广播工作。1947年归国，1948年春在清华大学音乐室任教。同年7月初，抵达[[华北解放区|华北解放区]]，在[[华北大学|华北大学]]文艺学院音乐系任教授。1949年春，随华北大学师生进入北平，在人民文工团任创作员<ref name=chentong/>。1949年，[[马思聪|马思聪]]、[[吕骥|吕骥]]、[[贺绿汀|贺绿汀]]、姚锦新获[[新政治协商会议筹备会|新政治协商会议筹备会]]第六小组聘为国歌词谱评选委员会顾问，参加[[中华人民共和国国歌|中华人民共和国国歌]]词谱评选工作<ref name=guichi/>。

1950年冬至1951年冬，姚锦新在[[天津人民艺术剧院|天津人民艺术剧院]]专门从事音乐创作。1951年冬至1953年10月，在[[中央音乐学院|中央音乐学院]]钢琴系任教，后来转到作曲系任教授，兼任作品分析教研室主任。1953年11月至1956年冬，在[[中华人民共和国文化部|文化部]]电影事业管理局音乐处专门从事音乐教育工作<ref name=chentong/>。1957年调回中央音乐学院作曲系任教。1964年调任[[中国音乐学院|中国音乐学院]]教授<ref name=zxnsh>{{Cite web |url =http://www.cssn.cn/ysx/ysx_ylx/201310/t20131026_611485.shtml  |title =中国近代女性音乐教育的先锋——上海中西女塾音乐教育研究（1917-1930）  |author = 陈晶 |date =  |publisher =  中央音乐学院学报(京)2010年4期，第78-86页|language =  |accessdate =  |quote =  }}</ref>。

1992年6月11日，姚锦新在北京病逝，享年81岁<ref name=chentong/>。

==著作==
*英文著作：《[[凤阳花鼓|凤阳花鼓]]及其它中国歌曲》
*译著：《[[勋伯格|勋伯格]]：月迷彼埃罗》、《器乐》、《音乐美学若干问题》（合译）

==家庭==
*父：[[姚捷勋|姚捷勋]]，包括其在内兄弟共6人，他排行第四。
*母：蔡亦民
*弟：[[姚依林|姚依林]]
*丈夫：[[陈世骧_(文学家)|陈世骧 (文学家)]]。民国29年（1940年），姚锦新在香港举办个人钢琴独奏会，与[[乔冠华|乔冠华]]邂逅恋爱。不久因战争阻隔等原因，姚锦新滞留[[华盛顿哥伦比亚特区|华盛顿哥伦比亚特区]]而与陈世骧结婚。4年后即与陈世骧离婚，未再婚。晚年姚锦新一直与养女共同居住在北京<ref name=guichi/>。
