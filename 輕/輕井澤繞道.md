'''輕井澤繞道'''（{{lang|ja|軽井沢バイパス}}）是[[日本|日本]][[長野縣|長野縣]][[北佐久郡|北佐久郡]][[輕井澤町|輕井澤町]]的[[國道18號|國道18號]][[繞道道路|繞道]]。由[[長野縣|長野縣]][[北佐久郡|北佐久郡]][[輕井澤町|輕井澤町]][[長倉字|長倉字]][[大小丸|大小丸]]至[[長野縣|長野縣]][[北佐久郡|北佐久郡]][[輕井澤町|輕井澤町]][[長倉字|長倉字]][[屋敷裏|屋敷裏]]。
全長6.7km。
<div style="clear: both;"></div>

== 概要 ==
* 起點 : 長野縣北佐久郡輕井澤町長倉字大小丸
* 終點 : 長野縣北佐久郡輕井澤町長倉字屋敷裏
* 全長 : 6.7[[km|km]]（繞道部為5.9 km）
* 道路規格 : 第3種1級
* 設計速度 : 80[[km/h|km/h]]
* 標準幅員 : 22[[m|m]]
* [[車線|車線]]數 : 2車線

[[日本道路公團|日本道路公團]]建設的[[碓冰繞道|碓冰繞道]]長野縣側延伸、當時由[[建設省|建設省]][[地方整備局|關東地方建設局]]長野國道工事事務所整備。

== 主要建造物 ==
* 借宿誇道橋
* 鳥井原橋
* 湯川橋
* 雨宮橋

== 沿革 ==
* 1969年（昭和44年）12月 : 建設開始。
* 1972年（昭和47年）7月 : [[車道|車道]]開通。
* 1973年（昭和48年）8月 : 步道・自轉車道建設開始。
* 1977年（昭和52年）11月 : 步道・自轉車道完成。
* 1987年（昭和62年）5月 : 綠化工程完成。
