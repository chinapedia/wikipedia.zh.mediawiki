'''养三站'''，位于[[北京市|北京市]][[石景山区|石景山区]][[广宁街道|广宁街道]]石景山脚下，是[[丰沙铁路|丰沙铁路]]上的一个小火车站。该站距离[[北京站|北京站]]35[[公里|公里]]，[[沙城站|沙城站]]86公里。

==简介==
1989年，石景山热电厂北侧的[[丰沙铁路|丰沙线]]建起'''石景山热电厂站'''，主要任务是接卸电煤。石景山热电厂站后改名为'''养三站'''，因位于{{站|养马场}}和{{站|三家店}}之间而得名。该站以电煤运输为主，站内每日通行120对客货列车，但没有客运列车停靠<ref>{{cite news|url=http://www.chinanews.com/sh/2015/02-17/7070532.shtml|title=铁路小站值班员：十小时发两千余次指令|publisher=[[新京报|新京报]]|date=2015-02-17|accessdate=2015-09-27}}</ref><ref>{{Cite web |url = http://news.sohu.com/20151220/n431929759.shtml |title =北京石景山最后一个工业火车站将关停 员工不舍  |author =  |date =2015-12-20  |publisher = 搜狐 |language =  |accessdate =  |quote =  }}</ref>。

本站在关闭前共有12股道，其中I、II股道为丰沙正线，3、4、5、6四股道为站内到发线，7、8、9、10、11、12道为京能电厂运煤专用线。

养三站一直是为京能热电有限公司接卸电煤的车站，但京能石景山热电厂的燃煤机组已于2015年3月19日关闭<ref>{{cite news|url=http://news.sina.com.cn/o/2015-03-20/144031627916.shtml|title=长安街最后烟囱 今“退休” 被燃气热电中心代替 百年京能石景山热电厂、华能热电厂分别于昨天和2016年关停|publisher=[[法制晚报|法制晚报]]|date=2015-03-20|accessdate=2015-12-13}}</ref>，且受[[长安街|长安街]]西延计划影响，丰沙铁路石景山段将新建地下隧道，养三站也将撤销。养三站道岔的拆除工作已于2015年11月13日开始准备，11月16日拆除沙城端（即三家店方向）道岔，24日拆丰台端（养马场方向）道岔，道岔拆除后车站暂时有人值守，但已无列车停靠。

2015年12月18日，养三站关闭车机联控信号<ref>{{cite news|url=http://news.sina.com.cn/o/2015-12-20/doc-ifxmszek7354886.shtml|title=再见，26岁的运煤小站|publisher=[[新京报|新京报]]|date=2015-12-20|accessdate=2015-12-22}}</ref>。养三站原址已于2017年改为丰沙铁路[[石景山隧道|石景山隧道]]西出口，该隧道于2017年12月21日启用。
