{{Vgname|家园：惊世浩劫|en=Homeworld: Cataclysm|official-cn=家园：惊世浩劫|tw=萬艦齊發：地動天驚}}是电子游戏《[[家园|家园]]》的[[资料片|资料片]]，于2000年9月首发。在中国大陆由[[育碧|育碧]]代理发行。<ref name="yesky">{{Cite web |url = http://www.yesky.com/282/208782.shtml |title = 《家园：惊世浩劫》简体中文版上市消息 |author = |date = 2001-12-07 |publisher = 天极网 |language = zh-cn |accessdate = 2017-02-26 }}</ref>本作中引入了补给的的概念。<ref name="yesky" />2017年6月，本作更名为'''家园：紧急状况'''（{{lang-en|Homeworld: Emergence}}）后在[[GOG.com|GOG.com]]发布。因“Cataclysm”一词在2010年发布的[[魔兽世界|魔兽世界]]第三部资料片——《[[魔獸世界：浩劫與重生|魔兽世界：大地的裂变]]》（{{lang-en|World of Warcraft: Cataclysm}}）中被使用，为避免侵权而更名为Emergence。<ref>{{Cite web |url = http://www.ali213.net/news/html/2017-6/304227.html |title = 策略游戏《家园：惊世浩劫》改名并已登陆GOG平台 |author = |date = 2017-06-26 |publisher = 游侠网 |language = zh-cn |accessdate = 2018-03-02 }}</ref>

== 设定 ==

在结束了与泰坦族的战争后，库申族回到了自己的母星希格拉（Hiigara）。4年后，库申族在希格拉上定都，且又分为6个族群（Kiith或Kiithid）。又过了7年，希格拉上的库申族人终于摆脱了粮食危机。但本图西人和泰坦族人在希格拉周围建立了军事基地，控制了直径10光年的空间。图拉尼海盗也趁机在希格拉周围烧杀劫掠。在母舰上从冷冻睡眠中醒来的库申族中，诞生了一个叫英弗利特复仇者的赏金猎人组织，以“永远的复仇”为口号，进行泰坦族战犯的抓捕。

本作的剧情发生在库申族与泰坦族战争结束后的15年。库申族中一个叫索托瓦（Kiith Somtaaw ）的小族群，在一次太空采矿中无意间发现了吊舱。在对吊舱进行研究时，无意间释放出了吊舱中的彼斯特族（the Beast）。<ref name=enet>{{Cite web |url = http://games.enet.com.cn/article/A0120011219001.html |title = 《家园：惊世浩劫》简体中文版圣诞节上市 |author = |date = 2001-12-19 |publisher = 硅谷动力 |language = zh-cn |accessdate = 2017-02-26 }}</ref>
