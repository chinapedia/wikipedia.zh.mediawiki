'''梭法螺属'''（學名：{{lang|la|'''''Cymatium'''''}}），又名'''象法螺屬'''<ref>{{cite web |url=http://digimuse.nmns.edu.tw/Demo_2011/showMetadata.aspx?ObjectId=090000018005b50e&TypeKind=suMeta&Type=invertebrates&Part=2-2&Domain=gg&Field=fi&Language=CHI |title=''Cymatium echo'' |access-date=2017-03-06 |language=zh-hant }}</ref>或'''嵌線螺屬'''{{r|Shelldoc 420090220098 04}}，是[[法螺科|法螺科]][[嵌線螺亞科|嵌線螺亞科]]之下的一個屬，包括從小型到大型的掠食性海螺，是一個[[海洋生物|海洋]][[腹足綱|腹足綱]][[軟體動物|軟體動物]]的[[分類單元|分類單元]]<ref name="WoRMS_138426" />。

本屬物種繁多，有差不多一百種，遍佈世界各地。因此有貝殼學家將本屬物種再細分為十個亞屬，也有建議將部分亞屬提升至屬級，但這些建議到現在都未有公論。

==化石紀錄==
本屬的化石紀錄已知從[[始新世|始新世]]到[[第四紀|第四紀]]（距今：從5580到1.2萬年前）<ref name=AA>{{fossilworks |id=9721 |taxon=Cymatium }}</ref>。

==形態描述==
這些海螺[[雌雄異體|雌雄異體]]，會產下[[卵筴|卵筴]]<ref>{{cite web |url=http://cindy0331.pixnet.net/blog/post/46605134 |title=蝸牛(螺)殺手-鑽紋螺 居然產卵了!!|date=2012-08-23 |accessdate=2017-05-29 |language=zh-hant}}</ref>。 孵化後，[[幼蟲|幼蟲]]有一個[[浮游生物|浮游生物]]階段，可以（在某些物種中）持續數月; 這是在某些物種中看到的廣泛分佈，因為在沉降到海底之前可以將浮游幼蟲攜帶很遠的距離。

==分類==
===亞屬===
目前本屬有至少十個亞屬<ref>{{cite web |url=http://www.jaxshells.org/genus.htm |title=? }}</ref>，部分亞屬有學者建議提升至屬級。不管如何，這些亞屬或屬都詳列如下：
* ''Cymatium'' <small>[[Peter_Friedrich_Röding|Röding]], 1798</small>, <small>type species: ''[[Cymatium_femorale|Cymatium femorale]]''</small>
* [[毛法螺屬|毛螺亞屬]] ''[[Monoplex|Monoplex]]'' <small>G. Perry, 1811</small>, <small>type species: ''[[Cymatium_parthenopeum|Cymatium parthenopeum]]''</small>
* ''[[Gelagna|Gelagna]]'' <small>Schaufuss, 1869</small>, <small>type species: ''[[Cymatium_succinctum|Cymatium succinctum]]''</small>
* ''[[Ranularia|Ranularia]]'' <small>Schumacher, 1817</small>, <small>type species: ''[[Cymatium_gutturnium|Cymatium gutturnium]]''</small>
* ''[[Gutturnium|Gutturnium]]'' <small>[[Otto_Andreas_Lowson_Mörch|Mørch]], 1852</small>, <small>type species: ''[[Cymatium_muricinum|Cymatium muricinum]]''</small>
* ''[[Reticutriton|Reticutriton]]'' <small>Habe and Kosuge, 1966</small>：<small>模式屬：''[[Cymatium_pfeifferianum|Cymatium pfeifferianum]]''</small>
* ''[[Linatella|Linatella]]'' <small>Gray, 1868</small>, <small>type species: ''[[Cymatium_cutaceum|Cymatium cutaceum]]''</small>
* ''[[Cymatium_lotorium|Lotoria]]'' <small>Emerson and Old, 1963</small>, <small>type species: ''[[Cymatium_perryi|Cymatium perryi]]''</small>
* [[籬螺亞屬|籬螺亞屬]] ''{{link-en|籬螺屬|Septa (gastropod)|Septa]]'' {{small|Perry, 1810}}：模式種為''[[Septa_rubecula|Septa rubecula]]''，被WoRMS提升至屬級。
* ''[[Turritriton|Turritriton]]'' <small>[[W.H._Dall|Dall]], 1904</small>, <small>type species: ''[[Cymatium_gibbosum|Cymatium gibbosum]]''</small>
