《'''聽說桐島退社了'''》（{{lang-ja|桐島、部活やめるってよ}}）為日本作家[[朝井遼|朝井遼]]的小說，第22屆[[小說昴新人獎|小說昴新人獎]]得獎作品<ref>{{Cite web |url= http://www.shueisha.co.jp/shuppan4syo/21nen/outline03.html |title= 第22回小說昴新人賞得獎作品《聽說桐島退社了》 |publisher= 集英社 |language= ja |accessdate= 2013-04-28 }}</ref>。本小說後改編成電影。

==故事簡介==
排球社社長桐島突然宣布退社，並失去蹤影，看似一件不起眼的事，卻帶給周遭同學們的生活不小的影響。

==登場人物==
;{{nihongo|菊池宏樹|菊池 宏樹|{{lang|ja|きくち ひろき}}}}
: 棒球社社員。運動萬能，經常翹掉社團活動，但總是背著社團背包。

;{{nihongo|小泉風助|小泉 風助|{{lang|ja|こいずみ ふうすけ}}}}
: 個子矮小的男子排球社社員，替代退社的桐島的「自由球員」位置。

;{{nihongo|澤島亞矢|沢島 亜矢|{{lang|ja|さわじま あや}}}}
: 管樂社社長，擅長吹奏[[薩克斯風|薩克斯風]]，暗戀龍汰。
: 電影中，暗戀宏樹。

;{{nihongo|前田涼也|前田 涼也|{{lang|ja|まえだ りょうや}}}}
: 電影社社員，在號稱「電影甲子園」的高中生電影競賽中獲得特別獎。國中時和霞同校。

;{{nihongo|宮部實果|宮部 実果|{{lang|ja|みやべ みか}}}}
: 壘球社社員。父親與姊姊在交通事故中去世，母親因此精神崩潰，把她當成已逝的姊姊小薰。

;{{nihongo|東原霞|東原 かすみ|{{lang|ja|ひがしはら かすみ}}}}
: 羽球社社員。馬尾女孩，愛好電影。

;{{nihongo|桐島|桐島|{{lang|ja|きりしま}}}}
: 排球社社長，全能型風雲人物，故事圍繞著「桐島」退社發展，全書中都沒有現身過。

;{{nihongo|孝介|孝介|{{lang|ja|こうすけ}}}}
: 排球社副社長，桐島退社後升任社長。實果的男朋友。

;{{nihongo|日野|日野|{{lang|ja|ひの}}}}
: 排球社社員，家裡開花店。

;{{nihongo|龍汰|竜汰|{{lang|ja|りゅうた}}}}
: 宏樹的好友。

;{{nihongo|梨紗|梨紗|{{lang|ja|りさ}}}}
: 桐島的女友。

;{{nihongo|沙奈|沙奈|さな}}
: 宏樹的女友。
