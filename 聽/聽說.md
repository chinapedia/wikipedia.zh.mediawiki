{{engname|《聽說》|''Hear Me ''}}是一部[[2009年電影列表|2009年]]的[[台灣電影|台灣電影]]，由[[台灣導演|台灣導演]][[鄭芬芬|鄭芬芬]]所執導，[[陳意涵|陳意涵]]、[[陳妍希|陳妍希]]及[[彭于晏|彭于晏]]主演，该電影於2009年8月28日在[[台灣|台灣]]上映。應為宣傳[[台北聽障奧運|台北聽障奧運]]而拍的電影。

==情節==
秧秧（[[陳意涵|陳意涵]]飾）常在選手訓練泳池旁為參加[[聽障奧運|聽障奧運]]的姊姊小朋（[[陳妍希|陳妍希]]飾）加油，一心想成就姊姊的夢想。天闊（[[彭于晏|彭于晏]]飾）某天替聽障游泳隊員送便當，巧見秧秧和小朋用[[手語|手語]]對話，上前用手語攀談，且對秧秧一見觸情。隨後秧秧摔車，天闊隨即載她去醫院，甚至跟她要了[[MSN|MSN]]的帳號。 

家裡開燒臘店的天闊為了製造兩人邂逅的機會，總是特地留一個便當給秧秧，甚至自燉煲湯加料和製作愛心便當給秧秧。秧秧感動之餘，仍是堅持手頭充裕之後一併還清。秧秧的打工是雕像藝術的街頭藝人，天闊等到秧秧工作結束後，一同前去[[寧夏夜市|寧夏夜市]]吃飯，兩人卻因付帳時的溝通誤會而發生口角。小朋獨自待在公寓，樓上失火而使得睡著的小朋喉嚨嗆傷送醫住院，亦造成失去參加聽奧的資格。秧秧對此事十分自責與內疚，也因為要照顧小朋，遂決定斷絕跟天闊的連絡。

天闊想要挽回對秧秧的情感，想盡辦法讓秧秧開心，送她玻璃[[水鳥|水鳥]]的存錢筒，甚至是扮成樹木在秧秧家門口等候，但秧秧卻不領情。小朋被取消資格而喝酒晚歸，引起秧秧的愧疚。小朋向秧秧表明不應該再依附她的夢想，應該要有自己的人生，找尋屬於自己的夢想，姊妹倆經過手語溝通後相擁而泣。最後秧秧悄悄地在燒臘店留下還清的便當錢，天闊也詢問父母能否接受他與聾人交往，在支持之下，天闊隨即前去向秧秧表白心意。

天闊在游泳池，背對著聽不見的秧秧自我排演對話，邀她來家裡打工。隔日，天闊帶著秧秧見父母，父母倆也用事先準備好的紙板寫上字句與秧秧溝通。正詢問秧秧是否有願意嫁給天闊的意願時，秧秧突然開口說出「我願意」。眾人震驚之餘，才曉得秧秧並不是聽障，而秧秧當初也誤以為天闊聽不見，所以兩人先前談了一段聽不見的戀愛。

==票房==
電影首映週末，台北票房即達89萬[[新台幣|新台幣]]，全台票房達到160萬[[新台幣|新台幣]]，超越2009年暑期檔的《[[不能沒有你|不能沒有你]]》、《[[陽陽|陽陽]]》和《[[野球孩子|野球孩子]]》等[[台灣電影|台灣電影]]，成為2009年純台灣資金與工作人員影片的開票冠軍<ref>{{cite web| url=http://tw.movie.yahoo.com/movieheadline/d/a/090831/3/gm7.html| title=《聽說》首週末票房創佳績 暑期檔台片冠軍| work=奇摩電影| accessdate=2009-09-01| deadurl=yes| archiveurl=https://web.archive.org/web/20090915220050/http://tw.movie.yahoo.com/movieheadline/d/a/090831/3/gm7.html| archivedate=2009-09-15}}</ref>。第二週，票房成長44%達到120萬，台北票房累積近300萬且全台票房達到500萬，成為2009年國片成長率最高電影。台北週末票房升到第五名，單廳票房在前十名中維持第三名，僅次於《[[變形金剛：復仇之戰|變形金剛：復仇之戰]]》[[IMAX|IMAX]]版和[[好萊塢|好萊塢]]愛情喜劇《[[愛情限時簽|愛情限時簽]]》<ref name="Taiwan">{{cite web | url=http://tw.movie.yahoo.com/movieheadline/d/a/090907/3/gqd.html | title=《聽說》上周末票房成長44% 全台破500萬 | work=奇摩電影 | accessdate=2009-09-07 | deadurl=yes | archiveurl=https://web.archive.org/web/20090928060639/http://tw.movie.yahoo.com/movieheadline/d/a/090907/3/gqd.html | archivedate=2009-09-28 }}</ref>。

第三週，台北僅剩八家戲院上映，但反倒開出151萬元的票房，與前週相比又成長27%。票房逆勢成長三成，是2009年9月11日到13日台北市週末票房排行前30名中唯一成長的電影。總票房累計至今，台北票房近600萬，全台票房確定破千萬<ref>{{cite web| url=http://www.cna.com.tw/SearchNews/doDetail.aspx?id=200909140262| title=《聽說》今年國片第一 票房衝破千萬| work=[[中央社|中央社]]| accessdate=2009-09-14| deadurl=yes| archiveurl=https://web.archive.org/web/20160305013009/http://www.cna.com.tw/SearchNews/doDetail.aspx?id=200909140262| archivedate=2016-03-05}}</ref>。最終票房票房已突破2800萬<REF>[http://newtalk.tw/news/view/2009-11-02/1679 聽說票房冠軍 聽看看鄭芬芬怎麼說] 2009.11.2 新頭殼</REF>。

[[香港|香港]]方面，在同年的11月24日，曙光影視發行公司與香港[[慈善機構|慈善機構]][[龍耳|龍耳]]合作舉辦《聽說》慈善首映禮<ref>{{cite web | language=zh-hk | url=http://www.silence.org.hk/file/HearMe.pdf | title=龍耳呈獻﹕《聽說》首映禮 | work=[[龍耳|龍耳]] | date=2009-11-24 | accessdate=2010-01-03}}</ref>，並獲得[[演藝界|演藝界]]人士親臨支持，亦獲得[[影評人|影評人]]一致讚賞。總票房累計至今，香港票房近80萬港元<ref>{{cite web | language=zh-hk | url=http://dailynews.sina.com/bg/ent/film/sinacn/20091221/1521980760.html | title=香港票房(2009.12.17) | work=[[新浪|新浪]] | date=2009-12-21 | accessdate=2010-01-03}}</ref>。

==公益活動==
在香港上映期間，曙光影視發行公司與慈善機構[[龍耳|龍耳]]合作舉辦一連串的公益活動。


'''宣揚「聾健共融」的理念'''

為發展公眾教育及推廣活動，曙光與龍耳在香港舉辦了電影《聽說》慈善首映禮<ref>{{cite web |language=zh-cn |url=http://v.ent.163.com/video/2009/11/E/G/V5O1QH3EG.html | title=《聽說》香港慈善首映 為聽障人士籌集愛心 |publisher=[[網易|網易]] |date=2009-11-25 |accessdate=2011-08-13}}</ref>。在此次籌辦首映禮，邀請[[香港明星足球隊|香港明星足球隊]]包括[[許志安|許志安]]及[[吳家樂_(演員)|吳家樂]]與龍耳的聾人會員進行足球友誼賽<ref>{{cite web |language=zh-hk |url=http://www.youtube.com/watch?v=yngQmhC_Dlk | title=4-11-2009 友誼賽 龍耳足球隊 + 彭于晏 VS 明星足球隊 |publisher=[[Youtube|Youtube]] |date=2009-11-20 |accessdate=2011-08-13}}</ref><ref>{{cite web |language=zh-hk |url=http://hk.apple.nextmedia.com/template/apple/art_main.php?iss_id=20091105&sec_id=462&art_id=13386668 | title=彭于晏樂得黐實蔡依林 |publisher=[[蘋果日報_(香港)|蘋果日報 (香港)]] |date=2010-11-05 |accessdate=2011-08-13}}</ref>，並得到[[勞工及福利局|勞工及福利局]]局長[[張建宗|張建宗]]、[[電影演員|影視明星]][[曾志偉|曾志偉]]、[[羅慧娟|羅慧娟]]及[[立法會議員|立法會議員]]梁耀宗及正言匯社[[張超雄|張超雄]]等人的致函支持，亦得到多家媒體報導<ref>{{cite web |language=zh-hk |url=http://www.youtube.com/watch?v=o_PMdfC_1i0 | title=「聽說」香港宣傳 彭于晏踢球為聽障募款  |publisher=[[中天新聞|中天新聞]] |date=2009-11-04 |accessdate=2011-08-13}}</ref><ref>{{cite web |language=zh-hk |url=http://www.hkheadline.com/ent/ent_content.asp?contid=93219&srctype=n | title=許志安極速減磅增賣點 |publisher=[[頭條日報|頭條日報]] |date=2010-11-05 |accessdate=2011-08-13}}</ref>，向社會大眾推廣「[[聾|聾]]健共融」的理念。


'''推廣手語'''

彭于晏及陳意涵在香港宣傳期間，龍耳派出手語導師協助下，拍攝一系列十集《[[手語教室|手語教室]]》片段<ref>{{cite web |language=zh-hk |url=http://www.youtube.com/watch?v=12_Q1miwNno | title=港版 聽說 手語教學  |publisher=[[Youtube|Youtube]] |date=2010-12-18 |accessdate=2011-08-13}}</ref>，於2009年11月每天早、午、晚時段，於[[亞洲電視|亞洲電視]]平台上播放，希望藉此加強公眾對手語的認識。
