《'''掇拾雜記'''》（[[越南語|越南語]]：'''''Xuyết Thập Tạp Ký'''''），[[越南|越南]][[文言文|漢文]]史籍，由[[阮朝|阮朝]]官員兼作家[[李文馥|李文馥]]撰，記錄文人間的詩文、聯語的趣事逸談，這些資料由作者與人交談所得，「掇拾而存之」，對越南古代文學具有研究價值。

== 本書的編撰 ==
本書的作者'''李文馥'''（生卒：1785年─約1849年），[[阮朝|阮朝]]時期官員，曾在翰林院、禮部、廣義鎮、廣南營、戶部、工部、禮部等歷任官職，奉命到過[[小西洋|小西洋]]、[[南海|南海]]、[[中國|中國]]等地，見聞甚豐，史稱他「所歷非一」，且「有文名」。<ref name="大南實錄李文馥">《大南實錄·大南列傳正編第二集·諸臣列傳·李文馥傳》，茲參考許文堂、謝奇懿編《大南實錄清越關係史料匯編》，中央研究院東南亞區域研究計畫，504頁。</ref>《掇拾雜記》的編寫，據序文所述，是作者自少時在學習之暇，從「家庭及場屋間居常說話」，這些談話內容「多可人語，乃皆野史稗官之所不見載者」。李文馥擔心這些聽聞「歷歲既久，遺忘殆盡」，便趁「今尚依稀記憶什之一二，因掇拾而存之，不拘門類倫次，有記輙錄」。有時雖記憶不清，亦「以己意附補」，於是完成此書。<ref>李文馥《掇拾雜記·序》，收錄於孫遜、鄭克孟、陳益源主編《越南漢文小說集成》第十六冊，上海古籍出版社，55頁。</ref>

《掇拾雜記》寫成的時間，據序文所載是在「[[嗣德|嗣德]]叁年」，即1850年。然而阮朝官修史書《[[大南實錄|大南實錄]]·大南列傳正編第二集》裡提到在1849年（嗣德二年），李文馥「擢光祿寺卿，尋卒」<ref name="大南實錄李文馥"/>，後世據此史文，一般認為李文馥卒於該年。學者[[陳慶浩|陳慶浩]]對此提出質疑，認為既然在《掇拾雜記·序》裡說本書寫於嗣德三年，李文馥卒年應在此年之後。<ref name="越南漢文小說集成51">《掇拾雜記》陳慶浩《提要》，收錄於孫遜、鄭克孟、陳益源主編《越南漢文小說集成》第十六冊，上海古籍出版社，51頁。</ref>

== 評價 ==
《掇拾雜記》的研究價值，據學者[[陳慶浩|陳慶浩]]指出，此書載有古時越南文人間的作品，因而「可視為民間文學的資料」<ref name="越南漢文小說集成51"/>。

== 流傳情況 ==
《掇拾雜記》寫成後，流傳於世的主要抄本有二，一本藏於[[越南國家圖書館|越南國家圖書館]]，當中附有其他[[李文馥|李文馥]]的作品，並在內文裡有硃墨眉批和圈點，抄錄甚為細心。<ref>《掇拾雜記》陳慶浩《提要》，收錄於孫遜、鄭克孟、陳益源主編《越南漢文小說集成》第十六冊，上海古籍出版社，49─50頁。</ref>另一抄本為[[漢喃研究院|漢喃研究院]]所藏，亦附有其他李文馥作品。<ref name="臺灣中央研究院"/>現代校點本方面，有[[陳慶浩|陳慶浩]]校點本，由[[上海古籍出版社|上海古籍出版社]]出版。<ref>《掇拾雜記》陳慶浩《提要》，收錄於孫遜、鄭克孟、陳益源主編《越南漢文小說集成》第十六冊，上海古籍出版社。</ref>
