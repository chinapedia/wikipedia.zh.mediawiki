== 東盧梅里亞納大陸 ==
=== 特爾齊亞將國（トルキエ將國） ===
*位於盧梅里亞納大陸東南部，國土被草原和沙漠所覆蓋的遊牧民族國家。是一個完全的實力主義社會國家，掌管國家政治和軍事大權的是被稱為「十三人將軍」的人們，最高長官為大將軍——位於首位的人必須要用顯赫的功績、豐富的才能和高尚人格。國內分為十三個洲，分別由十三人將軍管理。所有的國事，都在週三召開的將軍會議上決定。位於商隊之路和海之路這兩條相互交叉的交通要道上，同時也是一個商業國家，據說世界上90%的財富都從這裡通過。軍人都從通過武術測驗的庶民中選拔，由低到高非別是：下級士官——十人隊長——百人隊長——千人隊長——將軍——13人將軍——大將軍。建國於75年前，為了應對來自外部的壓迫，20個遊牧民族聯合起來建立了國家，其中16個民族構成了特爾齊亞。五大信仰中尤其崇拜水之精靈，來源於水資源的匱乏，每個村落都修建有水之神殿。
*推行獨特的特爾齊亞曆法，對應為：花月（ラーレ）＝1月，大鍋月（テンチェレ）＝2月，酒月（ラク）＝3月，白砂糖月（シュケル）＝4月，黒砂糖月（カラ・シュケル）＝5月，絨毯月（ハリ）＝6月，陶器月（チニリ）＝7月，胡椒月（ビベー）＝8月，塩月（トゥス）＝9月，白羊月（アク・コユンル）＝10月，黒羊月（カラ・コユンル）＝11月，茶月（チャイ）＝12月。

:位于将国西北方，是与帝国接壤的军事重镇，归第一州管辖。
; 欧亚西斯城（オアシスの町）
:位于交通要道上的商业重镇。
;耶尼·图古利鲁村（イェニ・トゥグリル村）
:处于将国最西北方，属第二州管辖。玛法穆特的故乡，在12年前与帝国的战争中被完全摧毁，现在的村子是以泉水为中心重建的。
; 水精灵之城（ナバラト） 
:位于特爾齊亞南部，全大陆最大的水之神殿坐落于此，是水精灵信徒心中的圣地。
; 泉之城切西梅（泉の町）
:地形崎嶇、利於埋伏，為“警告之鐘”鑄造工廠所在地。

=== 四将国 ===
*与特爾齊亞同文化根源的游牧民族国家，为特魯齊亞的衛星国。75年前建国时，实力最强劲的四支部族建立起了君主制国家。四将国的统治者为将王，实行君主世袭制，将王的權力等同於特魯齊亞的13将军。
==== 穆兹拉克将国(ムズラク将国) ====
*大特爾齊亞文化圈中唯一的临海国。也是良马的产地，骑兵尤其精锐，在四将国中战斗力首屈一指。目前將王為「聖官」巴亞吉特。
===== 主要地名 =====
; 港之城 里曼（リマン）
:大特爾齊亞文化圈中唯一面朝央海的城市。
; 東之城 夏爾克（シャルク） 
:穆兹拉克将国的首都，巴拉班的居所名为“红色宫殿（アル・サライ）”。

==== 布查克将国(ブチャク将国) ====
*特爾齊亞卫星国之一，出产宝石。其寶石加工的技術也是首屈一指，賣到黃金城經過加工為飾品，價格翻漲十倍。
*布查克的王子從小就會被送到各個商會去，當作外人養育。等哪天將王駕崩了，由最先徵招進宮的王子即位。故各王子之間都不曾見過自己的兄弟，並只想搶在他們之前奪得王位。目前將王為「武器商」伊斯梅爾。
===== 主要地名 =====
;宝石之城 穆朱赫（ムジュヘル）
:布查克将国的首都。宝石原石的生产地。

====库鲁裘将国(クルチュ将国) ==== 
*特爾齊亞卫星国之一，在四将国中国力最弱，并且与本国并未接壤。目前將王為「劍」奧爾罕。
===== 主要地名 =====
;奇岩之城 阿凱普（アカイブ）
:库鲁裘将国首都，如字面意思一样地形复杂。將王所住的宮殿為「藍色宮殿」。

==== 巴尔塔将国(バルタ将国) ====
*特爾齊亞卫星国之一，由于王位纷争频繁，多动乱。目前將王為凱末爾。
===== 主要地名 =====
;草原之城 奧瓦
:巴尔塔将国的首都。

=== 烏拉德王國(ウラド王国) ===
*位於大特爾齊亞文化圈北方，地處高原，終年寒冬，現任國王為齊格蒙特三世。因大部分資源能自給自足，自建國後的四百年都維持鎖國政策，嚴格管制出入，審查機制之謹慎可謂大陸第一。而正因鎖國帶來的效應，對於大陸上周邊諸國的動向特別關注，對每一外國都有專屬的外交大臣，例如對特爾齊亞的大臣為瑪露姬塔(王國第四王女，後任對東方諸國外交大臣)，對巴魯特萊茵的大臣為戈露特魯朵(王國第二王女，後任對心臟地區外交大臣)。而只要能有效獲得情報，不管手段為何，國王都不會反對。
*其軍隊以「槍騎兵」為主體。國家施行「全民皆兵」之政策，在農閒期進行訓練，已備有事之需。
*國內主要資源為散佈於國境內的小規模礦山群，以及廣闊的針葉林。農業土地雖貧瘠但仍有收穫，也發展近海捕撈漁業。在資源短缺、開發殆盡的同時，因瑪法穆特等人於海崖上發現高品質的「鳥糞肥」而重現生機。目前「鳥糞肥」交由特爾齊亞負責出口。
==== 主要地名 ====
;追鳥海岸
:王國狹長的海岸線，一路連接到帝國邊境。此處原有大批的海鳥在此棲息，隨著人們不斷地捕捉而不見蹤跡，但其排泄物長年堆積而成「鳥糞肥」，品質優良，預估會為王國帶來大量收益。

=== 銀色之都　阿吉羅斯(銀色の都　アルギュロス) ===
*位處東西大陸之交點，以商業繁榮聞名之城市。城市主要由「三大商人」管理，要在此處經商者須申請並持有「憑證」，才有得以貿易之資格。顧客對商隊主要是憑實績，資歷淺的人就算之前與顧客有過交情，也會被拒絕合作。若憑證商人被知道與外國政治家有所勾連，將被剝奪憑證，並終生驅逐出境。而能對極東大國「大秦」通商的，只有三大商人，若私下通商也會被驅除(前大秦禁軍的望·伊希除外)。

=== 岩之都　斯多格里奧公國（岩の都　スコグリオ）===

=== 燈塔之都　波尼基亞共和國（燈台の都　ポイニキア） ===
*曾為國祚長達1800年之久的大海洋帝國・波尼基亞帝國的首都。現任市長為伊歐斯·凱薩·阿波羅托洛斯。原本只是個小城市，為了自衛而於3400年前獲得名為「弗盧里奧」的城市作為屬地{{註|原作漫畫第77話中的地圖提及名稱。}}，從此開始大陸擴張之路，極盛時期領土涵蓋整個盧梅里亞納大陸。古波尼基亞帝國的初代皇帝名為「波尼科斯」。{{註|原作漫畫第80話中提及。}}
*其繼承了古波尼基亞帝國以來的體制，尤其是「元老會」的議會制度。其作為最高決策機關，若有人提出議案，即以投票表決。曆法則使用波尼基亞曆，已使用了3000年以上。
*城市東側的水晶山崖，不僅能防範外敵，從中開採出的水晶原石更是上等出口品，主要出口給威尼迪克(威尼迪克杯的原料)，特魯奇亞耳探的「水塔石」原料也出於此。
*波尼基亞的地理位置恰恰位於央海中心，與南方的薩洛斯王國連成分割央海的東西線。正因地勢重要，及其身為東西方重要的交易要沖，自3000年前就建設了堅固的城牆及燈塔，搭配水晶山崖防衛外敵，加上200年前與威尼迪克簽訂的互助條約(和平時期波尼基亞優先給予威尼迪克商業優惠，並作為商人們補給基地；戰時若收到波尼基亞求援訊息，威尼迪克即刻編制並派出艦隊協助防禦){{註|原作漫畫第17話中提及。}}，至今未被攻破。
*波尼基亞曆3022年(帝國曆451年)五月二十九日，在經過十七日的海上攻防，帝國軍暗中越過水晶山崖與海上艦隊前後夾擊，加上威尼迪克艦隊遲遲未來，宣告全面投降。戰後被歸為巴魯特萊茵帝國波尼基亞領，與同樣被控制的薩洛斯封鎖了海上東西線，造成西央海地區實質上被帝國控制。
