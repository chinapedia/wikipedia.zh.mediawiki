'''宇宙学视界'''是指能够接收信息的可测量[[距离|距离]]。这种对观测的限制来源于[[广义相对论|广义相对论]]，和[[ΛCDM模型|宇宙学标准模型]]。宇宙学视界界定了我们[[可观测宇宙|可观测宇宙]]的范围。本文将解释宇宙学上的几种不同的[[视界|视界]]的定义。本文中所用的距离单位是千[[秒差距|秒差距]]（kpc）或百万[[秒差距|秒差距]]（Mpc）。

== 粒子视界 ==
粒子视界是指在某个时刻<math>t=t_0</math>的[[观察者|观察者]]能够接收到其他地方的[[光|光]]信号的边界。粒子视界代表我们能够从过去获取信息的最远[[距离|距离]]，通常这也是[[可观测宇宙|可观测宇宙]]的大小。其对应的[[视界|视界]]半径可表示为：

<math> d_H=a(t_0)\int_0^{t_0} c a(t)^{-1}\mathrm{d}t </math>，

其中 <math>a(t)</math>对应于[[弗里德曼-勒梅特-罗伯逊-沃尔克度规|FRW度规]]中的[[尺度因子|尺度因子]]。

== [[事件视界|事件视界]] ==
[[事件视界|事件视界]]跟[[粒子视界|粒子视界]]有所不同，[[粒子视界|粒子视界]]是指在某个指定时刻远处[[光子|光子]]能够到达观察者的最远[[同移距离|同移距离]]，而事件视界指的是某个时刻发射的光子在未来所能传播的最大[[同移距离|同移距离]]，这里所说的未来时间由[[立体几何|时空几何]]本身决定，值得一提的是它未必是正无穷。一般的，时刻t发出的光子对应的固有距离可表示为：

<math> d_e(t)=a(t)\int_t^{t_{max}} c a(t')^{-1}\mathrm{d}t'</math>，

其中<math>t_{max}</math>是[[宇宙坐标系|宇宙坐标系]]时间轴的最大值，当宇宙一直膨胀时，它是正无穷。在[[ΛCDM模型|宇宙学标准模型]]中，<math>d_e(t_0)<\infty</math>。

== 哈勃视界 ==
所谓的哈勃视界指的是如果[[宇宙|宇宙]]不膨胀的话，一个光子可以传播的距离，这个距离是
<math>\chi = ct</math>，
其中t是从[[大爆炸|大爆炸]]开始的回视时间（[[宇宙年龄|宇宙年龄]]），根据[[弗里德曼方程|弗里德曼方程]]，

<math>t=\int_0^a \frac{}{H_0\sqrt{\Omega_{\gamma}a^{-2}+\Omega_ma^{-1}+\Omega_{\Lambda}a^2}} \mathrm{d}a</math>

其中<math>H_0</math>是[[哈勃常数|哈勃常数]]，<math>\Omega_{\gamma}</math>,<math>\Omega_m</math>,<math>\Omega_{\Lambda}</math>是密度参数，分别对应[[辐射|辐射]]，[[物质|物质]]，[[暗能量|暗能量]]的相对[[能量密度|能量密度]]。
今天，大概的，

<math>\chi_{0} \approx \frac{c}{H_0}</math>,

算得哈勃视界半径约为4.2Gpc。注意，这个尺度不是真正物理的尺度，但是由于历史的原因，通常我们用这个名词表示宇宙的[[半径|半径]]。

== 未来视界 ==
在一个正在[[宇宙加速膨胀|加速膨胀的宇宙]]中，事件在[[未来无穷远|未来无穷远]]（<math>t \to \infty</math>）将不可能被观测到，这是因为在比如指数膨胀的[[德西特时空|德西特时空]]中，事件信号将会被[[红移|红移]]到无线长的[[波长|波长]]，直至不可探测到。若以今天的固有距离来衡量，这给出了可以接收信号的最远距离的一个上限。更确切的说，选定[[参考系|参考系]]中同一[[时刻|时刻]]不同[[空间|空间]]位置发生的某些事件将永远不可能被我们观测到，即使我们可以观测到过去同一空间位置的事件的信号。也就是说，我们可以一直接收到此地事件发出的信号，但是无论多长时间，我们也不可能接受到那个空间位置事件发生的信号。实际上，从那个空间位置发出的信号的能量将会越来越弱，频率也会越来越低，也就是说不可观测到。在一个[[暗能量|暗能量]]主导的[[宇宙加速膨胀|加速膨胀]]的[[宇宙|宇宙]]中，[[尺度因子|尺度因子]]指数增长，在Kapteyn宇宙中被引力所束缚的银河系最终也将变得不可观测。
