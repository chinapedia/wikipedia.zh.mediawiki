{{vgname|宇宙巡航艦III－從傳說到神話－|ja=グラディウスIII -伝説から神話ヘ|en=Gradius III}}是一款由日本遊戲公司[[科樂美|科樂美]]所開發與發行之橫向捲軸射擊遊戲《[[宇宙巡航艦系列|宇宙巡航艦系列]]》作品，於1989年12月11日在日本上市。

== 劇情 ==

存在於未知領域的Bacterion軍團又開始行動了。這次它發出了一種強大的「Dark Force」能量，吞噬了一個又一個行星。Gradius軍派出Vic Viper超時空戰機迎接Bacterion軍團的攻擊。

== 遊戲系統 ==

; 能量膠囊

紅色能量膠囊可提升一格能量表。藍色能量膠囊除了可以消滅低耐久度的敵人，還可讓畫面上的敵方子彈全部消失。

; 能量表

每取得一次紅色膠囊即可提升一格能量表，每一格能量表各自表示一種裝備，供玩家選擇。

遊戲已設定好四種基本型態。

; 速度提升 (Speed Up)
: 機體飛行速度可以提升五次。

; 飛彈

本代可以讓飛彈連射兩發，且限定街機版，SFC版本沒有連射。

* 飛彈 (Missile)
: 向前下方發射對地飛彈，落到地面後會延著地面前進直到擊中敵人為止。

* 廣域破壞炸彈 (Spread Bomb)
: 向前方投擲的無誘導炸藥，可以摧毀一定範圍內的敵人。

* 光子魚雷 (Photon Torpedo)
: 投下之後會沿著直線前進，並可貫穿敵人。

* 雙向飛彈 (2-Way Missile)
: 向前上方與前下方拋射飛彈。

; 雙向機槍
* 雙向機槍 (Double)
: 原本的兩發前向子彈轉移一發往前上方發射。無法與雷射並用。

* 後方機槍 (Tailgun)
: 原本的兩發前向子彈轉移一發往後方發射。無法與雷射並用。

* 自由機槍 (Free Way)
: 原本的兩發前向子彈轉移一發往機體移動方向發射。無法與雷射並用。

; 雷射
* 雷射 (Laser)
: 具有貫穿能力的雷射。無法與雙向機槍並用。

* 波紋雷射 (Ripple Laser)
: 上下幅度會隨著發射距離而增加的雷射武器，威力與一般子彈相同。無法與雙向機槍並用。

; 子機 (Option)
: 子機會跟在自機後方並進行一樣的攻擊，最多可以攜帶四枚。

; ？

在遊戲開始之前，可供玩家選擇防禦裝備。

* 盾牌 (Shield)
: 於機首追加兩枚用來防禦來自前方的攻擊，可以抵擋16次攻擊。

* 力場 (Force Field)
: 機體周圍會出現防護用的力場，用來防禦來自各方位的攻擊，可以抵擋3次攻擊。

* 自由護盾 (Free Shield)
: 可抵擋8發子彈攻擊，可以同時擁有8顆護盾。
