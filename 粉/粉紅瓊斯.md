《'''粉紅瓊斯'''》（{{lang-ja|ピンキージョーンズ}}）是[[桃色幸運草|桃色幸運草]]的第2張的[[單曲|單曲]]<ref>包括獨立製作單曲通算是第4張</ref>。於2010年11月10日由[[STARCHILD|STARCHILD]]發售。

== 概要 ==
移籍[[STARCHILD|STARCHILD]]第1彈單曲。初回限定盤有3種（A、B、C）。每張DVD收錄1首不同PV。此外，附送發售活動應募抽選券。

表題曲「粉紅瓊斯」是[[電視動畫|電視動畫]]《[[緣之空|緣之空]]》使用的[[片尾曲|片尾曲]]。組合的樂曲是首次為電視動畫使用的主題曲。歌詞裡有6個成員的名字在入面<ref name = CD>{{Cite web |date= 2010年11月10日 |url= http://artist.cdjournal.com/d/pinky-jones/4110081652 |title= ももいろクローバー / ピンキージョーンズ |work= CDジャーナル |author= |accessdate= 2011年10月7日 |language=ja}}</ref>。
