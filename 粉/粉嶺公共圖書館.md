'''粉嶺公共圖書館'''（[[英文|英文]]：'''Fanling Public Library'''）是由[[香港公共圖書館|香港公共圖書館]]系統管理的[[分區圖書館|分區圖書館]]，位於[[香港|香港]][[新界|新界]][[粉嶺|粉嶺]][[聯和墟|聯和墟]]和滿街9號[[御庭軒|御庭軒]]基座2樓，於2003年1月正式運作<ref>http://hkpl.hk/tc_chi/ext_act/ext_act_pub/ext_act_pub_ln/ln_200303iss_01_01.html{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }} </ref>，取代原址位處粉嶺[[祥華邨|祥華邨]]祥樂樓地下的小型公共圖書館。現址佔地2200平方米，除了包括原有的成人圖書館、兒童圖書館、參考圖書館及報刊閱覽室外，另外增設多媒體圖書館、電腦資訊中心、推廣活動室及學生自修室，以及在成人圖書館開闢一角作為容許有限度飲食的咖啡閣。

2008年起，政府建立[[香港政府WiFi通|香港政府WiFi通]]，本館亦為熱點之一<ref>{{cite web |url=http://www.gov.hk/mobile/tc/wifi/location/address_plf.htm |title=存档副本 |accessdate=2008-11-04 |deadurl=yes |archiveurl=https://web.archive.org/web/20090213110546/http://www.gov.hk/mobile/tc/wifi/location/address_plf.htm |archivedate=2009-02-13 }} </ref>。2014年6月增設北區文化及歷史資源角。

== 歷史 ==
[[區域市政局|區域市政局]]管理新界各大公共圖書館時，粉嶺公共圖書館服務範圍原本不包括[[聯和墟|聯和墟]]東北鄉郊地區，由區域市政局每兩星期派出[[流動圖書車|流動圖書車]]到[[聯和道|聯和道]]與[[聯發街|聯發街]]交界的遊樂場提供借還圖書服務。直到2003年，粉嶺公共圖書館遷至現址，流動圖書車服務才取消。

==流動圖書車服務==
康樂及文化事務署由2009年4月27日起於粉嶺[[華心邨|華心邨]]提供流動圖書館服務<ref>http://www.hkpl.gov.hk/misc/mlsn/tc_chi/File/A3_WahSumPoster.pdf</ref>，填補粉嶺南圖書館服務的空缺。
