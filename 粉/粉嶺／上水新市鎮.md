'''粉嶺／上水新市鎮'''（'''Fanling-Sheung Shui New Town'''），是[[香港|香港]][[新界|新界]][[北區_(香港)|北區]]一個綜合式發展的[[香港新市鎮|新市鎮]]，範圍包括[[粉嶺|粉嶺]]及[[上水|上水]]<ref>[http://www.ozp.tpb.gov.hk/pdf/S_FSS_10_c.pdf 粉嶺／上水分區計劃大綱（pdf格式）] {{webarchive|url=https://web.archive.org/web/20070929080906/http://www.ozp.tpb.gov.hk/pdf/S_FSS_10_c.pdf |date=2007-09-29 }}</ref>。

== 地理 ==
粉嶺/上水新市鎮位於香港北區，是香港最北部的新市鎮，鄰近[[深圳市|深圳市]]。

新市鎮由上水和粉嶺兩個分區所組成，範圍東至[[粉嶺|粉嶺]][[聯和墟|聯和墟]]馬適路，南至粉嶺塘坑[[馬會道|馬會道]]、和興村、[[畫眉山|畫眉山]]交界，西至[[上水工業區|上水工業區]]、[[石上河|石上河]]一帶，北至馬會道與文錦渡路，梧桐河交界。新市鎮被[[麻芴河|麻芴河]]、畫眉山、石上河和[[梧桐河|梧桐河]]所圍繞，總發展面積為7.8平方公里。粉嶺／上水新市鎮一帶屬[[粉嶺低地|粉嶺低地]]，以平地、丘陵為主，与[[元朗|元朗]]及[[天水圍|天水圍]]是香港少數不涉及[[填海|填海]]建設的新市鎮。新市鎮大致可分為上水市中心及石湖墟、粉嶺市中心、聯和墟、粉嶺南和上水第36區（上水南）各部分。

== 歷史 ==
在粉嶺/上水新市鎮未發展之前，上水和粉嶺一帶早已有多條鄉村發展，現今新市鎮範圍原屬包括[[菜園村|菜園村]]、[[上水鄉|上水鄉]]、[[粉嶺圍|粉嶺圍]]、[[黃崗山村|黃崗山村]]、[[安樂村|安樂村]]、[[天平山村|天平山村]]、[[石湖村|石湖村]]、[[馬屎埔村|馬屎埔村]]、[[靈山村|靈山村]]、[[吳屋村|吳屋村]]、[[雞嶺村|雞嶺村]]、[[和合石村|和合石村]]、[[田心村|田心村]]、[[掃管埔村|掃管埔村]]等鄉村和農田的部分或全部用地。當時區內的商業和墟市發展主要在[[石湖墟|石湖墟]]和[[聯和墟|聯和墟]]。[[港英政府|港英政府]]於1970年代後期開始發展上水和粉嶺一帶，成為現代化新市鎮。其中首先發展上水市中心及石湖墟一帶；其次在80年代開始發展粉嶺鐵路站以北一帶；到了90年代初發展聯和墟和粉嶺南一帶為現代化社區；並在2000年代開始發展上水第36區。

== 上水市中心及石湖墟發展 ==
[[File:HK_Sheung_Shui_龍琛路_Lung_Sum_Avenue_view_Shek_Wu_Hui_building_facades_Jan_2017_Lnv2_04.jpg|thumb]]
新市鎮發展之初，第一個發展項目為上水鐵路站以南的[[香港公共屋邨|公共屋邨]][[彩園邨|彩園邨]]。該地原為菜園村的土地，後來先被政府收回及平整，並於1978年開始興建，1981年落成陸續入伙。當時，市鎮中心內的商場和社區並未開始興建，居民日常消費都在火車站北面的[[石湖墟|石湖墟]]進行。因此，石湖墟是新市鎮早期發展的重心地帶，即使今天，石湖墟依然是上水最繁盛的地方。另外，屋邨旁的居屋屋苑[[旭埔苑|旭埔苑]]和[[彩蒲苑|彩蒲苑]]分別於1982及1984年落成並陸續入伙，為市鎮內的第一批居屋屋苑。

其後於1980年代初，位於上水東北部的天平山村一帶開始發展，大批村屋被清拆以騰出土地興建公屋和屋苑。包括於1986年入伙的[[天平邨|天平邨]]、居屋屋苑[[安盛苑|安盛苑]]、[[翠麗花園|翠麗花園]]、私人住宅奕翠園等。而市鎮主要康樂設施亦集中在此區域，包括上水游泳池、北區運動場、龍琛路體育館等。

上水北部一帶主要是工業和倉庫中心，不但為新市鎮居民提供工作機會，更是為中港貨運貿易運作提供一大場地。近年，該區因自由行政策漏洞，成為[[平行進口貨品|水貨]]倉庫集中地，大量中港水貨客聚集和隨地散貨，製造大量垃圾，使當地衛生環境惡劣，亦引起區內居民對水貨客所造成的滋擾而不滿。

此外，新市鎮早期以上水石湖墟為發展中心，而石湖墟東南方被規劃為新市鎮的市中心，即包括現在上水廣場、新都廣場、上水中心、上水名都、龍豐花園及石湖墟綜合大樓在內的地帶。1990年代初期，市中心內逐漸有住宅屋苑及社區設施的落成。當中，上水廣場於1995年落成，為新界北唯一一幢甲級寫字樓大廈。

市中心商業區發展歷程：
*1991年：[[新都廣場|新都廣場]]落成入伙、[[龍豐花園|龍豐花園]]落成入伙
*1992年：[[上水名都|上水名都]]落成入伙
*1993年：[[上水中心|上水中心]]落成入伙
*1994年：[[石湖墟政府綜合大樓|石湖墟政府綜合大樓]]啟用
*1995年：[[上水廣場|上水廣場]]開幕
*2014年：[[上水匯|上水匯]]開幕

==粉嶺站一帶的發展==
[[File:Fanling_New_Town_201701.jpg|250px]]
新市鎮發展其後，工程不斷展開，[[粉嶺站|粉嶺站]]一帶為第二個發展的地帶。包括1984年落成的祥華邨，是市鎮內第二個公共屋邨，是北區最大型的公共屋邨。不過，當時粉嶺的發展才剛展開，鄰近火車站的屋苑和商場尚未興建。因此，邨內居民要購物便要到上水石湖墟，或到粉嶺北部的聯和墟。而在粉嶺站至粉嶺圍的一片地帶則逐漸發展為粉嶺的康樂和北區政府設施集中地，包括粉嶺游泳池、北區政府合署、粉嶺法院大樓、粉嶺遊樂場、粉嶺健康中心及前粉嶺女童院。

而位於粉嶺鎮東北的大型工業邨安樂村於1980年代中期開始發展，原有村屋陸續拆卸，並在興建工業大廈，為市鎮內一大通勤重地。

至於粉嶺站鄰近的屋苑及商場亦在90年代陸續落成。包括粉嶺中心於1990-1991年入伙；粉嶺名都亦於1993年入伙，為附近居民提供了購物的地方，因此粉嶺站一帶商場可算是粉嶺市中心。不過，這些商場始終不及上水市中心的購物中心集中和方便，相當部分粉嶺居民的消費購物仍選擇上水市中心。

== 粉嶺南的發展 ==
粉嶺南為現時市鎮內人口密度最高的地帶，各個公共屋邨、居屋和私人屋苑於90年代中期開始相繼落成，使這地帶成為粉嶺的主要住宅區。當中的[[華明邨|華明邨]]於1990年落成入伙。因為鄰近[[和合石墳場|和合石墳場]]及地處偏遠，所以早期出租情況並不理想，後來政府推出優惠政策吸引租戶入住。而[[華心邨|華心邨]]於1995年落成，其後於2000年，公共屋苑[[雍盛苑|雍盛苑]]入伙；私人住宅方面，[[花都廣場|花都廣場]]、[[碧湖花園|碧湖花園]]及[[牽睛間|牽睛間]]於90年代中期至後期陸續入伙；居屋方面，[[欣盛苑|欣盛苑]]於1993年落成入伙，[[景盛苑|景盛苑]]於1995年開始陸續入伙，[[昌盛苑|昌盛苑]]於2000年入伙。

2015年，位於花都廣場附近的粉嶺南政府綜合大樓開幕，為北區居民提供更多社區服務設施，亦減輕現有社區設施的需求壓力。

至於粉嶺西南部亦有住宅發展，包括公共屋邨[[嘉福邨|嘉福邨]]於1994年落成入伙；而邨內的居屋屋苑[[嘉盛苑|嘉盛苑]]則於1995年入伙。另外，區內較低密度的私人屋苑亦集中在這區發展，包括百福花園等私人住宅也位於這區，住宅帶沿[[百和路|百和路]]一直伸延至上水蔚翠花園、維也納花園附近。

== 聯和墟的發展 ==
[[聯和墟|聯和墟]]早於1951年建墟，但墟內及其附近一帶的現代化發展卻要到90年代才展開。

踏入90年代，聯和墟一帶開始有居屋屋苑和私人住宅落成。居屋屋苑[[榮福中心|榮福中心]]和[[榮輝中心|榮輝中心]]分別於1993及1996年落成入伙；私人住宅方面，海聯廣場為較早期的發展，於1994年入伙。該地從前為俗稱「白屋仔」的聯和新村，後來清拆村屋後才建成現在的私人住宅，附近很多屋苑也是填平田地和清拆村屋後建成的。1999年，[[綠悠軒|綠悠軒]]落成入伙，連同之後的帝庭軒和御庭軒（分別於2000及2003年入伙）均是聯和墟近年發展的較大型屋苑。

至於舊墟方面，2000年代初舊街市被拆卸，並搬到新建成、位於御庭軒的政府綜合大樓。包括街市、圖書館連自修室、體育館、社區會堂、環境資源中心等，為聯和墟一帶的居民提供服務。

2015年開始，聯和墟新發展大型私人住宅[[瓏山一號|瓏山一號]]和[[逸峯|逸峯]]，其中位於逸峯的大型商場更設有戲院，將成為北區另一個大型商業中心。

== 上水第36區（上水南）的發展 ==
在1980年代後期，1990年代初上水南部一帶已有初部發展，包括位於彩園邨及[[粉嶺公路|粉嶺公路]]以南一帶的太平邨於1989年落成入伙，為新市鎮內第四個公共屋邨，而鄰近的威尼斯花園亦於1993年入伙。同時，位於附近的[[北區醫院|北區醫院]]在1998年落成。

自2000年代開始，上水南一帶才有更大規模發展。主要發展為清曉路一帶，名為「粉嶺/上水第36區」<ref>[http://www.pland.gov.hk/pland_tc/info_serv/tp_plan/adopted/fss.html 部門內部圖則 - 粉嶺/上水]</ref>的新發展區及相關鄰近地區開始發展。成為新市鎮現今主要新發展地帶。

該發展區首期的發展為2003年開課之學校村，包括[[曾梅千禧學校|曾梅千禧學校]]、[[佛教正慧小學|佛教正慧小學]]及[[風采中學|風采中學]]三所學校。

而住宅發展方面，發展區首個[[公共屋邨|公共屋邨]][[清河邨|清河邨]]第三期於2006年開始陸續落成入伙，而第一、二期（包括屋邨商場）則因承建商違反合約而延至2008年7月落成。<ref>[http://www.housingauthority.gov.hk/tc/about-us/news-centre/press-releases/index.html?key=7913 房委會收回三個建築地盤]</ref>同時，上水南一帶的私人屋苑[[御皇庭|御皇庭]]、[[顯峰|顯峰]]及[[御景峰|御景峰]]相繼落成。

而上水南第二條公共屋邨[[祥龍圍邨|祥龍圍邨]]在2015年完工，位於清曉路/保健路/丙岡路交界，包括兩座樓高 33和37 層之大廈、一座樓高 4 層之商鋪/社會福利設施及其他相關設施。<ref>http://www.districtcouncils.gov.hk/archive/north_d/pdf/2010/N_2010_019_TC.pdf</ref>

同時，隨著區內一帶人口增多，為應付區內居民對體育場館之殷切需求，政府於粉嶺/上水第28A區，即[[上水官立中學|上水官立中學]]側興建多層室內體育館，設有多用途主場館、乒乓球室、室內緩跑徑等設施。體育館於2012年3月落成啟用，名為「保榮路體育館」。<ref>[http://www.lcsd.gov.hk/b5/ppr_release_det.php?id=4683 保榮路體育館為北區提供新穎設施] {{webarchive|url=https://web.archive.org/web/20131020022921/http://www.lcsd.gov.hk/b5/ppr_release_det.php?id=4683 |date=2013-10-20 }}</ref>而位於祥龍圍邨附近空地正興建東華三院馬錦燦紀念小學新校舍，以應付北區學位需求緊張問題。此外，因應北區人口持續增加，[[醫管局|醫管局]]落實擴建北區醫院，將附近一片政府用地作為醫院二期之用而應付需求。

== 缺點 ==
=== 位置偏遠 ===
粉嶺/上水新市鎮除了與[[深圳|深圳]]有近在咫尺的優勢外，距離其他[[香港市區|市區]]地方均極遠，車程往往超過一小時，令人卻步。

==未來發展==
根據香港政府最新規劃，[[古洞北|古洞北]]及[[粉嶺北|粉嶺北]]新發展區將是粉嶺／上水新市鎮的擴展部分，整合為'''粉嶺／上水／古洞新市鎮'''。全面發展完成後，估計新市鎮總人口可容納約460,000人居住

2014年，[[民建聯|民建聯]]獲[[香港建造商會|香港建造商會]]研究基金撥款，研究如何活化[[聯和墟|聯和墟]]、[[大埔墟|大埔墟]]和石湖墟。民建聯認為為了配合[[新界東北發展計劃|新界東北發展計劃]]，原墟市亦各有文化特色，既需要保育，亦需要注入新的元素。在[[香港大學|香港大學]]建築學院的協助下，研究團隊舉辦了多場地區工作坊，諮詢居民、商戶及學生等意見。最終，建議石湖墟[[符興街|符興街]]列為永久[[行人專用區|行人專用區]]，與[[新康街|新康街]]連接，令到區內商業活動更加聚集，而位於符興街的食物安全中心辦事處亦為特色建築，可以作為社區用途，在旁擺設露天茶座，有關建議獲得有關部門接納

惟時至今日，相關建議仍沒有進一步規劃和落實。<ref>[http://www.nentnda.gov.hk/doc/pe/info_digest.pdf 新界東北新發展區規劃及工程研究]</ref>。
