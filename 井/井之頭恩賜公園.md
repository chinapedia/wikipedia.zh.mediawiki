'''井之頭恩賜公園'''（{{jpn|j=井の頭恩賜公園|hg=いのかしらおんしこうえん|rm=Inokashira Onshi Kōen}}）是位在[[東京都|東京都]][[武藏野市|武藏野市]]和[[三鷹市|三鷹市]]的[[東京都|都立]][[公園|公園]]。通稱「井之頭公園」。

1917年（[[大正|大正]]6年）5月1日開園。園內的井之頭池和{{link-ja|三宝寺池|石神井公園|三寶寺池}}（{{link-ja|石神井公園|石神井公園}}）、{{link-ja|善福寺池|善福寺公園|善福寺池}}並稱「武藏野三大湧水池」。現由東京都西部公園綠地事務所管理，並被選定為[[日本櫻名所100選|日本櫻名所100選]]。

== 地理 ==
井之頭恩賜公園（以下簡稱「井之頭公園」）是橫跨武藏野市東南部與三鷹市東北部的公園。總面積約380,000[[平方公尺|m<sup>2</sup>]]。

井之頭公園其中心為井之頭池（約43,000[[平方公尺|m<sup>2</sup>]]）<ref name="gotokyo.org">[http://www.gotokyo.org/jp/kanko/mitaka/spot/10522.html 井の頭池 - 東京観光財団]</ref>。井之頭池為西北－東南方向延伸的細長水池，西北方分為兩端，東南方連結以井之頭池為源頭的[[神田川_(東京都)|神田川]]。另外，井之頭池西側是{{link-ja|御殿山 (武藏野市)|御殿山 (武蔵野市)|御殿山}}的雜木林，{{link-ja|吉祥寺通|吉祥寺通り}}對側則是{{link-ja|東京都建設局|東京都建設局}}的[[井之頭自然文化園|井之頭自然文化園]]。雜木林南側有流向東南方的[[玉川上水|玉川上水]]，其南邊是「西園」所在地。西園內有田徑場（競技場）與[[三鷹之森吉卜力美術館|三鷹之森吉卜力美術館]]。玉川上水下游旁則是擁有小廣場的「東園」。

從行政區劃來看，井之頭公園的井之頭池、神田川、井之頭公園站、西園、東園皆屬三鷹市，而井之頭自然文化園與雜木林所在的御殿山則屬於武藏野市。

井之頭池北方500公尺左右是[[中央線快速|JR中央線]][[吉祥寺站|吉祥寺站]]。吉祥寺站設有面向井之頭公園的公園口（南口），從車站到井之頭池之間的道路旁有許多以年輕人為客源的商店。東南方的神田川旁則有與其平行的[[京王井之頭線|京王井之頭線]][[井之頭公園站|井之頭公園站]]。井之頭公園站至井之頭池之間有許多小型廣場與小道。

== 歷史 ==
井之頭池以豐富的水源著稱。園內武藏野市側的御殿山遺跡已挖掘出[[繩文時代|繩文時代]]的{{link-ja|豎穴式住居|竪穴式住居}}遺跡與[[舊石器時代|舊石器時代]]石器、{{link-ja|敷石住居|敷石住居}}等等。

井之頭池西端的島上有座井之頭辯財天（[[別當寺|別當寺]]是[[天台宗|天台宗]]{{link-ja|大盛寺|大盛寺}}），傳聞最早是[[平安時代|平安時代]]中期[[源經基|六孫王經基]]在此設堂安置[[最澄|最澄]]（傳教大師）作的辯財天女像，之後[[源平合戰|源平合戰]]的[[源賴朝|源賴朝]]在平定東國後改建。[[鎌倉時代|鎌倉時代]]末期的[[元弘之亂|元弘之亂]]，[[新田義貞|新田義貞]]與{{link-ja|北條泰家|北条泰家}}的對戰造成辯財天燒毀。在經歷數百年後，才由[[江戶幕府|江戶幕府]]三代將軍[[德川家光|德川家光]]重建辯財天。

井之頭的名稱一說是由家光所命名，傳聞他以小刀在辯財天旁的[[日本辛夷|日本玉蘭木]]上刻下「井之頭」。現在該地立有石碑紀念此傳說。

另一說法是家光到此讚嘆「えんかしら、この水の美しさ」，之後轉變為「いのかしら」。

御殿山的地名是家光為了獵鷹來此時，選擇在能望見井之頭池的地點建造御殿。

{{link-ja|大盛寺|大盛寺}}的《神田御上水井之頭辯財天略緣起》中記載家康曾親手取水，並誇讚是關東第一名水。<ref>安田知代『井の頭公園まるごとガイドブック』株式会社文伸、2008年初版、33頁より引用</ref>
[[江戶時代|江戶時代]]，辯財天成為[[江戶|江戶]]市民的信仰之地與休閒場所。辯財天周邊可見當時商人與[[歌舞伎|歌舞伎]]捐贈的[[燈籠_(建築裝飾)|石燈籠]]、[[宇賀神|宇賀神]]像等。另外過去還曾有石造[[鳥居|鳥居]]，後因明治初年的[[神佛分離|神佛分離令]]而拆除，鳥居的柱石之後用在井之頭池與神田上水之間的[[水門|水門]]上。雖然水門已不再使用，但現在水池東端還可見到水門的遺跡。

江戶時代井之頭池一帶的樹林被指定為幕府御用林，[[明治維新|明治維新]]後由[[東京府|東京府]]收購。
* 1889年（[[明治|明治]]22年） - 為[[宮內省|宮內省]]（現在的[[宮內廳|宮內廳]]）御用林。
* 1913年（[[大正|大正]]2年）12月 - 66,245坪的帝室御料地下賜給[[東京市|東京市]]（其中8,980坪做為井之頭學校〈現在的[[井之頭自然文化園|井之頭自然文化園]]〉用地）<ref name="前島">前島康彥『井之頭公園』財團法人[[東京都公園協會|東京都公園協會]]發行、1980年</ref>。另外收購5,495.58坪、捐贈544坪、神社共用地896坪、池13,672坪<ref name="前島"/>。
* 1917年（大正6年）5月1日 - 作為{{link-ja|恩賜公園|恩賜公園}}開放<ref>大正6年東京市告示第51号</ref>。
* 1920年（大正10年） - 引用池水的游泳池、兒童池（丸池）竣工<ref name="前島"/>。
* 1929年（[[昭和|昭和]]4年）7月 - 開設划船場<ref name="前島"/>。
* 1933年（昭和8年）7月 - 原有的游泳池廢除，新設25公尺長的游泳池與兒童池<ref name="前島"/>。
* 1934年（昭和9年）5月 - 小動物園開園<ref name="前島"/>。
* 1935年（昭和10年）5月 - 新設中之島淡水魚生園（現在的水生物館）<ref name="前島"/>。
* 1937年（昭和12年） - 篤志家捐贈御殿山南斜面的梅林<ref name="前島"/>。
* 1942年（昭和17年）5月17日 - 井之頭自然文化園開園<ref name="前島"/>。
* 1944年（昭和19年） - [[第二次世界大戰|第二次世界大戰]]因木材不足，砍伐池畔15,000顆杉木<ref name="前島"/>。
* 1952年（昭和27年）11月 - 建立[[野口雨情|野口雨情]]歌謠碑<ref name="前島"/>。
* 1953年（昭和28年）5月 - 重建池中央的橋，並命名為七井橋<ref name="前島"/>。
* 1956年（昭和31年）11月3日 - 野外舞台完工<ref name="前島"/>。
* 1973年（昭和48年）3月 - 收購西園地區<ref name="前島"/>。
* 1974年（昭和49年）6月1日 - 西園付費設施多功能運動廣場、網球場開放<ref>昭和49年東京都告示第569号</ref>。
* 1993年（[[平成|平成]]5年）3月30日 - 井之頭恩賜公園連絡橋完工<ref name="前島"/>。
* 2013年（平成25年）5月1日 - 園內的親之井稻荷尊神社燒毀<ref>[http://www.asahi.com/national/update/0502/TKY201305020227.html 井の頭公園の稲荷神社全焼 放火容疑で捜査 東京・三鷹] 朝日新聞 2013年5月2日</ref>。

== 景點 ==
園內的井之頭池設有划船場。南側[[玉川上水|玉川上水]]以南的「西園」內有[[三鷹之森吉卜力美術館|三鷹之森吉卜力美術館]]。吉祥寺通（公園通）西側在內的園區西北部有[[井之頭自然文化園|井之頭自然文化園]]、[[動物園|動物園]]、{{link-ja|北村西望|北村西望}}雕刻館（收藏[[長崎市|長崎市]][[和平公園_(長崎市)|和平公園]]的{{link-ja|平和祈念像|平和祈念像}}原形）、日本庭園等。園外玉川上水附近還有{{link-ja|三鷹市山本有三紀念館|三鷹市山本有三記念館}}。

另外，公園附近有許多商店與餐廳。
