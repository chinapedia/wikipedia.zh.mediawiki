'''井之頭通'''（{{jpn|j=井ノ頭通り|hg=いのかしらどおり}}）是[[東京都|東京都]][[澀谷站|澀谷站]]前至[[武藏野市|武藏野市]]{{link-ja|境淨水場|境浄水場}}的道路名稱。一般日文寫作「井ノ頭通り」或「井の頭通り」。

== 起點、終點 ==
* 起點：東京都[[澀谷區|澀谷區]][[宇田川町|宇田川町]]（澀谷站前附近）
* 終點：[[武藏野市|武藏野市]]關前五丁目（境淨水場附近）

== 概要 ==
大半路段是直線。因利用{{link-ja|境淨水場|境浄水場}}至{{link-ja|和田堀給水所|和田堀給水所}}的水管土地，過去又稱水道道路，之後前首相[[近衛文麿|近衛文麿]]命名為井之頭街道，後[[東京都|東京都]]更名為井之頭通。

本道路在管理上可分為三個路段。
# 澀谷區道（澀谷站前附近 - 代代木公園交番前交差點）
# [[東京都道413號赤坂杉並線|東京都道413號赤坂杉並線]]（濱田山一帶的路段與[[東京都道14號新宿國立線|東京都道14號新宿國立線]]重複）：澀谷區代代木公園交番前交差點 - [[杉並區|杉並區]]環八井之頭交差點
# {{link-ja|東京都道7號杉並秋留野線|東京都道7号杉並あきる野線}}：環八井之頭交差點 - 武藏野市關前五丁目交差點
