'''狩野直喜'''（{{jpn|j=狩野 直喜|hg=かの なおき|rm=Kano Naoki}}，[[字|字]]'''子温'''，[[號|號]]'''君山'''，{{bd|1868年|2月11日|1947年|12月13日|Kano Naoki}}），[[日本|日本]][[汉学家|汉学家]]，历史学者，“[[京都支那学|京都支那学]]”的开创者，是日本近代的大儒。

== 简介 ==
狩野直喜毕业于[[东京帝国大学|东京帝国大学]]文学部汉学科。

1900年，狩野直喜来[[清朝|清朝]]留学，碰上了[[义和团|义和团]]，遭遇困境2个月后回国。

1906年到1928年，狩野直喜在[[京都帝国大学|京都帝国大学]]主持“[[中国哲学|中国哲学]]讲座”长达22年之久。

1911年，狩野直喜开始追踪[[敦煌|敦煌]][[文献|文献]]，访察[[英国|英国]]、[[法国|法国]]、[[俄国|俄国]]、[[美国|美国]]，并来[[中国|中国]]寻访，与中国学者[[王国维|王国维]]建立了深厚的友谊。

== 学术思想 ==
*严密的考证的方法。以弘扬[[清朝|清朝]][[考证学|考证学]]的方法为己任，同时很早就接受[[西方|西方]]的[[实证主义|实证主义]]思想。
*他的中国研究涉及的范围从[[上古|上古]]到[[清代|清代]]，领域从哲学到[[法律|法律]]到[[西学|西学]]，熟稔[[英语|英语]]和[[法语|法语]]。对[[中国哲学|中国哲学]]的研究继承了[[岛田篁村|岛田篁村]]合注疏、义理、考证三者为一的学问精神，而反对仅凭某种纯粹的理论来解释中国的学问。<ref>{{cite book |language =zh-hans|author=狩野直喜|title=《中国哲学史》|year=1953年|publisher=岩波书店|location=|pages=第4页|isbn= }}</ref>
*[[经学|经学]]的立场。狩野所涉猎的领域虽然十分广泛，但是经学的立场非常明确，认为[[戏曲|戏曲]]、[[戏剧|戏剧]]不属于经学。<ref>{{cite book |language =zh-hans|author=仓石武四郎、吉川幸次郎的谈话|title=《先学を語る――狩野直喜博士》|year=1971年|publisher=《东方学》|location=|pages=第142—145页|isbn= }}</ref>
*思想上归宗于[[儒学|儒学]]。[[王国维|王国维]]称他为一代“[[儒宗|儒宗]]”。{{noteTag|name= wangguowei|1=[[清朝|清朝]][[王国维|王国维]]《[[观堂集林|观堂集林]]》卷二十四《送日本狩野博士游欧洲》：“君山博士（狩野直喜，号君山）今儒宗，亭亭崛起[[东海|东海]]东……” }} 
*反对[[霸道|霸道]]，主张[[德治|德治]]，综述[[孔子|孔子]]之教。<ref>{{cite book |language =zh-hans|author=田冈领云|title=《和译汉文丛书》|year=1911|publisher=玄黄社|location=|pages=|isbn= }}</ref><ref>{{cite book |language =zh-hans|author=|title=《近代日本的孔子观》|year=2000年11月|publisher=《孔孟月刊》|location=台湾|pages=第39卷第3期|isbn= }}</ref>
