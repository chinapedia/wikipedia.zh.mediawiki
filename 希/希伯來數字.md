

'''希伯來數字'''系統是一個使用[[希伯來字母|希伯來字母]]並按字母順序的數字系統。

在這個系統，並沒有"零"的存在。而且各個數值都有自己的獨特的字母。

==計算==
'''希伯來語數字'''系統是以數值加起來的形式去組合出來。例如,177(קעז)是由100 + 70 + 7 = 177所組成。

在[[數學|數學]]上，這個系統使用22個[[字母|字母]]。最後一個字母是代表400的tav(ת)。而500(ת"ק)即使用100+400的方式去表示。但有時候，會使用最後五個[[希伯來語|希伯來語]]結尾寫法的[[字母|字母]]去代表500至900的數值。

==表達方式原則==

一般也將最大的數字放在最右為原則(希伯來文是從右邊開始寫)。例如100 + 70 + 7(קעז)與7 + 70 + 100(זעק)同樣是可以表達177這個數目，但一般會將最大的數目放在右面。

另外會以盡量簡短為原則。例如10(י)跟5 + 5(הה)或9 + 1(טא)同樣能表達10這個數目，但一般會採用較少字母的那種寫法。

===例外===

主要為了避免在數字上出現類似上帝的名字而忽視原則。例如15、16，以及115、116等數目，會約定俗成以9 + 6(טו)來表示15、以9 + 7(טז)來表示16，如此類推。

其次可能會因為特別的含意而選擇棄用一般的寫法。例如13的一般寫法是10 + 3(יג)，但是也會以1 + 5 + 2 + 5(אהבה)意思是「愛」來表達13，還有可以以1 + 8 + 4(אחד)意思是「一」(可以用來聲明"上帝是獨一"的"一"的意思)來表達13。
