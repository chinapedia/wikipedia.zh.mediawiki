'''希腊足球乙级联赛'''（Φούτμπολ Λιγκ 2），为[[希腊|希腊]][[足球|足球]]职业联赛的第三级别。联赛分为南区、北区两个组。

== 历史 ==
希腊足球乙级联赛成立于1965年，原为业余联赛，1983年更改为职业联赛。2010年8月3日，更名为乙级联赛。<ref>{{Cite web |url=http://www.epae.org/entry.fds?entryID=1067&pageCode=05.07&tablePageID=47&pageID=46&langid=1 |title=存档副本 |access-date=2010-12-08 |archive-url=https://web.archive.org/web/20120330092937/http://www.epae.org/entry.fds?entryID=1067&pageCode=05.07&tablePageID=47&pageID=46&langid=1 |archive-date=2012-03-30 |dead-url=yes }}</ref>

== 历届冠军 ==
* 1965年 [[莱瓦迪亚克斯足球俱乐部|莱瓦迪亚克斯]]等
* 1985-86年 [[克桑西足球俱乐部|克桑西]]、Charavghiakos
* 2009-10年 [[卡斯托利亚足球俱乐部|卡斯托利亚]]、[[韦里亚足球俱乐部|韦里亚]]
