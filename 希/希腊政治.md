{{希臘政治}}
'''[[希腊|希腊]]'''是一个[[议会民主制|议会民主制]]国家。

== 宪法 ==
希腊现行[[宪法|宪法]]为1975年宪法。

== 国家元首 ==
希腊[[国家元首|国家元首]]为[[希腊总统|希腊总统]]。总统任期为5年。

== 立法 ==
希腊议会为立法机构。希腊议会为[[一院制|一院制]]，共有300名议员，每届任期4年。

== 行政 ==
政府首脑为[[希腊总理|总理]]。總理是希臘的政府首腦，領導內閣。總理一般由國會最大黨的黨魁擔任。

== 司法 ==
希腊最高司法机构包括最高法院、最高行政法院以及检察机构。

== 政党 ==
希腊的主要政党有[[新民主党_(希腊)|新民主党]]、[[激进左翼联盟|激进左翼联盟]]、[[泛希腊社会主义运动|泛希腊社会主义运动]]、[[希腊共产党|希腊共产党]]、[[独立希腊人党|独立希腊人党]]、[[金色黎明党|金色黎明党]]等。
