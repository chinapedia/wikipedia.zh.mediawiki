'''恩贾梅纳'''（{{lang-fr|'''N'Djaména'''}}，{{lang-ar|'''أنجمينا'''}}），[[乍得|乍得]]首都，位于该国西部[[沙里河|沙里河]]畔，与[[喀麦隆|喀麦隆]][[库塞里|库塞里]]隔河相望，是该国第一大城市，2009年人口993492（2005年）。恩贾梅纳分为10个区。主要工业部门有肉类、棉织品加工。恩贾梅纳是乍得重要的内河港口，是[[家畜|家畜]]、[[谷物|谷物]]、[[食盐|食盐]]和[[枣椰|枣椰]]的集散地。恩贾梅纳建于1900年，当时称为'''拉密堡'''（Fort-Lamy），于1973年改为现名。

== 地理 ==
恩贾梅纳位于12°6′47″N 15°2′57″，在沙里河和洛贡河合流处。作为主要的行政中心，恩贾梅纳包括纳斯条状商业中心和住宅区。

== 气候 ==
恩贾梅纳是以短期雨季与长期旱季为特征的半干旱气候，年降水量为750毫米，蒸发量十分大。雨季从六月到九月，八月降水最集中。恩贾梅纳是世界上温度最高的首都之一，有11个月（除8月）平均气温超过32℃。

== 旅游和文化 ==
恩贾梅纳景点包括乍得国家博物馆、一座天主教堂和一些清真寺。沙里河的落日景观十分壮观。恩贾梅纳是2009年的伊斯兰文化之都。

== 教育 ==
恩贾梅纳有两所大学：以[[法语|法语]]为教学语言的[[恩贾梅纳大学|恩贾梅纳大学]]，以[[阿拉伯语|阿拉伯语]]为教学语言的[[费萨尔国王大学|费萨尔国王大学]]。
