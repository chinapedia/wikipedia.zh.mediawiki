《'''[[腦筋急轉彎_(漫畫)|脑筋急转弯]]'''》是[[台湾|台湾]]红极一时的一套画册。1993年1月，《脑筋急转弯》第7至12册以[[四川美术出版社|四川美术出版社]]的名义在[[中国大陆|中国大陆]]出版，然而其中第10册某条漫画引起了中国大陆[[穆斯林|穆斯林]]的强烈反应。

== 背景 ==
《[[腦筋急轉彎_(漫畫)|脑筋急转弯]]》原是台湾《[[中国时报|中国时报]]》家庭版上连载的单篇漫画笑话，1989年12月由[[时报文化|时报文化]]出版发行<ref>{{cite web|title=腦筋急轉彎(1)|url=http://www.readingtimes.com.tw/ReadingTimes/ProductPage.aspx?gp=productdetail&cid=mcig%28SellItems%29&id=FA0072|publisher=時報悅讀網|accessdate=2015-01-18}}</ref>，在台湾红极一时。

1992年8月，台湾人士郝某某联系[[华文出版社|华文出版社]]社长[[肖德荣|肖德荣]]，委托华文出版社出版《脑筋急转弯》第7至13册（后来，肖德荣只同意出版第7至12册）。通过层层委托，最终《脑筋急转弯》第7至12册以[[四川美术出版社|四川美术出版社]]名义办理了[[ISBN|ISBN]]书号，并于1993年1月在[[山东省|山东省]][[烟台市|烟台市]]滨海印刷厂付印出版<ref name="LawYearBook">{{cite book|author=中国法律年鉴编辑部|title=中国法律年鉴·1996|year=1996|publisher=中国法律年鉴社|pages=779}}</ref>。在该套画册第10册第43页、第44页，作者提出了一个[[脑筋急转弯|脑筋急转弯]]：「什麼人看過豬走路卻沒吃過豬肉？」，答案是「回教徒」<ref>{{cite news|author=趙靜瑜|title=雅言出版社：「捍衛閱讀自由」 《魔鬼詩篇》全球唯一中文版台灣發行|url=http://old.ltn.com.tw/2002/new/oct/17/life/art-1.htm|accessdate=2015-01-23|newspaper=[[自由時報|自由時報]]生活藝文網|date=2002-10-17}}</ref>，并且配了一个小[[穆斯林|穆斯林]]男孩朝一幅画着猪的画面做[[礼拜|礼拜]]的漫画<ref>{{cite journal|author=沈银汉|title=青海回民护教运动见闻|journal=北京之春|pages=50|url=http://beijingspring.com/bj2/1995/150/2003129133528.htm|accessdate=2015-01-15|quote=此书原是台湾时报出版公司出的漫画书，其中有两页画着一个小穆斯林男孩朝一幅画着猪的画面做礼拜。这引发了伊斯兰教民的愤怒和不满。|date=1995年1月}}</ref>，这引起了中国大陆穆斯林群體强烈反应<ref name="CASS1">{{cite book|author=江流 等|title=1993-1994年中国社会形势分析与预测|year=1994|publisher=中国社会科学出版社|isbn=7-5004-1489-7|pages=194|quote=四川美术出版社再版了台湾出版的《脑筋急转弯》全套连环画册，其中第10册42-43页，图文并茂采取提问的方式，在穆斯林为什么不吃猪肉的问题上歪曲事实，严重伤害了穆斯林群众的民族宗教感情。}}</ref>。

== 抗议 ==
抗议的浪潮首先从[[四川|四川]]掀起。1993年8月至9月，部分穆斯林群众要求按《[[古兰经|古兰经]]》处死、[[枪毙|枪毙]]四川美术出版社负责人员<ref>{{cite book|author=辛向阳|title=大国诸侯——中国中央与地方关系之结|year=1996|publisher=中国社会出版社|isbn=7-80088-652-2|pages=671}}</ref>，并扬言将四川美术出版社付之一炬<ref>{{cite book|author=吴海民|title=书号魔方——出版者的慢性自杀|year=1995|publisher=华艺出版社|location=北京|isbn=7-80039-662-2|pages=5}}</ref>；四川省伊斯兰教协会、成都市伊斯兰教协会代表穆斯林群众向四川美术出版社表达“最强烈的抗议”，提出了“严惩责任人”等六条意见。此后，[[甘肃省|甘肃省]]、[[兰州|兰州]]、[[临夏州|临夏州]]等地的伊斯兰教协会等召开宗教界人士特别会议，成立了“声讨抗辱委员会”，开展了要求严惩肇事者的签名活动，并分别在[[临夏州|临夏州]]和兰州举行了“抗辱声讨大会”和示威游行<ref name="CASS2">{{cite book|author=江流 等|title=1993-1994年中国社会形势分析与预测|year=1994|publisher=中国社会科学出版社|isbn=7-5004-1489-7|pages=207}}</ref>；伊斯兰教协会还出面组织了“甘肃省穆斯林起诉团”，赴四川、[[北京|北京]]等地进行控告、[[上访|上访]]<ref>{{cite book|author=临夏市地方志编纂委员会|title=临夏市年鉴·1986-1995|year=1996|publisher=兰州大学出版社|isbn=7-311-00948-0|pages=290}}</ref>；成都、甘肃、陕西、云南、湖南等地的穆斯林也采取了种种行动，表达对事件的态度<ref>{{cite book|author=李国光|title=中华人民共和国典型审判案例全书·刑事卷|year=1998|publisher=中国民主法制出版社|isbn=7-80078-312-X|pages=170}}</ref>，青海还发生了较为严重的对抗性冲突事件<ref name="CASS2"/>。

== 处理 ==
1993年8月，四川省新闻出版局对四川美术出版社作出处罚决定，并通知[[中华人民共和国|中国全国]]各地立即收缴、销毁《脑筋急转弯》一书。据统计，全国共收缴该书6000套。四川省内所收缴和查封的2384套，在[[四川省人民检察院|四川省人民检察院]]副院长亲自监督下于成都郊区进行了销毁。销毁的过程请公证处做了公证，还请有关单位负责人和穆斯林群众代表签字作证<ref name="CASS2"/>。四川美术出版社的停业整顿则一直延续到了1994年6月<ref>{{cite journal|author=木|title=四川美术出版社恢复出版业务|journal=出版参考|year=1994|issue=15|pages=3}}</ref>。

1993年10月，山东省新闻出版局对烟台市滨海印刷厂作出处罚决定，处[[罚金|罚金]][[人民幣|人民幣]]5万元，并责令该厂停业整顿<ref>{{cite book|author=石峰|title=中国出版工作调查报告·1994|year=1995|publisher=中国书籍出版社|isbn=7-5068-0506-5|pages=509}}</ref>；当时的[[中华人民共和国新闻出版署|中国新闻出版署]]发布了《关于全面检查涉及伊斯兰教的出版物的通知》，要求对涉及[[伊斯兰教|伊斯兰教]]的出版物进行严格的审查<ref>{{cite book|author=余振贵; 洪长有|title=中国伊斯兰教简志|year=2011|publisher=宗教文化出版社|isbn=978-7-80254-144-3|pages=508}}</ref>。

1994年6月，[[中华人民共和国国家民族事务委员会|中国国家民委]]、[[中共中央宣传部|中共中央宣传部]]、[[中共中央统战部|中共中央统战部]]、[[中华人民共和国文化部|中国文化部]]、[[中华人民共和国广播电影电视部|中国广播电影电视部]]、[[中华人民共和国新闻出版署|新闻出版署]]、[[国务院宗教局|国务院宗教局]]联合发布了《关于严禁在新闻出版和文艺作品中出现损害民族团结内容的通知》<ref>{{cite book|author=国家民族事务委员会经济司; 国家统计局国民经济综合统计司|title=中国民族统计年鉴·1995|year=1995|publisher=民族出版社|location=北京|isbn=7-105-02427-5|pages=45}}</ref>。

1995年5月3日，[[成都市人民检察院|成都市人民检察院]]以《[[中华人民共和国刑法|中华人民共和国刑法]]》玩忽职守罪将涉案的四川美术出版社副社长、总编辑室主任、助理编辑等向[[成都市中级人民法院|成都市中级人民法院]]提起公诉。1995年5月18日，成都市中级人民法院作出刑事附带民事判决，认为上述三人玩忽职守、不正确履行职责，分别处以5至3年的[[有期徒刑|有期徒刑]]。<ref>{{cite journal|author=马夫|title=《脑筋急转弯》案依法审理判决|journal=中国穆斯林|year=1995|issue=04|pages=6}}</ref>一审判决后，三名被告均提出[[上诉|上诉]]。1995年8月9日，[[四川省高级人民法院|四川省高级人民法院]]裁定驳回三被告人的上诉，维持原判<ref name="LawYearBook"/>。
