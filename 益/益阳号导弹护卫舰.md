'''“益阳”号导弹护卫舰'''（[[舷号|舷号]]548）简称'''“益阳”舰'''，是[[中国人民解放军海军|中国人民解放军海军]]第七艘[[054A导弹型护卫舰|054A导弹型护卫舰]]，可以单独或者协同海军其他[[兵力|兵力]]攻击敌[[水面舰艇|水面舰艇]]及[[潜艇|潜艇]]，具有较强的远程警戒和[[防空作战|防空作战]]能力。益阳舰由[[黄埔造船厂|黄埔造船厂]]于2006年开工建造，2009年4月3日命名，同年11月17日下水，2010年5月月底试航，同年10月26日入列，服役于[[东海舰队|东海舰队]][[驱逐舰第六支队|驱逐舰第六支队]]。<ref>{{cite web |url=http://epaper.zhoushan.cn/zsrb/20130512/index.htm |title=解密益阳舰 |publisher=舟山日报 |date=2013-05-12 |deadurl=yes |archiveurl=https://web.archive.org/web/20141006082553/http://epaper.zhoushan.cn/zsrb/20130512/index.htm |archivedate=2014-10-06 }}</ref>益阳舰舰名取自[[湖南省|湖南省]][[益阳市|益阳市]]，其入列命名授旗仪式于2010年10月26日在[[浙江省|浙江省]][[舟山市|舟山市]]某军港举行。<ref>{{cite web|url=http://dhnews.zjol.com.cn/dhnews/system/2010/10/27/012805633.shtml |title=海军“益阳舰”昨入列命名 |publisher=定海新闻网 |date=2010-10-27 }}</ref>

== 历史 ==
2010年4月，“益阳”号导弹护卫舰在某海域进行了一场由总部组织的联合对抗演练。<ref>{{cite web|url= http://navy.81.cn/content/2012-11/26/content_5111828_2.htm|title=某驱逐舰支队：深蓝航道，一支队伍踏浪前行|publisher=中国海军网| date=2012-11-26 }}</ref>益阳舰将厂家学习与出海训练相结合，于入列当年便完成全训，<ref>{{cite web|url= http://www.chinanews.com/mil/2013/09-03/5237901.shtml|title=益阳舰舰长揭示远海拦截作战训练检验三大方面|publisher=中新网| date=2013-09-03 }}</ref>并于次年参加了“东联-2011”远海训练和“蓝鲸-2011”演练。<ref>{{cite web|url= http://www.zgthj.net/Info.aspx?ModelId=1&Id=2506|title=益阳舰护航先进事迹报告团情系鲊埠回族乡寒门学子|publisher=桃江新闻网| date=2013-06-13 }}</ref>

2012年4月22日至27日，“益阳”号导弹护卫舰与[[哈尔滨号导弹驱逐舰|“哈尔滨”号导弹驱逐舰]]、[[舟山号导弹护卫舰|“舟山”号导弹护卫舰]]等20多艘中国舰艇与[[瓦良格号导弹巡洋舰|“瓦良格”号导弹巡洋舰]]、[[沙波什尼科夫海军元帅号反潜舰|“沙波什尼科夫海军元帅”号反潜舰]]等7艘[[俄罗斯|俄罗斯]]舰艇在[[山东省|山东省]][[青岛市|青岛市]]附近海域举行了“[[海上联合-2012|海上联合-2012]]”中俄联合军事演习。<ref>{{cite web|url= http://www.mod.gov.cn/reports/201204/zejy/2012-04/27/content_4362431.htm|title=“海上联合-2012”中俄联合军演在青岛海域举行|publisher=中华人民共和国国防部网| date=2012-04-27 }}</ref>

2012年7月3日，“益阳”号导弹护卫舰、[[常州号导弹护卫舰|“常州”号导弹护卫舰]]及[[千岛湖号综合补给舰舰|“千岛湖”号综合补给舰舰]]组成中国海军第十二批护航编队在浙江省舟山市某军港解缆起航，赶赴[[亚丁湾|亚丁湾]]、[[索马里|索马里]]海域接替第十一批护航编队执行护航任务。<ref>{{cite web|url= http://www.chinanews.com/tp/hd2011/2012/07-03/112079.shtml|title=中国海军第十二批护航编队起航|publisher=中新网| date=2012-07-03 }}</ref>该次护航历时201天，航程12449海里，编队完成46批204艘中外船舶护航任务，查证、驱离可疑船只35批62艘次，并访问[[澳大利亚|澳大利亚]]、[[越南|越南]]两国。益阳舰等于2013年1月19日返抵舟山市某军港。<ref>{{cite web|url= http://navy.81.cn/content/2013-01/19/content_5188088.htm|title=海军第十二批护航编队凯旋|publisher=中国海军网| date=2013-01-19 }}</ref>

2013年5月11日，纪念人民海军建立64周年舰艇开放暨第七届“海洋观教育日”活动在舟山市举行。“益阳”号导弹护卫舰等舰艇向公众开放参观。<ref>{{cite web|url=http://tv.cntv.cn/video/C10108/7f1f96747a733ddd01913a2e0a4e5c44|title=护卫舰“益阳舰”今天向公众开放|publisher=央视网|date=2013-05-11|access-date=2014-10-02|archive-url=https://web.archive.org/web/20141006072117/http://tv.cntv.cn/video/C10108/7f1f96747a733ddd01913a2e0a4e5c44|archive-date=2014-10-06|dead-url=yes}}</ref>

2013年8月26日，“益阳”号导弹护卫舰及“常州”号导弹护卫舰组成解放军海军东海舰队舰艇编队从舟山市某军港起航，赴[[西太平洋|西太平洋]]海域开展例行性训练。期间，编队开展了战备巡逻、舰机协同、远海综合攻防及非战争军事行动等训练。<ref>{{cite web|url= http://news.xinhuanet.com/mil/2013-08/26/c_117100232.htm|title=中国海军舰艇编队赴西太平洋开展远海训练 |publisher= 新华网| date=2013-08-26 }}</ref>9月10日，益阳舰等在完成历时16天的训练任务后，返回舟山市某军港。<ref>{{cite web|url= http://news.xinhuanet.com/politics/2013-09/11/c_125364367.htm|title=东海舰队远海训练编队返回舟山 |publisher= 新华网| date=2013-09-11 }}</ref>

2014年2月28日，“益阳”号导弹护卫舰与[[长春号导弹驱逐舰|“长春”号导弹驱逐舰]]、[[常州号导弹护卫舰|“常州”号导弹护卫舰]]等组成东海舰队远海训练舰艇编队从舟山市某军港起航。<ref>{{cite web|url=http://www.chinamil.com.cn/jfjbmap/content/2014-03/02/content_69030.htm|title=东海舰队远海训练舰艇编队起航|publisher=中国军网|date=2014-03-02|deadurl=yes|archiveurl=https://web.archive.org/web/20141006065602/http://www.chinamil.com.cn/jfjbmap/content/2014-03/02/content_69030.htm|archivedate=2014-10-06}}</ref>3月7日，益阳舰等穿越[[巴士海峡|巴士海峡]]进入西太平洋进行了高强度检验性训练。<ref>{{cite web|url= http://mil.news.sina.com.cn/2014-03-10/1136767992.html|title=中国海军编队西太演习：长春舰距敌千米实弹打击|publisher=新浪网| date=2014-03-10 }}</ref>

2014年6月27日，[[福建省|福建省]][[宁德市|宁德市]]一渔船在[[钓鱼岛|钓鱼岛]]（日本称尖阁诸岛，下同）以北海域作业时进水沉没、人员落水。接到通报后，正在附近海域执行任务的“益阳”号导弹护卫舰和[[三明号导弹护卫舰|“三明”号导弹护卫舰]]全速赶赴事发海域，与[[东海救助局|东海救助局]][[东海救112轮|“东海救112”轮]]及过往船只共同救起5位落水渔民。<ref>{{cite web|url= http://military.people.com.cn/n/2014/0627/c1011-25211261.html|title=海军两艘军舰前往钓鱼岛北部海域搜救遇险失踪人员 |publisher= 人民网| date=2014-06-27 }}</ref>

2014年9月21日，浙江省舟山市一渔船在钓鱼岛以北海域作业时进水沉没、人员落水。接到通报后，“益阳”号导弹护卫舰、东海救助局“东海救112”轮及[[海巡署|海巡署]][[基隆号巡防救难舰|“基隆”号巡防救难舰]]等两岸船舰前往该海域进行搜救并成功救起4位落水渔民。此為[[台灣海峽兩岸|台灣海峽兩岸]]公務船首次在釣島海域合作救援[[中國大陸|中國大陸]]漁民。<ref>{{cite web|url= http://www.huaxia.com/xw/twxw/2014/09/4080522.html|title=两岸公务船首次在钓鱼岛海域合作援救遇险大陆渔民 |publisher= 华夏经纬网| date=2014-09-21 }}</ref>

2014年12月12日，“益阳”号导弹护卫舰、[[泰州号导弹驱逐舰|“泰州”号导弹驱逐舰]]、[[郑州号导弹驱逐舰|“郑州”号导弹驱逐舰]]、[[舟山号导弹护卫舰|“舟山”号导弹护卫舰]]、[[千岛湖号综合补给舰|“千岛湖”号综合补给舰]]及[[北极星号电子侦察船|“北极星”号电子侦察船]]穿越[[宫古海峡|宫古海峡]]进入[[西太平洋|西太平洋]]。<ref>{{cite web|url=http://military.people.com.cn/n/2014/1214/c172467-26203247.html |title=中国海军舰队再次穿过宫古海峡 战机1周4次往返 |publisher=人民网| date=2014-12-14 }}</ref>

2015年11月7日，“益陽”號導彈護衛艦、“千島湖”號綜合補給艦及“濟南”號導彈驅逐艦與美國海軍舉行了為期六小時的聯合演習，這是中美首次在大西洋聯合軍演.此次演習在美國佛羅里達州梅波特港東南的大西洋海域進行,美方參演兵力為導彈驅逐艦“梅森”號和“斯托克”號，導彈巡洋艦“蒙特里”號。<ref>{{Cite web |url = http://udn.com/news/story/7992/1303362-%E5%88%A5%E7%AE%A1%E5%8D%97%E6%B5%B7%E4%BA%86%EF%BC%81-%E4%B8%AD%E7%BE%8E%E5%A4%A7%E8%A5%BF%E6%B4%8B%E8%BB%8D%E6%BC%94 |title = 別管南海了！ 中美大西洋軍演 |language = zh-tw |accessdate = 2015年11月13日 |deadurl = yes |archiveurl = https://web.archive.org/web/20151113100052/http://udn.com/news/story/7992/1303362-%E5%88%A5%E7%AE%A1%E5%8D%97%E6%B5%B7%E4%BA%86%EF%BC%81-%E4%B8%AD%E7%BE%8E%E5%A4%A7%E8%A5%BF%E6%B4%8B%E8%BB%8D%E6%BC%94 |archivedate = 2015年11月13日 }}</ref>
