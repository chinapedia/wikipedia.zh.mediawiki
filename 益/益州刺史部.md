'''益州刺史部'''，是漢朝十三州刺史部之一，州治在雒縣（今[[四川省|四川省]][[廣漢市|廣漢市]]北）。灵帝[[中平|中平]]年間移治綿竹縣（今四川省[[德陽市|德陽市]]東北）。献帝[[興平|興平]]年間又移治成都縣（今四川省[[成都市|成都市]]）。漢代州域範圍大致是今日的[[四川省|四川省]]、[[貴州省|貴州省]]、[[雲南省|雲南省]]大部分，[[陝西省|陝西省]]南部及[[湖北省|湖北省]]、[[甘肅省|甘肅省]]各一隅。

== 名稱由來 ==
「益州」一詞原稱「梁州」，出現始於春秋以後，大體指[[華山|華山]]、[[漢江|漢江]]以南的一片區域。春秋戰國時期，百家爭鳴，著書立說，把禹時的九州冠以稱謂，益州即其一。{{noteTag|九州即冀州、兗州、青州、徐州、揚州、荊州、豫州、幽州、雍州。}}《尚書·禹貢》載：「華陽、黑水惟梁州。」華指今陝西省華山，華陽即為華山之南。黑水其說不一：有今瀾滄江、金沙江、怒江、漢江等。《太康地記》云：「梁者，剛也，取西方金剛之氣剛強，以為名也。其地東據華山，西距黑水。」

== 西漢益州刺史部 ==
西漢時，領蜀郡、巴郡、漢中郡、廣漢郡、武都郡、犍為郡、越巂郡、牂柯郡、益州郡9個郡國。

== 東漢益州刺史部 ==
東漢時，領漢中郡、巴郡、廣漢郡、蜀郡、犍為郡、牂柯郡、越巂郡、益州郡、永昌郡、廣漢屬國、蜀郡屬國、犍為屬國、巴東屬國13個郡國，東漢末置上庸郡、房陵郡、永寧郡（後改稱巴郡）、固陵郡（後改稱巴東郡）、汶山郡、江陽郡等郡，廣漢屬國、蜀郡屬國、犍為屬國分別改稱梓潼郡、漢嘉郡、朱提郡。
