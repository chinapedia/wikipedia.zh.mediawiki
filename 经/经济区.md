'''经济区'''是一个[[区域经济|区域经济]]空间组织的概念，是指具有紧密联系的社会、经济一体化区域<ref name=jjq>{{cite journal|author=赵群毅|title=对环渤海经济圈概念的再认识|journal=北京社会科学|year=2006|issue=2}}</ref>。经济区是一个相对中国化的概念，且在中国大陆实践较多。

== 概述 ==
对于'''经济区'''的涵义、性质，学界多有争论。并且与[[经济圈|经济圈]]、[[都市圈|都市圈]]、[[城市群|城市群]]等经济学、地理学名词相混。然而根据各个名词的字面意义、本质含义以及具体实践，可知这些相关学科名词多有区别，不能混于一谈。

具体到经济区，其是在一定合理范围之内，包含农村、乡镇、城市等多种聚落类型的有着经济、文化、社会等各种联系的地域综合体<ref>{{cite journal|author=丁任重|title=有关经济区的几个概念的探讨|journal=天府新论|year=1985|issue=4}}</ref>。其形成的条件可分为两类：一是自然环境、历史、经济、社会条件等自然因素；二是人为力量（多为国家意志）<ref>{{cite journal|author=于光远|title=“经济区”概念小议|journal=江汉论坛|year=1986|issue=3}}</ref>。依照不同功能特征，经济区可以分为[[部门经济区|部门经济区]]、[[决策经济区|决策经济区]]和[[综合经济区|综合经济区]]三大类型<ref name=jjq/>。

== 具体实践 ==
在中国大陆，从改革开放始，中国逐渐形成了[[长江三角洲地区|长江三角洲地区]]、[[珠江三角洲地区|珠江三角洲地区]]和[[环渤海地区|环渤海地区]]三大经济发展先行区。然而，是时国家层面只是给予宽泛的政策支持，并未有具体的规划纲要引导、指导三大区域发展为联系十分紧密的经济区。自21世纪始，中国大陆实行“区域经济非均衡协调发展战略”，先后提出“西部大开发”、“振兴东北等老工业基地”及“中部崛起”的区域发展战略<ref>{{cite journal|author=杨小军|title=建国60年来我国区域经济发展战略演变及基本经验|journal=现代经济探讨|year=2009|issue=9}}</ref>。并先后出台具体意见、规划引导、指导各经济区的形成和发展。如先后出台《珠江三角洲地区改革发展规划纲要》、《长江三角洲地区区域规划》、《成渝经济区区域规划》、《海峡西岸经济区发展规划》等区域规划。
