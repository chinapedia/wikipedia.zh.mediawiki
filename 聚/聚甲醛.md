'''聚甲醛'''（英文：{{lang|en|Polyoxymethylene}}，以下简称'''POM'''），又称'''聚氧化亚甲基'''、'''聚缩醛'''，是一种在工程中使用的[[热塑性塑料|热塑性塑料]]，适用于高刚性，低摩擦和优异的尺寸稳定性的场合。

聚甲醛典型的[[注塑成型|注塑成型]]应用包括高性能工程部件，例如小[[齿轮|齿轮]]、[[紧固件|紧固件]]、刀柄、[[锁定器|锁定器]]等。该材料被广泛使用在[[汽车|汽车]]和消费电子产业。 [[M16突击步枪|M16突击步枪]]的某些部件也是用它制成。

==发展==

POM最先是由[[德国|德国]][[化学家|化学家]]，1953年[[诺贝尔化学奖|诺贝尔化学奖]]获得者[[赫尔曼·施陶丁格|赫尔曼·施陶丁格]]（德语：{{lang|de|Hermann Staudinger}}）发现的。他在
1920年代研究[[高分子|高分子]]时发现了POM的结构与[[聚合|聚合]]过程。但是由于热稳定性的问题，POM当时并未实现商业化。

在1952年，[[杜邦|杜邦]]公司的化学家合成了另一种POM，并且在1956年为其[[均聚物|均聚物]]申请了专利。杜邦公司将{{lang|en|R N MacDonald}}作为高分子量POM专利发明人。<ref>http://uk.news.dupont.com/site/contenu.asp?idtri=624&idcontenu=56221</ref>{{lang|en|MacDonald}}和他同事的专利描述了[[端基|端基]]为[[半缩醛|半缩醛]]（~O–CH<sub>2</sub>OH）的高分子量POM的制备方法<ref>{{cite patent|US|2768994}}</ref>。但是由于缺乏足够的热稳定性，这种POM还是不能够商用。具备热稳定性也意味着可以商用的POM是由{{lang|en|Dal Nagore}}发明的<ref>{{Cite patent|US|2998409}}</ref>，他发现用[[乙酸酐|乙酸酐]]对POM进行端基处理可以将容易[[解聚|解聚]]的半缩醛转变成热为稳定的，可以融化的[[塑料|塑料]]。

== 性质 ==
POM 的特点是在−40 [[摄氏度|℃]]时仍具有良好的[[强度|强度]]、[[硬度|硬度]]和[[刚性|刚性]]。由于 POM 是高[[结晶度|结晶度]]的[[化合物|化合物]]，所以它本质上是不透明的[[白色|白色]]，但是可以制成其他的[[颜色|颜色]]。POM 的[[密度|密度]]在 1.410–1.420 [[g/cm³|g/cm³]]<ref>[http://tools.ticona.com/tools/mcbasei/product-tools.php?sPolymer=POM&sProduct=CELCON Ticona MSDS for Hostaform <http://tools.ticona.com/tools/mcbasei/product-tools.php?sPolymer=POM&sProduct=HOSTAFORM/Celcon] {{webarchive|url=https://web.archive.org/web/20110512174003/http://tools.ticona.com/tools/mcbasei/product-tools.php?sPolymer=POM&sProduct=CELCON |date=2011-05-12 }}</ref>。

POM 的[[均聚物|均聚物]]是是一种[[半结晶|半结晶]]聚合物（结晶度75–85%），熔点为175℃。POM 的[[共聚物|共聚物]]的熔点稍低，约为162–173℃。

POM 是一种坚硬但[[摩擦系数|摩擦系数]]很低物质。但是它很容易在[[酸|酸]]的[[催化|催化]]下发生[[聚合物降解|聚合物降解]]反应。POM 的均聚物和共聚物都有通过[[封端|封端]]引入的抗解聚的连末端组。对于共聚物来说，第二单元通常是C2（[[乙二醇|乙二醇]]）或C4（[[1,4-丁二醇|1,4-丁二醇]]）单元。这些单元的氧链使用了醚键代替了缩醛基，从而更加稳定不易[[水解|水解]]。POM 对[[氧化剂|氧化剂]]敏感，因此通常加入[[抗氧化剂|抗氧化剂]]以提升材料等级。

POM 的优点：

* [[耐磨性|耐磨性]]高
* 摩擦系数低
* [[耐热性|耐热性]]高
* 良好的电气和[[介电|介电]]性能
* [[吸水率|吸水率]]低

== 生产过程 ==
不同的生产工艺可以制造出不同种类的均聚甲醛和共聚甲醛。

'''均聚甲醛'''

要制造均聚甲醛，首先要制造无水[[甲醛|甲醛]]。主要方法是首先通过水合甲醛（甲二醇，HCH(OH)<sub>2</sub>）与[[乙醇|乙醇]]的反应生成[[甲醛缩|甲醛缩]]（二乙氧基甲烷，CH<sub>2</sub>(OC<sub>2</sub>H<sub>5</sub>)<sub>2</sub>），再将甲缩醛与水的混合物通过[[萃取|萃取]]或[[真空蒸馏|真空蒸馏]]的方法脱水，然后通过加热甲缩醛的方式释放其中的甲醛。此时甲醛在[[阴离子催化|阴离子催化]]下开始聚合，然后通过[[乙酸酐|乙酸酐]]进行封端处理，得到稳定的均聚甲醛。本方法最典型的例子是[[杜邦|杜邦]]公司生产的特灵（Delrin）。

'''共聚甲醛'''

要制造共聚甲醛，首先要把甲醛转化为三氧杂环已烷（特别是[[1,3,5-三氧杂环己烷|1,3,5-三氧杂环己烷]]，又称三聚甲醛）
