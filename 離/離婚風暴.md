《'''離婚風暴'''》 ({{lang-fr|'''Gett: Le procès de Viviane Amsalem'''}}) 是一部2014年[[以色列|以色列]]和[[法国|法国]]合拍剧情片，[[Ronit_Elkabetz|Ronit Elkabetz]]和[[Shlomi_Elkabetz|Shlomi Elkabetz]]联合执导。为[[2014年戛纳电影节|2014年戛纳电影节]]“导演双周”单元的竞赛片。<ref name="Cannes2014">{{cite web |url=http://www.screendaily.com/festivals/cannes/cannes-directors-fortnight-2014-lineup-unveiled/5070844.article |title=Cannes Directors' Fortnight 2014 lineup unveiled |accessdate=26 April 2014 |work=Screendaily}}</ref>也入选[[多伦多影展|多伦多影展]]<ref name="TIFF">{{cite web |url=http://www.indiewire.com/article/toronto-international-film-festival-reveals-7-big-new-lineups-20140812 |title=TIFF Adds 'Clouds of Sils Maria' and 'Two Days, One Night,' Reveals 5 More Lineups |accessdate=28 August 2014 |work=Indiewire}}</ref>。

本片代表以色列角逐第87届[[奥斯卡最佳外语片|奥斯卡最佳外语片]]。<ref name="Israel">{{cite web |url=http://www.hollywoodreporter.com/news/oscars-israel-nominates-gett-trial-734564 |title=Oscars: Israel Nominates 'Gett, the Trial of Viviane Amsalem' in Foreign-Language Category |accessdate=21 September 2014 |work=Hollywood Reporter}}</ref>

== 剧情 ==
讲述在以色列发生的一件[[离婚|离婚]]诉讼。女主角薇薇安·阿姆萨勒姆三年前已提出结婚，然而她丈夫伊莱莎并不同意。面对宗教裁决人的暧昧态度，薇薇安已下定决心要争取自己的自由。
