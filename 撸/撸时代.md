'''撸时代'''是由[[喷TA秀工作室|喷TA秀工作室]]于2013年10月11日所出品的一部网络动画，该动画的发行者为[[penta工作室|penta工作室]]，导演为李豪凌（而第二季则为史涓生），编剧则为尔康（而第二季则为猴子骑狗），制片人则为[[伊芙蕾雅|伊芙蕾雅]]。该动画的风格类似于《[[十万个冷笑话|十万个冷笑话]]》，背景取材于《[[英雄联盟|英雄联盟]]》，而剧情则讲述了一群热爱《英雄联盟》的当代中国大陆大学生的日常生活故事。<ref>{{Cite web |url = http://games.qq.com/a/20131021/001187.htm#p=1 |title = LOL国产动画《撸时代》引追捧 不看你就OUT了 |date = 2013年10月21日 |publisher = 腾讯网 |language = zh-cn }}</ref>目前该动画已推出第一季与第二季，其中第一季于2013年10月18日开播，<ref>{{Cite web |url = http://news.uuu9.com/2013/201310/307780.shtml |title = 英雄联盟改编动画《撸时代》开播 崩三路的猪队友 |author = 小梦不着 |date = 2013年10月22日 |publisher =  游久网 |language = zh-cn }}</ref>而第二季则于2014年11月26日开播。
