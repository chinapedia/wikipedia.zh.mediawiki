《'''她叫妮可'''》（'''''Her Name Is Nicole'''''）是美國女子組合[[小野貓|小野貓]]的主唱[[妮可·舒辛格|妮可·舒辛格]]的首張個人專輯。<ref name=youtube>DJ Bonics （2007年4月16日）. [http://www.youtube.com/watch?v=0LyYnwDI3n8 DJ Bonics and Nicole from Pussy Cat Dolls!!!]. YouTube. 引用於2007年4月16日</ref>

== 專輯背景 ==
《她叫妮可》原定是在2007年推出的，其推出日期也更改了多次。最原先這張專輯會在2007年10月16日推出，後來更改至2007年11月6日，但由於該日遇上了兩名強勁對手──[[杰斯|杰斯]]（Jay-Z）和[[克里斯小子|克里斯小子]]（Chris Brown）的專輯推出，再推遲兩星期至2007年11月20日推出。<ref>[http://www.interscope.com/artist/news/default.aspx/nid/8471/aid/455 HerNameisNicole.com is now Live!]. Interscope Records （2007年8月4日）. 引用於2007年8月21日</ref><ref name=finalrelease>[http://www.rap-up.com/2007/09/28/nicole-scherzinger-gets-new-release-date/ Nicole Scherzinger gets new release date]. Rap-Up （2007年9月28日）. 引用於2007年9月30日.</ref><ref name=finalrelease/>因為缺乏宣傳令到單曲表現不佳，現在已推遲至在2008年1月推出。妮可亦會重新灌錄當中的歌曲。<ref>{{cite web |url=http://hollywoodinsider.ew.com/2007/10/nicole-scherzin.html |title=存档副本 |accessdate=2007-11-10 |deadurl=yes |archiveurl=https://web.archive.org/web/20071102080450/http://hollywoodinsider.ew.com/2007/10/nicole-scherzin.html |archivedate=2007-11-02 }}</ref>

這張專輯被證實將會在小野貓的第二張專輯推出後才會發市。

== 歌曲 ==
這張專輯由多名著名製作人製作，包括[[Akon|Akon]]、[[Kanye_West|Kanye West]]、Macy Gray、Ne-Yo、Pharrell Williams、Polow da Don、Sean Garrett、Timbaland、will.i.am等。<ref name=youtube/><ref name=nicole>James Montgomery, with reporting by Tim Kash （2007年8月23日）[http://www.mtv.com/news/articles/1567662/20070822/scherzinger__nicole.jhtml Nicole Scherzinger Nabs Kanye West for LP, Makes Snow Patrol Singer's Mom Cry] {{webarchive|url=https://web.archive.org/web/20081226225953/http://www.mtv.com/news/articles/1567662/20070822/scherzinger__nicole.jhtml |date=2008-12-26 }}. MTV. 引用於2007年8月26日.</ref> 曾創作[[蕾哈娜|蕾哈娜]]歌曲《雨傘》（Umbrella）的The-Dream，和妮可合作了歌曲《Power's Out》，這首歌曲會有[[Sting|Sting]]的特別出場。<ref name=billboard>Mariel Concepcion （2007年9月28日）[http://www.billboard.com/bbcom/news/article_display.jsp?vnu_content_id=1003648655 'Umbrella' Writer The-Dream Preps Solo Debut]. ''Billboard''. 引用於2007年9月28日.</ref><ref name=popculture>[http://thepopculturejunkie.blogspot.com/2007/09/what-dreams-are-made-of.html What Dreams are Made of...]. The Pop Culture Junkie （2007年9月3日）. 引用於2007年10月3日.</ref>由[[提姆巴蘭|提姆巴蘭]]（Timbaland）製作和特別出場的歌曲《Physical》在電影《[[神奇四俠：銀魔現身|神奇四俠：銀魔現身]]》出現。

早期在網上洩漏的歌曲《Steam》被指為第一首單曲，但是妮可稱這首歌曲不會在專輯出現。<ref name=youtube/>

第一首單曲《Whatever U Like》（想怎樣都行）在2007年7月24日在[[美國|美國]]和[[加拿大|加拿大]]同時推出，在2007年8月13日在[[英國|英國]]推出。第二首單曲《Baby Love》（寶貝的愛）由will.i.am製作和特別出場，在2007年9月18日正式在收音機推出。<ref>[http://www.fmqb.com/Article.asp?id=16691 CHR: Available for Airplay] {{webarchive|url=https://web.archive.org/web/20140628000000/http://www.fmqb.com/Article.asp?id=16691 |date=2014-06-28 }}. FMQB. Accessed 引用於2007年9月18日.</ref> 廠牌Interscope和妮可的官方網站把《Baby Love》稱為專輯的第一單曲，使媒體猜測是由於《Whatever U Like》強差人意的表現引起。<ref>{{Cite web |url=http://www.buddytv.com/articles/pussycat-dolls-present-the-search-for-the-next-doll/pussycat-dolls-trouble-ahead-f-12043.aspx |title=存档副本 |access-date=2007-11-10 |archive-url=https://web.archive.org/web/20071111023936/http://www.buddytv.com/articles/pussycat-dolls-present-the-search-for-the-next-doll/pussycat-dolls-trouble-ahead-f-12043.aspx |archive-date=2007-11-11 |dead-url=yes }}</ref>

在2007年11月16日，妮可的官方網站 [http://www.hernameisnicole.com/ HerNameIsNicole.com] 上載了《Her Name Is Nicole》的四首新歌，包括《Supervillain》、《Happily Never After》、《Who's Gonna Love You》和《Powers Out》，並讓她的支持者投出下一首單曲。後來加入了《Puakenikeni》和《Physical》再讓支持者投票。至投票終結日，《Puakenikeni》得到了最高票數。

《Puakenikeni》在12月10日經由官方網站證實了為第二首官方單曲，在12月18日透過[[ITunes|ITunes]]以及[[Amazon.com|Amazon.com]]發行為下載單曲。
