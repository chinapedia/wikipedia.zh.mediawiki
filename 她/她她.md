'''她她'''，[[香港|港]][[台灣|台]]女子組合，成員為'''Kartina'''和'''Penny'''<ref>{{cite news |language = zh-hk |url = http://orientaldaily.on.cc/cnt/entertainment/20141128/00282_112.html |title = 她她坦蕩蕩互爆閨密事 |publisher = 東方日報 |date = 2014-11-28 }}</ref>。兩人於2013年開始通過網絡上載翻唱流行歌曲的視頻，包括三天內超過300,000點擊率的《愛你》及超過200,000點擊率的《入陣曲》，曾被台灣媒體封“小[[S.H.E|S.H.E]]”的稱號，並先後被邀請上《[[康熙來了|康熙來了]]》，《[[大學生了沒|大學生了沒]]》及《[[綜藝大熱門|綜藝大熱門]]》等當紅節目作嘉賓<ref>{{cite news |language = zh-hk |url = http://www.stars-hk.com/wsp-topic.php?tid=123162 |title = 跨地美女組合“她她”美貌創作實力兼具 |publisher = Stars HK娛網 |date = 2014-09-04 |deadurl = yes |archiveurl = https://web.archive.org/web/20150924105809/http://www.stars-hk.com/wsp-topic.php?tid=123162 |archivedate = 2015年9月24日 |df =  }}</ref>。現為上騰制作旗下歌手，於2014年推出第一首派台曲《約定》。2015年參加浙江衛視節目《[[中國好聲音|中國好聲音]]》第四季，演唱《Bang Bang》。隨後，參加珠江頻道選秀節目《[[麥王爭霸|麥王爭霸]]》的比賽，最終止步於全國八強；同年作為踢館選手參加第八屆《[[超級星光大道|超級星光大道]]》第十二期的踢館賽，並踢館成功。2016年擔任廣東衛視全媒體造星音樂真人秀節目《我是直播歌手》形象大使、打氣導師及評委。

== 成員背景 ==
=== Kartina（奚漪）===
Kartina來自[[香港|香港]]，5歲的時候開始學習古典鋼琴；她6歲時在香港加入兒童合唱團；11歲時就已經代表香港，隨著合唱團到北京、馬來西亞、新加坡等國家參加比賽，並獲得優異成績；她曾經於香港校際音樂節榮獲多個歌唱獎項；12歲的時候更以鋼琴獎學金，考進英國[[北安普頓|北安普頓]]一名校；居英期間，她曾獲得「英國音樂節」歌唱及鋼琴多個獎項；2012年，於「英國現代音樂學院」歌唱表演系畢業<ref name="TaTa - 框格音樂事務">{{cite web|publisher=框格音樂|url=http://www.hongermusic.com/tata.php|title=TaTa - 框格音樂事務}}</ref>。

=== Penny（余姿昀）===
Penny來自[[台灣|台灣]]，5歲的時候開始學習古典鋼琴；她11歲時學習法國號（法國圓號）；12歲時已考取平鎮國中管樂班，擔任法國號首席，並於三年內共獲全國管樂比賽第一名等多個獎項；15歲的時候以主修法國號、副修鋼琴考取國立武陵高級中學的音樂班，高中時更參加了流行音樂社，開始接觸流行及搖滾音樂，擔任主唱三年，多次獲得校園歌唱比賽第一名；她曾跟隨臺灣知名歌唱老師秀珠老師學習，亦曾參與過配唱電視廣告等工作；大學時期，她參加不同的歌唱比賽、認識了前經理人、簽約上海創翊音樂臺灣部門、與兩位朋友Kartina、岡怡一同學習歌唱、舞蹈<ref name="TaTa - 框格音樂事務"/>。

== 組合簡介 ==
=== 她她組合 ===
來自台灣的余姿昀Penny，與來自香港的Kartina，她們本來互不相干。當年，Kartina到英國學習音樂，機緣巧合之下來到台灣參加一個歌唱比賽，後來更被唱片公司發掘。余姿昀Penny則同樣參加歌唱比賽而被發掘。其後，她們透過一個歌唱老師的介紹下認識彼此，發現大家同樣喜歡R&B的音樂作品，也視[[Avril_Lavigne|Avril Lavigne]]為偶像。由於對音樂的追求、音樂理念等不謀而合，並發現自己與對方的聲音更恰好能互補，Kartina的聲音較為厚和及低沉，Penny的聲音則高音及響亮，她們合唱和音時，更見特色。因此消除了兩人之間的語言隔閡，她們並開始改編及翻唱歌曲。<ref>{{cite web|publisher=V1周報|url=http://v1paper.com/archives/8352|title=打破地域的約定 ‧「她」和「她」的音樂夢 – 她她}}</ref>

2013年2月，余姿昀、Kartina和[[林岡怡|林岡怡]]（現改名為[[林妍柔|林妍柔]]）上載合唱數曲《愛你》到分享网站[[Youtube|Youtube]]，並成為臉書的熱門轉貼影片。其後持續上載合唱視頻到網上。三人因歌聲清亮，人氣高漲，更被稱為「新一代S.H.E」，備受网上關注。先后被邀請上“康熙來了”及“大學生了沒”等當紅節目，並接受媒體採訪。當時，有消息指三人會以女子組合出道，並推出專輯，但後來三人各自發展<ref>{{cite news|url=http://www.appledaily.com.tw/realtimenews/article/new/20130221/167180/|author=壹電視|title=新一代S.H.E」超萌美少女唱《愛你》爆紅|newspaper=[[蘋果日報|蘋果日報]]|date=2013-02-21|accessdate=2016-03-24}}</ref>。

2014年余姿昀與Kartina二人因翻唱[[五月天|五月天]]的「入陣曲」而重新受到關注<ref>{{cite news |language = zh-hk |url = http://www.setn.com/E/News.aspx?NewsID=28555 |title = 獨家／人美聲音也甜！網路正妹組樂團「她她」好夯 |publisher = 三立新聞網 |date = 2014-06-27 }}</ref>。並於同年正式成為框格音樂旗下歌手，組成了新晉女子組合「她她」<ref>{{cite news |language = zh-hk |url = http://the-sun.on.cc/cnt/entertainment/20141128/00470_096.html |title = 她她自爆半裸拍片 |publisher = 太陽報 |date = 2014-11-28 }}</ref>。

其組合名“她她”源自於她們認為「她」字很特別，不論廣東話或普通話，兩者發音均為一樣，而且繁體與簡體的寫法都為相同，因此認為可以代表他們兩人即使來自不同地方，處於不同的文化，音樂大同，信念都能夠一致。

=== 正式出道 ===
2014年初，她她經著名音樂人[[陳少寶|陳少寶]]慧眼相中，正式組成她她組合進軍樂壇，並以翻唱一首[[王菲|王菲]]的《約定》，於2014年5月正式出道樂壇。2015年夏天，她她參加了《[[中國好聲音|中國好聲音]]》第四季，演唱《Bang Bang》，雖然盲選未能獲4位導師轉身，但網上人氣卻因[[周杰倫|周杰倫]]事後大加讚賞而不斷飆升。<ref>{{cite news |language = zh-hk |url = http://dailynews.sina.com/bg/ent/music/sinacn/20150729/02026816379.html |title = 她她組合登好聲音 饒舌表演獲導師肯定 |publisher = 新浪網 |date = 2015-07-29 }}</ref>
