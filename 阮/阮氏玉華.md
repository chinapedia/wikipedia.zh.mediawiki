'''阮氏玉華'''（{{vie|v=Nguyễn Thị Ngọc Hoa|hn=阮氏玉華}}，{{bd|？||1645年|}}），今俗称作“阮福玉華”，是[[越南|越南]][[廣南國|廣南國]]君主[[阮熙宗|阮熙宗]]阮福源之[[養女|養女]]，嫁給[[日本|日本]][[安土桃山時代|安土桃山時代]]至[[江戶時代|江戶時代]]商人[[荒木宗太郎|荒木宗太郎]]為妻<ref name="Ngọc Hoa">{{Cite web |title=CÔNG NỮ NGỌC HOA, NÀNG LÀ AI?|url=http://www.erct.com/2-ThoVan/3-Unna/CongnuNgocHoa.htm}}</ref><ref name="Hội An">{{Cite web|title=Đặt tên đường Công nữ Ngọc Hoa ở Hội An|url=http://www.thanhnien.com.vn/van-hoa-nghe-thuat/dat-ten-duong-cong-nu-ngoc-hoa-o-hoi-an-320539.html}}{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref><ref name="肥後">{{Cite web |title=朱印船貿易と肥後|url=http://www.city.tamana.lg.jp/q/aview/472/1587.html}}</ref><ref name="王女">{{Cite web |title=荒木宗太郎と安南国王女|url=http://www.mekong.ne.jp/linkage/arakisoutaro.htm}}</ref><ref name="アーニオ姫">{{Cite web |title=荒木宗太郎とアーニオ姫|url=http://dream-nagasaki.jp/vietnam/anio-/anio-top.htm}}</ref>。

==稱謂==
阮氏玉華在越南及日本共有多個不同的稱謂，由於其養父阮福源統治期間為[[黎中興朝|黎中興朝]]時代，身為中部實際統治者的[[阮主|阮氏]]尚未稱王稱帝，名義上向後黎朝皇帝稱臣。阮福源當時僅稱瑞國公，其女兒皆稱[[公女|公女]]（{{vie|v=Công nữ|hn=公女}}），因此多稱為'''玉華公女'''（{{vie|v=Công nữ Ngọc Hoa|hn=公女玉華}}）<ref name="Hội An" />。雖然阮氏玉華沒有被封[[公主_(東亞)|公主]]（{{vie|v=Công Chúa|hn=公主}}）爵位，但「[[公主|公主]]」一詞在[[越南語|越南語]]裡常用作代稱君主女兒，因此也被後人通稱為'''玉華公主'''（{{vie|v=Công Chúa Ngọc Hoa|hn=公主玉華}}）。

阮氏玉華嫁到日本[[肥後國|肥後國]][[長崎|長崎]]（今[[長崎縣|長崎縣]][[長崎市|長崎市]]）後，改[[日本人名|日本名]]'''王加久'''（{{lang-ja|王加久}}、{{lang|ja|わかく}}）<ref name="嫁">{{Cite web |title=日本に嫁いだベトナムの王女|url=http://www.vietnamguide.com/newsdetail.php?catID=1&cat1ID=19&cat2ID=73&newsID=1701}}</ref>，又作'''王加久戶賣'''（{{lang-ja|王加久戸売}}、{{lang|ja|わかくとめ}}）<ref name="荒木">{{Cite web |title=荒木宗太郎|url=https://kotobank.jp/word/%E8%8D%92%E6%9C%A8%E5%AE%97%E5%A4%AA%E9%83%8E-27708}}</ref>、'''王加久戶女'''（{{lang-ja|王加久戸女}}、{{lang|ja|わかくとめ}}）<ref name="王女" />。她又被長崎人稱為「'''{{lang|ja|アニオーさん}}'''」，此名稱由來有不同說法，一種說法指這是越南語「阿娘」（{{vie|v=A Nương|hn=阿娘}}）的音轉<ref name="嫁" />，又有說法指是源自她平日說話常稱人為「{{lang|vi|anh}}」，而「{{lang|vi|anh}}」在[[越南語代詞|越南語代詞]]中是對同輩男性的敬稱，本義是[[哥哥|哥哥]]，在長崎人以訛傳訛下音變成「'''{{lang|ja|アニオー}}'''」<ref name="王女" />。又因為她出身高貴，被後世[[日本人|日本人]]尊稱為「'''{{lang|ja|アニオー姬}}'''」（{{lang|ja|アニオー姫}}）<ref name="アーニオ姫">{{Cite web |title=荒木宗太郎とアーニオ姫|url=http://dream-nagasaki.jp/vietnam/anio-/anio-top.htm}}</ref>。

==生平==
阮氏玉華是[[廣南國|廣南國]][[阮主|阮主]]阮福源之養女<ref name="Ngọc Hoa" /><ref name="Hội An" />，其親生父母不詳，身世有不同說法。一種說法指她是[[外戚|外戚]]之女<ref name="荒木" />，又有說法指她本是[[宮女|宮女]]，阮福源把她嫁給荒木宗太郎時，為了提高她的身份，就把她收為養女<ref name="南河">{{cite book |author=蔣國學 |title=越南南河阮氏政權海外貿易研究|publisher= 世界圖書出版社 |year= 2009 |isbn= 9787510028892 }}</ref>。由於她名字中的「華」（{{lang|vi|Hoa}}）字與阮福源其中一位親女[[阮氏玉誇|阮氏玉誇]]的「誇」（{{lang|vi|Khoa}}）字發音相近，常被混為一人。但阮氏玉誇[[和親|和親]][[占城|占城]]，嫁給占婆王[[波羅美|波羅美]]，並非嫁到日本，因此與阮氏玉華並非同一人<ref name="Ngọc Hoa" />。

後黎朝[[弘定|弘定]]二十年（1619年，日本[[元和|元和]]五年），阮福源在廣南國{{noteTag|日本文書中稱為「[[安南|安南]]」<ref name="荒木" />}}首都[[會安|會安]]接見來自日本的[[朱印船|朱印船]]商人荒木宗太郎，荒木宗太郎得到阮主的信任。為了發展與日本的[[貿易|貿易]]和使兩國關係緊密，阮福源就把養女阮氏玉華{{link-ja|許配|許婚}}給荒木宗太郎為妻，並賜荒木宗太郎[[越南人名|越南名]]阮太郎（{{vie|v=Nguyễn Thái Lang|hn=阮太郎}}）{{noteTag|一作「阮大良」<ref name="南河" />}}<ref name="南河" />。翌年，阮氏玉華隨同丈夫到日本[[肥後國|肥後國]]長崎，居於其夫在浦上淵村稲佐鄉[[飽之浦|飽之浦]]（{{lang-ja|飽の浦}}，今[[長崎市|長崎市]][[飽之浦町|飽之浦町]]）興建的大屋<ref name="人物">{{Cite web |title=荒木　宗太郎　 （？～１６３６年）|url=http://www.mekong.ne.jp/database/special/sengokugosho/arakisoutaro/}}</ref><ref name="荒木" />，並取日本名'''王加久'''，又稱'''王加久戶賣'''、'''王加久戶女'''。又被長崎人稱呼為「'''{{lang|ja|アニオーさん}}'''」<ref name="王女" /><ref name="嫁" />。

阮氏玉華與荒木宗太郎育有一女[[荒木家須|家須]]，自遠嫁日本後，阮氏玉華就一直沒有回到[[娘家|娘家]]，即使其夫在1622年（後黎朝[[永祚|永祚]]四年、日本元和八年）再到廣南國，她亦沒有隨行<ref name="王女" /><ref name="Hội An" />。[[日本鎖國|日本鎖國]]後，她與丈夫、女兒居住在長崎，其夫於1636年（日本[[寬永|寬永]]十三年，後黎朝[[陽和|陽和]]二年）農曆十一月初七於家中逝世，而阮氏玉華仍繼續於長崎生活，直至1645年（越南後黎朝[[福泰|福泰]]三年、[[莫朝|莫朝]][[順德|順德]]八年；日本[[正保|正保]]二年）逝世，在日本生活了26年<ref name="Hội An" />。她和丈夫合葬於長崎{{link-ja|大音寺|大音寺}}後山墓地<ref name="人物" /><ref name="荒木" /><ref name="アーニオ姫" /><ref name="墓地">{{Cite web |title=荒木宗太郎墓地|url=http://www.city.nagasaki.lg.jp/shimin/190001/192001/p000691.html}}</ref>。

==相關文化及紀念==
阮氏玉華在越南及日本皆被視為越日友好的象徵，兩地皆有以她為題材的文化及紀念。

在越南[[會安市|會安市]]有一條[[街道|街道]]名為公女玉華街（{{vie|v=Đường Công nữ Ngọc Hoa|hn=Đường 公女玉華}}），是以阮氏玉華命名<ref name="Hội An" />。2014年，會安市舉行會安日本祭，演出以阮氏玉華嫁給荒木宗太郎為題材的[[舞蹈|舞蹈]]<ref name="công nữ">{{Cite web |title=Hội An cổ sự: Duyên tình nàng công nữ|url=http://www.thanhnien.com.vn/van-hoa-nghe-thuat/hoi-an-co-su-duyen-tinh-nang-cong-nu-460630.html}}</ref>。

在日本長崎市，每年在[[鎮西大社諏訪神社|諏訪神社]]舉行的{{link-ja|長崎供日|長崎くんち}}的奉納-{踊}-中都有「{{lang|ja|アニオーさんの行列}}」，以荒木宗太郎和阮氏玉華的盛大[[婚禮|婚禮]]為題材，由演員扮演荒木宗太郎夫婦站在[[道具|道具]]朱印船上<ref name="人物" /><ref name="王女" /><ref name="アーニオ姫" />。而她從越南帶來的一面[[歐洲|歐洲]]製方[[鏡|鏡]]則是[[長崎歷史文化博物館|長崎歷史文化博物館]]的藏品<ref>{{Cite web |title=大ベトナム展|url=http://www.kyuhaku.jp/exhibition/exhibition_s31.html}}</ref>。
