'''阮氏芳草'''<ref>{{cite web |url=http://cn.nhandan.com.vn/newest/item/6644501 |title=越捷航空公司执行长阮氏芳草在全球最具权力女性榜上升11位|date=2018年12月11日|language=zh-hans|publisher=越南[[人民报_(越南)|人民报]]网|access-date=2019年2月28日}}</ref>（{{lang-vi|Nguyễn Thị Phương Thảo}}，{{bd|1970年|6月7日|||}}），生於[[越南|越南]][[河內|河內]]<ref>{{cite web |url=http://finance.sina.com.cn/roll/2016-03-31/doc-ifxqxcnz8949551.shtml |title=图文：70后阮氏芳草将成越南首位女亿万富豪|date=2016年3月31日|website=[[新浪网|新浪网]]|language=zh-cn|publisher=荆楚网-楚天金报|access-date=2019年2月28日|archive-url=http://ctjb.cnhubei.com/html/ctjb/20160331/ctjb2872292.html |archive-date=未知日期}}</ref>，[[企業家|企業家]]與[[億萬富豪|億萬富豪]]，[[越捷航空|越捷航空]]創辦人及其現任[[執行長|執行長]]。

==生平==
1988年，阮氏芳草在[[莫斯科|莫斯科]]學習[[金融|金融]]與[[經濟學|經濟學]]時開始從商，並由[[日本|日本]]、[[南韓|南韓]]與[[香港|香港]]等地，進口貨物至[[蘇聯|蘇聯]]境內<ref>{{cite web |url=http://view.inews.qq.com/a/NEW2017011202396202 |title=女博士白手起家 成越南首位女亿万富翁|date=2017年1月12日|website=[[腾讯网|腾讯网]]|language=zh-cn|publisher=|access-date=2019年2月28日}}</ref>。

2007年，她在[[胡志明市|胡志明市]]創辦[[越捷航空|越捷航空]]，是為[[越南|越南]]第一間[[私人企業|私人]][[航空公司|航空公司]]。

2016年，越捷航空首次公開募股，她也逐漸成為越南首位[[女性|女性]]億萬富豪<ref>{{cite web |url=http://www.sohu.com/a/124335987_320672 |title=比基尼空姐造就女富豪，越捷航空老板登上越南女首富宝座|date=2017年1月14日|website=[[搜狐网|搜狐网]]|language=zh-cn|publisher=[[彭博新聞社|彭博]]商业周刊/中文版|author=Sterling Wong|access-date=2019年2月28日}}</ref>。
