'''歙砚'''是指出产于[[歙州|歙州]]（[[徽州|徽州]]）的[[砚|砚]]，是中国四大名砚之一，与甘肃[[洮砚|洮砚]]、广东端砚、山西澄泥砚齐名。<ref>{{cite journal |title=解读歙砚 |author=胡中泰 |journal=收藏 |year=2010 |volume= |issue=9 |pages=121 |url=http://www.cqvip.com/qk/80240x/2010009/35041565.html }}</ref>

== 产地 ==
歙砚石产出地点较多，基本上分布于[[黄山|黄山]]山脉南麓、[[天目山|天目山]]与[[白际山|白际山]]西麓，而以[[婺源|婺源]]与[[休宁|休宁]]交界处的[[龙尾山|龙尾山]]（亦叫罗纹山，在现今的[[婺源县|婺源县]]溪头乡境内）下溪涧中所产为最优，称为“龙尾砚”。<ref>{{cite journal |title=天然板岩歙砚及龙尾砚石 |author=傅广生, 何永兰, 李雷 |journal=石材 |year=1996 |volume= |issue=3 |pages= |url=http://www.cnki.com.cn/Article/CJFDTotal-SCAA199603032.htm }}</ref><ref>{{cite journal |title=歙砚拾珍 |author=丁火 |journal=中国收藏 |year=2006 |volume= |issue=2 |pages=70-71 |url=http://www.cqvip.com/qk/83677x/2006002/21213761.html }}</ref>[[宋朝|宋]][[曹继善|曹继善]]《歙砚说》载：“龙尾山亦名罗纹山，下名芙蓉溪，石坑最多种，延蔓百余里，取之不绝。”<ref name="刘志斌">{{cite journal |title=歙砚在砚史发展中的历史概况及地位 |author=刘志斌 |journal=艺术生活 |year=2009 |volume= |issue=4 |pages=69-71 |url=http://www.cqvip.com/qk/80267x/2009004/30694274.html }}</ref>

== 历史 ==
一般认为歙砚的开采制作始于[[唐朝|唐]][[开元|开元]]年间，迄今已有1300多年的历史。[[北宋|北宋]][[治平|治平]]年间婺源县令[[唐积|唐积]]《婺源砚图谱》载：“婺源砚在唐开元中，因猎人叶氏逐兽至长城里，见叠石如城垒状，莹洁可爱，因携之以归，刊粗成砚，温润大过端溪者。后数世，叶氏诸孙持以与令，令爱之，访得匠手琢为砚，由是天下始传。”<ref name="刘志斌"/>也有人认为歙砚起源于[[汉代|汉代]]，而龙尾山歙砚则起源于唐代<ref>{{cite journal |title=便携砚及歙砚探源 |author=汪云孙 |journal=收藏 |year=2009 |volume= |issue=11 |pages=100-103 |url=http://www.cqvip.com/qk/80240x/2009011/32195694.html }}</ref>。

到[[南唐|南唐]]元宗[[李璟_(南唐)|李璟]]时已由官方组织开采制作，并设立砚务官。[[李煜|李煜]]称赞“歙砚为天下冠”。<ref>{{cite journal |title=硕大无朋举世无双——屯溪区博物馆藏宋坑龙尾歙砚板 |author=钱胜东 |journal=徽州社会科学 |year=2010 |volume= |issue=4 |pages=76 |url=http://www.cqvip.com/qk/83034x/2010004/33574191.html }}</ref>

[[北宋|北宋]]从[[景祐|景祐]]年间开始进行了几次规模较大的砚石开采，歙砚的影响越来越大。欧阳修、苏轼、黄庭坚、米芾等人均有赋诗称赞。南宋以后石尽山颓，开采量逐渐减少，时断时续。清乾隆四十二年的官方采石是清代唯一一次有记载的开采。直到1963年歙县工艺厂（安徽歙砚厂）根据历史记载的地点重新开采。<ref>{{cite journal |title=歙砚简说 |author=杨震 |journal=徽州社会科学 |year=2007 |volume= |issue=9 |pages=28-30 |url=http://www.cqvip.com/qk/83034x/200709/25473508.html}}</ref>

== 矿物学 ==
对婺源产歙砚的研究结果表明，其主要矿物为绢[[云母|云母]]，还含有[[石英|石英]]、[[绿泥石|绿泥石]]以及少量的[[锆石|锆石]]、[[正长石|正长石]]、[[高岭石|高岭石]]和[[斑铜矿|斑铜矿]]等，样品的平均[[摩氏硬度|摩氏硬度]]为4.45，平均密度为2.7485g／cm<sup>3</sup>。<ref>{{cite journal |title=歙砚的宝石矿物学特征研究 |author=张莹, 陈涛 |journal=宝石和宝石学杂志 |year=2008 |volume=10 |issue=3 |pages=12-15 |url=http://www.cqvip.com/qk/90026a/2008003/28269493.html }}</ref>

== 特点 ==
歙砚的特点是质地坚润不吸水，发墨如油不损毫，“涩不留笔，滑不拒墨”，而且“多年宿墨，一濯而莹”。具有“坚、润、柔、健、细、腻、洁、美”八德。<ref name="刘志斌"/><ref>{{cite journal |title=歙砚之前世今生 |author=张颐 |journal=董事会 |year=2005 |volume= |issue=6 |pages=104-105 |url=http://www.cqvip.com/qk/87931x/2005006/15873831.html }}</ref>
