'''檀'''或'''檀木'''，在[[漢語|漢語]]中不盡然是一種木材，或一類木材的統稱。有時候指一種香木；有時候指質地堅韌、紋理細緻的一種硬木；有時候指兼具以上這兩種特性的木材；但有香味或質堅紋細的木材，又未必稱為檀木──譬如[[沈香|沈香]]、[[黃楊木|黃楊木]]、[[檜木|檜木]]…等等。與其說檀木是一個[[自然科學|自然科學]]名詞，不如視之為歷經沿革、含義多元的文化、經濟概念。

== 檀木的種類 ==

市面上被稱為檀木的木材，至少包括幾大類；根據[[中華民國教育部|中華民國教育部]][[教育部國語推行委員會|國語推行委員會]]編纂《[[重編國語辭典修訂本|重編國語辭典修訂本]]》[http://dict.revised.moe.edu.tw/]，檀，指三種植物的名稱：
*[[檀香|檀香]]（''Santalum album''）：[[檀香科|檀香科]][[檀香屬|檀香屬]]的一種半寄生性常綠小[[喬木|喬木]]，其木材富含芳香精油，可蒸餾提取檀香油，作為[[香料|香料]]或[[藥材|藥材]]使用，東印度產的檀香精油被世界公認品質最佳。
*[[青檀|青檀]]：[[榆科|榆科]]植物的''Pteroceltis tatarinowii''；本種一屬一種，主要分佈在[[中國|中國]]的[[黃河|黃河]]中、下游，是漢語原始意義的檀木。
*[[紫檀|紫檀]]：主要指[[豆科|豆科]]植物[[紫檀屬|紫檀屬]]Pterocarpus spp.的''Pterocarpus santalinus''檀香紫檀、又稱[[小葉紫檀|小葉紫檀]]，以及在家具製作上被歸類為花梨木類、商品名稱padauk的''P.indicus''等多種同屬植物。

除此，根據羅良才的《檀木名實錄》，中國的[[白檀|白檀]]''Dalbergia hupeana''、[[降香黃檀|降香黃檀]]''Dalbergia odorifera''以及世界各地出產的[[黃檀屬|黃檀屬]]Dalbergia spp.(商品名稱rosewood、俗稱酸枝)；中國的茜草科的[[烏檀|烏檀]]''Nauclea afficinalis''以及世界各地出產的[[烏檀屬|烏檀屬]]Nauclea spp.；[[新烏檀|新烏檀]]''Neonauclea griffithii''(又稱黃楊木)以及世界各地出產的[[新烏檀屬|新烏檀屬]]Neonauclea spp.也都被視為檀木。

另外，在市面上，以檀為名的還有：紅檀Swartizia spp.、綠檀Guaiacum spp.、又稱非洲柚木的黑檀Guibourtia spp.以及又稱烏木的柿屬黑檀Diospyros spp.等。據說市面上稱檀木的樹種多達60種以上。

== 檀的歷史沿革 ==

檀是相當古老的漢字詞彙，《[[詩經|詩經]]》魏風有《伐檀》三章：

{{quote|
坎坎伐檀兮，寘之河之幹兮，河水清且漣猗。不稼不穡，胡取禾三百廛兮。不狩不獵，胡瞻爾庭有縣貆兮。彼君子兮，不素餐兮。

坎坎伐輻兮，寘之河之側兮，河水清且直猗。不稼不穡，胡取禾三百億兮。不狩不獵，胡瞻爾庭有縣特兮。彼君子兮，不素食兮。

坎坎伐輪兮，寘之河之漘兮，河水清且淪猗。不稼不穡，胡取禾三百囷。不狩不獵，胡瞻爾庭有縣鶉兮。彼君子兮，不素飧兮。
}}

估計成書於东周或更早的《詩經》，一般認為收錄的是中國3000年左右以前的黃河中、下游的民歌，當時中國此一區域的檀樹，就只有[[青檀|青檀]]一種，其他分佈在南方的黃檀等，其所在區域可能尚非中國版圖，所以一般推斷此處用以製輻、製輪的檀，就是青檀。

檀就是青檀的概念，到了[[印度|印度]][[佛教|佛教]]傳入中國以後，有了重大改變。印度盛產的[[檀香|檀香]]，無論在[[婆羅門教|婆羅門教]]或佛教中都享有殊勝尊崇，加上佛教重要的教意之一「布施」（dana），早期被譯為[[檀那|檀那]]。於是檀這個詞彙，因應佛教在中國的傳播興盛，有了宗教上的殊榮。來自印度的的檀香，漸漸成為檀一詞的主要意義。在16世紀中國自然科學家[[李時珍|李時珍]]的鉅著《[[本草綱目|本草綱目]]》中，就採用此一概念，認為檀即檀香，同時也指[[紫檀|紫檀]]，但無關青檀。

可能是自海外貿易發達的[[明朝|明朝]][[永樂|永樂]]年間、[[鄭和|鄭和]]的大航海時期開始，檀又有了新的意義。明、清兩代的中國家具工藝發展至歷史最高水平，當時用以製作高級家具的木材，產自中國者首推[[黃花梨|黃花梨]]、即[[海南|海南]]島出產的降香黃檀，而來自印度及[[印支半島|印支半島]]的各種紫檀屬木材，尤其年久會散發香味的印度小葉紫檀，被視為比黃花梨更加珍貴。

現代的中國以及台灣，根據文獻及科研，大體形成檀木括青檀、檀香及紫檀的官方說法。但在市俗商品市場上，商人為了提高木材的價格，往往將來自非洲、中南美洲、澳洲等世界各地的各種木材，都以檀命名，所以估計檀之概念，將持續擴充。
