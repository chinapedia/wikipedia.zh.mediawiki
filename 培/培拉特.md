'''培拉特'''（{{lang-sq|Berati}}）位于[[阿尔巴尼亚|阿尔巴尼亚]]中南部，是[[培拉特州|培拉特州]]的首府，人口约60,031（2011年）。

== 名稱 ==
培拉特又名'''千窗之城'''，是由於培拉特市內依山而建的逾千間有多個窗戶的白色小屋而來的。

== 歷史 ==

== 地理 ==
培拉特位於阿爾巴尼亞中南部，城市被[[奧蘇姆河|奧蘇姆河]]劃分，市內四周皆為山脈。培拉特的[[冬|冬]]天寒[[冷|冷]]而[[夏|夏]]天十分炎[[熱|熱]][[乾燥|乾燥]]。

== 文化 ==

== 旅遊 ==
培拉特擁有很多具歷史文化古蹟，包括有依山而建的矮小白色小屋、匯聚了[[東歐|東歐]]和[[中東|中東]]色彩的[[古堡|古堡]]、[[教堂|教堂]]、[[清真寺|清真寺]]，過千年歷史的古城（至今還有人居住），還有[[奧蘇姆河|奧蘇姆河]]與其兩岸的農田鄉村的自然美景等。雖然每年到訪培拉特旅遊的外國旅客很少，皆因這個城市對於外國人來說十分冷門，甚至有不少人更從來不知道有這個城市或沒有聽過這城市的名稱。但培拉特作為一個被[[聯合國教科文組織|聯合國教科文組織]]評為具[[自然文化遺產|自然文化遺產]]的城市，所以培拉特不論在[[阿爾巴尼亞|阿爾巴尼亞]]或國外仍有一定的吸引力。<ref>{{cite web|url=https://ol.mingpao.com/php/cultureleisure3.php?nodeid=1525717007994&subcate=travel&issue=20180508|title=阿爾巴尼亞世遺 千窗下散步 回到鄂圖曼年代|work=明報OL|accessdate=2018年5月8日}}</ref>

== 交通 ==
培拉特有巴士來往首都[[地拉那市|地拉那市]]和[[薩蘭達|薩蘭達]]，另外培拉特有公路連接，但交通網絡仍較阿爾巴尼亞其他城市為差，且培拉特沒有建設任何鐵路系統。

培拉特市內也有一個小型[[民用機場|民用機場]]，主要服務地區為阿爾巴尼亞國內。

== 軼聞 ==
培拉特市四周皆為山脈，而其中一座位於培拉特西邊的俠長山脈更刻上了'''NEVER'''一詞，又名「霍查山」，文字由多塊石頭堆積在山腰而成，意思是永不，即代表討厭[[恩維爾·霍查|恩維爾·霍查]]意思，是由培拉特一個農民刻上的。在2010年前稱為'''ENVER'''，因為昔日該農民喜歡[[阿爾巴尼亞勞動黨|阿爾巴尼亞勞動黨]]領袖[[恩維爾·霍查|恩維爾·霍查]]而這樣做。
