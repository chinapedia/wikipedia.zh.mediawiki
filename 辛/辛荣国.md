'''辛荣国'''（{{bd|1950年|11月|||catIdx=X}}），[[山东|山东]][[安丘|安丘]]人，[[中国人民解放军少将|中国人民解放军少将]]。

== 生平 ==
1970年至1977年，历任中国人民解放军陆军第五十五军一六三师四八七团战士、班长、排长，四八九团司令部作训股参谋。1971年10月加入[[中国共产党|中国共产党]]。1977年至1985年，任陆军第五十五军司令部作训处参谋、副处长、处长。其间，1983年9月至1985年7月在[[中国人民解放军军事学院|中国人民解放军军事学院]]指挥专业学习<ref name=jianli/>。

1985年至1989年，任[[中国人民解放军陆军第四十二集团军|陆军第四十二集团军]]一六三师四八八团团长。1989年至1994年，任陆军第四十二集团军一六三师参谋长。1994年至1999年，任陆军第四十二集团军一六三师师长。其间，1998年3月至1999年1月在[[中国人民解放军国防大学|中国人民解放军国防大学]]联合战役指挥员培训班学习。1999年至2001年，任陆军第四十二集团军参谋长。2001年至2003年，任陆军第四十二集团军副军长。其间，1998年9月至2001年7月在[[南京陆军指挥学院|南京陆军指挥学院]]合同战术专业学习；2002年5月至7月在[[国防科技大学|国防科技大学]]高科技知识培训班学习<ref name=jianli/>。

2003年至2004年，任[[广西军区|广西军区]]副司令员。2004年至2005年，任[[中共广西壮族自治区党委|中共广西壮族自治区党委]]常委，广西军区司令员。2005年起，任[[中共广东省委|中共广东省委]]常委、[[广东省军区|广东省军区]]司令员。是第九届、第十届中共广东省委委员<ref name=jianli>{{Cite web |url =http://cpc.people.com.cn/GB/64162/123659/123801/7328033.html  |title = 辛荣国同志简历 |author =  |date =  |publisher = 中国共产党新闻网 |language =  |accessdate =2017-07-11 |quote =  }}</ref>。2008年当选[[第十一届全国人大代表|第十一届全国人大代表]]<ref>{{Cite web |url = http://www.npc.gov.cn/npc/xinwen/dbgz/wj/2008-02/29/content_1401314.htm |title = 中华人民共和国第十一届全国人民代表大会代表名单 |author =  |date =  2008-02-29 |publisher =中国人大网  |language =  |accessdate =  |quote =  }}</ref>。
