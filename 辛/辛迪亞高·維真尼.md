'''辛迪亞高·維真尼'''（{{lang-es|Santiago Vergini}}，{{bd|1988年|8月3日|}}），[[阿根廷|阿根廷]]職業足球員，司職[[中堅|中堅]]，現時效力[[阿根廷足球甲级联赛|阿甲]]的[[小保加體育會|小保加]]。

==球會生涯==

===早期===
維真尼在[[沙士菲|沙士菲]]展開其足球生涯，但未曾加入過一隊。2009年，他在[[巴拉圭|巴拉圭]]球會[[奧林比亞會|奧林比亞會]]首次踢職業賽，在全季他共上陣14次並入一球，2010年共上陣四次。2011年外借加盟意大利球會{{fc|維羅納}}，上陣16場並射入一球。2011年他返回阿根廷，簽約[[紐維爾舊生|紐維爾舊生]]，他在這裡穩站常規位置，共上陣78場並入兩球，其後在2013年加盟[[學生隊|學生隊]]，僅上陣13場並入一球。<ref>[http://www.transfermarkt.co.uk/santiago-vergini/profil/spieler/94783 Santiago Vergini - player profile - Transfermarkt]</ref>

===新特蘭===
2014年1月20日，維真尼外借加盟[[英超|英超]]球會{{fc|新特蘭}}，效力至2013/14球季季尾<ref>{{Cite web |title=Defender Vergini joins |url=http://www.safc.com/news/team-news/2014/january/defender-vergini-joins |work=Sunderland AFC |date=20 January 2014}}</ref>。

2月8日他首度亮相英超，對{{fc|侯城}}一戰中因中堅[[韋斯·布朗|韋斯·布朗]]領紅牌而在上半場便入替[[波連尼|波連尼]]<ref>{{cite news|title=Sunderland 0 - 2 Hull|url=http://www.bbc.co.uk/sport/0/football/25994883|newspaper=[[BBC_Sport|BBC Sport]]|date=8 February 2014}}</ref>。由於布朗停賽，維真尼在2月22日作客[[阿仙奴|阿仙奴]]首次擔任正選，但卻因犯錯而以1:4落敗<ref>{{cite news|last=Hytner|first=David|title=Olivier Giroud and Arsenal keep up fight at top by beating Sunderland|url=http://www.theguardian.com/football/2014/feb/22/arsenal-sunderland-premier-league-match-report|newspaper=[[The_Guardian|The Guardian]]|date=22 February 2014}}</ref>。3月2日，維真尼在聯賽盃決賽獲選入球隊的大軍名單，但未獲安排上陣，最終球隊以1:3不敵[[曼城|曼城]]，屈居亞軍<ref>{{cite news|last1=McNulty|first1=Phil|title=Man City 3-1 Sunderland|url=http://www.bbc.co.uk/sport/0/football/26311649|accessdate=12 September 2014|work=[[BBC_Sport|BBC Sport]]|date=2 March 2014}}</ref>。經過數場賽事後，領隊[[普耶|普耶]]將他移往右邊，以暫時緩解球隊的傷患問題。他其後表現漸趨穩定，並助球隊護級成功，更最終在第14名完成球季。

2014年8月7日，他再與新特蘭簽訂借用合約，為期一年。由於球員的共同擁有權問題，他未能正式轉會<ref>{{Cite web |title=Vergini returns |url=http://www.safc.com/news/team-news/2014/august/argentinian-returns |work=Sunderland AFC |date=7 August 2014}}</ref>。

2014年10月18日，在球隊以0:8慘敗予{{fc|修咸頓}}一戰，維真尼在12分鐘擺下一個被[[BBC|BBC]]形容為「英超史上最詭異的[[烏龍球|烏龍球]]」，在禁區頂試圖解圍時以窩利收死己方門將[[文路尼|-{zh-cn:曼洛內; zh-tw:曼諾尼; zh-hk:文路尼}-]]，為球隊的慘敗揭開序幕。領隊[[普耶|普耶]]直指：「即使你翻看該球20次，也不知道發生什麼事。」<ref>{{cite web |last1=Reddy |first1=Luke |url=http://www.bbc.co.uk/sport/0/football/29580147 |title=Southampton 8 – 0 Sunderland |work=BBC Sport |date=18 October 2014 |accessdate=18 October 2014}}</ref>

==國家隊生涯==
2012年9月20日，維真尼在[[南美超级德比盃|南美超级德比盃]]首次代表{{nft|阿根廷}}國家隊，對戰{{nft|巴西}}。這項賽事只准本土聯賽球員參加，他在該仗在73分鐘後備替[[利辛度·艾斯基爾·盧比斯|-{zh-cn:洛佩斯; zh-tw:洛佩斯; zh-hk:盧比斯}-]]，不過這場比賽球隊以1:2落敗<ref>{{cite web|title=Brazil 2-1 Argentina|url=http://www.goal.com/en-india/match/brazil-vs-argentina/1384360/live-commentary?ICID=MP_MS_3|publisher=[[Goal.com|Goal.com]]|accessdate=8 August 2014}}</ref>。2014年10月14日，他獲國家隊重召，在作客7:0大勝{{nft|香港}}隊的友賽中踢足全場<ref>{{cite news|title=Hong Kong 0-7 Argentina|url=http://www.bbc.co.uk/sport/0/football/29619276|accessdate=18 October 2014|work=BBC Sport|date=14 October 2014}}</ref>。
