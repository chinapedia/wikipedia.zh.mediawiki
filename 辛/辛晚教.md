'''辛晚教'''{{BD|1938年|11月25日|2017年|10月7日|}}，為[[台灣|台灣]][[都市計畫|都市計畫]]學者，為台灣規劃界與都市計畫界重要導師，長期致力於台灣都市計劃制度之擬議、國土規劃制度之開創、土地問題與政策建言、文化生活圈與文化環境規劃之學術研究與社會實踐。辛晚教亦為[[國立台北大學|國立台北大學]]都市計劃研究所創辦人之一，於該所任教及工作達42年（1968-2009），期間擔任該所所長9年（1975-1981、1994-1997），並參與[[臺北大學城|台北大學特定區]]與[[國立台北大學|國立台北大學]]三峽校區之整體規劃<ref name="multiple">[http://www.ntpu.edu.tw/up/?p=1444&lang=zh-tw 辛晚教名譽教授]，[[國立臺北大學|國立臺北大學]]都市計劃研究所</ref>。曾擔任[[內政部營建署|內政部營建署]]副署長、內政部區域計畫委員會委員、內政部都市計畫委員會委員、中華民國都市計劃學會理事長等，此外也長期擔任[[臺北市政府文化局|臺北市政府文化局]]古蹟審議委員、[[臺北市都市計畫委員會|台北市政府都市計畫委員會]]委員，透過文資保存、都市計劃、環境共容的合宜並進方式，致力提升台北城市文化內涵素養。<ref>{{cite news |language = 中文 | |url = http://www.bcc.com.tw/newsView.3077963 |title = 辛晚教教授辭世 文資會悼念感謝貢獻 |publisher = 中廣新聞網 |date = 2017/10/18 |accessdate = 2018/03/16}}</ref><br/>
育有三子女，其一為導演[[辛建宗|辛建宗]]。

==生平==
===教育生涯===
辛晚教出生於[[台灣|台灣]][[澎湖|澎湖]]，後隨父母搬遷至高雄。<br />
1957年至1961年就讀[[國立中興大學法商學院|國立中興大學法商學院]]地政學系，1962年土地行政高考考取榜首<br />
1966年進入[[國立中興大學法商學院|國立中興大學法商學院]]任職，協助周一夔教授創辦都市計劃研究所，1968年至2009年之四十二年間任教及任職於都市計劃研究所，期間擔任三屆都市計劃研究所所長<ref name="multiple" />。任教期間，除協助[[台北大學|台北大學]]三峽校區之規劃以外，也積極推動台灣國土規劃體系，倡議台灣在地化的文化規劃，成為台灣文化規劃的先驅者。<br />
2000年擔任「財團法人環境規劃與城鄉研究文教基金會」董事長，致力於規劃學界人才的培養。

===國土規劃奠基者===
鑒於台灣都市化問題日趨嚴重，且當時台灣也缺乏都市規劃的經驗，1966年政府洽請聯合國技術顧問團來台，同時成立[[行政院國際經濟合作發展委員會|行政院國際經濟合作發展委員會]]（經合會）都市建設及住宅計畫小組，並在聯合國顧問孟森夫婦的指導下，帶入新的規劃思潮。<ref>{{Cite web |url = http://www.ntpu.edu.tw/up/?page_id=42&lang=zh-tw |title = 都市計劃研究所發展歷程 |author = 國立台北大學都市計劃研究所 |accessdate = 2018 | }}</ref>當時辛任教於國立中興大學法商學院，也成為[[行政院國際經濟合作發展委員會|經合會]]都市計劃小組研究員。極力倡議台灣國土規劃體系，促成台灣國土規劃體系的完整建立。<ref> {{Cite book | author = 呂芳上等 | title = 都市計畫前輩人物訪問紀錄 | location = 台北市 | publisher = 中央研究院近代史研究所 | date = 2000 | ISBN = 957-671-726-4 }} </ref><br />
1980至1990年間，辛晚教積極參與推動「國土綜合開發計畫」與「國土綜合發展計畫法」（草案），為現在[[國土計畫法|國土計畫法]]之前身。<ref> {{Cite book | author = 內政部營建署 | title = 基石：營建署30年實錄 | location = 台北市 | publisher = 內政部營建署 | date = 2011 |ISBN = 9789860292398| url = https://www.cpami.gov.tw/kids/filesys/annals/18/18_5eba0c9821a0aa657da6e82fd9802ac4.pdf }} </ref><br />
1982年出版《都市與區域計劃》<ref> {{Cite book | author = 辛晚教 | title = 都市與區域計劃 | location = 台北市 | publisher = 中國地政研究所初版，詹式書局再版 | date = 1982，2011 |ISBN = 978-957-705-4180 }} </ref>一書，為都市計劃學界重要書籍之一。

===文化貢獻===
1990年，辛晚教倡議台灣在地化的文化規劃，影響目前台灣國土規劃對於文化元素的重視。<br />
1990年創立[[南管|南管]]樂團《咸和樂團》，醉心於戲曲創作，也對於傳統戲曲與[[歌仔戲|歌仔戲]]有相關文獻考究。<br />
1995至1996年間，主持[[文化建設委員會|文化建設委員會]]（今[[文化部|文化部]]）之《[[辛晚教#學術與研究著作|全國文化生活圈文化展演設施發展綱要計畫]]》<ref>{{Cite book |url = https://books.google.com.tw/books/about/%E5%85%A8%E5%9C%8B%E6%96%87%E5%8C%96%E7%94%9F%E6%B4%BB%E5%9C%88%E6%96%87%E5%8C%96%E7%A1%AC%E9%AB%94_%E5%B1%95%E6%BC%94.html?id=_xTIkQEACAAJ&redir_esc=y |title = 全國文化生活圈文化展演設施發展綱要計畫 |author = 辛晚教 |accessdate = 1996 | }}</ref><br />
1997至2006年間，長期擔任[[台北市孔廟|台北市孔廟]]管理委員會委員<ref>{{Cite web |url = https://tct.gov.taipei/News_Content.aspx?n=AE0CAF9A04EBDEFF&sms=1B3E8B0202ECA8C5&s=4C7FC7E2A94C7E57 |title = 臺北市孔廟管委會成員歷任委員介紹 |author = 臺北市孔廟管理委員會 |accessdate = 2017 | }}</ref>，並於1994至2012年間擔任祭孔大典分獻官<ref>{{Cite web |url = http://www.tpml.edu.tw/ct.asp?xItem=34116873&ctNode=82316&mp=10200M |title = 紀念孔子2562週年誕辰釋奠典禮．9月28日清晨登場 |author = 臺北市孔廟管理委員會 |accessdate = 2012 |5 =  |archive-url = https://web.archive.org/web/20180315200027/http://www.tpml.edu.tw/ct.asp?xItem=34116873&ctNode=82316&mp=10200M |archive-date = 2018-03-15 |dead-url = yes }}</ref>，並對孔廟文化與祭祀空間深入研究發表論文<ref>{{Cite book |title= 臺北孔廟禮制空間與大龍峒文化環境專題研究 |author= 辛晚教 | location = 台北市 | publisher = 臺北市都市更新處  
 |date = 2008 |}}</ref>
