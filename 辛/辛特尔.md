'''《辛特尔》'''（又名'''《寻龙记》'''，{{lang-en|'''Sintel'''}}）是一部由[[Blender基金會|Blender基金會]]创作的[[內容開放|開放版權]]、[[創作共用|創作共用]]的[[動畫電影|動畫電影]]，代号Durian。片長15分鐘，讲述了一个[[吉普赛|吉普赛]]女孩寻找自己的宠物飞龙的故事。<ref>{{cite web

 | url = http://www.computerworld.com.au/article/159545/open_source_big_screen_matt_ebb_tells_tales_elephants_dream_/?fp=2&fpid=4
 | publisher = Computerworld
 |date=January 15, 2008}}</ref><ref>{{cite web
 | last = Sanguinheira Diogo | first = Rui Paulo
 | url = http://www.linux-magazin.de/heft_abo/ausgaben/2007/12/modelling_2_50
 | title = Modelling 2.50

 |date=December 2007}}</ref> 该电影全部使用[[開放源碼|開放源碼]]軟件製作（如 [[Blender|Blender]]、[[Linux|Linux]]），製作技術、流程和素材徹底公開。<ref>[http://www.blender3d.org/e-shop/product_info_n.php?products_id=120 Durian Open Movie DVD]</ref>

这部电影由Blender基金会董事长汤·罗森达尔和[[皮克斯动画工作室|皮克斯动画工作室]]的艺术家科林·利维执导。“辛特尔”这个名字来源于荷兰语中的sintel，意思是“灰渣”或“余烬”<ref>{{cite web

== 情节 ==
在影片的开头，一位名叫辛特尔的女孩正在寻找一个被她叫做斯盖尔斯的小龙。随后影片插叙了辛特尔发现翅膀受伤的小龙斯盖尔斯并照顾它，从而逐渐建立了深厚的关系。然而，当斯盖尔斯的翅膀完全恢复，能够自由飞翔时，却被一只成年巨龙所抓走。辛特尔从此走上了寻找斯盖尔斯的坎坷道路，与野兽和武士们战斗不断。最终，她来到一个山洞，里面有一只巨龙——旁边还有一只十分熟悉的小龙。巨龙发现了辛特尔并向她发动攻击，却在要杀死她的一瞬间犹豫了。辛特尔趁着这个机会给了巨龙致命一击，然而却在其翼上发现了一道熟悉的伤痕，这才意识到这只临死的巨龙竟是已成年的斯盖尔斯。再看看自己，亦已白发苍苍，原来时光已在寻龙之路上飞逝。辛特尔悲伤欲绝地离开洞穴，却不知，斯盖尔斯的孩子——那只幼龙——正步履蹒跚地跟随着她。

== 技术信息 ==
继《[[大象之梦|大象之梦]]》，《[[大雄兔|大雄兔]]》和《[[鼯鼠弗兰基|鼯鼠弗兰基]]》之后，这部短片是由Blender基金会创建的第四个项目。《辛特尔》由Blender协会创建，这是Blender基金会的一个部门，专门制作开源的电影和游戏。<ref name=about>{{cite web|url=http://peach.blender.org/index.php/about|title=About|work=Peach Open Movie|publisher=Blender Foundation|accessdate=2008-02-04}}</ref><ref name=institute>{{cite web|url=http://www.blender.org/blender-institute|title=Blender Institute|publisher=[[Blender_Foundation|Blender Foundation]]|accessdate=2008-02-04|deadurl=yes|archiveurl=https://web.archive.org/web/20080510143352/http://www.blender.org/blender-institute/|archivedate=2008-05-10}}</ref>
这部影片由Blender基金会资助，资金主要来自Blender社区，电影DVD的预售和商业赞助。最终的影片成品及制作素材，包括动画数据，文字和纹理均在[[知识共享协议|知识共享协议]]（Creative Commons）署名许可下发布。<ref name=about/>任何人都可以自由下載，而[[Youtube|Youtube]]上也提供在線觀看。

=== Blender軟件優化 ===
与Blender以前的开源电影项目一样，Blender的开发团队不懈地按照电影制作团队的需求来改进软件。这些改进包括[[用户界面|用户界面]]，[[粒子系统|粒子系统]]，造型，着色，[[渲染管线|渲染管线]]，约束以及烟雾模拟。在Blender软件从2.54 beta版到2.50 alpha的更新中，这些改进被公开发布。<ref>{{cite web | title = Blender 2.56a Beta | publisher = Blender Foundation | url = http://www.blender.org/development/release-logs/blender-254-beta/ Blender Release Log for 2.54 beta | deadurl = yes | archiveurl = http://archive.wikiwix.com/cache/20110223125504/http://www.blender.org/development/release-logs/blender-254-beta/ | archivedate = 2011-02-23 }}</ref>

=== 渲染電腦集群 ===
渲染辛特尔電影所使用的[[計算機集群|計算機集群]]只是常見的[[x86-64|x86-64]][[伺服器|伺服器]]，運行[[Linux|Linux]]系統。<ref>[http://www.sintel.org/development/system-benchmarks/ System Benchmarks]</ref> <ref>[http://www.sintel.org/news/render-farm-tour/ Farm Tour]</ref>

=== 超高清版本 ===
2011年2月下旬，《辛特尔》的4096×2160像素（4K）版本于网络上发布。这个视频文件比普通视频大得多，接近160GB。在2月25日，16bits版本亦可以下载，但这个版本则高达650GB。导演汤·罗森达尔认为，之前推出的《[[大雄兔|大雄兔]]》已成为全球范围内视频设备的参照。现在电影行业向4K版本迈进，而[[电影保护主义|电影保护主义]]依然盛行。因此在4K方面的开源研究对知识共享协议的使用者来讲意义重大。<ref>{{cite web|author=刘萍|publisher=Creative common|title=开源电影“Sintel” 4k 分辨率版正式面市|url=http://creativecommons.net.cn/2011/03/01/1-16/|accessdate=2012-05-26}}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>
