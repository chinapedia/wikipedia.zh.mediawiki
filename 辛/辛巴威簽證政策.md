==电子签==

| width  = 300px

外国公民可以通过津巴布韦移民局的[[电子签证|电子签证]]系统在线申请所有类型的签证（旅游，商务，居留和学习）。签证费可在线或在抵达时支付。<ref>[https://www.evisa.gov.zw/ eVisa Zimbabwe]</ref> 获得电子签证平均需要两个工作日。然而，由于许多因素，等待时间可能会有所不同。<ref>[https://www.evisa.gov.zw/#/Help/FAQ]</ref>电子签证自签发之日起三个月内有效。<ref>[https://www.evisa.gov.zw/#/Help/FAQ]</ref>

电子签可用于以下其中一个入境口岸：

*[[贝特桥|贝特桥]]
*[[哈拉雷国际机场|哈拉雷国际机场]]
*奇龙杜
*李子树
*[[穆塔雷|穆塔雷]]
*Nyamapanda
*[[卡里巴|卡里巴]]
*维多利亚瀑布
*约书亚Mqabuko Nkomo国际机场

==通用护照==
{{ external media
| align  = right
| width  = 300px
| image1 = [https://www.google.com/maps/d/edit?mid=zn2lpfogc40k.k8nVbPrZG61c Map of entry points where universal KAZA visa can be obtained]
}}
[[File:KAZA_visa_eligible_countries.png|450px]]
津巴布韦和[[赞比亚|赞比亚]]于2014年11月28日推出了一项名为KAZA签证的通用签证。此签证可在抵达口岸时申请，可在赞比亚和津巴布韦（包括博茨瓦纳的[[乔贝国家公园|乔贝国家公园]]）停留最长30天<ref>[http://www.theindependent.co.zw/2014/11/21/sadc-uni-visa-takes-shape/]</ref>。第二阶段[[纳米比亚|纳米比亚]]、[[安哥拉|安哥拉]]和[[博茨瓦纳|博茨瓦纳]]有望加入此项目<ref>[http://en.starafrica.com/news/zimbabwe-zambia-to-launch-univisa-project-on-november-28.html Zimbabwe, Zambia to launch UNIVISA project on November 28]</ref><ref>[http://ewn.co.za/2014/11/09/Zimbabwe-introducing-a-joint-tourist-visa-with-Zambia Zimbabwe introducing joint tourist visa with Zambia]</ref>。在第三阶段，计划将有三个[[南部非洲发展共同体|南部非洲发展共同体]]试点国家加入，在第四阶段，所有南共体国家都将计划成为通用签证项目的一部分<ref>[http://kazavisa.info/index.php/about/overview]</ref>。

通用签证项目于2015年由于签证贴纸用完以及两国之间的谅解备忘录到期而暂停。新的备忘录于2016年12月签署，将符合条件的国家（地区）名单从40个扩展到65个<ref>[http://www.chronicle.co.zw/zim-zambia-revive-kaza-uni-visa/]</ref>。在津巴布韦，可在Kazungula，津巴布韦维多利亚瀑布，[[哈拉雷国际机场|哈拉雷国际机场]]和维多利亚瀑布机场入境口岸申请此签证。<ref>[http://www.mukwatravel.com/kaza-uni-visa/]</ref>

可申请此签证的国家：<ref>[https://twitter.com/WeAreWilderness/status/811506604575232000]</ref><ref>[http://www.africareview.com/business-finance/Zimbabwe-and-Zambia-launch-single-tourist-visa/979184-3495978-wqa9nqz/]</ref>

==过境免签==
[[孟加拉国|孟加拉国]]和[[巴基斯坦|巴基斯坦]]公民即使过境也需要签证。所有其他国家的公民可以在同一天内通过任何津巴布韦国际机场过境6小时。
