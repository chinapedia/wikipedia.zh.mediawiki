'''联明抗清'''又称'''明末农民军联明抗清'''，是[[南明|南明]]时期原农民起义军余部同[[南明|南明]]皇室共同抗清的事件{{参考|name=抗清1|{{cite journal|author=曾磊|title=《“联明抗清”说质疑》|journal=《焦作师范高等专科学校学报》，南开大学历史学院|date=第4期|year=2012年|accessdate=2014-12-11}}}}。

在农民军里最早提出联合明朝官军抗清的而是在明末。农民军的主力进入[[山西|山西]]，进逼北京时。他们派人到北京会见[[崇祯帝|崇祯帝]]，提出联兵抵抗清军的主张，但是[[明朝|明政府]]拒绝了。关于这一说法，正史如《[[明史|明史]]》、《明鉴》等均不载{{参考|{{cite journal|author=刘蕙孙|title=《明末农民起义军联明抗清的史实》|journal=《历史教学问题》|date=06期|year=1958年|accessdate=2014-12-11}}}}。而南明政权的[[隆武帝|隆武帝]]最先提出联合各地农民军抗清的主张，这一政策对整个南明的抗清决策影响深远{{参考|{{cite web|author=张嵚|title=明末乱局之中谁最早提出联合农民军抗击清军？|url=http://news.ifeng.com/history/minjianshuoshi/zhangqin/detail_2012_01/15/11986673_0.shtml|publisher=凤凰网|accessdate=2014-12-11|language=中文|date=2012-01-15|year=2012}}}}。

== 背景 ==
{{main|甲申国难|南明}}
[[入關戰爭|清军入关]]后，清廷在全国推行一系列民族征服、民族压迫政策{{注|民族压迫政策详见条目[[清初六大弊政|清初六大弊政]]。}}，阶级矛盾变为民族矛盾{{参考|顾城，《[[南明史_(顧誠)|南明史]]·第三章·弘光朝廷的偏安江淮·第一节·基本国策——“借虏平寇”》：“对于清初我国社会的主要矛盾，史学界有不同意见。有的人认为从清兵入关占领北京起，民族矛盾就成了主要矛盾；也有人认为由明至清阶级矛盾一直处于主要地位。这些看法很值得商榷。因为如果认为阶级矛盾始终是主要矛盾，那就不能正确解释二十年左右仁人志士的抗清运动，更不能公正评价大顺、大西农民军联明抗清的正义性。而认为清军入关就标志着民族矛盾已经成为主要矛盾，显然不符合事实。甲申五月，无论是满洲贵族建立的清廷，还是在南京继统的弘光朝廷，都把大顺农民军视为死敌。直到清兵南下，弘光朝廷覆亡，清廷推行一系列民族征服、民族压迫政策，民族矛盾才上升为主要矛盾。”[[中国青年出版社|中国青年出版社]]，2003年。}}{{参考|1980年8月出版《清史论丛》第二辑载拙文《论清初社会矛盾》。}}。民族矛盾上升为主要矛盾。南明[[弘光帝|弘光政权]]建立初期采取[[联虏平寇|联虏平寇]]的策略想借助清军来消灭中原地区的农民军，没想到清军却以代明“复仇”为名推行灭明之策，导致[[弘光帝|弘光政权]]被清军消灭。此时[[满清|清国]]一方面打击李自成、张献忠的农民军余部力量，另一方面消灭南明政权，而李自成的被杀和弘光政权的覆灭使得农民军和南明政权走向联合，一起来抵抗清军{{参考|{{cite journal|title=《清史论丛》第二辑载拙文|journal=《论清初社会矛盾》|accessdate=2014-12-11|date=1980年8月}}}}{{参考|{{cite web|author=顾城|title=弘光朝廷的偏安江淮|url=http://www.guoxue123.com/new/0001/nms/014.htm|work=南明史|publisher=国学网|accessdate=2014-12-11|language=中文}}}}{{参考|{{cite web|title=《南明史·第四章　大顺政权的覆亡》|url=http://www.guoxue123.com/new/0001/nms/022.htm|publisher=国学导航|accessdate=2014-04-24}}}}。

== 过程 ==
=== 大顺军联明抗清 ===
{{main|大顺军}}
顺治二年(1645年)[[李自成|李自成]]牺牲后，余部将领[[李过|李过]]、[[高一功|高一功]]等率领[[大顺军|大顺军]]联明抗清，主要在[[湖南|湖南]]、[[湖北|湖北]]、[[广西|广西]]、[[贵州|贵州]]等省活动。[[李过|李过]]、[[高一功|高一功]]继殉难后，[[李来亨|李来亨]]率部自[[贵州|贵州]]进入[[川|川]]、[[鄂|鄂]]边境，同先到达的[[郝摇旗|郝摇旗]]、[[刘体纯|刘体纯]]部会师，并且联合当地的其他反清武装势力，组建了[[夔东十三家|夔东十三家]]。[[李来亨|李来亨]]选定湖北茅麓山作反清基地，并屯耕山田自给，队伍很快扩大到几十万人，成为当时抗清的主要力量之一{{参考|{{cite web|title=湖北兴山县发现李来亨抗清遗址的城墙|url=http://www.chinanews.com/sh/news/2009/03-06/1591817.shtml|publisher=中国新闻网|accessdate=2014-12-11|language=中文|date=2009年3月|date=2009-03-06}}}}。

[[大顺军|大顺军]]坚持抗清斗争达二十多年。康熙四年八月，清政府派李守戎率数万大军进山围剿，起义军伤亡惨重，最终失败{{参考|name=抗清2|{{cite web|title=陕西发现南明永历兴朝钱 为明末农民军遗留|url=http://tech.qq.com/a/20110711/000243.htm|publisher=腾讯网|accessdate=2014-12-11}}}}。

=== 大西军联明抗清 ===
{{main|大西政权}}
顺治四年(1647年)，[[张献忠|张献忠]]去世后，部下[[李定国|李定国]]、[[孙可望|孙可望]]、[[刘文秀|刘文秀]]率领[[大西政权|大西政权]]余部由[[四川|四川]]经[[贵州|贵州]]转战入[[云南|云南]]，以昆明为中心建立农民政权，联合南明[[永历帝|永历帝]]抗击清军{{参考|徐鼒：《小腆纪年附考·卷17》：“歃血，誓扶明室无二心”。}}。后迎永历帝入滇，[[昆明|昆明]]一度称滇都，继续抗清，前后坚持了16年{{参考|{{cite web|title=西军联明抗清|url=http://www.huaxia.com/qcyn/mzwh/wshc/2009/04/2699685.html|publisher=滇台连线|accessdate=2014-12-11|date=2009-04-09}}}}。

但是由于[[孙可望|孙可望]]引起的内讧{{参考|刘彬：《晋王李定国列传》：“今虽大局稍有转机，而敌势力张，成败尚未遂睹。正吾侪同心协力，共策兴复之秋。不宜妄听谗言，自相残杀，以败坏国家，愿明公深长思之。”}}，再加上农民军没有联合东南沿海抗清的[[郑成功|郑成功]]，导致局面日益恶化，最后[[永历帝|永历帝]]被擒，李定国吐血而死，大西军联明抗清失败了。

== 结果 ==
但是后期因为南明政权内部不和逐步被清军分化瓦解，农民军和南明诸政权纷纷被清军消灭。

== 影响 ==
农民起义军联合南明政权后，奔赴抗清斗争的第一线，转战南北，收复失地，纵横数省，曾一度有望中兴明朝{{参考| name=抗清2}}。

== 评价 ==
有人认为农民军余部同南明朝廷的关系称作“联合”值得商榷。事实上，原农民军余部同南明朝廷的关系依然是传统皇权下的君臣关系，并不存在“联合”的关系{{参考| name=抗清1}}。
