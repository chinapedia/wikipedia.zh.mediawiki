'''联虏平寇'''又称'''借虏平寇'''，指的是[[南明|南明]]时期的[[弘光帝|弘光政权]]，采取联合[[满清|清军]]力量来消灭农民军的[[李自成|李自成]]和[[张献忠|张献忠]]的政策。

== 背景 ==
[[明朝|明]][[崇祯|崇祯]]十七年（1644年），[[李自成|李自成]]攻入北京，[[崇祯帝|崇祯帝]]上吊殉国，后[[吴三桂|吴三桂]]引清军入关，李自成在清军入京之前仓皇出逃。而清军入关后推行民族压迫政策，强迫汉人[[剃发易服|剃发易服]]，导致当时中国的主要矛盾由阶级矛盾上升为民族矛盾{{参|1980年8月出版《清史论丛》第二辑载拙文《论清初社会矛盾》。}}。

崇祯帝死后，他的堂兄[[朱由崧|朱由崧]]在[[史可法|史可法]]、[[马士英|马士英]]等人的支持下在南京建立[[弘光帝|弘光政权]]，国号依旧为明，史称[[南明|南明]]。此时，南明虽然仍占据[[江南|江南]]半壁江山，但是仍然想收复[[中原地区|中原地区]]，于是就有人提出「联虏平寇」的策略{{参|{{cite web|title=清军入关"借虏平寇":借南明军队剿李自成农民军|url=http://www.chinanews.com/cul/2014/01-23/5773585.shtml|publisher=中国新闻网|accessdate=2014-04-24|language=中文}}}}。

== 过程 ==
[[弘光帝|弘光政权]]推行「联虏平寇」的策略，使得南明君臣上下都沉浸在借用清兵力扫灭“流寇”的幻想中，但是也有 人提出了相反的观点，主张自强为主，抗击农民军和清兵{{注|南明吏科都给事中[[章正宸|章正宸]]上疏道：「今日江左形势视之晋、宋更为艰难，肩背腹心，三面受敌。」并要求朝廷既需「念先帝、先后殉社稷之烈」，又应「念三百年生养黔黎尽为被发左衽」，「断宜以进取为第一义。进取不锐，则守御必不坚」。对形势的分析他说道：「近传闯渠授首，未可轻信。贼计甚狡，必亡走入秦，度暑必尽锐而出，与献贼合，睥睨长江。……又闻虏踞宫阙，动摇山东。而当国大臣仓惶罔措，但绍述陋说，损威屈体，隳天下忠臣义士之气，臣窃羞之，臣切痛之。」，「失今不治，转弭秋高，虏必控弦南指，饮马长、淮；而贼又驰突荆襄，顺流东下。瓦解已成，噬脐何及？」{{参|冯梦龙《[[甲申纪事|甲申纪事]]》。}}}}{{注|[[章正宸|章正宸]]在同年六月，给事中马嘉植上言：「今日可忧者，乞师突厥，召兵契丹，自昔为患。及今不备，万一饮马长、淮，侈功邀赏，将来亦何辞于虏？」{{参|[[李清|李清]]《[[南渡录|南渡录]]·卷二》。}}}}。而[[弘光帝|弘光政权]]并不理会这些，同年七月上旬，[[弘光帝|弘光政权]]召集群臣讨论同清方联络事宜，并派遣使臣前往清营。

而大顺军在[[山海关之战|山海关战役]]失败后，开始西撤，当时他们在[[河北|河北]]、[[河南|河南]]、[[山东|山东]]等地建立的农民政权随机逐步瓦解。当地的百姓并开始以“复明”的旗号来抗击大顺军{{参|张怡《闻续笔》卷一。「过德州界，一路乡勇团结，以灭贼扶明为帜，所在皆然。至济南，回兵数千自相纠合，队伍整肃，器械精好。浚河置榷，凡舟必盘诘乃得过。即以所浚之土堆集两岸，仅容步，不可骑。而沿河民家塞向墐户，留一窦以通出入，防守颇严。引领南师，如望时雨。既闻弘光登极，史公督师，无不踊跃思郊。每遇南来客旅，辄讯督师阁部所至。使斯时乘其锐而用之，数十万义士因粮于众，人自为战，大功可立也。日复一日，坐失事机，灰忠义之心，隳朝食之气，谋之不臧，土崩瓦解，伊谁咎哉！」}}{{参|乾隆五十年《济宁直隶州志·卷三十一·艺文》：「参与济宁、兖州、济南反叛大顺、恢复明政权的郑与侨在《倡义记》中写道：“是役也，当四海无主之日，前无所依，后无所凭，只以绅衿忠愤、乡勇血诚，遂使大憝立剪，名义以新。无奈江南诸执政鼠斗穴中，虎逸柙外，置李贼不共戴天之仇于不问，可胜叹哉！」}}。而[[弘光帝|弘光政权]]只在山东地区接收一下当地的政权外，其他的地方均无官员前去接收。

而此时在北京的清军也是对于南明的「联虏平寇」策略没有定见，而有人甚至向多尔衮建议：在北京屠城，然后退守沈阳，固守山海关，可无后患{{参|[[吴晗|吴晗]]《朝鲜李朝实录中的中国史料·上编·卷五十八》：「宜乘此兵威，大肆屠戮，留置诸王以镇燕都，而大兵则或还守沈阳，或退保山海，可无后患」”。}}。而当时由于清兵在入关初期，兵力有限，补充兵员是一件非常难办的事。而明朝虽然现在四分五裂，但地域辽阔，实力也相当可观。[[多尔衮|多尔衮]]摸不清底细，不敢贸然行事，主张徐图[[中原|中原]]{{参|[[吴晗|吴晗]]《朝鲜李朝实录中的中国史料·上编·卷五十八》：「若得北京，当即徙都，以图进取」。}}。而占领北京也是由于[[吴三桂|吴三桂]]的原因，所以多尔衮初期开始决定勾结[[南明|南明]]，一同打击“流寇”，实现南北分治{{参|name=转变|{{cite web|title=第二节清廷对南明弘光政权态度的变化|url=http://www.guoxue123.com/new/0001/nms/015.htm|publisher=国学网|accessdate=2014-04-26|language=中文}}}}。

然后清廷也看到了[[弘光帝|弘光政权]]控制的江南广大富庶地区，北方土地贫瘠，都仰赖于南方漕运的粮食和其他物资。1644年九月，清河道总督[[杨方兴|杨方兴]]劝说多尔衮要不惜一切代价夺取[[江南|江南]]地区{{参|《国榷·卷一百三》：「不得江南，则漕运阻矣，将何以成天下？」}}。而此时清廷对[[南明|南明]]政权态度也发生转变，清廷开始以正统自居，否认弘光朝廷的合法地位，要求南明君臣无条件投降{{参|name=转变}}。

1644年七月二十八日，多尔衮派人给史可法带了一封信，信中还扬言“联闯平南”，而[[史可法|史可法]]在给多尔衮回信中只是为[[弘光帝|弘光朝廷]]继统的合法进行辩解，没有拿出一个对付清军的办法，反而增长多尔衮的骄狂气焰{{参|name=转变}}。

== 结果 ==
由于弘光君臣一厢情愿地沉迷于“联虏平寇”，自身毫无振作之意，而清廷以代明“复仇”为名推行灭明之策，联虏平寇的策略失败了{{参|{{cite web|title=第三章 弘光朝廷的偏安江淮|url=http://www.guoxue123.com/new/0001/nms/014.htm|publisher=国学导航|accessdate=2014-04-24}}}}。1645年四月清军攻克扬州，清军屠城，扬州当地居民除少部分在破城前逃出和个别在清军入城后隐蔽较深幸免于难者以外，几乎全部惨遭屠杀，“城中积尸如乱麻”{{参|归庄《先兄监纪君行状》，见《归庄集》卷八。}}，史称[[扬州十日|扬州十日]]。{{注|多铎在“谕南京等处文武官员人等”的令旨中说：「昨大兵至维扬，城内官员军民婴城固守。予痛惜民命，不忍加兵，先将祸福谆谆晓谕，迟延数日，官员终于抗命。然后攻城屠戮，妻子为俘。是岂予之本怀，盖不得已而行之。嗣后大兵到处，官员军民抗拒不降，维扬可鉴。」{{参|云巢野史编《两都怆见录》《南都》，见胡慕椿辑《乡国纪变》第一册。}}}}。

同年清军攻占[[南京|南京]]，[[弘光帝|弘光帝]]被俘，被清军押送到北方被杀，[[弘光帝|弘光政权]]灭亡{{参|{{cite web|title=第五节　弘先帝出逃和清军占领南京|url=http://www.guoxue123.com/new/0001/nms/027.htm|publisher=国学导航|accessdate=2014-04-26|language=中文}}}}。
