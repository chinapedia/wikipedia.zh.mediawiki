《'''一半，藍色。'''》（日語：{{lang|ja|半分、青い。}}）為[[NHK|NHK]]於2018年4月2日至2018年9月29日播出的第98部[[晨間小說連續劇|晨間小說連續劇]]，由[[永野芽郁|永野芽郁]]主演。

== 企劃與製作 == 
時間設定於[[日本萬國博覽會|大阪萬博]]隔年的1971年，出生於[[岐阜縣|岐阜縣]]東美濃市梟町一個小村莊食堂的女主角鈴愛，是個活潑有朝氣的小女孩。然而，她在小學的時候因為生病（[[流行性腮腺炎|流行性腮腺炎]]）而造成左耳失聰，卻沒因此而氣餒，在父母與青梅竹馬的鼓勵下繼續開朗的活著。在高中畢業後，擁有豐富想像力的她以成為漫畫家為目標前往[[東京|東京]]，在老師與同伴的切磋琢磨下努力奮鬥，而後結婚並誕下女兒。但是，在[[日本泡沫經濟|泡沫時代]]結束後，她與丈夫離婚，成為單親媽媽的她帶著女兒返鄉，此時的她仍不知道，一個巨大的轉機正等著她<ref name="topics">{{Cite news|url=http://www.nhk.or.jp/dramatopics-blog/1000/263632.html|title=2018（平成30）年度前期 連続テレビ小説「半分、青い。」制作のお知らせ}}</ref>。2017年2月22日公布製作相關事宜，徵選參予人數共2366名，6月20日正式發表由17歲的[[永野芽郁|永野芽郁]]擔任女主角<ref>{{Cite news|url=http://www.nhk.or.jp/dramatopics-blog/1990/273633.html|title=ヒロインは永野芽郁さん！2018年度前期 連続テレビ小説「半分、青い。」}}</ref>，8月22日公布其他演員人選<ref>{{Cite news|url=http://www.nhk.or.jp/dramatopics-blog/1990/277913.html|title=連続テレビ小説「半分、青い。」おもな出演者【岐阜・故郷編】発表！}}</ref>，10月23日於[[惠那市|惠那市]]岩村町正式開鏡<ref>{{Cite news|url= http://www.asahi.com/articles/ASKCB4JJ6KCBOIPE00S.html |title= 来春朝ドラ、岐阜で撮影中　永野さん「みなさん優しい」}}</ref>。本劇為永野芽郁的晨間劇處女作<ref>{{Cite news|url=http://www.sponichi.co.jp/entertainment/news/2017/06/20/kiji/20170620s00041000156000c.html |title= 永野芽郁　朝ドラオーディション初参加で２３６６人の中から栄冠 |newspaper=SPONICHI ANNEX|date=2017-06-20|accessdate=2017-06-20}}</ref>。2018年8月17日正式殺青<ref>{{Cite news|url=https://www.oricon.co.jp/news/2117828/full/ |title= 永野芽郁、朝ドラヒロイン完走『半分、青い。』撮了「またやりたい」 |newspaper=ORICON NEWS|date=2018-08-18|accessdate=2018-08-18}}</ref>。

== 劇情簡介 == 
時間是1971年的岐阜，楡野家產下了他們家第一個小女嬰，取名為'''鈴愛'''（[[永野芽郁|永野芽郁]] 飾）；而同一天同一間醫院，萩尾家的小男嬰也誕生了，取名為'''阿律'''（[[佐藤健|佐藤健]] 飾），這造就了這兩人一生的不解之緣。過了9年，小學三年級的鈴愛仍然與阿律是青梅竹馬，但鈴愛活潑好動，阿律冷靜聰明。然而過了不久，鈴愛的左耳因為腮腺炎而失去聽覺，但是她不以為意，反而覺得只有一邊聽的到下雨的聲音而覺得有趣，也因為她有如此樂觀積極的心態及豐富的想像力，她嚮往成為一名漫畫家，決定在高中畢業後隻身前往東京向偶像'''秋風羽織'''（[[豐川悅司|豐川悅司]] 飾）學習，紡織出她精彩的半生......
