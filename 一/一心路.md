'''一心路（Yisin Rd.）'''為[[高雄市|高雄市]]的東西向重要道路，本道路共分為二個部份。東起[[凱旋路_(高雄市)|凱旋路]]接[[瑞隆路|瑞隆路]]，西至[[中山路_(高雄市)|中山二路]]與[[三多路_(高雄市)|三多三路]]口，僅經[[前鎮區|前鎮]]一區。

== 行經行政區域 ==
* [[前鎮區|前鎮區]]

== 分段 ==
一心路共分為二個部份：
* 一心一路；東起於[[凱旋路_(高雄市)|凱旋三路]]口接[[瑞隆路|瑞隆路]]，西於[[民權路_(高雄市)|民權二路]]口接一心二路
* 一心二路：東於民權二路口接一心一路，西止於[[中山路_(高雄市)|中山二路]]與[[三多路_(高雄市)|三多三路]]口。

== 歷史 ==
{{main|高雄市主要道路列表|2014年高雄氣爆事故}}
語出自《[[書經|書經]]‧秦誓》記載：「予臣三千惟一心。」、《[[管子|管子]]‧法禁》：「武王有臣三千而一心」。取「萬眾一心」之意。[[周武王|周武王]]朝臣雖然多達數千名，但是都能團結一條心，所以國家必然昌盛。

2014年7月31日深夜因發生[[2014年高雄氣爆事故|高雄氣爆事故]]的影響，造成整條一心一路（從[[凱旋路_(高雄市)|凱旋路]]口至[[光華路_(高雄市)|光華路]]口）無法通行。歷經3個月重建的日夜趕工之後，於同年11月20日午夜12時完工同時開放通車<ref>[http://news.ltn.com.tw/index.php/news/life/breakingnews/1161793 沒有慶祝！高雄氣爆道路修復 凌晨開放通車]. [[自由時報|自由時報]]. [2014-11-21]. {{ja icon}}</ref>。
