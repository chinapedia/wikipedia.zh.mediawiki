== 分佈 ==
一枝黃花生於海拔565－2850米<ref name = "植物通"/><ref name = "中國植物誌"/>的山坡、闊葉林緣、林下、路旁及草叢之中<ref name = "香港野花"/>。本品種原產於中國[[華東|華東]]、中南及西南等地<ref name = "中醫學圖庫"/>。在中國大陆境內，分佈於[[浙江|浙江]]、[[江蘇|江蘇]]、[[安徽|安徽]]、[[江西|江西]]、[[貴州|貴州]]、[[湖南|湖南]]、[[湖北|湖北]]、[[廣東|廣東]]、[[廣西|廣西]]、[[雲南|雲南]]及[[陝西|陝西]]南部；此外[[台灣|台灣]]等地均有分佈<ref name = "植物通"/>，亦為[[香港|香港]]原生物種<ref name = "香港野花網"/><ref>[http://www.hkherbarium.net/Herbarium/c_search_end.asp?id=12520 Solidago decurrens Lour.]香港植物標本室</ref>。

== 形態特徵 ==
一枝黃花是一種多年生[[草本植物|草本植物]]<ref name = "香港野花網"/><ref name = "香港野花"/>，高約35－100厘米，偶有高達1米<ref name = "香港野花網"/>。根狀莖粗短直立，圓柱形<ref name = "中國醫藥網"/>，單生或少數簇生，不分枝或中部以上有分枝，基部帶淡紫色<ref name = "香港野花網"/>，光滑無毛<ref name = "中國醫藥網"/>。葉為長橢圓形、卵形或寬披針形，互生，先端急尖，基部楔形，中部以上邊緣有細齒或全緣，葉質較厚，葉兩面、沿[[葉脈|葉脈]]及[[葉緣|葉緣]]披短柔毛或下面無毛，長約2－5厘米，寬約1－1.5厘米。花為[[頭狀花序|頭狀花序]]，長約6－8毫米，寬約6－9毫米；通常長於枝莖上部排列成傘狀圓錐花序或總狀花序，亦有時排列成復頭狀花序；總苞片披針形或披狹針形，4－6層，中內層長約5－6毫米，頂端漸尖或急尖，邊緣膜質<ref name = "植物通"/>；外圍舌狀雌性花<ref name = "香港野花網"/><ref name = "植物通"/>，花舌橢圓形，黃色，約八朵<ref name = "中醫學圖庫"/>，長約6毫米；中央為管狀兩性花<ref name = "香港野花網"/><ref name = "植物通"/>，多數，[[雄蕊|雄蕊]]5枚，聚藥<ref name = "植物通"/>，[[花藥|花藥]]基部稍鈍<ref name = "植物通"/>先端有帽狀附屬物<ref name = "中醫學圖庫"/>；柱頭2裂<ref name = "植物通"/>；[[子房|子房]]下位<ref name = "植物通"/>。果為[[瘦果|瘦果]]，圓筒形，無毛，有時頂端披疏柔毛<ref name = "植物通"/>。<ref name = "中國植物誌"/>

== 生長 ==
一枝黃花喜生長於涼爽濕潤的氣候，耐寒，宜栽種於肥沃疏鬆富含腐殖質排水良好的砂質土壤中。可採用種子、分株等方法繁殖<ref name = "中國醫藥網"/>。

== 藥性及毒性 ==
一枝黃花全草入藥，性味辛、苦，微溫<ref name = "中國植物誌"/>，能袪風清熱、解毒清腫<ref name = "植物通"/>等，可治毒蛇咬傷<ref name = "香港野花網"/>、[[感冒|感冒]]、[[扁桃体炎|扁桃体炎]]、[[支氣管炎|支氣管炎]]、[[上呼吸道感染|上呼吸道感染]]、咽喉腫痛、[[肺炎|肺炎]]、[[肺結核|肺結核]]、[[白喉|白喉]]、急性及慢性[[腎炎|腎炎]]、小兒疳積及口腔糜爛等<ref name = "植物通"/><ref name = "中國醫藥網"/>，见于[[清朝|清朝]]医学家[[吴其濬|吴其濬]]的《[[植物名实图考|植物名实图考]]》。全株含[[皂甘|皂甘]]，家畜誤食可引致痲痺及運動障礙等中毒症狀<ref name = "植物通"/>。

== 相似品種 ==
一枝黄花和著名的入侵物种[[加拿大一枝黄花|加拿大一枝黄花]]（''Solidago canadensis''）为同科不同种植物。
