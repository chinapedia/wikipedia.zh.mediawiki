《'''一代宗師'''》（{{lang-en|''The Grandmaster''}}），於2013年上映的一部華語電影。所講的是[[詠春|詠春]]高手[[葉問|葉問]]及一眾[[武術家|武術家]]的故事。[[王家衛|王家衛]]導演，[[梁朝偉|梁朝偉]]、[[章子怡|章子怡]]、[[張震|張震]]、[[趙本山|趙本山]]領銜主演。電影2013年1月先後在中國内地、港澳地區和台灣上映。2013年8月在美國首映。<ref>{{cite web |url=http://www.epochtimes.com/b5/13/8/14/n3940399.htm |title=《一代宗師》23日全美上映 王家衛談功夫 |author= |publisher=大紀元 |date=2013-08-14 |language= |accessdate= }}</ref>

《一代宗師》參與了全球多個地方的電影節。<ref>{{cite web |url=http://www.imdb.com/title/tt1462900/releaseinfo |title=The Grandmaster (2013) Release Info |publisher=IMDb |date= |language=en |accessdate=2016-05-20 }}</ref>在[[第33屆香港電影金像獎|第33屆香港電影金像獎]]中囊括12個大獎，在[[第50屆金馬獎|第50屆金馬獎]]囊括5項大獎(入圍11項)，在兩大頒獎典禮中皆為獲獎數及入圍數最多的電影。在不少海外電影頒獎禮中，亦多次獲得提名及獎項。<ref>{{cite web |url=http://www.imdb.com/title/tt1462900/awards |title=The Grandmaster (2013) Awards |publisher=IMDb |date= |language=en |accessdate=2016-05-20 }}</ref>在[[第86屆奧斯卡金像獎|第86屆奧斯卡金像獎]]中，得到「最佳攝影」及「最佳服裝設計」兩項提名。<ref>{{cite web |url=http://www.oscars.org/oscars/ceremonies/2014 |title=The 86th Academy Awards (2014) Nominees and Winners |publisher=Academy of Motion Picture Arts and Sciences (AMPAS) |date= |language=en |accessdate=2016-05-20 }}</ref>

《一代宗師3D》在2015年1月上映<ref name="3D北">{{cite web |url=http://news.mtime.com/2015/01/05/1535972.html |title=《一代宗师3D》北京首映 |author=飞鸟凉 |publisher=Mtime时光网 |date=2015-01-05 |accessdate= }}</ref>，王家衛稱這是影片的「終極版」。<ref>{{cite web |url=http://ent.163.com/14/1231/01/AEOP159S00032KMI.html |title=《一代宗师3D》深圳首映 视角不同梁朝伟赞新鲜 |publisher=网易娱乐 |date=2014-12-31 |accessdate= }}</ref>

== 劇情概述 ==

[[中華武士會|中華武士會]]會長、八卦形意門掌門宫寶森，從北方來到[[廣東|廣東]][[佛山|佛山]]舉辦引退儀式。其首徒馬三在宴上打傷了佛山武人，以致當地武術界推舉[[詠春|詠春]]高手[[葉問|葉問]]出戰宫寶森。葉問得勝後，宫寶森女兒宮若梅旋即挑戰葉問，贏回一仗。

[[日本侵華|日本侵華]]戰事爆發。宮若梅前赴大學習醫途中，在火車上相助一線天擺脫日本人追捕。馬三投日成了[[汉奸|汉奸]]，與宫寶森反目並把其打死。宮若梅為復仇推掉婚事、入道、發誓終身不嫁不授徒，以決戰馬三，最终取勝也同時受到嚴重內傷。

戰後，不少習武之人移居到香港。葉問1950年抵埗，開始傳授詠春，後來更與在港行醫的宮若梅重逢。一線天也流亡到了香港，開設了理髮店，同時在港傳授八極拳。宮若梅依誓沒有收徒傳藝，葉問卻把詠春發揚光大。<ref>{{cite web |url=http://zhongyi.apdnews.com/a/224007.html |title=水也先生也谈《一代宗师》里的人物原型 |publisher=亚太日报 |date=2013-12-27 |accessdate= }}</ref>

== 3D版 ==

2015年1月上映的《一代宗师3D》片长111分钟，比原来的130分钟少10多分钟，它以在北美上映的版本为基础，与原作相比进行了大幅剧情调整，比原版的故事剧情更紧凑连贯，新增剧情包括：叶问大战一线天、宫二重现六十四手，叶问和一线天在香港比武的戏份，叶问与宫二书信传情，叶问穿西装拍照的旁白，赵本山变[[片尾彩蛋|片尾彩蛋]]谈认清自我；删减剧情有：赵本山、宋慧乔、张智霖、张震、金士杰等人的戏份消失或大调整，其中张震和赵本山被剪的比例最大，宫二复仇不再是主情节，小沈阳被张震摆平戏份消失，广州沦陷后叶问生活困窘的描写、宫宝森出殡路上遭遇拦截等。<ref>{{cite web |url=http://ent.163.com/15/0105/02/AF5NDCVG00034R73.html |title=《宗师3D》改了啥:赵本山戏变少徒弟露脸30秒 |publisher=网易娱乐 |date=2015-01 |accessdate= }}</ref>

=== 幕後籌備 ===
該電影之中，男女主角梁朝偉、章子怡、張震均為了電影親身學習相關武術，並且達到真正的專業水準。如張震所學[[八極拳|八極拳]]，則在專業的武術比賽中取得冠軍殊榮，因此該電影中，包括梁朝偉在內的演員並沒有採用替身。
