'''攀雀'''（[[学名|学名]]：{{lang|la|''Remiz pendulinus''}}）为[[攀雀科|攀雀科]][[攀雀属|攀雀属]]的[[鸟类|鸟类]]，俗名'''洋红儿'''。

== 特征 ==
体长约11厘米；顶冠为灰色，自额先向后具有黑色过眼纹，背部为棕色；尾部为凹形。

== 分布 ==
分布于欧亚大陆各地，包括[[西伯利亚|西伯利亚]]、[[蒙古|蒙古]]、[[朝鲜|朝鲜]]、[[日本|日本]]、[[巴基斯坦|巴基斯坦]]、[[印度|印度]]，包括[[中国大陆|中国大陆]]的[[黑龙江|黑龙江]]、[[吉林|吉林]]、[[辽宁|辽宁]]、[[宁夏|宁夏]]、[[新疆|新疆]]、华北、长江中下游以至云南等地，一般栖息于近水的苇丛和柳、桦、杨等阔叶树间。该物种的模式产地在欧洲[[波兰|波兰]],[[立陶宛|立陶宛]],[[匈牙利|匈牙利]],[[意大利|意大利]]。<ref name='id4005'/>

== 亚种 ==
* '''攀雀东北亚种'''（[[学名|学名]]：{{lang|la|''Remiz pendulinus consobrinus''}}）。分布于朝鲜、日本以及[[中国大陆|中国大陆]]的[[黑龙江|黑龙江]]、[[吉林|吉林]]、[[辽宁|辽宁]]、[[宁夏|宁夏]]、[[河北|河北]]、[[天津|天津]]、[[河南|河南]]、[[山东|山东]]、[[长江流域|长江流域]]自[[湖北|湖北]]至[[江苏|江苏]]、[[上海|上海]]、[[云南|云南]]等地。该物种的模式产地在湖北[[沙市|沙市]]。<ref name='id4006'>{{cite web| url =http://www.bioinfo.cn/db05/BjdwSpecies.php?action=view&id=4006| title =攀雀东北亚种| work =《中国动物物种编目数据库》| publisher =中国科学院微生物研究所| author =中国科学院动物研究所| accessdate =2009-04-04| deadurl =yes| archiveurl =https://web.archive.org/web/20160305004824/http://www.bioinfo.cn/db05/BjdwSpecies.php?action=view&id=4006| archivedate =2016-03-05}}</ref>
