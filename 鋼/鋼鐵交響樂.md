{{日本作品|鋼鐵交響樂|はがねオーケストラ}}是Hanbit Ubiquitous Entertainment公司推出的[[即時戰術遊戲|即時戰術遊戲]]（SLG），於2016年10月在[[iOS|iOS]]與[[Android|Android]]平台上發行。動畫於同月在[[TOKYO_MX|TOKYO MX]]電視台播出，共12話<ref name="TV-1">{{Cite web |date= 2010-09-08 |url=http://mtgamer.com/rpg%E6%96%B0%E4%BD%9C%E3%80%8C%E9%8B%BC%E9%90%B5%E4%BA%A4%E9%9F%BF%E6%A8%82%E3%80%8Dtv%E5%8B%95%E7%95%AB%E5%8C%96%E6%B1%BA%E5%AE%9A%EF%BC%8110%E6%9C%88%E5%90%8C%E6%AD%A5%E6%8E%A8%E5%87%BA/ |title=RPG新作「鋼鐵交響樂」TV動畫化決定！10月同步推出 |publisher=MTGamer |accessdate=2016-10-18 |archiveurl=https://web.archive.org/web/20161018110100/http://mtgamer.com/rpg%E6%96%B0%E4%BD%9C%E3%80%8C%E9%8B%BC%E9%90%B5%E4%BA%A4%E9%9F%BF%E6%A8%82%E3%80%8Dtv%E5%8B%95%E7%95%AB%E5%8C%96%E6%B1%BA%E5%AE%9A%EF%BC%8110%E6%9C%88%E5%90%8C%E6%AD%A5%E6%8E%A8%E5%87%BA/ |archivedate=2016-10-18 |deadurl=no}}</ref><ref>{{Cite web |date= 2010-09-09 |url=http://www.animen.com.tw/NewsArea/NewsItemDetail?NewsId=18270&categoryId=800&realCategoryId=1&subCategoryId=5 |title=手機遊戲《鋼鐵交響樂》宣布電視動畫化，首波詳細情報一併曝光！ |publisher=Animen動漫平台 |accessdate=2016-10-18 |archiveurl=https://web.archive.org/web/20161018105832/http://www.animen.com.tw/NewsArea/NewsItemDetail?NewsId=18270&categoryId=800&realCategoryId=1&subCategoryId=5 |archivedate=2016-10-18 |deadurl=no}}</ref>。遊戲玩法以指揮模式進行，玩家將指揮一個名為「{{nihongo|鋼鐵|ハガネ}}」的機械來收集物品與對付敵人，同時也可多人合作共同對付敵人<ref name="GamePlay-1">{{Cite web |date= 2010-08-26 |url=http://gnn.gamer.com.tw/0/136550.html |title=機械軍武系手機新作《鋼鐵交響樂》於日本舉辦「駕駛測試會」 |publisher=4Gamer.net、GNN新聞 |accessdate=2016-10-18 |archiveurl=https://web.archive.org/web/20160827200926/http://gnn.gamer.com.tw/0/136550.html |archivedate=2016-08-27 |deadurl=no}}</ref>。

== 玩法 ==

遊戲以玩家操作的「{{nihongo|鋼鐵|ハガネ}}」來進行物品收集及對付敵人等，玩法以平面指揮及第三人稱互切換來相互輔助「鋼鐵」的前進與操作，除此外也能多人合作共同對付敵人。而玩家操作的「鋼鐵」有不同的部位，可使用收集的物品作材料組合成不同類型的「鋼鐵」<ref name="GamePlay-1" />。

== 開發 ==

「鋼鐵」須要使用車長、炮手、操作員共3個角色才能操作，而製作團隊表示角色方面比鋼鐵更加注入心力<ref name="GamePlay-1" />。

=== 合作 ===

聲音合成軟件[[VOICEROID|VOICEROID]]內的角色弦卷真紀與[[结月缘|结月缘]]會在遊戲內登場並有不同形式的合作活動<ref name="GamePlay-1" />。

== 動畫 ==

Hanbit Ubiquitous Entertainment公司於2016年9月8日公佈將會動畫化並於2016年10月9日在[[TOKYO_MX|TOKYO MX]]電視台播出，共播出12集<ref name="TV-1" /><ref>{{Cite web |date= 2010-09-08 |url=http://www.4gamer.net/games/307/G030794/20160908019/ |script-title=ja:「はがねオーケストラ」のTVアニメがTOKYO MXで10月9日に放送スタート。アニメと同時期にアプリ版の配信開始も予定 |publisher=4Gamer.net |accessdate=2016-10-20 |archiveurl=https://web.archive.org/web/20160909143908/http://www.4gamer.net/games/307/G030794/20160908019/ |archivedate=2016-09-09 |deadurl=no |language=ja}}</ref>。

=== 製作 ===

導演由[[太田雅彥|太田雅彥]]擔任，編劇為[[青島崇|青島崇]]，人物原案{{link-ja|今野隼史|今野隼史}}繪畫，作畫指導與人物設定為[[大隈孝晴|大隈孝晴]]，音樂由[[三澤康廣|三澤康廣]]負責，動畫製作為{{link-ja|Fanworks|ファンワークス}}<ref>{{Cite web |url=http://www.hagane-orchestra.jp/staff.html |script-title=ja:TVアニメ「はがねオーケストラ」公式サイト |accessdate=2016-10-20 |language=ja |title={title} |archive-url=https://web.archive.org/web/20160908112537/http://hagane-orchestra.jp/staff.html |archive-date=2016-09-08 |dead-url=yes }}</ref>。

=== 播放 ===

2016年10月10日於TOKYO MX電視台首播，同日於[[NicoNico|NicoNico]]網站播出。
