'''黎巴嫩阿拉伯語'''（اللهجة اللبنانية）（也叫「'''黎巴嫩語'''」）是[[黎凡特阿拉伯語|黎凡特阿拉伯語]]（اللهجات الشامية 或 لهجات سورية الكبرى）的一種，是[[黎巴嫩|黎巴嫩]]的主要口頭語言。黎巴嫩阿拉伯語深受其他[[中東|中東]]及[[歐洲|歐洲]]語言的影響，加之黎巴嫩的多元文化和多種語言環境，黎巴嫩阿拉伯語在一定程度上不同於[[阿拉伯語變體|其他地區的阿拉伯語]]。黎巴嫩阿拉伯語屬於[[北黎凡特阿拉伯語|北黎凡特阿拉伯語]]的中北黎凡特阿拉伯語。[[民族語|民族語]]網站為北黎凡特阿拉伯語設定的代號是<code>[http://www.sil.org/iso639-3/documentation.asp?id=apc apc]</code>。

== 語言還是方言 ==
與其他地方的阿拉伯語一樣，人們對於黎巴嫩阿拉伯語是一種獨立的[[語言|語言]]還是[[阿拉伯語|阿拉伯語]]的一種[[方言|方言]]也存在爭議。從語言學的角度來說，黎巴嫩阿拉伯語源於[[亚拉姆语|亞拉姆語]]而非阿拉伯語，而且與[[现代标准阿拉伯语|書面阿拉伯語]]及其他非黎凡特地區的阿拉伯語口語有很大不同。<ref>http://www.lebaneselanguage.org/language/lebanese-vs-arabic/ </ref>然而從政治及民族角度看，黎巴嫩是[[阿拉伯國家聯盟|阿拉伯國家聯盟]]的成員之一，且大部分[[黎巴嫩人|黎巴嫩人]]也認為自己是[[阿拉伯民族|阿拉伯民族]]的一部分，此時黎巴嫩阿拉伯語被認為是阿拉伯語的一種方言。

== 與書面阿拉伯語的不同 ==
黎巴嫩阿拉伯語與當代其他地區的阿拉伯語有許多共同特徵，與書面阿拉伯語也有著不同的音節結構。書面阿拉伯語的音節開頭必須是一個輔音，其後緊接著一個元音；而黎巴嫩阿拉伯語的音節開頭卻經常有兩個輔音。
* 句法：更為簡單，無語氣與格位標記。
* 數：動詞在任何時候都須與主語性數一致，無論主語是否已提及。
* 詞彙：有大量借詞，大約47%來自[[阿拉伯語|阿拉伯語]]，40%來自[[敘利亞語|敘利亞語]]，5%來自[[奥斯曼土耳其语|奥托曼土耳其語]]，5%來自[[希臘語|希臘語]]，3%來自[[法語|法語]]和[[英語|英語]]。
=== 例子 ===
以下是書面阿拉伯語與黎巴嫩阿拉伯語區別的例子。咖啡（قهوة），書面阿拉伯語：/ˈqahwa/，黎巴嫩阿拉伯語：[ˈʔahwe]。[[清小舌塞音|[q]]]對應[[聲門塞音|[ʔ]]]，末尾[[Taw|ة]]前的/a/變為[[半閉前不圓唇元音|[e]]]。
== 黎巴嫩阿拉伯語的方言 ==
儘管大多數黎巴嫩人能都理解通用的黎巴嫩阿拉伯語，黎巴嫩的不同地區也有自己獨特的方言，不論是在語音、語法還是詞彙方面。<br />
較為通用的方言包括：
* [[貝魯特|貝魯特]]方言
* [[北部省_(黎巴嫩)|北部]]方言
* [[南部|南部]]方言
* [[贝卡谷地|貝卡]]方言
* [[黎巴嫩山|黎巴嫩山區]]方言
== 其他语言的影响 ==
黎巴嫩在{{tsl|en|Arabization|阿拉伯化}}之前使用的是亚拉姆语。阿拉伯化后，与亚拉姆语相近的阿拉伯语变得流行起来，但也保留有很多亚拉姆语的词汇和语法。亚拉姆语在当代黎巴嫩口语中仍有着很大的影响。这导致了黎巴嫩阿拉伯语与标准阿拉伯语在发音、单词、语法上都有不同。<ref>أنيس فريحة، اللهجات وأسلوب دراستها، دار الجيل، 1989، ص130</ref>

== 書寫系統 ==
黎巴嫩阿拉伯語通常不用於書面，除了某些小說中的對話及用黎巴嫩阿拉伯語寫的詩歌中。黎巴嫩阿拉伯語也廣泛運用於黎巴嫩歌曲、戲劇、電視、廣播，特別是{{tsl|en|Zajal|زجل}}（一種歌唱形式）。<br />
黎巴嫩阿拉伯語通過{{tsl|en|Music_of_Lebanon|黎巴嫩歌曲}}流行於阿拉伯世界。黎巴嫩的正式出版物，例如報紙，通常用書面阿拉伯語、法語或英語發行。<br />
黎巴嫩阿拉伯語通常用[[阿拉伯字母|阿拉伯字母]]書寫。然而在非正式場合，如網上聊天室，也會用[[阿拉伯语聊天字母|拉丁字母轉寫]]。黎巴嫩詩人[[赛义德·阿克勒|赛义德·阿克勒]]曾推動使用拉丁字母，但未獲得廣泛認可。然而現在，許多阿拉伯網民在缺少阿拉伯語字母鍵盤時，會使用類似於سعيد عقل的拉丁轉寫，唯一的區別是網民們會用數字代替拉丁字母無法代表的阿拉伯語輔音。<br />
用拉丁字母轉寫黎巴嫩阿拉伯語仍無統一標準。2012年，Antoine Rizk分析了黎巴嫩人在聊天室、社交網絡、手機短訊、電子郵件中使用拉丁轉寫的情況，並將其總結在一個網頁上。<ref>http://www.facebook.com/LebaneseWritingRules</ref>這會幫助黎巴嫩阿拉伯語的學習者在網上學習。<ref>http://www.abjadiye.com</ref><br />
2010年，黎巴嫩語言學會（Lebanese Language Institute）發佈了黎巴嫩語鍵盤配置。這使得人們更易用與[[Unicode|Unicode]]相容的拉丁字母書寫黎巴嫩語，且可用特殊符號替代缺失音節。<ref>'''Lebanese Language Institute''': [http://www.lebaneselanguage.org/lll-system/lebanese-latin-letters/ Lebanese Latin Letters The Lebanese Latin Letters]</ref>
