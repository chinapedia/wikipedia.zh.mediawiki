'''黎元洪旧居'''位于中国[[北京市|北京市]][[东城区_(北京市)|东城区]][[王府井大街|王府井大街]]27号，是[[中华民国大总统|中华民国大总统]][[黎元洪|黎元洪]]在北京的旧居，曾经作为[[中华民国大总统府|中华民国大总统府]]，也是北京城内知名的私家园林之一<ref name=cy>[http://www.bjdclib.com/subdb/laneculture/yard/200906/t20090621_2223.html 王府井大街27号，东华流韵，于2013-04-20查阅]</ref>。

==历史==
该宅院南到[[东厂胡同|东厂胡同]]，北到[[翠花胡同|翠花胡同]]，东到[[王府井大街|王府井大街]]。此地[[明朝|明朝]]时为[[宦官|宦官]]控制的的特务机构“[[东厂|东厂]]”，其南现为[[东厂胡同|东厂胡同]]<ref name=cy/>。[[清朝|清朝]]时，该院落曾作为[[明史馆|明史馆]]。该院落是清朝[[镶白旗|镶白旗]]驻地<ref name=cai>[http://news.guoxue.com/article.php?articleid=32040 蔡美彪：四平调的“八零后”，国学资讯，2012-09-20]</ref>。[[清朝|清朝]][[咸丰|咸丰]]、[[同治|同治]]年间，[[两广总督|两广总督]]、大学士[[瑞麟|瑞麟]]在此兴建房屋及园林，命名为“馀园”。据记载，当时园内“河流甚长，草木尤佳”，园中的假山上有两块立石，分别刻有“崖半亭高”和“岭崎磊落”<ref name=cy/>。靠东围墙还依地势开凿了一条月牙河。继瑞麟之后，[[荣禄|荣禄]]迁入该宅院。荣禄在北京的住宅除了今[[东城区_(北京市)|东城区]]交道口[[菊儿胡同|菊儿胡同]]3号、5号和[[寿比胡同|寿比胡同]]6号外，此处是荣禄的另一旧居。据说荣禄迁居此处后，对宅院进行了改造，且安装了电灯，当时除了皇宫之外，这座宅院可说是全北京城内第一个装上电灯的住宅<ref name=qlw>[http://beijingww.qianlong.com/1470/2009/07/13/229@96412.htm 文化词解:王府井，千龙网，2009-07-13]</ref>。
[[File:SheKeBoYuanHotel.jpg|thumb]]东侧的原黎元洪旧居东墙及东大门]]
继荣禄之后，[[袁世凯|袁世凯]]为了拉拢[[黎元洪|黎元洪]]，乃以十万银元买下了荣禄宅的东半部送给黎元洪<ref name=qlw/>。该宅遂于1912年之后成为[[中华民国副总统|中华民国副总统]][[黎元洪|黎元洪]]的住宅。1916年[[袁世凯|袁世凯]]逝世后，黎元洪继任[[中华民国大总统|中华民国大总统]]，仍居住于此宅，当时还圈定了[[中华民国大总统府|中华民国大总统府]]界，建造了西式的围墙，并且在[[王府井大街|王府井大街]]上开门，如今北侧院墙的根基处尚存一块刻石，刻有“黎大德堂界址”<ref name=cy/>。[[张勋复辟|张勋复辟]]时，黎元洪被逐出北京。1922年，黎元洪再度出任大总统时，仍在该宅居住<ref name=qlw/>。实际上，当时该宅的正门设在[[东厂胡同|东厂胡同]]北侧，门牌号为“东厂胡同壹号”及“东厂胡同叁号”<ref>[http://book.yzdsb.com.cn/system/2012/05/28/011728537.shtml 东厂胡同有遗德——对黎元洪故宅的寻访（1），燕赵都市网，2012-05-28] {{webarchive|url=https://web.archive.org/web/20160304061722/http://book.yzdsb.com.cn/system/2012/05/28/011728537.shtml |date=2016-03-04 }}</ref>。

1928年黎元洪逝世后，此宅售予由[[日本|日本]]退还[[庚子赔款|庚子赔款]]而成立的“[[东方文化事业总委员会|东方文化事业总委员会]]”，且在院内兴建了一座日本式的三层图书馆<ref name=cy/>。[[抗日战争|抗日战争]]胜利后，该宅成为[[傅斯年|傅斯年]]担任所长的[[中央研究院历史语言研究所|中央研究院历史语言研究所]]的所在地，[[国立北京大学|国立北京大学]]校长[[胡适|胡适]]也曾在此居住。“东方文化事业总委员会”和[[胡适|胡适]]的大批藏书都留在该宅<ref name=cai/>。

1950年代后，此宅为[[中国科学院图书馆|中国科学院图书馆]]和[[中国科学院考古研究所|中国科学院考古研究所]]使用。[[范文澜|范文澜]]、[[汤用彤|汤用彤]]、[[梁思永|梁思永]]、[[张政烺|张政烺]]、[[邓广铭|邓广铭]]、[[季羡林|季羡林]]等人先后在此居住。1970年代末至1980年代初，黎元洪宅院西部被拆除，原址兴建了中国社会科学院近代史研究所和世界历史研究所大楼。1980年代，又兴建了六七栋研究所职工宿舍楼。小桥流水及戏楼亭阁等建筑大部分被毁。2000年代初，坐落在该宅院旧址东部的中国科学院图书馆被拆除，兴建了[[社科博源宾馆|社科博源宾馆]]和中国社会科学院考古研究所大楼。如今，此宅的残存部分为[[中国社会科学院考古研究所|中国社会科学院考古研究所]]使用<ref name=cy/><ref name=cai/><ref name=bei>{{Cite web |url =http://news.xinhuanet.com/mrdx/2016-01/08/c_134989479.htm  |title = 被“遮蔽”的黎元洪故居 |author =  |date =2016-01-08  |publisher =  新华网|language =  |accessdate =  |quote =  }}</ref>。

对此宅的保护一直非常薄弱，此宅长期未被列为文物加以保护<ref>{{Cite web |url = http://zqb.cyol.com/html/2013-01/28/nw.D110000zgqnb_20130128_6-03.htm |title = 袁世凯、黎元洪的故居要不要保护？ |author =  |date =  |publisher = 中国青年报2013年01月28日，第03版 |language =  |accessdate =  |quote =  }}</ref>。2012年，此宅以“黎元洪旧居”之名被列为[[北京市东城区普查登记文物|北京市东城区普查登记文物]]，时代为民国<ref>东城“文物古迹”监管大类小类细类名称及立结案标准，网格东城2012年11月10日总第35期文物古迹监管专刊</ref>。

==建筑==
如今，此宅的原有建筑所剩很少。21世纪初，普查登记文物黎元洪旧居内图书楼被拆毁<ref>黄钟，革故鼎新:文物保护区划新策略刍议，中国城市规划年会2014年</ref>。至21世纪初，大门的南北两侧分别留有一座小式硬山筒瓦建筑。院内靠南侧的假山上有一勾连搭式台轩，面阔三间，进深三间，其北残存一小段爬山廊。假山西北处是一座七檩带前廊硬山合瓦式建筑。再向南为一座七檩带前后廊硬山筒瓦建筑，面阔三间，东侧带有一间五檩小式硬山合瓦耳房<ref name=cy/>。后来这些残存建筑进一步被拆除。到2016年初，仅存很少的几处建筑。

*社科博苑宾馆西北侧残存建筑：社科博苑宾馆北侧开有一门，挂有中国社会科学院考古研究所的牌子，在拐子型楼的连接层中间夹有一座中式小房屋，房屋门脸上挂有中国社会科学院考古研究所文化遗产保护研究中心的牌子。与大门对望的是考古学家[[夏鼐|夏鼐]]的雕塑<ref name=bei/>。
*西边残存建筑：从考古研究所大门向西再过一个院门，有一栋南北向的九层大楼，再向南拐，可见大楼之间夹有两片中式老房屋。其中一个中式小平房位于社科博苑宾馆和考古研究所凹形大楼的中间，有铁栅栏围绕。另一栋是坐北朝南约有七八间的中式小平房，位于中国社会科学院近代史研究所和世界历史研究所的办公楼前<ref name=bei/>。

此宅作为大总统府时，兴建了高大精美的围墙，成为如今王府井大街北段的重要街景。在社科博源宾馆的兴建过程中，拆除了社科博源宾馆东侧的大部分围墙，仅留下正对宾馆大门的一小段作为影壁墙。2015年，正在施工的嘉德艺术中心工地拆除了工地东侧黎元洪旧居的大门和大部分围墙，在将要拆除最北段（含刻有“黎大德堂界址”的石碑）时被叫停。2016年1月，[[九三学社|九三学社]]北京市委在北京市[[两会|两会]]时提出并关注了该事件<ref>{{Cite web |url = http://news.xinhuanet.com/legal/2016-01/20/c_128646494.htm |title = 九三学社：建议东西城合并为北京中央区 |author =  |date =  2016-01-20 |publisher =新华网 |language =  |accessdate =  |quote =  }}</ref>。
