'''黎智'''（{{bd|1922年|4月22日|2001年|8月18日}}），[[湖北|湖北]][[浠水县|浠水县]]人，原名'''闻立志'''，1941年赴[[延安|延安]]时，取“立志”谐音改名为黎智<ref name=cctv2007>{{cite web |title=闻一多骨灰保存和安葬：血溅西仓坡魂归八宝山 |url=http://discovery.cctv.com/20071109/108975.shtml |publisher=[[CCTV|CCTV]].com |date=2007-11-09 |accessdate=2012-07-08}}</ref><ref name=bjds2004>{{cite journal |author=闻立树 |title=黎智的生活时代与片断经历 |journal=北京党史 |year=2004 |issue=05|pages=33-37 |url=http://www.cnki.com.cn/Article/CJFD2004-BJDY200405010.htm}}，[http://www.hprc.org.cn/pdf/BJDY200405010.pdf 拷贝1]</ref>。中华人民共和国官员。

==生平==
黎智出身书香门第——溪水闻氏。闻氏居于浠水县[[巴河镇_(浠水县)|巴河镇]]的[[望天湖|望天湖]]畔，乃当地大族。闻氏先祖是[[文天祥|文天祥]]的旁系家族成员，[[南宋|南宋]]灭亡后，逃亡至浠水，改姓为闻<ref name=jlg>{{cite web |author=闻立雕 |title=我的父亲闻一多（一） |url=http://www.wenyiduo.net/jinian/huiyi-huainian/2010/0429/134.html |publisher=闻一多纪念馆 |accessdate=2012-07-12}}</ref>。黎智的父亲闻家骢与著名学者[[闻一多|闻一多]]和[[闻家驷|闻家驷]]是同胞兄弟<ref>{{cite web |title=《黎智纪念集》 |url=http://www.whds.org.cn/content.jsp?id=2809 |publisher=武汉党史网 |date=2005-10-12 |accessdate=2012-07-08 |archive-url=https://web.archive.org/web/20160919131237/http://www.whds.org.cn/content.jsp?id=2809 |archive-date=2016-09-19 |dead-url=yes }}</ref>。

1938年，16岁的黎智在[[湖北省|鄂]]西[[巴东|巴东]][[野三关镇|野三关]]加入[[中国共产党|中国共产党]]。1940年高中毕业后先后担任[[来凤县|来凤县]]委宣传部长、[[利川县|利川县]]委组织部长。[[皖南事变|皖南事变]]后，代理利川县委书记，后因躲避国民党抓捕，经由[[重庆|重庆]]转往延安学习。1945年回重庆[[中共中央南方局|中共中央南方局]]工作，任青年组秘书。1948年，[[平津战役|平津战役]]前后，黎智负责领导[[北平|平]][[天津市|津]]两地学生运动，策反不少[[国民党|国民党]]高级官员的亲属和朋友，对中共和平接收[[北平|北平]]和顺利攻占[[天津|天津]]贡献很大<ref name=cjrb2002/>。他曾动员[[傅作义|傅作义]]的女儿[[傅冬|傅冬]]（原名傅冬菊）、老师[[刘厚同|刘厚同]]，以及傅作义部的[[少将|少将]]联络处长[[李腾九|李腾九]]等人劝说傅作义接受和平改编。[[天津战役|天津战役]]时，领导收集重要情报，使得共军能在29小时内攻占天津，歼敌13万，且对天津破坏很小。他还领导反对国民党当局南迁工科院校、工厂设备、以及工程技术人员，使得战后天津重要人力和技术资源得以较完整保留。
 
1949年，中共夺取武汉后，黎智一直在武汉工作，历任[[中共中央中南局|中南局]]青委委员、武汉青委副书记、团市委书记、中南局团工委副书记、武汉市委宣传部副部长、市委秘书长、市委常委、[[硚口区|硚口区]]委第一书记、武汉市委候补书记兼工业部部长、市经委主任<ref name=cjrb2001>{{cite web|title=黎智同志遗体告别仪式昨举行|url=http://www.cnhan.com/GB/content/2001-08/23/content_102835.htm|publisher=汉网（转载自《长江日报》）|date=2001-08-23|accessdate=2012-07-10|deadurl=yes|archiveurl=https://web.archive.org/web/20050817175054/http://www.cnhan.com/gb/content/2001-08/23/content_102835.htm|archivedate=2005-08-17}}</ref>。1955年担任市委宣传部副部长期间，曾兼任[[长江日报|长江日报]]社社长。1978年，黎智担任工程总投资高达40多亿元的[[武钢|武钢]]一米七工程的指挥部常务书记、副指挥长。1980年7月至1981年3月任[[武汉市|武汉市]]代市长；1981年3月至1983年3月任武汉市长<ref name=cjw2009>{{cite web |title=历任市长——黎智 |url=http://zt.cjn.cn/zt2009/wh60jl/sz/200908/t979328.htm |publisher=长江网 |date=2009-08-13 |accessdate=2012-07-08}}</ref>。他还担任过市委副书记兼市体改委主任，市人大常委会主任、党组书记等职。2001年在武汉逝世<ref name=cjrb2002>{{cite web|author=程秋萍|title=一位可敬的老共产党人——缅怀黎智同志革命业绩|url=http://www.cnhan.com/GB/content/2002-08/05/content_193173.htm|publisher=汉网（转载自《[[长江日报|长江日报]]》）|date=2002-08-05|accessdate=2012-07-08|deadurl=yes|archiveurl=https://web.archive.org/web/20050818120347/http://www.cnhan.com/gb/content/2002-08/05/content_193173.htm|archivedate=2005-08-18}}</ref>。

==家庭==
黎智的第一任妻子为魏克，在1982年病逝<ref name=bjds2004/>。他后来又与赵慧结为夫妻。有子女：闻心慧、闻心悦。
