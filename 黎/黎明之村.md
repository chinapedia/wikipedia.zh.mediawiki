'''黎明之村'''，又名'''曙光之城'''、'''曙光村'''、'''地球村'''（原文 Auroville<ref>Auro-源於聖哲[[奧羅賓多|奧羅賓多]] (Aurobindo) 之名的字首，-ville取自法文，意指「城市」。</ref>），是一個實驗性的寰宇城鎮，位於印度[[泰米爾納德邦|泰米爾納德邦]]的[[維魯普蘭縣|維魯普蘭縣]]，這裡靠近[[印度南部|印度南部]]的[[本地治里|本地治里]]市。係由[[密那·阿爾法薩|米拉·阿拉法薩]]夫人（又名「母親」）於 1968 年所創，並由建築設計師{{link-en|羅傑·安格爾|Roger Anger}}先生實際進行設計。<ref>{{cite web|url=http://www.boloji.com/architecture/00053.htm|title=建築師羅傑·安格爾|publisher=Boloji.com|date=|accessdate=2012-01-26|archive-url=https://web.archive.org/web/20100114073814/http://boloji.com/architecture/00053.htm|archive-date=2010-01-14|dead-url=yes}}</ref><ref>{{cite web |url=http://www.architectureweek.com/2005/1116/culture_1-1.html |title=密那·理查茲創建的黎明之村 |publisher=Architectureweek.com |date=2005-11-16 |accessdate=2012-01-26 |archive-url=https://web.archive.org/web/20161207170027/http://www.architectureweek.com/2005/1116/culture_1-1.html# |archive-date=2016-12-07 |dead-url=yes }}</ref><ref>{{cite web|url=http://www.auroville.info/ACUR/templates/avfuture.htm|title=密那·阿爾法薩，如同其他名字|publisher=Auroville.info|date=|accessdate=2012-01-26|deadurl=yes|archiveurl=https://web.archive.org/web/20040524231954/http://www.auroville.info/ACUR/templates/avfuture.htm|archivedate=2004年5月24日|df=}}</ref>這個城鎮正如母親首次公開發表的訊息那樣：「黎明之村旨在成為一座寰宇城鎮，來自世界各國的男男女女都能夠超越信條、政治、國籍，和平、進步且和諧地住在這裡。黎明之村的目的是實現人類大同。」

== 歷史 ==
黎明之村是[[聖奧羅賓多|聖奧羅賓多]]社群主導的一項計劃，並由[[密那·阿爾法薩|母親]]創建於 1968/2/28（週三）。母親是奧氏的靈性協作人，她相信「人類是處於過渡階段的靈性存有」。母親希望這個「寰宇城鎮」的實驗能將基於善良本意且渴願活在更美好世界的人們集結起來共創璀璨未來，藉此對人性的進化產生巨大貢獻。母親也相信像這樣的寰宇城鎮能對印度文明的復興作出決定性的貢獻。（參見《母親的天命議程》卷 9，dt.3.02.68）。基於這些理由，印度政府遂批准了黎明之村的成立，而在 1966 年，[[聯合國教科文組織|聯合國教科文組織]]（UNESCO）也背書邀請會員國加入黎明之村的發展。過去 40 年來，UNESCO 另四度為黎明之村背書。

1968 年 2 月 28 日，來自 124 國的代表參與了黎明之村的動土典禮，母親在會中發表了一份憲章，闡述她對整體生活的願景，包含四點：

 1.黎明之村不屬於任何人，它屬於全體人類，住在這裡的人須自願服膺於神聖意識。 
 2.黎明之村將是一個供人們進行終身教育、持續進化且永保青春的地方。
 3.黎明之村想要銜接過去和未來，人們能運用所有探索內外的發現，大膽躍向（人類）將來的徹悟。
 4.黎明之村將成為物質和精神研究的場域，在生活中體現真正的人類大同。

== 政府、信仰體系 ==
黎明之村基金會是一個隸屬於印度[https://en.wikipedia.org/wiki/Ministry_of_Human_Resource_Development_(India) 人力資源發展部（HRD）]的自主機構，由印度國會立法授權。<ref>{{cite web|url=http://education.nic.in/cd50years/x/7H/9D/7H9D0101.htm|title=黎明之村基金會法（1988年）|publisher=Education.nic.in|date=|accessdate=2012-01-26|deadurl=yes|archiveurl=https://web.archive.org/web/20120212095936/http://education.nic.in/cd50years/x/7H/9D/7H9D0101.htm|archivedate=2012-02-12}}</ref><ref>{{cite web |url=http://auroville.org/journals&media/newsandnotes/NewsNotes_no_251_July_12_08.zip |title=黎明之村新聞與備註No.251 |date= |accessdate=2012-01-26 |deadurl=yes |archiveurl=https://web.archive.org/web/20120216025529/http://auroville.org/journals%26media/newsandnotes/NewsNotes_no_251_July_12_08.zip |archivedate=2012-02-16 }}</ref>基金會由三部分組成：黎明之村居民（居民大會）、理事會，以及國際顧問委員會，由HRD部任命理事會的七名委員及國際顧問委員會的五名委員，再由理事會指派核心委員會，如基金及資產管理會、預算協調會、l'Avenir（城市規劃局）。基金會另設秘書長一職，亦由印度政府任命，秘書長須住在村內，村內也須有他的辦公室和組員。<ref>2013年黎明之村手冊第14頁</ref>基金會目前約擁有全鎮所需土地的一半。剩餘土地將在基金會有能力時購進。

黎明之村係為真理而生，故須超越所有社會、政治、宗教信仰等派系之外。黎明之村是人類尋求和平、真誠、真理的具現。<ref>黎明之村的母親，第2頁，1999年[[奧羅賓多修道院|奧羅賓多修道院]]信託出版</ref>

== 經濟 ==
[[File:Inuksuit_marking_Canada's_building_site_at_Auroville,_Tamil_Nadu,_India.jpg|thumb]]標示著黎明之村未來的加拿大館建址]]
除了紙幣和硬幣之外，村民都有一組數字帳號連到各自的中央戶頭。但訪客或遊客卻得申請一組臨時戶頭及黎明卡（Aurocard，一種金融卡）。

村民每月應繳納一筆奉獻金給社群，村民們也須盡可能以工作、金錢或關愛協助社群。黎明之村的部分預算來自「訪客奉獻」或以黎明卡支付的日費。黎明之村也有一套贍養系統，藉由這套系統，有需求的村民每月會從社群那裡收到一筆贍養費，足供應簡單的基本生活開銷。黎明之村的經濟及整體生活具有持續進化的本質，為了更接近這個願景，村內有多個進行中的實驗。<ref>{{cite web|url=http://business.in.com/article/work-in-progress/commerce-in-a-cocoon/3722/1 |title=富比世印度 |publisher=Business.in.com |date= |accessdate=2012-01-26}}</ref>

儘管印度政府擁有並管理黎明之村基金會，印度政府的金援只佔了黎明之村預算的一小部分；大部分的預算來自黎明之村營業單位的奉獻（其營收佔了黎明之村中央基金的 33%）以及其他捐獻。村內有民宿、營建單位、資訊科技業、製造並轉售如手工紙張等文具用品及著名線香品牌的中小型商業，黎明之村生產的線香亦可在[[本地治里|本地治里]]的自營店、印度各地和國外買到。這些單位各將盈利的相當比例奉獻給黎明之村。村內有逾五千人受雇在各產業或單位，他們多來自鄰近地區。

其他活動包括綠化、有機農業、基本教育研究、保健、村落發展、[[適用技術|適用技術]]、城鎮規畫、地下水位管理、文化活動及社群服務。

== 地理位置 ==
[[File:Auro_044.jpg|thumb]]
[[File:Auro_082.jpg|thumb]]

黎明之村的地產遍及[[本地治里|本地治里]]北方約 12 公里（7.5 英哩）處。連接[[清奈|清奈]]及本地治里的[https://en.wikipedia.org/wiki/East_Coast_Road 東岸道路]（ECR）是抵達黎明之村最便捷的通道。從 ECR 路標的叉路往西走 6 公里（3.7 英哩）可抵達遊客中心及[[黃金球_(曙光村)|黃金球]]。向東轉走約數百公尺則會直接通向黎明之村的私有海灘 Repos。

== 氣候 ==
黎明之村位於半濕潤熱帶地區的台地，最大海拔高度是[[黃金球_(曙光村)|黃金球]]所在的 32 公尺（105 英呎）處。年平均降雨量為 1.200 公厘（47 英吋），雨水主要來自六到九月的西南季風及十一到十二月的東北季風，乾季約有六個月。平均最高溫度是 32.2°C (90°F)，平均最低溫度是 20°C (68°F)。

== 傳播媒體 ==
黎明之村網站提供了各種計畫、活動、組織及外援主題的公開及非公開論壇，涵蓋了社群生活的各個面向。<ref>[http://www.auroville.org/journals&media/journals.htm 黎明之村期刊及新聞稿] {{webarchive|url=https://web.archive.org/web/20060830023315/http://www.auroville.org/journals%26media/journals.htm |date=2006-08-30 }}</ref>這些論壇上發表的意見不必然就代表社群大部分人的意見。黎明之村廣播電台網站播放大量的錄製節目及涵蓋村內各事件的每日新聞。此外，黎明之村也有一支稱為「外援媒體（OutreachMedia）」的小隊，負責控管記者及影視製作人，這組人馬的目標是確保所有記者及製片人都能取得最新的官方資訊，得從可靠消息來源取得具代表性的影片。
