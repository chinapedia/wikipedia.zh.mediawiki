'''黎明职业大学'''，简称'''黎大'''，位于[[福建省|福建省]][[泉州市|泉州市]]，成立于1984年，是经[[中华人民共和国教育部|中华人民共和国教育部]]备案的、具有独立颁发高等教育学历证书资格的全日制普通高等职业院校。其前身是1929年由[[中国同盟会|中国同盟会]]会员[[许卓然|许卓然]]、[[秦望山|秦望山]]等人创立的黎明高中。

== 学校简介 ==
学校旧校区位于泉州中山北路。新校区位于东海街道通港西街298号。

== 校史简述 ==
黎明职业大学的前身系成立于1929年的[[黎明高中|黎明高中]]，由[[中国同盟会|同盟会]]会员[[许卓然|许卓然]]、[[秦望山|秦望山]]等人创建。

1981年，[[梁披云|梁披云]]联络了海内外华侨，在黎明高中旧址发起创办了[[黎明学园|黎明学园]]。1984年正式创办黎明职业大学。
== 现任主要领导 ==
校[[党委书记|党委书记]]：林其天

校长：林松柏
