'''亞當斯州立大學'''（'''Adams State University'''，縮寫：'''ASU'''）是一所位于美國[[科羅拉多州|科羅拉多州]][[阿拉莫萨_(科罗拉多州)|阿拉莫萨]]、[[大沙丘国家公园|大沙丘国家公园]][[聖路易斯谷|聖路易斯谷]]的公立[[美國文理學院|文理學院]]。該大學創始于1921年，當時為一所師範學校。2012年5月22日升格為大學<ref>Adams State College. [http://www.adams.edu/news/may1214.php Adams State becomes a University] {{Wayback|url=http://www.adams.edu/news/may1214.php |date=20170104234838 }} viewed June 1, 2012.</ref>。2017年《[[美國新聞與世界報道|美國新聞與世界報道]]》未給予其全國排名資格。<ref name="USnews">{{cite web|title=Adams State University|url=http://colleges.usnews.rankingsandreviews.com/best-colleges/adams-state-college-1345|publisher=[[美國新聞與世界報道|美國新聞與世界報道]]|accessdate=2016-10-28}}</ref>
