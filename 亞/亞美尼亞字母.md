'''亞美尼亞字母'''是一種[[字母|字母]]，從5世紀開始用來書寫[[亚美尼亚语|亞美尼亞語]]，当时有36个字母。到19世紀為止，[[古典亞美尼亞語|古典亞美尼亞語]]都是文學語言。19世紀以後，亞美尼亞字母用來書寫兩種現代、文學和口語的方言[[東亞美尼亞語|東亞美尼亞語]]和[[西亞美尼亞語|西亞美尼亞語]]。兩者都在同一時期創造出來<ref>[http://armlet1600.sci.am/grerimasin.php?langid=3 G.Khandjian – 亞美尼亞字母的創造] {{webarchive|url=https://web.archive.org/web/20070626105218/http://armlet1600.sci.am/grerimasin.php?langid=3 |date=2007-06-26 }}</ref>。

亞美尼亞語中，“字母”稱為 {{lang|hu|այբուբեն}}（[[古典亞美尼亞語|古典亞美尼亞語]]和[[東亞美尼亞語|東亞美尼亞語]]：{{IPA|[ɑjbubɛn]}}或[[西亞美尼亞語|西亞美尼亞語]]：{{IPA|[ɑjpʰupʰɛn]}}），以首兩個字母命名。共38个。

=== 注釋 ===
在以上列表，上標的''h''（{{IPA|[ʰ]}}）在國際音標表示[[送氣|送氣音]]； 撇號（{{IPA|[ʼ]}}）表示[[擠喉音|擠喉音]]。 

# {{note|1}}在詞首時讀成 ''ye'' {{IPA|/jɛ/}}，在其它情況讀成 ''e'' {{IPA|/ɛ/}}。原因是 ''y'' {{IPA|/j/}}在[[複合詞|複合詞]]脫落，例如 {{lang|hy|եղբայր}}（ełbayr；{{IPA|/jɛʀˈbajɹ/}}“兄弟”）和 {{lang|hy|մորեղբայր}}（morełbayr, {{IPA|/moɹɛʀˈbajɹ/}} “舅父”）。
# {{note|2}}按照改革拼寫法，{{lang|hy|{{angbr|լ}}}}的名稱讀成{{IPA|[lyun]}}。
# {{note|3}}只在傳統拼寫法出現，位於詞首或詞語中的詞幹。
# {{note|4}}在詞首時讀成 ''vo'' {{IPA|/vo/}}，在其它情況讀成 ''o'' {{IPA|/o/}}。原因是{{IPA|/v/}}在複合詞脫落，例如 {{lang|hy|որդի}}（ordi；{{IPA|/voɹˈtʰi/}} “兒子”）和 {{lang|hy|քեռորդի}}（{{Unicode|kʻeṙordi}}；{{IPA|/kʰeroɹˈtʰi/}}“舅表兄弟”）。
# {{note|5}}按照改革拼寫法，{{lang|hy|{{angbr|տ}}}}的名稱讀成{{IPA|[tʼyun]}}。
# {{note|6}}實際上，只有伊朗亞美尼亞人讀{{IPA|[ɹ]}}；[[亚美尼亚|亞美尼亞]]的東亞美尼亞人將古典亞美尼亞語{{IPA|[ɹ]}}（{{lang|hy|ր}}）讀成 {{IPA|[ɾ]}}。
# {{note|7}}根據改革拼寫法，此字母被代表{{IPA|[u]}}的單元音{{lang|hy|{{angbr|ու}}}}取代。
# {{note|8}}它一般代表{{IPA|/v/}}，但是有例外情況。在古典亞美尼亞語，在詞首並在輔音前的 {{lang|hy|աւ}} 代表{{IPA|/au/}}，例如 {{lang|hy|աւր}}（awr；{{IPA|/auɹ/}} “白天”）。因為中世紀時[[語音演變|語音演變]]，此讀音已變成{{IPA|/oɹ/}}，而在13世紀以後寫成 {{lang|hy|օր}}（ōr）；單元音 {{lang|hy|ու}}（代表{{IPA|/ov/}}或{{IPA|/ou/}}）變成{{IPA|/u/}}；單元音 {{lang|hy|իւ}}（iw）代表{{IPA|/ju/}}。蘇聯統治下的亞美尼亞，將 {{lang|hy|ի}}（''i''）改成 {{lang|hy|յ}}（''y''），{{lang|hy|ւ}}（''w''）改成 {{lang|hy|ու}}（''ow''），產生雙元音 {{lang|hy|յու}}。
# {{note|9}}根據改革拼寫法，{{lang|hy|{{angbr|փ}}}}的名稱讀成{{IPA|[pʰjuɾ]}}。

== 連體字母 ==
[[意音文字|詞符]]{{lang|hy|{{angbr|և}}}}是以字母{{lang|hy|եւ}}組成的[[合體字母|合體字母]]，是亞美尼亞語的[[連詞|連詞]]，意為“和”。1920年代，亞美尼亞語的拼寫法改革引進此詞符。詞符與[[拉丁语|拉丁語]]的[[&|意符&]]類似。按照改革拼寫法，它經常在{{lang|hy|{{angbr|Օ}}}}前出現。一些使用傳統拼寫法的文本，也使用{{lang|hy|{{angbr|և}}}}。

== 標點符號 ==
在亞美尼亞語，「{{lang|hy|,}}」（storaket）是逗號，「{{lang|hy|:}}」（verjaket）是普通句號，「{{lang|hy|՟ }}」（pativ）則是縮寫詞使用的標記，現時已不再使用。問號「{{lang|hy|՞ }}」（hartsakan）寫在疑問詞最後字母和倒數第二字母之間。「{{lang|hy|՝ }}」（boot'）書寫位置與問號一樣，表示一個短暫停頓，但是比逗號長、比分號短。感嘆號「{{lang|hy|՛ }}」（yerkaratsman nshan）寫在感嘆詞最後字母和倒數第二字母之間。「{{lang|hy|« »}}」（čakertner）是引號。

== 歷史與演進 ==
{{alphabet}}

公元405年，[[聖梅斯羅布|聖梅斯羅布]]創造了亞美尼亞字母，主要是為了撰寫亞美尼亞語的[[聖經譯本|聖經譯本]]。數種字體被認為是亞美尼亞字母的雛形，包括[[帕拉維字母|帕拉維字母]]（{{lang|en|Pahlavi}}）、[[敘利亞字母|敘利亞字母]]（{{lang|en|Syriac}}）和[[腓尼基字母|腓尼基字母]]。然而，亞美尼亞字母的次序，意味著它很可能是以[[希腊字母|希臘字母]]為基礎的。

雖然現代亞美尼亞語的方言[[東亞美尼亞語|東亞美尼亞語]]和[[西亞美尼亞語|西亞美尼亞語]]都是用同樣的字母，但後者的語音轉變，使部分字母讀音不同。有關的字母包括<!--（以下的圖表有更多資料）-->：

* [[塞音|塞音]]
** {{lang|hy|բ}}（{{IPA|[b]}} 變為 {{IPA|[pʰ]}}）和 {{lang|hy|պ}}（{{IPA|[p]}} 變為 {{IPA|[b]}}）
** {{lang|hy|դ}}（{{IPA|[d]}} 變為 {{IPA|[tʰ]}}）和 {{lang|hy|տ}}（{{IPA|[t]}} 變為 {{IPA|[d]}}）
** {{lang|hy|գ}}（{{IPA|[g]}} 變為 {{IPA|[kʰ]}}）和 {{lang|hy|կ}}（{{IPA|[k]}} 變為 {{IPA|[g]}}）

* [[破擦音|破擦音]]
** {{lang|hy|ջ}}（{{IPA|[d͡ʒ]}} 變為 {{IPA|[t͡ʃʰ]}}）和 {{lang|hy|ճ}}（{{IPA|[t͡ʃ]}} 變為 {{IPA|[d͡ʒ]}}）
** {{lang|hy|ձ}}（{{IPA|[d͡z]}} 變為 {{IPA|[t͡sʰ]}}）和 {{lang|hy|ծ}}（{{IPA|[t͡s]}} 變為 {{IPA|[d͡z]}}）



字母的數目和次序隨著時代而變化。中世紀時，兩個新字母（{{lang|hy|օ}} {{IPA|[o]}}、{{lang|hy|ֆ}} {{IPA|[f]}}）出現，以更準確反映外語語音，使字母數目由36增至38。在古典亞美尼亞語，在詞首並在輔音前的 {{lang|hy|աւ}} 代表{{IPA|/au/}}，例如 {{lang|hy|աւր}}（awr；{{IPA|/auɹ/}} “白天”）。但後來，此讀音已變成{{IPA|/oɹ/}}，而在13世紀以後寫成 {{lang|hy|օր}}（ōr）。故此，現時亞美尼亞語一些本族詞以字母 {{lang|hy|օ}}（ō）為首，縱使此字母取自[[希腊字母|希臘字母]]，用來顯示以 ''{{lang|la|o}}'' {{IPA|[o]}}為字頭的外來詞讀音。

1922至24年，[[亚美尼亚苏维埃社会主义共和国|亞美尼亞蘇維埃社會主義共和國]]實行拼寫法改革。改革並沒有改變字母的讀法，但是有一些例外。流散在國外的亞美尼亞人（包括在[[黎巴嫩|黎巴嫩]]和[[伊朗|伊朗]]者）不接受改革，仍然使用古典亞美尼亞語的拼寫法。他們批評改革的某些問題，並宣稱改革有政治動機。

== 其他語言 ==
根据[[哥倫比亞大學|哥倫比亞大學]]的{{lang|hy|Petross Ter Matossian}}，從18世紀初期到大約1950年，超過二千本[[土耳其语|土耳其語]]書籍在250年之間印制，並採用亞美尼亞字母。亞美尼亞人閱讀這類書籍，其他族裔的（包括[[奥斯曼帝国|奧斯曼]]土耳其人）精英也有。在奧斯曼帝國，公文都使用亞美尼亞與阿拉伯字母，只是以[[奧斯曼土耳其語|奧斯曼土耳其語]]撰寫。例如，1851年時''{{lang|tr|Akabi}}''成為第一本在奧斯曼帝國推出的[[小说|小說]]，由亞美尼亞人{{lang|hy|Hovsep Vartan}}以亞美尼亞字母撰寫。而且，[[阿卜杜勒·邁吉德一世|阿卜杜勒·邁吉德一世]]在位時，亞美尼亞裔的{{lang|tr|Düzoğlu}}家族負責錢幣鑄造，用亞美尼亞字母、以土耳其語去撰寫紀錄。

=== Unicode ===

字母編碼範圍是從U+0530到U+058F。

=== 已廢止編碼 ===
==== ArmSCII-8 ====
ArmSCII-8是{{lang|en|Armenian Standard Code for Information Interchange}}的8-bit編碼，在1991至99年期間發展<ref>{{cite web | last = Melikyan | first = Hovik | title = ARMENIAN STANDARD CODE FOR INFORMATION INTERCHANGE -- ArmSCII | url = http://www.freenet.am/armscii/ | accessdate = 2006年11月30日 | deadurl = yes | archiveurl = https://web.archive.org/web/20070204043856/http://www.freenet.am/armscii/ | archivedate = 2007年2月4日 }}</ref>。它使用一個8-bit編碼上面128個字符去代表亞美尼亞字母，其餘128個字符代表另一種字母，通常是拉丁或西里爾字母。這容許以“一個”字形代表兩種字母，例如拉丁字母佔用一些首128個字符（如[[ASCII|ASCII]]），另一些則由亞美尼亞字母佔用。

ArmSCII-8在Windows 95和Windows 98視窗作業系統普遍使用。使用者如要閱讀亞美尼亞字母，要下載採用此編碼的字型。如要輸入字母，則要下載和使用一個在工具列{{lang|en|Armenian National Language Support}}運行的免費程序<ref>{{cite web | last = Melikyan | first = Hovik | title = Armenian National Language Support Version 2.0 | url = http://www.freenet.am/armnls/ | accessdate = 2006年11月30日 | deadurl = yes | archiveurl = https://web.archive.org/web/20070126055839/http://www.freenet.am/armnls/ | archivedate = 2007年1月26日 }}</ref>。這個程序讓使用者同時輸入亞美尼亞和另一種字母，而不需要轉字型。轉字型和鍵盤佈局時，只需使用熱鍵（通常是''Alt + Shift''）。

後來，更先進的[[Unicode|Unicode]]標準開始發展，並能在Windows 2000/XP/2003和Linux等系統使用。故此，ArmSCII-8編碼被廢棄。然而，一些尚未轉用Unicode的網頁仍然使用ArmSCII-8。

==== Arasan-compatible ====
Arasan-compatible的字型是以原本的Arasan字型為基礎，只是以亞美尼亞的[[ASCII|ASCII]]編碼取代拉丁字母字符，例如ASCII中的拉丁字符{{lang|la|{{angbr|A}}}}（65）代表亞美尼亞字母{{lang|hy|{{angbr|Ա}}}}。

這種的字型比ArmSCII-8優勝的是，輸入字母時並不需要多一個程序。只要安裝並選擇了字型，就可以用[[QWERTY|QWERTY]]鍵盤輸入亞美尼亞字母。然而，其缺點是只能使用一種字母。創建多於一種字母的文件時，需要轉換字型。而且，Arasan字型只有西亞美尼亞語音的鍵盤佈局。

這種字型在Windows 95和98系統很受歡迎，但隨著Unicode標準推出而被廢棄。然而，一些網頁仍然使用它。

Arasan字型的影響是，Windows 2000/XP/2003系統提供的亞美尼亞語語音鍵盤佈局，跟Arasan的幾乎一樣。

== 電腦字型 ==
數種作業系統上，亞美尼亞字母的字型都可以在個人電腦安裝和使用。Windows XP及以上版本提供的字型[[Sylfaen|Sylfaen]]採用傳統風格的Unicode亞美尼亞字母字符。開放源碼套裝[[DejaVu字型|DejaVu字型]]、[[GNU_FreeFont|GNU FreeFont]]也採用該風格的字符，並在[[Linux|Linux]]普遍使用。由於這些字型安裝輕便，可以在不同的作業系統使用。

== 鍵盤佈局 ==
使用者可以調整作業系統的[[鍵盤佈局|鍵盤佈局]]，例如增加佈局數目，用熱鍵轉換佈局（通常是 ''alt'' + ''shift''）。

=== Windows 2000/XP/2003 ===
Windows 2000、Windows XP和Windows 2003系統同時提供東亞美尼亞語和西亞美尼亞語鍵盤佈局<ref>{{Cite web |url=http://www.microsoft.com/globaldev/reference/keyboards.mspx |title=Windows Keyboard Layouts |access-date=2007-01-22 |archive-url=https://web.archive.org/web/20070123134145/http://www.microsoft.com/globaldev/reference/keyboards.mspx |archive-date=2007-01-23 |dead-url=yes }}</ref>。它們都是根據Windows 95系統中受歡迎的Arasan字型而創製。這些佈局大致是根據語音而設計。然而，一些亞美尼亞字母並不跟拉丁字母直接對應，於是通常配以大約讀音的按鍵，例如「''{{lang|hy|Խ}}''」對應「''{{lang|la|Q}}''」。而且，由於亞美尼亞字母（38個）數目比拉丁字母（26個）的多，前者一些字母在 [[QWERTY|QWERTY]]佈局的非字母按鍵出現，例如「''{{lang|hy|շ}}''」對應「''{{lang|la|,}}''」。

=== Linux ===
所有[[Linux套件列表|Linux發行版]]都預先設定獨特的鍵盤佈局。為了保持一致，Linux發行版通常採用來自[[Freedesktop.org|Freedesktop.org]]的{{lang|en|XKeyboard Configuration component}}之佈局。到2006年11月為止，Freedesktop.org包含五種亞美尼亞語鍵盤佈局，其中兩種與Windows XP的一樣<ref>[http://webcvs.freedesktop.org/xlibs/xkbdesc/symbols/ XKeyboard Config CVS] {{webarchive|url=https://web.archive.org/web/20061012050006/http://webcvs.freedesktop.org/xlibs/xkbdesc/symbols/ |date=2006-10-12 }} - 檔案''am''對應亞美尼亞語鍵盤佈局。</ref>。到版本10.1為止，[[SUSE_Linux|SUSE Linux]]支持兩種亞美尼亞語鍵盤佈局。雖然它不包括Windows XP的佈局，但仍然可以手動安裝<ref>[http://hy4linux.blogspot.com/2006/09/installing-new-armenian-keyboard.html Installing the new Armenian Keyboard layouts for GNU/Linux]</ref>。

如要在Linux使用亞美尼亞語鍵盤佈局，需要另行設定。使用[[GNOME|GNOME]]的人可以使用{{lang|en|GNOME Keyboard Indicator applet}}<ref>[http://hy4linux.blogspot.com/2006/09/gnulinux-typing-in-armenian-using.html GNU/Linux: Typing in Armenian using GNOME]</ref>。
