《'''亞爾斯蘭戰記'''》是[[田中芳樹|田中芳樹]]自1986年開始連載的[[奇幻|奇幻]][[小說|小說]]，1至10卷由[[角川書店|角川書店]]的[[角川文庫|角川文庫]]出版；後改為[[光文社|光文社]]出版<ref>{{cite web|title=『アルスラーン戦記』が光文社文庫に収められます！|url=http://a-hiro.cocolog-nifty.com/diary/2012/03/post-3587.html|accessdate=2015年2月9日|language=ja|date=2012年3月24日}}</ref>，1至10卷重新再版，並發行11卷以後的小說。本作在1990年代曾推出漫畫和動畫：漫畫版當時名為《阿爾斯朗戰記》，由{{link-ja|中村地里|中村地里}}繪畫，[[台灣東販|台灣東販]]代理中文版；動畫第一、二部以[[劇場版|劇場版]]分別於1991年和1992年推出，其後在1993年至1995再發行4部[[OVA|OVA]]。

2013年7月9日在《[[別冊少年Magazine|別冊少年Magazine]]》推出由《[[鋼之鍊金術師|鋼之鍊金術師]]》作者[[荒川弘|荒川弘]]繪畫的漫畫版<ref>{{cite web|title=《荒川弘版亞爾斯蘭戰記》荒川弘 x 田中芳樹新版亞爾斯蘭戰記連載漫畫開始|url=http://news.gamme.com.tw/538240|publisher=宅宅新聞|accessdate=2015年1月30日|date=2013年7月26日}}</ref>。2014年11月宣佈第三度動畫化，並將改編為電視動畫<ref>{{cite web|title=荒川弘x田中芳樹！漫畫《亞爾斯蘭戰記》確定動畫化，首部宣傳影片公開！|url=http://www.animen.com.tw/Page/News/NewsContent.aspx?ID=NS14110009|publisher=Animen新聞部|accessdate=2015年1月30日|date=2014年11月2日|deadurl=yes|archiveurl=https://web.archive.org/web/20150204130405/http://www.animen.com.tw/Page/News/NewsContent.aspx?ID=NS14110009|archivedate=2015年2月4日}}</ref>：與前作不同，電視動畫將以荒川弘的漫畫版為藍本用作改編，於2015年4月5日開始播放第1期<ref name="official">{{Cite web|url=http://www.arslan.jp/news/index00040000.html|title=ＴＶアニメ『アルスラーン戦記』 放送日決定＆メインキャスト第一弾発表！|work=動畫「亞爾斯蘭戰記」官方網站|date=2015年2月1日|accessdate=2015年2月1日|language=ja}}</ref>。2015年10月宣佈製作電視動畫第2期《亞爾斯蘭戰記 風塵亂舞》，於2016年7月播放。

== 故事簡介 ==
第一部為小說的第1至7集，講述大陸強國「帕爾斯王國」因鄰國「魯西達尼亞」入侵時，手下大將叛變而令首都淪陷，王太子亞爾斯蘭在猛將「達龍」的協助下死裏逃生，並前往請求智將「那爾撒斯」幫助，在兩人的輔佐之下途中招募猛將，並踏上艱辛的復國之路。

第二部從小說第8集開始，目前連載中。

== 主要國家 ==
; {{nihongo|帕爾斯|パルス}}
: 故事發生的主要王國。大陸公路（現實世界的[[絲綢之路|絲綢之路]]）的中轉站。
: 王都為[[埃克巴坦那|葉克巴達那]]。
: 原型為[[萨珊王朝|萨珊波斯]]。

; {{nihongo|魯西達尼亞|ルシタニア}}
: 帕爾斯西北面的王國。
: 原型是中世紀的西歐諸國。

; {{nihongo|辛德拉|シンドゥラ}}
: 位於帕爾斯東南面的王國，以卡威利河為兩國國界。擁有獨特的戰象部隊。
: 王都是烏萊優魯。
: 原型為[[印度|印度]]。

; {{nihongo|馬爾亞姆|マルヤム}}
: 位於帕爾斯西北面的王國。有安定的歷史和實力。和周邊各國的外交關係極為良好，雖然信仰依亞爾波特教，卻是由穩健的東方教會在指導。
: 王都是伊拉克里恩。
: 原型是[[拜占庭帝國|拜占庭帝國]]。

; {{nihongo|密斯魯|ミスル}}
: 位於帕爾斯西面的王國，以迪吉列河為界。有駱駝部隊及戰車。
: 王都是亞克密姆。
: 原型是[[埃及|埃及]]。

; {{nihongo|邱爾克|チュルク}}
: 位於帕爾斯東面的王國。
: 王都是赫拉特。
: 原型是[[阿富汗|阿富汗]]，連王都[[赫拉特|赫拉特]]的名字都完全是直接照搬了現實中阿富汗的城市名字。

; {{nihongo|特蘭|トゥラーン}}
: 位於帕爾斯東北面的[[遊牧|遊牧]]騎馬民族王國，與帕爾斯是世代的宿敵。
: 王都是沙曼岡。
: 原型為真實歷史上[[萨珊王朝|萨珊波斯]]的宿敵[[嚈噠帝國|嚈噠帝國]]。

; {{nihongo|絹之國|絹の国（セリカ）}}
: 在帕爾斯東面但不相鄰的文明國家，大陸公路東端位置的大[[帝國|帝國]]。
: 帝都是西都永安府，首都人口達200萬。
: 原型為[[中國|中國]]。

; {{nihongo|那巴達|ナバタイ}}
: 密斯魯南方的黑人之國，分成東西兩個王國，一起支配著第吉雷河上游廣大延伸的高地。
: 原型為[[努比亞|努比亞]]和[[埃塞俄比亞|埃塞俄比亞]]。

; {{nihongo|巴達夫夏|バダフシャーン}}
: 巴達夫夏公國本為帕爾斯東方的小國，在帕爾斯曆303年被帕爾斯時任大將軍的安德拉寇拉斯滅亡。

; {{nihongo|法爾哈爾|ファルハール}}
: 帕爾斯東方的公國，盛產[[翡翠|翡翠]]和[[紅玉|紅玉]]。
: 原型為[[樓蘭|樓蘭]]等中亚的[[都市國家|都市國家]]。

== 用語解說 ==
; {{nihongo|萬騎長|万騎長（マルズバーン）}}
: 帕爾斯授予武將的官職。第一部時是實領一萬名騎兵，並有十二人被授予此職位。之下則有千騎長、百騎長等。

; {{nihongo|大將軍|大将軍（エーラーン）}}
: 帕爾斯武將官職的最高職位。通常只授予一人。第一部時是巴夫利斯，安德拉寇拉斯三世即位前也曾擔任過此職位。亞爾斯蘭即位後則由奇斯瓦特擔任。

; {{nihongo|大陸公路|大陸公路}}
: 一條以帕爾斯王國為中心，向東西方各延伸八百法爾桑（約四千公里），連接廣大大陸兩端的交易之路。這條交易道路和通過此路的商隊都接受帕爾斯王的保護，納通行稅給帕爾斯王，鞏固了帕爾斯的繁榮。東方的盡頭是大陸公路另一強國──絹之國。

; {{nihongo|達爾邦內海|ダルバンド（ダルヴァンド）内海}}
: 位於帕爾斯北部、馬爾亞姆、特蘭之間的大湖。和現實世界的[[裡海|裡海]]相當。

; {{nihongo|迪馬邦特山|デマヴァンド山}}
: 帕爾斯境內的山。蛇王撒哈克被封印於此。帕爾斯人都將此山稱之為「魔山」而感到十分畏懼，山腳附近有村落在。英雄王凱·霍斯洛逝世後遺命將自己與寶劍魯克那巴德葬於此處，好繼續封印蛇王。

; {{nihongo|不死隊|不死隊（アタナトイ）}}
: 帕爾斯國王的親衛隊，是由五千人的出色騎兵所構成。

; {{nihongo|獅子獵人|獅子狩人（シールギール）}}
: 獨自一人獵殺獅子的戰士被授予的稱號。

; {{nihongo|芸香|芸香（ヘンルーダ）}}
: 是一種驅魔道具，散發的香氣有驅魔的功效，是商旅必備的物品。在野宿時商旅會將芸香放進篝火裏，火焰會在瞬間爆開，一股聞似橘子的強烈香氣會瀰漫在篝火四周。魔物都厭惡芸香的香氣而不會接近，而且嗅到後會露出真面目，因此有驅邪的作用。

; {{nihongo|紅玉|紅玉（ラアル）}}

; {{nihongo|綠玉|緑玉（エメラルド）}}

; {{nihongo|有翼猿魔|有翼猿鬼（アフラ・ヴィラーダ）}}
: 傳說中的怪物，體型大小與人類差不多，五官與四肢則近似猿猴，似人又似猿的身體上長著巨大的蝙蝠翼。牙齒與爪子含有劇毒能腐蝕生物，以人肉為食，尤其喜吃小孩與嬰兒的嫩肉。這群怪物過去曾遭聖賢王夏姆席德放逐到地底的熔岩城，成為蛇王撒哈克的侍從，之後隨著撒哈克的敗北而消聲匿跡。

; {{nihongo|鳥面人妖|鳥面人妖（ガブル・ネリーシャ）}}
: 傳說中的怪物，與有翼猿魔同為蛇王撒哈克的侍從。但與有翼猿魔不同的是，鳥面人妖會化身成人類，並具有相當的智慧隱瞞自己的身份。有嗜食人類胎兒的癖好。

; {{nihongo|四眼犬|四眼犬（シェムル）}}

; {{nihongo|食屍鬼|食屍鬼（グール）}}

; {{nihongo|客將軍|客将軍（アミーン）}}

; {{nihongo|孔雀姬|孔雀姫（ターヴース）}}

; {{nihongo|加斯|ガズ}}
: 帕爾斯的長度單位，1加斯約相當於現在1米。

; {{nihongo|法爾桑|ファルサング}}
: 帕爾斯的距離單位，1法爾桑約相當於現在5公里。

; {{nihongo|金幣、銀幣、銅幣|金貨（デーナール）、銀貨（ドラフム）、銅貨（ミスカール）}}

; {{nihongo|王族、貴族、騎士、自由民、奴隸|王族（ワースプフラーン）、貴族（ワズルガーン）、騎士（アーザーターン）、自由民（アーザート）、奴隷（ゴラーム）}}

; {{nihongo|奴隶制度|}}
: 帕尔斯与辛德拉、密斯鲁等邻近国家，都有实行奴隶制度。但自帕尔斯历321年，亚尔斯兰王即位后，帕尔斯的奴隶制度被废止，因此亚尔斯兰有「解放王」的称誉。在亚尔斯兰时代以前，帕尔斯的阶级制度，大致上可以这样分等∶由最高的王室，到贵族、诸侯，再到平民，最后才到奴隶。奴隶是贵重的财产，往往有以奴隶多寡来作为富裕程度的标准。奴隶没有薪水收入，只由主人供给饮食，而且身份是代代相传。如果碰上善良的主人则是奴隶的幸运，也有善良的主人会解放奴隶而给予他们自由民的身份。国家军队的编制也与奴隶制度有关，军队的骑兵都是由平民或以上的人担任，由国家支付薪水；而最低级的步兵则由奴隶担任，而且也不会支付薪水。而自亚尔斯兰王废止奴隶制度后，为邻近诸国的阶级制度带来冲击，诸国都防范解放奴隶的风气扩展到自己的国家，因此都加深了对帕尔斯的戒心和反抗。

== 動畫 ==
劇場版和OVA劇情至小說第5集，由[[松竹|松竹]]發行。第1、2集是以電影版形式推出，而其後則以OVA版推出，均由小說的原出版社角川書店發行。第一集劇場版推出於1991年，第二集1992年，兩集片長60分鐘，之後在93年再發行OVA版，片長30分鐘。動畫版內容相當跟從原著，並由插畫家[[神村幸子|神村幸子]]負責人物設定。無綫電視曾以《阿斯蘭戰記》為名播放劇場版和OVA。
