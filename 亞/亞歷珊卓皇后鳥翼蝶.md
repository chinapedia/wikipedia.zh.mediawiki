'''亞歷珊卓皇后鳥翼蝶'''（[[学名|学名]]：'''''{{lang|la|Ornithoptera alexandrae}}'''''），又名'''亞歷山大鳥翼蝶'''或'''亞歷山大鳳蝶'''，是[[世界|世界]]上最大的[[蝴蝶|蝴蝶]]。牠們是由[[羅斯柴爾德|羅斯柴爾德]]（Walter Rothschild）於1907年所命名，是為紀念英王[[愛德華七世_(英國)|愛德華七世]]的妻子[[亞歷山德拉皇后|亞歷珊卓皇后]]。牠們只分佈在[[新畿內亞|新畿內亞]]東部的[[北部省_(巴布亞紐幾內亞)|北部省]]。

亞歷山大鳥翼蝶以往是分類在[[裳鳳蝶屬|裳鳳蝶屬]]中，現已分類在[[鳥翼蝶屬|鳥翼蝶屬]]。另有建議將牠們分類在自己的[[屬|屬]]中。<ref>{{Cite journal en|author=Deslisle, G.|year=2007|title=''Straatmana'': a remplacement name for the subgenus ''Zeunera'' Deslisle 2001 (Lep. Papilionidae)|journal=Lambillionea (Brussels)|volume=1 & 2|pages=238}}</ref>

== 特徵 ==
雌性亞歷珊卓皇后鳥翼蝶較雄蝶大，翅膀也較圓及闊。雌蝶翼展達31厘米，體長8厘米，重12克。雌蝶的翅膀呈褐色，有白色斑紋，身體呈奶白色，[[胸部|胸部]]局部有紅色的絨毛。雄蝶較為細小，翅膀也是呈褐色，有虹藍光澤及綠色斑紋，[[腹部|腹部]]鮮黃色。雄蝶翼展約16-20厘米。牠們有一種形態，後翅有金點。<ref>{{cite web |title= Endangered Species - Queen Alexandra's Birdwing Butterfly Facts |url= http://www.earthsendangered.com/profile.asp?view=all&ID=4&sp=52 |accessdate=2009-02-17}}</ref>

== 寄生植物 ==
亞歷山大鳥翼蝶的[[幼蟲|幼蟲]]在[[擬馬兜鈴屬|擬馬兜鈴屬]]上覓食。牠們最初會吃嫩葉，在結[[蛹|蛹]]前會吃蔓藤。這些[[植物|植物]]的[[葉子|葉子]]及莖上有[[馬兜鈴酸|馬兜鈴酸]]，對[[脊椎動物|脊椎動物]]有毒及會積聚在幼蟲體內。成蟲會在[[木槿屬|木槿屬]]等的[[花朵|花朵]]上覓食。

== 生物學 ==
雌性亞歷珊卓皇后鳥翼蝶一生會產約27顆卵。初出生的[[幼蟲|幼蟲]]會先吃其卵殼，再而吃嫩葉。幼蟲呈黑色，有紅色的結節，在中間有奶白色的橫紋。幼蟲會環割[[寄生|寄生]]的[[植物|植物]]，並會到毗連的植物結[[蛹|蛹]]。蛹呈金黃色或黃褐色，有黑色斑紋。雄蝶蛹有木炭色的斑駁，最終會成為成蟲的特別鱗片。由卵至成蛹約需6星期，而蛹期約1個月或更長。成蟲會選擇在[[濕度|濕度]]較高的早上破蛹，以避免翅膀乾涸。成蟲壽命約為3個月。牠們的[[掠食者|掠食者]]包括[[大木林蛛|大木林蛛]]及一些細小的[[鳥類|鳥類]]。

亞歷珊卓皇后鳥翼蝶在早上及[[黃昏|黃昏]]十分活躍，並會在花間覓食。雄蝶在早上會在寄生植物附近尋找雌蝶。雄蝶會徘徊在雌蝶附近，釋出[[費洛蒙|費洛蒙]]來引發[[交配|交配]]。接受雌蝶會讓雄蝶降落，而不接受的雌蝶就會飛走或抗拒交配。雄蝶是極度地盤性的，並會趕走敵人。牠們一般飛得很高，當覓食或產卵時會降下到地上幾米高。

== 威脅 ==
亞歷珊卓皇后鳥翼蝶被[[世界自然保護聯盟|世界自然保護聯盟]]列為[[瀕危物種|瀕危物種]]<ref name=IUCN/>，只分佈在[[新畿內亞|新畿內亞]][[北部省_(巴布亞紐幾內亞)|北部省]]近岸[[雨林|雨林]]的100平方公里內。牠們在當地的數量豐富，但卻需雨林環境來生存。牠們主要的威脅是失去棲息地。而鄰近的[[拉明頓火山|拉明頓火山]]於1950年代爆發亦破壞了大片棲息地。由於牠們數量稀少，故在[[黑市|黑市]]價值很高。
