'''亞羅號事件'''（{{lang-en|'''The Arrow Incident'''}}）是引發[[第二次鴉片戰爭|第二次鴉片戰爭]]的關鍵事件，起源於1856年10月8日[[清朝|清朝]][[廣州|廣州]][[水師|水師]]在商船亞羅號（{{lang-en|Arrow}}）上進行的搜查及[[逮捕|逮捕]]海盗及嫌疑犯行動，以及後續[[大英帝國|英國]]的強硬抗議及武力恫嚇，相關衝突最終成為1857年英國對清用兵的理由。

== 背景 ==
1854年，英國政府援引中法《[[黃埔條約|黃埔條約]]》第三十五款：'''“双方若有應行更易章程條款之处，當就互換章程年月，核计满十二年之数，方可与中国再行筹议”'''以及中英《[[虎门條約|虎门條約]]》中的「[[最惠國待遇|最惠國待遇]]」條款，希望续十二年之約，以簽訂對英國更為有利的新約。見英國人要求修約，而美國和法國也要求修約。但清朝官員一味搪塞敷衍，互諉其事。見續約無望，三國公使憤懣。同時，[[商埠|通商口岸]]中國官民和「[[洋人|洋人]]」間的摩擦，也一直是清廷和[[西方世界|西方]][[大國|列強]]衝突的根源之一。

成為中英衝突導火線的亞羅號是由香港籍[[華人|華人]]蘇亞成所有，於1854年在中國內地製造的[[中国帆船|中西混合式帆船]] {{noteTag|name=英國議會|亞羅船是在1854年8月7日由蘇亞成製造，並且用一千元向香港政府購買執照，用於搬運米、豆等貨品（出自亞羅船水手的供詞）<ref name="孟引"></ref>}}，一日船隻被[[海盜|海盜]]劫去[[珠江|珠江]]，之後拾獲的民團交給廣州一家船公司<ref name="孟引">蔣孟引，《第二次鴉片戰爭》，北京：新知三聯書店（2009），p.34，ISBN 978-7-108-03284-3</ref>。後來賣給香港居民方亞明。1855年向[[港英政府|英屬香港政府]]登記並取得一年的[[執照|執照]]。除船長為[[愛爾蘭|愛爾蘭]]籍外，船上所有僱傭的[[水手|水手]]都是[[華裔|華裔]]<ref name="苦命天子"></ref>。

== 經過 ==
[[File:Chinese_officers_tear_down_the_British_flag_on_the_arrow.JPG|thumb]]

1856年10月8日早上八點<ref name="苦命天子"></ref> <ref name="清朝疆臣回憶錄,p.215">《清朝疆臣回憶錄》，第215頁</ref>，[[廣東水師|廣東水師]]接獲被劫商人舉報，懸掛[[英國國旗|英國國旗]]的亞羅號停泊在廣州[[黄埔区|黃埔]]<ref name="苦命天子">茅海建，《苦命天子 咸豐皇帝奕詝》，聯經出版事業股份有限公司，2008年，p.165，ISBN 9789570832600</ref>。巡河[[水師|水師]][[千總|千總]]梁國定登船搜索<ref>唐博，《清朝權臣回憶錄》，台北：遠流出版事業股份有限公司，初版：2010年9月1日，第212頁，ISBN 978-957-32-6702-7</ref>，當時船長正在另一條船上用早餐未做開航之準備。發現執照已過期12天，扣留了十二名涉嫌走私的中國[[水手|水手]]。但事後船長聲稱眼見英國國旗被撕毀的全部過程，清朝官方予以否認。對照英國的航海法律，任何一艘英國船駛入港口都必須卸下國旗。若當時升起國旗則表示水手準備私自潛逃（搜查那時船長不在船上），此說法的真實性已經起疑<ref name="苦命天子"></ref>。

英國驻廣州代理領事[[巴夏禮|巴夏禮]]向[[兩廣總督|兩廣總督]][[葉名琛|葉名琛]]抗議，要求放還水手及道歉，並要求移辦千總。事件後2天（10月10日）葉名琛答應放出9人，巴夏禮拒收。12月15日，廣州一帶中國居民見英軍退去，縱火焚燒「洋人」居室，使廣東十三行化為烏有。10月16日英公使[[寶寧|包令]]照會葉名琛：{{引用|如不速為彌補，自飭本國水師，將和約缺陷補足<ref>{{ja}} 佐佐木正哉，《{{lang|ja|鴉片戦争の中英抗争}}（資料篇稿）》，p.203</ref>。}}意即英國動武之意已十分明顯，也別有所求。10月21日巴夏禮遣人告訴葉名琛：「越日日中不如約，即攻城。」並要求24小時內答覆。葉名琛雖允諾放還全部12位水手，但以未扯下英國國旗拒絕道歉。

包令於是勒令香港駐華英軍駛入[[珠江口|珠江口]]，進逼廣州城前脅迫同意英人自由入城，至27日未見答覆便每隔五六分鐘砲轟一次總督衙門，葉不動聲色地在城內宣告：「殺一英人，賞銀三十元」。很快地英軍攻入城內，但未抓獲廣州總督<ref name="苦命天子,p.166">《苦命天子》，第166頁</ref>，不久退出，葉佯稱大勝。

== 影響 ==
翌年（1857年），[[英國|英國]]決定與[[法國|法國]]聯合出兵清朝，引發了[[英法聯軍之役|英法聯軍之役]]。
