'''《亞洲週刊》'''創刊於1987年12月13日，在[[香港|香港]]編輯發行。英文名稱為Yazhou Zhoukan，英文縮寫為YZZK。《亞洲週刊》由[[時代華納|時代華納]]屬下的Asiaweek（中文譯作“[[亞洲新聞|亞洲新聞]]”）創辦，創辦人是[[邁克爾·奧尼爾_(新西蘭傳媒工作者)|邁克爾·奧尼爾]]（Michael O'Neill）和[[康榮|康榮]]（Thomas Hon Wing Polin）。

《亞洲週刊》涵蓋的內容包括經濟、政治及社會文化等各個領域，廣獲全球華人關注。《亞洲週刊》總部位於[[香港|香港]][[柴灣|柴灣]][[嘉業街|嘉業街]]18號[[明報工業中心|明報工業中心]]，現任[[總編輯|總編輯]]為[[邱立本|邱立本]]。

==風格特色==
以“放之四海而皆準”的中文為要求和標準，為各地華人讀者構築文化橋樑.

== 歷史 ==
=== 時代集團時期 ===
[[邁克爾·奧尼爾_(新西蘭傳媒工作者)|邁克爾·奧尼爾]]是《亞洲週刊》的创始人。雖然他是[[新西蘭|新西蘭]]人，不懂中文，但是多年的亞洲工作經驗令他敏銳地意識到“亞洲時代將要來臨”。他不僅創辦了英文Asiaweek，還創辦了中文《亞洲週刊》。《亞洲週刊》現任總編輯[[邱立本|邱立本]]形容，奧尼爾是一位“富有專業精神和新聞激情的報人”。

《亞洲週刊》創刊的1987年，正值[[冷戰|冷戰]]進入尾聲，[[中國大陸|中國大陸]][[改革開放|改革開放]]經濟蓬勃發展；[[中華民國總統|中華民國總統]][[蔣經國|蔣經國]]撤銷[[台灣|台灣]][[黨禁|黨禁]]、[[報禁|報禁]]及開放赴中國大陸探親；[[中英香港談判|中英香港談判]]進展順利，[[香港|香港]]前途日趨明朗化；[[東南亞|東南亞]][[華人|華人]]隨著當地[[共產黨|共產黨]]的衰退，走出被歧視和被懷疑的陰影，重新恢復與神州大地的緊密關係。在這樣一個歷史背景下，奧尼爾和當時同在Asiaweek任職的[[康榮|康榮]]認為，亞洲華人讀者市場將會大有發展，於是決定創辦一本“打破國界”的[[中文|中文]]時事雜誌。

1987年12月13日，《亞洲週刊》正式創刊，發行商是“亞洲週刊有限公司”，創刊號上的主責編輯（相當於總編輯）是[[康榮|康榮]]，資深編輯是[[崔少明|崔少明]]和[[林和立|林和立]]，執行編輯是[[毛國倫|毛國倫]]與[[何良懋|何良懋]]，奧尼爾的名字不在其中。創刊號封面專題是“笑聲淚影回鄉行，隔別卅八載，兩岸初團圓”，封面圖片是隔絕了半個世紀的[[台海兩岸|台海兩岸]]親人在[[廈門|廈門]]會面時的悲喜交集。

《亞洲週刊》創辦之初，是[[Asiaweek|Asiaweek]]的姊妹刊物，風格模仿Asiaweek，封面以人物為主，大量使用新聞圖片，圖文並茂，設有“致讀者、中華天地、世界動態、怡情雅趣、大眾媒介、文化、男與女、現場報道、國際商務、會議室、經濟指標、貨幣市價、新知、人物剪影、電影、書籍與文學、健康生活、國際、旅遊、民眾之聲、觀點對照”等欄目，豎向排版，由後向前翻，香港零售價15港元。

《亞洲週刊》創刊時在每期首頁都印有自我介紹，稱“亞洲週刊有限公司為美國紐約時代集團（Time Inc.）的附屬公司”。1980年代末，《亞洲週刊》刊登的宣傳廣告自稱是：“接觸全球傑出華人的最佳媒介”、“第一本國際性中文時事週刊”、“是第一份跨地域報道世界時事的中文國際性刊物”。[http://www.yzzk.com/cfm/About.cfm]

1990年，香港出生、台灣[[國立政治大學|國立政治大學]]畢業的[[邱立本|邱立本]]到《亞洲週刊》應聘。他的名字首次出現在1990年11月25期的《亞洲週刊》上，職務是主筆，1993工作三年後，升為主責編輯。在此之前，《亞洲週刊》的主責編輯一直是[[康榮|康榮]]，但增加了助理主責編輯[[黎廷瑤|黎廷瑤]]，1990年首次出現華僑背景的資深編輯[[潘友來|潘友來]]及中國大陸背景的執行編輯[[曹景行|曹景行]]。

在1990年之前，《亞洲週刊》的編輯人員以香港本地精英為主，如[[康榮|康榮]]、[[崔少明|崔少明]]、[[林和立|林和立]]、[[何良懋|何良懋]]、[[黎廷瑤|黎廷瑤]]等。1990年以後，編輯隊伍開始逐漸走向多元化，除本地外，還有來自台灣、中國大陸、澳門以及東南亞等地的編輯，作者隊伍也日益多元化，在幾個主要國家和地區均設有特派員，特派員通常是本地華人，對當地情況非常了解。

1994年2月，即奧尼爾離開Asiaweek的那一年，[[明報集團|明報集團]]與[[時代華納|時代華納]]集團達成收購《亞洲週刊》協議，明報集團擁有《亞洲週刊》控制權。當時明報集團由香港商人[[于品海|于品海]]控股。1994年3月13日的《亞洲週刊》，宣布《亞洲週刊》的持有人變更為“明報雜誌有限公司”（明報集團的成員之一），總編輯是[[邱立本|邱立本]]，副總編輯是[[曹景行|曹景行]]，至此，《亞洲週刊》的股權由外資轉移到華資手上，而總部亦在1994年12月遷往柴灣嘉業街18號明報現址。

1995年10月，于品海由於被揭發早年在加拿大觸犯法律留有案底，被迫辭去明報集團董事長一職，並將手中明報集團股份出售予馬來西亞富商[[張曉卿|張曉卿]]擁有的[[常青集團|常青集團]]；張曉卿控股明報集團至今。

=== 邱立本主政時期 ===

1990年[[邱立本|邱立本]]從[[美國|美國]]回[[香港|香港]]加入[[中文|中文]]《亞洲週刊》，1993年成為[[總編輯|總編輯]]至今，開始了[[中文|中文]]《亞洲週刊》的全新時期。自1993年邱立本升任主責編輯以來，開始重視雜誌的評論，開設了時事評論專欄“筆鋒”（相當於社評），邱立本亦親自撰寫筆鋒文章，隨後又增加了“新思維”專欄，專供發表評論文章。《亞洲週刊》在原有欄目基礎上，進行了一些調整，形成了今天的欄目格局，增加了“畫中有話、封面專題、華人風雲、全球要聞、財經脈搏”等欄目，去除了“怡情雅趣、男與女、會議室、健康生活、民眾之聲”等欄目。整合後的欄目，更加突出“時事新聞週刊”的特點，同時也更加重視評論。

[[邱立本|邱立本]]在《亞洲週刊》的首要工作，是解決作為全球中文雜志面臨的[[語文|語文]]問題，即各地[[中文|中文]]因為[[方言|方言]]習慣、用詞遣句、文化[[歷史|歷史]]背景等因素造成交流溝通障礙甚至誤會，如：Kennedy在[[香港|香港]]和[[台灣|台灣]]叫「[[甘迺迪|-{甘迺迪}-]]」，在[[中國大陸|中國大陸]]則譯為「[[肯尼迪|-{肯尼迪}-]]」；North Korea在[[香港|香港]]和[[台灣|台灣]]譯為「[[北韓|北-{}-韓]]」，在[[中國大陸|中國大陸]]則譯為「[[北韓|朝-{}-鮮]]」；大陸說的“[[貓膩|貓膩]]”、“[[忽悠|忽悠]]”等，台港人士會不明白。《亞洲週刊》以“放之四海而皆準”的中文為要求和標準，在解決中文世界文字分歧上獲得突破，不但對其市場發展意義重大，而且也奠定了作為全球[[中文|中文]]刊物的地位。

《亞洲週刊》的報導立足亞洲，對亞洲各個國家和地區的報導比較平衡，除了台灣、中國大陸、香港以外，也報導亞洲其他地方發生的大事。近年有關中國大陸的關注焦點不斷增加，到2006年，《亞洲週刊》超過一半的封面專題是談論中國大陸，如果加上台灣和香港，這一年77%的雜誌封面專題集中在兩岸三地。

在《亞洲週刊》發展歷史上，特別值得一提的是1996年。這一年10月28日出版的《亞洲週刊》第43期，以封面文章報道《國民黨捐給白宮1500萬元？台灣捲入美政治獻金醜聞》，作者是《亞洲週刊》台灣編採中心資深編輯[[謝忠良|謝忠良]]和美國特約記者[[陳婉瑩|陳婉瑩]]。內文提及國民黨投資事業管理委員會主任委員[[劉泰英|劉泰英]]在1995年8月與前美國總統特別助理[[密道頓|密道頓]]見面時，曾提出願捐獻1500萬美元給[[克林頓|克林頓]]總統競選連任。報導刊出後，劉泰英認為妨害名譽提出控告，要求賠償1500萬美元。當時與劉泰英關係密切的[[國民黨主席|國民黨主席]][[李登輝|李登輝]]隨即以強硬措詞公開支持這一控告。官司於1995年12月7日首次開庭，這一官司在國際新聞界引起了強烈反響。美國“調查報道記者及編輯協會”（IRE）致函李登輝，抗議對《亞洲週刊》記者提出刑事誹謗，指出台灣政府支持這宗訴訟案對台灣帶來的名譽損失，將遠比有關報導造成的影響更大。總部設於紐約的[[保護記者委員會|保護記者委員會]]（CPJ）亦發表致李登輝的抗議信，譴責這一訴訟案違反聯合國[[世界人權宣言|世界人權宣言]]，要求台灣政府確保官方及國民黨的資源不會用來支持此案。美國《紐約時報》發表文章批評，有關訴訟是企圖以恫嚇來控制媒體，極不明智。數十位來自世界各地的新聞從業人員和學者，在互聯網上簽署致李登輝的聯名信，指責有關訴訟是對所有獨立報導的打擊，要求立即撤銷對兩位記者的控告。1997年4月22日，[[台北地方法院|台北地方法院]]經過4個多月的審理，認為媒體確從多位相關人士進行查證後才報導，即便報導與事實不符，也屬“對可受公評之事，而為適當評議”，宣判《亞洲週刊》三被告無罪，成為[[中華民國憲法|中華民國憲法]]上的重要[[判例|判例]]，也讓《亞洲週刊》名噪一時。

1998年，《亞洲週刊》網站成立，網址：www.yzzk.com。《亞洲週刊》每期有部分內容可以免費供讀者瀏覽，如果要閱覽全部內容，則需訂閱雜誌，訂戶可免費獲取密碼，瀏覽當期和過期雜誌。但在中國大陸，讀者無法登陸這一網站。（在2008年7月10日左右，北京奧運前夕，包括《亞洲週刊》在內的許多境外媒體網站被解封，中國大陸網民可以直接登錄。）

2001年3月，香港富商[[李嘉誠|李嘉誠]]兒子[[李澤楷|李澤楷]]旗下的Tom.com宣布以6000萬港元現金，收購《亞洲週刊》5成股份，Tom.com表示，希望進一步拓展《亞洲週刊》市場，特別是爭取增加中國大陸的訂閱量，目標是令《亞洲週刊》突破傳統雜誌範疇，成為一個提供全球新聞報導、商業信息科技研究以及公司評級等服務的權威多媒體中文平台。

然而，五年之後，Tom.com尚未達至上述目標，2006年3月，明報集團以發行新股方式，向Tom.com購回《亞洲週刊》五成股權，明報集團共發行1,200萬新股（相當於作價1,620萬港元）給TOM集團，明報集團持有《亞洲週刊》的全部股權。對於此次股份轉移，雙方都沒有透露原因。

2007年4月23日，[[張曉卿|張曉卿]]旗下的[[明報集團|明報集團]]與馬來西亞[[星洲集團|星洲集團]]、[[南洋商報集團|南洋商報集團]]簽署協議正式合併，對外統稱為[[明報集團|明報集團]]，《亞洲週刊》是新的明報集團的一員。2008年4月，香港的明報企業集團、馬來西亞的星洲媒體及南洋報業合併成[[世界華文媒體|世界華文媒體]]，《亞洲週刊》成為世界華文媒體的一員。

== 出版及發行 ==

《亞洲週刊》發行採取兩種方式，一種是訂閱，一種是報攤零售。香港市場零售價格每期為30港元，如果按年訂閱，可以根據訂閱期限的長短獲得相應折扣。《亞洲週刊》在香港設有發行部/訂戶服務部，同時亦委託專業發行公司開展市場推廣活動。[[台灣地區|台灣地區]]的零售總經銷是台灣英文雜誌社，在馬來西亞和新加坡也有發行代理。《亞洲週刊》被國泰航空、馬來西亞航空、新加坡航空、美國聯合航空、台灣中華航空、泰國航空指定為機艙刊物。

根據香港出版銷數公證會的統計數據，2006年，《亞洲週刊》銷量為7.3萬，香港本地3.4萬，香港以外地區3.9萬。據邱立本介紹，香港以外國家主要是台灣、新加坡和馬來西亞。中國大陸發行量約2000份左右。《亞洲週刊》採用正體字，2002年10月於新加坡、文萊及馬來西亞首次推出簡體字版本。

《亞洲週刊》的廣告語是“全球視野，本土情懷”。根據《亞洲週刊》網站公布的數據，《亞洲週刊》的讀者群大部分擁有高收入或較高的教育水平，約五成讀者更屬於最高管理階層及商界、金融界人士 。

== 定位及評價 ==

2004年《亞洲週刊》以《廣州媒體的冬天》為封面專題，獨家調查報導[[廣州|廣州]]《[[南方都市報|南方都市報]]》[[總編輯|總編輯]][[程益中|程益中]]、[[總經理|總經理]][[喻華峰|喻華峰]]被誣陷案，使[[程益中|程益中]]因此報道被釋放。《亞洲週刊》也因此報道獲得亞洲出版協會（SOPA）主辦的“亞洲傑出編輯獎”。

《亞洲週刊》力求“以華人熟悉的語言及價值觀報導和分析新聞”。《亞洲週刊》有一個欄目“歷史現場”，刊登大量的歷史圖片，特別是國共戰爭和抗日戰爭時期的歷史照片，以超越政治和黨派的視角來重新詮釋歷史。例如，《尋“匪諜”滄桑　探索國共和解　一個女共產黨員在台灣的身後事》，成為引人關注的話題。

《亞洲週刊》關注發生在各地的民眾衝突與維權示威，例如：中國大陸的孫志剛事件、太石村事件、汕尾徵地事件、山東盲人維權律師被拘事件、陝西民營石油股份糾紛、廈門市民反對設立海滄化工廠；澳門五一遊行衝突；香港七一遊行示威；台灣第一家族弊案引發的倒扁運動等等。每當中國大陸發生民眾衝突與維權示威等政治事件，《亞洲週刊》均會做細緻周到的報導。

《亞洲週刊》關注所有關乎[[新聞自由|新聞自由]]的議題和事件，例如，《南方都市報》喻華峰事件、《中國青年報》的《冰點週刊》事件、[[焦國標|焦國標]]撰文《討伐中宣部》事件、程翔間諜案等等。

雖然力求做到“客觀中立”，但基於發行的考慮，有關新加坡、馬來西亞的一些敏感議題，《亞洲週刊》的報導甚少。

在台灣問題上，《亞洲週刊》旗幟鮮明地主張統一，反對台獨。但對國內不同政治勢力，均有所報導。邱立本表示，“我們在統獨問題的考慮上，並非站在任何一個政黨或政府的立場，或是對於中國當局的逢迎，而是站在整個中華民族的立場上，我們認為這個違反了中華民族的利益與理想，所以我們提出這樣的看法。但我們也不同意當局採取疾風暴雨式的做法，而是和風細雨地解決統獨之爭，以溝通、交流來找到出路，而非靠武力解決問題。”{{cn}}台灣總統、副總統、[[立法院院長|立法院院長]]、[[行政院院長|行政院院長]]等政府首長與政治人物如[[馬英九|馬英九]]、[[王金平|王金平]]、[[蕭萬長|蕭萬長]]、[[呂秀蓮|呂秀蓮]]、[[施明德|施明德]]、[[吳敦義|吳敦義]]、[[邱毅|邱毅]]及歌手[[張惠妹|張惠妹]]等均接受過《亞洲週刊》的專訪。

《亞洲週刊》關注中華文化現象，從傳統文化到流行文化、從電影電視到音樂繪畫，每期均有介紹。2000年高行健獲諾貝爾文學獎，《亞洲週刊》刊登了《讓中國文學回歸中國人民》的封面專題，並將高行健評選為這一年的“年度人物”。《亞洲週刊》最早關注全球漢語熱，早在2002年，《亞洲週刊》就報道了韓國的學漢語熱潮。2007年，《亞洲週刊》又率先發現《中國風驚豔，台灣東南亞流行歌曲中國元素崛起》，中國大陸 “三國熱”和“論語熱”，也都成為《亞洲週刊》的關注內容。

《亞洲週刊》的財經報導並非重點，近年增加了財經報導以及財經評論，並且以幾項財經的排行榜為賣點，包括了銀行300大和全球華商1000等排行榜。

== 排行榜 ==

《亞洲週刊》每年均精心推出數個大型的財經排行專輯，為全球[[華商|華商]]提供宏觀視野的資訊平台和數量化的分析，主要排行榜有：
*亞洲銀行三百：以亞洲銀行上一年度的資產總規模為排名依據，同時提供資產增長排名、地區排名、盈利排名、純利增長排名、存貸比排名等。數據來源為各銀行（包括非上市銀行）公布的年度報表。排名範圍包括[[中國大陸|中國大陸]]、[[日本|日本]]、[[南韓|南韓]]、[[台灣|台灣]]、[[香港|香港]]、[[澳門|澳門]]、[[新加坡|新加坡]]、[[馬來西亞|馬來西亞]]、[[菲律賓|菲律賓]]、[[泰國|泰國]]、[[印度尼西亞|印度尼西亞]]、[[印度|印度]]、[[巴基斯坦|巴基斯坦]]等亞洲主要地區。每年6月初公布。
*國際華商五百：以[[中國大陸|中國大陸]]、[[香港|香港]]、[[澳門|澳門]]、[[台灣|台灣]]、[[新加坡|新加坡]]、[[馬來西亞|馬來西亞]]、[[泰國|泰國]]、[[印尼|印尼]]及[[菲律賓|菲律賓]]等地華商為主要股東的上市企業作為主要對象，依據當年6月30日收市的企業股價市值作出分析來排名，同時提供行業排名、資產增長排名、地區排名、盈利排名、純利增長排名等。數據來源為各公司公布的年度報表。排名範圍包括[[日本|日本]]、[[南韓|南韓]]、[[台灣|台灣]]、[[香港|香港]]、[[澳門|澳門]]、[[新加坡|新加坡]]、[[馬來西亞|馬來西亞]]、[[菲律賓|菲律賓]]、[[泰國|泰國]]、[[印度尼西亞|印度尼西亞]]、[[印度|印度]]、[[巴基斯坦|巴基斯坦]]等地區的[[華商|華商]]企業。每年9月底10月初公布。
*亞洲企業一千：以亞洲各國及地區企業（包括非上市企業）營業額為排名依據，同時提供行業排名、資產增長排名、地區排名、盈利排名、純利增長排名等。每年12月初公布。

== 主要欄目 ==
該刊闢有“封面專題”、“亞洲焦點”、“中華天地”、“歷史現場”、“人物剪影”、“財經脈搏”、“筆鋒”、“特別報道”、“文化藝術”、“資訊科技”等欄目。
*'''中華天地'''：重點介紹[[兩岸|兩岸]]三地的商業、經濟和社會發展。
*'''亞洲焦點'''：為[[華人|華人]]讀者提供亞洲各國的政治、經濟、社會民生的全方位報導。
*'''世界動態'''：突出報導國際經濟與政治的重要事件和發展變化，並跟進分析評論。
*'''文化藝術'''：為[[華人|華人]]讀者提供多個地區的文化指標，包括流行歌曲及暢銷書榜，還有新書推介、影評等，以[[華人|華人]]的角度，展示文化、消閑娛樂、運動、科學及健康等資訊。

== 訂閱 ==

《亞洲週刊》可於世界各地訂閱。2008年以前，在中國大陸，按照規定，《亞洲週刊》只限境外人士、外資企業及相關機構預訂，市面禁售。中國官方指定代訂境外報刊的「[[中國圖書進出口總公司|中國圖書進出口總公司]]」及「[[中國國際圖書貿易總公司|中國國際圖書貿易總公司]]」於2007年12月底表示，收到上級指示，2008年起不接受訂閱《亞洲週刊》，並且給訂戶退款，兩公司至今未解釋原因，並指這是全國統一的規定[https://archive.is/20130503092146/http://www.rfa.org/cantonese/xinwen/2007/12/28/china_media_freedom/?simple=1]。
