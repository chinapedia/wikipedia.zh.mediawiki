《'''咱们工人有力量'''》，[[马可|马可]]词曲。这首歌1948年诞生于[[佳木斯|佳木斯]]铁工厂，同年于[[沈阳|沈阳]][[皇姑屯机车车辆厂|皇姑屯机车车辆厂]]定稿，最后的定稿首发在《[[辽宁日报|辽宁日报]]》的前身《[[东北日报|东北日报]]》上。它运用了[[东北地区|东北地区]]民间[[秧歌调|秧歌调]]与[[号子|号子]]的节奏。<ref name="Bunny_fushun.nen"/>

== 历史 ==
1945年5月，[[马可|马可]]所在的[[东北鲁艺文工二团|东北鲁艺文工二团]]从农村回到[[佳木斯市|佳木斯市]]（当时[[合江省|合江省]]省会）到佳木斯铁工厂中深入生活，马可及团员为工人表演节目及教工人们唱歌，鉴于当时描写农民翻身的歌曲很多，却很少有专门写给工人的歌，使得马可决定创作该歌曲。马可在佳木斯铁工厂完成了这首歌的初稿，团员们当时就拿着歌谱到[[佳木斯发电厂|佳木斯发电厂]]和[[东北造币厂|东北造币厂]]等工厂教唱歌曲并听取工人对该曲的意见，马可在沈阳皇姑屯机车车辆厂教唱该曲后，做了最后一次修改，把“为什么？为了打老蒋！ ”改为“为什么？为了求解放！ ”<ref name="Bunny_fushun.nen">{{Cite web | url= http://fushun.nen.com.cn/74874736101818368/20110411/2441445.shtml |title= 《咱们工人有力量》在沈阳定稿 |author= 张阿春 |date= 2011-04-11 |publisher= 抚顺新闻网 |language= zh-hans |accessdate=2011-6-12导入 |quote= }}</ref>
== 词曲版权 ==
1976年7月27日，该曲词曲作者马可逝世。
== 使用及改编 ==
