'''铬酸钠'''（化学式：[[钠|Na]]<sub>2</sub>[[铬|Cr]][[氧|O]]<sub>4</sub>）是一种黄色固体，是[[铬酸|铬酸]]形成的[[钠|钠]]盐。铬酸钠用作石油工业中的腐蚀抑制剂、织物的染料助剂及木材防腐剂，<ref>{{cite web |url=http://www.pesticideinfo.org/Detail_ChemUse.jsp?Rec_Id=PC34364#ChemID |title=Sodium chromate - Pesticide use statistics for 2005 |accessdate=2008-06-20 |publisher=PAN Pesticides Database}}</ref>在医药学中也有应用。<ref>{{cite web |url=http://dailymed.nlm.nih.gov/dailymed/drugInfo.cfm?id=6525 |title=chromitope sodium (Sodium Chromate, Cr 51) injection, solution |accessdate=2008-06-20 |author=Bracco Diagnostics Inc. |publisher=DailyMed}}</ref>它可由[[重铬酸钠|重铬酸钠]]与[[氢氧化钠|氢氧化钠]]反应制备，具[[潮解|潮解]]性，可生成四水、六水和十[[水合物|水合物]]。

铬酸钠中的[[六价铬|铬为六价]]，具[[致癌性|致癌性]]。

水溶液中它与[[重铬酸钠|重铬酸钠]]存在平衡，加酸有利于红色重铬酸钠的生成：

:: 2 CrO<sub>4</sub><sup>2−</sup> + 2 H<sub>3</sub>O<sup>+</sup> ⇌ Cr<sub>2</sub>O<sub>7</sub><sup>2−</sup> + 3 H<sub>2</sub>O

铬酸钠有强[[氧化性|氧化性]]，溶于水<ref name="inchem">{{cite web |url=http://www.inchem.org/documents/icsc/icsc/eics1370.htm |title=SODIUM CHROMATE |accessdate=2008-06-20 |publisher=inchem}}</ref>且溶液呈微碱性。<ref>{{GESTIS|ZVG=4030|Date=2008-06-20}}</ref>
