'''寓·弍捌'''（英文：{{lang|en|'''Residence 228'''}}），是位於[[香港|香港]][[深水埗|深水埗]]的單幢住宅，樓高共26層，由[[華懋集團|華懋集團]]（以''強健國際有限公司''名義）發展，[[合安管理|合安管理]]管理，於2012年8月入伙。

== 歷史 ==
寓·弍捌 前身為位於「福榮街228-232A、234號」3幢6層高的[[唐樓|唐樓]]<ref>[http://gesu6133.pixnet.net/blog/post/30252182-%E9%A6%99%E6%B8%AF%E5%94%90%E6%A8%93%E6%8A%95%E8%B3%87-%E2%80%93-%E5%90%84%E5%8D%80%E6%94%B6%E8%B3%BC%E9%87%8D%E5%BB%BA%E7%9A%84%E6%83%85%E6%B3%81(%E6%96%B0%E4%BF%AE%E8%A8%82 香港唐樓投資 – 各區收購重建的情況]{{Dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=no }}</ref><ref>[http://www.unicorn-property.com.hk/articleDetail.php?searchnum=574 深水埗具重建潛力]</ref>，2006年12月[[華懋集團|華懋集團]]成功以1.4億[[港元|港元]]收購「福榮街228-232A號」2幢唐樓，地盤面積為6083方呎，及後至2009年[[華懋集團|華懋集團]]再透過[[田生集團|田生集團]]成功以2千萬收購「福榮街234號」。

最終，[[華懋集團|華懋集團]]合併兩地，發展成單幢商住項目 －寓·弍捌，並於2011年5月開售[[樓花|樓花]]，而[[示範單位|示範單位]]設於[[荃灣|荃灣]][[如心廣場|如心廣場]]5樓。

== 簡介 ==
物業實際樓高26層，惟由於不設4、14及24字樓，並設有天台，故有28字樓，其中基座佔2層，地下及一樓設有2個商鋪，而二樓則為[[會所|會所]]，其設施包括[[游泳池|游泳池]]、[[桑拿|桑拿]]房等，供住戶享用。

其餘23層為住宅單位 ，共設88個單位，建築面積由約450-2,000呎不等，實用率約73%，其中4伙為特色戶，而一梯有三伙或四伙。

== 商鋪 ==
基座地下及一樓為商鋪，地舖面積約4,216方呎，而1樓面積約9,613方呎，合共約1.38萬方呎，發展商於2014年2月委託[[中原地產|中原地產]]放售，意向價約2.28億元，其中若只購買1樓，意向價約1.44億元；同時接受洽租，意向每月租金約60萬元，若只租1樓則意向月租約38.5萬元<ref>[http://orientaldaily.on.cc/cnt/finance/20140227/00204_013.html 華懋寓‧弍捌舖嗌2.2億]</ref>。
