'''岳屏广场'''位于湖南衡阳[[岳屏公园|岳屏公园]]对面，于2002年6月1日开放<ref name=":0">{{Cite book|title=衡阳市志：1978-2008|last=|first=|publisher=方志出版社|year=2008年|isbn=9787514412246|location=|page=1704-1705|pages=|chapter=第二十编 城乡建设与环境保护}}</ref>。

== 位置与面积 ==
岳屏广场地处衡阳市先峰路，占地面积约6.1万平方米，其中绿化面积1.7万平方米<ref name=":0" />。

== 建设过程 ==
岳屏广场是在拆除原建材市场的基础上新建的，2002年2月7日开工、5月25日竣工<ref name=":0" />。

== 植物 ==
岳屏广场现在种植着金叶[[女贞|女贞]]、[[紅花檵木|红桎木球]]、[[月季|月季花]]、洒金柏树、红叶[[石楠属|石楠]]、[[唐菖蒲|剑兰]]、[[黄杨科|瓜子黄杨]]、[[细叶忍冬|细叶麦冬]]、[[杜鵑花屬|杜鹃]]、[[鸢尾|鸢尾]]、[[八角金盤|八角金盘]]等植物<ref>{{Cite news|url=http://news.focus.cn/hengyang/2014-11-12/5748805.html|title=岳屏广场启动绿化提质改造|last=|first=|date=2014年11月12日|work=搜狐焦点网|accessdate=}}</ref>。

== 景观 ==
岳屏广场由同济大学城市规划研究院、衡阳市规划设计院设计，广场总平面呈“凤凰”形布置，寓意衡阳市的发展如凤凰展翅，扶摇直上<ref name=":0" />。
