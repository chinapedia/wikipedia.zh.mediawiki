'''出羽重遠'''（{{lang|ja|出羽 重遠 でわ しげとお}}，[[大日本帝国海军|大日本帝国海军]]军人、[[華族_(日本)|華族]]，1856年1月17日（[[安政|安政]]2年12月10日）出生于[[陸奧國|陸奧國]][[會津藩|會津藩]]，1930年（[[昭和|昭和]]5年）1月27日逝世。最终军阶为海军大将。受封[[正二位|正二位]]、{{link-ja|勋等 (日本)|勲等|勋一等}}、[[金鵄勳章|功二级]]、[[男爵|男爵]]。

== 简历 ==
出羽其父为[[陸奧國|陸奧國]][[会津若松城|会津若松城]]的出羽佐太郎重信<ref>会津藩士人名録</ref>。出羽幼名房吉。其妻为会津藩士大沼亲诚之女；妻妹为[[铃木贯太郎|铃木贯太郎]]之妻阿丰<ref>半藤一利 51頁；星亮一 123-124頁</ref>；其妻弟为{{link-ja|大沼龙太郎|大沼龍太郎 (海軍軍人)}}海军轮机少将（“海軍機関少将”）。长子重夫为海军兵学校34期生，以海军大尉战死；次子泰邦为[[加藤定吉|加藤定吉]]的养嗣子；四子重芳为贵族院议员。重夫之女后来成为{{link-ja|寺垣猪三|寺垣猪三}}海军中将的养女<ref>『人事興信録』（1928年）</ref>。

[[戊辰戰爭|戊辰戰爭]]期间出羽加入[[白虎隊|白虎隊]]参加了战斗。战后其父以[[會津松平家|會津松平家]]的御用挂（日语“{{lang|ja|御用掛}}”，为皇室担当勤务者）身份上京，出羽亦随其父同行，后师从{{link-ja|秋月悌次郎|秋月悌次郎}}等人，进行学习。

1878年（明治11年）8月16日，出羽以[[海軍兵學校|海軍兵學寮]][[海军兵学校毕业生列表_(日本)|5期]]毕业；同期生中著名人物有[[三須宗太郎|三須宗太郎]]（海军大将）。出羽曾奉命任[[防护巡洋舰|防护巡洋舰]][[浪速号防护巡洋舰|浪速]]的返航委员赴英国处理接收事宜，并在返航时担任航海长一职。其后出羽又历任防护巡洋舰[[高千穂号防护巡洋舰|高千穂]]的分队长、[[无防护巡洋舰|无防护巡洋舰]]{{link-ja|高雄号无防护巡洋舰|高雄 (巡洋艦)|高雄}}的副舰长、炮舰{{link-ja|赤城号炮舰|赤城 (砲艦)|赤城}}的舰长等职，拥有着非常丰富的一线部队经验；其后又出任[[海军省|海军省]]第一局第一课长（后军务局第一课长）、大臣官房人事课长等海军行政职务。
[[File:LS75Z1.jpg|thumb]]、[[上村彦之丞|上村彦之丞]]、[[东乡平八郎|东乡平八郎]]、[[山本權兵衛|山本權兵衛]]、[[伊东祐亨|伊东祐亨]]、[[片冈七郎|片冈七郎]]、'''出羽重遠'''、[[齋藤實|齋藤實]]、{{link-ja|山下源太郎|山下源太郎}}；中列左起第6人为[[加藤友三郎|加藤友三郎]]]]

中日[[甲午战争|甲午战争]]爆发时，出羽正任[[常備艦隊|西海舰队]]参谋长。1894年12月27日，出羽晋升为海军大佐，同时调任为[[联合舰队|联合舰队]]参谋长，辅佐联合舰队司令长官[[伊东祐亨|伊东祐亨]]。甲午战争后再度出任军务局第一课长、军务局军事课长等职。1898年，出羽担任时任[[海軍大臣_(日本)|海軍大臣]]的[[山本權兵衛|山本權兵衛]]的次官。1900年，出羽由[[齋藤實|齋藤實]]推荐，赴英国担任[[装甲巡洋舰|装甲巡洋舰]][[常磐號裝甲巡洋艦|常磐]]的返航委员长<ref>松下芳男，263頁</ref>。1900年5月20日，出羽晋升为海军少将，同时叙任常备舰队司令长官；后又任军务局长兼[[軍令部_(日本海軍)|軍令部]]次长。

[[日俄战争|日俄战争]]爆发时出羽为第三战队司令。其率领着4艘巡洋舰，在第一舰队旗下，与第一战队共同参加了[[对马海峡海战|对马海峡海战]]（日方称为日本海海战）。1905年日本方面编组成{{link-ja|第四艦隊 (日本海軍)|第四艦隊 (日本海軍)|第四舰队}}，出羽担任司令长官一职，指挥了[[库页岛之战|库页岛登陆作战]]。日俄战争后出羽又先后任第二舰队司令长官、[[佐世保镇守府|佐世保镇守府]]司令长官、教育本部长等职，1912年年7月9日在第一舰队司令长官任内晋升为海军大将。在此之前日本的海军大将中除了皇族[[有栖川宮威仁親王|有栖川宮威仁親王]]外其余13人全部为旧[[薩摩藩|薩摩藩]]出身，而出羽不但是萨摩藩以外第一人、还更是来自于所谓“贼藩”的会津藩，其升任海军大将一事颇为轰动，媒体多有报道。1913年12月1日，出羽担任[[軍事參議院_(日本)|军事参议官]]。1914年1月28日，任[[西门子事件|西门子事件]]查问委员长。

晚年的出羽为[[會津戰爭|會津戰爭]]中的死难者建立纪念碑而奔走<ref>Yamakawa, ''Hoshū Aizu Byakkotai Jūkyūshi-den'', p. 64.</ref>。逝世后，葬于东京[[青山靈園|青山靈園]]。

== 履历 ==
1872年（明治5年）10月3日，就读[[海軍兵學校|海軍兵學寮]]。

1878年8月16日，以海軍兵學寮[[海军兵学校毕业生列表_(日本)|5期]]毕业，在43人中排名第6。任海军少尉补。

1880年8月12日，获海军少尉衔。同年12月4日，在护卫舰{{link-ja|龍驤号护卫舰|龍驤 (コルベット)|龍驤}}上服役。

1883年2月27日，晋升为海军中尉。同年10月4日，在护卫舰{{link-ja|天城号护卫舰|天城 (スループ)|天城}}上服役。

1885年4月27日，奉命任[[防护巡洋舰|防护巡洋舰]][[浪速号防护巡洋舰|浪速]]的返航委员，赴英国处理接收事宜，并在11月20日担任浪速航海长一职。

1886年4月7日，晋升为海军大尉。6月26日返抵日本。10月15日，任防护巡洋舰[[高千穂号防护巡洋舰|高千穂]]的分队长。

1887年10月27日，任[[常備艦隊|常备小舰队]]司令官参谋。

1889年7月29日，任无防护巡洋舰{{link-ja|高雄号无防护巡洋舰|高雄 (巡洋艦)|高雄}}的副舰长<ref>『官報』第1830号「叙任及辞令」1889年8月5日</ref>。

1890年10月16日，晋升为海军少佐。

1891年6月17日，任海军省第一局第一课长。

1893年5月20日，任海军省大臣官房人事课长。同年9月12日，任炮舰赤城舰长。

1894年5月5日，任通报舰{{link-ja|龙田号通报舰|龍田 (通報艦)|龙田}}返航委员长<ref>『官報』第3253号「叙任及辞令」1894年5月7日</ref>。6月5日任龙田舰长，并启程前往英国<ref>『官報』第3279号「叙任及辞令」1894年6月6日</ref>。12月7日，晋升为海军大佐，任西海舰队参谋长。12月17日，任常备舰队参谋长及联合舰队参谋长。

1895年7月25日，任海军省军务局第一课长。

1896年5月27日，兼任临时建筑部部员。

1897年4月1日，任海军省军务局军事课长<ref>『官報』第4122号「叙任及辞令」1897年4月2日</ref>。6月7日，获赐一级俸<ref>『官報』第4178号「叙任及辞令」1897年6月8日</ref>。

1898年4月5日，任装甲巡洋舰常磐的返航委员长<ref>『官報』第4426号「辞令」1898年4月6日</ref>，准备赴英国进行接收。10月3日，任常磐舰长。

1899年7月16日，返抵日本。

1900年5月20日，晋升为海军少将，任常备舰队司令。

1901年7月3日，任[[橫須賀鎮守府|橫須賀鎮守府]]舰政部长。

1902年10月29日，任海军省军务局长，兼军令部次长、海军将官会议议员。

1903年10月27日，再度出任常备舰队司令。12月28日，任第一舰队司令。

1904年6月6日，晋升为海军中将。

1905年6月14日，任第四舰队司令长官。12月20日，任第二舰队司令长官。

1906年9月13日，作为[[伏見宮博恭王|伏見宮博恭王]]的随员赴中国<ref>『官報』第6965号「叙任及辞令」1906年9月14日</ref>。11月22日，任海军教育本部长、将官会议议员。

1907年7月20日，任练习舰队特命检阅使<ref>『官報』第7218号「叙任及辞令」1907年7月22日</ref>。

1908年5月26日，任第二舰队司令长官。

1909年2月1日，任佐世保镇守府司令长官。

1911年12月1日，任第一舰队司令长官。
[[File:Dewa_Shigetō_and_Kobayashi_Seizō_2.jpg|right]]]]

1912年7月9日，晋升为海军大将。

1913年12月1日，任军事参议官。<ref>『官報』第404号「叙任及辞令」1913年12月2日</ref>

1914年1月28日，“[[西门子事件|西门子事件]]”查问委员长。3月20日，任特命检阅使<ref>『官報』第492号「叙任及辞令」1914年3月23日</ref> 。

1915年1月7日，作为日本政府代表之一参加[[巴拿马运河|巴拿马运河]]开通仪式，以及巴拿马太平洋万国博览会。小林躋造海军中佐也在随行人员之列。<ref>『官報』第728号「叙任及辞令」1915年1月8日</ref>

1916年9月6日，任特命检阅使。<ref>『官報』第1232号「叙任及辞令」1916年9月7日</ref>

1919年3月21日，任特命检阅使。<ref>『官報』第1989号「叙任及辞令」1919年3月24日</ref>

1920年12月17日，转为预备役。

1925年12月17日，退役。
