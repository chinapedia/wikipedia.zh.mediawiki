《'''出棋致勝'''》（{{lang-en|'''''Pawn Sacrifice'''''}}）是一部於2014年上映的[[美國|美國]][[傳記片|傳記片]]，為[[愛德華·茲維克|愛德華·茲維克]]執導，[[史蒂芬·奈特|史蒂芬·奈特]]撰寫劇本。由[[陶比·麥奎爾|陶比·麥奎爾]]、[[李佛·薛伯|李佛·薛伯]]、{{link-en|莉莉·拉貝|Lily Rabe}}、[[彼得·賽斯嘉|彼得·賽斯嘉]]與[[麥可·斯圖巴|麥可·斯圖巴]]主演。電影於2014年9月11日在[[多倫多國際電影節|多倫多國際電影節]]首映，在[[美國|美國]]則於2015年9月16日上映<ref name="deadlineill">{{cite web|last=Pedersen |first=Erik |url=http://deadline.com/2015/02/ill-see-you-in-my-dreams-release-date-pawn-sacrifice-1201366262/ |title=‘I’ll See You In My Dreams’ and ‘Pawn Sacrifice’ Get Release Dates |publisher=Deadline |date=2015-02-04 |accessdate=2015-05-30}}</ref>。

== 劇情 ==
故事背景設定於[[冷戰時期|冷戰時期]]，敘述美國西洋棋冠軍[[鮑比·費雪|鮑比·費雪]]（[[陶比·麥奎爾|陶比·麥奎爾]] 飾）將挑戰蘇聯帝國，而展現出他天才和瘋狂...。

== 製作 ==
電影於2013年10月初在[[冰島|冰島]]的[[雷克雅維克|雷克雅維克]]開始前期製作<ref>{{cite web|last=Siegel |first=Tatiana |url=http://www.hollywoodreporter.com/news/boardwalk-empires-michael-stuhlbarg-joins-650443 |title='Boardwalk Empire's' Michael Stuhlbarg Joins Bobby Fischer Chess Pic 'Pawn Sacrifice' (Exclusive) |publisher=Hollywood Reporter |date=2013-10-23 |accessdate=2015-05-30}}</ref>。10月中，在[[加拿大|加拿大]]的[[蒙特婁|蒙特婁]]開始了41天的拍攝，最後於同年12月11日在[[洛杉磯|洛杉磯]]結束<ref>{{cite web|last=Kay |first=Jeremy |url=http://www.screendaily.com/news/production/shooting-underway-on-pawn-sacrifice/5062849.article?referrer=RSS |title=Shooting underway on Pawn Sacrifice |publisher=Screendaily.com |date=2013-10-23 |accessdate=2015-05-30}}</ref>。

== 發行 ==
2014年9月11日，《出棋致勝》首映於[[多倫多國際電影節|多倫多國際電影節]]<ref>{{cite web |url=http://variety.com/2014/film/news/tiff-toronto-intl-film-festival-gala-special-presentations-1201266480/ |title=Toronto Film Festival Lineup Includes Denzel Washington’s ‘Equalizer,’ Kate Winslet’s ‘A Little Chaos’ |accessdate=2014-07-22 |work=Variety}}</ref>。2014年9月10日，{{link-en|Bleecker Street (公司)|Bleecker Street (company)|Bleecker Street}}收購了其在美國的發行權，同時也是該公司首次收購其他電影。本片原定在美國於2015年9月18日上映，但後來推前至9月16日<ref name="deadlineill"/>。

=== 評價 ===
《出棋致勝》獲得了普遍好評，影評人稱讚陶比·麥奎爾德演出和西洋棋天才鮑比·費雪的塑造。[[爛番茄|爛番茄]]基於77條專業影評持有73%的新鮮度<ref>{{cite web |url=http://www.rottentomatoes.com/m/pawn_sacrifice/ |title=Pawn Sacrifice (2015) |publisher=Rotten Tomatoes |accessdate=2015-10-02}}</ref>，[[Metacritic|Metacritic]]得分65，正評居多<ref>{{cite web |url=http://www.metacritic.com/movie/pawn-sacrifice |title=Pawn Sacrifice reviews |publisher=Metacritic |accessdate=2015-09-28}}</ref>。
