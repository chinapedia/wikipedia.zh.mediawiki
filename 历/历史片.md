'''历史片'''（{{lang-en|'''Historical fiction'''}}）是[[电影|电影]][[片种|片种]]之一。“历史”是文学中常见的，在历史片里，它包括“历史”和“历史上的人物”，包括当时的[[政治|政治]]、[[经济|经济]]、[[文化|文化]]、[[社会|社会]][[风俗|风俗]]，历史片必须关注“细节”，如果稍有误差，就会出现“[[穿帮|穿帮]]”镜头。<ref name="britannica">{{cite web|url=http://www.britannica.com/search?query=historical+fiction&ct= |title=Search - Encyclopedia Britannica |publisher=Britannica.com |date= |accessdate=2013-10-12}}</ref>

在[[大中华地区|大中华地区]]的历史片里，常常出现的穿帮是：[[明朝|明朝]]才有的[[线装书|线装书]]出现在明朝以前的朝代<ref>{{cite web |url=http://www.chinanews.com/yl/2014/02-26/5886675.shtml |title=范冰冰版《武则天》出现线装书 被批无常识  |accessdate=2014 |publisher=中新网 |quote =线装也称古线装。我国古代的纸本书，经历了卷轴和册页两个阶段。卷轴由卷、轴、缥、带组装成。汉、唐代只有这种卷轴形式的书。如今挂在墙上的轴画、书法，仍是卷轴装的遗风。晚唐以后，卷轴书向册页书过渡，其装订方法又有多种多样的演变，到了明代才正式出现了线装本的册页书。 |work= |language=zh-hans}}</ref>；“[[奉天承运皇帝诏曰|奉天承运皇帝，诏曰]]”也往往读成“奉天承运，皇帝诏曰”，而且出现在明朝[[朱元璋|朱元璋]]之前<ref>{{cite web |url=http://cul.sohu.com/20130906/n386012091.shtml |title=圣旨的读法：“奉天承运皇帝诏曰”被乱断句 |accessdate=2013 |publisher=搜狐网 |quote =最早使用“奉天承运皇帝诏曰”八个字作为圣旨开头语的，是明太祖朱元璋。也就是说，“奉天承运皇帝诏曰”八个字出现于圣旨中，始于明朝。“奉天承运皇帝诏曰”这八个字并不是时下影视作品中“奉天承运，皇帝诏曰”的读法，其正确的断句应该是“奉天承运皇帝，诏曰”。 |work= |language=zh-hans}}</ref>。服飾、禮儀與背景不符等。同时，这也是大中华地区历史[[电视剧|电视剧]]的常见错误。

== 概念 ==
“[[历史|历史]]”就是过去了的事情，历史片就是讲述过去了的事情的电影。<ref>{{cite book|last=Adamson|first=Lynda G.|title=World Historical Fiction|year=1999|publisher=Orxy Press|location=Phoenix, AZ|isbn=9781573560665|page=xi}}</ref>

历史片往往选取重大历史事件、历史题材、历史人物，它既呈现当时的真实情况，也允许虚构情节，但是“历史进程”必须尊重[[自然法则|自然法则]]，即过去发生了的朝代更替不能篡改，皇帝轮替也不能乱改等。<ref>{{cite web|title=Historical Fiction|url=http://www.readwritethink.org/files/resources/lesson_images/lesson404/HistoricalFictionDefn.pdf|work=ReadWriteThink|publisher=[[International_Reading_Association|International Reading Association]]|accessdate=10 January 2014}}</ref>

历史片往往根据史书来拍摄，有时候，人物名字或者事件地点会有所更换。艺术允许虚构，只要它不偏离正轨，自己建立“历史”，如果出现了自己建立或书写历史这样的情况，例如“[[韩国|韩国]]赢得[[南北战争|美国内战]]”，那就是荒诞不羁。

== 类型 ==
{{Main|史诗片}}
历史片下辖一个重要的类别就是“史诗片”，史诗片用全景拍摄、全程外景拍摄等方式，花费巨额资金，服装、器物等都是古代复制品，华丽的古代服饰、当年的语言环境、当时的风俗都是史诗片的重要细节。
