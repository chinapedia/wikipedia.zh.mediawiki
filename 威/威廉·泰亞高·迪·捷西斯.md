'''威廉·泰亞高·迪·捷西斯'''（'''Willian Thiego de Jesus'''，{{bd|1986年|7月22日|2016年|11月29日|catidx=Willian Thiego de Jesus}}），生于巴西[[塞爾希培州|塞爾希培州]]首府[[阿拉卡茹|阿拉卡茹]]，足球運動員，2016年11月28日時、於[[拉米亞航空2933號班機空難|拉米亞航空2933號班機空難]]罹難。

== 生涯 ==

=== 塞爾希培 ===

泰亞高的職業位置是後衛，從他的家鄉球隊塞爾希培出道，被甘美奧球探看中。

== 甘美奧 ==

泰亞高在2006年加盟甘美奧青年隊。2007年，他晉升至頂級隊，球隊該年贏得聯賽和盃賽冠軍。

2008年6月14日，他射入了他在聯賽的第一個入球，助球隊3-0擊敗戈亞斯。

== 逝世 ==
{{主條目|拉米亞航空2933號班機空難}}
泰亞高於哥倫比亞時間2016年11月28日登上由[[玻利維亞|玻利維亞]][[拉米亞航空|拉米亞航空]]出發，原定在位于[[哥倫比亞|哥倫比亞]][[麥德林|麥德林]]的[[何塞·瑪麗亞·科爾多瓦國際機場|何塞·瑪麗亞·科爾多瓦國際機場]]降落的航班，出戰[[2016年南美球會盃|2016年南美球會盃]]第一局總決賽，對手為哥伦比亚[[國民體育會_(哥倫比亞)|國民體育會]]，賽前便因空難逝世。
