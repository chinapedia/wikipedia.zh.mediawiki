'''威孚房开杯棋王争霸赛'''，一般简称为威孚房开杯，是由[[中国围棋协会|中国围棋协会]]、江苏[[无锡|无锡]]市政府共同主办的一项中国国内[[围棋|围棋]]比赛。
== 参赛棋手及比赛性质 ==
前三届参赛棋手全部为主办方特邀，第四届起，参赛棋手为中国国内围棋等级分前30名及主办方特邀2人共32人。
自2008年（第四届）起，该项赛事正式成为国家级正式比赛，纳入中国围棋棋手等级分计算。

== 比赛规则 ==
采用中国围棋规则，黑贴3又3/4子；比赛的用时为每方一小时，之后一次一分钟读秒。。

== 奖金 ==
第一届：冠军3万元人民币，亚军8000元人民币

第二届：冠军2万元人民币，亚军1万元人民币

第三、四届：冠军5万元人民币，亚军3万元人民币

第五－九届：冠军10万元人民币，亚军5万元人民币

第十－十一届：冠军15万元人民币，亚军6万元人民币

== 第一届 ==
首届威孚房开杯中国围棋棋王争霸赛于2003年10月3日~5日在[[太湖|太湖]]之滨无锡市进行。本次比赛是由中国围棋协会、[[无锡日报|无锡日报]]、无锡威孚房屋开发有限公司主办，北京弈友围棋文化传播公司承办，无锡威孚汽贸有限公司、[[无锡太极实业股份有限公司|无锡太极实业股份有限公司]]、无锡国联投资公司、无锡市人民大会堂、无锡电视台文体娱乐频道等协办，[[新浪网|新浪网]]支持，作为首届[[太湖博览会|太湖博览会]]上的一个亮点。

本届比赛邀请了[[陈祖德|陈祖德]]九段、聂卫平九段、马晓春九段、俞斌九段、[[常昊|常昊]]九段、周鹤洋九段、[[古力|古力]]七段、[[王煜辉|王煜辉]]七段8名著名棋手参赛，[[金同实|金同实]]任裁判长，特邀[[王汝南|王汝南]]八段、[[华以刚|华以刚]]八段、[[王元|王元]]八段作为讲棋嘉宾。

赛事具体日程安排：

　　1、10月3日下午4点：首届“威孚房开杯”中国围棋棋王争霸赛新闻发布会，地点：  无锡大饭店

　　2、10月3日下午6点：欢迎酒会，地点：无锡大饭店

　　3、10月4日上午9点半：首届“威孚房开杯”中国围棋棋王争霸赛开幕式，地点：无锡人民大会堂中厅

　　4、10月4日上午10点：第一轮比赛，地点：无锡人民大会堂湖滨厅，讲棋：大会堂中厅

　　5、10月4日下午2点半：参赛棋手与无锡本地棋手一对十指导活动，共100局，地点：无锡人民大会堂三楼中庭。

　　6、10月5日上午10点：第二轮比赛，地点：无锡人民大会堂湖滨厅，讲棋：大会堂中厅

　　7、10月5日上午2点半：第三轮比赛，地点：无锡人民大会堂湖滨厅，讲棋：大会堂中厅

　　8、10月5日下午5点：闭幕及发奖仪式，地点，无锡人民大会堂中厅

　　9、10月6日上午9点：棋手和无锡市领导及企业家联谊活动，地点：无锡大饭店

“[[棋圣|棋圣]]”聂卫平九段在决赛中执黑中盘战胜俞斌九段，继1990年在无锡夺得马山杯八强争霸赛后再次在无锡“封王”，并获得冠军奖金3万元，俞斌获得亚军8000元。
