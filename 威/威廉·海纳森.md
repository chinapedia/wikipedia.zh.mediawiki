'''安德里亚斯·威廉·海纳森'''（法罗语：Andreas William Heinesen，1900年2月15日-1991年3月12日）[[法罗群岛|法罗群岛]][[诗人|诗人]]、[[小说家|小说家]]、[[儿童文学|儿童文学]]家、[[作曲家|作曲家]]、[[画家|画家]]。<ref>http://www.goodreads.com/author/show/305752.William_Heinesen Goodreads.com</ref>

== 生平简介 ==

1900年，海纳森出生于[[法罗群岛|法罗群岛]]首府[[托尔斯港|托尔斯港]]，这里的学校使用[[法罗语|法罗语]]进行教学。海纳森平时使用法罗语，但写作时使用[[丹麦语|丹麦语]]。他的所有作品都有相应的法罗语译本。<ref>{{cite web |url=http://www.bbs.fo/content/view/565/58/ |title=存档副本 |accessdate=2012-12-28 |deadurl=yes |archiveurl=https://archive.is/20130106164157/http://www.bbs.fo/content/view/565/58/ |archivedate=2013-01-06 }} BBS.fo</ref><br />

21岁时，他出版了自己的第一部诗集。1934年，他创作了自己的第一部长篇小说《暴风雨中的黎明》。此前，他已经出版了三本书。1938年，受画家[[西莫尔·乔恩森-米基内斯|西莫尔·乔恩森-米基内斯]]的影响，他创作了长篇小说《诺欧》。<ref>{{cite web |url=http://www.snar.fo/alfroei/alfroei/rithoevundar/heinesen_william/ |title=存档副本 |accessdate=2011-07-19 |deadurl=yes |archiveurl=https://web.archive.org/web/20120328035301/http://snar.fo/alfroei/alfroei/rithoevundar/heinesen_william/ |archivedate=2012-03-28 }} Snar.fo (''in Faroese'')</ref>1949年出版的《黑锅炉》描绘了宗教狂热下的糜烂生活。在《迷途中的音乐家们》（1950）中，海纳森不再使用他早期作品中的社会题材，转而使用一种简单直白的叙述方式。<ref>http://www.forfatterweb.dk/oversigt/zheinesen00/print_zheinesen14 Forfatterweb.dk</ref><br />

50岁时，海纳森开始尝试创作[[短篇小说|短篇小说]]。这些短篇小说的绝大部分都收录于三本小说集中：《欢欣的时光》、《迦玛的蛊惑》以及《拯救邪恶的灵魂》。

== 荣誉 ==
* 1960年，获得[[丹麦文学奖|丹麦文学奖]]霍尔伯格奖章。<ref>{{cite web |url=http://www.dlb.dansklf.dk/iframe.dsp?author=243 |title=存档副本 |accessdate=2007-02-20 |deadurl=yes |archiveurl=https://web.archive.org/web/20070205083426/http://www.dlb.dansklf.dk/iframe.dsp?author=243 |archivedate=2007-02-05 }} Dansk Litteraturhistorisk Bibliografi</ref>
* 1965年，凭借长篇小说《好望角》获[[北欧理事会文学奖|北欧理事会文学奖]]。在小说中，海纳森描绘了[[十七世纪|十七世纪]]的[[丹麦|丹麦]]社会，这部作品也被普遍认为是他的代表作。<ref>http://www.abc-literature.dk/eng Website about William Heinesen with bibliography</ref>
* 1981年，有传言称海纳森将获得当年的[[诺贝尔文学奖|诺贝尔文学奖]]，但随后，他给[[瑞典文学院|瑞典文学院]]写了封信，要求将自己的候选人名额取消。<ref>http://www.dedalusbooks.com/catalog.php?s=4&id=74 Dedalus Books</ref>
* 1980年，获得[[丹麦批评家文学奖|丹麦批评家文学奖]]。<ref>http://www.dedalusbooks.com/catalog.php?authID=74&disp=2 Dedalus Catalogue</ref>
* 1984年，获得[[托尔斯港市议会儿童图书奖|托尔斯港市议会儿童图书奖]]。
* 1985年，[[丹麦学院|丹麦学院]]授予[[凯伦布里森奖章|凯伦布里森奖章]]。
* 1987年，[[瑞典文学院|瑞典文学院]]授予[[北欧文学奖|北欧文学奖]]。<ref>http://www.danskeakademi.dk/priser/vispriser.asp?id=36 Danskeakademi.dk</ref>
