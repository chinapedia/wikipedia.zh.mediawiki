==分佈==

威靈仙現分佈於[[中國大陸|中國大陸]]、[[台灣|台灣]]、[[越南|越南]]、[[日本|日本]][[琉球群島|琉球群島]]等地<ref name="ars"/>，[[中國|中國]]大陸分佈於[[廣西|廣西]]（[[海拔|海拔]]160－1000[[米|米]]）、[[廣東|廣東]]、[[福建|福建]]、[[貴州|貴州]]（[[海拔|海拔]]150－1000[[米|米]]）、[[雲南|雲南]]南部、[[安徽|安徽]][[淮河|淮河]]以南、[[四川|四川]]（[[海拔|海拔]]500－1500[[米|米]]）、[[湖北|湖北]]、[[湖南|湖南]]（[[海拔|海拔]]80－700[[米|米]]）、[[江蘇|江蘇]]南部（[[海拔|海拔]]140－320[[米|米]]）、[[江西|江西]]（[[海拔|海拔]]140－700[[米|米]]）、[[陝西|陝西]]南部（[[海拔|海拔]]1000[[米|米]]以下）、[[浙江|浙江]]及[[河南|河南]]等省份<ref name = "中國"/>，長於田基、溝邊、路邊草叢、山坡、山谷灌叢中及路旁<ref name = "生草藥">{{cite book|title=增訂生草藥性備要|author= 何克諫著, 關培生（增訂）|publisher=聚賢館文化|page=99|isbn= 962-436-270-X|date=1995年10月}}</ref>。

==形態特徵==

威靈仙是一種半常綠<ref name="浙江">{{cite encyclopedia| encyclopedia=浙江植物誌| title=威靈仙| volume=2| pages=294-295| url=http://v2.cvh.org.cn/difangzhi/zhejiang/page/2/294a.pdf}}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>多年生落葉<ref name="秦岭">{{cite encyclopedia| encyclopedia=秦岭植物誌| title=威靈仙| volume=1（2）| pages=293| url=http://v2.cvh.org.cn/difangzhi/qinling/page/1（2）/293.pdf}}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>[[藤本|藤本]][[植物|植物]]，木質，全株暗綠色<ref name="浙江"/>，乾後變為黑色<ref name="貴州">{{cite encyclopedia| encyclopedia=貴州植物誌| title=威靈仙| volume=3| pages=54| url=http://v2.cvh.org.cn/difangzhi/guizhou/page/3/054a.pdf}}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，長達數[[米|米]]<ref name="海南">{{cite encyclopedia| encyclopedia=海南植物誌| title=威靈仙| volume=1| pages=305-306| url=http://v2.cvh.org.cn/difangzhi/hainan/page/1/305.pdf}}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。鬚根叢生，多而細長<ref name="福建">{{cite encyclopedia| encyclopedia=福建植物誌| title=威靈仙| volume=2| pages=24-25| url=http://v2.cvh.org.cn/difangzhi/fujian/page/2/024a.pdf}}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，細長圓柱形<ref name="南方藥用"/>，外皮淺黃色<ref name="生草藥"/>，內為白色<ref name="生草藥"/>，咀嚼時帶有辛辣味<ref name="生草藥"/><ref name="浙江"/>；[[莖|莖]]幼時疏披短柔毛，後漸脫落，具細縱棱<ref name="福建"/>，長約3－9[[米|米]]<ref name="浙江"/>；小枝疏披短柔毛或近無毛，稍四棱形<ref name="秦岭"/>。<ref name = "中國"/>

===葉===

[[葉|葉]]為一回[[奇數|奇數]]<ref name="河南">{{cite encyclopedia| encyclopedia=河南植物誌| title=威靈仙| volume=1| pages=456| url=http://v2.cvh.org.cn/difangzhi/henan/page/1/456.pdf}}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>羽狀[[複葉|複葉]]，對生<ref name="浙江"/><ref name = "藥典">{{cite book|title=《中華人民共和國藥典中藥彩色圖集（1995年版）》|author=中華人民共和國衛生部藥典委員會|publisher=三聯書店|page=325-326|isbn=962-04-1326-1|date=1996年11月}}</ref>，長約7－20厘米<ref name="河南"/>；小葉5枚，或有時小葉3－7枚<ref name="浙江"/>；小葉薄紙質，狹卵形、卵形、卵狀披針形、卵狀橢圓形、卵狀三角形<ref name="浙江"/>、三角狀卵形、卵圓形或線狀披針形<ref name = "中國"/><ref name = "藥典"/>，先端鈍至銳尖，有時微凹<ref name="浙江"/>，基部圓形、截形<ref name="秦岭"/>、寬楔形至淺心形，全緣，兩面近無毛或背面及脈上疏披短柔毛<ref name="秦岭"/>，基出脈3－5條，兩面網脈皆不明顯<ref name="浙江"/>，長約1.2<ref name="河南"/>－10<ref name="安徽">{{cite encyclopedia| encyclopedia=安徽植物誌| title=威靈仙| volume=2| pages=336-337| url=http://v2.cvh.org.cn/difangzhi/anhui/page/2/336a.pdf}}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>[[厘米|厘米]]，寬約1－7[[厘米|厘米]]；[[葉軸|葉軸]]上部及小葉柄扭曲<ref name="浙江"/>；[[葉柄|葉柄]]長約2.5－8<ref name="貴州"/>厘米。<ref name = "中國"/>

===花===

[[花|花]]常為[[圓錐|圓錐]]狀聚傘[[花序|花序]]<ref name="安徽"/>，頂生或腋生<ref name = "藥典"/>，長約12－18[[厘米|厘米]]<ref name="秦岭"/>；花白色或綠白色<ref name="生草藥"/><ref name="海南"/>，多數，花[[直徑|直徑]]約1－2[[厘米|厘米]]<ref name = "中國"/>；[[花序軸|花序軸]]具縱條紋<ref name = "海南"/>；[[花梗|花梗]]纖細，中部以下近基部處常有一對線狀苞片<ref name = "海南"/><ref name = "福建"/>，長約1－2[[厘米|厘米]]<ref name = "海南"/>；萼片白色，4－5枚，長圓形或長圓狀倒披針形，[[花瓣|花瓣]]狀<ref name = "南方藥用"/>，先端常凸尖，表面密披短柔毛，中央披短柔毛及外邊緣密生灰白色絨毛<ref name = "福建"/>，內部無毛<ref name = "秦岭"/>，開展，長約0.5－1.5[[厘米|厘米]]；無花瓣<ref name = "浙江"/>；[[雄蕊|雄蕊]]多數<ref name = "海南"/>，無毛，短於萼片<ref name = "海南"/>；[[花藥|花藥]]線形，藥室側向開裂<ref name = "秦岭"/>，較[[花絲|花絲]]長兩倍<ref name = "貴州"/>；[[子房|子房]]披灰黃色柔毛；[[花柱|花柱]]下部披緊貼的長柔毛<ref name = "安徽"/>，頂部及柱頭披短毛<ref name = "安徽"/>，細長<ref name = "浙江"/>；[[心皮|心皮]]4－6枚，密披柔毛<ref name = "海南"/>。<ref name = "中國"/>

===果===

[[果|果]]為[[瘦果|瘦果]]<ref name = "藥典"/>，3－7枚，乾後變為黑色<ref name = "秦岭"/>，扁平，寬橢圓形至卵形，略披伸展的白色柔毛<ref name = "生草藥"/><ref name = "秦岭"/>，頂端披羽狀長毛<ref name = "生草藥"/>，長約3－7[[毫米|毫米]]<ref name="金門"/><ref name = "貴州"/>，寬約4[[毫米|毫米]]<ref name = "秦岭"/>；宿存羽毛狀[[花柱|花柱]]<ref name = "秦岭"/>，線形<ref name = "海南"/>，披長柔毛<ref name = "海南"/>，長約1.8<ref name = "浙江"/>－5[[厘米|厘米]]<ref name = "福建"/>。<ref name = "中國"/>

"醫大"/>，臨床應用於風濕性[[關節炎|關節炎]]、慢性[[膽囊|膽囊]]炎、腰肌勞損、外用於[[咽喉|咽喉]]炎、足跟痛、[[牙|牙]]痛<ref name = "中國"/>等治療上<ref name = "當代藥用"/>。本品性走竄，久服易傷正氣<ref name = "醫大"/>，血虛而致的筋骨拘攣疼痛者禁用<ref name = "趣話中藥"/>，氣虛血弱者<ref name = "南方藥用"/><ref name = "家有百草"/>、體虛氣弱者<ref name = "趣話中藥"/>、陰虛有熱<ref name = "止痛"/>、無風寒濕邪者<ref name = "止痛"/>及孕婦忌服<ref>{{cite encyclopedia | encyclopedia=財團法人 台灣必安研究所| title= 中醫藥常識 威靈仙【2010/02/24】|url=http://www.brion.org.tw/index.php?option=com_content&task=view&id=1392&Itemid=22}}</ref>。新鮮全株可治[[咽喉|咽喉]]炎及急性[[扁桃體|扁桃體]]炎，根部可治絲蟲病，外用可治牙痛<ref name = "中國"/>。

==藥材鑑定==

[[中藥|中藥]]威靈仙以乾燥[[根|根]]及[[根莖|根莖]]入藥，於[[秋季|秋季]]採挖，清除泥沙等雜質後曬乾<ref name="香港中藥材"/>；本品[[根|根]]部呈細長圓柱形，表面黑褐色，具細縱紋，稍為彎曲，有的皮部脫落，露出黃白色木部，質地堅脆，容易折斷，長約7－20[[厘米|厘米]]<ref name="易誤用">{{cite encyclopedia| encyclopedia=行政院衛生署食品藥物管理局. 易誤用中藥材.| title=威靈仙（Clematidis Radix）| url=http://www.fda.gov.tw/files/list/09-02_%E5%A8%81%E9%9D%88%E4%BB%99.pdf}}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，直徑約0.1－0.3[[厘米|厘米]]<ref name="香港中藥材"/>；根部乾時容易折斷<ref name="易誤用"/>，略呈方性，木部淡黃色，斷面皮部較廣，棕黃色<ref name="易誤用"/>，皮與木部間常有裂隙<ref name="香港中藥材"/>，易分離，具微臭味及微苦<ref name="易誤用"/>。根莖呈圓柱狀，橫生<ref name="易誤用"/>，表面淡棕黃色，頂端殘留木質化的莖基<ref name="易誤用"/>，下側及兩側著生多數細根，質地較堅韌，皮部常折斷，斷面纖維性，具隆起的節<ref name="易誤用"/>，長約1.5－10[[厘米|厘米]]，直徑約0.3－2[[厘米|厘米]]<ref name="香港中藥材"/><ref name="易誤用"/>。飲片多為切碎<ref name = "常用中藥材">{{cite book|title=常用中藥材鑑別圖典|author=趙中振, 陳虎彪|publisher=萬里機構|page=143|isbn= 978-962-14-4150-8|date=2010年1月}}</ref>；氣微，味淡；傳統經驗則認為以根粗、質堅實、斷面灰白色、條勻、地上殘基短者為佳<ref name = "醫大"/><ref name = "常用中藥材"/>。本種與同屬的[[棉團鐵線蓮|棉團鐵線蓮]]（''Clematis hexapetala ''）及[[東北鐵線蓮|東北鐵線蓮]]（''Clematis mandshurica''）同收錄於《[[中國藥典|中國藥典]]》2005年版中，定為[[中藥|中藥]]威靈仙的法定原植物來源種之一，主要透過浸出物、性狀、顯微鑒別、[[齊墩果酸|齊墩果酸]]薄層對照等指標測定，以控制[[藥材|藥材]]的質量<ref name = "當代藥用"/>。

==毒性==

本品全株具毒性，毒性成份為[[靈仙新甙|靈仙新甙]]及[[皂甙|皂甙]]，皂甙對腸胃道產生刺激，誤食可引致噁心、嘔吐、腹痛、腹瀉、頭暈等症狀，嚴重甚至死亡；皮膚接觸可引致[[接觸性皮炎|接觸性皮炎]] <ref>{{cite web| url=http://www3.ha.org.hk/toxicplant/hk/clematis_chinensis.html| title= Clematis chinensis 威靈仙| publisher=香港醫院管理局毒理學參考化驗室香港有毒植物圖鑑| deadurl=yes| archiveurl= https://web.archive.org/web/20180521154949/http://www3.ha.org.hk/toxicplant/hk/clematis_chinensis.html | archivedate=2018-05-21}}</ref>。
