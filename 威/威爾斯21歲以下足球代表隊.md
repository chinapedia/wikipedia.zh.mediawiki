'''威爾斯21歲以下足球代表隊'''（'''Wales national under-21 football team'''，簡稱'''威爾斯U-21'''）是[[威爾斯|威爾斯]]的21歲以下國家[[英式足球|足球]]代表隊，由[[威爾斯足球總會|威爾斯足球總會]]負責組織及管理，參加每兩年舉辦一次的[[歐洲U-21足球錦標賽|歐洲U-21足球錦標賽]]。

威爾斯U-21是於1976年[[歐洲足協|歐洲足協]]改革青年競賽後組成，首場比賽於同年12月15日作客[[伍尔弗汉普顿流浪足球俱乐部|狼隊]]的[[莫利纽克斯球场|莫利纽克斯球场]]對[[英格蘭21歲以下足球代表隊|英格蘭U21]]，雙方踢成0-0和氣收場。

威爾斯U-21是[[威爾斯|威爾斯]]境內最高級的青年足球隊，球員入選資格源於本地出生、父母或祖父母在威爾斯出生。

== 競賽歷史 ==
威爾斯U-21在[[歐洲U-21足球錦標賽|歐洲U-21足球錦標賽]]一向只求參與，歷屆均在外圍賽早早出局。直到[[2009年歐洲U-21足球錦標賽|2009年]]才取得突破，在外圍賽分組賽以8戰6勝2負的成績，主場憑[[捷特·伊雲斯|捷特·伊雲斯]]（Ched Evans）大演[[帽子戲法|帽子戲法]]4-2擊敗[[法國21歲以下國家足球隊|法國]]<ref>{{cite web |language=en |url=http://www.uefa.com/competitions/under21/fixturesresults/round=15084/match=300189/index.html |title=France dazzled by Evans hat-trick |publisher=[[UEFA|''uefa.com'']] |accessdate=2009-09-15}}</ref>及最後一場作客3-0擊敗[[羅馬尼亞21歲以下國家足球隊|羅馬尼亞]]<ref>{{cite web |language=en |url=http://www.uefa.com/competitions/under21/fixturesresults/round=15084/match=300194/index.html |title=Wales claim knockout victory in Romania |publisher=[[UEFA|''uefa.com'']] |accessdate=2009-09-15}}</ref>，最終僅以1分之微力壓法國出線次階段附加賽。附加賽碰上[[英格蘭21歲以下足球代表隊|英格蘭]]，首回合主場僅負2-3<ref>{{cite web |language=en |url=http://www.uefa.com/competitions/under21/fixturesresults/round=15085/match=304063/report=rp.html |title=Agbonlahor puts England in command |publisher=[[UEFA|''uefa.com'']] |accessdate=2009-09-15}}</ref>，次回合作客更迫成2-2平手<ref>{{cite web |language=en |url=http://www.uefa.com/competitions/under21/fixturesresults/round=15085/match=304071/report=rp.html |title=England withstand Welsh onslaught |publisher=[[UEFA|''uefa.com'']] |accessdate=2009-09-15}}</ref>，累計4-5飲恨出局。
