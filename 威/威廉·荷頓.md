'''威廉·荷頓'''（{{lang-en|William Holden}}，{{bd|1918年|4月17日|1981年|11月16日|catIdx=}}）[[美國|美國]]演員，1953年以電影《[[战地军魂|战地军魂]]》获得[[奧斯卡最佳男主角獎|奧斯卡最佳男主角獎]]。《[[日落大道_(電影)|日落大道]]》、《[[螢光幕後|螢光幕後]]》提名奧斯卡最佳男主角獎；《[[野宴|野宴]]》、《螢光幕後》提名[[英國電影和電視藝術學院|英國電影和電視藝術學院]]最佳外國男主角獎。1974年，他以電視短劇《[[蓝骑士|蓝骑士]]》得到[[艾美獎|艾美獎]]最佳男主角獎。他曾6度列名年度十大票房影星，其中1956年高居第一名。1999年，威廉·荷頓被[[美國電影學會|美國電影學會]]選為[[百年百大明星|百年來最偉大的男演員]]第25名。

晚年的荷頓為[[憂鬱症|憂鬱症]]所苦，甚至用[[酗酒|酗酒]]來紓解。1981年，威廉·荷頓在[[加州|加州]][[聖塔莫尼卡|聖塔莫尼卡]]海濱住宅跌倒身亡，享年63歲，依其遺願火化[[海葬|海葬]][[太平洋|太平洋]]。

== 事业 ==
威廉·荷頓擅長的電影類型甚廣，其中與導演[[比利·懷德|比利·懷德]]有過4次合作：《[[日落大道_(電影)|日落大道]]》、《[[戰地軍魂|戰地軍魂]]》（Stalag 17）、《[[龍鳳配|龍鳳配]]》、《[[麗人劫|麗人劫]]》（Fedora）。他在《[[蘇絲黃的世界|蘇絲黃的世界]]》演出風度翩翩的美國[[畫家|畫家]]與[[香港|香港]]酒吧女郎[[關南施|關南施]]談著[[東方|東方]]情調式美麗戀情；另一部作品《[[生死戀|生死戀]]》場景也是在香港，其電影主題曲是英語老式情歌的經典。其他知名作品尚有[[南北戰爭|南北戰爭]]背景的《[[魔鬼騎兵團|魔鬼騎兵團]]》（The Horse Soldiers）、場景[[亞洲|亞洲]]、二次大戰背景的《[[桂河大橋|桂河大橋]]》、[[韓戰|韓戰]]背景的《[[獨孤里橋之役|獨孤里橋之役]]》；[[西部片|西部片]]《[[日落黃沙|日落黃沙]]》；諷刺電影《[[螢光幕後|螢光幕後]]》（Network）；以及災難片《[[沖天大火災|火燒摩天樓]]》等。

荷頓在大銀幕上以「溫文儒雅、正直助人」形象著稱。當初他本來因為《戰地軍魂》劇中主角個性太過自私、憤世嫉俗而謝拒。導演比利·懷德又不肯將劇中人物改得更有同情心。但是威廉·荷頓最後還是被[[派拉蒙|派拉蒙]]「強迫」演出，並獲頒1953年奧斯卡最佳男主角獎。他有個綽號「Golden Boy」源自於1939年的成名作。

== 生活 ==
威廉·荷頓與女演員[[布倫達·馬歇爾|布倫達·馬歇爾]]結婚，他們生有兩子，荷頓並收養妻子與前夫之女，不過兩人在1971年，於多年分居之後離婚。

荷頓長期致力於[[野生動物|野生動物]]保育。1972年，他與志同道合提倡動物權益的女演員[[史蒂芬妮·鮑爾斯|史蒂芬妮·鮑爾斯]]有過戀情，但並無結果。他死后史蒂芬妮·鮑爾斯為其成立「威廉·荷頓野生動物基金會」。

荷頓的弟弟是一位[[美國海軍|海軍]][[戰鬥機|戰鬥機]]飛行員，並在1945年二戰期間殉職。10年後，荷頓演出以海軍飛行員為主角的《獨孤里橋之役》，被其弟的同僚認為神似他們已故的戰友。

== 其他 ==
威廉·荷頓是美國[[共和黨_(美國)|共和黨]]支持者，與[[隆納·雷根|隆納·雷根]]及[[南茜·雷根|南茜·雷根]]夫婦有著深厚的友情，荷頓夫婦是雷根夫婦婚禮中的男女儐相。[[二次大戰|二次大戰]]時，荷頓與雷根曾在[[美國陸軍航空軍|美國陸軍航空軍]]電影小組服役，兩人軍銜都是[[上尉|上尉]]。

威廉·荷頓為1954年《[[生活_(雜誌)|生活雜誌]]》的封面人物。
