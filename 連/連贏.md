為賽馬術語，是[[賽馬|賽馬]][[博彩|博彩]]的一個重要投注項目，在[[香港|香港]]、[[澳門|澳門]]、[[新加坡|新加坡]]、[[英國|英國]]、[[美國|美國]]、[[澳洲|澳洲]]、[[日本|日本]]、[[韓國|韓國]]等地方均有接受投注，澳門及香港粵語稱為「連贏」，投注者需選中一場賽事中的第一名及第二名，毋須順序。

== 投注金額 ==
以澳門及香港為例，連贏每注最少澳門幣或港幣10元；而新加坡為每注最少2新加坡元。

== 投注方法 ==
{|class="wikitable" width="70%"
|- style="background:#C0C0C0;" align=center
|width=10%|'''投-{注}-種類'''||width=60%|'''投-{注}-方法'''
|-
|align=center|「單式」投-{注}-||從自選任何馬匹編號中選出2匹不同馬匹編號。
|-
|align=center|「複式」投-{注}-||假設您的心水馬為7號、9號、11號，複式互串會將所選馬匹組成所有可能的組合，即分別為7+9、7+11 及 9+11，總共三注 ，好處是提高命中機會。
|-
|align=center|「馬-{膽}-」投-{注}-||當您對一匹馬有很大信心能跑入前兩名，您可選擇以該匹馬作「馬膽」，其他選擇則作為「配腳」。
假設重心馬為7號，以7號胆拖9號、11號，即分別為7+9 及 7+11，總共兩注，只要其中一個馬匹組合同時跑入前兩名，便為中彩，相比起「複式互串」好處是更能節省注項。
|}

== 中獎方法 ==
選中一場賽事中的第1名及第2名馬匹，毋須順序。

== 香港賽馬史上最冷連贏派彩 ==
紀錄於2017年3月5日第九場產生，由39倍的「天合」及99倍的「中游新星」包辦首兩名，每港幣$10派$23,863。<ref>【本港史上最冷連贏位】兩位本地騎師包Q買10蚊派近24000元 http://hk.racing.nextmedia.com/art_main.php?showdate=20170306&sec_id=3403910&art_id=17213129</ref>

舊紀錄是2013年2月17日「魅力知己」搭「總統之光」所構成的連贏位，派彩$18,948。
