'''妖魅變成夜話'''（'''{{lang|ja|妖魅変成夜話}}'''，'''{{lang|ja|ようみへんじょうやわ}}'''）是[[日本|日本]][[漫畫家|漫畫家]][[岡野玲子|岡野玲子]]的作品。故事背景為中國唐代，內容敘述主角李成潭身邊的各式靈異遭遇。一般認為本漫畫作品為岡野另一代表作「陰陽師」的番外篇，作者無法在嚴謹的「陰陽師」中表達的創作靈感，藉由本作品呈現。

== 內容簡介 ==

出生時帶著祥雲的李成潭，上京趕考的路上，遇上了女鬼及會變成美少年的妖狐，或許是他的幸運吧，女鬼及妖狐都跟隨著他不肯離去......

考中了進士後又被分配到沒血沒淚的龍玉將軍手下，他的工作是到全國各地去查訪各式怪異事件，繼女鬼及妖狐之後，幸運的李成潭又會遇上什麼樣的怪事呢？

== 登場人物 ==

=== 主要角色 ===

; 李成潭
: 實名李泉，字成潭。原為個性古板的鄉下秀才，志於實踐孔子教誨，為求考上科舉而一心向學。進士及第之後，被派往龍玉將軍麾下，負責調查超自然現象的始末，並製作給皇上過目的「奧秘事錄」。對美女毫無招架之力，不知何故，相當受到幽靈或妖怪的青睞。
; 龍玉將軍
: 率領直屬於皇帝的雙龍部隊的神秘人物，負責執行逮捕神仙的極機密任務。經常神出鬼沒，興趣為扮女裝。真實身分為謎。
; 白葉
: 原型為狐狸精，對李成潭一見鍾情，在涇水與李成潭有一夜之緣。
; 王珍春
: 美貌的女性幽靈，與李成潭有過一夜之緣。
; 寧寧兒、姚姚兒
: 龍玉將軍麾下的孿生童子，武功實力堅強。真實身分為謎。

=== 次要角色 ===

; 洞庭君
: 洞庭湖龍王，負責治理長江諸水。容貌魁偉，住在湖底的龍宮。
; 錢塘君
: 洞庭君之弟，錢塘江之神，水神族首長之一。個性急躁，曾以大水淹沒五嶽，而被天帝罰閉門思過。極度迷戀龍玉。
; 西湖公主
: 個性輕浮的仙界公主，興趣為打獵。
; 五雲龍
: 龍玉將軍的隨從，共五人。
