{{vgsname|妖怪手錶|ja=妖怪ウォッチ}}是[[LEVEL-5|LEVEL-5]]开发的[[任天堂3DS|任天堂3DS]][[遊戲軟件|遊戲軟件]]，在2013年7月11日發布，是LEVEL-5繼《[[閃電十一人|閃電十一人]]》和《[[紙箱戰機|紙箱戰機]]》系列后第三套[[跨媒體製作|跨媒體製作]]作品。宣傳歌曲是由[[King_Cream_Soda|King Cream Soda]]演唱的「妖怪碰碰之歌」與[[Dream5|Dream5]]演唱的「{{link-ja|妖怪體操第一|Break Out/ようかい体操第一}}」。

改編的[[漫畫|漫畫]]由小西紀行和{{link-ja|森千嘉子|もりちかこ}}創作，先後於2013年1月和2014年1月在《[[快樂快樂月刊|快樂龍]]》和《[[Ciao|Ciao]]》連載。是第38屆和第60屆「[[講談社漫畫賞|講談社漫畫賞]]」兒童部門，以及第27屆「[[小學館DIME品牌大賞|小學館DIME品牌大賞]]」優閒及娛樂部門的得奬作品<ref>{{Cite web |url=http://ebook.itmedia.co.jp/ebook/articles/1501/22/news103.html |title=第60回小学館漫画賞が発表、一般部門は『アオイホノオ』『あさひなぐ』のW受賞。 |date=2015-01-22 |accessdate=2016-12-27 |work=ITmedia |language=ja }}</ref><ref>{{Cite web |url=http://www.shogakukan.co.jp/st/files/20150121.pdf |title= 第60屆講談社漫畫賞官方得獎名單 |author=小学館 |language=ja}}</ref><ref>{{Cite web |url=http://www.nikkansports.com/entertainment/news/f-et-tp0-20141113-1395791.html |title= 流行りました「妖怪ウォッチ」トレンド大賞。 |work=日經體育新聞 |date=2014-11-13 |accessdate=2016-12-27 |language=ja}}</ref>。

與動畫制作公司[[OLM|OLM]]共同制作的[[電視動畫|電視動畫]]，於2014年在[[東京電視網|東京電視台系列]]播出。作品中的表現手法在日本小學生間引起了不少的话题，間接帶起作品的人氣。

作品中的重要道具「妖怪徽章」於同年1月11日發售，一度成为頗受日本小學生歡迎的人氣商品。此外，片尾曲「妖怪體操第一（{{lang|ja|ようかい体操第一}}）」的片段在YouTube上的點点击次數已超過1億，舞蹈動作既可愛又簡單，吸引很多網民學跳，把跳舞的片段上傳到[[niconico動畫|niconico動畫]]。

海外播出方面，香港由[[翡翠台|翡翠台]]在2015年5月3日开始播放；台灣由[[東森幼幼台|東森幼幼台]]在2015年5月9日开始播放，[[中華電視台|中華電視台]]在2016年1月23日开始播放，每週六下午17:00~17:30首播，每週日上午10:30~11:00重播。<p>

截至2018年，遊戲系列在全球的累計出貨量超過了1400萬份<ref>https://mediafactory.jp/Form/Product/ProductDetail.aspx?shop=0&pid=4935228180301</ref>。

== 故事概要 ==
一位普通的小學5年級學生'''天野景太'''在暑假中去捕蟲的時候偶然發現了樹下擺著一台破舊的扭蛋機，當他掏出身上唯一一枚100[[日圓|YEN]]硬幣擰出扭蛋之後，打開了一隻神秘的扭蛋。扭蛋中飛出了一個
自稱為「妖怪執事」 的妖怪'''[[威斯帕|威斯帕]]'''。'''威斯帕'''根本不是執事，牠要用「妖怪pad」查詢妖怪！但從此以後，景太就可以用妖怪手錶看到身邊出現的各種不可思議的妖怪了。(有可愛族、滑嘟嘟族、噁心族、不可思議族等)遇到困難的時候可以用妖怪徽章召喚妖怪朋友們呢!

第3部電影'''《[[電影版妖怪手錶：飛天鯨魚和雙重世界的大冒險喵！|電影版妖怪手錶：飛天鯨魚和雙重世界的大冒險喵！]]》'''（{{lang|ja|『妖怪ウォッチ』空飛ぶクジラとダブル世界の大冒険だニャン!}}）於2016年12月17日於日本上映。此部電影設定有二個世界，分別為「寫實」和「動畫」，「寫實」部分是由真人演出，而「動畫」世界則由動畫表現。台灣於2017年8月18日暑假歡樂首選。
