《'''妖力女神'''》是[[六道神士|六道神士]]的日本漫畫作品，在[[少年畫報社|少年畫報社]][[青年漫畫|青年漫畫]]雜誌《[[YOUNG_KING|YOUNG KING]]》2010年9月號開始連載，2011年6月20日改至《[[月刊YOUNG_KING|月刊YOUNG KING]]》2011年8月號連載，至《[[月刊YOUNG_KING_OURS_GH|月刊YOUNG KING OURS GH]]》（《月刊YOUNG KING》2013年10月號起改名）2016年4月號連載完畢。台灣及香港中文版由[[東立出版社|東立出版社]]發行。

== 故事概要 ==
宮森水城是個外表不起眼的大學法律系女學生，願望就是過著不起眼的生活。但是某一天，[[炒地皮|炒地皮]]的[[流氓|流氓]]跑到宮森家中企圖[[放火|放火]]，但流氓因為被發現而刺殺水城。就在水城以為自己就要死去的時候，眼前突然出現兩個美女——朱雀與竃門。但是下一個瞬間，水城發現自己沒死，自己卻跟朱雀結合在一起。原來，朱雀與竃門都是靠吃人類「時間」來活下去的妖怪；而且，若不吃「時間」，就會導致自己消失。水城「最差勁的日子」，拉開序幕！

== 出場人物 ==
; {{nihongo|宮森水城|宮森 水城（みやのもり みずき）}}
: 女主角，大學法律系女學生，外表不起眼的短髮[[貧乳|貧乳]][[眼鏡娘|眼鏡娘]]。無意間成為宮森家土地[[所有權|所有權]]人。在自己被刺殺時被朱雀救起，卻同時將朱雀束縛在體內，外貌也和朱雀融合變成身材超好的大美女。束縛不空後長出[[小雞雞|小雞雞]]，差點想剪斷，不過將不空以小雞雞變化白貂的模樣獨立出來。
: 在去小鳥家經營的民宿時，意外撞見疑是天狗的妖怪，被吸收後成為自己的翅膀。
; {{nihongo|朱雀|スザク}}
: 突然在水城面前出現的[[和服|和服]]美女，[[第一人稱|第一人稱]]為「吾」。在水城被刺殺時救了水城，卻被水城束縛在體內。
: 過去曾與打敗自己的和尚慧來結為夫妻。
: 真正的身份是由於不空的母親-不二的原因，穿越到古代的日本，忘卻了一切記憶的水城。
: 後來把「羅睺番眾」消滅，並決心以'''宮森水城'''的身份走完人生的路程。
; {{nihongo|竃門|カマド}}
: 與朱雀一起出現的和服美女，能夠聞到死亡的味道。稱朱雀為「姊姊」。朱雀說她「遲鈍又耐打，而且記性非常不好」。
; {{nihongo|宮森三笠|宮森 カサ（みやのもり ミカサ）}}
: 宮森家三姊弟中的長女，水城的養母。過去是號稱「千里眼」的美女，找東西之類非常準，但這只是表面的能力，實際上能夠看見無形的東西，年幼時是全盛時期，個性十分亂來、腹黑，據武藏所說此時的三笠是其他人。
; {{nihongo|宮森武藏|宮森 武蔵（みやのもり むさし）}}
: 宮森家三姊弟中的長男，擅長[[劍術|劍術]]。年青時是一個身材魁梧、擅長[[二刀流|二刀流]]的男性。
; {{nihongo|宮森延壽|宮森 延寿（みやのもり えんじゅ）}}
: 宮森家三姊弟中的次男，總能吸引一堆女性為他爭風吃醋。會使用[[數位相機|數位相機]]與[[個人電腦|個人電腦]]。三笠說他是「最差勁的花花公子」、「瘟神」，而且「跟他交往而變得不幸的人，全都會擺出似乎很幸福的表情」。年青時是一名長髮的美男子。擁有魅惑別人的能力「王之眼」。全盛時期是在15、6歲，因為教主不受他的誘惑，因此喜歡上她，更因此背叛了宮森家。
; {{nihongo|松川小鳥|松川 小鳥（まつごう ことり）}}
: 水城的同班好友，喜歡戴[[草帽|草帽]]的小個子女孩。曾經被不空作為誘餌而被吃了一半，丟在宮森家附近，之後又被水城復活了。
; {{nihongo|大町日和|大町 日和（おおまち ひより）}}
: 水城的同班好友，有常識的[[眼鏡娘|眼鏡女孩]]。
; {{nihongo|不空|フクウ}}
: 男夜叉，在宮森家門前對小鳥吃一半的時間棄屍，當做試探水城的反應。與水城正面對決時佔有優勢，但不小心反被水城束縛在體內。與朱雀一樣，沒水城的同意無法出面。不過將水城將自己以小雞雞變化白貂的模樣獨立出來，但無法遠離水城太遠。
; {{nihongo|旦月|旦月（たんげつ）}}
: 經營[[不動產仲介|不動產仲介]]業者「青山不動産」，常常向宮森家要求出售土地。對於殺人没有任何抵觸 ，左臂有著從和尚繼承過來的刺青，擁有自我意識，并且可以跟人交流。 
二十九話幫助宫森家修復结界之后，作为宫森家食客留下，被竃門變成老頭子，在宮森家負責打掃。
: 對殺人沒有絲毫的猶疑，對曾經殺掉自己的乙金圓的頸上捅了一刀。
; {{nihongo|小高|タカ}}
: 旦月的小弟，是某人委託旦月關照的(其死後，旦月被上司責怪看管不力而受罰)。吸食[[大麻|大麻]]。試圖縱火宮森家以示威脅，被水城發現慌張之餘刺殺水城，被竃門吃光時間，已死無全屍。但因水城改變過去，導致他沒有去過宮森家，在現在的世界裡，似乎還活著。
; {{nihongo|埴安秋葉|埴安 秋葉（はにやす あきは）}}
: 埴安家的孫女（[[偽娘|偽娘]]），喜歡漂亮女孩子。喜欢延壽，水城，朱雀，竃門。實際上就是一個變態，喜歡用錢解決事情。
;  {{nihongo|乙金圓|乙金 まどか（おとがね まどか）}}
: [[新興宗教|新興宗教]]「實濟智之光」教祖的代理人，言行讓人摸不着頭腦。有著能作為容器附身的才能。對她的教主大人十分忠心，因為不能忍受教主大人被當成道具利用，而服毒自殺。
; {{nihongo|慧來|慧来（エノキ）}}
: 曾經打敗朱雀的和尚，曾與朱雀結為夫妻。
