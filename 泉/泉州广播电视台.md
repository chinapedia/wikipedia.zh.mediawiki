'''泉州电视台'''是[[福建省|福建省]][[泉州市|泉州市]]的市级[[电视台|电视台]]，于1986年12月建台，1988年9月1日正式开播，现有[[国家广电总局|国家广电总局]]正式批准的“新闻综合”、“都市生活”、“影视剧”和“闽南语”四个频道，电视信号覆盖泉州地区以及金门、同安、厦门、三明大田、莆田仙游、福州永泰、龙岩漳平、漳州的部分地区，节目覆盖面积超13000[[平方公里|平方公里]]，收视人口超千万。


== 历史 ==
*1986年3月，泉州电视台筹建
*1987年10月，[[泉州市人民政府|泉州市人民政府]]批准泉州电视台编制建立
*1987年12月19日，[[广播电影电视部|广播电影电视部]]批准正式建台
*1988年2月16日，试办《泉州新闻》节目
*1988年9月1日，泉州电视台正式开播。开台初期，自制节目有《泉州新闻》、《侨乡周末》、《交通警报台》等
*1991年12月31日，泉州有线电视台开播。<ref>{{Cite web |url=http://www.qzt.gov.cn/szfzg/zs/201606/t20160604_3801.htm |title=《鲤城区志》第932页 |access-date=2017-10-14 |archive-url=https://web.archive.org/web/20171014183217/http://www.qzt.gov.cn/szfzg/zs/201606/t20160604_3801.htm |archive-date=2017-10-14 |dead-url=yes }}</ref>
*1997年，鲤城电视台收归泉州电视台，更名为泉州广播电视二台。当年12月28日，泉州电视台搬迁至现址田淮街66号的泉州市广播电视中心办公、播出。<ref>{{Cite web |url=http://www.qzt.gov.cn/szfzg/nj/201604/t20160421_2526.htm |title=《泉州年鉴1996-2000》第208页 |access-date=2017-10-14 |archive-url=https://web.archive.org/web/20171014183048/http://www.qzt.gov.cn/szfzg/nj/201604/t20160421_2526.htm |archive-date=2017-10-14 |dead-url=yes }}</ref>
*2001年7月1日，泉州电视台第一次更换台标，由红圆绿Z更换为金框QZ）。泉州有线电视台更名为泉州电视台2套，泉州广播电视二台更名为泉州电视台3套<ref>{{Cite web |url=http://www.qzt.gov.cn/szfzg/nj/201605/t20160504_3732.htm |title=《泉州年鉴2002》第747页 |access-date=2017-10-14 |archive-url=https://web.archive.org/web/20171014233646/http://www.qzt.gov.cn/szfzg/nj/201605/t20160504_3732.htm |archive-date=2017-10-14 |dead-url=yes }}</ref>。
*2002年9月30日，泉州电视台1-3套分别更名为新闻综合频道、经济生活频道、影视文艺频道<ref>[http://www.qzsfzw.fjqz.gov.cn/szfzg/nj/201604/t20160421_2527.htm 《泉州年鉴2003》报刊广播电视出版]</ref>。
*2005年，泉州电视台2套再次更名为都市生活频道，3套更名为影视剧频道
*2007年5月，泉州电视台闽南语频道开播。这是第一个全部采用闽南语播出的综合频道，以泉州话节目为主、以闽南人文为特色、以“传承闽南文化，服务两岸乡亲”为频道宗旨，汇集了《咱厝人》、《泉州讲古》、《学说泉州话》等精品节目。
*2012年6月，泉州电视台与[[泉州人民广播电台|泉州人民广播电台]]合并，成立泉州广播电视台
*2015年6月，泉州电视台更换为现有台标

== 电视台实力 ==
泉州电视台拥有一支优良的采、编、播队伍，现有采编播人员300多人，专业设备齐全的800平方米演播中心1座，50-100平方米演播室7个，采用BTC广播级设备制作和播出节目，并向数字化方向发展
。配备大型电视转播车，具备强劲的现场采摄、现场直播、实况转播、后期制作能力。节目在全国电视行业竞赛评比中屡获大奖，是福建省最具实力的电视媒体之一。
*1996年1月，首次购置6讯道模拟分量转播车，该转播车系用一部[[大客车|大客车]]改装而来。
*2005年，泉州电视台花费1000万元[[人民币|人民币]]，购置8讯道数字[[标清|标清]]转播车1部，采用[[沃尔沃|沃尔沃]]底盘总成及索尼控件。<ref>[http://www.c114.com.cn/tech/167/a204113.html 泉州电视台6+2数字转播车体系介绍]</ref>。2006年2月交付使用，随后淘汰旧有的6讯道模拟转播车<ref>[http://www.qzsfzw.fjqz.gov.cn/szfzg/nj/201606/t20160629_7715.htm 《泉州年鉴2006》-泉州广播电视中心]</ref>
*2013年10月，泉州电视台再次向[[索尼|索尼]]采购10讯道[[高清|高清]]转播车1部，并继续采用沃尔沃总成<ref>[https://www.pro.sony.com.cn/cn/pss/13740.htm 系统解决方案-高清转播车]</ref>，2014年3月20日交付使用。<ref>[http://tech.huanqiu.com/Enterprise/2014-03/4919602.html 索尼专业高清电视转播车交付泉州广播电视台]</ref>
*2017年8月，启动[[高清|高清]]化改造程序，先后完成高清[[虚拟演播室|虚拟演播室]]、非编制作网及摄录一体机等的采购招标。
*2018年7月，高清新闻网采购招标完成。由中科大洋获得本次标的。<ref>[http://qz.fjzfcg.gov.cn/350500/notice/d2bad35854053876b45269f56e50dee2/9aea2f0953fb42d889aaefc2725764cc/ 泉州广播电视台高清移动新闻网货物类采购项目]</ref><ref>[http://www.dayang.com.cn/folder78/folder79/folder86/2018-07-13/4278.html 大洋连续中标太仓、泉州两大地市级融媒体项目]</ref>
