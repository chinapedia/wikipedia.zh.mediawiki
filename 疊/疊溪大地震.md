'''疊溪大地震'''，又稱'''茂縣大地震'''，發生于1933年8月25日（[[民国|民国]]二十二年七月初五）<ref name="one">{{cite news|author1=郑其|title=叠溪古城海子里沉睡的庞贝？|url=http://morning.scol.com.cn/new/html/tfzb/20130828/tfzb667982.html|accessdate=2016-03-31|agency=天府早报|date=2013-08-28}}</ref>，震中位于中国[[四川省|四川省]]西北[[茂縣|茂縣]][[疊溪鎮|疊溪鎮]]。地震強度達7.5[[面波震級|M<sub>s</sub>]]，造成大規模的地表變動，山體滑坡，[[岷江|岷江]]河水倒灌，導致[[疊溪古城|疊溪古城]]被淹沒盡毀，形成今日的[[疊溪海子|疊溪海子]]。

这场地震的有感范围北至[[西安|西安]]，东至万县（今[[重庆市|重庆市]][[万州区|万州区]]），西到[[阿坝|阿坝]]，南至[[昭通|昭通]]。[[中国西部科学院|中国西部科学院]]地质研究所主任[[常隆庆|常隆庆]]携助手于10月赴往当地调查，在为期一个月的调查中得到如下结果：因地震而直接死难者6865人，受伤1925人，房屋倒塌5108间，受灾民众6352人，损失牲畜8878头<ref name="two">{{cite news|author1=韩福东|title=叠溪覆城80年祭|url=http://www.nandu.com/nis/201304/24/42909.html|accessdate=2016-03-31|agency=南方都市报|publisher=南都网|date=2013-04-24|deadurl=yes|archiveurl=https://web.archive.org/web/20160413003033/http://www.nandu.com/nis/201304/24/42909.html|archivedate=2016年4月13日|df=}}</ref>。地震后的10月9日又发生了次生洪水灾害，造成约两千多人遇难死亡。

== 背景与地理 ==
叠溪古镇位于[[茂县|茂县]]的西北部，今213国道处，位于茂县县城凤仪镇以北60多公里处，是川西平原经[[松潘草地|松潘草地]]前往青海、甘肃的一个交通要道<ref name="three">{{cite journal|author1=郑光路|title=叠溪古城瞬间消失之谜|journal=《丝绸之路》|date=2006年|issue=5|accessdate=2016-03-31}}</ref>。古镇地理上位于[[岷江|岷江]]上游旁的高台地上，未地震前据常隆庆的考察古镇高出当时河面有二百七十公尺。地震前古镇常住人口约为600余人，城内的房屋不到300所。历史上，[[汉代|汉代]][[元鼎|元鼎]]六年（公元前111年）在此地设蚕陵县。[[唐|唐]][[贞观|贞观]]四年（公元630年）隶属于翼州，是灌县（今都江堰市）经茂县到松潘的汉番中转站，也是在此时修筑了最早的叠溪城墙，但规模小于[[明朝|明朝]]时<ref name="three"/>。民国时期自1927年来叠溪这一岷江上游地区实际由[[川军|川军]]28路军军长[[邓锡侯|邓锡侯]]控制<ref name="four">{{cite journal|author1=王田|title=川西岷江上游地区的灾难叙述--以1933年叠溪地震为中心|journal=《西南民族大学学报:人文社科版》|date=2013年|issue=6|pages=第24至28页}}</ref>。

可肯定地震的发生属于断层活动引起的构造地震，震中最大烈度为Ⅹ度。青藏高原东缘来主要由岷江-虎牙断裂带和龙门山断裂带组成。因为在当时仅有北京和南京的两个地震台对地震有记录，使得关于地震的震源构造和机制存在争议。钱洪、周荣军等（1999）认为叠溪地震的发震构造是左旋为主的岷江断裂南段的延伸<ref>{{cite journal|author1=钱洪|coauthors=周荣军、马生浩、黎小刚|title=岷江断裂带南端与1933年叠溪地震研究|journal=《中国地震》|date=1999年|volume=15|issue=4|pages=第333至338页|accessdate=2016-04-02}}</ref>。这一说法也得到了沈正康和王康等人(2011)的认同，依据他们所搜索到的全球地震台站的记录提出地震发震构造是以左旋走滑为主兼具逆冲分量的岷江断裂带南段<ref name="six">{{cite journal|author1=沈正康|coauthors=王康|title=1933年叠溪地震的发震位置、震源机制与区域构造|journal=《地震学报》|date=2011年|volume=33|issue=5|pages=第557至567页|accessdate=2016-04-02}}</ref>。另外也有学者如黄祖智等（2002）和唐荣昌等（1983）认为地震的发震构造是位于较场弧形构造区内一条呈N52°-55°W的松平沟断裂，并分析是在川青块体近东西向的[[压应力|压应力]]下使得松平沟产生左旋的剪切活动引发了此次地震<ref>{{cite journal|author1=唐荣昌|coauthors=蒋能强、刘胜利|title=叠溪7.5级地震的地质构造背景及其发震构造条件的认识|journal=《地震研究》|date=1983年|volume=6|issue=3|pages=第327至337页}}</ref><ref>{{cite journal|author1=黄祖智|coauthors=唐荣昌、刘盛利|title=四川较场弧形构造与1933年叠溪地震发震构造的再讨论|journal=《中国地震》|date=2002年|volume=2|issue=18|pages=第183至192页}}</ref>。汶川大地震后，一些研究学者将叠溪地震和汶川地震推到大环境下分析，认为是巴颜喀拉地块（断块）向东滑行的结果，其中闻学泽（2009）进一步指出在[[青藏高原|青藏高原]]的东部川青块体（巴颜喀拉地块）的的东边界处，在近400年地震活动中处于由平稳到加速的应变能释放的过程，叠溪地震就是自1879年加速释放后的一场大地震，与之后2008年的汶川大地震一样皆为其边界的释放事件<ref name="seven">{{cite journal|author1=闻学泽|coauthors=张培震，杜方等|title=2008年汶川8.0级地震发生地历史与现今地震活动背景|journal=《地质物理学报》|date=2009年|volume=52|issue=2|pages=第444至454页|accessdate=2016-04-02}}</ref>。

== 破坏和伤亡 ==
1933年的地震导致岩石崩塌，截断岷江干流及支流形成10个堰塞湖。岷江被堵45天后，堰塞湖溃决，形成洪水。洪峰流量约10200立方米每秒，冲毁都江堰的水利工程。灌县（即[[都江堰市|都江堰市]]）死亡人数5000人以上，红十字会等单位沿途捞尸717具。<ref>[http://news.163.com/16/0722/21/BSK0G64700014Q4P.html 古代都江堰工程巧妙抗洪(组图).网易.2016-07-22]</ref>

地震导致岷江堵塞，形成大小海子11个，淹没了大量的田地和房屋，包括叠溪城及附近21个羌寨，死亡6800多人。<ref>{{cite news|title=四川茂县曾发生惨重地震灾害 叠溪古城消失(图)|url=http://news.sina.com.cn/zx/2017-06-24/doc-ifyhmtcf2834839.shtml|accessdate=2017-07-09|work=news.sina.com.cn|agency=新浪}}</ref>
