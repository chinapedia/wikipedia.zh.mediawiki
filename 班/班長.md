'''班長'''是组织团体的基层单位'''班'''或学校进行教育的基本单位'''班级'''的负责人。

== 学校 ==
=== 中国大陆 ===
在中国大陆，班长往往是班级委员会（班委会）的负责人，与[[中国共青团|团支部书记]]并列为班级中最高的学生干部，主要负责班级的全面工作，一般由[[教师|教师]]指定或由全班学生选举产生。

中国大陆的班委会往往设班长、副班长、学习委员、纪律委员、体育委员 、卫生委员、生活委员、劳动委员、文娱委员等职位，对应的[[中国共青团|团支部委员会]]另设组织委员、宣传委员等职位，负责具体班级事宜。

班长的职责一般包括
# 全面负责班级学生工作，领导班委会成员开展工作，配合同级团支部委员会的工作。
# 传达学校、班主任对班级的要求，并组织同学落实要求。
# 班主任在校时，听取班主任对班级管理的意见；班主任不在校时，代行班主任的职权。
# 了解同学们的思想和学习情况，并及时研究解决。

=== 台湾 ===
在台湾，班长或班代表是[[学校|学校]][[班聯會|班聯會]]成员之一，是[[班级|班级]]中最高的学生干部，主要负责班级的全面工作。一般由[[教师|教师]]指定或由全班学生选举产生。

现代学校的班级中常会举行“选举”，由班上同學選出班長、風紀股長、學藝股長、康樂股長、衛生股長等班幹部，由班長管理，班長由教師監督。班長管理班干部（例如風紀、衛生、學藝、總務等）及該班學生。

班长的职责一般包括，
#协助班級導師工作。
#组织开展各项班级集体活动。
#组织班级参加学校或年级集体活动。
#管理班干部工作。
#在某些学校负责收发“家庭聯絡簿”等。
#常被要求以身作则、帮助同学等。

== 政党组织 ==
班长是[[政党|政党]]的基层组织（如党支部）的负责人，引申为某些组织的部分成员的领导人。

== 军队 ==
班长是[[民兵|民兵]]团体、地方[[武装|武装]]或国家[[军队|军队]]中率领一班战士的[[士官|士官]]（通常，一个[[班|班]]下轄三個[[伍|伍]]，有9到15名[[士兵|士兵]]），一般由[[排长|排长]]或[[連長|連長]]等[[军官|军官]]指定。

班长在[[训练|训练]]、战斗和军营生活中负责管理本班战士（例如[[新兵訓練|新兵訓練]]課程的[[教育班長|教育班長]]），一般被要求以身作则、关心本班战士的生活等。

== 博彩业 ==
在澳門，荷官往往被稱呼為“班長”。擔當一個重要的角色。
