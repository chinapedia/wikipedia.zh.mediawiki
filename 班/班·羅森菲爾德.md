'''班·羅森菲爾德'''（{{lang-en|'''Ben Rosenfield'''}}，本名'''班傑明·「班」·羅森菲爾德'''（'''Benjamin "Ben" Rosenfield'''），{{bd|1992年|8月1日}}）是一名美國演員和音樂家。最知名於[[HBO|HBO]]的電視劇《[[酒私風雲|酒私風雲]]》第四、五季中飾演Willie Thompson。<ref>{{Cite web |url = http://www.hbo.com/boardwalk-empire/episodes/04/40-all-in/interview/ben-rosenfield.html |title = Interview With Ben Rosenfield |publisher = Home Box Office |language = en}}</ref><ref>{{Cite web |url = http://www.cinephiled.com/interview-ben-rosenfield-excesses-affluenza/ |title = Interview: Ben Rosenfield on the Excesses of ‘Affluenza’ |author = Danny Miller |date = 2014-07-24 |publisher = Cinephiled |language = en}}</ref>他於紀錄電影《{{link-en|少年 (電影)|Teenage (film)|少年}}》中飾演Tommie Scheel，音樂電影《{{link-en|紐約情弦|Song One}}》中飾演Henry Ellis，《{{link-en|富貴病 (電影)|Affluenza (film)|富貴病}}》中的Fisher Miller，以及愛情電影《{{link-en|我們的六年|6 Years}}》中的Dan Mercer。
