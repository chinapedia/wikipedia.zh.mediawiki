'''手樽站'''（{{jpn|j=手樽駅|hg=てたるえき|rm=Tetaru eki|f=y}}）是一位於[[日本|日本]][[宮城縣|宮城縣]][[松島町|松島町]]，隸屬於[[東日本旅客鐵道|東日本旅客鐵道]]（JR東日本）的[[鐵路車站|鐵路車站]]。

車站因2011年的[[東北地方太平洋沖地震|東北地方太平洋沖地震]]及引發的[[海嘯|海嘯]]影響，雖然整體車站結構及設施沒有太大受損，候車室及月台仍可保留使用，只有近出入口的月台表面有少量裂紋<ref>[https://dn3pm25xmtlyu.cloudfront.net/photos/large/269433613.jpg?1301814638&Expires=1434962648&Signature=N-Z48hL1cAL~ZLmSk~bR6VSgYndBGoaWqlslL12Ba3oHT4WMRx3Q9iQcIVAggBXBrvycfagvycpzKpLI5brig1cEKEm4TqQ0039jf9IYxUO4Vu~V6DWtvAbRTCTo6WzrBmPAZwucOS7RBUIJxu49XnCTKiTSSodtV6LGtEz-cpY_&Key-Pair-Id=APKAIYVGSUJFNRFZBBTA 手樽站近月台出入口的路面有少量裂痕。] {{webarchive|url=https://web.archive.org/web/20150621122855/https://dn3pm25xmtlyu.cloudfront.net/photos/large/269433613.jpg?1301814638&Expires=1434962648&Signature=N-Z48hL1cAL~ZLmSk~bR6VSgYndBGoaWqlslL12Ba3oHT4WMRx3Q9iQcIVAggBXBrvycfagvycpzKpLI5brig1cEKEm4TqQ0039jf9IYxUO4Vu~V6DWtvAbRTCTo6WzrBmPAZwucOS7RBUIJxu49XnCTKiTSSodtV6LGtEz-cpY_&Key-Pair-Id=APKAIYVGSUJFNRFZBBTA |date=2015-06-21 }}</ref>，但因為連接部份出現了大規模的破壞，令列車無法通行，因此須暫停營業，直至路段上其他受損的部份重建後，於2015年5月30日，隨全線復舊而重新投入服務。

== 車站結構 ==
現時採用簡易車站模式，不設有站舍，車站出入口設於月台末端向[[高城町站|高城町站]]方向，為無障礙斜道，並且在該處裝置了出、入站專用的[[Suica|Suica]]感應器。整個月台為全露天設計，有效長度為4輛，中央部份建有1座候車室，室內設置了數張座椅及1部乘車站證明書發行機。列車靠站時，往石卷方向為右側上落，往青葉通方向則為左側上落。

== 歷史 ==
* 1928年4月10日：{{link-ja|宮城電氣鐵道|宮城電気鉄道}}'''手樽站'''開業。
* 1944年5月1日：宮城電氣鐵道[[國有化|國有化]]，改為[[日本國有鐵道|日本國有鐵道]]車站，並改稱'''陸前富山站'''。
* 1952年11月22日：包括車站範圍內的用地，被日本政府根據「昭和28年文化財保護委員會告示第44號」規定，歸納為「特別名勝」內<ref name="MatsushimaPlan">{{PDFlink|[http://www.pref.miyagi.jp/bunkazai/matusima/matsushimahozonkanri.pdf 特別名勝松島保存管理計画]}}（宮城県 教育委員会 2010年3月）</ref>。
* 1956年：車站附近進行大規模[[围垦|围垦]]工程<ref name="reclamation">{{PDFlink|[http://www.pref.miyagi.jp/nosonshin/kouikisuirityousei/pdf/miyagi_nn_history.pdf みやぎの土地改良年表]}}（宮城県）</ref>，以增加該處用地。
* 1958年10月1日：無人化<ref>日本國有鐵道公示S33.9.27公345</ref>。
* 1968年：「手樽代行围垦事業」完成<ref name="reclamation"/>。
* 1987年4月1日：[[國鐵分割民營化|國鐵分割民營化]]、車站由JR東日本繼承。
* 2003年10月26日：可使用[[IC卡|IC卡]]「[[Suica|Suica]]」。
* 2011年3月11日：[[東北地方太平洋沖地震|東北地方太平洋沖地震]]，受海嘯影響而暫停營業。
* 2015年5月30日：仙石線全線復舊，再營業。但[[仙石東北線|仙石東北線]]列車並不在此停車<ref name="JReast1502">{{PDFlink|[http://www.jr-sendai.com/wp-content/uploads/2015/02/201505daiyakaisei.pdf 2015年5月 ダイヤ改正について]}} 東日本旅客鉄道 仙台支社（2015年2月26日）</ref>。
