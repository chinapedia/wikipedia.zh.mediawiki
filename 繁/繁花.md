[[File:The_Bookcover_of_Fanhua_(Shanghai_Literature_and_Art_Publishing_House,_2013).jpg|thumb]]于2013年出版的《繁花》单行本封面]]
《'''繁花'''》，原题为《'''上海阿宝'''》<ref name="shxs">{{cite web |author1=程德培 |title=程德培：我讲你讲他讲闲聊对聊神聊——《繁花》的上海叙事 |url=http://www.chinawriter.com.cn/wxpl/2015/2015-08-19/251192.html |website=中国作家网 |accessdate=2018-08-15}}</ref>，为《[[上海文学|上海文学]]》副主编[[金宇澄|金宇澄]]完成的一部[[话本|话本]]体[[长篇小说|长篇小说]]，以[[沪语|沪语]]风格描写上海市民生活。该小说创作伊始为[[网络小说|网络小说]]形式，发布于本地文化论坛“弄堂网”，后于《[[收获_(杂志)|收获]]》2012年秋冬长篇专号正式发表<ref>{{cite journal |author1=[[金宇澄|金宇澄]] |title=繁花 |journal=[[收获_(杂志)|收获]] |date=2012-09-23 |volume=2012年长篇专号（秋冬卷） |publisher=《收获》文学杂志社 |issn=0583-1288}}</ref>，小说单行本于2013年由[[上海文艺出版社|上海文艺出版社]]出版。

该书被认为确立了金宇澄在上海叙事白话小说上的历史地位，并获得2012年度中国小说排行榜长篇小说第一名、2013年首届[[鲁迅文化奖|鲁迅文化奖]]年度小说奖<ref name="新华悦读">{{cite web|website=新华悦读|url=http://news.xinhuanet.com/book/2014-03/26/c_126319658.htm|title=金宇澄谈成名：像老女人忽然怀孕 不自然|archiveurl=https://web.archive.org/web/20140422173931/http://news.xinhuanet.com/book/2014-03/26/c_126319658.htm |archivedate=2014-04-22|deadurl=yes|date=2014-03-26}}</ref>、第九屆[[茅盾文學獎|茅盾文學獎]]<ref>{{cite news|language=zh-hans |accessdate= 2015-08-16|title=第九届茅盾文学奖评奖办公室公告[2015年]第3号|url=http://www.chinawriter.com.cn/news/2015/2015-08-16/250886.html|date=2015-08-16|author= 第九届茅盾文学奖评奖办公室 |work=作协新闻|publisher=中国作家网}}</ref>等獎項。该作品后也被改编为[[舞台剧|舞台剧]]和[[苏州评弹|苏州评弹]]。

== 创作 ==
金宇澄欲于互联网上写一些无名者的市井事迹，故为自己在以上海话交流的本地文化论坛“弄堂网”注册了用户名“独上阁楼”，发表随笔。2011年5月10日开始，这部小说在该论坛文学版块开始连载<ref>{{cite web |author1=赵妍 |title=《繁花》作者金宇澄：耳闻的故事集中成了小说_频道_腾讯网 |url=http://cul.qq.com/a/20130408/000035.htm |website=腾讯文化 |publisher=时代周报 |accessdate=2018-08-15}}</ref>。在论坛网友的鼓励和建议下，金宇澄不断完善自己的文章，并开始考虑小说的结构问题。他每日写作，至当年11月4日完成初稿<ref>{{cite web |title=弄堂 longdang.org |url=http://www.longdang.org/index.php |website=弄堂网 |archiveurl=https://web.archive.org/web/20180815122802/http://www.longdang.org/index.php |archivedate=2018-08-15 |quote=自2011年5月10日11点42分 “独上阁楼（金宇澄）” 先生在 “弄堂论坛” 开帖创作 “繁花” ，每天创作一段并在 “弄堂论坛” 发布。2011年11月4日13点05分，论坛创作暂告一段落。}}</ref><ref name=":wenxuefanhua">{{cite web |author1=姜妍 |author2=吴月 |title=金宇澄和他的弄堂里开出的文学“繁花” |url=http://cul.sohu.com/20130422/n373571925.shtml |website=搜狐文化频道 |publisher=新京报}}</ref>。

金宇澄后将稿件交《收获》杂志社，杂志社建议其方言使用要让北方操[[官话|官话]]的读者也能看懂。金宇澄认可该观点，于是修改稿件，对语言进行了再创造。后应杂志社邀请，[[程德培|程德培]]和[[西飏|西飏]]对该文章做出评论，一并刊登于《收获》2012年秋冬长篇专号上<ref name=":wenxuefanhua" />。金宇澄在该书的跋中提及，他创作该作品时以口语形式铺陈，感兴趣的地方一是在当下的小说形态和旧文本之间的夹层，二则是在中国国民通识官话的当下以特定的方言内涵和样式转达文字信息的效果<ref name="fanhua">{{cite book |author1=金宇澄 |title=繁花 |date=2013 |publisher=[[上海文艺出版社|上海文艺出版社]] |location=上海 |isbn=978-7-5321-4800-4 |accessdate=2018-08-15}}</ref>{{rp|443-444}}。

该作品为上海市重大文艺创作项目，受到了[[上海文化发展基金会|上海文化发展基金会]]的资助，[[上海市作家协会|上海市作家协会]]也立项为重点创作资助项目<ref name="fanhua" />{{rp|封三}}。

== 内容与人物 ==
小说分为两个时间线书写，分为沪生的童年和当下。沪生和阿宝是小说的两个主要人物。阿宝的童年时间线上有小毛、姝华、银凤、大妹妹、兰兰、蓓蒂、绍兴阿婆、小珍、雪芝等人物。而沪生的当下时间线上则有陶陶、芳妹、梅瑞、小琴、阿宝、李李、汪小姐、葛先生、苏安、康总、玲子等人物<ref>{{cite journal |author1=刘涛 |title=花繁花衰——《繁花》论 |journal=艺术评论 |date=2014 |issue=05 |pages=105-109 |accessdate=2018-08-15 |publisher=[[中国艺术研究院|中国艺术研究院]] |location=北京 |issn=1672-6243 |doi= 10.16364/j.cnki.cn11-4907/j.2014.05.008}}</ref>。

《繁花》的叙事时间为上海的[[20世纪60年代|20世纪60年代]]至[[20世纪90年代|90年代]]，全文加上引言、尾声共计三十三章。其叙事分为两条线，一条线从[[1960年|1960年]]至[[文化大革命|文化大革命]]结束，另一条线则自[[1980年|1980年]]到[[21世纪|21世纪]]初，讲述了阿宝、沪生、小毛三个童年好友的上海往事，以十岁的阿宝开始，以中年的小毛去世结束，起于六十年代，终于九十年代。随着时间推移，最终两条线合二为一，归入了“海上”。<ref>{{cite journal |author1=黄平 |title=从“传奇”到“故事”——《繁花》与上海叙述 |journal=当代作家评论 |date=2013 |issue=04 |pages=54-62 |accessdate=2018-08-15 |publisher=当代作家评论杂志社 |location=辽宁 |issn=1002-1809 |doi= 10.16551/j.cnki.1002-1809.2013.04.014}}</ref>

== 语言风格 ==
金宇澄接受《[[南方都市报|南方都市报]]》采访时表示，《繁花》是一部基于上海话思维写成的小说，有别于普通话思维。该作品均采用方言句式完成，为防止拒读者于门外，故尽量改到与普通话同步，但仍存留了差异而存在沪语的语音特征。此外，金宇澄提到，他在找到一种合适的表达方式后，感觉很痛快。<ref name="nandu">{{cite web |author1=颜亮 |title=金宇澄专访：我用上海话思维写《繁花》 |url=http://cul.sohu.com/20130428/n374358498.shtml |website=搜狐文化频道 |publisher=南方都市报 |accessdate=2018-08-15}}</ref>

该小说在句式上以不分行的方式书写对白；用词上，该小说用了约1500个“不响”，甚至题记只有一句“上帝不响，像一切全由我定……”<ref name="fanhua" />{{rp|扉三}}，成为了小说显著的用词特征。金宇澄表示，他有意换一种写法，以这种方式写作，去除了内心描写，以对话替代。他认为：“……中国人最聪明，什么都懂了，什么都可以不响，小说可以大声疾呼，也该允许我一声不响。”<ref>{{cite web |title=1500个“不响”成就“金式语言” |url=http://cul.qq.com/a/20140218/019220.htm |website=腾讯文化 |accessdate=2018-08-15 |language=zh-CN}}</ref>

== 评价 ==
[[茅盾文学奖|茅盾文学奖]]评委[[王春林|王春林]]认为，按时序，关于上海的[[白话小说|白话小说]]有四位作家是「绝对绕不过去的」：[[韩邦庆|韩邦庆]]、[[张爱玲|张爱玲]]、[[王安忆|王安忆]]以及金宇澄，《繁花》一书确立了金宇澄的上海叙事白话小说的历史地位<ref name="新华悦读"/>；时任[[中国小说学会|中国小说学会]]会长、评论家雷达认为该小说是上海最好小说之一，并认为该小说对上海的伟大基点做了一次深度挖掘；《[[文艺报|文艺报]]》总编辑[[阎晶明|阎晶明]]认为，若该小说不运用方言风格，该小说不会成为经典作品；[[华东师范大学|华东师范大学]]教授[[陈子善|陈子善]]认为该作品应该放入[[海派文化|海派文化]]作品的脉络里来考察。<ref>{{cite web |author1=杨舟 |title=《繁花》被赞上海最好小说 专家称作者可比肩张爱玲 |url=http://www.xinhuanet.com/2013-10/25/c_125598454.htm |website=新华网 |publisher=新华社 |accessdate=2018-08-15 |date=2013-10-25}}</ref>

此外，[[程永新|程永新]]认为该作品建立了“一座与南方有关与城市有关的人情世态的博物馆”；[[程德培|程德培]]认为该小说恢复了小说的连载传统，是一部难得的、受上海各类人群喜爱的小说；[[郜元宝|郜元宝]]认为该作品没有明确的主线，并将这种结构和《[[红楼梦|红楼梦]]》相类比，认为难言这种结构是好是坏；[[李敬泽|李敬泽]]更是认为该作品深得《红楼梦》精髓，和中国古典小说一样描述了人生兴衰的过程；[[华东师范大学|华东师范大学]]教师毛尖认为该作品生产初期的模式回归了小说发展初期读者和作者交流并影响故事走向的模式，且作品中少用带“的”字的形容词，和上海本地生活不存在隔阂。<ref>{{cite news |title=文学名家眼中的《繁花》|work=[[人民日报|人民日报]]|date=2013-09-04|page=24|url=http://culture.people.com.cn/n/2013/0904/c87423-22799219.html |accessdate=2018-08-15}}</ref>

=== 舞台剧 ===
2018年1月26日至2月2日，[[舞台剧|舞台剧]]《繁花》第一季在上海[[美琪大戏院|美琪大戏院]]上演<ref name=":0">{{cite web |title=圆桌｜从小说到舞台剧，《繁花》的改编好在哪里 |url=http://news.163.com/18/0222/15/DB8Q0802000187VE.html |website=网易新闻 |accessdate=2018-08-15}}</ref>，引发轰动<ref name=":1">{{cite web |title=上海话演《繁花》，金宇澄不担心有障碍|url=http://www.bjnews.com.cn/ent/2018/06/14/491049.html |website=新京报网 |accessdate=2018-08-15}}</ref>；2018年6月21日至24日，舞台剧《繁花》在[[北京天桥艺术中心|北京天桥艺术中心]]连演六场<ref name=":2">{{cite web |title=《繁花》：何必为生命的片段哭泣，我们整个人生都催人泪下 |url=http://news.sina.com.cn/o/2018-07-09/doc-ihezpzwt8480817.shtml |website=新浪新闻 |accessdate=2018-08-15}}</ref>。金宇澄本人对该次改编并不担心；该舞台剧全程以[[沪语|沪语]]念白，金宇澄认为不会为观众带来理解障碍，反而会为该剧加分<ref name=":1" />。评论员[[今叶|今叶]]在《[[北京青年报|北京青年报]]》提出了批评，认为剧作成功转化了小说中的心理时空，但是未能在改变时对原作小说在剧场上演的效果进行考虑，以致未阅读原著的觉得情节琐碎、人物零散，也限制了原著读者的想象力<ref>{{cite web |author1=今叶 |title=把小说搬上舞台，问题开始了-新华网 |url=http://www.xinhuanet.com/book/2018-07/25/c_129919634.htm |website=www.xinhuanet.com |publisher=[[北京青年报|北京青年报]]}}</ref>。[[上海戏剧学院|上海戏剧学院]]教授[[荣广润|荣广润]]认为，该剧虽长达三个小时而不显枯燥，抓住原著主线；但剧本沿用双时空交错的方式却看不到两个时空的联系，让人遗憾；[[上海戏剧学院|上海戏剧学院]]教授[[丁罗男|丁罗男]]认为，第一季中蓓蒂和绍兴阿婆两个人物都未出现，是一种缺憾<ref name=":0" />。
