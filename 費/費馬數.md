{{unsolved|数学|当<math>n > 4</math>时，是否每个费马数都是[[合数|合数]]？}}
'''費馬數'''是以数学家[[费马|费马]]命名一组[[自然数|自然数]]，具有形式：

:<math>F_{n} = 2^{2^n} + 1</math>

其中''n''为非负整数。

若2<sup>''n''</sup> + 1是[[素数|素数]]，可以得到''n''必须是2的[[幂|幂]]。（若''n'' = ''ab''，其中1 < ''a'', ''b'' < ''n''且''b''为奇数，则2<sup>''n''</sup> + 1 ≡ (2<sup>''a''</sup>)<sup>''b''</sup> + 1 ≡ (−1)<sup>''b''</sup> + 1 ≡ 0('''mod''' 2<sup>''a''</sup> + 1)，即2<sup>''a''</sup> + 1是2<sup>''n''</sup> + 1的因數。）也就是说，所有具有形式2<sup>''n''</sup> + 1的素数必然是費馬數，这些素数称为'''費馬素數'''。已知的費馬素數只有''F''<sub>0</sub>至''F''<sub>4</sub>五個。

==基本性质==

费马数满足以下的[[递回关系|递回关系]]：

:<math>F_{n} = (F_{n-1}-1)^{2}+1\,</math>
:<math>F_{n} = F_{n-1} + 2^{2^{n-1}}F_{0} \cdots F_{n-2}</math>
:<math>F_{n} = F_{n-1}^2 - 2(F_{n-2}-1)^2</math>
:<math>F_{n} = F_{0} \cdots F_{n-1} + 2</math>

其中''n'' ≥ 2。这些等式都可以用[[数学归纳法|数学归纳法]]推出。从最后一个等式中，我们可以推出'''哥德巴赫定理'''：任何两个费马数都没有大于1的[[公因子|公因子]]。要推出这个，我们需要假设 0 ≤ ''i'' < ''j'' 且 ''F''<sub>''i''</sub> 和 ''F''<sub>''j''</sub> 有一个公因子 ''a'' > 1。那么 ''a'' 能把

:<math>F_{0} \cdots F_{j-1}</math>

和''F''<sub>''j''</sub>都整除；则''a''能整除它们相减的差。因为''a'' > 1，这使得''a'' = 2。造成矛盾。因为所有的费马数显然是奇数。作为一个推论，我们得到素数个数[[无穷|无穷]]的又一个证明。

其他性质：
*''F''<sub>''n''</sub>的位数''D''(''n'',''b'')可以表示成以''b'' 为[[进位制|基数]]就是

:<math>D(n,b) = \left\lfloor \log_{b}\left(2^{2^{\overset{n}{}}}+1\right)+1 \right\rfloor \approx \lfloor 2^{n}\,\log_{b}2+1 \rfloor </math> (参见[[高斯函数|高斯函数]]).
*除了F<sub>1</sub> = 2 + 3以外没有费马数可以表示成两个[[素数|素数]]的和。
*当''p''是奇素数的时候，没有费马数可以表示成两个数的''p''次方相减的形式。
*除了F<sub>0</sub>和F<sub>1</sub>，费马数的最后一位是7。
* 所有费马数{{OEIS|id=A051158}}的倒数之和是[[无理数|无理数]]。 ([[Solomon_W._Golomb|Solomon W. Golomb]], 1963)

== 历史 ==
1640年，[[费马|费马]]提出了一个猜想，認為所有的费马数都是素数。这一猜想对最小的5个費馬數成立，于是费马宣称他找到了表示素数的公式。然而，[[欧拉|欧拉]]在1732年否定了这一猜想，他给出了分解式：
:''F''<sub>5</sub> = 2<sup>32</sup> + 1 = 4294967297 = 641 × 6700417

歐拉證明費馬數的因數皆可表成''k''2<sup>''n''+1</sup> + 1，之後[[爱德华·卢卡斯|卢卡斯]]證明費馬數的因數皆可表成''k''2<sup>''n''+2</sup> + 1。

== 定理 ==
{{Main|可作图多边形}}
*[[卡爾·弗里德里希·高斯|高斯]]证明了：尺规作图[[正多边形|正多边形]]的边数目的[[充分条件|充分条件]]是2的非負整數次方乘以任意个（可为0个）不同的[[费马素数|费马素数]]的积，解决了兩千年来悬而未决的难题。這個條件也是[[必要條件|必要條件]]，但他沒有給出證明。

== 素性检验 ==

设<math>F_n=2^{2^n}+1</math>为第''n''个费马数。如果''n''不等于零，那么：

:<math>F_n</math>是素数，当且仅当<math>3^{\frac{F_n-1}{2}}\equiv-1\pmod{F_n}</math>。

===证明===

假设以下等式成立：
:<math>3^{\frac{F_n-1}{2}}\equiv-1\pmod{F_n}</math>
那么<math>3^{F_n-1}\equiv1\pmod{F_n}</math>，因此满足3<sup>k</sup>=1（mod<math>F_n</math>）的最小整数k一定整除<math>F_n-1=2^{2^n}</math>，它是2的幂。另一方面，k不能整除<math>\tfrac{F_n-1}{2}</math>，因此它一定等于<math>F_n-1</math>。特别地，存在至少<math>F_n-1</math>个小于<math>F_n</math>且与<math>F_n</math>互素的数，这只能在<math>F_n</math>是素数时才能发生。

假设<math>F_n</math>是素数。根据[[欧拉准则|欧拉准则]]，有：
:<math>3^{\frac{F_n-1}{2}}\equiv\left(\frac3{F_n}\right)\pmod{F_n}</math>,
其中<math>\left(\frac3{F_n}\right)</math>是[[勒让德符号|勒让德符号]]。利用重复平方，我们可以发现<math>2^{2^n}\equiv1\pmod3</math>，因此<math>F_n\equiv2\pmod3</math>，以及<math>\left(\frac{F_n}3\right)=-1</math>。因为<math>F_n\equiv1\pmod4</math>，根据[[二次互反律|二次互反律]]，我们便可以得出结论<math>\left(\frac3{F_n}\right)=-1</math>。
