'''費菲市'''是[[澳大利亞|澳大利亞]]雪梨大都會內，位於西南部的一個[[地方政府區域|地方政府]]。

== 沿革 ==
在歐洲人探索澳大利亞以前，[[澳大利亞原住民|原住民]]Cabrogal-Gandangara部族已於當地生活了3萬年。

== 市議會 ==
費菲市議會有議席12座，全市分3選區，各區選出議員4名代表市民和納稅人問政；市長為直選。<ref name=EC>{{cite web|url = http://www.lg.elections.nsw.gov.au/LGE2008/result.Fairfield.html|title = Fairfield City Council|work = 2008 Election results| publisher = Electoral Commission NSW| accessdate = 2009-06-16}}</ref>

==土地利用==
費菲市的住宅區與大片的工業區（Wetherill Park和Smithfield）為鄰；有4處主要的商業中心（費菲、卡巴瑪塔、Bonnyrigg、Prairiewood）；3座購物中心（費菲論壇、妮塔市、費菲蔡斯）。居中的「妮塔市」購物中心為當地帶來了大規模商機。由於費菲市內擁有多元族裔，許多咖啡館和餐廳的風格也各別洋溢著[[波斯尼亞|波斯尼亞]]式、[[義大利|義大利]]式、[[亞述|亞述]]式、[[西班牙|西班牙]]式和[[泰國|泰]]式等不同樣貌。

== 環境 ==
半世紀以來，市內大片的原生[[叢林|叢林]]經過土地管理與開發之後，已被清除。惟近年隨著地球溫度的上升、飛機與汽車排放廢氣、乾旱所造成的水質惡化，已促使市政府實施造林的政策以改善費菲市的居住環境。

== 學校 ==
費菲市內的學校包括有：Fairfield Public School、[[:en:Fairfield_High_School,_Sydney|費菲高中]]、St Johns Park High School、[[:en:Westfields_Sports_High_School|西區體育高中]]、[[:en:Fairvale_High_School|費維高中]]、[[:en:Yennora_Public_School|燕諾拉公立學校]]、Canley Vale High School等；天主教會學校有[[:en:Patrician_Brothers'_College,_Fairfield|Patrician Brothers' College]]、Our Lady of the Rosary Primary School、Mary Mackillop College、Freeman Catholic College、St Hurmizd's（[[東方亞述教會|東方亞述教會]]）primary school與Mar Narsai（[[東方亞述教會|東方亞述教會]]）college。

==媒體==
