'''溴化銫'''是一種[[無機化合物|無機]][[離子化合物|離子化合物]]，屬於[[溴化物|溴化物]]，為一種[[鹽類|鹽類]]，其[[化學式|化學式]]為CsBr，是一種[[白色|白色]]固體或[[白色|白色]]粒、塊狀物。 它具有簡單立方p型立方晶體結構，相當於[[氯化銫|氯化銫]]型，空間群為''Pm{{overline|3}}m'' ，晶格常數 為 ''a'' = 0.42953 nm. Cs和Br 的距離為0.37198 nm.

== 制備 ==
它可以透過下列反應製備：
===[[中和反應|中和反應]]===
將[[強鹼|強鹼]][[氫氧化銫|氫氧化銫]]和[[強酸|較強的酸]][[氫溴酸|氫溴酸]]混合[[酸鹼中和|酸鹼中和]]制得:
: CsOH (aq) + HBr (aq) → CsBr (aq) + H<sub>2</sub>O (l)
: Cs<sub>2</sub>(CO<sub>3</sub>) (aq) + 2 HBr (aq) → 2 CsBr (aq) + H<sub>2</sub>O (l) + CO<sub>2</sub> (g)

===直接合成===
直接將氣態溴和金屬銫放在一起，[[銫|銫]]會在[[氣態|氣態]][[溴|溴]]中[[燃燒|自燃]]
: 2 Cs (s) + Br<sub>2</sub> (g) → 2 CsBr (s)
直接合成是最快的方式，但由於成本高、實驗操作不易，所以一般不使用此方法。

== 用途 ==
'''溴化銫'''可作為[[分光鏡|分光鏡]]光學元件在長波段[[分光技術|分光光度儀]]。
