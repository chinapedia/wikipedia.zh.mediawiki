'''溴化镍'''是一种[[无机化合物|无机化合物]]，[[化学式|化学式]]为NiBr<sub>2</sub>，易溶于水。
==制备==
溴化镍的无水物可由单质在室温下的[[乙醚|乙醚]]中反应得到：<ref name=w />
:Ni + Br<sub>2</sub> → NiBr<sub>2</sub>
利用[[氢溴酸|氢溴酸]]和[[氢氧化镍|氢氧化镍]]、[[碳酸镍|碳酸镍]]反应，蒸發结晶，可以得到溴化镍的水合物：
: 2 HBr + Ni(OH)<sub>2</sub> → NiBr<sub>2</sub> + 2H<sub>2</sub>O
: 2 HBr + NiCO<sub>3</sub> → NiBr<sub>2</sub> + H<sub>2</sub>O + CO<sub>2</sub>↑

[[1,2-二溴乙烷|1,2-二溴乙烷]]和30%的[[氯化镍|氯化镍]]水溶液反应，可以制备溴化镍。这一反应避免了使用具有强腐蚀性的氢溴酸。反应时1,2-二溴乙烷过量至3倍摩尔比，以[[四丁基溴化铵|四丁基溴化铵]]为[[相转移催化剂|相转移催化剂]]，不额外加溶剂：<ref>张敬畅 等. 溴化镍制备方法的改进. 北京化工大学学报, 1998. 25(2): 64-68</ref>
: NiCl<sub>2</sub> + CH<sub>2</sub>BrCH<sub>2</sub>Br → NiBr<sub>2</sub> + CH<sub>2</sub>ClCH<sub>2</sub>Cl

==用途==
溴化镍可以和α-二亚胺形成配合物，用于催化乙烯聚合。<ref>义建军, 许学翔, 陈伟 等. 镍配合物[N,N]NiBr<sub>2</sub>的合成及其催化乙烯齐聚研究. 催化学报, 2002. 23(1): 69-71</ref>
