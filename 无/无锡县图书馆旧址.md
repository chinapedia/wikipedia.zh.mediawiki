'''无锡县图书馆旧址'''位于[[中国|中国]][[江苏省|江苏省]][[无锡市|无锡市]][[崇安区|崇安区]][[崇安寺_(无锡)|崇安寺]]商业步行街东侧，建造于民国初年，为一座中西合璧的四层砖木结构楼房，南侧为二泉映月广场，塑有[[钱绍武|钱绍武]]设计制作的[[阿炳|阿炳]]铜像。<ref>{{cite web|url=http://news.thmz.com/col63/2005/04/2005-04-2941847.html|title=〈瞎子阿炳铜像铸成〉|accessdate=2012年2月21日|work=|publisher=太湖明珠网}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

== 简介 ==
[[File:Wuxi_Old_Library_1920s.jpg|left]]

该建筑位于原无锡县城内的道观[[无锡洞虚宫|洞虚宫]]三清殿基址之上，辛亥革命后，1912年4月，无锡名流丁宝书、顾倬、侯鸿鉴、秦玉书等十二人发起，俞仲还等十四人赞助，经无锡军政府批准，并委任秦琢如、顾倬为筹建经董，于同年10月在三清殿旧址上兴建无锡县立图书馆。1914年年底建成，建筑面积1300多平方米，为当时无锡境内最高的楼阁式建筑，国学大师[[钱基博|钱基博]]亲任馆长。楼顶设有大自鸣钟，是无锡全城的校时标准，也堪称为当时无锡的地标。<ref>{{cite web |url=http://www.1street.com.cn/culture/historical/2011/1222/108.html|title=《崇安寺街区史略》|accessdate=2011年12月22日|work=小漪|publisher=崇安寺官方网站}}</ref><ref>{{cite web|url=http://news.thmz.com/col89/2010/01/2010-01-18688156.html|title=〈无锡地标：20米到339米〉|accessdate=2012年2月21日|work=|publisher=《无锡商报》|deadurl=yes|archiveurl=https://web.archive.org/web/20100122133330/http://news.thmz.com/col89/2010/01/2010-01-18688156.html|archivedate=2010年1月22日}}</ref>

1915年元旦，图书馆正式开馆，举办典礼，无锡县知事丁万谷致开幕词，观者达千人。1月5日，图书馆开门迎客，第一天到馆者27人，时任职教育司的[[黄炎培|黄炎培]]，亦亲临参观。当时开门时间自下午一点至五点，“不拘工农士商，只要识字的人，花费一二个铜元，就能进去看书。”<ref>{{cite web |url=|title=《崇安名胜史话》|pages=页67-70|accessdate=2006年12月|work=政协无锡市崇安区委员会、无锡市崇安区档案局编|publisher=山东画报出版社}}</ref>

这座被无锡人称为“大自鸣钟楼”的图书馆，是中国最早创建的公立图书馆之一，仅比[[国家图书馆|国家图书馆]]的前身“国立京师图书馆”晚3年，也一直承担着图书馆功能，直到2000年才迁至金匮桥边的图书馆新址。目前，钟楼已是江苏省文保单位，并被辟为无锡历史文献馆。<ref>{{cite web|url=http://news.thmz.com/col89/2009/03/2009-03-18512905.html|title=〈锡城崇安寺大自鸣钟“重出江湖”〉|accessdate=2009年3月18日|work=|publisher=《江南晚报》|deadurl=yes|archiveurl=https://web.archive.org/web/20150218205009/http://news.thmz.com/col89/2009/03/2009-03-18512905.html|archivedate=2015年2月18日}}</ref>

现建筑底层墙壁上存有记述建筑始末的《无锡图书馆碑记》一通，由[[钱基博|钱基博]]撰文、[[俞复|俞复]][[书丹|书丹]]。<ref>{{cite web |url=|title=《江苏文物综录》|pages=页303|accessdate=1988年10月|work=《江苏文物综录》编辑委员会编|publisher=南京博物院}}</ref>

== 逸事 ==

无锡县图书馆旧址的大自鸣钟每到整点或半小时，便会定时报时。1918年11月12日，为庆祝[[第一次世界大战|第一次世界大战]]结束，无锡县图书馆钟楼长鸣一百响，盛况空前。<ref>{{cite web|url=http://xh.xhby.net/mp2/html/2012-05/31/content_575970.htm|title=〈大自鸣钟〉|accessdate=2012年5月31日|work=|publisher=《新华日报》|deadurl=yes|archiveurl=https://web.archive.org/web/20150205045004/http://xh.xhby.net/mp2/html/2012-05/31/content_575970.htm|archivedate=2015年2月5日|df=}}</ref>
