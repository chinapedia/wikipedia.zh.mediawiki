'''無平方数因数的数'''（{{lang-en|square-free integer}}）是指其[[因數|因數]]中，沒有一個是[[平方數|平方數]]的[[正整數|正整數]]。簡言之，將一個這樣的數予以[[整數分解|質因數分解]]後，所有[[質因數|質因數]]的[[冪|冪]]都不會大於或等於2。例如：[[54|54]]={{質因數分解|54}}，由於54有因數是平方數（[[9|<math>3^2=9</math>]]），所以54不是无平方数因数的数；而[[55|55]]={{質因數分解|55}}，55沒有因數是平方數，所以55是无平方数因数的数。

以數學概念說明：若一個數<math>R</math>是無平方数因数的数，則對於任意平方數<math>S^2</math>且<math>S^2 \le R</math>則<math>S^2 \nmid R</math>；或者說當<math>R = P_1 P_2 P_3 ... P_n</math>且<math>P_1, P_2, P_3, ..., P_n</math>皆為[[質數|質數]]時，對於任意<math>1 \le i,j \le n</math>，<math>i \ne j</math>而言，<math>P_i \ne P_j</math>

另一方面，[[默比乌斯函数|默比乌斯函数]]<math>\mu (n) \ne 0</math>當且僅當<math>n \ge 1</math>且<math>n = 1</math>或<math>n</math>為無平方数因数的数時

前20個無平方因數的數是：[[1|1]]、[[2|2]]、[[3|3]]、[[5|5]]、[[6|6]]、[[7|7]]、[[10|10]]、[[11|11]]、[[13|13]]、[[14|14]]、[[15|15]]、[[17|17]]、[[19|19]]、[[21|21]]、[[22|22]]、[[23|23]]、[[26|26]]、[[29|29]]、[[30|30]]、[[31|31]]{{OEIS|id=A005117}}

由於「無平方数因数的数」的所有質因數指數均為一次方，故除1以外，有關數的正因數數目必定是[[2的冪|2的非負整數次方]]。

將无平方数因数的数分解為兩數之積，這兩數一定[[互質|互質]]。

依定義，顯然所有的[[質數|質數]]、[[楔形数|楔形数]]、[[質數階乘|質數階乘]]與有4個正因數的[[半質數|半質數]]都是无平方数因数的数。

==不含平方因子的数的分布==

如果用''Q''(''x'')来表示1和''x''之间的不含平方因子的数，则：

:<math>Q(x) = \frac{6x}{\pi^2} + O(\sqrt{x})</math>

因此，不含平方因子的数的[[自然密度|自然密度]]为：

:<math>\lim_{x\to\infty} \frac{Q(x)}{x} = \frac{6}{\pi^2} = \frac{1}{\zeta(2)}</math>

其中ζ是[[黎曼ζ函数|黎曼ζ函数]]。

类似地，如果用''Q''(''x'',''n'')来表示1和''x''之间的不含''n''次方因子的数，则我们可以证明：

:<math>\lim_{x\to\infty} \frac{Q(x,n)}{x} = \frac{1}{\zeta(n)}.</math>
