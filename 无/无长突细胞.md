'''无长突细胞'''是[[视网膜|视网膜]]中的[[中間神經元|中间神经元]]。<ref>{{Template:Cite journal|last1 = Kolb|first1 = H|last2 = Kolb|first2 = H|last3 = Fernandez|first3 = E|last4 = Nelson|first4 = R|title = Roles of Amacrine Cells|date = 1995|pmid = 21413397}}</ref>无长突细胞是抑制性神经元，其的树突分叉延伸至内丛状层（IPL），在此与视网膜神经节细胞和（或）双极细胞相接触。<ref name="Balas 2014">{{Template:Cite journal|pmid = 25170430|pmc = 4142557|year = 2014|author1 = Balasubramanian|first1 = R|title = Development of Retinal Amacrine Cells and Their Dendritic Stratification|journal = Current Ophthalmology Reports|volume = 2|issue = 3|pages = 100-106|last2 = Gan|first2 = L|doi = 10.1007/s40135-014-0048-2}}</ref>

== 概述 ==
无长突细胞发挥作用是在[[内丛状层|内丛状层]]（IPL），那是[[视网膜|视网膜]]的第二层[[突触|突触]]结构，其突触由[[双极细胞|双极细胞]]和[[视网膜神经节细胞|视网膜神经节细胞]]构成。根据其[[树突|树突]]形态和层级分类，无长突细胞就至少有33种不同的亚型。和水平细胞一样，无长突细胞将信息传递，不同的是，水平细胞输出来自[[视杆细胞|视杆细胞]]和[[视锥细胞|视锥细胞]]的信息，而无长突细胞则负责输出[[双极细胞|双极细胞]]的信息，且特异性更高。各型无长突细胞都释放一种或多种[[神经递质|神经递质]]来与其他细胞相联系。<ref name="Balas 2014">{{Template:Cite journal|pmid = 25170430|pmc = 4142557|year = 2014|author1 = Balasubramanian|first1 = R|title = Development of Retinal Amacrine Cells and Their Dendritic Stratification|journal = Current Ophthalmology Reports|volume = 2|issue = 3|pages = 100-106|last2 = Gan|first2 = L|doi = 10.1007/s40135-014-0048-2}}</ref>

无长突细胞通常根据它们的宽度以及它们的联系腹地在内丛状层的哪一阶层来分类，还可根据它们的神经递质类型来区分。大多抑制性神经元都以[[γ-氨基丁酸|γ-氨基丁酸]]或[[甘氨酸|甘氨酸]]为递质。

== 分型 ==
如上所述，对无长突细胞的亚型分类有多种方法。

'''γ-氨基丁酸能，甘氨酸能，或其他：'''无长突细胞根据它们所释放的神经递质分为可以分为γ-氨基丁酸能，甘氨酸能或其他类型（γ-氨基丁酸，某氨酸或二者皆非）。γ-氨基丁酸能无长突细胞通常都有较大的腹地，可以在[[神经节细胞层|神经节细胞层]]（GCL）和[[内核层|内核层]]（INL）中找到。有一种γ-氨基丁酸能无长突细胞被研究得较深入，即星暴无长突细胞。这种无长突细胞以表达[[胆碱乙酰转移酶|胆碱乙酰转移酶]]（ChAT）为特征，并被认为对方向选择和方向性活动的监测有一定作用。<ref name="Balas 2014">{{Template:Cite journal|pmid = 25170430|pmc = 4142557|year = 2014|author1 = Balasubramanian|first1 = R|title = Development of Retinal Amacrine Cells and Their Dendritic Stratification|journal = Current Ophthalmology Reports|volume = 2|issue = 3|pages = 100-106|last2 = Gan|first2 = L|doi = 10.1007/s40135-014-0048-2}}</ref> 其他无长突细胞也释放乙酰胆碱，但它们的功能并未被研究清楚。<ref name="Masland 2012">{{Template:Cite journal|pmid = 22416289|pmc = 3652807|year = 2012|author1 = Masland|first1 = R. H.|title = The tasks of amacrine cells|journal = Visual neuroscience|volume = 29|issue = 1|pages = 3-9}}</ref> 另一种γ-氨基丁酸能无长突细胞即为多巴胺能细胞，它们都表达单胺合成酶酪氨酸羟化酶（TH），调节光适应和昼夜节律。<ref name="Balas 2014">{{Template:Cite journal|pmid = 25170430|pmc = 4142557|year = 2014|author1 = Balasubramanian|first1 = R|title = Development of Retinal Amacrine Cells and Their Dendritic Stratification|journal = Current Ophthalmology Reports|volume = 2|issue = 3|pages = 100-106|last2 = Gan|first2 = L|doi = 10.1007/s40135-014-0048-2}}</ref> 它们都是广泛延伸的无长突细胞，且它们弥漫性释放多巴胺，且同时释放γ-氨基丁酸及其他所有突触常规释放的递质。<ref name="Masland 2012">{{Template:Cite journal|pmid = 22416289|pmc = 3652807|year = 2012|author1 = Masland|first1 = R. H.|title = The tasks of amacrine cells|journal = Visual neuroscience|volume = 29|issue = 1|pages = 3-9}}</ref> 其他许多γ-氨基丁酸能无长突细胞的亚型也被有注意到，但以上所列皆是被研究及讨论得最广少的类型。

甘氨酸能无长突细胞并不如γ-氨基丁酸能无长突细胞那样研究得广泛和明确。所有甘氨酸能无长突细胞都被认为以[[甘氨酸转运蛋白抑制剂|甘氨酸转运蛋白抑制剂]]（GlyT1）为特征性标记。有一种被研究得较明确的甘氨酸能无长突细胞仍被命名为无长突细胞（''Amacrine cell，首字母大写表示专有名词，区别于普通无长突细胞的amacrine cell的拼写——译注''）。这种细胞出现在内核层（INL）。<ref name="Balas 2014">{{Template:Cite journal|pmid = 25170430|pmc = 4142557|year = 2014|author1 = Balasubramanian|first1 = R|title = Development of Retinal Amacrine Cells and Their Dendritic Stratification|journal = Current Ophthalmology Reports|volume = 2|issue = 3|pages = 100-106|last2 = Gan|first2 = L|doi = 10.1007/s40135-014-0048-2}}</ref> AII无长突细胞的一个重要功能就是，它们从视杆双极细胞中获取细胞输入信息，并通过视锥双极细胞的突触后膜将信息重新分布其内，行使其适应功能。<ref name="Marc 2014">{{Template:Cite journal|pmid = 25237297|pmc = 4154443|year = 2014|author1 = Marc|first1 = R. E.|title = The AII amacrine cell connectome: A dense network hub|journal = Frontiers in Neural Circuits|volume = 8|pages = 104|last2 = Anderson|first2 = J. R.|last3 = Jones|first3 = B. W.|last4 = Sigulinsky|first4 = C. L.|last5 = Lauritzen|first5 = J. S.|doi = 10.3389/fncir.2014.00104}}</ref>

大约15%的无长突细胞即不是γ-氨基丁酸能也不是甘氨酸能。<div><ref name="Balas 2014">{{Template:Cite journal|pmid = 25170430|pmc = 4142557|year = 2014|author1 = Balasubramanian|first1 = R|title = Development of Retinal Amacrine Cells and Their Dendritic Stratification|journal = Current Ophthalmology Reports|volume = 2|issue = 3|pages = 100-106|last2 = Gan|first2 = L|doi = 10.1007/s40135-014-0048-2}}</ref>这些无长突细胞有时被称为nGnG无长突细胞，有研究认为作用于亲代无长突细胞的复制因子决定着无长突细胞的分型去向。一种在nGnG无长突细胞中的特异性表达的复制因子为Neurod6  <ref name="Kay 2011">{{Template:Cite journal|pmid = 21743471|pmc = 3144989|year = 2011|author1 = Kay|first1 = J. N.|title = Neurod6 expression defines new retinal amacrine cell subtypes and regulates their fate|journal = Nature Neuroscience|volume = 14|issue = 8|pages = 965-72|last2 = Voinescu|first2 = P. E.|last3 = Chu|first3 = M. W.|last4 = Sanes|first4 = J. R.|doi = 10.1038/nn.2859}}</ref></div>

'''树突分枝的长度：'''根据树突的长度和分布，无长突细胞可分为狭域无长突细胞（直径约70mm）、中域无长突细胞（直径约170mm），和广域无长突细胞（直径约350mm）。<ref name="Balas 2014">{{Template:Cite journal|pmid = 25170430|pmc = 4142557|year = 2014|author1 = Balasubramanian|first1 = R|title = Development of Retinal Amacrine Cells and Their Dendritic Stratification|journal = Current Ophthalmology Reports|volume = 2|issue = 3|pages = 100-106|last2 = Gan|first2 = L|doi = 10.1007/s40135-014-0048-2}}</ref> 不同的长度决定了这些无长突细胞能行使的不同特殊功能。狭域无长突细胞垂直沟通视网膜各层，同时辅助神经节细胞在其接受范围内形成功能单元。狭域无长突细胞及其与这些功能单元的交叠使得神经节细胞能感知视野中极小物点的细微运动。一种行使这类功能的狭域无长突细胞即为星暴无长突细胞<ref name="Masland 2012">{{Template:Cite journal|pmid = 22416289|pmc = 3652807|year = 2012|author1 = Masland|first1 = R. H.|title = The tasks of amacrine cells|journal = Visual neuroscience|volume = 29|issue = 1|pages = 3-9}}</ref>

中域无长突细胞也能垂直沟通视网膜各层，但它们更突出的功能仍然未知。由于它们的树突分树的大小和神经节细胞非常相似，它们可能在显微视野中与神经节细胞相混。同理，广域无长突细胞研究起来更困难，甚至连发现都困难，因为它们的分散在视网膜全层，因此视野中数量比较少。不过根据尺寸大小来推测，它们的一个主要功能是在单层中横向联系，虽然有些也参与层间的垂直联系。<ref name="Masland 2012">{{Template:Cite journal|pmid = 22416289|pmc = 3652807|year = 2012|author1 = Masland|first1 = R. H.|title = The tasks of amacrine cells|journal = Visual neuroscience|volume = 29|issue = 1|pages = 3-9}}</ref>

== 功能 ==
在多数情况下，无长突细胞的不同亚型都和它们的功能密切相关（结构决定功能）。然而某些视网膜无长突细胞的特殊功能仍值得一叙。
* 在内丛状层中拦截视网膜神经节细胞和（或）双极细胞<ref name="Balas 2014">{{Template:Cite journal|pmid = 25170430|pmc = 4142557|year = 2014|author1 = Balasubramanian|first1 = R|title = Development of Retinal Amacrine Cells and Their Dendritic Stratification|journal = Current Ophthalmology Reports|volume = 2|issue = 3|pages = 100-106|last2 = Gan|first2 = L|doi = 10.1007/s40135-014-0048-2}}</ref>
* 在许多神经节细胞的接受腹地内形成功能单元
* 为视网膜各层提供垂直联系
* 行使旁泌功能，如释放[[多巴胺|多巴胺]]和[[乙酰胆碱|乙酰胆碱]]<ref name="Masland 2012">{{Template:Cite journal|pmid = 22416289|pmc = 3652807|year = 2012|author1 = Masland|first1 = R. H.|title = The tasks of amacrine cells|journal = Visual neuroscience|volume = 29|issue = 1|pages = 3-9}}</ref>
* 通过它们的突触与视网膜其它细胞的联系及它们释放的神经递质，监测方向性活动，调节光适应和[[昼夜节律|昼夜节律]]<ref name="Balas 2014">{{Template:Cite journal|pmid = 25170430|pmc = 4142557|year = 2014|author1 = Balasubramanian|first1 = R|title = Development of Retinal Amacrine Cells and Their Dendritic Stratification|journal = Current Ophthalmology Reports|volume = 2|issue = 3|pages = 100-106|last2 = Gan|first2 = L|doi = 10.1007/s40135-014-0048-2}}</ref> ，并通过和视杆、视锥细胞的联系，控制[[暗视觉|暗视觉]]中的高敏感度 <ref name="Marc 2014">{{Template:Cite journal|pmid = 25237297|pmc = 4154443|year = 2014|author1 = Marc|first1 = R. E.|title = The AII amacrine cell connectome: A dense network hub|journal = Frontiers in Neural Circuits|volume = 8|pages = 104|last2 = Anderson|first2 = J. R.|last3 = Jones|first3 = B. W.|last4 = Sigulinsky|first4 = C. L.|last5 = Lauritzen|first5 = J. S.|doi = 10.3389/fncir.2014.00104}}</ref>
不同无长突细胞的不同功能仍有许多问题有待探索。它们广泛延伸的树突被认为能通过双极细胞及神经节细胞的反馈形成抑制性围合区。因此有观点认为它们对水平细胞的活动有辅助作用。

其他类型的无长突细胞则更多地行使调节作用，如调节适应[[明视觉|明视觉]]和[[暗视觉|暗视觉]]的的敏感度。AII无长突细胞就是暗视觉中视杆细胞的信号传递者。<ref name="Marc 2014">{{Template:Cite journal|pmid = 25237297|pmc = 4154443|year = 2014|author1 = Marc|first1 = R. E.|title = The AII amacrine cell connectome: A dense network hub|journal = Frontiers in Neural Circuits|volume = 8|pages = 104|last2 = Anderson|first2 = J. R.|last3 = Jones|first3 = B. W.|last4 = Sigulinsky|first4 = C. L.|last5 = Lauritzen|first5 = J. S.|doi = 10.3389/fncir.2014.00104}}</ref>

== 组织 ==
无长突细胞和其他视网膜中间神经元大都通常偶然出现，而不是和同类细胞邻接排列，故周围会形成一个“排它空间”使它们疏离开来。这种镶嵌式的排列方式能合各种细胞在视网膜上均匀分布，确保了视野各区域都能有完整的信息传导元素。<ref>{{Template:Cite journal|last = Wassle|first = H.|author2 = Riemann, H. J.|title = The Mosaic of Nerve Cells in the Mammalian Retina|journal = Proceedings of the Royal Society B: Biological Sciences|date = 22 March 1978|volume = 200|issue = 1141|pages = 441–461|doi = 10.1098/rspb.1978.0026}}</ref> 有小鼠模型研究显示，MEGF10和MEGF11跨膜蛋使无长突细胞和水平细胞呈星暴式分布，对这种镶嵌组织形式有重要作用。<ref>{{Template:Cite journal|last = Kay|first = Jeremy N.|author2 = Chu, Monica W.|author3 = Sanes, Joshua R.|title = MEGF10 and MEGF11 mediate homotypic interactions required for mosaic spacing of retinal neurons|journal = Nature|volume = 483|issue = 7390|pages = 465–9|date = March 2012|doi = 10.1038/nature10877|pmid = 22407321}}</ref>
