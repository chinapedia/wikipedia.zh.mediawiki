'''无限元素巡回演唱会'''（{{lang-en|The Freedom Tour}}）是美国[[创作歌手|创作歌手]][[艾莉西亚·凯斯|艾莉西亚·凯斯]]的第四场演唱会，以宣传她的第四张专辑《[[无限元素|无限元素]]》。演唱会始于2010年2月26日，[[渥太华|渥太华]]的[[加拿大丰业银行|加拿大丰业银行]]中心。这次演唱会将持续到北美及至欧洲。

==票房收入==
2010年上半年，艾莉西亚·凯斯的演唱会在北美取得11.7百万美元的巡演收入<ref>{{Cite web |url=http://www.pollstarpro.com/specialfeatures2010/2010MidYearTop100Tours.pdf |title=2010MidYearTop100Tours |accessdate=2010-07-31}}</ref>，平均票价78美元，总共卖出近15万张门票，全球数据则是22.0百万美元<ref>{{Cite web |url=http://www.pollstarpro.com/specialfeatures2010/2010MidYearTop50WorldwideTours.pdf |title=2010MidYearTop50WorldwideTours |accessdate=2010-07-31}}</ref>，。
