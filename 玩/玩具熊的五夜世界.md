{{Vgname|玩具熊的五夜世界|en=FNaF World}}是[[史葛·戈晃|史葛·戈晃]]（Scott Cawthon）在2016年1月21日透過遊戲平台[[Steam|Steam]]發表的[[獨立遊戲|獨立遊戲]]，類型為單人玩家以指向與點擊操作的[[角色扮演遊戲|角色扮演遊戲]]，為《[[玩具熊的五夜後宮|玩具熊的五夜系列]]》之衍生作品。

但因其故事并不完整，且内有大量[[程序错误|程序错误]]，一段时间后游戏就从Steam下架了。在2016年2月8日，这个游戏被更新并上架至[[Game_Jolt|Game Jolt]]。

==故事 ==
在一個全新的六層世界裡，《[[玩具熊的五夜後宮|玩具熊的五夜系列]]》以迷你姿態聚首一堂，踏上對抗邪惡敵人、發掘心儀夥伴與覓尋世界奧秘的冒險旅程。當一切變得理所當然之時，不同的派對組合和行走路向，將讓玩家發現看似獨立的世界，如何影響著其他世界，與及怎樣被其他世界影響。

==遊戲形式 ==
[[File:玩具熊的五夜世界_战斗.jpeg|缩略图]]

玩家可以選擇固定組合和自由組合8名起始角色，參與中等或高級難度的遊戲。期間可以選擇跟隨引導角色指示到達標準結局，亦可在不同位置達成特別結局或隱藏內容。旅程中途要操控角色穿越時空縫隙到達不同層數，亦會隨機觸發與敵對角色的指令式戰鬥，完成後將可得到貨幣「費斯金幣」（Faztokens），可用來購入輔助工具；部分戰鬥則會為帶來新的可玩角色，合共超過40名。此外，亦有一些操作截然不同的迷你遊戲。

==== 其他角色 ====

; 絨毛玩偶骨架（EndoPlush）
: 以絨毛陷阱為基礎出現的新角色。
; Animdude
: 來自同一作者製作之遊戲《There Is No Pause Button》，亦是史葛·戈晃本人的社交頭像。可以變為可使用角色。
; 咖啡機（Coffee）
: 來自同一作者製作之遊戲《The Desolate Hope》。
; 削片先生（Mr. Chipper）
: 來自同一作者製作之遊戲《Chipper & Sons Lumber Co.》。在某處以大型機械人姿態抱怨被本作系列搶去一切。可以變為可使用角色。

==== 非玩家角色 ====

; 虛擬佛萊迪（Virtua-Freddy）
: 只在特定武器下短暫出現，複製玩家最新一次動作。
; 盧比特（Lolbit）
: 給玩家提供攻擊用輔助工具「位元組昆蟲」（Bytes）。
; 蒂蒂（DeeDee）
: 給玩家提供垂釣珠寶遊戲，以賺取「費斯金幣」。
; 曼道（Mendo）
: 給玩家提供防禦用輔助工具「骨架身軀」。
; 8位元弗雷德熊（8-bit Fredbear）
: 於玩家閒置對話時閃現，並帶出影響其他世界的隱藏劇情。
; 惡果老人（Old Man Consequences）
: 在某處告知不按指示行動的玩家要承受惡果。
; 史葛·戈晃（Scott Cawthon）
: 於小遊戲裏作為BOSS，向玩家介紹未來會推出的續集。只有一個頭，但攻擊力驚人。
; 桌前男人（Desk Man）
: 在某處拼命工作的遊戲設計師，抱怨著其創作變質。
: 有研究推斷角色來自系列小說《銀色的雙眼》。
; 推魂者（Souldozer）
: 來自同一作者製作之遊戲《The Pilgrim's Progress》。
; 積卡的魔法彩虹（Chica's Magic Rainbow）
: 表裡不一的彩虹，為本遊戲最終Boss，不停嘲諷玩家，攻擊力驚人。
; 貝比（Baby）
: 於「姊妹地點篇」登場，但同時貫穿本作內容的隱藏角色。在結局殺死了桌前男人。
