== 倉田家 ==
; {{nihongo|倉田紗南|倉田紗南（くらた さな）||配音員：{{flagicon|JPN}} [[小田靜枝|小田靜枝]] / [[横山智佐|横山智佐]]；{{flagicon|TWN}} [[謝佼娟|謝佼娟]]（CN卡通頻道）／[[林美秀_(配音員)|林美秀]]（MOMO親子台）；{{flagicon|HKG}} [[吳小藝|吳小藝]]}}
: 本劇中心人物。1983年3月7日出生（※動畫是1985年）。[[雙魚宮|雙魚座]]。[[ABO血型系統|O型]]。是小迴劇團（又譯：小向日葵劇團）的人氣童星。最初主要參與綜藝節目的演出，由於演技備受好評，因此成為劇集和電影的演員。個性極為外向、活潑，行為舉止誇張，且有些早熟，十分重視身邊的人；時常將[[成語|成語]]讀錯（如把「現學現賣」讀作「先學先賣」、「合情合理」讀作「可情可理」等）。
: 實際上出生不久就被遺棄，是一名[[棄嬰|棄嬰]]，與母親實紗子並無血緣關係。由於是在3月7日被發現（後來發現該天也是出生日），因此名字被取為紗南<ref>[[日文|日文]]中3月7日的讀法為{{lang|ja|'''さ'''んがつ'''な'''のか}}，而紗南為{{lang|ja|'''さな'''}}。</ref>。紗南曾開玩笑說如果是3月6日出生的話，便要改名作山姆<ref>[[日文|日文]]中3月6日的{{lang|ja|'''さ'''んがつ'''む'''いか}}，而山姆為{{lang|ja|'''さむ/サム'''}}。</ref>。
: 一開始將經紀人相模玲當作自己的男朋友，在得知自己仰慕的女演員來海麻子是他的前女友後便視她為情敵。但是後來被看不下去的實紗子點破，了解到這只是大人給予小孩子的美夢而大受打擊。後來在放下對阿玲的感情後成全他與麻子復合。
: 原本就是個對情感欠缺敏感度的人，在經歷對阿玲的誤解後對戀愛更是失去了自信，害怕又會是自己的自作多情而變得遲疑，以致沒有發現羽山對自己的感情，也不確定自己對羽山的感覺到底算不算愛情。直至羽山和風花交往後才了解自己對羽山的心意，但因為無法背叛身為好友的風花而選擇退讓，並以不斷的工作來逃避兩人，動畫版中甚至為此以「與直澄一同到紐約進修」為名逃出國。
: 原作漫畫版中，紗南對羽山的思念反而因逃避而日益遽增，最後選擇回到學校面對，正式對風花表明自己喜歡羽山的感情，更表示總有一天要搶走羽山。直到在小森同學的事件中，風花在了解兩人的過去後選擇退讓，在事件結束後終於與羽山成為情侶。不久，因為羽山要去洛杉磯而受到打擊，使她曾經得過人偶病的心病復發，最後感受到羽山的心意而治好。
: 動畫版曾和五條武史（其實是親生父親）和直澄交往過，始終和羽山只是維持在死黨關係，但心中仍然喜歡羽山，讓看出這件事的直澄與風花都各自選擇退讓。雖承諾等到羽山拿到黑帶就和他告白，但是到了結局最後都沒如願。
: 雖然一開始與羽山是勢不兩立的關係，但隨著故事的演進兩人互相扶持面對不同的挑戰，成為彼此不可或缺的存在，亦可說是唯一能完全坦白的對象。和羽山的關係包含了不同的情感，是情人、朋友、對手更是戰友。此外，在給與羽山的愛情中包含了男女間的情感及母愛般的親情。
: 在漫畫番外《Deep Clear》中，26歲的她依舊以藝人的身分活耀。於23歲時和羽山結婚，在戶籍上已改名為羽山紗南，但藝名還是倉田紗南。三年後懷孕，因體質問題有早產的危險，讓因過於擔心的羽山不准紗南將孩子生下，兩人大吵一架後分居，陷入冷戰狀態。分娩期間曾一度昏迷，幸好感覺到丈夫的呼喚後醒來，最後平安的生下女兒，取名為羽山紗里。
: 這個角色是參考當時有名的童星[[有田氣惠|有田氣惠]]的。

; {{nihongo|倉田實紗子|倉田実紗子（くらた みさこ）||配音員：{{flagicon|JPN}} [[木野花|木野花]] / [[稀代樱子|稀代樱子]]；{{flagicon|TWN}} [[朱憶華|朱憶華]]（CN卡通頻道）／[[龍顯蕙|龍顯蕙]]（MOMO親子台）；{{flagicon|HKG}} [[徐愛貞|徐愛貞]]}}
: 4月1日出生。[[白羊宮|白羊座]]。AB型。紗南的養母，以《小白臉與我》（又譯：情夫與我）而得到青木獎的人氣作家，很多人都稱呼她「老師」。
: 髮型非常奇特且經常改變，言行也十分驚人，總是在家裡開著小型電動車。頭髮裡養了一隻[[松鼠|松鼠]]叫小丸（又譯：麻呂、小吉）。（一開始是紗南提議這樣做，但本人已經忘記了）。
: 為受孕機率只有5%的不孕體質，在年輕時對她造成非常重大的打擊，為此才以特異的打扮與言行紓解心中的壓力。就是在這個時候，她撿到被遺棄的紗南。她將紗南視為新人生的開始才決定領養她，更為了紗南立下要找尋她親生母親的計畫，努力成為知名的作家。
: 經常以相當特異的方式教育紗南，例如曾出過:「要睡過頭，上學遲到那是妳的自由，但想要不吃早餐就去上學，媽媽的心胸可沒這麼寬大」、「這個世界上沒有什麼事是不能鬧著玩的」等特異名言，但本身的思想成熟，也造就了紗南開朗堅強的個性。
: 對紗南採取放任式的管教，但仍然相當注意她的成長狀況，並幫助她度過各類難關。多次在人際關係和心理問題的方面上，開導心思不夠細膩的紗南。
: 出版了載有紗南身世秘密的《女兒與我》，是為了利用兩人的名氣來尋找紗南的生母。

; {{nihongo|相模玲|相模玲（さがみ れい）||配音員：{{flagicon|JPN}} [[内藤玲|内藤玲]] / [[菊池正美|菊池正美]]；{{flagicon|TWN}} [[陳進益|陳進益]]（CN卡通頻道）／[[陳宏瑋|陳宏瑋]]（MOMO親子台）；{{flagicon|HKG}} [[陳志強_(配音員)|陳志強]]}}
: 9月23日出生。[[室女宮|處女座]]。A型。紗南的經紀人，通稱「阿玲」。
: 高中時與來海麻子分手後，經歷了連考8間大學都落榜、父母意外雙亡、住所被收回而成為[[露宿者|露宿者]]等的不幸。後來紗南在車站前「撿」了他回家，在經過她母親面試後正式受聘為紗南的經紀人。由於紗南覺得他的眼晴太好看，因此要他一直都戴著太陽眼鏡。
: 後來由於紗南和麻子參與同一套電視劇，因此玲與麻子再次相遇並交往起來。不過在動畫版的惡搞完結篇中，麻子還是跟導演小野幹雄結婚了。
: 在漫畫番外《Deep Clear》中，依舊是紗南的經紀人。對紗南與羽山分居的狀況感到擔憂，暗中拜託身為偵探的音川珠里調查羽山。

; {{nihongo|志村千代|志村チヨ（しむら ちよ）||配音員：{{flagicon|JPN}} [[天野慶子|天野慶子]]；{{flagicon|TWN}} [[許雲雲|許雲雲]]（CN卡通頻道）／[[錢欣郁|錢欣郁]]（MOMO親子台）}}
: 倉田家的傭人。後來實紗子發覺是自己的童年朋友（互稱小千、小實），但似乎志村一直都記得。

== 羽山家 ==
羽山一家共四口，其名字正好是[[春天|春]]、[[夏天|夏]]、[[秋天|秋]]、[[冬天|冬]][[季節|四季]]的組合。下列就將羽山一家四名角色，做一番介紹。

; {{nihongo|羽山秋人|羽山秋人（はやま あきと）||配音員：{{flagicon|JPN}} [[中崎達也|中崎達也]] / [[緒方惠美|緒方惠美]]；{{flagicon|TWN}} [[許雲雲|許雲雲]]（CN卡通頻道）／[[錢欣郁|錢欣郁]]（MOMO親子台）；{{flagicon|HKG}} [[黄玉娟|黄玉娟]]}}
: 1982年10月12日出生（※動畫是1984年）。[[天秤宮|天秤座]]，B型。
:口頭禪-沒什麼
: 羽山秋人因為在秋天出生，所以叫做「秋人」（因為他的媽媽羽山小春超喜歡四季，因此她的兩個小孩都以季節命名）。紗南的同班同學，在故事的頭段也互相敵視，不但很喜歡捉弄紗南，而且奪去了紗南的初吻。在作品中從來不笑，是他最大的特徵之一。
: 由於出生時母親[[難產|難產]]而死，所以他姊姊非常討厭他並叫他「惡魔」、而他的性格也因此變得很乖戾。與紗南認識後性格慢慢變得好，並喜歡起紗南來。有懼高症。在作品中曾說「並不討厭」壽司，實際上是喜歡壽司，只是性格上不會說「喜歡」。特別鍾愛恐龍。解讀暗號能力差（由真理子及五味事件可知）。後來受了加村直澄的刺激和好好校長的建議後開始學起空手道。在漫畫及動畫中都曾受停學處分，但實際上在[[日本法律|日本法律]]中是禁止對中小學生處罰停學的。頭腦很好，在學業成績一直是第二名，尤其擅長數理（漫畫版第一集、動畫第二集紗南提到）。上中學後曾經和風花短暫交往過，後來和風花分手，漫畫版與紗南交往。
: 漫畫版中因小森事件而使右手神經受傷，為了治療受傷的手全家搬到洛杉磯去，一邊復健一邊繼續學習空手道，後獲得青少年  組的空手道冠軍，在高中時轉回日本就讀，回國後說話有時會夾帶英文。但在動畫中卻一直考不過黑帶。
: 故事開始時因家庭問題性格扭曲，後來是紗南救贖了他破碎不堪的心靈，個性冷漠不好相處但對於所作的事不會退縮承擔後果，紗南對他來說是唯一能坦誠一切的對象，也是支撐著他的心靈支柱，曾對風花說過“我一直都不討厭妳，但我非她不可，對不起”
: 在漫畫番外《Deep Clear》中，成為一名空手道講師兼針灸按摩師，23歲和紗南結婚，三年後，紗南懷孕，因自己的母親是生自己難產而死，所以覺得生孩子是件非常危險的事，一直不准紗南生孩子，只要一看到妻子的孕肚就會臉色發白，紗南分娩時更是陷入負面思考漩渦，後來在妻子陷入昏迷時以強烈的愛情氣波(精神呼喚)喚回妻子神智。
: 對於女兒的出生感到感激，非常疼愛女兒，最後決定了女兒的名字叫做羽山紗里。
; {{nihongo|羽山冬騎|羽山冬騎（はやま ふゆき）||配音員：{{flagicon|JPN}} [[松山鷹志|松山鷹志]] / [[小室正幸|小室正幸]]；{{flagicon|TWN}} [[陳進益|陳進益]]（CN卡通頻道）／[[陳宏瑋|陳宏瑋]]（MOMO親子台）}}
: 秋人的父親。2月14日B型。於有名的電腦公司工作，在故事初期經常不在家，個性缺根筋一直沒發現家人間的關係緊張並非對子女缺乏關心。後來被紗南罵是笨蛋後慢慢和家人改善關係。年輕時曾經是不良少年。有對著電視說話並邀請紗南到家裡來玩的奇怪舉動。
; {{nihongo|羽山夏美|羽山夏美（はやま なつみ）||配音員：{{flagicon|JPN}} [[岡村明美|岡村明美]] / [[根谷美智子|根谷美智子]]；{{flagicon|TWN}} [[朱憶華|朱憶華]]（CN卡通頻道）／[[雷碧文|雷碧文]]（MOMO親子台）}}
: 秋人的姊姊。B型。7月3日因為在夏天出生，所以叫做夏美。在故事初期因考生壓力常將氣撒在弟弟身上，叫他「惡魔」，並且將母親的去世歸咎於秋人。被紗南指責後對秋人的態度，而有所慢慢改善，在故事後期秋人受傷後顯出溫柔的一面。
; {{nihongo|羽山小春|羽山小春（はやま こはる）||配音員：{{flagicon|JPN}} [[南央美|南央美]]；{{flagicon|TWN}} [[許雲雲|許雲雲]]（CN卡通頻道）／[[錢欣郁|錢欣郁]]（MOMO親子台）}}
: 秋人的母親。生下秋人時難產而死，因此死忌是秋人的生日10月12日。在故事中除了在回憶中出現外，在秋人命危時曾以鬼魂形式出現。年輕時跟冬騎一起是不良少女。
; {{nihongo|羽山紗里|羽山紗里（はやま さり）}}
: 秋人和紗南的女兒。

== 同班同學和學校相關人物 ==
; {{nihongo|大木剛→佐佐木剛|大木剛→佐々木剛（おおき つよし→ささき つよし）||配音員：{{flagicon|JPN}} [[三澤真弓|三澤真弓]] / [[高山南|高山南]]；{{flagicon|TWN}} [[朱憶華|朱憶華]]（CN卡通頻道）／[[龍顯蕙|龍顯蕙]]（MOMO親子台）}}
: 1982年7月18日出生（※動畫是1984年）。[[巨蟹宮|巨蟹座]]。A型。紗南和秋人的同學，少數和秋人交好的朋友。故事途中由於父母離異因此在跟隨母親改[[姓氏|姓氏]]。性格上比女生還容易受到感動。
: 遺傳到父親發狂的個性，一旦發狂便一發不可收拾，只有羽山可以制止他（利用手刀）。發狂的頻率在5年級以前大概是一年一次（把老師叫成媽媽→被同學取笑→發狂），但在作品中的6年級以後就發了很多次狂(不過原因開始多樣化，例如紗南被奪去初吻時他也發飆）。
: 對他人贈送禮物沒有抵抗能力，曾因為紗南五年級亂送巧克力送給他而喜歡紗南，後來亞矢送他餅乾而跟亞矢交往，動畫第三十五集也被男同學拿這點惡作劇害他和亞矢感情差點出現危機。目前是亞矢的男朋友，十分關心秋人與紗南的關係進展。頭腦很好，在學業成績一直是第一名（漫畫版第一集、動畫第二集紗南提到）。
: 動畫中，小剛的生日也是在7月18日(因為日本的暑假只放8月份)，所以在上學時收到禮物無誤，詳情請見13集。

; {{nihongo|杉田亞矢|杉田亜矢（すぎた あや）||配音員：{{flagicon|JPN}} [[菊地晶子|菊地晶子]]；{{flagicon|TWN}} [[朱憶華|朱憶華]]、[[謝佼娟|謝佼娟]]（第82集至99集代配）（CN卡通頻道）／[[雷碧文|雷碧文]]（MOMO親子台）}}

; {{nihongo|松井風花|松井風花（まつい ふうか）||配音員：{{flagicon|JPN}} [[生駒治美|生駒治美]]；{{flagicon|TWN}} [[朱憶華|朱憶華]]（CN卡通頻道）／[[劉凱寧|劉凱寧]]（MOMO親子台）}}

; {{nihongo|尾崎里穗/尾崎里惠|尾崎里穂（おざき りほ）||配音員：{{flagicon|JPN}} [[岩坪里江|岩坪里江]]；{{flagicon|TWN}} [[朱憶華|朱憶華]]（CN卡通頻道）／[[錢欣郁|錢欣郁]]（MOMO親子台）}}
