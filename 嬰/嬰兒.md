'''嬰兒'''是指剛[[出生|出生]]的[[兒童|兒童]]，是人類一生的第一階段。

根據《[[說文解字|說文解字]]》所述，嬰兒的「嬰」字本意為女性的頸部飾物，後引申解作為抱在胸前哺乳之[[初生兒|初生兒]]。而嬰兒的[[英文|英文]]“{{lang|en|infant}}”源于[[拉丁文|拉丁文]]“{{lang|la|in-fans}}”一词，意为「不懂說話」。英語一般语境下是指小於一歲或未懂得走路的人類<ref name="dictionary">{{cite web | publisher = dictionary.com | title = Results for "infant" | url = http://dictionary.reference.com/browse/infant | accessdate = 2007-03-27}}</ref>。中文中「嬰兒」則指兩歲以下或未斷奶的兒童，不管是否能夠行走。

== 新生婴儿的外表 ==
[[File:HumanNewborn.JPG|right]]
剛剛出生的嬰兒[[臍帶|臍帶]]數星期後自己會脫落。有些嬰兒出世時有很多[[頭髮|頭髮]]，但亦有些卻相反。

[[File:Newborn_minutes_after_delivery.jpg|right]]（vernix caseosa），剛剛出生的嬰兒]]
剛剛出生的時候，嬰兒的皮膚呈暗藍色，一開始呼吸就會回復正常膚色。他們剛出生時還會有一層白色的東西，稱為{{tsl|en|Vernix caseosa|胎兒皮脂}}，有抗菌作用。手腳後來會脫皮，還會有[[胎記|胎記]]。

嬰兒的性器官會紅腫，男嬰的[[陰莖|陰莖]]會又大又紅，[[陰囊|陰囊]]異常的大，無論男女嬰兒的[[胸部|胸部]]都會大，是由於母體的激素造成的短暫情況。男女嬰兒的乳頭可能會產出奶，或女嬰從[[陰道|陰道]]流出血或奶狀的液體，兩個情況都是正常的，均是受母體激素的影響，很快會消失。

嬰兒的[[臍帶|臍帶]]呈藍白色，臍帶剪掉以後，會留下2至5公分，三星期左右，就會變乾、變黑並縮小，最後自己脫落。

這些種種剛剛出生的嬰兒的特徵很快就會消失。

== 婴儿的感官 ==
[[File:Infant_looking_at_shiny_object.jpg|thumb]]
哭泣是婴儿的本能，婴儿通过哭泣来传达饥饿、兴奋、厌倦、不适等感受。

绝大多数婴儿健康组织提倡[[母乳喂养|母乳喂养]]。当无法进行母乳喂养时，可以使用装有母乳或婴儿配方奶粉的奶瓶代替。婴儿喜歡啜手指、啜乳头、啜[[奶嘴|奶嘴]]。啜東西是他們的本能，讓他們可以自己吃奶。

婴儿最喜歡被轻轻拍打、拥抱和抚摸。婴儿的睡眠比成人多的多，新生儿达到每天18小时以上，随着年龄的增大睡眠逐渐减少。

新生嬰兒的[[視力|視力]]大约只有0.05，只能看到视力表上最大的那个"E"。随着年龄的增长，大约2岁时，才能提高到1.0。新生儿一生下来或几天后，便能睁开双眼，接受光线。大约6到8周时，开始能够凝视物体，并随着物体的移动而转动视线。大概4个月时，两眼的立体感逐渐发育，可以用双眼定位物体。

嬰兒還在母體時已可以在自己體內發出像母親般的心跳聲，亦能聽到外界的聲音。胎中的嬰兒雖然浸在羊水中，但亦可聽到聲音。嬰兒對女性的聲音作出的反應更大於男性的聲音。

嬰兒一-{}-出生就對甜、酸、苦、鹹有反應，甜的東西最為喜歡。一-{}-出生嗅覺就很發達，在第一個星期，已經可以分辨到母親跟其他女性的[[人奶|奶]]的味道。

== 嬰兒夭折率 ==
[[夭折率|夭折率]]是嬰兒一歲之前[[死|死]]的比率。這共可分為兩種：出世首27天死稱為新生兒死亡（neonatal death）；第28天以後死亡的稱為新生兒產後死亡率（post-neonatal death）。死因包括：缺水、感染、先天畸形、[[嬰兒猝死症|嬰兒猝死症]]（Sudden infant death syndrome）。

== 婴儿适宜食物 ==
[[File:Breastfeeding_a_baby.JPG|thumb]]]]

母乳是初出生嬰兒最理想的食物，世界衛生組織建議[[母乳餵養|母乳餵養]]至嬰兒六個月大，於六個月（180天）大之後，添加適宜嬰兒成長需求的副食品。<ref name="世界衛生組織|哪些是儿童生命早期的推荐食品？">[http://www.who.int/topics/infant_nutrition/zh/ 世界衛生組織|嬰兒營養]</ref><ref>[http://www.who.int/features/qa/57/zh/ 世界衛生組織|哪些是儿童生命早期的推荐食品？]</ref>

* 一（1～6个月龄）：纯母乳喂养，不添加母乳以外的任何食物，包括水，果汁，米糊等。
* 二（6～9个月龄）：母乳依然为主食，添加食物包括米糊、粥、水果汁、菜汁、蛋黄、鱼泥、肉糜等。
* 三（10～12个月龄）：添加烂面条、水果、碎菜、肝泥、和肉糜等容易消化的食物。
* 四（1岁以后）：尽量继续母乳至2岁。逐步增加稠粥、烂饭、面包、馒头、碎菜及肉末等食物。
* 五，固体食物添加顺序应该是谷类、蔬菜、水果、鱼肉类，婴儿喜爱甜食，先加水果就会拒绝蔬菜，一岁以后如果需要断奶，可以补充其他奶制品，如安全的鲜奶，酸奶，或者婴儿配方奶粉等，但配方奶粉并非首选，也非必需品。{{來源請求}}

== 婴儿饮食禁忌 ==
不宜过早添加辅助食品，应在婴儿4个月后，不宜过早添加含渣食品，要适应婴儿的消化能力，制作时调味品应少放，以免加重婴儿的肾脏负担。{{來源請求}}

嬰兒攝取一般飲食即可，若要攝取加工食品，則應詢問醫師意見，市售[[嬰兒食品|嬰兒食品]]並不一定合適。<ref>[http://m.news.sina.com.tw/article/20140625/12756335.html 台灣醒報：食品太營養 攝取維他命易過度]</ref>為預防嬰兒肉毒桿菌中毒，一週歲（12個月齡）以下之嬰兒不應餵食蜂蜜。<ref>[http://www.cdc.gov.tw/diseaseinfo.aspx?treeid=8d54c504e820735b&nowtreeid=dec84a2f0c6fac5b&tid=A664CCFBD51288B0 中華民國衛生福利部疾病管制署：傳染病介紹：肉毒桿菌中毒]</ref>
