'''甜蜜素'''（Sodium cyclamate），又称'''甜精'''，化学名[[环己基氨基磺酸钠|环己基氨基磺酸钠]]，一种[[甜味剂|甜味剂]]。

== 历史 ==
甜蜜素在1937年被[[伊利诺伊大学|伊利诺伊大学]]的学生麦克尔·斯维达（Michael Sveda）发现，1950年代开始应用于[[软性饮料|软性饮料]]工业，1960年代上市成为一般性[[代糖|代糖]]。1966年有研究发现甜蜜素可在肠菌作用下分解为可能有慢性毒性的[[环己胺|环己胺]]。1969年美国国家科学院研究委员会收到有关甜蜜素兑[[糖精|糖精]]的10:1混合物可致[[膀胱癌|膀胱癌]]的动物实验证据，不久后[[美国食品与药物管理局|美国食品与药物管理局]]即发出了全面禁止使用的命令。英、日、加拿大等国随后也禁用。

1973和1982年，[[美国雅培|美国雅培]]在多种证明甜蜜素食用安全性的实验报告基础上，两次向美国食品与药物管理局请愿希望能恢复甜蜜素的使用，但没有获得批准。美国食品与药物管理局认为现有证据虽然无法证明甜蜜素在大鼠和小鼠中的致癌作用，而且一些国际组织也发表评论表示甜蜜素是安全物质，但他们目前是不会考虑推翻现有的禁令，也不会对甜蜜素进行系统性的安全评估。

目前承认甜蜜素甜味剂地位的国家有超过55个，包括中国大陆和台湾在内。

== 性质 ==
甜蜜素是一种白色结晶或结晶性粉末，溶于水和[[丙二醇|丙二醇]]，几乎不溶于[[乙醇|乙醇]]、[[乙醚|乙醚]]、[[氯仿|氯仿]]和[[苯|苯]]。对空气、光、热稳定，在碱性介质中稳定，在酸性介质中略有分解。味甜，加热后稍有苦味。<ref name="chemyq">{{cite web|url=http://www.chemyq.com/xz/xz1/7314qqbcb.htm|title=环己基氨基磺酸钠；甜蜜素；环氨酸钠|publisher=化工引擎|accessdate=2009-09-24}}</ref>

== 制取 ==
由[[环己胺|环己胺]]与[[氯磺酸|氯磺酸]]或[[氨基磺酸|氨基磺酸]]发生[[磺化|磺化]]成[[环己基氨基磺酸|环己基氨基磺酸]]后，再与[[氢氧化钠|氢氧化钠]]成盐而得。<ref name="chemyq" />

== 用途 ==
甜蜜素是一种非营养性合成甜味剂，[[甜度|甜度]]约为[[蔗糖|蔗糖]]的30倍，优点是甜味好，后苦味比[[糖精|糖精]]低，成本较低；缺点是甜度不高，用量大，易超标使用，无营养价值。<ref name="食品化学">{{cite book|title=食品化学|author=夏延斌主编|publisher=中国轻工业出版社|location=北京|isbn=7-5019-3153-4|pages=365|date=2001年6月}}</ref>
