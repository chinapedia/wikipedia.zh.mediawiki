'''摄政台'''（Regent Terrace）是一条拥有34座古典住宅的住宅区道路，位于[[苏格兰|苏格兰]][[爱丁堡|爱丁堡]]的[[卡尔顿山|卡尔顿山]]。1995年，摄政台作为“爱丁堡的[[爱丁堡新城|新城]]和老城”的一部分，被列为[[世界遗产|世界遗产]]<ref>[http://news.scotsman.com/wheeliebins/UN-to-investigate-capitals-wheelie.2524550.jp  Newspaper article  mentioning the World Heritage Status of Regent Terrace], Howarth, Angus, "UN to investigate capital's wheelie bins", The Scotsman 2004-04-30.  Accessed 2009-08-09</ref><ref>[http://whc.unesco.org/en/list/728 UNESCO World Heritage Site Inscription]  Accessed 2009-08-10</ref>。

== 住宅 ==
摄政台得名于1822年[[乔治四世|乔治四世]]对爱丁堡的访问，他在父亲[[乔治三世|乔治三世]]生病期间担任摄政王。摄政台由建筑师威廉·亨利·普莱费尔规划于1825年，兴建于1831年到1833年<ref name = EdCouncil>{{Cite web |url=http://www.britishlistedbuildings.co.uk/sc-29618-1-regent-terrace-including-railings-and-b |title=1 Regent Terrace Including Railings and Boundary Walls, Edinburgh |accessdate=2015-12-08}}</ref>。普莱费尔同时设计了摄政台、皇家台和卡尔顿台，作为新城的向东延伸<ref name=EdCouncil2>[http://download.edinburgh.gov.uk/caca/CACANew_Town.pdf Report on The New Town Conservation Area by Edinburgh Town Council] {{webarchive|url=https://web.archive.org/web/20061008151020/http://download.edinburgh.gov.uk/caca/CACANew_Town.pdf |date=2006-10-08 }} Accessed 2009-08-10</ref>，比克雷格最初的新城规划更为壮观<ref name = Youngson/>。三个台被誉为爱丁堡最长的连续的乔治式建筑群<ref>[http://www.terracehotel.co.uk/ Description of the Terraces] Accessed 2009-08-09</ref> 普莱费尔希望吸引“时尚和富有的人”到摄政台<ref name = Youngson/>。这些住宅全部被列为A类登录建筑<ref name = EdCouncil/>。
[[Image:Plaque_28_Regent_Terrace_Edinburgh.JPG|thumb]]
这些住宅原先都只有两层（有些现在加高到三层）<ref name = EdCouncil/><ref name = Youngson>Youngson, A.J. (2001): “The Companion Guide to Edinburgh and the borders”, Chapter 9 (Calton Hill), [[Polygon_Books|Polygon Books]], Edinburgh, UK, ISBN 0-7486-6307-X</ref>。

自1951年起，美国领事馆设在摄政台3号<ref>[http://www.usembassy.org.uk/scotland/history.htm History of the United States Consulate General in Edinburgh] {{webarchive|url=https://web.archive.org/web/20100130074147/http://www.usembassy.org.uk/scotland/history.htm |date=2010-01-30 }} Accessed 2009-08-08</ref>。28号曾是戴高乐的自由法国大厦，后来成为法国总领事馆。2008年之前，32号曾是挪威总领事馆。

== 房价 ==
摄政台6号在1831年售价1500英镑，1877年售价2,700英镑<ref name = Anne>Mitchell, Anne (1993), “The People of Calton Hill”, Mercat Press, James Thin, Edinburgh, ISBN 1-873644-18-3. </ref>。此后价格下跌，到二战前只有1,000英镑，到战争结束时升至2,000英镑，1950年代中叶为4,000英镑，1993年为40万英镑<ref name = Anne/>，2008年超过了200万英镑<ref>[http://www.nethouseprices.com/index.php?con=sold_prices_street_detail&street=REGENT+TERRACE&locality=%20&town=EDINBURGH&cCode=SC&year=All&house_style=All&house_age=All&search_radius=15&outcode=EH7&incode=5BN&eastingToSearch=32650&northingToSearch=67410 Sold House prices for Regent Terrace] Accessed 2009-08-09</ref>。
