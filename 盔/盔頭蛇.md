'''盔頭蛇'''（''Hoplocephalus bungaroides''）是[[澳洲|澳洲]]特有的一種[[毒蛇|毒蛇]]。牠們在[[新南威爾士|新南威爾士]]被列為[[瀕危|瀕危]]。

== 特徵 ==
盔頭蛇是中等身型的[[毒蛇|毒蛇]]。牠們的身長60厘米，最長可達90厘米。牠們呈黑色，有不規則的黃色斑紋。

== 棲息地 ==
盔頭蛇分佈在[[澳洲|澳洲]][[新南威爾士|新南威爾士]]的[[悉尼盆地|悉尼盆地]]。牠們棲息地會視乎[[溫度|溫度]]、年齡、性別及[[繁殖|繁殖]]狀況而有所不同。在寒冷的季節，牠們會棲息在山崖石縫間，以從[[太陽|太陽]]取暖。當溫度上升時，牠們會走到林地，住在空心樹中；幼蛇及懷孕的雌蛇會留在較涼快及有遮蔽的岩石地帶。

盔頭蛇的棲息地受到城市化及非法清除岩石所破壞，令牠們暴露於[[掠食者|掠食者]]及失去棲息地。牠們所棲息的砂岩是園藝的材料。

== 行為 ==
=== 食性 ===
盔頭蛇主要吃細小的[[爬行動物|爬行動物]]及[[哺乳動物|哺乳動物]]。飼養的每個月只吃一或兩隻[[大家鼠|大家鼠]]就足以保持體重。幼蛇主要吃一種稱為''Oedura lesueuri''的[[壁虎|壁虎]]，及一些細小的[[石龍子|石龍子]]。

=== 繁殖 ===
雄蛇5歲就達至性成熟，雌蛇則要6歲。[[繁殖|繁殖]]期是於[[秋天|秋天]]至[[春天|春天]]間，雌蛇每兩年會生一胎。牠們是[[胎生|胎生]]的，出生期在每年1月至4月間，每胎約有4-12條幼蛇。牠們很多時會誕下未成熟的[[卵母細胞|卵母細胞]]或死胎。

== 保育狀況 ==
盔頭蛇在[[悉尼|悉尼]]曾很普遍。不過，牠們在[[新南威爾士|新南威爾士]]已被列為[[易危|易危]]。牠們衰落的原因是城市化造成的失去棲息地。
