'''沖南岩'''又被稱作'''南岩'''、'''大南小島'''或者是'''蛇島'''，在[[日本|日本]]則稱呼為'''沖之南岩'''（{{jpn|j= 沖の南岩 |hg= おきのみなみいわ |rm= Oki-no-Minami-iwa }}），是[[釣魚臺列嶼|釣魚臺列嶼]]中的一座無人[[荒島|荒島]]<ref group= "註">關於沖北岩在各地漢語字詞的使用習慣上皆不同處，其中中國大陸多稱呼為「-{南屿}-」，而香港與臺灣地區則常稱呼為「-{沖南岩}-」。</ref><ref>{{Cite web |url= http://mil.sohu.com/20101002/n275398296.shtml |title= 大陆台湾渔政船同时巡视钓鱼岛 日本巡逻舰无奈 |author= 刘峰 |date= 2010年10月2日 |publisher= [[搜狐|搜狐]] |language= zh-hans |accessdate= 2012年11月16日 |quote=  }}</ref><ref>{{Cite web |url= http://news.xinhuanet.com/ziliao/2003-01/23/content_703922.htm |title= 神聖的領土釣魚諸島 |author= 董曉 |date= 2003年1月23日 |publisher= [[新華網|新華網]] |language= zh-hant |accessdate= 2013年11月28日 |quote=  }}</ref><ref>{{Cite web |url= http://www.ifeng.com.tw/twnews/detail_2012_10/26/18591005_0.shtml |title= 釣魚島寶藏解祕 台日漁民冒險登島賺外快 |author= [[聯合新聞網|聯合新聞網]] |date= 2012年10月26日 |publisher= [[鳳凰衛視|鳳凰衛視]] |language= zh-hant |accessdate= 2013年11月28日 |quote=  }}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。沖南岩本身島嶼地質為火山島結構[[火山島|火山島]]，通常跟和它相應的[[沖北岩|沖北岩]]一同被歸類為[[釣魚臺|釣魚臺]]其附屬島嶼中。其相對位置位在[[石垣島|石垣島]]北方約160公里、釣魚臺列嶼主島釣魚臺東方約7公里處，島嶼整體[[面積|面積]]約0.01平方公里，而其海拔[[高度|高度]]最高僅有13公尺<ref name = "沖繩縣廳"/>。目前包括[[日本|日本]]、[[中華人民共和國|中華人民共和國]]和[[中華民國|中華民國]]都聲稱擁有沖南岩之主權，其中日本認定沖南岩等釣魚臺列嶼皆為其[[實際統治|實際統治]]的領土，但是另外一方面中華人民共和國與中華民國則多次表示自身才合法擁有該土地的主權<ref>{{Cite book|title=中国海岛开发与保护 地理学视角|last=张耀光编|first=|publisher=北京：海洋出版社|year=2012.06|isbn=978-7-5027-8291-7|location=|pages=126-128}}</ref><ref>{{Cite book|title=中国海洋权益争端的国际法分析|last=冯学智主编|first=|publisher=北京：中国政法大学出版社|year=2013.03|isbn=978-7-5620-4628-8|location=|pages=102-105}}</ref><ref>{{Cite book|title=钓鱼岛|last=疏震娅，张海文著|first=|publisher=北京：五洲传播出版社|year=2014.11|isbn=978-7-5085-2902-8|location=|pages=1-50}}</ref><ref>{{Cite book|title=中国海岛开发与保护 地理学视角|last=张耀光编|first=|publisher=北京：海洋出版社|year=2012.06|isbn=978-7-5027-8291-7|location=|pages=126-128}}</ref>。
