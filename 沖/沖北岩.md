'''沖北岩'''又被稱作'''北岩'''、'''大北小岛'''或者是'''鳥島'''，在[[日本|日本]]則稱呼為'''沖之北岩'''（{{jpn|j= 沖の北岩 |hg= おきのきたいわ |rm= Oki-no-Kita-iwa }}），是[[釣魚臺列嶼|釣魚臺列嶼]]中的一座[[荒島|荒島]]<ref group = "註">關於沖北岩在各地漢語字詞的使用習慣上皆不同處，其中中國大陸多稱呼為「-{北屿}-」，而香港與臺灣地區則常稱呼為「-{沖北岩}-」。</ref><ref>{{Cite web |url= http://mag.chinatimes.com/mag-cnt.aspx?artid=6886&page=1 |title= 釣魚台到底是誰的？ |author= 洪健昭 |date= 2011年2月18日 |publisher= [[中時電子報|中時電子報]] |language= zh-hant |accessdate= 2013年11月28日 |quote=  }}</ref><ref>{{Cite web |url= http://news.xinhuanet.com/ziliao/2003-01/23/content_703922.htm |title= 神聖的領土釣魚諸島 |author= 董曉 |date= 2003年1月23日 |publisher= [[新華網|新華網]] |language= zh-hant |accessdate= 2013年11月28日 |quote=  }}</ref>。沖北岩本身島嶼地質為[[火山島|火山島]]結構，通常跟和它相應的[[沖南岩|沖南岩]]一同被歸類為釣魚臺其附屬島嶼中。目前包括[[日本|日本]]、[[中華人民共和國|中華人民共和國]]和[[中華民國|中華民國]]都聲稱擁有沖北岩之主權，其中日本主張對沖北岩等釣魚臺列嶼擁有主權並為由其[[實際統治|實際統治]]的領土，但是另一方面中華人民共和國與中華民國則多次表示自身才合法擁有領土的主權。

== 地理 ==
沖北岩位在[[石垣島|石垣島]]北方約160公里、釣魚臺列嶼主島[[釣魚臺|釣魚臺]]北東方約6公里處，而其確實位置則為北緯25度46.5分、東經123度32分。島嶼整體[[面積|面積]]約0.05平方公里，而其海拔[[高度|高度]]最高僅有28公尺<ref name = "沖繩縣廳"/>。沖北岩主要由[[北嶼_(沖北岩主島)|北-{}-嶼]]、[[元寶島|元寶島]]和[[飛雲島|飛雲島]]3個主要島嶼所組成，而在北-{}-嶼和元寶島之間則隔著[[元寶門|元寶門]]海峽。而在最大的島嶼北-{}-嶼周邊還有數個小型島礁聚集，其中主要的包括有[[北屿仔岛|沖北岩仔島]]以及[[小元寶島|小元寶島]]，而有的時候亦會直接以「北-{}-嶼」作為主要島嶼北-{}-嶼和附近小礁石的統稱<ref group = "註">此處採用了在2012年9月，由[[中華人民共和國|中華人民共和國]]所公布的《釣魚臺海域部分地理實體標準名稱》命名稱呼。</ref><ref>{{Cite web |url= http://politics.people.com.cn/n/2012/0921/c1001-19076819.html |title= 海洋局、民政部公布我国钓鱼岛海域部分地理实体标准名称 |author= 崔东和耿聪 |date= 2012年9月21日 |publisher= [[人民網|人民網]] |language= zh-hans |accessdate= 2012年11月16日 |quote=  }}</ref><ref>{{Cite web |url= http://news.sina.com.cn/c/2012-09-22/091025229896.shtml |title= 钓鱼岛海域部分地理实体标准名称公布 |author= [[國際在線|國際在線]] |date= 2012年9月22日 |publisher= [[新浪|新浪]] |language= zh-hans |accessdate= 2012年11月16日 |quote=  }}</ref>。而另外一方面，日本則依照島嶼相對方位而將沖北岩分成東岩（{{lang|ja|東の岩}}）與西岩（{{lang|ja|西の岩}}）2個主要岩礁。
