'''卤仿反应'''是甲基酮类化合物，即含有[[乙酰基|乙酰基]]的化合物（R-CO-CH{{sub|3}}，R-可为[[氢|氢]]、[[烃基|烃基]]或[[芳基|芳基]]）在[[碱性|碱性]]条件下卤化并生成[[卤仿|卤仿]](三卤甲烷)与[[羧酸盐|羧酸盐]]的有机反应。<ref>Chakrabartty, in Trahanovsky, ''Oxidation in Organic Chemistry'', pp 343-370, Academic Press, New York, '''1978'''</ref>


[[Image:Haloform_Reaction_Scheme.png|center]]

==机理==
卤仿反应在机理上可以分为三步。以[[碘|碘]]参与的卤仿反应为例：
#羰基α-氢的连续卤化：
#*R-CO-CH<sub>3</sub> + 3 I<sub>2</sub> + 3 OH<sup>−</sup> → R-CO-CI<sub>3</sub>  +  3 I<sup>−</sup> + 3 H<sub>2</sub>O
#氢氧根的进攻：
#*R-CO-CI<sub>3</sub> + OH<sup>−</sup>  → [[羧酸|RCOOH]] + −CI<sub>3</sub>
#质子交换，卤仿最终形成：
#*RCOOH + −CI<sub>3</sub> → RCOO<sup>−</sup> + [[碘仿|CHI<sub>3</sub>]]
从本质上说，第1步是X<sup>+</sup>对氢的亲电取代；第2步是一个亲核取代。2、3步合起来可以看作三卤甲基酮的水解。

[[File:Iodoform_reaction.png|650px]]

机理如上图。图中第一步，是相對應甲基酮化合物的[[醇|醇]]類會因為被鹵素分子氧化成甲基酮進而產生鹵仿反應，因此此類化合物對於鹵仿反應通常也會呈現陽性反應。

参加反应的卤素是[[碘|碘]]时，稱為'''碘仿反應'''，氯、溴以此類推。氟氣不能与甲基酮类化合物进行这种反应，但是形如R-CO-CF<sub>3</sub>的化合物可以用来制备[[氟仿|氟仿]]（CHF<sub>3</sub>），这相当于卤仿反应机理的后半部分，是[[水解反应|水解反应]]。

卤仿反应只有在鹼性條件下才會產生[[鹵仿|鹵仿]]，酸性條件下則只會在与羰基相连的甲基帶上一個[[鹵素|鹵素]][[原子|原子]]。这相当于羰基α-氢的卤化，反應式：
*2 R-CO-CH<sub>3</sub> + X<sub>2</sub> －酸性條件→ 2 R-CO-CH<sub>2</sub>X

==應用==
[[File:Jodoformprobe.jpg|thumb]]
碘仿反应是卤仿反应的一种重要应用，在[[分析化学|分析化学]]中用来定性检测甲基酮类化合物(R-CO-CH<sub>3</sub>)及相应的醇（R-C(OH)-CH<sub>3</sub>）。该类化合物与[[氢氧化钠|氢氧化钠]]和[[碘|碘]]单质反应，生成常温[[溶解性|不溶于水]]、有消毒劑气味的黄色固体碘仿。

以[[丙酮|丙酮]]混合[[次氯酸鹽|次氯酸鹽]]可以用來製造[[氯仿|氯仿]]也是鹵仿反應的一種应用。在有机化学中，卤仿反应可用来利用甲基酮类化合物合成相应的羧酸。
