'''諸子百家'''是後世對[[華夏|華夏]][[先秦|先秦]]學術思想人物和派別之總稱。諸子是指[[中國|中國]][[先秦|先秦]]時期[[老子|老子]]、[[庄子|庄子]]、[[孔子|孔子]]、[[孟子|孟子]]、[[荀子|荀子]]、[[墨子|墨子]]、[[列子|列子]]、[[申子|申子]]、[[韓非子|韓非子]]等学术思想代表人物。[[春秋时期|春秋]]后期已出现颇有社会影响的[[道家|道家]]、[[儒家|儒家]]、[[法家|法家]]、[[墨家|墨家]]、[[阴阳家|阴阳家]]等不同学術流派，而至[[戰國時代|战国]]中期，'''百家爭鳴'''，眾多学说紛呈，丰富多彩，为[[中華文化|中華文化]]奠定了宽广基础。

== 沿革 ==
[[班固|班固]]《漢書·藝文志》認為諸子百家皆出於王官：「道家者流，蓋出於史官」；「儒家者流，蓋出於司徒之官」；「墨家者流，蓋出於清廟之守」；「法家者流，蓋出於理官」；「名家者流，蓋出於禮官」；「陰陽家者流，蓋出於羲、和之官」；「農家者流，蓋出於農稷之官」；「縱橫家者流，蓋出於行人之官」；「雜家者流，蓋出於議官」；「小說家者流，蓋出於稗官」。
[[春秋战国|春秋战国]]时代，王权衰落，[[诸侯|诸侯]]争霸，为了壮大自己的实力，各国开放政权以延揽人才，打破了本来的贵族政治体制，使得原本几乎没有资格参与政治的庶民可以发表自己的政见并参与政治决策，又因为[[人口|人口]]的增加，土地分配困难，[[社会|社会]]发生剧变。这一时期有思想的[[知识分子|知识分子]]，面对现实的社会问题、人生问题等，提出了解决的办法和思想。个人利害与国家之间的利害交互运用，相互影响，标新立异。各种学说、思想纷纷出现。

== 史家論述 ==

各个流派  
 
* [[西汉|西汉]][[司马迁|司马迁]]在《[[史记|史记]]》中引述了[[司马谈|司马谈]]（司马迁之父）对学术流派的见解，司马谈把[[先秦时期|先秦]]以来的[[学派|学派]]总归纳为六家([[論六家要旨|論六家要旨]])，即：[[道家|道家]]、[[儒家|儒家]]、[[墨家|墨家]]、[[法家|法家]]、[[名家|名家]]、[[阴阳家|阴阳家]]。
* [[东汉|东汉]][[班固|班固]]在《[[汉书|汉书]]》中则把[[先秦|先秦]]以来的学派归纳为十家([[九流十家|九流十家]])，即：[[道家|道家]]、[[儒家|儒家]]、[[墨家|墨家]]、[[法家|法家]]、[[名家|名家]]、[[阴阳家|阴阳家]]、[[农家|农家]]、[[纵横家|纵横家]]、[[杂家|杂家]]、[[小说家|小说家]]。
* [[吕思勉|吕思勉]]认为“[[术数|术数]]”、“[[方技|方技]]”、“[[兵书|兵书]]”三略，亦可称为先秦诸子，可算为'''十二家'''<ref>《先秦学术概论》</ref>。
* [[周非|周非]](中國业余歷史作者)反對現代人繼續沿用[[班固|班固]]對先秦諸子百家的分類，認為「這不過是[[班固|班固]]這些人，在兩千年前為諸子百家刻的一個記號而已」<ref>周非/著，《諸子百家大解讀》，遠流出版社，2011年4月初版，第12頁</ref>，必須重新審視當時各種思想學說、文學作品，以及各種新出現的科學知識及技術，然後重新歸納、分類。其在《諸子百家大解讀》一書，將春秋戰國時期的各種思想學說、文學作品，以及各種新出現的科學知識及技術，歸納為八大類<ref>周非將[[管子|管子]]學術思想歸納為「治國之道」；將[[道家|道家]]與[[陰陽家|陰陽家]]歸納為「天人之道」；將[[孔子|孔子]]與先秦[[儒家|儒家]]歸納為「君子之道」，將[[鄧析|鄧析]]、[[墨家|墨家]]、[[農家|農家]]、[[楊朱|楊朱]]、以[[魯班|魯班]]為代表的工程師、實業家、俠士等歸納為「平民之道」；將[[法家|法家]]、[[縱橫家|縱橫家]]、[[刑名之學|刑名之學]]、[[黃老之術|黃老之術]]等歸納為「帝王之道」與「功名之道」；將科學家歸納為「自然之道」；最後將文學及藝術作品歸為一類，稱之為「心靈的翅膀」。</ref>；並且舉出九本值得現代人重視的經典，稱之為「四經五書」<ref>「四經」是指《[[周髀算經|周髀算經]]》、《[[黃帝內經|黃帝內經]]》、《[[管子|管子]]》、《[[墨子|墨子]]》；「五書」則是指《[[左傳|左傳]]》、《[[詩經|詩經]]》、《[[莊子|莊子]]》、《[[論語|論語]]》、《[[考工記|考工記]]》。</ref>。其次，[[周非|周非]]主張要以「大視野」、「大背景」及「大系統」<ref>周非主張的「大視野」指的是「時代的視野」、「五千年文明史的視野」、「橫穿中西文明的視野」、「面向未來的視野」；「大背景」指的是「諸子百家之前兩千年的背景」、「當時五百年的背景」、「此後兩千年演變的背景」、「中西方文化交會的背景」、「現今的時候背景」；「大系統」指的是「天人系統」、「人文系統」、「學科系統」、「學派系統」、「諸子們個人的學術系統」。</ref>審視先秦諸子百家，如此才能全面、科學、系統地解讀諸子百家，進而對[[中國文化|中國文化]]正本清源，實現[[中國文化|中國文化]]的復興<ref>周非/著，《諸子百家大解讀》，遠流出版社，2011年4月初版，第5~6頁。</ref>。

== 影响 ==
基本上形成了中国的传统文化体系，这是中国历史上第一次大规模的思想解放运动，有力地推动了中国历史的发展。是中国学术文化思想发展史上的重要里程碑阶段，奠定了中国思想文化发展的基础。各家之间互相辩驳，又互相影响，互相取长补短，有力地促进了思想文化的发展。
