'''意大利使馆旧址'''，位于[[北京市|北京市]][[东城区_(北京市)|东城区]][[台基厂大街|台基厂大街]]1号，是意大利驻华使馆旧址。

==历史==
[[清朝|清朝]][[咸丰_(年号)|咸丰]]十年（1860年），[[英法联军|英法联军]]在[[第二次鸦片战争|第二次鸦片战争]]中攻入北京后，意大利继[[英国|英国]]、[[法国|法国]]之后在北京设立使馆。后于[[同治|同治]]八年（1869年）在[[东交民巷|江米巷]]与台基厂交叉路口的东北角的民宅内设公馆，该公馆路南对着[[大学士|大学士]][[徐桐|徐桐]]的住宅。[[光绪|光绪]]二十六年（1900年），[[义和团|义和团]]围攻使馆区时，意大利使馆被毁<ref name=dhly>{{Cite web |url = http://www.bjdclib.com/subdb/laneculture../historybuilding/200906/t20090610_1835.html |title =  意大利使馆旧址|author =  |date = 2009-06-10 |publisher =东华流韵  |language =  |accessdate =  |quote =  }}</ref>。

《[[辛丑条约|辛丑条约]]》签订后，意大利使馆要求扩大用地，另建使馆，乃占据西到御河，东到台基厂，北到[[东长安街|东长安街]]的地区。这片地区即[[堂子|堂子]]全部以及总税务司以北、[[肃亲王府|肃亲王府]]一部及其东的民宅，这些房屋被拆除，在其原址新建了位于东部的意大利使馆，以及位于西部的意大利兵营<ref name=dhly/>。

1928年[[北伐战争|北伐战争]]结束以后，各国使馆先后迁往[[南京国民政府|南京国民政府]]首都[[南京|南京]]，但各国在北平的使馆旧址未废，仍归各国所有<ref>{{Cite web |url = http://www.taiwan.cn/zt/wj/manbubeijingzhuanti/mbbj/200801/t20080102_516531.htm |title =  第四章　曲径千姿踏歌行（下）|author =  |date = 2003-09-12 |publisher = 中国台湾网 |language =  |accessdate =  |quote =  }}</ref>。

[[第二次世界大战|第二次世界大战]]期间，1943年9月8日，意大利宣布退出[[轴心国|轴心国]]。1943年9月9日，1000多名[[日军|日军]]在大炮及15辆轻型坦克的掩护下，向东交民巷的意大利使馆及电台发动攻击，仅有步枪及[[手榴弹|手榴弹]]的100名意大利守军防守一昼夜，9月10日上午9时投降，投降前毁掉了电台并焚烧了全部绝密文件。同时，日军还对在[[天津|天津]]、[[上海|上海]]租界及使领馆区的意大利军队以及在日本控制下的各港口的意大利舰船发动攻击，在这场被日本称为“九月义人降伏事变”中，日军共抓获将近1100名意大利公民。其中170名意大利官兵宣布效忠[[纳粹德国|纳粹德国]]扶植的“[[意大利社会共和国|意大利社会共和国]]”（即“萨洛共和国”），其他人则被日军关进位于[[满洲国|满洲国]]、[[日治朝鲜|朝鲜]]、[[菲律宾|菲律宾]]的集中营，在随后两年内饱受虐待<ref>{{Cite web |url =  http://news.163.com/09/0628/17/5CTP7S1000011MSF_2.html|title = 意大利侵华"远征军"：盟友日军屠刀下的冤魂 |author =  |date =  2009-06-28|publisher = 网易 |language =  |accessdate =  |quote =  }}</ref>。

意大利使馆旧址现由[[中国人民对外友好协会|中国人民对外友好协会]]使用。2001年作为东交民巷使馆建筑群的一部分，被定为[[第五批全国重点文物保护单位|第五批全国重点文物保护单位]]<ref name=dhly/>。

==建筑==
意大利使馆旧址现存原办公楼、教堂、官邸等建筑<ref name=dhly/>。

使馆大门位于台基厂大街西侧。南、北两座门房间距13米。门房为红砖墙，四坡锥形屋顶。檐下有金属花式支撑。两座门房之间有曲线形墙，墙高4米，红砖墙身上有石材镶边及大涡卷等装饰；内侧有石柱墩，石柱墩上部有盾徽形装饰，石柱墩内侧是曲线矮墙，矮墙上方的金属栅与3.7米宽的两扇大铁花门连成一体，其上挑起铁花灯架。该大门为[[新艺术运动|新艺术运动]]风格，在1970年代初被拆除，改建为如今的两个门柱及铁栅墙<ref name=dhly/>。

使馆主楼坐南朝北，地上二层、地下一层，东西37米、南北27米。因进深大，乃将中央的纵向承重墙处以南的体量较北部向内缩进。东、西侧立面分成各为三开间的两个部分。东、西面以及北面有通向室外的门。外墙用红砖砌成，窗套墙、转角檐部、水平腰线、基座均用石材。基座处地下室带有弧形窗，饰铁花。北主立面分成三个部分，中央的三开间微前凸，大门外罩有石材大门廊，正面和侧面的券洞上部带有金属花饰和吊灯。二层中央的三开间和大门廊连为整体，墙面饰有[[科林斯柱式|科林斯式壁柱]]，三个窗套上部饰弧形山花。屋顶的女儿墙处做成镂空栏杆。南立面中部的六开间是敞廊，饰有方形[[塔司干柱式|塔司干式柱]]，柱身带有水平凹槽划分，中间的两开间是大门，大门前有四开间长的水平台阶通向花园。如今南面这两层的外廊已经被装上窗户。室内层高五米，格局大致为原貌。从北大门进入门厅，左右各有前室可通向东西房间。穿过门厅便是大楼梯厅，可继续进入南部的房间。大楼梯厅东侧是三跑式大楼梯，大楼梯厅西侧设有封闭式楼梯间。二层大厅是跑马廊式，顶部是玻璃顶棚。全部栏杆铁花带有[[新艺术运动|新艺术运动]]风格。门厅和大楼梯厅的地面及墙裙采用[[大理石|大理石]]。粉色墙面用镏金线框划分，门是意大利式门套装饰。厅内采用[[爱奥尼柱式|爱奥尼式柱]]。所有房间是连列厅式，各房间全部贯通。各厅室内有壁炉。原敞廊的地面铺砌彩色[[马赛克|马赛克]]。1999年，该建筑进行了室内外维护和装修<ref name=dhly/>。

使馆大门内北侧是两幢二层官邸建筑。大门内南侧是一层接待用房，其中的一间内有[[南欧|南欧]][[伊斯兰|伊斯兰]]风格室内装修，刻有许多[[阿拉伯文|阿拉伯文]]祈祷经文。这樘装修原先在[[圆明园|圆明园]]西洋楼的“方外观”。“方外观”建于清朝[[乾隆|乾隆]]二十五年（1760年），是为当年进宫的和卓氏“和贵人”（即“[[香妃|香妃]]” ）兴建，形制仿效[[西班牙|西班牙]][[阿尔罕布拉宫|阿尔罕布拉宫]]中的“狮子院”。1901年后，意大利在清朝[[堂子|堂子]]的范围内新建使馆，可能当时从圆明园拆来此樘装修。使馆院内还有圆明园的一对铜狮以及若干石雕构件<ref name=dhly/>。

位于使馆院内东北部的官邸，坐北朝南，平面呈“U”字形，砖木结构，地上二层，长31米，宽17.1米，前有敞廊。灰砖清水墙面，一、二层之间有腰线，腰线饰有卷草纹。墙饰扁方壁柱，窗饰窗套。坡顶檐下有卷草饰带。该官邸在1949年后是[[路易·艾黎|路易·艾黎]]的住所，路易·艾黎逝世后辟为路易·艾黎纪念馆<ref name=dhly/>。

使馆院内西侧是教堂，坐西朝东，砖木结构，地上单层，供使馆人员和兵营士兵使用。长宽分别为26米、14米，平面呈长方形。正门入口朝东，半圆弧形祭坛位于西端。墙面是红色清水砖砌成，檐部及门窗套为灰白石材。立面被四根简化的科林斯式壁柱分成三个部分，中央部分设三个大门，每个大门过梁的上方有半圆形联拱窗。两侧开间的墙上有狭长小窗。侧墙开有长方形窗。壁柱上的檐壁环绕整座教堂。屋顶采用四坡瓦顶，已经经过改建。室内现已改造成礼堂，祭坛改成舞台，门窗已更换。教堂北面有一幢二层西式小楼，与教堂有廊相连，应当是当年神父的住所<ref name=dhly/>。
