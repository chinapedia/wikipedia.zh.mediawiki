'''意大利高速鐵路'''是一個位於[[意大利|意大利]]的[[高速鐵路|高速鐵路]]系統。意大利是歐洲最早實行鐵路高速化的國家，然而與法國[[TGV|TGV]]或日本[[新幹線|新幹線]]不同，意大利起初並不依賴建設全新路線，而採用傾斜列車提速，亦即[[潘多利諾|潘多利諾]]。

意大利的高速鐵路較為特殊。除有[[意大利高速列車|TAV]]公司（本身由意大利國鐵FS營運，公營）以外尚有一家民營高速鐵路企業，名為[[新旅客交通|新旅客交通]]（NTV），列車名為Italo。

==路線網絡==
意大利的首條高速鐵路路線Directissma，[[佛羅倫薩－羅馬高速鐵路|佛羅倫薩－羅馬高速鐵路]]於1970年動工。然而，全線於1992年才通車，前後一共22年。其餘路線則相對較快，由2005年至2009年左右，米蘭－那不勒斯高鐵走廊已經建設完成，都靈－威尼斯高鐵走廊也已部分通車。

=== 米蘭至薩萊諾路線 ===
米蘭至薩萊諾路線是義大利當地主要的南北向高速鐵路路線。

米蘭至博洛尼亞的路線於2008年12月13日分段完工通車。整條路線的建設總計耗費69億[[歐元|歐元]]，長達182公里的路線與[[A1高速公路_(義大利)|A1高速公路]]相互平行前進，總計穿越義大利境內的7個[[省份|省份]]與32個[[直轄市|直轄市]]。此路線沿途與8條既有路線互相連結。在雷焦艾米利亞有與該路線連結的車站，車站由[[瓦倫西亞|瓦倫西亞]]建築師[[聖地亞哥·卡拉特拉瓦|聖地亞哥·卡拉特拉瓦]]進行設計，並於2013年6月車站完工啟用。他還針對橫跨A1高速公路的高鐵路線設計了一座由他屬名的[[橋樑|橋樑]]。高鐵路線於博洛尼亞設有一座多重架構的車站（由於該地是義大利主要鐵路的交匯處）車站由[[日本|日本]]建築師[[磯崎新|磯崎新]]進行設計。

[[File:FS_ETR500-brandAV.jpg|thumb]] AV 在米蘭車站。此車輛是2009年2月4日運行在博洛尼亞 - 佛羅倫斯段上的ETR 500 Y1版本。其最高時速可達到 362 km/h，其曾在隧道運行時締造一個新世界速度紀錄。<ref name="FS355record">{{cite web |url=http://www.corriere.it/cronache/09_febbraio_04/treno_record_galleria_8806fdfa-f2ce-11dd-8878-00144f02aabc.shtml |title=Due record in prova per il Frecciarossa |date=2009-02-04 |publisher=Repubblica |language=it |accessdate=2009-02-05}}</ref>]]

博洛尼亞 - 佛羅倫斯線在2009年12月12日完工通車，使得兩座城市之間僅需耗費37分鐘的行車時間。然而，這段高鐵路線是極為複雜的路線，由於該路線大約有93％的路線（78.5公里）都需要興建[[隧道|隧道]]來通過[[亞平寧山脈|亞平寧山脈]]。博洛尼亞 - 佛羅倫斯線上，總計有9座長600公尺到18.5公里不等的隧道，中間僅有不到5公里的平面路線，然後到達佛羅倫斯後，新建有一座大型的多重結構高鐵車站，這座車站設立在[[貝爾菲奧雷|貝爾菲奧雷]]，是由[[英國|英國]]建築設計師[[諾曼·福斯特|諾曼·福斯特]]所設計。

佛羅倫薩 - 羅馬線的路線中包含了舊的“Direttissima”（字面意思：最直接的），而兩個城市之間的高鐵總長為240公里（150英里）。"''Direttissima''" 是[[歐洲|歐洲]]地區最早完成的高速鐵路路線，它在1977年與1986年之間完工。該路線的升級計畫由[[義大利高速列車|義大利高速列車]]（[[義大利語|義大利語]]：Treno Alta Velocità）完成。當高鐵列車進入羅馬後，列車在羅馬的 Tiburtina 車站可以選擇停在其他的聯運車站，由建築師ABD 設計，並由Paolo Desideri 或是 [[羅馬特米尼車站|特米尼車站]]聯繫推動。

羅馬 - 那不勒斯段的南端起始點位在義大利的[[首都|首都]]，該路線中第一個高速段是在2005年12月開始正式營運。這條路線總計通過在拉齊奧省與坎帕尼亞省境內的61座自治市，並在[[弗罗西诺内|佛羅西羅內]]北部、[[卡西諾|卡西諾]]南方以及[[卡塞塔|卡塞塔]]北方與現有的國家鐵路網連結。2009年12月13日，格里奇尼亞諾到[[拿坡里|拿坡里]]市中心最後的18公里正式完工。在[[坎帕尼亞大區|坎帕尼亞大區]]，高鐵路線穿越[[阿夫拉戈拉|阿夫拉戈拉]]，並且在此設立了一座重要的轉乘站，這座車站是由[[伊拉克|伊拉克]]籍的建築師[[扎哈·哈迪德|扎哈·哈迪德]]所設計。

=== 都靈至第里雅斯特路線 ===
都靈至第里雅斯特線中總長85公里的都靈 - 諾瓦拉段於2006年2月通車。諾瓦拉 - 米蘭段在2009年12月12日通車，讓[[米蘭中央車站|米蘭中央車站]]與[[都靈新門車站|都靈新門車站]]之間行車時間僅需要59分鐘。（米蘭的 Milano P. Garibaldi 車站至都靈的 Porta Susa 車站更只需要45分鐘。）這兩段路線合併後有125公里長，其中有97公里長（約占總長80%）的路線坐落在皮埃蒙特區（都靈，[[韋爾切利省|韋爾切利省]]和[[諾瓦拉省|諾瓦拉省]]），另外27公里（約占總長20％）的路線則坐落在[[倫巴第大區|倫巴第大區]]（[[米蘭省|米蘭省]]）。為了最大限度地降低對[[環境|環境]]的影響，都靈 - 米蘭線的高鐵都是基於既有的路廊上興建，因此該段高鐵路線是依附著A4都靈 - 米蘭高速公路而行的。

米蘭 - 威尼斯段包括從帕多瓦延伸至梅斯特（威尼斯），以及米蘭 - 特萊維奧段目前皆已投入營運。優先軌道工程部分，目前已正在興建中：預計在2016年時，將會開通[[特雷維格里奧|特雷維格里奧]] - [[布雷西亞|布雷西亞]]的高鐵路線。

=== 站點以及歐洲連通 ===
目前，自米蘭連結至熱那亞港口的高鐵路線已開始計劃，並且更進一步的，計畫將路線穿越[[阿爾卑斯山脈|阿爾卑斯山脈]]來使得義大利的高速鐵路系統與路網能夠與[[歐陸|歐陸]]地區的鐵路系統接軌，此計畫由[[歐盟|歐盟]]（EU）與大型聯運泛歐運輸通道（large intermodal pan-European transport corridors）來推動與計劃。

新的阿爾卑斯山鐵路連結其目的是為增加鐵路運輸的路線，然而其真正主要目的是在於支援未來預估的國際鐵路運輸量，使其能夠與歐洲各地的高速鐵路系統完全兼容，讓公路運輸轉換成鐵路運輸模式的運費成本能夠大比例的再平衡。在新的工程技術與修訂過的施工標準之下，穿越阿爾卑斯山的隧道將會擁有更高階的安全水準。

這項計畫中包含新建一條國際路線，並升級以下現有的義大利鐵路：[[弗雷瑞斯|弗雷瑞斯]]（都靈 - 里昂）；聖哥達（[[基亞索|基亞索]] - [[蒙扎|蒙扎]]、{{link-en|加拉拉泰|Gallarate}}以及[[貝林佐納|貝林佐納]]）；[[辛普朗隧道|
辛普朗]]（[[多莫多索拉|多莫多索拉]] - 諾瓦拉）；布倫納（豐澤 - 因斯布魯克）；[[塔爾維肖|塔爾維肖]] - [[塞默靈|塞默靈]]（[[烏迪內|烏迪內]] - 塔維西奧）； 穿越東部 ''Valico Orientale''（威尼斯 - 第里雅斯特 - [[龍基德伊萊焦納里|龍基德伊萊焦納里]]）。

== 未來路線 ==
=== 建設中 ===
* '''米蘭－威尼斯高速鐵路'''：雖然該路線之興建工程已分別在2003年以及2006年收到批准，然而維羅納 - 帕多瓦線至今尚未動工，米蘭 - 威尼斯線則在2011年時開始興建。<ref>{{cite web |url=http://www.rfi.it/cms/v/index.jsp?vgnextoid=3b8cb4050e64c110VgnVCM1000003f16f90aRCRD |title=存档副本 |accessdate=2013-04-26 |0=2013-05-22 |deadurl=yes |archiveurl=https://web.archive.org/web/20130522043357/http://www.rfi.it/cms/v/index.jsp?vgnextoid=3b8cb4050e64c110VgnVCM1000003f16f90aRCRD |archivedate=2013-05-22 }}</ref>米蘭 - 特雷維格里奧線和帕多瓦 - 梅斯特線將成為米蘭 - 威尼斯線的一部分。
* '''米蘭－熱那亞高速鐵路'''：該線路於2006年時批准，於2011年正式動工興建。<ref>{{cite web |url=http://www.rfi.it/cms/v/index.jsp?vgnextoid=fa5db4050e64c110VgnVCM1000003f16f90aRCRD |title=存档副本 |accessdate=2015-03-28 |0=2015-04-02 |deadurl=yes |archiveurl=https://web.archive.org/web/20150402151554/http://www.rfi.it/cms/v/index.jsp?vgnextoid=fa5db4050e64c110VgnVCM1000003f16f90aRCRD |archivedate=2015-04-02 }}</ref>
* '''{{link-en|都靈－里昂高速鐵路|Turin–Lyon high-speed railway}}'''：都靈－里昂線將會串連[[里昂|里昂]]、[[香貝里|香貝里]]和都靈，並且完成後將會加入了法國的 [[TGV|TGV]] 和義大利的 [[TAV|TAV]] 系統。該計劃將會使得[[弗雷瑞斯|弗雷瑞斯]]的鐵路成為重要的連接路線。

=== 規劃中 ===
* '''米蘭－[[基亞索|基亞索]]高速鐵路'''：為一條連結義大利、瑞士和德國的計劃路線，是瑞士 AlpTransit 專案的一個構想，該計劃中包含[[聖哥達基線隧道|聖哥達基線隧道]]和[[勒奇山基線隧道|勒奇山基線隧道]]。
* '''{{link-en|布倫納基線隧道|Brenner Base Tunnel}}'''：布倫納基線隧道將連結[[維羅納|維羅納]]、[[因斯布魯克|因斯布魯克]]和[[慕尼黑|慕尼黑]]，從而成為連接義大利，[[奧地利|奧地利]]和[[德國|德國]]三國的高速鐵路路線。布倫納基線隧道是連結三國高速鐵路一系列計劃中最重要的部分，這個計劃預計將連結德國[[柏林|柏林]]到[[西西里島|西西里島]]的[[巴勒莫|巴勒莫]]，並且納入全歐交通網絡的一部分。2008年12月，歐洲運輸署署長{{link-en|安東尼奧·塔亞尼|Antonio Tajani}}，批准了高達170億[[歐元|歐元]]的資金來針對條路線中11個興建案資助。目的在於完成橫跨[[歐洲|歐洲]][[大陸|大陸]]南北兩地的高鐵路線建設。
* '''[[第里雅斯特|第里雅斯特]] - 跨境[[斯洛維尼亞|斯洛維尼亞]] - 盧比安納'''：連結盧比安納的鐵路路線將能夠促進[[東歐|東歐]]的鐵路系統發展，以及連結義大利的 TAV 系統與斯洛維尼亞的[[潘多利諾|潘多利諾]]系統

==列車==
列車名稱以ETR（Electrico Treno Rapido）開頭，高速電動動車組之意。一共有ETR450、ETR460 、ETR480、ETR485、ETR470（[[齊薩爾皮諾|齊薩爾皮諾]]）、 ETR500（[[意大利歐洲之星|意大利歐洲之星]]）、ETR600（{{link-en|新潘多列諾|New Pendolino}}）、ETR610。另有ETR1000，即AGV575，營運Italo列車。
