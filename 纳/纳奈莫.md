'''纳奈莫'''（{{lang-en|Nanaimo}}）是[[加拿大|加拿大]][[不列颠哥伦比亚|不列颠哥伦比亚]]省西南部[[乃磨地區|乃磨地區]]一座城市，是該地區的行政中心。據2011年加拿大人口普查所示，乃磨市內人口為83810人<ref name="2011census"/>，是[[温哥华岛|温哥华岛]]上人口第二多的地方政體{{noteTag|1=建制為區域自治體的[[沙尼治|沙尼治]]2011年人口達109752人，為溫哥華島上人口最多的地方政體。}}。乃磨都會區人口則達98021人<ref name="2011censusmetro"/>，是溫哥華島上人口第二多的都會區。

“纳奈莫”这个名字来源于[[印第安|印第安]][[原住民|原住民]]的[[薩利希語系|薩利希语]]，其中一個意思为“聚会之地”<ref name="GeoBC"/>。因其天然的深水良港，纳奈莫的官方綽號為“海港之城”（{{lang|en|Harbour City}}）；1986年[[查尔斯王子|查尔斯王子]]和[[戴安娜王妃|戴安娜王妃]]到訪時簽署官方聲明正式確立此綽號<ref>{{Cite web |url=http://www.nanaimo.ca/EN/main/municipal/city-council/TheCitysEmblems.html |title=The City's Emblems |accessdate=2014-02-20 |work=乃磨市政府 }}</ref>。

== 地理位置 ==
纳奈莫位于[[西经|西经]]123度，[[北纬|北纬]]49度，与55千米以东的[[温哥华|温哥华]]隔[[佐治亚海峡|佐治亚海峡]]相望，離省府[[維多利亞_(卑詩)|維多利亞]]西北約111公里。人们直接可以从温哥华乘坐[[卑詩渡輪|卑詩渡輪]]到达这里，进而前往岛上的其他城市。纳奈莫是联系温哥华岛和[[北美大陆|北美大陆]]的重要交通枢纽和物资疏散地。

== 历史和经济 ==
纳奈莫最早作为一个[[殖民|殖民]]者的贸易站点于19世纪早期出现。1851年，当时的[[哈得孙湾公司|哈得孙湾公司]]通过当地原住民获悉这里有大量的[[煤矿|煤矿]]资源，遂于1853年在今天的纳奈莫市中心建立了一个[[碉堡|碉堡]]<ref name="hist">{{Cite web |url=https://www.nanaimo.ca/assets/Departments/Community~Planning/Heritage~Planning/Publications~Forms~Links/Timeline.pdf |title=A Historic Timeline for Nanaimo |accessdate=2014-02-20 |work=乃磨市政府 |deadurl=yes |archiveurl=https://web.archive.org/web/20130903111910/http://www.nanaimo.ca/assets/Departments/Community~Planning/Heritage~Planning/Publications~Forms~Links/Timeline.pdf |archivedate=2013-09-03 }}</ref>，这个碉堡至今仍作为这个城市的[[象征|象征]]而保存着。接下来的几十年里，纳奈莫作为一个重要的煤矿资源产出地而迅速发展，並於1874年正式建制為市<ref name="hist"/><ref name="GeoBC">{{Cite web |url=http://apps.gov.bc.ca/pub/bcgnws/names/9032.html |title=BC Geographical Names: Nanaimo |accessdate=2014-02-20 |work=GeoBC }}</ref>。

1887年，一起煤矿爆炸造成150位矿工丧生<ref name="hist"/>，这是加拿大迄今为止除了1917年的[[哈利法克斯|哈利法克斯]]大爆炸（{{lang|en|Halifax Explosion}}）外最严重的一起人为爆炸事故。20世纪40年代，木材开始取代煤矿成为了这里最主要的贸易资源。今天纳奈莫的重要产业包括[[渔业|渔业]]、林木业以及各种的[[第三产业|第三产业]]，例如[[旅游|旅游]]、[[交通|交通]]等，而[[房地产|房地产]]业在最近几年也得到了相当快的发展。

== 交通 ==
乃磨是溫哥華島東部的交通樞紐。[[卑詩渡輪|卑詩渡輪]]於市内設有三座碼頭：市中心碼頭提供航線前往加布里奧拉島（{{lang|en|Gabriola Island}}）、啓航灣碼頭（{{lang|en|Departure Bay}}）提供航線前往[[西溫哥華|西溫哥華]][[馬蹄灣|馬蹄灣]]、而公爵角碼頭（{{lang|en|Duke Point}}）則提供航線前往[[三角洲_(卑詩省)|三角洲]]托華信。

[[卑詩省1號公路|卑詩省1號公路]]（[[橫加公路|橫加公路]]）溫哥華島段的北端終點位於啓航灣碼頭，自此往南伸延至維多利亞。卑詩省19號公路的南端終點則位於公爵角碼頭，自此往西北伸延至哈迪港。19號公路繞過乃磨市中心的新路段開通後，舊有路段則改編為19A公路。

航空交通方面，離乃磨市以南約13公里的[[乃磨機場|乃磨機場]]（IATA代碼：YCD）提供航線前往[[溫哥華國際機場|溫哥華]]、[[維多利亞國際機場|維多利亞]]、[[亞博斯福國際機場|亞博斯福]]和[[卡加利國際機場|卡加利]]等地，而位於乃磨市以内的乃磨海港水上機場（IATA代碼：ZNA）則提供水上飛機航線前往溫哥華國際機場附近的水上機場以及溫哥華海港水上機場。

乃磨市内的巴士服務由乃磨地區交通局營運。

== 教育 ==
乃磨市內的公共基礎教育服務由68號校區提供。專上教育方面，[[溫哥華島大學|溫哥華島大學]]於2008年從馬拉斯皮納學院升格而成，其主校園設於乃磨市內。

據2011年加拿大人口普查所示，乃磨市內人口為83810人<ref name="2011census"/>，全國排名第63位；乃磨都會區人口則達98021人<ref name="2011censusmetro"/>，全國排名第38位。

== 文化 ==
以乃磨市命名的[[納奈莫條|納奈莫條]]是一種無需烤焗的糕點，在加拿大各地廣受歡迎。另外，每年一度在夏季舉辦的海港賽浴缸活動也是市内盛事之一。
