'''今里筋線'''（{{jpn|j=今里筋線|hg=いまざとすじせん|rm=Imazatosuji sen|f=y}}）是[[大阪府|大阪府]][[大阪市|大阪市]][[東淀川區|東淀川區]]的[[井高野車站|井高野車站]]和[[東成区|東成区]]的[[今里車站_(大阪市營地下鐵)|今里車站]]結成的[[大阪地下鐵|大阪地下鐵]]的路線。正式名稱為'''高速電氣軌道第8號線'''，鐵道要覽中以'''今里筋線（8號線）'''記載。路線代表的英文字母為「'''I'''」。於2006年（平成18年）12月24日開業。

本線在路線圖中以柑子色代表。

== 路線資料 ==
* 路線距離（實際距離）：11.9公里
* [[軌距|軌距]]：1435毫米
* 站數：11個（包括起終點站）
* [[複線鐵路|複線]]路段：全線
* [[電氣化鐵路|電氣化]]路段：全線（直流 1500 [[伏特|V]]、[[高架電纜|高架電纜]]）
* [[閉塞_(鐵路)|閉塞方式]]：車內信號式（[[自動列車制御裝置|CS-ATC]]，[[定位置停止裝置|TASC]]）
* 最高速度：70公里/小時
* 編組輛數：4輛（2006年－）
* 月台最大編組輛數：6輛
* 混雜率（往今里方向）：69.8%（2012年度：鴫野站→綠橋站間）<ref name="konzatsuritsu">{{PDFlink|[http://www.kotsu.city.osaka.lg.jp/library/ct/keieikaigi/h25_5/h24_subway_kotsuryo.pdf 平成24年度 地下鉄・ニュートラム 交通調査の結果について]}} - p.11、大阪市交通局、2014年4月5日閲覧。</ref><!-- 数値を変更する場合は年度・出典も修正を。-->
* 混雜率（往井高野方向）：55.9%（2012年度：鴫野站→蒲生四丁目站間）<ref name="konzatsuritsu"/><!-- 数値を変更する場合は年度・出典も修正を。-->

== 概要 ==
與[[長堀鶴見綠地線|長堀鶴見綠地線]]一樣，本線採用[[線性馬達|線性馬達]]列車。全線月台均在通車時已配備[[月台閘門|月台閘門]]。

[[淀川|淀川]]地下走的鐵道路線為繼[[JR東西線|JR東西線]]路線第二條路線。於JR[[大阪環狀線|大阪環狀線]]的內側通行初的地下鐵路線，[[國道479號|國道479號]]（大阪內環狀線）、[[國道163號|國道163號]]、[[國道1號|國道1號]]（[[京阪國道|京阪國道]]）、[[今里筋|今里筋]]（[[大阪市道大阪環狀線|市道大阪環狀線（都市計劃道路森小路大和川線）]]）的地下走行，於大阪市東部南北縱斷走行，放射狀延的地下鉄路線與[[京阪本線|京阪本線]]，[[片町線|片町線]]（學研都市線）連接。全是[[大阪市|大阪市]]內路線，[[太子橋今市車站|太子橋今市車站]]附近的[[守口市|守口市]]通行。沿線的住宅地居多。

建設費約2,718億日元（井高野 - 今里間）。今後，今里筋之地下的今里之湯里六丁目伸延計劃，本工事費用約1,320億日元。

今里筋線單獨站之改札，既存站新設有關今里筋線約改札，上部設現時刻・發車時刻・列車接近的案內的裝備。月台同樣有發車時刻的表示。

今里筋線開通後，[[今里站_(大阪市營地下鐵)|今里站]] - [[關目成育站|關目成育站]]地的上走行的[[大阪市營巴士|大阪市營巴士]]之班次減少，今里筋線的站間隔比較遠，地上電車換乘時間較久，巴士1小時約2 - 3班的間隔運行。

本線大部份車站設有地下單車泊位。

== 運行形態 ==
全線為[[井高野站|井高野站]] - [[今里站_(大阪市營地下鐵)|今里站]]的運轉，朝夕時有[[井高野站|井高野站]] - [[清水站_(大阪府)|清水站]]的區間列車行走。全線通行的列車平日は繁忙時間1小時有11 - 15班次，午間時1小時有7 - 8班次（約8分鐘的間隔），星期六・休日朝夕時有9 - 10班次，午間時1小時有7 - 8班次（約8分鐘的間隔）。井高野 - 清水的區間列車平日午前7班，午後5班，星期六・休日午前3班、午後2班的運轉方式。其他，平日有清水車站→今里車站的區間列車午前2班運轉。

清水車站與[[新森古市站|新森古市站]]間，在[[花博記念公園鶴見綠地|鶴見綠地（花博記念公園）]]掘削行走，車輛基地（鶴見綠地北車庫）建設。在1km內的[[長堀鶴見綠地線|長堀鶴見綠地線]]的[[鶴見綠地站|鶴見綠地站]]設置，本區間連接同線的工場（[[鶴見檢車場|鶴見檢車場]]）共用。專用的車輛工場與其他的[[大阪地下鐵|大阪地下鐵]]路線共用──[[千日前線|千日前線]]與[[中央線|中央線]]的[[森之宮檢車場|森之宮檢車場]]配置。

== 歷史 ==
地下鐵今里筋線行通今里筋，1957年以巴士運行，[[大阪市電|大阪市電]]與大阪市內東部與都心部連結的職責擔當。汽車交通量增加使該段巴士線在1969年廢止，以[[大阪市營巴士|市營巴士]]承繼路線，受塞車的交通機關非常困擾，放射鐵道路線補充的鐵道路線，新的地下鐵建設計劃誕生了。

1989年的[[運輸政策審議會|運輸政策審議會]]第10號報告，今後整備之檢討的路線為-{[[上新庄車站|上新庄車站]]}- - [[湯里六丁目車站|湯里六丁目車站]]間。今後、-{上新庄}-的經路為[[國道479號|國道479號]]（[[大阪內環狀線|大阪內環狀線]]）與[[東海道新幹線|東海道新幹線]]的交差點（大隅1交差點）的地下構建物的障礙，起点在1996年當時[[東淀川區|東淀川區]]內でも住宅開發，人口急增的[[井高野車站|井高野車站]]變更，1999年，井高野車站 - 今里車站間的軌道事業許可在翌年──2000年3月着工，2006年12月24日開業。餘下的今里車站 - [[湯里六丁目車站|湯里六丁目車站]]間未着手動工。東淀川區的人口約18萬人（2006年現在），大阪市的區內[[平野區|平野區]]、[[淀川區|淀川區]]第3人口多的區域。

* 2006年（[[平成|平成]]18年）7月6日 愛稱、車站名稱正式決定。
* 2006年（平成18年）12月24日 井高車野 - 今里車間(11.9km)開業。
