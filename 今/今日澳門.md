《'''今日澳門'''》（{{lang-pt|'''Hoje Macau'''}}），是在1990年創刊、於[[澳門|澳門]]發行的[[葡萄牙語|葡萄牙語]]綜合性日報[[報章|報章]]，該報的內容以當地消息為主，讀者群主要為澳門官員、商界以及葡人社群<ref>{{cite news|title=今日澳門|url=http://yoliving.com/index.php/tw/life-snapshot/28-life-snapshot/532-2014-11-12|accessdate=2018-02-01|agency=Yoliving|date=2014-11-12|deadurl=yes|archiveurl=https://web.archive.org/web/20180201192928/http://yoliving.com/index.php/tw/life-snapshot/28-life-snapshot/532-2014-11-12|archivedate=2018-02-01}}</ref><ref name ="wuchan2009">{{cite book|author1=吴志良|coauthors=陈震宇|title=《澳门人文社会科学研究文选：综合卷》|date=2009年|publisher=社会科学文献出版社|isbn=9787509711293|page=334}}</ref>。自2001年起報社所有權轉讓以來，社長的職位由在[[葡萄牙|葡萄牙]]出生、同時身兼[[記者|記者]]和[[作家|作家]]的[[左凱士|左凱士]]（{{lang-pt|Carlos Morais José}}）所擔任<ref>{{cite news|title=Uma semana para lembrar Pessanha, poeta de Macau|url=http://www.plataformamacau.com/macau/uma-semana-para-lembrar-pessanha-poeta-de-macau/|accessdate=2018-02-01|agency=Plataforma Macau|deadurl=no|archiveurl=https://web.archive.org/web/20180201192925/http://www.plataformamacau.com/macau/uma-semana-para-lembrar-pessanha-poeta-de-macau/|archivedate=2018-02-01}}</ref><ref>{{cite web|title=Carlos Morais José|url=http://macaucloser.com/zh-hant/magazine/carlos-morais-jos%C3%A9-0|website=Macau Closer|accessdate=2018-02-01|deadurl=no|archiveurl=https://web.archive.org/web/20180408133244/http://macaucloser.com/zh-hant/magazine/carlos-morais-jos%C3%A9-0|archivedate=2018-04-08}}</ref>。

《今日澳門》以日報形式發行，逢星期六、日及公眾假期休刊，平日一般刊印十六版，星期五內容會加印至約二十四版，常設版面內容可粗略分為[[新聞|新聞]]（[[政治|政治]]、[[社會|社會]]、特寫分析）、評論（社論及專欄）、[[體育|體育]]消息與[[藝術|藝術]]文化，較前版面及末頁則會加入國際及經濟新聞。該報全部內容均上載至官方[[網頁|網頁]]<ref>{{cite web|title=Jornal Hoje Macau|url=https://issuu.com/hojemacau|website=Hoje Macau|accessdate=2018-02-01|deadurl=no|archiveurl=https://web.archive.org/web/20180207073228/https://issuu.com/hojemacau|archivedate=2018-02-07}}</ref>。

該報為新聞出版廠有限公司（Fábrica de Notícias, Lda）所有，負責人為左凱士，該公司由一群股份數量和身份未披露的私人[[股東|股東]]組成<ref>{{cite web|title=澳門今日|url=http://www.gcs.gov.mo/index.php?PageLang=C&PageCode=PUB&PassType=PUB|website=澳門特別行政區新聞局|accessdate=2018-02-01|deadurl=no|archiveurl=https://web.archive.org/web/20170908054053/http://www.gcs.gov.mo/index.php?PageLang=C&PassType=PUB&PageCode=PUB|archivedate=2017-09-08}}</ref><ref name ="p2010"/>。

==歷史==
《今日澳門》創刊於1990年，其葡語名稱原為「Macau Hoje」。由於多年來受到十多起大部分由[[律師|律師]][[華年達|華年達]]（{{lang-pt|Jorge Neto Valente}}）所提出的涉嫌違反[[新聞自由|新聞自由]]和[[誹謗|誹謗]]的法律訴訟的關係，「Macau Hoje」最終在2001年因罰款和[[法律|法律]][[訴訟|訴訟]]所帶來的財務問題而停刊。同年，《今日澳門》的所有權轉讓至新聞出版廠有限公司，其葡語名稱從「Macau Hoje」改為「Hoje Macau」，中文譯名以及原來的[[記者|記者]]和員工得以保留，並於9月5日宣布正式出版<ref name="p2010">{{cite news|title=Diário "Hoje Macau" substitui "Macau Hoje"|url=https://www.publico.pt/2001/09/05/portugal/noticia/diario-hoje-macau-substitui-macau-hoje-38406|accessdate=2018-02-01|agency=Publico|date=2001-09-05|deadurl=no|archiveurl=https://web.archive.org/web/20180408133244/https://www.publico.pt/2001/09/05/portugal/noticia/diario-hoje-macau-substitui-macau-hoje-38406|archivedate=2018-04-08}}</ref><ref name ="wuchan2009"/>。新社長左凱士在社論中說，《今日澳門》（Hoje Macau）是首個把自身定義為當地的葡萄牙語報紙，而不是葡萄牙報紙<ref name="mdn01"/>，他又表示該報致力於反映澳門人的渴求和意見，以及維護葡語作為澳門特區[[官方語言|官方語言]]的地位<ref name ="p2010"/>。

1999年[[澳門主權移交|澳門主權移交]]後，閱讀葡文報刊的讀者驟減，為擴大讀者群，一些當地葡文報紙開始出版英文刊物 ，例如《[[句號報|句號報]]》曾於1999年首先出版英文增刊，但效果並不理想。《今日澳門》亦一度短暫附設[[英文|英文]]版，2002年2月，《今日澳門》出版英文周報《新時代》（New Times），每期銷售量約在600份左右，與大部份葡文報章的銷售量相若。同年八月，《新時代》因[[廣告|廣告]]收入無以為繼導致嚴重虧損而終刊<ref>{{cite web|author1=林玉鳳|title=回歸後澳門葡文報章的生存策略|url=http://rthk9.rthk.hk/mediadigest/20041014_76_120137.html|website=香港電台|accessdate=2018-02-01|deadurl=no|archiveurl=https://web.archive.org/web/20180201192916/http://rthk9.rthk.hk/mediadigest/20041014_76_120137.html|archivedate=2018-02-01}}</ref><ref>{{cite journal|author1=林良玉|coauthors=吳玫|title=《全球化視野下的澳門：澳門英文報業的 新聞框架研究》|journal=澳門研究|date=2011|issue=2|page=112-113}}</ref>。
