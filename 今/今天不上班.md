{{日本作品|今天不上班|きょうは会社休みます。}}，是[[日本|日本]]作家[[藤村真理|藤村真理]]的[[漫畫|漫畫]]作品，於少女[[漫畫雜誌|漫畫雜誌]]《{{link-ja|Cocohana|Cocohana}}》（[[集英社|集英社]]）的2012年1月號（創刊號）起開始連載。[[單行本|單行本]]至2014年9月已發行7本<ref name="kikan">{{Cite web |url=http://books.shueisha.co.jp/CGI/search/zen_list.cgi?siries_isbn=X78-4-08-846769-6&siries_kanren_isbn=&mode=2 |title=きょうは会社休みます。/ 第1巻〜第7巻｜藤村真理｜マーガレットコミックス｜BOOKNAVI |publisher=集英社 |accessdate=2014-09-25}}</ref>。

本漫畫於2013年獲選為第17屆日本[[文化廳媒體藝術祭|文化廳媒體藝術祭]]審查委員推薦作品<ref>[http://archive.j-mediaarts.jp/festival/2013/manga/works/17mj_KyowaKaishaYasumimasu/ 第17回 2013年 マンガ部門 審査委員会推薦作品 きょうは会社休みます。]</ref>。2014年10月，[[日本電視台|日本電視台]]將其連續劇化，於「[[日本電視台週三連續劇|週三連續劇]]」時段晚間十點播放。

== 故事內容 ==
在[[人生|人生]]之中尚未嘗試過與男性交往的[[OL|OL]]青石花笑，在她迎接自己33歲（電視劇中為30歲）[[生日|生日]]的時候悲嘆自己仍是[[處女|處女]]之身。當晚花笑跟在公司當[[兼職|兼職]]的青年田之倉一起飲酒，之後花笑醉倒了，直到另一日的早上睜開眼睛，才發覺自己原來與田之倉共度了一夜。因為酒醉失憶，花笑完全沒有「破處」的實感。在共度一夜後，田之倉竟向花笑提出交往。就這樣，花笑人生中的初次戀愛正式開始了……。

== 登場人物 ==
; 青石花笑
: 事務職[[OL|OL]]。33歲，33年來不曾有過男友，處女。
: 每天最早到公司，樸素不起眼、一板一眼的員工，33歲生日當晚與田之倉悠斗酒後發生了關係。雖好友建議應該分手結束關係，但青石仍然決定拿出勇氣踏出一步，開始從外觀之上改變自己的形象。在酒醉之後會做出大膽的行動。
; 田之倉悠斗
: 21歳。[[大學生|大學生]]。在花笑的公司打工。有豐富戀愛經驗。大学畢業後預定會進[[研究所|研究所]]學習。
; 朝尾侑
: 與花笑公司同一大廈，三樓入口食品公司的[[CEO|CEO]]。35歳。擅長應付女性，是展現出「做什麼都可以」的男性。
: 從他人眼中注意到外表俗氣平凡的花笑。是花笑的戀愛顧問，在調侃花笑時亦經常提出意見幫助她。後來離任CEO，成立了自己的公司。
