'''褐耳鹰'''（[[学名|学名]]：{{lang|la|''Accipiter badius''}}）为[[鹰科|鹰科]][[鹰属|鹰属]]的[[鸟类|鸟类]]，从[[南亚|南亚]]到[[撒哈拉以南非洲|撒哈拉以南非洲]]都有分布。一般栖息于开阔的林地、灌木间的草丛以及筑巢于多叶的树上。该物种的模式产地在[[斯里兰卡|斯里兰卡]]。<ref name='id1473'/>

== 保护 ==
* [[中國國家重點保護野生動物名錄|中国国家重点保护动物]]等级：二级

== 亚种 ==
* '''褐耳鹰新疆亚种'''（[[学名|学名]]：{{lang|la|''Accipiter badius cenchroides''}}）。在[[中国大陆|中国大陆]]，分布于[[新疆|新疆]]等地。该物种的模式产地在俄罗斯。<ref name='id1474'>{{cite web| url =http://www.bioinfo.cn/db05/BjdwSpecies.php?action=view&id=1474| title =褐耳鹰新疆亚种| work =《中国动物物种编目数据库》| publisher =中国科学院微生物研究所| author =中国科学院动物研究所| accessdate =2009-04-04| deadurl =yes| archiveurl =https://web.archive.org/web/20160305080242/http://www.bioinfo.cn/db05/BjdwSpecies.php?action=view&id=1474| archivedate =2016年3月5日| df =}}</ref>
* '''褐耳鹰南方亚种'''（[[学名|学名]]：{{lang|la|''Accipiter badius poliopsis''}}）。在[[中国大陆|中国大陆]]，分布于[[云南|云南]]、[[贵州|贵州]]、[[广西|广西]]、[[广东|广东]]、[[海南|海南]]等地。该物种的模式产地在缅甸北部。<ref name='id1475'>{{cite web| url =http://www.bioinfo.cn/db05/BjdwSpecies.php?action=view&id=1475| title =褐耳鹰南方亚种| work =《中国动物物种编目数据库》| publisher =中国科学院微生物研究所| author =中国科学院动物研究所| accessdate =2009-04-04| deadurl =yes| archiveurl =https://web.archive.org/web/20160305033132/http://www.bioinfo.cn/db05/BjdwSpecies.php?action=view&id=1475| archivedate =2016年3月5日| df =}}</ref>
