《'''這個世界的角落'''》（{{lang-ja|この世界の片隅に}}）為日本漫畫家[[河野史代|河野史代]]創作的[[日本|日本]][[漫畫|漫畫]][[作品|作品]]，於2007年1月23日號～2009年1月20日號的「[[漫画Action|漫画Action]]」（[[雙葉社|雙葉社]]）上連載。漫畫單行本同樣由雙業社出版，全書共分上、中、下3集。台灣中文版分為上、下2集，2014年6月26日由漫遊者出版社出版，書名為《'''謝謝你，在這世界的一隅找到我'''》，書名取自來自鈴與丈夫周作重遇時所說的一句話。

同名作品分別榮獲第13回（2009年）[[文化廳媒體藝術祭|文化廳媒體藝術祭]]漫畫部門優秀賞、「THE BEST MANGA 2010 {{lang|ja|このマンガを読め!}}｣（{{tsl|ja|フリースタイル (出版社)|FREE STYLE}}）和「{{Lang|ja|ダカーポ特別編集 最高の本!}} 2010」（Magazine House Mook）漫畫部門的第1名。

2011年8月5日由[[日本電視台|日本電視台]]改編為「終戰紀念特別劇」，由佐藤東彌執導，[[北川景子|北川景子]]、[[小出惠介|小出惠介]]主演。2012年8月17日由日本動畫公司[[MAPPA|MAPPA]]改編為同名劇場版動畫，由片淵須直執導，2016年11月在日本全國劇院上映<ref name=natalie160824>{{cite news|url=http://natalie.mu/eiga/news/199167|title=のんがアニメ映画「この世界の片隅に」で主演、「地面からふわっと浮いちゃいそう」|newspaper=映画ナタリー|date=2016-08-24|accessdate=2016-08-24}}</ref>。2018年[[TBS電視台|TBS電視台]]改編為連續劇，由[[土井裕泰|土井裕泰]]執導，[[松本穗香|松本穗香]]、[[松坂桃李|松坂桃李]]主演。

== 劇情概要 ==
出身[[廣島市|廣島市]]江波製[[海苔|海苔]]工家庭的浦野鈴，是個有點傻氣、擅長畫畫的女孩，在窮困、平凡卻又偶爾參雜著怪事地長大成人後、她與幼年時曾有一面之緣、居住在軍港[[吳市|吳市]]的海軍勤務兵北條周作結婚。在那裡過著受嚴格的小姑径子督導，被可愛的姪女晴美纏著的，再普通不過的夫婦生活。但隨著第二次世界大戰爆發，戰爭的影響開始深入生活的每個層面，雖然個性樂天的阿鈴努力地去適應這種生活，但逐漸延燒到日本本土的戰火終究從她那兒奪走了重要的事物……<ref>http://www.kingstone.com.tw/book/book_page.asp?kmcode=2018611145810</ref><ref>http://www.kingstone.com.tw/book/book_page.asp?kmcode=2018611145650</ref>

== 登場人物 ==
; {{nihongo|浦野鈴→北條鈴|浦野 すず（うらの すず） → 北條 すず（ほうじょう すず）}}
: 生于广岛市江波的女性，旧姓浦野。非常喜欢画画。幼年时曾有过遇到座敷童子等妖怪的不可思议的经历。二战末期，18岁的她来到吴市，嫁给北条家的长男周作，开始了以婆家北条家为中心的新生活。在战争中因被美軍空襲的定时炸彈而失去了一只手。
; {{nihongo|北條周作|北條 周作（ほうじょう しゅうさく）}}
: 北条家长男，在海军军事法庭做书记官。小时候就曾与铃有过一面缘，长大后找到了铃并与她结婚。婚后，非常温柔地对待在陌生的地方和环境下开始新生活的铃。战争最后阶段，身为文职官的他也不得不加入战斗部队。
; {{nihongo|北條燦|北條 サン（ほうじょう さん）}}
: 
; {{nihongo|北條圓太郎|北條 円太郎（ほうじょう えんたろう）}}
: 
; {{nihongo|黑村徑子|黒村 径子（くろむら けいこ）}}
: 周作的姐姐，在丈夫病死後与婆家断绝关系，带着女儿晴美回到北条家，儿子却留在黑村家。办事爽快麻利，与弟媳妇铃形成了鲜明的对比。平时对铃的态度很严厉。
; {{nihongo|黑村晴美|黒村 晴美（くろむら はるみ）}}
: 周作的外甥女，径子的女儿，5岁。随母亲来到位于吴市的外祖父家生活，与舅妈铃相处得很融洽，二人每天在一起画、眺望港口的大船。最後被美軍空襲的定时炸彈給炸死。
; {{nihongo|黑村久夫|黒村 久夫（くろむら ひさお）}}
: 周作的外甥，晴美的哥哥，被母親徑子留在黑村家，原本準備慶祝妹妹晴美入學而將自己使用的教科書送給了她。
; {{nihongo|無名孤女|}}
: 戰爭遺孤，原本是軍官的女兒，父親戰死後與母親在廣島相依為命。[[廣島原爆|廣島原爆]]時，不幸失去母親與家園，在挨餓流浪中偶遇鈴和周作。之後被兩人收養帶回北條家養育。
=== 江波 ===
; {{nihongo|浦野十郎|浦野 十郎（うらの じゅうろう）}}
: 鈴的父親，從事海苔收割業務。美軍投下原子彈後為了尋妻前往廣島市內，結果遭遇強烈輻射，10月份就因為輻射去世。
; {{nihongo|浦野澄|浦野 すみ（うらの すみ）}}
: 鈴的妹妹，昭和元年（1926年）出生，是個美人。在陸軍軍需工廠做女子挺身隊隊員時與一名男子交往。美軍投下原子彈後與父親一起進入廣島市尋母，後也因遭遇過量核輻射導致內出血而死。
; {{nihongo|水原哲|水原 哲（みずはら てつ）}}
:铃的小学同学、孩子王。加入海军后，成为巡洋舰“青叶”的船员。趁上岸的机会，到北条家来看望铃。他的出现，让铃和周作的心情变得复杂起来。最後戰死。
