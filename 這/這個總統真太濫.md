{{作品名稱|name=''Primary Colors''|hk=這個總統真太濫|tw=風起雲湧|equiv-cn=tw}}是一套1998年的美國政治電影，由[[米克·尼高斯|米克·尼高斯]]執導。內容改編自{{link-en|影射小說|Roman à clef}}《''{{link-en|Primary Colors (novel)|Primary Colors (novel)|Primary Colors: A Novel of Politics}}''》，影射{{link-en|1992年比爾·克林頓總統競選|Bill Clinton presidential campaign, 1992}}，最初以匿名出版，但在1996年被披露作者是記者{{link-en|Joe Klein|Joe Klein}}，他為《[[新聞週刊|新聞週刊]]》報道克林頓的競選。<ref name="what the">{{Cite web |url = http://articles.latimes.com/1998/mar/15/entertainment/ca-28949 |title = What the Movie Gets --and What It Doesn't |author = David Lauter |date = 1998-03-15 |publisher = The Los Angeles Times |language = en}}</ref><ref>{{Cite web |url = http://articles.latimes.com/1998/mar/02/entertainment/ca-24508 |title = Just What He Didn't Need Right Now |author = Faye Fiore |date = 1998-03-02 |publisher = The Los Angeles Times |language = en}}</ref><ref>{{Cite web |url = http://www.nytimes.com/1996/07/18/us/columnist-s-mea-culpa-i-m-anonymous.html |title = Columnist's Mea Culpa: I'm Anonymous |author = Doreen Carvajal |date = 1996-07-18 |publisher = The New York Times |language = en}}</ref>主演有[[尊·特拉華特|尊·特拉華特]]、[[愛瑪·湯遜|愛瑪·湯遜]]、[[比利·卜·科頓|比利·卜·科頓]]、[[嘉菲·比絲|嘉菲·比絲]]、{{link-en|毛拉·蒂爾尼|Maura Tierney}}、{{link-en|Larry Hagman|Larry Hagman}}和[[艾哲倫·雷斯特|艾哲倫·雷斯特]]。

比絲憑她的演出被提名[[奧斯卡最佳女配角|奧斯卡最佳女配角]]，而伊萊恩·梅則被提名[[奧斯卡最佳改編劇本|奧斯卡最佳改編劇本]]

== 製作 ==
繼1996年出版這本著作後，導演米克·尼高斯為電影版權支付了超過100萬美元。本片由伊萊恩·梅寫劇本，他與尼高斯1950年代和1960年代合作過{{link-en|尼高斯與梅|Nichols and May|喜劇雙人表演}}。<ref>{{Cite web |url = http://www.vanityfair.com/hollywood/2013/01/nichols-and-may-reunion-exclusive |title = Who's Afraid Of Nichols & May? |author = Sam Kashner |publisher = Vanity Fair |language = en}}</ref>

在[[康城影展|康城影展]]上，湯遜說她的演出沒有基於[[希拉莉·克林頓|希拉莉·克林頓]]，而特拉華達說他是基於幾個總統，但主要是[[比爾·克林頓|比爾·克林頓]]。<ref name="travolta reveals">{{Cite web |url = http://www.independent.co.uk/news/travolta-reveals-clintons-prime-sense-of-humour-1159912.html |title = Travolta reveals Clinton's prime sense of humour |author = David Lister |publisher = The Independent |language = en}}</ref>

尼高斯因從電影最終版本中剪走了一個不同種族之間的愛情場面而被批評。他回應已經刪除了這個場面是因為試映觀眾的負面反應。<ref name="travolta reveals" />

本片也引起了爭議，因為它描述了一個類似克林頓的角色，而當時本片的上映也接近[[萊溫斯基醜聞|萊溫斯基醜聞事件]]。<ref name="what the" /><ref>{{Cite web |url = http://articles.latimes.com/1998/may/16/entertainment/ca-50153 |title = With Its American Themes, Can 'Primary Colors' Cash In Abroad? |author = Mark Saylor |date = 1998-05-16 |publisher = The Los Angeles Times |language = en}}</ref><ref>{{Cite web |url = http://articles.latimes.com/1998/mar/15/entertainment/ca-28947 |title = They All Have a Secret |author = Patrick Goldstein |date = 1998-03-15 |publisher = The Los Angeles Times |language = en}}</ref><ref>{{Cite web |url = http://content.time.com/time/magazine/article/0,9171,987986,00.html |title = Cinema: Tale Of Two Bills |author = Eric Pooley |date = 1998-03-16 |publisher = Time |language = en}}</ref>
