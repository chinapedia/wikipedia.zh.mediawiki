== 位置 ==
这座车站位于[[莲湖路|莲湖路]]与[[洒金桥|洒金桥]]路口。

==事故== 
2009年8月2日上午9时20分，洒金桥站施工现场，在冠梁沟槽开挖支护过程中出现的约十方泥土塌方，两名正在作业的工人不幸被埋，送往医院后经全力抢救无效身亡。<ref>{{Cite web |url= http://news.hsw.cn/system/2009/08/02/050261599.shtml |title= 西安地铁一号线洒金桥站塌方 2名工人被埋身亡 |author=  |date=  2009-08-02 |publisher= 华商网 |language= zh-hans |accessdate= 2014年4月17日 |quote=  }}</ref>逝世的两名工人是53岁的四川籍工人甘祥宾和22岁的云南籍工人涂千军。<ref>{{Cite web |url= http://news.hsw.cn/system/2009/08/02/050261672.shtml |title= 西安地铁一号线施工现场塌方 2死亡工人身份确认 |author=  |date= 2009-08-02  |publisher= 华商网 |language= zh-hans |accessdate= 2014年4月17日 |quote=  }}</ref>
