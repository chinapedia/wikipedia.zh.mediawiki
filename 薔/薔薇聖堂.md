《'''薔薇聖堂'''》（[[日語|日語]]：{{lang|ja|薔薇の聖堂}}）是日本[[視覺系|視覺系]]搖滾樂團 [[MALICE_MIZER|MALICE MIZER]] 發行於2000年8月23日的第四張也是最後一張[[錄音室專輯|錄音室專輯]]，<ref name="CDJa1">{{cite web |url=http://www.cdjapan.co.jp/product/MMCD-13 |title=Bara no Seidou |accessdate=2017-03-31|publisher=CDJapan}}</ref>標題直譯成中文為「薔薇花的敎堂」之意。專輯於2007年1月26日由[[德國|德國]]唱片公司 {{link-en|Trisol Music Group}} 在歐洲發行，附加一本20頁寫眞圖冊以及樂團曾於2002年在日本單獨發行過的DVD《Cardinal》，<ref name="jame">{{cite web |url=http://jame-world.com/uk/news-19521-malice-mizer-european-release-for-bara-no-seidou.html |title=Malice Mizer: European release for Bara no Seidou |accessdate=2017-04-01|author=Geisha |date=14.01.2007 12:00 |publisher={{link-es|JaME}}. Japanese Music Entertainment}}</ref><ref name="CDJa3">{{cite web |url=http://www.cdjapan.co.jp/product/MMBV-28 |title=Cardinal |accessdate=2017-04-01|publisher=CDJapan}}</ref>DVD收錄專輯中三支[[單曲|單曲]]的[[音樂錄影帶|音樂錄影帶]]，此外還包括《Gardenia》和《Beast of Blood》的額外版[[音樂錄影帶|PV]] ({{lang|en|Alternative version}})。<ref name="Dis">{{cite web |url=https://www.discogs.com/Malice-Mizer-Bara-No-Seidou/release/1569241 |title=Bara No Seidou (CD, Album, Ltd + DVD-V, PAL) |accessdate=2017-03-31|publisher=[[Discogs|Discogs]]}}</ref>專輯於[[Oricon|Oricon]]音樂排行榜上名列17。<ref name="Ori">{{cite web |url=http://oriconv.seesaa.net/article/158589410.html |title=【MALICE MIZER】のオリコン順位と売り上げ枚数 |accessdate=2017-03-31|publisher=oriconv.seesaa.net}}</ref>

== 概述 ==
相較於 MALICE MIZER 以前的專輯，《薔薇聖堂》的風格有很大改變。這張作品不再如從前那樣注重雙吉他搭配的音色，但保留了類似[[創世紀樂團|創世紀樂團]]和[[羅西音樂|羅西音樂]]那種變幻無窮的音樂特質。樂團於此作中的整體音色風格由先前的[[藝術搖滾|藝術搖滾]]轉變成更為黑暗的[[哥德金屬|哥德金屬]]，不過他們仍然繼續使用[[小提琴|小提琴]]、[[古鍵琴|古鍵琴]]以及樂團一度慣用的其他樂器，但新增了[[管風琴|管風琴]]和[[合唱|唱詩班]]元素，使專輯充滿一種[[中世紀音樂|中世紀]]-[[巴洛克音樂|巴洛克式]]的暗魅[[抑鬱_(情緒)|憂鬱]]和宗教感。例如，《{{lang|ja|聖なる刻　永遠の祈り}}》一曲就是將管風琴和唱詩音色發揮到極致的作品，但也同時運用了雙吉他和[[電子鼓|電子鼓]]，以此讓歌曲風格在古典的基調下，又予人一種[[哥德金屬|哥德金屬]]的感覺。樂迷最愛的《{{lang|ja|虚無の中での遊戯}}》也是以管風琴作基調，再加入古鍵琴、電子鼓以及[[合成器|合成器]]的音色，讓此曲更貼近{{tsl|en|Dark wave|暗潮音樂}}風格。《{{lang|ja|虚無の中での遊戯}}》受到更多[[哥德搖滾|哥德搖滾]]和[[工業音樂|工業音樂]]的影響，因此，相較專輯裡的其他歌曲，此曲的[[音樂錄影帶|音樂錄影帶]]也給人一種不同的映像，但仍舊保持暗魅元素。<ref name="Eve">{{cite web |url=https://everything2.com/title/Bara+no+Seidou |title=Review of Bara no Seidou |accessdate=2017-04-01|publisher={{link-en|Everything2}}}}</ref>

== 演唱會 ==
一張題為《{{lang|ja|薔薇に彩られた悪意と悲劇の幕開け}}》的演唱會DVD於2000年11月22日發行，<ref name="CDJa2">{{cite web |url=http://www.cdjapan.co.jp/product/MMBV-15 |title=Bara ni Irodorareta Akui to Higeki no Makuake |accessdate=2017-04-01|publisher=CDJapan}}</ref>收錄了樂團以《薔薇聖堂》專輯為主題在[[日本武道館|日本武道館]]擧行的雙日演唱會。這是一場非常戲劇化的演唱會，某些表演帶有一定情節，也有穿着[[中世紀|中世紀]]修士袍服裝的「唱詩班」表演合聲。舞臺上設置了一幅取自專輯封面的[[哥德式建築|哥德式]]敎堂巨型圖像作為背景，敎堂上空的圓月也一併出場，並在演出過程中多次改變月亮的顏色，以此製造幻惑的氛圍。
