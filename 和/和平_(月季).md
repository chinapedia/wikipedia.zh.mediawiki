'''和平'''[[月季|月季]], 正确地说， “A.梅兰德夫人”("Madame A. Meilland")月季, 是一种著名且成功的[[园艺月季|园艺月季]]。截止1992年,此种月季已售出逾百万。其具有浅黄，接近奶油色的大花。花瓣边缘略微泛绯红。其适应力和生命力强，且具有一定的抗病性，这使得其在花卉贸易和园艺中相当出名。

在1935-1939年中，其被[[法国|法国]]园艺家[[弗朗西斯·梅兰德|弗朗西斯·梅兰德]]所开发。当梅兰德预见到德国将侵略法国时,为了保护这种新月季，其将枝条送到其在意大利、土耳其、德国和美利坚合众国的朋友。据信其在侵略之前最后一班可用的航班上被送往美国，并在战时于美国被[[科纳尔·派尔|科纳尔·派尔]]公司繁殖。 <ref name=meilland>{{cite book|title=Meilland, a Life in Roses |last1=Meilland |first1=Alain |last2=Lambert |first2=Gilles |pages=65-71 |date=April 1984 |publisher=Southern Illinois Univ Pr |isbn=0809311119 }}</ref>

== 培育 ==
[[File:Gloria_Dei_Bamberg.jpg|thumb]]
此栽培种在1935年被杂交出,并被编号为3-35-40 (1935年的第三个杂交种,为测试繁殖选定的第四十个栽培种). 由于第一次测试在次年夏天产出了外形可观的花，39年六月，第一个3-35-40植株被种到梅兰德月季田。是年夏季，枝条被送到其他国家的合伙人。据梅兰德回忆，“A.梅兰德夫人” 是从一个未命名植株和杂交茶香“玛格丽特·麦克格雷迪”("Margaret McGredy")月季杂交而来。<ref name=blv>{{cite book |title=Die BLV Rosen Enzyklopädie |publisher= BLV |author= Robert Markley |page=27 |ISBN= 978-3-8354-0272-0 |language= de |trans_title= The BLV Encyclopedia of Roses|date=2007}}</ref>

== 命名 ==
因为梅兰德在战事迫在眉睫的时候送出其枝条，栽培者间便不可能沟通，这也就是此月季有多重名字的原因。在法国，弗朗西斯·梅兰德和阿兰·梅兰德为了纪念弗朗西斯已故的母亲，阿兰的夫人克劳迪娅<ref name=blv>{{cite book |title=Die BLV Rosen Enzyklopädie |publisher= BLV |author= Robert Markley |page=27 |ISBN= 978-3-8354-0272-0 |language= de |trans_title= The BLV Encyclopedia of Roses|date=2007}}</ref>，决定命名此栽培种为“A.梅兰德夫人”。这是其早期的名字。其他的名字是由国际栽培种命名名录（International Code of Nomenclature for Cultivated Plants,ICNCP）命名的，作为贸易或是商品名使用。在意大利，其叫做"Gioia"（意大利语，欢乐），在德国，其叫做"Gloria Dei"（拉丁语，主（神）的荣耀），在美国、瑞典和挪威，其被称作"Peace"（英语，和平）。

在1945年早期，月季栽培者梅兰德给在第二次世界大战中使用的总策略的主要撰写者，英国陆军元帅艾伦·布鲁克（晚些被称作艾伦布鲁克子爵）写信，询问其是否可以以他的名字来给此月季命名，以感谢其对法国解放的重大贡献。布鲁克回绝了，但仍感谢其为此询问。他指出，其名将速被遗忘，而“和平”是一个更好的和更不朽的名字。就这样，此月季变得著名。

正式通过“和平”一名的决定被引种者[[科纳尔·派尔|科纳尔·派尔]]公司于1945年4月29日公布。这是个很特别的日子——是日柏林陷落，其被认作二战中欧洲战场的转折点。<ref name="meilland" />之后于是年和平月季被赠送给于位于旧金山的联合国机构，于谈判中的首次会议中被陈设。每朵月季花上都有一张便条，上面写道：“我们希望人们追求永续世界和平的想法，将伴着这和平月季协和而生。”（直译：“我们相信这朵和平月季将会影响人们关于永远的世界和平的想法。”）

英国的月季种植专家皮特·比尔斯，在其著作《月季》（Rose）中写道：“‘和平’，毫无疑问地，是最佳的杂交茶香品种，而其将会成为该品种群的一根不朽的准绳。”

== 性状 ==
“A.梅兰德夫人”植株会形成优雅的花苞，何者其全备绽放为硕大的中高形(英：high-centered)杯状花。花朵的平均直径是15cm，其颜色基于地点、天气状况等因素而定，一般由淡黄色和边缘的绯红色构成。其不易凋谢的花朵极其丰满，具有40-43片花瓣，能在雨季正常开花。其具有从温和到强不定的，似水果的甜香。<ref name=RHS_encyclopedia>{{cite book |title=Rosen - die große Enzyklopädie |publisher= Dorling Kindersley |author= Charles and Brigid Quest-Ritson |page=263 |ISBN=978-3-8310-1734-8 |language=de |trans_title=RHS Encyclopedia of Roses |date=2010}}</ref> “和平”不断地开花，到第一次[[霜冻|霜冻]]为止。

这一富有生命力的灌木，最终生长到120-200厘米高，90-125厘米宽。其在冬季至多能忍受-23°C的环境（[[USDA耐寒等级|USDA耐寒等级]]6），也可以接受全日照到半阴条件，且具有疾病抗性。其大而光滑的革质叶片相当暗。“A.梅兰德夫人”是个多面手，可以被用作[[切花|切花]]，或是[[园艺月季|园艺月季]]。可以单独栽种，也可以多棵一同栽种。其可以被修剪到同标准月季一样高（90cm），也可以进行盆栽。<ref>{{cite book |title=Alles über Rosen |publisher= Eugen Ulmer KG |author= Klaus-Jürgen Strobel |page=208 |ISBN= 3-8001-4471-9 |language= de |trans_title= Everything about Roses|date=2006}}</ref>

== 奖项 ==
“梅兰德夫人”曾获得过为数众多的奖项。其获得的第一个奖项是1942年[[莱茵|莱茵]]获得的“法兰西最美之花”金牌，之后是[[波特兰|波特兰]]（1944），[[海牙|海牙]]（1965），还有[[皇家月季协会|皇家月季协会]]（1947）。1944年，其被录入了[[全美月季锦集|全美月季锦集]].<ref>{{cite web |url=http://www.helpmefind.com/rose/pl.php?n=2203&tab=18 |title='Peace' rose Awards |publisher=HelpMeFind |date= |accessdate=2014-09-03}}</ref>

这是在“人民所爱之月季集”中第一个拿到月季所能获得最高奖项的人工培育品种，而且还被收录进了[[月季名伶堂|月季名伶堂]]。<ref>{{cite web |url=http://www.worldrose.org/awards/hof/hof.asp |title=Rose Hall of Fame |publisher=World Federation of Rose Societies |date=2012-09-25 |accessdate=2014-09-03 |archive-url=https://www.webcitation.org/6AI54x2Vv?url=http://www.worldrose.org/awards/hof/hof.asp |archive-date=2012-08-29 |dead-url=yes }}</ref>

== 变种和其他杂交种 ==
作为一个常见的栽培种，几个“A.梅兰德夫人”的变种也很有名。例如“藤本和平”（由W. Kordes' Söhne于1951年培育）,相比和平它具有更大的花，同时质量相对更差。“粉和平”，其颜色相对和平更饱满，是绯红色和杏黄色的混色。“古龙”（火和平，由麦克格雷迪在1966年培育），其具有红色的花瓣，然而与此同时，花瓣背部为黄色，形成了有趣的对比。<ref name=RHS_encyclopedia />

其也被用作亲本进行杂交，以便产生新的栽培种，并在著名月季的开发中扮演了重要的角色，例如[[游园会(月季)|游园会(月季)]]（由Swim在1959年研发），和[[明星(月季)|明星(月季)]](由Tantau在1960年研发)，和[[东方快车(月季)|东方快车(月季)]]（由Lim和Twomey在1991年研发）。
