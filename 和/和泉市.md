'''和泉市'''（{{jpn|j=和泉市|hg=いずみし|rm=Izumi shi|f=y}}）是位于[[大阪府|大阪府]]南部（{{tsl|ja|泉北}}地區）的行政區劃。轄區北部為平原，中部為丘陵，南部為{{Tsl|ja|和泉山脈}}；主要市區位於北部的平原區域。

== 歷史 ==
現在的和泉市轄區在令制國時代屬於[[和泉國|和泉國]][[和泉郡|和泉郡]]，位於西北部的主要市區在過去則為和泉國的[[國府_(日本歷史)|國府]]<ref name="History">{{Cite web |url=http://www.city.osaka-izumi.lg.jp/siseizyouho/profile/1315902857322.html| title=市の沿革| accessdate=2018-01-09| language=ja| publisher=和泉市政府}}</ref>，這部分區域在[[江戶時代|江戶時代]]大部分區域為[[一橋德川家|一橋德川家]]的領地，另有部分區域為{{tsl|ja|淀藩}}、[[小泉藩|小泉藩]]、{{tsl|ja|岸和田藩}}、{{tsl|ja|伯太藩}}的領地，其中伯太藩雖然主要轄區為北方的[[大鳥郡|大鳥郡]]，但由於藩廳設於此地的伯太村，因此被命名為「伯太藩」，此地也成為伯太藩的政治中心。而東南部區域則主要屬於{{Tsl|ja|関宿藩|關宿藩}}及{{tsl|ja|吉見藩}}的領地。

[[明治維新|明治維新]]後，這些區域一度在1871年全數被整併至[[堺縣|堺縣]]，但在1881年隨著堺縣被併入[[大阪府|大阪府]]而成為大阪府的轄區。1889年實施[[町村制|町村制]]，現在的和泉市範圍在當時分屬：{{tsl|ja|伯太村}}、{{tsl|ja|国府村 (大阪府)|國府村 (大阪府)|國府村}}、{{tsl|ja|郷荘村|鄉莊村}}、{{tsl|ja|北池田村}}、{{tsl|ja|南池田村}}、{{tsl|ja|東横山村 (大阪府)|東橫山村 (大阪府)|東橫山村}}、{{tsl|ja|西横山村 (大阪府)|西橫山村 (大阪府)|西橫山村}}、{{tsl|ja|南横山村|南橫山村}}、{{tsl|ja|北松尾村}}、{{tsl|ja|南松尾}}、{{tsl|ja|信太村 (大阪府)|信太村 (大阪府)|信太村}}、{{tsl|ja|八坂町|八坂町|南王子村}}共12個行政區劃；其中位於現在主要市區的伯太村、國府村、鄉莊村在1933年合併為{{Tsl|ja|和泉町}}，1956年和泉町與位於現在轄區東南部的行政區劃合併成立為'''和泉市'''<ref name="History"/>，1960年再併入位於北部市區周邊的信太村及八坂町（由南王子村改制而成），形成現在和泉市的轄區範圍。

== 交通 ==
[[西日本旅客鐵道|西日本旅客鐵道]][[阪和線|阪和線]]由轄區西北側主要市區旁通過，{{tsl|ja|和泉府中駅|和泉府中車站}}為市區最主要的車站；此外配合{{tsl|ja|泉北ニュータウン|泉北新市鎮}}所建的[[泉北高速鐵道線|泉北高速鐵道線]]也由[[堺市|堺市]]通往轄區中北部的新市鎮區域。但由於兩條鐵路路線並未通過轄內大部分地區，因此轄內交通主要仰賴{{tsl|ja|南海バス|南海巴士}}。

== 觀光資源 ==
[[File:Ikegami-sone_ruins04s3200.jpg|thumb]]

轄內的池上町地區發掘出{{tsl|ja|池上・曽根遺跡|池上、曾根遺跡}}，為[[彌生時代|彌生時代]]的{{tsl|ja|環濠集落|環濠聚落}}，挖掘出的面積達60[[公頃|公頃]]，因此在此設有以此為主題的池上曾根遺跡史跡公園、{{tsl|ja|大阪府立弥生文化博物館|大阪府立彌生文化博物館}}。

== 教育 ==
轄內唯一的高等教育學校為[[桃山學院大學|桃山學院大學]]，成立於1959年，為[[日本圣公会|日本圣公会]]所成立的私立學校。
