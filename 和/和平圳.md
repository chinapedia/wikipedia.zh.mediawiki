'''和平圳'''為一座位於[[臺灣|臺灣]][[臺東縣|臺東縣]][[鹿野鄉|鹿野鄉]]的小型[[灌溉|灌溉]][[水圳|水圳]]系統，水圳最初於[[日治時期|日治時期]]由當地平地[[原住民|原住民]]自行開鑿建成，其舊名為'''擺仔擺圳'''，目前由[[臺東縣|臺東]][[農田水利會|農田水利會]][[鹿野鄉|鹿野]][[工作站|工作站]]進行管理與維護。<ref>{{cite web |url=http://www.ttia.gov.tw/page4_luye.php |title=台東農田水利會 鹿野工作站 |accessdate=2015-01-23 |deadurl=yes |archiveurl=https://web.archive.org/web/20160121174149/http://www.ttia.gov.tw/page4_luye.php |archivedate=2016-01-21 }}</ref>

== 沿革 ==

和平圳於西元1911年（民國前1年，明治44年）[[台灣日治時期|台灣日治時期]]由當地原住民建成，當時水圳名稱為擺仔擺圳。<ref>{{cite web |url=http://www.ttia.gov.tw/page6.php |title=台東農田水利會 主要圳系建造年代表 |accessdate=2015-01-23 |deadurl=yes |archiveurl=https://web.archive.org/web/20141208020113/http://www.ttia.gov.tw/page6.php |archivedate=2014-12-08 }}</ref><ref name="建國百年宣導 線上成果展 民國40年擺仔擺圳(今和平圳)遭颱風侵襲搶修情形">{{cite web |url=http://doie.coa.gov.tw/2011/taiwan100/review-detail.asp?t=7&id=404&uc=ttia |title=建國百年宣導 線上成果展 民國40年擺仔擺圳(今和平圳)遭颱風侵襲搶修情形 | accessdate=2011}}</ref>

擺仔擺圳自日治時期建成以來時常遭到[[颱風|颱風]]侵襲後而圳路崩壞，導致當地民眾整年都必須重新修築圳道，甚至有棄圳另闢新圳的念頭產生。<ref name="建國百年宣導 線上成果展 民國40年擺仔擺圳(今和平圳)遭颱風侵襲搶修情形"/>

每逢颱風侵襲圳路損壞後重新修築的狀況直到[[臺灣戰後時期|中華民國接管臺灣後]]，1951年(民國40年)，當時[[台灣|台灣]][[省政府|省政府]]令台東農田水利會進行修整，於1951年(民國40年)2月1日將該工程技術部分發包開工，同時號召位於擺仔擺圳灌溉區內的民眾協助挖掘土方。然因施工期間已完成之圳路又遭洪水流失。導致整體改善工程遲至1956年(民國45年)12月底才完工通水。<ref name="建國百年宣導 線上成果展 民國40年擺仔擺圳(今和平圳)遭颱風侵襲搶修情形"/>

改善工程完工後，擺仔擺圳不再是因當時[[臺東縣|臺東]]地區最難治之埤圳而出名，而該圳名也由於擺仔擺之名諧音為'''壞了壞'''，故改名為和平圳。<ref name="建國百年宣導 線上成果展 民國40年擺仔擺圳(今和平圳)遭颱風侵襲搶修情形"/>

== 設施 ==

=== 進水口 ===

和平圳進水口設於[[鹿鳴溪|鹿鳴溪]]下游接鹿鳴溪左岸[[堤防|堤防]]處，總引入流量0.800[[C.M.S.|C.M.S.]]水量。因鹿鳴溪之溪水流量不穩定，因此台東農田水利會另開一條長達2.5公里的導水路將鄰近[[鹿野溪|鹿野溪]]之溪水穿過鹿野溪與鹿鳴溪之間[[沙洲|沙洲]]導進和平圳進水口前。<ref>{{cite web |url=http://student.kmes.tc.edu.tw/taiculture/water/14/C/04/13/C-14-04-13-menu.htm |title=灌排系統 台東農田水利會灌排系統 和平圳 |accessdate=2015-01-23 |deadurl=yes |archiveurl=https://web.archive.org/web/20150123171042/http://student.kmes.tc.edu.tw/taiculture/water/14/C/04/13/C-14-04-13-menu.htm |archivedate=2015-01-23 }}</ref>

=== 導水設施 ===

和平圳主幹線長約4公里，設有兩條排水線與三條支線，第一支線長1.275公里，第二支線長0.78公里，第三支線長1.268公里。

=== 沉沙池 ===

和平圳沉沙池設於進水口北方，並設有即時監控[[攝影機|攝影機]]於沉沙池上方。
