'''和晶'''（{{bd|1970年||catIdx=He,和}}），[[回族|回族]]，[[中國|中國]][[新疆|新疆]][[巴楚|巴楚]]人，電視節目[[主持人|主持人]]。

== 生平 ==
1988年，和晶考入[[上海戲劇學院|上海戲劇學院]]導演系，畢業後進入[[上海電視臺|上海電視臺]]工作，1993年起主持《[[智力大沖浪|智力大沖浪]]》而成名。<ref name="mab " />1997年考入[[上海大學|上海大學]]和[[北京電影學院|北京電影學院]]電影研究中心聯合辦學的中國電影史專業[[碩士|碩士]]研究生，次年畢業後轉任《[[有話大家說|有話大家說]]》欄目主持。

2001年，和晶辭職北上，進入[[中央電視臺|中央電視臺]]主持《選擇》欄目，2002年9月後主持《[[實話實說|實話實說]]》欄目。<ref name="mab " >{{cite web |url=http://money.591hx.com/article/2010-12-07/0000135913s_3.shtml |title=从地方台跳槽央视后大红大紫的女主播 |accessdate= 2010年|work= |publisher= 华讯|language =zh-cn}}</ref>

== 家庭 ==
和晶出生於[[新疆|新疆]][[巴楚縣|巴楚縣]]五十三團。父親和丕禪曾於[[石河子大學|石河子大學]]、[[浙江大學|浙江大學]]任教，經濟系教授。和晶是[[漢族|漢族]]和[[回族|回族]]混血。1999年，和晶結婚，2005年生下女兒。
