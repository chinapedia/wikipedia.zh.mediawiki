'''和田岬線'''（{{jpn|j=和田岬線|hg=わだみさきせん|rm=Wadamisaki sen|f=y}}）是一條連結[[兵庫縣|兵庫縣]][[神戶市|神戶市]][[兵庫區|兵庫區]]內屬於[[西日本旅客鐵道|西日本旅客鐵道]]（JR西日本）[[山陽本線|山陽本線]]的支線，是{{link-ja|兵庫站|兵庫駅}}－[[和田岬站|和田岬站]]間的通稱。此線連結本線與{{link-ja|和田岬|和田岬}}地區。

== 概要 ==
和田岬線，是山陽本線由兵庫站分支出來全長2.7km的支線。原來作為鐵道建設時資材輸送所用而於1888年敷設，其後長時間作貨物輸送之用，[[兵庫臨港線|兵庫臨港線]]亦有分支。現在已不作貨物輸送用途，只於朝夕的繁忙時間行駛以通勤客為對象的列車，2001年7月1日電氣化後由[[電聯車|電聯車]]行駛。

=== 路線資料 ===
* 管轄（事業類別）：西日本旅客鐵道（[[鐵路公司#第一種鐵道事業者|第一種鐵道事業者]]）
* 路線距離（{{link-ja|營業距離|営業キロ}}）：2.7公里
* [[軌距|軌距]]：1,067毫米
* 站數：2個（包括起終點站）
* [[複線鐵路|複線]]路段：沒有（全線[[單線鐵路|單線]]）
* [[電氣化鐵路|電氣化]]路段：全線電氣化（[[直流電|直流電]]1,500 V）
* [[閉塞_(鐵路)|閉塞方式]]：特殊自動閉塞式（軌道回路檢知式）
* 保安裝置：[[自動列車停止裝置|ATS-SW]]
* 最高速度：85公里/小時
* {{link-ja|運轉指令所|運転指令所}}：{{link-ja|大阪綜合指令所|大阪総合指令所}}

全線由{{link-ja|西日本旅客鐵道近畿統括本部|西日本旅客鉄道近畿統括本部|近畿統括本部}}管轄。

== 運行形態 ==
列車為周邊工場員工的通勤需要而特化，只在早上與晚上時間運行，平日為17往復，星期六為12往復，早上10點至下午4點之間完全沒有列車班次。週日及例假日只有朝夕各1往復共計2往復運行，不設[[臨時列車|臨時列車]]。不使用[[一人控制|一人控制]]。

== 使用車輛 ==
[[File:JRW-WadamisakiLine-PCtrain.jpg|thumb]]牽引OHAFU64型及OHA64型車廂。（攝於兵庫站）]]

2001年7月1日電氣化後，基本使用由[[網干總合車輛所|網干總合車輛所]]明石支所所屬的[[國鐵103系電車|103系]]電車6卡編成，因為只有一輛所以在檢査維修時會使用同支所配置的[[JR西日本207系電車|207系]]電車代走<ref>[http://railf.jp/news/2011/08/07/060000.html 和田岬線に207系が入線] - 『[[鐵道ファン_(雑誌)|鐵道ファン]]』[[交友社|交友社]] railf.jp鐵道ニュース 2011年8月7日</ref>。[[國鐵201系電車|201系]]電車撤離JR京都・神戶線前亦有使用過同系列把[[動車組拖車|動車組拖車]]拿掉1卡而作6卡編成。[[國鐵205系電車|205系]]・[[JR西日本321系電車|321系]]電車則未在和田岬線使用。

103系・201系的塗色為天藍色（[[青22號|青22號]]）。

== 利用狀況 ==
和田岬線以[[三菱重工業神戶造船所|三菱重工業神戶造船所]]等公司的員工通勤為主要目的。朝早是往和田岬方向、黃昏與晚上是兵庫方向非常擁擠，但逆方向的乘客則非常少，列車與[[回送|回送]]沒分別。

因為[[神戶市營地下鐵海岸線|神戶市營地下鐵海岸線]]的開業，使和田岬線的乘客數減少了三成，1日平均乘車人員約1万人次<ref name="kobe_20110215">[http://www.kobe-np.co.jp/news/shakai/0003805740.shtml 利用客3割減 和田岬線の廢止檢討 JR西日本] - [[神戶新聞|神戶新聞]] 2011年2月15日</ref>。

== 歷史 ==
和田岬線是因為[[山陽鐵道|山陽鐵道]]兵庫站 - [[姫路站|姫路站]]間的建設所需，把兵庫港輸入的資材輸送而於1888年敷設。資材輸送完了後1890年[[貨物線|貨物線]]開業，直至1984年為止設有往[[神戶市場站|神戶市場站]]等貨物支線（[[兵庫臨港線|兵庫臨港線]]）的分支。

1980年以前和田岬站前方有延長線路，是[[三菱重工業|三菱重工業]]貨物[[専用鐵道|専用線]]。車站前道路有[[神戶市電|神戶市電]]通過，市電有専用軌道用陸橋穿越和田岬線。這陸橋在市電廢止同時撤去。

另外，和田岬方向曾有與山陽本線明石方向直接接續線路的敷設計劃。這計劃最終沒有實現，線路敷設予定地現在成為細長的公園（梅ヶ香公園），在地圖上亦可以找到。

神戶市中心地[[三宮|三宮]]與和田岬直結的神戶市營地下鐵海岸線於2001年7月7日開業時，曾有和田岬線會廢止的傳聞<ref>[[川島令三|川島令三]]『全國鐵道事情大研究〈神戶篇〉』など</ref>，結果因為鐵路電氣化令成本大幅下降而得以存續。電氣化工程只花了三個月完成，因為路線短而不需要設置[[變電所|變電所]]，而且午間沒有列車運行而可以進行工程。

=== 年表 ===
* 1890年（[[明治|明治]]23年）7月8日：山陽鐵道貨物支線的兵庫站 - 和田崎町站間開業。新設和田崎町站。
* 1893年（明治26年）12月3日：途中新設[[新川站_(兵庫縣)|新川荷扱所]]。
* 1895年（明治28年）：和田崎町站改名和田岬站。
* 1906年（明治39年）12月1日：全線國有化。
* 1911年（明治44年）11月1日：旅客營業開始。新川荷扱所由和田岬線車站改變為其他路線車路。
* 1912年（明治45年）4月16日：兵庫站 - 和田岬站間的[[鐘紡前站|鐘紡前站]]開業。
* 1930年（[[昭和|昭和]]5年）4月1日：營業距離由[[英里|英里]]表記改變為公里表記 （1.7M→2.7km）。
* 1945年（昭和20年）2月：和田岬站轉移。
* 1948年（昭和23年）3月以降：鐘紡前站休止。
* 1962年（昭和37年）3月1日：鐘紡前站廢止。
* 1980年（昭和55年）10月1日：貨物營業廢止。
* 1987年（昭和62年）4月1日：[[國鐵分割民營化|國鐵分割民營化]]、由西日本旅客鐵道繼承。
* 1990年（平成2年）10月1日：客車更換為柴油車<ref name="jrr_1991">『JR氣動車客車編成表』91年版 ジェー・アール・アール 1991年 ISBN 4-88283-112-0</ref>。
* 2001年（平成13年）7月1日：全線電氣化、由電聯車運行。列車班數增加。
* 2010年（平成22年）12月1日：組織改正、由[[西日本旅客鐵道神戶支社|神戶支社]]管轄改變為近畿統括本部管轄<ref>[https://web.archive.org/web/20101204082729/http://www.westjr.co.jp/news/newslist/article/1175068_799.html 組織改正などについて]（[[インターネット・アーカイブ|インターネット・アーカイブ]]）- 西日本旅客鐵道プレスリリース 2010年11月16日</ref>。
