'''和谐2C型电力机车'''（HXD2C），是用于[[中华人民共和国铁路运输|中国铁路]]的[[变频器|交流电传动]]干线货运[[电力机车|电力机车]]车型之一。

==发展历史==
=== 背景 ===
自2003年[[刘志军|刘志军]]出任[[中华人民共和国铁道部|中华人民共和国铁道部]]部长后，提出并实施铁路“跨越式发展”，以尽快缩小在铁路机车车辆装备上与国际先进水平的差距。2003年11月，中国铁道部与[[中国南车集团|中国南车]]、[[中国北车集团|北车集团]]及其重点企业共同制定了《加快铁路机车车辆装备现代化实施纲要》，并选择了6家机车制造企业作为引进先进技术和自主创新依托的主体。2004年1月，[[中国国务院|中国国务院]][[国务院常务会议|常务会议]]通过了《中长期铁路网规划》；2004年4月，国务院常务会议研究通过的铁路机车车辆装备现代化实施方案明确指出，“加快我国铁路运输装备现代化，要按照引进先进技术、联合设计生产、打造中国品牌的总体要求，力争在较短时间内，使我国机车车辆生产能力达到世界先进水平”<ref>{{Cite web |url=http://www.china.com.cn/policy/txt/2008-12/02/content_16883929.htm |title=铁道部发展计划司解读我国中长期铁路网调整规划 |accessdate=2012-05-15 |publisher=中国网 |date=2008-12-02 }}</ref><ref>{{Cite web |url=http://news.xinhuanet.com/zhengfu/2004-02/23/content_1326091.htm |title=铁道部打造中国铁路机车车辆装备品牌 |accessdate=2012-05-15 |publisher=新华网 |date=2004-02-23 }}</ref>。2004年，[[中国北车集团|中国北车集团]][[大同电力机车有限责任公司|大同电力机车有限责任公司]]与[[法国|法国]][[阿尔斯通公司|阿尔斯通公司]]，引进了[[HXD2型电力机车|HXD2型]]双节八轴10000千瓦交流传动电力机车。2007年，大同电力机车再次与阿尔斯通公司合作，引进了[[HXD2B型电力机车|HXD2B型]]六轴9600千瓦电力机车。

===研制===
在HXD2、HXD2B型电力机车基础上，大同电力机车有限责任公司于2009年开始着手研制六轴7200千瓦电力机车，新机车被定型为HXD2C型，其中“HX”是“和谐”的汉语拼音首字母缩写、“D”代表电力机车、“2”代表大同电力机车公司的生产厂商代号，通称为“和谐”2C型电力机车。2009年底，首台HXD2C型机车完成组装，并开始进行厂内调试。2010年5月23日，HXD2B、HXD2C型电力机车下线剪彩仪式在大同电力机车公司举行<ref>{{Cite web |url=http://ip.people.com.cn/GB/11673079.html |title=新型自主知识产权电力货运机车下线 |accessdate=2012-09-03 |publisher=新华网 |date=2010-05-24 }}</ref>。2010年6月，铁道部向大同电力机车公司正式签订了220台HXD2C型电力机车的采购合同，总值58.5亿元人民币<ref>{{Cite web |url=http://ccnews.people.com.cn/GB/12161000.html |title=南车北车“开进”阿根廷 “计划外增长”可期 |accessdate=2012-09-15 |publisher=证券日报 |date=2010-07-16 }}</ref><ref>{{Cite web |url=http://business.sohu.com/20100712/n273450302.shtml |title= 中国北车和谐2C型电力机车中标铁道部采购项目 |accessdate=2012-09-15 |publisher=经济参考报 |date=2010-07-12 }}</ref>。

HXD2C型电力机车是在HXD2系列交流传动机车设计制造技术平台的基础上，为满足中国铁路重载货运需要而研发的大功率交流传动干线货运用六轴电力机车，牵引系统采用交—直—交流电传动、水冷[[IGBT|IGBT]]牵引逆变器、变频异步牵引电机、分布式网络控制系统，单轴功率1200千瓦，额定总功率为7200千瓦，能够在平原地区单机牵引5000～6000吨货物列车，最高运行速度为120公里/小时。机车总体采用“标准化、系列化、模块化、信息化”的设计原则，增加了设备的通用性、互换性。HXD2C型电力机车可根据用户需要，选择搭载不同供应商的牵引电传动系统，包括基于[[HXD3型电力机车|HXD3型电力机车]]的[[东芝|东芝]]系统（A方案）、基于[[HXD1C型电力机车|HXD1C型电力机车]]的[[株洲南车时代电气股份有限公司|株洲南车时代]]系统（B方案）、基于[[HXD3B型电力机车|HXD3B型电力机车]]的[[庞巴迪运输|庞巴迪]]系统（C方案）、中国北车集团自主研发的“北车心”系统（D方案）、以及基于HXD2系列电力机车的阿尔斯通系统（E方案）。目前采用东芝系统、庞巴迪系统及“北车心”系统的HXD2C型机车已经实现投产。
[[File:HXD2C0218.JPG|300px]]
首批167台HXD2C型电力机车（0001～0167）采用东芝牵引系统，首台机车于2010年5月正式下线后，被送往[[中国铁道科学研究院|中国铁道科学研究院]][[中国国家铁道试验中心（东郊分院）|北京环形铁道试验基地]]开始进行型式试验。2009年8月及12月，HXD2C型0001、0002号机车赴[[郑州铁路局|郑州铁路局]]新乡机务段，在[[京广铁路|京广铁路]][[新乡站|新乡]]至[[安阳站|安阳]]间和[[新焦铁路|新焦铁路]]新乡至[[晋城北站|晋城北]]间进行23、25吨轴重状态下动力学试验和重载牵引试验，机车在京广线上的最高试验速度达到130公里/小时。

第二批50台HXD2C型机车（0168～0217）采用庞巴迪牵引系统。2010年10月17日，大同电力机车公司与[[江苏常牵庞巴迪牵引系统有限公司|江苏常牵庞巴迪牵引系统有限公司]]（BCP）签订了7200千瓦电力机车牵引及控制系统的采购合同，为HXD2C型机车提供主变流柜、辅助变流柜、辅助滤波柜、牵引电动机和“Mitrac”微机控制系统等设备。首批设备于2011年3月开始交付同车公司<ref>{{Cite web |url=http://www.bcp-propulsion.com/news_display.asp?id=62&classname=1 |title=大同7200千瓦电力机车项目 |accessdate=2012-09-15 |publisher=江苏常牵庞巴迪牵引系统有限公司 |date= |deadurl=yes |archiveurl=https://web.archive.org/web/20131102065112/http://www.bcp-propulsion.com/news_display.asp?id=62&classname=1 |archivedate=2013-11-02 }}</ref>。

首笔订单的最后3台HXD2C型机车（0218～0220）采用中国北车集团自主研发的牵引系统，俗称“北车心”。2010年10月，中国北车集团开始立项研制7200千瓦电力机车电传动系统，以[[DJG2型电力机车|DJG2型]]交流传动高速客运电力机车牵引系统为研制技术平台，研制单位包括大同电力机车公司、[[永济新时速电机电器|永济电机公司]]、[[大连机车研究所|大连机车研究所]]和[[大连电力牵引研发中心|大连电力牵引研发中心]]，大连所和大连电牵研发中心主要负责网络控制系统、牵引变流控制单元、辅助变流器控制单元、辅助功率模块的研制；永济电机公司负责牵引电机、变流器主电路、牵引功率模块、柜体等的研制。2011年3月，“北车心”机车所使用的NECT牵引及网络控制系统通过技术评审<ref>{{Cite web |url=http://www.chinacnr.com/Page/272/SourceId/656/InfoID/3914/language/zh-CN/default.aspx |title=中国北车大连所公司HXD2C电力机车NECT牵引及网络控制系统通过技术评审 |accessdate=2012-09-15 |publisher=中国北车大连所公司 |date=2011-03-23 |deadurl=yes |archiveurl=https://web.archive.org/web/20131101191739/http://www.chinacnr.com/Page/272/SourceId/656/InfoID/3914/language/zh-CN/default.aspx |archivedate=2013-11-01 }}</ref>。2011年5月，首台“北车心”机车的配套产品，包括电传动系统、牵引辅助变流器、IGBT功率模块、滤波柜等，在永济电机公司完成试制组装及例行试验<ref>{{Cite web |url=http://www.yonge.chinacnr.com/djrjy/shownews.asp?id=2073 |title=HXD2C项目首台车产品完成试制 |accessdate=2012-09-15 |publisher= |date=2011-05-19 |deadurl=yes |archiveurl=https://web.archive.org/web/20131101151224/http://www.yonge.chinacnr.com/djrjy/shownews.asp?id=2073 |archivedate=2013-11-01 }}</ref>。

===运用===
====中國國鐵====
2010年12月，HXD2C型电力机车投入批量生产，并开始配属[[太原鐵路局|太原鐵路局]]（太原局集團）太原机务段；2010年12月20日，HXD2C型机车在[[石太铁路|石太铁路]]正式投入运营<ref>{{Cite web |url=http://finance.sina.com.cn/chanjing/gsnews/20101224/16043559479.shtml |title= 中国北车和谐2C型机车发力百年石太线 |accessdate=2012-09-15 |publisher=新浪财经 |date=2010-12-24 }}</ref>。2011年5月，根据铁道部安排，将太原机务段所有HXD2C型电力机车陆续调拨给[[郑州铁路局|郑州铁路局]]新乡机务段，担当[[侯月铁路|侯月铁路]]货运任务。此外，亦有部分HXD2C配屬太原局集團轄下非全資附屬機構-大同地方鐵路公司，而該等機車並無漆上國鐵路徽。

==技术特点==
===总体结构===
HXD2C型电力机车是六轴大功率干线货运用电力机车，车体结构与HXD2B型电力机车完全相同，采用框架式整体承载全钢箱型壳体焊接结构，车体侧墙采用波纹钢板，并设有宽度为787毫米的维修门。车体两端各设有一个司机室，司机可在任何一端司机室对机车进行控制。司机室采用框架式焊接钢结构，前窗采用一整体电热玻璃，为司乘人员提供了良好的视角，并满足机车在寒冷季节使用的需要。车内设备采用模块化结构、两侧屏柜化布置，并设宽600毫米的中间贯通走廊；车上并设有[[电水壶|电水壶]]、[[冰箱|冰箱]]、[[微波炉|微波炉]]、[[卫生间|卫生间]]等生活设施。每台机车车顶设有两台DSA-200型单臂式[[受电弓|受电弓]]，其他车顶外置设备包括[[断路器|真空断路器]]、接地开关、避雷器、高压连接器等高压设备。

机车采用独立通风方式，从侧墙过滤窗和车顶百叶窗吸入冷风，向发热部件冷却后从车底排出，并维持机械间呈[[正壓|微正压]]，改善机车防尘效果及防寒性能。[[机车轴式|机车轴式]]为[[Co-Co|Co-Co]]，持续功率为9600千瓦；机车标准轴重为23吨，并可以通过增加压铁提高到25吨。[[空气制动|空气制动]]系统采用[[法维莱公司|法维莱公司]]（Faiveley）的“Eurotrol”微机控制电空制动机，具有空电联合制动、阶段缓解、断钩保护、备用制动等功能。

===电气系统===
HXD2C型电力机车是交—直—交流电传动的单相工频交流电力机车，机车主电路由主变压器、牵引变流器、牵引电动机三大部分构成。机车设计采用“标准化、系列化、模块化、信息化”的开发原则，可以实现搭载不同供应商的牵引电传动系统。不论采用哪种牵引系统，所有HXD2C系列电力机车均采用单轴控制技术，每台牵引电机对应一套独立的四象限整流器和逆变器。

==== 东芝牵引系统 ====
HXD2C-A方案机车采用与HXD3型电力机车相同的东芝牵引传动系统，每台机车设有6套完全独立而又相同的牵引变流单元，分别安装在2个主变流器柜内，分别为2台转向架上的6台牵引电动机供电。每台主变流器柜内含有三组牵引变流器单元和一组辅助变流器；每组变流器单元由一个四象限脉冲整流器、一个中间直流环节、一个两点式电压型PWM逆变器组成；功率控制模块采用水冷IGBT变流模块（4500V/900A），中间直流电压为2800伏。

==== 庞巴迪牵引系统 ====
HXD2C-C方案机车采用基于HXD3B型电力机车的庞巴迪牵引传动系统，每台机车设有3个主变流器柜，每台主变流器柜由二个四象限脉冲整流器、一个中间直流环节、二个两点式电压型PWM逆变器和一个辅助变流器组成；功率控制模块采用水冷IGBT变流模块，中间直流电压为2800伏。

==== 北车牵引系统 ====
HXD2C-D方案机车采用中国北车集团大连电力牵引研发中心[[CNRERD|CNRERD]]自主研发的牵引传动系统，採用[[中車永濟電機|中車永濟電機]]生產的改良型號三相誘導電機。

==== 辅助电路 ====
HXD2C型机车将辅助变流装置集成在主变流柜中，每台机车设有两个IGBT辅助变流器，由主变压器二次侧辅助绕组供电。辅助变流器通过使用PWM整流器单元把从主变压器供电的单相交流电转换为恒定电压的直流电，再供通过逆变器单元转换为三相交流电。正常情况下两个变流器基本上以50%的额定容量工作，其中一个为恒频恒压变流器（CVCF），另一个为变频变压变流器（VVVF）。当某一套辅助变流器发生故障时，另一套辅助变流器可以承担机车全部的辅助电动机负载，此时该辅助变流器按照CVCF方式工作，确保机车辅助供电系统的可靠性。

=== 转向架 ===
机车走行部为两台三轴[[转向架|转向架]]，其结构与HXD2B型电力机车的转向架完全相同。转向架构架采用钢板焊成箱形结构的“目”字型构架，轮对轴箱采用拉杆定位。一系悬挂采用螺旋弹簧加双轴箱拉杆配垂向油压减震器；二系悬挂为橡胶堆。牵引力或制动力通过单推挽水平短牵引拉杆牵引装置传递。牵引电动机采用滚动轴承抱轴式半悬挂、单边刚性直齿传动。基础制动装置为单侧双闸瓦踏面制动单元，并带有闸瓦间隙调节器；其中三组制动单元具有停放制动功能。转向架并设有轮缘润滑装置。
