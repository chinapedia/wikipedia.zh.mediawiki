[[File:Pacific_moon.png|thumb]]'''Pacific Moon'''（'''{{lang-zh|和平之月}}'''），是一个专门制作New Age和World Music的亚洲著名音乐品牌，隶属于Chapter One公司，旗下精英荟萃，有[[贾鹏芳|贾鹏芳]]、[[城之内美莎|城之内美莎]]、[[京田诚一|京田诚一]]、[[邵容|邵容]]、[[姜小青|姜小青]]、[[菅井爱里|菅井爱里]]、[[恩田直幸|恩田直幸]]、[[小宫瑞代|小宫瑞代]]、[[吉田洁|吉田洁]]、[[渡边雅二|渡边雅二]]、[[吉冈一政|吉冈一政]]和[[里安娜|里安娜]]等人，融合民族和世界、东方乐器和西方配乐的典型效果创造属于Pacific Moon风格的作品，公司大部分专辑都用一个或两个[[汉字|汉字]]来作为专辑的名称，显示了专辑本身的特点和音乐人的风格。

== 特征 ==

和平之月系列是以东方乐器如[[琵琶|琵琶]]，[[古筝|古筝]]，[[二胡|二胡]]，[[扬琴|扬琴]]等中国乐器和[[日本筝|日本筝]]、[[尺八|尺八]]（中国唐朝时传入日本）等日本乐器为主奏，或加上[[钢琴|钢琴]]、[[长笛|长笛]]等西洋乐器，辅以电子合成乐的编曲，有的则以人声吟唱或以东洋的[[太鼓|太鼓]]及南洋的打击乐器，或者是宏大的交响乐团来表现旋律，创造出清新自然的New Age风格。
