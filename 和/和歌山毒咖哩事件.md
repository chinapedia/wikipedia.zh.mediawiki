'''和歌山毒咖哩事件'''是指1998年7月25日，[[日本|日本]][[和歌山縣|和歌山縣]][[和歌山市|和歌山市]]園部地區的一場祭典中所提供的[[咖哩|咖哩]]，遭加入[[砒霜|砒霜]]的事件。該事件共計造成67人送醫急救，其中4人不治身亡。嫌犯[[林真須美|林真須美]]於2009年4月21日遭[[最高裁判所|日本最高法院]]判處[[死刑|死刑]]定讞。

== 案發經過 ==
1998年7月25日晚間，食用和歌山縣和歌山市園部地區祭典所提供的咖哩的67名民眾因腹痛及嘔吐被送往醫院急救，其中4名男性民眾（分別64歲、54歲、16歲、10歲）於急救後宣告不治。

當地衛生所最初認定為一起單純的[[食物中毒|食物中毒]]意外。但和歌山縣警方卻在嘔吐物中化驗出[[氰化氫|氰化氫]]成分，因而判定為氰化氫中毒，但與受害者症狀又有所不符。最後經[[警察廳|警察廳]][[科學警察研究所|科學警察研究所]]化驗後，確認含有劇毒的[[亞砷酸|亞砷酸]]，即[[砒霜|砒霜]]的主要成分。

== 逮捕及審判 ==
1998年10月4日，一名涉嫌殺人未遂及詐領保險金的主婦林真須美遭到逮捕。她被指控在這個毒咖哩殺人事件中為了詐領保險金（林原從事保險推銷員的工作。她利用職務之便及掌握保險公司對被保險人的本人確認不完全的資訊下，在被害者不知情的情況下，將其加入保單中）。12月9日，警方在林真須美家中查獲大量亞砷酸，而林真須美當天也參與烹飪工作，因此再度遭到逮捕。

林真須美對警方及檢方的指控全盤否認，並在一審時全程行使[[緘默權|緘默權]]，但[[和歌山地方裁判所|和歌山地方法院]]一審、[[大阪高等裁判所|大阪高等法院]]二審均判處其死刑。2009年4月21日，[[最高裁判所|日本最高法院]]維持一、二審原判，判處林真須美死刑定讞，成為日本[[二戰|二戰]]後第11名女性死刑犯。

由於被告對指控全盤否認，而法院僅僅根據在場目擊者的證詞、在被告住處搜到的證物及被告曾有毒殺他人未遂的前科而判處其死刑，在日本社會中也引起廣泛議論。

== 後續 ==
*{{来源请求|由於媒體廣泛的以「遭下毒的咖哩」報導此一事件，使得許多民眾對咖哩的安全性產生疑慮，食品公司紛紛以廣告自清，料理節目也在菜單上多次保證絕對安全無虞。
*林真須美的住宅於2000年2月遭[[縱火|縱火]]燒毀，原地已改建為公園。
*在此事件發生後，日本陸續發生多起食物遭下毒案件。}}
