'''和谐号CRH380C型电力动车组'''，是[[中國鐵路總公司|中國鐵路總公司]]为营运新建的[[中國高速鐵路|高速城际铁路及客运专线]]，由[[中国北车集团|中国北车集团]][[长春轨道客车股份有限公司|长春轨道客车股份有限公司]]在[[和谐号CRH3型电力动车组|CRH3C]]、[[和谐号CRH380B型电力动车组|CRH380BL型电力动车组]]基础上自主研发的[[中国铁路高速|CRH]]系列高速[[動車組|動車組]]，也是“中國高速列車自主創新聯合行動計劃”的重点项目之一。值得注意的是[[中国铁路高速综合检测列车|中国铁路高速综合检测列车]]CRH380BJ-0301以CRH380B型电力动车组为基础，但采用了北车CRH380C的头型。中国鐵道部将所有自行发展关键技术、引进国外技术、联合设计生产的[[中国铁路高速|中国铁路高速]]（CRH）車輛均命名為「'''[[和谐号|和谐号]]'''」。

== 概要 ==
{{Seealso|和谐号CRH380B型电力动车组}}
[[京沪高速铁路|京沪高速铁路]]是世界上一次建成线路最长、标准最高的[[高速铁路|高速铁路]]，为之配套的新一代高速列车的开发是在消化吸收基础上一项庞大的再创新工程，为全面支撑中国高速列车技术自主创新的需求，[[中华人民共和国科技部|中华人民共和国科技部]]与[[中华人民共和国铁道部|铁道部]]于2008年2月26日共同签署了《中国高速列车自主创新联合行动计划》，提出研制符合京沪高铁运营需求的高速列车，由[[中国南车集团|中国南车]]、[[中国北车集团|北车集团]]在消化吸收相关技术的基础上，将动车组的时速从250～300公里提高到350公里及以上，为京沪高速铁路提供强而有力的装备保障，建立并完善具有自主知识产权、国际竞争力强的时速350公里及以上中国高速铁路技术体系。<ref>{{Cite web |url=http://xwcb.eastday.com/c/20080314/u1a410708.html |title=京沪高铁虹桥站年内开建 将成上海第一大火车站 |accessdate=2010-12-03 |publisher=新闻晨报 |date=2008-03-14 }}</ref>。其中，中国北车集团唐山轨道客车以CRH3型电力动车组为基础，研制新一代时速350公里级别高速动车组。2009年正式立项启动，当时研制项目名称为'''CRH3-350'''。

2008年9月，CRH3-350型动车组开始前期研发阶段，同时提高了车辆的设计标准，最高时速由时速350公里提高至380公里，项目名称为'''CRH3-380'''；列车在京沪高速铁路上的单程行车时间可由原规划的5小时，进一步缩短至4小时<ref>{{Cite web |url=http://www.railcn.net/news/railway-express/68405.html |title=京沪高铁国产列车 唐山投产　全球最快轮轨车 时速 |accessdate=2010-12-03 |publisher=铁道网 |date=2008-09-03 |deadurl=yes |archiveurl=https://web.archive.org/web/20140227184314/http://www.railcn.net/news/railway-express/68405.html |archivedate=2014-02-27 }}</ref>。更改标准后，新一代动车组的产品设计工作于2009年1月27日正式启动。

2009年3月16日，[[中华人民共和国铁道部|中华人民共和国铁道部]]正式与中国北车集团旗下的[[唐山轨道客车|唐山轨道客车公司]]和[[长春轨道客车|长春轨道客车股份公司]]、以及[[中国铁道科学研究院|中国铁道科学研究院]]签署了一系列采购合同，合作生产100列新一代高速動車組，为16節車廂的大編組座車，合同总金额达392亿元人民币。其中70列是由[[唐山轨道客车|唐山轨道客车]]制造，其餘30列則由[[长春轨道客车|长春轨道客车]]制造<ref name=sina0402>{{cite web | url = http://news.sina.com.cn/c/2009-04-02/083617533481.shtml| title = 西门子将向中方提供百列高速列车 不卖核心技术| date = 2009-04-02| publisher = 新浪網| accessdate = 2010-05-28}}</ref>。

由于此次签订合同的消息中中国官方并没有提到[[西门子|西门子]]的参与，次日铁道部仅在网站上发表“百列国产新一代高速动车组将于2011年驶上京沪高铁”为主题的新闻。至3月20日，德国西门子集团在其网站发布信息称，“唐山轨道客车有限责任公司、长春轨道客车股份有限公司、中国铁道科学院和西门子签署了一份关于提供100列高速列车的合同，西门子获得价值7.5亿欧元份额”，此时[[德国|德国]]《[[商报|商报]]》也援引西门子内部人士谈话，西门子表示“不会、也绝不出让核心技术”<ref name=sina0402 />。这则消息在海内外一度引起广泛关注，许多媒体以“西门子宣布获得中国政府购买100列高速列车的订单”类似标题进行了报道<ref name=sina0402 />。由于西门子网站消息与铁道部官方发布信息，措辞明显有出入，尤其在动车组知识产权归属上存在疑义，引起国内公众疑虑。3月25日，铁道部公开回应称：“近期并没有跟西门子签订任何合同”，形势几乎演变成由铁道部和西门子主演的一场公关危机，引发诸多猜测。3月31日，[[西门子交通集团|西门子交通集团]]媒体发言人回应媒体时表示，此次100列动车组的合约确已签署，列车将主要在中国生产。而对有关技术转让的合约细节等问题，则并未有详细透露<ref name=sina0402 />。实际上，按铁道部的招标规则，在京沪高铁的设备采购中，外资企业不能成为总承包商，但是容许外资企业通过与中国本土企业合作，原则上铁道部并不可直接与外资签订合同。因此，中国北车股份有限公司在3月16日签署协议以后，会将部分无法国产的零部件生产份额转包给了西门子公司，而西门子此次通过提供电气牵引系统等的方式可获得约7.5亿欧元（约70亿元人民币）的订单。而根据西门子的消息，这些部件的生产任务将分别由西门子位于德国纽伦堡、克雷菲尔德等地，以及奥地利和中国上海等工厂承担。

2009年6月，[[中华人民共和国铁道部|中华人民共和国铁道部]]向国内动车组制造企业招标采购共320列时速350公里的高速动车组<ref>{{Cite web |url=http://www.chinaitc.com.cn/chinaitc/news/zbgg/announcement00000442.htm |title=铁道部时速350公里速度级铁路动车组项目招标公告 |accessdate=2010-07-14 |publisher=中技国际招标公司 |date=2009年6月10日 |deadurl=yes |archiveurl=https://web.archive.org/web/20100705092257/http://www.chinaitc.com.cn/chinaitc/news/zbgg/announcement00000442.htm |archivedate=2010年7月5日 }}</ref>，中國北車集团获得了100列动车组的訂單<ref>{{cite web| url = http://finance.people.com.cn/BIG5/10134968.html| title = 中國北車再獲140列新一代動車組大額訂單| date = 2008-09-28| publisher = 人民網}}</ref>，其中80列为新一代CRH3-380型动车组，包含40列16辆编组动车组和40列8辆编组动车组，由长春轨道客车制造<ref>{{cite web| url = http://jlrbszb.chinajilin.com.cn/html/2009-10/06/content_559836.htm| title = 长客签下235亿订单 为铁道部制造新一代动车组| date = 2008-09-28| publisher = 吉林日报| deadurl = yes| archiveurl = https://web.archive.org/web/20110707014542/http://jlrbszb.chinajilin.com.cn/html/2009-10/06/content_559836.htm| archivedate = 2011-07-07}}</ref>。2009年9月28日，铁道部、长春轨道客车股份有限公司、北京铁路局于北京正式签订合同，总值235亿元人民币<ref>{{Cite web |url=http://www.cccar.com.cn/news_content.asp?mainid=22&id=250 |title=新一代时速350公里动车组第二单在北京签订 |accessdate=2010-12-03 |publisher=长春轨道客车股份有限公司 |date= }}</ref>。这批动车组计划于2011年4月开始交付，全部80列动车组于2012年6月完成交付。

2010年5月27日，首列CRH3-380列车完成样车试制，并在[[吉林省|吉林省]][[長春市|長春市]]绿园区轨道交通装备制造产业园首次向外界展示，并举行了“新一代高速动车组“和谐号”380A下线”仪式，同时并举行了长客股份高速车制造基地一期工程竣工典礼<ref>{{cite web |url = http://www.gov.cn/gzdt/2010-05/28/content_1615635.htm |title = 新一代高速动车组"和谐号"380A首辆车在长春下线 |publisher = 中国政府网 |date = 2010-05-28 |accessdate = 2010-05-28}}</ref><ref>{{cite web
| url = http://inews.mingpao.com/htm/INews/20100527/ca42128a.htm | title = 時速380公里高鐵出廠| publisher = [[明報|明報]]| date = 2010-05-28| accessdate = 2010-05-28}}</ref>。铁道部部长[[刘志军|刘志军]]、吉林省委书记[[孙政才|孙政才]]、铁道部副总工程师[[张曙光_(铁道部)|张曙光]]等出席了这次典礼，刘志军更表示，“长客能生产多少，铁道部就采购多少，前提是要保证质量”<ref>{{Cite web |url=http://news.qq.com/a/20100528/000211.htm |title=世界最快列车在长春下线 最高时速380公里 |accessdate=2010-12-03 |publisher=腾讯网 |date=2010-05-28 }}</ref>。

根据铁道部于2010年9月下发的《关于新一代高速动车组型号、车号及坐席号的通知》，原将将70列由[[唐山轨道客车|唐山轨道客车]]制造，110列由[[长春轨道客车|长春轨道客车]]（含新头型）制造的CRH3-380型动车组定型为CRH380B系列。至2010年12月中旬，铁道部下发《运装客车〔2010〕807号文件》，正式将25列由长春轨道客车生产的CRH3-380新一代新头型长编组座车动车组定型为'''CRH380CL'''。

目前，该型号的全部量产车已经在京沪高铁正式运行。

由2014年7月1日起，所有CRH380动车组編號均作出了更改，CRH380C原編號為（CRH380C-6301L～CRH380C-6325L），現改為（CRH380CL-5601～CRH380CL-5625）<ref name="铁总运(2014)150号">{{Cite web|url=http://www.ditiezu.com/thread-375095-1-1.html|title=《中国铁路总公司关于印发动车组型号、车号和坐席号编号规则的通知》|accessdate=2015-09-22|work=中国铁路总公司|publisher=中国铁路总公司|date=2014-06-13}}</ref><ref name="铁总运(2014)150号-weibo">{{Cite web|url=http://weibo.com/1736649017/BaPm4rmfQ|title=《中国铁路总公司关于印发动车组型号、车号和坐席号编号规则的通知》|accessdate=2015-09-22|work=中国铁路总公司|publisher=中国铁路总公司|date=2014-06-13}}</ref>。

== 技术特性 ==
CRH380C型动车组是在[[和谐号CRH3型电力动车组|CRH3C]]、[[和谐号CRH380B型电力动车组|CRH380BL]]基础上研发的新一代高速动车组，与[[和谐号CRH3型电力动车组|CRH3C]]相比，持续运营时速为由300公里提高至350公里，最高设计时速由350公里提高到380公里，最高试验时速为400公里以上，性能优化以提高牵引功率、降低传动比及动车组气动外形减阻为主，而列车舒适度优化方面主要采取提高列车减震性能、车厢降噪、加强车内气压控制等方式。

CRH380CL型动车组列车总数为25列（CRH380CL-5601～5625），全部由长春轨道客车生产，采用了8动8拖的编组方式，牵引功率为19200千瓦，将采用新头型及基于日立技术的永济牵引系统。列车由2辆商务车（又称VIP座车）、2辆一等座车、11辆二等座车和1辆餐车组成，定员为1015.<ref>{{cite web| url = http://www.worldrailway.cn/TEBIECEHUA/wjtb/200912/76284_9.html| title = 夯实基础求发展中国轨道交通产业喜结硕果——“2009年世界轨道交通发展研究会年会”在京召开| date = 2009-12-09| publisher = 世界軌道交通資訊網| accessdate = 2010-05-28| deadurl = yes| archiveurl = https://web.archive.org/web/20111007004735/http://www.worldrailway.cn/TEBIECEHUA/wjtb/200912/76284_9.html| archivedate = 2011-10-07}}</ref>。

值得一提的是，CRH380CL-5601的头型与其余24列的头型有区别：除5601之外其余车型车头增加了的银色装饰板，车侧車窗范围的黑色塗裝并非一体式而是分体式，实际上可以当5601作量产试制车。
