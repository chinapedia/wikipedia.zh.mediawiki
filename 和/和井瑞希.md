'''和井瑞希'''（{{bd|1991年|7月26日|catIdx=Takeshita Reina}}<ref name="chain">{{Cite web |url=http://www.joqr.co.jp/chainchronicle_audition/judge/|title=ェンクロキャラクターボイスオーディション合格者|accessdate=2016-07-03}}</ref>）是[[日本|日本]]的女性[[聲優|聲優]]、[[演員|演員]]，[[廣島縣|廣島縣]]出身。血型O型。[[JTB娛樂|JTB娛樂]]所屬<ref name="jtb">{{Cite web|url=http://www.jtb-entertainment-academy.com/pro/?info=%e6%96%b0%e6%89%80%e5%b1%9e%e3%81%ae%e3%81%8a%e7%9f%a5%e3%82%89%e3%81%9b-3。|title=新所属のお知らせ|publisher=JTB娛樂|accessdate=2016-07-03}}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

== 人物 ==
10歲時在租片店借《[[秀逗魔導士|秀逗魔導士]]》的片子時，被當時飾演莉娜·因巴斯一角的[[林原惠|林原惠]]的演技所感動，並在小學、國中、高中時都加入演劇社。之後藉由打工努力存錢打算到東京念書。高中畢業後，進入[[日本大學|日本大學藝術學部]]演劇學科演技課程就讀。後來以最優秀的成績畢業。畢業時更獲得日本大學藝術學部優等賞<ref>{{Cite web |url=http://koyu.art.nihon-u.ac.jp/publication/images/kaihou92.pdf|title=日本大学芸術学部交友会報92|accessdate=2016-06-03}}</ref>。

大學在學中曾經當過英雄秀的主持人<ref>{{Cite web |url=http://nakigatari.com|title=泣きがたりライブ〜郷愁〜ARTIST紹介|accessdate=2016-06-03}}</ref>。

2015年在手機遊戲《[[鎖鏈戰記|鎖鏈戰記]]》「鎖鏈戰記角色聲音甄選」內，在一般、專業合計2000人參與的激烈競爭中獲得[[岸田梅爾|岸田梅爾]]獎的大獎，並以壹領忍眾纏此一角色正式出道<ref>{{Cite web |url=http://sp.animatetimes.com/news/details.php?id=1436506385|title=審査委員長・緑川光さんが、新人声優に送ったひと言とは!?|accessdate=2016-06-02}}</ref>。

2016年4月1日成為[[JTB娛樂|JTB娛樂]]的正式成員<ref name="jtb"/>。5月時被拔擢飾演《[[初戀怪獸|初戀怪獸]]》中的橫內千秋一角，也是初次演出常規角色的作品<ref>{{Cite web |url=http://hatsukoimonster.jp/news/8.html|title=お知らせ 初恋モンスター|accessdate=2016-06-03}}</ref>。
