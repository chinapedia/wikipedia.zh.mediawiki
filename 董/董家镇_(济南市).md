'''董家镇'''，是{{PRC admin/fullname|37/01/12/106/000|3}}下辖的一个[[乡镇|乡镇]]级行政单位。

== 行政变革 ==

* 明崇祯十三年（1640）属巨冶河路
* 清乾隆三十六年（1771）属东北乡
* 民国十三年（1924）属老僧口乡
* 1948年'''董家区'''
* 1950年设'''第四区董家乡'''
* 1956设'''董家区董家集镇乡'''
* 1958设平原公社
* 1959属'''平原公社董家管理区'''
* 1961年将平原公社改称'''董家区'''
* 1968年撤区改为'''董家公社'''
* 1984年政社分开改称'''董家区'''
* 1985年撤区设立'''董家镇'''
* 2016年12月撤'''销董家镇设立董家街道办事处'''<ref>{{Cite web|url=http://www.licheng.gov.cn/art/2016/12/22/art_24473_1628356.html?xxgkhide=1|title=中共济南市历城区董家镇委员会关于撤销董家镇设立董家街道办事处的意见|accessdate=|author=中共济南市历城区董家镇委员会|date=2016年12月22日|publisher=}}</ref>

====== '''民国24年（1935）''' ======

* 秋  中共济南市委组织乡村讲演团，到历城农村宣传抗日救国。
* 是年  山东省政府在历城推行国民义务教育，'''董家区'''被选为省国民义务教育第一试验区。

====== '''民国26年（1937）''' ======

* 2月  王健民在刘家庄建立农民夜校， 在'''董家庄'''创办书刊文具消费合作社，开展党的工作。
* 3月  中共历城特别支部在'''董家庄'''女校成立读书会，在群众中组织抗日救国会、抗日民族解放先锋队。

* 7月  “七七” 事变后，中共历城特别支部撤离'''董家试验区'''，留王心崇在此坚持地下工作。

====== '''民国38年（1949）''' ======

*  5月  历城县划归鲁中南行政区泰安专区。全县共设9个区，即2个老区（西营、柳埠），6个新区（张马、遥墙、'''董家'''、东梧、邵而、泉泸），1个新收复区（卧龙）

====== '''共和国1951年''' ======

* 5月  县委、县政府机关迁至'''董家庄'''。

====== '''共和国1952年''' ======

* 2月  四区'''董家庄'''农民吕素贵办起全县第一个互助组。

====== 共和国1968年 ======

* 8月20日  西郊、南郊、唐王、'''董家'''、彩石、卧牛、吴家堡、窑头、党家等9个公社分别改为东方红、英雄山、红卫、平原、红旗、华山、太阳升、向阳、东风公社。

====== 共和国1973年 ======

* 3月15日  '''董家'''、港沟、邵而、柳埠医院转为历城县人民医院分院。

====== 共和国1984年 ======

* 6月1日  全县首届光荣的预备队少先队检阅式在'''董家镇'''举行。
* 11月20日  共青团济南市委、中共历城县委、县教育局、中共'''董家区'''委的负责人以及全县教师、青少年代表1200人，'''在董家'''区韩仓小学集会，悼念舍己救人、英勇献身的冯宝剑同学。会上，团县委追授冯宝剑“优秀少先队员”的光荣称号，并作出全县青少年向冯宝剑同学学习的决定。
