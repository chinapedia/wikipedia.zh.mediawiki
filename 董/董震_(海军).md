'''董震'''（{{bd|1963年|6月||}}）[[山东|山东]][[莱芜|莱芜]]人，[[中国人民解放军海军|中国人民解放军海军]]人物<ref name=rddb>{{Cite web|url =http://news.xinhuanet.com/zhengfu/2004-03/11/content_1359596.htm|title =十届全国人大代表董震|author =|date =2004-03-11|publisher =新华网|language =|accessdate =|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20040826140254/http://news.xinhuanet.com/zhengfu/2004-03/11/content_1359596.htm|archivedate =2004-08-26}}</ref>。

==生平==
1980年9月入伍。1985年12月加入[[中国共产党|中国共产党]]。[[中国人民解放军海军潜艇学校|中国人民解放军海军潜艇学校]]战术指挥专业毕业，大学本科学历<ref name=rddb/><ref name=jlz>{{Cite web|url =http://news.xinhuanet.com/mil/2013-10/26/c_125602333.htm|title =艇长立下“军令状”|author =|date =2013-10-26|publisher =新华网|language =|accessdate =|quote =}}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。后又进入高等学府深造，获研究生学历。2001年3月，已任艇长两年的董震出任海军某型潜艇艇员队队长。该艇员队驾驭的某型潜艇曾被授予“水下先锋艇”称号。董震出任艇员队队长后，与艇员队官兵连续3次完成重大战备巡逻及演习任务；第一次使用实弹攻击实体目标并一举击沉；艇员独立操作合格率达100％，一半艇员达到了两个以上岗位合格；先后填补了该型潜艇十多项战术训练空白<ref name=zhishi>{{Cite web |url =http://news.sohu.com/20050312/n224653030.shtml  |title = 董震：用知识和真心带兵大洋深处创辉煌 |author =  |date = 2005-03-12 |publisher = 搜狐 |language =  |accessdate =  |quote =  }}</ref>。董震还出任[[第十届全国人大代表|第十届全国人大代表]]<ref name=zhishi/><ref name=rddb/>。后来，董震升任[[中国人民解放军海军潜艇第一基地|中国人民解放军海军潜艇第一基地]]副司令员。2017年出任司令员。截至2013年，董震先后立二等功2次、三等功2次，获[[中国人民解放军海军北海舰队|中国人民解放军海军北海舰队]]评为“优秀基层个人标兵”；获中国人民解放军海军评为“人才建设先进个人标兵”、“海军优秀指挥军官”<ref name=jlz/><ref>{{Cite web |url =  http://mil.sohu.com/s2013/zgszqtbdbg/|title =“大国重器” 走进中国海军“杀手锏”部队  |author =  |date = 2013-10-26 |publisher = 搜狐 |language =  |accessdate =  |quote =  }}</ref>。
