'''董念台'''{{BD|？||||}}，籍貫甘肅蘭州<ref name="向呂秀蓮求婚"/>，台灣黑道人物，自稱「黑道狀元」<ref>{{cite book |author=董念台|title=《黑道狀元：董念台》 |date=2001-07-19|publisher=上登出版社 |location=台灣|language=zh-tw |isbn=9789577448644}}</ref>，以討債維生<ref>{{cite book |author=董念台|title=《壞到剛剛好》 |date=2003-10-01|publisher=大都會出版社|location=台灣|language=zh-tw |isbn=9789574114054}}</ref>，再生受刑人服務中心社長。

==生平==
董念台曾因《[[中華民國刑法|中華民國刑法]]》詐欺罪等罪入獄多次<ref>{{cite book |author=胡台麗|title=《燃燒憂鬱》 |date=1991-07-01|publisher=張老師文化 |location=台灣|language=zh-tw |isbn=9579486557}}</ref>。1989年，董念台在[[士林地方法院|士林地方法院]]以維護司法的最後尊嚴為理由，指責旁聽的[[藝術家|藝術家]][[許曉丹|許曉丹]]衣服穿太少<ref>{{cite book |author=李敖|title=《笑傲五十年》 |date=1999-05-01|publisher=商周出版 |location=台灣|language=zh-tw |isbn=9576673348}}</ref>。

2000年7月，時任受刑人再生協會理事長的董念台與愛滋病患中途之家負責人[[祁家威|祁家威]]簽訂合約，董念台經營的[[討債公司|討債公司]]將提供六個催討專員職務給[[愛滋病|愛滋病]]病患，並捐出[[新臺幣|新臺幣]]五萬元作為愛滋病患者救助基金。董念台表示，可藉著愛滋病患帶來債務人的精神壓力達到催討目的，同時提供愛滋病患者就業管道。祁家威表示，目前有很多雇主對愛滋病患「敬而遠之」，因此他樂見董念台的討債公司提供愛滋病患就業機會，讓愛滋病患走到陽光之下<ref>{{cite news| url =http://gsrat.net/news/newsclipDetail.php?ncdata_id=132| date=2000-07-25|accessdate=2015-02-15|title=「愛滋病患催討大隊」成軍？| author=張企群| publisher =《中國時報》| language =zh-tw}}</ref>。

2001年12月11日，董念台在二十四輛[[凱迪拉克|凱迪拉克]]豪華禮車前導下，朝向[[中華民國總統府|中華民國總統府]]開拔，公開向副總統[[呂秀蓮|呂秀蓮]]求婚但失敗<ref name="向呂秀蓮求婚">{{cite news| url =http://www.chinesedc.com/Authors/GeneralShen/2001_12/1220d.htm| date=2001-12-20|accessdate=2015-02-15|title=董念台向呂秀蓮求婚| author=沈敏| publisher =大華府網| language =zh-tw}}</ref>。

2013年，[[洪仲丘事件|洪仲丘事件]]，控告洪仲丘之姐[[洪慈庸|洪慈庸]]與[[公民1985行動聯盟|公民1985行動聯盟]]涉利用群眾逼迫政府修法，違反《組織犯罪防制條例》<ref>{{cite news| url =http://www.appledaily.com.tw/realtimenews/article/new/20130806/237838/| date=2013-08-06 |accessdate=2015-02-15|title=董念台告洪姊姊　網友狂轟：豬頭腦殘| author=| publisher =《蘋果日報》即時新聞| language =zh-tw}}</ref>，以及控告承審洪仲丘案的[[桃園地方法院|桃園地方法院]]法官涉嫌瀆職、圖利原告<ref>{{cite news| url =http://www.appledaily.com.tw/realtimenews/article/local/20130826/248622/3/| date=2013-08-26 |accessdate=2015-02-15|title=董念台穿西裝加短褲　告洪案3法官| author=胡守得| publisher =《蘋果日報》即時新聞| language =zh-tw}}</ref>。

2014年，前總統[[陳水扁|陳水扁]]保外就醫時，時任再生受刑人服務中心社長的他率眾至[[法務部|法務部]]抗議，要求所有的受刑人享有相同的福利，否則發動囚犯鬧房<ref>{{cite news| url =http://www.appledaily.com.tw/realtimenews/article/new/20141208/520241/| date=2014-12-08 |accessdate=2015-02-15|title=若只放扁　董念台：將發動囚犯鬧房| author=| publisher =《蘋果日報》即時新聞| language =zh-tw}}</ref>。他認為陳水扁裝病，告發為陳水扁鑑定的15名醫師偽造病歷<ref>{{cite news| url =http://setn.com/News.aspx?PageGroupID=6&NewsID=55738| date=2015-01-05 |accessdate=2015-02-15|title=控15名醫師偽造阿扁病歷　董念台：他只是身體老化而已| author=| publisher =三立新聞網| language =zh-tw}}</ref>。[[2015年高雄監獄挾持事件|2015年高雄監獄挾持事件]]後，他贊成6名受刑人的行為，打算策劃下一波監獄暴動<ref>{{cite news| url =http://news.ltn.com.tw/news/society/breakingnews/1233615| date=2015-02-13 |accessdate=2015-02-15|title=反扁保外就醫 董念台擬策劃下波監獄暴動| author=| publisher =《自由時報》即時新聞| language =zh-tw}}</ref>。

2014，董念台偕同律師至警局報案，表示自己遭黑道恐嚇。據了解，董念台在新北市汐止區有1店面、具有1/4股權，但近來他疑似遭到本省掛黑道騷擾恐嚇，疑似遭到道上兄弟「洗面」。<ref>{{cite news|title=討債專家董念台遭恐嚇 竟找警方報案|url=http://www.chinatimes.com/realtimenews/20141013005029-260402|publisher=中國時報即時新聞|date=2014-10-13|accessdate=2015-10-31}}</ref>

2018年5月，68歲董念台與35歲的金融業務員黃逸珊登記結婚，並於6月舉辦婚禮。<ref>[https://tw.news.appledaily.com/local/realtime/20180627/1380758/ 【專訪】金融OL收服討債專家　董念台「幸福來了！」]</ref>
