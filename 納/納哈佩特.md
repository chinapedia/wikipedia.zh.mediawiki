《'''納哈佩特'''》（{{lang-hy|Նահապետ}}），又名《'''生命得勝'''》（{{lang-en|''Life Triumph''}}），是一部在1977年上映的[[亞美尼亞|亞美尼亞]][[劇情片|劇情片]]，劇情乃是基於赫拉奇亞·科恰爾所創作同名[[小說|小說]]，講述一名[[亞美尼亞大屠殺|亞美尼亞大屠殺]]生還者的故事。

== 內容 ==
納哈佩特是一名亞美尼亞大屠殺生還者，他在大屠殺中被綁到木柱上，親眼看着妻子和兒女遭到[[鄂圖曼土耳其帝國|鄂圖曼土耳其帝國]]的士兵殺害{{sfn|Khiterer|Barrick|Misal|2014|p=299}}。大屠殺過後，納哈佩特來到亞美尼亞蘇維埃社會主義共和國偏遠的地方居住，當地的[[蘇聯共產黨|共產黨人]]推行[[土地改革|土地改革]]，引進[[電力|電力]]供應，又協助納哈佩特在家附近種一棵[[蘋果樹|蘋果樹]]；最終，納哈佩特逐漸步進了新生活，迎娶了另一名妻子，並與她生下一個兒子{{sfn|Khiterer|Barrick|Misal|2014|p=299}}。

這部電影經常使用[[倒敘|倒敘]]手法，展示納哈佩特一家人在亞美尼亞大屠殺期間的遭遇{{sfn|Whitehorn|2015|p=130}}。電影其中一幕是這樣的：一大堆紅蘋果從蘋果樹上掉下來，滾進了一條[[河流|河流]]裡，並一起順流而下，這是隱喻[[青年土耳其黨人|青年土耳其黨人]]在大屠殺期間把大量[[亞美尼亞人|亞美尼亞人]]遺體扔進[[幼發拉底河|幼發拉底河]]的情景{{sfn|Whitehorn|2015|p=130}}。電影沒有明確提及被指[[種族滅絕|種族滅絕]]亞美尼亞人的[[土耳其|土耳其]]{{sfn|Khiterer|Barrick|Misal|2014|p=299}}。

電影片名「納哈佩特」在亞美尼亞語是指[[族長|族長]]，可以回溯到亞美尼亞民族的創始人[[哈伊克|哈伊克]]{{sfn|Khiterer|Barrick|Misal|2014|p=299}}。

== 製作 ==
《納哈佩特》被認為是首部專門介紹亞美尼亞大屠殺的電影；在此之前，「拍攝以亞美尼亞大屠殺為題材的電影」是一個具突破性的想法{{sfn|Բաբաջանյան|2015}}。時任[[亞美尼亞共產黨_(1920年)|亞美尼亞共產黨]]領導人{{link-en|卡連·傑米爾強|Karen Demirchyan}}與導演根里赫·馬良的關係密切，在傑米爾強決定此電影將會基於赫拉奇亞·科恰爾所的小說《納哈佩特》首部分內容後，製作人員把計劃告知[[蘇聯|蘇聯]]當局；蘇聯政府最終在1976年批准拍攝這部電影，電影的製作也在同年開始{{sfn|Բաբաջանյան|2015}}。

此電影主要是在亞美尼亞[[阿拉加措特恩省|阿拉加措特恩省]]的城鎮{{link-en|塔林 (亞美尼亞)|Talin, Armenia|塔林}}和{{link-en|季安|Dian, Armenia}}取景，亦有部分鏡頭在[[電影製片廠|電影製片廠]]內拍攝{{sfn|Բաբաջանյան|2015}}。

== 反響 ==
《納哈佩特》曾在多個主要電影獎項中獲獎；有評論認為，這部電影有機會是導演根里赫·馬良最為人稱道的作品{{sfn|Tropvale Limited|1984}}。{{link-en|亞美尼亞通訊社|Armenpress}}曾在2015年10月刊出一篇評論，列出十部不可不看的亞美尼亞大屠殺電影，其中《納哈佩特》位處此列表的首位{{sfn|Grigoryan|2015}}。
