'''納瓦拉'''（{{lang-es|Navarra}} [naˈβara]；{{lang-eu|Nafarroa}} [nafaˈroa]；{{lang-oc|Navarra}} [naˈbaʁɔ]；{{lang-en|Navarre}}，{{IPAc-en|lang|n|ə|ˈ|v|ɑr}}） 是[[西班牙|西班牙]]北部一個[[西班牙行政區劃|自治區]]，正式名称是'''纳瓦拉特许共同体'''（{{lang-es|Comunidad Foral de Navarra}}），全区仅有纳瓦拉一省，总面积10,391平方千米<ref>[http://www.ine.es/inebaseweb/pdfDispacher.do?td=154090&L=0 Superficie y altimetría].INE.[2018-09-05]</ref>，2017年总人口643,234<ref>[http://www.ine.es/jaxiT3/Datos.htm?t=2852 Población por provincias].INE.[2018-09-05]</ref>。首府为[[潘普洛纳|潘普洛纳]]。

== 历史 ==
===地名语源===
第一次出现在书面文献上的纳瓦拉的名称可以追溯到9世纪早期，在[[法兰克王国|法兰克王国]]史学家[[艾因哈德|艾因哈德]]对[[神圣罗马帝国|神圣罗马帝国]]皇帝[[查理曼|查理曼]]大帝的编年史著作中，出现了诸如“navarros”的表记。<ref name="Auñamendi">Bernardo Estornés Lasa, [http://www.euskomedia.org/aunamendi/96788 Navarra] {{webarchive|url=https://web.archive.org/web/20120112133908/http://www.euskomedia.org/aunamendi/96788 |date=2012-01-12 }} in the [[Auñamendi_Entziklopedia|Auñamendi Entziklopedia]] (click on "NAVARRA – NAFARROA (NOMBRE Y EMBLEMAS)")</ref>其他的晚期著作如《[[法兰克王国编年史|法兰克王国编年史]]》记载有“nabarros”。该名称的词源主要有2种说法：<ref name="Auñamendi"/>
*[[巴斯克语|巴斯克语]]的''nabar''（[[通格|通格]]单数形式：''nabarra''，意为褐色的，多色的）。
*巴斯克语的''naba''（或[[卡斯蒂利亚语|卡斯蒂利亚语]]：''nava''，意为平原） + 巴斯克语的''herri''（意为人或土地）。<ref>{{Cite web |url=https://www.etymonline.com/word/Navarre#etymonline_v_45373 |title=Navarre |accessdate=2018-09-08 |publisher=etymonline.com}}</ref>

===纳瓦拉王国时期===
纳瓦拉自治区居民以[[巴斯克人|巴斯克人]]为主。778年，法兰克君主[[查理大帝|查理大帝]]南下进军受挫，他率领兵将穿过[[奥雷亚加|龙塞斯瓦列斯]]隘口回朝时，遭到巴斯克人[[隆塞斯瓦耶斯隘口战役_(778年)|伏击]]。大战后不久，巴斯克人独立，所建立的国家到905年改称为[[纳瓦拉王国|纳瓦拉王国]]。

[[桑乔三世_(纳瓦拉)|桑乔三世]]在位时，国势最强，靠着其文治武功和联姻，领土大为扩充。从13世纪起，王位转到法国人的手中。

1512年，上纳瓦拉被[[西班牙|西班牙]]兼并。下纳瓦拉继续保持独立，1589年因其国王[[亨利四世_(法兰西)|亨利]]继承法国王位而并入[[法国|法国]]。

亨利即法国[[波旁王朝|波旁王朝]]第一代君主亨利四世，他的四世孙后来成为[[西班牙国王|西班牙国王]]，即[[菲利浦五世_(西班牙)|菲利浦五世]]，创建了西班牙的波旁王朝。

== 地理 ==
纳瓦拉地处[[比利牛斯山|比利牛斯山]]西段南麓，这一带山区以丘陵为主，仅南部[[埃布罗河|埃布罗河]]有些平原。

== 经济 ==
纳瓦拉南部平原种植小麦、芦笋和青椒等。

矿业以开采[[钾盐|钾盐]]为主，工业集中在首府[[潘普洛納_(西班牙)|潘普洛纳]]一带。

== 文化 ==
{{main|圣费尔明节}}
纳瓦拉首府潘普洛纳以[[圣费尔明节|奔牛节]]闻名于世。

== 政治 ==
纳瓦拉与邻邦[[巴斯克地区|巴斯克地区]]来往密切，语言有相当的共通性，历史和风俗习惯也大致相同，但由于立场不同而各行其道。

[[西班牙内战|内战]]时，纳瓦拉的保守派加入了[[国民军_(西班牙内战)|弗朗哥阵营]]，而巴斯克地区三省则押注于{{Tsl|en|Republican faction (Spanish Civil War)|共和军 (西班牙内战)|共和政府}}这一边，双方势同水火。

[[弗朗哥|弗朗哥]]死后，巴斯克民族主义领袖，起草拟定区域性自治纲领，企图把纳瓦拉拉入加盟。但纳瓦拉还是成立了自己的自治区。
