'''屯田制'''，是[[中國|中國]][[東漢|東漢]]末年[[三国|三国]]时期[[曹操|曹操]]采纳部下[[枣祗|枣祗]]和[[韩浩|韩浩]]等的建议而施行的土地制度，对三国以及之后很长时间的[[中国历史|中国历史]]与社会产生了深远的影响。


{{Portal|經濟學|Emblem-money.svg}}
以军队戍边屯田，[[西汉|西汉]]时就常用于开拓和驻守西北边疆。[[汉文帝|汉文帝]]时，大臣[[晁错|晁错]]就曾建议“徙民实边”。[[漢武帝|漢武帝]]時，[[趙充國|趙充國]]建議「屯田」於邊防，戍衛與墾耕並顧，自[[敦煌|敦煌]]西至[[鹽澤|鹽澤]]往往起亭，而[[輪臺|輪臺]]和[[渠犁|渠犁]]皆有田卒數百人。此一方法用意，既可解決路途遙遠交通不便之下自力更生，又可使兵力在守防時亦不白花人力，乃一舉兩得之構想。但是直到东汉末年，所有屯田的构想与实施都只限于避免从异地长途运输粮食，解决边境守备军队之需，并不作为经济和社会制度。

而最早提出屯田制度的则是曹操麾下的[[枣祗|枣祗]]<ref>[http://www.guoxue.com/shibu/zztj/content/zztj_062.htm 《资治通鉴·卷第六十二·汉纪五十四》] 羽林监枣祗请建置屯田，曹操从之，以祗为屯田都尉，以骑都尉任峻为典农中郎将。</ref>，此外还有[[韩浩|韩浩]]<ref>[http://www.guoxue.com/shibu/24shi/sangzz/sgzz_001.htm　《三国志·魏书一·武帝纪第一》]　建安元年……是岁用枣祗、韩浩等议，始兴屯田。</ref> <ref>[http://www.guoxue.com/shibu/24shi/sangzz/sgzz_009.htm　《三国志·魏书九·诸夏侯曹传第九》] 韩浩者，河内人。……（裴松之注：）魏书曰：韩浩字元嗣。……时大议损益，浩以为当急田。太祖善之，迁护军。</ref>，而实行者包括[[任峻|任峻]]<ref>[http://www.guoxue.com/shibu/24shi/sangzz/sgzz_016.htm  《三国志·魏书十六·任苏杜郑仓传第十六》]任峻字伯达，河南中牟人也……是时岁饥旱，军食不足，羽林监颍川枣祗建置屯田，太祖以峻为典农中郎将，〔募百姓屯田於许下，得谷百万斛，郡国列置田官〕，数年中所在积粟，仓廪皆满。官渡之战，太祖使峻典军器粮运。贼数寇钞绝粮道，乃使千乘为一部，十道方行，为衤复陈以营卫之，贼不敢近。军国之饶，起於枣祗而成於峻。</ref>和[[曹魏|曹魏]]時期的[[鄧艾|鄧艾]]<ref>[http://www.guoxue.com/shibu/24shi/sangzz/sgzz_028.htm　《三国志·魏书二十八·王毌丘诸葛邓锺传第二十八》]　邓艾字士载，义阳棘阳人也。……时欲广田畜谷，为灭贼资，使艾行陈、项已东至寿春。艾以为“田良水少，不足以尽地利，宜开河渠，可以引水浇溉，大积军粮，又通运漕之道。”乃著济河论以喻其指。又以为“昔破黄巾，因为屯田，积谷于许都以制四方。今三隅已定，事在淮南，每大军征举，运兵过半，功费巨亿，以为大役。陈、蔡之间，土下田良，可省许昌左右诸稻田，并水东下。令淮北屯二万人，淮南三万人，十二分休，常有四万人，且田且守。水丰常收三倍於西，计除众费，岁完五百万斛以为军资。六七年间，可积三千万斛於淮上，此则十万之众五年食也。以此乘吴，无往而不克矣。”宣王善之，事皆施行。正始二年，乃开广漕渠，每东南有事，大军兴众，汎舟而下，达于江、淮，资食有储而无水害，艾所建也。</ref>等。曹操与他奠基的曹魏政权整合军屯与民屯，在各地设立田官专门负责屯田。屯田制逐步确立后，成为了国家政权的钱粮收入来源，同时也解决了屯田军民本身的生计。

早在随曹操镇压[[黄巾军|黄巾军]]餘部时，枣祗就对亦战亦耕、兵农合一的做法产生了极大兴趣，[[建安_(东汉)|建安]]元年（196年），曹操击败了颍川汝南的黄巾军，夺得了一大批耕牛、农具和劳动力。枣祗建议曹操利用这些农具，在[[许昌|许昌]]一带开垦土地，实行屯田，以解决粮食问题。曹操采纳了他的建议，并任命他为屯田[[都尉|都尉]]，全权负责屯田事宜。枣祗首先将荒芜的无主农田收归国家所有，将招募到的大批[[流民|流民]]按军队的编制编成组，由国家提供土地、种子、耕牛和农具，由他们开垦耕种，获得的收成由国家和屯田的农民按比例分成。屯田实施的第一年，就得[[榖|榖]]百万[[斛|斛]]，曹操于是下令郡国都置田官，招募流亡百姓屯田。后来又接受枣祗的建议，下令军队屯田，[[屯田制|屯田制]]得到广泛推行。

== 分類 ==
=== 軍屯 ===
{{main|军屯}}
軍屯是指設立土地予不用打仗的軍士，要求士兵能自行耕作而生產行軍所需糧食。軍屯以六十人為一營，且佃且守，士卒需繳納分成地租。

=== 民屯 ===
民屯則指發放土地（[[使用权|使用权]]）予一般民眾耕作，土地業權（[[所有权|所有权]]）仍歸於政府所有，而民眾耕作所獲收成則扣除佃租，餘下歸私人所有。負責耕作的民眾不能擅自棄下配給的土地不顧，否則將會受罰。一般以五十人為一屯，屯田民稱屯田部民、屯田客。屯設屯司馬，其上置典農都尉、典農校尉、典農中郎將等官員管理，直屬于中央。佃租一般采取分成方式，使用官牛者，繳納十分之六，使用私牛者，繳納一半。

== 影響 ==
東漢末年[[蝗災|蝗災]]為患，社會亦十分動盪。農耕設施受到戰爭破壞，更兼[[壯丁|壯丁]]被[[軍閥|軍閥]]強徵入伍，農作收成大為減少，糧食供應十分短缺，甚至傳聞有[[中国食人史|人吃人]]的事件發生，曹操这一制度纾緩了當時中國北方糧食需求的壓力。

枣祗首倡屯田制的实施，使长期遭受战争破坏的北方农业生产，在短期内得以恢复并稳定了下来。失去土地的农民又重新回到土地上来，许多荒芜的农田也被开垦，政府积存了大量的粮食。“数年中所在积粟，仓廪皆满”，使曹操“征伐四方，无运粮之劳”。屯田制的实施，不仅为曹操解决了令人头疼的军粮问题，而且还为他争取了大量的人口，加快了统一北方的进程。但由於常被半強迫性的遷移，必須遠離家鄉去拓荒，還要服勞役，導致百姓相繼逃亡以外，也爆發民變，像是屯田客呂並陳倉起義、[[侯音|侯音]]之亂等。

由枣祗首倡实施的“屯田制”，不仅在当时起了一定的积极作用，而且为后世开创了一种大规模的寓兵于农、兵农合一的先例，为历代封建统治阶级所不同程度地仿效，在中国政治、经济、军事发展史上占有重要的地位。曹魏時期的名將[[邓艾|鄧艾]]，在屯田實踐亦有重大貢獻。

但有其矛盾之處，進行官渡之戰時，曹操仍未軍糧所苦，是否有做到"數年中所在積粟，倉廩皆滿"還需考證

晉武帝時廢除「屯田制」，改實施[[占田制|占田制]]。

== 评价 ==
[[香港|香港]]首富及著名[[企业家|企业家]][[李嘉誠|李嘉誠]]在其為《世紀三國—曹操之霸者勝出》<ref>{{cite book zh|title=《世紀三國—曹操之霸者勝出》|isbn=9789577457783|author=羅吉甫|date=2005年7月7日|publisher=格林}}</ref>一書所作的序言中分析：「『屯田制』使魏國大大提高糧食產量，一年收穫百萬斛，使軍糧不再短缺。令曹操有雄厚的資本作支援，而在軍事上佔優。」
