'''屯門兒童合唱團'''成立於1981年，為[[香港政府|香港政府]]註冊的非牟利[[慈善團體|慈善團體]]。除男女童聲合唱外，屯門兒童合唱團於1989年成立自己的管弦樂隊<ref>{{Cite web |url= http://www.tmcc.org.hk/index.php?option=com_content&view=article&id=24&Itemid=40 |title= 宗旨及成立經過  |date=2006-08-29 |language= zh-tw }}</ref>。
屯門兒童合唱團亦常跟著名合唱團作交流或聯合演出，包括新界九區兒童合唱團、葉氏兒童合唱團等。

== 歷年海外演出 ==
<ref>{{Cite web |url= http://www.tmcc.org.hk/index.php?option=com_content&view=article&id=13&Itemid=28 |title= 赴外地演出一覽 |date=2010-06-08 |language =zh-tw }}</ref>
由一九九零起，合唱團經常赴外地演出，當中除交流表演外，亦包括全國性比賽、大型活動儀式等。
