'''蒋伸'''（{{bd|799年||881年|catIdx=J}}<ref name=birthdeath>此生卒年见于[http://daj.chinawuxi.gov.cn/web101/wxmr/zzjmr/227364.shtml 无锡市政府网对蒋伸的简介] {{webarchive|url=https://web.archive.org/web/20131213035853/http://daj.chinawuxi.gov.cn/web101/wxmr/zzjmr/227364.shtml |date=2013-12-13 }}，但没有引用出处。</ref>），字'''大直'''，'''乐安郡开国公'''<ref name=QTW83>[[:s:授蔣伸畢誠節度使制|:s:授蔣伸畢誠節度使制]]</ref>，[[唐朝|唐朝]]官员，[[唐宣宗|唐宣宗]]和[[唐懿宗|唐懿宗]]年间为[[宰相|宰相]]。

== 家世 ==
蒋伸祖籍[[常州|常州]][[宜兴市|义兴]]，其父[[蔣義_(唐朝)|蒋乂]]为官时全家迁到东都[[洛阳|洛阳]]。<ref name=NBT132>[[:s:新唐書/卷132|《新唐书》卷一百三十二]]</ref>蒋乂祖父[[蔣瓌|蔣瓌]]为太子洗馬、[[弘文館|弘文館]]學士，父亲[[蒋将明|蒋将明]]为國子司業、[[集賢殿|集賢殿]]學士、副知院使，蒋乂本人为[[祕書監|祕書監]]，<ref name=NBT75>[http://www.sidneyluo.net/a/a17/075.htm 《新唐书》卷七十五] {{webarchive|url=https://web.archive.org/web/20091219203329/http://www.sidneyluo.net/a/a17/075.htm |date=2009-12-19 }}</ref>还是卓越的史学家，著《大唐宰辅录》七十卷、《凌烟阁功臣》、《秦府十八学士》、《史臣》等传四十卷，封义兴县公，谥懿。蒋乂有五个已知的儿子，蒋伸是第二个，至少有一兄[[蒋係|蒋係]]，后也贡献突出，封淮阳郡公。蒋伸至少有三个弟弟[[蒋偕|蒋偕]]、[[蒋仙|蒋仙]]、[[蒋佶|蒋佶]]，都为官。蒋伸中[[进士|进士]]，效力各地的一些刺史为僚佐。<ref name=BT149>[[:s:舊唐書/卷149|《旧唐书》卷一百四十九]]</ref>

蒋伸曾设宴，酌酒对客人们说：“有孝于家、忠于国者饮此。”客人们肃然，只有尚书左丞[[李景让|李景让]]起身喝完一爵。蒋伸说：“公不宜。”<ref name=NBT177>[[:s:新唐书/卷177|《新唐书》卷一百七十七]]</ref>

== 唐宣宗年间 ==
[[大中_(唐朝)|大中]]二年（848年），蒋伸被召回[[长安|长安]]为[[右补阙|右补阙]]、史馆修撰。他和兄蒋係的儿子[[蒋兆|蒋兆]]都以文辞中进士，但在文才方面并不被高看，而是被视为良史。转驾部郎中，知制诰。<ref name=NBT132/>五年（851年），宰相[[白敏中|白敏中]]受任充招讨[[党项|党项]]行营都统、制置等使、南北两路供军使兼[[邠宁节度使|邠宁节度使]]，获准选择廷臣作为自己的将佐，四月，辟蒋伸为右庶子，充邠宁节度副使。<ref name=ZZTJ249>[[:s:資治通鑑/卷249|《资治通鉴》卷二百四十九]]</ref>后回长安为[[户部侍郎|户部侍郎]]。九年（855年），为[[翰林学士|翰林学士]]，进承旨。大中十年（856年），改[[兵部侍郎|兵部侍郎]]，判户部。<ref name=NBT132/>

宣宗尊敬蒋伸的意见。<ref name=NBT132/>十一年（857年）十二月，蒋伸以翰林学士承旨、通议大夫、守尚书户部侍郎、知制诰、上护军、赐紫金鱼袋为兵部侍郎，充职。<ref>[[:s:旧唐书/卷18下|《旧唐书》卷一十八下]]</ref>十二年（858年）十一月有一次，宣宗单独会见身为比宰相低一级的“次对官”的蒋伸，蒋伸说：“现在取得官位容易了，人要心怀侥幸了。”宣宗惊答：“这样下去会乱的。”蒋伸答：“还不至于生乱，但如果太多人心怀侥幸，生乱就不难了。”蒋伸将要离开时，宣宗三次留住他，说：“以后就不能单独与卿相见了。”蒋伸没有明白宣宗的意思，但十二月，在另一宰相备选户部侍郎判度支[[杜胜|杜胜]]被宦官诋毁的情况下，宣宗将群臣的名字放在容器内，在[[唐宪宗|唐宪宗]]神位前祈祷射取，于是蒋伸得中，以兵部侍郎判户部被授[[同中书门下平章事|同中书门下平章事]]，为[[实质宰相|实质宰相]]。<ref name=ZZTJ249/><ref>[[:s:新唐书/卷008|《新唐书》卷八]]</ref><ref>[[:s:新唐书/卷169|《新唐书》卷一百六十九]]</ref><ref>《旧唐书·宣宗纪》《册府元龟》作十三年四月事。</ref>为相以后，他只能和其他宰相一起面圣，再不能单独见宣宗了。而名位素在其上的御史大夫李景让则不得拜相。<ref name=NBT177/>四个月后，以通议大夫、尚书兵部侍郎、同中书门下平章事、判户部事、护军、赐紫金鱼袋被解除户部职权，<ref>[[:s:罢蒋伸判户部制|:s:罢蒋伸判户部制]]</ref>加[[中书侍郎|中书侍郎]]。<ref name=NBT132/>

== 唐懿宗和唐僖宗年间 ==
大中十三年（859年）八月，唐宣宗崩，唐懿宗继位。<ref name=ZZTJ249/>蒋伸留任宰相，十月，兼工部尚书，官爵一度为[[金紫光禄大夫|金紫光禄大夫]]、守中书侍郎、兼工部尚书、同中书门下平章事、上护军、'''乐安县开国男'''，食邑三百户。<ref>[[:s:授白敏中宏文馆大学士等制|:s:授白敏中宏文馆大学士等制]]</ref>[[咸通|咸通]]二年（861年）二月兼[[刑部尚书|刑部尚书]]，监修国史。<ref name=NBT132/>九月罢相，<ref>《资治通鉴》《新唐书·懿宗纪》《新唐书·宰相表下》作三年正月事。此从《新唐书》本传。详见[[郁贤皓|郁贤皓]]《[[唐刺史考全编|唐刺史考全编]]》。</ref>充[[河中节度使|河中节度使]]，仍保有同中书门下平章事为荣衔。<ref name=ZZTJ250>[[:s:資治通鑑/卷250|《资治通鉴》卷二百五十]]</ref>后又以河中节度使、金紫光禄大夫、检校兵部尚书、平章事、河中尹、上护军、'''乐安郡开国公'''、食邑二千户迁检校工部尚书、平章事、充[[宣武军节度使|宣武军节度汴宋亳等州观察处置等使]]。<ref name=NBT132/><ref name=QTW83/>五年（864年）五月，因病由中书侍郎、兼户部尚书、平章事被免为[[太子少保|太子少保]]分司东都。<ref name=NBT132/><ref>[[:s:旧唐书/卷19上|《旧唐书》卷一十九上]]作十年（869年）正月事。此从《新唐书·宰相表下》。</ref><ref>当时并无皇太子，这完全只是荣衔。</ref>七年（866年），用为[[华州|华州]]刺史。九年（868年）尚在任，<ref>《唐故华州衙前兵马使魏公（虔威）志铭》</ref>后再迁[[太子太傅|太子太傅]]，<ref name=NBT132/>[[致仕|致仕]]。死后赠[[太尉|太尉]]。<ref>《全唐文》</ref>

咸通年间蒋伸与宰相[[韦保衡|韦保衡]]及[[皇甫焕|皇甫焕]]撰武宗、宣宗两朝实录，但因战乱，在[[后晋|后晋]][[天福_(后晋)|天福]]六年（941年）四月时已失传，宰相[[赵莹|赵莹]]请求下敕书求购。<ref>[[:s:旧五代史/卷79|《旧五代史》卷七十九]]</ref><ref>[[:s:论修唐史奏|赵莹《论修唐史奏》]]</ref>

== 轶闻 ==
《新唐书·[[杜悰|杜悰]]传》载宣宗崩后，[[枢密使|枢密使]][[王归长|王归长]]、[[马公儒|马公儒]]等以遗诏立夔王[[李滋|李滋]]，而左军中尉[[王宗实|王宗实]]等入殿中，以王归长等为矫诏，而迎立郓王[[李温|李温]]。久后，[[宣徽使|宣徽使]][[杨公庆|杨公庆]]奉命去[[中书省|中书省]]，只对宰相杜悰作揖，蒋伸和其他宰相[[毕諴|毕諴]]、[[杜审权|杜审权]]则不敢动作，避于西轩。杨公庆要杜悰弹劾不联名请郓王监国的大臣抵罪，杜悰不肯，杨公庆失望离去。蒋伸等三宰相再来见杜悰问此事，杜悰无言，三宰相惶怖，乞求保存家族，杜悰说：“别有其他疑虑。”并没转达杨公庆的意思。后来郓王继位为唐懿宗，怒气已解，众臣遂无事。<ref>[[:s:新唐书/卷166|《新唐书》卷一百六十六]]</ref>《资治通鉴》亦记为咸通二年（861年）二月事。<ref name=ZZTJ250/>

但[[胡三省|胡三省]]指出此事出自杜悰家传，并引[[洪迈|洪迈]]《随笔》指出懿宗即位之日宰相为[[令狐綯|令狐綯]]、[[萧邺|萧邺]]、[[夏侯孜|夏侯孜]]、蒋伸，毕諴、杜审权是懿宗登基后用为相者，可见此事是野史妄言。
