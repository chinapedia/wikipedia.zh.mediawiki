'''島津氏'''是日本的[[氏族|氏族]]之一。在[[鎌倉時代|鎌倉時代]]到[[江戶時代|江戶時代]]期間是[[大名_(稱謂)|大名]]，另外其家族亦有不少分支。

== 概要 ==
從[[守護|守護]]職成為[[戰國大名|戰國大名]]，也是[[江戶時代|江戶時代]]的[[外樣大名|外樣大名]]（[[薩摩藩|薩摩藩]]藩主）。家族全盛期時以[[薩摩國|薩摩國]]為中心統治[[南九州|南九州]]。初代當主[[島津忠久|島津忠久]]是[[薩摩國|薩摩國]]、[[大隅国|大隅国]]及[[日向国|日向国]]3国[[守護|守護]]、亦出任[[越前国|越前国]]守護。鎌倉時代起家族開始發展，在[[越前国|越前]]、[[信濃国|信濃]]、[[駿河國|駿河]]、[[若狹國|若狹]]及[[近江国|近江]]出現了島津氏的支流，他們被稱為[[越前島津氏|越前島津氏]]、[[信濃島津氏|信濃島津氏]]、[[河州島津氏|河州島津氏]]、[[若狹島津氏|若狹島津氏]]及[[江州島津氏|江州島津氏]]。

此外，條目以後所記述的戰國時代的宗家是一個庶流[[伊作家|伊作家]]（他們同樣是[[相州家|相州家]]的當主），他們成為了當主。江戶時代薩摩藩主的島津氏的伊作家（相州家）的系統。

== 出自：島津莊的莊官 ==
[[File:KoyaSatsumaShimazuKeNoHaka.jpg|thumb]]
島津氏的祖先是[[日本秦氏|秦氏]]子孫[[惟宗氏|惟宗氏]]後代。[[惟宗基言|惟宗基言]]之子[[惟宗廣言|惟宗廣言]]的主君是[[藤原氏|藤原]][[攝關家|攝關家]]筆頭[[近衛家|近衛家]]日向國島津莊（現[[宮崎縣|宮崎縣]][[都城市|都城市]]）的[[莊官|莊官]]（下司）前往九州、兒子[[島津忠久|惟宗忠久]]在[[源賴朝|源賴朝]]時期出任該地[[地頭|地頭]]時開始稱為島津氏，都城市被說是島津家的發源地。

=== 被源賴朝重用背景 ===
到底惟宗忠久（島津忠久）是否為惟宗廣言的兒子則無從考究。另外關於「[[攝津國|攝津]][[大阪|大阪]][[住吉大社|住吉大社]]境內誕下忠久的[[丹後局|丹後局]]是[[源賴朝|源賴朝]]側室，忠久是賴朝私生子」在『島津国史』及『島津氏正統系圖』等資料記亦有所記載，學會對於島津氏是賴朝的私生子的說法是否定的。<!-- されているとの説が一部にあるが、その説自体も根拠は定かでないため、出自の問題は島津氏研究には欠かせない部分となる。 →ノート参照 -->現在關於島津氏忠久時期以前家譜沒有明確的家譜。

同樣是九州[[守護|守護]]一職的[[大友能直|大友能直]]，他與島津忠久的共通點是後來被稱為九州一族的祖先，他們的出自均自不明，並傳出是「母親是賴朝的妻妾，受到賴朝的擁戴」。正確來說，忠久是源自惟宗氏，能直並非出自[[近藤氏|近藤氏]]。

== 南北朝時代 ==
[[鎌倉幕府|鎌倉時間]]開始，當主以幕府[[御家人|御家人]]身份在鎌倉居住，由守護職及地頭職等一族人士負責開發領土。[[島津久經|久經]]時期，[[元朝|元]]軍入侵日本，氏族的居地有向南遷的趨勢。

1333年（[[元弘|元弘]]3年）[[島津貞久|島津貞久]]參與由[[後醍醐天皇|後醍醐天皇]]發動的鎌倉幕府討幕運動。鎌倉幕府滅亡後後醍醐天皇開始實行[[建武新政|建武新政]]，後醍醐親政謀反的[[足利尊氏|足利尊氏]]在攝津國戰敗而逃往九州，與少貳氏共同協助足利尊氏，在筑前国的[[多多良濱之戰|多多良濱]](今福岡縣福岡市)與[[菊池氏|菊池氏]]率領後醍醐宮方軍交戰。

1342年，南朝的征西將軍[[懷良親王|懷良親王]]進入[[南九州|南九州]]，有一段時間成為了南朝派的人物。1340年貞久嫡男[[島津宗久|島津宗久]]過身後，與守護国的[[島津氏久|島津氏久]]（[[奧州家|奧州家]]）及[[島津師久|島津師久]]（[[總州家|總州家]]）兩家分裂。

== 戰國時代 ==
進入戰國時代後，與領土內各地的[[国人|国人]]及其他島津一族展開了鬥爭，宗家因而逐漸衰落。庶流[[伊作忠良|伊作忠良]]（[[伊作家|伊作島津家]]）勢力漸漸增大，壓倒了其他家系，長男[[島津貴久|島津貴久]]以宗家的養子成為島津家家督。貴久之後將家督讓給長男[[島津義久|島津義久]]，1578年[[耳川之戰|耳川之戰]]當中大敗[[大友氏|大友氏]]，加上1584年[[沖田畷之戰|沖田畷之戰]]重創[[龍造寺氏|龍造寺氏]]、島津氏成為了「三州的太守」。此外，當時義久還擁有三個優秀弟弟（[[島津義弘|島津義弘]]、[[島津歲久|歲久]]及[[島津家久|家久]]），因而被稱為戰國最強的薩摩武士，他們嘗試率領大軍統一九州南部三州及九州。1587年[[豐臣秀吉|秀吉]]發動[[九州征伐|九州戰役]]，島津氏向秀吉投降，最終保住了三州的領土。

== 江戶時期以後 ==
[[關原之戰|關原之戰]]爆發期間，島津家支持西軍，與德川家陷入敵對關係。戰後於義弘退位下，成功保住了原有領土。江戶時代早期島津家入侵[[琉球|琉球]]，佔領[[奄美諸島|奄美諸島]]，並支配琉球王朝。[[幕藩體制|幕藩體制]]之下在[[寶曆治水事件|寶曆治水]]當中是被幕府的弱體化政策受壓的一方。[[德川綱吉|德川綱吉]]養女[[淨岸院|竹姫]]為[[島津繼豐|島津繼豐]]的後妻之後，與[[德川氏|德川氏]]女兒進行婚姻，因而加深了與幕府的關係。幸運的是，他們的當家相當長命，並沒出現子嗣繼承問題需要幕府介入，因此與幕府的關係比較穩定。

踏入[[幕末|幕末]]，對抗西方帝國主義情緒高漲，第28代當家[[島津齊彬|島津齊彬]]時期組成了以西式製鐵、造船、紡織為中心的近代產業（[[集成館事業|集成館事業]]），開始出現了新的發展。另外幕府進入了崩壞期，薩摩藩內出現不少尊皇倒幕的人物，他們與長洲藩共同為主要倒幕力量。明治時代[[島津忠義|島津忠義]]本家與父親[[島津久光|島津久光]]在明治維新過後各自分成[[玉里家|玉里家]]共兩家，被授予[[公爵|公爵]]。其他有實力的分家包括了[[昭和天皇|昭和天皇]]第五皇女子[[島津貴子|清宮貴子內親王]]降嫁的日向佐土原島津家（幕末時是領有2万7千石[[伯爵|伯爵]]）。其餘的被任命為[[男爵|男爵]]。

[[明仁|今上天皇]]的外祖母是島津忠義七女[[邦彥王妃俔子|島津俔子]]，因此現時的島津氏一門與[[日本皇室|皇室]]有血緣關係。[[今上天皇|今上天皇]]的妹妹[[貴子內親王|貴子內親王]]嫁給[[島津久永|島津久永]]。兒子[[島津禎久|島津禎久]]島津家與皇室有著根深蒂固的親戚關係

== 正式簽名所見的「姓」 ==
官方文書簽名是鎌倉時代初期的「惟宗朝臣○○」，由德川家賜姓「[[松平氏|松平]]」名字改用「松平朝臣○○」而簽署。自此以後，幕府公式文書等以「松平<u>薩摩守</u>（有變化）○○」作書寫。此外江戶時代中期以後、内部公式文書等就以「源朝臣○○」作簽名<ref>不過「源」的簽名使用方式是[[室町時代|室町時代]]中期左是右藤原姓與平行相等。</ref>。
