'''島式月台'''，又名'''中置式月台'''、'''中央月台'''、'''中间站台'''<ref>{{cite web|title=课程学习>>第五章 第二节 客运站的站场设施|url=http://zy.swust.net.cn/15/1/lkyszz/kcxx/chapter5/0502.html}}</ref>，是[[鐵路|鐵路]]月台的一種型態，為[[路軌|路軌]]在兩旁，[[車站月台|月台]]被夾在中間的設計。

== 優點 ==
相較於[[側式月台|側式月台]]，島式月台擁有以下之優點：
# 與[[月台|月台]]相關之設備（例如[[升降機|升降機]]、[[電動扶梯|電動扶梯]]等等）只需購置一組，可降低投資及營運成本。
# 較易於監控列车运作及乘客安全。
# 可以衍生出[[跨月台轉車站|同月台平行轉乘]]的設計，而大幅節省[[通勤|通勤]]時換車旅客的轉乘時間和徒步距離、提升系統運作的效率。
# 旅客若搭錯路線或方向時較易於換線返回。
# 若兩個月台目的地相同，旅客可在月台層選乘較早開出的列車，不需在大堂乘電梯前預先選定。

== 缺點 ==
#若設在平面，月台與車站出入口被軌道分離，不論上下行乘客皆必須透過天橋或地下道才能安全抵達月台。
#需要更長的站體容納月台與軌道，因此設在馬路正上方的高架車站不常採用此設計。
#軌道在車站內部與兩側必須分開，增加建設成本。
#如客流较大或者同台换乘站两侧列车同时到达时，人群易发生对冲情况。
#由於月台被軌道包夾，故未來運量增加時進行擴建工程較為困難。

== 使用 ==
此種月台較常使用於如下用途：
# 雙線鐵路中較小且高架或地下化的車站
# [[捷運系統|捷運系統]]（可分別方向、作為終點站緩衝）
