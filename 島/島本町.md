'''島本町'''（{{jpn|j=島本町|hg=しまもとちょう|rm=Shimamoto chō|f=y}}）是位於日本[[大阪府|大阪府]]東北部的行政區劃，屬[[三島郡_(大阪府)|三島郡]]。轄區主要位於淀川支流{{Tsl|ja|水無瀬川|水無瀨川}}的流域，轄內地形以丘陵地和山區為主，僅東南部靠淀川旁為平原，主要市區野味於此區；該區域北與[[京都府|京都府]][[大山崎町|大山崎町]]相鄰，為淀川眾多支流[[桂川_(淀川水系)|桂川]]、宇治川、{{Tsl|ja|木津川 (大阪府)|木津川 (大阪府)|木津川}}匯流處，也是連結[[京都盆地|京都盆地]]與[[近畿平原|近畿平原]]的交通要道，鐵路[[東海道新幹線|東海道新幹線]]、[[東海道本線|東海道本線]]、[[京都本線|京都本線]]及[[名神高速道路|名神高速道路]]都從此通過，因此目前也同時成為[[大阪市|大阪市]]與[[京都市|京都市]]外圍的[[卧城|卧城]]。<ref>{{Cite web| url=http://www.shimamotocho.jp/kankou_bunka/syoukai/matinogaiyou/1237902677659.html| title=町の概要| date=2015-03-17| publisher=島本町公所| accessdate=2018-09-25| language=ja}}</ref>

== 歷史 ==

過去在令制國時代屬於[[攝津國|攝津國]][[島上郡|島上郡]]，由於位於攝津國與[[山城國|山城國]]之間唯一不需穿越山區的通道上，自古即成為交通要道，也因此帶動此地的發展。[[東大寺|東大寺]]曾在此設有莊園，12世紀[[後鳥羽天皇|後鳥羽天皇]]也曾在此設立{{Tsl|ja|水無瀬離宮|水無瀨離宮}}<ref>{{Cite web| url=http://www.shimamotocho.jp/kankou_bunka/kankou_shizen_rekishi/1534226626435.html| title=島本町の地理的・歴史的環境| date=2018-08-16| publisher=島本町公所| accessdate=2018-09-25| language=ja}}</ref>；16世紀末[[豐臣秀吉|豐臣秀吉]]與[[明智光秀|明智光秀]]決戰的[[山崎之戰|山崎之戰]]即發生於本地北側的山崎地區。

[[江戶時代|江戶時代]]東南部的平原區域被分割為{{Tsl|ja|加納藩}}、[[公家|公家]][[水無瀨家|水無瀨家]]、[[離宮八幡宮|離宮八幡宮]]的領地，而西北部山區則屬於{{Tsl|ja|高槻藩}}的領地。19世紀[[明治維新|明治維新]]後，這修區域曾先後隸屬[[京都府|京都府]]、攝津縣、[[豐崎縣|豐崎縣]]、[[兵庫縣|兵庫縣]]、高槻縣、加納縣，直到1871年的第一次府縣整併中，除了山崎地區以外全數被劃入[[大阪府|大阪府]]，而山崎地區也在1873年由京都府改劃入大阪府。

1889年實施[[町村制|町村制]]後，此地設立了'''島本村'''，島本村在1940年後升格改制為'''島本町'''，至今轄區都未曾再有過變化。

== 交通 ==

由於東南部市區位於大阪與京都之間的主要通道上，鐵路[[東海道新幹線|東海道新幹線]]、[[東海道本線|東海道本線]]、[[京都本線|京都本線]]及[[名神高速道路|名神高速道路]]都從此通過，轄內主要車站是[[阪急電鐵|阪急電鐵]][[京都本線|京都本線]]的{{T|ja|水無瀨車站}}，而位於北側[[大山崎町|大山崎町]]境內的{{Tsl|ja|島本駅|島本車站}}也僅距離島本町山崎地區100公尺。

== 觀光資源 ==

[[File:Minasejingu03n2048.jpg|thumb]]

建於{{Tsl|ja|水無瀬離宮|水無瀨離宮}}原址的[[水無瀨神宮|水無瀨神宮]]是建於13世紀，主要祭祀後[[鳥羽天皇|鳥羽天皇]]、[[土御門天皇|土御門天皇]]、[[順德天皇|順德天皇]]，其境內的[[泉|湧泉]]「離宮之水」被列為日本[[名水百選|名水百選]]，甚至現在也是島本町自來水主要的水源。
