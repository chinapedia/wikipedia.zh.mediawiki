'''乔琳'''（{{bd|714年|8月21日|784年|7月28日|catIdx=Q}}），[[中国|中国]][[唐朝|唐朝]]官员，[[唐德宗|唐德宗]]初年短暂为[[宰相|宰相]]。他后来效力叛军首领[[朱泚|朱泚]]，唐军消灭朱泚政权后，他虽然年老，也被处决了。

== 家世和早期仕途 ==
乔琳生于[[唐玄宗|唐玄宗]][[开元|开元]]二年（714年），[[太原|太原]]人氏。他少年孤贫，但好学，以文词闻名。天宝元年（742年），中[[进士|进士]]，补[[成武|成武]]尉，后授[[兴平|兴平]]尉。大将[[朔方节度使|朔方节度使]][[郭子仪|郭子仪]]辟他为掌书记，不久又拜[[监察御史|监察御史]]。当时他健谈诙谐，常戏谑同僚。同院御史[[毕耀|毕耀]]和他互相嘲讽，发展为频繁争吵，最终都弹劾对方有错。乔琳被贬为[[巴州_(四川)|巴州]]员外司户。<ref name=BT127>[http://www.sidneyluo.net/a/a16/127.htm 《旧唐书》卷一百二十七] {{webarchive|url=https://web.archive.org/web/20131020062925/http://www.sidneyluo.net/a/a16/127.htm |date=2013-10-20 }}</ref><ref name=NBT224-2>[http://www.sidneyluo.net/a/a17/224b.htm 《新唐书》卷二百二十四下] {{webarchive|url=https://web.archive.org/web/20131020062923/http://www.sidneyluo.net/a/a17/224b.htm |date=2013-10-20 }}</ref>

乔琳后又被起用为[[南郭|南郭]]令，后又任[[山南西道节度使|山南西道节度使]][[张献诚|张献诚]]行军司马。<ref>鉴于原为叛军政权[[燕_(安史之乱)|大燕]]将领的张献诚于[[宝应|宝应]]元年（762年）降唐，随即任山南西道节度使，于大历二年（767年）离职，乔琳效力他肯定在此期间。见[http://www.sidneyluo.net/a/a16/122.htm 《旧唐书》卷一百二十二] {{webarchive|url=https://web.archive.org/web/20080621214340/http://www.sidneyluo.net/a/a16/122.htm |date=2008-06-21 }}和[[:s:資治通鑑/卷222|《资治通鉴》卷二百二十二]]。</ref>张献诚任满后，乔琳成为剑南[[东川节度使|东川节度使]][[鲜于叔明|鲜于叔明]]的判官，又改检校驾部郎中、[[果州|果州]]、[[绵州|绵州]]、[[遂州|遂州]]三州刺史，兼[[御史中丞|御史中丞]]。在绵州刺史任上曾惊于僧人神清年轻俊秀，亲自降礼请求为他削发。<ref>《宋高僧传》卷六《唐梓州慧义寺神清传》</ref>作风宽简，不亲自做事。他曾经对录事参军[[任绍业|任绍业]]说：“你能弹劾刺史吗？”任绍业一条条列出乔琳的过失给乔琳看，乔琳惊呼：“能知道我的过失，这是御史之材啊。”后又被召回朝中任大理少卿、[[国子祭酒|国子祭酒]]，约十二年（777年）又出为[[怀州|怀州]]刺史。<ref name=BT127/><ref name=NBT224-2/><ref>[[郁贤皓|郁贤皓]]《[[唐刺史考全编|唐刺史考全编]]》</ref>

== 唐德宗年间 ==
[[大历|大历]]十四年（779年），[[唐德宗|唐德宗]]登基，侍读国子博士[[张涉|张涉]]入翰林，迁散骑常侍，成为其一位有影响的顾问。<ref>[[:s:資治通鑑/卷225|《资治通鉴》卷二百二十五]]</ref>乔琳和张涉交好，德宗要张涉推荐宰相人选时，尽管乔琳能力未显，张涉还是推荐乔琳有见识度量，可堪大用。八月，德宗相信张涉所荐，任乔琳为[[御史大夫|御史大夫]]、京畿观察使，授[[同中书门下平章事|同中书门下平章事]]，为实质宰相，听闻此事的人都震惊了。乔琳衰老，听话困难，德宗咨询他，他几乎不能妥善对答，谋事也多疏漏，论事也不合时宜，只当了不到90天宰相，就在十一月被罢相改任[[工部尚书|工部尚书]]，<ref>[[:s:新唐书/卷7|《新唐书》卷七]]</ref>由[[司空|司空]][[崔宁_(唐朝)|崔宁]]代为御史大夫和宰相，<ref>[[:s:旧唐书/卷117|《旧唐书》卷一百一十七]]</ref>而另一宰相[[崔祐甫|崔祐甫]]因患病不理政事，于是另一宰相[[杨炎|杨炎]]因而得以专国政。<ref>[[:s:旧唐书/卷118|《旧唐书》卷一百一十八]]</ref><ref>[[:s:新唐书/卷145|《新唐书》卷一百四十五]]</ref>德宗也因此疏远了张涉。<ref name=NBT224-2/>[[建中_(唐朝)|建中]]元年（780年）十月，德宗以弟睦王[[李述_(唐)|李述]]为奉迎使，乔琳为副使，在全国范围内搜寻在[[安史之乱|安史之乱]]中失踪的母后[[睿真皇太后|沈太后]]。<ref name=BT12>[[:s:旧唐书/卷12|《旧唐书》卷一十二]]</ref><ref>[[:s:旧唐书/卷116|《旧唐书》卷一百一十六]]</ref><ref>[[:s:新唐书/卷82|《新唐书》卷八十二]]</ref><ref>[[:s:資治通鑑/卷226|《资治通鉴》卷二百二十六]]</ref>沈太后最终未能找到。<ref>[http://www.sidneyluo.net/a/a16/052.htm 《旧唐书》卷五十二] {{webarchive|url=https://web.archive.org/web/20080921035212/http://www.sidneyluo.net/a/a16/052.htm |date=2008-09-21 }}</ref><ref>[[:s:新唐书/卷77|《新唐书》卷七十七]]</ref>

建中四年（783年）七月，乔琳与群臣与[[吐蕃|吐蕃]]宰相[[区颊赞|区颊赞]]等会盟于丰邑里坛所。<ref>[[:s:旧唐书/卷196下|《旧唐书》卷一百九十六下]]</ref><ref>[[:s:新唐书/卷216下|《新唐书》卷二百一十六下]]</ref>

同年十月，在[[长安|长安]]等候被德宗派遣东进讨伐叛乱军阀[[淮西节度使|淮西节度使]][[李希烈|李希烈]]的泾原兵因未得德宗足够的赏赐，愤而兵变。德宗逃到[[奉天|奉天]]，乔琳也随驾，转[[吏部尚书|吏部尚书]]，迁[[太子少师|太子少师]]、<ref name=BT127/><ref>[[:s:資治通鑑/卷228|《资治通鉴》卷二百二十八]]</ref>[[太子太师|太子太师]]。<ref>《[[册府元龟|册府元龟]]》</ref>变军推将军[[太尉|太尉]][[朱泚|朱泚]]为首，朱泚不久自建秦政权，自称皇帝。但秦军不能在奉天俘虏德宗，唐、秦两军僵持。<ref>[[:s:資治通鑑/卷229|《资治通鉴》卷二百二十九]]</ref>[[兴元|兴元]]元年（784年）二月，唐军大将太尉[[李怀光|李怀光]]因德宗不见他而对德宗生怨，转而和朱泚结盟对抗德宗，德宗被迫远遁[[梁州|梁州]]。<ref>[[:s:資治通鑑/卷230|《资治通鉴》卷二百三十]]</ref>乔琳起初随驾，行至[[盩厔|盩厔]]，称年老且没有马，尽管德宗素来因他是旧臣且年老而以礼相待，给他一匹御马，他还是拒绝随驾。德宗怅然，将手中的马鞭赐给他，说：“好自为之，和卿决别了。”数日后，乔琳剃发在[[仙游寺|仙游寺]]为僧。朱泚闻讯，派数十骑迎乔琳回京，任为吏部尚书。乔琳起初拒绝，但最终接受。朱泚派自己所任的[[中书侍郎|中书侍郎]]、同平章事、判度支[[源休|源休]]给乔琳穿朝服、吃肉，他也不拒绝。但他显然对此事感情复杂，他为朱泚挑选的官员之一抱怨自己得到的官位不稳妥，乔琳模棱两可地答道：“足下觉得这次选官能稳妥吗？”<ref name=BT127/><ref name=NBT224-2/>

同年，另一唐军大将鄜坊、京畿、渭北、商华副元帅[[李晟|李晟]]灭朱泚政权（已改国号汉），收复长安，迎归德宗。包括乔琳在内的一批汉人官员都被认为将处极刑。李晟因乔琳年老而怜悯，上表为他请求宽恕。德宗恨乔琳变节，拒绝了，下令斩之。七月七日（7月28日），乔琳和两位汉人官员伪宰相[[蒋镇|蒋镇]]和伪节度使兼宰相[[张光晟|张光晟]]一同被李晟所诛。<ref name=BT12/><ref>[[:s:旧唐书/卷133|《旧唐书》卷一百三十三]]</ref><ref>[[:s:資治通鑑/卷231|《资治通鉴》卷二百三十一]]</ref>乔琳被杀前叹道：“我乔琳七月七日生，又在七月七日死。这不是我的命吗？”<ref name=BT127/><ref name=NBT224-2/>

== 评价 ==
*《旧唐书》史臣曰：乔琳巧辞真主，俯就伪官。<ref name=BT127/>
*[[洪迈|洪迈]]《[[容斋随笔|容斋随笔]]》卷九：[[安禄山|安禄山]]、朱泚之变，[[陈希烈|陈希烈]]、[[张均|张均]]、[[张垍|张垍]]、'''乔琳'''、[[李忠臣|李忠臣]]，皆以宰相世臣，为之丞弼。而[[甄济|甄济]]、[[权皋|权皋]]、[[刘海宾|刘海宾]]、[[段秀实|段秀实]]，或以幕府小吏，或以废斥列卿，捐身立节，名震海内。人之贤不肖，相去何止天冠地履乎！

== 轶闻 ==
[[:s:太平广记/卷第150|《太平广记》卷第一百五十<定数>]]引《[[前定录|前定录]]》：乔琳天宝元年从太原出发去科举，在[[大梁|大梁]]住宿，天寒，暴雪，乔琳的马死了，仆从也散去。乔琳听说[[浚仪|浚仪]]尉刘彦庄喜欢结交朋友，便前往求助。刘彦庄的朋友申屠生精通相术，他从不对刘彦庄的朋友宾客中的达官名流礼貌谦虚，对乔琳却非常亲热，刘彦庄感到很奇怪。乔琳出去时，刘彦庄对申屠生说：“我的宾客中，无论是有无才能的人，都未曾见过你同他们说一句话。乔琳只是个布衣平民，你为何对他异常客气？”申屠生笑着回答：“此人哪里是个平常的人呀！日后会成为你的上司。你应该好好地对待他，日后必能得到他的报答。我同他结交，也完全是为了你。可惜观察他的面相，有反叛的征兆。如做高官，不会超过百日，并且过了七十岁，必然死于非命。你记住我今天说过的话。”刘彦庄于是款待乔琳数日，并赠送车马，使其安全到达长安。申屠生也辞别刘彦庄，说自己得到刘彦庄的恩惠已经报答，从此不知其去向。大历中期，乔琳被任命为怀州刺史，刘彦庄正在怀州所辖的修武任县令，因断案误判致人屈死被家属上诉，乔琳到任，使其免罪。乔琳拜相只有八十七天，就因病辞官，后因追随朱泚而伏诛，死时七十一岁（正史只记载乔琳卒年七十余岁）。

[[:s:太平广记/卷第255|《太平广记》卷第二百五十五<嘲诮三>]]引《[[大唐新语|大唐新语]]》：朱泚作乱时，源休对朱泚任为[[黄门侍郎|黄门侍郎]]的[[蒋链|蒋链]]态度谦恭地说：“若衡量才能，那么我就是[[萧何|萧何]]，[[姚令言|姚令言]]就是[[曹参|曹参]]。”有识者听说后，都知道他不满自己的官职。乔琳喜好戏谑，于是对旧僚们说：“源公简直可以说是用火来逼迫酂侯（即萧何）了！”
