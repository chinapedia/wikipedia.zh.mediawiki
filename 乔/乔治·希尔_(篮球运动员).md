'''小喬治·杰西·希爾'''（{{lang-en|'''George Jesse Hill, Jr.'''}}，{{bd|1986年|5月4日}}），[[美國|美國]]職業[[籃球|籃球]]運動員，現效力於[[密爾瓦基公鹿|密爾瓦基公鹿]]，司職[[控球後衛|控球後衛]]、[[得分後衛|得分後衛]]。

== 早年 ==
喬治·希爾在[[印第安納州|印第安納州]]的[[印第安納波利斯|印第安納波利斯]]出生長大，他從小崇拜著像[[麥可·喬丹|麥可·喬丹]]、[[魔術強森|魔術強森]]和[[拉里·伯德|拉里·伯德]]這樣的球員，並夢想著有一天能在[[NBA|NBA]]打球。希爾高中就讀家鄉本地的{{link-en|Broad Ripple High School|Broad Ripple High School}}，在那裡他成為印第安納州頂尖高中生之一，與[[格雷格·奥登|格雷格·歐登]]、[[邁克·康利|邁克·康利]]、[[埃里克·戈登|埃里克·戈登]]、[[考特尼·李|考特尼·李]]、[[羅德尼·卡尼|羅德尼·卡尼]]及[[喬什·麥克羅伯茨|喬什·麥克羅伯茨]]等人被稱為當地的籃球七傑。<ref>{{Cite web |url=http://iw.newsbank.com/iw-search/we/InfoWeb?p_action=doc&p_theme=aggdocs&p_topdoc=1&p_docnum=1&p_sort=YMD_date:D&p_product=AWNB&p_docid=1241CDA351F36DB0&p_text_direct-0=document_id=(%201241CDA351F36DB0%20)&p_nbid=C5FQ62WWMTIyNjE2MDY0OC4zOTgwNzk6MToxMToxMjkuMTE1LjAuMA |title=存档副本 |access-date=2018-07-10 |archive-url=https://web.archive.org/web/20100809145945/http://iw.newsbank.com/iw-search/we/InfoWeb?p_action=doc |archive-date=2010-08-09 |dead-url=yes }}</ref>

希爾在高三那一年，場均可以得到36.2分，這是印第安納州高中體育協會歷史上第五高的單賽季平均水平。<ref>{{Cite web |url=http://www.iupuijags.com/roster.aspx?rp_id=7 |title=存档副本 |access-date=2008-07-30 |archive-url=https://www.webcitation.org/5ZgzSHEA1?url=http://www.iupuijags.com/roster.aspx?rp_id=7 |archive-date=2008-07-30 |dead-url= }}</ref>

== 大學生涯 ==
希爾高中畢業時得到來自[[天普大學|天普大學]]和[[印第安納大學|印第安納大學]]在內的幾所大學提供獎學金，但他最後選擇了[[印第安納大學與普渡大學印第安納波利斯聯合分校|IUPUI]]，目的是希望留在他生病的曾祖父Gilbert Edison身邊。然而就在他選擇就讀IUPUI幾個月後，他的曾祖父便去世了，從來沒有機會看到希爾在大學裡打球。希爾也遵循曾祖父的教導“永遠做一個忠誠的人”，所以希爾仍然拒絕了其他優惠的提案並留在IUPUI。<ref>{{Cite web |url=http://www.indystar.com/apps/pbcs.dll/article?AID=%2F20080626%2FSPORTS%2F806260437%2F1088%2FSPORTS04 |title=存档副本 |access-date=2008-07-30 |archive-url=https://www.webcitation.org/5Zh0E10IX?url=http://www.indystar.com/apps/pbcs.dll/article?AID=%2F20080626%2FSPORTS%2F806260437%2F1088%2FSPORTS04 |archive-date=2008-07-30 |dead-url= }}</ref>

希爾在他大二那一年，帶領IUPUI美洲虎隊取得了校史最佳戰績。大三那年由於受傷迫使希爾暫停參與比賽，希爾在下個賽季回歸並再次帶領美洲虎隊取得了創紀錄的26勝，並打進山頂聯盟錦標賽決賽但輸給了當屆冠軍{{link-en|Oral Roberts University|奧拉爾羅伯茨大學}}，因此失去了晉級[[NCAA|NCAA]]錦標賽的機會。若不計算希爾在2006-07賽季的傷病缺賽，在IUPUI的三個賽季中，美洲虎隊的戰績為61勝30負。

喬治·希爾最終以總得分1619分排名[[印第安納大學與普渡大學印第安納波利斯聯合分校|IUPUI]]校史第五，同時成為IUPUI第一個提前投入選秀的球員和山頂聯盟第一個在NBA選秀首輪就被選中的球員。

==NBA生涯==
===NBA選秀===
喬治·希爾雖然還有一年打NCAA的資格，但他決定提前投入[[2008年NBA選秀|2008年NBA選秀]]，但起初希爾選秀的預測順位並不理想，而他也表示有心理準備回大學打完最後一年，在選秀會前幾個月希爾參加了多個球隊的試訓，在奧蘭多的訓練營中希爾留下令人印象深刻的表現，也開始受到關注並被認為有機會在第二輪中被選中，最終在選秀會中[[聖安東尼奧馬刺|聖安東尼奧馬刺]]打破外界預想在首輪第26順位選中了喬治·希爾而沒有選擇[[堪薩斯大學|堪薩斯大學]]的[[馬里奧·查爾莫斯|馬里奧·查爾莫斯]]。<ref>https://www.webcitation.org/5ZamwW55P?url=http://www.iupuijags.com/news/2008/6/26/MBB_0626085013.aspx</ref>

===聖安東尼奧馬刺(2008-2011)===
[[2008-2009_NBA賽季|08-09賽季]]
2008年9月23日，馬刺隊與希爾簽下了一份標準新秀合同，該合同保證了前兩年，並在接下來的兩年中包括一個球隊選項。希爾因手指受傷錯過了賽季前兩場比賽，在11月4日與[[達拉斯小牛隊|達拉斯小牛隊]]的賽事中首次登場，並且得到11分1次抄截和1次助攻。原先希爾主要擔任[[東尼·帕克|東尼·帕克]]的替補，而因為帕克季初受傷，而得到先發的機會。在帕克和[[馬努·吉諾比利|馬努·吉諾比利]]歸隊後希爾的上場時間也相對減少，最後以場均5.7分、2.1個籃板和1.8次助攻結束了新秀球季。

[[2009-2010_NBA賽季|09-10賽季]]
在休賽季裡乔治·希尔致力於訓練控球技巧與強化三分投射能力，這些努力也在新賽季裡得到巨大的回報，在東尼·帕克受傷病困擾期間希爾做為球隊先發主控出賽了43場比賽，上場時間從16.5分鐘增加到29分鐘，場均得分也增加了一倍多，投籃命中率提高了7.5個百分點，三分命中率提高了7.9個百分點，賽季結束後希爾在[[NBA最佳進步球員|NBA最佳進步球員]]的評選中得到了第二名。

2010年4月1日對陣[[休斯敦火箭隊|休斯敦火箭隊]]的主場比賽中希爾取得了職業生涯新高的30分、5次抄截和7次助攻。<ref>https://www.mysanantonio.com/sports/spurs/article/Hill-leads-Spurs-past-Rockets-787228.php?c=y&page=2#storytop</ref>

希爾在季後賽首輪面對達拉斯小牛的系列賽裡稱職扮演先發控球，場均繳出14.3分、3.8籃板、1.1抄截，且在系列賽第四戰，馬刺GDP連線合計僅34投9中的戰況中拿下個人季後賽新高的29分帶領球隊取勝，幫助馬刺以總比分4：2淘汰小牛，但在西區半決賽裡遭到[[鳳凰城太陽|鳳凰城太陽]]橫掃出局。

[[2010-11_NBA賽季|10-11賽季]]
在東尼·帕克健康無虞的情況下，希爾重新回到替補後衛的定位，但希爾的各項數據沒有呈現下滑的跡象，甚至以場均11.6分為球隊提供穩定的板凳火力，幫助球隊成為西區第一種子，但卻在季後賽首輪遭到灰熊隊上演[[老八傳奇|老八傳奇]]。<REF>https://tw.appledaily.com/sports/daily/20110501/33356376</REF>

2011年3月28日，對陣[[曼菲斯灰熊隊|曼菲斯灰熊隊]]的比賽中希爾板凳出賽32分鐘，投籃12投9中，得到平個人生涯最高得分的30分。<REF>http://www.nba.com/games/20110327/SASMEM/gameinfo.html</REF>

2011年6月24日，[[2011年NBA選秀|2011年NBA選秀]]當天希爾被馬刺球團交易到[[印第安納溜馬|印第安納溜馬]]換取首輪第15順位的新秀[[科懷·雷納德|科懷·雷納德]]。<REF>http://www.espn.com/nba/draft2011/news/story?id=6698432</REF>

===印第安納溜馬(2011-2016)===
[[2011-12_NBA賽季|11-12賽季]]
希爾在溜馬的第一個賽季主要擔任[[达伦·科里森|达伦·科里森]]的替補，季中希爾因腳踝傷是缺席了12場比賽，季末科里森因傷缺賽希爾則成為球隊先發直到賽後賽終了，即使科里森傷癒回歸也未動搖希爾的先發位置。希爾本賽季場均得分9.6分，投籃命中率為44.2%。

季後賽首輪面對[[奧蘭多魔術|奧蘭多魔術]]的系列賽中，希爾五場比賽皆得分上雙，幫助球隊晉級，在東區半決賽面對[[邁阿密熱火|邁阿密熱火]]的系列賽裡，在第三戰希爾取得20分和5助攻幫助溜馬以總比分2：1領先，但那之後熱火連贏三場將溜馬淘汰。

[[2012-13_NBA賽季|12-13賽季]]
希爾與溜馬隊簽定了一份為期五年，價值4000萬美元的合約。<ref>http://www.insidehoops.com/blog/?p=10496</ref>
本賽季可以說是希爾做為先發主控的第一個完整賽季，希爾出賽76場全都是先發出賽，最終繳出14.2分、4.7助攻和3.7籃板成為溜馬隊陣中的第三得分手，僅次於[[保羅·喬治|保羅·喬治]]和[[大衛·韋斯特|大衛·韋斯特]]，並幫助球隊打入東區冠軍賽最終敗給了衛冕冠軍[[邁阿密熱火|邁阿密熱火]]。

[[2013-14_NBA賽季|13-14賽季]]
本賽季在兩名隊友保羅·喬治、[[蘭斯·史蒂芬森|蘭斯·史蒂芬森]]表現大躍進要的狀況下，希爾的場上表現與球權受到壓縮使得各項數據略有下滑。而在季後賽裡溜馬隊則是連續兩年在分區決賽敗給了熱火隊。

希爾在2014年2月7日對主場對陣[[波特蘭拓荒者|波特蘭拓荒者]]的比賽打出生涯最佳表現，包括生涯新高的單場37分外帶9籃板、8助攻和2抄截，以及正規時間終了前一記追平比數的三分球。<ref>https://www.indystar.com/story/sports/nba/pacers/2014/02/07/george-hill-scores-37-as-pacers-win-118-113-in-overtime/5300205/</ref>

[[2014-15_NBA賽季|14-15賽季]]
開季受到左膝傷勢困擾錯過了25場的賽事，希爾在2014年12月23日對陣[[紐澳良鵜鶘|紐澳良鵜鶘]]的比賽復出，替補出賽21分鐘取得15分、4籃板、3助攻和1抄截，幫助球隊取勝<ref>http://www.nba.com/games/20141223/NOPIND/gameinfo.html</ref>。在陣中主將保羅·喬治因傷幾乎整季缺賽的情況下，希爾本季出賽43場繳出16.1分、5.1助攻和4.2籃板，且場均得分與助攻皆領先全隊，同時在希爾出場的賽事裡溜馬隊取得了26勝17敗，可惜溜馬本季最終以38勝44敗的戰績無緣季後賽，這也是喬治·希爾生涯首次缺席季後賽。

2015年2月27日，主場對陣[[克里夫蘭騎士|克里夫蘭騎士]]的比賽中，希爾獲得15分、10籃板和12助攻達成了生涯首次的[[三双|大三元]]並帶領球隊獲勝。<ref>http://sports.sina.com.cn/nba/2015-03-01/12417528442.shtml</ref>
<ref>http://www.nba.com/games/20150227/CLEIND/gameinfo.html#nbaGIboxscore</ref>

[[2015-16_NBA賽季|15-16賽季]]
在球隊一哥保羅·喬治持續成長加上與新隊友[[蒙塔·艾利斯|蒙塔·艾利斯]]的角色重疊，身為主控的希爾反成第三持球者，這使得希爾的發揮受到限制，比賽表現顯得平淡，本季各項數據皆呈現小幅下滑，季後賽首輪與東區第二種子[[多倫多暴龍|多倫多暴龍]]纏鬥七場後遭淘汰。

賽季結束後，希爾被溜馬隊交易到了[[猶他爵士|猶他爵士]]<ref>https://nba.udn.com/nba/story/6780/1780639</ref>，這也讓身為家鄉球員的希爾感到意外。<ref>https://sports.ettoday.net/news/739647</ref>

===猶他爵士(2016-2017)===

[[2016-17_NBA賽季|16-17賽季]]
希爾來到[[猶他爵士|猶他爵士]]在場上獲得了很大的發揮空間，雖然因為傷勢困擾希爾本季只出賽49場<ref>http://sports.ltn.com.tw/news/breakingnews/1905343</ref><ref>http://sports.ltn.com.tw/news/breakingnews/1941396</ref>也不影響他亮眼的表現，且在10月31日至11月6日當周，以20.4分、5.5助攻帶領爵士取得3勝1敗，生涯首次獲選為NBA西區單周最佳球員<ref>http://www.nba.com/article/2016/11/07/players-week-lebron-james-george-hill</ref>，最終賽季繳出場均16.9分、4.2助攻、3.4籃板以及投籃命中率4成7、三分命中率4成03，各項數據皆可算是個人生涯最佳水準的表現，也幫助球隊取得51勝31敗的戰績，以西區第五種子躋身季後賽，這是爵士隊睽違三季首度重返季後賽，在首輪系列賽中希爾為球隊擊敗[[洛杉磯快艇|洛杉磯快艇]]的7場比賽中<ref>http://www.bcc.com.tw/newsView.3028649</ref>，貢獻了16.8分、3.7助攻、4.1籃板的穩定表現，但在西區半決賽對決[[金州勇士|金州勇士]]的系列賽第一戰中受傷後便沒有再出賽<ref>http://sports.ltn.com.tw/news/breakingnews/2057509</ref>，爵士也遭到勇士橫掃出局。<ref>https://nba.udn.com/nba/story/6780/2451558</ref>

2017年3月3日，希爾以賽季新高34分，帶領球隊在主場以 112-97 擊敗[[布魯克林籃網|布魯克林籃網]]。<ref>http://www.espn.com/nba/recap?gameId=400900310</ref>

賽季結束後，喬治·希爾成為自由球員，即便賽季出賽率不高，但其出色的表現依舊引發不少球隊的興趣，最終在2017年7月11日正式以三年5700萬美元加盟[[沙加緬度國王|沙加緬度國王]]。<REF>https://nba.udn.com/nba/story/6780/2576067</REF>

===沙加緬度國王(2017-2018)===

[[2017-18_NBA賽季|17-18賽季]]
希爾加盟重建中的國王隊，並沒有成為球隊的核心，在球團方針主力培養新秀[[德阿隆·福克斯|德阿隆·福克斯]]的原因，希爾多以2號出賽搭配福克斯，而在此希爾的場均被壓縮到26.6分鐘是新秀賽季以來的新低，且由於球隊戰績不佳、希爾發揮有限、新秀福克斯表現出色等原因，希爾對於球隊的重要性不在，最終在季中希爾被交易到了[[克里夫蘭騎士|克里夫蘭騎士]]。<ref>https://nba.udn.com/nba/story/6780/2977241</ref>

===克里夫蘭騎士(2018)===

騎士球團相中希爾的防守、三分能力及豐富的季後賽經驗，引進希爾期望他能成為球隊奪冠的重要拼圖，但希爾來到克里夫蘭表現一般，例行賽出賽24場表現與在國王隊時沒有太大的差別，而在季後賽則隨隊一路晉級至[[NBA總決賽|NBA總決賽]]最終敗給了[[金州勇士|金州勇士]]。
===密爾瓦基公鹿(2018～)===
[[克里夫蘭骑士|克里夫蘭骑士]]将[[喬治·希爾|喬治·希爾]]、[[山姆·德克爾|山姆·德克爾]]送到[[密爾瓦基公鹿|密爾瓦基公鹿]]，换回了[[馬修·德拉維多瓦|馬修·德拉維多瓦]]、[[約翰·亨森|約翰·亨森]]、2021一个首輪選秀籤以及一个次輪選秀籤
