'''乔莫·肯雅塔国际机场'''（'''Jomo Kenyatta International Airport'''）原名'''恩姆巴卡西机场'''（'''Embakasi Airport'''）、'''内罗毕国际机场'''（'''Nairobi International Airport'''），位于[[肯尼亚|肯尼亚]]首都[[内罗毕|内罗毕]]，是该国最大的机场，也是[[东非|东部非洲]]最繁忙、[[非洲|非洲]]第六繁忙的机场，得名于肯尼亚第一任总统[[乔莫·肯雅塔|乔莫·肯雅塔]]。坐落于内罗毕东南方向的恩姆巴卡西（Embakasi），距[[市中心|市中心]]15公里，地处建成区边缘。是[[肯尼亚航空|肯尼亚航空公司]]的枢纽机场。蒙巴萨高速与机场毗邻，是机场和内罗毕之间的主要通道。

机场设06/24跑道。06跑道配有[[仪表着陆系统|仪表着陆系统]]，用于起降。有一座建于20世纪70年代的航站楼。建于20世纪60年代的老航站楼有时被称为“原恩姆巴卡西机场”（Old Embakasi Airport），现用于货运和肯尼亚空军训练。<ref>[http://www.groundsupportworldwide.com/article/article.jsp?id=925&siteSection=1 GroundSupportWorldWide.com – The Creation of an African Aviation Epicenter]</ref>

2008年共输送旅客492 2542人。<ref>{{cite web |last=Kenya Airports Authority|title=JKIA Voted Best Marketed Airport In Africa Once Again|publisher=kenyaairports.co.ke |date= 2009-09-22|url=http://www.kenyaairports.co.ke/kaa/about/press_releases/2009/25sep09.html|accessdate=2010-01-14}}</ref>

== 历史 ==
'''内罗毕恩姆巴卡西机场'''（'''Nairobi Embakasi Airport'''）于1958年5月由最后一任肯尼亚总督埃沃林·巴陵（Evelyn Baring）宣布开放。原本应由[[伊丽莎白女王|伊丽莎白女王]]主持，但因在澳大利亚延误而未能参与开放仪式。<ref>[http://www.mccrow.org.uk/EastAfrica/NairobiAirport/Nairobi%20Airports.htm Nairobi Airports<!-- Bot generated title -->]</ref> 后来在跑道另一侧建起现在的航站楼，同时将机场改为今名。

== 航廈 ==

[[Image:Nairobi_Arrivals_Terminal.jpg|thumb]]

机场航站楼分为同时承担出发和到达的三座航廈，1、2航廈主要是国际航班，3航廈主要是国内航班。

出发的乘客在1、2航廈根据目的地办理登机手续，两个航廈均有使用通用航站楼设备系统的報到柜台，移民处位于一樓，之后乘客通过电梯或扶梯进入二樓候机厅。共有八个设有登机桥的登机門。国际到达大厅通过同样的登机門，在一楼行李提領處之前与通往移民处的线路合流。行李提領處设有行李轉盤。

到达区域设有银行、計程车、租车櫃檯、旅游服务中心和旅馆订房中心。往返市区的公车系統在1航廈和2航廈设有招呼站。所有航廈和到达區域都设有旅遊服務中心。设有航班訊息显示系统，可以顯示最新的航班到離資訊。

Simba餐厅位于中央大楼六樓。1航廈设有Home Park经营的自助餐厅，2航廈设有餐馆和酒吧，3航廈和国际到达區域设有自助餐厅和快餐店，均由NAS经营。每个航廈均设有饮料自动販賣机。

== 未来拓展 ==
<!-- Image with unknown copyright status removed: [[Image:Jomo_Kenyatta_International_Airport_Expansion.jpg|thumb]] -->

2005年10月14日，肯尼亚机场管理局宣布计划扩建机场。随后的两年内，当局宣布将升级全国各地机场，尤其是内罗毕。现机场设计年吞吐量为250万旅客，然而目前已达到400万，亟待进行扩建。扩建后的机场将比现有机场大一倍，面积从25662平方米扩大到55222平方米，现已捉襟见肘的停机坪将从200000平米扩大到300000平方米，还将建设新滑行道。到达和出发将完全分离，候机区将进行翻新改造。扩建后机场年吞吐量将达900万人，耗资一亿美元，其中[[世界银行|世界银行]]提供一千万美元。一期工程于2006年9月29日开始。由于在一次轻微事故中机场唯一的跑道被迫关闭一天，因此政府目前已經通過並且將建设第二跑道設為最優先工作。

== 事故 ==
* 1974年11月20日，[[德国汉莎航空公司540航班|德国汉莎航空公司540航班]]一架[[航空器注册编号|注册号]]为D-ABYB的[[波音747|波音747]]执行[[法兰克福|法兰克福]]-[[内罗毕|内罗毕]]-[[约翰内斯堡|约翰内斯堡]]航班，在从乔莫·肯雅塔国际机场24跑道起飞后不久坠毁，机上157人中59人遇难。
* 1989年5月，[[索马里航空公司|索马里航空公司]]一架[[波音707|波音707]]在起飞途中冲出跑道，机上70人无人伤亡。
* 1990年12月4日，苏丹货运航空公司一架波音707-321C在降落中坠毁，机上10人全部遇难。
* 2000年1月30日，[[肯尼亚航空|肯尼亚航空]]431航班执行内罗毕经[[拉各斯|拉各斯]]至[[阿布贾|阿布贾]]航班，但转向阿布贾，起飞飞往拉各斯时落入水中。
* 2007年5月5日，肯尼亚航空507航班从[[喀麦隆|喀麦隆]][[杜阿拉|杜阿拉]]起飞后失踪，在附近找到飞机时机上无人生还。
* 2013年8月7日，肯亞塔機場發生大火，入境航廈全部被燒毀。
