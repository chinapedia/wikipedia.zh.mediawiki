== 南洋參屬植物 ==
{{main|南洋參屬}}

南洋參屬植物({{lang|la|''Polyscias''}})旗下有114個品種，長有[[羽狀複葉|羽狀複葉]]。

== 形態 ==

裂葉福祿桐高約1-2公尺，側枝細長，表面布滿皮孔。葉[[互生|互生]]，中肋明顯，奇數羽狀複葉，小葉3-4對，卵形或卵橢圓形。鋸齒狀葉緣，鑲日邊新葉粉黃綠，後轉深綠。呈[[繖型花序|繖型花序]]，每年夏季開淡黃色小白花，有淡淡的花香味。結圓形球果，果實成熟時顏色由綠轉為黃。<ref>{{cite web|publisher=鳳西國中自然生態網|url=http://sjmm.loxa.edu.tw/02-03-06.htm|title=裂葉福祿桐|accessdate=2008-07-17|deadurl=yes|archiveurl=https://web.archive.org/web/20080911220234/http://sjmm.loxa.edu.tw/02-03-06.htm|archivedate=2008年9月11日|df=}}</ref>

== 品種 ==
按葉片的碎裂情度，分為「一回裂葉福祿桐」及「二三回裂葉福祿桐」兩類<ref name=hfjh />。

== 生長環境 ==

裂葉福祿桐一般被用作[[盆栽|盆栽]]觀葉及庭園造景樹而栽種，因為它生性強健，耐寒耐陰。不過，裂葉福祿桐對光的要求頗高（1500 到 10000 [[勒克斯|lux]]），所以只能在戶外環境生長。樹的汁液有毒，可用[[扦插法|扦插法]]繁殖<ref>{{cite web|url=http://www.dtes.chc.edu.tw/localspecial/plant/p55.htm|publisher=大同國小資訊看板|title=羽葉福祿桐|accessdate=2008-07-17}}</ref>。
