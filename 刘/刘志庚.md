'''刘志庚'''（{{bd|1956年|6月|||刘志庚}}），[[广东省|广东]][[兴宁市|兴宁]]人。1973年10月参加工作，1985年2月加入[[中国共产党|中国共产党]]。[[吉林大学|吉林大学]]经济系国民经济计划专业本科毕业，[[吉林大学|吉林大学]]政治经济学专业在职博士研究生。

== 生平 ==
早年曾担任过小学教师，恢复高考后考入[[吉林大学|吉林大学]]经济系国民经济计划专业学习。毕业后分配到[[深圳市计划局|深圳市计划局]]工作，仅用了8年多的时间就从一个科员升级至副局长。1992年12月后历任深圳市[[龙岗区|龙岗区]]区长、中共龙岗区委书记，[[清远市|清远市]]委副书记、市长，[[东莞市|东莞市]]市长、中共东莞市委书记。2011年12月，任[[广东省人民政府|广东省人民政府]]副省长，分管经济与信息化、交通运输、统计、通信、质量监督、安全生产工作。

== 落马 ==
2016年2月4日晚十時，中央紀委監察部網站公佈信息稱，劉志庚涉嫌嚴重違紀，正接受組織調查。<ref>{{cite web|url=http://www.ccdi.gov.cn/xwtt/201602/t20160204_74085.html|title=广东省副省长刘志庚涉嫌严重违纪接受组织调查|publisher=中央紀委監察部|accessdate=2016-02-04}}</ref>落马前一天，刘志庚还在韶关检查春运工作<ref>{{cite web|url=http://news.ifeng.com/a/20160204/47360224_0.shtml|title=刘志庚昨日在韶关检查春运工作|publisher=凤凰网|accessdate=2016-02-04}}</ref>。据媒体报道称，2014年2月，东莞掀起[[2014年东莞扫黄事件|扫黄运动]]，即与有关部门从外围着手调查刘志庚在东莞的违纪违法行为有关<ref name="扫黄是预兆">{{cite web|url=http://china.caixin.com/2016-02-04/100907492.html?_t_t_t=0.751900696195662|title=广东省副省长刘志庚落马 两年前东莞扫黄是预兆|publisher=财新网|accessdate=2016-02-04}}</ref>。而刘志庚在东莞期间，与原[[奥威斯太子酒店|奥威斯太子酒店]]的老板[[梁耀辉|梁耀辉]]，私下以“哥”相称<ref>{{cite web|url=http://news.youth.cn/sz/201602/t20160206_7612404.htm|title=刘志庚在东莞时不敢得罪“太子辉” 私下称“哥”|publisher=澎湃新闻|accessdate=2016-02-06|deadurl=yes|archiveurl=https://web.archive.org/web/20160206135120/http://news.youth.cn/sz/201602/t20160206_7612404.htm|archivedate=2016-02-06}}</ref>。

2016年4月18日，中纪委通报刘志庚已经被双开，移送司法机关<ref>{{cite web |url = http://www.ccdi.gov.cn/xwtt/201604/t20160418_77722.html |title = 广东省政府原党组成员、副省长刘志庚严重违纪被开除党籍 |accessdate = 2016年4月18日 |publisher = 中央纪委监察部网站}}</ref>。

2017年2月24日，[[南宁市|南宁]]铁路运输中级法院一审开庭审理了刘志庚受贿一案，认定他受贿共计折合9817万元人民币，法庭将择期宣判。同年5月31日，南宁铁路运输中级人民法院一审公开宣判，以受贿罪判处刘志庚无期徒刑，剥夺政治权利终身，没收个人全部财产；受贿所得财物予以追缴，上缴国库。刘志庚当庭表示服判，不上诉<ref>{{Cite web|url=http://news.hexun.com/2017-06-01/189436070.html|title=广东省原副省长刘志庚受贿案一审被判无期徒刑|accessdate=2017-06-01|author=|date=2017-06-01|work=|publisher=和讯网}}</ref>。

2017年7月4日，[[南宁铁路运输中级法院|南宁铁路运输中级法院]]一审公开开庭审理深圳市财政局办公室原调研员、东莞市华瑞房地产有限公司实际控制人曾小华行贿一案。公诉机关指控，2006年至2011年，曾小华请托时任中共东莞市委书记的刘志庚利用职务便利，为广东佳彩数码科技有限公司中标东莞有线数字电视[[机顶盒|机顶盒]]采购项目、东莞勤上光电股份有限公司业务发展提供帮助，分别从上述公司获利人民币1200万元和约2305万元。曾小华为感谢和继续得到刘志庚帮助，于2007年送给刘志庚位于深圳市[[宝安区|宝安区]]的别墅一栋，价值人民币约2617万元；2008年经刘志庚同意，送给刘志庚儿子刘文锐[[路虎|路虎]]揽胜越野车一辆，价值人民币约141万元。南宁铁路运输中级法院当天还对刘志庚受贿案另一起附属案进行开庭审理，广东锦龙发展股份有限公司法定代表人杨志茂被检方指控向刘志庚行贿人民币6411万余元<ref>{{Cite web|url=http://www.chinanews.com/sh/2017/07-04/8269050.shtml|title=涉嫌行贿广东省原副省长刘志庚逾亿元 广东两商人受审|accessdate=2017-07-04|author=|date=2017-07-04|work=|publisher=中新网}}</ref>。

== 评价 ==
===正面===
将[[东莞|东莞]]本地外来人口由“外地人”重新定义为“新莞人”，提高在[[东莞市|东莞]]外地人待遇，解决部分社会矛盾。<ref>{{cite web |url=http://dg.people.com.cn/BIG5/14153377.html |title=存档副本 |accessdate=2011-07-22 |deadurl=yes |archiveurl=https://web.archive.org/web/20160304044142/http://dg.people.com.cn/BIG5/14153377.html |archivedate=2016-03-04 }}《新莞人：改變的不僅僅是稱謂》</ref>

基础设施建设进一步推进，建成有虎门港、生态园、篮球中心等一系列基础市政工程设施。在施政的部分问题上开先河开始听取某些网络群众意见<ref>[http://wise.oeeee.com/a/20090914/270740.html]{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}《东莞书记刘志庚与民间智库人员座谈“拍砖”》</ref><ref>[http://news.sun0769.com/dg/video/t20110527_1036988.shtml] {{Wayback|url=http://news.sun0769.com/dg/video/t20110527_1036988.shtml |date=20110530073259 }}《刘志庚微博访谈网友提问均获回复》</ref>。社会治安虽仍不尽如人意但勉强可算有所好转，在任期间东莞最大的地下赌场被取缔<ref>[http://news.qq.com/a/20100113/001862.htm]《特警乘装甲车扫荡东莞赌场》</ref>。

=== 负面 ===
在[[东莞市|东莞]]任市委书记期间，[[东莞|东莞]]经济数据（[[国内生产总值|GDP]]）增速下降，由2007年的18.1%<ref>{{cite web |url=http://gzdaily.dayoo.com/html/2008-01/07/content_108040.htm |title=存档副本 |accessdate=2011-07-22 |deadurl=yes |archiveurl=https://web.archive.org/web/20160304081521/http://gzdaily.dayoo.com/html/2008-01/07/content_108040.htm |archivedate=2016-03-04 }}《广州日报2008年1月7日A31版：东莞预计2007年GDP达3151亿元》</ref>下降至2008年14%<ref>[http://nd.oeeee.com/data/200901/t20090119_965604.shtml]《2008年东莞GDP等宏观经济数据》</ref>、2009年5.3%<ref>{{cite web |url=http://www.eeo.com.cn/Politics/by_region/2010/01/28/161898.shtml |title=存档副本 |accessdate=2011-07-22 |deadurl=yes |archiveurl=https://web.archive.org/web/20100131082319/http://www.eeo.com.cn/Politics/by_region/2010/01/28/161898.shtml |archivedate=2010-01-31 }}《2009年东莞实现GDP逾3700亿元》</ref>、2010年10.3%<ref>{{cite web |url=http://gzdaily.dayoo.com/html/2010-12/28/content_1226993.htm |title=存档副本 |accessdate=2011-07-22 |deadurl=yes |archiveurl=https://web.archive.org/web/20110909113838/http://gzdaily.dayoo.com/html/2010-12/28/content_1226993.htm |archivedate=2011-09-09 }}《今年东莞GDP预计首破4千亿》</ref>，即使[[次貸危機|次贷金融危机]]过后东莞GDP增速仍未能回到2007年前水平。任期内受[[金融危机|金融危机]]企业倒闭、民工失业<ref>[http://www.youtube.com/watch?v=ieyONGC_Y_w]《中國東莞樟木頭最大玩具廠倒閉》</ref>，2011年经济复苏后仍由于施政问题<ref>[http://finance.jrj.com.cn/2011/07/23020210520616.shtml]《东莞加工企业困窘 政府施救心态矛盾》</ref>面临新一轮倒闭潮。

禁摩、禁养猪，其中禁养猪受到包括[[新华网|新华网]]等官方媒体质疑<ref>[http://society.people.com.cn/GB/6633621.html]《城市化地区该不该养猪?--东莞"禁猪令"风波调查》</ref>，被称为“刘治摩”<ref>{{cite web |url=http://www.ichangan.com/changan/1651 |title=存档副本 |accessdate=2011-07-22 |deadurl=yes |archiveurl=https://web.archive.org/web/20110814033145/http://www.ichangan.com/changan/1651 |archivedate=2011-08-14 }}《东莞书记刘治摩建议公安人性化执法 进房检查先敲门》</ref>、“[[牛魔王|牛魔王]]”、“三禁书记”<ref>[http://www.citure.net/info/2011511/2011511125343.shtml]《东莞书记刘志庚自嘲“三禁书记”》</ref>。

因企业被冒充其亲戚要钱而公布秘书电话<ref>[http://news.xinhuanet.com/politics/2008-01/14/content_7416029.htm]《東莞市委書記多次被冒名 公布秘書電話防他人被騙》</ref>，并表示如遇声明其亲戚要钱可致电秘书求证。在任期间，东莞被称为“性都”<ref>{{cite web |url=http://www.people.com.cn/h/2011/0623/c25408-4271868037.html |title=存档副本 |accessdate=2011-07-22 |deadurl=yes |archiveurl=https://web.archive.org/web/20160304091557/http://www.people.com.cn/h/2011/0623/c25408-4271868037.html |archivedate=2016-03-04 }}《东莞组建城市形象智囊团 欲甩掉“性都”等名号》</ref>，被指是东莞色情产业的“保护伞”<ref name="扫黄是预兆"/>。对于东莞人口减少刘志庚表示“從人口結構上看，這種調整未必是壞事”<ref>[http://gd.xinhuanet.com/newscenter/2008-11/04/content_14822755.htm]{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}《東莞市委書記：合俊倒閉為炒股和投資失誤所致》</ref>，如此言论自然受到东莞人质疑。网络小说《在东莞》作者元平被刑拘<ref>[http://www.fabao365.com/zt/hot/20100930_114.html]《东莞书案：“罪”在离现实太近》</ref>，部分网民表示刘志庚此做法言行不一致<ref>[http://cms.youth.cn/preview/wenhua.youth.cn/whpl/201009/t20100929_1349738.htm]{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}《“作家又被拘”，我们为何还要关注？》</ref>。提倡低碳生活，乘公交出行<ref>[http://news.xinhuanet.com/2011-06/15/c_121537537.htm]《廣東東莞市領導乘公交上班 體驗低碳出行》</ref>，但被网友指出图片内大巴空荡荡，刘志庚乘坐位置为残疾人座位，而且周围乘客亦较似东莞市政府公务员，有作秀嫌疑。治安问题：2011年7月初刘志庚表示“现在很多女同志敢背包上街了，敢戴首饰上街了”<ref>[http://roll.sohu.com/20110708/n312856971.shtml]《搜狐一周语录》</ref>，东莞市区即发生加油被抢车事件<ref>[http://news.timedg.com/2011-07/04/content_5050033.htm]{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}《两男抢车逃跑中放回“人质”》</ref>。
