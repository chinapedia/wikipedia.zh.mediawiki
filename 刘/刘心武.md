'''刘心武'''（{{bd|1942年|6月4日|||L刘}}），[[四川|四川]]安岳人，出生于[[四川|四川]][[成都|成都]]<ref name="multiple1">{{cite web |url=http://book.sina.com.cn/people/liuxinwu/|title=《名人堂：刘心武》|accessdate=2006|publisher=新浪网|language= zh-hans}}</ref>，[[中国|中国]]当代[[作家|作家]]、[[红学家|红学家]]，笔名'''刘浏'''、'''赵壮汉'''，擅长青年题材写作。研究《[[红楼梦|红楼梦]]》长达十余年，坚持从[[秦可卿|秦可卿]]这一人物入手，开创了[[红学|红学]]研究的一个新分支——[[秦学|秦学]]。除了《红楼梦》，他也对《[[金瓶梅|金瓶梅]]》研读了许多年。<ref name="解构">{{cite web|url=http://news.dahe.cn/2016/08-01/107247033.html | language= | title=刘心武全新解构《金瓶梅》：在巨大误解面前有压力 |publisher=成都商报 |date=2016年8月1日 |accessdate= }}</ref>

== 简历 ==
[[File:Liu_Xinwu-Hong_Loumeng.jpg|左]]
1942年刘心武出生于[[成都市|成都市]]。1950年后移居[[北京|北京]]。1961年毕业于[[北京师范专科学校|北京师范专科学校]]中文系，担任中学教师15年。1976年任北京出版社[[编辑|编辑]]，参与创刊《[[十月_(杂志)|十月]]》并任编辑。后任《[[人民文学|人民文学]]》[[杂志|杂志]]的主编<ref>{{Cite web |url=http://henan.163.com/12/0728/17/87H3J7VA022701RU.html |title=刘心武简介|accessdate=2015-11-03 |author= |date=2012-07-28 |work= |publisher=网易|language=Zh-hans }}</ref>。

1977年11月发表短篇小说《[[班主任_(小说)|班主任]]》，被视为“[[伤痕文学|伤痕文学]]”开山之作，也被认为是新时代文学的发轫作，<ref name="multiple1">{{cite web |url=http://book.sina.com.cn/people/liuxinwu/|title=《名人堂：刘心武》|accessdate=2006|publisher=新浪网|language= zh-hans}}</ref>引起轰动。

1985年出版長篇小說《[[鐘鼓樓_(小說)|鐘鼓樓]]》。另发表《[[5·19长镜头|5·19长镜头]]》、《[[公共汽车咏叹调|公共汽车咏叹调]]》，再次引发关注。

1986至1987在杂志《[[收获_(杂志)|收获]]》开辟《[[私人照相簿|私人照相簿]]》个人专栏，1999年出版《[[树与林同在|树与林同在]]》。

1992年后发表诸多随笔。

1993年发表研究《[[红楼梦|红楼梦]]》的学术论文，并用小说形式发表，长期以来坚持从[[秦可卿|秦可卿]]这一人物入手，开创了[[红学|红学]]研究的一个新分支——[[秦学|秦学]]。

1995年初步尝试建筑评论，1998年由[[中国建筑工业出版社|中国建筑工业出版社]]出版《[[我眼中的建筑与环境|我眼中的建筑与环境]]》，2004年由[[中国建材工业出版社|中国建材工业出版社]]出版《[[材质之美|材质之美]]》。

1993年出版《刘心武文集》8卷，直至2005年，海内外出版的个人专著的不同版本超过130种。其多部作品已被译为[[英文|英]]、[[法文|法]]、[[德文|德]]、[[日文|日]]、[[俄羅斯文|俄]]、[[意大利文|意]]、[[朝鲜文|韩]]、[[捷克文|捷克]]、[[瑞典文|瑞典]]、[[希伯来文|希伯来]]等文字出版。
 
2005年在[[中央电视台|中央电视台]][[教育频道|教育频道]][[百家讲坛|百家讲坛]]开讲《[[刘心武揭密〈红楼梦〉|刘心武揭密〈红楼梦〉]]》系列节目，原定为36集，但至23集时遭致[[红学家|红学家]]干预而停播。

2006年4月12日，刘心武应[[纽约华美人文学会|纽约华美人文学会]]之邀抵达[[美国|美国]]，4月16日于[[哥伦比亚大学|哥伦比亚大学]]做了两次“红楼揭密”演讲。2007年7月，继续在[[中央电视台|中央电视台]][[百家讲坛|百家讲坛]]讲述《[[刘心武揭密〈红楼梦〉|刘心武揭密〈红楼梦〉]]》系列节目。

2010年3月，在[[中央电视台|中央电视台]][[百家讲坛|百家讲坛]]讲述《[[〈红楼梦〉八十回后真故事|〈红楼梦〉八十回后真故事]]》系列节目。

2011年2月，《[[刘心武续红楼梦|刘心武续红楼梦]]》由江苏人民出版社出版发行，并且引发热议，谈论人群大致分为两派：一派质疑刘心武续写的权威性，力保名著不被“玷污”；另一派则期待尽早看到有别于以往的“红楼后二十八回”。<ref>{{cite web |url=http://culture.people.com.cn/GB/87423/13982304.html|title=刘心武续写《红楼梦》引发热议|accessdate=2011-02-23|work=|publisher=人民网|language= zh-hans}}</ref>

2016年7月，书籍《[[刘心武揭秘〈金瓶梅〉|刘心武揭秘〈金瓶梅〉]]》发行，他在书中揭秘了《金瓶梅》的成书之谜、西门庆死亡之谜、大结局之谜、影响《红楼梦》之谜等31个谜题。<ref name="解构"/>

== 作品风格 ==
刘心武常常從細節入手進行分析，因此其紅學理論多屬“索引派”。他所續的紅樓夢，以盡可能貼近原著中的判詞等細節為準，如賈寶玉和史湘云頭髮變白的情節，便是按照原著中的“因麒麟伏白首雙星”回目來寫的。由於苛求準確無誤地還原原著中的細微末節，常常不惜犧牲總體的合理性。<ref>{{cite web |url=http://culture.people.com.cn/GB/42223/68011/index.html|title=《文化：刘心武》|accessdate=2011|work=|publisher=人民网|language= zh-hans}}</ref>

== 评价 ==
在《刘心武续红楼梦》出版后，红学大家[[周汝昌|周汝昌]]写诗祝贺并褒奖刘心武，肯定了刘心武不惧指责勇于探佚的精神，称赞刘心武为“勇士”、“君子”。{{noteTag|1= 周汝昌，《辛卯立春大节纪事二首：贺刘心武续红楼梦竣工》，“一贺岁得芳讯，壮哉勇士风。为芹收佚稿，何论异还同。二淡交慕君子，何事互寻求。三节一通问，四时总顺流。彩牛迎渌酒，金兔上红楼。胜业为芹献，源同云梦头。”}} <br />
周汝昌：如今幸而来了一位名作家刘先生，心甘情愿弥补这一缺陷，对于红学界来说，增添了实力，注入了新的思想智慧，我们应该表示热烈欢迎。<ref>{{cite web|url=http://book.sohu.com/20071112/n253192292_2.shtml|title=《刘心武再登百家讲坛，周汝昌抱病之躯为其呐喊》|accessdate=2007年11月12日|work=|publisher=搜狐网|language=zh-hans|deadurl=yes|archiveurl=https://web.archive.org/web/20120315172637/http://book.sohu.com/20071112/n253192292_2.shtml|archivedate=2012年3月15日}}</ref><br />

刘心武曾经把[[北宋|北宋]]诗人[[黄庭坚|黄庭坚]]的计作《寄黄几复》中的名句“桃李春风一杯酒，江湖夜雨十年灯”，说是自己“梦中偶得”而发表，引发很多批评。在百家讲坛上讲《红楼梦》，又被人指[[杜撰|杜撰]]。<ref>{{cite web|url=http://www.openow.net/details/e1764.html|title=《又见“江湖夜雨十年灯”》|accessdate=2012-10-19|work=|publisher=|language=zh-hans}}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><br />

== 获奖作品 ==
*长篇小说《[[鐘鼓樓_(小說)|钟鼓楼]]》获第二届[[茅盾文学奖|茅盾文学奖]]。
*短篇小说《[[班主任_(小说)|班主任]]》获1978年全国首届优秀短篇小说一等奖。
*短篇小说《[[我爱每一片绿叶|我爱每一片绿叶]]》、儿童文学《[[看不见的朋友|看不见的朋友]]》《[[我可不怕十三岁|我可不怕十三岁]]》曾获过全国性奖项。
*长篇小说《[[四牌楼_(小说)|四牌楼]]》获得第二届上海优秀长篇小说大奖。
