'''刘联华'''（{{bd|1953年4月||}}），[[山东|山东]][[烟台|烟台]]人，[[汉族|汉族]]，[[中华人民共和国|中华人民共和国]]政治人物、[[第十二届全国人民代表大会|第十二届全国人民代表大会]]解放军代表。

毕业于[[华南师范大学|华南师范大学]][[政治学|政治学]]专业。加入[[中国共产党|中国共产党]]。

1999年起任[[中国人民解放军驻香港部队|中国人民解放军驻香港部队]][[参谋长|参谋长]]。2002年3月担任[[中国人民解放军驻澳门部队|中国人民解放军驻澳门部队]][[司令员|司令员]]。2008年2月起任广州军区副参谋长。2011年6月起任[[中共广东省委|中共广东省委]]常委、[[广东省军区|广东省军区]]司令员。2012年5月起，不再担任中共广东省委常委<ref>{{cite news |title = 刘联华不再担任广东省委常委(图/简历) |url = http://news.163.com/12/0514/10/81F7PL6P00014JB5.html |accessdate=2012-05-14 |website = 网易新闻}}</ref>。

2002年晋升少将军衔。

2008年起任[[第十一届全国人大代表|第十一届全国人大代表]]<ref>{{cite web |title = 全国人大代表信息-刘联华 |url = http://www.npc.gov.cn/delegate/viewDelegate.action?dbid=112794 |accessdate=2013-04-05 |website = 全国人大网}}</ref>。2013年起任[[第十二届全国人大代表|第十二届全国人大代表]]<ref>{{cite web |title = 全国人大代表信息-刘联华 |url = http://www.npc.gov.cn/delegate/viewDelegate.action?dbid=122807|accessdate=2013-04-05 |website = 全国人大网}}</ref>。
