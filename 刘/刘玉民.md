'''刘玉民'''{{BD|1951年|2月|||catIdx=L刘}}，[[山东|山东]][[济南|济南]]人，籍贯山东[[荣成|荣成]]，中国大陆小说家，[[中国作家协会|中国作家协会]]成员，[[中国共产党|中国共产党]]党员。<ref name="fish">{{cite web|url=http://www.sd.xinhuanet.com/wq/2008-01/02/content_12103548.htm|title="茅盾文学奖"获得者刘玉民|accessdate=|publisher=新华网|work=|language=zh-hans|deadurl=yes|archiveurl=https://web.archive.org/web/20130630132705/http://www.sd.xinhuanet.com/wq/2008-01/02/content_12103548.htm|archivedate=2013-06-30}}</ref><ref name="ssa" >{{cite web |url=http://news.sina.com.cn/o/2012-12-14/085925811322.shtml |title=刘玉民 |accessdate=2012 |publisher=新浪网 |work= |language=zh-hans}}</ref><ref name="us" >{{cite web |url=http://www.chinawriter.com.cn/zxhy/member/3211.shtml |title=刘玉民 (1951～) |accessdate= |publisher=中国作家网 |work= |language=zh-hans}}</ref>

==生平==
1951年，刘玉民出生在[[山东省|山东省]][[济南市|济南市]]的一个农民家庭。<ref name="fish" />
孩提时，刘玉民在荣成市上学。<ref name="fish" />
刘玉民父亲早逝，母亲在文化大革命中去世。<ref name="fish" />
中学之后，刘玉民在人民公社当交通员。<ref name="fish" />

1970年，刘玉民加入[[中国人民解放军|中国人民解放军]]。<ref name="fish" />

1971年，刘玉民开始发表作品，其作品发表在《[[济南日报|济南日报]]》和《[[大众日报|大众日报]]》。同年，刘玉民在[[济南军区|济南军区]]当战士，一直到1981年。<ref name="fish" />

1982年，刘玉民成了济南市文联的一名编辑，并且一步步爬上济南市作家协会主席、山东省文联副主席、山东文人书画院院长。<ref name="fish" />
