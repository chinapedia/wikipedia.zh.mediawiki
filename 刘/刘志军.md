'''刘志军'''（{{bd|1953年|1月29日|||L刘}}），[[中國|中國]][[湖北省|湖北]][[鄂州市|鄂州]]人，曾任[[中华人民共和国铁道部|中华人民共和国铁道部]]部长、党组书记，有一名兄弟[[劉志祥|劉志祥]]。[[中共中央党校|中共中央党校]]硕士学历<ref name="gov.cn">{{cite web |url = http://www.gov.cn/english/2005-09/20/content_65226.htm |title = Liu Zhijun |publisher = Gov.cn: Chinese Government's Official Web Portal |access-date = 2014-06-08 |archive-url = https://web.archive.org/web/20130927183711/http://www.gov.cn/english//2005-09/20/content_65226.htm |archive-date = 2013-09-27 |dead-url = no }}</ref>，[[工程师|工程师]]（中级[[职称|职称]]）。刘志军出生于农民家庭，於1972年[[参加工作|参加工作]]，曾担任[[郑州铁路局|郑州铁路局]]副局长、[[沈阳铁路局|沈阳铁路局]]局长、铁道部副部长等职。2003至2011年任铁道部部长。2011年因涉嫌严重违纪而被免职。

刘志军提出了“铁路跨越式发展”的方针，业界称之为“'''刘跨越'''”<ref>{{cite journal |url = http://www.nfcmag.com/article/3036.html |title = 高铁民航角力京沪线 |author = 李威 |journal = [[南风窗|南风窗]] |year = 2011 |issue = 15 |access-date = 2013-07-21 |archive-url = https://web.archive.org/web/20140111053445/http://www.nfcmag.com/article/3036.html |archive-date = 2014-01-11 |dead-url = no }}</ref>。在任内，刘志军动用多种资源，引进和发展高速铁路技术，开展大规模铁路建设。任内修建了长达1.8万公里的铁路，在建的铁路则高达3万公里。中国成为世界上高速铁路运营里程最长、在建规模最大的国家，刘志军因此被誉为“'''中国高铁之父'''”<ref>{{cite web |url=http://finance.sina.com.cn/china/20130711/110116090564.shtml |title=高铁之父刘志军与中国高铁建设的盛衰路 |publisher=Finance.sina.com.cn |date=2013-07-11 |accessdate=2018-02-12 |archive-url=https://web.archive.org/web/20171116091545/http://finance.sina.com.cn/china/20130711/110116090564.shtml |archive-date=2017-11-16 |dead-url=no }}</ref>。但也造成了铁道部的巨额债务。对刘的功过评价类似于[[日本|日本]][[新幹線|新幹線]]之父[[十河信二|十河信二]]<ref>{{cite web |author=更新于2011年2月23日 07:15 作者：英国《金融时报》中文网专栏作家 加藤嘉一 |url=http://www.ftchinese.com/story/001037078?page=1 |title=刘志军的高铁遗产 |publisher=Ftchinese.com |date=2017-12-14 |accessdate=2018-02-12 |archive-url=https://web.archive.org/web/20180212201845/http://www.ftchinese.com/story/001037078?page=1 |archive-date=2018年2月12日 |dead-url=no }}</ref>。

2013年7月8日，[[北京市第二中级人民法院|北京市第二中级人民法院]]一审以[[受贿罪|受贿罪]]、[[滥用职权罪|滥用职权罪]]判处刘志军[[死刑缓期执行|死刑，缓期两年执行]]，并处没收全部个人财产<ref>{{cite news |url = http://news.xinhuanet.com/legal/2013-07/08/c_116442565.htm |title = ==早年经历==
1953年1月，刘志军出生于[[湖北省|湖北省]][[鄂城市|鄂城市]]牌坊村刘金湾，是家里的长子，上有两个姐姐，下有一弟一妹<ref name=hq>{{cite journal|url=http://www.globalpeople.com.cn/index.php/news/cover-story/3574|title=刘志军的罪与罚|journal=环球人物|year=2013|issue=214|author=黄滢|deadurl=yes|archiveurl=https://web.archive.org/web/20130811123217/http://globalpeople.com.cn/index.php/news/cover-story/3574|archivedate=2013-08-11}}</ref>。因为家里穷，刘志军的父母当年卖了房子才供他读到[[初中|初中]]<ref name=hq/>。据刘志军自述，因为祖上有过雇工，家里被划为[[富农|富农]]，[[阶级成份|阶级成份]]不好，当初进入铁路系统很曲折。

1972年2月，初中毕业的刘志军赶上铁路招工，机会来临时，体检却查出肺部疑有钙化点。刘志军连夜赶路到县城医院覆診，拿到心肺无异议的确诊结果赶回，只差几分钟就到了截止的时间<ref name="xwzk">{{cite web| url=http://politics.inewsweek.cn/20130624/detail-46209-all.html| title=刘志军在秦城监狱托话给女儿：千万不要从政| publisher=中国新闻周刊| accessdate=2013-09-27| author=申欣旺}}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。19岁的刘志军遂成为武汉铁路分局武昌工务段的一名养路工<ref name=hq/>。因为字写得好，后来当了队里面的文书<ref name="woan">{{cite journal | url=http://www.capitalweek.com.cn/2013-05-20/76213491.html | title=铁道部窝案：多面刘志军 | journal=证券市场周刊 | year=2013 | issue=17 | author=戴小河 | access-date=2013-07-21 | archive-url=https://web.archive.org/web/20131105032807/http://www.capitalweek.com.cn/2013-05-20/76213491.html | archive-date=2013-11-05 | dead-url=no }}</ref>。

== 铁路系统升迁 ==
1972年2月至1981年9月，刘志军历任武汉铁路分局武昌工务段养路工、武昌工务段团委干事、团委书记、党委副书记，[[武汉铁路分局|武汉铁路分局]]团委书记。1973年8月，刘志军加入中国共产党。<ref name=
jianli>{{cite web |url = http://news.hexun.com/2008-02-28/104110143.html |title=铁道部部长刘志军简历 |publisher=新华网|date=2008-02-28}}</ref>

1981年9月至1982年9月，刘志军在[[华东交通大学|华东交通大学]]基础课部干部班学习。1982年9月至1984年9月在[[西南交通大学|西南交通大学]]运输系运输管理专业学习，补上了学历的不足<ref name="hq" />。刘志军的聪明与勤奋得到时任[[武汉铁路分局|武汉铁路分局]]副局长黄从全的赏识，黄把自己的女儿黄立平许配给他<ref name="woan"/><ref name="peiou">{{cite news |author= |url=http://news.163.com/11/0301/11/6U2CRB0R00014AED.html |title=刘志军倒在高铁招标潜规则上 |work=[[网易|网易]] |date=2011-03-01 |accessdate=2013-04-28 |archive-url=https://web.archive.org/web/20161009202058/http://news.163.com/11/0301/11/6U2CRB0R00014AED.html |archive-date=2016-10-09 |dead-url=no }}</ref>。此后的10年，刘志军搭上了“顺风车”。1984年9月至1987年4月，先后任[[郑州铁路局|郑州铁路局]][[武汉铁路分局|武汉铁路分局]][[江岸站|江岸站]]站长、党委副书记，武汉铁路分局党委常委、分局政治部副主任，1986年任武汉铁路分局党委书记。1987年4月，调任[[广州铁路局|广州铁路局]]党委常委、路局政治部副主任。1988年8月调任郑州铁路局武汉铁路分局局长、党委副书记。1991年9月，升任郑州铁路局副局长、党委常委。<ref name=jianli/>

在1992年郑州铁路局党委常委差额选举中，刘志军落选。落选原因除刘志军本人性格缺陷外，更大的原因在于离婚事件。刘志军在担任武汉铁路分局党委书记后不久，便与妻子离婚。刘志军落选郑州局党委常委后四处活动，得以在1992年8月调任湖北省国防科学技术工业办公室党组书记（正厅级），短暂离开了铁路系统<ref>{{cite web|url=http://news.ctdsb.net/shizheng/2013/0411/122776.shtml|title=刘志军的发迹及丑闻|publisher=楚天都市网|deadurl=yes|archiveurl=https://web.archive.org/web/20130927172201/http://news.ctdsb.net/shizheng/2013/0411/122776.shtml|archivedate=2013-09-27}}</ref>。

刘志军于1993年2月返回铁路系统。先是参加铁道部运输改革调研工作为期2个月。1993年4月出任[[沈阳铁路局|沈阳铁路局]]局长、党委副书记<ref name=jianli/>。1994年11月，刘志军进入铁道部，任铁道部党组成员、总调度长。其间1995年9月至1996年7月在[[中共中央党校一年制中青年干部培训班学员列表|中央党校一年制中青干部培训班]]学习。1995年9月至1998年7月在中央党校培训部哲学专业在职研究生班学习。1996年8月升任铁道部副部长、部党组成员。1997年1月至2000年4月，兼任[[中国联合通信有限公司|中国联合通信有限公司]]副董事长<ref name=jianli/>。

==铁道部部长==
[[File:Beijing-Tianjin_High_Speed_Train.jpg|thumb]][[蘭希·佩洛西|蘭希·佩洛西]]]]
2002年9月，刘志军担任铁道部[[党组书记|党组书记]]<ref name=ky/>、副部长。2002年11月，在[[中国共产党第十六次全国代表大会|中共十六大]]上当选为[[中央委员|中央委员]]<ref>{{cite web|url=http://cpc.people.com.cn/GB/64162/64168/64569/65444/4429139.html|title=中国共产党第十六届中央委员会委员名单|date=2002-11-14|publisher=中国共产党新闻网|access-date=2013-07-21|archive-url=https://web.archive.org/web/20130514193217/http://cpc.people.com.cn/GB/64162/64168/64569/65444/4429139.html|archive-date=2013-05-14|dead-url=no}}</ref>。2003年3月17日，[[第十届全国人民代表大会第一次会议|第十届全国人民代表大会第一次会议]]决定刘志军为铁道部部长。

===铁路跨越式发展===
[[File:HXD1_with_wagen.jpg|thumb]]。]]
铁道部是中国最有权力的部委之一，被称为“独立王国”<ref name=nytimes>{{cite web|url=http://www.nytimes.com/2013/03/21/world/asia/21iht-letter21.html?_r=0|title=Swift Demise of Last Major Monopoly in China|author=Didi Kirsten Tatlow|publisher=''The New York Times''|date=2013-03-20|access-date=2013-07-21|archive-url=https://web.archive.org/web/20130618113824/http://www.nytimes.com/2013/03/21/world/asia/21iht-letter21.html?_r=0|archive-date=2013-06-18|dead-url=no}}</ref>。铁道部有200万名职工，4.3万亿元人民币资产，并拥有自己的[[公安局|公安局]]、[[检察院|检察院]]和[[法院|法院]]<ref name=nytimes/>。然而铁路网整体能力严重不足，主要干线能力十分紧张，甚至处于超饱和状态<ref>{{cite news|url=http://finance.sina.com.cn/roll/20031025/0800489734.shtml|title=部长访谈全面建设小康社会需要发达完善的铁路网——铁道部部长刘志军谈铁路的跨越式发展|date=2003-10-25|publisher=经济日报|access-date=2013-07-21|archive-url=https://web.archive.org/web/20160304121039/http://finance.sina.com.cn/roll/20031025/0800489734.shtml|archive-date=2016-03-04|dead-url=no}}</ref>。

刘志军上任后，铁路改革的方向就发生逆转。前任部长[[傅志寰|傅志寰]]所推行的“网运分离”改革走到尽头，“跨越式发展”成为刘志军的目标<ref name=zjol>{{cite web|url=http://jrzb.zjol.com.cn/html/2013-04/11/content_2088348.htm?div=-1|title=刘志军沉浮记：靠妻子起家后离婚，高铁“大跨越”中落马|date=2013=04-11|access-date=2013-07-21|archive-url=https://web.archive.org/web/20130928072603/http://jrzb.zjol.com.cn/html/2013-04/11/content_2088348.htm?div=-1|archive-date=2013-09-28|dead-url=no}}</ref>。“主辅分离”成为刘志军在改革上的主导思路。该思路是将与高铁建设直接相关的权力、资产收归铁道部，而与之相关不大者予以剥离<ref name=gaigeyichan>{{cite news|author=孙春芳|title=铁路改革系列观察之二：刘志军的改革遗产|url=http://www.21cbh.com/HTML/2012-1-11/0NMDY5XzM5NTA0Ng.html|date=2012-01-11|deadurl=yes|archiveurl=https://web.archive.org/web/20131016235506/http://www.21cbh.com/HTML/2012-1-11/0NMDY5XzM5NTA0Ng.html|archivedate=2013-10-16}}</ref> 。

2002年12月，时任铁道部党组书记的刘志军在全国铁路工作会议上提出：“充分利用后发优势，学习借鉴发达国家铁路技术，实现我国铁路的跨越式发展，加快铁路现代化建设。”<ref>{{cite news|author=石玉国|title=百年陇海大提速(铁路大提速系列报道之四)|url=http://www.zytdb.com/news/article/8/2008/08/19/30665.html|newspaper=中原铁道报|date=2008-08-19}}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。刘志军认为，走常规发展之路已经不能解决中国铁路面临的主要矛盾，必须以“跨越式”的快速发展模式、吸收发达国家的经验，较常规发展更适合中国国情<ref>{{cite news|title=铁路实施跨越式发展战略 今后五年还将两次大提速|url=http://news.xinhuanet.com/newscenter/2003-09/17/content_1085395.htm|newspaper=经济参考报|date=2003-09-17|access-date=2013-08-20|archive-url=https://web.archive.org/web/20041030084802/http://news.xinhuanet.com/newscenter/2003-09/17/content_1085395.htm|archive-date=2004-10-30|dead-url=no}}</ref>。

2003年6月，铁道部在北京召开“铁路跨越式发展研讨会”，刘志军在讲话中阐述了中国铁路跨越式发展的内涵与任务，刘志军认为，跨越式发展是指在一定历史条件下落后者对先行者走过的某个发展阶段的超常规赶超行为。铁路跨越式发展，一是指运输能力的快速扩充。集中人力物力财力，加快铁路建设，在较短时间内解决铁路运输能力不适应的问题，早日使铁路运输能力适应国民经济和社会发展需要。二是指技术装备水平的快速提高。充分利用国际国内先进的技术资源，加快技术创新，在较短时间内，使中国铁路主要技术装备达到或接近发达国家水平<ref name=ky>{{cite journal|author=刘志军|title=落实“三个代表”要求 抓住新的历史机遇 努力实现中国铁路跨越式发展——刘志军部长在2003年6月28日召开的铁路跨越式发展研讨会上的讲话|journal=铁道工程企业管理|year=2003|issue=04}}</ref>。

===大规模铁路建设===
[[File:Shanghai_Hongqiao_Railway_Station_2010.jpg|thumb]]。]]
[[File:中国铁路基本建设投资（2002-2010）.png|thumb]]
2003年刘志军上任后不久，铁道部会同交通部、国家发改委等部门，制定了中国首个《[[中长期铁路网规划|中长期铁路网规划]]》，并于2004年1月经[[国务院常务会议|国务院常务会议]]审议通过。规划确定，到2020年全国铁路营业里程要达到10万公里，主要繁忙干线实现客货分线, [[复线铁路|复线]]率和[[铁路电气化|电化]]率均达到50％。同时，提出了[[“四纵四横”客运专线|“四纵四横”客运专线]]的宏伟计划，[[客运专线|客运专线]]总长度达到1.2万公里，客车速度目标值达到每小时200公里及以上<ref>{{cite journal|title=国家《中长期铁路网规划》内容简介|author=国家发展和改革委员会交通运输司|journal=交通运输系统工程与信息|year=2005|volume=5|issue=4}}</ref>。实现《中长期铁路网规划》的总投资达到20000亿元[[人民币|人民币]]<ref>{{cite news|url=http://www.chinanews.com/news/2005/2005-09-21/8/628835.shtml|title=实现《中长期铁路网规划》目标将需2万亿元资金|date=2005-09-21|publisher=中新社|access-date=2013-07-21|archive-url=https://web.archive.org/web/20110121173436/http://www.chinanews.com/news/2005/2005-09-21/8/628835.shtml|archive-date=2011-01-21|dead-url=no}}</ref>。

上万亿人民币的投资，使得刘志军的计划成为了自美国[[州际公路系统|州际公路系统]]以来，世界上投资最高的公共工程项目<ref name=osnos>{{cite web|author=Osnos, Evan.|url=http://www.newyorker.com/reporting/2012/10/22/121022fa_fact_osnos?currentPage=all&src=longreads&mobify=0|title="Boss Rail: The Disaster That Exposed the Underside of the Boom"|publisher=''The New Yorker''|date=2012-10-22|access-date=2012-10-26|archive-url=https://web.archive.org/web/20130313112445/http://www.newyorker.com/reporting/2012/10/22/121022fa_fact_osnos?currentPage=all&src=longreads&mobify=0|archive-date=2013-03-13|dead-url=no}}</ref>。为了在2008年之前建成中国第一条高速铁路，刘志军命令铁道部员工夜以继日地工作<ref name=osnos/>，并提出“要牺牲一代铁路人的利益换来中国铁路的发展”<ref>{{cite news|url=http://finance.sina.com.cn/roll/20110219/00099399565.shtml|title=刘志军功与过：媒体评其发展高铁功不可没|date=2011-02-19|publisher=华夏时报|access-date=2013-07-21|archive-url=https://web.archive.org/web/20110222072731/http://finance.sina.com.cn/roll/20110219/00099399565.shtml|archive-date=2011-02-22|dead-url=no}}</ref>。刘志军对内对外一律强势，对系统内反对意见坚决打击<ref name=zjol/>。在其任上，地方铁路局局长纷纷易人；对外则封杀呼吁铁路改革的媒体和专家。作为铁道部长，刘志军亲上火线，安排和指挥项目的分配<ref name=zjol/>。刘志军的勃勃雄心为他赢得了“刘跨越”的绰号，甚至有人称其为“刘疯子”<ref name=osnos/>。

刘志军建立“省部合作”的机制，把原先就对基础设施建设十分热衷的地方政府拉入到铁路建设的狂潮之中<ref name=gaigeyichan /> 。铁道部与31个省区市签订了加快铁路建设的战略协议，组建合资铁路公司。地方政府不仅承担征地拆迁的主体责任，而且对铁路建设的权益性投资达4000多亿元，改变了铁路建设仅靠中央政府投资的局面<ref>{{cite news | url=http://paper.cs.com.cn/epaper02/html/2008-08/13/content_17592719.htm | title=投融资体制改革步步深入，铁路修建主体进一步多元化 | work=中国证券报 | date=2008-08-13 | accessdate=2013-07-20 | pages=B05 | deadurl=yes | archiveurl=https://web.archive.org/web/20130928001117/http://paper.cs.com.cn/epaper02/html/2008-08/13/content_17592719.htm | archivedate=2013-09-28 }}</ref>。刘志军遗留下来的诸多铁路公司，初具一般商业公司的雏形，且拥有铁路路网和车辆资产。其运营模式则是让途经沿线的铁路局委托经营，这成为实行铁路改革区域分割的良好基础<ref name=gaigeyichan /> 。

刘志军任内，铁路建设投资达到历史最高峰。2003年铁路基本建设投资仅有533亿元人民币<ref>铁道部统计中心,中华人民共和国铁道部. 2003年铁道统计公报,2004.</ref>，2010年已达7075亿元人民币<ref>铁道部统计中心,中华人民共和国铁道部. 2010年铁道统计公报,2011.</ref>。铁路投资超过了军费<ref>{{cite news|title=李肇星：今年中国国防费预算5321.15亿元 增幅下降|url=http://2010lianghui.people.com.cn/GB/181624/11073574.html|newspaper=人民网|date=2010-03-04|access-date=2013-08-20|archive-url=https://web.archive.org/web/20120927002425/http://2010lianghui.people.com.cn/GB/181624/11073574.html|archive-date=2012-09-27|dead-url=no}}</ref> 。任内共修建了1.8万公里的铁路，建设中的铁路达3万公里。中国铁路运营里程从2002年的7.19万公里增长至2010年的9.1万公里。大规模铁路建设也给铁道部带来了沉重的债务压力，截止2011年上半年，铁道部负债达20907亿元，负债率为58.53%<ref>{{cite news|author=刘春瑞|title=铁道部总负债破两万亿元，上半年盈利近43亿|url=http://news.qq.com/a/20110802/000100.htm|newspaper=新京报|date=2011-08-02|access-date=2013-08-20|archive-url=https://web.archive.org/web/20180723110746/https://news.qq.com/a/20110802/000100.htm|archive-date=2018-07-23|dead-url=no}}</ref> 。

===引进和发展高速铁路技术===
[[File:CRH5A--京沪高铁.jpg|thumb]]动车组]]

自1990年代开始，中国开始自主研制高速列车，“[[中国铁路DJJ1型电力动车组|蓝箭]]”、“[[先鋒號電力動車組|先锋]]”、“[[中国铁路DJJ2型电力动车组|中华之星]]”、“[[長白山號電動車組|长白山]]”等高速列车先后问世。然而国产列车的设计要求低于国外高速铁路标准，整体技术不成熟，距离商业运营还有很大距离。同时，一些核心关键技术和设备可靠性、稳定性不足，列车的工艺水平比较差，关键设备需要进口<ref name="cz">{{cite web | url=http://news.xinhuanet.com/globe/2010-03/04/content_13096289.htm | title=中国式高铁的诞生与成长 | publisher=环球杂志 | date=2010-03-04 | accessdate=2013-09-27 | author=陈昕晔,王宁 | archive-url=https://web.archive.org/web/20160919015327/http://news.xinhuanet.com/globe/2010-03/04/content_13096289.htm | archive-date=2016-09-19 | dead-url=no }}</ref>。种种原因最终使得中国政府决定从国外引进高铁技术，但国外厂商必须向中国转让技术<ref name="cz"/>。2004年4月，[[中国国务院|中国国务院]]召开会议，专题研究铁路机车车辆装备有关问题，提出了“引进先进技术、联合设计生产、打造中国品牌”的基本原则，确立了引进少量原装、部分国内散件组装和大批量国内生产的项目运作模式<ref name="syf"/>。

在刘志军主导下，铁道部先后开展了三次动车组招标。铁道部在招标文件中明确规定，国外企业只能作为国内企业合作方，并通过国内企业参与竞争，并转让核心技术、分步实施国产化<ref name="syf">{{cite news | url=http://paper.people.com.cn/rmrb/html/2007-07/31/content_14137782.htm | title=重大技术引进必须发挥政府主导作用 ——访中国工程院院士孙永福 | work=人民日报 | date=2007-07-31 | accessdate=2013-07-20 | pages=14 | archive-url=https://web.archive.org/web/20160129151051/http://paper.people.com.cn/rmrb/html/2007-07/31/content_14137782.htm | archive-date=2016-01-29 | dead-url=no }}</ref>。2004年6月，铁道部启动了时速200公里动车组招标；2005年10月，“时速300公里动车组”的项目招标启动；2006年11月，中国第一条新建高速铁路[[京津城际|京津城际]]项目招标。铁道部运用了“战略买家”的策略，把国内几十家列车生产企业召集在一起，仅允许[[南车四方|南车四方]]和[[北车长客|北车长客]]与国外厂商谈判，其他企业一概不准与外方接触<ref name=tanpan/>。铁道部以中国铁路独一无二的市场优势作为吸引，迫使[[川崎重工|川崎重工]]、[[阿爾斯通公司|阿尔斯通]]、[[龐巴迪公司|庞巴迪]]和[[西門子公司|西门子]]等国外厂商降价<ref name=tanpan>{{cite news|url=http://news.sina.com.cn/c/2007-06-04/015311950828s.shtml|title=中国动车组技术引进谈判揭秘:首次招标节省90亿|date=2007-06-04|publisher=新京报|access-date=2013-07-21|archive-url=https://web.archive.org/web/20160129072706/http://news.sina.com.cn/c/2007-06-04/015311950828s.shtml|archive-date=2016-01-29|dead-url=no}}</ref>。

通过引进和消化国外技术，中国在较短时间内以较低的成本掌握了高速动车组多项关键技术和配套技术，完成了高铁动车组研制的第一阶段<ref name="kjrb">{{cite news | url=http://news.xinhuanet.com/tech/2011-06/28/c_121596120.htm | title=我国自主创新350公里高铁技术更快捷更安全更舒适 | work=科技日报 | date=2011-06-28 | accessdate=2013-07-20 | archive-url=https://web.archive.org/web/20110903090847/http://news.xinhuanet.com/tech/2011-06/28/c_121596120.htm | archive-date=2011-09-03 | dead-url=no }}</ref>。2007年4月18日，中国铁路进行[[中国铁路大提速|第六次大提速]]，在既有线上开行时速200-250公里的[[动车组|动车组]]列车，中国铁路跨入高速时代<ref>{{cite news|url=http://finance.people.com.cn/GB/8215/80960/5608354.html|title=铁路18日第六次大提速，中国铁路跨入高速时代|date=2007-04-13|publisher=人民日报|access-date=2013-07-21|archive-url=https://web.archive.org/web/20160304085645/http://finance.people.com.cn/GB/8215/80960/5608354.html|archive-date=2016-03-04|dead-url=no}}</ref>。第二阶段，即在掌握时速200—250公里高速列车技术的基础上，自主研制生产了时速350公里高速列车<ref name="kjrb" />。2008年4月，时速350公里的[[和谐号CRH3型电力动车组|CRH3型动车组]]下线。8月1日，中国第一条时速350公里的高速铁路——[[京津城际铁路|京津城际铁路]]开通运营。

而“[[中华之星|中华之星]]”等国产动车组由于不满足招标文件的要求，以及故障等因素，最终彻底出局。此举引起了强烈反弹。在2005年7月间铁道部主持召开的“中华之星”阶段验收总结会上，多方评委迫于刘志军的压力，纷纷认定此车最高只能跑160km／h<ref name="zhzx">{{cite web | url=http://www.infzm.com/content/61711 | title=【高铁之乱】中国高铁神话终结？ | publisher=南方周末 | date=2011-07-29 | accessdate=2013-09-27 | author=吕宗恕,沈念祖,祝杨 | archive-url=https://web.archive.org/web/20130928011523/http://www.infzm.com/content/61711 | archive-date=2013-09-28 | dead-url=no }}</ref>。“中华之星”的总设计师[[刘友梅|刘友梅]]联合原铁道部部长[[傅志寰|傅志寰]]在内的52名院士，向[[国务院|国务院]]呈送了一份《关于报送院士反映“中华之星”高速列车有关情况的签名信》，举报刘志军的行为，最终不了了之<ref>{{cite news | url=http://news.xinhuanet.com/fortune/2006-03/13/content_4298987.htm | title="中华之星"缘何成流星：高铁技术关键时刻掉链子 | publisher=商务周刊 | date=2011-03-05 | access-date=2013-09-27 | archive-url=https://web.archive.org/web/20061216133243/http://news.xinhuanet.com/fortune/2006-03/13/content_4298987.htm | archive-date=2006-12-16 | dead-url=no }}</ref>。

===高铁计划的进一步升级===
[[File:CRH380Afromshanghai.jpg|thumb]]]]
2007年，[[2007年－2008年环球金融危机|環球金融危機]]爆发。为了抵制经济衰退，刺激经济，中国政府进一步增加了在铁路上的投资。中国政府出台了[[扩大内需十项措施|“四万亿”经济刺激计划]]，其中1.5万亿被用于修建铁路、公路、机场等[[基础设施|基础设施]]<ref>{{cite news | url=http://politics.people.com.cn/GB/1027/8919115.html | title=张平详解四万亿投资安排，基础设施建设占重头 | work=人民网 | date=2009-03-06 | access-date=2013-08-20 | archive-url=https://web.archive.org/web/20130521144948/http://politics.people.com.cn/GB/1027/8919115.html | archive-date=2013-05-21 | dead-url=no }}</ref>。2008年10月，《中长期铁路网调整规划》经国务院常务会议批准，正式颁布实施。这是该规划自2004年修订以来的最新调整方案。根据调整规划，2020年全国铁路营业里程规划目标由10万公里调整为12万公里以上，其中高速铁路由1.2万公里调整为1.6万公里<ref>{{cite news | url=http://news.xinhuanet.com/newscenter/2008-11/27/content_10422275_1.htm | title=《中长期铁路网规划（2008年调整）》方案颁布实施 | work=新华网 | date=2008-11-27 | author=齐中熙 | access-date=2016-10-31 | archive-url=https://web.archive.org/web/20161031152953/http://news.xinhuanet.com/newscenter/2008-11/27/content_10422275_1.htm | archive-date=2016-10-31 | dead-url=no }}</ref>。铁路总投资将在3.5万亿元人民币以上，超过1949年以来铁路投资总和。

2008年2月，铁道部和[[中华人民共和国科技部|科技部]]签署了《中国高速列车自主创新联合行动计划》，刘志军和[[万钢|万钢]]出席仪式<ref>{{cite web|url=http://www.most.gov.cn/tpxw/200802/t20080227_59350.htm|title=科技部、铁道部联合签署《中国高速列车自主创新联合行动计划合作协议》|date=2008-02-27|publisher=中华人民共和国科技部|access-date=2010-05-28|archive-url=https://web.archive.org/web/20171027055512/http://www.most.gov.cn/tpxw/200802/t20080227_59350.htm|archive-date=2017-10-27|dead-url=no}}</ref>。该计划提出中国将研制符合京沪高速铁路运营需求的时速350公里及以上等级的高速列车<ref name="kjrb" />。包括[[中科院|中科院]]、[[清华大学|清华大学]]、[[北京大学|北京大学]]在内的近30家国内科研院所与高校、近50家骨干企业组成产学研用自主创新联合体，参与其中<ref>{{cite web | url=http://www.most.gov.cn/kjbgz/201106/t20110609_87419.htm | title=万钢部长考察调研京沪高速铁路科技创新工作 | publisher=中华人民共和国科技部 | work=2011-06-10 | accessdate=2013-07-21 | archive-url=https://web.archive.org/web/20131014173955/http://www.most.gov.cn/kjbgz/201106/t20110609_87419.htm | archive-date=2013-10-14 | dead-url=no }}</ref>。经过两年多的努力，第一种由中国研制的[[CRH380A型电力动车组|CRH380A型电力动车组]]于2010年5月在长春下线<ref>{{cite news|url=http://jlrbszb.chinajilin.com.cn/html/2010-06/22/content_629631.htm|title=速度的神话——世界最快列车CRH380A面世前后|publisher=吉林日报|date=2010-06-22|deadurl=yes|archiveurl=https://web.archive.org/web/20131002034938/http://jlrbszb.chinajilin.com.cn/html/2010-06/22/content_629631.htm|archivedate=2013年10月2日|df=}}</ref><ref name=CRH380A>{{cite journal|title=First Chinese designed HS train breaks cover|journal=International Railway Journal|year=2010|issue=09|pages=69-70}}</ref>。2010年12月3日在[[京沪高铁|京沪高铁]]先导段试验时，刘志军亲自试乘，并在驾驶室督阵，要求驾驶员冲到极限速度并长时间保持<ref name="woan"/>。最终列车创下486.1公里/小时的世界铁路运营试验最高速度<ref>{{cite news|url=http://news.xinhuanet.com/politics/2010-12/03/c_12844458.htm|title=时速486.1公里！中国再度演绎“高铁奇迹”|date=2010-12-03|publisher=新华网|access-date=2013-07-21|archive-url=https://web.archive.org/web/20101206063753/http://news.xinhuanet.com/politics/2010-12/03/c_12844458.htm|archive-date=2010-12-06|dead-url=no}}</ref>。

2010年12月，[[世界高速铁路大会|世界高速铁路大会]]首次在中国召开。刘志军在大会上称，中国高速铁路的营业里程已经达到7531公里，是全世界高铁运营里程最长、在建规模最大的国家<ref>{{cite web | url=http://news.xinhuanet.com/politics/2010-12/07/c_12855942.htm | title=铁道部：中国高速铁路运营和在建规模居世界第一 | work=新华网 | date=2010-12-07 | accessdate=2013-07-21 | archive-url=https://web.archive.org/web/20101210155025/http://news.xinhuanet.com/politics/2010-12/07/c_12855942.htm | archive-date=2010-12-10 | dead-url=no }}</ref>。

===行事风格===
刘志军被称为“站在火车头上的部长”，每逢关键时候，刘都要登乘机车现场指挥，创造了连续几十个小时检查提速线路的纪录<ref name="cnr">{{cite web | url=http://www.cnr.cn/09zt/bzftl/ftl/200909/t20090911_505466179.html | title=铁道部刘志军:站在火车头上的部长 | publisher=中广网 | date=2009-09-11 | access-date=2013-07-21 | archive-url=https://web.archive.org/web/20150504013708/http://www.cnr.cn/09zt/bzftl/ftl/200909/t20090911_505466179.html | archive-date=2015-05-04 | dead-url=no }}</ref>。2010年12月3日，刘亲自添乘[[和谐号CRH380A型电力动车组|CRH380A-6041L]]参加在[[京沪高铁|京沪高铁]]进行的冲高试验，此次试验创下了世界正常营运编组列车最高试验速度记录486.1km/h<ref>{{cite web|author1=新闻联播|title=京沪高速线CRH380AL冲高486.1km/h+立鸡蛋|url=https://www.bilibili.com/video/av18381889?from=search&seid=3667137603372688193|access-date=2018-05-09|archive-url=https://web.archive.org/web/20180510050433/https://www.bilibili.com/video/av18381889?from=search&seid=3667137603372688193|archive-date=2018-05-10|dead-url=no}}</ref>。[[中国工程院|中国工程院]]院士[[王梦恕|王梦恕]]称，新线路开通后，刘敢坐到司机的位置上，一出问题他最先倒霉，所以哪个施工单位也不敢乱来<ref>{{cite news|url=http://www.ycwb.com/news_special/2012-03/07/content_3737817.htm|title=王梦恕：不能因7·23事故否定高铁，高铁没乱花钱|publisher=羊城晚报|date=2012-03-07|access-date=2013-07-21|archive-url=https://web.archive.org/web/20130714080454/http://www.ycwb.com/news_special/2012-03/07/content_3737817.htm|archive-date=2013-07-14|dead-url=no}}</ref>。刘志军的同事称：“在列车所有的创纪录实验中，为了测试安全性，刘部长几乎每次都在驾驶室内亲自指挥。作为一个部长，能够用自己的生命，去检验高铁的安全性，的确让人感动。”<ref name="woan"/>

刘志军强硬的行事风格为他招来非议。一位与刘志军相识20多年的老铁路人表示，刘志军是个比较仗义的人，敢于大刀阔斧做事敢于承担责任，“铁路系统臃肿懒散，刘志军如果不够霸气，很多政策就难以推行下去。”但这种做事风格或许是推动铁路进步的必要手段，“他让你什么时候出成果就什么时候出，没有理由的。”<ref name="woan"/>

刘志军亦被称为[[工作狂|工作狂]]。刘志军声称，自己从在铁路工作起，就没有在家过一天[[春节|春节]]<ref name="cnr"/>。刘志军上任铁道部长后，[[北京市|北京市]][[复兴路|复兴路]]10号的铁道部大楼里有时会出现夜半依然灯火通明的加班场景。刘志军的同事称，刘志军处理问题雷厉风行十分强势。“他有时候半夜翻材料发现问题，就把相关部门负责人叫过去开会，马上研究解决思路，拿出方案，定出时间。”<ref name="woan"/>“家就在铁道部附近，每天早上6点多从家里出来，经常工作至深夜。”<ref name="shuangmian">{{cite web|url=http://www.chinatimes.cc//pages/moreInfo1.htm?id=127785|title=双面刘志军|author=公培佳, 张智|publisher=华夏时报|date=2013-04-12|deadurl=yes|archiveurl=https://web.archive.org/web/20130417100154/http://www.chinatimes.cc/pages/moreInfo1.htm?id=127785|archivedate=2013-04-17}}</ref>

刘志军相信[[风水學|风水學]]，在一些工程开工前，会邀请[[风水師|风水師]]选定奠基时间，以求吉利<ref>{{cite web|url=http://news.xinhuanet.com/politics/2013-05/17/c_124725545.htm|title=历数迷恋风水的中国贪官|publisher=新华网|accessdate=2013年5月17日|archive-url=https://web.archive.org/web/20130611211202/http://news.xinhuanet.com/politics/2013-05/17/c_124725545.htm|archive-date=2013年6月11日|dead-url=no}}</ref>。刘志军在会见[[律师|律师]]时称：每次开工都会找人按照[[黄历|黄历]]选一个好日子。他甚至也认为这个事情很奇怪，不信不行，先前没有选日子，开工的时候就会下雨，举行仪式还得临时搭棚子。选了日子的时候，一次都没下过，有时候明明还下着雨，仪式要举行的时候就停了<ref name="xwzk"/>。

=== 审判 ===
2013年4月10日，刘志军受贿、滥用职权一案，由北京市人民检察院第二分院向[[北京市第二中级人民法院|北京市第二中级人民法院]]提起[[公诉|公诉]]<ref>{{cite web |url=http://news.xinhuanet.com/legal/2013-04/10/c_115336978.htm |title=刘志军受贿、滥用职权案提起公诉 |publisher=新华网 |date=2013-04-10 |accessdate=2013-07-20 |archive-url=https://web.archive.org/web/20130413092834/http://news.xinhuanet.com/legal/2013-04/10/c_115336978.htm |archive-date=2013-04-13 |dead-url=no }}</ref>。案件于6月9日开庭审理<ref name="yishen">{{cite web |url = http://news.xinhuanet.com/legal/2013-06/09/c_116102677.htm |title = 刘志军受贿、滥用职权案一审开庭 |publisher = 新华网 |work = 2013-06-09 |accessdate = 2013-07-20 |archive-url = https://web.archive.org/web/20130612225542/http://news.xinhuanet.com/legal/2013-06/09/c_116102677.htm |archive-date = 2013-06-12 |dead-url = no }}</ref>。检察院指控，被告人刘志军利用职务便利，为[[邵力平|邵力平]]等11人提供帮助，先后非法收受财物共计人民币6460.54万元；应以受贿罪、滥用职权罪追究刘志军的刑事责任<ref name="yishen" />。检方亦提出，刘志军主动坦白了司法机关尚未掌握的其受贿的大部分事实<ref name="renhe" />，滥用职权造成的损失和受贿赃款基本已挽回，可从轻处罚<ref name="xjb-a">{{cite news |url = http://epaper.bjnews.com.cn/html/2013-06/10/content_438757.htm?div=-1 |title = 刘志军受审，承认受贿滥用职权 |work = 新京报 |date = 2013-06-10 |pages = A08 |access-date = 2013-08-20 |archive-url = https://web.archive.org/web/20130616110518/http://epaper.bjnews.com.cn/html/2013-06/10/content_438757.htm?div=-1 |archive-date = 2013-06-16 |dead-url = no }}</ref>。

因刘志军不愿自行委托[[辩护人|辩护人]]，北京市法律援助中心指定律师为其提供辩护<ref name="chinanews">{{cite web |url = http://www.chinanews.com/fz/2013/06-09/4917429.shtml |title = 刘志军案在京开庭，对指控无异议，辩护律师法院指定 |work = 中国新闻网 |date = 2013-06-09 |accessdate = 2013-07-21 |archive-url = https://web.archive.org/web/20130610055621/http://www.chinanews.com/fz/2013/06-09/4917429.shtml |archive-date = 2013-06-10 |dead-url = no }}</ref>。著名律师[[钱列阳|钱列阳]]为刘志军作罪轻辩护<ref>{{cite news |url=http://www.bjnews.com.cn/news/2013/06/07/267519.html |title=刘志军案周日上午开审，律师将做罪轻辩护 |work=新京报 |date=2013-06-07 |accessdate=2013-07-20 |author=张玉学, 张媛 |archive-url=https://web.archive.org/web/20130717081650/http://www.bjnews.com.cn/news/2013/06/07/267519.html |archive-date=2013-07-17 |dead-url=no }}</ref>。在检方所指控6460.54万当中，两笔共4900万的钱款成为控辩双方存在不同意见的焦点，两笔钱款为刘志军授意丁书苗为其疏通关系解决相关事宜而支出<ref name="dawen">{{cite news |url = http://news.xinhuanet.com/2013-07/08/c_116446520.htm |title = 刘志军受贿、滥用职权案一审宣判后审判长答记者问 |work = 新华网 |date = 2013-07-08 |accessdate = 2013-07-20 |archive-url = https://web.archive.org/web/20160822205047/http://news.xinhuanet.com/2013-07/08/c_116446520.htm |archive-date = 2016-08-22 |dead-url = no }}</ref>。钱列阳认为，刘志军并未直接将该款收为己有，该行为不应构成受贿罪。但法院最终认定其为受贿<ref name="dawen" />。

刘志军对被指控的事实和罪名都没有提出异议<ref name="chinanews" />，并且表示无论结果如何都不会上诉。<ref name="renhe">{{cite news |url = http://news.xinhuanet.com/politics/2013-07/08/c_124970928.htm |title = 刘志军案今日宣判，称任何结果都不上诉 |publisher = 新京报 |date = 2013-07-08 |access-date = 2013-07-08 |archive-url = https://web.archive.org/web/20130712221304/http://news.xinhuanet.com/politics/2013-07/08/c_124970928.htm |archive-date = 2013-07-12 |dead-url = no }}</ref><ref>{{cite news |url = http://news.qq.com/a/20130610/000797.htm |title = 刘志军站立出庭痛哭悔罪 应为中国梦做更多贡献 |author = 裴晓兰、孙思娅 |date = 2013-06-10 |work = [[京华时报|京华时报]] |accessdate = 2013-07-20 |archive-url = https://web.archive.org/web/20130614071607/http://news.qq.com/a/20130610/000797.htm |archive-date = 2013-06-14 |dead-url = no }}</ref>

2013年7月8日，北京市第二中级人民法院一审判处刘志军[[死刑缓期执行|死刑，缓期二年执行]]，并没收全部个人财产
<ref>{{cite web |url = http://news.xinhuanet.com/legal/2013-07/08/c_116442565.htm |title = 原铁道部部长刘志军受贿、滥用职权案一审宣判 |publisher = 新华网 |date = 2013-07-08 |accessdate = 2013-07-20 |archive-url = https://web.archive.org/web/20130712102714/http://news.xinhuanet.com/legal/2013-07/08/c_116442565.htm |archive-date = 2013-07-12 |dead-url = no }}</ref>。法院经审理查明：

{{quote|1986年至2011年，刘志军在担任郑州铁路局武汉铁路分局党委书记、分局长、郑州铁路局副局长、沈阳铁路局局长、原铁道部运输总调度长、副部长、部长期间，利用职务便利，为[[邵力平|邵力平]]、[[丁羽心|丁羽心]]等11人在职务晋升、承揽工程、获取铁路货物运输计划等方面提供帮助，先后非法收受上述人员给予的财物共计折合人民币6460万余元；刘志军在担任铁道部部长期间，违反规定，徇私舞弊，为丁羽心及其与亲属实际控制的公司获得铁路货物运输计划、获取经营动车组轮对项目公司的股权、运作铁路建设工程项目中标、解决企业经营资金困难提供帮助，使丁羽心及其亲属获得巨额经济利益，致使公共财产、国家和人民利益遭受重大损失。}}

此案与[[习近平|习近平]]上台后高调的[[反腐|反腐]]举措相联系<ref>{{cite web |url = http://www.guardian.co.uk/world/2013/jul/08/liu-zhijun-sentenced-death-corruption |title = News World news Xi Jinping Liu Zhijun, China's ex-railway minister, sentenced to death for corruption |publisher = ''The Guardian'' |date = 2013-07-08 |accessdate = 2013-07-20 |author = Kaiman, Jonathan |archive-url = https://web.archive.org/web/20130717194936/http://www.guardian.co.uk/world/2013/jul/08/liu-zhijun-sentenced-death-corruption |archive-date = 2013-07-17 |dead-url = no }}</ref>，被视为对新领导层打击腐败决心的考验<ref name="FT-1">{{cite web |url = http://www.ft.com/cms/s/0/f580534e-e77b-11e2-8a57-00144feabdc0.html |title = China ex-rail chief given suspended death sentence for corruption |work = ''The Financial Times'' |date = 2013-06-08 |accessdate = 2013-07-20 |author = Hille, Kathrin |archive-url = https://web.archive.org/web/20130712084117/http://www.ft.com/cms/s/0/f580534e-e77b-11e2-8a57-00144feabdc0.html |archive-date = 2013-07-12 |dead-url = no }}</ref>。然而根据《[[中华人民共和国刑法|中华人民共和国刑法]]》第五十条，「判处死刑缓期执行的，在死刑缓期执行期间，如果没有故意犯罪，二年期满以后，减为[[无期徒刑|无期徒刑]]」。有媒体分析，刘志军最早可能服刑27年在87岁时候出狱，若考虑60岁后可获“[[保外就医|保外就医]]”的情况，可能在9年后即保外就医<ref>{{cite web |url = http://www.cna.com.tw/News/aCN/201307100330-1.aspx |title = 貪官劉志軍，關9年換保外就醫 |publisher = 中央社 |date = 2013-07-10 |accessdate = 2013-07-21 |archive-url = https://web.archive.org/web/20130715011115/http://www.cna.com.tw/News/aCN/201307100330-1.aspx |archive-date = 2013-07-15 |dead-url = no }}</ref>。
