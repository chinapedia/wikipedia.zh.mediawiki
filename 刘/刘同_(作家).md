'''刘同'''（{{bd|1981年|2月27日|}}），曾用名'''刘童'''，出生于[[湖南省|湖南]][[郴州市|郴州]]，毕业于[[湖南师范大学|湖南师范大学]]中文系，中国当代[[作家|作家]]，[[光线影业有限公司|光线影业]]副总裁。2013年出版的励志小说《[[谁的青春不迷茫|谁的青春不迷茫]]》在中国大陆销量超过200万册<ref>{{Cite web|url = http://news.163.com/14/1010/15/A872CTT800014AED.html|title = 刘同书写青春 三月破百万|accessdate = 2015-08-02|author = |date = 2014-10-10|publisher = 网易新闻|language = zh-cn}}</ref>，并获得了第八届[[中国作家富豪榜|中国作家富豪榜]]“年度最佳励志书”，2014年出版《[[你的孤独,虽败犹荣|你的孤独，虽败犹荣]]》持续热销，同年，刘同跻身[[中国作家富豪榜|中国作家富豪榜]]第四。其“谁的青春不迷茫”系列作品被评价为“影响了85后、90后甚至00后”，被誉为是当下“最懂年轻人的作家”<ref>{{Cite web|url = http://book.ifeng.com/yeneizixun/detail_2014_12/22/1016208_0.shtml|title = 跨界独揽作家榜两项大奖 刘同成最懂年轻人作家|accessdate = 2015-08-02|author = |date = 2014-12-22|publisher = 凤凰网读书|language = zh-cn}}</ref>。2016年4月22日，由刘同的同名作品改编的电影《谁的青春不迷茫》上映。<ref>{{Cite web|url = http://news.163.com/16/0502/00/BM163T7N00014AED.html|title = 刘同：哭更能把握人性|accessdate = 2016-10-17|author = |date = 2016-05-02|publisher = 网易新闻|language = zh-cn}}</ref>。

== 生平 ==
*2001-2004年进入湖南电视台工作参与全国联播电台节目《城市日记》嘉宾主持，并以嘉宾身份参与接受央视、浙江卫视、旅游卫视等各大媒体访谈<ref>{{Cite web|url = http://talk.ifeng.com/professor/detail_2014_04/14/35748010_0.shtml|title = 刘同|accessdate = 2015-08-02|author = |date = 2014年4月14日|publisher = 凤凰讲堂|language = zh-cn}}</ref>。

*2011年10月，任[[光线传媒|光线传媒]]资讯事业部副总裁<ref>{{Cite web|url = http://www.wm23.com/wiki/48553.htm|title = 光线传媒刘同|accessdate = 2015-08-02|author = |date = |publisher = |language = zh-cn}}</ref>。

*2014年10月，担任光线影业副总裁，任大陆电影《[[親愛的_(2014年電影)|亲爱的]]》、《[[同桌的你_(电影)|同桌的你]]》制片人<ref>{{Cite web|url = http://yule.sohu.com/20140515/n399600525.shtml?recsyssohu=2|title = 《同桌的你》破四亿 制片人刘同欲再筹新电影|accessdate = 2015-08-02|author = |date = 2014年5月15日|publisher = 搜狐娱乐|language = zh-cn}}</ref>。
