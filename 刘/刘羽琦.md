'''刘羽琦'''（{{lang-en|'''Izumi Liu'''}}，{{bd|1987年|7月18日|||catIdx=L}}）江苏人，中国大陆演员、模特。

==生平==
1987年，刘羽琦出生在[[江苏省|江苏省]][[徐州市|徐州市]]。刘羽琦早年从事[[乒乓球|乒乓球]]运动，曾在[[上海市|上海市]]和[[广州市|广州市]]生活了很长一段时间。15岁时，刘羽琦北漂，在[[北京市|北京市]]开辟事业，因其身高1.71米而在模特界出道。<ref>{{cite web |url=http://ent.ifeng.com/idolnews/mainland/detail_2010_12/14/3498186_0.shtml |title=宅男女神刘羽琦全新造型 唯美气质美轮美奂 |accessdate=2010 |publisher=凤凰网 |work= |language=zh-hans}}</ref>本身擁有34D的傲人[[巨乳|巨乳]]。<ref>[https://star.ettoday.net/news/49510 《夜蒲2》劉羽琦當街脫光34D導彈胸任親任摸！] ETNEWS星光雲 2012-05-25.</ref>

2006年，刘羽琦遇到星探[[季忠平|季忠平]]，签约瀛凯文化传媒北京有限公司，开始了自己的演艺生涯。同年，她被[[中国中央电视台|中国中央电视台]]选为“徐州十大城市形象大使”。<ref>{{cite web |url=http://js.xhby.net/system/2013/10/31/019116534.shtml |title=刘羽琦：“徐州MM”是我的事业起点 |accessdate=2013 |publisher=新浪网 |work= |language=zh-hans}}</ref>2011年，刘羽琦签约[[海润影视|海润影视]]。

2012年，刘羽琦因出演香港電影《[[喜爱夜蒲2|喜爱夜蒲2]]》而走红。<ref>{{cite web |url=http://www.860714.net/article-453-1.html |title=中国模特刘羽琦借《喜爱夜蒲2》成功上位  |accessdate=2012 |publisher= |work= |language=zh-hans}}</ref>
