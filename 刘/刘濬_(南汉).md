'''刘濬'''（{{bd|9世纪||10世纪|catIdx=L}}），[[表字|字]]'''伯深'''，[[中国|中国]][[五代十国|五代十国]]时期[[政权|政权]][[南汉|南汉]]官员，一度担任[[宰相|宰相]]。

== 家世 ==
刘濬生年不详。父[[刘崇望|刘崇望]]，晚唐名臣，[[唐昭宗|唐昭宗]]年间拜相。包括伯父[[刘崇龟|刘崇龟]]在内的几位叔伯也是名臣。<ref name=BSH10>[https://archive.org/stream/02079315.cn#page/n2/mode/2up 《南汉书》卷十]</ref><ref name=BT179>[[:s:舊唐書/卷179|《旧唐书》卷一百七十九]]</ref>

唐昭宗年间，天下大乱。[[大顺_(唐朝)|大顺]]（890年—891年）初年刘崇龟为[[清海军节度使|清海军节度使]]后，刘濬为了避乱，来清海军依附刘崇龟，刘崇龟在[[乾宁|乾宁]]年间（894年—898年）过世后，他留在清海军部[[广州|广州]]。<ref name=BSH10/><ref name=NHFD65>[[:s:新五代史/卷65|《新五代史》卷六十五]]</ref><ref name=SGCQ62>[https://archive.org/stream/06072245.cn#page/n104/mode/2up 《十国春秋》卷六十二]</ref>

== 效力刘隐 ==
大约乾宁元年（894年），刘崇龟召[[刘隐|刘隐]]补右都押牙兼贺水镇使，不久又表奏刘隐为[[封州_(隋朝)|封州]][[刺史|刺史]]。<ref>[[:s:資治通鑑/卷259|《资治通鉴》卷二百五十九]]</ref>因而刘隐后来于[[天复_(唐朝)|天复]]元年（901年）权清海军[[留后|留后]]<ref>[[:s:資治通鑑/卷262|《资治通鉴》卷二百六十二]]</ref>并于[[天祐_(唐朝)|天祐]]元年（904年）被官方任为节度使后，<ref>[[:s:資治通鑑/卷265|《资治通鉴》卷二百六十五]]</ref>为表达对刘崇龟的感激，对刘濬加礼，任命他为幕僚，<ref name=NHFD65/>与[[李殷衡|李殷衡]]、[[倪曙|倪曙]]、[[周杰|周杰]]等同为宾客，经常与他们谋划秘密之事。<ref name=BSH10/><ref name=SGCQ62/>

== 效力刘䶮 ==
[[后梁|后梁]][[贞明_(朱友贞)|贞明]]三年（917年），原为后梁封臣的刘隐的弟弟和继承人[[刘䶮|刘䶮]]称帝建立大越国，后改汉，故史称南汉。<ref>[[:s:資治通鑑/卷270|《资治通鉴》卷二百七十]]</ref>他任刘濬为宗正卿兼工部侍郎。[[乾亨_(南汉)|乾亨]]年间（917年—925年），刘䶮曾调军屯[[潮州|潮州]]，准备伺机攻打东北邻国[[闽国|闽国]]。刘濬认为不可，与宰相[[杨洞潜|杨洞潜]]商议，杨洞潜认同他，意图劝阻刘䶮，但没有成功。八年（924年），刘䶮领军在闽国的[[汀州|汀州]]和[[漳州|漳州]]之间屯扎，被闽军所攻败走。<ref name=SGCQ62/><ref>[[:s:資治通鑑/卷273|《资治通鉴》卷二百七十三]]</ref>他因而很后悔没有听从杨洞潜和刘濬的劝说。<ref name=BSH10/>

[[大有_(年號)|大有]]九年（936年），杨洞潜卒，<ref>此按《南汉书·刘濬传》。但<杨洞潜传>称其卒于大有八年（935年）。见[http://archive.org/stream/02079314.cn#page/n40/mode/2up 《南汉书》卷九]</ref>刘䶮升刘濬为[[中书侍郎|中书侍郎]]，授[[同中书门下平章事|同中书门下平章事]]，代杨洞潜为相。<ref name=BSH10/><ref name=ZZTJ280>[[:s:資治通鑑/卷280|《资治通鉴》卷二百八十]]</ref>刘濬为官清简有操持，常劝刘䶮爱民息兵，言辞婉切。在宰相任上过世，时间不详。<ref name=BSH10/>鉴于刘䶮的继承者们年间没有他的记载，他可能比于十五年（942年）过世的刘䶮更早过世。<ref>[[:s:資治通鑑/卷283|《资治通鉴》卷二百八十三]]</ref>子孙在南汉也多有显达者。<ref name=SGCQ62/>

== 评价 ==
*《[[十国春秋|十国春秋]]》论曰：（倪）曙、[[何泽|（何）泽]]用文采显，'''（刘）濬'''以清执称，声施百粤，亦庶几名臣选焉。（倪曙、何泽以文采显名，'''刘濬'''以清廉有操守为人所称，声名遍及粤地，也或许可以是名臣之选。）
