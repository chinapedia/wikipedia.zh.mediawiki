'''刘滋'''（{{bd|729年||794年|11月19日|catIdx=L}}<ref name=BT13>[[:s:旧唐书/卷13|《旧唐书》卷一十三]]</ref>），[[表字|字]]'''公茂'''，[[中国|中国]][[唐朝|唐朝]][[官员|官员]]，[[唐德宗|唐德宗]]年间短期[[拜相|拜相]]。

== 家世 ==
刘滋出自[[彭城刘氏|彭城从亭里刘氏]]，祖父[[刘知几|刘知几]]为[[唐玄宗|唐玄宗]]年间[[左散骑常侍|左散骑常侍]]和史官，父[[刘贶|刘贶]]继为史官。<ref name=BT136>[[:s:旧唐书/卷136|《旧唐书》卷一百三十六]]</ref><ref name=NBT132>[[:s:新唐书/卷132|《新唐书》卷一百三十二]]</ref>

== 拜相前 ==
刘滋因祖先而[[蔭官|蔭官]]，年轻时就被授[[太子正字|太子正字]]。<ref>因任职时间不详，刘滋效力哪一位太子尚待考。</ref>历任[[涟水|涟水]]令等职。[[吏部侍郎|吏部侍郎]][[杨绾|杨绾]]荐刘滋堪为[[谏官|谏官]]，刘滋被拜为[[左补阙|左补阙]]，改[[太常卿|太常卿]]，又复为左补阙。<ref>杨绾上任为吏部侍郎的时间不详，仅知迟于[[唐代宗|唐代宗]][[广德|广德]]二年（764年），[[大历|大历]]五年（770年）他被调任，可见此事必定发生在此期间。见[[:s:旧唐书/卷119|《旧唐书》卷一百一十九]]。</ref>后来他为照顾当时在东都[[洛阳|洛阳]]的母亲而辞职，在洛阳期间，河南尹[[李廙|李廙]]署奏他为功曹参军。丧母后，他离职[[守丧|守丧]]，期满后，迁屯田员外郎，转司勋员外郎，判[[南曹|南曹]]（即负责审核吏部、兵部下级官吏的档案和政绩并向上级呈报）。他勤于公务，严格守法。后迁司勋郎中，屡次升迁后官拜[[给事中|给事中]]。<ref name=BT136/><ref name=NBT132/>

唐德宗[[建中_(唐朝)|建中]]四年（783年），因[[泾原兵变|泾原军兵变]]推[[太尉|太尉]][[朱泚|朱泚]]为首，德宗被迫逃离京城[[长安|长安]]，到[[奉天|奉天]]。<ref>[[:s:資治通鑑/卷228|《资治通鉴》卷二百二十八]]</ref>刘滋随驾，转太常少卿，掌礼仪。[[兴元|兴元]]元年（784年），改权知吏部侍郎。<ref name=LZ>[[:s:刘滋崔造齐映平章事制|陆贽《刘滋崔造齐映平章事制》]]</ref>当时因京师遭劫和蝗旱灾害，谷价上涨，南方如江淮、岭南等地的待选官员不能按例来长安参加[[科举|科举]]，德宗派刘滋去[[洪州|洪州]]典选他们，时人认为刘滋称职。<ref name=BT136/><ref name=NBT132/>

== 拜相及拜相后 ==
[[贞元_(唐朝)|贞元]]二年（786年）正月，充[[左散骑常侍|左散骑常侍]]，守本官，授[[同中书门下平章事|同中书门下平章事]]，与中书舍人[[齐映|齐映]]、给事中[[崔造|崔造]]同拜宰相。<ref name=LZ/>德宗因崔造在朝廷敢言而用之，刘滋為人端莊、文雅莊重、沈默寡言，幾乎沒甚麼上奏議論，只是謙卑保守，廉洁谨慎而已，与齐映兩人處理公事，多禮讓[[崔造|崔造]]，故政事多由崔造决定。

德宗令宰相去[[尚書|尚書]]六曹分別斷事，刘滋去了吏部、礼部。<ref>[[:s:旧唐书/卷130|《旧唐书》卷一百三十]]</ref>守[[抚州|抚州]][[刺史|刺史]][[戴叔伦|戴叔伦]]劝齐映、刘滋以[[中书省|中书省]]、门下省名义，不顾资历、远近、尊卑任命[[县令|县令]]、录事参军事之职，以免私吞钱粮，齐映、刘滋等重视其言。<ref>[[:s:新唐书/卷143|《新唐书》卷一百四十三]]</ref>当年，[[唐顺宗|皇太子]]为[[王皇后_(唐德宗)|王皇后]]服丧，左补阙[[穆质|穆质]]认为应以礼服丧，期满除丧服，太常博士[[柳冕|柳冕]]、[[张荐|张荐]]、[[畅当|畅当]]、[[李吉甫|李吉甫]]等则奏称皇后下葬后，皇太子即可除丧服，内心服丧三年即可。刘滋、齐映召畅当等，提出太子不应该以衰服侍膳直至下葬，应该令群臣服齐衰三十日除服，并以为常例，于是请求如[[南朝宋|南朝宋]]、[[南朝齐|南朝齐]]皇后为父母服丧三十日除服之礼，入谒时服「墨惨」，回宫后服「衰麻」。宰相们命太常卿[[郑叔则|郑叔则]]草奏，德宗认可柳冕所议，而穆质坚执依古礼，刘滋与齐映参酌群议，还是请求依郑叔则之议，皇帝最後聽從之。<ref>[[:s:旧唐书/卷149|《旧唐书》卷一百四十九]]</ref><ref>[[:s:新唐书/卷200|《新唐书》卷二百]]</ref>

十二月，浙江东西节度使[[韩滉|韩滉]]被加度支诸道转运盐铁等使后，屡次诬奏与自己有宿怨的右丞判度支[[元琇|元琇]]，致其被贬[[雷州|雷州]]司户，责罚太重，朝廷官員們都以为冤枉，多有私下議論者。尚书左丞[[董晋_(唐朝)|董晋]]对刘滋、齐映说元琇在国难时有功，如今被贬而不知道罪名，是权臣滥用刑罚，动摇人心，痛惜其遭遇，请他们奏请三司详断。但刘滋、齐映只是自我譴責，<ref>[[:s:旧唐书/卷129|《旧唐书》卷一百二十九]]</ref>卻不聽他的話。<ref>[[:s:新唐书/卷126|《新唐书》卷一百二十六]]</ref>

三年（787年）正月，刘滋被罢相，<ref>[[:s:新唐书/卷7|《新唐书》卷七]]</ref>仅守本官左散骑常侍。<ref>[[:s:旧唐书/卷12|《旧唐书》卷一十二]]</ref><ref>[[:s:資治通鑑/卷232|《资治通鉴》卷二百三十二]]</ref>

四年（788年），复为吏部侍郎。尚书左丞[[赵憬|赵憬]]称自己所荐的[[果州|果州]]刺史[[韦证|韦证]]因贪污被[[罷免|罷免]]，请求降低自己的考核成绩。刘滋时为校考使，认为他知错，反而升了他的考核成绩。<ref>[[:s:新唐书/卷150|《新唐书》卷一百五十]]</ref>

六年（790年）二月，迁吏部尚书。[[李晟|李晟]]被册为太尉兼[[中书令|中书令]]，按礼应由中书令读册、[[侍中|侍中]]赞礼，若中书令、侍中无人担任，则由宰相代理。宰相[[中书侍郎|中书侍郎]][[张延赏|张延赏]]与李晟有隙，削减礼数，仅用尚书[[崔汉衡|崔汉衡]]和刘滋代理。<ref>[[:s:新唐书/卷127|《新唐书》卷一百二十七]]</ref>[[窦参|窦参]]为宰相，想当[[吏部尚书|吏部尚书]]，于是刘滋被改[[刑部尚书|刑部尚书]]。<ref>窦参在贞元五年（789年）—八年（792年）期间为宰相，此次官职变动肯定发生在792年之前。见《旧唐书》卷一百三十六。</ref>

[[御史中丞|御史中丞]][[韦贞伯|韦贞伯]]劾奏刘滋先前在吏部选官太滥有疏漏导致官吏为恶，诏剥夺刘滋与侍郎[[杜黄裳|杜黄裳]]的金紫官服。刘滋博于[[经学|经学]]，善于评论，性格廉洁刻苦，嫉恨恶行，选官时谨慎，在资历材料上作假的尤其畏惧他。十年（794年）十月卒，<ref name=BT13/>赠[[陕州|陕州]]大都督，<ref name=BT136/>谥'''贞'''。<ref name=NBT132/>

== 评价 ==
*《旧唐书》
**史臣曰：（刘）滋、[[卢迈|（卢）迈]]家行修谨，临事可称，器虽龌龊，无废为君子矣。（刘滋、卢迈家风修谨，临事时的表现值得称赞，虽然为人拘谨，仍然可为君子。）
**赞曰：滋、迈之行，可以饰躬。（刘滋、卢迈的行为，可以修饰自身。）
