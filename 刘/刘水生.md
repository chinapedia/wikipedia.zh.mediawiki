'''刘水生'''（{{bd|1955年|9月||}}），[[江西|江西]][[南昌|南昌]]人，[[中华人民共和国|中华人民共和国]]官员。

==生平==
早年加入[[中国共产党|中国共产党]]。[[中共中央党校|中共中央党校]]研究生院经济管理专业毕业<ref name=xinxi/>。长期在[[人民大会堂|人民大会堂]]管理局工作，曾任人民大会堂管理局局长<ref>{{Cite web |url =http://news.163.com/16/0224/20/BGK7L01400014SEH.html  |title =解密人民大会堂管理局:多任局长亲历"重大事件"  |author =  |date = 2016-02-24 |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref>。2011年4月，任[[第十一届全国人大常委会|第十一届全国人大常委会]]副秘书长、机关党组成员。2013年，当选[[第十二届全国人民代表大会|第十二届全国人民代表大会]]江西地区代表，并担任[[第十二届全国人大常委会|第十二届全国人大常委会]]副秘书长、机关党组成员<ref name=xinxi>{{cite news|title=全国人大代表信息-刘水生|url=http://www.npc.gov.cn/delegate/viewDelegate.action?dbid=121110|accessdate=2013-04-05|newspaper=全国人大网|deadurl=yes|archiveurl=https://web.archive.org/web/20130713114651/http://www.npc.gov.cn/delegate/viewDelegate.action?dbid=121110|archivedate=2013-07-13}}</ref>。

2016年2月，因违纪被责令辞去第十二届全国人大代表，被免去全国人大常委会副秘书长职务，其第十二届[[全国人民代表大会华侨委员会|全国人民代表大会华侨委员会]]委员职务相应终止<ref>{{cite news|title=第十二届全国人民代表大会常务委员会代表资格审查委员会关于个别代表的代表资格的报告（2016年2月26日第十二届全国人民代表大会常务委员会第十九次会议通过）|url=http://www.npc.gov.cn/wxzl/gongbao/2016-06/03/content_1993633.htm|accessdate=2016-07-25|newspaper=中国人大网}}</ref><ref>{{cite web|accessdate=2016-02-23|url=http://www.thepaper.cn/newsDetail_forward_1434977|title=全国人大常委会副秘书长刘水生辞去第十二届全国人大代表职务|publisher=澎湃新闻}}</ref>。
