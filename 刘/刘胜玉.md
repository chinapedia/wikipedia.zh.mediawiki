'''刘胜玉'''（{{bd|1945年|2月|catIdx=L刘}}），[[山东省|山东省]][[荣成市|荣成市]]人，[[中国共产党|中国共产党]]及[[中华人民共和国|中华人民共和国]]官员，[[天津市人民代表大会常务委员会|天津市人大常委会]]主任<ref name=liushy>{{Cite web |url =http://politics.people.com.cn/GB/shizheng/252/9667/9684/6569100.html  |title = 刘胜玉同志简历 |author =  |date =  |publisher = 人民网 |language =  |accessdate = 2018-01-27 |quote =  }}</ref>。

== 生平 ==
1962年9月，在北京化工四厂参加工作，历任工人、厂团委书记。1966年2月加入中国共产党。“[[文革|文革]]”中下放劳动。1970年到1974年在[[北京大学|北京大学]]哲学系学习，后任北京化工四厂党委副书记、厂长。1979年起，历任西藏自治区团委副书记，中央直属机关团委副书记、书记。1984年起，历任[[中共中央直属机关委员会|中央直属机关党委]]常委、办公室主任，[[中共中央直属机关事务管理局|中央直属机关事务管理局]]局长、中央直属机关党委常委、[[中共中央办公厅|中央办公厅]]机关党委常委<ref name=liushy/>。

1993年4月，任[[中共中央党校|中央党校]]副校长。2000年4月，任[[中共天津市委|中共天津市委]]副书记。2007年6月，不再担任市委副书记。2006年1月20日，在天津市第十四届人大四次会议上当选为[[天津市人大常委会|天津市人大常委会]]主任。2008年1月28日，在天津市第十五届人民代表大会第一次全体会议上当选为天津市人大常委会主任<ref name=liushy/>。2011年1月，不再担任天津市人大常委会主任。2008年，当选为[[第十一届全国人大代表|第十一届全国人大代表]]。2013年，当选为[[第十二届全国人大代表|第十二届全国人大代表]]<ref>{{cite news|title=全国人大代表信息-刘胜玉|url=http://www.npc.gov.cn/delegate/viewDelegate.action?dbid=2008120068|accessdate=2013-03-06|newspaper=全国人大网}}</ref>，并当选第十二届[[全国人民代表大会民族委员会|全国人民代表大会民族委员会]]副主任委员<ref>{{Cite web |url = http://www.npc.gov.cn/wxzl/gongbao/2013-07/18/content_1810961.htm |title =第十二届全国人民代表大会民族委员会主任委员、副主任委员、委员名单（28名）  |author =  |date = 2013-03-16 |publisher = 中国人大网 |language =  |accessdate =  |quote =  }}</ref>。
