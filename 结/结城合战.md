'''结城合战'''是发生在[[永享|永享]]12年（1440年）[[室町幕府|室町幕府]]与[[关东地区|关东地区]]势力的一次战争。室町幕府第六代将军足利义教一直试图压制[[关东公方|关东公方]]足利持氏。后来[[足利持氏|足利持氏]]起兵反抗，[[关东管领|关东管领]][[上杉宪实|上杉宪实]]奉幕府命前往平叛。结城当地大名[[结城氏朝|结城氏朝]]举兵援助叛军。幕府军与叛军在结城展开战斗，最终以幕府军攻陷结城为终。

== 经过 ==
自永享7年(1435年)开始的[[關東公方|镰仓公方]][[足利持氏|足利持氏]]和[[關東管領|关东管领]][[上杉宪实|上杉宪实]]的对立在永享10年(1438)年终于发展为[[永享之乱|永享之乱]]，并以持氏兵败自尽，镰仓府灭亡告终。随后[[室町幕府|室町幕府]]六代将军[[足利义教|足利义教]]任命自己的儿子为新一任镰仓公方，在这一举动的刺激下，永享12年(1440年)持氏的残党，[[下总国|下总国]]的结城氏朝·[[结城持朝|持朝]]父子拥立持氏的遗孤起兵反叛室町幕府。

幕府随即派兵镇压，任命[[上杉清方|上杉清方]]为总大将，下辖[[今川範忠|今川範忠]]、[[小笠原政康|小笠原政康]]等人，并联合了关东的[[國人眾|国人众]]一同进军。永享12年7月29日，幕府军包围了氏朝坚守下的结城城。最终结城氏朝兵败身死。持氏的遗孤中，[[春王丸|春王丸]]和[[安王丸|安王丸]]在义教的授意下被[[长尾实景|长尾实景]]在[[美浓国|美浓国]]处决，永寿王丸（后来的[[足利成氏|足利成氏]]）被送往京都。

== 战后 ==
战火之后也蔓延到了当时在镰仓公方治下的奥州。持氏的叔父[[篠川公方|篠川公方]][[足利满直|足利满直]]因为在永享之乱中叛变到了幕府一方而受到了支持结城氏的武将们的围攻，被迫自杀。自此幕府失去了对奥羽的统治，令其成为了国人纷争的修罗场。

1441年以庆祝结城合战获胜为名在京都召开庆功会的将军足利义教被家臣[[赤松满祐|赤松满祐]]暗杀，是为[[嘉吉之乱|嘉吉之乱]]的开始。

文安3年(1446年)在关东诸将的要求下持氏的遗孤成氏回归镰仓，复兴了镰仓府。但随后成氏也与管领上杉氏开始对立并引发了[[享德之亂|享德之乱]]。

== 评价 ==
结城合战是永享之乱的延续，但战斗规模比永享之乱本身还要大。现存《结城合战绘词》<ref>{{cite web|author1=不详|title=結城合戦絵詞|url=http://bunka.nii.ac.jp/heritages/detail/85010|website=https://kotobank.jp/word/%E7%B5%90%E5%9F%8E%E5%90%88%E6%88%A6%E7%B5%B5%E8%A9%9E-1213069}}</ref>记载了合战的经过。另外读本《[[南總里見八犬傳|南总里见八犬传]]》也以[[里见季基|里见季基]]·[[里见义实|义实]]父子在结城合战中与结城氏并肩作战，最终父亲赴死儿子撤退为故事开端。
