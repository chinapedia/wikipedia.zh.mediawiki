'''艋舺清水巖'''，全名'''艋舺清水巖祖師廟'''，俗稱'''艋舺祖師廟'''，主祀[[清水祖師|清水祖師]]，位於[[臺北市|臺北市]][[萬華區|萬華區]]康定路81號。為一座受[[道教|道教]]、[[台灣民間信仰|台灣民間信仰]]影響極多的[[佛教|佛教]][[祖師廟|祖師廟]]。除主祀[[宋代|宋代]]高僧清水祖師外，並配祀天上聖母[[媽祖|媽祖]]、[[關帝君|關帝君]]、[[文昌君|文昌君]]、[[地藏菩薩|地藏菩薩]]、[[魁星|魁星]]及[[福德正神|福德正神]]等眾[[神祇|神祇]]。根據日本學者的研究，從艋舺清水巖[[分香|分香]]所建的廟，計有[[新莊|新莊]][[山子腳|山子腳]]、[[景美|景美]]溪子口等。<ref>鈴木清一郎《增訂台灣舊慣習俗信仰》馮作民譯，2000年</ref>

艋舺清水巖是[[中華民國|中華民國]][[直轄市_(中華民國)|直轄市]][[臺北市文化資產|定古蹟]]，它與[[艋舺龍山寺|艋舺龍山寺]]和[[大龍峒保安宮|大龍峒保安宮]]合稱為「臺北三大廟門」<ref>[http://kids.tpml.edu.tw/sp.asp?xdurl=DR/library03_1.asp&id=22641&mp=100 請問臺北三大廟門是指那三座廟？]</ref>；也與艋舺龍山寺和[[艋舺青山宮|艋舺青山宮]]<ref>原本是艋舺新興宮（今西門町天后宮），艋舺新興宮在日治時期被拆，遷至西門町現址後，以[[艋舺青山宮|艋舺青山宮]]代之</ref>合稱為「[[艋舺|艋舺]]三大廟門」<ref>[https://www.taipei-walkingtour.tw/t-tours193 民俗的博物館─艋舺三大廟]</ref>，或者把艋舺龍山寺、艋舺清水巖、艋舺青山宮與[[西門町天后宮|西門町天后宮]]（原艋舺新興宮）合稱「艋舺四大廟」<ref>[https://contest.plus1.today/okgo/index/stands?id=100206 台灣景點票選 艋舺龍山寺]</ref><ref>[http://www.shop2000.com.tw/%E4%B8%89%E8%8F%AF%E9%81%8A%E8%A6%BD%E5%85%AC%E5%8F%B8/product/p24870514 寫下半部萬華史的宗教觀光勝地 艋舺龍山寺]</ref>。

[[臺北盆地|臺北盆地]]周圍早期多為[[安溪|安溪]]移民所開拓，而清水祖師為安溪人的守護神，故[[大臺北|大臺北]]地區[[祖師廟|祖師廟]]林立。艋舺清水巖與[[三峽祖師廟|三峽祖師廟]]、[[淡水祖師廟|淡水祖師廟]]、[[瑞芳祖師廟|瑞芳祖師廟]]，稱為「大臺北四大祖師廟」。<ref>[http://www.rti.org.tw/m/program/detail/?recordId=299 中央廣播電台 台灣故事館]</ref>但這些祖師廟中以艋舺清水巖最具有歷史特色，不僅象徵安溪人的移民勢力，還見證了影響[[臺北市歷史|臺北市歷史]]甚鉅的[[分類械鬥|械鬥]]事件「[[頂下郊拼|頂下郊拼]]」。

== 興廟沿革 ==
[[臺灣清治時期|臺灣清治時期]]渡海來臺的[[福建|福建]][[泉州|泉州]][[安溪|安溪]]移民，公推翁有來為[[董事|董事]]，募得30,000[[銀元|銀元]]，為其從泉州府[[安溪清水岩|安溪縣蓬莱鄉清水巖]]，攜來的清水祖師[[香火|香火]]建廟。

1787年（[[清高宗|清高宗]][[乾隆|乾隆]]52年）在[[臺北|臺北]][[艋舺|艋舺]]現址修築廟宇，1790年（乾隆55年）落成。1817年（[[清仁宗|清仁宗]][[嘉慶|嘉慶]]22年）6月，[[颱風|颱風]]侵襲艋舺清水巖，翁有來於是又向鄉人募捐5,000銀元重修。《淡水廳志稿》記載：「祖師廟亦名清水巖，在艋舺街。乾隆年間捐建；嘉慶年間，捐題重修。」<ref>鄭用錫《淡水廳志稿 四卷》行政院文化建設委員會2006年</ref>

1853年（[[清文宗|清文宗]][[咸豐|咸豐]]3年），[[泉州三邑|泉州三邑]]人後來為了爭奪艋舺的商業利益，與[[同安縣|同安縣]]人間發生[[頂下郊拚|頂下郊拚]]事件。泉州三邑人，欲攻打同安人，苦於沼澤，想由清水巖通過，要求燒毀清水巖，故威脅安溪人，並答應事後會負責重修工作，但事件後三邑人毀約，只捐獻了一對龍柱，略表意思，安溪人於是由本廟董事[[白其祥|白其祥]]向安溪移民，募款25,000[[銀圓|銀圓]]重建清水巖，於1867年（[[清穆宗|清穆宗]][[同治|同治]]6年）整修，但是遲至1875年（[[清德宗|清德宗]][[光緒|光緒]]元年）才完工。清水巖原有三殿，格局完整，目前僅剩三川殿及正殿。

[[臺灣日治時期|日治初期]]的1896年（[[明治|明治]]29年），清水巖被[[臺灣總督府|臺灣總督府]]充做為[[國語學校|總督府國語學校]]的附屬學校。[[台北州立第二中學|臺北州立第二高級中學]]後來也在1922年（[[大正|大正]]11年）於清水巖址草創，並把此事記載入其校歌。「州立第二中學」也就是現在[[台北市立成功高級中學|成功高中]]的前身。1940年（[[昭和|昭和]]15年）後殿遭[[回祿|回祿]]之災，迄今未重建。<ref name="劉豐極">{{cite news  |url= http://ir.lib.ntnu.edu.tw/retrieve/47584/metadata_02_06_s_05_0008.pdf  |title= 劉豐極《艋舺清水巖祖師廟研究》台灣師範大學台灣文化及語言文學研究所，2009年  |deadurl= yes  |archiveurl= https://web.archive.org/web/20141213033726/http://ir.lib.ntnu.edu.tw/retrieve/47584/metadata_02_06_s_05_0008.pdf  |archivedate= 2014-12-13  }}</ref> 1985年，因闢建停車場而挖出一批長期埋在地下的清水巖後殿石材。 1997年，開始按修復計劃分期整建，至今已完成三川殿、正殿之整建。<ref name="劉豐極"/>

== 特色 ==
艋舺清水巖座東朝西，占地約2,000[[坪|坪]]，建地約500坪，原有三殿完整格局，後殿因火災倒燬，右[[護龍|護龍]]約有十分之三面積也因長沙街拓寬於1971年拆毀，現存為面闊七開間兩進兩廊兩護龍「七包三」形式的建築，兩殿間以左右廊相連[[中庭|中庭]]，兩側護龍則以過水廊連接正殿。

=== 重要[[文化財|文化財]] ===
三川殿開三門，明間[[步口|步口]]有[[抱鼓石|抱鼓石]]及蟠龍石柱一對，另有翁有來、翁有麟於1817年（嘉慶22年）重修時敬立的石柱楹聯：「為清水，為蓬萊，此地並分法界；是金身，是鐵面，入門便見真容」。[[檐廊|檐廊]]兩側龍、虎二堵牆分別有「[[南極仙翁|南極仙翁]]」及「[[麻姑|麻姑]]獻壽」磚雕，是臺灣現存落款（1817年）最早的磚雕作品。所繪門神為畫師[[陳壽彝|陳壽彝]]1968年力作，[[哼哈二將|哼、哈二將]]居中門，[[增長天王|南方]]、[[持國天王|東方]]、[[多聞天王|北方]]及[[廣目天王|西方]]等[[四大天王|四大天王]]則分居龍、虎門。屋頂為兩坡[[硬山式|硬山式]]廳堂結構，[[正脊|正脊]]與[[垂脊|垂脊]]排頭上飾有「[[八仙|八仙]]座騎」等[[交趾陶|交趾陶]]人物，兩廊棟架為[[卷棚|卷棚]]式，現存各殿棟架係1867年（同治6年）重建，此後屢經翻修，除彩繪多次外，迄今大致保持重建當年原貌，[[木結構建築|木結構建築]]幾乎為原物。

拜殿有1868年（同治7年）所立蟠龍石柱一對，正殿面闊三間進深六柱，為硬山翹脊屋頂，[[桁|桁]]木十六架，後坡多一架，未作附壁棟架，桁木及[[壽樑|壽樑]]直接插入[[山牆|山牆]]中，山牆內壁採粉刷，但前部露磚柱、磚雕及水車堵，呈前後之分，牆上有罕見的雙窗。此外，在樑架、石壁、龍柱上，也都刻有清朝皇帝[[年號|年號]]年代。正殿後方留有通道及[[格子門|格子門]]以穿廊形式相連原有的後殿，正殿後方步口對看牆分別有「平安」與「富貴」磚雕。廟後方保留回祿倖存的龍柱、石柱及柱礎等，包括龍邊一獨立式兩層閣樓遺跡。

艋舺清水巖廟埕應為開闊廣場，卻遭兩旁民宅[[違建|違建]]長期-{占}-用，僅留一狹長通道，儘管如此，整體建築仍呈現清代中葉臺灣廟宇渾厚樸拙特色與藝術水準，是臺北唯一能夠保存咸豐、同治年間建築原貌迄今者。

[[陳維英_(台灣)|陳維英]]在清水巖楹聯題字，共有三處：<ref name="劉豐極"/>
# 左次間門聯「（同治七年季冬敬立）封藏鬼洞來裂竹大梓羅漢松猶留佛蔭；度登仙橋去藥砧丹臼袈裟石爭訪師型（宗裔維英敬題 生員楊克彰敬書 楊姓弟子敬獻）」
# 左右後附點金柱柱聯「（大清同治七年戊辰佛生日）清五濁以覺人燈傳馬祖；水八功而潤物鉢出龍師（鄉進士孝廉方正四品花翎內外史官淡蘭主講宗裔維英敬撰并書 鄭姓弟子宗裔芳蘭長勝同再敬獻）」
# 後捲棚步口柱柱聯「祖籍桃源而出我穎川之派；師遊蓬島遂鎮化清水之嵓（宗裔維英再敬題 芳蘭 長勝又敬獻）」

==== 蓬萊老祖 ==== 
清水祖師的各尊神像中，以「蓬萊老祖」最受[[檀越|檀越]]崇敬，「蓬萊老祖」是一尊[[宋代|宋代]]的神像，相傳安溪清水巖在宋代得到一塊巨大的[[沈香木|沈香木]]，雕製六尊清水祖師神像，後代稱為「六古佛」。[[清治時期|清治時期]]，一僧自安溪清水巖，奉一尊「古佛」來臺，而後留在臺灣奉祀。[[清法戰爭|清法戰爭]][[西仔反|西仔反]]時，[[法國|法國]][[法軍|遠東艦隊]]侵犯[[淡水區|淡水]]，危及[[臺北城|臺北城]]，後來淡水[[鄉勇|鄉勇]]迎接了「蓬萊老祖」神像開赴戰場，居然擊退了[[法軍|法軍]]。[[清德宗|清德宗]]因[[清法戰爭|清法戰爭]]淡水告捷，賜予[[敕額|敕額]]「功資拯濟」。自此神蹟，艋舺祖師廟香火更為鼎盛。

艋舺方面認為，是因為淡水迎請了艋舺祖師廟的清水祖師「蓬萊老祖」神像助陣，祖師顯靈，才使法軍方敗於淡水。不過淡水善信方面卻認為「蓬萊老祖」神像原屬於淡水，是淡水沒有祖師廟，才寄放在艋舺的。由於有艋舺、淡水兩派信徒紛爭，兩廟在[[日治時期|日治時期]]為了「蓬萊老祖」神尊歸屬，還打官司，在[[日本|日本]][[法官|法官]]息事寧人的情況下，決定兩方共有，輪流供奉，農曆單月在艋舺，雙月在淡水，因應五月的「淡水清水祖師成道慶典」（淡水大拜拜），五月、六月份兩廟互換供奉月份，2016年起，「蓬萊老祖」不再接受其他廟宇迎請<ref>[http://news.ltn.com.tw/news/local/paper/1006015 〈台北都會〉蓬萊老祖神尊 不再掛單請出廟]</ref>。
