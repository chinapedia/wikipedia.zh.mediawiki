'''艋舺王氏家族'''是一個發跡於臺北[[萬華區|萬華地區]]的家族，其創始者為來自福建泉州[[惠安县|惠安]]溪底的'''王義德'''。

== 代表成員 ==
* '''王義德'''（[[1796年|1796年]] ([[嘉慶|嘉慶]]元年)－[[1858年|1858年]] ([[咸豐|咸豐]]八年)），官名王宗河，[[表字|字]]道輝，[[名諱|諱]]光洒，生於泉州惠安溪底，後於嘉慶十七年前後渡海至臺灣發展<ref name=":0">{{Cite web|url=https://nchdb.boch.gov.tw/assets/advanceSearch/monument/20060110000001|title=王義德墓|accessdate=|date=|last=|work=文化部文化資產局}}</ref>；有一說法為其曾先至臺南地區，但最終都落腳於艋舺地區。此後，王義德長年在當地從事對口貿易相關事業，並使王益興商號成為北部主要郊商之一。<ref>{{Cite web|url=http://blog.xuite.net/doublefish2/wretch/142671117-%E7%8E%8B%E7%BE%A9%E5%BE%B7%E5%A2%93(%E5%B8%82%E5%AE%9A%E5%8F%A4%E8%B9%9F)|title=王義德墓(市定古蹟)|accessdate=2017-01-05|last=(doublefish2)|first=二魚|work=隨意窩 Xuite日誌}}</ref>此外，王義德也經年致力於地方社會福利與治安維持，並因此獲清朝當局誥封為二品頂戴花翎，且於逝世時被追封五品藍翎贈奉直大夫頭銜。
* '''王家霖'''
* '''[[王則振|王則振]]'''，又稱'''馬俏哥'''，艋舺地區慈善家，創辦同仁堂協助當地無子嗣人士安葬，設立義倉、育嬰堂等<ref>{{Cite web|url=http://www.merit-times.com.tw/newspage.aspx?unid=152588|title=人間福報 : 清代善人 王義德墓列古蹟|accessdate=2017-01-05}}</ref>

== 文化資產 ==
* '''王義德墓'''：位在臺北市[[南港區|南港區]]，其形制屬清代典型墓形，置有墓碑、墓丘、墓肩、墓雙曲手、前埕等。<ref>{{Cite web|url=https://nchdb.boch.gov.tw/assets/advanceSearch/monument/20060110000001|title=王義德墓|accessdate=|work=文化部文化資產局}}</ref>
* '''艋舺王宅二進遺構'''：位於臺北市[[萬華區|萬華區]]恒安醫院內，為昔日王益興船頭行遺跡；該建物原為五進大厝，在[[台灣日治時期|日治時期]]被改建為診所，目前僅存原建物第三進木門框周邊結構。<ref name=":0" /><ref>{{Cite web|url=https://nchdb.boch.gov.tw/assets/advanceSearch/historicalBuilding/20060116000001|title=艋舺王宅二進遺構|accessdate=|date=|last=|work=文化部文化資產局}}</ref>

== 參見 ==
* '''[[王益順|王益順]]'''：同樣來自惠安溪底的建築匠師，曾參與艋舺龍山寺與新竹都城隍廟之營建
