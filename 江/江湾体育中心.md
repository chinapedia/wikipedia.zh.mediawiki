'''上海江湾体育中心'''或'''上海市江湾体育场'''，原名'''上海市运动场'''，位于今[[中国|中国]][[上海市|上海市]][[杨浦区|杨浦区]][[淞沪路|淞沪路]]245号。1935年8月底竣工，体育场东西两侧设有司令台，由人造白玉筑成，高20米，其左右顶巅置古铜色大鼎各一尊，曾有「远东第一体育场」之称<ref>http://www.sportzone.com.cn/a06002.html{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }} - www.sportzone.com - 江湾体育场场馆介绍</ref>。球场历尽70余年间多次大小修复和改建，现可容纳40,000人。

==历史==
=== 1935年建成 ===
江湾体育中心原名“上海市体育场”，由[[董大酉|董大酉]]建筑师设计，建成于1935年，占地约300亩，建筑总面积7.9万平方米，由体育场、体育馆和游泳池馆三大主体建筑组成。作为“大上海计划”中的重要公共建筑项目之一，是当时国内及远东地区规模最大、设备最先进，也最具影响力的大型综合性体育建筑群。<ref>{{cite web |url=http://www.kic-jwsc.com/jwsc_html/sch/history3.html |title=存档副本 |accessdate=2014-08-06 |deadurl=yes |archiveurl=https://web.archive.org/web/20130606003005/http://www.kic-jwsc.com/jwsc_html/sch/history3.html |archivedate=2013-06-06 }}</ref>

作为[[国民政府|国民政府]]发展上海的《[[大上海计划|大上海计划]]》的重要组成部分，上海市运动场于1934年8月正式开工，1935年8月底正式竣工，随后于10月10日在体育场内举行了[[中华民国全国运动会|中华民国第六次全国运动会]]。1936年上海市长[[吴铁城|吴铁城]]在体育场创办了「上海市立体育专科学校」。

1936年《上海市年鉴》中曾这样描述:“江湾体育场建筑之伟大、范围之广袤，其于体育场之地位，目下[[远东|远东]]殆无与匹”。江湾体育场是当时民国政府推出的“大上海计划”的主要建筑之一，该计划是一个全面、系统地建设近代上海（不包括[[租界|租界]]）的城市规划，选定以[[江湾镇|江湾]]南部今[[江湾五角场|五角场]]地区为中心，由政治区、交通设施、外围工业住宅区和道路系统四大建设项目组成。遗留在杨浦区的[[中华民国上海市政府大厦|旧上海市政府大厦]]、上海市体育场（江湾体育场）、[[上海市立图书馆|旧上海市立图书馆]]、[[上海市立博物馆|上海市立博物馆]]、[[旧上海市市立医院|旧上海市市立医院]]5处建筑都是「大上海计划」的组成部分。“大上海计划”目的是为了与上海市内的列强租界相抗衡，凸显民国政府的统治地位。[[抗日战争_(中国)|抗日战争]]爆发后，「大上海计划」被迫停止。<ref>{{Cite web |url=http://www.cnetnews.com.cn/2007/0721/428480.shtml |title=- www.cnetnews.com.cn - 探寻江湾体育中心的前世今生 第3段 |access-date=2010-04-15 |archive-url=https://web.archive.org/web/20100430134459/http://www.cnetnews.com.cn/2007/0721/428480.shtml |archive-date=2010-04-30 |dead-url=yes }}</ref>

1937年到1945年[[日军|日军]]占领期间，体育场作为日军军火库。[[第二次世界大战|第二次世界大战]]结束后，国民政府重新接管体育场，但仍用作军火库和兵营。1946年4月，军火库发生爆炸，田径场大部分看台被炸裂，震塌，其他建筑物不同程度地受到破坏。

1948年，[[中华民国第七次全国运动会|中华民国第七次全国运动会]]亦在此舉行。

===1954年全面修复===
江湾体育中心建筑组群庄严宏伟，主要墙体采用[[红砖|红砖]]砌筑，只在入口部分加以装饰，建筑各部分比例恰当，典雅大方，民族形式鲜明，颇为建筑界所赞扬。抗日战争爆发后，运动场建筑为日军毁坏。[[国共内战|国共内战]]结束后，经上海市人民政府拨款全面修复，修复设计由华东建筑工业部建筑设计公司（现华东建筑设计研究院）承担，于1954年8月竣工，[[陈毅|陈毅]]市长亲笔题名为“上海市江湾体育场”。<ref>http://www.cnetnews.com.cn/2007/0721/428480.shtml {{Wayback|url=http://www.cnetnews.com.cn/2007/0721/428480.shtml |date=20100430134459 }} - www.cnetnews.com.cn - 探寻江湾体育中心的前世今生 第4段</ref><ref>http://www.kic-jwsc.com/jwsc_html/sch/history2.html{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

===2006年改建===
两年多耗资亿元进行的修复，古稀之年的江湾体育场恢复了当年“东亚之最”的宏伟气势。在修旧如旧的基础上，江湾体育场主体建筑———运动场、体育馆和室内游泳池重现初时光彩。与众多知名学府为邻的江湾体育场已更名为「江湾体育中心」，并成为占地84公顷的杨浦知识创新区中央社区核心部分。这个上海市文物保护单位在修复改建后变身为国内第一个体育休闲中心。<ref>[http://sh.focus.cn/msgview/11787/99202542.html - http://sh.focus.cn - 江湾体育场将成国内首个体育休闲中心 首段]</ref>

占地约15公顷的江湾体育场西大门前，已挖建了地沉式广场，更加显现了这座历史建筑雄伟的视觉效果。经过「整容」，运动场、游泳池、体育馆三大场馆的外墙展现出当初建成时的风采。工程技术人员对风化程度深的墙砖施行“外科手术”，予以更换，对浅度风化的墙砖做了“面膜护理”，所有的表墙都展现出当初建成时的风采。<ref>[http://sh.xmnext.com/shehui/2006/12/29/116385.html - http://sh.xmnext.com - 70多岁江湾体育场“返老还童”] {{webarchive|url=https://web.archive.org/web/20071102061002/http://sh.xmnext.com/shehui/2006/12/29/116385.html |date=2007-11-02 }}</ref> 

2008年5月，江湾体育中心全面开放，由创智天地营运管理。它的主体包括综合体育馆、游泳馆、室外球场区、足球场四大部分，并设有大型停车场、中式庭院等配套设施。<ref>{{cite web |url=http://www.kic-jwsc.com/jwsc_html/sch/history.html |title=存档副本 |accessdate=2014-08-06 |deadurl=yes |archiveurl=https://web.archive.org/web/20130606002036/http://www.kic-jwsc.com/jwsc_html/sch/history.html |archivedate=2013-06-06 }}</ref>

==承办重要赛事==
从1950年代起江湾体育场就是上海[[足球|足球]]的大本营，最早的三支甲级球队———[[上海足球队|上海一队]]、上海青年队、上海工人队都在这里训练。后来联赛职业化后的[[上海申花|上海申花]]曾一度把这里作为训练基地，让江湾体育场跟上海足球有着紧密的联系。<ref>[http://www.jfdaily.com.cn/gb/node2/node17/node160/node46283/node46286/userobject1ai702015.html - 申江服务导报 - 上海体育半个世纪的圣地]{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>1977年球王[[贝利|贝利]]曾率[[美国|美国]][[纽约宇宙队|纽约宇宙队]]在江湾体育场和[[中国国家足球队|中国国家足球队]]打了一场友谊赛。

江湾体育场曾是上海承办国内外重大比赛的主要场馆，承办过的重要赛事有，1935年，[[中华民国第六届全国运动会|中华民国第六届全国运动会]]；1948年，[[1948年中華民國全國運動會|1948年中華民國全國運動會]]；1983年，[[中华人民共和国第五届全运会|第五届全运会]]开幕式；1993年，[[第一届东亚运动会|第一届东亚运动会]]的足球赛事；1996年，首届亚太区[[特殊奥林匹克运动会|特殊奥林匹克运动会]]；1997年，[[中华人民共和国第八届全运会|第八届全运会]]足球赛事；<ref>{{cite web |url=http://www.qqft.cn/browse/photocategorylist.php?action=Gallery_list&Group_id=21148 |title=存档副本 |accessdate=2007-09-14 |deadurl=yes |archiveurl=https://web.archive.org/web/20070905210532/http://www.qqft.cn/browse/photocategorylist.php?action=Gallery_list&Group_id=21148 |archivedate=2007-09-05 }} - www.qqft.cn - 江湾体育场介绍</ref>

改建完工的江湾体育场在2007年5月承办了[[亚洲|亚洲]][[极限运动|极限运动]]锦标赛，并且成为了[[2007年世界夏季特殊奥林匹克运动会|2007年世界夏季特殊奥林匹克运动会]]的闭幕式场地，<ref>[http://news.xinhuanet.com/sports/2007-08/01/content_6462125.htm - 新华网 - 闭幕式将于10月11日晚在上海江湾体育场举行]</ref>这也是体育馆翻新后首次承办综合性体育赛事。

== 场馆设施 ==
=== 足球场 ===
江湾足球场始建于1935年，是一座上海市市级文物保护单位和优秀历史建筑，可同时容纳25000人同时观看大型体育或者文化活动。
=== 综合体育馆 ===
江湾体育馆为全封闭的综合体育场馆，中心赛场面积达924平方米，可同时容纳约2100人观看及进行篮、排球等赛事或室内文艺演出。
=== 游泳馆 ===
江湾游泳馆为上海市市级历史保护建筑。游泳馆拥有标准池及为儿童准备的娃娃池两个泳池。标准池的宽20米，长50米，水深最浅处为1.2米，最深处为2.4米。
