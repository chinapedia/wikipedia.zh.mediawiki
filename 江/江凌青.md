'''江凌青'''{{BD|1983年|7月30日|2015年|1月17日|}}是[[中華民國|中華民國]]電影研究者暨藝術評論者、[[小說家_(職業)|小說家]]、[[藝術家|藝術家]]，在[[臺灣|臺灣]]文壇與藝壇有一定知名度。她出生於[[臺中縣|臺中縣]]，在就讀研究所之前便從事藝術與文學創作，同時在《[[中華副刊|中華副刊]]》、《[[國語日報|國語日報]]》、《[[藝術家_(雜誌)|藝術家]]》雜誌等刊物定期發表創作與藝評。她曾經由[[寶瓶文化|寶瓶文化]]出版短篇圖文小說集《男孩公寓》，並獲得2002年[[梁實秋文學獎|梁實秋文學獎]]、2003年[[臺北文學獎|臺北文學獎]]、2008年[[時報文學獎|時報文學獎]]、2006年與2009年[[全球華文學生文學獎|全球華文學生文學獎]]等藝文相關獎項，也曾獲選[[國家文化藝術基金會|國家文化藝術基金會]]97年度與98年度文學創作類補助。她也從事水彩和油畫等藝術創作，以及[[中華民國文化部|行政院文化建設委員會]]第一屆優秀青年藝術典藏<ref name="臺北市立美術館">{{Cite web |url = http://www.taipeibiennial2014.org/index.php/tw/author/237-ling-ching-chiang-tw |title = 江凌青/ Ling-Ching Chiang (台灣) |author =  |date =  |publisher = [[臺北市立美術館|臺北市立美術館]] |language = zh-hant |accessdate = 2015年1月21日 |quote =  }}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

早在就讀[[國立臺灣師範大學|國立臺灣師範大學]]美術研究所時，江凌青便開始研究當代藝術、新媒體藝術與電影美學理論，之後以[[中華民國教育部|中華民國教育部]]公費前往英國[[萊斯特大學|萊斯特大學]]美術與電影史系攻讀博士學位。其主要關心當代藝術跨域交流的美學樣態，並且從[[博物館|博物館]]研究之角度關注影像與新媒體藝術的發展<ref>{{Cite web |url = http://etat-heath.blogspot.com/2011/11/blog-post_10.html |title = 漫步，或是情節的歸位－從漢格〈游擊漫步〉思考數位藝術中的敘事 |author = 江凌青 |date = 2011年12月9日 |publisher = No Man's Land |language = zh-hant |accessdate = 2015年1月21日 |quote =  }}</ref>。而在一連串藝術評論與理論書寫中，也先後獲得2007年、2008年、2010年與2012年的[[數位藝術評論獎|數位藝術評論獎]]、2011年的世安美學論文獎以及國家文化藝術基金會藝評台首獎，另外也獲得國家文化藝術基金會101年度美術調查與研究類補助<ref name = "臺北市立美術館"/>。2014年於第九屆[http://digitalartfestival.tw/ 台北數位藝術節]中，再獲第八屆數位藝術評論獎最佳評論獎之肯定。2015年1月17日凌晨因為腦動脈堵塞逝世，得年31歲<ref name = "吳垠慧"/>。

== 成長 ==
=== 早年生活 ===
[[File:省議會紀念園區.jpg|thumb]][[霧峰區|霧峰鄉]][[臺灣省諮議會|臺灣省議會紀念園區]]附近。]]

江凌青於1983年7月30日出生於[[臺灣省|臺灣省]][[臺中縣|臺中縣]][[霧峰區|霧峰鄉]][[臺灣省諮議會|臺灣省議會紀念園區]]後方的議會山莊，是家中的長女。父親[[江和言|江和言]]就讀[[國立政治大學|國立政治大學]]政治系，由於家庭在[[南投縣|南投縣]]頗具影響力，使得其一度被安排其從政<ref name = "蘇孟娟">{{Cite web |url = http://www.epochtimes.com/b5/8/10/20/n2302667.htm |title = 小學畢業嫁大學生 自修變畫家 |author = 蘇孟娟 |date = 2008年10月20日 |publisher = 《[[大紀元時報|大紀元時報]]》 |language = zh-hant |accessdate = 2015年1月21日 |quote =  }}</ref>。然而在遇見江凌青的母親[[高碧玉|高碧玉]]後，欣賞後者在[[臺北市|臺北市]]作為養女、但是仍然不斷努力學習英語和日語，甚至僅有國小學歷但是考上遊覽車的隨車人員<ref name = "蘇孟娟"/>。江和言後來不顧親人反對與高碧玉結婚、自己則成為學校教官<ref name = "蘇孟娟"/>；而高碧玉在生下江凌青後發現女兒喜歡畫畫，自己也開始接觸水墨畫、水彩和油畫<ref name="喻文玟">{{Cite web|url=http://pro.udnjob.com/mag2/fn/storypage.jsp?f_ART_ID=44370 |title=養女出身、國小學歷 育女後畫出另片天 |author=喻文玟 |date=2008年10月14日 |publisher=聯合人力網 |language=zh-hant |accessdate=2015年1月21日 |quote= |deadurl=bot: unknown |archiveurl=https://archive.is/20150122104819/http://pro.udnjob.com/mag2/fn/storypage.jsp?f_ART_ID=44370 |archivedate=2015年1月22日 }}</ref>。1985年時，江凌青的弟弟江政凌出生<ref name = "高碧玉">{{Cite web |url = http://blog.artlib.net.tw/author_page.php?act=view&ename=valchiang |title = 高碧玉's 藝術家專屬網站 |author =  |date =  |publisher = 全球華人藝術網 |language = zh-hant |accessdate = 2015年1月21日 |quote =  }}</ref>。後來有積極人生觀的江和言更鼓勵妻子繼續從事創作工作，自己則從學校退休成為專業演說家<ref name = "蘇孟娟"/>。高碧玉在第52屆臺灣省政府全省美展獲得國畫類別首獎後，成為臺灣中部地區知名的畫家<ref name = "高碧玉"/>。

1989年江凌青就讀臺中縣霧峰國小，並且分別在1991年獲選臺中縣模範兒童、1992年獲選[[臺灣省|臺灣省]]特殊優良國小學生以及1993年入圍第一屆世界華人傑出青少年獎。1995年時則就讀臺中市立五權國民中學美術班，並且於1997年臺灣省獲選特殊優良國中生<ref name = "全球華人藝術網">{{Cite web |url = http://blog.artlib.net.tw/author_page.php?act=view&ename=abc |title = 江凌青's 藝術家專屬網站 |author =  |date =  |publisher = 全球華人藝術網 |language = zh-hant |accessdate = 2015年1月21日 |quote =  }}</ref>。其中江凌青從小對美術與文學有強烈的興趣，甚至小時候畫作裡就有一張是以自己為主題，將自己的畫作做展覽<ref name = "羅惠文">{{Cite web |url = http://tw.tranews.com/Show/Style1/News/c1_News.asp?SItemId=0271030&ProgramNo=A000001000003&SubjectNo=33461 |title = 以色彩寫詩　江凌青西畫創作個展 |author = 羅惠文 |date =  |publisher = TraNews台灣 |language = zh-hant |accessdate = 2015年1月21日 |quote =  }}</ref>。1998年就讀[[國立臺中第一高級中學|國立臺中第一高級中學]]美術班後，江凌青開始從事寫作工作，創作範圍橫跨[[小說|小說]]、[[新詩|新詩]]、[[散文|散文]]與評論等領域<ref name = "喻文玟"/>。而她除了在文學界發表許多作品外，同樣也參與多項繪畫比賽<ref name = "周美惠">{{Cite web |url = http://udn.com/NEWS/BREAKINGNEWS/BREAKINGNEWS9/9196536.shtml |title = 藝壇新秀江凌青遽逝 得年僅31歲 |author = 周美惠 |date = 2015年1月20日 |publisher = [[聯合新聞網|聯合新聞網]] |language = zh-hant |accessdate = 2015年1月21日 |quote =  }}</ref>。1999年江凌青獲選成為臺中市模範生，並且獲得[[國立國父紀念館|國立國父紀念館]]青年學生書畫比賽國畫第二名成績<ref name = "全球華人藝術網"/>。

到了隔年，她則分別獲得[[國立彰化生活美學館|國立彰化社會教育館]]中部五縣市春秋水彩寫生第一名、臺中市學生美術比賽國畫第二名、國立國父紀念館青年學生書畫比賽國畫第二名。2001年，江凌青以全校第一名的成績自國立臺中第一高級中學畢業，但是她捨棄[[國立臺灣大學|國立臺灣大學]]法律系、轉而前往[[國立臺灣師範大學|國立臺灣師範大學]]美術系就讀，並且選擇英語系作為輔系<ref name = "吳垠慧"/>。過去給予支持和鼓勵的江凌青父母，則對於其選擇自己的興趣抱持尊重。而在同一年，江凌青已經個別獲得國立國父紀念館第28屆全國青年學生書畫比賽國畫第二名、聯信文教基金會中部寫生比賽銅牌獎、該年度臺中縣美展水彩優選以及該年度全國學生美術比賽國畫優選，另外還獲得全國學生文學獎散文第四名和新詩第二名、全國文藝季散文第一名以及臺中縣文學獎小說獎<ref name="臺中市美術家資料館">{{Cite web|url =http://tcsac.webfuture.com.tw/tw/blog01.aspx?id=1434|title =美術家介紹 江凌青|author =|date =|publisher =[[臺中市美術家資料館|臺中市美術家資料館]]|language =zh-hant|accessdate =2015年1月21日|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20150122111700/http://tcsac.webfuture.com.tw/tw/blog01.aspx?id=1434|archivedate =2015年1月22日}}</ref>。

=== 大學時期 ===
[[File:江凌青父親江和言於追思會會場.JPG|thumb]]

== 生涯 ==
=== 研究所時期 ===
[[File:NTNUfrontdoor.jpg|thumb]]。]]

2005年，江凌青繼續就讀國立臺灣師範大學美術研究所西洋美術史組後，開始從事[[當代藝術|當代藝術]]、[[新媒體藝術|新媒體藝術]]與[[電影美學|電影美學]]的理論書寫<ref>{{Cite web |url = http://ntua.edu.tw/~arts/story/3_01.htm |title = 江凌青 Chiang, Ling-Ching |author =  |date =  |publisher = [[國立臺灣藝術大學|國立臺灣藝術大學]] |language = zh-hant |accessdate = 2015年3月23日 |quote =  }}</ref>。而從1999年開始，江凌青也陸陸續續在《[[中國時報|中國時報]]》、《[[聯合報|聯合報]]》、《[[自由時報|自由時報]]》、《[[幼獅文藝|幼獅文藝]]》、《[[幼獅少年|幼獅少年]]》、《[[中華日報|中華日報]]》、《[[中央日報|中央日報]]》、《[[台灣日報|台灣日報]]》、《[[明道文藝|明道文藝]]》、《[[誠品好讀|誠品好讀]]》、《[[乾坤詩刊|乾坤詩刊]]》、《[[聯合文學|聯合文學]]》等刊物發表文學作品，並且分別在2002年和2003年時獲得[[梁實秋文學獎|梁實秋文學獎]]、[[臺北文學獎|臺北文學獎]]等榮譽<ref name = "吳垠慧"/><ref name = "臺北市立美術館"/>。2006年時，江凌青以《夏季公寓》獲得由行政院文化建設委員會指導、《[[明道文藝|明道文藝]]》和《[[中央日報|中央日報]]》主辦的第24屆[[全球華文學生文學獎|全球華文學生文學獎]]<ref>{{Cite web |url = http://sec.ncu.edu.tw/news/press_content.php?P_ID=9370 |title = 第二十四屆全國學生文學獎得獎名單 |author = 《[[中央日報|中央日報]]》 |date = 2006年6月1日 |publisher = [[國立中央大學|國立中央大學]] |language = zh-hant |accessdate = 2015年1月21日 |quote =  |deadurl = yes |archiveurl = https://web.archive.org/web/20150402152836/http://sec.ncu.edu.tw/news/press_content.php?P_ID=9370 |archivedate = 2015年4月2日 }}</ref>。

=== 文字作品 ===
江凌青擅長小說、散文、新詩等各類文學創作，曾經榮獲[[時報文學獎|時報文學獎]]、[[梁實秋文學獎|梁實秋文學獎]]、[[林語堂文學獎|林語堂文學獎]]和全國各縣市文學獎項，以及世界華人女性新詩第二名殊榮。江凌青在接受採訪時表示比較希望能寫出或者看到，從很尋常的生活面向出發卻有驚奇發展的作品。江凌青提到自己沒有買過任何作家或者導演的全套著作品，但是在寫作養成階段看過許多[[村上龍|村上龍]]的文學著作，這包括有《村上龍料理小說》、《村上龍電影小說》、《京子》、《萊佛士酒店》、《悲傷的熱帶》等。對此她認為希望自己有一天可以變成像[[村上春樹|村上春樹]]那樣早起先慢跑的人，不過實際上仍然比較類似村上龍那種「不喜歡工作，所以都會盡快寫好出去玩」的概念<ref name = "聯合文學"/>。另外認為自己離真正小說家狀態還有一段距離的她還表示：

{{Cquote 3|在選擇標準上，其實很簡單就是要讓我覺得有趣；而若要說媒材之間的互補，我深信每種媒材都有一個獨立的世界，不是互補缺陷，而是相互映照各自的豐富之處。對我來說，小說彰顯的是一種比較原始的力量，因為使用的『工具』最單純。這些年看過各類藝術創作之後，我反而越來越嚮往這樣的單純。|江凌青<ref name = "聯合文學"/>}}

=== 視覺藝術 ===
江凌青的繪畫作品主要採融合現實與想像的手法，創造出充滿故事性的畫面空間<ref name = "羅惠文"/>。曾經與其有長期接觸、並且認為江凌青天資聰穎的國立臺灣師範大學美術研究所教授[[蘇憲法|蘇憲法]]表示：「綜觀凌青的作品，她早已脫離『具象寫實』的階段，進入『平面色塊』的概念，然而重點是去除具象、形體和描繪的特徵之後，並不會造成貧瘠或弱化的現象。<ref name = "全球華人藝術網"/>」而另一位教授[[王瓊麗|王瓊麗]]則認為：「畫中流露著活躍、熱情、細膩、創發的姿相，使其作品帶有哲學的精神對話；其作品有巧妙的構圖安排及揮灑自如的筆觸，在技巧及媒材的運用與色彩的表現上均有相對水準，可見其深厚的才學與功力。<ref name = "羅惠文"/>」

同時她也有許多繪畫作品獲得藏家收藏，其中霧峰澄清醫院收藏其4幅插畫連作、[[國立國父紀念館|國立國父紀念館]]收藏其國畫作品《業精於勤》和《皮影戲》、[[中國醫藥大學附設醫院|中國醫藥大學附設醫院]]收藏其1幅水彩作品、[[逢甲大學|逢甲大學]]收藏其油畫作品《魚群記憶的房間》、[[國立勤益科技大學|國立勤益科技大學]]收藏其1幅油畫作品、日月潭教師會館收藏其油畫作品《花》、霧峰區農會收藏其油畫作品《小鎮風光》等。同時江凌青在2003年時，也入選成為行政院文化建設委員會第一屆青年藝術家作品典藏計畫，使得其國畫作品《角色》和油畫作品《巷子的盡頭》獲得收藏，另外包括前中華民國內政部部長[[廖了以|廖了以]]等人也有收藏其畫作<ref name = "臺中市美術家資料館"/>。
