'''江苏省句东劳动教养管理所'''是[[中华人民共和国|中华人民共和国]][[江苏省|江苏省]]的7所省级[[劳动教养管理所|劳动教养管理所]]之一，隶属于[[江苏省司法厅|江苏省司法厅]][[江苏省戒毒管理局|戒毒管理局]]。位于[[句容市|句容市]][[石山头|石山头]]。与'''江苏省句东强制隔离戒毒所'''为[[一个机构两块牌子|一个机构、两块牌子]]<ref name="a1">{{Cite web |url=http://js.people.com.cn/html/2013/11/21/269935.html|title=江苏已无劳改人员 劳教所悄然变身戒毒所 |accessdate=2018-11-06 |author=施忆、韩振|date=2013-11-21 |work= |publisher=人民网|language=Zh-hans }}</ref>。

==沿革==
该所是江苏省最早的劳教所，成立于1950年代。劳教人员进行生产劳动。早期是茅草房，劳教人员都是所谓的“[[地富反坏右|地富反坏右]]”。1980年代初翻建为平房，劳教人员多为单位的“[[失足青年|失足青年]]”，包括“[[多吃多占|多吃多占]]”和“[[乱搞男女关系|乱搞男女关系]]”者。近年来改建楼房，[[吸毒|吸毒]]人员开始占据较大比重，因而管教方式也有所变化<ref>{{Cite web |url=http://www.legaldaily.com.cn/zt/content/2007-08/29/content_690690.htm?node=6641|title=从两代民警看劳教所变化 |accessdate=2018-11-06 |author=记者：蒋德|date=2007-08-29 |work=责任编辑：邢晖 |publisher=法制网—法制日报|language=Zh-hans }}</ref>。

2013年废除劳改制度，句东劳教所与江苏省现有的劳改所将转型为强制戒毒所<ref name="a1"/>。与句东强制隔离戒毒所同处石山头，隶属于同一系统的[[江苏句东农场|江苏句东农场]]网站公告显示，两者相关<ref>{{Cite web |url=http://www.jdztbw.com/list/?3_1.html|title=招标公告 |accessdate=2018-11-06 |author=|date= |work= |publisher=江苏句东农场有限公司招投标网|language=Zh-hans }}</ref>。
