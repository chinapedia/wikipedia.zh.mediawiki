'''江门鹤山反核事件'''是在2013年7月发生于[[中華人民共和國|中華人民共和國]][[广东省|广东]][[江门市|江门市]]的一起群体性抗议事件。缘起于[[中国核工业集团公司|中核集团]]计划在当地建设大型的核燃料加工厂，但遭到江门和[[鹤山市|鹤山市]]两地民众激烈反对，认为计划未进行[[环境评估|环境评估]]以及咨询公众时间太短，担心核燃料加工项目影响健康。项目最终由政府承诺撤销。

== 背景 ==
[[中国核工业集团公司|中国核工业集团公司]]（中核集团）计划在广东江门[[鹤山市|鹤山市]][[址山镇|址山镇]]兴建[[中国南部|中国东南]]沿海第一座[[核燃料|核燃料]]加工厂——龙湾工业园项目，选址于江门鹤山市址山镇大营工业区，这是江门鹤山击败40多个竞争对手后夺得的央企项目，中核集团与[[中国广核集团|中广核集团]]共同总投资370亿元开发。<ref>{{cite news|url=http://paper.wenweipo.com/2013/07/07/YO1307070010.htm|title=网民齐反江门建核燃料厂|publisher=|newspaper=[[文汇报|文汇报]]|date=2013-07-07}}</ref>

项目总用地面积约229[[公顷|公顷]]，总建筑规模约50万[[平方米|平方米]]。厂内将有[[铀|铀]]纯化转化、[[浓缩铀|铀浓缩]]、核燃料组件制造等设施，预料2017年投产<ref>{{cite news|url=http://orientaldaily.on.cc/cnt/china_world/20130710/00178_010.html|title=鹤山拟建核燃料厂激发民愤|newspaper=[[東方日報_(香港)|东方日报]]|date=2013-07-10}}</ref>，可供应[[大亚湾核电厂|大亚湾核电厂]]和[[台山核电站|台山核电站]]等。建成后，可望打造成为「一站式」核燃料加工产业链、国际一流核燃料加工产业集群，以至亚洲核燃料加工及装备制造中心。<ref name="nang0706">{{cite news|url=http://epaper.oeeee.com/H/html/2013-07/06/content_1889727.htm|title=中核拟在江门建核燃料加工厂|publisher=南都网|newspaper=[[南方都市报|南方都市报]]|date=2013-07-06}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref name="sina0706">{{cite news|url=http://news.sina.com.hk/news/20130706/-9-3010642/1.html|title=广东江门拟建核燃料加工厂 官方称安全无污染|publisher=新浪|newspaper=南方都市报|date=2013-07-06}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

项目发展方声称核燃料厂「绝对安全、无[[污染|污染]]，甚至在[[地震|地震]]等极端状况下也不会影响周边环境」。又谓涉及的鹤山当地居民都没有反对，拆迁徵地进展非常顺利，是因为居民详细了解项目后表示「有信心」。<ref name="nang0706"/>这个工业园项目徵地涉及鹤山市址山镇13个村庄。<ref name="sina0706"/>

当局希望计划能在2013年底动工。江门市政府在2013年7月4日发布《中核集团龙湾工业园项目社会稳定风险评估公示》征求公众意见，为期十天。

== 民众反应 ==
项目计划一出，江门市以至周边城市的民众随即表达了强烈反对，网上反对意见更是一面倒。

7月12日，江门市民响应网上发起之号召，走到江门市中心沿马路游行，最后到市政府门外集会示威，批评政府为经济利益牺牲市民健康，他们又认为十天的征询期太仓促。不少市民突破警察的封路，从四方八面涌出包围政府。市副市长与民众对话，被示威者包围。最后市政府决定延长咨询期。<ref>{{Cite news|url=http://news.tvb.com/world/51e015ac6db28c5e0c000004|title=江门鹤山市拟建核燃料加工厂触发抗议|publisher=[[无线新闻|无线新闻]]|date=2013-07-12}}</ref>

拟建的核燃料加工厂项目直线距离[[香港|香港]]约120公里，有关计划亦引起港人关注，在社交网站[[Facebook|Facebook]]多名网民留言表示对核厂安全感到担忧。

不过，核安全咨询委员会副主席李焯芬在香港核电站核安全咨询委员会记者发布会上，答复记者提问时称核燃料厂「不存在风险」。并谓核燃料棒不牵涉任何[[核分裂|核分裂]]作用在内，因此造好的核燃料棒，就是「拿着燃料棒都不会受辐射影响」。<ref>{{Cite news|url=http://news.wenweipo.com/2013/07/12/IN1307120037.htm|title=香港专家:江门建核燃料厂没风险|newspaper=[[广州日报|广州日报]]/文汇网|date=2013-07-12}}</ref>

== 项目撤销 ==
7月13日早上，江门市委、鹤山市委、中核集团、中广核集团等多方面证实，龙湾工业园区项目决定不予立项。江门市委、市政府高度重视12日的群众游行活动表达要求，并提出延长公示时间。他们称由于社会各界反对意见较多，经慎重考虑后充分尊重人民群众的意愿，作出终止项目的决定。<ref>{{Cite news|url=http://ndgd.oeeee.com/html/201307/13/82692.html|title=广东江门鹤山核燃料项目取消|newspaper=南都网|date=2013-07-13|deadurl=yes|archiveurl=https://web.archive.org/web/20130716185021/http://ndgd.oeeee.com/html/201307/13/82692.html|archivedate=2013-07-16}}</ref>

撤核计划消息传出后，市面一片欢欣鼓舞，民众庆祝「[[维权|维权]]」胜利。<ref>{{Cite news|url=http://paper.wenweipo.com/2013/07/14/CH1307140005.htm|title=江门顺民意 撤建核料厂|newspaper=文汇报|date=2013-07-14}}</ref>

14日下午，再有民众走到[[东湖广场|东湖广场]]聚集，民众对核燃料加工厂项目是否真正撤销表示忧虑，要求当局交出文件证明计划已被搁置，又希望市政府当众宣布项目取消，人数聚集过千。警察在现场戒备。<ref>{{Cite news|url=http://news.tvb.com/world/51e20ffb6db28cee1f000001|title=江门市民再示威促交搁建核料厂文件|newspaper=无线新闻|date=2013-07-14}}</ref>

之后江门市委书记刘海与民众对话，并发出红头文件人民政府公告，证明项目取消。聚集的民众欢呼着自行和平散去。<ref>{{Cite news|url=http://epaper.oeeee.com/A/html/2013-07/15/content_1895381.htm|title=江门发红头文件 确认取消核项目|newspaper=南方都市报|date=2013-07-15}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref>{{Cite news|url=http://paper.wenweipo.com/2013/07/15/YO1307150009.htm|title=江门红头文件尽释核疑虑|newspaper=文汇报|date=2013-07-15}}</ref>

== 后续 ==
在核厂项目选址附近的莲珠村，40多户共100多人早前收到政府每户几万元拆迁费不清楚是否需要交还给政府，不愿出镜的村长指出，对项目撤销感到很失望。<ref>{{Cite news|url=http://news.tvb.com/world/51e1586c6db28c527f000002|title=鹤山撤核料厂 每户数万元未知是否需退回|newspaper=无线新闻|date=2013-07-13}}</ref>

有报导称，中核集团龙湾工业园项目「胎死腹中」，带来大额经济损失。因为这项目如果建成投产，每年可为江门鹤山带工业产值390亿元，是2012年全年规模以上的工业总产值，还另加30亿元以上的税收。<ref>{{Cite news|url=http://ndfinance.oeeee.com/html/201307/20/85527.html|title=江门核项目胎死腹中 官员惋惜每年损失税收30亿元|publisher=南都网财经|date=2013-07-20|access-date=2013-07-29|archive-url=https://web.archive.org/web/20130723024119/http://ndfinance.oeeee.com/html/201307/20/85527.html|archive-date=2013-07-23|dead-url=yes}}</ref>
