[[江苏省|江苏省]]全境通用[[漢語|漢語]]，人口最多的漢語變體分别是[[江淮官话|江淮官话]]、[[吴语|吴语]]和[[中原官话|中原官话]]。关于[[连云港市|连云港市]]北部[[赣榆县|赣榆县]]方言存在争议。[[南通|南通]]部分地区属于吴淮混合方言。此外，由于近年来人口移动，[[苏南|苏南]]部分地区形成客籍、[[河南|河南]]等方言岛。

== 吴语 ==

江苏省操[[吴语|吴语]]（主要为[[太湖片|太湖片]]、[[宣州片|宣州片]]）的主要为[[苏锡常地区|苏锡常地区]]，实际面积较小，但人口密集且经济发达。江苏吴语区包括：[[苏州|苏州]]、[[无锡|无锡]]2市全部；[[常州|常州]]市区（包括武进区和新北区）、[[金坛|金坛]]东部，[[溧阳|溧阳]]；镇江[[丹阳市|丹阳市]]大部；南京[[溧水|溧水]]南部地区、[[高淳区|高淳区]]；南通市区南部、[[海门|海门]]、[[启东|启东]]2市全部，通州市小部，[[如东县|如东县]]小部；[[靖江市|靖江市]]大部。另外在苏北沿海，绵延至连云港沿海均有海门、启东两地于民国时期清末状元[[张謇|张謇]]带去的移民，该些移民及其后代绝大部分操苏沪嘉小片的启海话。

[[苏州话|苏州话]]是[[吴语|吴语]]的代表方言。

== 江淮官话 ==
范围包括[[扬州市|扬州市]]、[[淮安市|淮安市]]、[[盐城市|盐城市]]全部；[[南京市|南京市]]（除[[溧水|溧水]]、[[高淳|高淳]]）、[[镇江|镇江]]（除[[丹阳|丹阳]]）、[[泰州|泰州]]（除[[靖江|靖江]]）、[[连云港|连云港]]（除[[赣榆|赣榆]]、[[东海|东海]]西部和北部）[[宿迁|宿迁]]（除市区）大部；南通市市区、[[如皋市|如皋市]]、[[海安县|海安县]]、[[如东县|如东县]]大部，[[通州市|通州市]]一部分。

[[南京官話|南京官話]]和[[扬州话|扬州话]]是[[江淮官话|江淮官话]]的代表方言。

== 中原官话 ==
江苏省操[[中原官话|中原官话]]的地区包括：[[徐州市|徐州市]]全市、[[宿迁市|宿迁市]]市区、[[东海县|东海县]]西部及北部，[[赣榆县|赣榆县]]西部，[[赣榆县|赣榆县]]东部<sub>（有争议）</sub>。

徐州话是[[中原官话|中原官话]]的代表方言。

== 胶辽官话 ==
[[连云港市|连云港市]]北部[[赣榆县|赣榆县]]方言存在争议，此地位于[[胶辽官话|胶辽官话]]、[[中原官话|中原官话]]、[[淮语|淮语]]三者交汇地。有认为属于[[胶辽官话|胶辽官话]]（[[青州片|青州片]]、[[登连片|登连片]]仍有争议）<ref>《赣榆方言志》</ref>、有认为属于[[中原官话|中原官话]]<ref>《中国语言地图集》、《江苏省和上海市方言概况》（1960）</ref>。清声母入声字归并与[[中原官话|中原官话]]相同，今声母与[[胶辽官话|胶辽官话]]同。总体语调可能偏[[胶东话|胶东话]]。
