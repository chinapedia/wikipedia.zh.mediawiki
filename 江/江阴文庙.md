'''江阴文庙'''位于中国[[江苏省|江苏省]][[江阴市|江阴市]][[澄江街道|澄江街道]]人民中路196号，是历代奉祀[[孔子|孔子]]及儒门贤哲的祠庙，也是[[学宫|学宫]]和儒学教官衙署所在地。现在是[[无锡|无锡]]地区惟一保存较为完整的儒学建筑遗存，1995年被列为[[江苏省文物保护单位|江苏省文物保护单位]]。

==历史==
江阴文庙始建于[[北宋|北宋]]初年，后经多次修葺，到清末有[[照壁|照壁]]、[[德配天地坊|德配天地坊]]、[[道冠古今坊|道冠古今坊]]、[[金声门|金声门]]、[[玉振门|玉振门]]、[[棂星门|棂星门]]、[[泮桥|泮桥]]、[[戟门|戟门]]、[[大成殿|大成殿]]、[[明伦堂|明伦堂]]、[[尊新斋|尊新斋]]、[[时习斋|时习斋]]等建筑。1901年改名为校士馆，1903年兼办高等小学，1905年专办学堂（礼延高等小学），1912年改名为江阴县第一高等小学，根据学校需要进行了改扩建。[[抗日战争|抗日战争]]时期，文庙遭到了日军的严重破坏。抗战胜利后以及中华人民共和国成立后，文庙长期作为中学（江阴县立中学）校舍，大成殿作为废旧物资储藏处，沦为危房，明伦堂成为学生餐厅。<ref>{{cite web |title=民国时期的孔庙 |publisher= |url=http://www.confucianism.com.cn/Show.asp?id=52042}}</ref>1993年[[江苏省江阴高级中学|江阴县中]]迁至新址，原址改作[[暨阳中学|暨阳中学]]。1995年，文庙建筑从学校独立出来，并进行了全面整修。

2007年开始在明伦堂内开办国学启蒙班。<ref>{{cite web|title=退商进文"弘扬传统 江阴文庙重开国学讲坛|publisher=[[解放网|解放网]]|url=http://old.jfdaily.com/gb/jfxww/xinwen/node1223/node15574/userobject1ai1576348.html|deadurl=yes|archiveurl=https://web.archive.org/web/20160304201431/http://old.jfdaily.com/gb/jfxww/xinwen/node1223/node15574/userobject1ai1576348.html|archivedate=2016年3月4日|df=}}</ref> 2010年[[中秋节|中秋节]]，江阴文庙举行了首届[[祭孔|祭孔]]大典。<ref>{{cite web |title=江阴文庙举行首届祭孔大典 |publisher=[[中国新闻网|中国新闻网]]，原载[[江南晚报|江南晚报]] |url=http://www.js.chinanews.com/wx/news/2010/0925/29523.html}}</ref>

现在，文庙还经常作为美术、书法、摄影等艺术作品的展览馆。<ref>{{cite web |title=江阴市领导与市民共享春节“文化大餐” |publisher=[[人民网|人民网]]，原载[[江阴日报|江阴日报]] |url=http://unn.people.com.cn/GB/22220/80078/13883507.html}}</ref><ref>{{cite web|title=江阴市第四届退休职工书画展在文庙拉开帷幕|publisher=江阴市人力资源和社会保障局|url=http://www.jylss.com/index_Show_Class.asp?Article=1027}}{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref><ref>{{cite web |title=盟员朱文郁书法作品展在文庙举行 民盟江阴市委领导参观展出 |publisher=中国民主同盟江苏省委员会 |url=http://www.jsmm.org.cn/Article.asp?Id=1437}}</ref>

==建筑==
清末，文庙建筑配备齐全。有[[照壁|照壁]]、[[德配天地坊|德配天地坊]]、[[道冠古今坊|道冠古今坊]]、[[金声门|金声门]]、[[玉振门|玉振门]]、[[棂星门|棂星门]]、[[泮桥|泮桥]]、[[戟门|戟门]]、[[大成殿|大成殿]]、[[明伦堂|明伦堂]]、[[尊新斋|尊新斋]]、[[时习斋|时习斋]]等建筑。

现在的江阴文庙坐北朝南，由南至北依次为棂星门、泮池、[[大成门|大成门]]、[[大成殿|大成殿]]、明伦堂，大成殿的东西两侧有[[庑房|庑房]]40间。<ref>{{cite web |title=江苏江阴文庙 |publisher= |url=http://www.zhongsou.net/%E5%BB%BA%E7%AD%91%E6%96%87%E5%8C%96%E8%89%BA%E6%9C%AF/detail/article/1380989}}</ref>
