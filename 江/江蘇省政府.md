'''江蘇省政府'''是[[中華民國|中華民國]][[南京國民政府|南京國民政府]]在[[江苏省_(中華民國)|江蘇省]]的最高地方行政機關。自民國16年（1927年）11月1日[[南京國民政府|南京國民政府]]改組[[江蘇省政務委員會|江蘇省政務委員會]]為江蘇省政府開始，到民國39年（1950年）5月撤出[[嵊泗列島|嵊泗列島]]為止，前後歷時24年。

== 歷史沿革 ==
民國16年（1927年）4月18日，蔣中正在南京另立國民政府，接著又相繼组建了蘇、浙、皖、閩、黔、滬、京（南京）等省和[[特別市|特別市]]的政府機構。4月26日，南京國民政府下令改組江蘇省政務委員會，取消主席制，從委員中推選5人為常務委員，下設民政、財政、教育、建設、農工、司法、軍事7廳；同時裁撤臨時省政委員會、臨時軍事委員會、臨時財政委員會。10月28日，南京國民政府第11次常務會議議決，將江蘇省政務委員會改組為江蘇省政府委員會。11月1日，江蘇省政府在南京湖南路正式成立，下設民政、財政等廳。民國二十六年（1937年），因日軍入侵，國民政府於11月26日令改組江蘇省政府委員會。12月，日軍攻陷江蘇全省，江蘇省政府成為地方流亡政府。民國34年（1945年）8月，抗戰結束，江蘇省政府恢復其行政管轄區域。10月26日，國民政府下令改組江蘇省政府。民國37年（1948年）9月1日，再改組江蘇省政府<ref>《中華民國史事日志》，1948，中華民國三十七年戊子。</ref>。民國38年（1949年），因[[中国共产党|中共]][[中国人民解放军|解放軍]]進入蘇南，江蘇省政府遷往崇明、嵊泗一帶。江蘇省政府的行政機構，隨著民國39年（1950年）隨著國軍撤離嵊泗而消亡。

江蘇省政府改組換屆達十一次之多。省主席和委員的任職是沒有固定任期，有的最長任期達五年多，有的只有幾個月的時間。由[[行政院|行政院]]提名9至15人為省政府委員會委員，並於其中指定一人為省主席，和各廳、處長人選報請國民政府明令任命。省政府委員、主席、廳、處級，均為[[簡任|簡任]]一級<ref>{{Cite web |url=http://www.zjmcyj.com/zjmcyj/News_View.asp?NewsID=918&page=1 |title=民國江蘇省主席與主政官員 |access-date=2013-12-21 |archive-url=https://web.archive.org/web/20131224084808/http://www.zjmcyj.com/zjmcyj/News_View.asp?NewsID=918&page=1 |archive-date=2013-12-24 |dead-url=yes }}</ref>。

== 省政府駐地 ==
江蘇省政府成立初期，以[[江寧縣|江寧縣]]（今[[南京市|南京市]]）為省會。民國16年（1927年）7月27日，南京成立特別市，為國民政府所在地，不再受江蘇管轄，但江蘇省會依舊寄治南京；直到民國18年（1929年）二月，江蘇省會自南京移至[[鎮江縣|鎮江縣]]。民國26年（1937年）[[中国抗日战争|抗戰]]爆發不久，江蘇省境大部分淪陷。11月，省會暫遷至[[江都县|江都縣]]；12月，再遷至[[淮阴县|淮陰縣]]；民國28年（1939年）3月，再遷至[[兴化县_(江苏)|興化縣]]；民國三十年（1941年）4月，因日軍入侵，省政府遷至[[淮安县_(1914年)|淮安縣]][[车桥镇_(淮安市)|車橋]]，韩德勤在此主持修建了车桥机场；民國32年（1943年）10月，省政府暫遷[[安徽省_(中華民國)|安徽省]][[太和县|太和縣]]辦公；民國三十三年（1944年）省政府暫遷安徽省[[阜陽縣|阜陽縣]]李寨辦公<ref>[http://blog.163.com/public/read.php?tid=36201&uid=753]{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }} 行政區劃論壇「長白的民國系列之二——因抗日戰爭而被迫的駐地遷移
」。</ref>。民國34年（1945年）抗戰勝利後，江蘇省政府於9月遷回鎮江縣，民國38年（1949年）4月23日中共解放軍進入鎮江縣，江蘇省政府遷至[[崇明縣|崇明縣]]。6月又遷至[[嵊泗列島|嵊泗列島]]（1949年9月3日改置為[[嵊泗县|嵊泗縣]]）。

== 江南行署、徐州行署、江蘇特別行政區 ==
抗日戰爭期間，民國32年（1943年）1月28日設置[[江南行政公署|江南行政公署]]<ref>傅林祥、鄭寶恆：《中國行政區劃通史‧中華民國卷》，第156頁。</ref>（一作1939年1月1日置<ref>[http://blog.163.com/public/read.php?tid=12416&uid=374]{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }} 行政區劃論壇「國府的省轄行署」。</ref>），行署駐今[[安徽省|安徽省]][[黃山市|黃山市]][[屯溪区|屯溪區]]附近的梅林，代行江蘇省政府職權，管理長江以南地區；2月16日設置[[徐州行政公署|徐州行政公署]]<ref>傅林祥、鄭寶恆：《中國行政區劃通史‧中華民國卷》，第156頁。</ref>（一作1939年3月4日置<ref>[http://blog.163.com/public/read.php?tid=12416&uid=374]{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }} 行政區劃論壇「國府的省轄行署」。</ref>），行署駐地不詳，代行江蘇省政府職權，管理長江以北地區；12月，又以長江以北鹽城、寶應以東地方設置[[江蘇特別行政區|江蘇特別行政區]]，直隸[[重慶國民政府|重慶國民政府]]<ref>《國民政府公報》渝字第603號，民國32年12月11日，第19頁。</ref>。民國34年（1945年）抗戰勝利後，於9月裁撤各行署。
