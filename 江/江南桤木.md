'''江南桤木'''（[[学名|学名]]：{{lang|la|''Alnus trabeculosa''}}）是[[桦木科|桦木科]][[桤木属|桤木属]]的植物。

== 形态 ==
落叶[[乔木|乔木]]或[[灌木|灌木]]。广卵圆形叶，基部多为圆形，边缘有紧贴锯齿，叶柄长5-15毫米；早春开花，雌雄异株，柔夷[[花序|花序]]常有数个集生于上年小枝的末梢；果穗长1.5-2.5厘米，穗上木质鳞片，宿存，全形稍像松果。

== 分布 ==
分布于[[日本|日本]]以及[[中国大陆|中国大陆]]的[[河南|河南]]、[[湖北|湖北]]、[[福建|福建]]、[[江西|江西]]、[[江苏|江苏]]、[[浙江|浙江]]、[[广东|广东]]、[[安徽|安徽]]、[[湖南|湖南]]等地，生长于海拔200米至1,000米的地区，多生在山谷和河谷的林中，目前尚未由人工引种栽培。

== 异名 ==
