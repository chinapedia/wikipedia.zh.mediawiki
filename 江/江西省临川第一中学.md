'''临川一中'''是江西省重点中学，坐落在[[江西省|江西省]][[抚州市|抚州市]][[临川区|临川区]]，是一所规模大，而且非常有名的中学，曾获得[[中国十大名牌中学|中国十大名牌中学]]，[[中国百强中学|中国百强中学]]等荣誉<ref>http://www.lcyz.net/Item/336.aspx</ref>。

== 成就 ==
临川一中每年在[[高考|高考]]中均取得优秀的成绩，录取[[北京大学|北京大学]]、[[清华大学|清华大学]]等名校的学生数目常居江西之首<ref>http://www.lcyz.net/Item/336.aspx</ref>。拥有北京大学，清华大学等大学的[[自主招生|自主招生]][[校长推荐制|校长推荐]]资格。

== 争议 ==
{{ 地域中心 }}
临川一中以每年卓越的高考成绩著称，但有人评论称这是建立在巨大的学生基数（2007年为约11000人）上的，并且招收大量[[复读|复读]]考生。亦有人认为临川一中的教育理念过于[[应试教育|应试教育]]化，学习氛围“残酷”<ref>http://news.qq.com/a/20070910/002612.htm</ref>。
2010年高考人数约4000人，复读考生约2100，高考结果一本以上1800人，二本以上约3000人

== 竞争 ==
与省内其他重点中学，如[[江西师大附中|江西师大附中]]、[[临川二中|临川二中]]等构成竞争关系，每年高考成绩公布后，各校均会将該校的高考成绩张榜公布，以显示自身的教学质量。
