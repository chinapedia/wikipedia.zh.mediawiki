'''江口壽史'''（{{lang-ja|江口 寿史}}，{{bd|1956年|3月29日||}}），[[日本|日本]]男性[[漫畫家|漫畫家]]、[[插畫家|插畫家]]。妻子為前[[日本偶像|偶像歌手]]{{link-ja|水谷麻里|水谷麻里}}。O型[[血型|血]]。

== 來歷、人物 ==
江口壽史出生於[[熊本縣|熊本縣]][[水俁市|水俁市]]。代表作有《[[停止！！雲雀|停止!! 雲雀]]》、《{{link-ja|前進吧!! 海盜|すすめ!!パイレーツ}}》、《{{link-ja|江口壽史的爆發晚宴|江口寿史の爆発ディナーショー}}》等。

江口在小學3年級左右，受到了漫畫《[[七色假面|七色假面]]》、《[[月光假面|月光假面]]》和《[[原子小金剛|原子小金剛]]》等影響，開始創作漫畫<ref name = "eguchi">《KAWADE夢Mook 別冊文藝 總特集江口壽史》卷末略年譜（河出書房新社，2003年）。</ref>。1970年3月，因父親調職的關係與父親陪同搬到[[千葉縣|千葉縣]][[野田市|野田市]]居住<ref name = "eguchi" />。1974年3月從千葉縣立柏高等學校畢業<ref name = "eguchi" />。經過了一年的無業生活後，進入設計學院就讀，不過只通學1個月就未出現在課堂，留籍在校過著每日看電影、讀書和打[[彈珠機|柏青哥]]的日子<ref name = "eguchi" />。1976年，江口入院接受[[甲狀腺功能亢進症|甲狀腺功能亢進症]]的手術治療，住院長達1個半月，臥病在床的江口決定要走上職業漫畫家創作。

1977年投稿《恐怖的孩子們》至[[集英社|集英社]]入圍YOUNG JUMP獎（後來成為現在的十二傑新人獎），同年又投稿了《8點半的決鬥》準入選第6回{{link-ja|赤塚獎|赤塚賞}}，並開始連載《前進吧!! 海盜》，該作品以高質的搞笑而受到歡迎，甚至被讀者拿來與當時在《[[週刊少年Champion|週刊少年Champion]]》（[[秋田書店|秋田書店]]）的資深搞笑漫畫家{{link-ja|鴨川つばめ|鴨川つばめ}}的《{{lang|ja|{{link-ja|マカロニほうれん荘|マカロニほうれん荘}}}}》相提並論。1978年以《{{link-ja|HINOMARU劇場#隨時低潮的名探偵|ひのまる劇場#名探偵はいつもスランプ|隨時低潮的名偵探}}》入圍獲得第6屆愛讀者獎，1992年以《江口壽史的爆發晚宴》榮獲[[文藝春秋漫畫賞|文藝春秋漫畫屆]]。

1990年與前[[日本偶像|偶像歌手]]{{link-ja|水谷麻里|水谷麻里}}結婚（江口個人生涯的第二次）。此後，江口的插畫工作量逐漸增加，開始轉移專心創作插圖為中心。

1995年創刊「{{link-ja|COMIC CUE|COMIC CUE}}」，由江口就任責任編輯。並招集[[松本大洋|松本大洋]]、{{link-ja|望月峯太郎|望月峯太郎}}、{{lang|ja|{{link-ja|よしもとよしとも|よしもとよしとも}}}}、{{link-ja|魚喃霧子|魚喃キリコ}}、[[古屋兔丸|古屋兔丸]]等多位有意願投稿在本雜誌上的作家在傳統的漫畫界「競賽作品中定一個獨特主題作品」和「名作漫畫的封面插圖」等是否執行的企劃。

2003年參與擔任電視動畫《[[星際少年隊|星際少年隊]]》的角色原案，同年開始播放。

2004年12月在「[[Evening|Evening]]」（[[講談社|講談社]]）開始連載《壽丼》，但只連載3回立即休載。作者在自己個人官方網站說要調整身心及充電，可是Evening的編輯部卻公開宣佈正式連載終了。

2008年7月在《{{link-ja|A-ZERO|A-ZERO}}》（[[雙葉社|雙葉社]]）連載4頁全彩色漫畫《ZERO的笑點》，至2009年6月該誌休刊之後而休載。
