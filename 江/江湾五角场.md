'''江湾五角场'''，通称'''五角场'''（{{Sha|p1=hhngokshan|ipa=[ɦǹ̩kőʔzã᷆]}}）,位于上海市区东北角的[[杨浦区|杨浦区]]，是[[四平路_(上海)|四平路]]、[[黄兴路_(上海)|黄兴路]]、[[翔殷路|翔殷路]]、[[淞沪路|淞沪路]]和[[邯郸路_(上海)|邯郸路]]5条干道的交汇处。因此地原属[[宝山区_(上海)|宝山县]]江湾乡，故名江湾五角场。

== 历史 ==
民国初年五角场一带原来是上海[[江湾镇|江湾镇]]以东的一片荒野，1920年代[[复旦大学|复旦大学]]迁来此处。1930年代，[[上海市政府_(中华民国)|上海市政府]]为了与[[上海租界|租界]]竞争，在[[黄浦江|黄浦江]]下游方向开辟新区，修筑许多道路，这5条干道就是修筑于那时，形成交通枢纽。根据孙中山的《[[建国大纲|建国大纲]]》，[[上海特别市政府|上海特别市政府]]于1929年通过[[大上海計劃|「大上海计划」]]，将此处作为新上海市中心区。抗战前已在五角场的东北方建成市政府、博物馆、图书馆等中国古典式建筑，和[[江湾体育场|江湾体育场]]。抗战期间日本人继续经营，建成恒产株式会社（[[南京政治学院|南京政治学院]]分院）等。此后主要发展为文教区，附近有[[国防医学院|国防医学院]]及继承其校址的[[中国人民解放军海军军医大学|中国人民解放军海军军医大学]]、[[复旦大学邯郸校区|复旦大学邯郸校区]]及[[复旦大学江湾校区|江湾校区]]、[[同济大学|同济大学]]、[[上海财经大学|上海财经大学]]、[[上海体育学院|上海体育学院]]、[[中国人民解放军南京政治学院|中国人民解放军南京政治学院]]分院，以及[[长海医院|长海医院]]等。1990年代以后，5条干道交汇处附近陆续开设一批商店。<ref name="文汇报">{{cite web |url = http://2010.qq.com/a/20090915/000055.htm |title = 五角场历史变迁 老城区转型为上海城市副中心 |publisher=文汇报 }}</ref>

== 当今 ==
如今的杨浦五角场是上海市府确定的上海10大商业中心之一，和徐家汇、四川北路、淮海路、中山公园等商业中心齐名。现在的五角场已与「徐家汇繁华商业副中心」，「浦东花木高尚居住副中心」，「普陀真如交通枢纽副中心」并列为「上海四大城市副中心区」之一，能为上海居民提供现代化高端商业服务，是北上海旗舰型高端商业中心。五角场的发展较晚，但在整体规划上有着较高的起点，五角场并不是单一的大上海商业副中心，而是和复旦大学、同济大学等高校相结合，形成校区、知识产业园区、现代生活区和商业区融为一体，相互呼应发展的综合模式化城市副中心。

==江湾历史文化风貌区==
江湾历史文化风貌区规划范围由中原路－虬江－黑山路－政通路－国和路－翔殷路－黄兴路－国权路－邯郸路－淞沪路－闸殷路－世界路－嫩江路所围合的区域，总面积为457公顷（其中核心保护范围的面积为63.5公顷）。<ref>[http://www.shgtj.gov.cn/xxbs/yp/200904/t20090424_298910.htm 上海市规划和国土资源管理局·《上海市江湾历史文化风貌区保护规划修编》获批复]</ref>
