'''江苏北路'''是[[中国|中国]][[上海市|上海市]][[长宁区|长宁区]]的一条东西走向交通干道，为上海市区三横三纵主干道中“西纵”的一部分。道路南起[[长宁路|长宁路]]与[[江苏路_(上海)|江苏路]]连接，北至[[万航渡路|万航渡路]][[苏州河|苏州河]][[曹杨路桥|曹杨路桥]]，长0.648千米<ref>[http://www.shtong.gov.cn/newsite/node2/node2245/node68289/node68294/node68312/node68325/userobject1ai65726.html 上海市地方志 专业志 >> 上海市政工程志 >> 第一篇市区道路 >> 第二章主要干道 >> 节]</ref>。
== 历史 ==
20世纪70年代，上海市区跨苏州河的最后一座木质桥梁——曹杨路桥拆除，1980年在老桥原址重建一座新桥。新曹杨路桥北端连通[[曹杨路|曹杨路]]，而南端因原先与老桥连接的道路只有东西向万航渡路，与桥呈丁字型连接。而新桥桥面比老桥高，桥梁将直接跨过紧靠苏州河的万航渡路，下桥后无路连通，遂决定辟筑新路<ref>[http://www.shtong.gov.cn/newsite/node2/node2245/node70862/node70871/node70930/node70938/userobject1ai73739.html 上海市地方志 专业志 >> 上海地名志 >> 第五篇交通地名（二） >> 第三章桥梁、隧道 >> 节]</ref>。
<br>1980年3月，道路新辟工程开工，同年9月，道路建成。因道路南端连接[[江苏路_(上海)|江苏路]]，为江苏路北延伸段，故取名'''江苏北路'''<ref>[http://www.shtong.gov.cn/newsite/node2/node2245/node68289/node68294/node68312/node68325/userobject1ai65726.html 上海市地方志 专业志 >> 上海市政工程志 >> 第一篇市区道路 >> 第二章主要干道 >> 节]</ref>。
