'''江川路街道'''，是{{PRC admin/fullname|31/01/12/001/000|3}}下辖的一个[[街道办事处|街道办事处]]<ref>{{PRC admin/ref sgc|{{PRC admin/parent|31/01/12/001/000}}}}</ref>，即原先的'''闵行卫星城'''。[[上海人|上海人]]习惯将江川路街道所辖区域称之为“'''老闵行'''”，或直接称作“闵行”{{NoteTag|例如[[上海公交徐闵线|上海公交徐闵线]]，称其南端终点站在“闵行”。}}<ref>{{cite web |title=《老闵行》 |url = http://c.wanfangdata.com.cn/LocalChronicle-fz201206882.aspx |website =
[[万方数据|万方数据]] }}</ref><ref name="人文历史">{{Cite web |url=http://jcjd.shmh.gov.cn/ContPageDetail.aspx?objid=d5bb12f1-d555-45f4-91c1-6ed924565318 |title=上海市闵行区江川路街道网站>社区概况>人文历史 |access-date=2015-07-03 |archive-url=https://web.archive.org/web/20150704120936/http://jcjd.shmh.gov.cn/ContPageDetail.aspx?objid=d5bb12f1-d555-45f4-91c1-6ed924565318 |archive-date=2015-07-04 |dead-url=yes }}</ref>。而大致以今北街居委会所辖区域为'''闵行老街'''，但随着自1980年代起的改造和开发，闵行老街的历史风貌已基本消失<ref name="人文历史"></ref>。面积30.27平方公里。户籍人口13.2万人<sup>（2011年）</sup>。

==历史沿革==
'''闵行'''，亦称'''敏行'''<ref>{{cite book|author1=郭经|author2=唐锦|title=弘治《上海县志》|chapter=卷2《镇市》|quote=敏行市在十六保|publisher=上海市通志馆|year=1937|page=(影印明弘治本)23}}</ref>，处于原[[上海县|上海县]]的南界、原[[松江府|松江府]]的东北面<ref>《松江府志·卷一》：上海县在府东北……南至闵行镇、华亭县界七十二里</ref>，得名似与黄浦江在此水流湍急有关。因其地势较高、不遭水患，传有“三世修来住在闵行前”之谚<ref>顾翰《松江竹枝词》：“春申江上浪滔天，劫火烧来断水边。妾苦今生修未到，郎家不住闵行前。”评注：传有“三世修来住在闵行前”之谚，人以为无水旱之患也。庚申郡城失守，粤匪南北骚扰，独不渡浦，而闵行果依然无恙，前言始验。前人都务农最怕水旱，而居闵行则无忧。前人皆务农，最怕水旱之患，闵行无忧，故有是谚。</ref><ref name="rwls">{{cite web|url=http://jcjd.shmh.gov.cn/sites/mhjc/dyn/ViewCon_pg.ashx?ctgId=457b3730-e559-4f00-818b-07a487b88c50&infId=e3fc2f08-0a18-4ce9-86a7-8e4d2291c5d0&leftBarId=7932bcb3-3e89-45c6-ac65-6ae7e3ee9428|title=人文历史|website=上海市闵行区江川路街道办事处|archiveurl=https://web.archive.org/web/20171006131855/http://jcjd.shmh.gov.cn/sites/mhjc/dyn/ViewCon_pg.ashx?ctgId=457b3730-e559-4f00-818b-07a487b88c50&infId=e3fc2f08-0a18-4ce9-86a7-8e4d2291c5d0&leftBarId=7932bcb3-3e89-45c6-ac65-6ae7e3ee9428|archivedate=2017-10-06}}</ref>。该镇至今已约500年；但从元延祐七年和至正三年分别建有洞真道院、度门寺及明洪武六年利用民居建立黄浦巡检司署推断，元末明初该地已具集镇雏型，距今约近700年<ref>{{cite book|author=上海市闵行区人民政府|title=上海市闵行区地名志|chapter=行政区域|publisher=中国大百科全书出版社上海分社|year=1991|page=23}}</ref>。至清朝晚期，这里已是商贸兴盛之地。1928年市县分治后，闵行镇为上海县首镇，店铺林立，市面繁荣，人称“小上海”。1950年代，上海市开始建设闵行工业区；1980年代，闵行卫星城大开发，老镇因此被拆毁<ref name="rwls" />，留下的项家老宅旧址、节孝坊等遗址，现均为闵行区文物保护单位<ref>{{cite web |language =  |publisher = 上海市文化广播影视管理局 |title = 闵行区县级文物保护单位、登记不可移动文物 |url = http://wgj.sh.gov.cn/wgj/node1257/node1264/u1a75614.html |author = 市文广影视局 |date = 2011-10-28 |archivedate = 2012-05-12|archiveurl=https://web.archive.org/web/20120512173402/http://wgj.sh.gov.cn/wgj/node1257/node1264/u1a75614.html}}</ref>。在1991年新的闵行区建立后，原来的闵行镇区域被对应称为“老闵行”。2000年10月[[上海市人民政府|上海市人民政府]]下发文件，批准撤销华坪路、碧江路两个街道办事处，成立江川路街道办事处<ref name="mhb-kd">{{cite web|title=时间的刻度 历史的脚步|url=http://mhb.shmh.gov.cn/content/2012-09/07/content_8624.htm|website=闵行报|publisher=闵行报社|archiveurl=https://web.archive.org/web/20171014042425/http://mhb.shmh.gov.cn/content/2012-09/07/content_8624.htm|archivedate=2017-10-14|date=2017-10-14|deadurl=yes}}</ref>。

== 道路命名 ==

江川路街道内1949年后新建的道路大多以云南地名命名，例如[[江川路|江川路]]、[[剑川路|剑川路]]<ref>{{cite web |author1=闵行报社 |title=隐藏在闵行大街小巷的“秘密”，是该让你们知道了 |url=https://www.jfdaily.com/news/detail?id=38512 |accessdate=2019-03-08}}</ref>。此外，这些道路另有编号，其中单数道路为东西向，双数道路为南北向。

* 一号路：江川路
* 二号路：华宁路
* 三号路：剑川路
* 四号路：临沧路
* 五号路：景谷路
* 六号路：昆阳路
* 七号路：鹤庆路
* 十号路：瑞丽路
* 十二号路：兰坪路

== 文物保护点 ==

江川路街道共有16处闵行区文物保护点<ref>{{cite web |author1=上海市闵行区文化广播影视管理局 |title=闵行区不可移动文物保护要求责任告知函 |url=http://xxgk.shmh.gov.cn/mhxxgkweb/html/mh_xxgk/xxgk_whgb_ywxx/2018-06-19/Detail_45299.htm |accessdate=2019-02-24}}</ref>：

临沧路：积翠牌坊、八棱石井栏、崇德桥

建设路：李园旧址

新闵路：[[慕莲精舍|慕莲精舍]]、[[清净庵旧址|清净庵旧址]]、中孚化学制造染料厂股份有限公司闵行厂旧址

闵东路：[[亦庐旧址|亦庐旧址]]

华宁路：上海锅炉厂旧址、上海重型机床厂旧址

丽江路：闵行发电厂旧址

宾川路：电机新村旧址

江川路：上海汽轮机厂旧址、上海电机厂旧址、上海重型机器厂旧址

兰坪路：闵行饭店
