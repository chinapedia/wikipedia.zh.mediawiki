'''图森'''（{{lang-en|Tucson}}，又譯'''杜桑'''或'''土桑'''），位于[[美国|美国]][[亚利桑那州|亚利桑那州]]南部[[皮马县_(亚利桑那州)|皮马县]]，是该州南部第一大城市，全州第二大城市（[[凤凰城|凤凰城]]为第一大）、美国第32大城市和第52大都会区。根據[[美國2010年人口普查|2010年人口普查]]，图森市辖区共有人口520,116人，<ref>{{cite web|url=http://factfinder2.census.gov/faces/tableservices/jsf/pages/productview.xhtml?pid=DEC_10_PL_GCTPL2.ST13&prodType=table |title=Fact Finder |publisher=U.S. Census |date= |accessdate=2012-01-22 |deadurl=yes |archiveurl=https://web.archive.org/web/20110721034521/http://factfinder2.census.gov/faces/tableservices/jsf/pages/productview.xhtml?pid=DEC_10_PL_GCTPL2.ST13&prodType=table |archivedate=2011-07-21 }}</ref>图森大都会区（由图森市以及临近的[[南图森_(亚利桑那州)|南图森]]、[[萨瓦里塔_(亚利桑那州)|萨瓦利塔]]、[[奥罗谷_(亚利桑那州)|欧罗谷]]、[[马拉纳_(亚利桑那州)|马拉纳市]]组成）則共有人口989,569人<ref>[http://www.census.gov/popest/data/metro/totals/2011/tables/CBSA-EST2011-01.csv metro population totals]</ref>。图森也是[[皮马县|皮马县]]的[[县治|县治]]所在。{{GR|6}}该城位于凤凰城-图森太阳-{谷}-走廊，三面环山。干旱充满阳光的气候长期吸引了全国各地的退休者前来养老。光学制造、航空以及电子工业是该城的支柱产业。<ref>Long, Levy J. (October 10, 2006). [https://web.archive.org/web/20080528024424/http://www.azstarnet.com/sn/85704/150067 "Luxury labels: Tiffany, Louis Vuitton could inspire more high-end retailers to try Tucson."] Arizona Daily Star. Retrieved 2010-03-12.</ref>[[雷神公司|雷神]]和[[庞巴迪|庞巴迪]]公司在图森设有生产研发基地。<ref>{{Cite news|url=https://www.insidetucsonbusiness.com/news/tucson-s-largest-employers/article_84bac4f2-525c-11e5-a946-c7bcc58bb756.html|title=Tucson’s largest employers|last=Business|first=David Rupkalvis,  Inside Tucson|newspaper=Inside Tucson Business|accessdate=2018-11-02|language=en}}</ref>[[皮马社区大学|皮马社区大学]]、[[亚利桑那大学|亚利桑那大学]]與[[美國國家光學天文台|美國國家光學天文台]]總部位于图森。<ref>[http://www.visittucson.org/articles/index.cfm?action=View&ArticleID=355&contentid=24 University of Arizona Opens Sky Center To Public] (Press Release at VisitTucson.org). Retrieved 2010-12-16.</ref><ref>[http://skycenter.arizona.edu/about About us] {{Wayback|url=http://skycenter.arizona.edu/about |date=20101125152215 }}, Mt. Lemmon Sky Center. == 交通 ==
图森距离[[美墨邊界|美墨边境]]仅69英里，因此是两国贸易的前哨和重要枢纽。连接美墨两国的[[19號州際公路|19号州际公路]]北起图森南达[[诺加利斯_(亚利桑那州)|诺加列斯]]并直通墨西哥[[索諾拉州|索诺拉州]]。同时[[10號州際公路|10号州际公路]]斜穿全城，将图森与[[洛杉矶|洛杉矶]]，[[鳳凰城_(亞利桑那州)|凤凰城]]和[[艾爾帕索_(德克薩斯州)|埃尔帕索]]直接相连。<ref name="TDOT-1Oct2009">{{cite press release |title = Federal Transit Administration Gives Approval to the Tucson Modern Streetcar Project |publisher = Tucson Department of Transportation |date = October 1, 2009 |url = http://dot.tucsonaz.gov/news/details.cfm?id=604 |accessdate = 2010-03-12 |archive-url = https://web.archive.org/web/20100527213837/http://dot.tucsonaz.gov/news/details.cfm?id=604 |archive-date = 2010-05-27 |dead-url = yes }}</ref>

[[灰狗巴士|灰狗巴士]]和[[FlixBus|FlixBus]]为图森提供城际巴士服务，可借之到达全美和[[墨西哥|墨西哥]]北部几乎所有城市。

墨西哥跨国巴士公司Tufesa和[http://taproyal.com TapLoyal]也在图森设有车站，提供开往美国西海岸和墨西哥多个目的地的直达巴士。

美铁[[美鐵|Amtrak]]的[[黃昏特快號列車|日落特快号]]和德州之鹰号途径图森。每周有三班开往[[聯合車站_(洛杉磯)|洛杉矶联合车站]]，[[路易斯安那州|路易斯安那州]][[新奥尔良|新奥尔良]]，以及[[芝加哥|芝加哥]]的旅客火车<ref>{{Cite journal|title=Tucson, Arizona|url=https://en.wikipedia.org/w/index.php?title=Tucson,_Arizona&oldid=865257479|date=2018-10-22|journal=Wikipedia|language=en}}</ref>。

公共交通方面，有市区巴士SunTran，农村客运小巴SunShuttle，以及连接边远郊区的高速通勤巴士服务SunExpress<ref>{{Cite web|url=http://suntran.com/|title=Welcome to Sun Tran - Tucson, Arizona|accessdate=2018-11-02|work=suntran.com}}</ref>。同时，市内还有一条横向贯穿市中心的现代化电车线路SunLink<ref>{{Cite web|url=http://www.sunlinkstreetcar.com|title=Sun Link - The Tucson Streetcar|accessdate=2018-11-02|work=www.sunlinkstreetcar.com|language=en}}</ref>。

[[图森国际机场|图森国际机场]]是该区域主要的国际空港。

==地理==
[[File:TucsonAZ_ISS009-E-10382.jpg|thumb]]

根據[[美國2000年人口普查|2000年人口普查]]，本城面積為{{convert|227.0|sqmi|sqkm|2}}，其中{{convert|226.7|sqmi|sqkm|2}}為陸地，{{convert|0.3|sqmi|sqkm|2}}為水域。<ref name ="Gazetteer files">{{cite web|title=US Gazetteer files 2010|url=http://www.census.gov/geo/www/gazetteer/files/Gaz_places_national.txt|publisher=[[United_States_Census_Bureau|United States Census Bureau]]|accessdate=2013-01-25}}</ref>本城的海拔為{{convert|2643|ft|0|abbr=on}}。<ref>{{cite web|url=http://www.airnav.com/airport/TUS |title=KTUS Tucson International Airport Tucson, Arizona, USA |publisher=Airnav.com |date= |accessdate=2012-01-22}}</ref>

图森属副热带[[沙漠气候|沙漠气候]]，冬无严寒、夏酷暑漫长，受西南雨季的影响：7月和8月为降雨最多的月份。由高海拔，较[[鳳凰城_(亞利桑那州)|凤凰城]]与在周内沿[[科罗拉多河|科罗拉多河]]地区，图森的夏季气温低大约3–4 ℃。根据1981至2010年在图森国际机场的数据，日最低气温{{convert|0|C}}以下的平均日数为13天，日最高气温低于{{convert|10|C}}的年均日数为3.0天；日高温{{convert|30|C}}以上的日数年均有181天，{{convert|40|C}}以上的有28天。<ref name = "NWS Tucson, AZ"/>最冷月（12月）均温{{convert|51.9|F|1|disp=flip}}，极端最低气温{{convert|6|F|0|disp=flip}}（1913年1月7日）。<ref name = "NWS Tucson, AZ"/>最热月（7月）均温{{convert|87.0|F|1|disp=flip}}，极端最高气温{{convert|117|F|0|disp=flip}}（1990年6月26日）。<ref name = "NWS Tucson, AZ"/>年均降水量约{{convert|11.55|in|mm|disp=flip}}，年极端最少降水量为{{convert|5.07|in|mm|disp=flip}}（1924年），最多为{{convert|24.17|in|mm|disp=flip}}（1905年）。<ref name = "NWS Tucson, AZ"/>无霜期平均有290天（2月14日至11月30日）。<ref name = "NWS Tucson, AZ"/>雪为罕见，而36%的冬季出现可测量的降雪<!--1893至2013年当中41个冬季，不包括7个数据不足的冬季-->；冬季极端最多积累降雪量为{{convert|6.8|in|cm|disp=flip}}（1971–72年）。<ref name = "NWS Tucson, AZ"/>

== 文化 ==
在1853年[[蓋茲登購地|盖茨登购地]]之前，图森一直属于[[墨西哥|墨西哥]]领土。再加上由于十分临近墨西哥边境，拉丁裔移民对该城市文化带来的影响尤其显著。每年在图森举办的文化活动有全世界规模最大的[[图森宝石和矿产博览会|图森宝石和矿产博览会]]（Tucson Gem & Mineral Show），[[亡灵大游行|亡灵大游行]]（All Souls Procession）<ref>{{Cite web|url=https://allsoulsprocession.org|title=All Souls Procession Weekend|accessdate=2018-11-02|work=All Souls Procession|language=en-US}}</ref>以及在美国西南地区规模最大的图森美食节（Tucson Meet Yourself）。
