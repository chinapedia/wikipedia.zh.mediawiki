'''崇武镇'''是位于[[中华人民共和国|中国]][[福建省|福建省]][[泉州市|泉州市]][[惠安县|惠安县]]下辖的一个[[镇|镇]]，位于惠安县东南部的崇武半岛。崇武镇以当地的[[崇武古城|崇武古城]]及[[惠安女|惠安女]]著称。

== 历史沿革 ==

宋初，现崇武镇地界设小兜巡检寨。明洪武二十年(1387年)，建崇武城。民国初，崇武为第二区公所驻地。民国28年，建崇武镇。1961年，崇武人民公社成立。1965年，[[中国人民解放军海军|中国人民解放军海军]]东海舰队在此附近海域，击沉由[[澎湖|澎湖]]出航的中华民国海军“永昌”号护航炮舰和“永泰”号猎潜舰，史称“崇武海战”。1984年，恢复崇武镇，辖12个居委会。

== 气候 ==
崇武镇地处[[亚热带|亚热带]]，属[[海洋性季风气候|海洋性季风气候]]。年平均温度19℃，[[无霜期|无霜期]]达306天，年平均[[降水量|降水量]]为1010毫米，年日照2112小时。

== 教育 ==
[[File:Temple_of_People's_Liberation_Army.JPG|thumb]]供奉的是27名[[解放军|解放军]]，匾額寫“天下第一奇庙”]]
崇武镇有中学3所(崇武中学、五峰中学、岞港中学，其中崇武中学是一所完全中学设置初中部和高中部，五峰中学、岞港中学设置初中部)<ref>[http://www.qzedu.cn/content.aspx?uni=f6708005-85a1-4925-a091-10842cf07bb4 惠安县中学名册]</ref>、小学11所<ref>[http://222.77.71.165/work/shownews.asp?NewsID=3756 惠安县小学教育一览表] {{webarchive|url=https://web.archive.org/web/20090813034740/http://222.77.71.165/work/shownews.asp?NewsID=3756 |date=2009-08-13 }}</ref>。

== 旅游及名胜古迹 ==

崇武旅游业较为发达，主要景点及名胜古迹有崇武古城、西沙湾惠女风情园、大岞山新石器时代遗址等。

[[崇武古城|崇武古城]]为中华人民共和国[[全国重点文物保护单位|全国重点文物保护单位]]之一。

== 著名人物 ==
*[[黄吾野|黄克晦]]（1524年～1590年）：明朝诗人，著有《金陵稿》10卷，《北游草》、《苏州草》各6卷，《宛城集》3卷，《匡庐集》10卷等<ref>[http://www.fjsen.com/yhzh/2015-02/12/content_15685829_all.htm]，崇武诗人布衣名士——黄吾野。</ref>。
*[[戴一俊|戴一俊]]（1530年－1607年）：明朝官员，著有诗文结集为《石室藏稿》2卷。
*[[蒋馨|蒋馨]]（1873年－1933年）：著名石匠，在民国十五年（1926年）时曾负责提供南京[[中山陵|中山陵]]石碑的石材。
*[[张乾二|-{zh-hans:张乾二;zh-hant:張乾二}-]]（1928年8月－）：中国科学院院士、理论化学家。

== 地方志 ==
[[明朝|明朝]]，[[叶春及|叶春及]]撰有《惠安政书、崇武所城志》。[[嘉靖|嘉靖]]21年，崇武千户朱肜编纂的《莅官总要》是崇武的首部地方志书。

1996年，惠安县崇武镇地方志编纂委员会曾编印《崇武镇志》。
