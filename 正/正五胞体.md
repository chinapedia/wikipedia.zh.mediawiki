'''正五胞体'''是一种[[四维凸正多胞体|四维凸正多胞体]]，其展开为五个[[正四面体|正四面体]]。正五胞体的投影的形状可以想象成一个[[双正三棱锥|双正三棱锥]]的两顶点再加一条连线,或者是一个[[正四面体|正四面体]]的四顶点连线至中心，在这里，正五胞体作为正的正四面体面锥出现的。正五胞体有四个交面（[[等边三角形|等边三角形]]），十条棱和五个顶点。正五胞体是最简单的四维正多胞体（如同三角形是最简单的[[多边形|多边形]]）。<br>
正五胞体是四维的[[单纯形|正单纯形]]，这是一系列具有相同性质的多胞形的总称，这一家族的特性在正五胞体上也体现出来了。五胞体是四维最简单的多胞体，任何顶点数、棱数、面数、胞数比它小的多胞体都只能成为退化多胞体（即它们并不真正具有真实的、非零的超体积）。正五胞体的顶点排布是让五个点在四维空间中两两间距离都相等的唯一方案。正五胞体同其它面为正三角形的多胞形一样，具有稳定性，即如果正五胞体10条棱长都确定了，则正五胞体就被唯一确定了。
== 几何性质 ==
正五胞体作为一个单纯形，是自身对偶的。当它穿过三维空间时其截体积最大时，其截体是一个[[半正多面體|半正]]的[[正三棱柱|正三棱柱]]。它的[[二胞角|二胞角]]度数是cos<sup>-1</sup>(1/4)，约等于75.52°。对于一个边长为a的正五胞体，其超体积是<math>\cfrac{\sqrt{5}a^4}{96}</math>,表体积是<math>\cfrac{5\sqrt{2}a^3}{12}</math>。

若一个正五胞体的棱长为1，则其外接超球的半径为<math>\frac{\sqrt{10}}{5} \approx0.632455532</math>,外中交超球(经过正五胞体各棱中点的三维超球)半径为<math>\frac{\sqrt{15}}{10} \approx0.3872983346</math>，内中交超球(经过正五胞体各面中心的三维超球)半径为<math>\frac{\sqrt{15}}{15} \approx0.2581988897</math>，内切超球半径为<math>\frac{\sqrt{10}}{20} \approx0.158113883</math>。
=== 顶点坐标 ===
对于一个边长为2，中心在四维直角坐标系原点上的正五胞体，它的5个顶点坐标分别是

:<math>\left( \frac{1}{\sqrt{10}},\  \frac{1}{\sqrt{6}},\  \frac{1}{\sqrt{3}},\  \pm1\right)</math>
:<math>\left( \frac{1}{\sqrt{10}},\  \frac{1}{\sqrt{6}},\  \frac{-2}{\sqrt{3}},\ 0   \right)</math>
:<math>\left( \frac{1}{\sqrt{10}},\  -\sqrt{\frac{3}{2}},\ 0,\                   0   \right)</math>
:<math>\left( -2\sqrt{\frac{2}{5}},\ 0,\                   0,\                   0   \right)</math>

如果把正五胞体作为一个五维直角坐标系中的四维平面，则它的顶点坐标会简单得多，为(0,0,0,0,1)或(0,1,1,1,1)的全[[排列|排列]]（其中正五胞体棱长为<math>\sqrt{2}</math>），分别对应[[:en:5-orthoplex|五维正轴形]]（正三十二超胞体）或[[:en:5-demicube|五维半正方体]]。

=== 对称群结构 ===
正五胞体属于四维[[单纯形|单纯形]]，它有着A<sub>4</sub>对称结构，对应施莱夫利符号{3,3,3}，考斯特符号{{CDD|node_1|3|node|3|node|3|node}}，该群的群阶为120。
