'''正弦曲線'''或'''正弦波'''（'''{{lang|en|Sinusoid}}'''／'''{{lang|en|Sine wave}}'''）是一種來自[[數學|數學]][[三角函數|三角函數]]中的正弦比例的曲線。也是[[模拟信号|模拟信号]]的代表，與代表[[數位信號|數位信號]]的[[方波|方波]]相對。

== 一般形式 ==

'''正弦曲線'''的形狀就像完美的海上[[波浪|波浪]]般，以三角函數正弦比例改變而形成。

標準的純正弦函數公式為

: <math> y = \sin(x) </math>

sin(x) 為[[正弦函數|正弦函數]]。

而一般應用的正弦曲線公式為

: <math>y = A \cdot \sin ( \omega t \pm \theta )</math>

A 為[[波幅|波幅]]（縱軸）， ω 為[[角频率|角频率]]， t 為[[時間|時間]]（橫軸）， θ 為[[相位|相偏移]]（橫軸左右）。

以下的公式則擁有全部的可用參數

: <math>y = A \cdot \sin ( kx - \omega t - \theta ) + D</math>

k 為[[波數|波數]]（周期密度）， D 為（直流）偏移量（y軸高低）。

== 产生 ==
[[File:ComplexSinInATimeAxe.gif|thumb]]
'''正弦曲線'''的出現和應用非常廣泛，可經常見於研究和使用於：
* [[信號處理|信號處理]]的[[模擬信號|模擬信號]]
* [[物理|物理]]的[[簡諧運動|簡諧運動]]
* [[聲學|聲學]]的聲音空氣[[振動|振動]]
* [[樂器|樂器]][[音叉|音叉]]的振動波
* [[頻率產生器|頻率產生器]]的[[輸出|輸出]]
* [[交流電|交流電]]的[[電壓|電壓]]改變
等等。

即使是其它不規則的非正弦波，其實亦能夠以不同[[周期|周期]]和[[波幅|波幅]]的正弦波集合來表示。這類將複雜波段化成正弦波的技術稱為[[傅立葉分析|傅立葉分析]]。
