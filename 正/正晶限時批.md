《'''正晶限時批'''》是[[臺灣|臺灣]][[壹電視新聞台|壹電視新聞台]]（簡稱「壹新聞」）的[[政論節目|政論]][[談話性節目|談話性節目]]，採錄影播出。現任主持人為[[彭文正|彭文正]]、[[李晶玉|李晶玉]]搭檔。於2014年8月開播至2016年7月31日。2016年8月22日起，原主持人跳槽至[[民視|民視]]，主持全新節目《[[政經看民視|政經看民視]]》。

== 節目大事記 ==
*2014年8月4日，節目開播，並在[[2014年臺灣高雄氣爆事故|高雄氣爆事故]]現場進行LIVE直播。
*2014年8月11日，節目時段改為晚間八點至晚間十點。
*2014年11月1日，節目增加周末時段。
*2015年3月16日，彭文正、李晶玉辭去主持工作。<ref>[http://news.ltn.com.tw/news/politics/breakingnews/1258827 「慈濟封不了我口」 彭文正偕妻退出《正晶限時批》] 2015年03月16日</ref>
*2015年3月17日，汪潔民、蔡沁瑜、邱沁宜代班主持工作。
*2015年4月5日，彭文正、李晶玉與壹電視董事長練台生面談後，確定回鍋，汪潔民、蔡沁瑜、邱沁宜代班主持工作結束。
*2015年4月6日，彭文正、李晶玉回鍋，下午進棚錄影，節目名稱確定不變。
*2015年4月6日，壹電視將去年8月4日開播前的宣傳廣告再度拿出來播映，且從下午3點16分起，滾動字幕循環滾動「壹電視新聞台四月六日晚間八點推出節目「正晶限時批」。
*2015年4月17日，在節目進入廣告前，加入「下段預告」。
*2015年9月8日，壹電視因系統設備故障，將《正晶限時批》延後一小時播出。原播出的晚間八點至九點時段改由[[蕭子新|蕭子新]]播出《整點新聞》。<ref>{{cite news | author = [[千禧年雜誌|千禧年雜誌]]突發中心 |title=《壹電視》系統故障 《正晶限時批》今晚延後播出 |url=http://millenniummag-breakingnews.blogspot.tw/2015/09/08-1.html |work=《[[千禧年雜誌|千禧年雜誌]]》 |date=2015-09-08 |language=zh-tw}}</ref>
*2016年7月31日，因彭文正、李晶玉於同年8月22日於民視以《政經看民視》名稱，接下原《頭家來開講》主持人謝志偉的政論節目；因此《正晶限時批》確定停播<ref>[http://ent.ltn.com.tw/news/breakingnews/1774970 正晶拒向中國靠攏 確定跳槽民視再開講]，[[自由時報|自由時報]]，2016年07月26日。</ref>，當日也為《正晶限時批》最後一集。

== 節目性質 ==
*節目主持人彭文正認為：「其他政論節目沒頭沒尾，討論事件不看前不看後，只做一半」，《正晶限時批》在兩小時節目內討論出結果，並在節目最後寫一篇「限時批」做總結<ref>[http://www.chinatimes.com/newspapers/20140804000560-260112 彭文正攜李晶玉對戰９點政論節目]，[[中時電子報|中時電子報]]，2014年08月04日</ref>。
*節目初期中的「政經不正經」單元，是以網路影片為主，以詼諧輕鬆的方式，來了解政治時事。

== 收視率 ==
*彭文正曾在2015年3月20日的「蔻蔻早餐」專訪內提到<ref>[https://www.youtube.com/watch?v=qqDqtJGXbgQ (現場影片)蔻蔻早餐20150320蔻蔻's VIP-彭文正&李晶玉-評論慈濟遭封口 彭文正還原真相] 2015年03月20日</ref>，節目開播之初的收視率，大約只有0.05上下，在低檔徘徊，他還開玩笑的說：「我很怕隔天早上起床看《正晶限時批》的收視率，都很怕看到0.00的數字，因為這樣我昨天就白做了」。
*而在彭文正、李晶玉請辭前，該節目衝上了所有談話性節目的第一名（約在1.00左右），在凱擘的收視率報告中，甚至贏過了同時段的[[三立電視|三立]]鄉土劇《[[世間情|世間情]]》。
*有網友在[[PTT|PTT]]八卦版爆料，在兩位主持人請辭後，隔天收視率從0.80暴跌到0.50，甚至第二天更跌到0.3，期間最低點只有0.21。居所有談話性節目之末。
*2015年10月2日，節目邀請[[吳宗憲|吳宗憲]]與[[馮光遠|馮光遠]]針對雙方對於[[第50屆金鐘獎|第50屆金鐘獎]]相關發言爭議進行辯論，收視率達到0.85，較平時收視率高出0.3，為本節目開播以來第二高的紀錄(最高為釋昭慧因為[[內湖保護區反慈濟開發事件|內湖保護區反慈濟開發事件]]參加節目時的收視率1)。<ref>[http://star.ettoday.net/news/574069 吳宗憲神嗆馮光遠收視率衝史上第二高　僅次釋昭慧]，[[ETtoday新聞雲|ETtoday新聞雲]]，2015-10-03</ref><ref>[http://ent.ltn.com.tw/news/breakingnews/1279779 弱了？正晶回鍋限時批 收視掉一半]</ref>

== 封口事件 ==
*2015年3月15日晚間節目片尾的[[藏頭詩|藏頭詩]]「黑手來了，不如歸去」透露出了辭職念頭。
{{cquote 3|抹黑施壓告官不如自修止謗<br>
不擇手段干預心如止水應對<br>
否極泰來等待黎明歸真返璞<br>
公義真理-{了}-然於胸來去自如<br>}}
*2015年3月16日，彭文正因[[內湖保護區反慈濟開發事件|內湖保護區反慈濟開發事件]]在臉書粉絲頁上發文：「[[慈濟|慈濟]]封得了壹電視，封不了我的口，更封不了社會大眾的悠悠之口」、「你關得了正晶限時批，究竟還是會被全民限時批。」隨後蘋果日報報導，彭文正將辭去主持人職位，而其妻李晶玉也將跟進<ref>{{Cite web|url=http://news.ltn.com.tw/news/politics/breakingnews/1258827|author=劉力仁|language=zh-tw|title=「慈濟封不了我口」 彭文正偕妻退出《正晶限時批》|publisher=《自由時報》|date=2015-03-16|accessdate=2015-03-16}}</ref>。

*此事一出，在八卦版引發了熱烈討論，有人將矛頭指向慈濟施壓，也有人將矛頭指向同是年代的壹電視老闆[[練台生|練台生]]。甚至有網友發起拒看正晶限時批、壹電視、年代，並揚言要包圍壹電視大樓。

*彭文正在隔天凌晨再度發文向觀眾致謝，並表示：「我們相信，熄燈是別無選擇的最好抗議，希望它會點燃媒體老闆心中智慧的燈。」

== 爭議 ==
*2014年12月2日九合一大選結束後<ref>{{Cite web|url=http://www.ettoday.net/news/20141203/433906.htm|language=zh-tw|title=政論節目圖卡有玄機　彭文正：噓！唸了NCC會約我喝咖啡|publisher=ETtoday|date=2014-12-03|accessdate=2016-01-03}}</ref>，壹電視政論節目《正晶限時批》日前大搞KUSO，製作「英金郝吳立」的圖來暗諷國民黨，卻被資深政治記者黃暐瀚在臉書痛批低俗，直言「不敢相信台灣媒體墮落到這種程度」。《正晶限時批》用總統馬英九、國安會祕書長金溥聰、台北市長郝龍斌、副總統吳敦義及新北市長朱立倫等5人大頭貼，KUSO做出有諧音梗的圖表，主持人彭文正還提醒：「這個不要唸，唸出來會被NCC約去喝咖啡。」對於壹電視的惡搞，黃暐瀚在臉書寫道，這樣低俗的「節目效果」，真的能幫忙多賺幾趴的收視率嗎？為了收視率，在大學研究所教給學生的種種，都能一秒變身？「低俗至此，無言以對。」不久後有網友找出資深政治記者黃暐瀚上《新聞挖挖哇》的片段，指出他也引用網友說法，把范可欽的名字念成「可欽范」（可侵犯），而黃暐瀚也在4日於臉書發表聲明，指出自己當時思慮不周，願意道歉。
