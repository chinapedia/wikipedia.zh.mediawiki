'''正丁基鋰'''（英文簡稱'''BuLi'''），常简称为'''丁基鋰'''，是最重要的[[有機鋰化合物|有機鋰化合物]]。其被廣泛使用於[[彈性聚合物|彈性聚合物]]如[[聚丁二烯|聚丁二烯]]與[[苯乙烯-丁二烯-苯乙烯樹脂|苯乙烯-丁二烯-苯乙烯樹脂]]（SBS）的[[聚合|聚合]]起始劑。也常在工業上與實驗室中，用於有機合成中的[[強鹼|強鹼]]（[[超強鹼|超強鹼]]）。

正丁基鋰通常以[[烷烴|烷烴]]溶液（如[[戊烷|戊烷]]、[[己烷|己烷]]、[[庚烷|庚烷]]溶液）的形式販售。

雖然正丁基鋰本身為一種無色固體，但通常以淡黃色烷烴溶液的形式出現。這種溶液若保存得當能無限期保存，<ref name=Brandsma>{{cite book | author = Brandsma, L.; Verkraijsse, H. D. | title = Preparative Polar Organometallic Chemistry I | publisher = [[Springer-Verlag|Springer-Verlag]] | location = Berlin | year = 1987 | isbn = 3-540-16916-4}}</ref>但實際上通常會隨時間緩慢分解，產生[[氫氧化鋰|氫氧化鋰]]的白色細粉狀沉澱，溶液顏色並轉為橘色。

== 結構與鍵結 ==
{{main|有機鋰化合物}}
[[File:N-butyllithium-ionic-skeletal.png|150px]]

正丁基鋰在固態與大部分溶劑中，均以[[原子簇|原子簇]]的形式存在。有機鋰化合物通常有聚集的傾向。正丁基鋰的原子簇，是由鋰和丁基末端的碳以離域共價鍵結合。<ref>Elschenbroich, C. ”Organometallics” (2006) Wiley-VCH: Weinheim. ISBN 978-3-29390-6</ref>以正丁基鋰為例，原子簇為四聚體（在[[醚|醚]]中）或六聚體（在[[環己烷|環己烷]]中）。其四聚體形成扭曲的[[立方烷|立方烷]]結構，其中鋰與''C''H<sub>2</sub>R基的末端碳位於交錯的頂點上。也可視為Li<sub>4</sub>的正四面體與[''C''H<sub>2</sub>R]<sub>4</sub>的正四面體互相交錯而成的結構。原子簇內的鍵結類似[[乙硼烷|乙硼烷]]，但因包含八個原子而更為複雜。由於正丁基鋰缺乏電子的特性，對[[路易斯鹼|路易斯鹼]]具有高度反應性。

由於碳（2.55）與鋰（0.98）[[電負度|電負度]]的巨大差異，C-Li鍵是高度極性的，電荷分離度估計約在55-95%。因此在實際用途上，正丁基鋰通常可當作丁基陰離子''n''-Bu<sup>−</sup>與鋰陽離子Li<sup>+</sup>，儘管這是不正確的：正丁基鋰並不是[[離子化合物|離子化合物]]。

== 製備 ==
標準的正丁基鋰製備方式，是將[[溴丁烷|溴丁烷]]或[[氯丁烷|氯丁烷]]與金属[[鋰|鋰]]反應：<ref name=Brandsma/>
: 2 Li + C<sub>4</sub>H<sub>9</sub>X → C<sub>4</sub>H<sub>9</sub>Li + LiX
: X = Cl, Br
此反應中的鋰含有1-3%的[[鈉|鈉]]。反應使用的溶劑可為[[苯|苯]]、[[環己烷|環己烷]]或[[乙醚|乙醚]]。當使用溴丁烷為起始物時，產物為一均勻的溶液，含有溴化鋰及正丁基鋰的混合原子簇；而正丁基鋰與[[氯化鋰|氯化鋰]]形成較弱的錯合物，故若使用氯丁烷為起始物，會產生氯化鋰的沉澱。

== 反應 ==

正丁基鋰是一種強鹼，但依反應物的不同，也可作為強[[親核劑|親核劑]]與[[還原劑|還原劑]]。正丁基鋰中的鋰核心易與非質子性[[路易斯鹼|路易斯鹼]]，如醚與[[胺|叔胺]]結合，使原子簇解聚。當作為一種強鹼使用時，此反應通常稱為[[金屬化反應|金屬化反應]]（metalation）。反應通常於[[四氫呋喃|四氫呋喃]]及[[乙醚|乙醚]]中進行，因為醚類是有機鋰衍生物的優良溶劑（見下文）。

=== 金屬化反應 ===
{{main|有機鋰化合物}}

正丁基鋰最有用的化學性質之一，是它能將許多弱[[酸|布朗斯特酸]]去除質子。[[叔丁基鋰|叔丁基鋰]]與[[仲丁基鋰|仲丁基鋰]]的鹼性更強。正丁基鋰可將許多種碳-氫鍵去除質子，尤其當[[共軛鹼|共軛鹼]]可由電子[[離域電子|離域化作用]]或雜原子（非碳原子）所穩定時。例子包括末端[[炔|炔]]（''H''-CC-R）、甲基[[硫醚|硫醚]]（''H''-CH<sub>2</sub>SR）、[[硫縮醛|硫縮醛]]（''H''-CH(SR)<sub>2</sub>，如[[二硫烷|二硫烷]]）、甲基[[膦|膦]]（''H''-CH<sub>2</sub>PR<sub>2</sub>）、[[呋喃|呋喃]]、[[噻吩|噻吩]]、[[二茂鐵|二茂鐵]]（Fe(''H''-C<sub>5</sub>H<sub>4</sub>)(C<sub>5</sub>H<sub>5</sub>）等。<ref>{{cite journal | author = Sanders, R.; Mueller-Westerhoff, U. T. | title = The Lithiation of Ferrocene and Ruthenocene - A Retraction and an Improvement | journal = [[Journal_of_Organometallic_Chemistry|Journal of Organometallic Chemistry]] | year = 1996 | volume = 512 | pages = 219-224 | doi=10.1016/0022-328X(95)05914-B}}</ref>去除質子反應產物[[丁烷|丁烷]]的高穩定性與揮發性帶來使用上的方便，但在大規模反應中，也會因產生大量易燃氣體而成為問題。

: LiC<sub>4</sub>H<sub>9</sub> + R-H → C<sub>4</sub>H<sub>10</sub> + R-Li

正丁基鋰的動力學鹼性強度受到溶劑與共溶劑的影響。能與Li<sup>+</sup>形成錯合物的配體，如四氫呋喃（THF）、[[四甲基乙二胺|四甲基乙二胺]]（TMEDA）、[[六甲基磷醯胺|六甲基磷醯胺]]（HMPA）及1,4-二氮雜雙環[2.2.2]辛烷（[[DABCO|DABCO]]）能使Li-C鍵結更為極性化，故能增進金屬化反應的速率。這些錯合物亦對鋰化反應產物的分離有所助益，如著名的例子二鋰代二茂鐵。
:Fe(C<sub>5</sub>H<sub>5</sub>)<sub>2</sub> + 2 LiC<sub>4</sub>H<sub>9</sub> + 2 TMEDA → 2 C<sub>4</sub>H<sub>10</sub> + Fe(C<sub>5</sub>H<sub>4</sub>Li)<sub>2</sub>(TMEDA)<sub>2</sub>

[[Schlosser鹼|Schlosser鹼]]是一種混合正丁基鋰與[[叔丁氧鉀|叔丁氧鉀]]的超強鹼。在動力學上此鹼的反應性比正丁基鋰更佳，因此常被用於較為困難的金屬化反應。丁氧基陰離子與鋰形成錯合物，因此產生了比鋰試劑反應性更高的[[丁基鉀|丁基鉀]]。

使用正丁基鋰為鹼的一個實例是[[胺基|胺基]]與[[碳酸二甲酯|碳酸二甲酯]]的加成，其中正丁基鋰扮演將胺去除質子的角色：
: n-BuLi + R<sub>2</sub>NH + (MeO)<sub>2</sub>CO → R<sub>2</sub>N-CO<sub>2</sub>Me + LiOMe + BuH

=== 鹵素－鋰交換 ===

正丁基鋰與一些有機溴化物與碘化物發生置換反應，產生對應的有機鋰衍生物。此反應通常無法對有機氯化物或氟化物進行：

: C<sub>4</sub>H<sub>9</sub>Li + RX → C<sub>4</sub>H<sub>9</sub>X + RLi (X = Br, I)

此反應在製備許多有機鋰化合物上非常實用，尤其是[[芳基|芳基]]鋰與某些[[乙烯基|乙烯基]]鋰。然而，此反應的實用性依然嚴重受到限制，由於反應混合物中的丁基溴與丁基碘，會與新產生的有機鋰化合物反應。另外正丁基鋰亦可當作鹼，以[[脫鹵化氫反應|脫鹵化氫反應]]與置換反應競爭：

: C<sub>4</sub>H<sub>9</sub>Br + RLi → C<sub>4</sub>H<sub>9</sub>R + LiBr

: 2C<sub>4</sub>H<sub>9</sub>Li + R'CH=CHBr → 2C<sub>4</sub>H<sub>10</sub> + R'C≡CLi + LiBr

由於碘－鋰交換的速率比溴－鋰交換高數個數量級，當使用RI時這些副反應的傾向比使用RBr大幅降低。因此芳基、乙烯基與伯烷基碘是較佳的選擇，而通常使用[[叔丁基鋰|叔丁基鋰]]而非正丁基鋰為試劑，因為產生的叔丁基碘會迅速與叔丁基鋰發生脫鹵化氫反應而消耗掉（因此需要2當量的叔丁基鋰）。另外，乙烯基鋰也可由乙烯基鹵化物（如環己烯基氯）與鋰金屬直接反應，或經由錫－鋰交換而產生（見下一章節）。<ref name=Brandsma/>

=== 轉換金屬反應 ===
[[轉換金屬反應|轉換金屬反應]]是一類與此相關的反應，顧名思義，是兩種有機金屬化合物交換金屬的反應。許多此類反應包含鋰與[[錫|錫]]的交換：
: C<sub>4</sub>H<sub>9</sub>Li + Me<sub>3</sub>SnAr → C<sub>4</sub>H<sub>9</sub>SnMe<sub>3</sub> + LiAr
: 其中Ar代表芳基，Me代表甲基
在製備有機鋰化合物時，錫－鋰交換反應與鹵素－鋰交換反應相比，有一個重要的優點，因為產物中的錫化合物（如上述例子中的C<sub>4</sub>H<sub>9</sub>SnMe<sub>3</sub>）對鋰試劑的反應性比對應的鹵化物（如C<sub>4</sub>H<sub>9</sub>Br或C<sub>4</sub>H<sub>9</sub>I）低很多。其他可進行此類轉換反應的[[金屬|金屬]]與[[類金屬|類金屬]]包括有機[[汞|汞]]、[[硒|硒]]與[[碲|碲]]化物。

=== 羰基加成 ===
包括正丁基鋰在內的有機鋰化合物，可被用於某些[[醛|醛]]與[[酮|酮]]的合成。一種合成途徑是使用有機鋰與雙取代[[醯胺|醯胺]]反應：
: R<sup>1</sup>Li + R<sup>2</sup>CONMe<sub>2</sub> → LiNMe<sub>2</sub> + R<sup>2</sup>C(O)R<sup>1</sup>

=== 碳鋰化反應 ===
正丁基鋰能與某些活化的末端[[烯烴|烯烴]]，如[[苯乙烯|苯乙烯]]、[[丁二烯|丁二烯]]甚至[[乙烯|乙烯]]進行[[加成反應|加成反應]]，產生新的有機鋰化合物。此反應是商業上應用丁基鋰製造[[聚苯乙烯|聚苯乙烯]]與[[聚丁二烯|聚丁二烯]]的基礎：
: C<sub>4</sub>H<sub>9</sub>Li + CH<sub>2</sub>=CH-C<sub>6</sub>H<sub>5</sub> → C<sub>4</sub>H<sub>9</sub>-CH<sub>2</sub>-CH(Li)-C<sub>6</sub>H<sub>5</sub>

=== 四氫呋喃的分解 ===
四氫呋喃能被正丁基鋰去除與氧相鄰的四個質子之一，尤其在[[四甲基乙二胺|TMEDA]]的存在下反應更易進行。此反應消耗正丁基鋰並產生丁烷，並引發[[環加成反應|環加成反應]]的逆反應，產生[[乙醛|乙醛]]的[[烯醇|烯醇盐]]與[[乙烯|乙烯]]。因此以四氫呋喃為溶劑的正丁基鋰反應，通常在低溫（如-78 °C，可由[[乾冰|乾冰]]與[[丙酮|丙酮]]的混合冰浴簡易產生）下進行。稍高的溫度（-25 °C甚至-15 °C）亦有所使用。

=== 熱分解 ===
當正丁基鋰加熱時，發生與其他具有β氫的有機鋰化合物相似的反應，脫去β氫陰離子並產生[[丁烯|1-丁烯]]與氫化鋰：
: C<sub>4</sub>H<sub>9</sub>Li -Δ→ LiH + CH<sub>3</sub>CH<sub>2</sub>CH=CH<sub>2</sub>

== 安全 ==
在合理的化學認知中，正丁基鋰必須在惰性氣體的密封系統中儲存與使用，以防止試劑失去活性並維持安全。正丁基鋰與水劇烈反應：
: C<sub>4</sub>H<sub>9</sub>Li + H<sub>2</sub>O → C<sub>4</sub>H<sub>10</sub> + [[氢氧化锂|LiOH]]
正丁基鋰亦與二氧化碳反應，產生[[戊酸鋰|戊酸鋰]]：
: C<sub>4</sub>H<sub>9</sub>Li + CO<sub>2</sub> → C<sub>4</sub>H<sub>9</sub>CO<sub>2</sub>Li

叔丁基鋰極易與空氣與水分反應，其[[水解反應|水解反應]]放出的熱量足夠點燃溶劑（通常為四氫呋喃、乙醚或己烷），因此當接觸空氣時便會點燃。在某些情況下（如針頭中），正丁基鋰由於產生鋰的氧化物與氫氧化物作為屏障，可能會自行隔絕空氣，阻止進一步與氧氣反應。這種情況類似在[[鋁|鋁]]金屬表面形成的氧化層。叔丁基鋰具有[[自燃|自燃]]性，而正丁基鋰則無，但為防止分解仍應於乾燥的氮氣中使用。
