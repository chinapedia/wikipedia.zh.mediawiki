'''椎名高志'''（{{lang-ja|椎名 高志}}，{{bd|1965年{{來源請求}}|6月24日<ref name="sunday20080521">{{Cite web|url=http://www.websunday.net:80/backstage/siina.html|title=椎名高志 Vol.21|work=Sunday漫畫家支援官網|publisher=[[小學館|小學館]]|accessdate=2018-05-14|archiveurl=|archivedate=|deadlinkdate=|language=ja|deadurl=}}</ref>||}}），[[日本|日本]]男性[[漫畫家|漫畫家]]，[[ABO血型系統|O型]]血<ref name="sunday20080521" />。[[大阪府|大阪府]]出生<ref name="sunday20080521" />。

{{link-ja|大阪府立箕面高等學校|大阪府立箕面高等学校}}、{{link-ja|京都市立藝術大學|京都市立芸術大学}}畢業。妻子清水彩曾擔任[[高橋留美子|高橋留美子]]的助手{{來源請求}}。

== 來歷 ==
椎名從高中時期開始，以短篇（名稱未公開）入圍「{{link-ja|小學館新人漫畫大獎|小学館新人コミック大賞}}」佳作。並於在學期間製作過動畫，當中成員包含後來的[[動畫師|動畫師]][[吉松孝博|吉松孝博]]<ref group="注">吉松在2007年的電視動畫《[[大江戶火箭|大江戶火箭]]》擔任人物設定時，部分的人物設定則邀請椎名和跟椎名都從事漫畫家的{{link-ja|源太郎|みなもと太郎}}及[[內藤泰弘|內藤泰弘]]共同負責。還有，吉松在椎名的原作漫畫改編動畫《[[楚楚可憐超能少女組|楚楚可憐超能少女組]]》則是負責片頭動畫的原畫。</ref><ref name="amgakuin">{{Cite web|date=|url=http://www.amgakuin.co.jp/contents/?p=1255|title=動畫學科 吉松孝博的專訪 - Object id #3172｜AMUSEMENT MEDIA綜合學院－東京校|work=|author=|publisher={{link-ja|AMUSEMENT MEDIA綜合學院|アミューズメントメディア総合学院}}|accessdate=2017-02-14|language=ja}}</ref>。1989年，椎名於《[[週刊少年Sunday|週刊少年Sunday]]》（[[小學館|小學館]]）發表短篇「{{link-ja|Dr.椎名の教育的指導!!|Dr.椎名の教育的指導!!}}」之後，從此正式出道。主要在《週刊少年Sunday》等雜誌活動。

出道初期，椎名大多發表[[四格漫畫|四格漫畫]]短篇，之後陸續收錄在首部單行本《{{link-ja|(有) 椎名百貨店|（有）椎名百貨店}}》。在他的作品中，內容大多以[[戲仿|戲仿]]表現。

1991年，椎名以《[[GS美神_極樂大作戰！！|GS美神 極樂大作戰!!]]》成為他的長期連載出道作。同名漫畫改編成動畫後，主角美神令子由[[鶴弘美|鶴弘美]]獻聲，這是因為，椎名表示自己在國中時期看過鶴弘美的聲優出道作《[[佩琳物語|佩琳物語]]》（舊譯小英的故事），因此成為她的聲迷<ref group="注">椎名的另一部連載《楚楚可憐超能少女組》改編成動畫後，鶴弘美在作品中幫明石秋江配音，也是鶴弘美第二次在椎名的作品中獻聲。</ref>。還有，椎名與助手結婚時，則邀請鶴弘美和在《GS美神》另一位主角橫島忠夫的聲優[[堀川亮|堀川亮]]擔任婚禮主持人。

目前，椎名在《週刊少年Sunday》連載《楚楚可憐超能少女組》，並從連載開始有一段時間之後改成每週追加2篇的4格漫畫。至2014年13號為止中學生篇連載完結，從2014年39號開始連載最終章高中生篇。《楚楚可憐超能少女組》動畫化後，椎名曾在電視動畫第13話和OVA參加配音。

== 年譜 ==
* 1989年：以短篇「{{link-ja|Dr.椎名の教育的指導!!|Dr.椎名の教育的指導!!}}」出道。
* 1992年：《[[GS美神_極樂大作戰！！|GS美神 極樂大作戰!!]]》入圍第38屆[[小學館漫畫賞|小學館漫畫獎]]少年部門<ref name="sunday20080521" />。
* 1993年：《GS美神 極樂大作戰!!》電視動畫化。
* 1994年：《GS美神 極樂大作戰!!》電影動畫上映。
* 2008年：《[[楚楚可憐超能少女組|楚楚可憐超能少女組]]》電視動畫化。
* 2010年：《楚楚可憐超能少女組》OVA化。
* 2013年：《楚楚可憐超能少女組》衍生作品《THE UNLIMITED 兵部京介》電視動畫化。
