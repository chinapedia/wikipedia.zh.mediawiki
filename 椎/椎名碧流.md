'''椎名碧流'''（{{jpn|j=椎名 へきる|hg=しいな へきる|rm=Shiina Hekiru}}，{{bd|1974年|3月12日<ref name="arts-pro">{{Cite web|url=http://www.artsvision.co.jp/talent/802/|title=ARTSVISION公式官網的聲優簡介|publisher=[[ARTSVISION|ARTSVISION]]|accessdate=2017-02-20|language=ja}}</ref>||}}），[[日本|日本]]女性[[配音員|配音員]]、[[歌手|歌手]]。出身於[[東京都|東京都]]<ref name="arts-pro" />。[[身高|身高]]152cm。A型[[血型|血]]。

== 簡介 == 
[[ARTSVISION|ARTSVISION]]所屬，[[日本播音演技研究所|日本播音演技研究所]]出身<ref name="arts-pro" />。

為人熟悉的代表作品有動畫《[[貓狗寵物街|淘氣貓 喵喵三丁目]]》的桃、《[[偶像防衛隊|偶像防衛隊]]》的取石水無、《[[魔法騎士|魔法騎士]]》的獅堂光、《{{link-ja|YAT安心！宇宙旅行|YAT安心!宇宙旅行}}》的天上院桂、《[[橫濱購物紀行|橫濱購物紀行]]》的初瀨野阿爾法、《{{link-ja|米花兔|ウサハナ}}》花兔、《{{link-ja|獸爪|ケモノヅメ}}》的上月由香、《[[地獄少女|地獄少女 三鼎]]》的山童、《[[妄想學生會|妄想學生會]]》的轟寧寧、《[[槍彈辯駁_希望學園與絕望高中生|槍彈辯駁]]》的賽蕾絲媞亞·魯丁貝魯格及《[[少女與戰車|少女與戰車]]》的蝶野亞美；遊戲《[[兵蜂系列|兵蜂]]》系列的帕斯德爾等。

=== 來歷 ===
; 1991年
* 從[[日本播音演技研究所|日本播音演技研究所]]主辦的第6期特待生試鏡會中合格。同年，與同研究所的3名女性在籍者組成4人聲優偶像團體「'''Cristal A'''」（成員包含[[手塚千春|手塚千春]]、[[藤野薰|藤野薰]]、矢澤{{lang|ja|ゆめる}}）在配音業界出道。
; 1992年
* 參加由[[角川書店|角川書店]]、[[Sony_Music_Records|Sony Music Records]]聯手舉辦的[[OVA|OVA]]《{{link-ja|精靈使 (1995年OVA)|精霊使い|精靈使}}》女主角試鏡會獲得拔擢，同時從「'''Cristal A」'''退團。1991年10月發行的廣播劇CD《{{link-ja|花之飛鳥組！|花のあすか組!}}》飾演雲雀成為她的個人配音出道作。
; 1993年
* 正式成為經紀公司[[ARTSVISION|ARTSVISION]]的旗下聲優。加入ARTSVISION之後的首部配音角色是《[[偶像防衛隊|偶像防衛隊]]》的取石水無。
; 1997年
* 4月5日，於[[日本職棒|日本職棒]][[太平洋聯盟|太平洋聯盟]]公開賽「[[北海道日本火腿鬥士|日本火腿鬥士]]對決[[千葉羅德海洋|千葉羅德海洋]]」（比賽地點在[[東京巨蛋|東京巨蛋]]）擔任開球嘉賓<ref group="注">其中椎名的粉絲在觀眾人數占了過半，主要目的是來看椎名進行開球，開球儀式結束後，椎名的粉絲一致離去，僅剩下少數的球迷和觀眾欣賞正式比賽。</ref>。同時，擔任球隊吉祥物「{{lang|ja|ファイティー}}君」的配音（至2000年為止）。
* 4度在2月17日、1998年2月2日、1999年2月8日、2000年3月27日發行的《[[Oricon|Oricon Week The Ichiban]]》選為封面人物<ref group="注">今改名《Ori★Star》。</ref>。
; 2002年
* 6月，首次在真人電影《月{{lang|ja|のあかり}}》主演。
; 2008年
* 11月，首次在商業舞台劇「歌謠劇場『彈珠汽水』」參演。
; 2014年
* 11月8日，於自己的官方網站發表與一般男性結婚<ref>{{Cite web|date=|url=http://www.hekiru-shiina.jp/ja/informations/content/20141108.html|script-title=ja:4179＊LOVE｜【ご報告】|work=|author=|publisher=椎名碧流的官方網站|accessdate=2017-02-20|language=ja}}</ref>。

=== 人物 ===
其藝名「{{lang|ja|へきる}}」取自於成語「{{lang|ja|碧い流れ}}」漢字的「碧流」，再經由椎名她自己把它轉化成[[平假名|平假名]]<ref>{{Cite web|url=http://artist.cdjournal.com/a/shiina-hekiru/111923|title=椎名碧流|work=CDJournal|accessdate=2017-02-20|language=ja}}</ref>。因此[[漢語|中文]]的正式譯名為「'''椎名碧流'''」。

官方後援會的名稱是「4179*LOVE」（即「椎名粉絲團（{{lang|ja|しいなくらぶ}}）的日文發音」）。此外，同業同行[[間島淳司|間島淳司]]也表示以前是椎名粉絲團的會員之一。

興趣和特長有[[旅行|旅行]]、[[登山運動|登山]]、寶物探索、觀察[[鳥|野鳥]]、[[騎馬|騎馬]]<ref name="arts-pro" />。討厭的事物是和[[靈異故事|恐怖怪談]]有關的話題。

據同行[[尤加奈|尤加奈]]她表示，說椎名和[[手塚千春|手塚千春]]是長年來的好友及夥伴<ref>來自《The 聲優 1995》雜誌尤加奈的話（當時藝名「野上尤加奈」）。</ref>。還有，椎名表示自己是[[NHK7點新聞|NHK7點新聞]]平日天氣預報的[[氣象預報士|氣象預報士]]{{link-ja|半井小繪|半井小絵}}的粉絲<ref>專輯《{{link-ja|Face to Face (椎名碧流專輯)|Face to Face (椎名へきるのアルバム)|Face to Face}}》的發言。</ref>。

另外，椎名與在[[電視動畫|電視動畫]]《[[地獄少女|地獄少女 三鼎]]》共演的後輩[[佐藤聰美|佐藤聰美]]有不錯的交情，所以在自己主持的廣播節目與擔任嘉賓的佐藤常有話題可聊。

椎名她在連載雑誌或者廣播節目表示自己喜歡音樂種類是[[硬式搖滾|硬式搖滾]]·[[重金屬音樂|重金屬]]樂團的音樂等，另外喜歡的樂團有[[白蛇_(樂團)|白蛇]]、{{link-ja|BOWWOW|BOWWOW}}、[[夢劇院_(美國)|夢劇院]]、[[歐洲合唱團|歐洲合唱團]]、{{link-en|Michael Schenker Group|マイケル・シェンカー・グループ|M.S.G.}}、[[深紫|深紫]]、歌手有{{link-ja|蘭迪·羅茲|Randy Rhoads}}、{{link-en|狂人安德魯|Andrew W.K.}}等。

== 音樂活動 ==
; 1994年
* 8月21日：首張專輯《{{link-ja|Shiena|Shiena}}》發行，與Sony Artist Development簽下契約，成為[[Sony_Music_Records|Sony Music Records]]的旗下歌手，正式在歌壇出道。
; 1995年
* 3月29日：首次在{{link-ja|SHIBUYA Egg-man|Shibuya_eggman}}舉行個人公開演唱。
* 2月22、23日：於[[日本武道館|日本武道館]]舉行公開演唱，也是第一位踏足日本武道館的[[聲優|聲優]]。
* 8月29日：首次以聲優身分在[[朝日電視台|朝日電視台]]系列音樂番組[[MUSIC_STATION|MUSIC STATION]]出演。
* 11月21日：於[[JFN|JFN]]系深夜節目「Deeper Street 」負責企劃一環，與[[搖滾樂|搖滾樂團]][[月之海|月之海]]的鼓手{{link-ja|真矢 (鼓手)|真矢 (ドラマー)|真矢}}（負責週三）合作發行單曲《漂流者》。
; 1998年
* 3月8日：於{{link-ja|廣島厚生年金會館|広島市文化交流会館}}舉辦她的第100場達成演唱會。
; 1999年
* 1月21日：第6張《{{link-ja|Face to Face (椎名碧流專輯)|Face to Face (椎名へきるのアルバム)|Face to Face}}》發行，並首次登上[[Oricon公信榜|Oricon]]TOP10的第6名。
* 4月17日：首次以聲優身分在[[國立代代木競技場|國立代代木競技場]]第一體育館舉辦現場演唱。
; 2000年
* 12月31日：於[[幕張展覽館|幕張展覽館]]舉辦她的第200場紀念演唱會。
; 2002年
* 12月1日：以嘉賓身分出席在日本武道館舉辦的{{link-ja|Act Against AIDS|アクト・アゲインスト・エイズ}} LIVE 2002 THE VARIETY 10公演。
; 2006年
* 3月：與樂團[[TM_NETWORK|TM NETWORK]]的[[吉他|吉他]]手、[[鋼琴|鋼琴]]手{{link-ja|木根尚登|木根尚登}}組成雙人音樂組合「{{lang|ja|ひだまり}}」。
* 7月30日：於{{link-ja|Zepp Tokyo|Zepp}}舉辦她的第300場達成演唱會。
* 12月23日：首次以聲優身分參加[[埼玉超級競技場|埼玉超級競技場]]舉辦的活動<ref group="注">但是在稍早之前以聲優身分前往參加的第一人是[[水樹奈奈|水樹奈奈]]。</ref>。
; 2009年
* 6月15日：移籍進入成為大型唱片公司[[Lantis|Lantis]]的旗下歌手<ref>《Lantis Chronicle》的Lantis官方年表。</ref>。
; 2011年
* 3月：發生[[2011年日本東北地方太平洋近海地震|東日本大震災]]，在受到震災的影響下，採取延後措施將預定4月17日在Zepp Tokyo、5月5日在仙台darwin的公演場次延後到8月同時進行。
; 2012年
* 10月7日：參加{{lang|ja|みちのくアニソンフェス}}2012 ～Eastern Gale（地點在仙台SUNPLAZA Hall）舉辦的公演演唱。
