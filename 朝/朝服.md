'''朝服'''或稱'''朝衣'''、'''具服'''，是[[中國|中國]]古代的一種[[官服|官服]]，後來傳至[[朝鮮_(稱謂)|朝鮮]]、[[越南|越南]]、[[日本|日本]]、[[琉球|琉球]]。其制度形成于漢末，通常由皇帝与大臣在[[朝會_(政治)|朝会]]之时穿著，即皇帝「視朝之服」及百官「朝饗之服」，有時也包括[[后妃|后妃]]、[[命婦|命婦]]、[[女官|女官]]朝參之特定服裝。按朝会的大小、及時間不同，對朝服著裝的要求也不一樣。如唐代[[春節|元日]]、[[冬至|冬至]]等重大朝会時，朝服必須整潔而饰物俱全。退朝之後在一般的場合則不應穿著朝服。此外，朝服也可在重大典禮之時穿著。<ref name=李怡/>。
==起源==
[[File:章懷太子墓壁畫禮賓圖a.jpg|thumb]]壁畫[[禮賓圖|禮賓圖]]中身著朝服的文官]]

朝服最早見於[[先秦|先秦]]，按《周禮·春官·司服》：「視朝，則[[皮弁服|皮弁服]]」。[[东汉|东汉]]时開始使用袍子作朝服，[[三國|三國]]以後開始形成定制。
==各國朝服==
===中國===
按《[[后汉书|后汉书]]·舆服志下》：天子的穿著採用 “深衣制，有{{link-ja|袍 (漢服系服裝)|袍}}，随五时色”，天子以下則“皆通制袍，单衣、皂缘领、袖中衣，为朝服云”。[[北朝|北朝]]之後始稱具服，唐朝時沿用此名。按《[[新唐书|新唐书]]·车服志》：“具服者，五品以上陪祭、朝飨、拜表、大事之服也，亦曰朝服。冠帻，[[簪|簪]][[导|导]]，绛纱单衣，白纱[[中单|中单]]，黑领、袖，黑褾、襈、裾，白裙、襦，[[鞢𩎃帶|金钩鰈]]，[[大帶|假带]]，[[曲领方心|曲领方心]]，绛纱[[蔽膝|蔽膝]]，白袜，乌皮[[舄|舄]]，[[剑|剑]]，[[纷|纷]]，[[鞶囊|鞶囊]]，双[[佩|佩]]，双[[绶|绶]]”。文武官的朝服相似，以[[首服|首服]]和饰物來互相區別。按《[[武德令|武德令]]》，首服包括”[[远游冠|远游冠]]、[[进贤冠|进贤冠]]、[[武弁大冠|武弁大冠]]、[[獬豸冠_(梁冠)|獬豸冠]]等”。若首服相同，則以饰件區別：“五品以上双玉导，金饰，三品以上加宝饰，六品以下无饰”，即六品一下無權佩戴“剑、佩、绶”等。<ref name=李怡>{{cite journal |author=李怡 |date= 2012|title=唐代文官朝服与公服使用礼仪变迁之探研 |url= |journal=设计艺术研究 |volume= |issue=6 |pages=96-103+124|doi= |access-date=}}</ref>
北朝到唐朝之間形成的朝服制度很大程度上被[[五代十國|五代十國]]、[[宋代|宋代]]、[[元代|元代]]、[[明代|明代]]沿用<ref name=李怡/>，但亦有一定的變化。

女性穿著朝服的相關記載始見於[[東漢|東漢]]，《[[後漢書|後漢書]]》載二千石以上的夫人至皇后以[[蠶衣|蠶衣]]（親蠶禮所穿禮服）為朝服<ref name="後漢書"> {{cite book |author=范曄|year= |title=後漢书·舆服志 |quote=自二千石夫人以上至皇后，皆以蚕衣为朝服。 |publisher=| location= | isbn= |ignore-isbn-error=true| pages= |date=|language=}}</ref>。三國時一度中斷，後來[[晉代|晉代]]及[[南朝|南朝]]歷代皆沿用以蠶衣為皇后、命婦之朝服<ref name="通典1"> {{cite book |author=|year= |title=通典卷第六十二　禮二十二　沿革二十二　嘉禮七 |quote=晉依前漢制，…自二千石夫人以上至皇后，皆以蚕衣为朝服。……宋制，……自皇后至二千石命婦，皆以蠶衣為朝服。……齊因之，……陳依前制，……自二千石以上至皇后，皆以蠶衣為朝服。 |publisher=| location= | isbn= |ignore-isbn-error=true| pages= |date=|language=}}</ref>。[[北齊|北齊]]各級命婦朝服與[[公服|公服]]相同<ref name="通典2"> {{cite book |author=范曄|year= |title=通典卷第六十二　禮二十二　沿革二十二　嘉禮七 |quote=北齊……朝服公服，各與其命婦服同。 |publisher=| location= | isbn= |ignore-isbn-error=true| pages= |date=|language=}}</ref>。

唐代至明代女性朝服制度雖然不見於輿服志，但皇后、妃嬪、女官等[[內命婦|內命婦]]以及官員妻子、有爵女性等[[外命婦|外命婦]]在大朝會或宮中其他重大場合所穿之服飾也有規定，功能上亦視為朝服<ref name='周蘇平'>{{cite journal |author=周蘇平 |date= 2000|title=唐代女子服飾略說 |url= |journal=华夏文化 |volume=4 |issue= |pages=11-13|doi= |access-date=}}</ref>。例如《[[資治通鑑|資治通鑑]]》提及唐代[[長孫皇后|長孫皇后]]穿朝服<ref name="通鑑"> {{cite book |author=司馬光|year= |title=資治通鑑·唐紀十|quote=-{后}-退，具朝服立於庭，上驚問其故。 |publisher=| location= | isbn= |ignore-isbn-error=true| pages= |date=|language=}}</ref>，所指的應是皇后朝會大事之服[[褘衣|褘衣]]。

[[清朝|清朝]]的朝服並非直接繼承明代朝服，而是改造自明朝受元朝蒙古服裝影響而出現的常服[[曳撒|曳撒]]、[[贴里|贴里]]，再加入自己的特徵，如其“剪貂皮为缘饰”，從而創造了“緣貂朝衣”<ref>{{cite journal |author=王业宏、赵丰|date= 2009|title=清入关前朝服考|url= |journal=东华大学学报(社会科学版) |volume=9 |issue=4 |pages=253-258|doi= |access-date=}}</ref>。參見[[清朝官服|清朝官服]]。

===日本===
[[File:Yokohama_Silk_Museum_057.JPG|thumb]]
日本朝服制度始見於[[飛鳥時代|飛鳥時代]]，最初收錄「朝服」一詞的日本史書是《[[日本書紀|日本書紀]]》，在[[天武天皇|天武天皇]]十四年七月庚午條首次提到朝服，但朝服制度早於[[推古天皇|推古天皇]]時期已經出現。推古天皇定[[冠位十二階|冠位十二階]]，規定了各級文武官員及[[女官|女官]]服飾，此乃日本朝服之濫觴<ref name="朝日">{{Cite web |url=https://kotobank.jp/word/%E6%9C%9D%E6%9C%8D-98165 |title=朝服 |accessdate=2017-03-30 |author= |date= |work=コトバンク |publisher=朝日新聞|language=ja}}</ref><ref>{{Cite web |url=http://www.iz2.or.jp/fukushoku/f_disp.php?page_no=0000011 |title=推古朝、朝服 |accessdate=2017-03-30 |author= |date= |work= |publisher=風俗博物館|language=ja}}</ref><ref>{{Cite web |url=http://www.iz2.or.jp/fukushoku/f_disp.php?page_no=0000012 |title=推古朝女官朝服 |accessdate=2017-03-30 |author= |date= |work= |publisher=風俗博物館|language=ja}}</ref>，[[天智天皇|天智天皇]]時改制<ref>{{Cite web |url=http://www.iz2.or.jp/fukushoku/f_disp.php?page_no=0000013 |title=天智朝文官朝服 |accessdate=2017-03-30 |author= |date= |work= |publisher=風俗博物館|language=ja}}</ref>，天武天皇十一年參照唐代服飾又再改制，沿用至[[持統天皇|持統天皇]]時期又稍作改動<ref name="朝日" /><ref name="天武1">{{Cite web |url=http://www.iz2.or.jp/fukushoku/f_disp.php?page_no=0000014 |title=天武・持統朝文官朝服 |accessdate=2017-03-30 |author= |date= |work= |publisher=風俗博物館|language=ja}}</ref><ref name="天武2">{{Cite web |url=http://www.iz2.or.jp/fukushoku/f_disp.php?page_no=0000015 |title=天武・持統朝女官朝服 |accessdate=2017-03-30 |author= |date= |work= |publisher=風俗博物館|language=ja}}</ref>。[[大寶|大寶]]元年，[[文武天皇|文武天皇]]頒佈《[[大寶令|大寶令]]》，制定服飾制度，規定五位以上官員在特定儀式須穿著朝服<ref name="朝日" />。

[[奈良時代|奈良時代]]進一步參照唐代服飾制度，[[養老_(日本年號)|養老]]二年頒佈的《[[養老令|養老令]]》在《大寶令》的基礎上再修改服飾制度，規定了[[皇族|皇族]]、文官、武官、[[命婦|命婦]]、女官的朝服，按照品秩各有差異<ref name="奈良1">{{Cite web |url=http://www.iz2.or.jp/fukushoku/f_disp.php?page_no=0000016no=0000016 |title=養老の衣服令による文官礼服 |accessdate=2017-03-30 |author= |date= |work= |publisher=風俗博物館|language=ja}}</ref>，文武官朝服最大差異在於文官所穿為縫腋、有外襬的襴袍，武官所穿為無襴闕腋袍，顏色則按照品級有所差異<ref name="奈良3">{{Cite web |url=http://www.iz2.or.jp/fukushoku/f_disp.php?page_no=0000016no=0000017 |title=武官朝服 |accessdate=2017-03-30 |author= |date= |work= |publisher=風俗博物館|language=ja}}</ref>。[[內親王|內親王]]、[[女王_(日本)|女王]]、命婦、女官朝服的首服為[[花鈿|花鈿]]、[[釵|釵子]]，六位以下須戴[[義髻|義髻]]，體衣為[[襦裙|襦裙]]，外命婦上襦顏色視其夫品秩而定<ref name="奈良2">{{Cite web |url=http://www.iz2.or.jp/fukushoku/f_disp.php?page_no=0000016no=0000017 |title=養老の衣服令による命婦礼服 |accessdate=2017-03-30 |author= |date= |work= |publisher=風俗博物館|language=ja}}</ref><ref name="養老">{{Cite web |url=http://isariya.michikusa.jp/text/ebukuryou.htm |title=養老令＊衣服令第十九　凡壱拾肆条 |accessdate=2017-03-30 |author=逆名 |date= |work= |publisher=|language=ja}}</ref><ref name="朝日" />。

[[平安時代|平安時代]]初期基本上沿襲奈良制度，只是細節有變化<ref name="平安1">{{Cite web |url=http://www.iz2.or.jp/fukushoku/f_disp.php?page_no=0000024 |title=平安初期文官朝服 |accessdate=2017-03-30 |author= |date= |work= |publisher=風俗博物館|language=ja}}</ref><ref name="平安2">{{Cite web |url=http://www.iz2.or.jp/fukushoku/f_disp.php?page_no=0000025 |title=平安初期女官朝服 |accessdate=2017-03-30 |author= |date= |work= |publisher=風俗博物館|language=ja}}</ref>。中期發展出[[國風文化|國風文化]]，包括朝服在內的服飾開始有較大變化，本屬[[常服|常服]]的[[束帶|束帶]]變成男性皇族、官員的朝服<ref name="平安3">{{Cite web |url=http://www.iz2.or.jp/fukushoku/f_disp.php?page_no=0000026 |title=公卿冬束帯 |accessdate=2017-03-30 |author= |date= |work= |publisher=風俗博物館|language=ja}}</ref><ref name="平安4">{{Cite web |url=http://www.iz2.or.jp/fukushoku/f_disp.php?page_no=0000027 |title=公卿夏束帯 |accessdate=2017-03-30 |author= |date= |work= |publisher=風俗博物館|language=ja}}</ref><ref name="平安5">{{Cite web |url=http://www.iz2.or.jp/fukushoku/f_disp.php?page_no=0000028 |title=公家武官夏束帯 |accessdate=2017-03-30 |author= |date= |work= |publisher=風俗博物館|language=ja}}</ref>，女性朝服雖然仍為襦裙制，但樣式較之前變化頗大，稱為[[五衣唐衣裳|五衣唐衣裳]]<ref name="平安6">{{Cite web |url=http://www.iz2.or.jp/fukushoku/f_disp.php?page_no=0000030 |title=公家女房晴れの装い |accessdate=2017-03-30 |author= |date= |work= |publisher=風俗博物館|language=ja}}</ref>。此後大抵沿用，但亦有變化。直至[[明治維新|明治維新]]後，朝服制度全面廢除。

===朝鮮===
[[File:Joseon-Portrait_of_Heungseon_Daewongun-04.jpg|thumb]]朝服像]]
朝鮮的朝服最早見於[[統一新羅|統一新羅]]時期，[[真德女王|真德女王]]三年（649年）引進唐代服飾制度，包括朝服制度<ref name="三國史記"> {{cite book |author=金富軾 |year= |title=三國史記·新羅本紀第五·眞德王 |quote=春正月，始服中朝衣冠。 |publisher=| location= | isbn= |ignore-isbn-error=true| pages= |date=|language=}}</ref>，當時朝服以白色為主，<ref name="東夷"> {{cite book |author= |year= |title=舊唐書·列傳伝149上·東夷·新羅 |quote=風俗刑法與高麗同，而朝服尚白。| publisher=| location= | isbn= |ignore-isbn-error=true| pages= |date=|language=}}</ref>。

[[高麗|高麗]]初期，[[國王|國王]]以[[黃袍|柘黃袍]]為朝服，[[高麗文宗|文宗]]十二年四月改為黃、赭、絳三色袍皆可，至[[高麗毅宗|毅宗]]時期，把國王朝服分為三類，用於不同場合，在較盛大的場合如節日朝賀、[[大觀殿|大觀殿]]大宴、[[儀鳳門|儀鳳門]]宣赦、[[奉恩寺|奉恩寺]]謁祖先[[御真|御真]]、[[八關齋戒|八關齋戒]]、燃燈大會、祭天求豐收、[[王太子|王太子]]納[[太子妃|太子妃]]前醮戒、冊立[[王妃|王妃]]、王太子等穿赭黃袍，燃燈小會則以梔黃衣為朝服<ref name="輿服1"> {{cite book |author= |year= |title=高麗史·志·輿服·冠服 |quote=凡正至節日朝賀, 大觀殿大宴, ·儀鳳門宣赦, 奉恩寺謁祖眞, 八關會·燃燈大會·祈穀圓丘·出宮, 王太子納妃醮戒, 冊王妃·王太子, 臨軒發冊, 服赭黃袍. 燃燈小會, 則服梔黃衣。| publisher=| location= | isbn= |ignore-isbn-error=true| pages= |date=|language=}}</ref>。[[忠烈王|忠烈王]]二十七年五月模仿[[元代|元代]]制度，顏色改為芝黃，但不久又改回赭黃<ref name="輿服2"> {{cite book |author= |year= |title=高麗史·志·輿服·冠服 |quote=忠烈王二十七年五月五月服色擬於上國以芝黃代　袍未幾復用黃袍忠烈王二十七年五月服色擬於上國以芝黃代　袍未幾復用黃袍| publisher=| location= | isbn= |ignore-isbn-error=true| pages= |date=|language=}}</ref>。至[[恭愍王|恭愍王]]十九年五月獲[[明太祖|明太祖]]賜明制朝服一套。首服為七梁遠遊冠，上面有金[[博山|博山]]、蟬七隻以及綴以珠翠，用犀牛角製成的簪固定。體衣包括絳紗袍、紅裳、白紗中單、蔽膝、白假帶（大帶）、方心曲領、紅{{link-ja|銙帶|石帯|革帶}}、金鉤䚢，足衣為白襪黑[[舄|舄]]，恭愍王就以此作為國王朝服<ref name="輿服3"> {{cite book |author= |year= |title=高麗史·志·輿服·冠服 |quote=恭愍王十九年五月,太祖高皇帝賜遠遊冠七梁加金博山附蟬七首上施珠翠犀簪導. 絳紗袍紅裳白紗中單黑領靑緣袖紗蔽膝白假帶方心曲領紅革帶金鉤䚢白襪黑舃受群臣朝賀服之| publisher=| location= | isbn= |ignore-isbn-error=true| pages= |date=|language=}}</ref>。至於百官朝服，雖然毅宗時期規定百官在[[朝鮮新年|正月]]、冬至及每月初一的三大朝賀須穿朝服，但史料缺乏，[[忠烈王|忠烈王]]時期之前的百官朝服形制不明<ref name="张雪芳">{{Cite thesis |degree=碩士 |chapter= |title= 高丽王朝服饰研究|url= |author=张雪芳 |last= |first= |year= 2012|publisher=延邊大學 |accessdate= |docket= |oclc= |language=}}</ref>。恭愍王二十一年十一月改制，五品以下的官員朝服以木笏代替先前的象牙笏，衣服則改用紬色紵布改為先前的紅羅。<ref name="輿服4"> {{cite book |author= |year= |title=高麗史·志·輿服·冠服 |quote=恭愍王二十一年十一月敎: 象笏紅羅朝服皆非本國之産今後侍臣外東西班五品以下用木笏角帶紬紵朝服.| publisher=| location= | isbn= |ignore-isbn-error=true| pages= |date=|language=}}</ref><ref name="张雪芳" />。

[[朝鮮王朝|朝鮮王朝]]初期的朝服度度沿襲高麗制度，國王以遠遊冠、絳紗袍為朝服。此後一直沿用至[[朝鮮高宗|高宗]]稱帝前。高宗稱帝建立[[大韓帝國|大韓帝國]]後，皇帝以通天冠、絳紗袍為朝服<ref name="孙丽娟">{{Cite thesis |degree=碩士 |chapter= |title= 朝鲜王朝时期统治阶层服饰制度探究|url= |author=孙丽娟 |last= |first= |year= 2013|publisher=延邊大學 |accessdate= |docket= |oclc= |language=}}</ref>{{Rp|21—24}}。太宗八年始定[[王世子|王世子]]朝服，為明朝所賜的五梁冠、赤罗衣、白纱中单、赤罗裳、蔽膝、革带、佩綬、白袜、黑履、象笏。至世宗八年（1426年），[[礼曹|礼曹]]根据唐代、宋代制度以及明代賜服，详定世子朝服，其中冠為五梁冠，後獲明代賜六梁冠朝服，此後成為定-{制}-<ref name="孙丽娟" />{{Rp| 28}}。百官朝服則制定於[[朝鲜太宗|太宗]]十六年（1416年），太宗依据明代的《[[洪武礼制|洪武礼制]]》定立百官朝服之制，為梁冠、衣裳、佩綬制<ref name="太宗"> {{cite book |author= |year= |title=朝鮮王朝實錄·太宗實錄·十六年·一月·丙午 |quote=文武百官朝服, 依《洪武礼制》, 造梁冠衣裳佩绶| publisher=| location= | isbn= |ignore-isbn-error=true| pages= |date=|language=}}</ref>。世宗八年（1426年）又再參照唐代、宋代、《洪武礼制》、《[[通典|通典]]》、《[[文献备考|文献备考]]》等更改。至高宗時期把原來的白紗中單改為青紗中單，黑舄改為黑靴<ref name="孙丽娟" />{{Rp| 30—31}}。

===越南===
[[File:19世紀末的越南官員.jpg|thumb]]
越南的朝服制度始見於[[前黎朝|前黎朝]]，據《[[歷朝憲章類誌|歷朝憲章類誌]]·禮儀誌》載，當時的朝服模仿[[宋代|宋代]]制度。梁冠除了有宋制的進賢冠、貂蟬冠、獬豸冠等形制外，還自創了[[拱宸冠|拱宸冠]]、[[揚棠冠|揚棠冠]]，衣裳為朱色<ref name="陳光德"> {{cite book |author=陳光德 |year=2013 |title=NGÀN NĂM ÁO MŨ | publisher=COÂNG TY VAÊN HOÙA & TRUYEÀN THOÂNG NHAÕ NAM | location= | isbn=8936024919733 |ignore-isbn-error=true| pages= |date=2001年9月|language=vi}}</ref>{{Rp|72—75}}。[[陳朝_(越南)|陳朝]]初期朝服沿襲前代，後來因為皇室崇尚[[道教|道教]]，朝服亦仿照[[道士|道士]]服飾，如以羅巾包裹[[髮髻|髮髻]]而不戴冠，或戴[[芙蓉冠|芙蓉冠]]，身穿交領或圓領袍<ref name="陳光德" />{{Rp| 106—110 }}。

[[黎初朝|後黎朝初期]]，據《[[大越史記全書|大越史記全書]]》載，紹平四年的[[天聖節|天聖節]]，[[黎太宗|黎太宗]]穿著[[衮冕|衮冕]]朝服在會英殿視朝，大都督[[黎銀|黎銀]]率領出百官穿著朝服，行進慶下表禮，這是後黎朝皇帝穿衮冕、百官穿朝服之始<ref name="陳光德" />{{Rp| 161 }}，[[紹平|紹平]]四年定立百官朝服制度，樣式與[[公服|公服]]相同，首服為[[幞頭|幞頭]]<ref name="陳光德" />{{Rp| 171—75}}，衣為[[補服|補服]]。至[[景統|景統]]年間又改制<ref name="陳光德" />{{Rp| 第201頁 }}。後來[[莫登庸|莫登庸]]廢黜[[黎恭皇|黎恭皇]]，建立[[莫朝|莫朝]]，但各種制度仍然沿襲後黎朝。後來[[後黎朝|後黎朝]]復辟，是為[[黎中興朝|黎中興朝]]，並重新制定朝服制度，皇帝以[[衝天冠|衝天冠]]、黃袍、玉帶為朝服<ref name="陳光德" />{{Rp| 188 }}，百官仍以幞頭補服為朝服，但稍有改動，如[[皇太子|皇太子]]、[[皇子|皇子]]、[[王子|王子]]、[[國公|國公]]幞頭為[[楊棠冠|楊棠冠]]。[[永壽_(越南)|永壽]]四年又規定未有封爵的皇子、王子以黑涼巾、黑紗衣為朝服，[[保泰|保泰]]二年又改朝服冠為[[獬豸冠_(幞頭)|獬豸冠]]，但這種獬豸冠為幞頭式，與前代的梁冠式獬豸冠形制不同<ref name="陳光德" />{{Rp| 202—211}}。

[[阮朝|阮朝]][[嘉隆帝|嘉隆帝]]制定皇帝朝服為[[衝天冠|衝天冠]]、[[龍袍|黃龍袍]]<ref name="陳光德" />{{Rp| 288—293}}，皇太子朝服為七龍金珠飾[[唐巾_(阮朝)|唐巾]]、大赤色盤龍袍，皇子為[[平頂冠|平頂冠]]、盤領[[蟒袍|蟒袍]]，後改平頂冠為[[金冠_(越服)|金冠]]。百官朝服為幞頭、補服，文官用圓幞頭，武官用方幞頭。後期改為三品以上服蟒袍，四品服蛟袍，五品、六品服藍袍，五品的藍袍加花，稱為花袍<ref name="陳光德" />{{Rp| 307—322}}。
===琉球===
[[File:Dress_of_Oji_and_Aji.jpg|thumb]]
[[琉球|琉球]]的朝服制度始於[[琉球國|琉球國]][[第二尚氏王朝|第二尚氏王朝]]，又稱'''朝衣'''（{{ryu|r=朝衣|kn=チョージン}}），首服為巾或帽、簪，體衣為袍、帶<ref name="趙愛芹">{{cite journal |author=趙愛芹 |date= 2004|title=服飾文化與琉球社會 |url= |journal=哈尔滨学院学报 |volume=26 |issue=7 |pages=|doi= |access-date=}}</ref>。朝服制度的出現不遲於[[尚清王|尚清王]]時代，這時期的官員朝服以巾、帶顏色辮別等級，巾稱為以[[八卷|八卷]]（{{ryu|r=八卷|shin=八巻|kn=ハチマキ}}，又作[[缽卷_(琉裝)|缽卷]]（{{ryu|r=缽卷|shin=鉢巻|kn=ハチマキ}}，是纏頭用的色布，以黃色為最尊貴，然後是紅、青綠，白為最低等，國王則以錦帕纏頭。<ref name="吴永宁">{{Cite thesis |degree=碩士 |chapter= |title= 琉球民俗文化研究|url= |author=吴永宁 |last= |first= |year= 2008|publisher=福建师范大学 |accessdate= |docket= |oclc= |language=}}</ref>。<ref name="陳侃"> {{cite book |author= |year= |title=使琉球錄·群書質異 |quote=但結髻於首之右。凡有職者，簪一金簪；漢人之裔，髻則結於髮之中。俱以色布纏其首，黃者貴、紅者次之、青絲者又次之，白斯下矣。王首亦纏錦帕；衣則大袖寬博，製如道服。然腰束大帶，亦各如纏首之布之色；辨貴賤也。足則無貴賤，皆著草履|author=陳侃| publisher=| location= | isbn= |ignore-isbn-error=true| pages= |date=|language=}}</ref>。到了[[尚豐王|尚豐王]]時代開始規定朝服的體衣顏色，
據《[[琉球國由來記|琉球國由來記]]》載當時尚豐王看到有人穿大綠衣，覺得色彩光鮮、花紋美麗，於是下詔定[[王子|王子]]、[[按司|按司]]的朝服為大綠色，[[親方|親方]]以下朝服為大青色，皆以[[芭蕉布|芭蕉布]]製成<ref name="小林孝子">{{cite journal |author=小林孝子 |date= 1974-03|title=南日本の衣料について-3-芭蕉の朝衣(官服) |url= |journal= 鹿児島大学教育学部研究紀要. 自然科学篇 |volume=25 |issue= |pages= 67—74|doi= |access-date=|language=ja}}</ref>諸[[間切|間切]]之[[掟|掟]]、[[目指|目指]]的朝服為玉色，家來[[赤頭|赤頭]]（百姓出身的吏員）朝服為諸細工，<ref name="衣服">{{cite book |author= |year= |title=琉球國由來記·衣服 |quote=尚豐王世代，或人大緑色衣着，王看之，其色光輝而花美也。故有詔，定王子、按司朝服也。（練蕉布之単衣也）四季用之也。大青朝衣，親方以下朝衣也。玉色朝衣，諸間切掟、目指。家来赤頭，諸細工朝服。| publisher=| location= | isbn= |ignore-isbn-error=true| pages= |date=|language=}}</ref><ref name="小林孝子" />。《[[球陽|球陽]]》記載至[[尚質王|尚質王]]元年（1629年）更設[[芭蕉當職|芭蕉當職]]，專門管理製作朝服的芭蕉布<ref name="球陽"> {{cite book |author= |year= |title=球陽 |quote=尚質王，即位元年始メテ芭蕉当職ヲ置ク，モト芭蕉当卜称ス。後二奉行卜称ス。イマ偽ホ称シテ旧ノ如シ。| publisher=| location= | isbn= |ignore-isbn-error=true| pages= |date=|language=ja}}</ref>。

[[尚貞王|尚貞王]]三十八年頒佈《[[琉球國中山王府官制|琉球國中山王府官制]]》修改了朝服制度，首服由八卷改為冠（帽），與大帶皆以顏色區分等級，簪則以材質區別等級。簪方面一品及正二品用金簪，從二品用金花銀柱簪，三品及以下官員銀簪，不入品的[[吏員|吏員]]用銅簪。體衣方面，一品官員穿綠袍，二品以下官員穿青袍，[[吏員|吏員]]穿藍袍。冠方面，一品為二品為三品至七品戴黃冠，八至九品戴紅冠，吏員戴綠冠。帶方面，一品用錦帶，二、三品用繡龍黃帶，四品用繡龍紅帶，五品至九品用雜色花帶，吏員用青布妝花帶。簪方面，一品、正二品用金簪，從二品用金花銀柱簪，三品至九品以及蔭生用銀簪，吏員用銅簪。[[尚敬王|尚敬王]]時代大抵沿用，但稍有改動，《[[中山傳信錄|中山傳信錄]]》記載當時宮中雜職改穿綠袍、戴紅絹帽，地方吏員如[[里長|里長]]、[[保長|保長]]等仍穿藍袍（{{link-ja|藍染|藍染め}}袍，深藍近似黑色，又稱'''黑朝衣'''（{{ryu|r=黑朝衣|shin=黒朝衣|kn=クルチョージン}}）或'''黑朝'''（{{ryu|r=黑朝|shin=黒朝|kn=クルチョー}}）、戴紅或綠布帽，蔭生、官生改穿八、九品朝服<ref name="中山"> {{cite book |author= |year= |title=中山傳信錄·冠服 |quote=正一品以下，帽八等、簪四等、帶四等。……正、從一品：金簪，彩織緞帽，錦帶，綠色袍。正、從二品：正二品金簪、從二品金花銀柱簪，紫綾帽（有功者，賜彩織緞帽），龍蟠黃帶（有功者，賜錦帶），深青色袍（下至八、九品，朝服皆同）。正、從三品：銀簪，黃綾帽，龍蟠黃帶。正、從四品：簪、帽、袍同三品，龍蟠紅帶。正、從五品：簪、帽、袍同三品，雜色花帶。正、從六七品：簪、袍同三品，黃絹帽，帶同五品。正、從八九品：簪、袍同三品，大紅縐紗帽，帶同五品。雜職：簪、袍同三品，紅絹帽，帶同五品。里、保長：銅簪，藍袍，紅布帽或綠布帽。蔭、官生：簪、帽、服、帶俱同八品。| publisher=| location= | isbn= |ignore-isbn-error=true| pages= |date=|language=}}</ref><ref name="趙愛芹" /><ref name="吴永宁" />。而從《[[續琉球國志略|續琉球國志略]]》記記載所見，此制度至[[尚泰王|尚泰王]]時期並沒有大改變，只是尚泰王時各級官員體衣改為端青袍<ref name="續"> {{cite book |author= |year= |title=續琉球國志略·冠帶簪衣制 |quote=正、從一品：彩織緞帽，錦帶，金簪，端青袍。 正、從二品：紫綾帽，龍蟠黃帶，正二品金簪、從二品金花銀柱簪，端青袍。 正、從三品：黃綾帽，龍蟠黃帶，銀簪，端青袍。正、從四品：黃綾帽，龍蟠紅帶，銀簪，端青袍。正、從五品：黃綾帽，雜色花帶，銀簪，端青袍。正、從六七品：黃紬帽，雜色花帶，銀簪，端青袍。正、從八九品：大紅縐紗帽，雜色衣帶，銀簪，端青袍。百姓：紅布帽、綠布帽，青花布帶，銅簪，藍袍（有功者銀簪，黃紬帽，端青袍）。| publisher=| location= | isbn= |ignore-isbn-error=true| pages= |date=|language=}}</ref>。
