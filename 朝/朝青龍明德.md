'''朝青龍明德'''，本名'''道勒格爾蘇倫·達格瓦道爾吉'''<ref>http://news.xinhuanet.com/sports/2008-03/26/content_7862846.htm</ref>（蒙古名：{{lang|mn|Долгорсүрэн Дагвадорж}}，拉丁轉寫：{{lang|en|Dolgorsuren Dagvadorj}}，{{bd|1980年|9月27日|||Asashouryuu, Akinori}}），[[蒙古国|蒙古国]][[烏蘭巴托|烏蘭巴托]]人，[[日本|日本]][[大相撲|大相撲]]力士。他是第一個成為[[橫綱|橫綱]]的蒙古相撲力士，於2010年2月4日退役。

== 早年經歷 ==
朝青龍於1980年9月27日出生於[[蒙古国|蒙古国]]首都[[烏蘭巴托|烏蘭巴托]]，其家族是蒙古知名的博克（蒙古式摔跤）世家。朝青龍兄弟當中有三人從事相撲運動，其父的相撲成就也在蒙古赫赫有名。朝青龍從就讀[[第54中學校|第54中學校]]時開始接受[[柔道|柔道]]訓練，在接觸柔道的同年內即獲得中學運動會優勝，在其他同項運動賽事當中依舊有水準演出。

15歲時開始練習[[蒙古式摔跤|蒙古式摔跤]]，隔年立刻在[[那達慕|那達慕]]少年組蒙古式摔跤項目中獲取優勝。1997年，[[明德義塾高等學校|明德義塾高等學校]]校長偕同相撲部教練特地來蒙古探尋人才，朝青龍被校方看上後立刻輟學，同年9月與日後的[[朝赤龍太郎|朝赤龍太郎]]一同赴日。朝青龍赴日後獲獎無數，1998年受引薦進入[[高砂部屋|高砂部屋]]。

== 相撲生涯 ==
1999年1月以[[若松部屋|若松部屋]]所屬的相撲力士身份登上[[土俵|土俵]]。2001年即晉昇[[幕內力士|幕內力士]]。2002年晉昇為[[大關_(相撲)|大關]]，成為首位蒙古出身的大關力士，成就超越其偶像[[旭鷲山|旭鷲山]]。2003年1月，[[橫綱審議委員會|橫綱審議委員會]]針對其不當言論和行為作風仔細審核，最終仍一致推舉他晉升為[[橫綱|橫綱]]。

2004年，朝青龍在連續兩場所（一月場所、三月場所）獲得全勝優勝。在同年的五月場所，被[[北勝力英樹|北勝力英樹]]擊敗，連勝紀錄停在35場。後再與北勝力進行優勝決定戰，擊敗北勝力，拿下三場所連勝。同年的九月場所由大關[[魁皇博之|魁皇博之]]獲勝，朝青龍雖無緣拿下全年六場完全優勝，但同年的十一月場所開始至隔年達成了歷史新猷的「七連霸」與年間六場所完全優勝等歷史性記錄。2005年，朝青龍挑戰八連霸失敗。

2007年8月，朝青龍因涉及“裝病風波”而被[[日本相撲協會|日本相撲協會]]勒令停賽兩場所（9月及11月）、減薪，幾天後傳出罹患[[神经衰弱|神经衰弱]]，因而宣布回蒙古休養。日本媒體紛紛以此作為新聞素材，一般看法兩極；《[[讀賣新聞|讀賣新聞]]》指出，日本相撲協會的懲處簡直是封殺了朝青龍。

2010年2月4日，在東京酒後打傷友人，引發醜聞，因而宣布退出相撲界。

2010年10月3日，朝青龍在日本東京正式舉行退休儀式，最高潮就是由日本前首相[[森喜朗|森喜朗]]操刀，剪下一段朝青龍的頭髮，以及朝青龍的師傅、高砂親方，一一為朝青龍落髮。此時，朝青龍和親方都落下了感傷的眼淚，朝青龍在近萬名觀眾前，含淚親吻土俵，象徵他和這個個人生涯高峰的催生地，正式告別。

他回蒙古後從商，並成為[[民主党_(蒙古国)|民主黨]]的成員。他的外甥豐昇龍也是相撲力士。

== 爭議 ==
*2002年朝青龍與因傷休戰許久的[[貴乃花光司|貴乃花光司]]對戰，落敗退場後口出惡言，回休息室時曾表達“可惜自己沒往貴乃花傷處踢一腳”的言論，引發議論。

*2003年與[[旭鷲山|旭鷲山]]對戰時，朝青龍認為旭鷲山起身動作影響自身攻勢，要求審判委員重新判決，並對旭鷲山刻意作出有挑釁意味的動作。此後，朝青龍曾刻意扯下旭鷲山的[[髮髻|髮髻]]並破壞旭鷲山的[[汽車|汽車]]照後鏡，甚至在[[澡堂|澡堂]]與旭鷲山發生肢體衝突。

*朝青龍常以左手手刀領取懸賞金。

*朝青龍涉嫌裝病未參與練習訓練，曾受到日本相撲協會多次警告。

*朝青龍練習時，要求曾在正式土俵上擊敗他的力士作為他的練習對象。

*2007年，朝青龍以“腰部過勞[[骨折|骨折]]”為由請假退出[[夏季巡業|夏季巡業]]，卻在7月25日於蒙古與前[[日本國家足球隊|日本國家足球隊]]明星球員[[中田英壽|中田英壽]]參加[[足球|足球]]義賽，遭媒體披露。日本相撲協會理事[[伊勢之海|伊勢之海]]對此作出嚴詞批評，理事長[[北之湖一|北之湖一]]遂於8月1日召開臨時理事會，宣布朝青龍停賽四個月、減薪30%等嚴厲處罰。
