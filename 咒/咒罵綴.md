'''咒罵綴'''（Expletive infixation）是在詞語中插入的[[虛詞|虛詞]]（[[詞綴|詞綴]]），以加強[[髒話|髒話]]的語氣。

*unbelievable（難以置信） → un'''-fucking-'''believable（語感：真他媽的不敢相信）

== 概述 ==
插入的虛詞多為形容詞。
*[[分詞|分詞]]（[[fuck|fucking]]、mother-fucking, freaking, blooming, bleeding, damned）
*[[形容詞|形容詞]]（bloody）

[[發音|發音]]可能變化。（fucking → fuckin'）

大部分說話者在幼兒期不會使用到咒罵綴，然而一旦開始使用，就會很容易創造出新詞語，變得容許咒罵綴<ref name="McCarthy">McCarthy, John J. (1982). "Prosodic Structure and Expletive Infixation" (GIF). ''Language'' '''58''' (3): 574–590. doi:10.2307/413849. Retrieved on 2006-08-30.</ref>。

== 漢語 ==
漢語中的情況一般是置於句首，部分仍有咒罵意味，而部分則偏向[[感嘆詞|感嘆詞]]。例如
*肏超噁心的：有咒罵意味
*幹這句超棒：加強語氣或發語習慣<ref>{{cite web |language= zh |title= 罵「幹」判無罪 法官：是生活用語 |url= http://iservice.libertytimes.com.tw/liveNews/news.php?no=931302&type=%E7%A4%BE%E6%9C%83 |author=  |work= 自由電子報 |publisher= [[自由時報|自由時報]] |date= 2014-01-06 |accessdate= 2014-01-07 |deadurl= yes |archiveurl= https://web.archive.org/web/20140107135747/http://iservice.libertytimes.com.tw/liveNews/news.php?type=%E7%A4%BE%E6%9C%83&no=931302 |archivedate= 2014-01-07 }}</ref>
但也有例外的，例如：在粵語中就經常使用[[中綴|中綴]]插在動詞或形容詞中。在[[TVB|TVB]]的電視節目《[[男人食堂|男人食堂]]》中的〈[[男人食堂#粗口學堂|粗口學堂]]〉環節，Ben Sir甚至會用一下拍手來替代這些「咒罵綴」
