'''金属茂基配合物'''是一类含有一个或多个[[环戊二烯|环戊二烯]]基团（“[[茂基|茂基]]”，C<sub>5</sub>H<sub>5</sub><sup><nowiki>−</nowiki></sup>，简写为Cp<sup><nowiki>−</nowiki></sup>）的化合物。根据金属原子和环戊二烯基团之间键接方式的不同，金属茂基配合物可以分为三类：π配合物，σ配合物和离子配合物。

==茂基配体==
[[碱金属|碱金属]]茂基配合物会与多种[[过渡金属|过渡金属]]化合物反应，生成一系列新的配合物。

==五甲基环戊二烯基和茂基的比较==
五甲基環戊二烯基配体（Cp*）是有机金属化合物中的一种重要配体。

==五甲基环戊二烯基配合物的合成==
能够合成五甲基环戊二烯基金属配合物的代表性反应有：
::Cp*H  + C<sub>4</sub>H<sub>9</sub>Li  →   Cp*Li  +  C<sub>4</sub>H<sub>10</sub>

::2 Cp*Li + TiCl<sub>4</sub>  →   Cp*<sub>2</sub>TiCl<sub>2</sub>  + 2 LiCl

::Cp*<sub>2</sub>TiCl<sub>2</sub> + TiCl<sub>4</sub>  →  2 Cp*TiCl<sub>3</sub>

::Cp*Li + Me<sub>3</sub>SiCl  →  Cp*SiMe<sub>3</sub> + LiCl

::Cp*SiMe<sub>3</sub> + TiCl<sub>4</sub> →  Cp*TiCl<sub>3</sub>  + Me<sub>3</sub>SiCl

::2 Cp*H  +  2 Fe(CO)<sub>5</sub>   →   [Cp*Fe(CO)<sub>2</sub>]<sub>2</sub>  +  H<sub>2</sub>

某些五甲基环戊二烯基配合物是用六甲基[[杜瓦苯|杜瓦苯]]作为前体制备的，此法曾被用于合成[Rh(C<sub>5</sub>Me<sub>5</sub>)Cl<sub>2</sub>]<sub>2</sub>。
