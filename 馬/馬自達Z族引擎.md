'''馬自達Z族引擎'''是由[[日本|日本]][[馬自達|馬自達]]汽車公司研發、生產的[[往復式活塞引擎|往復式活塞]][[直列四缸引擎|直列四缸]][[汽油引擎|汽油引擎]]，其[[排氣量|排氣量]]分別由1.3L至1.6L不等，屬於舊款鑄鐵缸體的[[馬自達B族引擎|B族引擎]]的改良版本。

== 概要 ==
Z族引擎具有由[[頂置凸輪軸#雙頂置凸輪軸（DOHC）|雙頂置凸輪軸]]控制的16氣門，中間連結著正時鏈條。[[汽缸蓋|汽缸蓋]]與[[汽缸本體|汽缸本體]]皆以[[鋁合金|鋁合金]]製成，汽缸套則為鑄鐵製。為了提升[[剛度|剛度]]與[[強度|強度]]，採用上下分割缸體組合而成；特別加長的進氣歧管可以增加[[扭矩|扭矩]]，另採用[[S-VT可變氣門正時|S-VT可變氣門正時]]技術與不銹鋼排氣頭。

2010年10月20日馬自達宣布自隔年起不再發展此系列引擎，改用油耗表現更佳的[[創馳藍天技術|創馳藍天]]引擎取代<ref>參看{{en}}[http://www.mazda.com/publicity/release/2010/201010/101020a.html 【MAZDA】Mazda Reveals Next-Generation 'SKYACTIV' Technologies]。</ref>。不過，福特仍繼續生產Z與L族引擎，以供給其Duratec和[[EcoBoost|EcoBoost]]直列四缸引擎。

== Z5型 ==
所有Z族引擎均由此型衍生出來，排氣量為1,489c.c.的'''Z5-DE型'''，其缸徑和[[衝程|衝程]]分別為75.3mm和83.6mm。缸體、缸體內部和油底殼雖然近似[[馬自達B族引擎|B族引擎]]，不過[[頂置凸輪軸#雙頂置凸輪軸（DOHC）|DOHC]]汽缸頭可是全新的設計。油底殼由上下兩片組成，上片為鋁製，下片為沖壓鋼鐵製。車型：
# 1994年-1998年：[[馬自達323#第八代（1994年-1998年）|第八代323]]
# 1993年-1997年：[[馬自達Lantis|Lantis]]
# 1994年-1998年：[[馬自達323#Familia_Neo_/_323c_/_Laser_Lynx_/_Aztec|Familia Neo]]

== ZJ型 ==
[[File:Mazda_ZJ-VEM_engine.jpg|thumb]]
排氣量1,348c.c.的'''ZJ-VE型'''引擎，缸徑74mm、衝程78.4mm，具備電子節氣門控制、[[S-VT可變氣門正時|S-VT可變氣門正時]]和TSCV渦流控制閥（{{en}}tumble swirl control valve）等技術，區分成高、低馬力兩種版本：
* 高馬力版：最大馬力86ps / 6,000rpm，最大扭力122N·m / 3,500rpm。
* 低馬力版：最大馬力75ps / 6,000rpm，最大扭力121N·m / 3,500rpm。
另有'''ZJ-VEM型'''[[米勒循環引擎|米勒循環]]引擎，油耗表現更佳。車型：
# 2007年至今：[[馬自達2#第三代（2007年迄今）|第三代馬自達2]]
# 2009年迄今：[[福特Fiesta#第六代（2008年迄今）|第六代福特嘉年華（中國）]]

== ZL型 ==
此系列引擎的排氣量為1,498c.c.，缸徑78mm，衝程78.4mm。此外，該型引擎於1998年時最先搭配新開發的[[S-VT可變氣門正時|S-VT可變氣門正時]]技術，稱作'''ZL-VE型'''。按機械結構之不同區分成：
* '''ZL-DE型'''：除了進氣歧管與[[曲軸|曲軸]]行程外，其餘元件與Z6型 / ZM型相同<ref>參考[http://wanghan0201.wordpress.com/2011/03/22/%E8%80%81%E6%98%AF%E8%A2%AB%E5%95%8F%E5%8D%80%E5%88%A5mazda-zl-ve%E8%B7%9Fzl-de%E7%9A%84%E5%95%8F%E9%A1%8C/ 老是被問區別MAZDA ZL-VE跟ZL-DE的問題]。</ref>。
* '''ZL-VE型'''：因為配有[[S-VT可變氣門正時|S-VT可變氣門正時]]技術與高壓縮比（9.4:1）設計，該型引擎的馬力輸出大於排氣量較大的1.6L ZM-DE型引擎。

車型：
# 1998年-2003年：[[馬自達323#第九代（1998年-2003年）|第九代323]]

== ZY型 ==
[[File:Mazda_ZY-VE_Engine.JPG|thumb]]
'''ZY-VE型'''的排氣量為1,498c.c.，缸徑78mm，衝程78.4mm，最大馬力111hp，具備電子節氣門控制、[[S-VT可變氣門正時|S-VT可變氣門正時]]、TSCV渦流控制閥（{{en}}tumble swirl control valve）和VIS可變慣性進氣歧管系统（{{en}}variable induction system）等技術。車型：
# 2002年-2007年：[[馬自達2#第二代（2002年－2007年）|第二代馬自達2]]
# 2007年至今：[[馬自達2#第三代（2007年迄今）|第三代馬自達2]]
# 2004年至今：[[馬自達Verisa#第一代（DY系_2004年迄今）|第一代Verisa]]
# 2003年-2009年：[[馬自達3#第一代（2003年-2009年）|第一代馬自達3]]
# 2009年迄今：[[福特Fiesta#第六代（2008年迄今）|第六代福特嘉年華（中國）]]

== Z6型 / ZM型 ==
採用DOHC及16氣門設計、排氣量1,598c.c.的'''Z6型'''引擎之缸徑78.0mm、衝程83.6mm，乃改良自具有環型進、排氣道的Z5型。後來衍生出幾個亞型：
* '''ZM型'''：使用於[[亞洲|亞洲]]與[[歐洲|歐洲]]市場，壓縮比9.7:1，最大馬力97hp / 5,500rpm，最大扭力 145N·m / 3,700rpm。
* '''ZM-DE型'''：使用於[[北美洲|北美洲]]市場，壓縮比9.0:1<ref>但是供應南非的ZM-DE型引擎壓縮比則為9.4:1。</ref>，最大馬力105hp / 5,500rpm，峰值扭力145N·m / 4,000rpm。此型引擎與前述的ZL-VE型共用某些主要元件，可謂系出同源；但是兩者的[[曲軸|曲軸]]不同。車型：
# 1998年-2003年：[[馬自達323#第九代（1998年-2003年）|第九代323]]
# 2002年-2007年：[[馬自達2#第二代（2002年－2007年）|第二代馬自達2]]
* '''Z6-VE型'''：搭配[[S-VT可變氣門正時|S-VT可變氣門正時]]、VIS可變進氣系統（Variable Intake System之縮寫）與TSCV渦流控制閥（Tumble Swirl Control Valve之縮寫）等技術，壓縮比10.0：1，最大馬力106ps / 6,000rpm，最大扭力15.0kg·m / 4,000rpm。車型：  
# 2003年-2009年：[[馬自達3#第一代（2003年-2009年）|第一代馬自達3]]
# 2009年迄今：[[馬自達3#第二代（2009年-2013年）|第二代馬自達3]]
