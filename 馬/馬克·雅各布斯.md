'''馬克·雅各布斯'''（{{lang-en|'''Marc Jacobs'''}}，{{bd|1963年|4月9日|||catIdx=Jacobs, Marc}}）生於[[紐約|紐約]]<ref name="Bio">{{cite web |url=http://www.marcjacobs.com/#folder=/worldofmarcjacobs/biography |title=Biography |accessdate=2009-03-27 |work=''Marc Jacobs'' |publisher= |date= }}</ref>，是位[[美國|美國]]時尚設計師。馬克是同名品牌{{lang|en|Marc Jacobs}}以及副牌{{lang|en|Marc by Marc Jacobs}}的領銜設計師<ref name="vogue">{{cite web |url=http://www.vogue.co.uk/biographies/080420-marc-jacobs-biography.aspx |title=Marc Jacobs biography |accessdate=2009-03-27 |work= |publisher=''Vogue'' |date= }}</ref>。

2013年10月2日LVMH集團主席Bernard Arnault證實與LV合作長達十六年的Marc Jacobs10月約滿後將離任，專注發展其個人品牌。

== 生平 ==
雅各布斯的祖母在他幼時教導他如何縫製衣物，並成了影響雅各斯一生最大的人。15歲時，雅各布斯到路易威登在紐約的前衛精品店「胡鬧」擔任採購員，並認識了[[佩瑞·艾里斯|佩瑞·艾里斯]]，雅各布斯曾表示艾里斯「讓我覺得很酷、給了我很多希望<ref name="vogue" />。」1981年，雅各布斯於紐約[[藝術與設計高級中學|藝術與設計高級中學]]畢業、前往[[帕森設計學院|帕森設計學院]]就讀，並於1984年獲得佩瑞·艾里斯金環獎、切斯特溫伯格金環獎、帕森設計學院的年度設計學生等最高榮譽<ref name="Bio" /><ref name="vogue" />。

Marc Jacobs不時亦會穿上Prada、Balenciaga、Chanel的產品，亦曾去Chanel展時拍短片指出自己夢想成真<ref>[https://www.mings-fashion.com/marc-jacobs-mingsfashion-mj-3250/ 最識英雄重英雄的設計師：Marc Jacobs高調曬著Prada、Balenciaga、Chanel]</ref>。

2018年，與Marc Jacobs是設計學院同學的Anna Sui宣布推出限定系列，這次亦是他們多年以來的第一次合作。系列包括鎖匙扣、T-shirt和波鞋等等<ref>[https://www.mings-fashion.com/marc-jacobs-anna-sui-will-broome-1796/ 時裝界死黨：Marc Jacobs與Anna Sui聯手設計系列]</ref>。

== 私人生活 ==
雅各布斯是公開的[[同性戀者|同性戀者]]，更於2007年被[[OUT雜誌|OUT雜誌]]評價為「美國五十大同性戀男女」<ref name="OUT">{{cite web |author=Oxfield, Jesse, Idov, Michael |url=http://nymag.com/daily/intel/2007/04/out_ranks_the_top_50_gays_ande.html |title=''"Out" Ranks the Top 50; Anderson Is No. 2'' |accessdate=2009-03-27 |work= |publisher=紐約雜誌 |date= }}</ref>。

同年3月12日，雅各布斯因毒品及酗酒而接受治療<ref name="USAToday">{{cite web |url=http://www.usatoday.com/life/people/2007-03-13-jacobs-rehab_N.htm |title=Designer Marc Jacobs enters rehab |accessdate=2009-03-27 |work= |publisher=USAToday.com. |date= }}</ref>。

曾與[[巴西|巴西]]廣告經理羅倫佐·馬爾同（Lorenzo Martone）交往。2009年3月，[[女裝日報|女裝日報]]報導兩人將在交往一年後訂婚<ref>{{cite web |url=http://stylenews.peoplestylewatch.com/2009/03/18/marc-jacobs-gets-engaged/ |title=Marc Jacobs Gets Engaged! |accessdate=2009-03-27 |work= |publisher=People.com |date=2009-03-18 |deadurl=yes |archiveurl=https://web.archive.org/web/20090322020242/http://stylenews.peoplestylewatch.com/2009/03/18/marc-jacobs-gets-engaged/ |archivedate=2009-03-22 }}</ref>，可惜婚姻只維持了4個月。

2018年，雅各布斯與交往中的Charly DeFrancesco到快餐廳進餐時向Charly求婚，事後於社交網站分享喜訊<ref>[https://www.mings-fashion.com/marc-jacobs-%E7%B5%90%E5%A9%9A-charly-defrancesco-527/ Marc Jacobs 單膝跪地伴隨快閃舞蹈員向 Charly DeFrancesco 求婚]</ref>。一星期後，Charly DeFrancesco為了慶祝雅各布斯生日，為他準備了一個驚喜生日派對<ref>[https://www.mings-fashion.com/marc-jacobs-%E7%94%9F%E6%97%A5-charly-defrancesco-466/ Surprise！Fiancé 為 Marc Jacobs 辦驚喜生日派對]</ref>。

== 時裝風格 ==
MARC BY MARC JACOBS主要走青春時尚路線，深受時下青少年歡迎。

另外，MARC BY MARC JACOBS在香港亦設有特賣場<ref>[http://hk.ulifestyle.com.hk/spot/detail.html?id=ABdGDVo3BwpZEQNv MARC BY MARC JACOBS  HK.ULifestyle.com.hk]</ref>。

而品牌在紐約時裝周的地位十分高，曾多次為紐約時裝周作壓軸品牌<ref>[https://www.mings-fashion.com/%E6%98%A5%E5%A4%8F-marc-jacobs-%E6%99%82%E8%A3%9D%E5%91%A8-206375/ 紐約時裝周話題大騷：不得不看 Marc Jacobs 的四個原因]</ref>。
