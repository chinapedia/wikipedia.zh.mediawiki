'''馬亞爾II'''（'''[[仙女座星系|M31]]的[[衛星星系|G1]]'''），也稱為'''NGC-224-G1'''、'''SKHB 1'''、'''GSC 2788:2139'''、'''HBK 0-1'''、'''M31GC J003247+393440'''或'''仙女的星團'''，是一個繞著M31，也就是[[仙女座星系|仙女座星系]]的[[球狀星團|球狀星團]]。

它的位置在距離仙女座星系核心130,000[[光年|光年]]（40Kpc）<ref name ="Maetal2007" />，並且是[[本星系群|本星系群]]中[[絕對星等|絕對星等]]最亮<ref name ="Maetal2007" />的球狀星團，[[視星等|視星等]]為13.7等。G 1的質量被認為是[[半人馬座ω|半人馬座ω]]的兩倍，並且在核心可能有中介質量的黑洞（∼ 2{{e|4}} [[太陽質量|M<sub>⊙</sub>]]）<ref name ="Maetal2007" />。

它是由尼古拉斯馬亞爾（Nicholas Mayall）和O.J. Eggen在1953年首度從[[帕洛馬山天文台|帕洛馬山天文台]]48吋施密特望遠鏡在1948年拍攝的乾版上檢出的。

由於恆星[[金屬量|金屬量]]的分布很廣，包含了數個世代的[[恆星|恆星]]和許多的恆星創生期間，顯示經歷了數個世代的[[恆星|恆星]]創生，所以許多人主張他不是真正的球狀星團，許多人認為他不是個真正的球狀星團，而是被仙女座星系吞噬的[[矮星系|矮星系]]殘留下的星系核心殘骸<ref name ="Maetal2007" /><ref name="Meylanetal2001">{{cite journal

== 名稱的來源 ==
* 1991年，馬亞爾II因[[約翰·修茲勞|約翰·修茲勞]]、J.P. Brodie、S.M. Kent被稱為HBK 0-1.
* 1953年，[[尼古拉斯·U·馬亞爾|尼古拉斯·U·馬亞爾]]和O.J. Eggen發現馬亞爾II。
* 1977年，馬亞爾II因[[华莱士·萨金特|华莱士·萨金特]]、C.T. Kowal、F.D.A. Hartwick和Sidney van den Bergh被稱為SKHB 1，也稱之為G1。
