東洋工業（[[馬自達|馬自達]]汽車公司前身）於1960年代起自三輪商用車跨入四輪乘用[[轎車|轎車]]領域，一開始採用的是'''V型雙缸引擎'''。雖然這種氣冷式的[[發動機|引擎]]被使用了幾年的光景，但該公司旋即改採[[直列四缸|直列四缸]]OHV引擎。

== BA型 ==
東洋工業跨入四輪乘用轎車領域的里程碑為[[馬自達R360_Coupe|R360 Coupe]]，該款輕型轎車搭載一具排氣量356c.c.、氣冷式、夾角90°的V型雙缸引擎，缸徑60mm、衝程63mm，壓縮比8.0:1。採用OHV頂置四氣門推桿式設計，最大[[馬力|馬力]]16ps / 5,300rpm、最大扭力2.2kgm / 4,000rpm<ref>參考{{ja}}[http://www.asahi-net.or.jp/~rf7k-inue/izen/no-1/coop/c-siyo.html マツダ・R360クーぺ詳しい仕様書]。</ref>。車型：
# 1960年-1969年：[[馬自達R360_Coupe|馬自達R360 Coupe]]
# 1961年-1968年：[[馬自達B360|馬自達B360]]
# 1959年-1969年：[[馬自達K360|馬自達K360]]

== EA型 ==
此具'''EA型'''引擎乃是自356c.c.擴缸而來，採空冷式、四行程設計。最大馬力20ps / 4,300rpm，最大扭力3.8kg・m / 3,000rpm。車型：
# 1959年-1971年：[[馬自達T600|馬自達T600]]
# 1961年-1968年：[[馬自達B360|馬自達B600]]

== MB型 ==
排氣量1,105c.c.的'''MB型'''引擎，採空冷式設計，最大馬力32.5ps。車型：
# 1958年-1962年：[[馬自達Romper|馬自達Romper]]

== CT型 ==
排氣量1,157c.c.、氣冷式V型二缸'''CT型'''引擎之最大馬力32ps，它是日本汽車界首創採用頂置氣門技術的引擎，且配有油壓式氣門間隙自動調整裝置，有效抑制引擎震動與噪音。車型：
# 1950年9月：[[馬自達MAZDA號|MAZDA號CT型三輪卡車]]
