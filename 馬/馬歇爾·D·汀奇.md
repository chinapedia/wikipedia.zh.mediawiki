'''馬歇爾·D·汀奇'''是[[日本動漫|日本動漫]]作品《[[ONE_PIECE|ONE PIECE]]》中的人物<ref name=comic_people01>{{Cite web |url=http://comic.people.com.cn/GB/238131/238181/16915082.html |title=原王下七武海 四皇之一 马歇尔·D·汀奇 |date=2012年1月18日 |work=人民网动漫频道}}</ref>。黑鬍子海賊團提督、前任「[[王下七武海|王下七武海]]」之一。在第二部已擁有「[[四皇|四皇]]」稱號。

== 設計 ==
「黑鬍子」與其名汀奇（{{lang|en|Teach}}），皆是來自在第18世紀的[[加勒比海|加勒比海]]活躍的海盜[[愛德華·蒂奇|愛德華·蒂奇]]（{{lang|en|Edward Teach}}）<ref>{{cite news|title=海盜船殘骸 揭「黑鬍子」傳奇|newspaper=東方日報|date=2011-05-30}}</ref>。作者[[尾田榮一郎|尾田榮一郎]]曾透露設計「黑鬍子」的參考原型正是[[愛德華·蒂奇|愛德華·蒂奇]]<ref>《ONE PIECE》漫畫第49卷SBS詢問專欄。</ref>。

== 人物 ==
皮膚和頭髮顏色都是黑色，嘴裡有掉幾顆牙齒，是個粗獷的中年男子，獨特笑聲是「賊哈哈哈哈」<ref name="ONE PIECE動畫">《ONE PIECE》動畫第146集</ref>。

汀奇擁有追尋夢想的心，卻是個為了實現自己夢想，可以不擇手段的野心人物。相當有耐心的隱藏自己實力與野心，但只要被他抓到了機會就是個讓世界懼怕的存在。

童年時代的汀奇於漫畫單行本第63集SBS專欄登場，當時的他有著厚嘴唇，似乎因為滿腹委屈而哭泣著<small><ref>《ONE PIECE》漫畫單行本第63集SBS專欄</ref></small>。

== 劇中表現 ==
=== 第一部 ===
*隸屬白鬍子海賊團
:汀奇為了找到夢寐以求的[[惡魔果實|惡魔果實]]，不惜隱藏自己的實力加入「白鬍子」[[艾德華·紐蓋特|艾德華·紐蓋特]]成立的[[白鬍子海賊團|白鬍子海賊團]]，成為白鬍子海賊團的第二隊隊員。並在這段期間熟記每種惡魔果實的模樣。在前傳第零話當中短暫出現過，顯示他20幾年前就已經是白鬍子海賊團的成員。

:當「火拳」艾斯被推舉第二隊隊長時，艾斯向資歷較長的汀奇詢問他的看法。之後汀奇看到好友薩吉（第四隊隊長）獲得他一直夢寐以求的惡魔果實暗暗果實，為了要得到這顆惡魔果實而殺了薩吉，以致於犯下白鬍子海賊團中最惡劣的罪行殺害同伴，就此逃離白鬍子海賊團；而身為其直屬隊長的艾斯聽不進船員和白鬍子的勸告獨自追殺汀奇，間接促成未來無法收拾的局面<ref name="ONE PIECE動畫1">《ONE PIECE》動畫第235集</ref>。

*逃亡期間
:政府懸賞金是零的汀奇，為了進行下一步計畫迅速的先後跟四位成員組織自己的海賊團；分別為毒Q、范·歐葛、吉札士·伯吉斯和拉夫特<ref name="ONE PIECE動畫2">《ONE PIECE》動畫第151集。</ref>。到達磁鼓島時，汀奇利用自己的惡魔果實能力，與夥伴趕走了專制的瓦爾波，令磁鼓王國的護衛隊隊長多魯頓（後來為櫻花王國的開國君主）為此感到羞恥<ref>《ONE PIECE》動畫第80集</ref>。

:三個月後拉菲特親自跑去聖地·馬力喬亞，向正在召開會議的海軍本部將領和[[喬拉可爾·密佛格|「鷹眼」]]等人推薦讓汀奇取代[[克洛克達爾|克洛克達爾]]被捕後在「[[王下七武海|王下七武海]]」空缺的位置，汀奇本人則來到了加亞島的魔谷鎮休息等待回應<ref name="ONE PIECE動畫" />。

:在魔谷鎮的某一家酒吧裡面曾經跟[[魯夫|魯夫]]為了櫻桃派、汽水的事而吵架一陣子。不過在魯夫將要前往空島時才發現魯夫是一位價值1億貝里的懸賞犯，決定不等待拉夫特的回來，開始跟其他夥伴一起離開加亞<ref name="ONE PIECE動畫2" />。由於沖天海流的發生，使汀奇無法追捕到魯夫，不過他沒有因此而放棄這個念頭<ref>《ONE PIECE》動畫第152集</ref>。

*大事件的導火線
:汀奇到了巴納洛島大鬧後，和成員休息一陣子，後因[[草帽海賊團|草帽海賊團]]大鬧司法島事件被登上報紙，欲率領成員前往[[水之七島|水之七島]]追捕魯夫，同時遇到了正在追捕黑鬍子的艾斯。原本汀奇很開心地邀請自己以前的隊長艾斯加入，但艾斯正是因為黑鬍子殺掉船上的夥伴又逃出白鬍子海賊團而追捕他，再加上汀奇還表示他們準備要追殺魯夫，因此激怒了艾斯，並向汀奇展開攻擊。

:最後汀奇用黑暗果實的能力將艾斯擊敗，並將艾斯直接交給[[世界政府_(ONE_PIECE)|世界政府]]送到大監獄「[[推進城|推進城]]」<ref name="ONE PIECE動畫1" />，以此正式成為「王下七武海」的一份子，也不需要再追捕魯夫。曾在世界政府總部與其他七武海成員一起商討如何對付白鬍子<ref>《ONE PIECE》漫畫第49集</ref>。

*頂點戰爭事件
:後來汀奇帶領部下搶了一艘軍艦，闖入深海監獄推進城，和為了拯救艾斯闖入的魯夫再次碰面，魯夫得知汀奇就是將艾斯送到推進城的元兇後，馬上對其出手但反遭汀奇的「闇水」摔在地上，並在「海俠」[[吉貝爾|吉貝爾]]的制止下沒有再戰。之後遭推進城監獄長麥哲倫以「毒龍」擊敗，但被典獄長「雨」之矢龍救活。

:在推進城中放走了Level 6的囚犯，並拉攏了傳說中的4名囚犯為新成員，後來跟「雨」之史龍和傳說中的4名囚犯來到海軍本部，被虛弱的白鬍子打傷，而偷襲白鬍子，甚至在白鬍子死後偷取了其震動果實的能力，進而成為唯一一位擁有兩種果實能力的惡魔果實能力者。原因就在於白鬍子海賊團第一隊隊長「不死鳥」馬可表示，他的身體構造跟常人不同、根本是異形所導致。接著還昭告全世界：「從今以後就是我的時代了！」

*頂點戰爭過後
:「頂點戰爭」結束後，擄獲了「11超新星」之一的「大胃王」珠寶·波妮並率領海賊團到達了「新世界」某個燃燒中的島嶼。原本想將擄獲的波妮交與海軍以換得一艘軍艦，但因為海軍並沒有交易的意圖，加上不想和前來緝捕的上將「赤犬」盃起衝突而作罷<ref name="ONEPIECE漫畫_595">《ONE PIECE》漫畫第595話。</ref>。

=== 第二部 ===
在頂上戰爭結束後一年，黑鬍子率領其團員對白鬍子海賊團的餘黨發動戰爭，戰爭的結果以白鬍子海賊團餘黨的慘敗收場，黑鬍子獲得壓倒性勝利。汀奇更進一步稱霸原本屬於「白鬍子」的海域，被世人並列為新的四皇之一<ref name="ONEPIECE漫畫_650">《ONE PIECE》漫畫第650話。</ref>。根據吉貝爾表示，汀奇甚至在進行獵殺能力者的行動。事後汀奇閱報得知魯夫等人因為在圓蛋糕島策畫暗殺「BIG MOM」未遂（實為火戰車海賊團策畫暗殺計畫，魯夫等人和火戰車海賊團與太陽海賊團合作解救賓什莫克家族）的消息，認為魯夫要被稱做「皇帝」還早的很。

在新世界海賊樂園「蜂巢」閱報得知世界會議上革命軍與海軍上將發生衝突與在和之國出現以魯夫為首引發騷亂的消息，並告知月光·莫利亞強者之間的廝殺已經開始。

== 能力 ==
=== 惡魔果實 ===
黑鬍子爲自然系「[[惡魔果實|暗暗果實]]」能力者，惡魔果實歷史上最凶惡的能力，擁有把一切都吸進去的引力，連任何的光線都不會放過。能將與黑暗接觸的事物吸入、壓縮及粉碎，也能準確吸引惡魔果實能力者的實體，甚至使其碰觸到實體的惡魔果實能力者無法使用任何能力。缺點是由於黑暗的本質就是引力，因此無法用來躲避物理攻擊，而且受到的痛楚比一般人還強烈。

在「頂點戰爭」中，汀奇從戰死的[[艾德華·紐蓋特|「白鬍子」艾德華·紐蓋特]]身上奪走「[[惡魔果實|震動果實]]」的能力，成為目前唯一一個同時擁有兩種惡魔果實能力的人。但奪取方法尚未明朗。

== 人物關係 == 
=== 部下 ===
* 黑鬍子海賊團
{{main|ONE PIECE海賊列表#黑鬍子海賊團}}
: 由馬歇爾·D·汀奇所率領的海賊團，擁有恐怖的戰鬥力。兩年前的海賊船是巨大[[木筏|木筏]]，曾經毀滅位於偉大航路的磁鼓王國及襲擊大監獄[[推進城|推進城]]，「頂點戰爭」時網羅數位LEVEL 6的最強囚犯以及取得震震果實能力。一年後與「不死鳥」馬可率領的白鬍子海賊團餘黨發生了一次大規模衝突，世間稱為「了斷之戰」，最後馬可那方慘敗，隊長們失去音訊，白鬍子海賊團就此成了傳說，而「黑鬍子」藉此稱霸原本屬於白鬍子海賊團的地盤躋身至四皇之位。兩年後「黑鬍子」號稱提督，在兩年前集結的九名船員都成了幹部級人物，可見黑鬍子海賊團的勢力及人數不斷壯大。
