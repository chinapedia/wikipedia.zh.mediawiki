'''馬堅'''{{BD|1906年|6月6日|1978年|8月16日|catIdx=Ma马}}原名'''马玉书'''、'''马万清'''，[[表字|字]]'''子實'''，[[號|號]]'''自適'''，[[經名|經名]]'''穆罕默德·麥肯'''，[[回族|回族]]，[[云南|云南]][[箇舊|箇舊]]人，中国[[穆斯林|穆斯林]]學者與翻譯家，[[阿拉伯语|阿拉伯语]][[语言学家|语言学家]]，[[伊斯兰|伊斯兰]]学术文化研究家，通晓[[汉语|汉语]]、[[阿拉伯语|阿拉伯语]]，兼通[[波斯语|波斯语]]、[[英语|英语]]<ref name=zgzj>马开能，学富五车皓首译经——记马坚教授，中国宗教2004(10):20-22</ref>。

== 生平 ==
1906年6月，马坚生于[[云南省|云南省]][[个旧市|个旧县]][[沙甸镇|沙甸镇]]一个普通回族农民家庭，原名马玉书，字子实，经名穆罕默德·麦肯。马坚的父亲马忠吉是信仰虔诚的[[穆斯林|穆斯林]]，曾在1920年代中期到[[麦加朝觐|麦加朝觐]]。马坚自幼受到浓厚的[[伊斯兰教|伊斯兰教]]文化熏陶，7岁时便跟随马明福[[阿訇|阿訇]]学习，后又改投王富云阿訇接受《[[古兰经|古兰经]]》启蒙，学习[[阿拉伯文|阿拉伯文]]，一年后便能熟读《古兰经》并背诵若干章节。此后马坚到沙甸绅士白亮诚创办的[[私塾|私塾]]接受新式教育。白亮诚建议马忠吉让马坚到省城[[昆明|昆明]]深造，并愿意资助。马忠吉想让儿子当阿訇，所以起初并不赞同，后经白亮诚多次劝说才同意。1918年，马坚到昆明进入中阿并授的成德中学学习6年<ref name=zgzj/>。

中学毕业后，1926年3月马坚回到沙甸，在沙甸鱼峰小学任校务主任。其间，他将原来的初等小学改为初等和高等两级小学，采用新课本，增加作文、算术、体育、唱歌、图画等课程，并兼授阿拉伯语，使鱼峰小学成为“中阿并授”的完全小学。他说：“我相信，回族所需要的是教育，中国所需要的也是教育，惟教育可以救国，惟教育可以振教。” 1928年3月，马坚为去[[宁夏|宁夏]][[固原县|固原县]]三营大寺教长虎嵩山阿訇处学习[[波斯文|波斯文]]，和人结伴自沙甸出发，经[[蒙自|蒙自]]乘火车沿[[滇越铁路|滇越铁路]]出[[河口瑶族自治县|河口]]，从[[越南|越南]][[海防市|海防]]乘船经[[香港|香港]]赴[[上海|上海]]，从上海乘火车到[[河南|河南]][[洛阳|洛阳]]。同年6月，他独自从洛阳出发，最终抵达虎嵩山阿訇帐下。当年[[中国西北|中国西北]]地区战乱不断、匪患严重、[[霍乱|霍乱]]流行，民众生活困苦。他沿途与当地回民同样常以野菜及植物块根为食。晚年他回忆称：“这段旅途，也可以说是[[唐僧|唐僧]]取经，唐僧穿过[[火焰山|火焰山]]，[[三打白骨精|三打白骨精]]，我是穿过赤地千里的[[黄土高原|黄土高原]]，几战霍乱症。如果当时没有坚强的毅力，是不可能到达目的地的。”<ref name=zgzj/><ref name=bkdd>{{Cite web |url =http://cpc.people.com.cn/GB/165240/166721/10012718.html  |title = 马坚：一位不可多得的学者 |author =  |date = 2009-09-08 |publisher = 中国共产党新闻网 |language =  |accessdate =  |quote =  }}</ref>

在虎嵩山阿訇帐下，马坚学习3年，熟练掌握了波斯语的文法、语法，还汉译了波斯文《伟嘎业》和《真境花园》<ref name=zgzj/>。不久他返回上海，进入[[哈德成|哈德成]]阿訇创办的上海伊斯兰师范学校学习。1931年4月，获[[中国回教学会|中国回教学会]]派赴[[埃及|埃及]]留学。当时埃及是[[英国|英国]][[殖民地|殖民地]]，[[签证|签证]]须向英国驻上海领事馆申请，领事馆官员见马坚出身卑微，乃故意刁难，不给签证。马坚一行只好从上海回昆明，请人帮忙才获得英国驻昆明领事馆的赴埃及签证。同年底，马坚一行抵达埃及。在基本学完[[爱资哈尔大学|爱资哈尔大学]]的课程后，马坚又选修达尔·欧鲁米阿拉伯语专科学校的课程，继续学习阿拉伯语言学、文学、史学、哲学、宗教学。 马坚在埃及留学8年，经济来源主要是校方每月发放的2[[英镑|英镑]]奖学金，所以他生活俭朴，有套[[西装|西装]]穿着时间较长，经历埃及8任内阁总理，被同学们戏称作“八内阁”西装。贫困的生活并没有摧毁马坚的意志和信念,他的求知欲望反而更加强烈。有一次他病愈后，同学林兴华约他去公园散步，但他边走边看书。林兴华后来回忆：“我接触过不少同学和学者，像马坚这样刻苦读书和善于钻研问题的人并不多见。”留学埃及留学期间，马坚用阿拉伯文著《中国回教概观》、用阿拉伯文译《[[论语|论语]]》、《茶神》、《[[河伯|河伯]]娶妻》、《中国格言谚语》，陆续在埃及[[开罗|开罗]]出版。同时他还将《回教[[基督教|基督教]]与学术文化》等伊斯兰哲学专著译成中文，寄回国内发表<ref name=zgzj/>。

1937年[[抗日战争|抗日战争]]全面爆发后，马坚等中国留学生很关心国内局势，向埃及当地民众及其他各国留学生宣传中国人民的抗日斗争，揭露[[日本|日本]]侵略者的暴行。在开罗的华侨成立了“中国战区灾民救济会”，马坚任该会秘书，该会向阿拉伯友人介绍中国抗日进展，并四处募捐汇往国内支援抗日战争。1939年，[[侵华日军|侵华日军]]从沦陷区回族人里挑选出一批亲日卖国分子组成朝觐团，准备赴麦加朝觐，以制造国际舆论。马坚和几位同学得知后，立即致信哈德成阿訇，哈德成阿訇随即向[[国民政府|国民政府]]建议，以中国伊斯兰教名义派出朝觐团赴麦加朝觐，以抵制日伪朝觐团的影响。国民政府采纳了该意见，决定由埃及爱资哈尔大学的中国留学生组成中国朝觐代表团赴麦加朝觐，并拨500英镑路费。马坚随团朝觐，并开展了抗日宣传<ref name=zgzj/>。

1939年，马坚自埃及学成归国，哈德成阿訇代表中国回教学会从上海赴香港迎接，并和马坚商议了翻译《古兰经》之事。马坚短暂回云南家乡省亲后，1940年经[[滇缅公路|滇缅公路]]辗转抵达上海，参加了中国回教学会译经委员会，在[[伍特公|伍特公]]、[[沙善余|沙善余]]帮助及[[马晋卿|马晋卿]]资助下，翻译[[文言文|文言文]]《古兰经》，同时用空闲时间翻译[[白话文|白话文]]《古兰经》。经14个月共译出《古兰经》9卷。1941年[[太平洋战争|太平洋战争]]爆发后，马坚因一直参加抗日救亡宣传，被日伪列入黑名单，乃乔装成商人经[[浙江|浙江]][[杭州|杭州]]、[[兰溪|兰溪]]、[[金华|金华]]回到家乡沙甸，在中阿并授的沙甸私立养正学校教授阿拉伯语、阿拉伯文选、伊斯兰经典，并和[[中法大学|中法大学]]教授[[夏康农|夏康农]]、[[西南联合大学|西南联合大学]]教授[[郑秉壁|郑秉壁]]等人共同任教于沙甸鱼峰中学（夏康农任校长）。他在教学中提倡学生独立思考，他教的文史和伊斯兰教课程均由他自己编印教材和讲义，他还推动伊斯兰教经堂教育改革，率先在国内采用新教学方法教授阿拉伯语和宗教学。抗日战争胜利前夕，他到昆明继续翻译《古兰经》及其他译著<ref name=zgzj/><ref name=bkdd/>。

1945年抗日战争胜利后，由[[北京大学|北京大学]]、[[清华大学|清华大学]]、[[南开大学|南开大学]]组成的西南联合大学分别迁回[[北平|北平]]、[[天津|天津]]复校，1946年马坚获聘为北京大学教授，成为北京大学东方语言文学系的创始人之一，并在该系建立了阿拉伯语专业，使阿拉伯语在中国首次进入[[高等教育|高等教育]]，奠定了中国阿拉伯语教学的基础。至1949年，北京大学出版部和[[商务印书馆|商务印书馆]]已先后出版了他译的《古兰经》上册（前8卷译本和注释），随后又出版了他写的《〈古兰经〉简介》单行本<ref name=zgzj/><ref name=bkdd/>。

1949年，他和清华大学、北京大学部分教授联名上书[[傅作义|傅作义]]，为[[北平和平解放|北平和平解放]]起了推动作用。1949年9月，马坚作为宗教界和少数民族的代表，参加了[[中国人民政治协商会议第一届全体会议|中国人民政治协商会议第一届全体会议]]，并出席10月1日的[[中华人民共和国开国大典|中华人民共和国开国大典]]。[[中华人民共和国|中华人民共和国]]成立后，马坚历任[[第一届全国政协委员|第一届全国政协委员]]，第一至五届[[全国人大|全国人大]]代表。他还中国伊斯兰教协会常委、中国亚非学会理事、北京大学教授和东方语研究室主任等职。<ref name=bkdd/><ref name=sohu>{{Cite web|url = http://www.sohu.com/a/215230584_142697|title = 中国现代伊斯兰学者—马坚|author = |date = 2018-01-07|publisher = 搜狐|language = |accessdate = |quote = |archive-url = https://web.archive.org/web/20180313154009/http://www.sohu.com/a/215230584_142697|archive-date = 2018-03-13|dead-url = yes}}</ref>。

1950年代初，马坚撰写了《[[穆罕默德|穆罕默德]]的宝剑》和《回民为什么不吃猪肉》两篇文章，澄清了当时人们的一些模糊认识，在社会上产生很大反响<ref name=bkdd/>。他继续任北京大学东方语言文学系教授，并任东方语研究室主任。1950年代，在国家领导人关怀和教育部门支持下，马坚领导北京大学阿拉伯语专业按新教学大纲开展阿拉伯语新教材和《阿拉伯语汉语词典》的编写工作。他曾为[[毛泽东|毛泽东]]、[[周恩来|周恩来]]等[[党和国家领导人|党和国家领导人]]做过阿拉伯语翻译，将毛泽东著作《[[论人民民主专政|论人民民主专政]]》译成阿拉伯文出版，并参加了[[五四宪法|中华人民共和国第一部宪法]]的阿文翻译工作。他还是[[中国伊斯兰教协会|中国伊斯兰教协会]]的发起人之一，长期担任中国伊斯兰教协会常务委员。他还担任中国亚非学会理事等职。1957年中国伊斯兰教协会机关刊物《中国穆斯林》创刊，从第2期开始连载马坚译的带注释《古兰经》译本。此后因教学和社会活动繁忙，马坚一度无暇进行注释工作。“[[文化大革命|文化大革命]]”期间，马坚身心受到摧残，只剩一只眼睛勉强有微弱视力，用高倍放大镜看稿吃力，只好靠助手读译稿，他边听边口述修改意见，直到病逝前十多个小时，他还在最后一次修改核校译稿<ref name=zgzj/><ref name=bkdd/><ref name=sohu/>。另外从1957年到1965年，马坚翻译了《阿拉伯通史》，后又根据原书1970年第十版作了修改。

1978年8月16日，马坚在[[北京|北京]]病逝，享年72岁。8月26日，追悼会在[[北京八宝山革命公墓|北京八宝山革命公墓]]礼堂举行，由[[中华人民共和国教育部|教育部]]副部长、北京大学党委书记[[周林|周林]]主持，北京大学党委副书记、副校长[[高铁|高铁]]致悼词<ref>人大代表、中国伊斯兰教协会常务委员　北大教授马坚先生追悼会在京举行，人民日报1978年9月8日</ref>。1981年，[[中国社会科学出版社|中国社会科学出版社]]出版了马坚译《古兰经》现代汉语通译本，卷首由[[白寿彝|白寿彝]]作序，书中还保留了马坚1949年为前8卷带注释本撰写的《〈古兰经〉简介》（节录）和《译者序》，书末附《参考书目举要》及马坚夫人马存真写的《后记》<ref name=zgzj/>。1987年，经[[沙特阿拉伯王国|沙特阿拉伯王国]]朝觐义产部督导，中阿對照版本由[[麥地那|麥地那]]的{{link-en|法赫德國王古蘭經印刷廠|King Fahd Complex for the Printing of the Holy Quran}}出版，向世界发行，成为迄今全世界影响最大的《古兰经》汉译本<ref name=bkdd/>。

==著作==
马坚一生著作颇丰，同时他还翻译了大量的阿语作品，又将大量的汉语作品译成阿语。主要著作有：

;专著
*《中国回教概观》
*《阿拉伯语法教材》

;阿译汉作品
*《[[古兰经|古兰经]]》
*《[[阿拉伯通史|阿拉伯通史]]》， [[美国|美国]] 希提（Philip Khuri Hitti）著
*《阿拉伯简史》，[[美国|美国]] 希提著
*《回教真相》，[[黎巴嫩|黎巴嫩]]侯赛因·爱勒·吉斯尔著
*《回教哲学》，[[埃及|埃及]] 穆罕默德·阿布笃著
*《伊斯兰哲学史》，[[荷兰|荷兰]] 第·博尔著
*《伊斯兰教育史》
*《认主学大纲》
*《教义学大纲》（即《教典诠释》）
*《默罕默德宝剑》
*《回历纲要》
*《教义读本》
*《回教基督教与学术文化》
*《阿拉伯文学概况》
*《埃及宪法》
*《悬诗》（手稿，文革中遗失）

;汉译阿作品
*《[[论语|论语]]》
*《[[中国神话故事|中国神话故事]]》

;编著
*《阿拉伯语汉语词典》
