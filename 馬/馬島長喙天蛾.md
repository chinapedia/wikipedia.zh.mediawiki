'''馬島長喙天蛾'''（学名：{{lang|la|''Xanthopan morgani''}}），又名'''非洲長喙天蛾'''、'''馬達加斯加長喙天蛾'''，是[[天蛾科|天蛾科]]的一个物种，分布於[[非洲|非洲]]東部。

== 習性與特徵 ==
[[File:Wallacesesquipedale.jpg|thumb]]在1867年所繪的想像圖，「假設」某種蛾類能為大彗星蘭授粉，30多年後果然發現長喙天蛾的存在<ref name=kritsky/>]]
馬島長喙天蛾最顯著的特徵就是那演化出超長的口器，體色大部分為褐色，身上有許多黑色條紋。授粉方式是飛到花的附近，同時將口器伸進花裡。幼蟲則以[[番荔枝|番荔枝]]、[[紫玉盤|紫玉盤]]與[[木瓣樹|木瓣樹]]等[[番荔枝科|番荔枝科]]植物為食。

== 異名 ==
除了[[馬達加斯加島|馬達加斯加島]]外，馬島長喙天蛾在[[辛巴威|辛巴威]]與[[馬拉威|馬拉威]]也有分布，在早期，馬島長喙天蛾被分為兩個亞種：分布於[[非洲大陸|非洲大陸]]的族群與[[馬達加斯加|馬達加斯加]]島上的族群（Xanthopan morganii praedicta），直到後來，這兩個亞種才被確定是同一個族群。

== 發現 ==
在1830年代，[[法國|法國]][[植物學家|植物學家]]{{link-en|Louis-Marie Aubert du Petit-Thouars|Louis-Marie Aubert du Petit-Thouars}}發現了[[大彗星蘭|大彗星蘭]]，其蜜管的長度將近40公分，這點令科學家不禁懷疑，怎麼可能會有動物能幫這種植物授粉<ref>{{cite web |url=http://www.darwinproject.ac.uk/darwinletters/calendar/entry-3411.html |title=Darwin Correspondence Project - Letter 3411 — Darwin, C. R. to Hooker, J. D., 25 Jan (1862) |accessdate=2009-02-09 }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>？[[生物學家|生物學家]][[查爾斯·達爾文|查爾斯·達爾文]]在1862年的著作《蘭花的授粉》(Fertilisation of Orchids)，對這種植物做出了「在有這種植物生長的地方一定有某種長舌（或口器）生物能替其授粉」的假設，後來在1903年昆蟲學家真的發現了馬島長喙天蛾，確定這種生物能幫大彗星蘭授粉<ref name=kritsky>{{Cite news| last = Kritsky| first = Gene| year = 1991| title =Darwin's Madagascan Hawk Moth prediction| periodical =American Entomologist|volume = 37| pages =206–9| url =http://www.darwinproject.ac.uk/images/darwinsmoth_optimised.pdf| accessdate =2009-02-09| ref = harv| postscript = <!--None-->}}</ref>。

馬島長喙天蛾同種異名中的praedicta在[[拉丁文|拉丁文]]中有「預測」之意，指的就是[[查爾斯·達爾文|查爾斯·達爾文]]對大彗星蘭作的推論。
