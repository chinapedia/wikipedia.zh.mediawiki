'''馬克·艾沙姆'''（{{lang-en|'''Mark Ware Isham'''}}，{{bd|1951年|9月7日}}），是一名美國小號手、[[合成器|合成器演奏家]]以及電影作曲家，也是[[山達基|山達基]]（科學教）教徒<ref>{{Cite web |url = http://www.billboard.com/photos/6501902/musicians-scientologists/1 |title = 13 Musicians Who Are (Or Were) Scientologists |date = 2015-03-14 |publisher = Billboard |language = en}}</ref>。他的作品有多樣的種類，包括[[爵士樂|爵士樂]]、[[電子音樂|電子音樂]]和電影。<ref>{{Cite web |url = http://www.allmusic.com/artist/mark-isham-mn0000236246/biography |title = Mark Isham Biography |author = Steve Huey |publisher = Allmusic |language = en}}</ref>他已為接近一百部電影配樂。

== 生涯 ==
艾沙姆出生於紐約市，是小提琴手Patricia（née Hammond）和[[人文學|人文學]]教授Howard Fuller Isham的兒子。<ref>{{Cite web |url = http://www.filmreference.com/film/39/Mark-Isham.html |title = Mark Isham Biography (1951-) |publisher = Film Reference |language = en}}</ref>

他的音樂作品廣泛和各種各樣，包括參與多個小組，如Group 87、{{link-en|David Torn|David Torn}}的{{link-en|Cloud About Mercury|Cloud About Mercury}}項目，以及與[[瓊尼·米歇爾|瓊尼·米歇爾]]、{{link-en|Terry Bozzio|Terry Bozzio}}、{{link-en|Bill Bruford|Bill Bruford}}、{{link-en|XTC|XTC}}和Doug Lunn合作錄音。

艾沙姆是一名[[山達基|山達基教徒]]。<ref>{{Cite web |url = http://www.newyorker.com/magazine/2011/02/14/the-apostate-lawrence-wright |title = THE APOSTATE: Paul Haggis vs. the Church of Scientology. |author = Lawrence Wright |date = 2011-02-14 |publisher = The New Yorker |language = en}}</ref>他與唐娜·艾沙姆（Donna Isham）結緍。
