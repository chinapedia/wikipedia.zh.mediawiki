'''馬吉安主義'''是由[[馬吉安|馬吉安]]於144年所發展的神學思想，將[[二元論|二元論]]放在[[基督教|基督教]]的信仰中，是當時甚受歡迎的一種思想。他認為《[[希伯來聖經|希伯來聖經]]》（又稱《[[舊約|舊約]]》）內所記載的[[上帝|上帝]]與[[耶穌基督|耶穌基督]]的父的論述，有所不同，且《希伯來聖經》內本身充斥著矛盾。馬吉安指《希伯來聖經》中的上帝是「創造的上帝」，講求[[天譴|天譴]]，是充滿憤怒且暴力的神；基督之[[聖父|聖父]]則是溫柔敦厚的神。他摒棄了《希伯來聖經》，且只集合了數卷[[使徒|使徒]][[保羅|保羅]]及[[路加|路加]]的著作成為正典。

由於馬吉安主義沒有任何著作留下來，包括相信是記載馬吉安思想的《對比論》（''Antitheses''），他的主張及思想只有從反對他的人的著作中重新構築出來。這些作者包括反對馬吉安主義最厲害的[[鐵徒良|鐵徒良]]（著有五卷《反馬吉安》）、[[愛任鈕|愛任鈕]]（著有《反異端》）等。在325年的[[第一次尼西亞公會議|第一次尼西亞公會議]]，馬吉安主義被判為[[異端|異端]]，而《[[聖經|聖經]]》的正典亦同時被確立。

== 發展歷史 ==
馬吉安主義的開展要追索至約在144年，馬吉安被[[羅馬教廷|羅馬教廷]]處以[[破門律|破門律]]開始。馬吉安原是[[黑海|黑海]]一帶的商人，在[[本都|本都]]的富有[[基督徒|基督徒]]，亦是一名主教的兒子。他約在140年到達[[羅馬|羅馬]]，其後發生[[巴爾·科赫巴的暴亂|巴爾·科赫巴的暴亂]]。這次的暴亂與其他[[猶太羅馬戰爭|猶太羅馬戰爭]]（即[[猶太起義|猶太起義]]等），提供了馬吉安主義發展的歷史台階。

馬吉安利用羅馬教廷向他歸還的捐款開展自己的教會，與羅馬教廷展開對抗。馬吉安主義接著在西方發展接近300年，後數個世紀發展至東方，尤其是在[[拜占庭帝國|拜占庭帝國]]以外充斥[[摩尼教|摩尼教]]的地區。湊巧地，現今相信[[摩尼|摩尼]]是一個[[曼達安教|曼達安教]]擁戴者，而曼達安教與馬吉安主義在某些程度上有關。舉例說，曼達安教及馬吉安主義均相信[[德謬哥|德謬哥]]。

馬吉安主義在325年舉行的[[第一次尼西亞公會議|第一次尼西亞公會議]]內，被判為異端。在《[[聖經|聖經]]》正典被確立的同時，於5世紀後消聲匿跡。

== 思想要點 ==
馬吉安主張[[基督教|基督教]]與[[猶太教|猶太教]]非但是不同的宗教，還屬於敵對，基於[[基督|基督]]的教導與[[猶太教|猶太教]]的互不協調，與及《[[希伯來聖經|希伯來聖經]]》的自我矛盾等，馬吉安摒棄《希伯來聖經》，並指當中的[[上帝|上帝]][[耶和華|耶和華]]，雖然創造了這個世界，但卻是邪惡的源頭，所以不是完美的，而是一個低階的{{link-en|德謬哥|Ddmiurge}}。馬吉安主義受到當時[[希臘文化|希臘文化]]、[[羅馬文化|羅馬文化]]的影響，並從[[柏拉圖主義|柏拉圖主義]]的觀點批判《希伯來聖經》。馬吉安主張《希伯來聖經》的耶和華是矛盾、嫉妒、憤怒，還會實行[[種族滅絕|種族滅絕]]的，而由祂所創造的物質世界，是一個充滿患難的世界。創造這樣的世界的神，必然是一個笨拙的或惡毒的德謬哥。

管理[[自然|自然]]及[[人類|人類]]的法則對馬吉安而言，就與這個[[造物主|造物主]]因嚴厲的執行正義而易怒、好鬥以及殘酷的的特質相似，且必然是從祂而來，祂所以是世界的創造者及主宰。然而這套管理世界的法則是如此的不靈活，甚至充滿矛盾及暴虐，這正與《希伯來聖經》表現的上帝一致。[[基督|基督]]並非[[猶太人|猶太人]]的[[彌賽亞|彌賽亞]]，反倒是由[[獨一個體|獨一個體]]所派來的靈體，將存在的真理彰顯，使人性從德謬哥的陷阱中解放。很明顯的，馬吉安主義的[[基督論|基督論]]是根據[[幻影說|幻影說]]：否認基督的人性。

馬吉安在[[使徒|使徒]]中獨推崇[[保羅|保羅]]及與保羅關係密切的[[路加|路加]]，認為其他的使徒均被受猶太教的薰陶，唯獨保羅從基督處獲得救贖的真意。使徒們的著作當中，他只認同保羅及路加的作品。任何其他的著作及與《希伯來聖經》有關聯的傳統，他都認定是與[[真理|真理]]相違背，而真理的重點就是由保羅所提倡律法與[[福音|福音]]、[[憤怒|憤怒]]與[[恩典|恩典]]、[[行為|行為]]與[[信心|信心]]、[[肉體|肉體]]與[[靈魂|靈魂]]、[[罪|罪]]與[[義|義]]、[[死亡|死亡]]及[[生命|生命]]的爭議。他從而將這些爭議歸咎於《希伯來聖經》中公義及憤怒的上帝，世界的創造者，福音的次等神，為滿有愛心及憐憫的[[基督|基督]]所不認識的。

馬吉安修改了《[[路加福音|路加福音]]》，刪去了所有有關基督降生的預言，加上[[保羅書信|保羅書信]]（但不包括[[教牧書信|教牧書信]]及《[[希伯來書|希伯來書]]》）及[[偽經_(基督宗教)|偽經]]的《[[老底嘉書|老底嘉書]]》，成為《對比論》。這本《對比論》雖不被接納為《[[聖經|聖經]]》正典，但卻是最早編輯成書的《[[新約聖經|新約聖經]]》。

馬吉安主義雖然與[[諾斯底主義|諾斯底主義]]並不盡相同，但卻與它的某些信念有著關係。但諾斯底主義是建基於他們認為他們所擁有的靈智，而馬吉安主義則是著重[[保羅書信|保羅書信]]及[[耶穌|耶穌]]的講話的內容—亦即經典的內容。早期基督教[[神學家|神學家]]對於是否將馬吉安納入為諾斯底主義的一派而產生分歧。如[[阿道夫·馮·哈納克|阿道夫·馮·哈納克]]就拒絕這種說法，他認為馬吉安本身拒絕說明創造的傳說，亦沒有聲稱擁有任何特別啟示或秘密知識；但[[米德|米德]]卻認為馬吉安的[[二元論|二元論]]等思想與諾斯底主義不謀而合，應屬於其中一派。

== 對主義的批判 ==
根據[[俄利根|俄利根]]註釋《[[馬太福音|馬太福音]]》的著作的備註中，馬吉安禁止使用寓意釋經法來解釋經典。[[特土良|特土良]]就曾在其著作中質疑馬吉安的做法，並聯同[[伊皮法紐|伊皮法紐]]指控馬吉安將《馬太福音》、《[[馬可福音|馬可福音]]》及《[[約翰福音|約翰福音]]》摒棄及只使用《[[路加福音|路加福音]]》。特土良推論馬吉安這種做法是要回復[[耶穌|耶穌]]的教導，並與[[愛任紐|愛任紐]]批評馬吉安主義救贖的不完全。

[[希坡律陀|希坡律陀]]指馬吉安是相信[[幻影說|幻影說]]：即[[基督|基督]]以人的形象出現，本質卻不是人，且沒有真正在[[十字架|十字架]]上死去。

因為馬吉安拒絕《[[希伯來聖經|希伯來聖經]]》，一些[[基督徒|基督徒]]相信馬吉安是[[反猶太主義|反猶太主義者]]。事實上，現今「馬吉安主義」亦會受來指一些存有反猶太主義的[[教會|教會]]。另一方面，馬吉安並不稱基督徒為[[取代論|取代論]]的「新以色列」，亦從未曾引用《希伯來聖經》來支持自己的觀點。但是，馬吉安從未曾顯示他是一個反猶太主義者，他只是反對[[猶太人|猶太人]]的經典而已。

在一些《[[聖經|聖經]]》的古拉丁抄本中，[[保羅書信|保羅書信]]的序言（並非內文，只有數行簡介的文字）相信是由馬吉安或是他的跟隨者所寫的。據稱，在最先有序言的[[富爾達抄本|富爾達抄本]]及其他其後的抄本中，該序言是由馬吉安主義者所寫的。相反，其他早期的拉丁[[福音書|福音書]]抄本亦有反馬吉安主義的序言。

另外，相信馬吉安對其跟隨者施以嚴厲的道德要求，而當中有些曾在逼害中殉道，他更不接納那些曾在[[羅馬|羅馬]]的逼害中放棄信仰的信徒。他的其他跟隨者，如[[亞皮利|亞皮利]]，則獨自創造另一些不同的教派。

==現代的馬吉安主義==
原有的馬吉安主義已於約5世紀消聲匿跡，但是馬吉安的影響及他對《[[舊約聖經|舊約聖經]]》的批判一直延伸至今天。馬吉安所謂在《舊約聖經》中找出的矛盾，亦是今天很多[[哲學家|哲學家]]對[[基督教|基督教]]批判所引用，尤其是在經文內明顯認可的暴力及種族滅絕。很多[[無神論|無神論者]]、[[不可知論|不可知論者]]及[[人道主義|人道主義者]]均認同馬吉安有關聖經暴行的例子，並引用同樣的經文來批評基督教及[[猶太教|猶太教]]。有些[[基督徒|基督徒]]亦同意馬吉安所指《舊約聖經》內的種族滅絕並非一個好的模範去跟從，亦並非「公義的懲罰」。

對於某些現今自稱為馬吉安主義者，同樣放棄《舊約聖經》而著重《[[新約聖經|新約聖經]]》。[[純潔派|清潔派]]，或現今的[[純潔派|清潔派]]運動，持馬吉安同樣的理由去拒絕《舊約聖經》，究竟他們是源自於馬吉安主義或是自行創立的就不得而知了。[[托马斯·杰斐逊|托马斯·杰斐逊]]及[[托馬斯·潘恩|托馬斯·潘恩]]都是出名的[[自然神論|自然神論者]]拒絕《舊約聖經》。其他如[[伏爾泰|伏爾泰]]、[[查爾斯·狄更斯|查爾斯·狄更斯]]、[[查爾斯·羅伯特·達爾文|查爾斯·羅伯特·達爾文]]、[[聖雄甘地|聖雄甘地]]及[[阿爾伯特·愛因斯坦|阿爾伯特·愛因斯坦]]等均表示反對《舊約聖經》的[[耶和華|耶和華]]及欽佩耶穌。[[西蒙·威爾|西蒙·威爾]]被認為是當代的馬吉安主義者，儘管她是跟隨重建後的馬吉安主義，而非歷史遺產的延續。
