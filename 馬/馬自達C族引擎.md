'''馬自達C族引擎'''是1960年代橫跨1980年代由[[日本|日本]][[馬自達|馬自達]]汽車公司研發、生產的往復式活塞[[汽油引擎|汽油引擎]]，[[排氣量|排氣量]]分別從1.0L至1.8L不等；此系列是該公司最早發展的[[往復式發動機|往復式活塞引擎]]。

== PC型 ==
排氣量985c.c.的'''PC型'''引擎採[[頂置凸輪軸#單頂置凸輪軸（SOHC）|SOHC]]、8汽門設計，缸徑70mm、衝程64mm。這具縱置式引擎被設計成[[前置後驅|前置後驅]]型式，在外銷版裡可榨出45hp的最大馬力、69 N·m的峰值扭力。車型：
# 1965年-1967年：[[馬自達323#第一代（1963年-1968年）|馬自達1000 Coupé]]
# 1970年-1978年：[[馬自達323#第二代（1967年-1978年）|Familia Presto / 馬自達1000]]
# 1977年-1980年：[[馬自達323#第四代（1977年-1980年）|第四代323]]

== TC型 ==
排氣量1,272c.c.的直列四缸SOHC '''TC型'''之壓縮比為1：9.2，缸徑為73mm，衝程76mm，乃自PC型擴缸而來。可達最大馬力87ps / 6,000rpm、最大扭力則為11.0kg·m / 3,500rpm，極速155km/hr<ref>參看{{ja}}[http://www.spacecatalog.biz/mazda/245_1.html マツダファミリアプレスト]。</ref>。值得注意的是，馬自達基於旋渦吸氣的[[稀薄燃燒|稀薄燃燒]]方式，採用[[排氣再循環|排氣再循環]]（Exhaust Gas Recirculation）技術，其汽化器可控制空燃比自動補正，有效降低排出氣體中的[[氮氧化物|氮氧化物]]（NOx），因而獲得財團法人汽車技術會的「技術240選」<ref>參考{{ja}}[http://www.jsae.or.jp/autotech/data/9-8.html 日本の自動車技術240選：マツダSCSエンジン(TC)] {{Wayback|url=http://www.jsae.or.jp/autotech/data/9-8.html |date=20130310182952 }}。</ref>。車型：
# 1970年-1978年：[[馬自達323#第二代（1967年-1978年）|Familia Presto / 馬自達1300]]
# 1971年-1978年：[[馬自達Grand_Familia|Grand Familia / 馬自達Mizer]] / 馬自達808 / 馬自達818
# 1974年-1981年：起亞Brisa
# 1977年-1980年：[[馬自達323#第四代（1977年-1980年）|第四代323]]
# 1977年-1983年：[[馬自達Bongo#第二代（BA2型_1977年至1983年）|第二代Bongo / E1300]]
# 1982年-1984年：大宇Maepsy<ref>參看{{en}}[http://joongangdaily.joins.com/article/view.asp?aid=2913903 Small sedan made a big contribution to Daewoo Motor Co.]一文，第五段。</ref>

== UC型 ==
1,415c.c.的'''UC型'''引擎缸徑77mm、衝程76mm，最大馬力70hp。車型：
# 1977年-1980年：[[馬自達323#第四代（1977年-1980年）|第四代323 / 馬自達GLC]]
# 1977年-1983年：[[馬自達Bongo#第二代（BA2型_1977年至1983年）|第二代Bongo / E1400]]
# 1984年-1985年：福特Econovan

== UB型 ==
排氣量1,490c.c.之'''UB型'''引擎採直列四缸SOHC設計，缸徑與衝程皆為78mm，最大馬力78ps、最大扭力11.8kg·m；此具引擎後來被更大排氣量的NA型取代。車型：
# 1966年-1972年：[[馬自達Luce#第一代（1966年-1972年）|第一代Luce]]
# 1970年-1974年：[[馬自達626#第一代（SNA/S122A系　1970年-1974年）|第一代Capella / 馬自達616]]
# 1971年-1978年：[[馬自達Grand_Familia|Grand Familia / 馬自達808 / 馬自達818]]

== NA型 ==
直列四缸SOHC、排氣量1,586c.c.的'''NA型'''引擎，缸徑為78mm、衝程為83mm，配置一具尼崎公司（{{ja}}株式会社ニッキ）製造的[[化油器|化油器]]。依不同地區市場特性調校成：
* 日規：最大馬力100hp / 6,000rpm，最大扭力14.0kg·m / 3,500rpm。
* 美規：最大馬力70hp / 5,000rpm，峰值扭力82lb·ft / 3,500rpm。

此外，1970年代為了因應新頒布的汽車廢氣排放標準，改良成「1600AP型」（anti-pollution之意），但馬力輸出卻下降：
* 日規AP型：最大馬力90hp / 6,000rpm，最大扭力13.0kg·m / 3,500rpm<ref>參考{{ja}}[http://www.spacecatalog.biz/mazda/244_1.html 1970年代のマツダファミリア1600APは低公害車だ]一文。</ref>。
車型：
# 1970年-1974年：[[馬自達626#第一代（SNA/S122A系　1970年-1974年）|第一代Capella / 馬自達616]]
# 1971年-1978年：[[馬自達Grand_Familia|Grand Familia / 馬自達808 / 馬自達818]] 
# 1971年-1975年：[[馬自達B系列#第三代（1971年-1978年）|第三代Proceed / B1800]]
# 1972年-1973年：[[馬自達323#第三代（1973年-1977年）|馬自達808]]（[[美國|美國]]）
# 1974年-1978年：[[馬自達626#第二代（CB125系　1974年-1978年）|第二代Capella / 626]]
