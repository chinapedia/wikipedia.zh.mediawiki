'''馬來糕'''，常稱為'''馬拉糕'''，是著名的[[廣東菜|粵式]]糕點，是[[香港|香港]]及[[廣東|廣東]][[茶樓|茶樓]]常見的[[點心|點心]]，是一種[[蒸|蒸]]製而成的[[蛋糕|蛋糕]]<ref>[http://hk.apple.nextmedia.com/etw/%E9%A3%B2%E9%A3%9F%E5%86%B7%E7%9F%A5%E8%AD%98/article/20160923/3_55571874/CNN%E4%B8%96%E7%95%8C%E5%8D%81%E4%B8%83%E5%A4%A7%E8%9B%8B%E7%B3%95-%E9%A6%99%E6%B8%AF%E9%A6%AC%E6%8B%89%E7%B3%95%E4%B8%8A%E6%A6%9C- CNN世界十七大蛋糕 香港馬拉糕上榜？]，[[蘋果日報|蘋果日報]]，2016年09月23日</ref>。

== 概要 ==
馬拉糕的特色是[[氣孔|氣孔]]有三層，頂層是直的，而低層是橫的。茶樓內的馬拉糕通常製成一大型圓狀，切開成小塊發售。馬拉糕呈金黃色，新鮮吃時非常蓬鬆、柔軟，帶有輕微的香味。有些愛吃馬拉糕的人會專挑原來是在圓狀糕點周圍的部份來吃，因為旁邊的馬拉糕水蒸氣較少，口感特別好。

== 起源 ==
馬拉糕雖然是香港的代表性糕點，但其起源已不可考<ref>[http://hk.on.cc/hk/bkn/cnt/news/20160908/bkn-20160908060034167-0908_00822_001.html 馬拉糕登世界殿堂 揭糕邊美味謎團]，[[東方日報|東方日報]]，2016年09月08日</ref>，其中一個較為普遍的說法是[[英國人|英國人]]統治[[馬來半島|馬來半島]]後，仍保留[[下午茶|下午茶]]進食[[糕點|糕點]]的習慣，但因為在馬來半島[[椰奶|椰奶]]比[[牛奶|牛奶]]常見，烤箱也不易找，於是使用椰奶取代牛奶，並使用蒸製方法而無需使用烤箱製作蛋糕<ref>[http://www.hkcna.hk/content/2016/0830/492188.shtml 馬拉糕入圍全球最正蛋糕]，香港新聞網，2016年08月30日</ref>。原本馬來亞的糕點是加有[[斑蘭葉|斑蘭葉]]調味而成為綠色的蛋糕，廣東廚師雖然參考了這種蛋糕的製作方法，但因為在廣東難找斑蘭葉，於是把斑蘭葉省去，卻蒸出金黃色的馬拉糕，並突出了雞蛋的味道<ref>[https://www.e123.hk/ElderlyPro/details/197605/74 小朋友大問題﹕點心師 點點心思]，長青網，2012年10月31日</ref>。

== 製作 ==
正宗的馬拉糕由[[麵粉|麵粉]]、[[雞蛋|雞蛋]]、[[豬油|豬油]]、[[奶油|牛油]]混合[[發酵|發酵]]超過24小時<ref>[http://hk.apple.nextmedia.com/realtime/news/20161119/55936974 添好運主理人培哥：泡打粉含鋁 點心才鬆軟]，[[蘋果日報|蘋果日報]]，2016年11月19日</ref>，最後放在[[蒸籠|蒸籠]]內[[蒸|蒸]]製而成，亦有簡易馬拉糕製法，使用[[紅糖|紅糖]]且無需[[發酵|發酵]]，但鬆軟程度會稍低。

== 泡打粉含鋁 ==
食肆基於時間及成本考量，大部分都不會使用長時間的發酵方法製作馬拉糕，而是於麵糊加入泡打粉，使完成後的馬拉糕有鬆軟口感<ref>[http://orientaldaily.on.cc/cnt/news/20090521/00176_065.html 馬拉糕班戟鬆餅含鋁高]，[[東方日報|東方日報]]，2009年05月21日</ref>。雖然使用泡打粉可大幅縮短製作時間，但泡打粉普遍含有[[鋁|鋁]]，人體長期攝取鋁會有損健康，對[[腦部|腦部]]及[[生殖系統|生殖系統]]造成損害<ref>[http://www.example.org 海蜇含鋁勁高 每周4件馬拉糕超標]，[[成報|成報]]，2016-11-19</ref>。
