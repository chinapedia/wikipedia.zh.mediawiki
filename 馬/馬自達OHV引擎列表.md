雖然目前[[馬自達|馬自達]]所有的輕型車款（kei car）都委託[[鈴木株式會社|鈴木汽車]]代工製造，不過1960年代與1970年代之間都使用自製的OHV頂桿式汽門機構引擎。

== AA型 ==
[[排氣量|排氣量]]359c.c.的'''AA型'''引擎為水冷式直列二缸二衝程之設計，搭配單[[化油器|化油器]]後可輸出最大馬力35ps / 6,500rpm、最大扭力4.0kgm / 5,500rpm。車型：
# 1972年-1976年：[[馬自達Chantez|馬自達Chantez]]
# 1973年-1976年：[[馬自達Porter#Porter（1968年至1976年）|馬自達Porter]]
# 1973年-1989年：[[馬自達Porter#Porter_Cab（1969年至1989年）|馬自達Porter Cab]]

== DA型 ==
358c.c.水冷式OHV四行程[[直列四缸引擎|直列四缸]]'''DA型'''引擎的缸徑46mm、衝程54mm，引擎壓縮比10.0:1。該具引擎曾是世界上最小的量產[[四冲程循环|四行程汽車引擎]]，此紀錄後來被[[本田技研工業|本田汽車]]的356c.c.直列四缸AK250E型引擎改寫。馬力輸出可分成：
# 最大馬力18ps / 6,800rpm，峰值扭力2.1kg·m / 5,000rpm<ref>參看{{ja}}[http://www.asahi-net.or.jp/~rf7k-inue/izen/no-8/caroru/cr-siyo.html マツダ・キャロル詳しい仕様書]。</ref>。
# 最大馬力20ps / 7,000rpm，峰值扭力2.4kg·m / 5,000rpm<ref>參看{{ja}}[http://blogs.yahoo.co.jp/rara1735/GALLERY/show_image_v2.html?id=http%3A%2F%2Fblogs.c.yimg.jp%2Fres%2Fblog-8f-fc%2Frara1735%2Ffolder%2F432680%2F09%2F9006609%2Fimg_6%3F1337002908 春のバンまつり（サブロク編）④]。</ref>。
車型：
# 1961年-1968年：[[馬自達B360|馬自達B360]]
# 1962年-1970年：[[馬自達Carol#第一代（1962年-1970年）|第一代Carol]]
# 1968年-1973年：[[馬自達Porter#Porter（1968年至1976年）|馬自達Porter]]

== RA型 ==
586c.c.水冷式OHV四行程直列四缸'''RA型'''引擎算是DA型的加大版，其缸徑54mm、衝程64mm，最大馬力28hp。車型：
# 1961年-1968年：[[馬自達B360|馬自達B600]]
# 1962年-1964年：[[馬自達Carol#Carol_600_（1962年-1964年）|馬自達P600]]

== SA型 ==
排氣量782c.c.水冷式OHV直列四缸'''SA型'''引擎之缸徑58mm、衝程74mm，最大馬力達42ps / 6,000rpm，峰值扭力6kg·m / 3,200rpm，壓縮比8.5:1。由於該具引擎之汽缸頭以[[鋁合金|鋁合金]]製成，打開引擎蓋時可見其白色外殼，馬自達遂以「白色引擎」（{{ja}}白しエンジン）作為推銷用詞。車型：
# 1963年-1968年：[[馬自達323#第一代（1963年-1968年）|第一代323]]
# 1966年-1975年：[[馬自達Bongo#第一代（1966年至1975年）|第一代Bongo]]

== PB型 ==
987c.c.水冷式OHV直列四缸'''PB型'''引擎之方形缸徑與衝程均為68mm，最大馬力依調校程度可輸出48至58ps不等。車型：
# 1963年-1968年：[[馬自達323#第一代（1963年-1968年）|馬自達1000]]

== TA型 ==
排氣量1,139c.c.的'''TA型'''引擎採水冷式、直列四缸、頂置汽門設計，缸徑70mm、衝程74mm，引擎[[壓縮比|壓縮比]]7.8:1<ref>參看{{ja}}[http://www.asahi-net.or.jp/~rf7k-inue/izen/no-5/mazda/mz-siyo.html 詳しい仕様書]。</ref>。最大馬力46ps / 4,600rpm，最大扭力8.0kg·m / 3,000rpm，極速為87km/hr。車型：
# 1959年-1962年：[[馬自達T1500#概要與歷史|馬自達T1100]]
# 1959年：馬自達D1100

== TB型 ==
1,169c.c.水冷式OHV直列四缸'''TB型'''引擎之缸徑70mm、衝程76mm，並使用[[日立|日立]]製造的[[化油器|化油器]]，具8.6:1的[[壓縮比|壓縮比]]。最大馬力達68ps / 6,000rpm、最大扭力則為9.6kg·m / 3,000rpm。車型：
# 1968年-1978年：[[馬自達323#馬自達R100|馬自達1200]]

== UA型 ==
1,484c.c.水冷式OHV直列四缸'''UA型'''引擎之缸徑75mm、衝程84mm，可區分成幾個亞型：搭載於D1500四輪貨車的亞型最大馬力60ps / 4,600rpm、最大扭力10.4kg·m / 3,000rpm<ref>參看{{ja}}[http://www.asahi-net.or.jp/~rf7k-inue/izen/no-14/d1500/d-siyo.html D1500詳しい仕様書]。</ref>，而第一代B1500的亞型最大馬力59hp，但第二代B1500因為改善了引擎的汽缸頭與閥門，且把原本的側吸式[[化油器|化油器]]（sidedraft carburetor）改為下吸式（downdraft carburetor），最大馬力增至72ps / 5,200rpm、最大扭力12.0kg·m / 3,400rpm。車型：
# 1956年-1971年：[[馬自達T1500|馬自達T1500]]
# 1958年-1962年：[[馬自達Romper|D1500]]
# 1961年-1965年：[[馬自達B系列#第一代（1961年-1965年）|第一代B1500]]
# 1965年-1971年：[[馬自達B系列#第二代（1965年-1971年）|第二代B1500]]
# 1965年-1977年：[[馬自達Kraft|馬自達Kraft]]

== UB型 ==
排氣量1,484c.c.的'''UB型'''同樣仍屬水冷式OHV四缸設計，不過稍作改良。最大馬力60ps / 4,600rpm，最大扭力10.4kg・m / 3,000rpm。車型：
# 1956年-1971年：[[馬自達T1500|馬自達T1500]]

== VA型 ==
水冷式OHV四行程直列四缸'''VA型'''引擎之排氣量為1,985c.c.，最大馬力81ps / 4,600rpm，峰值扭力15.5kg・m / 2,000rpm<ref>請見{{ja}}[http://sanrinsya.xxxxxxxx.jp/page022.html Ｔ２０００　ＴＶＡ１２Ｓ]。</ref>。車型：
# 1962年-1974年：[[馬自達T2000|馬自達T2000]]
# 1965年-1972年：[[馬自達Light_Bus|馬自達Light Bus]]
# 1971年-1980年：[[馬自達Titan#第一代（1971年-1980年）|第一代Titan]]
# 1972年-1982年：[[馬自達Parkway#第一代（1972年-1982年）|第一代Parkway]]

== VB型 ==
水冷式OHV直列四缸'''VB型'''引擎同樣為1,985c.c.。車型：
# 1962年：D2000（[[馬自達Romper|馬自達Romper]]後繼車款）
