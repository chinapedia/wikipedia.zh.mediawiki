'''馬尿酸'''（{{lang|en|Hippuric acid}}），学名'''苯甲酰胺乙酸'''，分子式C<sub>9</sub>H<sub>9</sub>NO<sub>3</sub>，结构简式C<sub>6</sub>H<sub>5</sub>CONHCH<sub>2</sub>CO<sub>2</sub>H，该词起源于[[希臘語|希臘語]]的''Hippos''（{{lang|en|horse}}）和''ouron''（{{lang|en|urine}}）。它是一種在[[馬|馬]]和[[草食性動物|草食性動物]][[尿液|尿液]]中發現的[[有机酸|有机酸]]。苯甲醯胺乙酸晶体易溶於熱水，在187℃左右熔化，大约于240℃分解。许多种的芳香化合物（比如[[苯甲酸|苯甲酸]]和[[甲苯|甲苯]]）摄入体内后，都会通过和[[氨基酸|氨基酸]]（如[[甘氨酸|甘氨酸]]）反应转化为苯甲醯胺乙酸，因此，体内马尿酸浓度过高可能是由甲苯中毒导致的。由于引起苯甲醯胺乙酸中毒的因素还有很多，因此科学家对两者之间的联系仍存怀疑。<ref>http://www.ncbi.nlm.nih.gov/pubmed/19891605</ref><ref>http://en.wikipedia.org/wiki/Toluene_%28toxicology%29#Measure_of_exposure</ref>。

== 物化性質 ==
* 苯甲醯胺乙酸可溶解于[[氫氧化鈉|氫氧化鈉]]或[[氫氧化鉀|氫氧化鉀]]水溶液，經[[水解|水解]]會分解為[[苯甲酸|苯甲酸]]和[[甘氨酸|甘氨酸]]。
* 和[[亞硝酸|亞硝酸]]反應，可轉換成[[苯甲酸乙醇酸酯|苯甲酸乙醇酸酯]]C<sub>6</sub>H<sub>5</sub>C(=O)OCH<sub>2</sub>CO<sub>2</sub>H。马尿酸乙[[酯|酯]]可以和[[肼|肼]]进行反应可得到马尿酰肼C<sub>6</sub>H<sub>5</sub>CONHCH<sub>2</sub>CONHNH<sub>2</sub>。该化合物由[[:en:Theodor_Curtius|Theodor Curtius]]发现可用于制备[[偶氮亚胺化合物|偶氮亚胺化合物]]。

== 歷史 ==
*1829年：[[尤斯圖斯·馮·李比希|尤斯圖斯·馮·李比希]]發現苯甲醯胺乙酸和[[苯甲酸|苯甲酸]]是兩種不同物質。
*1839年：尤斯圖斯·馮·李比希確定了苯甲醯胺乙酸的分子式。
*1873年：Victor Dessaignes利用[[苯甲醯氯|苯甲醯氯]]與[[甘氨酸鋅|甘氨酸鋅]]之間的反應，合成苯甲醯胺乙酸<ref>"Ueber die Regeneration der Hippursäure" Justus Liebigs Ann. Chem. '''1853''', ''87'', 325-327. DOI: [http://dx.doi.org/10.1002/jlac.18530870311 10.1002/jlac.18530870311]</ref>。

== 制备 ==
除了上述利用的化學反應合成外，在[[:en:Theodor_Curtius|Theodor Curtius]]的报告中还描述了另一种合成方式，將[[氨基乙酸|氨基乙酸]]溶於[[氫氧化鈉|氫氧化鈉]]溶液中，並且攪拌加入[[苯甲酰氯|苯甲酰氯]]及[[氫氧化鈉|氫氧化鈉]]溶液，保持反應物呈[[鹼性|鹼性]]。令[[液體|液體]][[飽和_(化學)|飽和]]後加[[盐酸|盐酸]]使[[pH值|pH值]]下降至2，過濾並將固體溶於水中[[重結晶|重結晶]]後可以得到苯甲醯胺乙酸。<ref>T. Curtius "Synthese von Hippursäure und Hippursäureäthern" ''Ber. Deutsch. Chem. Ges.'' '''1884''', ''17'', 1662-1663. == 體內產生 ==
[[甲苯|甲苯]]等芳香族化合物進入體內，會被[[肝臟|肝臟]]中的[[細胞色素P450|細胞色素P450]]（CYP2E1）甲羥基化，產生[[苯甲醇|苯甲醇]]（同時產生5%的[[環氧化合物|環氧化合物]]）；苯甲醇被[[醇脫氫酶|醇脫氫酶]]氧化，產生[[苯甲酸|苯甲酸]]；[[苯甲酸|苯甲酸]]与[[甘氨酸|甘氨酸]]脱水缩合，產生苯甲醯胺乙酸。见下圖：
