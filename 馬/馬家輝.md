'''馬家輝'''博士（{{lang|en|Dr. MA Ka Fai}}，{{bd|1963年|catIdx=M马}}），生於[[香港|香港]]，长于[[灣仔|灣仔]]，父為資深報人[[馬松柏|馬松柏]]，曾任广告方案、杂志记者、报社编辑、專欄作家、文化評論學者，亦為台灣問題研究員；現為[[香港城市大學|香港城市大學]]中文及歷史學系助理教授，亦為台灣及香港報刊專欄作家，並有多本著作結集出版。其偶像為[[李敖|李敖]]、[[梁文道|梁文道]]、[[繆雨|繆雨]]、[[胡菊人|胡菊人]]等。与[[梁文道|梁文道]]、[[萧若元|萧若元]]、[[刘细良|刘细良]]、[[王慧麟|王慧麟]]、[[陶杰|陶杰]]等并称为“香江才子”。

== 简介 ==
1963年，马家辉生于[[香港|香港]]，他是家中的独子，在[[湾仔|湾仔]]长大，<ref name="《賤嘴馬“賤”是一種批判的態度》">{{cite web |url=http://news.ifeng.com/gundong/detail_2011_05/28/6682493_0.shtml|title=《賤嘴馬“賤”是一種批判的態度》|accessdate=2011年5月28日|work=广州日报|publisher=凤凰网|language=zh-hant}}</ref>就讀[[佛教黃鳳翎中學|佛教黃鳳翎中學]]。17岁，马家辉在一家书店发现一本谈[[李敖|李敖]]的书籍，<ref name="马家辉：年纪越大我越崇拜李敖">{{cite web|url=http://news.xinhuanet.com/book/2009-12/24/content_12696331.htm|title=《马家辉：年纪越大我越崇拜李敖》|accessdate=2009年12月24日|work=|publisher=新华网|language=zh-hans}}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>19岁那年起，他开始迷恋上[[李敖|李敖]]的作品，并立志21岁前写出一本关于李敖的著作。高考之后，马家辉未得到香港兩間大學的取錄，只被[[香港浸會學院|香港浸會學院]]文憑課程取录，於是选择了去[[台湾大学|台湾大学]]入讀[[心理系|心理系]]學士課程。<ref name="马家辉：年纪越大我越崇拜李敖">{{cite web|url=http://news.xinhuanet.com/book/2009-12/24/content_12696331.htm|title=《马家辉：年纪越大我越崇拜李敖》|accessdate=2009年12月24日|work=|publisher=新华网|language=zh-hans}}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref> 马家辉在台大读书时认识了李敖。大二时马家辉出版了《消灭李敖，还是被李敖消灭》，李敖读完后对马家辉说：“小马，我当你面这样说，背后也是这样说——我以前研究[[胡适|胡适]]，胡适跟我说，李敖，你比我胡适更了解胡适。那我今天跟你说，你比我李敖更了解李敖”，并且在1997年的回忆录中将马家辉列为平生所交好友名单。<ref name="《賤嘴馬“賤”是一種批判的態度》">{{cite web |url=http://news.ifeng.com/gundong/detail_2011_05/28/6682493_0.shtml|title=《賤嘴馬“賤”是一種批判的態度》|accessdate=2011年5月28日|work=广州日报|publisher=凤凰网|language=zh-hant}}</ref>

1997年1月，马家辉和妻儿住在[[台北|台北]][[大直|大直]]，春节时节，接到[[高信疆|高信疆]]的电话，邀请他去香港《[[明报|明报]]》工作，他成了《明报》的副总编辑，且开设专栏，开始写尖刻的时评，被人们称为“贱嘴马”。<ref name="《賤嘴馬“賤”是一種批判的態度》">{{cite web |url=http://news.ifeng.com/gundong/detail_2011_05/28/6682493_0.shtml|title=《賤嘴馬“賤”是一種批判的態度》|accessdate=2011年5月28日|work=广州日报|publisher=凤凰网|language=zh-hant}}</ref> 馬家輝曾任[[香港城市大學|香港城市大學]]中國文化中心助理主任，現任香港城市大學中文及歷史學系助理教授。<ref>{{cite web |url=https://cah.cityu.edu.hk/people/cima/ |title=馬家輝博士
|accessdate=2011-05-28 |work= |publisher=香港城市大學中文及歷史學系 |language=zh-hant}}</ref>

== 履历 ==
马家辉獲[[台灣大學|台灣大學]]心理學系學士，美國[[芝加哥大學|芝加哥大學]]社會科學碩士，[[威斯康辛大學|威斯康辛大學]]麥迪遜校區社會學博士，曾以繆思為筆名，曾任台灣華商廣告公司文案企劃、台灣《大地》地理雜誌記者、《[[明報|明報]]》世紀版和讀書版策劃顧問（同為該報專欄作家），[[鳳凰衛視|鳳凰衛視]]、[[香港有線電視|香港有線電視]]、[[香港電台|香港電台]]節目主持等。

馬家輝活躍於民間政治活動，參加不少公開論壇，分析政治問題。2008年8月28日，馬家輝出席[[有線電視_(香港)|有線電視]]舉辦的立法會新界西選舉論壇時，作為嘉賓主持卻在提問時指[[自由黨_(香港)|自由黨]]是「富貴黨」及「吸血」。自由黨的[[周梁淑怡|周梁淑怡]]在論壇後向有線高層投訴身為嘉賓主持之一的馬家輝此舉有違選舉公平<ref>[http://appledaily.atnext.com/template/apple/art_main.cfm?iss_id=20080829&sec_id=4104&subsec_id=11867&art_id=11535174 蘋果日報 - 20080829 - 自由黨被叫富貴黨　周梁投訴馬家輝] {{webarchive|url=https://web.archive.org/web/20080831151610/http://www1.appledaily.atnext.com/template/apple/art_main.cfm?iss_id=20080829&sec_id=4104&subsec_id=11867&art_id=11535174 |date=2008-08-31 }}</ref>。

== 個人生活 ==
馬家輝妻子是台灣作家林美枝（筆名：張家瑜），她的作品包括：《我開始輕視語言》和《告別式從明天開始》。在1980年代，馬家輝和林美枝都在台灣一家地理雜誌當記者，馬家輝負責採訪東南亞的緬甸、老撾、越南等地，林美枝負責採訪歐洲和中國大陸。<ref>{{cite web |title=人生就是要放肆：馬家輝和張家瑜的愛情故事 |url=https://new.qq.com/omn/20180125/20180125A0QYIE.html}}</ref> 兩人育有一女兒，合著的《小妹》一書是兩人對女兒成長過程的記錄。

== 奖项荣誉 ==
馬家輝於2008 年底獲《[[南方人物周刊|南方人物周刊]]》評選為「年度中國魅力五十人物」之一; 2010 年底, 獲《[[南方都市報|南方都市報]]》評選為“年度深圳香港意見領袖”之一。
與林美枝合著之《小妹》散文集, 2014年獲票選為香港學生好書龍虎榜首名.
