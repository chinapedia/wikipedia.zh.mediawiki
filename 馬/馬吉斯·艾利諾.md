'''馬吉斯·賴恩·艾利諾'''（{{lang-en|'''Marcus Ryan Elliott'''}}，{{bd|1984年|4月24日|}}），生於[[美國|美國]][[加利福尼亞州|加利福尼亞州]]，[[美國|美國]]籃球運動員，司職[[控球後衛|控球後衛]]，有「'''戰神'''」之稱，為現時[[東南亞職業籃球聯賽|ABL東南亞職業籃球聯賽]]史上單季最高得分球員，亦是[[2016–17年東南亞職業籃球聯賽|2016/17年度ABL常規賽]]最有價值球員、得分王兼助攻王，為2017年[[東方籃球隊|東方龍獅]]奪得[[2016–17年東南亞職業籃球聯賽|ABL總冠軍]]成員，現時效力[[東南亞職業籃球聯賽|東南亞職業籃球聯賽]]及[[香港男子甲一組籃球聯賽|香港甲一籃球聯賽]]球隊[[東方籃球隊|香港東方]]。

== 早年生活 == 
艾利諾生於[[美國|美國]][[佛羅里達州|佛羅里達州]]的[[華爾頓堡灘_(佛羅里達州)|華爾頓堡灘]]，於二年級開始打籃球，及後到中學畢業後升讀[[路易斯安那理工大學|路易斯安那理工大學]]，可是由於他與校隊教練團的關係欠佳，使他從來沒有嘗試進入[[NBA|NBA]]，更不清楚何時是[[NBA選秀|NBA選秀會]]，甚至沒有詢問有關職業籃球的事情，結果他畢業後就回家在地盤工作，不過艾利諾的籃球夢沒有就此完結，因為他深知他自己最想做個職業籃球員，故他便繼續苦練球技並開始打電話及寄電郵給相關人士。

== 籃球生涯 ==
直到2008年的一天下班後，當時正打算到[[北卡羅萊納州|北卡羅萊納州]]度假過周末的艾利諾突然接到[[智利|智利]]職業隊{{link-en|瓦爾迪維亞籃球隊|CD Valdivia|瓦爾迪維亞}}的電話，邀請他加盟展開職業球員生涯，到2015年轉會[[阿根廷全國籃球聯賽|阿根廷全國聯賽]]球隊{{link-en|阿爾馬格羅聖羅倫素籃球隊|San Lorenzo de Almagro (basketball)|聖羅倫素}}，並於首季協助球隊奪得[[阿根廷全國籃球聯賽|全國聯賽冠軍]]，

到2016年與球隊約滿後，艾利諾獲[[東方籃球隊|東方]]教練[[艾度·托利斯|艾度托利斯]]邀請加盟，並隨隊參與[[東南亞職業籃球聯賽|東南亞職業籃球聯賽]]到不同國家比賽，而他於加盟即於11月27日作客擊敗[[台灣|台灣]]球隊[[高雄聖徒職業籃球隊|高雄聖徒]]的賽事，首度為登場更取得全場最高的30分。<ref>[http://hk.on.cc/hk/bkn/cnt/sport/20170418/bkn-20170418222739952-0418_00882_001.html 東南亞職籃　勝在團結！東方龍獅搏盡追番場] on.cc東網 2017年4月18日</ref>

到2017年3月23日，艾利諾於擊敗[[西貢熱火籃球隊|西貢熱火]]的常規賽以[[大三元|三雙]]成績，再度當選賽事最有價值球員，兼累積497分成功以2分之差刷新紀錄，並打破[[ABL|ABL]]已保持長達五年的紀錄成為於[[ABL|ABL]]單季得分最多的球員，其後更獲選為[[ABL|ABL]]常規賽最有價值球員和得分王及助攻王，惟隨後艾利諾於4月18日對[[新加坡騰飛之獅籃球俱樂部|新加坡騰飛之師]]的季後賽總決賽第二回合，在封阻對方球員得分後被球證吹罰，而不服裁決的他跟球證理論時，再領2個技術犯規首度被直接逐出場，<ref>[http://hk.on.cc/hk/bkn/cnt/sport/20170322/bkn-20170322221557426-0322_00882_001.html 東南亞職籃　R1預演！東方龍獅OT撲火] on.cc東網 2017年3月23日</ref>結果他在4月23日於5場3勝制的總決賽，協助球隊以場數3–1首度參賽即奪得[[2016–17年東南亞職業籃球聯賽|ABL總冠軍]]，而他則榮膺最有價值世界外援及總決賽最有價值球員。

到2017年1月5日，[[東方籃球隊|東方]]宣佈與艾利諾續約到2018年球季，而他於2018年居港滿一年後，取代[[基斯·班尼斯|班尼斯]]的居港球員身分註冊出戰香港賽事，<ref>[http://hk.on.cc/hk/bkn/cnt/sport/20170423/bkn-20170423183411712-0423_00882_001.html 東南亞職籃　盡掃個人獎！東方龍獅封王] on.cc東網 2017年4月23日</ref>而他更在1月24日對[[印尼CLS騎士籃球隊|印尼騎士]]後，再度打破紀錄成為[[ABL|ABL]]史上首位連續三場得[[三雙|三雙]]記錄的球員。其後他在其生涯第43場[[ABL|ABL]]賽事，3月15日，對[[莫諾吸血鬼籃球隊|曼谷吸血鬼]]時錄得在[[ABL|ABL]]的1000分成為最快完成創舉的球員，5月8日對[[滿貫籃球隊|滿貫]]時，艾利諾首度在[[香港男子甲一組籃球聯賽|甲一組聯賽]]亮相，即錄得全場最高的31分，協助[[東方籃球隊|東方]]以79–67勝出。艾利諾協助東方於5場3勝制的總決賽，以場數3–2擊敗南華，奪得冠軍，而他則榮膺最有價值球員。

== 球場以外 ==
由於在[[加利福尼亞州|加利福尼亞州]]出生，艾利諾兒時擁護的球隊是[[洛杉磯湖人|洛杉磯湖人]]，由於其父母曾在[[西雅圖|西雅圖]]讀書，使艾利諾亦一度是[[西雅圖超音速|西雅圖超音速]]的粉絲並視[[蓋瑞·裴頓|披頓]]為偶像，至於他歷來最喜歡的球員便是戰神[[艾倫·艾佛森|艾佛遜]]，艾利諾最欣賞的本地球員是有被譽為「[[香港|香港]][[浪花兄弟_(NBA)|浪花兄弟]]」的[[陳兆榮|陳兆榮]]和[[李琪_(籃球運動員)|李琪]]及[[劉子禮|劉子禮]]。<ref>[https://www.am730.com.hk/news/%E6%96%B0%E8%81%9E/%E3%80%90am%E5%B0%88%E8%A8%AA%E3%80%91%E9%A0%98%E9%A6%99%E6%B8%AF%E7%90%83%E9%9A%8A%E7%99%BBabl%E6%A6%9C%E9%A6%96-%E6%B5%AA%E4%BA%BA%E8%89%BE%E5%88%A9%E8%AB%BE%EF%BC%9A%E6%88%91%E5%B0%B1%E6%98%AF%E6%83%B3%E6%89%93%E7%B1%83%E7%90%83%EF%BC%81-68449 【am專訪】領香港球隊登ABL榜首 浪人艾利諾：我就是想打籃球！] [[am730|am730]] 2017-03-06</ref>
