'''馬蒂亞斯·雅各布·許萊登'''（{{lang|de|'''Matthias Jakob Schleiden'''}}，{{bd|1804年|4月5日|1881年|6月23日|S}}）是一位[[德國|德國]][[植物學家|植物學家]]，[[細胞學說|細胞學說]]的建立者之一。他出生於德國[[漢堡|漢堡]]，大學時原本研讀法律，但後來轉向其興趣[[植物學|植物學]]。

他喜歡使用[[顯微鏡|顯微鏡]]來觀察植物的結構，他在擔任[[耶拿大學|耶拿大學]]教授時，紀錄了植物不同部位是由細胞所構成的現象。後來他也辨識出1831年時由植物學家[[羅伯特·布朗|羅伯特·布朗]]所發現的[[細胞核|細胞核]]<ref>{{cite web en|author = Trisha Creekmore|title = The Science Channel :: 100 Greatest Discoveries: Biology|publisher = [[探索通信|探索通信]]|url = http://science.discovery.com/convergence/100discoveries/big100/biology.html|accessdate = 2006-10-17|deadurl = yes|archiveurl = https://web.archive.org/web/20061024155730/http://science.discovery.com/convergence/100discoveries/big100/biology.html|archivedate = 2006-10-24}}</ref>。
