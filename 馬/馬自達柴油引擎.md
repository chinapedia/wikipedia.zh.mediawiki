=== RF 2002型 ===
這具排氣量2.0L的'''RF 2002型'''採用比較先進的高壓共軌缸內直噴技術（common rail direct-injection），亦稱作'''MZR-CD型'''。由於配備了可變幾何渦輪噴油嘴（variable turbine geometry）和[[EVVC膨脹縱旋渦燃燒|EVVC膨脹縱旋渦燃燒]]技術（expansive vertical vortex combustion），使得燃油效率與[[氮氧化物|氮氧化物]]排出量獲得改善。依動力傳動系統的差異，輸出動力可分成兩種版本：121ps / 3,500rpm和136ps / 3,500rpm，而最大扭力皆為310N·m / 2,000rpm、壓縮比皆是18.3:1。

這具引擎具有雙質量[[飛輪|飛輪]]（dual-mass flywheel，用來取代傳統自排變速箱的扭矩轉換器，將傳統飛輪拆分成兩片組合式鐵盤以內鐵和螺絲組合，再用環狀齒輪予以接合；一邊連接離合器，另一邊連接傳動軸。兩片飛輪之間置放減震彈簧，以達到傳動時的平穩效果）及日本[[電裝|Denso]]製造、最大壓力1,800bar的高壓共軌噴射裝置。

=== RF 2005型 ===
為了達到[[歐洲汽車尾氣排放標準|歐盟第四期汽車廢氣排放標準]]（European emission standards stage IV），2005年此具亦稱作'''MZR-CD型'''或'''RF-CDT型'''的柴油引擎隨著[[馬自達6|馬自達6]]小改款而改善下述：
* 石川島播磨重工業的RHF5 VJ33型可變幾何渦輪增壓器（variable geometry turbocharger）減少了14%的[[轉動慣量|慣性矩]]，且設計得更緊湊，譬如渦輪軸和廢氣入口之間的距離更短。
* 日本[[電裝|Denso]]開發的第二代共軌缸內直噴系統，壓力高達1,800bar，噴油嘴有六孔。在不同的行駛狀態下，可採用多段式燃油噴射，甚至每個衝程循環可多達九次噴射。
* 相對較低的壓縮比16.7:1改善了馬力輸出與扭矩，同時降低燃燒噪音與廢氣排放。
* 降低引擎燃燒衝程的溫度以減少[[氮氧化物|氮氧化物]]排出。
* 含氧感知器會不斷偵測廢棄的含氧量並傳送至32[[位元|位元]]的ECU行車電腦，以優化燃油和空氣的混合比例。

此具引擎分成兩種馬力版本：標準馬力版最大馬力121ps / 3,500rpm、最大扭力320N·m / 2,000rpm；高馬力版最大馬力143ps / 3,500rpm、最大扭力360N·m / 2,000rpm。兩種版本都裝置了[[柴油碳微粒濾清器|柴油碳微粒濾清器]]，以符合環保法規。因此，該具引擎排放的黑碳顆粒比歐盟第四期汽車廢氣排放標準規定的還要少80%。車型：
# 2005年7月以後出廠的[[馬自達6#第一代（2002年-2008年）|第一代馬自達6]]
# 2005年-2010年：[[馬自達5#第二代（CREW/CR3W型_2005年-2010年）|第二代馬自達5]]
# 2007年：[[馬自達3#第一代（2003年-2009年）|第一代馬自達3]]小改款

=== R2 2008型 ===
也稱為'''MZR-CD型'''或者'''R2-CDT型'''但排氣量2,184c.c.的'''R2 2008型'''渦輪增壓柴油引擎乃自RF型擴缸而來，缸徑保持86mm，但衝程比RF型多了8mm；且重心較低的鋁合金[[汽缸本體|汽缸本體]]結合了低摩擦係數、前鏈條驅動的平衡軸。其共軌缸內直噴系統係日本[[電裝|Denso]]製造，噴射壓力達2,000bar（200MPa）。噴油嘴上佈滿10個孔徑0.119mm的噴孔與高感度電磁閥，且因16.3:1的壓縮比達到高度霧化的噴射效果。可變噴嘴渦輪增壓器（variable nozzle turbocharger，縮寫成VNT）具有彎曲葉片和耐磨密封（abradable seal）；而雙頂置凸輪軸（DOHC）氣門系統由鏈條驅動。另外，新型柴油碳微粒濾清器更耐熱、新型[[排氣再循環|EGR]]（Exhaust Gas Recirculation的縮寫）冷卻器更有效率，達到歐盟第五期汽車廢氣排放標準。

更甚者，此具引擎採用最新「選擇性觸媒還原脫硝技術」（{{en}}Selective Catalytic Reduction，簡稱SCR），以[[AdBlue|AdBlue]]車用尿素溶液透過選擇性觸媒催化還原作用，把廢氣中的氮氧化物還原成[[氮|氮]]及[[氧|氧]]，藉以減少空氣污染。不過目前馬自達僅自2009年尾起於歐規版[[馬自達CX-7|馬自達CX-7]]搭載這項裝置，將15.5公升的容器安裝於後行李廂下方，在正常的行駛條件下可使用約2萬公里<ref>參看{{ja}}[http://www.mazda.co.jp/corporate/publicity/release/2009/200903/090316a.html 【MAZDA】マツダ、日本メーカー初の乗用車用尿素SCRシステムを開発]。</ref>。該引擎依馬力大小可區分成下列五種：
* 高馬力型：最大馬力185ps / 3,500rpm、最大扭力400N·m / 1,800-3,000rpm。
* 中馬力型：最大馬力163ps / 3,500rpm、最大扭力360N·m / 1,800-3,000rpm。
* 低馬力型：最大馬力125ps / 3,500rpm、最大扭力310N·m / 1,800-2,600rpm。
（前述三者搭載於[[馬自達6#第二代（2008年迄今）|第二代馬自達6]]）
* 高馬力型：最大馬力185ps / 3,500rpm、最大扭力400N·m / 1,800-3,000rpm。
* 標準馬力型：最大馬力150ps / 3,500rpm、最大扭力360N·m / 1,800-2,600rpm。
（前述二者搭載於[[馬自達3#第二代（2009年-2013年）|第二代馬自達3]]）

=== SH-VPTS型 ===
[[File:MAZDA_SH-VPTS_DIESEL_1.JPG|thumb]]
'''另參見：[[創馳藍天技術#柴油引擎|創馳藍天技術]]'''<br>
該具排氣量2,188c.c.、直列四缸高壓共軌缸內直噴的引擎亦稱作'''SKYACTIV-D型'''，採用馬自達力推的「[[創馳藍天技術|創馳藍天技術]]（{{en}}SKYACTIV Technology）」，包括下列特點<ref>參看{{en}}[http://www.mazda.com/publicity/release/2010/201010/101020a.html 【MAZDA】Mazda Reveals Next-Generation 'SKYACTIV' Technologies]。</ref>：
* 創下全球最低的[[柴油引擎|柴油引擎]]壓縮比14.0:1，可達到提高20%的燃油使用效率。
* 二階段式開啟渦輪增壓系統（也就是雙渦輪增壓設計），以提升引擎扭力表現與動力輸出平順度。
* 符合歐盟第六期汽車廢氣排放標準、北美Tier2 Bin5廢氣排放規定與日本新長期廢氣排放標準（{{ja}}平成22年排出ガス規制，一般通稱為「ポスト新長期規制」）。該具引擎之衝程為94.2mm、缸徑86.0mm、壓縮比14.0:1，最大馬力175ps / 4,500rpm，扭力峰值42.8kg·m / 2,000rpm。

該型SKYACTIV-D 2.2柴油引擎經多數使用者質疑本身有設計缺陷，經車友組成自救會 "Mazda CX-5 SOS Group Taiwan/自救會/セルフヘルプ"、[http://carinf.mlit.go.jp/jidosha/carinf/opn/search.html?selCarTp=1&lstCarNo=1410&txtFrDat=1000%2F01%2F01&txtToDat=9999%2F12%2F31&txtNamNm=%20&txtMdlNm=%20&txtEgmNm=SH&chkDevCd=7 "日本國土交通省不具合情報"]及澳洲及泰國皆有多起回報因SKYACTIV-D 2.2引擎缺陷導致冷卻系統高溫及冷卻水噴出，導致動力限制造成行車危險，台灣馬自達於2017年8月30日經車友發動大規模圍廠後台灣馬自達發出[https://www.mazda.com.tw/news/offers/20170830-1/ 聲明]表示開始免費檢修缺陷引擎及調查原因，迄今尚未公布調查結果。

2017年12月推出的CX-8搭載新改良的SH-VPTS型柴油引擎，採用類似[[富豪汽車|富豪汽車]]「i-ART系統」技術，馬自達稱之為「急速多段燃燒」<ref>{{ja}}[https://motor-fan.jp/tech/10001986 MotorFan: マツダCX-8 デンソーi-ARTインジェクターと急速多段燃焼で洗練度を上げたSKYACTIV-D2.2]。</ref>，在四支壓電晶體式噴油嘴（piezo injector）上安裝可偵測燃油噴射量及壓力的[[感測器|感測器]]。因此最大馬力調整至190ps / 4,500rpm，最大扭力45.9kgf·m / 2,000rpm。

車型：
