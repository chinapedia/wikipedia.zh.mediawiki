'''馬自達MAZDA號'''（{{lang-ja|マツダ·マツダ号}}）是一輛由日本東洋工業（現稱[[馬自達|馬自達]]）於1930至1950年代期間製造、委託[[三菱商事|三菱商事]]代為銷售的三輪[[載貨汽車|卡車]]。該款三輪載貨卡車為馬自達跨入汽車製造行業的濫觴，在[[馬自達博物館|馬自達博物館]]門口便置放了一輛展示品。

== 概要 ==
1931年10月，原本製造[[機床|工作機械]]的東洋工業跨入機動車輛的領域，推出命名為「'''MAZDA號'''」（DA型）的三輪貨車。關於車名的由來，以當時的[[董事長|社長]][[松田重次郎|松田重次郎]]的姓氏，經[[波斯|古波斯]][[瑣羅亞斯德教|拜火教]]光明之神[[阿胡拉·馬茲達|阿胡拉·馬茲達]]（{{lang|fa|Ahura Mazdā}}）之名變體的「'''MAZDA'''」。由於當時的銷售業務委託三菱商事代為處理，故油箱上除了「MAZDA」字樣外，尚印有三菱的紅色[[商標|商標]]。銷售區域除日本境內，尚包括[[中國|中國]]、[[滿州國|滿州]]、[[巴西|巴西]]等地。

為了打響MAZDA號三輪車的名號，該公司甚至於1936年（[[昭和|昭和]]11年）發起一項名為「鹿兒島至東京[[商隊|商隊]]宣傳」（{{lang|ja|鹿児島―東京間キャラバン宣伝}}）的[[廣告|廣告]]行銷活動，派出4輛KC型和1輛DC型三輪車，從[[鹿兒島|鹿兒島]]出發前往[[東京|東京]]，花了25天的時間行走約2,700公里<ref>參看{{ja}}[http://seez.weblogs.jp/car/cat3843142/page/7/ <span lang="ja">旧きをたずねて新しきを知る！　重次郎とマツダヒストリー　第６回</span>]之段落。</ref>，趁機宣傳產品耐用度。

東洋工業的三輪貨車產品在1961年時達到最高峰，當年度總計生產了45,685輛，[[市場佔有率|市場佔有率]]超過80%以上<ref>請見{{ja}}[http://seez.weblogs.jp/car/cat3843142/page/6/ <span lang="ja">旧きをたずねて新しきを知る！　重次郎とマツダヒストリー　第１１回</span>]之段落</ref>。

== 型式 ==
[[File:Mazda_Type_DC.JPG|thumb]]
[[File:1953_Mazda_CT_01.jpg|thumb]]
MAZDA號歷經好幾代的變革，車型計有：
# '''DA型'''：1931年10月上市，搭載482c.c.氣冷式側氣門單汽缸引擎，缸徑85.5mm、衝程84mm、最大馬力9.4ps / 3,300rpm，搭配三速手動變檔設計<ref>參考{{en}}[http://www.jsae.or.jp/autotech/data_e/3-4e.html Society of Automotive Engineers of Japan: Mazda DA Three-wheeled Truck] {{Wayback|url=http://www.jsae.or.jp/autotech/data_e/3-4e.html |date=20130127033515 }}一文。</ref>，後車斗的最大載貨容積為200kg。
# '''DB型'''：1932年推出，採用雙重車殼設計。
# '''DC型'''：1934年上市，動力改為485c.c.引擎，最大載貨容積400kg。
# '''KA型'''：1934年10月推出，但改為強化過的單層車殼，改成[[排氣量|排氣量]]654c.c.的引擎，且引擎與變速箱之結構整合成一體。
# '''TCS型'''：1935年登場，採雙重車殼、654c.c.引擎之設計，制動系統有手動、腳踏板兩種，當年售出了424輛<ref>參看{{ja}}[http://www.asahi-net.or.jp/~ip3t-nksn/Pages/B-classic/Qrp826.htm マツダTCS型]第二段。</ref>。
# '''KC型'''：也在1935年上市，在其2年8個月的生產周期裡，共計出售了5,595輛<ref>參看{{ja}}[http://seez.weblogs.jp/car/cat3843142/page/7/ 旧きをたずねて新しきを知る！　重次郎とマツダヒストリー　第６回]之段落。</ref>。
# '''GA型'''：1938年4月推出，以取代舊式K型系列。引擎排氣量擴大到669c.c.，變速系統也從三速進化至四速，最大載貨容積500kg。
# '''GB型'''：[[第二次世界大戰|第二次世界大戰]]期間，東洋工業被徵召生產軍需品，沒有多餘的產能可以繼續製造三輪卡車。戰後為了國內重建的市場需求，該公司恢復製造，重新改良GA型而在1949年推出GB型。該款三輪卡車搭載新開發的701c.c.單汽缸SV型引擎，採全鋁合金鑄造，最大馬力可輸出15.2hp。變速箱系統採低壓鑄造法（low pressure die casting）製造，與引擎整合成一體。最大載貨容積雖然維持500kg不變，但車斗尺寸改變，因輪胎而拱起的空間也修改得更平坦以利載貨。
# '''PB型'''：二戰結束後，各日本機動車輛製造商積極嘗試從載貨車進化至乘用車，而業界中仍持續發展三輪車型的二大勢力應屬[[大發工業株式會社|大發工業]]與東洋工業。大發工業於1951年10月推出三輪車型的雙門乘用轎車{{link-ja|大發Bee|ダイハツ・Bee}}，反觀東洋工業則於1950年發售PB型三輪卡車。該型車將後半部以手工打造的車殼包覆起來，後來車殼改以油壓沖床製造。全車含駕駛共可容納六人乘坐，相對而言車重也比GB型大約多了250kg。引擎一樣採701c.c.單汽缸SV型引擎。該型車至1952年停產為止，共發售了PB-1、PB-2至PB-5等共六款，總計生產了690輛<ref>參看{{ja}}[http://www.asahi-net.or.jp/~ip3t-nksn/Pages/B-classic/Qrp810.htm マツダPB型]第二段。</ref>。其實這款車是因為東洋工業所在的[[廣島市|廣島市]]，因戰爭結束後為了載客運輸需求而開發，作為「90日圓均一價[[出租車|計程車]]」之用。此外在東京與[[大阪|大阪]]市區內也有PB型三輪計程車，被市民暱稱為「{{lang|ja|輪タク}}」（汽車taxi之意）。在全盛時期的1951年（昭和26年），兩地的PB型計程車加起來超過1千輛，直到翌年生產了690輛後而停產<ref>參看{{ja}}[http://seez.weblogs.jp/car/cat3843142/page/6/ 旧きをたずねて新しきを知る！　重次郎とマツダヒストリー　第１０回]之段落。</ref>。
# '''LB型'''：1950年3月發售，車斗特別應市場需求而加大，可說是GB型的放大版本（比GB型長70cm），載物容積達750kg。
# '''VB型'''：1950年4月推出，以LB型改良發展的三輪廂型車。
# '''CA型'''：1950年6月推出，這是馬自達第一款小型「四輪」貨車，以三輪車為基礎改良而成。
# '''CT型'''：1950年9月上市，動力為日本首創採OHV頂置氣門技術的1,157c.c.氣冷式V型二缸引擎，可輸出32ps，且配有油壓式氣門間隙自動調整裝置，有效抑制引擎震動與噪音。此型車除了駕駛者膝蓋部位前覆有防護罩，車頭上也有玻璃擋風罩，當時常被改裝成[[消防車|消防車]]（稱為CF型）、清潔垃圾車、油罐車等特殊車輛使用。1951年追加車斗加長車型。
# '''HB型'''：1951年3月上市，以LB型為基礎而改良。
# '''CTL型'''：1951年發售，駕駛座上方覆蓋著可遮風避雨的斗篷，車斗加長至3公尺。翌年推出最大載貨容積2公噸的車型，為業界首創。
# '''CTD型'''：1952年3月發售，屬於三輪[[傾卸卡車|傾卸卡車]]（dump truck）。
# '''CTA型'''：1953年推出，以CT型為基礎而改良；同年12月改款，將頭燈改為雙圓燈。
# '''CHTA型'''：和前者於同年推出，設計者為[[小杉二郎|小杉二郎]]。
# '''GCZ型'''：1954年2月推出，最大載貨容積750kg。
# '''GDZA型'''：1955年上市，底盤承襲自GCZ型，動力為701c.c. OHV頂置氣門式引擎，最大馬力17ps。頭燈為雙燈式淚滴造型，且首度在車頭防護罩裝上方向指示燈。
# '''GLTB型'''：1956年推出，為GDZA型的後繼車款。
# '''MRA型'''：1957年11月上市，此款車已屬馬自達製造三輪卡車的末期產品。
# '''MRB型'''：自MRA型改良而來。
# '''HBR型'''：1957年推出，設計者為小杉二郎。
