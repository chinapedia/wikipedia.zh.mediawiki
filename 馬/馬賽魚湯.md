'''馬賽魚湯'''（法語：'''bouillabaisse'''，{{IPA-fr|bu.ja.bɛːs}}；{{lang-oc|bolhabaissa}}），法国美食的至尊料理之一，是來自[[法國|法國]][[地中海|地中海]]沿岸的複雜魚湯，也是[[普羅旺斯-阿爾卑斯-藍色海岸|普羅旺斯]]美食的光榮，做法隨地方而變，但通常公認[[馬賽|馬賽]]地區的配方暨烹調方式都屬地道。

馬賽魚湯也被稱為世界三大湯頭之一，另外兩個是[[中國|中國]][[粵菜|粵菜]]的[[上湯|上湯]]和[[泰國|泰國]]的[[冬陰湯|冬陰湯]]。<ref name="jiten2007_p650">{{harvnb|日仏料理協会|2007|p=650}}</ref>

== 详情 ==
名字“bouillabaisse”來自動詞「烹煮」'''bouillir''' 和「降低、減少」'''abaisser'''。除了種類混雜的[[魚|魚]]隻（通常不含[[貝|貝]]類、[[蝦|蝦]]或[[龍蝦|龍蝦]]）外，還必須加入[[橄欖油|橄欖油]]、[[洋蔥|洋蔥]]、[[番茄|番茄]]、[[大蒜|大蒜]]、[[西洋芹|西洋芹]]、[[茴香|茴香]]、[[百里香|百里香]]、[[蔥|蔥]]、[[桂葉|桂葉]]、[[番紅花|番紅花]]和[[柳橙|柳橙]]皮。真正的馬賽魚湯必須使用地中海的魚產，包括基本的[[牙鱈|牙鱈]]、[[海鰻|海鰻]]、[[鯡魚|鯡魚]]、[[赤鮋屬|赤鮋]]<ref>{{cite web |url=http://www.wretch.cc/blog/dionysus1101/9809545 |author=[http://www.wretch.cc/blog/dionysus1101/ Alfred Lee] |title=當馬賽魚湯裡不再有赤鮋 |date=2012-05-19 |accessdate=2012-01-04 |language=zh-hant }}</ref>等魚種，也含有多種其他魚類，同時所有的材料都要快速的一起烹煮。上桌時，一種稱為 rouille 的佐[[醬|醬]]、[[烤|烤]]過或[[煎|煎]]過的[[麵包|麵包]]丁或是碎麵包，還有烹煮後分離的魚肉，會連同魚湯的[[湯|湯]]體分開呈裝，作為佐餐或配料以增香添色，其本質與[[葡萄牙|葡萄牙]]的[[葡式海鮮湯|葡式海鮮湯]]（Caldeirada）有不少相似處。由於單純從外觀而論或許不怎麼吸引人，為了吸引觀光客，或是那些不了解地道的馬賽魚湯的人，有時[[龍蝦|龍蝦]]或[[螃蟹|螃蟹]]也會隨著這些魚種一同成為盤中飧，可能還包含[[淡菜|淡菜]]等貝類作為號召，這就成為巴黎版本的菜色，這種迎合式的美食變體，卻不是道地的馬賽魚湯。

== 文學作品的看法 ==
馬賽魚湯不僅已列入國際美食之列，在文壇也佔有一席之地，無論是[[韻文|韻文]]或[[散文|散文]]，[[萨克雷|萨克雷]]便曾在詩句中大肆讚揚在[[巴黎|巴黎]]享受馬賽魚湯的快感。[[英國|英國]]作家[[喬安·凱瑟琳·羅琳|喬安·凱瑟琳·羅琳]]在長篇[[小說|小說]][[哈利波特|哈利波特]]第四冊十六章提及馬賽魚湯也是[[霍格華茲|霍格華茲]][[魔法|魔法]]學院用來招待，這道佳餚在該作品中似乎使一位來自[[法國|法國]]的魔法師感到非常滿意。

不過華人作家[[余秋雨|余秋雨]]卻曾在其著作《行者無疆》中對馬賽魚湯作出較負面的描述：
<blockquote>「魚湯上得很快，先是一桌一大海碗，由服務員一勺勺分到每人的淺盆上。湯呈渾褐色，趁著熱氣先進一口，便立即皺起了眉。不能說難吃，但又腥又鹹，是一種平庸的口味，以前在海邊一些貧困的農家可以喝到。我喜歡吃魚，不怕腥，但對這種完全不作調理的腥，還是不敢恭維。
<br><br>
第二道是正菜，其實與第一道湯出於同鍋，只不過把熬湯的實物盛起來罷了。樣子不錯，紅色的是[[小龍蝦|小龍蝦]]，黑色的是[[蛤蜊|蛤蜊]]殼，白色的是魚肉，三兩塊黃色是[[馬鈴薯|土豆]]，與湯合成一盆，一人一份。先喝一口湯，與頭道湯完全相同，於是吃實物。小龍蝦肉要剔出來十分費事，終於剔出，小小一條，兩口咽下，不覺鮮美。然後吃魚，一上口才發現又老又柴，原來這些水產在一個大鍋裡不知熬了多久，魚怎麼禁得起這樣熬呢？只得嘆一口氣，挾一塊土豆，揪半片麵包入口，算是用完了馬賽魚湯。」</blockquote>
