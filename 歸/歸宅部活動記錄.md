{{日本作品|歸宅部活動記錄|帰宅部活動記録}}（或譯名《回家社活動日誌》）是{{lang|ja|[[黑羽|くろは]]}}創作的[[日本|日本]][[漫畫|漫畫]]作品。

== 簡介 ==
在經過[[史克威爾艾尼克斯|史克威爾艾尼克斯]]的「漫畫選集 歸宅部（<small>{{lang|ja|コミックアンソロジー極 帰宅部}}</small>）」刊載後，同社的[[網路漫畫|網路漫畫]]配信網站「[[GANGAN_ONLINE|GANGAN ONLINE]]」從2011年8月18日開始每月第2、4週更新連載。作者的另一部作品《{{lang|ja|マジカルロリポップ}}》也開始連載。

動畫方面，2013年7月至10月播放同名[[電視動畫|電視動畫]]<ref name=natalie>{{Cite web|date=2013-05-09|title=動畫「歸宅部活動記錄」製作為NOMAD，片頭曲由乙女新黨演唱。|url=http://natalie.mu/comic/news/90268|work=[[Natalie_(網站)|Comic Natalie]]|publisher=Natasha|accessdate=2013-05-09|language=ja}}</ref>。

== 故事簡介 ==
剛成為高中生的安藤夏希，在被朋友古橋愛問及要加入哪個社團時，隨意說出要加入「[[歸宅部|歸宅部]]」。而這時同班同學塔野花梨向夏希提出了要參觀「歸宅部」的請求。夏希沒想到「歸宅部」這個社團真的存在，並加入了這個社團……。

== 登場角色 ==
=== 歸宅部 ===
; {{nihongo|安藤夏希|安藤 夏希（あんどう なつき）|聲：[[木戶衣吹|木戶衣吹]]}}
: 1年級生。受花梨邀請參觀歸宅部。其「我可以回家了嗎」的發言，讓社員稱讚她為「相當具有歸宅精神的孩子」，最後加入歸宅部。
: 在歸宅部負責[[吐槽|吐槽]]，頭頂有不能剪的[[呆毛|呆毛]]（一剪掉，旁邊的頭髮會自動補過來，不論幾次都一樣，連只用[[修圖軟體|修圖軟體]]去掉都會讓電腦[[當機|當機]]）。有時候會和櫻做出相同的反應。動畫第4集後的小劇場表示，她的名字源自[[甜甜圈|豆餡甜甜圈]]（日文發音很像）。

; {{nihongo|塔野花梨|塔野 花梨（とうの かりん）|聲：[[結名美月|結名美月]]}}
: 1年級生。邀請夏希參觀歸宅部。
: 覺得歸宅部社員在招募社員時的舉止很有趣，而想加入歸宅部，實際並不清楚歸宅部的活動內容。
: 個性相當純真，相當[[天然呆|天然呆]]，常擺出如[[小動物|小動物]]般令人無法抗拒的可愛表情和反應，被歸宅部所有成員視為可愛到會讓人想引誘犯罪的存在。受克蕾雅寵愛，甚至要一輩子養她都願意。擅長製作點心、縫紉手工藝，加上溫柔純真的性格，女子力超高（被櫻以女子力檢測力測出數值高達五萬，而且還不斷飆升至爆表）。

; {{nihongo|道明寺櫻|道明寺 桜（どうみょうじ さくら）|聲：[[小林美晴|小林美晴]]}}
: 2年級生，歸宅部社長，牡丹的第一個朋友。做事不經大腦，常一時興起搞出些莫名其妙的活動，也是歸宅部最主要的麻煩製造者。被學生會長稱為{{nihongo|{{ruby|擾亂手榴彈|Grenade Rager}}|グレネード乱痴気}}。
: 血型：A型、身高：163cm、星座：牡羊座。

; {{nihongo|大萩牡丹|大萩 牡丹（おおはぎ ぼたん）|聲：[[相內沙英|相內沙英]]}}
: 2年級生，古武術「萩月流」第十六代繼承者。擅長不少格鬥技和武術，連用[[手刀|手刀]]隔空切開[[寶特瓶|寶特瓶]]都會，戰鬥實力能擊倒各種熊（包含[[北海道|北海道]]的[[日本棕熊|日本棕熊]]、[[阿拉斯加灰熊|阿拉斯加灰熊]]和[[北極熊|北極熊]]都是她的手下敗將）。身心鍛鍊到自稱不論受任何的攻擊都能毫髮無傷（櫻親自嘗試後證實連鐵椅砸都無效，不過不知為何好像有[[受虐癖|受虐癖]]似的打越狠越高興），卻敵不過花梨以柔克剛的精神攻擊。被學生會長稱為{{nihongo|{{ruby|漆黑處刑台|Midnight Murder}}|ミッドナイトマーダー}}。

; {{nihongo|九重克蕾雅 (九重久禮亞) |九重 クレア（ここのえ クレア）|聲：[[千本木彩花|千本木彩花]]}}
: 2年級生，九重財閥社長的女兒，金錢觀和一般人不同。花梨入部後受其可愛的性格影響而非常寵愛她，喜歡到要把她帶回家養一輩子都沒問題，全身上下的衣服都是仿造校服外貌特別訂製的，不論是制服還是運動服，因此從沒穿過真的校服。動畫第6集後的小劇場表示，她的名字源自[[泡芙|法式泡芙]]（Eclair），如果念「這裡的法式泡芙」的日文讀音幾乎一樣。

=== 萩調流四天王 ===
; {{nihongo|殘酷的青龍|残酷の青龍|聲：[[上村祐翔|上村祐翔]]}}

; {{nihongo|殘忍的白虎|残忍の白虎|聲：[[小沼英希|小沼英希]]}}

; {{nihongo|殘虐的朱雀|残虐の朱雀|聲：[[伊藤美来|伊藤美来]]}}
: 萬綠叢中一點紅，四天王唯一的女性。負責偵查敵對流派首領牡丹，對常受到寵愛的花梨不滿。本名叫{{nihongo|「詩天使」|しえる}}。

; {{nihongo|殘念的玄武|残念の玄武|聲：[[齋藤夢人|齋藤夢人]]}}
: [[動漫迷|動漫迷]]，外號是四人中最不中聽的（曾試過更換外號，但反而更糟），是四天王中負責吐槽的角色。

; {{nihongo|高圓禮奈／高圓蕾娜|高円 レイナ（たかまど －）|聲：[[三澤紗千香|三澤紗千香]]}}
: 學生會會長，頭髮黑長直的美少女，有「KOUSOKU<ref group="注">應該是指'''校規'''，但也可能是指'''光速'''、'''高速'''或'''束缚'''，因為這些詞讀音都一樣。</ref>」這個外號。領域型的性格，凡在她周遭一定範圍裡必無法違反校規，以「高速」地用「校規」「束缚」一切而獲得那樣的稱號，並被稱為{{nihongo|{{ruby|學生手銬|Handbook Bind}}|ハンドブック·バインド}}[[超能力|能力者]]。

=== CG短篇動畫 ===

* 分鏡、演出：松浦有紗
* CG動畫：松浦有紗（4、7、9話）、蔦佳穗里（5話）、山口瞳美（6、8、10、11、12話）、寺田良介（11、12話）
* 背景：高田真理
* 人物造型：-{鎌}-田麻友美
* 製作管理：市川量也、坂元久留美
* 製作：[https://cgworld.jp/stdatabase/30540.html domerica]
