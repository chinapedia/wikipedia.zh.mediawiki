'''群星匯正音'''是自1995年3月4日至1995年4月1日由香港電台電視部播放的教育節目。由[[何文匯|何文匯]]博士和[[陳玉蓮|陳玉蓮]]主持。是第一部以粵語正音為題材的節目。

此節目透過娛樂與教育並重的戲劇形式，引起觀眾對粵語正音的關注；並且提供正確發音示範。幫助大眾消除一般粵語發音的毛病。

== 每集內容 ==

·第一集（1995年3月4日播放，編導：李慶華 / 徐志文）

[[何文匯|何文匯]]博士發起粵語正音大行動，設計了電腦過三關的正音測試來甄選粵語正音大使，為他宣揚正音的訊息。首先得到這榮譽的是[[何家勁|何家勁]]飾演的展昭大俠、[[張學友|張學友]]和[[梁朝偉|梁朝偉]]。[[陳玉蓮|陳玉蓮]]成爲何博士的助手後，推動正音不遺餘力，分別糾正了好友梅小惠和車房工人阿張（[[施介強|施介強]]飾）的粵語發音。而[[梁朝偉|梁朝偉]]駕車險些撞傷樓南光，在討論賠償時，帶出了一般人讀‘我(ngo5)’‘你(nei5)’等字的起音錯誤和‘八(baat8)’，百(baak8)’等字的收音錯誤。

·第二集（1995年3月11日播放，編導：李慶華 / 徐志文）

淘氣的[[郭富城|郭富城]]抵受不了雪糕的引誘，一本正經地過了三關而成為了正音大使。[[周慧敏|周慧敏]]和[[李克勤|李克勤]]則與何博士歌曲匯正音。正音大行動漸受關注，[[羅慧娟|羅慧娟]]飾演的銀行經理也邀請何博士想辦法糾正銀行職員在工作上常犯的錯誤發音。小侍應[[古巨基|古巨基]]遇上飛越時空而來的展大俠，討教了不少正音問題，更目睹他如何以不凡的身手和正確的發音從容應付歹徒。

·第三集（1995年3月18日播放，編導：李慶華 / 徐志文）

粵語正音大使[[羅文|羅文]]在接受[[楊吉璽|楊吉璽]]訪問時，道出自己的成功之道和正音要訣‘準確發音，增強信心！’。郭富城在公園與上小賊[[羅啟新|羅啟新]]，不但喚醒了他的良知，而且糾正了他的發音。在圖書館中，學生[[文頌嫻|文頌嫻]]正為學習粵語正音感到困惑之際，碰上了何博士，獲受粵音拼音法。而何博士在夢中見到字正腔圓的[[李嘉欣|李嘉欣]]，不待她過三關，便為她扣上正音大使襟章。就在這時，[[陳玉蓮|陳玉蓮]]突然出現，開展了一段神奇詭秘的劇情。

·第四集（1995年3月25日播放，編導：李慶華 / 徐志文）

餐廳中，[[李克勤|李克勤]]遇上滿口錯音的侍應[[黃一山|黃一山]]，忍不住給他上了正音一課。而[[黎明|黎明]]不愧為正音大使，出口成章，發音正確，與何博士惺惺相惜。在化妝間裏，發音毫不準確的賈公子（[[羅君左|羅君左]]飾）向[[周慧敏|周慧敏]]示愛，卻被教訓一頓。[[林憶蓮|林憶蓮]]在實驗室過三關時遇上停電，好事多磨；但堅毅的她最終也成為正音大使。何博士主持‘N計劃’講座，強調鼻音聲母[n-]的重要性，連說普通話的[[陳德容|陳德容]]也感到莫大興趣，請他和下午茶，向他討教一番。

·第五集（1995年4月1日播放，編導：李慶華 / 徐志文）

癡情歌迷[[鄭敬基|鄭敬基]]向[[林憶蓮|林憶蓮]]示愛，雖然遭到拒絕，但卻學會了準確發音。[[鄭柏林|鄭柏林]]在[[黎明|黎明]]哥哥的指點下，分辨出‘橙(caang2)’和‘鏟(caan2)’收音的不同之處。[[洪楗華|洪楗華]]和[[鄭嘉穎|鄭嘉穎]]兩個籃球小子，不懂得‘男(naam4)’、‘藍(laam4)’和‘女(neoi5)’、‘呂(leoi5)’起音有別，常常因此而發生爭執，最後得到[[張學友|張學友]]解開疑團。[[江希文|江希文]]和[[梁漢文|梁漢文]]在卡拉OK因漠視[[羅文|羅文]]的正音大使身份而被[[羅文|羅文]]催眠做懲罰，卻因而糾正了錯誤的發音。而壓軸出場的正音大使正是何博士的得力助手[[陳玉蓮|陳玉蓮]]。

== 其他事項：日常錯讀字10選 ==

[[何文匯|何文匯]]博士在第五集介紹了10個日常錯讀字，粵語正音之外，還強調粵音正讀。

·‘'''愉'''快’的‘'''愉'''’應讀‘'''娛(jyu4)'''’，不應讀‘'''遇(jyu6)'''’。

·‘活'''躍'''’的‘'''躍'''’應讀‘'''藥(joek9)'''’，不應讀‘'''約(joek8)'''’。

·‘'''緋'''紅’的‘'''緋'''’應讀‘'''非(fei1)'''’，不應讀‘'''匪(fei2)'''’。

·‘'''綜'''合’的‘'''綜'''’應讀‘'''眾(zung3)'''’，不應讀‘'''宗(zung1)'''’。

·‘'''緩'''慢’的‘'''緩'''’應讀‘'''換(wun4)'''’，不應讀‘'''援(wun6)'''’。

·‘'''銘'''感’的‘'''銘'''’應讀‘'''名(ming4)'''’，不應讀‘'''茗(ming5)'''’。

·‘眼'''眸'''’的‘'''眸'''’應讀‘'''謀(mau4)'''’，不應讀‘'''茂(mau6)'''’。

·‘友'''誼'''’的‘'''誼'''’應讀‘'''義(ji6)'''’，不應讀‘'''宜(ji4)'''’。

·‘嬌'''媚'''’的‘'''媚'''’應讀‘'''味(mei6)'''’，不應讀‘'''眉(mei4)'''’。

·‘'''頒'''獎’的‘'''頒'''’應讀‘'''斑(baan1)'''’，不應讀‘'''攀(paan1)'''’。

特別是，‘友'''誼'''’的‘'''誼'''’和‘'''義'''’是同音同義字。
