《'''群青日和／遭難'''》（{{lang|ja|群青日和／遭難}}，''Ideal Days for Ultramarine / A Distress (First Run Limited Edition)''）是[[東京事變|東京事變]]的第一張[[黑膠唱片|黑膠唱片]]，跟第一張專輯《[[教育_(東京事變專輯)|教育]]》，一起於2004年11月25日發行。總計銷售額2,281張。

== 簡介 ==
本張專輯為東京事變首張黑膠唱片，採完全限定發行，購買者必須要在10月23日以前預約至唱片行預約，11月25日再來拿取唱片<ref>{{cite web |url=http://www.vibe-net.com/news/?news=0020545 |title=東京事変　完全生産限定アナログ盤をリリース |author=vibe-net.com |language=ja |deadurl=yes |archiveurl=https://web.archive.org/web/20160304115313/http://www.vibe-net.com/news/?news=0020545 |archivedate=2016-03-04 }}</ref>。而這張以25公分的黑膠盤收錄了單曲《群青日和》和《遭難》中的曲目，封面則是與首張專輯《教育》有異曲同工之妙，專輯《教育》是紅色的紙鶴、紅色的框，到了黑膠唱片整個就變成白色的複刻版，十分有趣。
