'''鈴木園子'''（日語：{{lang|ja|鈴木 園子}}/{{lang|ja|すずき そのこ}}，Suzuki Sonoko），是[[日本漫画|日本漫画]]和[[日本动画|动画]]作品《[[名侦探柯南|名侦探柯南]]》中的[[名偵探柯南角色列表|虚构角色]]。

作者[[青山剛昌|青山剛昌]]原本設定園子只在一個故事登場，不過因為「園子家非常有錢」的設定便於故事創作，於是改變主意。<ref group="參">{{Cite web|url=http://news.ltn.com.tw/news/life/breakingnews/1951398 |title=花輪才第6？網友票選10大令人印象深刻的動漫富二代 |work=自由時報電子報 |date=[2017年1月19日]}}</ref>

== 人物 ==
帝丹高中二年B班學生，是[[毛利蘭|毛利蘭]]和[[工藤新一|工藤新一]]同班同學，是小蘭的好姐妹，同時也是鈴木財團二千金。

她時常邀請毛利一家參加各式的宴會活動。當柯南因應[[毛利小五郎|毛利小五郎]]不在命案現場時，就會轉為麻醉她來破案<ref group="參">初見於漫畫第5卷，TV 35《山莊繃帶怪人之殺人事件（後篇）》</ref>，遂自稱「推理女王」。

總是向他人介紹小蘭是新一的老婆，並以小蘭的口氣調侃新一和小蘭之間的關係，常使小蘭感到害羞。有時也會因輕浮或誇張的舉止而令柯南暗自在心中吐槽或譏諷。

== 形象 ==
個性外向活潑、樂觀，待人真誠，做事直接、不拘小節，還略帶輕浮；雖出身名門，卻從不炫耀自己的千金身份；在學校是[[網球|網球]]社社員<ref group="參">TV 331-332《疑惑的辣味咖哩》、705《密室中的柯南》</ref>。

性格花痴，喜歡穿著較性感暴露的服裝來吸引男性目光（但造成京極真反感<ref group="參">漫畫第22卷，TV 154《園子的危險夏日物語（後篇）》</ref>），不過風采總是被小蘭搶走；憧憬的白馬王子是[[怪盜基德|怪盜基德]]，稱呼他「基德大人」，一直想要和他見面。

== 情感 ==
最要好的朋友是同班同學也是從小就認識的[[毛利蘭|毛利蘭]]。極為重視友情，曾對失去記憶的小蘭說：「就算你一輩子都無法恢復記憶，我也永遠是你的朋友」<ref group="參">M4《[[瞳孔中的暗殺者|瞳孔中的暗殺者]]》</ref>，也曾表示她的背後只願交給小蘭<ref>M11《[[紺碧之棺|紺碧之棺]]》，園子在得知女海盜'''[[安妮·伯妮|安·寶妮]]'''和'''[[瑪莉·瑞德|瑪麗·麗德]]'''是背對背奮戰後如此表示。</ref>。

男友是全國空手道冠軍——[[名偵探柯南角色列表|京極真]]，外號「'''蹴擊貴公子'''」。兩人穩定交往中，曾送過他[[杯子|杯子]]，但被拿來當[[花瓶|花瓶]]使用<ref group="參">漫畫第30-31卷，TV 228-229《暗藏殺機的陶藝教室》</ref>。收到京極真發來的視頻郵件，得知京極真也在接受網球訓練。後來京極真返回日本，但二人交往遭到園子父母反對。京極與園子的母親，園子與怪盜基德雙雙做出賭注，二人的戀情得到了發展和認同。
