'''鈴蘭'''（[[學名|學名]]：''{{lang|la|Convallaria majalis}}''，英文：Lily of the Valley，法文：Muguet de mai），也称'''山谷百合'''、'''風鈴草'''、'''君影草'''，是[[铃兰属|铃兰属]]中的唯一[[種|種]]，味甜，高毒性。原產[[北半球|北半球]][[溫帶|溫帶]]，[[歐洲|歐]]、[[亞洲|亞]]及[[北美洲|北美洲]]和[[中國|中國]]的[[中国东北地区|东北]]、[[華北|華北]]地區[[海拔|海拔]]850～2500處均有野生分佈。经典法语歌曲'''''le temps du muguet'''''即是指铃兰花。

== 形态 ==
鈴蘭是[[多年生草本植物|多年生草本植物]]，株高20－30cm。地下具横生而分枝的根状茎，茎端具肥大的地下芽。叶2－3枚，基部互抱成鞘状，卵圆形或窄卵圆形，具孤形叶脉，有光泽。偏向一侧的总状花序，高15－20cm，夏季开钟状白色花，小花约10朵，径约8mm，下垂并具芳香。果实为[[浆果|浆果]]，熟时红色，直径5－7毫米，其中包含数个较大的淡白或淡褐色种子，种子干燥，珠状，清澈半透明，宽1－3毫米。<ref name=keiskei>Ohara, Masashi; Araki, Kiwakoi; Yamada, Etsukoi; Kawano, Shoichi, Life-history monographs of Japanese plants, 6: ''Convallaria keiskei'' Miq. (Convallariaceae), ''Plant Species Biology'', Vol 21, No 2, August 2006, pp. 119–126(8), Blackwell Publishing</ref>

== 分类 ==
铃兰共分为3个变种，而部分植物学家认为这3个变种应该成为独立的物种<ref>{{en}}http://www.efloras.org/florataxon.aspx?flora_id=1&taxon_id=107908</ref>：

* ''Convallaria majalis'' [[栽培品种|var.]] ''keiskei''：分布于[[中国|中国]]和[[日本|日本]]，有红色的果实和[[铃|铃]]形的花朵。<ref name="Plantfinder">{{Cite book |title=RHS Plant Finder 2009–2010 |year=2009 |publisher=[[Dorling_Kindersley|Dorling Kindersley]] |location= |isbn=978-1-4053-4176-9 |pages=195, 196 }}</ref><ref name=keiskei/>
* ''Convallaria majalis'' var. ''majalis''：分布于[[欧亚大陆|欧亚大陆]]，花上有白色中脉。
* ''Convallaria majalis'' var. ''montana''：分布于[[美国|美国]]，花上有淡绿色中脉。
* ''Convallaria majalis'' var. ''rosea''：花朵为粉红色，不少学者承认这一变种。<ref name="Plantfinder"/>

== 象征 ==
*铃兰在法国是“拥有幸福”的象征。因其在每年的4月到5月间开放，所以在每年的5月1日，法国人有互赠铃兰互相祝愿一年幸福的习俗，获赠人通常将花挂在房间里保存全年，象征幸福永驻，[[比利时|比利时]]、[[荷兰|荷兰]]、[[瑞士|瑞士]]和[[安道尔|安道尔]]等也有同样的习俗。有人认为只有一株刚好13朵小花的铃兰才会带来好运。
*法国民间习俗“铃兰婚礼”上常常可以看到这种花，象征结婚13年。
*铃兰的花语为“幸福的回归”，也有一個較為具貶意的花語：“忌妒”。
*自1967年，其成为[[芬兰|芬兰]]国花<ref>{{cite web | url = http://www.ts.fi/teemat/luonto/50453/Kansalliskukka++on+myrkyllinen | title = Kansalliskukka on myrkyllinen | language = fi | publisher = Turun Sanomat | date= 2009-05-26 | accessdate = 2017-11-08 }}</ref>。

== 毒性 ==
[[File:Lillyvalleystriiped.jpg|thumb]]
[[File:C._m._cv._Plena.JPG|thumb]]
[[File:Convallaria_majalis_0003.JPG|thumb]]
鈴蘭含有[[强心苷|强心苷]]类有毒物質[[铃兰毒苷|铃兰毒苷]]（Convallatoxin）、[[铃兰苦苷|铃兰苦苷]]（Convallamarin）和[[铃兰糖苷|铃兰糖苷]]（Convalloside）等，全草有毒，花、根毒性比較強。
部分人服用其乾品或製劑，造成[[噁心|噁心]]、[[嘔吐|嘔吐]]、流涎、[[腹瀉|腹瀉]]等症狀，亦可能出現[[頭暈|頭暈]]、[[頭痛|頭痛]]、心律不整、[[心衰竭|心衰竭]]等病症。

鈴蘭全株及果实均是高[[毒性|毒性]]的<ref>http://health.nytimes.com/health/guides/poison/lily-of-the-valley/overview.html</ref><ref>{{cite web |url=http://museum.gov.ns.ca/poison/?section=species&id=79 |title=存档副本 |accessdate=2009-07-30 |deadurl=yes |archiveurl=https://web.archive.org/web/20081123041510/http://museum.gov.ns.ca/poison/?section=species&id=79 |archivedate=2008-11-23 }}</ref>，目前在铃兰中已发现大概38种不同的[[强心苷类|强心苷类]]毒素，包括：

* [[铃兰苷|铃兰苷]]（convallarin）
* [[铃兰苦苷|铃兰苦苷]]（convallamarin）
* [[铃兰毒苷|铃兰毒苷]]（convallatoxin）
* convallotoxoloside
* [[铃兰毒原苷|铃兰毒原苷]]（convalloside）
* [[新铃兰毒原苷|新铃兰毒原苷]]（neoconvalloside）
* [[葡萄糖铃兰毒原苷|葡萄糖铃兰毒原苷]]（glucoconvalloside）
* [[铃兰种苷|铃兰种苷]]（majaloside）
* convallatoxon
* [[铃兰总苷|铃兰总苷]]（corglycon）
* [[卡诺醇|卡诺醇]]-3-''O''-α-<small>L</small>-[[鼠李糖苷|鼠李糖苷]]（cannogenol-3-''O''-α-<small>L</small>-rhamnoside）
* 卡诺醇-3-''O''-β-<small>D</small>-[[甲基阿洛糖苷|甲基阿洛糖苷]]（cannogenol-3-''O''-β-<small>D</small>-allomethyloside）
* 卡诺醇-3-''O''-6-脱氧-β-<small>D</small>-[[阿洛糖苷|阿洛糖苷]]-β-<small>D</small>-[[葡萄糖苷|葡萄糖苷]]（cannogenol-3-''O''-6-deoxy-β-<small>D</small>-allosido-β-<small>D</small>-glucoside）
* 卡诺醇-3-''O''-6-脱氧-β-<small>D</small>-阿洛糖苷-α-<small>L</small>-鼠李糖苷（cannogenol-3-''O''-6-deoxy-β-<small>D</small>-allosido-α-<small>L</small>-rhamnoside）
* [[毒毛旋花子苷元|毒毛旋花子苷元]]-3-''O''-6-[[脱氧|脱氧]]-β-<small>D</small>-阿洛糖苷-α-<small>L</small>-鼠李糖苷（strophanthidin-3-''O''-6-deoxy-β-<small>D</small>-allosido-α-<small>L</small>-rhamnoside）
* 毒毛旋花子苷元-3-''O''-6-脱氧-β-<small>D</small>-阿洛糖苷-α-<small>L</small>-阿拉伯糖苷（strophanthidin-3-''O''-6-deoxy-β-<small>D</small>-allosido-α-<small>L</small>-arabinoside）
* 毒毛旋花子苷元-3-''O''-α-<small>L</small>-鼠李糖苷-2-β-D-葡萄糖苷（strophanthidin-3-''O''-α-<small>L</small>-rhamnosido-2-β-D-glucoside）
* [[沙门苷元|沙门苷元]]-3-''O''-6-脱氧-β-<small>D</small>-阿洛糖苷-α-<small>L</small>-鼠李糖苷（sarmentogenin-3-''O''-6-deoxy-β-<small>D</small>-allosido-α-<small>L</small>-rhamnoside）
* 沙门苷元-3-''O''-6-脱氧-β-<small>D</small>-[[古洛糖苷|古洛糖苷]]（sarmentogenin-3-''O''-6-deoxy-β-<small>D</small>-guloside）

* 19-羟基沙门苷元（19-hydroxy-sarmentogenin）

* locundioside

== 传说 ==
* 在森林守护神[[圣雷欧纳德|圣雷欧纳德]]死亡的土地上，开出了白色又具有香味的铃兰。铃兰绽放在那块冰凉的土地上，就是圣雷欧纳德的化身。
* 铃兰是[[聖母瑪利亞|聖母瑪利亞]]哀悼基督的眼淚變成的，把铃兰稱為「Our Lady's Tears」即聖母之淚，很多人也譯為女人的眼淚。
* 烏克兰有個美麗的傳說：很久以前有一位美麗的姑娘，痴心等待遠征的愛人，思念的淚水滴落在林間草地，變成芳馨四溢的铃兰。 
* 铃兰是[[白雪公主|白雪公主]]斷了的珍珠項鏈灑落的珠子，還有人說那是7個小矮人的小小燈籠。 
* 一個叫「琅得什」（俄文：{{lang|ru|Ландыш}}，意为铃兰）的少年，為了他的愛人「維絲娜」（俄文：{{lang|ru|Весна}}，意为春天）離他而去而傷心欲絕，少年的淚水變成了白色的花朵，而少年破碎的心流出的鮮血變成了鈴蘭豔紅的漿果。
