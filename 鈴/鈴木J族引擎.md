'''鈴木J族引擎'''是1990年代至2000年代之間[[日本|日本]][[鈴木公司|鈴木公司]]開發製造的[[往复式发动机|往復式]][[汽油引擎|汽油引擎]]，共通設計為[[直列四缸引擎|直列四缸]]、[[顶置凸轮轴#双顶置凸轮轴（DOHC）|DOHC]]、16汽門的結構。其中J18A型引擎有時記作「G18K型」，實際上兩者是相同的。

== J18/J18A型 ==
[[排氣量|排氣量]]1,839c.c.的'''J18/J18A型'''，其[[壓縮比|壓縮比]]9.8：1，缸徑84.0mm、行程83.0mm，最大[[馬力|馬力]]135ps / 6,500rpm、[[扭力|扭力]]峰值16.0kg·m / 3,000rpm。車型：
# 1995年-2002年：第三代鈴木Cultus Crescent Wagon、[[鈴木Esteem|鈴木Esteem]]（鈴木Cultus外銷版名稱）
# 1996年-2000年：第一、二代[[鈴木Escudo|鈴木Escudo]]

== J20/J20A型 ==
'''J20/J20A型'''的排氣量乃1,995c.c.，缸徑84.0mm、衝程90.0mm，壓縮比10.5：1，最大馬力145ps / 6,000rpm、最大扭力19.7kg·m / 4,000rpm。車型：
# 1997年-1999年：第二代[[馬自達Proceed_Levante|馬自達Proceed Levante]]
# 1997年-2008年：第二、三代鈴木Escudo
# 1999年-2004年：第二代{{link-en|雪佛蘭Tracker|Chevrolet Tracker}}
# 2001年-2007年：[[鈴木Liana|鈴木Liana]]
# 2006年迄今：第一、二代[[鈴木SX4|鈴木SX4]]

== J23A型 ==
排氣量2,290c.c.的'''J23A型'''引擎之缸徑90.0mm、衝程90.0mm，最大馬力156hp / 5,400rpm、最大扭力20.1kg·m / 3,000rpm。車型：
# 2001年-2007年：[[鈴木Liana|鈴木Liana]]

== J24B型 ==
[[File:Suzuki_J24B_engine.jpg|thumb]]的J24B型引擎]]
'''J24B型'''的排氣量是2,393c.c.，缸徑92.0mm、衝程90.0mm，壓縮比10.0：1，可輸出最大馬力178ps / 6,500rpm、扭力峰值170lb-ft / 4,000rpm。此外，這具引擎具有鈴木公司史上首度採用的旋轉閥式可變進氣系統、平衡軸等元件。車型：
# 2008年迄今：第三代鈴木Escudo
# 2009年-2014年：[[鈴木Kizashi|鈴木Kizashi]]
