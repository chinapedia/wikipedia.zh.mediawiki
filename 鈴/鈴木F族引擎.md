'''鈴木F族引擎'''是1970年代起[[日本|日本]][[鈴木公司|鈴木公司]]開發製造的[[往复式发动机|往復式]][[汽油引擎|汽油引擎]]，[[排氣量|排氣量]]自0.5L至1.1L不等，大部份搭載於該公司所生產的[[輕型車|輕型車]]。若不計入後文所敘的F8B型，1994年後逐漸由[[鈴木K族引擎|鈴木K族引擎]]替代。

== 沿革與概要 ==
此族引擎源起第二代鈴木Jimny外銷專用（LJ80系），1972年開發之際乃鈴木第一具[[汽車|汽車]][[四冲程循环|四衝程循環]]引擎。後來該公司惟恐搭載[[两冲程循环|二衝程循環]]引擎的第四代鈴木Fronte無法通過新制的[[昭和|昭和]]50年度汽車廢氣排放標準<ref>於是1977年6月上市的小改款Fronte中，部分車型搭載由[[大發工業株式會社|大發工業]]提供的{{link-ja|大發AB族引擎|ダイハツ・AB型エンジン|AB族引擎}}。後來，二衝程循環引擎的第四代鈴木Fronte順利通過該排放標準的檢驗。</ref>，遂以[[直列四缸引擎|直列四缸]]、四衝程循環的F8A型為基礎，修改成直列三缸、排氣量543c.c.的F5A型，於是F5A型逐漸廣泛搭載於其他車種。

F族引擎的特徵在於上死點附近，[[汽缸蓋|汽缸蓋]]和[[活塞|活塞]]皆有心形凹陷（即{{link-en|燃燒室|Combustion chamber}}），油氣燃燒時形成渦漩狀而加快燃燒速度。但是這個特殊的構造並未使用於多汽門結構的型式。此族引擎的共同點為：直列型四衝程循環，[[凸輪軸|凸輪軸]]以正時皮帶驅動，以及冷卻方式皆為水冷。

== F8C型（SOHC） ==
排氣量796c.c.的'''F8B型'''也是四衝程、SOHC、水冷式的設計，但為直列三缸。缸徑68.5mm、行程72.0mm、壓縮比8.7：1，搭配[[化油器|化油器]]可輸出最大馬力49ps / 6,000rpm、扭力峰值71.5 N·m / 4,000rpm。[[南韓|南韓]]{{link-en|大宇 (汽車)|Daewoo Motors|大宇汽車}}基於此具引擎，1991年開發出S-TEC型引擎。車型：
車型：

* 1991年迄今：[[鈴木Carry#大宇Damas_/_Labo_/_Attivo|大宇Damas/Labo/Attivo]]

== F8D型（SOHC） ==

* 2000年-2014年：[[鈴木Alto|馬魯蒂Alto 800]]
* 2012年迄今：[[鈴木Alto|馬魯蒂Alto 800]]

== F10A型（SOHC） ==

* 1982年-1985年：[[鈴木Jimny#SJ40型Jimny_1000|第二代鈴木Jimny 1000/SJ410/Samurai]]、鈴木Potohar（[[巴基斯坦|巴基斯坦]]）

* 1985年-2000年：[[馬魯蒂Gypsy|馬魯蒂Gypsy MG410]]（[[印度|印度]]）
* 1988年-1999年：[[鈴木Cultus#第二代（SF系_1988年-1999年）|第二代鈴木Cultus]]/Swift/鈴木Forsa II SF410（某些[[發展中國家|發展中國家]]市場，如[[厄瓜多爾|厄瓜多爾]]）

== F10D型（SOHC） ==
排氣量1,061c.c.、直列四缸、SOHC構造的'''F10D型'''引擎乃自2001年起由[[馬魯蒂鈴木公司|馬魯蒂鈴木公司]]開發製造，專門供應[[印度|印度]]國內市場。缸徑68.50mm、行程72.00mm、壓縮比10.0：1，供油系統採MPFi多點噴射，可輸出62ps / 6,000rpm的最大馬力及85.0N·m / 3,200rpm的最大扭力。後來換裝32[[位元|位元]]的[[发动机控制器|ECU]]後，馬力小昇至64ps。由於無法通過[[欧洲汽车尾气排放标准|歐盟四期汽車廢氣排放標準]]，2009年9月馬魯蒂鈴木宣佈此具引擎將自2010年起退役。車型：
* 2000年-2010年：[[鈴木Wagon_R|馬魯蒂Wagon R]]
* 2001年-2014年：[[鈴木Alto|馬魯蒂Alto 1000]]
* 2006年-2009年：[[鈴木MR_Wagon#印度馬魯蒂Zen_Estilo|馬魯蒂Zen Estilo]]
