'''鈴真'''（{{lang-ja|りんしん}}{{bd}}）<!--鈴木真矢→鈴真→{{lang|ja|りんしん}}-->，[[日本|日本]][[三重縣|三重縣]]出身的[[動畫師|動畫師]]、[[插畫家|插畫家]]。[[Arms|Arms]]所屬。O型[[血型|血]]{{r|presepe}}。

== 人物簡介 ==
年齡、性別、本名不公開。

高校畢業後立即前往[[東京|東京]]，[[福田皖|福本完]]為師事遠親。最初在《[[霹靂貓|霹靂貓]]》擔任動畫工作，因勾勒線條很漂亮，不到一年就升任[[原畫|原畫]]<ref name=rsb2001>[[#rsb2001|#rsb2001]]</ref>；經[[OVA|OVA]]《[[淫獸學園|淫獸學園2]]》負責[[作畫監督|作畫監督]]、《淫獸學園3》負責[[人物設計|人物設定]]，持續投入[[美少女|美少女]]動畫作品。遊戲原畫初作同樣是《淫獸學園》<ref group="注">從首本畫集介紹就誤植成《淫獸教室》至今。</ref>，並藉由動畫《[[同級生2|同級生2]]》人物設定作業之緣故，參加遊戲《[[WORDS_WORTH|WORDS WORTH]]》的人物設定、原畫{{r|presepe}}。曾以[[成人動畫|成人動畫]]製作為中心，隨[[Arms|Arms]]策略改變轉從事一般動畫活動，因此《{{link-ja|緋忍傳 呀宇種|緋忍伝 呀宇種}}》乃Arms也是鈴真的最後成人動畫作品<ref>[http://www.pinkpineapple.co.jp/web/gausu/interview/ {{lang|ja|緋忍伝 呀宇種（ガウス） - インタビュー}}]. 動畫「緋忍伝 呀宇種」公式官網 [2018-09-19] {{ja icon}}.</ref><ref>{{Cite web |script-title=ja:ピンパイ最新作『緋忍伝 -ガウス-』は、りんしん氏最後の18禁作品!? |date=2005-09-16 |url=http://galge.com/galge/nomember/sbp/topics/05/09/16b/index.html |accessdate= |archiveurl=https://web.archive.org/web/20090208234343/http://galge.com/galge/nomember/sbp/topics/05/09/16b/index.html |publisher=Galge.com |archivedate=2009-02-08 |language=ja |deadurl=yes }}</ref><ref>{{Cite web |script-title=ja:OVA『緋忍伝-ガウス-』の制作スタッフにインタビューしました！ |date=2005-11-14 |url=http://www.galge.com/galge/nomember/sbp/topics/05/11/14/index.html |accessdate= |archiveurl=https://web.archive.org/web/20090812063842/http://www.galge.com/galge/nomember/sbp/topics/05/11/14/index.html |publisher=Galge.com |archivedate=2009-08-12 |language=ja |deadurl=yes }}</ref>。
