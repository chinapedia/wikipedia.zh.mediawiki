'''杰克·威廉·绍斯塔克'''（{{lang-en|'''Jack William Szostak'''}}，{{bd|1952年|11月9日|catIdx=Szostak, Jack}}），[[美国|美国]]<ref name="nobel_press">{{cite web |url=http://www.nobelprize.org/nobel_prizes/medicine/laureates/2009/press.html |title=The 2009 Nobel Prize in Physiology or Medicine - Press Release |publisher=[[诺贝尔基金会|诺贝尔基金会]] |date=2009年10月5日 |accessdate=2011年11月13日 |language=en }}</ref>[[生物学家|生物学家]]、[[霍华德·休斯医学研究所|霍华德·休斯医学研究所]]研究员、[[哈佛医学院|哈佛医学院]][[遗传学|遗传学]]教授、[[麻省总医院|麻省总医院]]亚历山大·里奇杰出研究员。<ref name="hhmi">{{cite web |url=http://www.hhmi.org/research/investigators/szostak_bio.html |title=HHMI Scientist Bio: Jack W. Szostak, Ph.D. |publisher=[[霍华德·休斯医学研究所|霍华德·休斯医学研究所]] |date= |accessdate=2011年11月13日 |language=en }}</ref><ref name="lab">{{cite web |url=http://genetics.mgh.harvard.edu/szostakweb/people.html |title=Szostak Lab: People |publisher=绍斯塔克实验室 |date= |accessdate=2011年11月13日 |language=en }}</ref>他与[[伊丽莎白·布莱克本|伊丽莎白·布莱克本]]和[[卡罗尔·格雷德|卡罗尔·格雷德]]因为“发现[[端粒|端粒]]和[[端粒酶|端粒酶]]如何保护[[染色体|染色体]]”而一起获得2009年[[诺贝尔生理学或医学奖|诺贝尔生理学或医学奖]]。<ref name="nobel_press"/>

== 生平 ==
绍斯塔克1952年出生于[[英国|英国]][[伦敦|伦敦]]，父母分别来自[[波兰|波兰]]和[[英格兰|英格兰]]。当时他的父亲在[[帝国理工学院|帝国理工学院]]学习航空工程。后来由于父亲工作原因，绍斯塔克跟随家人多次搬家，最后定居[[加拿大|加拿大]]。父母对绍斯塔克的科学兴趣非常支持，还允许他在地下室尝试一些危险的科学实验。<ref name="nobel_bio">{{cite web |url=http://www.nobelprize.org/nobel_prizes/medicine/laureates/2009/szostak.html |title=Jack W. Szostak - Biographical |publisher=[[诺贝尔基金会|诺贝尔基金会]] |date= |accessdate=2011年11月13日 |language=en }}</ref><ref name="sinovision">{{cite web |url=http://www.sinovision.net/index.php?module=newspaper&act=details&col_id=331&news_id=171062 |title=探索生命起源之門——記諾貝爾奬獲得者杰克·紹斯塔克博士 |publisher=美国中文网，原载《僑報周末》 |date=2011年5月16日 |accessdate=2011年11月13日 |language= }}</ref>

1968年，绍斯塔克不满16周岁就从高中毕业，进入[[麦吉尔大学|麦吉尔大学]]学习，1972年获[[细胞生物学|细胞生物学]]的[[学士|学士]]学位。之后，他在[[康奈尔大学|康奈尔大学]]师从[[华裔|华裔]]生物学家[[吴瑞|吴瑞]]教授<ref>[http://www.springerlink.com/content/g274621682085040/ Ray Wu, as remembered by a former student]</ref>，攻读[[生物化学|生物化学]][[博士|博士]]学位，1977年毕业后继续在[[吴瑞|吴瑞]]教授指导下开展[[博士后|博士后]]研究。<ref name="nobel_bio"/><ref name="sinovision"/><ref>{{cite journal en |author=Szostak, Jack |title=Ray Wu, as remembered by a former student |journal=Science in China Series C: Life Sciences |volume=52 |issue=2 |pages=108–110 |doi=10.1007/s11427-009-0023-6 |url=http://www.springerlink.com/content/g274621682085040/ |date=February 2009}}{{En icon}}</ref><ref name="mgh">{{cite web|url=http://ccib.mgh.harvard.edu/vitas/SzostakCV.pdf |title=Curriculum Vitae of Jack W. Szostak |publisher=[[麻省总医院|麻省总医院]] |date= |accessdate=2011年11月13日 |language=en |format=PDF |deadurl=yes |archiveurl=https://web.archive.org/web/20080828190726/http://ccib.mgh.harvard.edu/vitas/SzostakCV.pdf |archivedate=2008年8月28日 }}</ref>

1979年，绍斯塔克加入哈佛医学院，在悉尼·法伯癌症研究所（现为达纳－法伯癌症研究所）担任助理教授，建立自己的实验室。期间，他与[[伊丽莎白·布莱克本|伊丽莎白·布莱克本]]在合作研究中发现，[[端粒|端粒]]中一段独特的[[DNA序列|DNA序列]]保护[[染色体|染色体]]免于退化，而布莱克本的博士生[[卡罗尔·格雷德|卡罗尔·格雷德]]又进一步鉴别出了参与端粒DNA复制的一种[[逆转录酶|逆转录酶]]——[[端粒酶|端粒酶]]。他们发现，端粒变短则细胞衰老，而端粒酶的活性能够维持端粒的长度。这些研究成果对[[癌症|癌症]]及其他疾病的研究具有重要影响。<ref name="sfbt">{{cite web |url=http://sanfrancisco.bizjournals.com/sanfrancisco/stories/2009/10/05/daily1.html |title=UCSF's Elizabeth Blackburn wins Nobel Prize |first=Ron |last=Leuty |publisher=San Francisco Business Times |date=2009年10月5日 |accessdate=2011年11月13日 |language=en }}</ref>期间，他还参与了世界上最早的关于[[酵母菌人工染色体|酵母菌人工染色体]]的研究。1983年，绍斯塔克升任副教授；次年，实验室整体搬迁到[[麻省总医院|麻省总医院]]。1988年，升任教授，取得哈佛医学院终身教职。

== 研究 ==
绍斯塔克在[[遗传学|遗传学]]领域有贡献。他被相信是构造了世界上第一个[[酵母菌人工染色体|酵母菌人工染色体]]。这一成就有助于科学家绘制在哺乳动物[[基因|基因]]的位置，和开发操纵基因技术。他在这方面的成就，也有助于[[人类基因组计划|人类基因组计划]]。

1998年起，兼任[[霍华德·休斯医学研究所|霍华德·休斯医学研究所]]研究员。2000年起，兼任麻省总医院亚历山大·里奇杰出研究员。<ref name="hhmi"/><ref name="mgh"/>

近年来，他的实验室将研究重点转向了[[生命起源|生命起源]]问题，<ref name="nobel_bio"/><ref>{{cite web |url=http://www.hms.harvard.edu/dms/bbs/fac/szostak.php |title=Jack W. Szostak |publisher=哈佛医学院 |date= |accessdate=2011年11月13日 |language=en }}</ref><ref>{{cite web |url=http://exploringorigins.org/about.html |title=About the Exploring Origins Project |publisher=exploringorigins.org |date= |accessdate=2011年11月13日 |language=en }}</ref>和在实验室建造的人造细胞生命。<ref>http://exploringorigins.org/ Exploringorigins.org </ref>

== 荣誉与奖励 ==
因为他的贡献，绍斯塔克已获得多个奖项和荣誉。绍斯塔克是[[美国国家科学院|美国国家科学院]]、[[美国文理科学院|美国文理科学院]]及[[纽约科学院|纽约科学院]]院士。<ref name="hhmi"/>他曾先后获得过如下奖项：
* 美国国家科学院分子生物学奖（1994年）<ref>{{cite web |url=http://nas.nasonline.org/site/PageServer?pagename=AWARDS_molbio |title=NAS Award in Molecular Biology |publisher=[[美国国家科学院|美国国家科学院]] |date= |accessdate=2011年11月13日 |language=en |deadurl=yes |archiveurl=https://web.archive.org/web/20120722011217/http://nas.nasonline.org/site/PageServer?pagename=AWARDS_molbio |archivedate=2012年7月22日 }}</ref>
* Hans Sigrist奖，瑞士伯尔尼大学
* 美国遗传学会奖章（2000年）<ref>{{cite web |url=http://www.genetics-gsa.org/pages/pastawards.shtml |title=Past GSA Award Recipients |publisher=美国遗传学会 |date= |accessdate=2011年11月13日 |language=en }}</ref>、
* [[拉斯克奖|艾伯特·拉斯克基础医学研究奖]]（2006年）<ref>{{cite web |url=http://www.laskerfoundation.org/awards/2006basic.htm |title=2006 Basic Medical Research Award |publisher=拉斯克基金会 |date= |accessdate=2011年11月13日 |language=en }}</ref>和
* Heineken奖，荷兰皇家艺术与科学院（2008年）
* [[诺贝尔生理学或医学奖|诺贝尔生理学或医学奖]]（2009年）<ref name="nobel_press"/>。
