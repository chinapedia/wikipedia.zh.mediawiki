'''杰拉德·特·胡夫特'''（{{lang-nl|'''Gerard 't Hooft'''}} ，{{bd|1946年|7月5日|catIdx=T}}），[[荷兰|荷兰]][[理论物理学|理论物理]]學家，[[乌得勒支大学|乌得勒支大学]]教授，於1999年因為「闡明[[物理学|物理学]]中[[電弱相互作用|電弱相互作用]]的[[量子|量子]]結構」與其指導教授[[馬丁紐斯·韋爾特曼|馬丁紐斯·韋爾特曼]]一同獲得[[诺贝尔物理学奖|诺贝尔物理学奖]]。<ref name=nobel>{{cite web | title = The Nobel Prize in Physics 1999| publisher = Nobel Foundation | url = http://nobelprize.org/nobel_prizes/physics/laureates/1999/index.html|accessdate=2008-10-09}}</ref>二十世纪中后期最重要的理论家之一。

==生平==
===早年===
1946年7月5日，杰拉德·特·胡夫特出生在[[荷蘭|荷蘭]][[登海爾德|登海爾德]]，但主要在[[海牙|海牙]]長大。在家中三個孩子裡排行第二。特·胡夫特生長在一個充滿學者的家庭，他的外祖母是諾貝爾獎得主[[弗里茨·澤爾尼克|弗里茨·澤爾尼克]]的姊妹，外祖父是[[萊登大學|萊登大學]]知名的[[動物學|動物學]]教授。舅舅{{le|尼可·范坎彭|Nico van Kampen}}在[[烏特勒支大學|烏特勒支大學]]擔任物理學教授。雖然他的母親作為一名女性，並未從事科學事業，但她嫁給了一位海洋工程師。<ref name="autobio">{{Cite web
 |first=G. |last='t Hooft
 |url=http://nobelprize.org/nobel_prizes/physics/laureates/1999/thooft-autobio.html
 |title=Gerardus 't Hooft - Autobiography
 |publisher=Nobel web
 |year=1999
 |accessdate=06-10-2010
}}</ref>在父親的影響下，特·胡夫特很早就展現對科學的興趣。念小學時當學校老師問起未來志向時，他表示要成為一位「知曉一切事物的人」。<ref name="autobio"/>

小學畢業之後，特·胡夫特進入道爾頓學園（Dalton Lyceum）就讀。這是一間實行{{le|道爾頓計劃|Dalton Plan}}的教育機構，這套教學方法相當適合特·胡夫特。他輕鬆通過了科學、數學等科目，但對於語言課程不是很擅長。儘管如此，他還是完成了英語、法語、德語、[[古希臘語|古希臘語]]、和[[拉丁語|拉丁語]]的課程。十六歲那年，他參加了[[國際奧林匹克數學競賽|奧林匹克數學競賽]]並獲得了銀牌。<ref name="autobio">{{Cite web
 |first=G. |last='t Hooft
 |url=http://nobelprize.org/nobel_prizes/physics/laureates/1999/thooft-autobio.html
 |title=Gerardus 't Hooft - Autobiography
 |publisher=Nobel web
 |year=1999
 |accessdate=06-10-2010
}}</ref>

===求學===
1964年，特·胡夫特通過了中學測驗，進入烏特勒支大學學習物理。因為希望就讀他舅舅所在的物理系，他選擇了烏特勒支大學而非距離更近的萊登大學。由於特·胡夫特太過專注在科學，他的父親堅持他參加學校裡的一個精英社團「烏得勒支學生隊」（Utrechtsch Studenten Corps），以期望他能從事一些課外活動。這某些程度上確實展現了一些效果，他曾在賽艇俱樂部「特里同」（Triton）擔任指揮，與科學討論社團「克里斯蒂安·惠更斯」（Christiaan Huygens）一起規劃了一次科學類的全國學生代表大會。

在物理的學習上，特·胡夫特決定追求他所認為理論物理中的核心—[[粒子物理學|基本粒子物理]]。他的舅舅並不喜歡這個領域，所以當特·胡夫特要寫「doctoraalscriptie」（在荷蘭相當於碩士論文）時，他找上了新進的教授[[馬丁紐斯·韋爾特曼|馬丁紐斯·韋爾特曼]]。韋爾特曼當時在研究[[楊-米爾斯理論|楊-米爾斯理論]]，由於這個理論在物理計算上無法被[[重整化|重整化]]，以當時而言並不算是主流的研究課題。特·胡夫特被交付的題目是{{le|阿德勒-貝爾-傑克威反常|Adler–Bell–Jackiw anomaly}}，一個理論無法解釋的電中性[[π介子|π介子]]衰變現象。當時的理論不允許π介子衰變產生光子，然而實際的實驗卻觀測到這類的衰變。特·胡夫特當時也無法給出合理的解釋。

1969年，特·胡夫特選擇韋爾特曼作為他博士學位的指導教授。他開始研究韋爾特曼正在研究的課題，楊-米爾斯理論的重整化。1971年他發表了第一篇論文，裡頭說明如何重整化楊-米爾斯場中質量為零的場。<ref>{{cite journal|url=http://linkinghub.elsevier.com/retrieve/pii/0550321371903956|pages=173–199|title=Renormalization of massless Yang-Mills fields|journal=Nuclear Physics B|volume=33|issue=1|accessdate=2018-04-02|doi=10.1016/0550-3213(71)90395-6|author=G. 'tHooft}}</ref>這項工作並未引起太多的注意，但韋爾特曼對於長久以來的問題被解決感到相當興奮。在這之後他們發展了一套稱為{{le|因次正規化|dimensional regularization}}的方法。隨後特·胡夫特的第二篇論文也發表，探討了楊-米爾斯理論在[[自發對稱性破缺|自發對稱破缺]]之後帶質量場的重整化。<ref>{{cite journal|url=http://linkinghub.elsevier.com/retrieve/pii/0550321371901398|pages=167–188|title=Renormalizable Lagrangians for massive Yang-Mills fields|journal=Nuclear Physics B|volume=35|issue=1|accessdate=2018-04-02|doi=10.1016/0550-3213(71)90139-8|author=G.'t Hooft}}</ref>這篇文章漸漸受到廣泛注意，在1999年為兩人贏得[[諾貝爾物理學獎|諾貝爾物理學獎]]。

以這兩篇文章為基礎，1972年特·胡夫特完成了他的學位論文《楊-米爾斯場的重整化方法》。同一年，他與烏特勒支大學醫學院的學生Albertha A. Schik結為夫妻。<ref name=autobio/>

===學術生涯===
[[Image:Gerardus_t'_Hooft_at_Harvard.jpg|thumb]]
完成博士學位之後，特·胡夫特到了[[歐洲核子研究組織|歐洲核子研究組織]]擔任研究員。他和韋爾特曼一起試著完善楊-米爾斯理論的重整化計算。這段期間，他開始對[[強交互作用|強交互作用]]感到興趣，認為這類的交互作用可以用零質量的楊-米爾斯理論來描述。根據計算的結果，這樣的理論具有[[漸進自由|漸進自由]]的性質，符合{{link-en|深度非彈性散射|deep inelastic scattering}}實驗的結果。這與物理學家所認知的楊-米爾斯理論相違背，過去我們所知的如[[重力|重力]]和[[電磁作用|電磁作用]]其作用的強度隨距離增加而減弱。然而這種人們熟知的性質無法解釋深度非彈性散射的實驗結果。1972年，特·胡夫特在一個小型會議中談論到這個計算成果，一位物理學家{{le|庫爾特·希門奇克|Kurt Symanzik}}便建議他發表這個成果。<ref name=autobio/>然而之後特·胡夫特仍未將此計算發表成論文。不久之後，[[休·波利策|休·波利策]]、[[大衛·葛羅斯|大衛·葛羅斯]]、[[弗朗克·韋爾切克|弗朗克·韋爾切克]]等人於1973年也得到了同樣的結果，三人因此貢獻獲得了2004年的諾貝爾物理獎。<ref>{{cite web
 |url=http://nobelprize.org/nobel_prizes/physics/laureates/2004/
 |title=The Nobel Prize in Physics 2004
 |publisher=Nobel Web
 |year=2004
 |accessdate=2010-10-24
}}</ref><ref>{{cite web
 |url=http://nobelprize.org/nobel_prizes/physics/laureates/2004/politzer-lecture.pdf
 |format=PDF
  |title=The Dilemma of Attribution
 |first=H. David
 |last=Politzer
 |publisher=Nobel Web
 |year=2004
 |accessdate=2010-10-24
}}</ref>

== 研究 ==
特·胡夫特的研究興趣大致分為三個方向：基本粒子物理的[[規範理論|規範理論]]、[[量子重力|量子重力]]和[[黑洞|黑洞]]、量子力學的基礎研究。<ref name="homepage">{{cite web
 |first=G. |last='t Hooft
 |title=Gerard 't Hooft
 |url=http://www.phys.uu.nl/~thooft/
 |accessdate=24-20-2010
}}</ref>

===基本粒子物理的規範理論===
物理上特·胡夫特最為人所知的貢獻是有關規範理論的研究。

* 证明了[[规范场论|规范理论]]的可[[重整化理论|重整化]]
* [[引力|引力]][[协变量子化|协变量子化]]
* 独立发现了[[量子色动力学|量子色动力学]]的[[渐近自由|渐近自由]]
* [[磁单极|特·胡夫特-泊里雅可夫磁单极]]解
* 在规范理论中的其他重要工作，包括[[夸克禁闭|夸克禁闭]]、[[大N展开|大N展开]]、[[QCD弦|QCD弦]]、[[瞬子|瞬子]]、[[反常|反常]]等
* 独立提出了[[量子引力|量子引力]]中的[[全息原理|全息原理]]
