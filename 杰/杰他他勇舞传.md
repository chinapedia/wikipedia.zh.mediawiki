《'''舞勇傳杰他他'''》是一部由日本[[漫畫家|漫畫家]][[衛藤浩幸|衛藤浩幸]]（{{lang-ja|衛藤ヒロユキ}}）於免費線上[[期刊|期刊]]《[[GANGAN_ONLINE|GANGAN ONLINE]]》上連載的[[漫畫|漫畫]]作品，是《[[咕嚕咕嚕魔法陣|咕嚕咕嚕魔法陣]]》的外傳<ref name="nari"/><ref>{{Cite web |url= http://natalie.mu/comic/news/18084|title= 「舞勇伝キタキタ」1巻発売記念、未収録分を無料配信
|accessdate=2011-10-19|publisher=natalie.mu |date= 2009-06-27}}</ref>。各話標題均以「第○蓑」表示。

另外，該作是《GANGAN ONLINE》首批被印製成[[單行本|單行本]]發售的十一部作品之一<ref>{{Cite web |url= http://natalie.mu/comic/news/17109|title= ガンガンONLINEから初単行本、一挙11タイトル|accessdate=2011-10-19|publisher=natalie.mu |date= 2009-06-01}}</ref>、以及是其連載網站二週年紀念活動的主打作品之一<ref>{{Cite web|url=http://www.square-enix.com/jp/magazine/ganganonline/event/2nd/|title=ガンガンONLINE 2周年 コミックスフェア|accessdate=2011-10-19|publisher=[[GANGAN_ONLINE|GANGAN ONLINE]]|deadurl=yes|archiveurl=https://web.archive.org/web/20111018142501/http://www.square-enix.com/jp/magazine/ganganonline/event/2nd/|archivedate=2011-10-18}}</ref>。

== 角色 ==
=== 主要角色 ===
;杰他他老伯(キタキタおやじ)
:本名艾特柏傑·安德魯(TVB譯：-{雅度柏·艾爾圖}-)。上身赤裸、下身著[[草裙|草裙]]的[[禿頭|禿頭]]老伯，特點是腳毛長。曾是杰他他村的村長，惟一的絕技是跳杰他他舞。經常纏著仁傑和歌莉，四處尋找傳人。可惜因形象問題，普通人已經避之則吉，根本沒有人願意學。惟一的用處是作為主角的擋箭牌。

;奇基(チキ)
:13歲，男，克萊姆村居民，雙親被妖怪殺死。把封印了的杰他他老伯再次召喚出來，後來和杰他他老伯踏上了成為勇者的旅途。
