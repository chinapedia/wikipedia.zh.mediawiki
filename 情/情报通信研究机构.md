國立研究開發法人'''情報通信研究機構'''（英語譯名：{{lang|en|National Institute of Information and Communications Technology}}，簡寫為{{lang|en|'''NICT'''}}）是隶属于[[日本|日本]][[总务省|总务省]]的[[独立行政法人|独立行政法人]]機構（{{link-ja|國立研究開發法人|国立研究開発法人}}）。该机构成立于2004年4月，目前总部位于日本[[东京都|东京都]][[小金井市|小金井市]]，主要进行[[信息技术|信息技术]]领域的研究和开发，同时对信息[[通信|通信]]提供业务支持，目前的理事长是[[宮原秀夫|宮原秀夫]]。

== 历史沿革 ==
[[File:NICT_National_Institute_of_Information_and_Communications_Technology.jpg|thumb]]
[[日本|日本]]的情报通信研究机构的历史可以追溯到1896年10月[[遞信省|遞信省]]的“电气试验所”成立[[无线电|无线电]]通信研究部门。1948年6月，“电波物理研究所”划归[[文部省|文部省]]管辖。1952年8月，[[邮政省|邮政省]]成立“电波研究所”。1988年4月，电波研究所更名为“通信综合研究所”。2001年4月，通信综合研究所随着日本政府机构的调整划归[[总务省|总务省]]，并成为独立[[行政法人|行政法人]]。

另一方面，1979年8月，日本设立“通信·放送卫星机构”，负责卫星通信和广播的管制服务，并于1992年10月更名为“通信·放送机构”。2002年3月，该机构终止卫星管制的业务，自此其主要业务为通信和广播技术的研究和开发。

2004年4月，日本政府决定合并原有“通信综合研究所”和“通信·放送机构”，成立“情报通信研究机构”。<ref>{{cite web |url=http://www.nict.go.jp/about/history.html |title=沿革 |publisher=情报通信研究机构 |accessdate=2009-03-28 |language=ja }}</ref>

== 组织机构 ==
情报通信研究机构目前在日本各地设有15处研究所、3处观测站和2处无线电发射站，其研发机构分为7个研究部门。<ref>{{cite web |url=http://www.nict.go.jp/about/organization.html |title=組織紹介 |publisher=情报通信研究机构 |accessdate=2009-03-28 |language=ja |deadurl=yes |archiveurl=https://web.archive.org/web/20090324013038/http://nict.go.jp/about/organization.html |archivedate=2009-03-24 }}</ref>

;第一研究部门
:[[下一代网络|下一代网络]]技术的基础研究部门，下设：下一代网络技术研究中心、下一代无线技术研究中心和未来信息通信技术研究中心
;第二研究部门
:通用通信技术的基础研究部门，下设：知识创新通信技术研究中心和通用媒介研究中心
;第三研究部门
:信息通信安全技术的基础研究部门，下设：信息通信安全技术研究中心和电磁波计量研究中心
;联合研究部门
:下设：JGNII项目推进研究中心
;研究推进部门
:主要负责各种研究的成果转化，知识产权的确立，以及各个领域的国际合作，并负责其成果的[[JIS|JIS]]或[[ISO|ISO]]标准化工作。
;基本研究促进部门
:主要负责公开招募民间无法进行的高风险研究、招募海外科研人员。
;信息通信振兴部门
:主要为信息通信行业的创业者提供信息，并对创业和基础设施的建设提供资助和业务支持。

== 公共服务 ==

=== 标准频率和时间 ===

根据日本法律，情报通信研究机构是设定标准频率和时间的官方机构，并负责标准电波和时间的发送工作。<ref>{{cite web |url=http://jjy.nict.go.jp/index.html |title=日本標準時（JST）プロジェクト |publisher=情报通信研究机构 |accessdate=2009-03-28 |language=ja }}</ref>

情报通信研究机构采用一个特定的测量系统每天一次对其放置于4个钟室的[[铯|铯]][[原子钟|原子钟]]和[[氢|氢]][[微波发射器|微波发射器]]原子钟进行时差的测量，通过平均、合成这些原子钟的时间，得到该机构的[[世界协调时|世界协调时]]，并在此基础上增加9个小时成为日本标准时。在使用氢原子钟之前，其世界协调时与[[国际度量衡局|国际度量衡局]]的差距在±50[[纳秒|纳秒]]以内，在2006年2月7日引入氢原子钟后，目前的差距在±10纳秒以内。

两座标准电波的发射台位于[[福岛县|福岛县]][[田村市|田村市]]的[[大鹰鸟谷山|大鹰鸟谷山]]上和[[佐贺县|佐贺县]][[佐贺市|佐贺市]]的[[羽金山|羽金山]]上，[[無線電呼號|無線電呼號]]「[[JJY|JJY]]」，发射频率为40千[[赫兹|赫兹]]及60千赫兹。除了接收[[长波电台|长波电台]]的广播外，情报通信研究机构还通过[[NTP|NTP]]服务以及[[电话|电话]]回路提供标准频率和时间的服务。

=== 宇宙天气预报 ===

为减少[[磁暴|磁暴]]对信息通信活动带来的损失，情报通信研究机构还对磁场的状态进行观测，并提供宇宙天气预报的服务，发布[[太阳活动|太阳活动]]、[[电离层|电离层]]观测和[[地磁|地磁]]活动的有关信息。<ref>{{cite web |url=http://www2.nict.go.jp/y/y223/sw_portal/sw_portal.html |title=宇宙天気予報 |publisher=情报通信研究机构 |accessdate=2009-03-28 |language=ja |deadurl=yes |archiveurl=https://web.archive.org/web/20090324010636/http://www2.nict.go.jp/y/y223/sw_portal/sw_portal.html |archivedate=2009-03-24 }}</ref>
