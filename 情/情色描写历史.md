'''情色描写'''（英语：{{lang|en|erotic depictions}}）包括以[[繪畫|繪畫]]、[[雕塑|雕塑]]、[[攝影|攝影]]、[[戲劇|戲劇]]、[[音樂|音樂]]以及[[寫作|寫作]]的手法描述[[人类的性|人类的性]]相關的場面，這在歷史上幾乎所有文明都曾產生過，不分古今東西。在早期人類文化上，性型態與動作上常與超自然力量相聯繫，在一些宗教儀式可見端倪。在亞洲國家如[[印度|印度]]、[[尼泊爾|尼泊爾]]、[[斯里蘭卡|斯里蘭卡]]、[[中國|中國]]和[[日本|日本]]，情欲的表達與{{link-en|情色藝術|Erotic art|情色藝術的作品}}在當地本土宗教（諸如[[印度教|印度教]]、[[佛教|佛教]]、[[道教|道教]]或[[神道教|神道教]]等）之中有其各自特殊的精神上的意義。而[[古希臘|古希臘]]與[[古羅馬|古羅馬]]人产製了諸多情色藝術與裝飾，也多與他們的宗教信仰及文化習俗相融合。<ref name = east>{{cite book| last = Rawson| first = Phillip S.| title = Erotic art of the east; the sexual theme in oriental painting and sculpture| publisher = Putnam| date = 1968| location = New York| pages = 380| id = {{LCC|N7260.R35}}}}</ref><ref name = romans>{{cite book| last = Clarke| first = John R.| title = Roman Sex: 100 B.C. to A.D. 250| publisher = Harry N. 古代，情色描写往往屬於當地本土或宗教藝術文化的一部分，無異於其他類型的作品，現代所謂[[色情物品|色情物品]]（pornography）的概念在早期尚未成型，直到後來19世紀的[[維多利亞時代|維多利亞時代]]，才有將描寫[[妓女|妓女]]的文章作為色情物品的最初定義，如1857年的英國醫學字典：「色情物品——以公共衛生的角度對妓女或賣淫的描述」。<ref>Dunglison, Robley (1857). ''Medical lexicon. A dictionary of medical science'', 1857 edition, s.v. [http://dictionary.oed.com/help/bib/oed2-d2.html#dunglison "Pornography"]. From the [[牛津英語詞典|牛津英語詞典]], second edition (1989), Oxford University Press, Retrieved on 11月30日, 2006年.</ref>

色情物品的定義，到了1860年代已開始與現代意義相當接近<ref name = pleasures>{{cite book| last = Sigel| first = Lisa| title = Governing Pleasures. Pornography and Social Change in England, 1815-1914| publisher = Rutgers University Press| date = 2002| id = ISBN 978-0-8135-3001-7}}</ref>1864年，[[韋氏詞典|韋氏詞典]]首現其現代定義：「用作裝飾供人於[[酒神節|酒神節]]狂歡的房間牆壁之淫褻圖畫，如[[龐培城|龐培城]]的壁畫。」<ref>''An American dictionary of the English language'', new and revised edition (1864), s.v. [http://dictionary.oed.com/help/bib/oed2-w.html#webster "Pornography"]. From the [[牛津英語詞典|牛津英語詞典]], second edition (1989), Oxford University Press, Retrieved on 11月30日, 2006年.</ref>這標誌着在英美國家，人們開始對性愛描述賦予不雅的性質。1857年，英國通過法例對觀看情色描写作品的人加以懲罰，這在之前是沒有的。<ref name = history/>1860年代，龐培城的大量文物出土，[[古羅馬|古羅馬]]大量情色描写作品重見天日，使一向自認為承襲[[羅馬帝國|羅馬帝國]]智慧的維多利亞時代的英国人大感震驚。他們採取逃避策略。大量出土文物被收藏到[[那不勒斯|那不勒斯]]的秘密博物館，不可移動的文物則被掩蓋起來，更派員把守。
