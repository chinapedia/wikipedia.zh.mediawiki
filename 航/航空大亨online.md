{{vgname|航空大亨online|en=Air Tycoon Online(ATO)}}是一款[[经营模拟游戏|经营模拟游戏]]，適用於[[iPhone|iPhone]]、[[iPad|iPad]]及任何[[Android|Android]]裝置，本游戏可以让玩家建立自己的航空公司。游戏以回合制的方式進行，每回合兩個小時。透過本遊戲可以與同一个服务器世界各地的玩家進行互動，在詳細的統計數據支援下，建立及管理全球的[[飛機|航班]]，包括客運及貨運，甚至是[[機場|機場]]以及附屬營利設施。遊戲需要以[[WiFi|WiFi]]或行動數據連接[[網際網路|網際網路]]進行。

遊戲時間設定為公元1960年至2020年，公元1969年12月前真實時間每一小時為遊戲中的一個月，公元1970年1月後真實時間每兩小時相等於遊戲內的一個月（即一個回合）。

== 基本功能介紹 ==

* 每次進入伺服器擁有20C幣(付費版50C幣)，資本額視登錄年代之總平均資產而定
* 申請航額每次需15分鐘，單次申請上限為20個
* 付運飛機每次需30分鐘，單次上限5架
* 每架飛機壽命20年
* 空閒航額至多擁有100個

== 续作 ==

官方在Android和iOS推出了续作{{vgname|[[航空大亨online_2|航空大亨online 2]]|en=Air Tycoon Online 2(ATO2)}}。游戏玩法基本相同，但新增(更改)了部分内容：

# 更改遊戲回合時間(1960.01~2030.01為一回合二小時)
# 更改飛機付運及航額申請數量與時間
# 更改可租賃的機型(如：[[波音|波音]][[波音727|727]]-100、[[747|747]]-100、[[757|757]]-200；麥道[[DC-8|DC-8]]-61、DC-9-10...)
# 更改附加功能之使用制度
# 更改財務報表內容&單位
# 更改獎勵制度
# 更改空閒航額最大值(100→50)
# 增加可使用的飛機餐
# 增加可使用的製造商(如：[[荷蘭|荷蘭]][[福克公司|福克公司]]；[[英國|英國]][[英國航太|英國航太]]...)與機型(如：[[洛克希德|洛克希德]][[L1049G|L1049G]]；[[麥道|麥道]][[DC-9|DC-9]]-21、[[DC-10|DC-10]]-40；[[福克F28|福克F28]]、[[福克F70|福克F70]]、[[福克F100|福克F100]]...)
# 將飛機、機場&地圖改為3D顯示
# 新增"延遠"功能(飛機航程達9800km，且具"SO"標章即可加開)
# 新增"稅"、"保險費"等財務支出
# 新增"收購"功能(可收購退出玩家之公司)
# 新增"任務"獎勵
# 新增"模擬降落"(不論成功失敗，一回合限定2次。成功+1000K，失敗-100K)
# 新增"空難事故"、"天氣異常"等突發狀況
# 新增公司等級制度(與"稅"、"保險費"等財務支出、飛機付運及航額申請數量與時間...等相關)
# 新增飛機"喜好度"功能
# 新增機場(如:[[北韓|北韓]][[平壤|平壤]]、[[南韓|南韓]][[襄陽郡_(韓國)|襄陽]]、[[木浦|木浦]]、[[大邱|大邱]]、[[澳大利亞|澳洲]][[凱恩斯_(昆士蘭州)|凱恩斯]]......)
