'''辰巳唯'''（{{lang-ja|辰巳ゆい}}、{{bd|1984年|6月8日}}），出身自[[岐阜縣|岐阜縣]]，[[日本|日本]]前[[賽車女郎|賽車女郎]]、[[AV女優|AV女優]]<ref name="アリスJAPAN"/>。現在為[[演員|演員]]和[[日本電視藝人|藝人]]<ref name="official">{{Cite web|url=http://yui-tatsumi.com/profile.html |title=公式プロフィール |work=辰巳ゆい OFFICIAL WEBSITE |accessdate=2015-10-3 |language=ja}}</ref><ref>{{cite web|url=http://www.playno1.com/article-19048-1.html#.Wy6eOaczaUk |title=辰巳ゆい近況update! |publisher=AVNo.1-PLAYNO.1 |date=2016-02-22}}</ref>。

== 人物 ==
興趣是[[購物|購物]]，擅長[[料理|料理]]<ref name="アリスJAPAN"/>和做[[餜子|餜子]]。最喜歡的食物是[[蕎麥麵|蕎麥麵]]、[[銀鱈|銀鱈]]西京燒和[[日式燒肉|日式燒肉]]，最愛喝的飲料是咖啡歐蕾。最喜歡的[[藝人|藝人]]是韓國[[男子音樂組合|男團]][[BIGBANG|BIGBANG]]。平日休閒時最喜歡看[[電視|電視]]。最好的朋友是AV女優[[小倉奈奈|小倉奈奈]]<ref>{{cite web|url=http://www.playno1.com/article-11424-1.html#.Wy6m_KczaUk |title=辰巳ゆい的11個小秘密! |publisher=AVNo.1-PLAYNO.1 |date=2013-08-13}}</ref>。

喜歡溫柔有[[大男子主義|男子氣概]]的男生，體型不要太壯也不要太高。平常會用[[按摩棒_(性玩具)|按摩棒]][[自慰|自慰]]，第一次性經驗是在19歲念[[大學|大學]]時候，對方比自己小一歲<ref>{{cite web|url=http://wpb.shueisha.co.jp/2012/06/30/12297/ |title=ＡＶアイドルたちと至福の私服デート　ＶＯＬ．０３６　辰巳ゆい |publisher=[[週刊Playboy|週プレNEWS]] |date=2012-06-30 |language=ja}}</ref>。

曾以「'''朝倉亞矢'''」<ref>跟另一位賽車女郎{{lang|ja|'''朝倉あや'''}}不同人<!--辨別關鍵字以利查證-->。</ref>名義擔任賽車女郎，後因興趣加入[[日本性產業|AV界]]。在前三部作品中公司則讓她不完全退去身上衣物演出，在第四部作品《完全AV》之後，則開始解禁。

== 簡歷 ==
2008年1月11日，成為日本[[AV片商|AV片商]]「[[Alice_Japan|Alice Japan]]」旗下專屬AV女優[[AV|AV]][[出道|出道]]<ref name="アリスJAPAN"/><ref name="official"/>，並簽有「無限期契約」。

2011年11月，成為Alice Japan旗下AV女優團體「Alice Stars」（アリスた～ず☆）的成員之一，並是該團較資深的女優。2013年6月，經過兩年後首次公開[[出道|出道]]<ref>{{Cite web |url=http://tower.jp/article/feature_item/2013/06/17/0705 |title=辰巳ゆい、優希まこと、葵つかさ、奥田咲、小島みなみ、川上奈々美によるアリスた～ず★が待望のデビュー |publisher=[[淘兒唱片|タワーレコード オンライン]] |date=2013-06-17 |language=ja}}</ref>。

2012年8月10日，前往[[韓國|韓國]]宣傳電影《[[AV女優_(電影)|AVアイドル]]》<ref>{{cite news|url=http://contents.innolife.net/news/list.php?ai_id=153682 |title=日本AVスター辰巳ゆい「ウォンビンと熱い夜を過ごしたい」 |publisher=innolife.net |date=2012-08-11 |language=ja}}</ref><ref>{{cite news|url=https://www.nownews.com/news/20130317/309738 |title=AV女優辰巳唯揭露血淚史　演完興奮喊還要 |publisher=[[NOWnews今日新聞|NOWnews今日新聞]] |date=2013-03-17}}</ref>。

2013年，與[[葵司|葵司]]、[[小森愛|小森愛]]到[[香港|香港]]宣傳寫真集，這也是她們首次發行寫真集<ref>{{cite news|url=https://tw.appledaily.com/new/realtime/20130112/161146 |title=AV女優辰巳唯　倒瀉奶慶出道5周年 |publisher=[[蘋果日報_(台灣)|蘋果日報]] |date=2013-01-12}}</ref><ref>{{cite news|url=https://star.ettoday.net/news/153402 |title=AV女優悲歌！任全人類觀賞肉體　28歲辰巳唯暗地垂淚 |publisher=[[ETtoday新聞雲|ETtoday新聞雲]] |date=2013-01-16}}</ref>。10月15日，與港星[[杜汶澤|杜汶澤]]合拍[[香港電影|香港]][[三級片|三級片]]《[[豪情3D|豪情3D]]》<ref>{{cite news|url=https://star.ettoday.net/news/284077 |title=AV女優辰巳唯「床戰抓傷」杜汶澤　女搖D奶畫面辛辣 |publisher=ETtoday新聞雲 |date=2013-10-18}}</ref>。
