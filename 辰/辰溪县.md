'''辰溪县'''<ref>原为「-{辰谿}-」，转换为简化字为「-{辰溪}-」，但[[因用字生僻难认而更名的中国地名列表|并未更改县名]]，故繁体字应对应「-{辰谿}-」。</ref>，古称“辰阳”，位于[[湖南|湖南省西部]]、[[怀化|怀化市]]中部偏北。地图坐标为东径109°54'-110°32'、北纬27°53'-28°13'。辖域总面积1977平方公里，总人口51.61万人（2004年），其中非农业人口为9.85万人，农业人口41.76万人。[[县治|县治]][[辰阳镇|辰阳镇]]。

== 地理 ==
辰溪地处[[雪峰山脉|雪峰山脉]]和[[武陵山脉|武陵山脉]]，[[罗子山|罗子山]]主峰海拔1378.7米，为辰溪境内最高峰。全县共有大小河89条，流域在10平方公里以上的溪流达58条，主要[[河流|河流]]为[[沅水|沅水]]在境内流程99.7公里、[[辰水|辰水]]（沅水之[[酉水_(沅水支流)|酉水]]支流）流程33.7公里，长田湾水库。相邻[[县|县级政区]]有，北临[[沅陵|沅陵县]]和[[泸溪|泸溪县]]，东界[[溆浦|溆浦县]]，南与[[中方|中方县]]、怀化[[市辖区|市辖区]][[鹤城区|鹤城区]]相连，西与[[麻阳|麻阳民族自治县]]为邻。（2009年末全县总人口53.52万人）

辰溪属中[[亚热带|亚热带]][[季风|季风]][[气候|气候]]区，年均降雨量1400mm；年平均温度16℃，无霜期293天，日照时数1477时。

== 民族和语言 ==
现在辰溪县总人口中，[[汉族|汉族]]占88.8%，[[瑶族|瑶族]]占9.1%，[[苗族|苗族]]、土家族等其他少数民族占2.1%。全县各民族通用[[湘语|湘语]][[辰溆片|辰溆片]]，苗族使用[[苗语湘西方言|苗语湘西方言]]。

== 历史 ==
今辰溪辖境，战国属楚黔中地，屈原《涉江》有记载：“朝发枉渚兮，夕宿辰阳”<ref>{{cite book |author1=屈原 |title=楚辞}}</ref>。秦属黔中郡地；于[[西汉|西汉]]高祖二年（前205年）置“辰陵县”，高祖五年易名为“辰阳县”；隶义陵郡（见《舆地广记》），治铜山，即今潭湾镇之杉林、溪边二村。 西汉高祖五年（公元前202），改义陵郡为武陵郡，同时，改辰陵县为辰阳县。三国时，辰阳先属蜀，獍亭之战（222）后，归于孙吴。无阳从辰阳析出，改置舞阳县。晋武帝太康元年（280），裁辰阳并入镡成县。东晋安帝义熙六年（410），裁镡成并人舞阳县。 南朝宋武帝永初元年（420）复置辰阳县。辖地除贵州铜仁一地已划人烊j可郡外，余皆不变。梁武帝天监十年（511），改名建昌县，隶南阳郡。至陈宣帝，废建昌，仍复名辰阳，隶沅陵郡。隋文帝开皇九年（589），辰阳县治迁于沅水北岸（即今县治），与辰阳之名不符，又适当辰水（又名辰溪）入沅水之口，故易县名为辰溪。同时废静人县（今吉首）人之。南阳郡先后改名寿州、充州。炀帝大业二年（606年）废州复郡。充州、辰州合并为沅陵郡，辖沅陵、辰溪、大乡、盐泉、龙标五县。 唐高祖武德三年（620年）复改沅陵郡为辰州。辰溪析出原义陵县地置溆浦县；析出辰溪、沅陵两县之部分地置麻阳县；原静人县地亦析出归泸溪县管辖：武后垂拱三年（687），复析出所属之今凤凰县地增置渭阳县。辰溪县域大大缩小。宋元明清时期，辰溪先后隶属于辰州泸溪郡、辰州路、辰州府。康熙二十八年（1689）颁发县印时，改「溪」为「-{谿}-」。 民国初沿袭清制，民国3年（1914），辰溪隶属辰沅道。民国11年废道，辰溪直隶于湖南省。民国24年设湘西绥靖处，辰溪属沅辰泸溆督察区。民国25年以后，先后隶第一、第三、第九行政督察区。现隶属怀化市。
