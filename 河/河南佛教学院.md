'''河南佛教学院'''，位于中国[[河南省|河南省]][[南阳市|南阳市]][[桐柏县|桐柏县]]龙潭河风景区内，成立于2011年，占地约40亩。是河南省唯一的一所全日制四年[[本科|本科]]的[[佛教|佛教]]高等院校，创立时据称也是世界上最大的佛教学院。<ref name="dhfx">{{cite web|url=http://newpaper.dahe.cn/hnsb/html/2011-03/01/content_468707.htm|title=河南建世界最大佛学院 4年可拿本科文凭|publisher=[[大河报|大河报]]|date=2011-03-01}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>是中国六大佛教高等院校之一。

==简介==
2005年，经[[国家宗教事务局|国家宗教事务局]]（国宗函〔2005〕191号）、[[河南省人民政府|河南省人民政府]]宗教事务局（豫宗〔2005〕52号）批准，河南佛教学院由河南省佛教协会筹建，由[[中华慈善总会|中华慈善总会]]、河南省慈善总会定点援建，建设资金来源于中华慈善总会名誉副会长、企业家[[王国全|王国全]]个人捐赠的1.5亿元人民币。2004年由[[复旦大学|复旦大学]]设计院完成总体设计。2006年6月第一期工程开工，2010年10月竣工投入使用，2012年4月河南佛教学院举行落成典礼。河南佛教学院建成后，受河南省人民政府宗教事务局行政领导，王国全任该学院的法人代表、常务副院长。依照惯例，河南省佛教协会会长兼任该学院院长，首任院长为河南省佛教协会会长[[释永信|释永信]]<ref>{{cite web|url=http://www.tbxx.net/show.aspx?id=110&cid=5|title=河南佛教学院简介|publisher=桐柏信息|date=2010-10-01}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref>{{Cite web |url = http://www.thepaper.cn/newsDetail_forward_1611115 |title =就释永信被举报问题，调查组负责人接受专访  |author =  |date = 2017-02-03  |publisher =澎湃新闻 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://www.hnfjxy.org/plus/list.php?tid=18 |title = 河南佛教学院简介 |author =  |date =  |publisher = 河南佛教学院 |language =  |accessdate = 2017-02-03 |quote =  }}</ref>。在这里读书，学杂费和食宿全免，每月还有补助、并设有奖学金<ref name="dhfx"/>。
