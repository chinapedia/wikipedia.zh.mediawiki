'''河田末吉'''（{{jpn|j=河田 末吉|hg=かわた すえきち|rm=Kawata Suekichi|f=h}}，{{bd|1901年1月28日|||？}}<ref name="datatakao"/><ref name="世界人名翻译大辞典">{{cite book|title=世界人名翻译大辞典|year=2007|publisher=新华通讯社译名室|location=北京市|isbn=9787500107996|accessdate=2015-07-03|language=zh-cn}}</ref>），[[日本|日本]][[大阪府|大阪府]]人<ref name="臺灣總督府職員錄系統">{{cite web|url=http://who.ith.sinica.edu.tw/corrigendumList.action?xmlId=0001759423&dtdId=000088 |language=zh-tw |title=臺灣總督府職員錄系統 |publisher=中央研究院臺灣史研究所 |accessdate=2015-06-28 |archiveurl=https://web.archive.org/web/20150628070828/http://who.ith.sinica.edu.tw/corrigendumList.action?xmlId=0001759423&dtdId=000088 |archivedate=2015-06-28 |deadurl=no }}</ref>，為[[京都帝國大學|京都帝國大學]]理學部[[物理學|物理學]]博士<ref name="張幸真"/>{{rp|77,119}}。河田末吉曾任第二任[[臺北帝國大學|臺北帝國大學]]物理學講座教授與[[國立臺灣大學物理學系|國立臺灣大學物理學系]]教授。其專長為[[宇宙射線|宇宙射線]]物理、[[結晶|結晶]]物理學以及[[熱力學|熱力學]]<ref name="張幸真"/>{{rp|122}}。

1936年11月，河田末吉成為第二任[[國立臺灣大學物理學系|臺北帝國大學物理學講座]]教授，並領導[[太田賴常|太田賴常]]與內藤實、伊藤貢二位助手在臺灣的[[玉山|玉山]]與[[阿里山山脈|阿里山]]等高山地區的高山營林所，利用探空氣球收集[[中子|中子]]數據以進行[[宇宙射線|宇宙射線]]研究，並持續接受以[[仁科芳雄|仁科芳雄]]為首的{{link-ja|日本學術振興會|日本学術振興会}}原子核小組以及[[帝國學士院|帝國學士院]]等單位的補助。[[臺灣戰後時期|二戰結束後]]，河田末吉等日籍學者被臺大物理系首任系主任[[戴運軌|戴運軌]]留任續聘，並協助重組[[荒勝文策|荒勝文策]]等人製作、曾完成世界第二次人工撞擊[[原子核|原子核]]實驗的考克饒夫－瓦耳頓型加速器，完成了[[中華民國|中華民國]]暨戰後全[[亞洲|亞洲]]首次人工撞擊原子核實驗。1947年，河田末吉受[[二二八事件|二二八事件]]影響而被[[國民政府|國民政府]]遣返日本，並於其後任職於[[京都大學|京都大學]]工學部應用物理學教室，在[[聯合國軍最高司令官總司令部|聯合國軍最高司令官總司令部]]（GHQ）的監督之下與[[東京大學|東京大學]]合作進行[[核物理|核物理]]與宇宙射線物理的研究。

==生平==
===早期===
河田末吉於1901年生於日本[[大阪府|大阪府]][[大阪市|大阪市]]北區<ref name="datatakao"/><ref name="世界人名翻译大辞典"/><ref name="臺灣總督府職員錄系統"/>，並在就讀京都帝國大學物理學系時期，與爾後在多變數[[解析函數|解析函數]]論研究有重大貢獻的[[數學家|數學家]][[岡潔|岡潔]]結交為好友<ref>{{cite journal lua|author=顏一清 |title=多變數解析函數開拓者岡潔(1901.4/19 – 1978.3/1) 一位民族主義數學家 |journal=數學傳播 |volume=20 |issue=4 |publisher=[[中研院|中研院]]數學研究所 |pages=21-32 |url=http://w3.math.sinica.edu.tw/math_media/d204/20403.pdf |accessdate=2015-07-02 |language=zh-tw |archiveurl=https://web.archive.org/web/20150702143543/http://w3.math.sinica.edu.tw/math_media/d204/20403.pdf |archivedate=2015-07-02 |deadurl=no |date=1996年12月 }}</ref>。畢業後，河田末吉進入京都帝國大學物理科第四講座{{notetag|前身為[[新城新藏|新城新藏]]創立的「宇宙物理講座」，石野又吉為其第二任講座教授<ref name="張幸真"/>{{rp|107}}。}}的「石野研究室」，跟隨[[石野又吉|石野又吉]]從事[[X射線|X射線]]實驗，並一路從講師升遷為助理教授<ref name="張幸真"/>{{rp|120}}。在石野又吉於1935年8月退休後，身為助理教授的河田末吉與[[玉成嘉十郎|玉成嘉十郎]]一同暫代其職位，直到次年由[[荒勝文策|荒勝文策]]教授接任之<ref name="張幸真"/>{{rp|107}}。

===臺北帝國大學時期===
[[File:NTU_PhysicsOldBuilding.JPG|200px]]理化學教室校舍，即臺大二號館，物理學講座即設於其中。現內設有[[臺大物理文物廳|臺大物理文物廳]]。]]

1936年（昭和11年）11月，由於荒勝文策轉任京都帝國大學<ref name="科學史通訊">{{cite journal|author=劉廣定 |title=史實再探──六十年前臺灣大學完成的我國第一個核分裂實驗 |url=http://www4.isu.edu.tw/sites/chsas_taipei/doc/add_32.pdf |accessdate=2014-11-02 |journal=科學史通訊 |volume=32 |language=zh-tw |archiveurl=https://web.archive.org/web/20150122090542/http://www4.isu.edu.tw/sites/chsas_taipei/doc/add_32.pdf |archivedate=2015-01-22 |deadurl=yes |date=2009年3月 }}</ref>，河田末吉接手第二任臺北帝國大學物理學講座教授<ref name = "張幸真"/>{{rp|119}}<ref name="中華科技">{{cite journal lua|author=劉廣定 |title=初探民國 34～37 年的中國核子科學 |journal=中華科技史學會會刊 |issue=10 |pages=7-10 |url=http://readopac2.ncl.edu.tw/nclJournal/search/detail.jsp?dtdId=000040&search_type=detail&la=ch&checked=&unchecked=&sysId=0006558169 |accessdate=2015-01-26 |language=zh-tw |archiveurl=https://web.archive.org/web/20150402104521/http://readopac2.ncl.edu.tw/nclJournal/search/detail.jsp?dtdId=000040&search_type=detail&la=ch&checked=&unchecked=&sysId=0006558169 |archivedate=2015-04-02 |deadurl=no |date=2006年12月 }}</ref>。在這段時期，河田末吉主要帶領內藤實與伊藤貢二位助手{{notetag|二位皆自臺北帝國大學物理學講座畢業。}}以及[[太田賴常|太田賴常]]助理教授，在臺灣的[[玉山|玉山]]與[[阿里山山脈|阿里山]]等高山地區的高山營林所利用探空氣球收集[[中子|中子]]數據，借用[[國立臺灣大學醫學院|醫學部]][[竹中繁雄|竹中繁雄]]的設備分析，進行[[宇宙射線|宇宙射線]]研究<ref name="張幸真">{{cite journal zh|author=張幸真 |title=台灣知識社群的轉變－以台北帝國大學物理講座到台灣大學物理系為例 |pages=20 |url=http://ndltd.ncl.edu.tw/cgi-bin/gs32/gsweb.cgi?o=dnclcdr&s=id=%22091NTU00493013%22.&searchmode=basic |accessdate=2012-06-11 |language=zh-tw |archiveurl=https://web.archive.org/web/20150628071851/http://ndltd.ncl.edu.tw/cgi-bin/gs32/gsweb.cgi?o=dnclcdr&s=id%3D%22091NTU00493013%22.&searchmode=basic |archivedate=2015-06-28 |deadurl=no |date=2003-07-31 }}</ref>{{rp|123}}，並持續接受以[[仁科芳雄|仁科芳雄]]為首的{{link-ja|日本學術振興會|日本学術振興会}}原子核小組以及[[帝國學士院|帝國學士院]]等單位的補助<ref name = "張幸真"/>{{rp|122,123}}。河田末吉也曾帶領內藤和伊藤二人利用[[X光|X光]]以單晶[[繞射|繞射]]法分析[[北投石|北投石]]結晶<ref name = "張幸真"/>{{rp|123}}。當時，太田賴常曾向河田末吉提議重建[[荒勝文策|荒勝文策]]團隊的加速器，但該建議並未被採用<ref name = "張幸真"/>{{rp|122}}。

在河田末吉任職臺北帝大期間，曾遭遇理農學部內部對立而分裂為理學部與農學部之事件<ref name = "張幸真"/>{{rp|124}}。但由於其與荒勝文策領導之物理學講座屬於京都帝大系統，而非當時臺大主要的二大派系：[[東京帝國大學|東京帝大]]系統與[[東北帝國大學|東北帝大]]-[[北海道帝國大學|北海道帝大]]系統，因此並未捲入當時二派的激烈鬥爭之中<ref name = "張幸真"/>{{rp|125}}。

而由於宇宙射線研究在[[臺灣總督府|臺灣總督府]]看來對戰爭並無幫助，因此當時河田末吉未被捲入戰時學術動員的浪潮，再加上經費來源充足，河田末吉得以繼續專心進行其純學術研究<ref name = "張幸真"/>{{rp|126,127}}。然而，助理教授太田賴常於昭和16年3月轉任[[臺北高校|臺北高校]]教授物理，繼而與助手伊藤貢一同被總督府工業部[[天然瓦斯|天然瓦斯]]研究所聘任，協助解決戰爭期間的能源問題，接任助理教授的內藤實也在被授予勳八等白色桐葉章後遭軍方強制徵兵，臺北帝國大學物理學講座於是僅存河田末吉一人獨力研究<ref name = "張幸真"/>{{rp|127}}。

===國立臺灣大學時期===
[[File:臺大物理文物廳展覽廳.jpg|200px]]的考克饒夫－瓦耳頓型[[粒子加速器#直線加速器|直線粒子加速器]]。]]
[[臺灣戰後時期|二戰結束後]]，河田末吉教授與[[太田賴常|太田賴常]]副教授等日籍學者被[[臺大物理系|臺大物理系]]首任系主任[[戴運軌|戴運軌]]留任續聘<ref name = "張幸真"/>{{rp|77}}，並協同二人與[[許雲基|許雲基]]、[[許玉釧|許玉釧]]、周木村、林松雲等人，重組[[荒勝文策|荒勝文策]]等人製作、曾完成世界第二次人工撞擊[[原子核|原子核]]實驗的{{le|考克饶夫-瓦尔顿加速器|Cockcroft–Walton generator}}{{notetag|該事件本末請詳見條目[[荒勝文策|荒勝文策]]。}}<ref name="科學史通訊"/><ref name="臺灣物理史數位典藏計畫">{{cite web |url=http://ntur.lib.ntu.edu.tw/bitstream/246246/77859/1/952422H002015.pdf |language=zh-tw |title=臺灣物理史數位典藏計畫 |publisher=[[行政院|行政院]][[國家科學委員會|國家科學委員會]] |date=2007-03-29 |accessdate=2015-01-27 |archiveurl=https://www.webcitation.org/6ZcOMqGof?url=http://ntur.lib.ntu.edu.tw/bitstream/246246/77859/1/952422H002015.pdf |archivedate=2015-06-28 |deadurl=no }}</ref>{{rp|21}}<ref name="Hi! NTU">{{cite book|author=劉子銘 |coauthors=楊松翰、蔡明達、劉建甫 |title=Hi! NTU 解讀臺大的82個密碼 |url=http://books.google.com.tw/books?id=Lxl578rOVhYC&pg=PA149&hl=zh-TW&source=gbs_toc_r&cad=3#v=onepage&q&f=false |year=2010 |publisher=[[國立臺灣大學|國立臺灣大學]]出版中心 |location=[[臺北市|臺北市]] |isbn=978-986-02-2869-4 |date=2010-04 |accessdate=2014-12-03 |language=zh-tw |archiveurl=https://web.archive.org/web/20150128172924/http://books.google.com.tw/books?id=Lxl578rOVhYC&pg=PA149&hl=zh-TW&source=gbs_toc_r&cad=3 |archivedate=2015-01-28 |deadurl=no }}</ref>。然而起初在1946年9月<ref name="中華科技"/>{{rp|8}}，戴運軌諮詢河田末吉相關事宜時，河田末吉曾對重組加速器一事表示消極態度<ref name="中華科技"/>{{rp|13}}。戴運軌事後寫道：
{{cquote|卅五年九月初，鄙人考查原有儀器，即擬定計劃，為欲建立柯克勞夫式之直流高壓電源，以供原子核物理學之研究。乃商請留用之日籍教授河田末吉氏，共同搜集高壓電所需一切儀器。惟河田教授以為儀器太少，恐難以成功，表示消極，鄙人認為電壓雖不能到達預期之高，即略低亦可，故排除一切困難，勉力進行，當時有林嘉仁助教、洪金和技工二人同任搬運試裝之勞。<ref name="中華科技"/>{{rp|13}}}}

爾後在戴運軌與太田賴常等人的堅持與努力之下，重組加速器之計畫得以在該年10月開始進行<ref name="中華科技"/>{{rp|8}}，並於1948年5月13日下午8時35分{{notetag|當時河田末吉本人已返日。}}完成了[[中華民國|中華民國]]暨戰後全亞洲首次人工撞擊原子核實驗<ref name = "張幸真"/>{{rp|161,162}}<ref name="臺灣物理史數位典藏計畫"/>{{rp|4}}<ref name="原子核物理實驗室">{{cite web|url=http://web.phys.ntu.edu.tw/gphyslab/physmu/nuclear.html |title=原子核物理實驗室 |publisher=[[臺大物理文物廳|臺大物理文物廳]] |accessdate=2014-12-02 |language=zh-tw |archiveurl=https://web.archive.org/web/20150604004614/http://web.phys.ntu.edu.tw/gphyslab/physmu/nuclear.html |archivedate=2015-06-04 |deadurl=no }}</ref>。該加速器目前被[[國立臺灣大學|國立臺灣大學]]校方展示於[[臺大物理文物廳|臺大物理文物廳]]，即原物理學講座[[原子核|原子核]]實驗室<ref name ="原子核物理實驗室"/>。

當時，河田末吉與戴運軌一同進行[[計數器|計數器]]之組合與X射線晶體分析研究<ref name = "張幸真"/>{{rp|154}}，內藤實也在此時自[[中國抗日戰爭|中國戰場]]返日，繼續跟隨河田末吉進行研究<ref name = "張幸真"/>{{rp|128}}。另外，河田末吉亦與內藤實和林家仁助教共同規劃大一普物實驗課程之內容<ref name = "張幸真"/>{{rp|155}}。
然而，由於1947年[[二二八事件|二二八事件]]的發生，在臺日籍學者遭當時的[[臺灣省政府|臺灣省政府]]全數遣返，河田末吉等人亦在遣返名單之中<ref name = "張幸真"/>{{rp|77,150}}。該年4月，河田末吉等大批日籍教授被集中於[[基隆|基隆]]的集中營等待船隻接駁，並於10日後在臺灣大學數十名師生的送別之下，搭乘{{link-ja|海王丸|海王丸}}經[[青島|青島]]、[[上海|上海]]等地返回日本<ref name = "張幸真"/>{{rp|150}}。

===晚年===
河田末吉受[[二二八事件|二二八事件]]影響，於1947年5月3日返日，並於其後任職於[[京都大學|京都大學]]工學部應用物理學教室<ref name = "張幸真"/>{{rp|124}}，擔任熱力學教授<ref name = "張幸真"/>{{rp|127}}。河田末吉在臺灣所做的宇宙射線研究，也提供了[[仁科芳雄|仁科芳雄]]等物理學者許多寶貴的數據，並在[[聯合國軍最高司令官總司令部|聯合國軍最高司令官總司令部]]（GHQ）的監督之下與[[東京大學|東京大學]]的研究社群合作進行[[核物理|核物理]]與宇宙射線物理的研究<ref name = "張幸真"/>{{rp|128}}。爾後，河田末吉轉任[[荒勝文策|荒勝文策]]創設的私立[[甲南大學|甲南大學]]理學部，並於1959年成為通過{{link-ja|日本原子能委員會|原子力委員会}}認證的輻射防護主任<ref>{{cite journal lua|title=認定による放射線取扱主任者名 |journal=原子力委員会月報 |volume=4 |issue=6 |publisher=原子力委員会 |url=http://www.aec.go.jp/jicst/NC/about/ugoki/geppou/V04/N06/19590610V04N06.html |accessdate=2015-07-03 |language=ja |archiveurl=https://web.archive.org/web/20140822110124/http://www.aec.go.jp/jicst/NC/about/ugoki/geppou/V04/N06/19590610V04N06.html |archivedate=2014-08-22 |deadurl=no |date=1959年6月 }}</ref>。
