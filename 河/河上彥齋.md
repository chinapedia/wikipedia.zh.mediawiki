'''河上彥齋'''（{{bd|1834年|12月25日|1872年|1月13日}}，[[天保|天保]]五年[[十一月廿五|十一月廿五]] - [[明治|明治]]四年[[十二月初四|十二月初四]]）。[[尊王攘夷|尊王攘夷]]派的日本[[武士|武士]]和[[熊本藩|熊本藩]]士。[[諱|諱]]'''玄明'''。[[幕末四大人斬|幕末四大人斬]]之一。

== 生涯 ==
[[文久|文久]]三年（1863年）30歳時，被同格的幹部[[宮部鼎藏|宮部鼎藏]]選拔為[[熊本藩|熊本藩]]親衛兵。一般被人稱為「人斬彥齋」，彥齋所斬殺過的人物確實有[[佐久間象山|佐久間象山]]，但以後斬殺了幾個人卻沒有明確的流傳下來。其身高五[[丈|丈]]多（150厘米）身材短小但皮膚白皙，據說初次見面的人會誤會他是[[女性|女性]]。而彥齋的[[劍術|劍術]]為[[我流|我流]]，被傳為是單手拔刀的高手（單膝跪在地面上以那樣的低姿勢使出[[逆袈裟斬|逆袈裟斬]]來攻擊）。

不過也有說法認為他修行的[[劍術|劍術]]是[[伯耆流|伯耆流]]。而這個說法的根據，為當時熊本藩最盛行的拔刀術就是伯耆流，並且有人提出彥齋所使用的劍術雖然和伯耆流有些不同但大多有著一樣的招式，所以用這個為理由來證明。但除了在白天公然的暗殺了[[佐久間象山|佐久間象山]]以外，就完全沒有殘留其他關於他使用什麼劍術的跡象。因此他可能是[[幕末四大人斬|幕末四大人斬]]裡面最令人害怕的人。

在「[[八月十八日的政變|八月十八日的政變]]」之後，彥齋前往[[長州|長州]]。在長州擔任[[三條實美|三條實美]]的護衛。[[元治|元治]]元年（1864年）6月的[[池田屋事件|池田屋騷動]]後，彥齋為了要幫在事件中死去的[[宮部鼎藏|宮部鼎藏]]復仇於是再次前往[[京都|京都]]企圖討伐[[新選組|新選組]]。7月11日，彥齋暗殺了[[公武合體|公武合體]]派的開国論者[[佐久間象山|佐久間象山]]。從[[暗殺|暗殺]]象山以後，彥齋就沒再進行暗殺的行動了。

[[幕長戰爭|第二次長州征伐]]時，參加長州軍，為長州軍增加勝利的機會。[[慶應|慶應]]三年（1867年）歸藩，但由於那時熊本藩為[[佐幕|佐幕]]派所掌權，彥齋因此而下獄。所以當[[大政奉還|大政奉還]]、[[王政復古_(日本)|王政復古]]、[[鳥羽伏見之戰|鳥羽伏見之戰]]等重大事件發生時，彥齋都是在獄中度過。慶應四年（1868年）2月彥齋出獄，而佐幕派的熊本藩想利用彥齋踏上[[明治維新|維新]]的浪潮，但遭彥齋的拒絕。

[[明治維新|明治維新]]後，走向[[開国|開国]]政策的新政府相當害怕彥齋會提出[[攘夷|攘夷]]的建議。而且彥齋也被質疑和[[二卿事件|二卿事件]]（意圖推翻[[明治政府|明治政府]]的事件）有所關係，然後又被懷疑是暗殺参議員[[廣澤真臣|廣澤真臣]]的兇手，於是在[[明治|明治]]四年（1871年）12月遭斬首。但是這個暗殺事件和彥齋並無關係，所以一般認為是因為彥齋不遵守明治政府的政策方針，而因此被殺。

彥齋的[[辞世之歌|辞世之歌]]為

 君が為め　死ぬる骸に　草むさば　赤き心の　花や咲くらん 
 君を思い君の御法に死ぬる身を　ゆめ見こりなそつくせ世の人 
 かねてよりなき身と知れど君が世を　思う心ぞ世に残りける

== 逸話 ==
在志士集會的時候，酒席上大家批評了一位幕臣。並且紛紛說出「絕不原諒這傢伙」、「砍了他」等話，而這些喝了酒的志士們因為這樣講著講著而怒氣上身，河上彥齋同時也在那個場合。這時聽到這些話的彥齋暫時離開了座位，然後據說當大家還在講這個話題時，彥齋已經提了這個幕臣的頭回來了。

== 小知識 ==
* 漫畫《[[神劍闖江湖|神劍闖江湖]]》（浪客劍心）的主角[[緋村劍心|緋村劍心]]，即是以河上彥齋為原型所創作出來的人物。
* 在漫畫《[[銀魂|銀魂]]》中，有位幫倒幕派工作的劍客河上萬齊，而他的名字由來就是河上彥齋。
