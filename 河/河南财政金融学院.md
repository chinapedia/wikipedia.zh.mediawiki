'''河南财政金融学院'''，位于[[河南省|河南省]][[郑州市|郑州市]]，于2016年由[[河南财政税务高等专科学校|河南财政税务高等专科学校]]和[[河南教育学院|河南教育学院]]合并组建而成<ref>{{cite news|title=我省将新增3所本科院校 |url=http://www.haedu.gov.cn/2015/10/19/1445222812340.html |language=zh-cn |publisher=河南省教育厅 |date=2015-10-19  }}</ref>。

== 简介 ==
河南财政税务高等专科学校前身是创建于1963年的河南省会计学校，位于郑州市[[金水区|金水区]]农业路23号，[[文革|文革]]爆发后学生造反，校长尹萍、党委书记翟肇基以及部分教师被打成“走资派”遭批斗，受尽各种非人的迫害，学校停止一切教学管理工作，学校先后由校革命委员会、工宣队统管。1970年学校停办，1976年7月复校，1985年经河南省人民政府批准组建成立河南省财政税务专科学校，1992年更名为河南财政税务高等专科学校，2010年10月主校区迁到郑州新区郑开大道北侧。该校隶属[[河南省财政厅|河南省财政厅]]，业务受[[河南省教育厅|河南省教育厅]]领导，是河南最早培养财税、金融、会计等专门人才的学校<ref name="学院">{{cite web |url=http://www.haie.edu.cn/intro.html |language= |title=河南财政金融学院 |publisher=河南财政金融学院 |date= |accessdate=2016-09 |deadurl=yes |archiveurl=https://web.archive.org/web/20161003063825/http://www.haie.edu.cn/intro.html |archivedate=2016年10月3日 |df= }}</ref>。河南教育学院前身是1955年成立的河南省教育行政干部学校，1978年成为河南教育学院，它是[[河南省|河南省]]唯一一所以培养培训中小学[[教师|教师]]、教育行政干部和师范性、应用型本专科人才的成人本科高等[[师范院校|师范院校]]<ref name="简介">{{cite web|url=http://www.haie.edu.cn/ | language= | title=河南教育学院简介 |publisher=河南教育学院 |date= |accessdate=2015-08-08 }}</ref>。

2015年，[[河南省人民政府|河南省人民政府]]向[[中华人民共和国教育部|中华人民共和国教育部]]呈交《河南省人民政府关于申报设置河南财政金融学院的函》（豫政函〔2015〕75号）。2016年3月22日，中华人民共和国教育部下发《教育部关于同意合并建立河南财政金融学院的函》（教发函〔2016〕64号），同意河南财政税务高等专科学校、河南教育学院合并建立河南财政金融学院，同时撤销河南财政税务高等专科学校、河南教育学院的建制<ref>{{Cite web |url = http://www.moe.edu.cn/srcsite/A03/s181/201603/t20160331_236127.html |title = 教育部关于同意合并建立河南财政金融学院的函 |author =  |date =  2016-03-22|publisher =中华人民共和国教育部  |language =  |accessdate =  |quote =  }}</ref>。

2016年5月19日，河南财政金融学院发展战略研讨会暨河南财政金融学院揭牌仪式举行<ref>{{Cite web |url =http://finance.ifeng.com/a/20160519/14398385_0.shtml  |title =河南又添一所本科院校 河南财政金融学院揭牌  |author =  |date = 2016-05-19 |publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref>。2017年2月，河南省人民政府下发关于建立河南财政金融学院的实施意见<ref>{{Cite web |url =  http://henan.sina.com.cn/news/2017-02-04/detail-ifyafcyw0109560.shtml|title = 河南新增一所本科高校 河南财政金融学院获批建立 |author =  |date =2017-02-04  |publisher = 新浪 |language =  |accessdate =  |quote =  }}</ref>。

河南财政金融学院成立后，拥有象湖校区和[[龙子湖高教园区|龙子湖]]校区两个校区，总占地1509亩，建筑面积100.6万平米。有全日制在校生近2万人，教职工931人。设专业70个，以经济学、管理学为主，拥有12门省级精品课程。并建有软件工程实训中心、物流专业实训室等271个校内实训室、242个校外实训基地<ref name="学院"/>。
