'''乌头属'''又名'''附子属'''（[[学名|学名]]：''{{lang|la|Aconitum}}''）是一种属于[[毛茛科|毛茛科]]的开花植物的[[属_(生物)|属]]，这个属有250多个[[种_(生物)|种]]。又名'''萴'''（音同“仄”）。

这些[[多年生植物|多年生植物]]主要生活在北半球山地潮湿的草地上。它们的[[叶子|叶子]]呈深绿色，没有托叶。叶子呈掌状，分五至七瓣。每瓣又分三片，边缘粗糙锋利。叶子在茎上交错或者螺旋排列，下部的叶子有长的柄。

乌头属植物非常美观，在高大直立的主干的顶端有一簇[[总状花序|总状花序]]的蓝色的、紫色的、白色的、黄色的或者粉色的两侧对称的、拥有许多[[雄蕊|雄蕊]]的[[花|花]]。其五片[[花萼|花萼]]中的一片形似一个圆筒状的帽子。每朵花有二至十枚蜜腺状的[[花瓣|花瓣]]。最上面的两个花瓣很大，它们位于帽状的花萼下，有很长的柄。它们的顶端有一个空的针，里面含有[[花蜜|花蜜]]。其它花瓣小或者完全退化了。三至五片[[心皮|心皮]]在底部融在一起。

[[果实|果实]]为[[蓇葖果|蓇葖果]]。

乌头属的学名''Aconitum''可能来自小亚细亚的一座山的名字。在[[希腊神话|希腊神话]]中传说[[海格力斯|海格力斯]]将冥界[[哈得斯|哈得斯]]的三頭犬[[刻耳柏洛斯|刻耳柏洛斯]]捕到地面上。这条恶狗的唾液淌到的地方就长出了乌头。

== 普及 ==
乌头属植物是[[第三纪|第三纪]]时的极地植物，随[[冰川期|冰川期]]从[[西伯利亚|西伯利亚]]传播到欧洲、亚洲和美洲。

== 传粉 ==
传播花粉的昆虫不易接近乌头属的花蜜，只有有很长的虹吸管的[[大黄蜂|大黄蜂]]才能获得它的花蜜，因此两者之间互相配合发展，以至于两者的分布地区一致。

== 药用和毒性 ==
《[[本草纲目|本草纲目]]》记载，乌头“辛、温、有大毒”。乌头属毒性非常高，它们也是非常重要的[[药用植物|药用植物]]。其毒性主要来自[[乌头碱|乌头碱]]。未受伤的[[皮肤|皮肤]]就已经可以吸收乌头碱。皮肤敏感的人可以通过接触而[[过敏|过敏]]。在[[药学|药学]]中主要使用乌头的[[根|根]]，这里的药性浓度非常高。依照2015年版《[[中国药典|中国药典]]》之规范，未经[[炮制|炮制]]的生草乌、生川乌“内服宜慎”；炮制后的乌头属药材，如需内服，应先煎、久煎，以减低毒性，每次用量不得超过3克；即便外用，也应控制药量。

由于乌头属的剧毒，中国政府严禁私人买卖未经炮制的乌头属药材（如草乌、川乌），[[保健食品|保健食品]]中不得添加乌头。然而至今仍经常有擅自烹制乌头药膳或者制作乌头药酒，食用或饮用后导致中毒甚至死亡的案例。
