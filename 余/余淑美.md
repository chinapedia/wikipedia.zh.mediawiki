'''余淑美'''（{{bd|1952年|11月|||}}），[[台灣|台灣]]女性[[分子生物學|分子生物學]]家，[[中央研究院院士|中央研究院院士]]，現擔任[[中央研究院分子生物研究所|中央研究院分子生物研究所]]特聘研究員。她以研究水稻基因而聞名，在台灣學界有「水稻教母」之譽。

== 經歷 ==
余淑美出身[[臺中市|臺中市]][[外埔區|外埔區]][[客家|客家]]農家，是家中的長女<ref>[http://www.taiwan-panorama.com/show_issue.php?id=200929802086c.txt 台灣光華雜誌- 余淑美的水稻情緣]</ref>。[[臺北市立第一女子高級中學|臺北市立第一女子高級中學]]畢業後獲得[[國立中興大學|國立中興大學]][[植物病理學|植物病理學]]學士、碩士。之後前往美國，於1984年獲得[[阿肯色大學|阿肯色大學]]植物病理學及[[植物學|植物學]]博士。獲得博士學位後曾在[[冷泉港實驗室|冷泉港實驗室]]、[[羅徹斯特大學|羅徹斯特大學]]、[[康乃爾大學|康乃爾大學]]進行研究。1989年回到台灣進入中研院分子生物研究所任職至今。

== 研究 ==
余淑美目前的研究方向有植物糖訊息傳遞及基因調控、水稻功能性基因體、植物分子農場。

== 家庭 ==
余淑美的丈夫是中央研究院分子生物研究所研究員[[趙裕展|趙裕展]]，兩人育有一子一女<ref>[https://www.gvm.com.tw/Boardcontent_16518.html 余淑美 為全球糧食未來打拚]</ref>。
