'''端野站'''（{{jpn|j=端野駅|hg=たんのえき|rm=Tanno eki|f=y}}）位於[[北海道|北海道]][[北見市|北見市]]端野町端野，是[[北海道旅客鐵道|北海道旅客鐵道]]（JR北海道）[[石北本線|石北本線]]上的站。

== 車站構造 ==
擁有對向式月台2面2線的地面車站。往來月台間必須利用站內的平交道。

目前是無人站，由[[北見站|北見站]]管理。
沒有設置自動售票機。
雖然站房與北見市商工會（原為端野町商工會）的名產中心併設，
但實際上是在站房旁的新建築物營業。

== 站名由來 ==
[[阿伊努語|阿伊努語]]nup-kes「野地的邊緣」（ヌプ、ケシ，野の端）。

== 車站周邊 ==
原為[[端野町|端野町]]的中心站。
鐵道被夾於國道39號與北海道道1024號川向端野線之間。
車站出入口設於北海道道1024號，若由國道39號必須繞路方可抵達。

* 北見警察署端野駐在所
* 北見市公所端野總合分所（原為端野町公所）
* 端野郵局（與日本郵便北見分店端野集配中心併設）
* 北見信用金庫端野分店
* 北見未來農業協同組合（JA北見未來）端野分所
* 北海道北見巴士「端野郵局」車站

== 历史 ==
* 1912年（大正元年）10月5日 - [[日本國有鐵道|日本國有鐵道]]設立。。
* 1933年（[[昭和|昭和]]8年） - 改建站房。
* 1984年（昭和59年）2月1日 - 廢止貨物與行李處理的業務。
* 1984年（昭和59年）11月10日 - 簡易委託化。
* 1987年（昭和62年）4月1日 - [[國鐵分割民營化|國鐵分割民營化]]後成為北海道旅客鐵道（JR北海道）轄下的車站。
* 1991年（[[平成|平成]]3年）2月 - 改建站房。
* 時期不詳　廢止簡易委託，完全無人化。
