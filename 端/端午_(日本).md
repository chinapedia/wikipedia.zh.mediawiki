'''端午'''（{{lang-ja|端午}}、{{lang|ja|たんご}}）是[[日本|日本]][[大和民族|大和民族]]一個傳統節日，又稱'''端午節'''（{{lang|ja|端午の節句}}、{{lang|ja|たんごのせっく}}）、'''菖蒲節'''（{{lang|ja|菖蒲の節句}}、{{lang|ja|しょうぶのせっく}}），在日本已有悠久的歷史，與[[人日|人日]]、[[上巳|上巳]]、[[日本七夕|七夕]]、[[重陽|重陽]]統稱為「{{link-ja|節句|節句|五節句}}」{{noteTag|有季節特徵的特定傳統節日}}<ref>{{cite web | url = http://iroha-japan.net/iroha/A03_goseku/ | title = 五節句 | publisher = Synergy Marketing, Inc. | accessdate = 2014-05-18}}</ref>。雖然日本的[[端午節|端午節]]是由[[中國|中國]]傳入，但經過多年的發展，已經成為具有當地特色的節日。

[[明治維新|明治維新]]後日本廢除農曆，改用新曆，於是端午節的法定日期也改為新曆5月5日，但部份地區的端午傳統行事仍依照舊曆日期進行。[[日本政府|日本政府]]自1948年起把新曆5月5日定為[[日本兒童節|兒童節]]，列為[[日本公共假日|公眾假期]]，也是[[黃金週_(日本)|黃金週]]長假期的最後一天<ref>{{Cite web |title=ゴールデンウィーク |url=http://iroha-japan.net/iroha/A01_event/07_gw.html |accessdate=2014-05-18}}</ref>。

==歷史==
[[File:田植姿_南秋田・金足.jpg|thumb]]
===起源===
一般認為日本的端午節是始於[[飛鳥時代|飛鳥時代]]由中國[[唐朝|唐朝]]傳入，據《[[日本書紀|日本書紀]]》載，[[推古天皇|推古天皇]]十九年（611年）時的五月五日，推古天皇去獵藥<ref name="推古天皇十九">{{cite book | title = 《日本書紀·推古天皇十九年》 | quote =夏五月五日，藥獵於菟田野。取雞鳴時，集于藤原池上、以會明乃往之。粟田細目臣爲前部領、額田部比羅夫連爲後部領。是日諸臣、服色皆隨冠色各著髻花、則大德小德並用金、大仁小仁用豹尾、大禮以下用鳥尾。}}</ref>，而把此日成為正式節日的最早記載見於於[[養老|養老]]二年（718年）作成、[[天平勝寶|天平勝寶]]九年（757年）实施的《養老令》<ref name="諸節日條">{{cite book | title = 《養老令·雜令四十》諸節日條| quote =凡正月一日。七日。十六日。三月三日。五月五日。七月七日。十一月大嘗日。皆為節日其普賜。臨時聽敕。}}</ref>，明確把五月五日稱為「端午（五）」則始見於《[[續日本後紀|續日本後紀]]》中[[承和_(日本)|承和]]六年的相關內容<ref name="由來">{{Cite web |title=关于中日两国端午节由来的思考|url=http://www.cqvip.com/QK/89748X/201310/47263551.html|accessdate=2014-05-18}}</ref><ref name="承和六年">《續日本後紀·仁和天皇》承和六年五月：「乙酉，是端五之節也，天皇御武德殿，觀騎射。」</ref>。但有研究認為端午節實際傳入日本的時間可能比史書記載為早，有可能早於[[彌生時代|彌生時代]]已由來自中國[[春秋時代|春秋時代]][[吳國|吳]][[越國|越]]兩國的[[渡來人|渡來人]]帶到當地。而日本早於有明文記載端午節之前，當地農民在農曆五月有一個稱為“[[五月忌|五月忌]]”（{{lang|ja|五月忌み}}）的祭祀活動，這是因為五月氣候多雨[[潮濕|潮濕]]，物品容易[[發霉|發霉]]，田間[[害蟲|害蟲]]滋生，同時也是[[蠅|蠅]]患之時，故諸事不吉，人們避免在這天嫁娶、建屋、搬家、造船等<ref name="五月忌み">{{Cite web |title=五月忌み |url=http://vv.81study.com/archives/1010779/ |accessdate=2014-05-18}}</ref>，與中國把五月視為「惡月」、諸事不宜的情況相同。五月忌是一個在[[插秧|插秧]]前的祭{{link-ja|田神|田の神}}儀式，舉行前會選稱為「早乙女」的少女負責祭祀，被選為早乙女的少女要在[[神社|神社]]舉行奉告儀式，然後進入一間屋頂插上[[菖蒲|菖蒲]]、稱為「女之家」（{{lang|ja|女の家}}）或「女之屋根」（{{lang|ja|女の屋根}}）的小屋進行潔淨身心、袚除污穢的程序，包括飲菖蒲酒和[[洗百病|用菖蒲洗澡]]，至插秧祭舉行時才可以出來。由於這節日的習俗行事主要和女性相關，而當時日本的耕種工作多由女性負責，所以這節日也是女性暫停農事，休息潔淨之日子，故五月五日或前一天又稱為「女之夜」（{{lang|ja|女の夜}}），現時部份地區仍有此習之遺風。這習俗與吳越人農耕[[巫俗|巫俗]]儀式相近，故此五月忌可能是吳越移民農耕巫俗儀式的一種變化<ref name="由來" /><ref name="從女人">{{Cite web |title=從女人的節日到男兒的節日--漫談日本的端午節 |url=http://d.wanfangdata.com.cn/periodical_ryzs200107021.aspx |accessdate=2014-05-18}}</ref><ref name="論端午">{{Cite web |title=论端午节在日本的变异——明治以前日本端午的特色|url=http://www.cqvip.com/QK/83215X/200604/23377805.html|accessdate=2014-05-18}} </ref>。

此外，在日本發現的一面製作於彌生時代晚期的銅[[鏡|鏡]]，[[铭文|铭文]]中有“五月丙午之日”之字，而其他用詞又似是當時日本人對[[漢語|漢語]]的錯用，有研究認為端午節本是取五月第一個[[午|午]]日，阳气至极，於是取這天炼[[剑|剑]]、炼镜、采药以禳解阴毒之气，後來才定於五月五日；而該面刻有「五月丙午之日」之鏡則代表於彌生時代晚期，關於「五月午日」的观念意识很可能已传入了日本<ref name="在日本">{{Cite web|title=端午在日本|url=http://lib.cnki.net/cjfd/WSZS200207015.html|accessdate=2014-05-18|deadurl=yes|archiveurl=https://web.archive.org/web/20140502054430/http://lib.cnki.net/cjfd/WSZS200207015.html|archivedate=2014-05-02}}</ref>。

後來五月忌與午日觀念，以及後來模仿中國唐代端午習俗的內容結合，開始形成了具備日本本土特色的端午節<ref name="在日本" /><ref name="傳承">{{Cite web |title=論中國端午節文化在日本的傳承與發展|url=http://big.hi138.com/wenhua/wunhuayanjiu/201101/279241.asp|accessdate=2014-05-18}}</ref>。

===發展===
[[File:薬玉の枇杷_美人花競1.jpg|thumb]]
====飛鳥至平安時代====
飛鳥時代這段時期的端午儀式主要是宮廷貴族進行，主要有采草药、[[騎马|騎马]][[射箭|射箭]]等強身抗病的事项。《日本書紀》載推古天皇當時穿著華麗去菟田野藥獵，所謂药猎，[[江戶時代|江戶時代]][[河村益根|河村益根]]父子的《書紀集解》認為是主要是採藥兼為狩獵，但[[幕末|幕末]]至[[明治|明治]]時代學者{{link-ja|飯田武鄉|飯田武郷}}所著的《日本書紀通釋》則認為是猎取可以成为药材的[[鹿茸|鹿茸]]<ref>《日本書紀通釋》：「關於藥獵，集解上云係採藥草兼為田獵也。其實，主要目的在於夏獵，亦即切取鹿之嫩角也。係依據通證上記載，藥謂鹿茸，月令仲夏鹿角解。別錄曰，四月五月解角，時取陰乾 而來。」</ref>，姑勿論是何種解法，都是受中國於端午採藥以禳解毒气的風俗影響<ref name="在日本" /><ref name="推古天皇十九" />。群臣在雞鳴之時於藤原池集合，天亮進宮，按照{{link-ja|七色十三階冠|七色十三階冠}}{{noteTag|大化三年（647年）制定的各級官員服飾制度。}}制定的冠位規定的服色、冠色簪上髻花，[[大德_(日本)|大德]]{{noteTag|飛鳥時代[[冠位十二階|冠位十二階]]的第一階。}}、{{link-ja|小德 (日本)|小徳|小德}}{{noteTag|飛鳥時代冠位十二階的第三階。}}用[[金|金]]，{{link-ja|大仁 (日本)|大仁|大仁}}{{noteTag|飛鳥時代冠位十二階的第二階。}}、{{link-ja|小仁|小仁}}{{noteTag|飛鳥時代冠位十二階的第四階。}}用豹尾，{{link-ja|大禮 (日本)|大礼|大禮}}{{noteTag|飛鳥時代冠位十二階的第二階。}}以下用鳥尾。翌年又有記載推古天皇於五月五日藥獵，以及與群臣集於羽田，相連參趣於朝，裝束與菟田之獵相同，<ref>《日本書紀·推古天皇二十年》：「夏五月五日、藥獵之、集于羽田、以相連參趣於朝、其裝束如菟田之獵。」</ref>。到[[天智天皇|天智天皇]]時，宮中西小殿有端午宴會，天皇駕臨，當時的[[皇太子|皇太子]][[大海人皇子|大海人皇子]]{{noteTag|實為[[皇太弟|皇太弟]]，後登基為[[天武天皇|天武天皇]]。}}與群臣侍宴和奏田舞<ref>{{cite book | title = 《日本書紀·天智天皇十年》 | quote =五月辛丑（五日），天皇御西小殿，皇太子、群臣侍宴，于是再奏田舞}}</ref>。除宴飲、田舞外，飛鳥時代宮中端午節還會舉行{{link-ja|日本傳統賽馬|古式競馬 (日本)|走馬}}（[[賽馬|賽馬]]），《[[續日本紀|續日本紀]]》載[[大寶_(日本)|大寶]]元年（701年）五月五日，文武天皇命令五位以上的大臣走馬，並親臨觀看<ref name="大寶元年">{{cite book | title = 《續日本紀·大寶元年》|quote= 五月丁丑（五日），令群臣五位以上出走馬。}}</ref>。這段時期的宮中端午宴會曾規定以[[菖蒲|菖蒲]]為髮飾，稱為[[菖蒲鬘|菖蒲鬘]]或[[菖蒲縵|菖蒲縵]]，後來一度廢止<ref name="天平">{{cite book | title = 《續日本紀·元正天皇》|quote= 十九年五月丙子朔庚辰条：「天皇御南苑觀騎射走馬。是曰。太上天皇詔曰。昔者五月之節常用菖蒲爲縵。比來已停此事。從今而後。非菖蒲縵者勿入宮中。」}}</ref><ref name="論日本">{{Cite web |title=論日本對中國端午文化的接受|url=http://d.g.wanfangdata.com.hk/Periodical_gzshkx201303025.aspx|accessdate=2014-05-18}}</ref>。

自[[飛鳥時代|飛鳥時代]]起，日本人到端午節就會繫上[[五彩絲|五彩絲]]（又稱續命縷、長命縷）以抗病，除絲線外還會繫上菖蒲、時花等，這也是來自中國端午節的習俗<ref name="在日本" />。至[[平安時代|平安時代]]，宮中除了賜續命縷外，還有賜{{link-ja|藥玉|薬玉}}，藥玉由續命縷演變而成，本來是一種以菖蒲、艾、雜花等為原料，再繫上續命縷的[[香囊|香囊]]<ref name="藥玉料">{{cite book | title = 《延喜式·左近衛府》|quote=凡五月五日药玉料，菖蒲、艾、杂花十捧，三日平旦申内侍司，列设南殿前。}}</ref>，平安時代初期續命縷和藥玉經常混同，如成書於[[貞觀_(清和天皇)|貞觀]]年間的《儀式》一書中提及的「續命縷」是指藥玉，至[[延喜|延喜]]年間才分成兩種不同事物<ref name="藥玉料" /><ref name="續命縷">{{cite book | title =《延喜式·中務省》|quote=五月五日續命縷料、絲五十絇、紅花大三斤。}}</ref>，再後期藥玉又不再是香囊，演變成一種球形純裝飾品<ref name="薬玉と長命縷">{{cite web | url =http://origamio.com/en/kusudama-lab/research/were-kusudama-and-choumeiru-the-same-things.html  | title =薬玉と長命縷は同じモノか| accessdate = 2014-05-18}}</ref><ref name="在日本" />。當時宮中的一些地方如[[中宮|中宮]]住處之內的柱子也會掛上藥玉，並且會掛到[[重陽節|重陽節]]過後才取下<ref name="中宮">{{cite book | title = 《枕草子》第三十七段|quote=空の気色曇りわたりたるに、中宮などには、縫殿より御薬玉とて色々の糸を組み下げて参らせたれば、御帳立てたる母屋の柱に、左右に付けたり。九月九日の菊を、あやしき生絹の衣に包みて参らせたるを、同じ柱に結ひつけて月ごろある、薬玉に解きかへてぞ棄つめる。また、薬玉は菊の折まであるべきにやあらむ。されど、それは、みな糸を引き取りて、もの結ひなどして、しばしもなし。}}</ref>。藥玉不僅由天皇賜予臣下和用於宮廷，還有賜予各[[寺院|寺院]]<ref name="薬玉を誰に賜">{{cite web | url =http://origamio.com/kusudama-lab/research/whom-was-kusudama-given-to.html | title = 薬玉を誰に賜ったのか| accessdate = 2014-05-18}}</ref>。還有些人會佩上避邪用的[[物忌|物忌]]{{noteTag|以木或草製作的辟邪護符}}<ref name="御節供" />。

平安時代宮中的端午相關儀式其實早於四月二十八日就開始了，隨後的日子有献菖蒲、赐续命缕、骑射、走马、[[杂技|杂技]]、[[奏乐|奏乐]]等活動。四月二十八日先舉行「駒牽」儀式，把五月三至六日騎射和走馬所用、由[[馬寮|馬寮]]飼養與皇族、公卿以及各[[令制國|令制國]]進獻的馬向天皇展示，進獻馬之數目依[[位階|位階]]高低而定。各令制國的将马展示给天皇时必須由[[国司|国司]]亲自牵马，因此也是國司表示保衛和效忠天皇的儀式。騎射、走馬之儀在[[武德殿|武德殿]]覺行，天皇會駕臨觀看，稱為「[[觀馬射式|觀馬射式]]」，如[[仁明天皇|仁明天皇]]在承和六年於[[武德殿|武德殿]]观看騎射<ref name="承和六年" />。這是一個重要的儀式，各官員皆要戴菖蒲鬘出席<ref>{{cite web |url =http://miko.org/~uraki/kuon/furu/text/rituryou/engi/engi11.htm | title = 《延喜式·太政官》|quote=凡五月五日，天皇觀騎射并走馬，辨及史等檢校諸事，所司設御座於武德殿。是日，內外群官皆著菖蒲鬘，諸司各供其職。| accessdate = 2014-05-18}}</ref>。五位以上的官員若未能進獻走馬，則要於四月三十日前申送其狀；進了不堪狀者倘若於端午節當日又進獻馬，就視為落敗<ref>{{cite web |url =http://miko.org/~uraki/kuon/furu/text/rituryou/engi/engi11.htm| title =  《延喜式·太政官》|quote=凡五位已上不堪進五月五日走馬，四月卅日以前申送其狀。已進不堪狀之後，若當日若前日進馬之類，並為負馬。| accessdate = 2014-05-18}}</ref>。仪式结束后就奏起[[日本雅樂|雅樂]]，勝方向天皇献舞谢恩，敗方则将马匹如数献给天皇。此时的骑射已經不單是當初祈雨、健身的原本内涵，還有比武练兵的意義。此外，各貴族府邸也有騎射比賽，其场面之热闹并不亚于宫中<ref name="在日本" /><ref name="論日本" /><ref name="傳承" /><ref name="論端午" />。

平安時代中期的端午節行事已非皇室貴族專美，成为了从宫廷到民间的國家定期节日。平安時代女作家[[清少納言|清少納言]]的[[隨筆|隨筆]][[散文|散文]]集《[[枕草子|枕草子]]》就有記載當時社會各階層過端午的情況，當時上至宮廷，下至民家都争相着在自己家中插上菖蒲、艾草，或用菖蒲作為身上的佩飾<ref name="御殿">{{cite book | title = 《枕草子》第三十七段|quote=節は、五月にしく月はなし。菖蒲、蓬などのかをりあひたる、いみじうをかし。九重の御殿の上をはじめて、言ひ知らぬ民の住家まで、いかでわがもとにしげく葺かむと葺きわたしたる、なほいとめづらし。いつかはこと折に、さはしたりし。}}</ref><ref name="御節供" />。孩子們都跑到街出遊玩，女孩們打扮漂亮，互相比較誰的衣服袖子比較美，調皮的男孩則扯女孩的袖子戲弄她們<ref name="童女">{{cite book | title = 《枕草子》第三十七段|quote= 地歩く童女などの、程々につけて、いみじきわざしたりと思ひて、常に袂まぼり、人のに比べなど、えもいはずと思ひたるなどを、そばへたる小舎人童などに引き張られて泣くも、をかし。}}</ref>。又有人以各種顏色的信紙配上菖蒲或其他當季花草寫信給意中人，如紫色紙裹著[[楝|楝]]花、青色紙包著打結的菖蒲葉、白紙捲著菖蒲根等。收到信的女子想回信，也會和好朋友親密地互相出示來函商量<ref name="紫紙">{{cite book | title = 《枕草子》第三十七段|quote= 紫紙に楝の花、青き紙に菖蒲の葉細く巻きて結ひ、また白き紙を根してひき結ひたるもをかし。いと長き根を、文の中に入れなどしたるを見る心地ども、艶なり。返 事 書かむと言ひ合はせ語らふどちは、見せかはしなどするもいとをかし。人の女、やむごとなき所々に御文など聞こえ給ふ人も、今日は心ことにぞなまめかしき。夕暮れの程に、郭 公の名乗りて渡るも、すべていみじき。」}}</ref>。節俗內容豐富多采，趣味盎然<ref name="在日本" /><ref name="論日本" /><ref name="論端午" />。中國的端午食俗如吃[[粽子|粽子]]、飲菖蒲酒也於平安時代[[日本皇室|皇室]]及貴族階層盛行，吃粽子於[[寬平|寬平]]二年（890年）之前已很普遍，當時的粽子以五色絲線纏裹，故稱「五色粽」<ref>{{cite book | title =  《宇多天皇御記》|quote=宽平二年二月卅日丙午，仰善日，正月十五日七种粥，三月三日桃花饼，五月五日五色粽⋯⋯ 俗间行来，以为岁事，自今以后，每色弁调宜供奉之。」}}</ref><ref name="在日本" />。神社如[[皇大神宮|皇大神宮]]也有端午的相關祭典<ref name="皇大神宮">{{cite book | title =  《皇大神宮儀式帳·一年中行事并月記事》五月例|quote=五日節菖蒲蓬等供奉太神并荒祭宮、月讀宮、瀧原宮、伊雜宮、及諸殿〈仁〉供奉、然則藥御酒神宮并荒祭宮供奉、然後禰宜内人物忌等集御厨院、菖蒲蘰藥酒直會被給畢、即御厨參向〈氐〉大饗被給畢。}}</ref>。

[[鎌倉時代|鎌倉時代]]時，人們注意到「菖蒲」與「尚武」在日語中同音{{noteTag|「菖蒲」、「尚武」、「勝負」日語漢字音讀皆為「{{lang|ja|しょうぶ}}」}}，又注意到菖蒲形狀似[[劍|劍]]，於是菖蒲就不再只是驅邪禳毒之物，而被賦予「尚武」的意義。出現這種轉變，一方面是因為日本的騎射走馬在端午節舉行，尚武思想是繼承平安時代端午比武練兵的傳統；另一方面是因為當時[[武士_(日本)|武士]]勢力崛起，由武士掌權的[[幕府時代|幕府時代]]來臨，成為實際統治者的[[武家|武家]]就把原有的端午節俗注入反映自身生活的內容，稱之為「'''尚武之節'''」，亦因此出現一些新的節俗，舊有的節俗也因為加入了尚武觀念而有所發展<ref name="在日本" /><ref name="論日本" /><ref name="論端午" />。

當時武士階層繼承了前代騎射比武練兵之習，但過程中更重視检验接近于实战要求的骑乘擒拿技巧，即在相互追逐中看谁先把对手拉下马，落馬者常常被摔死或被马踩死，因此較之前代有更鮮明的武鬥色彩。在民間則流行[[打石戰|打石戰]]{{noteTag|日語中有多個名稱，如「石合戰」（{{lang|ja|石合戦}}）、「印地打」（{{lang|ja|印地打}}）、「伊牟地打」（{{lang|ja|伊牟地打}}）、「飞磔」（{{lang|ja|飛磔}}）、石合战（{{lang|ja|石合戦}}）、印阵打（{{lang|ja|印陣打}}）、因阵打（{{lang|ja|因陣打}}）、阴地打（{{lang|ja|陰地打}}）、阴阵打（{{lang|ja|陰陣打}}）、磔合戰（{{lang|ja|磔合戦}}）等<ref name="鬥石考" />。}}，是一個由几十或几百名相邻村落的男孩在野外分阵对垒，互相投掷石块以决胜负的打仗游戏。打石戰本來是對天地陰陽之氣激烈變化的模擬展演，因此會在端午節或其他被視為陰陽激烈變化的時節進行以逐疫祛病，也有“以血驱邪避邪”的意義<ref name="鬥石考">{{Cite journal|author=刘晓峰|title=东亚“克仗斗石”考|journal = 《文化学刊》|date=2007年}}</ref>。打石戰在日本古代同時也是四至五月舉行的[[成男戒|成男戒]]的其中一項需要通過的[[勇气|勇气]]考验{{noteTag|亦称「成年戒」，是日本古代男性成年禮以及取得戰士資格的一種通過儀禮}}。在尚武的氛圍之下，鎌倉時代的端午打石戰更著重「戰鬥」的精神甚於逐疫祛病的初衷<ref name="蒙昧惡俗">{{cite web |url =http://www.th.gov.tw/files/0622_8.pdf |title =從祛疾避疫到狂歡暴動的「蒙昧惡俗」：論臺灣端午節的石戰習俗 |accessdate =2014-05-18 |deadurl =yes |archiveurl =https://web.archive.org/web/20140502194142/http://www.th.gov.tw/files/0622_8.pdf |archivedate =2014-05-02 }}</ref><ref name="印地打">{{cite web |url =http://www.kashiwashobo.co.jp/new_web/column/rensai/r03-44.html |title =端午の節供の印地打 |accessdate =2014-05-18 |deadurl =yes |archiveurl =https://web.archive.org/web/20140502062137/http://www.kashiwashobo.co.jp/new_web/column/rensai/r03-44.html |archivedate =2014-05-02 }}</ref>。

菖蒲在鎌倉時代依然是不可或缺之節物，但它已經成為武力的象徵，而不止是驅瘟避禍。人們插菖蒲之目的也由消極的避邪禳災演變為積極的斬妖除魔，把菖蒲視為斬除妖邪的刀劍<ref name="菖蒲酒">{{cite web |url =http://www.gekkeikan.co.jp/enjoy/culture/summer/summer02.html |title= 菖蒲酒（しょうぶさけ）| accessdate = 2014-05-18}}</ref>。又出現了菖蒲兜{{noteTag|又稱菖蒲胄，「兜」是指「胄」，即古代軍人用的頭盔<ref name="五月人形？">{{cite web |url =http://www.koderaningyo.com/tango/gogatsu/ |title= そもそもなぜ五月人形やこいのぼりを飾るのでしょうか？| accessdate = 2014-05-18}}</ref>。}}，由前代的菖蒲鬘結合[[馬印|馬印]]{{noteTag|作战时立于主将马旁的
一种标志。}}演變而成，是作為祈求男孩健康成長之物，也是送給通過成男戒者的禮物<ref name="論端午" />。男孩子又會玩[[菖蒲打|菖蒲打]]遊戲，是用菖蒲枝擊打地面，以大聲者為勝、先擊斷者為輸的一種遊戲<ref name="菖蒲湯">[http://i-nekko.jp/nenchugyoji/gosekku/tango/shobuyu/ 菖蒲湯]</ref><ref name="端午の節句">{{cite web |url =http://okazarian.web.fc2.com/05_tango.html|title= 端午の節句| accessdate = 2014-05-18}}</ref>。因為盘马弯弓在古代日本是男子之事，尚武氛圍促使端午節從前代以女性為中心的節日轉為以男性為中心<ref name="５月">{{cite web |url =http://www.bite-japan.com/kako/saijiki05-j.html|title= ５月の歳時記| accessdate = 2014-05-18}}</ref>。這時燕子花代表的「勝利」也有了戰爭勝利之意<ref name="大伴家持" />。相對武家與民間越來越盛行的端午行事，此時宮中端午行事日漸衰落，但仍保持騎射儀式以及藥玉、菖蒲的贈答，從鎌倉前期公家{{link-ja|近衛家實|近衛家実}}所著的《{{link-ja|豬隈關白記|猪隈関白記}}》至鎌倉末期的《{{link-ja|花園天皇宸記|花園天皇宸記|花園院宸記}}》都有宮中端午節使用藥玉、菖蒲的記載<ref name="贈答">{{cite web |url =http://origamio.com/kusudama-lab/research/until-when-did-the-custom-to-exchange-kusudama-continue.html |title= 薬玉を贈答する習慣はいつまであったのか| accessdate = 2014-05-18}}</ref>。鎌倉時代末期[[元日戰爭|蒙古入侵日本]]，第一次入侵無功而還，[[镰仓幕府|镰仓幕府]]也加强防禦來抵抗蒙古再次侵襲。[[弘安|弘安]]二年（1279年）的端午節，幕府下令家家户户陈列[[兵器|兵器]]、竖起[[旗幟|旌旗]]以鼓舞士氣，自此端午之日陳兵器、豎軍旗就成為習俗<ref name="論端午" />。亦有另說指豎軍旗的習俗源於[[室町幕府|室町幕府]]建立者[[足利尊氏|足利尊氏]]擊退[[元朝|蒙古]]，並於五月五日統一全國，於是武家社會就紛紛豎軍旗慶祝，成為習俗<ref name="5月5日">{{cite web |url =http://www.ja-aichi.or.jp/pearlrice/saijiki/tango.html |title =5月5日は端午の節句 |accessdate =2014-05-18 |deadurl =yes |archiveurl =https://web.archive.org/web/20100706213049/http://www.ja-aichi.or.jp/pearlrice/saijiki/tango.html |archivedate =2010-07-06 }}</ref>。

[[室町時代|室町時代]]至[[戰國時代_(日本)|戰國時代]]戰亂頻仍，尚武風氣持續，也興起更多與男子、尚武相關的端午習俗。當時製作擺設用冑的材料除菖蒲外還加上了[[檜木|檜木]]，檜木製的冑稱為檜兜。武士會在端午節時於家門前擺飾菖蒲兜和檜兜祈求武運昌隆，並相信這些冑能保護他們，也有把災害轉移到冑身上的意思<ref name="5月5日" />。後來製作擺設用冑的工匠技術越來越精湛，冑就發展成「兜人形」（{{lang|ja|かぶとにんぎょう}}）<ref name="菖蒲酒">{{cite web |url =http://www.gekkeikan.co.jp/enjoy/culture/summer/summer02.html |title= 菖蒲酒（しょうぶさけ）| accessdate = 2014-05-18}}</ref>。[[應永|應永]]至[[嘉吉|嘉吉]]年間由{{link-ja|權官|権官|權}}[[大納言|大納言]][[中山定親|中山定親]]所著的[[日記|日記]]《{{link-ja|薩戒記|薩戒記}}》記載當時宮中保持獻菖蒲和賜菖蒲的傳統<ref name="薩戒記">{{cite book | title =  《薩戒記》|quote= 應永卅三年五月六日己亥、晩頭藏人中務丞重仲〈一臈〉來臨、談曰、昨日自所々所進於内裏之菖蒲在三ケ所、一所者南殿南面、〈簀子上〉一所者御殿東面簀子、一所者内侍所也、入夜主上〈◯稱光〉密々被召具女房出御、件所令取菖蒲給之事、例年之儀也、而今年所置南殿之昌蒲無之、若遲參歟之由被思食、數刻令相待御之處、遂不進之、違例不便之由頗逆鱗、然而彼昌蒲所二候所々不知食、又祗候之輩不二覺悟、仍召衞士被尋問之、申云、所進置御殿、并内侍所之昌蒲者、衞士所持參也、於南殿事者、不存知云々、又申云、可被尋典藥寮歟云々、仍内裏御使則向典藥頭郷成朝臣宅問之、答不存知之由、猶不審之餘、向官外記許、問件所進之司、各申下不覺悟之由、如此間、後日可有御沙汰之由被仰了、件事今日猶八講堂諸人成不審之處、日野中納言談曰、去比自滿濟僧正〈三寶院〉知行所領、可進菖蒲於内裏、可爲何様哉之由、可伺申入道殿之由、彼祗候女房被我、然而付他人可申之由答了、若件事歟云々、其後付藤中納言入道申入道殿無御返事云々、仍不及進入歟、不便々々、予〈◯藤原定親〉案此事、五月三日、六衞府獻昌蒲花之事、見年中行事、是則當時衞士所持來之昌蒲也、又同五日、書司供昌蒲云々、當時如何、又同日典藥寮供昌蒲云々、是近來所進二兩殿之昌蒲也、其體如屋形作之飾昌蒲云々、是昌蒲輿也、抑彼典藥寮所進昌蒲事、寮頭郷成朝臣不覺悟歟、不足言事歟、彼寮頭爲別納、故廣成朝臣女〈◯註略〉知行之、存生之間、毎年進件昌蒲、死去之後、彼所領爲入道内相府殿沙汰、被寄進三條八幡宮滿濟僧正、依爲二彼宮別當知行件領、無左右被寄附寺社了條、不可然事歟、尤可被付二寮頭一事也。}}</ref>，而據《年中定例記》和《成氏年中行事》記載，當時幕府在端午節也有相關儀式，也有使用藥玉，如{{link-ja|伊勢貞陸|伊勢貞陸}}的《簾中舊記》就記載[[足利義政|足利義政]]時幕府端午有使用藥玉<ref>{{cite book | title = 《簾中舊記》|quote= 五月五日の御くすだまは、御所さまへは十二すぢづヽのが參り候、上らふたちより御下までは九すぢにて候、御びでうは六すぢづヽにて候、内裏伏見殿こりやう殿より、大なる御くす玉參。}}</ref>。神社繼續有端午祭典，以五色粽為[[祭品|祭品]]<ref name="鹿島宮">{{cite book | title = 《鹿島宮年中行事·五月》|quote= 五日、大宮祭、大宮司、大禰宜、物忌、今日端午之節會ニハ、御供ニ粽、是ヲ付テ奉備神前、此例、昔高辛氏之惡子、五月五日船ニテ海ヲ渡ルトテ、惡風俄ニ吹出浪沈、水神ト成、常人ヲナヤマス有時、五色之糸ヲ以、粽ヲシテ海中エ提ゲ入ケレバ、青龍ト成リ、其ヨリ海神人ヲナヤマス事ナシ、自爾以降、諸社ニテ粽ヲ神ニ奉リ、君臣上下祝之。}}</ref>前代盛行的打石戰遊戲到這段時期，在一些情況下已發展為成年男性之間具有實戰性質的武鬥，稱為[[菖蒲合戰|菖蒲合戰]]（{{lang|ja|菖蒲合戦}}）。戰況激烈者甚至用到刀劍[[火器|火器]]，常导致血流漂柞、死伤遍野的惨状；又由於端午殺人無罪，有些人就趁機殺人報私仇，因此菖蒲合戰也是解決人際矛盾的一種手段<ref name="論端午" /><ref name="鬥石考" /><ref name="印地打" />。

[[江戶時代|江戶時代]]日本端午節延續尚武之風，人們在屋外繼續掛軍旗、陳兵器。掛軍旗在這時代演變成掛[[鯉魚旗|鯉魚旗]]和{{link-ja|武者繪|武者絵|武者繪幟}}，是出於[[鯉躍龍門|鯉躍龍門]]的中國傳說，又因為鯉魚離水後不會掙扎，從容就死。兩者都是勇敢的象徵，掛鯉魚旗就保佑家中男孩健康成長的意義。這最初是武家的習俗，在江戶中期流傳至[[町人|町人]]家庭和市民階層。武者繪幟則是繪上武將或驅除病魔之神[[鍾馗|鍾馗]]畫像之旗幟，表示希望男孩健康和勇敢<ref name="５月" />，後來又發展出室內用的[[掛軸|掛軸]]式{{link-ja|武者繪|武者絵}}。這段時期仍然盛行打石戰及由此衍生的武鬥習俗，稱為[[菖蒲合戰|菖蒲合戰]]（{{lang|ja|菖蒲合戦}}），由於傷亡慘重，[[德川幕府|德川幕府]]曾多次下令禁止，卻屢禁不止，直至明治時代之後才開始式微<ref name="印地打" /><ref name="論端午" />。

====明治時代之後====
[[File:山本昇雲画「子供遊び」1a.jpg|thumb]]]]
[[File:Japanese_Boys_Festival_1957.jpg|thumb]]的男孩和他的五月人形、甲冑兵器模型，後方為花菖蒲[[掛軸|掛軸]]]]
[[明治維新|明治維新]]之後，日本政府废除农历，端午节也就从农历的五月初五日改为公历5月5日。江戶後期的端午習俗大致保留。1948年，日本把這天定為兒童節，與4月尾至5月初其他節日組成[[黃金週_(日本)|黃金週]][[公眾假期|公眾假期]]，以端午節為最後一天<ref name="淺談">{{cite web |url =http://www.cqvip.com/QK/90473X/201223/43139518.html |title= 浅谈日本的端午文化| accessdate = 2014-05-18}}</ref>。

有部份地區的傳統端午活動則仍然於農曆五月或接近農曆五月的新曆6月舉行。至今日本人仍然十分重視端午節，節日氣氛十分熱鬧，到處都會看見色彩鮮豔的鯉魚旗，商家、政府部门、市民团体、公共机构等都會开展各种各样的應節活动。又因為新曆計算的端午節為兒童節，同時接近[[母親節|母親節]]，也是商家促銷的時機，推出一些節日相關的新型商品，如戰國武士造型的[[巧克力|巧克力]]、鯉魚造型的[[蛋糕|蛋糕]]等吸引孩子的食品，也有櫃檯售賣粽子等傳統端午食品。還有些商家同時推出兒童節和母親節櫃檯，方便人們一同選購端午節、兒童節和母親節商品。不少地方政府部門或[[社區|社區]]會在這天舉辦一些針對兒童或親子的活動，例如[[群馬縣|群馬縣]][[太田市|太田市]]和[[神奈川縣|神奈川縣]][[橫濱市|橫濱市]]的兒童節音樂會（こどもの日コンサート），又有些地區會宣傳保護兒童權益的法規<ref name="淺談" />。

===使用菖蒲及其他當季花草===
[[File:菖蒲湯.png|thumb]]
菖蒲是日本端午節的必備節物，還有艾蒿、蘭草、花菖蒲、燕子花等都是端午節的當季花草。菖蒲在日本不單有禳邪避瘟的作用，其劍狀的外形令人聯想到斬妖除魔，加上其日語讀音與「尚武」、「勝負」相同，更是勇武的象徵。在端午節時，人們把菖蒲和艾蒿插在屋檐上，稱為「軒菖蒲」，或用來鋪屋頂，稱為「菖蒲葺」，睡覺時把菖蒲墊在[[枕頭|枕頭]]下，稱為「菖蒲枕」。又有些人會在農具甚至家畜身上插菖蒲。還有把菖蒲插在頭上或製成菖蒲鬘、菖蒲鉢卷等首服，據說菖蒲鉢卷可以治頭痛，而給男孩子戴菖蒲鉢卷也有尚武的意義。菖蒲也是端午節供奉神明的[[供品|供品]]，全國多個神社都會舉行祈祷消灾祛病和延年益寿的[[菖蒲祭|菖蒲祭]]，祭品為古式{{link-ja|神饌|神饌}}，包括菖蒲與艾草，以及粽子、麻糬等食品。家中的神壇也會放上菖蒲。人們又會喝菖蒲酒和用菖蒲水沐浴，日本把用菖蒲水沐浴稱為[[菖蒲湯|菖蒲湯]]，不但在家中自行用菖蒲煮水沐浴，[[錢湯|錢湯]]在端午都要把菖蒲切成段放进浴池裡供客人泡澡。還有一些用菖蒲玩的節日遊戲，如菖蒲打、菖蒲合戰、[[菖蒲合|菖蒲合]]、[[菖蒲占卜|菖蒲占卜]]等遊戲。菖蒲打通常是男孩玩的遊戲，把菖蒲編成一束，擊打地面，以聲音大者為勝，最先折斷者為輸。菖蒲合戰也是男孩子的遊戲，以菖蒲刀或菖蒲太刀互相擊打的模擬[[格鬥遊戲|格鬥遊戲]]<ref name="太刀" /><ref name="印地打" />。菖蒲合又稱根合，是[[鬥草|鬥草]]的一種，是採摘菖蒲然後比較誰的菖蒲根較長，同時伴以歌詠決定勝負。菖蒲占則是把菖蒲打結，插在屋簷下，用來許願、占卜，如果菖蒲上有蜘蛛結網就是吉兆。艾蒿則常與菖蒲一起使用<ref name="藥物">{{cite web |url =http://jominken.kanagawa-u.ac.jp/triangle/uploads_media/2013/05/ZHOU_Xing_01.pdf |title =东亚的端午：以“药物”为中心 |accessdate =2014-05-18 |deadurl =yes |archiveurl =https://web.archive.org/web/20140502032057/http://jominken.kanagawa-u.ac.jp/triangle/uploads_media/2013/05/ZHOU_Xing_01.pdf |archivedate =2014-05-02 }}</ref><ref name="端午の節句" /><ref name="論日本" /><ref name="在日本" /><ref name="菖蒲湯" />。不少神社如[[春日大社|春日大社]]、[[鶴岡八幡宮|鶴岡八幡宮]]等都會舉辦[[菖蒲祭|菖蒲祭]]神事<ref name="菖蒲祭1">{{cite web |url =https://www.hachimangu.or.jp/shintou/matsuri/cont05.html |title= 菖蒲祭（しょうぶさい）| accessdate = 2014-05-18}}</ref><ref name="菖蒲祭2">{{cite web |url =http://www.kasugataisha.or.jp/calendar/spring05.html|title= 菖蒲祭| accessdate = 2014-05-18}}</ref><ref name="鶴岡">{{cite web |url =http://news.hachimangu.or.jp/dispdtinfo.asp?prm=&M_ID=155&C_ID=3|title= 菖蒲祭| accessdate = 2014-05-18}}</ref>。

[[花菖蒲|花菖蒲]]和[[燕子花|燕子花]]都是端午節當季盛開的花卉，同樣可作藥用，兩者同屬[[鳶尾科|鳶尾科]]，日語把鳶尾科稱為[[溪蓀|溪蓀]]（渓蓀{{lang|ja|あやめ}}），汁液可作為青色染料染布，青色又是祓除邪氣之色，因此以溪蓀汁染布裁成的衣服也是可以避邪之服。燕子花開頭的讀音又日語的「勝利」相通，而溪蓀汁染的衣服又代表神聖的愛情，故又象徵戀愛勝利的祝願，而尚武風氣盛行後，也可以指在鬥爭中獲勝，符合尚武精神<ref name="大伴家持" /><ref name="書き付け花" />。人們在端午節期間除了把溪蓀入藥和作為染料外，也會把它們和其他花草一起用作裝飾，如用作[[華道|華道]]的花材，插好後作為端午節的應節擺飾<ref>{{cite web |url =http://www.about-childrensday.org/tangonosekkuno.html |title= 端午の節句には菖蒲がつきもの| accessdate = 2014-05-18}}</ref>。

===應節裝飾===
[[File:冑に菖蒲の図_里見雲嶺筆.jpg|thumb]]
[[File:五月人形.jpg|thumb]]
[[File:Tango_no_sekku_baby_2.jpg|thumb]]]]
日本人過端午節不但會擺飾以應季花卉為花材的華道作品，還有很多應節的[[裝飾|裝飾]]，稱為[[五月飾|五月飾]]，主要有驅邪和尚武的含義，也是包含對家中男孩健康的祝願<ref name="端午の節句" /><ref name="論日本" /><ref name="在日本" />。五月飾可分為於屋內擺放的屋內飾（屋内飾り）和掛於室外的屋外飾（屋外飾り）。屋內飾主要有五月人形，有男孩的家庭都會在端午節於家中陳設。五月人形的原型都是日本歷史上或傳說中的著名的戰士，人物隨著時代有所增加，包括上古傳說中的[[神武天皇|神武天皇]]、[[神功皇后|神功皇后]]，鎌倉時代的[[源義經|源義經]]、[[武藏坊辨慶|武藏坊辨慶]]，戰國武將[[豐臣秀吉|豐臣秀吉]]、[[加藤清正|加藤清正]]、[[伊達政宗|伊達政宗]]，江戶幕府首位將軍[[德川家康|德川家康]]，以及童話故事中的[[金太郎|金太郎]]和[[桃太郎|桃太郎]]等，這些人偶的服飾一般依照歷史上或傳說中的代表戰鬥服飾製作，並配以武器，造型則是可愛的孩子模樣，故稱為「孩子大將」（{{lang|ja|子供大将}}）、「腕白大將」（{{lang|ja|わんぱく大将}}）{{noteTag|「腕白」（わんぱく）在日語裡是對男孩的一種稱呼，來自[[關白|關白]]一詞。}}。也有些人會在家中陳設仿照各武將所使用的鎧甲製作的模型鎧甲。兩者都是祝願男孩健康成長，成為勇敢的人，並有保佑男孩的意義<ref>{{cite web |url =http://www.kougetsu.co.jp/tangonosekku/ningyou_erabikata.html |title =五月人形 鎧 兜の選び方と買い方～ |accessdate =2014-05-18 }}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref name="五月人形" /> 。放於室內的還有{{link-ja|武者繪|武者絵}}軸，是繪有武將圖像的[[浮世繪|浮世繪]]畫作掛軸<ref>[http://www.musyae.net/sales05.html 室内飾り（軸）]</ref>，現代也有鑲在[[畫框|畫框]]的武者繪<ref>{{cite web |url =http://www.musyae.net/sales06.html |title= 室内飾り（額）| accessdate = 2014-05-18}}</ref><ref name="人形の由来">{{cite web |url =http://www.koboku.co.jp/tango/5-tango/tango-setumei.htm |title =五月人形の由来 |accessdate =2014-05-18 |deadurl =yes |archiveurl =https://web.archive.org/web/20140624141120/http://koboku.co.jp/tango/5-tango/tango-setumei.htm |archivedate =2014-06-24 }}</ref>。還有給男孩穿上[[羽織|陣羽織]]{{noteTag|一種無袖和服外套，常穿在鎧甲外。}}和戴上冑，現代又有以[[報紙|報紙]]摺成冑戴在男孩頭上<ref name="印地打" />。

室外飾主要有武者繪幟和鯉魚旗。武者繪幟是繪上武將圖像的浮世繪旗幟，掛在室外。鯉魚旗是布制或纸制的圆筒空心、上绘鲤鱼图案、可随风飘舞的旗状物。一則是來自[[鯉躍龍門|鯉躍龍門]]的傳說，鯉魚不畏艰险、逆流而上的坚定信念，与来到龙门前奋起一跃时的决然态度，是勇敢的象徵；而鯉魚跳過龍門就化成[[龍|龍]]，有出人頭地的吉祥意義。另外又因為鯉魚離水後不會掙扎，即使放在砧板上也一动不动，甚至身受刀伤也依然不动，这种从容就死的凛然态度，代表危难之际能够镇定自若地献出生命的勇敢精神。故此鯉魚旗寄託了對男孩健康成長、奮發向上，將來出人頭地，成為勇敢戰士的祝願<ref name="５月" /><ref name="論日本" /><ref name="在日本" />。初期的鯉魚旗是繪上鯉魚圖案的旗子，然後演變成鯉魚狀的旗幟，江戶時代多是一條黑色大鯉魚配上五色插在武者繪幟旁邊，到明治時期發展為代表父親的黑色「真鯉」和代表母親的紅色「緋鯉」，大正時代則發展為成套的旗幟，由位置的高低順序為[[風車_(玩具)|風車]]、五色[[風向袋|風向袋]]、真鯉、緋鯉、以及代表兒子的青鯉（子鯉），有多個兒子的家庭會再於青鯉之下加上綠色、紫色等冷色調的鯉魚，青鯉及之下的鯉魚數目與家中兒子的數目相同，於是也有了顯示家中人丁興旺的意思。五色風向袋則源自端午辟邪用的續命縷，又寓意子孫昌盛。現代的鯉魚旗又有再加上暖色鯉魚代表女兒。在藍天上飄揚的鯉魚旗，看上去有如在碧水中暢游的鯉魚，是代表夏季的景物<ref name="菖蒲の節供">{{cite web |url =http://i-nekko.jp/nenchugyoji/gosekku/tango/|title= 端午（たんご）：5月5日　菖蒲の節供| accessdate = 2014-05-18}}</ref><ref name="豆知識">{{cite web |url =http://www.hinakoubou.jp/maycontents/mametishiki.html |title= 五月人形・鯉のぼり豆知識| accessdate = 2014-05-18}}</ref><ref>{{cite web |url =http://www.musyae.com/originkoinobori.html |title=鯉のぼり（鯉幟）の由来、武者のぼりとの関係}}</ref><ref name="端午の節句" /><ref name="論日本" /><ref name="在日本" />。傳統的[[和紙|和紙]]鯉魚旗以[[埼玉縣|埼玉縣]][[加須市|加須市]]的出產的最為著名，在二戰前更是全國產量第一<ref>{{Cite web |url=http://www.koinobori.co.jp/koinobori-yurai.html |title=端午の節句 |accessdate=2014-05-18 |deadurl=yes |archiveurl=https://web.archive.org/web/20140604075259/http://koinobori.co.jp/koinobori-yurai.html |archivedate=2014-06-04 }}</ref><ref name="加須">{{cite web |url =http://www.j-pca.net/locality/history/kazo/index.html|title= 加須とこいのぼり| accessdate = 2014-05-18}}</ref>。

近年隨著附設[[庭園|庭園]]的獨立[[屋|屋]]減少，居於[[住宅大廈|住宅大廈]]裡的家庭增加，不適合於窗外掛大型的五月飾，於是不少室外飾都有室內化的版本，市面就有不少適合放於室內的小型鯉魚旗和武者繪幟售賣，受到居於住宅大廈的日本民眾歡迎，還些人會自行製作室內用鯉魚旗，成為一種普及的[[手工藝|手工藝]]品<ref>{{cite web |url =http://www.egoma.jp/%E3%82%A4%E3%83%99%E3%83%B3%E3%83%88/270/|title= 鯉のぼりの意味や由来！| accessdate = 2014-05-18}}</ref>。

===放風箏===
[[File:浜松まつり.jpg|thumb]]
日本端午節的放風箏習俗起源有多種說法。據江戶時代成書的《{{link-ja|濱松城記|浜松城記}}》載，端午放風箏起源於[[室町時代|室町時代]][[永祿|永祿]]年間，當時[[引間城|引間城]]主[[飯尾連龍|飯尾連龍]]為慶祝長子[[飯尾義廣|飯尾義廣]]出生，就在城內放風箏<ref name="初凧">{{cite web |url =http://hamamatsu-daisuki.net/matsuri/history/|title= 子供の誕生を祝う「初凧」の伝統| accessdate = 2014-12-02}}</ref>，該書又記載[[安土桃山時代|安土桃山時代]]時[[元龜|元龜]]六年（1572年），[[德川家康|德川家康]]的次子[[結城秀康|結城秀康]]出生在，其家臣就在端午節放風箏<ref name="凧揚げ">{{cite web |url =https://www.scm.or.jp/portal/oyumino/story/back-10.html |title =端午の節句と凧揚げ |accessdate =2014-05-18 |deadurl =yes |archiveurl =https://web.archive.org/web/20140502061333/https://www.scm.or.jp/portal/oyumino/story/back-10.html |archivedate =2014-05-02 }}</ref>。但結城秀康於1574年出生，年代有誤，而《濱松城記》錯誤頗多，被視為[[偽書|偽書]]，故書中說法並不可靠<ref name="検証">{{cite web|url =http://plaza.across.or.jp/~shijo-t/kigen2.html|title =『浜松城記』を検証する|accessdate =2014-12-02|deadurl =yes|archiveurl =https://web.archive.org/web/20160304080603/http://plaza.across.or.jp/~shijo-t/kigen2.html|archivedate =2016-03-04}}</ref>。而端午放風箏的現存的最早可靠記載是[[寬政|寬政]]年間，因為濱松一帶端午期間有強風從[[遠江國|遠江國]]吹來，被稱為「遠州之勁風」（{{lang|ja|遠州のからっ風}}），人們就放風箏慶祝兒子誕生<ref name="初凧" />。

此習俗至今仍在關東以西至中部地方流行，而這些地區的男嬰出生後過的第一個端午節，父母都要放風箏祝福兒子健康幸福地成長，稱為「初凧」。一些地區有較大型的端午放風箏活動，例如5月3至5日在[[埼玉縣|埼玉縣]][[春日部市|春日部市]][[江戶川|江戶川]]旁舉行的「放庄和之大風箏祭」（大凧あげ祭り）就有日本最大、達「百疊敷」大的「庄和之大風箏」（{{lang|ja|庄和の大凧}}）<ref name="庄和">{{cite web |url =http://gt.kouryu.or.jp/detail/11214/5494.html|title= 庄和の大凧あげ祭り| accessdate = 2014-05-18}}</ref>，[[靜岡縣|靜岡縣]][[濱松市|濱松市]]在5月3日至5日舉行的{{link-ja|濱松祭|浜松まつり}}有放風箏合戰，[[滋賀縣|滋賀縣]][[東近江市|東近江市]]源於江戶時代端午節習俗、現於5月下旬舉行的八日市放大風箏（八日市大凧揚げ）會放一隻長1.3米、闊1.2米的巨型風箏，並以「近江八日市的放大風箏習俗」（{{lang|ja|近江八日市の大凧揚げ習俗}}）名義列入日本{{link-ja|選擇無形民俗文化財|選択無形民俗文化財}}<ref name="八日市">{{cite web |url =http://www.higasioumi.gaido.jp/kankou/maturi/oodako.html |title =八日市大凧まつり |accessdate =2014-05-18 }}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。其他地區也有各具特式的端午風箏會<ref name="人形の由来" /><ref name="凧揚げ" />。
===賽馬、騎射與射箭===
[[File:Yabusame00.jpg|thumb]]
端午節有不少習俗與馬相關，這是因為「午」代表馬<ref name="藤森">{{cite web |url =http://plaza.yanase.co.jp/kyoto_shiga/yanase-concierge/backnumber200805.html |title =藤森神社 |accessdate =2014-05-18 |deadurl =yes |archiveurl =https://web.archive.org/web/20140502050839/http://plaza.yanase.co.jp/kyoto_shiga/yanase-concierge/backnumber200805.html |archivedate =2014-05-02 }}</ref>，其中最常見的是賽馬和騎射。端午節前後，日本各地不少神社都會有賽馬和流鏑馬的神事，有些依照新曆日期舉行，有些沿用舊曆，也有些在6月接近舊曆端午節的日子進行，全國各地於端午節期間舉行的流鏑馬神事超過30個<ref name="開催日">{{cite web |url =https://sites.google.com/site/yabunavi/home/list2|title= 開催日| accessdate = 2014-05-18}}</ref>。其中[[京都府|京都府]][[京都市|京都市]][[北區_(京都市)|北區]]的[[賀茂別雷神社|賀茂別雷神社]]（通稱[[上賀茂神社|上賀茂神社]]）於新曆5月5日舉行的的{{link-ja|競馬會神事|競馬会神事}}、[[左京区|左京区]]的[[賀茂御祖神社|賀茂御祖神社]]（通稱[[下鴨神社|下鴨神社]]）於新曆5月初舉行的流鏑馬神事以及[[靜岡縣|靜岡縣]][[富士宮市|富士宮市]][[富士山本宮淺間大社|富士山本宮淺間大社]]舉行的流鏑馬祭皆被列為[[世界文化遺產|世界文化遺產]]。上賀茂神社的競馬會神事始於[[寬治|寬治]]七年（1093年），當時[[堀河天皇|堀河天皇]]敕令原來於宮中[[武德殿|武德殿]]舉行的端午賽馬儀式移師上賀茂神社舉行，明治維新後更改日期為新曆5月，在5月1日會先舉行「足汰式」（初賽）。下鴨神社的流鏑馬神事也源自宮中端午行事，自原為舊曆四月酉日舉行的[[葵祭|葵祭]]定為新曆5月15日舉行後，5月5日的流鏑馬神事也被視為葵祭前的儀式<ref name="壱">{{cite web |url =http://www.sekku.or.jp/gogatsungyou1.html |title= 五月人形　壱、五月人形の生い立ちと発展| accessdate = 2014-05-18}}</ref>。淺間大社於5月5日舉行的流鏑馬祭則起源於[[建久|建久]]四年（1193年）端午節[[源賴朝|源賴朝]]在該社進行流鏑馬祈求武運昌隆、天下太平之事，舉行之前一天要先進行富士川川原祓、馬場祓、前日祭、末社巡拜等儀式<ref name="5月の行事">{{cite web |url =http://fuji-hongu.or.jp/sengen/festivals/05_09.html |title= 流鏑馬祭・5月の行事（平成25年）| accessdate = 2014-05-18}}</ref>。而射箭儀式則是藥獵行事的遺風<ref name="皐月">{{cite web |url =http://www.iz2.or.jp/rokushiki/5.html |title= 皐月（五月）| accessdate = 2014-05-18}}</ref>。
===迎田神、插秧祭、女之天下===
[[File:月次風俗図屏風4a_-_複製.jpg|thumb]]
日本農村在古代端午節舉行的迎田神和插秧祭，現代多由神社舉行，稱為{{link-ja|御田植祭|御田植祭}}（お田植え祭），是祈求五穀豐登的祭祀儀式，也是負責農事的女性休息、遊玩之日子，現時一些地區對端午節的別稱如[[愛知縣|愛知縣]]和[[德島縣|德島縣]]稱5月4日為「女之晚」（{{lang|ja|女の晩}}）、[[高知縣|高知縣]]山區稱端午節為「女之天下」（{{lang|ja|女の天下}}），以及女之夜（{{lang|ja|女の夜}}）、女之家（{{lang|ja|女の家}}）、女之宿（{{lang|ja|女の宿}}）、女之屋頂（{{lang|ja|女の屋根}}）等別稱，以及一些地區在端午節以女性為主人、男性為客人，或讓女性優先享用[[菖蒲湯|菖蒲湯]]的習俗，都反映端午為女性休息遊玩日的古風<ref>{{cite web |url =http://www.shitsurai.com/003/002d03.html |title =端午の節供 |accessdate =2014-05-18 |deadurl =yes |archiveurl =https://web.archive.org/web/20140502044710/http://www.shitsurai.com/003/002d03.html |archivedate =2014-05-02 }}</ref><ref name="女の天下">{{cite web |url =http://kotobank.jp/word/%E5%A5%B3%E3%81%AE%E5%A4%A9%E4%B8%8B |title= 女の天下| accessdate = 2014-05-18}}</ref>。

明治維新後，部份神社把端午御田植祭改為新曆，有些則繼續依舊曆端午節日期舉行。現在部份神社的御田植祭，如[[愛媛縣|愛媛縣]][[今治市|今治市]][[大三島町|大三島町]]的[[大山祇神社|大山祇神社]]於舊曆五月初五舉行的御田植祭<ref name="一人角力">{{cite web |url =http://www.shimacon.jp/watch/otaue.html |title= お田植え祭一人角力| accessdate = 2014-05-18}}</ref>、[[東京都|東京都]][[中央區_(東京都)|中央区]][[築地|築地]]的{{link-ja|波除稻荷神社|波除稲荷神社}}於5月6日舉行的御田植祭<ref name="第１3号">{{cite web |url =http://www.namiyoke.or.jp/sazanami13.html |title =【さざなみ】平成1７年皐月８日　第１3号 |accessdate =2014-05-18 |deadurl =yes |archiveurl =https://web.archive.org/web/20140502040209/http://www.namiyoke.or.jp/sazanami13.html |archivedate =2014-05-02 }}</ref><ref name="波除1">{{cite web |url =http://www.namiyoke.or.jp/event_taue.html |title =お田植え祭 |accessdate =2014-05-18 |deadurl =yes |archiveurl =https://web.archive.org/web/20120608053314/http://www.namiyoke.or.jp/event_taue.html |archivedate =2012-06-08 }}</ref>、[[奈良縣|奈良縣]][[吉野郡|吉野郡]][[天川村|天川村]]的{{link-ja|天河大辨財天社|天河大弁財天社}}於新曆5月舉行的御田祭等<ref name="弁財天社">{{cite web |url =http://www.tenkawa-jinja.or.jp/kongetsu/5-2ondasai.html |title =お ん だ 祭 （お 田 植 え 祭） |accessdate =2014-05-18 |deadurl =yes |archiveurl =https://web.archive.org/web/20150127232659/http://www.tenkawa-jinja.or.jp/kongetsu/5-2ondasai.html |archivedate =2015-01-27 }}</ref>，都仍依照古俗選出年輕女性為早乙女來進行儀式。[[鹿兒島縣|鹿兒島縣]][[霧島市|霧島市]][[隼人町|隼人町]]的[[鹿兒島神宮|鹿兒島神宮]]的御田祭除了選早乙女外，還有選年輕男性為早男，與早乙女一起進行儀式<ref name="鹿児島神宮">{{cite web |url =http://www.minc.ne.jp/~hotei/otauekago26.html |title= 御田植祭　（鹿児島神宮）| accessdate = 2014-05-18}}</ref>。有些地區神社雖然有舉行御田植祭，但已經沒有再選早乙女，而是由男性進行儀式<ref name="由來" /><ref name="從女人" />。

===牛休息日===
在[[奈良縣|奈良縣]]，[[岡山縣|岡山縣]]的[[備中國|備中地方]]北部、[[美作國|美作地方]]，[[愛媛縣|愛媛縣]]部份地區，[[山口縣|山口縣]]等地區，端午節期間的五月被視為[[牛|牛]]的休息日。把五月端午節期間列為牛休息日可能是因為五月惡疫流行，令牛病死，故此有了這段時期讓牛休息的習俗。奈良縣和山口縣部份地區在五月不會要牛工作，其他地區的牛在五月的休息日由兩天至一整個月不等，各地讓牛休息的日子皆有不同，有些是一個月內特定數天，有些則是連續多天。例如江戶時代初期成書的[[農書|農書]]《{{link-ja|清良記|清良記}}》就把五月初一、初五、十五、十六、廿九為牛的休息日<ref>{{cite book |title=《清良記·親民鑑月集》|quote=扨五月朔日、五日、十五日、十六日、廿九日、此の五日は牛を休め申日に定めたり。此日牛を遺へば急雨ふらずと古来より申伝。大殿清貞公は此日牛を遣ふたる者はきつく御せっかん有りし。}}</ref>，南宇和郡[[奧野川|奧野川]]一帶則把牛休息日定為五月五日和十六日，[[東宇和郡|東宇和郡]]{{link-ja|城川町|城川町}}則定為五月五日和廿八日，[[宇和島市|宇和島市]]祝森的牛休息日定於五月十六日，[[喜多郡|喜多郡]]{{link-ja|肱川町|肱川町}}予子林在五月初一、初五、十六、廿八讓牛休息。在牛休息期間，人們會幫牛清洗全身，有些地區會給牛享用菖蒲浴。當中不少地區皆認為在牛休息日期間要牛工作會招致不幸，如斷腳、帶來[[旱災|旱災]]、引起[[火災|火災]]等。還有些地區會在端午節期間為牛祈福，祈求神明保佑家中的牛健康<ref name="牛">{{cite web |url =http://ilove.manabi-ehime.jp/system/regional/index.asp?P_MOD=2&P_ECD=2&P_SNO=49&P_FLG1=5&P_FLG2=2&P_FLG3=3&P_FLG4=0 |title =牛馬市と牛の民俗 |accessdate =2014-05-18 |deadurl =yes |archiveurl =https://web.archive.org/web/20140502061326/http://ilove.manabi-ehime.jp/system/regional/index.asp?P_MOD=2&P_ECD=2&P_SNO=49&P_FLG1=5&P_FLG2=2&P_FLG3=3&P_FLG4=0 |archivedate =2014-05-02 }}</ref>。

===模擬戰===
端午節的模擬戰習俗包括上面提及的以菖蒲刀或菖蒲太刀互相擊打的模擬[[格鬥遊戲|格鬥遊戲]]<ref name="太刀" />，在明治時代之前還流行打石戰。打石戰本是為了逐疫祛病，後來從兒童遊戲發展為成年人的武鬥<ref name="印地打" />。自鐮倉時代開始，打石戰成為端午節的主要習俗之一。到了江戶時代，德川幕府因其危險性而屢次禁止，可是成效不彰。直至明治時代仍有些地方保留此習俗，現代已經式微<ref name="印地打" /><ref name="蒙昧惡俗" />。

日本端午節的食品主要有[[粽|粽]]和[[柏餅|柏餅]]，[[關西地區|關西地區]]多於端午節吃粽子，而柏餅則多見於[[關東地區|關東地區]]<ref name="豆知識" />。粽子在日本古代稱為「茅卷」（{{lang|ja|茅巻き，ちがやまき}}），最初用茅葉包裹，呈圓錐形，後來又出現以[[菰|菰]]葉、[[菖蒲|菖蒲]]葉、竹葉、竹[[箬|箬]]、[[蘆葦|蘆葦]]葉等包裹的粽子，不但是應節食品，也是端午祭祀用的供品<ref name="鹿島宮" /><ref>{{cite book |title=《倭名類聚鈔》|quote=粽は、菰の葉で米を包み、灰汁をもってこれを煮て、爛熟せしむるものなり。五月五日にこれを喰う}}</ref>。現時日本各地的粽子多種多樣，如[[靜岡縣|靜岡縣]][[藤枝市|藤枝市]]{{link-ja|朝比奈村 (靜岡縣志太郡)|朝比奈村 (静岡県志太郡) |朝比奈村}}的[[朝比奈粽|朝比奈粽]]、[[新潟縣|新潟縣]]的{{link-ja|笹糰子|笹団子}}和三角粽（{{lang|ja|三角ちまき}}）、[[宮城縣|宮城縣]]和[[福島縣|福島縣]]的大豆三角粽、[[岐阜縣|岐阜縣]][[中津川|中津川]]地方的以[[青茅|青茅]]葉包裹的[[刈安粽|刈安粽]]（{{lang|ja|カリヤすちまき}}）、[[京都府|京都府]]的[[水仙粽|水仙粽]]（{{lang|ja|すいせんちまき}}）、[[羊羹粽|羊羹粽]]（{{lang|ja|ようかんちまき}}）和[[茅卷|茅卷]]、[[長野縣|長野縣]]與[[岐阜縣|岐阜縣]]的[[朴葉卷|朴葉卷]]（ほおばまき）、[[島根縣|島根縣]]的[[笹卷|笹卷]]、鹿兒島縣以[[孟宗竹|孟宗竹]]箬包裹的[[鹼水粽|鹼水粽]]等<ref name="節句考">{{cite web |url =http://washimo-web.jp/Tango/chimaki.htm |title= 端午の節句考| accessdate = 2014-05-18}}</ref><ref>{{cite web |url =http://www.echigo-iwafuneya.com/c-story1.html |title= ちまきアラカルト！～はじめに～| accessdate = 2014-05-18}}<</ref><ref>{{cite web |url =http://www.echigo-iwafuneya.com/c-story3.html |title= ちまきアラカルト！～ちまきの種類～| accessdate = 2014-05-18}}</ref><ref name="論日本" /><ref name="在日本" />。柏餅則是一種以洗淨的[[粳米|粳米]]乾燥後磨成的[[上新粉|上新粉]]{{noteTag|中間粗度的粉。}}製成，內有餡料，再於外面-{}-包上[[槲櫟|槲櫟]]葉的[[和菓子|和菓子]]，由於人們日語稱為「柏」的槲櫟葉是神依附之處，而槲櫟要在新芽長出之後，老葉才會脫落，故有子孫昌盛且繫衍不絕的寓意<ref name="節句考" /><ref>{{cite web |url =http://www.wagasiya.net/manner/tango.html |title= 端午の節句（子供の日）・柏餅（かしわもち）とちまき| accessdate = 2014-05-18}}</ref><ref name="豆知識" /><ref name="人形の由来" /><ref name="菓子">{{cite web |url =http://www.maff.go.jp/j/pr/aff/1205/spe2_02.html |title= 端午の節句のお菓子 / さなぶりのごちそう| accessdate = 2014-05-18}}</ref>。

常見的端午節食品還有[[赤飯|赤飯]]、[[鯛魚|鯛魚]]、[[蝦|蝦]]、[[鯉魚|鯉魚]]、{{link-ja|出世魚|出世魚}}、[[鰹魚|鰹魚]]、[[竹筍|竹筍]]等。赤飯、連頭尾的鯛魚、蝦是日本常見的慶賀場合食品。端午節食用的鯉魚通常烹調成[[糖醋炸鯉魚|糖醋炸鯉魚]]，其寓意與鯉魚旗相同，同樣取自「鯉躍龍門」和「離水鯉魚」之說，祝願男孩健康成長、勇敢無畏、出人頭地。{{link-ja|出世魚|出世魚}}是指成長的不同階段有不同日語名稱的魚，常見的有[[鰤魚|鰤魚]]、[[鱸魚|鱸魚]]、[[鯔魚|鯔魚]]等，出世魚有如古代日本武士也會因應人生不同階段去使用不同的名字，故有祝願男孩平安長大、出人頭地的含義。鰹魚的日語讀音與「勝男」同音，寓意「勝利的男兒」。竹筍是端午節當季的蔬菜，竹子又生長迅速，竹筍料理代表希望男孩快高長大。傳統的端午節句料理其中一種常見組合是插上菖蒲根的蝦湯、糖醋炸鯉魚、鹽燒鯛魚、{{link-ja|煮筍|若竹煮}}、竹內皮拌[[梅乾|梅乾]]和[[青豆|青豆]]飯<ref>{{cite web |url =http://www.hirotaya.com/iware5gatu.htm |title= 知っておきたい　　端午の節句のならわし| accessdate = 2014-05-18}}</ref><ref>{{cite web |url =http://www.miyakan-h.com/news/restaurant/post_504.html |title =端午の節句お祝い料理豆知識 |accessdate =2014-05-18 |archiveurl =https://web.archive.org/web/20140502063529/http://www.miyakan-h.com/news/restaurant/post_504.html |archivedate =2014-05-02 |deadurl =yes }}</ref>。

有些地區還有其他特別的端午節食品。一些地區會在端午節食用[[草餅|草餅]]。[[北海道|北海道]]尤其是南部[[日本海|日本海]]沿岸地區有食用一種以上新粉和砂糖製作再蒸成、稱為{{link-ja|牛餅|べこもち}}的和菓子。[[德島縣|德島縣]]{{link-ja|脇町|脇町}}由於舊曆五月初五正值[[麥子|麥子]]收穫的時節，人們會用收穫的麥子磨成[[麵粉|麵粉]]製作成餅，再用山上採的葉片包裹之，製成稱為[[麥糰子|麥糰子]]（{{lang|ja|麦だんご}}）的應節和菓子。[[宮崎縣|宮崎縣]][[宮崎市|宮崎市]]{{link-ja|佐土原町|佐土原町}}的端午節食品包括{{link-ja|鯨羊羹 (宮崎市)|鯨羊羹 (宮崎市)|鯨羊羹}}，是一種[[鯨魚|鯨魚]]形狀的[[羊羹|羊羹]]，相傳[[元祿|元祿]]年間{{link-ja|佐土原藩|佐土原藩}}[[島津氏|島津氏]]六代藩主{{link-ja|島津惟久|島津惟久}}之繼承人{{link-ja|島津忠雅|島津忠雅}}一出生就顯現王者之風格，島津惟久就命菓匠製作鯨魚形的和菓子，自此就成為佐土原藩端午節的食品<ref>{{cite web |url =http://matome.naver.jp/odai/2136732823937662601 |title= 端午の節句に食べるのは、ちまき・柏餅以外にも全国で色々あるみたい| accessdate = 2014-05-18}}</ref>。[[長崎縣|長崎縣]]有一種叫[[鯉生菓子|鯉生菓子]]的端午菓子，是鯉魚形的米糕<ref>{{cite web |url =http://www18.ocn.ne.jp/~wagasi18/season.html |title= 鯉生菓子| accessdate = 2014-05-18}}</ref>。

==各地特色==
[[File:鶴岡八幡宮菖蒲祭1.jpg|thumb]]
===北海道、東北、關東地方===

在[[北海道|北海道]]及[[东北地区_(日本)|東北地區]]，由於新曆5月仍會下雪，故此該地區的端午節習俗多依照舊曆日期舉行。[[岩手縣|岩手縣]][[盛岡市|盛岡市]][[瀧澤村|瀧澤村]]於每年舊曆五月五日會舉行稱為「{{link-ja|嚓咕嚓咕馬|チャグチャグ馬コ}}」的馬祭，目的是為了在端午插秧之前的農閒期慰勞農耕馬、祈求健康消災。1958年起改為每年新曆6月15日，2001年起再改為每年6月第二個星期六舉行。這天90多匹馬會穿上綴上多個[[鈴|鈴]]的鮮豔裝束，從瀧澤村的[[鬼越蒼前神社|鬼越蒼前神社]]走到[[八幡宮|八幡宮]]，「嚓咕嚓咕」是馬行進時的鈴響聲。有此習俗是因為當地盛產馬匹，1996年被環境省選為「日本音之風景百選」之一<ref>{{cite web |url =http://gigaplus.makeshop.jp/chagmaru/chag_freepaper_vol_1.pdf |title= 《チャグまるしぇ滝沢》，2012年11月號| accessdate = 2014-05-18}}</ref><ref>{{cite web |url =http://morioka.keizai.biz/headline/912/ |title= 岩手の風物詩「チャグチャグ馬コ」、90頭が盛岡八幡宮へ―距離短縮も例年通り開催| accessdate = 2014-05-18}}</ref><ref>{{cite web |url =http://umatohito.com/culture/2012-12-18-1472.html |title= 鬼越蒼前神社　チャグチャグ馬コを見てみよう| accessdate = 2014-05-18}}</ref>。[[花卷市|花卷市]]的[[佛寺|佛寺]][[成島毗沙門堂|成島毗沙門堂]]在5月3至5日會舉行[[全國哭泣相撲大會|全國哭泣相撲大會]]（全国泣き相撲大会），是為了祈祷小孩健康成長所而舉辦的{{link-ja|哭泣相撲|泣き相撲}}比賽。參加者為一歲[[嬰兒|嬰兒]]，分成兩名一組，各由一位[[相撲|相撲]][[力士|力士]]抱著，互相叫喊，先哭者輸。這本來是成年人的[[相撲|相撲]]角力比賽，人們相信贏的一方農作物會豐收，比賽也就逐漸激烈，甚至發生過流血事件，因此改為嬰兒哭啼比賽。比賽中嬰兒的「大哭」、「不理不睬」、「微笑」等各種招式也令[[觀眾|觀眾]]大笑不止<ref>{{cite web |url =http://www.zipangguide.net/travel/sight/iwate/bishamon_temple.html |title= 成島毗沙門堂| accessdate = 2014-05-18}}</ref>。

[[埼玉縣|埼玉縣]][[北葛飾郡|北葛飾郡]][[春日部市|春日部市]][[江戶川|江戶川]]旁於每年5月3—5日舉行「放庄和之大風箏祭」，源於江戶時代當地的[[養蠶|養蠶]]豐收[[占卜|占卜]]和端午慶祝男孩出生的習俗，活動當中的「庄和之大風箏」是日本最大的風箏<ref name="庄和" /><ref name="地方別">{{cite web |url =http://shaddy.jp/kankonsosai/sai/may/may_content.html#collumsekku |title= 地方別・五月節句| accessdate = 2014-05-18}}</ref>。以出產和紙鯉魚旗見稱的[[加須市|加須市]]，每年5月上旬會在[[利根川|利根川]]上空懸掛一條全長111米、重730公斤、全世界最大的鯉魚旗，有如巨鯉魚在空中暢泳<ref name="加須" />。

===中部地方===
[[北陸|北陸]]、[[甲信地方|甲信]]地方的端午節在現代通常定於6月初。[[新潟縣|新潟縣]]民多在6月5至6日過端午節，小孩會玩[[菖蒲打|菖蒲打]]遊戲，有祈求豐收的意義。[[山梨縣|山梨縣]]的端午裝飾是在[[黃金週_(日本)|黃金週]]前擺飾至6月6日後才會收起來，超過一個月<ref name="地方別" />。

[[靜岡縣|靜岡縣]]有在長子出生後的第一個端午節放風箏，現代發展成放風箏合戰，以[[濱松市|濱松市]]在5月3日至5日舉行的{{link-ja|濱松祭|浜松まつり}}以及[[榛原郡|榛原郡]][[相良町|相良町]]的「相良之風箏合戰」最為有名。[[富士宮市|富士宮市]]的[[富士山本宮淺間大社|富士山本宮淺間大社]]在舉行流鏑馬祭起源於[[建久|建久]]四年（1193年）端午節，當時[[源賴朝|源賴朝]]在該社進行流鏑馬以祈求武運昌隆、天下太平，整個儀式武由5月4日開始至5月5日結束，被列為[[世界文化遺產|世界文化遺產]]<ref name="5月の行事" />。
