《'''異世界悠閒農家'''》（{{lang-ja|異世界のんびり農家}}）是{{lang|ja|内藤騎之介}}創作、{{link-ja|やすも}}負責插圖的[[日本|日本]][[輕小說|輕小說]]，2016年12月29日於小說投稿網站“[[成為小說家吧|成為小說家吧]]”發表的[[网络小说|網路連載小說]]。連載初期小說名是'''異世界農家'''，後發現有相同小說名，因此改為'''異世界悠閒農家(異世界のんびり農家)'''<ref>{{cite web|title=内藤騎之介活動報告|url=http://mypage.syosetu.com/mypageblog/view/userid/611609/blogkey/1603034/}}</ref>。

== 簡介 ==
: 主角街尾火樂，大學畢業後於黑心公司就業，由於身體過度操勞工作帶來的傷害，以至於三十歲後就一直躺在醫院病床上，最終享年39歲<ref>{{cite web|title=網路小說 第1話|url=https://ncode.syosetu.com/n3289ds/1/}}</ref>。
: 在主角離世後，出現在眼前的神給了主角轉移而不是轉生到異世界的機會，並詢問主角有什麼願望，主角希望能有具不會生病的身體，因此獲得「健康的身體」；主角希望能轉移稍微沒人的地方，因此轉移到世界最危險的地方「死之森」；主角希望轉移後能從事農業，因此獲得「萬能農具」，可以根據想像自在變行為農業用具。
: 開始異世界第二人生中挖掘、砍伐及耕作的農家生活!

== 登場人物 ==
===大樹村===
====主要角色====
; {{nihongo|街尾火樂|街尾火楽}}
: 主角，種族為人類(?)<ref>{{cite web|title=【登場人物表】種族と名前一覧３|url=https://ncode.syosetu.com/n3289ds/473}}</ref>，本名「街尾火樂（街尾火楽）」，大樹村村長，在轉移後世界是個完全沒有一般常識，外表為一般普通人。
: 轉移異世界後，在世界最危險的地方「死之森」從事農業，一點一滴地從無到有，建立村莊，聚集一群魔物及各式種族共同生活著。
: 使用武器為「萬能農具」，戰鬥力規格外，是個能無視魔法及物理法則武器，能突破龍族與九尾狐等傳說生物的所有結界與障壁，直接擊穿身軀。<ref>{{cite web|title=網路小說 第278話|url=https://ncode.syosetu.com/n3289ds/278/}}</ref>。

: 吸血姬，女主角之一，種族為吸血鬼，種族代表。
: 與村長育有一子一女，擅長魔法道具與醫學。
; {{nihongo|緹雅|ティア}}

: 過後居住在大樹村，負責外交擔當，並與村長育有一女。村長評價其人類形態為很像認真的班長或風紀委員長。

: 與村長育有一女，負責大樹村對外的商業交涉。村長評價其很像運動社團的主將。
