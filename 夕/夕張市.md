'''夕張市'''（{{jpn|j=夕張市|hg=ゆうばりし|rm=Yūbari shi|f=y}}）為位于[[日本|日本]][[北海道|北海道]]中部[[空知綜合振興局|空知綜合振興局]]的城市，近年以生產{{Tsl|ja|夕張メロン|夕張哈密瓜}}而聞名。

轄區位於{{Tsl|ja|夕張山地}}的南端，主要市區位於西部的河谷，及南部{{tsl|ja|夕張川}}沿岸，其餘地區皆為山林地。過去曾經因{{tsl|ja|石狩炭田|石狩煤礦}}而繁榮，在1960年人口數最高曾達到116,908，但之後隨著煤礦產業的衰退人口隨之逐漸減少，目前為全日本人口第二少的城市，僅多於同樣因煤礦產業而沒落的[[歌志內市|歌志內市]]。也因為煤礦產業的萎縮，產業移出後留下來的週邊設施造成市政府的嚴重負擔，市政在1980年代起開始出現財政問題，最終在2007年3月6日被日本政府列為「財政重建團體」。

市名源自[[阿伊努語|阿伊努語]]的「yu-paro」（礦泉湧出的地方）<ref name="brief">{{cite web |url=http://www.city.yubari.hokkaido.jp/contents/compendium/index.html |title=夕張市的概要 |publisher=夕張市政府 |accessdate=2008年1月28日 |language=ja |deadurl=yes |archiveurl=https://web.archive.org/web/20080131211628/http://www.city.yubari.hokkaido.jp/contents/compendium/index.html |archivedate=2008年1月31日 }}</ref>或「i-par」（出口）<ref>{{Cite web|url=http://www.pref.hokkaido.lg.jp/ks/ass/grp/131140P.pdf|title=アイヌ語地名リスト|accessdate=2017-08-17|author=北海道庁|date=|publisher=北海道庁}}</ref>。

== 歷史 ==

1874年由[[開拓使|北海道開拓使]]聘僱的[[美國|美國]][[地質學|地質學]]學者{{tsl|en|Benjamin Smith Lyman|班傑明·史密司·萊曼}}所率領的調查隊來到{{Tsl|ja|夕張川}}上游進行首次的煤礦探勘並確認本地擁有煤礦資源<ref name="brief" />，以及1888年日本工程師[[坂市太郎|坂市太郎]]在此發現了[[露天煤礦|露天煤礦]]之後，[[煤礦|煤礦]]成為帶動夕張地區發展的主要產業。

1890年在本地正式設立'''登川村'''，並開始招募移民進入開墾，1892年正式開始開採煤礦，{{tsl|ja|北海道炭礦鉄道|北海道煤礦鐵道}}也完成了[[追分車站_(北海道)|追分]]至[[夕張車站|夕張]]之間的鐵路（現在的[[石勝線|石勝線]]）。

1893年登川村一度與角田村（現在的[[栗山町|栗山町]]）、長沼村（現在的[[長沼町|長沼町]]）一起被併入由仁村（現在的[[由仁町|由仁町]]），不過僅四年後在1897年7月又恢復獨立設置為'''登川村'''。

隨著人口的成長，登川村在1918年2月改制並改名為'''夕張町'''，1920年人口突破5萬人，1943年4月升格改制為'''夕張市'''，此時夕張市境內共有24個大小煤礦，人口之後也在1960年達到最高峰的116,908人。

但從1965年起，各個煤礦逐一開始停採，使得夕張的發展開始停滯，之後雖然曾規劃以煤礦的歷史為主題來發展觀光產業，設立了包括「煤礦博物館」、「煤礦生活館」、「模擬煤礦」等重現夕張過去歷史和生活的觀光設施，但依然無法重振繁榮。

現在則利用本地氣候的溫差生產哈密瓜，建立夕張哈密瓜之品牌；並引入[[花畑牧場|花畑牧場]]，舉辦{{Tsl|ja|ゆうばり国際ファンタスティック映画祭|夕張國際奇幻電影節}}來刺激觀光發展。

==行政==

現任市長{{tsl|ja|鈴木直道}}原為東京都政府衛生局的職員，2008年由於夕張市政府為了重建財政降低支出，刪減市政府職員後，被[[東京都|東京都]]政府派來支援市政業務，主要負責醫療保險業務，同時也主動協助觀光事務的推展；在原定一年的支援任務期滿後，主動延長支援時間，繼續留在夕張市協助財政重建的工作。

由於這兩年期間的活躍，在2010年3月支援任務結束時，許多市民特地到市政府歡送鈴木直道，並開始出現支持其參選下任夕張市市長的建議。2010年11月，鈴木直道決定辭去東京都政府的工作，參選夕張市市長，並於2011年4月贏得選舉，當時也以30歲的年紀成為全日本最年輕的市長。

== 交通 ==

[[北海道旅客鐵道|北海道旅客鐵道]][[石勝線|石勝線]]以東西向通過轄區南部，並在[[新夕張車站|新夕張車站]]向北分出支線進入夕張市主要市區，但此段支線由於運量過低，北海道旅客鐵道已決定將在2019年4月停止此段支線的營運。

除北海道旅客鐵道外，過去也曾有{{Tsl|ja|夕張鐵道}}和{{Tsl|ja|三菱石炭鉱業|三菱煤礦業}}在此經營{{Tsl|ja|北海道炭礦汽船夕張鉄道線|夕張鐵道線}}和{{Tsl|ja|三菱石炭鉱業大夕張鉄道線|大夕張鐵道線}}兩條鐵路路線，其中夕張鐵道線從夕張市市區往西連結至[[栗山町|栗山町]]，大夕張鐵道線則是自現在的[[清水澤車站|清水澤車站]]往東延伸進入轄內東部山區，並可再連結山區內的{{Tsl|ja|森林鉄道 (大夕張営林署)|森林鐵道 (大夕張營林署)|森林鐵道}}。但這些鐵路自1960年代起陸續停駛，最終在1987年後僅剩下北海道旅客鐵道所經營的鐵路路線。

其中夕張鐵道停止經營鐵路後，則以經營巴士客運為主，目前已「夕張巴士」的品牌經營夕張市內的客運路線及往返[[札幌市|札幌市]]的長途客運。此外{{Tsl|ja|北海道中央バス|北海道中央巴士}}也有經營夕張往返札幌市和[[岩見澤市|岩見澤市]]的長途路線。

由於地理位置的因素，[[道東自動車道|道東自動車道]]以與[[石勝線|石勝線]]相近的路線通過轄區南部，並在新夕張車站附近設有[[夕張交流道|夕張交流道]]。

== 教育 ==

過去夕張市境內曾經有6間高中、10間中學校、27間小學校，但隨著人口減少，學校亦隨之裁撤，目前全市的高中、中學、小學僅都僅剩下一間，包括{{Tsl|ja|北海道夕張高等学校|北海道夕張高等學校}}、{{Tsl|ja|夕張市立夕張中学校|夕張市立夕張中學校|夕張中學校}}和夕張小學校。此外也有一間特殊學校－{{Tsl|ja|北海道夕張高等養護学校|北海道夕張高等養護學校}}

而夕張市也利用1988年與[[東海大學_(日本)|東海大學]]合作，由東海大學在此設立了夕張生物試験農場。
