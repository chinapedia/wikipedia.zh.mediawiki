'''工藤优作'''（{{lang|ja|工藤 優作}}/{{lang|ja|くどう ゆうさく}}，Kudō Yūsaku），是[[日本|日本]][[漫画|漫画]]和[[动画|动画]]作品《[[名侦探柯南|名侦探柯南]]》中的[[名偵探柯南角色列表|虚构角色]]。世界首屈一指的推理小說家，是本作男主角高中生[[偵探|偵探]][[工藤新一|工藤新一]]的父親；暢銷全球的'''《暗夜男爵》'''系列小說是其代表作品，並以'''《緋色的搜查官》'''一劇獲得[[奥斯卡最佳原创剧本奖|奧斯卡最佳劇本獎]]<ref group="參" name="85卷"/>。

名字參考自曾在1979年由日本[[偵探小說|偵探小說]]改編的同名電視劇《{{tsl|ja|探偵物語|偵探物語 (電視劇)}}》中飾演主角'''工藤俊作'''的日本演員[[松田優作|松田'''優作''']]。

== 人物 ==
妻子是曾經風靡全球的女演員[[工藤有希子|工藤有希子]]，與鄰居[[阿笠博士|阿笠博士]]、警視廳警部[[目暮十三|目暮十三]]是老朋友，和'''黑羽盜一'''（初代[[怪盜基德|怪盜基德]]）是亦敵亦友的關係；目前和有希子定居[[洛杉磯|洛杉磯]]，並委託阿笠博士管理原住處。

經常協助警方偵查案件，20年前在查明命案的真相後向有希子求婚<ref group="參">漫畫第26卷，TV 192《生死一瞬間 新一回來了》</ref>；婚後儘管和有希子感情很好，但兩人也會有小爭執的時候，有希子曾幾次賭氣而「離家出走」，但都會被他哄回。

曾偽裝成「暗夜男爵」與妻子有希子一起惡整兒子新一，而這只是測試其偵探能力，最主要的目的是勸他能一起移居洛杉磯，同時也計畫借助自己在[[國際刑警組織|國際刑警組織]]的朋友幫忙調查[[APTX-4869|APTX-4869]]的解藥，讓他能盡快恢復原狀，不料新一因為想靠自己恢復原形（另一方面則是怕小蘭會受到連累）而拒絕<ref group="參">漫畫第6卷，TV 43《江戶川柯南誘拐事件》</ref>。

== 形象 ==
個性沉著穩重，在任何情況下都能保持冷靜，有著英式[[Gentleman_(消歧義)|紳士]]的模樣及風度，偶爾還帶有幽默感。因為外貌英俊而深受女性讀者歡迎；然而他時常因外出旅行，擅自停止寫稿而被各家出版社追稿。

相當信任及理解新一的想法，新一也視其為良師。工藤家族皆知道柯南的真實身分。

推理能力的敏鋭度遠在其子新一之上，同時觀察力極佳，能夠一眼看穿事件真相，是目暮警部、[[阿笠博士|阿笠博士]]遇到瓶颈时咨询的对象；[[灰原哀|灰原哀]]推測新一的推理能力是繼承自其父親<ref group="參">漫畫第37卷，TV 335《東都顯像所的秘密（前篇）》</ref>

== 技能 ==
在各方面能力都很出色，擁有多項交通工具的駕照，曾在[[夏威夷|夏威夷]]教導[[江户川柯南|新一]]射擊<ref group="參" name="M2">M2《[[名侦探柯南_第十四个目标|第十四號獵物]]》</ref>，駕駛直升機<ref group="參" name="M2"/>、快艇<ref group="參">M4《[[名偵探柯南_瞳孔中的暗殺者|瞳孔中的暗殺者]]》</ref>、汽車<ref group="參">M5《[[名偵探柯南_往天國的倒數計時|往天國的倒數計時]]》</ref>、飛機<ref group="參">M8《[[名偵探柯南_銀翼的奇術師|銀翼的奇術師]]》</ref>等等，也曾設計過遊戲場景<ref group="參">M6《[[名偵探柯南_貝克街的亡靈|貝克街的亡靈]]》</ref>。

== 其他 ==
曾經將一名字跡潦草的記者所寫的「怪盜'''1412'''號」讀成「'''KID'''」，意外的成為「怪盜'''基德'''」稱號的由來；也是黑羽盜一之子'''[[黑羽快斗|黑羽快斗]]'''的命名者<ref group="參">漫畫第55卷，TV 473《工藤新一的少年冒險（後篇）》</ref>。
