《'''來戀愛吧'''》是[[豐田實|豐田實]]的[[漫畫|漫畫]]作品。在「[[月刊afternoon|月刊afternoon]]」連載。全5巻。

== 概要 ==
其背景是以作者所住的[[江戶川區|江戶川區]]為原型，所發生在江戶川週圍的純愛[[愛情喜劇|愛情喜劇]]。

== 故事大綱 ==
某高中的1年級生・根岸由美子突然被不認識的同班同學星野一告白。最初由美子是加以拒絶，但在阿一的積極攻勢下而答應。就這樣兩人開始發展了從朋友變成情侶的笨拙愛情生活。升上二年級時，也是在整個過程中一邊發生動盪不安的事情一邊卻又慢慢發展他們的愛情。整個故事也就是從他們一年級到升上三年級之前瘋狂的展開。

== 主要登場人物 ==
;星野 一（{{lang|ja|ほしの はじめ}}）
:主角。是個有點笨拙的角色。個性老實但做事很固執。做事非常認真，不過很欠缺一般的常識。將來的夢想是想要當[[太空人|太空人]]所以預定要到[[美國|美國]]讀大學。作菜好吃到讓敏男說出「你到底是什麼樣的人啊」。輕音樂部的幽靈部員。

;根岸 由美子（{{lang|ja|ねぎし ゆみこ}}）
:女主角。被大家叫作「小根」(除了阿一以外)。相對於阿一和（星野家）是個很突兀的角色。實際上個性有些粗暴（?!）。料理實力是能夠殺死人般的差勁，在最後一集想辦法做出能讓阿一吃的東西。

;目黒 陽子（{{lang|ja|めぐろ ようこ}}）
:從小學時代就和由美往來的朋友。總是有著一副酷樣。一直都會含著一根棒棒糖。常常會給阿一很多奇怪的戀愛建議並且躲在旁邊享受他出糗的樣子。從TRACK15話正式（?）和敏男成為情侶。

;塚原 敏男（{{lang|ja|つかはら としお}}）
:從小學時代就和阿一往來的朋友。總是沉默寡言。在本作中是少數的正常人之一。有著他所說「正在增加百分之20」的體型，不過原因是過去被阿一的老媽強迫吃蛋糕所造成的。不喜歡麻煩，會積極參與有趣的事情。屬於輕音樂部。

;和田（{{lang|ja|わだ}} - ）
:陽子的朋友。對於[[神秘學|神秘學]]和[[魔法|魔法]]有所研究並且也知道當地的靈異地點。會用水晶球、塔羅牌和星座等來幫人占卜。在便當裡會放入來源不明的東西當作食材。角色設定是有點陰暗的個性而且似乎有個哥哥。屬於神秘學研究會。

;羽柴 守／{{lang|ja|ハシバ}}（{{lang|ja|はしば まもる}}）
:阿一的朋友。擅長[[撞球|撞球]]，馬拉松也相當強。屬於輕音樂部。

;水元 義經／{{lang|ja|ヨシツネ}}（{{lang|ja|みずもと よしつね}}）
:阿一的朋友。剛滿十八歲（總之就是能夠對十八禁的東西出手）喜歡女人（這點和智弘很合得來）但是在本作中還沒有女朋友(不過後面喜歡上冴木)，本人說過「自己過著彷彿嚼沙般的高中生活」而且是邊哭邊講。有個在O島（以[[伊豆大島|伊豆大島]]為原型。作者的出生地）開民宿的叔叔。輕音樂部。

;林先生（{{lang|ja|はやし}} - ）
:家庭科老師。從二年級開始擔任（一・由美子）等人的導師。是一個大美人。在作便當比賽時傳授由美子怎麼收服男人胃口的技藝（但因為由美子的恐怖實力而敗北）。酒量很好。

;杉本 凉子（{{lang|ja|すぎもと りょうこ}}）
:喜歡上向由美子告白的阿一，不過卻是一個沒有結果的單戀少女。也被叫作「阿杉」（ケースケ會叫她「小涼」）。個性軟弱，但內心卻相當堅強。相當擅長[[保齡球|保齡球]]。屬於美術部。

;牧村 {{lang|ja|ケースケ}}（{{lang|ja|まきむら}} - ）
:和凉子是青梅竹馬，對凉子單戀。也被叫作「小牧」。個性輕浮，但也有認真的一面。

;星野 零（{{lang|ja|ほしの れい}}）
:阿一的姐姐。個性冰冷。以一臉酷樣在她的學校(女校？)受到女生的歡迎。在阿一的性格形成中或多或少受到她的影響。不輕易表現感情，但實際上是個害羞的人。對於由美子的弟弟・カツヤ（幼兒）一見鍾情。而星野姐弟的名字好像是來自於「二進位思考」（[[二進位|二進位]]只有1和0，所以他們的思考不是對就是錯）。

;屋敷 智弘（{{lang|ja|やしき ともひろ}}）
:是阿一打工的咖啡店「咖啡屋敷」老闆的兒子。和星野性格相反，常常一副慵懶狀而且喜歡女人。在輕音樂部是阿一等人的後輩。

;若葉 大介（{{lang|ja|わかば だいすけ}}）
:和由美子是國中時代的同學（高中是不同校）。並且是同個吹奏樂部。在畢業後的同學會中雖然對由美子告白，但因為由美子已經有了阿一而失敗。由於常常講錯話所以會保持沉默，不過因沉默時的可愛樣子而因此在學妹們間有愛好者俱樂部。和阿一上同樣的補習班。

;冴木 醒（{{lang|ja|さえき せい}}）
:因為阿一的幫忙而和阿一相識，實際上都是同樣年級的學生。樣子有點像是大人。因為有著兩條辮子和眼鏡的標準的班長造型，而被大家選為班長，不過本人倒是很排斥這種事情。在私生活上會打扮的相當流行。做事很有條理而且絕不會欠人家一份情。因為這個原因而嫉妒和阿一很好的由美子，不過後來兩個人的關係總算有變好了。
