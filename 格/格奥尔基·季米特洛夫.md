'''格奥尔基·迪米特罗夫·米哈伊洛夫'''（{{lang-bg|Георги Димитров Михайлов}}；{{bd|1882年|6月18日|1949年|7月2日|catIdx=D}}），通常被称为'''格奥尔基·米哈伊洛维奇·季米特洛夫'''（{{lang-ru|Георгий Михайлович Димитров}}），[[保加利亚共产党|保加利亚共产党]][[保加利亚共产党中央委员会总书记|中央委员会总书记]]和[[保加利亚总理|部长会议主席]]（总理），国际[[共产主义|共产主义]]活动家。

== 早年生涯 ==
格奥尔基·季米特洛夫于1882年6月18日诞生在[[保加利亚|保加利亚]][[拉多米尔|拉多米尔]]县[[科瓦切夫齐|科瓦切夫齐]]村一个小手工业者家庭，有八个兄弟姐妹，12岁就被迫离开学校到[[索非亚|索非亚]]一家很小的[[印刷厂|印刷厂]]当学徒工。1902年他加入[[保加利亚社会民主工党|保加利亚社会民主工党]]，次年该党分裂，他参加[[保加利亚社会民主工党（紧密派社会主义者）|保加利亚社会民主工党（紧密派社会主义者）]]。

== 政治生涯 ==
1909年当选社会民主党中央委员，1913年，季米特洛夫被选为[[保加利亚国民议会|保加利亚国民议会]]议员。1919年5月，[[保加利亚社会民主工党（紧密派社会主义者）|保加利亚社会民主工党（紧密派社会主义者）]]改组为[[保加利亚共产党|保加利亚共产党]]，季米特洛夫继续担任党的领导工作。1923年领导[[九月起义|九月起义]]，失败后[[流亡|流亡]]国外。

== 莱比锡审判 ==
1933年2月27日，震惊世界的“[[国会纵火案|国会纵火案]]”发生。3月9日，纳粹警察局以“参与[[纵火|纵火]]”的罪名逮捕了当时正在[[柏林|柏林]]的季米特洛夫。9月21日，德国在[[莱比锡|莱比锡]]开庭审讯季米特洛夫。季米特洛夫在法庭上宣称纵火案是纳粹的策划。最终，法院宣判季米特洛夫无罪释放。

== 共产国际领袖 ==
1934年2月27日到达苏联，受到[[苏联共产党|苏联共产党]]总书记[[斯大林|斯大林]]接见，经斯大林提议，由季米特洛夫领导[[共产国际|共产国际]]的工作，从1935年至1943年任[[共产国际执行委员会|共产国际执行委员会]]总书记。1943年12月至1945年12月转入[[苏联共产党中央委员会国际部|苏联共产党中央委员会国际部]]。

== 保加利亚领袖 ==
1944年9月9日，保加利亚君主制被推翻，保加利亚工人党和祖国阵线成立了祖国阵线政府。1945年11月4日，季米特洛夫结束流亡返国。1946年9月，保加利亚举行[[全民投票|全民投票]]，废除了[[君主制|君主制]]，建立[[共和政体|共和政体]]，11月，季米特洛夫出任共和国第一任总理。1948年12月，保加利亚工人党改名为保加利亚共产党，季米特洛夫当选为[[保加利亚共产党中央委员会总书记|保加利亚共产党中央委员会总书记]]和[[保加利亚总理|部长会议主席]]，成为保加利亚的最高领导人。

1949年7月2日，季米特洛夫因病医治不愈，在[[莫斯科|莫斯科]]逝世，终年67岁。遺體經防腐處理置於[[索菲亞|索菲亞]]{{link-en|季米特洛夫陵寢|Georgi Dimitrov Mausoleum}}的水晶棺內供人參觀。[[保加利亞共產黨|保加利亞共產黨]]政權瓦解後，陵寢被撤除，遺體下葬普通公墓。
