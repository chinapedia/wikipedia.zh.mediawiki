'''格奈森瑙號戰列艦'''（{{lang-de|Gneisenau}}）是[[納粹德國海軍|納粹德國海軍]]的一艘[[戰列艦|戰列艦]]，為[[沙恩霍斯特级战列舰|沙恩霍斯特级]]的2號艦，在[[第二次世界大戰|第二次世界大戰]]中投入使用。該艦是第3艘被命名為「格奈森瑙號」的軍艦，以紀念[[普魯士王國|普魯士王國]]陸軍元帥—[[奧古斯特·馮·格奈森瑙|奧古斯特·馮·格奈森瑙]]，為繼1879年下水的{{Link-en|格奈森瑙號戰艦 (1879)|SMS Gneisenau (1879)|格奈森瑙號}}和在[[第一次世界大戰|第一次世界大戰]]的[[福克蘭群島戰役|福克蘭群島戰役]]中被擊沉的[[裝甲巡洋艦|裝甲巡洋艦]][[格奈森瑙號戰艦|格奈森瑙號]]後的同名艦。

==設計與建造==
{{main|沙恩霍斯特级战列舰}}
格奈森瑙號為沙恩霍斯特级的2號艦，於1934年2月於[[基爾|基爾]][[德意志造船廠|德意志造船廠]]開始建造，但它的設計一直作修改而在1935年5月重新建造，也延誤了工程進度。當格奈森瑙號建造完成後，它的滿載排水量超過了世界列強於1922年簽訂的[[华盛顿海军条约|华盛顿海军条约]]規定的35,000噸，而德國也並非簽約國<ref>{{en}}{{cite web |url= http://www.scharnhorst-class.dk/technicallayout/generaldetails.html |title= 沙恩霍斯特號和格奈森瑙號 - 設計技術細則 |publisher= scharnhorst-class.dk |accessdate= 2010-04-09}}</ref>。格奈森瑙號配有厚達350毫米的裝甲，相當於當時的現代化戰鬥艦厚度，也超過了英國一戰留下的戰鬥巡洋艦[[卻敵號|卻敵號]]和{{Link-en|聲望號|HMS Renown (1916)}}，以及法國的快速戰鬥艦[[敦克爾克號戰艦|敦克爾克號]]和[[史特拉斯堡號戰艦|史特拉斯堡號]]。格奈森瑙號配有9門口徑280毫米的主砲，雖然因為它的高初速而擁有不錯的穿甲力和射程，但仍無法與當時主力戰鬥艦的380毫米主砲匹敵，它這樣的武裝也是倉促服役所造成的一個後果。
[[File:Bundesarchiv_DVM_10_Bild-23-63-37,_Schlachtschiff_"Gneisenau".jpg|thumb]]
如果格奈森瑙號裝配的是原預定要改裝的6門380毫米主砲，將會是英國皇家海軍的強大對手，其擁有同英國主力艦相等的裝甲，卻有快得多的航速。但格奈森瑙號在設計當時德國海軍並未有380毫米的火砲可裝配，也因此它最後採用280毫米主砲，因為其設計目的本就是攻擊商船，而未打算與敵人主力艦交戰，也因此與較弱的火力相反地，它擁有能脫離戰艦追擊的高航速。由於二戰前設計它的時候受到條約的限制，它仍保留其280毫米的主砲，並打算用在海上破交戰中。
格奈森瑙號與姊妹艦沙恩霍斯特號一般被認為是當時德國設計最優秀的船艦，唯一的缺點的是其乾舷較低，容易在不平靜的海面上將甲板打溼，也因此在1938年冬季調整了其舷弦和安裝了[[大西洋艦艏|大西洋艦艏]]（Atlantic Bow），還曾在1938年8月至11月進行海上戰鬥訓練。

==服役歷史==
1939年9月9日，英國對德宣戰後6天，停泊於[[布倫斯比特爾科格|布倫斯比特爾科格]]（Brunsbüttelkoog）的格奈森瑙號受到[[皇家空軍|皇家空軍]]的轟炸，但無損傷。10月8日，它與[[科隆號輕巡洋艦|科隆號輕巡洋艦]]和9艘驅逐艦組成一支誘餌艦隊去分散盟軍對[[德國號裝甲艦|德國號裝甲艦]]的搜尋注意力。它和[[沙恩霍斯特號戰鬥巡洋艦|沙恩霍斯特號]]被盟軍稱作「蛇蠍姊妹」，因為它們總是一起出擊，並對英國通商船團造成很大的破壞。1939年11月，這兩艘軍艦擊沉了英國偽裝巡洋艦[[拉瓦平迪號|拉瓦平迪號]]（Rawalpindi），但格奈森瑙號於接下來的海上暴風雨中受到了重創<ref>{{en}}{{cite web |url= http://www.scharnhorst-class.dk/gneisenau/gneiseoperationhist.html|title= 格奈森瑙號作戰歷史：北方海域的巡弋 |publisher= scharnhorst-class.dk |accessdate= 2010-04-09}}</ref>。
===威瑟演習作戰===
1940年4月，格奈森瑙號參與了入侵[[丹麥|丹麥]]和[[挪威|挪威]]的[[威瑟演習作戰|威瑟演習作戰]]，並和[[沙恩霍斯特號戰鬥巡洋艦|沙恩霍斯特號]]一起與英艦聲望號交戰，格奈森瑙號的前砲塔和主砲指揮儀受到了損壞，但兩艘德艦很快地從這場戰鬥中脫離了<ref>{{en}}《與敵人更近迫的戰鬥，二次大戰中的皇家海軍》（Engage the Enemy More Closely, the Royal Navy in the Second World War）， Correlli Barnett著，W W Norton & Co Inc出版社（1991），ISBN 034-033-901-2，第111頁</ref>。5月5日，格奈森瑙號於離港口海域21公尺處觸雷（磁性水雷），受到了破壞而喪失航行能力18分鐘，直到5月21日於基爾修復。英軍於6月8日撤出挪威，而當天下午4點，格奈森瑙號和[[沙恩霍斯特號戰艦|沙恩霍斯特號]]撞見了英國艦隊，並展開攻擊，[[英國皇家海軍|英國皇家海軍]]的{{Link-en|光榮號航空母艦|HMS Glorious}}、驅逐艦{{Link-en|阿卡司塔號|HMS Acasta (H09)}}和{{Link-en|熱情號|HMS Ardent (H41)}}均被擊沉，但在被擊沉前，阿卡司塔號仍發射了一枚魚雷重創了[[沙恩霍斯特號戰鬥巡洋艦|沙恩霍斯特號]]，造成船身內部淹水<ref>{{en}}《德國戰鬥艦 1939-45》（German Battleships 1939-45），Gordon Williamson著，Osprey出版社（2008年），ISBN 978-184-176-498-6，第10頁。</ref>。而格奈森瑙號於20日和[[希佩爾海軍上將號重巡洋艦|希佩爾海軍上將號重巡洋艦]]前往[[冰島|冰島]]一帶做佯動誘敵航行，以掩護[[沙恩霍斯特號戰艦|沙恩霍斯特號]]駛回德國的途中，遭到英國潛艇{{Link-en|克萊特號|HMS Clyde (N12)}}的魚雷攻擊受創，因而兩艦也經由挪威[[特隆赫姆|特倫汗]]，同沙恩霍斯特號駛回德國。

===大西洋的戰鬥===
{{main|柏林行動}}
維修完畢後，格奈森瑙號一同與沙恩霍斯特號參加大西洋的破交戰行動，在1941年1月到3月內的[[柏林行動|柏林行動]]中，格奈森瑙號擊沉了14艘船，而沙恩霍斯特號擊沉了8艘，大多來自沒有護航的船團。它們也成功避開了與英國戰鬥艦護航的船團戰鬥<ref>{{en}}{{cite web |url= http://www.scharnhorst-class.dk/scharnhorst/history/scharnnorthernpatrolattack.html |title= 格奈森瑙號作戰歷史：北方海域的巡弋 |publisher= scharnhorst-class.dk |accessdate= 2010-04-09}}</ref>。在這期間，兩艘船一直自[[法國|法國]][[布雷斯特|布雷斯特]]修整後前往大西洋出擊。1941年4月初，英國[[皇家空軍|皇家空軍]]轟炸司令部派出轟炸機轟炸該港，迫使格奈森瑙號離開乾塢，停泊於港內。4月6日，格奈森瑙號被皇家空軍飛行中尉的{{Link-en|康乃斯·坎貝爾|Kenneth Campbell}}的轟炸機擊傷重創，後來就一直於布雷斯特港中整修直到同年12月<ref>{{en}}{{cite web |url= http://www.scharnhorst-class.dk/scharnhorst/history/scharnbrest.html |title= 於法國布雷斯特的格奈森瑙號和沙恩霍斯特號（1941年3月22日—1942年2月11日） |publisher= scharnhorst-class.dk |accessdate= 2010-04-09}}</ref><ref>{{en}}{{cite web |url= http://www.scharnhorst-class.dk/gneisenau/gallery/gallgneisebrest.html|title= 法國布雷斯特的格奈森瑙號 |publisher= scharnhorst-class.dk |accessdate= 2010-04-09}}</ref>。
===海峽衝刺作戰===
{{main|海峽衝刺作戰}}
1942年，格奈森瑙號與沙恩霍斯特號，隨同[[歐根親王號重巡洋艦|歐根親王號重巡洋艦]]，和多艘魚雷艇與驅逐艦的陪同下，執行了[[塞博勒斯行動|塞博勒斯行動]]，要大膽地於白天從法國港口出發，穿越[[英吉利海峽|英吉利海峽]]回到德國。這三艘主要戰艦從激烈地海空戰鬥中逃脫，格奈森瑙號和歐根親王號還重創了英國驅逐艦{{Link-en|烏斯特號驅逐艦|HMS Worcester|烏斯特號}}，格奈森瑙號對它齊射，摧毀了右舷方的艦橋，並使第1和第2鍋爐起火，歐根親王號更擊中了該驅逐艦4次，使它著火。後來格奈森瑙號於荷蘭北部觸雷，又被拖到基爾維修<ref>{{en}}{{cite web |url= http://www.scharnhorst-class.dk/gneisenau/gneiseoperationhist.html |title=格奈森瑙號作戰歷史|publisher= scharnhorst-class.dk |accessdate= 2010-04-09}} </ref>。
===基爾港空襲===
1942年2月26日至2月27日，英國皇家空軍對基爾港—格奈森瑙號因觸雷而維修的港口浮動船塢發動空襲，出動了178架轟炸機，並打到了它的船頭。與平時修理不同，由於原本就預計2星期內就會修完，所以沒有卸載彈藥，因為這次的攻擊而被引爆，格奈森瑙號的整個船頭部份因為爆炸而被毀。經過緊急搶修後，格奈森瑙號卸下它的火藥和砲管於[[格丁尼亚|格丁尼亚]]，並轉入後備役大修。
===重建、拆除與作為阻塞艦===
[[Image:Gdynia_Gneisenau.jpg|thumb]]
雖然格奈森瑙號已於1942年至1944年間維修完畢，但自1943年7月它就被擱置著，沒有投入現役部隊中。在此期間，海軍試著將其280毫米砲塔換成[[俾斯麥級戰艦|俾斯麥級戰艦]]使用的380毫米雙連裝砲塔，還有要將它的船頭延長10公尺，並將它所有105毫米和150毫米的火砲全部換成22門128毫米的火砲。然而在1943年12月沙恩霍斯特號沉沒後該計畫就被放棄了。格奈森瑙號最後在1945年3月23日被鑿沉作為阻塞艦，戰後還被[[波蘭|波蘭]][[打撈|打撈]]了上來，切碎作為廢鐵處理。

==戰後遺物==
[[Image:Gneisenau-1.jpg|thumb]] 
格奈森瑙號其中一支計畫安裝的380毫米火砲現在保存於丹麥的{{Link-en|漢斯索勒姆|Hanstholm}}的一家博物館中。該火砲也僅是德軍的岸防砲台「鐵必制」計畫的其中一部分，其他的第70、71、74和75火砲已於20世紀50年代被摧毀。格奈森瑙號砲塔上的280毫米砲紛紛被拆除，名為「安東」（Anton）的火砲被送往荷蘭，而名為「凱薩」（Cäsar）和「布魯諾」（Bruno）的砲塔和火砲則被送往挪威的岸防砲兵那裡。它的船尾砲塔「凱薩」後來被送到挪威特倫汗附近的[[奧蘭|奧蘭]]一帶，名為奧斯塔特（Austråt）的岸防砲台，今天該處建成了一座博物館，也因此保留了它。而第2座砲塔「布魯諾」則被作為[[卑爾根|卑爾根]]附近的要塞「菲耶爾堡壘」（Fjell fortress）的一部分。另外還有它的150毫米副砲和其雙連裝砲塔至今仍保留在丹麥的史坦佛斯特（Stevnsfort），而「安東」砲塔的部份火砲則保留於荷蘭的[[荷蘭之角|荷蘭之角]]。
