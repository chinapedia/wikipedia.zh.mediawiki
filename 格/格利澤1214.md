'''格利澤1214''' ''([[格利澤近星星表|Gliese-Jahreiss]] 1214)''是在[[蛇夫座|蛇夫座]]的一顆[[視星等|視星等]]14,7等，暗淡的[[恆星分類#_M型|M4.5]]<ref name="SIMBAD-GJ1214"/>[[紅矮星|紅矮星]]<ref name="SIMBAD-GJ1214">{{cite web
  |title=LHS 3275  -- High proper-motion Star 
  |work=[[SIMBAD|SIMBAD]]
  |publisher=[[史特拉斯堡天文資料中心|史特拉斯堡天文資料中心]]
  |url=http://simbad.u-strasbg.fr/simbad/sim-id?Ident=GJ+1214
  |accessdate=2009-12-16}}</ref>。它與[[地球|地球]]的距離約40光年，大小約為[[太陽|太陽]]的一半<ref name="harvardpress"/>，表面溫度約為3000 [[熱力學溫標|K]]<ref name="harvardpress"/> (2700 °C)，光度只有太陽的0.3% <ref name="harvardpress">{{cite web |url=http://www.cfa.harvard.edu/news/2009/pr200924.html |title=Astronomers Find Super-Earth Using Amateur, Off-the-Shelf Technology |author=David A. 這顆恆星的半徑比理論模型估計所預測的大15% <ref name="disco-charbonneau"/>。

==行星系==

在2009年12月中旬，哈佛史密松物理天文台的一組天文學家宣布發現一顆[[系外行星|系外行星]]的伴星，[[GJ_1214_b|GJ 1214 b]]，是一顆有大量的水，和質量與半徑都與地球相近的[[超級地球|超級地球]]<ref name="disco-charbonneau"> {{cite journal |last1=Charbonneau |first1=David |coauthors=Zachory K.
