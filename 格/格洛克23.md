'''格洛克23'''（Glock 23）是一款由[[奧地利|奧地利]]槍械製造商[[格洛克|格洛克]]（Glock）公司所設計及生產的[[手枪|手枪]]，是[[格洛克19|格洛克19]]的[[.40_S&W|.40 S&W]][[口徑|口徑]]版本，標凖彈匣為13發。

== 設計 ==
格洛克23發射[[.40_S&W|.40 S&W]][[彈藥|彈藥]]，裝有改良過的[[手槍套筒|套筒]]及套筒導軌，對應.40 S&W的槍管，外型與[[格洛克19|格洛克19]]非常相似。

格洛克23經歷了四次修改版本，最新的版本稱為'''第四代格洛克23'''（{{Lang-en|4th generation Glock 23}}）。第四代會在[[手槍套筒|套筒]]上型號位置加上“Gen4”以玆識別。

2010年开始，新推出的格洛克23為了大大提高[[人因工程学|人機工效]]，採用了新紋理，握把由粗糙表面改凹陷表面，而握把略為縮小，亦且由過往不能更換改為可以更換握把片（分別是中形和大形，亦可以不裝上握把片直接使用），以調整握把尺寸，更適合不同的手形。[[套筒|套筒]]內部的復進簧改為雙復進簧式設計，大大降低了後坐力和提高了全槍的壽命。為了適應雙復進簧式設計，套筒下的聚合物槍身前端部分較前一代格洛克23略為加寬。亦會有經改進的[[彈匣|彈匣]]設計，以便左右手皆可以直接按下加大化的[[彈匣卡榫|彈匣卡榫]]以更換彈匣，亦可以與舊式彈匣共用，但只可以右手按下彈匣卡榫以更換彈匣。

== 採用 ==
格洛克23是美國境內最受民間及執法部門喜愛的手槍之一，而格洛克也籍著格洛克22和23成為第一間成功打敗史密斯威爾森在市場的位置的公司（1990年計算）。

== 改進型 ==
=== 格洛克23C ===
'''格洛克23C'''和格洛克23的主要分別是23C裝有'''槍口補償裝置'''（'''C'''ompensated），在槍管頂部開兩個橢圓形的孔以利用射擊時排出氣體抑制槍口方向。

=== 格洛克23RTF2 ===
'''格洛克23RTF2'''（{{Lang-en|Glock 23 RTF2}}；RTF2，全稱：{{Lang|en|Rough Texture Frame 2}}，意為：粗糙[[槍身|槍身]]2型）是'''格洛克23'''的表面修改版本。採用了新紋理，握把由粗糙表面改為凹陷表面，和將[[手槍套筒|套筒]]後方的花紋會由直線改為彎線。<ref>[http://hueysgunsight.blogspot.com/2010/03/glock-23-gen-3-rtf2.html Huey's Gunsight—GLOCK 23 Gen 3 RTF2]</ref>
