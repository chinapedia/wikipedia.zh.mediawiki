《'''格伦佐'''》（{{lang|en|Greenzo}}）是[[美国|美国]][[情景喜剧|情景喜剧]]《[[我为喜剧狂|我为喜剧狂]]》[[我为喜剧狂_(第2季)|第2季]]的第5集，也是整个剧集的第26集<ref name="205release">{{cite web|url=http://www.screenrush.co.uk/television/television-813/season-5594/ |title=30 Rock: Episodes Season 2 |publisher=Screenrush (AlloCiné) |accessdate=2013-08-24 |location=London |archiveurl=https://web.archive.org/web/20120730001603/http://www.screenrush.co.uk/television/television-813/season-5594/ |archivedate=2012-07-30 |deadurl=no }}</ref>。由乔恩·波拉克编剧，唐·斯卡迪诺执导<ref name="205episode">{{cite episode |title=Greenzo |episodelink=格伦佐 |series=30 Rock |serieslink=我为喜剧狂 |credits=Jon Pollack (writer); Don Scardino (director) |network=NBC Universal |station=NBC |airdate=2007-11-08 |season=2 |number=5}}</ref>，2007年11月8日通过[[全国广播公司|全国广播公司]]在[[美国电视|美国]][[首演|首播]]<ref name="205release" />。[[友情客串|客串出演]]本集的演员包括凯文·布朗、格里兹·查普曼、约翰·卢兹、马里克·潘考利、[[大卫·史威默|大卫·史威默]]、{{link-en|麦迪逊·麦金利|Madison McKinley}}<ref>{{cite web |url=http://starcasm.net/wp-content/uploads/2011/01/Madison_Garton_star_magazine.jpg |title=Rising Star Madison McKinley |accessdate=2013-08-31}}</ref><ref>{{cite web |url=http://madisonmckinley.com/media/ |title=Madison McKinley Media |accessdate=2013-08-31 |deadurl=yes |archiveurl=https://web.archive.org/web/20130831044924/http://madisonmckinley.com/media/ |archivedate=2013年8月31日 |df= }}</ref>、宝拉·佩尔（{{lang|en|Paula Pell}}和迪翁·萨普（{{lang|en|Dion Sapp}}），此外梅莉迪丝·维埃拉和美国前[[美国副总统|副总统]][[阿尔·戈尔|阿尔·戈尔]]都在节目里以自己的真实身份出镜。

在这一集中，[[通用电气|通用电气]]副总裁杰克·唐纳吉（{{lang|en|Jack Donaghy}}，[[亚历克·鲍德温|亚历克·鲍德温]]饰）请来一位名叫杰拉德（{{lang|en|Jared}}）的男演员扮演格伦佐（{{lang|en|Greenzo}}），这是“美国首位不会评判他人，对商务活动态度友好的环保倡导者”。但后者的表演却很快演变成一场灾难。全国广播公司杂工跟班肯尼斯·帕塞尔 （{{lang|en|Kenneth Parcell}}，[[杰克·麦克布瑞尔|杰克·麦克布瑞尔]]饰）决定举办一场聚会，但首席编剧丽兹·莱蒙（{{lang|en|Liz Lemon}}，[[蒂娜·菲|蒂娜·菲]]饰）知道没有人会出席，于是和男演员崔西·乔丹（{{lang|en|Tracy Jordan}}，[[崔西·摩根|崔西·摩根]]饰）决定一起“想办法”让这次聚会成功。女演员简娜·玛隆妮（{{lang|en|Jenna Maroney}}，{{link-en|简·科拉克斯基|Jane Krakowski}}饰）和丽兹都怀疑[[制片人|制片人]]皮特·霍恩伯格（{{lang|en|Pete Hornberger}}，{{link-en|斯科特·安第斯|Scott Adsit}}饰）有[[外遇|外遇]]。<ref name="205episode" />

== 剧情 ==
打算[[退休|退休]]的通用电气公司[[首席执行官|首席执行官]]唐·盖斯（{{lang|en|Don Geiss}}）交给副总裁杰克和其他有意竞争自己职位的手下一项任务，想个办法从[[环境保护主义|环境保护主义]]者手中赚到钱。杰克请来一位名叫杰拉德的男演员，由其扮演全国广播公司的环境[[吉祥物|吉祥物]]格伦佐。这个角色起初很成功，在《[[今天_(NBC)|今日秀]]》上与梅莉迪丝·维埃拉的互动也获得了好评。然而随着名气的增长，格伦佐也变得越来越不可一世，并在节目上严厉批评[[喜剧小品|喜剧小品]]《{{lang|en|TGS}}与崔西·乔丹》剧组成员浪费纸张，破坏环境。到了再一次上节目时他变本加厉，直接批评通用电气和杰克这样的“大公司及其两面三刀的肥猫高管”。愤怒的杰克开除了杰拉德，并试图请来前[[美国副总统|美国副总统]]阿尔·戈尔替补空缺但也无济于事。之后，格伦佐还喝多了酒来到现场，更是给已经糟透了的情况雪上加霜。<ref name="205episode" />

与此同时，电视台杂工跟班肯尼斯·帕塞尔正计划举办一场聚会。首席编剧丽兹·莱蒙回忆起肯尼斯举办过的以往几场聚会，每次除了他本人外就都只有自己一个人参加。男演员崔西觉得这样真太可怜了，于是四处造谣说[[T.I.|T.I.]]将会出席此次聚会，然后又通过各种各样的谎言说服他人来参加。结果聚会成了一场彻头彻尾的灾难，难过的肯尼斯决定这辈子再也不办什么聚会了。<ref name="205episode" />

丽兹在自己公寓里发现有别的女人的口红，于是和女演员简娜·玛隆妮都怀疑暂时与太太分居的制片人皮特有了外遇。经过“调查”丽兹意外地发现，皮特竟然是在和自己的老婆“偷情”，原来两人都喜欢这种新鲜刺激的感觉。之后皮特还问起丽兹，自己是否还能暂时借宿在她公寓里，因为夫妻俩终于可以再次体验[[约会|约会]]的感觉了。<ref name="205episode" />

== 反响 ==
