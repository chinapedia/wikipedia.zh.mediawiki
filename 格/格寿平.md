'''格寿平'''（{{lang|fr|Fernand Lacretelle}}，{{bd|1902年|4月23日|1989年|5月5日|catIdx=L}}），法国耶稣会士，[[天主教|天主教]]来华传教士，[[天主教海州监牧区|天主教海州监牧区]]首任监牧。

1902年4月23日，格寿平出生于法国巴黎。1921年（18岁）加入耶稣会。1927年（25岁）来华传教。取中文名格寿平，字养和。1934年6月7日（32岁）晋升神甫。1942-1946年，格寿平担任徐家汇修院院长。1947年12月8日担任上海[[耶稣会|耶稣会]]会长
<ref>[http://www.shtong.gov.cn/newsite/node2/node2245/node75195/node75203/node75283/node75297/userobject1ai91951.html 上海宗教志 >> 第四编天主教 >> 第二章团体 >> 第一节　耶稣会]</ref>。1950年，罗马教廷任命苏州教区主教[[龚品梅|龚品梅]]兼任[[天主教上海教区|天主教上海教区]]主教，龚品梅到上海上任后，任命格寿平为神学顾问
<ref>[http://www.shtong.gov.cn/newsite/node2/node2245/node75195/node75203/node75283/node75297/userobject1ai91953.html 上海宗教志 >> 第四编天主教 >> 第二章团体 >> 第二节　天主教上海教区]</ref>。1951年2月21日（48岁）被梵蒂冈任命为[[天主教海州监牧区|天主教海州监牧区]]（1949年6月9日成立）首任监牧，但他留在上海未赴任。1953年6月15日夜，格寿平和其他外国传教士一同被捕，罪名是“[[帝国主义|帝国主义]][[间谍|间谍]][[特务|特务]]首要分子”，关在上海市第三看守所。1954年7月2日，上海市军管会军法处将格寿平驱逐出境，并在《信鸽》报登载他供认“反对人民政府，并帮助别人也同样地反对政府，我承认，我是帝国主义分子。”的请求宽大书的照片<ref>[http://www.shtong.gov.cn/newsite/node2/node2245/node75195/node75203/node75287/node75301/userobject1ai92012.html 上海宗教志 >> 第四编天主教 >> 第六章反帝爱国运动和独立自主自办教会 >> 第一节　反帝爱国运动]</ref>。1954年，格寿平去台湾服务。1957年5月31日到达越南西贡。1976年又返回台湾，担任多个本堂的神父。1983年辞去海州监牧职务。1989年5月5日去世，享年87岁。
