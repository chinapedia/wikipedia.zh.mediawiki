'''格伦·达菲·施莱弗'''（{{lang|en|Glenn Duffie Shriver}}，{{bd|1981年|11月23日|||catIdx=S施}}），[[美国|美国]]公民，因为试图为[[中华人民共和国|中华人民共和国]]窃取情报而被捕。

== 案情 ==
施莱佛生于美国[[弗吉尼亚州|弗吉尼亚州]][[亨利科縣|亨利科縣]]，他中文普通话流利，曾于2002年到中国上海的[[华东师范大学|华东师范大学]]留学一年，毕业后曾留在中国。施莱佛2004年在上海居住期间，认识了三个人，而这些人他后来得知，都是中国安全部门的特工。应这些特工的要求，施莱佛答应返回美国，申请美国情报部门或执法部门的工作。施莱佛在法庭承认，他知道，他的最终目标就是设法得到联邦政府的职位，这样，他就可以有机会得到美国国防机密情报，然后就可以转交给中国情报官员，换取现金收入。

2005年到2010年，有5年时间，施莱佛一直企图得到美国国务院一份外交官的工作，或是[[中央情报局|中央情报局]][[国家秘密行动处|国家秘密行动处]]的工作。施莱佛承认，在此期间，他和中国情报官员经常联系，分三次得到7万多美元（现金）的报酬。中国情报官员说，这是对施莱佛表现出来的“友谊”的回报。

2009年12月，施莱佛得到通知，说让他在2010年5月来华盛顿，参加中情局录取前最后阶段的考察和面试。他立刻把这个信息通报给中国情报官员，说他已经获得一些进展，有望得到一份美国中央情报局的工作，但此时他无法分身前去中国再次同他们见面，因为这样做，就会引起调查他背景的美国官员的怀疑。

施莱佛承认，他在申请中央情报局工作填表时，没有据实填写他在过去7年中同外国政府或外国政府之代理人之间的关系。但实际上，他从2004年起，起码同外国的一名或多名情报人员有过大约20次面谈。另外，施莱佛还故意没有填写他在2007年曾去过中国，并因为申请中央情报局工作而得到了4万美元的报酬。另外，施莱佛在申请中情局工作最后的面试阶段，多次对同他面试的官员说谎。施莱佛承认，他这样做，是为了隐瞒他和中国情报官员的非法关系。

== 认罪 ==
施莱佛2010年10月22日在联邦法庭认罪，承认试图混入美国中央情报局，为中国输送情报。施莱佛是跟当局达成认罪协议之后在法庭上认罪的。按照这个新闻稿，施莱佛受到的指控只有一项：阴谋向中国输送美国国防机密情报。检方指控施莱佛企图向“某人”递送情报，但没有点明该人的身份。按照认罪协议，明年1月宣判，刑期可能是4年。

美国助理司法部长克里斯（Kris）的话说，该被告试图谋求一个联邦政府的职位，得到美国国防机密情报，然后向中华人民共和国的情报官员传递这些情报。他说：“通过检察人员、特工、分析人员的辛勤工作，该被告的阴谋被揭露出来而无法付诸实施。”检察官麦克布莱德（MacBride）的话说，“施莱佛先生背叛了美国，多次采取行动，试图为别国政府刺探情报。”麦克布莱德还说，“我们对任何危害国家安全的行为保持警惕，将尽全力查获和惩罚那些背叛国家的人。”麦克布莱德指，施莱佛先生选择为中国刺探情报的道路，就放弃了他的教育背景、他的职业生涯，他的未来。施莱佛没有意识到，中国发展同他个人的友谊，只是想利用他。美国检察官麦克布莱德说，这对于其它试图步其后尘的人来说，也是一个极为深刻的教训。

== 中国反应 ==
针对这一报导，中国驻美国大使馆新闻参赞王保东第一时间发表声明说：此事是无中生有。中新社的报导援引王保东的话说：中国永远不会参与损害其它国家利益的活动。另外，BBC 和大公网援引中国使馆的声明说：“任何通过‘杜撰来诋毁中国’的意图，都将‘徒劳无功’。

== 中国媒体报导 ==
针对美国司法部这一新闻稿，美国许多媒体都做了相关的报导。中国许多媒体，包括北京媒体如[[中国日报|中国日报]]网、[[人民网|人民网]]、[[环球网|环球网]]等转载了中新社的报导。香港親北京傳媒[[大公网|大公网]]、[[凤凰网|凤凰网]]，美国的《[[侨报|侨报]]》等等，也都加以报导，大多数的报导都是源自[[中新社|中新社]]王欣的报导：题目是：美报诬称中国在美国情报部门安插间谍窃取机密。

中新社2010年25日发表的这篇报导说，美国媒体10月25日报导称，曾在中国生活的美国留学生施莱佛为了减轻惩罚，承认自己“密谋向中国传递美国国防情报”。美国媒体“捕风捉影地发表评论，称中国方面正在努力向美国情报部门输送间谍，以窃取机密信息。”大公网的题目是：美汉称中国出钱供其打入CIA 中方驳斥。不过，中国环球时报英文网报导题目是：美男子承认试图刺探情报罪名。

== 美国媒体报导 ==
美国《[[华盛顿时报|华盛顿时报]]》发表记者戈茨的报导，题目是，间谍被捕说明北京努力吸收特工。报导说，施莱佛被捕事件，充分表明北京千方百计想渗透中情局。报导援引[[CIA|CIA]]发言人维丝(Paula Weiss)的话说，这是美国反间谍部门的成功行动。报导说，施莱佛不是第一个希望渗透CIA的中国间谍。1999年，当时的CIA局长[[特内特|特内特]]曾披露，起码有3名CIA官员曾为中国刺探情报。但这几个人都没有被抓，其中一名官员也曾拿到北京的6万美金。1985年，曾在CIA外围单位工作的美籍华人[[金无怠|金无怠]]被捕。司法当局指控他长期为中国充当间谍。但金无怠最后被判刑前在监狱自杀身亡。

报导说，CIA最近几年来遭到批评，说它在反中国间谍方面，心有余而力不足。报导援引美国资深情报界人士(Kenneth E. deGrafenreid)的话说，中国很可能想把施莱佛像金无怠那样，培养成长期战略间谍。金无怠上个世纪40年代就开始为美国政府部门服务，一直为中国提供消息，一直到1985年事发被捕。华盛顿时报的报导说，另外，中国还展现了很强的间谍能力，洛杉矶的陈文英就是其中一例。陈文英曾给联邦调查局当线民，但同时，她也给中国当间谍，并同起码两名FBI高级官员保持暧昧关系。报导还援引情报官员的话说，中国的军事和情报部门，往往通过在美国办的公司为掩护，但实际上，这些公司都在设法窃取或购买不许出口中国的美国高科技。<ref>[http://www.voanews.com/chinese/news/20101026_Spy_Arrest_Underscores_BJ_Bid_Agents-105777518.html 对比新闻：美国男子承认试图为中国窃情报]美国之音</ref>

但制作不佳，评价不高。<ref>[http://cn.nytimes.com/usa/20140416/c16shriver/ FBI警告美国青年别被中国政府利用]</ref>
