'''格罗塞托'''（[[意大利语|意大利语]]：'''{{lang|it|Grosseto}}'''）位于[[意大利|意大利]]中部[[第勒尼安海|第勒尼安海]]畔，是[[格罗塞托省|格罗塞托省]]的首府。

== 地理 ==
離[[第勒尼安海|第勒尼安海]]約14公里，[[翁布羅內河|翁布羅內河]]平原中心，海拔不超過10米。

== 歷史 ==
格羅塞托的起源可以追溯到[[中世紀|中世紀]]，在公元803年，就有格羅塞托的記載。 到了十二世紀，格羅塞托已經成為托斯卡納的主要城市之一。

== 經濟 ==
以[[葡萄酒|葡萄酒]]和[[香水|香水]]生產為主。

== 主要景點 ==
*[[格羅塞托主教座堂|格羅塞托主教座堂]]
*老城區
*葡萄園
