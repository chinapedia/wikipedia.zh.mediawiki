'''肺塵病'''或'''尘肺病''' （Pneumoconiosis），又稱'''黑肺症'''（Black Lung）（[[香港|香港]]稱'''肺塵埃沉着病'''，中国大陆和[[台灣|台灣]]俗稱'''矽肺病'''），又稱'''尘肺'''、'''砂肺'''，是一種[[肺部|肺部]][[纖維化|纖維化]][[疾病|疾病]]。患者通常長期處於充滿[[灰塵|塵埃]]的場所，因吸入大量灰塵，導致末梢[[支氣管|支氣管]]下的[[肺泡|肺泡]]積存灰塵，一段時間後肺內發生變化，形成[[纖維化灶|纖維化灶]]。

矽肺症主要是人體吸入結晶[[矽|矽]]的粉塵所造成，結晶矽的主要成分主要有土、[[沙|沙]]、[[花岡岩|花岡岩]]、以及其他岩石成分。

矽肺的主要症狀為：呼吸短促、[[發燒|發燒]]、[[疲倦|疲倦]]、無食慾、胸痛、乾咳、呼吸衰退，最後有可能致死。

矽肺容易引發其他的病變，包括[[肺癌|肺癌]]、支氣管炎、慢性阻塞性肺部病變、[[肺結核|肺結核]]、硬皮病、甚至會造成[[腎臟|腎臟]]病變。

== 塵肺病之定義 ==
英文''Pneumoconiosis''意即「塵肺病」，其中''pneumo''代表肺臟；konis代表粉塵。早於1700年[[義大利|義大利]]的[[拉馬齊尼|拉馬齊尼]]（Ramazzini）就提出吸入粉塵對肺的影響。 

依據Robinns等人所編著的[[病理學|病理學]]教科書及Ladou所編撰之職業與環境醫學教科書，沉積於肺部的礦物性粉塵（通常是在工作環境中吸入），導致肺部的[[非腫瘤性反應|非腫瘤性反應]]（nonneoplastic lung reaction）之所有症狀，統稱為肺塵症。 

1930年時[[台大醫院|台大醫院]]院長楊思標亦對肺塵病下了定義：長期吸進有害於人體的某種塵埃所引起的[[肺纖維病變|肺纖維病變]]（fibrotic change）。但目前肺塵病一詞，已較為廣義，包括因化學煙霧或蒸氣（chemical fume or chemical vapor）所導致的肺部疾病。

== 发病机制 ==
当长期吸入含有游离的[[二氧化硅|二氧化硅]]粉尘后，粉尘进入肺部，不能及时被清除，被肺内的[[巨噬细胞|巨噬细胞]]所吞噬，但二氧化硅不能被巨噬细胞所自行分解，导致巨噬细胞死亡，释放所吞噬的粉尘，然后恶性循环，造成更多细胞受损。在粉尘周围因受损的巨噬细胞刺激，造成纤维细胞增生，形成纤维性结节，这种结节被叫做-{矽}-结节。在-{矽}-结节内含有较多的[[免疫球蛋白|免疫球蛋白]]，病人血清中常出现[[γ-球蛋白|γ-球蛋白]]水平增高。故矽肺病可看作-{矽}-结节的形成与肺组织的弥漫性纤维化。<ref name=sun>{{cite web|url=http://www.fx120.net/disease4/201004/604098.html|title=-矽肺病的发病机制| publisher = 南都周刊 | year = 2010|language=zh-hans}}</ref>

== 特徵 ==
患者因吸入的塵埃日漸增多，病情也因而逐漸惡化，出現自覺症狀或是肺機能的障礙。

肺器官是人體主要內臟之一。吸入的空氣在肺泡內進行氣體交換，排出[[二氧化碳|二氧化碳]]，使血液帶氧，再送回[[心臟|心臟]]。藉着心臟的跳動，含氧的血液輸送到體內各部位。

患上肺塵病的人，因為交換氣體的作用受阻，使得[[血液|血液]]中的氧氣不足，二氧化碳過多，這種現象若長期持續，會導致[[右心室|右心室]]肥大或[[肺性心|肺性心]]或[[心不全|心不全]]等現象。

因右心室肥大、肺性心或心不全等症，患者出現全身性的衰弱現象，抵抗力隨之減弱，如果再併發[[急性肺炎|急性肺炎]]，病情嚴重會死亡。此症常併發肺結核、慢性支氣管炎、[[肺氣腫|肺氣腫]]，但併發肺癌並不常見。

== 罹患頃向 ==
在充滿塵埃或易於引起此症之場所工作的人，患病率自然偏高。過去以[[礦工|礦工]]最容易罹患，此外營建類工作者也容易罹患，或者工作場所與事務上與噴砂表面處理、岩石壓碎及鑽孔、混凝土、[[隧道|隧道]]工程、[[硅|硅]]、[[水泥|水泥]]或[[瀝青|瀝青]]的製造等相關者也容易罹患。因此是一種[[職業病|職業病]]。

一般而言，四十歲至五十歲以上的人，較易發病，且無男、女之別。{{Fact|但是，部份人的[[鼻腔|鼻腔]]或氣管較不易沾染塵埃，或是某些人的內分泌機能可阻止纖維的異常增長時，罹患此症的機會因此降低。|time=2019-01-04T03:01:38+00:00}}

== 症狀 ==
此症是因吸入的塵埃，堆積於肺內，影響空氣的交換，而逐漸發展的疾病。

疾病未發作前，患者毫無不適現象，但是若在環境不佳的場所，{{Fact|工作十年以上（也有短至幾年），會因疾病的長期發展而顯露病徵。發病後，即使不再置身於工作場所，病患依然繼續惡化。|time=2019-01-04T03:01:38+00:00}}

初期，患者只出現[[咳嗽|咳嗽]]、咳[[痰|痰]]等呼吸系統的疾病，若病情繼續惡化，就會發生[[呼吸困難|呼吸困難]]、全身倦怠、衰弱、[[貧血|貧血]]等病徵。

=== 檢查時所能發現的症狀 ===
經由各種檢查法，可發現患者的[[紅血球|紅血球]]數量減少、[[血紅素|血紅素]]的含量降低等貧血症狀，另外，血液中的含氧量不足，二氧化碳卻偏多。

患者的肺機能衰退，經[[X光|X光]]檢查，可發現肺部的異常現象，這些現象會隨著時間而逐漸惡化，即使經過治療亦無法恢復健康。

=== 併發症 ===
一旦發病後，因病情發展，十分緩慢，所以常引發各種肺疾，在併發症中，最應預防的是[[肺結核|肺結核]]。若屬輕度肺結核，可利用各種抗結核劑治癒之，較為容易。但是，若患了肺塵肺結核，即使服用各種抗結核劑亦無痊癒之日，治療只是暫時延長患者的生命罷了。

另有些突發性的併發症，需做緊急的處置，最具代表性的是急性肺炎和急性心不全。出現這種併發症時，患者需保持絕對的靜養，並接受住院治療。

== 症狀緩和 ==
可以透過[[洗肺|洗肺]]來緩和部分病人的症狀，洗肺沒辦法治癒病人，但可以延緩病情繼續惡化，身體狀況可以接受洗肺的塵肺病病人，在洗肺後會延長壽命。

== 世界各地现状 ==

===中國大陸===
2009年，中国大陆发生[[开胸验肺|开胸验肺]]事件，而使得尘肺病广为人知。

中華人民共和國《2013年全国职业病报告情况》显示，根据全国30个省、自治区、直辖市（不包括西藏）和[[新疆生产建设兵团|新疆生产建设兵团]]职业病报告，2013年共报告职业病26393例，其中尘肺病23152例，煤工尘肺和矽肺分别为13955例和8095例。尘肺病报告病例数占2013年职业病报告总例数的87.72%。<ref>[http://www.ce.cn/xwzx/gnsz/gdxw/201406/30/t20140630_3064986.shtml 去年新增2.3万尘肺病例 占职业病报告数近九成]，中国经济网，2014年6月30日</ref>

《2014年全国职业病报告》，职业病29972例，其中尘肺病占89.66%<ref>{{cite web|url=https://view.news.qq.com/original/intouchtoday/n3365.html|title=流水的矿工，铁打的尘肺病|date=2015-12-06|website=[[腾讯|腾讯评论]]|archiveurl=https://archive.fo/QAoTp|archivedate=2019-03-20}}</ref>。

尘肺病在中国大陆的盛行，亦催生了相关作品，如摄影组照《一个尘肺病家庭的爱与绝望》（[[沈绮颖|沈绮颖]]）<ref>{{cite web|url=https://news.qq.com/original/living/l488.html|website=[[腾讯|腾讯新闻]]|title=一个尘肺病家庭的爱与绝望|archivedate=2019-03-20|archiveurl=https://archive.fo/gEWaK|language=zh-hans}}</ref>。

===香港===

香港特区政府设立[[肺尘埃沉着病补偿基金委员会|肺尘埃沉着病补偿基金委员会]]，研究相关事项。
