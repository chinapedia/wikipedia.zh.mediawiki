'''维罗纳'''（[[意大利语|意大利语]]：'''{{lang|it|Verona}}'''）是位于[[意大利|意大利]]北部[[威尼托|威尼托]][[阿迪杰河|阿迪杰河]]畔的一座历史悠久的城市，2000年入选[[联合国教科文组织|联合国教科文组织]]的[[世界遗产|世界遗产]]。

[[莎士比亞|莎士比亞]]的名作《[[羅密歐與茱麗葉|羅密歐與茱麗葉]]》以此城為背景，有「愛之都」之稱。

== 历史 ==

维罗纳最初由古代意大利的[[埃乌加内|埃乌加内]]人建造，公元前550年被古[[高卢人|高卢人]]占领，公元前约300年[[波河|波河]]流域成为[[罗马人|罗马人]]的领土，公元前89年维罗纳成为[[罗马帝国|罗马帝国]]的殖民地，维罗纳城内现存的圆形露天剧场（竞技场）建造于此后的一个世纪。

维罗纳因为处在多条贸易道路的交界，成为欧洲一座重要的城市。[[东哥特人|东哥特人]]从489年开始统治意大利，维罗纳成为[[东哥德王国|东哥德王国]][[狄奥多里克大帝|狄奥多里克大帝]]的住所。569年[[伦巴底人|伦巴底人]][[阿尔博因|阿尔博因]]取而代之，维罗纳成为[[伦巴第|伦巴第]]的第二大城市。774年伦巴底人在维罗纳做出对[[法兰克王国|法兰克王国]][[查理大帝|查理大帝]]的最后一次抵抗失败，维罗纳成为意大利国王的住所。952年德意志国王[[奥托一世_(神圣罗马帝国)|奥托一世]]入侵意大利征服伦巴第，自称意大利国王，维罗纳归属于[[巴伐利亚公爵|巴伐利亚公爵]]家族，976年起隶属于[[卡林西亞|卡林西亞]]。12世纪开始维罗纳才获得城市的自治权，1164年创建并加入[[伦巴第|伦巴第]]城市联盟，中世纪后期由[[斯卡拉|斯卡拉]]家族统治。

1509年至1517年维罗纳由[[神圣罗马帝国|神圣罗马帝国]]的[[马克西米连一世_(神圣罗马帝国)|马克西米连一世]]统治。维罗纳在1797年被[[拿破仑|拿破仑]]占领，他结束了[[威尼斯共和国|威尼斯共和国]]的统治，在与意大利一起击败[[第一次反法同盟|第一次反法同盟]]后，与[[奥地利|奥地利]]签署合约，将维罗纳划为奥地利城市。意大利在1866年的[[普奥战争|普奥战争]]中与[[普鲁士|普鲁士]]结盟并最终战胜奥地利，维罗纳回归[[意大利王国|意大利王国]]。

== 旅游景点 ==
[[File:Duomo_(Verona)_-_Facades.jpg|thumb]]
如今的维罗纳因坐落在[[阿达湖|阿达湖]]和[[威尼斯|威尼斯]]的附近，成为旅游胜地和[[威尼托|威尼托]]地区的经济中心。

=== 圆形竞技场 ===
维罗纳最雄伟的建筑是[[维罗纳圆形竞技场|维罗纳圆形竞技场]]，它同时也是罗马式的圆形露天剧场，它是世界上现存的第三大圆形竞技场，每年夏天都举行[[歌剧|歌剧]]演出，吸引了众多的歌剧迷。

=== 教堂 ===
维罗纳有多座教堂。[[維羅納主教座堂|维罗纳圣母主教座堂]]建于1187年，结合了[[罗马式建筑|罗马式建筑]]和[[哥特式建筑|哥特式建筑]]两种风格，大教堂因[[文艺复兴|文艺复兴]]后期伟大画家[[提香|提香]]的作品《玛利亚升天》而出名。[[圣亚纳大削堂_(维罗纳)|圣亚纳大削堂]]是维罗纳最大的教堂，由[[多明我会|多明我会]]建造于1290年至1481年间，它的正面保持着没有完工的状态，内饰豪华，并以驼背雕像背着的圣水池而出名。[[圣费尔莫教堂|圣费尔莫教堂]]由[[本笃会|本笃会]]建造于1065年至1138年间，教堂正面展现了罗马式和哥特式两种不同的建筑风格。[[圣柴诺圣殿|圣柴诺圣殿]]建于12世纪和13世纪，整栋建筑呈现罗马式，但屋顶却是哥特式，引人注目的是教堂的钟楼和教堂正面的巨型圆窗。

=== 朱丽叶的阳台 ===
[[File:Balkon_der_Julia_Verona.jpg|thumb]]
[[莎士比亞|莎士比亞]]剧作《[[羅密歐與茱麗葉|羅密歐與茱麗葉]]》的故事发生在维罗纳，[[羅密歐·蒙泰古|羅密歐]]曾在[[朱麗葉·凱普萊特|茱麗葉]]家的阳台向朱丽叶求爱，这段悲剧爱情故事和朱丽叶的阳台一起留给了后世。维罗纳城内仍保留着朱丽叶的故居，故居中“朱丽叶的阳台”每年吸引着数百万的游客。虽然这部戏剧的最早版本发生在[[锡耶纳|锡耶纳]]而不是维罗纳，实际上也并不存在“朱丽叶的阳台”，维罗纳人不得不一次又一次地向慕名而来的游客解释这一遗憾的事实，但他们出于旅游业的目的还是在故居后院中按照莎士比亚的巨作建造起一座象征性阳台，以供游客朝拜这座“爱的圣地”，故居墙上也留下了无数游客爱情誓言的[[涂鸦|涂鸦]]。阳台下有一座朱丽叶的铜像，右胸因传会带来好运而已经被游客抚摸得发亮。

== 体育 ==
维罗纳有两个足球俱乐部，[[切沃足球俱乐部|-{zh-hans:切沃;zh-hant:基爾禾;}-]]成立于1929年，现时在[[意大利甲级联赛|意大利甲级联赛]]角逐。-{zh-hans:切沃;zh-hant:基爾禾;}-曾在2002年出战[[欧洲联盟杯|欧洲联盟杯]]，2006年出战[[欧洲冠军联赛|欧洲冠军联赛]]。维罗纳的另一間球會[[維羅納足球俱樂部|維羅納足球俱樂部]]成立于更早的1903年，曾获得1984年至1985年赛季的意大利冠军，并出战第二年的[[欧洲联赛冠军杯|欧洲联赛冠军杯]]。

== 重要人物 ==
维罗纳曾留下过许多重要人物的足迹，同欧洲历史息息相关，[[恺撒|凯撒大帝]]曾选择维罗纳作为他的度假地，意大利[[东哥德王国|东哥德王国]]的创始者[[狄奥多里克大帝|狄奥多里克大帝]]和他的继任者们，意大利[[伦巴第王国|伦巴第王国]]的建立者[[阿尔博因|阿尔博因]]和伦巴第的公爵们、[[法兰克王国|法兰克王国]]国王[[查理大帝|查理大帝]]和他的儿子意大利国王[[丕平_(意大利)|丕平]]、意大利国王[[贝伦加尔一世|贝伦加尔一世]]、意大利[[文艺复兴|文艺复兴]]诗人[[但丁|但丁]]等都曾居住在维罗纳，[[罗马天主教会|罗马天主教会]]产生[[罗马主教|罗马主教]]的[[教宗选举|教宗选举]]也曾在维罗纳举行。维罗纳也出现在德国作家[[歌德|歌德]]、法国作家[[司汤达|司汤达]]和[[瓦勒里|瓦勒里]]的旅行日记中。
