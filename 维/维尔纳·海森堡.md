'''维尔纳·海森堡'''（{{lang-de|'''Werner Heisenberg'''}}，{{bd|1901年|12月5日|1976年|2月1日|catIdx=Heisenberg}}），[[德国|德国]][[物理学家|物理学家]]，[[量子力学|量子力学]]创始人之一，“[[哥本哈根学派|哥本哈根学派]]”代表性人物。1932年，海森堡因為“创立[[量子力学|量子力学]]以及由此导致的[[氢|氢]]的[[同素异形体|同素异形体]]的发现”而榮获[[诺贝尔物理学奖|诺贝尔物理学奖]]。

他对物理学的主要贡献是给出了量子力学的矩阵形式（[[矩阵力学|矩阵力学]]），提出了“[[不确定性原理|不确定性原理]]”（又称“海森堡不确定性原理”）和[[S矩阵|S矩阵]]理论等。他的《量子论的物理学原理》是量子力学领域的一部經典著作。

== 生平 ==
维尔纳·海森堡1901年12月5日出生于德国[[维尔茨堡|维尔茨堡]]，1920年从[[慕尼黑|慕尼黑]]的一所中学毕业后，在[[慕尼黑大学|慕尼黑大学]]学习物理学，师从[[阿諾·索末菲|阿諾·索末菲]]、[[威廉·維恩|威廉·維恩]]等，1922年冬天转去[[哥廷根大学|哥廷根大学]]，师从[[马克斯·玻恩|马克斯·玻恩]]（1954年诺贝尔物理学奖）、[[詹姆斯·弗兰克|詹姆斯·弗兰克]]（1925年诺贝尔物理学奖）和[[大卫·希尔伯特|大卫·希尔伯特]]。1923年他在慕尼黑大学获得博士学位，并去哥廷根大学担任玻恩的助手，1924年获得大学任教资格。

從1924年至1927年，海森堡在哥廷根擔任講師。1924年9月17日至1925年5月1日，由於一項國際教育委員會[[洛克菲勒基金會|洛克菲勒基金會]]獎學金的支持，海森堡與哥本哈根大學物理主任玻爾進行研究。之後他回到[[哥廷根|哥廷根]]和[[尼尔斯·玻尔|尼尔斯·玻尔]]（1922年诺贝尔物理学奖）共事並共同發表了[[矩陣力學|矩陣力學]]。1926年又前往哥本哈根大学教授理论物理学。1927年，發展了海森堡[[不確定性原理|不確定性原理]]，為後來知名的[[哥本哈根詮釋|哥本哈根詮釋]]奠定了基礎，於同年，年仅26岁的海森堡被任命为[[莱比锡大学|莱比锡大学]]的教授，他的第一篇論文發表於[[萊比錫|萊比錫]]。1929年，他又前往[[美国|美国]]、[[日本|日本]]和[[印度|印度]]巡回讲学。1941年成为[[柏林洪堡大学|柏林洪堡大学]]的物理学教授和物理研究所主任。

[[第二次世界大战|第二次世界大战]]结束后，他和其他德国物理学家作为囚犯，被美国军队送往[[英国|英国]]{{tsl|en|Farm Hall}}。1946年重返德国后，他和他的同事们重建了哥廷根物理研究所，该研究所在1948年改名为[[马克斯-普朗克物理学研究所|马克斯-普朗克物理学研究所]]。1948年他在英国[[剑桥|剑桥]]讲学数月，1950年和1954年又应邀前往美国讲学，1955年冬天他在[[苏格兰|苏格兰]]的[[圣安德鲁斯大学|圣安德鲁斯大学]]教书，这些课程后来出版成书。1955年作为马克斯-普朗克物理学研究所的主任，与研究所一起迁往慕尼黑，1958年成为慕尼黑大学的物理学教授，他的研究所被改名为“马克斯-普朗克物理学和天体物理学研究所”（现为[[马克斯-普朗克天体物理学研究所|马克斯-普朗克天体物理学研究所]]）。

海森堡爱好古典音乐，是个出色的钢琴手，1937年与伊丽莎白·徐赫（Elisabeth Schophauer）结婚，生有7个孩子，居住在慕尼黑。海森堡在1976年2月1日因癌症於家中逝世。

== 研究 ==

海森堡的名字一直都和他的[[量子力学|量子力学]]理论联系在一起，这一理论发表时他年仅23岁，他也因为提出这一理论及其应用（尤其是[[氢的自旋異構體|氢的自旋異構體]]发现），获得了1932年的[[诺贝尔物理学奖|诺贝尔物理学奖]]。

海森堡提出的新理论，是完全基于对[[原子辐射|原子辐射]]的观察，他认为，在某一个给定的时间点，一个[[电子|电子]]所处的位置是无法确定的，也无法跟踪它的轨迹，所以[[玻尔|玻尔]]假定的[[电子轨道|电子轨道]]并不存在；诸如[[位置|位置]]、[[速度|速度]]等[[力学|力学]]量，无法用通常的数字来描述，但可以用抽象的数学结构即[[矩阵|矩阵]]来表达，海森堡用矩阵形式给出了他的新理论（[[矩阵力学|矩阵力学]]）。

此后，海森堡又提出了著名的“[[不确定性原理|不确定性原理]]”（又称“海森堡測不準原理”），在一个量子力学系统中，一个运动[[粒子|粒子]]的位置和它的[[动量|动量]]不可被同时确定，位置的[[不确定性|不确定性]]<math>\Delta x</math>和动量的不确定性<math>\Delta p</math>是不可避免的，它们的乘积不小于<math>h/4\pi</math>（<math>h</math>为[[普朗克常数|普朗克常数]]），这些误差对于人类来说虽然是微小的，但是在原子研究中并不能被忽略。

在[[莱比锡|莱比锡]]期间，海森堡为[[原子核物理学|原子核物理学]]做出了重要贡献，为[[基本粒子|基本粒子]]理论引入了内部对称[[量子数|量子数]]（1932年，1933年），发展了一种[[铁磁性|铁磁性]]理论（1928年），和[[沃尔夫冈·泡利|沃尔夫冈·泡利]]对[[量子场论|量子场论]]进行了开创性研究工作。海森堡和[[約翰·惠勒|約翰·惠勒]]同为[[S矩阵|S矩阵]]（1942年，1944年）之父，他很早就研究了量子场论的基本长度模型（1938年）。1940年代，他还研究了[[宇宙射线|宇宙射线]]及其产生的离子碎片，导致不久后在英国发现了第一个[[介子|介子]]。

1957年起，海森堡的研究兴趣转向了[[等离子体|等离子体]]物理和[[高热原子核反应|高热原子核反应]]问题，并与[[日内瓦|日内瓦]]国际原子物理研究所紧密合作，他担任该研究所的科学政策委员会主席，并一直是该委员会的成员。在他于1953年成为[[洪堡基金会|洪堡基金会]]主席后，为基金会做了很多促进工作，他邀请各国科学家来德国，并协助他们在德国开展研究工作。

1953年起，他的理论工作偏向基本粒子的[[统一场论|统一场论]]，这对于他来说，是理解基本粒子物理学的关键。

== 荣誉 ==

除了获得[[马克斯·普朗克奖章|马克斯·普朗克奖章]]、德国[[联邦十字勋章|联邦十字勋章]]等奖章，诺贝尔物理学奖等奖项外，海森堡还被[[布鲁塞尔大学|布鲁塞尔大学]]、[[卡尔斯鲁厄大学|卡尔斯鲁厄大学]]和[[布达佩斯大学|布达佩斯大学]]授予荣誉博士头衔。他是伦敦[[皇家学会|皇家学会]]的会员、以及哥廷根、巴伐利亚、萨克森、普鲁士、瑞典、罗马尼亚、挪威、西班牙、荷兰、罗马、美国等众多科学学会的成员，[[德国科学院|德国科学院]]和[[意大利科学院|意大利科学院]]的院士。1953年成为洪堡基金会的主席。

==纳粹德国的核武器计划==
海森堡曾经是[[纳粹德国|纳粹德国]][[核武器|核武器]]研制的领导人，不过纳粹德国自始至终都没有能力将核武器从理论变为现实。海森堡本人提供的一种说法称，他其实并不信任[[希特勒|希特勒]]政权，因此在尽力拖延纳粹德国的研究计划。但曼哈顿计划重要领导人[[古德施密特|古德施密特]]等一些人对这种说法嗤之以鼻，这也被称之为“海森堡之谜”。<ref>{{cite news|title=海森堡是纳粹帮凶？ 诺贝尔奖师生书信公开(附图)|url=http://news.sina.com.cn/cl/2002-02-09/1747472946.html|accessdate=2017-02-18|agency=北京晚报|publisher=新浪新闻|date=2002-02-09}}</ref><ref>{{cite book|author1=曹天元|title=量子物理史話|publisher=八方出版股份有限公司|isbn=9789865810443|page=海森堡之谜|url=http://hx.cnd.org/2015/05/06/%E6%9B%B9%E5%A4%A9%E5%85%83%EF%BC%9A%E6%B5%B7%E6%A3%AE%E5%A0%A1%E4%B9%8B%E8%B0%9C%EF%BC%88%E4%B8%80%EF%BC%89%E6%91%98%E8%87%AA%E3%80%8A%E9%87%8F%E5%AD%90%E5%8F%B2%E8%AF%9D%E3%80%8B/}}</ref>
