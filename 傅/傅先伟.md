'''傅先伟'''（{{bd|1944年|2月|2018年|8月20日}}），[[浙江|浙江]][[宁波|宁波]]人，[[中国基督教三自爱国运动委员会|中国基督教三自爱国运动委员会]]主席<ref name=ifeng>{{Cite web |url = http://phtv.ifeng.com/a/20160427/41600705_0.shtml |title = 中国基督教三自爱国运动委员会主席傅先伟简介 |author =  |date = 2016-04-27 |publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref>。

==生平==
傅先伟自[[金陵協和神學院|金陵協和神學院]]教牧研究生班毕业<ref>[http://www.christianweekly.net/4DACTION/W4D_CWREAD/15732/11/BACK 中國基督教兩會選出新一屆兩會委員會領導同工]{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}，《基督教周报》第2265期（2008年1月20日）</ref><ref name=ifeng/>。曾经在上海周家渡船厂和学校工作。1988年进入上海市基督教两会（上海市基督教三自爱国运动委员会、上海市基督教教务委员会）。1996年5月出任[[上海沐恩堂|上海沐恩堂]]长老。1997年1月，当选为上海市基督教三自爱国运动委员会常务委员。1997年10月，当选为上海市基督教三自爱国运动委员会第七届秘书长。2002年11月1日，当选为上海市基督教三自爱国运动委员会第八届主席。2003年起，任第十届[[全国政协|全国政协]]委员。2008年1月13日，在中国基督教第八次代表会议上接替[[季剑虹|季剑虹]]当选[[中国基督教三自爱国运动委员会|中国基督教三自爱国运动委员会]]第八届主席，此后连任第九届主席<ref name=shsh/><ref name=ifeng/><ref>[http://www.usqiaobao.com:81/qiaobao/html/2008-01/14/content_64073.htm 傅先伟接掌中国基督教]{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。2008年，担任[[中国人民政治协商会议第十一届全国委员会委员名单|第十一届全国政协常务委员]]。2013年，担任[[中国人民政治协商会议第十二届全国委员会委员名单|第十二届全国政协常务委员]]。2018年，担任[[中国人民政治协商会议第十三届全国委员会委员名单|第十三届全国政协常务委员]]<ref>{{Cite web|url=http://sports.163.com/18/0126/06/D92BRKK400051C89.html|title=中国人民政治协商会议第十三届全国委员会委员名单|accessdate=2018-01-26|last=|work=网易}}</ref><ref name=ifeng/>。

2018年8月20日，傅先伟在[[上海华山医院|上海华山医院]]安息主怀，享年74岁<ref name=shsh>{{Cite web |url =https://www.thepaper.cn/newsDetail_forward_2365379  |title =74岁中国基督教三自爱国运动委员会主席傅先伟长老逝世  |author =  |date =  2018-08-21|publisher = 澎湃新闻 |language =  |accessdate =  |quote =  }}</ref>。
