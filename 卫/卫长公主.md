'''卫长公主'''{{BD|前138年||？|catIdx=L}}，又称'''当利公主'''，[[西汉|西汉]][[公主|公主]]，[[汉武帝|汉武帝]]和[[孝武衛皇后|孝武衛皇后]]<ref>《史记》和《汉书》皆未记载卫长公主的生母，根据[[司马贞|司马贞]]的《[[史记索隐|史记索隐]]》可知卫长公主为卫子夫女。</ref>的女儿。本名、生卒、排行皆不详。

== 生平 ==

《[[史記|史記]]》和《[[漢書|漢書]]》中對衛長公主的記載，都是關於她的兩次婚姻。第一次，衛長公主下嫁第五代[[平陽侯|平陽侯]][[曹襄|曹襄]]<ref>曹襄的父親是平陽侯[[曹時|曹時]]，[[嫡母|嫡母]]（或生母）是[[漢武帝|漢武帝]]的姐姐[[平陽公主_(漢景帝)|平陽公主]]。</ref>。曹襄逝世於[[元鼎|元鼎]]三年（公元前114年）。曹襄之子[[曹宗|曹宗]]承襲平陽[[侯爵|侯]]。第二次下嫁方士[[欒大|欒大]]，欒大後因方術不驗而被腰斬。

元鼎五年，武帝寵信欒大，把衛長公主嫁給他。婚後，武帝將公主的[[食邑|封地]]改名為[[当利县|当利县]]，衛長公主又號為當利公主。

其後，史書再沒有公主的記載。[[征和|征和]]二年（前91年），曹襄之子曹宗死於[[巫蠱之禍|巫蠱之禍]]，《[[漢書|漢書]]》<ref>《漢書 卷三十九 [[蕭何|蕭何]][[曹參|曹參]]傳第九》子宗嗣，有罪，完為城旦。</ref>則記為處以[[完刑|完刑]]，為[[城旦|城旦]]<ref>{{cite web|url=http://www.eywedu.com/zonghetd/ywyz2007/ywyz20071152.html|title=「城旦」註解探微|publisher=語文教學與研究·綜合天地 作者：武健|date=2007年第11期 |accessdate=2009-12-16 |language=Zh-hans}}</ref>。

== 生卒、排行、封号 ==

母亲[[卫子夫|卫子夫]]于[[建元_(汉武帝)|建元]]二年春<ref>《[[史记|史记]] 卷一百一十一 卫将军骠骑列传第五十一》建元二年春，青姊子夫得入宫幸上。</ref>（公元前139年）入宫。而根据《[[汉书|汉书]]》记载<ref>《汉书 卷九十七上 外戚传第六十七上》入宫岁余，不复幸。武帝择宫人不中用者斥出之，子夫得见，涕泣请出。上怜之，复幸。遂有身，尊宠。</ref>，卫子夫入宫一年多，得到汉武帝再次[[宠幸|宠幸]]而怀孕。因此卫长公主的生年不会早于公元前138年。至于卫长公主的逝世时间，已经无从考证了。

根据司马贞在《[[史记索隐|史记索隐]]》中的注释，可知卫长公主有同母姐妹兄弟三人，是卫太子[[刘据|刘据]]、[[诸邑公主|诸邑公主]]、[[石邑公主|石邑公主]]<ref>[[颜师古|颜师古]]在《[[汉书|汉书]]》的注释，则称[[诸邑公主|诸邑公主]]和[[阳石公主|阳石公主]]是卫子夫的女儿。</ref>。除刘据<ref>卫太子刘据生于公元前128年。</ref>外，诸邑公主、石邑公主的出生年份皆不详，因此无法得知四人的排行。同时刘据十四岁时，卫长公主就已成为[[寡妇|寡妇]]，所以一般推断卫长公主是刘据的姐姐。

卫长公主最初的[[封号|封号]]已不得而知，卫长公主越级称长公主，唐代[[司马贞|司马贞]]在《[[史记索隐|史记索隐]]》<ref>[http://www.eywedu.com/Historian/sj_028.htm 《史记 集解索隐正义》]</ref>对此的解释是“卫子夫之子曰卫太子，女曰卫长公主。是卫后长女，故曰长公主，非如帝姊曰长公主之例。”

== 影视形象 ==
电视剧《[[大汉天子3|大汉天子3]]》中，[[熊乃瑾|熊乃瑾]]饰卫长公主。剧中卫长公主为刘据之妹。剧中虚构了其与匈奴王子金童相爱，却因道士李长生欺瞒武帝而被下嫁李长生，李长生伏法后才嫁给金童，但不久即遭[[江充|江充]]陷害被捕，在江充严刑拷问下不肯攀诬刘据终咬舌自尽，被悔悟后的武帝命[[司马迁|司马迁]]将其记载为与其余[[巫蛊之祸|巫蛊之祸]]受害者一同死于宫中瘟疫等情节，与历史记载不符。且剧中武帝称其为“卫长”，但其封号的正确断句应是“卫 长公主”而非“卫长 公主”。
