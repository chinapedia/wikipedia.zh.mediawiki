'''招宝山街道'''，是[[中国|中国]][[浙江省|浙江省]][[宁波市|宁波市]][[镇海区_(宁波市)|镇海区]]的一个[[街道_(行政区划)|街道]]和镇海区政府曾经所在地。位于[[甬江|甬江口]]北岸，南面隔甬江与[[北仑区|北仑区]][[小港街道_(宁波市)|小港街道]]相望，西面和北面与[[蛟川街道|蛟川街道]]相连，东面隔[[灰鳖洋|灰鳖洋]]与[[舟山市|舟山市]][[金塘岛|金塘岛]]相望。总面积20.8平方公里，人口6.23万，[[街道办事处|街道办事处]]位于胜利路200号。

== 沿革 ==
'''招宝山街道'''原属城关镇，古称'''浃口'''，别名'''蛟川'''，2001年析城关镇设立招宝山街道和蛟川街道。

== 交通 ==
招宝山街道通过[[招宝山大桥|招宝山大桥]]与北仑区[[戚家山街道|戚家山街道]]相连，骆霞线[[甬江水底隧道|甬江水底隧道]]与[[小港街道|小港街道]]相连，[[宁镇公路|宁镇公路]]与宁波市中心城区相连，[[镇骆路|镇骆路]]与镇海[[骆驼街道|骆驼街道]]新城区相连。

== 旅游 ==
招宝山街道为全国重点文物保护单位[[镇海口海防遗址|镇海口海防遗址]]所在地，[[招宝山|招宝山]]为4A级景区。
