'''恰蒂斯加尔邦'''（[[恰蒂斯加尔语|恰蒂斯加尔语]]／[[印地语|印地语]]：'''छत्तीसगढ़'''，[[拉丁字母|拉丁字母]]转写：{{Unicode|Chattisgarh}}）是位于印度中部的一个行政邦，它于2000年11月1日从[[中央邦|中央邦]]脱离，成为印度的第26个邦，全国第十大邦，首府为[[赖布尔|赖布尔]]。“恰蒂斯加尔”一词的印地语直译为“36个部落”，意指古代此处的36个部落。

恰蒂斯加尔邦西北部与中央邦接壤，西部与[[马哈拉施特拉邦|马哈拉施特拉邦]]接壤，南部与[[安得拉邦|安得拉邦]]接壤，东部与[[奥里萨邦|奥里萨邦]]接壤，东北部和北部分别与[[贾坎德邦|贾坎德邦]]和[[北方邦|北方邦]]接壤。

[[恰蒂斯加尔语|恰蒂斯加尔语]]作为该邦官方语言之一，是[[印欧语系|印欧语系]]的一个重要分支，亦是恰蒂斯加尔邦的主要语言。许多语言学家将其视为[[印地语|印地语]]在当地的一种演化方言；此外在恰蒂斯加尔邦的一些山区也聚居有许多讲[[達羅毗荼語系|達羅毗荼語系]]（{{lang|en|Dravidian languages}}）的[[岡德人|岡德人]]（{{lang|en|Gond}}）；在恰蒂斯加尔邦北部的少部分地区也通用[[蒙達語族|蒙達語族]]（{{lang|en|Munda Language Family}}）的一些方言。

== 地理 ==
恰蒂斯加尔邦的北部为广阔的[[印度河恒河平原|印度河恒河平原]]，恒河的支流[[汉德河|汉德河]]（{{lang|en|Rihand}}）流经这个区域。[[索德布尔山脉|索德布尔山脉]]（{{lang|en|Satpura Range}}）的东端和[[焦达讷格布尔高原|焦达讷格布尔高原]]（{{lang|en|Chota Nagpur Plateau}}）的西端东西贯穿将[[默哈讷迪河|默哈讷迪河]]（Mahanadi River）盆地从印度河恒河平原分割而出。邦中部是富饶的默哈讷迪河及其支流平原，土地肥沃，是广阔的[[稻米|稻米]]种植区。邦南部位于[[哥达瓦里河|哥达瓦里河]]（{{lang|en|Godavari River}}）与其直流[[印德拉瓦提河|印德拉瓦提河]]（{{lang|en|Indravati River}}）交汇处的[[德干高原|德干高原]]。全邦的[[森林覆盖率|森林覆盖率]]为44％。

印度最大河流之一的默哈讷迪河发源于恰蒂斯加尔邦，邦内另有[[赫斯多河|赫斯多河]]（{{lang|en|Hasdo}}，默哈讷迪河的一个支流）、汉德河、[[因德勒瓦蒂河|因德勒瓦蒂河]]（{{lang|en|Indravati}}）、[[琼克河|琼克河]]（{{lang|en|Jonk}}）和[[阿帕河|阿帕河]]（{{lang|en|Arpa}}）等其他的主要河流。

恰蒂斯加尔邦的南部和北部多是[[山丘|山丘]]地带，中部以[[平原|平原]]为主，气候为典型的印度中部气候。

== 政治 ==
恰蒂斯加尔邦北部和南部活动着一个反叛武装团体，[[纳萨尔派|纳萨尔派]]（{{lang|en|Naxalites}}），该组织主张通过[[农民|农民]]武装斗争夺取政权，并在一些偏远地区建立了对立政府。

此外，该邦有为数众多的无党派者和4个主要政治[[党派|党派]]。

* [[印度人民党|印度人民党]]，当前（2013年）为恰蒂斯加尔邦立法会多数党；
* [[印度国大党|印度国大党]]；
* [[社会民主党_(印度)|社会民主党]]（{{lang|en|Bahujan Samaj Party}}）；
* [[冈瓦那党|冈瓦那党]]（{{lang|en|Gondwana Party}}）。

2006年2月28日，纳萨尔组织突袭了丹德瓦达区的数辆[[卡车|卡车]]，导致55名平民丧生，该事件也成为印度历史上受纳萨尔武装的袭击丧生人数最多的事件之一。在后继的纳萨尔武装的袭击中，仍有少民众丧生。

2010年5月恰蒂斯加尔邦通过的一部法律中规定，任何[[宗教|宗教]]信仰的转变必须提前30天向相关机构通报。违反此条令的公民将被处以1年的监禁。根据[[BBC|BBC]]的报道，该邦民众宗教信仰的转变实为一个主要的政治问题。

== 经济 ==
恰蒂斯加尔邦2004年的经济总产值估计为120亿美元，在分裂后，这个矿产区占据了原[[中央邦|中央邦]]（{{lang|en|Madhya Pradesh}}）30％的经济输出。

恰蒂斯加尔邦境内有多家大型企业，诸如[[比莱钢铁厂|比莱钢铁厂]]（{{lang|en|Bhilai Steel Plant}}）、SEC铁路区（{{lang|en|S.E.C.Railway Zone}}）、BALCO铝业公司（{{lang|en|BALCO Aluminium Plant}}，科尔巴）、国家热电有限公司（{{lang|en|National Thermal Power Corporation Ltd}}，科尔巴）和SE煤炭公司（{{lang|en|S.E. Coal Ltd}}），在分裂后，经济取得了进一步的增长。科尔巴区是一个国家级的电力中枢，为印度其他数个邦提供电力。然而恰蒂斯加尔邦的电力短缺却十分严重，尤其在炎热的夏天供电很难保障，居住条件很差。恰蒂斯加尔邦的南部地区[[铁矿|铁矿]]丰富，丹德瓦达区有一家NMDC公司开采此处铁矿为印度市场提供原料，同时一部分铁矿也被送往[[日本|日本]]等其他国家。近来ESSAR公司也已经开通了一条连通[[维萨格|维萨格]]（{{lang|en|vizag}}）的铁矿运输线路。

== 交通 ==
恰蒂斯加尔邦[[铁路|铁路]]网以比拉斯布尔为中心，通过赖加尔、杜尔格、詹吉查姆帕等几大城市连同印度各地。同时，比拉斯布尔也是印度东南地区的铁路运输中心。

该邦的公路建设发展较慢，[[孟买|孟买]]通向[[加尔各答|加尔各答]]的NH 6号公路穿越该邦，邦内另有一条连通赖普尔和维萨格的公路NH 43。

该邦的空中运输规模较小，首府赖普尔有全邦唯一一座商业运营机场，已开通了通往印度四个主要城市[[德里|德里]]、孟买、加尔各答和[[金奈|金奈]]的航线。在本邦内，比拉斯布尔建有一座小型机场，[[比莱|比莱]]（{{lang|en|Bhilai}}）、[[杰格德尔布尔|杰格德尔布尔]]（{{lang|en|Jagdalpur}}）、苏古贾、科尔巴各建有简易跑道。

== 文化 ==
恰蒂斯加尔邦是印度发展较慢的邦之一，2001年人均收入约为250美元，城市人口约20％。大多数地区，一些基本生活设施缺乏，许多地区[[电话|电话]]都没有接通，但在邦内几个主要地区，[[移动电话|移动电话]]业务已经开通。

恰蒂斯加尔邦有接近34％的设籍部落（{{lang|en|Scheduled Tribe}}），12％的设籍阶级（{{lang|en|Scheduled Caste}}），超过50％的其他背景的阶层。全部人口中超过96％的人口属于印度种姓（{{lang|en|Hindu castespage}}）。该邦有多宗教教派，主要有[[萨特拿米教|萨特拿米教]]（{{lang|en|Satnami}}）、[[Kabirpanth教|Kabirpanth教]]、[[Ramnami_Samaj教|Ramnami Samaj教]]，其他教派主要为反对当地世袭统治的外来人群。

因为当地特色的社会形式和复杂的社会群体，恰蒂斯加尔邦很受[[人类学家|人类学家]]和[[社会学家|社会学家]]偏爱。

在恰蒂斯加尔邦，每到周日，许多集市将在一些特色的村庄举办，当地政府正在努力发展当地的特色农庄，鼓励私营机构对一些重要的遗址进行专业的管理和有效的维护。

* [[查姆帕兰|查姆帕兰]]（{{lang|en|Champaran}}），位于赖普尔的一个小镇，以宗教圣地闻名于世。

== 女性权益 ==
恰蒂斯加尔邦[[女性|女性]]人口比例较高，在所有邦中位列第二位，农村人口中女性比例高于[[男性|男性]]，城市人口中男性比例高于女性。

印度社会中，女性权益较多受到限制，但恰蒂斯加尔邦乡村社会受印度教正统宗教影响较小，许多文化风俗都保有当地特色。因此尽管当地仍较为贫穷，但相对于印度许多其他邦而言，该邦的女性却享有相对较高的权利，女性较为独立，能够参与社会组织，发表社会言论，能够自主选择[[婚姻|婚姻]]，甚至能够自主选择[[离婚|离婚]]。当地许多古老的[[庙宇|庙宇]]都与女性有关，如：Shabari，摩诃摩耶（{{lang|en|Mahamaya}}），Danteshwari等。

但是，当地的也同样有根据当地政府的调查资料显示，当地20岁至49岁妇女的平均结婚年龄在15.4岁，15岁至19岁的女孩结婚率高达34％。此外，当地的社会信仰普遍相信[[巫术|巫术]]的存在，女性则被认为能够使用某种超自然力，但一些人也质疑巫术的存在，称其为“Tonhi”，女性也经常因此而受到迫害。

随着经济的发展，知识的普及，尤其是一些国外[[移民|移民]]的回归，许多倡导女权主义的观点也被逐步带入到印度社会之中，虽然恰蒂斯加尔邦相对比较闭塞，但也仍旧受到一些影响，也取得了一定程度上的改善。

21世纪以后，恰蒂斯加尔邦女性人口比率正逐年降低。
* 恰蒂斯加尔邦男女比例为1901年1：1.046、1941年1：1.032、1981年1：0.996、2001年1：0.99；
* 同期的印度全国的男女比例为1901年1：0.972、1941年1：0.945、1981年1：0.934、2001年1：0.933。

== 语言 ==
恰蒂斯加尔邦的语言除了印度官方语言[[北印度语|北印度语]]外，主要是一些大多数人通用的当地方言，但无论哪种语言都受到了北印度语的影响，与其有密不可分的关系。这些语言大致可归于几大语系，[[蒙达语系|蒙达语系]]（{{lang|en|Munda}}，[[南亚语系|南亚语系]]的一个分支）、[[德拉威语系|德拉威语系]]（{{lang|en|Dravidian}}）和[[印欧语系|印欧语系]]，在恰蒂斯加尔每一个语系都下属有数十种不同方言（总计93种）。其中，最主要的[[恰蒂斯加尔语|恰蒂斯加尔语]]（{{lang|en|Chhattisgarhi}}）属于印欧语系并结合了很多蒙达语系和德拉威语系词汇。

恰蒂斯加尔邦北部，三大语系的方言都有使用；恰蒂斯加尔邦中部，仅有印欧语系的方言还在被使用；恰蒂斯加尔邦南部，则德拉威语系和印欧语系的方言都有使用。根据[[语言学家|语言学家]]H.L. Shukla的研究，蒙达语系的方言正进入加速消亡阶段；德拉威语系的方言中，[[帕尔吉语|帕尔吉语]]（{{lang|en|Parji}}）也面临很严重的消亡的问题，[[库卢克语|库卢克语]]（{{lang|en|Kurukh}}）和[[龚德语|龚德语]]（{{lang|en|Gondi}}）则在艰难地生存；而印欧语系所面临的问题则是多种方言正在丢失他们的一致性。

=== 蒙达语系 ===
根据1971年印度[[人口普查|人口普查]]，[[科尔库语|科尔库语]]（{{lang|en|Korku}}）是有超过200,000人使用的主要的蒙达语系方言（尚不清楚这个统计是针对印度全国还是仅限于恰蒂斯加尔邦）。另有[[卡里亚语|卡里亚语]]（{{lang|en|Kharia}}）和[[科尔巴语|科尔巴语]]（{{lang|en|Korba}}也是恰蒂斯加尔邦主要使用的蒙达语系方言。

与以上三种主要方言相比，[[慕阿斯语|慕阿斯语]]（{{lang|en|Muasi}}）、[[图日语|图日语]]（{{lang|en|Toori}}，赖加尔地区）、[[尼哈利－曼卡日语|尼哈利－曼卡日语]]（{{lang|en|Nihali-Mankari}}）、[[哈尔瓦日语|哈尔瓦日语]]（{{lang|en|Khaerwari}}）、[[比好德语|比好德语]]（{{lang|en|Birhord}}）、[[科达库语|科达库语]]（{{lang|en|Kodaku}}，苏古贾地区）、[[德亥尔其语|德亥尔其语]]（{{lang|en|Dhelki}}）、[[马哈托语|马哈托语]]（{{lang|en|Mahto}}）、[[科拉－马吉语|科拉－马吉语]]（{{lang|en|Kora-Majhi}}）、[[蒙达语|蒙达语]]（{{lang|en|Munda}}）和[[桑塔尔语|桑塔尔语]]（{{lang|en|Santhali}}）等其他的蒙达语系方言，则为次方言或正在消亡的方言。

已知的唯一一个[[盖德巴语|盖德巴语]]（{{lang|en|Gadba}}）的使用者（在巴斯塔地区）已经在80岁时逝世。

=== 德拉威语系 ===
根据1971年印度人口普查，恰蒂斯加尔邦有300,609人以[[库卢克语|库卢克语]]（{{lang|en|Kurukh}}）或[[奥昂语|奥昂语]]（{{lang|en|Oraon}}）作为母语，约有30000人使用[[帕尔佳语|帕尔佳语]]（{{lang|en|Parja}}）或[[杜尔比语|杜尔比语]]（{{lang|en|Dhurbi}}）。

[[龚德语|龚德语]]是一种主要由[[龚德人|龚德人]]使用的德拉威语，龚德族自称为Koitor或Koitol，因此语言学家H.L. Shukla也将他们的语言称作Koitor，并整理出了隶属于此语言之下的[[哆米语|哆米语]]（{{lang|en|Dormi}}）、[[丹达米－玛依语|丹达米－玛依语]]（{{lang|en|Dandami-Maria}}）、[[胡依语|胡依语]]（{{lang|en|Bhuria}}）、[[布吉－玛依语|布吉－玛依语]]（{{lang|en|Abujh-Maria}}）、[[考亚语|考亚语]]（{{lang|en|Koya}}）和[[高图－穆里亚语|高图－穆里亚语]]（{{lang|en|Ghotul-muriya}}）多种语言。

根据1961年印度人口普查，印度总共有3,900,000龚德人。1971年的人口普查显示他们中的三分之二都分布在中央邦。至今，几乎超过一半的龚德人还使用他们自己的方言。

=== 印欧语系 ===
在恰蒂斯加尔邦93种方言中，有70种被归类于[[印欧语系|印欧语系]]，其中最主要的一种语言即为[[恰蒂斯加尔语|恰蒂斯加尔语]]，其下又包括[[嘎依亚语|嘎依亚语]]（{{lang|en|Agariya}}）、[[宾吉瓦依语|宾吉瓦依语]]（{{lang|en|Binjhwari}}）、[[拜嘎尼语|拜嘎尼语]]（{{lang|en|Baigani}}）、[[布里亚语|布里亚语]]（{{lang|en|Bhuliya}}）、[[拉依亚语|拉依亚语]]（{{lang|en|Lariya}}）、[[丹瓦尔语|丹瓦尔语]]（{{lang|en|Dhanwar}}）、[[潘卡语|潘卡语]]（{{lang|en|Panka}}）、[[丁德瓦尔语|丁德瓦尔语]]（{{lang|en|Dindwar}}）等许多方言。除此之外，与恰蒂斯加尔语同属印欧语系的[[沙德里语|沙德里语]]（{{lang|en|Sadri}}）也是各部族之间能够相互通用的一种语言，甚至从恰蒂斯加尔邦和奥利沙邦到西孟加拉的范围内都有使用。此外，[[亥比语|亥比语]]（{{lang|en|Halbi}}）是在恰蒂斯加尔南部地区各部族较为通用的一种语言（早前曾有不同的观点认为亥比语应定位为一个独立语系，但现在一般将其作为印欧语系的一种方言。），在1951年的人口普查中，调查发现有99％的亥比语使用者能够使用至少另外一种方言。
