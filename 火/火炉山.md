
'''火炉山'''是[[中华人民共和国|中华人民共和国]][[广东省|广东省]][[广州市|广州市]][[天河区|天河区]]的一个[[免费|免费]][[森林公园|森林公园]]，南面是[[岑村|岑村]]火炉山南路，与[[育新街|育新街]]、[[华观路|华观路]]、[[科韵路|科韵路]]相连，北面为[[柯木塱|柯木塱]]，与[[广汕公路|广汕一路]]、柯木塱南路相连，东面有长700米的东门登山路径<ref>{{Cite web |url=http://www.southcn.com/nfdaily/bo/timeline/content/2014-03/24/content_95804901.htm|title=升级版火炉山森林公园今天正式开门迎客|accessdate=2014-03-24 |work= |publisher=南网 |date=2014-03-24 }}</ref>至[[天河智慧城|天河智慧城]]核心区（高唐）、[[凌塘|凌塘]]村，西侧连至龙眼洞林场。南北有环山公路相连接<ref>{{Cite web |url=http://politics.caijing.com.cn/2014-05-07/114159739.html|title=骑车登山有了新去处火炉山环山公路投入使用|accessdate=2014-05-07 |work= |publisher=财经网 |date=2014-05-07 }}</ref>，最高点白架顶，海拔321.8米，还有猪头石、鸡枕石、三间二廊、竹园广场等景点，南面有[[周恩来|周总理]]视察岑村纪念林。据统计，火炉山森林公园多个地段的[[负离子|负离子]]普遍达到每[[立方厘米|立方厘米]]3500—4000个 <ref>{{Cite web |url=http://news.southcn.com/d/2011-05/25/content_24558305.htm|title=火炉山负离子是办公室80倍 上班族多去洗洗肺 |accessdate=2011-05-25 |work= |publisher=南方网 |date=2011-05-25 }}</ref>。 火炉山临近[[凤凰山|凤凰山]]、[[龙眼洞|龙眼洞]]、[[华南植物园|华南植物园]]、[[广东树木公园|广东树木公园]]。 
==隐患==
2014年9月，火炉山森林公园基建项目已基本完工,但目前公园的道路[[照明|照明]]、[[公共廁所|公共厕所]]等水电配套设施尚未完善,存在一定的安全隐患。<ref>{{Cite web |url=http://news.ifeng.com/a/20140929/42111082_0.shtml|title=重阳夜登火炉山危险！|accessdate=2014-09-29 |work= |publisher=凤凰网 |date=2014-09-29 }}</ref>
==名称来源==
火炉山，因从天空俯视其形状象[[葫芦|葫芦]]，原名为葫芦山，因其山上泥土多为红泥土，空中看上去为火红色，所以称火葫芦，又简称“火炉”，因此又名火炉山。
