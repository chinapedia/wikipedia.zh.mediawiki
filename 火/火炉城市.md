'''火炉城市'''是对[[夏季|夏季]]天气酷热的[[中国|中国]]大城市的称呼，传统意义上，一般集中在中国[[长江|长江]]流域。现在，统计范围扩大了。<!-- 这里的南方指的是中国prc，不是所谓的中华民国roc，所以指南北不是台南与台北的概念，他可能是哈尔滨与武汉的对比，他可能是西安与广州的对比，夏季，夏季大陆季风太弱了，海洋季风太强了，全国普遍高温，北方不一定就比南方凉快，但是我们说南方比北方热，或者比方比南方冷，并不奇怪，另外条目中列出的最热城市排名也佐证如此，所以我觉得不需要参考来源 -->

== 成因 ==
在夏季，长江流域受西[[太平洋|太平洋]][[副热带高压|副热带高压]]控制，高温、高湿天气较多，人们形容这种感觉是像生活在“火炉”当中。

== 民间说法 ==
旧时火炉城市的说法都源于民间，并没有气温统计学上的依据。

=== 三大火炉 ===
*长江流域三个较具知名度的大城市，夏季炎热，在[[民国|民国]]时期被传称为“三大火炉”<ref>[http://www.gzkp.org.cn/show.aspx?id=180&cid=5 为什么重庆、武汉、南京有“三大火炉”之称?] {{webarchive|url=https://web.archive.org/web/20141112042247/http://www.gzkp.org.cn/show.aspx?id=180&cid=5 |date=2014-11-12 }}，广州科普资讯网</ref>。
**[[重庆|重庆]]、[[武汉|武汉]]、[[南京|南京]]

=== 四大火炉 ===
*旧时“四大火炉”：长江流域的四个夏季气候炎热的大城市被传称为“四大火炉”。有两种说法：
**[[重庆|重庆]]、[[武汉|武汉]]、[[南京|南京]]、[[长沙|长沙]]
**[[重庆|重庆]]、[[武汉|武汉]]、[[南京|南京]]、[[南昌|南昌]]
其中，人教版地理教材采用第二种说法。

=== 七大火炉 ===
*[[重庆|重庆]]、[[武汉|武汉]]、[[南京|南京]]、[[长沙|长沙]]、[[杭州|杭州]]、[[南昌|南昌]]、[[福州|福州]]

== 官方说法 ==
专业气象部门根据气温统计数据提出的火炉城市，因为统计项目和时段不同，也有不同说法。
*专业气象部门根据1981-2010年的高温天数将[[福州|福州]]、[[重庆|重庆]]、[[杭州|杭州]]评为中国“三大火炉”。<ref>http://news.house365.com/gbk/xaestate/system/2013/07/17/022296010.html</ref>
*2010年7月，中国大陸气象部门根据近20年高温情况，统计中国最热城市前10名依次为：[[福州|福州]]、[[杭州|杭州]]、[[重庆|重庆]]、[[长沙|长沙]]、[[武汉|武汉]]、[[海口|海口]]、[[南昌|南昌]]、[[广州|广州]]、[[西安|西安]]、[[南宁|南宁]]。<ref>[http://house.people.com.cn/GB/12323114.html 人民网 武汉从“四大火炉”除名 福州成中国最热城市]</ref>
*2012年8月，中國大陸[[国家气候中心|国家气候中心]]专家根据1981年到2011年共31年的气象观测资料，统计中国大陸最热的10个省会城市或直辖市为：[[重庆|重庆]]、[[福州|福州]]、[[杭州|杭州]]、[[南昌|南昌]]、[[长沙|长沙]]、[[武汉|武汉]]、[[西安|西安]]、[[南京|南京]]、[[合肥|合肥]]、[[南宁|南宁]]。<ref>[http://www.cma.gov.cn/2011xwzx/2011xqhbh/2011xdtxx/201208/t20120816_182112.html 中国气象局 国家气候中心专家：“火炉”城市越来越多]</ref>

== 争议 ==
部分人士指出：如果考虑[[湿度|湿度]]因素，[[长江流域|长江流域]]的“火炉城市”夏季[[相对湿度|相对湿度]]普遍较高，应该称为“蒸笼城市”更为准确。<ref>[http://czrb.bohaitoday.com/html/2011-07/05/content_3813395.htm 沧州日报] {{webarchive|url=https://web.archive.org/web/20110907123540/http://czrb.bohaitoday.com/html/2011-07/05/content_3813395.htm |date=2011-09-07 }}</ref>
目前，[[北京|北京]]、[[天津|天津]]、[[石家庄|石家庄]]、[[西安|西安]]等北方城市也呈现高温态势，北京、石家庄更为严重。这些现象引起人们对于[[环境|环境]]的关注。

注1：上述城市所列举的平均数据原则上取自代表该城市的国际交换站1981-2010年的数据，极端数据原则上取自1951-2016年（更新中）的数据。

注2：如果在这期间出现迁站的情况，则极端数据的统计仍然延续，平均数据的统计将视迁站后新站的代码和老站是否一致而定，如果不一致，则将分开计算，并取期间最长的一段列出（一般来说如果迁站后新站和老站之间距离较远或者海拔差距较大或者自然环境有明显差异的，将使用新的代码）。

注3：西安的平均数据采自1981-2005年的西安老站（57036），2006年起西安的国际交换站迁至泾河站（57131）。泾河站2006-2010年的平均数据不计入统计，但2006-2016年的极端数据计入统计。

注4：长沙的平均数据采自1987-2010年的望城坡站（57687），1951-1986年长沙的国际交换站是马坡岭站（57679）。马坡岭站1981-1986年的平均数据不计入统计，但1951-1986年的极端数据计入统计。

注5：上海的平均数据采自1991-2010年的宝山站（58362），1951-1990年上海的国际交换站是龙华（徐家汇）站（58367）。龙华（徐家汇）站1981-1990年的平均数据不计入统计，但1951-1990年的极端数据计入统计。
