《'''火箭兵报'''》是[[中国人民解放军火箭军政治工作部|中国人民解放军火箭军政治工作部]]主办的报纸，为中共[[中国人民解放军火箭军|中国人民解放军火箭军]]党委机关报，1991年创刊。

==简介==
1991年7月1日，《火箭兵报》创刊<ref>{{Cite web |url = http://live.people.com.cn/note.php?id=488110228085605_ctdzb_028  |title = 第11届韬奋奖获得者陈德春讲述军营中的新闻梦想 |author =  |date =  |publisher =人民网  |language =  |accessdate =  |quote =  }}</ref>。由[[中国人民解放军第二炮兵|中国人民解放军第二炮兵]]政治部主办。1991年9月12日，[[邓小平|邓小平]]为《火箭兵报》题写报头<ref>{{Cite web |url =http://www.china.com.cn/zhuanti2005/txt/2004-08/02/content_5625228.htm  |title = 1991年9月 |author =  |date =2004-08-02  |publisher = 中国网 |language =  |accessdate =  |quote =  }}</ref><ref>孙临平，《火箭兵报》的歌——纪念邓小平同志为《火箭兵报》题写报名20周年座谈会即席赋，军事记者2011年10期</ref>。

2000年代，在社长陈德春的领导下，《火箭兵报》从四开四版改成对开大报，从每周三期改成每周四期，从黑白改成全彩印刷。陈德春本人也获得了第十一届[[长江韬奋奖|长江韬奋奖]]<ref name=nian/>。

《火箭兵报》原先由中国人民解放军第二炮兵政治部主办。2015年底，在[[深化国防和军队改革|深化国防和军队改革]]中，中国人民解放军第二炮兵升格为[[中国人民解放军火箭军|中国人民解放军火箭军]]，该报改由中国人民解放军火箭军政治工作部主办。
