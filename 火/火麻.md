'''火麻'''（{{lang-en|''hemp''}}<!--防止有人将它与Sativa条目混淆-->），[[古中國|古中國]]稱'''漢麻'''、'''枲麻'''{{NoteTag|不結實的大麻，即大麻的雄株。莖皮纖維可織成夏布。}}（{{標音|注音=ㄒㄧˇ|拼音=xǐ}}）、'''苴麻'''{{NoteTag|能結子的麻，即大麻的雌株。}}（{{標音|注音=ㄐㄩ|拼音=jū}}），通称'''大麻'''，俗稱'''麻'''，是[[Cannabis_sativa|大麻]]（學名：''Cannabis sativa''）的[[变种|变种]]之一，源产于[[中國|中國]]<ref name="中國">{{Cite web|url=http://www.cmd.gov.hk/html/b5/health_info/publication/Interesting_Chinese_Herbs_2.html|title=衞生署中醫藥事務部|accessdate=2018-11-15|work=www.cmd.gov.hk|language=zh-HK|archive-url=https://web.archive.org/web/20181009022031/http://www.cmd.gov.hk/html/b5/health_info/publication/Interesting_Chinese_Herbs_2.html#|archive-date=2018-10-09|dead-url=no}}</ref>。该变种是[[THC|THC]]含量<0.3%的一种经济作物，是大麻很久以前散播到[[亚洲|亚洲]]各地的两个[[變種|變種]]之一<ref>{{Cite web|url=http://bjwb.bjd.com.cn/html/2018-07/27/content_268420.htm|title=麻：亦真亦幻的历史|accessdate=2018-11-15|last=北京晚报|work=bjwb.bjd.com.cn|language=en}}</ref>，在中國已有6000餘年的種植歷史<ref name="中國"/>，可以被當成食用作物、經濟作物、藥物與工業原料。

火麻纖維可以進行[[紡織|紡織]]，也可製造[[紙|紙]]、[[生物降解塑料|生物降解塑料]]、[[建築材料|建築材料]]、[[食品|食品]]、動物[[飼料|飼料]]以及[[燃料|燃料]]等物。主要用作工業的火麻可稱為'''工業大麻'''。火麻[[種子|種子]]可以做成藥材[[火麻仁|火麻仁]]，中醫用來治療[[便祕|便祕]]、[[腹瀉|腹瀉]]；火麻仁也可以榨取[[火麻油|火麻油]]用作烹飪，這類火麻可稱為'''食用大麻'''<ref>{{Cite web|url=http://hk.bbwc.cn/dfsn4y.html|title=價值十億新“商機”，食用類大麻能否成為下一個羽衣甘藍？ - 智趣生活 - 彭博商業周刊|accessdate=2018-11-15|work=hk.bbwc.cn}}</ref>。

人類長久以來種植[[大麻属|大麻]]作為工業纖維，植物油脂、宗教用途以及藥用。火麻种植简便、生长迅速、对环境的适应能力极强<ref>{{Cite web|url=http://www.fibercrops.com/html/kejituiguang/200912/22-74.html|title=无毒大麻---云麻一号 - 麻类作物营养与施肥网|accessdate=2018-11-16|work=www.fibercrops.com|language=zh-cn|archive-url=https://web.archive.org/web/20161221064940/http://fibercrops.com/html/kejituiguang/200912/22-74.html#|archive-date=2016-12-21|dead-url=no}}</ref>、耐贫瘠、抗逆性强、适生性广、喜光照、光合作用效率高、不需施肥<ref name=gov/>，所以大麻是一種對[[環境友善|環境友善]]的[[經濟作物|經濟作物]]。目前在許多國家{{NoteTag|如包括中國、日本、韓國、西班牙、英國、愛爾蘭、法國、加拿大、南非、埃及在内的许多国家。}}，含有微量[[四氫大麻酚|四氫大麻酚]]從而無法進行[[娛樂性用藥|娛樂性用藥]]的大麻通常允許作為一種合法的[[經濟作物|經濟作物]]。而[[大麻|用作吸食的大麻]]在世界各地通常會[[大麻的合法性|受到不同程度的管制]]，很多國家把它列為[[毒品|毒品]]的一種。

== 形态 ==
一年生[[草本|草本]]，根系发达，梢部有分支；茎梢及中部呈方形，基部为圆形，皮粗糙有沟纹，被短腺毛；掌状复叶，普遍7片披针形小叶，边缘有锯齿；花单性，雌雄异株，雄花为白色，花柄细长，圆锥花序；雌花小，无花柄，穗状花序，[[种子|种子]]为深绿色或灰褐色<ref>{{Cite book|title=精选草药彩色图谱 1|last=汪毅主编|first=|publisher=贵阳：贵州科技出版社|year=2014.05|isbn=978-7-80662-989-5|location=|pages=83}}</ref>。强行压扁时，表面有点状突起<ref>{{Cite book|title=中华本草 2|last=国家中医药管理局《中华本草》编委会编|first=|publisher=上海：[[上海科学技术出版社|上海科学技术出版社]]|year=1999|isbn=7-5323-5106-8|location=|pages=549}}</ref>。

== 用途 ==
火麻的用途很廣泛，主要用于纺织、造纸、製[[绳|绳]]等<ref name=gov>{{Cite web|url=http://www.gov.cn/jrzg/2009-07/21/content_1370717.htm|title=云南成功培育毒性均低于0.3%的4个工业大麻品种|accessdate=2018-11-15|work=www.gov.cn|archive-url=https://web.archive.org/web/20090825060347/http://www.gov.cn/jrzg/2009-07/21/content_1370717.htm#|archive-date=2009-08-25|dead-url=no}}</ref>。

大麻在[[中國|中國]]有六千多年的種植歷史<ref name="中國"/>，是重要的經濟作物。傳統的[[六谷|六谷]]就包括大麻。其韌皮的[[纖維|纖維]]可以用於紡織粗、細麻[[布|布]]，製造繩索、麻線等，或用於造[[紙|紙]]。火麻種子可以加工成[[中藥|中藥]]「[[火麻仁|火麻仁]]」，火麻仁榨成的油称作「火麻油」。亦可將火麻仁和芝麻用慢火炒到金黃色，然後將它們打成類似顆粒狀的物質後加水用[[紗布|紗布]]隔渣，再將火麻仁汁加熱即為[[火麻仁茶|火麻仁茶]]<ref>{{cite web |url = http://www.cuhk.edu.hk/scm/CMPromotionGroup/news/060717.html |title=存档副本 |accessdate=2012-04-10 |deadurl=yes |archiveurl=https://web.archive.org/web/20110615005949/http://www.cuhk.edu.hk/scm/CMPromotionGroup/news/060717.html |archivedate=2011-06-15 }}</ref>。

==各國種植狀況==

===中國===
中國的法律中，[[四氢大麻酚|四氢大麻酚]]含量低于0.3%（干物质重量百分比）的大麻属原植物及其提取产品，稱為工业大麻，為合法產品<ref>{{Cite web |url=http://www.law-lib.com/law/law_view.asp?id=330738# |title=2009年9月21日云南省人民政府第30次常务会议通过的《云南省工业大麻种植加工许可规定》第二条第1款 |access-date=2014-08-22 |archive-url=https://web.archive.org/web/20140826120615/http://www.law-lib.com/law/law_view.asp?id=330738# |archive-date=2014-08-26 |dead-url=no }}</ref>，云南、黑龙江等地均有工业大麻种植场。
