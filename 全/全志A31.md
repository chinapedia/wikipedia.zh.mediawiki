A31是一款基于[[ARM架構|ARM Cortex-A7]]的四核移动应用处理器，同时配备Imagination公司的[[PowerVR|PowerVR]] SGX544 MP2 GPU <ref name=A31>[http://www.pcpop.com/doc/0/863/863764.shtml A7四核芯片全志A31深度解析],泡泡网。</ref>。A31支持[[4K解析度|4Kx2K]]视频解码，[[蓝光|蓝光]]3D，支持四路720P同时编解码。

== A31规格<ref name=A31规格>[http://www.allwinner.com/product/A31 全志A31简介]{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }},全志官网。</ref> ==
'''CPU'''
* 四核 Cortex-A7
* 1MB L2 Cache
* 256KB L1 Cache
'''GPU'''
* PowerVR SGX544 MP2
* 8 logic cores
* 遵守Open GL ES2.0, Open CL1.x, DX9_3
'''Video'''
* 4K视频解码
* 全高清视频解码
* BD Directory BD ISO, BD m2ts视频解码
* H.264 HP 1080P@60fps视频编码
* 3840x1080@30fps 3D解码
* 3840x1080@30fps 3D编码
* 遵守[[RTSP|RTSP]],[[HTTP|HTTP]],HLS,RTMP,MMS五大流媒体协议
'''Display'''
* 双通路[[LVDS|LVDS]] 1920X1080@60fps
* 4-lane MIPI DSI 1920X1200@60fps
* 双路 RGB/CPU 2048X1536@60fps
* 内置HDMI 1.4 4Kx2K@30fps
'''Memory'''
* 双通道/16位NAND
* 双通道/64位DDR
'''Camera'''
* 内置Parallel&MIPI接口
* 内置ISP，支持 Raw data CMOS
* 支持500万，800万，1200万摄像头
* 支持8/10/12位 RGB bayer sensor
* 支持1080P@60fps录像，可实现人脸检测，全景拍摄等
'''PMIC'''
* A31定制电源芯片AXP221
* 支持21路输出，充电电流高达2.2A
* 遵守USB3.0充电标准
'''Audio'''
* 内置HI-FI 100dB音频编解码
* 3个MIC

'''OS'''
* Android 4.2及以上
* Windows RT
