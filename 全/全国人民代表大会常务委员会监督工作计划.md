'''全国人民代表大会常务委员会监督工作计划'''（简称'''全国人大常委会监督工作计划'''）是[[全国人大常委会|全国人大常委会]]每年制定的对本年度全国人大常委会监督工作的具体安排。

==简介==
2006年8月27日第十届全国人民代表大会常务委员会第二十三次会议通过《中华人民共和国各级人民代表大会常务委员会监督法》，第八条规定：“各级人民代表大会常务委员会每年选择若干关系改革发展稳定大局和群众切身利益、社会普遍关注的重大问题，有计划地安排听取和审议本级人民政府、人民法院和人民检察院的专项工作报告。”“常务委员会听取和审议专项工作报告的年度计划，经委员长会议或者主任会议通过，印发常务委员会组成人员并向社会公布。”<ref>{{Cite web |url =http://www.npc.gov.cn/wxzl/wxzl/2006-09/26/content_354987.htm  |title = 中华人民共和国各级人民代表大会常务委员会监督法 |author =  |date = 2006-08-27 |publisher = 中国人大网 |language =  |accessdate =  |quote =  }}</ref>另外，这部《监督法》及《中华人民共和国立法法》还对各级人民代表大会常务委员会审查和批准决算、对规范性文件的备案审查等方面职权进行了规定<ref name=ellq/>。

2006年12月15日第十届全国人民代表大会常务委员会第五十四次委员长会议通过《全国人大常委会2007年监督工作计划》，这是全国人大常委会在《监督法》施行后出台的首个年度监督工作计划。《全国人大常委会2007年监督工作计划》主要包括：（1）听取和审议“[[一府两院|一府两院]]”的专项工作报告（2）审查和批准决算，听取和审议审计工作和计划执行情况的报告（3）法律实施情况的检查（4）规范性文件的备案审查<ref name=ellq>{{Cite web |url =  http://www.npc.gov.cn/wxzl/gongbao/2007-05/29/content_5366931.htm#|title =  全国人大常委会2007年监督工作计划|author =  |date =2006-12-15  |publisher = 中国人大网 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://news.sohu.com/20070228/n248401905.shtml |title =全国人大常委会出台首个年度监督工作计划  |author =  |date =2007-02-28  |publisher =搜狐  |language =  |accessdate =  |quote =  }}</ref>。

此后年度监督工作计划的内容不断拓展。例如《全国人大常委会2017年监督工作计划》主要包括：（1）听取和审议专项工作报告（2）听取和审议计划、预决算监督报告（3）检查法律的实施情况（4）开展专题询问（5）开展专题调研（6）做好宪法实施监督的相关工作<ref name=elyq/>。
