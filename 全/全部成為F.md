{{日本作品|全部成為F|すべてがFになる|THE PERFECT INSIDER}}，是[[日本|日本]][[推理小說|推理小說]]作家[[森博嗣|森博嗣]]之「[[犀川&萌繪系列|犀川&萌繪系列]]」的第一本著作。日語原文由[[講談社|講談社]]於1996年出版，中譯本由尖端於2005年出版。《全部成為F》曾獲第一屆[[梅菲斯特賞|梅菲斯特賞]]（{{lang|ja|メフィスト賞}}）。

== 劇情簡介 ==
故事敘述建築系準教授犀川創平、以及同系學生西之園萌繪，為了見傳說中的天才[[程式設計師|程式設計師]]真賀田四季，而一同旅行來到真賀田研究所，結果發現了穿著婚紗、兩手兩腳都被切斷的詭異[[屍體|屍體]]，於是兩人開始嘗試解開[[密室殺人|密室殺人]]的真相。

== 登場人物 ==
=== 主角 ===
; {{nihongo|犀川創平|犀川 創平（さいかわ そうへい）|[[聲優|聲優]]：[[加瀨康之|加瀨康之]]}}
: 本作男主角，國立N大學工學部建築系[[副教授|副教授]]，曾是西之園萌繪父親的學生，在萌繪失去雙親後一直關照她。擁有非凡的邏輯思維。

; {{nihongo|西之園萌繪|西之園 萌絵（にしのその もえ）|聲優：[[種崎敦美|種崎敦美]]}}
: 本作女主角，國立N大學工學部建築系1年級生，19歲。年幼時因最自豪的魔術表演被犀川破解而曾厭惡他，但在雙親因[[空難|空難]]喪生陷入低潮時受犀川照顧才漸走出陰霾，轉而對他抱有好感。腦袋聰明，觀察細微，有跳躍式思維，但也常受感情左右。

=== 真賀田研究所 ===
; {{nihongo|真賀田四季|真賀田 四季（まがた しき）|聲優：[[木戶衣吹|木戶衣吹]]}}
: 真賀田研究所的博士，天才程式設計師，年幼時就發揮超常的才智而成名。14歲時真賀田研究所落成那天突然殺害自己雙親而引起社會譁然，因未成年且被視為[[無意識|無意識]]行為而判無罪，之後便把自己關在研究所房裡進行研究，過著不與房外人直接接觸的生活。其間開發出「Red magic」的獨家[[作業系統|操作系统]]，並做出名為「戴博拉」的研究所管理程序。

: 15年後突然出現在螢幕前再度接受媒體間接採訪，身穿著純白的洋裝並戴著手套，樣貌幾乎與15年前無異。曾接受萌繪申請與她私下會談，談話中冒出許多意味不明的語句而令她不知所措。之後犀川與萌繪親自來會面時，卻成一具穿著婚紗失去四肢的屍體，並在電腦裡的日記留下「當全部成為F」的不明語句和一台名為道流的開鎖用機器人。

: 具有[[多重人格|多重人格]]，其角色分別為栗本其志雄、佐佐木棲麻和真賀田道流，全為四季身邊已故的要人或人偶。有比犀川更加非凡的邏輯思維，追尋自由且不受世間[[道德|道德]]觀念影響。

: 真身並沒有死去，屍體與螢幕前現身的實際上是15年前與新藤清二發生不倫之戀下在房內所生的女兒，因此沒人知道其間房內有兩個人。原計畫是讓女兒15年後殺死自己與所長，卻因她跟萌繪的會談造成思想變動，才變更計畫由自己親手殺死女兒跟所長，並偽裝成真賀田未來混在犀川的學生中一同離開。引誘警方去東京後再度與犀川見面，之後又失去蹤影。

; {{nihongo|新藤清二|新藤 清二（しんどう せいじ）|聲優：[[咲野俊介|咲野俊介]]}}
: 真賀田研究所的所長，四季的叔叔。在犀川來訪那天駕駛直升機去接應真賀田未來，回來時得知四季遇害的事件被請求用直升機上的無線電與警方聯絡，然而隨後也遇害而沒能及時報警。

: 15年前目擊過四季殺害雙親。曾與年輕時的四季發生不倫之戀。
