'''全球移动互联网大会'''（{{lang-en|'''G'''lobal '''M'''obile '''I'''nternet '''C'''onference}}，简称'''GMIC'''）是由基于[[北京|北京]]长城会（{{lang-en|'''G'''reat '''W'''all '''C'''lub}}，简称'''GWC'''）<ref>http://www.linkedin.com/company/1594648?trk=tyah</ref>{{deadlink}} 每年所举办的亚洲区最有影响力的[[移动互联网|移动互联网]][[会议|大会]]，侧重[[移动互联网|移动互联网]]行业里的新兴[[创新|创新]]和竞争。<ref name="GMICsite">{{cite web|url=http://thegmic.com/|title=Global Mobile Internet Conference|work=thegmic.com}}</ref> 它始于2009年。2013GMIC大会于2013年10月22日到23日在[[旧金山|旧金山]] Moscone 中心举办。<ref name=GMICsite>{{cite web|url=http://sv.thegmic.com/|title=GMIC|work=GMIC}}</ref> 其主要意图是增强全球移动互联网<ref>{{Cite web |url=http://www.linkedin.com/company/global-mobile-internet-conference-gmic- |title=存档副本 |access-date=2013-10-10 |archive-url=https://web.archive.org/web/20121214093748/http://www.linkedin.com/company/global-mobile-internet-conference-gmic- |archive-date=2012-12-14 |dead-url=yes }}</ref> 执行管之间的合作和机遇。全球移动互联网大会有三大吸引特色：主舞台，G-Startup竞争和appSpace。自始，GMIC大会在[[北京|北京]], [[旧金山|旧金山]]举办。<ref name="GMICsite"/>
