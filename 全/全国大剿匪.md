'''全国大剿匪'''是在[[中华人民共和国|中华人民共和国]]成立后，于1950年开始的消灭[[中华民国政府|中华民国政府]]在[[中国大陆|中国大陆]]残余军事力量及[[土匪|土匪]]的全国性军事行动，是为[[第二次国共内战|第二次国共内战]]的延续。它与同期开始的[[镇压反革命运动|镇压反革命运动]]彻底消灭了中华民国政府在中国大陆的残余軍事力量，肃清了中国大陆境内反对[[中华人民共和国政府|中华人民共和国政府]]的武装势力，巩固了中华人民共和国政权。

== 背景 ==

[[第二次国共内战|第二次国共内战]]初期，[[中国共产党|中国共产党]]在攻占的[[中国东北地区|东北地区]]后，即开展“[[东北剿匪|剿匪]]”行动。1949年，随着第二次国共内战进入尾声，[[中华民国政府|中华民国政府]]军事败局已定。3月，[[中共七届二中全会|中共七届二中全会]]提出<ref name="a1"/>，[[中国人民解放军|中国人民解放军]]攻占[[中国南方|南方]]后，解放军“首要任务之一就是消灭国民党的反动武装力量，在乡村中则是首先有步骤地展开清剿[[土匪|土匪]]的斗争。”4月，解放军在[[渡江战役|渡江战役]]中，攻占[[中华民国首都|中华民国首都]][[南京市_(中華民國)|南京]]，随后[[上海戰役|攻占全国经济中心上海]]，标志中国共产党基本取得第二次国共内战的胜利。此后，解放军迅速向南方、西北方推进，逐步占领中国大陆全境。至10月1月，中华人民共和国成立。11月，解放军取得[[西南战役|西南战役]]胜利，消灭了[[中华民国国军|中华民国国军]]在中国大陆的最后一支重兵集团，控制[[中国西南地区|西南地区]]，至1950年初，基本攻占中国大陆全境。但中华民国政府残存的军事力量仍极为庞大，除中华民国国军残部、[[地方武装|地方武装]]外，各类[[杂牌军|杂牌军]]、[[土匪|土匪]]活跃<ref>{{Cite web |url=http://www.hnszw.org.cn/data/news/2015/12/80100/ |title=剿匪、镇反和抗美援朝|accessdate=2017-04-04 |author= |date=2015-12-17 |work= |publisher=海南史志网，来源：中国共产党海南历史（第二卷）|language=Zh-hans }}</ref>。

中华人民共和国建立之初的1950年1月至10月，发生试图颠覆新生政权的[[武装暴动|武装暴动]]816起。效忠中华民国政府的“匪特”在西南地区攻打、攻陷[[县城|县城]]计百余座，近四万名中共党员及新政府[[干部|干部]]、支持者被杀害<ref name="a1"/>。[[中南军区|中南军区]]广西剿匪工作考察组的报告指出，1950年整个[[廣西省_(中華人民共和國)|广西]]“匪情”严重，以[[宜山县|宜山地区]]为例，反政府武装控制区占55%，活跃区约占13%，当地政府控制区仅占31.9%<ref name="a3">{{Cite web |url=http://www.hprc.org.cn/gsyj/yjjg/zggsyjxh_1/gsnhlw_1/d13jgsxsnhlw/201411/t20141105_300680.html |title=毛泽东与广西剿匪|accessdate=2017-04-04 |author=姬文波 |date=2015-06-30 |work= |publisher=中华人民共和国国史网，来源：国史网 2014-11-05，第十三届国史学术年会论文集 |language=Zh-hans }}</ref>。[[华东地区|华东]]、[[中南地区|中南]]、[[中国西南地区|西南]]、[[中国西北地区|西北地区]]的边沿地带、偏远山区，是中华人民共和国政府“剿匪”的重点地区<ref name="a1">{{Cite web |url=http://www.zgdsw.org.cn/n/2015/0817/c244520-27474614.html |title=大规模的剿匪斗争|accessdate=2017-04-03 |author= |date=2015-08-17 |work=责编：吴伟锋 |publisher=中国共产党历史网，来源：中共中央党史研究室著：《中国共产党历史》（第二卷）（1949-1978），中共党史出版社2011年1月版|language=Zh-hans }}</ref>。

== 剿匪经过 ==

1950年3月16日，[[中国共产党中央委员会|中国共产党中央委员会]]、[[中央军事委员会|中央军事委员会]]发布《剿灭土匪，建立革命新秩序》的指示<ref name="a2">{{Cite web |url=http://www.people.com.cn/GB/historic/0316/860.html |title=1950年3月16日 全国展开大规模剿匪斗争 |accessdate=2017-04-03 |author= |date=2003-08-01 |work= |publisher=人民网|language=Zh-hans }}</ref>：必须明确，剿灭土匪，是当前全国革命斗争不可超越的一个重要阶段，是建立和恢复各级人民政权，以及开展其他一切的必要前提，是彻底消灭国民党在大陆的残余武装，迅速恢复革命新秩序的保证。中国人民解放军调出39个军140个师计150万人参加全国大剿匪<ref name="a2"/>。10月，[[中国人民解放军|中国人民解放军]]以[[中国人民志愿军|中国人民志愿军]]的名义参加[[朝鲜战争|朝鲜战争]]，部分完成剿匪任务的解放军被派往朝鲜战场<ref name="a3"/>。

1951年上半年，中国各地已剿匪逾百万。军事行动持续至1953年，消灭、投降者260余万人<ref name="a1"/>（或计240万人<ref name="a2"/>），至此中华民国政府在中国大陆的残余军事力量被消灭殆尽。
