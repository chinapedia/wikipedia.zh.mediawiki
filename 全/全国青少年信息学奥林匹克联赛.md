'''全国青少年信息学奥林匹克联赛'''（{{lang-en|'''National Olympiad in Informatics in Provinces'''}}，简称：'''{{lang|en|NOIP}}'''）是[[中华人民共和国|中华人民共和国]]组织的、面向中国（含中港澳）中学生的[[信息学|信息学]]竞赛<ref>{{Cite web|url = http://www.noi.cn/newsview.html?id=66&hash=9CB0C5&type=5|title = 全国青少年信息学奥林匹克竞赛系列活动简介|accessdate = 2018-11-08|author = CCF|date = |publisher = |archive-url = https://web.archive.org/web/20181108184504/http://www.noi.cn/newsview.html?id=66&hash=9CB0C5&type=5|archive-date = 2018-11-08|dead-url = no}}</ref>，由[[中国计算机学会|中国计算机学会]]主办，其中[[香港|香港特别行政区]]由[[香港電腦教育學會|香港電腦教育學會]]承办<ref>{{Cite web|url=http://hkoi.org/zh/noip-intro/|title=NOIP 簡介 {{!}} 全國青少年信息學奧林匹克聯賽(香港賽區)|accessdate=2017-02-26|author=|date=|publisher=|archive-url=https://web.archive.org/web/20170227063931/http://hkoi.org/zh/noip-intro/|archive-date=2017-02-27|dead-url=no}}</ref>。初赛在每年10月某个星期六，复赛在每年11月某个周末举办。

== 背景 ==

1984年2月16日，[[邓小平|邓小平]]参观上海市展览馆举办十年科技成果展。邓小平在观看过计算机演示后指出“计算机的普及要从娃娃做起”。<ref>{{Cite web|url =http://news.sohu.com/2004/02/16/54/news219085448.shtml|title =廿年前一句鼓励话 “计算机娃娃”成微软研究员|accessdate =|author =|date =|publisher =|archive-url =https://web.archive.org/web/20170105214257/http://news.sohu.com/2004/02/16/54/news219085448.shtml|archive-date =2017-01-05|dead-url =no}}</ref>同年，[[中国计算机学会|中国计算机学会(CCF)]]创办[[NOI|全国青少年计算机'''程序设计'''竞赛(NOI)]]。当年参加竞赛的人数超过8000人<ref name=":0">{{Cite web|url = http://www.noi.cn/about/summary|title = 全国青少年信息学奥林匹克竞赛系列活动简介-背景|accessdate = 2016-01-16|author = |date = |publisher = |archive-url = https://web.archive.org/web/20171028073327/http://www.noi.cn/about/summary#|archive-date = 2017-10-28|dead-url = yes}}</ref>。中共中央领导[[王震|王震]]出席了首届竞赛颁奖大会，并对此项活动给予了充分肯定。<ref name=":0" />

1985年起，CCF每年固定举办[[全国青少年信息学奥林匹克竞赛|全国青少年信息学'''奥林匹克竞赛'''(NOI)]]。1995年起，举办NOIP，每年由中国计算机学会统一组织。

== 宗旨 ==
NOIP是一项面向中国(含陆港澳)中学生的信息学竞赛和普及活动。CCF官方称NOIP的宗旨是“旨在向那些在中学阶段学习的青少年普及计算机科学知识；给学校的信息技术教育课程提供动力和新的思路；给那些有才华的学生提供相互交流和学习的机会；通过竞赛和相关的活动培养和选拔优秀计算机人才”。<ref name=":0" />NOIP的官方定义是“课外性质的因材施教活动”“在业余时间自愿参加”，目的是“在更高层次上推动(信息学技术)普及”<ref name=":0" />。但事实上，NOIP的专业性比较强，在学习深度、时间、参加者素质上由较高的要求，远非“业余时间参加的课外活动”。

== 竞赛内容 ==
目前参赛者可以选择使用[[C|C]]、[[C++|C++]]、[[Pascal_(程式語言)|Pascal]]三种编程语言，初赛不同语言考题不同，复赛试题相同。具体的内容则在竞赛大纲中体现。

2016年11月1日，[[中国计算机学会|中国计算机学会]]发布通知：2020年开始，除NOIP以外的NOI系列其他赛事（包括冬令营、CTSC、APIO、NOI）将不再支持Pascal和C语言；从2022年开始，NOIP竞赛也将不再支持Pascal语言。即从NOIP2022开始，NOI系列的所有赛事将全部取消Pascal语言。在无新增程序设计语言的情况下，NOIP系列赛事自2022开始将仅支持C++语言。<ref>{{Cite web|url=http://www.noi.cn/about/rules/744-ccfnoi|title=CCF关于NOI系列赛事程序设计语言变更的公告|accessdate=2017-02-02|author=|date=|publisher=|archive-url=https://web.archive.org/web/20170203075234/http://www.noi.cn/about/rules/744-ccfnoi#|archive-date=2017-02-03|dead-url=yes}}</ref>

==考试形式==
NOIP分为'''初赛'''与'''复赛'''；按参赛对象分为'''普及组'''（主要为初中）和'''提高组'''（主要为高中）但不限于年级。提高组试题比普及组略难。

===初赛===
初赛通过笔试，筛选出10%～15%的报名者参加复赛。

===复赛===
复赛为上机现场编程。并于当日或次日公布初测结果（2011年以前）。2011年后由CCF统一测评，并于大约一周后公布结果。复赛最初为三道题，自1999年起改为4道题。以上都是于同一天测试，其中提高组为上午，普及组为下午。2011年起，NOIP提高组改为6题，分两天测试（周六、周日上午，每日三题）<ref>{{Cite web |url=http://www.noi.cn/about/rules/395-ccf-noip2011# |title=关于CCF NOIP2011有关规则的公告 |access-date=2014-07-15 |archive-url=https://web.archive.org/web/20140705165628/http://www.noi.cn/about/rules/395-ccf-noip2011# |archive-date=2014-07-05 |dead-url=yes }}</ref>。可以使用的编辑器是 [[MinGW|MinGW]] Develop Studio、Free Pascal IDE以及北航GAIT研究组开发的[https://web.archive.org/web/20170318174225/http://www.noi.cn/guide-10 GUIDE]（GAIT Universal IDE）等。自[[NOI_Linux|NOI Linux]]发行之后NOI、NOIP被要求使用NOI Linux，但许多省因为技术原因仍在使用[[Windows|Windows]]系统，其中C/C++仍在使用[[Dev-C++|Dev-C++]]作为IDE。

== 时间 ==
初赛一般在每年10月第三个星期六，复赛一般在每年11月第三个星期六举办（2011年NOIP开始提高组复赛分为周六、周日两天），竞赛时间及赛程据CCF当年规定执行。复赛参赛者要提前一天赶到考点，以交纳参赛费、领取准考证并试用机房。目前，复赛提高组的比赛时间是每场3.5小时。<ref>{{cite web|author1=中国计算机学会|title=全国青少年信息学奥林匹克联赛组织指南|url=http://history.noi.cn/about/rules/60-2008-12-26-07-29-34|accessdate=2018-11-10|archive-url=https://web.archive.org/web/20181110120426/http://history.noi.cn/about/rules/60-2008-12-26-07-29-34|archive-date=2018-11-10|dead-url=no}}</ref><ref>{{cite web|author1=浙江省青少年信息学竞赛委员会|title=浙江省青少年信息学奥林匹克竞赛实施办法（2017年11月修订）|url=http://www.xiaoxiaotong.org/AttachFile/2018/10/40050021/636765120532625951.pdf|accessdate=2018-11-10|archive-url=https://web.archive.org/web/20181110120226/http://www.xiaoxiaotong.org/AttachFile/2018/10/40050021/636765120532625951.pdf|archive-date=2018-11-10|dead-url=no}}</ref>

== 保送政策 ==
2010年11月19日，教育部宣布取消了各项奥林匹克竞赛全国决赛一等奖以下的[[保送生|高校保送]]资格，改由所在地招生委员会决定是否给予20分及以下的加分。调整政策从2011年秋季进入高中阶段一年级的学生开始适用，2010年(含)以前已进入高中阶段学习的学生，仍可适用调整前的相关政策。

根据教育部现行《普通高校招收保送生办法》中关于保送生选拔条件的规定，获得NOIP一等奖的应届高中毕业生，均具有保送进入高校就读的资格。部分地区一等奖获奖选手还享有高考加分优惠，具体情况视省招办政策而定。获奖选手可申请参加[[自主招生|高校自主招生]]和保送生考试，经高校测试通过，可享受高考降分优惠或直接保送录取。

NOIP中成绩优秀的非高三选手，可以作为省代表队成员参加全国决赛(NOI)及夏令营比赛（部分省市代表队人员须经过选拔赛决出）。NOIP获奖选手同样具有保送资格，并且成绩优秀的选手能够当场获得高校点招，免试进入名牌大学。夏令营作为全国决赛的扩大赛，本身不具有保送资格，但如果选手之前已获得NOIP一等奖，则同样可以参与现场保送。2013届及以前获得提高组复赛一等奖的高中毕业生可免高考，而通过大学的保送生考试直接被录取。

== 争议 ==
自[[NOI_Linux|NOI Linux]]发行之后NOI、NOIP被要求使用NOI Linux，但许多省因为技术原因仍在使用[[Windows|Windows]]系统，其中C/C++仍在使用[[Dev-C++|Dev-C++]]作为IDE。由于其头文件检查不严格，导致许多参加者在复赛NOI Linux的测评环境下失去题目的全部分数。<ref>{{Cite web|url=http://www.noi.cn/noi-news/noi/754-ccf-noip2016|title=CCF NOIP2016公告|accessdate=2017-02-02|author=|date=|publisher=|archive-url=https://web.archive.org/web/20170203075232/http://www.noi.cn/noi-news/noi/754-ccf-noip2016#|archive-date=2017-02-03|dead-url=yes}}</ref>

参赛者一般将NOIP视为大学“敲门砖”，2011年之前中国[[高考|高考]]较为重视在各类奥赛中获奖的“专才”，NOIP获奖者可以获得高校保送或优惠录取资格，因此参赛者多以拿奖、争取[[保送生|保送生]]资格为目的。2010年11月19日，[[中华人民共和国教育部|中华人民共和国教育部]]宣布取消各项[[奥林匹克竞赛|奥林匹克竞赛]]全国决赛一等奖以下的高校保送资格，2011年起正式施行<ref>{{Cite web|url=http://www.ce.cn/xwzx/gnsz/gdxw/201011/19/t20101119_21982817.shtml|title=教育部调整高考加分 奥赛国内奖等取消保送_中国经济网|accessdate=|author=|date=|publisher=|archive-url=https://web.archive.org/web/20160304200212/http://www.ce.cn/xwzx/gnsz/gdxw/201011/19/t20101119_21982817.shtml|archive-date=2016-03-04|dead-url=yes}}</ref>，包括NOIP在内的多项奥赛遇冷。此后NOIP参赛者多以争取[[自主招生|自主招生考试]]资格为目的。<ref>{{Cite web|url=http://www.cnbeta.com/articles/75785.htm|title=论NOIP取消保送与CCF被黑——NOIP选手自白|accessdate=2016-01-16|author=|date=|publisher=|archive-url=https://web.archive.org/web/20150127031136/http://www.cnbeta.com/articles/75785.htm|archive-date=2015-01-27|dead-url=no}}</ref>

由于信息学竞赛应试的需要，参赛者常常需要使用一些与软件设计理念相悖的技巧，例如为了在时限内完成子任务，完成选手使用自己编写的整数读入函数“快速读入”<ref>{{Cite web|url=https://blog.csdn.net/qq_40515553/article/details/80732719|title=【模板】快速读入|accessdate=2018-06-19|author=|date=|publisher=|archive-url=https://web.archive.org/web/20180813175321/https://blog.csdn.net/qq_40515553/article/details/80732719|archive-date=2018-08-13|dead-url=no}}</ref>广为流行；选手为了快速录入竞赛的程序，普遍不重视源程序的缩进、排版、变量命名；设计算法时往往过于注重实际效率和常数因子的优化<ref>{{Cite web|url=https://blog.csdn.net/ez_yww/article/details/78472427|title=一些卡常技巧|accessdate=2017-11-07|author=|date=|publisher=|archive-url=https://web.archive.org/web/20180813175038/https://blog.csdn.net/ez_yww/article/details/78472427|archive-date=2018-08-13|dead-url=no}}</ref>，忽略算法本身的理论复杂度，甚至只是针对特殊的数据而设计算法获取一定的分数。
