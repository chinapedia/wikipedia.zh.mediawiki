'''全国人大培训中心'''，隶属[[全国人民代表大会常务委员会办公厅|全国人民代表大会常务委员会办公厅]]。

==简介==
全国人大培训中心承担人大系统干部的培训工作，同时承担[[全国人大代表|全国人大代表]]培训工作<ref>张宝山，建设学习型的国家权力机关——就2006年全国人大代表培训工作访全国人大培训中心，中国人大2007(5):37-39</ref><ref name=jj>王理洲，人大干部的培训基地——全国人大干部培训中心简介，人大工作通讯1995年10期</ref>。对全国人大代表的培训是该中心承担的重要任务之一，例如[[第十二届全国人大代表|第十二届全国人大代表]]选举产生后，2013年全国人大常委会办公厅组织近2700位全国人大代表参加初任学习、1200多位全国人大代表参加履职学习，2014年到2017年已举办12期代表专题学习班及研讨班、2期少数民族代表学习班，平均每年有1000多位第十二届全国人大代表参加学习<ref>{{Cite web |url = http://news.youth.cn/gn/201703/t20170301_9195666.htm |title =人民日报盘点：2016年全国人大都干了啥？  |author =  |date = 2017-03-01 |publisher =中国青年网  |language =  |accessdate =  |quote =  }}</ref>。

该中心在多地设有培训基地：

*'''全国人大北戴河培训基地'''：即'''[[全国人大机关服务中心|全国人大机关服务中心北戴河休养院]]'''（'''碧螺湖宾馆'''），位于[[河北省|河北省]][[秦皇岛市|秦皇岛市]][[北戴河区|北戴河区]]金山路5号<ref name=jj/>。

*'''全国人大深圳培训基地'''：即'''深圳人大干部培训中心'''，位于[[广东省|广东省]][[深圳市|深圳市]][[福田区|福田区]]深南大道6002号人民大厦。原名“深圳全国人大干部培训中心”，是全国人大常委会办公厅与深圳市人大常委会办公厅合作兴建的全国人大系统最高层次培训机构。该中心于1993年筹建，1996年12月5日正式开业，是深圳市人大常委会直属的正局级[[事业单位|事业单位]]。2007年，深圳市事业单位改革，该中心的人事和经营仍由深圳市人大常委会办公厅管理，资产转至深圳市国资委。2010年起，实行深圳市人大常委会办公厅和深圳市投资控股有限公司共同监管模式。2010年7月，该中心获全国人大常委会批准冠名为“全国人大深圳培训基地”。该中心自有的人民大厦高25层，总建筑面积3万余平方米，2011年按照四星级酒店标准重新装修。该中心集培训、住宿、餐饮、写字楼为一体，设有客房、中餐厅、会议室、教室，以及3间大型会议室（深圳厅、福田厅、龙岗厅）<ref>{{Cite web|url = http://szrdpxzx.com/page/html/company.php |title =中心介绍|author =|date =|publisher =全国人大深圳培训基地 深圳人大干部培训中心 |language =|accessdate =2017-03-27|quote =|deadurl =yes|archiveurl = https://web.archive.org/web/20170317025130/http://szrdpxzx.com/page/html/company.php|archivedate =2017-03-17}}</ref>。

*'''全国人大北京培训基地'''：即[[全国人大会议中心|全国人大会议中心]]下设的'''顺喜山庄'''，位于[[北京市|北京市]][[顺义区|顺义区]][[高丽营镇|高丽营镇]]府荣街76号。“全国人大北京培训基地”匾额由[[全国人大常委会委员长|全国人大常委会委员长]][[吴邦国|吴邦国]]题写<ref>{{Cite web |url =http://www.anzhen.org/Hospitals/Journals/ArticleIndex/1066 |title =  我院圆满完成全国“两会”医疗保障任务|author =  |date = 2016-05-11 |publisher = 首都医科大学附属北京安贞医院 |language =  |accessdate =  |quote =  }}</ref>。
