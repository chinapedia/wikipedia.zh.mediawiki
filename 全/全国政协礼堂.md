'''全国政协礼堂'''，位于[[北京市|北京市]][[西城区|西城区]][[太平桥大街|太平桥大街]]23号，由[[全国政协办公厅|全国政协办公厅]]直属[[事业单位|事业单位]]'''全国政协礼堂管理处'''管理。

== 简介 ==
全国政协礼堂自1954年开始筹建，1956年竣工，欧式建筑风格，是[[中华人民共和国|中华人民共和国]]较早的重要建筑之一。该礼堂是全国政协举行[[中国人民政治协商会议全国委员会常务委员会|常务委员会]]会议、专题协商会议、各专门委员会会议等的会场。这里还曾是[[中国共产党第八次全国代表大会|中国共产党第八次全国代表大会]]的主会场<ref>[http://www.ltcppcc.com/jianjie.asp 礼堂简介]，全国政协礼堂，于2011-11-3查阅 {{webarchive|url=https://web.archive.org/web/20111026013609/http://www.ltcppcc.com/jianjie.asp |date=2011-10-26 }}</ref><ref name=jj>{{Cite web |url =http://cppcc.people.com.cn/GB/34961/51372/51373/3650486.html  |title =全国政协礼堂简介  |author =  |date = 2005-08-29 |website = 中国政协新闻网  |language =  |accessdate =  |quote =  }}</ref>。1995年全国政协礼堂进行了整体加固改造，1996年竣工后实行全面开放。礼堂南侧新建了政协文化广场<ref name="jj" />。

全国政协礼堂占地面积5600平方米，总建筑面积1.6万平方米，拥有其南侧的政协广场。是集会议、演出、餐饮、健身、娱乐休闲、社交为一体的综合性多功能场所<ref name=jj/>。

全国政协礼堂隶属于全国政协办公厅，对内称为'''全国政协礼堂管理处'''，属全国政协机关差额补贴[[事业单位|事业单位]]<ref>[http://www.ltcppcc.com/jigou.asp 机构建制，全国政协礼堂，于2011-11-3查阅] {{webarchive|url=https://web.archive.org/web/20110712183259/http://www.ltcppcc.com/jigou.asp |date=2011-07-12 }}</ref><ref name=jj/>。礼堂管理处实行处长（总经理）分工负责和科长（部门经理）负责制相结合的企业管理方式<ref name=jj/>。

全国政协礼堂管理处由[[全国政协办公厅服务开发中心|全国政协机关服务中心]]（全国政协机关服务局）举办，负责管理全国政协礼堂，为党和国家及政协系统举办重大活动提供场所和服务。全国政协礼堂管理处负责全国政协礼堂场所管理与设施维护，并提供会议与宴请场所、展览与文化活动场所及相关服务<ref>{{Cite web |url =http://www.gjsy.gov.cn/cxzl/  |title = 全国政协礼堂管理处 |author =  |date =  |publisher =事业单位在线  |language =  |accessdate = 2018-02-08 |quote =  }}</ref>。
