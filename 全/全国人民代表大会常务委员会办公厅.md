'''全国人民代表大会常务委员会办公厅'''（简称'''全国人大常委会办公厅'''）是[[全国人民代表大会常务委员会|全国人民代表大会常务委员会]]的综合办事机构，在[[全国人民代表大会常务委员会秘书长|全国人民代表大会常务委员会秘书长]]领导下工作，全国人民代表大会常务委员会副秘书长协助秘书长工作<ref name="banshi" />。

== 沿革 ==
1954年9月，全国人大常委会成立。全国人大常委会办公厅同时成立<ref>{{Cite web |url = http://news.163.com/14/0908/07/A5JS3NNC00014AEE.html |title =  我的60年人大情缘|author =温卓文  |date =2014-09-08  |publisher =网易 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url =http://cpc.people.com.cn/GB/69112/99985/100003/9743964.html  |title = 在全国《民法通则（草案）》座谈会上的讲话要点 |author = [[彭真|彭真]] |date =  |publisher =中国共产党新闻网  |language =  |accessdate =  |quote =  }}</ref>。1954年办公厅成立时，下设五室一局二处：法律室、民族室、人民接待室、编译室、顾问室、典礼局（前身是1949年设立的中央人民政府典礼局）、秘书处、总务处。1955年增设研究室<ref name=fourty> {{Cite book | author =全国人大常委会办公厅研究室  | title = 人民代表大会制度建设四十年 | location =  | publisher = 中国民主法制出版社 | date =  1991年| pages = 924-925 | ISBN =  | accessdate =  | url =  | language =  | quote =  }} </ref>。

1956年，撤销顾问室，增设国际议会活动办公室。1958年后，撤销研究室、编译室、典礼局、国际议会活动办公室、总务处。1958年7月，将秘书处改为秘书室。1965年，将办公厅秘书室下设的外事组改为外事室，办公厅由此下设五个室：法律室、民族室、人民接待室、秘书室、外事室<ref name=fourty/>。

[[文革|文革]]爆发后，1966年8月， [[康生|康生]]、[[徐冰|徐冰]]、[[刘宁一|刘宁一]]、[[周荣鑫|周荣鑫]]组成全国人大常委会机关领导小组，主持常委会机关工作。1968年6月，根据[[中共中央|中共中央]]决定，全国人大常委会机关和全国政协机关实行军管，全国人大常委会机关设立军管组。1969年11月，全国人大常委会机关在[[湖北|湖北]][[沙洋县|沙洋]]设立[[五七干校|五七干校]]，大部分人员被下放到五七干校劳动，法律室、民族室、人民接待室、秘书室、外事室随之停止工作，全国人大常委会机关在北京成立留守处, 在军代表[[李启煌|李启煌]]领导下处理日常工作。1973年3月军管组撤离<ref name=fourty/>。

1975年1月13日至17日，[[第四届全国人民代表大会|第四届全国人民代表大会]]第一次会议在北京召开。同年2月，全国人大常委会机关在[[人民大会堂|人民大会堂]]恢复办公，常委会下设民族政策研究组、宗教政策研究组、办公厅，办公厅下设秘书组、外事组、政法组、总务组<ref name=fourty/>。

1978年2月26日到3月5日, [[第五届全国人民代表大会|第五届全国人民代表大会]]第一次会议在北京召开。五届全国人大一次会议恢复设立[[全国人民代表大会代表资格审查委员会|全国人民代表大会代表资格审查委员会]]，办公厅下设秘书组、外事组、政法组、总务组、信访组<ref name=fourty/>。1979年设立机关纪委<ref name=fourty/>。

1980年，全国人大办公厅下设各组改名为室，总务组划归秘书室，调整为四个室：秘书室、政法室、外事室、信访室。1981年增设研究室，共计五个室<ref name=fourty/>。

1983年6月6日到21日，[[第六届全国人民代表大会|第六届全国人民代表大会]]第一次会议在北京召开。会议根据新《[[中华人民共和国宪法|中华人民共和国宪法]]》的规定，在全国人大设立代表资格审查委员会和民族、法律、财政经济、教育科技文化卫生、外事、华侨6个专门委员会。办公厅下设秘书室、政法室、信访室、外事室、研究室。1983年室改名为局，政法室更名为联络局，成为四局一室：秘书局、联络局、信访局、外事局、研究室。1986年5月增设行政管理局。1987年7月增设新闻局、人事局，成为七局一室。六届人大一次会议以后，设立机关党委办公室，为局级单位<ref name=fourty/>。后来，秘书局拆分为秘书一局、秘书二局，行政管理局改为机关事务管理局，还设立了离退休干部局。1988年，人民大会堂管理局由[[中共中央办公厅|中共中央办公厅]]划归全国人大常委会办公厅管理。

按照[[中共十五大|中共十五大]]关于推进机构改革的总体部署，全国各级人大、政协机关，法院、检察院机关，群众团体机关从2000年开始分别进行机构改革<ref name=quanshan/>。2000年12月26日，《中共中央关于全国人大机关、全国政协机关机构改革的意见》（中发〔2000〕18号）下发<ref>政协甘肃省第九届委员会常务委员会关于设置研究室、专门委员会的决定，甘肃日报，2003-02-22</ref><ref name=quanshan/>。按照中央精神，全国人大机关各工作委员会和专门委员会办事机构职责作适当调整，重点理顺办公厅研究室与部门专门委员会办事机构、全国人大财政经济委员会内设机构与全国人大常委会预算工委内设机构的职责关系，将资料管理、后勤服务和办刊等工作划转[[事业单位|事业单位]]承担。此次改革，全国人大机关、全国政协机关对人员编制适当精简，把精简人员编制与深化干部人事制度改革结合起来，充实了专门人才，加强业务工作力量，优化了干部队伍结构<ref name=quanshan>{{Cite web|url = http://www.xzqs.gov.cn/Article/Show?id=36232|title = 2000年全国各级人大、政协机关，法院、检察院机关，群众团体机关机构改革的情况|author = |date = 2012-02-06|publisher = 徐州市泉山区人民政府办公室|language = |accessdate = |quote = |archive-url = https://web.archive.org/web/20180317231805/http://www.xzqs.gov.cn/Article/Show?id=36232|archive-date = 2018-03-17|dead-url = yes}}</ref>。

2007年3月，撤销秘书二局，将秘书一局重新改为秘书局<ref>{{Cite web |url = http://news.cctv.com/china/20080220/100113_1.shtml |title = 十一届人大将增加专职常委人数 法工委扩编扩权 |author =  |date =2008-02-20  |publisher = 央视网 |language =  |accessdate =  |quote =  }}</ref>。

== 职责 ==
根据有关规定，全国人大常委会办公厅承担下列职能<ref name=banshi>[http://www.npc.gov.cn/huiyi/dbdh/11_2/2009-03/01/content_1475563.htm 全国人大常委会的办事机构和工作机构，中国人大网，2009-03-01]</ref>：
{{columns-list|2|
# 承担全国人大会议、全国人大常委会会议、委员长会议的各项筹备和会务工作；
# 受委员长会议的委托，拟定有关的议案草案和常委会工作的有关计划；
# 承担全国人大及其常委会行使职权的调查研究工作；
# 负责全国人大及其常委会有关报告、文件的起草和准备工作；
# 组织办理全国人大代表提出的议案及建议、批评和意见；
# 承担全国人大常委会同全国人大代表和省、自治区、直辖市的人大常委会的联络工作；
# 承办全国人大及其常委会同外国议会、议会国际组织的交往联系工作；
# 负责全国人大及其常委会的新闻发布和宣传工作；
# 办理和接待全国人大代表和人民群众的来信来访；
# 承担全国人大及其常委会的后勤保障和会议服务工作等。
}}

== 中央纪委国家监委驻全国人大机关纪检监察组 ==
