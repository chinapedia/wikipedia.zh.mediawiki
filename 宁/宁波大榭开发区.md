'''宁波大榭开发区'''是[[中国|中国]][[浙江省|浙江省]][[宁波市|宁波市]]一个[[国家级经济技术开发区|国家级经济技术开发区]]。开发区位于[[北仑区|北仑区]][[大榭街道|大榭街道]]，由[[中国中信集团公司|中国中信集团公司]]（CITIC）成片开发，设立于1993年3月<ref name="main-intro">{{cite web|url=http://www.daxie.gov.cn/|title=大榭简介|publisher=宁波大榭开发区管理委员会|accessdate=2011-10-08}}</ref>。根据宁波市人大常委会2002年修订的《宁波大榭开发区条例》，大榭开发区管理委员会作为宁波市政府派出机构代管大榭街道<ref>{{cite web|url=http://www.nbrd.gov.cn/art/2013/9/2/art_4960_1021753.html|title=宁波大榭开发区条例|publisher=宁波市人大常委会|date=2002-09-13|accessdate=2011-10-08|deadurl=yes|archiveurl=https://web.archive.org/web/20160304054116/http://www.nbrd.gov.cn/art/2013/9/2/art_4960_1021753.html|archivedate=2016-03-04}}</ref>。

== 自然地理 ==
大榭开发区位于[[大榭岛|大榭岛]]及周边岛屿。其中，大榭岛面积30.84平方公里，是区域内最大的岛屿。第二大岛屿为[[穿鼻岛|穿鼻岛]]，面积2.16平方公里<ref name="main-intro" />。大榭岛地势东南高，西北低，距离陆地最近点为0.5公里，最高峰七顶山海拔334.1米<ref name="shizhi-environment">{{cite book|title=宁波市志|year=1995|chapter=第三卷 自然环境|publisher=中华书局|isbn=978-7-101-01507-2}}</ref>。

== 重要企业 ==
* [[三菱|三菱]]化学
* 中海石油宁波大榭石化有限公司
* 韩华石油[[聚氯乙烯|聚氯乙烯]]项目
* 林德气体大型空分项目
* 宁波万华聚氨酯公司
* 招商国际集装箱码头

== 交通 ==
大榭开发区通过[[大榭跨海大桥|大榭跨海大桥]]与大陆连接，[[大榭第二大桥|大榭第二大桥]]目前正建设中，铁路[[大榭支线|大榭支线]]也正在规划中。
