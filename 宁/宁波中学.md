'''浙江省宁波中学'''<ref>{{cite web |url=http://www.nbzx.org/ |title=浙江省宁波中学 |language=zh |publisher=浙江省宁波中学 |accessdate=2017-08-07 }}</ref>，是[[中国|中国]][[浙江省|浙江省]][[宁波市|宁波市]]的一所全寄宿制[[高中|高中]]，由[[张之洞|张之洞]]等倡议创办于1898年。宁波中学始名储才学堂，后历次更名为宁波府中学堂、浙江省立第四中学、浙江省宁波市第一中学、浙江省宁波中学。1995年被评为浙江省一级[[重点中学|重点中学]]。现任校长为2016年到任的高级教师邵迎春<ref>{{cite web |url=http://www.ziyuanku.com/xiaozhang/introduction/1212.html |title=邵迎春 浙江省宁波市宁波中学 |language=zh |accessdate=2017-08-07 }}</ref>女士，她亦为该校成立近120年以来的首位女性校长。
