'''宁乡县第九高级中学'''，位于[[湖南省|湖南省]][[长沙市|长沙市]][[宁乡县|宁乡县]][[大屯营乡|大屯营乡]]，为宁乡县高级中学之一。<ref name=" bok1" >{{cite book |author=	吴石平、贺明兰 |title=《宁乡县志》|year=2008年 |publisher=方志出版社 |location= |pages= |isbn= 9787802382855 |language = zh-cn}}</ref>

== 历史 ==
1906年，[[中华民国|中华民国]][[教育家|教育家]][[朱剑凡|朱剑凡]]先生在[[大屯营乡|大屯营乡]]建立了一所学校，只接收小学生和初中生。1949年，[[中华人民共和国|中华人民共和国]][[宁乡县人民政府|宁乡县人民政府]]将学校收为国有。1969年，学校被更名为宁乡县第九高级中学，并且开始接收高中生。2011年7月10日，宁乡县第九高级中学举行百年一聚校友会。<ref>{{cite web |url=http://www.nx123.ccoo.cn/news/local/1105935.html |title=冬日暖阳耀宁乡九中 |accessdate= 2012-12-12|work= |publisher=宁乡在线 |language =zh-cn}}</ref>
