'''宁波东站'''是[[中国|中国]][[浙江省|浙江省]][[宁波市|宁波市]]一座铁路车站，曾在宁波站改建时作为临时车站。车站位于[[鄞州区|鄞州区]]站北路，福明路与桑田路之间，现代装潢市场陶瓷城东面，甬台温铁路北面。<ref name="ready-service">{{Cite web |url=http://news.cnnb.com.cn/system/2010/09/06/006668299.shtml |title=火车东站今天进行铁路设备调试 8日投用 |accessdate=2010-09-06 |publisher=中国宁波网 |date=2010-09-06 }}</ref>距[[宁波站|宁波站]]5公里，距[[奉化站|奉化站]]32公里，为[[甬台温铁路|甬台温铁路]]和[[北仑支线|北仑支线]]车站。宁波东站于1986年开通货运，最初经营危险品运输业务<ref name="ningbo-trans-cat">{{cite book|title=宁波市交通志|chapter=第四编 陆路（铁路运输）|publisher=海洋出版社|date=1996-01|isbn=7-5027-4080-5|pages=358}}</ref>，后发展成为除[[宁波北站|宁波北站]]外重要的货运站<ref>{{cite article|title=宁波东货场搬迁的必要性及新站站址方案的探讨|author=高琦|publisher=铁道勘测与设计|year=2010|issue=1}}</ref>。由于甬台温铁路的建设，[[宁波站|宁波站]]运营的列车迅速增加，加之立体交通和城市轨道交通对新建车站的需求，铁道部和宁波市政府决定对宁波站实施改建。宁波站改建期间，经过改建的宁波东站作为过渡客运站，于2010年9月8日启用<ref>{{Cite web |url=http://zjnews.zjol.com.cn/05zjnews/system/2010/08/31/016890062.shtml |title=新铁路宁波东站9月8日启用 届时南站暂停客运 |accessdate=2010-09-04 |publisher=浙江在线 |date=2010-08-31 }}</ref><ref>{{Cite web |url=http://zh.ningbo.gov.cn/zwgk/gggs/bmgz/201008/t20100830_36675.htm |title=铁路宁波东站将于9月8日零时起正式启用 |accessdate=2010-09-04 |publisher=宁波市镇海区人民政府 |date=2010-08-30 |deadurl=yes |archiveurl=https://web.archive.org/web/20131226050412/http://zh.ningbo.gov.cn/zwgk/gggs/bmgz/201008/t20100830_36675.htm |archivedate=2013-12-26 }}</ref>。2013年12月28日，宁波站改建结束后，宁波东站停运<ref name="ningbo-start-service">{{cite web|url=http://news.cnnb.com.cn/system/2013/12/25/007944089.shtml|title=12月28日起铁路宁波站投入使用 宁波东站停办客运业务|publisher=中国宁波网|date=2013-12-25|accessdate=2013-12-25}}</ref>，并计划在未来与[[庄桥站_(宁波市)|庄桥站]]一道作为辅助客运站<ref name="future-usage">{{Cite web |url=http://news.cnnb.com.cn/system/2010/09/08/006670595.shtml |title=铁路宁波东站启用 甬城将步入“一城三站”时代 |accessdate=2010-09-08 |work=陈锦源 |publisher=中国宁波网 |date=2010-09-08 }}</ref>。

== 沿革 ==
宁波东站原为宁波铁路运输货运站。车站位于[[鄞州区|鄞州区]][[潘火街道|潘火街道]]，建于1986年，为[[北仑支线|北仑支线]]车站。当时的车站包含正线1条，到发线2条，货物线2条，牵出线、存车线各1条，渡线2条，站台1座<ref name="ningbo-trans-cat" />。为辅助宁波站改造为枢纽站，宁波东站客运部分于2009年开工，2010年建成，并于当年9月8日取代宁波站成为宁波临时的主要客运站。改建的宁波东站具有3台9线，并预留1条到发线及基本站台<ref name="begin-construction">{{Cite web |url=http://g.ningbo.gov.cn/art/2009/11/12/art_4067_394394.html |title=铁路宁波东站试桩开工 |accessdate=2010-09-08 |work= |publisher=宁波市重大项目稽察办公室 |date=2009-11-12 |deadurl=yes |archiveurl=https://archive.is/20131228015653/http://g.ningbo.gov.cn/art/2009/11/12/art_4067_394394.html |archivedate=2013-12-28 }}</ref>。2013年7月，[[杭甬客运专线|杭甬客运专线]]建成通车，宁波东站办理列车对数从建成初期的53对增长到88对，逐渐达到饱和。2013年12月27日晚11时，从厦门北开往宁波东的D3212次成为宁波东站办理的最后一趟列车<ref>{{cite web|url=http://news.cnnb.com.cn/system/2013/12/28/007946683.shtml|title=再见！东站 火车东站昨晚11时送完最后一波旅客|publisher=宁波日报|date=2013-12-28|accessdate=2013-12-28}}</ref>。次日，随着宁波站正式投入营运，宁波东站停运，并进入改造阶段<ref name="ningbo-start-service" />，计划在将来作为永久辅助车站<ref name="future-usage"/>。

== 设施 ==
宁波东站站房坐南朝北，面积8000平方米，采用钢筋混凝土框架结构，外立面采用全屏玻璃幕墙。站房分为上下两层和地下一层票房。上层候车室为动车和直达列车候车室，下层为其他列车候车室。<ref name="checkin">{{Cite web |url=http://zt.cnnb.com.cn/system/2010/08/31/006661688.shtml |title=铁路东站候车方式和检票时间有变化 |accessdate=2010-09-08 |work= |publisher=中国宁波网 |date=2010-08-31 }}</ref>站台规模为3台9线（预留1条到发线及基本站台），雨棚18300平方米<ref name="begin-construction"/><ref>{{Cite web |url=http://zt.cnnb.com.cn/system/2010/08/31/006661296.shtml |title=宁波东站专题 站台数 |accessdate=2010-09-06 |publisher=中国宁波网 |date=2010-08-31 }}</ref>，采用[[人行过街天桥|天桥]]进出站台。<ref name="checkin"/>站前广场设有管理用房、商业用房、铁路行车房、出租车停靠站和公交车站，总面积1.2万平方米。<ref>{{Cite web |url=http://zt.cnnb.com.cn/system/2010/08/31/006661363.shtml |title=宁波东站专题 广场建筑 |accessdate=2010-09-06 |publisher=中国宁波网 |date=2010-08-31 }}</ref>

== 列车 ==
[[File:CRH380B-6238L_in_Ningbodong_Railway_Station.jpg|thumb]]高速动车组列车]]
宁波东站停运前平日开行旅客列车91对，其中始发、终到57对，经停34对。其中包括开往[[安徽|安徽]]、[[广东|广东]]、[[四川|四川]]、[[贵州|贵州]]等地的普通和快速旅客列车、开往[[武昌|武昌]]的一对直达特快列车（[[Z31/34、Z32/33次列车|Z31/34、Z32/33次]]）和开往[[北京|北京]]、[[天津|天津]]、[[南京|南京]]、[[上海|上海]]、[[杭州|杭州]]、[[福州|福州]]等地的[[中国铁路高速列车|高速列车]]。高峰时车站每5分钟就将办理一趟列车<ref name="railserv">{{Cite web |url=http://news.cnnb.com.cn/system/2013/06/28/007765902.shtml|title=铁路宁波东站高峰时每隔５分钟开出一趟列车|date=2013-06-28|accessdate=2013-06-30}}</ref>。

== 站外交通 ==
宁波东站站前广场设有[[出租车|出租车]]停靠点和公交始发站。<ref name="ready-service" />建成初期，共有15条公交线路始发于宁波东站，分别前往望春、宁波大学、慈城、骆驼、洪塘、孔浦、镇海中心城区、宁波客运中心、白峰码头、北仑中心城区、东钱湖和咸祥。<ref name="pub-trans">{{Cite web |url=http://zjnews.zjol.com.cn/05zjnews/system/2010/08/31/016889382.shtml |title=票哪里买公交车怎么坐 宁波东站换乘全攻略 |accessdate=2010-09-06 |publisher=浙江在线 |date=2010-08-31 }}</ref>此外，从宁波东站前往宁波客运中心和汽车南站乘车的乘客可以乘坐专线巴士，其车票可在购买客车车票时作为等值现金使用，相当于免费。<ref name="pub-trans1">{{Cite web |url=http://news.cnnb.com.cn/system/2010/08/31/006660042.shtml |title=东站站前路实行单向通行 设15条公交线路 |accessdate= 2010-09-06 |publisher=中国宁波网 |date=2010-08-31 }}</ref>将来，[[宁波轨道交通|宁波轨道交通]][[宁波轨道交通4号线|4号线]]也将到达宁波东站。<ref name="checkin"/>

== 服务 ==
宁波东站停运前设有21个售票窗口和1个独立的退票窗口。<ref>{{Cite web |url=http://zt.cnnb.com.cn/system/2010/08/31/006661482.shtml |title=宁波东站专题 票房 |accessdate= 2010-09-06 |publisher=中国宁波网 |date=2010-08-31 }}</ref>候车区域采用“先检票，后上车”的组织方式<ref>{{Cite web |url=http://zt.cnnb.com.cn/system/2010/08/31/006661495.shtml |title=宁波东站专题 进站候车 |accessdate= 2010-09-06 |publisher=中国宁波网 |date=2010-08-31 }}</ref>，开车前10分钟停止检票。<ref>{{Cite web |url=http://zt.cnnb.com.cn/system/2010/08/31/006661499.shtml |title=宁波东站专题 提前检票 |accessdate= 2010-09-06 |publisher=中国宁波网 |date=2010-08-31 }}</ref>
