'''附屬動議'''（{{lang|en|subsidiary motion}}）是一項[[議事程序|議事程序]]，因[[主動議|主動議]]而生，旨在改變主動議內容或處理方式，例如與會者可提出[[修正_(動議)|修正動議]]，增添、刪除、修改、潤飾主動議內容；或提出[[付委_(動議)|付委動議]]，將主動議轉交委員會討論、調查。根據《[[羅伯特議事規則|羅伯特議事規則]]》，依{{Link-en|位階 (議事學)|Floor (legislative)|位階}}排序，為擱置動議、[[停止討論|停止討論動議]]、[[延期討論動議|延期討論動議]]、付委動議、修正動議和[[延期討論動議|無期延期動議]]<ref name="RONR">{{Cite book|author=Henry M. Robert|coauthor=Daniel H. Honemann; Thomas J. Balch; William J. Evans; Sarah Corbin Robert|title=''Robert's Rules of Order: Newly Revised''|publisher=Da Capo Press|edition=10|language=en|isbn=073820384X|pages=60-61|date=2010年11月}}</ref>。

《{{Link-en|議事程序準則|The Standard Code of Parliamentary Procedure}}》刪除延期討論動議，以擱置替代；它還認為停止討論議已經過時，用詞容易使人混淆，應該改成「關閉討論」（{{lang|en|Close Debate}}）；擱置又可稱為臨時延期（{{lang|en|Postpone Temporarily}}），有別於《羅伯特議事規則》，要刪除一個動議案毋需討論或表決；但《議事程序準則》認為需要三分之二同意<ref>{{Cite book|author=Alice Sturgis |title=''The Standard Code of Parliamentary Procedure''|publisher=McGraw-Hill|edition=4|isbn=0071365133|pages=234-235|language=en|date=2000年12月}}</ref>。
