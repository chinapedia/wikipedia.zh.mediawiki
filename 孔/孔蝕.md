'''孔蝕'''，或稱為'''斑蝕'''、'''麻點腐蝕'''，是[[金屬|金屬]]浸置在[[溶液|溶液]]時，發生的一種局部的[[腐蚀|腐蝕]]，會在金屬形成小孔洞。孔蝕的原動力為: 小區域成為[[陽極|陽極]]；其他表面區域成為[[陰極|陰極]]，形成局部的[[电化学|电化学]]反應。孔蝕不易偵測，且孔蝕可穿透至金屬內部，短時間卻減少質量甚微，危害不容忽視。<ref name="Corrosion : Understanding the Basics">{{cite book|url=https://books.google.com.tw/books?id=D0nAMorpSIYC|author=Joseph R. Davis |title=Corrosion : Understanding the Basics |page=105|publisher=ASM International|year=2000|isbn=978-0-87170-641-6}}</ref>

== 機制 ==
[[File:Fe_corrosion.PNG|right]]
傳統的解釋為自發性的電化學反應。氧化[[半反應|半反應]]發生在孔洞內，還原半反應發生在孔洞表面周圍。當孔洞向內成長時，孔洞內端溶液的[[離子|離子]][[濃度|濃度]]會逐漸增加，提升孔蝕速率，有此可知，不流動的溶液較易發生孔蝕。又由於重力，孔洞傾向垂直方向生長。<ref>ASM Handbook, Volume 13, "Corrosion", ISBN 0-87170-007-7, ASM International, 1987</ref>

孔蝕通常腐蝕速率緩慢，重量損失十分微小，卻會深入破壞金屬結構。當伴隨其他類型的腐蝕時，表面的孔蝕經常被腐蝕產物覆蓋。

孔蝕常源自於微小的表面缺陷（刮傷表面或表面塗層），拋光後的表面具有較優的耐孔蝕性。不鏽鋼也可能發生孔蝕，添加[[鉬|鉬]]的不鏽鋼耐孔蝕性較佳。

== 孔蝕的 合金 / 環境 組合 ==
有些金屬或合金在特定環境下會失去原有的活性，此狀態稱為[[鈍化|鈍態]](Passivity)。鋁、鎳、鈦及其合金具有鈍化行為，在氧化性環境中，表面形成薄而緊密附著的氧化層，保護金屬不受近一步氧化腐蝕。但具有鈍化行為的金屬或合金還是會被腐蝕，而孔蝕被認為是此情況下局部的腐蝕([[微米|微米]]至[[毫米|毫米]]尺度)。如鋁合金會在氯離子(Cl-)溶液孔蝕。相比之下，氧化層不具保護力的合金 / 環境組合較少形成孔蝕。

除了氯離子，造成孔蝕的陰離子包含[[硫代硫酸盐|硫代硫酸根離子]] (S<sub>2</sub>O<sub>3</sub><sup>2−</sup>)、[[氟化物|氟離子]]及[[碘化物|碘離子]]。硫代硫酸根是特別強烈的腐蝕劑，為[[黃鐵礦|黃鐵礦]]氧化或[[硫酸根|硫酸根離子]]還原的產物。硫代硫酸根在許多的工程設備會造成腐蝕的問題 : [[硫化物|硫化物]]礦物冶煉設備、[[油井|油井]]及運輸[[:en:Sour_crude_oil|高硫原油]]的管線、[[牛皮纸|牛皮纸]][[造紙廠|造紙廠]]、[[甲硫氨酸|甲硫氨酸]]及[[離胺酸|離胺酸]]製造設備等。 

添加抑蝕劑於於封閉系統可大幅降低腐蝕速率。
