'''张宏根''' ({{bd|1935年|11月27日|2003年|11月25日|catIdx=Z张}})，出生于[[上海|上海]]，是[[中华人民共和国|中华人民共和国]]第一代足球运动员，作为球员曾多次代表[[中国国家足球队|中国队]]出场并进球，退役后成为教练。

== 球员生涯 ==

张宏根早年在华南队踢球。18岁时被[[中国国家足球队|中国队]]第一任主教练[[李凤楼|李凤楼]]选入国家队。在1957年的[[世界杯足球外围赛|世界杯外围赛]]中，他打入[[中国国家足球队|中国队]]在该赛事上的第一球。然而[[中国国家足球队|中国队]]终因实力不济而未出线。时有“三张”之说：张宏根、[[张京天|张京天]]、[[张俊秀|张俊秀]]。他们三人成了当时[[中国国家足球队|中国队]]的顶梁柱。1958年张宏根被评为“'''全国十佳足球运动员'''”之首。

== 教练生涯 ==

因为伤病困扰，张宏根不到30岁就不得不退役。[[贺龙|贺龙]]、[[邓小平|邓小平]]都为之惋惜。之后张宏根成为教练。他一度担任国家队教练，不过没有什么突出成绩。1985年他带队取得[[世界大学生运动会|世界大学生运动会]]足球季军。在1994年中国足球职业化后的首届[[中国足球甲级A组联赛|中国足球甲级A组联赛]]上，他带领[[大连实德|大连万达]]队赢得了首届联赛冠军。1999年，他在担任[[成都五牛|成都五牛]]队教练时，因诊断为[[胃癌|胃癌]]而辞职。

== 去世 ==

因受[[胃癌|胃癌]]病痛折磨，张宏根于2003年11月25日在[[北京|北京]]逝世，当时正值最后一届[[中国足球甲级A组联赛|甲A联赛]]第29轮（倒数第二轮，当时大连实德第30轮轮空）前。在第29轮比赛上，[[大连实德|大连实德]]队全体队员佩戴黑纱表示哀悼，并以胜利向张宏根致敬。<ref>{{cite news|url=http://www.dlxww.com/gb/daliandaily/2003-11/27/content_234587.htm|title=甲A为张宏根默哀|publisher=[[大连日报|大连日报]]|date=2003-11-27|accessdate=2015-10-31|deadurl=yes|archiveurl=https://web.archive.org/web/20160305001717/http://www.dlxww.com/gb/daliandaily/2003-11/27/content_234587.htm|archivedate=2016-03-05}}</ref><ref>{{cite web|url=http://sports.sina.com.cn/j/2003-11-26/1425673759.shtml|title=实录-二王闪光先后进球 实德谢幕战2-1胜健力宝|publisher=新浪体育|date=2003-11-26|accessdate=2015-10-31}}</ref>他的骨灰盒按他自己的要求，安葬于[[上海|上海]]。

== 相关 ==

1958年，[[越南|越南]]发行了一套体育[[邮票|邮票]]，张宏根带球的形象被印在上面。这是[[中国|中国]]运动员首次出现在其他国家的[[邮票|邮票]]上。后来，[[缅甸|缅甸]]发行的一套[[邮票|邮票]]上同样出现了张宏根的形象。
