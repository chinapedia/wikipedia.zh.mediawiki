'''張少懷'''{{BD|1980年|11月4日||}}，[[台灣|台灣]]男演員，畢業於[[新加坡|新加坡]][[南洋藝術學院|南洋藝術學院]]戲劇系。

== 生平 ==
學生時期曾組樂團，擅長[[吉他|吉他]]、[[薩克斯風|薩克斯風]]等樂器。熱愛戲劇表演，參與多部電影和電視劇演出。

2006年以《[[聖稜的星光|聖稜的星光]]》入圍[[第41屆金鐘獎|第41屆金鐘獎]]戲劇節目類最佳男配角獎。

2013年以《[[含苞欲墜的每一天|含苞欲墜的每一天]]》入圍[[第48屆金鐘獎|第48屆金鐘獎]]戲劇節目男配角獎。

== 家庭成員 ==
父親為綜藝節目主持人[[張菲|張菲]]。叔叔是歌手[[費玉清|費玉清]]。姑姑是[[費貞綾|費貞綾]]。妻子為演員[[翁馨儀|翁馨儀]]。<ref>{{Cite web |url=http://www.mombaby.com.tw/article/%E5%BC%B5%E5%B0%91%E6%87%B7%E3%80%81%E7%BF%81%E9%A6%A8%E5%84%80%E2%80%A7%E5%96%9C%E8%BF%8E%E6%84%9B%E5%A5%B3%E5%B0%8F%E6%AB%BB%E6%A1%83-.html |title=張少懷、翁馨儀‧喜迎愛女小櫻桃 |access-date=2015-04-30 |archive-url=https://web.archive.org/web/20151122005039/http://www.mombaby.com.tw/article/%E5%BC%B5%E5%B0%91%E6%87%B7%E3%80%81%E7%BF%81%E9%A6%A8%E5%84%80%E2%80%A7%E5%96%9C%E8%BF%8E%E6%84%9B%E5%A5%B3%E5%B0%8F%E6%AB%BB%E6%A1%83-.html |archive-date=2015-11-22 |dead-url=yes }}</ref>
