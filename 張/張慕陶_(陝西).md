'''张慕陶'''（{{bd|1902年||1941年|1月5日|Z张慕陶}}），[[陕西|陕西]][[旬邑|旬邑]]人，本名'''张金刃'''，又名'''金印'''，號'''镜英'''，[[齋號|齋號]]'''信斋'''，化名'''張慕陶'''、'''张渊明'''、'''馬雲程'''。1925年冬，成为[[中国共产党|中国共产党]]党员；但被当时的[[中共中央|中共中央]]和[[河北|河北]]省委作为“[[右倾|右倾]]机会主义”典型来批判的，甚至被定以“右派叛徒”、“奸细”、“[[托派|托派]]”等罪名，并最终被开除出党。後投靠[[閻錫山|閻錫山]]，官拜[[少將|少將]]。因長期反蔣，1941年遭到[[蔣介石|蔣介石]][[枪決|枪決]]。

直至今天，有关张慕陶的历史问题仍是疑团重重，没有得到澄清。

== 青年领袖 ==
在考入“陕西省立第三中学”后不久，22岁的张慕陶经[[魏野畴|魏野畴]]、张性初等介绍加入[[中国社会主义青年团|中国社会主义青年团]]，开始参加革命活动，宣传新文化新思想；1925年冬，转为[[中国共产党|中国共产党]]党员，并出任[[共青团|共青团]]西安地方执行委员会委员。此时，张慕陶作[[冯玉祥|冯玉祥]]部下的政治处长。

1927年[[共青团|共青团]]第四次代表大会后，张慕陶受[[共青团|共青团]]中央指示，开始筹组[[共青团|共青团]][[陕西|陕西]]省委；当年7月7日，[[共青团|共青团]][[陕西|陕西]]省委成立，张慕陶任书记。经过努力[[陕西省|陕西省]]的团员人数不断壮大，在短短半年时间里，就原来的1924人增加到3678人。由于熟识[[西北|西北]]地区的军阀，他受[[中共|中共]]组织的委托，在[[陕西|陕西]][[渭南|渭南]]、[[华阴|华阴]]、[[富平|富平]]、[[三原|三原]]、[[栒邑|栒邑]]等县发动“华北暴动”<ref>[http://news.xyl.gov.cn/content/ylrb-wz/2006-11/2/21553778563.html 刘志丹在“肃反”中] 《[[榆林|榆林]]日报》</ref>。

因工作突出，张慕陶于当年出任[[中共|中共]][[陕西|陕西]]省委常委。第二年，[[中共六大|中共六大]]在[[莫斯科|莫斯科]]举行，张慕陶作为[[陕西省|陕西省]]的唯一代表参加了这次会议，时年26岁。

== 领导顺直 ==
1928年12月，[[中共中央政治局|中共中央政治局]]决定，任命张慕陶为中共顺直省委常委、兼组织部部长。不久，由于当时的[[省委书记|省委书记]]的王藻文和省委常委李德贵的叛变，[[彭真|彭真]]等20多名中共顺直省委重要成员被捕。张慕陶临危受命，领导重组中共顺直省委；并在1929年9月，正式出任当选中共顺直省委书记<ref>《中国的噩梦》 孟伯谦著，香港自联出版社</ref>。

==第一次被开除党籍==
在中共六届四中全会后，因为和[[王明|王明]]等中央负责人的意见不同，张慕陶接受并参与到[[罗章龙|罗章龙]]等人另立“中央非常委员会”，在[[河北|河北]]进行反“四中全会”、组织“第二省委”的活动，被认为是“右倾机会主义分裂党的行动”，受到开除党籍的处分。

1932年11月，[[中共中央|中共中央]]决定恢复张慕陶的党籍，候补期一年，继续留在[[河北|河北]]工作。

==再次被开除党籍==
1933年1月，为帮助[[冯玉祥|冯玉祥]]在察哈尔省建立和发展民众抗日武装，受[[中共|中共]]北方特科的派遣，张慕陶到[[察哈尔民众抗日同盟军|察哈尔民众抗日同盟军]]，主持该军[[中共|中共]]地下党的工作，并兼任同盟军军事委员会常委、中共张家口特委书记<ref>[http://www.gmw.cn/02blqs/2006-09/07/content_497551.htm 《国共两党与察哈尔抗日》写作札记] 王晓荣著，人民出版社 2005年8月版</ref>。

在[[察哈尔民众抗日同盟军|同盟军]]失败后，张慕陶所领导的中共张家口特委被认为是“最有害、最可怕的机会主义者”，而他所主张的“联日反蒋”也受到批判，并被第二次开除出党<ref>《冯玉祥与中国共产党》 王光远 著，中央文献出版社 2004年6月版 第223页</ref>。

==继续反蒋==
在政治上失利后的张慕陶一直在[[天津|天津]]、[[北平|北平]]、[[太原|太原]]一带游走，联络地方实力派继续进行“反[[蒋介石|蒋]]抗日”活动，曾遭[[国民党|国民党]]通缉。“[[西安事变|西安事变]]”时，应[[杨虎城|杨虎城]]的邀请，张慕陶回到[[西安|西安]]。他迎合东北军和西北军中“少壮派”急于营救[[张学良|张学良]]回西安的心理，鼓动和中央军打仗，竭力主张杀[[蒋介石|蒋介石]]，并认为“国共合作是阶级投降”。

[[抗战|抗战]]爆发后，张慕陶经人介绍，在[[阎锡山|阎锡山]]处当少将参议，帮助阎进行反蒋反共工作。[[薄一波|薄一波]]等[[中共|中共]]党员到[[阎锡山|阎锡山]]的军队中进行[[统战|统战]]工作时，曾争取张慕陶回到[[中共|中共]]党内，但遭拒绝。[[王明|王明]]、[[康生|康生]]等人利用手中掌握的权力和宣传工具，给张慕陶扣上了“[[托派|托派]][[汉奸|汉奸]]”的罪名。后[[阎锡山|阎锡山]]将张慕陶送到了“国民政府军委会委员长西安行营”，被关押在“陕西省第一监狱”<ref>《陈独秀大传》 任建树 著，上海人民出版社 1999年5月版， ISBN 720802884</ref>。

1940年12月8日，[[蒋介石|蒋介石]]电令陕西省主席[[蒋鼎文|蒋鼎文]]处决张慕陶。1941年1月5日，张慕陶被枪杀于[[陕西|陕西]][[南郑|南郑]]，成为[[蒋介石|蒋]][[阎锡山|阎]]矛盾的牺牲品。
