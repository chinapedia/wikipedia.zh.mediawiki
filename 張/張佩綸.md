'''張佩綸'''（{{bd|1848年|11月24日|1903年|2月4日|catIdx=Z張}}），[[表字|字]]'''幼樵'''，[[号|号]]'''蒉斋'''，[[直隶|直隶]][[丰润县|丰润县]]齐家坨人（今[[河北|河北]][[唐山|唐山]][[丰润|丰润]]），[[晚清|晚清]]名臣。[[李鸿章|李鸿章]]女婿。堂侄[[張人駿|張人駿]]歷任[[兩廣總督|兩廣總督]]、[[兩江總督|兩江總督]]，為[[袁世凱|袁世凱]]之親家。

== 生平 ==

父親[[張印塘|張印塘]]，曾任[[安徽|安徽]][[按察使|按察使]]，但於張佩綸幼時逝世，令張佩綸兒時家景清寒。

[[同治|同治]]九年（1870年）舉人，同治十年（1871年）辛未科二甲[[進士|進士]]，選[[庶吉士|庶吉士]]，散館授[[翰林院|翰林院]][[編修|編修]]，升[[侍讲|侍讲]]，早年在京城與[[李鴻藻|李鴻藻]]、[[潘祖蔭|潘祖蔭]]、[[張之洞|張之洞]]、[[陳寶琛|陳寶琛]]、[[寶廷|寶廷]]等同為「[[清流党|清流]]」，以[[彈劾|彈劾]]大臣而聞名，例如[[李鴻章|李鴻章]]及對[[俄國|俄國]]談判示弱的[[崇厚|崇厚]]等。

1884年[[福州|福州]][[馬尾|馬尾]]對[[法國|法國]][[馬江海戰|之戰]]前夕，張佩綸自[[左副都御史|左副都御史]]被外放至[[福建軍務會辦|福建軍務會辦]]，然張不理會[[詹天佑|詹天佑]]等人的警告，對以商船示人而停泊在馬尾港達一月的[[法軍|法軍]][[戰艦|戰艦]]掉以輕心，致令[[馬尾海戰|馬尾海戰]]中[[福建水師|福建水師]]甫開戰即全軍覆沒，張佩綸亦因兵敗後逃跑而被革職，充軍[[中国东北地区|东北]]。因張佩綸早年奏章中好談軍事而領兵時棄陣潛逃，故被時人譏為[[马谡|马谡]]。

1888年，張佩綸充軍回京後卻得早年政敵[[李鴻章|李鴻章]]欣賞其才華而招為[[婿|婿]]，将长女[[李菊藕|李菊藕]]（1866年-1912年）嫁给他作[[填房|填房]]。然因[[李鴻章|李鴻章]]為避私嫌而不願保舉張氏，李鴻章逝後張佩綸更覺起復無望，55歲时以布衣之身憂鬱離世。

[[袁世凯|袁世凯]]说，“[[天下|天下]][[翰林|翰林]]真能通的，我眼裏只有三个半，[[张佩纶|张幼樵]]、[[徐世昌|徐菊人]]、[[杨士骧|杨莲府]]，算三个全人，[[张謇|张季直]]算半个。”

== 家庭 ==

張佩綸結過三次婚，有三子一女。元配朱芷薌（?-1879年）生有二子：長子張志滄（早夭），二子[[張志潛|張志潛]]。繼室邊粹玉（?-1886年），在張佩綸被流放期間病歿，無子女。與李菊藕生有一子一女，儿子[[张志沂|张志沂]]<!-- 张廷重 -->（1896——1953）即近代著名女作家[[張愛玲|張愛玲]]的父亲，女儿[[张茂渊|张茂渊]]（1898-1991）。

張佩綸和李菊藕曾合作編輯張佩綸在被流放時所著寫的幾部書，包括《[[管子注|管子注]]》二十四卷與《[[莊子古義|莊子古義]]》十卷和《[[澗于集|澗于集]]》與《[[澗于日記|澗于日記]]》。夫婦二人更合寫了一本[[食譜|食譜]]與一部[[武俠小說|武俠小說]]。

== 詩評 ==
<poem>
*檢蕢齋手劄愴然有感 其五（淸末民初•[[陳寶琛|陳寶琛]]）　　
與君同戊申，百歲已過半。
桑榆縱及收，橫流浩無岸。
豈惟倉頡泣，姬孔亦流歎。
世方無夷吾，被髪但坐看。
多君注管書，不朽出憂患。
奈何傳世業，期勗及樗散？
往者不可追，來日更誰諫？
衰病空倚樓，從今斷江雁。
</poem>
