'''張文順'''（{{bd|1938年|10月17日|2009年|2月16日|catIdx=Zhang張文順}}），中國相聲演員，中国相声21世纪复兴的奠基者之一，[[德云社|德云社]]創始人之一。2009年2月16日凌晨5点25分逝世，享年71嵗。

==生平==
北京市曲艺团第一科学员，师承[[佟大方|佟大方]]先生。复跟随[[架冬瓜|架冬瓜]]先生学习[[滑稽大鼓|滑稽大鼓]]。退休后与郭德纲搭档表演传统相声，并和[[郭德纲|郭德纲]]、[[李菁_(相声演员)|李菁]]一起发起创建了北京德云相声艺术研习社，后更名为北京[[德云社|德云社]]。德云社早期的很多相声作品都得益于他的挖掘和整理。

张文顺因患[[癌症|癌症]]，医治无效，于2009年2月16日5时25分在[[北京市中医院|北京市中医院]]去世。<ref>{{cite web |url=http://ent.sina.com.cn/h/2009-02-16/11142377181.shtml |title=张文顺今晨逝世 德云社停演一周以示纪念 |publisher=新浪网 |accessdate=2009-02-16 |work= |language =zh-cn}}</ref>

== 评价 ==
[[郭德纲|郭德纲]]：“张先生是德云社的创始人，没有他就不会有[[德云社|德云社]]。那时候如果不是张先生的帮忙，我会非常非常难……十多年来，老先生不遗余力地为德云社，他是一个最聪明的人。”<ref>{{cite web |url= http://ent.sina.com.cn/j/2009-02-16/13512377387.shtml|title=郭德纲哭别张文顺：没有他就没有德云社 |publisher=新浪网 |accessdate=2009-02-16 |work= |language =zh-cn}}</ref>

张文顺的义子[[高峰|高峰]]：“他是一个平和谦恭的人，也是一个聪明的人，没有任何的老人架子，和德云社所有人关系都不错，这就是为什么他的去世会让全社的人动容。”<ref>{{cite web |url=http://ent.sina.com.cn/j/2009-02-16/14082377408.shtml |title=义子高峰追忆张文顺：平和谦恭的老人 |publisher=2009-2-16 |accessdate= |work= |language =zh-cn}}</ref>
