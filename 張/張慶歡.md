'''張慶歡'''<ref>{{cite web |url=http://lpl.qq.com/es/player_detail.shtml?mbid=555|title=TOPMaRin|author=官方選手資料|date= |website= |publisher= |access-date= |quote=}}</ref>（{{Kor|諺=장경환|漢=張慶歡|羅=Jang Gyeong-Hwan|f=y}}，{{bd|1991年|2月12日}}，绰号：'''MaRin'''），韩国《[[英雄联盟|英雄联盟]]》职业选手，原为韩国[[SK_Telecom_T1|SK Telecom T1]]战队上单选手，后于2015年12月10日加入中国大陆[[LGD_Gaming|LGD Gaming]]战队并担任上单位置，後回到韓國成為[[Afreeca_Freecs|Afreeca Freecs]]上單選手。2018年轉為中國[[Topsports_Gaming|Topsports Gaming]]战队上单选手。<ref>{{cite web |url=https://weibo.com/5449734852/H1HjncZGD|title=TOP.MaRin断开连接|author=Topsports电子竞技俱乐部微博|date=2018年11月8日 |website= |publisher= |access-date= |quote=}}</ref> 2019年1月19日於直播中宣布退役。

==早年经历==

張慶歡于1991年2月12日出生于[[韩国|韩国]][[忠清南道|忠清南道]][[天安市|天安市]]，在服兵役期间，張慶歡在他的前辈们的推荐下开始接触《[[英雄联盟|英雄联盟]]》，此外，他还接触了《[[魔兽争霸3|魔兽争霸3]]》，在此之后他还接触了《[[Chaos|Chaos]]》。

2013年，金靖均组建了[[SK_Telecom_T1|SK Telecom T1]]战队，但在该战队组建时，由于缺乏上单选手，使得金靖均建议張慶歡练习上单英雄，于是他开始练习上单英雄。在大约1个月后，他加入了SKT T1 #2，开始职业生涯。<ref>{{Cite web |url = http://games.qq.com/a/20151211/000223.htm#p=1 |title = MaRin自述离开SKT原因：Faker过于耀眼 |date = 2015年12月11日 |publisher = 腾讯网 |language = zh-cn }}</ref>
