'''張榮財'''（{{bd|1958年|catIdx=C張}}），[[台灣|台灣]][[臺南市|臺南市]][[關廟區|關廟區]]東勢里人，從事[[磚頭|磚頭]]搬運工作，因在1985年駕駛拼裝車不慎撞上[[陳水扁|陳水扁]]的妻子[[吳淑珍|吳淑珍]]，導致吳終生下半身[[癱瘓|癱瘓]]，從此被捲入政治鬥爭之中。這場車禍因被宣傳為「政治謀殺」，張榮財遂長期遭到[[民進黨|民進黨]]支持者指責為「[[中國國民黨|國民黨]]的劊子手」。此車禍案件亦曾與[[林宅血案|林宅血案]]、[[江南案|江南案]]相提並論。

== 吳淑珍車禍 ==
1985年11月18日，陳水扁參選臺南縣長敗選，由親友五十餘人陪同，到關廟鄉謝票。下午1時許，他們搭車到關廟鄉中正路關廟鄉公所前，將汽車停放路旁，大家下車準備徒步到中正路368號對面巷內「齒留香餐廳」用餐。另一方面，張榮財替盛建磚廠載3000塊磚到[[太保市|龍潭]]給買主謝榮文後回到關廟，至[[北勢村|北勢村]]益山修理行修理拼裝車大燈開關後駕車準備回家。

事故當時，陳水扁和太太吳淑珍及部分友人已轉入巷內，其他人仍在中正路上。而張榮財駛到中正路關廟鄉農會前時發現[[煞車|煞車]]失靈，他聲稱當時不知所措，只顧閃躲路上人車，到了中正路368號前時，發現前面路上站滿行人，他因閃躲不及，乃將方向盤向左打，準備將車轉向人較少的巷內，不料便撞上正步行到餐廳用餐的吳淑珍及同行友人余慧珠。

== 調查結果 ==
當時台南地檢處檢察官[[呂光華|呂光華]]調查結果，基於下述理由認為張榮財非蓄意要謀殺吳淑珍：

#張榮財係單純[[鄉民|鄉民]]，所受教育不多，平日為生計忙碌，未加入任何政治團體或民間社團，亦未與地方政治人物往來，更未結交任何不法分子，與陳水扁夫婦素不相識，無任何恩怨可言，亦無法預知陳妻等臨時會步入該巷內，應無蓄意殺害陳妻之可能。
#該車禍發生時，眾目睽睽，光天化日，被告果欲殺人，儘可採隱密手段，擇無人處所為之，殊無出此下策，開拼裝車於鬧市下手之理。
#當時五輪拼裝車都是利用真空煞車輔助器<ref>煞車真空倍力器（Brake Vacuum Booster）</ref>來煞車，如真空吸力不足，就會有煞車失靈現象發生。檢察官認為張榮財於肇事時雖曾踩煞車，但緊張之下，真空吸力不足，未能打開風門，致未煞住。如被告蓄意殺人，事先即可更動機件，故意佈置成煞車失靈狀態，豈非更能掩人耳目。故經查並無任何確切證據可資證明張榮財蓄意謀殺吳淑珍。

此外根據陳水扁樁腳同時也是車禍目擊者林昭泰的說法，案發後他有過一段明查暗訪，也證實張榮財只是個為三餐奔忙的磚頭載運工。<ref>【2004/09/13 聯合報】一個鐵牛車司機無言的意外人生</ref>

此案由陳水扁第二次出庭時主動撤銷告訴<ref>TVBS獨家揭祕 當年撞殘吳淑珍 張榮財折磨20年 -溫紳</ref>，故張榮財未經由司法判決證明其清白，或定讞其罪刑。

== 張榮財家人的回應 ==
雖說司法調查結果認為此一事故純屬意外，但張榮財因為對造成吳淑珍殘疾的良心愧疚，儘管每每配合參加[[民進黨|民進黨]]的選舉造勢時屢遭眾人的言語羞辱，張榮財都無言承受，每次的回答都是：“我對不起大家、我不該去害阿扁與阿珍。”直到有一次，張榮財在臺上忍不住哭了，台下眾人紛紛鼓譟：“他終於真正懺悔了，劊子手也有這樣的一天。”

據時報周刊記者於2006年的採訪記錄，在張榮財的故鄉台南縣[[關廟區|關廟鄉]]的東勢村，一些村民們為了保護張榮財往往謊稱張榮財已死於意外，而張榮財家人對於陌生人的來訪亦非常排斥，甚至回應：「不要再叫阿榮去演戲了，二十一年了，能不能給他一個平靜的生活？」<ref>〈阿扁抹黑張榮財21年〉，時報周刊vol.1482</ref>

== 吳淑珍家人的回應 ==
對於這起車禍，[[吳淑珍|吳淑珍]]的家人至今仍認為是一場「政治車禍」。其夫婿[[陳水扁|陳水扁]]擔任[[總統|總統]]時，在2006年6月20日公開宣稱：「還害我太太在我選[[臺南縣|臺南縣]]縣長謝票的期間，遭受'''政治車禍'''，已經坐輪椅二十一年了。」其女兒[[陳幸妤|陳幸妤]]也是持相同看法，在2008年10月9日於[[自由時報|自由時報]]投書提道：「作為一個政治人物的兒女是可悲，尤其是[[泛绿|綠營]]的政治人物，我九歲時我媽媽'''政治車禍'''才出院半年，我爸爸就被抓去關了。」
