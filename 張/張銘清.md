'''张铭清'''（{{bd|1946年||||catIdx=Z张}}）祖籍[[山東|山東]]，[[中华人民共和国|中华人民共和国]]官员、记者、学者。

==生平==
张铭清早年长期生活于[[福建|福建]]，在[[三明市|三明]]的高中毕业<ref>{{Cite web |url =http://www.cctv.com/health/special/fangtan/zhangmingqing/index.shtml   |title = 对话张铭清 |author =  |date =  |publisher =央视网  |language =  |accessdate =  2017-10-11|quote =  }}</ref>。1969年自[[福建农林大学|福建农学院]]土壤化学专业毕业<ref>{{Cite web |url = http://news.fznews.com.cn/kjww/2010-6-7/201067WRD1QFV8NN224215.shtml |title = 福建农大杰出校友张铭清 7日被母校聘为客座教授 |author =  |date = 2010-06-07 |publisher =福州新闻网  |language =  |accessdate =  |quote =  }}</ref>。1978年到1981年，在[[中国社会科学院研究生院|中国社会科学院研究生院]]新闻系，是该院首届硕士研究生。1981年到1993年，历任《[[人民日报|人民日报]]》福建记者站记者、首席记者、站长，《人民日报》记者部副主任、高级记者。1993年到2006年，任[[中共中央台湾工作办公室|中共中央台湾工作办公室]]、[[国务院台湾事务办公室|国务院台湾事务办公室]]主任助理、新闻局局长、新闻发言人、[[中华全国新闻工作者协会|中华全国新闻工作者协会]]常务理事<ref name=jl>{{Cite web |url = http://news.sina.com.cn/o/2008-10-22/072814611062s.shtml |title = 海协会副会长张铭清简历 |author =  |date = 2008-10-22 |publisher = 新浪|language =  |accessdate =  |quote =  }}</ref>。2006年起，任[[海峡两岸关系协会|海峡两岸关系协会]]副会长<ref>{{Cite web |url =  http://www.huaxia.com/lasd/lbyl/2003/06/509080.html?id=19190/3b9a9  |title =  海峡两岸关系协会|author =  |date =  2003-06-04|publisher =华夏经纬网  |language =  |accessdate =  |quote =  }}</ref>。2007年4月，受聘为新成立的[[厦门大学|厦门大学]]新闻传播学院院长、教授、[[博士生导师|博士生导师]]，研究领域是对台和对外宣传<ref name=jl/><ref>{{Cite web |url = http://news.sohu.com/20070414/n249428263.shtml |title = 张铭清正式受聘厦大新闻传播学院院长 |author =  |date =  2007-04-14|publisher =搜狐  |language =  |accessdate =  |quote =  }}</ref>。2008年6月，任[[海峡两岸关系协会|海峡两岸关系协会]]第二届理事会副会长<ref>{{Cite web |url =http://www.chinanews.com/tw/dlyw/news/2008/06-03/1270906.shtml  |title = 新闻资料：海峡两岸关系协会 |author =  |date =2008-06-03  |publisher = 中国新闻网 |language =  |accessdate =  |quote =  }}</ref>。他还曾任[[中国人民解放军国防大学|中国人民解放军国防大学]]兼职教授、[[北京联合大学|北京联合大学]]兼职教授、顾问<ref name=jl/>。

2000年代，張銘清被[[台灣|台灣]][[綜藝節目|綜藝節目]]《[[全民最大黨|全民最大黨]]》的[[邰智源|邰智源]]多次搞笑模仿（劇中化名為張名清），因此在台灣具有相當高的知名度。邰智源到[[北京|北京]]来时，还曾和張銘清见面<ref>[http://www.southcn.com/weekend/culture/200604060046.htm 电视］《全民大闷锅》：“解闷救台湾”] {{webarchive|url=https://web.archive.org/web/20080223090120/http://www.southcn.com/weekend/culture/200604060046.htm |date=2008-02-23 }}</ref><ref>{{Cite web |url =  http://news.ifeng.com/taiwan/zhangmingqing/others/200810/1021_4861_840485.shtml|title = 张铭清在厦大遇台生“追星” |author =  |date = 2008-10-21 |publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref>。2008年10月，張銘清到訪[[陳水扁|陳水扁]]的故鄉[[臺南縣|臺南縣]][[官田區|官田鄉]]，10月20日出席[[國立臺南藝術大學|國立臺南藝術大學]][[學術研討會|學術研討會]]，交流《兩岸傳播暨影像藝術》，10月21日參觀[[台南市孔廟|台南市孔廟]]時，遭[[民進黨|民進黨]][[市議員|市議員]][[王定宇|王定宇]]率領亲绿群眾包圍，[[张铭清访台遇袭事件|遇袭]]摔倒在地。事件经媒体报道，震惊[[海峡两岸|海峡两岸]]<ref>{{Cite web |url = http://tw.people.com.cn/BIG5/14810/8207124.html  |title =  張銘清在台南遭綠營支持者暴力攻擊|author =  |date = 2008-10-21 |publisher =人民网  |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web|url =http://www.takungpao.com/news/08/10/22/LT1-978730.htm|title =大公報社論：要求嚴懲暴力分子|author =|date =2008-10-22|publisher =大公网|language =|accessdate =|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20081025051253/http://www.takungpao.com/news/08/10/22/LT1-978730.htm|archivedate =2008-10-25}}</ref>。
