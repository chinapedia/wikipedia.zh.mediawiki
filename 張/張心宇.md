'''張心宇'''（{{bd|1978年|8月9日||}}），[[台灣|台灣]][[新聞主播|新聞主播]]。其妻為[[東森財經新聞台|東森財經新聞台]]主播[[廖廷娟|廖廷娟]]<ref>{{cite web |title=「帥哥主播」張心宇走過婚變風波再婚　娶甜美廖廷娟 |url=https://star.ettoday.net/news/200822 |website=ETtoday新聞雲 |accessdate=2018-09-22}}</ref><ref>{{cite web |author1=阮馨儀 |title=主播夫妻張心宇、廖廷娟杜拜遇險　飯店大火整棟燒光！ |url=https://www.nownews.com/news/20160101/1944382/ |website=今日新闻 |accessdate=2018-09-22}}</ref>。現任[[壹電視新聞台|壹電視新聞台]]《1600整點新聞》主播、[[壹電視新聞台|壹電視新聞台]]體育主播，其兄是補習班數學老師[[張心豪|張心豪]]
張心豪-補教界王牌教師，目前在板橋教書。
