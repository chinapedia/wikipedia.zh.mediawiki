《'''張雨生大兵日記'''》是台灣漫畫家[[曾正忠|曾正忠]]的漫畫作品。<ref>[http://ent.msn.com.tw/news/content.aspx?sn=0901011088 張雨生死忠歌迷超有心 主辦『張雨生20週年紀念展』上千件絕版品展出], 2007/11/9</ref><ref>[http://catalog.digitalarchives.tw/item/00/66/d4/f0.html 張雨生大兵日記], 數位典藏與數位學習國家型科技計畫 Taiwan e-Learning & Digital Archives Program</ref>

== 簡介 ==
《張雨生大兵日記》為曾正忠以1989年台灣歌手[[張雨生|張雨生]]入伍從軍的新聞作為題材所發表的[[四格漫畫|四格漫畫]]作品，內容以張雨生在營區和各級長官、同袍所發生的各種趣味事件為主。起初於漫畫雜誌《[[星期漫畫|星期漫畫]]》上發表，後續一共發行了三集[[單行本|單行本]]。

== 登場人物 ==
; [[張雨生|張雨生]]
: 以實際存在歌手為範本的角色，本作品主角。身材不高，但聲音高亢。常在軍營裡發生各種烏龍狀況而遭殃。
: 因身為歌手而擅長歌唱，也有能用嘴吞槍的專長。<ref>張雨生大兵日記I：41頁</ref>
; 芭樂班長
: 張雨生的班長，為作品第一男配角。
: 帶兵嚴謹的鐵漢，但有喜歡看《星期漫畫》（連載本作品的雜誌）的嗜好，<ref>張雨生大兵日記I：29頁</ref>喜歡軍中福利社的女店員阿蘭。<ref>張雨生大兵日記I：38頁</ref>
; 鐵齒班長
: 外貌「大門牙」的班長，曾創下在軍隊[[除夕|除夕]]晚會用門牙連開十二瓶[[黑松汽水|黑松汽水]]的紀錄。<ref>張雨生大兵日記I：7頁</ref>偶有在睡夢中大喊全體成員『起床！』的毛病。<ref>張雨生大兵日記I：65頁</ref>
; 老鼠班長
: 有喜歡躲在廁所裡偷聽別人講[[八卦新聞|八卦新聞]]的習慣，<ref>張雨生大兵日記I：15頁</ref>是軍營裡[[政戰官|政戰官]]獲取各種情報的來源。<ref>張雨生大兵日記II：28頁</ref>
; 連長
: 外號『嚴厲的老爸』的[[上尉|上尉]]，<ref name="volI-67">張雨生大兵日記I：67頁</ref>偶有跑錯連隊的情況。<ref>張雨生大兵日記I：23頁</ref>
; 政戰官
: 外號『慈祥的大媽』，喜愛念佛經。<ref name="volI-67"/>喜愛將政戰採用『藝術化』的表現。<ref name="volI-62">張雨生大兵日記I：62頁</ref>
; 營長
: 營區內最高階級長官，張雨生的歌迷。<ref>張雨生大兵日記I：10頁</ref>
; 007
: 張雨生的同袍之一，有[[口吃|口吃]]、反應慢的毛病，喜愛從事[[乩童|乩童]]的行為，<ref>張雨生大兵日記I：69頁</ref>崇拜的團體為[[義和團|義和團]]。<ref>張雨生大兵日記I：71頁</ref>
; 高伙房
: 一名記憶力差而被調到廚房的老兵，<ref>張雨生大兵日記II：45頁</ref>常感慨自己遭遇而欲尋求知音，和軍犬TATAMI是奇妙的組合。<ref>張雨生大兵日記II：51頁</ref>
; TATAMI
: 體型巨大的軍犬。外貌看似性格兇猛，但常有被其它小動物嚇跑的情況。<ref>張雨生大兵日記II：32頁</ref>
: 名字和外型貌似曾正忠的另一部漫畫作品《無膽狗熊TATAMI》裡頭的狗角色『TATAMI』。
; 阿蘭
: 軍中福利社之花，外貌為[[林青霞|林青霞]]模樣的臉蛋、[[鄒美儀|鄒美儀]]模樣身材的女子，<ref>張雨生大兵日記I：6頁</ref>芭樂班長心儀的對象。
; 阿三
: 號稱「波斯灣之虎」，[[阿拉伯世界|阿拉伯世界]]的石油大王，<ref>張雨生大兵日記III：86頁</ref>曾[[綁架|綁架]]張雨生。<ref>張雨生大兵日記III：85頁</ref>
: 以當時[[伊拉克總統|伊拉克總統]][[萨达姆·侯赛因|海珊]]為範本的角色。
