《'''警徽天职3'''》（[[英语|英文]]：{{lang|en|'''C.L.I.F. 3'''}}），[[新加坡|新加坡]][[新傳媒私人有限公司|新傳媒私人有限公司]]警匪电视剧，由[[李南星|李南星]]、[[瑞恩|瑞恩]]、[[戚玉武|戚玉武]]、[[白薇秀|白薇秀]]、[[黃俊雄_(藝人)|黃俊雄]]、[[李美玲|李美玲]]及[[蘇智誠|蘇智誠]]領銜主演，監製為[[張龍敏|張龍敏]]，製作人爲[[郭令送|郭令送]]。此劇為《[[警徽天職2|警徽天職2]]》續集，由[[新加坡警察部队|新加坡警察部队]]與豐城Gain City、[[昇菘集團|昇菘集團]]及Viz Branz所贊助。<ref>http://www.hometeam.sg/article.aspx?news_sid=20131114I2F6nOahpAPC</ref>

== 故事大纲 ==
当警方破门而入时，两名泰籍匪徒差点成功靠伪装脱身。误打误撞之下，交通警察张贵祥（[[黃俊雄_(藝人)|黄俊雄]]飾）揭穿了他们的身份。来自警察情报组的唐耀佳（[[戚玉武|戚玉武]]飾）和匪徒正面冲突，经过一轮激战后被匪徒逃走了。另一方面，廖心怡（[[白薇秀|白薇秀]]飾）完成了海外的维和任务并回到新加坡。尽管她百般抗拒，耀佳坚持要重续前缘。但耀佳不知道，心怡在一场炸弹事件后患上了创伤后压力症候群。 

魏蓝天（[[李南星|李南星]]飾）是CID特别罪案调查组的组长，與黄芷婕（[[瑞恩|瑞恩]]飾）結爲夫妻。他在一次的罪案调查中碰上了旧友杨忠正（[[曹国辉|曹国辉]]飾）。忠正特地到新加坡来挽回与女儿的关系，并阻止她成为应召女郎。 

蓝天非常高兴能重遇好友，但他并不知道原来忠正是一个国际犯罪集团的头目。当蓝天调查的案子可能会揭发他真正的身份时，忠正设下陷阱陷害蓝天。最后，两名昔日好友被迫正面对决。这些警务人员是否能够解决个人生活和警方任务之间的难题呢？

== 軼事 ==
*此劇是[[李南星|李南星]]、[[方展發|方展發]]繼《X元素》、《[[警徽天職2|警徽天職2]]》、《[[信約：唐山到南洋|信約：唐山到南洋]]》后，連續第四度合作的電視劇。
*飾演銀幕情侶的[[戚玉武|戚玉武]]、[[白薇秀|白薇秀]]已連續登場三季《警徽天職》系列。
*[[曹国辉|曹国辉]]已連續登場三部同時段電視劇，其為《[[砂煲肉骨茶|砂煲肉骨茶]]》、《[[Missy先生|Missy先生]]》。
*此劇是[[黃俊雄_(藝人)|黄俊雄]]、[[瑞恩|瑞恩]]首度以非銀幕情侶身份合作。
*此劇是[[李南星|李南星]]、[[黃俊雄_(藝人)|黄俊雄]]繼《雙子星》、《[[信約：唐山到南洋|信約：唐山到南洋]]》后，三度合作。
*此劇是[[李美玲|李美玲]]首度與[[李南星|李南星]]、[[蘇智誠|蘇智誠]]合作。
*此劇是[[瑞恩|瑞恩]]、[[李美玲|李美玲]]繼《[[微笑正義|微笑正義]]》后，再度合作。
*此劇是[[黃俊雄_(藝人)|黄俊雄]]、[[馬藝瑄|馬藝瑄]]首度飾演銀幕情侶。

== 與史實不合之處 ==
*此剧出现时间不合之处，详情如下：
**韩俊希在第2集里撞死陈彦均的父母陈国邦和王美芳，车祸在交通警察通告被指发生在2014年1月10日，不在从闭路电视来看，2014年4月3日蓬萨通回组屋单位和2014年4月12日王美芳的葬礼范围之间。
**第25集里，杨忠正被开枪重伤事故发生后的结尾里，警察部队升值典礼在2014年5月13日举行。但是，第1集一开始杨忠正被中枪，之后就作了一年前的冒充警察案（从闭路电视来看，2014年4月3日蓬萨通回组屋单位），此剧故事足足不到一年。
*演员表里，[[方展发|方展发]]是以“陈彦均”的角色鸣谢，但在第5集里，却是相反的——“陈彦均”是以[[方展发|方展发]]的角色鸣谢。
*此剧一共讲了几次“王八蛋”，详情如下：
**第6集里，翁选美乘搭在德士上碰见路霸，被骂是个“死三八”、“王八蛋”。后者的不雅用语通常是用在男人身上，因此不合理，但中文字幕没有显示，还是与实际不合。
**蜘蛛明在咖啡店遇见“王八蛋”，想打他却被张贵祥及时阻止，中文字幕在此剧播出时被改成“混蛋”。这不符合实际，因为在《[[信约：动荡的年代|信约：动荡的年代]]》里，“王八蛋”改成“混蛋”没有配音，“走狗”则被改成“傀儡”，但很显然的被附上配音。
