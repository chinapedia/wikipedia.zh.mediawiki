'''警視長'''是[[日本警察|日本警察官]]的階級之一，位居[[警視正|警視正]]之上、[[警視監|警視監]]之下，相當於[[自衛隊|自衛隊]]階級「將補」（[[少將|少將]]）<ref>{{Cite web |url=http://www.mod.go.jp/pco/chiba/kaikyu.html |script-title=ja:自衛官の階級章、防衛記念章、精勤章、職種き章、各種き章等について紹介します |author={{lang|ja|自衛隊千葉地方協力本部}} |accessdate=2015-08-02 |archiveurl=https://archive.is/20170525163014/http://www.mod.go.jp/pco/chiba/kaikyu.html |archivedate=2017-05-25 |language=ja |deadurl=yes }}</ref>。

屬這階級的役職包括[[警察廳|警察廳]]本廳課長、參事官、[[警察大學_(日本)|警察大學校]]部長、管區警察局部長；[[警視廳|警視廳]]生活安全、地域、交通、警備、暴力團對策的各部長、主要參事官；道府指定縣[[警察本部|警察本部]]的部長、主要參事官；中小規模[[警察本部|警察本部]]的本部長、北海道警察本部的方面本部長。

只有少數[[非國家公務員|非國家公務員]]（{{lang|ja|ノンキャリア}}）能昇任到此階級。[[國家公務員|國家公務員]]組（{{lang|ja|キャリア}}）大概於40歲後半昇任到此階級。還有推薦組的全体人員可以昇進警視長，大規模的警察本部的部長有時候會在退休時昇任警視長。

== 負責的主要職位 ==
* 警察廳 －　各課課長、室主任
* 警察廳地方機關（管區警察局）　-　總務監察·廣域調整部長·學校校長·首席監察官（只有關東管區）
* 警察大學　－ 各教養部長·財務捜査研修所所長
* [[科學警察研究所|科學警察研究所]] － 總務部長·研究調整官
* 警視廳　－　生活安全·地域·交通·組織犯罪對策部長·学校校長·主要參事官
* 警視廳方面本部長
* 警察本部長
* 北海道警察　－ 警務、總務部長、方面本部長
* 宮城縣警察　－ 刑事部部長
* 府、指定縣警察本部　－ 警務、總務、刑事、生活安全、地域、交通、警備部長
* 中小規模的縣警察本部　－ 本部長
