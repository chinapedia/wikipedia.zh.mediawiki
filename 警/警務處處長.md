'''警務處處長'''（[[英文|英文]]：'''C'''ommissioner of '''P'''olice，[[縮寫|縮寫]]：'''CP'''） 俗稱「'''一哥'''」，是[[香港警察職級|香港警察職級]]中最高級[[香港警務處憲委級職級|憲委級]]職級；位於[[警務處副處長|警務處副處長]]之上，為[[香港警務處|香港警務處]]首長。於[[香港主權移交|香港主權移交]]後，警務處處長為[[香港政府主要官員|香港政府主要官員]]，須由[[香港特別行政區行政長官|行政長官]]提名，[[中華人民共和國國務院|中華人民共和國國務院]]任命，在《[[香港政府憲報|香港政府憲報]]》中公佈。

除非有特別原因令政府特別批准，否則'''警務處處長'''的退休年齡和一般警察一樣都是57歲。因此任期和[[香港政府主要官員|香港政府主要官員]]不一樣。

== 徽章衣飾 ==
警務處處長的[[襯衣|襯衣]]為[[白色|白色]]，[[肩章|肩章]]以[[英国陆军|英国陆军]][[上将|上将]]军衔标志为蓝本，為一枚雙杖嘉禾花、一枚市花及嘉禾及一枚軍星，[[領章|領章]]為葉片紋，警帽有雙條銀色葉片邊飾，[[權杖|權杖]]一枝。

== 歷史 ==
香港警察職級乃參考自[[英國|英國]]警銜，英國警銜則參考自英國[[軍銜|軍銜]]，一般分為[[將軍|將軍]]、[[校尉|校尉]]、[[士官|士官]]以及[[兵|兵]]的職級。警務處處長為[[上將|上將]]職級。

== 歷任首長 ==
因為早年[[香港政府|香港政府]]負責翻譯的人選每次皆不同，警察首長之中文官方譯名亦時有不同。政府檔案中曾經出現的中文譯名包括'''總緝捕官'''、'''總差役'''和'''香港差頭'''等等；時至1909年，官方名稱首次定為'''香港巡警道'''，至1917年改稱為'''警察司'''，至1930年再改輪為'''警察總監'''，於1933年一度改為'''公安局長'''。直至1935年，正式定為警務處處長，並且一直沿用至今。<ref>[http://www.police.gov.hk/offbeat/779/chi/f05.htm 警隊的名稱與職級] 《警聲》第779期</ref>

== 俗稱由來 ==
於[[世界大戰|世界大戰]]前，[[香港人口|香港人口]]比較少，屬於[[殖民地|殖民地]]管治模式，英國派駐香港的行政長官為[[香港總督|總督]]，負責行使英國所制定的[[政策|政策]]以及發出行政[[命令|命令]]。當時的香港政府的結構非常精簡。對下擁有最大[[權力|權力]]的第2號人物，即為治安首長（即當時的警務處處長），管理範圍包括領導[[警察|警察]]、[[香港出入境事務|出入境事務]]、[[海關|海關]]、[[消防|消防]]和[[監獄|監獄]]管理及[[囚犯|囚犯]]更生等等，並且執行所有一切與[[香港治安|香港治安]]相關的行政及命令。

=== 一號車牌 ===
首批香港官員的車輛車牌號碼可以參考到各人的重要性排列位置。在香港，於1907年共有6輛[[汽車|汽車]][[登記|登記]]。當時，[[香港車輛註冊號牌|香港車輛註冊號牌]]由警察司（即現今的警務處處長）負責登記。

#皇冠徽章──[[香港總督|香港總督]]；首位在香港使用汽車的總督為[[梅含理|梅含理]]，於1912年起以汽車代步。被委任為香港總督前，梅含理於1893至1901年間出任警察司，兼任[[消防隊監督|消防隊監督]]（即現今的[[消防處處長|消防處處長]]）。
#1號車牌──[[治安首長|治安首長]]（架構改革後，正名為警務處處長，及後再設回保安司（即現今的保安局局長），以管理所有[[香港紀律部隊|香港紀律部隊]]，惟1號車牌續由警務處處長擁有。）
#2號車牌──[[財政司|財政司]]（即現今的[[財政司司長|財政司司長]]），財政司改用FS車牌後，2號車牌被[[王明維|王明維]]於1993年2月27日以950萬港元投得。
#3號車牌──[[周錫年|周錫年]]爵士，華人代表，於1983年9月地產商人[[鄭公時|鄭公時]]以103萬港元投得，他認為“3”有“三三不盡”之意。
#4號車牌──[[醫務衛生署署長|醫務衛生署署長]]（即現今的[[衛生署署長|衛生署署長]]），後來被[[趙世曾|趙世曾]]于1978年以147,000港元投得。
#5號車牌──由一位梁姓女士代表一位不願透露姓名的商人於1993年以250萬港元投得，後來為[[劉鑾雄|劉鑾雄]]所有。
#6號車牌──為[[邵逸夫|邵逸夫]]（六叔）擁有，于1978年以330,000港元投得。
#7號車牌──原為[[香港輔助警察隊|香港輔助警察隊]]總監[[曹峻安|曹峻安]]擁有，他去世後，運輸署將車牌號碼收回拍賣，[[香植球|香植球]]於1989年1月以480萬港元投得。
#8號車牌──[[羅定邦|羅定邦]]於1988年2月以500萬港元投得。
#9號車牌──原為[[鄧肇堅|鄧肇堅]]擁有，後來被杨受成於1994年3月19日以1,300万港元投得，該价格是當時历来香港最高的投標纪录，亦一度為全球最昂贵。
#10號車牌──[[工務司|工務司]]（即現今的[[發展局局長|發展局局長]]），後來為[[容永道|容永道]]擁有。<ref>http://xing.glyx.cn/machine/2568.shtml</ref>

除1號車牌外，各司署的無字頭車牌均已經售予私人擁有，就應否出售1號車牌的問題，曾經一度引起公眾議論，時任警務處處長[[李君夏|李君夏]]表示，出售1號車牌有違[[香港警察傳統|警隊傳統]]，並且打擊警隊[[士氣|士氣]]，他強烈抗拒出售之。後期的警務處處長[[李明逵|李明逵]]亦表示，1號車牌不是給警務處處長，是[[香港市民|香港市民]]認為維持[[香港治安|治安]]的警隊最為重要，是給警隊的。他認為如果警隊表現不好，有20個1號車牌亦沒有用處。故此他贊成保留，認為可以作為對警察的一種鞭策。<ref>[http://stu.hksyu.edu/~ourvoice/release2007/200706/release200706_07a.htm 一哥的1號車牌] 《仁聞報》電子版 2007年6月</ref>
