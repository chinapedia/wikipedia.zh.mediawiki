'''警部'''為[[日本警察|日本警察官]]之[[階級|階級]]之一，位於[[警視|警視]]之下、[[警部補|警部補]]之上，相當於[[自衛隊|自衛隊]]的三佐（[[少校|少校]]）至一尉（[[上尉|上尉]]）<ref>{{Cite web |url=http://www.mod.go.jp/pco/chiba/kaikyu.html |script-title=ja:自衛官の階級章、防衛記念章、精勤章、職種き章、各種き章等について紹介します |author={{lang|ja|自衛隊千葉地方協力本部}} |accessdate=2015-08-02 |archiveurl=https://archive.is/20170525163014/http://www.mod.go.jp/pco/chiba/kaikyu.html |archivedate=2017-05-25 |language=ja |deadurl=yes }}</ref>。其職位包括[[警察本部|警察本部]]之系長至副課長、[[警察署|警察署]]之課長、[[派出所|派出所]]所長等。

警察組織之階級金字塔中結構中、警部以上之人員為數相當少。階級為警部以上者、享有[[逮捕狀|逮捕狀]]申請權。普通警員中，有能力的現場警員一般於30歲後半～40歲升任為警部。

任職警部有兩種管道：普通組和特考組，普通組是[[警部補|警部補]]任滿4年後，先3個月研修、8個月實習，再1個月研修，可升警部。另一種是國家特考組（[[精英官僚|精英官僚]]），也就是國家[[公務員|公務員]]（[[國家公務員|國家公務員]]I種合格者）在前線警察署分局實習9個月，之後自動昇為警部。理論上，最年輕23歲即可担任警部。

== 相等職級 ==
*香港[[督察級|督察級]]（日本警部英譯「Chief Inspector」，同香港的[[總督察|總督察]]）
*[[中華民國警察|臺灣警察]]初階警官的'''兩線二星'''（第七、八序列職務；職銜：中隊長、警備隊長、所長、警務員）、中階警官的'''兩線三星'''（第六序列職務；職銜：組長、偵查隊長、主任）。
*[[美國警察|美國警察]]，因為警察為各州治安事務，各州警察階級或有不同，以[[加州|加州]][[洛杉磯警察局|洛杉磯警察局]]或[[紐約州|紐約州]][[紐約市警察局|紐約市警察局]]為範例，相對應之警察階級者為 Lieutenant。
