'''《警部補 矢部謙三》'''是在[[朝日電視台|朝日電視台]]周五深夜劇場（每週五23:15 - 24:15日本標準時間）播出的連續電視劇，由[[生瀨勝久|生瀨勝久]]主演。

第一季於2010年4月9日至[[5月14日|5月14日]]播出。第二季於[[2013年|2013年]][[7月5日|7月5日]]至[[8月30日|8月30日]]播出。

[[2017年|2017年]][[3月7日|3月7日]]以《'''警部補 矢部謙三〜人工頭腦VS人工頭毛〜'''》為標題、於「{{lang|ja|ビデオパス}}」與「{{lang|ja|テレ朝動画}}」上傳<ref>{{Cite news |title=生瀬勝久、『刑事・矢部謙三』4年ぶり完全新作ドラマ配信スタート |newspaper=[[ORICON_NEWS|ORICON NEWS]] |date=2017-01-26 |url=http://www.oricon.co.jp/news/2085036/full/ |accessdate=2017-02-01}}</ref>。

== 概要 ==
本劇為2000年開始的[[圈套_(電視劇)|圈套]]系列的衍生作品，以圈套裡出現的刑警-矢部謙三為主角，解開許多困難案件的故事。本劇為生瀨勝久的初主演。

==角色介紹==
===主要演員===
★代表曾經在圈套系列出場的人物。

; <span id="矢部謙三">矢部謙三</span> - [[生瀨勝久|生瀨勝久]]★（中學時代：[[吉井克斗|吉井克斗]] / 小學時代：[[谷端奏人|谷端奏人]]）

警視廳公安部刑警。官位警部補。對自己戴著假髮有自卑感，對假髮相關字眼相當敏感，對除了自己的助手外的人隱瞞此事。沒有什麼權威，會對上司拍馬屁，喜歡研究小短劇，說著關西方言。

; <span id="秋葉原人">秋葉原人</span> - [[池田鐵洋|池田鐵洋]]★

矢部的第三任助手，崇拜矢部的電腦宅男。

;第一季
; <span id="桂美晴">桂美晴</span> - [[貫地谷栞|貫地谷栞]]

警視廳公安部庶務員。隨身攜帶曾經是傳說的刑警父親.桂太郎所留下的筆記本「刑警百科」。對於現場搜查很憧憬，會利用機會任性前往現場搜查。對不講理的矢部感到反感。有不時轉動脖子發出喀喀聲的癖好。
矢部稱呼她為「小姑娘」。

; <span id="桜木健一郎">桜木健一郎</span> - [[鈴木浩介|鈴木浩介]]
警視廳公安部庶務員。個性安逸，有時會去支援搜查活動，有不錯的推理能力。思考時有把手指放在[[人中|人中]](鼻下方的溝)的癖好。矢部稱呼他為「刑事君」。

; <span id="菊池愛介">菊池愛介</span> - [[姜暢雄|姜暢雄]]★

警視廳參事官。東大跳級畢業後曾經是矢部的助手，不過現在已升官，經常炫耀自己的高學歷。

; <span id="片桐優奈">片桐優奈</span> - [[原幹惠|原幹惠]]
警視廳公安部的臥底搜查員。説話有濃重鄉音。[[昭和39年|昭和39年]]生，是矢部的同期。
