'''警視廳'''（{{jpn|j=警視庁|hg=けいしちょう|rm=Keishichō|f=y}}；英語譯名：{{lang|en|Metropolitan Police Department}}，簡寫{{lang|en|'''MPD'''}}）是[[日本|日本]][[東京都|東京都]]的[[警察|警察]]機關，由[[東京都廳|東京都政府]]下設的[[東京都公安委員會|東京都公安委員會]]管轄。至2015年度，警視廳設有102個[[警察署|警察署]]，[[警官|警官]]總數43,422人<ref>[http://www.keishicho.metro.tokyo.jp/sikumi/gaiyo/sosikizu.htm {{lang|ja|グラフ警視庁 組織図・体制 ：警視庁}}] {{webarchive|url=https://web.archive.org/web/20140623090101/http://www.keishicho.metro.tokyo.jp/sikumi/gaiyo/sosikizu.htm |date=2014-06-23 }}{{ja}}</ref>。

== 概要 ==
警視廳是日本各[[日本行政區劃|都道府縣]]中，唯一使用此名稱的地方警察機關（依據日本《警察法》第47條第1項；其他均以「警察本部」命名），因而常以「警視廳」一詞代稱東京都警察。另外有別於其他的地方警察機關，警視廳與[[北海道警察|北海道警察]]不屬警察廳下設的{{link-ja|管區警察局|管区警察局}}監管，而由[[警察廳|警察廳]]直接監管。警視廳本部位於[[霞關|霞關]]，由於附近的舊地名為「外櫻田門」，因而被通稱為「[[櫻田門|櫻田門]]」。警視廳在東京都内的各警察署被稱為「支店」、警視廳本廳則稱為「本店」。警視廳的首長為[[警視總監|警視總監]]。

由於[[東京|東京]]為[[日本首都|日本首都]]，聚集大多數的國家重要機關，使得警視廳有別於地方警察本部所擔當的特殊任務：
* [[天皇|天皇]]、[[日本皇室|皇族]]之警衛
* [[日本國會|立法機關]]、[[日本行政機關|行政機關]]、[[外國駐日本機構列表|駐日大使館]]、[[總理大臣官邸|首相官邸]]等重要設施之警備
* [[日本內閣總理大臣|內閣總理大臣]]等要人（[[國務大臣_(日本)|國務大臣]]、[[日本政黨列表|主要政黨]]領袖、[[執政黨|執政黨]]幹部）之警護

此外，與其他道府縣警察相比，警視廳轄區內警官與居民的比例達日本全國平均值的兩倍。

== 歷史 ==
*1874年（[[明治|明治]]7年）1月15日：東京警視廳創立
*1931年（[[昭和|昭和]]6年）：警視廳廳舍竣工
*1948年（昭和23年）5月25日：警視廳預備隊創立
*1954年（昭和29年）7月1日：[[警察法|警察法]]之制定與統合（現在之體制）
*1957年（昭和32年）4月1日：預備隊改名[[機動隊|機動隊]]
*1980年（昭和55年）：警視廳本部廳舍竣工
*1994年（[[平成|平成]]6年）4月1日：新式警察制服開始使用
*2000年（平成12年）4月：警視廳成立核生化搜查隊
*2002年（平成14年）：全國警務人員更換新型[[警察手冊|警察手冊]]，其參考美國[[FBI|FBI]]的設計
*2007年（平成19年）：巡邏警車車身上加注「POLICE」英文字樣。隨後，地域課、交通機動隊和新購的警車亦會加注。

== 規模 ==
警視廳管轄範圍為東京都全境，所屬職員之人数跟預算額都是[[日本警察|日本警察]]組織中規模最大的。

:所屬警察官：
*警察官　43,422人（2015年4月1日） 
*事務及技術人員　2,907人（同上）

== 方面本部 ==
