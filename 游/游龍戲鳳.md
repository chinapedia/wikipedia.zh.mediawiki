'''游龍戲鳳'''，即以[[明武宗|明武宗]]与民间女子李凤姐相爱为题材的民间文学，[[小说|小说]]、[[杂剧|杂剧]]、[[戏曲|戏曲]]、多有演绎，近代以来多被改编为[[电影|电影]]、[[电视剧|电视剧]]。[[京剧|京剧]]除游龙戏凤之名外，有称'''梅龙镇'''、'''下江南'''<ref>{{Cite web |url=http://www.pingshuxiazai.com/soft/2498.html |title=京剧《梅龙镇》剧本唱词全文 又名《游龙戏凤》《下江南》|accessdate=2017-03-28 |author= |date=2014-10-16 |work= |publisher=中国曲艺网|language=Zh-hans }}</ref>、'''美龙镇'''、'''骊珠梦'''、'''白凤冢'''，[[昆曲|昆曲]]和[[花部乱弹|花部乱弹]]称'''缀白裘·戏凤'''，地方剧称'''解带封宫'''、'''下江南'''、'''卖酒'''、'''龙戏凤'''、'''李凤姐卖酒'''、'''美龙镇'''、'''正德戏凤'''等<ref name="kf1">{{Cite web |url=http://www.kaiwind.com/culture/hot/201401/16/t20140116_1347965.shtml |title=明武宗与“游龙戏凤”|accessdate=2017-03-28 |author= |date=2014-01-17 |work=校对：李元，责任编辑：周原 |publisher=凯风网，来源：《北方文学》|language=Zh-hans }}</ref>。

故事主要讲述明武宗[[微服出巡|微服出巡]]，至梅龙镇，投宿于酒家李龙家，与其妹（或女）李凤姐相爱。最终结局亦有不同演绎，武宗或对李凤姐始乱终弃，或接入后宫，有情人终成眷属<ref name="kf1"/>。

== 历史背景 ==

明朝[[正德_(明朝)|正德]]年间，明武宗宠幸佞臣[[江彬_(明朝)|江彬]]。正德十二、十三年间，武宗以微服巡幸[[大同镇_(明朝)|大同]]、[[太原府|太原]]。江彬等人掠夺良家妇女数十车，跟随武宗前行，途中有死者。正德十四年（1519年），[[寧王之亂|寧王之亂]]起，使得明武宗有借口前往江南。江彬亦在[[扬州|扬州]]，搜罗处女、寡妇，又引武宗渔猎。武宗宠爱的[[刘美人_(明武宗)|刘姬]]劝谏，稍止<ref>《[[明史|明史]]•卷三百七•列传第一百九十五》◎佞幸</ref>。
