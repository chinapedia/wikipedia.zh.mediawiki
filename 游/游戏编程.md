'''游戏编程'''指利用计算机编程语言，如[[C编程语言|C编程语言]]、[[C++|C++]]编程语言、[[組合語言|汇编]]等，编制计算机或[[游戏机|游戏机]]上的游戏。

目前流行的游戏编程语言为C++编程语言，目前流行的游戏编程接口为[[DirectX|DirectX]] 9.0，还有[[OpenGL|OpenGL]]、[[SDL|SDL]]等。

== 游戏脚本 ==

游戏脚本是指在制作[[电子游戏|电子游戏]]的[[程序|程序]]时，[[程序员|程序员]]为电子游戏程序编写的[[脚本语言|脚本语言]]。脚本语言是为了缩短传统的编写-编译-链接-运行过程而创建的计算机[[编程|编程]]语言，而游戏脚本语言在电子游戏程序的编写中被广泛使用。

目前的[[电子游戏|电子游戏]]编程过程中大量使用'''游戏脚本'''来进行游戏程序的编写。例如在网络游戏[[魔兽世界|魔兽世界]]中就使用了[[Lua|Lua]]脚本。一般游戏脚本是由开发电子游戏的[[程序员|程序员]]来编写，但也可能由一些其他的游戏开发人员编写简单的游戏脚本。

== 游戏工具编程 ==

'''游戏工具编程'''是指采用各种开发工具进行开发修改[[电脑|电脑]]、[[电视|电视]][[游戏|游戏]]的过程。主要的开发工具有以下几大类：

# [[编程语言|编程语言]]工具以及[[集成编程环境|集成开发环境]]：[[Visual_C++|Visual C++]]、[[Visual_Basic|Visual Basic]]、[[Delphi|Delphi]]、[[.NET_Framework|.NET Framework]]系列（Visual C++.NET、Visual Basic.NET、Visual C#.NET）等等。
# 开发工具包：最有名的是[[DirectX|DirectX]]系列。
# 各种游戏厂家自己开发的[[封装度|封装度]]极高的制作工具，如[[電子角色扮演遊戲|角色扮演游戏]]制作工厂等等。

这类工具是将游戏必要的人物、道具以及场景绘制好后，由玩家自由调用组合，而很少甚至不需要制作者敲入一行代码就可完成整个游戏的制作过程。但是这种制作过程无法充分发挥制作人的创作才智，例如各种人物的设计就无法由创作者加以改变。
