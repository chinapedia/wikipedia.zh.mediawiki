{{vgname|游戏发展国|ja=ゲーム発展途上国}}是[[开罗游戏|开罗游戏]]开发的一款[[模擬遊戲|模拟类]][[电子游戏|电子游戏]]。游戏1997年4月在日本於Microsoft Windows平台發行。2010年10月9日，遊戲移植到了iOS和Android平台。<ref name="IGN">{{cite web|title=Game Dev Story iPhone Review|author=Trueman, Doug|publisher=[[IGN|IGN]]|url=http://uk.wireless.ign.com/articles/112/1128588p1.html|accessdate=2011-06-29|deadurl=yes|archiveurl=https://web.archive.org/web/20111001052851/http://uk.wireless.ign.com/articles/112/1128588p1.html|archivedate=2011-10-01}}</ref>游戏中，玩家在一个[[电子游戏产业|电子游戏公司]]进行决策，使公司壮大。游戏获得了评论家的好评，许多评论家赞赏其让人上瘾与游戏对电子游戏产业[[流行文化|流行文化]]的借鉴。

== 玩法 ==

游戏中，玩家在一个[[电子游戏产业|电子游戏公司]]进行决策，使公司创造热门游戏以获取利益。<ref name="IGN" />玩家能操控一系列影响游戏品质的因素，如员工的人事变动、游戏开发的方向和速度。玩家还可以使用一些道具提高员工的工作效率或改变员工工作的项目。随着游戏的进展，公司规模扩大，可以雇佣更多的工作人员，从而更容易创造品质更高的游戏。<ref name="IGN" />

一款游戏制作完成后会获得不同评论家评出的范围从1-10分的评分。分数通常影响游戏的销量，每周会根据游戏的销量对游戏进行排名。<ref name="IGN" />

当玩家制作出一款评分达到32分及以上的游戏后，游戏公司就达到了名人堂水平。

== 发行 ==

《游戏发展国》最初于1997年4月在日本于[[Microsoft_Windows|Microsoft Windows]]平台发行。<ref>{{cite web|title=Important Importables: Kairosoft|author=Lada, Jenni|date=2012-06-01|publisher=Technology Tell|url=http://www.technologytell.com/gaming/93512/important-importables-kairosoft/|accessdate=2015-06-01}}</ref>2010年，游戏移植到了[[iOS|iOS]]和[[Android|Android]]平台。<ref name="interviewta">{{cite web|title='Game Dev Story 2' Coming To iOS, But Not 'Soon'|author=Nicholson, Brad|date=2010-11-30|publisher=Touch Arcade|url=http://toucharcade.com/2010/11/30/game-dev-story-2-coming-to-ios/|accessdate=2012-05-10}}</ref>

== 评价 ==

《游戏发展国》发行后收获了评论的好评。许多评论家赞赏其让人上瘾与游戏对电子游戏产业[[流行文化|流行文化]]的借鉴。游戏在[[Metacritic|Metacritic]]获得了86%的评分。<ref name="mc">{{cite web|title=Game Dev Story|publisher=[[Metacritic|Metacritic]]|url=http://www.metacritic.com/game/ios/game-dev-story/critic-reviews?dist=positive|accessdate=2012-01-31}}</ref>《[[影音俱樂部|影音俱樂部]]》嘉奖游戏中玩家对公司发展决策的自由度。<ref name="AVCLUB">{{cite web|author=Heisler, Steve|date=2010-11-01|url=http://origin.avclub.com/articles/november-1-2010,46598/|title=Games: November 2010|publisher=''[[影音俱樂部|The A.V. Club]]''|accessdate=2012-01-31|deadurl=yes|archiveurl=https://web.archive.org/web/20111002124924/http://origin.avclub.com/articles/november-1-2010,46598/|archivedate=2011-10-02}}</ref>[[Eurogamer|Eurogamer]]的凯扎·麥當勞则称《游戏发展国》为“自己在iPhone上玩过的最棒的游戏”，他称赞了游戏的不确定性与令人上瘾的模拟玩法。<ref name="eurogamer">{{cite web|title=Game Dev Story Review|author=MacDonald, Keza|date=2010-11-02|publisher=[[Eurogamer|Eurogamer]]|url=http://www.eurogamer.net/articles/2010-11-02-game-dev-story-review|accessdate=2012-01-31}}</ref>《[[猴島小英雄系列|猴島小英雄系列]]》的开发者[[羅恩·吉伯特|羅恩·吉伯特]]告诉《[[连线|连线]]》称自己已对游戏上瘾。<ref name="wired">{{cite web|title=What's Right (and Wrong) With Game Dev Story's Addictive Simulation|author=Schreier, Jason|date=2010-12-03|publisher=''[[连线|Wired]]''|url=https://www.wired.com/gamelife/2010/12/game-dev-story/?pid=711&viewall=true|accessdate=2012-01-31}}</ref>

游戏的结尾阶段遭到了一些人的批评，《{{tsl|en|Pocket Gamer|口袋游戏人|口袋游戏人}}》认为游戏的结尾阶段太容易做出较为成功的游戏了。<ref name="pocketgamer">{{cite web|title=Game Dev Story review|author=Mundy, Jon|publisher=''{{tsl|en|Pocket Gamer|口袋游戏人|Pocket Gamer}}''|url=http://www.pocketgamer.co.uk/r/iPhone/Game+Dev+Story/review.asp?c=24481|accessdate=2012-01-31}}</ref>{{tsl|en|Giant Bomb|巨型炸弹|巨型炸弹}}的瑞恩·戴维斯表示，游戏的菜单風格较为老旧。但讚揚了遊戲富有魅力和令人上瘾的元素让其值得一玩。<ref name="GB">{{cite web|title=Game Dev Story review|author=Davis, Ryan|date=2010-11-02|publisher=[[Giant_Bomb|Giant Bomb]]|url=http://www.giantbomb.com/game-dev-story/61-32908/reviews/|accessdate=2012-01-31}}</ref>Thunderbolt认为，游戏只能吸引那些真正了解电子游戏产业的人。<ref name="thunderboltreview">{{cite web|title=Game Dev Story – iPhone review|author=Kemph, Calvin|date=2011-03-01|publisher=Thunderbolt|url=http://www.thunderboltgames.com/reviews/article/game-dev-story-review-for-iphone.html|accessdate=2012-01-31}}</ref>
