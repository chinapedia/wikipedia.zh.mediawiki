'''奈端数理'''（[[拉丁語|拉丁語]]：Philosophiæ Naturalis Principia Mathematica）是[[艾薩克·牛頓|艾薩克·牛頓]]所著《[[自然哲學的數學原理|自然哲學的數學原理]]》的一個十九世紀的[[漢語|漢語]]譯本，譯本已佚。奈端即為牛頓之譯名。

== 歷史 ==
牛頓的《自然哲學的數學原理》原文為拉丁文。中國清代數學家[[李善蘭|李善蘭]]及英國人[[傅蘭雅|傅蘭雅]]（John Fryer）、[[偉烈亞力|偉烈亞力]]（Alexander Wylie）將其中第一編共14章譯為漢語，時間約為1858至1860年。<ref>"学贯中西——李善兰传" by 杨自强</ref>1882年，李善蘭將譯稿交由[[華蘅芳|華蘅芳]]校訂至1897年。譯稿後為[[梁啟超|梁啟超]]所丟失。<ref name="梁启超丢失《奈端数理》译稿 戴念祖">{{cite web |url=http://catalogs.ihns.ac.cn/bitstream/311051/2515/1/%E6%A2%81%E5%90%AF%E8%B6%85%E4%B8%A2%E5%A4%B1_%E5%A5%88%E7%AB%AF%E6%95%B0%E7%90%86_%E8%AF%91%E7%A8%BF_%E6%88%B4%E5%BF%B5%E7%A5%96.pdf |title=存档副本 |accessdate=2014-02-03 |deadurl=yes |archiveurl=https://web.archive.org/web/20140221055225/http://catalogs.ihns.ac.cn/bitstream/311051/2515/1/%E6%A2%81%E5%90%AF%E8%B6%85%E4%B8%A2%E5%A4%B1_%E5%A5%88%E7%AB%AF%E6%95%B0%E7%90%86_%E8%AF%91%E7%A8%BF_%E6%88%B4%E5%BF%B5%E7%A5%96.pdf |archivedate=2014-02-21 }}</ref>

== 語言 ==
李善蘭翻譯與高等數學有關的書籍時創造了漢語中的「指數」、「微分」等詞，後亦傳入[[日語|日語]]之中。
