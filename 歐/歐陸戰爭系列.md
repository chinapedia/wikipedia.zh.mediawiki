'''歐陸戰爭'''（{{lang-en|''European War''}}）是由遊戲開發商[[Easy_Tech|Easy Tech]]開發的[[回合制策略遊戲|回合制策略遊戲]]系列。Easy Tech於2011年起陸續發佈歐陸戰爭系列遊戲。

== 概述 ==

歐陸戰爭系列遊戲，是以[[歐洲|歐洲]]作為主要戰場的回合制策略遊戲。該系列遊戲主要圍繞19世紀末至20世紀中期在歐洲發生的各場戰役。

== 系列作品 ==

=== 歐陸戰爭 ===

《歐陸戰爭》設定於18至19世紀的歐洲，戰鬥單位分為三種：步兵、[[騎兵|騎兵]]和大炮。遊戲供玩家選擇6個勢力，分別是：[[德國|德國]]、[[英國|英國]] 、[[法國|法國]] 、[[俄國|俄國]] 、[[奧匈帝國|奧匈帝國]] 和[[土耳其|土耳其]] 。玩家可以購買不同的指令牌達到目的，各國家的指令牌價錢亦不同。各主要勢力均有獨有的特別卡，以扭轉局勢。玩家只可以在[[首都|首都]]生產戰鬥單位。<ref>[http://itunes.apple.com/cn/app/ou-lu-zhan-zheng/id345541700?mt=8 欧陆战争 app store]</ref>

=== 歐陸戰爭2 ===

《歐陸戰爭2》設定於[[二戰|二戰]]前夕，戰鬥單位分為三種：步兵、[[坦克|坦克]]和大炮。玩法與歐陸戰爭大致相同，玩家可以在首都及第5級城市生產戰鬥單位，且可選勢力明顯增加。<ref>[http://itunes.apple.com/cn/app/ou-lu-zhan-zheng2/id409567943?mt=8 欧陆战争2 app store] </ref>

=== 歐陸戰爭3 ===

《歐陸戰爭3》設定於19世紀末至20世紀初，戰鬥單位分為九種：步兵、騎兵、大炮、[[機槍|機槍兵]]、[[裝甲車|裝甲車]]、坦克、[[驅逐艦|驅逐艦]]、[[巡洋艦|巡洋艦]]和[[戰艦|戰艦]]。玩法與歐陸戰爭、歐陸戰爭2和[[世界征服者系列|世界征服者 1945]]相差極大。戰鬥單位不再以數量計算，而是以生命值計算。單位的攻擊人數亦以生命值高低計算。擲骰的總和改為將會扣掉敵方的生命值。而且一塊領土可以有最多四個戰鬥單位。攻佔敵方首都亦不會致使敵方滅亡。而稅收亦分為兩種：金錢和工業點數。一部份指令牌需要金錢和工業點數，缺一不可。金錢在所有領土均可收到，[[城市|城市]]和首都能收取更多。而工業點數只可於工業區中收到。不同級別的戰鬥單位要於不同級別的工業區或城市生產，例如步兵、騎兵能於第3級或以上城市或第1級工業區中生產，坦克只能在第3級工業區中生產，而所有艦隻均要在碼頭中生產。國家獨有的指令牌被取消，而所有國家的指令牌價錢都相同。歐陸戰爭3亦新加了科技等級的設定，不同國家開始時有不同的科技等級。而科技等級越高，能使用的指令牌就越多。例如步兵和騎兵只需科技等級1便能生產，坦克則要科技等級5才能生產，遊戲新增機場，可以派遣戰鬥機、轟炸機或使用空軍技能。

除此之外，遊戲新加了一個元素：指揮官。裝備了指揮官的戰鬥單位戰鬥力會大大地提高。升級指揮官或購買戰役均需獎章，而獎章的獲得是在戰鬥時隨機贈送。遊戲還有一個新玩法：多人模式。多人模式允許玩家以一對一、二對二或三對三的模式戰鬥。遊戲的領土版圖亦大大增加，遊戲不再只局限於歐洲，還包括[[北美洲|北美洲]]和[[東亞|東亞]]。在歐陸戰爭3中亦也新增了帝國模式，其實就是玩家挑選國家後，再由遊戲給予的關卡進行。關卡是固定的，也較符合史實。例如北美洲的美國有[[南北戰爭|南北戰爭]]，以及大清帝國對戰[[太平天國|太平天國]]。

=== 欧陆战争4 ===
在欧陆战争4中，玩家将扮演一位将军去指挥部队与对手争夺资源。并且需要发展经济和工业，升级建筑，建造工事，攻占对方的领土。无论是使用陆军、舰队都需要您合理的安排。因每个部队都有自己的特性，在弱势时建设防御工事，扩张时建设要塞吸引火力。部队会在战斗后增加经验，当成为王牌部队后，战斗力将大大增加。指挥官可以亲自指挥部队进行战斗，自带的将军养成系统，可以提升将军所带部队的能力，所以提高指挥官军衔以及爵位也变得十分重要。游戏中可以选择战役模式、征服模式还有提供玩家之间对抗的多人模式进行游戏。

歐陸戰爭4將主遊戲分為：戰役、征服。在征服中有6隔年代可以選擇，分別為：1798、1789、1803、1806、1809...。進入征服模式，玩家選擇一個國家進入，國家被分為挺法派與反法派互相戰鬥。

== 玩法 ==

整個遊戲系列均以一塊塊[[領土|領土]]組成一個[[國家|國家]]。遊戲主要分為[[征服|征服]]模式和[[戰役|戰役]]模式。玩家命令戰鬥單位（主要是[[步兵|步兵]]和[[大炮|大炮]]以及[[騎兵|騎兵]]）移動和進攻，並且要使用指令牌加強戰鬥單位的攻擊和防禦、生產新單位或消滅敵軍。戰鬥單位以數量計算，並且進攻或防守時最高只能出動5個單位。進攻時，攻守雙方均要擲骰。若高點數的一方為進攻方，則防守方陣亡。若高點數的一方為防守方，則進攻方陣亡。若點數相同，則雙雙陣亡。當一塊領土沒有士兵時，則敵方可以進駐。而攻取敵方首都便能滅掉該國家。每回合開始時，玩家會收到一定稅收作為購買指令牌之用。

歐陸戰爭3起，玩法大有改變。歐陸戰爭4參考[[将军的荣耀|將軍的榮耀]]模式：作戰目標是攻陷所有指定的敵方重鎮，擊斃所有指定的敵方將領，同時守住所有指定己方重鎮，保護好至少一位指定的己方將領。相反一旦所有指定的己方將領陣亡或所有指定的己方重鎮陷落，則作戰失敗。指定的將領或重鎮會以紅色閃動標示出來。

=== 戰鬥單位 ===

; 歐陸戰爭

《歐陸戰爭》主要分為3個戰鬥單位：步兵、騎兵和大炮。

* 步兵：軍隊中最便宜的單位，缺點為攻擊力低。
* 騎兵：攻擊力比步兵高，因此價錢亦相對高。
* 大炮：大炮是一個特殊的戰鬥單位，擲骰所得點數為消滅敵方單位的數量。

; 歐陸戰爭2

《歐陸戰爭2》與《歐陸戰爭》相似，仍然分為3個戰鬥單位：步兵、大炮和坦克。

* 步兵：軍隊中最便宜的單位，缺點為攻擊力低。
* 坦克：攻擊力比步兵高，價錢亦為最高，但全滅5個敵人便可繼續攻擊。
* 大炮：大炮是一個特殊的戰鬥單位，攻擊步兵及坦克均不會被回擊。缺點是全滅敵方後不能進駐。


; 歐陸戰爭3

《歐陸戰爭3》分為9個戰鬥單位：步兵、騎兵、大炮、機槍兵、裝甲車、坦克、驅逐艦、巡洋艦和戰艦。

* 步兵：軍隊中最便宜的單位，缺點為攻擊力低。步兵的移動力為2，可於第三級或以上城市或第一級工業區中生產。
* 騎兵：騎兵的移動力非常高，移動力為3，且其戰鬥力比步兵高，因此價錢比步兵稍高，可於第三級或以上城市或第一級或以上工業區生產。
* 大炮：大炮繼承了《歐陸戰爭2》中大炮的特色，當大炮攻擊其他陸上單位時，其他單位並不能還擊（坦克除外）。同樣，大炮在全滅領土上的敵軍後不能進駐該地。大炮的移動力為1，可於第一級或以上工業區生產。
* 機槍兵：機槍兵是一個新的戰鬥單位，其特色為反擊。當與步兵或騎兵戰鬥時，機槍兵在傷害結算後再次攻擊敵方，造成雙重傷害。但其缺點為生命值低。機槍兵的移動力為1，可於第二級或以上工業區生產。
* 裝甲車：裝甲車是軍隊中的中堅部隊，攻擊力高且生命值高，而且消滅一單位後仍然可以繼續攻擊。但裝甲車價錢相對較高。裝甲車的移動力為2，可於第二級或以上工業區生產。
* 坦克：坦克是軍隊中的超級武器，其攻擊力、生命值非常高、消滅一單位後仍然可以繼續攻擊，且不會被戰壕所阻擋並能夠反擊大炮。唯一缺點就是，坦克價錢非常高。坦克的移動力為2，只可於第三級工業區生產。
* 驅逐艦：驅逐艦是海軍中最便宜的戰鬥單位，因此戰鬥力亦相對低。移動力為1。
* 巡洋艦：巡洋艦擁有中等程度的戰鬥力，因此價錢比驅逐艦高。移動力為1。
* 戰艦：稱霸海洋的最強海軍，攻擊力為海軍中最高，但價錢高昂。移動力為1。


; 歐陸戰爭4
《歐陸戰爭4》分設有4種兵種：步兵，砲兵，騎兵，海軍；並有特殊固定單位要塞

* 步兵
# [[民兵|民兵]]：最廉價而戰鬥力最弱的單位
# [[线列步兵|線列步兵]]：生存力強的基礎步兵
# [[散兵|輕步兵]]：攻擊力穩定，機動性強但生存力較低的步兵
# [[擲彈兵|擲彈兵]]：全面戰鬥力較強的步兵，配備用以打擊碉堡工事的手榴彈
# [[帝国卫队_(拿破仑一世)|近衛軍]]: 全面戰鬥力最強的步兵，配備用以打擊碉堡工事的手榴彈
# [[加特林机枪|機槍]]: 威力巨大專門殺傷敵人有生力量的單位，但是射程短，生存力不高，機動性差

* 騎兵
# [[轻骑兵|輕騎兵]]：使用馬刀的騎兵部隊，機動性強
# [[槍騎兵|重騎兵]]：使用長槍的騎兵部隊，戰鬥力強
# [[龍騎兵|近衛騎兵]]：使用滑膛槍的騎兵部隊，戰鬥力最強，配備用以打擊碉堡工事的手榴彈
# [[裝甲車|裝甲車]]： 搭載機槍的裝甲車輛，戰鬥力和生存力極強，但是造價昂貴

* 砲兵
# [[炮|輕炮]]：具備遠程攻擊能力，射程不足的目標敵人無法反擊
# [[砲|重砲]]：具備遠程攻擊能力，射程不足的目標敵人無法反擊；火力強勁但機動性差
# [[臼炮|圍城砲]]：專門打擊碉堡工事的特殊火砲，只能執行遠程攻擊或反擊
# [[火箭炮|火箭炮]]：原型為英國的康格里夫火箭炮，攻擊時能連帶擊傷與目標相鄰的敵人或友軍，且目標敵人無法以任何方式還擊；火力強勁但生存力不高，只能執行遠程攻擊或反擊

* 海軍
# [[私掠船|私掠船]]：造價低廉的輕型軍艦，航速高，火力中等但射程短
# [[護衛艦|護衛艦]]：中型軍艦，航速高，火力中等且射程遠
# [[戰艦|戰艦]]：主力重型軍艦，航速中等，火力強而射程遠
# [[鐵甲艦|鐵甲艦]]：具備極高抗打擊能力的重型軍艦，航速中等，火力強而射程遠

* 要塞
# [[要塞|小型要塞]]：步槍兵駐守的木質臨時要塞
# [[菱堡|中型要塞]]：裝備火砲的菱堡，抗打擊能力高
# [[星形要塞|大型要塞]]：裝備火砲的星形要塞，抗打擊能力極高
# [[海岸炮|海岸炮]]：專門打擊艦船的固定火砲，不適宜攻擊陸軍

歐陸戰爭4 主要將領列表

歐陸戰爭4採取了《將軍的榮耀》形式的將領系統，並大幅拓展將領數目。遊戲取錄了大部分參與十九世紀初幾場大規模戰事的將領。
