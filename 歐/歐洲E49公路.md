'''欧洲E49公路'''（'''E49'''）是欧洲的一条[[欧洲国际公路|国际公路]]，起点是[[德国|德国]]的[[馬德堡|馬德堡]]，终点为[[奧地利|奧地利]]的[[維也納|維也納]]，全長約759公里。<ref>{{Cite web |url = http://www.elbruz.org/eroads/E49.htm |title =  E49  |publisher = Elbruz |accessdate = 20 April 2015}}</ref>
[[Image:A22_Kaisermuehlen.JPG|thumb]]段]]
== 线路 ==
欧洲E49公路穿过以下主要城市：<ref>{{Cite web |url = https://treaties.un.org/doc/Publication/UNTS/Volume%201302/volume-1302-I-21618-English.pdf |title =  European Agreement on main international traffic arteries. (AGR) - Consolidated version  |publisher = UN Treaty Collection |accessdate = 20 April 2015}}</ref>
*德国：[[馬德堡|馬德堡]] - [[哈雷_(德國)|哈雷]] - [[普勞恩|普勞恩]]
*捷克：[[海布|海布]] - [[卡羅維瓦利|卡羅維瓦利]] - [[皮爾森|皮爾森]] - [[皮塞克|皮塞克]] - [[捷克布傑約維采|捷克布傑約維采]] - [[特熱邦|特熱邦]] 
*奧地利：[[霍恩_(奧地利)|霍恩]] - [[施托克勞|施托克勞]] - [[維也納|維也納]]
