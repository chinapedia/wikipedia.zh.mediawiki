'''歐鯰'''（[[学名|学名]]：''{{lang|la|Silurus glanis}}''）又稱'''歐洲巨鯰'''，為[[輻鰭魚綱|輻鰭魚綱]][[鯰形目|鯰形目]][[鯰科|鯰科]]的其中一[[種|種]]，是在[[歐洲|歐洲]]生活的[[淡水|淡水]]鯰{{efn|“[[wikt:鯰|鯰]]”、“[[wikt:鮎|鮎]]”二字常通用。}}和[[汽水_(水域)|汽水]]鯰，是世界上最大的[[淡水魚|淡水魚]]之一以及世界最大的[[鯰魚|鯰魚]]之一。

== 體态 ==
=== 體色 ===
歐鯰的体色各不相同，有的身体全黑，而有的头部则是深黄绿色或蓝黑色。歐鯰肋部颜色稍浅，还有极少数体表-{}-带有棕红色光泽；而其腹部是白色的。全身均为蓝黑色或白色的歐鯰个体也出现过。除了底色以外，歐鯰的身上还有云片状或点状的花纹。[[鳍|鳍]]色为深棕红色或紫褐色。<ref>[http://www.scotcat.com/glanis.html Silurus glanis] {{webarchive|url=https://web.archive.org/web/20080915212827/http://www.scotcat.com/glanis.html |date=2008-09-15 }}, ScotCat.com</ref>

歐鯰的體色主要會隨著水的清澈程度而受影響，生活在混濁水域中歐鯰體色是褐色，而生活在清澈水域中歐鯰體色則是偏向黑色，此外，季節也會影響歐鯰的體色。<ref>[http://www.yarfish.ru/en/som]{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}, yarfish.ru</ref>

=== 特徵 ===
歐鯰身体很长，前端圆柱形，末端较扁平，[[头部|头部]]较大且扁平，[[口部|口部]]宽而大，有3对触须，上颌触须放松时尤其长，可以触及胸前。[[脊鳍|脊鳍]]十分小，向头部方向插入身体，但无脂鳍。臀鳍基部很长，這些特徵使歐鯰十分容易就可辨識得出來。

=== 體型 ===
傳說中最大的歐鯰體長達到5公尺，重達336公斤（但大部分的歐鯰體長大約在2公尺左右，體重很少會超過100公斤），生長於19世紀的[[俄羅斯|俄羅斯]]，此紀錄比已知的任何[[鯰形目|鯰形目]]都要大許多，包括[[湄公河巨鯰|湄公河巨鯰]]（''Pangasianodon gigas''）
的3公尺，重350公斤、[[撒旦鴨嘴|撒旦鴨嘴]]([[絲條短平口鯰|絲條短平口鯰]])（''Brachyplatystoma filamentosum''）最大可達360公分，重200公斤，以及[[鮡科|鮡科]]的鯰魚，俗稱[[坦克|坦克]]鴨嘴([[巨魾|巨魾]])（''Bagarius yarrelli''）的2公尺，重90.7公斤，及兩個近親[[琵琶湖鯰|琵琶湖鯰]]（''Silurus biwaensis''）的2.4公尺(240公分)，重140 公斤，以及[[懷頭鯰|懷頭鯰]]（''Silurus soldatovi''）的4公尺(400公分)，重40公斤。

== 习性 ==

=== 栖息 ===
歐鯰喜欢居住在广阔温暖、有大量植被的的[[湖|湖]]或死水塘中，以及水流缓慢的深水江[[河|河]]中，有時他們也會棲息在[[水庫|水庫]][[上游|上游]]。它们喜欢栖息在隐蔽的地方，如河床上的孔洞和沉木等。歐鯰在开放水域或水体底部捕食。人们将歐鯰作为[[食用鱼|食用鱼]]饲养在[[鯉鱼|鯉鱼]]塘中，同时也将它们视为宠物。

=== 食性 ===
歐鯰居[[河川|河川]][[食物鏈|食物鏈]]的上層，幾乎什麼東西都吃，主要以[[环节动物门|环虫]]、[[腹足纲|螺]]、[[昆虫|昆虫]]和[[甲壳亚门|甲壳类]]维生，有时也吃[[蛇|蛇]]、[[龜|龜]]、[[魚|魚]]、[[鼠|鼠]]、[[水鸟|水鸟]]（如[[鴨|鴨]]），甚至[[田鼠|田鼠]]，而歐鯰也吃過一條[[狗|狗]]，因此有一個可怕的外號為「殺手庫諾」（不過殺手庫諾懷疑已在2003年死亡）。在[[中世紀|中世紀]]時，就有許多攻擊人的傳聞，那時，歐鯰被形容成殺人不眨眼的吃人怪物，能一口將嬰孩吞下，在有廣泛歐鯰棲息的水域中，常有巨大的身影攻擊人類事件；目前只能確認有歐鯰吃過人、但不知道是吃活人還是吃屍體，也有歐鯰嘗試捕捉活人但未成功。

== 繁育 ==
在其栖息地，雌歐鯰一般于5月－7月将卵产在由雄歐鯰挖掘的浅[[洼地|洼地]]或是河岸中，产下的卵每千克体重可以达300000个，光是[[夏季|夏季]]繁殖期，一隻歐鯰的卵，加總起來共有70萬顆以上，卵由雄魚照顧直至小鲶鱼孵化出，此过程需3－10天，所需时间取决於水温。如果水位变化过大或过快，雌歐鯰会用其强健有力的尾鳍向卵上泼水，以保持卵的湿润；此外，若是有類似淡水[[龍蝦|龍蝦]]、鯉魚等偷蛋賊，歐鯰就會強制驅離甚至攻擊，就連人類也不例外，這也是牠們會攻擊小[[船|船]]及泳客的原因。

== 壽命 ==
歐鯰生长极快，只需4至5[[年|年]]就可以发育成熟，寿命一般可达到20或30年，而目前已知的最长寿的歐鯰甚至活到80岁。

== 分布 ==
歐鯰原產於[[中亚|中亚]]的[[鹹海|鹹海]]、[[南歐|南歐]]、[[中歐|中歐]]和[[東歐|東歐]][[德國|德國]]的[[萊茵河|萊茵河]]上游、[[波羅的海|波羅的海]]和[[里海|里海]]、[[亞洲|亞洲]]東北部，目前甚至持續入侵到[[西班牙|西班牙]]等歐洲國家，也就是後引入到[[中國|中國]]大陸以及其他[[國家|國家]]，其分布範圍十分廣泛。

== 渔业 ==
=== 市场 ===
歐鯰雖然生活於歐洲的各個大小國家中，但最不到當地人的愛戴，因為歐鯰的肉容易發黃，不易保存，加上土味又太重，使的市場反應十分不佳；近年來[[養殖業|養殖業]]者為了滿足中國饕客的需求，而將歐鯰大量引進中國，並且迅速的在該國市場中落地生根。而且因為出口限制，大部分市場很少機會見到白子歐鯰。歐鯰的[[卵|卵]]有毒，不可食用。

=== 捕捞 ===
诱捕歐鯰的最佳鱼饵之一是[[水蛭|水蛭]]，可谓百发百中，几乎从来不会失败。制作钓饵要小心，注意不能破坏水蛭的体节，这样其能生存较长时间，发出的声音可以吸引歐鯰，而另外一種魚餌是淡水[[鰻魚|鰻魚]]，跟[[水蛭|水蛭]]一樣幾乎沒有失誤過，並且可保證一定吸引歐鯰。其他常用的鱼饵包括小鱼、大[[蚯蚓|蚯蚓]]团、其他昆蟲、鯉魚、动物内脏和[[鱿鱼|鱿鱼]]块。冬天，大量的歐鯰会聚集在河床的低洼地，这样人们可用[[拖网渔船|拖网渔船]]大量捕获歐鯰。抓住的歐鯰很滑，而且不停挣扎，在[[船|船]]上不易控制，一种解决办法是用手指夹住触须，因为其触须很敏感，这样做会使它们痛得不能动弹。

=== 相似種 ===
有些鯰形目的魚類長的與歐鯰白子差不多，例如[[懷頭鯰|懷頭鯰]]。這種鯰魚體色、體態與歐鯰白子相似，不過歐鯰的頭皆比這種[[鯰魚|鯰魚]]尖，而且歐鯰白子的體色帶一點粉紅色，相比有些差異。

=== 體型 ===
歐鯰在最大的淡水魚名單中，佔有一席之位，不論用什麼方法比較，歐鯰都比大部分的淡水魚大，可以媲美[[美國白鱘|美國白鱘]]、[[歐洲鱘|歐洲鱘]]、[[中華鱘|中華鱘]]、[[大西洋鱘|大西洋鱘]]、[[巨骨舌魚|巨骨舌魚]]、近親[[懷頭鯰|懷頭鯰]]。

== 觀賞用途 ==
歐鯰不仅為[[養殖|養殖]]與食用魚類，也是十分常見的[[觀賞魚|觀賞魚]]。在[[水族館|水族館]]裡，最常見的種類是小型歐鯰的白化幼仔，體型一般可以達到1.85公尺（6英尺），生長一段時間後可達到2.78公尺（9英尺），但因為空間過小，所以用來觀賞的歐鯰最大通常不會超過3公尺。不过，若使用室內游泳池，最大紀錄可以達到4.75公尺，幾乎可以達到野生種5公尺的最大體型紀錄。關於食物，飼養者可以先餵食歐鯰[[鴨|鴨]]、[[雞|雞]]，之後便可以進一步餵食[[鱈魚|鱈魚]]、[[蟑螂|蟑螂]]等食物；而另一種餵食方法，適用於大型歐鯰，飼養者可將[[魷魚|魷魚]]或大型魚類的肉切成塊，但若想餵食其他食物也是可以的，如[[小龍蝦|小龍蝦]]（螯蝦）。

== 鯰魚吞籃球 ==
網路上曾流傳[[大洋洲|大洋洲]][[悉尼|悉尼]]北部一隻大鯰魚吞籃球的圖片，顯然這隻鯰魚餓壞了，才把籃球誤以為食物，結果吐不出來。這隻鯰魚在體型上與外型上看起來跟歐鯰極為相似，不過也有可能是其他大型鯰魚，例如[[坦克鴨嘴|坦克鴨嘴]]（''Bagarius jarrelli''）、[[奧圖鯰|奧圖鯰]]（''Wallago attu''）。<ref>[http://www.gamez.com.tw/showthread.php?t=281274 鲶鱼吞篮球卡在嘴里被迫漂浮湖面]</ref>

== 汙染物含量 ==
在最近[[土耳其|土耳其]]进行的魚類体内污染物（[[汞|汞]]、[[鉛|鉛]]等[[重金屬|重金屬]]與其他物質）含量的調查報告中指出，歐鯰與其他三種魚類（''Achanthobrama marmid''、''Chondrostoma regium''和''Cyprinus carpio''）汙染物含量較多，雖然其汙染物對於魚體本身是幾乎無害的，但卻可以造成體質較敏感的[[哺乳類|哺乳類]]、[[鳥類|鳥類]]死亡，雖然此區已有限制污染物不可超過一定含量的標準，並且，這些魚類污染物含量低於該地區標準，但隨著[[農業|農業]]與[[工業|工業]]的發展，污染造成的問題也可能會發生。
