'''稗田阿禮'''（{{Jpn|j=稗田阿礼|kyu=稗田阿禮|hg=ひえだの あれ}}），生卒年不詳<ref>可能是7世紀後期至8世紀初期。</ref>，是[[日本|日本]][[奈良時代|奈良時代]][[貴族|貴族]]的雜役，亦是《[[古事記|古事記]]》的編纂者之一。

== 生平略歷 ==
稗田阿禮的出身至今仍是個謎，同時代編成的《[[日本書紀|日本書紀]]》及[[平安時代|平安時代]]編成的《[[續日本紀|續日本紀]]》皆無記載此人之資料。僅從《[[古事記|古事記]]》[[序|序文]]得知他是[[元明天皇|元明天皇]]的[[舍人|舍人]]<ref>在皇族貴族家中擔任警備、打雜之雜役。</ref>，年齡為28歲，博聞強記的他能背誦《[[帝紀|帝紀]]》、《[[舊辭|舊辭]]》等史料，故天皇召來[[太安萬侶|太安萬侶]]記錄稗田阿禮口誦之事，編纂成《古事記》三卷。

《古事記》之序文原文為：
{{Quote|時有舍人名稗田阿禮，年廿有八，為人聰穎，過目而能誦，耳聞而不忘，敕語阿禮，令誦習帝皇日繼及先代舊辭。然運移世異，未行其事矣。<br>
故天皇嘆於舊辭誤亂，欲修帝紀之亂，和銅四年九月十八，詔臣安萬侶以撰稗田阿禮所誦之舊辭，而成書獻上，故謹隨詔旨慎加採錄。
}}

== 爭議問題 ==
通常幫日本[[貴族|貴族]]料理雜役的「[[舍人|舍人]]」由男性擔任，不過[[江戶時代|江戶時代]]卻曾出現「稗田阿禮是女性」的說法，由民俗學者[[柳田國男|柳田國男]]、神話學者{{link-ja|西鄉信綱|西郷信綱}}等人提出。他們認為{{link-ja|稗田氏|稗田氏}}與{{link-ja|猿女君|猿女君}}的始祖同樣為[[天鈿女命|天鈿女命]]（{{lang-ja|アメノウズメ}}），猿女君本為朝廷中擔任神職角色的[[巫女|巫女]]或[[女孺|女孺]]<ref>在日本天皇後宮裡擔任打掃、照明等雜役之下級女官。</ref>。而「阿禮」這個詞本來就是巫女的代稱，更加印證了「稗田阿禮是女性」的說法。

多年來此見解幾成定論，不過鑽研日本古代史甚力的學者[[梅原猛|梅原猛]]，基於《古事記》大膽狂放的文筆寫法及年齡推算，提出了「稗田阿禮乃是[[藤原不比等|藤原不比等]]的別名」的新見解。除此之外，寶賀壽男編著的《古代氏族系譜集成》將稗田阿禮當成中臣磐余的孫子。

== 信仰 ==
主祀稗田阿禮的[[神社|神社]]包含下列：
* {{link-ja|賣太神社|賣太神社}}（[[奈良縣|奈良縣]][[大和郡山市|大和郡山市]]稗田町）：主祭神為稗田阿禮命，由於附近居住著{{link-ja|猿女君|猿女君}}稗田氏後代，被視為稗田阿禮的出身地。
* 稗田神社（[[兵庫縣|兵庫縣]][[揖保郡|揖保郡]][[太子町_(兵庫縣)|太子町]]）：主祭神為阿禮比賣命。
此外，[[岐阜縣|岐阜縣]]的{{link-ja|飛驒細流街道|飛騨せせらぎ街道}}也豎立著「稗田阿禮誕生之地」的看板。
