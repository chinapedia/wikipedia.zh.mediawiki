'''管制塔'''（{{jp|j=管制塔|hg=かんせいとう|rm=Kanseitō|lead=yes}}）是一部由{{link-ja|三木孝浩|三木孝浩}}執導，[[山崎賢人|山崎賢人]]和[[橋本愛|橋本愛]]主演的2011年日本电影。電影以[[日本|日本]][[樂團|樂團]][[Galileo_Galilei_(樂團)|Galileo Galilei]]的歌曲《管制塔》為主題<ref name="sohu">{{cite web | url=http://yule.sohu.com/20110409/n305549046.shtml | title=《管制塔》东京上映 山崎贤人桥本爱主演(图) | date=2011-04-09 | accessdate=2014-01-11}}</ref>。

== 劇情簡介 ==
“作為一個僅僅15歲，什麼也做不了的普通少年，現在，我能為小米做的，只有做樂曲和繼續唱下去。” 主人公中學生藤田駈成長在日本最北端的城市，[[稚內|稚內]]。藤田在平淡無奇的生活中感受不到任何歸屬感，直到他遇到了轉校生瀧本瑞穗，一個像動畫片[[姆明|姆明]]裡的小米一樣的女孩。一天，藤田偶然在家裡的倉庫發現了舊[[吉他|吉他]]，於是他和瀧本決定組建樂隊，接下來會發生什麼呢？<ref>{{cite web | url=http://movie.douban.com/subject/5951271/ | title=管制塔 - 豆瓣電影 | accessdate=2014-01-11}}</ref>

== 製作背景 ==
電影以正處嚴寒冬季的[[Galileo_Galilei_(樂團)|Galileo Galilei]]故鄉[[北海道|北海道]][[稚內市|稚內市]]為舞台，講述了對未來迷茫的孤獨少年藤田駈（山崎賢人飾）與轉校女生瀧本瑞穂（橋本愛飾）通過追求音樂的夢想，將二人的心緊密聯結的純愛物語<ref>{{cite web | url=http://ent.163.com/11/0411/08/71BKMJHI000300B1.html | title=桥本爱新片《管制塔》首映 自曝冬天穿短裤上课_网易娱乐 | date=2011-04-11 | accessdate=2014-01-11}}</ref>。

電影根據Galileo Galilei的同名歌曲「管制塔」歌詞內容延展編創而成，講述青春期少男少女以樂抒懷的故事，帶著些許[[自傳|自傳]]性的元素，具象化了樂隊的音樂世界。對於創作這個故事的起源，導演三木孝浩表示說：「去看Ga​​lileo Galilei演出時，對於他們『在什麼地方長大』這個問題產生了興趣。」「雖說物質生活富足，但十幾歲的少年人都會有無人理解自己的感覺，電影描寫的正是這份感覺，希望無論是否為Galileo Galilei的樂迷，大家都能來看這部具有普遍性的電影。」<ref>{{cite web | url=http://www.m1905.com/mdb/film/2207205/feature/ | title=管制塔幕后花絮_电影网 | accessdate=2014-01-11}}</ref>

由於影片從頭至尾都在冬天的稚內市作為外景地進行拍攝，過程中山崎賢人曾說：“太冷了，嘴凍僵了，[[台詞|台詞]]也說不流利了，穿著戲服，真是太難受了。”橋本愛則笑著說：“初頭真是冷得不行，後來反而習慣了。結果回來以後，上體育課時只有我還穿著短袖衫。”對於和山崎賢人的合作，橋本則回憶道：“第一次見面時還擔心能和他好好對話嗎？合作要不要緊哪？不過拍攝過程中就了解到他是個親切的人。”山崎賢人則評價橋本愛“雖然比我還小一歲，不過非常成熟穩重。”<ref>{{cite web | url=http://www.headline01.com/yule/rhyl/dianying/63439.html | title=《管制塔》东京上映 讲述美好的青春故事 | date=2013-10-30 | accessdate=2014-01-11 }}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>
