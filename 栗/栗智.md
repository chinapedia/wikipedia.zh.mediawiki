'''栗智'''（{{bd|1950年|11月|||L栗}}），[[安徽省|安徽省]][[利辛县|利辛县]]人，[[新疆维吾尔自治区|新疆维吾尔自治区]]原[[人民代表大会制度|人大常委会]]副主任、党组成员，自治区第十二届人大常委会党组成员。是第十、十一届[[全国人民代表大会|全国人民代表大会]]代表，[[中国共产党第十七次全国代表大会|中共十七大]]代表。2015年3月11日被[[中国共产党中央纪律检查委员会|中纪委]]宣布调查，系[[中国共产党第十八次全国代表大会|中共十八大]]后新疆首位落马省部级官员<ref>{{cite web|title=栗智系新疆首位落马省部级官员|url=http://news.163.com/15/0311/15/AKEF046O00014JB5.html|accessdate=2015-03-11|publisher=网易}}</ref>。2015年7月6日被双开并移送司法机关。据中纪委的通报消息称，他除涉嫌收受贿赂、通奸、转移、藏匿赃款赃物、收受礼金外，还涉嫌档案造假，向组织隐瞒本人真实年龄<ref name="开除党籍">{{cite web|title=新疆维吾尔自治区人大常委会原副主任、党组成员栗智严重违纪违法被开除党籍 |url=http://www.ccdi.gov.cn/xwtt/201507/t20150706_58796.html|date=2015-07-06 |accessdate=2015-07-06|publisher=中央纪委监察部网站}}</ref>。

== 基本情况 ==
栗智1969年12月参加[[中国人民解放军|中国人民解放军]]，1971年3月加入[[中国共产党|中国共产党]]，1974年退伍后在[[新疆维吾尔自治区|新疆]][[尼勒克县|尼勒克县]]工作，同年考入[[西华师范大学|南充师范学院]]（现[[西华师范大学|西华师范大学]]）生物系生物专业。1977年10月毕业后进入新疆维吾尔自治区轻工厅供销公司工作。2000年7月毕业于[[新疆大学|新疆大学]]经济管理专业，在职研究生，高级经济师。

1993年，栗智出任新疆啤酒花股份公司董事长、总经理。1996年升任新疆维吾尔自治区轻工厅副厅长，1997年12月任轻工厅党组书记。2000年调任[[昌吉回族自治州|昌吉回族自治州]]副书记，次年升任[[博尔塔拉蒙古自治州|博尔塔拉蒙古自治州]]书记，并兼任[[新疆生产建设兵团|新疆生产建设兵团]]农五师党委[[第一书记_(中国)|第一书记]]、第一[[政治委員|政委]] ；2005年1月调任昌吉州委书记，兼任新疆生产建设兵团农六师党委第一书记、第一政委； 2006年年11月升任[[乌鲁木齐市|乌鲁木齐]]市委书记，兼任[[新疆生产建设兵团|新疆生产建设兵团]]农十二师党委第一书记、第一政委，2008年1月当选为[[新疆维吾尔自治区|新疆维吾尔自治区]]第十一届[[人民代表大会制度|人大常委会]]副主任。2009年9月5日，经[[中国共产党中央委员会|中共中央]]同意，新疆维吾尔自治区党委决定，免去栗智同志[[乌鲁木齐市|乌鲁木齐]]市委书记职务。西方媒體评论认为，栗智此次免职是当了“[[替罪羊|替罪羊]]”，以削弱[[乌鲁木齐七五事件|乌鲁木齐七五事件]]后要求自治区党委书记[[王乐泉|王乐泉]]下台的呼声。

== 人物简历 ==
*1969年12月－1974年7月，[[新疆维吾尔自治区|新疆维吾尔自治区]][[乌鲁木齐市|乌鲁木齐]]警备区消防营文书；
*1974年7月－1974年10月，新疆维吾尔自治区尼勒克县粮食局门市部工人；
*1974年10月－1977年10月，南充师范学院（现[[西华师范大学|西华师范大学]]）生物系生物专业学习；
*1977年10月－1985年4月，新疆维吾尔自治区轻工厅供销公司原料科干部；
*1985年4月－1987年2月，新疆维吾尔自治区轻工厅供销公司销售科副科长；
*1987年2月－1990年2月，新疆维吾尔自治区经协办干部、服务公司副经理；
*1990年2月－1993年2月，新疆维吾尔自治区轻工原料基地开发公司酒花科科长、经理助理；
*1993年2月－1996年10月，新疆维吾尔自治区轻工供销公司党委副书记、总经理兼新疆啤酒花股份公司董事长、总经理（1995年9月－1996年7月，在新疆维吾尔自治区党委党校中青班学习）；
*1996年10月－1997年12月，新疆维吾尔自治区轻工厅党组成员、副厅长；
*1997年12月－2000年7月，新疆维吾尔自治区轻工厅党组书记、副厅长（1998年9月－2000年7月，在[[新疆大学|新疆大学]]经济系经济管理专业研究生课程进修班学习，2000年3月－*2000年7月，在中央党校进修班学习）；
*2000年7月－2001年1月，新疆维吾尔自治区[[昌吉回族自治州|昌吉回族自治州]]党委副书记；
*2001年1月－2005年1月，新疆维吾尔自治区[[博尔塔拉蒙古自治州|博尔塔拉蒙古自治州]]党委书记，2001年3月起兼任[[新疆生产建设兵团|新疆生产建设兵团]]农五师党委[[第一书记_(中国)|第一书记]]、第一[[政治委員|政委]]；
*2005年1月－2006年11月，新疆维吾尔自治区乌昌党委副书记，昌吉回族自治州党委书记 ，2005年4月起兼任新疆生产建设兵团农六师党委第一书记、第一政委；
*2006年11月－2008年1月，新疆维吾尔自治区乌昌党委书记，乌鲁木齐市委书记，新疆生产建设兵团农十二师党委第一书记、第一政委；
*2008年1月－2015年3月，新疆维吾尔自治区人大常委会副主任（至2013年1月）、党组成员（至2014年3月<ref>{{Cite web|url=http://www.ts.cn/special/content/2008-04/02/content_2520213.htm|title=自治区党委近期干部任免|accessdate=2017-05-24|author=|date=2008-04-02|work=|publisher=新疆天山网}}</ref>），新疆维吾尔自治区乌昌党委书记、乌鲁木齐市委书记（至2009年9月），新疆生产建设兵团农十二师党委第一书记、第一政委（至2009年9月）；
*2015年3月11日，因涉嫌严重违纪违法，接受[[中国共产党中央纪律检查委员会|组织]]调查<ref>{{cite web|publisher=中纪委监察部网站|accessdate=2015-03-11|title=新疆维吾尔自治区人大常委会原副主任栗智涉嫌严重违纪违法接受组织调查|url=http://www.ccdi.gov.cn/xwtt/201503/t20150311_53153.html}}</ref>
*2015年7月6日被双开并移送司法机关<ref name="开除党籍"/>。
*2016年6月23日在[[青海省|青海省]][[西宁市|西宁市]]中级人民法院审讯，栗智被控受贿1319万，当庭表示认罪、悔罪<ref>[http://news.enorth.com.cn/system/2016/06/23/031034431.shtml 新疆人大原副主任栗智被控受贿1319万 当庭认罪]</ref>。
*2016年11月18日栗智因犯受贿罪被判处有期徒刑12年，并处罚金人民币100万元<ref>{{cite web |url = http://www.chinacourt.org/article/detail/2016/11/id/2352556.shtml |title = 新疆人大常委会原副主任栗智一审被判刑十二年 |accessdate = 2016年11月18日 |publisher = 中国法院网}}</ref>。
