'''男鹿和雄'''（{{bd|1952年2月29日|catIdx=O}}）是出身[[日本|日本]][[秋田縣|秋田縣]]的男性[[美術指導|美術監督]]，曾參與多部[[川尻善昭|川尻善昭]]執導或[[吉卜力工作室|吉卜力工作室]]動畫作品的背景美術製作。

== 經歷 ==
自秋田縣立角館高校畢業後，男鹿和雄在1972年進入動畫美術公司的[[小林製片|小林製片]]工作，中間曾有一段時間轉職從事看板設計方面的職務，之後1977年則再度回到小林製片參與動畫的背景繪製。

後續男鹿和雄轉到[[MADHOUSE|MADHOUSE]]動畫公司下就職。在MADHOUSE期間參與由[[川尻善昭|川尻善昭]]改編自[[菊地秀行|菊地秀行]]小說《[[魔界都市〈新宿〉|魔界都市〈新宿〉]]》同名動畫中，曾影響了男鹿和雄的美術表現技法<ref>{{Cite web |url=http://www.adnet-sakigake.com/kyo/interview/oga/ |script-title=ja:Kazuo Oga　男鹿　和雄さん　アニメーション美術監督・挿画家 |work=adnet-sakigake |date=2005-06 |accessdate=2012-09-13 |deadurl=yes |archiveurl=https://web.archive.org/web/20131103014437/http://www.adnet-sakigake.com/kyo/interview/oga/ |archivedate=2013-11-03 }}</ref>。

1987年在友人的介紹下，男鹿和雄加入了[[吉卜力工作室|吉卜力工作室]]。在吉卜力工作室男鹿和雄首次擔當的事務是負責由[[宮崎駿|宮崎駿]]執導動畫《[[龍貓|龍貓]]》中的森林風景繪製，當時男鹿和雄所畫出自然風景皆獲得宮崎駿與[[高畑勳|高畑勳]]的好評<ref>{{Cite book |script-title=ja:映画を作りながら考えたことII |location=日本 |publisher=[[德間書店|德間書店]] |author=[[高畑勳|高畑勳]] |date=1999 |pages=165頁  |ISBN=4-19-861047-9 |accessdate=2012-09-05 }}</ref><ref>{{Cite web |url=http://www.ntv.co.jp/oga/about/ogakazuotoha.html#miryoku1 |script-title=ja:男鹿和雄とは |publisher=[[日本電視台|日本電視台]] |date=2006-07-28 |accessdate=2012-09-13 }}</ref>。

之後男鹿和雄參與了吉卜力數部動畫的美術、背景事務，並在2006年發表了第一部個人執導的動畫《[[種山原之夜|種山原之夜]]》<ref>{{Cite web |url=http://www.ghibli.jp/taneyamagahara/00info/000832.html#more |script-title=ja:角館イベント・ミニレポート |publisher=[[吉卜力工作室|吉卜力工作室]] |date=2006-07-28 |accessdate=2012-09-13 }}</ref>，以及在2007至2010年期間在日本[[東京都|東京都]]、[[愛知縣|愛知縣]]等九個縣市巡迴展開個人作品美術展<ref>{{Cite web |url=http://www.ntv.co.jp/oga/jyunkaiten/index.html |script-title=ja:ジブリの絵職人・男鹿和雄展 |publisher=日本電視台 |date=2005-06 |accessdate=2012-09-13 }}</ref>。

男鹿和雄現今以位於[[八王子市|八王子市]]自宅內的畫室作為主要工作點，除仍持續接手吉卜力動畫事務外，也曾與吉卜力另一人員[[武重洋二|武重洋二]]參與由[[細田守|細田守]]執導的《[[跳躍吧！時空少女|跳躍吧！時空少女]]》、《[[夏日大作戰|夏日大作戰]]》等動畫作品的美術背景繪製。
