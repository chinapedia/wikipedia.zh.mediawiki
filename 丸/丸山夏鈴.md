'''丸山夏鈴'''（{{jp|j=丸山 夏鈴|hg=まるやま かりん|rm=Maruyama Karin|lead=yes|extra1={{bd|1993年|8月2日|2015年|5月22日}}}}）是一名[[福島縣|福島縣]][[郡山市|郡山市]]出身的[[日本偶像|日本偶像]]、[[模特兒|模特兒]]與[[歌手|歌手]]，生前所屬[[經紀公司|經紀公司]]為{{lang|en|Happy Strike}}。丸山自小以成為[[日本偶像|偶像]]為目標，但在小學二年級時就已經被發現[[人腦|大腦]]內長有[[腦腫瘤|惡性腫瘤]]，自此往往需要頻繁出入[[醫院|醫院]]<ref name=sjt/>。然而丸山一直沒有放棄自己的夢想，不僅在住院期間參與多個藝能選拔活動<ref name=totte/><ref name=ITG/>，在高中時更當上了[[學生會|學生會]]會長，熱心為學校服務<ref name=shoshi/>。

丸山於2012年通過了選拔，開始偶像活動，此時的她已經歷四次開腦[[手術|手術]]。其後丸山腦內的腫瘤不斷復發，在進行額外四次手術後還是未能根治，翌年[[癌細胞|癌細胞]]更擴散至[[肺部|肺部]]，肺功能衰退<ref name=ITG2/>。即使如此，丸山依然戴上氧氣管錄製歌曲，並於2015年2月在[[群眾募資|群眾募資]]的協助下正式[[出道|出道]]，推出第一張[[單曲|單曲]]《{{lang|en|Eternal Summer}}》<ref name=natalie/>。丸山的事務所曾計劃在募資金額達200萬[[日圓|日圓]]後製作第二張單曲，而群眾的捐款最後也達到230多萬<ref name=cf/>，惟丸山的病情在2015年3月惡化，並於同年5月22日因[[肺癌|肺癌]]於家鄉[[郡山市|郡山市]]病逝，終年21歲。而這也令丸山的出道單曲《{{lang|en|Eternal Summer}}》成為她唯一的音樂作品。

== 經歷 ==
=== 小學時代：發現腫瘤 ===
丸山夏鈴在1993年出生於[[福島縣|福島縣]][[郡山市|郡山市]]。她在小學時期開始迷上了「[[早安少女組。|早安少女組。]]」，並希望長大後能成為像她們般的[[日本偶像|日本偶像]]<ref name=tokudane>{{Cite web |language=ja |script-title=ja:がんと闘ったアイドル丸山夏鈴さん死去　48本の動画に刻んだ思い｜WEBとくダネ！〜とくダネです。 |url=http://blog.fujitv.co.jp/tokudane-official/E20150525002.html |archiveurl=https://archive.is/20150526121423/http://blog.fujitv.co.jp/tokudane-official/E20150525002.html |archivedate=2015-05-26 |website=WEBとくダネ |publisher=[[富士電視台|テレビフジ]] |accessdate=2015-08-13 |date=2015-05-25 |deadurl=yes }}</ref>。然而在小學二年級的[[冬天|冬天]]，丸山在老師面前忽然暈倒，體育課[[跳繩|跳繩]]時也時常感到不適，又出現[[發燒|發燒]]和[[噁心|噁心]]等病徵，但那時丸山以為只是[[感冒|感冒]]而沒有理會，後來詳細檢查才發現[[人腦|大腦]]內出現[[腦腫瘤|惡性腫瘤]]，需要進行腫瘤摘除手術<ref name=sjt>{{Cite web|language=ja|publisher=丸山夏鈴の「かりんの夢への階段」|url=http://ameblo.jp/pukarin-cho/entry-11734905673.html |script-title=ja:手術から一週間 |accessdate=2015-05-23 |author=丸山夏鈴 |archiveurl=https://web.archive.org/web/20150701023130/http://ameblo.jp/pukarin-cho/entry-11734905673.html|archivedate=2015-07-01|date=2013-12-23 }}</ref>。留院期間，丸山依然沒有放棄當偶像的夢想，還在醫院裏一間學校的院友面前跳舞<ref name=totte>{{Cite web|language=ja|script-title=ja:私にとってアイドルは・・・|publisher=丸山夏鈴の「かりんの夢への階段」|url=http://ameblo.jp/pukarin-cho/entry-11097676709.html|accessdate=2015-06-03|archiveurl=https://web.archive.org/web/20150701043156/http://ameblo.jp/pukarin-cho/entry-11097676709.html|archivedate=2015-07-01|date=2011-12-05}}</ref>。

=== 中學和高中時代 ===
丸山在中學二年級再次病發，需要入院進行第二次手術。住院期間，她看過[[Perfume|Perfume]]的影片，並十分敬佩她們為了出道而作出的多年努力，更表示這段時間是[[Perfume|Perfume]]教導了她努力和奮鬥。到高中時代，丸山開始積極參與各種活動，如[[義工|義工]]服務、社團活動（{{lang|ja|部活}}），甚至當上了學校第46任[[學生會|學生會]]會長<ref name=shoshi>{{Cite web|script-title=ja:￼丸山夏鈴さんが来ました！！|url=http://www2.shoshi.ed.jp/news/2014.12.17_cosmos_2.pdf.pdf|archiveurl=https://web.archive.org/web/20150714035426/http://www2.shoshi.ed.jp/news/2014.12.17_cosmos_2.pdf.pdf|archivedate=2015-07-14|publisher=尚志高等学校生徒会|accessdate=2015-06-03|date=2014-12-17|language=ja}}</ref>。丸山在學生會任期內十分忙碌，工作時也受到不少壓力和批評<ref name=ITG/>。有一次，她在音樂節目《[[MUSIC_STATION|MUSIC STATION]]》中偶然看見《[[裙襬飄飄|裙襬飄飄]]》一曲，隨即對[[AKB48|AKB48]]產生了興趣，並參加了其選拔，最後在第三次審查中落選。然而她被[[前田敦子|前田敦子]]於[[AKB48第22張單曲選拔總選舉|AKB48第22張單曲選拔總選舉]]上的發言所感動，決意繼續為學生會的職務奮鬥。<ref name=totte/>

在丸山逝世前一小時，其母曾於[[Twitter|Twitter]]上批評Happy Strike事務所的社長下村誠沒有遵守約定，將早前籌款活動中募集回來的捐款轉達給她們，又無視丸山向他傳送的電郵，覺得「籌款活動的善意夏鈴一日圓也收不到，實在太可憐了」<ref name=nispo/>，催促社長交代<ref>{{Cite web |language=zh-cn |title=日本女星丸山夏玲去世 粉丝捐款被其公司扣压 |url=http://www.chinanews.com/yl/2015/05-25/7300232.shtml |archiveurl=https://archive.is/20150727153619/http://www.chinanews.com/yl/2015/05-25/7300232.shtml |archivedate=2015-07-27 |publisher=[[中国新闻网|中国新闻网]] |accessdate=2015-07-27 |date=2015-05-25 |deadurl=yes }}</ref>。雖然下村在翌日已馬上把剩餘的703,986日圓親手交給丸山家人<ref>{{Cite web |language=ja |script-title=ja:アイドル周辺でトラブルも…クラウドファンディングの危険な落とし穴 (2) |url=http://www.tokyo-sports.co.jp/entame/entertainment/410830/2/ |publisher=《[[東京體育|東スポ]]》 |accessdate=2015-09-06 |archiveurl=https://archive.is/20150906145332/http://www.tokyo-sports.co.jp/entame/entertainment/410830/2/ |archivedate=2015-09-06 |date=2015-09-06 |deadurl=yes }}</ref>，並在一星期後於事務所網頁上公開收支報告<ref>{{Cite web |language=ja |script-title=ja:【ご報告】丸山夏鈴の活動収支経過報告（5/31付） |publisher=丸山夏鈴の「かりんの夢への階段」 |url=http://ameblo.jp/happystrike-fanclub/entry-12032453983.html |archiveurl=https://archive.is/20150601080630/http://ameblo.jp/happystrike-fanclub/entry-12032453983.html |archivedate=2015-06-01 |accessdate=2015-06-03 |date=2015-05-31 |deadurl=yes }}</ref>，惟其行動的遲鈍受到了網民的抨擊<ref name=nispo>{{Cite web |language=ja |script-title=ja:丸山夏鈴さん支援金巡り家族と事務所が意見食い違い |url=http://www.nikkansports.com/entertainment/news/1481841.html |archiveurl=https://archive.is/20150603123420/http://www.nikkansports.com/entertainment/news/1481841.html |archivedate=2015-06-03 |publisher=《[[日刊體育|日刊スポーツ]]》 |accessdate=2015-06-03 |date=2015-05-24 |deadurl=yes }}</ref>。

== 個人生活 ==
丸山家中養有一隻名叫「櫻桃」（{{lang|ja|チェリー}}）的[[芝娃娃|-{zh-hans:吉娃娃;zh-hant:芝娃娃;zh-tw:吉娃娃}-]]，而喜歡的動物是[[哈氏異康吉鰻|花園鰻]]<ref>{{Cite web |language=ja |script-title=ja:丸山夏鈴の読み方やプロフィールやひとりごと動画について調べてみる。 |url=http://trendboxs.com/maruyamakarin-profile-hitorigotodouga/ |archiveurl=https://archive.is/20150813071011/http://trendboxs.com/maruyamakarin-profile-hitorigotodouga/ |archivedate=2015-08-13 |publisher=トレンドボックス |accessdate=2015-08-13 |date=2015-05-23 |deadurl=yes }}</ref>。由於丸山在江戶川大學屬影像研究部（{{lang|ja|映像研究部}}）和傳播媒體學科（{{lang|ja|マスコミュニケーション学科}}）<ref>{{cite web|script-title=ja:学園祭で追悼ライブ　丸山夏鈴さんしのぶ　がんと闘いながらアイドル活動　明るく楽しいステージに　２日から、江戸川大学|url=http://www.chibanippo.co.jp/news/local/286119|publisher=千葉日報|accessdate=2016-02-29|language=ja|date=2015-10-31}}</ref>，她喜歡使用[[Adobe_Premiere|Adobe Premiere]]與{{link-ja|AviUtl|AviUtl}}等程式剪輯影片和製作[[三維|3D]]模型，而她在首次單獨現場演唱活動中更播放了自己拍攝和製作的影片<ref>{{Cite web |language=ja |script-title=ja:ハッピーストライク丸山夏鈴さんが肺がんで死去〜脳腫瘍の手術を繰り返す〜 |url=http://arigato-jpn.com/news/maruyama |archiveurl=https://archive.is/20150813071326/http://arigato-jpn.com/news/maruyama |archivedate=2015-08-13 |publisher=エンタメNOW!! |accessdate=2015-08-13 |date=2015-05-22 |deadurl=yes }}</ref>。雖然丸山自小已多次接受手術，但她討厭別人跟她談論有關疾病的話題、或是說一些可憐她的話，因為她認為這會令氣氛變得沉重<ref>{{cite journal|author1={{link-ja|吉田豪|吉田豪}}|script-title=ja:ぶらり豪|journal=《{{link-ja|BEAUTIFUL Lady & TELEVISION|BEAUTIFUL Lady & TELEVISION}}》2013年10月号|date=2013-08-24|language=ja}}</ref>。
