'''會津若松市'''（{{jpn|j=会津若松市|hg=あいづわかまつし|rm=Aizuwakamatsu shi|f=y}}）是[[日本|日本]][[福島縣|福島縣]]西部（[[會津|會津]]地方）的都市，也是會津地方的中心城市。在[[江戶時代|江戶時代]]這里是[[会津藩|会津藩]]的[[城下町|城下町]]。由于在二戰中免于盟軍空襲等因素，這裡遺留了不少古蹟，成為了豐富的觀光資源。也因為是日本自古以來的兵家必爭之地，因此被稱為「武士之鄉」，<ref>http://t.samurai-city.jp/</ref>目前以會津若松市為中心的會津若松都市圈的人口約有19萬人。

== 氣候 ==
由於城市中心位於盆地，年溫差大（1月平均氣溫0℃，8月平均氣溫 25℃）。通常冬季會降雪，這一地區已被指定為大雪地區,也充滿了日本海的氣候，而氣候的特點是遠離大海的內陸。年平均氣溫為 11.4℃，年平均降雨量為1133.8毫米。

== 人口 ==
* 推計人口 : 127,759人 (2009年10月1日)

== 觀光 ==
會津若松車站是會津鐵道以及JR東日本的磐越西線與只見線的匯合處，可在此處轉車前往[[喜多方市|喜多方市]]、[[豬苗代湖|豬苗代湖]]區域與[[只見線|只見線]]、[[南會津|南會津]]等區域，因此多數外地旅客會將會津若松作為其中繼的住宿與遊玩地點<ref>http://www.aizutetsudo.jp/</ref><ref>http://www.jreast.co.jp/</ref>。另外會津若松觀光當局推出的「會津周遊卡」（{{lang|ja|会津ぐるっとカード}}）<ref>http://www.aizukanko.com/kk/aizucard/</ref>，可在兩日內無限次搭乘指定區間內的鐵路、巴士、市內觀光巴士前往會津若松以及其鄰近區域，一張售價為2,700日圓。

===寺院===
由於會津藩是[[戊辰戰爭|戊辰戰爭]]中舊幕府勢力（{{link-ja|佐幕|佐幕|佐幕派}}）的主力，會津若松市境內許多寺院也成了在內戰中陣亡的將士最後的安葬地。
* {{link-ja|天寧寺 (會津若松市)|天寧寺 (会津若松市)|天寧寺}}：戊辰戰爭敗戰後[[新選組|新選組]]局長[[近藤勇|近藤勇]]遭斬首，部分的遺體葬於此地。
* {{link-ja|阿彌陀寺 (會津若松市)|阿弥陀寺 (会津若松市)|阿彌陀寺}}：新選組三番隊隊長[[齋藤一|齋藤一]]埋葬於此。
* {{link-ja|興德寺 (會津若松市)|興徳寺 (会津若松市)|興德寺}}：[[蒲生氏鄉|蒲生氏鄉]]之墓所在地。
* {{link-ja|八葉寺|八葉寺}}：本堂阿彌陀堂是日本國家指定[[重要文化財|重要文化財]]。
* {{link-ja|延命寺 (會津若松市)|延命寺 (会津若松市)|延命寺}}：境內的地藏堂（又稱「藤倉二階堂」）也是日本國定重要文化財。
