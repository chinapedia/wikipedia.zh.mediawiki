'''窦易直'''（{{bd|？||833年|4月29日|catIdx=D}}<ref>[[:s:舊唐書/卷17下|《旧唐书》卷一十七下]]</ref>），[[表字|字]]'''宗玄'''，谥号'''晋阳恭惠公'''，[[中国|中国]][[唐朝|唐朝]][[官员|官员]]，[[唐敬宗|唐敬宗]]和[[唐文宗|唐文宗]]年间为[[宰相|宰相]]。

== 家世 ==
窦易直生年不详，[[京兆|京兆]]始平人。<ref name=BT167>[[:s:舊唐書/卷167|《旧唐书》卷一百六十七]]</ref><ref name=NBT151>[[:s:新唐書/卷151|《新唐书》卷一百五十一]]</ref>家族认祖[[汉朝|汉朝]][[皇后|皇后]][[窦妙|窦妙]]父[[大将军_(东亚)|大将军]][[窦武|窦武]]。祖父窦元昌，[[彭州|彭州]][[九泷|九泷]]县令。父窦彧，[[庐州|庐州]][[刺史|刺史]]。<ref>[http://www.sidneyluo.net/a/a17/071b.htm 《新唐书》卷七十一]</ref>

窦易直举[[明经|明经]]，为[[秘书省|秘书省]]校书郎，此后十年不应辟，以考试书判入等，授[[蓝田|蓝田]]尉。曾在华阴令任上被升为左拾遗，士大夫闻讯，都高兴地说“窦易直被提拔了，正直之道可以推行了！”<ref>[[:s:送窦拾遗赴朝廷序|梁肃《送窦拾遗赴朝廷序》]]</ref>历任右司、兵部、吏部三郎中。<ref name=BT167/><ref name=NBT151/>

== 唐宪宗年间 ==
[[元和_(唐朝)|元和]]六年（811年）六月，迁[[御史中丞|御史中丞]]。谢日，赐绯鱼袋。<ref>[[:s:舊唐書/卷14|《旧唐书》卷一十四]]</ref>因病被免，八年（813年），迁[[给事中|给事中]]。<ref>[[:s:窦易直可给事中制|:s:窦易直可给事中制]]</ref>九月，出为陕虢都防御观察使，仍赐金紫。入为[[京兆尹|京兆尹]]。[[万年|万年]]尉[[韩晤|韩晤]]贪赃事发，窦易直命曹官[[韦正晤|韦正晤]]问讯，查出案值达三十万。[[唐宪宗|唐宪宗]]认为案值不够，下诏重查，查出案值达三百万。<ref name=BT167/>因此，十二年（817年），窦易直贬为[[金州|金州]]刺史，韦正晤长流[[昭州|昭州]]。十三年（818年）六月，迁[[宣州|宣州]]刺史、宣歙池都团练观察等使。<ref name=BT167/>次年五月，为[[润州|润州]]刺史，充[[浙西观察使|浙西观察使]]。<ref name=NBT151/><ref name=BT15>[[:s:舊唐書/卷15|《旧唐书》卷一十五]]</ref>

== 唐穆宗年间 ==
[[唐穆宗|唐穆宗]][[长庆|长庆]]二年（822年）七月，[[宣武军|宣武军]]兵变，逐[[节度使|节度使]][[李愿|李愿]]。窦易直闻变，担忧本部士兵也会如法炮制。他想以金帛赏赐军士，但有人对他说：“赏赐无名，将士反而更加生疑。”窦易直因而没有这么做，但是军士已经知道他想赏赐金帛了。时江、淮旱灾，漕物淹积不能前运，州大将[[王国清|王国清]]因而以取漕货激众军谋乱，九月，事泄，窦易直械王国清下狱，王国清党羽数千入狱劫之，想大行剽掠。窦易直登楼发令：“能诛乱者的人，一个首级赏千万！”众军喜，擒王国清及其党羽二百余人（《两唐书》本传皆作三百余人），窦易直皆杀之。<ref>[[:s:資治通鑑/卷242|《资治通鉴》卷二百四十二]]</ref>但窦易直为赏赐众军，倾其府藏，军队更加骄气，财物也告竭。同月，窦易直被[[李德裕|李德裕]]所代，<ref name=BT174>[[:s:舊唐書/卷174|《旧唐书》卷一百七十四]]</ref><ref>[[:s:新唐書/卷180|《新唐书》卷一百八十]]</ref>回京为吏部侍郎。十一月，<ref>《旧唐书·穆宗纪》作十二月。</ref>改户部侍郎，兼[[御史大夫|御史大夫]]，判度支。<ref name=BT167/><ref name=NBT151/><ref name=BT16>[[:s:舊唐書/卷16|《旧唐书》卷一十六]]</ref>

== 唐敬宗年间 ==
长庆四年（824年）正月唐穆宗驾崩，唐敬宗登基后不久，五月窦易直即因宰相[[李逢吉|李逢吉]]推荐，以朝议郎、守尚书户部侍郎、兼御史大夫、判度支、[[上柱国|上柱国]]、赐紫金鱼袋身份被任为朝散大夫、守尚书户部侍郎、判度支，与吏部侍郎[[李程|李程]]一同被授[[同中书门下平章事|同中书门下平章事]]，拜为宰相。<ref name=NBT8>[[:s:新唐書/卷8|《新唐书》卷八]]</ref><ref name=ZZTJ243>[[:s:資治通鑑/卷243|《资治通鉴》卷二百四十三]]</ref><ref>[[:s:授窦易直平章事制|:s:授窦易直平章事制]]</ref>改[[门下侍郎|门下侍郎]]。辞让判度支，七月，给其三月俸禄，罢判度支。<ref name=BT167/>十一月封'''晋阳郡公'''，<ref name=NBT151/>食邑三千户。<ref name=BT17-1>[[:s:舊唐書/卷17上|《旧唐书》卷一十七上]]</ref>

先前[[鄠县|鄠县]]令[[崔发|崔发]]因越权逮捕殴打百姓的五坊使而获罪，遭品官五十余人击打伤了脸落了牙齿，遇赦也不被免，李逢吉、窦易直、李程为崔发求情，提到崔发母是故宰相[[韦贯之|韦贯之]]的姐姐，年已八十，为崔发入狱积忧成疾，希望敬宗基于孝道恩赦崔发。敬宗先前不知崔发有老母，闻言感到同情，遂释放崔发并抚慰其母。<ref name=BT171>[[:s:舊唐書/卷171|《旧唐书》卷一百七十一]]</ref>

[[宝历_(唐朝)|宝历]]元年（825年）四月，敬宗被群臣上尊号文武大圣广孝皇帝，发布赦文，窦易直以摄[[侍中|侍中]]读宝官门下侍郎平章事的身份被赐一子出身。<ref>[[:s:受尊號赦文_(李湛)|《受尊号赦文》]]</ref>

窦易直以避免表现[[裙带关系|裙带关系]]闻名，后来[[开成|开成]]二年（837年）唐文宗对当时的宰相们评价他：“你们推荐人的时候不要在意亲疏。朕听闻窦易直为相时，不用亲人故人，如果这些人有才而因为避嫌而遭他弃用，这也不是至公之举。”<ref name=NBT151/><ref name=BT173>[[:s:舊唐書/卷173|《旧唐书》卷一百七十三]]</ref><ref name=NBT182>[[:s:新唐書/卷182|《新唐书》卷一百八十二]]</ref><ref>[[:s:資治通鑑/卷245|《资治通鉴》卷二百四十五]]</ref>

== 唐文宗年间 ==
宝历二年（826年）十二月，敬宗遇弑，文宗继位。<ref name=ZZTJ243/>[[大和_(唐朝)|大和]]二年（828年）十月，窦易直在尚书右仆射任上罢相，<ref name=NBT8/>改[[检校|检校]][[左仆射|左仆射]]（《新唐书》作检校[[尚书|尚书]][[右仆射|右仆射]]）、[[襄州_(西魏)|襄州]]刺史、充[[山南东道节度使|山南东道节度使]]、临汉监牧等使，仍保有同中书门下平章事为荣衔。<ref name=BT17-1/>四年（830年），宰相[[王播|王播]]去世，[[裴度|裴度]]病重，由[[刘禹锡|刘禹锡]]代为上表，提到文宗登基时的四位宰相已故两人（[[韦处厚|韦处厚]]、王播），自己又病重时日无多，只有出镇的窦易直安然无恙，可见权位难居，请求辞官。<ref>[[:s:為裴相公讓官第一表|:s:為裴相公讓官第一表]]</ref>五年（831年），窦易直入为左仆射，判[[太常卿|太常卿]]事。当年宰相[[宋申锡|宋申锡]]被神策军军虞候豆卢著诬告勾结皇弟漳王[[李凑|李凑]]谋反，窦易直认为宋申锡罪当诛，闻者不赞同。<ref>[[:s:新唐書/卷152|《新唐书》卷一百五十二]]</ref>十一月，任[[凤翔|凤翔]]尹、[[凤翔陇右节度使|凤翔陇右节度使]]，加检校[[司空|司空]]。六年（832年），因病请求回京，获准。七年（833年）四月卒，<ref name=BT17-2>[[:s:舊唐書/卷17下|《旧唐书》卷一十七下]]</ref>赠[[司徒|司徒]]，谥'''恭惠'''。<ref name=BT167/><ref name=NBT151/>

== 评价 ==
*韦处厚：窦易直良厚，忠事先朝，陛下（指唐文宗）固当委信。<ref>[[:s:舊唐書/卷159|《旧唐书》卷一百五十九]]</ref>/窦易直长厚忠实，经事先帝，陛下所宜亲重委信之。<ref>[[:s:新唐书/卷142|《新唐书》卷一百四十二]]</ref>
*窦易直入仕十多年，常居散官，不应请辟；担任节度使时，也以公廉闻名。在相位，未尝论用亲党，但在公事上举荐人时则无所避嫌。然而元和年间，吏部尚书[[郑余庆|郑余庆]]曾著《仆射上仪》议仆射上日仪制，不与隔品官抗礼。时任御史中丞的窦易直奏驳郑余庆所议，<ref>[[:s:详议仆射答拜仪注奏|窦易直《详议仆射答拜仪注奏》]]</ref>但等自己做了左仆射，却行隔品致敬之礼，被时论非议鄙视嘲笑。<ref name=BT167/><ref>[[:s:新唐书/卷131|《新唐书》卷一百三十一]]</ref><ref>[[:s:新唐書/卷181|《新唐书》卷一百八十一]]</ref><ref>[[:s:仆射上仪依三公奏|陈夷行《仆射上仪依三公奏》]]</ref>
*窦易直曾在边境增加使者、官吏，因此被[[金献贞|金献贞]]上书劝谏。<ref>[[:s:谏窦易直|金献贞《谏窦易直》]]</ref>
*唐文宗曾对[[李珏|李珏]]说“窦易直劝我，凡是宰相启奏拟议的，五个取三个，两个取一人。他应该劝我选择宰相，不应该劝我怀疑宰相。”开成四年（839年），当时已是宰相的李珏重提此事，文宗说：“易直此言甚鄙。”<ref name=BT173/>/“易直此言殊可鄙。”<ref name=NBT182/>
*[[北宋|北宋]]判太常礼仪院赵昂、判考功张洎：窦易直之公举无避，乃谥曰“恭惠”。<ref>[[:s:宋史/卷264|《宋史》卷二百六十四]]</ref>

== 子孙 ==
