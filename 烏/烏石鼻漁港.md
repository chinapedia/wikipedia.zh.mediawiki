'''烏石鼻漁港'''為一座位於[[臺灣|臺灣]][[臺東縣|臺東縣]][[長濱鄉|長濱鄉]]的漁[[港|港]]，距離北方同在長濱鄉的[[長濱漁港|長濱漁港]]約8公里。目前在[[行政院|行政院]][[農委會|農委會]][[漁業署|漁業署]]新制的[[漁港法|漁港法]]之中，被歸類在第二類漁港中，由新港區[[漁會|漁會]]進行管理。

烏石鼻漁港興建在烏石鼻的[[火成岩|火成岩]]地形上，目前設籍[[漁船|漁船]]數8艘，漁港捕撈漁獲以[[龍蝦|龍蝦]]、[[白帶魚|白帶魚]]等近海魚種為主，在2009年（民國98年）時曾名列在漁業署所舉辦的十大魅力漁港之中。<ref name="行政院海岸巡防署東部地區巡防局 烏石鼻安檢所"/><ref name="臺東縣政府 臺東觀光網 烏石鼻漁港"/>

烏石鼻漁港因無防坡堤之設置，故港區泊地穩靜度極差，如將漁船安置在烏石鼻漁港碼頭邊，會因風浪關係讓漁船衝撞[[碼頭|碼頭]][[混凝土|混凝土]]構造物，因此，當地漁民都將漁船停至烏石鼻內海區，利用混凝土塊沉至內海區，以作為[[錨|錨]]供漁船停靠，再從港區內的曳船道搭乘小型竹筏至內海泊地上的漁船出海。<ref name="臺東縣政府 臺東觀光網 烏石鼻漁港">{{cite web |url=http://tour.taitung.gov.tw/zh-tw/Travel/ScenicSpot/355/%E7%83%8F%E7%9F%B3%E9%BC%BB |title=臺東縣政府 臺東觀光網 烏石鼻漁港 | accessdate=2015-02-23}}</ref>

== 沿革 ==
烏石鼻漁港最早興建於1975年（民國64年），當時僅有曳船道一座以供當地旦曼與寧埔地區的漁筏起卸使用，後陸續興建有碼頭、聯外道路陸上設備等相關設施。

目前烏石鼻漁港尚無任何增建或改善計畫。

== 設施 ==
烏石鼻漁港目前有曳船道一座以及[[行政院|行政院]][[海岸巡防署|海岸巡防署]][[東部地區巡防局|東部地區巡防局]]所設立的安檢所，名為'''烏石鼻安檢所'''，隸屬在'''岸巡八二大隊'''旗下。<ref name="行政院海岸巡防署東部地區巡防局 烏石鼻安檢所">{{cite web |url=http://www.cga.gov.tw/GipOpen/wSite/ct?xItem=48123&ctNode=6223&mp=9995 |title=行政院海岸巡防署東部地區巡防局 烏石鼻安檢所 | accessdate=2015-02-23}}</ref>
