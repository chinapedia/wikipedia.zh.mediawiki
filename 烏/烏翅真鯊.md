'''烏翅真鯊'''（[[學名|學名]]：''{{lang|la|Carcharhinus melanopterus}}''），又名'''污翅白眼鮫'''、'''黑翼鯊'''、'''伯爵鯊'''、'''黑鰭鯊'''或'''黑鰭礁鯊'''，是一種生活在熱帶及溫和[[海洋|海洋]]的[[鯊魚|鯊魚]]。牠經常被誤會為[[黑邊鰭真鯊|黑邊鰭真鯊]]（學名''Carcharhinus limbatus''）。

== 分佈 ==
烏翅真鯊是現今在印度、太平洋及[[加勒比海|加勒比海]][[珊瑚礁|珊瑚礁]]附近淺水區（有時會在30厘米深的淺水區）最常見的[[鯊魚|鯊魚]]之一。牠們生活在20℃至27℃水溫的[[海洋|海洋]]，並且不會冒險進入一些遠離海洋的熱帶湖或河流中。

== 外表 ==
[[File:Carcharhinus_melanopterus_Luc_Viatour.jpg|thumb]]
就如牠的名字所言，烏翅真鯊的胸鰭及背鰭的頂端是[[黑色|黑色]]的，而下部是[[白色|白色]]。牠們身體上半部的皮是呈[[褐色|褐色]]。最大的烏翅真鯊可達3.5米長。牠的前端是呈圓鈍的。一種稱為[[灰珊瑚鯊|灰珊瑚鯊]]的鯊魚外形與牠很相似，但從其較粗狀及[[灰色|灰色]]的身體及沒有黑端的背鰭，就能分辨牠們。由於烏翅真鯊體型較小，所以並不認為是[[人類|人類]]的威脅。

== 攝食 ==
烏翅真鯊主要獵食珊瑚魚，但牠們也會捕獵[[鱘魚|鱘魚]]及[[烏魚|烏魚]]。

== 生殖、行為及與人類的接觸 ==
[[File:Snorkeler_with_blacktip_reef_shark.jpg|thumb]]
烏翅真鯊是胎生的，每胎約有2至4頭小鯊。母鯊的懷胎期為16個月。出生的小鯊約為33至52厘米長。

烏翅真鯊並非群居的，但卻有時會一小群出沒。一般烏翅真鯊都是較害羞的，但卻會對潛水員及[[水肺潛水|水肺潛水員]]感到好奇。與其他[[鯊魚|鯊魚]]無異，當牠們感覺到危險時，會把身體捲曲成S型。除非是感到憤怒，否則牠們是無害的。一些情況如用手餵食或以矛捕魚，加上低視野，都會令烏翅真鯊感到憤怒。

烏翅真鯊是少有能跳躍出水面的鯊魚，這種行為稱為「[[躍身擊浪|躍身擊浪]]」。牠們亦會在海面徘徊。

[[File:Blacktip_reef_shark.jpg|thumb]]

== 數量下降 ==
烏翅真鯊有時會被漁業混獲及丟棄。牠們的數量，與其他[[鯊魚|鯊魚]]一樣亦在下降。由於牠們的鰭會作為珍貴的[[魚翅|魚翅]]，牠們的鰭在被捕獲時會被切去，而鯊魚本身會被丟回海中等待死亡。這可能是烏翅真鯊數量減少的原因。
