'''烏鱧'''（[[學名|學名]]：''{{lang|la|Channa argus}}''），又名'''北方蛇頭魚'''、'''斑魚'''、'''黑魚'''、'''生鱼'''、'''烏魚'''、'''烏棒'''、'''蛇頭魚'''、'''文魚'''和'''才魚'''，属[[鱧科|鱧科]][[鱧屬|鱧屬]]，在中国是一种常见的食用鱼。

== 體型 ==
[[File:Channa_argus_after_berg_1933.jpg|thumb]]. Retrieved 2007-[[07-15|07-15]].</ref>]]
烏鱧有[[雄性|雄性]]大於[[雌性|雌性]]的現象，一般成年烏鱧的體長約在40至70[[公分|公分]]之間，不過有些大型烏鱧的體長長達80公分以上，目前為止發現最大的烏鱧有接近1.5[[公尺|公尺]]之長（也有人說是1.2公尺），重達15磅（7[[公斤|公斤]]左右），據說是在[[俄羅斯|俄羅斯]]被發現的。

== 特徵 ==
[[File:Channa_argus_3.JPG|thumb]]
烏鳢的[[口|口]]部延伸至[[眼|眼]]部後方，長長的背鰭大約有49至50個[[鰭|鰭]]刺，而臀鰭則有31至32個鰭刺。烏鳢的體色是其最明顯的特徵，一般大多為棕色，不過有少數部位可能為[[金色|金色]]或蒼[[白色|白色]]，烏鳢身體兩側的斑點呈現鞍狀排列。

== 經濟價值 ==
[[File:Channa_argus_7.JPG|thumb]]
烏鳢在[[韓國|韓國]]、[[中華人民共和國|中華人民共和國]]都被視為一種珍贵的食用魚，廣東人尤喜加入[[西洋菜|西洋菜]]、[[陳皮|陳皮]]、[[鴨|鴨]][[腎|腎]]等作湯；贵州人用番茄做辅料炖制而成的[[酸汤鱼|酸汤鱼]]是贵州名菜；大韓民國在1992年時出產了500[[公噸|公噸]]左右的烏鳢，但烏鳢卻未成為[[地理|地理]]位置相近的[[日本|日本]]市場中十分普遍的魚種，而在其他地區，如[[鹹海|鹹海]]和[[塔拉河|塔拉河]]，每年都各會有1－5公噸和10公噸的產量，目前[[台灣|台灣]]某些[[餐廳|餐廳]]也引入了此種作為[[食用魚|食用魚]]，與台灣其他原生一樣可當作食用魚的[[鱧屬|鱧魚]]([[斑鱧|斑鱧]]（''Channa maculata''）、[[七星鱧|七星鱧]]（''Channa asiatica''}}）)有競爭的意味。

== 習性 ==
牠們有鰓,也有輔助呼吸器官,只要有水氣就可以在旱災時躲在泥中度過。

=== 生存環境 ===

烏鳢分布于[[印度|印度]]、东南亚至[[俄罗斯远东|俄罗斯远东]]地区.朝鲜,日本以及各大水系等，常栖息于湖泊、河流及沟渠中以及常潜伏在水草茂盛处，喜好生存於底部為泥土、有水生植物與水流緩慢平靜的[[湖泊|湖泊]]、[[河川|河川]]、[[池塘|池塘]]、[[水庫|水庫]]以及[[運河|運河]]等水域。该物种的模式产地在[[舟山群岛|舟山群岛]]。<ref name='id8426'/>

=== 哺育與成長 ===
[[File:Channa_argus_8.JPG|thumb]]
烏鳢的幼魚由兩隻成魚守護，並會把卵放置於60至80公分深的巢穴。烏鳢的在孵化時的第一天，長度大約是5.7毫米；在3天後則會長至7.3毫米，並且開始用魚鰓呼吸，血管會明顯增少，而胸鰭會變大；兩星期後，身長會達到1.1公分長；而1.8公分之後，就會開始食用小[[甲殼纲|甲殼纲]][[動物|動物]]以及更為幼小的魚；直到長至4到4.5公分時，才會有斑點出現。成魚守護时会攻击游近的动物。

=== 食性 ===
烏鳢是肉食性動物，並且食量非常大，食物種類有[[蜻蜓|蜻蜓]]幼蟲、[[蟑螂|蟑螂]]、[[甲蟲|甲蟲]]、[[小龍蝦|小龍蝦]]（螯蝦）、[[蛙類|蛙類]]、[[魴|魴]]、[[鯉魚|鯉魚]]以及其他淡水魚等等。可吃下體型一半的魚。
尤其喜欢在水面捕食。

== 入侵物种 ==
作為食用魚，烏鳢曾在[[美國|美國]]，[[紐約市|紐約]]、[[休士頓|休士頓]]、[[奧蘭多_(佛羅里達州)|奧蘭多]]、[[聖路易斯_(密蘇里州)|聖路易]]等城市有大量販賣，但因有部份烏鳢流出，在野外大量繁衍，成为入侵物种，对当地生態造成破壞，2007年7月29日，美國當局禁止了烏鳢的進口。

== 大眾文化 ==
烏鳢在2004年被拍成《[[恐怖食人魚|恐怖食人魚]]》（''Snakehead Terror''）以及《[[科學怪魚|科學怪魚]]》（''Frankenfish''）兩部[[電影|電影]]，故事中是以烏鳢對生態的破壞為背景，不過《科學怪魚》劇中的烏鳢經基因改造體型龐大，而《恐怖食人魚》則是寫說烏鳢被加入[[赫爾蒙|赫爾蒙]]異常巨大。

==經濟利用==
可作為食用魚及觀賞魚。
