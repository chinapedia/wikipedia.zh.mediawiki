'''烏布蘇湖'''（{{Lang-mn|'''Увс Нуур'''}}；{{Lang-tyv|'''Успа-Холь'''}}；{{Lang-ru|'''Убсу-Нур'''}}）屬[[鹹水湖|鹹水湖]]，主体位于[[蒙古国|蒙古国]][[乌布苏省|乌布苏省]]，東北部屬[[俄羅斯联邦|俄羅斯联邦]][[圖瓦共和國|圖瓦共和國]]。平均海拔 759米，面積 3,350 平方公里，是[[蒙古国|蒙古国]]水域面积最大的[[湖泊|湖泊]]<ref name="medeelel">{{cite web

==地理==
烏布蘇湖长84公里，宽79公里，平均深度6米。注入该湖的主要河流是从[[杭爱山|杭爱山]]流向东方的[[图伦河|图伦河]]（[[:en:Turuun_River|Baruunturuun]]）、纳尔河（Nariin gol）、[[特斯河|特斯河]]（[[:en:Tes_River|Tes]]）， 以及从西边的[[阿尔泰山|阿尔泰山]]流过来的[[哈尔黑拉河|哈尔黑拉河]]（[[:en:Kharkhiraa_River|Kharkhiraa River]]）以及桑格河（Sangil gol）。<ref name="davies">{{cite web

烏布蘇湖所在的[[乌布苏盆地|乌布苏盆地]]是一个[[内流盆地|内流盆地]]<ref name="medeelel" />，是[[亞歐大陸|亞歐大陸]]保持得最好的[[溫帶草原|溫帶草原]]景觀。乌布苏盆地中尚有其他較小的湖泊，最著名的有海拔1,450米的[[乌列格湖|乌列格湖]]（Ureg Lake）。

烏布蘇湖所在的[[乌布苏盆地|乌布苏盆地]]于2003年被[[聯合國教科文組織|聯合國教科文組織]]列入[[世界遺產|世界遺產]]名錄<ref name="unesco">{{cite web

==生态环境==
这片面积庞大的集水区，由于没有出口，所以成为高盐度水域（盐度为千分之18.8，是海水盐度的1.5倍），盐度主要来自[[硫酸钠|硫酸钠]]离子。

该湖每年10月到次年5月为冰期。夏季，该湖水面温度为25℃，水底温度为19℃<ref name="davies" />。
