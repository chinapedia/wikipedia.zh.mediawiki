'''乌隆他尼府'''（{{thai|t=จังหวัดอุดรธานี|r=Changwat Udon Thani}}，{{IPA-th|t͡ɕāŋ.wàt ʔù.dɔ̄ːn tʰāː.nīː}}），简称'''乌隆府'''<ref name="zhou">{{cite book|title=[[世界地名翻译大辞典|世界地名翻译大辞典]]|author=周定国|publisher=中国对外翻译出版公司|year=2008|isbn=978-7-5001-0753-8|page=1235}}</ref>（{{thai|t=อุดร|r=Udon}}，{{IPA-th|ʔù.dɔ̄ːn}}），是[[泰国|泰国]]的一个府，属于[[依善地区|东北地区]]，其人口数为1,467,158（2000年），土地面积为11,730平方公里，乌隆府的首府为[[乌隆他尼|乌隆]]（Udon Thani）。

== 历史 ==
该府原名“莫肯村”，于1893年，国王御命亲王少将把在廊开府的指挥部迁至莫肯村。以后更名为“乌隆城”，于1907年晋格为乌隆府。

== 地理 ==
面积约为11,730平方公里，距曼谷约为564公里。其疆域北临[[廊开府|廊开府]]；南连[[孔敬府|孔敬府]]和[[加拉信府|加拉信府]]；东界[[沙功那空府|沙功那空府]]；西邻[[農磨蘭普府|農磨蘭普府]]。

== 行政 ==
乌隆府行政区共分为16个县（''Amphoe''）和2个次级县（''King Amphoe''），下分155区(tambon)、1682 村(muban)。

== 名胜古迹 ==
* 廊巴乍湖：位于直辖县，为一大型湖泊，除作为繁殖各种淡水鱼外，尚作为休闲娱乐的公园。
* 乌隆兰花园：位于直辖县，有各类品种的兰花圃，并有花费十年时间而繁殖成功的新品种，园内环境优美。
* 目曼那卡村和曼堂村：位于直辖县，出产著名织布制品，花纹鲜艳夺目，物廉价美。

==广播理事会乌隆府短波中继站==
[[美国|美国]][[广播理事会|广播理事会]]（BBG）在乌隆府设有[[短波|短波]]中继站，有7部500千瓦的短波发射机，主要转播[[美国之音|美国之音]]、[[自由欧洲电台|自由欧洲电台]]、Radio Farda、Radio Aap Ki Dunyaa 和[[泰国广播电台国际台|泰国广播电台国际台]]的节目。
