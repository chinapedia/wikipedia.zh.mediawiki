《'''烏龍麵之國的金色毛毬'''》是[[日本|日本]][[漫畫家|漫畫家]][[筱丸和|筱丸和]]（{{lang|ja|篠丸のどか}}）創作的[[日本漫畫|漫畫]]作品，[[新潮社|新潮社]]出版，於新潮社漫畫雜誌《[[月刊Comic@Bunch|月刊Comic@Bunch]]》2012年8月號開始連載<ref>{{cite web|url=http://www.shinchosha.co.jp/bunch/backnumber/20120621/|script-title=ja:月刊コミック＠バンチ　2012年8月号|publisher=[[新潮社|新潮社]]|accessdate=2016年11月3日|language=ja}}</ref>，是以日本[[香川縣|香川縣]]著名兩大「特產」──[[烏龍麵|烏龍麵]]與[[貍貓|貍貓]]（妖怪）為主要舞台的日常系漫畫。

== 故事簡介 ==
俵宗太是個住在東京的網頁設計師，在自家的烏龍麵店發現一名奇怪的小孩，而這個孩子真面目是狸貓妖怪。以香川縣為舞台，兩名角色開始了同居生活。

該漫畫作品是以香川縣著名的烏龍麵與貍貓為主的故事。

: 與宗太約定，不在他人前露出耳朵和尾巴，避免遭人發現；但過於疲憊或懈怠或睡覺或生病時，還是會崩露馬腳。能夠變化外，也能幻現人類過去的經歷片段。

; {{nihongo|中島忍|中島 忍（なかじま しのぶ）|聲：[[杉田智和|杉田智和]]<ref name="登場人物"/>、[[嶋村侑|嶋村侑]]（幼少期）；台灣：[[何志威|何志威]]}}

; {{nihongo|大石凜子|大石 凛子（おおいし りんこ）|聲：[[中原麻衣|中原麻衣]]<ref name="登場人物"/>；台灣：詹雅菁}}
: 俵宗太的姊姊。已婚後住在高知縣。原姓「俵」，結婚後改姓「大石」。
: 少時即期盼長大成為稱職的母親，懷孕後在母親忌日時返家掃墓家祭稟告。母親過世後一度希望撐起家務，但廚藝始終不佳，最後反而是宗太的廚藝更好。

: 僧侶。提醒宗太此地自古盛有狸妖之事。會從事舞台DJ的工作。喜好狸貓，大量收集狸貓相關物品、書籍。
; {{nihongo|藤山紗枝|藤山 紗枝（ふじやま さえ）|聲：[[花澤香菜|花澤香菜]]<ref name="登場人物"/>；台灣：[[鄭郁欣|鄭郁欣]]}}

; {{nihongo|濱田吾郎|浜田 吾郎（はまだ ごろう）|聲：[[黑田崇矢|黑田崇矢]]<ref name="登場人物"/>；台灣：何志威}}
: 宗太在東京的老闆，重視屬下，按才提拔並富有人情味。。宗太中學腳傷時在病床間巧遇宗太與中島在鄰床學習主頁製作，乃教導宗太並邀他至東京招待遊玩。宗太離開香川後，在其東京的公司服務。相當器重宗太，雖不太情願卻願意讓宗太拿長假在香川照顧波可，在宗太呈辭後還替他在香川介紹友人處的工作機會。
; {{nihongo|永妻宏司|永妻 宏司（ながつま ひろし）|聲：[[立花慎之介|立花慎之介]]<ref name="登場人物"/>；台灣：[[鈕凱暘|鈕凱暘]]}}
: 宗太的後輩。對宗太相當敬仰。一度發現波可的耳朵和尾巴。怕黑卻能夠立刻在眼睛蒙上後睡去。體力驚人，能自東京騎腳車至小豆島。小豆島上愛上宗太的姐姐，旋得知為人妻而失戀。
: 情緒不太鎮靜。同事覺得他是怪人，打扮、行為怪異，惟宗太接受他。
; {{nihongo|田中舞|田中 舞（たなか まい）|聲：[[皆口裕子|皆口裕子]]<ref name="登場人物"/>；台灣：[[石采薇|石采薇]]}}
: 宗太暗戀的初戀。宗太為波可添裝時於童裝店相遇，乃與宗太成為媽媽之友。
: 已有二孩子。結婚後改姓「田中」，但宗太還是習慣叫原姓「真鍋」。
; {{nihongo|田中望|田中 のぞみ（たなか のぞみ）|聲：[[本渡楓|本渡楓]]<ref name="登場人物"/>；台灣：[[鄭郁欣|鄭郁欣]]}}
: 舞的長女。以為波可是女生。兩者衝撞後復交好，贈送波可珠珠花形的髮飾。較波可年長，待波可如弟弟。
; {{nihongo|冴木學|冴木 学（さえき まなぶ）|聲：[[鳥海浩輔|鳥海浩輔]]；台灣：林正騏}}

: 傳說中的創意總監。掀起該動畫世界中虛構動畫角色「嘎嗷嘎嗷」熱潮的人物。
