'''央视网'''（'''CCTV.COM'''），2016前称为'''中国网络电视台'''（{{lang-en|China Network Television}}，[[縮寫|縮寫]]：{{lang|en|CNTV}}），[[中国中央电视台|中国中央电视台]]下辖的新媒体平台。它提供所有中国中央电视台频道與中國大陸各省級衛視和地面频道的视频直播和节目点播，亦是CCTV的官方网站，在中华人民共和国境外的用户也可以進入該网站收看、阅览所有中国中央电视台视频直播和新闻资讯。<ref>{{cite web|title=中国网-{}-络电视台今日正式开播|url=http://it.sohu.com/20091228/n269250120.shtml|publisher=sohu|accessdate=2014-07-13}}</ref>

==站点历程==
{{Refimprove section|time=2013-12-13T05:52:46+00:00}}
[[Image:CNTV.jpg|right]][[Image:CNTV_new_logo.png|right]]
* 1996年12月，CCTV.COM建立并试运行，是中国最早发布中文信息的网站之一；
* 2000年10月，CCTV.COM被列入中央重点新闻网站，成为新闻媒体网站的国家队和主力军；
* 2001年5月25日，中央电视台成立网络宣传部，将网站纳入节目宣传部门。自此，CCTV. COM建设全面推进，并迅速成长为有影响力的网络媒体；
* 2006年4月28日，中央电视台正式成立网络传播中心和央视国际网络有限公司，央视国际（CCTV. COM）同时实现全新改版；
* 2007年12月18日，CCTV. COM正式与[[国际奥委会|国际奥委会]]签约，成为中国大陆和澳门地区[[2008北京奥运|2008北京奥运]]唯一官方互联网/移动平台转播机构。
* 2010年7月1日，央视网全面并入中国网络电视台。<ref>{{cite web|title=新闻|url=http://www.dayoo.com/roll/201008/04/10000307_103123614.htm|accessdate=2014-07-13|deadurl=yes|archiveurl=https://web.archive.org/web/20140714153913/http://www.dayoo.com/roll/201008/04/10000307_103123614.htm|archivedate=2014-07-14}}</ref>
* 2010年7月5日，央视网与中国网络电视台实现业务归并，www.cctv.com和www.cntv.cn域名共同指向中国网络电视台首页。
* 2017年1月1日，CNTV.COM中国网络电视台网站改回CCTV.COM央视网域名，其央视节目点播亦变回了CCTV.COM标志。事实上，现在的CNTV已经转型为IPTV和OTT业务服务的“中国-{互}-联网电视”（“爱上传媒CHINA IPTV”和“NewTV未来电视”均为央视新媒体下属单位）。
* 2017年2月，央视设立“CCTV央视专区”，面向全国各地广电网络有线电视互动平台，目前已在年内完成全国大多地区的对接<ref>[http://www.sarft.gov.cn/art/2016/6/30/art_114_31150.html 中央电视台正式上线有线互动电视“央视专区”]</ref>。并于2018年2月全线上线面向4K有线互动电视的“央视专区”<ref>{{cite web |url = https://bp.imaschina.com/show/8/26411.html |title=CCBN2018：央视4K超高清频道2018年10月1日开播 |author=依马狮广电网 |website =依马狮广电网}}</ref>。
* 2017年12月31日，央视网后缀为@cntv.cn邮箱服务将正式关停。
* 2019年2月19日，央视网全面改版。与此同时，同属于[[中央广播电视总台|中央广播电视总台]]的央广网（[[中央人民广播电台|中央人民广播电台]]官网）、[[国际在线|国际在线]]（[[中国国际广播电台|中国国际广播电台]]官网）也一起改版，三家网站改版后风格大致相同。<ref>{{cite web|title=中央广播电视总台“全国县级融媒体智慧平台”暨央视网新版全终端正式上线|url=http://news.cctv.com/2019/02/19/ARTIejmGGarMgT9G5pEmvn8l190219.shtml?spm=C96370.PsikHJQ1ICOX.Em32AuyOHUeL.25|accessdate=2019-02-19}}</ref>

==央視搜索==
央視搜索是央視網于2010年1月14日發布的[[搜索引擎|搜索引擎]]服務。起初，央視搜索提供“[[视频|视频]]”、“[[网页|网页]]”、“[[新闻|新闻]]”、“[[图片|图片]]”、“[[博客|博客]]”五項搜索服務。<ref>{{Cite web |url=http://tech.sina.com.cn/i/2010-01-14/18523772205.shtml |title=央视网悄然测试独立搜索功能 |accessdate=2010-01-14 |work=新浪科技 |publisher=新浪科技 |date=2010-01-14 }}</ref>目前，央视搜索仅支持视频和网页搜索。<ref>[https://search.cctv.com/index.php 央视搜索]</ref>

==IP电视服务==
央视国际网络子公司[[爱上传媒|爱上传媒]]系央视IPTV总平台，为全国31省市三大运营商IPTV提供央视频道直播、节目轮播、点播等信息服务。前身为“'''CCTV-IP电视'''”。
