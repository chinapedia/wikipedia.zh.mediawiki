'''哌替啶盐酸盐'''（Pethidine），別名'''杜冷丁'''、度冷丁、唛啶、配西汀、地美露（Demerol）、盐酸哌替啶。为白色、无嗅、结晶状的粉末，能溶于水，一般制成针剂的形式。用作[[麻醉药|麻醉药]]。

== 作用 ==
作为人工合成的麻醉药物，哌替啶普遍地使用于[[临床|临床]]，它对人体的作用和机理与吗啡相似，但镇痛、麻醉作用较小，仅相当于吗啡的1／10－1／8，作用时间维持2－4小时左右。主要作用于中枢神经系统，对心血管、平滑肌亦有一定影响。毒副作用也相应较小，恶心、呕吐、便秘等症状均较轻微，对呼吸系统的抑制作用较弱，一般不会出现呼吸困难及过量使用等问题。

== 細節 ==
主要治療在鎮痛方面，目前列為管制藥品，相對的禁忌如下：

* 哌替啶對正在使用或近期內曾使用[[单胺氧化酶|单胺氧化酶]]（MAO）抑制剂者為使用禁忌。
* 14天內服用MAO抑制劑者，如給予治療劑量之哌替啶會引起不可預期的、嚴重的、甚至致命的反應，其症狀包括昏迷、嚴重呼吸抑制、[[低血壓|低血壓]]、及類似急性[[麻醉|麻醉]]藥品過量之症狀。其他的明顯反應有過動、全身痙攣、心跳加速、高熱、[[高血壓|高血壓]]。
* 嚴重反應者可[[靜脈注射|靜脈注射]][[氢化可的松|氢化可的松]]或[[强的松龙|强的松龙]]（prednisolone），對[[高血壓|高血壓]]及高熱者可併用靜脈注射[[氯丙嗪|氯丙嗪]]。

== 合成 ==
由[[环氧乙烷|环氧乙烷]]与[[甲胺|甲胺]]加成为''N''-甲基[[二乙醇胺|二乙醇胺]]，用[[氯化亚砜|氯化亚砜]]在[[苯|苯]]中[[氯化|氯化]]为二氯代衍生物，与[[苯乙腈|苯乙腈]]在[[氨基钠|氨基钠]]存在下缩合为1-甲基-4-苯基-4-氰基[[哌啶|哌啶]]，水解为[[羧酸|羧酸]]，再与[[乙醇|乙醇]][[酯化|酯化]]为乙酯，最后与[[盐酸|盐酸]]成盐而得。

== 注意事項 ==
* 有嗜眠、眩暈等現象。服用期間如須駕車、從事操作機械等危險性工作者，應特別小心。
* 對於某些老年人或虛弱、嚴重肝或腎功能不全、[[甲狀腺|甲狀腺]]機能不足、[[阿狄孫氏症|阿狄孫氏症]]、及[[前列腺|前列腺]]肥大或尿道狹窄患者，給予哌替啶時應小心，同時起始劑量應予減低。
* 對頭部傷害、顱內傷害、顱內壓已上升者，給予哌替啶會加重呼吸抑制及腦[[脊髓|脊髓]]液壓明顯升高。
* [[氣喘|氣喘]]及其他呼吸異常者使用哌替啶應格外小心。
* 哌替啶會產生類似[[嗎啡|嗎啡]]之藥物依賴性，而有濫用之虞。

== 常見副作用 ==
過量使用：
* [[嗜睡|嗜睡]]
* [[噁心|噁心]]
* 嘔吐
* 呼吸抑制
* [[便秘|便秘]]
* 尿滯留
* [[瞳孔|瞳孔]]縮小

非過量使用：偶爾有[[頭暈|頭暈]]目眩、[[噁心|噁心]]嘔吐、[[便秘|便秘]]的情形

==合成法==
哌替啶可以在兩步合成中製備。第一步是[[苯乙腈|苯乙腈]]和[[氯甲烷|氯甲烷]]的反應在[[氨基钠|氨基钠]]存在下形成[[哌啶|哌啶]]環。然後將[[腈|腈]]轉化成[[酯|酯]]。<ref>Patent Appl. DE 679 281 IG Farben 1937.</ref>
