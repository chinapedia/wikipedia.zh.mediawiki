[[Image:Saffron_gatherersSantorini-3.jpg|thumb]][[壁畫|壁畫]]，[[愛琴海|愛琴海]][[聖托里尼|聖托里尼]]島嶼[[亞克羅提利_(聖托里尼)|亞克羅提利]]的出土物。]]

人類栽種與使用'''番紅花的歷史'''，可以追溯到三千五百年前<ref name="Deo_1">{{Harvnb|Deo|2003|p=1}}。</ref><ref name="Hogan"/>，橫跨多個文化、大陸與文明。[[番紅花|番紅花]]，一種以[[番紅花|番紅花]]（''Crocus sativus''）的乾燥[[雌蕊|柱頭]]製成的[[香料|香料]]，自古一直是世界上最奢華的物質之一。有著略帶苦味的口感，似乾草的[[氣味|氣味]]，與微微金屬般的光澤，番紅花常用作[[調味料|食材調味]]、[[香水|香水]]、[[染料|染色]]與[[藥物|醫藥]]。番紅花土產自[[西南亞|西南亞]]<ref name="Grigg_287">{{Harvnb|Grigg|1974|p=287}}。</ref><ref name="Hill_272">{{Harvnb|Hill|2001|p=272}}。</ref>，於[[希臘|希臘]]首次種植。<ref name="McGee_422">{{Harvnb|McGee|2004|p=422}}。</ref>

現今種植的番紅花源自野生的[[卡萊番紅花|卡萊番紅花]]（''Crocus cartwrightianus''）。人工栽種者藉由選擇突變的長柱頭來[[植物育種|繁殖]]卡萊番紅花。例如，[[青銅器時代|青銅器時代]]末期[[克里特島|克里特島]]，出現的卡萊番紅花與番紅花的突變種<ref name="Goyns_1">{{Harvnb|Goyns|1999|p=1}}。</ref>。番紅花最早記載於西元前七世紀的[[亞述|亞述]]，[[亞述巴尼拔|亞述巴尼拔]]所匯編的植物文獻中。自此以後，便陸續揭露了四千年以來，番紅花可治療約莫九十種疾病等[[番紅花貿易和用途|用途]]的記載<ref name="Honan">{{Harvnb|Honan|2004}}。</ref>。番紅花緩慢的傳播、遍及整個[[歐亞大陸|歐亞大陸]]，而後並到達[[北非|北非]]、[[北美|北美]]與[[大洋洲|大洋洲]]的部分地區。

== 希臘羅馬時期 ==

[[Image:Cueilleuse_de_safran,_fresque,_Akrotiri,_Grèce.jpg|thumb]]古[[米諾斯文明|米諾斯]]居地裡，許多描繪有關番紅花的壁畫的其中之一。]]

番紅花在[[希臘羅馬|希臘羅馬]]古典時期（西元前八世紀至西元後三世紀）扮演著重要的角色<ref name = "Willard_41">{{Harvnb|William|2001|p=41}}</ref>。然而，希臘文化的番紅花其實起源於[[青銅器時代|青銅器時代]]。[[克諾索斯|克諾索斯]]宮殿的[[米諾斯文明|米諾斯]][[克里特島|克里特]]的壁畫記載了番紅花的收成<ref name="Hogan">{{Harvnb|Hogan|2007|p=3}}。</ref>，描繪著年輕女子與[[猴子|猴子]]採集番紅花花朵的情景。其中一個壁畫遺跡位於希臘[[聖托里尼|聖托里尼]]島（古希臘時稱做錫拉島 (Thera)），[[亞克羅提利_(聖托里尼)|亞克羅提利]]的[[賽斯特三號|賽斯特三號]]（"Xeste 3"）建築。賽斯特三號壁畫始於西元前1600–1500年間<ref name = "Honan" />，另有各種其他的年代說，如西元前3000–1100年間<ref name = "Ferrence"/>，以及西元前十七世紀等<ref name="Dalby_2002_124">{{Harvnb|Dalby|2002|p=124}}。</ref>。在他們的畫中，[[希臘神話|希臘女神]]監督著花朵的采摘與柱頭的挑選，以製成治療的藥材<ref name = "Ferrence"/>。同一址出土的另一壁畫也做了一名婦女以番紅花治癒其出血的腳的描畫<ref name="Honan"/>。這些錫拉島壁畫是最早紀錄了將番紅花用作草藥的精密植物學畫像<ref name="Ferrence">{{Harvnb|Ferrence|2004}}。</ref>。米諾斯在聖托里尼島亞克羅提利種植番紅花的居地，最後於西元前1645年至1500年間，被強大地震與併發的[[米諾斯火山爆發|火山爆發]]摧毀<ref name="Willard_37">{{Harvnb|Willard|2001|p=37}}。</ref>。[[火山灰|火山灰]]掩埋並保存了番紅花壁畫。<ref name="Willard_37-38">{{Harvnb|Willard|2001|pp=37–38}}。</ref>

根據古希臘傳說敘述，曾有一群粗獷的水手登船、經過了艱辛危險的航海，抵達了遙遠的[[奇里乞亞|奇里乞亞]]，為了取得他們心目中最有價值的番紅花<ref name="Willard_2-3">{{Harvnb|Willard|2001|pp=2–3}}.</ref>。與番紅花有關的希臘傳說中，最有名的則是番紅花與[[菝葜|牛尾菜]]的悲劇故事：英俊青年番紅花開始追求雅典附近樹林裡的[[寧芙|木精靈]]牛尾菜。在一段短暫的田園詩般的戀情中，牛尾菜對番紅花的殷勤感到心滿意足，但很快的轉變為厭煩。番紅花持續的求愛，牛尾菜於是蠱惑番紅花，施法讓他變成一朵番紅花朵，有著亮橙色的柱頭，做為番紅花對牛尾菜不朽熱情的渺小象徵<ref name = "Willard_2-3"/>。[[奧維德|奧維德]]後來回憶起此悲劇和香料時寫道：<ref name = "Willard_1"/>

[[Image:Man_gathering_saffron_Knossos_Crete_crocus_sativus_fresco.jpg|thumb]][[克諾索斯|克諾索斯]]米諾斯壁畫不精確的重建品<ref>Hood, S. (1978) ''The Arts in Prehistoric Greece'', Yale University Press, pp. 48-49; Platon, N. (1947) ''Kritika Chronika'', pp. 505-506</ref>，描繪一個男人（應是一隻猴子）採收番紅花的情景。]]

:{{quote|width=300px|"Crocus and Smilax may be turn'd to flow'rs,<br />And the Curetes spring from bounteous show'rs<br />I pass a hundred legends stale, as these,<br />And with sweet novelty your taste to please"<br />     — 奧維德，《[[變形記|變形記]]》<ref name="Willard_1">{{Harvnb|Willard|2001|p=1}}.</ref>}}

對古代地中海一帶的人們而言，奇里乞亞的沿海城市[[所羅易|所羅易]]採集來的番紅花最有價值，尤其是用於[[香水|香水]]和藥膏<ref name = "Willard_41"/>。然而，[[希羅多德|希羅多德]]與[[老普林尼|老普林尼]]等名人則認為產自[[肥沃月彎|肥沃月彎]]、[[亞述|亞述]]與[[巴比倫|巴比倫]]的番紅花，才是用於治療腸胃與腎臟疾病的最上等良藥<ref name = "Willard_41"/>。產於[[寇里坑恩洞穴|寇里坑恩洞穴]]與[[巴納塞斯山|巴納塞斯山]]的希臘番紅花同樣也變得知名。奇里乞亞番紅花的顏色，在[[羅得島的阿波羅尼奧斯|羅得島的阿波羅尼奧斯]]的《[[阿爾戈船英雄記|阿爾戈船英雄記]]》中被用來作為比喻<ref>[http://www.gutenberg.org/dirs/etext97/1argn10.txt Argonautica of Apollonius Rhodius] at Project Gutenberg.</ref>，而其香氣則出現在[[馬提雅爾|馬提雅爾]]的《[[雋語|雋語]]》中。<ref>{{cite web|url=http://books.google.com/books?id=gl5T47CvuDsC&dq=crocus+and+(korykian%7Ccorycian%7Ckorikian)&source=gbs_summary_s&cad=0|title=Ancient Rome in so many words|author=Christopher Francese|year=2007|page=162}}</ref>
