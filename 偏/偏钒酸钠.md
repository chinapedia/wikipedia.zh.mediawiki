'''偏钒酸钠'''是一种[[无机化合物|无机化合物]]，化学式为NaVO<sub>3</sub>。

== 性质 ==

无色棱柱状[[单斜|单斜]]结晶，溶于水。在自然界以少见的[[付穆水钒钠石|付穆水钒钠石]]（metamunirite，无水物）和[[水钒钠矿|水钒钠矿]]（munirite，二水物）存在。

== 制备 ==

由[[五氧化二钒|五氧化二钒]]溶于[[氢氧化钠|氢氧化钠]]溶液中，经浓缩结晶而得。

: <math>\rm \ V_2O_5+2NaOH\rightarrow 2NaVO_3+H_2O</math>

== 用途 ==

用于化肥、照相、煤气脱硫，也用作媒染剂等。
