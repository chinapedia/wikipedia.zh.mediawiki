'''玉蕈离褶伞'''（[[学名|学名]]：''{{lang|la|Lyophyllum shimeji}}''），又名'''占地菇'''、'''本占地菇'''、'''本菇'''、'''九月菇'''，为[[口蘑科|口蘑科]][[离褶伞属|离褶伞属]]的[[蕈类|蕈类]]，日文称'''本占地'''、'''大黒占地'''，分布於[[日本|日本]]以及中国大陆的[[云南|云南]]等地，仅分布於[[东亚|东亚]]。该物种的模式产地为日本。<ref name="shimeji"/>

== 特征 ==
菌体高2－8厘米，开始为馒头半球形或斗笠形，成熟後展开变平为菌盖，菌盖直径2－12.5厘米，不黏且平滑，中部稍突起或凹陷，颜色为灰褐色或淡灰色，上有白色斑纹，盖缘平滑内卷。菌柄長3－8厘米，直径一般为1.2－2.5厘米，最粗可达3.2厘米，白色，实心，下部酒坛或腰鼓状膨大。肉质緻密，白色肥厚，美味。[[孢子|孢子]]直径4－6微米，无色，光滑球形。子实层内及其他部位无囊状体。玉蕈离褶伞为外生[[菌根真菌|菌根真菌]]，秋季9月－11月初生於阔叶林中，[[子实体|子实体]]生长在林地上，单生或群生。<ref name="shimeji">{{cite journal |author=刘培贵 |year=1993 |month= |title=介绍一种滇产珍贵食用菌—玉蕈离褶伞 |journal=中国食用菌 |volume=12|issue=2 |pages=29 |url=http://www.cqvip.com/qk/90633X/199302/1246040.html |accessdate= 2009-08-05 |id={{ISSN|1003-8310}} |language=zh-hans}}</ref>

玉蕈离褶伞在日本一般生於[[枹栎|枹栎]]（''Quercus serrata''）和[[赤松|赤松]]林中，在中国首次发现是於云南[[景谷|景谷]]的[[染木树|染木树]]（''Saprosma ternatum''）和[[梨树|梨树]]林的腐烂枝叶中<ref name="shimeji"/>。

从菌株的研究可得知，菌丝生長温度为5－32℃，最適温度为25－26℃。子实体发育温度为15℃左右。<ref name="jpo" />

== 食用 ==
日本有一句著名的俗语，称“[[松茸|松茸]]香，占地美”（{{lang|ja|香り松茸、味占地}}），这表明玉蕈离褶伞的味道十分鲜美，味道鲜美胜过松茸，是稀少且高級的食用菌，在[[日本清汤|日本清汤]]（{{lang|la|すまし汁}}）、[[蘑菇饭|蘑菇饭]]（{{lang|la|キノコご飯}}）、[[奶油炖菜|奶油炖菜]]中都有使用，烤食尤其味美，被日本人认为是食用菌的珍品<ref name="shimeji"/>。占地的鲜味来自其富含[[鲜味|鲜味]]物质[[单磷酸鸟苷|单磷酸鸟苷]]、[[谷氨酸|谷氨酸]]和[[天冬氨酸|天冬氨酸]]<ref>{{ja}}{{cite web|url=http://www.hyoei.ac.jp/cooking/column_n/colum72.html |title=日本人はキノコが大好き |accessdate=2009-08-06 |work=Hyoeiオリジナルコラム（なでしこ通信） |publisher=兵庫栄養調理製菓専門学校 |deadurl=yes |archiveurl=https://web.archive.org/web/20071213100021/http://www.hyoei.ac.jp/cooking/column_n/colum72.html |archivedate=2007-12-13 }}</ref>。天然的玉蕈离褶伞非常珍贵，如在2007年1千克的价钱可高达10000－20000日元，是松茸价格的数倍。

== 名称误用 ==
在日语中，玉蕈离褶伞与[[真姬菇|真姬菇]]（''Hypsizigus tessellatus''，异名''Hypsizygus marmoreus''）均被称为“占地”（{{lang|la|シメジ}}），这造成了名称上的混淆。

== 人工栽培 ==
=== 菌床栽培 ===
玉蕈离褶伞是一种不易栽培且稀有的[[菌根真菌|菌根真菌]]，但仍可利用菌床进行人工栽培<ref>{{ja}}{{cite journal |author=寺下隆夫 |coauthors=永井勝、坂井拓夫 |title=菌根形成食用担子菌, ホンシメジおよびマツタケの子実体形成研究の現状 |journal=近畿大学農学部紀要 |volume=34|issue= |pages=95－105 |url=http://ci.nii.ac.jp/naid/110001044440/ |accessdate= 2009-08-06 |date=2001年3月}}</ref>。1999年[[寶生物工程株式會社|寶生物工程株式會社]]（TaKaRa Bio）<ref>{{ja}}[http://agribio.takara-bio.co.jp/kinoko/kinoko.html 寶生物工程株式會社（TaKaRa Bio）]</ref>发现在特殊情况下一部分菌根真菌的菌株在特殊情况下会分解[[淀粉|淀粉]]，之後用[[大麦|大麦]]等[[穀物|穀物]]作为主要成分，成功利用菌絲瓶法人工栽培玉蕈离褶伞<ref name="jpo">{{ja}}{{cite web |url=http://www.jpo.go.jp/shiryou/s_sonota/hyoujun_gijutsu/kinoko/2-2-1.pdf#1 |title=きのこ栽培方法 2-2-1-1 ホンシメジ（''Lyophyllum shimeji''） |accessdate=2009-08-06 |date=2007-03-14 |work=きのこの栽培方法 |publisher=日本經濟產業省特許廳 |deadurl=yes |archiveurl=https://web.archive.org/web/20110323084447/http://www.jpo.go.jp/shiryou/s_sonota/hyoujun_gijutsu/kinoko/2-2-1.pdf#1 |archivedate=2011-03-23 }}[http://www.jpo.go.jp/shiryou/s_sonota/hyoujun_gijutsu/kinoko/mokuji.htm 目录] {{webarchive|url=https://web.archive.org/web/20090310173234/http://www.jpo.go.jp/shiryou/s_sonota/hyoujun_gijutsu/kinoko/mokuji.htm |date=2009-03-10 }}</ref>。菌丝生长需要足够的氧和铁。单糖也可用作除[[淀粉|淀粉]]外的碳源。最适生長[[pH|pH]]为5.4左右<ref name="jpo"/>。

=== 林地栽培 ===
玉蕈离褶伞需要树龄15－25年的枹栎和赤松林，环境为小乔木和草地，落葉需清除。为获得更高的生长率，可将菌丝定植在未生长玉蕈离褶伞的苗木上，子实体成熟後菌根就可固定。

=== 盆钵栽培 ===
剥开赤松树皮，然後用[[泥炭藓属|泥炭藓]]（''Sphagnum''）包裹使其长出菌根，然後将菌丝块移植入[[花盆|花盆]]中使子实体发育。
