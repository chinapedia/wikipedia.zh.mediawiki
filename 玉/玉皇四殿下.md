'''玉皇四殿下'''為一台灣的道教神祇，[[玉皇上帝|玉皇上帝]]四子，亦稱'''玉皇四太子'''，聖誕為每年農曆五月初二<ref>{{Cite web|url=http://www.kaigi.com.tw/story.htm|title=台南開基玉皇宮|accessdate=2018-08-02|work=www.kaigi.com.tw}}</ref>。

== 由來 ==

台灣的玉皇四殿下信仰來自泉州，但因祖廟早已於文革時期毀壞，其信仰起源目前尚不可考。根據[[台灣|台灣]][[台南市|台南市]][[開基玉皇宮|開基玉皇宮]]沿革的紀載，[[明朝|明朝]][[天啟_(明)|天啟]]四年（1624年），有一[[福建|福建]][[泉州府|泉州府]]商人-{欲}-至台經商，與同伴至鄉廟恭請[[玉皇上帝|玉皇上帝]]、[[玉皇三公主|玉皇三公主]]娘的香火，與玉皇四太子殿下木雕神像來台南赤崁，將香火與神像供奉於尖山頂，並建一天壇，為台灣供奉玉皇上帝之始<ref>{{Cite web|url=http://www.yuhlonggong.com/index.php?option=com_content&view=article&id=99:znchin1&catid=42:2011-01-07-05-52-5|title=玉皇四殿下聖紀|accessdate=2018-08-02|author=|date=|last=玉欽居士|work=www.yuhlonggong.com|publisher=|language=zh-tw}}</ref>。明[[永曆_(南明)|永曆]]二十四年（1670年）正式建廟，廟名為「玉皇太子宮」，又稱四舍廟，奉玉皇四太子為主神，為全台最先供奉四殿下的祖廟<ref>{{Cite web|url=https://www.moi.gov.tw/files/civil_download_file/15.%E5%8F%B0%E5%8D%97%E5%B8%82-99%E5%B9%B4%E8%AA%BF%E6%9F%A5%E5%AF%BA%E9%99%A2%E5%AE%AE%E5%BB%9F%E5%9F%BA%E6%9C%AC%E8%B3%87%E6%96%99.pdf|title=全國寺院宮廟基本資料調查表}}</ref>。<br>[[台灣清治時期|清]][[嘉慶|嘉慶]]五年（1800年）地震，廟身年久失修，於是大行修廟並雕塑玉皇上帝、玉皇四殿下、玉皇三公主之神像，此時才正式改名[[開基玉皇宮|開基玉皇宮]]，並尊奉玉皇上帝為主神，同年玉皇四太子被改稱為玉皇四殿下<ref>{{Cite web|url=http://crgis.rchss.sinica.edu.tw/temples/TainanCity/north/2104050-KJYHG|title=開基玉皇宮|accessdate=2018-08-07|author=謝奇峰|work=文化資源地理資訊系統|language=zh-tw}}</ref>。

== 信仰 ==

玉皇四殿下為開基玉皇宮出巡遶境的主神。開基玉皇宮二樓左龕獨立奉祀的玉皇四殿下前，對聯寫著「殿前不讓妖魔祟，下界難容惡詐逞。」，可見玉皇四殿下在信仰中扮演避妖保民的角色。另根據《中國方志叢書：台南文化新刊》的記載，昔玉皇太子廟，因太子年少時喜好戲水，海濱漁民也期望祂能保護戲水的兒童<ref>{{Cite book|url=https://books.google.com.tw/books?id=t170AAAAMAAJ&q=%E7%8E%89%E7%9A%87%E5%9B%9B%E5%A4%AA%E5%AD%90&dq=%E7%8E%89%E7%9A%87%E5%9B%9B%E5%A4%AA%E5%AD%90&hl=zh-TW&sa=X&ved=0ahUKEwiCivzhnczcAhUIA4gKHdZ-C4YQ6AEIUjAH|title=中國方志叢書: 臺灣地區|last=|first=|date=1983|publisher=成文出版社|year=|isbn=|location=|pages=|language=zh-TW|chapter=}}</ref>。

玉皇四殿下的隨駕護法神分別為黃侍讀與關太尉。其中黃侍讀左手執書卷，右手執硃砂筆，陪侍玉皇四殿下校錄善惡、燮理陰陽；關太尉右手執掌彈丸，左手執弓箭，護持玉皇四殿下巡視凡間。兩位護法神聖壽皆與玉皇四殿下同祀<ref>{{Cite web |url=http://www.yuhlonggong.com/index.php?option=com_content&view=article&id=131&catid=42&Itemid=66  |title = 關太尉與黃侍讀聖紀|accessdate=2018-08-02|author=|date=|last=玉欽居士|work=www.yuhlonggong.com|publisher=|language=zh-tw }}</ref>。

開基玉皇宮沒有[[乩童|乩童]]，光復以前問事方式是以鑾轎，但僅限於玉皇四殿下，光復後則改採擲筊方式至今<ref>{{Cite web|url=https://ndltd.ncl.edu.tw/cgi-bin/gs32/gsweb.cgi?o=dnclcdr&s=id=%22099CCU00045013%22.&searchmode=basicf|title=解讀在地神明----玉皇三公主信仰研究}}</ref>。目前玉皇四殿下分靈以南部地區為多，而同樣奉祀於開基玉皇宮的玉皇三公主分靈則以北部較多。<ref>{{Cite web |url=https://www.th.gov.tw/new_site/05publish/03publishquery/02journal/01download.php?COLLECNUM=401054206  |title = 臺灣文獻季刊臺南開基玉皇宮與諸天信仰 }}</ref>

== 神蹟 ==

* 明朝永曆二十四年（1670年），[[鄭成功|鄭成功]]部將李世輝身患頑疾，經其子李世金設香案禱告後，果真痊癒。
* 玉皇四太子殿下於臺南市[[白河區|白河區]][[關子嶺|關子嶺]][[火山碧雲寺|火山天公廟]]降駕，留詩云：「玉皇陛下眾群真，皇命在深究迷群，四御天星護真人，殿前共議渡亡靈，下降人間救良民。」此乃齊頭詩，揭示「玉皇四殿下」名號<ref>{{Cite web |url=https://scontent.fkhh1-2.fna.fbcdn.net/v/t1.0-9/35414337_1688457634570162_2569508023563190272_n.jpg?_nc_cat=107&_nc_ht=scontent.fkhh1-2.fna&oh=ddb1aaccb94e3b6a3c4419a8b81a53e2&oe=5CD6F5A8  |title = 火山天公廟藏頭詩 }}</ref>。該廟虎邊牆上還留有當初四殿下奉旨巡視全台的事蹟。
* [[民國紀年|民國]]77年（1988年），玉皇四殿下於台灣[[高雄縣|高雄縣]][[六龜鄉|六龜鄉]]寶來玉皇亭（今寶來開基玉皇玉聖宮），令乩身閉關，降著仁正聖經傳世<ref>{{Cite web |url=http://www.yuhlonggong.com/daoism_scripture/scripture_5.pdf  |title = 仁正聖經全文 }}</ref>。

== 宮廟 ==

* [[台南市|台南市]][[北區_(臺南市)|北區]][[開基玉皇宮|台南開基玉皇宮]]：明永曆二十四年（1670年）創立，為台灣玉皇四殿下信仰之起源，也是台灣最古老的天公廟。<ref>{{Cite web|url=http://crgis.rchss.sinica.edu.tw/temples/TainanCity/north/2104050-KJYHG|title=開基玉皇宮|accessdate=2018-08-07|author=謝奇峰|work=文化資源地理資訊系統|language=zh-tw}}</ref>
* 台南市北區玉皇玉聖宮：民國77年（1988年）創立，為目前台灣信仰玉皇四殿下的最大宮廟。並提倡不焚化金紙與燃放鞭炮的道教信仰。<ref>{{Cite web|url=http://www.yuhlonggong.com/index.php?option=com_content&view=article&id=173&Itemid=70|title=玉皇玉聖宮沿革|accessdate=2019-01-09|author=玉欽居士|work=www.yuhlonggong.com|language=zh-tw}}</ref>
* 台南市[[中西區_(台灣)|中西區]]凌霄寶殿：民國66年（1977年）創立，主祀玉皇上帝，配祀玉皇四殿下與玉皇三公主。<ref>{{Cite web|url=http://crgis.rchss.sinica.edu.tw/temples/TainanCity/westcentral/2108025-LXBD|title=凌霄寶殿|accessdate=2019-01-09|author=謝奇峰|work=文化資源地理資訊系統|language=zh-tw}}</ref>
* 台南市[[永康區|永康區]]龍潭天壇：[[昭和|昭和]]十一年（1936年）創立，主祀玉皇四殿下、吳府三千歲。
* [[高雄市|高雄市]]六龜區寶來開基玉皇玉聖宮天公廟：民國77年（1988年）創立，主祀玉皇上帝，配祀玉皇四殿下與玉皇三公主。
* 高雄市[[小港區|小港區]]太方宮：主祀 觀音佛祖、玉皇四殿下、天官武財神、神農大帝、九天太子、黑虎將軍。
