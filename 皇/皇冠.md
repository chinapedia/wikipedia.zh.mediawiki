'''皇冠'''或'''王冠'''是世界許多地區[[君主|君主]]或[[神|神]]頭上所載的權力與正統性的[[象徵物|象徵物]]。中華文化圈各國的[[冕冠|冕冠]]與之相似。

== 類別 ==
皇冠共分為三類，詳細如下：

# [[加冕皇冠|加冕皇冠]]，加冕典禮用的皇冠，只有在君主加冕時戴上，可能是自行戴上或由[[教會|教會]][[主教|主教]]戴上。
# [[君主皇冠|帝冠]]，君主平時所戴的皇冠，代表著無上的權力。 *貴族有時也會戴著類似的寶冠，代表其身份。
# -{[[后冠|后冠]]}-，皇后所戴的皇冠，並沒有權力的意義，只是代表其身份高貴。

在某些古代國家裡，頂冠可能戴在平民頭上而非君主頭上。如古[[羅馬|羅馬]]凱旋而歸的將軍，古希臘奧運項目得勝的選手等，而這種頂冠大多是用桂葉做成的桂冠。

== 歷史 ==
[[File:Jacques-Louis_David_006.jpg|thumbnail]]
在[[拜占庭帝國|拜占庭帝國]]與[[歐洲|歐洲]]諸國的[[基督教|基督教]]傳統裡，君主的權力來自基督教會，因此新君即位有時會舉行加冕典禮，由基督教會的[[主教|主教]]為其加冕，以表示其正統性。部份[[神聖羅馬帝國|神聖羅馬帝國]]皇帝還遠赴[[義大利|義大利]][[羅馬|羅馬]]讓教宗親自為其加冕。法國的[[拿破侖|拿破侖]]亦依照此傳統，由教宗[[庇護七世|庇護七世]]加冕為[[法蘭西帝國|法蘭西帝國]]皇帝。详细請參看[[加冕皇冠|加冕皇冠]]。

歐洲君主的皇冠最初是由[[君士坦丁大帝|君士坦丁大帝]]由[[波斯|波斯]][[阿開民王朝|阿開民王朝]]引進的，被其後的羅馬的皇帝沿用。

美國[[自由女神像|自由女神像]]頭上所戴的頂冠為象征光明之{{link-en|光照冠|Radiant crown}}；[[羅得斯島巨像|羅得斯島巨像]]（古希臘太陽神[[希里歐斯|希里歐斯]]之像）頭上所戴的頂冠也可能是光照冠，代表太陽之光輝。其後大多數仍為[[異教徒|異教徒]]的羅馬皇帝在羅馬傳統的[[太陽節|太陽節]]時配戴亦戴上光照冠。

今天只有[[英國|英國]]仍沿襲君主頭戴王冠的傳統，不過很多其他歐洲國家的君主仍保留王冠作為其家族徽章。

全球很多古老民族的統治者在信史時代前已經有頭戴頂冠的習慣。而皇冠通常會由很多珠寶及黃金組合而成，不過[[美洲土著|美洲土著]]在歐洲人移民至[[美洲|美洲]]前則經常頭戴由[[麗鵑|麗鵑]][[羽毛|羽毛]]所製的頂冠，在[[波里尼西亞|波里尼西亞]]地區亦有這情種情形出現。

* 加冕通常與其他儀式一起舉行，如登基大典與基督教的[[塗油禮|塗油禮]]。

== 象徵意義 ==

皇冠通常是君主身份的象徵。然而有很多國家的君主可能從未真正戴過皇冠，皇冠的圖案可能只作為其家族的徽章，如[[比利時|比利時]]王國便沒有加冕典禮。

* [[東正教|東正教]]婚體裡有一個儀式名為加冠，即新人們被加冠為其未來家庭的''國王''與''王后''。在[[希臘|希臘]]婚礼中，頂冠通常由花朵(人工轧制或者由自然分叉的嫩枝)所製，並由新人作為紀念品保存起來。在[[斯拉夫|斯拉夫]]婚禮裡，頂冠通常由金屬所製，並會模仿真實的皇冠而製，但通常並不會在每次婚禮時新製一對，反而是由教區自行保留一對，在該教區所有新人舉行婚禮時為其戴上，因為其造價遠比希臘式的為高。

* 兒童，特別是女孩，通常會戴上[[花環|花環]]，就像是戴上-{后}-冠。

* 皇冠在宗教上通常被視為神聖的代表，舉例來說，[[黑寶冠|黑寶冠]]有時會被虔誠的宗教信徒視為神聖的模範。
