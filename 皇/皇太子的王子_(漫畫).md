《'''皇太子的王子'''》（{{lang-ko|프린스의 왕자}}，{{lang-en|''Prince of Prince''}}）是一部[[韓國|韓國]]耽美[[網絡|網絡]][[漫畫|漫畫]]，由作家Jea-Ah與畫家SE聯手製作，2014年開始於[[Naver_webtoon|Naver webtoon]]連載<ref>{{Cite web|url=http://comic.naver.com/webtoon/list.nhn?titleId=622640 |title=프린스의 왕자|publisher=[[Naver_(公司)|NAVER]]|accessdate=2015-07-05|language=ko}}</ref>。後於2015年改編成[[皇太子的王子_(電視劇)|同名電視劇]]<ref name="korea56191">{{cite web|title=FTIsland鍾訓、AOA酉奈、林允浩 出演網路劇《Prince的王子》|url=http://www.koreastardaily.com/tc/news/56191 |publisher=KPOP 韓星網|date=2015-02-06|accessdate=2015-07-05|language=zh-hant}}</ref><ref name="appguru87680">{{cite web|title=LINE TV強檔韓劇 人氣漫畫改編網路劇《皇太子的王子》首播 BL情愫+KUSO劇情超吸睛|url=http://www.appguru.com.tw/appguru/blog/87680|publisher=AppGuru|date=2015-06-17|accessdate=2015-07-05|language=zh-hant}}{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref><ref name="kbs3094165">{{Cite web|url=http://news.kbs.co.kr/news/view.do?ncd=3094165 |title=[연예] KBS 첫 웹드라마 ‘프린스의 왕자’ 오늘 드디어 TV공개 |publisher=KBS NEWS|date=2015-06-12|accessdate=2015-07-05|language=ko}}</ref><ref name="joins17974182">{{Cite web|url=http://article.joins.com/news/article/article.asp?total_id=17974182&ctg=1502 |title=신예 우혁 vs FT아일랜드 최종훈…´프린스의 왕자´ 끝판왕은 유나? |publisher=중앙일보 연예|date=2015-06-08|accessdate=2015-07-05|language=ko}}</ref><ref name="hancinema82937">{{Cite web|url=http://www.hancinema.net/upcoming-korean-web-drama-prince-of-prince-82937.html |title=Upcoming Korean Web drama "Prince of Prince" |publisher= HanCinema|date=2015-06-05|accessdate=2015-07-05|language=en}}</ref>。

== 劇情簡介 ==
身為大企業MS集團社長之子，年青有為的'''朴時賢'''甚少有機會與業務繁忙的父母見面，與他相處得最長時間的就只有妹妹'''朴尤娜'''。為了照顧及保護妹妹，他不時干涉尤娜的學業，甚至是令她厭倦的程度。某次時賢發現尤娜為了玩遊戲而不去補習班，更被親自飾演遊戲角色的遊戲製作公司老闆'''李夢龍'''深深吸引住，完全不把時賢的勸告聽進去。為了救尤娜脫離遊戲世界，時賢只好潛入該遊戲的製作公司，下定決心找出公司的機密文件並從內部摧毀，還假扮成男[[同性戀|同性戀]]者以方便接近李夢龍，結果卻與職員們發生一連串好笑的衝突。

; 朴尤娜（{{lang|ko|박유나}}，{{lang|en|Park Yu-na}}）
: 女性，朴時賢的妹妹，比他小7歲，就讀聖雅女中高二{{sfn|皇太子的王子|loc=第一季 第1話}}{{sfn|皇太子的王子|loc=第一季 第2話}}{{sfn|皇太子的王子|loc=第一季 第4話}}。本來對遊戲及動漫方面完全陌生，一次無意中接觸到遊戲「秘密的花園」，愛上了飾演其中一名遊戲角色的李夢龍，開始收集秘密的花園的周邊，放滿整個房間{{sfn|皇太子的王子|loc=第一季 第4話}}。但最愛的人是哥哥。

; 江林（{{lang|ko|강림}}，{{lang|en|Gang Lim}}）
: 年齡：23歲、生日：1993年7月18日、星座：[[巨蟹座|巨蟹座]]、身高：170cm、血型：O型{{sfn|皇太子的王子|loc=第二季 預告}}
: 男性，於城堡軟體工作，負責圖面設計，於男子高中及專門大學遊戲設計部畢業。有一個姐姐。尚未服役。形象可愛，挺受歡迎，在男中時已有不少人氣，畢業時全校同學集體抱頭痛哭{{sfn|皇太子的王子|loc=第一季 第5話}}{{sfn|皇太子的王子|loc=第二季 預告}}。

; 芈香丹（{{lang|ko|미향단}}，{{lang|en|Mi Hyang-dan}}）

: 女性，城堡軟體的代理，負責圖面設計，神學大學出身，有一個妹妹。曾為修女，但因對信徒施暴而被趕出{{sfn|皇太子的王子|loc=第一季 第5話}}{{sfn|皇太子的王子|loc=第二季 預告}}。

; 蔣秘書（{{lang|ko|강비서}}，{{lang|en|Secretary Gang}}）
: 男性，時賢的秘書，經常受不了時賢的各種脫軌行為，暗地裡把他稱為「反社會大便人（{{lang|ko|싸이코 똥쟁이}}）」。

; 原繼佑（{{lang|ko|원계위}}，{{lang|en|Won Gye-wi}}）
: 男性，比時賢大5歲的表哥，李真喜的丈夫。經常被誤稱為「原基友（{{lang|ko|원게이}}，{{lang|en|Won Gei}}）」。初戀為穿女裝的時賢，因而在親戚之間被嘲笑為蘿莉控同志，並對時賢懷恨在心，決心把跟他有關的一切毀掉{{sfn|皇太子的王子|loc=第一季 第25話}}{{sfn|皇太子的王子|loc=第一季 第26話}}。

; 李敏赫（{{lang|ko|이민혁}}，{{lang|en|Lee Min-hyuk}}）
: 男性，李真喜的弟弟。小時候曾為尤娜的同學，因欺負她而被強制退學。現時協助原繼佑毀掉時賢的一切，為了對付他而接近尤娜{{sfn|皇太子的王子|loc=第一季 第19話}}。

; 李真喜（{{lang|ko|이진희}}，{{lang|en|Lee Jin-hui}}）
: 女性，李敏赫的姐姐，原繼佑的妻子。以前曾為時賢的未婚妻。

; 原玟慧（{{lang|ko|원민혜}}，{{lang|en|Won Min-hye}}）
: 女性，尤娜的母親，繼佑的姑姑，不顧家族反對與一介平凡小職員結婚，因而受到家族的唾棄，更跟父親斷絕父女關係{{sfn|皇太子的王子|loc=第一季 第25話}}。

; 春香（{{lang|ko|춘향}}，{{lang|en|Chun-hyang}}）
: 夢樂園孤兒院的其中一個孤兒，在繪畫方面很有天賦，夢龍被欺負時對他伸出援手，因而成為朋友。腐女，喜歡畫BL漫畫。初二時被收養，但16歲時就患上了一個全身出血的不治之症，不希望繼續試新藥苟然殘存，懇求夢龍幫她拔喉放棄治療，臉帶笑容地去世。

; 姜申愛（{{lang|ko|강신애}}，{{lang|en|Gang Sin-ae}}）
: 女性，尤娜的高中同學，喜歡動漫及[[角色扮演|角色扮演]]，被金韶云欺凌。因尤娜幫助她而非常感激她，把卞學道的服裝送了給她{{sfn|皇太子的王子|loc=第一季 第1話}}{{sfn|皇太子的王子|loc=第一季 第3話}}。

; 金韶云（{{lang|ko|김소윤}}，{{lang|en|Gim So-yun}}）
: 女性，尤娜的高中同學，未來系統老闆的女兒。欺凌姜申愛，被尤娜向老師告發，因而對尤娜懷恨在心，跟她打架，但敗給尤娜。因時賢下令撤回未來系統的一切投資資金而導致公司倒閉，父親受不了打擊而一病不起，母親為了籌錢而到處低聲下氣地求人，於是威迫全班同學一起排擠尤娜並綁架她，把時賢引出來，打算殺死他。

; 徐勝熙（{{lang|ko|서승희}}，{{lang|en|Seo Seung-hui}}）
: 女性，尤娜的高中同學，因為長得太醜，小學時被叫作半獸人，國中時被叫作麻子，後來整容。受金韶云指使，威迫全班同學一起排擠尤娜並綁架她。
