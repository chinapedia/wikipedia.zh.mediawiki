'''皇甫暉'''（{{bd|9世纪||956年|catIdx=H}}），[[五代|五代]]時期軍事人物，[[魏州|魏州]][[興唐府|興唐府]]（今[[河北省|河北省]][[大名縣|大名縣]]）人，曾以一介小卒之姿發動[[鄴都之變|鄴都之變]]，引發了[[後唐莊宗|後唐莊宗]]之死，與[[後唐明宗|後唐明宗]]的[[即位|即位]]，[[後晉|後晉]]時繼續擔任[[將領|將領]]，因[[契丹|契丹]]的攻擊，投奔[[南唐|南唐]]，官拜[[奉化|奉化]][[節度使|節度使]]，後與[[後周|後周]]作戰，被後周將領[[趙匡胤|趙匡胤]]擊傷，傷重而死。

== 效力后唐 ==
{{main|鄴都之變}}
[[後唐莊宗|後唐莊宗]]時為[[魏博軍|魏博軍]]的一名[[小兵|小兵]]。[[同光|同光]]四年（926年），庄宗[[刘玉娘|刘皇后]]传教令给在蜀地的儿子魏王[[李继岌|李继岌]]令其杀死同在蜀地的[[成德节度使|成德节度使]][[侍中|侍中]][[郭崇韬|郭崇韬]]，不久庄宗又杀[[河中节度使|河中节度使]][[尚书令|尚书令]][[朱友谦|朱友谦]]，魏博监军武德使[[史彦琼|史彦琼]]则奉命去杀朱友谦子[[澶州|澶州]]刺史朱建徽。史彦琼出城时没有说明所为何事，于是出现谣言称：郭崇韬在蜀地杀死李继岌自立为王，刘皇后为儿子之死迁怒庄宗而弑君，史彦琼出城是去与刘皇后议事。

其时，皇甫晖所属之部隊戍守[[瓦橋關|瓦橋關]]（今[[河北省|河北省]][[雄縣|雄縣]]）期滿，原應回[[魏州|魏州]]（今河北省[[大名縣|大名縣]]），行经[[貝州|貝州]]（今河北省[[南宮市|南宮市]]），庄宗却担心魏州空虚，这支军队会生乱，命他们就地[[屯田|屯田]]，由於不能歸鄉的關係，軍心大亂。

皇甫暉與同袍在夜間[[賭博|賭博]]大敗，沒錢付帳，突心生一計，趁此軍心浮動之時機煽動士兵，士兵們[[兵變|兵變]]之後，就不會被追討債務。皇甫暉擁立指挥使[[楊仁晸|楊仁晸]]為[[首領|首領]]：「主上有天下是我們魏博軍的功勞，魏博軍征戰十多年，如今天下已定，天子不念舊勞，反而愈發猜忌我們了。我們遠戍一年多，離家咫尺，卻不讓我們相見。現在聽聞皇后弒逆，京師已亂，將士願與公俱歸，表聞朝廷。即使天子安泰，興兵征討，以我魏博兵力也足以拒之，怎知不是圖富貴的資本？」軍士們都呼應，仁晸不從，說：「您各位這樣算計也太過分了罷！現在英明的君主在朝廷之上，天下像是一家，主上的精甲銳兵，不下數十萬，各位也都有家屬，怎麼會講出這麼不祥的話？」皇甫暉斬之；又立一個裨將為首領，裨將不從，皇甫暉又斬之。偏將[[趙在禮|趙在禮]]越牆而走，被皇甫暉看見，皇甫暉把趙在禮從牆上拉下來，趙摔落地面，皇甫暉以[[獵首|伐取]]的楊仁晸、裨將兩個[[首級|首級]]來恐嚇趙，趙立刻同意[[叛變|叛變]]，於是皇甫暉自稱為「魏博馬步軍都指揮使」，立趙為[[大帥|大帥]]。

皇甫暉率兵攻入[[鄴城|鄴城]]，立赵在礼为[[留后|留后]]，邺都[[留守|留守]][[王正言|王正言]]被迫迎接。皇甫暉大肆殺害民眾，擄掠民財，到了一個民宅，聽說屋主[[國姓|姓國]]。皇甫暉說：「我就是來破國的！」就把姓國的屋主全家殺了。又到了下一個民宅，聽說屋主[[萬姓|姓萬]]。皇甫暉說：「我殺了『萬家』就夠了！」又把姓萬的屋主[[滅門|滅門]]了。種種殘暴類似於此。

莊宗派归德军节度使[[元行欽|元行欽]]前往討伐。元行钦攻邺都南门，以[[詔書|詔書]]诏谕，但史彦琼却大骂要破城将叛军碎尸万段，皇甫晖遂对士兵們说「投降也不会得到宽恕」，遂毁了[[詔書|詔書]]，不降。庄宗闻讯大怒，下令破城后[[屠城|屠城]]，发大军讨之，但元行钦始终不能破城。

莊宗又命[[李嗣源|李嗣源]]前往討伐，嗣源及[[武宁军节度使|武宁军节度使]][[霍彦威|霍彦威]]被軍士張破敗挾持，张破败意欲與魏博叛軍合兵造反，但皇甫晖不接纳，反而[[斩杀|斩杀]]张破败，城外乱军溃散，李嗣源、霍彦威等反被皇甫晖劫持。皇甫晖素忌霍彦威，想杀他，但霍彦威以口才說服了皇甫晖，免于一死。<ref>《旧五代史》卷六十四霍彦威传</ref>李嗣源以出城收拢旧部为由与霍彦威离开邺都后，李嗣源的女婿[[石敬瑭|石敬瑭]]、部將康義誠告訴李嗣源，造反局勢已成，後悔也勢難免罪。嗣源只好占據[[汴州|汴州]]（今河南[[開封|開封]]），進軍[[洛邑|洛邑]]。莊宗決定親征，指揮使[[郭從謙|郭從謙]]發動[[興教門之變|興教門之變]]，殺死莊宗。李嗣源進兵[[洛城|洛城]]而[[稱帝|稱帝]]，是為後唐明宗，封皇甫暉為[[陳州|陳州]][[刺史|刺史]]。

== 效力后晋 ==
[[後晉|後晉]]成立，皇甫暉繼續為官，任[[卫将军|卫将军]]，而[[趙在禮|趙在禮]]已為[[節度使|節度使]]，皇甫暉為人貪婪而暴虐，竟然趁赵在礼来朝，去恐嚇官位比他高的赵在礼，說趙在禮的富貴是由自己而得，如不給錢，當下就要攻擊趙，趙在禮畏懼，給了皇甫暉許多錢財，並擺設酒席相待，皇甫暉喝了酒拿了錢就走了，根本不拜謝趙在禮。后为[[密州|密州]]刺史。

== 效力南唐 ==
後晉遭到[[契丹人|契丹人]]攻滅時，皇甫暉率州人逃奔[[南唐|南唐]]。南唐中主[[李璟|李璟]]派舟楫相迎。皇甫晖担心自己出身凶贼，不为所容，行经[[秦淮河|秦淮河]]，跳水自杀，被左右救起，自称如同踩着大石头。<ref name=SGCQ24>《十国春秋》卷二十四·皇甫晖传</ref>皇甫晖入南唐，被封為[[歙州|歙州]]刺史、神卫军都虞候、[[奉化|奉化]]節度使，加[[同平章事|同平章事]]，镇[[江州|江州]]。南唐[[保大|保大]]六年（948年），[[后汉隐帝|后汉隐帝]]年幼，中原衰弱，淮北群盗多送款于李璟，李璟遣皇甫晖出海、泗诸州招纳。<ref>《新五代史》卷六十二南唐世家第二</ref>次年，[[蒙城|蒙城]]镇将咸师朗投降皇甫晖，但徐州将[[成德钦|成德钦]]在峒峿镇打败了皇甫晖，俘斩六百级，皇甫晖等回师。<ref>《资治通鉴》卷二百八十八</ref>

保大十四年（956年），[[後周世宗|後周世宗]]柴榮攻淮南，皇甫晖被任为北面行营应援使，会合神武统军北面行营都部署[[刘彦贞|刘彦贞]]、都监[[姚凤|姚凤]]提兵十万前去迎敌。刘彦贞败死，皇甫晖、姚凤退保清流关。柴荣進兵[[壽州|壽州]]（今[[安徽省|安徽省]][[壽縣|壽縣]]），命大將[[趙匡胤|趙匡胤]]（後來的[[宋太祖|宋太祖]]）攻清流关。皇甫晖先大败周军数千。<ref>《史纂左编》</ref>后赵匡胤用[[赵普|赵普]]计，在皇甫晖等率部在山下列阵时，带着部队绕过山后，从后面偷袭南唐军，皇甫暉大败，且战且走逃入[[滁州|滁州]]，但滁州刺史王绍颜已弃城逃跑，皇甫晖断桥自守。周军涉水越城而入。<ref name=SGCQ24/>皇甫晖逃亡失路，要求決鬥，趙匡胤武藝甚高，笑而應允，奮馬直前，一刀中皇甫暉脑部，砍落馬下，<ref>《史纂左编》则称皇甫晖为赵匡胤三纵三擒。</ref>姚凤一并被擒。

皇甫晖被送交世宗親自審問，说：“臣疲惫了，想暂时坐下。”坐下后又说：“想暂时躺下。”不等命令就躺下，神色自若，仰头说：“我皇甫晖从贝州军伍起兵辅佐李嗣源，促成了唐庄宗之祸；后来率众投江南，位兼将相，前后南北二朝大小数十战从未失败。今天被擒，是上天褒奖大朝兴盛，也是南北两军的勇怯不相當。<ref>《史纂左编》则称皇甫晖称上天褒奖赵匡胤，盛赞赵匡胤之神武。</ref>”很淡定地告訴世宗，後周部隊極為勇猛頑強，比[[契丹人|契丹人]]還優秀。世宗見了皇甫暉的傷勢非常同情，賞賜金带、鞍马等許多禮物。但几天后，皇甫暉伤情加重，自知時日不多，拒絕醫療而死。滁州人感念于他，一天有五个时辰鸣钟祭献于他。<ref name=SGCQ24/>

有子[[皇甫继勋|皇甫继勋]]。

== 评价 ==
*[[欧阳修|欧阳修]]《新五代史》：“晖为人骁勇无赖。”
*[[南宋|南宋]][[陆游|陆游]]《[[南唐书_(陆游)|南唐书]]》、[[清朝|清朝]][[吴任臣|吴任臣]]《十国春秋》：“晖独持重，部分甚整，士亦乐为用。周人颇惮之。”
*[[清朝|清朝]][[赵翼|赵翼]]《[[廿二史札记|廿二史札记]]》：“[[刘仁赡|刘仁赡]]、皇甫晖、姚凤皆完节于南唐者。”
*《十国春秋》论曰：皇甫晖以乱卒位刺史，拔身南奔，滁州之败，义不求生，庶几可为晚盖者矣。
