'''皇家旗'''或'''王室旗'''（{{lang-en|'''Royal Standard'''}}）是[[英國|英國]]及其他[[英联邦王国|英联邦王国]][[英國君主|女王]][[伊莉莎白二世|伊莉莎白二世]]在英國與其他[[英聯邦|英聯邦]]王国作為[[元首|元首]]的象征旗帜。在英國，[[旗幟|旗幟]]由[[國徽|國徽]]中央的盾徽圖案形成。而在英國之外的一些较大的英聯邦王国，皇家旗旗幟則是將皇家徽章放於該英聯邦王国國徽內的盾徽圖案中央（試比較英格兰、蘇格蘭、澳洲、巴貝多、加拿大、牙買加、紐西蘭各自的皇家旗及各自的國徽），在其他没有各自的皇家旗的其他英联邦王国，则统一使用皇家徽章作为皇家旗。皇家徽章為藍色底，中央为大写字母“E”（代表“伊麗莎白”），字母E上为加冕皇冠，四周以金色玫瑰花圈包圍。

皇家旗是英国及多数英联邦王国的旗帜礼仪中位居首位。此旗只能在女王所在的建築物或车船上使用，如同时有其他旗帜使用的话，皇家旗的高度必须超过[[英國國旗|英國國旗]]、其它王室成員旗幟和其它旗帜。此外，皇家旗从不降半旗，表示女王不向任何人屈首。

==英國==
在[[英格蘭|英格蘭]]、 [[威爾士|威爾士]]、 [[北愛爾蘭|北愛爾蘭]]及其他[[英國海外領地|英國海外領地]]的英國王室旗幟分成四個象限。
第二個和第四個象限代表[[英格蘭|英格蘭]]並且包括三頭向前直走姿勢的金獅子在紅色區塊；第一象限代表[[蘇格蘭|蘇格蘭]]包括一頭紅獅子以後腳直立在金色區塊；第三象限代表[[愛爾蘭|愛爾蘭]]包括金豎琴在藍色區塊。

在[[蘇格蘭|蘇格蘭]]使用跟英格蘭相似的旗幟，但是紅獅子金色區塊卻在第二與第四象限，而三頭向前直走姿勢的金獅子卻在第一象限。

現在的王室旗幟，有些小改變（尤其[[豎琴|豎琴]]曾經用來代表愛爾蘭），從[[維多利亞女王|維多利亞女王]]王朝時代始用。而在此前[[乔治一世_(大不列颠)|喬治一世]]至[[喬治三世|喬治三世]]時期的王室旗幟則合併了[[漢諾威|漢諾威]]的盾徽（以代表國君乃系[[神聖羅馬帝國|神聖羅馬帝國]][[選帝候|選帝候]]）與[[法國|法國]]盾形紋徽，宣示英國與漢諾威的[[共主邦聯|共主邦聯]]之餘同時[[英格蘭王室對法國王位的主張|要求法國承認共主]]。

其中最為有名的旗幟包括英國[[安妮女王|安妮女王]]的''蘇格蘭統一者''、[[喬治一世_(大不列顛)|喬治一世]]至[[喬治三世|喬治三世]]的''漢諾威選帝侯''與[[喬治三世|喬治三世]]至[[威廉四世_(英國)|威廉四世]]的''漢諾威國王''。當1837年英王威廉四世駕崩後，英國王位傳給了血緣關係較近的姪女[[維多利亞女王|肯特郡主維多利亞]]。不過卻因德意志地區施行[[歐洲大陸|歐洲大陸]]的[[撒利法|撒利法]]，禁止女性繼承王位。維多利亞三伯父的漢諾威國王寶座，傳給了她的五叔[[恩斯特·奥古斯特一世_(汉诺威)|坎伯蘭公爵]]。之後，添加了漢諾威的盾徽的[[汉诺威王朝|漢諾威王室]]旗幟便沿用至今（儘管該旗幟中英格蘭、愛爾蘭與蘇格蘭的紋章有顯微象限位置不同）。

==澳洲==
在[[澳洲|澳洲]]的王室旗幟被劃成六等分，第一等分的聖喬治紅十字代表[[新南威爾士州|新南威爾士州]]，中間是一頭金獅子，四方位則有各一顆金色星星。第二等分的皇冠和六顆白色星星代表[[維多利亞州|維多利亞州]]。第三等分的藍色馬爾他十字架與中間的皇冠代表[[昆士蘭州|昆士蘭州]]。第四等分的食火鳥代表[[南澳大利亞州|南澳大利亞州]]。第五等分的黑天鵝代表[[西澳大利亞州|西澳大利亞州]]。最後的第六等分，一頭紅色的獅子則代表[[塔斯馬尼亞州|塔斯馬尼亞州]]。在最上面的王室徽章則有一顆光芒耀射七個方向的星星，代表著澳洲六個州與[[北領地|北領地]]。

==巴貝多==
在[[巴貝多|巴貝多]]的王室旗幟是黃色的，中間是有根的[[無花果|無花果]]樹（葉子是藍色的），[[巴貝多的自尊花|巴貝多的自尊花]]則是在左右兩側之上。無花果樹上還有個王室代表勳章。最具特點是中間的無花果樹，因該國是以無花果樹名為名的。

==加拿大==
[[加拿大|加拿大]]的王室旗幟上的三分之二部份與英格蘭王家旗幟是相同的，除了第四象限的三個金色[[鸢尾花|鸢尾花]]的法國旗幟。旗子的底部則有三片加拿大的紅色[[楓葉|楓葉]]。當然，在旗幟最上面也是王室勳章。

==牙買加==
[[牙買加|牙買加]]的王室旗幟是白底兼紅色的[[聖喬治十字|聖喬治十字]]，在十字架的底端則各有一顆金鳳梨。王室勳章則放在十字架的中心。

==紐西蘭==
[[紐西蘭|紐西蘭]]的王室旗幟被劃分為四個象限，但是有白色垂直條紋通過中心。第二象限有四顆紅色的星星。第一象限有一個金色的羊毛。第三象限有一捆金色的麥子。最後的第四象限包括兩把金鎚子。中間條紋則有三艘船，旗子最上面則有王室勳章遮住了中間的船。

==其他英聯邦國家==
其他的[[英聯邦|英聯邦]]國家沒有自己的王室旗幟。在元首不是英國女王的聯邦國家，代表旗幟通常使用一個藍色圖上有個'E'，而E上有皇冠、再由金玫瑰花圈包圍著的女王旗幟。

[[獅子山|獅子山]]、[[馬爾他|馬爾他]]及[[千里達及托巴哥|千里達及托巴哥]]一開始都有王室旗幟，但是當他們成為[[共和國|共和國]]之後就廢止了。

==其他皇家成員==
其他[[英國皇家|英國皇家]]成員的旗幟都是王室旗幟按照旗帜礼仪加上与其身份相应的图案，修改而成的。

===威爾士親王===
威爾士親王有四面不同的個人代表旗幟：一面用在英格蘭(與北愛爾蘭)、一面用在蘇格蘭、一面用在康沃爾、最後一面用在威爾士。

; 英格蘭與北愛爾蘭: 使用王室在英格蘭使用的旗幟，加上白條與三個點。在旗子的中心，有威爾士公國的盾形徽章疊在旗幟上：盾章上的四個象限中，第二與第四象限是一頭紅獅子在金區塊、第一與第三象限則是由一頭金獅子在紅色區塊。這威爾士王子的英格蘭旗幟，通常也用在海外地區(出訪時，也是使用這面旗幟)。

; 威爾士: 旗幟被劃分為四個象限，第二與第四象限是紅獅子在金色區塊，第一和第三項限是金獅子在紅色區塊。而疊在上面的是綠色的盾牌與冠冕。

; 蘇格蘭 : 根據威爾士親王的兩種頭銜：[[羅撒西公爵|羅撒西公爵]]與[[群島之主|群島之主]]，旗子劃分為四個象限。第二與第四象限有一條藍色與白色的棋盤帶在中間的金色區塊。第一與第三象限有一艘船在白色區塊。而金盾上有以後腳直立的蘇格蘭紅獅子。

; 康沃爾 : 黑底上鋪十五枚金幣。

===有皇室血緣的王子與公主们===
其他的皇室成員有他們自己的個人旗幟。這些旗幟的樣子都是以王室旗幟為基礎加以變造的，上面以一個白色長帶橫放與放上三個點或是垂飾(當代君王的子女)，或五個點(當代君王之孫)。傳統上所有擁有皇室血緣的王子與公主(君王的後裔)，都會在他們的18歲生日被授予盾形徽章，並且授予他們的個人旗幟可以在自己往後居住的地方飄揚。

以下是皇家成員的個人旗幟
* 皇室[[約克公爵|約克公爵]]殿下: 白色長帶中間是藍色船錨，第一與最後的點則什麼都沒有。
* 皇室[[比阿特丽斯公主_(约克)|约克比阿特丽斯公主]]殿下: 白色長帶第一、中間(第三)與最後(第五)的點都是蜜蜂，而第二與第四個點什麼都沒有。
* 皇室[[爱德华王子|威塞克斯伯爵]]殿下: 白色長帶中間是一朵紅色玫瑰(英國國花)，第一與最後的點什麼都沒有。
* 皇室[[安妮公主|王室長公主]]殿下: 白色長帶第一與最後是紅十字，中間是紅色的心形。
* 皇室[[威廉王子_(威爾斯)|威廉王子]]殿下: 不同於其他君王的孫子，威廉王子使用擁有三個點的長條帶(應該用五個點的長條帶)。中間點有紅色貝殼，第一與最後的點什麼都沒有。
* 皇室[[哈利王子_(威尔士)|哈利王子]]殿下: 第一、中間(第三)與最後(第五)的點都是鑲上紅色貝殼，而第二與第四個點什麼都沒有。
* 皇室[[格洛斯特公爵|格洛斯特公爵]]殿下: 使用五個點的白長條帶，單數點使用聖喬治十字架，偶數點使用向前直走姿勢、面對面的紅獅子。
* 皇室[[愛德華親王_(根德公爵)|肯特公爵]]殿下: 使用五個點的白長條帶，單數點使用藍船錨，偶數點聖喬治十字架。
* 皇室[[邁克爾王子|肯特麥克王子]]殿下: 使用五個點的白長條帶，單數點使用聖喬治十字架，偶數點藍船錨。
* 皇室[[雅麗珊郡主|亞莉珊卓拉公主]]殿下: 使用五個點的白長條帶，第一與第五使用心形，第二與第四使用船錨，中間則是聖喬治十字架。

===英國國君的配偶===
男性國君的配偶（國王的皇后）有她們自己的個人旗幟。這些旗幟的樣子都是以王室旗幟及她們父親的紋章圖案做組合，在蘇格蘭，使用蘇格蘭版本的王室旗幟及她們父親的旗幟做組合。例如[[太后|皇太后]][[伊麗莎白·鮑斯-萊昂|伊麗莎白·鮑斯-萊昂]]的個人旗幟為右側放上她父親家族斯特拉絲摩伯爵紋章圖案，左側放上王室旗幟。在英國之外具有自身皇家旗的英联邦領土國家沒有為她們另外設計旗幟。

女性國君的配偶（女王的王夫）不會被授予此類旗幟，他們使用自身家族的紋章圖。王夫[[菲利普亲王_(爱丁堡公爵)|愛丁堡公爵]]殿下，就有自己特別的旗幟。根據其血統，旗子劃分為四格象限，第一象限代表[[丹麥|丹麥]]，包括三頭向前方向的藍獅子與九顆紅心。第二象限代表[[希臘|希臘]]，白色十字放在藍區塊上。第三象限代表蒙巴頓家族，分為五等分，並以兩條黑縱線放在白色區塊上。而第四象限代表愛丁堡，一座紅頂黑堡。

===其他皇室成員===
其他的皇室成員未必使用英國王室旗幟。像是沒有皇室血緣的威爾士王妃[[黛安娜王妃|黛安娜王妃]]，葬禮上就看不到她自己的旗幟。
