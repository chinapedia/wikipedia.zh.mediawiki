'''野牡丹'''（[[学名|学名]]：{{lang|la|''Melastoma candidum''}}，[[英文|英文]]名稱：{{lang|en|Common Melastoma}}）<ref>[http://www.cfh.ac.cn/63860.sp Melastoma malabathricum L.野牡丹] 中國自然標本館</ref><ref>[http://www.producegreen.org.hk/hkwildflower/100flower/flower_38.htm 野牡丹] 香港野花網</ref>，为[[野牡丹科|野牡丹科]][[野牡丹属|野牡丹属]]的植物，又名'''山石榴'''（台湾）、'''大金香炉'''、'''猪古稔'''（广东）、'''豹牙兰'''（云南）、'''豹牙郎木'''、'''倒罐草'''、'''九螺子花'''、'''高腳稔'''、'''罐罐草'''、'''地茄'''、'''金石榴'''、'''老虎杆'''、'''埔筆仔'''、'''痢疾罐'''、'''毛張口'''、'''小毛香'''、'''王不留行'''、'''王不留'''<ref name = "浸會大學"/><ref name = "公務人力">[http://www.hrd.gov.tw/WebForm/wfrmPlant.aspx?menu_id=181&page=4 中心植物介紹—野牡丹]公務人力發展中心</ref>。原產於[[巴西|巴西]]，分布于[[台湾|台湾]]、[[印度|印度]]以及[[中国大陆|中国大陆]]的[[广东|广东]]、[[福建|福建]]、[[广西|广西]]、[[云南|云南]]等地，生长于海拔120米的地区，多生长于山坡松林下和开朗的灌草丛中。<ref>{{cite web| url =http://www.bioinfo.cn/db05/KmzwSpecies.php?action=view&id=12202| title =野牡丹| work =《中国高等植物数据库全库》| publisher =中国科学院微生物研究所| author =昆明植物研究所| accessdate =2009-02-24| deadurl =yes| archiveurl =https://web.archive.org/web/20160305072746/http://www.bioinfo.cn/db05/KmzwSpecies.php?action=view&id=12202| archivedate =2016-03-05}}</ref>花期5－9月，果期10－12月<ref name = "浸會大學">[http://libproject.hkbu.edu.hk/was40/detail?channelid=1288&searchword=herb_id=D01032 藥用植物圖像數據庫]香港浸會大學中醫藥學院</ref>。

因野牡丹在眾多野花之中，最為綺麗頗有[[牡丹|牡丹]]的雍容華貴，故而得名<ref>[http://www.tesri.gov.tw/plant/change03.htm 行政院農業委員會] {{webarchive|url=https://web.archive.org/web/20080527215620/http://www.tesri.gov.tw/plant/change03.htm |date=2008-05-27 }}特有生物研究保育中心</ref>。

==形態特徵==
野牡丹是一種常綠小灌木，高約0.5－1.5米，可高達3米。莖鈍四棱形或近圓柱形，披淡褐色鱗片狀糙毛。單葉對生，長橢圓形或卵形，先端鈍尖，基部近圓形或淺心形，全緣，葉片堅紙質，兩面披淡褐色糙毛及短柔毛，長約4－12厘米，寬約3－8厘米；葉脈5－7條基出；葉柄長約2－5厘米。花為聚傘花序，長於分枝頂端，近頭狀，由3－7朵花組成，稀單生，基部具葉狀總苞2；花梗密披鱗片狀糙毛，長約3－20[[亳米|亳米]]；花萼5裂，裂片卵形或略寬，內側紅色，外側披淡褐色鱗片狀糙毛，先端漸尖，萼管壺形，長約1厘米；花瓣5片，倒卵形，先端圓形，粉紅色或玫瑰紅色，密披緣毛，離瓣花；雄蕊10枚，5長5短，較長的雄蕊基部黃色直立，上部呈關節狀彎曲，狀似鐮刀，連接紫紅色半圓形的[[花藥|花藥]]，短的雄蕊黃色，並未分兩節，連接黃色的花藥；雌蕊柱狀，墨綠色；花柱線形，紫紅色；子房下位，5室，密被糙伏毛，先端具一圈剛毛。果為[[蒴果|蒴果]]，壺形，與宿存萼貼生，密披褐色鱗片狀糙毛，成熟後開裂，果肉紫色，長約1－1.5厘米，直徑約8－12[[亳米|亳米]]；種子白色，鑲於肉質胎座內。<ref name = "浸會大學"/><ref name = "公務人力"/><ref>[http://digital-kinmen.kmnp.gov.tw/files/806-1000-2425,c83-1.php 金門國家公園]數位典藏網</ref><ref>http://nrch.culture.tw/twpedia.aspx?id=6542 臺灣大百科全書</ref>

==醫藥用途==
野牡丹全株可入藥。全株，中藥名為野牡丹，味酸、澀，性涼，歸脾、胃、肺、肝經，能清熱解毒、消炎止血、消積利濕、消食止痢，主治腹瀉、便血、吐血、咳血、月經過多、乳汁不下、腸癰、肝炎、白帶、產後腹痛、跌打損傷及外傷出血等<ref>[http://www.tcmlib.com/zy/html68/showdetail-363730322ce9878ee789a1e4b8b92c7a79.html 野牡丹]{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}意古中醫</ref>。根，中藥名為野牡丹根，別名痢疾罐、王不留，味酸、澀，性平，能健脾利濕、活血止血，主治消化不良、便血、月經不調、食積腹痛、頭痛、風濕痹痛及跌打損傷等<ref>[http://www.tcmlib.com/zy/html68/showdetail-363730342ce9878ee789a1e4b8b9e6a0b92c7a79.html 牡丹根]{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}意古中醫</ref>。果實或種子，中藥名為野牡丹子，別名豹牙郎子，味苦，性平，能通經下乳、活血止血，主治乳汁不通、產後腹痛、難產等<ref>[http://www.tcmlib.com/zy/html68/showdetail-363730332ce9878ee789a1e4b8b9e5ad902c7a79.html 野牡丹子]{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}意古中醫</ref>。<ref name = "浸會大學"/>

==醫藥典故==
相傳[[東漢|東漢]]的開國功臣[[邳彤|邳彤]]，因他家鄉的民眾不齒王莽及王郎等人的篡漢行為，故當王莽率大軍追殺[[劉秀|劉秀]]而路經此地時，拒絕為王莽的大軍提供食宿，王莽被迫連夜氣憤地離開邳彤家鄉。邳彤因精通藥理，常為人治病，人稱「藥王」，當他發現家鄉田園裡所產的野牡丹有舒筋活血、通乳止痛之用時，為反映此事，故而命名為「王不留」 或「[[王不留行|王不留行]]」<ref name = "公務人力"/>。

==相似品種==
*[[毛菍|毛菍]]（''Melastoma sanguineum''），同為野牡丹科野牡丹屬植物，分別在於毛菍的葉片無毛或像鋪上一層細毛<ref>[http://hkcww.org/hkplant/monthtopic/topic1.htm 野牡丹與毛稔分別]柴娃娃植物網</ref>及手感較為粗糙，枝條上有長毛<ref>[http://www.hktree.com/tree/Melastoma%20candidum.htm 野牡丹]樹木谷</ref><ref>{{cite book|title=《香港野花 第一冊》|author= |series=魏遠娥、洪烈招、陳淑瓊、陳愛強|publisher=效野公園之友會、天地圖書|page=第268-269頁|isbn= 978-988-211-344-2|date=2007年7月}}</ref>；
*[[地菍|地菍]]（''Melastoma dodecandrum''），是野牡丹科野牡丹屬，分別在於地菍的花與野牡丹、毛菍及崗棯的花比較下，為最細者<ref>{{cite book|title=《香港野花 第一冊》|author= |series=魏遠娥、洪烈招、陳淑瓊、陳愛強|publisher=效野公園之友會、天地圖書|page=第234-235頁|isbn= 978-988-211-344-2|date=2007年7月}}</ref>；
*[[崗棯|崗棯]]（''Rhodomyrtus tomentosa''），則為桃金娘科桃金娘屬植物<ref>[http://www.hktree.com/tree/Rhodomyrtus%20tomentosa.htm 崗棯]樹木谷</ref>。
