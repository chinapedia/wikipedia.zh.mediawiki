'''野國總管'''（{{ryu|r=野國總管|shin=野国総管|kn=ヌグンソークァン|rm=Nugun Sōkwan}}；生卒年不詳）是[[琉球國|琉球]][[第二尚氏王朝|第二尚氏王朝]]時期的一位人物。由中國福建，他將[[番薯|番薯]]引入琉球的[[沖繩島|沖繩島]]。

==姓名考據==
野國總管只是這位人物的稱謂，野國指的是[[北谷間切|北谷間切]]的野國村（今[[中頭郡|中頭郡]][[嘉手納町|嘉手納町]]野國），總管則是[[進貢船|進貢船]]船員的一個役職名。<ref name="Kadena">{{Cite web |title=野國總管コーナー |url=http://www.town.kadena.okinawa.jp/kadena/soukan/soukancorner/index.html |work=（公式ウェブサイト）|publisher=嘉手納町}}</ref><ref name="Kawasaki pdf">{{Cite web |title=焼き芋小百科 |url=http://www.jrt.gr.jp/yaki_imo/zenbun.pdf#search=%27 |format=pdf |work=（公式ウェブサイト）|publisher=川崎いも友の会}}</ref><ref name="JCRD">{{Cite web|title=沖縄県（嘉手納町）　わが国に甘藷をもたらした「芋大主」　野國總管|url=http://www.chiiki-dukuri-hyakka.or.jp/1_all/jirei/100furusato/html/furusato100.htm|work=伝えたいふるさとの100話（公式ウェブサイト）|publisher=財団法人 地域活性化センター[http://www.chiiki-dukuri-hyakka.or.jp/]|deadurl=yes|archiveurl=https://web.archive.org/web/20120720011109/http://www.chiiki-dukuri-hyakka.or.jp/1_all/jirei/100furusato/html/furusato100.htm|archivedate=2012-07-20}}</ref>野國總管的詳細事蹟不詳，根據近代學者的考證，野國總管出生在野國村，其祖先世世代代都是野國村的百姓。<ref name="Soukan03, Kadena">
{{Cite web |author=伊波勝雄 |title=3. 野國總管の身分とその子孫 |url=http://www.town.kadena.okinawa.jp/kadena/soukan/soukancorner/03.html |work=野國總管コーナー（公式ウェブサイト）|publisher=嘉手納町}}</ref>

野國總管的姓名曾長期是個謎，不過根據分析總姓系譜圖，可以多多少少瞭解其情況。總姓系譜圖的「一世」條記載「大宗藩初野國總管，號松岩；室：号永屋」。而今日總姓與那霸家第十三代家族中保存的[[位牌|位牌]]則記載「總管野國松岩，名乘裕初，言名總世健」。由此可以推斷，野國總管的[[童名|童名]]為「'''松'''」，名乘「'''裕初'''」，[[和名|和名]]為「'''與那霸裕初'''」、[[唐名|唐名]]為「'''總世健'''」。另外，野國總管之妻戒名「永屋」，可以推斷，其妻的童名為「永」。<ref name="Soukan02, Kadena">
{{Cite web |author=伊波勝雄 |title=2. 野國總管の姓名を考える |url=http://www.town.kadena.okinawa.jp/kadena/soukan/soukancorner/02.html |work=野國總管コーナー（公式ウェブサイト）|publisher=嘉手納町}}</ref>

==生平==
野國總管於1605年前往[[明朝|明朝]]之際，在[[福州|福州]]見到番薯，將其植於盆上帶回琉球，在野國村進行試種。番薯的能夠克服惡劣的天氣，易於扎根，因此該村農民廣為栽種。首里士族[[麻平衡|麻平衡]]（儀間親方真常）得知後，向野國總管求取了番薯苗，並學習了番薯的種植方法，在琉球各地廣為傳播。歷經十五年，琉球各地皆知曉了種植番薯之法，有效防止了饑荒。野國總管也因功被昇為首里士族。<ref name="Kawasaki pdf" /><ref name="JCRD" /><ref name="kyuyo4">《[[:s:球陽記事/卷之四|球陽記事·卷四]]》</ref>

==後世紀念==
琉球當地人對野國總管甚為尊敬，稱之為「甘藷大主（ンムウスー）」、<ref name="Soukan44, Kadena"/>「芋大主」（ウムウフシュ）。<ref name="JCRD" />直至[[二戰|二戰]]結束以前，琉球人仍以番薯為主食。<ref name="Soukan44, Kadena">
{{Cite web |author=伊波勝雄 |title=44. 甘藷の開拓者、野國總管 |url=http://www.town.kadena.okinawa.jp/kadena/soukan/soukancorner/44.html |work=野國總管コーナー（公式ウェブサイト）|publisher=嘉手納町}}</ref>

1700年，擔任野國村地頭職的[[野國正恒|野國親方正恒]]在野國港西地洞中創建石堂，將其遺骨遷葬於此，每年兩次進行祭祀。<ref name="kyuyo4"/>1751年，野國總管六世孫比嘉筑登之在墓旁建立「甘藷撥祥之地」六字顯彰碑。<ref name="Soukan04, Kadena" />

1936年，[[沖繩縣|沖繩縣]]廳在[[那霸市|那霸市]][[奧武山|奧武山]]建立[[世持神社|世持神社]]，將發展沖繩產業的三大恩人野國總管、[[蔡溫|蔡溫]]（具志頭親方文若）、[[麻平衡|麻平衡]]（儀間親方真常）供奉其中。<ref name="Soukan04, Kadena" />

野國總管的墓於1945年[[沖繩島戰役|沖繩島戰役]]中被毀，放置其遺骨[[石廚子|石廚子]]遭破壞，只剩下屋頂形的石蓋，但其遺骨尚存。<ref name="Soukan04, Kadena">
{{Cite web |author=伊波勝雄 |title=4. 野國總管の墓の建立と顕彰碑 |url=http://www.town.kadena.okinawa.jp/kadena/soukan/soukancorner/04.html |work=野國總管コーナー（公式ウェブサイト）|publisher=嘉手納町}}</ref>1955年，在[[嘉手納町|嘉手納村]]建立「野國總管宮」，分其遺骨祭祀。

==祭祀==
在第二尚氏王朝時期，北谷間切的野國、野里、砂邊三村人民，於二三月中擇吉祭祀其墓，以祈豐年。[[小祿村|小祿間切]]儀間村的百姓，也在儀間村的赤平祭祀野國總管，以報答其種植番薯之恩。<ref name="kyuyo4"/>1922年（大正十一年）的陰曆八月十七，北谷村（今北谷町）舉行祭祀其墓。<ref name="Soukan04, Kadena" />今日嘉手納町亦以其遺德不小，於每年秋季舉行「野國總管祭」。<ref name="JCRD" />。
