'''野邊地町'''（{{jpn|j=野辺地町|hg=のへじまち|rm=Noheji machi|f=y}}）是位於[[青森縣|青森縣]][[下北半島|下北半島]]南部[[陸奥灣|陸奥灣]]的[[町|町]]。由於受到山背的影響而整年都十分寒冷，冬天時為縣內少有的大雪地區。

== 歷史 ==
野邊地的名字是來自[[阿伊努語|阿伊努語]]的「nup-un-pet」，有「在原野中流著的清澈之河」的意思。古時作為交通的重要地點而繁榮。而現在亦作為下北地方及上北地域的中核都市而有廣泛的交流，發揮其機能的話則繁榮是指日可待。

* 1889年（明治22年）4月1日 - 施行町村制，野邊地村、馬門村、有戶村合併為'''野邊地村'''。
* 1897年（明治30年）8月28日 - 町制施行，改為'''野邊地町'''。

== 地理 ==
位於[[下北半島|下北半島]]的根部，北面被[[陸奧灣|陸奧灣]]、南面被[[烏帽子岳|烏帽子岳]]所包圍。

== 經濟 ==
=== 產業 ===
擔當上北郡北部地區（[[東北町|東北町]]的千曳以北地區、[[橫濱町|橫濱町]]、[[六所村|六所村]]）及[[東津輕郡|東津輕郡]][[平內町|平內町]]的政治及商業核心，不過由於政府機關、事業所的重組，道路的高規格化等的緣故，令本町的重要性及人口不斷下降。

==== 奶酪畜牧業 ====
* 有戶地區

==== 漁業 ====
主要的海產為養殖的扇貝。
* 野邊地漁港
* 有戶漁港

==== 商業 ====
擁有MAEDA（マエダストア）、MAXVALU（マックスバリュ）等的食品超市，かんぶん、SUNDAY（サンデー）等HOME CENTRE，及HAPPY DRUG（ハッピードラッグ）、藥王堂及橫濱藥房等藥店。在2009年，河童（かっぱ）壽司開業。

備註：MAXVALU（マックスバリュ）野邊地店是繼承自以前營業但經營失敗的龜屋みなみチェーン的店舖。
