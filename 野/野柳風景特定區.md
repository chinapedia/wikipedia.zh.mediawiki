'''野柳地質公園'''，又稱為'''野柳風景區'''，是位於[[臺灣|臺灣]][[新北市|新北市]][[萬里區|萬里區]]野柳里的風景區，位於[[北海岸及觀音山國家風景區|北海岸及觀音山國家風景區]]範圍內。由於其天然的地質景觀，而成為[[台灣北海岸|臺灣北海岸]]知名的{{link-en|旅遊景點|Tourist attraction}}之一。

== 概要 ==
野柳地區為[[大屯山|大屯山]]山脈延伸至[[東海|東海]]之[[岬角|岬角]]，全區長約1700[[公尺|公尺]]、寬約250公尺。當地地景因[[造山運動|造山運動]]、[[風化|風化]]及[[海蝕|海蝕]]等[[地质作用|作用]]而呈現奇特形態，其中有「[[女王頭|女王頭]]」、「仙女鞋」、「蕈狀石」、「燭臺石」、「薑石」、「[[單面山|單面山]]」等特殊景觀。<ref>{{Cite web|url=http://www.ylgeopark.org.tw/content/info/YlIntroduction.aspx|title=野柳地質公園全球資訊網|accessdate=2017-03-26|last=http://www.ylgeopark.org.tw/|work=www.ylgeopark.org.tw}}</ref>

== 歷史 ==
[[大航海時代|大航海時代]]，[[西班牙|西班牙]]人經常乘船航行於[[基隆|基隆]]與[[淡水|淡水]]之間，往往在當地[[岬角|岬角]]觸礁擱淺並遭[[原住民|原住民]]洗劫。因此，他們在[[地圖|地圖]]上將當地標示為Punto Diablos（在[[西班牙文|西班牙文]]中，Punto意為「岬角」、Diablo意為「魔鬼」）。後來，[[臺灣西班牙統治時期|荷蘭人在打敗西班牙人後]]將當地標示為Duijvel hoek或Caap Diable（原名在[[荷蘭文|荷蘭文]]中之翻譯）。<ref>{{Cite web|url=http://gushi.tw/%E3%80%90%E8%87%BA%E7%81%A3%E5%9C%B0%E5%90%8D%E7%9C%9F%E7%9B%B8%E3%80%91%E9%87%8E%E6%9F%B3%E8%B7%9F%E9%87%8E%E7%94%9F%E7%9A%84%E6%9F%B3%E6%A8%B9%E6%9C%89%E9%97%9C%E4%BF%82%E5%97%8E%EF%BC%9F/|title=【臺灣地名真相】野柳跟野生的柳樹有關係嗎？|accessdate=2017-03-27|author=|date=2017-03-27|publisher=[[故事|故事]]|deadurl=yes|archiveurl=https://web.archive.org/web/20170327191352/http://gushi.tw/%E3%80%90%E8%87%BA%E7%81%A3%E5%9C%B0%E5%90%8D%E7%9C%9F%E7%9B%B8%E3%80%91%E9%87%8E%E6%9F%B3%E8%B7%9F%E9%87%8E%E7%94%9F%E7%9A%84%E6%9F%B3%E6%A8%B9%E6%9C%89%E9%97%9C%E4%BF%82%E5%97%8E%EF%BC%9F/|archivedate=2017-03-27}}</ref>

== 地理 ==
當地[[地層|地層]]主要為[[中新世|中新世]]野柳群大寮層；該地層分布於萬里區龜吼村至萬里區國聖埔海灘，主要由1000萬年至2500萬年前生成之厚層[[砂岩|砂岩]]所構成，[[顏色|顏色]]多呈[[青|青]][[灰|灰]]色或[[黃|黃]]灰色。

== 經營 ==
[[戰後台灣|戰後]]，野柳風景區一度為臺北縣[[萬里鄉_(台灣)|萬里鄉]][[公共造產|公共造產]]。

1978年，當地改由[[臺北縣政府|臺北縣政府]]管理，並設置「臺北縣野柳風景特定區管理所」負責經營；當時，園區[[門票|門票]]正面印有「野柳[[烏來|烏來]]風景特定區」字樣，背面印有「實踐[[三民主義|三民主義]]　建立安和社會　復興[[中華文化|民族文化]]　堅守[[中華民國與美國關係|民主陣容]]」字樣。

2002年，[[交通部觀光局|交通部觀光局]]成立[[北海岸及觀音山國家風景區|北海岸及觀音山國家風景區]]管理處，並將野柳風景特定區納入轄下。2006年，野柳風景特定區[[委外經營|委外經營]]由「新空間國際有限公司」營運，並隨後改稱「野柳地質公園」。

2010年，「野柳地質公園委託經營案」獲得[[行政院公共工程委員會|行政院公共工程委員會]][[金擘獎|金擘獎]]。
