'''野犬式全方位防護運輸車'''（[[德文|德文]]：'''A'''llschutz-'''T'''ransport-'''F'''ahrzeug Dingo，[[縮寫|縮寫]]：'''ATF Dingo'''），是[[德国联邦国防军|德国联邦国防军]]现役的一款军用[[装甲车|装甲车]]

== 设计和特点 ==
“野犬”式全防護車由德国[[克劳斯-玛菲·威格曼|克劳斯-玛菲·威格曼]]公司设计生产，使用[[烏尼莫克|烏尼莫克]]底盘。Allschutz-Transport-Fahrzeug的[[德文|德文]]意思解作“全方位防护运输车辆”，顾名思义，良好的防卫性能是该车的一大特点。根据“野犬”的设计，该车能够承受恶劣的路况、[[机枪|机枪]]扫射、小型[[反坦克武器|反坦克武器]]的攻击以及通过[[大规模杀伤性武器|大规模杀伤性武器]]防护测试认证。

== 改型 ==
“野犬2型”是“野犬”的改进型。和“野犬1”相比，“野犬2”仍旧采用了 [[烏尼莫克|烏尼莫克]]U 5000底盘，但提高了防护能力（“野犬2”可以加挂模块式附加装甲）和载荷，並配備了后视摄像机，有利于在城市环境下驾驶车辆。除此之外，野犬2还降低了红外信号特征，在[[紅外線熱像儀|紅外線熱像儀]]前面具有一定的隐身能力。“野犬2型”推出了两个版本：一款为3,250mm[[轴距|轴距]]（3.5吨有效负载），另一款为3,850mm轴距（4吨有效负载）。“野犬2”能容纳8名乘员。目前KMW正在研发内部容纳空间更大的“野犬2 GFF”。

由于载荷和内部空间得到提高，相比起“野犬1”，“野犬2”能够执行更多任务，目前已開發出人员输送车、救护车、货车、指挥控制车、防空车和前线观察车合共6种车型。作为前线观察车时，“野犬2”可安装可伸缩桅杆，桅杆上安装有传感組件、白光/热成像观察设备和护眼型激光测距仪等设备 。<ref name="野犬式全防護車">[http://www.hudong.com/wiki/%E8%A3%85%E7%94%B2%E6%B1%BD%E8%BD%A6]</ref>

== 武装 ==
“野犬”的预设[[武器|武器]]为一挺置于车顶的7.62 mm[[遥控|遥控]][[机枪|机枪]]，这挺机枪也可以用[[白朗寧M2重機槍|12.7 mm机枪]]或[[HK_GMG自動榴彈發射器|HK GMG]]自动[[榴弹发射器|榴弹发射器]]取代。

== 实战纪录 ==
2005年6月3日，一辆隶属联邦德国国防军的“野犬”在波斯尼亚进行[[巡罗|巡罗]]时遭受一枚6千克[[反坦克地雷|反坦克地雷]]攻击，但车内乘员安然无恙。
