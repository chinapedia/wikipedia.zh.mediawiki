'''野田市'''（{{jpn|j=野田市|hg=のだし|rm=Noda shi|f=y}}）是[[千葉縣|千葉縣]]西北部的[[市|市]]。2003年6月6日，{{link-ja|東葛飾郡|東葛飾郡}}'''{{link-ja|關宿町|関宿町}}'''（{{lang|ja|せきやどまち}}）編入後，成為千葉縣最北端的自治體。


== 地理 ==
位于千葉县最北端，約在[[關東平原|關東平原]]中央位置，東有[[利根川|利根川]]、西有[[江戶川|江戶川]]、南有{{link-ja|利根運河|利根運河}}。利根川對岸為[[茨城縣|茨城縣]]，江戶川對岸為[[埼玉縣|埼玉縣]]。

野田地名的由來眾說紛紜，目前較可信的說法是野田右馬助在此建館而得名，但也有「因野田的地名而稱野田右馬助」的說法。

[[東武鐵道|東武鐵道]][[東武野田線|野田線]]「[[野田市站|野田市站]]」周邊是[[龜甲萬|龜甲萬]]創業地，此地每日輸出[[醬油|醬油]]至日本各地。

== 歷史 ==
[[野田市|野田市]]與舊{{link-ja|關宿町|関宿町}}地區的發展歷史不同，因此分為兩部分分別介紹。

=== 野田市 ===
; 江戶時代以前
野田在[[江戶時代|江戶時代]]以前缺乏紀錄，但市內有一些小型[[遺跡|遺跡]]，如野田[[貝塚|貝塚]]、{{link-ja|山崎貝塚 (千葉縣)|山崎貝塚 (千葉県)|山崎貝塚}}等。

至於中世，傳聞此地有活躍於[[後三年之役|後三年之役]]的{{link-ja|鎌倉景政|鎌倉景政}}居城與[[古河公方|古河公方]]重臣野田右馬助居城。但並未發現任何遺跡。

尾崎傳聞是[[鎌倉幕府|鎌倉幕府]][[御家人|御家人]]{{link-ja|野本家定|野本家定}}城館所在地。木野崎則有[[國人眾|國人]]{{link-ja|一色數馬|一色数馬}}城館遺跡。

[[德川家康|德川家康]][[江戶|江戶]]入府後，[[岡部長盛|岡部長盛]]在1590年（[[天正|天正]]18年）於{{link-ja|山崎 (野田市)|山崎 (野田市)|山崎}}建造居城。

; 江戶時代
野田在進入江戶時代後，紀錄逐漸增加。受惠於水運，供應江戶的醬油產業在此興起，成為[[醬油|醬油]]一大產地。陸上交通也因{{link-ja|日光東往還|日光東往還}}，{{link-ja|山崎 (野田市)|山崎 (野田市)|山崎宿}}、中里宿等宿場蓬勃發展。

; 近代以後
* 1889年（明治22年）4月1日
** 野田町、清水村、上花輪村、中野台村、堤台村、櫻台村飛地、今上村飛地合併成立{{link-ja|野田町 (千葉縣)|野田町 (千葉県)|野田町}}。

1970年代以後，在通勤路線[[東武野田線|東武野田線]]的帶動下成為東京的[[臥城|臥城]]。但在1990年代以後受到{{link-ja|都心回歸|都心回帰}}現象的影響，青年人口流出，逐漸出現高齡化。

近年，[[都市再生機構|都市再生機構]]在市南部的{{link-ja|水木 (野田市)|みずき (野田市)|水木}}（舊{{link-ja|山崎 (野田市)|山崎 (野田市)|山崎}}）地區開發野田水木之街（野田みずきの街），加上[[筑波快線|筑波快線]]通車使得前往東京都心更為便利，人口再次增加。

=== 舊關宿町 ===
; 江戶時代以前
室町時代由建築{{link-ja|關宿城|関宿城}}的簗田氏所支配。戰國時代，簗田氏與[[後北條氏|後北條氏]]爆發關宿合戰。

; 江戶時代
江戶時代屬{{link-ja|關宿藩|関宿藩}}，由德川[[譜代|譜代]][[大名|大名]]擔任城主。除了日漸繁榮的關宿城城下町，現在江戶川與利根川分流點因是[[水上交通|水上交通]]的要衝設有[[關隘|關所]]，加上{{link-ja|日光東往還|日光東往還}}的宿場，是下總非常繁華的地區。

; 近代以後
廢藩與船運衰退使得本地失去過去的榮景。

泡沫經濟破滅後，因不在[[東京|東京通勤圈]]內，地價下跌率在縣內名列前茅，人口也持續減少。最大聚落大字木間瀨每年減少超過100人<ref>野田市役所ウェブサイト上の大字別人口統計に基づく</ref>。相反地，南部地區人口逐年上升。

; 舊關宿町年表
* 1889年（明治22年）4月1日－設關宿町、{{link-ja|二川村 (千葉县東葛飾郡)|二川村 (千葉県東葛飾郡)|二川村}}、{{link-ja|木間瀨村|木間ヶ瀬村}}。
* 1955年（昭和30年）7月20日－（舊）關宿町、二川村、木間瀨村合併為'''{{link-ja|關宿町|関宿町}}'''。
* 2003年（平成15年）6月6日－關宿町編入野田市。
