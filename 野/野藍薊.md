'''野藍薊'''（[[學名|學名]]：{{lang|la|''Echium wildpretii''}}）是一種兩年生的[[草本植物|草本植物]]，可以生長達3米高。它們是[[西班牙|西班牙]][[特內里費島|特內里費島]]上的[[特有種|特有種]]，主要生長在[[泰德峰|泰德峰]]。它的俗名為'''寶石塔'''。<ref name=Bramwell>{{Cite book en|author=Bramwell, D and Bramwell, Z|year=2001|title=Wild flowers of the Canary Islands|edition=2nd edition|publisher=Editorial Rueda SL|location=Madrid, Spain|id=ISBN 84-7207-129-4}}</ref>

== 生長環境 ==
野藍薊生長在[[泰德峰|泰德峰]]峽谷的[[亞高山帶|亞高山帶]]。它們需要大量的陽光生長，並會長在乾旱的環境，能抵禦低至零下15℃的[[氣溫|氣溫]]。

== 特徵 ==
野藍薊是兩年生的[[植物|植物]]。頭一年會長出茂密的[[葉子|葉子]]，第二年就會長出直立的[[花序|花序]]，可以高達1-3米。

== 開花 ==
[[File:Echium_wildpretii3.jpg|thumb]]
野藍薊會在[[特內里費島|特內里費島]]的春末至初夏間開花。

== 用途 ==
野藍薊可以作為觀賞植物。其高含量的[[花蜜|花蜜]]及[[花粉|花粉]]也受到養蜂業的青睞。
