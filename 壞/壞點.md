'''壞點'''（{{lang-en|Defective pixel}}）通常是指[[液晶螢幕|液晶螢幕]]面板上的某些[[畫素|螢幕畫素]]全部或部分喪失其作用{{notetag|在[[自動光學檢測|自動光學檢測]]技術上也是瑕疵的一種。}}，而造成變成永久性的'''亮點'''、'''暗點'''與'''色點'''<ref name="HP"/>。液晶面板生產製造時，偶爾會出現有壞點的螢幕，其中部分的亮點可能是黑點、白點、紅點、綠點或藍點等局部的小點產生變化，但即使有壞點依舊可以正常顯示，對於液晶螢幕整體的功能不至於有所影響，但會造成視覺上困擾<ref name="LCD達人"/>。

一般的情況下，若是由於技術因素所產生的壞點是上不可修復的。但是如果是因為保持在一個靜止不動的畫面很長時間所產生的壞點，則有可能使用軟體修復或者擦拭方式去除<ref>[http://lcd.intozgc.com/108/108635_3.html 液晶出现坏点的处理方法-3]{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

== 產生原因 ==
液晶面板內部組成是由小單元的[[像素|畫素點]]，每個畫素並由三個各自發出紅、綠、藍三原色光點（也稱為[[次畫素|次畫素]] 英文為dot或subpixel）所構成，結構簡單但製造過程並不容易<ref name="處理方法"/>。[[三原色|三原色]]混合就能產生人類[[視覺|視覺]]所能接收的色彩，由多個畫素一起發光或發暗就可產生正常[[影像|影像]]。舉例19吋顯示器的解析度是1280 X 1024，其中單元格的數量是3,932,160個（1280 X 1024 X 3），並且每個像素還有一套獨立的[[電晶體|電晶體]]等元件所控制<ref name="HP"/>。
壞點大部分情况下於面板製程时所產生，也有可能是液晶面板自然损耗、内部受過热产生轻微变形或者外部受到不正常[[壓力|壓力]]所導致<ref>[http://www.pangoo.com/article-164.html 液晶电视坏点产生的原因及预防]{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }},Pangoo中国{{zh-cn}}</ref> <ref name="菜鳥學堂"/>。

== 瑕疵種類 ==
液晶面板裡頭，僅是一個色彩的小點無法正常顯示，便會產生所謂的壞點。依據{{link-en|ISO 13406-2|ISO 13406-2}}國際標準，當一個螢幕畫素的[[三原色|三原色]]光點喪失其256種亮度變化時，此瑕疵即會影響該畫素的表現。螢幕畫素瑕疵通常可再細分為'''亮點'''、'''暗點'''、或'''色點'''三種。依照壞點內部的損壞程度，依照[[光|光]]的原理，三原色的光點若全亮則構成白光，反之，三原色光點全不亮則是黑色<ref name="HP"/>。

;暗點：
某個三原色（[[RGB|RGB]]）光點永恆不發光<ref name="HP"/>。

;亮點：
某個三原色（RGB）光點永恆只能發出某個亮度<ref name="HP"/>。

;色點：
某個三原色光點只能表現出一部分的亮或暗，因此色點仍會有不同程度的色彩表現。色點發生的機率遠超過亮點或暗點的機率。

儘管有些供應商提供零壞點保證，但多強調保證無亮點，而少有保證無色點或暗點<ref name="HP"/>。

== 容許標準 ==
正常情況下，面板製造商必須遵循ISO國際標準組織的規範，供應LCD面板給供應商也要符合可靠度的要求。但是以現在的工業生產技術的限制，還無法保證每一個生產出來的液晶螢幕都沒有壞點，因此ISO國際標準組織的規範容許些許程度的不完美<ref name="HP"/>，而且各國的[[國家標準|國家標準]]通常也會對於螢幕壞點有確實的規定範圍，以保證消費者的權益。根據[[ISO13406-2|ISO13406-2]]，對於壞點的容許範圍摘錄如下<ref name="處理方法"/>：

*整體像素瑕疵：
:解析度1024 X 768的液晶螢幕容許值為2個
:解析度1280 X 1024的液晶螢幕容許值為3個
:解析度1600 X 1200的液晶螢幕容許值為4個
:解析度2048 X 1536的液晶螢幕容許值為6個

*紅、綠、藍三者之一子像素的瑕疵：
:解析度1024 X 768的液晶螢幕容許值為4個
:解析度1280 X 1024的液晶螢幕容許值為7個
:解析度1600 X 1200的液晶螢幕容許值為10個
:解析度2048 X 1536的液晶螢幕容許值為16個

*子像素在任意5x5區域內瑕疵：
:解析度1024 X 768的液晶螢幕容許值為2個
:解析度1280 X 1024的液晶螢幕容許值為3個
:解析度1600 X 1200的液晶螢幕容許值為4個
:解析度2048 X 1536的液晶螢幕容許值為6個

面板製造商有不同的壞點判定標準。由於製造商之間的激烈競爭，現時品質控制的標準已經提高。如果液晶螢幕擁有四個或以上的壞點是比較容易察覺到的，因此[[消費者|消費者]]可以據此要求更換新的一台。液晶螢幕的壞點位置同樣是不可忽略的，生產商常會因為壞點出現在螢幕中央區域而降低標準<ref name="菜鳥學堂"/>。

==檢測方法==
要能判斷液晶螢幕的面板畫素是否有瑕疵，必須在光線絕對充足和黑暗的室內環境下，否則難以正確辨識出所有的亮點、暗點、與色點<ref name="HP"/>。
最佳檢測亮點、暗點缺陷的方法，是在畫面螢幕全綠的時候找暗點，畫面螢幕全藍的時候找亮點，這樣可以輕易的找出亮點或暗點。在對液晶屏幕進行檢測的點缺餡時，可連接有安裝檢測軟體的電腦或者檢測軟體，可有效幫助尋找壞點。這些具有相關的功能的軟體有 Nokia Monitor Test、[http://www.phoenixstudio.org/cn/displayx.htm DisplayX]、Monitors Matter CheckScreen等<ref>[http://news.xinhuanet.com/it/2008-05/21/content_8219005_3.htm 致命的物理損傷 液晶面板壞點全解密 ]</ref>。

==其他==
在[[電荷耦合元件|電荷耦合元件]]是[[數位相機|數位相機]]方面，壞點則是指無法正確捕捉光線的感光元件，若面板出現壞點會直接造成成像的暇疵<ref name="CCD"></ref>。
