《'''問答魔法學院'''》（[[日語|日語]]：{{lang|ja|クイズマジックアカデミー}}，[[英語|英語]]：{{lang|en|QUIZ MAGIC ACADEMY}}；簡稱：'''QMA'''與'''{{lang|ja|マジアカ}}'''）是[[日本|日本]][[科樂美|科樂美數位娛樂]]所開發的[[問答遊戲|問答遊戲]][[街机|街機遊戲]]平台系列，以及依此改編的[[OVA|OVA]]以及[[漫畫|漫畫]]。

最大的特徵是使用[[電腦網路|電腦網路]]與其他玩家連線對戰，測驗的問題每週新增不同的問題。問題的修正以及新增功能都藉由網路來進行。

玩家主要扮演魔法學園的學生，並且回答各式各樣的問題。並藉由淘汰賽獲得晉級到上位階級的機會。

== 概要 ==
=== 作品 ===
《'''問答魔法學院'''》採用e-AMUSEMENT，容易新增和修正問題、儲存成績等，最多可以十六人參與遊戲，本作品採用[[觸控式螢幕|觸控式螢幕]]來取代遊戲搖桿與鍵盤的輸入。
在本條目依照整個遊戲系列來做解說，以下略稱為QMA1、QMA2、QMA3、QMA4、QMA5。

能夠以網路連線作軟體更新動作，以及問題的題庫新增以及修正（題目隨著時間的改變而修正。）較為容易，解決了以往問答遊戲無法修正的缺點。

QMA系列的舞台受到了「[[哈利·波特|哈利·波特]]」裡魔法學校的影響<ref>《{{lang|ja|マジックアカデミーへ行こう!～QUIZ MAGIC ACADEMY FAN BOOK～}}》科樂美數位娛樂，2006年，154頁</ref>。玩家為「[[學生|學生]]」。QMA設置的遊樂中心賦予分校地位<ref>「QUIZ MAGIC ACADEMY本校」以及「QUIZ MAGIC ACADEMY分校」為製作團隊除錯用的機器。</ref>。

=== 機體 ===
機體由擁有[[伺服器|伺服器]]功能的中央螢幕，以及遊戲玩家所使用的[[客戶端|客戶端]]（中央螢幕一台最多對十六台）所構成。

用戶端的機體使用[[觸控式螢幕|觸控式螢幕]]，回答時選擇在畫面所表示的文字來回答。機體在配置時採取面對面設計，看不到別人身姿為主。

在用戶端的[[作業系統|作業系統]]採用了[[嵌入式系統|嵌入式系統]]專用的[[Windows_XP_Embedded|Windows XP Embedded]]。在啟動時的畫面顯示「KONAMI Windows XP Embedded」，為了控制觸控式螢幕每台機體都有獨立的[[BIOS|BIOS]]。

在中央螢幕則是採用[[Windows_Server_2003|Windows Server 2003]]（QMA4），各種動作呼叫「center.exe」程式來進行。從QMA2升級到QMA3時連同遊戲系統基板一同被更換（PC平台）<ref>{{cite web|url=http://lyrical-magical.net/diarylog/200606.html#060609|2=|script-title=ja:DoRaGooN PRoJeCT since 2003/12/09 ここに来てPreslerコアのPenDが欲しくなる罠 2006年6月のログ|accessdate=2008-04-09|language=ja|deadurl=yes|archiveurl=https://web.archive.org/web/20080919131440/http://lyrical-magical.net/diarylog/200606.html#060609|archivedate=2008-09-19}}</ref>。
