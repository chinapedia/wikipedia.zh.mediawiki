'''拱頂'''是一種拱形的一個建築學用语，指平行的多個[[拱形|拱形]]製造出的天花板樣式，用於提供天花板的或屋頂的空間<ref name="concise">{{cite web
 |url         = http://concise.britannica.com/ebc/article-9381834/vault
 |title       = Vault
 |publisher   = Encyclopædia Britannica
 |accessdate  = 2007-07-18
 |deadurl     = yes
 |archiveurl  = https://web.archive.org/web/20070522214831/http://concise.britannica.com/ebc/article-9381834/vault
 |archivedate = 2007-05-22
}}</ref>。英語中的拱頂（''vault''）一詞來自於法語''voûte''，而法語中的拱頂一詞則來自於義大利語''volta''。拱頂的部分施加橫向[[推力|推力]]，需要反方向的[[摩擦力|阻力]]。 當在地面上建造拱頂時，地面會提供所需的所有的阻力。 然而，當拱頂在地上空中被建造時，採用各種替換方式來提供所需的阻力。 一個例子是在{{le|桶形拱頂|barrel vault|桶形或連續拱頂}}的情況下使用的較厚的牆壁。 當使用相交的拱頂時，[[扶壁|扶壁]]被用於提供阻力。

最簡單的拱頂是{{le|桶形拱頂|barrel vault}}（也稱為隧道拱頂），其形狀通常為半圓形。 桶形拱頂是連續拱形，長度大於其直徑。 在建造拱時，需要臨時支撐，同時構造{{le|拱石|voussoir}}環並將環放置在適當位置。 直到最重要的拱石，拱心石（keystone）又名拱顶石被定位，拱頂仍不是自我支持的。 在容易獲得木材的情況下，這種臨時支撐是通過由具有半圓形或扇形頭部的木框架桁架組成的中心來提供的，該桁架支撐著拱石直到整個拱形環完成。 有了一个桶形拱頂，然後可以移動定心以支撐下一個環{{sfn|Spiers|1911|p=956}}。

拱頂這種建築形式在羅馬帝國時期得到發展，廣泛出現在中世紀時期的教堂建築當中。
