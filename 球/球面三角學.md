'''球面三角學'''是[[球面幾何學|球面幾何學]]的一部分，主要在處理、發現和解釋[[多邊形|多邊形]] (特別是[[三角形|三角形]]) 在球面上的角與邊的聯繫和關聯。在[[天文學|天文學]]上的重要性是用於計算天體軌道和地球表面與太空航行時的天文導航。

==球面上的線==
在球殼的表面，最短的距離是[[大圓|大圓]]上接近直線的弧線，也就是圓弧的圓心與球殼的球心是同一點。例如：地球上的[[子午線|子午線]]和[[赤道|赤道]]都是大圓。所謂[[行星|行星]]表面的直線，就是球面上兩點之間最近距離的大圓弧線（如果把自己拘束在球面上的直線上）。

在球面上，由大圓的[[弧|弧]]所包圍的區域稱為球面[[多邊形|多邊形]]，但要注意，不同於平面上的情形，在球面上[[二角形|二角形]]是可能存在的。（兩個弧夾出兩個角的三角形類似物）

這些多邊形的邊長(弧長)，可以利用球心角很方便的來測定，將弧的兩端所對應的球心角乘上半徑便是邊長。要注意的是，這些角都必須用[[弳度量|弳度量]]來量度。.

因此，對一個'''球面三角形'''而言，是由他的弧長與球心角來具體描述的，只是弧的長度是用弳度量來標示。

值得注意的是，球面三角形的三個內角的和總是大於180°，但在平面上只有180°。超過180°的數值稱為'''球面剩餘''' E：E = α + β + γ - 180°，這些結餘給出了球面三角形的面積。確定這個值，球面剩餘必須以弳度量來測定，表面積A依據球面的半徑和球面剩餘來測量：
:A = R<sup>2</sup> · E
這是[[高斯-博內定理|高斯-博內定理]]，這很明顯的顯示沒有相似的球面三角形(三角形有相同的角，但-{邊長和面積}-不同)。而在特殊的情況下，球的半徑為1，則球面三角形的面積A = E。

要解球面幾何的問題，要點是能剖析出其中的''直角三角形''(三個角中有一個是90°)，因為這樣就可以利用[[約翰·納皮爾|納皮爾]]的多邊形求解。

== 恆等式 ==
球面三角形滿足[[球面餘弦定理|球面餘弦定理]]
:<math>\cos c= \cos a \cos b + \sin a \sin b \cos C \!</math>

這個恆等式的証明需要利用平面的[[餘弦定理|餘弦定理]]和球面三角形的對角"C"延升的切線，而且，在小角度時可以引用平面幾何的公式。

他也滿足並且有相似於平面形式的[[正弦定理|正弦定理]]
:<math>\frac{\sin a}{\sin A}=\frac{\sin b}{\sin B}=\frac{\sin c}{\sin C}.</math>

更詳盡的公式列表可以點選：[http://mathworld.wolfram.com/SphericalTrigonometry.html 此處]
