'''與魔鬼共騎'''（{{lang-en|Ride with the Devil}}）1999年公映的[[美国|美国]]剧情片[[电影|电影]]。根据{{le|丹尼尔·伍德瑞尔|Daniel Woodrell}}的小说《Woe to Live On》改编，剧本由[[詹姆斯·夏慕斯|詹姆斯·夏慕斯]]完成。小说和电影都反映了在[[南北战争|南北战争]]时在[[密苏里|密苏里]]不断升级的[[游击战|游击战]]。

该片在1999年上映6周，票房仅有635,096美元，相比三千八百万美元的制作成本，可谓惨败。

==剧情==
[[19世纪60年代|19世纪60年代]]，[[美国内战|美国内战]]一触即发。1861年，美国成立了[[美利坚联盟国|美利坚联盟国]]，南方人自恃经济力量的雄厚，许多南方人开始用[[游击战|游击战]]来挑动内战的敏感神经。[[堪萨斯州|堪萨斯州]]及[[密苏里州|密苏里州]]的边境，两个儿时玩伴杰格·罗德（[[托贝·马奎尔|托贝·马奎尔]]饰演）和杰克·布尔·柴斯（[[斯基特·奥里奇|斯基特·奥里奇]]饰演）不经意间卷入了战争。杰格是在[[密苏里州|密苏里州]]土生土长的[[德国|德国]]裔，其父亲原本希望他支持北方军队，但是他却违背了父亲的意愿，和杰克一起加入了南方军队，他们在队长布莱克·约翰（[[詹姆斯·卡维泽|詹姆斯·卡维泽]]饰演）的领导下，和队友彼特·迈克森（[[乔纳森·雷·麦耶斯|乔纳森·雷·麦耶斯]]饰演）、乔治·克莱德（[[西蒙·贝克|西蒙·贝克]]饰演）、[[黑奴|黑奴]]丹尼尔·赫特（[[杰弗瑞·莱特|杰弗瑞·莱特]]饰演）一起在[[边境|边境]]地界开始了漫长的[[游击战争|游击战争]]，直到寒冷冬季的来临。南方军队在游击战时，杰克认识了年轻貌美的寡妇寡妇苏·李·雪莱（[[珠儿·克尔奇|珠儿·克尔奇]]饰演），苏刚刚因为战争失去丈夫，寂寞难耐，于是两人迅速坠入爱河，但是战争却没有能够给他们过上幸福甜蜜生活的机会，他们不得不东躲西藏，为生活奔走异乡，最后在[[南北战争|南北战争]]结束时，他们开始组建了新生活。<ref>{{cite web |url= http://data.ent.sina.com.cn/movie/moviedetail.php?id=4241&detail=intro|title= 与魔鬼共骑|accessdate= 2005年|work=|publisher=新浪网 |language = zh-cn}}</ref>

== 花絮 ==
导演[[李安|李安]]解释片名的含义时说：“那一群打游击的南军是史上有名的残暴，所以大家都称他们‘[[魔鬼|魔鬼]]’，当然他们也是史上最有名的[[骑士|骑士]]。”该片是导演李安的第六部作品，情节大部分发生在寒冷的冬季，题材选取了几个年轻人为幸福生活而不懈努力的故事，打破了以往[[南北战争|南北战争]]一贯的白描战争的老旧手法。<ref>{{cite web |url=http://data.ent.sina.com.cn/movie/moviedetail.php?id=4241&detail=shortcomment |title=与魔鬼共骑 |accessdate=2005年 |work= |publisher=新浪网 |language=zh-cn |archive-url=https://web.archive.org/web/20160304194900/http://data.ent.sina.com.cn/movie/moviedetail.php?id=4241&detail=shortcomment |archive-date=2016-03-04 |dead-url=no }}</ref>
