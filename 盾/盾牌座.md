'''盾牌座'''，'''Scutum'''，（[[拉丁文|拉丁文]]"[[盾|盾]]"）是现代88[[星座|星座]]中的一个小星座。它被幾個大星座包圍，包括[[天鷹座|天鷹座]]、[[人馬座|人馬座]]及[[巨蛇座|巨蛇座]]。盾牌座包含中国[[星官|星官]]：天弁。

== 历史传说 ==
盾牌座並無有關之神話故事，這星座代表了[[波蘭|波蘭]]國王[[约翰三世_(波蘭)|約翰三世]]的"揚尼那[[盾章|纹章]]"。其名字和[[后发座|后发座]]一样和真实的历史人物有关。原来的名字'''Scutum Sobiescianum'''（蘇比斯基之盾）是[[赫维留|赫维留]]在1683年后期为了纪念[[维也纳保卫战|维也纳保卫战]]中带领基督教军队的的波兰英雄和国王[[約翰三世_(波蘭)|约翰三世]]而起的。

== 特點 ==
盾牌座以天區面積計名列倒數第五，只佔109平方度。它的亮星不多，其主星是[[盾牌座α|盾牌座α]], [[視星等|視星等]]3.85. [[銀河|銀河]]穿越這個星座，[[盾牌座星雲|盾牌座星雲]]可於盾牌座的東北角邊界找到。

== 深空天體 ==
盾牌座包含幾個[[疏散星團|疏散星團]]、一個[[球狀星團|球狀星團]]及一個[[行星狀星雲|行星狀星雲]]。

當中兩個[[疏散星團|疏散星團]]被收錄於[[梅西爾天體列表|梅西爾天體列表]]，包括M11"[[野鴨星團|野鴨星團]]"（NGC 6705）和[[M26|M26]]星團（NGC 6694）。

盾牌座的東部可找到[[球狀星團|球狀星團]][[NGC_6712|NGC 6712]]及[[行星狀星雲|行星狀星雲]][[IC_1295|IC 1295]]，兩者相距24[[弧秒|弧秒]]
