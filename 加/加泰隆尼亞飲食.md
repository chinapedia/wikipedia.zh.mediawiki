'''加泰隆尼亞飲食'''是指西班牙[[加泰隆尼亞|加泰隆尼亞]]地區的飲食文化。在8世紀之後，摩爾人曾統治加泰隆尼亞的部分地區，他們將砂糖、香辛料、茄子、[[菜薊|菜薊]]、大米、義大利麵等食材帶入加泰隆尼亞。[[摩爾人|摩爾人]]到來之前，[[猶太人|猶太人]]也曾來到加泰隆尼亞。外國人給加泰隆尼亞的飲食帶來很大影響<ref name=telegraph>{{cite web |url=http://www.telegraph.co.uk/foodanddrink/9156020/Catalan-cuisine-through-history.html |title=Catalan cuisine through history |publisher=Telegraph |date=2012-03-24 |accessdate=2016-02-07}}</ref>。。中世紀時，阿拉貢-加泰隆尼亞王國曾支配地中海大部分地區，使得加泰隆尼亞飲食在歐洲擁有影響力。同時，加泰隆尼亞飲食也開始受到法國飲食的影響。在弗朗哥執政時期，加泰隆尼亞獨特的飲食文化曾受到壓制。不過在弗朗哥政權結束之後，加泰隆尼亞人已經開始重新認識他們的飲食文化。加泰隆尼亞飲食以蔬菜、肉類、魚類和水果為主要原料<ref name=lonely>{{cite web |url=http://www.lonelyplanet.com/spain/barcelona/travel-tips-and-articles/68463 |title=An introduction to Catalan cuisine |publisher=Lonely Planet |accessdate=2016-02-07}}</ref>。加泰羅尼亞北部的飲食文化較為傳統，而南部則較多使用香辛料，最有名的是[[西班牙海鮮飯|西班牙海鮮飯]]。<ref name=gencat>{{cite web |url=http://www.gencat.cat/culturcat/portal/site/culturacatalana/menuitem.be2bc4cc4c5aec88f94a9710b0c0e1a0/en_GB/indexbc02.html?vgnextoid=37d3ef2126896210VgnVCM1000000b0c1e0aRCRD&vgnextchannel=37d3ef2126896210VgnVCM1000000b0c1e0aRCRD&vgnextfmt=detall2&contentid=5c48161da99d7210VgnVCM1000008d0c1e0aRCRD&newLang=en_GB |title=Cultura Catalana |publisher=加泰隆尼亞政府觀光局 |accessdate=2016-02-07 |deadurl=yes |archiveurl=https://web.archive.org/web/20160616173808/http://www.gencat.cat/culturcat/portal/site/culturacatalana/menuitem.be2bc4cc4c5aec88f94a9710b0c0e1a0/en_GB/indexbc02.html?vgnextoid=37d3ef2126896210VgnVCM1000000b0c1e0aRCRD&vgnextchannel=37d3ef2126896210VgnVCM1000000b0c1e0aRCRD&vgnextfmt=detall2&contentid=5c48161da99d7210VgnVCM1000008d0c1e0aRCRD&newLang=en_GB |archivedate=2016-06-16 }}</ref>。
