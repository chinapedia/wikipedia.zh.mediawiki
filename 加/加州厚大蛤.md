'''加州厚大蛤'''（[[学名|学名]]：{{lang|la|'''''Epilucina californica'''''}}）<ref name = "col3"/><ref name = "col402"/><ref name = "col418"/>，又名'''加州满月蛤'''，是[[帘蛤目|帘蛤目]][[满月蛤科|满月蛤科]]的一种<ref name = "COL"/><ref name = "source"/>，原屬[[厚大蛤屬|厚大蛤屬]]<ref>[http://www.biolib.cz/en/taxon/id521855/ Biolib]</ref>，今單獨獨立出來成為''[[Epilucina|Epilucina]]''屬<ref name="WoRMS_464045"/>。
''[[Epilucina|Epilucina]]''屬本來就只有加州厚大蛤這一個物種，後來才再加入另一個[[化石種|化石種]][[華盛頓厚大蛤|華盛頓厚大蛤]]<ref name="WoRMS_464045"/>。

根據[[Catalogue_of_Life|Catalogue of Life]]，本物種並沒有亞種<ref name = "COL"/>。
