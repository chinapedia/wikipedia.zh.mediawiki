'''加德納'''（{{lang-he|גדנ"ע}}）是[[以色列|以色列]]的一個軍事計畫，內容是要求該國青年在名叫「[[:en:Nahal|納哈爾]]」的一個[[步兵|步兵]]旅，為預備參加[[以色列國防軍|以色列國防軍]]和[[:en:Israel_Border_Police|邊防警察]]的[[義務兵役制|義務兵役]]，進行為期一周的軍事和紀律訓練。每年約有19,000名本國青年受訓，而外國青年也可以參加。<ref name="haaretz">{{Cite news|url=http://www.haaretz.com/hasen/spages/807850.html|title=New IDF Gadna youth program criticized as overly militaristic|publisher=[[Haaretz|Haaretz]]|author=Kashti, Or|date=2007-01-01|accessdate=2008-08-15}}</ref>
== 歷史 ==
'''加德納'''是'''Gdudei No'ar'''（{{lang-he|גדודי נוער}}）的縮寫，[[以色列獨立宣言|以色列獨立]]前就已經成立了<ref>{{Cite web|url=http://www.zionism-israel.com/dic/Gadna.htm|title=Gadna|publisher=Zionism-Israel.com|accessdate=2008-08-15}}</ref>，其最早的目的是要培養滿足各種需要的戰鬥人員。受訓者投入了1948年的[[第一次中東戰爭|阿以戰爭]]，尤其是在[[耶路撒冷|耶路撒冷]]的前線。

== 服裝 ==
[[File:PikiWiki_Israel_2886_Gelil_1963_תמונת_מחזור_מאימון_גדנע_בגליל_(גליל_ים)_1963.jpg|thumb]]
加德納計畫裡，指揮官、警官與軍官需配戴[[:en:Nahal|納哈爾步兵旅]]的綠色[[貝雷帽|貝雷帽]]，但是帽子上的納哈爾徽章以及衣服肩膀上的納哈爾標記，則用[[:en:Education_and_Youth_Corps|教育暨青年軍團]]的徽章和標記取代。

指揮官在他們的左肩配戴飾帶。受訓者配發工作服、B級制服、軍用腰帶、水壺和水壺袋。入訓時，每人還會發一頂鴨舌帽，必須隨時配戴。女性則必須束髮。

參加計畫的青年被分成數小隊，並住在[[帳篷|帳篷]]或軍隊營房。每一個小隊會選出兩個青年指揮官，最高的青年指揮官會在他衣服的左肩繡上多種顏色條紋的肩章。第二高的青年指揮官是[[中士|中士]]，他衣服的左肩會繡上全黑的肩章。

== 結構組織 ==
自'''加德納計畫'''成立以來，就有不少軍事基地是計畫執行的場所，但時代變遷，截至2008年底，僅剩3個基地依舊存在。<ref name="bamahane3014">{{cite journal|title=什麼是加德納 （So What Exactly is Gadna?）|author=Danieli, Michal|journal=[[Bamahane|Bamahane]]|date=November 27, 2009|issue=3014|page=p. 34}} {{he icon}}</ref>

===現役基地===
依基地規模大小排列：

* 博克加德納基地（Sdeh Boker Gadna Base），位於以色列 [[內蓋夫_(以色列)|內蓋夫沙漠]]。
* 泰西門加德納基地（Tzalmon Gadna Bas），位於以色列 [[:en:Lower_Galilee|加利利下]]。
* 捷諾加德納基地（Joara Gadna Base），位於以色列 [[:en:Plain_of_Manasseh|馬拿西平原]]。

以色列國防軍預計會在2009年至2010年對這些基地進行設備升級和改造計畫。<ref name="bamahane">{{Cite web|url=http://dover.idf.il/IDF/News_Channels/today/08/08/1206.htm |author=Belekh, Dor |title=Serving the Gadna |publisher=[[Bamahane|Bamahane]] |date=2008-08-13 |accessdate=2008-08-15 |deadurl=yes |archiveurl=https://web.archive.org/web/20080917215329/http://dover.idf.il/IDF/News_Channels/today/08/08/1206.htm |archivedate=2008-09-17 }} {{he icon}}</ref>

== 青年在軍營的生活 ==
[[File:Gadna.JPG|thumb]]]]
在這為期一周的訓練裡[[以色列|當地]]青年與外國青年所受的訓練是差不多的，不過當地青年要接受的訓練更嚴苛，甚至時間更長，以達到[[以色列國防軍|以色列國防軍]]的基本要求。'''加德納'''參與者過著早睡晚起的生活。他們的課程包括如何拆解清潔武器、學會使用[[M16突擊步槍|M16突擊步槍]]。他們還必須了解[[以色列軍事|以色列軍事的歷史]]與國防軍軍歷代著名人物。參與者也得做些維護基地的工作，像是清潔環境和在廚房做事。
