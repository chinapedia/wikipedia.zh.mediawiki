'''加里宁格勒州'''（{{lang-rus|Калинингра́дская о́бласть|Kaliningradskaya oblast}}），又譯'''卡-{里}-宁格勒州'''，位於[[波羅的海|波羅的海]]沿岸[[立陶宛|立陶宛]]西方的[[小立陶宛|小立陶宛]]。

加里寧格勒州是[[俄羅斯|俄羅斯]]最西部的一個行政區，面積15,100平方公里（陸地面積13,300平方公里），人口941,873人（2010年人口普查），[[首府|首府]]為[[加里寧格勒|加里寧格勒]]，该州与立陶宛、波兰接壤，和俄罗斯本土完全不相連，是俄罗斯的[[外飞地|外飞地]]。該地原為[[普魯士|普魯士]]的主要部份，[[德意志統一|德意志統一]]後成為了[[德意志帝國|德意志帝國]][[東普魯士|東普魯士]]的一部分，[[二戰|二戰]]后割讓[[蘇聯|蘇聯]]，屬[[俄羅斯蘇維埃聯邦社會主義共和國|俄羅斯蘇維埃聯邦社會主義共和國]]管轄。1991年[[蘇聯解體|蘇聯解體]]后，原本屬蘇聯一部份的[[立陶宛|立陶宛]]和[[白俄罗斯|白俄罗斯]]脫離獨立，導致加里寧格勒與俄羅斯本土分開，而成為該國的[[外飛地|外飛地]]。這個城市看不出其被德國統治時的歷史，因為很多在普魯士興盛時期興建的建築物都在二戰中被炸毀，當地僅剩的德國建築是經過修復的大教堂。

加里寧格勒州原為東普魯士的一部分，中心城市是[[哥尼斯堡|哥尼斯堡]]，[[二戰|二戰]]結束之後，根據蘇美英三國簽署的[[波茨坦協定|波茨坦協定]]，哥尼斯堡和東普魯士北部地區劃歸[[蘇聯|蘇聯]]版圖。1946年，這片土地成為了[[俄羅斯蘇維埃聯邦社會主義共和國|俄羅斯蘇維埃聯邦社會主義共和國]]的一個州，為了紀念不久前去世的[[蘇聯最高蘇維埃主席團|蘇聯最高蘇維埃主席團]]主席[[加里寧|加里寧]]，該州改名為加里寧格勒州，亦將該州的首府名訂為加里寧格勒。蘇聯政府強行將這裡的[[德國|德國]]人遷走，同時遷入大批俄羅斯人。

1990年3月，[[立陶宛|立陶宛]]宣布獨立，因此俄羅斯本土和加里寧格勒州被分開。該州居民現在如要前往俄羅斯，通常會利用海空交通工具，但如果居民走陸路進出俄羅斯，就必須經過立陶宛和[[白俄羅斯|白俄羅斯]]。2004年，[[波蘭|波蘭]]和[[立陶宛|立陶宛]]加入[[歐盟|歐盟]]後，俄羅斯人必須取得歐盟的簽証才能進出加里寧格勒。

== 歷史 ==
加里寧格勒州成立於1945年，首府加里寧格勒原名為[[哥尼斯堡|哥尼斯堡]]，大約在13世紀建立，隨著德國商人[[漢撒同盟|漢撒同盟]]的興起，哥尼斯堡也繁榮起來。

在15世紀末，哥尼斯堡已擁有約100多艘商船，在1660年前哥尼斯堡由[[普魯士|普魯士]]控制。

1701年，普魯士首位國王[[腓特烈一世_(普魯士)|腓特烈一世]]在這兒的大教堂加冕，並將哥尼斯堡定為[[普魯士王國|普魯士王國]]的首都。哥尼斯堡當時成為[[普魯士|普魯士]]和[[俄羅斯帝國|俄羅斯]]貿易的中心，並一直作為普魯士的首都，直至普魯士[[遷都|遷都]]至[[柏林|柏林]]為止。

哥尼斯堡的著名教育中心，阿爾貝蒂娜大學在1544年成立，這間大學最著名的學者是[[哲學家|哲學家]][[康德|康德]]，他大部份的思想均在此形成。

哥尼斯堡在[[20世紀|20世紀]]遭逢巨變，[[第一次世界大戰|第一次世界大戰]]德國戰敗，戰勝國決定劃出「[[波蘭走廊|波蘭走廊]]」，令[[波蘭|波蘭]]可連接沿海地區，但這決定使哥尼斯堡和[[東普魯士|東普魯士]]跟德國本土隔離了。

[[第二次世界大戰|第二次世界大戰]]後，[[蘇聯|蘇聯]]、[[美國|美國]]和[[英國|英國]]在[[波茨坦會議|波茨坦會議]]上決定取消[[東普魯士|東普魯士]]，其北部劃歸為蘇聯的一部分。

1946年4月7日，蘇聯[[最高蘇維埃主席團|最高蘇維埃主席團]]通過了成立[[俄羅斯蘇維埃聯邦社會主義共和國|俄羅斯]]哥尼斯堡州的命令。7月4日，行政中心以該主席團的其中一名主席[[加里寧|加里寧]]之名命名，改為加里寧格勒，州名亦改為加里寧格勒州。

自此，加里寧格勒再也沒有德國人，有些在[[第二次世界大戰|戰爭]]中死亡，有些則逃走了，其餘都被蘇聯放逐至[[東德|東德]]或[[西伯利亞|西伯利亞]]，取而代之的是大量遷入的俄羅斯人，他們大多在附近的蘇聯[[波羅的海艦隊|波羅的海艦隊]]總部工作。

== 政治 ==
加里寧格勒州[[杜馬|杜馬]]和州政府行使國家權力，是根據俄羅斯聯邦[[憲法|憲法]]、聯邦立法和州章程成立的機構，加里寧格勒州杜馬是常設國家權力機構。

州杜馬議員（32名）由常駐該州，擁有選舉權的俄羅斯公民通過無記名投票方式選舉出來，任期4年。

州長是領導常設執行權力機構。

== 地理和氣候 ==
[[File:KaliningradOB.png|thumb]]，歷史上曾經是[[普魯士|普魯士]]東北面的一個地區。]]
加里寧格勒州是俄羅斯聯邦最小的州。首府加里寧格勒瀕臨[[波羅的海|波羅的海]]，人口45萬，數目幾乎接近全州的一半。加里寧格勒州主要城市還有[[波羅的斯克|波羅的斯克]]、[[蘇維埃斯克|蘇維埃斯克]]等。從加里寧格勒到[[華沙|華沙]]距離為400公里，到[[柏林|柏林]]、[[哥本哈根|哥本哈根]]、[[斯德哥爾摩|斯德哥爾摩]]的距離均在600公里左右。州內的行政單位數量有13個區、8個州屬市、14個區屬市、5個市內區、5個城鎮、97個行政農庄。加里寧格勒州是俄羅斯的重要[[海軍|海軍]]基地，[[波羅的海艦隊|波羅的海艦隊]]司令部就設在此。

加里寧格勒州與[[立陶宛|立陶宛]]共管[[庫爾斯沙嘴|庫爾斯沙嘴]]。在北部和東部，它與立陶宛接壤（邊界線長280.5公里），在南部，與波蘭交界（邊界線長約232公里）。波羅的海海岸線大約184公里。加里寧格勒州從東到西的最遠距離為205公里；從北到南——108公里。加里寧格勒市到波蘭邊界的距離為35公里，到立陶宛邊界的距離為70公里。

加里寧格勒市離最近的[[俄羅斯|俄羅斯]]州中心——普斯科夫的距離為800公里，離[[莫斯科|莫斯科]]的距離為1289公里。而與歐洲許多國家首都的距離卻較近（距[[立陶宛|立陶宛]][[維爾紐斯|維爾紐斯]]——350公里；距[[拉脫維亞|拉脫維亞]][[里加|里加]]——390公里；距[[波蘭|波蘭]][[華沙|華沙]]——400公里；距[[德國|德國]][[柏林|柏林]]——600公里；距[[瑞典|瑞典]][[斯德哥爾摩|斯德哥爾摩]]——650公里；距[[丹麥|丹麥]][[哥本哈根|哥本哈根]]——680公里；距[[挪威|挪威]][[奧斯陸|奧斯陸]]——850公里） 

加里寧格勒州的大部分位於普列戈利亞河流域，西部及西南部環繞著[[波羅的海|波羅的海]]和波羅的灣。地形以低[[平原|平原]]、[[沼澤|沼澤]]和平緩丘陵為主，百分之二十的土地生長著混生林。受[[海洋|海洋]]影響，氣候濕潤，年降雨量600-700毫米。1月平均氣溫-9℃，7月17℃。[[土壤|土壤]]以灰化土為主。礦藏有[[石油|石油]]、[[褐煤|褐煤]]、[[泥炭|泥炭]]等。種植業不發達，唯亞麻種植較多。畜牧、家禽業較為發達。加里寧格勒州內主要的河流有涅曼河、普列戈利亞河。在該州境內還分布著大大小小的[[湖泊|湖泊]]一百多個，還有許多[[沼澤|沼澤]]地。加里寧格勒州的氣候已由海洋性氣候向溫帶大陸性氣候逐漸過渡。

== 經濟 ==
加里寧格勒地區是[[北歐|北歐]]以及[[波羅的海|波羅的海]]地區經濟最為落後的地方。考慮到加里寧格勒州特殊的地理位置，為了盡量吸引外資，俄羅斯國家[[杜馬|杜馬]]議會批准加里寧格勒州經濟特區法（1996年生效），該項法律規定加里寧格勒州全境將不再征收任何進出口關稅，從而使該州成為一個無關稅地區，確定了特殊的、優惠的投資和經營活動制度，其中包括對外經營活動。經濟特區法的有效期為25年。有效期內，如果俄羅斯進入戰爭狀態或政府宣布國家進入緊急狀態時間長達3個月時，國家杜馬就有權中止該經濟特區法的實施。由於上述州經濟特區法律的實施，州對外貿易額1998年比1995年提高了70％。加里寧格勒1996年成為特別經濟區，享有稅務優惠，吸引到[[南韓|南韓]][[KIA|KIA]]和[[德國|德國]][[BMW|BMW]]車廠到來設廠；2004年加里寧格勒地區的經濟增長達百分之11.5，工業產量增長百分之25.8。

加里寧格勒是俄羅斯的貿易港和漁港，工業的基礎領域有造紙、燃料、機械制造、食品、輕工業，也有造船業和[[琥珀|琥珀]]開採加工業。琥珀生產儲量佔世界儲量的百分之九十。每年生產琥珀多達四百多噸，產品多數出口到[[日本|日本]]、[[波蘭|波蘭]]、[[立陶宛|立陶宛]]和[[德國|德國]]。加里寧格勒州自然資源比較缺乏，工廠所需百分之九十的原材料主要靠鐵路和汽車從俄羅斯本土運進，而百分之八十的電能都是來自俄羅斯本土。機械製造企業大部分都集中在加里寧格勒市，包括有造船業，車廂、起重機、照明設備、以及其他複雜的技術設備。

由於[[歐盟|歐盟]][[簽証|簽証]]政策對俄羅斯非常嚴苛，俄羅斯方面認為仿效20世紀70年代中期在[[西柏林|西柏林]]和[[西德|西德]]其它地區間實行的無簽証制度，但這個建議被歐盟否決。歐盟認為，一旦加里寧格勒居民獲得特殊照顧，俄羅斯公民都可以利用這個缺口進入歐盟國家。俄羅斯又要求經波蘭和立陶宛開設連接加里寧格勒與俄羅斯本土的運輸走廊，且在波立兩國入盟後為俄羅斯居民發放一日簽証，以方便他們自由來往於加里寧格勒和俄羅斯本土之間，但歐盟目前並沒有對此建議加以正視。

== 人口結構 ==
加里寧格勒州人口有94.68萬人，其中城市人口佔77％，農村人口佔23％。平均人口密度每平方公里63人。女性佔51.7％，男性佔48.3％。勞動人口——60.9％，兒童——19.6％，退休人員和老年人——19.5％。

州内人口屬於[[多民族|多民族]]地區，有30個大小民族，其中:
* [[俄羅斯族|俄羅斯族]]——佔78.1％
* [[白俄羅斯族|白俄羅斯族]]——佔7.7％
* [[烏克蘭族|烏克蘭族]]——佔7.4％
* [[立陶宛人|立陶宛人]]——佔1.9％
* [[亞美尼亞族|亞美尼亞族]]——佔0.8％
* [[德意志族|德意志族]]——佔0.6％
* [[波蘭人|波蘭人]]——佔0.5％。

== 運輸 ==
[[File:TramKaliningrad.jpg|thumb]]

[[哈拉波罗夫国际机场|哈拉波罗夫国际机场]]位於加里寧格勒市北部24公里，該機場主要連接加里寧格勒到俄羅斯其他城市，而且還提供航班到[[西歐|西歐]]某些[[城市|城市]]和[[以色列|以色列]]。在[[波羅的斯克|波羅的斯克]]可以坐船前往[[聖彼得堡|聖彼得堡]]，[[哥本哈根|哥本哈根]]，[[里加|里加]]和[[基爾|基爾]]。

在德國統治時期，中央火車站稱為哥尼斯堡中央火車站。現稱為Passazhirskiy國際中央火車站。火車提供班次可以到達[[馬爾堡|馬爾堡]]，[[柏林|柏林]]，[[波羅的斯克|波羅的斯克]]，[[莫斯科|莫斯科]]，[[聖彼得堡|聖彼得堡]]，[[明斯克|明斯克]]，[[哈爾科夫|哈爾科夫]]，[[阿納帕|阿納帕]]和[[巴格拉季奧諾夫斯克|巴格拉季奧諾夫斯克]]等各大城市。

目前市中心位於勝利廣場的區域火車站，可以到達[[澤列諾格拉茨克|澤列諾格拉茨克]]和[[斯韋特洛戈爾斯克|斯韋特洛戈爾斯克]]，並每天一次開至[[蘇維埃茨克|蘇維埃茨克]]。在1881年，哥尼斯堡已開始使用有軌[[電車|電車]]，一直運行至今。而在1975年無軌[[巴士|巴士]]系統亦投入服務。

== 「波羅的海的香港」計劃 ==
自從[[蘇聯解體|蘇聯解體]]後，多個[[蘇聯加盟共和國|蘇聯加盟共和國]]都相繼宣告獨立。而地處[[波蘭|波蘭]]以北的加里寧格勒和俄羅斯本土分隔了，故令當地陷入嚴重經濟危機，失業率高企，以及很多年輕人離開。但自從[[普京|普京]]總統上台之後，他致力於把這個經濟落後的地方發展成“[[波羅的海|波羅的海]]的[[香港|香港]]”。普京總統將加里寧格勒列為[[經濟特區|經濟特區]]，鼓勵外商投資，新投資者可以享有6年豁免利得稅的優惠，[[莫斯科|莫斯科]]還致力促進加里寧格勒與[[歐盟|歐盟]]合作，並建成一個國際的機場，令這個加里寧格勒城成為俄羅斯當中十個城市連繫英國、德國、意大利和西班牙等地方。

加里寧格勒的發展，很大程度上與這裏的港口有關。加里寧格勒港口是俄羅斯唯一一個不結冰的波羅的海港口（[[不凍港|不凍港]]），現在每年處理至少800萬噸[[貨櫃|貨櫃]]。海事商業港口局計劃擴大當地貨倉，並興建新的貨櫃碼頭。當地不少俄羅斯商人亦開始在沿海興建不少豪宅。普京又計劃，於2009年把加里寧格勒定為俄羅斯四個[[賭博|賭博]]合法化的地方之一，到時當地將會有賭場區，區內將有3萬多個酒店房間，令這裏變成為“波羅的海的[[拉斯維加斯|拉斯維加斯]]”。為了進一步推動旅游業，俄羅斯當局還計劃，重建加里寧格勒的歷史風貌以吸引更多世界各地的游客。

== 軍事 ==
[[File:Baltic_fleet_headquarters.jpg|thumb]]加里寧格勒州曾經是俄羅斯聯邦軍事化最重的地區，其軍事設施及密度亦是全歐洲最多最高的。[[前蘇聯|前蘇聯]][[波羅的海軍區|波羅的海軍區]]總部曾經設在此地。加里寧格勒現在還作為俄羅斯波羅的海艦隊的總部，為[[切爾尼亞霍夫斯克|切爾尼亞霍夫斯克]]，[[頓斯科耶|頓斯科耶]]和加里寧格勒契卡洛提供空軍基地及海軍空軍基地。

俄时任總統[[迪米悌·阿納托利耶維奇·梅德維傑夫|梅德韋傑夫]]在2008年11月在俄羅斯議會發表國情咨文時表示可能會在加里寧格勒州部署[[伊斯坎德爾|“伊斯坎德爾”導彈]]。他聲稱這是為了應對美國可能在波蘭和捷克部署[[反導系統|反導系統]]。鑒於[[美國|美國]]總統[[奧巴馬|奧巴馬]]在2009年9月17日宣佈放棄在[[捷克|捷克]]和[[波蘭|波蘭]]部署反導系統，[[俄羅斯|俄羅斯]]亦準備採取回應，暫時沒有打算在加里寧格勒州部署“伊斯坎德爾”導彈。
