'''加太線'''（{{jpn|j=加太線|hg=かだせん|rm=Kada sen|f=y}}），是一條連接[[和歌山縣|和歌山縣]][[和歌山市|和歌山市]]{{link-ja|紀之川站|紀ノ川駅}}與{{link-ja|加太站 (和歌山縣)|加太駅 (和歌山県)|加太站}}，屬於[[南海電氣鐵道|南海電氣鐵道]]的鐵路路線。
==歷史==
* 1912年（明治45年）6月16日，加太輕便鐵道從[[和歌山口|和歌山口]]到[[加太|加太]]間開始營業。
* 2016年4月29日，觀光列車「めでたいでんしゃ」開始運行<ref>{{cite web |url=https://news.mynavi.jp/article/20160417-a076/|title=南海電鉄、加太さかな線「めでたいでんしゃ」は4/29運行開始 - 記念企画も|publisher=マイナビニュースについて}}</ref>。
* 2017年10月7日，觀光列車「めでたいでんしゃ」水藍色版本開始運行<ref>{{cite web |url=https://news.mynavi.jp/article/20171007-a047/|title=南海加太線、水色「めでたいでんしゃ」10/7デビュー! 水族館のような電車に|publisher=マイナビニュースについて}}</ref>。

== 路線資料 ==
* 路線距離（{{link-ja|營業距離|営業キロ}}）：9.6公里
* [[軌距|軌距]]：1067毫米
* 站數：8個（包括起終點站）
* 複線路段：沒有（全線[[單線鐵路|單線]]。但是紀之川站合流部分為[[複線鐵路|複線]]）
* 電氣化路段：全線電氣化（直流1500V）
* [[閉塞_(鐵路)|閉塞方式]]：自動閉塞式
* 最高速度：50公里/小時

== 車站列表 ==
* 所有車站都位於[[和歌山縣|和歌山縣]][[和歌山市|和歌山市]]。
* 只限普通車運行，所有列車各站停車。
* ※：和歌山市－紀之川為[[南海本線|南海本線]]
* 除西之-{庄}-站以外所有車站均可進行[[列車交會|列車交會]]。
