'''徐家汇圣衣院'''（{{lang-fr|'''Zi-Ka-Wei Convent des Carmelites'''}}）又称'''苦修院'''，是[[加尔默罗会|加尔默罗会]]在中国[[上海|上海]]建立的一个[[修女|修女]][[隐修院|隐修院]]，曾为[[加尔默罗会|加尔默罗修会]]中国分会所，创建于1870年，关闭于1953年<ref name="徐汇区志">{{cite web |url = http://www.shtong.gov.cn/newsite/node2/node4/node2249/xuhui/node38592/node38601/node63522/userobject1ai23775.html |title = 第二节 徐家汇上海天主教教务中心|date=1997|author=徐汇区志-宗教民族篇-天主教章}}</ref><ref name="lkjhgfds">{{cite web |url = http://lkjhgfds.world.edoors.com/CvD-sehJFhk0 |title = 一、徐家汇圣衣院的始末（下） |date = 2011-07-22 |author = 奈何天 |deadurl = yes |archiveurl = https://web.archive.org/web/20131218120028/http://lkjhgfds.world.edoors.com/CvD-sehJFhk0 |archivedate = 2013年12月18日 |df =  }}</ref>。圣若瑟圣衣院大楼兴建于1873年，现属上海大型国有企业[[上影集团|上影集团]]的资产。

==圣衣院历史==
===早期发展===
1869年2月3日，应[[耶稣会|耶稣会]][[天主教南京总教区|天主教江南代牧区]]主教[[郎怀仁|郎怀仁]]之邀，5位来自法国[[拉瓦尔_(伊泽尔省)|拉瓦尔]][[加尔默罗会|加尔默罗会]]（圣衣会）的修女抵达上海徐家汇圣母院。<ref name="徐汇区志" />

1870年2月19日，早些时日来华的[[拯亡会|拯亡会]]法国修女和中国守贞姑娘将徐家汇附近王家堂的一片房屋借居给了5位加尔默罗会修女。2月24日，郎怀仁主教宣布封门，修女在王家堂开始了与外界隔绝的隐修，加尔默罗会会规严格，包括守斋、苦行、缄默不语、与世隔绝。次年，又有两位来自拉瓦尔的修女加入了修行行列。<ref name="徐汇区志" />

===圣若瑟圣衣院启用===
1873年12月8日，圣衣会在[[土山湾孤儿院|土山湾孤儿院]]对岸的[[肇嘉浜路|肇嘉浜]]东侧动工兴建了圣衣会新会院，大院为一幢外表朴实的假四层砖木结构宗教建筑，称作「圣若瑟圣衣院」。1874年，圣衣院有修女17人，1948年为18人<ref name="徐汇区志" />。1950年，有修女14人，其中外籍修女6人，院长为外籍的革克郎<ref name="上海宗教志">{{cite web |url = http://www.shtong.gov.cn/newsite/node2/node2245/node75195/node75203/node75283/node75297/userobject1ai91966.html |title = 第六节　其他团体 |publisher = 上海宗教志 |deadurl = yes |archiveurl = https://web.archive.org/web/20131219141831/http://www.shtong.gov.cn/newsite/node2/node2245/node75195/node75203/node75283/node75297/userobject1ai91966.html |archivedate = 2013-12-19 }}</ref>。

===关闭后旧址现状===
[[中华人民共和国|中华人民共和国]]建立后的1953年，圣衣会被[[中国共产党|中共]]当局关闭并移交由[[中国天主教爱国会|上海市徐汇区天主教爱国会]]管理。1958年，会院大楼划归给[[上海电影制片厂|上海电影制片厂]]<ref name="lkjhgfds"/><ref name="上海徐汇发布">{{cite web|url=http://shanghai.kankanews.com/shgov/2013-06-03/1703602.shtml|title=玩转徐汇|date=2013-06-03|author=上海徐汇发布|deadurl=yes|archiveurl=https://web.archive.org/web/20131215104959/http://shanghai.kankanews.com/shgov/2013-06-03/1703602.shtml|archivedate=2013-12-15}}</ref>。2008年，[[上影集团|上影集团]]启动建造上影集团大楼和上海电影博物馆，以「建造电影博物馆」为由，将具有上百年历史的圣衣院旧址整体拆除后原址原样重建，引发了不小的争议<ref name="lkjhgfds" />。重建后的圣衣院旧址于2013年完工，现为上影集团亚太电影会所。
