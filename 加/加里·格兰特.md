'''加里·格兰特'''（{{lang-en|Cary Grant}}，{{bd|1904年|1月18日|1986年|11月29日}}），另譯'''-{zh-hans:卡莱·葛伦;zh-hk:加里·格蘭;zh-tw:加里·格蘭特;}-'''，真名'''阿齊博尔德·亚历山大·李奇'''（Archibald Alexander Leach），英国[[電影演員|電影演員]]，加里·格兰特为他的[[艺名|艺名]]。[[美国电影学会|美国电影学会]]的[[AFI百年百大明星|AFI百年百大明星]]排名中将他列为男影星第2名 。

== 生平 ==
加里·格兰特生长于[[英国|英国]]貧家，14歲失學加入[[馬戲團|馬戲團]]巡迴演出，16歲在美國[[紐約|紐約]]演[[舞台劇|舞台劇]]5年，成名于西岸[[好莱坞|好莱坞]]，擔綱演出數部「緊張大師」[[希區考克|希區考克]]執導影片；

1981年他与[[考特·贝茜|考特·贝茜]]、[[海伦·海斯|海伦·海斯]]、[[杰洛姆·罗宾斯|杰洛姆·罗宾斯]]、[[鲁道夫·塞尔金|鲁道夫·塞尔金]]一同获得[[肯尼迪中心荣誉奖|肯尼迪中心荣誉奖]]。

== 事业 ==

== 个人生活 ==
卡萊·葛倫是美國[[共和黨_(美國)|共和黨]]支持者，雖然他認為影星不宜公開發表其政見。
