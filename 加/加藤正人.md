'''加藤正人'''（{{bd|1963年|3月28日}}）是[[日本|日本]][[电子游戏|电子游戏]]美工师、[[编剧|编剧]]与[[电子游戏总监|总监]]。他在生涯早年使用“Runmaru”和“Runmal”的假名<ref>{{cite web |url=http://www.1up.com/do/blogEntry?bId=9008133 |archiveurl=https://www.webcitation.org/5x0etnLIt?url=http://www.1up.com/do/blogEntry?bId=9008133 |title=The Chrono Trigger - Ninja Gaiden Connection |last=Parish |first=Jeremy |date=2009-10-28 |work=[[1UP.com|1UP.com]] |publisher=[[UGO_Entertainment,_Inc.|UGO Entertainment, Inc.]] |archivedate=2011-03-07 |accessdate=2011-03-07 |deadurl=yes }}</ref>。之后他加入[[史克威尔|史克威尔]]，最知名的工作是为《[[时空之轮|时空之轮]]》、《[[Radical_Dreamers|Radical Dreamers]]》、《[[异度装甲|异度装甲]]》、《[[穿越时空|穿越时空]]》以及部分《[[最终幻想VII|最终幻想VII]]》编写剧本。

== 简介 ==
加藤最早为[[特库摩|特库摩]]制作《[[足球小将_(FC游戏机)|足球小将]]》和[[忍者龙剑传系列|忍者龙剑传系列]]。在他的前三个游戏中，他从图形，图形兼编剧，干到动作元素总监<ref name="hard">{{cite web |url=http://www.hardcoregaming101.net/ninjagaiden/ninjagaiden9.htm|title=Interview with Masato Kato|date=|accessdate=2013-03-01 |author=|publisher=Hardcore Gaming 101}}</ref>。在尝试将“龙剑传”翻译为英语时，团队遇到了麻烦，最终他们因为“好听”而选择使用“Gaiden”，尽管这词通常意为“外传”<ref name="hard" />。后来加藤转入[[Gainax|Gainax]]工作。

加藤之后进入[[史克威尔|史克威尔]]，并参与《[[时空之轮|时空之轮]]》、《[[Radical_Dreamers|Radical Dreamers]]》、《[[异度装甲|异度装甲]]》和《[[穿越时空|穿越时空]]》的编剧工作。加藤称，因硬件限制，《穿越时空》的游戏世界扩充遭遇困境，他们尽可能将更多的数据塞入游戏碟片<ref name="cram">{{cite web |url=http://www.chronocompendium.com/Term/GamePro_Interview.html|title=Interview with Masato Kato|date=|accessdate=2013-03-01 |author=|publisher=[[GamePro|GamePro]]}}</ref>。相反，和《时空之轮》那样为通关玩家开放额外结局，这种多重结局的开发显得更容易<ref name="cram" />。

在为《[[最终幻想XI_吉拉德的幻影|最终幻想XI 吉拉德的幻影]]》设计剧情后的2002年，加藤离开史克威尔，成为自由编剧。除了为不同公司设计游戏外，加藤还继续参与史克威尔的项目，如[[玛娜世界|玛娜世界]]系列、任天堂DS强化移植版《时空之轮》以及另外三部《[[最终幻想XI|最终幻想XI]]》资料片。

2005年，加藤正人和老友音乐家[[光田康典|光田康典]]合作，编写[[短篇小说|短篇小说]]《克丽坦的五季》，而光田也随之创作了音乐专辑《{{tsl|en|Kirite|kiЯitɘ|克丽坦}}》。剧情和原声还以[[歌剧|歌剧]]形式演出<ref>{{cite web |url=http://www.procyon-studio.com/kirite/kirite_and_endless.html |title=Five Seasons of kiЯitɘ |accessdate=2007-02-16 |author=Procyon Studio staff |publisher=[http://www.procyon-studio.com/ Our Millennial Fair] |language=ja |deadurl=yes |archiveurl=https://www.webcitation.org/678Z9q0mM?url=http://www.procyon-studio.com/kirite/kirite_and_endless.html |archivedate=2012-04-23 }}</ref>，而光田在此之前为加藤《[[穿越时空|穿越时空]]》和《[[异度装甲|异度装甲]]》等数个游戏作曲。2008年，加藤和光田一同参与了[[世嘉|世嘉]]公司出品的游戏《[[世界毀滅|世界毀滅]]》的制作<ref>{{cite web|title=光田康典操刀！《世界毁灭》特典原声公布|url=http://www.cnetnews.com.cn/2008/0708/969847.shtml|publisher=CNETNews.com.cn|accessdate=2013-12-10|language=zh-cn|archive-url=https://web.archive.org/web/20131212110425/http://www.cnetnews.com.cn/2008/0708/969847.shtml|archive-date=2013-12-12|dead-url=yes}}</ref>。

2010年5月，加藤正人曾在自己的[[Twitter|Twitter]]上表示，“[[iPad|iPad]]的发售预示着[[任天堂|任天堂]]时代已宣告结束”，并称“[[任天堂DS|DS]]和[[Wii|Wii]]就是他们给自己挖好的坟墓。”<ref>{{cite news|url=http://games.sina.com.cn/t/n/2010-06-02/1138404851.shtml|title=加藤正人：iPad上市意味着任天堂时代终结|publisher=新浪游戏 |date=2010年6月2日|accessdate=2013-12-12|language=zh-cn}}</ref><ref>{{cite news|url=http://digi.163.com/10/0602/10/685S1DQI001624J7.html|title=偏执游戏从业者称任天堂正在自掘坟墓|publisher=网易数码|work=兔友网|date=2010-06-02|accessdate=2013-12-12|language=zh-cn}}</ref><ref>{{cite news|url=http://tjgame.enorth.com.cn/system/2010/06/02/004737736.shtml|title=《FF7》创作者加藤正人说老任时代要结束|publisher=北方网|date=2010-06-02|accessdate=2013-12-12|language=zh-cn|deadurl=yes|archiveurl=https://web.archive.org/web/20160304080226/http://tjgame.enorth.com.cn/system/2010/06/02/004737736.shtml|archivedate=2016-03-04}}</ref>
