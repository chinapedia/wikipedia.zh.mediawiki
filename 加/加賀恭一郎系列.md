「'''加賀恭一郎系列'''」（有時被稱為「'''加賀系列'''」）指的是日本推理作家[[東野圭吾|東野圭吾]]的[[小說|小說]]中，由加賀恭一郎為主人公的系列作品的總稱。

== 概要 ==
加賀恭一郎首次出現於1986年東野出道的第二本作品《[[畢業—雪月花殺人遊戲|畢業—雪月花殺人遊戲]]》。在該作裡他仍然是國立T大學在讀的大學生，在學校裡被捲入的連續殺人事件中充當偵探角色。在1989年的《[[沉睡的森林_(小說)|沉睡的森林]]》裡因為「作者的少許惡作劇心態」，加賀以[[警視廳|警視廳]][[警視廳#刑事部_Criminal_Investigation_Bureau|搜查一課]]的警察身份出場。90年代中期後在《[[惡意_(小說)|惡意]]》、《[[誰殺了她|誰殺了她]]》、《[[我殺了他|我殺了他]]》、《[[再一個謊言|再一個謊言]]》等作品中均有出場。

然後在2006年開始連載的《[[紅手指|紅手指]]》裡，加賀以良知拷問迷失了親情的一家人，同時也披露了自己與父親的往事；2004年開始連載，2009年出版的《[[新參者|新參者]]》中，剛調配至[[日本橋|日本橋]][[人形町|人形町]]的加賀為查案而來回奔走。

== 登場人物 ==
=== 主人公 ===
; <span id="加賀恭一郎">{{nihongo|加賀 恭一郎|{{ruby-ja|加賀|かが}} {{ruby-ja|恭一郎|きょういちろう}}|Kaga Kyouichirou}}</span>
: 初登場時是大學生，為二十一至二十二歲，畢業之後曾當過兩年教師，但因為學生之間發生的某件事，認為自己沒有資格做教師而辭職<ref>出自《[[惡意_(小說)|惡意]]》。</ref>，然後跟父親一樣成為警官，原任職警視廳搜查一課，之後在[[練馬區|練馬]]署搜查一系擔任[[巡查部長|巡查部長]]。之後調配到久松署，如今在[[日本橋|日本橋]]署，職位是[[警部補|警部補]]。
: 身形高大健碩。因為五官深邃的關係，逆光看的話會好像有黑眼圈和尖下巴。因不吸煙所以牙齒很白，笑容很爽朗。周遭的人對加賀的印象是：外表不錯、很受歡迎的樣子，在《[[新參者|新參者]]》中頭髮稍微長了一點。
: 興趣是[[茶道|茶道]]和欣賞[[古典芭蕾|古典芭蕾]]。也是美國暢銷書作家[[麥可·克萊頓|克萊頓]]的書迷<ref>在《[[我殺了他|我殺了他]]》，神林貴弘問加賀：「你是克萊頓的書迷嗎？」。</ref>。在國立T大學中是[[劍道|劍道]]部的部長，劍道六段，曾贏得全日本選手權優勝。加賀雖然心地善良，又有領導和協調才能，成為警官後卻經常單獨行動。雖不善辯亦不沉默寡言，雖感情豐富亦有沉著冷靜的一面，以他鉅細無遺的洞察目光快速將案件看穿。認為只要是在案件中内心受傷的人都是受害者，對於加害者也能表示理解，非常有人情味。出身於社會學部的文科生的他，對[[工學|工學]]、[[化學|化學]]和[[電腦科學|電腦科學]]也很熟悉。但性格也有奇怪一面，在《[[新參者|新參者]]》裡曾數次詢問與案件無關的問題，被[[町|町]]內一部分人當作是怪人。
: 感情方面，在《[[畢業——雪月花殺人遊戲|畢業——雪月花殺人遊戲]]》裡曾向一同渡過高中和大學時代的相原沙都子求婚，但兩人對畢業後有各自不同的計畫展望，因而沒有進一步交往。但在《[[沉睡的森林_(小說)|沉睡的森林]]》裡，有記述他每年會從海外收到1、2次由「大學時代的戀人」沙都子寄來的信件，恭一郎認為那是「來自過去的信」。接著和新進芭蕾舞蹈員淺岡未緒相戀，之後並未交代兩人進展，但在《[[新參者|新參者]]》裡某件事件的審判中，有記述他作為辯方的證人出庭，因此事他也被調離了轄區。如今獨身，在《[[紅手指|紅手指]]》中是30多歲。
: 家人方面，母親在他幼時離家出走，恭一郎似乎認為原因是因為父親不顧家的緣故，因此和父親關係不怎麼好。在《[[沉睡的森林_(小說)|沉睡的森林]]》裡不時以電話和父親通話，也有跟他提過未緒的事。
: 最新登場的作品為《[[當祈禱落幕時|當祈禱落幕時]]》。

=== 家族成員 ===
; <span id="加賀隆正">{{nihongo|加賀 隆正|{{ruby|加賀|かが}} {{ruby|隆正|たかまさ}}|Kaga Takamasa}}</span>
: 恭一郎的父親，已故。曾是警察，辭職後擔任保安公司的顧問。在《畢業——雪月花殺人遊戲》裡，曾向恭一郎提供案件建議。在《[[沉睡的森林_(小說)|沉睡的森林]]》裡，在電話中談過恭一郎的婚事和自己身邊發生的小事。但恭一郎在母親百合子出事之後就開始疏遠父親。
<!--:『どちらかが彼女を殺した』では恭一郎が和泉康正に、「無意味な復讐は[[赤穂浪士|赤穂浪士]]だけでたくさんだ」が父の口癖である、と語っている。-->
: 在《紅手指》中因患[[肝癌|肝癌]]和[[膽囊癌|膽囊癌]]而入院，後來安詳地離世。
; {{nihongo|田島 百合子|{{ruby|田島|たじま}} {{ruby|百合子|ゆりこ}}|Tajima Yuriko}}
: 恭一郎的母親，已故。年輕時曾在俱樂部酒店上班，與隆正在辦案中相識，隨後結婚。在恭一郎12歲那年，與隆正離婚後便失蹤了十六年。直到《當祈禱落幕時》，恭一郎才真正了解母親當年離開的原因。
; <span id="松宮脩平">{{nihongo|松宮 脩平|{{ruby|松宮|まつみや}} {{ruby|脩平|しゅうへい}}|Matsumiya Shuuhei}}</span>
: 恭一郎的表弟，警視廳搜查一課的警官。自父親死後就對舅舅，也就是加賀的父親如親生父親般尊敬，因為舅舅在松宮父親死後，對他們多加照顧。成為警官的原因也是因為尊敬的舅舅曾當過警察。非常不滿一次也沒來探望隆正的恭一郎。
: 稱恭一郎為「恭哥」，但恭一郎讓他在工作中稱自己為「加賀」。
: 在隆正死後，了解到恭一郎和隆正背後的故事。

=== 其他 ===
