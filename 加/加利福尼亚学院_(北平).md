'''加利福尼亚学院'''，旧址位于[[北京市|北京市]][[东城区_(北京市)|东城区]]，是[[中华民国|中华民国]]时期设在[[北平市|北平市]]的一座学院。<ref name=ifeng>[http://house.ifeng.com/news/society/detail_2013_09/30/30009225_0.shtml 北京闹市“鬼楼”将修缮 官方否认军官太太自杀，凤凰网，2013年09月30日]</ref>

==简介==
该学院的旧址位于[[朝阳门内大街|朝阳门内大街]][[北京外交人员服务局|北京外交人员服务局]]后面。1910年兴建。原是[[美国|美国]]教会兴建，作为培训传教士[[中文|中文]]并提供休息处的“华北协和话语学校”<ref name=ifeng/>。[[美国|美国]]第一代专业[[汉学家|汉学家]]，大部分是1930年代靠[[哈佛燕京学社|哈佛燕京学社]]的奖学金到[[北平|北平]]进修。为这批青年提供语言训练的正是华北协和话语学校。[[韦慕庭|韦慕庭]]和[[费正清|费正清]]也是在该校的网球场上开始了友谊。韦慕庭于1932年秋自[[哥伦比亚大学|哥伦比亚大学]]来到北平，因奖学金不足，他在华北协和话语学校学习时兼充该校图书馆的管理员<ref>[http://dangshi.people.com.cn/n/2013/0923/c85037-23002079.html 美国一流汉学家的一段共同的学术经历——第一代美国留学生在北京，中国共产党新闻网，2013年09月23日]</ref>。

1930年后，该学校更名为“加利福尼亚学院”，开始招生。后来该学校停办<ref name=ifeng/>。

2005年，北京市东城区编纂出版《东华图志》，其中对[[朝阳门内大街81号|朝阳门内大街81号]]的历史进行了错误介绍。在本书编纂出版时，有位[[美国|美国]]记者错将该院的两座小楼与由此向西一站地的[[北京外交人员服务局|北京外交人员服务局]]后边的原“华北协和话语学校”建筑混淆，写出了《一栋记录中美历史关系的学校将被拆除》的报道，《东华图志》编纂人员不察，将此错误说法收入书中。实际上根据档案显示，华北协和话语学校（及加利福尼亚学院）与朝阳门内大街81号没有任何关系<ref name=wy>{{Cite web |url =http://money.163.com/14/0923/16/A6REAMQI00254TI5_all.html  |title =档案解密：北京朝内81号身世之谜  |author =  |date =2014-09-23  |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref>。
