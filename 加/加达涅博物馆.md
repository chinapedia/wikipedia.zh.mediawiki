'''加达涅博物馆'''（Musée Gadagne）是一个博物馆，位于[[里昂第五区|里昂第五区]]（ [[里昂老城|里昂老城]]）中心的 圣若翰区。它是由里昂历史博物馆和和国际木偶博物馆组成。该建筑于1920年被列为历史古迹。在1902年到1941年之间，逐步被里昂市收购。<ref>{{cite web |title=Af fil du temps > Architecture > Musée de Gadagne |url=http://www.vieux-lyon.org/musee-de-gadagne_f01260.htm |publisher=Vieux Lyon |language=法语|accessdate=2010年5月8日}}</ref>1998年关闭维修并加以扩建达十余年之久，直到2009年6月12日才重新开放。<ref>{{cite web |title=Le musée Gadagne rouvre ses portes à Lyon |url=http://culturebox.france3.fr/all/11917/le-musee-gadagne-rouvre-ses-portes-a-lyon/#/all/11917/le-musee-gadagne-rouvre-ses-portes-a-lyon/ |publisher=France 3 |date=2009 6月12日 |language=法语 |accessdate=2010年5月8日 |deadurl=yes |archiveurl=https://web.archive.org/web/20111009153759/http://culturebox.france3.fr/all/11917/le-musee-gadagne-rouvre-ses-portes-a-lyon/#/all/11917/le-musee-gadagne-rouvre-ses-portes-a-lyon/ |archivedate=2011年10月9日 }}</ref>

里昂历史博物馆自1921年起设于该建筑内，展出此前收藏在[[里昂市政厅|里昂市政厅]]的展品。目前博物馆共有80,000件展品，<ref>{{cite web |title=Les Musées Gadagne |url=http://www.culture.lyon.fr/culture/sections/fr/musees__expositions/a_lyon/le_musee_gadagne/entite?entiteId=364 |publisher=Culture Lyon |language=法语|accessdate=2010年5月8日}}</ref> 30个展厅，分布在四个楼层，体现自中世纪直到19世纪里昂的文化历史，包括考古遗迹、雕塑、绘画、家具和陶瓷，还有许多地图和版画，说明这个城市的成长和主要建筑物的建设。

自1950年以来，世界木偶博物馆入驻加达涅，包括2000个木偶。2000年代大规模翻新后，分布于一楼九个独立的展厅。

三楼设有一个150个座位的容量小剧场，四楼设有700平方米的花园。

==历史==
该博物馆位于加达涅公馆（Hôtel Gadagne）。该建筑由Pierrevive兄弟建于十六世纪初。1545年，加达涅兄弟重新设计。这个富有的[[佛罗伦萨|佛罗伦萨]]家族在十五世纪初住在里昂，拥有巨额家产。当时里昂人会说：“富比加达涅”。
