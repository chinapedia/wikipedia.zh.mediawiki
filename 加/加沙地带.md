'''加沙地带'''（{{Lang-ar|قطاع غزة}}或''{{Transl|ar|DIN|Qiṭāʿ Ġazzah}}''；[[希伯來語|希伯來語]]：'''רצועת עזה'''）是[[西奈半岛|西奈半岛]]东北部地中海沿岸占地363[[平方公里|平方公里]]（140平方哩）的区域，位於[[巴勒斯坦|巴勒斯坦]]西岸、[[西奈半島|西奈半島]]東北部的狹長地帶，在二戰後原本在聯合國規劃中將與約旦河西岸成立一個新的[[阿拉伯國家|阿拉伯國家]]，但1948年阿拉伯聯軍對以色列發動[[第一次中東戰爭|阿以戰爭]]，加沙為埃及吞併，直到1967年[[六日戰爭|六日戰爭]]又被以色列占领。目前为[[巴勒斯坦国|巴勒斯坦国]]政府统治。

== 歷史 ==
加沙地带自古為肥沃月灣與埃及之間的要道。自16世紀起，加沙地带先後被[[奧斯曼土耳其|奧斯曼土耳其]]、[[英國|英國]]、[[埃及|埃及]]，和[[以色列|以色列]]統治。1994年，以色列將加沙地带的自治權交給[[巴勒斯坦民族權力機構|巴勒斯坦民族權力機構]]，一直以來都不曾是一個真正獨立的國家。<ref name="eccpalestine.org">{{cite web|url=http://www.eccpalestine.org/eu-heads-of-missions-report-on-gaza/ |title=EU Heads of Missions’ report on Gaza 2013 |publisher=Eccpalestine.org |date= |accessdate=2014-08-02}}</ref>

2005年之前，加沙地带69%的土地与[[約旦河西岸地區|西岸]]的部分地區由[[巴勒斯坦民族权力机构|巴勒斯坦民族权力机构]]管理，其他部分（主要是以色列公民居住的地方）则由以色列管理。由于巴勒斯坦民族权力机构不允许拥有正规军，治安由巴勒斯坦的公共治安队和民警负责。

2005年2月，以色列政府决定在[[夏天|夏天]]将以色列军队撤出加沙地带，并放弃所有以色列在加沙地带的居民点。撤军后，以色列将依然控制加沙地带的海岸线，以及加沙地带与埃及之间的一个狭长区域。但以色列国内对撤军计划有很大分歧。

2005年8月15日，以色列关闭加沙古什·卡提夫犹太定居点，正式撤出加沙。儘管如此，邊境、海岸與領空依舊被以色列控制。

由於主政的哈馬斯堅持消滅以色列，因此加沙地带的邊境、海岸與領空依舊被以色列重重封鎖，水電基礎設施的使用權亦依然被以色列嚴密管控<ref>{{cite web |url=http://www.coolloud.org.tw/node/51506 |title=帝國恐怖主義 斷送巴勒斯坦和平 | 苦勞網 |publisher=Coolloud.org.tw |date=2008-08-26 |accessdate=2014-08-02}}</ref>。

2007年6月，[[哈马斯|哈马斯]]通過[[加沙之戰_(2007年)|加沙之战]]从[[法塔赫|法塔赫]]手中夺得加沙地带的控制权，自此法塔赫仅控制[[约旦河西岸|约旦河西岸]]地区，而加沙地带由哈马斯控制<ref name=jieguan/>。以色列從此開始嚴密封鎖加沙地带，[[埃及|埃及]]亦關閉加沙地带南部的邊界，居民只能依靠救濟和從與埃及間的地道掙扎求生。國際救援組織前往加沙的援助車隊則常在埃及受到阻撓，然而2010年5月的以色列軍隊襲擊援助加沙自由船隊的[[加沙船隊衝突|加沙船隊衝突]]卻造成了轉機，世界各國人民與政治家為了支援加沙人民，開始自發地結合起來<ref>{{cite web |url=http://www.dailystar.com.lb/article.asp?edition_id=10&categ_id=5&article_id=116262#axzz0rhIK7jHB |title=A big corner has been turned in Gaza |publisher=Dailystar.com.lb |date=2010-06-23 |accessdate=2014-08-02}}</ref>。加沙地带與[[約旦河西岸|約旦河西岸]]地区作为[[巴勒斯坦國|巴勒斯坦國]]在2012年正式被[[聯合國|聯合國]]接納為[[聯合國大會觀察員列表#觀察員國|觀察員國]]。<ref>{{cite web|url=http://www.un.org/News/Press/docs/2012/ga11317.doc.htm |title=General Assembly Votes Overwhelmingly to Accord Palestine ‘Non-Member Observer State’ Status in United Nations |publisher=Un.org |date= |accessdate=2014-08-02}}</ref>

2008年[[2008年加沙戰爭|加沙戰爭]]，以色列摧毀了包括警察局、監獄和指揮中心等450個哈馬斯目標。聯合國說，頭兩天的空襲共造成320名巴勒斯坦人被打死，1400人受傷。為報復以色列軍事打擊，哈馬斯向以色列境內目標發射火箭彈和迫擊炮還擊。同時以色列北部也受到來自黎巴嫩方向發射的火箭彈襲擊。空襲持續一週後，以色列於次年1月3日派出地面部隊進入加沙地帶。

2012年[[雲柱行動|雲柱行動]]，以色列國防軍於11月14日開始在加沙地帶展開的軍事行動，以報復此前哈馬斯方面對以色列南部城鎮的襲擊。

[[2014年以巴衝突|2014年以巴衝突]]，以色列大規模空襲加沙，7月8日起在哈馬斯統治加沙地帶展開軍事行動，在衝突進行了50天後，以色列與巴勒斯坦抵抗组织哈马斯已同意埃及提出的方案，從格林威治時間26日16時開始在加沙實施無限期停火，為50天來造成超過2100多人死亡的加沙戰事畫下句點。

2011年5月，经埃及斡旋，法塔赫和哈马斯在埃及[[开罗|开罗]]签署和解协议。2012年2月，双方在[[卡塔尔|卡塔尔]]签署《多哈宣言》。2014年4月，双方宣布执行上述和解协议，但未能落实。2017年9月17日，哈马斯宣布解散掌握加沙地带管理权的“行政委员会”，同意由巴勒斯坦政府接管加沙地带并行使行政权。2017年9月25日，巴勒斯坦政府发言人优素福·穆罕默德在[[拉姆安拉|拉姆安拉]]宣布，巴勒斯坦总理[[拉米·哈马达拉|拉米·哈马达拉]]与总统[[马哈茂德·阿巴斯|阿巴斯]]协商后决定，巴勒斯坦政府将于2017年10月3日在加沙地带召开政府每周例会，为此哈姆达拉和部分政府部长将在2017年10月2日进入加沙地带，与控制加沙地带的哈马斯进行权力交接<ref name=jieguan>{{Cite web |url = http://www.thepaper.cn/newsDetail_forward_1807257 |title =  缺位多年后，巴勒斯坦政府将于下周接管加沙地带|author =  |date =2017-09-26  |publisher = 澎湃新闻 |language =  |accessdate =  |quote =  }}</ref>。

== 人口 ==
在2010年加沙地带的人口数达到160万，其中约一百万是联合国认证的难民。<ref>{{cite web|url=https://www.un.org/unrwa/refugees/gaza.html|archiveurl=https://web.archive.org/web/20071217211906/http://www.un.org/unrwa/refugees/gaza.html|archivedate=17 December 2007 |title=UNRWA: Palestine refugees |publisher=Un.org |date= |accessdate=1 June 2010}}</ref> 在2015年末估计当地有185万人<ref name=pcbs_end_2015>[http://pcbs.gov.ps/Downloads/book2176.pdf ''Table 3: Projected Population in the State of Palestine by Governorate, End Year 2015'']. PCBS, ''Palestinians at the End of 2015'', p. 36. [http://pcbs.gov.ps/PCBS_2012/Publications.aspx Source:]</ref>。大部分的加沙地带居民都是1948年[[第一次中东战争|第一次中东战争]]中逃出巴勒斯坦的难民及其后代。加沙地带的人口一直快速增长，2014年估计当地的总和生育率在每名妇女4.24个孩子。联合国曾经声称，如果不采取必要的措施，修复基础设施。到2020年，人口会增长50万人，那时加沙地带将不适合人类居住。<ref>[https://www.un.org/depts/dpa/qpal/docs/2013Rome/P2%20MS.%20AL%20WEHAIDY%20EN.pdf  'United Nations Seminar on Assistance to the Palestinian People,'] FAO Seminar, Rome
27 and 28 February 2013</ref> 大多数当地人是逊尼派[[穆斯林|穆斯林]]，少数为[[巴勒斯坦基督徒|基督徒]]（0.7%）（约两三千人）。<ref>[http://news.bbc.co.uk/1/hi/world/middle_east/4514822.stm Middle East Christians: Gaza pastor] BBC News, 21 December 2005</ref>

加沙地带位于[[中东|中东]]，与[[埃及|埃及]]接壤的邊界長11公里，（[[拉法省|拉法]]就位於邊境附近），而與以色列接壤的邊界則长約51公里。保守的犹太人認為加沙地带是以色列的领域，而巴勒斯坦人则聲称這是将来要成立的巴勒斯坦国的領土。

2005年時任[[以色列總理|以色列總理]][[阿里埃勒·沙龙|阿里埃勒·沙龙]]打算单方面放弃所有加沙地带的以色列居民点，这些居民点主要集中于西南边靠海岸的喀什卡提夫。此外，加沙地带与[[地中海|地中海]]还有一條约40公里长的海岸线。但由于以色列的军事管辖，[[巴勒斯坦民族权力机构|巴勒斯坦民族权力机构]]对其海岸线並没有控制权，海域仍由以色列控制。

加沙地带气候温和，冬季温暖，夏季则炎热干旱。地形平坦，有些地方是丘陵，海岸有沙丘。最高点海拔105公尺。自然资源有可耕地（加沙地带约三分之一的地区被灌溉），最近还发现[[天然气|天然气]]。环境问题包括[[沙漠化|沙漠化]]、淡水鹹化、垃圾处理、饮水不洁带来的疾病、土壤恶化和[[地下水|地下水]]资源的消耗。

加沙地带被认为是人类摇篮之一，在四千年前是[[非利士人|非利士人]]眾居的地方。人类最早的用火遗址是在加沙地带发现，一些最古老的人類化石也是在这里发现的，一些非常古老的抽象符号有可能是人类最老的文字。

== 经济 ==
1994年5月，根据[[开罗协议|开罗协议]]，加沙地带的经济由巴勒斯坦民族权力机构管理。

由於[[亚西尔·阿拉法特|亚西尔·阿拉法特]]领导下政府贪污和管理不良，加上以色列在遭受一系列恐怖袭击后，将加沙地带的边境关闭，期內加沙地带的经济萎缩了三分之一。在边境关闭前，有许多加沙地带的人到以色列工作。经济不景導致高失业率。

1998年，以色列改变对巴勒斯坦的政策，开始减轻封锁巴勒斯坦的经济，並减缓对巴勒斯坦货物和劳工运輸的限制。这使經濟連續三年恢复。但2000年爆发的[[第二次巴勒斯坦武装起义|第二次巴勒斯坦武装起义]]，导致以色列再度封锁。在此后两年中，巴勒斯坦内部的斗争和以色列的军事行动，摧毁了加沙地带主要的工厂和管理机构，许多企业倒闭，国家总生产力大降，巴勒斯坦在以色列的劳工的收入也大降。

根据美国中央情报局的[[世界概况|世界概况]]，2001年经济下降35%，人均收入为每年625美元。60%的人口生活在贫穷线以下。加沙地带的工业主要是小型的家庭企业，其产品为[[纺织品|纺织品]]、[[肥皂|肥皂]]、[[橄榄树|橄榄树]]木雕刻和旅游纪念品。以色列人在一个工业中心建立了一些小型现代化的工业。[[电力|电力]]由以色列提供。主要的农产品是[[橄欖_(木樨科)|橄榄]]、[[柠檬|柠檬]]、[[蔬菜|蔬菜]]、[[牛肉|牛肉]]和[[奶制品|奶制品]]。主要出口柠檬和鲜花，主要进口食品、消耗品和建设物资。主要的贸易对象为以色列、埃及和西岸。

[[约翰·霍普金斯大学|约翰·霍普金斯大学]]和[[国立阿拉伯巴勒斯坦大学|国立阿拉伯巴勒斯坦大学]]于2002年末为[[国际援外合作署|国际援外合作署]]作的一个研究表明，巴勒斯坦人普遍缺乏营养。17.5%6岁到59个月的儿童患慢性[[营养不良|营养不良]]，53%的年轻和中年妇女以及44%的儿童患[[贫血症|贫血症]]。

== 运输和通讯 ==
从南到北加沙地带有一条[[标准轨距|标准轨距]]的[[铁路|铁路]]，但已荒廢，只有少数轨道保存。这条铁路过去在南部连接埃及的铁路，在北部连接以色列的铁路系统。此外，加沙地带还有一个小的、原始的[[公路网|公路网]]，它唯一的海港是[[加沙市|加沙市]]，现在已被关闭。[[亚西尔·阿拉法特国际机场|加沙国际机场]]于1998年11月24日开放。2000年10月被以色列下令关闭，2001年12月以色列军队摧毁了它的跑道。

加沙地带有簡陋的[[PSTN|电话服务系統]]，以色列電訊商如[[Cellcom|Cellcom]]亦有提供服務。加沙地带有两个[[电视台|电视台]]（由巴勒斯坦民族权力机构管理），没有[[广播电台|广播电台]]，此外还有四个[[互聯網服務供應商|互联网服务商]]提供主要為ASDL服務。大多数巴勒斯坦家庭擁有[[收音机|收音机]]和[[电视机|电视机]]。

== 被以色列實行經濟封锁 ==
2008-2009年间的[[加沙战争|加沙战争]]结束后，以色列虽允许有限数量的[[医疗|医疗]][[人道|人道]]救援物资进入加沙，但在对加沙地带实施封锁。[[红十字国际委员会|红十字国际委员会]]表示封锁破坏了加沙经济，使加沙的医疗状况持续恶化<ref>{{cite web|url=http://www.icrc.org/web/chi/sitechi0.nsf/html/palestine-interview-250610 |title=搜索 - 红十字国际委员会 |publisher=Icrc.org |date=2010-07-01 |accessdate=2014-08-02}}</ref>。

以色列安全局局长尤瓦尔·迪斯金（Yuval Diskin）表示他并不反对放松贸易限制，但他认为西奈的走私隧道以及加沙地带的开放港口威胁了以色列的安全。迪斯金认为，哈马斯和伊斯兰圣战组织已经走私了“5000多枚射程达40公里的火箭弹”。有些火箭弹射程甚至能远至特拉维夫市区。

以色列总理办公室发言人马克·雷格夫（Mark Regev）表示以色列的做法是“制裁”而非封锁，但联合国近东巴勒斯坦难民救济和工程处的一名加沙法律顾问称封锁是“不符合国际法的行为”。

2010年7月，[[英国首相|英国首相]][[大卫·卡梅伦|大卫·卡梅伦]]（David Cameron）对封锁表示谴责，他说：“人道物资和救援人员必须能够双向通行。加沙不能也绝不应成为[[集中营|集中营]]。”作为回应，以色列驻伦敦使馆发言人表示：“加沙人民是恐怖组织[[哈马斯|哈马斯]]的囚犯。哈马斯的统治和行动重点直接导致了加沙局势。”

阿拉伯联盟指责[[以色列|以色列]]发动了一场[[经济|经济]][[战争|战争]]。

[[以色列国防军|以色列国防军]]严格控制以色列和加沙地带之间过境点区域内的通行，并封锁了与加沙交界的地区。加沙地带及边界地区（包括与埃及交界地区和海岸沿线）都非常危险，局势随时可能发生变化。

面对国际上日益高涨的要求放松或解除[[封锁|封锁]]的呼声，埃及和以色列从2010年6月起放松了限制。[[埃及|埃及]]部分开放了埃及到加沙的拉法过境点，埃及外交部清楚表示过境点的开放主要是为人员通行，不能运送供应品。以色列宣布将允许严格意义上的民用物品进入加沙，但要防止某些[[武器|武器]]和具有双重用途的物品进入哈马斯控制的加沙地带。

2011年5月28日，埃及开放了与加沙地带陆路相连的拉法口岸，外界对加沙的封锁至此结束。

[[埃及總統|埃及總統]][[阿卜杜勒-法塔赫·塞西|阿卜杜勒-法塔赫·塞西]]上任後再度封鎖埃及西奈半島與加沙地带之間的地道，削弱哈馬斯武力、資源的後送。<ref>[http://www.cna.com.tw/news/firstnews/201407265001-1.aspx 加薩以色列戰火 3分鐘看懂（懶人包）]，中央社，2014年7月26日</ref>

2009年1月至2010年6月间，以色列向加沙地带运送了约20亿磅救援物资、1.37亿升燃料以及5万吨厨用煤气。
