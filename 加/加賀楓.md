'''加賀楓'''（{{bd|1999年|11月30日}}）日本東京都出身。日本女子偶像組合「[[早安少女組。|早安少女組。]]」第十三期成員。<ref name="HPprofile">[http://www.helloproject.com/morningmusume/profile/kaede_kaga/ モーニング娘。’17：プロフィール（ハロー！プロジェクト　オフィシャルサイト）]. 2017年1月1日閲覧。</ref>原[[Hello!_Pro研修生|Hello! Pro研修生]]第十七期。隸屬演藝經紀公司[[UFG#UP-FRONT_AGENCY|UP-FRONT AGENCY]]，血型A型，個人代表色為義大利紅。

== 個人簡歷 ==
;2012年
*因憧憬[[早安少女組。|早安少女組。]]九期成員[[鞘師里保|鞘師里保]]，而參加モーニング娘。11期メンバー『スッピン歌姫オーディション』，於第三次審查中落選。
*同年11月加入[[Hello!_Pro研修生|Hello! Pro研修生]]，並於12月9日在[[TSUTAYA_O-EAST|TSUTAYA O-EAST]]中開演的『ハロプロ研修生発表会 2012 12月の生タマゴShow!』中初亮相。

;2013年
*1月在[[中野太陽廣場|中野太陽廣場]]中開演的『Hello!Project 誕生15周年記念ライブ』，初次與[[Hello!_Project|Hello! Project]]正規成員共演。
*2月初次與同為研修生[[山岸理子|山岸理子]](現為[[Tsubaki_Factory|Tsubaki Factory]])演出舞台劇『熱帯男子』。
*4月16日~9月15日出演由アイドル専門チャンネルPigoo放送的電視節目『ハロプロ研修生のただ今研修中！』。
*10月5日~9月27日出演由[[BS日本|BS日本]]放送的電視節目『〜おねだりエンタメ!〜はぴ★ぷれ』。
*10月20日發行了[[Hello!_Pro研修生|Hello! Pro研修生]]DVD單曲『おへその国からこんにちは/天まで登れ!』，亦為首次單曲音源化。

;2014年
*3月，演出『演劇女子部 僕たち可憐な少年合唱団』。
*6月，演出『演劇女子部 LILIUM-リリウム 少女純潔歌劇-』。

;2015年
*曾獲得2015年實力公開診斷第一名。

;2016年
*11月3日~20日初次主演『演劇女子部 ネガポジポジ』。
*12月12日『モーニング娘。'16コンサートツアー秋～MY VISION～』最終場公演於[[日本武道館|日本武道館]]中，由[[早安少女組。|早安少女組。]]隊長[[譜久村聖|譜久村聖]]發表為第十三期新成員。

;2017年
*從1月2日開始的『Hello! Project 2017 WINTER ～ Crystal Clear ～』，首次以[[早安少女組。|早安少女組。'17]]的成員身份出演。

;2018年
*2月23日以嘉賓身份再次演出由原為[[℃-ute|℃-ute]]成員[[矢島舞美|矢島舞美]]主演的舞台劇『熱帶男子』。
*4月19日因為「加賀」的緣故，做為[[石川縣|石川縣]]的[[加賀市|加賀市]]、[[小松市|小松市]]的加賀溫泉鄉觀光大使。

== 人物 ==
*有個姊姊。
*小時候非常喜歡在外玩耍，小學1～2年級時曾學過體操運動但在過程中因為受傷而辭退，之後便開始學習劍道。剛開始非常討厭學劍道，但後來慢慢的變得越來越喜歡。小學3年級時曾在全國大賽中出場。一直到中學一年級加入[[Hello!_Pro研修生|Hello! Pro研修生]]仍然持續著，當時官網上的研修生資歷上寫著喜歡的運動為武術包含劍道。
*以早安少女組。成員的身份參加了第一次的家族演唱會，跳舞時的模樣被[[Kobushi_Factory|Kobushi Factory]]成員[[和田櫻子|和田櫻子]]給予「重力を感じさせない」、「無重力ダンス」的評價。
*加入研修生約4年的時間，有20人以上的同期及後輩相繼出道，加入早安少女組。時被媒體稱為「苦勞人」，相對同期的[[橫山玲奈|橫山玲奈]]則創下研修生史上最快出道的紀錄(二個月)。
*曾有過辭退研修生的想法是出自於當時研修生同期的[[牧野真莉愛|牧野真莉愛]]已被選為早安少女組。十二期成員。在雙親詢問她「之後該怎麼辦」的時候，認為自己成為研修生也才二年，所以決定繼續留在研修生。2016年的時候，因為同時考慮到高校二年級的課業以及將來的問題，將想要辭退的想法告訴經紀人受到了慰留。在與雙親討論之下，獲得了「12月為止加油吧」的結論並決定繼續留下來，最後在12月時以早安少女組。13期成員身份於武道館中初次登場。
*研修生時期與十二期成員[[牧野真莉愛|牧野真莉愛]]做為同期，但由於早安少女組。上下關係嚴格，於是稱謂從「まりあ」改為「牧野さん」。另外，十二期成員[[羽賀朱音|羽賀朱音]]雖為研修生時期的後輩，但還是改為「羽賀さん」。
*在部落落與雜誌的問卷調查中，提及喜歡與推薦的動漫為[[進擊的巨人|進擊的巨人]]、[[排球少年|排球少年]]、[[Free!|Free!]]、[[境界的彼方|境界的彼方]]、[[革命機Valvrave|革命機Valvrave]]、[[流浪神差|流浪神差]]、[[幼女戰記|幼女戰記]]、[[不愉快的妖怪庵|不愉快的妖怪庵]]、[[文豪Stray_Dogs|文豪Stray Dogs]]、[[亞爾斯蘭戰記|亞爾斯蘭戰記]]、[[終結的熾天使|終結的熾天使]]。共演舞台劇「ネガポジポジ」的前[[Berryz工房|Berryz工房]]成員[[須藤茉麻|須藤茉麻]]在其官方推特上提及能夠哼出[[排球少年|排球少年]]的主題曲。在接受「月刊ガンダムエース」的攝影工作準備時，將「[[機動戰士鋼彈_鐵血孤兒|機動戰士鋼彈 鐵血孤兒]]」全50話觀看完畢，並獲得其作品模型做為禮物。且將過程全程記載於部落格中。
*非常喜歡玩[[深淵傳奇|深淵傳奇]]或者[[傳奇系列|傳奇系列]]的電玩遊戲，為了同系列的遊戲，還特別買了PS4，並於部落格中報告。同時也很擅長音樂類型的遊戲，例如：「[[LoveLive!|LoveLive!]]」、[[IDOLiSH7|IDOLiSH7]]或是[[Hello!Project|Hello!Project]]全員登場的音樂手遊「ハロプロタップライブ」。
*曾與九期成員[[譜久村聖|譜久村聖]]及十期成員[[飯窪春菜|飯窪春菜]]一起到訪南夢宮主題樂園所舉辦的「キミと愛ドリッシュないと！」。
*從2017年8月到現在，不曾在部落格中放上自己的自拍照。

== 特徵 ==
*興趣是卡拉OK、欣賞動漫。喜歡漫畫與GAME，聲優[[悠木碧|悠木碧]]的粉絲。
*喜歡的食物為燉糖漬醬漢堡、不喜歡任何辛辣的食物。
*加入[[早安少女組。|早安少女組。]]之前的志向是成為警察。
*特技是[[小號|小號]]、[[劍道|劍道]]。
*憧憬的前輩與目標為前[[早安少女組。|早安少女組。]]九期成員的[[鞘師里保|鞘師里保]]，現以十期成員[[石田亞佑美|石田亞佑美]]為目標。
*13期二人的目標是希望成為像[[道重さゆみ|道重さゆみ]]、[[田中れいな|田中れいな]]時期般，成為[[早安少女組。|早安少女組。]]中堪稱最強的時期。
