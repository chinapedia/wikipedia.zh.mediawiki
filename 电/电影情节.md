《'''电影情节'''》<ref>{{cite web |url=https://www.5music.com.tw/CDList-C.asp?cdno=431455679043|title=凱蒂·佩芮 花漾年華 |work=五大唱片 |accessdate=2016-10-24 |date=2010-08-31}}</ref>（{{lang-en|'''Not Like the Movies'''}}）是[[美国|美国]]歌手[[凯蒂·佩芮|凯蒂·佩芮]]演唱的一首歌曲，出自其第三张专辑《[[花漾年华|花漾年华]]》。佩芮与该歌的制作人{{le|格雷格·威爾斯|Greg Wells}}共同创作歌曲歌词。2010年8月3日，[[国会音乐集团|国会唱片]]将这首歌曲作为专辑的宣传单曲发行。这首歌是一首以少女角度创作的强力情歌。歌词讲述了一个女孩想像着她的第一次，但她感觉「那实在不对劲」。然而，她意识到就算有这样的经历，她的「王子」依然在「外面」等待着她。歌曲在佩芮与其未来丈夫[[羅素·布蘭德|羅素·布蘭德]]第一次约会之前创作出来。

《电影情节》一曲获得评论界的广泛好评，有些乐评人称其为是一首有力量且美丽的歌曲，同时也是一首令人熟思的情歌，但是感觉到歌曲的歌词极少提到佩芮的未来丈夫罗素·布兰德。《电影情节》同时获得了商业上的成功，登上了美国[[告示牌百强单曲榜|告示牌百强单曲榜]]第53位和[[加拿大百强单曲榜|加拿大百强单曲榜]]第41位。佩芮在她所领衔的{{le|加州之夢巡迴演唱會|California Dreams Tour}}上表演了这首歌曲。2011年2月13日，她连同《[[花漾年華_(歌曲)|花漾年华]]》，在[[第53屆葛萊美獎|第53屆葛萊美獎]]表演了这首歌曲。这一场表演在评论界获得了广泛好评。

== 创作概念 ==
2010年8月，佩芮在[[Youtube|Youtube]]采访关于《[[花漾年华|花漾年华]]》的细节时称《电影情节》是在她完成了[[凯蒂你好巡回演唱会|凯蒂你好巡回演唱会]]后，第一首为专辑所创作的歌曲。这首歌的制作和歌曲协助创作则由格雷格·威爾斯负责，他曾经和佩芮共同创作她的第二张录音室专辑《[[花漾派对|花漾派对]]》。佩芮认为这首歌可以分成两部分，因为她在认识布兰德之前已经开始制作《电影情节》，而他们开始约会之后这首歌才制作完毕。她觉得这首歌对于她来说是「非常特别」的一首歌，歌曲的第一部分描写了她需要挣脱和放开这是很多人声称知道但其实他们并不知道的一些往事。佩芮表示将那些往事中的感觉放进《电影情节》里对她来说是一种解脱<ref name=youtube>{{cite web |url=http://www.youtube.com/watch?v=oKW2WvSUHQI |title=Katy Perry talks "Not Like the Movies" |date=2010-08-03|work=Katy Perry Music |publisher=[[YouTube|YouTube]] |accessdate=2011-09-05|language=en}}</ref>。

《电影情节》是一首强力情歌<ref name="usatoday"/>，时长四分零一秒<ref name="itunes"/>，以[[降A大調|降A大調]]写成，并持续使用四分之四拍的[[小节|小节]]，[[速度_(音樂)|速度]]是每分鐘100拍。佩芮演唱这首歌的音域从F<sub>3</sub>到E{{music|flat}}<sub>5</sub>之间，超过了一个[[純八度|純八度]]<ref name="sheet">{{cite web|url=http://www.musicnotes.com/sheetmusic/mtdVPE.asp?ppn=MN0088658|title=Katy Perry Not Like the Movies – Digital Sheet Music|work=''MusicNotes.com'' |publisher=Alfred Music Publishing |accessdate=2011-09-04|language=en}}</ref> 。这首歌有一个基本序列A{{music|flat}}5–Fm7–Cm–E{{music|flat}}来作为其和弦进程<ref name="sheet"/>。一位《泰若》评论家的观点解释，《电影情节》的歌词是关于一场男女之间的恋爱关系，但是女方却觉得自己并没有沉浸在爱情的甜蜜中，而依旧等待她的梦中情人，或者是那所谓「迷人的王子」<ref>{{cite web |url=http://en.terra.com/music/news/katy_perry_new_song_not_like_the_movies_out_now/oci36714 |title=Katy Perry New Song 'Not Like the Movies' Out Now |work=Terra |publisher=Terra Networks |accessdate=2011-09-04|language=en}}</ref>。
[[File:California_Dreams_Tour_(Not_Like_the_Movies).jpg|220px]]
《MTV新闻》的克里斯·瑞安（Chris Ryan）解释这首歌主要是说当你没有遇到你的理想对象时，你要如何去爱，还有不要经常去想象那种「在大螢幕中的电影魔力爱情故事」<ref name="mtv">{{cite web |url=http://buzzworthy.mtv.com/2010/08/03/song-you-need-to-know-katy-perry-not-like-the-movies/ |title=Song You Need To Know: Katy Perry, 'Not Like The Movies' |publisher=[[Viacom|Viacom]] |work=[[MTV|MTV]] |last=Ryan |first=Chris |accessdate=2010-08-05 |date=2011-09-04|language=en}}</ref>。

PopMatters的史蒂夫·萊特里奇（Steve Leftridge）认为这张单曲有一个「枯燥乏味的」的[[旋律|旋律]]<ref name="popmatters">{{cite web |url=http://www.popmatters.com/pm/review/130887-katy-perry-teenage-dream |title=Katy Perry: Teenage Dream |first=Steve |last=Leftridge |work=PopMatters |publisher=PopMatters, Media Inc |date=2010-10-04 |accessdate=2011-09-04|language=en}}</ref>。其旋律被指与[[布兰妮|布兰妮]]的《[[每一次|每一次]]》和[[伊凡塞斯|伊凡塞斯]]的《我那不朽回忆》（My Immortal）相似<ref name="mtv"/>。《[[今日美国|今日美国]]》的艾丽莎·加德納（Elysa Gardner）和一位来自《肖像》杂志（Portrait Magazine）的匿名评论家的观点中，歌曲的[[副歌|副歌]]中，凯蒂唱道为爱而坚持是「有電影戲劇性的/有一个完美结局」，并之后期望自己是一个「傻女孩」。一位《肖像》杂志的评论家将它和《花漾年华》其他单曲一起评述，而认为它在专辑中是最慢节奏的歌曲。他们写道：「它拿走了专辑里所有的能量，以及所有的快速心跳和无法抗拒爱的感觉，但全都为了一首关于寻找爱情的歌而慢了下来。」<ref name="pm">{{cite web |url=http://www.portraitmagazine.net/reviews/albums/teenagedream.html |title=Katy Perry: Teenage Dream |publisher=Portrait Magazine |accessdate=2011-09-04 |language=en }}{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

==评价和反响==
《电影情节》获得了评论家的赞赏。[[MTV|MTV]]的克里斯·瑞安称这首歌是「相当的美丽」，但注意到其歌词「很少的」谈及了佩芮的未来丈夫羅素·布蘭德<ref name="mtv" />。《今日美国》的艾丽莎·加德納称《电影情节》是一个「不可抗拒的强力情歌」并推荐读者去购买这首歌曲<ref name="usatoday">{{cite web |url=http://www.usatoday.com/life/music/reviews/2010-08-19-katyperry19_VA_N.htm |title=Review: Katy Perry is 'Dream'-ing of Madonna|last=Gardner |first=Elysa |work=[[USA_Today|USA Today]] |date=2010-08-18 |publisher=Gannett Company |accessdate=2011-09-04|language=en}}</ref>。一位《肖像》杂志的编辑作家认为《电影情节》来作为《花漾年华》的最后一首歌曲是一个不错的选择，这样充分展示了歌手不错的歌声，因为她有「对她有利的歌词，而不是只是去单一的尝试。」因此这位评论家觉得这是一首「非常棒的歌曲」。<ref name="pm" />

{{quote box|width=25%|quote=「当凯蒂·佩芮对情歌失去控制，就像《电影情节》一样，为了她那悲剧式的爱情生活而在地上哭泣，她具有[[郊区|郊区]]女孩身份传统而感到骄傲，就像他们在[[荷里活|荷里活]]规模般的情绪崩溃。」|source=《[[滚石杂志|滚石杂志]]》的罗布·謝菲爾德（Rob Sheffield）<ref>{{cite web|url=http://www.rollingstone.com/music/albumreviews/teenage-dream-20100823 |title=Teenage Dream by Katy Perry – Music Review |last=Sheffield |first=Rob |work=[[Rolling_Stone|Rolling Stone]] |date=2010-08-23| publisher=Wenner Media LLC |accessdate=2011-09-04|language=en}}</ref>}}

虽然《电影情节》被发行于[[宣传单曲|宣传单曲]]，但还是成功进入到以[[数位音乐下载|数位下载]]销售为准则的音乐榜单。在2010年8月11日那一周中，《电影情节》以第53名的成绩在美国[[告示牌百大单曲榜|告示牌百大单曲榜]]首度亮相<ref name="billboarddigital">{{Cite news |url=http://www.billboard.com/articles/news/956910/taylor-swift-makes-sparkling-hot-100-entrance |first=Silvio |last=Pietroluongo |location=New York |title=Taylor Swift Makes Sparkling Hot 100 Entrance |work=[[Billboard|Billboard]] |publisher=[[Prometheus_Global_Media|Prometheus Global Media]] |date=2010-08-11 |accessdate=2011-09-04|language=en}}</ref>，却只在该榜上呆了一周。这首歌以卖出57,000数位下载，成功进入了《告示牌》成分榜单[[热辣数位歌曲榜|热辣数位歌曲榜]]第22位。在[[加拿大百强单曲榜|加拿大百强单曲榜]]上，这首歌在榜单的最高位置为第41位<ref name="allmusic">{{Cite web|url=http://allmusic.com/artist/katy-perry-p1010533/charts-awards/billboard-singles|title=Katy Perry Charts & Awards: Billboard Singles|publisher=Rovi Corporation |work=[[Allmusic|Allmusic]] |accessdate=2011-09-04|language=en}}</ref>。

== 现场表演 ==
[[File:KatyPerryBudapest03.png|thumb]]
《电影情节》是凯蒂2011年{{le|加州之夢巡迴演唱會|California Dreams Tour}}的演出曲目。在表演这首歌时，凯蒂拖着一袭巨大的婚礼裙摆，坐在以花缠绕的秋千上，从舞台上慢慢升起，而作为[[佈景|佈景]]的一部分<ref>{{cite web |url=http://www.nme.com/reviews/katy-perry/11946 |title=Live Review: Katy Perry |work=[[NME|NME]] |publisher=IPC Media |first=Emily |last=MacKay |date=2011-04-04 |accessdate=2011-08-24|language=en}}</ref>。在同一时间，一个泡泡机开始射出泡泡，之后，一张床单悬挂在她的后面并展示一些[[卡通|卡通]]动物坠入爱河的片段。观看她其中一场演出的《亞利桑那共和国》（The Arizona Republic）的埃德·莫斯利（Ed Masley）称《电影情节》的表演是「甜蜜的」<ref name="tar">{{cite web |url=http://www.azcentral.com/thingstodo/music/articles/2011/08/04/20110804katy-perry-brings-her-candy-coated-california-dreams-phoenix.html |first=Ed |last=Masley |title=Katy Perry brings candy-coated California Dreams to Phoenix |work=The Arizona Republic |publisher=Gannett Company |date=2011-08-04 |accessdate=2011-08-24|language=en}}</ref>。

这首歌曲还在[[第53屆葛萊美獎|第53屆葛萊美獎]]上演出。她穿着亮色粉红、似「公主般」的裙子，并化着亮彩的妆容。当佩芮演唱《电影情节》的歌词时，她坐着的秋千持续地升上舞台的高处。《MTV新闻》的记者马武塞·施吉比（Mawuse Ziegbe）称其是个「诚挚的禮物」<ref name="mtvgrammy">{{cite web |url=http://www.mtv.com/news/articles/1657862/katy-perry-grammy-performance.jhtml |title=Katy Perry Shares Russell Brand Wedding Footage At Grammys  |publisher=MTV News. Viacom |first=Mawuse |last=Ziegbe |date=2011-02-13 |accessdate=2011-09-05|language=en}}</ref>。《[[時代_(雜誌)|时代]]》杂志的克萊爾·南达特（Claire Suddath）与凯蒂的结婚片段作为连接，说这是「[[情人节|情人节]]」主题的表演，她因为高兴而摇摆着，就像「高中年刊时照片的佈景」。她给这次表演一个南达特「B+」级别<ref name="time">{{cite web |url=http://www.time.com/time/specials/packages/article/0,28804,2048962_2048961_2048945,00.html |title=Katy Perry; 'Not Like the Movies,' 'Teenage Dream' (Medley) |work=[[Time_(magazine)|Time]] |publisher=Time Warner Inc |first=Claire |last=Suddath |date=2011-02-14 |accessdate=2011-09-05|language=en}}</ref>。《星星论坛》（Star Tribune）的乔恩·鲷（Jon Bream）给这次表演一个B等级<ref>{{cite web |url=http://www.startribune.com/entertainment/blogs/116143149.html |title=Grading the Grammy performances |first=Jon |last=Bream |work=Star Tribune |publisher=Michael J. Klingensmith |date=2011-02-14 |accessdate=2011-09-05|language=en}}</ref>。
