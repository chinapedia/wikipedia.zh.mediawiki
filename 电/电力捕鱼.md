'''电力捕鱼'''（{{lang-en|'''Electrofishing'''}}），简称'''电鱼'''，是使用[[电|电力]]进行[[捕鱼|捕鱼]]。电力捕鱼的效果是捕鱼数量多，成本低，操作简单，这种方法被广泛使用，目前，这种捕鱼方法在[[中国大陆|中国大陆]]是被法律禁止的。<ref name="fish" >{{cite web |url=http://cj.t56.net/html/2012/rdpl_0306/7971.html |title=电网捕鱼为何屡禁不止？ |accessdate=2012 |publisher=泰州网 |work= |language=zh-hans}}</ref>

==危害==
电力捕鱼期间，不论大小鱼类，全部都难以逃脱被电死的结果，虽然有时候鱼类只是被电晕，但是也损坏了鱼类的[[性腺|性腺]]，导致鱼类[[不孕|不孕]]、鱼[[卵|卵]]、虾卵、泥鳅卵等都会被杀死，同时还会对水中的[[无脊椎动物|无脊椎动物]]造成伤害，电力捕鱼所到之处，鱼类灭绝，生态系统受损，渔业资源枯竭。<ref name="fish" /><ref name="bull" >{{cite web |url=http://news.sohu.com/20050314/n224671457.shtml |title=电网捕鱼、殃及子孙 |accessdate=2005 |publisher=搜狐网 |work= |language=zh-hans}}</ref><ref name="bee" >{{cite web |url=http://www.qingdaonews.com/gb/content/2011-11/10/content_9003741.htm |title=浙江渔船来山东青岛捕鱼，所经之处鱼类灭绝 |accessdate=2011 |publisher=青岛新闻网 |work= |language=zh-hans}}</ref>

电力捕鱼对人也有危险，有时候，为了攫取更大的利益，人们常常使用更高的[[电压|电压]]，[[漏电|漏电]]或人在水中防护措施不当，则伤害自身。<ref name="fish" />

电力捕鱼也影响[[水质|水质]]，由于鱼类被电死之后，尸体在水中[[腐烂|腐烂]][[变质|变质]]，造成水质的损坏。<ref name="fish" />

==物理原理==
电力捕鱼时，鱼类对电流强度的不同而产生3中反应，分别是：感电反应、趋阳反应、麻痹反应。

===感电反应===
使用最小电压或临界电压捕鱼时，鱼类感知到电压后，会颤动和逃避。

===趋阳反应===
中等电压捕鱼时，鱼类会迅速向阳极方向逃走，在交流电场的鱼群就会停滞不前。

===麻痹反应===
趋阳电场强度继续提高，鱼类就会昏迷麻痹，呼吸减弱，腹部上翻，最后被电压击毙。

==相關法例==
*《[[中华人民共和国渔业法|中华人民共和国渔业法]]》第三十条规定：“禁止使用炸鱼、毒鱼、电鱼等破坏渔业资源的方法进行捕捞”<ref name="fish" /><ref name="bee" />；《[[中华人民共和国刑法|中华人民共和国刑法]]》：“对违反保护水产资源法规，在禁渔区、禁渔期或者使用禁用工具、方法捕捞水产品，情节严重的，处三年以下[[有期徒刑|有期徒刑]]、拘役、管制或罚金”<ref name="mao" >{{cite web |url= http://www.xsnet.cn/xs/sh/2009/6/8/904243.shtml|title=电力捕鱼屡禁不止 |accessdate=2009 |publisher=萧山网 |work= |language=zh-hans}}</ref>。
*《[[中華民國漁業法|中華民國漁業法]]》第48條非經核准不得使用之採捕方法：採捕水產動植物，不得以左列方法為之：三、使用電氣或其他麻醉物。 中華民國５９年４月３０日起公布施行。
