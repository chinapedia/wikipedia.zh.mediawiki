== 企劃 ==
參酌英國作家[[路易斯·卡羅|路易斯·卡羅]]兒童文學作品《[[爱丽丝梦游仙境|爱丽丝梦游仙境]]》與其續集《[[愛麗絲鏡中奇遇|愛麗絲鏡中奇遇]]》為背景，衍生親情復仇劇。<ref>この段落の出典。{{Cite web |author=島影真奈美 |date=2014-04-05 |url=http://www.excite.co.jp/News/reviewmov/20140405/E1396662980289.html |title=女子の復讐、鍵は「不思議の国のアリス」。上野樹里主演TBS新ドラマ『アリスの棘』第一話試写会レポ |work=エキレビ! |publisher=[[エキサイト|エキサイト]] |accessdate=2014-04-08}}</ref>

== 故事概要 ==
大學醫院的新人醫師・水野明日美。因14歲時手術中父親致死，在意外中得到父親的真實病歷，經15年的計劃，向惡質醫師復仇。
