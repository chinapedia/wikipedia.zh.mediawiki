《'''愛·天使·音樂·寶貝'''》（{{lang-en|''Love. Music. Angel. Baby.''}}）為[[美國|美國]]歌手[[關·史蒂芬妮|關·史蒂芬妮]]發行的首張[[錄音室專輯|錄音室專輯]]，於2004年11月12日透過[[新視鏡唱片|新視鏡唱片]]發行。史蒂芬妮作為搖滾樂隊[[不要懷疑|不要懷疑]]的主唱已發行過五張錄音室專輯，並於2003年初開始準備個人出道作品。在不要懷疑樂隊休息期間，史蒂芬妮開始投入《愛·天使·音樂·寶貝》的專輯創作，作為樂隊以外的個人計劃。史蒂芬妮參與專輯所有歌曲的創作，並與{{tsl|en|André 3000|安德烈 3000||安德烈 3000}}、{{tsl|en|Dallas Austin|達拉斯·奥斯丁||達拉斯·奥斯丁}}、[[德瑞醫生|德瑞醫生]]、{{tsl|en|The Neptunes|海王星 (組合)|海王星}}與[[琳達·佩里|琳達·佩里]]等多名[[詞曲作家|詞曲作家]]與製作人合作。

《愛·天使·音樂·寶貝》的創作意念為聽起來像1980年代作品，並受到[[治療樂隊|治療樂隊]]、[[流行尖端|流行尖端]]、[[辛蒂·羅波|辛蒂·羅波]]與[[麥當娜|麥當娜]]多名歌手影響。專輯揉合各種風格的歌曲，包括[[電音流行|電音流行]]、{{tsl|en|dance-rock|搖滾舞曲||搖滾舞曲}}、[[新浪潮|新浪潮]]、[[当代节奏布鲁斯|節奏藍調]]、[[嘻哈音樂|嘻哈]]、[[靈魂樂|靈魂]]與[[迪斯科|迪斯科]]，而歌曲的歌詞則主要關於時尚與財富。專輯的宣傳包括發行六張於商業上獲得成功的[[單曲|單曲]]以及舉行北美地區的「{{tsl|en|Harajuku Lovers Tour|哈拉美眉巡迴演唱會||哈拉美眉巡迴演唱會}}」。史蒂芬妮亦經常帶同名為{{tsl|en|Harajuku Girls|原宿女孩||原宿女孩}}的伴舞團隊進行專輯宣傳。

《愛·天使·音樂·寶貝》獲得樂評家整體正面的評價，並於[[格林美獎|格林美獎]]獲得{{tsl|en|Grammy Award for Album of the Year|格林美獎年度專輯||年度專輯}}等總共六項提名。專輯首週以309,000張的銷量空降美國[[告示牌二百強專輯榜|告示牌二百強專輯榜]]第7位，並於其後上升至最高第5位。專輯獲多個國家的[[音乐唱片销售认证|複數白金認證]]，並於全球售出超過700萬張。

== 發行背景 ==
史蒂芬妮在樂隊[[不要懷疑|不要懷疑]]的時候便已開始以個人名義與{{tsl|en|Eve (rapper)|夏娃 (說唱歌手)|夏娃}}與[[莫比|莫比]]等歌手合作。在製作樂隊的第五張錄音室專輯《[[搖滾萬萬歲|搖滾萬萬歲]]》時，不要懷疑曾與[[王子_(音樂家)|王子]]、{{tsl|en|The Neptunes|海王星 (組合)|海王星}}與{{tsl|en|David A. Stewart|大衛·史都華||大衛·史都華}}合作，並獲{{tsl|en|Spike Stent|斯佩克·史滕特|馬克「斯佩克」史滕特}}進行專輯{{tsl|en|audio mixing (recorded music)|混音 (音樂錄製)|混音}}。在樂隊進行專輯宣傳巡迴演唱會期間，史蒂芬妮聽到{{tsl|en|Club Nouveau||Club Nouveau}}1987年的單曲《{{tsl|en|Why You Treat Me So Bad||Why You Treat Me So Bad}}》，並考慮錄製一些把1980年代風格融入現代的作品<ref name="scared1">{{cite web |last=Vineyard |first=Jennifer |url=http://www.mtv.com/bands/s/stefani_gwen/news_feature_112204/ |title=Gwen Stefani: Scared Solo |publisher=[[音樂電視網|MTV]] |page=1 |year=2004 |archiveurl=https://web.archive.org/web/20090104203228/http://www.mtv.com/bands/s/stefani_gwen/news_feature_112204/ |archivedate=2009-01-04 |accessdate=2007-02-26}}</ref>。不要懷疑的[[貝斯手|貝斯手]]與史蒂芬妮的前男友{{tsl|en|Tony Kanal|托尼·卡納爾||托尼·卡納爾}}，向她推介了王子、{{tsl|en|Lisa Lisa and Cult Jam|莉莎莉莎與神秘樂團||莉莎莉莎與神秘樂團}}以及{{tsl|en|Debbie Deb|黛比·德布||黛比·德布}}的音樂，其後二人在卡納爾的睡房裡談論了關於製作歌曲的事<ref>{{cite web |last=Eliscu |first=Jenny |url=https://www.theguardian.com/music/2005/jan/30/popandrock.gwenstefani |title='I'll cry just talking about it' |work=[[衛報|The Guardian]] |date=2005-01-30 |accessdate=2013-09-23 |deadurl=no |archiveurl=https://web.archive.org/web/20130927160105/http://www.theguardian.com/music/2005/jan/30/popandrock.gwenstefani |archivedate=2013-09-27 }}</ref>。

史蒂芬妮在2003年展開個人作品的錄製工作<ref name="recording">{{cite web |last1=Moss |first1=Corey |last2=Downey |first2=Ryan |url=http://www.mtv.com/news/1471367/gwen-stefani-recording-solo-material/ |title=Gwen Stefani Recording Solo Material |publisher={{le|MTV新聞|MTV News|MTV News}} |date=2003-04-18 |archiveurl=https://web.archive.org/web/20151119095537/http://www.mtv.com/news/1471367/gwen-stefani-recording-solo-material/ |archivedate=2015-11-19 |accessdate=2015-06-02}}</ref>。她提及自己曾在為原聲帶錄製單曲、繼續與別人的合作以及以化名「GS」發行專輯之間作考慮<ref name="recording"/><ref name="hb3">{{cite web|last=Collins |first=Clark |url=http://www.ew.com/ew/article/0,,1562569_3,00.html |title=Holla Back |work=[[娱乐周刊|Entertainment Weekly]] |page=3 |date=2006-11-22 |archiveurl=https://web.archive.org/web/20131020221431/http://www.ew.com/ew/article/0%2C%2C1562569_3%2C00.html |archivedate=2013-10-20 |accessdate=2015-06-02 |deadurl=yes }}</ref>。{{tsl|en|Jimmy Iovine|吉米·奧文||吉米·奧文}}（新視鏡的總裁與共同創辦人）說服了史蒂芬妮創作這張專輯<ref name="hb3"/>。在與[[琳達·佩里|琳達·佩里]]第二天的會面，二人以史蒂芬妮的[[腦閉塞|腦閉塞]]以及對個人專輯的恐懼為主題創作了一首歌曲。這首歌後來成為歌曲《[[還等什麼?|還等什麼?]]》，作為專輯首張主打[[單曲|單曲]]發行<ref name="scared2">{{cite web |last=Vineyard |first=Jennifer |url=http://www.mtv.com/bands/s/stefani_gwen/news_feature_112204/index2.jhtml |title=Gwen Stefani: Scared Solo |publisher=MTV |page=2 |year=2004 |archiveurl=https://web.archive.org/web/20081202003104/http://www.mtv.com/bands/s/stefani_gwen/news_feature_112204/index2.jhtml |archivedate=2008-12-02 |accessdate=2015-11-28}}</ref>。

當二人開始創作一首史蒂芬妮覺得太過私人的歌曲時，她去找了卡納爾見面。卡納爾向她演奏了一首正在創作的歌曲，而這首歌後來成為專輯最後一張單曲《{{tsl|en|Crash (Gwen Stefani song)|天雷地火||天雷地火}}》。二人嘗試一起創作新作品，但於兩星期後便放棄了。直至六個月後，當二人再次回顧那些被扔在一旁的歌曲時，史蒂芬妮才發覺自己給予自己太多壓力。當她調整好心態好後便再次找其他創作人與製作人合作。史蒂芬妮再次找回琳達·佩里合作，並獲她介紹{{tsl|en|Dallas Austin|達拉斯·奥斯丁||達拉斯·奥斯丁}}，以及包括{{tsl|en|Outkast|流浪者合唱團||流浪者合唱團}}的{{tsl|en|André 3000|安德烈 3000||安德烈 3000}}、海王星、[[德瑞醫生|德瑞醫生]]等多個歌手<ref name="scared2"/><ref name="scared3">{{cite web |last=Vineyard |first=Jennifer |url=http://www.mtv.com/bands/s/stefani_gwen/news_feature_112204/index3.jhtml |title=Gwen Stefani: Scared Solo |publisher=MTV |page=3 |year=2004 |archiveurl=https://web.archive.org/web/20081201084424/http://www.mtv.com/bands/s/stefani_gwen/news_feature_112204/index3.jhtml |archivedate=2008-12-01 |accessdate=2015-11-28}}</ref>。史蒂芬妮於2004年初確認專輯的發行<ref>{{cite web |last=Orshoski |first=Wes |url=http://www.mtv.com/news/1485633/gwen-stefani-warns-watch-out-for-solo-lp-summer-single/ |title=Gwen Stefani Warns, 'Watch Out' For Solo LP, Summer Single |publisher={{le|MTV新聞|MTV News|MTV News}} |date=2004-03-10 |archiveurl=https://web.archive.org/web/20151208093404/http://www.mtv.com/news/1485633/gwen-stefani-warns-watch-out-for-solo-lp-summer-single/ |archivedate=2015-12-08 |accessdate=2015-06-02}}</ref>，並把專輯定位為「舞曲作品」以及「惡趣味」<ref name="scared1"/>。

== 宣傳 ==
[[Image:GwenStefaniTheRealThing.jpg|right]]

:主題目：{{le|哈拉美眉巡迴演唱會|Harajuku Lovers Tour}}

史蒂芬妮於2005年10月16日展開「{{tsl|en|Harajuku Lovers Tour|哈拉美眉巡迴演唱會||哈拉美眉巡迴演唱會}}」，以宣傳《愛·天使·音樂·寶貝》。演唱會只有一個巡迴階段，並在北美地區舉行42場演出，最終於2005年12月21日結束。演唱會的暖場嘉賓包括嘻哈團體[[黑眼豆豆|黑眼豆豆]]、說唱歌手[[M.I.A.|M.I.A.]]以及歌手[[席亞拉|席亞拉]]。演唱會獲得不同的反響，一些評論家讚揚史蒂芬妮在演出的聲音，而有些則從演出的其他方面進行批評。根據《[[告示牌_(雜誌)|告示牌]]》的資料，演唱會37場演出的總票房成績達2,200萬美元，而其中20場演出更爆滿<ref>{{cite web |last=Cohen |first=Jonathan |url=http://www.billboard.com/articles/news/56532/gwen-stefani-unveils-solo-tour-dates |title=Gwen Stefani Unveils Solo Tour Dates |work=Billboard |date=2006-12-01 |accessdate=2014-07-22 |deadurl=no |archiveurl=https://web.archive.org/web/20140922012935/http://www.billboard.com/articles/news/56532/gwen-stefani-unveils-solo-tour-dates |archivedate=2014-09-22 }}</ref>。演唱會的映像其後在2006年12月4日以[[數碼多功能影音光碟|DVD]]的形式發行<ref name="AllMusic live album review">{{cite news|title=Gwen Stefani – ''Harajuku Lovers Live''|url=http://www.allmusic.com/album/harajuku-lovers-live-mw0001232097|accessdate=2017-04-22|publisher=AllMusic|deadurl=no|archiveurl=https://web.archive.org/web/20180106120340/https://www.allmusic.com/album/harajuku-lovers-live-mw0001232097|archivedate=2018-01-06}}</ref>。《愛·天使·音樂·寶貝》的迷你混音專輯亦於2005年11月22日發行，收錄曲包括《紙醉金迷》、《超酷》、《哈啦美眉》與《還等什麼?》等混音版本<ref name="AllMusic extended play">{{cite web|title=Gwen Stefani – ''Love.Angel.Music.Baby.: The Remixes''|url=http://www.allmusic.com/album/loveangelmusicbaby-the-remixes-mw0001224905|publisher=AllMusic|accessdate=2017-04-23|deadurl=no|archiveurl=https://web.archive.org/web/20161011154249/http://www.allmusic.com/album/loveangelmusicbaby-the-remixes-mw0001224905|archivedate=2016-10-11}}</ref>。

== 商業成績 ==
《愛·天使·音樂·寶貝》首週以309,000張的銷量空降[[告示牌二百強專輯榜|告示牌二百強專輯榜]]<ref>{{cite web |last=Whitmire |first=Margo |url=http://www.billboard.com/articles/news/65450/u2s-bomb-explodes-at-no-1 |title=U2's 'Bomb' Explodes At No. 1 |work=Billboard |date=2004-12-01 |accessdate=2007-02-27 |deadurl=no |archiveurl=https://web.archive.org/web/20130702112442/http://www.billboard.com/articles/news/65450/u2s-bomb-explodes-at-no-1 |archivedate=2013-07-02 }}</ref>。在2005年6月18日的週榜，專輯以83,000張的銷量上升至第5位的新高<ref>{{cite web |last=Whitmire |first=Margo |url=http://www.billboard.com/articles/news/62658/careys-emancipation-returns-to-no-1 |title=Carey's 'Emancipation' Returns To No. 1 |work=Billboard |date=2005-06-08 |accessdate=2014-01-25 |deadurl=no |archiveurl=https://web.archive.org/web/20141002194825/http://www.billboard.com/articles/news/62658/careys-emancipation-returns-to-no-1 |archivedate=2014-10-02 }}</ref>。專輯於同年12月獲[[美國唱片業協會|美國唱片業協會]]認證為三白金唱片<ref name="RIAA"/>，截至2009年5月已經在美國售出超過400萬張<ref name="USsales">{{cite web |last=Harding |first=Cortney |url=http://www.billboard.com/articles/news/268611/no-doubt-hits-the-road |title=No Doubt Hits The Road |work=Billboard |date=2009-05-18 |accessdate=2013-02-08 |deadurl=no |archiveurl=https://web.archive.org/web/20140914135018/http://www.billboard.com/articles/news/268611/no-doubt-hits-the-road |archivedate=2014-09-14 }}</ref>。

專輯於歐洲同樣取得商業成功，發行首週以45,484張的銷量空降[[英國專輯排行榜|英國專輯排行榜]]第14位<ref>{{cite web |last=Jones |first=Alan |url=http://www.musicweek.com/news/read/take-that-make-it-three-weeks-atop-singles-chart/035283 |title=Take That make it three weeks atop singles chart |work={{tsl|en|Music Week||Music Week}} |date=2006-12-11 |archiveurl=https://www.webcitation.org/69lXFkrpZ?url=http://www.musicweek.com/news/read/take-that-make-it-three-weeks-atop-singles-chart/035283 |archivedate=2012-08-08 |accessdate=2012-07-03}}</ref>。在發行的第25週，《愛·天使·音樂·寶貝》於2005年5月15日的週榜以21,271張的銷量升至第4位的新高<ref>{{cite web |last=Jones |first=Alan |url=http://www.musicweek.com/news/read/gorillaz-fail-to-dethrone-akon/033944 |title=Gorillaz fail to dethrone Akon |work=Music Week |date=2005-05-16 |archiveurl=https://www.webcitation.org/6R1LJYakY?url=http://www.musicweek.com/news/read/gorillaz-fail-to-dethrone-akon/033944 |archivedate=2014-07-12 |accessdate=2014-01-26}}</ref>。專輯於2005年9月16日獲[[英國唱片業協會|英國唱片業協會]]認證為三白金唱片<ref name="BPI"/>，亦成為英國2005年銷量第20高的專輯<ref name="UK-yearend2005"/>。截至2016年3月，專輯已經在英國售出1,068,242張<ref name="UKsales">{{cite web |last=Jones |first=Alan |url=http://www.musicweek.com/analysis/read/official-charts-analysis-adele-holds-off-james-iggy-at-albums-summit/064396 |title=Official Charts Analysis: Adele holds off James, Iggy at albums summit |work=Music Week |date=2016-03-25 |accessdate=2017-01-23 |subscription=yes |deadurl=no |archiveurl=https://web.archive.org/web/20170202030636/http://www.musicweek.com/analysis/read/official-charts-analysis-adele-holds-off-james-iggy-at-albums-summit/064396 |archivedate=2017-02-02 }}</ref>。專輯亦打進丹麥、芬蘭、愛爾蘭、挪威與瑞典的專輯前10位，以及打進奧地利、比利時、法國、德國、荷蘭以及瑞士專輯榜的前20位<ref name="ultratop"/><ref name="ire"/>。[[國際唱片業協會|國際唱片業協會]]於2005年5月認證專輯為白金唱片，代表在歐洲的銷量達到100萬張<ref name="IFPIeurope"/>。

在澳洲，《愛·天使·音樂·寶貝》於2005年2月連續兩週獲得[[澳大利亞唱片業協會榜|澳大利亞唱片業協會榜]]冠軍，並在榜單上逗留了56週<ref name="aus"/>。專輯成為2005年銷量第4高的專輯<ref name="aus_yearend"/>，並獲[[澳洲唱片業協會|澳洲唱片業協會]]認證為四白金唱片，代表出貨量達到280,000張<ref name="ARIA"/>。專輯在[[加拿大專輯榜|加拿大專輯榜]]獲得非連續的兩週季軍<ref>{{cite web |last=Williams |first=John |url=http://jam.canoe.com/Music/Artists/G/Game/2005/02/02/918106.html |title=The Game stays on top in Canada |publisher={{tsl|en|Jam!||Jam!}} |date=2005-02-02 |accessdate=2015-06-02 |deadurl=yes |archiveurl=https://web.archive.org/web/20170202020802/http://jam.canoe.com/Music/Artists/G/Game/2005/02/02/918106.html |archivedate=2017-02-02 }}</ref>，並在2006年4月獲[[加拿大音樂協會|加拿大音樂協會]]認證為五白金唱片，代表達到500,000張的銷量<ref name="CRIA"/>。截至2006年11月，《愛·天使·音樂·寶貝》於全球一共售出超過700萬張<ref>{{cite web |last=Infantry |first=Ashante |url=https://www.thestar.com/article/125206 |title=Stefani has it all, baby |work=[[多倫多星報|Toronto Star]] |date=2006-11-30 |archiveurl=https://web.archive.org/web/20090831091830/http://www.thestar.com/article/125206 |archivedate=2009-08-31 |accessdate=2010-10-10}}</ref>，並成為2005年全球銷量第12高的專輯<ref name="global"/>。

== 影響 ==

production|嘻哈音樂製作|嘻哈製作人}}[[提姆巴蘭|提姆巴蘭]]及{{tsl|en|Danja (record producer)|丹賈}}創作及製作。費塔朵亦從{{tsl|en|worldbeat|世界節拍}}的創作歌手轉為跟史蒂芬妮一樣向城市當代音樂靠攏的歌手<ref>{{cite web |last=Erlewine |first=Stephen Thomas Furtado: Loose |work=Rolling Stone |date=2006-06-15 |accessdate=2013-02-09 |deadurl=no |archiveurl=https://web.archive.org/web/20140331035128/http://www.rollingstone.com/music/albumreviews/loose-20060615 |archivedate=2014-03-31 |accessdate=2007-07-16}}</ref>。專輯的主打單曲《{{tsl|en|London Bridge (Fergie song)|倫敦大橋 (菲姬歌曲)|倫敦大橋}}》被拿來與《哈啦美眉》相比，而《{{tsl|en|Glamorous (Fergie song)|魅力四射}}》亦被拿來與《紙醉金迷》作比較<ref>{{cite magazine |last=Vincentelli |first=Elisabeth |url=http://www.timeout.com/newyork/music/fergie |title=Fergie: The Dutchess (Interscope) |magazine=[[Time_Out|Time Out New York]] |issue=573 |date=2006-09-21
