《'''愛的幸運曲奇'''》是中国偶像组合[[SNH48|SNH48]]的第三张[[EP|EP]]，分北京版、上海版兩個版本推出，北京在2013年11月25日于SNH48網上官方商城与[[拍拍网|拍拍网]]开始预售<ref>{{Cite web |url=http://www.snh48.com/html/zixun/2013/1123/399.html |title= SNH48最新EP《爱的幸运曲奇》北京专场版11月25日开启预售 |publisher=SNH48官方网站|date=2013-11-23|accessdate=2013-12-28}}</ref>，上海版在11月29日开始预售<ref>{{Cite web |url= http://www.snh48.com/event/s33/ |title= SNH48新年EP『愛的幸運曲奇』 爱的礼物甜蜜奉献|publisher=SNH48官方网站|accessdate=2013-12-28}}</ref><ref>{{Cite web|url=http://www.macaodaily.com/html/2013-11/26/content_856717.htm|title=SNH48穗開唱打響頭炮|publisher=[[澳门日报|澳门日报]]|date=2013-11-26|accessdate=2013-12-28|deadurl=yes|archiveurl=https://web.archive.org/web/20131228154106/http://www.macaodaily.com/html/2013-11/26/content_856717.htm|archivedate=2013-12-28}}</ref>。

EP的北京版与上海版区别在于附赠的握手券使用地区的不同，其餘收录的曲目及附赠的DVD都是相同的。本张EP五首曲目均为[[AKB48|AKB48]]歌曲的中文翻唱。
