《愛在貓咪療癒時》（日語：ねこばん）為2010年10月至12月於多家電視台連播（[[神奈川電視台|tvk]]、[[埼玉電視台|TVS]]、[[千葉電視放送|CTC]]、[[三重電視放送|MTV]]、[[京都放送|KBS]]、[[太陽電視台|SUN]]）及在日本電視台系統的[[札幌電視放送|STV]]、東京電視台系統的[[TVQ九州放送|TVQ]]播出、[[光TV|光TV]]及[[VOD|VOD]]上架播出的連續劇。<ref>同枠では『[[ねこタクシー|ねこタクシー]]』→『[[犬飼さんちの犬|犬飼さんちの犬]]』→『[[幼獣マメシバ|幼獣マメシバ]]』に続けて[[東名阪ネット6|東名阪ネット6]]の動物シリーズの放送となった。</ref>

== 概要 ==
劇中演出的貓咪皆以徵選方式選出，由日本娛樂媒體總合學院（アミューズメントメディア総合学院）配音部門的在校生負責劇中演出的貓咪的配音工作。

* 根本勳：[[伊武雅刀|伊武雅刀]]
: 主角，前電車駕駛員。
* 根本敏子：[[Lily|Lily]]
: 阿勳的妻子。
* 阿勳的女兒：[[片山圭|片山圭]]
: 與雙親分居。
* 阿勲的女婿：[[松田信行|松田信行]]
: 前部下。
* 千鶴：[[大和田萌|大和田萌]]
: 阿勳的孫女。4歳。照片演出。

== 電影版 ==
2011年1月22日以3D版本在日本上映。由原班人馬擔當演出。
