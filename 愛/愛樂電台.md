'''台北愛樂廣播股份有限公司'''（{{lang-en|Philharmonic Radio Taipei Co., Ltd.}}，簡稱'''愛樂電台'''、'''PRT'''），是臺灣第一家古典音樂電台，也是全世界，第一家華人古典音樂電台及資料庫電台。
== 歷史 ==
西元1994年，在「每一個偉大的都會都應該擁有一家古典音樂電台」的理念下，由一群臺灣的中產階級在廣播電台第二梯次開放時，率先發起並申設成功，於1995年11月9日開播，秉持著「臺灣最沒有壓力的聲音」觀念，推廣精緻文化。

頻道全天24小時發聲，開播至今，始終堅持不談政治、不賣藥、不外包時段、多聽音樂少說話、「來自古典，進入生活」的節目理念，以最好聽的音樂提供聽眾「另外一種選擇，而不是另一種同質性的選擇」，以符合多元化社會、市場區隔發展趨勢。

== 愛樂平台-「官網正統，頻道多元」 ==
因應數位匯流時代來臨，愛樂電台於2017年正式轉型愛樂平台，進行各項通路整合，包括：

* 頻道FM99.7、FM90.7
* 樂997APP，2016年6月正式推出
* 官網「就是愛樂」，2017年10月全新改版上架
* 網路爵士頻道(Jazz on Line)
* 網路西洋經典老歌頻道(Oldies but Goodies)
* 節目隨選隨聽(Audio On Demand)
* 電子報
* Shopping Mall
* 音樂會活動
* FB粉絲專頁
* YouTube專頁

== 愛樂電台開播至今，品質第一，得獎記錄居全臺電台之冠 ==
秉持著專業與敬業的精神，愛樂電台創台至今，獲獎已經打破臺灣過去傳統廣播電台的紀錄。並在2004年，第四次參加金鐘獎，創下入圍18項，得獎9項臺灣史上最佳紀錄後，即停止再參與外界任何比賽，轉向國際競爭！
