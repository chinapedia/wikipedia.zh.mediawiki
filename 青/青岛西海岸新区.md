'''青岛西海岸新区'''（简称'''西海岸新区'''），位于[[山东省|山东省]][[青岛市|青岛市]]，范围与[[黄岛区|黄岛区]]行政区域重合，是中国一个以海洋经济发展为主题的[[国家级新区|国家级新区]]。2014年6月3日，国务院正式批准设立青岛西海岸新区，成为中国第9个国家级新区{{参|{{cite|url=http://politics.people.com.cn/n/2014/0610/c1001-25126231.html|title=青岛西海岸新区成第9个国家级新区|publisher=新京报|date=2014-06-10}}}}。

== 区位概况 ==
黄岛新区位于[[胶州湾|胶州湾]]西岸，与青岛市主城区隔湾相望，包括青岛市[[黄岛区|黄岛区]]全部行政区域，其中陆域面积约2096平方公里、海域面积约5000平方公里。新区处于[[京津冀城市群|京津冀]]和[[长江三角洲经济圈|长三角]]两大都市圈之间，[[环渤海经济圈|环渤海经济圈]]的南缘，[[山东半岛蓝色经济区|山东半岛蓝色经济区]]的核心地带，是[[黄河流域|黄河流域]]主要出海通道和[[亚欧大陆桥|亚欧大陆桥]]东部重要端点，与[[韩国|韩国]]、[[朝鲜|朝鲜]]隔[[黄海|黄海]]相望{{参|{{ cite|url=http://qd.ifeng.com/special/sdjjxq/detail_2013_04/25/747457_0.shtml|title=青岛西海岸经济新区简介|publisher=青岛西海岸官网|date=2014-04-25}}}}。新区海岸线282公里，滩涂83平方公里，岛屿21处，沿岸分布自然港湾23处。陆域开发空间广阔，适宜建设用地约450平方公里{{参|{{cite|url=http://www.xihaian.gov.cn/n306146/n306285/index.html|title=自然资源|publisher=青岛西海岸官网|date=2014-06-10|deadurl=yes|archiveurl=https://web.archive.org/web/20140812122030/http://www.xihaian.gov.cn/n306146/n306285/index.html|archivedate=2014-08-12}}}}。青岛西海岸新区区位条件、科技人才、海洋资源、产业基础、政策环境等综合优势明显，具备推进陆海统筹、城乡一体、军民融合发展的独特条件{{参|{{cite|url=http://news.qingdaonews.com/qingdao/2014-06/10/content_10505725.htm|title=青岛西海岸新区诞生始末 五大定位详细解读|publisher=青岛早报|date=2014-06-10}}}}。

== 沿革背景 ==
*2012年1月31日，'''青岛西海岸经济新区'''党工委、管委会正式成立，青岛西海岸经济新区开发建设全面启动。
*2012年3月，青岛市发布《青岛西海岸经济新区发展规划》。根据规划，西海岸经济新区争取成为[[国家级新区|国家级新区]]。
*2012年12月，经[[中华人民共和国国务院|国务院]]批准，调整青岛市部分行政区划，合并原[[黄岛区|黄岛区]]、[[胶南市|胶南市]]为新的黄岛区。为西海岸新区获批国家级新区奠定基础。
*2014年6月3日，国务院通过国函〔2014〕71号文批准同意设立青岛西海岸新区，是为第9个国家级新区。{{参|{{cite|url=http://www.gov.cn/zhengce/content/2014-06/09/content_8870.htm|title=国务院关于同意设立青岛西海岸新区的批复|publisher=中国政府网|date=2014-06-09}}}}
*2017年5月10日，《青岛西海岸新区发展总体规划》经山东省政府同意，通过鲁发改区域〔2017〕489号文公开发布。{{参|{{cite|url=http://www.sdfgw.gov.cn/art/2017/5/13/art_48_249760.html|title=关于印发青岛西海岸新区发展总体规划的通知|publisher=山东省发展和改革委员会网站|date=2017-05-13}}{{Dead link|date=2018年7月 |bot=InternetArchiveBot |fix-attempted=yes }}}}
*2017年12月1日，《山东省青岛西海岸新区条例》经山东省第十二届人民代表大会常务委员会第三十三次会议通过，于2018年1月1日起实施。{{参|{{cite|url=http://www.sdrd.gov.cn/articles/ch00023/201712/f1df5850-ee30-460b-a146-ac616f01fb17.shtml|title=山东省青岛西海岸新区条例|publisher=山东人大门户网站|date=2017-12-01}}}}

== 规划 ==
以海洋经济发展为主题，服务于青岛建设区域性经济中心和国际化城市的发展定位，把建设青岛西海岸新区作为全面实施海洋战略、发展海洋经济的重要举措，为促进东部沿海地区经济率先转型发展、建设海洋强国发挥积极作用{{参|{{cite|url=http://business.sohu.com/20140610/n400624379.shtml|title=青岛西海岸获批国家级新区 海洋经济成发展新引擎|publisher=每日经济新闻|date=2014-04-25}}}}。努力发展成为海洋科技自主创新领航区、深远海开发战略保障基地、军民融合创新示范区、海洋经济国际合作先导区、陆海统筹发展试验区，为探索全国海洋经济科学发展新路径发挥示范作用{{参|name=国家级新区}}{{参|{{cite|url=http://news.bandao.cn/news_html/201406/20140610/news_20140610_2421327.shtml|title=青岛西海岸新区海洋经济唱主角 重点发展六产业|publisher=半岛网-半岛都市报|date=2014-06-10}}}}。

青岛海洋科技实力雄厚，拥有[[985工程|985工程]]院校[[中国海洋大学|中国海洋大学]]和[[青岛海洋科学与技术国家实验室|青岛海洋科学与技术国家实验室]]，以及20多个[[青岛科研院所|国家级科研院所]]，海洋科技人才数量占全国的50%。[[舟山群岛新区|舟山群岛新区]]注重岛屿开发和陆海统筹，而青岛西海岸新区则强调海洋经济对陆域经济的带动。青岛西海岸新区尝试从近海经济向远洋经济发展，通过延长产业链条，带动更多海洋相关的产业发展{{参|{{cite|url=http://money.163.com/14/0610/03/9UBLROAI00253B0H.html|title=青岛西海岸新区获批 拟建深远海开发战略保障基地|publisher=21世纪经济报道|date=2014-06-10}}}}。
