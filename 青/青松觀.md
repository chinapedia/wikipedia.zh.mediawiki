[[File:HK_ChingChungKoon_MainGateway.JPG|thumb]]
'''青松觀'''是一個[[香港|香港]][[道教|道教]][[慈善團體|慈善團體]]，「青松」取其「松柏長青，老當益壯」的意思。青松觀，亦指場所，青松觀的「青松觀建築群」，位於[[新界|新界]][[屯門|屯門]][[麒麟圍|麒麟圍]][[青松觀路|青松觀路]]青松徑。

== 歷史及服務 ==
[[File:HK_ChingChungKoon_HauPoWoon.JPG|thumb]]
[[File:HK_ChingChungTaoistAssociation_Head_Office.JPG|thumb]]青松觀總辦事處]]
青松觀，創立於1950年，屬[[全真教|全真教]]龍門派，[[道派|道派]]溯源於[[廣東|廣東]][[廣州|廣州]][[至寶台|至寶台]]。何啟忠、陸至真、葉至和、盧寶經、易寶雲、陳台鏡等18位創辦人於[[九龍|九龍]][[偉晴街|偉晴街]]建壇，1952年遷壇至[[彌敦道|彌敦道]]，直到1960年購置[[屯門|屯門]][[麒麟圍|麒麟圍]]建立永久觀址，於1974年增設九龍[[大南街|大南街]]觀址和總辦事處。

;贈醫施藥
青松觀自創辦之初於木屋區辦有中醫贈醫贈藥服務；1975年於大南街青松觀內興辦第一所西醫診所，1977年於屯門青松觀內開辦第二所西醫診所，由註冊西醫主診；2003年開辦第一間中醫藥贈診所。

;興學育才
青松觀早於1957年已在[[九龍|九龍]][[窩打老道|窩打老道]]開設青松學校，其後遷往[[汝洲街|汝洲街]]，高峰期學生超逾300人。直到[[香港政府|香港政府]]施行小學免費教育後，已無義學之需才停辦。1980年代初期開始相繼開辦兩所中學、兩所小學及三所幼稚園。

;敬老扶弱
青松觀於1960年代興辦「青松安老院」，其後在屯門觀內增建「晚晴樓」，容納更多長者；1986年於[[元朗|元朗]]開辦「青松護理安老院」；1997年於[[天水圍|天水圍]][[天慈邨|天慈邨]]開辦「青松侯寶垣老人服務中心」。

== 屯門青松觀路的青松觀 ==
[[File:HK_ChingChungKoon_WuTinGateway.JPG|thumb]]
青松觀佔地20萬[[平方英尺|平方英尺]]，除了是信徒道侶聚合參道之宗教場所，亦是中外遊客及香港市民參觀遊覽之道教聖地。觀內園地清雅；建築以「純陽寶殿」為中心，於1961年落成，恭奉著[[呂純陽|呂純陽]]祖師、[[王重陽|王重陽]]祖師和[[丘長春|丘長春]]祖師三代聖像，自純陽殿建成後，「青華堂」、「翊化宮」、「旭日亭」、「眾妙之門」牌樓等也相繼落成，增設池榭樓台、「九龍彩壁」等。觀內也有[[安老院|安老院]]和西醫贈診所。每年四月，觀內都有[[盆景|盆景]][[展覽|展覽]]，展出中國傳統盆栽供大眾[[免費|免費]]觀賞。

2009年香港歷史建築評級重新評定，青松觀10幢包括「純陽殿」、「翊化宮」、「涼亭」、「景華堂和明華堂」、「晚晴樓」、「清華堂」、「梅萼軒」、「乾安樓」、「怡和齋」及「頤怡逸園」原本列為[[香港二級歷史建築列表|二級歷史建築物]]，其中有8幢被剔出名單，只建議「純陽殿」及「翊化宮」評級不變。在新評級下，青松觀計劃在未來5年以1億元，分階段重建作為「青松安老院」的「乾安樓」、「晚晴樓」及「梅萼軒」，新安老院樓高5至6層，並提供療養和善終服務<ref>{{cite web |language=zh-hk |url=http://news.singtao.ca/toronto/2009-04-22/hongkong1240391400d1732778.html |title=青松觀二級除名 重建安老院 |publisher=[[星島日報|''singtao.ca'']] |date=2009-04-22 |accessdate=2009-11-24}}</ref>。

== 青松仙苑 ==
[[File:HK_ChingChungSianYuan.JPG|thumb]]
青松仙苑，位於新界屯門[[青松觀路|青松觀路]][[青新徑|青新徑]]8號。在[[青山醫院|青山醫院]]以下。為市民提供地方安置[[祖先|祖先]][[骨灰|骨灰]]的場所。

2005年青松仙苑擴建進行新靈龕大樓工程，原預計於2006年初落成，其後因工程「爛尾」而需重新招標興建，直到2007年底才完成，工程期間暫託該苑存放先人骨灰者，需繳付一年暫託費用招不滿<ref>{{cite web |language=zh-hk |url=http://hk.apple.nextmedia.com/template/apple/art_main.php?iss_id=20070921&sec_id=4104&art_id=10183787 |title=骨灰龕遲落成招不滿 |publisher=[[蘋果日報|''hk.apple.nextmedia.com'']] |date=2007-09-25 |accessdate=2009-11-24}}</ref>。

2009年9月1日青松仙苑一連五日分批發售700個骨灰龕位，售價由5至20萬港元不等，開售前引來逾千市民輪候，首日140個骨灰龕位以先到先得方式售罄後，青松仙苑突宣布改以抽籤方式發售餘下龕位，引起逾千名排隊認購的善信不滿群起鼓噪<ref>{{cite web |language=zh-hk |url=http://hk.news.yahoo.com/article/090902/3/e0y2.html |title=青松觀賣骨灰龕再起衝突 |publisher=[[雅虎|''hk.news.yahoo.com'']] |date=2009-09-03 |accessdate=2009-11-24}}</ref>。
