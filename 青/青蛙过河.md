{{vgname|青蛙过河|ja=フロッガー|en=Frogger}}是一款1981年的街机游戏，由[[科乐美|科乐美]]开发。游戏中要达成的目标是将青蛙一只一只地引回家。每个青蛙必须穿过马路时躲避车辆，还要度过危机四伏的河流。高级的玩家还能在路上得到一些奖励。该游戏被认为是[[街机游戏黄金时代|街机游戏黄金时代]]的经典作品之一，游戏元素和主题都值得关注。它还是早期街机游戏使用一个以上[[CPU|CPU]]的例子，该游戏需要两个[[Zilog_Z80|Z80]]支持。<ref>{{KLOV game|7857|Frogger}}</ref>

== 游戏元素 ==

一开始玩家有五只青蛙（生命），玩家控制的青蛙一开始出现在屏幕底部。屏幕的下半部分包括一条[[机动车|机动车]]来来往往的马路，在不同的版本中，这些车辆包括[[小汽车|小汽车]]、[[卡车|卡车]]、[[巴士|巴士]]、[[沙滩车|沙滩车]]、[[推土机|推土机]]、[[货车|货车]]、[[的士|的士]]、[[自行车|自行车]]、[[摩托车|摩托车]]等，在马路上水平疾行。屏幕的上半部分包括一条小河，里面有浮木、[[鳄鱼|鳄鱼]]、[[乌龟|乌龟]]，同样也是水平移动。屏幕最顶端有5个洞，是青蛙的目的地。每关都是计时的。

穿过马路后到达中线，玩家必须为过河作好准备。玩家可跳在迅速移动的浮木和龟背上安全过河，但要避免撞上鳄鱼、[[水蛇|水蛇]]和[[水獭|水獭]]。抓甲虫或护送母青蛙，可获得奖励。青蛙到家后，可开始下一关。

青蛙死亡方式有几种：
# 被车撞
# 跳入河水
# 撞上鳄鱼、蛇或獭
# 跳进被鳄鱼进驻的青蛙洞中
# 呆在龟背上过长，以致乌龟潜入水中
# 呆在龟背或浮木上过长，以致木头或乌龟移出屏幕
# 跳进已有青蛙进驻的青蛙洞中
# 跳进青蛙洞旁边或草丛
# 青蛙还没回家，时间用完
