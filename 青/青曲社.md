'''青曲社'''是位于[[陕西省|陕西省]][[西安市|西安市]]的一个[[相声|相声]]社团，成立于2007年<ref name=jinghua>{{Cite web |url= http://epaper.jinghua.cn/html/2014-02/21/content_65298.htm |title= 亏得同行帮衬 走哪儿都不饿 |author= 田超 |date= 2014年2月21日 |publisher= 京华时报 |language= zh-hans |accessdate= 2014年3月16日 |quote=  |deadurl= yes |archiveurl= https://web.archive.org/web/20140308000743/http://epaper.jinghua.cn/html/2014-02/21/content_65298.htm |archivedate= 2014年3月8日 }}</ref>。社团名称蕴含“青云直上，曲故情长”的意思。青曲社茶楼位于[[文昌门|文昌门]]内的[[柏树林街|柏树林街]]。

==发展历程==
[[File:Qing_Qu_She.JPG|thumb]]
青曲社由[[苗阜|苗阜]]创立于2007年，全名为“陕西青年曲艺社”。<ref name=hsw>{{Cite web |url= http://news.hsw.cn/system/2012/05/28/051331246.shtml |title= 青曲社茶楼一周年店庆 本土特色振兴“西北相声” |author=  |date= 2012-05-28 |publisher= 华商网 |language= zh-hans |accessdate= 2014年3月16日 |quote=  }}</ref>

在2014年[[北京卫视|北京卫视]]马年春晚上，青曲社相声演员苗阜、[[王声|王声]]合作的一段《满腹经纶》受到观众欢迎，演出视频也在网上被大量转发。<ref name=jinghua/><ref>{{Cite web |url=http://www.chinanews.com/ad/2014-02-20/735.html |title= 青曲社首次进京•苗阜、王声相声专场 |author=  |date= 2014年2月20日 |publisher= 中国新闻网 |language= zh-hans |accessdate= 2014年3月16日 |quote=  }}</ref>同年[[中央电视台|中央电视台]]元宵晚会，苗阜和王声表演了相声《学富五车》。<ref name=jinghua/>

==成员==
苗阜，陕西[[铜川|铜川]]人，搭档王声的小学同学。曾在[[富平县|富平县]][[梅家坪站|梅家坪站]]当电工<ref name=inewsweek>{{Cite web|url=http://culture.inewsweek.cn/20140228/detail-80019-1.html|title=从满腹辛酸到《满腹经纶》|author=刘子倩|date=2014年2月28日|publisher=中国新闻周刊网|language=zh-hans|accessdate=2014年3月16日|quote=}}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。后创办青曲社，师承相声艺人[[郑小山|郑小山]]。<ref name=hsw/>

王声，陕西铜川人，青曲社秘书长，搭档苗阜的小学同学。毕业于[[陕西师范大学|陕西师范大学]]中文系，曾任出版社编辑。<ref>{{Cite web |url= http://yule.sohu.com/20140326/n397231825.shtml |title= 西安青曲社中兴陕派相声 喜欢相声不需要理由 |author=  |date=  |publisher= 搜狐 |language= zh-hans |accessdate= 2014年3月26日 |quote=  }}</ref><ref name=inewsweek/>

青曲社亦有鹤翔、张再驰、吕纳超、王小遒等演员。
