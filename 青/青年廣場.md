{{engname|青年廣場|Youth Square}}<ref>{{Cite web |url=http://www.youthsquare.hk/en/ |title=青年廣場 官方網址 |access-date=2010-07-11 |archive-url=https://web.archive.org/web/20100717233939/http://www.youthsquare.hk/en/ |archive-date=2010-07-17 |dead-url=yes }}</ref>，前稱{{engname|香港青年發展中心|Hong Kong Centre for Youth Development，HKCYD}}，俗稱'''Y-Square'''、'''YS'''、'''柴灣青年發展中心'''及'''柴灣青年廣場'''，是一座位於[[香港|香港]][[柴灣|柴灣]][[柴灣道|柴灣道]]238號的[[建築|建築]]，為[[香港特別行政區|香港特別行政區]][[民政事務局|民政事務局]]所擁有，現時由2010年開始[[新世界集團|新世界集團]]轄下的新世界設施管理有限公司以非牟利形式負責管理及營運。<ref>[http://news.mingpao.com/pns/dailynews/web_tc/article/20160320/s00002/1458410882591 明報：柴灣青年廣場連年蝕新世界加價續管理]</ref>

青年廣場於2001年6月興建，到2003年12月至2005年7月期間暫停工程，最終於2008年11月落成。建設費總額約7.5億港元，[[香港特別行政區政府|政府]]撥款5億5,090萬港元興建，其餘2億港元由[[華人永遠墳場管理委員會|華人永遠墳場管理委員會]][[贊助|贊助]]。廣場於2009年4月開始為部份設施陸續進行試業，直至2010年3月6日正式啟用。 <ref>[http://www.info.gov.hk/gia/general/200812/30/P200812300205.htm 香港政府公告: 香港青年發展中心管理諮詢委員會]</ref><ref>[http://news.sina.com.hk/cgi-bin/nw/show.cgi/32/1/1/989017/1.html 香港青年發展中心管理諮詢委員會成立] [[星島日報|星島日報]]</ref><ref>[http://www.hab.gov.hk/tc/policy_responsibilities/Social_Harmony_and_Civic_Education/Centre_youth.htm 民政事務局: 柴灣青年發展中心簡介]</ref>設施包括商場（主要由補習社及教育中心租用）、設有643個座位的Y綜藝館、Y劇場、多用途場地及設有148間客房的Y旅舍等。

== 歷史 ==
青年廣場前身為柴灣[[社區中心|社區中心]]、[[環翠里|環翠里]]休憩處及[[柴灣道|柴灣道]][[迴旋處|迴旋處]][[行人天橋|行人天橋]]的柴灣道（樓梯）和[[環翠道|環翠道]]（緩陂樓梯）出口之所在地。

1998年，前任[[香港特別行政區行政長官|行政長官]][[董建華|董建華]]於[[施政報告|施政報告]]提出對[[青少年|青少年]]服務加強重視，宣布耗資9億元將柴灣社區中心[[重建|重建]]為青年發展中心。原計劃樓高24層，總面積為35,000平方米。中心設青少年發展活動使用的場地、由青年[[實業家|實業家]]以及有意推動青年發展的商業機構經營的[[咖啡室|咖啡室]]和商店、以至150間旅舍房間。

不過該建議公佈後，引起各界嘩然。單是[[建築費|建築費]]便預計要花8億5,630萬元。另加顧問費5,200萬元。不少[[立法會議員|議員]]皆批評此計劃太過昂貴，是浪費資源。此外，當時[[市政總署|市政總署]]（今[[康樂及文化事務署|康樂及文化事務署]]）轄下的很多場地租用率欠佳，並沒有增加的必要。<ref>新報：九億元建柴灣青年中心 議員驚呼昂貴 官稱自負盈虧 (1999年6月17日)</ref>

2000年12月，[[香港特別行政區政府|政府]]開始進行柴灣社區中心清拆工程，於2001年5月完成。

到了2001年，政府向[[香港特別行政區立法會|立法會]]申請5.5億元，在柴灣興建青年發展中心，預期在2005年底落成。並以[[自負盈虧|自負盈虧]]模式運作，政府透過[[貸款|貸款]]或資助形式，設立一項營運基金作開辦費，毋須倚靠政府的經常資助，政府將成立有限公司專責管理並委任董事會成員。同年6月，開始進行[[地基|地基]]暨興建主座大樓[[地庫|地庫]]工程，至2003年12月完成。

不過，香港當時經濟環境欠佳，在2003年政府發現中心首10年營運，將有9,000萬的[[赤字|赤字]]。同年年底，青年發展中心的地基和地庫工程完工，由於未能確定中心的營運模式，當局將中心的上蓋建築工程延至2005年中才興建。工程延誤18個月，每個月還要付出20萬，合共360萬的保養地盤費用。直到2005年7月，主座及旅舍兩座大樓建築工程正式開始恢復。

青年廣場是香港首座以青少年發展為主題的綜合中心，由主座大樓及旅舍大樓兩部分組成，分別樓高15層及12層，總樓面面積約42,000平方米。

=== 地庫樓層（LG3至LG1）：Y 綜藝館 ===
{{engname|「Y 綜藝館」|Y Theatre}}是一座可容納643個座位的劇院，觀眾席設上下兩層。台口有兩個獨立的升降台，這個設備在亞洲只有三個劇院設有。

=== 地下 ===
{{engname|「地面廣場」|Plaza}}為露天及全開放式場地，可舉辦草地音樂會、地攤市集等活動。

=== 1樓 ===
{{engname|「Y 展覽平台」|Y Platform}}

=== 2樓 ===
2樓為{{engname|「Y 劇場」|Y Studio}}多用途禮堂，內設224個活動劇院式座椅及活動牆身門間隔，可用作小型文藝表演，亦可用作容納300人的國際性青年會議場地及開放式展覽，甚至進行開放式的展覽之用，最多可容納500人。

=== 3樓至4樓 ===
主座大樓的3樓和4樓，名為{{engname|「Y 生活館」|Y Mart}}，原計劃分別有24個車仔舖和36間店舖，不過到了2010年7月，車仔舖已被移走。原來的位置用作藝術展覽用途。「生活館」現有的38間店舖已全部租出，租戶包括青年創業家、社會企業、文化藝術機構、教育中心等，與其他商場無異。

主座大樓的4樓為{{engname|「美食館」|Food Mall}}，又稱為{{engname|「賞食新天地」|Iconic Square}}，佔地2,171平方米。現有食肆為提供上海菜的「太極樓」、PB Cafe及[[板長壽司|板長壽司]]。

=== 5樓至6樓 ===
主座大樓5樓至6樓為多用途空間，可供公眾進行多項活動。5樓設有3間鋼琴室、1間興趣小組室及1間創意工作室。部份通道現已作為「印‧藝‧廊」展覽區，5樓另設2間辦公室，佔地710呎。

6樓設有音樂工作室、舞蹈劇場、舞蹈室、Y 攀石地帶、及5間多用途室及舞蹈劇場。

=== 7樓至9樓 ===
主座大樓7樓為公民教育及青年事務資源中心，提供一站式的展覽、資料教材庫及活動室等設施。其餘部份為[[民政事務局|民政事務局]]辦公室及新世界設施管理辦公室。8樓及9樓為辦公室，佔地15,302呎。現有的租戶以青年團體為主，8樓租戶包括[[香港青年聯會|香港青年聯會]]、[[香港青少年發展聯會|香港青少年發展聯會]]、[[香港菁英會|香港菁英會]]、[[香港童軍總會|香港童軍總會]]（港島地域）、[[香港青年交流促進會等|香港青年交流促進會等]]。

9樓租戶包括[[香港青年聯會|香港青年聯會]]、[[樂群社會服務處|樂群社會服務處]]、[[香港志願者協會|香港志願者協會]]、[[垂誼樂社|垂誼樂社]]、[[新生精神康復會|新生精神康復會]]、[[國際扶輪3450地區|國際扶輪3450地區]] 、[http://www.unleashfoundation.org/default.php%5D 騁志發展基金]等。

=== 香港藝術學院 ===
此外，[[香港藝術學院|香港藝術學院]]亦於2M，5樓和12樓，設有繪畫，攝影和應用媒體學科課室。行政辦公室位於8樓中庭，圖書館位於9樓玻璃房。

=== Y旅舍 ===
主座大樓及旅舍大樓合共有148個旅舍房間，包括101間雙人房、37間雙人房連露台、8間三人房、1間六人房及1間複式六人房，其中6間雙人房為無障礙房間設於主座9樓。<ref>[http://www.youthsquare.hk/chi/yloft_hostel_rooms/ 青年廣場 - 旅舍客房]</ref>其他設施有自助洗衣房、精品角、商務角、茶水間咖啡室、悠閒區聚會空間、健身區、空中花園及位於14樓的空中緩跑徑等。除了4樓的3間餐廳，12樓也有CocoaLab咖啡室，佔地146.5平方米。<ref>[http://www.youthsquare.hk/chi/yloft_hostel_facilities/ 青年廣場 - 旅舍設施]</ref>

== 建築特色 ==
青年廣場是一座[[鋼筋混凝土|鋼筋混凝土]]建築，以[[玻璃幕牆|玻璃幕牆]]作為[[外牆|外牆]]。[[建築風格|建築風格]]和[[立面|立面]]跟[[日本|日本]][[東京|東京]][[新宿|新宿]]、[[丸之內|丸之內]]、[[日本橋|日本橋]]的[[辦公大樓|辦公大樓]]近似。此建築設計是在2000年6月，由[[民政事務局|民政事務局]]聯同[[香港建築師學會|香港建築師學會]]合辦的國際建築設計比賽中，挑選出以黃德明、朱國勇、謝錦榮及曾永璋的設計團隊的作品為藍本。<ref>[http://www.info.gov.hk/gia/general/200012/07/1207164.htm 香港特區政府新聞處：行政長官頒獎予青年發展中心建築設計比賽得獎者（2000年12月7日）]</ref> 可是不論是柴灣區居民或外界，均指出青年廣場建築設計與以屋苑為主的環境顯得格格不入。<ref>[http://www.inmediahk.net/node/284055 香港獨立媒體：青年發展中心專題（二）：來看看這個「世紀之作」]</ref><ref>[http://news.sina.com.hk/cgi-bin/nw/show.cgi/2/1/1/690982/1.html 香港新浪網新聞：柴灣青年中心「丟空」被轟（來源：星島日報）]</ref><ref>[http://www.facebook.com/topic.php?uid=2246379594&topic=4674 Facebook「U live in Chai Wan?」群組討論帖：柴灣地標]{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

== 活動 ==
青年廣場於2009年10月18日舉辦首個對外[[開放日|開放日]]。<ref>[http://www.youthsquare.hk/detail.php?s=news&t=23&n=a8d8d05ae059e5a4720c59b93b1bee53&l=tc 2009年10月18日星期日 - 開放日]</ref> 活動包括[[香港漫畫|香港漫畫]]歷史展覽、[[漫畫|漫畫]]工作坊、[[香港運動員|香港運動員]][[相片|相片]]展和分享會、戶外[[舞蹈|舞蹈]]表演，以及[[敲擊樂|敲擊樂]]表演和工作坊。該天亦安排導賞服務，帶領參加人士親身參觀廣場各項不同設施。

到了2010年3月6日，舉行青年廣場暨[[青年高峰會|青年高峰會]]開幕日，來自香港、[[中國內地|中國內地]]及[[澳門|澳門]]的600多名青年代表齊聚一堂，以「'''創建未來──青年人面對的挑戰與機遇'''」的主題，分享心得，交流觀點。

2016年4月23日青年廣場與國際義務工作機構義遊及 [[ViuTV|ViuTV]] 攜手舉辦的「求同．傳義」名人沙龍，邀請了[[曾鈺成|曾鈺成]]與[[梁國雄|梁國雄]]同台對談，與主持[[方健儀|方健儀]]一起探索有關文化、價值觀、待人處事方式以至生活習慣等的差異，認識矛盾，並探討旅遊、義務工作、世界公民、朋友及消費等議題，啟發青年人思考當中的意義。<ref>[http://hk.apple.nextmedia.com/enews/realtime/20160426/55030371 蘋果日報：【溫馨遞杯】關心曾鈺成喉嚨唔舒服 長毛被指好HEHE：關你X事]</ref>

== 使用狀況 ==
青年廣場自2009年4月開始分階段啟用以來，設施的使用率一直有所提升。現時主要設施（如Y旅舍、Y綜藝館、Y劇場）均錄得八成的使用率，而辦公室和商舖出租率更接近百分之百。廣場的收回成本比率亦一直維持在百分之五十之上。<ref>[http://www.info.gov.hk/gia/general/201604/13/P201604130486.htm 立法會二十題：柴灣青年廣場的管理及營運（2016年4月13日）]</ref>

2010年3月27日，有立法會議員批評廣場人流冷清，擔心長遠都要由港府「包底」。民政事務局強調廣場仍處於開業階段，會採取措施吸引更多旅客和租戶，又指「[[掟鞋|掟鞋]]」事件有助為廣場宣傳，引起議員嘩然。<ref>[http://orientaldaily.on.cc/cnt/news/20100327/00176_013.html 東方日報：青年廣場冷清靠公帑補貼（2010年3月27日）]</ref> <ref>[http://news.mingpao.com/pns/dailynews/web_tc/article/20160320/s00002/1458410882591 明報：柴灣青年廣場連年蝕新世界加價續管理（2016年3月20日）]</ref>

2010年7月6日，有曾租用青年廣場設施的人士指出，內裡配套設備保養不足及無法給使用者自行設定，更在表演期間發生火警鐘誤鳴，完全不符合實際需要。<ref>[http://orientaldaily.on.cc/cnt/news/20100706/00176_016.html 東方日報：青年廣場舞台多甩漏（2010年7月6日）]</ref>

2012年4月18日，[[審計署|審計署]]發表第五十八號審計報告，批評青年廣場的設施租用率低，選址錯誤，更指出青年廣場的實際營運虧損達3320萬元，比預期的500萬元虧損，高出超過五倍。<ref>{{cite news |title = 青年廣場年蝕3320萬 |date = 2012-04-19 |url = http://news.sina.com.hk/news/5/1/1/2639681/1.html |work = 星島日報 |quote =  |accessdate = 2012-04-26 |deadurl = yes |archiveurl = https://web.archive.org/web/20120426050954/http://news.sina.com.hk/news/5/1/1/2639681/1.html |archivedate = 2012-04-26 }}</ref>

2013年6月，[[香港藝術學院|香港藝術學院]]正式遷入青年廣場。

2016年4月18日，民政事務局指出青年廣場設施使用率一直有所提升，大部分設施使用率均達至十分高的水平。數據顯示，截至今年1月，除辦公室單位及零售單位使用率達100%外，Y劇場使用率為79.6%。<ref>[http://hk.on.cc/hk/bkn/cnt/news/20160418/bkn-20160418184119078-0418_00822_001.html 東方日報：特別財會：柴灣青年廣場蝕3790萬　歷屆最多（2016年4月18日）]</ref>

== 鄰近交通 ==
青年廣場位於[[香港|香港]][[柴灣|柴灣]][[柴灣道|柴灣道]]238號，即柴灣道與[[環翠道|環翠道]]交界，毗鄰[[柴灣戲院大廈|柴灣戲院大廈]]、[[四號幹線|四號幹線]]起點的柴灣道[[迴旋處|迴旋處]]、[[港鐵|港鐵]][[柴灣站|柴灣站]]、[[新翠花園|新翠花園]]、[[環翠商場|環翠商場]]及[[興華邨|興華邨]]，並設有數條相連的有蓋[[行人天橋|行人天橋]]。遊人可由港鐵[[柴灣站|柴灣站]]A出口步行，經新翠花園及柴灣道迴旋處行人天橋（有往「青年廣場」的行人指示牌），抵達主座大樓一樓入口，或由港鐵柴灣站E出口步行，經行人天橋前往Y 綜藝館。附近設有巴士站、專線小巴站等交通配套。
