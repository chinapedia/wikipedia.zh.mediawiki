'''青山里战役'''，又称'''青山里抗日大捷'''，发生在1920年10月21至26日。[[独立军_(朝鲜)|韩国独立军]]与[[日军|日军]]先后在满洲[[和龙县|和龙县]]青山-{里}-的大山裡进行了十余次的战斗。此次战役是韩国独立军历史上取得战果最显著，最有意义的一次战役。由于中方军队的事先通告，2000多名韩国独立军在当地[[中国朝鲜族|朝鲜族]]村民的大力帮助下，借助天时地利，以少胜多，抵挡了5000余名的日军，對其構成重创{{r|金光先|金宇钟}}。

==背景==
[[File:青山里-베개봉_-_panoramio.jpg|thumb]]青山里]]
“[[三一运动|三一运动]]”被日本人镇压后，[[韩国独立运动|韩国独立运动]]的中心转移到了[[满洲|满洲]]。韩国独立军的武装力量也在满洲不断壮大。为了给出兵消灭满洲的[[独立军_(朝鲜)|韩国独立军]]力量提供借口，日本人用钱买通马帮袭击[[珲春市|珲春]]{{r|回忆录|page=146}}。1920年10月2日，数百名日本人收买的马帮武装袭击珲春。日本人为马帮打开了城门。这伙马帮中除了亲日的马帮外，还混有许多反日马帮。反日马帮冲进城门后，捣毁了日本领事馆，杀光了日本警察，拿走领事馆地下室的五六百件武器，并释放了被关押在领事馆的独立军义士，史称“[[珲春事件|珲春事件]]”{{r|回忆录|金宇钟|page=149}}。

1920年10月5日，日本以“珲春事件”为借口，无视中国主权派1万余驻朝日军进犯[[延边朝鲜族自治州|延边]]，并要求中国出兵配合镇压延边反日武装{{r|回忆录|page=150}}。[[张作霖|张作霖]]任命他的部下孟富德旅团长担任消灭韩国独立军的总司令。由于中国人和韩国人对日本人的共同怨恨，孟富德的军队并没有对韩国独立军开火，而是与驻扎在[[汪清县|汪清县]]十里坪的最大规模韩国独立军，[[金佐镇|金佐镇]]将军率领的{{link-ko|北路军政署|북로군정서}}谈判达成协议，北路军政署一个月内搬出西大坡{{r|回忆录|page=158-160}}。

==经过==
[[File:Battle_of_Qingshanli.JPG|thumb]]
按照对中国军队的承诺，北路军政署准备返回韩国掀起更大规模的民族抗日斗争，通过国际舆论给日本巨大的压力。部队行军至西大坡与[[和龙镇|和龙镇]]中间地带时，孟富德派副官王德林通知北路军政署有约两个师的日军正在准备围捕北路军政署。应孟富德之邀，金佐镇派副总裁玄天默、他的秘书和[[李范奭|李范奭]]骑快马到帽儿山去见孟富德。韩国独立军在孟富德那里，得到了更加详细、具体的敌情。孟富德对日军没有事先任何通报的无礼军事冒犯很愤怒，并表示如果需要，他愿意为韩国独立军提供协助。{{r|回忆录|page=172-176}}

1920年10月20日下午4时，北路军政署在[[和龙县|和龙县]]青山里（又叫“三道沟”）与由日军第37旅团、骑兵第27连队、野炮第25连队组成的混成部队相遇，彼此在相距10里左右发现对方{{r|回忆录|page=173-176}}。在10月21至26日的6天里，韩国独立军与日军先后在青山里的白云坪、完楼沟、渔郎村、泉水坪、古洞河、二道沟、三道沟的大山里进行了十余次的战斗。其中白云坪战斗、完楼沟战斗、泉水坪战斗和渔郎村战斗是青山里战役中的四场最大的战斗{{r|金光先|金宇钟}}。通过利用有利地形，以及得到当地[[中国朝鲜族|朝鲜族]]村民的大力帮助，2000多名韩国独立军以少胜多，抵挡了5000多名的日军，日军伤亡惨重，受到重创{{r|金光先|金宇钟}}<ref>{{cite web |url=http://travel.sina.com.cn/china/2014-04-03/0829256060.shtml |title=红色延边州 山山金达莱村村烈士碑 |publisher=新浪网 |accessdate=2016-03-29}}</ref>。

==后续==
1920年11月20日，[[洪范图|洪范图]]、[[池青天|池青天]]的部队，以及大韩独立军、义军府、血诚团、正义军政司等与{{link-ko|北路军政署|북로군정서}}大联合，成立“{{Link-ko|大韩独立军团|대한독립군단}}”，兵力规模达到3500余人{{r|金佐镇}}。1921年6月，大韩国独立军团进入[[俄罗斯|俄罗斯]]参加武装革命{{r|回忆录|page=319}}。

== 争议 ==
日韩皆宣称己方胜利，日方称己方战死11人,負傷24人，朝鲜人死130人,傷90以上,逃亡200人。韩方宣称朝鲜人用3,050 ~ 4,700人击败30,000~35,000名日军，歼灭日军1000余人。

==纪念==
2000年10月，[[和龙市|和龙市]]政府为纪念青山里抗日大捷80周年在和龙青山林场厂西部的山上修建“青山里抗日大捷纪念碑”<ref>{{cite web |url=http://you.ctrip.com/sight/helong1446012/1708118.html |title=青山里抗日大捷纪念碑 |publisher=携程网 |accessdate=2016-03-29}}</ref>。[[金佐镇|金佐镇]]将军的曾孙韩国男演员[[宋一国|宋一国]]多次参加青山里长征活动，并曾为青山里抗日大捷纪念碑献花。
