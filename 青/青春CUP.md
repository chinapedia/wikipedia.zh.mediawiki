《'''青春CUP'''》（{{lang-ja|ちゅーぶら!!}}）是[[日本|日本]]漫画家[[中田ゆみ|中田ゆみ]]的作品，為[[雙葉社|雙葉社]]月刊漫畫雑誌《[[Comic_High!|Comic High!]]》所連載校園漫画。改編的電視動畫於2010年1月於日本播放，漫畫由臺灣東立出版社代理發行。

== 简介 ==
主人公葉山奈由在學校創立貼身衣物社的相关故事。

== 登場人物 ==
; {{nihongo|叶山奈由|葉山 奈由（はやま なゆ）||[[聲優|聲]]：[[茅原实里|茅原实里]]}}
: 本作品主人公，一年级A班的班长。[[双马尾|双马尾]]发型，带着[[眼镜|眼镜]]。入学考试时得到第一名。虽然成绩相当优秀，不过也有[[天然呆|天然呆]]的一面。在祖母和圭吾的影响下，对内衣有强烈的执着，从周围女性的外观就能看出其内衣穿得不对，因為无法保持沉默而突然去抓别人的胸部，而多次被误解。还有着检查别的女孩子穿什么内衣的习惯。
: 小学6年级时，由于穿着[[吊袜带|吊袜带]]上学，受到误解，被同学敬而远之。在初中入学式的时候，由于试穿成人向的艳丽短裤（圭吾造）被同学看到，被误解成在“做[[援助交际|援助交际]]”，不过也因此结识遥和弥子两个朋友。从那之后就对她们关于如何穿着适当的内衣进行指导，并为了让更多的女孩子得到指导，在遥的提案下，决定创立“内衣部”。在有着“下流”、“太让人不好意思”之类固定观念的其它人的偏见、时不时无心的嘲笑面前，在遥、弥子、小町及水野老师的帮助下，创立了内衣同好会，并在其活动中努力着。
: 她的双亲已经去世，现在和义兄圭吾一起生活。祖母好像也是和圭吾同样做内衣设计相关工作的样子，在奈由小时候给了她少年用的“决胜内衣”。
; {{nihongo|白石遥|白石 遥（しらいし はるか）||聲：[[矢作纱友里|矢作纱友里]]}}
: 奈由的同班同学，因為是大阪出身所以有關西腔調。與神宮寺弥子是好友。擔當著三位弟弟的姊姊。
: 小學時代因為自己的胸部比同齡生發育的早且曾被男生們因此給言語上的霸凌，所以在一段時間內對此很自卑。
; {{nihongo|神宮寺弥子|神宮寺 弥子（じんぐうじ やこ）||聲：[[寿美菜子|寿美菜子]]}}
: 奈由的同班同学。家中經營[[劍道|劍道]]道場，母親為學校教師。好像经常在看历史小说的单行本<ref>弥子读的各单行本的标题有：STEP.1『剣豪伝説』、STEP.2『家康的影武者』、STEP.3『戦国武***（以下不明）』、STEP.4『信長的野望』、STEP.5『最後的武士3』等等。其它集数中登场时也有描写她在一边读单行本一边走路的样子。</ref>。
; {{nihongo|天原清乃|天原 清乃（あまはら きよの）||聲：[[日笠阳子|日笠阳子]]}}
: 就读于私立樱南中学1年级E班。喜歡化妝，曾在學校的周會進行服裝儀容檢查時因此仗義執言。
: 家境比其他三人富裕，擔任過模特兒與時尚雜誌的封面人物。對於內衣的知識遠超過奈由所知道的，同時也知道跟情趣內衣有關的一切。

=== 清乃的同班同學 ===
; {{nihongo|小町阳树|小町 陽樹（こまち ひろき）||聲：[[佐藤雄大|佐藤雄大]]}}
: 和清乃同班，E班的班长。因在小學和補習班中多次被葉山奈由奪去第一名，而向奈由宣戰。
: 之後在一半不情願之下加入了內衣同好會，擔任會中唯一的男性會員和助理。

=== 櫻南中学校老師 ===
; {{nihongo|水野环|水野 環（みずの たまき）||聲：[[大原沙耶香|大原沙耶香]]}}
: 私立樱南中学一年级A班的副班主任，教国语课程。擔任內衣同好會的顧問老師，但本人並不願意。曾打籃球不錯，而曾被學生邀請擔任籃球部的顧問老師，但本人拒絕。葉山圭吾是她高中時代的學長。
; {{nihongo|大熊老師|大熊先生（おおくま - ）||聲：[[寶龜克壽|寶龜克壽]]}}
: 私立樱南中学一年级A班的班導，授課科目未知。有著地中海禿頭且戴眼鏡的中年男性。柔道社的顧問老師，在奈由提出內衣社的構想有些驚訝時就非常大方的指導他們社團的創建辦法，然後推薦了「如果是女性就能理解內衣的一切」的環當顧問老師。
; {{nihongo|塚本老師|塚本先生（つかもと - ）||聲：[[真山亞子|真山亞子]]}}
: 負責手工部的顧問老師。
: 在一開始對於奈由在學校內開辦內衣部的構想很不看好，同時要他打退堂鼓。在學園祭上因為他們三人在雜誌封面上曝光的事情而強烈反對他們的立場，最後在看到他們與話劇社共同努力之的成果下才終於認同他們。
; {{nihongo|高遠老師|高遠先生（たかとお - ）||聲：[[淺川悠|淺川悠]]}}
: 私立樱南中学一年级C班的女班導，漫畫研究會的顧問老師。與环互為好友，偶而會互相揉胸部。
; {{nihongo|谷原老師|谷原先生（たにはら - ）||聲：[[中村悠一|中村悠一]]}}
: 附近小學的年輕教師，奈由與紗繪的前任班導，同時是紗繪憧憬的老師。
: 在單方面的批判奈由成立內衣同好會的時候被塚本老師給制止，外表雖然是一般的誠實青年但會認為女性內衣是不該拿出來的東西。

=== 主角的親人與好友 ===
; {{nihongo|叶山圭吾|葉山 圭吾（はやま けいご）||聲：[[樱井孝宏|樱井孝宏]]}}
: 奈由的哥哥，内衣设计师（經常叫女主角幫他試穿他所设计的内衣）。因女主角雙親在她小時候去世，所以便自小照顧女主角。由於故事尾端意外地要調職去京都進行某個大案子所以要搬家，然後也讓奈由在這之間進退兩難。

; {{nihongo|鷺澤葉子|鷺沢 葉子（さぎさわ ようこ）||聲：[[高島雅羅|高島雅羅]]}}
: 奈由的祖母，與圭吾相同為內衣設計師。以前曾經跟圭吾聯手經營過「珠寶牌」名牌。
: 由於孫子女的父母雙亡的關係所以由他撫養該兄妹，有時太過忙碌所以多少會給兩人添麻煩，奈由對於內衣的觀念多數是由此影響來的。

; {{nihongo|康太|コータ||聲：[[高城元氣|高城元氣]]}}
: 弥子的兒時玩伴，經常來到神宮寺道場上課。
