'''青花瓷'''是[[方文山|方文山]]作词、[[鍾興民|鍾興民]]編曲、[[周杰伦|周杰伦]]作曲演唱的[[中国风|中国风]]歌曲。歌曲旋律采用“宫调式”的主旋律<ref>{{Cite web|url = http://ent.163.com/09/0821/14/5H8FPS0J000333J5.html|title = 《青花瓷》《秋风渐》探讨中国风歌曲本质|accessdate = |author = |date = 2009-08-21|publisher = 网易娱乐}}</ref>。2008年，周杰伦参加[[2008年中国中央电视台春节联欢晚会|央视春晚]]演唱青花瓷<ref>{{Cite web|url = http://ent.163.com/08/0110/02/41QJ360G00032DGD.html|title = 周杰伦春晚要唱《青花瓷》|accessdate = |author = |date = 2008-01-10|publisher = 网易}}</ref>。

原本方文山要将歌曲取名为“[[青铜器|青铜器]]”，因青铜器不够浪漫改用[[宋朝|宋朝]]的“[[汝窑|汝窑]]”为歌名，但“汝”字的发音会有不雅联想，因青花瓷浪漫优雅，最终方文山决定使用青花瓷作为歌曲名<ref>{{Cite web|url = http://ent.qq.com/a/20140128/000220.htm|title = 周杰伦《青花瓷》差点被毙 公司：唱青蛙池？|accessdate = |author = |date = 2014-01-28|publisher = 腾讯}}</ref><ref name=":1">{{Cite web|url = http://news.xinhuanet.com/photo/2013-12/23/c_125901699.htm|title = 方文山称养分来自阅读 青花瓷曾被误为“青蛙池”|accessdate = |author = |date = 2013年12月23日|publisher = 中国日报网}}</ref>。青花瓷一开始被周杰伦误听成青蛙池<ref name=":1" /><ref>{{Cite web|url = http://news.youth.cn/yl/201403/t20140323_4905788.htm|title = 方文山：周杰伦曾将“青花瓷”理解成“青蛙池”|accessdate = |author = |date = 2014-03-23|publisher = 中国新闻网}}</ref>。

歌曲获得2008年[[金曲奖|金曲奖]]最佳年度歌曲<ref name=":2">{{Cite web|url = http://ent.qq.com/a/20080706/000105.htm|title = 金曲奖落幕 周杰伦《青花瓷》获最佳年度歌曲|accessdate = |author = 华商报|date = 2008年7月6日|publisher = 腾讯}}</ref>、年度最佳作词奖和年度最佳作曲奖，而周杰伦也获得最佳作曲人、最佳专辑制作人等奖项<ref name=":2">{{Cite web|url = http://ent.qq.com/a/20080706/000105.htm|title = 金曲奖落幕 周杰伦《青花瓷》获最佳年度歌曲|accessdate = |author = 华商报|date = 2008年7月6日|publisher = 腾讯}}</ref>，由于周杰伦缺席，由方文山代为领奖<ref>{{Cite web|url = http://ent.sina.com.cn/y/2008-07-05/20032089085.shtml|title = 获最佳年度歌曲奖：周杰伦《青花瓷》|accessdate = |author = |date = 2008年7月5日|publisher = 新浪娱乐}}</ref>。

== 反响 ==
武汉一所高中将青花瓷歌词作为语文考试默写题目<ref>{{Cite web|url = http://qjwb.zjol.com.cn/html/2007-12/06/content_1862096.htm|title = 周杰伦《青花瓷》成考题引质疑|accessdate = |author = |date = 2007年12月6日|publisher = 钱江晚报}}</ref>，2008年6月13日山东高考考题中，青花瓷出现在基本能力测试考题上<ref>{{Cite web|url = http://www.chinanews.com/yl/ytcf/news/2008/06-13/1280805.shtml|title = 周杰伦《青花瓷》歌词上2008年山东高考考题|accessdate = |author = |date = 2008年6月13日|publisher = 中国新闻网}}</ref>。

青花瓷为了营造朦胧的意境省略了带有语法标志性的词语<ref>{{Cite web|url =http://barb.sznews.com/html/2009-01/06/content_477689.htm|title =歌曲《青花瓷》有语病？|accessdate =|author =|date =2009-01-06|publisher =宝安日报}}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref>{{Cite web|url =http://news.xinhuanet.com/comments/2009-01/26/content_10720380.htm|title =周杰伦《青花瓷》歌词中有病句吗|accessdate =|author =|date =2009年1月26日|publisher =解放日报|deadurl =yes|archiveurl =https://web.archive.org/web/20161109162804/http://news.xinhuanet.com/comments/2009-01/26/content_10720380.htm|archivedate =2016年11月9日}}</ref>。2009年[[元旦|元旦]][[成都|成都]]举行的[[北大|北大]]招生考题中，考题要求学生指出“素胚勾勒出青花笔锋浓转淡，瓶身描绘的牡丹一如你初妆。”有什么语法错误<ref name=":0">{{Cite web|url = http://qnck.cyol.com/content/2009-01/06/content_2497546.htm|title = 《青花瓷》错误有几多？|accessdate = |author = |date = 2009-01-06|publisher = 中国青年网}}</ref>。[[马未都|马未都]]表示“在瓶底书汉隶仿前朝的飘逸”中的“[[汉隶|汉隶]]”应改成“[[小篆|小篆]]”，因为清代乾隆以后青花瓶底上只写小篆不写隶书，还有“临摹宋体落款时却惦记着你”中的“宋”应改成“楷”，因为中国陶瓷史上，[[宋体|宋体]]落款仅见康雍乾三朝[[珐琅|珐琅]]彩瓷器，青花上没有写过宋体字<ref name=":0" /><ref>{{Cite web|url = http://qjwb.zjol.com.cn/html/2008-03/18/content_2106330.htm|title = 收藏家：《青花瓷》歌词有错|accessdate = |author = |date = 2008年3月18日|publisher = 钱江晚报}}</ref>。

青花瓷有非官方改编版本化学版<ref>{{Cite web|url = http://mt.sohu.com/20150722/n417274447.shtml|title = 化学版青花瓷：记住这首歌就记住了中学化学知识点|accessdate = |author = |date = 2015-07-22|publisher = 搜狐}}</ref>，由乐山一中老师邱伟改编，周杰伦称：“这个化学老师很厉害！”并邀请与其同台演唱化学版的青花瓷<ref>{{Cite web|url = http://scnews.newssc.org/system/2012/12/13/013688372.shtml|title = 乐山一中老师受邀与周杰伦同台演绎化学版《青花瓷》|accessdate = |author = |date = 2012-12-13|publisher = 四川新闻网}}</ref>。此外青花瓷还有致敬[[乔布斯|乔布斯]]版<ref>{{Cite web|url = http://v.ifeng.com/news/society/201110/e7550d32-10e2-469e-869a-e08796308c9e.shtml|title = 网友改编歌曲《青花瓷》向乔布斯致敬|accessdate = |author = |date = 2011年10月8日|publisher = 凤凰网}}</ref>。

此外歌曲引发一定的青花瓷收藏热，2008年[[成都|成都]]、[[重庆|重庆]]两地的青花瓷收藏爱好者从2007年的几百人陡增到上万人<ref>{{Cite web|url =http://www.wccdaily.com.cn/new/html/hxdsb/20080730/hxdsb169501.html|title =周杰伦引发青花瓷收藏热|accessdate =|author =|date =2008-07-30|publisher =华西都市报}}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

[[厦门六中|厦门六中]]初中部室内合唱团[[阿卡贝拉|无人声伴奏]]改编了《[[稻香_(歌曲)|稻香]]》和《青花瓷》两首歌曲<ref>{{cite web |title=14名初中生用杯子“敲出”《稻香》：排练一个月，反响不错 |url=http://www.sohu.com/a/225402276_260616 |website=搜狐 |accessdate=2018-06-13 |date=2018-03-12}}</ref>，其中[[周杰伦|周杰伦]]本人曾与合唱团合唱《稻香》一曲<ref>{{cite web |title=厦门六中合唱团和周杰伦同台演出 周董：唱得比我好 |url=http://fj.sina.com.cn/news/s/2018-04-01/detail-ifysuufh4222997.shtml |website=新浪网 |accessdate=2018-06-13 |date=2018-04-01}}</ref>。
