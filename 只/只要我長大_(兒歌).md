'''只要我長大'''是[[臺灣|臺灣]]早期風靡一時的愛國歌曲，[[白景山|白景山]]詞曲。1950年，[[中華文藝獎金委員會|中華文藝獎金委員會]]舉辦徵曲比賽，白景山以《只要我長大》四段詞曲獲第一名；1952年，收於《新選歌謠》第6期，後由[[中央廣播事業管理處|中央廣播事業管理處]]以「中華愛國歌曲」為題灌錄唱片。原版歌詞具有反共的[[政治意識形態|政治意識形態]]。<ref>{{cite web |title=只要我長大 |url=https://audio.nmth.gov.tw/audio/zh-TW/Item/Detail/bcf4eeac-aeaa-45cf-86c5-c0d3b3ba688b |website=臺灣音聲一百年}}</ref><ref>{{cite web |title=只要我長大 |url=http://www.youtaker.com/video/svb5-2e86f2cc15bf451ab3a17525c04197bf001.html |website=優特客}}</ref>之後的新版歌詞，以《哥哥爸爸真偉大》兒歌面貌，傳唱各方，但大部分都只唱第一段歌詞。<ref>{{cite web |title=只要我長大(哥哥爸爸真偉大) |url=https://mojim.com/twy100614x10x30.htm |website=魔鏡歌詞網}}</ref><ref>{{cite web |title=只要我長大 - 快樂童年童歌精選3 |url=https://www.kkbox.com/tw/tc/song/LZZvpaR06XqKHQEpVQEpV0P4-index.html |website=KKBOX}}</ref>

==歌詞==
===舊版===
{{cquote|
哥哥爸爸真偉大，名譽照我家，為國去[[剿匪|剿匪]]，當兵笑哈哈。<br>
走吧！走吧！哥哥爸爸，家事不用您牽掛，只要我長大，只要我長大。

叔叔伯伯真偉大，榮光滿鄉下，救國去剿匪，壯志賽奔馬。<br>
去吧！去吧！叔叔伯伯，我也要把奸匪殺，只要我長大，只要我長大。

街坊鄰居真偉大，造福為人家，[[反攻大陸|反攻]]去剿匪，生死全不怕。<br>
幹吧！幹吧！街坊鄰家，我也挺身去參加，只要我長大，只要我長大。

革命軍人真偉大，四海把名誇，[[反共抗俄|抗俄]]去剿匪，犧牲為國家。<br>
殺吧！殺吧！革命軍呀，我也要把[[殺朱拔毛|朱毛]]殺，只要我長大，只要我長大。
}}

===新版===
{{cquote|
哥哥爸爸真偉大，名譽照我家，為國去打仗，當兵笑哈哈。<br>
走吧！走吧！哥哥爸爸，家事不用你牽掛，只要我長大，只要我長大。

叔叔伯伯真偉大，榮光滿天下，救國去打仗，壯志賽奔馬。<br>
走吧！走吧！叔叔伯伯，我也挺身去參加，只要我長大，只要我長大。

街坊鄰家真偉大，造福給大家，奮勇去殺敵，生死全不怕。<br>
幹吧！幹吧！街坊鄰家，我也要把敵人殺，只要我長大，只要我長大。

革命軍人真偉大，四海把名誇，拼命去殺敵，犧牲為國家。<br>
殺吧！殺吧！革命軍呀，我也要把奸匪殺，只要我長大，只要我長大。
}}
