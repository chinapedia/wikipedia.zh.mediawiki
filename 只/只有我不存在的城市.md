《'''只有我不存在的城市'''》（{{jp|j=僕だけがいない街|hg=ぼくだけがいないまち|lead=yes}}，略稱：'''僕街'''）是[[日本漫畫家|日本漫畫家]][[三部敬|三部敬]]的[[日本漫畫|日本漫畫]]作品，由[[KADOKAWA|KADOKAWA]]出版發行，於2012年6月4日發售的《[[Young_Ace|Young Ace]]》2012年7月號開始連載<ref>{{cite web|url=http://www.amazon.co.jp/dp/B008459536/|title=ヤングエース 2012年 07月号|publisher=[[Amazon.co.jp|Amazon.co.jp]]|accessdate=2016年2月28日|language=ja}}</ref>，於2016年3月4日發售的《Young Ace》2016年4月號結束連載<ref>{{cite web|url=http://www.amazon.co.jp/dp/B01BL2P7WG/|title=ヤングエース 2016年4月号|publisher=[[Amazon.co.jp|Amazon.co.jp]]|accessdate=2016年2月28日|language=ja}}</ref>。

於2014年獲得「[[這本漫畫真厲害！|這本漫畫真厲害！]]」第15位<ref>{{cite web |url= http://www.animenewsnetwork.com/news/2013-12-09/top-manga-ranked-by-kono-manga-ga-sugoi-2014-voters|title= Top Manga Ranked by Kono Manga ga Sugoi 2014 Voters|date= December 9, 2013|accessdate= November 16, 2014|work= [[Anime_News_Network|Anime News Network]]}}</ref>，2014年「[[漫畫大賞|漫畫大賞]]」第二位<ref>{{cite web |url= http://www.animenewsnetwork.com/news/2014-03-27/kaoru-mori-a-bride-story-wins-7th-annual-manga-taisho-award|title= Kaoru Mori's A Bride's Story Wins 7th Annual Manga Taisho Award|date= March 27, 2014|accessdate= November 16, 2014|work= [[Anime_News_Network|Anime News Network]]}}</ref>，2015年「[[漫畫大賞|漫畫大賞]]」第八位<ref>{{cite web |url= http://www.animenewsnetwork.com/news/2015-01-18/8th-manga-taisho-awards-nominates-14-titles/.83431|title= 8th Manga Taisho Awards Nominates 14 Titles|date= January 18, 2015|accessdate= January 19, 2015|work= [[Anime_News_Network|Anime News Network]]}}</ref>。

== 劇情簡介 ==
身為漫畫家'''藤沼悟'''因為現實生活不順遂而持續掙扎，擁有著「再上映」的穿越時空能力，是可主動或被動地將時間反覆倒帶重播，藉此阻止「事件」的發生。而他某天下班回到家時，從家鄉來訪的母親因在超級市場意外目睹到18年前綁架犯的真實身分，然後認為與過去的同班同學的女孩'''雛月加代'''的遇害事件有關，爾後在悟住的公寓遭到殺害，同時悟還被嫁禍成弒親兇手；這時「再上映」能力啟動，自己卻回到18年前……。來到18年前2月15日的'''藤沼悟'''開始決定要阻止整起事件的發生，為了'''雛月加代'''與母親他將出手改寫一切歷史……。

此外另有外傳，故事描述主角'''藤沼悟'''在昏睡狀態中，以另一個視角描述的新故事，成為中學生的'''雛月加代'''和'''小林賢也'''等人的外傳故事。

== 登場人物 ==
=== 主要人物 ===
; {{jp|c=藤沼悟|j=藤沼 悟|hg=ふじぬま さとる|rm=Fujinuma Satoru|extra2=聲優：[[滿島真之介|滿島真之介]]{{small|（成年）}}／[[土屋太鳳|土屋太鳳]]{{small|（童年）}}；電影演員：[[藤原龍也|藤原龍也]]{{small|（成年）}}／[[中川翼|中川翼]]{{small|（童年）}}；電視劇演員：[[古川雄輝|古川雄輝]]{{small|（成年）}}／[[内川蓮生|内川蓮生]]{{small|（童年）}}}}
: 本作男主角，[[北海道|北海道]]出身。1977年3月2日出生<ref name=birthday>當時為「29歳」。通过「僕だけがいない街.com」了解，2015年1月29日第19刷之后的版本全部修正至「29歲」，推测初版是错字。</ref>。2006年5月時為29歲，為發表過作品的漫畫家，在披薩店「Oasi Pizza」打工。1988年時就讀市立美琴小學。期人際關係相當薄弱，容易把在心裡想的話順口說出。擁有'''「再上映」'''的能力，能夠改變過去發生的事情。
: 某次在打工途中啟動能力，為了救人而遭遇到交通事故，雖然幸好只是輕傷，但他的母親因為擔心他而決定暫時搬過來照顧他。後和母親前往超市的回程中，重播的現象再次降臨到藤沼悟身上。這次他沒有發生任何事，然後打工回到家的他，卻看到母親死亡。通常是短時間的時間跳躍，「再上映」時候卻從2006年回到1988年。
: 在第一次時間線，被警方認為是殺害母親的兇手，其後跳回1988年開始第二次時間線，後來經歷失敗後在2006年稍作停留，在被警察拘捕時與真凶視線相對了一下，「再上映」再次回到1988年開始第三次時間線。
: 在第三次時間線，成功阻止三人被殺害，但被連續綁架殺人的真正元兇淹沒在冰冷的湖水中，落入長期植物人狀態，15年後甦醒（2003年8月、漫畫第33話）之後，忘記當時在湖中的記憶與殺人兇手的身分，遇到愛梨後又再昏睡一年多的時間（漫畫第36話，2005年5月11日甦醒，昏睡386天），後來和小林賢也合作取回記憶，並阻止犯人的新計畫。在事件完結後，其再上映能力沒有再出現過，最後在橋下與躲雪的愛梨相遇。
: 動畫版中，最後成為人氣漫畫家，漫畫作品即將動畫化。因為躲雪的關係再次和愛梨在橋下相遇。

; {{jp|c=雛月加代|j=雛月 加代|hg=ひなづき かよ|rm=Hinazuki Kayo|extra2=聲優：[[悠木碧|悠木碧]]{{small|（童年和成年）}}；電影演員：[[鈴木梨央|鈴木梨央]]{{small|（童年）}}／[[森寬和|森寬和]]{{small|（成年）}}；電視劇演員：[[柿原梨花|柿原梨花]]{{small|（童年）}}／[[吉谷彩子|吉谷彩子]]{{small|（成年）}}}}
: 悟的小學同班同學，北海道出身。1977年3月2日出生（與悟同一天）。非常擅長黑白棋。性格封閉內向，曾寫下'''《只有我不在的街道》'''（{{jp|j=私だけがいない街|lead=yes}}）作文獨白。舊姓為高島，母親遭受父親暴力相待後離異，隨母親居住的單親家庭小孩。1988年期間經常被母親施暴，總是遍體鱗傷地被關在置物間。因為母親總是在假日進行毆打，因此星期一常常遲到。
: 在第一次時間線，為1988年連續誘拐殺人事件的第一位被害者。3月1日失蹤之前，當日悟放學後曾經看到雛月獨自在公園徘徊的身影，卻沒有出聲叫喚。（屍體直到融雪後才被發現）。在第二次時間線，在悟第一次回到過去時與悟的關係發生改變，被悟改變自己的內心不再封閉，在3月1日到來時被悟邀請去科技館，成功渡過這一天，然而生日隔天卻失踪。屍體是在儲藏室被發現，死亡時間延遲至3月3日（女兒節）悟的第一次救援行動以失敗告終。
: 在第三次時間線悟從科技館那裡開始行動，成功把雛月安頓在隱密處，並聯同佐知子和八代等使雛月之母失去撫養權。之後雛月與外祖母生活在一起，經八代努力回到班級繼續上課，成為悟的朋友們一份子，不再落單一人。
: 15年前藤沼悟不僅成功改寫雛月加代原本被謀殺的命運，更幫助加代從家暴的陰影走出來，稱悟為其心目中的英雄。15年後當悟甦醒時，已和杉田廣美結婚，曾帶著孩子{{nihongo|'''未来'''|ミライ|Mirai}}前來探望復康中的悟。
: 外傳故事中描寫悟出事後，升上國中的雛月加代，奔走醫院協助藤沼佐知子照料悟的片段。

; {{jp|c=片桐愛梨|j=片桐 愛梨|hg=かたぎり あいり|rm=Katagiri Airi|extra2=聲優：[[赤崎千夏|赤崎千夏]]；電影演員：[[有村架純|有村架純]]；電視劇演員：[[優希美青|優希美青]]}}
: 悟在2006年時的披薩店同事，使一名女高中生。性格開朗大方，對於認定的事情毫不扭捏勇往直前，是個散發太陽般溫暖光輝的女孩。夢想是帶著一台照相機，環繞世界去旅行。愛梨的熱情和存在深深感染悟，對悟的記憶起極關鍵的作用，因為她的信念，推動著悟一次又一次地面對命運的挑戰。 
: 在第一次時間線，當時對於心思冷漠的悟，愛梨卻直爽地稱呼他為'''「尊敬的朋友」'''而進一步認識。雖與悟相差12歲，仍令悟的母親認為「有機會」。在第二次時間線，堅信悟並非弒母兇手，並幫其躲避警察追捕。因此她曾被真犯人到居住地縱火而陷入昏迷，還好被悟和高橋救出。但被警方監視保護，並拜託{{nihongo|母親||聲優：[[中根久美子|中根久美子]]}}偽裝，並逃出醫院與悟會面後，提供兇手可能是'''西園'''的情報，之後因為警方暗中跟蹤自己而使悟被逮捕。
: 在第三次時間線，由於悟回到過去改變歷史，悟昏迷15年後在醫院遇見的愛梨並不認識他。在悟改變了的未來因看不慣偷拍甦醒的悟企圖亂編新聞的記者而搶走記者的相機使底片曝光並大打出手，因而與悟相見，悟也因為愛梨才恢復記憶（動畫是在雛月加代探望時恢復記憶）。最後故事結尾以躲為由而遇見悟。
: 動畫版中刪除愛梨在第三次時間線的劇情（搶走記者相機的人變更為八代學），但故事結尾一樣與悟在大雪的橋下重聚。外傳故事中描寫愛梨在第三次時間線的劇情，是一名攝影師，並且解釋愛梨並不是單純躲雪才遇見悟。

; {{jp|c=藤沼佐知子|j=藤沼 佐知子|hg=ふじぬま さちこ|rm=Fujinuma Sachiko|extra2=聲優：[[高山南|高山南]]；電影演員：[[石田百合子|石田百合子]]；電視劇演員：[[黑谷友香|黑谷友香]]}}
: 悟的母親，52歳（2006年5月）。獨自一人把悟從小的時候把之養大，煮咖哩超級好吃。洞察力高，讀心術一般的發言令她常被悟形容為「妖怪」；在悟的救援行動中不時充當協助的角色。本是石狩電視台報導部的電視播音員。
: 然而在第一次和二次時間線都因為目擊誘拐未遂現場以及找到兇手的線索慘遭刺殺。在第三次的時間線，因為悟改變歷史所以並沒有遭到刺殺。在病房細心的照顧悟長達15年，在悟又陷入昏迷後再度照料一年。是悟最為感謝的人。最終回中，可以得知在悟成為漫畫家後，依然與其一起生活。

; {{jp|c=八代學|j=八代 学|hg=やしろ がく|rm=Yashiro Gaku|extra2=聲優：[[宮本充|宮本充]]；電影演員：[[及川光博|及川光博]]；電視劇演員：[[戶次重幸|戶次重幸]]}}
: 悟在1988年時的小學班級導師。觀察力敏銳，口才便給，聰明並有自己一套做人道理，悟曾將父親的形象投注在八代身。曾經於愛情上碰壁；戒煙後心癮起時要吃糖果減壓，是以車子裡總有一大堆糖果。
: 在第一次時間線，八代在小學畢業致詞中，曾以一段「人總會有不足，把這些不足填好就是人生」的話，一直記在悟的心裡。在第二次時間線中揭露，很早洞悉雛月受到虐待，併已通知相關部門。
: 在第三次時間線中因為悟的行動，與佐知子聯手救回雛月，後來在他努力下雛月回到校園。在悟遇害後仍密切關注著他，其甦醒後在公園吊橋中再次見面。動畫版中則代替愛梨阻止並喝斥偷拍甦醒的悟的記者而與悟相見，在雨中的樓頂與記憶已經恢復的悟相會。
: 曾偽裝市議會議員並改名為{{nihongo|'''西園學'''|にしぞの まなぶ|聲優：大泉一平}}，並和高橋店長家中關係頗好。

; {{jp|c=小林賢也|j=小林 賢也|hg=こばやし けんや|rm=Kobayashi Ken'ya|extra2=聲優：[[大地葉|大地葉]]{{small|（童年）}}／[[柄本佑|柄本佑]]{{small|（成年）}}；電影演員：[[福士誠治|福士誠治]]{{small|（成年）}}；電視劇演員：小田陽翔{{small|（童年）}}／[[白洲迅|白洲迅]]{{small|（成年）}}}}
: 悟在1988年時的小學同學兼好友，同為基地的伙伴，喜歡踢足球，是個觀察力敏銳的少年。
: 由於石狩電視台引發的事件的報導限制，看出擁有大人的意識的悟的變化，而被悟付諸行動的勇氣深深感動。在第二次時間線，發現悟非常在意雛月，亦有份策劃驚喜生日派對，也注意到加代身上的瘀青以及相當於求救信的作文。
: 在第三次時間線，被悟極力想保護雛月的決心感動，並加入了悟的救援行動，阻止了悟對加代之母的襲擊，協助悟阻止雛月加代、中西、杉田的死亡。推崇悟是他心目中的英雄。15年後成為律師，仔細記錄著悟昏迷後事件的後續，並持續追查真正的綁架犯。

=== 其他人物 ===
; {{jp|c=白鳥潤|j=白鳥 潤|hg=しらとり じゅん|rm=Shiratori Jun|extra2=聲優：[[水島大宙|水島大宙]]；電影演員：[[林遣都|林遣都]]；電視劇演員：[[矢野聖人|矢野聖人]]}}
: 悟在1988年時認識的青年，個性十分溫柔體貼，但有些口吃，擅長剪紙工藝。口頭禪是“拿出勇氣”，因而被悟稱作{{nihongo|「勇氣哥」|ユウキさん|}}。
: 被警方認為是連續兒童殺人案兇手，一直不認罪，然而還是被判處死刑。在第三次的時間線，因為悟暗中的行動而得到不在場證明，因此沒被逮捕。15年後因為工作移居東南亞一帶，並與當地女子結婚生子，並在那裡成家，繼承父親{{nihongo|'''清一郎'''||聲優：手塚秀彰}}「白鳥食品」之業。

; {{jp|c=杉田廣美|j=杉田 広美|hg=すぎた ひろみ|rm=Sugita Hiromi|extra2=聲優：[[鬼頭明里|鬼頭明里]]{{small|（童年）}}／[[田丸篤志|田丸篤志]]{{small|（成年）}}；電視劇演員：城桧吏{{small|（童年）}}／[[溝口琢矢|溝口琢矢]]{{small|（成年）}}}}
: 悟在1988年時的小學同學兼好友，同為基地的伙伴，喜歡太空戰士，外表像女孩子。為1988年連續誘拐殺人事件的第三名受害者。
: 在第一次時間線兇手為誤導和嫁禍兇手混淆警察的偵查方向而殺害外貌像女孩子的廣美，用以使警察誤以為凶手不知道廣美是男的。在第三次時間線成功被藤沼悟解救。15年後已成為實習醫生，並和雛月加代結婚並育有一子。

; {{jp|c=和|hg=カズ|rm=Kazu|extra2=聲優：{{tsl|ja|菊池幸利|菊池幸利}}；電視劇演員：三澤拓冬}}
: 悟在1988年時的小學同學兼好友，同為基地的伙伴，喜歡射擊​​遊戲。在第三次時間線，在悟等人向中西彩搭話都失敗的情況下，以其男子氣概吸引到中西彩到祕密基地，在誤打誤撞的情況下使其避開真兇的魔爪。

; {{jp|c=修|hg=おさむ|rm=Osamu|extra2=聲優：[[七瀨彩夏|七瀨彩夏]]；電視劇演員：福田徠冴}}
: 悟在1988年時的小學同學兼好友，同為基地的伙伴，喜歡勇者鬥惡龍。在漫畫和動畫中的造型不盡相同，但都是貪吃鬼。

; {{jp|c=柳原美里|j=柳原 美里|hg=やなぎはら みさと|rm=Yanagihara Misato|extra2=聲優：[[木野日菜|木野日菜]]}}
: 悟在1988年時的小學同學，坐在悟的隔壁。在第三次的時間線中，原本被真兇盯上的加代、中西和廣美在悟的努力下被迫放棄計畫，於是悟推測真兇可能會尋找替代人選下手，美里曾因先前第二次時間線發生的餐費竊盜事件，被誤會成小偷開始被孤立，成為真兇誘騙悟的契機。
: 動畫版中因為悟是為救成為真兇目標的自己而陷入昏迷而發起救助悟的籌款。漫畫裡募捐是由基地的成員們主導的，外傳中則提及確實由美里提出此議，並也挺身參與募款活動。

; {{jp|c=濱田光一|j=浜田 コウイチ|hg=はまだ こういち|rm=Hamada|extra2=聲優：[[吉田麻實|吉田麻實]]}}
: 悟在1988年時的小學同學，為冰球隊好手，為校隊主力，頗受女同學歡迎。體育課時溜冰比賽被分到與悟同組勝出，對於悟在最後衝線突然停頓的舉動不諒解，並為此拉扯悟的衣領表達憤怒。

; {{jp|c=中西彩|j=中西 彩|hg=なかにし あや|rm=Nakanishi Aya|extra2=聲優：[[金子彩花|金子彩花]]；電視劇演員：髙木彩那}}
: 悟在1988年時鄰鎮小學的朋友，因為要上補習班無法與同學玩而為之苦惱，放學後常獨自坐在基地河堤對面的涼亭等待補習班上課，白鳥潤偶爾會與之搭話聊天。動畫與漫畫的外表大不相同。在第一次時間線與第二次時間線，都為1988年連續誘拐殺人事件的第二位受害者。
: 在第三次時間線，在悟等人的努力下，與悟一行人成為朋友而不再成為真兇的殺害目標，因為和的一席話而對和有好感。

; {{jp|c=雛月明美|j=雛月 明美|hg=ひなづき あけみ|rm=Hinazuki Akemi|extra2=聲優：[[岡村明美|岡村明美]]；電影演員：[[安藤玉惠|安藤玉惠]]；電視劇演員：{{link-ja|江口德子|江口のりこ}}}}
: 雛月加代的母親。因為長期被丈夫家暴而帶著加代離婚，但在離婚後反從受害者變加害者，在1988年時長期虐待自己的女兒。在第三次時間線中，最後被悟、佐知子、八代等人阻止暴力行徑，最終八代帶她的母親來跟她道歉而投案自首，並被褫奪監護權交由{{nihongo|雛月祖母||聲優：高橋博子}}領養。

; {{jp|c=高橋店長|j=高橋|hg=たかはし|extra2=聲優：{{tsl|ja|竹内栄治|竹内榮治}}；電影演員：[[高橋努|高橋努]]；電視劇演員：今野浩喜}}
: 悟在2006年時打工的披薩店「Oasi Pizza」店長。在第二次時間線，因擔心愛梨被“殺母嫌疑犯”悟傷害，試圖報警卻被愛梨阻止並痛斥，後在從火災中與悟救出了愛梨並隱瞞悟的行蹤。在第三次時間線中，改變後的未來的他並不認識悟，騎著送外賣的機車與悟擦肩而過。

; {{jp|c=澤田真|j=澤田 真|hg=さわだ まこと|rm=Sawada|extra2=聲優：[[大川透|大川透]]；電影演員：[[杉本哲太|杉本哲太]]；電視劇演員：[[真島秀和|真島秀和]]}}
: 藤沼佐知子在石狩電視台報導部的同事，曾與小時候的悟見過幾次面。在第二次時間線中，悟第一次拯救加代失敗後的未來世界中選擇相信悟，並告訴悟他的母親是因為發現真兇真實身份而被滅口。在第三次時間線中，在悟成功改變未來後與賢也一同追查這一系列連續殺人案的真兇。

; {{jp|c=北村醫生|j=北村|hg=きむら|rm=Kimura|extra2=聲優：[[濱田賢二|濱田賢二]]}}
: 在第三次時間線中是悟的主治醫生，認為悟的甦醒以及快速復健可以稱之為奇蹟，同時也對悟提到奇蹟與人的意志相關。是個不願拿患者當小白鼠的好醫生，對藤沼佐知子烹調的咖喱讚譽有加。

; {{jp|c=北丸久美|j=北丸 久美|hg=きたまる くみ|rm=Kitamaru Kumi|extra2=聲優：{{tsl|ja|宇山玲加|宇山玲加}}}}
: 在第三次時間線中和悟住在同一間醫院的病患，因患有白血病，因其姊捐贈的骨髓而得到治療。與悟及佐知子的感情不錯，對悟似乎有些微的憧憬之情，後來被真兇盯上，誘以逼迫悟與自己對抗。
