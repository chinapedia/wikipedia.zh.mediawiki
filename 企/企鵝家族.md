'''企鵝家族'''（[[英語|英語]]：''Pingu''）是[[瑞士|瑞士]]一部由歐特馬·顧特曼（Otmar Gutmann）<ref name=tg2016>{{Cite web |url=https://www.theguardian.com/tv-and-radio/2016/jan/11/how-we-made-pingu-childrens-tv-animation |title=How we made Pingu |publisher=The Guardian |date=2016-01-11}}</ref>所[[創作|創作]]的[[兒童|兒童]][[黏土動畫|黏土動畫]]，風格為[[歐式動畫|歐式動畫]]及[[3D動畫|3D動畫]]，這部動畫最早是在1986年製作，一開始在瑞士的[[電視臺|電視臺]]播放，後來擴展至各國，每一集的內容不長，約5分鐘。

這部動畫的故事背景在[[南極洲|南極洲]]，講述[[企鵝|企鵝]]PINGU和他住在[[冰屋|冰屋]]的家人以及和他的好朋友[[海豹|海豹]]Robby之間的故事，故事內容詼諧生動，幽默風趣，在這部動畫裡，角色彼此之間使用「企鵝語」及肢體動作如[[手勢|手勢]]等交談，而且對話時，沒有字幕，這也是本片的一大特色。



==遊戲==
GAME BOY遊戲:企鵝家族世界健康企鵝1號(B-AI 企鵝家族)，以及Nintendo DS遊戲:Pingu的興奮♪ 狂歡節!(SQUARE ENIX 酷兒·恩尼克斯)。

== 登場角色 ==
在許多華人中文網頁中，常稱企鵝家族中的企鵝品種為「高烏天伊企鵝」，但實際上官方並無說明品種為何，且也不存在「高烏天伊企鵝」（參見[[企鵝#分類|企鵝列表]]）。

* Pingu：本動畫主角
* Pinga: Pingu 的三歲妹妹(英文版維基是sister)
* Vater：Pingu 的爸爸
* Mutter：Pingu 的媽媽
* Opa：Pingu 的爺爺
* Pingi：Pingu 的女友
* Pingo 和 Punky：Pingu 的朋友兼死對頭
* Robby：Pingu 的海豹朋友
