'''连二硫酸'''（H<sub>2</sub>S<sub>2</sub>O<sub>6</sub>）是一种只能在[[溶液|溶液]]中存在的化合物。<ref name=G&E>{{Greenwood&Earnshaw}} pp. 715-716</ref>

== 性质 ==
连二硫酸是一种较稳定的强酸。室温下，稀的连二硫酸溶液较稳定。溶液被浓缩或者受热时，缓慢歧化分解为[[硫酸|硫酸]]和二氧化硫：

:H<sub>2</sub>S<sub>2</sub>O<sub>6</sub> → SO<sub>2</sub> + H<sub>2</sub>SO<sub>4</sub>

连二硫酸中的硫元素氧化态+5，理因能够参与许多[[氧化还原反应|氧化还原反应]]。但实际上它与多数氧化剂（例如[[高锰酸钾|高锰酸钾]]、[[氯气|氯气]]、[[次氯酸钠|次氯酸钠]]）和一般的还原剂都不发生作用，这可能动力学原因造成的。

[[钠汞齐|钠汞齐]]可将连二硫酸盐还原为[[亚硫酸盐|亚硫酸盐]]或[[连二亚硫酸盐|连二亚硫酸盐]]<ref>{{cite book|title=《无机化学丛书》第五卷：氧硫硒分族|pages=P166-211-212|author=张青莲|isbn=7-03-002238-6|location=北京|publisher=科学出版社|date=1991年11月}}</ref>：
:S<sub>2</sub>O<sub>6</sub><sup>2-</sup> + 2 Na-Hg → 2 SO<sub>3</sub><sup>2-</sup> + 2Na<sup>+</sup> + 2Hg
:S<sub>2</sub>O<sub>6</sub><sup>2-</sup> + 4 Na-Hg + 4 H<sup>+</sup> → S<sub>2</sub>O<sub>4</sub><sup>2-</sup> + 2H<sub>2</sub>O + 4Na<sup>+</sup> + 4Hg

== 盐 ==
{{main|连二硫酸盐}}
虽然电导实验证明，连二硫酸是一种二元酸，但是目前为止还没发现它的[[酸式盐|酸式盐]]。所有的连二硫酸盐都易溶于水。<ref name=G&E />它们是温和的氧化剂和还原剂。连二硫酸根离子的结构与[[乙烷|乙烷]]类似，但两个SO<sub>3</sub>基团采取的构象与乙烷并不相同，这可能是由于键长不同造成的。S—S键长度是2.15 Å，而S—O键的键长短得多，只有1.43 Å。

== 制备 ==
连二硫酸盐可以由氧化[[亚硫酸盐|亚硫酸盐]]的方法制得（[[氧化态|氧化态]]由+4到+5），但在大规模生产中用[[二氧化锰|二氧化锰]]氧化冷的[[二氧化硫|二氧化硫]]水溶液来获得连二硫酸锰：

:2MnO<sub>2</sub> + 3SO<sub>2</sub> → MnS<sub>2</sub>O<sub>6</sub> + MnSO<sub>4</sub>

可以通过[[复分解反应|复分解反应]]将刚才获得连二硫酸锰转化为其他连二硫酸盐：

:Ba<sup>2+</sup>(aq) + MnS<sub>2</sub>O<sub>6</sub>(aq) + MnSO<sub>4</sub>(aq) → BaSO<sub>4</sub>(s)↓ + BaS<sub>2</sub>O<sub>6</sub>·2H<sub>2</sub>O(aq)

高浓度的连二硫酸后来用硫酸与连二硫酸钡反应的方法成功制得：

:BaS<sub>2</sub>O<sub>6</sub>(aq) + H<sub>2</sub>SO<sub>4</sub>(aq) → H<sub>2</sub>S<sub>2</sub>O<sub>6</sub>(aq) + BaSO<sub>4</sub>(s)↓
