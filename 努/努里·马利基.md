'''努里·卡迈勒·马利基'''（[[阿拉伯语|阿拉伯语]]：'''{{rtl-lang|ar|نوري كامل المالكي}}'''，{{bd|1950年|6月20日|catIdx=M}}），前[[伊拉克总理|伊拉克总理]]。2006年4月26日马利基将他在流亡时期用的名字“贾瓦德·马利基”，恢复为原名“努里·马利基”。

== 早年 ==
马利基生于[[什叶派|什叶派]]圣城[[卡尔巴拉|卡尔巴拉]]。早年曾在[[巴格达大学|巴格达大学]]学习阿拉伯文学。

== 流亡和回到伊拉克 ==
马利基1968年加入[[達瓦黨|伊斯蘭達瓦黨]]，1979年7月16日，被發現是遭取締的伊斯蘭達瓦黨的成員後，馬利基逃離伊拉克。據伊斯蘭達瓦黨的網站上的相關簡歷，他在十月離開伊拉克，並經由[[約旦|約旦]]很快逃到[[敘利亞|敘利亞]]，採用化名「賈瓦德」。他於1982年離開[[敘利亞|敘利亞]]及[[伊朗|伊朗]]，並住在[[德黑蘭|德黑蘭]]直到1990年。之後他一直住在大馬士革，直到2003年[[伊拉克戰爭|美國聯軍入侵伊拉克和推翻海珊政權]]。<ref name="islamicdawaparty.com">{{cite web|url=http://www.islamicdawaparty.com/?module=home&fname=leaderdesc.php&id=78|title=Leader Description|publisher=Islamic Dawa Party|accessdate=2 December 2011}}</ref>雖然生活在敘利亞，他仍做為達瓦黨的政治幹事，並發展與真主黨，特別是與伊朗的密切關係，得到該國的支持推翻海珊政權。<ref>{{cite web |url=http://www.lebanonwire.com/0604MLN/06042403CSM.asp |title=New Iraqi Leader Seeks Unity |accessdate=2014-08-13 |0=2011-11-25 |deadurl=yes |archiveurl=https://web.archive.org/web/20111125111553/http://www.lebanonwire.com/0604MLN/06042403CSM.asp |archivedate=2011-11-25 }} ''Christian Science Monitor''</ref>

雖然生活在大馬士革，馬利基仍編輯黨報《Al-Mawqif》並領導達瓦黨大馬士革分部。1990年，他加入了聯合行動委員會，並擔任輪值主席之一。該委員會是一些反海珊的反對派以大馬士革為基地的聯盟。<ref name="islamicdawaparty.com"/>達瓦黨在1992年至1995年間參加了伊拉克國民大會，因為領導出現分歧而退出。<ref>{{cite web|url=http://www.bostonreview.net/BR28.5/cole.html |title=The Iraqi Shiites |accessdate=2014-08-13 |0=2003-12-14 }} ''Boston Review, Juan Cole''</ref> 當海珊倒台後他回到他的祖國伊拉克，馬利基成為[[伊拉克臨時政府|伊拉克臨時政府]]的最高國家委員會副領導人。馬利基是伊拉克过渡政府总理[[易卜拉欣·贾法里|易卜拉欣·贾法里]]的重要盟友，伊斯兰政党达瓦党的“二把手”，同时任达瓦党的发言人。

2005年1月，马利基当选为[[伊拉克過渡國民議會|過渡國民議會]]议员，担任议会安全和国防委员会负责人，並從軍隊和政府中清除前[[復興社會黨|復興社會黨]]官員。2005年10月任伊拉克宪法起草委员会副主席，並起草新[[伊拉克憲法|憲法]]。

== 总理 ==
[[File:Barack_Obama_&_Nouri_al-Maliki_in_Baghdad_4-7-09_2.JPG|thumb]]在[[巴格達|巴格達]]握手]]
在2005年12月的議會選舉中，[[伊拉克團結聯盟|伊拉克團結聯盟]]贏得了多數席位，並提名[[易卜拉欣·賈法裡|易卜拉欣·賈法裡]]為伊拉克的首次戰後總理。2006年4月20日，前过渡政府总理[[易卜拉欣·贾法里|易卜拉欣·贾法里]]宣布不再连任。21日，什叶派政党联盟指定马利基取代贾法里成为新的总理候选人，22日总统[[塔拉巴尼|塔拉巴尼]]任命他组建内阁。

马利基领导的[[什葉派|什葉派]]政府被批评漠視國內佔少數的[[遜尼派|遜尼派]]和[[庫爾德人|庫爾德人]]，造成族群关系紧张。<ref>{{Cite news|url=http://www.voachinese.com/content/iraq-20140625/1944468.html |title=伊拉克总理拒绝组建紧急政府 |publisher=美國之音 |date=2014-06-25}}</ref>2014年，[[伊拉克和黎凡特伊斯兰国|伊拉克和黎凡特伊斯兰国]]的武裝分子接連攻佔伊拉克西部和北部的多个城镇，有二百萬人口的[[摩蘇爾|摩蘇爾]]在6月失守，同月武裝分子已逼近首都[[巴格達|巴格達]]。8月11日，伊拉克总统[[福阿德·马苏姆|福阿德·马苏姆]]邀请[[海德尔·阿巴迪|海德尔·阿巴迪]]担任总理组成新政府。8月14日，马利基宣布放弃权力。
