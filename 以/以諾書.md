'''以諾書'''（{{lang-gez|መጽሐፈ ሄኖክ}}，''mets'iḥāfe hēnoki''；{{lang-en|Book of Enoch}}）是[[啟示文學|啟示文學]]作品之一，自称記載了在[[大洪水|大洪水]]之前[[以諾|以諾]]與[[上帝|上帝]]同行三百年（''詳見[[以諾|以諾]]''）期间所見的[[異象|異象]]。以諾書分成以諾一書（1 Enoch）、以諾二書（2 Enoch）和以諾三書（3 Enoch）三本。大部份的[[基督教會|基督教會]]以及現代的[[猶太教|猶太教]]团体多視其為[[偽經|偽經]]。

== 以諾一書 ==
以諾一書，又名為《埃塞俄比亞以諾啟示錄》，因為它最完整的抄本是用[[阿姆哈拉语|埃塞俄比亞文]]寫成的。此書亦有[[亞蘭文|亞蘭文]]和[[拉丁文|拉丁文]]的抄本殘卷，所以有學者認為其原文可能是以亞蘭文或[[希伯來文|希伯來文]]寫成的。以諾書；其中以諾一書應是在不同時期和不同人寫成的，而成書的地方亦被學者認為是在[[巴勒斯坦|巴勒斯坦]]。在[[死海古卷|死海古卷]]中亦見有其殘卷，因此學者認為[[猶太教|猶太教]]的[[愛色尼人|愛色尼人]]可能時常誦讀這書。

以諾書在[[猶太人|猶太人]]和[[基督徒|基督徒]]當中，以諾一書是最為人熟悉的，而《[[新約|新約]]·[[猶大書|猶大書]]》的作者被認為深受此書影響。在其他兩約之間時期的次經中，以諾一書多次被引用。初期[[基督教會|教會]]及[[教會父老|教父]]們十分重視這書，其中以[[特土良|特土良]]為代表；但第四世紀開始，教會開始改變態度，因為[[奧古斯丁|奧古斯丁]]和[[耶柔米|耶柔米]]也不看重此書。自此以後，此書僅在[[埃塞俄比亞|埃塞俄比亞]][[東正教|東正教]]教會中和[[貝塔以色列|埃賽俄比亞猶太人]]常被提及。

成書時間方面，學者認為此書的成書時間較長，大約由公元前三世紀到一世紀。
=== 書卷 ===
以諾一書共分成五卷：
# 卷一（第一到三十六章）：第一到五章是這本書的導言，描繪[[末世|末世]]時代罪人受罰和義人等到獎賞；第六至三十六章是關於[[天使|天使]]和宇宙的事，講述天使跟人間的女人結合，因此人類敗壞。以諾為這世界向神代求，但失敗。以諾被天使帶往[[陰間|陰間]]、[[耶路撒冷|耶路撒冷]]和其他地方。
# 卷二（第三十七到七十一章）：第三十七到七十一章是比喻或是明喻，講述將來會有的審判、[[彌賽亞|彌賽亞]]、義人和被選者。內容主要講及罪人滅亡和義人得勝。第四十五到五十七章是關於彌賽亞和被選的那一位。彌賽亞的描述和[[以西結書|以西結書]]那人間之子不同，卻跟[[但以理書|但以理書]]所說那從天上而來的人子相近。
# 卷三（第七十二到八十二章）：這是有關天上的光體和[[天文|天文]]現象的描述。此部份內提到曆法的計算將不像[[法利賽人|法利賽人]]以[[陰曆|陰曆]]計算，而是以[[陽曆|陽曆]]。另外，又提到一年將不是三百六十五又四分之一日，而是三百六十四日；這些天文知識主要是啟示將來的日子是如何的混亂。
# 卷四（第八十三到九十章）：這是以諾在夢中所見的異象，主要有兩個部份；第一部份是描述從洪水之前以諾對將來的異象，是關於整個世界和以色列的，以及[[上帝|上帝]]將要以洪水處罰罪惡的世界；第二部份則是由[[亞當|亞當]]到[[猶大·馬加比|馬加比]]時代的[[以色列歷史|以色列歷史]]。
# 卷五（第九十一到一百零七章）：是關於義人與罪人的兩條道路，在第九十一到一百零五章中，以諾預言義人將[[復活|復活]]以及罪人將受罰，還特別提及啟示文學中的星期；就是由以諾開始，「將來」要分成十個日子不一樣的星期。第二個星期是指挪亞的時代，接著是[[亞伯拉罕|亞伯拉罕]]、[[摩西|摩西]]等等；第八個星期是[[猶大·馬加比|馬加比]]的時代，並指出公義將要掌權；而新天新地則將在第十個星期出現。第一百零六到一百零七章講述挪亞的出生，並有關罪惡的增加和洪水的[[預言|預言]]。

另外，以諾一書還有第一百零八章，稱為另一卷以諾所寫的書，是描述罪人受罰的異象，以勉勵義人忍受在世上的痛苦，等待將來的審判。

=== 影響 ===
在[[基督教神學|基督教神學]]上，《以諾一書》是十分重要的著作，因為它幫助讀者了解[[兩約之間|兩約之間]]的[[猶太教|猶太教]]思想；例如，以諾以不同的名字稱呼神，包括：至高者、至聖者和至大者等等。此書又特別解釋[[創世記|創世記]]第六章的一至四節，視此經文為講述天使墜落。在《[[舊約聖經|舊約聖經]]·[[但以理書|但以理書]]》第七章中所描述的「人子」與《[[新約聖經|新約聖經]]》中所指的「天上的彌賽亞」都可以在以諾一書中找到相同的記載，並且指出，人子是那位公義，並且將會審判世界的彌賽亞。此書又以對末世論的描述詳盡而聞名，書中詳細地描述到末日的審判，罪人在陰間永受刑罰，而義人則復活並得到永生。在末日，彌賽亞將再臨，並且祂的王國和新耶路撒冷城將被建立。

以諾一書和但以理書都是[[猶太教|猶太教]]中場重要的啟示文學，並且都對新約聖經中的作者們有很多的影響。與以諾一書最有直接關係的經文是[[猶大書|猶大書]]第十四節到第十五節：「[[亞當|亞當]]的七世孫子以諾曾經預言這些人說：看哪，主帶著祂的千萬聖者降臨，要在眾人身上行審判，證實那一切不敬虔的人所妄行的一切不敬虔的事，又證實不敬虔之罪人所說頂撞祂的剛愎話。」另外，聖經新約對人子、彌賽亞與祂的國度、天使和[[魔鬼|魔鬼]]等思想都有直接或間接地跟以諾一書有關。

== 以諾二書 ==
以諾二書不同於以諾一書，它的來歷、文本的完整性和寫作日期都有很多爭議，所以甚至有部份[[基督宗教|基督宗教]]的宗派視之為[[偽典|偽典]]；但又有神學家認為，書中所記載的內容都與基督宗教對聖經的看法同出一轍。

以諾二書分成四卷：
# 卷一: 第一到二十章。
# 卷二: 第二十一到四十章。
# 卷三: 第四十一到六十章。
# 卷四: 第六十一到七十三章。

== 以諾三書 ==
《以諾三書》是以希伯來文寫成的猶太啟示文學，與《以諾一書》和《以諾二書》沒有直接的關係。

《以諾三書》據稱是出自大祭司以實瑪利之手，記述他得到天使[[梅丹佐|梅丹佐]]('''天使化的以諾''')的幫助，漫遊天庭，看到神的寶座、戰車、和眾天使。

以諾三書分成三卷：
# 卷一: 第一到二十章。
# 卷二: 第二十一到四十章。
# 卷三: 第四十一到四十八章。
