'''以色列工党'''（{{lang-he|העבודה}}，{{transl|he|HaAvoda}}）是一個[[以色列|以色列]][[中間偏左|中間偏左]][[政黨|政黨]]，黨員約30萬人。領導人的祖輩多屬[[歐洲人|歐洲裔]][[猶太人|猶太人]]，不少來自[[波蘭|波蘭]]和[[蘇聯|蘇聯]]的猶太人，比較傾向代表來自[[西方世界|西方世界]]的猶太人利益。

== 歷史 ==
過去工黨曾經長期領導以色列聯合政府，以色列立國以來，從未有單一政黨單獨取得過半數的[[议会_(以色列)|议会]]席位。1948至1977年，1984至1990年，1992至1996年，1999至2003年，2009至2011年，工黨是以色列的[[執政黨|執政黨]]，過去曾執政長達45年。

2007年6月12日，举行工党主席选举，[[埃胡德·巴拉克|埃胡德·巴拉克]]睽違十年再度当选<ref>{{cite news |language = zh-hans |author = 朱剑慧、马晓燕 |url = http://news.xinhuanet.com/world/2007-06/13/content_6235891.htm |title = 以色列前总理巴拉克当选工党主席 |publisher = [[新華網|新華網]] |date = 2007-06-13 |accessdate =2010-12-28}}</ref>。巴拉克在2007年－2013年擔任以色列國防部長，2010年底巴拉克退出工黨後，工黨退出聯合政府，成為最大反對黨。

2013年底，[[艾薩克·赫爾佐格|艾薩克·赫爾佐格]]當選以色列工黨主席。

== 意識形態 ==
工黨支持[[猶太復國主義|猶太復國主義]]，但亦承認[[巴勒斯坦人|巴勒斯坦人]]的[[自決|自決]]權，不反對建立一個擁有有限主權（如不能擁有[[軍隊|軍隊]]、不能與任何[[國家|國家]][[軍事|軍事]]上結盟、[[以色列空軍|以色列空軍]]享有[[領空|領空]]使用權）的[[巴勒斯坦國|巴勒斯坦國]]，與反對巴勒斯坦立國的[[利庫德集團|利庫德集團]]不同，工黨支持「兩國方案」（Two-state Solution），同意向巴勒斯坦國讓出部分土地（西岸的部分地區），及終止擴建殖民區，以換取和平。
