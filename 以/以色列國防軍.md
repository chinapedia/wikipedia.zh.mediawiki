'''以色列國防軍'''（{{lang-he|צבא ההגנה לישראל}}，{{Audio|He-Israeli Defense Forces.ogg|Tsva HaHagana LeYisrael}}，字面意思为“以色列保衛軍”，{{lang-en|Israel Defense Forces, IDF}}），通常縮寫成Tsahal{{nowrap|（{{rtl-lang|he|צה"ל}}）}}，是[[以色列|以色列]]的军队名称，由陆军、[[以色列海军|海军]]和[[以色列空军|空军]]组成。也有其他一些準軍事的部門負責不同層面的以色列國家安全（例如邊界警察和內部安全局）。

以色列国防军创立于1948年，是[[中東地區|中東地區]]國防預算最高的軍隊之一，作为世界上最有战争经验的武装部隊，在建國半個世紀以來參與了五場主要的大型戰爭和其他無數的小型衝突。<ref>[http://www.mfa.gov.il/mfa The Israeli Ministry of Foreign Affairs] {{webarchive|url=https://web.archive.org/web/20060818011042/http://www.mfa.gov.il/mfa |date=2006-08-18 }}</ref> 在人員上，以色列國防軍的主要優勢是其人員訓練的精良品質以及完善的制度，而不是人員的數量多寡。以色列國防軍也大量依賴於高科技的武器系統，以色列國內也有一些專門替國防軍生產和開發武器科技的機構或企業，其他的則進口自國外（大多來自[[美國|美國]]）。据2016年5月的官方數據，以色列国防军现役部队為164,000人，其中陆军130,000人，海军20,000以及空军14,000。经动员可迅速增加预备役500,000人使武装力量总数达到700,000人。<ref>"The Europa World Year Book, 2005, Volume 1", London & New York: Routledge, 2005, p 2301.ISBN 1-85743-306-8</ref>

以色列實行[[全民皆兵|全民皆兵]]，大多數以色列人（無論男女）都在18歲時[[徵兵制|徵召]]入伍<ref>{{cite web

}}</ref>。有些[[移民|移民]]也會自願加入以色列國防軍，不過[[阿拉伯裔以色列人|阿拉伯裔以色列人]]例外：大多數阿拉伯裔公民都沒有列為徵召對象，以避免以國與其他[[阿拉伯國家|阿拉伯國家]]開戰時可能爆發的[[利益衝突|利益衝突]]。其他不需當兵的還包括殘障人士、已婚婦女、或者出於宗教原因者。男性的義務役役期是三年，女性則是兩年。以色列國防軍裡也有[[貝都因人|貝都因人]]等[[少數族群|少數族群]]參與。自從1956年來，[[德魯茲派|德魯茲]]族群也和猶太人一樣應召進入以色列國防軍。在宗教機構就學的人，可以獲得延緩徵召。多数[[犹太教正統派|犹太教正統派]]人士会不停地延迟服役，直到超出法定服役年龄，这种做法在以色列引起相当大的争议。

雖然阿拉伯裔的公民不在徵兵範圍內，他們也可以自願從軍。同樣的政策也套用在貝都因人和許多非猶太裔的公民上。

服过义务兵役后，以色列男子转入国防军[[预备役|预备役]]部队，每年从事几个星期的服务，直到40岁为止。2015年7月以色列宣佈裁軍40,000人，以因應戰爭型態的改變，以色列情報部門則增加2000人。

== 历史 ==
===前身===
以色列國防軍的前身是[[以色列英管时期|以色列英管时期]]的犹太人[[准军事组织|准军事组织]][[哈加拿|哈加拿]]，包含其动員部门[[帕尔马赫|帕尔马赫]]，這是猶太領土上的长駐武装部隊。
[[File:Spike_LR_3.jpg|缩略图]]

===正式成立===
[[File:Nahal-kumta.jpg|thumb]]
[[File:Israeli_F-16s_at_Red_Flag.jpg|thumb]]
{{Main|以色列國防軍歷史}}
以色列國防軍於1948年5月14日隨著[[以色列|以色列]]的正式建國而成立。其主要目的為：保護以色列之領土、社會不受侵犯；保護以色列居民；與日常生活中的任何危害戰鬥。<ref>"以色列國防軍官方網站"  {{cite web |url=http://www1.idf.il/DOVER/site/homepage.asp?clr=1&sl=EN&id=-8888&force=1 |title=存档副本 |accessdate=2006-08-14 |deadurl=yes |archiveurl=https://web.archive.org/web/20060813113136/http://www1.idf.il/DOVER/site/homepage.asp?clr=1&sl=EN&id=-8888&force=1 |archivedate=2006-08-13 }}</ref>
以色列国防军继承了以色列英管时期的犹太人准军事组织[[哈加纳|哈加纳]]，特别是其动員部门[[帕尔马赫|帕尔马赫]]，成为犹太领土上的长駐武装部隊。二战时期[[英军|英军]]犹太团的主要力量在以色列国防军成立时加入，两个犹太地下组织[[伊尔贡|伊尔贡]]和[[莱希|莱希]]随后也以松散联盟的形式成为其一部分，他们獲准在某些方面独立运作，直到1948年的以色列独立战争，战后伊尔贡和莱希解散其成员则并入以色列国防军。 

以色列国防军利用从1956年到1966年这段相对和平的时间采购了大量新型装备並且完成了将军队从临时组建的武装到专业军队的转变。同时以色列的[[核武|核武]]力量也在此期间得到了大力发展。

經過一段時期的和平，以色列国防军面對一系列的宗教戰爭，阿拉伯鄰國的目標是摧毀猶太人國家。

== 核武能力 ==
[[File:INS_Hetz.JPEG|thumb]]
許多人推測以色列具有使用[[核武器|核武器]]的能力，還有預估認為以色列的軍火庫可能儲藏了高達40枚的核武器<ref>[http://www.janes.com/regional_news/africa_middle_east/news/jir/jir990901_1_n.shtml]</ref>。自從20世紀中旬以來，以色列一直有一座[[內蓋夫_(以色列)|內蓋夫]]核能研究中心負責此領域的研究，並且有能力生產堪足歸類為核武器的材料。這座研究中心從來沒有經過[[國際原子能機構|國際原子能機構]]檢查過，國際原子能機構宣稱他們相信以色列「是個擁有核子武器的國家」，然而以色列政府對此不置可否。由於以色列並非[[不擴散核武器條約|不擴散核武器條約]]的簽署國，雖然以色列核子武器軍火庫的詳細數量一直有待爭議，一般相信以色列擁有至少一百枚核子武器裝置。

有關以色列核武器投射能力的資料比詳細的核武器研究計畫資料還要容易取得。以色列是中東地區研發[[中程彈道飛彈|中程彈道飛彈]]最為領先的國家之一，名為「耶利哥」（Jericho）的[[彈道飛彈|彈道飛彈]]系列早在1970年代便已開始部署，目前已有三個系列：耶利哥一型、二型和三型。耶利哥二型飛彈從1980年代中旬開始服役，預估射程有至少1,500公里，而最新的耶利哥三型飛彈則預估至少有4,500公里射程<ref> {{cite web |url=http://www.wisconsinproject.org/countries/israel/howfar.html |title=存档副本 |accessdate=2007-03-27 |deadurl=yes |archiveurl=https://web.archive.org/web/20061231040821/http://www.wisconsinproject.org/countries/israel/howfar.html |archivedate=2006-12-31 }}</ref>，其他的估計則認為耶利哥三型的射程高達7,800公里<ref>{{Cite web |url=http://www.nuclearfiles.org/menu/key-issues/nuclear-weapons/basics/nuclear-stockpiles.htm |title=存档副本 |access-date=2012-01-29 |archive-url=https://web.archive.org/web/20090418013952/http://www.nuclearfiles.org/menu/key-issues/nuclear-weapons/basics/nuclear-stockpiles.htm |archive-date=2009-04-18 |dead-url=yes }}</ref>。

除了彈道飛彈科技以外，以色列保有完整系列的[[海豚級潛艇|海豚級]]傳統動力[[潛艦|潛艦]]，一般廣泛推測海豚級潛艦搭載有以色列自製的中程[[巡弋飛彈|巡弋飛彈]]（射程可達1450公里），有能力攜帶核子彈頭<ref> http://www.fas.org/news/israel/e20000619israelmakes.htm </ref>。
