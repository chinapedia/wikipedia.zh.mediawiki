《'''以諾遊蹤'''》（[[英文|英文]]：{{lang|en|'''Enoch's Footprint'''}}）是一個[[香港|香港]]製作的宗教旅遊節目，由[[飛躍網絡國際|飛躍網絡國際]]製作，[[電視廣播有限公司|電視廣播有限公司]]購入版權並先後於旗下的[[J2|J2]]、[[高清翡翠台|高清翡翠台]]及[[翡翠台|翡翠台]]播映。節目以[[基督教|基督教]][[聖經|聖經]]的歷史背景作主題，每輯節目均由[[林以諾|林以諾]]牧師主持，並帶同多位藝人參與行程。節目播放完畢後由飛躍網絡國際推出[[DVD|DVD]]銷售。

==節目簡介==
*第一輯節目名為《土耳其七教會之旅》，於[[土耳其|土耳其]]拍攝，拍攝期10天，<ref>[http://www.upwill.org/article/3503.html 《以諾遊蹤》驚喜處處： 邊走、邊看 愛琴海邊嚐甜品] {{webarchive|url=https://web.archive.org/web/20100125174150/http://www.upwill.org/article/3503.html |date=2010-01-25 }} upwill.org</ref>於2009年12月20日至2010年1月24日逢星期日早上9:30-10:00，及晚上8:00-8:30於J2台播放。<ref>[http://www.upwill.org/article/3286.html 《以諾遊蹤》無綫J2台播：超神蹟 林以諾牧師：上帝意念高於人意念] {{webarchive|url=https://web.archive.org/web/20091223194818/http://www.upwill.org/article/3286.html |date=2009-12-23 }} upwill.org</ref>其後曾於高清翡翠台重播。

*第二輯節目《希臘保羅疾風之旅》，林以諾牧師到[[希臘|希臘]]重踏使徒[[保羅|保羅]]宣教足跡，於2010年4月17日起逢星期六下午3:00於高清翡翠台首播，及於[[myTV|myTV]]提供節目重溫。

*第三輯節目《保羅勇闖高峰之旅》，到訪多個使徒保羅曾前往傳道的[[歐洲|歐洲]]城市，<ref>[http://www.upwill.org/news/picture-story/5604-florence 《以諾遊蹤III》在翡冷翠漫步] {{webarchive|url=https://archive.is/20130503131933/http://www.upwill.org/news/picture-story/5604-florence |date=2013-05-03 }} upwill.org</ref>於2010年12月11日起逢星期六下午3:00於高清翡翠台首播，及於myTV提供節目重溫。2月14日至3月21日凌晨於無綫翡翠台重播。<ref>[http://www.upwill.org/channel/5-2010-10-07-01-51-36/5440-2010-11-29-12-49-52 《以諾遊蹤：保羅勇闖高峰》節目簡介 (於無綫翡翠台播放)] {{webarchive|url=https://web.archive.org/web/20110204233530/http://www.upwill.org/channel/5-2010-10-07-01-51-36/5440-2010-11-29-12-49-52 |date=2011-02-04 }} upwill.org</ref>

*第四輯節目《以色列 穌哥行傳》以[[耶穌|耶穌]]的事蹟為主題，於2011年1月18日至30日前往[[以色列|以色列]]拍攝，參與藝人包括[[關心妍|關心妍]]、[[黃智賢|黃智賢]]及[[裕美|裕美]]，行程由[[新幹線旅遊|新幹線旅遊]]贊助；<ref>[http://www.upwill.org/news/daily-news/5788-chinachristian 由農村到城市到全球化 《中國基督徒三十年》展示中國基督教演變] {{webarchive|url=https://web.archive.org/web/20110211092554/http://upwill.org/news/daily-news/5788-chinachristian |date=2011-02-11 }} upwill.org</ref>於2011年4月9日起，逢星期六下午3:30在高清翡翠台播放。<ref>[http://www.gospelherald.com.hk/news/cul-1485/%E3%80%8A%E4%BB%A5%E8%AB%BE%E9%81%8A%E8%B9%A4IV%E4%B9%8B%E7%A9%8C%E5%93%A5%E8%A1%8C%E5%82%B3%E3%80%8B4%E6%9C%889%E6%97%A5%E6%92%AD%E6%94%BE 《以諾遊蹤IV之穌哥行傳》4月9日播放] 基督日報 (香港)</ref>

*第五輯節目《德國‧改革與轉化之旅》，於2011年7月11日至22日前往[[德國|德國]]拍攝，參與藝人包括[[李詩韻|李詩韻]]、[[何基佑|何基佑]]及[[苟芸慧|苟芸慧]]，於2011年12月3日起逢星期六下午6:00於高清翡翠台首播，及於myTV提供節目重溫。<ref>[http://upwill.org/channel/5-channel/7757-enochfootprint 《以諾遊蹤V：德國改革與轉化之旅》預告片(將於無綫高清翡翠台播放)] {{webarchive|url=https://archive.is/20120715131359/http://upwill.org/channel/5-channel/7757-enochfootprint |date=2012-07-15 }} upwill.org</ref>

*第六輯節目《南洋‧赤道朝陽》，林以諾牧師與藝人[[何基佑|何基佑]]、[[何傲兒|何傲兒]]及[[裕美|裕美]]到[[馬來西亞|馬來西亞]]及[[新加坡|新加坡]]拍攝，於2012年4月7日起逢星期六下午6:00於高清翡翠台首播，及於myTV提供節目重溫。<ref>[http://programme.tvb.com/foodandtravel/enochsfootprint6 《以諾遊蹤VI: 南洋‧赤道朝陽》 tvb.com] </ref>

*第七輯節目《日本‧沃土再生》，林以諾牧師與藝人[[黃智賢|黃智賢]]、[[李璧琦|李璧琦]]及[[何傲兒|何傲兒]]到[[日本|日本]]拍攝，於2012年12月16日起逢星期日下午5:30於高清翡翠台首播，及於myTV提供節目重溫。<ref>[http://www.channelj.org.hk/index.php?option=com_content&view=article&id=1287:enochfootprintjapan&catid=60:sepcial 「以諾遊蹤」之「日本 - 沃土再生」] ChannelJ</ref>

*第八輯節目《關西·歲月的洗禮》，林以諾牧師與藝人[[利嘉兒|利嘉兒]]及[[裕美|裕美]]等到[[日本|日本]][[近畿地方|關西]]拍攝，於2013年11月10日起逢星期日下午5:30於高清翡翠台首播，及於myTV提供節目重溫。
