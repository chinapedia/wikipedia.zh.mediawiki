'''以利亞'''（{{lang-he|אֵלִיָּהוּ}} ''eliyahu''）是《[[聖經|聖經]]》人物之一，是一位[[先知|先知]]。'''以利亞'''名字的意義為“[[耶和華|耶和華]]是神”。

以利亞出生於公元前9世紀，當時[[以色列王國_(前期)|以色列王國 (前期)]]進入南北分裂時期，分為[[以色列王國_(後期)|以色列王國 (後期)]]及[[南猶大國|南猶大]]。《[[聖經|聖經]]》紀載，以利亞按照神的旨意，警告[[亞哈|亞哈]]王，如果繼續崇拜偶像，神將審判以色列，讓以色列經歷旱災。以利亞寄住[[撒勒法|撒勒法]]的寡婦家時，也施行了救活寡婦兒子的神蹟及油與麵不短缺的神蹟。<ref>{{Cite book|title=聖經|last=|first=|publisher=|year=|isbn=|location=|pages=列王記上 17章}}</ref>以色列經歷三年的旱災後，以利亞再次與亞哈王見面，並約好在[[迦密山|迦密山]]，透過[[燔祭|獻燔祭]]，確認哪邊敬拜的神才是真神。結果以利亞的[[祭壇|祭壇]]興起降火的神蹟，讓所有人不得不承認"[[耶和華|耶和華]]才是真神!"並解除了三年旱災的窘境。<ref>{{Cite book|title=聖經|last=|first=|publisher=|year=|isbn=|location=|pages=列王記上 18章}}</ref>後來'''以利亞'''仍被以色列王室逼迫。最後在[[約旦河|約旦河]]乘旋風升天去了，後來由[[以利沙|以利沙]]繼承以利亞[[先知|先知]]的使命。<ref>{{Cite book|title=聖經|last=|first=|publisher=|year=|isbn=|location=|pages=列王記下 2章}}</ref>

== 背景 ==
列王時代，因為以色列民族的國家分裂，是极混乱，堕落，敗壞，黑暗的時期；因此這段時間也是出現許多[[先知|先知]]来教訓百姓的時代。然而在眾先知中，名字在[[新約聖經|新約聖經]]中經常被提及的，就是以利亞，他與神國有一定的關係，但也只是上帝所差遣的其中一個僕人而已，真正重要的乃是耶和華以色列的神。

在以利亞生平重大的事件中，經常面對一位特出，有权而兇惡的婦人[[耶洗别|耶洗别]]（Jezebel）。她是一個鐵石心肠剛愎冷酷的女人。她是[[西頓|西頓]]王的女兒，在熱心事奉[[巴力|巴力]]（Baal）的氣氛中長大，當她做了[[以色列王國|以色列]]王[[亞哈|亞哈]]的妻子以後继续信奉她先祖的宗教，並且藉着软弱的丈夫強迫以色列人随從她的宗教。亞哈為了討好她，特别在[[撒馬利亞|撒馬利亞]]為巴力築了一座壇，並開始迫害耶和華的信徒。耶洗别殺害[[耶和華|耶和華]]的先知，奉養四百五十名巴力的先知。在亚哈离道、[[叛教|叛教]]的过程中，神便差遣以利亞來到。

崇拜巴力是世界上最古老的信仰之一，巴力的信徒不但把巴力當作天上的王，又是土地之神，五穀，並認為牲畜的昌盛都在巴力的管制之下。人們在崇拜巴力时要烧香[[祈福|祈福]]，甚至把[[人牲|儿女当祭品]]献上，崇拜时常有飲宴，荒淫堕落狂歡相伴，觸怒耶和華（[[列王纪上|列王纪上]]14：23-24），以利亞適時而起。对耶和華敬畏的热诚與忠心。

== 隱藏的靈修 ==
以利亞在神的领導之下，接受各種操练造就其靈性，以后才顯出能力，成就被托付的事工。

*1. 迦密之先，必有基立 – “去藏在基立溪旁”，這種等候神的供應比事奉神更難。然而，他凭信心抓住神给他的第一個應許：“我已吩咐[[烏鴉|烏鴉]]在那里供養你”。儘管溪里的水一天天减少，也卻學會謙卑，顺服，遵神旨而行。

*2. 在撒勒法寡婦家的操練 – 以利亞在基立（Cherith Brook）所學的功課，他便繼續更深一層的造就。當他到達撒勒法(Zarephath)的时候，發現神所指定的那位寡婦正为自己和兒子預備最後餐，然后便等死。然而在基立的體驗，使他大有信心信靠神會天天施行神蹟，以供應他及那寡婦母子的需要。當他信心往前邁向一大步，神果然使藉以利亞所說“罈內的麵必不减少，瓶裡的油必不缺短，直到[[耶和華|耶和華]]使雨降在地上的日子。”（[[列王紀上|列王紀上]]17：14）的話完全應驗。

*3. 面對孩子死亡的操練 – 寡婦的兒子忽然死去，住在這家裏的以利亞突遭此不測，實在是很難學習的課程。此時以利亞應該是如經上記的是與大家一樣性情的人，所以以利亞求告耶和華，「耶和華我的神啊，我寄居在這寡婦的家裡，你就降禍於她，使她的兒子死了嗎？」 以利亞三次伏在孩子的身上，求告耶和華說：「耶和華我的神啊，求你使這孩子的靈魂仍入他的身體！」，耶和華應允以利亞的話，孩子的靈魂仍入他的身體，他就活了。

== [[迦密山|迦密山]]的火 ==
*當全國偏離正道，崇拜耶和華的情緒降至最低潮的時候，神在這樣的時候興起了以利亞。為了讓全國上下明白兩種相對的宗教不可能有和平共存的餘地，以利亞向巴力的先知挑戰，要他们試試耶和華的威力。“那降火顯應的神就是神”，這是十分乾脆而公允的挑戰。既然他們認為巴力是掌火之神，那么便讓他試試它的本行。其结果也清清楚楚說明：“若耶和華是神，就當順從耶和華；若巴力是神，就當順從巴力”（列王紀上18:21）。百姓會長久心持两意，如今应该是黑白分明，表明忠貞的時候了。

*此時，以利亞也面臨極關鍵性的挑戰，但是最後以利亞成功地求耶和華-以色列的神從天降下火来燒盡燔祭，木柴，石頭及溝裏的水，以利亞的成功除了顺服神的差遣以外，尚可由其獻祭禱告時所作的幾件事，窺知其成功的要索：1.重修已经毁壞的祭壇，2.又取十二塊石頭築壇——是應驗了經上說的那樣建立聖所。以利亞的禱告蒙雅耶和華垂聽，有火从天而降，耶和華-以色列神的至高权威得以彰顯，此時以色列眾民看見了，就俯伏在地，說：「耶和華是神！耶和華是神！」 
並且此時，以利亞趁著挑戰巴力先知的比賽獲勝，下令眾人拿住巴力的先知，眾人就拿住他們。以利亞帶他們到基順河邊，在那裡殺了他們。這是要應驗先知書中所記那樣摔碎仇敵。然後以利亞恳切地禱告，天降下大雨解除旱象。

== 升天 ==
當[[亞哈謝|亞哈謝]]為王的時候，仍效法其父母，一味行惡，輕忽至高神的存在。亞哈謝患病不求耶和華，耶和華便指示以利亞傳警告，亞哈謝不悔改，反而派兵捉拿以利亞，上帝使用這事情，使用自己的僕人以利亞而显出能力，從天降火下來燒滅那些捉拿的兵。

最後以利亞直接被接走回到令人想念的天家，一生為信仰而战。

並且被接升天時, 擲下他的外衣給耶和華所揀選的接班人以利沙 (列王記上19:18 - 並膏亞伯米何拉人沙法的兒子以利沙作先知接續你)，他升天前, 曾問以利沙要求什麼. 以利沙求：願感動你的靈雙倍感動我。
