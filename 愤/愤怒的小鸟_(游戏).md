{{vgname|憤怒鳥|en=Angry Birds}}是[[芬蘭|芬蘭]][[Rovio娱乐|Rovio娱乐]]推出的一款[[益智射擊遊戲|益智射擊遊戲]]。在遊戲中玩家控制一架彈弓發射無翅小鳥來打擊建築物和小豬，並擊殺以關中所有的小豬為最終目的。2009年12月首先發佈於[[蘋果公司|蘋果公司]]的[[iOS|iOS]]平台，自那時起，已經有超過1200萬人在[[App_Store|App Store]]付費下載，因而促使該遊戲公司開發新的遊戲版本以支持包括[[Android|Android]]、[[Symbian_OS|Symbian OS]]等操作系統在內的擁有觸控式功能的[[智慧型手機|智慧型手機]]。Rovio Mobile公司也通過免費升級的方式為憤怒鳥增加大量的遊戲內容，并推出節日的獨立版本<ref>[http://www.beautytalk.com.hk/angry-birds-valentine 情人節特別版本]等。</ref>。

支持者称赞《憤怒鳥》游戏内容让人著迷、具有滑稽趣味的風格和低廉的價格。遊戲的流行使憤怒鳥擁有了[[個人電腦|個人電腦]]和[[遊戲機|遊戲機]]版本、一個專賣遊戲角色相關商品的市場甚至是關於特色電影或是電視劇的長期計劃。包括在所有平台下的所有版本，本游戏的总下载量超过3亿<ref name="BBC-300M">{{cite web|url=http://www.bbc.co.uk/news/business-14245047|title=Angry Birds maker Rovio sued over app patents|author=BBC News|work= |date=July 22, 2011| accessdate=July 22, 2011}}</ref>，被誉为“目前最主流的游戏之一”<ref name="dt">{{cite web| author=Van Camp, Jeffrey (November 23, 2010).| title=Israeli Angry Birds satire goes viral| url=http://www.digitaltrends.com/gaming/israeli-angry-birds-satire-goes-viral/| publisher=digitaltrends.com| accessdate=2011-09-10|}}</ref>、“2010年最畅销作品之一”<ref name="shaer">{{cite web|author=Shaer, Matthew（2010年11月29日）|title="Angry Birds bound for Xbox, PlayStation"|url=http://www.csmonitor.com/Innovation/Horizons/2010/1129/Angry-Birds-bound-for-Xbox-PlayStation|accessdate=2011-09-10}}</ref>以及“迄今移动产品应用中最大的成功”<ref>{{Cite web|title=Angry Birds will be bigger than Mickey Mouse and Mario. Is there a success formula for apps?|url=http://miter.mit.edu/article/angry-birds-will-be-bigger-mickey-mouse-and-mario-there-success-formula-apps|date=February 18, 2011|accessdate=March 27, 2011|coauthors=Erik Holthe Eriksen, Azamat Abdymomunov, Dagens Næringsliv|work=MIT Entrepreneurship review|deadurl=yes|archiveurl=https://web.archive.org/web/20110430114108/http://miter.mit.edu/article/angry-birds-will-be-bigger-mickey-mouse-and-mario-there-success-formula-apps|archivedate=2011年4月30日}}</ref>。目前是[[App_Store_(iOS)|App Store]]最畅销的应用。

2012年1月13日，Rovio在其[[Youtube|Youtube]]專門頻道上載中國龍年春節主題預告片，中國大陸將此遊戲名稱定為「-{zh-cn:愤怒的小鸟; zh-hk:'''憤怒的小鳥'''; zh-tw:'''憤怒的小鳥''';}-」。其餘海外華人社區包括台港澳則翻譯為「-{zh-cn:憤怒鳥; zh-hk:'''憤怒鳥'''; zh-tw:'''憤怒鳥''';}-」<ref>http://www.youtube.com/watch?v=wUNuuneCtqU</ref>。同年，Rovio將憤怒鳥內的小豬轉變成主角，推出《[[搗蛋豬|搗蛋豬]]》。

== 遊戲內容 ==
[[File:Angry_Bird_Screenshot.jpg|400px]]

在遊戲中，玩家需要控制一群小鳥，幫助它們奪回被一群綠色的小豬偷走的蛋。在每一關中，小豬被由[[木材|木材]]、[[冰塊|冰塊]]和[[石塊|石塊]]等材料搭成的建築保護著（有時甚至會有皮球作緩衝保護小豬），遊戲的目標是消滅關中所有的小豬。玩家通過使用[[彈弓|彈弓]]發射小鳥，直接打擊小豬，或通過打擊建築使其垮塌而間接壓倒小豬。在許多關卡中，場景中會提供額外的道具，例如有[[炸藥|炸藥]]的箱子或者石塊，可以和小鳥的撞擊配合，用以擊敗那些處在難以接近位置的豬。

在遊戲中有11種不同種類的小鳥可供使用。在最早期的幾個關卡中，只有最基本的紅色小鳥可供使用。不過隨著遊戲進程的推移，會有更多種類的小鳥使用，其中的一些小鳥在空中飛行時輕敲觸摸屏便可以施放出特殊能力。例如，藍色小鳥（體型最小，擅長撞碎冰塊）將會分裂成三隻略小的鳥，黃色小鳥（擅長鑽木）可以加速衝刺，黑色小鳥（體型笨重，擅長破壞石頭）會爆炸，白色小鳥會投下可爆炸的蛋然後高速往右上飛走等，綠色小鳥會迴旋飛行。小豬也有各種不同的大小尺寸，體型小的豬相對來說也更為脆弱。脆弱的小豬通過直接打擊或者利用被摧毀的建築殘骸都能摧毀，較大的豬能夠承受更多的傷害。還有一些小豬戴著頭盔，使它們有更高的防禦力。

每一關初始的小鳥數量、類型和順序是遊戲預先指定的。若在最後一隻小鳥飛出之後或之前，所有的小豬都被消滅，那麼這關便完成了，同時下一關也將解鎖。在完成關卡後，緊接著玩家將根據遊戲中獲得分數的多少獲得1星到3星的評價，玩家可以任意多次重試，當然也可以重玩之前打過的關卡嘗試獲得更高的分數。

== 開發歷程 ==
Angry Birds的創意[[山寨文化|山寨]]自2008年的Castle Clout和2009年的{{link-en|Crush the Castle|Crush the Castle}}，後兩作均是Liam Bowmers在網頁遊戲平臺{{link-en|Armor Games|Armor Games}}上發佈的作品。在[[App_Store_(iOS)|移动平台]]上，它们也比Angry Birds上架更早。<ref>{{cite news|url=http://game.chinanews.com/feature/201207/1322956.html|title=欧美山寨游戏简史——移动平台的无主之地|publisher=中新网|date=2012-07-13|deadurl=yes|archiveurl=https://web.archive.org/web/20120910052948/http://game.chinanews.com/feature/201207/1322956.html|archivedate=2012-09-10}}</ref>

在2009年初，Rovio的職員開始考慮潛在遊戲的製作計劃。其中一個建議來自資深遊戲設計師Jaakko Iisalo：在一個模擬的截圖中，一些看不見腿和翅膀的、憤怒鳥十分有特色。儘管這副截圖並沒有指示出這將是個怎樣的遊戲，職員們依舊喜歡其中的角色，團隊成員最後選擇了這些角色，並為它們打造了一個相關的遊戲。作為一個開發遊戲的概念，職員們認識到應為小鳥們設計敵人。因為[[豬流感|豬流感]]正是那時新聞的話題，所以製作團隊讓豬成為了小鳥們的敵人。初始用於開發憤怒鳥的投資成本，估計超過10萬歐元，其中不包括後續開發的投資。

當Rovio開始為其他設備開發憤怒鳥版時，新的問題暴露出來了。當團隊開始為[[Android|Android]]版本開發時，它們發現Android軟件有大量的設備類型和版本的配置。[[CPU|CPU]]速率甚至[[用戶界面|用戶界面]]比原先開發iOS版本的遊戲來得更大。仍然還有近30種Android手機無法運行遊戲，其中不乏[[Motorola_Backflip|Motorola Backflip]]等新出品手機。在發佈Android平台遊戲的初始版本的一個月後，Rovio Mobile開始為其他設備設計一個遊戲的簡化版本。

Windows Phone 7版本由Microsoft Studio發行，2011年5月27日上架，售價2.99美元，「Poached eggs」第一節共21個關卡可供免費試玩。WP版本較其他平台版本不同之處，在於遊戲與玩家的[[Xbox_Live|Xbox Live]]戶口連結，共有20個隱藏成就可供解開。

=== 篇章 ===

; 第一章「Poached Eggs」
: 在遊戲最初的iOS版本中包含了標題為「Poached Eggs」的章節，章節中包括三個主題的關卡，每個主題都含有21關。Rovio時常發行免費升級包，升級內容包括增加新關卡、新的遊戲中物件甚至新種類的鳥。每當升級包發佈，所有平台的應用商店中，都將整合最新升級的完整版遊戲。
; 第二章「Mighty Hoax」
: 首個升級包於2010年1月10日發行，增加了新的名為「Mighty Hoax」的章節。章節中包含兩個新的主題，每個主題又擁有21關。在2010年4月發行的升級包中，增加了名為「金蛋」的新要素：「金蛋」被藏在遊戲中的各個角落，一旦金蛋被擊中，新的附加關卡將被開啟。
; 第三章「Danger Above」
: 在2010年3月4日，升級包中更有名為「Danger Above」的新章節加入，最初章節只包括了一個主題和15個關卡，不過在隨後發行的兩個升級包中，兩個新的主題被加入到「Danger Above」章節中，兩個新加入的主題都含有15個關卡。
; 第四章「The Big Setup」
: 在2010年7月1日，「The Big Setup」章節被加入，其中包含兩個15關主題和附加的「金蛋」關卡。不久後，「The Big Setup」又被增加了兩個15關主題。
; 第五章「Ham 'Em High」
: 於2010年12月23日發佈。此章是為了慶祝遊戲在Apple Store發佈的第一年而發佈。「Ham 'Em High」包含一個15關主題，一隻稱為「Mighty Eagle」的鳥，這隻鳥可以用來清除任意的關卡。「Mighty Eagle」只能在遊戲中購買且只能在未完成的關卡中使用（每小時只能使用一次）。不過只要關卡使用標準的鳥類通關，使用限制便被取消；「Mighty Eagle」也可以在曾經完成的關卡中使用，來玩一個稱為「Total Destruction」的遊戲，在小遊戲中玩家可以使用標準的小鳥以及「Mighty Eagle」來摧毀儘可能多的場景元素，達成100%摧毀的玩家將在那一關獲得一片「Mighty Eagle」的羽毛。2011年2月4日對「Ham 'Em High」進行第一次擴充，增加一個由15個關卡組成的主題。2011年3月22日，Rovio公司發行《憤怒鳥里約大冒險》（''Angry Birds Rio''），設有45關卡。同名的[[動畫電影|動畫電影]]將由[[二十世紀福斯|二十世紀福斯]]（20th Century Fox）公司發行<ref>{{cite news|url=http://tw.nextmedia.com/applenews/article/art_id/33155233/IssueID/20110201|title=憤怒鳥新版遊戲《大冒險》設45關卡|publisher=台灣蘋果日報|date=2011-02-01|accessdate=2011-02-03}}</ref>。2011年3月17日，「Ham 'Em High」進行第二次擴充，使這一章節完整。
; 第六章「Mine And Dine」
: 於2011年6月16日發佈。2011年5月，[[Google_I/O|Google I/O]]大會期間，發表了網頁瀏覽器版本，只要支援[[WebGL|WebGL]]的瀏覽器均可使用。遊戲並在[[Chrome_Web_Store|Chrome Web Store]]上架<ref>{{cite news|url=http://chinese.engadget.com/2011/05/11/angry-birds-gets-a-web-version-in-google-chrome/|title=Angry Birds多了網頁版，現身於Google Chrome|publisher=癮科技|date=2011-05-12}}</ref>。
; 第七章「Bird day Party」
: 於2011年12月11日發佈、從2012年開始每年12月11日都會追加關卡。設計公司與[[國際鳥盟|國際鳥盟]]合作，聯手拯救世界瀕危鳥類，並推出專為此開發的新版Angry birds，以瀕危鳥類為主角。使用者可以在玩遊戲的過程中進行捐款，拯救189種鳥類<ref>{{cite news|url=http://tw.news.yahoo.com/憤怒鳥真的-怒-了-出手救瀕危鳥類-121643131.html|title=憤怒鳥真的「怒」了！ 出手救瀕危鳥類|publisher=新頭殼newtalk |date=2012-01-05}}</ref>。
; 第八章「Bad Piggies」
: 於2012年10月9日發佈。与Rovio发行的另一款游戏[[捣蛋猪|捣蛋猪]]相关。
; 第九章「Red's Mighty Feathers」
: 於2013年7月2日發佈。初次開啟關卡只用 Red Mighty Feathers Bird(升级红鳥)，後於2013年9月16日新增了新關卡進入新章節，並原本只用強化紅鳥關卡放進Bouns Egg Defender關卡。
; 第十章「Short Fuse」
: 於2013年11月26日發佈。章節裏的黑鳥全化為強化黑鳥
; 第十一章「Flock Favorites」
: 于2014年7月22日发布。由粉丝投票决定的全新章节;五周年章节「BirdDay 5」
: 于2014年11月11日发布。愤怒鸟发布五周年特别作品，基于粉丝设计投稿创作的新章节
; 特別章節「Surf and Turf」
: 2012月2月13日首次於Facebook發佈，並於2012年3月20日發佈。
