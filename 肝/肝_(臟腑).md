在[[中醫|中醫]]傳統[[臟象學說|臟象學說]]中，'''[[肝臟|肝]]'''屬[[五行|五行]]中的木，其主要功能為「疏泄」與「藏血」。其中疏泄包含了精神情緒、[[消化|消化]]，以及與氣、血、[[水|水]]的流通；而藏血則包括了[[血液|血液]]的儲藏及調節<ref>陳健仲。《[http://www.ctmd.org.tw/dissertator.aspx?dir_tree=WDD0000304&data_id=WDD0000415 中醫和西醫不同觀點下肝的生理功能]》。中國醫藥研究叢刊，1999年，第二十期，31-33頁。</ref>。除此之外，肝與[[膽_(臟腑)|膽]]互為表裡，影響[[膽汁|膽汁]]的分泌。

==生理==
《[[內經|內經]]》之後的醫學家在描述肝時，常會將其與[[情緒|情緒]]，尤其是其中的[[怒|怒]]氣連結。舉例而言，《[[素問|素問]]‧臟氣法時》寫道：「肝病者，令人善怒。」《素問‧陰陽應象大論》則說：「在志為怒，怒傷肝。」<ref name = "A">田安然。《[http://www.ctmd.org.tw/dissertator.aspx?dir_tree=WDD0000304&data_id=WDD0000422 論中醫的肝臟與西醫肝臟的不同]》。中國醫藥研究叢刊，2002年，第廿三期，94-97頁。</ref>

肝主「藏血」及「疏泄」，是對肝生理作用的常見敘述，而此時所說的血也與[[精神|精神]]活動有關，例如《[[靈樞|靈樞]]‧平人絕穀》所說的「血脈和利，精神乃居。」<ref name = "A"/>肝主疏泄是[[朱丹溪|朱丹溪]]所提出的想法，他在《[[格致餘論|格致餘論]]‧陽有餘陰不足論》中寫道：「主閉藏者腎也，司疏泄者肝也，二藏皆有相火，而其系上屬於心。<ref name = "A"/>」

==病理==
與肝相關的症狀包括了肝鬱、肝火與肝風等。肝鬱是指因為情緒的變化而產生的身體不適。例如[[胸悶|胸悶]]或是[[月經|月經]]不順。肝火則是指人在生氣的時候，所產生出來的臉紅、口亁或是[[頭痛|頭痛]]等症狀，又稱為「肝氣盛」。

肝風有兩種意義，一種是指使身體抖動的病症，也就是「肝風內動」，造成這種抖動的原因並不一定，有些可能源自肝火<ref name = "A"/>。而另一個意思是指肝受[[风邪|风邪]]影響而產生的疾病<ref>[http://www.39kf.com/cooperate/cmdic/g/2006-01-11-144487.shtml  医源资料库 > 中医词典]</ref>。
