'''宗庆后'''（{{bd|1945年|10月12日|catIdx=Zong宗}}）<ref>[http://www.dfzjsh.com/xy/ShowArticle.asp?ArticleID=536 宗庆后：解剖“娃哈哈”20年] ，新浙商网，2008年6月7日</ref>，[[中国|中国]][[浙江省|浙江省]][[杭州市|杭州市]]人。他是[[娃哈哈集团|娃哈哈集团]]的创始人，1991年至今担任杭州娃哈哈集团有限公司[[董事长|董事长]]兼[[总经理|总经理]]。宗庆后曾经是第十一届全国人大代表。在2010年的[[胡润百富榜|胡润百富榜]]中成为中国首富<ref>[http://www.apdnews.com/news/22658.html 中国“首富”宗庆后] {{webarchive|url=https://web.archive.org/web/20131029185721/http://www.apdnews.com/news/22658.html |date=2013-10-29 }}，亚太日报，2013年9月23日</ref><ref>
[http://money.msn.com.cn/rich/20100929/17331141300.shtml 2010胡润百富榜：宗庆后800亿元成中国首富]</ref>。及後於2016年胡润百富榜中，以1120億人民幣財富排名第5位。<ref>
[hthttp://hurun.net/CN/HuList/Index?num=612C66A2F245 2016年胡润百富榜]</ref>

== 生平 ==
1945年10月12日出生于江苏省[[宿迁县_(唐朝)|宿迁县]]（今[[宿迁市|宿迁市]]），祖父曾是[[张作霖|张作霖]]手下的财政部长，父亲宗启騄在国民政府当过职员。宗庆后父亲宗启騄夫妇曾在[[南京|南京]]和宿迁居住过，他的曾祖父辈均为[[杭州府|杭州府]][[钱塘县|钱塘县]]籍，所以1949年以后迁归浙江杭州祖籍工作。 

中學畢業後曾在[[舟山|舟山]][[鹽場|鹽場]]工作，1964年，在浙江绍兴茶叶厂任调度。1979年任[[小學|小學]]教師的母親退休，於是他返回家鄉，但因文化水平低，只可在一所小學裡當[[校工|校工]]，做推销员。1987年，杭州市[[上城區|上城區]]校辦企業經銷部成立，這即成為後來的娃哈哈。當時42歲的宗慶后與兩名退休教師，以14萬元借貨承包了连年亏损的杭州上海校办企业经销部，代銷人家的[[汽水|汽水]]、棒冰及[[文具|文具]]賺錢。<ref>[http://unn.people.com.cn/BIG5/41495/41500/3037676.html 人民網:“娃哈哈”的新童話] {{webarchive|url=https://web.archive.org/web/20080425095207/http://unn.people.com.cn/BIG5/41495/41500/3037676.html |date=2008-04-25 }}(轉載自証券日報)</ref>。1989年，创建杭州娃哈哈营养食品厂，任厂长。1991年，拿出8000万兼并杭州罐头食品厂成立了娃哈哈食品集团公司。

2007年, 公司遭到达能强行并购, 宗庆后辞职引发员工抗议，商标终审判归娃哈哈。

他指控[[法国|法国]][[达能|达能]]公司强行以40亿元人民币的低价并购杭州娃哈哈集团有限公司总资产达56亿元、2006年利润达10.4亿元的其他非合资公司51%的股权，并最后辞职。此事在中国大陆引起轩然大波。目前事态仍在发展之中。<ref>{{cite web|url=http://finance.sina.com.cn/focus/2007wahaha/index.shtm|title=聚焦娃哈哈遭遇达能强行并购|work=新浪财经|publisher=新浪网}}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref>{{cite web |url=http://www.caijing.com.cn/newcn/home/todayspec/2007-06-10/21753.shtml|title=宗庆后辞任 达能“释兵权”|work= |publisher=財經網|date=2007-06-10}}</ref>

在2011年"两会"上，宗庆后有一项颇为引人瞩目的提案，即建议国企员工都可持股，并按贡献大小定期调整持股额。从1993年开始，娃哈哈集团就已采用全员持股。

2012年10月12日，宗庆后以100亿美元身价登上《[[福布斯|福布斯]]》中国首富。

在2013年"两会"期间的新闻见面会上，宗庆后建议国家建立阶梯式的供房制度，给低薪阶层提供一室一厅的廉租房，月租金控制在其月收入的10%以下；给中产阶层提供一套面积90平米的经济适用房，这个价格约为其月收入的20%，并且在15年内能还清本息；对高收入群体，购买商品房可以不限面积、不限价格，但是必须使用自有资金，不得贷款，而且必须退还原有经适房。同时为了避免房地产市场投资过热还应该向持有多套商品房的炒房者加大征税力度和额度。

== 美国绿卡==
美国一位不愿透露姓名的移民律师说:宗庆后本人持有美国绿卡已长达9年。对此，作为第十二届全国人大代表的宗庆后在2013年回应说: 我们全家都没有入外国籍, 也没有外国绿卡，目前也没有移民计划；而原来的美国绿卡已取消。

== 遇襲事件 ==
2013年9月13日，宗慶後在其[[杭州|杭州]]住宅附近遭襲擊，左手被砍傷，四個手指的[[肌腱|肌腱]]被砍斷<ref>[http://www.bbc.co.uk/zhongwen/trad/china/2013/09/130918_china_tycoon_attack.shtml 港媒：娃哈哈老總宗慶后遭砍傷手部 - BBC中文網 - 兩岸]</ref>。當時宗慶後在家門口突遇一位持刀的陌生中年男子。在受到威脅後，宗慶後被對方的刀割傷了左手兩根手指的肌腱，該男子隨後被刑拘。據審查，該男子今年上半年借了3萬元來到杭州找工作，因年紀較大，一直沒有找到。在[[電視|電視]]節目上看到過宗慶後幫助[[農民工|農民工]]的訪談，也想得到宗慶後的援助，在其公司安排一個工作，便到宗慶後的住處附近找到宗慶後，因未如願，于是實施了違法行為。遇襲事件後各種猜測的聲音不斷涌現，坊間還出現了「尋仇說」、「報復說」等，還有傳聞稱，娃哈哈公司的內部高管發生重大調整，娃哈哈的經營出現嚴重問題等。對于一係列傳言，宗慶後在12月接受媒體訪問中做了澄清，他表示，遇襲事件應該是個偶發事件，該事件後，自己的生活也沒什麼太大變化<ref>[http://news.xinhuanet.com/fortune/2013-12/17/c_125869273.htm 宗慶後首談遇襲事件:偶發事件 生活沒太大變化-新華網]</ref>。

== 评价 ==
因为对民族品牌以及民族产业的巨大贡献，2013年10月，全国工商联60华诞之际，《中国工商》杂志、华商韬略编辑委员会、中华工商联合出版社联合发起《民营力量璀璨中国梦想——100位对民族产业贡献卓著的民营功勋企业家》荣誉报道活动，彰显民营经济及民营企业家的民族成就与国家贡献，宗庆后获对“民族产业贡献卓著的民营功勋企业家”荣誉。
