'''宗室'''（{{lang-mnc|ᡠᡴᠰᡠᠨ|z=uksun}}），音譯'''烏克孫'''，俗名'''黄带子'''，是出身清朝皇族[[愛新覺羅氏|愛新覺羅氏]]嫡系的皇族成員所享有的稱號。沒有爵位的皇族宗室，可以在名字（gebu）之前冠上「宗室」，以表示其身份。

==概論==
烏克孫（uksun），在明朝[[女真|女真]]時，就在女真部落中使用，意指源出同支的近親兄弟姐妹們，其親緣關係較[[穆昆|穆昆]]更為親近。在清朝官修「滿蒙漢三體」的《[[:s:滿洲實錄|滿洲實錄]]》中，對譯為漢語的「族」。

1635年（天聰9年）正月二十六日，皇太極發布上諭，凡太祖後代，稱[[阿哥_(清朝)|阿哥]]，其他六祖子弟，稱覺羅，皆為[[汗|汗]]的宗室。皆用紅帶子，一般人不可直呼其名，應用阿哥，或覺羅來稱呼。

在在清朝入關之後，1636年（[[清太宗|清太宗]][[崇德|崇德]]元年），[[皇太极|皇太极]]下诏，规定
#其祖父[[清显祖|清显祖]]塔克世的直系子孙为“'''宗室'''”
#清興祖[[福滿|福滿]]與清景祖[[覺昌安|覺昌安]]等其余伯叔兄弟旁支子孙称“[[觉罗|觉罗]]”。
#烏克孫（uksun），被譯為宗室，成為清朝皇室嫡系子孫專有用的名稱。

此外，另外創了一個單字，烏克蘇拉（uksura），用來取代之前烏克孫的意義，用以指近親家庭中的成員<ref>[[劉小萌|劉小萌]]〈明代女真的血缘家族"乌克孙"〉，收入[[劉小萌|劉小萌]]著《滿族的社會與生活》</ref>。

金朝舊例，金朝的皇族直系稱為宗室，不直呼其姓；皇族旁系則稱[[完顏|完顏]]，此外，還有非皇族的完顏氏<ref>《金史》卷59〈表〉：「金人初起，完顏十二部，其後皆以部為氏，史臣記錄有稱「宗室」者，有稱完顏者。稱完顏者亦有二焉，有同姓完顏，蓋疏族，若石土門、迪古乃是也；有異姓完顏，蓋部人，若歡都是也。大定以前稱「宗室」，明昌以後避睿宗諱稱「內族」，其實一而已，書名不書氏，其制如此。宣宗詔宗室皆稱完顏，不復識別焉。」</ref>。[[鄭天挺|鄭天挺]]認為將皇族分為宗室與覺羅，是仿效金制度。非皇室的同姓覺羅，則稱民覺羅。

宗室系「黄带子」，觉罗系「红带子」，由此以示区别。宗室被革退者系「红带子」，位列皇族家谱《[[玉牒|玉牒]]》之末。例如，签署《[[中英南京条约|中英南京条约]]》的中方代表之一[[伊里布|伊里布]]就是原为宗室，后被革退为红带子。

身為宗室，依其親疏，可受封爵位，最高為親王、郡王、貝勒、貝子以下，共分十四等<ref>《欽定大清會典則例》卷1：「宗室封爵列十有四等，一和碩親王，二世子，三多羅郡王，四長子，五多羅貝勒，六固山貝子，七鎮國公，八輔國公，九不入八分鎮國公，十不入八分輔國公，十一鎮國將軍，十二輔國將軍，十三奉國將軍，十四奉恩將軍。皇子生十五歲，由府請封其爵，級出自欽定。」</ref>，隨著與皇室的親疏，逐代遞降。

在十四等之下，稱閒散宗室，只享有宗室頭銜，可用四品頂戴，但沒有爵位<ref>吳振域《養吉齋叢錄》卷1：「宗室封爵，自親王、郡王、貝勒、貝子以下，凡十四等，以世遞降。以下則為閒散宗室，用四品頂戴。惟禮親王、睿親王、肅親王、鄭親王，世襲不降封，或獲罪革爵，仍以旁支襲封，京師俗諺謂之鐵帽子王。」</ref>。

== 世襲 ==
根據[[光緒|光緒]]《[[大清會典|大清會典]]》卷一《[[宗人府|宗人府]]》記載，清王朝分封皇室爵位共有'''功封'''、'''恩封'''、'''襲封'''和'''考封'''四種形式和十二等爵（參見男子宗室的爵位有12等）。

清代为了防止如明代一样宗室爵位泛滥的情况，制定降襲制度，这在[[中国|中国]]歷朝歷代中是最為特殊的。

其中輔國將軍以上還分'''世襲罔替'''和'''世襲遞降'''兩類。一般情況下，因為功封王爵者多屬世襲罔替，也就是俗話說的“[[铁帽子王|鐵帽子王]]”。因為恩封爵位者則多屬世襲遞降，但在遞降到輔國將軍這一爵位時便不再遞降。獲得親王、郡王、貝勒、貝子等爵位的宗室並不能世襲，每世遞降一等。惟有12大“鐵帽子王”因為其祖先功勳卓著，被賜世襲不降封典。如果某“鐵帽子王”獲罪奪爵，以其旁支襲爵，但其後世也不乏被奪爵的歷史記載。

===世襲罔替===
{{Main|铁帽子王}}
[[世袭罔替|世襲罔替]]，即以原爵位承襲。例如大清十二家“ 鐵帽子王 ”。

[[清兵入關|清兵入關]]時，戰功昭著的幾个著名[[親王|親王]]與[[郡王|郡王]]，拥有[[世袭罔替|世袭罔替]]的权利。他们的主爵位（[[入八分|入八分]]的[[爵位|爵位]]）不必世袭递降，分别为[[礼亲王|礼亲王]]、[[睿亲王|睿亲王]]、[[豫亲王|豫亲王]]、[[肃亲王|肃亲王]]、[[郑亲王|郑亲王]]、[[庄亲王|庄亲王]]、[[克勤郡王|克勤郡王]]、[[顺承郡王|顺承郡王]]。这八个清初王公，嫡系後裔享有世袭罔替的[[王爵|王爵]]（俗稱鐵帽子）。但是除了[[嫡子|嫡子]]以外，其餘子孫还跟其他王公一样，世袭递降，并且进行考封。

后来也有四位[[親王|親王]]因為輔佐[[皇帝|皇帝]]、[[太后|太后]]政治鬥爭有功，而得到此殊榮，即[[怡亲王|怡亲王]]、[[恭亲王|恭亲王]]、[[醇亲王|醇亲王]]、[[庆亲王|庆亲王]]。

除此之外，还有几家清初立过军功的[[入八分公|入八分公]]，得到世袭罔替，如褚英后人的奉恩镇国公，就是世袭罔替的。

===世襲遞降===
降等世襲，即每承襲一次要降一級，但降級若干次（親王降至鎮國公，郡王降至輔國公，貝勒降至不入八分鎮國公，貝子降至不入八分輔國公，鎮國公降至鎮國將軍，輔國公降至輔國將軍）後便不再遞降，以此爵傳世。

== 閒散宗室 ==
根据世降一等的袭爵方式和嫡庶有别的原则，必然会导致有些宗室无法取得爵位，無爵位的宗室叫“閒散宗室”，用四品頂戴、武职补服，但无四品实职。

闲散宗室本无品级顶戴。至[[乾隆|乾隆]]四十七年（1782年），[[乾隆帝|乾隆帝]]认为闲散宗室“谱列银潢，名登《玉牒》”，却“身无职级，竟至与齐民无别”，有辱皇室风采，于是钦赐闲散宗室着四品顶戴、武职补服。

== 现状 ==

1912年2月，[[清政府|清政府]]颁布[[宣統帝退位詔書|宣統帝退位詔書]]，正式结束清朝296年统治。此后，满族人通遍使用[[汉姓|汉姓]]，放弃传统[[滿族姓氏|滿族姓氏]]，宗室亦如此，[[爱新觉罗氏|爱新觉罗氏]]改为[[金姓|金姓]]、[[趙姓|趙姓]]，亦有[[随名姓|随名姓]]。[[中华人民共和国|中华人民共和国]]在成立1949年后，在相当一段时间内强调[[阶级成分|阶级成分]]，因此个人对为清朝宗室后裔身份较为忌讳。1970年代末，社会政治风气改变。此后，宗室后裔不再成为已忌讳。当代学者[[阎崇年|阎崇年]]，将清朝皇族后裔归为三类：近亲宗室、宗室嫡系子孙、闲散宗室。包括[[溥仪|溥仪]]兄弟在内的近亲宗室的正支较为低调，旁支更喜欢使用爱新觉罗氏这个姓氏。有认为这是“时代功利性”。更有皇族后裔将自称爱新觉罗者归为“骗子”<ref>{{Cite web |url=http://news.ifeng.com/a/20140911/41937752_2.shtml |title=学者：正支后裔很低调 旁支就喜欢管自己叫爱新觉罗 |accessdate=2017-08-11 |author= |date=2014-09-11 |work= |publisher=凤凰网，来源：中国网|language=Zh-hans }}</ref>。
