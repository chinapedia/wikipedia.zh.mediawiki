'''宗教法人法'''（しゅうきょうほうじんほう、昭和26年4月3日法律第126號）是[[日本|日本]]關於[[宗教團體|宗教團體]]舉行[[宗教|宗教]]活動等、尊重[[信仰自由|信仰自由]]的目的，賦予宗教團體[[法人格|法人格]]（4條）的[[法律|法律]]。最終改正是2006年（[[平成|平成]]18年）6月2日法律第50號。

== 概要 ==
; 目的
: 「<cite>賦予宗教團體法律上的能力</cite>」（第一條）
; 定義
: 「<cite>推廣宗教的教義、舉行儀式行事、及教化育成信者</cite>」（第二條）
: 「<cite>備有禮拜的設施的團體</cite>」
: 或這些團體的「<cite>包括團體</cite>」

== 經過 ==
1939年，制定「宗教團體法」，1940年4月1日開始施行。1945年12月28日，因[[駐日盟軍總司令部|駐日盟軍總司令部]]（GHQ）指示，[[天皇|天皇]]以[[勅令|勅令]]廢止「宗教團體法」，宗教團體由從來的認可制（許可制）改為屆出制（報備制），宗教法人可自由設立、規則變更、解散等的「宗教法人令」在即日施行<ref>[http://www.mext.go.jp/b_menu/hakusho/html/hpbz198101/hpbz198101_2_260.html 第二編　戰後の教育改革と新教育制度の発展 第三章　学術・文化 第五節　宗教] {{webarchive|url=https://web.archive.org/web/20120421161654/http://www.mext.go.jp/b_menu/hakusho/html/hpbz198101/hpbz198101_2_260.html |date=2012-04-21 }}（「文部科学省」公式ウェブサイト）</ref><ref>ウィリアム・ウッダード、「天皇と神道 GHQの宗教政策」サイマル出版会、1988年</ref>。

1951年4月3日，取代「宗教法人令」的「宗教法人法」在同日公布，即日施行。
