'''收发器'''（'''transceiver'''）是由一个[[发送器|发送器]]一个和[[无线电接收机|接收器]]构成的设备，它们组合并共享共用电路或单个外壳。当发送与接收功能之间没有共用电路时，该设备为发送机-接收机。这个词起源于[[20世纪|20世纪]]20年代初。类似的设备包括{{tsl|en|Transponder||应答器}}、{{tsl|en|Transverter|转换器}}和[[中继器|中继器]]。

== 无线电技术 ==
[[File:Icom.png|缩略图]]（HF）收发器，具有[[频谱分析仪|频谱分析仪]]和[[数字信号处理|数字信号处理]]（DSP）功能]]
在[[无线电|无线电]]技术中，收发器意味着单元包含接收机和发射机。在无线电最开始，接收机与发送机是独立的设备，直至1920年左右。[[业余无线电|业余无线电]]的无线电操作员（也称“火腿”）可以自己制作设备，并且现在更容易制作一个包含两个功能（发送与接收）的简易单元。几乎所有的现代业余无线电设备都使用收发器，但还有一个活跃的市场为纯无线电接收器，主要是为短波收听（SWL）操作员。收发器的例子有[[无线对讲机|无线对讲机]]、{{tsl|en|CB radio|民用频带无线电}}（CB radio）。

== 电话 ==
在有线[[电话|电话]]上，听筒包含[[錄音|音频]]的发射机和接收机，在20世纪通常用{{tsl|en|Tinsel wire|金属线缆}}有线连接到基本单元。整个单元通俗的被称为“接收器”。在[[移动电话|移动电话]]或其他{{tsl|en|Radiotelephone|无线电手机|无线电话}}上，整个单元是用于音频和无线电的收发器。

[[无绳电话|无绳电话]]在听筒上使用一个音频和无线电收发器，并且[[基站|基站]]上也有一个无线电收发器。如果有线电话基站或无声基站包含{{tsl|en|speakerphone|送受话器}}，该基站还可成为一个手机之外的音频收发器。

[[调制解调器|调制解调器]]类似收发器，但它调制和解调信号而非直接发送和接收。发送的信号被调制，接收的信号被解调。

== 以太网 ==
[[File:Ethernet-media-converter-transceiver-0a.jpg|缩略图]]
收发器在[[IEEE_802.3|IEEE 802.3]]文档中被称为[[介质|介质]]附件单元（MAU），并被广泛应用于{{tsl|en|10BASE2}}和{{tsl|en|10BASE5}}[[以太网|以太网]]网络。[[光導纖維|光纤]][[吉比特以太网|吉比特以太网]]、[[10吉比特乙太網路|10吉比特乙太網路]]、[[100吉比特以太网|40吉比特乙太網路]]和[[100吉比特以太网|100吉比特以太网]]使用的收发器包括：[[千兆乙太網路介面轉換器|千兆乙太網路介面轉換器]]（GBIC）、[[小封装可插拔收发器|小封装可插拔收发器]]（SFP）、[[SFP+|SFP+]]、{{tsl|en|QSFP}}、[[XFP|XFP]]、[[XAUI|XAUI]]、{{tsl|en|CXP (connector)||CXP}}和{{tsl|en|C Form-factor Pluggable||CFP}}。
