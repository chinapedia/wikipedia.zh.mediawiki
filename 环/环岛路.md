'''环岛路'''（Huandao Rd.）是环绕[[福建省|福建省]][[厦门市|厦门市]][[厦门岛|厦门岛]]的城市干道，跨越厦门市的[[思明区|思明区]]和[[湖里区|湖里区]]两个区级行政区，全长43公里，地方道路编号为'''县道401'''。<ref>{{cite news |url=http://www.xmhouse.com/information/News/xwzx/bd/200901/t20090120_142696.htm |title=厦门公路局局长：将投资30亿加快路网建设 |publisher=海峡导报 |date=2009-01-20 |accessdate=2009-05-04}}</ref>

== 道路概况 ==
[[File:厦门环岛路沙滩.jpg|thumb]]
环岛路全长约43公里，自1991年开始建设起总投资近38亿元，分三期建设，是厦门市编号为县道401的一级次干道地方道路，全程为双向四车道或双向六车道。

环岛路以厦门岛北部的[[石鼓山立交桥|石鼓山立交桥]]为起点，途经[[疏港路|疏港路]]、[[湖滨西路|湖滨西路]]连通与[[鼓浪屿|鼓浪屿]]隔海相望的[[鹭江道|鹭江道]]，并从[[和平码头|和平码头]]起接[[演武大桥|演武大桥]]直达[[厦门大学|厦门大学]]、[[胡里山炮台|胡里山炮台]]。自胡里山炮台起的路段称为[[环岛南路|环岛南路]]，经过[[曾厝垵|曾厝垵]]、黄厝通往位于[[前埔|前埔]]的[[厦门国际会议展览中心|厦门国际会议展览中心]]，接[[环岛东路|环岛东路]]路段向北到达位于[[五通|五通]]的[[五缘大桥|五缘大桥]]，沿[[厦门高崎国际机场|厦门高崎国际机场]]南侧的[[枋钟路|枋钟路]]经过[[高崎互通立交|高崎互通立交]]和[[嘉禾路|嘉禾路]]回到石鼓山立交桥终止。<ref>{{cite web|url=http://www.amoy-roadbridge.com/project/project-hdl.htm|title=精品工程 - 环岛路|publisher=厦门市路桥建设投资总公司|accessdate=2009-05-04|deadurl=yes|archiveurl=https://web.archive.org/web/20090421165157/http://www.amoy-roadbridge.com/project/project-hdl.htm|archivedate=2009-04-21}}</ref>

环岛路位于厦门岛东南，自[[厦门大学|厦大]]白城起至何厝[[香山_(厦门)|香山]]的一段约13公里的路段为厦门市著名的旅游景点，在1997年厦门市评定的[[厦门名景|新20景]]中占有一席之地，称为'''东环望海'''。这一路段依海而筑，路面平曲线随海岸线延展，与沿线的景点相结合，正对[[小金门|小金门]]、大担、二担等岛屿，竖立有“一国两制统一中国”大标语牌。<ref>{{cite web|url=http://www.xmnn.cn/hxwlypd/xmjdlb/200608/t20060828_1823.htm|title=东环望海|publisher=厦门网|accessdate=2009-05-04|deadurl=yes|archiveurl=https://web.archive.org/web/20071016221756/http://xmnn.cn/hxwlypd/xmjdlb/200608/t20060828_1823.htm|archivedate=2007-10-16}}</ref>

== 主要桥梁 ==
=== 演武大桥 ===

{{main|演武大桥}}

'''演武大桥'''位于环岛路西南角海军码头至[[厦门大学|厦大]]白城段，全长2.2公里，由主桥体、[[成功大道|成功大道]]互通立交、演武互通立交和白城互通立交组成。桥梁建设总投资5.6亿元，于2003年9月6日正式通车。在结构上采取了鱼腹式梁，椭圆型墩，桥身通体漆为白色。<ref>{{cite web|url=http://www.fjqhdmw.com/Html/20081124/2008112414483.shtml|title=阅读厦门穿桥过|publisher=福建区划地名网|accessdate=2009-05-04|deadurl=yes|archiveurl=https://web.archive.org/web/20081212093338/http://www.fjqhdmw.com/Html/20081124/2008112414483.shtml|archivedate=2008-12-12}}</ref>为了避免影响沿海景观，桥梁照明采取附设在桥梁护栏上的照明灯而不设灯杆。由于演武大桥低桥位段的桥面标高只有5.5米，被认为是目前世界上离海平面最近的桥梁。<ref>{{cite web |url=http://www.cbi360.com/news/Article.aspx?articleId=8486 |title=厦门演武大桥投用五年首次全面体检 |publisher=中国建筑网 |accessdate=2009-05-04}}</ref>

=== 五缘大桥 ===

{{main|五缘大桥}}

'''五缘大桥'''原名'''钟宅湾大桥'''，2006年2月更为现名<ref>{{cite news|url=http://www.soufun.com/news/2006-02-21/642262.htm|title=钟宅湾大桥更名为五缘大桥|publisher=东南早报|date=2006-02-21|accessdate=2009-05-04}}{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。桥梁位于位于围海造地形成的内湖[[五缘湾|五缘湾]]（原名钟宅湾）的出海口，连接环岛路的[[五通|五通]]和[[墩上|墩上]]段。桥梁为钢结构中承式提篮拱桥，总长810米，主桥宽34.9米，引桥宽32米。<ref>{{cite web|url=http://www.amoy-roadbridge.com/project/project-zzwbridge.htm|title=精品工程 - 钟宅湾大桥|publisher=厦门路桥建设投资总公司|accessdate=2009-05-04|deadurl=yes|archiveurl=https://web.archive.org/web/20081010215104/http://www.amoy-roadbridge.com/project/project-zzwbridge.htm|archivedate=2008-10-10}}</ref>桥梁于2004年7月2日建成通车，是中国建造的第三座这种类型的桥梁，建成时按跨度大小排名第二。<ref>{{cite web |url=http://www.sasac.gov.cn/n1180/n1226/n2410/n314274/1216445.html |title=中铁大桥局厦门钟宅湾大桥通车 |publisher=国有资产监督管理委员会 |accessdate=2009-05-04}}</ref>
