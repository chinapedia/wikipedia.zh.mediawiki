'''环氧丙烷'''，分子式C<sub>3</sub>H<sub>6</sub>O。

== 性质 ==
无色[[醚|醚]]味液体，低沸点、易燃。有[[手性|手性]]，工业品一般为两种[[对映体|对映体]]的[[外消旋|外消旋]]混合物。与水部分混溶，与[[乙醇|乙醇]]、[[乙醚|乙醚]]混溶。与[[戊烷|戊烷]]、[[戊烯|戊烯]]、[[环戊烷|环戊烷]]、[[环戊烯|环戊烯]]、[[二氯甲烷|二氯甲烷]]形成二元共沸混合物。有毒，对粘膜和皮肤有刺激性，可损伤眼角膜和结膜，引起呼吸系统疼痛，皮肤灼伤和肿胀，甚至组织坏死。<ref name="zghg" /><ref>[http://www.guidechem.com/dictionary/75-56-9.html 环氧丙烷字典-Guidechem.com{{en}}]</ref>

== 制备 ==
1、[[丙烯|丙烯]]、[[氯气|氯气]]与水於常压、60 °C加成产生[[氯丙醇|氯丙醇]]，后者经[[氢氧化钙|氢氧化钙]]处理、凝缩、蒸馏，得到环氧丙烷。

<br />
:[[File:Propylenoxid_Darstellung_1.svg|579px]]

:<math>\mathrm{2 \ C_3H_6 \ + \ Cl_2 \ + \ H_2O \longrightarrow \ 2 \ C_3H_7ClO}</math>


:[[File:Propylenoxid_Darstellung_2.svg|442px]]

:<math>\mathrm{C_3H_7ClO \ + \ OH^- \longrightarrow \ C_3H_6O \ + \ Cl^- \ + \ H_2O}</math>
<br />

2、[[乙苯|乙苯]]、[[异丁烷|异丁烷]]或[[异丙苯|异丙苯]]氧化产生[[有机过氧化物|有机过氧化物]]如[[氢过氧化乙苯|氢过氧化乙苯]]、[[叔丁基氢过氧化物|叔丁基氢过氧化物]]或[[氢过氧化异丙苯|氢过氧化异丙苯]]等，再在[[环烷酸钼|环烷酸钼]]催化下与[[丙烯|丙烯]]进行[[环氧化|环氧化]]反应生成环氧丙烷。

<br />
:[[File:Propylenoxid_Darstellung_3.svg|566px]]

:<math>\mathrm{C_3H_6 \ + \ C_8H_{10}O_2  \xrightarrow{Kat.}  \ C_3H_6O \ + \ C_8H_{10}O}</math>

== 用途 ==
环氧丙烷为有机合成中的重要原料，用于合成[[丙醛|丙醛]]、[[丙烯醇|丙烯醇]]、[[丙二醇|丙二醇]]、[[甘油|甘油]]、合成树脂、有机酸、表面活性剂、乳化剂、增塑剂、泡沫塑料、杀菌剂、熏蒸剂、洗涤剂等。
