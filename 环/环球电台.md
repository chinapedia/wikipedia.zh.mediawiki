'''环球电台'''（英语：Trans World Radio，缩写为：'''TWR'''）是一个跨国[[基督教|基督教]]福音[[广播电台|广播电台]]。环球电台从14个国家，使用[[中波|中波]]或大功率调频发射器、[[短波|短波]]以及通过当地[[电台|电台]]、[[电缆|电缆]]、[[人造卫星|人造卫星]]和[[互联网|互联网]]，达到160个国家、230种[[语言|语言]]和方言区的数百万人。

环球电台拥有许多节目，其中一些由电台职员制作，另一些来自其他机构，例如“Thru the Bible”和“Insight for Living”。

环球电台的国际总部位于美国[[北卡罗来纳州|北卡罗来纳州]][[卡瑞_(北卡罗莱纳州)|卡瑞]]。总部和世界各地的职员来自美国、加拿大和各个广播目标国。

2006年，其英国总部从[[巴斯|巴斯]]迁往[[曼彻斯特|曼彻斯特]]。

== 历史 ==
环球电台开始于1952年，当时保罗·弗里德（Paul Freed）从[[摩洛哥|摩洛哥]]向[[西班牙|西班牙]]发射广播。后来，环球电台被驱逐出[[摩洛哥|摩洛哥]]，转移至[[摩纳哥|摩纳哥]]。在[[摩洛哥|摩洛哥]]曾经多年使用的大功率发射器是[[二战|二战]]后[[纳粹|纳粹]]所抛弃的。其他主要的发射地点还有[[关岛|关岛]]、[[博奈爾島|博奈爾島]]、[[斯里兰卡|斯里兰卡]]、[[塞浦路斯|塞浦路斯]]和[[斯威士兰|斯威士兰]]。

== 语言 ==
环球电台使用230种语言向全球广播。

[[汉语普通话|汉语普通话]]、[[广州话|广州话]]、[[蒙古语|蒙古语]]、[[维吾尔语|维吾尔语]]、[[藏语|藏语]]、[[彝语|彝语]]、[[英语|英语]]、[[日语|日语]]、[[韩语|韩语]]、[[越南语|越南语]]、[[高棉语|高棉语]]、[[乌兹别克语|乌兹别克语]]、[[阿拉伯语|阿拉伯语]]、[[土耳其语|土耳其语]]、[[法语|法语]]、[[意大利语|意大利语]]、[[葡萄牙语|葡萄牙语]]、[[马拉加斯语|马拉加斯语]]等等。

== 中文广播 ==
环球电台的中文广播开播于1954年。平均每日播出6小时，由[[关岛|关岛]]发射，呼号为'''KTWR-SW'''。其中文广播总部设在[[香港|香港]]。
