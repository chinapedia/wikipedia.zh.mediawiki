'''菲利浦·道格拉斯·杰克逊'''（{{lang-en|'''Philip Douglas Jackson'''}}，{{bd|1945年|9月17日|catIdx=Jackson}}），暱稱'''菲尔·杰克逊'''（{{lang|en|'''Phil Jackson'''}}），[[美国|美国]]前[[NBA|NBA]]籃球運動員、教练，球员时代曾效力于[[纽约尼克斯队|纽约尼克斯队]]和[[新泽西网队|新泽西网队]]。退役后成为教練，1989-1998执教於[[芝加哥公牛队|芝加哥公牛队]]，助公牛队拿下6个总冠军。2000年-2011年执教[[洛杉矶湖人队|洛杉矶湖人队]]，助湖人队拿下5个总冠军。这11个总冠军也让他超过了前辈红衣主教[[阿诺德·奥尔巴赫|阿诺德·奥尔巴赫]]的9个总冠军的记录。再加上他作为球员在纽约尼克斯队拿到的两个总冠军，一共13个总冠军让他成为NBA历史上球员和教练中拿冠军最多的人。他仍保持NBA教练最高执教胜率记录，也曾带领两支不同球队获得总冠军（[[公牛隊|公牛隊]]及[[湖人隊|湖人隊]]），NBA历史上仅有三人获此殊荣，另两人是[[帕特·莱利|帕特·莱利]]和[[亚历克斯·汉纳姆|亚历克斯·汉纳姆]]。他执教公牛队和湖人队时都成功地使用了[[三角戰術|三角进攻]]战术。由于他对东方哲学特别是[[禅宗|禅宗]]的深厚兴趣，他也常被媒体尊称为“禅师”。

== 球员时代 ==
杰克逊於球員時代效力[[紐約尼克斯隊|紐約尼克隊]]，場上位置為[[大前鋒|大前鋒]]，並於1970年及1973年获得總冠軍戒指。1978年轉至[[新澤西網隊|新澤西網隊]]，並兼任助理教練。隨後於1980年退休，留下6.7分、4.3籃板的職業生涯平均數據。

== 教练时代 ==
1982年，杰克逊到美国[[大陆篮球协会|大陆篮球协会]]（篮球小联盟）擔任[[奥尔巴尼地主|奥尔巴尼地主]]的總教練，不但帶領球隊奪得總冠軍，也獲得CBA年度最佳教練獎。五年之內，他帶領球隊取得117勝90負的成績。期间他一直在寻求在NBA执教的机会，但未能如愿。
===[[芝加哥公牛|芝加哥公牛]]===
1987年杰克逊开始在公牛擔任助理教練。1989年接替[[道格·科林斯|道格·科林斯]]出任總教練一職，开始在公牛推行[[三角戰術|三角戰術]]，给乔丹拉开空间，並於1990-1991赛季取得61勝21負，更讓球隊取得成立25年來第一個總冠軍。

1990-1993和1995-1998六个赛季，杰克逊指导公牛取得两个三连冠。1996年得到年度最佳教練獎，當年他帶領公牛取得史無前例的72勝10負的傲人成績。這項記錄直到2016年被[[金州勇士|金州勇士]]的73勝9負所打破。

虽然战绩很好，但杰克逊和公牛经理杰里·克劳斯的关系每况愈下。杰里·克劳斯当年聘用了杰克逊为公牛助教，但他认为杰克逊对他并未感恩戴德，而且认为自己在建立公牛王朝中的贡献被忽视了。1997年杰里·克劳斯邀请了公牛所有助教参加自己养女的婚礼，独不邀请杰克逊。在乔丹的努力下，杰克逊与公牛队签下了97-98最后一年合同，杰里·克劳斯明确表示要杰克逊在赛季结束后走人。98赛季后，杰克逊卸任公牛總教练，并称自己不会再任教练。隨著乔丹再次退役，皮朋转隊，公牛王朝就此落幕。

===[[洛杉磯湖人|洛杉磯湖人]]===
1999-2000赛季，禪師开始擔任湖人教练。并指导湖人队获得1999-2002三个赛季的总冠军。但主力球星奥尼尔和科比、科比和杰克逊之间的关系日益紧张。2003年，湖人招来“邮差”马龙和“手套”佩顿，组成F4冲击总冠军，但在总决赛上1:4被[[底特律活塞|底特律活塞]]击败。杰克逊旋即宣布辞去總教练职务。随后奥尼尔被交易至[[迈阿密热火|迈阿密热火]]。

在经历了失败的2004-2005赛季后，湖人重新聘请杰克逊担任主教练。他指导湖人队在2009和2010年再拿两个总冠军。他希望获得湖人队更大的管理权限，但遭到巴斯家族一些成员，尤其是老巴斯的儿子吉姆·巴斯强烈反对。2011年，挑戰3連霸的湖人被[[達拉斯小牛|達拉斯小牛]]直落4橫掃，杰克逊卸任湖人總教练。

截至2010至11年賽季，他的例行賽成績為1155勝485負（70.4%得勝率）；季後賽成績（截至2011年4月25日）則為229勝104負（68.8%的勝率）。2007年，杰克逊以教練身分入選[[籃球名人堂|籃球名人堂]]。

== 總裁時代 ==
===紐約尼克===
2014年3月15日，[[紐約尼克|紐約尼克]]與杰克森正式簽約，成為該隊總裁。<ref>http://sports.163.com/14/0315/07/9NC3L17D00051CA1.html</ref>

因為過去三角戰術的成功，所以強迫教練與球員只能使用三角戰術，也因此尼克隊教練皆被外界視為傀儡教練。實施三角戰術後尼克隊戰績直落，菲尔杰克森將矛頭原因指向球星[[卡梅隆·安东尼|卡梅隆·安东尼]]，不斷向媒體放話要求[[卡梅隆·安东尼|卡梅隆·安东尼]]離隊甚至已經與各隊談論交易內容，但因[[卡梅隆·安东尼|卡梅隆·安东尼]]動用拒絕交易條款使球隊陷入尷尬場面，尼克隊未來基石[[克里斯塔普斯·波爾津吉斯|克里斯塔普斯·波爾津吉斯]] (KP) 支持安東尼，認為禪師此舉極度不尊重球員憤而拒絕隨隊練球以示抗議，菲尔杰克森獲悉後將[[克里斯塔普斯·波爾津吉斯|克里斯塔普斯·波爾津吉斯]]擺上交易檯面（但索價太高無球隊願交易），此舉讓禪師與KP徹底撕破臉，KP更向高層表態自己與禪師只能有一個留在尼克隊。

在菲爾傑克森的領導下，戰績與更衣室氣氛低迷，尼克高層於2017年6月28日對菲爾傑克森開鍘。雖然球迷對傑克森的批評聲音眾多，但傑克森在2015年NBA選秀在廣大反對聲音下堅決選下[[克里斯塔普斯·波爾津吉斯|波爾津吉斯]]為尼克隊留下了球隊新的重要基石是他的一大功績。
