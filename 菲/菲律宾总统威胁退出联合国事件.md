'''菲律宾总统威胁退出联合国'''是一起因菲律宾禁毒而引发的外交事件。

== 禁毒争议 ==
2016年6月30日[[罗德里戈·杜特尔特|罗德里戈·杜特尔特]]在菲律宾首都马尼拉的马拉卡南宫正式宣布就职，成为了菲律宾的16任[[总统|总统]]，杜特尔特曾担任菲律宾南部城市达沃市市长22年，使得原本治安较差的达沃市荣获了“世界第九大安全城市”的称号。<ref>{{ Cite web |url = http://www.thepaper.cn/newsDetail_forward_1508789|title = 澎湃网:菲律宾强力禁毒：滥杀致毒贩横尸街头，超十万人自首}}</ref> 当杜特尔特担任菲律宾总统后，宣布击毙或逮捕一名毒贩可获奖励300万比索，并声称“民众可以随时报警，如果有枪的话也能自己动手，我支持你。毙了他，我会给你颁奖。”还允许禁毒警察就地枪决贩毒者而不需要经过司法程序。在这种高压的环境下，上万名吸毒者或毒贩被逮捕，还有超过60万人自首。然而，随着行动规模的不断扩大，部分民众开始质疑，一些毒贩家属认为尽管他们确实吸毒，但没有拒捕却依旧被杀害。<ref>{{ Cite web |url = http://news.sina.com.cn/c/2016-08-23/doc-ifxvcnrv0698427.shtml|title = 新浪网新闻中心:媒体：禁毒本是好事 为何菲却遭“千夫所指}}</ref>

== 退出联合国 ==

[[联合国人权高级专员办公室|联合国人权高级专员办公室]]8月18日发表声明对菲律宾“非法处决”“杀戮行径”表示谴责，杜特尔特随后声称:“我会向世界证明，你们所谓的专家都是蠢货。”并表示“我无意侮辱你，但我们得考虑下是否要退出联合国了，为什么要听这帮蠢货的话？”继而指着联合国无力解决饥饿问题，任由恐怖主义蔓延，对叙利亚和伊拉克战争束手无策，听任大国狂轰乱炸等。还声称他一点也不在乎这些话所带来的后果。<ref>{{ Cite web |url = http://mil.news.sina.com.cn/china/2016-08-22/doc-ifxvcsrm2133773.shtml|title = 新浪军事:菲总统威胁退出联合国 称要联合中国另建国际组织}}</ref>

== 回应 ==
菲律宾外长亚赛在22日声称“尽管我们对联合国表示担心，但是我们仍然忠实于该国际组织。”并表示杜特尔特总统的言论是为了表达失望和愤怒。菲律宾不会退出联合国。<ref>{{ Cite web |url = http://world.chinadaily.com.cn/2016-08/25/content_26590117.htm|title = 中国日报中文网:菲律宾外长称不会退出联合国 总统仅“表达失望” }}</ref>
