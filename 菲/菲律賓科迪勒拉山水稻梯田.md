'''菲律賓科迪勒拉山水稻梯田'''（{{lang-fil|Mga Hagdan-Hagdang Palayan ng Kordilyera ng Pilipinas}}; [[:en:Ifugao_language|Ifugao]]: ''Payew''；{{lang-en|Rice Terraces of the Philippine Cordilleras}}）或稱'''巴拿威梯田'''指的是位於[[菲律賓|菲律賓]][[呂宋島|呂宋島]]科迪勒拉山脈（{{lang-en|Cordilleras}}）東面的[[梯田|梯田]]，位於{{tsl|en|Banaue|巴拿威鎮}}附近，其梯田規模為[[世界|世界]]最大，也擁有世界最大的人造灌溉系統。1995年被[[聯合國教科文組織|聯合國教科文組織]]登錄為[[世界文化遺產|世界文化遺產]]。科迪勒拉山脈的梯田，也享有「天國的階梯」、「通往天堂的天梯」等美譽。<ref name = "世界文化與自然遺產_2005"> {{ cite book | title = 世界文化與自然遺產 | url = https://books.google.com.tw/books?id=JPpGDwAAQBAJ&pg=PT169&dq=%E8%8F%B2%E5%BE%8B%E8%B3%93+%E6%A2%AF%E7%94%B0+%E4%B8%96%E7%95%8C%E9%81%BA%E7%94%A2&hl=zh-TW&sa=X&ved=0ahUKEwiSr7Hk3-LcAhULUt4KHTPuACcQ6AEIJTAA#v=onepage&q=%E8%8F%B2%E5%BE%8B%E8%B3%93%20%E6%A2%AF%E7%94%B0%20%E4%B8%96%E7%95%8C%E9%81%BA%E7%94%A2&f=false | publisher = 漢宇國際文化, 2005 | author = 由 李偉主編 編輯 | isbn = 9789867185426 | language = zh-tw | date = 2005-01-01 }} </ref>

== 歷史 ==
位於呂宋島中央的科迪勒拉山脈，有許多標高超過1,000[[公尺|公尺]]的山峰。「科迪勒拉」（Cordilleras）即是[[西班牙語|西班牙語]]的「山脈」。[[碧瑤|碧瑤]]這個熱門的避暑聖地即位於此山脈上。在碧瑤東北方的巴拿威鎮（Banaue）、邦圖克、巴達特（Batad）等多處地區為梯田主要分佈的區域。梯田的總長度大約是2,000[[公里|公里]]，故以世界最大規模梯田的名義，在1995年登錄為世界文化遺產。<ref name = "【世界遺產之旅8】菲律賓─南島VS. 巴洛克_2012"> {{ cite book | title = 【世界遺產之旅8】菲律賓─南島VS. 巴洛克 | url = https://books.google.com.tw/books?id=OZ9oBAAAQBAJ&pg=PA14&dq=%E8%8F%B2%E5%BE%8B%E8%B3%93+%E6%A2%AF%E7%94%B0+%E4%B8%96%E7%95%8C%E9%81%BA%E7%94%A2&hl=zh-TW&sa=X&ved=0ahUKEwjA8uXM7uLcAhWJF4gKHRh4BeAQ6AEIMjAC#v=onepage&q=%E8%8F%B2%E5%BE%8B%E8%B3%93%20%E6%A2%AF%E7%94%B0%20%E4%B8%96%E7%95%8C%E9%81%BA%E7%94%A2&f=false | publisher = 國立臺北藝術大學, 2012 | author = 作者：國立臺北藝術大學文化資源學院、童憶 | isbn = 9789860339598 | page = 14 | language = zh-tw | date = 2012-10-01 }} </ref>

單一梯田的大小最大的有2,500平方公尺，最小的梯田只有4平方公尺；用石塊修建的梯田外壁最高4公尺，最矮的外壁不到2公尺高；灌溉梯田的盤山台階渠道總長度達1.9公里。

科迪勒拉山水稻梯田的起源被認為與菲律賓的山地民族[[伊富高省|伊富高]]人（{{lang-en|Ifugao}}）有關。伊富高人是居住在菲律賓呂宋島北部多山地區以[[水稻|水稻]]種植為主要[[經濟|經濟]]活動的原住民部落。他們可能在西元前約1,000-100年時開始耕作這一片梯田。<ref name = "世界文化與自然遺產_2005" />

被列為世界文化遺產的柯迪樂拉山的水稻梯田美麗景色也被繪製在菲律賓20元[[菲律賓披索|披索]]紙鈔背面。<ref name = "亞洲鈔票故事館_2015"> {{ cite book | title = 亞洲鈔票故事館 | url = https://books.google.com.tw/books?id=W-GvDAAAQBAJ&pg=PA103&dq=%E8%8F%B2%E5%BE%8B%E8%B3%93+%E6%A2%AF%E7%94%B0+%E4%B8%96%E7%95%8C%E9%81%BA%E7%94%A2&hl=zh-TW&sa=X&ved=0ahUKEwiSr7Hk3-LcAhULUt4KHTPuACcQ6AEIVjAH#v=onepage&q=%E8%8F%B2%E5%BE%8B%E8%B3%93%20%E6%A2%AF%E7%94%B0%20%E4%B8%96%E7%95%8C%E9%81%BA%E7%94%A2&f=false | publisher = 書泉出版社, 2015 | author = 作者：莊銘國  | isbn = 9789864510061 | page = 103 | language = zh-tw | date = 2015-08-01 }} </ref>

== 維護 ==
有人認為科迪勒拉山水稻梯田被列為世界文化遺產，促使菲律賓政府開始推動梯田保護政策。

水稻梯田被列入世界文化遺產後，有越來越多無視於灌溉溝渠走向的[[建築物|建築物]]出現，農民也因為觀光客眾多，但受益者是旅遊業者而非農民，生產水稻的收入比不上經營觀光業，導致也有農民荒廢梯田、廢耕，水稻梯田美景維持成為[[菲律賓政府|菲律賓政府]]的一大難題，也因此科狄勒拉山水稻梯田世界遺產在2001年被列入《[[濒危世界遗产|瀕危世界遺產名錄]]》，世界瀕危遺產名錄旨在喚醒國際組織、在地的菲律賓政府、以及當地伊富高人民需要趕緊保護水道梯田遺產。<ref name = "世界文化與自然遺產_2005" /><ref name = "大洋洲的物件與文化_ 2008 - 302 頁"> {{ cite web | title = 大洋洲的物件與文化 | url = https://books.google.com.tw/books?id=JV_jT9PI_BMC&q=%E8%8F%B2%E5%BE%8B%E8%B3%93+%E6%A2%AF%E7%94%B0+%E4%B8%96%E7%95%8C%E9%81%BA%E7%94%A2&dq=%E8%8F%B2%E5%BE%8B%E8%B3%93+%E6%A2%AF%E7%94%B0+%E4%B8%96%E7%95%8C%E9%81%BA%E7%94%A2&hl=zh-TW&sa=X&ved=0ahUKEwilo8W26uLcAhUG5bwKHfbaCVs4ChDoAQhUMAg | publisher = 國立自然科學博物館 | author = 王嵩山 ((人類學, 1957- )) | language = zh-tw | accessdate = 2018-08-10 }} </ref>

== 登錄基準 ==
科迪勒拉山的巴拿威水稻梯田因為符合以下標準，而被登錄為世界文化遺產：
*（iii）獨一無二或至少是非常特別地代表了一種文化傳統或是一種現存或已經滅絕的文明。
*（iv）明顯代表某一類建築或[[技術|技術]]，並且展示了[[人類|人類]][[歷史|歷史]]上的某一段或幾段非常重要的時期。
*（v）可作为传统的人类居住地或使用地的杰出范例，代表一种（或几种）文化，尤其在不可逆转之变化的影响下变得易于损坏。
