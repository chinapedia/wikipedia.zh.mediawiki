'''短體魚屬'''（[[學名|學名]]：{{lang|la|''Guizhoubrachysomus''}}，或依其學名譯為'''貴州短體魚'''）為生活於[[三疊紀|三疊紀]]時期的小型[[盧加諾魚科|盧加諾魚科]][[輻鰭魚|輻鰭魚]]，全長約4公分<ref name = Liu03/>。

[[模式種|模式種]]為'''小短體魚'''（''G. minor''，又稱'''小貴州短體魚'''），[[正模標本|正模標本]]目前收藏於[[南京地質古生物研究所|南京地質古生物研究所]]（編號'''NIGP 136039'''），為一具幾近完整的單一個體化石，於[[中國|中國]][[貴州|貴州]]的[[興義市|興義市]][[烏沙鎮_(興義市)|烏沙鎮]]被發現，屬於[[竹竿坡組|竹竿坡組]]地層，年代相當於[[三疊紀|三疊紀]]晚期的{{link-en|卡尼階|Carnian}}<ref name = Liu03/>。

== 詞源與命名歷史 ==
短體魚的學名最初是 ''Brachysomus''，由[[希臘文|希臘文]]的「''{{lang|gre|βραχῦς}}''」及「''{{lang|gre|σώμα}}''」衍生而組成；其中「''{{lang|gre|βραχῦς}} / brachy-''」意為「短的」，而「''{{lang|gre|σώμα}} / soma''」則意指「身體」，主要是形容這種魚短而高的軀體。但「''Brachysomus''」一名早在1823年就已被一個[[鞘翅目|鞘翅目]][[象鼻蟲科|象鼻蟲科]]昆蟲的[[屬|屬]]所使用，由於[[國際動物命名規約|命名規約]]中明訂同樣的學名先發表者具有保留權，而後發表者將視為無效，因此隔年短體魚的屬名以「''Guizhoubrachysomus''」更改並取代，意思是「來自[[貴州|貴州]]的短體魚」<ref name = Liu04/>。模式種的種名在[[拉丁文|拉丁文]]裡意為「小的」<ref name = Liu03>{{ cite journal | author = 刘冠邦; 尹恭正; 王雪华; 罗永明; 王尚彦 | title = "贵州兴义晚三叠世贵州龙层新发现的鱼类" | journal = 古生物学报 | volume = 42 | issue = 3 | pages = 346–366 | year = 2003 | issn = 1000-3118 | url = http://159.226.74.248:8000/pagelinks/136039.pdf }}</ref>。

== 特徵描述 ==
短體魚是小型魚類，魚體短而高。鰓蓋骨和下鰓蓋骨呈三角形。魚鰭具有明顯的棘鱗；[[背鰭|背鰭]]稍大而位置偏前方，呈三角形，起點位置與[[腹鰭|腹鰭]]的起點相對，鰭條約11根；臀鰭大而偏後位，起點和背鰭的背鰭的終點相對，鰭條約7根；尾鰭的背葉和腹葉大小與外型相當，尾裂淺，鰭條約11根。鱗片大而硬，硬鱗表面光滑。身體前中部有兩列狹而高的縱列側鱗<ref name = Liu03/>。
