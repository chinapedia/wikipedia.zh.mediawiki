'''短吻螠虫'''（學名：{{lang|la|'''''Listriolobus brevirostris'''''}}）是[[螠蟲目|螠蟲目]][[螠蟲科|螠蟲科]]的一個物種<ref name="WoRMS_266790" /><ref name = "COL"/><ref name = "source"/>。螠蟲目現時屬於[[多毛綱|多毛綱]]。牠們散佈於[[日本|日本]]<ref name=Nishikawa2004 />、[[中國|中國]][[山東省|山東省]]及[[澳大利亞|澳大利亞]]<ref name=SE1972 />的近岸海底<ref name=csdb />，居住在於軟沉積物建造的[[棲管|棲管]]裡。
