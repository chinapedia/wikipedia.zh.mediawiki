在[[几何|几何]]中，一个'''圆盘'''（{{lang|en|disk 或 disc}}）是由[[平面|平面]]中一个[[圆|圆（circle）]]围成的区域。一個圓只包含[[边界_(拓扑学)|邊界]]，而一個圓盤包含[[内部|内部]]區域。
<br />
在度量幾何與凸分析中，圓盤是[[凸集|凸集]]，因為每兩點之間的[[直线|直线]]點都落在該點集合中；但是圓不是[[凸集|凸集]]，因為它是中空的。

== 開圓盤與閉圓盤 ==
不含[[边界_(拓扑学)|邊界]]的圓盤稱為'''開圓盤'''，包含[[边界_(拓扑学)|邊界]]的圓盤稱為'''閉圓盤'''。

開圓盤與閉圓盤是開區間與閉區間在二維上的推廣（參見[[區間|區間]]）。就[[点集拓扑学|点集拓扑学]]來說，它們都是[[开集|开集]]或[[闭集|闭集]]，開／閉區間是一維的開／閉集，而開／閉圓盤是二維的開／閉集。因此，在[[数学分析|数学分析]]中，如同區間被使用在[[實數線|實數線]]上，圓盤被使用在[[复平面|複數平面]]上用來表示[[邻域|邻域]]。
<br /><br />
要注意的是，因為一個集合可能是一個聯集，所以一個開集不一定是開區間或開圓盤，例如，<math>(0,1)\cup (2,3)</math> 是一個開集，但是它不是一個開區間，因為它不連續。

== 圓盤的度量空間定義 ==
在[[笛卡儿坐标|笛卡儿坐标]]中，以 <math>(a, b)</math> 为中心半径为<math>R</math>的开圆盘由公式
:<math>D=\{(x, y)\in {\mathbb R^2}: (x-a)^2+(y-b)^2 < R^2\}</math>
给出，而同样中心与半径的闭圆盘为
:<math>\overline{ D }=\{(x, y)\in {\mathbb R^2}: (x-a)^2+(y-b)^2 \le R^2\}.</math>

一个半径为<math>R</math>的开圆盘或闭圆盘的[[面积|面积]]是<math>\pi R^2</math>（见 [[圆周率|圆周率]]<math>\pi</math>）。

== 圓盤與球 ==
{{main|球 (數學)}}
[[球_(數學)|球]]是圆盘在[[度量空间|度量空间]]中的推广。不过，'''球'''被用來當作一個一般性的概念，以推廣到多維空間，在這概念下，圓盤是二維空間（歐幾里得平面）中的球。因此，開圓盤是二維的開球，閉圓盤是二維的閉球。

== 物理學中的圓盤 ==
在理论物理学中，圆盘也被用來當作[[二维气体|二维气体]]的氣體分子模型，通常它被视为刚体，所以它們的碰撞是[[弹性碰撞|弹性的]]。
