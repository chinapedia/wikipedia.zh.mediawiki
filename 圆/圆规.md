'''圓規'''在[[數學|數學]]和[[製圖|製圖]]裏，是用來繪製[[圓|圓]]或[[弧|弧]]的工具，常用於[[尺规作图|尺规作图]]。圓規通常是由金屬製成，包括兩部分，由一個[[鉸鏈|鉸鏈]]連接着，其中可作調整，其中一邊尖銳是用作圓心，另一邊通常可裝上[[筆|筆]]。圓規分普通圓規、彈簧圓規、點圓規、樑規等。現代的圓規則多與[[三角尺|三角尺]]、[[量角器|量角器]]、[[直尺|直尺]]等成套裝出售。

== 歷史 ==
圆规的發明最早可追溯至中國夏朝，《[[史記|史記]]·夏本記》載[[大禹|大禹]]治水“左凖繩，右規距”，公元前15世紀的甲骨文中，已有規、矩二字，當時稱為「規」，即今日的圓規，《[[周禮|周禮]]·考工記·匠人》記載：“匠人建國，平地以懸，置槷以懸，眡以景。爲規，識日出之景與日入之景。晝參諸日中之景，夜考之極星，以正朝夕。”。山東嘉祥[[武梁祠|武梁祠]]內有「東漢伏羲女媧磚刻像」，其中[[女媧|女媧]]執規，[[伏羲|伏羲]]執矩，這裡的規是古式梁規，形狀與甲骨文「癸」的字形相似<ref>馮時《中國天文考古學》第33頁</ref>。

== 變型 ==
[[File:Zirkel.jpg|thumb]]
* 橫桿圓規（'''Beam-compass'''）是一種圓規夾有木製或黃銅製的橫桿。
* [[椭圆规|椭圆规]]是一種能夠繪畫[[椭圆|椭圆]]的工具，其中一個款式與圆规近似。
