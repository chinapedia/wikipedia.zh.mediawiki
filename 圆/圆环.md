'''环岛'''也称'''环形交叉'''或'''环形交通'''，是交通节点的一种特殊形式，属于[[平面道路交叉|平面道路交叉]]。圓環由'''环形车道'''和一个'''中心岛'''组成，这种设置使得任何一個方向而來的交通流量進入迴旋處後，均需以迴旋處的中心圈以單一方向旋轉行駛，直至轉向所需的行駛方向而離開。因为它不需要信号灯、减少了交通冲突点、提高了交通安全系数、造价比[[高架桥|高架桥]]便宜等原因，通常认为圓環优于十字路口。有些車流量較高的圓環，車輛需要在圓環外排隊較久的時間才有機會進入圓環。在[[中国|中国]]及台灣某些車流量大的圓環會安装[[交通信号灯|交通信号灯]]来控制交通{{NoteTag|如北京[[上地环岛|上地环岛]]、武汉[[大东门环岛|大东门环岛]]等。}}，至此，圓環已经完全失去了它的意义，車輛會一直卡在環形車道內，完全失去了圓環每分每秒都在消化車流的優點。特別在[[台灣|台灣]]，還會在圓環中間開十字路，這種做法比一般的十字路口和正確的圓環都要更沒效率。

== 优点 ==
[[File:Swindon_Magic_Roundabout.svg|thumb]][[威爾特郡|威爾特郡]]的[[魔術迴旋處_(斯溫頓)|魔術圓環]]，由多個大小不一的圓環構成]]

有不少研究指出，除非道路同時間有其他非機動車使用者，否則迴旋處在統計學實驗上較傳統的[[平面道路交叉|平面道路交叉]]及交通圈安全<ref>{{cite web |url = http://findarticles.com/p/articles/mi_qa3734/is_200703/ai_n18755716/pg_8 |title = A Comparative Evaluation of the Safety Performance of Roundabouts and Traditional Intersection Controls |author1 = Shashi S. Nambisan |author2= Venu Parimi |date = March 2007-03 |work = Institute of Transportation Engineers |accessdate = 2007-11-27 }}</ref>。迴旋處對於交通流量大、平均車速高或[[高速道路|高速道路]]的交通流量管理效率偏低。这种形式使[[车辆|车辆]]以较慢的行驶速度通过交叉口，但是却给[[驾驶员|驾驶员]]提供了更好的通视效果。环形交通的冲突的比普通交叉口少得多。由于较低的行驶速度使环形交通的事故发生率明显较少。通过环形交叉的交通流比普通交叉路口要大。通过环形交叉的平均车流速度较高，因为车辆几乎一直在行驶而不需要停车。

当一个十字路口有逾四个通道时，其交通控制在本质上是比较复杂的。这时环形交通所能联系的通道的数量近取决于环岛的大小。原则上，环形交通能够将此节点处所有道路联系起来。

一个设计得好的环形交通可以很有效率的疏导庞大的[[交通量|交通量]]。环形交通这种简单明了的交通设施具有更好的经济型，平均尾气排量较少，噪声小，其维护费用也少于复杂的交通信号设施。特别是对于大型汽车和载重汽车在平面交叉口的出入口处经常会对於对向车流造成很大的影响。而在环形交叉处，如果直径足够大，对其他车流的影响基本上十分轻微。在环形交叉处，车流可以在无干扰的情况下调头。

與燈控路口相比，迴旋處能應付的車流量不一定比燈控路口為少。在交通流理論而言，以下因素主要影響兩者間應付車流量能力的分別：
* 迴旋處的大小，一般而言較大的迴旋處能處理較大的交通量，反之路口面積對燈控路口的容量幫助不大（較大的路口反而需要更多的紅燈的時間讓上一個時相放行的車輛離開路口）；
* 車輛轉向的比率，對車輛靠左駛的國家和地區而言，右轉車輛對圓形地負荷影響較大；反之對燈控路口而言, 左轉/右轉車輛的比例對路口負荷影響較輕微。

== 缺点 ==
[[File:Bochum_KFZEbene_Kreisverkehr.jpg|thumb]][[波鸿|波鸿]]市中心的一个地下环形交叉]]

环形交叉所需要的占地面积較大，使其不能在任何需要它的地方建造。相比之下，一个使用交通信号灯的普通平面交叉在占地上显得更加经济。

对行人和自行车来说，通过一个环形交叉比跨越一般十字路口要走較遠的路，穿越道會在遠離環形交叉的地方。在环形交叉处没有信号灯，行人具有优先權。自行车行驶的时候，车上的人很难看到身后的机动车辆，所以这就要求机动车驾驶员在驶入环形交叉之前特别注意自行车辆。

交通压力问题在車道量高的环形交叉处一样存在。鉴于“驶出车辆路权优于驶入车辆”的原则，机动车经常要在环形交叉的入口处排队等候。当然这个问题在安装交通信号灯的普通平交道口一样存在，如果它的设计没有满足使用要求的话。虽然大的环形交叉能够承担很繁重的交通压力并完成任务，但它同样也会变得更加危险，因为相对于越大的环形交叉来说，其入口和出口就相对越小。

== 建筑方式 ==
[[File:Boulevard_mosenor_romero.jpg|thumb]]一个立體环形交叉]]

为了减小环形交叉中间区域（即'''[[安全島|中心岛]]'''）的土地浪费，人们经常会在中心岛上做一些建筑小品、雕塑、喷泉或者纪念碑。

城市有轨电车的轨道经常会穿过环形交叉，这就要设置专门的交通信号灯或者特殊的提示标志，以保证有轨电车的优先权。特殊情况下有轨电车也会环绕中心岛行驶。公共汽车站、有轨电车站和地铁站可以设置在中心岛上。

== 歷史 ==
[[File:ColumbusCirclefromTimeWarnerCenterNYC20050807.jpg|thumb]][[哥倫布圓環|哥倫布圓環]]]]

[[法國|法國]][[建築師|建築師]][[尤金·海納德|尤金·海納德]]（Eugène Hénard）於1877年時就已設計出單向繞行的環形道路交叉。[[美國|美國]]建築師{{Link-en|威廉·菲爾普斯·伊諾|William Phelps Eno|威廉·伊諾}}則為現代圓環的設計概念開了先河，並設計了[[紐約市|紐約市]]的[[哥倫布圓環|哥倫布圓環]]，於1905年啟用。其他的圓環亦隨後於美國各地落成，其中有許多圓環採用更大的直徑設計以利高速車輛合流，並規定駛進圓環的車輛優先於原本於圓環內行駛的車輛，然而這種設計卻終告失敗，因為儘管有些圓環的直徑可大至100[[公尺|公尺]]，但仍不足以提供足夠空間已利高速行駛的車輛在圓環內合流，而導致[[交通事故|交通事故]]的比例增高；再者，將優先權給予駛進圓環車輛的規定，也造成圓環內部的[[交通堵塞|交通堵塞]]。以上這種圓環設計的流弊直至1950年代中期才在美國完全消失，世上其他地區圓環的交通問題則直到1960年代期間，現代圓環在[[英國|英國]]發展之後才有所改善。

世界上最有名的圓環位於[[法國|法國]][[巴黎|巴黎]]的[[戴高樂廣場|戴高樂廣場]]。該圓環為十二條主要道路的交匯點，共有八條行車綫，包圍著[[巴黎凱旋門|凱旋門]]。另外，[[澳大利亞|澳大利亞]][[首都|首都]][[坎培拉|坎培拉]]有圓環環繞著[[澳大利亞國會|澳大利亞國會]]，其內的每一個交叉口都設有[[交通號誌|交通號誌]]。目前世界上最大的圓環則位於[[沙烏地阿拉伯|沙烏地阿拉伯]][[東部省_(沙烏地阿拉伯)|東部省]]之首府達曼，其長度大約有1.20[[英里|英里]]。
