'''羅賓·東尼'''（{{lang-en|Robin Tunney}}，{{bd|1972年|6月19日|catIdx=}}），是一位[[美國|美國]][[女演員|女演員]]。

== 簡歷 ==
羅賓·東尼出生於[[美國|美國]][[芝加哥|芝加哥]][[伊利諾伊州|伊利諾伊州]]，父親是[[汽車|汽車]][[推銷員|推銷員]]，母親是[[調酒師|調酒師]]，父母二人也是[[愛爾蘭裔美國人|愛爾蘭裔美國人]]，表哥是[[芝加哥市|芝加哥市]][[議員|議員]]{{tsl|en|Tom Tunney|湯姆·東尼}}。羅賓·東尼在[[芝加哥|芝加哥]]的Chicago Academy for the Performing Arts學習演技，18歲時搬到[[洛杉磯|洛杉磯]]開始其職業生涯。

== 個人生活 ==
東尼在1997年10月4日與[[製片人|製片人]]{{tsl|en|Bob Gosse|鮑伯·高斯}}[[結婚|結婚]]，他們在2006年[[離婚|離婚]]，在2009年東尼與[[澳洲|澳洲]][[導演|導演]]{{tsl|en|Andrew Dominik|安德魯·多米尼克}}成為情人，他們之間的關係由2010年8月結束，2012年12月25日東尼與[[室內設計師|室內設計師]]Nicky Marmet在[[里約熱內盧|里約熱內盧]]度假時訂了婚。
