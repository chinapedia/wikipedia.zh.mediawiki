'''羅利達拉姆國際機場'''位於[[美國|美國]][[北卡羅來納州|北卡羅來納州]]，是該州[[罗利_(北卡罗来纳州)|羅利]]（Raleigh）和[[德罕_(北卡羅來納州)|達拉姆]]（Durham）雙子城共用的民用國際機場。該機場位於羅利市區西北約20公里處。機場佔地4929公頃，共有3條跑道，通航國內國際共38個機場，日起降352架次。2011年旅客吞吐量逾900萬。RDU機場管理局負責機場設施及運營，RDU機場管理局的控制方為[[达勒姆县_(北卡罗来纳州)|达勒姆县]]、[[韦克县_(北卡罗来纳州)|韋克縣]]選出的代表。

== 航線 ==
現時主要是美國國內航線。

輝煌時期有直航到達[[英國|英國]][[倫敦|倫敦]][[希斯路機場|希斯路機場]] 和[[加拿大|加拿大]][[多倫多|多倫多]]。更曾有航空公司視作中樞站。
<!-- 參看 英文版 和 機場網站 -->

截至2009年夏天沒有直航到達亞洲的航線。

油價高以及經濟不景, 各航空公司縮減航線。

== 歷史 ==
1929年開設。1943年正式用作商業客運。

2009年正進行擴建工程。

==跑道==
羅利達拉姆國際機場有三条跑道。2个平行跑道是 5L/23R 和 5R/23L，一个直角跑道 14/32。2个平行跑道有 LED 灯。
