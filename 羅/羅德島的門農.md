'''門農''' （{{lang-el|Mέμνων }}; 前380年 — 前333年），在前334年亞歷山大東征時，門農作為希臘雇傭軍將領，為波斯[[大流士三世|大流士三世]]效力。

== 前期 ==
前358年時，兄長[[羅德島的門托耳|門托耳]]和門農起初受雇於波斯[[赫勒斯滂弗里吉亞|赫勒斯滂弗里吉亞]]總督[[弗里吉亞的阿爾塔巴左斯|阿爾塔巴左斯]]，然而阿爾塔巴左斯和其他總督於前356年發動對波斯[[阿爾塔薛西斯三世|阿爾塔薛西斯三世]]的叛變，但阿爾塔巴左斯的勢力終究被國王軍擊敗，於是門托耳在前353年流亡到埃及，而門農、阿爾塔巴左斯和巴耳賽涅逃到[[馬其頓王國|馬其頓王國]]，投靠[[腓力二世_(馬其頓)|腓力二世]]。

之後，門托耳被波斯俘虜後，向阿爾塔薛西斯三世投降並為波斯效力。在前342年門托耳率軍消滅埃及後，阿爾塔薛西斯三世任命門托耳為帝國西部最高統帥，門托耳向阿爾塔薛西斯三世請求赦免阿耳塔巴左斯等人，讓門農、阿爾塔巴左斯和[[巴耳馨|巴耳馨]]回到波斯，在兄長門托耳去世後，門農繼承雇傭軍統帥，並娶巴耳馨為妻。

== 初次與亞歷山大交手 ==
[[File:BattleofGranicus.JPG|300px]]
當馬其頓的[[亞歷山大大帝|亞歷山大大帝]]繼承其父親[[腓力二世_(馬其頓)|腓力二世]]的遺志，穩定希臘和馬其頓後，開始朝亞洲進軍。而波斯[[小亞細亞|小亞細亞]]的各總督也紛紛帶領自己的部隊集結一起，並在[[澤雷亞|澤雷亞]]城外紮營。在波斯的軍事會議上，深知馬其頓軍事實力的門農，建議不與馬其頓軍正面對抗而改施行焦土政策，焚燒周邊的田地，迫使亞歷山大得不到補給而撤退（當時亞歷山大國庫透支，軍糧不到一個月）。但波斯[[赫勒斯滂弗里吉亞|赫勒斯滂弗里吉亞]]總督[[阿西提斯|阿西提斯]]說：「絕不允許自己領地上有一間房屋被燒」而拒絕這項提議，而其他波斯將領都認為門農不可信任，並決定在前334年與亞歷山大決戰。儘管門農也出戰這場[[格拉尼庫斯河戰役|格拉尼庫斯河戰役]]，卻因波斯陣列遭到馬其頓突破，手下的希臘雇傭軍慘遭馬其頓人屠殺，但門農從這場戰役生還。

== [[哈利卡那索斯圍城戰|哈利卡那索斯圍城戰]] ==
戰後，門農更受大流士三世信任和依靠，任命門農為下小亞細亞總督，統帥波斯艦隊<ref>[[阿利安|阿利安]] 《亞歷山大遠征記》卷一 p.045 </ref>。於是門農開始在[[哈利卡那索斯|哈利卡那索斯]]加強防禦，在那集結許多波斯部隊和希臘雇傭軍，並在港口駐守艦隊防備，準備與亞歷山大長期對抗。當前334年亞歷山大來到哈利卡那索斯後，試圖搭建投石塔把城牆轟倒，但接連幾天的攻城戰都未有戰果，儘管曾把城牆攻破一個缺口，城內泥瓦匠又很快砌了一道新牆<ref>[[阿利安|阿利安]] 《亞歷山大遠征記》卷一 p.047 </ref>，波斯守兵也以居高臨下之勢，向馬其頓軍攻擊。

儘管城內守軍一直試圖衝出城外焚燒馬其頓的投石塔，但遭到馬其頓擊退，戰況讓波斯軍損失慘重。加上城牆一部分已經傾倒，軍中傷員也持續增加，門農決定放棄哈利卡那索斯，趁著夜晚連夜撤出。並焚燒哈利卡那索斯城內的軍需物資，又因當晚風勢過大，哈利卡那索斯全城陷入火海中，最後亞歷山大僅獲得一座殘破的城市。

== 進攻愛琴海島嶼 ==
門農知道希臘各城邦對馬其頓的統治相當反感，開始計畫入侵[[愛琴海|愛琴海]]島嶼來鼓動[[希臘城邦|希臘城邦]]反抗馬其頓人。趁著亞歷山大在[[法西里斯|法西里斯]]休息整頓，門農率領波斯艦隊成功奪下[[希俄斯島|希俄斯島]]。接著又前往[[萊斯博斯島|萊斯博斯島]]進攻，這時島上僅剩[[米蒂利尼|米蒂利尼]]尚未落入門農手中，而[[雅典|雅典]]的反馬其頓派[[狄摩西尼|狄摩西尼]]聽到門農在希俄斯島的勝利，開始鼓吹雅典與其他城邦反叛，這時斯巴達國王[[亞基斯三世|亞基斯三世]]也正在籌備與馬其頓戰爭。

== 病逝 ==
或許亞歷山大相當幸運，門農在圍攻米蒂利尼期間，於前333年病逝，並把指揮權暫時移交給[[法那巴佐斯|法那巴佐斯]]。而許多學者認為，門農若沒因病去世，他的戰略將可能成功，亞歷山大在征服亞洲勢必會增加許多困難，甚至遭到失敗，儘管門農的繼任者法那巴佐斯繼續門農的戰略<ref>[[阿利安|阿利安]] 《亞歷山大遠征記》卷二 p.057 </ref>，但大流士三世無法像信任門農那樣去信任法那巴佐斯，到了[[伊蘇斯戰役|伊蘇斯戰役]]前夕，大流士三世招回法那巴佐斯的希臘雇傭軍，使得希臘只剩斯巴達這支反馬其頓的力量，最終門農的戰略失敗。
