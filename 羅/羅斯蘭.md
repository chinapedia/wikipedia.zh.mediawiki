'''羅斯蘭'''（{{lang-en|Rossland}}）是[[加拿大|加拿大]][[卑詩省|卑詩省]]南部古特尼邊界地區一座城市。據2011年加拿大人口普查所示，羅斯蘭市內人口為3556人<ref name="2011census" />。

羅斯蘭一帶於1890年發現金礦，吸引大批民眾到此淘金<ref name="hist">{{Cite web |url=http://www.rossland.ca/history |title=Rossland History |accessdate=2014-02-25 |work=羅斯蘭市政府 }}</ref>。同年羅斯·湯普森（{{lang|en|Ross Thompson}}）獲取此處160英畝土地的業權，並在此劃下鎮址，本來打算將此處稱為「湯普森」，但由於卑詩省内已有一座名為「湯普森」的城鎮，因此改名為「羅斯蘭」<ref name="GeoBC">{{Cite web |url=http://apps.gov.bc.ca/pub/bcgnws/names/21861.html |title=BC Geographical Names: Rossland |accessdate=2014-02-25 |work=GeoBC }}</ref>。羅斯蘭於1897年3月18日正式建制為市<ref name="GeoBC"/>，人口於同年達致7000，為當時[[加拿大西部|加拿大西部]]最大城市之一<ref name="hist"/>。

卑詩省3B和22號公路於羅斯蘭交匯，兩條公路在市内共構運行。3B公路在兩端皆駁上[[鴉巢公路|3號公路]]；22號公路則向北通往[[卡斯爾加|卡斯爾加]]，向南直達[[美加邊境|美加邊境]]。
