《'''羅密歐與茱麗葉之後現代激情篇'''》（{{lang-en|'''''William Shakespeare's Romeo + Juliet'''''}}）是一部[[1996年電影|1996年]]首映的[[美國電影|美]][[澳洲電影|澳]][[愛情片|愛情片]]，為[[莎士比亞|莎士比亞]]名劇《[[羅密歐與朱麗葉|羅密歐與朱麗葉]]》的眾多電影版之一。電影由[[巴茲·雷曼|巴茲·雷曼]]執導，男女主角則由[[里安納度·狄卡比奧|里安納度·狄卡比奧]]及[[克萊兒·丹妮絲|克萊兒·丹妮絲]]扮演。

== 摘要 ==
導演雖然沒有改變重要對白，卻對故事背景作了相當大程度的改編，如故事發生在現代，地點改成一個名為[[维罗纳|-{zh-hans:维罗纳;zh-hk:凡羅拿;zh-tw:維羅納;}-灘]]的虛構城市，角色決鬥的武器從刀劍變成[[槍械|槍械]]（但槍械牌子有「刀牌」和「劍牌」之分），而蒙特鳩與卡帕萊特兩個家族的對立亦不只是因世仇，更是為了黑幫戰爭；不少人將此片稱為《[[羅密歐與朱麗葉|羅密歐與朱麗葉]]》的[[MTV|MTV]]版。

== 反映 ==
此片叫好亦叫座，不但於1997年[[柏林影展|柏林影展]]獲得多個獎項，在票房收入上也非常成功，只用了約1千4百50萬[[美元|美元]]拍攝，國際票房卻總共有1億4千7百萬美元。<ref name=mojo>{{cite web|url=http://www.boxofficemojo.com/movies/?id=romeoandjuliet.htm |title=Romeo + Juliet (1996) |accessdate=2007-10-14 |publisher=[[Box_Office_Mojo|Box Office Mojo]]}}</ref>

== 芭蕾舞版 ==
2007年，此片被國際盛名的[[芭蕾|芭蕾]]編舞家Peter Martins改編成芭蕾舞劇，配樂則沿用了[[謝爾蓋·謝爾蓋耶維奇·普羅科菲耶夫|謝爾蓋·普羅科菲耶夫]]的音樂。
