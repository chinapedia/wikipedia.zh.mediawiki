[[Image:Codex_Rohonczi_51.jpg|thumb]]
'''羅洪特抄本'''（Rohonc Codex）是一本內容不明的神秘[[書籍|書籍]]，全書共448頁；以當時的發現地[[哈布斯堡君主國|哈布斯堡君主國]]西部都市[[羅洪特|羅洪特]]（今[[奧地利|奧地利]][[布爾根蘭州|布爾根蘭州]][[上瓦特縣|上瓦特縣]][[雷希尼茨|雷希尼茨鎮]]）命名。

寫本用紙檢測出為約1430年時所製造的[[威尼斯紙|威尼斯紙]]，然而書寫的內容卻可能是在紙張製造很久之後才寫上去的：而相反的也有可能是抄寫於更古老的書本內容，因此仍然無法確定切確的成書時間。

== 歷史 ==
可以確定的是，此書最初的擁有者是19世紀[[匈牙利|匈牙利]]貴族[[Gusztáv_Batthyány|Gusztáv Batthyány]][[伯爵|伯爵]]；之後他將此書及其他藏書全部捐贈給[[匈牙利科學院|匈牙利科學院]]，而這本書直到1838年為止都還是伯爵的私人藏書。

不過，這本書的正確起源不明；Batthyány伯爵家在1743年的藏書目錄內有著「[[匈牙利語|匈牙利語]]的[[祈禱文|祈禱文]]・十二開的書本一本」的紀錄，該書的大小和推測的書本頁數和羅洪特寫本的特徵一樣。然而目錄內只有一小段敘述內容，並無法確認是不是羅洪特寫本。

而後於1840年左右，先是由匈牙利學者[[Ferenc_Toldy|Ferenc Toldy]]開始研究寫本內文，而後[[Hunfalvy_Pál|Hunfalvy Pál]]、澳大利亞古文學專家[[Mahl|Mahl]]博士分別繼續研究。1884年至1885年，[[捷克|捷克]][[布拉格|布拉格]]的大學教授[[Josef_Jireček|Josef Jireček]]和[[Konstantin_Jireček|Konstantin Jireček]]研究其中的32頁；1885年送往[[因斯布魯克大學|因斯布魯克大學]]教授[[Bernhard_Jülg|Bernhard Jülg]]處研究，1890年至1892年被匈牙利知名畫家[[Munkácsy_Mihály|Munkácsy Mihály]]送往[[巴黎|巴黎]]研究，不過依舊是徒勞無功。

現在寫本保存於匈牙利科學院內，可利用寫本的[[微縮膠片|微縮膠片]][[拷貝|拷貝]]來進行研究，但是必須要先獲得特殊的許可。

== 內容 ==
[[Image:Codex_Rohonczi_44.jpg|thumb]]
[[Image:Codex_Rohonczi_41.jpg|thumb]]和[[基督徒|基督徒]]]]
羅洪特寫本以十二開的[[威尼斯紙|威尼斯紙]]構成，內頁文字約9-14行；此外還付有類似[[宗教|宗教]]或是[[軍事|軍事]]之類的87幅插畫，其中可以看見諸如[[十字架|十字架]]、[[新月|新月]]和[[卍|卍]]字的圖案，似乎是描寫[[基督徒|基督徒]]、[[穆斯林|穆斯林]]和[[佛教徒|佛教徒]]等不同宗教信仰的人們共同生活的場景。

寫本內使用的文字種類比現在已知的任何一種[[字母系統|字母系統]]還要多上約10倍，不過有一部分記號使用的次數很少；因此也有推測書中文字並非[[字母系統|字母系統]]，而是類似[[日語|日語]]的[[假名|假名]]一樣是[[音節文字|音節文字]]或是[[漢字|漢字]]的[[語素文字|語素文字]]的推測。此外，由左邊不對稱的段落推斷，文字是由右至左書寫。

== 文字 ==
[[Image:Szekely_Hungarian_Rovas_alphabet_Szekely_magyar_rovas_ABC.svg|thumb]]

寫本內書寫的文字和語言都不明，語言推測可能是[[匈牙利語|匈牙利語]]、[[達基亞語|達基亞語]]、[[早期羅馬尼亞語|早期羅馬尼亞語]]或[[欽察語|欽察語]]；此外也有以[[婆羅米文|婆羅米文]]書寫[[印地語|印地語]]的說法，然而以上推測都沒有證據可以加以證實。

1889年，由[[Személyek|Némethi Kálmán]]整理出寫本的文字一覽；1970年，[[Gyürk_Ottó|Gyürk Ottó]]進行有組織性的研究；得出文字是'''由右往左，由上往下'''順序書寫的結論並得知了寫本的頁數順序。1990年代中期，[[Locsmándi_Miklós|Locsmándi Miklós]]利用電腦分析文字內容；發現有一個類似[[羅馬字母|羅馬字母]][[i|i]]的記號似乎是用來分隔句子，同時似乎也代表著[[11|11]]。[[Locsmándi_Miklós|Locsmándi Miklós]]還研究類似[[附加符號|附加符號]]的記號，不過並沒有發現規則性；然而寫本內的文字有著類似[[匈牙利語|匈牙利語]]的語格變化的特徵，因此[[Locsmándi_Miklós|Locsmándi Miklós]]認為寫本不是惡作劇或是偽造品。

不過[[匈牙利|匈牙利]]學者則多半認為，寫本是[[特蘭西瓦尼亞|特蘭西瓦尼亞]]系匈牙利古董商[[Literáti_Nemes_Sámuel|Literáti Nemes Sámuel]]所偽造的。他除了是[[匈牙利國立圖書館|匈牙利國立圖書館]]的共同創設者之一，還是惡名昭彰的贋作作家，甚至還騙過了不少當時有名的學者專家。最先主張[[Literáti_Nemes_Sámuel|Literáti Nemes Sámuel]]偽造說的是匈牙利歷史學家[[Szabó_Károly|Szabó Károly]]於1866年提出，而後1878年[[Fejérpataky_László|Fejérpataky László]]、1899年[[Tóth_Béla|Tóth Béla]]、1930年[[Pintér_Jenő|Pintér Jenő]]、1973年[[Csapodi_Csaba|Csapodi Csaba]]等人的後繼研究也認同這個說法。

== 解讀法 ==
[[Nyíri_Attila|Nyíri Attila]]在僅僅研究寫本的其中2頁後，提出了解讀的方法；他先將文字顛倒順序排列，並找出其中類似[[字母系統|字母系統]]的文字。這個時候，同樣的文字就可能發出不同的發音；相反的，不同的文字也可能發出同樣的發音。不過為了拼出有意義的單字，[[Nyíri_Attila|Nyíri Attila]]亦有將原本文字的順序調整的情況。<ref>Attila Nyíri, ''Theologiai Szemle'', 39 (1996), pp. 91-98.</ref>

以下是[[Nyíri_Attila|Nyíri Attila]]提出的翻譯。

<blockquote>
Eljött az Istened. Száll az Úr. Ó. Vannak a szent angyalok. Azok. Ó<br />
你的神降臨了。主他飛舞著。噢，神聖的[[天使|天使]]們就在那裡。天使們啊。噢。
</blockquote>

羅馬尼亞語言學者[[Viorica_Enăchiuc|Viorica Enăchiuc]]則認為寫本是[[Vlach人|Vlach人]]與[[欽察人|欽察人]]或[[佩切涅格人|佩切涅格人]]的對戰紀錄；因此使用的語言應該是[[世俗拉丁語|世俗拉丁語]]或[[早期羅馬尼亞語|早期羅馬尼亞語]]的一種，[[Viorica_Enăchiuc|Viorica Enăchiuc]]提出的寫本序章的一節翻譯如下。

<blockquote>
Solrgco zicjra naprzi olto co sesvil cas<br />
噢，燃燒的[[太陽|太陽]]啊，無論何時都留下了印記。
</blockquote>

其他還有[[印度|印度]]人[[Mahesh_Kumar_Singh|Mahesh Kumar Singh]]的解讀法，他認為寫本最初的24頁都是將[[印地語|印地語]]直接音譯成[[匈牙利語|匈牙利語]]，並且是尚未發現的聖經[[經外書|經外書]]的開頭部分。內容則是以[[冥想|冥想]]般的方式開始序章，接著敘述[[耶穌|耶穌]]還是孩童時代的事情。

這個說法被刊載在匈牙利的研究雜誌[[Turán|Turán]]上。<ref>Turán, 2004/6 = 2005/1, pages 12-40.</ref>。
