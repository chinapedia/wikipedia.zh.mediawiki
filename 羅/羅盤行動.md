'''羅盤行動'''（{{lang-en|Operation Compass}}）是[[第二次世界大戰|第二次世界大戰]]中盟軍在[[北非戰場|北非戰場]]所進行的第一次軍事行動，行動結果是[[大英國協|大英國協]]的軍隊橫越[[利比亞|利比亞]]大部及俘虜超過100,000名[[義大利|義大利]]士兵，自己只付出很少的代價。

== 序幕 ==
當[[意大利|意大利]]於1940年6月10日向[[法國|法國]]及[[英國|英國]]宣戰後，在利比亞的義大利軍隊及在[[埃及|埃及]]的[[大英國協|大英國協]]軍隊開始發生一系列邊境衝突，在這一系列衝突中較值得注意的是在6月17日[[英軍第11騎兵旅|英軍第11騎兵旅]]攻佔了[[卡普措堡|卡普措堡]]，在之前的6月12日，英軍俘虜了義軍63名士兵，[[貝尼托·墨索里尼|貝尼托·墨索里尼]]摧促利比亞總督[[伊塔洛·巴爾博|伊塔洛·巴爾博]]對埃及的英軍發動大規模攻勢，墨索里尼的即時目標是佔領[[蘇伊士運河|蘇伊士運河]]，最終希望將[[義大利殖民地|義屬北非]]與[[義大利殖民地|義屬東非]]連成一起，不過，基於很多原因，巴爾博不願意這樣做，當巴爾博於6月28日因意外而死去後，墨索里尼即摧促其繼任人，[[魯道夫·格拉齊亞尼|魯道夫·格拉齊亞尼]]元帥發動進攻，與巴爾博一樣，格拉齊亞尼亦不願意這樣做。

格拉齊亞尼是在利比亞的[[義大利第10軍團|義大利第10軍團]]的指揮官，在這片[[義大利殖民地|義大利殖民地]]上，[[義大利第5軍團|義大利第5軍團]]防守[[的黎波里塔尼亞|的黎波里塔尼亞]]，而第10軍團則防衛[[昔蘭尼加盆地|昔蘭尼加盆地]]，當在[[突尼西亞|突尼西亞]]的法國人不再威脅的黎波里塔尼亞後，第5軍團更多資源被抽調出來以支援第10軍團。

當時盟軍在埃及只有30,000人以對抗在利比亞的250,000名義軍，雙方在沙漠使用的均是舊式戰機，[[義大利人|義大利人]]擁有[[菲雅特CR.32|菲雅特CR.32]]及[[菲雅特CR.42|菲雅特CR.42]]式戰鬥機，而[[英國人|英國人]]則擁有[[羅斯特鬥士型戰鬥機|羅斯特鬥士型戰鬥機]]。

格拉齊亞尼解釋他這支數量龐大但機械化程度很低的部隊沒有足夠力量擊敗英國人那支人數雖少但機械化程度很高的部隊，他唯一一支機械代部隊是部份摩托化及只配有輕型裝備的'''馬萊蒂集團'''，該集團由[[彼得羅·馬萊蒂|彼得羅·馬萊蒂]]將軍指揮，但此集團缺乏運輸車輛，它的坦克主要是只有裝配機關槍的[[菲雅特_L3|菲雅特 L3]][[輕型戰車|輕型戰車]]，其最重型的是[[菲雅特_M11/39|菲雅特 M11/39]]，這種坦克配備一門火力較弱的37毫米反坦克炮，而且數量較少，相比之下，英國人則有擁有一些更快的[[巡航坦克|巡航坦克]]([[Mk_I巡航戰車|A9]]、[[Mk_II巡航戰車|A10]]及[[Mk_III巡航戰車|A13]])，它們的數量更超越M11/39，英軍更擁有少量重型的[[馬提爾達2號步兵坦克|馬提爾達]][[步兵坦克|步兵坦克]]，這種坦克速度雖然較慢，但裝甲很厚及火力強大，當時沒有任何一種義大利的[[反坦克炮|反坦克炮]]可將其擊破，雖然同[[德國|德國]][[88毫米高射炮|88毫米高射炮]]一樣的義大利軍[[高射炮|高射炮]]可做到。

== 第10軍團的進攻 ==
雖然如此，格拉齊亞尼仍然遵照墨索里尼的命令，在1940年9月13日率領第10軍團發起進攻，意大利軍越過利比亞與埃及之間的邊境進入埃及，除了數個步兵師及馬萊蒂集團外，還有一個由[[利比亞人|利比亞人]]組成的步兵師。

當佔領了卡普措堡後，進展十分緩慢，意大利人之進攻在3天內只前進了95千米，9月16日，他們便在[[西迪巴拉尼|西迪巴拉尼]]鎮停止前進，在此地，意軍挖掘戰壕、構築防禦工事、等待增援及補給物資。

但是，英國[[皇家海軍|皇家海軍]]已調動力量，包括[[航空母艦|航空母艦]][[:en:HMS_Illustrious_(R87)|光輝號]]到[[地中海|地中海]]，以防備意大利在非洲北部的進一步行動<ref name=HMS-ILLUSTRIOUS>{{cite web

== 英軍主動進攻 ==
在義軍進攻後，韋維爾命令埃及英軍司令[[亨利·梅特蘭·威爾遜|亨利·梅特蘭·威爾遜]]將軍策劃一個有限的攻勢計劃將義軍趕回，羅盤行動最初被計劃進行5天，後由於初期的成功而擴大<ref name="Mead331">Mead, p.331</ref>，1940年12月8日，英軍及[[英屬印度軍隊|英屬印度軍隊]]在[[理查德·奧康納|理查德·奧康納]]中將的指揮下突然向入侵的義大利軍發起進攻，奧康納的部隊是通過在西迪巴拉尼南面之走廊進攻的，整個行動之策劃由[[埃里克·多爾曼·史密斯|埃里克·多爾曼·史密斯]]准將負責。

根據[[情報組織|情報組織]]資料，大部份參戰部隊直至進攻前還以為行動只是[[演習|演習]]，進攻部隊得到[[25磅大炮|25磅大炮]]、[[布倫海姆式轟炸機|布倫海姆式]]及馬提爾達2號步兵坦克的支援。

行動的開始部份被義大利人稱為'''馬爾馬里卡戰役'''，英國人稱為'''軍營戰役'''，馬爾馬里卡戰役這個名字來自位於作戰區域的海岸平原，而軍營戰役這個名字來源自義大利在西迪巴拉尼鎮外的防線上的個別獨立的義軍軍營。

行動開始後5個小時，義軍便開始潰敗，彼得羅·馬萊蒂將軍陣亡，超過4,000名義軍被殺或被俘，3天內，237門大炮、73輛輕型及中型坦克被擊毀或被俘，大約38,300名士兵陣亡或被俘，進攻的英軍沿維多利亞公路向西推進，通過[[哈法雅隘口|哈法雅隘口]]，再次攻佔利比亞境內的卡普措堡。

理查德·奧康納中將想繼續進攻，他希望最少能佔領[[班加西|班加西]]，但是，[[阿奇博爾德·珀西瓦爾·韋維爾|阿奇博爾德·珀西瓦爾·韋維爾]]已命令[[英印第4步兵師|英印第4步兵師]]轉去參與進攻義東非的[[義屬東非戰役|義屬東非戰役]]，奧康納事後表示：「這令他感到十分震驚及不高興……」，他的即時反應是要為此而付出代價。12月14日，[[澳洲第6步兵師|澳洲第6步兵師]]被派來代替印度軍隊，他們剛剛接受完訓練，缺乏裝備，只有一連配有新型的25磅大炮的炮兵連支援。

== 巴迪亞, 圖卜魯格及德爾納 ==
[[File:Awm_005392_2nd11th.jpg|300px]]第2及第11步兵團所組成的聯合部隊經過與[[義大利|義大利]]守軍的激烈戰鬥後佔領了[[圖卜魯格|圖卜魯格]]港]]

對於奧康納的進攻速度，防守剩餘的利比亞之義軍感到震驚，當大英國協的軍隊向前推進時，數個義軍大型單位因物資供應中斷及戰敗而投降，經過激烈戰鬥，一個跟一個單位投降，澳洲軍隊於1941年1月5日佔領[[巴迪亞|巴迪亞]]、1月22日攻佔[[圖卜魯格|圖卜魯格]]及於2月3日佔領[[德爾納|德爾納]]。

墨索里尼在西迪巴拉尼任命[[安尼巴萊·貝爾貢佐利|安尼巴萊·貝爾貢佐利]]中將指揮部隊防守巴迪亞，但40,000名義軍士兵在澳洲第6步兵師進攻巴迪亞一天後便投降，貝爾貢佐利因此在大英國協進攻圖卜魯格時指揮部隊守城。

最終，他在第10軍團於比達夫門被包圍及戰敗時成為了俘虜。

== 比達夫門戰役 ==
英軍之快速進攻令義軍決定放棄昔蘭尼加，1941年1月底，英軍知道義軍從比達夫門撤出昔蘭尼加，由[[米高·摩爾·默克爾|米高·摩爾·默克爾]]少將率領的[[英軍第7裝甲師|英軍第7裝甲師]]截擊正在撤退中的義大利第10軍團。 

默克爾的決定是進攻將通過默基利、墨斯及安特拉特，而[[澳洲第6步兵師|澳洲第6步兵師]]將沿由阿卡达山脈向北到海岸的公路追擊義軍，惡劣的地形令坦克很難前進，因此他要求在利比亞沙漠西南面峽谷的部隊協助，這部隊包括第11騎兵旅、第1皇家禁衛軍旅、皇家來福槍旅第2營、皇家空軍反坦克中隊、第3皇家騎炮兵部隊及第4皇家炮兵團C分隊，整個部隊大約有2,000人，只配備一些MK VI輕型坦克及巡邏坦克。

1941年2月4日，峽谷部隊到達比達夫門鎮及建立防禦工事以截擊撤退中的第10軍團，第2日，義軍前鋒部隊到達及展開進攻，他們的攻勢猛烈而不顧一切，但終於失敗。

2天後，峽谷部隊的步槍手、坦克及大炮包圍了這支由超過100輛坦克及200門大炮支援的大約20,000名意軍，大部份坦克均是新到的[[M13/40坦克|M13/40坦克]]，M13是M11的改良型，擁有一門較佳的47毫米反坦克炮，這能擊毀英軍的輕型及巡邏戰車。

戰事在近距離內發生及經常出現白刃戰。最後一支義軍在2月7日到達，當時最後20輛義軍坦克衝破了由少數步槍手及反坦克炮所組成的包圍圈，但這次突圍卻因為部署在幾碼外的英軍反坦克炮火力而被制止，當最後一次突圍失敗後，英軍第7裝甲師到達及從班加西到來澳洲第6步兵師會合，義軍集體投降。

== 總結 ==
經過10個星期的戰事，義大利第10軍團全軍覆沒，大英國協的軍隊前進了800公哩，擊毀了400輛坦克及1,292門大炮，俘虜義軍超過130,000人，相反，大英國協只有494人陣亡及1,225人受傷，但是，將義大利人趕出非洲北部的進攻被迫停止，當前進到[[阿格海拉|阿格海拉]]時，[[溫斯頓·邱吉爾|溫斯頓·邱吉爾]]首相命令停止進攻及將軍隊因[[希臘-義大利戰爭|希臘-義大利戰爭]]調往[[希臘|希臘]]協防，還有，在1941年1月11日，航空母艦光輝號因為遭受[[俯衝轟炸機|俯衝轟炸機]]轟炸而受損，這容許[[德國|德國]][[非洲軍團|非洲軍團]]能到達的黎波里塔尼亞（[[向日葵行動|向日葵行動]]），沙漠上的戰事又出現完全不同的局面<ref>''The Battle of Alamein: Turning Point, World War II'', pg. 50</ref>。
