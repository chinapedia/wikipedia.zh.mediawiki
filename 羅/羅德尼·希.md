'''羅德尼·希'''（[[英文|英文]]：{{lang|en|Rodney Sieh}}）是一位[[賴比瑞亞|賴比瑞亞]]報紙編輯；2005年，他在賴比瑞亞創立了名為《[[非洲頭版|非洲頭版]]》的日報，該報原只是網路報，後來於2008年開始印製實體報紙，並擴大發行至1,500份，內有多個版面<ref>{{cite web |url=http://pulitzercenter.org/reporting/liberia-front-page-africa-investigative-newspaper-print |title=Liberia: FrontPage Africa, an Investigative Daily |author=Jina Moore|date=19 March 2012 |publisher=The Pulitzer Center |accessdate=2013-12-05}}</ref>。

==遭控誹謗==

2010年，《非洲頭版》、羅德尼·希和該報記者[[山沃·法拉赫|山沃·法拉赫]]（{{lang|en|Samwar Fallah}}）被賴比瑞亞前農業部長[[克里斯多福·托爾|克里斯多福·托爾]]（{{lang|en|Christopher Toe}}）控告誹謗，並求償200萬元，原因是《非洲頭版》曾報導他挪用高達數百萬元的公共資金。對此[[世界报业协会|世界報業協會]]曾表示，該報對該案件的是非曲直沒有意見，只是單純的陳述，該前農業部長所提出的求償是懲罰性的要求<ref>{{cite web |url=http://www.highbeam.com/doc/1G1-221768105.html |title=WAN-IFRA condemns libel judgement in Liberia |date=19 March 2010 |publisher=World Association of Newspapers {{Subscription required|via=HighBeam Research}} |accessdate=2013-12-05 |archive-url=https://web.archive.org/web/20150924152208/http://www.highbeam.com/doc/1G1-221768105.html |archive-date=2015-09-24 |dead-url=yes }}</ref>。

但後來《非洲頭版》、羅德尼·希和山沃·法拉赫仍被判有罪，必須償付罰金<ref>{{cite web |url=http://www.ifex.org/liberia/2011/03/01/fine_upheld/ |title=Court upholds fine against "Frontpage Africa" newspaper in libel case |date=25 February 2011 |publisher=International Freedom of Expression Exchange |deadurl=no |archivedate=2012年9月25日 |archiveurl=https://www.webcitation.org/6Ax5XZl80?url=http://www.ifex.org/liberia/2011/03/01/fine_upheld/ |accessdate=2013-12-05 }}</ref>。

==險遭暗殺==

2011年7月，有政府官員通知羅德尼·希，指該國前高級部長[[威爾斯·納葛斯|威爾斯·納葛斯]]（{{lang|en|Willis Knuckles}}）曾試圖將他暗殺——該部長因為2007年，羅德尼·希報導了他的性醜聞而黯然下台<ref>{{cite web |url=http://allafrica.com/stories/201107210620.html |deadurl=no |title=Liberia: Editor Receives Death Threat |date=20 July 2011 |publisher=IFEX via All Africe |archivedate=2012年9月25日 |archiveurl=https://www.webcitation.org/6Ax5lQFeS?url=http://allafrica.com/stories/201107210620.html |accessdate=2013-12-05 }}</ref>。

==入獄==

2013年8月，羅德尼·希因付不出2010年遭控誹謗的150萬美元罰金而入獄<ref>http://www.economist.com/news/middle-east-and-africa/21585017-president-keeps-peace-fails-reduce-graft-skin-deep-success Liberia: Skin-deep success</ref>。
