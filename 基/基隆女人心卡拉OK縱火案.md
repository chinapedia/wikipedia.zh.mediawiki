'''基隆女人心卡拉OK縱火案'''是於2004年6月16日晚間發生的一起縱火殺人案件，是[[臺灣|臺灣]][[基隆市|基隆市]]的重大刑案。林旺仁於女人心卡拉OK投擲[[燃烧瓶|汽油彈]]縱火，火勢延燒3樓另一家「波麗路PUB」，造成5人死亡8人受傷。<ref name="epochtimes-2004-n570915"/><ref>{{cite news|author=|title=火海救人 三英雄殉難|url=http://www.appledaily.com.tw/appledaily/article/headline/20040618/1015703|accessdate=2016-03-28|newspaper=[[蘋果日報_(台灣)|蘋果日報]]|date=2004-06-18}}</ref><ref name="tvbs-2013-206987">[http://news.tvbs.com.tw/entry/206987 【破案一線間】 縱火害五死！急診室查訪1分鐘　揪嫌] (TVBS)，2013年4月21日</ref>

== 案發過程 ==
林旺仁因不滿其妻在該KTV工作，夫妻倆多次爭吵，一氣之下使用[[塑膠袋|塑膠袋]]填裝汽油，至該KTV投擲汽油袋縱火，火勢一路延竄整棟建築，經過搶救仍造成5死8傷的慘劇。<ref name="epochtimes-2004-n570915"/><ref name="tvbs-2013-206987"/>

5名死者中包括3名消防隊員，他們因本案殉職。<ref name="tvbs-2013-206987"/>

== 後續 ==
案發過後引發附近民眾不滿。由於該案發建築的3樓波麗路卡拉PUB，雖然有小吃店營業執照，但實際上與營業項目不符，且案發時搶救路線也非常狹窄，無法迅速救援。<ref name="epochtimes-2004-n570915"/><ref name="tvbs-2013-206987"/>

後來凶嫌林旺仁在案發後在[[宜蘭縣|宜蘭縣]][[冬山鄉|冬山鄉]]被捕到案，隨後移送[[基隆地檢署|基隆地檢署]]開始偵辦。

== 判刑 ==
2011年11月24日[[最高法院_(中華民國)|最高法院]]判處[[死刑|死刑]]、[[褫奪公權|褫奪公權]]終身定讞。<ref>http://www.appledaily.com.tw/realtimenews/article/local/20111124/95619/applesearch/疑妻討客兄縱火奪5命　醋夫判死刑定讞 蘋果日報</ref> 林旺仁現羈押在[[台北看守所|台北看守所]]等待槍決。
