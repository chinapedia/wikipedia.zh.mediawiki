{{基督教}}

'''基督教哲學'''是包括所有代表二千年來試圖用[[哲學|哲學]]的知識中的理性思考來闡釋[[基督教|基督教]]的教導的理性思考的傳統。

== 怎樣調解基督教和哲學的分歧 ==
在[[宗教|宗教]]及哲學的融合上，過往的努力及嘗試的遭遇到不少困難。這是因為經典[[哲學家|哲學家]]在他們的研究上沒有任何預設的結論，相反經典宗教信眾就已經存在著一套宗教理念。事實上，由於他們不同的目的及立場，有人認為{{who}}沒有人能同時是哲學家，又是[[信仰|信仰]]啟示的真實跟隨者。若基於此論點，所有試圖尋找融合的努力也是白費的。

但是，亦有一些人認為這種融合是可能的。其中一個方法是運用哲學論證去證明一個預設的宗教理念是正確的。這就是所謂的[[護教學|護教學]]，亦是多種宗教傳統，如[[猶太主義|猶太主義]]、基督教及[[伊斯蘭教|伊斯蘭教]]作品的一般技巧，但卻多被不認同是真正的哲學。另一方法是完全捨棄宗教理念是正確的想法，除非在哲學分析上已獲得相同的結論。但是，這方法反而不被該宗教信眾所接受。第三種方法，亦是較少用的方法，就是把[[分析哲學|分析哲學]]應用到個別的宗教上。在此情況上，一個宗教家只要詢問以下問題，亦可以成為一個哲學家：

*[[神|神]]的本質是甚麼？我們怎樣知道神的存在？
*[[啟示|啟示]]的本質是甚麼？我們怎樣知道神將自己的旨意啟示給人類？
*哪一種宗教傳統須以[[文學|文學]]方法解釋？
*哪一種宗教傳統須以[[寓意|寓意]]方法解釋？
*一個人須相信甚麼才算是該宗教的擁護者？
*怎樣調解哲學上的成果與宗教的分歧？
*怎樣調解[[科學|科學]]上的成果與宗教的分歧？

以上鈎畫了基督教哲學家怎樣構築他們的工作。至於其他哲學家則並不是用這些方式建搆他們的工作，例如有人認為{{who}}對於神存在的論證是沒有意義的，因為神的存在在基督教信仰上是沒有懷疑的。

==基督教哲學家與非基督教哲學家的相互影響==
在基督教哲學、[[猶太哲學|猶太哲學]]及[[伊斯蘭哲學|伊斯蘭哲學]]之間有著很多的相互影響。很多基督教哲學家都參考其他猶太及伊斯蘭哲學家的著作，而他們在某一信仰上的討論往往會導致另一信仰上的討論。

有些現今的伊斯蘭哲學家與[[天主教|天主教]]哲學家會共同研究某一課題。

== 基督教哲學的起源 ==
*[[耶穌|耶穌]]：記載在[[四福音書|四福音書]]內耶穌的生平及教導是基督教的基礎。

*[[保羅|保羅]]：大數的掃羅是一個逼迫早期[[基督教教會|基督教教會]]、間接導致[[聖司提反|司提反]]殉道的[[猶太人|猶太人]]。掃羅戲劇性的轉變皈依基督教，並改名保羅。他成為基督教領袖，並寫作多卷書信教導教義及[[神學|神學]]予早期教會。他的一些講章及與[[古希臘哲學家|希臘哲學家]]的辯論亦記載在《[[聖經|聖經]]》的[[使徒行傳|使徒行傳]]。他的書信成為後期基督教哲學家的重要資源。

== 希羅主義下的基督教哲學 ==
希羅主義（Hellenism）是對在耶穌、聖保羅及其後世代的[[羅馬帝國|羅馬帝國]]的[[希臘化文明|希臘文化]]的傳統稱呼。除了[[畢達哥拉斯|畢達哥拉斯]]、[[柏拉圖|柏拉圖]]及[[亞里士多德|亞里士多德]]（他的著作經已遺失了多個世紀）傳統的小數繼承者外，希臘的經典哲學經已過時及被削弱。新興起的哲學派有[[犬儒學派|犬儒學派]]、Skeptics及[[斯多亞學派|斯多亞學派]]，亦是把希羅哲學帶進我們世界的思想及演說家。慢慢地，在希羅主義下產生了一個較完整的吹向，但在某些範疇上，仍有一些反對的聲音。

以下是一些與希羅基督教哲學有著緊密關係的思想家：

* [[特土良|特土良]]：特土良在他皈依基督前已是一位哲學家，他是2世紀的多產[[作家|作家]]，更被稱為“西方教會之父”。他發展了[[靈魂傳殖說|靈魂傳殖說]]（即[[靈魂|靈魂]]是從父母傳給下一代）、神是有形體的（雖然不是肉體）存在及四福音書的權威。他大力打擊[[馬吉安主義|馬吉安主義]]，並且認為希臘哲學與基督教智慧是不相和的。在他末年時，他加入了[[孟他努主義|孟他努主義]]的[[異端|異端]]小教派，因此他不獲[[天主教教會|天主教教會]]封聖。

* [[愛任紐|愛任紐]]：愛任紐最受觸目的是他反對[[諾思底主義|諾思底主義]]，申辯神的獨一性的著作。他論證[[原罪|原罪]]是潛藏在人性中，而[[基督|基督]]的降生成人把從[[亞當|亞當]]而來的原罪“還原”，因此所有人類的生命得以聖化。愛任紐認為基督是人類智慧的老師。

* [[亞歷山太的革利免|亞歷山太的革利免]]

* [[俄利根|俄利根]]：俄利根受[[柏拉圖主義|柏拉圖主義]]的影響，把柏拉圖主義的元素加入[[基督教|基督教]]內。他套用柏拉圖的[[唯心主義|唯心主義]]在[[道|道]]及兩個教會（一個完美的教會，及另一個現實的教會）的觀念上。他亦堅持柏拉圖的完美神觀。

*[[聖奧古斯丁|聖奧古斯丁]]：奧古斯丁發展了經典基督教哲學，大大地受希伯來及[[古希臘|希臘]]思想所影響。他引用希臘思想家[[柏拉圖|柏拉圖]]、[[新柏拉圖主義|新柏拉圖主義]]及[[斯多亞主義|斯多亞主義]]，及從啟示修訂了基督教及《[[聖經|聖經]]》的教導。

*[[亞他那修|亞他那修]]
*[[約翰一世_(君士坦丁堡牧首)|約翰一世]]
*4世紀的[[加帕多家教父|加帕多家教父]]

==中世紀基督教哲學==
{{see also|經院哲學|中世紀的科學發展}}

*[[安瑟倫|安瑟倫]]：安瑟倫最為出名的是他論證神存在的“[[本體論說|本體論說]]”。安瑟倫是其中一位西方思想家直接引用亞里士多德的著作，但是他並沒有亞里士多德的所有著作，而他所有亦只是[[亞拉伯|亞拉伯]]文的翻譯。

*[[托馬斯·阿奎納|托馬斯·阿奎納]]：托馬斯·阿奎納是[[大阿爾伯特|大阿爾伯特]]的學生，是一個忠實的[[多明我會|多明我會]]實踐者。他與13世紀的[[方濟各會|方濟各會]][[罗吉尔·培根|罗吉尔·培根]]（Roger Bacon）十分相似。阿奎納 在基督教重新引入[[亞里士多德|亞里士多德]]之哲學。他相信在信仰與理性之間不存在任何矛盾，而亞里士多德經已達至人類追求[[真理|真理]]的頂峰。他於是運用亞里士多德的哲學作為他構築神學及哲學的藍圖。他是[[巴黎大學|巴黎大學]]的教授，與他同期的有方濟各會的教授[[聖文德|聖文德]]（St Bonaventure）。

== 現代及當代的基督教哲學 ==
以下是現代及當代的基督教哲學家，並以英文字首排序：

*[[卡爾·巴特|卡爾·巴特]]：兩次[[世界大戰|世界大戰]]的[[德國|德國]]幸存者，巴特撰寫一部偉文的《教會教義學》，但他未能完成此書而於1968年歿。巴特強調人類思想與神自身有別的，縱然人嘗試去了解明白神，而神亦透過人類語言與文化啟示祂自己，但人對神的觀念必定與神自身不盡相同。巴特否認自己是一個哲學家，他宣稱自己只是一個教會教義學者及傳道人。

*[[卻斯特頓|卻斯特頓]]：[[英國|英國]]基督教作家，他應用基督教思想於非小說、[[小說|小說]]及[[詩|詩]]的作品上，引發不同的神學、[[倫理|倫理]]、[[政治|政治]]及[[經濟|經濟]]的關注，尤其是在於尋找真理的重要性、反對[[優生學|優生學]]及[[Distributivism|Distributivism]].
