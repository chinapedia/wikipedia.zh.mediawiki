'''基迪昂·泽拉勒姆'''（{{lang-de|'''Gedion Zelalem'''}}，{{bd|1997年|1月26日}}），是一名[[美国|美国]][[足球|足球]]运动员，曾被外借至[[蘇格蘭|蘇格蘭]]球會[[格拉斯哥流浪|格拉斯哥流浪]]，现效力于[[英格兰足球超级联赛|英格兰足球超级联赛]]的[[阿森纳足球俱乐部|阿森纳足球俱乐部]]。

== 早年生活 ==
施拉林生於[[德國|德國]][[柏林|柏林]]，雙親為[[埃塞俄比亞|埃塞俄比亞]]人。五歲時，他加入BFC Germania 1888青年軍，及後於2003年轉至[[哈化柏林足球會|哈化柏林]]青年軍。

施拉林於2006年移民至美國華盛頓<ref>{{cite web|last=Goff|first=Steven|title=Arsenal takes look at Gedion Zelalem, a 14-year-old Ethiopian-German living in Washington|url=http://www.washingtonpost.com/blogs/soccer-insider/post/arsenal-takes-look-at-gedion-zelalem-a-14-year-old-ethiopian-german-living-in-washington/2011/08/21/gIQAAbl5UJ_blog.html|publisher=Washington Post|accessdate=May 4, 2014|date=August 21, 2011}}</ref> ，並為MSC United、Bethesda Soccer Club及Olney Rangers三間球會的青年軍上陣。

== 職業生涯 ==
=== 阿仙奴 ===
[[阿仙奴|阿仙奴]]球探於施拉林為Olney Rangers上陣時所發現，並邀請他與阿仙奴青年隊一同訓練。於訓練營完結後，被阿仙奴簽入。<ref>{{cite web |url=http://arsenalyouth.wordpress.com/first-years/gedion-zelalem/ |title=8. Gedion Zelalem |publisher=Jeorge Bird's Arsenal Youth |accessdate=November 10, 2013}}</ref>

2013年，施拉林隨阿仙奴一線隊參與亞洲之旅<ref>{{cite web |url=http://www.arsenal.com/news/news-archive/arsenal-name-24-man-squad-for-asia-tour |title=Arsenal name 24-man squad for Asia Tour |publisher=Arsenal F.C. |date=July 11, 2013 |accessdate=November 10, 2013}}</ref> ，於對陣越南明星隊、印尼明星隊以及[[名古屋鯨魚|名古屋鯨魚]]時有好表現，被視為明日之星，並與前隊長[[法比加斯|法比加斯]]相提並論。<ref>{{cite web |url=http://worldsoccertalk.com/2013/07/19/no-longer-a-secret-gedion-zelalem-shines-for-arsenal/ |title=No Longer a Secret: Gedion Zelalem Shines for Arsenal |publisher=World Soccer Talk |date=July 19, 2013 |accessdate=November 10, 2013}}</ref><ref>{{cite web |url=http://espnfc.com/blog/_/name/arsenal/id/1863?cc=5739 |title=Starlet Gedion Zelalem excelling on Arsenal tour |publisher=ESPN FC Blog |date=July 22, 2013 |accessdate=November 10, 2013 |first=Andrew |last=Magnan}}</ref><ref>{{cite web |url=http://www.givemesport.com/362690-gedion-zelalem-is-arsenals-latest-teenage-prodigy |title=Gedion Zelalem is Arsenal's latest teenage prodigy |publisher=GiveMeSport |date=July 25, 2013 |accessdate=November 10, 2013 |first=William |last=Cordova}}</ref>

2014年1月24日，施拉林首次為一線隊上陣，於[[英格蘭足總盃|英格蘭足總盃]]對陣[[高雲地利足球會|高雲地利]]時入替[[阿歷士·張伯倫|張伯倫]]。同年3月，施拉林續約阿仙奴，新合約直至2017年結束。<ref>{{Cite web |title=Cazorla, Ramsey, Zelalem sign contracts |url=http://www.arsenal.com/news/news-archive/20140318/cazorla-ramsey-zelalem-sign-contracts |work=Arsenal FC |date=March 18, 2014}}</ref>隨後，施拉林亦為阿仙奴於[[歐聯|歐聯]]上陣。<ref>{{cite web |url=http://www.uefa.com/uefachampionsleague/season=2015/matches/round=2000548/match=2014401/index.html |title=Podolski, Ramsey inspire Arsenal at Galatasaray|publisher=UEFA |date=December 9, 2014 |accessdate=February 11, 2015}}</ref><ref>{{cite web |url=http://www.washingtonpost.com/blogs/soccer-insider/wp/2014/12/09/gedion-zelalem-plays-for-arsenal-in-champions-league-match/|title=Gedion Zelalem plays for Arsenal in Champions League match |date=December 9, 2014 |accessdate=February 11, 2015 |first=Steven |last=Goff}}</ref>

=== 格拉斯哥流浪(外借) ===
2015-16賽季，施拉林被外借至[[格拉斯哥流浪|格拉斯哥流浪]]半季<ref>{{cite news|last1=Hytner|first1=David|title=Arsenal’s Gedion Zelalem to join Rangers on loan until January|url=https://www.theguardian.com/football/2015/aug/18/arsenal-gedion-zelalem-rangers-loan|accessdate=23 May 2016|work=The Guardian|date=18 August 2015}}</ref><ref>{{cite web|title=Gedion Zelalem joins Rangers on loan|url=http://www.arsenal.com/news/news-archive/20150824/gedion-zelalem-joins-rangers-on-loan|publisher=Arsenal FC|accessdate=August 24, 2015|date=August 24, 2015}}</ref> ，於對陣[[艾迪爾聯足球會|艾迪爾聯]]首次為流浪上陣，助攻[[占士·泰雲尼亞|占士·泰雲尼亞]]及[[馬田·域漢|馬田·域漢]]，最終球隊大勝5-0。2016年1月，流浪延長施拉林的租借，直至季尾。<ref>{{cite news|title=Rangers extend Gedion Zelalem loan from Arsenal until summer|url=http://www.bbc.com/sport/football/35247539|accessdate=23 May 2016|work=BBC Sport|date=6 January 2016}}</ref>

== 國家隊 ==
施拉林有資格為美國、德國或埃塞俄比亞國家隊上陣。埃塞俄比亞教練曾希望施拉林為埃塞俄比亞上陣，可惜遭到拒絕。<ref>{{cite web |url=http://www.bbc.co.uk/sport/0/football/24537467 |title=Gedion Zelalem: Ethiopia keen on Arsenal teenager |publisher=BBC Sport |date=October 15, 2013 |accessdate=November 10, 2013}}</ref> 他曾為[[德國|德國]]U15、U16及U17上陣，隨後他退出德國U17大軍名單，並拒絕為德國U18上陣，因為他希望為美國隊上陣。其後他申請成為美國公民，有資格為美國隊上陣。

2015年5月，施拉林獲美國U20邀請，參與[[2015年U20世界盃足球賽|2015年U20世界盃足球賽]]，可惜於八強被[[塞爾維亞|塞爾維亞]]淘汰出局。<ref>{{cite web|title=Gedion Zelalem makes U.S. debut in under-20 friendly|url=http://www.espnfc.us/united-states/story/2457677/gedion-zelalem-makes-us-debut-in-under-20-friendly|website=ESPNFC |date=May 19, 2015|accessdate=May 19, 2015}}</ref>
