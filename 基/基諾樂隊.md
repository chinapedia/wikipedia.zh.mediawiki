'''基諾樂隊'''是2004年成立於[[中華人民共和國|中華人民共和國]][[湖北省|湖北省]][[武漢市|武漢市]]的一個男子[[搖滾樂隊|搖滾樂隊]]，最初名爲「靜態滑行」樂隊<ref name="基诺The Chinos">{{cite web | url=http://ent.enorth.com.cn/system/2009/04/24/003981080.shtml| title=草莓艺人：基诺The Chinos| author=北方網| accessdate=2009-04-28}}</ref>，2009年後不見組合活動。

== 成員 ==
樂隊的三個成員皆來自[[湖北|湖北]][[武漢|武漢]]。主唱謝欣，吉他胡坤，貝斯手張裔。<ref name="基诺The Chinos"/> 

== 發展史 ==
2002年，謝欣胡坤表兄弟二人和幾個好友組建了一個非職業的樂隊。2004年底，胡坤和興趣相投的大學同學張裔等一起組建了基諾樂隊的前身「靜態滑行」樂隊，並開始參加武漢一些[[搖滾|搖滾]]演出。2006年，樂隊的作品開始通過網絡傳播，並於當年簽約了先生小姐唱片公司。2007年，由原[[達達樂隊|達達樂隊]]的前吉他手吳濤擔任製作人，樂隊在北京錄製了首張專輯《再見！維多利亞》，並於2008年推出。<ref name="基诺The Chinos"/> ，<ref name="基诺《再见维多利亚》：只剩下英伦的英伦乐队">{{cite web | url=http://ent.sina.com.cn/y/m/2008-07-01/21182083614.shtml| title=基诺《再见维多利亚》：只剩下英伦的英伦乐队| author=新浪音樂| accessdate=2008-07-01}}</ref>
