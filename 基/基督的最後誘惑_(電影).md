《'''基督的最後誘惑'''》（{{lang-en|'''''The Last Temptation of Christ'''''}}）是一套1988年的[[美國電影|美國]]宗教電影，改編自[[尼可斯·卡赞扎基斯|尼可斯·卡山札基]]有爭議性的1955年[[基督的最後誘惑|同名小說]]，描繪[[耶穌|耶穌]]的一生。電影由[[馬田·史高西斯|馬田·史高西斯]]執導，[[威廉·達福|威廉·達福]]飾演耶穌，另外的主要演員還有[[夏菲·基圖|夏菲·基圖]]、{{link-en|芭芭拉·荷西|Barbara Hershey}}、[[哈里·迪恩·斯坦頓|哈里·迪恩·斯坦頓]]、[[大衛·寶兒|大衛·寶兒]]。本片完全在[[摩洛哥|摩洛哥]]拍攝。

像小說一樣，這部電影描述了耶穌基督的生活和他對各種形式[[引誘|誘惑]]的掙扎，包括恐懼、懷疑、抑鬱、不情願和性慾。這導致了原著和電影中描繪基督想像自己參加性活動而被誘惑，這個觀念對一些基督徒引起了冒犯。本片包括一個免責聲明，說明電影偏離了普遍公認耶穌生活的{{link-en|新約聖經中的耶穌生活|Life of Jesus in the New Testament|聖經形象}}，而且不是基於[[福音書|福音書]]。

本片得到了兩極化評價。史高西斯獲得提名[[奧斯卡金像獎|奧斯卡金像獎]]的[[奧斯卡最佳導演獎|最佳導演]]，而荷西飾演抹大拉的馬利亞為她獲得提名[[金球獎最佳電影女配角|金球獎最佳電影女配角]]，而基圖飾演的猶大為他獲得提名{{link-en|金草莓獎最差男配角獎|Golden Raspberry Award for Worst Supporting Actor|金草莓獎最差男配角}}。

== 製作 ==
史高西斯從童年時已經想製作一部耶穌一生的電影版本。在1972年電影《[[列車女賊|列車女賊]]》中指導{{link-en|芭芭拉·荷西|Barbara Hershey}}時，她給了他一本卡山札基的小說。史高西斯在1970年代末買下這部小說的電影版權，並把它交給[[保羅·許瑞德|保羅·許瑞德]]去改編。《基督的最後誘惑》最初是史高西斯的《[[喜劇之王_(1983年電影)|喜劇之王]]》的下一步作品，製作安排於1983年的[[派拉蒙電影|派拉蒙電影]]開始，預算約1400萬美元，並在[[以色列|以色列]]拍攝。最初的演員包括[[艾丹·奎因|艾丹·奎因]]飾演耶穌、[[史田|史田]]飾演本丟·彼拉多、{{link-en|雷·戴維斯|Ray Davies}}飾演加略人猶大，以及{{link-en|Vanity (singer)|Vanity (singer)|Vanity}}飾演抹大拉的馬利亞。派拉蒙的管理層及其當時的母公司{{link-en|Gulf and Western Industries|Gulf and Western Industries|Gulf+Western}}由於開始憂慮暴漲的預算和來自宗教團體的抗議信函，該項目進入了周轉期，並最終於1983年12月被取消，而史高西斯則接下來製作《[[三更半夜|三更半夜]]》。

1986年，[[環球影業|環球影業]]對該項目感興趣。史高西斯提出在58天內以700萬美元拍攝本電影<ref>{{Cite web |url = http://www.christianitytoday.com/ct/2013/august-web-only/last-temptation-of-christ.html |title = Last Temptation Turns Twenty-Five |author = Kenneth R. Morefield |date = 2013-08-07 |publisher = Christianity Today |language = en}}</ref>，環球影業對製作亮起[[綠燈_(電影製作)|綠燈]]。評論家和編劇{{link-en|傑伊·庫克斯|Jay Cocks}}與史高西斯一起修改了許瑞德的劇本。[[艾丹·奎因|艾丹·奎因]]推辭耶穌的角色，史高西斯重新揀選演員威廉·達福來飾演。史田也推辭了彼拉多的角色，角色則由大衛·寶兒飾演。[[主要拍攝|主要拍攝]]於1987年10月開始。在摩洛哥的拍攝（對於史高西斯來說是第一次）是困難的，而且由於匆忙的計劃進度而增加困難。「我們都在緊急狀態下工作，」史高西斯回憶說。場景必須是即興創作，而且很少深思熟慮去設計出來的，使史高西斯為本片開發了一個極簡抽象派藝術的美學。拍攝在1987年12月25日殺青。

=== 音樂 ===
本片的[[原聲音樂|原聲音樂]]由[[彼得·蓋布瑞爾|彼得·蓋布瑞爾]]作曲，1988年獲得提名[[金球獎最佳原創配樂獎|金球獎最佳原創配樂獎]]，並在CD上發行，標題為《''{{link-en|Passion (Peter Gabriel album)|Passion (Peter Gabriel album)|Passion}}''》，在1990年獲得了一項最佳新時代專輯格萊美獎（Best New Age Album）。本片的配樂本身幫助了普及[[世界音樂|世界音樂]]。蓋布瑞爾隨後合輯了一張名為《''{{link-en|Passion – Sources|Passion – Sources}}''》的專輯，包括不同音樂家的額外材料，這些音樂家啟發他製作原聲帶，或他為原聲帶採樣。

=== 巴黎聖米歇爾戲院襲擊 ===
1988年10月22日，一群天主教徒於[[巴黎|巴黎]]的聖米歇爾戲院（Saint Michel theatre）公映該電影的期間縱火，襲擊令13人受傷，其中4人嚴重燒傷。<ref name="james">{{Cite web |url = http://www.nytimes.com/1988/11/09/world/religious-war-ignites-anew-in-france.html |title = Religious War Ignites Anew in France |author = James M. Markham |date = 1988-11-09 |publisher = 
The New York Times |language = en}}</ref><ref name="steven">{{Cite web |url = http://www.nytimes.com/1988/10/25/movies/police-suspect-arson-in-fire-at-paris-theater.html |title = Police Suspect Arson In Fire at Paris Theater |author = Steven Greenhouse |date = 1988-10-25 |publisher = The New York Times |language = en}}</ref>聖米歇爾戲院被嚴重破壞，3年後才重新開放。<ref name="steven"/>[[天主教巴黎總教區|巴黎大主教]]{{link-en|Jean-Marie Lustiger|Jean-Marie Lustiger}}在沒有觀看本片前預先譴責電影，但他也譴責這次襲擊，稱這些犯案者為“基督的敵人”。<ref name="steven"/>

這次襲擊後來歸咎於一個基督教原教旨主義者團體的成員Bernard Antony，他是極右派[[國民陣線_(法國)|國民陣線]]的代表，以及[[絕罰|被逐出教會]]的大主教{{link-en|馬歇爾·勒費弗爾|Marcel Lefebvre}}的追隨者。<ref name="james"/>類似襲擊包括塗鴉、引爆催淚彈和臭彈，及毆打戲院觀眾。<ref name="james"/>至少有九名被認為是該團體的成員被捕。<ref name="james"/>一個被稱為「反對種族主義以尊重法國和基督教身份的聯盟」（Alliance générale contre le racisme et pour le respect de l'identité française et chrétienne）的五名武裝分子團體被判15至36個月的緩刑監禁，還要為損害賠償450,000[[法國法郎|法郎]]罰款。<ref>{{Cite web|url=http://www.humanite.fr/node/4317 |title=L'ABSOLUTION DES TERRORISTES |date=1990-04-04 |publisher=L'Humanité |language=fr |deadurl=bot: unknown |archiveurl=https://web.archive.org/web/20140714230115/http://www.humanite.fr/node/4317 |archivedate=2014-07-14 }}</ref>

=== 家庭媒體 ===
雖然《基督的最後誘惑》在[[VHS|VHS]]和[[鐳射影碟|鐳射影碟]]上發售，但許多錄像租賃店，包括當時占主導地位的[[百視達|百視達]]，也因為電影的爭議性反響而拒絕出租本片。<ref name=Criterion>{{cite video|people=Martin Scorsese|title=''The Last Temptation of Christ'' [audio commentary]|medium=Laserdisc/DVD/Blu-ray Disc|publisher=The Criterion Collection|location=New York|date=1997|display-authors=etal}}</ref>1997年，[[標準收藏|標準收藏公司]]在鐳射影碟上發行了《基督的最後誘惑》的特別版，標準收藏於2000年在DVD上重新發行，並於2012年3月發行區碼A[[藍光光碟|藍光光碟]]。<ref>{{Cite web |url = http://www.blu-ray.com/news/?id=7905 |title = Criterion Blu-ray in March: Scorsese, Kalatozov, Hegedus & Pennebaker, Baker, Lean (Updated) |author = Josh Katz  |date = 2011-12-15 |publisher = Blu-ray.com |language = en}}</ref>

== 反響 ==
=== 票房 ===
《基督的最後誘惑》於1988年8月12日在123間劇院上映，並在上映週末獲得401,211美元。本片在財政上不是很成功。本片落畫時，在國內只獲得8,373,585美元（拍攝成本超過7,000,000美元），在墨西哥則獲得487,867美元，全球總額為8,861,452美元。<ref>{{Cite web |url = http://www.boxofficemojo.com/movies/?page=intl&id=lasttemptationofchrist.htm |title = The Last Temptation of Christ |publisher = Box Office Mojo |language = en}}</ref>
