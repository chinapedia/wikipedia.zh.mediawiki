在[[阿卡德|阿卡德]]史诗《[[埃努玛·埃利什|埃努玛·埃利什]]》（“Enuma Elish”，也称作“创世的七块泥板”）中，'''基沙尔'''（Kishar）<ref>《神话词典》，“基沙尔”，第200页，辽宁人民出版社，1989年</ref>是拉赫穆（Lahmu）与拉哈穆（Lahamu）-[[迪亚马特|提阿玛特]](Tiamat)和阿勃特（Abzu）的第一个孩子所生的女儿。她是安沙尔（Anshar）的姐姐和妻子，[[安努|安努]]（Anu）的母亲。<ref>{{cite web |url=http://oracc.museum.upenn.edu/amgg/listofdeities/anarandkiar/|title=安沙尔和基沙尔(男神和女神)|last=Brisch|first=Nicole|date=2012年|website=古代美索不达米亚众神|publisher=Oracc 和英国高等教育学院|accessdate=2013年6月19日}}.</ref>基沙尔可能作为天空神安沙尔的对偶神，<ref>布兰克，杰里米和安东尼·格林, 1992年. 《神仙、妖魔和古美索不达米亚符号：一部图解词典》'' 伦敦: 大英博物馆出版社. 第34页.</ref>代表了大地，可能被看作是[[地母神|地母神]]。

在《埃努玛·埃利什》中，基沙尔只出现过一次，在史诗开头的几行，然后从故事中消失。她似乎只是在公元前一世纪的其它文献中偶尔出现，并可能同等于与苍天神[[安努|安努]]的女性对应神-安图（Antu）女神。<ref>{{cite web |url=http://oracc.museum.upenn.edu/amgg/listofdeities/anarandkiar/|title=安沙尔和基沙尔(男神和女神)|last=Brisch|first=Nicole|date=2012|website=古代美索不达米亚众神|publisher=Oracc 和英国高等教育学院|accessdate=2013年6月19日}}.</ref>
