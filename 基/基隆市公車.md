'''基隆市公車'''（[[英語|英語]]譯名：{{lang|en|Keelung City Bus}}）是[[台灣|台灣]][[基隆市|基隆市]]的[[台灣汽車客運|公共汽車]]系統，主管機關為[[基隆市政府|基隆市政府]]（承辦單位：基隆市政府交通旅遊處）。其狹義上僅指'''基隆市公共汽車管理處'''（簡稱'''基隆市公車處'''）所經營之公共汽車系統，廣義上則包含基隆市境內所有的市區公車路線，包含基隆市公車處與[[基隆客運|基隆客運]]兩家業者。
[[File:Keelung_City_Bus_Management_Office_20170429a.jpg|thumb]]

== 概要 ==
[[File:Keelung_City_Bus_Station_and_YM_OCAM_20170429.jpg|thumb]]
路線主要以[[基隆車站|基隆車站]]為中心，向基隆市內各大[[道路|道路]]延伸，行車路線計45條，平均每日行駛約1,400班次，每日載客人數約74,000人次。除一般路線外，尚包括遊園公車（[[中正公園|中正公園]]遊園公車）、愛心公車（[[和平島|和平島]]─[[長庚醫院|長庚醫院]]、[[基隆市立信義國民中學|信義國中]]─[[國家新城|國家新城]]）、聯線公車（堵南里─國家新城）、休閒農業公車（七堵郵局─各休閒農場─七堵停車場）。

== 歷史 ==
基隆市的公車發展，起源於[[台灣日治時期|日治時代]]中期。1945年[[臺灣戰後時期|中華民國接管臺灣後]]，鑑於市區交通之需要，[[基隆市政府|基隆市政府]]利用日治時代[[基隆市役所|基隆市役所]]擁有的大型[[自動車|自動車]]（[[巴士|巴士]]），於1946年4月9日成立「基隆市公共汽車管理所」，負責經營市區交通業務。1947年3月初，基隆市公共汽車管理所一度改隸[[交通路公路總局|臺灣省公路局]]，但於同年6月5日恢復市營。1952年8月1日，奉[[臺灣省政府|臺灣省政府]]核准，基隆市公共汽車管理所改為「基隆市公共汽車管理處」，為市屬交通事業單位。

2012年8月9日，[[基隆客運|基隆客運]]開闢基隆市內[[臺鐵捷運化|臺鐵捷運化]][[接駁公車|接駁公車]]路線，為基隆市公車首度有民營業者加入營運。2017年7月1日，基隆市公共汽車管理處乘客申訴專線由基隆市政府交通旅遊處電話號碼改為基隆市政府[[1999專線|1999專線]]。

== 路線 ==
目前全線設有總站（位在[[基隆車站|基隆車站]]前）及12處分站。各路線以總站為營運中心，向市內各地（分站）展開。

一般路線的編號採取『●◎◎』的型式，『●』表示終點，『◎◎』則是流水號(例如01 02 ~09 10)；這是從1987年2月10日開始實施的編號方式，見1987年2月的《[[聯合報|聯合報]]》基隆市版。

== 乘車規則 ==
*前上後下：前門上車，後門下車。
*上車收費：一律上車投現，或使用[[一卡通_(台灣)|一卡通]]、[[基隆交通卡|基隆悠遊卡]]、[[悠遊卡|悠遊卡]]、[[有錢卡|有錢卡]]<ref>[https://www.happycashcard.com.tw/corporate/message/in.do?id=20170614112531477493 基隆市公車全路線上線!]2017-06-15,遠鑫電子票證公司</ref>、[[愛金卡|愛金卡]]<ref>[https://www.icash.com.tw/Home/NewsDetail/?ID=9147 持icash2.0搭乘基隆市公車即享OPENPOINT點數天天5倍送]2017-12-12,愛金卡公司</ref>等電子票證付費。
*公車離站以後、靠站以前，一律不開門。公車停靠招呼站以前，若車上無人按[[下車鈴|下車鈴]]表示下車意願，且站內無人招手-{表示}-乘車意願，則過站不停。若乘客按下車鈴時已明顯不可能安全停靠最近站牌，視為下一站下車意願。
*車上禁止吸[[香菸|菸]]、飲食、嚼食[[口香糖|口香糖]]或[[檳榔|檳榔]]。

== 費用 ==
[[File:Keelung_Transportation_Card(Students).JPG|thumb]]

全票15元，半票8元，學生優待卡9元。

基隆市公車的收費方式-{}-採取投現收費與[[智能卡|電子票證收費]]並行，車費則是全部皆以一段票收費。投現收費不找零，因此須於乘車前自備足額零錢。初期基隆市公車的電子票證，稱為'''[[基隆交通卡|基隆交通卡]]'''，採用與[[悠遊卡|悠遊卡]]相容的介面系統，2004年7月1日起發行。2007年9月1日起，基隆市公車處停售基隆交通卡，改為使用[[悠遊卡公司|悠遊卡公司]]發行的'''[[悠遊卡|基隆悠遊卡]]'''。

2006年6月30日以前，票價為全票12元、半票9元，一律上車投錢。2006年7月1日起，單段票價調整為全票15元、學生票9元（僅限於使用學生交通卡，若為投現則票價比照全票）；老人、身心障礙者與兒童優待票8元。另外，持有基隆交通卡敬老愛心卡、敬老卡、關懷卡，可免費搭乘公車，關懷陪伴卡緊接在關懷卡後使用可享半票優惠，惟這些卡種必須在各區公所提出申請。
