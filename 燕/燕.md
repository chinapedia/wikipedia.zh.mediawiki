== [[中國古代|中國古代]][[地方政府|地方政權]] ==
*'''[[燕京戎|燕京戎]]'''，简称“燕”。[[商朝|商朝]]末期[[山西省|山西]][[汾河|汾河]]流域的游牧民族。
*'''[[南燕国|南燕国]]'''，[[周朝|周朝]]姞姓诸侯小国，位于现[[河南省|河南省]]。
*'''[[燕国|燕国]]'''，[[周朝|周朝]][[召公|召公]]建立的政權，位于[[河北省|冀]][[辽宁省|辽]]一带的[[姬姓|姬姓]]燕国，一稱'''北燕國'''，战国七雄之一。
*'''[[燕国_(楚汉战争)|燕国 (楚汉战争)]]'''（前208年—前202年），[[秦朝|秦]]末[[汉朝|漢]]初[[韩广|韩广]]、[[臧荼|臧荼]]建立地方政權。 
*'''[[广阳郡|燕国]]'''（前202年—前195年），[[汉朝|漢]]初高祖[[刘邦|刘邦]]册封[[卢绾|卢绾]]建立诸侯国。
*'''[[广阳郡|广阳郡]]'''，[[汉朝|汉朝]][[幽州|幽州]]地区的一个郡，建立时称'''燕国'''，[[汉昭帝|汉昭帝]]时改名广阳郡，[[三国|三国]]及[[晋朝|晋朝]]复称燕国。
*'''[[燕国_(公孙渊)|燕国 (公孙渊)]]'''，[[三国|三国]]时期'''[[公孫淵|公孙渊]]'''自称燕王建立的政權，位于[[辽东郡|辽东郡]]，后被[[曹魏|曹魏]]消灭。
*'''[[慕容氏諸燕|慕容氏諸燕]]'''，指[[五胡十六国|十六國]]時代，由[[鲜卑|鲜卑]][[慕容姓|慕容氏]]成員及其養子等，所建立的數個政權。
**'''[[前燕|前燕]]'''（337年-370年），[[鲜卑|鲜卑]]人[[慕容皝|慕容皝]]建立的政權。
**'''[[西燕|西燕]]'''（384年-394年），[[鲜卑|鲜卑]]人[[慕容泓|慕容泓]]建立的政權。
**'''[[後燕|後燕]]'''（384年—407年），[[鲜卑|鲜卑]]人[[慕容垂|慕容垂]]建立的政權。
**'''[[南燕_(十六國)|南燕 (十六國)]]'''（398年—410年），[[鲜卑|鲜卑]]人[[慕容德|慕容德]]建立的政權。
**'''[[北燕|北燕]]'''（407年—436年），[[慕容垂|慕容垂]]的養子[[高雲|高雲]]所建之政權。
*'''燕'''（615年—616年），[[隋朝|隋]]末[[王須拔|王須拔]]、[[魏刀兒|魏刀兒]]起事自立之国号，后改魏。
*'''燕国'''（618年—624年），[[隋朝|隋]]末[[唐朝|唐]]初[[高开道|高开道]]建立地方政權。
*'''[[燕_(安史之乱)|大燕]]国'''（756年—763年），[[唐朝|唐朝]][[安史之亂|安史之亂]]时期叛軍將領[[安祿山|安祿山]]建立的政權。
*'''[[燕_(五代)|燕 (五代)]]'''（911年—913年），[[五代十国|五代十國]]時期[[劉守光|劉守光]]建立的政權。

== 地理名称 ==
*[[河北省|河北省]]北部一帶，「'''燕地'''」是該地的別稱。
*[[北京市|北京]]、[[保定市|保定]]一帶，「'''[[北京市|燕京]]'''」是該地的雅稱。
*'''[[燕市|燕市]]'''，[[日本|日本]]城市名稱，在[[新潟县|新潟县]]。
