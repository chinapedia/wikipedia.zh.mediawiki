'''燕话'''，又名'''卫里话'''，是[[中国|中国]][[浙江省|浙江省]][[慈溪市|慈溪市]]的一个[[闽语|闽语]][[方言岛|方言岛]]，位于[[观海卫镇|观海卫镇]]，形成于[[明代|明代]]，目前面临消失的危险。

== 分布 ==
燕话的使用者主要分布于观海卫镇西北角<ref>{{cite web|url=http://www.cxnews.cn/gb/node2/node218/node223/userobject1ai142774.html|date=2010-09-20|accessdate=2010-10-05|title=慈溪“非遗”掠影|author=方印华|deadurl=yes|archiveurl=https://web.archive.org/web/20101005104553/http://cxnews.cn/gb/node2/node218/node223/userobject1ai142774.html|archivedate=2010-10-05}}</ref>，卫西村
、卫北村一带，目前仅有不足千人使用。<ref name="main">{{cite web|url=http://news.cnnb.com.cn/system/2009/12/25/006370727.shtml|publisher=中国宁波网|date=2009-12-25|accessdate=2010-10-05|title=观海卫老城六百年燕话何去何从？}}</ref>

== 来源 ==
目前，对观海卫燕话的来源有两种说法。第一种说法认为，明代将领[[戚继光|戚继光]]在浙江沿海抗击[[倭寇|倭寇]]时，带领的福建军人将家乡话带到了观海卫。第二种说法认为，观海卫建卫时，征调的福建省[[福宁|福宁]]卫旗军将当地方言带入观海卫，逐步形成了燕话。<ref name="main" />

== 轶事 ==
著名[[宁波帮|宁波帮]]人士，镇海龙山（今属慈溪市[[龙山镇_(慈溪市)|龙山镇]]）人[[虞洽卿|虞洽卿]]曾经使用燕话作为证券交易的暗语。[[抗日战争|抗日战争]]时期，[[中国共产党|中国共产党]]领导的抗日力量三五支队也曾经使用燕话作为秘密联络的话语。<ref name="main" />
