'''燕子湖'''是[[綠島|綠島]]島上的一個臨海地區的名稱，目前屬[[綠島人權文化園區|綠島人權文化園區]]轄管範圍內。

== 慈航宮 ==
'''慈航宮'''為一座位處當地的廟宇，其特色為其廟前之兩座紅色石獅。<ref>{{Cite web|url=http://ero0321.blogspot.tw/2014/09/blog-post_3.html|title=綠島‧靈異的傳說 - 鬼洞(燕子洞)裡的黃金與百位先人遺骨|accessdate=2016-12-29|work=ero0321.blogspot.tw}}</ref>

== 十三中隊 ==
{{Seealso|綠島人權文化園區#十三中隊}}
'''十三中隊'''之遺址位於慈航宮旁前往燕子洞之路徑上，為新生訓導處埋葬政治受難者遺體之主要地點。<ref>{{Cite web|url=http://mmweb.tw/72952/n|title=燕子洞|accessdate=2016-12-29|work=mmweb.tw}}</ref>

== 燕子洞 ==
'''燕子洞'''是燕子湖地區內的一處[[海蝕洞|海蝕洞]]，當地居民傳言該地曾為[[台灣日治時期|日治時期]]之行刑場所及歷經多個時代的墳場<ref>{{Cite news|url=http://www.appledaily.com.tw/appledaily/article/supplement/20100729/32695610/|title=台東 綠島 探海蝕洞 訪昔日刑場|last=|first=|date=|work=|newspaper=蘋果日報|accessdate=2016-12-29|language=zh-TW}}</ref>；近年來，當地已陸續發掘出許多骨骸碎片<ref>{{Cite journal|title=綠島行|author=林秀嫚|url=http://beta.nmp.gov.tw/enews/no276/page_03.html|journal=史前館電子報|issue=276|doi=|others=|year=2014|volume=|page=|pmid=}}</ref>。<ref>{{Cite news|url=http://taichung2050.pixnet.net/blog/post/5197960-%E7%B6%A0%E5%B3%B6%E7%99%BE%E4%BD%8D%E5%8F%A4%E4%BA%BA%E7%8F%BE%E8%BA%AB%E7%B5%82%E7%B5%90%E9%BB%83%E9%87%91%E5%A4%A2|title=綠島百位古人現身終結黃金夢 @ 台中學研究中心 :: 痞客邦 PIXNET ::|last=(taichung2050)|first=台中學研究進化版|newspaper=台中學研究中心|accessdate=2016-12-29|language=zh-TW}}</ref>
