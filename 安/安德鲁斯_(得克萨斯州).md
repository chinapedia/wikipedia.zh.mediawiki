'''安德鲁斯'''（{{lang en|'''Andrews'''}}）位于[[美国|美国]][[得克萨斯州|得克萨斯州]]西部，是[[安德鲁斯县_(得克萨斯州)|安德鲁斯县]]的县治所在{{GR|6}}，面积12.4平方公里。根据[[美国2000年人口普查|美国2000年人口普查]]，共有9,652人，其中[[白人|白人]]占75.65%、[[非裔美国人|非裔美国人]]占2.04%。{{GR|2}}這個數字於2010年上升至11,088人。<ref>[http://factfinder2.census.gov/ 2010 census factfinder2]</ref>安德魯斯縣退伍軍人紀念館位於本城內。<ref name="andrewscountyveteransmemorial.com">Andrews County Veterans Memorial  {{cite web |url=http://www.andrewscountyveteransmemorial.com/index.htm |title=存档副本 |accessdate=2008-01-13 |deadurl=yes |archiveurl=https://archive.is/20070801130913/http://www.andrewscountyveteransmemorial.com/index.htm |archivedate=2007-08-01 }} Accessed 1-13-2008</ref>美國職業高爾夫球選手[[查德·坎貝爾|查德·坎貝爾]]生於本城。<ref>{{cite web
