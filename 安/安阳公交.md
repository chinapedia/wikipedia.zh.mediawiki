本条目介绍[[河南省|河南省]][[安阳市|安阳市]]市区的[[公交|公交]]线路。

== 公交公司 ==

安阳市区公交系统由两家公司及其下属分公司运营：
*安阳公交总公司
*安阳扬子巴士有限公司

== 票价 ==

[[无人售票|无人售票]]票价一元，部分总里程较长线路分段收费。

== 运营线路 ==

截至2015年1月，安阳市区有公交线路48条。
以[[安阳站|安阳火车站]]、公交东站、公交西站、公交车站、[[安阳工学院|安阳工学院]]为始发及终点站的线路较多。

== 运营车辆 ==

安阳公交系统截至2014年初，共拥有运营车辆647台，其中空调车辆60台。车辆品牌较为多样，以[[宇通客车|宇通客车]]为主，另有[[黄海客车|黄海]]、[[扬子客车|扬子]]、[[恒通客车|恒通]]、[[中通客车|中通]]、[[东风客车|东风]]、[[少林客车|少林]]、[[友谊客车|友谊]]等品牌客车。运营车辆以单层客车为主，有10辆双层巴士于2014年10月中旬投入运营。在此之前安阳公交也曾拥有过少量“亚星”双层巴士，但早已全数退役。
