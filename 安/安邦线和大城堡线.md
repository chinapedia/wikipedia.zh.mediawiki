'''安邦线/大城堡线'''（{{lang-en|'''Ampang Line/Sri Petaling Line'''}}），是[[巴生谷|巴生谷]]第三和四条列车线路，也是马来西亚首条轻快铁线。这条路线属于[[陆路公共交通委员会|陆路公共交通委员会]]（SPAD）的大吉隆坡/[[巴生谷综合运输系统|巴生谷综合运输系统]]。在路線編碼中，通往[[安邦站|安邦]]為{{rint|my|3}}號線，代表色是[[橘色|橘色]]，通往[[大城堡站|大城堡]]則是{{rint|my|4}}號線，代表色是[[褐色|褐色]]。

安邦綫和大城堡线前稱「实达线」（STAR LRT），並由Sistem Transit Aliran Ringan（簡稱STAR）的公司管理。現今所有的輕快鐵則由馬來西亞國家基建公司（Prasarana）旗下的[[快捷通轨道|快捷通轨道]]公司管理。此輕快鐵系統共長45.1[[公里|公里]]，並是马来西亚的第一条[[标准轨|标准轨]]线路。

安邦線與大城堡線的部分路軌，是沿用在20世紀初，雪蘭莪鐵道所經營之鐵路，該路線由吉隆坡市中心的蘇丹街其自[[陳秀連站|陳秀連站]]一帶，接著分岔經馬魯里一帶至[[安邦|安邦]]，另一條則經[[沙叻秀|沙叻秀]]至[[加影|加影]]，形成今日輕快鐵路線之雛形。<ref>[蘇丹街火車站 http://www.sinchew.com.my/node/1212030]</ref>

== 历史 ==
[[马来西亚|马来西亚]]自1980年代起突飞猛进的经济发展，使[[首都|首都]][[吉隆坡|吉隆坡]]的[[机动车|机动车]]数量和道路交通流量不断增加；加上[[马来西亚|马来西亚]]政府采取扶持国产汽车工业的政策，使[[马来西亚|马来西亚]]成为国民汽车拥有率最高的[[东南亚国家|东南亚国家]]。根据[[马来西亚|马来西亚]]政府在1980年代初进行的[[吉隆坡|吉隆坡]]总体交通规划，预测到2000年首都私人汽车数量将比1980年增加2.8倍，当局有必要改变交通政策和提高[[公共交通|公共交通]]的比重，否则[[吉隆坡|吉隆坡]]未来不得不面对严重的[[交通堵塞|交通堵塞]]，因此该规划报告建议在[[吉隆坡|吉隆坡]]市中心形成以[[城市轨道交通|城市轨道交通]]为主、[[公共汽车|公共汽车]]为辅的交通运输系统。

1990年，由[[马来西亚|马来西亚]]和外国公司组成的联合体完成吉隆坡轻轨系统（LRT）的可行性研究，建议借鉴[[香港|香港]]和[[新加坡|新加坡]]兴建地铁的经验，利用以往市區通往[[安邦|安邦]]的廢棄鐵路路線，优先興建从市中心通往东南方向、连接[[安邦|安邦]]和[[冼都|冼都]]的轻轨铁路。1992年11月，轻轨运输系统私人有限公司（Sistem Transit Aliran Ringan Sdn Bhd）由9家公司合股成立，负责这条轻轨铁路的建设和经营，因此这条路线亦得名为“'''实达轻轨'''”（'''STAR LRT'''）或“'''实达线'''”（'''STAR Line'''）。

“STAR”第一期工程全长12.1公里，由[[安邦站|安邦站]]至[[蘇丹依斯邁站|蘇丹依斯邁站]]，于1993年8月完成融资协议并正式动工，至1996年12月建成通车。该线是[[马来西亚|马来西亚]]第一条轻轨铁路，轨道采用1,435毫米标准轨距，牵引供电制式为750伏特直流[[第三轨供电|第三轨供电]]；由[[德国|德国]][[AEG|AEG公司]]（后来为[[Adtranz|Adtranz]]）和[[英国|英国]]{{tsl|en|Taylor Woodrow|泰勒·伍德罗公司}}分别作为机电和土木工程承建商；投资模式为[[BOO模式|BOO模式]]（建设－拥有－经营）及土地补偿，以配合当时马来西亚政府的[[公用事业|公用事业]]民营化政策。“STAR”二期工程包括北延线（[[苏丹依斯迈站|苏丹依斯迈站]]至[[冼都东站|冼都东站]]）和南延线（[[陈秀连站|陈秀连站]]至[[大城堡站|大城堡站]]），两条延线均于1998年建成通车。

然而，实达线和后来的布达拉线建成通车之后，两家轻轨铁路营运商都面临严重的财政困难，主要原因是基础建设和车辆购置费用等固定投资的债务偿还压力，而且客流量水平亦远低于建设规划时的估计。由于[[吉隆坡|吉隆坡]]的私人汽车拥有比率非常之高，加上政府缺乏对泊车设施和接驳交通等配套的规划，使许多[[吉隆坡|吉隆坡]]市民仍然习惯使用私人汽车出行，阻碍了城市轨道交通系统使用率的提高。根据安邦线原本的建设规划，最初预计到2003年的日均客流量将达到50万人次，但实际客流量仅达到11万人次。

2001年11月，马来西亚联邦政府不得不负起最后承担者的角色，接管了两家频临破产危机的轻轨铁路营运公司，并由企业债务重组委员会（CDRC）负责对其进行{{tsl|en|Debt restructuring|债务重组}}。随后，为了加强整合吉隆坡的公共交通系统、提升公共交通的营运效率和服务质素，马来西亚联邦政府于2004年7月注资成立吉隆坡快捷通公司（RapidKL），作为{{tsl|en|Prasarana Malaysia|马来西亚国家基建公司}}（Prasarana）的子公司，自2005年1月1日起成为吉隆坡的公共交通营运商，负责经营巴生河流域的轻轨铁路和巴士服务。

2005年7月19日起，吉隆坡快捷通公司宣布将实达轻轨的两条路线，分别命名为'''安邦線'''（'''Ampang Line'''）及'''大城堡線'''（'''Sri Petaling Line'''），所有车站标识于2006年完成更名。2007年4月，吉隆坡快捷通公司又再决定将這兩條路線合併及统一命名為'''安邦線'''（Ampang Line），不再對兩條支線分别赋予不同名称。2016年7月，吉隆坡快捷通公司又再决定将这两条路线拆分为独立的线路，而中间共相连11站。

== 路線 ==
安邦線屬於中運量系統，全線皆為地上路線，部分路線直接架設在[[巴生河|巴生河]]上方，全長為27[[公里|公里]]，设有25个车站，包括12个高架车站、13个地面车站。安邦線由兩條支線組成，其營運模式為“安邦站-冼都東站”（安邦线）與“布特拉高原站-冼都東站”（大城堡线），兩線於[[陳秀連站|陳秀連站]]匯合，繼續駛往終點站[[冼都東站|冼都東站]]。1996年12月16日,[[安邦站|安邦站]]至[[苏丹依斯迈站|苏丹依斯迈站]]之間的線段通車，為[[馬來西亞|馬來西亞]]最早營運的[[輕快鐵|輕快鐵]]路線。為了迎接1998年舉辦的[[英聯邦運動會|英聯邦運動會]]而建設'''大城堡支線'''，本路線從[[陳秀連站|陳秀連站]]延伸至[[大城堡站|大城堡站]]，該次[[英聯邦運動會|英聯邦運動會]]的比賽場合主要集中於[[武吉加里爾站|武吉加里爾站]]附近的體育園區。1998年12月6日，[[苏丹依斯迈站|苏丹依斯迈站]]至[[冼都東站|冼都東站]]之間的路線全線通車。

[[File:Bandaraya_station_(Star_Line)_(exterior),_Kuala_Lumpur_(January_2007).jpg|thumb]]是安邦線上的一個典型的高架車站]]

[[人民廣場站_(吉隆坡)|人民广场站]]至[[蒂蒂旺沙站|蒂蒂旺沙站]]的線路是沿著[[鵝嘜河|鵝嘜河]]與[[巴生河|巴生河]]而建，從[[人民廣場站_(吉隆坡)|人民广场站]]到[[冼都東站|冼都東站]]之間皆為高架路線。[[陳秀連站|陳秀連站]]以東至[[安邦站|安邦站]]皆為地面路線，其他的路線則是高架與路面並存，全線皆為地上路線。

安邦和大城堡線总共設有38站，[[陳秀連站|陳秀連站]]-[[冼都東站|冼都東站]]之間有11站，安邦站-陳秀連站之間有7站，布特拉高原站-陳秀連站之間有20站。另外，在[[安邦站|安邦站]]與[[丽阳站_(安邦)|丽阳站]]之間及[[大城堡站|大城堡站]]設有兩個轻轨车辆段，提供列車存放於維修服務。

馬來西亞的軌道運輸系統多為[[軌距|米軌]]，而安邦線採用的卻是[[標準軌|標準軌]]，這是[[馬來西亞|馬來西亞]]的第一條標準軌線路。

==安邦線延長計劃==
隨著吉隆坡人口的外移，位於吉隆坡西南部的蒲種人口迅速增長，當地的交通幹道經常面臨嚴重塞車問題。2011年5月26日，[[馬來西亞|馬來西亞]]政府宣布安邦線延長計劃。在這計劃下，安邦線將延長17.7公里，延長路線從[[大城堡站|大城堡站]]繼續往南延伸11站（另有2站等至周圍的人口增長後，才會啟用）至[[布特拉高原站|布特拉高原站]]，與[[格拉那再也線|格拉那再也線]]的延長線交會，第一階段將於2015年10月通車，第二階段將於2016年3月通車，第三階段與[[格拉那再也線|格拉那再也線]]的延長線已於2016年6月30日全線通車<ref>{{cite web|url=http://www.nanyang.com/node/602836|title=安邦延长线首阶段增4 站 轻快铁明年10月到蒲种|publisher=|deadurl=yes|archiveurl=https://web.archive.org/web/20141227083638/http://www.nanyang.com/node/602836|archivedate=2014-12-27}}</ref>。

==延長線爭議及工程意外頻發==
安邦延長線工程計劃公佈後，開始爆發許多與政黨議題相關的爭議，其興建工程的承包商多為無鐵路興建經驗及與政府有密切關係的土著集團，被人質疑是政府圖利特定集團及土著企業<ref>{{cite web|url=http://www.orientaldaily.com.my/index.php?option=com_k2&view=item&id=18493:&Itemid=113|title=乔治肯特標获　轻快铁延长工程}}</ref>。在路線規劃上，也遭到部分沿線居民的抗議，指出鐵路路線與民宅或學校的距離過近，希望能更改路線。在興建過程中，也經常發生各式工程意外，因延長路線多為建在高速公路上的高架鐵路，大部分意外多為建材墜落，壓毀下方行經的車輛，被市民懷疑工程當局沒有做好工程監督，才會導致意外頻發<ref>{{cite web|url=http://www.orientaldaily.com.my/index.php?option=com_k2&view=item&id=155563:&Itemid=242|title=高空工程多意外　驾驶人出入难安|work=東方網 - 馬來西亞東方日報}}</ref>。在2014年12月24日，臨近布特拉高原的工程路段中，發生了支撑鹰架倒塌事故，造成2员工被活埋丧命、4人受伤及2人自行脱困，全部死傷者皆為[[孟加拉|孟加拉]]籍外勞，該次事件也是2014年內，[[馬來西亞|馬來西亞]]第4宗轻快铁或捷运工程內發生的意外事故<ref>{{cite web|url=http://www.orientaldaily.com.my/index.php?option=com_k2&view=item&id=155408:2&Itemid=114|title=轻快铁工程又夺2命|work=東方網 - 馬來西亞東方日報}}</ref>。

== 接驳的其他轨道系统 ==
[[File:Inside_hang_tuah_station.jpg|thumb]]
2012年前因各路線屬於不同的公司營運，因此各路線的票券亦不相通，需要出站轉車。近年來，[[吉隆坡快捷通|吉隆坡快捷通]]統一管理目前的[[輕快鐵|輕快鐵]]與[[單軌|單軌]]路線後，推出了新一代的投幣式車票後，並開始將旗下的路線的票券整合起來，目前乘客可於[[漢都亞站|漢都亞站]]、[[占美清真寺站|占美清真寺站]]與[[蒂蒂旺沙站|蒂蒂旺沙站]]直接轉乘至其他的路線，不需另外出站購票。於[[吉隆坡中央車站|吉隆坡中央車站]]交會的[[吉隆坡單軌|吉隆坡單軌]]與[[格拉那再也線|格拉那再也線]]因月台之間有一段距離，需要進行站外轉乘。

[[芙蓉線|芙蓉線]]與[[巴生港線|巴生港線]]因屬於[[馬來亞鐵道|馬來亞鐵道]]管理，所以仍不能與[[吉隆坡快捷通|吉隆坡快捷通]]旗下的路線進行站內轉車。目前兩家公司的月台與票券的整合工作正在進行中。

== 使用车辆 ==
[[File:Ampang_LRT_train_for_Ampang_Line_&_Sri_Petaling_Line.png|缩略图]]
安邦线自1996年起使用30组由[[Adtranz|Adtranz]]设计及授权[[澳大利亚|澳大利亚]]{{tsl|en|Walkers Limited|沃尔克斯公司 (澳大利亚)|沃尔克斯公司}}制造的轻轨列车，列车部分设计借鉴自澳大利亚为[[香港輕鐵|香港輕鐵]]制造的[[輕鐵第一期列車|第一代輕鐵車輛]]。轻轨列车以6辆编组方式运行，每两节车厢组成一个动力单元。车辆分为带有司机室的控制动车（Mc）和无司机室的中间动车（M）两种形式。每个单元设有三个转向架，包括两个动力[[转向架|转向架]]和一个中间[[铰接式转向架|铰接式转向架]]。轻轨车辆采用[[不锈钢|不锈钢]]车体，每节车厢设有三对双页外挂滑动式车门。每列6辆编组列车的座席数量为168个，整列列车最大载客量为1200人<ref>{{Cite web |url=http://www.myrapid.com.my/corporate/our-business/our-fleet/ampang-line |title=AMPANG LINE (2 coaches of 6 cars train) |accessdate=2014-10-23 |publisher=Prasarana Malaysia Berhad |date= |deadurl=yes |archiveurl=https://web.archive.org/web/20141009124507/http://www.myrapid.com.my/corporate/our-business/our-fleet/ampang-line |archivedate=2014-10-09 }}</ref>。

马来西亚国家基建公司为了配合即将完成的安邦延长线工程，于2012年4月向[[中国|中国]][[南车株洲电力机车|南车株洲电力机车有限公司]]采购20列（120辆）轻轨列车，合约总值约3亿7000万令吉；列车以南车株机公司为[[土耳其|土耳其]][[伊兹密尔地铁|伊兹密尔地铁]]提供的轻轨车辆为基础，其中6组列车在马来西亚本土的南车马来西亚轨道交通装备制造基地进行组装<ref>{{Cite web|url=http://www.csrgc.com.cn/g782/s7710/t57745.aspx|title=中国南车株机公司中标马来西亚吉隆坡 “安邦线”轻轨列车|accessdate=2014-10-23|publisher=中国南车股份有限公司|date=2012-04-09|deadurl=yes|archiveurl=https://web.archive.org/web/20141023131039/http://www.csrgc.com.cn/g782/s7710/t57745.aspx|archivedate=2014-10-23}}</ref>。2014年10月，马来西亚国家基建公司再度向南车株机公司采购30列（180辆）轻轨列车，合约总值约6亿2000万令吉，这批列车将用于替换已经老化的第一代轻轨车辆，其中9组列车在马来西亚本土的南车基地实现本地化生产；国家基建公司与南车株机公司是以[[折价贴换|折价贴换]]方式合作，国家基建公司从中国南车增购30组新列车的同时，南车株机公司则负责回收及处理安邦线的30组旧列车<ref>{{Cite web |url=http://www.csrgc.com.cn/g3479/s6895/t251447.aspx |title=中国南车再获马来西亚30列地铁车辆订单 |accessdate=2014-10-23 |publisher=中国南车股份有限公司 |date=2014-10-21 |deadurl=yes |archiveurl=https://web.archive.org/web/20141023132206/http://www.csrgc.com.cn/g3479/s6895/t251447.aspx |archivedate=2014-10-23 }}</ref><ref>{{Cite web |url=http://www.nanyang.com/node/657240?tid=460 |title=国家基建与中国南车签6亿合约 安邦轻快铁增30组列车 |accessdate=2014-10-23 |publisher=南洋网 |date=2014-10-23 |deadurl=yes |archiveurl=https://web.archive.org/web/20141023092737/http://www.nanyang.com/node/657240?tid=460 |archivedate=2014-10-23 }}</ref>。

自2016年12月1日起，其餘的[[中國|中國]][[中车株洲电力机车|南車株洲電力機車有限公司]]（今[[中國|中國]][[中车株洲电力机车|中車株洲電力機車有限公司]]）製造的“AMY”列車將正式投入服務。與此同時，已服務將近20年的老舊[[Adtranz|Adtranz]]列車將完成其歷史任務，正式退役。這也意味著馬來西亞史上第一代輕軌列車型將畫下完美的句點。
