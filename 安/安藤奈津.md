'''《安藤奈津》'''（''{{lang-ja|あんどーなつ}}''）是在2008年7月7日到9月22日間，於[[東京放送|TBS]]在每週一20:00－20:54（[[日本標準時間|日本標準時間]]）播放，由[[貫地谷詩穗梨|貫地谷詩穗梨]]及[[國村隼|國村隼]]主演的[[電視劇|電視劇]]，共播出12集<ref>http://www.tbs.co.jp/ando-natsu/</ref>。

電視劇是改編自由[[小學館|小學館]]出版、[[西祐治|西祐治]]原作、[[Terry山本|Terry山本]]畫的同名漫畫。

== 概要 ==
劇情主要是描述在[[東京|東京]][[淺草|淺草]]一間歷史悠久的[[和菓子|和菓子]]店工作的女主角，以專業日式點心師傅為目標而成長的故事，以及在她周圍所發生的各式充滿人情味的小故事。

每集除了會介紹各式的和菓子外，也介紹了許多與和菓子相關的[[俳句|俳句]]及[[落語|落語]]。

== 劇情 ==
想成為西式點心師傅的'''安藤奈津'''（[[貫地谷栞|貫地谷栞]]飾演），原本工作的糕餅店因為老闆娘去世而結束營業，她只好前往另一間點心企業面試，但並沒有獲得錄取，在回程途中她陰錯陽差遇見在「滿月堂」工作的和菓子師傅－'''梅吉'''（國村隼飾演），她被對方專注製作和菓子的態度所吸引，於是拜對方為師……。

== 主要演員、角色簡介 ==
* '''安藤奈津'''：[[貫地谷栞|貫地谷栞]]
*: 故事的主人公，[[福井縣|福井縣]]出身，畢業於專門西式點心學校，以作為一個西點師傅，製作讓人溫馨的點心作為人生的目標，然而卻陰差陽錯地走進“滿月堂”中拜了安田梅吉為師，步上作為一個和菓子職人的漫長旅途。雖然父母雙亡，仕途失利，卻總能保持著明亮堅強的性格面對人生。

* '''安田梅吉'''：國村隼
*: 百年老店「滿月堂」的當家師傅，平時被人愛稱為“梅”，有著三十年的和菓子製作經驗。平時看似不茍言笑，但其實是個細膩的人。

* '''丸岡竹藏'''：尾美利德
*: 「滿月堂」的師傅，梅吉的弟子，奈津的前輩。平時被人愛稱為“竹”。原本是為了繼承老家的和菓子店，十八岁时被送到「滿月堂」學習，也有十五年以上的在職經驗了，目前還在「滿月堂」修業中。

* '''月岡光子'''：{{lang|ja|風吹ジュン}}
*: 「滿月堂」的老闆娘，平時總是一身和服的打扮，丈夫過世之後便獨力經營滿月堂。
