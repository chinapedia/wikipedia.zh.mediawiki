'''安德魯·「邦尼」·黃'''（{{lang-en|Andrew "bunnie" Huang}}；{{bd|1975年||}}）是一名美國[[駭客|駭客]]，擁有[[麻省理工學院|麻省理工學院]][[電子工程|電子工程]][[博士|博士]]學位，是2003年出版的「Xbox:逆向工程入門」<ref>{{cite web | url=http://nostarch.com/xboxfree | title=An open letter from bunnie, author of Hacking the Xbox | date=March 2013}}</ref>的作者。他也是 Zeta Beta Tau 兄弟會的會員。從2012年起定居在新加坡。<ref>{{cite web | url=http://dangerousprototypes.com/2012/08/23/workshop-video-36-beers-in-bunnies-workshop/ | title=Workshop Video #36: Beers in Bunnie’s Workshop | date=2012-08-23}}</ref>

==計劃==
安德魯·黃是 [[Chumby|Chumby]] 硬體方面的要角，他負責設計並生產 chumby 裝置，以及更寬的 chumby 硬體平台的策略性規劃和生態系統的發展。他已經完成的主要計劃，範圍從修改 Xbox，到跟 [[Luxtera,_Inc|Luxtera, Inc]]一起設計世界第一個以10 Gbit/s速度運作、完全整合的[[矽光子|矽光子]][[晶片|晶片]]，以及跟[[加州理工學院|加州理工學院]]（Caltech）合作建造一些供矽[[奈米線|奈米線]]裝置研究使用、最早的硬體原型。他也參與了以下的設計計劃：跟 [[Mobilian|Mobilian]] 合作設計使用在[[802.11b|802.11b]]和[[藍芽|藍芽]]上的無線收發器、在[[Qualcomm|Qualcomm]]設計了[[數位電影|數位電影]]的[[編解碼器|編解碼器]]，以及在1999年由國際無人載具系統協會（[[AUVSI|AUVSI]]）所舉辦、由[[麻省理工學院|麻省理工學院]]團隊獲勝的自主式機器潛水艇設計。他偏好受挑戰去窺究以矽為基礎的硬體，而同時負責了許多安全系統的「非傳統設計」（un-design，在設計中結合科學、工程和生物科技）部份。 [[邦尼|邦尼]]是[[Make_(雜誌)|Make (雜誌)]]的投稿作者，也是他們技術顧問群的成員。他同時也寫了不少關於在中國製造的文章。<ref>{{cite web|url=http://boingboing.net/2007/07/13/bunnie_huangs_blogse.html |title=Bunnie Huang's blog-series on Chinese manufacturing |date=March 2013 |deadurl=yes |archiveurl=https://web.archive.org/web/20100529104051/http://www.boingboing.net/2007/07/13/bunnie_huangs_blogse.html |archivedate=2010-05-29 }}</ref>

邦尼也使用了逆向工程技術說明某些[[MicroSD|MicroSD]]品質低劣的原因。<ref>{{cite web | url=http://boingboing.net/2010/02/16/sleuthing-uncovers-t.html | title=Sleuthing uncovers the mystery of Kingston MicroSD cards' crappy QA | date=2010-02-16}}</ref>2013年，他與Sean 'xobs' Cross合作提出經由植入到內嵌控制器的後門來載入任意程式碼的方法。<ref>{{cite web | url=http://www.youtube.com/watch?v=CPEzLNh5YIo | title=The Exploration and Exploitation of an SD Memory Card | date=2013-12-29}}</ref>

邦尼曾被排訂以專家證人的身分出席聯邦政府控告Crippen的審判庭，來決定修改Xbox是否違反[[數位千禧年著作權法|數位千禧年著作權法]]。<ref>{{cite web | url=http://www.wired.com/threatlevel/2010/10/xbox-modder-tria/ | title=Threat Level Privacy, Crime and Security Online Previous post Next post Prosecutors Seek to Block Xbox Hacking Pioneer From Trial | date=2010-10-21}}</ref>此案在開庭的第三天突然在陪審團和代表聯邦的控方前被撤銷<ref>{{cite web | url=http://www.wired.com/threatlevel/2010/12/crippen-dismissed/ | title=Prosecutors Dismiss Xbox-Modding Case Mid-Trial | date=2010-10-02}}</ref>，在邦尼出庭作證前，此案即被駁回。<ref>{{cite web | url=http://www.theregister.co.uk/2011/09/16/chumby_nettv_hack/ | title=How gizmo maker's hack outflanked copyright trolls | date=March 2013}}</ref>

他也設計了NeTV，這是所知第一個公開使用的[[HDCP|HDCP]]（高畫質數位內容保護）「萬能鑰匙」。視訊設備使用這個萬能鑰匙在現有受保護的HDCP連結上，用一個據稱不會違反DMCA法律的方式來執行一個視訊覆蓋（Video Overlay）。NeTV的硬體和軔體可以[[CC-BY-SA|CC-BY-SA]]的授權方式公開取得。<ref>{{cite web | url=http://kosagi.com/w/index.php?title=NeTV_Main_Page | title=Source code}}</ref>

他也以一個志工的身份設計了[[開源硬體|開源硬體]]「Safecast蓋革計數器」來回應[[2011年日本東北地方太平洋近海地震|2011年日本東北地方太平洋近海地震]]，以及接著發生的[[福島第一核電廠事故|福島第一核電廠事故]]。<ref>{{cite web | url=http://boingboing.net/2012/03/15/bunnie-huangs-open-geiger-co.html | title=Bunnie Huang's open Geiger counter: design notes and reference | date=2012-03-15}}</ref><ref>{{cite web | url=http://geigercounter.com/soul-geiger-counter/ | title=Inside the Heart and Soul of a new Geiger counter | date=2012-09-27}}</ref>

2012年2月，邦尼接受[[The_Amp_Hour|The Amp Hour]]節目第84集的專訪，他談及他的電子作品在中國的情形，以及逆向工程。<ref>{{cite web | url=http://www.theamphour.com/the-amp-hour-84-bunnies-bibelot-bonification/ | title=The Amp Hour #84 - An Interview with Bunnie Huang - Bunnie's Bibelot Bonification | date=2013-12-22}}</ref>

2013年，邦尼宣布他與Sean 'xobs' Cross攜手合作致力於名為[[Novena_(computing_platform)|Novena (computing platform)]]的[[筆記型電腦|筆記型電腦]]的開發。這部[[筆記型電腦|筆記型電腦]]被期待是一個首見的筆電種類。Novena的硬體和軟體全部是開放的，而且只包含製造廠商在無需保密協訂下而能取得所需設計的零件。<ref>{{cite web|url=http://makezine.com/magazine/make-38-cameras-and-av/building-an-open-source-laptop/ |title=Building an Open Source Laptop |date=2014-01-08 |deadurl=yes |archiveurl=https://web.archive.org/web/20140316080440/http://makezine.com/magazine/make-38-cameras-and-av/building-an-open-source-laptop/ |archivedate=2014-03-16 }}</ref>除了一般[[筆記型電腦|筆記型電腦]]的零件外，Novena的主機板還包含了一個[[現場可程式邏輯門陣列|現場可程式邏輯門陣列]]、雙乙太網路埠、三軸[[加速規|加速規]]，以及容易擴充的硬體。<ref>{{cite web | url=http://www.kosagi.com/w/index.php?title=Novena_Main_Page | title=Novena Main Page | date=2014-03-14}}</ref>2014年5月7日，Novena的[[群眾募資|群眾募資]]達到預期目標的250,000美元，在不計入隨後的預接訂單下，朝722,880美元的總金額繼續籌募。
邦尼目前和[[MIT媒體實驗室|MIT媒體實驗室]]博士生Jie Qi合作「電路貼紙」（註冊名稱為chibitronics<ref>{{cite web | url=http://chibitronics.com/ | title=Crafting with electricity | date=June 2013}}</ref>）計劃，這是一種用來手作電子產品、可撕或貼的電子材料。<ref>{{cite web | url=http://www.crowdsupply.com/chibitronics/circuit-stickers | title=Circuit Stickers | date=December 2013}}</ref>

邦尼將參與2015年3月13-15日在新加坡舉辦的[[FOSSASIA|FOSSASIA]] 2015峰會，他受邀擔任峰會主講人之一。<ref>{{cite web | url=http://fossasia.org/#speakers | title=FOSSASIA Singapore 2015 Keynote Speakers}}</ref>
