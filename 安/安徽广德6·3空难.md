'''安徽广德“6·3”空难'''發生在2006年6月3日，一架隸屬[[中國人民解放軍空軍|中國人民解放軍空軍]]<ref name="xinhuanet_2006Sept07">{{Cite web |url = http://news.xinhuanet.com/politics/2006-09/07/content_5061228.htm |title=中央军委通报:近期两起特大事故相关责任人受到严肃处理|accessdate=2011-04-03|work=新华网|date=2006-09-07|language=zh-hans}}</ref>的[[空警-200|空警-200]]預警機在执行任务途時<ref name="xinhuanet_2006Jun04">{{Cite web |url = http://news.xinhuanet.com/mil/2006-06/04/content_4642641.htm |title=中国一架军用运输机失事 |accessdate=2011-04-03 |website =新华网 |date=2006-06-04|language=zh-hans}}</ref>，在[[安徽省|安徽省]][[廣德縣|廣德縣]]柏垫镇姚村上空失事墜毀<ref name="gdly_2008Aug11">{{Cite web|url=http://www.gdly.org.cn/lyzh/HTML/726.html|title=我县建成6·3空难烈士纪念碑|accessdate=2011-04-03|author=今日广德|work=广德旅游|publisher=广德县旅游局|date=2008-08-11|language=zh-hans|deadurl=yes|archiveurl=https://archive.is/20080930142119/http://www.gdly.org.cn/lyzh/HTML/726.html|archivedate=2008-09-30}}</ref>，機上包括機組人員在內的40人全部罹難<ref name="xinhuanet_2006Sept07" />。這是[[中國人民解放軍歷史|中國人民解放軍歷史]]上最嚴重的空難，亦是中國空軍2006年發生的四起空難中的第二起<ref name="stnn">{{Cite web |url = http://www.stnn.cc/weekly/weekly0633/weekly0633_10/200612/t20061206_413344.html |title=紧急应变措施缺乏 中国军机坠毁事故频发 |accessdate=2011-04-03 |work=星岛网刊 |language=zh-hans}}</ref>。

2006年9月7日，中國官方通訊社[[新華社|新華社]]公佈了調查結果，認定空難的直接原因是飛機多次穿越结冰区域，造成飞机空中结冰並导致失控坠毁<ref name="xinhuanet_2006Sept07" />。

在最初的官方新聞中，失事飛機的機型和失事地點的準確位置並沒有透露，僅使用「軍用運輸機」和「安徽某地」等籠統的表述帶過<ref name="xinhuanet_2006Jun04" />。隨著事件的發展和報道的深入，肇事飛機的機型被證實是當時正在研發的、由[[運-8|運-8]]改裝的<ref name="yzzk_2006Jul02">{{Cite web |url = http://www.yzzk.com/cfm/Content_Archive.cfm?Channel=tl&Path=236390221/26tl1.cfm|title=中國預警機撞山墜毀案中案 |accessdate=2011-04-03 |author=林一民 |work=亞洲週刊 |date=2006-07-02 |language=zh-hk}}</ref>空警-200預警機，失事地點為安徽廣德<ref name="gdly_2008Aug11"/>。

== 事件經過 ==
飛機在當日約13時30分從安徽[[肥東軍用機場|肥東軍用機場]]起飛，約16時在安徽廣德上空失事墜毀。<ref name="yzzk_2006Jul02"/>事發當時有目擊者看到兩架軍機一前一后，低飛進入縣城上空，並發出異常的聲音。隨後，其中的一架飛機突然朝下墜落，之後飛機又突然之間猛然上升朝山上飛去。飛機墜毀前曾發生爆炸，墜毀後飛機斷成兩截並再次發生爆炸。失事現場被砸出一個大坑，同時發現5具完整的屍體。<ref name="stnn"/>

空難發生後，廣德縣縣委、縣政府和縣人武部共組織了5600人次和部隊官兵對空難現場進行搜索。<ref name="guangdemz_2011Sept26">{{Cite web|url=http://www.guangdemz.gov.cn/arts.asp?id=872|title=双拥花开别样红|accessdate=2011-12-25|author=王志兵|work=广德县民政局|date=2011-09-26|language=zh-hans|deadurl=yes|archiveurl=https://web.archive.org/web/20120119094858/http://www.guangdemz.gov.cn/arts.asp?id=872|archivedate=2012-01-19}}</ref>
=== 遇難者 ===
機上40人中除5人為試飛員外，其他人員均為電子專家。<ref name="stnn"/>其中軍方和地方技術人員各佔一半，大部分人員來自安徽[[國防科工委|國防科工委]]「第三十八研究所」，機上還包括幾位將軍級高級工程師。<ref name="yzzk_2006Jul02"/>

== 事件調查 ==
空難發生後，[[中央军委|中央军委]]派遣了总部工作组连夜赶赴現場，组织搜救和善後工作。<ref name="xinhuanet_2006Jun04"/>該工作組由中共中央政治局委员、中央军委副主席[[郭伯雄|郭伯雄]]率領<ref name="xinhuanet_2006Jun04_mourning">{{Cite web|url=http://news.xinhuanet.com/newscenter/2006-06/04/content_4645035.htm|title=胡锦涛发唁电哀悼我失事军用运输机机上遇难人员|accessdate=2011-04-04|work=新华网|date=2006-06-04|language=zh-hans}}</ref>。

空難調查發現，除直接原因結冰之外，還有多個原因導致空難發生：
#機翼除冰系統出現故障<ref name="yzzk_2006Jul02"/>；
#飛機超載（因當時飛機正在進行測試，裝有大量設備，機上不應搭載超過30人）<ref name="yzzk_2006Jul02"/>；
#機上大部分乘客沒有綁好安全帶<ref name="yzzk_2006Oct29">{{Cite web|url=http://www.yzzk.com/cfm/Content_Archive.cfm?Channel=kk&Path=2217536212/43kk1.cfm|title=高級預警機毀於低級錯誤|accessdate=2011-04-04|author=王健民|work=亞洲週刊|date=2006-10-29|language=zh-hk}}</ref>。
=== 隱瞞案情 ===
空軍有關方面在空難之後刻意將遇難人數數字從40人減少至29人，企圖使事故級別從「重大事故」（30人以上）降低一級，並對飛機的[[飛行紀錄儀|飛行紀錄儀]]（俗稱黑匣子）的資料進行修改，將黑匣子長度為1分03秒資料刪除。事件被揭發後，郭伯雄曾憤怒地說「誰敢造假我槍斃誰。」<ref name="yzzk_2006Jul02"/>

=== 事後處分 ===
空難原因查明後，中央军委對空難相關的責任人進行處分：<ref name="xinhuanet_2006Sept07"/>
*[[南京军区|南京军区]]副司令员兼南京军区空军司令员[[江建曾|江建曾]]：记大过处分
*南京军区副政治委员兼南京军区空军政治委员[[王伟_(中将)|王伟]]：记过处分
*南京军区空军副参谋长[[郭春广|郭春广]]：降职处分
*南京军区空军某团团长张广建：'''撤职处分'''
*空军装备部副部长[[岳留安|岳留安]]：记过处分
還有6人被給予记大过、记过、严重警告等处分。

== 哀悼和紀念 ==
;中央
6月4日，时任中共中央總書記、國家主席、中央軍委主席[[胡錦濤|胡錦濤]]發出唁電，對空難中罹難人員表示深切哀悼，並向遇難者家屬表示慰問。<ref name="xinhuanet_2006Jun04_mourning"/>6月15日，遇難人員的追悼會在江蘇舉行，由空军司令员[[乔清晨|乔清晨]]主持，空军政委[[邓昌友|邓昌友]]致悼词。<ref>{{Cite web|url=http://news.xinhuanet.com/mil/2006-06/16/content_4705105.htm|title=空军运输机遇难者追悼会举行 胡锦涛送花圈 |accessdate=2011-04-04|work=新华网|date=2006-06-16|language=zh-hans}}</ref>軍方據稱在這次追悼會之後，又在北京單獨為遇難的軍方人員舉行了一個不公開的追悼會。<ref name="yzzk_2006Jul02"/>

;地方
2008年，空難發生地安徽廣德縣政府投資了30多萬元人民幣，在該縣[[英烈山|英烈山]]脚建造了'''英烈坊'''牌坊，紀念空難中罹難的人員。牌坊於2008年5月開始設計，同年8月建成，牌坊总高8米，宽10米，正中的兩條立柱刻有對聯，左柱內容為「国家栋梁」，右柱內容為「民族英雄」，横联为「英烈坊」。<ref name="gdly_2008Aug11"/><ref>{{Cite web|url=http://www.xuancheng.gov.cn/include/web_content.php?id=31015|title=广德建6.3空难烈士英烈坊|accessdate=2011-04-03|work=宣城市政府|date=2008-08-12|language=zh-hans}}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>牌坊上做了特殊的設計以紀念該事件：如牌坊6.3米高的外柱和左右兩個各3個、中間6個的斗拱寓意空難發生的日期——6月3日；牌坊4根立柱和立柱中間間距4米則寓意遇難的40名人員。<ref>{{Cite web|url=http://www.guangdemz.gov.cn/arts.asp?id=653|title=广德6.3空难烈士纪念牌|accessdate=2011-12-25|work=广德县民政局|date=2010-04-22|language=zh-hans|deadurl=yes|archiveurl=https://web.archive.org/web/20120119114712/http://www.guangdemz.gov.cn/arts.asp?id=653|archivedate=2012-01-19}}</ref>除此之外，廣德縣和軍方還一同興建了柏梨公路、航空學校和「英烈亭」等纪念设施。<ref name="guangdemz_2011Sept26"/>

== 類似事件 ==
*1988年6月26日，執飛[[法國航空296號班機空難|法國航空296號班機]]的[[空中客车A320|空中客车A320-111型客機]]在載有136名乘客的情況下，在法國一個航空展中表演低空飞越機場時墜毀，空難造成機上3人死亡。事後黑匣子在被[[法國民航局|法國民航局]]違規保存10天之後才被警方收回，但資料疑似遭到纂改而無法讀取。這次事件成為空中客車A320的首次空難。
*1994年10月31日，執行[[美鷹航空4184號班機空難|美鷹航空4184號班機]]的[[ATR-72|ATR 72-212型客機]]在美國[[芝加哥国际机场|芝加哥国际机场]]等待降落时，因机身結冰和飛機設計缺陷，导致飞机失速墜毀，机上包括機組人員在內的68人全部罹难。
