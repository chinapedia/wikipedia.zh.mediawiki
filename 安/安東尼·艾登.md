'''羅伯特·安東尼·艾登'''，'''第一代亞芬伯爵'''，[[嘉德勳章|KG]]，[[軍功十字勳章|MC]]，[[樞密院|PC]]（'''Robert Anthony Eden, 1st Earl of Avon'''，{{bd|1897年|6月12日|1977年|1月14日|catIdx=Eden, Anthony}}），[[英國|英國]][[政治家|政治家]]，[[第二次世界大戰|第二次世界大戰]]時擔任[[外相|外相]]，後來在1950年代出任[[英國首相|英國首相]]。大眾普遍認為，他要對1956年的[[蘇伊士運河危機|蘇伊士運河危機]]負上絕大部分責任。在2004年[https://web.archive.org/web/20060328180554/http://www.mori.com/polls/2004/leeds.shtml]，一所研究機構（MORI）舉行了一場選舉，讓139位學者對20世紀的英國首相進行評分，結果艾登得分最低。同樣地，英國[[BBC|BBC]]電台早前亦進行了類似調查 [http://news.bbc.co.uk/1/hi/uk_politics/575219.stm]，結果[[邱吉爾|邱吉爾]]得票最多，而艾登則得票最少。

== 早年生涯 ==
艾登出生於[[達拉謨郡|達拉謨郡]][[西奧克蘭|西奧克蘭]]（West Auckland），一個非常保守的地主家庭，父親是[[從男爵|從男爵]]，而母親（Sybil Grey）則來自[[諾森伯蘭郡|諾森伯蘭郡]]有名的格雷家族（Grey family）。艾登早年入讀[[伊頓公學|伊頓公學]]，大學則畢業於[[牛津大學|牛津大學]]的[[牛津大学基督堂学院|牛津大学基督堂学院]]。他在大學主修外語，除了能說[[俄語|俄語]]和[[阿拉伯语|阿拉伯语]]外，更精通[[法語|法語]]、[[德語|德語]]和[[波斯語|波斯語]]。艾登在[[第一次世界大戰|第一次世界大戰]]時加入[[英國陸軍|英國陸軍]]，並獲頒贈[[軍功十字勳章|軍功十字勳章]]。他於戰時的1918年5月26日至第198步兵旅擔任{{Link-en|旅參謀長|Brigade major}}職務<ref name="Aster 1976, pp. 5–8">{{cite book|last=Aster|first=Sidney|title=Anthony Eden|publisher=St Martin's Press|location=London|year=1976|isbn=978-0-312-04235-6 |page=5-8}}</ref><ref name="Rhodes James">James, Robert Rhodes. ''Anthony Eden: A Biography'' (1986), detailed scholarly biography</ref>{{rp|p52}}，以21歲之齡成為英國陸軍最年輕的旅參謀長<ref name="Rhodes James"/>{{rp|p55}}。

1919年6月13日，艾登自英國陸軍[[復員|復員]]<ref name="Aster 1976, pp. 5–8"/>，最終階級為[[上尉|上尉]]<ref>{{London Gazette|issue=31479 |supp=y|page=9661|date=29 July 1919}}</ref><ref>{{London Gazette|issue=32034 |supp=y|page=8846|date=27 August 1920}}</ref>。在1923年，他獲選為[[英國保守黨|保守黨]]的[[英國下議院|下議院]]議員，代表[[沃里克和萊明頓_(英國國會選區)|沃里克和萊明頓]]選區，從此投身於政治事業。同年，他迎娶了派翠斯·伯吉，並生了兩名兒子，可惜這段婚姻並不美滿，最後以[[離婚|離婚]]收場，成為他政治生涯上的一個污點。

1926年，艾登在[[英国外交和联邦事务部|外務部]]成為國會私人秘書（Parliamentary Private Secretary），後於1931年獲擢升為外務次官（Under-Secretary for Foreign Affairs）。1934年，他加入了[[斯坦利·鮑德溫|鮑德溫]]的[[內閣|內閣]]，除了獲委任為[[掌璽大臣|掌璽大臣]]，又被指派為[[國際聯盟|國際聯盟]]代表。他與其他經歷過[[第一次世界大戰|第一次世界大戰]]的政客一樣，極力主張反戰，並努力以國際聯盟來維持[[歐洲|歐洲]]和平。可是，他很快發現，對[[納粹德國|納粹德國]]和[[法西斯|法西斯]][[意大利|意大利]]推行[[綏靖政策|綏靖政策]]，無助於推動歐洲和平。結果，於1935年，正當外相[[塞繆爾·霍爾|塞繆爾·霍爾]][[爵士|爵士]]（Sir Samuel Hoare）企圖向意大利行綏靖政策，以平息意大利入侵[[阿比西尼亞|阿比西尼亞]]一事，艾登便私底下提出反對。最後，塞繆爾·霍爾爵士因簽署《[[霍爾-賴伐爾協定|霍爾-賴伐爾協定]]》失敗而辭職，艾登遂接任為外相。

在那時，艾登曾被認為是領導[[潮流|潮流]]的人物。當時的[[绅士|绅士]]階級一般愛戴[[圓頂硬禮帽|圓頂硬禮帽]]，但艾登卻戴[[杭堡帽|杭堡帽]]（即Homburg hat，這種帽有點像圓頂硬禮帽，但帽緣卻稍微上捲，帽冠中央有摺痕，而帽身則飾以緞帶），結果，在[[英國|英國]]成為了新風尚。

艾登有一位[[哥哥|哥哥]]和一位[[弟弟|弟弟]]，分別叫提摩泰和尼古拉。但兩人不幸地在1916年的[[日德蘭海戰|日德蘭海戰]]中陣亡。

== 外務大臣 ==
[[File:FDR_and_Anthony_Eden_at_the_Quebec_Conference.jpg|thumb]]，圖中艾登與[[美國總統|美國總統]][[富蘭克林·德拉諾·羅斯福|羅斯福]]握手。]]
艾登上任外相之時，英國正值外交政策方針的調整期，以對應日益增長的[[法西斯|法西斯]]力量。故此，他支持了對[[西班牙內戰|西班牙內戰]]採取不干預政策。後來[[內維爾·張伯倫|內維爾·張伯倫]]上任首相後，艾登亦支持他對[[納粹德國|納粹德國]]作合理讓步，以換取和平。在1936年，當[[英國|英國]]和[[法國|法國]]阻止納粹德國進佔[[萊茵蘭|萊茵蘭]]失敗後，艾登也同樣沒有提出抗議。但到了1938年2月，艾登終於因為不能再接受張伯倫與[[意大利|意大利]]展開協商而辭職。辭職以後，他變成[[邱吉爾|邱吉爾]]的盟友，成為了與保守黨的綏靖政策大唱反調的後坐議員。當時有很多意見，認為艾登將會帶領其他意見一致的議員，起來反對張伯倫的綏靖政策。艾登雖然的確反對過《[[慕尼黑協定|慕尼黑協定]]》的簽訂，但他實際上卻保持低調作風，以避免黨內出現分裂。結果，艾登雖然在國內仍是受歡迎的人物，但在政壇上的影響力卻大大下降。

在1939年9月，[[第二次世界大戰|第二次世界大戰]]爆發，艾登重新被邀請加入張伯倫的政府，成為[[自治領大臣|自治領大臣]]（Secretary of State for Dominion Affairs），但由於這並非「戰時內閣」的職位，故此，到了1940年5月，即[[納粹德國|納粹德國]]入侵[[法國|法國]]和[[内维尔·张伯伦|張伯倫]]請辭以後，艾登沒有被考慮為接任[[英國首相|首相]]的人選，結果，首相一職由[[邱吉爾|邱吉爾]]接任。邱吉爾上任後，任命艾登為[[陸軍大臣|陸軍大臣]]（Secretary of State for War），後在1940年尾獲重任為外相，並於1941年成為[[政治戰爭行政部|政治戰爭行政部]]（Political Warfare Executive）內，行政委員會的委員。雖然艾登甚得邱吉爾的信任，但艾登的權限卻受到了限制。因為很多重要的談判，例如和[[富蘭克林·德拉諾·羅斯福|羅斯福]]與[[史達林|史達林]]的會談，都是由邱吉爾本人所主持。然而，艾登依然忠誠地為邱吉爾擔當副手。到了在戰爭末期，艾登則主要被指派處理[[英國|英國]]與[[戴高樂|戴高樂]]的關係。在1942年，艾登又兼任[[下議院領袖|下議院領袖]]。

當[[英國工黨|英國工黨]]於1945年的大選勝出以後，艾登的保守黨成為了反對黨，他本人則出任副黨魁。在那時很多人認為[[邱吉爾|邱吉爾]]應該退休，讓艾登繼任為黨魁。但一方面邱吉爾不願考慮這個問題，而另一方面艾登仍十分效忠於邱吉爾，所以他沒有為此催迫邱吉爾。反而，在這段時候，大戰雖然已近尾聲，但艾登卻因為第一段婚姻的破裂和兒子的死訊而大受打擊。

1951年，[[英國保守黨|英國保守黨]]重新掌政，艾登第三次被聘任為外務大臣。今次邱吉爾只是為名義上的首相，而同時在[[冷戰|冷戰]]氣氛日漸濃厚下，艾登則終於首次可以不受干預下，掌理[[英國|英國]]的外交事務。縱然英國的力量在戰後已經今非昔比，但他仍能在一系列的外交危機中作出有效的應對。早於1950年，艾登已經和第一任妻子離婚，因此在1952年迎娶了邱吉爾的姪女，[[克萊麗莎·史賓塞-邱吉爾|克萊麗莎·史賓塞-邱吉爾]]小姐（1920年—）─一位名過其實的[[天主教|天主教]]教徒。結果，儘管艾登的這段婚姻比第一段要美滿得多，卻仍然遭到信奉天主教的作家[[伊芙琳·沃|伊芙琳·沃]]所猛烈抨擊。1953年，艾登因為[[膽囊|膽囊]]出現輕微疾病，入院進行了一連串手術。但不幸地，他的健康卻未能完全恢復，並阻礙了他日後的工作。後在1954年，他出席了[[日內瓦會議_(1954年)|日内瓦会议]]，并与[[中华人民共和国国务院总理|中国总理]]兼[[中华人民共和国外交部部长|外长]][[周恩来|周恩来]]达成了两国之间互派[[外交官|代办]]的协议，从而标志着两国正式[[外交|建立外交关系]]，从而也使英国成为了第一个和[[中华人民共和国|中国]]建立[[外交|正式外交关系]]、实现[[外交|邦交正常化]]、互派[[外交官|大使、公使、代办]]与互设[[外交代表机构|大使馆、公使馆、代办处]]与互相拥有[[外交豁免权|外交特权与豁免权]]的第一个西方大国。也是在同一年，在从[[日内瓦会议|日内瓦会议（1954年）]]回国后，他受勳為[[嘉德勋章|嘉德勳爵士]]。

== 首相 ==
[[File:Anthony-Eden-arms.PNG|thumb]]
1955年4月，[[邱吉爾|邱吉爾]]終於宣佈退休，艾登遂接替他出任[[英國首相|英國首相]]。艾登憑藉著俊朗的外表、魅力和他在大戰的政績，使他在國內成為受歡迎的人物。因此當他一上任首相，便立即舉行了[[1955年大選|1955年大選]]，並為[[英國保守黨|保守黨]]在[[英國下議院|下議院]]帶來更大的多數優勢。可是，艾登從未出任過有關本土事務的職位，故缺乏處理經濟事務的經驗。為此，艾登把這些事務一概交給他的副手們，如[[拉博·巴特勒|拉博·巴特勒]]等，至於自己則專注在外交事務，並和[[美國總統|美國總統]][[艾森豪|艾森豪]]結成緊密的同盟。而艾登的名言：「要永遠以和平為大前提。」（Peace comes first, always）更使他的受歡迎程度有增無減。

但是，這個英、美同盟在日後被證明是徒具虛名的。在1956年，艾登聯合[[法國|法國]]，企圖阻止[[埃及總統|埃及總統]][[納賽爾|納賽爾]]將[[蘇伊士運河|蘇伊士運河]]國有化，因為蘇伊士運河自[[十九世紀|十九世紀]]以來，都是{{link-en|蘇伊士運河公司|Suez Canal Company}}的資產，而該公司則由英國和法國的股東所擁有。面對著蘇伊士運河國有化，艾登根據他在1930年代的外交經驗，視納賽爾為[[墨索里尼|墨索里尼]]的翻版，認為兩人皆如出一轍，都是急激的民族社會主義者，並會入侵其他國家。然而其他意見卻認為，納賽爾的舉動是愛國主義的表現，合情合理。

結果在1956年10月，經過了幾個月的談判和嘗試斡旋後，最終勸阻[[纳赛尔|納賽爾]]失敗。於是英國和法國，聯同[[以色列|以色列]]出兵[[埃及|埃及]]，並佔領了蘇伊士運河地區。不過在出兵以後，[[艾森豪|艾森豪]]立即表示了強烈的反對，譴責這是[[殖民主義|殖民主義]]的死灰復燃。同時，這位美國總統是[[非殖民地化|非殖民地化]]的支持者，認為這樣除了可使[[殖民地|殖民地]]得到自由以外，又有利於[[美國|美國]]的對外貿易，而且更有助和[[阿拉伯|阿拉伯]]與[[非洲|非洲]]地區的領導人建立良好關係。至於艾登則忽略了，自[[第二次世界大戰|第二次世界大戰]]以來，英國政府的財政狀況都要依賴美國，以致艾登最後不得不向美國的壓力屈服，進行撤兵。是次事件稱為「[[蘇伊士運河危機|蘇伊士運河危機]]」，標誌著[[大英帝國|大英帝國]]和[[法國|法國]]的正式衰落，脫離列強行列。

在蘇伊士的徹底失敗，使他的聲望受挫之餘，健康亦因而惡化。但他的財相[[麥美倫|麥美倫]]，儘管是蘇伊士運河危機的其中一位策劃人，在此時卻乘機巧妙地操縱艾登，使他終於在1957年1月辭去了[[英國首相|首相]]一職，並由自己接任。艾登辭職後，成功挽留了自己的聲望，並於1961年被冊封為[[亞芬伯爵|亞芬伯爵]]。

== 退休 ==
離開政壇以後，艾登與第二任妻子在維特郡過著平靜的生活，期間又寫了一本受好評的個人傳記，名為《''Another World''》，除此之外，他亦寫了幾本政治回憶錄。在1977年，艾登在[[美國|美國]]外遊時，身體狀況突然急劇惡化，在他的要求下，當時的[[英國首相|英國首相]][[卡拉漢|卡拉漢]]派出[[英國皇家空軍|英國皇家空軍]]接他回國，最後亞芬伯爵在1977年1月14日，因[[肝癌|肝癌]]在[[索爾斯伯里|索爾斯伯里]]逝世，終年79歲。

從1945年至1973年，艾登是[[英格蘭|英格蘭]][[伯明翰大學|伯明翰大學]]的校監。

艾登唯一存活的兒子，[[尼古拉·艾登|尼古拉·艾登]]在1977年以前稱為艾登子爵，也是一位[[政治家|政治家]]，曾在-{zh-hans:[[玛格利特·撒切尔|玛格利特·撒切尔]]夫人;zh-hant:[[玛格利特·撒切尔|瑪格利特·柴切爾]]夫人;zh-hk:[[戴卓爾夫人|戴卓爾夫人]]}-的政府供職，一直至54歲時，因[[愛滋病|愛滋病]]早逝。

艾登的長子，西門·艾登（Simon Eden），於[[第二次世界大戰|第二次世界大戰]]期間(1945年6月)，隨英國皇家空軍在[[緬甸|緬甸]]的一次行動中陣亡。艾登與西門的關係十分要好，故他的死訊對艾登造成了很大打擊。

有關艾登的文件和政府檔案，現正存放於[[伯明翰大学|伯明翰大學]]，以作特別館藏。

=== 變動 ===
* 1955年12月，拉博·巴特勒取代[哈利·克魯克香克，成為掌璽大臣及下議院領袖。麥美倫則接替拉博·巴特勒為財政大臣。塞爾文·勞埃接替麥美倫為外務大臣。沃爾特·唐納·蒙克頓爵士取代勞埃為國防部長。伊安·麥克勞德（Iain Macleod）獲接替蒙克頓爵士的勞工及國民服務部長一職。[[喬治·道格拉斯-漢彌爾頓，第十代塞爾扣克伯爵|塞爾扣克勳爵]]（Lord Selkirk）取代伍爾頓勳爵為蘭卡斯特公爵領地總裁。另一方面，[[工務部長|工務部長]]（Minister of Public Works），[[派翠克·巴肯-赫本|派翠克·巴肯-赫本]]（Patrick Buchan-Hepburn），加入內閣，而退休金及國民保險部長一職在奧斯伯特·皮克退休後脫離內閣。

* 1956年10月，蒙克頓爵士改任[[財政部主計長|財政部主計長]]，[[安東尼·亨利·海德|安東尼·亨利·海德]]（Anthony Henry Head）則接任為國防大臣。
