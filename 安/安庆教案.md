'''安庆教案'''是[[清朝|清朝]][[同治|同治]]八年九月三十日（1869年11月3日）在[[安徽省|安徽省]][[安庆|安庆]]发生的一次[[教案_(宗教)|教案]]。在基督教和当地士绅百姓冲突不断的背景下，当时来安庆参加[[府试|府试]]的各地考生砸毁英法两国教堂，后来[[两江总督|两江总督]]、[[安徽巡抚|安徽巡抚]]和英法方面签署《南京协定》，事件才平息下来。

== 背景 ==
1700年，[[天主教|天主教]]的[[比利时|比利时]][[方济会|方济会]]会士[[卫方济|卫方济]]在安庆设立教堂传教，但是没有持续下去。[[鴉片戰爭|鴉片戰爭]]后，传教士在中国活动迅速增加，1865年，[[天主教江南代牧区|天主教江南代牧区]]曾希望在安庆建立活动地点，但是时任[[两江总督|两江总督]]的[[李鸿章|李鸿章]]禁止外国传教士在安庆购买房产。1867年，[[曾国藩|曾国藩]]允许[[法国|法国]]传教士在安庆东右坊建立教堂。1869年，[[英国基督教新教|英国基督教新教]][[差会|差会]][[内地会|内地会]]传教士[[宓道生|宓道生]]{{efn|也作“密道生”。}}和[[卫养生|卫养生]](James Williamson)在西右坊建立圣爱堂。

== 事件的发生 ==
[[File:JJMeadows.jpg|thumb]]传教士[[宓道生|宓道生]]]]

[[天主教|天主教]]、[[新教|新教]]在安庆建立组织之后，和当地民众矛盾日益激化，外地的[[反基督教|反基督教]]思想和组织也流入安庆。同治八年八月十二日（1869年9月27日），[[怀宁县|怀宁县]]即将举行[[科举|科举]]考试，安庆知府派人告诉英国传教士“[[县试|县试]]完毕，即是[[府试|府试]]日期，应考童生甚易滋事”，希望他们暂停活动或者离开安庆。但是英国传教士没有听从知府。

同年[[农历|农历]]九月（[[公历|公历]]10月）[[府试|府试]]即将在安庆举行，从各地赶到安庆应考的考生纷纷传阅外地有关反基督教的[[传单|传单]]，一时间考场外也出现揭贴，称洋教“不敬祖宗神佛”，“恃金钱诱人入教”，“迷拐男女，剖取孩子心目”。安庆知府要求考生不要滋事，加强了城内的治安，同时亲自前往英法两国教堂要求传教士在府试期间暂停活动。两位英国传教士不愿离去，要求知府查办造谣之人。10月31日，有考生前往教堂和传教士理论，不欢而散。农历九月二十九日（公历11月2日）出现了“匪教猖獗，与考童为难，订于初二日拆圣爱堂”的揭帖。冲突一触即发，英国传教士两人恐教堂被毁，与第二天早上赶往[[安庐道|安庐道]][[衙门|衙门]]{{efn|安庐道衙门兼理外交事务。}}，要求追查惩办[[揭贴|揭贴]]之人，并要求给予人身保护。[[道台|道台]][[刘传祺|刘传祺]]让下属告诉两人，“所有此事情形，俱已知悉，应向知府衙门告知。”于是两人准备乘轿前往知府，但在刚出道署大门就遇上了赶考的[[武举|武举]]考生。考生推翻轿子，两人逃回署内，冲突就此开始。

考生先冲到了[[内地会|内地会]]的圣心堂打砸，传教记录被毁。传教士[[宓道生|宓道生]]的妻儿被官兵保护送到了道署。然后考生又冲向了法国[[耶稣会|耶稣会]]天主堂。法国[[神父|神父]][[韩伯禄|韩伯禄]]因病当时在[[长江|长江]]码头等待轮船准备去[[上海|上海]]，听到天主堂被毁的消息连忙赶去，沿途不断遭到袭击，赶到教堂时被人群围住，无法去知府，只得返回码头去[[上海|上海]]报信。

安庆知府后来带人将人群疏散，又保护起两座教堂。两位英国传教士要求送他们去[[镇江|镇江]]，后又改为去[[九江|九江]]，农历十月一日安庆官府派人送二人到九江，并给100[[银元|银元]]做路费。至此事件稍稍平息下来。

== 后续发展 ==
负责天主教江南教区西教区传教的法国[[耶稣会|耶稣会]]会士金式玉{{efn|又作“金缄三”。}}（Joseph Seekinger），于农历十月初七（公历11月10日）赶到安庆，要求与官员会面讨论赔偿，但遭到拒绝。后来他向[[法国驻上海总领事馆|法国驻上海总领事馆]]求助。英法两国后来向[[总理衙门|总理衙门]]提交抗议照会。农历十月十六日（公历11月19日），[[英国驻华公使|英国驻华公使]][[阿礼国|阿礼国]]率舰队行驶至安庆江面武力威胁，[[安徽巡抚|安徽巡抚]][[英翰|英翰]]表示考试结束后再办案。

== 南京协定 ==
农历十月十九日(公历11月22日)，法国驻沪总领事达伯理照会英翰，要求赔偿法国天主教堂损失4000银元。后来[[法国驻华公使|法国驻华公使]][[罗淑亚|罗淑亚]]率4艘法国军舰，由上海驶往南京、安庆，并且直接与[[两江总督|两江总督]][[马新贻|马新贻]]交涉。清廷向[[总理衙门|总理衙门]]发去谕告，要求“无论如何变通，总期拿获滋事之人，赔偿所失之物，以便迅速了结，免致彼国藉此要挟，丛生枝节”，同时也要求马新贻、英翰两人快速结案。最终，两人和英法方面签署了《南京协定》，内容主要有：
{{Quote|
* 赔偿英国内地会1735银元，赔偿法国耶稣会4000银元。
* 将肇事者逮捕审问刑罚。
* 出告示严禁反教活动。}}

== 影响 ==
同治九年正月二十四日(1870年2月23日)，英国公使[[威妥玛|威妥玛]]照会总理衙门，要求处理安庆教案中相关官员。同年阴历四月，马新贻、英翰两人在安庆立石碑，表明教会可以自由买卖土地，地方官员处理此类事件时需迅速公平办案，民教和睦相处。

《南京协定》赔偿给法国耶稣会的4000银元中有2000银元由安庆知府代为购买建立教堂的土地，这片土地选在黄家狮，教堂自1872年建成，命名为[[安庆耶稣圣心主教座堂|安庆耶稣圣心主教座堂]]，后来作为[[天主教安庆总教区|天主教安庆总教区]]的[[主教座堂|主教座堂]]保留到现在。

此后安徽省内教案仍然不断发生，影响大的如1891年的[[芜湖教案|芜湖教案]]。直到光绪二十四年(1898年)，颁布了《安徽保护教士章程》。随后教案虽然有所减少，至清末大大小小的冲突仍然不断发生。
