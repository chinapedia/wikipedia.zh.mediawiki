'''安諾星人Anodite'''來自於'''安諾星Anodyne'''，是BEN 10<ref>{{Cite web|url=https://www.cartoonnetworkasia.com/show/ben-10-ultimate-alien|title=少年駭客|publisher=[[BEN10|BEN10]]}}</ref>裡擁有自由心靈的[[天能|天能]]生物。

成年的安諾星人是純粹能量，可以自由截取天地萬物的[[天能|天能]]。 

== 外貌 ==

* 在Ben10外星英雄和終極英雄的劇情中，安諾星人的身體是暗紫色的人形能量體，以及統一有著一頭像觸手的粉紅色能量頭髮，他們並沒有耳朵、鼻子、嘴唇，也沒有穿衣服。<ref>{{Cite web|url=https://www.cartoonnetworkasia.com/show/ben-10-omniverse|title=安諾星人|publisher=[[BEN10|BEN10]]}}</ref>
* 在Ben10全面進化裡，他們的頭髮不再是一樣的而各有不同的形態，身體也變成亮紫色，更像人類一樣有耳朵、鼻子和嘴唇，並有穿衣服。<ref>{{Cite web|url=https://www.cartoonnetworkasia.com/show/ben-10-omniverse|title=安諾星人|publisher=[[BEN10|BEN10]]}}</ref>

== 性格特質 ==
安諾星人的性格非常自由奔放，不拘小節，他們並不像大多數人類一樣嚴肅地看待生活，導致部分的安諾星人為了滿足自己的樂趣，而做出無視道德的行為，不過並不代表安諾星人不願意幫助別人，事實上他們幫助過非常多的人，

但他們的確會有點瞧不起非[[天能|天能]]生物的凡人們。

== 繁衍 ==

* 安諾星人能夠透過製造人類型體來和人類繁衍後代。
* 如果安諾星人和人類繁衍後代的話，則不能保證後代能夠繼承安諾星人的超能力，安諾星人的超能力(薇朵娜稱之為[[天能|天能]]天賦)，也有可能繼承到孫輩的身上，就像薇朵娜的孩子們都沒有繼承到[[天能|天能]]天賦，但她的孫女卻有。

== 超能力 ==

* 安諾星人可以自由操縱生命能量(魔法的本質)。他們可以透過這種粉紅色能量產生爆炸或製造防護罩等等，而這些能量的威力取決於安諾星人本身所掌握[[天能|天能]]的技巧，不過當他們露出外星人型體時，[[天能|天能]]的威力會大幅度的提升，技巧熟練的安諾星人   可以直接控制比他們弱的安諾星人所製造的[[天能|天能]]，並能自由飛行到高處而不是只浮在空中幾公尺。

* 在外星人型態中，安諾星人強大的力量可以做有限度的修改現實，並且能夠在太空中生存和飛行，或讓自己或他人在各個空間象限進行瞬間轉移，也能夠自由製造偽裝用的身體，截取天地萬物的[[天能|天能]]。他們的身體能夠承受強力的攻擊而毫髮無傷。     在生氣的時候可以改變自身的型體大小。

* 安諾星人可以透過[[天能|天能]]磁場互相溝通，就像小玟和桑妮戰鬥時Call薇朵娜那樣。以及心電感應能力，像是薇朵娜在人類型態被困住時，還是能夠透過[[天能|天能]]將思緒裡的人事物注入田馬克的腦海中。                                                                                   安諾星人還擁有感知追蹤的能力，他們可以透過殘留的生命能量來追蹤目標。

* 安諾星人截取的[[天能|天能]]越多，就越強大。

* 要完全掌握安諾星人的超能力至少需要受七十五年的正規訓練，如果缺乏訓練就使用強大的超能力將會失去自我。<ref>{{Cite web|url=https://www.cartoonnetworkasia.com/show/ben-10-ultimate-alien|title=少年駭客|publisher=[[BEN10|BEN10]]}}</ref>

== 弱點 ==

* 安諾星人無法追蹤伊克透瑞星人(鬼影)，因為他們不具備[[天能|天能]]，而追蹤能力對於無生命物體或是具有相同[[天能|天能]]的目標也沒有效果。
* 安諾星人的能量也對科技品犯沖，例如在一個充滿科技品的地方，安諾星人會很難追蹤目標，而部分科技品還能夠封印安諾星人的能力，例如智能機器人雷射爆有一個手鐲能夠以人形囚禁並封印安諾星人除了心電感應外大部分的能力。
* 加拉巴哲龜對於安諾星人的[[天能|天能]]攻擊完全免疫，但是魔咒卻不一定。
* 魔法也可以用來對抗[[天能|天能]]攻擊。
* 安諾星人的[[天能|天能]]並不是源源不絕的，是會耗盡的，當耗盡時就會死亡。
* 有人提出安諾星人不能製造[[天能|天能]]，但可以使用預備好的[[天能|天能]]。

== 名稱由來 ==
安諾星人英文Anodite來自拉丁語Anodyne，意思是減輕身體的疼痛。 

== 關於安諾星人 ==

* 安諾星人沒有DNA。
** 因此安諾星人和另一物種繁衍的後代，基本上都會擁有安諾星人基因，但在生物學的角度，後代幾乎百分之百只會遺傳到非安諾星人的基因。
** 這也表示Omnitrix無法將使用者變成安諾星人，因為沒有DNA樣本可以採納。
** 但安諾星人確實擁有類似於遺傳物質的特性。 
* 假設安諾星人使用Omnitrix或Ultimatrix，他們還是可以使用[[天能|天能]]。
* 目前已知一種外星人可以直接透視安諾星人的人類型態直接看到本體。<ref>{{Cite web|url=https://www.cartoonnetworkasia.com/show/ben-10-ultimate-alien|title=少年駭客|publisher=[[BEN10|BEN10]]}}</ref>
