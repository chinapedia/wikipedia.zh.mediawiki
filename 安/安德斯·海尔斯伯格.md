'''安德斯·海尔斯伯格'''（'''Anders Hejlsberg'''，{{bd|1960年|12月}}），[[丹麦|丹麦]]人，[[Borland|Borland]] [[Turbo_Pascal|Turbo Pascal]][[编译器|编译器]]的主要作者。进入[[微软|微软]]公司后，先后主持了[[Visual_J++|Visual J++]]、[[.Net|.Net]]， [[C♯|C#]] 和 [[TypeScript|TypeScript]]。

== 生平 ==
1960年出生於[[哥本哈根|哥本哈根]]，曾在[[丹麦技术大学|丹麦技术大学]]学习工程，但沒有畢業，大學時期他曾替Nascom microcomputer撰寫程式，他曾為Nascom-2電腦撰寫藍標籤（Blue Label）Pascal compiler，到了DOS時代他又重新改寫這套compiler。当时他在丹麦拥有个叫[[Poly_Data|Poly Data]]的公司，他编写了[[Compass_Pascal|Compass Pascal]]编译器核心，后来叫[[Poly_Pascal|Poly Pascal]]。1986年他首次認識了[[Philippe_Kahn|Philippe Kahn]]（[[Borland|Borland]]的创立者）。

== 在Borland公司 ==
安德斯·海尔斯伯格把Compass Pascal编译器核心授权给了[[美国|美国]][[Borland|Borland]]公司，并作为雇员加入Borland公司，并且是后来所有[[Turbo_Pascal|Turbo Pascal]]版本与[[Delphi|Delphi]]前3个[[版本|版本]]的[[架构师|架构师]]。[[Philippe_Kahn|Philippe Kahn]]为第一个版本的[[Turbo_Pascal|Turbo Pascal]]添加了[[用户界面|用户界面]]与[[编辑器|编辑器]]。

== 加入[[微软|微软]]公司 ==
1996年Hejlsberg加入[[微软|微软]]公司。据说，[[比尔·盖茨|比尔·盖茨]]亲自参与了这次挖角行动，年薪三百萬美元，并许诺安德斯·海尔斯伯格在微软将得到技术上的足够自由和资源支持<ref>{{cite book |author= [[李維_(台灣)|李維]] |title= 《BORLAND傳奇》 |publisher= 碁峰資訊股份有限公司 |isbn= 986421294X |date=2003年4月}}</ref>。据说该事件也是微软公司和Borland公司后续官司的导火索。

== 在微软公司 ==
进入微软公司后，首先主持了[[Visual_J++|Visual J++]]的开发工作，后来由于在Java开发工具授权问题上和[[升阳|Sun]]公司的纠纷，微软停止了Visual J++的后续开发。

这之后，作为[[.Net|.Net]]概念的发起人之一，安德斯·海尔斯伯格被任命为微软[[.Net|.Net]]的首席架构师，主持.Net的开发工作。
