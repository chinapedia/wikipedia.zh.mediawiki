'''安哥拉蟒'''（學名：''Python anchietae''）是[[蛇亞目|蛇亞目]][[蟒科|蟒科]][[蟒屬|蟒屬]]下的一種無毒蟒蛇，主要分布於[[安哥拉|安哥拉]]及[[納米比亞|納米比亞]]，是一種與[[球蟒|球蟒]]有著密切聯繫的罕有蛇類，目前未有任何亞種被確認。<ref name="ITIS">{{ITIS|ID=634782|taxon=''Python anchietae''|year=2007|date=21 September}}</ref>

== 特徵 ==
成年的安哥拉蟒最長能達180[[公分|公分]]，身體顏色主要以[[紅色|紅色]]及[[棕色|棕色]]作基調，表皮上有為數頗多的[[白色|白色]]小圓點斑紋。腹部位置呈[[黃色|黃色]]，是一種無論在野外還是人工飼養的環境下均十分罕見的蟒蛇。<ref name="Meh87">Mehrtens JM. 1987. Living Snakes of the World in Color. New York: Sterling Publishers. 480 pp. ISBN 0-8069-6460-X.</ref>

== 地理分布 ==
安哥拉蟒主要分布於[[非洲|非洲]]，包括[[安哥拉|安哥拉]]南部及[[納米比亞|納米比亞]]北部地區。其標準產地為安哥拉的「Catumbella」。<ref name="McD99"/>

== 生態習慣 ==
安哥拉蟒多棲息於多石地形，也會出沒於有岩石的草原及灌木林間。<ref name="Meh87"/> 牠們多於日間活動，平常置身於小型岩洞、巨石縫隙等地方，主要進食各種小型的[[哺乳動物|哺乳動物]]及[[鳥類|鳥類]]。<ref name="Meh87"/>

繁殖方面，安哥拉蟒屬於[[卵生|卵生]]動物，雌蛇每次能誕下4至5枚蛇卵，但雌蛇是否具有孵卵習慣則尚是未知之數。安哥拉蟒的初生幼蛇達43至46公分長。<ref name="Meh87"/>

== 分類學 ==
根據學者布朗利（Broadley, 1990）指出，安哥拉蟒與分布於西非的[[球蟒|球蟒]]（''Python regius''）有緊密的血緣關係。<ref name="Meh87"/>
