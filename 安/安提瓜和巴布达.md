'''安提瓜和巴布达'''（{{lang-en|'''Antigua and Barbuda'''}}），简称“'''安巴'''”，是[[中美洲|中美洲]]的一个[[岛国|岛国]]，位于[[加勒比海|加勒比海]]和[[大西洋|大西洋]]之间，往南邻近[[法国|法]]属[[瓜德罗普|瓜德罗普]]，西南方向靠近[[英国|英]]属[[蒙特塞拉特|蒙特塞拉特]]，西面则为[[圣基茨和尼维斯|圣基茨和尼维斯]]，其國際代碼為AG。安提瓜和巴布达有两个主要的岛屿，分别是[[安提瓜岛|安提瓜岛]]和[[巴布达岛|巴布达岛]]，还有一些更小的岛屿。安提瓜和巴布达是[[背风群岛|背风群岛]]的一部分。

安提瓜和巴布达是世界上人口最少的国家之一。根据2016年数据，安巴共有人口10.10万，绝大多数为非裔。[[英语|英语]]为官方语言和通用语。多数居民信奉[[基督教|基督教]]。

== 历史 ==
{{main|安地卡及巴布達歷史}}
最早在[[安提瓜岛|安提瓜岛]]和[[巴布达岛|巴布达岛]]定居的人是[[西波內人|西波內人]]（''Ciboney''），他们早在前25世纪就已经在这里居住了。后来[[阿拉瓦人|阿拉瓦人]]把他们排挤掉了。好战的[[加勒比人|加勒比人]]后来虽然占据了安提瓜和巴布达周围的岛屿，但没有占据这两个岛。

1493年[[克里斯托弗·哥伦布|克里斯托弗·哥伦布]]在他的第二次新世界航行中在安提瓜登陆，他将这个岛命名为''安提瓜的圣玛丽亚''。但西班牙并没有在此殖民，因为安提瓜岛缺乏淡水，还有好战的[[加勒比人|加勒比人]]。从1632年开始[[英国|英国]]人开始在安提瓜岛殖民，1634年殖民邻近的巴布达岛。1666年法国人曾短暂入侵安提瓜。英国人建立甘蔗种植园，并从[[非洲|非洲]]贩运奴隶。直到1834年安提瓜的奴隶制才废除。奴隶虽然获得解放，但他们在经济上依然依赖农场主。
1939年[[安提瓜商工会|安提瓜商工会]]组成。1946年从这个工会中演变出的[[安提瓜工党|安提瓜工党]]参加竞选。从1951年工党成为长期的执政党，直到2004年大选联合进步党获胜。联合进步党执政到2014年，工党当年再次夺回政权。

1981年安提瓜和巴布达成为[[英联邦|英联邦]]中的一个[[独立国家|独立国家]]，国家元首是[[安提瓜和巴布达女王|安提瓜和巴布达女王]]。

== 地理 ==
{{main|安地卡及巴布達地理}}
安提瓜和巴布达由一些岛屿组成，其中安提瓜岛是最大的（281平方公里），为火山岛，地势低平；巴布达岛其次（161平方公里），为珊瑚岛。这些岛一般都相当低，全国最高点[[奥巴马峰|奥巴马峰]]（Mount Obama），[[海拔|海拔]]只有402米<ref>[http://news.163.com/09/0806/02/5G0G9CNU0001121M.html 加勒比岛国将最高峰命名“奥巴马峰”为其庆生 (网易新闻)]</ref>。

属[[热带草原气候|热带草原气候]]，年均气温27℃，年均降水量1100毫米。

== 政治 ==
安提瓜和巴布达是[[英联邦|英联邦]]的成员，因此它的国家首脑是英国女王[[伊丽莎白二世|伊丽莎白二世]]。她在岛上的代表是她任命的总督。这个总督一般作为总理大臣的顾问。

安提瓜和巴布达行[[兩院制|兩院制]]，上院的17名成员是由总督提名任命的（一般在与总理大臣和反对党黨魁协商后），下院的17名成员是选举出来的。总理大臣一般是下院的多数党黨魁，他和他的政府成员一起管理国家事物。选举至少每5年一次，但总理可以随时要求进行新的选举。

安提瓜和巴布达有多个党派，它们之间的斗争往往很激烈。反对党指责多数党利用它多年执政的机会来取得更大的选举结果。

安提瓜和巴布达的宪法保障言论、出版、信仰、行动和结社的自由。它的法系主要是基于[[普通法|普通法]]。

== 经济 ==
{{main|安地卡及巴布達經濟}}
安提瓜和巴布达的最主要的经济来源是旅游业，占国内生产总值的一半以上。自2000年开始的旅游业的萧条为安提瓜和巴布达带来了很大的困难。农业主要是自给。此外还有少数工业，主要是生产出口产品如床上用品、手工艺品和电子器件。离岸金融行业也是重要的经济部门，但美国人艾伦·斯坦福曾经以安提瓜为基地，制造了史上最大的庞氏骗局之一，骗取投资人80亿美元以上。这个事件很大程度上打击了金融业。

安提瓜的美利坚大学及安提瓜医科大学对当地的经济也很重要。安提瓜和巴布达政府试图召集运输业、通讯业和金融业企业来减轻它对旅游业的依赖。

== 人口 ==
{{main|安地卡及巴布達人口}}
大多数人是过去[[黑奴|黑奴]]的后代，但也有少数[[欧洲人|欧洲人]]，主要是[[英国人|英国人]]和[[葡萄牙人|葡萄牙人]]。几乎所有居民信奉[[基督教|基督教]]。

== 文化、体育与宗教 ==
安提瓜和巴布达的文化主要受到英国和非洲传统的影响。国球是[[板球|板球]]，曾经于2007年举办[[板球世界杯|板球世界杯]]。足球、帆船等也是很受欢迎的运动。每年八月这里为庆祝废除奴隶制举办[[狂欢节|狂欢节]]。
安提瓜的两家报纸是《每日观察》(Daily Observer)和《加勒比时报》(Caribbean Times)。
77%的人口信仰基督教，其中英国国教(17.6%)是信众最多的。

== 軍事 ==
{{main|安地卡及巴布達軍事}}
安地卡及巴布達國防軍（{{lang|en|Antigua and Barbuda Defence Force}}；{{lang|en|ABDF}}），為[[安地卡及巴布達|安地卡及巴布達]]的軍事力量，常備兵力約170人，預備兵力約75人，是世界上最小的國家軍事部隊。
