'''安远炮台'''是[[中国|中国]][[浙江省|浙江省]][[宁波市|宁波市]][[镇海区_(宁波市)|镇海区]]的一座炮台。炮台位于甬江北岸，[[招宝山|招宝山]]南麓，现存炮台始建于光绪十三年（1887年），次年建成。炮台主体为圆形，[[三合土|三合土]]修筑。1996年，炮台与其他镇海口海防设施一道被列入[[第四批全国重点文物保护单位|第四批全国重点文物保护单位]]中的[[镇海口海防遗址|镇海口海防遗址]]项目<ref name="nbwb">{{cite web|url=http://www.nbwb.net/pd_wwbh/info.aspx?Id=866&type=2|title=镇海口海防遗址|publisher=宁波文化遗产保护网|date=2006-01-04|accessdate=2013-07-21|archive-url=https://web.archive.org/web/20140314221549/http://www.nbwb.net/pd_wwbh/info.aspx?Id=866&type=2|archive-date=2014-03-14|dead-url=no}}</ref>。

== 历史 ==
最初的安远炮台由浙江提督[[歐陽利見|歐陽利見]]建造于[[镇海之战_(1885年)|中法战争镇海之战]]爆发前<ref name="海天雄镇">{{cite book|title=海天雄镇：镇海口海防遗址|author=夏炳章|author2=王闰清|publisher=中国大百科全书出版社|year=1998|isbn=7-5000-6028-9|location=北京}}</ref>{{rp|17-21}}，当时炮台配备三门阿姆斯特朗前膛炮<ref>{{cite web|url=http://www.zhkhfsg.com/hfyz_detail.asp?cataid=290|title=安远炮台|publisher=镇海口海防历史纪念馆|accessdate=2013-08-17|archive-url=https://web.archive.org/web/20141216171617/http://www.zhkhfsg.com/hfyz_detail.asp?cataid=290|archive-date=2014-12-16|dead-url=no}}</ref>。中法战争结束后，鉴于甬江口原有炮台位置偏内且射程有限，同知杜冠英、参将吴杰主持在招宝山下方的石矶修建新的安远炮台，与对岸[[金鸡山_(北仑区)|金鸡山]]下的绥远炮台相对，扼守甬江口。工程于1888年冬完工<ref>[[薛福成|薛福成]]《[[s:宏遠炮台銘記|宏遠炮台銘記]]》：……又稍進，則金雞山下與招寶山後各聳一磯，夾江相望，於此分建二台，曰「綏遠」，曰「安遠」。</ref>，设置21厘米口径[[克虏伯|克虏伯]]后膛海岸炮一门<ref name="xianzhi-military">{{cite book|title=镇海县志|publisher=中国大百科全书出版社|year=1994|isbn=7-5000-0425-7|location=|pages=|chapter=第七编 军事}}</ref>。

1932年，炮台由国军宁波防守司令部管辖<ref name="shizhi">{{Cite book|title=宁波市志|last=|first=|publisher=中华书局|year=1995|isbn=978-7-10101-507-2|location=北京|pages=}}</ref>{{Rp|2090}}。[[抗日战争|抗日战争]]前的1936年，国民政府认为安远炮台位置仍过于靠内，将炮台裁撤，克虏伯后膛炮被迁往位于钳口门炮台山的新镇远炮台<ref name="xianzhi-military" />，并在抗日战争中发挥作用<ref name="nbwb" />。

== 形制 ==
安远炮台位于镇海招宝山南麓乌龙岗下，为镇海口海防遗址中现存最为完整的炮台<ref name="海天雄镇" />{{rp|79}}。炮台占地面积328平方米，外形为圆形，内径16.5米，高6米，厚2米，设有前后炮门，分别面对东海和[[甬江|甬江]]，中心设有环形铁轨，可用于旋转炮口位置。主体为沙砾、黄泥、石灰混合后形成三合土夯筑，相传还混有糯米以增强其强度<ref name="nbwb" />。抗日战争前后，曾使用水泥修筑门和顶部，今洞门尚存，高5米，宽3米，而顶部已毁<ref name="xianzhi-culture">{{cite book|title=镇海县志|publisher=中国大百科全书出版社|year=1994|isbn=7-5000-0425-7|location=|pages=|chapter=第二十四编 文化 新闻}}</ref>。

== 保护 ==
{{external media|width=250px|image1=[http://www.nbwb.net/UploadFile/images/2018012211221396.jpg 镇海口海防遗址（安远炮台）保护区划图]，原始内容[https://web.archive.org/web/20180609021844/http://www.nbwb.net/UploadFile/images/2018012211221396.jpg 存档]于2018年6月9日}}
安远炮台于1981年7月被列入镇海县文物保护单位，1989年12月12日和[[威远城_(招宝山)|威远城]]、[[月城_(招宝山)|月城]]等海防遗迹一同被列入浙江省文物保护单位<ref name="xianzhi-culture" />，1996年以[[镇海口海防遗址|镇海口海防遗址]]之名升格为[[第四批全国重点文物保护单位|第四批全国重点文物保护单位]]<ref name="nbwb" />。炮台周边曾有球场、油库等建筑，炮台部分开裂且周边环境不佳，1995年由镇海区海防办出资维修，拆除周边建筑，同时在炮台旁布置了三门清代[[紅衣大炮|紅衣大炮]]的复制品<ref name="海天雄镇" />{{rp|79}}。
