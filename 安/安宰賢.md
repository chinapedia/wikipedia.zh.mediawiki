'''安宰賢'''<ref>[http://news.nate.com/view/20120801n08233 안재현│My name is...]{{ko}}</ref>（{{lang-ko|안재현}}，{{bd|1987年|7月1日}}），[[韓國|韓國]][[男|男]][[模特兒|模特兒]]、[[演員|演員]]、[[珠寶設計師|珠寶設計師]]，自創珠寶品牌AA.Gban。

== 演藝經歷 ==
2012年，安宰賢在[[JTBC|JTBC]]綜藝《[[李秀根和金炳万的上流社會|李秀根和金炳万的上流社會]]》做為快遞員的角色頗受歡迎。

2013年，接拍《[[來自星星的你|來自星星的你]]》，飾演女主角千頌伊的弟弟千允才，正式成為演員。

2014年，與歌手[[鄭俊英|鄭俊英]]一同擔任[[Mnet媒體|Mnet]]音樂節目《[[M!_Countdown|M! Countdown]]》節目主持人。

2015年，在[[韓國放送公社|KBS]]月火連續劇《[[Blood_(電視劇)|Blood]]》中與[[具惠善|具惠善]]、[[池珍熙|池珍熙]]、[[丁海寅|丁海寅]]合作演出，首次擔任男主角。

== 個人品牌 ==

Gban 中的Gban，來自韓文「戒指」倒轉來的讀音，靈感源自一次他在街上看到一位男士手捧著花，安宰賢對於那單純心意留下深刻印象。花的生命短，所以結合花與飾物，延續花的生命，以花、藤蔓等作主題，推出一系列飾物。<ref>[https://www.niusnews.com/=P024g2w3 外表看似小孩，時尚感卻過於常人！不只是千頌伊弟的安宰賢]</ref><ref>[http://www.jessicahk.com/articles/qian-song-yi-di-zai-xian-xi-gang-kai-aagbanxian-ding-shi-dian 「千頌伊弟」 安宰賢襲港！開AA.Gban限定飾店]</ref><ref>[https://news.housefun.com.tw/news/article/73035457513.html 《來自星星的你》安宰賢也紅了！　自創品牌設櫃奢華百貨]</ref>
