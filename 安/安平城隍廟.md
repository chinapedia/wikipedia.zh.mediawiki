'''安平城隍廟'''位於[[臺灣|臺灣]][[臺南市|臺南市]][[安平區|安平區]]，昔日為水師官兵所祭祀的廟宇，現與[[安平觀音亭|安平觀音亭]]、[[安平天后宮|安平天后宮]]、[[安平大眾廟|安平大眾廟]]同為安平地區四大公廟{{r|黃}}。其歷史可追溯至清乾隆年間，不過今天的建築是1989年重建的{{r|傅}}。

== 沿革 ==
安平城隍廟據《續修臺灣縣-{志}-》所載，為水師協鎮[[沈廷耀|沈廷耀]]於[[乾隆|乾隆]]十四年（1749年）所建，而後協鎮[[丁朝雄|丁朝雄]]（1785年）、守備[[陳景星|陳景星]]（1801年）與游擊詹勝、守備陳廷梅和李文瀾（1804年）曾修葺之{{r|黃}}。而同治二年（1863年）時的整修得到當地漁戶、地方官員與水師的響應，其姓名與捐款金額都留在三支梁籤上。廟宇於隔年修葺完成，並留下當時臺灣水師中協副總府[[游紹芳|游紹芳]]所贈之「被靈夑理」的匾額{{r|黃}}。

而在進入[[臺灣日治時期|日治時期]]後，該廟由官廟轉為安平六角頭的公廟，但當時香火並不盛。[[二次大戰|二次大戰]]結束後，該廟於民國七十八年（1989年）重修而成今貌{{r|黃}}。

== 建築 ==
安平城隍廟面寬三開間，由三川殿、拜殿與正殿相連而成，為鋼筋水泥建築{{r|傅}}。其門神為[[牛頭馬面|牛頭馬面]]，與臺南的[[臺灣府城隍廟|府城隍廟]]及[[全臺首邑縣城隍廟|縣城隍廟]]有別，為[[陳壽彝|陳壽彝]]之作{{r|黃}}。

== 祀神 ==
該廟除供奉城隍之外，尚供奉有城隍夫人、[[福德正神|福德正神]]、[[註生娘娘|註生娘娘]]、[[陰陽司|二十四司]]、[[韓盧將軍|韓盧將軍]]與[[范謝將軍|范謝將軍]]{{r|黃}}。
