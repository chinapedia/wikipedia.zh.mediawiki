'''安康医院'''是[[中华人民共和国|中华人民共和国]]对精神病人进行强制医疗或加以监护的专门机构。此类医院隶属于[[公安机关|公安机关]]。

安康医院的性质之一是根据《[[中华人民共和国刑法|中华人民共和国刑法]]》的规定，由政府对造成危害结果但不负[[刑事责任|刑事责任]]的精神病人进行强制医疗的机构。其性质之二是根据《[[s:中华人民共和国人民警察法|中华人民共和国人民警察法]]》的规定，对严重危害公共安全或者他人人身安全的精神病人进行监护的指定场所之一。

== 建立依据 ==
《[[中华人民共和国刑法|中华人民共和国刑法]]》第十八条规定，“精神病人在不能辨认或者不能控制自己行为的时候造成危害结果，经法定程序鉴定确认的，不负[[刑事责任|刑事责任]]，但是应当责令他的[[家属|家属]]或者[[监护人|监护人]]严加看管和医疗；'''在必要的时候，由政府强制医疗。'''”

《[[s:中华人民共和国人民警察法|中华人民共和国人民警察法]]》第十四条规定，“[[公安机关|公安机关]]的[[人民警察|人民警察]]对严重危害公共安全或者他人人身安全的精神病人，可以采取保护性约束措施。'''需要送往指定的单位、场所加以监护的'''，应当报请县级以上[[人民政府|人民政府]]公安机关批准，并及时通知其[[監護人|监护人]]。”

{{Wikisource|关于加强精神卫生工作的意见}}
1987年4月20日[[中华人民共和国卫生部|中华人民共和国卫生部]]、[[中华人民共和国民政部|民政部]]、[[中华人民共和国公安部|公安部]]《[[s:关于加强精神卫生工作的意见|关于加强精神卫生工作的意见]]》是全国第二次精神卫生工作会议之后出台的文件。该文件要求“开展精神卫生立法和司法鉴定以及对精神病人的管治工作。抓紧起草《精神卫生法》，公安机关从速组建精神病管治院，并会同司法、卫生部门组建司法鉴定委员会。”

{{Wikisource|全国公安机关第一次精神病管治工作会议纪要}}
1988年1月29日[[中华人民共和国公安部|中华人民共和国公安部]]印发的《[[s:全国公安机关第一次精神病管治工作会议纪要|全国公安机关第一次精神病管治工作会议纪要]]》（公发〔1988〕5号）要求各地抓紧建立公安机关管理的精神病管治院，并将各地此类机构统一定名为“安康医院”。

在2001年召开的全国第三次精神卫生工作会议上，[[中华人民共和国卫生部|中华人民共和国卫生部]]也要求各地建立安康医院。

{{Wikisource|关于进一步加强精神卫生工作指导意见的通知}}
2004年9月20日[[中华人民共和国国务院|中华人民共和国国务院]]国办发〔2004〕71号通知转发的[[中华人民共和国卫生部|卫生部]]、[[中华人民共和国教育部|教育部]]、[[中华人民共和国公安部|公安部]]、[[中华人民共和国民政部|民政部]]、[[中华人民共和国司法部|司法部]]、[[中华人民共和国财政部|财政部]]、[[中国残疾人联合会|中国残联]]《[[s:关于进一步加强精神卫生工作指导意见的通知|关于进一步加强精神卫生工作指导意见的通知]]》再次要求没有安康医院的省、自治区、直辖市要尽快建立安康医院。

== 收治对象 ==
根据1988年1月29日[[中华人民共和国公安部|中华人民共和国公安部]]印发的《[[s:全国公安机关第一次精神病管治工作会议纪要|全国公安机关第一次精神病管治工作会议纪要]]》（公发〔1988〕5号）要求，公安机关管理的精神病管治院收治对象包括：
{{引用|
#有[[杀人|杀人]]、[[放火|放火]]、[[强奸|强奸]]、[[爆炸|爆炸]]行为的；
#严重扰乱党政军机关办公秩序和[[企业|企]][[事业单位|事业单位]]生产、工作秩序的；
#严重扰乱公共秩序、交通秩序，危害[[公共安全|公共安全]]的；
#当众出丑，有伤风化的；
#影响社会安定，造成严重后果的。}}

该文件还强调，“公安机关管理的精神病管治院收治上述精神病人，都应经[[精神病|精神病]]司法医学鉴定。”

2010年5月26日在[[武汉|武汉]]召开的全国安康医院工作会议上，公安部监所管理局特别强调了安康医院的收治对象及收治法定程序。在会后接受[[新华网|新华网]]记者采访时，该局表示，“强制医疗是一种特殊的行政强制措施，是由我国《[[中华人民共和国刑法|刑法]]》作出规定的。”<ref name=xinhua/>

决定进行强制医疗必须满足两个先决条件，<ref name=xinhua/>
{{引用|
*第一，被强制医疗人员已发生了触犯刑法的行为，涉嫌犯罪；
*第二，被强制医疗人员是经法定鉴定程序，确认无刑事责任能力的精神病人。<br>两个条件缺一不可，如果该人涉嫌犯罪经鉴定不是精神病人，那么相关部门将按照刑事诉讼程序处理；如果该人虽然是精神病人，但没有发生触犯刑法的行为，公安机关不得对其采取任何限制自由的措施，包括强制医疗措施。}}

该局对上面最后一句话又进行了进一步说明，指出：<ref name="xinhua">{{Cite web|url=http://news.xinhuanet.com/legal/2010-05/28/c_12154943.htm|title=公安部监所管理局就安康医院工作答新华网记者问|accessdate=2017-05-24|author=|date=2010-05-28|work=|publisher=新华网}}</ref>

{{引用|安康医院的收治必须符合法定程序，对有触犯[[中华人民共和国刑法|刑法]]行为、经鉴定无刑事责任能力的精神病人，须经县级以上[[公安机关|公安机关]]决定，安康医院才得接收；所收治的必须是法定对象，对没有发生触犯[[中华人民共和国刑法|刑法]]行为的普通精神病人，安康医院一律不得接收。}}

== 名称 ==
1988年《全国公安机关第一次精神病管治工作会议纪要》中称，
{{quote|关于统一院名的问题。鉴于目前公安机关管理的精神病管治院名称不统一，为了便于工作，统称“安康医院”，即：××省（自治区、市）公安厅（局）安康医院。}}

但实际上，各安康医院名称也并不都是前加某某公安厅（局），比如[[广州市安康医院|广州市安康医院]]等。

由于“安康医院”并未成为公安机关所建此类医院的专用名称，故大批和公安机关毫无联系的一般医疗机构也在使用“安康医院”作为自己的名称。

== 性质 ==
安康医院大多数均属公安机关下属的[[事业单位|事业单位]]，不属行政编制，在安康医院工作的民警不享受公务员待遇。2004年《[[人民公安|人民公安]]》杂志的报道指，由于有的地方政府财政投入不足，致使一些安康医院经费严重困难，甚至达到无以为继的境地。为此，各安康医院想方设法自筹经费，如进行精神疾病司法鉴定，成为综合医院并对外开放门诊，收治自愿就诊的精神病人以及接收自愿戒毒人员，对外开展[[体检|体检]]等。<ref>[http://www.cpd.com.cn/gb/newspaper/2004-08/05/content_323007.htm 非比寻常的安康事业，人民公安2004年第14期（总第668期）] {{Wayback|url=http://www.cpd.com.cn/gb/newspaper/2004-08/05/content_323007.htm |date=20041215033417 }}</ref>

也有的地方如[[北京市|北京市]]已经改变安康医院的事业单位性质，将其改制为政府行政机构，以解决其资金及人员待遇问题。
