'''安立敏'''（{{bd|1954年|12月||}}）女，[[山西|山西]][[广灵|广灵]]人，[[中华人民共和国铁路|中华人民共和国铁路]]高管<ref name=alm>{{Cite web |url = http://www.china-railway.com.cn/zgsgk/gsld/201501/t20150129_46651.html |title =  安立敏　中国铁路总公司党组成员|author =  |date = 2017-09-21 |publisher = 中国铁路总公司 |language =  |accessdate =  |quote =  }}</ref>。

==生平==
1975年3月参加工作，1981年4月加入中国共产党，[[北方交通大学|北方交通大学]]机械系车辆专业毕业，大学学历，工学学士学位，高级政工师，高级工程师。1975年3月至1978年3月，任[[内蒙古自治区|内蒙古自治区]]呼和浩特市一中教师。1978年3月至1982年1月，在[[北方交通大学|北方交通大学]]机械系车辆专业学习。1982年1月至1984年7月，任北方交通大学党委学生工作部部员、校团委副书记。1984年7月至1984年12月，任全国铁道团委负责人。1984年12月至1990年3月，任全国铁道团委副书记。1990年3月至1993年4月，任全国铁道团委书记。其间，1991年9月至1992年7月在[[中央党校一年制中青年干部培训班|中央党校一年制中青年干部培训班]]学习<ref name=alm/><ref>{{cite news|title=北京交通大学原创话剧《茅以升》引师生热议|url=http://edu.people.com.cn/n/2014/0328/c367001-24768760.html|date=2014-03-28|newspaper=人民网}}</ref>。

1993年4月至1994年6月，任铁道部政治部办公室副主任。1994年6月至1998年9月，任铁道部人事司副司长（政治部干部部副部长）。1998年9月至2002年1月，任铁道部人事司司长（政治部组织部部长）<ref name=alm/>。

2002年1月至2002年3月，任铁道部纪委书记、党组成员，人事司司长（政治部组织部部长）。2002年3月至2013年3月，任铁道部纪委书记、党组成员。2013年3月至2017年9月，任[[中国铁路总公司|中国铁路总公司]]党组纪检组组长、党组成员。2017年9月起，任中国铁路总公司党组成员<ref name=alm/><ref>{{cite news|title=徐守盛安立敏慰问黔张常铁路建设者|url=http://news.gmw.cn/newspaper/2014-12/18/content_102999353.htm|date=2014-12-18|newspaper=光明网}}</ref>。

她是第十六届、十七届、十八届中央纪委委员<ref name=alm/><ref>{{cite news|title=第十八届中央委员中央候补委员中纪委委员最新职务|url=http://www.zgg.org.cn/zggxx/xxchsh/dyshch/201502/t20150209_499853.html|date=2015-02-09|newspaper=[[紫光阁_(杂志)|紫光阁]]网|deadurl=yes|archiveurl=https://web.archive.org/web/20150402122055/http://www.zgg.org.cn/zggxx/xxchsh/dyshch/201502/t20150209_499853.html|archivedate=2015-04-02}}</ref>。
