'''安德里亞斯·加爾'''（{{lang-en|'''Andreas Gal'''}}，{{bd|1976年}}）是[[Mozilla|Mozilla]]前任[[首席技術長|首席技術長]]，著名於幾項[[開放原始碼|開放原始碼]]專案和Mozilla相關技術方面的工作。

加爾出生於[[匈牙利|匈牙利]][[塞格德|塞格德]]，在[[德國|德國]][[呂北克|呂北克]]長大<ref name="telekom">{{cite web|title=Andreas Gal - Der Freiheitskämpfer|url=http://dreisechsnull.telekom.de/mozilla-forschungschef-andreas-gal-der-freiheitskmpfer/|accessdate=26 April 2014|author=Iris Quirin|language=de|deadurl=yes|archiveurl=https://archive.is/20140426231223/http://dreisechsnull.telekom.de/mozilla-forschungschef-andreas-gal-der-freiheitskmpfer/|archivedate=2014年4月26日}}</ref>。在高中期間，他開發了各種開放原始碼AX.25網路堆疊，並為[[業餘無線電|火腿電台]]網路節點（INP3<ref>{{cite web|url=http://nordlink.dyndns.org/cms/index.php?id=27 |archive-url=https://archive.is/20130414142300/http://nordlink.dyndns.org/cms/index.php?id=27 |dead-url=yes |archive-date=2013-04-14 |title=Inter Node Protocol 3 }}</ref>）設計了一種路由協議，該協議得到了AX.25網路路由器的廣泛支援<ref>{{cite web|url=http://he.fi/archive/linux-hams/200110/0032.html|title=INP3 support for Linux|deadurl=yes|archiveurl=https://web.archive.org/web/20141124213729/http://he.fi/archive/linux-hams/200110/0032.html|archivedate=2014-11-24}}</ref><ref>{{cite web|url=http://www.langelaar.net/projects/jnos2/news/|title=JNOS news|deadurl=yes|archiveurl=https://web.archive.org/web/20140427044607/http://www.langelaar.net/projects/jnos2/news/|archivedate=2014年4月27日|df=}}</ref><ref>{{cite web|url=http://www.swiss-artg.ch/xnet/pdf/intern.pdf|title=(X)Net user manual|deadurl=yes|archiveurl=https://web.archive.org/web/20071019194831/http://www.swiss-artg.ch/xnet/pdf/intern.pdf|archivedate=2007-10-19}}</ref>。

在馬格德堡大學攻讀研究生期間，他是AspectC ++的共同設計者，AspectC ++是[[C語言|C]]和[[C++|C++]]語言的[[面向侧面的程序设计|面向侧面]]擴充<ref>{{cite web|url=https://www.academia.edu/267150/AspectC_Language_Proposal_and_Prototype_Implementation|title=AspectC++: Language Proposal and Prototype Implementation}}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。後來，他在[[加州大學爾灣分校|加州大學爾灣分校]]獲得[[電腦科學|電腦科學]][[哲學博士|哲學博士]]學位。他的論文介紹了使用{{link-en|追蹤樹|Trace tree}}的高階語言追蹤即時編譯的概念<ref>
[http://static.usenix.org/events/vee06/full_papers/p144-gal.pdf "HotpathVM: an effective JIT compiler for resource-constrained]
Andreas Gal, Christian W. Probst, Michael Franz - Proceeding VEE '06 Proceedings of the 2nd international conference on Virtual execution environments
{{DOI|10.1145/1134760.1134780}}.</ref>。

加爾於2008年加入Mozilla，並在Google宣布[[Google_Chrome|Chrome]]和[[V8_(JavaScript引擎)|V8]]引擎的幾週前，構建了網頁瀏覽器第一個[[即時編譯|即時編譯]]引擎[[TraceMonkey|TraceMonkey]]<ref>{{cite web|url=https://brendaneich.com/2008/08/tracemonkey-javascript-lightspeed/|title=TraceMonkey: JavaScript Lightspeed}}</ref>。在完成了TraceMonkey的工作之後，加爾成為Mozilla的研究總監。他開始了一個值得關注的計畫[[PDF.js|PDF.js]]，這是一個JavaScript和HTML5編寫的PDF渲染器，取代了[[Firefox|Firefox]]中的[[Adobe|Adobe]] PDF[[外掛程式|外掛程式]]<ref>{{cite web|url=http://www.webpronews.com/mozilla-kills-the-pdf-plugin-in-firefox-19-2013-01|title=Mozilla Kills The PDF Plugin In Firefox 19|access-date=2014-12-04|archive-url=https://web.archive.org/web/20141126045520/http://www.webpronews.com/mozilla-kills-the-pdf-plugin-in-firefox-19-2013-01|archive-date=2014-11-26|dead-url=yes}}</ref>。

2011年，加爾共同建立了Boot to Gecko計畫，後來成為[[Firefox_OS|Firefox OS]]<ref>{{cite mailing list |last = Gal |first = Andreas |title = Booting to the web |mailinglist = mozilla.dev.platform |date = 2011-07-25|url=http://groups.google.com/group/mozilla.dev.platform/browse_thread/thread/7668a9d46a43e482?pli=1 |accessdate = 2011-11-20 }}</ref><ref>{{cite news|url=http://www.latimes.com/business/technology/la-fi-tn-firefox-os-summer-twitter-sony-20130226,0,5519806.story|title=Firefox OS to launch this summer; big phone makers pledge support | work=Los Angeles Times | first=Salvador|last=Rodriguez|date=2013-02-26}}</ref>。

自2013年起，加爾被任命為Mozilla行動工程副總裁<ref>{{cite web|url=http://www.linkedin.com/in/andreasgal|title=Andreas Gal's Linkedin page}}</ref>。2014年4月，加爾成為Mozilla首席技術長<ref>{{cite web|url=https://blog.mozilla.org/blog/2014/04/24/mozilla-cto-andreas-gal/|title=Mozilla CTO: Andreas Gal}}</ref>。2015年6月，他離開Mozilla<ref>{{Cite web|title = New Adventure|url = http://andreasgal.com/2015/06/05/new-adventure/|website = Andreas Gal|accessdate = 2015-06-05}}</ref>，與Firefox OS團隊的另外兩名成員共同創立了[[物聯網|物聯網]]初創公司Silk Labs<ref>{{Cite web |title = Mozilla CTO Andreas Gal Leaving for IoT Startup |url = http://www.internetnews.com/blog/skerner/mozilla-cto-andreas-gal-leaving-for-iot-startup.html |publisher = InternetNews |accessdate = 2015-06-05}}</ref>。同年，加爾成為[[Acadine_Technologies|Acadine Technologies]]顧問，這是由Mozilla公司前總裁[[宮力|宮力]]新成立的一家初創公司，旨在開發以Firefox OS為基礎的系統<ref name="torch">{{Cite web|url = http://www.cnet.com/news/startup-acadine-picks-up-the-torch-for-mozillas-troubled-firefox-os/|title = Startup picks up the torch for troubled Firefox OS|last = Shankland|first = Stephen|date = 2015-12-10|website = CNET|accessdate = 2015-12-12}}</ref>。
