'''安大略省'''，位于[[加拿大|加拿大]]东部，该省面积1,076,395 平方公里，按面积为加拿大第二大省份。该省西邻[[曼尼托巴省|曼尼托巴省]]，北邻[[哈得逊湾|哈得逊湾]]，东邻[[魁北克省|魁北克省]]，南邻[[美国|美国]][[明尼苏达州|明尼苏达州]]、[[密歇根州|密歇根州]]、[[俄亥俄州|俄亥俄州]]、[[宾西法尼亚州|宾西法尼亚州]]、[[纽约州|纽约州]]。

== 地貌 ==
[[安大略省|安大略省]]可分为三个地理区域：
*北部（也即除南部以外的大部分地区）为高地，属于[[加拿大地盾|加拿大地盾]]；
*“东北部”为[[哈得逊湾|哈得逊湾]]沿岸低地；
*南部为伊利湖和安大略湖沿岸，属于五大湖－圣劳伦斯河谷地。
该省最高点位于{{le|伊示巴蒂那山脊|Ishpatina Ridge}}（Ishpatina Ridge），海拔693米。在南部，[[达弗林县|达弗林县]]和[[格雷縣_(安大略省)|格雷縣]]海拔最高，达到430~540米；毗邻的[[威灵顿县|威灵顿县]]与{{le|滑铁卢县|Waterloo County, Ontario}}也有300~400米。

主要河流有[[渥太华河|渥太华河]]、[[圣劳伦斯河|圣劳伦斯河]]、[[尼亚加拉河|尼亚加拉河]]、[[奥尔巴尼河|奥尔巴尼河]]等。该省最长的河流为[[奥尔巴尼河|奥尔巴尼河]]，长约980 km。[[五大湖|五大湖]]中的[[苏必利尔湖|苏必利尔湖]]（Lake Superior）、[[休伦湖|休伦湖]]（Lake Huron）、[[伊利湖|伊利湖]]（Lake Erie）和[[安大略湖|安大略湖]]（Lake Ontario）位于该省与[[美国|美国]]的交界处。尼亚加拉河上的[[尼亚加拉瀑布|尼亚加拉瀑布]]为著名旅游景点。

== 气候 ==
安大略省有3个主要的气候带。安大略省西南部与中南部为[[湿润大陆性气候|湿润大陆性气候]]。安大略省北部为[[亚寒带|亚寒带]]气候。

== 人口 ==
安大略省是加拿大人口最多的省。该省的最大城市[[多伦多|多伦多]]，也是该省的[[省会|省会]]。加拿大[[首都|首都]][[渥太华|渥太华]]也位于该省.该省94%的人口位于[[南安大略|南安大略]]，为加拿大人口密度最高的地区之一。北部人口相对稀少。
