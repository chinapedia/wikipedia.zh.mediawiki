'''安庆市'''，又名'''宜城'''，是[[中华人民共和国|中华人民共和国]][[安徽省|安徽省]]下辖的[[地级市|地级市]]，位于安徽省西南部，[[长江|长江]]下游北岸。长江沿岸著名的港口城市，中国民族工业的发源地；历史悠久，二千多年前为皖国，安徽省简称“'''皖'''”即源于此。[[中国历史文化名城|中国历史文化名城]]、[[中国优秀旅游城市|中国优秀旅游城市]]、[[国家园林城市|国家园林城市]]，有中国“[[黄梅戏|黄梅戏]]乡”之称。自清[[康熙|康熙]]六年（1667年）安徽建省始，直至中共建政初的二百多年里，安庆一直是[[安徽省|安徽省]]的[[省会|省会]]。

安庆为安徽省[[区域中心城市|区域中心城市]]，是安徽省皖西南政治、经济、文化、科教、交通和航运中心，是皖赣鄂三省交界处全国重要的[[交通枢纽|交通枢纽]]和军事战略要地。长江沿岸重要的滨江港口城市，长三角西门户，[[长江中游城市群|长江中游城市群]]重要枢纽，皖南国际旅游文化示范区重点城市，[[皖江城市带承接产业转移示范区|皖江城市带承接产业转移示范区]]核心城市，地理位置重要，有“万里长江此封喉，吴楚分疆第一洲”之喻。

== 释名 ==
因[[宋朝|宋朝]]当地两支驻军「[[同安军|同安军]]」和「[[德庆军|德庆军]]」而得名，又一说[[南宋|南宋]][[绍兴|绍兴]]十七年（1147年）改舒州德庆军为舒州安庆军，“安庆”自此得名（含“平安吉庆”之意）。安庆城始建于公元1217年，至今已有近800年的历史。东晋诗人[[郭璞|郭璞]]曾称“此地宜城”，故安庆又别名“宜城”。

== 历史 ==
安庆历史悠久，考古工作者曾在此发现过多处[[新石器时期|新石器时期]]文化遗址，其中比较著名的有[[张四墩遗址|张四墩遗址]]、[[薛家岗遗址|薛家岗遗址]]<ref>{{cite news |url=http://www.china.com.cn/aboutchina/data/wwbhdw/txt/2006-12/12/content_7495497.htm |title=薛家岗遗址简介|publisher=中国网}}</ref>、[[孙家城遗址|孙家城遗址]]等新石器文化遗址。

[[先秦时期|先秦时期]]，周天子曾分封[[皖国|皖国]]，地域位于现在的安庆市潜山县<ref>[http://www.aq365.com/html/200906/29/050601830.htm 古皖国封地——潜山]</ref>，这也是安徽简称“[[皖|皖]]”的由来。[[春秋|春秋]]时市境地属桐、皖二国。皖、桐先后为楚灭，属楚国。后属吴国。[[战国|战国]]时，复属楚国。

[[秦汉|秦汉]]时期，江北重镇'''皖城'''位于今[[潜山|潜山]]。[[秦朝|秦]]统一后，属[[九江郡|九江郡]]。[[西汉|西汉]]初，改九江郡为[[淮南国|淮南国]]。[[汉文帝|文帝]][[前元|前元]]十六年（前164年），分淮南国，立淮南、[[衡山国|衡山]]、[[廬江國|庐江]]3国，属衡山国。后衡山国除，为[[廬江郡_(秦朝)|庐江郡]]。[[汉武帝|武帝]]分全国为13[[刺史部|刺史部]]（州），隶属[[揚州刺史部|扬州刺史部]]庐江郡[[舒县|舒县]]、[[皖縣|皖县]]，直到[[东汉|东汉]]。

[[三国时期|三国时期]]，[[曹魏|曹魏]]和[[东吴|东吴]]在此多次交兵。初属魏，后属吴。舒县废，属吴[[庐江郡|庐江郡]][[皖县|皖县]]（[[郡治|郡治]]）。

[[魏晋南北朝|魏晋南北朝]]时期，在今天的安庆市区和怀宁县山口镇（现已划归安庆市）附近，曾先后修筑了[[吕蒙|吕蒙]]城和皖口城，是[[长江|长江]]北岸的军事要塞。后在战争结束后被废弃。[[西晋|西晋]]平吴，复置舒县，属庐江郡舒县及皖县。[[永嘉_(西晋)|永嘉]]末（313年），皖县废。[[东晋|东晋]][[义熙_(东晋)|义熙]]年间（405—418年），于皖县旧地立怀宁县（怀宁之名始见于此，县治在今潜山县）及新冶县（今望江县），并置[[晋熙郡_(义熙)|晋熙郡]]。分属庐江郡舒县及晋熙郡怀宁县（郡治）。[[刘宋|南朝宋]]初，于舒县侨立阴安县，隶晋熙郡。宋、[[南齐|齐]]时属晋熙郡阴安县、怀宁县（宋时郡治）。[[梁_(南朝)|梁]]置[[豫州_(南梁)|豫州]]，复改晋州，治晋熙郡怀宁。[[北齐|北齐]]以晋州为[[江州_(北齐)|江州]]，[[陳_(南朝)|陈]]又为晋州。属晋熙郡怀宁县、枞阳郡阴安县。

[[隋朝|隋]][[开皇|开皇]]初，改晋熙郡为[[熙州_(隋朝)|熙州]]，领怀宁、宿松、太湖、望江、同安5县。[[大业_(年号)|大业]]三年（607年），熙州改为[[同安郡|同安郡]]，怀宁仍为郡治。分属同安郡同安县与怀宁县。[[唐朝|唐]][[武德|武德]]九年（626年），改同安郡为东安州，旋为[[舒州_(唐朝)|舒州]]。[[天宝_(唐朝)|天宝]]元年（742年）又改为同安郡。[[至德_(唐朝)|至德]]二载（757年），更名[[盛唐郡|盛唐郡]]（同安县同时改名桐城县，县治于开元年间移至今桐城县城），第二年复名舒州。属舒州（同安郡）桐城县、怀宁县。

[[五代十国|五代十国]]时，先后属[[杨吴|吴]]、[[南唐|南唐]]、[[后周|后周]]，仍为舒州桐城县及怀宁县辖地。

[[北宋|北宋]][[建隆|建隆]]元年（960年），舒州（同安郡）团练州升为防御使。[[政和_(年号)|政和]]五年（1115年），置舒州德庆军。[[南宋|南宋]][[紹興_(南宋)|绍兴]]十七年（1147年），改为舒州安庆军（因与[[德庆府|德庆府]]重名，故取同安郡旧称之“安”，易名“安庆”。安庆之名始于此）。[[慶元_(年號)|庆元]]元年（1195年），升为[[安庆府|安庆府]]，府治在今[[潜山|潜山]]县城，领怀宁、桐城、太湖、宿松、望江5县。市境地属安庆府桐城县、怀宁县。[[嘉定_(年号)|嘉定]]十年（1217年）四月，金军南下破[[光州|光州]]（今河南[[潢川县|潢川]]）。安庆知府黄干奏请朝廷，在“盛唐湾宜城渡之阴”即今城区所在地建筑新城，以备战守。此为安庆建城之始。同时，安庆府治从怀宁县（在今潜山县）移于安庆新城，怀宁县治亦迁皖口（今怀宁县山口镇）。后由于金兵威胁减轻，建城时辍时续。[[宋理宗|宋理宗]][[绍定|绍定]]四年（1231年），安庆知府赵某（佚名）第二次筑安庆城，未竣工又停建。[[端平|端平]]三年（1236年），蒙古军南进，府治移罗刹洲（今[[池州市|池州市]][[贵池区|贵池区]]境内），又移杨槎洲。府城渐荒墟。[[景定|景定]]元年（1260年）三月，为防御蒙古军，沿江制置大使[[馬光祖|马光祖]]复筑安庆城，怀宁县始迁附郭。安庆城从此府、县同城而治，地域全属怀宁县，一直延续到清末。 安庆城从建初到现在已有770多年的历史了<ref>[http://www.anqing.gov.cn/include/web_view.php?ty=7&id=690 安庆市历史沿革]</ref>。[[元朝|元]][[至元_(忽必烈)|至元]]十三年（1276年），改安庆府为安庆路，立安抚司。次年改为总管府（治怀宁县），属蕲黄宣慰司。二十三年（1286年），罢宣慰司，直属河南江北行中书省。[[至治_(元朝)|至治]]三年（1323年），以怀宁县西部的清朝、玉照两乡立潜山县，治怀宁旧城。从此怀宁县境域相对稳定。安庆路领怀宁、桐城、太湖、宿松、望江、潜山6县。元朝后期，爆发[[紅巾軍|红巾军起义]]，元军倚安庆城为屏障。[[至正|至正]]十三年（1353年），行中书省于淮东改宣慰司为都元帅府，治淮西。以宣慰副使佥都元帅府事分兵守安庆城。 

至正二十一年（1361年）八月，[[朱元璋|朱元璋]]改安庆路为宁江府，次年四月改宁江府为安庆府。明初直隶南京（又称[[南直隶|南直隶]]）。[[天顺_(明朝)|天顺]]（1457—1464年）时，在今城区境内，安庆府设10坊。明末，[[张献忠|张献忠]]起义军逼近安庆，明廷为加强安庆的军事力量，[[崇祯|崇祯]]十年（1637年）七月，从应天10府中划安庆、庐州、池州、太平4府别设巡抚（简称安庐巡抚），驻节安庆城。这是安庆第一次设置巡抚，其领辖范围除安庆等4府外，还包括河南光州（今潢川县）的光山、固始、湖广[[蕲州|蕲州]]（今湖北[[蕲春县|蕲春县]]）的[[武穴市|广济]]、[[黄梅县|黄梅]]、江西德化（今[[九江市|九江市]]）的湖口等地。

[[顺治|顺治]]二年（1645年）因明制，仍立安庐巡抚，驻节安庆城。五年，巡抚移驻池州。六年，归并操江，称操抚。领上下江防事的操江先驻[[南京市|江宁]]（今南京市），至此移驻安庆。[[康熙|康熙]]元年（1662年），操江军务归并[[兩江總督|两江总督]]（驻江宁），遂专设巡抚驻安庆。清初，安庆府隶属[[江南布政使司|江南布政使司]]（俗称江南省）。顺治十八年（1661年），江南布政使司分为左右二司，均治江宁。安庆隶江南左布政使司。康熙六年（1667年）改江南左布政使司为安徽布政使司（安徽省），仍寄治江宁，此为安徽省建置之始。[[乾隆|乾隆]]二十五年（1760年），安徽布政使司自江宁移治安庆，从此安庆正式为省治（省会）。[[咸丰_(年号)|咸丰]]三年（1853年）[[太平天国|太平军]]攻占安庆，省治迁庐州府（合肥市）。咸丰十一年（1861年），[[湘军|湘军]]攻下安庆，巡抚衙门迁回安庆，湘军司令部短暫搬遷至此。[[同治|同治]]元年（1862年）迁回安庆。

民国元年（1912年）1月1日，中华民国成立，安徽首任都督兼民政长是同盟会员[[柏文蔚|柏文蔚]]。1938年1月，[[日本|日本]]侵略军攻进安庆，安徽省政府先期迁往[[六安|六安]]、[[金寨|金寨]]等地。故从乾隆二十五年起到1937年止，安庆一直是安徽省会，前后共有170多年的历史。

从抗战胜利时起到[[中华人民共和国|中华人民共和国]]成立前夕，安庆是[[国民政府|国民政府]]安庆专员公署和怀宁县治所在地。1949年4月，[[解放军|解放军]]占领安庆。1988年地、市合併。安庆遂辖周围八县，桐城县于1996年8月20日经民政部批复（民行批[1996]59号）撤销，设立县级桐城市。2016年1月4日，枞阳县划归铜陵市。2018年8月3日经国务院批准撤销潜山县，设立县级潜山市。

== 地理 ==
安庆北依[[大别山|大别山]]，与[[六安市|六安市]]、[[合肥市|合肥市]]、[[铜陵市|铜陵市]]相接；东南濒长江，与[[池州市|池州市]]、[[江西省|江西省]][[九江市|九江市]]为邻；西连[[湖北省|湖北省]][[黄冈市|黄冈市]]。全市总面积共1.53万[[平方公里|平方公里]]，其中市区面积820平方公里。

安庆地理位置十分重要，具有相当的战略意义，[[曾国藩|曾国藩]]就曾经认为'''“安庆之得失，定乾坤之转与不转。安庆关系淮南之全局，为克复金陵之张本。”'''<ref>[http://www.hnrku.net.cn/zgfyj/kNews_typenews.asp?id=26 曾国藩军事战略辩证法论略] {{webarchive|url=https://web.archive.org/web/20110704122056/http://www.hnrku.net.cn/zgfyj/kNews_typenews.asp?id=26 |date=2011-07-04 }}</ref>

属[[亚热带季风气候|亚热带季风气候]]，四季分明、雨热同期、光照充足。1月平均气温4.3℃，7月平均气温29.2℃，全年均温17.2℃。极端最高气温40.9℃（2003年8月2日），极端最低气温-12.5℃（1969年2月5日）。降水量1200-1500毫米，日照时数1800小时左右。

根据2010年[[第六次全国人口普查|第六次全国人口普查]]，全市[[常住人口|常住人口]]为531.1万人<ref>{{cite book |author=安庆市统计局|title=《安庆市2010年第六次全国人口普查主要数据公报》}}</ref>，同[[第五次全国人口普查|第五次全国人口普查]]相比，十年共增加13.1万人，增长2.5%，年平均增长0.25%。其中，男性人口为268.5万人，占50.56%；女性人口为262.6万人，占49.44%。总人口性别比（以女性为100）为102.26。0－14岁人口为87.8万人，占16.53%；15－64岁人口为388.4万人，占73.13%；65岁及以上人口为54.9万人，占10.34%。居住在城镇的人口为195.3万人，占36.77%，居住在乡村的人口为335.8万人，占63.23%。

== 经济 ==
安庆市初步形成了石油化工、轻纺食品、机械及汽车零部件、商贸流通四大支柱产业，以及农业、文化旅游、建材器械、医药、船舶、航空、电子商务等新兴环保、高科技、服务产业。国务院批复的国家级《皖江城市带承接产业转移示范区规划》中，明确定位：安庆，是全省三大区域中心城市之一、三大产业组团之一、六大交通枢纽之一、现代化历史文化名城，打造带动皖西南、辐射皖赣鄂交界地区的区域中心城市，建设全国重要的石化和轻纺产业基地。2013年，安庆市以GDP1500亿元跻身安徽省前三甲，跨越千亿元，创历史新高

工业以机械制造、纺织、石油化工为主。[[国有企业|国有企业]]发展缓慢，[[民营企业|民营企业]]相对发展较快，外资正在扩大投资。对外主要向[[日本|日本]]和[[美国|美国]]出口。

安庆市的经济在中国现代化建设中取得了长足的发展，现有大型石化企业[[安庆石化|安庆石化]]，有全国纺企业的先进单位[[华茂股份有限公司|华茂股份有限公司]]，更有众多的中外合资企业，[[安庆大电厂|安庆大电厂]]一期二期均已建成，[[安庆公路长江大桥|安庆公路长江大桥]]和安庆铁路长江大桥已建成，安庆至南京高速铁路开通，安庆至合肥、安庆至九江高速铁路正在建设。
