'''安文镇'''，是[[中国|中国]][[浙江省|浙江省]][[金华市|金华市]][[磐安县|磐安县]]下辖的一个[[镇|镇]]，为磐安县[[县城|县城]]所在地<ref>{{Cite book|title=当代中国城市-区域：权力·空间·制度研究丛书 中国市制的历史演变与当代改革|last=刘君德，范今朝著|first=|publisher=南京：东南大学出版社|year=2015年7月|isbn=978-7-5641-5604-6|location=|pages=267}}</ref>。

东连[[墨林鄉_(磐安縣)|墨林乡]]和[[大盘镇|大盘镇]]，南邻盘峰、双峰、[[深泽乡|深泽乡]]，西北与东阳市马宅镇接壤，总面积90.03平方公里，人口33320人。

== 沿革  ==
安文镇古名'''惠化'''，又有'''瑞山'''、'''文溪'''等名。地处东阳江支流南江上游文溪两岸，位于磐安县西边沿。镇区面积5．2平方公里，四周群山环抱，文溪、根溪、东溪等于此汇合后，穿镇而北向，素有“万山管钥”。南宋末朝官[[陆宰|陆宰]]、[[龚茂良|龚茂良]]、[[晏景初|晏景初]]携文书藏其裡而不毁，遂以“平安昌文”之意，名安文。

1935年，安文从[[东阳县|东阳县]]析出，归属于[[大盘山区|大盘山区]]绥靖公署管辖，1939年，磐安县建立，归属[[磐安县|磐安县]]。1949年，中国人民解放军进入，同年10月30日成立磐安县人民政府，安文始为县城。1950年设[[安文乡|安文乡]]。1958年，改公社。1958年，磐安县撤销，安文隶属东阳县。1983年，磐安县恢复，安文镇复为县城<ref>{{Cite book|title=中华人民共和国地名大词典 第1卷|last=崔乃夫主编|first=|publisher=北京：商务印书馆|year=1998年1月|isbn=7-100-01836-6|location=|pages=1872}}</ref>。

== 产业 ==
安文镇是新兴工业城镇<ref>{{Cite book|title=中国城镇 浙江卷|last=李家鸿主编；《中国城镇·浙江卷》编辑委员会编|first=|publisher=北京：中国城市出版社|year=1991年12月|isbn=7-5074-0607-5|location=|pages=1098}}</ref>。盛产香菇、茶叶。有冶炼、机械、橡胶、丝绸等厂。境内有石印、石龟、石踝、石牛及双溪合流，连木双莺等“四奇八景”。

1953年，安文镇创办火力发电厂。之后当地创办了酒厂、印刷厂、火腿厂。1957年，当地政府[[公私合营|公私合营]]，将铁业合作社、木业合作社合并为机械厂。1958年撤县后，安文一些工厂迁到东阳县。1983年恢复县建制后，工业重新起步。1988年，有镇属工厂39家，村办企业41家，联办企业42家，个体企业212家。县肉制品厂是定点的[[金华火腿|金华火腿]]产品厂，生产的火腿，1987年获[[国家银质奖|国家银质奖]]。此外磐安机械厂、制冷设备厂、磐安电器厂、生物化学厂等都在安文镇<ref name=a>{{Cite book|title=浙江省名镇志|last=浙江省名镇志编纂委员会|first=|publisher=上海：上海书店出版社|year=1991年5月|isbn=7-80569-398-6|location=|pages=599-601}}</ref>。

全镇耕地面积4300亩，林业面积58000亩。水田主种双季水稻，部分种蔬菜和药材，旱地主要种[[小麦|小麦]]、[[大豆|大豆]]、[[玉米|玉米]]、[[番薯|番薯]]。种植的药材有[[白术|白术]]、[[元胡|元胡]]、[[贝母|贝母]]、[[白芍|白芍]]、[[玄参|玄参]]、[[桔梗|桔梗]]、[[黄芪|黄芪]]、[[丹皮|丹皮]]等。安文盛产[[红心李|红心李]]、[[散花梨|散花梨]]、[[水蜜桃|水蜜桃]]、[[端午桃|端午桃]]、[[西瓜|西瓜]]、[[樱桃|樱桃]]、[[杏梅|杏梅]]等水果<ref name=a />。安文镇还有浙江最大的[[银杏|银杏]]基地<ref>{{Cite web|url=http://zjnews.zjol.com.cn/05zjnews/system/2008/10/07/010001525.shtml|title=浙江最大银杏基地很头大 已有两企业要来收购白果-银杏-浙江在线|accessdate=2017-07-03|work=浙江新闻}}</ref>。

安文镇在大盘山山麓的花溪村附近修建花溪风景名胜区、[[大盘山地质公园|大盘山地质公园]]，为浙江著名旅游胜地<ref>{{Cite web|url=http://panews.zjol.com.cn/panews/system/2013/12/04/017389308.shtml|title=平板长溪秀江南——安文镇花溪村--磐安新闻网|accessdate=2017-07-03|work=panews.zjol.com.cn}}</ref>。

== 行政区划 ==
安文镇辖有8个[[社区|社区]]与23个[[行政村|行政村]]，分别为：
*龙山社区、五指社区、上章社区、壶厅社区、市口社区、东溪社区、荷塘社区、康庄社区
*后坞村、羊山头村、联谊村、联进村、溪文村、殊闲村、台口村、小岭村、岭脚村、元岭坑村、根溪村、岩里村、应古村、花溪村、双坑村、上马石村、中田村、上葛村、岭外村、胡村、白云山村、石头村、石贝村。
