'''安德烈·亞歷山德雷斯庫'''（{{lang-en|'''Andrei Alexandrescu'''}}，{{bd|1969年}}），[[罗马尼亚裔美国人|罗马尼亚裔美国人]]，[[C++|C++]]和[[D语言|D语言]]专家。

从[[D语言|D语言]]2.0版本之后，他加入D语言核心开发团队，是继[[沃爾特·布萊特|沃爾特·布萊特]]之后的最主要设计师。他撰写了《D程序设计语言》一书，全面系统地介绍了D语言的设计和实现。

他因通过模版元程序设计实现基于政策设计的开创性成果而名闻遐迩。他撰写了《现代C++设计》一书阐述了他的理念，首次在他编写的程序库[[Loki|Loki]]里加以实现。他在MOJO库里实现了'''move constructors'''。<ref name="movectors">{{cite web |url=http://www.ddj.com/cpp/184403855 |title=Move Constructors |last=Alexandrescu |first=Andrei |date=1 February 2003 |publisher=[[Dr._Dobbs_Journal|Dr. Dobbs Journal]] |accessdate=25 March 2009 |archiveurl=https://web.archive.org/web/20090507104956/http://www.ddj.com/cpp/184403855 |archivedate=2009年5月7日 |deadurl=no }}</ref>

他目前在[[Facebook|Facebook]]公司就职，担任研究科学家。

他于2004年8月入[[美国国籍|美国国籍]]。<ref>[http://www.reddit.com/r/pics/comments/2di6ik/sixteen_years_ago_at_28_i_landed_in_new_york_with/ Reddit.com: "Sixteen years ago, at 28, I landed in New York with $300 to my name. Today I became a US citizen. It's been a wild ride that I hope will go on!"]</ref>

== 教育和职业 ==
亞歷山德雷斯庫1994年7月毕业于布加勒斯理工大学，获得电子工程学士学位。<ref name="resume">[http://erdani.com/resume.html Andrei Alexandrescu: Resumé] {{webarchive|url=https://web.archive.org/web/20110407082008/http://erdani.com/resume.html |date=2011-04-07 }}</ref><ref>[http://accu.informika.ru/events/public/accu0103.htm ACCU Spring Conference 2001] {{webarchive|url=https://web.archive.org/web/20110811025520/http://accu.informika.ru/events/public/accu0103.htm |date=2011-08-11 }}</ref>

1998年9月他在[[C/C++_Users_Journal|C/C++ Users Journal]]上首次发表文章。 从1999年4月至2000年2月在Netzip公司担任程序经理。Netzip公司被RealNetworks公司后，他继续任职到2001年9月。<ref name="resume" />

亞歷山德雷斯庫先后于2003年和2009年获得美国华盛顿大学计算机科学硕士和博士学位 。<ref>[http://accu.org/index.php/conferences/accu_conference_2008/accu2008_speakers  ACCU 2008: Speakers]</ref><ref>[http://accu.org/index.php/conferences/accu_conference_2010/accu2010_speakers ACCU 2010: Speakers]</ref><ref>[http://www.cs.washington.edu/education/grad/RecentPHDGraduates.html Computer Science & Engineering, Recent Ph.D. Graduates (Summer 2009)]. University of Washington.</ref>

近年来，他一直协助[[沃爾特·布萊特|沃爾特·布萊特]]开发[[D语言|D语言]]，于2010年5月出版了《D语言程序设计》。

亞歷山德雷斯庫、[[Herb_Sutter|Herb Sutter]]和{{link-en|斯科特·邁爾斯|Scott Meyers}}三人主持名为 [http://cppandbeyond.com/ C++ and Beyond]的小型年度技术讨论会议。

== 贡献 ==
'''Expected''' 作为C++模版类纳入[[Boost_(C++_libraries)|Boost]]库<ref>{{cite web |author1=Botet |author2=Talbot |title=A proposal to add a utility class to represent expected monad |url=http://www.hyc.io/boost/expected-proposal.pdf |deadurl=yes |archiveurl=https://web.archive.org/web/20140819082937/http://www.hyc.io/boost/expected-proposal.pdf |archivedate=2014-08-19 }}</ref>。Alexandrescu 建议 <ref>{{cite web |last1=Alexandrescu|title=Systematic Error Handling in C++ |url = http://channel9.msdn.com/Shows/Going+Deep/C-and-Beyond-2012-Andrei-Alexandrescu-Systematic-Error-Handling-in-C}}</ref>使用 Expected<T> 作为一个类用于返回值用来表示要么包含一个类型T或者例外防止其被创建。这是对于要么返回代码要么显示调用例外的一项改进。Expected<T>有如下好处：
* 关联计算目标和错误关联
* 自然构成多种例外
* 在错误处理和抛出例外两种风格中切换
* Teleportation possible across thread boundaries, across nothrow subsystem boundaries and across time (现在保存, 之后抛出)
* 收集、集合和组合例外

=== 例子 ===
他建议采用

<code>
Expected<int> parseInt(const string&);  // Returns an expected int: either an int or an exception
</code>

来代替如下的函数声明。

<code>
int parseInt(const string&);  // Returns 0 on error and sets errno
</code>

或者

<code>
int parseInt(const string&);  // Throws invalid_input or overflow
</code>
