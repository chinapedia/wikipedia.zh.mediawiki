'''安托万·沃克'''（{{lang-en|'''Antoine Walker'''}}, {{bd|1976年|8月12日|catIdx=W沃}}），已退役[[美国|美国]]职业[[篮球|篮球]]运动员，1976年8月12日出生在[[伊利诺伊州|伊利诺伊州]]的[[芝加哥|芝加哥]]，可司职[[大前锋|大前锋]]和[[小前锋|小前锋]]两个位置。曾在五支不同的NBA球队效力过，2006年[[NBA总冠军|NBA总冠军]]球队成员。

== 大学篮球生涯 ==
沃克在伊利诺伊州的Dolton长大，最后进入了芝加哥的Mount Carmel高中。他是肯塔基大学以外卡身份取得1996年NCAA总冠军的主力核心球员。在大学的第一年就当选东南分区(SEC)的最有价值球员。二年级时荣获东南分区第一阵容和NCAA全明星。在肯塔基大学打了2年大学联赛后沃克与1996年决定参加那一年的NBA选秀。

== NBA生涯 ==
在[[1996年NBA选秀|1996年NBA选秀]]中沃克被[[波士顿凯尔特人队|波士顿凯尔特人队]]以第一轮第六顺位摘下，其后与[[保罗·皮尔斯|保罗·皮尔斯]]成为该队两大领袖之一。2003年新赛季开始前，沃克被交易到[[达拉斯小牛|达拉斯小牛]]，在小牛沃克没有得到施展的空间，2004年8月8日，沃克和[[托尼·德尔克|托尼·德尔克]]被送往[[亚特兰大鹰队|亚特兰大鹰队]]以换取[[杰森·特里|杰森·特里]]、[[阿伦·亨德森|阿伦·亨德森]]和一个选秀权。

2005年8月2日，在一场涉及到5队13名球员的NBA史上最大规模的球员交易中，沃克从凯尔特人换往迈阿密热火。2006年6月20日他随热火赢得了其职业生涯首个总冠军，在总决赛系列赛中安东尼·沃克场均贡献了14分和11个篮板。

2007年10月24日，沃克与替补中锋[[迈克尔·多列亚克|迈克尔·多列亚克]]，时常受伤的前锋[[韦恩·西米昂|韦恩·西米昂]]以及下一年的一个首轮选秀权和明尼苏达森林狼的前热火队球员[[里奇·戴维斯|里奇·戴维斯]]和老中锋[[马克·布朗特|马克·布朗特]]进行了交换。

2008年夏，沃克被交易至[[孟菲斯灰熊|孟菲斯灰熊]]。

沃克是灵活的大个球员，热衷于投射[[三分球|三分球]]。他在每场获取19分左右的同时，也能抢下8.3个[[篮板球|篮板球]]，并有3.9次[[助攻|助攻]]，颇为全能。

== NBA记录 ==
沃克持有两项NBA记录。在2002年季候赛首轮对阵[[费城76人|费城76人]]的系列赛中，沃克在一场比赛的第二节射中了6个三分球，是NBA季后赛单节3分球命中数最多的球员；另一项纪录则不是很光彩，在2002-03赛季对阵[[克里夫兰骑士|克里夫兰骑士]]的比赛中，沃克的3分球10投0中，成为单场比赛连续3分不中次数最多的球员。<ref>{{cite web |url=http://www.basketballreference.com/teams/boxscore.htm?yr=2002&b=20021221&tm=cle |title=存档副本 |accessdate=2006-11-24 |deadurl=yes |archiveurl=https://web.archive.org/web/20070210025145/http://www.basketballreference.com/teams/boxscore.htm?yr=2002&b=20021221&tm=cle |archivedate=2007-02-10 }}</ref>
