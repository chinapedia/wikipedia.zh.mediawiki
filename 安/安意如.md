'''安意如'''（{{bd|1984年|6月20日}}），本名'''张莉'''，女，生于[[安徽|安徽]][[宣城|宣城]]的一个工人家庭<ref>{{Cite web|url = http://www.hi.chinanews.com/hnnew/2015-04-30/4_48362.html|title = 畅销书作家安意如：其实我是很帅气的人|accessdate = 2015-08-03|author = |date = 2015年4月30日|publisher = 中国新闻网|language = zh-cn}}</ref>，患有先天轻度[[脑瘫|脑瘫]]。[[中国|中国]]当代[[作家|作家]]，现居[[北京|北京]]。因创作诗词评赏“漫漫古典情”系列成名。作品以[[中国|中国]]古典[[诗|诗]][[詞_(文學)|词]]鉴赏以及爱情故事为主。

==人物争议==
2006年底，安意如出版的《人生若只如初见》、《思无邪》两本古典诗词赏析作品被指大量抄袭网友“江湖夜雨”的图书和网文，引发媒体及网民讨论<ref>{{Cite web|url = http://news.artxun.com/xiqu-1485-7422564.shtml|title = 古典文学赏析丛书抄袭多多误人子弟|accessdate = 2015-08-03|author = |date = 2009-07-21|publisher = 博宝艺术网|language = zh-cn}}</ref>。2007年1月14日，安意如回应自己并没有抄袭，“我借鉴和引用别人的文字没有标明出处，这是我的疏忽大意。可能因为我出身于网络，版权意识比较淡薄吧。”<ref>{{Cite web|url = http://book.sina.com.cn/news/c/2007-01-16/1203209140.shtml|title = 安意如回应抄袭事件坚称自己是“借鉴”|accessdate = 2015-08-03|author = |date = 2007年1月16日|publisher = 新浪读书|language = zh-cn}}</ref>，并在博客中向“江湖夜雨”道歉。其之后出版的部分作品也被指抄袭。
