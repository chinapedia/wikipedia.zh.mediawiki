'''安陽王'''（{{vie|v='''An Dương Vương'''|hn=安陽王}}）本名'''蜀泮'''（{{vie|v='''Thục Phán'''|hn=蜀泮}}），原是[[古蜀|古蜀]]的王子，乃為[[鱉靈|鱉靈]]的後裔。在[[秦國|秦國]]於西元前316年滅[[古蜀|古蜀]]之後，輾轉到達現在[[越南|越南]]北部，建立[[甌貉國|甌貉國]]，自稱為安陽王，建都於[[古螺|古螺]]，乃今[[河内|河内]]近郊的[[東英縣|東英縣]]。

==滅文郎而建國的傳說==
根據[[越南|越南]]官修史書《[[大越史記全書|大越史記全書]]》和《[[欽定越史通鑑綱目|欽定越史通鑑綱目]]》的說法，蜀泮本為[[中國|中國]][[戰國|戰國]]時代[[古蜀|蜀國]]的王子。因先世蜀王要求與[[雄王|雄王]]通婚遭拒，蜀國因此與文郎國結為世仇。在蜀國被[[秦國|秦國]]滅亡之後，蜀泮率軍南下，到達現在[[廣西|廣西]]和[[雲南|雲南]]，隨後輾轉南下，進攻世仇[[文郎國|文郎國]]。最後一代[[雄王|雄王]]自恃強勇，不為兵備，在蜀泮進攻的時候因酒醉墜井而死。於是蜀泮自立為王，稱安陽王，改國號為[[甌貉|甌貉]]（现代學者認為“甌貉”就是中國古籍中的“[[甌駱|甌駱]]”，“貉”又可以寫作“雒”）。安陽王在[[越裳|越裳]]地區建立新都[[思龍城|思龍城]]，因其盤旋如螺形，所以又稱[[古螺城|古螺城]]<ref>《大越史記全書》（外紀卷之一）：甲辰元年〈周赧王五十八年〉（公元前二五七年）。王既併文郎國，改國號曰甌貉國。初王屢興兵攻雄王。雄王兵強將勇，王屢敗。雄王謂王曰：「我有神力，蜀不畏乎。」遂廢武備而不修，需酒食以為樂。蜀軍逼近，猶沈醉未醒，乃吐血堕井薨，其衆倒戈降蜀。王於是築城于越裳，廣千丈，盤旋如螺形，故號螺城。又名思龍城〈唐人呼曰崑崙城，謂其城最高也〉。其城築畢旋崩，王患之，乃齋戒禱于天地山川神祇，再興功築之。</ref>。

== 稱王時期的傳說 ==
關於安陽王統治時期的事蹟，越南史書中的記載甚少。越南歷史學者則稱安陽王是一位偉大的君主。在他在位期間，製造了[[靈弩|靈弩]]，抵禦了[[秦朝|秦朝]]幾次的進攻。

西元前210年，秦朝[[任囂|任囂]]、[[趙佗|趙佗]]率軍入侵。趙佗在[[北江|北江]]的[[僊遊山|僊遊山]]與安陽王交戰，安陽王用靈弩擊退了趙佗。趙佗退守[[武寧山|武寧山]]，遣使講和。雙方約定以[[平江|平江]]為界，北為趙佗界，南為安陽王界。趙佗子[[趙仲始|趙仲始]]娶安陽王的女兒[[媚珠|媚珠]]為妻，入贅安陽王家。仲始竊取了[[靈弩|靈弩]]，用假弩將其掉包。仲始以省親之名義北歸，臨行前對媚珠說：「夫婦恩情不可相忘，如兩國失和，南北隔別，我來到此，如何得相見。」媚珠說：「妾有鵞毛錦褥常附於身，到處拔毛置岐路，以示之。」仲始回到北方後，將這番話告訴了趙佗。

西元前208年，[[任囂|任囂]]病逝，臨死前囑令[[趙佗|趙佗]]掌管嶺南事務。趙佗在嶺南發兵拒險，成為一路割據勢力。趙佗發兵南下進攻安陽王。安陽王不知[[靈弩|靈弩]]被竊，為趙佗所敗，與女兒[[媚珠|媚珠]]同乘一馬逃跑。趙仲始認准鵝毛緊追不捨，安陽王逃至海濱，下馬，在湧出海面的[[金龜|金龜]]的協助下，持七寸文犀走入海中。

安陽王在海濱下馬之時，因金龜指稱乘坐馬後的人「是賊也，蓋殺之」，於是揮劍斬殺了媚珠。媚珠的血流入海中，化為了明珠。仲始痛哭不已，將媚珠的屍體歸葬[[螺城|螺城]]，其屍體化為玉石。後來因思念媚珠，投井而死。

== 中國古籍中的記載 ==
中國古籍中最早所記安陽王相關事蹟見於《[[史記索引|史記索引]]》及《[[水經注|水經注]]》所引用的《廣州記》及《交州外域記》。
《史记·南越传》《索引》引《广州记》曰：“交趾有骆田，仰潮水上下，人食其田，名为骆人，有骆王骆侯，诸仙自名为骆将，铜印青绶，即今之令长也。后蜀王子将兵讨骆侯，自称为安阳王，治封溪县，后南越王尉他攻破安阳王，令二使典主[[交趾郡|交趾]]、[[九真郡|九真]]二郡人。”
《水经·叶榆水注》引《交州外域记》云：“交趾昔未有郡县之时，土地有骆田……”，与《广州记》略同。

== 近現代學術界對甌雒國的闡釋 ==
關於甌雒國的存在年代，及其與秦朝政府的關係，後代學者有不同說法。

[[法國|法國]][[漢學家|漢學家]][[鄂盧梭|鄂盧梭]]認為，蜀王子（蜀泮）在公元前210年[[秦始皇|秦始皇]]死時，乘中國內亂，奪取[[象郡_(秦代)|象郡]]地區（鄂盧梭認定象郡在越南，並指出該地雖為中國秦代領土，實為封建土酋所治。蜀王子套取該地後自號為安陽王。但不久被趙佗所反擊，到公元前207年將之消滅。<ref>鄂盧梭《秦代初平南越考》第三章《輯子史文》，馮承鈞譯，臺灣商務印書館版，64─65頁。</ref>

[[周振鶴|周振鶴]]對鄂盧梭的說法提出質疑，認為象郡既非鄂盧梭所認為的[[日南郡|日南郡]]（鄂盧梭認為秦象郡即漢代之日南郡，周振鶴已斥其非），而以秦之強大，亦不至於為一蜀王子所敗，且若安陽國僅存在三年（鄂盧梭認為是前210年至前207年），其事蹟亦不可能成為傳說流傳後世千百年後。<ref>周振鶴《西漢政區地理》，人民出版社，1987：8:1-203.</ref>周氏認為，安陽國之存在時間絕不像鄂盧梭所認為的那樣絕對，要之，根據《史記》，南越王趙佗滅安陽國併其地為交趾、九真二郡時間當在[[呂雉|呂后]]、[[漢文帝|文帝]]間<ref>《史記·南越傳》：“秦已破滅，佗即擊併桂林、象郡。”“歲餘，高后崩，（漢朝）即罷兵。佗因此以兵威邊，財物賂遺閩越、西甌、駱、役屬焉，東西萬餘里。”</ref>。

[[呂士朋|呂士朋]]認為，甌雒國當於[[戰國時代|戰國時代]]末期建國於[[紅河|紅河]]下游的[[紅河三角洲|平原]]一帶。秦軍南下設象郡後，安陽王退據一隅，情況像部落酋長，依舊統治一小地區。呂士朋指出秦朝對象郡，終未能施行直接統治，而是間接統治，方式是把越人的統治權，委託給當地若干土酋手中。秦朝滅亡時，安陽王便捲土重來，在象郡擴張勢力，成為當地較強大的領導者。直到趙佗來攻時才被消滅。<ref>呂士朋《北屬時期的越南》第一章《上古時期的中越關係》，香港中文大學新亞研究所東南亞研究室刋，17頁。</ref>

[[郭振鐸|郭振鐸]]、[[張笑梅|張笑梅]]則對古籍裡「甌雒國」的存在表示質疑，認為它本身具有傳疑性質，可能並非現代意義的「真正國家」，只是初具政權組織的部落聯盟。另外，又提出可以把「甌雒」理解為「甌」指「西甌」（位於現時廣西西南、越北的越族），而「雒」則指「[[雒越|雒越]]」（位於現時越南西北，屬古[[百越族|百越族]]一支），而蜀泮則是西甌及雒越各部落的領袖。然而因生產力尚屬低下，在趙佗的入侵下遂將之平定。<ref>郭振鐸、張笑梅《越南通史》第二編第三章第三節，中國人民大學出版社版，128─133頁。</ref>

== 相關傳說 ==
*[[皐魯|皐魯]]：傳說是安陽王時期的能工巧匠，為安陽王造[[靈弩|靈弩]]。
*[[李翁仲|李翁仲]]：傳說中安陽王時期來自[[交趾|交趾]] 、獲得[[秦始皇|秦始皇]]徵召出征[[匈奴|匈奴]]的一位巨人。其後李翁仲退役，回鄉歸隱，秦始皇欲再召回翁仲，在越南古代小說《[[粵甸幽靈集|粵甸幽靈集]]》及《[[嶺南摭怪|嶺南摭怪]]》裡，便有所描述。<ref>《粵甸幽靈集錄·歷代人臣·校尉英烈威猛輔信大王》，收錄於《越南漢文小說叢刊》第二輯第二冊《神話傳說類》，臺灣學生書局版，第26-27頁；《嶺南摭怪·李翁仲傳》，收錄於《嶺南摭怪等史料三種》，中州古籍出版社版，24頁。</ref>據[[中國|中國]]學者[[戴可來|戴可來]]指出，這則故事是取材自中國傳說裡曾威振匈奴的南海人[[阮翁仲|阮翁仲]]事跡，後被搬到越南而成為「李翁仲」。<ref>戴可來《關於<嶺南摭怪>的編者、版本和內容》，附錄於《嶺南摭怪等史料三種》，中州古籍出版社版，265頁。</ref>
