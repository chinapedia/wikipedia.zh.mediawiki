'''安·多吉·雪巴'''（{{lang-en|Ang Dorje Sherpa}}；{{bd|1970年}}）是一位來自[[尼泊爾|尼泊爾]][[潘波崎|潘波崎]]的[[夏爾巴人|雪巴族]]登山嚮導、登山家及搬運工，登頂[[珠穆朗瑪峰|珠穆朗瑪峰]]18次。他知名於在造成8名登山者死亡的[[1996年聖母峰事故|1996年聖母峰事故]]中，擔任[[羅勃·霍爾|羅勃·霍爾]]的{{link-en|冒險顧問登山隊|Adventure Consultants}}登山嚮導，這是目前珠穆朗瑪死亡人數第三多的山難。

== 早年生涯 ==
安·多吉於1970年出生<ref name="Outside">{{cite web |url=http://www.outsideonline.com/adventure-travel/asia/nepal/mount-everest/Impossible-to-Forget.html |title=Impossible To Forget |publisher=Outside Online |date=August 15, 2006 |accessdate=2012年7月5日 |author=Sohn, Tim}}</ref>在位於[[尼泊爾|尼泊爾]]珠穆朗瑪峰山坡下[[坤布|坤布谷]]的村莊[[潘波崎|潘波崎]]。他出身於[[喜馬拉雅|喜馬拉雅山脈]]登山家庭，父親尼瑪·丹增·雪巴（{{lang|en|Nima Tenzing Sherpa}}），是英國登山家{{link-en|克里斯·波寧頓|Chris Bonington}}在1970和1980年代的探險隊的登山嚮導<ref name="View">{{cite web |url=http://www.canada.com/ottawacitizen/story.html?id=e3d30d97-d299-4186-9293-5cd03e02072c&k=11749 |title=A view from the top |publisher=Calgary Herald |date=October 17, 2006 |accessdate=2012年7月5日}}</ref>。

安·多吉於12歲時就跟隨父親擔任探險隊的搬運工。安·多吉說「當我小時候時總是想攀登」。因他工作認真，得以在喜馬拉雅地區為數個探險隊從事登山工作<ref name="View" />。 1992年，正值22歲的他首度登頂珠穆朗瑪峰<ref name="everestnews">{{cite web |url=http://www.everestnews.com/angdorje.htm |title=Mt. Everest 2005: Chhuldim Ang Dorje Sherpa aka Ang Dorje Sherpa |publisher=Everest News |accessdate=2012年7月5日}}</ref>。

== 嚮導生涯 ==
安·多吉帶領許多探險隊攀登珠穆朗瑪峰，並經常替[[羅勃·霍爾|羅勃·霍爾]]所創立的{{link-en|冒險顧問登山隊|Adventure Consultants}}服務。在[[1996年聖母峰事故|1996年聖母峰事故]]期間<ref name="high ambitions">{{cite web |url=http://www.highambitions.net/film/climbers.html |title=High Ambitions in the Himalaya |publisher=BeLegato Productions |year=2006 |accessdate=2012年7月5日}}</ref>，他與拉帕·澤林·雪巴（{{lang|en|Lhakpa Tshering Sherpa}}）在惡劣的天氣條件下，大膽設法營救霍爾和其他人，最終有8人喪生。他們爬升900公尺至[[珠穆朗瑪南峰|南峰]]，僅在距離霍爾100公尺處因為難以抵抗暴風雨而停止救援。等待45分鐘後決定撤退。安·多吉事後說：「這很悲傷。非常困難。」<ref name="View" />。此項事蹟後來由[[強·克拉庫爾|強·克拉庫爾]]在其著作《{{link-en|聖母峰之死|Into Thin Air}}》中進行了描述<ref name="Outside" /> 。

安·多吉在協助攀登珠穆朗瑪峰的工作之外，也擔任珠穆朗瑪峰、[[阿空加瓜山|阿空加瓜山]]、[[瑞尼爾山|瑞尼爾山]]、[[吉力馬札羅山|吉力馬札羅山]]與[[島峰|島峰]]的高山嚮導<ref name="adventureconsultants">{{cite web |url=http://www.adventureconsultants.com/adventure/guidesandsherpas/#_Ang_Dorje |title=The Adventure Consultants International Expedition Guiding Team |publisher=Adventure Consultants Limited |accessdate=2012年7月5日}}</ref>。

他在1980年第二度攀登珠穆朗瑪峰時，協助[[印度|印度]]女性登山家{{link-en|巴城德里·帕爾|Bachendri Pal}}成為首度登頂珠穆朗瑪峰的印度女性。他通常以無氧攀登的形式攀爬，很活躍且敏捷。

== 攀登成就 ==
截至2016年止，安·多吉自1992年開始已登頂珠穆朗瑪峰18次（皆在春季及秋季，且都由東南脊路線經{{link-en|南坳|South Col}}）登頂，自1995年起登頂[[卓奧友峰|卓奧友峰]]7次，另外也登頂[[布洛阿特峰|布洛阿特峰]]（1995年）、[[加舒爾布魯木II峰|加舒爾布魯木II峰]]（1997年）與[[阿瑪達布拉姆峰|阿瑪達布拉姆峰]]（1996年，經西南脊路線）<ref name="everestnews" />。

== 個人生涯 ==
安·多吉於2002年在[[珠穆朗瑪峰基地營|珠穆朗瑪峰基地營]]遇見美國[[計算語言學|計算語言學家]]蜜雪兒·格雷戈里（{{lang|en|Michelle Gregory}}）後移民美國。這對伴侶一年後結婚，並與兩個小孩居住於[[華盛頓州|華盛頓州]][[里奇蘭_(華盛頓州)|里奇蘭]]。每年春天回尼泊爾攀登珠穆朗瑪峰，部分原因是可順道探親。他於2012年完成第16次登頂珠穆朗瑪峰<ref name="Outside" /><ref name="spokesman">{{cite web |url=http://www.spokesman.com/stories/2011/jun/05/field-reports-ang-dorje-returns-from-15th-mount/ |title=Field reports: Ang Dorje returns from 15th Mount Everest ascent |publisher=Spokane Spokesman-Review |date=June 5, 2011 |accessdate=2012年7月5日}}</ref>。當他不是在喜馬拉雅山領引導客戶時，安·多吉在華盛頓家附近的風力發電場工作，為一位風力渦輪機技工<ref name="tri-city herald">{{cite web |url=http://www.tri-cityherald.com/2012/06/07/1974005/richland-man-returns-from-mount.html |title=Richland Man Returns From Mount Everest |publisher=Tri-City Herald |date=June 7, 2010 |accessdate=2012年7月5日 |author=O'Neal, Dori}}</ref>。
