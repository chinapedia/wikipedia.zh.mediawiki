'''龜派氣功'''（日文：かめはめ波；英文：Kame Hame Ha），又稱'''神龜衝擊波'''，也有直接翻譯為「卡曼哈曼波」或「卡美哈梅哈」。是鳥山明所創作的日本漫畫《[[七龍珠|七龍珠]]》中的虛構氣功。依據設定，此氣功由劇中一位名為[[龟仙人|龟仙人]]的武術大師所创，需要聚集「氣」來發動，有尾隨性，龜仙人用了50年創造出來，但主角[[孫悟空_(七龍珠)|孫悟空]]只是看了一眼便學會，其後[[克林|克林]]、[[飲茶_(七龍珠)|飲茶]]等人也輕易學會。

== 設計 ==
鳥山明的妻子提議以夏威夷國王的開創者來命名，「{{lang|ja|かめはめ波}}」（[[日语罗马字|羅馬拼音]]：Kamehameha）意即[[卡美哈梅哈一世|卡美哈梅哈]]。

== 劇中使用 ==
=== 架勢 ===
[[File:Kamehameha_DB_scheme.svg|right]]

# {{lang|ja|'''か'''}}　…　在身體前方將兩隻手合成[[花萼|花萼]]状。（此時發音「KA」）
# {{lang|ja|'''め'''}}　…　將花萼狀的兩手放在腰間一側。（此時發音「ME」）
# {{lang|ja|'''は'''}}　…　體內集氣。（此時發音「HA」）
# {{lang|ja|'''め'''}}　…　在双手掌心聚出一个光球。（此時發音「ME」）
# {{lang|ja|'''波'''}}　…　兩手朝向目標，放出氣功波。打出时为一个蓝光弹，像流星一样，有条尾巴。（喊「HA！」時放出）

有時只唸「波」（HA），也可放出氣功。

中文版因音節數問題，有時也譯作「'''神龜衝撃波'''」或「'''龜-{派}-氣功波'''」。

===威力===
初登場時龜仙人便以龜派氣功一擊消滅了火焰山的大火，連同城堡也一起摧毀了，悟空則是在第一次使用時打壞了車子。

在天下第一武鬥大會中，龜仙人以威力最強的龜派氣功轟走了月亮，讓看見月亮變成猩猩的悟空回復成原來的模樣。

後來隨著悟空與其他角色越來越強，龜派氣功的威力也不斷的提升，甚至於能一擊摧毀整個星球。

=== 變化 ===
* 十倍形，红色光弹，只有变成超級[[赛亞人|赛亞人]]4的孙悟空能使用，威力是普通形的十倍。
* 大爆炸龜派氣功，由超級賽亞人4的悟吉塔所使用，雙手伸直於前方聚氣，然後發射。
* 植曼形，由小林所使用，雙手伸直於上方聚氣，直於前方慢慢地發射上天空，然後变成无数光弹来發射。

== 影響 ==
七龍珠的标志，除了“Y”的手势外，龜波氣功的手势成为七龍珠的特色之一。在[[日本|日本]]、[[台灣|台灣]]和[[香港|香港]]連載與播出期间，成为愛好者们的热仿动作。此外[[香港電影|香港電影]]《[[唐伯虎點秋香|唐伯虎點秋香]]》結尾，與太師夫人鬥法一幕，作為其中一項招式；《[[武状元苏乞儿|武状元苏乞儿]]》的最后一式[[亢龙有悔|亢龙有悔]]也擺出同样的動作。其他致敬、仿效、甚至抄襲的作品也非常多。[[电子游戏|电子游戏]][[快打旋風系列|-{zh-tw:快打旋風;zh-cn:街头霸王;zh-hk:街頭霸王}-]]的'''龙'''与'''KEN'''的[[波动拳|波动拳]]亦与龜波氣功相仿。

龜派氣功後來也被廠商製作成玩具、體感電子遊戲。<ref>{{cite web|title=用這個《七龍珠》玩具 你也可以發龜派氣功|url=http://www.appledaily.com.tw/realtimenews/article/life/20170318/1079438/|website=蘋果日報|language=zh-TW|date=2017年3月18日}}</ref><ref>{{cite web|title=アニメ『ドラゴンボールZ』の必殺技“かめはめ波”を撃つ体験ができるVRグッズが登場【動画あり】|url=https://www.famitsu.com/news/201704/13131070.html|website=ファミ通.com|language=ja|date=2017年4月13日}}</ref>
