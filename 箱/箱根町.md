'''箱根町'''（{{jpn|j=箱根町|hg=はこねまち|rm=Hakone machi|f=y}}）是[[日本|日本]][[神奈川縣|神奈川縣]]西部，[[箱根峠|箱根峠]]東側位置的[[町|町]]，屬[[足柄下郡|足柄下郡]]。

== 概要 ==

自古已是温泉町，在[[江戶時代|江戶時代]]設置[[箱根關|箱根關]]。近代由於[[西武鐵道|西武鐵道]]集團與[[小田急電鐵|小田急電鐵]]致力於觀光開發而得到急速發展。'''[[箱根溫泉|箱根溫泉]]'''是湯本、塔之澤、大平台、宮之下、小涌谷等地溫泉的總稱。

== 地理 ==
*山：[[箱根山|箱根山]]、[[箱根峠|箱根峠]]
*川：[[蛇骨川|蛇骨川]]、[[須雲川|須雲川]]、[[早川|早川]]、[[明神川|明神川]]、[[大澤|大澤]]
*湖：[[蘆之湖|蘆之湖]]（{{lang|ja|芦ノ湖}}）、{{lang|ja|お玉ヶ池}}、[[精進池|精進池]]

== 历史 ==

; 律令時代
[[律令制|律令時代]]屬[[東海道|東海道]][[相模國|相模國]]、[[足柄郡|足柄郡]]。

; 鎌倉時代
[[源賴朝|源賴朝]]舉兵的序盤之戰於石橋山展開。參考[[石橋山之戰|石橋山之戰]]。

; 江戶時代
[[德川家康|德川家康]]時、[[東海道|東海道]]路線由足柄峠改為箱根峠、現在的箱根町蘆之湖畔設置[[關所|關所]]（[[箱根關所|箱根關所]]）。

; 明治維新以後
[[廢藩置縣|廢藩置縣]]之後屬[[足柄縣|足柄縣]]。於1876年8月21日足柄縣分割之後屬神奈川縣。

1889年、町村制施行時分成箱根町、湯本村、溫泉村、宮城野村、仙石原村、元箱根村、蘆之湯村等1町6村。1927年湯本村施行町制、1954年箱根町、元箱根村、蘆之湯村合併為新的箱根町、1956年9月5町村合併成現在的箱根町。

== 觀光 ==
[[File:HakoneAndFuji.jpg|thumbnail]]]]
箱根是日本知名的觀光地。近年來有很多來自台灣和韓國的觀光客來到箱根，因而景點的標記牌上也增加了[[繁體中文|繁體中文]]和[[諺文|諺文]]表記。
=== 主要觀光景點 ===
*[[芦之湖|芦之湖]]为箱根火山活动休眠而形成的火山湖，湖面积690公顷，湖深43公尺，以[[富士山|富士山]]为其背景，山色绝美，随风乘[[芦之湖海盗船|芦之湖海盗船]]漫游，景致十分怡人。
*[[強羅公園|強羅公園]]是一座修建於早雲山麓下的法式岩石庭園。
*[[箱根神社|箱根神社]]建于西元8世纪，位于芦之湖南端的元箱根，拥有悠久的历史和珍藏许多文化遗产而闻名。
*[[箱根美術館|箱根美術館]]與位於[[靜岡縣|靜岡縣]][[熱海市|熱海市]]的[[MOA美術館|MOA美術館]]為姐妹館。

== 以箱根為背景的作品 ==
*动画片《[[新世纪福音战士|新世纪福音战士]]》中的[[第三新东京市|第三新东京市]]以箱根为原型，动画中经常取箱根当地实景作画。
*輕小說及動畫作品《[[GATE_奇幻自衛隊|GATE 奇幻自衛隊]]》也有將該地作為背景的劇情。
*動畫片[[爆走兄弟Let's_&_Go!!|爆走兄弟Let's & Go!!]] WGP世界盃篇，總決賽的第二天賽事，以箱根山道為背景。
*2015年10月新番《温泉幼精小箱根》就是以箱根為背景的動畫作品。劇中出現的地點名稱和場景均為現實中所有，如箱根、強羅、蘆之湖、元箱根港等
