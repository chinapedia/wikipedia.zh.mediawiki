'''羽根田卓也'''（{{lang-ja|はねだ たくや}}；{{bd|1987年|7月17日}}）是[[日本|日本]]男子[[輕艇激流|輕艇激流]]運動員，從2000年初開始參賽。他在[[2014年亞洲運動會|2014年亞洲運動會]]輕艇激流C1奪得金牌<ref>{{cite web
 |url          = http://www.incheon2014.kr/Sports/Biographies/Athletes_Profile/?ParticCode=5104623&lang=en
 |title        = HANEDA Takuya
 |website      = Incheon2014.kr
 |publisher    = The 17th Incheon Asian Games Organizing Committee
 |accessdate   = 2016年5月11日
 |archiveurl   = https://web.archive.org/web/20141106192333/http://www.incheon2014.kr/Sports/Biographies/Athletes_Profile/?ParticCode=5104623&lang=en
 |archivedate  = 2014年11月6日
 |deadurl      = yes
}}</ref>，[[2016年夏季奧林匹克運動會|2016年夏季奧林匹克運動會]]他在同一個項目奪得銅牌，成為亞洲第一位奪得奧運獎牌的輕艇激流運動員<ref>[http://www.asahi.com/ajw/articles/AJ201608100008.html OLYMPICS/ Japan’s Haneda takes bronze in men's canoe slalom]. Associated Press via Asahi (August 10, 2016)</ref>。

羽根田卓也自18歲開始移居[[斯洛伐克|斯洛伐克]]接受訓練。他的教練為前斯洛伐克輕艇運動員[[Milan_Kubáň|Milan Kubáň]]<ref>{{cite web|title=Hanedov bronzový príbeh so slovenským rukopisom|url=http://sport.aktuality.sk/c/229876/rio-2016-hanedov-bronzovy-pribeh-so-slovenskym-rukopisom/|accessdate=10 August 2016}} {{sk icon}}</ref>。
