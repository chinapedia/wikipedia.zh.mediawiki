'''羽生市'''（{{jpn|j=羽生市|hg=はにゅうし|rm=Hanyū shi|f=y}}）是[[埼玉縣|埼玉縣]]東北部的一個人口約5萬5千人的[[市|市]]。以衣料街而聞名。往[[加須市|加須市]]的通勤率為10.1%（平成22年國勢調査）。

== 地理 ==
位於埼玉縣東北部，北有[[利根川|利根川]]流經。市域大略平坦。由於地處加須低地北部的沖積台地，海拔約17公尺左右。除接壤[[群馬縣|群馬縣]]，也鄰近[[茨城縣|茨城縣]]與[[栃木縣|栃木縣]]。[[東武伊勢崎線|東武伊勢崎線]]南北貫穿本市，通往[[秩父市|秩父市]]的[[秩父鐵道|秩父鐵道]]也以[[羽生站|羽生站]]為起點。另外[[東北自動車道|東北自動車道]]通過本市東部。

== 歷史 ==
* 1889年（[[明治|明治]]22年）4月1日 - [[町村制|町村制]]施行，[[北埼玉郡|北埼玉郡]]{{link-ja|羽生町|羽生町}}成立。
* 1903年（明治36年）4月23日 - [[東武鐵道|東武鐵道]][[東武伊勢崎線|伊勢崎線]][[羽生站|羽生站]]開業。
* 同年9月13日 - 東武鐵道伊勢崎線須影站（現[[南羽生站|南羽生站]]）開業。
* 1921年（[[大正|大正]]10年）3月30日 - 北武鐵道（現[[秩父鐵道|秩父鐵道]]）羽生站開業。
* 1922年（大正11年）7月28日 - 北武鐵道{{link-ja|新鄉站 (埼玉縣)|新郷駅 (埼玉県)|新鄉站}}開業。
* 1954年（[[昭和|昭和]]29年）9月1日 - [[北埼玉郡|北埼玉郡]]羽生町、{{link-ja|新鄉村 (埼玉縣北埼玉郡)|新郷村 (埼玉県北埼玉郡)|新鄉村}}、{{link-ja|須影村|須影村}}、{{link-ja|岩瀨村 (埼玉縣)|岩瀬村 (埼玉県)|岩瀨村}}、{{link-ja|川俣村 (埼玉縣)|川俣村 (埼玉県)|川俣村}}、{{link-ja|井泉村|井泉村}}、{{link-ja|手子林村|手子林村}}合併成立'''羽生市'''。
* 1955年（昭和30年）1月1日 - 北埼玉郡{{link-ja|三田谷村|三田ヶ谷村}}、{{link-ja|村君村|村君村}}合併成立{{link-ja|千代田村 (埼玉縣)|千代田村 (埼玉県)|千代田村}}。
* 1959年（昭和34年）4月1日 - 千代田村併入羽生市。
* 1981年（昭和56年）9月1日 - 秩父鐵道{{link-ja|西羽生站|西羽生駅}}開業。
* 1992年（[[平成|平成]]4年）3月 - [[東北自動車道|東北自動車道]][[羽生IC|羽生交流道]]通車。
* 2006年（平成18年）3月25日 - [[國道122號|國道122號]]新{{link-ja|昭和橋 (利根川)|昭和橋 (利根川)|昭和橋}}通車。

== 市名由來 ==
市內[[神社|神社]]的懸佛刻有「太田'''埴生'''庄」，埴（はに、赤土）生（う、多）後來轉變為音近的羽生。另也說是演變自[[埴輪|埴輪]]（はにわ）<ref name="hanyu_rekishi">[http://www.city.hanyu.lg.jp/docs/2011022800088/ 羽生市サイト 羽生の歴史]</ref>。

在文字上，[[太田道灌|太田道灌]]在[[文明_(年號)|文明]]10年（1478年）曾寫下「武州'''羽生'''の嶺にたてこもり」。而「小田原舊記」則曾記載「武州'''羽丹生'''城代中條出羽守」，代表過去曾有埴生、羽生、羽丹生等寫法。<ref>羽生市史上卷</ref>
