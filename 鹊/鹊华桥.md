'''鹊华桥'''，原位于[[山东省|山东省]][[济南市|济南市]][[历下区|历下区]][[大明湖|大明湖]]南门东侧，[[百花洲_(济南)|百花洲]]北侧，为一座东西向单孔石拱桥，桥下通连大明湖与百花洲。

该桥始建于宋代，曾因百花洲而称为'''百花桥'''，又名'''白雪桥'''，元代改称鹊华桥（一说百花桥位于百花洲南，与鹊华桥两桥相望），明代曾重建，清[[嘉庆|嘉庆]]二年（公元1797年）及[[道光|道光]]五年（公元1825年）又两次重建。原桥因侵华日军破坏，桥面凸凹不平，坡陡路滑，造成行车隐患，1946年被拆除，改建为木板平桥，后分别在1948年和1953年更换为石板平桥和钢筋混凝土平桥。1983年拓宽明湖路时再次改建，桥面与路面持平，已看不出原桥的痕迹<ref>{{cite web|url=http://www.jnup.gov.cn/qcyx/mqzc/qszg/200711/3190a10b-83d4-4f4f-a014-819e1d356786.htm|title=《鹊华桥、百花洲、曲水亭》|accessdate=2009-10-29|work=|publisher=济南市规划局|archive-url=https://web.archive.org/web/20160305042918/http://www.jnup.gov.cn/qcyx/mqzc/qszg/200711/3190a10b-83d4-4f4f-a014-819e1d356786.htm|archive-date=2016-03-05|dead-url=yes}}</ref><ref>{{cite web |url=http://news.163.com/09/0324/04/5554TC0F000120GU.html|title=《闲话鹊华桥》|accessdate=2009-10-29 |work=|publisher=网易新闻}}</ref>。

当时为方便游船出入，鹊华桥净跨4米，桥面最高处高出路面约2米<ref>{{cite web|url=http://www.jndaxxw.gov.cn/dangan/zhuanti/jinanjiuying/jinanmingcheng/mc/9-2.asp|title=《济南旧影-鹊华桥》|accessdate=2009-10-29|work=|publisher=济南档案信息网}}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，清末民初时期曾为游览大明湖的重要码头，在清代[[刘鹗|刘鹗]]所著的《[[老残游记|老残游记]]》中，老残便是在鹊华桥登舟，继而游览大明湖。另外，当时大明湖北岸很少有高大建筑，空气也很少污染，游人站在桥顶向北可以眺望到济南北郊遥遥相对、朦胧隐现于烟雨之中的[[鹊山|鹊山]]和[[华不注山|华山]]，即为[[济南八景|历下八景]]中的“鹊华烟雨”<ref>{{cite web |url=http://news.sina.com.cn/s/2005-08-24/03166765168s.shtml|title=《鹊华桥跃然纸上露真容》|accessdate=2009-10-28 |work=|publisher=舜网-济南时报}}</ref>，元代[[赵孟頫|赵孟頫]]的《[[鹊华秋色图|鹊华秋色图]]》即根据这一景观绘成。

为恢复“鹊华烟雨”景观，2009年在大明湖扩建工程中异地重建了鹊华桥，位于“鹊华烟雨”石牌坊以北，总长度约90米，为景区内唯一一座五孔拱桥<ref>{{cite web |url=http://www.sdta.gov.cn/zhengwunews1/new1/lydtlist.asp?id=8809|title=《大明湖扩建工程中新修的鹊华桥雏形显现》|accessdate=2009-10-29 |work=|publisher=山东旅游政务网}}</ref>，亦作为“七桥风月”<ref>即七座景观桥，包括芙蓉桥、百花桥、秋柳桥、水西桥、鹊华桥、湖西桥、北池桥，七桥环湖，风格各异，曾巩曾有诗：“从此七桥风与月，梦魂常到木兰舟”。</ref>的组景之一<ref>{{cite web|url=http://www.jncc.gov.cn/News/newsView.aspx?Fid=4644&FclassID=17500&FnewsNo=100100|title=《大明湖新区今天开门纳客　28座石桥串起新美景》|accessdate=2009-10-29|work=|publisher=济南建设网}}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。
