《'''癲噹'''》（Din Dong）為[[香港|香港]][[動漫|動漫]]作者[[貓室|貓室]]的跨媒體創作，涵蓋具劇情[[動畫|動畫]]、漫畫、角色形象商品及聯合文化創作等範疇。

== 概要 ==
本作於[[2007年－2008年環球金融危機|金融海嘯]]時投入設計工作，而作者亦表明其中一個創作理念，是盼望觀眾讀者可以在該時勢反思過份執著金錢與物質的生活<ref name="nowTV" />，亦表現一點逆境自強精神<ref name="headline" />。最初以專欄漫畫形式發佈，其後於2010年2月27日把第一部CG動畫短片放上Youtube播放，並邀得[[香港配音員|香港配音員]][[林保全|林保全]]，為主角癲噹配音<ref>{{cite news |language = 中文 | author = 彭碧珊 | url = http://forum.vlshk.com/view.php?id=957 | title = 港產動畫《癲噹》叮噹配音員聲演 配音老手盼港卡通風行 | publisher = 明報 | date = 2010年3月30日}}</ref>。後來於4月，作者獲邀請參展[[第三十四屆香港國際電影節|第三十四屆香港國際電影節]]之時，所有動畫短片與其他國際比賽得獎作品結合為《貓室動畫大集結》，在展場內亮相。<ref>{{Cite web |url= http://34.hkiff.org.hk/chi/film/detail/34163.html |title= 第34屆香港國際電影節 - 貓室動畫大集結 - 電影詳情 |date= 2010年1月 |publisher= 香港國際電影節協會有限公司 |language= 中文 |accessdate= 2013年8月18日 |deadurl= yes |archiveurl= https://web.archive.org/web/20160304131003/http://34.hkiff.org.hk/chi/film/detail/34163.html |archivedate= 2016年3月4日 }}</ref>同年2010年10月，本作再以公募作品形式參與第4屆日本東京ASIAGRAPH，並獲評審團嘉許為特別上映作品。<ref>{{Cite web |url= http://www.asiagraph.jp/2010/program/report/works2/index.html|title= ASIAGRAPH2010 CGアートギャラリー公募展示部門　入選者紹介|date=2010年9月|publisher= ASIAGRAPH2010 |language= 中文 |accessdate= 2013年8月18日}}</ref>。

除了[[動漫|動漫]]外，主角癲噹的形象亦成為帶領[[貓室|貓室]]角色走向香港大眾潮流的主軸，在以其為名的網頁及Youtube頻道展示[[貓室|貓室]]大小創作、商場展覽及品牌廣告。

; 癲噹
: 動畫配音：[[林保全|林保全]]
: 本作主角，治癒系的貓咪，作者「貓室」的家貓，充滿正能量，經常說笑話和扮鬼扮馬，希望為忙亂的都市人帶來歡樂。<ref>{{Cite web |url= http://din-dong.net/shop.html#!/~/product/category=0&id=4549061 | title= online shop Book 2 - 免費去旅行 |publisher= dindong.net|language= 中文 |accessdate= 2013年8月18日}}</ref>
; 戴熊
: 動畫配音：洪文婷
: 香港人，於金融海嘯中失業<ref name="headline" />，擁有大部分「[[宅男|宅男]]」個性<ref name="nowTV">{{Cite web |url= http://www.youtube.com/watch?v=Igx3TYdOsv8|title= 癲噹(din-dong) @ NOW新聞台|author=何海淩、陳大偉 |date=2010年8月 |publisher= [[NOW新聞台|NOW新聞台]] 文藝館 |language= 中文 |accessdate= 2013年8月18日}}</ref>，有問題就會找上癲噹幫忙，但最後都不會有完滿的好下場。
; 賤香
: 戴熊的女朋友，身材肥胖，具[[公主病|公主病]]的特徵<ref name="nowTV" />。
; 阿讚貓
: 又名阿正，癲噹的哥哥，外表可愛，性格粗獷，可是心地善良。只要好好跟他相處，他就會展露溫柔的一面。<ref>{{Cite web |url= http://din-dong.net/shop.html#!/~/product/category=0&id=4549061 | title= online shop Book 2 - 免費去旅行 |publisher= dindong.net|language= 中文 |accessdate= 2013年8月18日}}</ref>

== 作品 ==
=== 漫畫 ===
漫畫故事最早以[[漫畫|漫畫]]專欄形式於《[[東Touch|東Touch]]》連載，及後亦有於馬來西亞《Comic King》上出現<ref name="headline">{{cite news |language = 中文 | author = 盧倩敏 | url = http://news.hkheadline.com/dailynews/headline_news_detail_columnist.asp?id=78388&section_name=wtt&kw=8 | title = 新聞故事—— 「癲噹」「戴熊」勉勵港人 創意藝術展散播快樂 | publisher = 頭條日報 | date = 2009年5月15日}}</ref>，目前則於《[[晴報|晴報]]》連載。內容圍繞主角癲噹與作者之間的生活。部分作品輯印成書，透過[[三聯書店（香港）|香港三聯書店]]出版。

=== 動畫 ===
動畫故事改編自漫畫，每段長約1分鐘，由[[貓室|貓室]]單獨負責所有映像製作，並邀得[[The_Pancakes|The Pancakes]]設計貫通全篇之背景音樂。
