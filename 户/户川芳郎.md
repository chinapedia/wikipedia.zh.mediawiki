'''户川芳郎'''（{{bd|1931年|8月27日}}），[[日本|日本]][[汉学家|汉学家]]，[[东京大学|东京大学]]“[[中国|中国]]文史部”掌门人。<ref name="日本汉学家：儒学在日本为什么被称为“儒教” ">{{cite web |url=http://www.cccrx.com/2008-1/2008122224511.asp|title=日本汉学家：儒学在日本为什么被称为“儒教”|accessdate=2008年1月22日|work=王宇|publisher=中国当代儒学网|language=zh-cn}}</ref>

== 简介 ==
2005年，户川芳郎被[[汤一介|汤一介]]邀请作为《[[儒藏|儒藏]]》的编委，他说：“[[日本|日本]]选定的[[文献|文献]]选目应该体现出日本[[儒学|儒学]]的最高水准。”<ref>{{cite web|url=http://www.bj.xinhuanet.com/bjpd_sdzx/2009-04/25/content_16355371.htm|title=《儒藏》工程编纂工作吸引韩日等国学者参与|accessdate=2009-04-25|work=|publisher=新华网|language=zh-cn|deadurl=yes|archiveurl=https://web.archive.org/web/20140221195247/http://www.bj.xinhuanet.com/bjpd_sdzx/2009-04/25/content_16355371.htm|archivedate=2014-02-21}}</ref>

== 学术思想 ==
户川芳郎长期致力于研究[[汉学|汉学]]，把[[儒学|儒学]]尊称为“[[儒教|儒教]]”，更加偏重[[儒学|儒学]]的教化意义。{{noteTag|1=儒教：不仅仅是单一的[[宗教|宗教]]，还包含了教学、教养、教化等意义。<ref name="日本汉学家：儒学在日本为什么被称为“儒教” "/>}} 他一生主要精力用于协助日本[[学者|学者]]搜集整理日本的[[儒家|儒家]]著作，探寻[[中华文化|中华文化]]的内核和意义，推动汉学在世界的发展。<ref name="日本汉学家：儒学在日本为什么被称为“儒教” "/>
