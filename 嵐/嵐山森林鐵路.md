'''嵐山森林鐵路'''為一座位在[[臺灣|臺灣]][[花蓮縣|花蓮縣]][[秀林鄉|秀林鄉]]山區的林業[[鐵路|鐵路]]運輸系統，該森林鐵路的2號索道曾是全球最長的林業用[[索道|索道]]，並且該鐵路所在的太魯閣林場曾是臺灣東部三大林場之一<ref name="台灣的森林鐵路簡介 嵐山山地鐵道">{{Cite web|url=http://blog.xuite.net/sujaushi/twblog1/226659908|title=台灣的森林鐵路簡介 嵐山山地鐵道|accessdate=2015-06-30|author=蘇昭旭|date=2014-07-14|publisher=Xuite|language=zh-hant}}</ref>，當前該座森林鐵路系統已停用廢棄。<ref name="東部探險">{{Cite web |url=https://www.youtube.com/watch?v=UYxwIKMz8BA |title=東部探險-嵐山工作站 |accessdate=2015-06-30 |author=|publisher=東部探險|language=zh-hant |date=}}</ref>

== 沿革 ==
===日治時期 ===
嵐山森林鐵路最早是在臺灣[[日治時期|日治時期]]1943年（昭和18年）由當時的[[南邦林業株式會社|南邦林業株式會社]]進行山區林業開發，並且隸屬在該公司旗下的[[花蓮港|花蓮港]]支社。最早的嵐山鐵道是於花蓮港支社正式開採的兩年後進行鋪設，總長6公里。<ref name="東部探險"/><ref name="《台灣的鐵道》">《台灣的鐵道》，遠足文化，出版日期：民國91年5月，ISBN 9868015472</ref>

南邦林業株式會社持續經營嵐山，直到戰後移交林產管理局<ref name=":0">{{Cite book|url=https://www.worldcat.org/oclc/837777397|title=《失序的森林：日治末期太魯閣林業開發》|last=張雅綿|first=|date=|publisher=花蓮縣文化局|year=2012|isbn=9789860342437|edition=初版|location=花蓮市|pages=|chapter=|oclc=837777397}}</ref>{{rp|205}}。

=== 戰後至今 ===
二戰結束後，因接收不及，太魯閣的森林伐採仍由南邦林業株式會社繼續經營<ref name=":0" />{{rp|205}}。直到1946年（民國35年）9月後，[[臺灣省行政長官公署|行政長官公署]]農林廳林產管理委員會才接收，嵐山森林鐵路也包含在內一併移交，並改名為花蓮港山場<ref name=":0" />{{rp|205}}。1947年，林產管理局成立，太魯閣林場被暫劃為隸屬於[[太平山林場|太平山林場]]之下的太魯閣分場<ref name="《台灣的鐵道》 "/><ref name=":0" />{{rp|205}}<ref>{{Cite web|url=http://subject.forest.gov.tw/web/publication/main_01-3-1.htm|title=林務局局誌-光復初期的林務局|accessdate=2015-06-30|author=姚鶴年|date=1997-03-12|publisher=行政院農委會林務局|language=zh-hant}}</ref>。

1948年（民國37年）太平山林場太魯閣分場正式獨立為太魯閣林場。<ref name="《台灣的鐵道》 "/>

1951年（民國40年），太魯閣林場計畫開闢新路線，於隔年1952年（民國41年）完工，一共興建有總長9.5公里，並正式命名為太魯閣主線的鐵道，以及長1.4公里的支線。此外也興建一條連接位在[[佐倉|佐倉]]的木材存放場至[[臺鐵|臺鐵]][[臺東線|臺東線]]與林場之間的鐵路路線長3.7公里。<ref name="東部探險"/>

1951年（民國40年），由於佐倉的木材存放場腹地過小，加上[[砂婆噹溪|砂婆噹溪]]的[[橋梁|橋梁]]時常損壞，經太魯閣林場重新檢討後進行改線計畫，平地鐵道不再過溪，改由太昌興建1.5公里路線至新建的新1號索道著點。<ref name="台灣山岳雜誌">{{Cite web |url=http://www.twmount.com.tw/content/Content-Detail.aspx?seri=3218&issue=97 |title=嵐山新一號索道 |accessdate=2015-06-30 |author=古庭維|publisher=台灣山岳雜誌|language=zh-hant |date=}}</ref>

而新1號索道著點往山區前進至嵐山工作站的新線上共有架設3條索道，於1956年（民國45年）5月完成，其中，三條索道的斜距依序分別為1,560、1,600、1,400公尺，新2號索道更號稱'''「東亞第一長索道」'''。<ref name="東部探險"/><ref name="台灣的森林鐵路簡介 嵐山山地鐵道"/>

1960年（民國49年）後，太魯閣主線繼續延伸擴建，最終延伸總長為42公里。<ref name="東部探險"/>

1977年（民國66年）以後，臺灣省政府實施「台灣林業經營改革方案」，將臺灣的森林經營改以多目標的保育與利用為前提，而使得伐木業務逐漸減少，當時的哈崙工作站也開始結束伐木業務，改以造林為主。於1989年（民國78年）7月嵐山工作站正式裁撤並且連帶的嵐山森林鐵路也結束營運使用荒廢至今。<ref name="東部探險"/><ref name="台灣的森林鐵路簡介 嵐山山地鐵道"/><ref name="台灣山岳雜誌"/>

=== 車輛 ===

* [[蒸汽|蒸汽]]機關車<ref name="哈崙伐木場2">{{Cite web |url=http://library.taiwanschoolnet.org/cyberfair2002/C0236970055/color-htm/index.htm |title=繁華盡落的山林-哈崙伐木場 |accessdate=2015-06-30 |author=孫承偉、童如妤、官美媛所帶領之本專題製作學生|publisher=花蓮縣鑄強國民小學|language=zh-hant |date=2002年2月8日}}</ref>
[[美國|美國]]製造，以[[煤炭|煤炭]]為燃料，運轉整備重量18噸，製動機具有手剎車及[[空氣|空氣]]剎車兩種，發動後可牽引重量95公噸，為世界上甚少見會爬山的機關車。
* [[汽油|汽油]]機關車<ref name="哈崙伐木場2"/>
日本製造，以汽油為燃料，最大馬力為50匹馬力，總排氣量1600立方公分，總重量45噸，發動後可牽引10節小台車，載重量40公噸左右，服務於木瓜林區管理處約30年（包含嵐山、哈崙、林田山的森林鐵路），1980年（民國69年）6月30日後停駛。
* 台車

=== 工作站 ===
* 嵐山工作站
嵐山工作站設置於太魯閣主線5.4公里處，站內設置有林務局行政單位、機械工廠、小型的製材廠以及林務局聘僱伐木工之宿舍區，總駐紮人員至正式裁撤前約有50多名工作人員及行政人員<ref name="東部探險"/>，總占地面積約2[[公頃|公頃]]。<ref>{{Cite web |url=http://www.twmount.com.tw/content/Content-Detail.aspx?seri=3562&issue=105 |title=台灣山岳雜誌-嵐山工作站 |accessdate=2015-06-30 |author=古庭維|publisher=台灣山岳雜誌|language=zh-hant |date=}}</ref>
