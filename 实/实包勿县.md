'''实包勿县'''（''{{lang-en|Sebauh District}} {{lang-ms|Daerah Sebauh}}''）是[[砂拉越|砂拉越]][[民都鲁省|民都鲁省]]的一个县，总面积有5229.48平方公里<ref>[http://www.bintuluro.sarawak.gov.my/modules/web/page.php?id=93 民都鲁省公署] {{webarchive|url=https://archive.is/20120722020402/http://www.bintuluro.sarawak.gov.my/modules/web/page.php?id=93 |date=2012-07-22 }} 2011年12月9日更新 [2011年12月10日查阅] {{ms}}</ref>，人口约2.1万（2010年人口普查）<ref>[http://www.bintuludo.sarawak.gov.my/page.php?id=84&menu_id=0&sub_id=128 实包勿副县概况]{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }} 2011年12月9日更新 [2011年12月10日查阅]{{ms}}</ref>。县里有4个小镇，6个甘榜和202个长屋，多数的地区可以通过陆路交通到达，但一些较深郊地区还是需要通过水路到达如：乌鲁班丹（Ulu Pandan） 宾友（Binyo） 和乌鲁浙拉隆（Ulu Jelalong）。

== 经济 ==
在实包勿县里，70%的经济是来自[[农业|农业]]，如耕种[[油棕|油棕]]，[[稻谷|稻谷]]与[[胡椒|胡椒]]。县里也有少数的木材业在进行。 

== 人口分布 ==
在实包勿县里，[[华人|华人]]，[[马来人|马来人]]与[[马兰诺族|马兰诺族]]居住得较集中，多数住在县里的几个小镇上，如：[[实包勿|实包勿]]镇（Pekan Sebauh），[[都胞|都胞]]镇（Pekan Tubau），[[班丹|班丹]]镇（Pekan Pandan）和[[拉芒|拉芒]]镇（Pekan Labang）。[[伊班族|伊班族]]及[[乌鲁人|乌鲁人]]（Orang Ulu）居住的比较分散，而也都分布在乡郊地区：'''卡库斯'''（Kakus）和'''浙拉隆'''（Jelalong）。
