'''渡瀨晶'''（{{lang-ja|わたせ あきら}}，{{bd|1980年|7月2日|catIdx=W渡}}）是[[日本|日本]]知名的[[AV女優|AV女優]]。

== 人物 ==
渡瀨晶出身於日本[[東京都|東京都]]。
*於[[美國|美國]][[洛杉磯|洛杉磯]]拍攝《BLACK SEMEN》系列，與五名黑人男優作對手戲，2002年7月15日發行。
*於《Dream Party》與多名女優合演，突破性與同性女優作對手戲。
*引退作品之一《露出大都會東京3》，於東京街頭當眾與男優作對手戲。

== 略歷 ==
*2000年 - 到日本AV製作公司應徵，同年5月獲日本「宇宙企画」錄用，並於2000年5月28日發表其第一部個人AV作品。
*2001年5月 - 轉投日本「MOODYZ」AV製作公司。
*2002年 - 接拍女性[[內衣|內衣]]及[[絲襪|絲襪]]廣告。獲得2002年度成人廣播美腳女優大獎。
*2003年 - 開始參演多人合演的影片，與其他女優混熟成好友。
*2004年 - 遊走於「[[Cinamagic|Cinamagic]]」、「[[AUDAZ|AUDAZ]]」、「[[WANZ_Factory|WANZ Factory]]」、「[[ATHENA|ATHENA]]」、「[[SOD|SOD]]」等多間AV製作公司之間，出演作品已超過50部。
*2005年 - 作品大幅減少並於11月隱退。
*2006年 - 嫁予一名日本[[警察|警察]]。
*2007年 - 舊片重出，改名為「なるみ礼」。
