'''渡邊謙'''（{{lang-ja|渡辺 謙}}，[[假名|假名]]：{{lang|ja|わたなべ けん}}，{{lang-en|'''Watanabe Ken'''}}，{{bd|1959年|10月21日|}}）是一名[[日本|日本]][[男演員|男演員]]。出生於[[新潟縣|新潟縣]][[北魚沼郡|北魚沼郡]][[小出町|小出町]]（今[[魚沼市|魚沼市]]）。

渡邊謙从出演電視劇出道，後成為電影演員。他的代表作包括NHK大河劇《[[獨眼龍政宗|獨眼龍政宗]]》及電影《[[末代武士|末代武士]]》<ref>[http://movies.about.com/cs/thelastsamurai/a/smkw120103.htm Watanabe in the Last Samurai]</ref><ref>{{cite web|last=Corkill |first=Edan |url=http://www.japantimes.co.jp/life/2013/07/06/people/from-hollywood-to-hirohito/#.UeFONmQY1F8 |title=From Hollywood to Hirohito |publisher=The Japan Times |date= |accessdate=2014-02-19}}</ref>。他憑《[[末代武士|末代武士]]》於2004年获[[奧斯卡|奧斯卡]]最佳男配角奖提名<ref>[http://articles.cnn.com/2010-08-18/entertainment/ken.watanabe_1_theatrical-actor-ken-watanabe-roles?_s=PM:SHOWBIZ Watanabe nominated for Academy Award] {{webarchive|url=https://web.archive.org/web/20120214050315/http://articles.cnn.com/2010-08-18/entertainment/ken.watanabe_1_theatrical-actor-ken-watanabe-roles?_s=PM:SHOWBIZ |date=2012-02-14 }}</ref>，此片更使他为國際電影界所知。此后他与好莱坞导演多次合作，出演过[[克林特·伊斯特伍德|克林特·伊斯特伍德]]导演的《[[來自硫磺島的信|來自硫磺島的信]]》（2006）以及[[克里斯托弗·诺兰|克里斯托弗·诺兰]]导演的《[[蝙蝠俠：開戰時刻|蝙蝠俠：開戰時刻]]》（2005）、《[[全面啟動|全面啟動]]》（2010）。

== 經歷 ==
渡邊謙高中畢業後於1978年加入了劇團{{lang|ja|ひまわり}}，但不久退團。同年，由於觀看[[演劇集團圓|演劇集團圓]]後大為感動，加入了劇團。他演出的第一部電視剧為《未知的反亂》。其後他在電影《{{tsl|ja|海と毒薬 (映画)|海與毒藥 (電影)|海與毒藥}}》及《{{tsl|ja|タンポポ (映画)|蒲公英 (電影)|蒲公英}}》中表現優異，開始受到关注。

1986年渡邊謙在[[連續電視小說|NHK晨間劇]]《[[跳駒|跳駒]]》中表現良好，从而得以饰演1987年[[NHK|NHK]][[大河劇|大河劇]]《[[獨眼龍政宗|獨眼龍政宗]]》中的主角[[伊達政宗|伊達政宗]]，一演成名。此剧平均收視率為39.7%，为歷年大河劇最高，政宗的印象深入人心，2003年日本人票選最想重播的電視劇，仍是「獨眼龍政宗」。

1990年原本应邀演出[[角川映畫|角川映畫]]的《天與地》男主角[[上杉謙信|上杉謙信]]，但因為罹患[[白血病|白血病]]無緣演出。2002年1月1日離開了他所屬的演劇集團，2003年首次出演日本以外制作的作品，首部作品為《末代武士》，劇中飾演森勝元一角獲得好評。

1983年和第一任妻子由美子結婚，育有一子一女，儿子[[渡邊大|渡邊大]]也是演員，女儿[[杏_(演員)|杏]]是[[模特兒|模特兒]]及演員。2005年離婚，同年與演員[[南果步|南果步]]結婚。他的2004年，他被美國《[[時人_(雜誌)|People]]》雜誌评选为「50位最美丽人士」之一，並被評為日本中年演員四天王之一（其他三人為[[中井貴一|中井貴一]]、[[真田廣之|真田廣之]]、[[佐藤浩市|佐藤浩市]]）。

2017年3月，《[[週刊文春|週刊文春]]》的報導指渡邊謙與某位比他年輕21歲的女子搞婚外情長達三年。2018年5月17日，渡邊謙透過事務所宣布與南果步離婚<ref>{{Cite web |url = https://www.sponichi.co.jp/entertainment/news/2018/05/17/kiji/20180517s00041000288000c.html |title = 渡辺謙、南果歩との離婚を発表　結婚１３年目の破局　昨年３月に不倫発覚 |author = Sponichi Annex |date = 2018-05-17 |language = ja |accessdate = 2018-05-18}}</ref><ref>{{Cite web |url = http://hk.on.cc/hk/bkn/cnt/entertainment/20180517/bkn-20180517195025119-0517_00862_001.html |title = 為小三棄正印　渡邊謙與南果步離婚 |author = on.cc東網 |date = 2018-05-17 |publisher = 東方報業集團 |accessdate = 2018-05-18}}</ref>。
