《'''渡君的xx即將崩壞'''》（{{lang-ja|渡くんの××が崩壊寸前}}），由日本女性漫畫家[[鳴見奈留|鳴見奈留]]繪畫的漫畫，在2014年8月4日開始連載於《[[Young_Ace|Young Ace]]》<ref name="Natalie1">{{cite web|url=http://natalie.mu/comic/news/122755|title=JAの鳴見なる新作、畑荒らしに遭った兄妹描く|date=2014-08-04|accessdate=2016-10-16|publisher=[[Natalie_(網站)|Comic Natalie]]|language=ja}}</ref>，後來於2015年11月20日移籍至《{{link-ja|月刊Young Magazine|月刊ヤングマガジン}}》連載<ref name=Natalie2>{{cite web|url=http://natalie.mu/comic/news/166652|title=鳴見なるの三角関係＋妹のラブコメ「渡くんの××が崩壊寸前」月刊YMに移籍|date=2015-11-20|accessdate=2016-10-16|publisher=Comic Natalie|language=ja}}</ref>。

== 概要 ==
《渡君的xx即將崩壞》是{{link-ja|鳴見奈留|鳴見なる}}的第5部作品，前作為已改編為電視劇的《[[愛吃拉麵的小泉同學|愛吃拉麵的小泉同學]]》<ref name="Natalie1"/>。作品在2014年8月於[[日本漫畫雜誌|漫畫月刊]]《[[Young_Ace|Young Ace]]》連載，後來於2015年11月移籍至[[講談社|講談社]]旗下的《{{link-ja|月刊Young Magazine|月刊ヤングマガジン}}》<ref name=Natalie2/>。[[角川書店|角川書店]]出版的[[單行本|單行本]]第1集於2015年2月12日發售，但後來因移籍的關係，講談社決定重印第1冊，內頁亦與角川版有許多的不同<ref>{{cite web|url=http://life-hacking.net/%E3%80%8E%E6%B8%A1%E3%81%8F%E3%82%93%E3%81%AExx%E3%81%8C%E5%B4%A9%E5%A3%8A%E5%AF%B8%E5%89%8D%E3%80%8F%EF%BC%88%E7%AC%AC1%E5%B7%BB%EF%BC%89%E3%83%A4%E3%83%B3%E3%83%9E%E3%82%ACkc%E3%82%B9/|title=『渡くんの××が崩壊寸前』（第1巻）ヤンマガKCスペシャル版とカドカワコミック・エース版の違いは？比較してみた。|publisher=LH Magazine|date=2016-06-04|accessdate=2016-10-16|language=ja}}</ref>。而第1冊與第2冊的發行時間也相差逾1年半。

== 故事簡介 ==
生活的全部都以妹妹為中心的高中生渡直人，在姑姑家過著平和的日子，但時隔6年再次出現的青梅竹馬使這日常受到威脅⋯

== 登場人物 ==
; {{nihongo|渡 直人|渡 直人（わたり なおと）}}
: 本作主人公。16歲，高中2年級生，為學生委員會成員之一。原居於[[長野縣|長野縣]]{{註|作中以長野縣的古名信州稱之}}，但因後來父母雙亡{{註|時間為故事開始的2年前}}的關係而輾轉寄居於各親戚的家，現居於[[東京都|東京都]]郊外的姑姑家<ref>單行本第1冊第10至11頁</ref>。與妹妹鈴白相依為命，日常生活亦以妹妹為中心，也因此被同學認為是妹控。個性細心、為人著想，擅長廚藝，雖甘願為妹妹做各種事，但意識到如果她一直依賴自己的話長大後會一無是處，所以希望她能自立。與紗月為青梅竹馬，但6年前紗月無故破壞他的田地後不再與直人見面而疏遠，對轉學至此的紗月處處防備，但實質上仍很關心她。
:在19話被紫告白，於第22話以「試試看」的名義開始交往。

; {{nihongo|館花 紗月|館花 紗月（たちばな さつき）}}
: 女主角，16歲，高中2年C班學生。直人的青梅竹馬，但早於6年前疏遠，因某些原因破壞直人家的田地而被直人稱為「田地毀滅者」。有著一頭遮著左眼的長金髮的美少女，性格冷靜沉著，不擅長應付女生團體。喜歡直人，從長野轉學至東京後便一直跟蹤他，希望透過在各方面幫助直人來彌補自己6年前的過錯，由於是農家女兒的關係<ref>單行本第1冊第100頁</ref>，在耕作方面幫上直人許多，但在游泳方面則因經常翹掉游泳課而一竅不通。有在一間飯店裡打兼職。

; {{nihongo|石原 紫|石原 紫（いしはら ゆかり）}}
: 直人的同班同學，16歲，學生委員會成員之一。有著一頭齊肩的短黑髮與均等的身材，是全校公認的美少女，曾拒絕過《全國美少女圖鑑》星探的發掘。性格溫柔文靜。在初中時曾與總一郎有過一段「戀情」，但實質上只是總一郎的單方面追求，因此在升上高中後對這段往事產生陰影與改變了她對戀愛的定義，也因此對「與眾不同，無私、誠實而且純潔」的直人產生好感。
:在19話向直人告白，於第22話以「試試看」的名義開始交往。

; {{nihongo|德井 重|徳井 重（とくい しげ）}}
: 直人的同班同學與好朋友，在學校上稱得上是直人唯一要好的朋友。在學校中挺受女生歡迎但不擅長應付她們。

; {{nihongo|美尋|美尋（みひろ）}}
: 直人的同班同學，重的朋友。個性活潑，喜愛甜食，曾在超市購物時與直人相遇，對溫柔的直人抱好感。

; {{nihongo|渡 鈴白|渡 鈴白（わたり すずしろ）}}
: 直人的妹妹，10歲，小學4年生。重度兄控，日常生活都需要直人的幫忙，甚至只要分開便會大吵大鬧，但其實已經懂得自立，不過仍希望一直依賴哥哥。身高比同齡人矮，姑姑多摩代認為是遺傳，在第1話時向直人要求能種東西的田地，導致多摩代答應給予自己荒蕪的院子作耕作。

; {{nihongo|渡 多摩代|渡 多摩代（わたり たまよ）}}
: 直人的姑姑與房東，28歲，擁有與年齡不相稱的蘿莉身高與外貌，而自己亦很在意自己的身高，整天大部分時間都在房間中工作，似乎是職業作家<ref>單行本第1冊第13頁</ref>。由於擁有大房子的關係而讓直人與鈴白兩兄妹寄於籬下，雖然看似對他們很不友善，但實質上很關心他們兩兄妹。家裡飼養著一隻名叫「貓呆」的貓。

; {{nihongo|靜間|静間（しずま）}}
: 多摩代的美女責任編輯，經常到多摩代家催促她交原稿，偶爾會幫助直人照顧鈴白。

; {{nihongo|藤岡 總一郎|藤岡 総一郎（ふじおか そういちろう）}}
: 初中時期的紫的學長，比紫大兩屆的學生會長，家裡開大醫院的獨生子，全國模擬考試的名次也一直名列前茅。給人一種溫柔友善的印象，但不太理解別人感受。曾單方面追求紫，雖然被紫拒絕但仍一直希望她能與自己交往，甚至還提到結婚的事情，導致紫在升上高中後對這段往事產生陰影。在畢業後一直沒與紫聯絡，後來在她與直人們在海邊進行合宿時碰見她，並再次向紫提出交往的請求，最後再次被紫拒絕。
