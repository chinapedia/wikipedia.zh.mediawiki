'''八軒車站'''（{{jpn|j=八軒駅|hg=はちけんえき|rm=Hachiken eki|f=y}}）是一由[[北海道旅客鐵道|北海道旅客鐵道]]（JR北海道）所經營的[[鐵路車站|鐵路車站]]，位於[[日本|日本]][[北海道|北海道]][[札幌市|札幌市]][[西區_(札幌市)|西區]]，是JR北海道[[札沼線|札沼線]]（學園都市線）的一站，駐有站員。車站編號為'''G03'''。電報略號為'''{{lang|ja|ハケ}}'''。

== 歷史 ==
* 1988年（昭和63年）11月3日 - [[北海道旅客鐵道|北海道旅客鐵道]]（JR北海道）[[札沼線|札沼線]]'''八軒站'''開始營運，位於[[桑園車站|桑園車站]]至[[新川車站_(北海道)|新川站]]之間。當時為設有[[平交道|平交道]]的1面2線無人車站。同時桑園站與桑園站至本車站部份路段[[交通立體化|高架化]]<ref name=school>{{cite web|url=http://www.jrhokkaido.co.jp/press/2009/090909-2.pdf |language=ja |title=札沼線（学園都市線）の電化について |publisher=JR北海道 | date=2009-09-09 |accessdate=2017-08-31 |archive-url=https://web.archive.org/web/20091006234745/http://www.jrhokkaido.co.jp/press/2009/090909-2.pdf |archive-date=2009-10-06 }}</ref><ref name=school2>{{cite web |url=http://www.jrhokkaido.co.jp/press/2011/111013-3.pdf |language=ja |title=札沼線（学園都市線）の電化開業時期について |publisher=JR北海道 |date=2011-10-03 |accessdate=2017-08-31 |archive-url=https://web.archive.org/web/20111205160350/http://www.jrhokkaido.co.jp/press/2011/111013-3.pdf |archive-date=2011-12-05 |deadurl=yes }}</ref>。
* 1991年（平成3年）3月16日 - 札沼線的暱稱設定為「'''學園都市線'''」<ref name=school/><ref name=school2/>。
* 1996年（平成8年）6月23日 - 札沼線（學園都市線）桑園站至本車站，以及本車站至新川站路段高架化的同時，本車站改建成側式月台2面2線的高架車站<ref name="sone24"/>。
* 1997年（平成9年）3月22日 - 新站房落成。同時車站成為業務委託車站，駐有站員。
* 2000年（平成12年）
** 3月11日 - 新車站大樓落成。札沼線（學園都市線）本車站至太平站新設下行路線，成為[[複線鐵路|複線鐵路]]<ref name=school/><ref name=school2/>。
** 日期不詳 - 札沼線桑園車站至[[石狩月形車站|石狩月形車站]]路段引入自動進路制御裝置（PRC）。
* 2006年（平成18年）1月 - 引入自動列車導向裝置。並設置按方向的導向用液晶顯示屏。
* 2007年（平成19年）10月1日 - 設定車站編號（'''G05'''）<ref>{{cite web |url=http://www.jrhokkaido.co.jp/press/2007/070912-3.pdf |language=ja |title=駅番号表示（駅ナンバリング）を実施します |publisher=JR北海道 |date=2007-09-12 |accessdate=2017-08-31 |archive-url=https://web.archive.org/web/20070930015220/http://www.jrhokkaido.co.jp/press/2007/070912-3.pdf |archive-date=2007-09-30 |deadurl=yes }}</ref>。
* 2008年（平成20年）10月25日 - 開始使用[[儲值車票|智能車票]]「[[Kitaca|Kitaca]]」<ref>{{cite web |url=http://www.jrhokkaido.co.jp/press/2008/080910-1.pdf |language=ja |title=Kitacaサービス開始日決定について |publisher=JR北海道 |date=2008-09-10 |accessdate=2017-08-31 |archive-url=https://web.archive.org/web/20080913103748/http://www.jrhokkaido.co.jp/press/2008/080910-1.pdf |archive-date=2008-09-13 |deadurl=yes }}</ref>。
* 2012年（平成24年）6月1日 - 札沼線（學園都市線）桑園站至北海道醫療大學站正式電氣化（[[交流電|交流電]]20,000[[伏特|V]]、50[[赫兹|Hz]]）<ref>{{cite web|url=http://www.jrhokkaido.co.jp/press/2012/120314-1.pdf |language=ja |title=学園都市線電化開業に伴う電車の投入（第一次）について |publisher=JR北海道 | date=2012-03-14 |accessdate=2017-08-31 |archive-url=https://web.archive.org/web/20120925203437/http://www.jrhokkaido.co.jp/press/2012/120314-1.pdf |archive-date=2012-09-25 }}</ref><ref>{{cite web |url=http://www.jrhokkaido.co.jp/press/2012/120803-1.pdf |language=ja |title=平成24年10月ダイヤ改正について |publisher=JR北海道 |date=2012-08-03 |accessdate=2017-08-31 |archive-url=https://web.archive.org/web/20120925203427/http://www.jrhokkaido.co.jp/press/2012/120803-1.pdf |archive-date=2012-09-25 |deadurl=yes }}</ref>。
* 2017年（平成29年）2月27日 - 簡易[[驗票閘門|自動閘機]]更新為對應[[智慧卡|IC卡]]的新型號。

== 車站結構 ==
2面2線[[側式月台|側式月台]]（月台長度：135米<ref name=sapporojr>{{cite web|url=http://www.city.sapporo.jp/sogokotsu/kotsutaikei/documents/databook_3.pdf |language=ja |title=札幌の都市交通データ（3. JR） |publisher=札幌市公式ホームページ | date=2017-04-20 |accessdate=2017-08-31 |archive-url=https://web.archive.org/web/20170818094412/http://www.city.sapporo.jp/sogokotsu/kotsutaikei/documents/databook_3.pdf |archive-date=2017-08-18 }}</ref>）的高架車站。與月台以樓梯、升降機與上行專用扶手電梯連結。

由[[桑園車站|桑園車站]]管理的業務委託車站（北海道JR Servicenet），設有[[綠窗口|綠窗口]]（營業時間7時00分至22時00分）、[[自動售票機|自動售票機]]、自動[[驗票閘門|自動閘機]]（對應[[Kitaca|Kitaca]]及磁卡車票）。

直至2012年5月31日，本車站與[[新川車站_(北海道)|新川站]]及[[新琴似車站|新琴似車站]]是少數非電氣化的高架及複線鐵路車站。離開桑園站並進入八軒站前，列車於高架路段需要先緩慢下降，向右（東側）急轉後再緩慢上升，這是該路段進行高架化殘留的痕跡。此外，八軒站高架化後使用較少下方的空間及柱子，因此現時站房下方仍然殘留舊月台及路軌等。此外，高架化完成後，有一段時間使用當時位於新川站側的渡線，直至現時下行線於2000年3月11日完成後才移除。而進行電氣化工程期間亦使用[[鋼結構|鋼結構]]加強月台構造。

== 車站周邊 ==
位於札幌的市中心，周邊有大量建築物及公寓等。車站西側為札幌市立琴似中央小學，而東側稍遠的地方則有札幌工業學校、[[北海道武藏女子短期大學|北海道武藏女子短期大學]]及[[北海道大學|北海道大學]]等。
* [[函館本線|函館本線]][[琴似車站_(JR北海道)|琴似站]]
* 西警察署八軒交番
* 札幌八軒郵局
* 小樽信用金庫札幌分店
* [[北洋銀行|北洋銀行]]八軒分店
* Fretty{{lang|ja|フレッティ}}琴似店
* MaxValu八軒5條店
* 北海道中央巴士「八軒6條東2丁目」站牌
