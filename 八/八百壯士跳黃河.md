'''八百壯士跳黃河'''或称'''八百冷娃（楞娃<ref name="a1"/>）跳黄河'''，是指在1939年6月对[[日本|日本]]作战的[[中条山战役|中条山战役]]中，八百名十六、七岁的[[陕西人|陕西籍]][[国民革命军|国民革命军]]战士，弹尽粮绝后跳[[黄河|黄河]][[自杀|自杀]]的事迹。此事並未載於[[中華民國國軍|中華民國國軍]]任何戰史紀錄，其真伪在[[中国大陆地区|中国大陆地区]]亦存在争议，有些人认为“八百壮士跳黄河”是[[信史|信史]]，甚至有媒体还采访到一名事件的亲历者<ref>[http://www.yijun.gov.cn/xwzx/zwyw/15316.htm “八百壮士跳黄河”中唯一在世者]</ref>；也有人认为“八百壮士跳黄河”是艺术创作，并不是真实历史。<ref name="a1"/><ref name="a2"/>。

2011年，在[[山西省|山西省]][[芮城县|芮城县]][[陌南镇_(芮城县)|陌南镇]]建有“中条山抗日英雄 跳黄河殉国纪念碑”。报道称是为纪念[[中条山|中条山]]“八百壮士跳黄河”的事迹<ref>{{Cite web |url=http://view.news.qq.com/a/20120417/000015.htm |title=中条山跳黄河八百壮士纪念碑圣天湖畔落成 |accessdate=2015-10-03 |author=记者：董应赞|date=2012-04-17 |work= |publisher=，来源：三晋都市报2011年4月6日报道|language=Zh-hans }}</ref>。

2015年6月，[[中華民國陸軍軍官學校|中華民國陸軍軍官學校]]将
“[[八百壮士|八百壮士]]”[[2015年“八百壯士跳黃河”烏龍事件|误植为“八百壯士跳黃河”]]，引起台湾媒体广泛批评。

== 背景 ==

1949年10月，[[中国共产党|中国共产党]]取得[[第二次国共内战|第二次国共内战]]的胜利，建立[[中华人民共和国|中华人民共和国]]。战事失利的[[中华民国政府|中华民国政府]]被迫[[国府迁台|迁台]]。[[中华人民共和国政府|中华人民共和国政府]]及[[中华人民共和国宣传|宣传]]中，对于中华民国政府和[[中华民国国军|中华民国国军]]及领导人[[蒋介石|蒋介石]]在[[抗日战争|抗日战争]]中的评价，通常是“[[消极抗战、积极剿共|消极抗战、积极剿共]]<ref>{{Cite web |url=http://www.fusong.gov.cn/djzl/130909.jhtml |title=历史的见证和启示 ——纪念抗日战争暨世界反法斯战争胜利70周年 |accessdate=2015-10-03 |author=松江河镇关工委 |date=2015-08-12 |work= |publisher=抚松县人民政府门户网站，来源：抚松县人民政府 |language=Zh-hans |deadurl=yes |archiveurl=https://web.archive.org/web/20151005082520/http://www.fusong.gov.cn/djzl/130909.jhtml |archivedate=2015-10-05 }}</ref>”。对于中国共产党在抗日战争中的作用，通常称为“[[中流砥柱|中流砥柱]]<ref>{{Cite web |url=http://theory.people.com.cn/n/2015/0723/c40531-27346993.html |title=中国共产党是抗战中当之无愧的中流砥柱 |accessdate=2015-10-03 |author=胡德坤|date=2015年7月23日 |work= |publisher=人民网，来源：人民日报2015年07月23日07版|language=Zh-hans }}</ref>”。

中华民国政府兴建的抗日战争阵亡将士公墓和纪念性建筑物在1949年后，尤其是[[文化大革命|文化大革命]]期间，均遭受不同程度的破坏或被用于它途。进入1980年代后，中华人民共和国政府恢复了部分遭到破坏的国军将士墓葬和纪念性建筑，如[[国殇墓园|国殇墓园]]、[[国民革命军陆军第八十八师淞沪抗日阵亡将士纪念坊|国民革命军陆军第八十八师淞沪抗日阵亡将士纪念坊]]等。

中国大陆普通民众中，有赞扬国民革命军在抗日战争中表现的舆论观点，甚至被认为“拔高国军战绩，到神话的地步<ref>{{Cite web |url=http://www.guancha.cn/society/2014_04_20_223534.shtml|title=钓鱼文“国军血战诺曼底”登上湖北高考模拟题 网友：侮辱考试严肃性|accessdate=2015-10-03 |author= |date=2014-04-20 |work= |publisher=观察者网|language=Zh-hans }}</ref>”。反对者则有用“[[钓鱼文|钓鱼文]]”来反讽（参见[[国军52军浴血诺曼底|国军52军浴血诺曼底]]）。在网络流传的“八百壮士跳黄河”中，一直以来被中华人民共和国政府作为典型宣传的[[狼牙山五壮士|狼牙山五壮士]]则成为对比对象，八百壮士有着比狼牙山五壮士更高的评价。

== 考證 ==
八百壯士跳黃河之說，最早是因作家[[陳忠實|陳忠實]]在2004年9月4日發表的一篇文章：《一段几乎湮没的史实：中条山八百壮士血祭黄河》，後來為[[歷史小說|歷史小說]]《[[立马中条|立马中条]]》正式出版時的[[序|序]]。陳文引述了書中第17章:「血祭黃河」的情節，稱1939年6月6日，日軍進攻[[中條山|中條山]]的[[西北軍|西北軍]]防禦陣地，經過一番激戰後，有800名士兵不願被俘虜，選擇跳入[[黃河|黃河]]之中；而陳文之中又稱曾與該部小說的作者之一談論過此事的細節<ref name=news.qq.com>{{cite web|title=“八百国军投黄河”之真假|url=http://view.news.qq.com/zt2012/babailengwa/index.htm|publisher=[[騰訊網|騰訊網]]|accessdate=2015-06-22|language=zh-cn|deadurl=yes|archiveurl=https://web.archive.org/web/20150421233130/http://view.news.qq.com/zt2012/babailengwa/index.htm|archivedate=2015-04-21}}</ref>。2005年7月，《[[南都周刊|南都周刊]]》的一位記者採訪了陳文之中所提到的作者[[张君祥|张君祥]]。張稱，800名壯士集體跳黃河自盡，是「在历史真实的基础上，用艺术手法塑造的」，而800人的數字，是戰後[[第四集團軍|第四集團軍]]在河灘撈起了800多具[[屍體|屍體]]<ref name=nbweekly>{{cite web|author1=谢海涛|title=寻访“八百壮士集体投河”真相|url=http://www.nbweekly.com/column/xiehaitao/200903/25138.aspx|publisher=[[南都周刊|南都周刊]]|accessdate=2015-06-22|date=2009-03-26|deadurl=yes|archiveurl=https://web.archive.org/web/20150423200539/http://nbweekly.com/column/xiehaitao/200903/25138.aspx|archivedate=2015-04-23}}</ref>。

不過，一位中國大陸學者張恆則堅稱壯士跳黃河之事確實存在。他花費數年在當地訪查，並根據受訪者的記憶與說法，認為若併入前後各日計算，當年跳河而死的士兵絕對不只800人，可能達數千人<ref name=mod.gov.cn>{{cite web|author1=張亞利|title=1939年中條山保衛戰︰800陝西冷娃跳黃河|url=http://www.mod.gov.cn/big5/hist/2014-11/18/content_4559056_3.htm|publisher=[[中華人民共和國國防部|中華人民共和國國防部]]|accessdate=2015-06-22|date=2014-11-18}}</ref>。

中國知名企業家[[潘石屹|潘石屹]]的祖父潘尔燊，1925年考入[[黄埔军校|黄埔军校]]六期。任165师九八六团上校团长，于41年5月率部参加中条山战役，为掩护全军5万余官兵的撤退，率部在张店镇固守,2300多名勇士在撤离阵地时只剩7人。长子潘钟麟在战役中也壮烈殉国。據潘石屹在[[微博|微博]]上曾經自述大伯潘钟麟是掉进黄河。 <ref name=weibo>{{cite web|author1=潘石屹|title=清明节，我来到大伯墓前，给他的墓上捧了一筐新土。这是一座空坟，大伯潘钟麟在中条山抗战时阵亡了，没有找到尸首，村上与他一起当兵的回来说，大伯掉进黄河了。三叔说：你大伯没儿没女，你来多给他烧些纸钱，他就不受穷了。|url=http://weibo.com/1182391231/CdYe6atFZ|publisher=[[新浪微博|新浪微博]]|accessdate=2016-10-25|date=2015-04-19}}</ref>。

==== 至6月6日的記載 ====
根據日方其戰前的諜報資料，當時在此區域的部隊有陳碩儒指揮的177師主力，與17師、35師、47師，合計約2萬；王鎮華、孔從周的獨立46旅及47旅，其兵力合計亦為2萬，先前已經數次「討伐」此區的部隊，造成其極大損害，但由於其躲避追捕的方法巧妙，故未能根絕；此區部隊的老、幼雜兵甚多，裝備與士兵素質亦不佳，但抗日意識旺盛<ref name=26th />{{rp|6}}。此區主要「妨礙」治安工作的是177師，與楊子良、楊振芳指揮的[[游擊隊|游擊隊]]，並且會跟盤據在[[稷王山|稷王山]]的敵軍相互呼應，是地域警備上的禍根<ref name=26th />{{rp|2}}<ref>日方戰報中，並未將6月6日的陌南鎮戰鬥單獨列出，而是其平陸周圍地區掃蕩戰的一部份；而綜合日方戰報中，陌南鎮的戰鬥最多不超過七小時即已結束，當日中午以前，日軍便已攻下該鎮並設立指揮所，故日方反而是詳寫了6月6日以後的激戰。而陳文描述的6月6日「戰役」，在日方記錄中，僅為當時日軍一系列掃蕩戰的其中一個片段，綜合來源參照亦僅得數行敘述，所以本文另外摘入6日前的經過</ref>。

6月3日，高橋與關的兩個旅團，已經消滅了平陸[[張店鎮|張店鎮]]的敵軍<ref name=26th />{{rp|8}}；6月4日20時，奉第20師團5月26日命令，各隊開始往[[解縣|解縣]]集結。此前，山口集成飛行隊已協力將黃河對岸的敵砲兵陣地肅清<ref name=26th-2>{{cite web|author1=野砲兵第２６連隊|title=藤室部隊命令|url=http://www.jacar.go.jp/DAS/meta/image_C11111387900?IS_STYLE=default&IS_KIND=SimpleSummary&IS_TAG_S1=InfoD&IS_KEY_S1=%E8%97%A4%E5%AE%A4%E9%83%A8%E9%9A%8A&IS_LGC_S32=&IS_TAG_S32=&|accessdate=2015-06-23|language=ja|date=1939年}}</ref>{{rp|83}}；主力藤室大佐的部隊6月5日起，開始掃蕩解縣至陌南鎮的沿道敵軍。6月6日1時30分，開始佔領陌南鎮兩端的高地；3時30分佔領完成，6時30分，開始由高地向下射擊<ref name=26th />{{rp|16}}。由於沒有預想中的激烈抵抗，故分兵至陌上村再占領北側陣地<ref name=26th />{{rp|18}}；11時，藤室部隊已經進入陌南鎮建立隊本部，並且將敗走的敵軍壓迫到黃河邊消滅；葛趙村與周圍的敵軍也已一併包圍後徹底掃蕩，協力部隊也將大溝村至大溝南側地區的敵軍壓迫至黃河邊「徹底壞滅」<ref name=26th-2 />{{rp|91}}；18時，聯隊長抵達本部，下令砲擊仍在黃河左岸的敵軍<ref name=26th />{{rp|21}}；20時之前，在砲兵協力下，包括棗樹巷與道東村周圍地區的敵軍，也同樣被壓迫至黃河邊消滅，旅團司令部在陌南鎮設置完成<ref name=26th-2 />{{rp|97}}<ref>綜合日方參戰部隊戰報，有中國人大量遺棄屍體的記載。如砲兵26聯隊發現20至數十具屍體，步兵78聯隊目擊約500，在張店鎮發現「數千」被中國人遺棄的屍體，拋棄的武器彈藥「不能計數」，不過都是在6月7至10日先後發現，且都不是在黃河邊；而參謀本部記載的真正交戰日期是6月7日和8日，中國人交戰兵力是1400，都與6月6日不符合，故略過未摘。</ref>。

而投降的師團副官向日方坦承，第一日（6月6日）時，177師在須頭村，預定第二日往平陸地區移動；因第二日時遇上了出擊的日軍而措手不及，而改往西移動；但又碰到了向西出擊的日軍，此時部隊支離破碎，師部此後下落不明<ref name=78th />{{rp|1002}}。第一日時，177師原本要前往山上增援奪回陣地，但為時已晚，交戰時全線崩潰，緊急往平陸地區退卻，而日軍已經遮斷全部退路，最後只好由窯頭村往右逃至黃河邊；而師部在8日正午左右告知有3000人戰死、傷，其後即失去聯繫，副官告訴日方，傳聞師長從平陸東北方的谷地逃走<ref name=78th />{{rp|1003}}。

鄭培元還向日方上交了「感想文」。首先，他感謝皇軍對投降者的優待；其次，他抱怨抗戰將領只能聽從上官的強制命令，無怪乎戰必敗；第三，日本支那一強一弱明顯，勝敗已分；第四，皇軍聖戰的目的是為了世界和平，與防止東亞被赤化<ref name=78th />{{rp|1005}}。第五，日支提攜是東洋的真和平基礎，日支停戰則世界永久和平，他已有為了皇軍添力的覺悟；第六，日軍戰術巧妙，我軍雖知道由解縣集結，但不知第二天早上就發動攻擊，以致全線一齊崩潰；而日軍追擊至山下時又無對應處置，最後我軍左來右往間被殲滅。他更告訴日方，雖然昨（9日）一天之中日軍就消滅3000名敵人，但還有3000人自平陸以北地區遁走；第七，支那軍的缺點就是戰法呆滯不知變通，只會呆然防守陣地<ref name=78th />{{rp|1006}}；第八，中央對我軍的軍需費太過苛刻，以致我軍得向當地住民強制徵收，武器使用陝西省共產軍的裝備；第九，日支事變後支那軍還要再敗，[[蔣介石|蔣介石]]前途黯淡<ref name=78th />{{rp|1007}}。
