'''国民革命军第八路军西安办事处旧址'''位于[[西安市|西安市]][[新城区_(西安市)|新城区]]北新街附近的七贤庄1号；曾是全国15个[[八路军|八路军]]、[[新四军|新四军]]办事处中成立最早、坚持时间最长、影响最大的办事机构；<ref>{{Cite web |url=http://xatb.mofcom.gov.cn/article/tupxw/201308/20130800241446.shtml|title=西安特办开展专题党日活动|accessdate=2013-12-22|author= |date=2013-08-12 |work= |publisher=中华人民共和国商务部网|language=Zh-hans }}</ref> 1959年改建为'''八路军西安办事处纪念馆'''，包括七贤庄一、三、四、七号院；1988年公布为[[全国重点文物保护单位|全国重点文物保护单位]]。

== 历史沿革 ==
[[File:Xi'an_Balujun_Banshichu_Jiuzhi_2014.01.04_09-44-59.jpg|200px]]
1936年4月，原驻[[东北军|东北军]]的[[中国共产党|中国共产党]]代表[[刘鼎|刘鼎]]根据[[周恩来|周恩来]]的指示在西安古城内租用了七贤庄一号，作为中共的地下交通站和办事处；1936年12月，[[西安事变|西安事变]]和平解决后，这里成为半公开的“[[中国工农红军|红军]]联络处”。1937年4月,红军总供给部部长[[叶季壮|叶季壮]]率领人员在西安七贤庄4号院设立了与红军联络处平行的采办委员会，下设有运输科。

1937年8月，[[抗日民族统一战线|抗日民族统一战线]]成立后，[[中国工农红军|中国工农红军]]改编为[[国民革命军第八路军|国民革命军第八路军]]，原“红军联络处”也改名为“八路军驻陕办事处”，可以公开活动；[[林伯渠|林伯渠]]、[[董必武|董必武]]先后担任党代表；[[伍云甫|伍云甫]]、[[周子建|周子建]]等曾担任办事处处长。

到1939年,采办委共有汽车22辆。为加强汽车的使用管理,1939年10月,成立了“八路军西安采办委员会汽车队”。因车队驻地在西安市七贤庄15号,故社会上习惯称“七贤庄汽车队”。汽车队设有队长、指导员。每车配有正副司机各1名,并设有“押车排”,有押车副官10多人。

[[第二次国共内战|第二次国共内战]]全面爆发前夕，1946年9月，八路军西安办事处撤离。<ref>{{Cite web |url=http://travel.sina.com.cn/china/2008-10-14/140330021.shtml|title=八路军西安办事处旧址|accessdate=2013-12-22|author= |publisher=新浪网|language=Zh-hans}}</ref>

== 主要作用 ==
八路军驻西安办事处是1937年至1946年期间，中共设在国民党统治区的一个公开的合法机构；其主要任务是：宣传共产党的抗日主张，开展[[抗日民族统一战线|抗日民族统一战线]]工作；招收、输送进步青年去[[延安|延安]]；为[[陕甘宁边区|陕甘宁边区]]和抗日前线领取、转运和采购战争物资。[[周恩来|周恩来]]、[[朱德|朱德]]、[[张闻天|张闻天]]、[[任弼时|任弼时]]、[[刘少奇|刘少奇]]、[[秦邦宪|秦邦宪]]（[[博古|博古]]）、[[叶剑英|叶剑英]]、[[邓小平|邓小平]]、[[吴玉章|吴玉章]]等人曾多次在办事处工作、生活；[[白求恩|白求恩]]、[[柯棣华|柯棣华]]和[[史沫特莱|史沫特莱]]等也曾在此居住；人员最多时，七贤庄一号有300多人；后来的中共高级领导人，如[[江青|江青]]、[[陈慕华|陈慕华]]等，也都是经八路军西安办事处赴延安的<ref>{{Cite web |url=http://news.xinhuanet.com/book/2008-01/21/content_7462469.htm |title=江青进延安恰巧搭乘毛泽东坐的卡车|author=王凡 东平著 |publisher=中央党史出版社|language=Zh-hans}}</ref><ref>{{Cite web |url=http://www.xabb.org.cn/contents/13/6.html|title=原国务院副总理陈慕华重游“八办”|accessdate=2013-12-22|author=|publisher=八办官网|language=Zh-hans }}</ref>。

== 建筑 ==
七贤庄位于[[西安市|西安市]]北新街附近，毗邻[[西安市后宰门小学|西安市后宰门小学]]，是一处坐北朝南的庭院建筑群落，共有庭院10座。八路军西安办事处就设立在这座院落西头的一号院内。办事处南北长82米，东西宽17米，占地面积1300多平方米；建筑为砖木结构平房，硬山顶，土坯心表砖墙壁；南、西、北各设1门。内有接待室、会客室、办公室、重要领导人的住房、库房、厨房、电台室、译电室、机要室、救亡室等。
