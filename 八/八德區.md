'''八德區'''為[[中華民國|中華民國]][[桃園市|桃園市]]的[[區_(中華民國)|市轄區]]，面積33.7111平方公里，是桃園市面積最小的行政區。位於桃園市的東北部，北鄰[[桃園區|桃園區]]，東鄰[[新北市|新北市]][[鶯歌區|鶯歌區]]，西鄰[[中壢區|中壢區]]，西南連[[平鎮區|平鎮區]]，南接[[大溪區|大溪區]]，地勢南高北低，有[[茄苳溪|茄苳溪]]貫穿南北。至3月底[[人口|人口]]數有203,236人，人口密度約6,029人/平方公里，密度為[[桃園市|桃園市]]第二高行政區，僅次於[[桃園區|桃園區]]。本區近年來因廣豐自辦重劃及八德擴大都計開發，人口大為增長，成為全台發展快速區域之一。桃園縣於2014年12月25日升格為直轄市，原縣轄八德市改制為'''八德區'''。

== 歷史 ==
'''八德區'''舊名「八塊厝」（下含小字莊頭、莊尾、稻埕頭、連城、舊城、公館、租倉、城外等八處，今八德區公所、[[八德三元宮|三元宮]]一帶地區），相傳是[[清|清]][[乾隆|乾隆]]年間有[[謝姓|謝]]、[[蕭姓|蕭]]、[[邱姓|邱]]、[[呂姓|呂]]、[[賴姓|賴]]、[[黃姓|黃]]、[[吳姓|吳]]、[[李姓|李姓]]等八戶人家至此開墾，每姓各築一屋，因而得名。1920年改設「-{[[八塊庄|八塊庄]]}-」，劃歸[[新竹州|新竹州]][[桃園郡|桃園郡]]管轄。戰後改名[[桃園市|桃園縣]]八德鄉；1995年1月改制「八德市」，同時「村」改為「里」，但商業活動主要聚集在[[大湳_(桃園市)|大湳]]一帶。2014年12月25日桃園縣改制直轄市，八德市改制「八德區」。
記憶八德，八塊厝(閩南話)是八隻屋(詔安話)翻譯而來，清治時期邱呂為八德兩大姓，而邱呂祖籍皆是福建漳州詔安，因此八德幾乎是講詔安話，來到此地取名為八隻屋，而後清末日據時代閩南人移入，才被強勢語言慢慢同化。而八隻屋取名原因猜測是，墾兵來開墾土地時在這生活搭建的八間草屋而起名的。

*1995年，1月1日人口滿15萬，改制桃園縣八德市。
*2010年，11月[[特力屋|特力屋]]進駐八德。
*2012年，4月14日[[巧克力共和國|巧克力共和國]]開幕。
*2014年，5月3日[[興仁花園夜市|興仁花園夜市]]於長興路開幕。11月10日，統聯1659八德(大湳)-土城(永寧)通車。12月25日，改制桃園市八德區。
*2015年，1月桃拾生活廣場開幕。10月1日統聯1659改為A/B/C三線。A/B線延長至八德擴大都計介壽路二段與豐德路口(近大溪區交界)。
*2016年，2月大湳交流道下福德一路往八德方向，將190公尺道路拓寬成10線道46米寬。12月，[[廣豐新天地|廣豐新天地]]試營運。2017年1月12日正式開幕。
*2017年，4月茄苳路多年瓶頸由3米拓寬為8米。6月，桃園市立兒童美術館於[[廣豐新天地|廣豐新天地]]5F開幕。6月16日，位於八德擴大都計的「八德區棒壘球場」落成啟用。
*2018年，6月20日人口突破20萬人<ref>[https://udn.com/news/story/7324/3218190  桃園升格至今 人口破220萬 六都最年輕]</ref>。8月,八德國民運動中心位在國際路與廣福路口(土地面積5,000坪)原「大湳北景雲營區」動工。10月1日，統聯1659A/B/C移撥給桃園市政府，變更為市區公車，路線編號變更為713、715、716。10月15日桃園捷運綠線在八德擴大都計GC01標高架八德段舉辦動工典禮。12月位於長興路尾(霄裡與龍岡交界)的簡易綠美化公園「花舞霄裡HAKKA」落成<ref>[http://news.ltn.com.tw/news/life/breakingnews/2639736  花舞霄裡HAKKA進行曲 七彩花海迎賓]</ref>。興豐路榮興路口至八德埤塘公園之間增設約600公尺人行道。
*2019年，1月人口密度每平方公里突破6000人大關。2月,茄冬溪成功橋改建工程完成，橋體拓寬到長30公尺、寬8公尺。3月4日新增統聯717龍岡經大湳到永寧站班次。

== 行政 ==
以八德區公所為最高行政機關，設區長1人，承桃園市市長之命綜理區政，置主任秘書、秘書、視導及民政課、社會課、農經課、工務課、秘書室、人事室、人文課、會計室、政風室等課室，公有八德市場、公有大湳市場等附屬單位。

清潔隊、托兒所、圖書館已回歸由市政府相關局處管理。

都市發展課、公園路燈管理所相關業務合併至工務課及農經課。

計畫室合併至秘書室。

財政課相關業務回歸至稅務局及國稅局。

== 都市計劃 ==
[[File:2019八德擴大都計.jpg|thumb]]

=== 大湳地區都市計劃 ===
八德大湳地區都市計畫區範圍以[[大成國小|大成國小]]附近商業區為中心，包括介壽路兩旁，南至大潤發賣場，北至桃園都市計畫界線，總面積約409.27[[公頃|公頃]]；目前為八德區人口聚集的主要地區。

=== 八德地區(擴大)都市計劃 ===
[[File:八德都市計畫區.png|thumb]]及遠處的[[觀音山_(新北市)|觀音山]]；右後為[[陽明山|陽明山]]]]

==== 歷史 ====
本特定區原是配合[[台灣高速鐵路|台灣高鐵]][[桃園車站_(高鐵)|桃園車站]]與[[國家衛生研究院_(台灣)|國衛院]]等兩大基礎建設將在該區設立而規劃，惟後來皆未在該區設立。目前則是以八德區公所為中心，1994年起由地方開始規劃，2003年送交[[中華民國內政部|內政部]]審議，經過無數次的溝通協調，面積由原來650公頃驟減至164.78公頃。2009年1月內政部審核通過第一期164公頃的開發計劃案（土地變更），已於2010年初完成，未來還將持續規劃進行第2期、第3期的都市計劃開發案。2016年桃園市政府整併全市33處都市計畫區，將此區整併入桃園都會區都市計畫。

==== 規模 ====

===== 第一期 =====
八德擴大都市計劃區內，目前有二條寬達50米的大道（建德路、豐德路），以及建置長達28公里的寬頻管道，還有[[桃園捷運|桃園捷運]]綠線的捷運車站專區，形成環狀的捷運網絡系統，並與大台北捷運系統接軌（目前尚在規劃中）。八德擴大都市計畫廣置公園，瑞興、福興、興仁、瑞豐、瑞德等里都畫設公園用地，還特意保存區內4口水利會大池塘，讓它成為自然生態公園，成為八德區主要特色。還有總長8公里多環區的自行車道。區內西邊還保留2.4公頃作為[[國立中央大學|中央大學]]八德校區<ref>[https://udn.com/news/story/7324/2600475  中大未放棄八德校區 鄭文燦：先做公園]</ref>。東側豐德路介壽路口增設國道三號大鶯豐德交流道(暫名)可行性評估市政府已於2018年9月送交通部高公局審議。<ref>[http://www.tycg.gov.tw/ch/home.jsp?id=9&parentpath=0,1&mcustomize=news_view.jsp&dataserno=201605020004&aplistdn=ou=news,ou=chinese,ou=ap_root,o=tycg,c=tw&toolsflag=Y  八德第二交流道可行性研究送中央審議]</ref>
八德擴大都計的第三條大道(園三)，路寬40米，本為生活圈六號道路，現更名為「八德區介壽路至建德路新闢道路」。長度約1.7公里，未來會從八德擴大都市計畫建德路、仁德一路口沿仁德一路，穿越建國路、豐田路沿著產業道路，最後再連接介壽路二段六八五巷（伍陽農場）旁，新闢道路會配合捷運綠線土木工程，將會委託捷運施工單位一併施工。

==== 與文化資產的衝突 ====
葉金萬是桃園八塊厝當地人，生於[[清代|清代]][[道光|道光]]21年（1841年），歿於[[台灣日治時期|日治時代]][[昭和|昭和]]5年（1930年），享年89歲，是台灣當時著名的建廟老師傅，八德區三元宮殿內的木樑及雕飾，古樸典雅，就是出自此名木作大師葉金萬之手。其故居「樹德堂」，原為葉氏祖厝宗祠，就位於三元宮旁；古厝建造於[[咸豐|咸豐]]4年（1854年），為傳統三合院格局。卻在此次八德擴大都市計劃中，古厝被拆除，夷為平地。<ref>[http://www.tonyhuang39.com/tony0531/tony0531.html  桃園八德．三元宮．呂氏著存堂．日本神風特攻隊隊長駐所舊址 【Tony的自然人文旅記 】]</ref>。
