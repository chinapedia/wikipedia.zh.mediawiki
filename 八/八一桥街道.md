== 简介 ==

== 历史沿革 ==
1949年成立积谷仓街行政事务所，1951年改积谷仓街公所，1953年分为大士院、半步街居委会，1954年并为大士院街道，1960年更名八一桥公社，1979年改称八一桥街道。

1997年时辖区面积0.43平方千米，人口3.6万，辖大士院住宅小区一区一，二区一、二、三、四，三区一、二、三、四，四区一、二、三、四，阳明路西，中山堂，中山堂侧，豫章后街、芭茅巷一、二、裘家厂一巷、三巷、东万宜巷、西濠街等23个居委会，办事处驻半步街72号。<ref>[http://www.xzqh.org/html/show/jx/9739.html 东湖区八一桥街道_行政区划网(区划地名网)]</ref>

2012年西濠街社区改名为叠山路西社区<ref>[http://www.stats.gov.cn/zjtj/tjbz/tjyqhdmhcxhfdm/2011/36/01/02/360102003.html 2011年] {{webarchive|url=https://web.archive.org/web/20150924124950/http://www.stats.gov.cn/zjtj/tjbz/tjyqhdmhcxhfdm/2011/36/01/02/360102003.html |date=2015-09-24 }}和[http://www.stats.gov.cn/zjtj/tjbz/tjyqhdmhcxhfdm/2012/36/01/02/360102003.html 2012年] {{webarchive|url=https://web.archive.org/web/20140714151501/http://www.stats.gov.cn/zjtj/tjbz/tjyqhdmhcxhfdm/2012/36/01/02/360102003.html |date=2014-07-14 }}</ref>。

== 人口 ==
