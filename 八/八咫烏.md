'''八{{ruby-py|咫|-{zh-cn:zhǐ;zh-tw:ㄓˇ}-}}烏'''（{{jpn|j=八咫烏|hg=やたがらす（やたのからす）|rm=Yatagarasu (Yatanokarasu)}}）乃[[日本神話|日本神話]]故事[[神武東征|神武東征]]之際，受[[高皇產靈神|高皇產靈神]]和[[天照大神|天照大神]]之命，從{{link-ja|熊野國造|熊野国造|熊野國}}到[[大和國|大和國]]為[[神武天皇|神武天皇]]帶路的[[烏鴉|烏鴉]]，一般描繪為三隻腳的形象。

== 概要 ==
在[[熊野三山|熊野三山]]，烏鴉作為「ミサキ」神（[[神使|神使]]）受到信仰，日本神話中登場的八咫烏被視為太陽的化身。近世以前常用在起請文的熊野{{link-ja|熊野牛王符|熊野牛王符|牛玉寶印}}繪有烏鴉。《[[新撰姓氏錄|新撰姓氏錄]]》中，八咫烏是高皇產靈尊的曾孫[[賀茂建角身命|賀茂建角身命]]的化身，之後成為鴨縣主之祖，故[[奈良縣|奈良縣]][[宇陀市|宇陀市]]榛原區的{{link-ja|八咫烏神社|八咫烏神社}}以[[賀茂建角身命|賀茂建角身命]]為祭神。此外，八咫烏也是[[戰國_(日本)|戰國時代]][[紀伊國|紀伊國]]的[[雜賀眾|雜賀眾]][[鈴木氏|鈴木家]]的旗印。

== 和金鵄的關係 ==
[[File:Emperor_Jimmu.jpg|thumb]]《大日本名將鑑》的「神武天皇」。場面是金鵄，但是圖卻是黑色的「金鵄」。]]
《[[記紀|記紀]]》二書曾提到八咫烏，在《[[日本書紀|日本書紀]]》中神武東征時，在天皇和[[長髓彦|長髓彦]]作戰之際，協助神武天皇的[[金鵄|金鵄]]（金色的[[鳶|鳶]]）經常被混淆為八咫烏。

== 象徵 ==
在現代，八咫烏成為[[日本足球協會|日本足球協會]]等組織的象徵。

== 小行星 ==
[[群馬縣|群馬縣]][[大泉町|大泉町]]的天文家[[小林隆男|小林隆男]]將1997年發現的[[小行星|小行星]]（[[天文學臨時編號|臨時編號]]1997 AY1）命名為「八咫烏」，2004年8月9日以（9106）[[八咫烏_(小行星)|八咫烏]]之名登錄。
