'''八寶粥'''，[[琉球語|琉球語]]稱為'''{{lang|ryu|あまがし}}'''，英语称为Mixed Congee，本意是指用八种不同的材料熬制成的粥，在今天是一種以多種[[豆類|豆類]]、[[穀物|穀物]]等食材烹調而成的甜[[粥|粥]]，其材料因應地區和個人喜好有所變化，不同品牌的八宝粥用料也不同，但是基本上是米，豆类，干果类，中药材。家庭熬制“八宝粥”，有时还会加板栗、胡萝卜、香肠、咸肉等。八寶粥常用作節日食品，如[[中國|中國]]部份地區於[[臘八節|臘八節]]食用八寶粥，稱為[[臘八粥|臘八粥]]，中國南方部份地區和[[琉球|琉球]]於[[端午節|端午節]]食用，也有些地區是[[十二月廿五|十二月廿五]]、[[正月|正月]]等日子吃八寶粥。八宝粥所用食材与制作都很简单，食用时可根据个人口味选择加不加糖、牛奶等。

== 起源 ==
八宝粥也叫腊八粥，关于八宝粥的来历有不同的说法，其中一个是讲，朱元璋在当皇帝之前，在一次战争中饥寒交迫，他四处找吃的。在田边发现了一只很肥的老鼠，跑进老鼠洞里，他走进那个老鼠洞，往里面掏了掏，发现了各种各样的粮食，其中就有小米，花生，红豆等。他将这些粮食清洗干净，混在一起煮成了粥，吃起来非常好吃。他在做皇帝后，仍然想念当时自己做的粥，便让御厨做这种粥，并且加入了莲子，核桃等，使其更美味，命名“腊八粥”，后流传民间，延续至今。

== 八宝 ==
传统八宝即红小豆，小米，绿豆，核桃仁，花生，红枣，山药，莲子

== 制作方法 ==
'''腊八粥'''
* 材料：主料：大米 配料：红豆，枸杞，银耳，香菇，红枣，莲子，玉米
* 制作方法：
# 红豆，枸杞用冷水浸泡40分钟；银耳，香菇用热水浸泡直至水冷却
# 将洗净的红枣莲子玉米大米和浸泡好的材料一起放入锅中，大火煮开改小火慢熬2-3小时即可

'''儿童八宝粥'''
* 材料：主料：糯米 配料：红小豆，绿豆，蜜枣，桂圆肉，莲子，花生，枸杞
* 制作方法：
# 红小豆，绿豆，枸杞冷水浸泡40分钟
# 将洗净的糯米倒入砂锅，加入其余材料，中火熬20分钟改小火继续熬至浓稠。
* 注：莲子，枸杞3-5克即可不易过多。

'''孕妇八宝粥'''
* 材料：主料：糯米  配料：红小豆，红豆，莲子，枸杞，蜜枣，山楂条，葡萄干
* 制作方法：
# 红小豆，红豆，枸杞浸泡40分钟
# 将洗净的糯米放入砂锅，加入材料大火熬开改小火慢熬1-2小时，出锅前加入白糖在熬10分钟
* 注：白糖适量即可

'''老人八宝粥'''
* 材料： 主料：糯米，小米  配料：红小豆，绿豆，红枣，香芋，莲子，花生
* 制作方法：
# 将糯米浸泡一段时间，将红小豆，绿豆，花生浸泡40分钟
# 香芋去皮切块，将糯米，小米倒入砂锅，下剩余洗净的材料熬煮。中火慢熬40分钟，改小火慢熬1-2小时即可。
* 注：将食物尽量熬至软糯。

'''健胃八宝粥'''
* 材料：主料：大米，薏仁  配料：红豆，百合，绿豆，莲子，枸杞，山药
* 制作方法：
# 薏仁，红豆，绿豆浸泡一段时间
# 将洗净的材料放入锅中，开火慢熬，小火熬制软烂即可。
* 注：百合，莲子适量。

== 营养价值==
八宝粥的食材非常丰富，富含谷类豆类。能够补充我们一餐当中吃不到的许多食物；并且清淡，热量较低可以去除身体内多余的脂肪油脂。谷类与豆类的平衡搭配，补充蛋白质，混合煮粥能够充分发挥氨基酸作用。对于老人，孩童可以平衡膳食，补充营养，预防疾病；孕妇可以补血补气。同时对于减肥的人群也可以补充身体所需，健康减肥

== 八宝粥品牌对比 ==
* 娃哈哈八宝粥：味道偏甜比较稀，红豆颗粒大，加热后使用味道更佳，是先熬煮在封罐。
* 银鹭八宝粥：粥更为浓稠，味道甜，种类多样，是生料入罐在进行熬煮。
