'''橫芝光町'''（{{jpn|j=横芝光町|hg=よこしばひかりまち|rm=Yokoshibahikari machi|f=y}}）是[[千葉縣|千葉縣]]東北部的一[[町|町]]。

合併前的橫芝町與光町分別屬不同郡（歷史上分別屬[[上總國|上總國]]與[[下總國|下總國]]），現在橫芝光町屬山武郡。

[[File:Hikari_Chiba.JPG|缩略图]]

== 地理 ==
位於千葉縣[[太平洋|太平洋]]側、{{link-ja|九十九里平原|九十九里平野}}中央。町北部地勢較高，但起伏不大。九十九里平原最大的河川{{link-ja|栗山川|栗山川}}流經町中央。

== 歷史 ==
* [[繩文時代|繩文]]前期：大部分仍為海洋（{{link-ja|繩文海進|縄文海進}}）。
* 繩文中期：北部中台等地已有人類生活（[[貝塚|貝塚]]）。
* 繩文後期：栗山川[[河口|河口]]形成[[潟湖|潟湖]]{{link-ja|坂田池|坂田池}}與{{link-ja|乾草沼|乾草沼}}。
* 繩文晩期：[[東京灣|東京灣]]岸人口減少，九十九里沿岸人類持續活動，本町姥山已發現當時的{{link-ja|標式遺跡|標式遺跡}}{{link-ja|山武姥山貝塚|山武姥山貝塚}}。
* [[彌生時代|彌生時代]]：海岸線後退，現在的平原地帶稻作興盛。
* [[古墳時代|古墳時代]]：本町小川台發現與{{link-ja|下海上國造|下海上国造}}有關的{{link-ja|小川台古墳群|小川台古墳群}}，中台發現與{{link-ja|武社國造|武社国造}}有關的{{link-ja|芝山古墳群|芝山古墳群}}。
* [[平安時代|平安時代]]：本町屋形是{{link-ja|平高望|平高望}}與{{link-ja|平良兼|平良兼}}等[[平氏|平氏]]據點。
* [[室町時代|室町時代]]：建造{{link-ja|篠本城|篠本城}}（本町篠本）、{{link-ja|坂田城|坂田城}}（坂田篠本）。

=== 沿革 ===
==== 橫芝光町成立前 ====
* 1897年（[[明治|明治]]30年）6月1日 - {{link-ja|橫芝站|横芝駅}}開業。
* 1953年（[[昭和|昭和]]28年）5月18日 - 制定{{link-ja|國道126號|国道126号}}。
* 1998年（[[平成|平成]]10年）3月30日 - [[千葉東金道路|千葉東金道路]][[松尾橫芝IC|松尾橫芝交流道]]啟用。
* 2006年（平成18年）3月25日 - {{link-ja|銚子連絡道路|銚子連絡道路}}[[橫芝光IC|橫芝光交流道]]啟用。

==== 橫芝光町成立後 ====
* 2006年（平成18年）3月27日 - [[山武郡|山武郡]]{{link-ja|橫芝町|横芝町}}與{{link-ja|匝瑳郡|匝瑳郡}}{{link-ja|光町 (千葉縣)|光町 (千葉県)|光町}}合併成立'''橫芝光町'''。
* 2011年（平成23年）3月11日 - [[東日本大地震|東日本大地震]]，海嘯造成水災<ref>{{cite web |url=http://www.pref.chiba.lg.jp/kouhou/h23touhoku/yokoshibahikari.html |title=横芝光町の被災状況 |publisher=千葉県 |date=2011-07-15 |accessdate=2011-07-18 |deadurl=yes |archiveurl=https://web.archive.org/web/20120121024828/http://www.pref.chiba.lg.jp/kouhou/h23touhoku/yokoshibahikari.html |archivedate=2012-01-21 }}</ref>。
