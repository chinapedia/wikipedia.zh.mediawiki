'''橫內尚樹'''（{{lang-ja|横内なおき}}，1966年12月3日 - ），[[日本|日本]]人，居住在[[北海道|北海道]]的男性[[漫畫|漫畫]]家，講談社旗下漫畫月刊[[Comic_BomBom|Comic BomBom]]（已休刊）作者。

== 作品 ==

*[[霹靂酷樂貓|霹靂酷樂貓]]（日文：サイボーグクロちゃん，Comic BomBom 1997年9月号～2001年12月号連載，2015年11月份推出新裝版）

*ウッディケーン（Comic BomBom 2002年3月号～2003年5月号連載 全3集）

*{{lang|ja|サイボーグクロちゃん番外バトル}}（Comic BomBom 2002年12月号～2005年12月号連載 原作：橫內尚樹，漫畫：內田淳太（{{lang|ja|内田じゅんた}}）全3集）

*[[霹靂酷樂貓_格林機關槍再次上膛|霹靂酷樂貓 格林機關槍再次上膛]]（日文：サイボーグクロちゃんガトリングセレクション リローデッド，講談社、2012年7月23日）

*宇宙のガズゥ（2011年於繪圖網站PIXIV不定期刊載的網路漫畫，2016年講談社宣佈單行本化）

== 其他 ==
*2009年8月擔任[[迪士尼|迪士尼]]機器人短篇動畫[[火球_(动画)|FIRE BALL]]（ファイヤーボール）官方部落格[[LOGO|LOGO]]設計以及NEW TYPE月刊ファイヤーボール的[[四格漫畫|四格漫畫]]繪製，目前最新作品是在PIXIV公開的漫畫『{{lang|ja|宇宙のガズゥ}}』，2012年1月出版了霹靂酷樂貓漫畫精選集『{{lang|ja|復活！サイボーグクロちゃん ガトリングセレクション}}』，第二部霹靂酷樂貓漫畫精選集『{{lang|ja|復活！サイボーグクロちゃんガトリングセレクション リローデッド}}』於2012年7月發行。
