'''橫紋擬麗魚'''，為[[輻鰭魚綱|輻鰭魚綱]][[鱸形目|鱸形目]][[隆頭魚亞目|隆頭魚亞目]][[慈鯛科|慈鯛科]]的一[[種|種]]。被[[IUCN|IUCN]]列為次級[[保育動物|保育動物]]。

== 分布 ==
本魚分布於[[東非|東非]][[馬拉威湖|馬拉威湖]]。

== 特徵 ==
本魚體色以棕色為底，身體上顏色有相當多種，有黃色、藍色、橘色甚至白色，因此外貌特徵難以描述。魚體上具有許多模糊的深色直條紋。雌魚和雄魚相比，顏色較淡。眼大，高位。背鰭圓形。體長可達14公分。

== 生態 ==
本魚棲息在湖泊的岩石區，屬雜食性，以[[藻類|藻類]]、[[浮游生物|浮游生物]]為食，具有領域性。

== 經濟利用 ==
屬於高經濟價值的觀賞魚。
