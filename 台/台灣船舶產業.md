'''台灣船舶產業'''，'''台灣造船產業'''是指在[[台灣|台灣]]開設製造修繕船艦相關公司組成的產業，2015年台灣有135家船廠加入造船同業公會，從業人數2萬5千人，從業人員平均年齡為45歲，1997年到2001年的年均產值為340.418億新台幣；2006年到2015年的年均產值為569.18億新台幣；2016年台灣船舶產值達到623億元新台幣。以船舶噸位數統計，台灣排名世界第11名，佔有噸位比率百分之2.62；以交船數量統計，台灣在2016年排名世界第5，佔有率百分之0.7；台灣製造的遊艇市佔率排在世界第7名。<ref name = 艦船與海洋工程產業特性與發展前景_2016> {{ cite web | title = 艦船與海洋工程產業特性與發展前景 | url = http://ocean.kh.edu.tw/wSite/public/Data/f1473650680039.pdf | publisher = 台灣國際造船公司 | author = 陳豊霖 總經理 | date = 2016-07-18 }} </ref><ref name = 臺灣船舶產業概況_陳宗慶_2017>{{cite news |author1=陳宗慶 |title=臺灣船舶產業概況 |url=http://www.chinatimes.com/newspapers/20170525000136-260210 |accessdate=2018-08-14 |work=工商時報 |date=2017年05月25日}}</ref><ref name = "taiwan ship net! 台灣船舶網_2018"> {{ cite web | title = taiwan ship net! 台灣船舶網 | url = http://www.ship.org.tw/ | language = zh-tw | accessdate = 2018-08-14 }} </ref>

== 歷史 ==
[[鄭成功政權|鄭成功政權]]時期，在台南設有修造船廠。

[[滿清政府|滿清政府]]時期，有官營修造中式帆船廠。

1916年，[[台灣日治時期|台灣日治時期]]，日本礦業家木村九太郎在基隆成立木村鐵工所。1918年建造船塢，從事修造船舶與小汽艇。

1922年，台灣日治時期，日本總督府投資成立員工400人的「基隆船渠株式會社」。

1937年，台灣日治時期，[[日本|日本]][[三菱重工業|三菱重工業株式會社]]（目前世界第二大造船廠）投資成立「'''台灣船渠株式會社'''」，設廠於[[基隆市|基隆]][[和平島|和平島]]，為台灣第一家以[[詹姆斯·瓦特|瓦特]][[蒸汽機|蒸汽機]]為基礎現代化造船廠。<ref name = 台灣造船工業之演進簡史_2011> {{ cite web | title = 台灣造船工業之演進簡史 | url = http://www.cie.org.tw/khc/magaz100Y/18--%E5%8F%B0%E7%81%A3%E9%80%A0%E8%88%B9%E5%B7%A5%E6%A5%AD%E4%B9%8B%E6%BC%94%E9%80%B2%E7%B0%A1%E5%8F%B2.pdf | publisher = 百年紀念專刊 (中工高雄會刊 第18卷 第4期)  | author = 黃正清 教授，何政龍 助理驗船師 | date = 2011-05-11 }} </ref>

1946年7月，台灣進入[[台灣戰後時期|中華民國時期]]，[[資源委員會|資源委員會]]介入重整日本在台工業；原日企「台灣船渠株式會社」、「株式會社台灣鐵工所」合併成立「'''台灣機械造船公司'''」。下轄基隆造船廠、高雄機器廠，為資源委員會與[[台灣省政府|台灣省政府]]合資之[[台灣省政府|台灣省營]]事業機構。

1948年，台灣機械造船公司機器廠、造船廠分家，分別改組為「'''台灣機械公司'''」（簡稱'''台機'''，位於[[高雄市|高雄市]]）<ref>1990年起，鋼鐵廠售予[[統一企業|統一集團]]、船舶廠售予[[東南水泥|東南水泥]]、合金鋼廠售予[[隆成發鐵工廠|隆成發鐵工廠]]；2001年11月19日，台機公司製造廠及總公司售予[[中國鋼鐵|中國鋼鐵]]，民營化終於劃上句點。《鏗鏘已遠：台機公司獨特的一百年》，陳政宏，行政院文建會文化資產總管理處籌備處（今行政院[[文化部|文化部]]），216頁，2007年4月，ISBN 978-986-009-388-9</ref><ref>{{cite web |url=http://www.cw.com.tw/article/article.action?id=23763 |title=存档副本 |accessdate=2013-09-26 |deadurl=yes |archiveurl=https://web.archive.org/web/20130927124508/http://www.cw.com.tw/article/article.action?id=23763 |archivedate=2013-09-27 }} - [[天下雜誌|天下雜誌]]86期，作者：楊瑪利，1988-07-01</ref>及「'''台灣造船公司'''」（簡稱'''台船'''，位於[[基隆市|基隆市]][[和平島|和平島]]），個別獨立經營。

1957年，「台灣機構造船有限公司」與美國合資組成「英格爾斯台灣造船幹塢公司」，該公司到1962年解散。

1965年，台灣政府與日本石川島播磨重工簽訂造船協議。

1976年，成立高雄船廠。1977年7月基隆船廠與高雄船廠合併成為「中國造船公司」，簡稱中船，當時中國造船公司的造船量約佔全台灣造船量的百分之90。

1970年代末期，中船公司藉由南韓的設計圖，以及美國的技術指導，開始建造小型軍用艦艇。

=== 遊艇廠 ===
台灣的遊艇製造始於1958年到1990年代，是駐台[[美軍|美軍]]引進與國內[[舢舨|舢舨]]廠商結合而開啟；後續與國外客戶以遊艇設計圖找尋國內業者[[代工生產|代工生產]]方式承接訂單，從國外引進[[玻璃纖維強化塑膠|玻璃纖維強化塑膠]]製造技術、某些廠商直接向客戶購買設計圖進行量產降低成本，訂單快速成長後利潤加大；在1990到1999年代，因國內廠商只專注在產業鍊的代工生產，遊艇主要市場的美國市場萎縮，遊艇廠商開始分散市場與尋求技術升級，以及加強客製化與提升附加價值；到了2000年到2011年代以後，台灣遊艇廠商開始進行[[策略聯盟|策略聯盟]]、改善製造工程的基礎設施後，走向客製化與巨型遊艇等更大市場發展。<ref name = 臺灣遊艇產業發展脈絡之研究_2012>{{cite web | title = 臺灣遊艇產業發展脈絡之研究 | url = http://ir.csu.edu.tw/dspace/bitstream/987654321/3149/2/%E5%AD%B8%E4%BD%8D%E8%AB%96%E6%96%87.pdf | publisher = 國立高雄師範大學地理系 | author = 陳振杰 | language = zh-tw | date = 2012-06-01}}</ref>

=== 遊艇廠 ===
目前台灣遊艇廠商製造技術發展包括：<ref name = 國際造船產業及航運市場趨勢暨船舶中心簡介_2009> {{ cite web | title = 國際造船產業及航運市場趨勢暨船舶中心簡介 | url = http://w3.sname.ncku.edu.tw/ncku_chinese/ocean/pdf/20090227class.pdf | author = 黃正利 | date = 2009-03-02 }} </ref>
* 提升表面精質度：新型態纖維編織法可以提升船殼表面精緻度更細緻平滑
* 組合式傢俱：將遊艇內傢俱元件模組化，可以提高遊艇傢俱生產製造效率，降低遊艇傢俱生產成本
* 真空輔助樹脂轉注成型：可以提高傳統手工注入樹脂的效率與穩定性，並更容易製造模具、減少傳統樹脂轉注成型的氣泡、可以製造更大船殼

== 造船廠 ==
目前以造船船體與零件規模由大到小分類如下：

=== 大型船廠 ===
[[台灣國際造船|台灣國際造船]]，主要造船為貨櫃輪、散裝貨輪、巡防艦等大型船舶，以及油輪、半潛式重載船、冷藏船、升降式鑽油平台、自卸式水泥運輸船等各型船舶。其中目前最大船舶為2016年間所建造14,198[[TEU|TEU]]級載貨能力，載重噸位146,073頓的[[陽明海運|陽明海運]]貨櫃輪。廠區分布在[[高雄|高雄]]與[[基隆|基隆]]地區。

=== 中型船廠 ===
有[[中信造船|中信造船]]、[[慶富造船|慶富造船]]、[[龍德造船|龍德造船]]與[[三陽造船廠|三陽造船廠]]等四家建造海巡艦艇、公務船、鰹鮪延繩釣漁船、（美式）圍網漁船、遠洋漁船，交通船、渡輪、觀光船、旅客船、拖船、海洋研究船與平台船等各類型船舶、[[漁船|漁船]]與海工業務的中大型船廠，分布在[[高雄港|高雄港]]、[[基隆港|基隆港]]與宜蘭[[蘇澳港|蘇澳港]]等地區。

軍艦方面可製造巡防艦、（飛彈）巡邏艦、戰鬥支援艦、海巡艦艇與飛彈快艇等船艦。

=== 小型船廠 ===
有喜長發、健富、靖海等78家生產玻纖漁船、小型巡邏艇為主的小型修造船廠。

=== 遊艇廠 ===
有嘉鴻、高鼎、嘉信、東哥、大瑞、隆宜企業與統怡等36家廠商，製造船舶有玻璃纖維遊艇、豪華遊艇、客製化中大型動力遊艇與高級訂製帆船；遊艇建造範圍由38呎到150呎的超級遊艇，多數為60呎到80呎的豪華遊艇。

=== 船舶零件廠 ===
除了船舶整體建造外，台灣也有許多零件製造廠商，包括船舶螺槳廠、船舶五金廠、船舶資訊設備廠與船舶艤品等37家廠商。

* 船外機螺旋槳廠：般若科技股份有限公司
*船內機螺旋槳廠:宏一新螺旋槳公司、瑞孚宏昌船舶推進系統股份有限公司、日昇船舶科技有限公司、宏昇螺旋槳股份有限公司等公司，主要生產直徑5公尺以下螺槳。<ref name = "自己的潛艇自己造　台灣潛艦靠他們_2016"> {{ cite web | title = 自己的潛艇自己造　台灣潛艦靠他們 | url = https://www.cw.com.tw/article/article.action?id=5076635 | publisher = 天下雜誌 | author = 鄧凱元 | language = en | date = 2016-06-01 }} </ref>
* 五金廠：包括奉珊、緯航、銘船等公司，主要生產甲板機械與遊艇五金。
* 資訊設備廠：有融程電訊公司，提供船舶所需之資訊設備。
* 艤品廠：包括金鉅、建大、南鐵等公司，主要生產桅桿、欄杆、鋼製船舵與船殼等零件。<ref name = 臺灣船舶產業概況_陳宗慶_2017 />

== 船舶產品 ==

=== 用途 ===
除本題目章節以船隻大小分類外，依據台灣造船公會的船舶產品種類分類為以下五種船舶，其中商船是目前台灣產值比例最大的船舶種類。<ref name = "關於公會_2018"> {{ cite web | title = 關於公會 | url = http://www.tsba.org.tw/list_28.htm | language = zh-tw | accessdate = 2018-08-15 }} </ref>

* 商船：運輸貨物、人員之船舶，比如貨櫃船、油輪、液化氣體船、水泥船、木材船、冷凍船、客輪

* 漁船：漁撈作業用船舶，比如拖網漁船、圍網漁船、釣船

* 遊樂船：從事海上休閒娛樂之船舶，比如遊覽觀光船、遊艇、海釣船、快速賽艇、水下觀光潛艇

* 軍用船：軍事用途之船舶，比如巡防艦、巡邏艦、潛艇、快艇

# 巡防艦：目前台灣對於巡防艦的建造零組艦自給率約30%，其中包括船舶設計約佔6%、船體組裝約佔15%、推進軸（螺旋槳）系統約佔5%，但是其他如通訊設備約佔2%、導航儀器約佔9%、柴油推進主機約佔13%，戰鬥系統40%仍須要向美國、德國、日本、芬蘭等國廠商採購。<ref name = "國艦國造：台灣船舶產業的全面提升_2017"> {{ cite web | title = 國艦國造：台灣船舶產業的全面提升 | url = http://ww2.money-link.com.tw/RealtimeNews/NewsContent.aspx?sn=5625&pu=News_0021&eturec=1 | language = zh-tw | accessdate = 2018-08-16 }} </ref>

* 特種船：包括拖船、駁船、工程船、軍警用艦艇、交通船

=== 建造材質 ===
依據船舶建造材質分類可以分為以下3種類<ref name = 台灣造船工業之演進簡史_2011 /><ref name = "關於公會_2018" /><ref name = KHTan_2018> {{ cite web | title = 小船/遊艇船體材質選擇及其優缺點 | url = https://file.motcmpb.gov.tw/home/FileConversion?filename=files/201801/40a9dd7c-8159-454c-a101-efbfb2557aa0.pdf&nfix=&name=5.%E8%88%B9%E9%AB%94%E6%9D%90%E8%B3%AA%E6%AF%94%E8%BC%83.pdf | author = KHTan | date = 2018-01-15 }} </ref>：

* 鋼殼船：軟鋼製造的鋼殼船特性是強度最強，比重也最大因此船體重量最重；可以在拆解鋼材後將鋼材資源回收。

* 鋁殼船：鋁合金製造的鋁殼船特性是強度次之，比重為軟鋼的34%，船體重量次之約為鋼殼船的45%~50%。

* 玻璃纖維船：玻璃纖維材質（包括[[玻璃纖維強化塑膠|玻璃纖維強化塑膠]]、[[碳纖維|碳纖維]]、[[克維拉|克維拉]]纖維等複合材料）船相對鋼殼船與鋁殼船的強度較差，依材質不同比重為軟鋼的17%~23%，船體重量最輕約為鋼殼船的37%~60%。
