'''CK100型蒸汽機關車'''（日治時為'''400型'''），為[[台灣總督府交通局鐵道部|台灣總督府鐵道部]]及戰後[[臺灣鐵路管理局|臺灣鐵路管理局]]的支線客貨兩用之[[過熱式蒸汽機關車|過熱式蒸汽機關車]]，共有8輛。

== 出現、採購 ==
1917年，[[日本|日本]]自製過熱式蒸汽機關車成功，決定了日後[[台灣|台灣]]火車成為日系火車天下的命運，當時的[[台鐵|台鐵]]，仍大量使用[[汽車製造_(公司)|日本汽車株式會社]]製之50型（[[台鐵CK50型蒸汽機車|CK50型]]）14部，為[[縱貫線|縱貫線]]上牽引力較佳的主力車。總督府鐵道部為提升原有機車性能，向汽車會社購入原50型的改良型，以代替50型。此型車頭在1917年首度購入4部後，又於1919年增購4部，並被編為400型。

== 構造 ==
此型車與[[台鐵CT240型蒸汽機車|200型]]、[[台鐵EK900型蒸汽機車|300型]]同屬早期過熱式機車，本身承襲了原50號型的設計（軸配置與50號型同為1C1配置，中間動輪無輪緣），但已改成華氏[[汽門|汽門]]，機關車本身也已改成過熱式，提高了其蒸汽壓力及機械效能。

日治時期，[[CK101|400]]及401於大正六年7月啟用後，由於本身因為耗煤量少於50號型、操縱輕快，獲得大多好評。

== 特殊意義 ==
此型車是汽車會社自行發展1C1實驗性質的水櫃式蒸汽機車，但本身只是發展時期的過客，在日本國內推銷失敗，[[日本國鐵|日本國鐵]]內沒有量產型編號。當年購買此型車的，除台鐵外，尚有[[常總鐵道|常總鐵道]]（後編為8號、9號，比400型多一砂箱），除此之外，並無其他私鐵採購，因此，在不久後此型車即停產，意外使此型車成了「台灣唯一」的珍寶，造就了1998年CK101復駛時，日人組團來台的「盛況」。

== 運用 ==
=== 日治 ===
啟用之初，400與401原本配置於[[臺北車站|台北]]、[[基隆車站|基隆]]，用於[[宜蘭線|宜蘭線]]前段，八輛全到齊之後轉往[[臺中車站|台中]]、[[嘉義車站|嘉義]]，與50號型同用於幹線行駛。因其1C1軸配置以及水櫃式設計方便前進後退之故，在[[台鐵CT150型蒸汽機車|500型]]等大型煤水車式車頭陸續引進後，逐漸進入支線服務，在1926年後，有五部配置於基隆，三部留在台中。1937年[[台鐵CK120型蒸汽機車|C12型]]抵台，400型（當時已改稱C44型）全移往北部，行駛於[[八堵車站|八堵]]、宜蘭之間以及[[平溪線|平溪線]]，僅一輛留在高雄（可能用於[[高雄臨港線|高雄臨港線]]、[[屏東線|屏東線]]）。

=== 戰後 ===
戰後，由於本身是過熱式中的第一型水櫃式機車，幸運地成的了百位數中的首型機車，在戰後的40年光景中，曾多次出現在平溪線、[[新店線|新店線]]等北部支線，與C12型（這時已改稱CK120型）同為最佳的支線客貨兩用車。然而，隨著時代變遷，由於本身設計較為老舊，故自[[民國63年|民國63年]]7月起，台鐵電化工程動工後，同年12月1日，CK100型正式退役，結束了將近一甲子的運轉生涯。

== 保存車 ==
民國68年，電化工程竣工，蒸汽火車全數退役，如今只有CK101被選為鐵博館展示車而保存下來，之後就一直停放在[[嘉義車站|嘉義車站]]的機務段內。[[民國87年|民國87年]]6月9日鐵路節，CK101正式復駛，成為台鐵史上，第一部動態保存的蒸汽機車。
