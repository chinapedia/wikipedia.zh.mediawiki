'''台湾海峡复航'''，以中华人民共和国籍货轮眉山轮于1979年5月29日在[[台湾海峡|台湾海峡]][[金門島|金门]]至[[马祖岛|马祖]]海域的11个小时的航行为标志<ref name="a1"/>，中华人民共和国方面恢复使用本国民用船只在台湾海峡通行的事件。

1950年起，[[两岸分治|两岸分治]]，[[中华民国政府|中华民国政府]]在[[美国政府|美国政府]]的支持下，奉行[[关闭政策|关闭政策]]，长期军事[[封锁|封锁]]中国大陆东南沿海。中华人民共和国籍船只无法通过[[台湾海峡|台湾海峡]]，中国大陆南北港口间的航运因此中断二十年，直到1968年以绕行[[太平洋|太平洋]]的方式[[打通南北航线|打通南北航线]]。此后，中华人民共和国方面船只仍不能在[[台湾海峡|台湾海峡]]，通行长期绕行太平洋运行南北航线，解放军军用水面舰艇亦如此。

1972年，[[美国总统|美国总统]][[尼克松访华|尼克松访华]]，中美关系解冻。1974年，[[西沙海战|西沙海战]]爆发，[[中国人民解放军海军东海舰队|解放军东海舰队]][[海军舰队首航台湾海峡|南下编队首过台湾海峡]]。1978年12月，[[中美斷交|美国与中华民国断交]]，次年1月，[[中美建交|与中华人民共和国建交]]。在此背景下，[[中华人民共和国交通部|中华人民共和国交通部]]于5月17日，批准眉山轮、富春江轮、红旗121轮等五艘轮船首航台湾海峡。由[[中国人民解放军总参谋部|中国人民解放军总参谋部]]和交通部组建首航领导小组，军地多方给予支持。眉山轮配备了[[军用电台|军用电台]]、[[高射机枪|高射机枪]]、[[重机枪|重机枪]]等武器装备<ref name="a1"/>，并对船员进行了军事训练<ref name="a2"/>。5月27日，眉山轮从广州[[黄埔港|黄埔港]]起航。29日，抵达[[金門島|金门]]以南海域，黎明后眉山轮开始穿越台湾海峡中段，于17时30分越过马祖岛海域。此段航程共计168[[海里|海里]]，用时11小时。期间，[[中華民國國軍|国军]]派出一架飞机<ref name="a1"/>、两条[[军舰|军舰]]<ref name="a2"/>，但未对眉山轮做出军事反应。31日，眉山轮抵达日本[[名古屋|名古屋]]，后顺利返回黄埔港<ref name="a1"/>。

6月22日，中华人民共和国交通部召开“恢复台湾海峡正常通航会议”，并颁布《[[s:交通部关于我商船通行台湾海峡的暂行规定|交通部关于我商船通行台湾海峡的暂行规定]]》<ref name="a1">{{Cite web |url=http://www.coscogz.com.cn/gzyyxcw/ztzx_show.asp?Tid=1030&bid=24&mid=85&sid=0 |title=新中国三十年第一程——记眉山轮首航台湾海峡|accessdate=2017-03-12 |author= |date=2011-03-28 |work= |publisher=中远航运宣传网|language=Zh-hans }}</ref>，南北航行全部取道台湾海峡。9月12日，[[中华民国政府|中华民国政府]]公告中止《[[s:戡亂時期截斷匪區海上交通辦法|戡乱时期截断匪区海上交通办法]]》，被视为中华民国方面在法理上终止[[关闭政策|关闭政策]]<ref name="a6">{{cite thesis|chapterurl=http://nccur.lib.nccu.edu.tw/bitstream/140.119/49868/8/15300108.pdf |chapter=结论|title=《封鎖大陸沿海──中華民國政府的「關閉政策」，1949-1960》 |format=PDF |author=林宏一| date=2009-06-00 |degree=碩士|accessdate=2019-07-09|language=Zh-hant}}</ref>。

1983年11月30日，[[客轮|客轮]]紫罗兰号、长柳号分别从黄埔港、[[上海港|上海港]]相向开出，完成客轮通过台湾海峡的首航<ref name="a2">{{Cite web |url=http://www.dangan.ningbo.gov.cn/dadt/yndt/201008/t20100812_686461.html |title=亲历者口述档案再现台湾海峡复航传奇|accessdate=2017-03-12 |author=钟鸣、吴晓琼 |date=2010-08-12 |work= |publisher=宁波档案网，来源:《中国档案报》2010年8月12日总第2040期第三版|language=Zh-hans }}</ref>。此后，中华人民共和国船只以在[[臺灣海峽中線|臺灣海峽中線]]西侧航行的方式通过台湾海峡。
