'''台灣穿山甲'''（[[学名|学名]]：{{lang|la|''Manis pentadactyla pentadactyla''}}）又稱'''台灣鯪鯉'''、悶仔、土龜，是一種身上覆有[[鳞|鱗片]]的[[哺乳類|哺乳動物]]。該物種為[[穿山甲属|穿山甲属]][[中華穿山甲|中華穿山甲]]的一個[[亞種|亞種]]，也是[[臺灣|台灣]]的一個[[特有種|特有亞種]]。目前，因[[棲地破碎化|棲地流失]]及[[人類|人類]]過度捕食等因素，該物種已瀕臨[[絕滅|絕滅]]。<ref name=":0">{{Cite news|url=http://teia.tw/zh-hant/natural-valley/species/9616|title=穿山甲 - 台灣環境資訊協會|last=teia|date=2016-02-04|newspaper=台灣環境資訊協會|accessdate=2017-03-21|language=zh-TW}}</ref><ref name=":1">{{Cite web|url=http://np.cpami.gov.tw/chinese/index.php?option=com_content&view=article&id=3984&Itemid=34 |title=臺灣國家公園 |accessdate=2017-03-21 |author= |date= |work=臺灣國家公園 |publisher= |language=zh-tw |deadurl=bot: unknown |archiveurl=https://web.archive.org/web/20170205174743/http://np.cpami.gov.tw/chinese/index.php?option=com_content&view=article&id=3984&Itemid=34 |archivedate=2017-02-05 }}</ref><ref name=":2">{{Cite web|url=http://www.merit-times.com.tw/newspage.aspx?unid=429915|title=人間福報 : 台灣吉祥動物 穿山甲|accessdate=2017-03-21}}</ref><ref>{{Cite news|url=http://guide.cpshs.hcc.edu.tw/bin/downloadfile.php?file=WVhSMFlXTm9MelExTDNCMFlWODVNMTgxTkRjNU1EQTJYekU1T0RFMUxuQmtaZz09&fname=NktPMDVhNjI2YWlQNXBXWjVvNkk2S3liNTZpLzVMcU1MbkJrWmc9PQ%3D%3D |title=圖 2、苗栗通霄地區五隻共域的石虎個體由無線電追蹤所產生的活動範圍（95% MCP； 線框）和核心 |date=2017-03-21 |newspaper=archive.is |accessdate=2017-03-21 |deadurl=bot: unknown |archiveurl=https://archive.is/20170321200736/http://guide.cpshs.hcc.edu.tw/bin/downloadfile.php?file=WVhSMFlXTm9MelExTDNCMFlWODVNMTgxTkRjNU1EQTJYekU1T0RFMUxuQmtaZz09&fname=NktPMDVhNjI2YWlQNXBXWjVvNkk2S3liNTZpLzVMcU1MbkJrWmc9PQ== |archivedate=2017-03-21 }}</ref>

== 習性 ==
台灣穿山甲的前爪很大因此很有力，非常適合爬樹，但後腳因爪較小而不適於抓住[[樹幹|樹幹]]，所以牠們會用尾巴代替後腳，用尾部的鱗片插入樹幹，因此不易滑落。身體以螺旋的方式向上攀爬，在樹枝則可倒掛，向下則以頭上腳下的方式倒退爬行。台灣穿山甲以[[舉尾蟻|舉尾蟻]]和[[白蟻|白蟻]]為食，台灣穿山甲會用前爪挖開蟻巢，使螞蟻逃出，再用長[[舌|舌頭]]舔食。<ref>{{Cite news|url=http://www.tanews.org.tw/info/7143|title=先打贏食物 好鼻師穿山甲享用大餐|newspaper=台灣動物新聞網|accessdate=2017-03-21|language=zh-hant}}</ref>台灣穿山甲的舌頭具有有黏性的[[唾液|唾液]]，可以沾黏食物後再送進口中。當台灣穿山甲遇到危險時，會縮成球狀，讓[[掠食者|掠食者]]沒辦法下手。<ref name=":0" /><ref name=":1" /><ref name=":2" />

== 特徵 ==
全長75~95公分，體重約3~6.5公斤。全身布滿鱗甲。<ref name=":0" /><ref name=":1" />

== 分布 ==
台灣中高海拔山區。<ref name=":1" />

== 巢穴 ==

=== 覓食洞穴 ===
在山坡上斜著挖地洞，洞口深度較淺，因此較容易見到洞底，洞口常堆著挖出來的廢土。覓食洞穴常在枯樹附近，可做為臨時休息或避敵之處。

=== 居住洞穴 ===
洞口多在陡直山坡，周圍遮蔽少、排水佳，洞口深度較淺深，是長久居住的地洞。<ref name=":1" />

== 主要死亡原因 ==
* 被[[人類|人類]]捕殺
* [[路殺|路殺]]
* 遭[[流浪犬|野狗]]咬噬

== 人工飼養 ==
食物不能吃得太細緻，易有消化潰瘍出血症狀。台灣穿山甲因對[[生態系統|自然生態系統]]依賴極大而難以適應人類[[圈養|圈養]]環境。<ref>{{Cite news|url=http://www.chinatimes.com/newspapers/20170319000681-260309|title=習性特殊 穿山甲人工養殖難度大|last=中時電子報|newspaper=中時電子報|accessdate=2017-03-21|language=zh-TW}}</ref>
