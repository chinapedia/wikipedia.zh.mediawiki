'''台灣賽馬'''舉辦於[[台灣日治時期|日治時代]]，當時稱為「競馬」。[[昭和|昭和]]3年（1928年）起，每年依序在[[臺北市|台北]]、[[新竹市|新竹]]、[[臺中市|台中]]、[[嘉義市|嘉義]]、[[臺南市|台南]]、[[高雄市|高雄]]、[[屏東市|屏東]]舉辦，每地2─3場，多為春季、秋季各1場，每場2─6天不等。

== 歷史 ==
=== 黎明期 ===
[[大正|大正]]4年（1915年）11月，在台北江瀕街埋立地舉辦為期1週的賽馬會，此為台灣賽馬之始。而正式的賽馬則從[[昭和|昭和]]3年（1928年）開始，台北武德會馬術部在台北圓山運動場（[[中山足球場|中山足球場]]）舉行賽馬，此為台灣馬券賽馬之嚆矢。昭和4年（1929年）至昭和6年（1931年），各地的團體也開始自行主辦賽馬。此時期的中獎馬券，需在指定的商店換取獎品而非現金。

=== 台灣競馬協會時代 ===
昭和6年（1931年），台北馬事協會、台中產馬會、嘉義產馬會、台南愛馬會、高雄產馬會、屏東愛馬會，共同組織「台灣競馬協會」，制定「台灣競馬協會規約」、「競馬施行規程」，確立共通的規範。規約的要點是春・秋兩季賽馬、競賽內容、馬體檢查、裁判、獎金等細目。中獎的馬券，需在指定的商店換取獎品而非現金，額度最高以入場券10倍價值為限。昭和7年（1932年），台北春季賽馬除原來的單勝式馬券，再加上複勝式馬券的販賣。此時期的賽馬，各競馬場在春秋兩季各辦4日（共2週、各六日舉辦），春季在2月～5月、秋季在9月～12月舉行。

=== 台灣競馬令時代 ===
昭和13年（1938年）5月，[[台灣總督府|台灣總督府]]公佈「台灣競馬令」，使台灣賽馬有明確的法規可遵循，也更改部分競馬規則的規定。「台灣競馬令」施行後，春季在3月～7月、秋季在9月～翌年2月，也就是除了8月以外，全台灣毎個月都有賽馬。規定競馬場需有周長1600公尺以上、寬30公尺以上的馬場。此時期的中獎馬券，是換取現金而非獎品，上限仍維持10倍。也維持未成年・賽馬相關者禁止購買馬券等規定。

=== 1944年 ===
昭和16年（1941年），[[太平洋戰爭|太平洋戰爭]]爆發。昭和18年（1943年），為了獎勵馬産，調降馬券稅<ref name="furuya1943">[[#競馬の必要性と機構に付いて|競馬の必要性と機構に付いて]]</ref>。根據《[[台灣日日新報|台灣日日新報]]》、《台灣日報》（原名《台南新報》），台灣各地的競馬場在昭和18年的春・秋兩季仍舉辦了各7日間的賽馬。台灣的賽馬舉辦到昭和19年（1944年）2月為止。

=== 可能再起 ===
* [[韓國瑜|韓國瑜]]當選高雄市長後，提起高雄發展賽馬產業的構想。但是因為會涉及牴觸「對動物不能有直接或間接賭博的搏鬥或競技行為」、「賭博是違法行為」等問題需要，而且不管怎樣過程{{#tag:ref|韓存在幾種版本：1.杜拜的賽馬就沒有賭博性質、2.加入運彩減少爭議|group="†"}}{{#tag:ref|目前尚無場地可尋。|group="†"}}，都必須透過國家特許法令核准。

==競馬場==
[[File:Taichu_keiba_ad_on_1939_magazine.jpg|thumb]]》雜誌上的台中競馬廣告。]]

起初競馬場大多設在公園空地、運動場或陸軍練兵場，後來皆改在新設立的常設競馬場舉行。台北曾先後於圓山運動場（[[中山足球場|中山足球場]]）、練兵場（[[青年公園|青年公園]]）、川端競馬場（同安街附近的河堤公園）、北投競馬場（[[政治作戰學校|政治作戰學校]]）舉行賽馬。新竹的常設競馬場初設在赤土崎，後遷至香山。其他各地的常設競馬場設於大肚山（台中）、堀川町（嘉義）、桶盤淺（台南）、前金（高雄）、六塊厝（屏東）。昭和13年（1938年）「台灣競馬令」施行後，規定台灣的競馬場需擁有一周長1600公尺以上的馬場。

===台北競馬場===
昭和3年（1928年），在台北市内的圓山運動場建設周長400公尺、寬13公尺的馬場，舉辦台灣最初的正式賽馬。昭和5年（1930年）春季，在圓山運動場以周長550公尺舉行。昭和5年（1930年）秋季開始，借用陸軍練兵場（現・青年公園）建設周長800公尺，之後擴大為1000公尺的馬場舉行賽馬。昭和9年（1934年），在[[新店溪|新店溪]]北側建設周長1000公尺的常設競馬場舉行賽馬，此即川端競馬場（位在新店溪畔，橫跨[[水道町|水道町]]和[[川端町_(台北市)|川端町]]，所以也稱作水道町競馬場或水道町川端競馬場）<ref name="sai2003">[[#載2003|載2003]]</ref><ref name ="gaichi 131">外地及満洲国馬事調査書、131頁。</ref><ref> 加藤寿子 『旧台北イラストマップ』1993年</ref>。昭和14年（1939年），決定在[[北投區|北投]]建設新競馬場。昭和15年（1940年）開始，在北投競馬場（現・國防大學政治作戰學院）舉行賽馬。昭和15年（1940年）10月26日落成的北投競馬場擁有周長1800公尺的馬場，雖為木造設施，但是，具備馬見所（看台）、投票所、厩舎等設施都設有遮雨棚，可避免日曬雨淋享受競馬之樂。[[臺鐵淡水線|台鐵]]在競馬期間也會加開臨時列車<ref>『台灣畜産會會報』3巻11号、1940年11月、57-58頁。</ref>。

賽馬的主辦者在昭和3年（1928年）是台北武德會馬術部、1930年（昭和5年）起是台北馬事協會<ref name="masani">[[#本島競馬の沿革竝に之が現況|本島競馬の沿革竝に之が現況]]</ref>、昭和12年（1937年）起是台北州畜産組合連合<ref name ="taiwannochikusann 6-1-66">台灣畜産協會『台灣之畜産』6巻1号、1938年、66頁</ref>、1941年的北投競馬是由台北州畜産會主辦<ref>『台灣日日新報』1941年6月21日</ref>。

===新竹競馬場===
[[File:Sinchiku_1945.jpg|thumb]]

昭和5年（1930年）秋季和昭和6年（1931年）春季，在[[新竹市|新竹]]市内的公園内運動場建設周長450公尺的馬場，舉行賽馬。昭和10年（1935年），在[[十八尖山|十八尖山]]麓的赤土崎建設周長1000公尺、寬25公尺的常設競馬場<ref name="masani"/>（現在的國立新竹高商校地<ref name="bunka">[http://www.hcccb.gov.tw/chinese/16museum/mus_b02a.asp?station=108&bull_id=1727&bull_top=1724&cate_id=3 新竹市文化局]</ref>）。之後，因赤土崎競馬場場地狹小而決定移轉，在新竹郊外的香山建設當時台灣最大的周長1600公尺的競馬場（現在的香山工業區）{{#tag:ref|日後被1800公尺的台北北投競馬場和台南新競馬場超越，但是，香山競馬場開設時是台灣最大的競馬場。|group="†"}}，之後在新設的香山競馬場舉行新竹的賽馬<ref name="bunka"/><ref name ="ntl">[http://www.ntl.edu.tw/public/Attachment/39181049133.pdf 台灣国立圖書館『庶民生活史-消失的體育活動-賽馬與相撲』]</ref>。

賽馬的主辦者在昭和5-6年是新竹馬事協會、昭和10年是新竹産馬期成會<ref name="masani"/>、昭和11年起是新竹州畜産組合連合會<ref name="sai2003"/>。

===台中競馬場===
昭和4年（1929年）秋季，在[[台中市|台中]]市内的練兵場建設周長550公尺、寬15公尺的馬場，舉行賽馬。昭和7年（1932年），練兵場的馬場擴大為周長800公尺<ref name="masani"/>，地點是台中市南區萬安里正義街付近，之後作為軍司令部，現在是台中長春公園。台中長期借用練兵場舉行賽馬，是台灣最晚建設常設競馬場之地<ref name="masani"/>，最後終於在[[大肚山|大肚山]]麓（現在的[[成功嶺|成功嶺]]基地）建設台中競馬場<ref name ="ntl"/>。新設的台中競馬場周長1600公尺<ref name="furuya1943"/>。

===嘉義競馬場===
昭和4年（1929年）春季，在[[嘉義市|嘉義]]的公園内的空地建設周長410公尺的馬場，舉行賽馬<ref name="masani"/>。昭和5年（1930年）秋季，在堀川町（現在的嘉義市東區新開里及其周邊）的常設競馬場落成，在此舉行賽馬。堀川町競馬場最初周長800公尺、昭和9年（1934年）擴大為周長1000公尺<ref name="masani"/>。

昭和13年（1938年），因堀川町競馬場場地狹小而決定移轉。當初預定移轉至下路頭[[八掌溪|八掌溪]]畔白鷺橋西側（白鷺橋在現在的軍輝橋西側，現在的嘉義市東區安業里八掌溪畔）<ref>『台灣日日新報』1938年7月21日</ref>，但是，最後在嘉義北方的後湖，現在[[嘉北車站|嘉北車站]]的東北方付近新設競馬場（現在的後湖工業區）<ref name ="taiwanairpower">[http://taiwanairpower.org/blog/?p=9689 taiwanairpower.org]</ref>。新設的嘉義競馬場周長1600公尺<ref name="furuya1943"/>。

===台南競馬場===
[[File:Map_of_Tainan_racecourse_1945.jpg|thumb]]
昭和4年（1929年），在[[台南市|台南]]運河旁的空地建設周長400公尺的馬場，開辦賽馬。空地的賽馬在春・秋兩季舉辦，昭和6年（1931年）秋季起，借用練兵場建設周長800公尺的馬場，舉行賽馬。昭和7年（1932年），在桶盤棧建設1周1000公尺的常設競馬場<ref name="masani"/>。桶盤棧在日治時代稱作[[汐見町|汐見町]]、現在稱作[[水交社_(臺南市)|水交社]]。

之後，競馬場移轉至台南市後甲（現在的東光國小及其周邊、台南紡織）<ref name ="taiwanairpower"/>。擁有獨特的三角形馬場的<ref name ="taiwanairpower"/>台南新競馬場周長1800公尺，和台北北投競馬場同樣，擁有台灣最大周長<ref name="furuya1943"/>。

===高雄競馬場===
昭和6年（1931年）秋季，在[[高雄市|高雄]][[苓雅寮|苓雅寮]]的海軍航空基地内借地，建設周長800公尺的馬場，是高雄賽馬之嚆矢。苓雅寮海軍基地的賽馬僅辦1年即終了，昭和7年（1932年）秋季起，在[[入船町_(高雄市)|入船町]]的空地建設1周600公尺、之後擴大為800公尺的馬場，舉行賽馬。昭和10年（1935年），在[[前金區|前金]]的大港埔開設周長800公尺、寬20公尺的常設競馬場<ref name="masani"/>。之後又在[[灣子內_(三民區)|灣子內]]建設競馬場<ref name ="taiwanairpower"/><ref>[https://web.archive.org/web/20131225015423/http://www.wretch.cc/blog/skylong/27723342 高雄文史：找尋灣仔內競馬場]</ref>。根據美國陸軍1945年的地圖，可以確認灣子内有楕圓形的競馬場<ref>[http://www.sunriver.com.tw/images/maps/map_012xb.jpg 1945年高雄地圖]</ref>。新設的高雄競馬場周長1600公尺<ref name="furuya1943"/>。

===屏東競馬場===
昭和4年（1929年）秋季，在[[屏東市|屏東]]飛行場借地，建設周長600公尺的馬場，是屏東賽馬之嚆矢。昭和5年（1930年）春季，在屏東公園運動場建設周長400公尺的馬場，舉行賽馬。公園運動場的馬場屬於暫時性設施，賽馬僅辦上述兩回即終了，昭和5年（1930年）秋季，在六塊厝開設常設競馬場。六塊厝競馬場周長800公尺，之後漸次擴大，昭和9年（1934年）擴大為1000公尺、昭和10年（1935年）秋季已有1600公尺的馬場<ref name="masani"/>。六塊厝競馬場位在屏東飛行場和台鐵的鐵道線之間<ref name ="taiwanairpower"/>。
