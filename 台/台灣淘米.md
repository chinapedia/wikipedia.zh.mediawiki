'''台灣淘米科技公司'''，原名'''異想數位娛樂股份有限公司'''，於2010年1月更名改為「'''台灣淘米'''」，2012年10月成立[[917play娛樂平台|917play娛樂平台]]。目前代理美國{{link-en|王島娛樂|KingsIsle Entertainment}}，以及中國大陆[[上海淘米|上海淘米網路]]、[[第七大道科技有限公司|第七大道科技]]、上海星果網路、福州天盟数码、武汉聚网科技等公司的部分遊戲。

台灣淘米的前身為'''異想數位娛樂'''，初期代理[[爭霸Online|爭霸Online]]，於2008年12月代理[[摩爾莊園|摩爾莊園]]，並與網站摸摸耳小遊戲合作，將遊戲發佈於該站。之後便有許多網站採取此模式配合摩爾莊園推廣知名度，使得摩爾莊園短時間內在台灣迅速竄紅。隨著異想數位科技轉型成台灣淘米之後，原代理之爭霸Online停止營運，並關閉異級棒網站的內容。

公司更名為台灣淘米後，開始代理[[賽爾號|賽爾號]]、[[小花仙|小花仙]]等遊戲，並代理其他遊戲公司的遊戲（如[[彈彈堂|彈彈堂]]、[[庫庫馬力|庫庫馬力]]等），並積極與眾多廠商合辦活動或贊助，以及周邊商品的積極推出和活動的大量舉辦，讓台灣淘米獲得大量營收。

後來，台灣淘米成立[[917play娛樂平台|917play娛樂平台]]，遊戲年齡層主要在青少年與成人，與台灣淘米平台主打的兒童為主要年齡層不同，目前已經代理多款遊戲。

2016年1月，成立[[就醬玩H5Play|就醬玩H5Play]]遊戲平台，經營以[[HTML5|HTML5]]運行的遊戲。<ref>{{Cite web |url = http://h5play.com.tw/about |title = 關於我們 |author =  |date =  |publisher = 台灣淘米 |language = zh-tw |accessdate = 2016-02-06 |quote = 【就醬玩H5Play】遊戲平台是台灣淘米科技股份有限公司成立的HTML5遊戲平台，台灣淘米科技股份有限公司為一間社群網路遊戲平台營運公司，同時也是身兼品牌授權及廣告媒體，致力打造華人地區最大線上樂園。 }}</ref>

== 歷史 ==
2007年12月27日，'''異想數位娛樂股份有限公司'''在[[經濟部商業司|經濟部商業司]]辦理登記成立，為台灣淘米的前身。隔年5月取得《[[爭霸Online|爭霸Online]]》台灣代理權<ref>{{zh-tw}}{{cite web|url=http://gnn.gamer.com.tw/3/30523.html|title=以《山海經》為藍本 異想數位宣布代理《爭霸 Online》預計暑期封測|accessdate=19 May 2008|website=巴哈姆特電玩資訊站}}</ref>，並於12月取得《[[摩爾莊園|摩爾莊園]]》[[台港澳|台港澳]]代理權。

2009年11月13日，'''台灣淘米科技股份有限公司'''由開曼群島 ELYN CORPORATION 轉投資核准，並登記成立，與異想數位的董事長為同一人<ref name=":0">{{Cite web|url=http://www.61.com.tw/investors/About_1.php|title=關於淘米|accessdate=2018-12-21|work=www.61.com.tw}}</ref>。隔年1月8日，宣布取得《[[賽爾號|賽爾號]]》台港澳代理權，並舉行公測<ref>{{zh-tw}}{{cite web|url=http://gnn.gamer.com.tw/9/41629.html|title=科學與遊戲並重 台灣淘米取得寓教於樂《賽爾號》台港澳營運權|accessdate=8 Feb 2010|website=巴哈姆特電玩資訊站}}</ref>。1月26日，異想數位將旗下遊戲《摩爾莊園》代理權移交給台灣淘米、另一款遊戲《爭霸Online》則停止營運；2月16日，異想數位辦理結束登記。並於2011年5月3日解散並清算完結。

===台灣淘米論壇===

2010年12月16日，公告將由原台灣淘米代理的摩爾莊園、賽爾號、彈彈堂附設論壇合併為'''台灣淘米論壇'''，簡稱'''淘論'''，12月21日完成合併。

台灣淘米論壇是作為台灣淘米旗下所代理之遊戲而附設的討論區，以遊戲作為版塊區分，另設有類似於[[巴哈姆特電玩資訊站|巴哈姆特]]場外休憩區的討論版。

==爭議事件==
===台版摩爾莊園名次內定爭議===
因摩爾莊園在台灣是代理的，從異想時期到台灣淘米時期，因每次的拉姆運動會、火神杯、旋風拉力賽等莊園競賽名次都與大陸版本相同，「名次」的問題備受玩家質疑<ref>{{zh-tw}}[http://bbs.61.com.tw/viewthread.php?tid=75744 【已回覆】比賽內定問題(請官方進來)]</ref>。但直到2010年火神杯後，第一名與大陸版本不同，謠言不攻自破。

===公司更名爭議===
公司更名時，於摩爾莊園官方網站跳出提示窗<ref>{{zh-tw}}[http://www.51mole.com.tw/sta.html 共同聲明稿]</ref>，稱因「異想數位娛樂中止代理」而由台灣淘米承接[[代理|代理權]]。摩爾論壇（今台灣淘米論壇）上就因此出現抗議聲浪，玩家希望異想數位娛樂能夠繼續代理摩爾莊園遊戲，但在淘米的公司簡介<ref>{{zh-tw}}[http://www.61.com.tw/taomee_tw/index.html 台灣淘米]</ref>中，提到了「台灣淘米科技股份有限公司，前異想數位娛樂，……」，證明了異想是台灣淘米的前身。

===搶乳牛事件===
官方在2010/04/14星期三早上於摩爾論壇（今台灣淘米論壇）發布4/16改版訊息<ref>{{zh-tw}}[http://bbs.61.com.tw/viewthread.php?tid=67976 4/16【改版預告】大伯出書啦？大伯的新書發表會！]（原爭議段已移除）</ref>中，有「搶一萬頭[[乳牛|乳牛]]」的活動。但是在下午卻無預警地將該段撤下，引起大批玩家不滿，[[批評|批評]]一個接著一個的來，官方於隔天（2010/04/15）早上發表一篇解釋文章<ref>{{zh-tw}}[http://bbs.61.com.tw/viewthread.php?tid=68052 【官方公告】乳牛事件說明(新增最新消息)]</ref>中，並向玩家道歉，才逐漸平息這場風波。事後恢復「搶牛」活動，但數量從原來公布的一萬隻變成三千隻。

===2012年8月遊戲異常事件===
2012年8月16日下午，台灣淘米的旗下遊戲（包括論壇、米米校巴、米米福利社）突然都出現異常情況，不能登入。玩家瘋狂進入[[Facebook|Facebook]]的遊戲粉絲團抱怨，官方表示將會在8/17改版時修復，因此屆時無法進行遊戲，官方也在遊戲中發布補償。


===2014年11月29日駭客入侵官方人員帳號事件===
2014年11月29日，[[摩爾莊園|摩爾莊園]]玩家友達（本名[[黃宥翔|黃宥翔]]，目前藝名黃豪），將遊戲摩爾莊園官方人員麼麼公主及絲爾特的帳號進行破解登入，在遊戲內大肆招搖，而遊戲管理員也立刻查出他的IP與米米號，將他的所有遊戲帳號永久凍結，並且提出和解以及在[[淘米論壇|淘米論壇]]上發佈公開致歉文<ref>{{zh-tw}}[http://bbs.61.com.tw/viewthread.php?tid=428143 黃豪在淘米論壇的公開致歉文]</ref>，最後以和解結束這場事件，也因此事件他在遊戲以及社團內爆紅、討論。
