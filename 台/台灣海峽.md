'''臺灣海峽'''（簡稱'''臺海'''；歐洲早期稱'''福爾摩沙海峽'''）指的是介於[[福建|中國大陸]]與[[台湾|台湾]]之間的海域和[[海峽|海峽]]<REF name="NicholsWilliams2009"/><REF name="Ichiye2000"/>。

由於臺灣海峽位於[[东海|东海]]及[[南海|南海]]之間，為東洋海洋史上貿易路線上的重要水域<REF name="Schottenhammer2010"/><REF name="Tsai2008"/>；位於[[亞洲|亞洲]]及[[太平洋|太平洋]]之間，成為近代地緣政治戰略要點<REF name="StaffUsa1999"/><REF name="Li2011"/>。

地理學上為[[东海|东海]]<ref>{{Cite web |url=http://www.mac.gov.tw/public/Attachment/052811542991.pdf |title= 台灣海峽的法律地位暨美中間潛存的海事爭議 }}</ref>及[[西太平洋|西太平洋]]的一部分，北以[[台湾|台湾]][[新北市|新北市]][[三貂角|三貂角]]經[[釣魚臺列嶼|釣魚臺列嶼]]之[[黃尾嶼|黃尾嶼]]與[[福建省|福建省]][[平潭島|平潭島]]連線；南則是以[[屏東縣|屏東縣]][[鵝鑾鼻|鵝鑾鼻]]與[[福建省|福建省]][[东山岛|东山岛]]連線為界<ref>{{Cite web |url=http://nrch.culture.tw/twpedia.aspx?id=1459 |title= 台灣海峽 }}</ref>，南北長約300公里，平均寬度180公里。

主要以[[大陸棚|大陸棚]]為主，其水深（岩床最大深度）為70米<ref name="Ichiye2000" />。海域上的島嶼，除靠近福建的沿海[[大陸島|大陸島]]外，尚有[[澎湖群島|澎湖群島]]與[[屏東縣|屏東縣]]的[[小琉球|小琉球]]。

==形成==
[[更新世|更新世]]后期[[冰河期|冰河期]]開始，海水退去，海峡深度下降了130～180米，变成陆地，並與大陆相连接。大量[[古人类|古人类]]、古动物从大陆经过成为陆地的台湾海峡迁徙至台湾。
早在6亿年以前古生代晚期造山运动开始时，在台湾海峡地区出现“台湾滩”，这是海峡中的最淺处，深度仅20米，但两岸间许多地区依然连为一体。

20世纪70年代，在台湾[[北港|北港]]地区的石油勘探中，发现[[菊石|菊石]]化石，这是[[中生代|中生代]]的[[标准化石|标准化石]]，由此证明，在[[中生代|中生代]][[三叠纪|三叠纪]]和[[侏罗纪|侏罗纪]]期间，两岸间虽然许多地方已经出现海水，但主要还是陆地。台湾海峡位置上的地壳运动没有停止。到1.92亿年前的中生代侏罗纪和[[白垩纪|白垩纪]]之间，两岸间发生剧烈的地壳运动，此时台湾开始成为陆地，地质史上台湾称之为“[[南澳运动|南澳运动]]”（即由[[大南澳片岩|大南澳片岩]]演变为[[大理石|大理石]]），大陆称之为“[[燕山运动|燕山运动]]”。

海峡最初的形成。两岸分离开始于[[白垩纪|白垩纪]]和[[古新世|古新世]]时期，也就是在约5400万年以前，台湾中部地区开始被水淹，中间成为浅海，开始成为海峡。在始新世早期的地壳运动中，台湾开始“[[太平运动|太平运动]]”，大陆开始“[[茅山运动|茅山运动]]”，两岸陆地连成一片。直到[[第四纪|第四纪]][[更新世|更新世]]时期，华南地区的[[花岗岩|花岗岩]]被沙化，岩石中的[[石英|石英]]和[[云母|云母]]被冲积到台湾海峡中的低部地区，今天台湾北部和中部地区大量开采，成为[[新竹|新竹]]玻璃工业的重要原料。

自[[第四紀冰河時期|第四紀冰河時期]]（1800万年至6000年之间）以来，大海进入海进期，海水面上升约有100至130米，形成了今天的海平面，台湾海峡开始形成。自此开始，两岸间的来往由陆地转为海上。

===亞洲及太平洋===
臺灣海峽位於[[亞洲|亞洲]]及[[太平洋|太平洋]]之間，並連結[[南海|南中國海]]及[[東海|東中國海]]<ref name="NicholsWilliams2009" /><ref name="Ichiye2000" />。

因為臺灣及其海峽位於亞洲及太平洋之間，在[[第二次世界大戰|第二次世界大戰]]戰後被視為华盛顿監視中國東南沿海活動的窗口或[[中华人民共和国|中华人民共和国]]打破[[第一島链|第一島链]]進入太平洋的據點<REF name="StaffUsa1999"/><REF name="Li2011"/>（相關位置參見不同視角的兩圖如右）。[[两岸分治|两岸分治]]后，[[中华民国政府|中华民国政府]]凭借军事优势，推行[[关闭政策|关闭政策]]，得以封锁台湾海峡三十年。随着[[中华人民共和国军事|中华人民共和国军事]]力量的增长和政治局势的变化。中国人民解放军军用水面舰艇在[[福州军区|福州军区]]岸防火力策应下，于1974年1月第一次穿越台湾海峡，驰援[[西沙海战|西沙海战战场]]。民用货船于1979年在[[台湾海峡复航|台湾海峡复航]]。此后，台湾海峡中形成一条无形界线——[[臺灣海峽中線|海峽中線]]。

[[第七艦隊_(美國海軍)|美國第七艦隊]]進入臺灣海峽的歷史說明臺灣海峽的重要地緣政治意義<ref name="徐宗懋2007">{{cite book|author=徐宗懋|title=20世紀台灣:精選版,光復篇|url=http://books.google.com/books?id=euiiIj1Ei98C&pg=PA151|accessdate=5 March 2013|year=2007|publisher=台灣書房出版有限公司|isbn=978-986-7332-82-0|pages=151–}}</ref><ref name="張延廷2007">{{cite book|author=張延廷|title=國防通識教育|url=http://books.google.com/books?id=-GtQcoqYi-4C&pg=PA294|accessdate=5 March 2013|year=2007|publisher=五南圖書出版股份有限公司|isbn=978-957-11-4492-4|pages=294–}}</ref>。1996年，中華民國舉行[[1996年中華民國總統選舉|首次總統民選]]，{{Verify source|[[中華人民共和國|中華人民共和國]]方面坚持“一个中国”原则，以「臺灣為中國領土」為依据對臺灣本島[[基隆|基隆]]、[[高雄|高雄]]的外海發射短程彈道飛彈}}{{Request quotation}}，中華民國實際控制的區域均感受戰爭威脅。<ref name="Chang2001">{{cite book|author=Jaw-ling Joanne Chang|title=後冷戰時期美國海外出兵案例研究|url=http://books.google.com/books?id=O66CAAAAIAAJ|accessdate=5 March 2013|year=2001|publisher=中央硏究院歐美硏究所|isbn=978-957-671-815-1}}</ref>，史称[[臺湾海峡导弹危机|臺湾海峡导弹危机]]。往後，中華民國[[中華民國總統及副總統在臺灣之公民直接選舉與罷免|每屆總統大選]]前後，美國為防止中国大陸方面武力干涉中華民國選舉，會在每屆中華民國總統選舉到就職典禮那一段期間，常態性的巡弋臺灣海峽<ref name="劉慶祥2010">{{cite book|author=劉慶祥|title=兩岸和平發展與互信機制之研析|url=http://books.google.com/books?id=ucm9K1xN_rAC&pg=PA55|accessdate=5 March 2013|year=2010|publisher=秀威資訊科技股份有限公司|isbn=978-986-221-467-1|pages=55–}}</ref>。

===東海及南海===
因為臺灣海峽位於東中國海及南中國海之間<REF name="国民经济计划与管理"/>，在[[海洋史|海洋史]]（特別是東亞貿易）<REF name="Schottenhammer2010"/><REF name="Tsai2008"/>、國際關係、地緣政治研究中，和[[霍尔木兹海峡|霍尔木兹海峡]]及[[麻六甲海峽|麻六甲海峽]]等有相關比较研究，以探讨海峡区域的自然、人文发展<REF name="AutoMP-1"/><REF name="杨晓辉"/>。

臺灣海峽在東海及南海的海域交通的位置，也成為明清時期的[[武装海商集团|武装海商集团]]，如[[李旦|李旦]]、[[郑芝龙|郑芝龙]]、[[林道乾|林道乾]]、[[林鳳|林鳳]]等活動的區域之一<REF name="黃淑鈴高永謀2006"/><REF name="史明1980"/><REF name="汤锦台2001"/><REF name="吳法1976"/><REF name="林仁川黄福才1997"/><REF name="庄国土2001"/><REF name="Lee李恩涵2003"/>，其中台湾海峡更成為[[郑芝龙|郑芝龙]]及其後繼的鄭氏舰队的内湖<ref name="Tsai2008"/><REF name="廈门市郑成功纪念馆2003"/><REF name="海峡两岸关系史:第1卷.开发・融合"/><REF name="法国汉学"/><REF name="台湾研究"/><REF name="历史学"/>（相關位置參見不同視角的兩圖如右）。[[郑芝龙|郑芝龙]]以[[華南|華南]]、[[臺灣|臺灣]]及[[日本|日本]]等地為活躍舞臺，經營日本[[平戶島|平戶島]]、[[長崎|長崎]]、[[馬尼拉|馬尼拉]]、和[[華南|華南]]及[[東南亞|東南亞]]華人移民據點形成主要的商業航道為其主要經濟命脈，台灣海峽亦為[[武裝海商集團|武裝海商集團]]活動及征戰的主要海域。<ref name="Leonard1984">{{cite book|author=Jane Kate Leonard|title=Wei Yuan and China's Rediscovery of the Maritime World|url=http://books.google.com/books?id=wQCm-bequYgC&pg=PA66|accessdate=4 March 2013|year=1984|publisher=Harvard Univ Asia Center|isbn=978-0-674-94855-6|pages=66–}}</ref><ref name="湯錦台2011">{{Cite book| edition = 初版| publisher = 如果出版:大雁出版基地發行| isbn = 978-986-670-279-2| last = 湯錦台| title = 大航海時代的台灣| location = 臺北市| series = 發現台灣| date = 2011}}</ref>。

因為90%以上的中國进口石油通过海上运输，而三條主要航線，都包含[[马六甲海峡|马六甲海峡]]及台湾海峡，學者[[杨晓辉|杨晓辉]]認為這兩個海峡成為中国石油战略的两个“敏感区”（相關位置參見不同視角的兩圖如右）<ref name="杨晓辉"/>。

== 海峽水域 ==
臺灣海峽最窄處是福建省[[平潭島|平潭島]]與新竹商港之間，直線距離{{convert|130|km}}；最寬處是[[屏東縣|屏東縣]]的[[貓鼻頭|貓鼻頭]]到福建[[東山島|東山島]]的[[澳角|澳角]]，直線距離{{convert|410|km}}。海域面積為80000平方公里，是[[臺灣島|臺灣島]]近海中最大的[[海峽|海峽]]<ref name="東山陸橋" />。

臺灣海峽水深各處相差很大，雖然平均水深不足{{convert|50|m}}，但是，海峽南部斷續分佈與中國大陸之間的一片砂質淺灘的水深最淺，小於{{convert|30|m}}；東南部的大陸坡則水深超過{{convert|1000|m}}<REF name="東山陸橋"/>。

臺灣海峽自然環境複雜，水域內多有淺灘、溝谷、海島礁石；河流入海口和海灣也很多，水團和海水流系多，漁場和水產資源也很豐富。台灣海峽以風大，浪大，[[流速|流速]]大著稱<ref name="東山陸橋" />。

臺灣海峽海流存在明顯季節變化，[[冬季|冬季]]以從北向南的沿岸流為主，[[夏季|夏季]]以由南向北的暖水以及從[[太平洋|太平洋]]進入的[[黑潮|黑潮]]支流為主<ref name="東山陸橋" />。

== 气候 ==
臺灣海峽[[季风|季风]]交替明显，频繁的偏北风非常强劲。每年10－次年4月[[東北季風|东北风]]为主；6－8月[[西南氣流|西南风]]为主。

每年强烈的[[颱风|颱风]]伴随暴雨，造成潮水位变化剧烈，对沿岸侵蚀很强烈。

== 歷史人文 ==
* 16世紀中葉，[[葡萄牙人|葡萄牙人]]航經台灣附近海峽，看到青翠碧綠的台灣島時，忍不住讚嘆「Ilha Fermosa」，[[葡萄牙文|葡萄牙文]]意即「美麗（Formosa）島（Ilha）」。因此早期也被[[歐洲|歐洲]]稱為「'''福爾摩沙海峽'''」<REF name="AutoMP-2"/><REF name="AutoMP-3"/>。
* 早期的臺灣海峽被稱為「[[黑水溝|黑水溝]]」（因有[[黑潮|黑潮]]流經），許多來自[[閩|閩]]、[[粵|粵]]的移民渡過臺灣海峽時，不慎發生[[海難|海難]]，有[[河洛|河洛]]與[[客家|客家]]都有[[民謠|民謠]]《[[渡臺悲歌|渡臺悲歌]]》稱偷渡黑水溝來臺者「六死三留一回頭」，意即十人當中，有六人會死在臺灣海峽，有三人會留在臺灣，而一人會不敢繼續渡過黑水溝而重回中國大陸。原因常為偷渡船艙環境險惡與超載<ref>[http://gushi.tw/archives/12930 博物館裡的「潮男」：私渡臺灣的移民與他們的海上歷險] 渡臺有一定程度的失事機率，但不會高得太誇張，至少也會在商人們願意承擔的風險範圍裡面。不過，偷渡是另一回事。...時常升高偷渡風險的一種人為因素：超載。...會悶死人...[[閩平漁事件|閩平漁事件]]
臺灣一位海洋史學者談論他的觀點，他認為「六死三留一回頭」最初可能不是用來描述移民渡臺，而是採借自閩南移民下南洋的「六亡三在一回頭」</ref>。
* 1949年6月，[[中华民国政府|中华民国政府]]推行[[关闭政策|关闭政策]]，不久，[[两岸分治|两岸分治]]。1950年6月22日，[[朝鲜战争|朝鲜战争]]爆发后，[[美国海军第七舰队|美国海军第七舰队]]随即开入台湾海峡实行军事[[封锁|封锁]]，禁止中华人民共和国籍船只航行。在关闭政策的影响下，以台湾海峡为界，中国大陆民用海运分为南北方各自通航，中华人民共和国方面必须租用外国轮船来进行外贸海运。
*与民用船舶类似，[[中国人民解放军海军|解放军海军]]水面舰艇在1974年前亦不从未通过台湾海峡。大陆南北调动小型舰艇——[[鱼雷艇|鱼雷艇]]时，用火车运输<ref>{{Cite web |url=https://finance.ifeng.com/a/20170614/15454521_0.shtml |title=西沙战事紧哪：1974年我海军战舰首次过海峡为何台军不拦截|accessdate=2019-03-30 |author= |date=2017-06-14 |work= |publisher=凤凰网|language=Zh-hans }}</ref>。大型舰艇则以通过台湾岛以东，绕行[[太平洋|太平洋]]的方式进行南北航行。1965年，[[八六海战|八六海战]]和[[崇武以东海战|崇武以东海战]]后，[[中華民國海軍|中华民国海军]]不再对解放军海军主动出击<ref>{{Cite web |url=https://www.guancha.cn/society/2017_03_29_401115.shtml |title=《西沙风云》电影开拍 会出现蒋介石“协助大陆”的剧情吗|accessdate=2019-03-30 |author= |date=2017-03-29 |work= |publisher=观察者网|language=Zh-hans }}</ref>。1974年1月，[[西沙海战|西沙海战]]爆发，为抢时间，[[中国人民解放军海军东海舰队|解放军东海舰队]]南下编队在[[福州军区|福州军区]]岸防火力策应下第一次通过台湾海峡。1990年代，大陆出现解放军[[海军舰队首航台湾海峡|海军舰队首航台湾海峡]]得到[[蒋介石|蒋介石]]允许的伪史——[[西沙战事紧|西沙战事紧]]。
* 1964年[[越南战争|越南战争]]升级后，美国为防止中华人民共和国公开介入参战，停止了美军舰机对中国远洋轮船的拦截、登临检查。1966年3月，[[中华人民共和国交通部|中华人民共和国交通部]]向国务院呈送了《关于中国自营轮船开辟南北航线的请示报告》。1966年5月11日，周恩来总理批复要求交通部尽快拟定试航方案。试航。1968年4月25日，[[广州远洋运输公司|广州远洋运输公司]]“黎明”远洋货轮在[[湛江港|湛江港]]装载11000吨蔗糖，交通部分管海运的副部长[[于眉|于眉]]担任组长，交通部远洋局副局长袁之平任副组长的工作小组随船指导，各相关单位派出40余人携带轻武器随船护航并协助工作。“黎明”轮配备双船长、双政委，沿国际习惯南下航行到北纬10°附近，转向[[巴拉望岛|巴拉望岛]]与[[加里曼丹岛|加里曼丹岛]]之间的[[巴拉巴克海峡|巴拉巴克海峡]]，进入[[苏禄海|苏禄海]]，沿菲律宾[[棉兰老岛|棉兰老岛]]北岸通过[[苏里高海峡|苏里高海峡]]及[[圣贝纳迪诺海峡|圣贝纳迪诺海峡]]进入太平洋[[菲律宾海|菲律宾海]]，再向东北驶至东经130°折向日本沿海的[[大隅海峡|大隅海峡]]，进入[[东海|东海]]，历时共十五天驶抵[[青岛|青岛]]，航程4533海里。黎明轮于6月2日按照原航线南下，交通部远洋局局长、中远公司总经理张公忱随船同行，经过12个昼夜的行驶，于6月14日安全抵达湛江。随后又经“九江”轮、“红旗”轮（船长[[钱永昌|钱永昌]]后任交通部部长）试航成功，避开中华民国空军500海里作战半径。1968年10月22日，周恩来总理批准绕道[[西太平洋|西太平洋]]深处的[[打通南北航线|海上南北航线正式开航]]。到1972年已有63艘船舶318个航次航经南北航线。
* 1972年10月22日，为开辟较近的经巴林塘海峡绕过台湾东部海域的南北航线，广州海运局的沿海散货船“五指山”轮悬国旗，满载9011吨铁矿石从海南[[八所港|八所港]]启航。16天3185海里航行，经菲律宾[[圣地哥角|圣地哥角]]，[[佛提岛|佛提岛]]航道，入[[锡布延海峡|锡布延海峡]]，再经提卡俄航道，出[[圣柏那提诺海峡|圣柏那提诺海峡]]进入西太平洋，绕过台湾岛和琉球群岛，折入[[大隅海峡|大隅海峡]]，然后进入东海，驶抵大连港，于11月7日抵达大连港，开辟了新的南北航线。 
*  1979年5月29日广州远洋局“眉山”轮悬挂国旗在白天公开穿越台湾海峡，从黄埔港驶抵日本[[名古屋|名古屋]]。随后，上海远洋“江城”轮、广州远洋“富春江”轮、“明华”轮，天津远洋“天门”轮，广州海运局“红旗121”、烟台救捞局“沪救101”、“德安”两个拖轮船队。1979年6月22日颁布了《交通部关于我商船通行台湾海峡的暂行规定》。这标志着中国海运业在[[台湾海峡复航|台湾海峡复航]]。 
* 臺海兩岸在1996年3月8日至3月25日間，[[中華人民共和國|中華人民共和國]][[中國人民解放軍|解放軍]]在臺灣海峽進行飛彈試射演習，对中華民國的[[1996年中華民國總統選舉|首次總統民選]]造成了一定的影响，[[美國|美國]]出動[[獨立號航空母艦_(CV-62)|獨立號]]與[[尼米茲號航空母艦|尼米茲號]]兩艘[[航空母艦|航空母艦]]前往臺灣海峽鄰近水域警戒，協防臺灣的安全。這個引發國際關注的事件被稱為「[[臺灣海峽飛彈危機|臺灣海峽飛彈危機]]」。
* 2013年11月18日，[[中華人民共和國|中華人民共和國]][[中國人民解放軍|解放軍]][[遼寧號航空母艦|遼寧號航空母艦]]首度穿越台灣海峽，引起台灣軍方緊張；而此次是第一次有美軍以外航母通過台海；而在2017年7月初，遼寧號前赴[[香港|香港]]訪問及離開香港途中，第三及第四次穿越台灣海峽。
* 2018年7月7日，美国两艘军舰[[馬斯廷號驅逐艦_(DDG-89)|馬斯廷號驅逐艦]]及[[本福尔德号驱逐舰|本福尔德号驱逐舰]]由台湾南部海域航经台湾海峡，向东北还行<ref>[http://cn.rfi.fr/%E4%B8%AD%E5%9B%BD/20180707-%E4%B8%A4%E8%89%98%E7%BE%8E%E5%9B%BD%E4%B8%BB%E5%8A%9B%E9%A9%B1%E9%80%90%E8%88%B07%E6%9C%887%E6%97%A5%E9%80%9A%E8%BF%87%E5%8F%B0%E6%B9%BE%E6%B5%B7%E5%B3%A1 两艘美国主力驱逐舰7月7日通过台湾海峡].rfi.</ref>。由于[[中华民国|中华民国]][[蔡英文|蔡英文]]当选总统以来，两岸关系不断紧张，在中国大陆不断绕飞台湾岛——[[绕台岛巡航|绕台岛巡航]]，并进行一系列军事演习后，美国决定海军定期、但不频繁的穿越台湾海峡行动<ref>[http://www.guancha.cn/internation/2018_06_05_459019.shtml 外媒称美军考虑派军舰通过台湾海峡 中方回应].观察者.</ref>。同年10月及11月，美軍先後派遣[[柯蒂斯·威爾伯號驅逐艦|柯蒂斯·威爾伯號驅逐艦]]、{{link-en|安提坦號巡洋艦|USS Antietam (CG-54)}}、{{link-en|史托戴爾號驅逐艦|USS Stockdale (DDG-106)}}及{{link-en|佩科斯號補給艦|USNS Pecos (T-AO-197)}}通過台灣海峽<ref>{{Cite web |url = https://fnc.ebc.net.tw/FncNews/Content/74698 |title = 美艦9個月內6度通過台灣海峽！國防部這樣說 |author = 東森財經新聞 |date = 2019-03-26 |publisher = 東森電視事業股份有限公司 |language = zh-tw |accessdate = 2019-04-22}}</ref>。
* 2019年3月24日，美軍再次派遣柯蒂斯·威爾伯號驅逐艦及隸屬[[美國海岸防衛隊|美國海岸防衛隊]]的[[巴索夫號緝私艦|巴索夫號緝私艦]]穿越台灣海峽<ref>{{Cite web |url = https://www.hk01.com/%E5%8D%B3%E6%99%82%E5%9C%8B%E9%9A%9B/310008/%E7%BE%8E%E5%9C%8B%E5%86%8D%E6%B4%BE%E8%89%A6%E9%9A%BB%E7%A9%BF%E8%B6%8A%E5%8F%B0%E7%81%A3%E6%B5%B7%E5%B3%BD-%E7%B6%AD%E8%AD%B7%E5%8D%B0%E5%A4%AA%E5%9C%B0%E5%8D%80%E8%87%AA%E7%94%B1%E9%96%8B%E6%94%BE |title = 美國再派艦隻穿越台灣海峽　維護印太地區自由開放 |author = 歐敬洛 |date = 2019-03-05 |publisher = 香港01有限公司 |language = zh-hk |accessdate = 2019-04-22}}</ref>。同年4月，印度海軍的{{link-en|加爾各答號驅逐艦|INS Kolkata}}經台灣海峽北上青島，參與[[中國人民解放軍海軍成立70周年海上閱兵|中國人民解放軍海軍成立70周年海上閱兵]]<ref>{{Cite web |url = https://news.ltn.com.tw/news/world/breakingnews/2766433 |title = 印度軍艦經台灣海峽北上青島 參加中國閱艦式 |author = 自由時報 |date = 2019-04-22 |language = zh-tw |accessdate = 2019-04-22}}</ref>。
