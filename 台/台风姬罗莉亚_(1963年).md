'''颱風姬羅莉亞'''（{{lang-en|'''Typhoon Gloria'''}}，國際編號：'''6314'''，中國大陸編號：'''6312'''，[[聯合颱風警報中心|聯合颱風警報中心]]：'''14W'''，[[菲律賓大氣地球物理和天文管理局|菲律賓大氣地球物理和天文管理局]]：'''Oniang'''）是[[1963年太平洋颱風季|1963年太平洋颱風季]]中的一個[[熱帶氣旋|熱帶氣旋]]。葛樂禮形成後向西北方向前進，對[[臺灣|臺灣]]和[[中國大陸|中國大陸]]等地造成了嚴重的影響。

葛樂禮颱風是標準的「[[西北颱|西北颱]]」<ref name="西北颱的代名詞--葛樂禮颱風回顧(一)"/>，西北颱不論在強風或豪雨都是威力驚人。葛樂禮颱風除了造成全臺相當慘重的災情外，也讓臺灣人民對於往後的「西北颱」可謂聞之色變<ref name="民國52年葛樂禮颱風災害與臺灣省政府救災"/><ref name="西北颱的代名詞--葛樂禮颱風回顧(一)"/>。

== 發展過程 ==
1963年9月初，正當費依颱風通過[[巴士海峽|巴士海峽]]時，在[[關島|關島]]西方近海有一擾動正在醞釀<ref name="西北颱的代名詞--葛樂禮颱風回顧(二)"/>。隨著費依颱風進入[[南海|南海]]，關島附近的擾動也增強成為[[輕度颱風|輕度颱風]]，9/6早上8時命名為「葛樂禮」<ref name="西北颱的代名詞--葛樂禮颱風回顧(二)"/>。1963年9月6日，姬羅莉亞在[[關島|關島]]西北方的海面上生成，初向西行，並迅速發展。7日轉為西北西行進。9日台灣各地風雨漸增。10日凌晨，暴風半徑已到達台灣東海岸<ref name="鄭懿瀛"/>。11日從[[台灣|台灣]][[基隆北海岸|基隆北海岸]]擦肩而過後續以西北方向前進，強度減弱，12日在中國福建[[连江|连江]]登陆。

==== 當地發布之最高颱風警特報 ====
:當地發出最高颱風警告：[[海上颱風警報|海上]][[陸上颱風警報|陸上颱風警報]]

9月10日傍晚，「臺灣省氣象所」發佈颱風警報單指出，葛樂禮將偏北移動。

==== 經過 ====
9月9日凌晨兩點，台灣北部東北部東部地區受葛樂禮颱風外圍環流影響，逐漸出現陣雨<ref name="西北颱的代名詞--葛樂禮颱風回顧(二)"/>。

9月10日早上到下午間，颱風的行進方向偏西北到北北西，讓當時的「臺灣省氣象所」（[[交通部中央氣象局|交通部中央氣象局]]前身）認為葛樂禮將被北方[[低壓槽|槽線]]所牽引。

9月10日下午兩點，葛樂禮颱風此時中心距離[[花蓮市|花蓮]]東方僅剩350公里左右。

9月10日傍晚，「臺灣省氣象所」發佈颱風警報單指出，葛樂禮將偏北移動，不料北方槽線並不夠深，無法導引葛樂禮北上。

9月10日晚上八點，葛樂禮已經來到[[宜蘭市|宜蘭]]東方不到300公里海面上，並轉向西北西移動<ref name="西北颱的代名詞--葛樂禮颱風回顧(二)"/>。

9月11日，葛樂禮侵襲[[台灣北部|台灣北部]]，豪雨如注。

9月12日，颱風中心行至[[彭佳嶼|彭佳嶼]]附近，造成台灣中北部嚴重的水患，溪流氾濫<ref name="1963 年葛樂禮(GLORIA)颱風"/><ref name="鄭懿瀛"/>。

==== 災情 ====
台灣受颱風葛樂禮風向影響，颱風掠過宜蘭、臺北等地區，造成北部地區相當大的豪雨。由於雨量太大及正值漲潮期，並由於排水系統過於老舊，再加上原來可以含水的地區正在蓋起大樓，導致大雨無處宣洩，積水非常嚴重，雖增加抽水站仍緩不濟急，加上石門水庫洩洪<ref name="民國52年葛樂禮颱風災害與臺灣省政府救災"/>，造成[[台北盆地|台北盆地]]大淹水，整個[[大台北地區|大台北地區]]包括[[三重區|三重]]、[[蘆洲|蘆洲]]等地頓成澤國，整個大台北地區在水裡泡了三天三夜，尤以[[臺北市|臺北市]][[大同區_(臺北市)|大同區]]最為慘重，台北市日新國小附近的淹水達一樓深<ref name="民國52年葛樂禮颱風災害與臺灣省政府救災"/>。

台北縣則以[[板橋區_(新北市)|板橋]]浮洲里婦聯一、二村淹水最為嚴重<ref name="鄭懿瀛"/>。

全台共有224人死亡、450人受傷及88人失蹤<ref name="民國52年葛樂禮颱風災害與臺灣省政府救災"/>，房屋全倒者計13,950戶，半倒者計10,783戶，農林漁牧、交通產業與民眾財產等損失，總計高達[[新臺幣|新臺幣]]14億元以上<ref name="民國52年葛樂禮颱風災害與臺灣省政府救災"/>。

==== 統計 ====
甫落成的[[石門水庫_(臺灣)|石門水庫]]緊急採取洩洪措施，洩洪速率高達每秒一萬立方公尺左右<ref name="西北颱的代名詞--葛樂禮颱風回顧(一)"/>。

這次是石門水庫正式竣工的首度洩洪，根據水庫管理統計，洩洪量為五億七千四百餘萬立方公尺<ref name="鄭懿瀛"/>。

==== 後期研究 ====
根據後來的研究和統計，葛樂禮颱風之所以會為台灣帶來如此嚴重的災情，主要是破紀錄的降雨量。

氣象資料顯示，當葛樂禮從9月9日到12日過境台灣期間，雨量最大集中地有兩個區域，一為[[淡水河|淡水河]]、[[頭前溪|頭前溪]]及[[後龍溪|後龍溪]]上游，四天的總雨量在1,700到1,800公釐之間，影響範圍更及於[[大安溪|大安溪]]和[[大甲溪|大甲溪]]上游<ref name="鄭懿瀛"/>；另一個區域則為[[阿里山鄉|阿里山]]區，總雨量約1,500公釐<ref name="鄭懿瀛"/>。

至於台北與桃園之間，總雨量雖然只有400公釐左右，但因[[大屯山|大屯山]]區有一處降雨量超過1,000公釐的中心，加上[[淡水河|淡水河]]上游[[大漢溪|大漢溪]]降下暴雨，水流匯合才會導致氾濫成災<ref name="鄭懿瀛"/>，加上當時市區排水能力不足。

=== {{CHNML}} ===
中國大陸[[浙江|浙江]]、[[上海|上海]]等部份地區則因暴雨造成188人丧生。

== 後續檢討 ==
=== {{TWN}} ===
葛樂禮風災後，政府及民眾也從葛樂禮風災及水患後開始重新檢討各項防風防洪的建設工作<ref name="鄭懿瀛"/>。例如，在葛樂禮颱風來襲時造成極大損失的[[社子島|社子島]]，雖然自古以來便飽受水患之害，但直到被葛樂禮重創之後，才引發幾項重大的變遷措施，例如[[基隆河|基隆河]][[截彎取直|截彎取直]]工程<ref name="鄭懿瀛"/>。1964年開始，將基隆河舊河道（即現在士林區公所所在的基河路）填平，另闢目前較直的新河道。兩地之間則新築了現在的百齡橋。

葛樂禮風災後，政府有關單位認為水患係因[[關渡|關渡]]門過窄宣洩不及所致，遂於1964年進行拓寬淡水河道工程，將[[淡水河|淡水河]]流出臺北盆地出口處西側的象鼻、獅子頭等隘口炸毀<ref name="鄭懿瀛"/>。
