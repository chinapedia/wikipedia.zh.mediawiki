'''台制'''是[[台灣|台灣]]所採用的[[度量衡|度量衡]]制度。在[[台灣日治時期|台灣日治時期]][[臺灣總督府|臺灣總督府]]多次頒布度量衡相關法規，逐步以[[日本|日本]]的[[尺貫法|尺貫法]]統一臺灣的度量衡制度。<ref>{{Cite web |url=http://www.digital.ntu.edu.tw/workshop0121/ppt/05.pdf |title=二、並行的近代化 （二）臺灣度量衡制度的近代化 |accessdate=2012-10-04 |author=陳慧先 |date=2011年1月21日星期五 |work=斷裂、衝突與延續──戰後臺灣度量衡的改制與爭議 第6頁|publisher=國立臺灣大學歷史學研究所博士生}}</ref>故部份臺制度量衡單位的數值與日制相同，如現今慣用的「台斤」、「台尺」。另外也有部分單位是延續自其他時期，如源自於荷治時期的面積單位「甲」。[[大正|大正]]十三年（1924年），臺灣總督府試圖推動[[公制|公制]]度量衡（[[国际单位制|国际单位制]]），但不成功。<ref>{{Cite web |url=http://www.digital.ntu.edu.tw/workshop0121/ppt/05.pdf |title=二、並行的近代化 （二）臺灣度量衡制度的近代化 |accessdate=2012-10-04 |author=陳慧先 |date=2011年1月21日星期五 |work=斷裂、衝突與延續──戰後臺灣度量衡的改制與爭議 第6-7頁|publisher=國立臺灣大學歷史學研究所博士生}}</ref>
{{Wikisource|度量衡法 (民國73年)}}
民國73年（1984年），《度量衡法》全文修正後，主管度量衡相關事務的[[經濟部中央標準局|經濟部中央標準局]]曾一度強制取締非公制的度量衡器，以利全面推行公制、淘汰台制，但遭民間反彈，並引發中央與臺灣省議會之爭，最後並未徹底落實。<ref>{{Cite web |url=http://www.digital.ntu.edu.tw/workshop0121/ppt/05.pdf |title=四、1980年代公制與臺制之爭（一）背景經過 |accessdate=2012-10-04 |author=陳慧先 |date=2011年1月21日星期五 |work=斷裂、衝突與延續──戰後臺灣度量衡的改制與爭議 第11頁|publisher=國立臺灣大學歷史學研究所博士生}}</ref>在臺灣本島，傳統市場至今仍普遍使用台制單位，市場中所稱之「[[斤|斤]]」即為[[台斤|台斤]]。此外，雖然目前政府機構以公制單位登記面積，但源自日本的[[面積|面積]]單位「[[坪|坪]]」及[[荷蘭|荷蘭]]的「[[甲_(單位)|甲]]」等台制單位，在臺灣民間還是頗為常見。
