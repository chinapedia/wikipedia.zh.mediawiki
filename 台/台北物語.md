《'''台北物語'''》（{{lang-en|'''''Story in Taipei'''''}}），是一部[[2017年電影列表|2017年]]上映的[[台灣電影列表|台灣電影]]。由擔任過[[金馬獎|金馬獎]]評審的劇作家[[黃英雄|黃英雄]]編導，[[陳怡安|陳怡安]]、[[李宓_(台灣藝人)|李宓]]、[[邱志宇|邱志宇]] 領銜主演，2017年5月19日於台北與高雄[[有限上映|小型上映]]。電影本身製作品質不佳，反被觀眾認為「爛得有特色」，劇情台詞洗腦。上映後產生[[網路爆紅|網路爆紅]]現象，在[[台灣|台灣]][[端午節|端午節]]假期時，造成播映戲院場場爆滿的熱潮，並引發媒體報導。<ref>[https://stars.udn.com/star/story/10090/2491887?from=udn-ch1_breaknews-1-cate8-news 神片「台北物語」爛口碑逆襲 5大奇點不可不看！]</ref> 映期進入第二週後爆滿熱潮依然不退減，上映第三週時在台北市單以兩廳戲院播映，卻仍得到178萬票房的好成績。<ref>[https://my-cte.com/2017/06/06/%E5%BD%B1%E5%BB%B3%E8%A7%80%E7%9C%BE%E7%8B%82%E5%91%BC%E5%96%8A%E3%80%8Ctony%E3%80%8D-%E7%86%AC11%E5%B9%B4%E5%9B%A0%E3%80%8A%E5%8F%B0%E5%8C%97%E7%89%A9%E8%AA%9E%E3%80%8B-%E7%88%86%E7%B4%85%E5%9D%A6/ 影廳觀眾狂呼喊「Tony」熬11年 因《台北物語》爆紅坦言心情複雜]</ref>在長達一個月的上映後，台北市票房已累積至450萬。<ref>[https://stars.udn.com/star/story/10090/2549835 「台北物語」苦撐多日賺450萬 Tony哥出頭天]</ref>並於下檔半個月後重新再度上映。<ref>[http://www.chinatimes.com/newspapers/20170713000648-260112　《台北物語》神片重新上映]</ref>高話題的口碑延燒至海外，入選香港坎坷影展開幕片，風潮席捲香港。<ref>[https://news.tvbs.com.tw/entertainment/765629　《台北物語》紅到香港　開幕片電影票秒殺 ]</ref>

== 劇情簡介 ==
議員酒後駕車，撞了開計程車的男偷車賊。而車上女乘客在這場車禍認出了酒駕者是名政客，並且識破了身旁的助理為第三者，因此起了貪念，決定與偷車賊向議員索求500萬的『封口費』。但就在兩人前往取款的過程裡，卻驚爆了女乘客與議員妻子之間的醫療糾紛，同時也抖出了包商與政商勾搭的弊案，弊案裡卻也牽連到偷車賊與女助理過去的家庭。
為什麼包商會出現在車禍當中的後車廂裡?是謀殺?綁架?還是另有隱情?! 身為議員的柯錫恩與這弊案是否有關連? 如後宮傳般錯綜複雜的正宮與小三的對決，最後該如何收場? 權貴政商與庶民間恩恩怨怨又到底發生了什麼事?在這個怪異的日子，每個人都竭盡所能的探索別人的罪狀，卻總是可以為自己的罪狀找到無罪的理由。有罪無罪的界線出現一種矛盾和詭異，而這一切也都直接指向複雜的人性。

== 網路爆紅 ==
1.本片早於2014年拍攝完成<ref>[http://ent.ltn.com.tw/news/breakingnews/2082967 《台北物語》爛到爆紅 主角資生堂千金這麼說]</ref>。初期反應冷淡，但因有網友撰文指出片中許多劇情不合邏輯 ( 如台詞過於文青及不能理解含意的另類主角「吊扇與陶瓷狗」)，以及技術層面低劣 ( 如畫面搖晃、切換方式 ) 等部分，該文章在[[臉書|臉書]]上引發大量分享，引發討論熱潮後，在2017年台灣端午節假期時，其播放戲院均取得爆滿的票房成績，吃掉同檔期《[[神力女超人_(2017年電影)|神力女超人]]》在喜滿客影城放映廳數，更創造觀影後起立鼓掌及討論度極高的盛況，影迷甚至主動為劇中演員 TONY [[邱志宇|邱志宇]] 舉辦粉絲見面會，250人座位瞬間秒殺。<ref>[http://star.ettoday.net/news/935558　超荒唐！《台北物語》TONY哥見面會20分完售 粉絲哭排候補]</ref>並獲香港坎坷影展青睞，獲邀成為開幕片，人氣爆棚。<ref>[http://star.ettoday.net/news/1004960?t=%E3%80%8A%E5%8F%B0%E5%8C%97%E7%89%A9%E8%AA%9E%E3%80%8B%E7%B4%85%E5%88%B0%E5%9C%8B%E5%A4%96%EF%BC%81%E5%B7%A8%E6%98%9F%E8%A6%8F%E6%A0%BC%E8%AB%8BTony%E9%A3%9B%E5%8E%BB%E2%8B%AF%E7%A5%A8%E5%88%B8%E7%A7%92%E6%AE%BA　《台北物語》紅到國外！巨星規格請Tony飛去⋯票券秒殺]</ref>

2.影迷於上映一週年自掏腰包舉辦《台北物語》回魂場，使得電影四度重回首輪電影院放映<ref>[https://news.ebc.net.tw/news.php?nid=114285  粉絲包場回味《台北物語》 導演曝：續集將開拍]</ref>
