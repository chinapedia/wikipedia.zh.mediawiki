=== 研發 ===
許志聖花了12年孕育「台梗九號」。許憶述， 1975年（民國64年）至1981年（民国70年）間，台灣[[水稻|水稻]]栽培面積逐年遞增，在經濟發展之下，食[[米|米]]消費量卻逐年減少。當時稻作研究人員興起以高品質米提高消費量的構想，開始研究如何種出好吃的新品種。74年間，許志聖進入台中農改場，接下前人做了一半的研究，後選育出黏性、硬度適中，咀嚼後還泛香甜的新品系'''台梗育8269號'''（台稉九號）。在[[台灣|台灣]]推出之後，廣受消費者喜愛，甚至一向以本土稻米為傲的[[日本|日本]]，都曾到台中農改場要求技術轉移。<ref name="凌筠婷">{{Cite web |url = http://udn.com/news/story/7314/1399899 |title = 孕育出台灣好米 許志聖功成退休 |author = 凌筠婷 |date = 2015-12-25 |publisher = 聯合新聞網 |language = zh-tw |accessdate = 2016-09-07 |quote = 許志聖說，台稉九號命名他只開心一天，隔天繼續下田，許下心願「要育出能打敗台稉九號的品種」，21年來他又育出台稉糯五號、台稉15號、台中191號。 }}</ref>

之後，他又以台稉九號為母本，雜交其他品種，再選育出具有[[七葉蘭|七葉蘭]]香味的新米，並於2009年命名為「台中194號」開始推廣。<ref name="凌筠婷" /><ref>{{Cite web |url = http://www.merit-times.com.tw/NewsPage.aspx?unid=425541 |title = 台梗九號之父退休 許志聖 相信台灣米會更好 |author =  |date = 2015-12-25 |publisher = 人間福報 |language = zh-tw |accessdate = 2016-09-10 |quote = 「育種要常下田」許志聖說，育種要做好，基本功一定要紮實，他下田和農人討論遇到的困難和問題，從中找靈感。雖然到退休，他仍無法打敗台梗九號，但發現：「世界上沒有完美品種，只有順應趨勢，最適合市場產品」。 }}</ref>

=== 品種特性 ===
產量偏低，易感染白葉枯病<ref>{{Cite web |url = http://www.airitilibrary.com/Publication/alDetailedMesh1?DocID=U0042-0905201415251552 |title = 水稻台稉9號細胞培養及篩選耐白葉枯病菌細胞系之初探 |author = 張雅嵐 |date = 2013-01-01 |publisher = 屏東科技大學植物醫學系所 |language = zh-tw |accessdate = 2013-0910 |quote = 本研究之目的為研發水稻癒傷組織誘導培養技術、懸浮細胞建立培養及初代培養技術與篩選抗白葉枯病細胞系。…… }}</ref>。氮肥施用過多，更易感染白葉枯病。農政單位一般建議，栽種台稉9號減少氮肥<ref>{{zh-tw}} [http://kmweb.coa.gov.tw/Category/d.aspx?documentId=17846&fileName=Exten-6204.pdf&ver=2 台稉9號合理化施肥技術]</ref>，降低施用率10～20％，儘早施用追肥。<ref>{{Cite web |url = http://www.mdais.gov.tw/view.php?catid=1589 |title = 適合苗栗地區之良質米品種 |author = 張素貞 |date = 2016-09-09 |publisher = 苗栗區農業改良場 |language = zh-tw |accessdate = 2016-09-09 |quote = 適合苗栗地區良質米品種有台稉 8 號、台稉9號、台稉14號及台中秈10號。就產量及栽培特性而論，除台稉9號較低產難管理外，其他三個品種產量及病蟲害管理較容易。唯台稉8號在近山區地 區栽培要注意稻熱病防治，台稉14號第二期作後期遇低溫，抽穗不整齊問題，要加強注意。米質表現則以台稉9號食味較穩定，在第二期表現仍可達到A級優異表 現，所以苗栗地區主要生產「台灣好米」原料米以台稉9號為主。近幾年新品種陸續推出，台農71號（益全香米）也在苗栗後續竄起，但因其不易管理，目前在苗 栗栽培面積有限，分散於西湖、苑裡、公館等鄉鎮，其在苗栗適栽性仍在評估中。對於新品種適應性，本場將計畫持續示範試驗之，期為苗栗地區尋求到最好的良質米品種。 }}</ref>

== 市場 ==
由於食用口感媲美[[越光米|越光米]]，在台灣地區，[[7-Eleven|7-Eleven]]旗下商品御飯糰曾以採用「台稉9號」為主打訴求<ref name="汪文豪" />。

2012年，[[農糧署|農糧署]]經過21個米穀公會會員調查市場偏好的水稻品種。台梗9號因米粒透明飽滿、黏性佳、口感好，廣受喜愛。其平均價格達到每公斤80元以上（[[新台幣|新台幣]]），已超過一般市售白米的2倍價。雖受消費者歡迎，但種植面積僅1萬[[公頃|公頃]]上下，不到台灣水稻面積5％。<ref name="鍾麗華">{{cite news |language = zh-tw |author = 鍾麗華 |coauthors =  |url = http://news.ltn.com.tw/news/life/breakingnews/621306 |title = 國人最受歡迎稻米　台梗九號居榜首 |publisher = 自由時報 |pages =  |date = 2012-03-29 |accessdate = 2016-09-07}}</ref>

=== 混充情形 ===
市場上，台稉9號參雜其他品種的情況非常嚴重<ref>{{cite news |language = zh-tw |author = 洪素卿、劉曉欣、吳俊鋒、王秀 |coauthors =  |url = http://iservice.ltn.com.tw/2013/specials/foodsafty/news.php?type=a&no=733645 |title = 號稱台稉9號 10粒找不到1粒 |publisher = 自由時報 |pages =  |date = 2013-11-27 |accessdate = 2016-09-07}}</ref>。民102年第三季，[[農糧署|農糧署]]抽驗發現，陸和碾米廠「金農頂級金の米-純台稉9號米」10粒米中，沒有1粒台稉9號；「台糖有機米台稉9號」在10粒米中，只有1粒。前銷糧商泉順「山水花東台稉9號米」及苗栗糧商的「健康稻好運米」，於10粒米抽驗中，也僅有7粒，抽驗結果同時發布於同年度第四季。<ref name="汪文豪">{{cite news |language = zh-tw |author = 汪文豪 |coauthors = 上下游News&Market |url = http://health.businessweekly.com.tw/AArticle.aspx?ID=ARTL000007546&p=2 |title = 小心被騙！消費者最愛吃的台梗9號，混米也最嚴重 |publisher = 商業周刊 |pages = 1-4 |date = 2014-01-21 |accessdate = 2016-09-07}}</ref>

[[中華民國|中華民國]]米穀商業同業公會全國聯合會理事長[[吳源昌|吳源昌]]將原因推給改良場與秧苗場，認為其提供的秧苗品種不純。更有糧商抱怨，台稉9號米有姊妹系品種；[[台灣大學|台灣大學]]農藝系判定為台稉9號，而農委會委託[[食品工業研究所|食品工業研究所]]檢驗判定為台稉8號<ref name="蘋果資料室" />。翁良材駁斥吳源昌的說法，他表示台灣有和日本相同的「水稻良種三級繁殖制度」，三級繁殖制度下要求提供的秧苗品種純度在99.95%以上（低於此值、發芽率過低、無生命物質過多，均全部打掉），再怎麼不純，也不可能出現十粒米抽驗，卻只有七粒是台稉9號的情形。<ref name="汪文豪" />對於米穀公會的不滿，[[農糧署|農糧署]]回應，已考量國內生產環境，並參考國際食米標示方式，會檢討調整相關規定，與國際接軌；至於糧商提到的，市售包裝米的純度不足，農糧署解釋，若是農民自行留種子，未從指定的種苗繁殖業者購買，就可能發生純度不足的問題。<ref>{{Cite web |url = http://newtalk.tw/news/view/2014-01-15/43666 |title = 市售米抽驗 東田台稉9號米沒1粒真 |author = 翁嫆琄 |date = 2014-01-15 |publisher = 新頭殼newtalk |language = zh-tw |accessdate = 2016-09-07 |quote = 不過，針對農糧署的抽驗規範，米穀公會則是出面喊冤，強調白米分級標準的各等級間，最小可能只有0.1%的落差，加上抽檢方式難免有誤差值，以致易造成產品標示不符規定，並非為劣米混充。 }}</ref>

== 抗癌效果 ==
[[馬偕紀念醫院|馬偕紀念醫院]]臺北總院和[[嘉義大學|嘉義大學]]進行偕同研究<ref>{{cite news |language = zh-tw |author = 李書璇 |coauthors = 中廣新聞 |url = http://www.epochtimes.com/b5/6/7/24/n1397034.htm |title = 防癌米當紅引發買氣 台灣米詢問度大幅增加 |publisher = 《大紀元》轉發 |pages =  |date = 2006-07-24 |accessdate = 2016-09-07}}</ref>，發現台稉九號的成份能抗[[癌|癌]]，該篇[[文獻|文獻]]並登上國際[[學術期刊|學術期刊]]。<ref>{{cite news |language = zh-tw |author = 朱詠薇 |coauthors =  |url = http://news.tvbs.com.tw/local/358717 |title = 抗癌米就是它！　台稉9號白米、糙米 |publisher = TVBS |pages =  |date = 2006-07-10 |accessdate = 2016-09-07}}</ref>研究團隊拿台稉九號進行糊化實驗，發現其[[白米|白米]]、[[糙米|糙米]]具有提升[[免疫|免疫]]力，扼殺[[癌症|癌症]]細胞的效果。<ref>{{zh-tw}} [http://www.freeweb.com.tw/gg/upload/ftp/rice.pdf 市售稻米提升抗腫瘤免疫功能之評估]</ref>研究結果提到，其中的[[醇溶蛋白|醇溶蛋白]]成分具有抗[[白血病|白血病]]免疫功能，可刺激[[單核球|單核球]]等[[免疫細胞|免疫細胞]]分泌[[細胞激素|細胞激素]]，抑制人類白血病細胞的生長和誘導[[巨噬細胞|巨噬細胞]]分化。<ref>{{Cite web |url = http://ndltd.ncl.edu.tw/cgi-bin/gs32/gsweb.cgi/login?o=dnclcdr&s=id=%22097NCYU5103015%22.&searchmode=basic |title = 分離並鑑定台梗九號稻米蛋白質和其抗白血病的免疫能力 |author = 吳佳恬 |date = 2010-03-03 |publisher = 嘉義大學生化科技研究所 |language = zh-tw |accessdate = 2016-09-12 |quote = 本研究目的是分離台粳九號蛋白質並且使用蛋白質體技術進行分析。藉由二維電泳與質譜分析白米（胚乳）和米糠中的蛋白質，我們鑑定出幾個有差異的蛋白質點，包含代謝、運輸、儲存、抗氧化、抵抗疾病和發育相關蛋白。其中，儲存蛋白可能是重要的調控免疫反應物質。我們進一步分別使用二次水、鹽溶液，鹼溶液和酒精分離出四種儲存蛋白，包含：白蛋白（albumin）、球蛋白（globulin）、穀蛋白（glutelin）和醇溶蛋白（prolamin）。經由抗白血病的免疫能力分析後，顯示prolamin對於刺激人類周邊血液單核球細胞（human mononuclear cells, MNC）並抑制人類白血病U937細胞生長，具有高度活性。且抗體的中和反應也證明稻米的prolamin是一個主要的活性成分。此外，發現在小麥的gluten和gliadin萃取成分中有表現會引起腹腔過敏疾病（celiac disease）的gliadin蛋白，但在稻米蛋白中並未被測得。因此我們認為稻米是個理想的不含gluten的食物，它不會引起乳糜瀉等相關疾病。總結，本研究證明稻米的prolamin在抗白血病免疫功能方面可能扮演重要的角色。 }}</ref>
