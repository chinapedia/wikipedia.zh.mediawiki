'''台北植物園'''，位於[[中華民國|中華民國]][[臺北市|臺北市]][[中正區_(臺北市)|中正區]]龍安里[[南海路|南海路]]53號，佔地約8.2[[公頃|公頃]]，隸屬於[[行政院農業委員會林業試驗所|行政院農業委員會林業試驗所]]，是一個備有完整植物蒐集紀錄文件，並進行科學研究、保育、展示及教育的場所。園內熱門景點包括兩處市定古蹟[[欽差行臺|欽差行臺]]及[[台北植物園腊葉館|台北植物園腊葉館]]、[[南門町三二三|南門町三二三]]、荷花池與各主題植物園區，比鄰[[南海學園|南海學園]]，包括[[國立歷史博物館|國立歷史博物館]]、[[教育廣播電台|教育廣播電台]]等。

== 歷史 ==
1895年（明治28年），[[台灣總督府|台灣總督府]]殖產課為瞭解本島樹木性狀及配合造林需要而研議成立苗圃，於12月30日選定[[臺北府城小南門|臺北府城小南門]]的[[大日本帝國陸軍|大日本帝國陸軍]]用地作為苗圃用地，約當今延平南路216巷、延平南路258巷至中華路二段區域。

1900年（明治33年），於小南門外龍匣口庄購入農地12,953坪，進行開墾整地及樹苗移植，始稱「台北苗圃」，範圍約現今植物名人園、植物分類園區、雙子葉植物區至林業試驗所行政大樓一帶。全區劃為15區，除苗圃設施外，新設置母樹園、果樹園及花卉園，從事育苗並進行苗木栽培試驗。

1911年（明治44年），台北苗圃改制為「林業試驗場」，轄有本場（台北苗圃）、嘉義林業試驗支場、恆春林業試驗支場，隸屬於台灣總督府殖產局，負責台灣林業經營及森林資源之調查研究工作。

1921年（大正10年），成立[[植物園|植物園]]。該園創立目的乃是為了培育[[熱帶植物|熱帶植物]]，熟悉[[南洋|南洋]]各種植物與[[生態環境|生態環境]]。因此，該園有著非台灣原生的樹木，如[[大王椰子|大王椰子]]、[[亞歷山大椰子|亞歷山大椰子]]等各種[[棕櫚科|棕櫚科]]植物。

1924年（大正13年），林業部臘葉標本館落成，即為現在座落於[[欽差行臺|欽差行臺]]南側的[[台北植物園腊葉館|台北植物園腊葉館]]古蹟。

1933年（昭和8年），因[[台北公會堂|台北公會堂]]（今[[台北市中山堂|台北市中山堂]]）興建需要，原址既有之欽差行臺廳舍一部份移築至台北植物園，另一部份移至[[圓山動物園|圓山動物園]]。

1949年（民國38年），台灣省立林業試驗所改為台灣省農林廳林業試驗所。

1999年（民國88年），台灣省農林廳林業試驗所改為[[行政院農業委員會林業試驗所|行政院農業委員會林業試驗所]]。

== 園區設施 ==
[[File:植物園腊葉館.jpg|thumb]]]]
[[File:欽差行臺.jpg|thumb]]]]
園內有二級古蹟[[欽差行臺|欽差行臺]]、市定古蹟[[台北植物園腊葉館|台北植物園腊葉館]]、仿日式木造平房[[南門町三二三|南門町三二三]]，在部分大王椰子樹幹上甚至還留有[[二戰|二次世界大戰]]遺留的砲彈痕跡。

十二生肖植物園區、裸子植物園區、多肉植物園區、圓形溫室（仿自[[紐約植物園|紐約植物園]]溫室）、植物名人園…等不同的植物生態園區。
