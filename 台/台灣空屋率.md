'''台灣空屋率'''是計算[[台灣|台灣]]全國[[空屋|空屋]][[比率|比率]]的數據，計算方式為每年[[11月|11月]]、[[12月|12月]][[電力|用電]]量低於60度空屋數量除以房屋稅籍[[住宅|住宅]]量所得的[[百分比|百分比]]率。<ref name = "全台逾86萬宅是「空屋」，但仍有7萬多新建屋_2018"> {{ cite web | title = 全台逾86萬宅是「空屋」，但仍有7萬多新建屋 | url = https://www.thenewslens.com/article/103951 | publisher = The News Lens 關鍵評論 | author = 羊正鈺 | language = zh-tw | date = 2018-09-12 }} </ref><ref name = "全台空屋率10.2% 高達86.3萬宅_2017"> {{ cite web | title = 全台空屋率10.2% 高達86.3萬宅 | url = http://ec.ltn.com.tw/article/paper/1130029 | publisher = 自由時報 | author = 徐義平 | language = zh-tw | date = 2017-08-26 }} </ref><ref name = "地政學訊2015.03.11(第49期)_2015"> {{ cite web | title = 地政學訊2015.03.11(第49期) | url = https://landeconomics.nccu.edu.tw/download.php?filename=936_e9e6cc1d.pdf&dir=archive&title=%E7%AC%AC%E5%9B%9B%E5%8D%81%E4%B9%9D%E6%9C%9F%282015%2F03%2F11%29 | publisher = 政治大學地政學系學術發展委員會 林子欽 系主任 | author = 徐世榮 | date = 2015-03-18 }} </ref>

== 低用電度標準 ==
因為空屋調查執行時難以有準確判斷標準，故使用低用電度標準判斷空屋數，低用電度標準空屋以[[台灣電力公司|台灣電力公司]]網站，250公升中型[[電冰箱|電冰箱]]與[[電視機|電視機]]一個月耗電量計算而得，每戶基本用電量為55-62度，因此以60度整數為標準計算，從2008年開始，於每年11月與12月計算年度低用電量戶數。<ref name = "低度使用(用電)住宅_2018"> {{ cite web | title = 低度使用(用電)住宅 新建餘屋(待售)住宅 統計資訊簡冊 中華民國 107 年 9 月 | url = http://pip.moi.gov.tw/V2/Z/SCRZ0004.aspx?Func=SCRE0104&Param=20180910313827.pdf%2c%e7%b5%b1%e8%a8%88%e8%b3%87%e8%a8%8a%e7%b0%a1%e5%86%8a(107%e5%b9%b49%e6%9c%88%e5%87%ba%e5%88%8a) | author = 內政部營建署 | date = 2018-09-07 }} </ref>

== 影響 ==
政治大學地政學系研究住宅本身具有異質性，無法完全取代，空屋率高低對於房地產市場無明顯賣壓影響；政治大學張金鶚教授認為，過多的空屋會損耗社會成本與浪費社會資源；政府可以加強租賃制度、制訂租賃條例鼓勵空屋屋主將房子出租。<ref name = "空屋等無人_1995"> {{ cite web | title = 空屋等無人 | url = https://www.gvm.com.tw/article.html?id=3838 | publisher = 遠見天下文化出版股份有限公司 | author = 林惠君 | language = zh-tw | date = 1995-04-15 }} </ref>
