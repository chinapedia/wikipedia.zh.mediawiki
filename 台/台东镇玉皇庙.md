'''台东镇玉皇庙'''，又称'''下村庙'''、'''清溪庵'''、'''台东大庙'''，是中国[[青岛市|青岛市]][[市北区|市北区]]的一座[[道教|道教]]寺庙，原址在[[青岛台东|台东]]道口路、姜沟路路口西侧，可能是青岛市中心范围内已知最早的宗教建筑，现已不存。

==历史沿革==
关于玉皇庙的初建时间，有说法称可追溯到元朝末年，但关于该庙的可考史料最早可上溯至明代，属[[道教|道教]][[全真|全真]][[华山派|华山派]]。德租时期的地图能看到该庙的存在，一般以德文标为“{{lang|de|Tempel}}”（庙宇）。1941年庙宇重建。1952年“扫除封建迷信”的运动中庙宇的宗教活动停止，神像被毁。[[文革|文革]]时大殿、山门等部分被拆毁，仅剩东西两组厢房。庙址先后成为台东半导体零件厂厂房、台东区文化馆、台东区少年宫的活动场地。90年代[[台东区_(中国)|台东区]]撤销，庙址改为市北区文化馆和市北区少年宫。2005年6月仅剩的两组厢房连同附近建筑一并被拆除<ref>{{cite web |url=http://club.qingdaonews.com/showAnnounce_29_2169214_1_1.htm|title=谁来拯救青岛市区最早的寺庙旧址？}}</ref>。

2008年3月29日，道口路9号工地（玉皇庙原址）上发现了两块功德碑，长2.4米，宽0.98米，一块碑上刻有“万善同归”字样，另一块刻有“百世流芳”，同时刻有[[赵琪|赵琪]]所撰碑文“重修清溪庵碑记”，年代均为1941年<ref>{{cite web|url=http://www.51fdc.com/html/2008-03-31/00020627.htm|title=道口路施工出土两块石碑 印证萝卜会最早发源地|date=2008-03-31|work=刘鑫 刘震 周辉 徐新东|publisher=半岛都市报}}</ref>。

==结构==
该庙坐北朝南，东邻杨家村，南临清溪河，故有“清溪庵”之名。庙门内为青龙白虎殿，庙内与传统庙宇结构相同，为四合院式天井，正殿供奉[[玉皇大帝|玉皇大帝]]，西殿供奉[[太上老君|太上老君]]，东殿供奉[[关羽|关帝圣君]]<ref Name=miao>{{cite web |url=http://club.qingdaonews.com/showAnnounce_57_3817020_1_1.htm|title=青岛台东镇的三大庙宇道观|date=2009-10-09}}</ref>。

==玉皇庙萝卜会==
每年农历正月初九为玉帝生日，玉皇庙庙会三天。庙会正值[[立春|立春]]，民间有吃萝卜之习俗，称为“咬春”，故庙会上买卖萝卜者甚多，久而久之，玉皇庙庙会便有“萝卜会”之称。每值庙会，玉皇庙内香火旺盛，周边辽宁路、威海路、台东一路、桑梓路人头攒动，甚是热闹。五十年代后庙会停止。1991年[[青岛市人民政府|市政府]]为招商引资、搞活经济，恢复萝卜会，并将其与元宵山会合并，每年无固定举办场所，但自2010年起萝卜会回到玉皇庙附近的昌乐路（即“青岛文化街”）举办<ref Name=miao/><ref>{{cite web |url=http://www.qingdaonews.com/content/2010-02/22/content_8298114.htm|title=萝卜会今日盛大开幕 地方民俗各具特色(组图)|date=2010-02-22|work=王国强|publisher=青岛新闻网}}</ref>。

==关于清溪河==
清溪河又名小清河、清水河、溪水河，因流经杨家村，德租时期地图统一标为“{{lang|de|Yang tschia tsun Fluß}}”（杨家村河），后来因干流中下游与昌乐路平行，又被称为“昌乐河”。该水体自[[太平山_(青岛)|太平山]]北麓发源，河道走向大致为今聚仙路、桑梓路、道口路、昌乐路。由于城市化进程，该河径流量大幅减小，干支流大部分也已被填埋或人工改为地下径流。该河原本在今昌乐路、大港纬五路路口附近汇入胶州湾，由于上世纪末大港填海造陆，改为在大港港区内汇入[[海泊河|海泊河]]。原入海口以下河段现为地表径流。
