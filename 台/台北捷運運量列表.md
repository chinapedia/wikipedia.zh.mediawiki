'''[[台北捷運|台北捷運]]運量'''自1996年通車後便持續增長，於兩年後的1998年12月22日打破了累積運量1億人次的紀錄，而這個數字也隨著時間的推移，不斷刷新到2005年8月16日的20億人次。除此之外，台北捷運的年運量也在2010年12月29日首度突破5億人次、還有2012年2月4日日運量突破200萬人次，2019年3月31日突破100億人次。<ref name="旅運量">{{cite web |language=zh-tw |author= |title=關於我們>統計資料>旅運量 |url=https://www.metro.taipei/cp.aspx?n=FF31501BEBDD0136&s=1B495F57E6E58102 |publisher=臺北大眾捷運股份有限公司 |date=2019-01-16更新 |accessdate=2019-01-16}}</ref>

==里程碑==
*1998年12月22日：累計突破1億人次。<ref name="旅運量">{{cite web |language=zh-tw |author= |title=關於我們>統計資料>旅運量 |url=https://www.metro.taipei/cp.aspx?n=FF31501BEBDD0136&s=1B495F57E6E58102 |publisher=臺北大眾捷運股份有限公司 |date=2019-01-16更新 |accessdate=2019-01-16}}</ref>
*2001年1月4日：累計突破5億人次。<ref name="旅運量" />
*2005年8月16日：累積突破20億人次，並於2005年8月13日於淡水站前廣場舉行「運量突破20億 健康城市有活力」慶祝活動。<ref name="旅運量" />
*2008年2月22日：運量累計突破30億人次。<ref name="旅運量" />
*2010年4月22日：運量累計突破40億人次。<ref name="旅運量" />
*2010年12月29日：年運量首度突破5億人次。<ref name="旅運量" />
*2012年2月4日：平常日運量首度突破200萬人次，達2,058,072人次<ref name="誕生與成長">{{cite web|language=zh-tw |publisher=臺北大眾捷運股份有限公司 |title=臺北燈節參觀人潮踴躍捷運平常日運量破紀錄達205.8萬 新莊線免費搭乘優惠活動超過256萬人次 |url=http://www.trtc.com.tw/ct.asp?xItem=17223762&ctNode=22278&mp=122031 |author= |date=2012-02-09 |accessdate= |deadurl=yes |archiveurl=https://web.archive.org/web/20140115073023/http://www.trtc.com.tw/ct.asp?xItem=17223762&ctNode=22278&mp=122031 |archivedate=2014-01-15 }}</ref>。
*2012年2月17日：運量累計突破50億人次，並於2012年2月26日（週日）下午2點在捷運淡水站廣場舉行「幸福感謝50億慶祝音樂會」。<ref name="2012年報">{{cite book|language=zh-tw|author=|title=《臺北捷運公司2012年年報》|url=http://www.trtc.com.tw/public/Attachment/31291639891.pdf|publisher=臺北大眾捷運股份有限公司|date=|accessdate=2014-12-20|deadurl=yes|archiveurl=https://web.archive.org/web/20141220082618/http://www.trtc.com.tw/public/Attachment/31291639891.pdf|archivedate=2014-12-20}}</ref>
*2013年10月7日：運量累計突破60億人次，於2013年10月6日（週日）下午2點在台北車站、忠孝復興、忠孝新生、古亭、民權西路舉辦「捷運好行‧橘紅藍綠棕猜猜看」發送恬恬圈手環活動，並於10月12日當天配戴「捷運恬恬圈」手環，可以免費搭乘臺北捷運。<ref name="2013年報">{{cite book |language=zh-tw |author= |title=《臺北捷運公司2013年年報》 |url=http://www.tpech.gov.tw/public/Attachment/482911541995.pdf |publisher=臺北大眾捷運股份有限公司 |date= |accessdate=2014-12-20}}</ref>
*2013年12月8日：運量累計突破61億人次，於2013年12月（週日）上午9點半起在台北車站、市政府站、西門站等三個車站，邀請乘客參加「尋找億載捷客」活動，符合參加資格乘客，可以現場領取號碼牌，獎品包括一百八十天、九十天、及三十天捷運票卡、「好運獎一日票」、「神秘捷客獎」等。選定2013年5月至10月平均日運量最高的台北車站、市政府及西門等3個車站舉行。只要乘客的悠遊卡近6次的交易紀錄中，有1次捷運搭乘紀錄，就符合參加資格，可以在活動現場領取號碼牌。<ref name="旅運量" />
*2017年2月：運量累積達80億人次。<ref>{{cite web |language=zh-tw |publisher=中國時報 |title=中視《60分鐘》軌道經濟帶頭衝 |url=http://www.chinatimes.com/realtimenews/20170323002060-260405 |author=楊婕 |date=2017-03-22 |accessdate=2017-04-16 }}</ref>
*2018年12月：運量累積達98億人次。<ref>{{cite web |language=zh-tw |publisher=臺北大眾捷運股份有限公司 |title=臺北捷運感謝旅客支持及同仁努力　2018年營運可靠度及運量雙創新高|url=https://www.gov.taipei/News_Content.aspx?n=F0DDAF49B89E9413&sms=72544237BBE4C5F6&s=CA51ACDC0D202A97 |author= |date=2019-01-09 |accessdate=2019-01-16 }}</ref>
*2019年3月：運量累積達100億人次。

==歷次跨年==
