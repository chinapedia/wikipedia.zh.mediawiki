'''台兒莊戰役'''為1938年3月至4月中旬[[國民革命軍|國軍]]在[[山東省|山東省]]南邊[[台兒莊|台兒莊]]與意圖由[[山東|山東]]分兩路進攻[[徐州|徐州]]的[[日軍|日軍]]進行的戰役，這場戰役是[[中国抗日战争|中国抗日战争]]中[[徐州會戰|徐州會戰]]的一部分。國軍在以[[徐州|徐州]]為中心地區與日軍激戰，史稱徐州会战。日軍[[第5師團|第5師團]]主攻[[臨沂|臨沂]]，[[第10師團|第10師團]]主攻[[滕縣|滕縣]]、[[臨城|臨城]]、[[台兒莊|台兒莊]]，均遭國軍擊退。因為是對日抗戰爆發後[[國民革命軍|國軍]]首次取得的勝利，又稱為'''台兒莊大捷'''。但當時的[[日本帝國陸軍|日本帝國陸軍]]論點並不認為他們撤退代表戰敗<ref name="kikuchi70-71" />，日軍認為即便這場局部衝突伤亡不小，但是這場戰役只是在擴張縱深的接觸戰，而非帝國大本營派令的正式作戰任務。

== 背景 ==
日軍在1937年[[華北|華北]]與[[華中|華中]]地區的戰役獲得勝利後察覺國民政府並未如原本預想的失去戰力求和，反而是將首都內遷冀求長期戰爭勝利，因此日本陸軍一方面將軍隊從16師擴張到22師45萬人，同時策劃新攻勢希望打通[[津浦線|津浦線]]（今[[京滬鐵路|京滬鐵路]]）將南北佔領區連成一塊，並強迫國府軍在無險可守的四戰之地[[徐州|徐州]]進行大規模決戰以殲滅當時內遷的[[蔣中正|蔣中正]]政府底下所屬戰力。

1938年1月1日，國民政府宣佈改組，提出要通過持久戰才能取得最後勝利，國防委員會成員由30人增加至75人，實行交通軍事管制，拒絕日本倡議和平<ref name="人">{{Cite_book|author=[[伊斯雷爾·愛潑斯坦|伊斯雷爾·愛潑斯坦]]|title=《人民之戰》|location=香港|publisher=和平圖書|year=2016}}</ref>{{rp|119}}。2月10日，林彪統率八路軍115師奪回冀中保定和石家莊之間4個車站，切斷平漢鐵路日軍後方，繳獲500支步槍、30挺輕機槍和100多匹馬<ref name="人" />{{rp|140}}。3月初，蔣親自在鄭州坐鎮，中共八路軍於晉西打退企圖於黃河南岸平漢鐵路渡河之日軍<ref name="人" />{{rp|144}}。

== 概要 ==
原先日軍的構想是在擴軍完成後的1938年夏季發動新一波攻勢，為此日本參謀本部在2月16日的御前會議時先向天皇匯報其作戰意圖，同時向前線部隊告知作戰方針並劃出防衛縱深圈，並派遣傾向不擴大現有事端的作戰科長[[河邊虎四郎|河邊虎四郎]]上校前往中國對底下部隊進行協調。3月，日軍參謀本部同意第二軍增加部分兵力，但須遵守不得超越[[臨沂|臨沂]]、[[棗莊|棗莊]]之線，前線日軍卻無視上級命令，第十師團擅自向南推進，從[[濟南|濟南]]發兵朝西南方的[[濟寧市|濟寧市]]進攻，第五師團則派出一個營的兵力朝濟南東南方的[[臨沂市|臨沂市]]助攻，在協調無用下河邊虎四郎遭到撤換<ref name="hata43-45">『太平洋戦争への道 第4巻　日中戦争 下』、43-45頁。</ref>，繼任的[[稻田正純|稻田正純]]下達了新的意向，同意第十師團殲滅[[山東|山東]]省[[大運河|大運河]]一線的國軍<ref name="masui167-168">益井、167-168頁。</ref><ref name="ochi84-86">越智、84-86頁。</ref>。

[[徐州|徐州]]在[[津浦铁路|津浦铁路]]（[[天津|天津]]—[[浦口|浦口]]）和[[陇海铁路|陇海铁路]]（[[兰州|兰州]]－[[连云港|连云港]]）交叉点，国民政府第五战区总部。

台兒莊位於徐州東北30公里的[[大運河|大運河]]北岸，与徐州东北的邳县（今[[邳州市|邳州市]]）接壤，扼守運河的咽喉。<ref>{{cite book|title=《[[良友|良友]]》畫報第一三七期|date=1938-05|quote=台兒莊是魯南部之一個小鎮，西北距嶧縣三十四公里，西南距隴海路之的運河四十三公里。水陸交通便利，全莊居民四千餘戶，頗為繁盛。南下運河口，可以截斷徐海一的鐵路，西北兩面和韓莊呼應，可以直逼徐州。所以敵將磯谷之必以主力出台兒莊，乃與坂垣之出臨沂，為同一之意義，即目的在確實控制魯南，免除後顧之憂﹐然後自魯南山地縱兵徐州隴海之間，迂迴側擊徐州，以完成其貫通津浦線之野心。|chapter=台兒莊大破敵軍}}</ref>。[[桂系|桂系]]將領[[第五戰區|第五戰區]]司令[[李宗仁|李宗仁]]與時任國軍副參謀總長兼軍訓部長的[[白崇禧|白崇禧]]合作指挥[[國民革命軍|國民革命軍]]将两路日军分别阻止在山东[[临沂|临沂]]和台儿庄。

由於日軍的作戰並未協調，這種以少量兵力孤軍深入的作法反而遭到國軍利用，第五戰區指揮官[[李宗仁|李宗仁]]看準時機調派了其手下的20%兵力到台兒莊周邊冀求一舉全殲南侵日軍，除了各派系的部隊以外，軍委會甚至抽調了大部分[[SFH_18榴彈砲|150榴]]與反戰車炮前去增強戰區火力，以事後來看這項作為讓國府軍取得了部分火力優勢。李宗仁遂以西北军[[孫連仲|孫連仲]]第二十六路军在徐州以北的台兒莊與日軍反復爭奪。

雖然國軍戰史中會以日軍師團長之名稱代稱進攻部隊，但日軍在戰役中並未將全部師團兵力投入，因此用這種做法並不精確，也無法反映實際日軍作戰規模。<br>
台兒莊戰役中日軍動用2支步兵旅（其中一支僅有半數兵力），另外還增派2支野戰炮兵團，2支山砲連，1支野戰重炮兵團，1支工兵營，2支輕裝甲車連等強化火力，考量連月作戰耗損，實際規模應在2萬人上下。

4月5日，湯恩伯第十三軍在台兒莊以北20英里之洪山鎮俘虜幾百名日本人並繳獲野戰炮<ref name="人" />{{rp|165}}。以中央军[[湯恩伯|湯恩伯]]军团卷击台儿庄日軍侧背，最終取得台兒莊戰役勝利，以毙伤日軍一万多人的戰果成爲近代日本第一次軍事挫敗<ref>{{Cite_book |authors=David Boyle著，李靜宜、張麗瓊、李俊融译 |title=《第二次世界大戰：戰火中的鏡頭》 |location=台北 |publisher=明天國際圖書 |ISBN=9867256433 |year=2005 }}</ref><ref>{{Cite_book |author=[[黃仁宇|黃仁宇]] |title=《從大歷史的角度讀蔣介石日記》 |location=台北 |publisher=時報出版 |year=1994 }}</ref>{{rp|202}}。中國老百姓給國軍當嚮導，照顧傷員，甚至在日軍炮火下也不離開部隊，許多人揀起死者槍支，參加戰鬥<ref name="人" />{{rp|151}}。{{quote|4月5日，[[蔣中正|蔣中正]]電令第二十軍團：<br />
台兒莊附近會戰，我以十師之眾，對一師半之敵，歷時旬餘，未獲戰果，該軍團居敵側背，態勢尤為有利，攻擊竟不奏功，其將何以自解，即應嚴督所部，於六七兩日，奮勉圖功，殲滅此敵，毋負厚望，究竟有無把握，仰即具報為要。<ref>《抗日戰爭正面戰場》上，頁608。</ref>

{{quote|李宗仁電軍令部：
……當面之敵約三聯隊以上，炮40餘門，戰車三四十輛，……今日復以戰車連續攻擊，炮火集中轟炸2000發以上，附近村落民房毀成平地，我陣地官兵亦多同歸於盡……。統計各部傷亡最為慘重：第27師僅餘戰鬥員約2000名，第31師餘1400名……}}
同日，坂本支隊得到直屬上級第五師團師部命令：「準備將部隊轉移至攻略沂州」。為何日軍在台兒莊尚未攻陷時下達這個命令，板垣徵四郎後來辯解的理由是說他以為台兒莊已經被攻下<ref name="masui169-170" />，但也承認「當時師團內部的情報極為混亂所以誤認」<ref name="ochi88-89">越智、88-89頁。</ref>。無論理由，在台兒莊周邊的日軍認為不用打下去了，因此坂本支隊長當天晚上和瀨谷支隊長連繫，表示明天（6日）將要反轉戰線後撤。這決定了隔天瀨谷支隊的命運。

4月6日，李宗仁赶到台儿庄附近亲自指挥部队，對第十师团发起了全线出击，日軍機動車輛多被擊毀，其餘也因缺乏[[汽油|汽油]]而陷於癱瘓。瀨谷支隊在台兒莊與嶧縣城周邊。
{{quote|4月6日，[[湯恩伯|湯恩伯]]呈覆[[蔣中正|蔣中正]]：
當面之敵經各軍奮勇猛擊，殲敵過半，已嚴令各軍迅向台兒莊攻擊前進，[[臨沂|臨沂]]西南中村附近之敵，似有南進模樣，……如敵不再增援，當可於今明兩日，將該敵壓迫於台兒莊以北附近地區殲滅之<ref>第二次中日戰爭各重要戰役史料彚編，台兒莊會戰，國史館，1984年7月，第62頁，轉引自中華民國國防部編，徐州會戰(三)</ref>。[[蔣中正|蔣中正]]令白崇禧李宗仁轉知會：張自忠部、龐炳勛部於向城臨沂等處游擊，阻斷日軍合圍台兒莊，防止敵軍會合。

在此同時，昨晚得知友軍撤退的瀨谷支隊長在「沒有接到攻佔台兒莊命令」、「沒有接到撤退命令」的情況下權衡後，也在4月6日日落開始準備撤退。

4月7日凌晨1时，国军發動反攻，以孙连仲第2集团军为主组成的左翼兵团和以汤恩伯第20军团为主组成的右翼兵团在台儿庄及其附近地区大举反攻。双方便展开了巷战、肉搏战，一时间，台儿庄城内枪林弹雨，血流成河。李宗仁命令部队猛追。日軍向峄城、枣庄撤退。至此台儿庄战役結束。

==戰後==
一批中国学者利用日方资料做出如下解读：1938年6月，日本華北方面軍參謀部第三課對臺兒莊戰役前後日軍傷亡的一個統計，第5、第10師團合計傷亡11984名<ref>{{cite book|title=《父親與民國——白崇禧將軍身影集上冊【戎馬生涯】》|author=白先勇編著|location=香港|publisher=[[天地圖書|天地圖書]]|date=2012-05|isbn=9789882198111|quote=台兒莊之役自三月二十三日日軍發動攻勢起，至四月七日晚，莊內日軍殘餘被掃清止，歷時半月。據李宗仁估計「敵軍總死傷當在二萬人以上」}}</ref>{{rp|126}}，其中第五師團2月20日至5月10日共戰死1281人， 受傷5478人，第10師團3月14日至5月12日戰死1088人， 受傷4137人 。日本冈山大学日本近代史教授[[姜克實|姜克实]]认为此说是中国学者对日文资料的断章取义，并认为“在1938年2月20日-4月7日间的“台儿庄会战”，即“台儿庄大捷”中的日本军的死伤者(包括在济宁，嘉祥扫荡的长濑支队)总数应为5800名。其大多数不是死伤于台儿庄，而是东部的临沂战场”<ref name=":0">{{Cite web|url=http://www.aisixiang.com/data/89218.html|title=台儿庄战役日军死伤者数考|accessdate=2016-07-09|author=姜克实|date=2015-06-12|publisher=}}</ref>。

根據日軍第63步兵聯隊第一手戰鬥詳報，負責台兒莊主攻的日軍第63步兵聯隊連同配合作戰的後勤與支援單位在內，共計有8,464人參與台兒莊之戰，其中陣亡346人，受傷1,154人，死傷總數為1,500人<ref>「JACAR(アジア歴史資料センター)Ref.C11111254400、歩兵第６３連隊　台児庄攻略戦闘詳報　昭和１３年３月２日～昭和１３年４月６日（２分冊の１）(防衛省防衛研究所)」</ref>。

#長期抗戰的主要著眼點在於消耗敵軍戰力，而獲得最後勝利。須深知不在一城一市之得失；避免對持久抗戰心理發生不良影響。

*日本[[岡山大學|冈山大学]]日本史教授[[姜克实|姜克实]]认为：“从中国军队在1938年5月16日放弃了战略要地徐州这点上看，‘台儿庄大捷’并没有多大战略上胜利的意义。只不过把徐州的陷落时间推迟了二个月之久”，“也就是说‘台儿庄大捷’的意义不一定在军事面，而在于政治面，精神面和宣传面之中。其打破了‘皇军百战百胜’的旧观念，给予了中国民众和抗战部队奋勇抗日的精神力量和勇气。对于一直节节败北于日军攻势的中国军来讲，在这一层面多少夸张一点实事，多宣传一点胜利，多创造出几个像王铭章一样的英雄人物，从精神战这一角度上看还是有必要的”。<ref name=":0" />

== 宣传与纪念 ==
1980年前中国大陆在教科书与传媒中极少提及台儿庄战役。例如1960年上海教育出版社《中学历史教师手册》和1979年徐州师范学院历史系所编教学参考用《世界历史大事纪年》就抗日战争总体提的大部分都是共产党的活动介绍，1938年条目前下和国民党相关的只有“日军占领广州、武汉”，在[[八一三事变|八一三事变]]后唯一提到的国军战役是[[豫湘桂战役|豫湘桂战役]]。80年代後有所改變，中国大陆电影《[[血战台儿庄|血战台儿庄]]》展现了台儿庄战役中奋力抵抗的国军将士。

現今，在中華人民共和國通用教科書介紹臺兒莊大捷，宣稱是後來投共之[[李宗仁|李宗仁]]在抗战初期正面战场取得的一次重大胜利。<ref>中國大陸 八年级第二学期课本 在第五战区司令长官李宗仁指挥下，中国军队将敌军一路阻止在临析，使另一路敌军孤军深入。日军猛攻台儿庄，庄内守军凭借断墙惨壁英勇抵抗，拖住日军。4月初，中国军队发动全线反攻，日军伤亡惨重，向北溃逃。台儿庄战役歼敌1.2万人，是抗战初期正面战场取得的一次重大胜利。上海教育出版社出版 ISBN 978-7-5320-5131-1</ref>台儿庄大捷75週年時，當地舉行隆重紀念活動。<ref>[http://ent.163.com/13/0307/20/8PD19FSN00032DGD.html 台儿庄大战胜利75周年纪念活动启动]</ref>
