'''台灣加油隊'''源自於[[澳洲|澳洲]]球迷組織[[狂人團|狂人團]]，當時狂人團在[[1997年美國網球公開賽|1997年美國網球公開賽]]為澳洲著名選手[[帕特里克·拉夫特|-{zh-hans:拉夫特;zh-hk:拉夫達;zh-tw:拉夫特;}-]]加油，後來果真讓拉夫特拿下那屆[[美網|美網]]冠軍，後來在澳洲唸書的台灣人[[李昆霖|李昆霖]]，1999年，獲得「狂人團」的隊服：2000年[[2000年夏季奧林匹克運動會|雪梨奧運]]，李昆霖看到「狂人團」為澳洲加油的情況，他頗有感觸，便想創造一個仿效「狂人團」的台灣加油隊{{参考|name=第三節 他鄉遇故知─海外僑胞與學生的支持 |1={{Cite book|author=盧彥勳 |title=《盧彥勳的堅持》〈第五章 為我豐富生命色彩的貴人 〉 |year=2010年 |pages=第196頁–199頁}}}}。2006年初起與[[楊蕙如|楊蕙如]]共同發起成立台灣加油隊並擔任隊長，在世界各地號召球迷在運動場上為台灣運動選手加油，並四處發送[[中華民國|中華民國]]國旗。{{参考|1={{cite web |url=http://blog.udn.com/candy5661/256969 |publisher=民生報 |title=台灣加油隊 讓世界看見台灣 |accessdate=2011-02-28}}}}

== 成員 ==
台灣加油隊的年齡層，其成員年紀從20歲到70歲，他們來自各行各業，有國小教師、業務工程師、系統工程師，也有公務人員{{参考|1={{cite web |url=http://tw.nextmedia.com/applenews/article/art_id/3268151/IssueID/20070221 |publisher=蘋果日報 |title=台灣加油團 見證光榮 |accessdate=2011-02-28}}}}。
