'''社團法人台灣維基媒體協會'''（{{lang-en|Wikimedia Taiwan}}）簡稱'''台灣維基媒體協會'''、'''維基台灣分會'''<ref name = "王則文 01">{{Cite web |url = https://web.alcd.tw/uploads/2017/12/14/3a4f303013921268ad78f50d743d9b80.pdf |title = 維基協會十週年慶 原住民族語維基列發展重點 |author = 王則文 |date = 2017年12月14日 |publisher = 《原教界》 |language = Zh-hant |accessdate = 2018年3月27日 |quote =  |deadurl = no |archiveurl = https://web.archive.org/web/20180329112125/https://web.alcd.tw/uploads/2017/12/14/3a4f303013921268ad78f50d743d9b80.pdf |archivedate = 2018年3月29日 }}</ref>，原名'''中華民國維基媒體協會'''，是[[中華民國|中華民國]]（[[臺灣|臺灣]]）的非營利全國性社會團體。台灣維基媒體協會為經[[美國|美國]][[維基媒體基金會|維基媒體基金會]]授權在臺灣的[[維基媒體分會列表|維基媒體分會]]組織，負責推廣如[[維基百科|維基百科]]、[[維基新聞|維基新聞]]等各項維基媒體基金會所屬的[[維基媒體計劃|維基媒體計劃]]，及推動相關的合作專案<ref name = "Accupass 活動通 01">{{Cite web |url = https://old.accupass.com/org/detail/r/1408130908591926449990/4/0 |title = 中華民國維基媒體協會 |author = 台灣維基媒體協會 |date =  |publisher = Accupass 活動通 |language = Zh-hant |accessdate = 2018年3月27日 |quote =  |deadurl = no |archiveurl = https://web.archive.org/web/20180329112125/https://old.accupass.com/org/detail/r/1408130908591926449990/4/0 |archivedate = 2018年3月29日 }}</ref>。

2006年，臺灣在地的維基媒體使用者開始進行組織籌備程序，最早共有35位創始會員參與。2007年，中華民國維基媒體協會於[[臺北市|臺北市]]註冊，報經[[中華民國內政部|中華民國內政部]]立案成立，在同年7月4日正式獲得維基媒體基金會的在地分會認可。台灣維基媒體協會的會址曾先後設在[[新北市|新北市]][[淡水區|淡水區]]、臺北市[[萬華區|萬華區]]，於2017年11月遷至臺北市[[大安區_(臺北市)|大安區]]。從2018年1月起，台灣維基媒體協會分別由[[張遠_(維基人)|張遠]]及[[陳瑞霖|陳瑞霖]]出任理事長和監事長，底下並有2名雇員。

台灣維基媒體協會也是使用者與維基百科交流的重要管道，大多數剛開始想要編輯維基百科的新使用者，最常遇到的案例是不熟悉編輯方法，並因為其內容或用字遣詞的表達錯誤而[[維基百科的刪除主義和保留主義|遭到刪除]]<ref name = "善耕365基金會 01"/>；台灣維基媒體協會便能在編輯時提供協助、把關，從而減少錯誤，這除了確保[[維基百科的關注度|重要的知識]]不會遭維基百科刪除，同時也能協助使用者正確的編輯<ref name = "善耕365基金會 01"/>。台灣維基媒體協會還經常收到覺得自己的條目遭到亂改的[[對維基百科的批評|批評]]，但依照由社群撰寫內容的方針，協會只能幫助抱怨者找到撰寫的網友溝通<ref name = "陳瑞霖 01"/>，或協助讓該條目轉為僅開放[[管理員_(維基百科)|管理員]]編輯、或封禁特定帳號以防惡意[[維基百科的破壞|破壞]]<ref>{{Cite web |url = https://tw.appledaily.com/headline/daily/20140803/35998362/ |title = 林杰樑逝周年 維基遭惡搞 |author = 邱俊吉 |date = 2014年8月3日 |publisher = 《[[蘋果日報_(台灣)|蘋果日報]]》 |language = Zh-hant |accessdate = 2018年3月26日 |quote =  |deadurl = no |archiveurl = https://web.archive.org/web/20180329112129/https://tw.appledaily.com/headline/daily/20140803/35998362/ |archivedate = 2018年3月29日 }}</ref>。

台灣維基媒體協會是經美國維基媒體基金會授權，設在臺灣的維基媒體地方分會組織<ref name = "Accupass 活動通 01"/><ref name = "KKTIX 01"/><ref name = "維基媒體基金會 01"/>，也是維基媒體基金會在全世界正式承認的40個地區分會之一<ref name = "羅佳旼 01"/><ref name = "Reke 01"/>。而依照理事會任期，台灣維基媒體協會定期與維基媒體基金會簽訂分會協議書<ref name = "章程"/>。台灣維基媒體協會還是報經主管機關中華民國內政部立案，依法成立的公益性社團法人<ref name = "陳瑞霖 01"/>，屬於非營利性質、全國性的社會團體<ref name = "亞太多元學習協會 01"/><ref name = "羅佳旼 01"/>。在此，台灣維基媒體協會將中華民國[[中華民國行政區劃|全國行政區域]]作為組織區域，其目的事業也受到事業主管機關的指導與監督<ref name = "章程"/>。

台灣維基媒體協會的最高權力機構為[[會員大會|會員大會]]，並且分別設置有[[理事會|理事會]]（共9名理事）與[[監事會|監事會]]（共3名監事）<ref name = "章程"/>。其中，會員大會的職權包括：訂定與變更章程；選舉及罷免理事、監事；議決入會費、常年會費、事業費、會員捐款面額及方式；議決年度工作計畫、報告、預算及決算；議決會員代表除名處分；議決財產處分；議決解散協會；議決與會員權利義務有關的其他重大事項<ref name = "章程"/>。理事會和監事會則分別由會員代表選舉而成，而當新任理事會選出的6個月內，仍未與維基媒體基金會簽訂協議書時，台灣維基媒體協會則宣告解散<ref name = "章程"/>。在經理事會通過後，台灣維基媒體協會也能成立各種委員會、小組或其他內部作業組織<ref name = "章程"/>。

理事會設有經過理事互選而產生的[[理事長|理事長]]，對內綜理督導會務，對外則是代表台灣維基媒體協會，並擔任會員大會、理事會主席<ref name = "章程"/>，現任理事長為[[張遠_(維基人)|張遠]]<ref name = "台灣維基媒體協會 03"/><ref name = "台灣維基媒體協會 03">{{Cite web |url = https://mailchi.mp/b7f5175947f6/1072 |title = 中華民國維基媒體協會會訊（2018年02月號） |author = 王則文和上官良治 |date = 2018年8月 |publisher = 台灣維基媒體協會 |language = Zh-hant |accessdate = 2018年3月26日 |quote =  }}</ref><ref name = "Organization">{{Cite web |url = https://meta.wikimedia.org/wiki/Wikimedia_Taiwan/Organization |title = Wikimedia Taiwan/Organization |author = 台灣維基媒體協會 |date = 2018年1月15日 |publisher = [[元維基|元維基]] |language = Zh-hant |accessdate = 2018年3月26日 |quote =  }}</ref>。而根據法律，監事會負責監督組織內部的運作，現任監事長是[[陳瑞霖|陳瑞霖]]<ref name = "台灣維基媒體協會 03"/><ref name = "Organization"/>。另外還設有1名[[秘書長|秘書長]]，接受理事長的命令，處理台灣維基媒體協會事務<ref name = "章程"/>，現任秘書長為[[王則文|王則文]]<ref name = "Organization"/>。其他工作人員則是由理事長提名、經理事會通過聘免，並報主管機關備查<ref name = "章程"/>。此外，台灣維基媒體協會還能經由理事會，聘請1名名譽理事長，及數名名譽理事或顧問，其聘期與理事、監事的任期相同<ref name = "章程"/>。

=== 互動模式 ===
[[File:Wikimedia_Taiwan_Education_Program_workshop_3.jpg|thumb]]（Nichole Saad）訪-{問}-台灣維基媒體協會。]]

在維基社群中，維基媒體基金會、分會與社群志工彼此屬於夥伴關係，這種機制鼓勵維基人互相監督、勉勵、分享與貢獻，也是維基公信力和發展遍及世界各處的最主要原因<ref name = "張家豪 01"/>。作為在臺灣協助維基百科發展的維基媒體組織，台灣維基媒體協會的會員主要是由充滿知識維護熱忱、且贊同協會工作與宗旨的臺灣維基人組成<ref name = "羅佳旼 01"/>，不過實際的會員編輯群並不龐大<ref name = "善耕365基金會 01"/>。台灣維基媒體協會的會員申請資格，依照章程又分成個人會員、團體會員、贊助會員與永久會員，並擁有遵守章程、決議及繳納會費之義務<ref name = "章程"/>。

在互動模式部分，維基媒體基金會、分會與社群之間有著明顯的分權平行，關係上彼此互不干涉，但是各方在必要情況下也都能夠互相合作<ref name = "張家豪 01"/>。其中，維基媒體基金會只負責不斷發想各種專案、及維持網站的順利運作，而分會、地方社群則可自行發想計畫，並向維基媒體基金會請求資源或是諮詢<ref name = "張家豪 01"/>；例如台灣維基媒體協會過去便曾申請過數次經費，分別用於教材、女性維基社群的活動、以及[[維基百科亞洲月|維基百科亞洲月]]上<ref>{{Cite web |url = https://g0v.news/社群報導-他山之石-用獎助金創造公民科技蝴蝶效應-3c8d49e60535 |title = 【社群新聞】他山之石：用獎助金創造公民科技蝴蝶效應 |author = 簡韻真 |date = 2017年1月19日 |publisher = [[g0v零時政府|g0v零時政府]] |language = Zh-hant |accessdate = 2018年3月26日 |quote =  }}</ref>。而分會與地方社群關係亦然<ref name = "張家豪 01"/>。

相對而言，各地的社群聚會被視為是維基的根本基礎，協會是為了社群而存在的<ref name = "張家豪 01"/>；也因此，台灣維基媒體協會還有擔當集結本地社群意見的工作<ref name = "陳瑞霖 01"/>。除了舉行定期的在地聚會外，台灣維基媒體協會還會透過國際會議，關心各國維基媒體協會彼此的狀況，並與同樣願意為知識貢獻的成員交流<ref name = "善耕365基金會 01"/>。例如在2013年抗議封鎖境外盜版網站期間，台灣維基媒體協會便曾設想通報給維基媒體基金會，藉此引起世界各地方的維基分會關注<ref name = "ETtoday新聞雲 01"/>，對臺灣形成國際輿論壓力<ref name = "鄭景雯 01"/><ref name = "呂炯昌 01"/>。

=== 名稱與會址 ===

台灣維基媒體協會的全名是「社團法人台灣維基媒體協會」，而其過去的全名則是「社團法人中華民國維基媒體協會」<ref name = "台灣維基媒體協會 02">{{Cite web |url = https://mailchi.mp/7fa6b1753fed/1071 |title = 中華民國維基媒體協會會訊（2018年01月號） |author = 王則文和上官良治 |date = 2018年1月 |publisher = 台灣維基媒體協會 |language = Zh-hant |accessdate = 2018年3月26日 |quote =  }}</ref>。台灣維基媒體協會在成立之初，便透過在中文維基百科上的籌備頁面進行討論，最終決定採用「社團法人中華民國維基媒體協會」的稱呼，而「中華民國維基媒體協會」這個名稱爾後就使用長達10多年<ref name = "台灣維基媒體協會 02"/>。

然而由於台灣維基媒體協會的國際聯繫事務大幅增加，[[漢語|漢語]]名稱「中華民國維基媒體協會」與[[英語|英語]]名稱「Wikimedia Taiwan」難以對應的問題，便逐漸被提出討論<ref name = "台灣維基媒體協會 02"/>。受限於可能產生的行政成本考量，台灣維基媒體協會遲遲未針對更名案進行主動提案<ref name = "台灣維基媒體協會 02"/>。直到2017年12月24日，台灣維基媒體協會在第4屆第1次會員大會經由臨時動議，以壓倒性票數通過協會更名案，從而更改為「台灣維基媒體協會」<ref name = "台灣維基媒體協會 02"/>。

而根據《台灣維基媒體協會章程》，台灣維基媒體協會會址應設於主管機關所在的地區<ref name = "章程"/>。台灣維基媒體協會會址最早是設在[[新北市|新北市]][[淡水區|淡水區]]，並在2014年進駐到位於臺北市[[萬華區|萬華區]][[西門町|西門町]]的共同工作空間「[[卡市達創業加油站|卡市達創業加油站]]武昌起義」<ref name = "亞太多元學習協會 01"/><ref>{{Cite web |url = https://www.inside.com.tw/2016/09/23/custard-cream |title = 【台北共同空間巡遊】卡市達加油站：將鑒於歐洲經驗，啟動共同空間轉型！ |author = Chris |date = 2016年9月23日 |publisher = INSIDE |language = Zh-hant |accessdate = 2018年3月26日 |quote =  |deadurl = no |archiveurl = https://web.archive.org/web/20161106223413/http://www.inside.com.tw/2016/09/23/custard-cream |archivedate = 2016年11月6日 }}</ref><ref name = "台灣維基媒體協會 01">{{Cite web |url = https://mailchi.mp/66d7ebe4de6a/10610 |title = 中華民國維基媒體協會會訊（2017年10月號） |author = 王則文、上官良治 |date = 2017年10月 |publisher = 台灣維基媒體協會 |language = Zh-hant |accessdate = 2018年3月26日 |quote =  }}</ref>。爾後，台灣維基媒體協會向卡市達創業加油站武昌起義承租2個辦公桌位、以及工商登記權<ref name = "台灣維基媒體協會 01"/>。到了2017年中，卡市達創業加油站武昌起義歇業。2017年11月，台灣維基媒體協會將辦公室遷至國立臺灣大學後門、[[臺北捷運|臺北捷運]][[科技大樓站|科技大樓站]]附近的「滾出趣人生旅所」，與多個原卡市達武昌起義進駐團隊繼續共享辦公空間<ref name = "台灣維基媒體協會 01"/>。

== 計畫 ==
=== 各式專案 ===
[[File:維基交流─Dr._James_(24490885541).jpg|thumb]]前來臺灣訪問，台灣維基媒體協會成員與其交流意見。]]

台灣維基媒體協會所構思的「台灣知識種子計畫」維基推廣計畫，是希望能夠擴大維基寫作領域，並提升維基條目[[維基百科的可靠性|質量]]<ref name = "張家豪 01"/>；當中除了廣為招募、培訓成員，以增加臺灣維基志工數量外，同時藉由知識分享的風氣，期望能夠增添與在地議題連結、及對臺灣發展有所幫助的關鍵知識，並將這些內容寫入維基媒體的各項計畫<ref name = "張家豪 01"/>，形成改變社會的能量<ref name = "均優學習 01"/>。台灣知識種子計畫的執行方式，則是由台灣維基媒體協會整合資源、提供諮詢，並且結合臺灣在地的維基社群，號召大專院校學生為主要寫手，來編輯各式條目<ref name = "張家豪 01"/>。

== 財務 ==
[[File:Post_Script_the_Wikimania_puzzle_globe_(1119900855).jpg|thumb]]

根據《台灣維基媒體協會章程》，台灣維基媒體協會的經費來源包括：常年會費、事業費、會員捐款、委託收益、基金與其利息等收入<ref name = "章程"/>。2015年，台灣維基媒體協會共有新臺幣近60萬元的收入，來源包含本地捐款（新臺幣423,017元，70.66%）、專案執行收入（新臺幣151,428元，25.29%）{{noteTag|在維基百科的募款廣告會由維基媒體基金會收取，再依硬體設備、維基媒體基金會人員薪資、各地分會的專案分配<ref name = "陳瑞霖 01"/>。由於各地維基組織社群的動員能量及組織經驗不一，所以各地維基媒體協會所分配、執行的資源也不同，除了位於舊金山的維基媒體基金會總部之外，經費執行最多者是德國、再來是法國及英國維基媒體協會<ref name = "廖漢騰 01"/>，因此捐款並非按各地區比例給予<ref name = "陳瑞霖 01"/>。}}、會費收入（新臺幣24,000元，4.01%）、利息及其他收入（新臺幣260元，0.04%）<ref name = "陳瑞霖 01"/>。在這當中，募款是台灣維基媒體協會的重要工作之一<ref name = "CNET 01"/>。

台灣維基媒體協會每年會在會計年度開始前2個月，由理事會編列年度工作計畫、收支預算表、員工待遇表，並提請提會員大會通過（當會員大會因故未能如期召開時，則會先提交給理監事聯席會議通過），在會計年度開始前提交給主管機關核備<ref name = "章程"/>。在會計年度結束後的2個月內，理事會也會編列年度工作報告、收支決算表、現金出納表、資產負債表、財產目錄及基金收支表，經過監事會審核後，提出審核意見書交給理事會，並提請會員大會通過，在3月底前報主管機關核備（當會員大會未能如期召開時，則會先呈報給主管機關）<ref name = "章程"/>。
