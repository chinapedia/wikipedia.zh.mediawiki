'''左心室'''是[[人類|人類]][[心臟|心臟]]四個[[心室|心室]]之一，而四心室則包括兩個[[心房|心房]]和兩個[[心室_(心臟結構)|心室]]。它會接收來自[[左心房|左心房]]的[[動脈血|動脈血]]，再把之泵入[[主動脈|主動脈]]以把含氧血供應全身。在此途中，含氧血會經過兩個[[瓣|活瓣]]，一是位於左心房和左心室之間的[[二尖瓣|二尖瓣]]，另一個就是位於大動脈的[[大動脈瓣|大動脈瓣]]，它們都用以防止[[血液|血液]]倒流。

== 形狀 ==
與[[右心室|右心室]]比之，左心室的形狀是更長，亦更像圓錐。其[[橫切面|橫切-{面}-]]也呈現一個[[橢圓形|橢圓形]]或接近[[圓形|圓形]]的輪廓。[[胸肋面|胸肋面]]，即心臟的前面，主要由右心室組成，亦有小部份是由左心室組成。然而，左心室不但組成了心尖，也組成心的隔面的大部份，而隔面就是器官與[[橫膈膜|橫膈膜]]接觸的那一面。為了以一個高的[[血壓|血壓]]把血液泵出，左心室的[[肌肉|肌肉]]比右心室的更厚，也更發達。

== 發展 ==
由青少年的年紀開始，左心室的肌壁會變厚，使之比右心室的厚三至六倍。這亦反映左心室在泵血過程中會造出比右面多五倍的壓力。在每一次[[心跳|心跳]]中，左心室會接受來自肺靜脈大約處於80[[托|托]]（相當於接近11千[[帕斯卡|帕斯卡]]的壓力）的血，接著把血壓推高至大約120托（相當於16.3千帕斯卡）以將之送入大動脈。（所有壓力值是在心臟處於休息狀態下量度，亦為與被訂為零的參考壓力的周邊大氣層壓力相對的相對值。）

== 功能 ==
一個正常的心臟，其左心室的肌肉一定可以︰

# 在每一次收縮後瞬速放鬆以讓來自肺靜脈的含氧血快速地填充，此即心臟[[舒張期|舒張期]]的放鬆與填充。
# 瞬速的激烈收縮以推動大量的血液進入大動脈，克服非常高的大動脈壓力及提供額外的壓力以延伸大動脈和其他的主要動脈且為突然上升的血量提供足夠的空間，此即心臟[[收縮期|收縮期]]的收縮和外排。
# 在中樞系統控制下，瞬速地增加或降低其抽泵能力。

== 抽泵容量 ==
一個正常成人心臟在處於靜息狀態下可以每分鐘大約泵出五升血液。最大的輸出能力可以由非運動員的大約每分鐘二十五升至達[[奧林匹克|奧林匹克]]水平的運動員的大約每分鐘四十五升。

== 左心室肥大 ==
{{Main|左心室肥大}}

[[左心室肥大|左心室肥大]]本身並非一種[[疾病|疾病]]，但往往是[[心臟病|心臟病]]的先兆。<ref name="pmid18030317">{{cite journal |author=Meijs MF, Bots ML, Vonken EJ, ''et al'' |title=Rationale and design of the SMART Heart study: A prediction model for left ventricular hypertrophy in hypertension (在高血壓中左心室肥大的預計模型) |journal=Neth Heart J |volume=15 |issue=9 |pages=295–8 |year=2007 |pmid=18030317 |doi=}}</ref>左心室肥大可以是一種心肌對[[有氧運動|有氧運動]]和[[力量訓練|力量訓練]]的自然反應，也會是對[[心血管疾病|心血管疾病]]和[[高血壓|高血壓]]的病理反應，不過更可以由增加心臟[[後負荷|後負荷]]或心肌的疾病引起。<ref name="titleAsk the doctor: Left Ventricular Hypertrophy">{{cite web |url=http://www.clevelandclinic.org/heartcenter/pub/guide/askdoctor/lvh.htm |title=Ask the doctor: Left Ventricular Hypertrophy (左心室肥大)|accessdate=2007-12-07 |format= |work=}}</ref>當然，無論如何這種擴大並非永久。在一些個案中，這種增生會隨著血壓的下降而回復。<ref name="pmid16627048">{{cite journal |author=Gradman AH, Alfayoumi F |title=From left ventricular hypertrophy to congestive heart failure: management of hypertensive heart disease (從左心室肥大到充血性心臟衰竭︰管理高血壓疾病)|journal=Prog Cardiovasc Dis |volume=48 |issue=5 |pages=326–41 |year=2006 |pmid=16627048 |doi=10.1016/j.pcad.2006.02.001}}</ref>
