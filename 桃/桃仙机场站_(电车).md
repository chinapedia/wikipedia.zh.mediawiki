'''桃仙机场站'''位于[[辽宁省|辽宁省]][[沈阳市|沈阳市]][[浑南区|浑南区]]，是[[沈阳有轨电车|沈阳有轨电车]][[沈阳有轨电车2号线|2号线]]的终点站，车站于2013年8月15日正式投入运营，车站主要服务于往来[[沈阳桃仙国际机场|沈阳桃仙国际机场]]的乘客。桃仙机场站是中国大陆机场联络轨道交通中唯一一座有轨电车车站。有轨电车开通后，可从沈阳市内开车或乘地铁到奥体中心站下车，换乘有轨电车2号线到达桃仙机场，整条线路长14.8公里，按照平均23公里/小时的时速，从奥体中心到桃仙机场只需39分钟。

== 车站布局 ==
本站为单线双侧式月台（西班牙式月台）设计。列车进站后开启右侧车门落客，清客后司机进行换端操作，再开启左侧车门上客。

== 车站周边 ==
桃仙机场3号航站楼、停车场

== 争议 ==
由于计划中的沈阳地铁2号线南延伸段（全运路-桃仙机场）迟迟未能开工建设，乘客如乘搭轨道交通进入市区，只能乘坐有轨电车，加上有轨电车2号线与1号线存在大量共线区间，列车运行间隔较大，故大量乘客仍然选择乘搭机场巴士以及乘坐其他车辆离开机场。
