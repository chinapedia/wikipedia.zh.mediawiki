== 背景 ==

桃力民是鄂托克旗[[木凯淖尔镇|木凯淖尔镇]]的一个[[自然村|自然村]]，桃力民来源于[[蒙古语|蒙古语]]“查汉桃日莫（或察干桃日莫<ref name="a2"/>）”，意为“向阳的白泥壕<ref>{{Cite web |url=http://www.eq.gov.cn/qq_etkq_0009/mkn/mkn_cqjs/201112/t20111206_531462.html |title=木凯淖尔镇桃力民村概况 |accessdate=2018-04-02 |author= |date=2011-12-06 |work= |publisher=鄂尔多斯木凯淖尔党政网 |language=Zh-hans |archive-url=https://web.archive.org/web/20180402101410/http://www.eq.gov.cn/qq_etkq_0009/mkn/mkn_cqjs/201112/t20111206_531462.html |archive-date=2018-04-02 |dead-url=yes }}</ref>”。桃力民及附近地区属[[伊克昭盟|伊克昭盟]]管辖。[[抗日战争|抗日战争]]开始后，成为抗日前沿地带。当地[[蒙古王公|蒙古王公]]或投靠日本，或态度暧昧<ref name="a2"/>。[[中国共产党|中国共产党]]亦积极争取此地。1938年6月，中国共产党绥蒙工作委员率部进驻桃力民地区的克泊尔庙（在今木凯淖尔镇大克泊尔村）。中共绥蒙工委以八路军绥蒙游击司令部政治部名义对外展开工作，主任[[白如冰|白如冰]]，并创建[[桃力民抗日根据地|桃力民抗日根据地]]。1939年11月，在[[大青山_(内蒙古)|大青山]]建立[[中共绥远省委|中共绥远省委]]，中共绥蒙工委撤消。其后，在桃力民地区的乌素加汗村建立中共伊盟工委，受[[陕甘宁边区|陕甘宁边区]]党委领导，对外称八路军联络参谋处。而随着[[国共摩擦|国共摩擦]]加剧，八路军于1941年撤出桃力民地区，并撤消中共伊盟工委<ref>{{Cite web |url=http://inews.nmgnews.com.cn/system/2015/09/01/011761328.shtml |title=草原深处的抗战记忆——鄂托克旗桃力民地区的抗日活动 |accessdate=2018-04-02 |author=仁庆道尔吉|date=2015-09-01 |work=责任编辑：魏莉 |publisher=内蒙古新闻网，来源：鄂尔多斯日报|language=Zh-hans }}</ref>。

== 桃力民办事室 ==

当时，为了更好的管理地方及民众，[[国民政府|国民政府]]对[[伊克昭盟|伊克昭盟]]部分蒙旗实行[[蒙汉分治|蒙汉分治]]。1941年春，[[绥远省政府|绥远省政府]]在东胜组建第三行政专员公署，负责管理伊克昭盟。专员公署下辖七旗一县，及耳字壕民众组训处、桃力民办事室<ref name="a2"/>。

桃力民办事室负责人刘实朴，驻地姆呼尔梢村。管辖地区，包括今杭锦旗南部和东部的亚斯图、四十里梁、胜利、阿门其日格、特然高勒等乡镇（苏木），北部的杭锦淖、[[独贵特拉镇|独贵特拉]]、永胜、吉尔格朗图、[[巴拉贡镇|巴拉贡]]、[[呼和木独镇|呼和木独]]、建设等乡镇，鄂托克旗东部和北部的汉族聚居开垦的粮地（或称明地）地区，包括今公其日格、早梢、沙井等乡及木肯淖部分地区。最初划分三个区，以[[保甲制|保甲制]]管理<ref name="a2"/>。

== 建立及废置 ==

1943年，绥远省政府将桃力民办事室升级为县级编制的行政机构——桃力民办事处。当时，桃力民办事处夹在杭锦旗、鄂托克旗、[[乌审旗|乌审旗]]、[[郡王旗|郡王旗]]、[[东胜县|东胜县]]和[[达拉特旗|达拉特旗]]之间，将诸旗县全部分隔。而桃力民办事处所辖地区，又将杭锦旗黑地（未开垦的蒙地）包围<ref name="a2"/>。除了动员汉族民众参加抗日战争的需要外<ref name="a1"/>，有认为绥远省政府的划分与杭锦旗末代[[扎萨克|扎萨克]][[阿拉坦敖其尔|阿拉坦敖其尔]]投靠[[日伪|日伪]]有关<ref name="a2"/>。

桃力民办事处首任处长傅观澜，是当时绥远省主席[[傅作义|傅作义]]的亲信。办事处下设五科：民政、财政、建设、教育、军事，一个会计室（实即办公室）。另有警察局、法院、监狱<ref name="a2"/>。

办事处原管辖的三个区，后改为十个[[乡|乡]]。十个乡，分别是杭锦旗的慧民乡、慧成乡、慧德乡，梁外地区的民泰乡、民乐乡、民兴乡、民立乡、民和乡、民福乡和隆兴乡，并设立[[乡公所|乡公所]]<ref>{{Cite web |url=http://www.shuofang.co/portal.php?mod=view&aid=7209 |title=巴拉亥“独贵龙”运动 |accessdate=2018-04-02 |author=崔永峰 |date=2016-06-12 |work= |publisher=朔方网——杭锦旗新闻地情网，来源：《杭锦简史》|language=Zh-hans }}</ref>。1945年，抗日战争胜利后，又合并为七个乡。1949年时，所辖七乡{{noteTag|1=李中林的原文遗漏一个乡。}}：惠民乡、隆兴乡、民乐乡、民立乡、民和乡、民泰乡<ref name="a1"/>。共辖三十三个保，一百九十六个甲，共计三千一百六十六户。驻地仍是姆呼尔梢村<ref name="a2"/>。1948年4、5月间，傅观澜奉绥远省保安司令部委任，组建桃力民自卫一团、二团。同年，王培仁任接任办事处处长<ref>{{Cite web |url=http://www.shuofang.co/portal.php?mod=view&aid=5263 |title=国民党桃力民自卫一团第一大队反抗整编事件 |accessdate=2018-04-02 |author= |date=2015-03-09  |work= |publisher=朔方网——杭锦旗新闻地情网，选自《杭锦文史》第一辑|language=Zh-hans }}</ref>。办事处机构规模类似于[[乙级县|乙级县]]，1949年1月、6月有过两次精简。至12月时，内设一室二科（秘书室、政务科、军事科），另有财粮等附属机构，以及司法处、警察局<ref name="a1"/>。

1949年9月，绥远省主席[[董其武|董其武]]投降中共，宣布绥远省起义，即[[九·一九起义|九·一九起义]]。桃力民办事处奉命等待[[中国人民解放军|解放军]]接收<ref name="a2"/>。1949年末，仍为[[綏遠省_(中華人民共和國)|绥远省]]伊克昭盟管辖。12月21日，[[绥远省军政委员会|绥远省军政委员会]]发出《关于解决乌伊两盟问题纪要》，其中决定取消桃力民办事处和[[达拉特旗组训处|达拉特旗组训处]]两个特殊的县级行政区，将桃力民办事处辖区划回杭棉旗和鄂托克旗<ref>{{Cite web |url=http://ordos.nmqq.gov.cn/sqgl/jzqh/2015-09-15/1243.html |title=伊克昭盟当代行政区划调整 |accessdate=2018-04-02 |author= |date=2015-09-15 |work= |publisher=鄂尔多斯市地情网，来源：《内蒙古自治区志·行政区域建制志》|language=Zh-hans }}</ref>。27日，绥远省军政委员会成立大会上，副主席[[高克林|高克林]]在讲话中明确了绥远省军政委员会管辖的行政区，确定取消桃力民办事处、达拉特旗组训练处<ref name="a1"/>。

1950年3月8日，绥远省人民政府设立桃力民中心区人民政府。王得胜为中心区人民政府代理主任。王得胜等人通过解放军的协助，在第二次接收时，完成了对桃力民办事处的接收工作。原办事处长王培仁已在此前出走。3月18日，桃力民中心区人民政府正式办公。至5月4日，完成各乡交接手续，任命新的[[乡长|乡长]]，而大部分职员自愿留用。不久，辖下的惠民乡改属杭锦旗。其余的乡改为区，以数字命名。同时，废除[[保甲制|保甲制]]，改保为[[行政村|行政村]]，甲为[[自然村|自然村]]。1951年5月4日，[[绥远省人民政府|绥远省人民政府]]撤消桃力民中心区人民政府，辖区归属原旗<ref name="a2"/>。
