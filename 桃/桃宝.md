'''桃宝'''（{{bd|1993年|1月1日|}}），本名'''黄妃'''，北京人，中国大陆[[配音員|配音员]]，[[模特兒|模特]]。2011年参与网络动画《[[我叫MT|我叫MT]]》的配音工作，曾担任[[中国网络电视台|CNTV]]《第一游戏》的主持人<ref>{{Cite web|url = http://games.cntv.cn/2011/fg_mc_0831/45183.shtml|title = 90后萌脸萌音妹子——桃宝|accessdate = 2015-08-03|author = |date = 2011年8月31日|publisher = 中国网络电视台|language = zh-cn}}</ref>，以其独特有趣的[[娃娃音|娃娃音]]被网友誉为“萌音教主”<ref>{{Cite web|url = http://game.china.com/mobile/gamenews/11106783/20150618/19866368.html|title = 萌音教主悄然驾临|accessdate = 2015-07-22|author = |date = 2015-06-18|publisher = 中华网|language = zh-cn|deadurl = yes|archiveurl = https://web.archive.org/web/20150722181359/http://game.china.com/mobile/gamenews/11106783/20150618/19866368.html|archivedate = 2015年7月22日|df = }}</ref>。
