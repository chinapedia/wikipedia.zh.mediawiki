'''桃園市政府客家事務局'''，為[[桃園市政府|桃園市政府]]所屬一級機關，設於[[龍潭區|龍潭區]]，第一任局長[[蔣絜安|蔣絜安]]，為[[客家|客家]]女兒、[[龍潭|龍潭]]媳婦，現任局長則是[[何明光|何明光]]。

== 組織 ==
===行政單位===
* 秘書室
* 人事室
* 會計室

===業務單位===
*綜合企劃科
*文教發展科
*園區經營科

== 業務 ==
主要業務包括桃園市內[[客家|客家]]文化及語言推廣、客家特色產業行銷、[[桃園市客家文化館|客家文化館]]館舍活化等。

== 歷任局長 ==
*第一任：[[蔣絜安|蔣絜安]]
*第二任：[[何明光|何明光]]
