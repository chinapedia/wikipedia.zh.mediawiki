'''牛頓鸚鵡'''（''Psittacula exsul''），又名'''羅德里格斯環頸鸚鵡'''，是[[羅德里格斯島|羅德里格斯島]]特有的一種[[鸚鵡|鸚鵡]]。牠們約長40厘米，呈藍灰色。<ref name = Fuller1987/>

牛頓鸚鵡的[[種小名|種小名]]是為紀念最初紀錄牠們的François Leguat的難民身份。最後是於1875年觀察到牠們，目前只餘下兩組完整的標本及數個[[亞化石|亞化石]]。<ref name =Birdlife-1531/>

==標本==
牛頓鸚鵡有兩組完全的標本存放在[[劍橋大學|劍橋大學]]，一個雄性及一個雌性。<ref name = Fuller1987/>雌鳥標本是由[[毛里裘斯|毛里裘斯]]的殖民官{{link-en|愛德華·牛頓|Edward Newton}}交給他的哥哥{{link-en|阿弗烈·牛頓|Alfred Newton}}，而阿弗烈於1872年正式作出描述。雄鳥標本是於1875年射殺的，從其喙的顏色及雙翼沒有紅點而估計是一隻雛鳥。<ref name = Fuller1987>{{cite book en |title = Extinct Birds |pages = 143–4 |first = Errol |last = Fuller | year = 1987| publisher = Penguin Books (England) |isbn = 0670817972}}</ref>目前亦有發現一些[[亞化石|亞化石]]。<ref name =Birdlife-1531/>

==描述==
牛頓鸚鵡長約40厘米，其中不半是牠們的尾羽。<ref name = Fuller1987/>牠們的外觀像其他[[鸚鵡|鸚鵡]]，頸上有黑環，不過牠們卻呈藍灰色，而非綠色。<ref name = Fuller1987/>[[瞳孔|瞳孔]]呈黃色，上身較深色，下身則較淺色。雄鳥從蠟膜至[[眼睛|眼睛]]間有一條較深色的線。<ref name = Fuller1987/>雌鳥的[[頭部|頭部]]較灰色，其頸上黑環不大明顯。<ref name = Fuller1987/>早期的報告指牠們肩膀上有紅點，但現存的兩個標本都沒有，因而估計雄鳥標本是一隻雛鳥。<ref name = Fuller1987/>另外亦有發現綠色的牛頓鸚鵡。

==滅絕==
牛頓鸚鵡最初是由François Leguat紀錄。<ref name = Fuller1987/>他認為牛頓鸚鵡很豐富，主要是吃[[橄欖|橄欖]]樹類的[[果實|果實]]。他發現牛頓鸚鵡可以作為食物及能夠模仿說話，於離開[[毛里裘斯|毛里裘斯]]時就帶走了一隻。<ref name = Fuller1987/>於1726年有另一個有關的紀錄。後來[[法國|法國]][[數學家|數學家]][[彭革雷|彭革雷]]（Alexandre Pingré）於1761年到了[[羅德里格斯|羅德里格斯]]觀察[[金星凌日|金星凌日]]時，發現牛頓鸚鵡已很稀少。

牛頓鸚鵡的[[滅絕|滅絕]]可能是因失去棲息地及獵殺所致。<ref name =Birdlife-1531>[http://www.birdlife.org/datazone/species/index.html?action=SpcHTMDetails.asp&sid=1531&m=0 BirdLife Species Factsheet: ''Psittacula exsul'']</ref>
