'''牛运震'''（{{bd|1706年||1758年|catIdx=N牛}}），[[表字|字]]'''阶平'''，[[号|号]]'''真谷'''，人称'''空山先生'''，[[山东|山东]][[滋阳|滋阳]]马青（今[[兖州市|兖州市]][[新兖镇|新兖镇]]牛楼村）人<ref name="勤政爱民的文学家牛运震 ">{{cite web|url=http://www.gs.xinhuanet.com/dfpd/2005-04/05/content_4005151.htm|title=《勤政爱民的文学家牛运震》|accessdate=2005-04-05|work=|publisher=新华网|language=zh-hans|deadurl=yes|archiveurl=https://web.archive.org/web/20060601050300/http://www.gs.xinhuanet.com/dfpd/2005-04/05/content_4005151.htm|archivedate=2006-06-01}}</ref>，[[清朝|清朝]]政治人物、[[文学家|文学家]]，同[[進士|進士]]出身。

== 简介 ==
牛运震的高祖牛天铸、曾祖牛起宗、祖父牛洪范都是[[贡生|贡生]]，他的父亲牛梦瑞也是[[拔贡|拔贡]]，家族世代温柔敦厚，使得他自幼便受到良好的教育，十岁便能诵读[[黃帝四經|四經]]、撰写文章，有“牛才子”之美誉。[[雍正|雍正]]元年（1723年）牛运震在十七岁时补选为县禀生，以文章文彩出众，声名远扬。[[雍正|雍正]]七年（1729年）选为[[拔贡生|拔贡生]]。[[雍正|雍正]]十年（1732年）应[[顺天|顺天]][[鄉試|鄉試]]中式，次年聯捷[[进士|进士]]。[[雍正|雍正]]十三年（1735年）[[山东|山东]]奉诏举办[[博学鸿词科|博学鸿词科]]考试，[[山东巡抚|山东巡抚]]岳睿選推他應試，牛运震显示出过人的才华和韬略。当时的许多硕学饱读之士与其相比亦略逊一筹，遗憾的是他到[[京都|京都]]参加[[殿试|殿试]]时，却因其文章不符合体例规定而落榜。[[乾隆|乾隆]]三年（1738年）牛运震被授予[[甘肃|甘肃]][[秦安县|秦安县]][[知县|知县]]一职，当时，[[京城|京城]]的群贤公卿都对他到如此偏僻的外地补缺任职而惋惜，他却欣然赴任。他在秦安开凿了九条河沟，灌溉了万亩粮田<ref name="勤政爱民的文学家牛运震 "/>，并且亲自教农民使用新式农具。在职期间，他平反了许多冤狱。他又在县里开设了陇山书院，授课教书。[[乾隆|乾隆]]六年（1741年）[[朝廷|朝廷]]又令牛运震兼任[[徽县|徽县]][[知县|知县]]，又任[[兩當縣|兩當縣]][[知縣|知縣]]，由于他辖治三县，便在三县中心点的[[大门镇|大门镇]]（别名小甘州）设堂办公，听讼断案。并且解决了徽县多年来的虎患<ref>[[民國|民國]]·[[趙爾巽|趙爾巽]]等，《[[清史稿|清史稿]]》（卷477）：“ 設隴川書院，日與諸生講習，民始向學。兼攝徽縣，又攝兩當縣，舍於三縣之中，曰大門鎮，以聽訟。徽縣多虎，募壯士殺虎二十六，道始通。”</ref>。[[乾隆|乾隆]]二十三年（1758年）正月二十二日，牛运震因病于家乡[[兖州|兖州]]辞世。

== 著作 ==
*牛运震学识渊博，博古通今，对[[天文|天文]]、[[地理|地理]]、[[兵法|兵法]]等均有钻研，[[文学|文学]]尤其优秀，著有《空山堂文集》。

== 评价 ==
牛运震一生廉洁勤政，兴利除弊，被后世誉为：“威惠甚重”。

[[毛泽东|毛泽东]]读其著作时批注：“读《游五姓湖记》，则见篇中人物，皆一时之豪；吾人读其文，恍惚与之交矣。游者岂徒观览山水而已哉，当识得其名人巨子贤士大夫，所谓友天下之善士也。”<ref >{{cite book |language =zh-hans|author=中共中央文献研究室，中共湖南省委《毛泽东早期文稿》编辑组|title=《毛泽东早期文稿》|year=2008-11-01|publisher=湖南人民出版社|location=湖南长沙|pages=第587页|isbn=9787543855113}}</ref> 。
