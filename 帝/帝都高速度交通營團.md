'''帝都高速度交通營團'''（{{jpn|j=帝都高速度交通営団|hg=ていとこうそくどこうつうえいだん|rm=Teito Kōsokudo Kōtsū Eidan|f=y}}），簡稱'''交通營團'''、或逕稱為'''{{link-ja|營團|営団}}'''，是[[日本|日本]]曾存在的一個{{link-ja|特殊法人|特殊法人}}機構，主要經營[[東京都|東京都]]內（[[東京都區部|東京都區部]]）的[[地下鐵|地下鐵]]路線。成立於1941年，2004年後{{link-en|公司化|Corporatization}}改制為[[東京地下鐵|東京地下鐵株式會社]]。在[[都營地下鐵|都營地下鐵]]開業前，是東京唯一的地下鐵營運單位。

== 沿革 ==
[[File:Tokyo_Metro_Office_Building.jpg|225px]]
營團是由當時東京兩間地下鐵的營運公司「[[東京地下鐵道|東京地下鐵道]]」及「{{link-ja|東京高速鐵道|東京高速鉄道}}」合併而成，前者在1927年開設日本乃至於亞洲的第一條地下鐵線，後者則在1938年開設第二條地下鐵線（兩條路線之後互相串聯，成為今日的[[銀座線|銀座線]]）。在[[太平洋戰爭|太平洋戰爭]]時期的運輸體制下，以及根據1941年（[[昭和|昭和]]16年）3月6日頒布的《帝都高速度交通營團法》，為管理東京與鄰近地區的“地下都市高速度交通事業”而成立，並由負責交通業務之[[中央省廳|中央省廳]]監管（[[鐵道省|鐵道省]]→[[運輸通信省|運輸通信省]]→[[運輸省|運輸省]]→[[國土交通省|國土交通省]]）。其[[英語|英語]]譯名為'''{{lang|en|Teito Rapid Transit Authority}}'''、簡稱'''{{lang|en|TRTA}}'''。

「地下都市高速度交通事業」即是指地下鐵，因為[[鐵路|鐵路]]在城市內比[[道路|道路]]交通快，而營團開始營運時，東京市區的軌道交通是以[[路面電車|路面電車]]為主，故當時在法人名稱中加上「高速」兩字，以強調地下鐵的速度性；「[[帝都|帝都]]」則是指「[[大日本帝國|帝國]]的[[首都|首都]]」東京，是當時常見的用詞。至於「{{link-ja|營團|営団}}」是「經營財團」的通稱，這是日本在[[中國抗日戰爭|日中戰爭]]與太平洋戰爭爆發後，為加強戰爭時期國家對部分產業的控制與管理，透過立法的方式由政府與民間共同出資的一種組織型態，以介於[[公法人|公法人]]與[[私法人|私法人]]之間的身分實施[[獨佔|獨佔]]式的經營。當時成立的營團還有住宅營團、農地開發營團等，但大部分在戰後因[[駐日盟軍總司令|盟軍占領當局]]的指令被改制為{{link-ja|公團|公団}}（採取公法人形式經營的[[公營事業|公營事業]]機構）或直接廢除，僅有帝都高速度交通營團被認為非為戰爭目的而設，而被允許存續下來，但民間[[資本|資本]]被排除，成為一個近似於公團的公營事業機構。

營團於1941年7月4日成立，同年9月1日開始營運。開業時的資本額為6000萬[[日圓|日圓]]，由[[日本政府|日本政府]]、[[東京市|東京市]]役所（1943年起為[[東京都廳|東京都廳]]）和相關鐵路業者（多為[[私鐵|私鐵]]）集資而成：日本政府佔4000萬日圓，東京市役所佔1000萬日圓，{{link-ja|東京橫濱電鐵|東京横浜電鉄}}（今[[東京急行電鐵|東京急行電鐵]]前身之一）、[[東武鐵道|東武鐵道]]各200萬日圓，[[京成電鐵|京成電氣軌道]]、[[小田急電鐵|小田急電鐵]]各100萬日圓，[[西武鐵道|西武鐵道]]、[[武藏野鐵道|武藏野鐵道]]（現被併入[[西武鐵道|西武鐵道]]）、國有鐵道共濟組合則各50萬日圓。戰後的1951年，改為純政府出資，由[[日本國有鐵道|日本國有鐵道]]（[[國鐵分割民營化|國鐵分割民營化]]後改由日本政府）與東京都廳共同出資，並由日本[[國庫|國庫]]融資進行地下鐵路網的建設。

早期在稱呼東京的「地下鐵」時，指的通常都是營團。營團經營的地下鐵路線稱作「'''營團地-{}-下鐵'''」或「'''營團線'''」，車站入口的標誌就寫上「'''{{lang|ja|地下鉄}} SUBWAY'''」。而在稱呼路線時，為免和[[東京都|東京都]]自力經營的[[都營地下鐵|都營地下鐵]]混淆，營團經營的地下鐵路線有時會稱為「營團○○線」或「地-{}-下鐵○○線」（後者使用較多）。雖然「營團地-{}-下鐵」是對營團旗下鐵路系統的[[暱稱|暱稱]]，但在營團解散前反而成為對整個組織的通稱，並被使用於提供旅客的資料上<ref>[http://members.jcom.home.ne.jp/miyayume/diary/200602/chikatetsu-rosenzu-1.jpg メトロカード]{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

2004年（[[平成|平成]]16年）4月1日，營團依照《東京地下鐵株式會社法》被廢止，營團的一切[[權利|權利]]與[[義務|義務]]由營團改組的[[東京地下鐵|東京地下鐵株式會社]]承繼。

== 4S ==
營團的標誌以'''S'''作圖案，同時在'''Subway'''的S字引申出'''4S'''，作為營團的基本理念。
*'''Speed'''：速度
*'''Safety'''：安全
*'''Security'''：可靠
*'''Service'''：服務

由於英語在戰時屬[[敵性語|敵性語]]（敵軍語言），所以營團從成立到1964年（昭和39年）10月10日是使用圓形[[隧道|隧道]]與[[鐵軌|鐵軌]]的斷面之組合圖案作為徽章<ref>[http://d.hatena.ne.jp/taketake-1998/20130817/1376745855 東京旅行二日目 - 栗原鉄道の静岡日記]</ref>。

== 重大事件及意外事故 ==
*1963年（昭和38年）9月5日，一連串的「[[草加次郎事件|草加次郎事件]]」中，最大型事件就是「[[地下鐵銀座線爆炸案|地下鐵銀座線爆炸案]]」。一列銀座線的列車最前方座位掛上[[計時炸彈|計時炸彈]]。13名乘客受傷。
*1995年（平成7年）3月20日，[[奥姆真理教|奥姆真理教]]發動[[東京地鐵沙林毒氣事件|東京地鐵沙林毒氣事件]]，12人[[死亡|死亡]]，5510人受傷。千代田線、丸之內線、日比-{谷}-線全日停止服務。
*2000年（平成12年）3月8日，日比谷線[[中目黑站|中目黑站]]附近發生-{出}-軌事故，5人死亡。
