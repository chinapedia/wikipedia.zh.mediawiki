'''芝山町'''（{{jpn|j=芝山町|hg=しばやままち|rm=Shibayama machi|f=y}}）是[[千葉縣|千葉縣]][[山武郡|山武郡]]的一[[町|町]]。依據國家{{link-ja|構造改革特別區域|構造改革特別区域}}法，2003年4月21日指定為{{link-ja|國際機場特區|国際空港特区}}。往[[成田市|成田市]]的通勤率為18.6%（平成22年國勢調査）。

== 地理 ==
位於[[成田機場|成田機場]]南側，町內有部分為機場用地。芝山町全部都在成田機場起降航道下方，由於飛機起降高度低，使得本町成為噪音汙染最大的自治體。

町域全體屬丘陵地，居民多以農業為主。

== 歷史 ==
* 本地已發現5世紀至7世紀的{{link-ja|芝山古墳群|芝山古墳群|古墳群}}。
* 「芝山」來自於南部的[[天台宗|天台宗]]寺院「{{link-ja|觀音教寺|観音教寺|芝山仁王尊}}」。芝山仁王尊傳聞建於[[奈良時代|奈良時代]]前期。
* 芝山仁王尊在[[江戶時代|江戶時代]]以前與成田市的[[成田山新勝寺|成田山新勝寺]]並列。過去曾規劃通過芝山的鐵道路線，但未能實現。在成田開始建造鐵道後（20世紀初），兩地參拜客數差距逐漸拉開。

=== 近世 ===
* 1889年（明治22年）4月1日 - [[町村制|町村制]]施行，{{link-ja|武射郡|武射郡}}{{link-ja|二川村 (千葉縣山武郡)|二川村 (千葉県山武郡)|二川村}}與武射郡{{link-ja|千代田村 (千葉縣)|千代田村 (千葉県)|千代田村}}成立。
* 1955年（昭和30年）7月1日 - 山武郡二川村與山武郡千代田村合併成立山武郡芝山町。
* 1966年（昭和41年）7月4日 - 閣議通過三里塚機場建設促進決議案。新東京國際機場建設預定地為千葉縣成田市{{link-ja|三里塚|三里塚}}{{link-ja|宮內廳下總御料牧場|宮内庁下総御料牧場}}。
** 7月20日 - {{link-ja|三里塚芝山聯合機場反對同盟|三里塚芝山連合空港反対同盟}}成立，反對鬥爭白熱化。
* 1978年（昭和53年）5月20日 - 新東京國際機場（現成田國際機場）啟用。
* 1985年（昭和60年）3月24日 - 芝山町、成田市、[[佐倉市|佐倉市]]、[[榮町_(千葉縣)|榮町]]指定為國家國際觀光示範區。
* 1997年（平成9年）12月7日 - 三里塚芝山聯合機場反對同盟前成員{{link-ja|相川勝重|相川勝重}}<ref>[http://www.asahi.com/politics/update/1014/TKY200910140112.html 朝日新聞－「頭にきて眠れず」森田知事] (2009.10)</ref><ref>[http://www.zck.or.jp/essay/2683.html 全国町村会－地域と空港と「真の共栄」を目指して～千葉県芝山町長・相川勝重(全国町村会)] {{webarchive|url=https://web.archive.org/web/20111110234036/http://www.zck.or.jp/essay/2683.html |date=2011-11-10 }}</ref>當選町長。
* 2002年（平成14年）10月27日 - [[芝山鐵道|芝山鐵道]][[芝山鐵道線|芝山鐵道線]]開通。同時[[芝山千代田站|芝山千代田站]]開業。

== 經濟 ==
芝山町緊鄰機場，臨空工業團地（芝山第2工業團地、空港南部工業團地）等有許多企業進駐，為依據國家構造改革特別區域法成立的國際機場特區一部分。因此，本町的就業機會、[[物業稅|物業稅]]收入等皆優於鄰近市町村，是普通交付稅
不交付團體。

=== 第一級產業 ===
農業從事者多，約占總人口30%左右。近年[[西瓜|西瓜]]與花的產量增加。

=== 第二級產業 ===
町內有芝山（向野地區－千葉縣企業廳。木崎地区－千葉縣社區營造公社）、第2芝山（千葉縣企業廳）、機場南部（千葉縣企業廳）三處工業團地。芝山工業團地內有{{link-ja|日本奧的斯電梯|日本オーチス・エレベータ）}}的電梯試驗塔「芝山測試塔」（高154.2公尺，地上39層）。此外現在物流園區也不斷擴展。

== 名勝、古蹟、觀光地、祭典、活動 ==
; 芝山仁王尊{{link-ja|觀音教寺|観音教寺}}
: 創建於781年的[[天台宗|天台宗]]古刹，江戶時代與[[成田山新勝寺|成田山新勝寺]]（成田不動）同為民間信仰中心。以安放在仁王門的仁王像與本尊厄除十一面觀世音菩薩聞名。院前的三重塔已被列為千葉縣有形文化財。

; {{link-ja|航空科學博物館|航空科学博物館}}
: 日本第一座展示飛機歷史、技術的航空博物館。館內展示日本首次飛行的Farman III雙翼機複葉機全尺寸模型與飛機實體引擎，另有飛行模擬器可體驗。5樓的360度展望台可觀賞成田機場飛機起降。室外有戰後第一隻國產客機「[[YS-11|YS-11]]試作1號機」。

; 芝山公園
: 占地15公頃的園內除了梅花與草地外，還有千葉縣有形文化財的江戶時代農家（舊藪家住宅），以及休閒設施、棒球場、迷你兒童活動場等。

; {{link-ja|芝山古墳·埴輪博物館|芝山古墳・はにわ博物館}}
: 芝山公園內，展示縣內、九十九里地方出土的埴輪與[[古墳時代|古墳時代]]（3世紀末〜7世紀）考古遺跡。

; 埴輪祭
: 1982年起每年11月第2個星期日舉行。

; {{link-ja|Satellite成田|サテライト成田}}·f-keiba成田
: {{link-ja|競輪場外車券售場|競輪場外車券売場}}、{{link-ja|場外勝馬投票券發售所|場外勝馬投票券発売所}}。
