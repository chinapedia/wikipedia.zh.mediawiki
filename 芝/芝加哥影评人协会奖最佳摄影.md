'''芝加哥影评人协会奖最佳摄影'''（{{lang-en|'''Chicago Film Critics Association Award for Best Cinematography'''}}）是[[芝加哥影评人协会|芝加哥影评人协会奖]]的主要奖项之一。<ref>{{cite web |url=http://yule.sohu.com/20131218/n391983985.shtml |title=芝加哥等影评人协会扎堆颁奖 《为奴》所向披靡 |accessdate= 2013|publisher=搜狐网 |work= |language=zh-hans}}</ref><ref>{{cite web |url=http://ent.163.com/12/1219/11/8J3778M600034QN3.html |title=芝加哥影评人协会奖揭晓 《猎杀本拉登》横扫 |accessdate= 2012|publisher=网易 |work= |language=zh-hans}}</ref>

== 概述 ==
罗杰·狄金斯是提名和得奖次数最多的人，他被提名10次，得奖2次。
