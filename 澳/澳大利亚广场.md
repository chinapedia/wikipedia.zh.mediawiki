'''澳大利亚广场'''（{{lang|en|Australia Square}}）是[[澳大利亚|澳大利亚]][[悉尼|悉尼]][[中心商务区|中心商务区]]的一座集办公和商业为一体的建筑，其地址为悉尼乔治大街264号。
 
建筑设计师为[[哈里·塞德勒|哈里·塞德勒]]（{{lang|en|Harry Seidler}}）及其合作人。如今，它是悉尼[[地标|地标性]]建筑，并被认为是澳大利亚建筑的象征。<ref name="arch"/><ref name="smh">{{cite web |url=http://www.smh.com.au/articles/2004/05/14/1084289865200.html |title=Top of the town |author= |accessdate=2010-05-04 |date=2004-05-15 |work=Sydney Morning Herald}}</ref>其最突出的特征为从落成的1967年到1976年之间其塔楼为悉尼最高建筑。<ref name="arch">{{cite web |url=http://www.architecture.com.au/i-cms?page=6364 |title=Biography: Harry Seidler AC OBE LFRAIA |accessdate=2010-05-04 |date=2005-05-06 |work=architecture.com.au |publisher=The Australian Institute of Architects }}</ref>

澳大利亚广场由GTP集团拥有。1967年由于其创新和吸引人的设计而获得约翰·萨尔曼爵士勋章（{{lang|en| Sir John Sulman Medal }}）。1990年代中期，该建筑被完全翻新。<ref>{{cite web |url=http://www.australiasquare.net/LoadIntroPageLinks.asp?IntroPageLinks=BuildingProfile |title=Building Profile |accessdate=2010-05-04 |work=Australia Square Online |publisher=Core Vision |deadurl=yes |archiveurl=https://web.archive.org/web/20110718100416/http://www.australiasquare.net/LoadIntroPageLinks.asp?IntroPageLinks=BuildingProfile |archivedate=2011-07-18 }}</ref>2003年进行了另一个耗值1100万美元的整修方案，其中包括更换所有公共区域的铺路石、新的照明系统及户外桌子。<ref>{{cite web |url=http://www.infolink.com.au/c/Bovis-Lend-Lease/Australia-Square-gets-a-makeover-n756508 |title=Australia Square gets a makeover |accessdate=2010-05-04 |date=2003-07-04 |work=Infolink |publisher=Reed Business Information |deadurl=yes |archiveurl=https://web.archive.org/web/20110615153107/http://www.infolink.com.au/c/Bovis-Lend-Lease/Australia-Square-gets-a-makeover-n756508 |archivedate=2011-06-15 }}</ref> 

==塔楼==
[[File:Australia_sq_sydney.jpg|thumb]]
建筑的正式名称为塔楼（{{lang|en|Tower Building}}），在其落成的时候为世界上最高的轻质混凝土建筑。<ref name="arch"/>建筑高度约为170米，<ref name="hsa">{{cite web |url=http://www.seidler.net.au/projects/014.html |title=Australia Square |accessdate=2010-05-04 |date= |work= |publisher=Harry Seidler and Associates |deadurl=yes |archiveurl=https://web.archive.org/web/20080719194527/http://seidler.net.au/projects/014.html |archivedate=2008-07-19 }}</ref>最初的设计为58层，后来减为50层。<ref name="smh"/>47楼为[[旋转餐厅|旋转餐厅]]，48楼为观景台。<ref name="smh"/>建筑包括了悉尼最大的地下停车场，可容纳400辆车。<ref name="rc">{{cite web |url=http://www.realcommercial.com.au/cgi-bin/rsearch?a=o&id=5459588&f=0&p=10&t=com&ty=&fmt=&header=&cc=&c=97169014&s=nsw&snf=as&tm=1224653690 |title=Australia Square - Sydney's Icon |accessdate=2010-05-04 |work=realcommerical.com.au |publisher=REA Group |deadurl=yes |archiveurl=https://archive.is/20121208200357/http://www.realcommercial.com.au/cgi-bin/rsearch?a=o&id=5459588&f=0&p=10&t=com&ty=&fmt=&header=&cc=&c=97169014&s=nsw&snf=as&tm=1224653690 |archivedate=2012-12-08 }}</ref>
