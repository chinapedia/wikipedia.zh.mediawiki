2001年4月11日，在[[2002年世界杯足球赛|2002年韩日世界杯]]的一场[[2002年世界杯外围赛|预选赛]]中，[[澳大利亚国家足球队|澳大利亚国家足球队]]跟[[美属萨摩亚足球代表队|美属萨摩亚足球代表队]]在位于[[考夫斯港|考夫斯港]]的[[BCU国际体育场|BCU国际体育场]]舉行的一場賽事。澳大利亚以31:0的比分击败了对手，并且创造了国际比赛最大分差的世界纪录。在比赛中，[[阿奇·湯普森|湯普森]]独进13球，同样打破了国际比赛中个人单场进球最多的记录。<ref name=Telegraph>{{cite web |url=http://www.telegraph.co.uk/sport/football/international/3003021/Australia-score-31-without-loss-in-record-win.html |title=Australia score 31 without loss in record win |date=2001-04-11 |accessdate=2009-08-05 |work=Telegraph.co.uk |publisher=Telegraph Media Group Limited|language=英语}}</ref> 

比赛的结果引发了对世界杯预选赛赛制的争论，汤普森和澳大利亚的主教练都认为应该引进分组初赛来避免这種強弱懸殊的比赛发生。最终，这个意见被[[国际足联|国际足联]]采纳，并于[[2006年世界杯足球赛|2006年德国世界杯]]的[[2006年世界杯外围赛_(大洋洲区)|大洋洲区预选赛]]开始正式实施。

== 背景 ==
自从1986年起，国际足联开始在[[大洋洲足球协会|大洋洲]]设立一个单独的[[世界杯外围赛|预选赛]]。此前，大洋洲的球队和亚洲的球队一起参加世界杯[[亚洲足球协会|亚洲区]]的预选赛。[[2002年世界杯足球赛|2002年世界杯]]，一共有10支球队参加了大洋洲区的预选赛。这些球队被分为两个小组，每个小组五支球队。获得小组第一的两支球队将分别在主客场与对方交锋，获胜者将参加与南美区第五名球队的洲际附加赛，最终获胜的一方将参加2002年韩日世界杯。[[澳大利亚国家足球队|澳大利亚]]、[[美属萨摩亚国家足球队|美属萨摩亚]]、[[斐济国家足球队|斐济]]、[[萨摩亚国家足球队|萨摩亚]]以及[[汤加国家足球队|汤加]]被分在第一小组。所有的比赛于2001年4月在[[澳大利亚|澳大利亚]]的[[考夫斯港|考夫斯港]]举行。<ref name=FIFA>{{cite web |url=http://www.fifa.com/worldcup/archive/edition=4395/preliminaries/preliminary=3863/index.html |title=2002 FIFA World Cup Korea/Japan Preliminaries; Results, Oceanian Zone |date= |accessdate= 2009-08-05 |work=FIFA.com |publisher=Fédération Internationale de Football Association|language=英语}}</ref>

澳大利亚和[[新西兰国家足球队|新西兰]]被认为是大洋州区最强的球队，只有这两支球队获得过[[大洋洲国家杯|大洋洲国家杯]]。澳大利亚和新西兰分别在[[1974年世界杯足球赛|1974年]]和[[1982年世界杯足球赛|1982年]]参加过世界杯正赛。另一方面，美属萨摩亚是世界上最弱的足球代表队之一，从1998年加入国际足联起，他们输掉了所有的正式比赛。比赛之前，澳大利亚在国际足联的[[国际足协世界排名|排名]]是第75位，而美属萨摩亚排名203位，是所有成员中排名最低的。<ref name=second />

在比赛的两天以前，澳大利亚以22比0的悬殊比分战胜了汤加，打破了原来由[[科威特国家足球队|科威特]]保持的国际比赛最大分差记录。在2000年，科威特以20比0战胜了[[不丹国家足球队|不丹]]。<ref name=Kuwait>{{cite web |url=http://www.independent.co.uk/sport/football/internationals/exposed-tonga-lose-220-680944.html |title='Exposed' Tonga lose 22-0 |date=2001-04-10 |accessdate=2009-08-05 |work=The Independent |publisher=Independent News and Media Limited|language=英语}}</ref>而美属萨摩亚刚刚经历了两连败，分别是0比13输给斐济以及0比8输给萨摩亚。<ref name=FIFA/>

== 比赛详情 ==
[[File:Archie_Thompson.jpg|thumb]]
澳大利亚的大部分主力球员并没有在本场比赛中登场亮相，其中包括了在22比0横扫汤加的比赛中联合贡献10球的前锋[[约翰·阿洛伊西|约翰·阿洛伊西]]和[[达米安·莫里|达米安·莫里]]。而美属萨摩亚的球员正受到护照问题的困扰，在20名正选球员中只有门将尼基·萨拉普能够参赛。美属萨摩亚在召唤他们的U-20球员代替成年队参加比赛时同样遇到了麻烦，因为当时小球员们正在参加高中考试。因此，美属萨摩亚只能派遣国少队球员参赛，所有球员的平均年龄只有18岁，其中还包括三个15岁的球员。<ref>{{cite web |url=http://www.telegraph.co.uk/sport/football/international/3003225/FIFA-ruling-which-left-Samoans-singing-the-blues.html |title=FIFA ruling which left Samoans singing the blues |date=2001-04-14 |accessdate=2009-08-05 |work=Telegraph.co.uk |publisher=Telegraph Media Group Limited|language=英语}}</ref>

美属萨摩亚一直保持没有失球直到布西亚尼斯在第10分钟依靠角球打破僵局。第12分钟，[[阿奇·汤普森|阿奇·汤普森]]打进了自己在本场比赛的第一粒进球。一分钟后，他的搭档大卫·日德里奇也打入一球。在第17分钟和第19分钟，[[托尼·波波维奇|托尼·波波维奇]]连入两元，使得澳大利亚以6比0领先。第25分钟，日德里奇完成了帽子戏法，此时，澳大利亚已经拥有9球的领先优势。在接下来的20分钟，汤普森一共打入6球，并率领球队以16比0的比分领先。上半场，汤普森已经攻入8球，日德里奇也有4球入账。<ref name=ESPN>{{cite web |url=http://soccernet.espn.go.com/archive/worldcup/news/2001/0411/20010411australiawr.html |title=Australia smash two world records |date=2001-04-11 |accessdate=2009-08-05 |work=ESPN.com Soccernet |publisher=ESPN Internet Ventures|language=英语}}</ref><ref name=BBC>{{cite web |url=http://news.bbc.co.uk/sport2/hi/football/world_cup_2002/1271854.stm |title=Aussie footballers smash world record |date=2001-04-11 |accessdate=2009-08-05 |work=BBC Sport |publisher=British Broadcasting Corporation|language=英语}}</ref>

在上半场打破僵局的布西亚尼斯在第50分钟再度为球队建功，之后不久，他便完成了帽子戏法。汤普森和日德里奇在下半场也分别攻入5球和4球，这使得他们在本场比赛的进球数分别达到了13粒和8粒。[[奥雷里奥·维德马|奥雷里奥·维德马]]和西蒙·科洛西莫在本场比赛中梅开二度，替补登场的佛斯托·德·亚米契斯则为球队打进一球。<ref name=ESPN/><ref name=BBC/> 在第86分钟，已经29球落后的美属萨摩亚由帕蒂·斐吉埃完成了他们在本场比赛中的第一脚也是唯一一脚射门，可惜被澳大利亚门将[[迈克尔·佩特科维奇|迈克尔·佩特科维奇]]扑出。<ref name=CBC>{{cite web |url=http://www.cbc.ca/sports/story/2001/04/11/fifa010411.html |title=Australians set World Cup soccer scoring record |date=2001-04-11 |accessdate=2009-08-05 |work=CBCSports.ca |publisher=Canadian Broadcasting Corporation|language=英语}}</ref>

巨大的比分导致了现场记分的错误。在比赛结束以后，赛场的大屏幕打出了澳大利亚以32比0战胜美属萨摩亚，汤普森打进14球。<ref>{{cite web |url=http://www.nytimes.com/2001/04/12/sports/plus-soccer-was-it-31-0-or-32-0-australia-wins.html?n=Top/News/U.S./U.S.%20States,%20Territories%20and%20Possessions/American%20Samoa |title=Was It 31-0 or 32-0?; Australia Wins |date=2001-04-12 |accessdate=2009-08-05 |work=The New York Times |publisher=The New York Times Company|language=英语}}</ref> 但在统计员重新确认以后，这一结果被更改为了31比0，汤普森的进球总数也从14球变为了13球。<ref name=ESPN/> 比赛结束以后，FIFA公布了官方统计数据，根据裁判和在场官员的报告，本场比赛澳大利亚以31比0获胜，汤普森一共打入13球。<ref>{{cite web |url=http://www.fifa.com/worldcup/archive/edition=4395/preliminaries/preliminary=3863/matches/match=22243/report.html |title=Match Report: Australia - American Samoa |year=2001 |accessdate=2009-08-05 |work=FIFA.com |publisher=Fédération Internationale de Football Association|language=英语}}</ref>

== 记录 ==
澳大利亚31比0战胜美属萨摩亚的比赛打破了两天前由他们创造的国际比赛最大分差胜利记录。此前，他们以22比0的悬殊比分战胜了[[汤加国家足球队|汤加]]。这个记录的先前保持者是[[科威特国家足球队|科威特]]，他们在[[2000年亚洲杯足球赛|2000年亚洲杯]]的预选赛中20比0击败[[不丹国家足球队|不丹]]。<ref name=Kuwait/> 这场比赛同样打破了世界杯最大比分胜利记录，该记录先前的保持者是[[伊朗国家足球队|伊朗]]，他们在2002年世界杯亚洲区预选赛中19比0大胜[[关岛国家足球队|关岛]]。<ref name=Telegraph/>

除了最大分差的记录被打破，许多个人记录在这场比赛后也被改写。澳大利亚的[[阿奇·汤普森|阿奇·汤普森]]在本场比赛中攻入13球，打破了原先由澳大利亚球员加里·科尔和伊朗人[[卡里姆·巴盖里|卡里姆·巴盖里]]共同保持的国际比赛个人单场进球数量记录，在[[1982年世界杯足球赛|1982年世界杯]]预选赛澳大利亚对阵[[斐济国家足球队|斐济]]以及[[1998年世界杯足球赛|1998年世界杯]]预选赛伊朗对阵[[马尔代夫国家足球队|马尔代夫]]比赛中，加里·科尔和卡里姆·巴盖里分别为球队打进7球。<ref>{{cite web |url=http://www.theage.com.au/news/Soccer/The-allimportant-Cole-difference/2004/12/12/1102625590072.html |title=The all-important Cole difference |date=2004-12-12 |accessdate=2009-08-05 |work=TheAge.com.au |publisher=The Age Company Ltd|language=英语}}</ref><ref>{{cite web |url=http://www.guardian.co.uk/Archive/Article/0,4273,4110868,00.html |title=Iran fanatics keep close eye on the Valley |date=2000-12-30 |accessdate=2009-08-05 |work=guardian.co.uk |publisher=Guardian News and Media Limited|language=英语}}</ref> 在本场比赛中打进8球的[[大卫·日德里奇|大卫·日德里奇]]紧随汤普森之后，成为国际比赛单场进球第二多的球员。但也有人认为，[[丹麦国家足球队|丹麦]]的索弗斯·尼尔森和[[德国国家足球队|德国]]的戈特弗里德·福克斯才是国际比赛单场进球数量第二的保持者，他们曾在[[1908年夏季奥林匹克运动会|1908年奥运会]]和[[1912年夏季奥林匹克运动会|1912年奥运会]]的一场比赛中攻入10球。<ref name=second/> 但这些比赛虽然为国际足联所承认，但参赛者均为业余球员。汤普森的记录还平了成人正式比赛的单场最高进球记录，这项记录由约翰·皮特里在1885年的[[苏格兰足总杯|苏格兰足总杯]]的一场比赛中所创造的。<ref>{{cite web |url=http://www.dailymail.co.uk/sport/article-1054747/On-day--September-12.html |title=On this day - September 12 |date=2008-09-11 |accessdate=2009-08-05 |work=Daily Mail |publisher=Associated Newspapers Ltd|language=英语}}</ref>

== 赛后反应 ==
[[File:Frank_Farina_cropped.jpg|thumb]]
赛后，澳大利亚的主教练[[弗兰克·法里纳|弗兰克·法里纳]]批评了世界杯预选赛的制度并质疑世界杯预选赛是否应该出现这种差距悬殊的比赛。攻入13球的阿奇·汤普森对自己创造了新的记录表示高兴，但是，他也同意法里纳的观点。[[国际足联|国际足联]]发言人基斯·库珀表示同意他们关于比赛的评论以及改制的想法。但[[大洋洲足球协会|大洋洲足协]]主席巴希尔·斯卡尔塞拉并不认同他们的观点并宣称小球队也有权面对澳大利亚和[[新西兰国家足球队|新西兰]]这样的球队，正如澳大利亚有权面对[[巴西国家足球队|巴西]]和[[法国国家足球队|法国]]一样。<ref name=Telegraph/><ref name=CBC/>

{{Cquote 2|打破世界纪录让我梦想成真，这种事并不是天天都能发生的。但是，你应该看看你的对手，并问问自己：我们需要这样的比赛吗？<ref name=CBC/>|阿奇·汤普森}}

为了避免这種強弱懸殊的比赛再度发生，国际足联在[[2006年世界盃外圍賽_(大洋洲區)|2006年世界杯大洋州区的预选赛]]引入了一项新的赛制，即小球队需要进行初赛，初赛的胜者才能挑战澳大利亚和新西兰。<ref>{{cite web |url=http://www.fifa.com/newscentre/news/newsid=538353.html |title=Road to SA 2010 starts in Samoa |date=2007-06-21 |accessdate=2009-08-05 |work=FIFA.com |publisher=Fédération Internationale de Football Association|language=英语}}</ref> 在2006年底，为了提高在国际比赛中的竞争力，澳大利亚离开大洋洲足协加入[[亚足联|亚足联]]，他们和其他球队的巨大差距是促使他们离开的原因之一。<ref>{{cite web |url=http://www.smh.com.au/articles/2005/03/10/1110417621684.html |title=Goal at last: Australia joining Asia |date=2005-03-11 |accessdate=2009-08-05 |work=smh.com.au |publisher=The Sydney Morning Herald|language=英语}}</ref> 从[[2010年世界杯外围赛|2010年世界杯预选赛]]开始，澳大利亚将和其他亚洲球队争夺进入世界杯的入场券。<ref>{{cite web |url=http://www.telegraph.co.uk/sport/football/international/england/2326661/Fifa-World-Cup-2010-The-draw-in-full.html |title=Fifa World Cup 2010: The draw in full |date=2007-11-26 |accessdate=2009-08-05 |work=Telegraph.co.uk |publisher=Telegraph Media Group Limited|language=英语}}</ref>

== 比赛过后 ==
美属萨摩亚以0比5败给汤加结束了他们的2002年世界杯预选赛旅程。他们以四场比赛57失球并且没有进球排名小组垫底。<ref name=FIFA/>

澳大利亚在本场比赛后则分别以2比0和11比0的比分战胜斐济和萨摩亚，并以四场比赛66个进球没有失球占据了小组第一。之后，澳大利亚又以6比1战胜了另外一个小组的第一新西兰，<ref name=FIFA/> 成功晋级洲际附加赛，对手是南美区第五名[[乌拉圭国家足球队|乌拉圭]]。最终，他们以总比分1：3落敗，未能出線2002年日韩世界杯。<ref>{{cite web |url=http://www.telegraph.co.uk/sport/football/international/3017518/World-Cup-Qualifier-Australian-hopes-sunk-by-Morales.html |title=World Cup Qualifier: Australian hopes sunk by Morales |date=2001-11-25 |accessdate=2009-08-05 |work=Telegraph.co.uk |publisher=Telegraph Media Group Limited|language=英语}}</ref>
