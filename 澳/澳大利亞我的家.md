'''澳大利亞我的家'''，([[英文|英文]]：I still call Australia home)，又被稱作'''澳大利亞是我的家'''；'''澳洲是我家'''，是一首澳洲民歌。於1980年由{{link-en|彼得·阿倫|Peter Allen}}（Peter Allen）譜曲并演唱。

這首歌在推出后即在澳洲廣為流行，并成為其中一首最為著名的澳洲愛國歌曲。另外，一些工作和生活在歐洲和美國的澳洲人更是將其視為一種精神食糧，一個真實的例子是[[澳洲航空|澳洲航空]]自1980年代以來就將這首歌作為其廣告歌曲，而客戶的受眾群體則是大多數生活在海外的澳洲人。<ref name="InspirationRoom">{{cite web |url=http://theinspirationroom.com/daily/2006/qantas-i-still-call-australia-home |title=Qantas I Still Call Australia Home |publisher=The Inspiration Room Daily |accessdate=2 July 2009 |author=Duncan Macleod}}</ref>

在1984年洛杉磯奧運會的開幕式的電視特別節目中，[[奥莉维亚·纽顿-约翰|奥莉维亚·纽顿-约翰]]在一個唱詩班的配合下演唱該曲和《馬特達的華爾滋》。隨後兩隻歌曲出現在美國歌劇《來自澳洲的男孩》中，歌劇描述彼得·阿倫的生平，由[[荷裡活|荷裡活]]澳洲著名演員[[休·杰克曼|休·杰克曼]]飾演彼得·阿倫。

== 演唱 ==
* 《澳大利亞我的家》曾被[[塔斯馬尼亞州|塔斯馬尼亞州]]的[[TNT電視臺|TNT電視臺]]用於收臺歌曲。<ref>{{cite web |url=http://www.youtube.com/watch?v=6Xvo0L_oUWQ |title=TNT 9 Closedown Early 1980s |accessdate=1 September 2010 |date=circa 1980}}</ref> 和{{link-en|塔斯馬尼亞電視臺|TasTV}}.<ref>{{cite web |url=http://www.youtube.com/watch?v=DNWNSGDDhXY |title=TAS TV End of Transmission |accessdate=16 March 2012 |date=circa 1986}}</ref>
* 在澳洲航空的廣告版本中，歌詞中的[[里約熱內盧|里約熱內盧]]被[[羅馬|羅馬]]代替；因為澳洲航空並沒有開通直飛里約熱內盧的航線，而[[墨爾本|墨爾本]]以及[[悉尼|悉尼]]直飛羅馬則是公司當時極力宣傳的航線之一。<ref name="InspirationRoom"/>
** 在2009年澳洲航空的广告中，歌詞中出現了出现的“Kala Lagaw Ya”為[[托雷斯群島|托雷斯群島]]當地的土著語言。<ref name="InspirationRoom"/> 
* 這首歌亦曾被[[凱莉·米洛|凱莉·米洛]]於2009年3月在[[2009年澳洲熱浪|維州熱浪]]導致的山火義演中演唱。
* 這首歌亦以澳洲航空的名義出現在2009年的[[澳大利亞澳式足球聯盟|澳大利亞澳式足球聯盟]]([[AFL|AFL]]) 總決賽的賽前表演上，當時亦有一架澳洲航空的[[空中客車A330|空中客車A330]]出現在[[墨爾本木球場|墨爾本木球場]]上空。<ref>{{cite web |url=http://www.youtube.com/watch?v=tcqwpkcWT2U |title=I Still Call Australia Home - Qantas Choir Live Performance AFLGrand Final 2009 |accessdate=23 Feburary 2014 |date=circa 2009}}</ref>
* 這首歌亦在[[奥普拉·温弗里秀|奥普拉·温弗里秀]]的最後一季的表演節目中被[[荷裡活|荷裡活]]的“[[桉樹葉幫|桉樹葉幫]]”部份成員演唱，包括[[妮可·基德曼|妮可·基德曼]]，[[凯斯·艾尔本|凯斯·艾尔本]]，[[休·杰克曼|休·杰克曼]]，[[奥莉维亚·纽顿-约翰|奥莉维亚·纽顿-约翰]]和[[羅素·高爾|羅素·高爾]]於悉尼歌劇院外圍的臺階上演唱。
* 這首歌曲隨著澳航的廣告曾風靡於世界各地，包括在[[中國|中國]]亦具有一定的知名度。
