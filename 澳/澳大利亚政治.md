'''[[澳大利亚|澳大利亚]]'''是[[君主立宪制|君主立宪制]]下的[[议会民主制|议会民主制]]国家。

== 国家元首 ==
澳大利亚是联邦制[[君主立宪制|君主立宪制]]国家。国家元首是澳大利亚君主（澳大利亚与[[英国|英国]]及其他[[英联邦王国|英联邦王国]][[英国君主|共戴一君]]）。[[澳大利亚总督|澳大利亚总督]]为澳大利亚君主不在澳大利亚时的代表，在《[[澳大利亚宪法|澳大利亚宪法]]》下代替君主行使职权。现任国家元首为[[伊丽莎白二世|伊丽莎白二世]]，现任总督[[彼得·科斯格罗夫|彼得·科斯格罗夫]]。在联邦制下，澳大利亚君主同时也是各州的君主，在各州直接任命各州{{en-link|澳大利亚州督|Governors_of_the_Australian_states|州督}}（而不由联邦总督或政府指派）。根据现代的澳大利亚宪政惯例，君主除了在按照澳大利亚总理和各州{{en-link|澳大利亚州长|Premiers_of_the_Australian_states|州长}}的提名任命或撤换总督和州督时，不过问澳大利亚政治。

按照各英联邦王国在20世纪中叶确定的宪政惯例，澳大利亚君主王位的承袭规则根据历代英国法律确立、由所有王国共同承认。修改此规则需要各王国一致同意并分别立法方可有效。

== 议会 ==
{{main|澳大利亚国会}}
根据现行宪法，澳大利亚议会由澳大利亚君主、[[澳大利亚众议院|众议院]]、[[澳大利亚参议院|参议院]]组成。

众议院有150个议席，任期3年。但在其任期内，政府可以随时解散众议院提前大选。由于众议院可以对政府发动[[不信任动议|不信任动议]]迫使政府倒台，这使得政府必须对众议院负责，因此政府都由众议院多数黨产生。目前最近一次提前大选是2016年。

参议院则共有76个议席，任期6年，每3年改选一半，如果政府的立法不能被参议院通过，政府可提请君主或总督“解散参众两院”。参议院有6个州都平均获12席，另外2个领地各获得2席。由于参议院任期固定，最近一次提前大选是1998年，因此即使参众两院同時选举，选举后，参议员在翌年7月1日才就任。例如2013年9月7日举办的[[2013年澳洲聯邦大選|2013年澳洲聯邦大選]]，参议院当选后，在2014年7月1日才就任。

众参两院在平常的立法权力上相同，但根据宪法规定，财政预算案只能在众议院提出及修改，参议院只能表决财政预算案并不能对其进行修改。但尽管如此，由于澳大利亚的参议院权力比同是实行[[西敏制|西敏制]]的英国和加拿大等的参议院（或上议院）都要获得更多实质性的权力，因此导致了有些时候参议院与政府的关系的紧张，这被认为是[[1975年澳大利亞宪政危机|1975年澳洲宪政危机]]的一个原因之一。

== 政府 ==
[[File:Scott_Morrison_2014_crop.jpg|150px]]
澳大利亚的元首虽为君主，而总督为代表，在名义上有很大权力。但总督基本都会授权众议院的多数党组织以总理为首的内阁执行权力，总督只是充当礼仪上的职务，甚至总督本身都由总理提名后再提请君主任命。因此总督基本不会与总理發生冲突，除了[[1975年澳大利亚宪政危机|1975年澳大利亚宪政危机]]以外。

澳大利亚联邦政府由众议院多数党或政党联盟组成。政府首脑为[[澳大利亚总理|澳大利亚总理]]，一般由众议院的多数党领袖担任，除了[[约翰·戈顿|约翰·戈顿]]曾在1968年大选中以参议员身份获任总理。现任总理为[[联盟_(澳大利亚)|政党联盟]]及[[澳大利亚自由党|自由党]]党魁[[斯科特·莫里森|斯科特·莫里森]]。尽管在澳大利亚宪法规定中并没明文记载，但是根据宪法64条任命部长的规定，总理作为首席的政府部长而可以任命，而作为政府首脑，总理掌有包括提请领导政府，任命部长，解散议会大选等最高的权力。

根据西敏制惯例，总理从国会议员中选择部长，以此组成[[澳大利亚内阁|内阁]]。政策决定都由内阁决定，除了正式对外宣布的决定以外，内阁讨论的内容并不公开，而且所有内阁部长都必须遵守「内阁一致」的原则。同总理职位一样，内阁不是宪法所规定的国家机构，因此内阁的决定没有直接法律效力，须经以总督为首的[[联邦行政会议|联邦行政会议]]象征性通过方得实行。

== 司法 ==
澳大利亚最高司法机构是[[联邦高等法院|联邦高等法院]]。各个州设有州最高法院、区法院和地方法院；[[澳大利亚首都领地|首都领地]]和[[北领地|北领地]]仅设最高法院和地方法院。

== 政党 ==
澳大利亚主要政党有[[澳大利亚工党|澳大利亚工党]]、[[澳大利亚自由党|澳大利亚自由党]]、[[澳大利亚国家党|澳大利亚国家党]]；规模较小的政党有[[澳大利亚民主党|澳大利亚民主党]]、[[绿党|绿党]]、[[家庭第一党|家庭第一党]]、[[单一民族党|单一民族党]]等。现执政党为澳大利亚自由党和澳大利亚国家党组成的[[聯盟_(澳洲)|联盟]]。

虽然澳大利亚法律允许多党竞争，但其选举制度下的自由竞争使得澳大利亚政坛从20世纪初开始就形成了相对稳定的两大党派竞争的形势。此两大党派中的一方自从20世纪初就一直是较左翼的工党，另一方则是较右翼的[[聯盟_(澳洲)|政党联盟]]，此联盟的成员党历史上有过更替，现在主要是自由党和国家党。

工党作为一个社会民主的中间偏左的政党，被认为是劳工阶层的代表，近年也吸纳进步中产阶级的支持，赢得[[2007年澳洲大选|2007年大选]]，并在[[2010年澳大利亚联邦大选|2010年大选]]后联合独立议员继续执政。而自由党与国家党则被认为是两个偏右的保守性政党，分别代表较保守的城市中产阶级及资产阶级，和乡村的农场主阶层。自由、国家两党在选举中和在议会内部都组成[[聯盟_(澳洲)|長期合作聯盟]]，而在[[昆士兰州|昆士兰州]]两党甚至已经合并为[[昆士蘭自由國家黨|自由国家党]]，而在[[北领地|北领地]]则为{{en-link|乡村自由党|Country Liberal Party}}，这四个党派在全国性的选举中通常联合起来，并连续赢得[[2013年澳洲聯邦大選|2013年]]、[[2016年澳大利亞聯邦大選|2016年]]两次大选。
