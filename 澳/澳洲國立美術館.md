'''澳洲國立美術館'''（{{lang|en|National Gallery of Australia}}），1967年由[[澳洲聯邦|澳洲聯邦]]政府成立的公共美術館，位於[[澳洲首都特區|澳洲首都特區]][[坎培拉|坎培拉]]<ref>{{Cite web |url=http://nga.gov.au/AboutUs/building/building.cfm |title=About the building |publisher=澳洲國立美術館 |accessdate=2012-10-25 |language=en}}</ref>。1970年選定[[伯利·格里芬湖|伯利·格里芬湖]]畔現址後，美術館於1982年完工並正式對外開放<ref>{{Cite web |url=http://nga.gov.au/AboutUs/building/history.cfm |title=History of the building |publisher=澳洲國立美術館 |accessdate=2012-10-25 |language=en}}</ref>。美術館現有館藏16萬件，除了當代藝術和西洋藝術品外，也典藏了澳洲原住民的工藝創作<ref name="report">{{Cite web |url=http://www.nga.gov.au/AboutUs/Reports/NGA_AR_11-12.pdf |title=National Gallery of Australia Annual Report 2011-12 |publisher=澳洲國立美術館 |accessdate=2012-10-25 |format=PDF |page=42 |language=en}}</ref>。
