'''澳洲航空1號班機'''（QF1，QFA1）是由[[澳洲航空|澳洲航空]]運營的一班由[[悉尼|悉尼]]始發，目的地為倫敦的定期航班，曾於1999年9月23日在曼谷[[廊曼國際機場|廊曼國際機場]]經停時因跑道濕滑衝出跑道，飛機起落架損毀。

== 航班 ==
澳洲航空往來澳大利亞與倫敦之間，經停[[東半球|東半球]]的航線被稱為“[[袋鼠航線|袋鼠航線]]”，QF1號班機便是其中之一。1999年9月23日當天，該航班於澳洲當地時間16時45分從[[悉尼機場|悉尼機場]]起飛，預定在八個多小時的飛行之後於泰國當地時間22時45分降落曼谷廊曼國際機場。

== 事故經過 ==
1999年9月23日晚，VH-OJH執行QFA1號班機，準備向曼谷機場的21L跑道進近，當時曼谷正經歷雷雨天氣。22時40分，曼谷機場的能見度為1500米，此時[[泰國國際航空|泰國國際航空]]的一架[[空中巴士A330|空中巴士A330]]正常降落。22時43分，澳洲航空的另一班機QFA15（雪梨經曼谷往羅馬）因進近時能見度過低而復飛。<ref name="TMWK QANTAS">{{Cite book| last=Benns| first=Matthew| title=The Men Who Killed Qantas| publisher=William Heinemann:Australia| place=Sydney| year=2009| isbn=978-1-74166-891-9}}</ref>{{rp|1}}然而，QFA1號班機的機師並未留意以上事實。22時44分，塔台告知機組跑道濕滑的情況。

QFA1號班機在最後進場時由副機長操縱，高度降到200英尺以下，這時廊曼機場雨勢轉急，跑道燈光只能在駕駛艙外玻璃雨刷器掃過之後隱約可見。飛機通過跑道入口時速度為169節，高度為76英尺。<ref name="ASN"/>在飛機離地只有10英尺時，機長看不見跑道末端，要求副機長執行[[重飛|復飛]]<ref>{{cite web|title=资料：1999年世界民航事故|url=http://news.sina.com.cn/world/2000-06-22/100007.html|publisher=新浪网|date=2000-06-22|accessdate=2013-10-05}}</ref>，副機長隨即推動推力操縱杆，但沒有按壓TO/GA（起飛/復飛）電門，飛機主起落架接地。之後，機長自行將推力操縱杆復位，在沒有發出口令指示的情況下取消了復飛，[[推力反向器|推力反向器]]也沒有開啟。

由於機長取消重飛，飛機直至在跑道上滑行約1625米後才減速，但偏出了跑道中心線。機組人員採用了降落時的[[標準作業程序|標準作業程序]]：襟翼角度為25度（而非最大的30度）<ref name="TMWK QANTAS"/>{{rp|17}}，無反推，無自動制動，但此程序不適用於在積水跑道的降落。事後調查顯示，飛機滑行時，機輪在積水的跑道上發生了打滑。由於推力反向器未開啟，機組人員無法使飛機停在跑道盡頭，飛機以88節的速度衝出跑道盡頭100米長的停止道，繼續在跑道外的草地上滑行220米，撞上地面的無線電天線之後停住，機鼻觸及跑道邊界的道路，而路的另一側是一個高爾夫球場。

== 涉事客機 ==
此次衝出跑道的波音747-400於1990年8月14日首飛，生產序列號為24806，裝有四台[[勞斯萊斯RB211|勞斯萊斯RB211-524G]]引擎，同年9月交付給澳洲航空。事發前，該機飛行小時數為41151，起降次數為6002。<ref name="ASN">{{cite web |title=Accident description |publisher=aviation-safety.net |url=http://aviation-safety.net/database/record.php?id=19990922-0 |accessdate=2013-10-05}}</ref>

===损毁情况===
飛機與天線相撞之後，飛機的前起落架和右起落架損壞，前起落架被收回，機身前側及右側塌下，右側的兩個引擎及引擎掛架也因此受損。

機上人員在飛機急停後20分鐘內全部撤離，38人輕傷，無人重傷。涉事客機修復後於2000年4月8日重新投入運營<ref>{{cite web|title=The Qantas Source - Boeing 747-400.|url=http://www.theqantassource.com/b744.html|publisher=The QANTAS Source|accessdate=2013-10-05|deadurl=yes|archiveurl=https://web.archive.org/web/20131006174404/http://www.theqantassource.com/b744.html|archivedate=2013-10-06}}</ref>，2012年9月退出現役，同年10月在美國被拆解。

事後，澳洲航空在悉尼至倫敦的航線上繼續使用QF1這一航班號，但機型由波音747-400改為[[空中客车A380|空中客车A380]]。2013年4月起，因澳航與[[阿聯酋航空|阿聯酋航空]]合作，QF1號班機的中停點曾改為[[迪拜國際機場|杜拜]]<ref>{{cite web|title=澳洲航空公司 (QF) #1 ✈ 2013年 09月 05日 ✈ YSSY / SYD - OMDB / DXB 航班跟蹤 ✈ FlightAware|url=http://zh.flightaware.com/live/flight/QFA1/history/20130905/0550Z/YSSY/OMDB|publisher=FlightAware|accessdate=2013-10-05}}</ref>，但已在2018年3月25日改同新加坡。
