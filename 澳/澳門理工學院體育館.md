'''澳門理工學院體育館'''（{{lang-pt|'''Pavilhão Polidesportivo do Institito Politécnico de Macau'''}}）位於[[澳門|澳門]]理工學院街的一座體育場館，現屬澳門[[體育發展局|體育發展局]]轄下設施；體育館於2003年10月30日由[[澳門行政長官|澳門行政長官]][[何厚鏵|何厚鏵]]主持開幕。

==位置==
澳門理工學院體育館位於澳門[[新口岸|新口岸]]理工學院街，座落於[[澳門理工學院|澳門理工學院]]校園之內並與澳門理工學院匯智樓相連，佔地6,212平方米<ref name="open" />，體育館設有[[行人天橋|行人天橋]]和[[行人隧道|地下行人隧道]]與毗鄰的[[澳門綜藝館|澳門綜藝館]]互相連接<ref name="open" />。

==歷史==
耗資1.201億澳門元興建的澳門理工學院體育館連同澳門理工學院匯智樓一併興建{{notetag|1=建築費為1.16億澳門元，設計費為410多萬<ref name="cost">{{cite web |language=zh-hant |publisher=[[澳門日報|澳門日報]] |date=2004年10月29日 |title= Ｏ五東亞運主要場館設備先進環保 理工體育館明開幕 }}</ref>}}<ref>{{cite web |language=zh-hant |publisher=[[市民日報|市民日報]] |date=2002年11月28日 |title= 體發局長蕭威利在理工學院體育館奠基禮上稱重整南灣湖辦東亞運水上比賽 }}</ref>，於2002年11月7日由[[2005年東亞運動會|第四屆東亞運動會澳門組織委員會]]主席[[蕭威利|蕭威利]]、[[運輸及工務司司長|運輸及工務司司長]][[歐文龍|歐文龍]]、[[社會文化司司長|社會文化司司長]][[崔世安|崔世安]]以及[[土地工務運輸局|土地工務運輸局]]局長[[賈利安|賈利安]]共同主持奠基儀式<ref name="build">{{cite web |language=zh-hant |publisher=[[華僑報|華僑報]] |date=2002年11月8日 |title= 理工體館奠基料明秋竣工 將作為東亞運動會主場館  }}</ref>。2003年8月，澳門理工學院體育館按第84/2003號社會文化司司長批示被列入為[[體育發展局|體育發展局]]轄下體育設施之一<ref>{{cite journal|journal=澳門政府公報 |date=2003年8月25日 |issue=34 |title=第84╱2003號社會文化司司長批示 |url=http://images.io.gov.mo/bo/i/2003/34/despsasc-84-2003.pdf }}</ref>。

[[File:Pavilhão_Polidesportivo_do_Institito_Politécnico_de_Macau_13.jpg|thumb]]
2003年10月30日，[[澳門行政長官|澳門行政長官]][[何厚鏵|何厚鏵]]為澳門理工學院體育館暨理工學院綜合樓主持開幕典禮，是繼[[澳門奧林匹克游泳館|澳門奧林匹克游泳館]]後第二個落成的[[澳門東亞運動會|澳門東亞運動會]]體育場館<ref name="open">{{cite web |language=zh-hant |publisher=[[澳門日報|澳門日報]] |date=2003年10月31日 |title=理工體育館昨開幕 設施完善有助推動體育事業發展 }}</ref>；同日由第四屆東亞運澳門組委會接管<ref>{{cite web |language=zh-hant |publisher=[[正報|正報]] |date=2003年10月17日 |title=理工體育館落成 }}</ref>，而該組委會也於翌月2日起在體育館開設澳門首家以運動主題的精品店<ref>{{cite web |language=zh-hant |publisher=[[市民日報|市民日報]] |date=2003年11月3日 |title=2005精品廊昨開幕 }}</ref>。

澳門理工學院體育館於2005年至2007年曾作為[[2005年東亞運動會|第四屆東亞運動會]]<ref>{{cite news|language=zh-hant |publisher=[[澳門日報|澳門日報]] |date=2005年10月28日 |title=東亞運場館簡介 }}</ref>、[[2006年葡語系運動會|第一屆葡語系運動會]]以及[[2007年亞洲室內運動會|第二屆亞洲室內運動會]]的比賽場館<ref>{{cite news|language=zh-hant |publisher=[[澳門日報|澳門日報]] |date=2006年10月11日 |title=澳男女排對決葡國 }}</ref><ref>{{cite news|language=zh-hant |publisher=[[澳門日報|澳門日報]] |date=2007年6月27日 |title=亞室運設 477獎牌 }}</ref>。2006年1月24日起，澳門理工學院體育館按第1/2006號行政法規被歸入體育發展局轄下體育設施<ref>{{cite web |language=zh-hant |publisher=[[澳門日報|澳門日報]] |date=2006年1月24日 |title=體發局的組織及運作 }}</ref>。澳門理工學院體育館也舉辦澳門書市嘉年華<ref>{{cite news|language=zh-hant |publisher=[[新華澳報|新華澳報]] |date=2011年7月23日 |title=澳門書市嘉年華開幕 }}</ref>、亞洲體育舞蹈錦標賽等活動及賽事<ref>{{cite news|language=zh-hant |publisher=[[濠江日報|濠江日報]] |date=2010年10月3日 |title=亞洲舞林高手聚本澳 IDSF昨開賽 }}</ref>。

==設施==
澳門理工學院體育館樓高四層及兩層地下停車場<ref name="open" />，其建築面積超過17,339.12平方米{{notetag|1=澳門理工學院官方網頁指該建築面積為17,339.12平方米，而由[[澳門東亞運|澳門東亞運組委會]]於2005年出版的《澳門理工學院體育館暨綜合樓》官方刊物中表示該建築面積為19,255.3平方米<ref name="venue" />}}。體育館設有3,742個座位{{notetag|1=3,742個座位中，2,625個為固定座位、181個貴賓座、168為記者席以及768個自由拉動式的座位<ref name="venue" />}}；其比賽場地淨高18.5米，而面積則為1440平方米（若而把自由拉動式的座位拉出後，面積也有990平方米<ref name="venue">{{cite web |language=zh-hant |publisher=[[澳門理工學院|澳門理工學院]] |accessdate=2012年2月2日 |title=場地資料 |url=http://www.ipm.edu.mo/cweb/stadium/index_iframe.html#place }}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>）；體育館主要用作澳門理工學院體育暨運動高等學校的教學場地外，也可作[[籃球|籃球]]、[[排球|排球]]、[[室內運動|室內運動]]等體育項目比賽用途<ref name="build" />；也設有記者室、評述室、運動員及裁判員專用室等設備。體育場館設計採用滲透式冷暖氣調節系統及裝有電動隔光幕簾以有效控制天然採光強度，以節省能源<ref name="cost" />。澳門理工學院體育館的兩層地下停車場共設有106個車位<ref name="cost" />。
