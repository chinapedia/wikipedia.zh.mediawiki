'''伸缩像素显示器'''（{{lang-en|'''Telescopic pixel display'''}}，简称'''TDP'''）是一种新型显示技术，介于[[液晶显示器|液晶显示器]]（LCD）和{{tsl|en|Digital micromirror device|数字微镜设备}}（DMD）（基于[[数字光处理|数字光处理]]投影仪）之间，同时具有两种新型显示器的优点。

== 工作原理 ==
伸缩像素的设计原理基于[[光學望遠鏡|光學望遠鏡]]。“伸缩”是指每个[[像素|像素]]都是一个由主镜和次镜组成的微型望远镜，主镜和次镜可以通过施加特定电压而变形。

当像素为关闭时，主镜和次镜为平行，因此两者都阻止了[[光|光]]的通过，使其反射[[背光|背光]]，这样像素看起来为暗。当[[像素|像素]]为开启时，主镜变形为抛物线形状，将光聚焦到次镜上。在从次镜反射后，[[光|光]]透过薄膜中的孔传播并且像素出现[[明度|明度]]。

== 优势 ==
伸缩式[[像素|像素]]阵列的性能测试表明，它们对未来的[[显示器|显示器]]具有实质性前景。[[背光|背光]]透射率经测量为36%，而模拟表明设计改进后可以达到56%。在一个有5小时续航时间的现代[[筆記型電腦|筆記型電腦]]上，这种效能提高可以多增加45分钟的额外续航时间而无需降低屏幕[[明度|亮度]]。[[像素|像素]]响应时间为0.625毫秒——快于[[液晶显示器|LCD]]的2至10毫秒响应时间。这样的响应时间也足够进行顺序颜色处理，通过对每个[[像素|像素]]显示红色、蓝色和绿色的快速脉冲来显示颜色，简化工艺和设备设计。伸缩像素的其他优点是潜在的低成本以及相对容易制造和控制。

== 缺点 ==
TPD的唯一缺点是对比度。使用非准直[[光|光]]进行的实验测量显示出非常低的[[對比度|對比度]]，仅有20:1。模拟表示最高达800:1的[[對比度|對比度]]是可能的，但这用目前的[[液晶显示器|LCD]]技术也能做到。等离子和[[有機發光二極體|有機發光二極體]]具有更高的对比度，因此仍需大量工作才能与这些技术竞争。
