'''迴圈量子重力論'''（{{lang|en|loop quantum gravity}}，{{lang|en|LQG}}），又譯-{zh-hans:'''回圈量子引力论'''; zh-hant:'''環圈量子重力論''';}-，英文別名'''圈引力'''（loop gravity）、'''量子幾何學'''（{{lang|en|quantum geometry}}）；由[[阿贝·瓦桑特·阿希提卡|阿貝·阿希提卡]]、[[李·斯莫林|李·斯莫林]]、[[卡洛·羅威利|卡洛·羅威利]]等人發展出來的[[量子引力|量子引力]]理論，与[[弦理论|弦理论]]同是當今將[[量子重力|重力量子化]]最成功的理論。

利用[[量子场论|量子场论]]的[[微扰理論|微扰理論]]来实现引力论的量子化的理论是不能被[[重整化|重整化]]的。如果主張[[时空|时空]]只有[[四维|四维]]而從[[廣義相對論|廣義相對論]]下手，结果可以把廣義相對論转变成类似[[規範場論|規範場論]]的理論，基本正則變量为{{link-en|阿希提卡-巴貝羅聯絡|Ashtekar-Barbero Connection}}而非[[度规张量|度规张量]]，再以联络定义的[[平移算子|平移算子]]（holonomy）以及{{link-en|通量變數|flux variable}}为基本變量來實現[[量子化|量子化]]。

在此理論下，時空描述是呈[[背景獨立|背景獨立]]，由關係性[[迴圈|迴圈]]織出的[[自旋網路|自旋網路]]鋪成時空[[幾何|幾何]]。網絡中每條邊的[[長度|長度]]為[[普朗克長度|普朗克長度]]。迴圈並不存在於時空中，而是以迴圈扭結的方式定義時空幾何。在[[普朗克尺度|普朗克尺度]]下，時空幾何充滿隨機的[[量子漲落|量子漲落]]，因此自旋網絡又稱為[[自旋泡沫|自旋泡沫]]。在此理論下，時空是[[離散|離散]]的。

== 迴圈量子引力通論與目標 ==
多數[[弦論學家列表|弦論學家]]相信無法在[[時空|3+1維時空]]中，將引力量子化而不產生物質與能量有關的人工產物。然而[[弦論|弦論]]所預測的物質有關的人工產物也未被證明是否真的與實際觀測到的物質不相同。不過若迴圈量子引力成功地成為引力的量子理論，則已知的物質場必須「事後」再加到此一理論中，而不是從理論中自然而然地出現。迴圈量子引力論的創始者之一[[李·斯莫林|李·斯莫林]]已思索過弦論與迴圈量子引力兩者可能分別是一個終極理論兩相不同的近似這樣的可能性。
[[Image:Spinnetwork.jpg|right]]形态]]

目前迴圈量子引力聲稱具有的成功之處有：
# 其為3維空間幾何的[[微擾理論_(量子力學)|非微擾]][[量子化_(物理學)|量子化]]，具有量子化的面積與體積[[算符|算符]]。
# 其包含了對於[[黑洞|黑洞]][[熵|熵]]的計算。
# 其為弦論以外另一可行的理論，但僅只涉及引力的量子化（即非[[萬有理論|萬有理論]]）。

然而，這樣的聲稱尚未被完全接受。雖然許多迴圈量子引力的核心成果都是來自於嚴謹的[[數學物理|數學物理]]，不過它們的物理詮釋仍多為推敲性質。迴圈量子引力是有可能成為引力或者是幾何的改進方案；舉例來說，(2)中的熵計算事實上是針對一種形式的「洞」來做的，這個洞可能是，也可能不是黑洞。

量子引力的其他方案，比如[[自旋泡沫|自旋泡沫]]模型，與迴圈量子引力密切相關。

== 迴圈量子引力的假设 ==

迴圈量子引力的两个最重要的假设为
# [[广义协变|广义协变]] - 物理学的定律可以用任何的坐标系来表示，这也是[[广义相对论|广义相对论]]的基本假设。
# [[背景獨立|背景獨立]] - 不存在可以作为背景的独立不变的度规，坐标系等。

迴圈量子引力也假设[[量子论|量子论]]的基本原理是正确的。举例广义协变的理论有[[广义相对论|广义相对论]]，非广义协变的理论有[[狭义相对论|狭义相对论]]（[[狭义协变|狭义协变]]），非背景獨立的理论有[[牛顿力学|牛顿力学]]（假设存在一条独立不变的时间轴），[[狭义相对论|狭义相对论]]（其背景为[[闵可夫斯基空间|闵可夫斯基空间]]，背景度规为[[闵可夫斯基度规|闵可夫斯基度规]]），在背景电磁场中运动的电子的方程等，背景獨立的理论有[[广义相对论|广义相对论]],度规张量的值完全由理论决定。

== 迴圈量子引力的基本内容 ==

迴圈量子引力可以从[[广义相对论|广义相对论]]的[[ADM形式|ADM表示法]]推导。ADM表示法的正则变数为[[三维空间|三维空间]]的度规张量<math>q_{ab}</math>以及其[[正则动量|正则动量]]<math>P^{ab}</math>。使用[[狄拉克约束|狄拉克约束]]处理方法可得ADM表示法有两个[[第一类约束|第一类约束]]：

# [[微分同胚约束|微分同胚约束]]: <math>-2q^{-\frac{1}{2}}\;^3\nabla_a P^a_{\ b} = 0</math>
# [[哈密顿约束|哈密顿约束]]: <math>-^3R + q^{-1}(P^{ab}P_{ab} - \frac{1}{2}P^2) = 0</math>

=== 阿希提卡-巴贝罗联络 ===
此时用卡当的几何法，用[[三足一次型|三足（triad）一次型]]来表示度规张量,

<math>e^i_a e^j_b \delta_{ij} = q_{ab}</math>

假设与[[三足相容|三足相容]]的[[联络|联络]]为<math>\Gamma^i_a</math>(称为[[自旋联络|自旋联络]]),三维空间的[[外部曲率张量|外部曲率张量]]为<math>K^i_a</math>,<math>\gamma</math>为任何实数，定义一个新的联络

<math>A^i_a = \Gamma^i_a + \gamma K^i_a</math>

即[[阿希提卡-巴贝罗联络|阿希提卡-巴贝罗联络]]。其正则动量为<math>E^a_i = det(e)e^a_i</math>。使用狄拉克约束处理方法可得三个第一类约束:

# [[高斯约束|高斯约束]]: <math>D_aE^a_i = 0</math>
# [[微分同胚约束|微分同胚约束]]: <math>F^i_{ab}E^a_i = 0</math>
# [[哈密顿约束|哈密顿约束]]: <math>\frac{1}{\sqrt{|det(E)|}}F^i_{ab}E^a_jE^b_k\epsilon_{ijk} + ... = 0</math>

<math>D_a</math>为阿希提卡-巴贝罗联络定义的[[协变微商|协变微商]]，<math>F^i_{ab}</math>为阿希提卡-巴贝罗联络定义的[[曲率张量|曲率张量]].由于有高斯约束的关系,所以迴圈量子引力是一种类似[[规范场论|规范场论]]的理论.
