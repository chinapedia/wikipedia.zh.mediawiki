《'''迴響天空／每個女孩子啊'''》（{{lang|ja|空が鳴っている／女の子は誰でも}}，''Sora ga Natteiru/Onna no Ko wa Dare Demo''；''The Reverberation/Fly Me To Heaven''）是[[東京事變|東京事變]]的第七張[[單曲|單曲]]，由[[EMI音樂_(日本)|EMI Music Japan]]／[[維京唱片|Virgin Music]]於2011年5月11日發行<ref name="JPN" />，也是東京事變首張雙Ａ面單曲。同時「迴響天空」是[[江崎固力果|江崎固力果]]『Watering Kiss Mint』口香糖廣告的主題曲<ref name="TheReverberationDRAMASONG" />，而「每個女孩子啊」則作為資生堂「MAQuillAGE」系列的廣告主題曲<ref name="FlyMeToHeavenDRAMASONG" />，兩者皆由主唱椎名林檎擔任產品的代言人。初回限定為《[[Discovery_(東京事變)#演唱會簡介|東京事變 live tour 2011 Discovery]]》巡迴演唱會門票優先抽選券。

在單曲銷售成績方面，本張單曲在日本Oricon銷售榜上最高位居單週第6名<ref name="SoragaNatteiruOriconWeek" />，名列2011年年度銷售榜第142位<ref name="2011salessingle" />。而在日本唱片協會所公佈的RIAJ數位下載榜中，歌曲「迴響天空」最高曾名列單週第40名<ref name="SoragaNatteiruRIAJDigitalChart" />，「每個女孩子啊」最高則曾名列單週第26名<ref name="OnnanoKowaDareDemoRIAJDigitalChart" />。

== 單曲概要 ==
《'''迴響天空／每個女孩子啊'''》是東京事變繼2010年7月發表數位單曲《[[DOPA-MINT!|DOPA-MINT!]]》九個月後，也是繼2009年12月發行單曲《[[能動的三分間|能動的三分間]]》一年多後的新作品。同時這張單曲也是東京事變及椎名林檎歌手生涯中第一次發行雙Ａ面單曲。在歌曲部分，這次主唱椎名林檎特別選用「女孩子」作為歌曲的標題，這樣的選擇在椎名林檎的歌手生涯中十分罕見，因為關於女性的詞彙中，椎名林檎連「雌」這個字都用過了，但是這次卻選用「女孩子」來代表女性<ref name="SoragaNatteiruOnnanoKowaDareDemoInterview">{{cite web |url=http://www.tokyojihen.com/sp/notes/interview/toct40320/ |title=「空が鳴っている／女の子は誰でも」オフィシャル・インタビュー |publisher=EMI |language=ja |deadurl=yes |archiveurl=https://web.archive.org/web/20140622083013/http://www.tokyojihen.com/sp/notes/interview/toct40320/ |archivedate=2014-06-22 }}</ref>。對此，椎名林檎表示以前會堅持不用許多詞彙，主要是想要確立自己及自我認同兩個部分，但是她已經過了那個時期，因此對於現在的她來說，不同詞彙的使用代表的是音樂特性上的不同，就像是「女人」、「少女」、「女子」等，它們的概念不都不一樣嗎？椎名林檎更進一步的表示按照這首歌的概念，如果不用女孩子來命名的話是無法成立的<ref name="SoragaNatteiruOnnanoKowaDareDemoInterview" />。

在單曲發售時程上，新單曲發行的消息最早於2010年12月10日就在官網發布<ref>{{cite web|url=http://natalie.mu/music/news/41860 |title=東京事変、2月に大型タイアップ付きシングルを発表 |publisher=natalie |language=ja}}</ref>。到了2011年1月20日日本資生堂則宣布椎名林檎將代言‘MAQuillAGE’系列產品，廣告曲「每個女孩子啊」則由東京事變演唱<ref>{{cite web |url=http://www.vibe-net.com/news/?news=2005976 |title=東京事変 CM出演&新曲がCM音楽に抜擢！！ |publisher=vibe-net |language=ja |deadurl=yes |archiveurl=https://web.archive.org/web/20160305045530/http://www.vibe-net.com/news/?news=2005976 |archivedate=2016-03-05 }}</ref>。在02月8日，官方網站不但宣布歌曲「迴響天空」將成為江崎固力果公司廣告主題曲，同一天也開始提供歌曲「迴響天空」數位下載的服務<ref name="JPNDigitalPrevious">{{cite web|url=http://www.tokyojihen.com/news/ |title=2/23発売ニューシングルより新曲「空が鳴っている」モバイル先行配信開始！|publisher=EMI |language=ja}}</ref>。但受到鼓手刄田綴色喝酒鬧事影響，新單曲發售時程延期<ref name="SoragaNatteiruOnnanoKowaDareDemoReleaseDelayNews">{{cite web|url=http://natalie.mu/music/news/44912 |title=東京事変ドラマー刄田綴色が公務執行妨害の容疑で逮捕 |publisher=natalie |language=ja}}</ref><ref name="SoragaNatteiruOnnanoKowaDareDemoReleaseDelaySorry">{{cite web|url=http://www.kronekodow.com/gaiko/20110212.html |title=この度の刄田綴色の不祥事について |publisher=黑貓堂 |language=ja}}</ref>，改至5月11日發售<ref name="JPN" />。另外，原訂發售時，並無附任何的特典。但後來於5月發行時，官方決定隨單曲附贈《東京事變 live tour 2011 Discovery》巡迴演唱會門票優先抽選券。台灣則於05月13日發行台壓版<ref name="TWN" />，等到05月20日才代理進口初回版<ref name="TWNJPN" />。

在歌曲銷售方面，歌曲「迴響天空」首先在2011年2月8日提供數位下載<ref name="JPNDigitalPrevious" />，並在日本唱片協會所公佈的RIAJ數位下載榜中，最高名列單週第40名<ref name="SoragaNatteiruRIAJDigitalChart" />。後來，因為受鼓手鬧事所影響，在02月12日暫停數位下載的服務<ref name="JPNDigitalPrevious" />。到了4月20日，「迴響天空」及「每個女孩子啊」兩首歌曲再次接受歌迷下載<ref name="JPNDigital" />，而歌曲「每個女孩子啊」則在RIAJ數位下載榜中，最高名列第26名<ref name="OnnanoKowaDareDemoRIAJDigitalChart" />。而在單曲銷售方面，本張單曲在發行當週以4.1萬張的銷售量在日本Oricon銷售榜上位居第6名<ref name="SoragaNatteiruOriconWeek" />，總計在Oriocn銷售榜上榜8週<ref name="SoragaNatteiruOriconConsecutiveWeek" />，累積銷售達5.4萬張，名列2011年年度銷售榜第142名<ref name="2011salessingle" />。

=== 發售延期 ===
本張單曲原定2月23日發行，但是在2月12日時，鼓手刃田綴色在酒後鬧事，並涉嫌妨礙警方執行公務而遭逮捕，東京事變立刻發表聲明道歉，並宣佈東京事變及新單曲發行時程無限期暫停<ref name="SoragaNatteiruOnnanoKowaDareDemoReleaseDelayNews" /><ref name="SoragaNatteiruOnnanoKowaDareDemoReleaseDelaySorry" />。而受鼓手鬧事所累，不但唱片公司暫停歌曲「迴響天空」線上下載的服務<ref>{{cite web |url=http://www.vibe-net.com/news/?news=2006208 |title=東京事変 椎名林檎出演CMが本日よりリニューアル！ |publisher=vibe-net |language=ja |deadurl=yes |archiveurl=https://web.archive.org/web/20120612211438/http://www.vibe-net.com/news/?news=2006208 |archivedate=2012-06-12 }}</ref>，連原本預定連續兩週要上朝日電視台音樂節目「Music Station」的計畫也一併取消<ref name="MS">{{cite web |url=http://www.daily.co.jp/gossip/article/2011/02/13/0003801006.shtml |title=東京事変のＴＶ出演キャンセル |publisher=DSO |language=ja |deadurl=yes |archiveurl=https://web.archive.org/web/20110217153334/http://www.daily.co.jp/gossip/article/2011/02/13/0003801006.shtml |archivedate=2011-02-17 }}</ref>。另外，以新單曲中的歌曲作為廣告曲的資生堂「MAQuillAGE」系列及江崎固力果「Watering Kiss Mint口香糖」廣告也將原本的歌曲改由椎名林檎單獨演唱，分別是「迴響天空〜深海版〜」（無伴奏版）及「每個女孩子啊〜素肌編〜」（鋼琴演奏）<ref name="SoragaNatteiruOnnanoKowaDareDemoReleaseDelayThemeSongChange">{{cite web|url=http://headlines.yahoo.co.jp/hl?a=20110216-00000001-dal-ent|title=椎名林檎ＣＭ曲ソロに…メンバー逮捕余波|publisher=yahoo|language=ja|archiveurl=https://web.archive.org/web/20110219065115/http://headlines.yahoo.co.jp/hl?a=20110216-00000001-dal-ent|archivedate=2011-02-19|deadurl=yes}}</ref>。

日後，配合單曲再次發售，江崎固力果公司將廣告配樂從椎名林檎清唱的版本，改為東京事變的歌曲「能動的三分間」，而不是恢復採用歌曲「迴響天空」。而資生堂則是將原本東京事變所演唱的歌曲「每個女孩子啊」，再度套入新的廣告中。在節目宣傳方面，先前因故取消錄影的音樂節目「Music Station」，在單曲發行後，改以椎名林檎為名義參加演出<ref name="MS2">{{cite web |url=http://news.livedoor.com/article/detail/5564902/ |title=今週Mステに椎名林檎、Perfume、Superfly、西野カナら |publisher=natalie |language=ja |deadurl=yes |archiveurl=https://web.archive.org/web/20110603030527/http://news.livedoor.com/article/detail/5564902/ |archivedate=2011-06-03 }}</ref>。

=== 音樂錄影帶 ===
歌曲「迴響天空」的音樂錄影帶由兒玉裕一擔任製作，他同時也是廣告的導演。音樂錄影帶的概念是以自我探詢的影像為主，散發出戀物及戲劇性。

=== 跨業合作 ===
[[File:Shiina_CM_Glico4.JPG|thumb]]
*;「迴響天空」
:（江崎固力果）‘Watering Kiss Mint口香糖’的廣告主題曲<ref name="TheReverberationDRAMASONG">{{cite web|url=http://natalie.mu/music/news/44659 |title=演技はすべて逆回転！椎名林檎出演「キスミント」新CM |publisher=natalie |language=ja}}</ref>
椎名林檎參與了江崎固力果『Watering Kiss Mint』口香糖廣告的演出，為此系列第四個電視廣告。廣告由兒玉裕一擔任製作，他同時也是本首歌音樂錄影帶的導演。導演為了與主題曲「迴響天空」的歌詞「慢動作」相互呼應，連帶的廣告也採慢動作播放。在這個廣告中，導演使用[[高速攝影機|高速攝影機]]拍攝椎名林檎在水中吃口香糖的畫面，並將不同顏色的口香糖口味用霓虹燈的方式映射至背景上，而當椎名林檎開始在水中吹口香糖氣泡時，剎那間的靜默代表著如夢幻般靜謐的時光
<ref name="TheReverberationDRAMASONG" />。這樣的拍攝手法也讓椎名林檎吃不消，大喊「這好難喔<ref>{{cite web|url=http://www.news24.jp/entertainment/news/1616157.html |title=椎名林檎、高難度演技に「難しい～」ＣＭ曲は東京事変 |publisher=news24 |language=ja}}</ref>!」

但後來受鼓手刄田綴色酒後鬧事的影響，江崎固力果公司將原本的廣告配樂改由椎名林檎單獨演唱，取名為「迴響天空〜深海版〜」（無伴奏版）<ref name="SoragaNatteiruOnnanoKowaDareDemoReleaseDelayThemeSongChange" />。在單曲重新發售後，江崎固力果公司並不是把歌曲「迴響天空」重新當作廣告主題曲，反而是採用東京事變以前的歌曲「能動的三分間」。

[[File:Shiina_CM_MAQuillAGE.JPG|thumb]]
*;「每個女孩子啊」
:（資生堂）‘MAQuillAGE’系列廣告主題曲<ref name="FlyMeToHeavenDRAMASONG">{{cite web|url=http://natalie.mu/music/news/43689 |title=椎名林檎資生堂CMモデルに抜擢、音楽は東京事変が担当 |publisher=natalie |language=ja}}</ref>
椎名林檎參與了日本資生堂『MAQuillAGE』系列的電視廣告的演出，並於2011年2月20日發佈在官方網站上<ref name="FlyMeToHeavenDRAMASONG" />。這個廣告由兒玉裕一擔任製作，他同時也是本首歌音樂錄影帶的導演。廣告是在美國[[新墨西哥州|新墨西哥州]]著名景點[[白沙國家公園|白沙國家公園]]所拍攝，搭配椎名林檎的冷凝無垢的臉龐與獨特歌聲，成功帶來「純白無垢」的震撼感官<ref name="FlyMeToHeavenDRAMASONG" />。而事後發表單曲時，椎名林檎表示在拍資生堂廣告的時候十分緊張，因為她自認為不是被拍攝的專家，因此她總是盡她最好的一面去完成，希望能對廣告有些許貢獻<ref name="SoragaNatteiruOnnanoKowaDareDemoInterview" />。

但後來受鼓手刄田綴色酒後鬧事的影響，資生堂將還沒公開的廣告配樂改為椎名林檎單獨演唱，取名為「每個女孩子啊〜素肌編〜」（鋼琴演奏）<ref name="SoragaNatteiruOnnanoKowaDareDemoReleaseDelayThemeSongChange" />。直到單曲重新發售後，資生堂再將東京事變的版本，套入新的廣告中。

== 批評及評價 ==
在音樂網站''CDJournal''的音樂評論家認為「迴響天空」是首冷硬派搖滾曲風，同時帶點不安氣氛的歌曲，而吉他的伴奏更增加了緊張、快速的感覺，並且也凸顯出椎名林檎的歌聲<ref name="REVIEW1">{{cite web |url=http://artist.cdjournal.com/d/-/4110120572 |title=東京事変 / 空が鳴っている / 女の子は誰でも |publisher=CDJournal |language=ja}}</ref><ref name="REVIEW2">{{cite web |url=http://artist.cdjournal.com/d/-/4111050247 |title=東京事変 / 大発見 |publisher=CDJournal |language=ja}}</ref>。同時也提到「每個女孩子啊」則是一首帶著搖擺爵士管弦樂風格的曲目<ref name="REVIEW1" /><ref name="REVIEW2" />。音樂網站''HOTEXPRESS''的音樂評論家武川春奈則認為這首歌曲很有東京事變的風格，首次聽到的感覺就跟《[[群青日和|群青日和]]》所受到的衝擊相當，而她特別喜歡這首歌曲的歌詞和椎名林檎歌唱的歌聲<ref name="REVIEW3" >{{cite web |url=http://www.hotexpress.co.jp/review/3062/ |title=空が鳴っている/女の子は誰でも |publisher=Hot Express |language=ja}}</ref>。同時也提到「每個女孩子啊」則是一首「包含所有女孩子的元素」的歌曲，而當她也表示這樣的標題在椎名林檎的歌手生涯中相當少見，這也讓她想起了椎名林檎個唱時期的歌曲《[[幸福論|幸福論]]》（1998）、《[[在這裡接吻|在這裡接吻]]》（1999）及《兩人的時間》（2009）<ref name="REVIEW3"  />。

== 銷售排行 ==
=== Oricon 銷售榜(日本) ===
《迴響天空／每個女孩子啊》於2011年5月11日發行。發行首週，在日本公信榜Oricon的單曲榜2011年5月第2週付的榜單中以41,110張銷售量居單週第六名，同週第一名是銷售量突破15萬大關的男子組合[[關西傑尼斯8|關西傑尼斯8]]所發行的單曲「[[My_Home|My Home]]」，第二名則是女團體[[French_Kiss_(音樂團體)|French Kiss]]的單曲「[[太遜了I_love_you!|太遜了I love you!]]」<ref name="SoragaNatteiruOriconWeek" />。第二週則以7,646張銷售量位居第十七位，同週第一、二名分別是組合[[KAT-TUN|KAT-TUN]]的單曲「[[WHITE_(KAT-TUN單曲)|WHITE (KAT-TUN單曲)]]」及電子音樂組合[[Perfume|Perfume]]的單曲「[[Laser_Beam/微香|Laser Beam/微香]]」<ref>{{Cite web |url=http://www.oricon.co.jp/rank/js/w/2011-05-30/p/2/ |title=2009年5月第3週の邦楽シングルランキング情報 |publisher=Oricon |language=ja}}</ref>。總計《迴響天空／每個女孩子啊》在日本公信榜Oricon的單曲榜上榜11週，累積銷售達7.1萬張<ref name="SoragaNatteiruOriconConsecutiveWeek" />。同時在2011年年銷售榜中，《迴響天空／每個女孩子啊》以5.4萬張銷售量居全年第142名<ref name="2011salessingle" />。
