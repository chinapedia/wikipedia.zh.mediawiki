'''要町站'''（{{jpn|j=要町駅|hg=かなめちょうえき|rm=Kanamechō eki|f=y}}）是位於[[日本|日本]][[東京都|東京都]][[豐島區|豐島區]][[要町_(豐島區)|要町]]一丁目，屬於[[東京地下鐵|東京地下鐵]]的[[鐵路車站|鐵路車站]]。

此站有[[有樂町線|有樂町線]]、[[副都心線|副都心線]]交會。包括此站的[[小竹向原站|小竹向原站]]－[[池袋站|池袋站]]路段，有樂町線在上層，而副都心線在下層。因此此站的有樂町線月台正下方就是副都心線月台。有樂町線的[[車站編號|車站編號]]是'''Y 08'''、副都心線是'''F 08'''。

== 歷史 ==
* 1983年（[[昭和|昭和]]58年）6月24日：[[帝都高速度交通營團|營團地下鐵]]'''要町站'''隨有樂町線[[地下鐵成增站|營團成增]]（現稱地下鐵成增）至池袋段一同開業。
* 2003年（[[平成|平成]]15年）2月：成為業務委託站。
* 2004年（[[平成|平成]]16年）4月1日：營團地下鐵民營化，本站由東京地下鐵接手經營。
* 2008年（平成20年）6月：副都心線站區開業。
* 2011年（平成23年）1月10日：有樂町線月台啟用月台門。
* 2011年（平成23年）2月25日：導入發車音樂<ref>[http://www.switching.co.jp/news/archives/194 有楽町線の発車メロディを制作しました] {{webarchive|url=https://web.archive.org/web/20140724184255/http://www.switching.co.jp/news/archives/194 |date=2014-07-24 }} - 株式会社スイッチ 2011年4月6日</ref>。

== 車站結構 ==
要町站位於[[東京都道441號池袋谷原線|要町通]]地下，並與[[東京都道317號環狀六號線|山手通]]、[[首都高速中央環狀線|首都高速中央環狀線]][[山手隧道_(東京都)|山手隧道]]交會。由於本站結構與[[首都高速中央環狀線|首都高速中央環狀線]]及周邊地區的維生管線形成一體化的巨大地下構造，因此被稱為「要町盒子」、「環六盒子」。由於環六盒子的興建歷史緣故，本站月台比山手隧道更深。另外，副都心線月台下有四根山手隧道的換氣管，則是環六盒子進行擴建時增設的。


本站月台採上下2層[[島式月台|島式月台]]各1面2線設計，上方為有樂町線月台，下方為副都心線月台，與小竹向原－池袋段其他車站相同。其中副都心線月台實際上與有樂町線月台一同落成，但直到副都心線站區啟用時才開放使用。因此，有樂町線月台往副都心線月台的樓梯空間皆被遮蔽。剪票口至月台間的電梯雖能通至副都心線月台，但當時上僅開放至有樂町線本線（當時）月台。另外，隨著副都心線通車與有樂町線[[和光市站|和光市站]] - 池袋站間的準急開始運行。有樂町線月台的廣播機器也隨之更新，廣播內容加入列車編組長度。但有樂町線內的準急已在2010年3月廢止。

月台設備方面，初期有樂町線月台不設[[月台閘門|月台閘門]]，於2010年才安裝閘門，並於2011年1月8日啟用<ref>[http://www.tokyometro.jp/news/2010/2010-45.html 東京地下鐵新聞稿（2010年9月6日）]</ref>。而副都心線月台自啟用起已設有月台閘門。
