'''驯鹿'''（學名''Rangifer tarandus''），又名'''角鹿'''。是[[鹿科|鹿科]]'''驯鹿属'''下的唯一一種[[動物|動物]]。

== 特征 ==
不同的驯鹿亚种之间的形态有很大差距。总的来说，生活在南部地区的驯鹿要比北部的同类体形更大。驯鹿的肩宽可以达到120厘米，身长在1.5米到2.3米之间。雄性和雌性之间也有体形差异，某些亚种的雄性体形可以达到雌性的两倍，但雄性和雌性驯鹿头上都长角，这也是驯鹿区别于其它鹿种的显著特点之一<ref name="ADW">{{cite web|title =Rangifer tarandus(Caribou) |url=http://animaldiversity.ummz.umich.edu/site/accounts/information/Rangifer_tarandus.html|publisher= Animal diversity web |accessdate=2009-11-14}}</ref>；长角分枝繁复，有时超过30叉，蹄子宽大，悬蹄发达，尾巴极短。

驯鹿的身体上覆盖着轻盈但极为抗寒冷的毛皮。不同亚种、性别的毛色在不同的季节有显著不同，从雄性北美林地驯鹿在夏季时的深棕褐色，到[[格陵兰岛|格陵兰岛]]上的白色。主要毛色有褐色、灰白色、花白色和白色。花色中白色一般出现在腹部、颈部和蹄子以上部位<ref name="ADW"/>。

== 习性 ==
繁殖形态为[[胎生|胎生]]，每次产出幼仔需时''4-6''个月。　刚出生的驯鹿生长速度很快。母鹿一般在冬季怀胎，在春季迁移的途中生育。幼仔生下两三天后，就可以跟着母鹿赶路，一个星期之后，它们的速度就能像父母一样快，每小时可以迁移48公里。

== 分布 ==
驯鹿主要分布於北半球的环[[北极地区|北极地区]]，包括在[[欧亚大陆|欧亚大陆]]和[[北美洲|北美洲]]北部及一些大型岛屿。在中国，目前驯鹿只见于[[大兴安岭|大兴安岭]]东北部林区。

== 保护 ==
驯鹿已被列入[[中国国家林业局|中国国家林业局]]2000年8月1日发布的《国家保护的有益的或者有重要经济、科学研究价值的陆生野生动物名录》。

== 用途 ==
中国[[鄂温克族|鄂温克族]]使用驯鹿作为交通工具。另外[[圣诞节|圣诞节]]时候[[圣诞老人|圣诞老人]]的[[雪橇|雪橇]]也是靠驯鹿来拉的。

== 常见誤解 ==
拉雪橇用的是驯鹿而不是[[麋鹿|麋鹿]]。[[麋鹿|麋鹿]]是中国的特有物种，虽然后来有被带到世界其他地区，但是还没有早到出现在西方国家的古老传说中，而驯鹿在许多西方国家特别是寒冷地区都有所分布。<ref name="x">{{cite web|title =麋鹿vs驯鹿，到底哪个拉着圣诞老人的雪橇？ |url=http://blog.renren.com/blog/228567347/702263480 |publisher= 人人网 |accessdate=2014-12-26}}</ref>
