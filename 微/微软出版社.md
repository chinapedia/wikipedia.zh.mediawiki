'''微軟出版社'''（{{lang-en|Microsoft Press}}）是[[微軟|微軟]]負責針對其產品與技術出版著作的事業部門，著作包含了一般性的入門書、教學手冊、專家手冊、參考用書等等。[[微軟認證|MCP]]考試的參考用書也是由微軟出版社出版，稱為「自學教材」 (Self-Paced Training kit)。

微軟出版社成立于1983年<ref>[[马文方|马文方]]：《从无序到有序》。《[[中国计算机报|中国计算机报]]》1997年第39期。http://media.ccidnet.com/media/ciw/679/00750002.htm{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，是微软的直属分支机构<ref>《MS Press与人民邮电出版社携手打造中国金牌微软技术书籍》，微软公司，http://www.microsoft.com/china/learning/HotspotFocus/20071115.aspx {{Wayback|url=http://www.microsoft.com/china/learning/HotspotFocus/20071115.aspx |date=20080724120529 }}</ref>，目前配屬於Microsoft Learning事業群中。

==知名作者==

微軟出版社自成立至今，已培養出數位知名的技術作家，例如：

* Charles Petzold，以 Programming Windows 一書揚名國際，此書為公認的 Windows API 學習用書，目前已出至第六版。
* Jeff Prosise，其知名著作為 Programming Windows with MFC。
* Jeffrey Richter，以 Advanced Windows 一書揚名，被列為 Windows API 的進階讀物，目前則改名為 Programing Applications for Microsoft Windows。

==知名書籍==

* Programming Windows系列，自1988年出版以來，被公認是[[Windows_API|Windows API]]的聖經級學習與參考用書<ref>[http://www.charlespetzold.com/bio.html Charles Petzold Personal Site]</ref>。此書在台灣最早由松格資訊翻譯，並分為上下兩冊 (Programming Windows 95)，後由[[華彩軟體|華彩軟體]]接手，目前則是由[[文魁資訊|文魁資訊]]出版並銷售。
* Programming Windows with MFC系列，公認的[[MFC|MFC]]學習用書，此書曾由台灣的松格資訊翻譯為中文版，全書厚約1200頁，目前由文魁資訊出版。
* Windows Internals，以描述Windows核心的內容為主，全書描寫了Windows的系統層次架構，深受低階或系統層次程式開發人員的喜愛。
* Advanced Windows，以描述高階Windows API應用為主，後改名為Programming Applications for Microsoft Windows，與 Charles Petzold的Programming Windows 並列Windows開發人員必讀聖典之一。
