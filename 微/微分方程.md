'''微分方程'''（{{lang-en|Differential equation}}，'''DE'''）是一種[[數學|數學]][[方程|方程]]，用來描述某一類[[函数|函数]]與其[[导数|导数]]之间的关系。微分方程的解是一個符合方程的[[函數|函數]]。而在初等数学的代数方程裡，其解是常数值。

微分方程的应用十分广泛，可以解决许多与导数有关的问题<ref name="劉睦雄">
{{cite book
 |author=劉睦雄
 |coauthors=張任業
 |title=微分方程
 |year=1988
 |publisher=華泰書局
<!-- |isbn=書籍編號 -->}}</ref>{{rp|p.1}}。[[物理|物理]]中许多涉及变力的[[运动学|运动学]]、[[动力学|动力学]]问题，如空气的阻力為速度函數的落体运动等问题，很多可以用微分方程求解。此外，微分方程在[[化学|化学]]、[[工程学|工程学]]、[[经济学|经济学]]和人口统计等领域都有应用。

数学领域对微分方程的研究着重在几个不同的面向，但大多数都是关心微分方程的解。只有少数简单的微分方程可以求得[[解析解|解析解]]。不过即使没有找到其解析解，仍然可以确认其解的部份性质。在无法求得解析解时，可以利用[[数值分析|数值分析]]的方式，利用电脑来找到其数值解。 [[动力系统|动力系统]]理论强调对于微分方程系统的量化分析，而许多数值方法可以计算微分方程的数值解，且有一定的准确度。

== 分類 ==
微分方程可分為以下幾類，而隨著微分方程種類的不同，其相關研究的方式也會隨之不同。

===常微分方程及偏微分方程===
* [[常微分方程|常微分方程]]（ODE）是指一微分方程的未知數是單一自變數的函數<ref name="episte">
{{cite web
 | url=http://episte.math.ntu.edu.tw/entries/en_dif_equation/ 
 | author=翁秉仁
 | language=中文 
 | title=微分方程 
 | work=EpisteMath
 | publisher=中央研究院數學所、台大數學系 
 | accessdate=2014-01-15 
}}</ref>。最簡單的常微分方程，未知數是一個實數或是複數的函數，但未知數也可能是一個向量函數或是矩陣函數，後者可對應一個由常微分方程組成的系統。微分方程的表达通式是：

:<math>f\left(x, \frac{d^n y}{dx^n},\frac{d^{(n-1)} y}{dx^{(n-1)}},\cdots, \frac{dy}{dx}, y\right)=0</math>

:常微分方程常依其階數分類，階數是指自變數導數的最高階數<ref name="劉睦雄"/>{{rp|p.3}}，最常見的二種為一階微分方程及二階微分方程。例如以下的贝塞尔方程：

::<math>x^2 \frac{d^2 y}{dx^2} + x \frac{dy}{dx} + (x^2 - \alpha^2)y = 0</math>
:（其中y為[[應變數|應變數]]）為二階微分方程，其解為[[贝塞尔函数|贝塞尔函数]]。

* [[偏微分方程|偏微分方程]]（PDE）是指一微分方程的未知數是多個自變數的函數<ref name="episte"/>，且方程式中有未知數對自變數的[[偏微分|偏微分]]。偏微分方程的階數定義類似常微分方程，但更細分為[[橢圓算子|橢圓型]]、[[雙曲線偏微分方程|雙曲線型]]及[[抛物偏微分方程|拋物線型]]的偏微分方程，尤其在二階偏微分方程中上述的分類更是重要。有些偏微分方程在整個自變數的值域中無法歸類在上述任何一種型式中，這種偏微分方程則稱為混合型。像以下的方程就是偏微分方程：

:: <math> \frac{\partial u}{\partial t} + t\frac{\partial u}{\partial x} = 0. </math>

===線性及非線性===

常微分方程及偏微分方程都可以分為線性及非線性二類。

若微分方程中沒有出現自变數及微分項的平方或其他乘積項，也沒有出現应变數及其微分項的乘積，此微分方程為[[線性微分方程|線性微分方程]]，否則即為'''非線性微分方程'''。

'''齊次線性微分方程'''是線性微分方程中更細的分類，微分方程的解乘上一係數或是與另一個解相加後的結果仍為微分方程的解。

若線性微分方程的係數均為常數，則為'''常係數線性微分方程'''。常係數線性微分方程可以利用[[拉氏轉換|拉氏轉換]]轉換為代數方程<ref name="劉睦雄"/>{{rp|p.315-316}}，因此簡化求解的過程。

針對非線性的微分方程，只有相當少數的方法可以求得微分方程的解析解，而且這些方法需要微分方程有特別的[[對稱性|對稱性]]。長時間時非線性微分方程可能會出現非常複雜的特性，也可能會有[[混沌理论|混沌]]現象。有關非線性微分方程的一些基本問題，例如解的存在性、唯一性及初始值非線性微分方程的[[適定性問題|適定性問題]]，以及邊界值非線性微分方程都是相當難的問題，甚至針對特定非線性微分方程的上述基本問題都被視為是數學理論的一大突破。例如2000年提出的7個[[千禧年大獎難題|千禧年大獎難題]]中，其中一個是[[納維-斯托克斯存在性與光滑性|納維-斯托克斯存在性與光滑性]]，都是探討[[納維－斯托克斯方程式|納維－斯托克斯方程式]]其解的數學性質<ref>[http://www.claymath.org/millennium/Navier-Stokes_Equations/navierstokes.pdf Official statement of the problem] {{webarchive|url=https://web.archive.org/web/20120418004019/http://www.claymath.org/millennium/Navier-Stokes_Equations/navierstokes.pdf |date=2012-04-18 }}, Clay Mathematics Institute.</ref>，截至2018年8月此問題仍尚未被證明。

線性微分方程常常用來近似非線性微分方程，不過只在特定的條件下才能近似。例如單擺的運動方程為非線性的微分方程，但在小角度時可以近似為線性的微分方程。

=== 舉例 ===

以下是常微分方程的一些例子，其中<math>u</math>為未知的函數，自變數為<math>x</math>，<math>c</math>及<math>\omega</math>均為常數。

* 非齊次一階常係數線性微分方程：

:: <math> \frac{du}{dx} = cu+x^2. </math>

* 齊次二階線性微分方程：

:: <math> \frac{d^2u}{dx^2} - x\frac{du}{dx} + u = 0. </math>

* 描述[[諧振子|諧振子]]的齊次二階常係數線性微分方程：

:: <math> \frac{d^2u}{dx^2} + \omega^2u = 0. </math>

* 非齊次一階非線性微分方程：

:: <math> \frac{du}{dx} = u^2 + 1. </math>

* 描述長度為<math>L</math>的[[單擺|單擺]]的二階非線性微分方程：

:: <math> L\frac{d^2u}{dx^2} + g\sin u = 0. </math>

以下是偏微分方程的一些例子，其中<math>u</math>為未知的函數，自變數為<math>x</math>及<math>t</math>或者是<math>x</math>及<math>y</math>。

* 齊次一階線性偏微分方程：

:: <math> \frac{\partial u}{\partial t} + t\frac{\partial u}{\partial x} = 0. </math>

* [[拉普拉斯方程|拉普拉斯方程]]，是橢圓型的齊次二階常係數線性偏微分方程：

:: <math> \frac{\partial^2 u}{\partial x^2} + \frac{\partial^2 u}{\partial y^2} = 0. </math>

* [[KdV方程|KdV方程]]，是三階的非線性偏微分方程：

:: <math> \frac{\partial u}{\partial t} = 6u\frac{\partial u}{\partial x} - \frac{\partial^3 u}{\partial x^3}. </math>

== 性質 ==
=== 普遍性的數學描述 ===
許多[[物理|物理]]或是[[化學|化學]]的基本定律都可以寫成微分方程的形式。在[[生物學|生物學]]及[[經濟學|經濟學]]中，微分方程用來作為複雜系統的[[數學模型|數學模型]]。微分方程的數學理論最早是和方程對應的科學領域一起出現，而微分方程的解就可以用在該領域中。不過有時二個截然不同的科學領域會形成相同的微分方程，此時微分方程對應的數學理論可以看到不同現象後面一致的原則。

例如考慮光和聲音在空氣中的傳播，以及池塘水面上的波動，這些都可以用同一個二階的偏微分方程來描述，此方程即為[[波動方程|波動方程]]，因此可以將光和聲音視為一種波，和水面上的水波有些類似之處。[[約瑟夫·傅立葉|約瑟夫·傅立葉]]所發展的熱傳導理論，其統御方程是另一個二階偏微分方程－[[熱傳導方程式|熱傳導方程式]]，[[扩散作用|扩散作用]]看似和熱傳導不同，但也適用同一個統御方程，而經濟學中的布萊克-休斯方程也和熱傳導方程有關。

=== 微分方程的解 ===

微分方程的解通常是一个[[函数|函数]]表达式<math>y=f(x)\,</math>（含一个或多个待定常数，由初始条件确定）。例如：

:<math> \frac{dy}{dx}=\sin x</math>，

的解是

:<math> y=-\cos x+C</math>，

其中<math>C</math>是待定常数；

例如，如果知道

:<math> y=f(\pi)=2</math>，

则可推出

:<math>C=1</math>，

而可知 <math> y=-\cos x+1</math>，

=== 簡易微分方程的求解方法 ===
====一階線性常微分方程====
對於一階線性常微分方程，常用的方法是常數變易法：

對於方程：<math>y'+p(x)y+q(x)=0</math>

可知其通解：<math>y=C(x)e^{-\int p(x)\, dx}</math>

然後將這個通解代回到原式中，即可求出<math>C(x)</math>的值

====二階常係數齊次常微分方程====
對於二階常係數齊次常微分方程，常用方法是求出其特征方程的解

對於方程：<math>y''+py'+qy=0</math>

其特征方程：<math>r^2+pr+q=0</math>

根據其特征方程，判斷根的分佈情況，然後得到方程的通解：
<math>y=c_1 y_1+c_2 y_2</math>

一般的通解形式為

（在<math>r_1=r_2</math>的情況下）:<math>y=(C_1+C_2 x) e^{r x}</math>

（在<math>r_1\neq r_2</math>的情況下）:<math>y=C_1 e^{r_1 x}+C_2 e^{r_2 x}</math>

（在共軛複數根的情況下）：<math>y=e^{\alpha x} (C_1 \cos (\beta x) + C_2 \sin (\beta x))</math>

=== 約束條件 ===
微分方程的約束條件是指其解需符合的條件，依常微分方程及偏微分方程的不同，有不同的約束條件。

[[常微分方程|常微分方程]]常見的約束條件是函數在特定點的值，若是高階的微分方程，會加上其各階導數的值，有這類約束條件的常微分方程稱為[[初值問題|初值問題]]。

若是二階的常微分方程，也可能會指定函數在二個特定點的值，此時的問題即為[[邊界值問題|邊界值問題]]。若[[邊界條件|邊界條件]]指定二點數值，稱為[[狄利克雷邊界條件|狄利克雷邊界條件]]（第一類邊值條件），此外也有指定二個特定點上導數的邊界條件，稱為[[諾伊曼邊界條件|諾伊曼邊界條件]]（第二類邊值條件）等。

[[偏微分方程|偏微分方程]]常見的問題以[[邊界值問題|邊界值問題]]為主，不過邊界條件則是指定一特定[[超曲面|超曲面]]的值或導數需符定特定條件。

=== 解的存在性及唯一性 ===
存在性是指給定一微分方程及約束條件，判斷其解是否存在。唯一性是指在上述條件下，是否只存在一個解。

針對常微分方程的[[初值問題|初值問題]]，[[皮亚诺存在性定理|皮亚诺存在性定理]]可判別解的存在性，[[柯西-利普希茨定理|柯西-利普希茨定理]]則可以判別解的存在性及唯一性。

針對偏微分方程，{{link-en|柯西-克瓦列夫斯基定理|Cauchy–Kowalevski theorem}}可以判別解的存在性及唯一性。
[[皮亚诺存在性定理|皮亚诺存在性定理]]可以判斷常微分方程[[初值問題|初值問題]]的解是否存在。

== 歷史 ==
微分方程的起源約在十七世紀末，为了解决物理及天文学问题而產生，大約和微積分的發展同時。[[惠更斯|惠更斯]]在1693年的《教师学报》中提到常微分方程，[[雅各布·白努利|雅各布·白努利]]在1691年建立[[悬链线|悬链线]]的微分方程，並求得其函數。微分方程在十八世紀中期成為一個獨立的學科<ref>
{{cite web| url=http://219.239.227.53/pages/courses/gdsx/station/pages/KCZY/GSGG_11.html| language=中文| title=常微分方程的发展史况| publisher=北京航空航天大学现代远程教育学院| work=高等數學| accessdate=2014-01-18| deadurl=yes| archiveurl=https://archive.is/20140118022938/http://219.239.227.53/pages/courses/gdsx/station/pages/KCZY/GSGG_11.html| archivedate=2014-01-18}}</ref>，而微分方程也帶動許多當時的科學發展，例如海王星的發現就和微分方程的分析有關<ref name=IntrodPDE>
{{cite web
 | url=http://staff.ustc.edu.cn/~bjxuan/IntrodPDE.doc
 | language=中文
 | title=偏微分方程理论学习 
 | publisher=中国科学技术大学
 | accessdate=2014-01-18}}</ref>。

偏微分方程是由[[傅立葉|傅立葉]]開始的，他在1822年發表《熱的解析理論》，提出[[熱傳導方程|熱傳導方程]]的偏微分方程，並且利用[[分離變數法|分離變數法]]求得級數解，並且開始有關[[傅立葉級數|傅立葉級數]]的研究。另外在十九世紀有關 [[拉普拉斯方程|拉普拉斯方程]]的研究也是偏微分方程的重要发展。[[拉普拉斯|拉普拉斯]]和[[西莫恩·德尼·泊松|泊松]]都有許多的貢獻，後來[[喬治·格林|喬治·格林]]提出了相關[[格林函數|格林函數]]及[[格林公式|格林公式]]等概念，並帶動[[斯托克斯|斯托克斯]]、[[麦克斯韦|麦克斯韦]]及後來[[電磁學|電磁學]]相關的研究。而[[流體力學|流體力學]]的[[纳维-斯托克斯方程|纳维-斯托克斯方程]]及彈性介質的[[柯西方程|柯西方程]]也是在十九世紀提出的偏微分方程。<ref name=IntrodPDE/>。後來許多的理論都是以偏微分方程的形式出現，[[量子力學|量子力學]]的基礎方程式[[薛丁格方程|薛丁格方程]]也是偏微分方程，[[廣義相對論|廣義相對論]]中的[[愛因斯坦重力場方程式|愛因斯坦重力場方程式]]也有類似偏微分的[[協變導數|協變導數]]。

== 相關概念 ==
* [[時滯微分方程|時滯微分方程]]（DDE）是一個單一自變數的方程，此變數一般稱為時間，未知數在某一時間的導數和特定函數在之前時間的值有關。

* [[隨機微分方程|隨機微分方程]]（SDE）是一個未知數為[[隨機過程|隨機過程]]，且方程中有包括已知隨機過程（例如[[维纳过程|维纳过程]]）的方程，不過雖名為微分方程，其中沒有微分項。

* {{link-en|微分代數方程|differential algebraic equation}}（DAE）是包括自變數微分項的方程，但是為自變數微分項的[[隱函數|隱函數]]。

== 和差分方程的關係 ==
{{See also|时标微积分}}

微分方程的理論和[[差分方程|差分方程]]的理論有密切的關係，後者的座標只允許離散值，許多計算微分方程數值解的方法或是對於微分方程性質的研究都需要將微分方程的解近似為對應差分方程的解。
