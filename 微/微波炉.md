'''微波炉'''是指利用[[微波|微波]]波段的電磁波加热物体（主要是極性分子，例如水）的[[家用电器|家用]]或工业用[[電器|電器]]。

== 历史 ==
使用微波来烹饪食物的方法是首先由任職[[雷神公司|雷神公司]]的[[培西·史賓賽|培西·史賓賽]]想到的，[[培西·史賓賽|培西·史賓賽]]过去为公司建造雷达设备的磁电管。一天，他在一个启动的[[雷达|雷达]]设备上工作时，突然发觉自己放在口袋里的花生[[巧克力|巧克力]]融化了。經[[培西·史賓賽|培西·史賓賽]]的思索和研究，發現他的[[巧克力|巧克力]]是被[[微波|微波]]所融化，所以認為[[磁控管|磁控管]]也能用來[[烹飪|烹飪]]。

1960年代，利顿电子公司设计了一种磁控管，将其应用于微波炉中，但利顿公司并未把微波炉的市场设想为家庭。而[[佐佐木正治|佐佐木正治]]认为这项技术可以用于家用烹调，并说服日本早川公司（即[[夏普|夏普]]公司的前身）实践他的想法，當然雖然這是一項新科技，實際上以物理學的角度看，電磁波早就在烹飪當中使用了，就是點燃爐火的紅外光熱能來加熱食物，不過微波的加熱形式更方便，在1962年人類烹飪革命的微波炉开始大规模生产。<ref>{{cite book

== 工作原理 ==
如果一些物体（比如液态[[水|水]]）含有[[极性|电极性分子]]并且这些分子可自由震荡，那么它们可被微波炉加热。当这些物体被置于[[微波|微波]]传播空间中，在微波高频振荡（微波炉常采用2.45 GHz的微波）的[[电磁场|电磁场]]作用下，物体中的[[极性|电极性分子]]（尤其是水分子，可强烈地对微波作出响应）的方向会随振荡[[電場|電場]]一起振动，一个分子的固有[[电磁场|电磁场]]被改变并影响邻近分子，于是分子的振動便在分子之間傳遞開去，分子振動就是內能，增加內能就是加熱，微波能令物質中的內能增加，也即是能令物質加熱。虽然非电极性分子也会因[[電場|電場]]产生一些[[位移极化|位移极化]]，但这本身对内能几乎没有贡献。有的食物本身就有自由的[[极性|电极性分子]]，因此可以被微波直接加热；其他食物只要与水均匀混合，也可以通过水间接地被微波加热。另外，有說微波加熱是分子[[共振|共振]]的結果，這是錯誤的，水分子的[[共振頻率|共振頻率]]為1THz以上。<ref>{{cite book |last=Schmitt |first=Ron |title=Electromagnetics Explained: a handbook for wireless/RF, EMC, and high-speed electronics |url=http://books.google.fi/books?id=7gJ4RocvEskC |accessdate=3 December 2012 |year=2002 |publisher=Elsevier |location=Burlington, Mass., USA |isbn=978-0-7506-7403-4 |page=343}}</ref>

有些情況下，微波加熱可以做到由內而外，即內部溫升比表面快。例如內部有較多水份而表面較乾，這樣內部加熱就會比表面快；又例如內部的[[熱容量|熱容量]]比表面低，內部溫升就比較快。但並不是所有食物被微波加熱時都有此情況出現。

== 效率 ==
{{Unreferencedsect}}
微波爐只能將其耗用的部份電能轉為熱能，以一個1100W的微波爐為例，一般只有700W的微波產生，效率為64%。雖然產生微波的效率不高，但因為微波加熱不會浪費熱能於器皿上，也較小在空氣中散失，所以仍有不差的效率。

== 使用須知 ==
===容器和包装===
* 摺疊的金屬在微波加熱的情況下會產生[[電弧|電弧]]（蓝色火花）。當微波使金屬累積電能，捏成一團的金屬使電流跳過空隙，並產生電弧。如：用鋁箔摺成的小球放在微波爐加熱時，表面出現火花。此爆閃的現象對現代微波爐而言不容易產生危害，但仍有風險且使用金屬容器會影響加熱效果。<ref name="流言终结者">{{cite web|title =流言终结者：铜板砸死人（下）|url =http://www.56.com/u16/v_MTk1MzM4MDU.html|accessdate =2013-05-21}}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>
* 平面的金屬在微波爐中碰到爐壁時，電流會經爐壁傳回[[磁控管|磁控管]]，使磁控管熔化，引致[[短路|短路]]。<ref name="流言终结者"/>
* 將[[保鮮紙|保鮮紙]]與食品一起加熱時，保鮮紙中的[[PVC|PVC]]會釋出[[氫氯酸|氫氯酸]]和[[塑化劑|塑化劑]]。若使用保鮮膜，應選擇「不含有PVC」或「可用於微波爐加熱」者，並避免直接碰觸到食物。<ref>{{cite web

===食物===
* 使用微波爐加熱[[脂肪|脂肪]]含量高的食物，如：含有大量脂肪的[[豬肉|豬肉]]時，應在容器上方加上蓋子，以免[[脂肪|脂肪]]因過熱噴離肉類至爐內部。
* 不可把帶殼的全[[蛋|蛋]]直接入微波爐烹調，因為蛋殼不透氣，加熱會因內部氣壓增加造成爆裂。而加热去壳鸡蛋，应用针、筷子等'''刺破蛋黄'''，否则高温加热时也会爆裂喷溅。但對熟蛋來說，即便刺破蛋黃，因為蛋黃不流動，氣化的水分子不一定可以由刺破的孔逃出，也有爆裂的可能；也有可能因為水的體積膨脹速度趕不上水逃出蛋的速度而爆裂。但有刺破的蛋爆裂威力通常不大，不會造成危險。
* 加熱液體時應避免單獨入微波爐加熱，要放置攪拌棒等以助熱能釋放，加熱後不應立即取出，以免[[暴沸|暴沸]][[灼傷|灼傷]]。一般放进家用自来水单独加热会沸腾，因为其中含有少许杂质，但蒸馏水在加热超过沸点后仍然不沸腾，一旦受到扰动则会暴沸。<ref name="流言终结者"/>
*婴儿食品谨慎使用微波炉加热，[[中国农业大学|中国农业大学]]食品学院营养与食品安全系副教授范志红表示，这主要是为了避免加热不均匀带来的营养素损失，也避免过高的温度可能产生有害物质。因为婴儿对营养素的需求是非常严格的，身体的解毒能力也远未发育完全。<ref>人民网[http://paper.people.com.cn/xaq/html/2013-04/26/content_1231970.htm?div=-1]</ref>
*微波烹饪对于高水分食品如粥、饭、面条、牛奶、蔬菜等是合适的。而脂肪含量高水分含量低的食物，如坚果、奶酪、五花肉等，用微波炉加热时由于温度上升快而容易焦煳。

===其他===
* 留意時間上限問題。
*医学界的确禁止使用普通微波炉融化冰冻的血浆，原因在于用普通微波炉加热时，血液制品因受热不均匀，容易出现局部过热。而血液一旦温度升高到40℃以上就会出现溶血，严重的溶血反应可以导致身体组织死亡，危及患者生命。这种医疗事故并非是微波加热本身会产生毒性。
