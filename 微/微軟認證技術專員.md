'''微軟認證技術專員'''（MCTS）是微軟於2007年推出的新一代微軟專業認證，等級在MCP（Microsoft Certified Professional）之上的進階認證，持有MCTS代表對微軟某個技術或某個產品具有充份的技術能力，可以執行組態設定（針對網管或系統管理人員）或程式撰寫能力（針對開發人員），MCTS是以產品和技術為認證的標的，與MCP系列不同，每一個MCTS認證均只有一科考試科目，只要通過該科考試，即可取得MCTS資格，但開發技術的MCTS要求考生必須要先通過Exam 70-536: TS: Microsoft .NET Framework Application Development Foundation才可以取得開發技術的MCTS認證的資格（此要求在.NET Framework 4.0發布之後已經取消）。

MCTS的認證通常以MCTS:所認可的技術或產品方式表示，例如SQL Server 2005的MCTS認為表示法為MCTS:SQL Server 2005。

MCTS的考試名稱皆以TS開頭，方便識別。

微軟在2012年重新變更認證架構，新的MCSA需要考過兩科才能取得認證，MCTS亦將於考試停考後走入歷史。

==Windows Server 2008系列==

以Windows Server 2008的組態設定為主，並做為MCITP: Enterprise Administrator與MCITP: Server Administrator的前置需求。

<source lang="text">
Exam 70-640: TS: Microsoft Windows Server 2008 Active Directory, Configuring
Exam 70-642: TS: Microsoft Windows Server 2008 Networking Infrastructure, Configuring
Exam 70-643: TS: Microsoft Windows Server 2008 Application Platform, Configuring
</source>

Windows的另兩個版本：Small Business Server 2008以及Essential Server 2008也有考試，但不列入MCITP: Enterprise Administrator與MCITP: Server Administrator的前置需求。

<source lang="text">
Exam 70-653: TS: Microsoft Windows Small Business Server 2008, Configuring
Exam 70-654: TS: Microsoft Windows Essential Server 2008, Configuring
</source>

針對微軟在Windows Server 2008推出的虛擬化解決方案（Virtualization）也有推出考試：

<source lang="text">
Exam 70-652: TS: Microsoft Windows Server Virtualization, Configuring
</source>

現有的MCSA 2003與MCSE 2003認證專家可以利用升級考試來取得Windows Server 2008系列的MCTS認證集，依升級考試的不同，取得的MCTS的認證數也有所不同。

<source lang="text">
Exam 70-648: TS: Upgrading your MCSA on Windows Server 2003 to Windows Server 2008, Technology Specialists 
Exam 70-649: TS: Upgrading your MCSE on Windows Server 2003 to Windows Server 2008, Technology Specialists 
</source>

==.NET Framework 2.0系列==

以Microsoft .NET 2.0技術為主要認證內容，並做為下列三個MCPD認證的前置需求：

* MCPD: Web Developer (70-528)
* MCPD: Windows Developer (70-526)
* MCPD: Enterprise Application Developer（70-529, 70-526, 70-528）

<source lang="text">
Exam 70-526: TS: Microsoft .NET Framework 2.0 Windows Client Development
Exam 70-528: TS: Microsoft .NET Framework 2.0 Web Client Development
Exam 70-529: TS: Microsoft .NET Framework 2.0 Distributed Application Development
</source>

以上的認證要求必須要先通過Exam 70-536: TS: Microsoft .NET Framework Application Development Foundation才可得到認可。

持有MCAD on Microsoft .NET與MCSD on Microsoft .NET認證的專業人員，可以利用升級考試來快速獲得MCTS的認證。

<source lang="text">
Exam 70-558: 
UPGRADE: MCAD skills to MCTS: .NET Framework 2.0 Windows Client Development 
Exam 70-559: 
PGRADE: MCAD skills to MCTS: .NET Framework 2.0 Web Client Development
</source>

下列三科升級考試不但可獲得MCTS，也可以獲得MCPD認證：
<source lang="text">
Exam 70-551: 
UPGRADE: MCAD skills to MCPD: Web Developer by using Microsoft .NET Framework 
Exam 70-552: 
UPGRADE: MCAD skills to MCPD: Windows Developer by using Microsoft .NET Framework
Exam 70-553: 
UPGRADE: MCSD Microsoft .NET skills to MCPD: Enterprise Application Developer by using Microsoft .NET Framework, Part 1
Exam 70-554: 
UPGRADE: MCSD Microsoft .NET skills to MCPD: Enterprise Application Developer by using Microsoft .NET Framework, Part 2
</source>

* MCPD: Windows Developer 3.5（70-505）
* MCPD: Enterprise Application Developer 3.5（70-505, 70-561, 70-562, 70-503）

Exam 70-503: TS: Microsoft .NET Framework 3.5, Windows Communication Foundation Applications

==.NET Framework 4.0系列==

以[[.NET_Framework|.NET Framework]] 4.0技術為主要認證內容，並做為下列兩個MCPD認證的前置需求：

* MCPD: Web Developer 4 (70-519)
* MCPD: Windows Developer 4 (70-518)

<source lang="text">
Exam 70-511: TS: Microsoft .NET Framework 4, Windows Applications
Exam 70-513: TS: Microsoft .NET Framework 4, Service Communication Applications
Exam 70-515: TS: Microsoft .NET Framework 4, Web Applications
Exam 70-516: TS: Microsoft .NET Framework 4, Data Access
</source>

與.NET Framework 2.0與3.5系列的MCTS不同的是，.NET Framework 4的MCTS不要求先考70-536即可獲得MCTS。

持有MCPD .NET Framework 3.5系列認證的專業人員，可以利用升級考試升級到.NET Framework 4.0版本的MCTS系列認證，同時也可以獲得.NET Framework 4.0的MCPD認證。
