'''氯化汞'''（[[化学式|化学式]]：[[汞|Hg]][[氯|Cl]]<sub>2</sub>）俗称'''升汞'''，室温下为白色晶体，是实验室常用试剂。可溶于水，加热易[[升华|升华]]，可引起[[汞中毒|汞中毒]]，因为毒性极大，使用时必须小心。

== 性质 ==

氯化汞为[[正交晶系|正交晶系]]，容易升华，具有明显的[[共价|共价]]特性。固态含直线型Cl-Hg-Cl分子，以片状排列。通常认为每个汞原子与六个氯原子[[配位|配位]]，两个短的Hg-Cl距离为2.38Å，与其他四个则相距3.38Å。<ref>Wells, A.F. (1984) Structural Inorganic Chemistry, Oxford: Clarendon Press. ISBN 0-19-855370-6.</ref>

氯化汞可溶于水，[[溶解度|溶解度]]随温度升高而增大。对其水溶液进行[[电导|电导]]、冰点降低、光谱等一系列实验表明，大多数氯化汞以未离解的HgCl<sub>2</sub>存在，只有少量发生离解：

: <math>\rm \ HgCl_2 \rightleftharpoons HgCl^+ + Cl^-</math>

并且也存在少量的HgCl<sub>3</sub><sup>−</sup>和极少量的Hg<sup>2+</sup>、[HgCl<sub>4</sub>]<sup>2−</sup>离子。[[水解|水解]]反应为：

: <math>\rm \ HgCl_2 + H_2O \rightleftharpoons Hg(OH)Cl + H^+ + Cl^-</math>

于存在Cl<sup>−</sup>的溶液中溶解，生成四面体型[HgCl<sub>4</sub>]<sup>2−</sup>配离子。

== 制备 ==

氯化汞的制备方法有：
1、[[汞|汞]]或[[氯化亚汞|氯化亚汞]]与[[氯气|氯气]]反应：

: <math>\rm \ Hg + Cl_2 \rightarrow HgCl_2</math>

: <math>\rm \ Hg_2Cl_2+Cl_2\rightarrow 2HgCl_2</math>

2、[[盐酸|盐酸]]与硝酸亞汞溶液混合：

: <math>\rm \ Hg_2(NO_3)_2 + 4 HCl \rightarrow 2 HgCl_2 + 2 H_2O + 2 NO_2</math>

3、加热固态[[硫酸汞|硫酸汞]]和[[氯化钠|氯化钠]]的混合物。产物HgCl<sub>2</sub>升华后冷凝为细小的菱方晶体。

== 反应 ==

氯化汞于碱性溶液中水解，反应为：

: <math>\rm \ 2 HgCl_2 + 2OH^- \rightleftharpoons HgO + H_2O + HgCl_3^- + Cl^-</math>

碱不足时生成[[氯氧化物|氯氧化物]]。

与[[氨|氨]]反应，根据条件不同，可以有不同的产物：

: <math>\rm \ HgCl_2 + 2NH_3 \rightarrow HgCl_2\cdot 2NH_3</math>
: <math>\rm \ HgCl_2 + 2NH_3 \rightarrow HgNH_2Cl + NH_4^+ + Cl^-</math>
: <math>\rm \ 2HgCl_2 + 4NH_3 + H_2O \rightarrow Hg_2NCl\cdot H_2O + 3NH_4^+ + 3Cl^-</math>

== 应用 ==

氯化汞主要用作[[催化剂|催化剂]]，催化[[乙炔|乙炔]]转化为[[氯乙烯|氯乙烯]]，而后者是制取[[聚氯乙烯|聚氯乙烯]]的前体：

: <math>\rm \ C_2H_2  +  HCl \rightarrow CH_2\!=\!CHCl</math>

上述反应已被[[1,2-二氯乙烷|1,2-二氯乙烷]][[高温裂解|高温裂解]]的反应所取代。

氯化汞也被用作电池中的[[去极剂|去极剂]]、医药中的防腐杀菌剂、染色的[[媒染剂|媒染剂]]、木材的防腐剂、照相乳剂的增强剂及[[有机合成|有机合成]]和[[分析化学|分析化学]]中的试剂，<ref>Matthias Simon, Peter Jönk, Gabriele Wühl-Couturier, Stefan Halbach "Mercury, Mercury Alloys, and Mercury Compounds" in Ullmann's Encyclopedia of Industrial Chemistry 2006: Wiley-Interscience. DOI: 10.1002/14356007.a16_269.pub2</ref> 但由于毒性而使其应用受到限制。

=== 化学试剂 ===

氯化汞在化学中的应用有：

* 制取金属[[汞齐|汞齐]]。例如制取[[铝|铝]]汞齐时，将铝浸入氯化汞溶液中，铝的表面便逐渐生成薄层的单质汞，与铝结合为汞齐。汞齐的生成祛除了表面氧化膜对铝的保护，使铝“活化”并发生许多反应。如，铝汞齐能溶解于水并放出[[氢气|氢气]]；可与[[卤代烃|卤代烃]]发生[[Barbier反应|Barbier反应]]；可作为[[还原剂|还原剂]]参与[[有机合成|有机合成]]，等等。相应烷基铝化合物类似[[格氏试剂|格氏试剂]]，具[[亲核性|亲核性]]。
* 氯化汞可脱去[[羰基|羰基]]的[[缩硫醛|缩硫醛]]/[[缩硫酮|酮]][[保护基|保护基]]，同时发生[[极性翻转|极性翻转]]。
* 制取[[甘汞|甘汞]]、[[四碘合汞酸钾|四碘合汞酸钾]]等汞化合物。
