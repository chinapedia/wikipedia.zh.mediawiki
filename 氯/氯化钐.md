'''氯化釤(III)'''（[[化学式|化学式]]：[[钐|Sm]][[氯|Cl]]<sub>3</sub>），是[[稀土金屬|稀土金屬]][[釤|釤]](III)的[[氯化物|氯化物]]。它是一種淡黃色的固體。若將之暴露在潮濕的空氣中，它會迅速的吸收水分子形成六[[水合物|水合物]]，SmCl<sub>3</sub>·6H<sub>2</sub>O。

單純加熱氯化釤的水合物可能造成小部份的產物水解。在110 °C時會失去五個水分子。氯化釤是強的[[路易士酸|路易士酸]]，若依[[软硬酸碱理论|软硬酸碱理论]]的分類，它被歸類為「硬酸」。

== 製備 ==
氯化釤(III)可以由金属釤或[[碳酸釤(III)|碳酸釤(III)]]與[[盐酸|盐酸]]反應而得。

無水氯化釤(III)可以由將含水氯化釤脫水制得。有兩種方法：在高度真空下與[[氯化銨|氯化銨]]慢慢地加熱到400 °C，或者与大量的[[氯化亚砜|SOCl<sub>2</sub>]]一起加熱五小時。無水氯化釤也可以由金属釤與[[氯化氫|氯化氫]]反應而得。<ref>L. F. Druding, J. D. Corbett, ''[[J._Am._Chem._Soc.|J. Am. Chem. Soc.]]'' '''83''', 2462 (1961)</ref><ref>J. D. Corbett, ''Rev. Chim. Minerale'' '''10''', 239 (1973). </ref>
:2 Sm(s) + 6 HCl(aq) → 2 SmCl<sub>3</sub>(aq) + 3 H<sub>2</sub>(g)
:Sm<sub>2</sub>(CO<sub>3</sub>)<sub>3</sub>(s) + 6 HCl(aq) → 2 SmCl<sub>3</sub>(aq) + 3 CO<sub>2</sub>(g) + 3 H<sub>2</sub>O(l)

高純度的氯化釤可通过高溫高度真空的環境下將氯化釤[[升华|升华]]得到。<ref name="Edelmann" />

== 用途 ==
氯化釤(III)可以用來製備釤金屬，而釤金屬有多種用途，主要在[[磁鐵|磁鐵]]方面。無水氯化釤(III)是氯化鈉和氯化鈣的[[助熔劑|助熔劑]]。無水氯化釤也可以用來製備一些[[釤|釤]]的[[有機金屬化合物|有機金屬化合物]]，如用作[[氢化反应|氢化反应]]及[[烯烃|烯烃]][[硅氢化反应|硅氢化反应]][[催化剂|催化剂]]的双(五甲基环戊二烯基)钐(III)配合物。<ref>G. A. Molander, E. D. Dowdy, ''Lanthanides: Chemistry and Use in Organic Synthesis'', Springer-Verlag, Berlin, 1999, p119-154. </ref>它也可以用來製備其他的釤化合物，比如[[氢氧化钐(III)|氢氧化钐(III)]]与[[氟化钐(III)|氟化钐(III)]]：
:SmCl<sub>3</sub>(aq) + 3 NaOH(aq) → Sm(OH)<sub>3</sub>(s) + 3 NaCl(aq)
:SmCl<sub>3(aq)</sub> + 3 KF(aq) → SmF<sub>3</sub>(s) + 3 KCl(aq)
