'''氯乙醛'''，结构式ClCH<sub>2</sub>CHO。

== 性质 ==
无色具刺鼻辛辣气味的液体。易燃；有毒，具酸腐蚀性，对皮肤和粘膜有刺激性。溶于水、[[甲醇|甲醇]]、[[乙醚|乙醚]]、[[丙酮|丙酮]]。40％水溶液为无色透明液体；浓度超过50％时析出半水物片晶，半水物溶于水、[[乙醇|乙醇]]和[[乙醚|乙醚]]。

== 制备 ==
[[氯乙烯|氯乙烯]]和[[氯|氯]]在氧化塔中於40～45 °C反应6～7小时至氯乙醛含量达10％，终止反应、分离副产物[[三氯乙烷|三氯乙烷]]，得氯乙醛水溶液。
:<math>\ H_2C\!=\!CHCl+Cl_2+H_2O \rightarrow ClCH_2CHO+2HCl\uparrow</math>

== 用途 ==
用作有机合成原料，例如制取杂环化合物（如与[[硫脲|硫脲]]衍生物反应合成[[磺胺噻唑|磺胺噻唑]]）和杀菌剂等。
