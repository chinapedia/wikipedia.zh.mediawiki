{{Expand English|Magnesium chloride}}
'''氯化镁'''是一种[[氯化物|氯化物]]，[[化学式|化学式]]MgCl<sub>2</sub>。无色而易[[潮解|潮解]][[晶体|晶体]]。這些鹽是典型的離子鹵化物，高度易溶於水。水合氯化鎂可以從鹽水或[[海水|海水]]中提取。通常带有6[[分子|分子]]的[[结晶水|结晶水]]。但加热至95[[摄氏度|℃]]时失去结晶水。135℃以上时开始分解，并释放出[[氯化氢|氯化氢]]（HCl）气体。工业上生产[[镁|镁]]的原料。在海水和[[盐卤|盐卤]]中找到。水合氯化鎂是處方口服鎂補充劑通常使用的物質。

==制备==
氯化镁一般用[[氢氧化镁|氢氧化镁]]和[[盐酸|盐酸]]反应得到：
:<chem>{Mg(OH)2} + 2HCl -> {MgCl2} + 2H2O</chem>
此外，镁的氧化物、碳酸盐和盐酸反应，也能制得氯化镁。

==毒物學==
鎂離子有苦味，氯化鎂溶液的苦的程度，決定於不同程度的鎂離子濃度。氯化鎂一旦滲入泥土，會造成[[鎂害|鎂害]]<ref name="colostate1">{{cite web  |url=http://www.ext.colostate.edu/pubs/garden/07425.html |title=Magnesium Chloride Toxicity in Trees |first1=B.A. |last1=Goodrich |first2=W.R. |last2=Jacobi |page=1 |date=2008年7月}}</ref><ref>Pesticide Action Network Database, http://www.pesticideinfo.org/Detail_Chemical.jsp?Rec_Id=PC32916</ref>。
