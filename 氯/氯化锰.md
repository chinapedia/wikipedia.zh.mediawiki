'''氯化锰'''是化学式为MnCl<sub>2</sub>·''x''H<sub>2</sub>O的化合物，其中x的值为0或2或4。四水合氯化锰是一种粉红色的盐，在自然界中以稀有的[[氯锰矿|氯锰矿]]形式存在。通常，氯化锰指的是MnCl<sub>2</sub>·4H<sub>2</sub>O，为八面体构造的''反''-Mn(H<sub>2</sub>O)<sub>4</sub>Cl<sub>2</sub>分子。二水合物MnCl<sub>2</sub>·2H<sub>2</sub>O也存在。许多Mn(II)化合物均表现出特征的浅粉红色，是由[[过渡金属|过渡金属]]的d<sup>5</sup>[[自旋|自旋]]引起的。

==制备==
可用[[锰|锰]]金属和[[氯化氢|氯化氢]]气体制备，或用[[碳酸锰|碳酸锰]]和[[盐酸|盐酸]]制备。前者得到无水盐，后者得到带有结晶水的盐。
:<math>\mathrm{Mn} + \mathrm{2HCl}  \longrightarrow \mathrm{MnCl}_2 + \mathrm{H}_2 \uparrow</math>
:<math>\mathrm{MnCO}_3  + \mathrm{2HCl} \longrightarrow \mathrm{MnCl}_2  + \mathrm{H}_2\mathrm{O} + \mathrm{CO}_2\uparrow</math>

[[二氧化锰|二氧化锰]]在与浓[[盐酸|盐酸]]加热时也会生成氯化锰，此反应在实验室中用于制备[[氯气|氯气]]：
:<math>\mathrm{MnO}_2 + \mathrm{4HCl} \longrightarrow \mathrm{MnCl}_2 + \mathrm{2H}_2\mathrm{O} + \mathrm{Cl}_2 \uparrow</math>

这个反应证明了二氧化锰的氧化性。

==化学性质==
氯化锰由于水解，其水溶液呈弱酸性，[[pH|pH]]约为4。它可以用来制备许多锰化合物： 
:<math>\mathrm{MnCl}_2(aq)  +  \mathrm{K}_2\mathrm{CO}_3(aq) \longrightarrow \mathrm{MnCO}_3(s)  +  \mathrm{2KCl}</math>

Mn<sup>2+</sup>离子在水溶液中的实际存在形式为[Mn(H<sub>2</sub>O)<sub>6</sub>]<sup>2+</sup>。

氯化锰是弱[[路易斯酸|路易斯酸]]，与[[氯离子|氯离子]]反应生成以下离子：[MnCl<sub>3</sub>]<sup><nowiki>−</nowiki></sup>，[MnCl<sub>4</sub>]<sup>2<nowiki>−</nowiki></sup> 以及 [MnCl<sub>6</sub>]<sup>4<nowiki>−</nowiki></sup>。

某些有机[[配体|配体]]的作用下，锰(II)易被空气中的氧气氧化为锰(III)配合物。离子包括[Mn([[EDTA|EDTA]])]<sup><nowiki>−</nowiki></sup>、 [Mn([[氰化物|CN]])<sub>6</sub>]<sup>3<nowiki>−</nowiki></sup>以及[Mn([[乙酰丙酮|Hacac]])]<sub>3</sub>。  [[三苯膦|三苯膦]]与之形成不稳定的2:1加合物：
:<math>\mathrm{MnCl}_2 + \mathrm{2Ph}_3\mathrm{P} \longrightarrow \mathrm{MnCl}_2(\mathrm{Ph}_3\mathrm{P})_2</math>

无水氯化锰是制备许多锰化合物的起始点。例：
:<math>\mathrm{MnCl}_2 + \mathrm{2NaC}_5\mathrm{H}_5 \longrightarrow \mathrm{Mn}(\mathrm{C}_5\mathrm{H}_5)_2 + \mathrm{2NaCl}</math>


==毒性==
锰元素对于[[需氧生物|需氧生物]]来说是必需的，且通常锰化合物毒性也并不大。锰中毒可能由长时间暴露在含锰尘土或烟雾中所导致。
