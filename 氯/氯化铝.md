'''氯化鋁'''，或'''三氯化铝'''，化学式为AlCl<sub>3</sub>，是[[氯|氯]]和[[鋁|鋁]]的[[化合物|化合物]]。氯化鋁[[熔點|熔點]]、[[沸點|沸點]]都很低，且會[[昇華|昇華]]，為有離子性的共價化合物。熔化的氯化鋁不易導電<ref name="Greenwood">N. N. Greenwood, A. Earnshaw, ''Chemistry of the Elements'', Pergamon Press, Oxford, United Kingdom, 1984.</ref>，和大多數含[[鹵素|鹵素]]離子的鹽類（如[[氯化鈉|氯化鈉]]）不同。 <!-- It exists in the solid state as a six-coordinate layer lattice. -->

AlCl<sub>3</sub>采取“[[三氯化钇|YCl<sub>3</sub>]]”结构，为Al<sup>3+</sup>立方最密堆积层状结构，<ref name="Wells">A. F. Wells,  ''Structural Inorganic Chemistry'', Oxford Press, Oxford, United Kingdom, 1984.</ref> 而AlBr<sub>3</sub>中Al<sup>3+</sup>却占Br<sup>−</sup>最密堆积框架的相邻四面体间隙。熔融时AlCl<sub>3</sub>生成可挥发的[[二聚体|二聚体]]Al<sub>2</sub>Cl<sub>6</sub>，含有两个三中心四电子氯桥键，更高温度下Al<sub>2</sub>Cl<sub>6</sub>二聚体则离解生成平面三角形AlCl<sub>3</sub>，与[[三氟化硼|BF<sub>3</sub>]]结构类似。

[[File:Aluminium-trichloride-3D-structures.png|middle]]


氯化鋁是很常用的[[路易斯酸|路易斯酸]]。在[[化學工業|化學工業]]，它常是[[傅-克反應|傅-克反應]]的[[催化劑|催化劑]]。它亦用於[[烴|烴]]的[[聚合|聚合]]和[[异构化反应|异构化反应]]中。

鋁亦可以生成[[一氯化鋁|一氯化鋁]]（AlCl），但這种化合物很不穩定，只能以[[氣態|氣態]]的形式出現。

工業上，它經[[鋁土礦|鋁土礦]]加工，由鋁和氯之間的放熱反應所製。

== 化學性質 ==

氯化鋁是強[[路易斯酸|路易斯酸]], 可和[[路易斯鹼|路易斯鹼]]作用產生化合物，甚至也可和[[二苯甲酮|二苯甲酮]]和[[均三甲苯|均三甲苯]]之類的弱路易斯鹼作用。<ref name="Friedel">G. A. Olah (ed.), ''Friedel-Crafts and Related Reactions'', Vol. 1, Interscience, New York, 1963.</ref>
若有氯離子存在，氯化鋁會生成(四)氯鋁酸根離子AlCl<sub>4</sub><sup>−</sup>：

: AlCl<sub>3</sub>(aq) + Cl<sup>−</sup>(aq) {{unicode|⇌}}  AlCl<sub>4</sub><sup>−</sup>(aq)

在水中，氯化鋁會部分水解，形成[[氯化氫|氯化氫]]气体或H<sub>3</sub>O<sup>+</sup>離子。其水溶液和其他含鋁物质的溶液相同，含有水合铝离子，跟適當份量的[[氫氧化鈉|氫氧化鈉]]反應可生成[[氫氧化鋁|氫氧化鋁]][[沉淀|沉淀]]：<ref>[http://www.guidechem.com/dictionary/7446-70-0.html 氯化铝字典-Guidechem.com]</ref>

: AlCl<sub>3</sub>(aq)  +  3 NaOH(aq)  →  Al(OH)<sub>3</sub>(s)  +  3NaCl(aq)
: AlCl<sub>3</sub>(aq)  +  3 H<sub>2</sub>O  → AlO<sub>2</sub><sup>-</sup>+  3HCl  +  H<sub>3</sub>O<sup>+</sup>

== 用途 ==

氯化鋁主要用在[[傅-克反应|傅-克反应]]<ref name="Friedel"/> 中，例如以[[苯|苯]]和[[光气|光气]]为原料制备[[蒽醌|蒽醌]]，应用于[[染整工業|染整工業]]中。<ref name="Greenwood"/> 在廣義的傅-克反应中，[[醯氯|醯氯]]或[[鹵代烷|鹵代烷]]和[[芳香族|芳香族]]物質的反應式如下:<ref name="Friedel"/>

:: [[File:AlCl3_FriedelCrafts.gif|480px]]

苯及其[[衍生物|衍生物]]在发生上述反應时，主產物是對位的異構物。相比较下，烷基化反应涉及的问题较多，不如酰基化反应应用广泛。无论是哪种反应，氯化铝和其他原料和仪器都必须是中等干燥的，少量的水有助于反应进行。

由于氯化铝可与反应产物配位，因此应用在傅克反应时，它的用量必须与反应物相同，而非“催化量”。反应后的氯化铝很难回收，会产生大量的腐蚀性废料。为了达到[[绿色化学|绿色化学]]的要求，化学家开始使用[[三氟化釔|三氟化釔]]或[[三氟化鏑|三氟化鏑]]来替代氯化铝，减少污染。

氯化鋁也常用來將[[醛基|醛基]]加在苯環上，如[[加特曼-科赫反應|加特曼-科赫反應]]用[[一氧化碳|一氧化碳]]、[[氯化氫|氯化氫]]、[[氯化鋁|氯化鋁]]及[[氯化亞銅|氯化亞銅]]為催化劑<ref>L. G. Wade, ''Organic Chemistry'', 5th edition, Prentice Hall, Upper Saddle River, New Jersey, United States, 2003.</ref>

:: [[File:AlCl3_formylation.gif|450px]]

氯化铝在有机化学中有很广泛的应用。<ref>P. Galatsis, in: ''Handbook of Reagents for Organic Synthesis: Acidic and Basic Reagents'', (H. J. Reich, J. H. Rigby, eds.), pp12-15, Wiley, New York, 1999.</ref> 它可以催化[[Ene反應|Ene反應]]，比如[[3-丁烯-2-酮|3-丁烯-2-酮]]（甲基乙烯基甲酮）与[[香芹酮|香芹酮]]加成：<ref>B. B. Snider, ''Accounts of Chemical Research'' '''13''', 426 (1980).</ref>

:: [[File:AlCl3_ene_rxn.gif|500px]]

AlCl<sub>3</sub>也常用在烃类[[聚合反应|聚合反应]]和[[异构化反应|异构化反应]]中，重要的例子包括<ref name="Greenwood"/> 工业上[[乙苯|乙苯]]的生产。乙苯可用于进一步制备[[苯乙烯|苯乙烯]]、[[聚苯乙烯|聚苯乙烯]]以及用作[[清洁剂|清洁剂]]的[[十二烷基苯|十二烷基苯]]。

芳烃存在下，氯化铝与铝混合可用于合成二(芳烃)金属配合物。例如，[[二苯铬|二苯铬]]就是通过特定金属卤化物经由[[Fischer-Hafner合成|Fischer-Hafner合成]]制备的。

低浓度的[[碱式氯化铝|碱式氯化铝]]常是[[防汗药|防汗药]]的成分之一，而[[多汗症|多汗症]]患者在使用时浓度会高些（12%或更高）。

== 注意事項 ==

無水氯化鋁會和[[鹼|鹼]]劇烈反應，因此需妥善處理。含結晶水的氯化鋁較無此問題。

如果吸入無水氯化鋁或接觸眼睛和皮膚會造成刺激。
