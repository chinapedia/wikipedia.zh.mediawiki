'''氯酸'''，化学式为[[氢|H]][[氯|Cl]][[氧|O]]<sub>3</sub>，是[[氯|氯]]的[[含氧酸|含氧酸]]之一，其中氯的[[氧化态|氧化态]]为+5。它具强[[酸性|酸性]]（[[酸离解常数|p''K''<sub>a</sub>]]≈−1）及强[[氧化性|氧化性]]，可用于制取多种[[氯酸盐|氯酸盐]]。

它可由[[氯酸钡|氯酸钡]]与[[硫酸|硫酸]]反应，并滤去[[硫酸钡|硫酸钡]]沉淀得到：
:Ba(ClO<sub>3</sub>)<sub>2</sub> + H<sub>2</sub>SO<sub>4</sub> → 2HClO<sub>3</sub> + BaSO<sub>4</sub>

或用[[次氯酸|次氯酸]]加热[[歧化|歧化]]的反应制取：
:3HClO → HClO<sub>3</sub> + 2 HCl

浓度在30%以下的氯酸冷溶液都是稳定的，40%的溶液也可由减压下小心蒸发制取，但是在加热时会分解，产物不一：
:8HClO<sub>3</sub> → 4HClO<sub>4</sub> + 2H<sub>2</sub>O + 2Cl<sub>2</sub> + 3 O<sub>2</sub>
:3HClO<sub>3</sub> → HClO<sub>4</sub> + H<sub>2</sub>O + 2 ClO<sub>2</sub>

[[热力学|热力学]]上，氯酸是不稳定的，会自发发生[[歧化反应|歧化反应]]。
