'''氯磺酸'''（[[化学式|化学式]]：HSO<sub>3</sub>Cl）是一种无色或淡黄色的液体，為劇毒。具有辛辣气味，在空气中发烟，是[[硫酸|硫酸]]的一个[[羟基|－OH]] 基团被[[氯|氯]]取代后形成的化合物。分子为[[四面体|四面体]]构型，取代的基团处于[[硫酸|硫酸]]与[[硫酰氯|硫酰氯]]之间，有催泪性，主要用于[[有机化合物|有机化合物]]的[[磺化|磺化]]，制取药物、染料、农药、洗涤剂等。

== 制取 ==
工业上用[[三氧化硫|三氧化硫]]和[[氯化氢|氯化氢]]发生化合反应制得氯磺酸：
::HCl + SO<sub>3</sub> → HSO<sub>3</sub>Cl

干燥的[[氯化氢|氯化氢]]气体通入[[发烟硫酸|发烟硫酸]]，或[[五氯化磷|五氯化磷]]与发烟硫酸反应都可以得到氯磺酸，多用于实验室中：
::PCl<sub>5</sub> + SO<sub>2</sub>(OH)<sub>2</sub> → HSO<sub>3</sub>Cl + POCl<sub>3</sub> + HCl

== 应用 ==
氯磺酸遇水发生剧烈的水解反应，生成[[硫酸|硫酸]]和[[盐酸|盐酸]]：
::HSO<sub>3</sub>Cl + H<sub>2</sub>O → H<sub>2</sub>SO<sub>4</sub> + HCl

氯磺酸可用于制备[[磺酸|磺酸]]。磺酸类常是[[染料|染料]]和[[药物|药物]]（如[[磺胺|磺胺]]）的合成前体。
::R-H + HSO<sub>3</sub>Cl → RSO<sub>2</sub>OH + HCl
