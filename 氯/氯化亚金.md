'''氯化亚金'''（'''氯化金(I)'''），又称'''一氯化金'''。是[[金|金]]元素的一种氯化物，化学式为'''AuCl'''。

== 制备 ==
氯化亚金可以通过[[三氯化金|三氯化金]]的热分解来制备。

== 反应 ==
虽然更高温度下如果[[氯|氯]]气的压强合适它可以保持稳定，但是它在环境条件下处于亚稳态。与水共热时，氯化亚金会歧化成金属金和三氯化金：
: 3 AuCl → 2 Au + AuCl<sub>3</sub>

氯化亚金与[[溴化钾|溴化钾]]反应通过金的歧化反应得到[[溴金酸|溴金酸]]钾和[[氯化钾|氯化钾]]： 
: 3 AuCl + 4 KBr → KAuBr<sub>4</sub> + 2 Au + 3 KCl

== 安全 ==
氯化亚金对[[眼睛|眼睛]]和[[皮肤|皮肤]]具有刺激性，能损害肾脏功能并降低血细胞数目。一旦不慎接触，用[[肥皂|肥皂]]水冲洗皮肤15分钟。
