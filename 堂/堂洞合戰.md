'''堂洞合戰'''（{{jpn|j=堂洞合戦|hg=どうほらがっせん|rm=Dōhoragassen}}）是在[[永祿|永祿]]8年8月28日（1565年9月22日），[[織田信長|織田信長]]、{{Link-ja|加治田眾|加治田衆}}與[[齋藤龍興|齋藤龍興]]勢力的[[岸信周|岸信周]]於[[美濃國|美濃國]]{{Link-ja|堂洞城|堂洞城}}一帶（現[[岐阜縣|岐阜縣]][[加茂郡|加茂郡]][[富加町|富加町]]夕田一帶）爆發的一場[[戰役|合戰]]。

==背景==
為了阻擋信長攻佔美濃，{{Link-ja|關城 (美濃國)|関城 (美濃国)|關城}}主[[長井道利|長井道利]]、{{Link-ja|加治田城|加治田城}}主{{Link-ja|佐藤忠能|佐藤忠能}}和堂洞城主岸信周結盟，並且接納道利的意見，將忠能之女{{Link-ja|八重綠|八重緑}}作為岸氏的養女（人質），從而鞏固同盟關係。然而，忠能派居於加治田城的平民{{Link-ja|梅村良澤|梅村良澤}}前往[[織田氏|織田氏]]的[[犬山城|犬山城]]，通過[[丹羽長秀|丹羽長秀]]的介紹成為信長的內應。{{Link-ja|鵜沼城|鵜沼城}}和{{Link-ja|猿啄城|猿啄城}}先後落入織田軍手中，而猿啄城將多治見修理亮則逃往[[甲斐國|甲斐]]，其餘戰敗的士兵則進入堂洞城與岸軍匯合<ref name="tomikachoshi chunosanjo">{{Cite book|last= |first= |author= 富加町史編集委員会|authorlink= |coauthors= |year= 1980|title= 富加町史|volume = 下巻 通史編||publisher= [[富加町|岐阜県加茂郡富加町]]|location= |pages= 184 - 192|chapter = 中濃三城の盟約|id= |isbn= }}</ref>。

信長派[[金森長近|金森長近]]為使者前往堂洞城招降，但是遭到信周拒絕，其嫡子岸信房在長近面前將自己的兒子斬首，以表示其決心，長近只好離去<ref name="tomikachoshi chunosanjo"/>。岸軍在準備合戰期間，在堂洞城對面的長尾丸山將作為人質的八重綠處以[[十字架|磔刑]]。當晚，忠能的家臣{{Link-ja|西村治郎兵衛|西村治郎兵衛}}潛入岸軍陣營，成功盜取八重綠的遺體，並且將其安葬於加治田的{{Link-ja|龍福寺 (富加町)|龍福寺 (富加町)|龍福寺}}<ref>{{Cite book|last= |first= |author= 富加町史編集委員会|authorlink= |coauthors= |year= 1980|title= 富加町史|volume = 下巻 通史編||publisher= 富加町|location= |pages= 192 - 194|chapter = 堂洞合戦|id= |isbn= }}</ref>。

==堂洞城攻城戰==
織田和加治田眾聯軍分別從西、南和北三個方位進攻堂洞城，西和南由織田氏家臣長秀、[[河尻秀隆|河尻秀隆]]和[[森可成|森可成]]負責，北面則有忠能率領的加治田眾負責。另一方面，岸氏則由信周防守堂洞城的南和西面，北面則由信周之嫡子信房負責。信長在高畑山（現[[富加町|富加町]]高畑）設下本陣，從而將堂洞和關城分隔開，關城主道利率領援軍渡過{{Link-ja|津保川|津保川}}，信長便在高畑的森林將其擊退。信長騎馬到處指揮各部隊<ref name="tomikachoshi ikusanokeika">{{Cite book |last= |first= |author= 富加町史編集委員会|authorlink= |coauthors= |year= 1980|title= 富加町史|volume = 下巻 通史編||publisher= 富加町|location= |pages= 196 - 198|chapter = 堂洞合戦 戦の経過|id= |isbn= }}</ref>。

從西面進攻的織田軍受制於險峻的地形，無法突破岸軍的伏兵組成的防線，但是從北面進攻的佐藤父子倚仗熟悉地形的優勢，沿山路攻向堂洞城，信房雖然多次將其擊退，但是隨著戰事陷入苦戰，岸軍死傷慘重，信房自己也有三處地方受傷，最終他[[切腹|切腹]]自殺<ref name="tomikachoshi ikusanokeika"/>。

南面織田軍的[[太田牛一|太田牛一]]則放箭攻擊對手，立下戰功。岸軍則一邊撤退一邊還擊，信周之妻亦持長刀揮舞抵抗。然而，織田軍的秀隆和可成在日落時分攻進堂洞城，長秀則攻進本丸。在敵我難分的亂戰中，信周夫妻在詠唱辭世之歌後互刺身亡，信周之弟岸信貞亦奮戰而死，堂洞城最終被攻佔<ref name="tomikachoshi ikusanokeika"/>。

==戰後==
攻佔堂洞城當晚，信長在加治田城的佐藤父子的屋敷暫住一晚，翌日在城下實地巡視。信長在返回犬山途中，於井之口遭到道利軍和[[齋藤龍興|齋藤龍興]]的3,000兵力突襲，織田軍當時只有800名士兵，在難以戰鬥的情況下倉遑撤退至鵜沼<ref>{{Cite book |last= |first= |author= 富加町史編集委員会|authorlink= |coauthors= |year= 1980|title= 富加町史|volume = 下巻 通史編||publisher= 富加町|location= |pages= 198|chapter = 堂洞合戦 信長帰陣|id= |isbn= }}</ref>。其後，道利進攻加治田城，信長便派[[齋藤長龍|齋藤利治]]前往支援，最終在這場{{Link-ja|關、加治田合戰|関・加治田合戦}}中，關城亦被攻佔<ref>{{Cite book |last= |first= |author= 富加町史編集委員会|authorlink= |coauthors= |year= 1980|title= 富加町史|volume = 下巻 通史編||publisher= 富加町|location= |pages= 198 - 199|chapter = 堂洞合戦 関・加治田合戦、関 落城|id= |isbn= }}</ref>。

織田軍攻佔堂洞城後雖然將其廢棄，不過後來在[[加治田、兼山合戰|加治田、兼山合戰]]中，[[森長可|森長可]]曾經以堂洞城遺址作本陣<ref>{{Cite book |last= |first= |author= 富加町史編集委員会|authorlink= |coauthors= |year= 1980|title= 富加町史|volume = 下巻 通史編||publisher= 富加町|location= |page= 239|chapter = 加治田・兼山合戦 加治田城に攻め寄せる|id= |isbn= }}</ref>。
