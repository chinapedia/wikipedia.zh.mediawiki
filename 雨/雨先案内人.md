'''雨先案内人'''（あめさきあんないにん）是[[日本|日本]]的一支[[搖滾樂隊|搖滾樂隊]]。

== 成員 ==
;雨ノ地晴太郎（雨之地晴太郎）
:擔任主唱、[[鼓手|鼓手]]（三人時期）、[[吉他手|吉他手]]（個人時期）。

== 原成員 ==
;西山小雨
:擔任主唱、[[鍵盤手|鍵盤手]]。
;齋藤ぽんちょ（齋藤雨披）
:擔任[[和聲歌手|和聲]]、[[貝斯手|貝斯手]]。

== 來歷 ==
“雨先案内人”原本為[[雨之地晴太郎|雨之地晴太郎]]個人表演時期所使用的名稱<ref name="about">{{Cite web|url=http://amesakiannainin.jimdo.com/about-1/|title=自傳|publisher=雨先案内人|accessdate=2012-04-05|deadurl=yes|archiveurl=https://web.archive.org/web/20130115091040/http://amesakiannainin.jimdo.com/about-1|archivedate=2013年1月15日|df=}}</ref>。雨之地晴太郎曾經以彈奏[[原聲吉他|原聲吉他]]演唱的形式進行了三年左右的活動，2010年6月[[西山小雨|西山小雨]]和[[齋藤駿介|齋藤駿介]]接受邀請加入，“雨先案内人”作為[[樂隊|樂隊]]開始活動<ref name="about" />。2013年12月27日，貝斯手齋藤ぽんちょ、鍵盤手西山小雨退出樂隊，雨先案内人暫停活動。2014年起，雨ノ地晴太郎由“主唱&鼓手”轉為“主唱&吉他手”，2014年12月，繼續以雨先案内人的名義開始活動。
