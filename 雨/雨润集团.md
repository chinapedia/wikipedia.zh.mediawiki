'''雨润集团'''（{{sehk|1068}}）是中国一家食品工业企业，总部设立于南京市建邺区，是中国最大的肉制品生产企业之一。雨润旗下拥有雨润食品（1068.HK）、南京中商（600280.SH）两家上市公司。2014年位列中国民营企业500强第5位。<ref>{{Cite web|url=http://finance.ifeng.com/a/20141026/13218463_0.shtml|title=2014民营企业500强发布：苏宁联想魏桥集团分列前三|accessdate=2018-02-20|work=finance.ifeng.com}}</ref>

== 历史发展 ==
*1993年，“南京市雨润食品公司”在南京市注册。
*1996年，公司租用了南京罐头厂的生产线，并在后来收购该厂。
*1997年，公司收购了江苏东海肉联厂（连云港福润的前身）。
*1998年，公司改制为[[有限责任制|有限责任制]]，并更名为“南京雨润肉食品有限公司”。
*2002年，“南京雨润肉食品有限公司”更名为“江苏雨润食品产业集团有限公司”。
*2003年，成功收购哈尔滨肉联厂，并引进大众肉联品牌；
*2005年10月3日，雨润在[[香港联合交易所|香港联合交易所]]成功上市。
*2015年9月，該公司己與[[融創中國|融創中國]]達成戰略合作協議。<ref>[http://www.mpfinance.com/htm/finance/20150909/news/ec_eci1.htm 融創成雨潤戰略伙伴] 明報，2015年9月9日</ref>

== 遭到收购 ==
雨润集团于2010年受到美国[[高盛公司|高盛公司]]收购，这是高盛在中国继[[雙匯集團|双汇]]以来的第二次收购中国食品企业。

== 品牌运作 ==
雨润集团现有雨润、旺润、福润、大众肉联四个品牌，其产品包括[[冷鲜肉|冷鲜肉]]、[[冷冻肉|冷冻肉]]以及以[[猪肉|猪肉]]为主的低温肉制品、高温肉制品。

== 负面事件 ==
2009年4月[[安徽省|安徽省]][[质监局|质监局]]通报，[[阜阳|阜阳]]雨润肉类加工有限公司2009年3月16日生产的[[午餐肉|午餐肉]]经抽查被发现含有“[[克伦特罗|克伦特罗]]”([[瘦肉精|瘦肉精]])<ref>[http://news.qq.com/a/20090423/000527.htm 雨润午餐肉检出瘦肉精 院士称堪比毒奶粉]</ref>。雨润集团在事发之后表示该批次产品的总量只有100盒左右，并已全部[[召回|召回]]并销毁。同时也表示只有部分午餐肉由阜阳雨润肉类加工有限公司加工生产，产量很小，一年销售额大约只有10万元左右。

2011年5月雨润火腿疑掺过期肉被安徽合肥一家5星级酒店厨师揭发，公司欲以1万元私了。<ref>[http://finance.ifeng.com/news/bgt/20110525/4068338.shtml 雨润火腿疑掺过期肉被安徽合肥厨师揭发 公司欲一万元私了]</ref>

2011年6月，雨润集团旗下的渭南生秦肉类加工有限公司被发现一批含有大量三腺的“问题肉”，在不到几个小时的时间内，这些本应被处理的“问题肉”竟被“放行”。被新闻媒体曝光后，引来消费者一片骂声。<ref>[http://business.caing.com/2011-06-30/100274564.html 陕西渭南雨润“问题肉”被查扣后又放行]</ref>

2011年9月，雨润集团旗下的河南子公司有產品被驗出有瘦肉精殘留，有關消息令雨润集团股價一日下跌16.13%。<ref>[http://finance.sina.com.hk/news/41/2/1/4079057/1.html 雨潤再中招 暴跌16%]</ref>
