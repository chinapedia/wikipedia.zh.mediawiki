'''雨傘人'''（{{Lang-en|'''Umbrella Man'''}}）是指在[[香港|香港]][[雨傘革命|雨傘革命]]期間所拍攝的照片人物，後來引申至泛指參與雨傘革命及[[香港民主運動|香港民主運動]]的抗議群眾<ref name = "於慶中">{{Cite web |url = http://www.appledaily.com.tw/realtimenews/article/new/20141009/485076/ |title = 港榮休主教罵學生愚蠢　黃之鋒：佔領非少數人決定 |author = 於慶中 |date = 2014-10-09|publisher = 《[[蘋果日報_(台灣)|蘋果日報]]》 |language = zh-hant |accessdate = 2014-11-25 |quote =  }}</ref>。

== 背景 ==
=== 佔領發起 ===
[[File:Occupy_Central_Press_Con.jpg|thumb]]、[[戴耀廷|戴耀廷]]和[[陳健民_(社會學者)|陳健民]]。]]
早在2013年年初，[[香港大學|香港大學]]法律系副教授[[戴耀廷|戴耀廷]]、[[香港中文大學|香港中文大學]]社會學系副教授[[陳健民_(社會學者)|陳健民]]和[[基督教|基督教]][[浸信會|浸信會]]牧師[[朱耀明|朱耀明]]提出發起[[讓愛與和平佔領中環|佔領中環]]行動，試圖藉由佔領香港金融區[[中環|中環]]的交通要道這一[[公民不服從|公民不服從]]行動，來讓中國政府同意依照《[[中華人民共和國香港特別行政區基本法|中華人民共和國香港特別行政區基本法]]》[[提名委員會|第四十五條]]實施[[2016年及2017年香港政治制度改革|2016年及2017年香港政治制度改革]]。其中佔領中環主辦單位認為香港應當依照《[[公民權利和政治權利國際公約|公民權利和政治權利國際公約]]》，遵循其中的國際標準公平實踐「選舉權」、「被選舉權」和「提名權」的行政長官和立法會普遍選舉<ref>{{Cite web |url = http://oclp.hk/index.php?route=occupy/book_detail&book_id=10 |title = 和平佔中信念書 |author =  |date =  |publisher = [[讓愛與和平佔領中環|讓愛與和平佔領中環]] |language = zh-hant |accessdate = 2014年11月25日 |quote =  }}</ref><ref>{{Cite web |url = http://www.cna.com.tw/news/firstnews/201406215001-1.aspx |title = 香港「全民投票」與占領中環【特企】 |author =  |date = 2014年6月21日 |publisher = [[中央通訊社|中央通訊社]] |language = zh-hant |accessdate = 2014年11月25日 |quote =  }}</ref>。而如果政府對於這一平等選舉制度之要求無法實現的話，佔領中環主辦單位則整個行動最終會以非暴力的佔領模式進行<ref>{{Cite web |url = http://www.theguardian.com/world/2014/mar/06/occupy-central-hong-kong-democracy-campaign |title = Occupy Central gives downtown Hong Kong a taste of disobedience |author = Tania Branigan |date = 2014年3月6日 |publisher = 《[[衛報|衛報]]》 |language = en |accessdate = 2014年11月25日 |quote =  }}</ref>。反對者則擔憂活動會受到非理性行為的影響，而可能的暴力衝突將會嚴重影響香港經濟和市民日常生活。

從2013年至2014年期間，佔領中環主辦單位組織了3次商討會，並且就香港特別行政區行政長官選舉方式舉行[[6.22民間全民投票|6.22民間全民投票]]。之後主辦單位亦將公民投票結果寄送給[[香港特別行政區政府|香港特別行政區政府]]，但是對此政府則抱持強烈反對態度。

2014年8月31日，[[全國人民代表大會常務委員會|全國人民代表大會常務委員會]]作出有關2017年[[香港特別行政區行政長官|香港特別行政區行政長官]]選舉的決議<ref>{{Cite web |url = http://news.xinhuanet.com/english/china/2014-08/31/c_133609238.htm |title = Full text of NPC decision on universal suffrage for HKSAR chief selection |author = Yang Yi |date = 2014年8月31日 |publisher = [[新華網|新華網]] |language = en |accessdate = 2014年11月25日 |quote =  }}</ref>。其中全國人民代表大會常務委員會否決了有關於公民提名之意見，明確提到將由1,200名界別分組成員組成[[提名委員會_(香港)|提名委員會]]。之後必須在中華人民共和國政府嚴密監督下，由2名至3名獲得提名委員會過半票數者擔任香港特別行政區行政長官候選人，之後才讓香港民眾直接選舉，這項決定被視為是為了禁止香港民主派成為行政長官候選人的機會，即使獲得參選也必須得到建制派委員的支持，是變相的「篩選」<ref>{{Cite web |url = http://www.scmp.com/news/hong-kong/article/1582079/beijing-scholar-says-hong-kongs-candidate-nominating-system-out |title = Hong Kong's candidate nominating system out of balance, says Beijing scholar |author = Tony Cheung |date = 2014年8月31日 |publisher = 《[[南華早報|南華早報]]》 |language = en |accessdate = 2014年11月25日 |quote =  }}</ref>。

=== 抗爭展開 ===
[[File:2014.09.27_添美道-DSC_3516_(15192486828).jpg|thumb]]聚集。]]
佔領中環主辦單位原本預定在同年10月1日的[[中華人民共和國國慶節|中華人民共和國國慶節]]當天，對於框架決定作出實質的佔領中環行動作為回應<ref>{{Cite web |url = http://www.theguardian.com/world/2014/aug/31/hong-kong-activists-open-elections-ruled-out-beijing |title = Hong Kong activists vow to take over financial centre in election protest |author = Tania Branigan |date = 2014年8月31日 |publisher = 《[[衛報|衛報]]》 |language = en |accessdate = 2014年11月25日 |quote =  }}</ref>。不過早在9月22日時，[[香港專上學生聯會|香港專上學生聯會]]和[[學民思潮|學民思潮]]便宣布發起為期一周的[[2014年香港學界大罷課|罷課行動]]，藉由[[罷課|罷課]]的方式要求全國人民代表大會常務委員會退回所確定[[2016年香港立法會選舉|2016年香港立法會選舉]]、[[2017年香港特別行政區行政長官選舉|2017年香港特別行政區行政長官選舉]]的框架和候選人提名方案，以及反對香港特別行政區政府向全國人民代表大會的政治改革報告、爭取香港特別行政區行政長官選舉的[[真普選|公民提名權]]和廢除立法會[[功能组別|功能组別]]，以及香港特別行政區政府重新啟動政治改革諮詢與重新提交報告等訴求<ref>{{Cite web |url = http://www.cna.com.tw/news/firstnews/201409295004-1.aspx |title = 香港占領中環運動 關鍵10問 |author =  |date = 2014年9月30日 |publisher = [[中央通訊社|中央通訊社]] |language = zh-hant |accessdate = 2014年11月25日 |quote =  }}</ref><ref>{{Cite web |url = http://www.bbc.com/news/world-asia-china-29306128 |title = Thousands of Hong Kong students start week-long boycott |author =  |date = 2014年9月22日 |publisher = [[BBC新聞網|BBC新聞網]] |language = en |accessdate = 2014年11月25日 |quote =  }}</ref>。香港專上學生聯會和學民思潮鼓勵中學生與大學生參與罷課並應積極參與政治和社會事件之討論，並且聚集在政府總部門外等地抗議全國人民代表大會常務委員會的決定，然而政府的反應使得罷課行動逐步升級<ref>{{Cite web |url = http://www.theguardian.com/world/2014/sep/22/hong-kong-students-democracy-protest |title = Hong Kong students begin democracy protest |author =  |date = 2014年9月22日 |publisher = 《[[衛報|衛報]]》 |language = en |accessdate = 2014年11月25日 |quote =  }}</ref>。

9月26日晚上，學民思潮召集人[[黃之鋒|黃之鋒]]表示由於[[行政署|行政署]]突然興建鐵欄圍封位於[[政府總部_(香港)|政府總部]]大樓前方的公民廣場，因而要求重新開放公民廣場予以公共集會。之後其率領數百名參與罷課行動的示威學生突破安全圍欄，進佔自2014年7月便長期封鎖的[[重奪「公民廣場」行動|公民廣場]]，香港專上學生聯會亦很快給予支持。集會人士隨即分成兩路，一部分人推開立法會停車場出入口大門，另外一部分人則從立法會綜合大樓旁通道攀越3公尺高的圍欄，進佔公民廣場。很快地警方將廣場內的抗議群眾包圍並且限制其行動，最後鎮暴警察在第二天進駐政府總部並且強制清場，總共拘捕74人<ref>{{Cite web |url = http://www.scmp.com/news/hong-kong/article/1603471/scholarisms-joshua-wong-released-high-court-judges-instruction |title = Scholarism's Joshua Wong released at High Court judge's instruction |author =  |date = 2014年9月29日 |publisher = 《[[南華早報|南華早報]]》 |language = en |accessdate = 2014年11月25日 |quote =  }}</ref><ref>{{Cite web |url = http://www.bbc.com/news/world-asia-china-29378961 |title = Hong Kong democracy protesters enter government complex |author =  |date = 2014年9月27日 |publisher = [[BBC新聞網|BBC新聞網]] |language = en |accessdate = 2014年11月25日 |quote =  }}</ref>。發起罷課行動的學民思潮和香港專上學生聯會則繼續聚集在[[添美道|添美道]]，而學生罷課的舉動和香港警方的清場也使得大量民眾聚集在政府總部附近表達支持。到了當天晚上8時至11時，學民思潮和香港專上學生聯會更發起大型集會活動以要求釋放學生，集會結束後示威群眾仍然繼續留守<ref>{{Cite web |url = http://www.appledaily.com.tw/realtimenews/article/new/20140927/477608/ |title = 5萬港人晚間續抗命　藝人何韻詩感動站台 |author =  |date = 2014年9月2日 |publisher = 《[[蘋果日報_(台灣)|蘋果日報]]》 |language = zh-hant |accessdate = 2014年11月25日 |quote =  }}</ref>。

== 相片 ==
=== 使用催淚彈 ===
[[File:港人聲援佔中抗拒警方封鎖_(15).jpg|thumb]]
為了因應當晚局勢，9月28日凌晨1時40分佔領中環發起人戴耀廷宣布正式啟動其提出1年又8個月的[[讓愛與和平佔領中環|公民不服從行動]]，並且佔領中環之行動將從佔領香港政府總部作為起點<ref>{{Cite web |url = http://www.cna.com.tw/news/firstnews/201409280012-1.aspx |title = 港占中運動啟動 從政府總部起 |author = 張謙 |date = 2014年9月28日 |publisher = [[中央通訊社|中央通訊社]] |language = zh-hant |accessdate = 2014年11月25日 |quote =  }}</ref>。在宣布啟動佔領中環後，香港警方在同日上午開始封鎖政府總部，這使得示威群眾無法前往[[添馬|添馬]]支援佔領人士，這導致群眾於下午衝出[[夏慤道|夏慤道]]等主要幹道。很快地數千名示威群眾便佔領了[[香港島|香港島]]北部[[金鐘|金鐘]]兩條東西向的交通要道，成功封鎖了香港政府總部外的中央金融區主要幹道<ref>{{Cite web |url = http://www.bbc.com/news/world-asia-china-29390770 |title = Hong Kong police clear pro-democracy protesters |author =  |date = 2014年9月27日 |publisher = [[BBC新聞網|BBC新聞網]] |language = en |accessdate = 2014年11月25日 |quote =  }}</ref>。其中示威群眾在前往抗議現場的同時亦開始準備雨傘，準備雨傘的目的一方面是為了應對抗議現場的日曬雨淋等情況，另外一方面也可以防禦警察的[[胡椒噴霧|胡椒噴霧]]噴射或者接近示威群眾，也因此在抗議現場到處出現民眾攜帶雨傘的景象<ref name = "外媒關注盛讚">{{Cite web |url = http://www.appledaily.com.tw/appledaily/article/headline/20141002/36120918/ |title = 外媒關注盛讚 最有禮貌抗爭 |author =  |date = 2014年10月2日 |publisher = 《[[蘋果日報_(台灣)|蘋果日報]]》 |language = zh-hant |accessdate = 2014年11月25日 |quote =  }}</ref>。之後許多面套保鮮膜和醫用口罩等防護裝備的示威群眾數次嘗試突破警方設下的封鎖線，並且試圖藉由張開雨傘防禦來抵禦警方的胡椒噴霧之使用，進而擴大佔領範圍至政府總部周遭。

隨後[[香港警務處|香港警務處]]同意鎮暴警察針對抗議民眾使用[[催淚彈|催淚彈]]，接連使用了87枚催淚彈嘗試逼迫示威群眾[[9·28催淚彈驅散行動|撤離佔領現場]]<ref>{{Cite web |url = http://www.heraldsun.com.au/news/hong-kong-police-use-tear-gas-to-disperse-thousands-of-prodemocracy-demonstrators/story-e6frf7jo-1227073369950 |title = Hong Kong police use tear gas to disperse thousands of pro-democracy |author = [[美聯社|美聯社]] |date = 2014年9月30日 |publisher = 《{{Link-en|先驅太陽報|Herald Sun}}》 |language = en |accessdate = 2014年11月25日 |quote =  }}</ref>。隨著催淚彈的發射一度使得大量示威群眾短暫撤離抗議現場，然而大量新聞媒體在過程中拍攝到一名身穿黑衣、面戴口罩的男子，手持雨傘高舉並且走過警方發射的催淚彈煙霧之相片。有關相片以及其他民眾手持雨傘抵禦的相片隨即成為香港大規模民主示威的象徵，而身穿黑衣者甚至被稱作「雨傘人」（Umbrella Man）。其中[[雨傘革命的藝術#「雨傘人」相片|「雨傘人」相片]]很快在[[Twitter|Twitter]]上被轉發超過數千次，並且在其他社群網站服務上也被廣泛轉載<ref>{{Cite web |url = http://www.usatoday.com/story/news/world/2014/09/30/hong-kong-protests/16473507/ |title = Key social moments of Hong Kong protests |author = Alia E. Dastagir和Brett Molina |date = 2014年10月2日 |publisher = 《[[今日美國|今日美國]]》 |language = en |accessdate = 2014年11月25日 |quote =  }}</ref>。而許多群眾也將這天晚上所拍攝的警方嘗試以胡椒噴霧和催淚彈嘗試驅散示威群眾相片，陸陸續續標記為「雨傘革命」<ref>{{Cite web |url = http://www.usatoday.com/story/news/world/2014/09/29/hong-kong-democracy-protests/16413195/ |title = Hong Kong's 'umbrella revolution' opens wide |author = Calum MacLeod |date = 2014年9月30日 |publisher = 《[[今日美國|今日美國]]》 |language = en |accessdate = 2014年11月25日 |quote =  }}</ref>。對於鎮暴警察施放大量催淚彈和威脅開槍以驅散群眾的做法，大量不滿的示威群眾除了前往佔領金鐘、中環與灣仔路段外，當天晚上包括銅鑼灣、旺角和尖沙咀等人群密集地區也陸續遭到佔領<ref name = "於慶中"/>。

=== 「雨傘革命」 ===
由於示威群眾試圖以雨傘防禦胡椒噴霧和催淚彈的舉動，使得外國傳播媒體普遍將此次行動稱為「Umbrella Revolution」或「Umbrella Movement」，而中文直譯便是「雨傘革命」或「雨傘運動」。另外也有香港民眾將這次示威行動稱作「遮打運動」，其中「遮」在粵語中為傘、「打」寄意行動以打擊香港特別行政區行政長官[[梁振英|梁振英]]和推翻[[全國人民代表大會|全國人民代表大會]]決定為目標。同時「遮打」二字亦恰好呼應位於中環的遮打道，不過實際上遮打道並非有關行動的集結地點。

10月3日，美國雜誌《[[時代_(雜誌)|時代]]》亞洲版中以香港的示威行動作為封面專題報導，其中分別以「雨傘革命」（The Umbrella Revolution）為主標題、「香港爭取自由對中國而言是一次挑戰」（Hong Kong's fight for freedom is a challenge for China）為副標題。在報導中指出香港當前情況是因中央集權所造成，並且使得[[香港人|香港人]]發現其在政治上的無力感<ref>{{Cite web |url = http://hk.apple.nextmedia.com/realtime/international/20141002/52969090 |title = 【和平佔中】時代雜誌封面報道港佔中行動 |author =  |date = 2014年10月2日 |publisher = 《[[蘋果日報_(香港)|蘋果日報]]》 |language = zh-hant |accessdate = 2014年11月25日 |quote =  }}</ref>。而《時代》雜誌在封面照片挑選上，則採用香港警方在施放催淚彈時黑衣男子雙手手持雨傘的正面照，另外一旁還有同樣遭催淚彈攻擊的[[壹傳媒|壹傳媒]]董事會主席[[黎智英|黎智英]]<ref>{{Cite web |url = http://www.appledaily.com.tw/appledaily/article/headline/20141003/36122942/ |title = 「雨傘人」登《時代》封面 |author =  |date = 2014年10月3日 |publisher = 《[[蘋果日報_(台灣)|蘋果日報]]》 |language = zh-hant |accessdate = 2014年11月25日 |quote =  }}</ref>。

== 後續 ==
=== 事件評論 ===
由於雨傘實際上並無法阻擋催淚彈所發射的刺激性氣體，因此「雨傘人」拿著雨傘在催淚氣體中行走的圖像隨即引起大眾注意<ref>{{Cite web |url = http://www.881903.com/Page/ZH-TW/internetbitsdetail.aspx?itemid=750197&csid=821_2842 |title = 【薪火相傳】「雨傘人」成新抗爭標誌 |author =  |date = 2014年9月29日 |publisher = [[香港商業電台|香港商業電台]] |language = zh-hant |accessdate = 2014年11月25日 |quote =  }}</ref>，甚至成為雨傘革命中民主示威活動的象徵<ref name = "外媒關注盛讚"/>。《[[商業內幕|商業內幕]]》形容有關「雨傘人」的相片將成為香港攝影界的標誌影響，並且在往後多年仍然會不斷提及<ref>{{Cite web |url = http://www.businessinsider.com/umbrella-man-hong-kong-2014-9 |title = This Is The Iconic Hong Kong Photo The World Will Never Forget |author = Joe Weisenthal |date = 2014年9月29日 |publisher = 《[[商業內幕|商業內幕]]》 |language = en |accessdate = 2014年11月25日 |quote =  }}</ref>。而《[[今日美國|今日美國]]》則是於評論了在催淚彈煙霧中行進的黑衣人，將之與過去1989年[[六四事件|六四事件]]爆發後所拍攝的[[王維林|王維林]]阻擋坦克之形象相互對比，將其視為香港佔領中環版本的「坦克人」（Tank Man）<ref>{{Cite web |url = http://www.appledaily.com.tw/realtimenews/article/new/20141001/479788/ |title = 外媒連結香港與天安門　雨傘人vs坦克人 |author =  |date = 2014年10月3日 |publisher = 《[[蘋果日報_(台灣)|蘋果日報]]》 |language = zh-hant |accessdate = 2014年11月25日 |quote =  }}</ref>。臺灣資深傳媒工作者[[楊泰興|楊泰興]]則表示在催淚彈煙霧中持傘挺立的抗議身影，已經贏得了臺灣民眾的尊敬並且對於香港人有所改觀<ref>{{Cite web |url = http://hk.apple.nextmedia.com/news/art/20141009/18894497 |title = 台港命運重新連動 |author = [[楊泰興|楊泰興]] |date = 2014年10月2日 |publisher = 《[[蘋果日報_(香港)|蘋果日報]]》 |language = zh-hant |accessdate = 2014年11月25日 |quote =  }}</ref>。而前[[德國之聲|德國之聲]]中文部記者蘇雨桐更是在[[Twitter|Twitter]]上表示：「你們有槍我們有傘！」

除了新聞媒體外，許多示威群眾也開始紛紛使用「雨傘革命」來稱呼這次的抗議行動，並且把雨傘視為爭取民主和團結的象徵。「雨傘革命」和有關相片與影像很快便在各式[[社會化媒體|社會化媒體]]廣泛宣傳而在世界各地廣為熟知，然而隨著香港抗議活動持續進行使得中國政府仍盡可能阻止相關騷動消息流傳。早在9月28日時，中華人民共和國政府有關部門已經下達審查指示：「各網站對香港學生暴力衝擊政府和『佔中』相關信息一定要立即清理，有問題及時報告。嚴格管好互動欄目，堅決刪除有害信息。要嚴格執行。<ref>{{Cite web |url = http://chinadigitaltimes.net/2014/09/minitrue-delete-harmful-information-hong-kong/ |title = Minitrue: Delete Harmful Information on Hong Kong |author = Josh Rudolph |date = 2014年9月28日 |publisher = [[中國數字時代|中國數字時代]] |language = en |accessdate = 2014年11月25日 |quote =  }}</ref><ref>{{Cite web |url = http://www.scmp.com/news/hong-kong/article/1603840/doomed-protests-will-not-change-beijings-decision-political-reform |title = 'Doomed' protests will not change Beijing's decision on political reform, warns Chinese state paper |author = James Griffiths |date = 2014年9月29日 |publisher = 《[[南華早報|南華早報]]》 |language = en |accessdate = 2014年11月25日 |quote =  }}</ref><ref>{{Cite web |url = http://www.reuters.com/article/2014/09/29/us-hongkong-china-internet-idUSKCN0HO1KV20140929 |title = China censors try to blunt Hong Kong protests, don't always succeed |author = Paul Carsten |date = 2014年9月29日 |publisher = [[路透社|路透社]] |language = en |accessdate = 2014年11月25日 |quote =  }}</ref>」這使得網際網路審查人員迅速刪除包括「香港」、「路障」、「佔領中環」、「雨傘」等相關之訊息<ref>{{Cite web |url = http://www.nytimes.com/2014/10/01/world/asia/chinese-web-censors-struggle-with-hong-kong-protest.html |title = Chinese Web Censors Struggle With Hong Kong Protest |author = Andrew Jacobs |date = 2014年9月30日 |publisher = 《[[紐約時報|紐約時報]]》 |language = en |accessdate = 2014年11月25日 |quote =  }}</ref>。對此中國外交關係專家[[亞當·席格|亞當·席格]]（Adam Segal）便認為隨著時間經過中國政府將會封鎖更多關鍵字，甚至「我們很快見會看見『雨傘』遭到封鎖」<ref>{{Cite web |url = http://www.usatoday.com/story/news/world/2014/09/29/hong-kong-protests-social-media/16444213/ |title = Hong Kong vs. Tiananmen: Social media fuel 'umbrella revolution' |author = Alia E. Dastagir和Rick Hampson |date = 2014年9月30日 |publisher = 《[[今日美國|今日美國]]》 |language = en |accessdate = 2014年11月25日 |quote =  }}</ref>。

=== 形象使用 ===
之後佔領活動的集會人士經常一起撐起手中的[[雨傘|雨傘]]，顯示團結及堅持<ref>{{Cite web |url = http://news.now.com/home/local/player?newsId=113823 |title = 家長帶子女到佔中據點授公民教育 |author =  |date = 2014年10月1日 |publisher = [[now新聞台|now新聞台]] |language = zh-hant |accessdate = 2014年11月25日 |quote =  }}</ref>。而在10月1日所舉辦的[[中華人民共和國國慶節|中華人民共和國國慶節]]升旗儀式上，至少有數千名抗議民眾出現在升旗地點外圍並且撐開象徵性的雨傘，並且唱起歌曲《[[海闊天空_(Beyond歌曲)|海闊天空]]》以表示不滿<ref>{{Cite web |url = http://big5.soundofhope.org/node/512465 |title = 「雨傘人」成香港佔中運動象徵 爆紅網路 |author =  |date = 2014年10月2日 |publisher = [[希望之聲|希望之聲]] |language = zh-hant |accessdate = 2014年11月25日 |quote =  }}</ref>。而在稍後舉辦的國慶節酒會上，南區區議員[[司馬文|司馬文]]襟前貼上黃絲帶，並在酒會會場上打開黃色雨傘，以表示希望將外面示威群眾的聲音帶進會場<ref>{{Cite web |url = http://www2.hkej.com/instantnews/current/article/909626/%E5%8F%B8%E9%A6%AC%E6%96%87%E9%BB%83%E5%82%98%E7%8F%BE%E8%BA%AB%E5%9C%8B%E6%85%B6%E9%85%92%E6%9C%83 |title = 司馬文黃傘現身國慶酒會 |author =  |date = 2014年10月1日 |publisher = 《[[信報|信報]]》 |language = zh-hant |accessdate = 2014年11月25日 |quote =  }}</ref>。

不少藝術家在網際網路上自發地為「雨傘革命」設計標語和口號，並起加上黃絲帶來象徵和平追求普通選舉。其中視覺藝術家[[黃國才|黃國才]]便開設Facebook專頁以集合不同的設計，對此他指出 :「看到市民只是用雨傘抵擋，就感到對比是如此強烈，警方用殘暴手法對付市民，但市民只是用雨傘。雨傘本是平常物品，如今竟變成象徵反抗之物。<ref>{{Cite web |url = http://hk.apple.nextmedia.com/news/art/20141001/18885421 |title = 「雨傘人」成佔中版王維林 |author =  |date = 2014年10月1日 |publisher = 《[[蘋果日報_(香港)|蘋果日報]]》 |language = zh-hant |accessdate = 2014年11月25日 |quote =  }}</ref>」而網際網路著名漫畫家巴丟草則根據法國畫家[[歐仁·德拉克羅瓦|歐仁·德拉克羅瓦]]為紀念1830年[[法國七月革命|法國七月革命]]而創作的油畫《[[自由引導人民|自由引導人民]]》，另外改編了一幅命名為《雨傘領導人民》的漫畫。對此他表示：「香港佔中又過一夜，傘成為抗暴的標誌，她不比劍鋒利，不比盾結實，卻保護香港民主自由的種子，待明天發出新芽！<ref>{{Cite web |url = http://www.voachinese.com/content/error-404-umbrella-revolution-20141003/2471943.html |title = 火墙内外：公民抗争 雨伞革命 |author = 江河和海伦 |date = 2014年10月2日 |publisher = [[美國之音|美國之音]] |language = zh-hans |accessdate = 2014年11月25日 |quote =  }}</ref>」
