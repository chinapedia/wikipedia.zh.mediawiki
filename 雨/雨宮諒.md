'''雨宮諒'''{{BD|1980年|||}}，[[日本|日本]][[輕小說|輕小說作家]]。出身於[[愛媛縣|愛媛縣]]。代表作是2008年1月至3月播出同名電視動畫的《[[死後文|死後文]]》。

== 人物簡介 ==
2003年，以短篇《{{lang|ja|おじいちゃんの宝}}箱》參加[[電擊文庫|電擊文庫]]（[[MediaWorks|MediaWorks]]／[[ASCII_Media_Works|ASCII Media Works]]）舉辦的第10回[[電擊小說大獎|電擊小說大獎]]獲得選考委員獎勵獎。翌年2004年正式出道。

== 作品概覽 ==
所有作品在[[電撃文庫|電撃文庫]]刊載（[[MediaWorks|MediaWorks]]／[[ASCII_Media_Works|ASCII Media Works]]發行）。
