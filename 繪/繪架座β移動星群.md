'''繪架座β移動星群'''是位於附近相對上較接近[[地球|地球]]的一個年輕[[移動星群|移動星群]]。在[[天文學|天文學]]中，移動星群是在太空中有著相同的起源並有著共同運動的一群恆星集團。

繪架座 β移動星群是天文學上一個很重要的研究對象，因為他與[[太陽系|太陽系]]的距離相當接近，而且在恆星發展的過程中也還很年輕。已經知道繪架座β有氣體和塵埃組成的巨大盤面，可能是[[原行星盤|原行星盤]]；也有證據顯示有年輕的[[類木行星|氣體巨星]]環繞著<ref>[http://www.extrasolar.net/planettour.asp?StarCatId=%20&PlanetId=58 Extrasolar Visions - beta Pictoris b<!-- Bot generated title -->] {{webarchive|url=https://web.archive.org/web/20101117125022/http://extrasolar.net/planettour.asp?StarCatId=&PlanetId=58 |date=2010-11-17 }}</ref>。

這個星群的年齡和距離使它成為搜尋[[系外行星|系外行星]]，和研究行星系統是如何形成的首選目標。

== 組織 ==
繪架座β移動星群由28顆單獨恆星組成，包含了17個行星系，還有被確認的[[棕矮星|棕矮星]]。集團的核心距離地球大約115光年，成員的年齡在1,000萬年至3,000萬年之間。

集團主要由低溫、低亮度的K和M[[光譜類型|類型]]的恆星構成，多數都是肉眼看不見的。繪架座β是一個例外，其他肉眼可見的還有：

*[[望遠鏡座η|望遠鏡座η]]
*[[波江座51|波江座51]]
*[[HD_203|HD 203]]
*[[HD_146624|HD 146624]]
*[[HD_165189|HD 165189]]
*[[HD_172555|HD 172555]]

集團成員所在的空間主要區域被侷限在[[南半球|南半球]]，散佈在[[波江座|波江座]]、[[天兔座|天兔座]]、[[繪架座|繪架座]]、[[天蝎座|天蝎座]]和[[望遠鏡座|望遠鏡座]]。

== 發現 ==
早先估計繪架座β的年齡是1,000萬年，但由於恆星在空間中幾乎都是被隔離的而難以証明。依據當前關於[[恆星演化|恆星演化]]的理論，極端年輕的恆星在太空中應該仍然和在同一區域內同時形成的年輕恆星在一起，而還沒有到被[[引力|引力交互作用]]造成與其他的兄弟姐妹離散的階段。

直到1999年才發現一對昏暗的[[紅矮星|紅矮星]]有著和繪架座β相似的年齡和空間速度，使這個問題得以解決，對年齡的估計也得到了佐證<ref>{{cite journal|author = Barrado y Navascués, David; Stauffer, John R.; Song, Inseok; Caillault, J.-P.|title = The Age of beta Pictoris|journal = The Astrophysical Journal|date = 1999-08-01 |volume = 520|issue = 2|pages = L123–L126|url = http://adsabs.harvard.edu/abs/1999ApJ...520L.123B|accessdate = 2007-05-14|doi = 10.1086/312162}}</ref>。

在2001年發布的資料更近一步確認有17個[[恆星系統|恆星系統]]有著和繪架座β相似的運動和年齡，構成了繪架座β移動星群，並以最明顯的繪架座β做為命名的依據<ref>http://www.journals.uchicago.edu/doi/full/10.1086/337968{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。
