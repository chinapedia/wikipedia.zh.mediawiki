'''觸角'''亦稱為'''觸鬚'''，是指某些[[有爪動物|有爪動物]]、[[節足動物|節足動物]]或是[[軟體動物|軟體動物]]等生長於[[頭部|頭部]]的一種[[感覺器官|感覺器官]]。大部份都生長於頭部的兩側，具有[[聽覺|聽覺]]、[[觸覺|觸覺]]以及[[嗅覺|嗅覺]]等功能<ref>{{cite web |language=zh-hant |url=http://dict.revised.moe.edu.tw/cgi-bin/newDict/dict.sh?idx=dict.idx&cond=%C4%B2%A8%A4&pieceLen=50&fld=1&cat=&imgFont=1 |title=教育部重編國語辭典修訂本：詞句的解釋 }}{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。形狀有許多種類，常見的有絲狀、鞭狀、念珠狀、鋸齒狀、櫛齒狀、羽狀等<ref name="a">{{cite web |language=zh-hant |url=http://icontent.nkps.tp.edu.tw/insectinfo/Intro.aspx?taxonID=52&specID=697 |title=認識昆蟲-昆蟲的頭部 }}；觸角種類的圖文說明。</ref>。除此之外，例如[[蝸牛|蝸牛]]頭上的軟角，或是[[蝦|蝦]]的長鬚等，也都是屬於觸角。

=== 節足動物 ===
{{main|節肢動物門}}
大部份的節足動物都會擁有觸角，不過也有些是因為[[進化|進化]]影響造成的[[變態|變態]]，像是[[螯肢亞門|螯肢亞門]]（''Chelicerata''）雖然是屬於節肢動物門，但是就沒有觸角，例如[[蜘蛛|蜘蛛]]、[[蠍子|蠍子]]或是[[鱟|鱟]]等<ref>{{cite web |language=ja |url=http://www5b.biglobe.ne.jp/~hilihili/keitou/mizika/seesk-others/kyoukaku.html |script-title=ja:ケリケラータ／鋏角類 |trans_title=螯肢亞門的解說 }}</ref>。

長在[[口前葉|口前葉]]上的觸角是節足動物重要的感覺器官，也是一種特徵。通常節足動物的口前葉上在口器的後面的稱為[[附肢|附肢]]，而在口器前方的就是觸角<ref>{{cite web |language=ja |url=http://www.weblio.jp/content/%E9%A0%AD%E9%83%A8#.E7.92.B0.E5.BD.A2.E5.8B.95.E7.89.A9.E3.81.AE.E5.A0.B4.E5.90.88 |script-title=ja:頭部 とは とうぶ }}；參看其中節足動物說明一節。}}</ref>。節足動物門裡也有些是以觸角的型態來分類，例如[[昆蟲綱|昆蟲綱]]的觸角是一對，[[甲殼綱|甲殼綱]]的觸角有兩對以及[[蛛形綱|蛛形綱]]沒有觸角等<ref>{{cite web |language=zh |url=http://esky.chc.edu.tw/~science/4/%B8U%BF%B3%A9%F8%C2%CE/%AFS%BCx.htm |title=昆蟲特徵 }}{{dead link}}</ref>。

=== 軟體動物 ===
{{main|軟體動物門}}
在大多數的軟體動物頭部上方也會有一對柔軟的突起物，尖端呈現比較細的針狀或是膨脹變成球狀的都有，而且有數種類在其尖端的地方會有[[眼睛|眼睛]]，所以這類與一般常見觸角的功能之外還擁有[[視覺|視覺]]。{{Fact}}<!-- 日語維基並非[[WP:可靠來源|WP:可靠來源]] -->

=== 有爪動物 ===
{{main|有爪動物門}}
在其頭部的背上擁有一對觸角，大致上與部份節足動物的觸角一般，主要的特徵在於觸角的結構是有許多細小關節所組成，{{Fact}}<!-- 日語維基並非[[WP:可靠來源|WP:可靠來源]] -->其代表性的就是[[櫛蠶|櫛蠶]]。

== 主要功能 ==
觸角一般都是長在頭部上方或前方所延伸出來的，普通節肢動物之中都會在其基部擁有關節，是個可動的感覺器官。感覺功能在每個分類之下的動物都會有些不同，有觸覺、熱覺、音覺、味覺、視覺或是氣流變化的感覺等，甚至雄性的[[蛾|蛾類]]觸角還能捕捉到[[雌性|雌性]]蛾所發出類似性引誘的[[信息素|信息素]]。還有像是[[水蚤|水蚤]]（[[:en:Daphnia|Daphnia]]）之類的動物也會將觸角當作在水中游泳的器官。<ref>{{ja}}[http://star310.com/90478A70/w96F08A84 {{lang|ja|触角の役割}}]（觸角的功能）。 {{webarchive|url=https://web.archive.org/web/20071008195059/http://star310.com/90478A70/w96F08A84 |date=2007-10-08 }}</ref>
