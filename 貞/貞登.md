'''貞登'''（{{jpn|j=貞登|hg=さだ の みのる/さだ の のぼる|rm=Sada no Minoru/Sada no Noboru}}，{{bd|？||？||catIdx=Sada no Minoru}}）是[[日本|日本]][[平安時代|平安時代]]前期[[貴族|貴族]]，父親是[[仁明天皇|仁明天皇]]。官位為[[正五位|正五位]]下[[備中國|備中守]]。

== 生涯 ==
[[承和_(仁明天皇)|承和]]年間初期（835年左右），貞登被父親[[仁明天皇|仁明天皇]]賜姓[[源氏|源]]{{Link-ja|朝臣 (日本)|朝臣|朝臣}}而[[臣籍降下|臣籍降下]]。承和12年（845年），貞登的生母三國町懷疑與{{Link-ja|藤原有貞|藤原有貞}}通姦而被褫奪[[更衣_(職官)|更衣]]身份，貞登也被逼出家，改稱法號深寂。[[嘉祥|嘉祥]]年間後期（851年左右），貞登獲其兄長[[文德天皇|文德天皇]]賜予{{Link-ja|時服|時服}}和{{Link-ja|月料|月料}}，並且在文德天皇抱恙期間，與時康親王（後來的[[光孝天皇|光孝天皇]]）一同負責在藥中{{Link-ja|試毒|毒見}}。然而，由於貞登是出家人，因此文德天皇死後無法分得其任何財產<ref name="a">《[[日本三代實錄|日本三代實錄]]》貞観8年3月2日条</ref>。

其後，貞登需要過著一般人的生活，亦有生兒育女，但是身份仍然是僧侶，無籍貫的同時亦未能出仕於朝廷，生活相當拮据。對此，在[[貞觀_(清和天皇)|貞觀]]8年（866年），貞登的兄弟時康親王與本康親王等向朝廷上奏，讓貞登[[還俗|還俗]]以及恢復{{Link-ja|本姓|本姓}}，並且獲得批准，可是由於按照[[嵯峨天皇|嵯峨上皇]]的遺言，生母有過失之人無法成為源氏，因此貞登無法恢復源朝臣姓，只能冠以貞朝臣登的姓名，並且升至[[正六位|正六位]]上，籍貫則獲分配為右京一條之一坊（現[[京都府|京都府]][[京都市|京都市]][[右京區|右京區]]）<ref name="a" />

貞觀9年（867年），貞登{{Link-ja|叙爵|叙爵}}，升至[[從五位|從五位]]下<ref>《日本三代實錄》貞観9年正月7日条</ref>、貞觀19年（877年）升至從五位上<ref>《日本三代實錄》貞観19年正月3日条</ref>、[[元慶|元慶]]9年（885年）出任[[備中國|備中守]]<ref>《日本三代實錄》元慶8年正月16日条</ref>。[[宇多天皇|宇多天皇]]在位期間的[[寬平|寬平]]6年（894年），貞登升至[[正五位|正五位]]下，但是死亡日期則沒有文獻記載。

與此同時，貞登作為勅撰[[歌人|歌人]]，有一首[[和歌|和歌]]收錄於《[[古今和歌集|古今和歌集]]》<ref>《勅撰作者部類》</ref>。
