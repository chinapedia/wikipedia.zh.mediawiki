== 创作背景 ==
1947年5月，[[上海交通大学|交通大学]]护校运动中被捕关押于[[上海警备司令部|上海警备司令部]]的[[邢幼青|邢幼青]]等40多位学生将当时流行歌曲《'''[[跌倒算什么|跌倒算什么]]'''》改成《'''坐牢算什么'''》。<ref>{{Cite web |url= http://topics.sjtu.edu.cn/newsnet/9thdisplay.php?id=18232 |title= 地下党领导的新民主主义革命运动（1925-1949） |author=  |date=  |publisher= 上海交通大学第九次党代会专题网 |language= zh-hans |accessdate= 2011-05-13 |quote= 8、前“捕”后继的宣传队 }}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

== 东方红 ==
该曲在1964年10月2日[[人民大会堂|人民大会堂]]演出的大型音乐舞蹈史诗《[[东方红_(音乐舞蹈史诗)|东方红]]》中为“[[领唱|领唱]]、[[齐唱|齐唱]]”，位于第五场《'''埋葬蒋家王朝'''》<ref name="wyzxsx.com">{{Cite web|url=http://www.wyzxsx.com/Article/Class12/200902/69889.html|title=大型音乐舞蹈史诗《东方红》歌词和朗诵词|accessdate=2011-05-08|date=2009-02-14|format=|publisher=乌有之乡|language=zh-hans|deadurl=yes|archiveurl=https://web.archive.org/web/20090903035839/http://www.wyzxsx.com/Article/Class12/200902/69889.html|archivedate=2009-09-03}}</ref>。
