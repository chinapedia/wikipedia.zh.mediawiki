'''激进左翼聯盟'''（{{lang-el|Συνασπισμός Ριζοσπαστικής Αριστεράς|Synaspismós Rizospastikís Aristerás}}），简称'''激左盟'''（{{lang|el|ΣΥΡΙΖΑ}}，SYRIZA），是[[希腊|希腊]]的一个政党。激进左翼联盟原为由多個反對財政緊縮政策的[[左翼|左翼]]政黨和獨立人士組成的聯盟，后改组为统一的政党。其前身是“争取左翼团结和共同行动对话空间”。

== 起源 ==
1989年－1991年，受[[苏东剧变|苏东剧变]]影响，[[希腊共产党|希腊共产党]]内部发生严重分裂，主张全面否定[[马列主义|马列主义]]和取消党的“革新派”，联合[[运动和生态左翼联盟|左翼和进步联盟]]（当时由希腊共产党领导的左翼[[政党联盟|政党联盟]]）内其它党派（主要是[[希腊左翼|希腊左翼]]），试图从改变党的纲领和改选领导机构成员入手，进而全面改变联盟性质，将其变成一个“统一、自主、多派别”的政党。<ref name=tu>于海青. 希腊“激进左翼联盟”的崛起与发展前景[J]. 科学社会主义, 2013(4):151-154.</ref>1991年6月23日，希共“正统派”在牢固掌握党的领导权、挫败“革新派”改变党的性质的企图后，将“革新派”的代表人物清除出中央委员会和政治局，并宣布退出左翼和进步联盟。包括近一半希共中央委员和五分之二党员在内的“革新派”遂自行退党，加入左翼和进步联盟，将其改组为一个统一的政党。<ref>{{cite web |url=http://myy.cass.cn/news/747055.htm |title=存档副本 |accessdate=2015-01-27 |deadurl=yes |archiveurl=https://web.archive.org/web/20150225102448/http://myy.cass.cn/news/747055.htm |archivedate=2015-02-25 }}</ref> 

2001年，左翼和进步联盟与其它数个左翼党派（这些党派大部分脱胎于[[希腊共产党|希腊共产党]]，包括[[托洛茨基主义|托洛茨基主义]]者、[[毛主义|毛主义]]者、[[欧洲共产主义|欧洲共产主义]]者）组成名为“争取左翼团结和共同行动对话空间”的松散联盟。2003年，左翼和进步联盟更名为“运动和生态左翼联盟”。2004年，在“争取左翼团结和共同行动对话空间”的基础上，组建了名为“激进左翼联盟”的政党联盟，运动和生态左翼联盟是该联盟的主要力量。

== 发展 ==
2004年1月，激进左翼聯盟作为一个[[政党联盟|政党联盟]]成立，由以下单位组成：
* [[运动和生态左翼联盟|运动和生态左翼联盟]]
* [[革新共产主义生态左翼|革新共产主义生态左翼]]
* [[国际主义工人左翼|国际主义工人左翼]]
* [[争取行动左翼联合运动|争取行动左翼联合运动]]
* [[行动公民_(希腊)|行动公民]]
* 其他独立左翼团体和活动家

[[2004年希腊议会选举|2004年希腊议会选举]]，在[[希臘議會|希臘議會]]中有6席。[[2007年希腊议会选举|2007年希腊议会选举]]中增加至14席，[[2009年希腊议会选举|2009年希腊议会选举]]中降为13席。

2011年[[欧债危机|欧债危机]]爆发后，在希臘反對經濟緊縮的呼聲中，激进左翼聯盟在[[2012年5月希腊议会选举|2012年5月希腊议会选举]]中大有斬獲，得票率超过16%，取的52个席次，躍升成第二大黨，第一大黨的保守派[[新民主黨_(希臘)|新民主黨]]組閣失敗後，獲得組閣權，但最終失敗，而向總統交出組閣權。同月底正式组成统一政党，不少成員由原[[泛希臘社會主義運動|泛希臘社會主義運動]]退黨加入，激进左翼联盟取代泛希臘社會主義運動成為最大的[[中间偏左|中左翼]]政黨。

[[2012年6月希腊议会选举|2012年6月希腊议会选举]]中，该党取得26%的支持率，并取得71席，成为议会第二大党和最大[[反对党|反对党]]。同月25—26日，该党组织罢工者和抗议者联合举行雅典总罢工。在此次罢工后，激进左翼联盟的民测支持率明显提升<ref>{{cite web |url=http://myy.cass.cn/news/727610.htm |title=存档副本 |accessdate=2015-01-27 |deadurl=yes |archiveurl=https://web.archive.org/web/20150723215429/http://myy.cass.cn/news/727610.htm |archivedate=2015-07-23 }}</ref>。

[[2015年希腊国会选举|2015年1月希腊大选]]中，该党取得36.34%的支持率，并取得149席<ref>http://www.skai.gr/ekloges2015</ref>，成为议会第一大党與希臘政府組織主導權，並在稍後與在本次選舉中排名第六的右翼政黨[[獨立希臘人|獨立希臘人]]共組聯合政府<ref>http://www.bbc.com/news/world-europe-30981950</ref>。

2015年8月21日，以[[帕纳约蒂斯·拉法扎尼斯|帕纳约蒂斯·拉法扎尼斯]]为首的25名希腊议会议员为表达对时任希腊总理、激进左翼联盟领袖[[阿列克西斯·齐普拉斯|阿列克西斯·齐普拉斯]]所推行的政策的不满，宣布退出激进左翼联盟，另外组建名为“[[人民团结_(希腊)|人民团结]]”的政党。

== 意识形态 ==
激进左翼联盟现时自我定位为既非[[共产党|共产党]]也非[[社会民主党|社会民主党]]的[[民主社会主义|民主社会主义]]多元左翼政党。它支持[[混合经济|混合经济]]，尤其强调[[女权主义|女权主义]]、民主权利和[[环境保护|环境保护]]等“新政治议题”，又称「[[另类全球化|另类全球化]]」。其成员主要包括那些致力于克服[[资本主义|资本主义]]弊病的具有[[激进民主主义|激进民主主义]]和[[生态|生态]]导向的力量，以及一些要求发展[[民主|民主]]和强调民族自决权的进步人士。因此，激进左翼联盟虽然起源于[[工人运动|工人运动]]，但并非一个[[工人阶级|工人阶级]]政党。如[[政治学家|政治学家]]大卫•阿特所言：“我们不再是共产党人，我们永远不可能是社会民主党人。”<ref>{{cite web |url=http://myy.cass.cn/news/727610.htm |title=存档副本 |accessdate=2015-01-27 |deadurl=yes |archiveurl=https://web.archive.org/web/20150723215429/http://myy.cass.cn/news/727610.htm |archivedate=2015-07-23 }}</ref><ref>{{cite web |url=http://myy.cass.cn/news/747055.htm |title=存档副本 |accessdate=2015-01-27 |deadurl=yes |archiveurl=https://web.archive.org/web/20150225102448/http://myy.cass.cn/news/747055.htm |archivedate=2015-02-25 }}</ref>
