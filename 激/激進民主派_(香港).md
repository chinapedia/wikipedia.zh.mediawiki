'''激進民主派'''（Radical democracy camp），亦稱為'''進步民主派'''（Progressive Democrats），是自2010年政改爭議以來，用以分辨同為廣義[[泛民主派|泛民主派]]的[[溫和民主派_(香港)|主流民主派]]（Moderate Democrats）。[[親中媒體|親中媒體]]廣義稱之為「激進反對派」<ref>[http://news.wenweipo.com/2015/04/27/IN1504270081.htm 各界譴責激進反對派毒化政改氛圍]</ref>或「極端反對派」<ref>[http://paper.people.com.cn/rmrbhwb/html/2014-09/03/content_1473333.htm 香港极端反对派不要沦为历史罪人]</ref>。現組成[[香港立法會|香港立法會]]泛民主派第四大黨[[議會陣線|議會陣線]]。雖然在民主派中佔少數，但具有重要影響力並長期主導議會抗爭，2018年[[毛孟靜|毛孟靜]]首次成為[[民主派會議|民主派會議]]召集人。

激進民主派要求香港立即實現[[真普選|真普選]]，要求行政長官和立法會儘快推行全面[[普遍選舉|普選]]、注重監察政府運作、視[[人權|人權]]、[[平等|平等]]、[[公義|公義]]、[[言論自由|言論自由]]等為社會的重要價值，與主流民主派人士相同，最初源於抗爭手段有異，激進民主派主張在立法會進行激烈的非暴力抗爭，與{{DPHK}}、{{CPHK}}等主流民主派出現政治分歧，但皆認同以[[非暴力|非暴力]]手段抗爭，並反對民主派分裂（早在2011年部分激進民主派反對與{{DPHK}}撞區以達致「票債票償」的局面）。激進民主派在其他政治或經濟理念上多偏向[[左翼|左翼]]，包括[[中港矛盾|中港矛盾]]及[[酷刑聲請|難民黑工]]問題，在[[本土主義|本土主義]]的立場也有分歧。

== 被視為激進的主因 ==
2010年[[五區總辭|五區總辭]]一事之前，此概念較不明顯，只用以形容立場較不鮮明的泛民主派成員；並作為相對概念，但整體泛民均會有所合作。不過，自[[社民連|社民連]]發動「[[2010年香港立法會地方選區補選|五區公投]]」後，傳媒就開始使用此詞形容並參加此項目的泛民主派，當中包括社民連及社運青年。

== 分裂 ==
[[民主黨_(香港)|民主黨]]於2010年1月宣布不支持五區公投，其後更於6月23日支持[[2012年香港政治制度改革|2012年政改方案]]，支持並推動五區公投的激進民主派指民主黨支持政改是導致泛民分裂的主因，故民主黨為[[建制派_(香港)|建制派]]而非真正民主陣營。不過，泛民內部亦有不少聲音指出泛民不應在此時分裂，反對狙擊民主黨。

數月後，由於社民連不少黨員反對在[[2011年香港區議會選舉|2011年香港區議會選舉]]「狙擊」民主黨，導致不少黨員退黨，於2011年1月23日另組[[人民力量|人民力量]]。人民力量在[[2011年香港區議會選舉|2011年香港區議會選舉]]「狙擊」民主黨，但最終，人民力量所派候選人除[[民主陣線|民主陣線]]的麥業成外全數落敗，包括主席[[劉嘉鴻|劉嘉鴻]]、立法會議員[[陳偉業|陳偉業]]、錢偉洛、[[歐陽英傑|歐陽英傑]]、[[任亮憲|任亮憲]]、[[李偉儀|李偉儀]]、[[陳志全|陳志全]]、趙植東、符偉樂等人。而且有16人因得票少於5%遭沒收保証金。黃毓民宣佈區議會「票債票償」運動失敗。

2012年後，因[[中港矛盾|中港矛盾]]、[[中港關係|中港關係]]及香港政治改革爭議等問題，以及對[[香港新移民|內地來港人士]]態度等問題，激進民主派出現嚴重分裂，分裂出[[本土派_(香港)|本土派]]。

其中[[社會民主連線|社會民主連線]]、[[工黨_(香港)|工黨]]等[[左翼|左翼]]傾向的政黨因主張同情內地來港[[香港人士權益，而過去曾狙擊民主黨的[[人民力量|香港人士權益，而過去曾狙擊民主黨的[[人民力量]]自2013年黃毓民退黨，以及2015年與民主黨共同政改否決後，亦不再狙擊民主黨，全面與民主派恢復合作的關係，在[[2016年香港立法會地方選區補選|2016年香港立法會地方選區補選]]，社民連及人力均支持公民黨的[[楊岳橋|楊岳橋]]。

黃毓民參與的[[普羅政治學苑|普羅政治學苑]]、[[熱血公民|熱血公民]]等組織轉向[[本土派|本土派]]。

== 政黨合作 ==
[[2016年香港立法會新界東地方選區補選|2016年新界東補選]]，激進民主派與溫和民主派共同推舉及支持[[公民黨_(香港)|公民黨]]的候選人[[楊岳橋|楊岳橋]]。自從[[2016年農曆新年旺角騷亂|2016年旺角騷亂]]後，本土派人氣大升。激進民主派繼續支持楊岳橋，最終楊岳橋勝出。但補選結果卻反映激進民主派不少支持者流向本土派。

新界東補選後，為了應付選戰，社民連和人民力量組成[[進步民主連線|進步民主連線]]參與[[2016年香港立法會選舉|2016年立法會選舉]]<ref>[http://linepost.hk/?uid=4471 LinePost線報 - 合久必分，分久必合？社民連和人力，計劃合組「進步民主派」出戰立會選舉]</ref>。最終只有新界東的原議員[[陳志全|陳志全]]（人民力量）及[[梁國雄|梁國雄]]（社會民主連線）成功當選，而其他地區如九龍西及香港島低票落敗，九龍東及新界西則因與其他泛民及本土派競爭而高票落敗。

== 以往被視為「激進民主派」的組織 ==
* [[港同盟|港同盟]]（1990－1994）
[[港同盟|港同盟]]未和[[匯點|匯點]]合併前被視為「激進民主派」，是[[激進|激進]][[民主|民主]]力量的[[急先鋒|急先鋒]]，雖然主張激進，但從未採取暴力抗爭，主張議會民主路線，港同盟和滙點合併後成立[[民主黨_(香港)|民主黨]]，走溫和的路線，為[[溫和民主派_(香港)|溫和民主派]]。其後民主黨再和[[前綫_(1996年)|前綫]]（1996-2008）合併，繼續主張以和平方式促進民主。
* [[前綫_(1996年)|前綫]]（1996－2008）
政綱內明示香港應全民制憲，廢棄由中英雙方商議而無港人廣泛共識的《[[香港基本法|香港基本法]]》。2008年11月23日併入民主黨，惟不少成員並未跟隨加入。
* [[社會民主論壇|社會民主論壇]]（1997－2002）
