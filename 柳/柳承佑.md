'''柳承佑'''（{{lang|ko|류승우}}，{{bd|1993年|12月17日|||Ryu, Seung-woo}}），[[韩国|韩国]][[足球|足球]]运动员，现效力于[[K聯賽1|南韓甲組職業足球聯賽]]球队[[濟州聯足球俱樂部|濟州聯]]。

==俱乐部生涯==
2013年11月6日，柳承佑和[[经典K联赛|经典K联赛]]球队[[濟州聯足球俱樂部|济州联]]签订职业生涯的首份合同，不过一个月后，球队宣布将他租借到[[德国足球甲级联赛|德国足球甲级联赛]]球队[[勒沃库森足球俱乐部|勒沃库森]]。<ref>{{cite web |url=http://sports.donga.com/3/all/20131213/59530002/2|language=ko |title=류승우, 제주 입단 한 달만에 ‘레버쿠젠 임대’ |publisher=Sports DongA |date=13 December 2013}}</ref> 2014年1月25日，柳承佑在勒沃库森2比3不敌[[弗赖堡体育俱乐部|弗莱堡]]的比赛下半场替补球队中的另一个韩国人[[孙兴慜|孙兴慜]]出场，上演职业生涯首秀。<ref>{{cite web |url=http://www.bundesliga.com/en/liga/news/2013/0000284342.php |title=Fearless Freiburg Fight Back Twice for Leverkusen Scalp |publisher=Bundesliga |date=25 January 2014 |accessdate=25 January 2014 |deadurl=yes |archiveurl=https://web.archive.org/web/20140201080058/http://www.bundesliga.com/en/liga/news/2013/0000284342.php |archivedate=2014年2月1日 }}</ref> 他在2013/14赛季一共在德甲联赛中得到2次出场机会。
