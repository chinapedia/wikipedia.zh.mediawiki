'''柳浑'''（{{bd|716年||789年|2月23日|catIdx=L}}<ref>[[:s:旧唐书/卷13|《旧唐书》卷一十三]]</ref>），本名'''柳载'''，本字'''元舆'''，后字'''夷旷'''，一字'''惟深'''，[[襄州_(西魏)|襄州]]人，祖籍[[河东郡_(中国)|河东郡]][[解县|解县]]（今[[山西省|山西省]][[运城市|运城市]][[临猗县|临猗县]]），封爵'''宜城县伯'''<ref>柳浑为伯爵的记载是根据他在两唐书中的传记。《旧唐书·德宗本纪》称他为宜城县子，而非伯爵。比较[[:s:旧唐书/卷125|《旧唐书》卷一百二十五]]、[[:s:新唐书/卷142|《新唐书》卷一百四十二]]、《旧唐书》卷一十三和[[柳宗元|柳宗元]]《故银青光禄大夫右散骑常侍轻车都尉宜城县开国伯柳公行状》。</ref>，谥号'''贞'''，[[唐朝|唐朝]][[官员|官员]]，[[唐德宗|唐德宗]]年间拜为[[宰相|宰相]]。

== 家世 ==
柳载出自[[河东柳氏|河东柳氏]]东眷，六代祖[[柳惔|柳惔]]是南梁仆射。曾祖父柳善才，荆王侍读。祖父柳尚素，[[润州|润州]][[曲阿|曲阿]]县令。父柳庆休，[[渤海郡|渤海郡]]县丞。<ref>{{cite web |url=http://www.sidneyluo.net/a/a17/073.htm |title=存档副本 |accessdate=2009-04-08 |0=2009-02-02 |deadurl=yes |archiveurl=https://web.archive.org/web/20090202171923/http://sidneyluo.net/a/a17/073.htm |archivedate=2009-02-02 }}[http://www.sidneyluo.net/a/a17/table/form65.htm 《新唐书》卷七十三]  {{webarchive|url=https://web.archive.org/web/20120203002025/http://www.sidneyluo.net/a/a17/table/form65.htm |date=2012-02-03 }}</ref>后赠[[蔡州_(西魏)|蔡州]][[刺史|刺史]]、[[工部尚书|工部尚书]]。<ref name=LZY>柳宗元《故银青光禄大夫右散骑常侍轻车都尉宜城县开国伯柳公行状》</ref>

柳载父柳庆休早卒。他好学，但因父亲早亡而贫穷。十多岁时，有巫者告之：“你相貌早夭且贱，做和尚可缓死，禄位不是你的事。”叔伯们想从其言，柳浑说：“抛弃圣教，行异术，不如速死。”更加好学，与名士们同游。唐玄宗[[开元|开元]]年间中[[汝州|汝州]][[进士|进士]]，<ref>此从《行状》。《两唐书》皆作[[天宝_(唐朝)|天宝]]（742年-756年）初年事。</ref>礼部侍郎[[韦陟|韦陟]]以为异，录为上第。授[[宋州|宋州]][[单父|单父]]尉。<ref name=BT125>《旧唐书》卷一百二十五</ref><ref name=NBT142>《新唐书》卷一百四十二</ref>加云骑尉。<ref name=LZY/>

== 唐肃宗年间 ==
[[唐肃宗|唐肃宗]][[至德_(唐朝)|至德]]年间（756年-758年），[[江西|江西]]采访使[[皇甫侁|皇甫侁]]闻柳载之名，辟为判官，守永丰令。有能名，被表为[[洪州_(隋朝)|洪州]][[丰城|丰城]]令。<ref>鉴于皇甫侁于至德二载（757年）被罢免此职，柳载为他效力肯定在此之前。见[[:s:資治通鑑/卷219|《资治通鉴》卷二百一十九]]。</ref>累除[[衢州|衢州]]司马，未就职，弃官归隐武宁山。

== 唐代宗年间 ==
[[唐代宗|唐代宗]]年间，又召拜[[监察御史|监察御史]]，认为君命不可逃，装束赴任。监察御史被要求谨慎规矩，而柳载性不拘束，为上司不喜，因而不乐，寻求他任。宰相们认识到他的才能，奏留他为[[右补阙|右补阙]]。<ref name=NBT142/><ref name=CFYG>《册府元龟》</ref><ref>此从《行状》。《两唐书》本传及《册府元龟》作[[左补阙|左补阙]]。</ref>次年，除殿中侍御史，赐绯鱼袋，知江西租庸院事。<ref name=LZY/><ref name=BT125/>

迁[[侍御史|侍御史]]。唐代宗[[大历|大历]]年间（766年-779年），江西监察使[[魏少游|魏少游]]奏柳载充江南西路都团练判官。累授检校司封郎中。柳载效力魏少游期间，有开元寺僧人和徒弟夜饮，醉倒后寺庙起火。他们归咎于守门的聋仆，贿赂魏少游派去调查的军候，使后者上状如是称。魏少游信以为真，准备处罚那个仆人，很多人知道仆人无辜却不敢言。柳载和同僚[[崔祐甫|崔祐甫]]却入内报告给魏少游。魏少游惊讶，讯问僧人，僧人招供。魏少游感谢他们：“若不是二位君子，老夫就要成暗劣之人了。”从此柳载以公正闻名。后[[路嗣恭|路嗣恭]]接替魏少游，以柳载为都团练副使。改祠部员外郎，转司勋郎中，大历十二年（777年），拜[[袁州|袁州]]刺史。<ref name=LZY/><ref name=BT125/><ref name=NBT142/>

== 唐德宗年间 ==
十四年（779年），唐代宗崩，子唐德宗继位。崔祐甫拜相，[[建中_(唐朝)|建中]]初年荐柳载为[[谏议大夫|谏议大夫]]。吏部侍郎[[邵说|邵说]]以才干闻名，有人以为他能做宰相，金吾将军裴儆对柳载说邵说原为叛将[[史思明|史思明]]、[[史朝义|史朝义]]父子旧臣，掌兵权，前后掳掠名家子女数十人以为婢仆，还抢夺宝货，力屈了才投降，朝廷宥以不死，做官了又厚颜无耻，遑遑求财，装饰府邸，结托权贵以求大用，不知愧惧而有得色，不能长久，故揣测他祸不远了。[[建中_(唐朝)|建中]]元年（780年）二月，柳载又因崔祐甫推荐出任浙江东西黜陟使，奏润州刺史[[马炫|马炫]]（《[[册府元龟|册府元龟]]》或误作“冯炫”）清白，使其得征拜为太子右庶子、迁[[左散骑常侍|左散骑常侍]]。<ref name=CFYG/><ref>[[:s:旧唐书/卷134|《旧唐书》卷一百三十四]]</ref>加[[朝散大夫|朝散大夫]]，又拜左庶子、集贤殿学士。后召回[[长安|长安]]迁尚书左丞。<ref name=BT125/>建中三年（782年）五月，邵说果然被贬死。<ref>[[:s:旧唐书/卷137|《旧唐书》卷一百三十七]]</ref><ref>[[:s:新唐书/卷203|《新唐书》卷二百零三]]</ref>六月，柳载以右庶子为尚书右丞。<ref name=NBT142/><ref name=BT12>[[:s:旧唐书/卷12|《旧唐书》卷一十二]]</ref>加[[银青光禄大夫|银青光禄大夫]]，迁[[右散骑常侍|右散骑常侍]]。<ref name=LZY/>

四年（783年）正月，宰相[[关播|关播]]任用[[李元平_(唐朝)|李元平]]守[[汝州|汝州]]抵御叛将[[淮西节度使|淮西节度使]][[李希烈|李希烈]]，柳载说：“这是衔着玉卖石头啊。去了一定被擒，能抵御什么贼呢？”并举[[王衍_(西晋)|王衍]]、[[殷浩_(东晋)|殷浩]]华而不实的例子，时人不信，等李元平果然被李希烈生擒，众人才叹服。<ref name=NBT142/>十月，在长安待命的[[泾原|泾原]]兵兵变，德宗被迫逃往[[奉天|奉天]]。柳载全家逃到终南山躲藏，泾原兵推[[太尉|太尉]][[朱泚|朱泚]]为首，朱泚很快建立秦政权，自称皇帝。他素闻柳载之名，为离间德宗君臣，下诏以柳载[[同平章事|同平章事]]、[[崔寧_(唐朝)|崔宁]]为[[中书令|中书令]]，都拜为宰相，并查访到柳载躲藏处，要配给他宰相的印信。柳载改名换姓欺骗叛军，还将家人交给叛军为人质。叛军将其爱子带回拷问，右腿都被打折。柳载乘机从小道翻越秦岭，历时十多天，找到了德宗的队伍。德宗以为崔宁和朱泚串通，处死了崔宁，<ref>[[:s:旧唐书/卷117|《旧唐书》卷一百一十七]]</ref><ref>[[:s:新唐书/卷144|《新唐书》卷一百四十四]]</ref>但没有处置柳载。<ref>[[:s:資治通鑑/卷228|《资治通鉴》卷二百二十八]]</ref>后德宗被迫远遁[[梁州|梁州]]，柳载随驾，被任为左散骑常侍。[[兴元|兴元]]元年（784年），朱泚政权被灭，德宗重返长安，柳载称自己的名字上了朱泚的诏书，引以为耻，“载”字又有“戈”旁，而国家正需要偃武，请求改名为“浑”，<ref>[[:s:请改名奏|:s:请改名奏]]</ref>并改表字，获准。<ref name=LZY/><ref name=BT125/>

[[贞元_(唐朝)|贞元]]元年（785年）七月，朝廷论功，柳浑获赐轻车都尉，拜[[兵部侍郎|兵部侍郎]]，<ref>此从《旧唐书·德宗纪》。《旧唐书》本传作二年（786年）事。</ref>封宜城县伯。三年（787年）正月，因权相[[韩滉|韩滉]]推荐，柳浑被授[[同中书门下平章事|同中书门下平章事]]，拜为宰相，<ref name=BT12/><ref name=NBT7>[[:s:新唐书/卷7|《新唐书》卷七]]</ref>判[[门下省|门下省]]。德宗曾亲自选县令们主管京畿城邑，有政绩，召宰相说此事，宰相们如[[左仆射|左仆射]][[张延赏|张延赏]]等都贺德宗得人，唯独柳浑不贺，说：“这是[[京兆尹|京兆尹]]的职责。陛下应该选择臣等辅政，臣应当选京兆尹，京兆尹应当寻求县令主管琐事。代京兆尹选择县令不是陛下所宜做的。陛下舍弃此道，使地方大治，可谓爱人，但不是王政的大伦，臣不知道有何可贺。”德宗以为然。德宗命玉工做腰带，玉工摔坏了一个銙，不敢上报，秘密买了其他玉补上；献上后，德宗指出买来的銙材料不像，玉工伏罪，德宗命处决，诏书到中书省，柳浑说：“陛下如果当时便杀了就罢了，若下达有司，就需要议罪量刑。且才春季就行刑，容臣条奏定罪。”以误伤乘舆器服之罪，杖当事玉工六十，其余玉工释放，诏从之。柳浑又奏：“故尚书左丞[[田季羔|田季羔]]，公忠正直，先朝名臣。其祖、父都以孝行得以旌表门闾，京城的隋朝旧第只有季羔一家而已。如今他的堂侄田伯强要卖了房子募兵讨[[吐蕃|吐蕃]]。这个例子一旦开了，恐怕滋长为非作歹之风。讨贼自有国家大计，岂能便宜侥幸之徒？且毁弃义门，有损风化，希望稍作责罚，也可惩劝。”<ref>[[:s:请禁田季羔货宅奏|:s:请禁田季羔货宅奏]]</ref>德宗准奏。时韩滉专权，其他宰相形同虚设，柳浑虽然是韩滉所荐，但厌恶韩滉专政且认为他过苛，以前宰相[[齐映|齐映]]因狷察拜相不到一年就被罢为戒，严斥他杖责下层官吏致死的行为，韩滉惭愧，自此收敛所为。韩滉与[[光禄卿|光禄卿]][[裴腆|裴腆]]不和，趁柳浑休假，秘密请贬裴腆。诏书已下，柳浑坚执不肯，请求讯问、校验，依法定罪量刑，裴腆最终获赦仍任本职。二月，柳浑反对德宗提拔原近臣[[果州|果州]]刺史[[白志贞|白志贞]]为[[浙西观察使|浙西观察使]]，认为白志贞是个谄媚之徒，即使有廉洁谨慎之名，也不宜提拔为要职，屡次上表使此事中止。<ref name=LZY/>但当柳浑病假时，德宗还是提拔了白志贞。柳浑康复后请辞，被德宗优诏拒绝。<ref name=ZZTJ232>[[:s:資治通鑑/卷232|《资治通鉴》卷二百三十二]]</ref>其判门下省期间，主管官吏说应该审定吏、兵部拟任命的六品以下官员，柳浑不悦地说：“各部门分管事务，又去改变他们的决定，这不合礼制法律。官员们离家千里只为这点俸禄，主持一个小地方的事务，难道担心办不好吗？况且奖赏善政贤才，不靠这个。”因此这年审定任命的官员没有被更改。<ref name=BT125/><ref name=NBT142/>

柳浑曾与张延赏奉诏撰《[[昭德皇后_(唐朝)|昭德皇后]]庙乐章》，但德宗认为词句不美，留中不用，而令学士[[吴通玄|吴通玄]]另外撰写献上。<ref>[[:s:旧唐书/卷52|《旧唐书》卷五十二]]</ref><ref>[[:s:旧唐书/卷190下|《旧唐书》卷一百九十下]]</ref><ref>[[:s:新唐书/卷77|《新唐书》卷七十七]]</ref>

闰五月，德宗不顾大将太尉[[李晟_(唐朝)|李晟]]反对，和吐蕃宰相[[尚结赞|尚结赞]]结盟。当日，另一大将[[浑瑊|浑瑊]]作为德宗使者，准备在[[平凉川|平凉川]]正式和尚结赞签署和约。德宗在便殿对宰相们说这是社稷之福，柳浑却直言自己为吐蕃不可信而担忧，李晟也同意。德宗为他们持续反对和议而愤怒，斥责了他们：“柳浑是书生，不懂边事；以大臣（指李晟）的智慧谋略，怎么也说出这样的话！”他俩都顿首俯伏，当即被下令回到中书省，柳浑所奏被留中。但当夜，[[邠宁节度使|邠宁节度使]][[韩游瑰|韩游瑰]]关于吐蕃伏击浑瑊且几乎俘获他的报告就到了。德宗大惊柳浑预言到了此事，遣使将报告给柳浑。次日，他召柳浑于前殿，对柳浑说：“你只是个书生，竟然能如此精准地料及万里之外的此事。”对他尤其礼敬。柳浑先前曾和流放江南的[[李泌|李泌]]及[[顾况|顾况]]交好、吟咏，柳浑拜相后，征顾况为校书郎，时任[[陕州|陕州]][[长史|长史]]的李泌也得以回朝为宰相。<ref>[[:s:旧唐书/卷130|《旧唐书》卷一百三十]]</ref>六月，李泌与李晟、[[司徒|司徒]][[马燧|马燧]]、柳浑一同入见德宗，说服德宗当面承诺善待身为功臣的李晟、马燧。<ref name=ZZTJ232/>同时，因为柳浑敢直言，和德宗信任的张延赏关系变差。德宗以张延赏管吏事，而以柳浑管刑法。<ref>[[:s:新唐书/卷127|《新唐书》卷一百二十七]]</ref>张延赏弄权自矜，嫉恨柳浑守正。一次，张延赏派亲信对柳浑说：“相公您因为品德而受到很大的尊敬，如果在朝堂上慎言，能长保相位。”柳浑答：“代我谢谢张公，柳浑的头可断，但舌头不可禁！”于是被张延赏排挤。德宗喜欢说话文雅含蓄的人，而柳浑直率且常用俚语，因而不久即冒犯了德宗。德宗想贬柳浑为亲王的师傅，但在李泌劝说下只是于八月贬他再任右散骑常侍。<ref name=LZY/><ref name=BT125/><ref name=NBT142/><ref name=BT12/><ref name=NBT7/><ref>[[:s:旧唐书/卷129|《旧唐书》卷一百二十九]]</ref><ref name=ZZTJ233>[[:s:資治通鑑/卷233|《资治通鉴》卷二百三十三]]</ref><ref>《旧唐书》本传及《资治通鉴》作罢为左散骑常侍。</ref>

柳浑善作文，但追随时风以求功业，作文不及胞兄[[柳识|柳识]]深思。又善辩、幽默、旷达，与他人交往时，不像高官通常表现的那样骄傲。罢相后数日，他就和老友们旅游，完全没有表现出为罢相而沮丧的样子。前宰相[[李勉|李勉]]、[[卢翰_(唐朝)|卢翰]]评价道：“我们和柳宜城相比，都是过于拘束的人。”贞元五年二月五日（789年3月6日），柳浑卒于昌化里，官终银青光禄大夫、右散骑常侍、[[轻车都尉|轻车都尉]]，年七十四。<ref name=NBT142/><ref>《两唐书》本传作卒年七十五，此从《行状》。</ref>有《柳浑集》十卷。<ref name=BT125/><ref>[[:s:新唐书/卷60|《新唐书》卷六十]]</ref>节俭不事产业，俸禄都分给宗族、姻亲，身后没有地产。<ref name=LZY/>

[[唐宣宗|唐宣宗]][[大中_(唐朝)|大中]]初年，柳浑续图[[凌烟阁|凌烟阁]]。<ref>[[:s:新唐书/卷191|《新唐书》卷一百九十一]]</ref>

== 评价 ==
*《[[旧唐书|旧唐书]]》：
**史臣曰：[[张镒|张镒]]、[[萧复|萧复]]、'''柳浑'''，节行才能訏谟亮直，皆足相明主，平泰阶，而[[卢杞|卢杞]]忌之于前，延赏排之于后，[[管仲|管仲]]有言：「任君子，使小人间之，害霸也。」德宗黜贤相，位奸臣，致朱泚、[[李怀光|怀光]]之乱，是失其人也，岂尤其时哉！
**赞曰：得人则兴，失人则亡。镒、复、'''浑'''去，宗社其殃。
