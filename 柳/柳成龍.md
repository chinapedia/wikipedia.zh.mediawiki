'''柳成龍'''，生於1542年11月7日（嘉靖二十一年十月初一），卒於1607年6月7日（萬曆三十五年五月十三日），[[字|字]]'''而見'''，[[號|號]]'''西厓'''，安东丰山（今[[韩国|韩国]][[庆尚北道|庆尚北道]]安东市丰山邑）人，是[[朝鮮王朝|朝鮮王朝]]時期的政治家。他是[[柳仲郢|柳仲郢]]的次子，曾於[[朝鮮宣祖|朝鮮宣祖]]時期擔任[[領議政|領議政]]一職。[[封號|封號]]'''豐原府院君'''，死後[[諡號|諡號]]'''文忠'''。

== 早年仕途 ==
少年时，曾向著名[[儒學|儒學]]學者退溪先生[[李滉|李滉]]學習儒學，获得李滉的高度赞赏。1557年柳成龙中乡试，1563年中生员进士试，1564年中生员会试，1566年及第，選補承文院權知副正字。1573年父亲去世而[[丁忧|丁忧]]，1575年丧期满后两次拒绝出仕，次年才接受[[弘文馆|弘文馆]]校理的任命。1589年，郑汝立谋反案爆发，[[西人党|西人党]]借机大加株连，柳成龙作为礼曹判书上疏解救。不久后调任吏曹判书，选派“沉静谨慎”的吴亿龄调查郑汝立谋反同党，挫败了[[郑澈|郑澈]]等西人的阴谋。<ref>{{cite book|author1=佚名|title=西厓集|pages=《西厓先生年谱》}}</ref>

== 战中表现 ==
柳成龙於1590年擔任[[右議政|右議政]]一職，執掌朝政。他在政治派別上屬於[[東人黨|東人黨]]。在东人党成功击败[[西人党|西人党]]以后，他主张采取稳健的态度，与东人党内激进派意见不合，不久后东人党分裂为[[南人党|南人党]]和[[北人党|北人党]]，柳成龙成为了南人党的领袖。1591年，柳成龍提拔少年時的老相識[[李舜臣_(朝鮮)|李舜臣]]為[[全羅左道|全羅左道]]水軍節度使，這引起了与北人党相善的[[元均_(朝鲜)|元均]]的不滿。同时他还主张提拔了[[权慄|权慄]]，后者成为了战争期间的朝鲜军最高统帅——都元帅。[[豐臣秀吉|豐臣秀吉]][[萬曆朝鮮戰爭|侵略朝鮮]]之際，柳成龍歷任[[領議政|領議政]]、[[都體察使|都體察使]]（主管军事）等要職，主持朝鮮王朝的政治、军事，并负责来援明军的后勤补给。柳成龙是坚决的主战派，在[[碧蹄馆之战|碧蹄馆之战]]后依旧坚持催促明军进兵，但是最终明军没有继续前进。停战期间，元均勾結反對南人黨的大臣，污蔑李舜臣，最终使得李舜臣被罢免。柳成龙争辩无效后上疏请辞，不被允许。

== 被谗去职 ==
1598年，[[丁应泰|丁应泰]]诬告朝鲜同日本同谋攻击明朝。[[朝鲜宣祖|宣祖]]有意要柳成龙赴北京辩解，柳成龙以被[[北人党|北人党]]攻击而拒绝。北人党借机大加攻讦，柳成龙被褫职，虽然两年后又复职，但是他已经无意于政治，不久后辞官回乡。1604年被封为二等扈圣功臣（用以表彰在万历朝鲜战争期间随同[[李昖|李昖]]逃亡的大臣），1607年去世。

== 评价 ==

明军经理[[杨镐|杨镐]]：汝國事，當使如柳某者輔之。<ref>{{cite book|author1=佚名|title=西厓集|pages=《西厓先生年谱》}}</ref>

[[朝鮮宣祖|朝鮮宣祖]]：將抱冤而死，地下相見，豈安於心？<ref>《宣祖實錄》宣祖30年1月6日</ref>

《宣祖修正實錄》：成龍素負重望, 當國累年, 大爲群小所惎。 初以使行之不自請爲罪案, 而乘機搆捏, 以無根之說、不近之謗, 肆意羅織, 終至於慈母投杼<ref>此處用典出自《戰國策》，指在眾多人的誹謗下，親近的人也開始被懷疑。</ref>, 痛矣!

== 其他 ==
[[File:Letter_of_Ryu_Sung-ryong.PNG|thumb]]
柳成龍為官清廉，經常出資資助百姓。他被罢职的时候，因为没有路上吃的粮食，甚至要写信让家人运粮来，“子弟皆徒步，顚沛萬狀”。<ref>{{cite book|author1=佚名|title=西厓集|pages=《西厓先生年谱》}}</ref>

他在战争期间建议使用[[戚继光|戚继光]]的练兵法练习军队（他倡议组建的“三手军”成为了朝鲜军队后来的主力），利用朝鲜多山的地形建造山城以作为防御，并请求恢复镇管制（朝鮮初期的一種軍事體制）以维护军队的战斗力<ref>{{cite book|author1=柳馨远|title=磻溪随录}}</ref>，可见他虽然不是军人，却对军事有着自己的见解。在战争期间提拔的李舜臣、权慄都成为了大将并被[[朝鲜王朝|朝鲜王朝]]追封为一等宣武功臣（用以表彰万历朝鲜战争期间的将领）。

其著作《[[懲毖錄|懲毖錄]]》，是柳成龍晚年對[[萬曆朝鮮戰爭|萬曆朝鮮戰爭]]期間執政的反省，其书名来自《[[诗经|诗经]]·周颂·小毖》的“予其惩，而毖后患”，意思是“我必须深刻吸取教训，用以免除后患”。其书具有較高的史料價值，但是也有人认为柳成龙在该书中掩饰自己的过失，攻讦别人，特别是明军将领。<ref>《宣祖修正實錄》宣祖40年6月</ref>（这可能与双方在战和问题上的态度不同有关）其原稿被[[韩国|韩国]]指定为第132号国宝。

他还编纂了自己老师的文集《退溪集》、[[王氏高丽|王氏高丽]]末期儒学者[[郑梦周|郑梦周]]的文集《圃隐集》等。

柳成龍的後代有不少名人。韓國藝人[[柳時元|柳時元]]是其後裔。日本藝人[[柳熺俊|柳熺俊]]是其17世孫。圍棋棋手[[柳時熏|柳時熏]]也是他的後裔。
