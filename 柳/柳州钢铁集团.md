'''广西柳州钢铁集团有限公司'''始建于1958年，2007年2月，子公司「柳钢股份」在[[上海证券交易所|上海证券交易所]]上市。2005-2015年[[武漢鋼鐵集團|武漢鋼鐵集團]]與[[广西壮族自治区人民政府国有资产监督管理委员会|广西国资委]]透過合伙公司[[廣西鋼鐵集團|廣西鋼鐵集團]]以80-20比控股柳州钢铁集团。2015年9月广西国资委退出廣西鋼鐵集團，柳州钢铁集团重新成為广西国资委直屬企業。<ref>[http://www.sse.com.cn/disclosure/listedinfo/announcement/c/2015-09-09/601003_20150909_1.pdf 柳州钢铁股份有限公司重大事项公告]</ref>

== 生产能力 ==
具备铁975万吨、钢1100万吨、钢材1400万吨以上的年综合生产能力。热轧带肋钢筋（螺纹钢）为国家免检产品，生产的船用钢板和钢带获得了CCS（中国）、GL（德国）、DNV（挪威）、BV（法国）、LR（英国）、NK（日本）、ABS（美国）、KR（韩国）等八国船级社的工厂认可。并先后投入20亿元治理污染。<ref>广西柳州钢铁公司 {{cite web |url=http://www.liuzhousteel.com/index.asp |title=存档副本 |accessdate=2011年6月5日 |deadurl=yes |archiveurl=https://web.archive.org/web/20110428121510/http://www.liuzhousteel.com/index.asp |archivedate=2011年4月28日 |df= }}</ref>

== 技术中心 ==
建设有[[柳钢技术中心|柳钢技术中心]]，提供的仪器包括：电感耦合等离子体发射光谱仪、四球摩擦机、自动闪点仪等设备具有国际先进水平；电脑量热仪、运动粘度测定仪、60t万能材料试验机、分光光度计、原子吸收分光光度计及红外碳硫仪、水中油份浓度分析仪、微电脑烟尘平行采样仪、高温物性测试仪、微机控制全自动压力试验机、全自动高温抗折试验机、平板法导热系数测定仪、高温热膨胀仪。<ref>广西柳州钢铁公司 {{cite web |url=http://www.liuzhousteel.com/Channel.asp?id=40 |title=存档副本 |accessdate=2011年6月5日 |deadurl=yes |archiveurl=https://web.archive.org/web/20110428121826/http://www.liuzhousteel.com/Channel.asp?id=40 |archivedate=2011年4月28日 |df= }}</ref>

开展的测试及试验项目：钢铁、铁合金、铁矿化学分析；钢铁产品力学性能、金相检验及分析；耐火材料及相关非金属的检验试验及分析研究；煤、焦、油分析检验；焦碳热反应性及反应后强度试验；工业废水、废气、粉尘、厂界噪音等监测分析；钢材、金属备件的超声波探伤以及耐热、耐磨铸钢件的生产试验。<ref>广西柳州钢铁公司 {{cite web |url=http://www.liuzhousteel.com/Channel.asp?id=40 |title=存档副本 |accessdate=2011年6月5日 |deadurl=yes |archiveurl=https://web.archive.org/web/20110428121826/http://www.liuzhousteel.com/Channel.asp?id=40 |archivedate=2011年4月28日 |df= }}</ref>
