'''面積二次轴矩'''（'''second axial moment of area'''），又称'''面積慣性矩'''，或面積对某一轴的'''惯性矩'''，通常是对受弯曲作用物体的横截面而言，是反映截面的形状与尺寸对弯曲变形影响的物理量。弯曲作用下的变形或[[挠度|挠度]]不仅取决于荷载的大小，还与横截面的几何特性有关。如[[工字梁|工字梁]]的抗弯性能就比相同截面尺寸的矩形梁好。它和反映截面抗扭转作用性能的[[面積极惯性矩|面積极惯性矩]]是相似的。

面積二次轴矩虽然也称“惯性矩”，但它和用以计算旋转物体[[角加速度|角加速度]]的[[質量惯性矩|質量惯性矩]]（常称为转动惯量）是不同的两个概念。二者有相同的符号<math>I</math>（<math>I</math>是英文中惯性 inertia 的首字母），但依据上下文二者不致混淆。而且二者的[[因次|因次]]或[[单位_(度量衡)|单位]]不同：面積二次轴矩的单位是长度的四次方，而后者的单位是长度的二次方乘以质量。

==定义==
截面的面积为''A''，则
: <math>I_x = \int y^2\,\mathrm dA</math>
: <math>I_y = \int x^2\,\mathrm dA</math>
分别表示截面对坐标轴''x''与''y''的惯性矩，第一式中的''y''和第二式中的''x''分别表示面积微元d''A''到''x''和到''y''轴的垂直距离。

在[[国际单位制|国际单位制]]（SI）中，截面二次轴矩的单位是m<sup>4</sup>，常用mm<sup>4</sup>表示。

==坐标变换==
计算截面惯性矩时常根据截面形状采用方便计算的[[坐标系|坐标系]]，然后可以通过坐标变换应用到其他坐标系中。
===平行轴定理===
{{main|平行轴定理}}
在已知对过截面[[形心|形心]]轴的惯性矩和轴间距离的情况下，[[平行轴定理|平行轴定理]]可以确定对变换后新轴的惯性矩。
: <math>I_x = I_{xCG}+A d^2\,</math>
*''I<sub>x</sub>'' ：对''x''轴的惯性矩
*''I<sub>xCG</sub>'' ：对与''x''轴平行并且过截面形心的轴（与[[中性轴|中性轴]]重合）的惯性矩
*''A'' ：截面面积 
*''d'' ：两轴之间的距离

===转轴公式===
下列公式可以计算坐标轴旋转一个角度后截面对新坐标轴的惯性矩
:<math>{I_x}^* = \frac{I_{x} + I_{y}}{2} + \frac{I_{x} - I_{y}}{2} \cos(2 \phi) -  I_{xy} \sin(2 \phi)</math>

:<math>{I_y}^* = \frac{I_{x} + I_{y}}{2} - \frac{I_{x} - I_{y}}{2} \cos(2 \phi) +  I_{xy} \sin(2 \phi)</math>

*<math>\phi</math> ：旋转的角度（逆时针）
:<math> x^* = x \cos \phi + y \sin \phi</math>
:<math> y^* = -x \sin \phi + y \cos \phi</math>
*''I<sub>x</sub>'' 和 ''I<sub>y</sub>'' ：原坐标系下的惯性矩
*''I<sub>x</sub><sup>*</sup>'' 和 ''I<sub>y</sub><sup>*</sup>'' ：坐标系转动后新坐标系下的惯性矩


==简单截面的惯性矩==
以下是几种简单截面对"截面形心"所在轴的惯性矩
===矩形截面===
:<math>I_{x}=\frac{bh^3}{12}</math>
*''b'' ：宽度（''x''方向）
*''h'' ：高度（''y''方向）

:<math>I_{y}=\frac{hb^3}{12}</math>
*''b'' ：宽度（''x''方向）
*''h'' ：高度（''y''方向）

===圆形截面===
:<math>I_x = \frac{\pi}{64} D^4 = \frac{\pi}{4} r ^4</math>
:<math>I_o = 2I_x = \frac{\pi}{2} r ^4</math>
*''D'' ：直径
*''r'' ：半径

===三角形截面===
以底边方向为''x''方向
:<math>I_{x}=\frac{bh^3}{36}</math>
*''b'' ：底边宽度（''x''方向）
*''h'' ：高（''y''方向）

==梁的弯曲正应力==
以中性轴为原点，单向受弯[[梁（结构）|梁]]横截面上''y''处的正应力为
:<math>{\sigma}= \frac{My}{I_x}</math>
*''M'' ：作用在梁上的[[弯矩|弯矩]]
*''y'' ：到通过形心的''x''轴的距离
*''I<sub>x</sub>'' ：对通过形心的''x''轴的惯性矩

由该式可见截面的惯性矩越大，弯曲正应力越小，抗弯性能越好。

==与[[极惯性矩|极惯性矩（截面二次极矩）]]的关系==
由于<math>\rho^2=y^2+z^2</math>，极惯性矩<math>I_{P}=\int_{A} \rho^2 dA</math>
根据截面二次轴矩的定义，可知：

<math>I_{P}=I_{y}+I_{z}</math>

即截面对于任何一点的极惯性矩，等于该截面对以该点为原点的任意一组正交坐标系的截面二次轴矩之和
