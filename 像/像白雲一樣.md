《'''像白雲一樣'''》<ref>{{cite web|title=藤井郁彌/天空的模樣|url=http://www.books.com.tw/exep/cdfile.php?item=0020077144|publisher=博客來|accessdate=2014-06-19|deadurl=yes|archiveurl=https://web.archive.org/web/20160304141827/http://www.books.com.tw/exep/cdfile.php?item=0020077144|archivedate=2016-03-04}}</ref>（{{jp|j=白い雲のように|hg=しろいくものように|rm=Shiroi Kumo no Yōni|lead=yes}}），又譯《'''一如白雲'''》<ref>{{cite web|title=myMusic - 音樂 - 歌詞 - 一如白雲 (猿岩石)|url=http://www.mymusic.net.tw/song/show?song_id=p000020-a0072908-s000747-t005-c6&cmd=ezpc|publisher=台灣酷樂}}</ref>，是[[日本|日本]][[搞笑藝人|搞笑組合]][[猿岩石|猿岩石]]的出道[[單曲|單曲]]，於1996年12月21日透過[[日本古倫美亞|日本古倫美亞]]發行。單曲在發售首週僅名列[[Oricon公信榜|Oricon週榜]]第23名，不過隨後銷量繼續上升，最終共計賣出114萬張，成為{{link-ja|Million seller|ミリオンセラー|百萬歌曲}}<ref name="oriconcd">{{cite web|url=http://www.oricon.co.jp/news/50991/|title=猿岩石 シングル・アルバム売上一覧表|publisher=[[Oricon|Oricon]]|accessdate=2014-05-30}}</ref>。

== 簡介 ==
[[日本電視網協議會|日本電視台系]]《{{link-ja|前進！電波少年|進め!電波少年}}》的歐亞大陸橫斷企劃終結後，猿岩石所屬的事務所以及《電波少年》的[[製作人|製作人]]{{link-ja|土屋敏男|土屋敏男}}認為兩人的歌喉「比想像中好」，於是決定為兩人製作歌曲，進行歌手[[出道_(演藝用語)|出道]]<ref>{{cite web|title=猿岩石の由来|url=http://moto-neta.com/stage-name/saruganseki/|publisher=タネタン|accessdate=2014-06-10}}</ref>。本作在發行當初並未受到大眾的關注，在開售首週只取得[[Oricon公信榜|Oricon公信榜]]週榜第23名，銷量約2萬張。但是隨著兩人的知名度開始浸透到大眾文化中，單曲在第四週登上週榜頭十名，並上升至第三名<ref name=spon/>。此後單曲繼續保持在榜上，銷量之高為搞笑藝人發行的CD中較為罕見的情況。最終，單曲在[[Oricon公信榜|Oricon公信榜]]的總銷量超越一百萬，獲認證為{{link-ja|Million seller|ミリオンセラー|百萬歌曲}}<ref name="oriconcd"/>。

在製作過程中，原本播放《電波少年》的[[日本電視台|日本電視台]]打算找回同屬母公司[[日本電視控股|日本電視控股]]旗下的唱片公司[[VAP|VAP]]發售本作品<ref>{{cite web|title=猿岩石のある暮らし - 資料集 - 商品 - AIRnet|url=http://www3.airnet.ne.jp/saru/data/goods.html|accessdate=2014-06-10|date=2005-06-11}}</ref>，但猿岩石的所屬藝能事務所[[太田製作|太田製作]]最終選擇了[[日本古倫美亞|日本古倫美亞]]，結果引起日本電視台不滿，不再在節目中起用太田製作的藝人，日本電視台內隸屬太田事務所的藝人{{link-ja|松村邦洋|松村邦洋}}也被降職<ref>{{cite book|author=宝泉薫|title=芸能界「一発屋」外伝|publisher=彩流社|isbn=4-88202-609-0|pages=48-49|chapter=永遠のヒッチハイク人生／純正アイドル・猿岩石|date=1999-12-01}}</ref>。

本曲後來被A面曲《像白雲一樣》之作詞者{{link-ja|藤井郁彌|藤井フミヤ}}和作曲者{{link-ja|藤井尚之|藤井尚之}}所組成的兄弟組合{{link-ja|F-BLOOD|F-BLOOD}}進行翻唱<ref>{{cite web|title=『F-BLOOD』F-BLOOD｜シングル、アルバムのダウンロード（配信）、歌詞、音楽ランキング【レコチョク】33425248|url=http://recochoku.jp/album/A33425248/|publisher=[[RecoChoku|RecoChoku]]|accessdate=2014-06-10}}</ref>，另外在猿岩石的第8張單曲《擁抱昨天之前的你》（{{lang|ja|昨日までの君を抱きしめて}}）中也以「'98 Piano Strings Version」名義收錄了本單曲B面曲《為甚麼我會踏上路途呢》的[[鋼琴|鋼琴]][[弦樂|弦樂]]演奏版本。

== 評價 ==
猿岩石在歌曲推出數年後就宣布解散，有音樂評論認為歌曲[[副歌|副歌]]部分的歌詞內容「被風吹拂消失了吧 我的腳印 被風吹著而去了啊 像白雲一樣」（{{lang|ja|風に吹かれて消えてゆくのさ 僕らの足跡 風に吹かれて歩いてゆくのさ 白い雲のように}}）將猿岩石的未來準確預測了出來<ref name=revkako>{{cite web|title=猿岩石　シングルレビュー|url=http://moma-bd.com/kako.saruganseki.htm|accessdate=2014-06-19}}</ref>。組合於2004年因兩人事業方向出現分歧而解散<ref>{{cite web|title=猿岩石が3月でコンビを解散。|url=http://www.narinari.com/Nd/2004032069.html|publisher=Narinari.com|accessdate=2014-06-04|date=2004-03-18}}</ref>，因此實際上也如同[[白雲|白雲]]般「被風吹拂消失」。另外，單曲附送的歌詞本清楚分列出兩人在曲中演唱的部分，有吉和森脇的[[獨唱|獨唱]]時間大致獲得平均分配，但是評論認為在副歌[[合唱|合唱]]的部分則明顯是森脇的聲音較為響亮，相較之下有吉只有總聲量的兩成左右，即使是在電視節目上現場演唱時主[[旋律|旋律]]也只聽到森脇的聲音。不過即使如此，歌曲仍獲得四星的評級（五星為滿分）。<ref name=revkako/>
