'''溫偉耀'''博士（{{lang|en|Dr. Milton Wan Wai-yiu}}，{{bd|1952年||||catIdx=W}}），[[牛津大學|牛津大學]]及[[香港中文大學|香港中文大學]]博士，基督教學者，靈修神學和漢語神學專家，曾任[[香港中文大學崇基學院神學院|香港中文大學崇基學院神學院]]副教授、[[香港中文大學|香港中文大學]]文化及宗教研究系副教授，及[[漢語基督教文化研究所|漢語基督教文化研究所]]神學教授。由於曾經歷喪妻、喪父、喪幼女的悲慘遭遇，他亦被稱為「當代約伯」<ref>{{ cite web |url=http://www.gospelherald.com.hk/news/min-3060/ |title=專訪：「當代約伯」溫偉耀靠上帝走過人生黑暗 |date=2013-07-26 |publisher=基督日報 |author= }}</ref>。

==簡歷==

溫偉耀1952年生於香港，中學就讀[[香港培正中學|香港培正中學]]<ref>{{ cite web |url=http://www.puiching.org/club-level-img/1969-%E5%B9%B4%E7%B4%9A%E6%98%87%E7%A4%BE/ |title=1969 年級昇社 |date= |publisher=培正同學會 |author= }}</ref>，1970年入讀[[香港中文大學|香港中文大學]]物理系，1973年獲物理學理學士學位。1975年以量子力學研究取得[[香港中文大學|中大]]物理學哲學碩士學位。1978年在[[中國神學研究院|中國神學研究院]]取得基督教研究碩士學位。1978年他參與[[趙天恩|趙天恩]]博士創立的「中國教會研究中心」關心中國大陸的教會，並協助趙博士所創的期刊《中國與教會》。<ref name="wan60">{{ cite book |url=http://www.logos.com.hk/bf/acms/content.asp?site=logosbf&op=show&type=product&code=9789628450367  |title=溫偉耀六十自選集：作者簡介 |date=2012-01-15 |publisher=卓越使團 |author=溫偉耀 |location=香港 | pages=  |ISBN=9789628450367 }}</ref>
<ref name="tbts090109">{{ cite web |url=http://tbts119.blogspot.hk/2009/01/blog-post_1923.html |title=基督教與中國文化講座報導 |date=2009-01-09 |publisher= |author=吳昶興 }}</ref>

1980年舉家到英國[[牛津大學|牛津大學]]留學，研究現代西方哲學及基督教神學，1984年取得博士學位後返回香港。1985年9月在[[香港中文大學|香港中文大學]]攻讀第二個哲學博士的學位，主要是研究北宋二程修養工夫的哲學，師從[[勞思光|勞思光]]，1990年取得哲學博士學位。<ref name="wan60" />

1989年「[[六四事件|六四事件]]」後，溫氏移民加拿大。1998年，溫氏返回香港，擔任[[香港中文大學崇基學院神學院|崇基學院神學院]]「中國文化與基督教」課程的老師，其後他負責「當代基督教教學資源中心」，以暑期密集課程方式為內地學者提供全面的基督教教學訓練，並提供基督宗教研究相關的教材。<ref name="gh050927">{{ cite web |url=http://www.gospelherald.com.hk/news/min-691/ |title=溫偉耀博士：今日我有一個中國夢﹙三﹚ |date=2005-09-27 |publisher=基督日報 |author= }}</ref>

1999年起任教於香港中文大學文化及宗教研究系，並兼任[[漢語基督教文化研究所|漢語基督教文化研究所]]莫慶堯醫生傑出神學教授、加拿大[[多倫多大學|多倫多大學]]亞洲神學中心研究員。溫氏並致力參與中國學術界對基督教的研究及教學，曾被邀擔任[[中國人民大學|中國人民大學]]、[[浙江大學|浙江大學]]、[[武漢大學|武漢大學]]、[[山東大學|山東大學]]、[[四川大學|四川大學]]等七所大學的客座教授。<ref name="gh130726">{{ cite web |url=http://www.gospelherald.com.hk/news/min-3060/ |title=專訪：「當代約伯」溫偉耀靠上帝走過人生黑暗 |date=2013-07-26 |publisher=基督日報|author= }}</ref><ref name="gh050924">{{ cite web |url=http://www.gospelherald.com.hk/news/min-689/ |title=溫偉耀博士：今日我有一個中國夢﹙二﹚ |date=2005-09-24 |publisher=基督日報 |author= }}</ref>

由1984年至2016年，溫氏先後任教於香港[[中國神學研究院|中國神學研究院]]、加拿大安省神學院 (Ontario Theological Seminary，後改稱為[[天道神學院|天道神學院]]  Tyndale University and Seminary)、[[香港中文大學崇基學院神學院|香港中文大學崇基學院神學院]]。溫氏亦同時擔任加拿大麥恆華人長老會 (Markham Chinese Presbyterian Church)、加拿大城北華人基督教會(Richmond Hill Christian Community Church)及香港播道會泉福堂的顧問牧者。<ref>{{ cite web |url=https://cclcbooks.wordpress.com/new_releases/a_spiritual_reading_of_the_song_of_songs/ |title=作者簡介 |date= |publisher=基督教文藝出版社 |author= }}</ref>

2016年5月，溫氏退休，退休後繼續從事靈修神學及漢語神學的研究和寫作。<ref>{{ cite web |url=http://christiantimes.org.hk/Common/Reader/News/ShowNews.jsp?Nid=95307&Pid=2&Version=1508&Cid=583&Charset=big5_hkscs |title=溫偉耀將赴日研究禪學與靈修學  |date=2016-07-21 |publisher=時代論壇 |author= }}</ref>

== 學術研究 ==
=== 靈修神學 ===
1980年至1984年，當溫氏在[[牛津大學|牛津大學]]修讀[[系統神學|系統神學]]期間，他在課餘研讀了大量[[靈修神學|靈修神學]]的經典，包括中世紀天主教神學典籍，開展了他「靈修神學」的探索。1985年返回香港後，溫氏大力推廣靈修神學，在八十年代在香港掀起一股靈修神學的熱潮。1984年至1989年，溫氏經歷了幼女嚴重智障、妻子和父親逝世的不幸事件，他稱這些「面對苦難」經歷令他對靈修神學有再進一步探索。<ref name="inm100612">{{ cite news |url=http://www.inmediahk.net/node/1007340 |title=三問溫偉耀 |date=2010-06-12 |publisher=獨立媒體 |author=何信 }}</ref>

在靈修學方面，溫氏尤其欣賞被稱為「基督的小花」的法國修女[[利雪的德蘭|小德蘭]]（Thérèse of Lisieux）。溫氏形容小德蘭的靈修學以「愛」為中心，對靈性有極深的體會和表達，她的自傳感動人心。<ref>{{ cite web |url=http://christiantimes.org.hk/Common/Reader/News/ShowNews.jsp?Nid=150905&Pid=102&Version=0&Cid=2141&Charset=big5_hkscs |title=深閱靈修生活，溫偉耀︰將思想化為不住的禱告 |date=2017-03-31 |publisher=時代論壇 |author= }}</ref> 

心理輔導和靈修學大師[[盧雲|盧雲]]神父（Henri J.M. Nouwen）是其中一位影響溫氏生命最深刻的人<ref>{{ cite book |url= |title=溫序 |work=心靈愛語：當我陷入靈命低潮的時候 |date=1997 |publisher=卓越使團 |author=溫偉耀 |location=香港 | pages=頁3 |ISBN=9789627409960 }} </ref>。1987年溫氏妻子去世後，他帶著嚴重弱智的女兒從香港到多倫多找盧雲。盧雲說了一段他「終生不會忘記的說話」：「你的女兒是天使，是上天差派她來保護你，保護你不會墮落成爲一個滾逐名與利的人。」1991年，溫氏出版盧雲的《無能者的大能》曾引起風波，因為有基督教的牧者和機構反對他以天主教神父爲師和介紹天主教的靈修學。<ref name=""nouwen">{{ cite book |url= |title=盧雲與我-記一段屬靈的友誼 |work=心靈愛語：當我陷入靈命低潮的時候 |date=1997 |publisher=卓越使團 |author=溫偉耀 |location=香港 | pages=頁185-192 |ISBN=9789627409960 }} </ref>

=== 漢語神學 ===
1999年溫氏從加拿大返回香港，開始了漢語神學的建構，研究基督教文化與中國文化之間的深層問題，嘗試以基督教傳統與當代中國文化作出互動和對話。研究包括：基督教對當代中國社會的積極意義，基督教的『公共性』和『獨一性』，基督教與儒學天道與人性等漢語神學的核心問題等。主要對像為中國學術界。<ref name="life2009">{{ cite book |url=http://www.logos.com.hk/bf/acms/content.asp?site=logosbf&op=show&type=product&code=9787802540996 |title=生命的轉化與超拔：我的基督宗教漢語神學思考 |date=2009 |publisher=宗教文化出版社 |author=溫偉耀 |location=北京 | pages=  |ISBN=9787802540996 }} </ref>

== 個人生活 ==

溫偉耀元配羅雅玲，1972年畢業於[[香港中文大學|香港中文大學]][[新亞書院|新亞書院]]藝術系<ref>{{ cite web |url=http://www.alumni.cuhk.edu.hk/aafaa/namelist.htm |title=歷屆系友名單 |date= |publisher= |author= }}</ref>。1980年，溫氏舉家到英國留學，期間誕下患有嚴重智障的幼女溫曉華。1984年學成回港，發現妻子得了末期癌症，他的父親亦罹患失智症。在1987年不到半年間，他的父親與妻子相繼離世。<ref name="tbts090109" />

1989年，由於經歷喪父喪妻之痛，加上發生天安門[[六四事件|六四事件]]，中國對示威學生開槍，溫氏認為中國沒有希望，決定離開香港，移居加拿大。<ref name="tbts090109"/> 之後在加拿大續弦再娶葉麗芬，但再婚後仍經歷種種困難：長女全然不能接受新母，以為父親遺棄她；葉麗芬亦表示不曉得如何當後母，寧願離開以免爭端。<ref>{{ cite web |url=http://www.hkbibleconference.org/component/content/article/214-1973-2006-sermons/deeper-life-message/2002-74th12/1412-.html |title=軟弱與剛強-寶貝與瓦器 |date=2002-08-01 |publisher=港九培靈研經會 |author=溫偉耀 }}</ref> 1994年，嚴重智障的幼女突然被加拿大政府勒令離境，當溫氏將她帶往出生地英國，英政府因不願意負責高昂的醫護費，亦不願接納她，並懷疑葉麗芬想遺棄繼女。<ref>{{ cite web |url=https://midnightsam.wordpress.com/tag/%E6%BA%AB%E5%81%89%E8%80%80/ |title=溫偉耀：使人在絕望中有盼望的神 |date=2005-09-11 |publisher= |author= }}</ref> 2008年2月11日，患有嚴重智障的幼女溫曉華逝世，享年26歲。<ref>{{ cite web |url=http://christiantimes.org.hk/Common/Reader/News/ShowNews.jsp?Nid=45746&Pid=6&Version=0&Cid=150 |title=再見，曉華 |date=2008-02-28 |publisher=時代論壇 |author=溫偉耀、溫葉麗芬 }}</ref>
