《'''LOVE SO LIFE～溫馨的愛～'''》（{{Lang-ja|LOVE SO LIFE}}）是日本漫畫作家{{Link-ja|河內楓|こうち楓}}所畫的[[少女漫畫|少女漫畫]]作品。

==故事簡介==
因為年幼喪母在孤兒院長大的中村詩春，為了成為托兒所老師，所以在托兒所打工，在那認識了一對可愛的雙胞胎松永葵、松永茜。然而他們的爸爸因為承受不住妻子過世的打擊，把他們託給自己在電視台當主播的弟弟松永政二。不善照顧小孩的政二，被小葵小茜整得團團轉，因此請詩春到家裡擔任褓母。經過相處後，政二開始對詩春產生不同的情感…

==人物==
; {{nihongo|中村 詩春|なかむら しはる||[[聲優|聲優]]：[[能登麻美子|能登麻美子]]}}
:女主角，乐观，温柔，责任心强，喜欢小孩子，梦想成为托儿所老师，因为在孤儿院长大的关系，很擅长照顾别人。
; {{nihongo|松永 政二|まつなが せいじ||聲優：[[小野大輔|小野大輔]]}}
:男主角，电视台人气主播，因某些原因"被逼"照顾哥哥的双胞胎孩子，工作时形象清新，干练，与家里的形象产生严重反差，为人体贴。
; {{nihongo|松永 茜|まつなが あかね||聲優：[[澤城美雪|澤城美雪]]}}
: 雙胞胎中的姐姐<ref>78話提及茜是姐姐，外傳1中翻將-{双}-子のきょうだい(兄弟姐妹)錯翻成兄妹，但根據78話應為姐弟。</ref>，2歲。非常有元氣，有時候無法控制自己，即使跌倒也要繼續向前衝。非常貪吃。

; {{nihongo|松永 葵|まつなが あおい||聲優：[[澤城美雪|澤城美雪]]}}
: 雙胞胎中的弟弟，2歲。雖然看起來像個小大人，但只要一熱衷某個東西，就會對周遭的事物視若無睹。喜歡星星、撒嬌。

; {{nihongo|宮川 健|みやかわ たける||聲優：[[柿原徹也|柿原徹也]]}}

;{{nihongo|吉井 梨生|よしい りお||聲優：[[高垣彩陽|高垣彩陽]]}}
