'''登莱胡同'''，是位于[[北京市|北京市]][[西城区|西城区]]南部的一条胡同。

==简介==
登莱胡同位于[[广安门内大街|广安门内大街]]西端南侧。原先北到[[广安门内大街|广安门内大街]]，南到宣武体育场（今[[广安体育馆|广安体育馆]]）北围墙，折向西至[[南线阁街|南线阁街]]，长518米，平均宽度4米。[[明朝|明朝]]时称为干面胡同，属于白纸坊。清朝时沿用干面胡同的名称。胡同内有[[宝应寺_(北京)|宝应寺]]。[[清朝|清朝]]《[[宸垣识略|宸垣识略]]》记载：“宝应寺在广安门东南，地名南燕角，相传唐刹也。”宝应寺旁边有[[山东|山东]]登莱胶寄骨所（登莱胶是指[[山东省|山东省]]的[[登州|登州]]、[[莱州|莱州]]、[[胶州|胶州]]），迤南是登莱胶义地，又称山左义地（今为广安体育馆所在处）。[[中华人民共和国|中华人民共和国]]成立前，宝应寺及其四周都是义地以及旷地，极少有居民。中华人民共和国成立后，干面胡同及宝应寺附近逐渐形成街巷。1965年北京市整顿地名，将干面胡同更名为“登莱胡同”。宝应寺后来被列为[[宣武区文物保护单位|宣武区文物保护单位]]<ref>{{Cite web |url = http://www.bjdclib.com/subdb/laneculture/lane/200907/t20090722_12037.html |title =登莱胡同  |author =  |date = 2009-07-22 |publisher = 东华流韵 |language =  |accessdate =  |quote =  }}</ref>。
