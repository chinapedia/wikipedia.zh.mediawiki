《'''鲁达圣诞节'''》（{{lang|en|''Ludachristmas''}}）是[[美国|美国]][[情景喜剧|情景喜剧]]《[[我为喜剧狂|我为喜剧狂]]》[[我为喜剧狂_(第2季)|第2季]]的第9集，也是整个剧集的第30集。由塔米·赛格编剧，唐·斯卡迪诺执导，2007年12月13日通过[[全国广播公司|全国广播公司]]在[[美国电视|美国]]首播<ref name="nbcmv">{{cite web |url=http://www.screenrush.co.uk/television/television-813/season-5594/ |title=30 Rock: Episodes Season 2 |publisher=Screenrush (AlloCiné) |accessdate=2013-08-24 |location=London |archiveurl=https://web.archive.org/web/20120730001603/http://www.screenrush.co.uk/television/television-813/season-5594/ |archivedate=2012-07-30 |deadurl=no }}</ref><ref name="nbcepisode">{{cite web |url=http://www.nbc.com/30-rock/episode-guide/season-2/224681/ |title=ALL EPISODE GUIDES : NEWEST |publisher=NBC |accessdate=2013-09-13 |deadurl=no |archivedate=2012-11-10 |archiveurl=https://web.archive.org/web/20121110025309/http://www.nbc.com/30-rock/episode-guide/season-2/224681/ }}</ref>。[[友情客串|客串出演]]本集的演员包括凯文·布朗、凯·加农、格里兹·查普曼、阿尼塔·吉莱特、塞莱娜·冈萨雷斯（{{lang|en|Selena Gonzalez}}）、[[巴可·亨利|巴可·亨利]]、约翰·卢茨、约翰·F·穆尼（{{lang|en|John F Mooney}}）、马里克·潘考利、[[安迪·里查克|安迪·里查克]]和伊莲妮·斯楚奇<ref name="yahoo">{{cite web |archivedate=2011-06-15 |deadurl=yes |archiveurl=https://web.archive.org/web/20110615092147/http://tv.yahoo.com/episode/155258/castcrew%3B_ylt%3DAs06.AsMpOeY2u4mmYi_aPyvo9EF |url=http://tv.yahoo.com/episode/155258/castcrew;_ylt=As06.AsMpOeY2u4mmYi_aPyvo9EF |title=''30 Rock'': "Episode 209" |accessdate=2008-08-18 |publisher=Yahoo }}</ref>。

在这一集中，全国广播公司[[电视节目|电视节目]]《{{lang|en|TGS}}与崔西·乔丹》首席[[编剧|编剧]]丽兹·莱蒙（{{lang|en|Liz Lemon}}，[[蒂娜·菲|蒂娜·菲]]饰）的家人都趁假期来[[纽约|纽约]]看她，副总裁杰克·唐纳吉（{{lang|en|Jack Donaghy}}，[[亚历克·鲍德温|亚历克·鲍德温]]饰）的[[母亲|母亲]]科琳·唐纳吉也在[[圣诞节|圣诞节]]期间来看儿子。节目的一众演员和编剧准备参加一年一度的“鲁达圣诞节”聚会。男演员崔西·乔丹（{{lang|en|Tracy Jordan}}，[[崔西·摩根|崔西·摩根]]饰）被法院命令必须佩戴一个酒精监探装置。

==剧情==
[[通用电气|通用电气]]公司副总裁杰克·唐纳吉相信自己的母亲科琳因为-{飓}-风导致航班取消，应该是不会在圣诞节期间来看望自己了，对此他非常高兴。可没想到科琳选择先坐[[汽车|汽车]]到达[[亚特兰大|亚特兰大]]，再从[[哈茨菲尔德－杰克逊亚特兰大国际机场|亚特兰大国际机场]]乘飞机来到[[纽约|纽约]]。丽兹·莱蒙的家人也在假日里来到了[[GE大楼|GE大楼]]，包括她因一次事故导致[[海马回|海马回]]受损，大脑失去[[短期记忆|短期记忆]]功能的弟弟米奇·莱蒙，他一直以为现在还是1985年12月7日，自己还只有17岁，但他实际上已经40岁了。杰克的母亲立刻就对莱蒙的家人感到讨厌，因为他们总是显得很开心，很乐观。科琳于是决心让这家人露出真面目，给儿子看看他们个个都一团糟，他们相互的关系也一样。杰克、科琳和丽兹一家一起度过了融洽的一天，之后还一起去吃晚餐。科琳在餐桌上说，看起来丽兹是两老最喜欢的孩子，米奇于是透露1985年时双亲在一个本该前去观看丽兹参加足球赛的晚上实际上是带他去看电影《[[七宝奇谋|七宝奇谋]]》了。丽兹于是又意外说漏了嘴，让米奇知道自己其实已经40岁了。然后莱蒙一家开始大吵起来，科琳的计划实现了。

与此同时，《{{lang|en|TGS}}与崔西·乔丹》节目组的一众编剧和演员正准备在编剧办公室举办一年一度的鲁达圣诞节聚会。而崔西由于前不久刚被一名[[法官|法官]]判决必须在脚踝上戴一个用来监测汗液[[酒精|酒精]]含量的装置，因此不能参加这次聚会，并对此深感烦恼。[[电视台|电视台]]杂工跟班肯尼斯·帕塞尔（{{lang|en|Kenneth Parcell}}，[[杰克·麦克布瑞尔|杰克·麦克布瑞尔]]）也很烦恼，因为他觉得这些同事没有一个了解圣诞节的真谛。于是他取消了鲁达圣诞节，让所有人都在他和加里牧师（{{lang|en|Reverend Gary}}，约翰·F·穆尼饰）的带领下坐下来交谈，了解圣诞节的真正含意。结果众演员和编剧从中得到“启发”，跑出电视台大楼要去拉倒楼前那棵大圣诞树。肯尼斯最终阻止了他们，崔西则透露自己已经在不该喝酒的时间里喝酒了。

这一集结束时，《{{lang|en|TGS}}与崔西·乔丹》的演员和剧组成员在一起庆祝圣诞节；而餐厅里，莱蒙一家仍然在争吵，杰克对科琳说：“圣诞快乐，妈妈”，原来他也对看到莱蒙一家并不那么完美而感到高兴。

==制作==

本集节目中，崔西·乔丹因[[醉酒驾驶|醉酒驾驶]]而被法院判决在脚踝上戴一个可以监探汗液中酒精含量的装置，这一故事情节实际上是根据男演员崔西·摩根的真实经历改编<ref name="ign" /><ref name="dui_3">{{cite news |first=Michael |last=Wilson |title=Humor Sliced and Diced |url=http://www.nytimes.com/2008/01/06/fashion/06nite.html |work=The New York Times |date=2008-01-06 |accessdate=2013-09-13}}</ref>，他曾两次因酒驾被捕，第一次是2005年12月，第二次是2006年11月。2005年被捕后他被判罚款390[[美元|美元]]，36个月[[缓刑|缓刑]]，并需强制参加一个酒精教育课程<ref name="dui_1">{{cite web |url=http://www.tv.com/news/tracy-morgan-arrested-for-drunk-driving--7306/ |title=Tracy Morgan arrested for drunk driving |accessdate=2013-09-13 |last=Mahan |first=Colin |date=2006-11-28 |publisher=TV.com |deadurl=no |archiveurl=https://web.archive.org/web/20070627131444/http://www.tv.com/story/7306.html |archivedate=2007-06-27 }}</ref>。而2006年被捕后，他被处以罚款1000美元，[[驾驶证|驾驶证]]也被暂停使用6个月<ref name="dui_2">{{cite web |url=http://www.eonline.com/news/54403/tracy-morgan-cops-to-uncool-charge |title=Tracy Morgan Cops to Uncool Charge |accessdate=2013-09-13 |last=Keller |first=Julie |publisher=E!}}</ref>，还需要在脚踝上戴一个安全连续远程酒精监控器（{{lang|en|Secure Continuous Remote Alcohol Monitoring}}，简称{{lang|en|SCRAM}}），该仪器可以对皮肤表面汗液的酒精浓度进行实行监控<ref name="dui_4">{{cite web |url=http://www.tmz.com/2007/04/28/tracy-morgan-cops-a-plea/ |title=Tracy Morgan Ordered to SCRAM, Escapes Jail Time |accessdate=2008-08-19 |date=2007-04-28 |publisher=TMZ.com |archivedate=2012-10-11 |deadurl=no |archiveurl=https://web.archive.org/web/20121011132555/http://www.tmz.com/2007/04/28/tracy-morgan-cops-a-plea/ }}</ref>。

==反响==
