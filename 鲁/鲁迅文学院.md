'''鲁迅文学院'''，位于[[北京市|北京市]][[朝阳区_(北京市)|朝阳区]]文学馆路45号（芍药居校区）、北京市朝阳区八里庄南里27号（八里庄校区），是[[中国作家协会|中国作家协会]]直属单位，是[[中华人民共和国|中华人民共和国]]唯一一所国家级的以联系作家、服务作家、团结作家、培养作家为宗旨的教学与研究机构<ref name=jj>{{Cite web |url = http://www.chinawriter.com.cn/z/luyuan/index.shtml |title = 鲁迅文学院简介 |author =  |date =  |publisher =中国作家网  |language =  |accessdate =2018-01-05  |quote =  }}</ref>。

==简介==
1949年[[中华人民共和国|中华人民共和国]]成立后不久，[[丁玲|丁玲]]等从[[延安|延安]]来的革命作家即向[[中华人民共和国中央人民政府_(1949–1954)|中央人民政府]]呈报了《创办文学院建议书》，提出：“按文学艺术各部门来说，文学是一种基础艺术，但我们有戏剧、音乐、美术各学院，恰恰缺少文学院，所以有创办文学院之必要。”1950年，[[中央人民政府文化部|中央人民政府文化部]]批文同意成立'''中央文学研究所'''。丁玲任所长，[[张天翼|张天翼]]任副所长。经半年多筹备，1950年10月，中央文学研究所在[[北京市|北京市]][[鼓楼东大街|鼓楼东大街]]103号的[[四合院|四合院]]内宣布成立。出席成立大会的有[[郭沫若|郭沫若]]、[[茅盾|茅盾]]、[[周扬|周扬]]、[[沙可夫|沙可夫]]、[[李伯钊|李伯钊]]、[[李广田|李广田]]等，郭沫若称这是个伟大事件，“在中国历史是第一次”<ref name=jje>{{Cite web |url =http://www.chinawriter.com.cn/zxjg/lxwxy/  |title = 鲁迅文学院简介 |author =  |date =  |publisher = 中国作家网|language =  |accessdate = 2018-01-05 |quote =  }}</ref><ref name=jj/>。

1950年代，郭沫若、[[胡乔木|胡乔木]]、周扬、茅盾、[[郑振铎|郑振铎]]、[[老舍|老舍]]、[[曹禺|曹禺]]、[[叶圣陶|叶圣陶]]、[[吴组缃|吴组缃]]、[[艾青|艾青]]、[[何其芳|何其芳]]、[[张天翼|张天翼]]、[[田间|田间]]等诗人、作家、理论家、教育家、戏剧家在中央文学研究所授课。中央文学研究所培养出一批优秀作家，包括[[马烽|马烽]]、[[西戎_(作家)|西戎]]、[[胡正|胡正]]、[[陈登科|陈登科]]、[[唐达成|唐达成]]、[[邓友梅|邓友梅]]、[[玛拉沁夫|玛拉沁夫]]、[[张志民_(作家)|张志民]]、[[苗得雨|苗得雨]]，《[[小兵张嘎|小兵张嘎]]》作者[[徐光耀|徐光耀]]，《[[红色娘子军|红色娘子军]]》作者[[梁信|梁信]]等人。作家[[梁斌|梁斌]]在此边工作边创作，写成了《[[红旗谱|红旗谱]]》<ref name=jje/>。

1953年，中央文学研究所更名'''中国作家协会文学讲习所'''，归属[[中国作家协会|中国作家协会]]。1958年，因“[[丁玲|丁玲]]、[[陈企霞|陈企霞]]”问题，讲习所停办<ref name=jje/><ref name=jj/>。

[[中共十一届三中全会|中共十一届三中全会]]后，文学讲习所开始恢复。但因无固定校舍，只能到处租房办学。1984年，文学讲习所更名'''鲁迅文学院'''，在[[北京市|北京市]][[朝阳区_(北京市)|朝阳区]]八里庄南里27号建立校舍。在新时期的20年中，鲁迅文学院培养出一批青年作家，如[[蒋子龙|蒋子龙]]、[[叶文玲|叶文玲]]、[[叶辛|叶辛]]、[[陈国凯|陈国凯]]、[[王安忆|王安忆]]、[[张抗抗|张抗抗]]、[[高洪波_(作家)|高洪波]]、[[竹林_(作家)|竹林]]、[[邓刚|邓刚]]、[[赵本夫|赵本夫]]、[[朱苏进|朱苏进]]、[[乔良_(军人)|乔良]]、[[刘毅然|刘毅然]]、[[刘兆林|刘兆林]]、[[莫言|莫言]]、[[毕淑敏|毕淑敏]]、[[余华|余华]]、[[刘震云|刘震云]]、[[周大新|周大新]]、[[迟子建|迟子建]]、[[陆天明|陆天明]]、[[张平_(作家)|张平]]、[[周梅森|周梅森]]、[[何申|何申]]、[[范小青|范小青]]、[[秦文君|秦文君]]、[[叶广苓|叶广苓]]、[[王旭峰|王旭峰]]等。2001年评选的四部[[茅盾文学奖|茅盾文学奖]]作品中，获奖作者王安忆、张平、王旭峰都曾是鲁迅文学院学员<ref name=jje/><ref name=jj/>。

2001年，在[[中共中央宣传部|中共中央宣传部]]支持下，鲁迅文学院校舍重新改建，添置了教学设施<ref name=jje/>。

2002年9月至2003年1月，鲁迅文学院举办首届中青年作家高级研讨班，主要作家有[[关仁山|关仁山]]、[[马丽华|马丽华]]、[[徐坤_(作家)|徐坤]]、[[谈歌|谈歌]]、[[柳建伟|柳建伟]]、[[孙惠芬|孙惠芬]]、[[巴音博罗|巴音博罗]]、[[竹雄伟|竹雄伟]]、[[凌可新|凌可新]]、[[张梅_(作家)|张梅]]、[[红柯_(作家)|红柯]]、[[陈继明|陈继明]]、[[衣向东|衣向东]]、[[金瓯_(作家)|金瓯]]、[[戴来|戴来]]等<ref name=jje/>。

2003年，鲁迅文学院举办第二届高级研讨班（主编班）<ref name=jje/>。

2004年3月至2004年7月，鲁迅文学院举办第三届高级研讨班（中青年作家班），主要作家有[[邱华栋|邱华栋]]、[[张旻|张旻]]、[[程青|程青]]、[[刘亮程|刘亮程]]、[[祝勇|祝勇]]、[[赵光鸿|赵光鸿]]、[[潘小平|潘小平]]、[[徐剑|徐剑]]、[[鲍玉学|鲍玉学]]、[[胡学文|胡学文]]等<ref name=jje/>。

2004年9月9日至2005年1月，鲁迅文学院举办第四届高级研讨班（少数民族中青年作家班），集中了全国24个民族的青年作家<ref name=jje/>。
