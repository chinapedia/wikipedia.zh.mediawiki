'''竺規身'''（{{bd|1884年|6月29日|1978年|2月27日}}），中国著名基督教[[牧师|牧师]]。

== 慕尔堂牧师 ==
1884年6月29日，竺規身出生於[[浙江省|浙江省]][[宁波府|宁波府]][[奉化|奉化]]縣大埠鄉後竺村。年幼時父母相繼亡故，1900年由親戚介紹往上海當學徒，1906年与店主之女结婚后独立开店经营。1908年，竺規身在[[衛理公會|衛理公會]][[慕尔堂|慕尔堂]]受洗。同年開始作義務傳道。1911年棄商作全職傳道员。他追求被圣灵充满，在1915年5月20日晚禱時受靈洗，1917年被按立為牧師。

1924年，竺規身被[[衛理公會|衛理公會]]派往[[哈爾濱|哈爾濱]]開展宣教事奉，1928年返回，继续担任[[慕尔堂|慕尔堂]]牧师。

1930年代，竺規身說服衛理公會高層，借出[[慕尔堂|慕尔堂]]給[[宋尚節|宋尚節]]開佈道會，空前成功，全國各地的邀請函紛至踏來。宋博士前往各地传道时，竺规身的长女竺臨恩担任譯員。

== 自办錫安堂 ==
1937年，中日战争爆发，[[希腊|希腊]]侨民[[裴雅各|裴雅各]]回国，将他创办的康脑脱路荣耀会交给竺規身。竺辭去慕爾堂主任牧師，将荣耀会改为中华基督教堂。1940年代，上海数十个独立堂会联合成立[[灵工团|灵工团]]。1946年，竺規身被選為上海靈工團終身職監督。

1949年，竺规身发起信徒祷告、奉献，购置延安中路1125号的1亩多土地，建造[[锡安堂_(上海)|锡安堂]]，至1953年5月落成，能容纳600多人，这是1949年以后上海兴建的第一座大教堂，当时的《天风》杂志亦加以报道<ref>竺規身：《中華基督教錫安堂落成獻堂典禮》，《天风》366期，1953.5.18</ref>。此后锡安堂成為上海[[灵工团|灵工团]]的总堂。<ref>{{cite web

== 在文革中 ==
1966年8月，[[文化大革命|文化大革命]]爆發，所有牧師都遭到批鬥。1969年，在強大的政治壓力下，九成傳道人公開放棄信仰。84歲高齡的竺牧師寧願受死亦不肯否认基督，批判圣经，不肯写大字报批斗地方教会女同工[[左弗如|左弗如]]，而被戴上反革命分子的帽子，在里弄监督劳动十余年之久<ref>{{cite book
  | last = 张锡康

1978年2月，因竺规身在巴西的长女写信给华国锋，而获平反。一周后，2月27日凌晨四時，竺規身无疾而终，在世95年。
