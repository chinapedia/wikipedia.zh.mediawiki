'''下游工程'''也称'''生物分离工程'''（{{lang|en|bioseparation}}），是[[生物工程|生物工程]]的一个组成部分。生物化工产品通过微生物发酵过程、[[酶反应|酶反应]]过程或动植物细胞大量培养获得，从上述发酵液、反应液或培养液中分离精制有关产品的过程即为'''[[下游加工过程|下游加工过程]]'''（{{lang|en|Downstream Processing}}）。传统上由一些化学工程的单元操作组成，但由于生物物质的特性，有其特殊要求，且其中的某些单元操作在一般[[化学|化学]]工业中应用较少，所以近年来其研究领域越发接近生物工程方向。

== 特点 ==
下游加工过程的生物产品，由于氧传递限制、细胞量和[[产物抑制|产物抑制]]，往往是产物浓度低的水溶液。其组分复杂，含大分子、小分子、可溶物、不可溶物和化学添加物等；产物稳定性差，易发生[[化学降解|化学降解]]和[[微生物降解|微生物降解]]；由于培养过程[[分批操作|分批操作]]导致生物变异性大。然而生物工程成品主要面向药品和食品，所以其质量要求较一般化学工程产品高。 

== 单元操作 ==
# 细胞回收技术（[[固液分离|固液分离]]）：絮凝，离心，过滤，微过滤。
# [[细胞破碎|细胞破碎]]技术：球磨，高压匀浆，化学破碎技术
# 初步纯化技术：[[吸附法|吸附法]]，离子交换，[[沉淀|沉淀]]（盐析法，有机溶剂沉淀，等电点，沉淀剂），[[溶剂萃取|溶剂萃取]]，[[两水相萃取|两水相萃取]]，超临界萃取，逆胶束萃取，[[膜分离|膜分离]]技术
# 高度纯化技术：各类层析，亲和，疏水，聚焦，[[离子交换|离子交换]]，[[结晶法|结晶法]]，[[色层分离|色层分离]]
# 成品加工：浓缩，除菌与热原，喷雾干燥，气流干燥，沸腾干燥，冷冻燥
