'''下島站'''（{{jpn|j=下島駅|hg=しもじまえき|rm=Shimojima eki|f=y}}）是位於[[日本|日本]][[長野縣|長野縣]][[伊那市|伊那市]]西春近小出島區的[[東海旅客鐵道|東海旅客鐵道]]（JR東海）[[飯田線|飯田線]]車站。

== 車站構造 ==
*單式月台，1面1線，不可以[[列車交會|列車交會]]
*[[地上車站|地上車站]]。
*由[[伊那市站|伊那市站]]管理的[[無人站|無人站]]。
*設有候車室。

== 車站周邊 ==
車站是位於田園地帶與河岸之間。近年開始發展住宅地區，部分是用作[[公寓|公寓]]發展。

*[[天龍川|天龍川]]
*[[長野縣道146號南箕輪澤渡線|長野縣道146號南箕輪澤渡線]]
*[[國道153號|國道153號]]

== 利用狀況 ==
以下為每年平均每日上落車人次：
* 63人（2005年）
* 68人（2006年）
* 79人（2007年）
* 78人（2008年）
* 89人（2009年）

== 历史 ==
* 1913年（[[大正|大正]]2年）12月27日 - 伊那電車軌道（1919年改名為[[伊那電氣鐵道|伊那電氣鐵道]]）宮田站至伊那町（現在的伊那市）延伸段啟用，新設'''下島站'''。
* 1932年（[[昭和|昭和]]7年） - 車站改變位置。
* 1943年（[[昭和|昭和]]18年）8月1日 - 伊那電氣鐵道線國有化，成為[[日本國有鐵道|國鐵]]飯田線車站。[[東海道本線|東海道本線]][[濱松站|濱松站]]至[[名古屋站|名古屋站]]、飯田線各站、[[中央本線|中央本線]][[上諏訪站|上諏訪站]]至[[鹽尻站|鹽尻站]]、[[松本站|松本站]]上車的乘客可使用這站。
* 1954年（昭和29年）12月1日 - 東京都區內各站與[[長野站|長野站]]上車的乘客也可使用這站。
* 1968年（昭和43年）4月1日 - 乘客上下車站的限制取消，同時車站成為[[無人站|無人站]]。
* 1987年（昭和62年）4月1日 - 因[[國鐵分割民營化|國鐵分割民營化]]，本站成為東海旅客鐵道的車站，變成[[無人站|無人站]]。
