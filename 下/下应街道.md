'''下应街道'''是[[中国|中国]][[浙江省|浙江省]][[宁波市|宁波市]][[鄞州区|鄞州区]]一个街道，辖区面积22.2平方公里，户籍人口1.3万人，登记外来人口2.8万人。<ref>[http://xiaying.nbyz.gov.cn/Article/about.asp 下应简介] {{webarchive|url=https://web.archive.org/web/20120605112738/http://xiaying.nbyz.gov.cn/Article/about.asp |date=2012-06-05 }}</ref>

== 沿革 ==
2012年4月10日，下应街道鄞县大道以北部分拆分出[[潘火街道|潘火街道]]。

== 地理位置 ==
下应街道地处宁波市东南，东邻[[东钱湖|东钱湖]]，南接[[云龙镇|云龙镇]]，西连[[首南街道|首南街道]]、[[中河街道|中河街道]]，北至[[潘火街道|潘火街道]]。

== 下辖 ==
* 社区：东兴、天宫
* 居委会：下应
* 行政村：黎明、史家码、江六、河东、河西、胜利、东升、姜村、湾底

== 交通 ==
[[环城东路|环城东路]]和[[环城南路_(宁波)|环城南路]]依傍而过，离[[北仑港|北仑港]]17公里，离[[宁波栎社国际机场|宁波栎社国际机场]]8公里。
