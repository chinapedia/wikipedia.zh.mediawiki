'''下瓦爾登州'''（[[德語|德語]]：Nidwalden；[[法語|法語]]：Nidwald；[[意大利語|意大利語]]：Nidwaldo；[[羅曼語|羅曼語]]：Sutsilvania）位於[[瑞士|瑞士]]中部。人口39,866，首府設於[[施坦斯|施坦斯]]（Stans）。為傳統上的「半州」。

== 地理 ==
此州位於瑞士中部，北面連接[[琉森湖|琉森湖]]，東南西面被山脈環繞。西面和南面接壤[[上瓦爾登州|上瓦爾登州]]，東接[[烏里州|烏里州]]，北面與[[琉森州|琉森州]]及[[施維茨州|施維茨州]]隔[[琉森湖|琉森湖]]相望。州內最高點是洛特施托利峰（Rotstöckli）（海拔2901米）。

== 歷史 ==
下瓦爾登在1291年與[[烏里州|烏里州]]及[[施維茨州|施維茨州]]在[[呂特利|呂特利]]（Rütli）締結盟約，成立[[瑞士|瑞士]]邦聯。州政府要到[[十四世紀|十四世紀]]末才成立，包括議會及法庭。在[[十四世紀|十四]]及[[十五世紀|十五世紀]]期間[[上瓦爾登半州|上]]、下瓦爾登人民在大事上會共同商議，可是兩個半州從來沒有合併過。例如[[上瓦爾登半州|上瓦爾登]]並無參與侵略[[貝林佐納|貝林佐納]]、[[里維埃拉|里維埃拉]]及[[布連尼奧|布連尼奧]]地區的行動。

在1500年左右有很多下瓦爾登人去當僱傭兵，他們有很多當兵後索性移居外地，例如法國的[[阿爾薩斯|阿爾薩斯]]（Alsace）地區，這紓緩了下瓦爾登人口過多的問題。[[法國大革命|法國大革命]]後，[[拿破崙|拿破崙]]開始擴張版圖，下瓦爾登人民拒絕接受他頒布的新憲法後，法軍就在1798年9月9日攻擊下瓦爾登，最少400名居民被殺。[[拿破崙|拿破崙]]對下瓦爾登的統治在1814年結束，統治期間作出的改變都被回復過來。之後，下瓦爾登只在1877年修改過憲法。

== 政治 ==
州行政局（Regierungsrat）由七位成員組成。本地議會有60席。下瓦爾登因為是半州，因此在瑞士聯邦議會只能擁有一名代表。（全州有兩名）

== 經濟 ==
在1500年左右，下瓦爾登人主要是當僱傭兵。由[[十九世紀|十九世紀]]中葉以降，貿易、工業及旅遊業得到了長足的發展。到[[二十世紀|二十世紀]]中葉為止，下瓦爾登的經濟產業主要是[[農業|農業]]，牛隻及[[乳酪|乳酪]]主要出口到[[意大利|意大利]]北部。現在主導經濟的是中小型企業，行業有機械製造、醫療用品、國際貿易、光學及電子儀器等。州內最大的僱主是一間叫皮拉圖斯的飛機建造商。

傳統行業仍然很重要，尤其是伐木業和農業。上瓦爾登的農業專注於蓄牧業，供應奶類及肉類。多數農莊是家族經營。

== 旅遊業 ==
在冬季及夏季，湖光山色吸引大量遊客到下瓦爾登遊覽。主要的渡假村集中在Klewenalp、[[施坦斯山|施坦斯山]]（Stanserhorn）、Bannalp和Bürgenstock。

== 文化 ==
當地團體致力保護上瓦爾登的傳統文化，例如是傳統音樂、[[約德爾調|約德爾調]]（Yodelling）、舞蹈、服飾、戲劇及節日等。當地不時有舉辦演奏會和畫展。

== 行政區劃 ==
由於下瓦爾登面積太小因此並無分區，但有十一個市級行政區：
* Beckenried
* Buochs
* Dallenwil
* Emmetten
* Ennetbürgen
* Ennetmoos
* Hergiswil
* Oberdorf
* [[施坦斯|施坦斯]]（首府）
* Stansstad 
* Wolfenschiessen
