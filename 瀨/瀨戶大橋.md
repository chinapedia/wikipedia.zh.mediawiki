'''瀨戶大橋'''（{{jpn|j=瀬戸大橋|hg=せとおおはし|rm=Seto Ōhashi|f=y}}），是橫跨[[日本|日本]][[本州|本州]]（[[岡山縣|岡山縣]][[倉敷市|倉敷市]]）到[[四國|四國]]（[[香川縣|香川縣]][[坂出市|坂出市]]）間之[[瀨戶內海|瀨戶內海]]上、共十座[[橋樑|橋樑]]的總稱，也是{{link-ja|本州四國聯絡橋|本州四国連絡橋}}的三條路線之一。雖亦有將瀨戶大橋誤寫為「瀨戶內海大橋」的情況，但瀨戶內海大橋是指[[西瀨戶自動車道|西瀨戶自動車道]]。過去為避免混淆兩橋而曾將瀨戶大橋稱為「備讚瀨戶大橋」。瀨戶大橋是[[岡山市|岡山市]]至[[高松市|高松市]]的{{link-ja|國道30號|国道30号}}所屬路段之一。

== 概要 ==
[[File:Seto-Ohashi_Bridge.jpg|210px]]
瀨戶大橋自1978年開始施工，修建耗時9年6個月，在1988年4月10日開始通車，總建設費約11,330億日元<ref name="gaiyo">{{cite web |url=https://www.mlit.go.jp/hakusyo/transport/shouwa63/ind000202/frame.html |script-title=ja:瀬戸大橋の概要 |accessdate=2017-07-11 |publisher={{lang|ja|国土交通省}} |language=ja}}</ref>。

瀨戶大橋上層的部分是4車道的[[瀨戶中央自動車道|瀨戶中央自動車道]]，下層是[[四國旅客鐵道|JR四國]][[本四備讚線|本四備讚線]]（暱稱[[瀨戶大橋線|瀨戶大橋線]]），是一座兩層的[[公路鐵路兩用橋|公路鐵路兩用橋]]。瀨戶大橋由{{link-ja|鹽飽群島|塩飽諸島}}五個島嶼之間的六個橋樑和連接這些橋樑的高架橋構成，橋樑部分全長9,368米，算上高架部分則全長13.1公里，是世界最長的鐵路公路兩用橋<ref name="gaiyo"></ref>。瀨戶大橋包括[[懸索橋|懸索橋]]、[[斜拉橋|斜拉橋]]、[[桁架橋|桁架橋]]三種橋樑。

[[File:Seto-Ohashi_Line_001.JPG|210px]]
瀨戶大橋在施工時使用了海底無線爆破等當時世界首次應用的技術，並且也是世界首個在容易因氣象條件和負重而變形的大型懸索橋修建鐵路的橋樑。為使鐵軌免受橋樑變形的影響，日本鐵路部門專門研究緩衝裝置以吸收鐵軌變形的影響<ref>{{cite web |url=http://bunken.rtri.or.jp/PDF/cdroms1/0004/2008/0004004832.pdf |script-title=ja:鉄道吊橋と緩衝桁軌道伸縮装置 |accessdate=2017-07-11 |publisher={{lang|ja|鉄道総合技術研究所}} |language=ja}}</ref>。

瀨戶大橋的鐵路部分可容納[[新幹線|新幹線]]、[[在來線|在來線]]共四條鐵軌，現在僅鋪設並使用了在來線部分的中央兩條鐵軌。在[[四國橫斷新幹線|四國橫斷新幹線]]的計劃中，瀨戶大橋將增設兩條兩條鐵軌，東側兩條鐵軌用於在來線，西側兩條鐵軌用於新幹線。瀨戶大橋的設計最高限速是公路100km/h（第1種第2級）、鐵路在來線部分130km/h、新幹線部分160km/h<ref name="sakaide">{{cite web |url=http://www.city.sakaide.lg.jp/soshiki/seisaku/setoohashibridge.html |script-title=ja:瀬戸大橋（本州四国連絡橋） |accessdate=2017-07-11 |publisher={{lang|ja|坂出市}} |language=ja}}</ref>。

瀨戶大橋開始通車之後，倉敷市下津井田之浦的部分居民和橋下的各島民相繼抱怨瀨戶大橋線的[[噪音|噪音]]及[[光污染|光污染]]。現在瀨戶大橋為了減少光污染，只在部分時間點燈，以降低對住民生活的影響。另外，橋上的電燈本來的用途是維護、檢修橋樑。

== 歷史 ==
-{zh-hant:[[File:Map_of_Honshu-Shikoku_Bridge_Project_(zh-hant).svg|210px]]; zh-hans:[[File:Map_of_Honshu-Shikoku_Bridge_Project_(zh-hans).svg|210px]];}-
自[[明治|明治]]時期，香川縣就有興建瀨戶大橋的設想<ref>{{cite web |url=http://www.jb-honshi.co.jp/seto-ohashi/shoukai/rekishi1.html |script-title=ja:架橋構想の芽生え |accessdate=2017-07-11 |publisher={{lang|ja|本四高速}} |language=ja}}</ref>。1955年，{{link-ja|宇高連絡船|宇高連絡船}}紫雲丸發生事故，造成包括修學旅行的小學生在內168人死亡。此後香川縣議會對日本政府提出要求修建本州、四國兩地之間鐵路的要求<ref>{{cite web |url=http://www.mlit.go.jp/jmat/monoshiri/judai/30s/30s_siun_3ukoui.htm |script-title=ja:汽船紫雲丸機船第三宇高丸衝突事件 |accessdate=2017-07-11 |publisher={{lang|ja|国土交通省}} |language=ja}}</ref>。1958年，香川縣議會決定通過橋樑連接兩地，並對國會提出架橋計劃書。1959年，建設省開始調查本四連絡橋各路線，並提出了「宇野-高松線」（B案）、「玉野市日比-高松市下笠居線」（C案）、「兒島-坂出線」（D案）三個候選方案。最終因B案和C案的施工難度及費用遠高於D案而決定採用D案<ref>{{cite web |url=http://www.jb-honshi.co.jp/seto-ohashi/shoukai/rekishi2.html |script-title=ja:ルートの選定 |accessdate=2017-07-11 |publisher={{lang|ja|本四高速}} |language=ja}}</ref>。

[[File:Shimotsui-Seto_Bridge_who_saw_from_Okayama_Prefecture.JPG|210px]]
1973年，日本政府決定同時興建西瀨戶自動車道、瀨戶大橋、神戶淡路鳴門自動車道三條本四連絡橋路線，但同年因[[第一次石油危機|第一次石油危機]]爆發，該計劃被凍結。1975年，日本政府決定解凍其中一條路線，並在1977年決定在三條路線中首先興建瀨戶大橋。1978年10月10日，瀨戶大橋開始施工<ref>{{cite web |url=http://www.jb-honshi.co.jp/seto-ohashi/shoukai/rekishi3.html |script-title=ja:着工までの経緯 |accessdate=2017-07-11 |publisher={{lang|ja|本四高速}} |language=ja}}</ref>。

1987年1月17日時，瀨戶大橋所有公路和鐵路的隧道實現貫通。同年8月12日，瀨戶大橋的全部橋樑部分實現貫通，本州和四國首次實現陸路相連。1988年4月2日至3日，瀨戶大橋實施了橋上馬拉松等開放活動，並在4月10日正式通車<ref name="sakaide"></ref>。2005年，由於日本道路公團等民營化關係法成立，瀨戶大橋改由JB本四高速管理。2008年，為紀念通車20週年，瀨戶大橋舉辦了健康馬拉松大會，這是瀨戶大橋首次因惡劣天氣以外的原因而全面禁止汽車行駛<ref>{{cite web |url=http://www.jb-honshi.co.jp/seto-ohashi/shoukai/rekishi4.html |script-title=ja:開通から20周年まで |accessdate=2017-07-11 |publisher={{lang|ja|本四高速}} |language=ja}}</ref>。2013年，為紀念瀨戶大橋開業25週年，[[JR西日本|JR西日本]]和[[JR四國|JR四國]]開通了[[黃昏特快號列車|黃昏特快號列車]]瀨戶大橋開業25週年紀念號列車<ref>{{cite web |url=https://www.westjr.co.jp/press/article/2013/03/page_3445.html |script-title=ja:瀬戸大橋線開業25周年記念トワイライトエクスプレス車両瀬戸大橋線開業25周年記念号発売 |accessdate=2017-07-11 |publisher={{lang|ja|JR西日本}} |language=ja}}</ref>。

== 瀨戶大橋的紀念設施 ==
[[File:Setonaikai_National_Park,_Great_Seto_Bridge_-_panoramio.jpg|250px]]
倉敷市瀨戶大橋架橋紀念館和瀨戶大橋架橋紀念公園位於[[岡山縣|岡山縣]][[倉敷市|倉敷市]]{{link-ja|兒島地區|児島地域}}。瀨戶大橋架橋紀念公園以岡山縣史跡下津井城遺跡為中心，可在此一覽瀨戶內海景色，面積約16.7公頃<ref>{{cite web |url=http://www.kurashiki-tabi.jp/see/1282/ |script-title=ja:瀬戸大橋架橋記念公園 |accessdate=2017-07-11 |publisher={{lang|ja|倉敷観光WEB}} |language=ja}}</ref>。

{{link-ja|瀨戶大橋紀念公園|瀬戸大橋記念公園}}和{{link-ja|瀨戶大橋紀念館|瀬戸大橋記念館}}位於[[香川縣|香川縣]][[坂出市|坂出市]]番之州緑町（番之州高架橋西側），是1988年{{link-ja|瀨戶大橋架橋紀念博覽會|瀬戸大橋架橋記念博覧会}}（瀨戶大橋博'88/四國）的舉辦地點。瀨戶大橋紀念館即是博覽會當時的展廳之一，設有展示瀨戶大橋相關資料的展示室。瀨戶大橋紀念公園面積10.2公頃，公園也是[[道之驛|道之驛]]，西側的瀨戶大橋塔在108米處設有旋轉瞭望室。瀨戶大橋塔的南側有{{link-ja|香川縣立東山魁夷瀨戶內美術館|香川県立東山魁夷せとうち美術館}}<ref>{{cite web |url=http://www.my-kagawa.jp/point/172/ |script-title=ja:瀬戸大橋記念公園 |accessdate=2017-07-11 |publisher={{lang|ja|倉敷観光WEB}} |language=ja}}</ref>。

== 瀨戶大橋的紀念發行物 ==
1988年，為紀念瀨戶大橋通車，日本郵政部門發行了4種面值60日元的紀念[[郵票|郵票]]。日本造幣局亦在1988年發行了面額500日元的瀨戶大橋通車白銅[[紀念幣|紀念幣]]<ref>{{cite web |url=http://www.mint.go.jp/popup/data_kinenkahei_set_page05.html |script-title=ja:青函トンネル・瀬戸大橋開通記念貨幣 |accessdate=2017-07-11 |publisher={{lang|ja|造幣局}} |language=ja}}</ref>。
