'''喀戎'''（{{lang-el|'''Χείρων'''}}，[[羅馬化|轉寫]]：Chiron）也譯'''奇戎'''，是希腊神话中非常著名的贤人，属于[[半人马|半人马]]族。

== 介绍 ==
在[[希腊神话|希腊神话]]中，[[半人马|半人马]]居住于[[希腊|希腊]]中东部屏达思山和[[爱琴海|爱琴海]]之间、叫做[[忒萨利|忒萨利]]和[[阿耳卡狄亚|阿耳卡狄亚]]的两个地区。他们经常因为放荡和好色而被描述成酒神[[狄俄倪索斯|狄俄倪索斯]]的追随者，他们中的一个例外便是喀戎。

== 傳說 ==
凱隆是[[水仙女|水仙女]][[菲呂拉|菲呂拉]]與[[克洛諾斯|克洛諾斯]]的孩子。無法接受凱隆樣貌的菲呂拉，請[[宙斯|宙斯]]將自己變成菩提樹，拋棄了凱隆。<br>
與其他兇殘野蠻的[[半人馬|半人馬]]不同，凱隆以其和善及智慧著稱，所以在中文裏也常被美称为[[人马|人马]]。他從阿波羅和阿緹蜜絲學習了音樂、醫術、狩獵...等各種技藝，爾後成為多位希腊英雄的導師，當中包括[[珀耳修斯|珀耳修斯]]、[[忒修斯|忒修斯]]、[[阿基里斯|阿基里斯]]、[[伊阿宋|伊阿宋]]和[[赫拉克勒斯|赫拉克勒斯]]。他也是医药之神[[阿斯克勒庇俄斯|阿斯克勒庇俄斯]]的老师。

赫拉克勒斯在一次追杀[[半人马|半人马]]强盗的过程中误伤了他的老师喀戎，使用的正是当年老师赠送的无坚不摧的箭，那支箭穿过一个半人马强盗的身体，射中了后面喀戎的膝蓋；喀戎本是不死之身，但赫拉克勒斯的箭浸泡了九头蛇[[许德拉|许德拉]]毒血，喀戎感到疼痛的折磨超过死亡。当时[[普罗米修斯|普罗米修斯]]因為偷了天[[火|火]]給人類使用，正被宙斯綁在[[高加索山|高加索山]]上受苦刑，喀戎决定将自己与普罗米修斯交换，自己放弃永生让普罗米修斯恢復自由，使两方都得以解脱。后被[[宙斯|宙斯]]升上天空，成为[[人马座|人马座]]（罗马诗人[[奧維德|奧維德]]: ''Fasti v.379''）。

由于希腊人继承了[[苏美尔|苏美尔]]人的黄道星座体系，常有将黄道星座[[人马座|人马座]]（也称“射手座”）與后者半人马座相混淆。在[[埃拉托斯特尼|埃拉托斯特尼]]的希腊神话体系中，半人馬座是[[缪斯|缪斯]]女神们为了纪念陪伴她们的[[薩堤爾|薩堤爾]]克洛托斯（[https://en.wikipedia.org/wiki/Krotos Krotos]）而请求宙斯将后者升上天空的星座。由于克洛托斯发明了[[射箭|射箭]]并擅长骑马狩猎，宙斯为成为星座的克洛托斯添了马身和弓箭。

== 天文學上的凱隆 ==
'''凱隆'''在[[天文学|天文学]]上是指类似凱龍星（[[小行星2060|小行星2060]]号）的，又称“柯瓦尔天体”的天体，是一颗位于[[土星|土星]]和[[天王星|天王星]]轨道之间的远日[[小行星|小行星]]。1977年由[[查爾斯·科瓦爾|查爾斯·科瓦爾]]（''C.T.Kowal''）发现。

== 流行文化 ==
* 喀戎在[[輕小說|輕小說]]《[[Fate/Apocrypha|Fate/Apocrypha]]》中以黒之陣營的「Archer」的職階登場，在故事中被描述以人類的身體現世以免真名被識破，寶具為「天蠍一射」；以後在手機遊戲《[[Fate/Grand_Order|Fate/Grand Order]]》中沿用《Fate/Apocrypha》的資料，並且追加半人馬形態登場。
