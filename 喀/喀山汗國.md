'''喀山汗國'''（{{lang-tt|'''Qazan xanlığı/Казан ханлыгы'''}}；{{lang-ru|'''Казанское ханство'''}}）是[[金帳汗國|金帳汗國]]瓦解後產生的[[汗国|汗国]]（1441—1552年），由[[拔都|拔都]]的兄弟[[禿花帖木兒|禿花帖木兒]]後人[[兀魯·穆罕默德|兀魯·穆罕默德]]和兒子[[馬赫第提克|馬赫第提克]]住在[[喀山|喀山]]建立的獨立汗國，範圍大約是[[伏尔加保加利亚|伏尔加保加利亚]]區域、現今的[[楚瓦什共和國|楚瓦什共和國]]、[[馬里埃爾共和國|馬里埃爾共和國]]、[[韃靼斯坦共和國|韃靼斯坦共和國]]、[[烏德穆爾特共和國|烏德穆爾特共和國]]和[[巴什科爾托斯坦共和國|巴什科爾托斯坦共和國]]。

== 历史 ==
在[[馬赫第提克|馬赫第提克]]在位期間，汗國正式立國。1452年，他的一位兄弟[[卡西姆|卡西姆]]逃亡[[莫斯科|莫斯科]]。[[莫斯科公国|莫斯科公国]]把一塊[[奥卡河|奥卡河]]畔的、以他的名字命名土地封給他，名為[[卡西莫夫|卡西莫夫]]。其屬下一部分逃亡至[[克里米亞汗國|克里米亞汗國]]。

馬赫第提克在1452年冬季，進攻[[莫斯科公國|莫斯科公國]]。他的继承者喀山汗[[易卜拉欣|易卜拉欣]]甚至1468年征服了[[维亚特卡|维亚特卡]]，但是，他不久就被迫与俄国人媾和，并交还了他所俘获的人。易卜拉欣的两个儿子[[伊尔哈姆|伊尔哈姆]]和[[穆罕默德·阿明|穆罕默德·阿明]]爭立。阿明逃亡俄國，俄國派軍護送，使他即位。但他在1505年反叛，次年他打败了一支莫斯科公国的军队。

莫斯科為了控制[[伏爾加河|伏爾加河]]，向喀山进攻。1516年阿明死後，喀山汗國絕嗣。接著克里米亞[[格來王朝|格來王朝]]和莫斯科公國的[[瓦西里三世|瓦西里三世]]輪流強加自己的人選於喀山。1545—1552年，罗斯军队为阻止蒙古鞑靼人的侵袭而对喀山汗国进行征战，叫“[[喀山远征|喀山远征]]”。经过数次的征战，1552年10月2日[[伊凡四世|伊凡四世]]对喀山实施围攻和强击，结果喀山汗国被消灭，伏尔加河中游地区并入俄国。俄罗斯的象征[[圣瓦西里大教堂|圣瓦西里大教堂]]即为纪念[[伊凡四世|伊凡四世]]攻陷喀山汗国所建。

==经济和人口==

居民是操[[突厥語|突厥語]]的[[巴什基爾人|巴什基爾人]]、[[楚瓦什人|楚瓦什人]]，[[诺盖人|诺盖人]]和[[烏拉爾語系|烏拉爾語系]]的[[摩爾多瓦人|摩爾多瓦人]]、[[馬里人|馬里人]]（舊稱'''切列米斯人'''）、[[烏德穆爾特人|烏德穆爾特人]]，以及一些[[科米人|科米人]]部落。最主要是[[保加尔人|保加尔人]]。

汗国居民主要從事农业(黑麥、大麥和燕麥)，也有手工业（生產粘土製品，木材和金屬手工，皮革，盔甲，犁和珠寶）和養蜂。汗国治下的城市贸易商多与高加索、中亚、莫斯科公国贸易，到十六世纪俄罗斯是最重要贸易夥伴，主要市場是喀山和伏爾加河上的大[[巴扎|巴扎]]。

== 政治 ==
國家最高統治者是可汗，他的行为决定是基於一个内阁会议的協商。貴族組成[[伯克|伯克]]，[[埃米爾|埃米爾]]，[[米儿咱|米儿咱]]，神職人員也有相當大影響力，有經學院。軍隊二萬至六萬人。由當地[[保加爾人|保加爾人]]組成貴族，但汗的[[保鑣|保鑣]]由[[欽察人|欽察人]]与诺蓋人出任，也有韃靼人，他們是由俄羅斯的精英、[[突厥语族|突厥语族]]人、蒙古人结合而成，由四大[[貴族|貴族]]（阿尔根、八林、钦察、Shirin）主持汗國。该国以伊斯兰教为国教。多数人是平民。战俘多卖给[[奥斯曼帝国|奥斯曼帝国]]、[[布哈拉汗国|布哈拉汗国]]和[[希瓦汗国|希瓦汗国]]，非穆斯林要交[[吉茲亞税|吉茲亞税]]。

縱觀其歷史，汗國容易出現內亂和爭位。汗國115年更換了19次可汗，一些多次登基。汗常常被當地貴族選出。

== 文化 ==
俄罗斯消息指出，至少有五种语言被用在喀山汗国。首要是[[鞑靼语|鞑靼语]]，其中包括喀山鞑靼人中的方言（其前身为穆斯林保加尔人）和西部方言的mishars。其书面形式（旧鞑靼语）。[[楚瓦什语|楚瓦什语]]言是保加尔语言的一个分支，该保加尔语言，也强烈地影响了鞑靼语的中部方言。其他三組，大概是[[马里语|马里语]]，[[摩爾達維亞语|摩爾達維亞语]]和[[巴什基尔语|巴什基尔语]]。
