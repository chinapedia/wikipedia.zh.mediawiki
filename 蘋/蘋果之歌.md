《'''蘋果之歌'''》（{{lang|ja|りんごのうた}}，''Ringo no Uta／A Song of Apples''）是日本音樂家[[椎名林檎|椎名林檎]]的第九張[[單曲|單曲]]，由[[EMI音樂_(日本)|東芝EMI]]／[[維京唱片|Virgin Music]]於2003年11月25日發行<ref name="JPN" />。在跨業合作部分，A面歌曲「蘋果之歌」成為日本[[NHK|NHK]]節目‘{{link-ja|大家之歌|みんなのうた}}’節目主題曲<ref name="RingonoUtaDRAMASONG" />。初回生產限定盤為「特殊黑痣仕樣盒」。

在單曲銷售成績方面，本張單曲在日本Oricon銷售榜上最高位居單週第2名<ref name="RingonoUtaOriconWeek2" />，名列2004年年度銷售榜第85位<ref name="2004sale" />。而在銷售認證上，單曲《蘋果之歌》銷售突破10萬張，在2003年11月通過日本唱片協會認定為金唱片<ref name="RIAJ200311" />。

== 單曲概要 ==
《'''蘋果之歌'''》是椎名林檎繼2003年1月發表《[[莖(STEM)～諸侯出遊篇～|莖(STEM)～諸侯出遊篇～]]》後，睽違十個月的新單曲，也是椎名林檎日後以樂團[[東京事變|東京事變]]出道前，以個人名義所發行的最後一張單曲。同時，歌曲「蘋果之歌」更為NHK節目「大家之歌」的主題曲<ref name="RingonoUtaDRAMASONG" /><ref name="songofapple1" />。歌曲「蘋果之歌」最早在椎名林檎第三次日本巡迴演唱會《[[Electric_Mole#演唱會簡介|雙六高潮]]》中演唱，日後也有被收錄至影音專輯中。之後，樂團東京事變改編了歌曲「{{nihongo|蘋果之歌|りんごのうた}}」，並改以「{{nihongo|蘋果之歌|林檎の唄}}」為名收錄至首張專輯《[[教育_(專輯)|教育]]》中，然而東京事變演唱的版本並沒有收錄至椎名林檎的專輯裡。

同時，這也是椎名林檎點痣後的首張作品，而椎名林檎也將點痣創意發揮到極限，她特地在單曲的塑膠盒子上印製一個黑點，當單曲封面放入塑膠盒中時，就會發現那顆黑點準確地對在椎名林檎的嘴角上，如此以來她的痣又回來了<ref name="intro">{{cite web |url=http://ent.sina.com.tw/music/news/content/1832 |title=有影無？椎名林檎「招牌黑痣」代“痣”大條啦!! |publisher=玫瑰唱片 2003.11.25 |language=zh-tw |deadurl=yes |archiveurl=https://web.archive.org/web/20160201232426/http://ent.sina.com.tw/music/news/content/1832 |archivedate=2016年2月1日 |df= }}</ref>。日後，唱片公司將A面歌曲以特別曲收錄至2008年11月25日發行的套裝專輯《[[MoRA|MoRA]]》中，藉以慶祝椎名林檎出道十週年。

至於單曲發行的時間恰逢椎名林檎出道屆滿五年，為了慶祝出道五年，椎名林檎特地選在她的生日11月25日發行。而在單曲發售時程上，新單曲發行的消息最早於2003年9月19日就在官網上發佈<ref name="songofapple1" />，到了11月6日則在官方網站上發表單曲的音樂錄影帶<ref name="songofapple2" />。日本於11月25日發行[[Copy_Control|Copy Control]]的版本<ref name="JPN" />，台灣也於當天同步發行進口版<ref name="TWN" />與台壓版<ref name="TWN" />。後來因為Copy Control的格式已經絕版，則改成CD-DA格式收垃，並與椎名林檎出道十周年紀念專輯《[[我·放電|我·放電]]》一同於2008年7月2日發售<ref name="JPNRENEW" />。

最後，在單曲銷售成績方面，本張單曲在發行當週以2.2萬張的銷售量在日本Oricon銷售榜2003年12月第一週（11月24日發表）付的榜單上位居第八名<ref name="RingonoUtaOriconWeek" />，而在單曲發售日前就登上銷售排行榜前十名的紀錄是繼歌手[[中森明菜|中森明菜]]的單曲《北翼》後<ref name="Killer" group="注">1984年1月9日・週間第2名</ref>第二次發生<ref name="songofapple1" />。之後，在日本Oricon銷售榜上最高位居單週第2名<ref name="RingonoUtaOriconWeek2" />，總計在Oriocn銷售榜上榜13週<ref name="RingonoUtaOriconConsecutiveWeek" />，累積銷售達11.4萬張，名列2004年年度銷售榜第85位<ref name="2004sale" />。同時，單曲銷售突破10萬張，在2003年11月通過日本唱片協會認定為金唱片<ref name="RIAJ200311" />。

=== 音樂錄影帶 ===
[[Image:Shiina01-10-mv.jpg|thumb]]
在歌曲「蘋果之歌」的[[音樂錄影帶|音樂錄影帶]]中，導演{{link-ja|番場秀一|番場秀一}}、松本正明將主軸定位為椎名林檎出道以來造型的總回顧，像是出道作品「幸福論」的學生服裝、歌曲「本能」的護士服、「石膏」的蕾絲造型，甚至是「罪與罰」的紅色緊身衣等都出現在音樂錄影帶中，而劇組也重現了當時拍音樂錄影帶的場景<ref name="songofapple2" /><ref name="intro" />。甚至連當時因為懷孕的緣故而改成動畫版音樂錄影帶的歌曲「真夜中的純潔」，也以原本拍攝的造型登場<ref name="intro" />。

但椎名林檎本人並沒有再度穿上這些服裝，而是請到演員{{link-ja|富田靖子|富田靖子}}、[[麻生久美子|麻生久美子]]及歌舞伎演員{{link-ja|中村七之助|中村七之助}}三人跨刀演出<ref name="RINGOUTANA">{{cite web |url=http://www.epochtimes.com/b5/3/11/18/n413621.htm |title=椎名林檎發新曲宣傳又出新招 |publisher=中央社 |language=zh-tw}}</ref><ref name="songofapple3">{{cite web |url=http://www.vibe-net.com/news/?news=0016446 |title=椎名林檎　「りんごのうた」の特殊広告が解禁 |publisher=vibe-net |language=ja |deadurl=yes |archiveurl=https://web.archive.org/web/20160213233548/http://www.vibe-net.com/news/?news=0016446 |archivedate=2016年2月13日 |df= }}</ref>。演員富田靖子在音樂錄影帶中模仿歌曲「本能」中，穿護士服出拳的動作<ref name="RINGOUTANA" /><ref name="songofapple3" />；演員麻生久美子則是模仿歌曲「石膏」中，懷抱吉他的鏡頭<ref name="RINGOUTANA" /><ref name="songofapple3" />；歌舞伎演員中村七之助則演出歌曲「莖」中，彈古箏的畫面<ref name="RINGOUTANA" /><ref name="songofapple3" />。

== 銷售排行 ==
=== Oricon 銷售榜(日本) ===
《蘋果之歌》於11月25日發行單曲。發行首週，在日本公信榜Oricon的單曲榜2003年12月第1週付的榜單中以22,360張銷售量居單週第8名，同週冠軍是銷售量突破28萬大關的搖滾樂團[[Mr.Children|Mr.Children]]的單曲「[[掌/Kurumi|掌/Kurumi]]」，第二名則是銷售9萬張的R&B雙人組合[[化學超男子|化學超男子]]的單曲「{{link-ja|YOUR NAME NEVER GONE/Now or Never/You Got Me|YOUR NAME NEVER GONE/Now or Never/You Got Me}}」<ref name="RingonoUtaOriconWeek" />。第二週則以銷售量48,529張位居第2位，創下本張單曲最高的單週排行名次<ref name="RingonoUtaOriconWeek2" />。總計《蘋果之歌》在日本公信榜Oricon的單曲榜上榜13週，累積銷售達11.4萬張<ref name="RingonoUtaOriconConsecutiveWeek" />。同時在2003年年銷售榜中，《蘋果之歌》以11.4萬張銷售量居全年第85名<ref name="2004sale" />。
