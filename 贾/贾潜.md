'''贾潜'''（{{bd|1903年|5月9日|1996年|12月25日}}）原名'''贾荣卿'''，字'''光尧'''，抗战时期改名'''贾潜'''，[[河南省|河南省]][[滑县|滑县]]杨公店村人，[[中华人民共和国|中华人民共和国]]法学家、官员、法官<ref name=shishi>法学家贾潜逝世，人民日报1996年12月31日，第四版</ref><ref name=hebing>何兵，最高法院曾经的“右派”，21世纪2013(5)</ref>。

==生平==
贾潜毕业于[[北平|北平]][[朝阳大学|朝阳大学]]法律科。大学毕业后历充法院、民政厅职员和官吏，后到[[山东|山东]][[济南|济南]]长期从事[[律师|律师]]工作<ref name=hebing/><ref name=qindian>{{Cite web |url =http://mt.sohu.com/20150903/n420378321.shtml  |title = 贾潜：毛主席钦点的特别军事法庭庭长 |author =  |date = 2015-09-03 |publisher = 搜狐 |language =  |accessdate =  |quote =  }}</ref>。1940年2月参加[[中国共产党|中国共产党]]领导的革命工作，先后在[[冀鲁豫边区|冀鲁豫边区]][[滑县|滑县]]抗日民主政府和冀鲁豫行署担任科长、参议员。1946年7月加入中国共产党。[[第二次国共内战|第二次国共内战]]时期，历任[[晋冀鲁豫边区|晋冀鲁豫边区]]政府行政委员会委员、晋冀鲁豫边区高等法院副院长、[[华北人民法院|华北人民法院]]审判长、[[华北人民政府|华北人民政府]]法制委员会委员<ref name=shishi/><ref name=qindian/>。

[[中华人民共和国|中华人民共和国]]成立后，贾潜历任[[中央人民政府最高人民法院|中央人民政府最高人民法院]]党组成员、中央人民政府最高人民法院委员兼刑事审判庭庭长、审判委员会委员，中央法制委员会民事法规委员会副主任委员等职。贾潜是《[[中华人民共和国人民法院组织法|中华人民共和国人民法院组织法]]》的起草人之一。1956年，[[中华人民共和国最高人民法院特别军事法庭|中华人民共和国最高人民法院特别军事法庭]]审判[[日本|日本]]战犯，贾潜被任命为特别军事法庭庭长<ref name=shishi/>。当时审判日本战犯时，中华人民共和国尚无刑法和刑事诉讼法，经贾潜建议，由[[全国人大常委会|全国人大常委会]]专门颁布了一个决定，根据该决定审判了日本战犯<ref name=hebing/>。

1957年，在[[反右运动|反右运动]]中，贾潜被打成[[中华人民共和国最高人民法院|中华人民共和国最高人民法院]]内部的头号“右派分子”。1957年12月，《[[人民日报|人民日报]]》宣布了贾潜的右派罪状。《人民司法工作》1957年第12期总计七篇文章，四篇针对贾潜：《揭露和批判贾潜的反党言行》，《驳贾潜的审判独立谬论》，《贾潜在为谁着想？》，《对贾潜的所谓学习法律科学的刮视》。在最高人民法院举办的辩论会上，最高人民法院交通运输审判庭庭长[[曾汉周|曾汉周]]等人“揭露和批判贾潜的反党言行”<ref name=hebing/>。1958年2月，贾潜被划为“右派分子”<ref name=canshi/>。

1958年6月3日，全国人大常委会决定撤销右派分子贾潜的最高人民法院审判委员会委员、庭长的职务；撤销右派分子[[林亨元|林亨元]]、[[朱耀堂|朱耀堂]]的最高人民法院副庭长职务；撤销右派分子[[刘寅夏|刘寅夏]]的最高人民法院审判委员会委员、审判员的职务；撤销右派分子[[杨显之|杨显之]]、[[张向前|张向前]]、[[郝绍安|郝绍安]]、[[田明中|田明中]]的最高人民法院审判员职务<ref>{{Cite web |url =http://www.npc.gov.cn/wxzl/gongbao/2000-12/06/content_5000530.htm  |title = 全国人民代表大会常务委员会撤职人员（1958年6月3日） |author =  |date =2000-12-06  |publisher = 中国法院网 |language =  |accessdate =  |quote =  }}</ref>。

1961年7月9日，贾潜被任命为[[国务院参事|国务院参事]]<ref name=canshi>{{Cite web |url =  http://www.counsellor.gov.cn/lsyg/dsj/2012-02-08/255.shtml|title = 1978年大事记 |author =  |date =  2012-02-08|publisher = 国务院参事室|language =  |accessdate =  |quote =  }}</ref>。

1978年12月18日，最高人民法院向[[国务院参事室|国务院参事室]]寄送了[[中国共产党最高人民法院机关委员会|中国共产党最高人民法院机关委员会]]关于给现任国务院参事贾潜平反的决定。决定说：经重新审查，贾潜的“右派分子”属于错划, 经报请[[中共中央组织部|中共中央组织部]]批准，予以改正，撤销原划为右派分子的决定和开除中国共产党党籍的决定<ref name=canshi/>。

1980年2月，贾潜被任命为[[中华人民共和国司法部|中华人民共和国司法部]]顾问。1982年离职休养后，先后任司法部咨询委员会副主任，[[中国法学会|中国法学会]]、[[中国律师协会|中国律师协会]]顾问，[[中国社会科学院|中国社会科学院]]法学研究所学术指导等工作。1989年获全国老有所为精英奖<ref name=shishi/>。

1996年12月25日，贾潜在北京病逝，享年93岁<ref name=shishi/>。
