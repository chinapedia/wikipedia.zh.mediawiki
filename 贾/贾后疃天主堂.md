'''贾后疃天主堂'''，全名'''天主教北京教区贾后疃圣母无染原罪堂'''，位于[[北京市|北京市]][[通州区_(北京市)|通州区]][[潞城镇|潞城镇]]贾后疃村，隶属[[天主教北京教区|天主教北京教区]]，是通州区乃至京东现存最大的天主教堂。<ref name=dytz/>

==简介==
贾后疃天主堂位于潞城镇贾后疃村，地处通州城东偏南30里。历史上教友人数曾超过万人。21世纪初有数千教友。<ref name=dytz>{{Cite web|url =http://www.dayuntongzhou.com/zhdt_2012/2012/xinwen/tongzhou/tongzhouzixun/tushuotongzhou/2012/0413/63679.html|title =京东最大的教堂——通州贾后疃天主堂|author =|date =2012-01-01|publisher =大运通州网|language =|accessdate =|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20170920121228/http://www.dayuntongzhou.com/zhdt_2012/2012/xinwen/tongzhou/tongzhouzixun/tushuotongzhou/2012/0413/63679.html|archivedate =2017年9月20日|df =}}</ref>

[[清朝|清朝]][[顺治|顺治]]年间，贾后疃贾某（号半仙）在北京[[宣武门|顺治门]]一带艰难谋生，多次到[[宣武门天主堂|宣武门天主堂]]听神父讲道，并受洗入教。后来回到家乡传教，从此贾后疃的教友逐渐增加。20年后，北京[[西什库天主堂|西什库天主堂]]主教[[樊国良|樊国良]]（[[法国|法国]]人）先后派王、李、邹等神父来到贾后疃村中及通州城内外传教。1873年，先后在贾后疃、[[牛牧屯天主堂|牛牧屯]]、通州西门外兴建天主教堂，并且在通州设立教区，管辖[[房山|房山]]、[[良乡|良乡]]、[[长辛店镇|长辛店]]、[[琉璃河地区|琉璃河]]、[[顺义|顺义]]、[[宝坻|宝坻]]、[[香河|香河]]等地的传教、建堂事宜。<ref name=dytz/>

1873年，王神父在贾后疃监工兴建天主堂，为罗马式建筑，可容600多人。院落坐北朝南。神父宿舍为砖券，前廊合瓦券棚顶，墙壁嵌有木制门窗，棂苇帘灰顶，前有两株[[洋槐|洋槐]]。西部是教堂，坐西朝东，南北长约30米，东西10多米，高14米，面檐砌有连弧砖券，门上顶端立有[[十字架|十字架]]。<ref name=dytz/>

1900年夏，在[[义和团运动|义和团运动]]中，贾后疃天主堂受到义和团重创。后来，西什库天主堂派李神父重整教堂及教会事务，并且在附近传教，后来另派董神父（[[荷兰|荷兰]]人）、王玛窦，1927年以后又先后派王公覆、王汝楫、孟焕禹等人继任本堂神父。这段时期，贾后疃堂区管辖谭台、榆林庄、卜落垡、八百户、豆各庄、城子等20多个村庄，远大于[[牛坊天主堂|牛坊堂区]]和[[牛牧屯天主堂|牛牧屯堂区]]。<ref name=dytz/>

[[中华人民共和国|中华人民共和国]]成立后，从1955年到1957年，先后有胡明德神父、贾西中神父任本堂神父，胡明新任副本堂神父。1958年后，教堂停止宗教活动，被改成小学校。[[文革|文革]]期间曾用作礼堂、仓库，1972年至1973年间教堂被彻底拆除。<ref name=dytz/><ref name=xhw/>

1970年代，贾后疃村人胡明新神父在南口农场[[劳改|劳改]]结束后回村里居住。[[中共十一届三中全会|中共十一届三中全会]]后，宗教政策获得落实。根据教友们的要求，人民政府同意重建贾后疃天主堂。经协商，决定由天主教北京教区拨款5万元人民币，[[通县|通县]]人民政府补偿1万元，此外还有教友奉献8000元人民币，有关方面赞助3万多元人民币，以上合计10万元人民币。依照原天主堂的外观及构造，由侉子店乡建筑施工队负责设计及施工。1986年4月18日动工。同年8月15日即[[圣母升天|圣母升天]]瞻礼之际祝圣。同年12月7日竣工。1986年12月8日即竣工次日，北京教区主教[[傅铁山|傅铁山]]主持了开堂典礼，通县人民政府等各级领导前来祝贺。新教堂可容400多人就座，堂内各种设施齐全。开堂后第一任本堂为胡明新神父，之后便是本堂鲍静坡神父、副本堂贺斌神父。<ref name=dytz/><ref name=xhw>{{Cite web|url =http://www.bj.xinhuanet.com/bjpd_sdwm/2006-02/08/content_1845024.htm|title =贾后疃天主堂|author =|date =2006-02-08|publisher =新华网|language =|accessdate =|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20161226221359/http://www.bj.xinhuanet.com/bjpd_sdwm/2006-02/08/content_1845024.htm|archivedate =2016-12-26}}</ref>

2005年，教堂顶部塌方。2006年底，教堂由于损坏严重而全部拆除。由北京市第五建筑工程有限公司承建，在老教堂的原址上重建了新教堂，2008年7月新教堂落成。新教堂长150米，宽80米，占地面积近20亩，雄伟壮观，内部装修朴实庄重。<ref name=dytz/><ref name=zoujin>{{Cite web |url = http://www.tianguangbao.org/2013.10.1/13.htm |title = 圣神的宫殿 纯洁的子民——走进通州区贾后疃堂 |author = 李妍 |date = 2013-10-01 |publisher = 天光 |language =  |accessdate =  |quote =  }}</ref>2013年5月25日，北京教区[[李山|李山]]主教来到贾后疃堂看望本堂闫一杰神父及其他神长教友，并主持了贾后疃天主堂圣母山、宗徒广场祝圣典礼。<ref>{{Cite web |url =  http://www.chinacatholic.cn/html/ccic/report/14051993-1.htm|title =北京教区李山主教主持贾后疃天主堂圣母山、宗徒广场祝圣典礼  |author =  |date = 2013-06-05 |publisher = 中国天主教 |language =  |accessdate =  |quote =  }}</ref>
