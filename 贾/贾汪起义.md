'''贾汪起义'''也稱為'''賈汪投共'''，是[[中国|中国]][[第二次国共内战|第二次国共内战]]中[[淮海战役|淮海战役]]初期，由[[中国共产党|中国共产党]]策划在[[中華民國國軍|中华民国国军]]内部进行的一场大规模倒戈。

== 背景 ==
在淮海战役前夕，[[冯玉祥|冯玉祥]][[西北军|西北军]]旧部第三绥靖区在[[徐州|徐州]]和[[黄百韬|黄百韬]]兵团之间运河一线和[[中国人民解放军第三野战军|解放军华东野战军]]对峙。第三绥靖区司令部在徐州，前线指挥部在[[徐州|徐州]][[贾汪|贾汪]]。第三绥靖区包括第59军、第77军。这两个军其前身是[[卢沟桥事变|卢沟桥事变]]中的抗日军队第29军和[[张自忠|张自忠]]领导的第33集团军。[[冯治安|冯治安]]为第三绥靖区总司令，[[何基沣|何基沣]]、[[张克侠|张克侠]]为副司令官。何、张两人都是共产党地下党员。何基沣去过[[延安|延安]]，张克侠是[[周恩来|周恩来]]亲自介绍入党的。

== 经过 ==
华东野战军副司令员[[粟裕|粟裕]]遣使华东野战军13纵队的联络部长[[杨斯德|杨斯德]]去和[[何基沣|何基沣]]、[[张克侠|张克侠]]联系，争取他们动员更多将领叛變，另一方面在绝对优势兵力的情况下，组织包围这支孤军，并计划对其进行围歼。[[何基沣|何基沣]]、[[张克侠|张克侠]]投誠的态度很积极，但也坦诚他们没有能力令2个军的人都投诚，因为他们两个都是只有副司令这个虚职。要争取士兵和军里其他干部的意见。张克侠的态度比较谨慎，表示没有把握争取冯治安。虽然[[何基沣|何基沣]]、[[张克侠|张克侠]]二人力劝部下投誠，但其下面所辖77军与59军的态度并不同，77军倾向投誠，59军则倾向抵抗。

1948年11月6日，淮海战役开始。解放军重点攻击第三绥靖区东面的黄百韬兵团。11月7日，粟裕命令华野第七纵队立刻向驻守在[[万年闸|万年闸]]地区的59军守军发动猛攻，59军守军不敌，几小时就损失一个营，对59军的士气造成巨大震动。何、张二人再次召开了两军团级以上干部大会，大家一致认为，不投降就会被全歼的态势已经非常明显。而且第三绥靖区由于非蒋嫡系部队，平时蒋就对他们多加防范，少恩寡德，苛扣经费，这次又将这支军队推上第一线作战，令军人非常不满。在会上，何基沣愤怒的说道：“以前蒋介石对我们西北军像是后娘养的，现在他蒋介石危急了，又要我们给他戴孝帽子，当然我们不给他戴。”在全部团级以上干部无人反对的情况下，11月8日，59军和77军部分共2.3万人决定投奔中共的解放军。没有参加起义的5000余人，撤往徐州方向<ref name="a1"/>。

== 影响 ==

贾汪起义造成国军防线在中间开了一个大口子。华东野战军迅速插到徐州和正在逃往徐州的黄百韬兵团之间，使得[[黄百韬|黄百韬]]兵团被包围。黄百韬兵团的歼灭是[[淮海战役|淮海战役]]徐蚌戰役中解放军取得的第一个军事大胜利，为中共在淮海战役取得全盘胜利打下了基础。11月18日，[[毛泽东|毛泽东]]在电报中写道“北线何、张起义是（淮海战役）第一个大胜利”<ref name="a1">{{Cite web |url=http://js.ifeng.com/xz/humanity/detail_2015_04/13/3776881_0.shtml |title=鲜为人知的徐州“贾汪起义”|accessdate=2017-12-17 |author= |date=2015-04-13 |work= |publisher=凤凰网|language=Zh-hans }}</ref>。
