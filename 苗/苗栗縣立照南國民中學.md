'''照南國中'''位於[[苗栗縣|苗栗縣]][[竹南鎮|竹南鎮]]公教路2號，學生總人數1，385人，其特別之處在於其操場與鄰近的照南國小共用，故常可在下課時間見到國中生出現在同一個操場上的情景，該校創立於1969年，校地三．二四四一公頃.

== 教室狀況 ==
具有各年級教室共48間，資優班三間，電腦教室兩間，導師室五間，科任導師室一間，輔導室一間，多媒體視聽中心一間，總務處一間，教務處一間，特教教室兩間，理化教室兩間，保健室一間，腳力場一間，活動中心及專科教室大樓各一棟，音樂教室兩間，本校設有蒸飯室一間，籃球場數座，實驗室兩間，蔣公銅像一座(蔣公銅像在興建新校舍時,已被拆除)。為了因應入學人數增加，故在原教師[[停車場|停車場]]處加蓋新校舍，已完工啟用多時。
原照南國中北棟因校舍老舊於104年2月拆除
106年4月完工9/20剪綵啟用今｛敦品樓｝
由於照南國中圖書館地處偏遠
現今改為在敦品樓｛敦品圖書館｝

敦品圖書館以誠品風格以及退休張凰謙老師捐贈書法字畫為風格。

敦品樓簡介：敦品樓興建量體為三層樓二十一間教室，大樓設計以學校傳統文化作為建築與景觀的延續，並融合學校願景及校本課程為設計概念主軸，其中敦品圖書館結合文學和空間美學成為幽雅的閱讀環境，每層樓有7間教室。

班級數（106學年度）：

1.普通班方面～國一（七年級）17班、國二（八年級）16班、國三（九年級，畢業班）15班，合計48班。歸屬學校[[教務處|教務處]]來管轄。

2.特殊教育班方面～設置『[[身心障礙類|身心障礙類]]』[[特殊教育班|特殊教育班]](“身障類特教班”)，交由學校[[輔導室|輔導室]][[特教組|特教組]]所管理。

== 飲水 ==
其校內多處設有飲水機，大多數學生都自備杯子水壺以盛裝飲用水，濾心雖然定期更換，且有定時作水質檢測，但也有部份學生表示其不喜歡飲水機的味道，寧可自家中帶水或者去福利社買水，其飲水機少數外觀有灰塵，不過純屬班級清理問題。

== 福利社 ==
其原名為照南國中員工福利社，每位學生入學前皆會預繳10元的股份，於學期末在回饋給學生，但目前回饋金額有嚴重下跌的趨勢，從早期的500元到150元，現在只剩10元。其內部置有多個冷藏[[冰箱|冰箱]]，一個冷凍櫃及蒸包子機一個，包子採用桂冠冷凍包子，其水採用[[金車公司|金車公司]]出產的波爾[[礦泉水|礦泉水]]，及小包[[零食|零食]]由（福壽食品公司）出產還有小包的小饅頭由旺旺公司出產，另外也有多種果汁飲料，也有22元的（統一公司）瑞穗鮮乳早上有三明治，蔥抓餅，鐵蛋可供選擇，架上擺放基本文具學用品等，另有販售制服書包的服務，也販售學生請假用的假卡。
