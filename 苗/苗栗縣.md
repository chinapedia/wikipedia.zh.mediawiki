'''苗栗縣'''（官方中文羅馬字名稱：'''Miaoli County'''，{{lang-nan-tw|Biâu-le̍k-koān}}；{{lang-hak|Mèu-li̍t-yen|syhp=Meuˇ lid ien|hlhp=Miau lidˋ rhanˇ}}），是[[中華民國|中華民國]][[臺灣省|臺灣省]]的[[縣_(中華民國)|縣]]，有'''[[山城|山城]]'''之別稱，於[[臺灣本島|臺灣本島]][[北臺灣|西北部]]，東倚[[雪山山脈|雪山山脈]]，西濱[[臺灣海峽|臺灣海峽]]，北部中港溪地帶與[[新竹縣|新竹縣]][[新竹市|市]]接壤形成竹苗生活圈，南部大安溪地帶與[[臺中市|臺中市]]相鄰互動頻繁，領域範圍雖然方整適中，但由於地勢崎嶇多山地阻隔，交通廊道受限而各鄉鎮交流不易，在發展上有諸多挑戰。

在地緣關係上，苗栗縣位於臺灣本島的西北部，不同的單位會依據其需求而將本縣歸屬到中部或北部。例如[[中華民國國家發展委員會|行政院國發會]]將本縣劃歸為[[中臺灣|臺灣中部]]區域；而[[交通部中央氣象局|交通部中央氣象局]]則將本縣劃歸為[[北臺灣|臺灣北部]]區域<ref>[http://www.cwb.gov.tw/V7/life/ 中央氣象局－氣象]</ref><ref>[http://tw.weather.yahoo.com/today.html Yahoo奇摩－氣象]</ref>，此外本縣於2006年1月時加入「北臺區域發展推動委員會」，並簽署了《北臺區域合作宣言》，2016年1月則與中部縣市簽署《中彰投苗區域治理宣言》<ref>[http://news.ltn.com.tw/news/politics/breakingnews/1562028 張菁雅，〈苗栗加入中彰投區域治理平台4縣市長簽署合作宣言〉，《自由時報》，2016年1月5日]</ref>。

==縣名由來==
苗栗縣縣名來自[[原住民|原住民]][[道卡斯族|道卡斯族]]巴利社（Bari），其社名之原住民語意為平原。最初遷移至此的[[臺灣客家人|客家人]]與[[河洛人_(台灣)|閩南人]]以其音近似[[靈貓科|靈貓科]]動物[[山貓|-{貓貍}-]]（俗體字作-{貓狸}-）之發音，即以此作為地名<ref>《重修苗栗縣誌》，卷六語言志，上冊，第1頁</ref>。其後官方文書亦有使用近音之-{貓裏}-<ref name="http://www.libertytimes.com.tw/2009/new/jun/24/today-north4.htm">自由電子報，2009-6-24，苗栗舊名劉政鴻宣布正名貓裏</ref>、-{貓裡}-或-{貓里}-者。[[清朝|清]][[光绪|光緒]]15年（1889年）[[臺北府|臺北府]]新竹縣分拆為2縣之際，將貓狸依改為近音雅字之'''苗栗'''，以作為新的縣名及堡名<ref name="苗栗市誌31頁">《苗栗市誌》，31頁</ref>，其後沿用至今。

== 歷史 ==
{{main|苗栗縣歷史}}
依據考古學家的研究，苗栗地區在史前時期已有先民在此地生活，並留下許多歷史遺跡。一千多年以前，[[臺灣|臺灣]][[原住民|原住民]]移居此地，'''苗栗'''地區成為[[平埔族|平埔族]][[道卡斯族|道卡斯族]]的生活範圍。自17世紀中葉起，[[漢人|漢人]]開始移入屯墾，原住民被迫同化或向高山遷移。[[明鄭時期|明鄭時期]]，苗栗地區歸屬[[天興縣|天興縣]]管理。[[臺灣清治時期|清治時期]]，天興縣改稱為[[諸羅縣|諸羅縣]]。1889年（清[[光绪|光緒]]15年），苗栗始設縣。1895年日本統治台灣後，清代的[[苗栗縣_(清朝)|苗栗縣]]遭廢除。1901年到1908年間，另置[[苗栗廳|苗栗廳]]，1920年廢廳置州郡，今苗栗縣轄區被劃分為[[苗栗郡|苗栗郡]]、[[竹南郡|竹南郡]]、[[大湖郡|大湖郡]]，屬[[新竹州|新竹州]]管轄。戰後初期屬新竹縣之一部，1950年[[臺灣|臺灣]]施行[[地方自治|地方自治]]後，苗栗縣恢復設置，其行政區域沿用至今<ref>[http://www.miaoli.gov.tw/index/history/his1_01.asp 《認識苗栗 歷史研革》，苗栗縣政府全球資訊網]</ref>。

== 地理 ==
=== 位置 ===
苗栗縣位於臺灣本島中北部，北邊和東北邊與[[新竹縣|新竹縣]]、[[新竹市|新竹市]]為鄰，南邊和東南邊隔著[[大安溪|大安溪]]、[[雪山山脈|雪山山脈]]與[[臺中市|臺中市]]接壤，西濱[[台灣海峽|臺灣海峽]]。極東點是[[泰安鄉|泰安鄉]]梅園村的[[大霸尖山|大霸尖山]]；極西點位於[[苑裡鎮|苑裡鎮]]房裡海岸；極南點是[[卓蘭鎮|卓蘭鎮]]內灣里南端；極北點是[[竹南鎮|竹南鎮]]崎頂里北端，中心點則約在[[獅潭鄉|獅潭鄉]]豐林村。全縣東西寬約64公里，南北長約50公里，面積1820.31平方公里，外型頗似一顆鑽石。海岸線長度自竹南鎮崎頂-{里}-海岸北端起，向南延伸至苑裡鎮房裡海岸南端止，全長約50公里<ref name="苗栗縣政府地理環境">
[http://www.miaoli.gov.tw/index/history/his2_01.asp 《認識苗栗 地理環境》，苗栗縣政府全球資訊網]</ref>。

=== 地形 ===
境內山多平原少，主要是原屬於[[雪山山脈|雪山山脈]]西側的河川[[沖積扇|沖積扇]]，因不斷受到各河川侵蝕，漸漸切割成今日的[[丘陵|丘陵]]、台地地形，數條河川隨著地勢穿梭其間，形成多變化的山川風貌。地形上大致可依序分為[[平原|平原]]、[[丘陵|丘陵]]、[[臺地|臺地]]及[[山脉|山脈]]等四種，除了臺地之外，其他地形都非常有規律的由西向東排列<ref name="苗栗縣政府地理環境"/>。

苗栗縣雨量豐沛，大、小河川密佈。主要河川由北至南有[[中港溪|中港溪]]、[[後龍溪|後龍溪]]、[[西湖溪|西湖溪]]及[[大安溪|大安溪]]。中港溪發頭於[[鹿場大山|鹿場大山]]（樂山）西北側，流域涵蓋本縣北部及[[新竹縣|新竹縣]]西南部；後龍溪發源於鹿場大山西南側，流經本縣中部；西湖溪發源於三義鄉[[關刀山_(苗栗縣)|關刀山]]北側，流經本縣中西部；大安溪發源於大霸尖山西側，流經本縣南部及台中市北部。

=== 氣候 ===
苗栗縣的位置，在北迴歸線以北，其氣候屬亞熱帶季風型氣候，年平均氣溫在20~22°C之間。但氣候的變化因素，在於地形和冬夏不同與方向季風的影響，因此沿海的平原地區及鄰近丘陵，是屬副熱帶氣候，而1,000公尺以上的山區，則屬溫帶氣候，但雪山山脈已有點偏向寒帶氣候。雨量的分佈是山區多於丘陵地，丘陵地多於平原，東部山區均在2,500公厘以上，西部沿海也有1,200公厘，年雨量約1,600公釐，年降雨日數100~125天，降雨集中在5~8月。受到季風的影響，夏季時吹西南季風，冬季時則東北季風盛行，南端的大安溪、火炎山一帶是臺灣南北氣候的分界線，為冬季季風雨的南界，因此苗栗縣的冬季比臺灣中部其他縣市較為陰冷。夏、秋之際的颱風，往往給臺灣帶來狂風豪雨，造成災害，但苗栗縣受到中央山脈的阻擋，相形之下，受害情形和其他地區相較，要來得輕微。在頭屋鄉有一特殊氣候現象：每年9月時期，因冬、夏季風風向更替，形成由上往下的強風，使得人車行走困難，尤其在空曠處更為強烈，稱「九降風」。

== 族群 ==
苗栗縣的族群由[[漢族|漢族]]（[[客家人|客家人]]為主，[[閩南人|閩南人]]次之）及[[臺灣原住民族|臺灣原住民族]]這兩類民族共同構成。
;原住民族
現在原住民人口佔縣總人口1.9%。苗栗縣昔日為[[道卡斯族|道卡斯族]][[蓬山社群|蓬山社群]]及[[後壠社群|後壠社群]]的原鄉，[[台灣荷西殖民時期|荷西時期]]對於這些[[原住民|原住民族]]部落已有詳細的記載，清領初期[[郁永河|郁永河]]的《裨海記遊》及日治初期[[伊能嘉矩|伊能嘉矩]]的《踏查日記》亦予以詳盡調查。這群原鄉人逃過了[[大航海時代|大航海時代]]殖民當局的壓迫，卻逃不過[[臺灣清治時期|清領時期]][[汉族|漢族]]先民如潮水般湧入，在經濟上及文化上的衝擊。最終僅少部份離開原鄉，遠走[[埔里盆地群#埔里盆地|埔里盆地]]。大部份選擇融入[[汉族|漢族]]文化，與所有人和平共處，而漸漸消失於歷史舞台。

[[賽夏族|賽夏族]]居住於今[[新竹縣|新竹縣]][[五峰鄉_(台灣)|五峰鄉]]及苗栗縣-{[[南庄鄉_(台灣)|南庄鄉]]}-一帶，[[臺灣清治時期|清領時期]]族群勢力最盛北達[[鳳山溪|鳳山溪]]，南至[[後龍溪|後龍溪]]，東與[[泰雅族|泰雅族]]人為鄰，西至竹苗海岸；但受到漢族先民及泰雅族的雙重壓迫，及-{[[南庄事件|南庄事件]]}-的打擊，族群勢力日漸衰退。現今總人口約5,000人，是人口僅次於[[邵族|邵族]]的第2稀少民族，其日常生活深受泰雅族及客家族群雙重影響，傳統文化急速消逝中。〈相關參見：賽夏族[[頭目|頭目]][[日阿拐|日阿拐]]及主導抗日的-{[[南庄事件|南庄事件]]}-〉。

泰雅族相傳發源於[[南投縣|南投縣]][[仁愛鄉|仁愛鄉]]山區，在[[臺灣清治時期|清領時期]]開始向外拓展，勢力遠達[[花蓮縣|花蓮]]、[[南澳鄉|南澳]]及[[烏來區|烏來]]，是北部最為強勢的[[原住民族|原住民族]]。在苗栗縣主要居住地是[[泰安鄉|泰安鄉]]的[[雪山山脈|雪山山脈]]地區。

;漢族
苗-{栗}-縣在清朝時代，是苗栗[[客家|客家人]]開墾最盛時期<ref>[http://content.edu.tw/local/mauli/chushin/1/2.htm 苗栗縣《歷史淵源》，鄉土教材皮皮遊苗栗網站]</ref>。目前沿海四鎮[[竹南鎮|竹南]]、[[後龍鎮_(台灣)|後龍]]、[[通霄鎮|通霄]]及[[苑裡鎮|苑裡]]，主要居民為使用[[臺灣話|臺灣閩南語]]的族群。除沿海四鎮及泰安鄉以外的各鄉鎮，主要人口都是使用[[臺灣客家語|臺灣客家語]]的族群；地理環境多是[[丘陵|丘陵]][[台地|台地]]，因此以各項[[農作物|農作物]]栽培為主要產業。佔人口大宗的客語族群，在地方政治上人才輩出。（此處族群是以使用語言來劃分，非民族。）<br />

所以，苗栗縣18個鄉鎮市，除廣義[[海線|海線]]4個鎮，與一個[[山地鄉|山地鄉]][[泰安鄉|泰安鄉]]外，其餘13個鄉鎮市，都是[[客家人|客家人]]為主。

所以，苗栗縣也跟[[新竹縣|新竹縣]]一樣，是臺灣的「客家大縣」。

== 區劃人口 ==
[[File:苗栗行政區劃.png|thumb]]
民國三十九年（1950年）調整行政區域，將[[新竹縣|新竹縣]]所屬[[苗栗區|苗栗區]]的3鎮5鄉、[[竹南區|竹南區]]的2鎮4鄉及[[大湖區|大湖區]]的4鄉等18個鄉鎮合併獨立為苗栗縣，縣治設於苗栗鎮，轄5鎮、13鄉。民國四十年（1951年）後龍鄉改制後龍鎮、卓蘭鄉改制卓蘭鎮，民國七十年（1981年）苗栗鎮升格為縣轄市，民國一百零四年（2015年）頭份鎮升格為縣轄市，全縣轄有2[[縣轄市|市]]（苗栗市、頭份市）、5[[鎮_(中華民國)|鎮]]（竹南鎮、後龍鎮、通霄鎮、苑裡鎮、卓蘭鎮）、11[[鄉_(中華民國)|鄉]]（造橋鄉、西湖鄉、頭屋鄉、公館鄉、銅鑼鄉、三義鄉、大湖鄉、獅潭鄉、三灣鄉、南-{庄}-鄉，泰安鄉為[[山地鄉|山地鄉]]），共有18個鄉鎮市。

=== 大眾運輸 ===
==== 高鐵、臺鐵 ====
*[[臺灣鐵路管理局|臺鐵]][[縱貫線_(北段)|縱貫線]]在本縣設有[[崎頂車站|崎頂車站]]及[[竹南車站|竹南車站]]，再往南則分岔為[[臺中線|山線]]及[[海岸線_(臺鐵)|海線]]。山線在本縣設有[[造橋車站|造橋車站]]、[[豐富車站|豐富車站]]、[[苗栗車站|苗栗車站]]、[[南勢車站|南勢車站]]、[[銅鑼車站|銅鑼車站]]、[[三義車站|三義車站]]及[[舊山線|舊山線]]的[[勝興車站|勝興車站]]；海線則設有[[談文車站|談文車站]]、[[大山車站|大山車站]]、[[後龍車站|後龍車站]]、[[龍港車站|龍港車站]]、[[白沙屯車站|白沙屯車站]]、[[新埔車站_(苗栗縣)|新埔車站]]、[[通霄車站|通霄車站]]及[[苑裡車站|苑裡車站]]。竹南、苗栗為[[一等站_(台鐵)|一等站]]，其餘均為[[三等站_(台鐵)|三等站]]以下，其中[[後龍車站|後龍]]、[[通霄車站|通霄]]、[[苑裡車站|苑裡]]為[[三等站_(台鐵)|三等站]]的較大站，有較高等級列車停靠，其餘[[三等站_(台鐵)|三等站]]及以下以區間車為主。

*[[台灣高速鐵路|台灣高鐵]]南北穿越本縣，在[[後龍鎮_(台灣)|後龍]]豐富地區設立[[苗栗車站_(高鐵)|苗栗站]]，[[豐富車站|豐富站]]北移與高鐵共站。[[苗栗車站_(高鐵)|苗栗站]]於2015年12月1日通車營運。

==== 輕軌 ====
原本有規劃[[竹竹苗輕軌|竹竹苗輕軌]]系統，將苗栗－後龍、竹南－頭份、竹南－新竹等輕軌路網列入初期規劃內容<ref name="自由時報20080426">[http://www.libertytimes.com.tw/2008/new/apr/26/today-north2.htm 《竹竹苗輕軌捷運 縣長建議延長》，自由時報電子報，2008年4月26日]</ref>。但於2010年1月被交通部退回申請計畫，短期內不會興建<ref>[http://dailynews.sina.com/bg/tw/twlocal/bcc/20100118/16101123247.html 交通部退回竹竹苗捷運 翻案難度高]</ref>。

== 教育 ==

== 體育 ==
苗栗縣最有名的是有一個尖山國小,足球全國賽得到三十幾座全國冠軍~
苗栗縣的主要體育設施有苗栗縣立體育場、[[苗栗縣立巨蛋體育館|苗栗縣立巨蛋體育館]]、苗栗縣立體育館、苗栗縣立網球場及[[竹南運動公園|竹南運動公園]]等。主要的運動競賽則有苗栗縣全民運動大會、苗栗縣中小學聯合運動會及苗栗縣籃球聯賽等。

在[[2006年中華民國全民運動會|2006年全民運動會]]裡，苗栗縣積分是114分，在25個參賽隊伍中排名第13名。

在[[2007年中華民國全國運動會|2007年全國運動會]]裡，苗栗縣獲得3面金牌、7面銀牌及5面銅牌，在25個參賽隊伍中排名第21名，僅勝[[新竹市|新竹市]]、[[澎湖縣|澎湖縣]]、[[金門縣|金門縣]]與[[連江縣_(中華民國)|連江縣]]<ref name="96全運會">[http://score96.tn.edu.tw/prize.aspx 《中華民國九十六年全國運動會 -獎牌統計》，96年全國運動會官方網站] {{webarchive|url=https://web.archive.org/web/20090125030316/http://score96.tn.edu.tw/prize.aspx |date=2009-01-25 }}</ref>。

然而苗栗縣在田徑、籃球等少數幾個運動種類仍然有突出的表現。在田徑方面，共獲得3金3銀3銅，排名第6名，表現極佳<ref name="96全運會"/>。

== 產業發展 ==
苗栗縣經濟活動以往以農牧業為主，近年來工商業發展迅速，產業結構已有大幅改變。為協助傳統農牧業生產及營運技術昇級，在[[公館鄉_(臺灣)|公館鄉]]設有「[[行政院農業委員會|行政院農業委員會]]苗栗區農業改良場」，在[[竹南鎮|竹南鎮]]設有[[臺灣動物科技研究所|臺灣動物科技研究所]]。

苗栗縣的傳統[[制造业|製造業]]以[[陶瓷|陶瓷]]業、[[手工業|手工業]]為主。其中陶瓷業曾經風光一時，產品銷售到世界各地。近年來產業逐漸外移，已不復往日盛況。[[國立臺灣工藝研究發展中心|國立台灣工藝研究所]]在[[苗栗市|苗栗市]]設有「苗栗技術輔導中心」，以輔導相關產業再創新機<ref name="苗栗市公所工商發展">[http://www.mlcg.gov.tw/mlcg/2-4.php?menuID=1260 《認識苗栗：工商發展》，苗栗市公所全球資訊網]</ref>

位於苗栗市北郊的西山工業區，有長春化工苗栗廠、台肥苗栗廠等大型[[石化|石化]]廠，均得力於鄰近之[[公館鄉_(台灣)|公館鄉]]生產豐富的[[天然气|天然氣]]與[[石油|石油]]氣<ref name="苗栗市公所工商發展"/>。為此，[[台灣中油|台灣中油]]在苗栗市設有「探採研究所」<ref>[http://www.cpc.com.tw/big5_BD/edri/home/ 台灣中油股份有限公司探採研究所官網首頁] {{webarchive|url=https://web.archive.org/web/20090519005400/http://www.cpc.com.tw/big5_BD/edri/home/ |date=2009-05-19 }}</ref>。

苗栗縣其現有的科技園區及工業區尚有[[銅鑼工業區|銅鑼工業區]]、[[竹南廣源科技園區|竹南廣源科技園區]]、[[新竹科學工業園區|新竹科學工業園區]]竹南基地、[[銅鑼中興工業區|銅鑼中興工業區]]、[[竹南工業區|竹南工業區]]、[[頭份工業區|頭份工業區]]等；另有開發中的新竹科學工業園區銅鑼基地，及規劃中的[[後龍鎮_(台灣)|後龍]]科技園區與銅鑼中平工業區，另有依[[都市計畫|都市計畫]]興建的[[苑裡北區工業區|苑裡北區工業區]]與[[苑裡南區工業區|苑裡南區工業區]]。

2015年苗栗縣出現債務破表危機。[[劉政鴻|劉政鴻]]執政的9年期間，因舉辦百年燈會、國慶煙火、世界男高音演唱以及推動客家桃花源等重大設施，到了2014年未償債務已達546億元，到[[徐耀昌|徐耀昌]]上任後，苗栗縣債務已達648億元。2015年7月，苗栗縣政府因7月份員工薪資延期發放，縣長徐耀昌尋求行政院協助。行政院打算介入苗栗財務控管，以解決危機<ref>。{{Cite news|url=http://udn.com/news/story/1/1053653 |title=苗栗提償債計畫 中央有條件出手|publisher=聯合新聞網 |date=2014-09-03}}</ref>

=== 文學 ===
;文學史及相關論述
第1屆到第5屆的「苗栗縣文學研討會」論文集。

至於苗栗縣的文學獎、縣內的[[客家山歌|客家山歌]]、「[[魍神|魍神]]」傳說，以及傳統詩社等，已有碩博士生從事研究，成果有：張宏誠〈台灣解嚴後縣市文學獎研究(1987~2010)〉、張莉涓〈苗栗[[客家山歌|客家山歌]]研究─以[[頭份市|頭份市]]、[[造橋鄉|造橋鄉]]、[[頭屋鄉|頭屋鄉]]、[[公館鄉_(臺灣)|公館鄉]]為例〉、鍾愛玲〈徘徊在「[[鬼|鬼]]」「[[怪|怪]]」之間：苗栗地區「[[魍神|魍神]]」傳說之研究〉、王幼華〈日治時期苗栗縣傳統詩社研究─以[[栗社|栗社]]為中心〉等論文。

== 政治 ==
{{main|苗栗縣政府|苗栗縣縣長|苗栗縣議會|苗栗地方法院}}

1951年，苗栗縣首屆民選縣長即歷經兩次選舉、五次投票才順利選出合法縣長，是臺灣選舉史上的異數。第一次選舉由[[黃運金|黃運金]]、[[劉定國_(1913年)|劉定國]]捉對撕殺，投了兩次票才由劉定國當選，卻因劉定國具軍人身份而被判當選無效，重新公告選舉<ref name="重修苗栗縣志自治志">[http://book.mlc.gov.tw/book/DefaultImage2.aspx 《重修苗栗縣志》〈自治志〉第261頁]</ref>。

第二次選舉共有七人參選，第一次投票結果[[黃焜發|黃焜發]]最高票（13,644票），[[李白濱|李白濱]]（12,063票）、[[賴順生|賴順生]]（11,570票）分居二、三名，其餘諸人票數稍低。由於投票人數未達全縣公民數的半數，必須再行投票。第二次投票結果李白濱獲得30,275票排名第一，高於賴順生（29,145票）與黃焜發（29,111票），但仍無人過半數而無效。后来李白滨（青年党）为时任台湾省主席吴国桢劝退当省政府参议，才由賴順生當選<ref name="重修苗栗縣志自治志"/>。

之後，劉派、黃派派系形成，壁壘分明，政治趨於穩定。縣長選舉完全是由國民黨勝選，且由「劉派」（劉定國）與「黃派」（黃運金）兩大派系輪流執政。

直到了[[1993年中華民國縣市長選舉|1993年縣長選舉]]，[[何智輝|何智輝]]代表退出國民黨，以無黨籍身分，擊敗[[中國國民黨|國民黨]]提名頭份林家所支持的[[張秋華|張秋華]]，首次打破「由國民黨提名一定當選」之慣例。此後[[苗栗縣縣長|苗栗縣長]]依序為[[傅學鵬|傅學鵬]]、[[劉政鴻|劉政鴻]]等人，也因這20多年間，[[民主進步黨|民主進步黨]]的逐步進入苗栗縣，劉派、黃派又逐漸放棄成見逐步合作，形成號稱「劉黃聯軍」共抗「民進黨」進入的局面。但「劉黃聯軍」內部還是維持「劉派」與「黃派」兩大派系既合作又競爭的格局。

== 財政問題 ==

財政問題曾是苗栗縣嚴重的問題之一，苗栗縣在劉政鴻任內，強調大建設、大發展，廣開不排富福利，長期浮編預算再「調借」中央統籌分配款和專戶資金，加上議會二十二年不曾刪預算，造成長短債從2012年開始超過法定債限，債務佔歲入比率達一六九．八九％，每位縣民平均負債七萬多，居非六都縣市第一，在全國僅次於高雄。2015年曾一度出現[[2015年苗栗縣財政危機|財政危機]]，無法發給縣政府公務員薪資。
