'''陸駿光'''（'''Luk Chun Kwong'''，{{bd|1978年|2月9日||}}），香港男演員，[[香港演藝學院|香港演藝學院]]畢業，曾為[[電視廣播有限公司|香港無綫電視]]旗下[[演員|演員]]，於2012年轉到[[香港電視網絡|香港電視網絡]]工作，至2015年4月約滿<ref>{{cite web|url=http://hk.apple.nextmedia.com/realtime/magazine/20150430/53687375/|title=港視覆核成功又如何？約滿陸駿光：諗過做地盤|date=2015-04-30|work=[[香港蘋果日報|香港蘋果日報]]}}</ref>，期間拍攝《[[導火新聞線|導火新聞線]]》、《[[大眾情性|大眾情性]]》、《[[驚異世紀|驚異世紀]]》等劇集而廣為人知。因其樣貌與日本電視劇《[[半澤直樹|半澤直樹]]》男主角[[堺雅人|堺雅人]]相似，因有港版半澤直樹之稱號，而其本人亦曾於2013年10月[[香港電視網絡歷史#.E7.94.B3.E8.AB.8B.E5.A4.B1.E6.95.97.E8.88.87.E6.8A.97.E8.AD.B0.E6.B4.BB.E5.8B.95|反對香港政府不發牌予香港電視集會]]中，飾演半澤直樹一角。<ref>[http://hk.apple.nextmedia.com/entertainment/art/20131027/18480803 娛樂名人- 【壹週刊】港版半沢直樹你認得陸駿光嗎?]</ref>2018年首次於[[ViuTV|ViuTV]]拍劇。

陸駿光曾與前無綫藝員[[李穎芝|李穎芝]]交往，後於2011年分手。現時與圈外女文員茵茵（Sherraina）交往，兩人已於2015年7月成婚。<ref>[http://hk.apple.nextmedia.com/entertainment/art/20140814/18831041 陸駿光陪未婚妻揀婚戒| 蘋果日報| 娛樂| 20140814]</ref>陸駿光有二兄一姊<ref>{{cite web |url=https://www.thestandnews.com/society/%E9%99%B8%E9%A7%BF%E5%85%89-%E4%B8%8D%E8%A6%81%E5%BE%8C%E6%82%94/ |title=陸駿光 不要後悔 |last= |first= |date=2015-4-11 |website= |publisher=立場新聞 |access-date= 2019-1-12|quote=}}</ref>，曾就讀[[德貞小學|德貞小學]]（1991年上午校畢業），出身於中醫世家。在校時曾在小五年級留級一年。及後於升中後為校內排球隊成員。他當時入讀過兩所香港中學，中二時亦曾留級一年。<ref>{{cite web |url=http://www.dmx.hk/hidden-menu-01/21-cat-audio-06/1838-%E5%AD%90%E7%A8%8B%E6%89%AE%E7%86%9F-%E9%BB%98%E9%BB%98%E8%80%95%E8%80%98%E9%9D%9C%E5%BE%85%E6%94%B6%E7%A9%AB%E7%9A%84%E9%99%B8%E9%A7%BF%E5%85%89-byid25.html |title=[子程扮熟]默默耕耘靜待收穫的陸駿光 |last= |first= |date=2015-1-12 |website= |publisher=創動力媒體 Dynamix |access-date= 2019-1-12|quote=}}</ref>
