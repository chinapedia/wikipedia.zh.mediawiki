'''陸基中段防禦系統'''（GMD）過去被稱為[[国家导弹防御系统|國家飛彈防禦系統]]（NMD），在2002年為了和其他的反飛彈計畫有所區分而改變稱呼[https://web.archive.org/web/20060903050641/http://www.mda.mil/mdalink/pdf/kadish13mar02.pdf]，例如太空基和海基的攔截計畫，以及再入大氣層階段與加速階段彈道飛彈的攔截計畫（參見[[洲際彈道飛彈#发射后的各个飞行阶段|飛行階段]]）。<ref>''[http://www.mda.mil/mdalink/pdf/kadish13mar02.pdf Missile Defense Reorganization statement] {{webarchive|url=https://web.archive.org/web/20060903050641/http://www.mda.mil/mdalink/pdf/kadish13mar02.pdf |date=2006-09-03 }}'', [http://www.mda.mil/ Mda.mil]  {{webarchive|url=https://web.archive.org/web/20060301165436/http://www.mda.mil/ |date=2006-03-01 }}</ref>

波音綜合國防系統集團是此計畫主要承包商，負責監督和整合其他分包商。陸基中途防禦系統的主要關鍵系統包括:
*[[陸基攔截器|陸基攔截器]]（GBI） - [[洛克西德·马丁公司|洛克西德·馬丁公司]]
**彈頭 -[[外大氣層截殺載具|外大氣層截殺載具]]（EKV）- [[雷神公司|雷神公司]]
*戰場管理指揮管制通訊系統（BMC3） - [[诺斯洛普·格鲁门公司|諾斯洛普‧格魯曼公司]]
*陸基雷達（GBR）
*改良式早期預警雷達（UEWR）或[[鋪路爪長程預警雷達|鋪路爪長程預警雷達]](PAVE PAWS)
*前沿配置X波段雷達（FBXB）例如雷神AN / TPY-2早期飛彈預警雷達或[[海基X波段雷達|海基X波段雷達]]平台

2002年，[[美国政府|美国政府]]将“国家导弹防御系统”改名為[[陸基中途防禦系統|陸基中途防禦系統]](GMD)專案，以分辨它和其他飛彈防禦計畫的不同處，例如太空衛星攔截、海基攔截、上升段攔截、重返段攔截等諸多方案。

2004年7月22日，第一具陸基攔截系統部署於[[阿拉斯加|阿拉斯加]] Ft. Greely, ({{coord|63.954|N|145.735|W}})。2004年底為止已經部署六枚還有兩枚於加州范登堡空軍基地，在Ft. Greely於2005又加裝了兩枚，本系統已經可以提供基礎防禦能力。

2004年12月15日，馬紹爾群島舉行的攔截測試失敗，因為[[阿拉斯加|阿拉斯加]]科迪亞克島的攔截器發射後16分鐘出現異常運動。

五角大廈發言人Larry DiRita於2005年1月13日五角大廈記者會上說"我不認為宣告系統可以運作就是目標達成。我只是說初步運作能力的目標已經在2004年底達成"。總之，最大的問題是資金"有一些部分已經可以運作但是有一些部分還不行，如果國會更關注和給更多資金於本專案，它將會相對上更快運作。" 

2005年1月18日，[[美國戰略司令部|美國戰略司令部]]指揮官督導設立「整體飛彈防禦聯合機制指揮部.JFCC IMD」，一旦該機構啟動，將推動發展科技和能力進行全球飛彈防衛和支援。

2005年2月14日，其他測試失敗的攔截發射都是因為[[瓜加林環礁|瓜加林環礁]]的地面支援設施故障導致，不是攔截器飛彈本身原因。<ref>[[Missile_Defense_Agency|Missile Defense Agency]]. [http://www.mda.mil/mdalink/pdf/05news7.pdf Missile Defense Flight Test Conducted] {{webarchive|url=https://web.archive.org/web/20090713120043/http://www.mda.mil/mdalink/pdf/05news7.pdf |date=2009-07-13 }}. (PDF).2005-02-14.</ref>

2005年2月24日，美國國防部飛彈防禦處測試了[[神盾系統|神盾系統]]海基攔截效能，成功攔截靶彈。這是首次[[標準三型飛彈|標準三型飛彈]]（SM-3）攔截器成功運作也是第五次神盾系統實測成功。在2005年11月10日 USS Lake Erie（CG-70）號軍艦偵測追蹤並攔截到一枚兩節式靶彈，於該彈發射後兩分鐘。<ref>[[Missile_Defense_Agency|Missile Defense Agency]].[http://www.mda.mil/mdalink/pdf/05news0008.pdf Aegis Ballistic Missile Defense Flight Test Successful] {{webarchive|url=https://web.archive.org/web/20090713115954/http://www.mda.mil/mdalink/pdf/05news0008.pdf |date=2009-07-13 }}. (PDF). 2005-02-24.</ref>

2006年9月1日，[[陸基中途防禦系統|陸基中途防禦系統]]測試成功。一枚攔截器從范登堡空軍基地發射攔截從[[阿拉斯加|阿拉斯加]]發射的靶彈，地面支援人員都在[[科羅拉多泉|科羅拉多泉]]進行操作。本測試後飛彈防衛處指導官Trey Obering將軍說：「我們已經可以進行長程飛彈防禦系統的全程測試。」<ref>美國國防部. 在2007年2月份，美國開始接觸波蘭和[[捷克|捷克]]商量設立陸基中程飛彈防禦基地於這些國家的可能。根據捷克官方報告（將近67%捷克民眾不同意<ref>Citizens on U.S. Anti-Missile Radar Base in Czech Republic{{cite web |url=http://www.cvvm.cas.cz/upl/zpravy/100761s_pm80306.pdf |title=存档副本 |accessdate=2008-11-27 |deadurl=yes |archiveurl=https://web.archive.org/web/20081130053315/http://www.cvvm.cas.cz/upl/zpravy/100761s_pm80306.pdf |archivedate=2008-11-30 }}</ref>）建立飛彈防禦雷達以支援將要建在波蘭的飛彈防禦基地，該基地主要是為了歐洲防禦來自伊朗的長程飛彈。<ref>Center for Security Studies (CSS), ETH Zurich [http://www.isn.ethz.ch/pubs/ph/details.cfm?lng=en&id=29898 "US Missile Defense: A Strategic Challenge for Europe"], Daniel Möckli, ''CSS Analyses in Security Policy no. 12'', April 2007</ref>.
