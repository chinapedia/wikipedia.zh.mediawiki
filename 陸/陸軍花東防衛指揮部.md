'''陸軍花東防衛指揮部'''（{{lang-en|'''Hua-tung Defense Command'''}}；簡稱：「'''陸軍花東防衛部'''、'''陸軍花防部'''、'''花防部'''」，隊名：「'''正義部隊'''」），為[[中華民國陸軍|中華民國陸軍]]下轄[[防衛指揮部|防衛指揮部]]之一，係[[中華民國國軍|中華民國國軍]]駐守於[[東臺灣|東臺灣]]（[[花蓮縣|花蓮縣]]、[[臺東縣|臺東縣]]）{{Link-en|責任區|Area of responsibility|轄區}}的[[陸軍|陸軍]]部隊，總兵力約六千餘人，指揮官編階陸軍中將。戰時負責[[第二作戰區|第二作戰區]]的作戰指揮和[[軍事管制|軍事管制]]。前身為'''陸軍第三十二軍'''。

== 沿革 ==
=== 戡亂時期 ===
*1948年，在[[南京市_(中華民國)|南京市]]成立「'''陸軍第六十七軍'''」。
*1949年5月，與陸軍第十軍併編，改[[番號|番號]]為「'''陸軍第十軍'''」。
*1949年10月，移駐[[浙江省_(中華民國)|浙江]][[舟山|舟山]]。
*1949年11月，恢復「'''陸軍第六十七軍'''」番號，並且參加[[登步島戰役|登步島戰役]]。
*1950年，駐防[[臺灣|臺灣]][[桃園|桃園]]。
*1954年，在[[臺灣|臺灣]][[彰化|彰化]]整編為「'''陸軍第二軍'''」。
*1976年1月，更改番號為「'''陸軍第三十二軍'''」，移駐[[臺灣|臺灣]][[新竹|新竹]]。
*1989年，因「陸精七號案」移駐[[臺灣|臺灣]][[花蓮|花蓮]]，同時改編為「'''陸軍花東防衛司令部'''」。司令[[顏忠誠|顏忠誠]]中將，隸屬[[陸軍總司令部|陸軍總司令部]]。

=== 精進時期 ===
*2006年3月，因應《國防組織法》，更銜為「'''陸軍花東防衛指揮部'''」迄今。<ref>{{Cite web |url = http://army.mnd.gov.tw/Publish.aspx?cnid=824&p=9661&Level=2 |title = 陸軍花東防衛指揮部 |author =  |date =  |publisher = 中華民國陸軍司令部 |language = zh-tw |accessdate = 2012-06-16 |quote =  |archive-url = https://web.archive.org/web/20140304035049/http://army.mnd.gov.tw/Publish.aspx?cnid=824 |archive-date = 2014-03-04 |dead-url = yes }}</ref>
