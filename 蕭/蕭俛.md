'''蕭俛'''（{{bd|8世纪||842年|3月16日|catIdx=X蕭}}<ref name=BT18-1>[[:s:旧唐书/卷18上|《旧唐书》卷一十八上]]</ref>），字'''思謙'''，封'''徐国公'''，[[唐朝|唐朝]]官员，[[唐穆宗|唐穆宗]]年间任宰相。在传统史料中，他因正直而受到褒扬，也因做出错误决定导致朝廷对河北藩镇失控而受到谴责。

== 家庭背景 ==
蕭俛生年不详。他来自一个显赫的家族，是[[南朝梁|南朝梁]]的后代，祖父[[蕭華_(唐朝)|蕭華]]、曾祖[[萧嵩|萧嵩]]和萧嵩的叔伯曾祖父[[蕭瑀|蕭瑀]]都在唐朝当过宰相。<ref>[[:s:新唐书/卷101|《新唐书》卷一百零一]]</ref>蕭俛的父亲[[蕭恆|蕭恆]]任[[御史|御史]]。<ref>《新唐书》卷七十一</ref>蕭俛于[[唐德宗|唐德宗]][[貞元_(唐)|貞元]]七年（791年）擢[[進士|進士]]第<ref>[[徐松|徐松]]《[[登科記考|登科記考]]》卷十二·貞元七年</ref>，和同年中進士的[[皇甫鎛|皇甫鎛]]和[[令狐楚|令狐楚]]发展出深厚的友谊。<ref name=BT172>[[:s:旧唐书/卷172|《旧唐书》卷一百七十二]]</ref>

== 唐宪宗年间 ==
[[唐宪宗|唐宪宗]][[元和_(唐)|元和]]元年（806年）四月二十八日，蕭俛通过了「才識兼茂、明於體用」的贤良方正[[制科|制科]]<ref>[[徐松|徐松]]《[[登科記考|登科記考]]》卷十六·元和元年、[[:s:资治通鉴/卷237|《资治通鉴》卷二百三十七]]</ref>，为第五上等，<ref>[[:s:处分及第举人诏|:s:处分及第举人诏]]</ref>被任为[[右拾遺|右拾遺]]，<ref>《资治通鉴》卷二百三十七</ref>后升为[[右補闕|右補闕]]。当时他的朋友[[柳宗元|柳宗元]]被贬[[永州|永州]]司马，写信给他。<ref>[[:s:新唐书/卷168|《新唐书》卷一百六十八]]</ref>三年（808年），仆射、判度支[[裴均|裴均]]交结权幸欲求为宰相，其党羽意欲动摇宰相[[李吉甫|李吉甫]]的地位，也扬言直言极谏科中讥刺时政、忤犯权幸的言辞都是直指李吉甫。萧俛与其他谏官[[李约|李约]]、[[独孤郁|独孤郁]]、[[李正辞|李正辞]]秘密上疏陈奏，才使宪宗意解。<ref name=BT148>[[:s:旧唐书/卷148|《旧唐书》卷一百四十八]]</ref>六年（811年）正月，奉敕与谏议大夫[[孟简|孟简]]、给事中[[刘伯刍|刘伯刍]]、工部侍郎[[归登|归登]]等于丰泉寺翻译《大乘本生心地观音经》。<ref>[[:s:旧唐书/卷14|《旧唐书》卷一十四]]</ref><ref>[[:s:旧唐书/卷149|《旧唐书》卷一百四十九]]</ref><ref>[[:s:旧唐书/卷163|《旧唐书》卷一百六十三]]</ref><ref>[[:s:新唐书/卷160|《新唐书》卷一百六十]]</ref><ref>[[:s:答孟简萧俛等贺御制新译大乘本生心地观经序状|白居易《答孟简萧俛等贺御制新译大乘本生心地观经序状》]]</ref>同年被召充[[翰林学士|翰林学士]]。李吉甫任用萧俛却罢免同时被召为学士的李正辞，人们都疑惧。<ref>[[:s:新唐书/卷146|《新唐书》卷一百四十六]]</ref>除守起居舍人。<ref>[[:s:除萧俛起居舍人制|白居易《除萧俛起居舍人制》]]</ref>第二年，萧俛迁[[司封員外郎|司封員外郎]]，仍任翰林学士。九年（814年），任[[駕部郎中|駕部郎中]]，仍任翰林学士，知制诰。<ref name=BT172/>

当年，宪宗正在讨伐控制[[淮西节度使|彰义军]]（军部在今[[河南|河南]][[驻马店|驻马店]]）的叛将[[吴元济|吴元济]]，主战的宰相李吉甫刚刚去世。蕭俛的朋友[[張仲方|張仲方]]认为李吉甫主战太草率，反对谥他为“敬憲”。宪宗认为这是在批评自己，把張仲方贬官，但也把李吉甫的谥号改为“忠懿”。<ref name=BT148/>由于蕭俛与張仲方交好，宪宗也免了他的翰林学士，改任[[太僕少卿|太僕少卿]]。<ref name=BT172/>但十一年（816年）正月，蕭俛和翰林学士[[錢徽|錢徽]]等众多官员上疏请求罢兵。<ref name=BT170>[[:s:旧唐书/卷170|《旧唐书》卷一百七十]]</ref><ref name=NBT173>[[:s:新唐书/卷173|《新唐书》卷一百七十三]]</ref>为了警告这些官员，宪宗免了錢徽的翰林学士和蕭俛的知制诰职位，虽然让他们继续守本官，但疏远了他们。<ref>[[:s:旧唐书/卷15|《旧唐书》卷一十五]]</ref><ref name=ZZTJ239>[[:s:资治通鉴/卷239|《资治通鉴》卷二百三十九]]</ref>[[荆南节度使|荆南节度使]][[袁滋|袁滋]]本来揣度宪宗厌兵，自己上表入朝欲议罢平淮西事，途中听闻萧俛、钱徽因提议罢兵被黜，就改言必胜迎合宪宗之意，才得以回到本镇。<ref>[[:s:新唐书/卷151|《新唐书》卷一百五十一]]</ref>

十三年（818年），蕭俛的朋友皇甫鎛因为宪宗敛财而得宠，拜为宰相。他推荐令狐楚拜相，<ref name=NBT166>[[:s:新唐书/卷166|《新唐书》卷一百六十六]]</ref>而后二人又一齐推荐蕭俛为相。于是蕭俛也得宠于宪宗，被授朝议郎、飞骑尉，赐绯鱼袋，任[[御史中丞|御史中丞]]。<ref name=BT172/>同年九月，他上疏弹劾因千余百姓欠朝廷钱而将他们逮捕的宦官五坊使[[楊朝汶|楊朝汶]]，并得到宰相[[裴度|裴度]]、[[崔群|崔群]]的支持，十月，宪宗赐楊朝汶死，并释放被其逮捕的人。<ref name=BT170/><ref name=NBT173/><ref name=ZZTJ240>[[:s:资治通鉴/卷240|《资治通鉴》卷二百四十]]</ref>

十一月，考功员外郎[[李渤|李渤]]上奏称萧俛、段文昌、[[崔植|崔植]]等宰相不称职，不能劝阻宪宗游幸[[骊山|骊山]]，请求将三宰相与翰林学士[[杜元颖|杜元颖]]考中下。<ref>[[:s:考校京官奏|:s:考校京官奏]]</ref>该奏折被留中。<ref name=BT171>[[:s:旧唐书/卷171|《旧唐书》卷一百七十一]]</ref><ref>《册府元龟》作十五年十二月事。</ref>

十四年（819年）十月，萧俛奏应诸司诸军诸使公廨要求下放诸色本利钱等并设为定制，获准。<ref name=CFYG>《[[册府元龟|册府元龟]]》</ref>

== 唐穆宗、唐敬宗年间 ==
十五年（820年）正月，唐宪宗驾崩，唐穆宗继位。穆宗很快流放了他所不喜的皇甫鎛。谏议大夫[[李景俭|李景俭]]性矜诞，使酒纵气，言语间冒犯宰相，萧俛和翰林学士[[段文昌|段文昌]]诉于穆宗，李景俭被贬为[[建州_(唐朝)|建州]]刺史。<ref name=BT171/><ref>[[:s:新唐书/卷81|《新唐书》卷八十一]]</ref>闰正月，当穆宗思考用谁为相时，令狐楚推荐蕭俛。当时已袭徐国公的蕭俛被任为朝散大夫、守[[中書侍郎|中書侍郎]]，<ref name=BT16>[[:s:旧唐书/卷16|《旧唐书》卷一十六]]</ref>并和段文昌一同被授予[[同中書門下平章事|同中書門下平章事]]，为实质宰相，<ref name=NBT8>[[:s:新唐书/卷8|《新唐书》卷八]]</ref>赐金紫之服、紫金鱼袋。<ref>[[:s:授萧俛中书侍郎平章事制|:s:授萧俛中书侍郎平章事制]]</ref>此后穆宗想杀皇甫鎛，被蕭俛和宦官们阻止。<ref name=ZZTJ241>[[:s:资治通鉴/卷241|《资治通鉴》卷二百四十一]]</ref>人们以为令狐楚通过皇甫镈做宰相而驱逐[[裴度|裴度]]，群情共怒，但因萧俛的缘故，无人敢言。<ref name=NBT166/>二月，监察御史[[杨虞卿|杨虞卿]]以穆宗频繁出游上疏切，穆宗令中使宣付宰相，说：“杨虞卿所上疏切直可奖。”令狐楚、萧俛、段文昌在延英殿奏事时趁机以纳谏为贺。萧俛又加朝议大夫，<ref>[[:s:令狐楚等加阶制|元稹《令狐楚等加阶制》]]</ref>加勋，<ref>[[:s:萧俛等加勋制|元稹《萧俛等加勋制》]]</ref>加封爵，但将新增封地让给弟弟。<ref>[[:s:萧俛等加封爵制|元稹《萧俛等加封爵制》]]</ref>六月，赠萧华太保、萧恒吏部尚书，封萧俛母韦氏为岐国太夫人。<ref name=CFYG/>八月，转[[门下侍郎|门下侍郎]]。<ref>[[:s:授萧俛门下侍郎平章事制|:s:授萧俛门下侍郎平章事制]]</ref>十月，[[吐蕃|吐蕃]]入寇[[泾原|泾原]]，穆宗命中使以禁军援之，对宰相们说：“用兵有必胜之法吗？”萧俛对道：“兵是凶器，战是危事，圣主不得已而用之。以仁讨不仁，以义讨不义，先务招怀，不为掩袭。古之用兵，不绝人祀，不杀病人，不擒老人，不犯田稼。安人禁暴，是出师的上等。如救之甚于水火。所以王者之师，有征无战，此是必胜之道。如果因小忿就轻动干戈，使敌人怨结，师出无名，不但不胜，还是自危之道，本就应该深慎！”<ref>[[:s:对穆宗问兵法有必胜疏|:s:对穆宗问兵法有必胜疏]]</ref>穆宗以为然。<ref name=BT172/>谏议大夫[[郑覃|郑覃]]、[[崔玄亮|崔玄亮]]劝穆宗在吐蕃入寇之际稍减游乐，留心政事，穆宗不悦，问萧俛：“这些是什么人？”萧俛答：“谏官谏议大夫郑覃等。”穆宗意解，说臣下规劝自己的过失是忠。<ref>[[:s:旧唐书/卷173|《旧唐书》卷一百七十三]]</ref><ref>[[:s:新唐书/卷165|《新唐书》卷一百六十五]]</ref>

作为宰相的蕭俛处事谦逊而谨慎，善善恶恶。他为人孝顺，像普通人子一样小心侍奉母亲韦夫人。当穆宗命他为已故节度使[[王士真|王士真]]写碑文（可能是为了取悦刚归顺朝廷的其子[[王承元|王承元]]）时，他拒绝了，认为王士真生前不忠于朝廷。<ref>[[:s:辞撰王承宗先铭奏|:s:辞撰王承宗先铭奏]]</ref>但他和段文昌因为一项错误决策遭到谴责。他俩以为经过宪宗对藩镇的一系列作战，很多军阀已被击溃，其他军阀也同意遵奉朝廷号令了，于是提议朝廷密令各军队每年裁减8%的士兵。穆宗新继位，沉湎饮宴，没有看出问题，同意了。<ref name=BT16/>结果，被开除的士兵结为土匪，盧龍（军部在今[[北京|北京]]）、[[成德|成德]]（军部在今[[石家庄|石家庄]]）两镇在[[朱克融|朱克融]]和[[王廷凑|王廷凑]]带领下在[[长庆|长庆]]元年（821年）（此时蕭俛已不是宰相）反叛，匪兵们都加入了他们的队伍。由于匪兵们有经验，匆忙召集的官军没有经验，卢龙、成德的2万叛军竟然在和15万官军的对垒中取得了优势。<ref name=BT172/><ref>[[:s:资治通鉴/卷242|《资治通鉴》卷二百四十二]]</ref>

萧俛为官清直，太仆卿[[韦弘景|韦弘景]]常辅佐萧俛。<ref>[[:s:旧唐书/卷157|《旧唐书》卷一百五十七]]</ref><ref>[[:s:新唐书/卷116|《新唐书》卷一百一十六]]</ref>

[[裴垍|裴垍]]、[[韦贯之|韦贯之]]、[[李绛|李绛]]、[[崔群|崔群]]、萧俛都是补阙[[韦顗|韦顗]]的布衣旧交，先后都做了宰相，但朝廷典章之事多咨询韦顗，说：“我们五人的智力比不上一个韦公。”<ref>[[:s:新唐书/卷118|《新唐书》卷一百一十八]]</ref>

同年正月，穆宗飨太庙，礼毕，出朱雀门中路，日抱珥呈现五色，萧俛等率两省供奉官称贺于马前。<ref name=CFYG/>同月，[[西川节度使|西川节度使]][[王播|王播]]给穆宗进贡并贿赂当权宦官，想当宰相，段文昌也支持。蕭俛在延英殿坚决反对，认为王播奢侈纳贿且人品奸邪内外皆知，不可入相。由于穆宗不从，蕭俛累次上表辞相，优诏不许。当月王播到[[长安|长安]]后，蕭俛被罢相，任[[右僕射|右僕射]]，<ref name=NBT8/><ref name=ZZTJ241/>勋赐如故。<ref>[[:s:萧俛守尚书右仆射制|:s:萧俛守尚书右仆射制]]</ref>蕭俛自认为辅政日浅，升官太快，三上表章辞仆射，于是于二月被改任为守[[吏部尚書|吏部尚書]]。<ref>[[:s:萧俛除吏部尚书制|白居易《萧俛除吏部尚书制》]]</ref><ref>[[:s:萧俛守吏部尚书诏|:s:萧俛守吏部尚书诏]]</ref>萧俛一子得授官，萧俛却将此让给三从弟[[萧伸|萧伸]]，使其得任河中府参军。<ref>[[:s:萧俛一子回授三从弟伸制|白居易《萧俛一子回授三从弟伸制》]]</ref>吏部尚书要处理大量乏味的工作，蕭俛认为不适合自己，也拒绝了，坚持称病，乞求改任散官。七月，改[[兵部尚書|兵部尚書]]，<ref>[[:s:授李绛吏部尚书萧俛兵部尚书制|:s:授李绛吏部尚书萧俛兵部尚书制]]</ref>十月，进阶正议大夫。同年，大理卿[[刘元鼎|刘元鼎]]为盟会使，右司郎中[[刘师老|刘师老]]副之，萧俛奉诏与宰相们与尚书右仆射[[韩皋|韩皋]]、御史中丞[[牛僧孺|牛僧孺]]、吏部尚书李绛、户部尚书[[杨于陵|杨于陵]]、礼部尚书[[韦绶|韦绶]]、太常卿[[赵宗儒|赵宗儒]]、司农卿[[裴武|裴武]]、[[京兆尹|京兆尹]][[柳公绰|柳公绰]]、右金吾将军[[郭鏦|郭鏦]]及[[吐蕃|吐蕃]]使者论讷罗在京师西郊会盟。<ref>[[:s:新唐书/卷216下|《新唐书》卷二百一十六下]]</ref>第二年，他上表称病请求分司[[洛阳|洛阳]]，虽然没有被当即批准，但三月他就被任为[[太子少保|太子少保]]，<ref name=BT16/><ref>当时穆宗并未立敬宗为太子，太子少保只是荣衔。</ref>分司洛阳。<ref name=BT172/>

== 唐文宗、唐武宗年间 ==
[[唐敬宗|唐敬宗]][[宝历_(唐朝)|宝历]]元年（825年）闰七月，蕭俛在[[同州|同州]]（今[[陕西|陕西]][[渭南|渭南]]）刺史任上时，治下韩城县左神策军渚田内放牧马群牧小将刘兴裔擅自鞭打在神策军渚田内放牧的百姓王文秀等，被摄韩城令李元珪遣县吏擒送萧俛处。<ref name=CFYG/>二年（826年）三月，萧俛再任[[太子太保|太子太保]]，分司洛阳。同年，敬宗驾崩，弟[[唐文宗|唐文宗]]继位，<ref>[[:s:《资治通鉴》卷二百四十三|资治通鉴/卷243]]</ref>于[[大和_(唐朝)|大和]]元年（827年）正月召回蕭俛，二月改任为检校右仆射，兼礼部尚书，四月又改任为检校[[左僕射|左僕射]]兼[[太子少师|太子少师]]分司洛阳。<ref>[[:s:旧唐书/卷17上|《旧唐书》卷一十七上]]</ref><ref name=ditto>当时没有太子，所以太子太保、太子少师也只是荣衔。</ref>五年（831年）七月，时为太子少师分司、[[上柱国|上柱国]]、袭徐国公的蕭俛再次称病，文宗准他以[[银青光禄大夫|银青光禄大夫]]守左仆射[[致仕|致仕]]。<ref name=BT172/><ref>[[:s:旧唐书/卷17下|《旧唐书》卷一十七下]]</ref><ref>[[:s:答萧俛乞致仕诏|:s:答萧俛乞致仕诏]]</ref>

大约此时，蕭俛的母亲韦夫人去世，蕭俛为她[[服丧|服丧]]，有过于礼。丧期过后，文宗还想召他，但他称病拒绝了。他认为洛阳这个都市太繁忙，也不愿见客，于是在[[濟源|濟源]]山间养老。八年（834年），唐文宗立[[李永_(唐朝)|李永]]为太子，想召回蕭俛任李永的太子太傅，蕭俛让弟弟[[蕭傑|蕭傑]]奉表持诏书到长安表示谢绝，坚称自己有痼疾。文宗准许了，许他以[[太子太傅|太子太傅]]致仕。<ref>[[:s:答萧俛辞少师诏|:s:答萧俛辞少师诏]]</ref>[[开成|开成]]二年（837年），萧俛弟谏议大夫[[萧俶|萧俶]]被授[[楚州|楚州]]刺史，辞行之日，文宗对他称萧俛为先朝名相，赐萧俛诏书匹帛，要萧俶带到济源表达心意，希望萧俛可以来京。下诏赐绢三百匹，令萧俶宣示。<ref name=BT172/><ref>[[:s:赐萧俛手诏|:s:赐萧俛手诏]]</ref>[[唐武宗|唐武宗]][[会昌_(唐朝)|会昌]]二年（842年）二月，萧俛以[[太子太师|太子太师]]致仕身份去世。<ref name=BT18-1/>

== 评价 ==
*《旧唐书》赞曰：萧、李相才，致之外篇。<ref name=BT172/>
*《新唐书·藩镇宣武彰义泽潞传》指萧俛为导致唐朝藩镇之乱的庸佐。<ref>[[:s:新唐书/卷214|《新唐书》卷二百一十四]]</ref>
*[[北宋|北宋]]宰相[[王安石|王安石]]认为萧俛裁军是正确而必要的，酿成藩镇复叛是因为措施不得法。<ref>[[:s:宋史/卷193|《宋史》卷一百九十三]]</ref>
*[[南宋|南宋]]秘书郎[[唐文若|唐文若]]《文思箴》：萧俛去兵，祸乱乃萌。<ref>[[:s:宋史/卷388|《宋史》卷三百八十八]]</ref>
