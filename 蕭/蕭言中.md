'''蕭言中'''{{BD|1965年|7月6日|catIdx=X}}是[[台灣|台灣]]的[[漫畫家|漫畫家]]、[[舞台劇|舞台劇]][[編劇|編劇]]、[[節目主持人|節目主持人]]，[[法文|法文]]名'''Loïc'''。[[血型|血型]]O型。[[台北縣私立復興高級商工職業學校|復興商工]]美工科畢業。
为男性漫画家，1965年出生，<ref>{{Cite news|url=http://comic.qq.com/a/20100124/000006.htm|title=个人介绍——萧言中|language=zh-cn|publisher=腾讯网|author=|date=2006-02-08|accessdate=2015-02-03}}</ref>自1984年开始发表漫画。<ref>{{Cite news|url=http://kid.qq.com/a/20120419/000044.htm|title=台湾著名漫画大师萧言中个人简介|language=zh-cn|publisher=腾讯网|author=|date=2006-02-08|accessdate=2015-02-03}}</ref>其作品《笨贼一箩筐》被[[朱德庸|朱德庸]]、[[敖幼祥|敖幼祥]]一起推荐，<ref>{{Cite news|url=http://comic.qq.com/a/20100124/000005.htm|title=朱德庸敖幼祥联袂推荐《笨贼一箩筐》|language=zh-cn|publisher=腾讯网|author=|date=2006-02-08|accessdate=2015-02-03}}</ref>
也是继蔡志忠、朱德庸之后第三个到[[北京|北京]]出版的台湾漫画家。<ref>{{Cite news|url=http://beijing.qianlong.com/3825/2006/07/07/2601@3290338.htm|title=漫画家萧言中带着《笨贼一箩筐》来北京|language=zh-cn|publisher=千龙网|author=|date=2006-02-08|accessdate=2015-02-03}}</ref>与敖幼祥、[[蔡志忠|蔡志忠]]、朱德庸并齐“台湾四大漫画才子”。<ref>{{Cite news|url=http://shcci.eastday.com/eastday/shwc/r/20120817/u1a6792129.html|title=萧言中：最不务正业的漫画家[图]|language=zh-cn|publisher=东方文创 |author=|date=2006-02-08|accessdate=2015-02-03}}</ref><ref>{{Cite news|url=http://www.stardaily.com.cn/3.1/1405/22/448389.html|title=内容不拷贝漫画 话剧《笨贼一箩筐》昨天首演|language=zh-cn|publisher=北京娱乐信报|author=|date=2006-02-08|accessdate=2015-02-03}}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>曾参演过[[屏风表演班|屏风表演班]]的《莎姆雷特》、《三人行不行》等[[舞台剧|舞台剧]]。<ref>{{Cite news|url=http://beijing.qianlong.com/3825/2006/07/07/2601@3290338.htm|title=漫画家萧言中带着《笨贼一箩筐》来北京|language=zh-cn|publisher=千龙网|author=|date=2006-02-08|accessdate=2015-02-03}}</ref>被叫“笨贼之父”<ref>{{Cite news|url=http://enjoy.eastday.com/e/20140522/u1a8104628.html|title=改编自王牌漫画 《笨贼一箩筐》网络剧将诞生|language=zh-cn|publisher=东方网 |author=|date=2006-02-08|accessdate=2015-02-03}}</ref>。被读者冠以“单幅漫画第一人”<ref>{{Cite news|url=http://ent.163.com/14/0522/15/9SS20BCO00031GVS.html|title=网络剧《笨贼一箩筐》将拍 跨界改编王牌漫画|language=zh-cn|publisher=网易 |author=|date=2006-02-08|accessdate=2015-02-03}}</ref>。

== 人物 ==
1985年蕭言中於[[時報文化|時報文化]]出版漫畫作品《童話短路》、《Rocky弱雞》後累積出版三十餘部的漫畫創作，其中《童話短路》曾創下五十萬本銷售紀錄<ref>{{Cite web |url=http://www.books.com.tw/exep/prod/booksfile.php?item=0010348456 |title=博客來書籍館 |accessdate=2008-09-14 |archiveurl=https://web.archive.org/web/20151227020452/http://www.books.com.tw/exep/prod/booksfile.php?item=0010348456 |archivedate=2015-12-27 |deadurl=yes }}</ref>。蕭言中多元涉足各種領域，除了舞台劇、主持外，也曾和歌手[[陳昇|陳昇]]合唱3首歌，並曾擔任陳昇[[跨年|跨年]][[演唱會|演唱會]]的[[導演|導演]]。[[曹蘭|曹蘭]]歌曲〈白雪壞公主〉由蕭言中填詞、[[紀宏仁|紀宏仁]]作曲。

== 簡歷 ==
*1965年出生，學生時期常因為[[塗鴉|塗鴉]]而遭到責罰，曾將人物塗成綠色後辯解其為[[外星人|外星人]]。後來就讀復興商工美工科畢業。
*1984年參加《[[中國時報|中國時報]]》[[全國漫畫大擂台|全國漫畫大擂台]]，受到《中國時報》的看重。
*1985年出道，於[[時報文化|時報文化]]出版漫畫作品《童話短路》，以輕髮幽默的風格創造出銷售佳績，曾在採訪中提及[[童話|童話]]故事自幼耳熟能詳，「將童話中的任何一個點，比如時間、人物某方面的性格稍做改變，整個故事就不一樣了！」。
*1987年成立「糖果屋默劇團」，以[[默劇|默劇]]的方式表演[[兒童話劇|兒童話劇]]，編導有作品《魔鏡的蒼蠅日記》、《白雪壞公主》、《聖誕老公公浩劫餘生記》等多部舞台劇。
*1994年和陳昇合作進[[錄音室|錄音室]]錄製歌曲《Last Oder》，收錄於陳昇《風箏》專輯。
*2005年舉行《短路誕生20年》漫畫原稿展，於全台灣巡迴。
