'''臺中市立圖書館'''（簡稱'''中市圖'''）為臺中市的市立公共圖書館系統，轄有臺中市44個分館，共392萬冊（件），[[臺中市政府文化局|臺中市政府文化局]]為其主管機關。

成立之初以原北屯四張犁分館作為[[臺中市立圖書館總館|總館]]，未來待設在[[水湳經貿園區|水湳經貿園區]]的總館建成後再遷入，預計2022年落成。

== 歷史 ==
2010年12月25日，[[臺中縣|臺中縣]]、[[臺中市_(省轄市)|市]][[2010年中華民國縣市改制直轄市|合併升格改制為直轄市]]，原臺中市文化局和原臺中縣文化局所管理的縣市級圖書館，以及原臺中市各區公所和原臺中縣各鄉鎮市公所管理的鄉鎮市區級圖書館，皆交由[[臺中市政府文化局|臺中市政府文化局]]圖書資料科統一管理。2014年12月25日《地方制度法》修正後，和平區再次改制為具有公法人地位的地方自治團體<ref>[http://www.libertytimes.com.tw/2014/new/jan/15/today-p12.htm 直轄市原民區 可選區長、區代表] {{webarchive|url=https://web.archive.org/web/20140217182535/http://www.libertytimes.com.tw/2014/new/jan/15/today-p12.htm |date=2014-02-17 }}，自由時報，2014-1-15</ref>，臺中市和平區圖書館改由區公所管理。

2016年3月1日，合併各區圖書館<ref>和平區圖書館仍由和平區公所管理編制</ref>與[[大墩文化中心|大墩文化中心]]圖書館、[[葫蘆墩文化中心|葫蘆墩文化中心]]圖書館，正式成立「'''臺中市立圖書館'''」，下設8個課室及43個分館，由原北屯四張犁分館暫為總館。<ref>[http://www.library.taichung.gov.tw/public/content/index-1.asp?Parser=1,20,189,45 臺中市立圖書館-本館簡介-簡 介]</ref>同年10月18日，北區分館新館落成啟用。<ref>[http://www.taichung.gov.tw/ct.asp?xItem=1756021&ctNode=24068&mp=1001D 中市圖書館北區分館啟用]，台中市新聞局，2016-10-18</ref>
2018年4月29日，西屯西溪分館落成啟用。同年9月20日，李科永紀念圖書分館落成啟用。

臺中市立圖書館成立之初以2003年完工啟用的北屯四張犁分館作為總館。

位於西屯區[[水湳經貿園區|水湳經貿園區]]的新總館，興建經費約新臺幣10.28億，預計2022年完工。啟用後，預計藏書將達到70萬冊，將是臺中市最大型的市立公共圖書館，也是繼[[國立公共資訊圖書館|國立公共資訊圖書館]]之後，臺中市第二大的公共[[圖書館|圖書館]]，竣工後的臺中市立圖書館樓高7層，另包括地下2層<ref>{{Cite web|url = http://rdnet.taichung.gov.tw/ct.asp?xItem=69900&ctNode=3007&mp=115030|title = 委託研究報告-臺中市立圖書館總館空間規劃|accessdate = 2014-10-14|author = 印紀聯合建築師事務所}}</ref><ref>{{Cite web|url = https://tw.news.yahoo.com/%E5%8F%B0%E4%B8%AD%E5%A4%A7%E5%AE%85%E9%96%80%E7%89%B9%E5%8D%80-%E6%8E%A5%E8%BB%8C%E5%9C%8B%E9%9A%9B%E9%83%BD%E5%B8%82-213000937.html|title = 台中大宅門特區 接軌國際都市|accessdate = 2014-10-14|author = 王妙琴|publisher = 工商時報|date = 2013-02-27}}</ref>。
