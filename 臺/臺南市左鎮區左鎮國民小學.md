'''臺南市左鎮區左鎮國民小學'''位於臺灣[[臺南市|臺南市]][[左鎮區|左鎮區]]，其校史可追溯至1920年4月1日設立的「大目降公學校崗仔林分教場菜藔分離教室」{{r|志2}}。

== 沿革 ==
左鎮區在日治時期直到1918年3月才在[[外新化南里|外新化南里]]崗仔林庄設立「大目降公學校崗仔林分教場」，為當時左鎮地區唯一的官辦小學，後來因為菜寮居民覺得學生前往崗仔林路途遙遠，遂爭取設立分離教室，並計畫設在菜寮仕紳郭先助的房子前{{r|志2}}。而後四年制的「大目降公學校崗仔林分教場菜藔分離教室」於1920年4月1日正式成立，最初入學學生有60多人，大多是十幾歲的少年{{r|志2}}。校名在這之後因為崗仔林分教場在1921年4月獨立成「崗仔林公學校」與校址在1922年因仕紳[[羅來受|羅來受]]提議下從菜寮搬到拔馬之故，改稱「崗仔林公學校左鎮分教場」，不久又在1924年獨立為「左鎮公學校」，同時崗仔林公學校因學生數不足反而變成「左鎮公學校崗仔林分教場」{{r|志2}}。1941年4月1日，因臺灣總督府之政策，改稱「左鎮國民學校」{{r|志2}}。

二次大戰後，左鎮國民學校在1946年5月改稱「左鎮第一國民學校」，次年（1947年）3月又改回「左鎮國民學校」{{r|志2}}。而在1947年時，該校有14個班級，學童842人，並增設澄山分校{{r|志2}}。1963年增設光榮分校，次年（1964年）1月18日因[[1964年白河地震|白河大地震]]導致校舍重創，時任校長溫禎祥爭取到經費後於同年重建校舍{{r|志2}}。

1968年改制為左鎮國民小學，而後在1970年代校園進行大幅改建，包括拆除日治時期的教室與整建球場、操場等{{r|志2}}。1980年代因為左鎮地區學子外流，部分學校降為分校，如澄山國小與岡林國小就先後成為左鎮國小的分校，而後澄山分校在1996年改隸光榮國小（後在2006年併入光榮國小），岡林國小則在數次變動後於2006年併入左鎮國小{{r|志2}}。

== 國樂隊 ==
左鎮國小國樂隊於1994年2月成立，曾榮獲83－86學年度臺南縣音樂比賽絲竹室內樂合奏第一名與臺灣省音樂比賽甲等等獎項{{r|志2}}。而由於該校學生少，故採取讓學生只要生上三年級便成為國樂隊成員的方式以免隊員不足，同時樂器由校方提供以減輕學生負擔{{r|志2}}。

== 知名校友 ==
*[[陳春木|陳春木]]：左鎮地區重要的化石採集者與文史工作者，有「化石爺爺」之稱；2000年成為該校傑出校友。
