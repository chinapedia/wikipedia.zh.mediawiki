'''臺中刑務所演武場'''為[[臺中市|臺中市]]僅存[[臺灣日治時期|臺灣日治時期]]武術道館式建築，建於1937年，當初是供[[臺中刑務所|臺中刑務所]]（今[[臺中監獄|臺中監獄]]）司獄官練武之用。

== 歷史 ==
*1924年 第一代台中刑務所修道館建成。
*1937年8月16日第二代台中刑務所演武場（今址）落成。
*1945年[[中華民國政府|中華民國政府]]接收[[日本帝國|日本]][[台灣總督府|台灣總督府]]轄區後，演武場轉為[[中華民國法務部|法務部]][[臺中監獄|臺中監獄]]，週邊全建為[[眷村|眷村]]。
*2004年2月20日，被[[臺中市文化局|臺中市文化局]]登錄為[[台灣歷史建築|歷史建築]]。
*2006年11月15日發生火災，木構建築部分焚毀，隨後臺中市文化局成立緊急應變小組，開始著手災後清理與記錄工作{{r|陳}}，之後進行修復工程。 
*2010年11月，主體與附屬建築修復完成，移交給[[臺中市警察局|臺中市警察局]]使用並開放參觀。
*2012年3月，[[臺中市政府|臺中市政府]]委託[[財團法人|財團法人]][[道禾教育基金會|道禾教育基金會]]經營，定名為「'''道禾六藝文化館'''」；隔年5月正式開館啟用。

== 所屬建築 ==
*惟和館（主館）：和洋折衷磚造的日式建築，屋頂採[[歇山頂|入母屋]]式寺院形式；原為[[柔道|柔道]]、[[劍道|劍道]]使用空間。
*心行館（附屬建築）：位於主館左側，為傳統日式木構建築。附設「小書房」經營經典、教育、文學、藝術出版品、台中主題文創作品、六藝文化及具紀念性產品
*傳習館（附屬建築）：位於主館後方，為之後增建的空間。 
*大樹下劇場（庭院）：以老[[榕樹|榕樹]]為主體的戶外展場。

== 臺中地區武道建築 ==
;[[武德殿|武德殿]]
*第一代：「[[台中武德殿|台中武德殿]]」，全名「[https://web.archive.org/web/20060719150631/http://www.dnbk.org/ 大日本武德會]臺中支部武德殿」。建於1913年的木造建築，1931年因改建而被拆遷到員林，成為[[員林鎮|員林]]「金鳳山[[員林寺|員林寺]]」。1990年代因改建新寺，被解體收藏於[[彰化縣|彰化縣]][[花壇鄉|花壇鄉]][[台灣民俗村|台灣民俗村]]。
*第二代：建於1931年，原址為今臺中市[[三民路_(台中市)|三民路一段]]158號，1997年被[[臺中市政府|臺中市政府]]拆建為辦公大樓（今[[臺中市政府地政局|臺中市政府地政局]]本部）。

;演武場
*第一代：「台中刑務所修道館」，建於1924年，使用1942年後已無資料。
*第二代：「台中刑務所演武場」，建於1937年。

;民間
*台中武德館：建於1957年，位於臺中市興中街90號（原為[[日本|日本]][[佛寺|佛寺]]－[[中尊寺_(台中市)|中尊寺]]庭院），目前為臺中市[[柔道|柔道]]聯誼會使用。
