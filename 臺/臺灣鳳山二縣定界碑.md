'''臺灣鳳山二縣定界碑'''位於臺灣高雄市[[茄萣區|茄萣區]]白砂崙萬福宮後方，是清朝[[臺灣府|臺灣府]]底下[[臺灣縣_(1684年-1887年)|臺灣縣]]與[[鳳山縣_(臺灣)|鳳山縣]]的界碑{{r|文2}}。該石碑在2012年7月4日被公告為高雄市的一般古物{{r|文1}}，後來在2013年1月21日改列為重要古物{{r|文2}}。

該石碑立於清乾隆二十八年（1763年）八月，上刻當時鳳山縣知縣王瑛的告示，明定臺灣縣與鳳山縣之間南北以二層行溪（今[[二仁溪|二仁溪]]）為界，東西以新開港為界，是為臺灣現存最早的地理行政區界碑{{r|文2}}{{r|解}}。

== 沿革 ==
雍正九年（1731年）臺灣縣與鳳山縣調整行政區劃，位於二層行溪南邊的「鯤身頭」（今白砂崙、白雲里）及「茄藤仔」（今頂茄萣、下茄萣）從臺灣縣改歸鳳山縣管轄，後來在乾隆廿四年（1759年）因新開港的疏濬導致兩縣縣界產生紛爭{{r|文1}}。之後在臺灣縣知縣陶紹景及鳳山縣知縣王瑛會勘後，於乾隆廿八年（1763年）立此界碑定界{{r|文1}}。

== 碑文 ==
{{Quote|
鳳山縣正堂王，為題案歸割事。<br/>乾隆二十八年七月二十九日，詳蒙本府憲正堂余，轉蒙本道憲覺羅四，檄飭臺、鳳會勘兩邑疆界；南北以二層行溪為界，東西以新開港定界。溪之南、港以東，草仔藔、蟧里塭等處田地糧課均歸鳳邑管轄。立石定界，永遠遵行，毋得紛爭滋訟。須至碑者。
<br/>
乾隆二十八年八月 日立，歸[[文賢里_(堡里)|文賢里]]管保看守，無致損壞。
}}

== 碑體 ==
該石碑的材質是[[花崗岩|花崗岩]]，高193cm，寬42.5cm，厚19cm{{r|解}}。
