'''臺北市立忠孝國民中學'''，簡稱'''忠孝國中'''，是[[臺北市|臺北市]][[大同區|大同區]]一間市立[[國民中學|國民中學]]，建校於1970年。校內設有太陽能光電設備、交安教室、智慧電力網路系統等設施，並發展數理資優課程。
[[File:Taipei_Municipal_Zhongxiao_Junior_High_School_main_entrance_20171111.jpg|thumb]]

== 校史 ==
忠孝國中原址為[[德意志帝國|德意志帝國]]駐台領事館、[[臺灣鐵路管理局|臺灣鐵路管理局]]維修場，民國58年7月籌備處成立，首任校長張良彬奉派擔任籌備工作，依法辦理校地申請調撥用暨學校規劃事宜。民國59年8月1日正式招生。民國69年成立資優班，招收校內數理資優學生。

== 建築與校園環境 ==
迄64年6月全校校舍陸續完工，暨興建校舍三棟，司令台一座，操場及跑道，籃球場及排球場等。民國68年興建學生活動中心，提供師生及社區民眾活動場所。
目前有:皆為4層樓+地下1層的校舍三棟、活動中心一棟(內含舞台、室內籃球場、2樓看台、男女盥洗室、舞蹈教室、桌球室...)、司令台(五育台)1座、升旗台1座(含3支旗桿)、籃球場、跳遠跑道與沙坑、操場(2016年修建，顏色由全紅變成由內而外分別為:紅-藍-綠-藍-紅。)、操場中的草地(為少數保留草地未將操場中央建設成球場的學校)、電梯2座、烤肉台、1樓平面停車場、生態池、性別友善廁所、無障礙廁所、親子廁所、集哺乳室等

== 地理特色 ==
台北市立忠孝國中位於台北市大同區西寧北路32號，鄰近大稻埕、迪化街、忠孝橋、捷運北門站、北門(承恩門)、西門町、台北地下街、台北車站、中興醫院等。
