'''臺灣臺東農田水利會'''為隸屬於[[行政院農委會|行政院農委會]]旗下並負責管理[[臺灣|臺灣]][[臺東縣|臺東縣]]境內之[[灌溉|灌溉]][[水利|水利]]設施的[[公法人|公法人]]行政單位，主要進行台東縣各大農業灌溉水利設施之管理、營運、維護、修繕等相關業務。<ref name="台灣台東農田水利會" >{{cite web |url=http://www.ttia.gov.tw/page1.php|title=台灣台東農田水利會 |  accessdate=2015-02-05}}</ref>

=== 公共埤圳組合時期 ===

* 台東農田水利會最初創立於台灣[[日治時期|日治時期]]1918年（大正7年）1月16日，當時依據[[台灣總督|台灣總督]][[兒玉源太郎|兒玉源太郎]]在1901年（明治34年）7月4日頒布之'''「[[台灣公共埤圳規則|台灣公共埤圳規則]]」'''而建立'''「卑南[[公共埤圳組合|公共埤圳組合]]」'''。「卑南公共埤圳組合」隸屬於[[日本|日本]][[台灣總督府|台灣總督府]][[台東廳|台東廳]]，並管轄同於日治時期所建設的[[卑南大圳|卑南大圳]]。<ref name="台灣台東農田水利會" /><ref name="國立台灣大學 圖書館藏系統 台灣省台東農田水利會會誌 / [臺灣省臺東農田水利會會誌編審委員會編]" >{{cite web |url=http://tulips.ntu.edu.tw/record=b3424666*cht|title=國立台灣大學 圖書館藏系統 台灣省台東農田水利會會誌 / [臺灣省臺東農田水利會會誌編審委員會編] |  accessdate=1990}}</ref>

=== 水利組合時期 ===

* 1921年（大正10年）12月28日，日本台灣總督府頒布'''[[台灣水利組合令|台灣水利組合令]]'''，並於隔年1922年（大正11年）公告水利組合令實施細則，「卑南公共埤圳組合」則在1925年（大正14年）11月25日依據該細則改組為'''「卑南[[水利組合|水利組合]]」'''。<ref name="國立台灣大學 圖書館藏系統 台灣省台東農田水利會會誌 / [臺灣省臺東農田水利會會誌編審委員會編]" />
* 1940年（昭和15年）12月29日，將南方的'''「[[知本|知本]][[水利組合|水利組合]]」'''合併至「卑南水利組合」之中，並在1941年（昭和16年）將仍為私營階段的[[利家圳|利家圳]]接收管理，當時首長為[[藤田淳敦|藤田淳敦]]。<ref name="國立台灣大學 圖書館藏系統 台灣省台東農田水利會會誌 / [臺灣省臺東農田水利會會誌編審委員會編]" />
* 1943年（昭和18年）11月30日，「卑南水利組合」正式改稱為'''「台東水利組合」'''。<ref name="國立台灣大學 圖書館藏系統 台灣省台東農田水利會會誌 / [臺灣省臺東農田水利會會誌編審委員會編]" />

=== 農田水利協會時期 ===

* 1945年（民國34年），[[臺灣戰後時期|中華民國接管臺灣後]]隔年 1946年（民國35年）11月30日，「台東水利組合」奉當時[[台灣省|台灣省]][[行政長官公署|行政長官公署]]之令改名為'''「臺灣省臺東農田水利協會」'''並針對第[[二次世界大戰|二次世界大戰]]戰後日本[[殖民|殖民]]政府所遺留下荒廢之埤圳進行修復通水，其中，受損輕微之圳道，由農田水利協會指導當地農民進行修復，而受損嚴重之圳道，則陳請中央政府撥款修建。<ref name="國立台灣大學 圖書館藏系統 台灣省台東農田水利會會誌 / [臺灣省臺東農田水利會會誌編審委員會編]" />

=== 水利委員會時期 ===

* 1948年（民國37年）2月20日，依據當時[[台灣省政府|台灣省政府]]頒佈之[[「臺灣省各地農田水利委員會設置辦法」|「臺灣省各地農田水利委員會設置辦法」]]再度改名為'''「臺東水利委員會」'''。<ref name="國立台灣大學 圖書館藏系統 台灣省台東農田水利會會誌 / [臺灣省臺東農田水利會會誌編審委員會編]" />

=== 農田水利會時期 ===

* 1956年（民國45年）7月，全台各地水利委員會均奉台灣省政府之令進行改組，並在同年11月2日與'''「[[關山水利委員會|關山水利委員會]]」'''以及'''「[[關山水利整理委員會|關山水利整理委員會]]」'''合併，並且改稱為'''「臺灣省台東農田水利會」'''。<ref name="國立台灣大學 圖書館藏系統 台灣省台東農田水利會會誌 / [臺灣省臺東農田水利會會誌編審委員會編]" />
* 1970年（民國59年）5月18日，「[[新港農田水利會|新港農田水利會]]」正式併入「臺灣省台東農田水利會」，並且管理臺東縣全縣之灌溉水利設施。
* 1975年（民國64年）1月1日，依據台灣省政府頒佈之'''「台灣省加速農村建設時期健全農田水利會實施要點」'''之規定進行改組，會長改由中央政府指派，原有之會員代表在任期結束後暫停改選，並在此期間推動[[卑南上圳|卑南上圳]]之興建計畫。<ref name="國立台灣大學 圖書館藏系統 台灣省台東農田水利會會誌 / [臺灣省臺東農田水利會會誌編審委員會編]" />
* 1981年（民國70年）12月31日「台灣省加速農村建設時期健全農田水利會實施要點」結束實施，恢復舊有體制，並持續進行灌溉水利設施之更新與改善，包含區域排水改善工程，輪灌改善工程，實施農地重劃，並且先後改建了旗下多個工作站的辦公廳。<ref name="國立台灣大學 圖書館藏系統 台灣省台東農田水利會會誌 / [臺灣省臺東農田水利會會誌編審委員會編]" />
* 1982年（民國71年）4月，重新開始辦理會員代表選舉，並於同年5月20日舉辦第六任會長選舉。<ref name="國立台灣大學 圖書館藏系統 台灣省台東農田水利會會誌 / [臺灣省臺東農田水利會會誌編審委員會編]" />
* 2002年（民國90年）6月20日，[[總統|總統]]華總(一)義字第9000120780號令公佈修正農田水利會組織通則，會長及會務委員均由會員直接投票選舉之，任期四年，會長連選得連任一次，會務委員連選得連任。當時台東農田水利會第一任及第二任直選會長為[[梁俊良|梁俊良]]<ref name="台灣台東農田水利會" /><ref name="國立台灣大學 圖書館藏系統 台灣省台東農田水利會會誌 / [臺灣省臺東農田水利會會誌編審委員會編]" />
* 2013年（民國103年）1月，'''「臺灣省台東農田水利會」'''正式改稱為'''「臺灣臺東農田水利會」'''，第三任及第四任直選會長為[[羅應鑑|羅應鑑]]。<ref name="台灣台東農田水利會" />

* '''水利小組'''

水利小組為該工作站管區內的農田水利會會員義務勞動所分列的組別，每年利用第二期作水稻收穫後之農閒期，由各工作站發動小組義勞辦理各小組轄區內之小給水小排水路之浚喋補修伐草等工作。<ref name="台灣台東農田水利會 水利小組" >{{cite web |url=http://www.ttia.gov.tw/page8-4-5.php|title=台灣台東農田水利會 水利小組 |  accessdate=2015-02-05}}</ref>

== 管轄設施 ==

{{main|臺東縣圳道}}

台東農田水利會下轄多個工作站針對台東地區各灌溉水圳系統進行管理與維護，其中以[[卑南大圳|卑南大圳]]，[[卑南上圳|卑南上圳]]，[[鹿野圳|鹿野圳]]，[[關山大圳|關山大圳]]，[[池上圳|池上圳]]，[[長濱大圳|長濱大圳]]等水圳為主要圳系，以及其他小型灌溉水圳系統，總圳路幹支分線長823,023公尺，小給水路1,408,785公尺。<ref name="台灣台東農田水利會" /><ref name="農田水利入口網 台灣台東農田水利會" >{{cite web |url=http://doie.coa.gov.tw/about/about.asp?md=a&uc=ttia |title=農田水利入口網 台灣台東農田水利會 |  accessdate=2015-02-05}}</ref>

其中，分別在卑南上圳與鹿野圳設有一座調整池，分別為十股調整池與馬背調整池。引水來源多屬於地面河川水流，另有救旱用地下水抽水站9口，總抽水量0.395C.M.S。<ref name="台灣台東農田水利會" /><ref name="農田水利入口網 台灣台東農田水利會" />

另目前有[[尚武圳|尚武圳]]，[[利吉圳|利吉圳]]兩座水圳當前處於停灌廢棄狀態。<ref name="台灣台東農田水利會" />

台東農田水利會總事業區達14,318公頃，其中，發源自[[中央山脈|中央山脈]]屬於[[卑南溪|卑南溪]][[流域|流域]]的灌溉區達9,619公頃，屬於[[秀姑巒溪|秀姑巒溪]]流域的灌溉區為143公頃，其他[[太平溪|太平溪]]、[[利嘉溪|利嘉溪]]、[[知本溪|知本溪]]、[[太麻里溪|太麻里溪]]、[[朝庸溪|朝庸溪]]等引水自上述溪流的灌區面積為1,409公頃，發源於[[海岸山脈|海岸山脈]]者有[[水母溪|水母丁溪]]、[[馬海溪|馬海溪]]、[[竹湖溪|竹湖溪]]、[[宜灣溪|宜灣溪]] 、[[都威溪|都威溪]] 、[[富家溪|富家溪]]、 [[成功溪|成功溪]]、 [[跋便溪|跋便溪]]、 [[馬武窟溪|馬武窟溪]] 、[[都蘭溪|都蘭溪]]等引水自上述溪流之灌區面積有3,147公頃，並且，其灌溉區面積分散，多屬梯田。<ref name="台灣台東農田水利會" /><ref name="農田水利入口網 台灣台東農田水利會" />
