'''臺中市尚未列入保護的文化資產列表'''，是羅列統計[[臺中市|臺中市]]境內尚未依《[[文化資產保存法|文化資產保存法]]》列入保護的文化資產潛力點（以1945年前就已存在的建物為主）。尚待[[文化部文化資產局|文化部文化資產局]]或[[臺中市文化資產處|臺中市文化資產處]]邀請專家學者、相關單位再行履勘，評估相關文化資產價值後，依文化資產保存法相關規定辦理。

== 官署類 ==
*[[台中高農|台中高農]]第二校區倉庫：[[西屯區|西屯區]]惠中路上（鄰後備司令部旁...疑似日治時期僅存的台中州立農業學校建物）。
*[[原台中測候所|原台中測候所]]：[[台中公園|台中公園]]內的[[中央氣象局|中央氣象局]]台中氣象站。
*水質化驗所：今水源地文化休閒中心，位於[[北區_(台中市)|北區]]體二用地內（[[國立臺灣體育運動大學|國立臺灣體育運動大學]]後方，有拆除危機）。
*第一唧筒室：位於[[台灣自來水公司|台灣自來水公司]]內，建於1922年。
*<s>[[太平車站|太平車站]]：位於[[太平區_(台中市)|太平區]]的糖鐵[[中南線|中南線]]車站，2015年風災已全倒。</s>
*[[臺中孔廟|臺中孔廟]]：雖建於1976年，但同年份建成的文英館亦能列為古蹟，能列為保存更好。
*[[臺中市忠烈祠|臺中市忠烈祠]]：建於1970年，但亦有列為保存的價值。
*[[台中體育場|台中體育場]]：入口處立面的[[鑲嵌畫|鑲嵌畫]]壁畫，是1961年7月由臺灣美術家[[顏水龍|顏水龍]]創作之大作品。
*[[原臺灣省菸酒公賣局菸葉試驗所|原臺灣省菸酒公賣局菸葉試驗所]]辦公室：菸葉試驗所已改為[[臺中軟體園區|臺中軟體園區]]，此為僅存的日治時期建物。

== 商業類 ==
*[[梨山賓館|梨山賓館]]：建於1965年，位於[[梨山風景區|梨山風景區]]內
*[[台中協贊信用組合總社|台中協贊信用組合總社]]：建於1911年，位於[[中區_(台中市)|中區]]中山路202號，今台中市第二信用合作社總社。
*[[有限責任台中信用組合總社|有限責任台中信用組合總社]]：建於1938年，位於[[中區_(台中市)|中區]]市府路59號，今[[三信商業銀行|三信商業銀行]]總行
*[[第二市場|新富市場六角樓]]：位於[[中區_(台中市)|中區]]的中正路與[[三民路_(台中市)|三民路]]口第二市場內，僅存的日治時期市場建物。
*[[文青果組合會社|文青果組合會社]]：位於[[東區_(台中市)|東區]][[復興路_(台中市)|復興路]]四段的台中後站旁。
*何藥房：位於[[中區_(台中市)|中區]]成功路253號，日治時期大正年間的藥房建物。
*金振源：位於[[中區_(台中市)|中區]]繼光街129號（近[[成功路_(台中市區)|成功路]]），大正紅磚洋樓式建築。
*東平診所：位於[[中區_(台中市)|中區]][[中正路_(台中市中區)|中正路]]311號，與[[中華路_(台中市中區)|中華路]]交叉口，建於1957年，曾為東平戲院。
*[[中央書局|株式會社中央書局本店]]：位於[[中區_(台中市)|中區]][[中正路_(台中市中區)|中正路]]125號，與[[市府路_(台中市中區)|市府路]]交叉口，昭和元年（1926年）[[台灣文化協會|台灣文化協會]]創設的中央俱樂部。
*台中武德館：位於[[中區_(台中市)|中區]]興中街90號（原為中尊寺庭院），建於1957年的武道館建物。

== 宅第類 ==
*[[福建源|福建源]]、[[梅鏡堂|梅鏡堂]]、[[勝昌堂|勝昌堂]]：位於[[北區_(台中市)|北區]]體二用地內（[[國立臺灣體育運動大學|國立臺灣體育運動大學]]後方，有即將拆除危機）。
*[[旱溪余氏下邳堂|旱溪余氏下邳堂]]：位於[[東區_(台中市)|東區]]旱溪街61巷1號，台中少見的日治時期閩式三合院土角厝。。
*[[林文欽|林文欽]]墓：位於[[大里區|大里區]]爽文路449號的大里花市供應中心側方田埂小路右轉進去。另有墓道碑於爽文路與德芳路三段交叉口的進士公園內。
*[[更樓|更樓]]：原[[吳鸞旂|吳鸞旂]]公館正門門樓，1889年建於新莊子（今[[台中車站|台中車站]]後站區域）。1983年遷建至台中公園內，為台灣僅存的中式更樓建築物。
*大安中庄黃宅：黃卿宅第，1935年中部大地震後改建為帶有日式洋樓的四合院。

== 宗教類 ==

*[[寶覺禪寺|寶覺禪寺]]舊正殿：建於1927年，位於[[北區_(台中市)|北區]]健行路140號
*[[天主教|天主教]][[瑪利諾會|瑪利諾會]]語言服務中心：位於[[西區_(台中市)|西區]]的三民路一段與貴和街口。
*[[豐原慈濟宮|豐原慈濟宮]]：現況為1933年修建完工時之樣貌，位於[[豐原區|豐原區]]中正路179號。
