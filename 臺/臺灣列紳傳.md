'''《臺灣列紳傳》'''為[[臺灣日治時期|臺灣日治時期]][[臺灣總督府|總督府]]委託[[鷹取田一郎|鷹取田一郎]]編纂的一本人物[[列傳|列傳]]，以[[文言文|漢文]]簡要記述當時[[臺灣|臺灣]]社會知名且持有[[臺灣紳章|臺灣紳章]]的諸多[[仕紳|仕紳]]菁英，共計1020位。本書於1915年（大正4年）開始編纂，隔年出版，為[[台灣日治時期#始政時期|始政時期]]的刊物，目的在於彰顯臺灣仕紳對[[大日本帝國|帝國]]與[[天皇|天皇]]陛下的貢獻，並且從而攏絡心人。本書除了當時的政治用途之外，亦有其不可抹滅的史學價值。

== 背景 ==
1896年（[[明治|明治]]29年）10月23日，初期執政的日本政府為了攏絡[[臺灣人|臺灣人]]的社會菁英以及有名望者，[[臺灣總督府|臺灣總督府]]遂以府令第50號公佈〈臺灣紳章條規〉，以優卹遺賢、慰撫逸民、勸獎德化及維持風教為由，頒授[[臺灣紳章|臺灣紳章]]。1915年（[[大正|大正]]4年），總督府為緩和[[西來庵事件|西來庵事件]]對統治方所帶來的衝擊，同時也為了表彰受紳章者對臺灣的貢獻，因此藉由編纂《臺灣列紳傳》宣揚[[儒家|儒家]]價值來攏絡人心、穩定政局。時任[[臺灣總督|臺灣總督]]的[[安東貞美|安東貞美]]遂將整理之1020位持有紳章的菁英名單，委託[[鷹取田一郎|鷹取田一郎]]編纂這批仕紳各自的簡要傳記，並於1916年（大正5年）出版。<ref>[http://www.ntl.edu.tw/sp.asp?xdurl=MP15AP/BOOK_CP.asp&xItem=15691&ctNode=1427&mp=15 臺灣列紳傳 :: 國立臺灣圖書館100週年館慶]</ref>

為了處理西來庵事件而來抵臺灣的[[民政長官|民政長官]][[下村宏|下村宏]]於本書之序文中寫道：「紳章條規優卹[[清朝|前朝]]遺賢，並勸獎鄉黨模楷，……，今算所授紳章，自條規設定之始，以迄今日，鄉賢莊者，……，凡一千有餘名。已入鬼籍者，不亦鮮。今而不立其傳記，應雖有殊功偉德非常之跡，千古堙滅而不著如此畢矣。遺憾何忍哉？於是乎考定紳章佩有者，閱歷逸聞，命編次《臺灣列紳傳》，或以補足文獻乎？讀者庶幾以知雍熙之和，遠有其由來。」充分彰顯編纂本書的用意。<ref name="國史館臺灣文獻館">[http://www.th.gov.tw/epaper/view2.php?ID=54&AID=727 臺灣列紳傳 :: 國史館臺灣文獻館-電子報]{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

== 內容 ==
[[File:臺灣列紳傳內文1.jpg|right]]。]]
《臺灣列紳傳》以所收錄的仕紳之[[姓氏|姓氏]]與出生地作為分類索引，以便閱讀者翻查。文中詳述該仕紳之簡要傳記，並多少記載台籍仕紳於[[乙未戰爭|乙未戰爭]]之後的事蹟，暗示其普遍對帝國及[[天皇|天皇]]陛下的效忠與奉獻；例如本書對[[辜顯榮|辜顯榮]]的介紹中有著「[[馬關條約|乙末割讓]]之時，我軍已上陸於三貂角，人心鼎沸，紛擾不可名狀。君單身赴基隆，表誠[[天皇|皇]]師，邀到於臺北，從南進軍而有功。」等敘述記載。藉由本書除了能夠了解所載仕紳個人的生平簡介之外，更能配合日本時代總督府有系統的檔案及田野調查等考證史料，來更深入地研究該仕紳所處的家族歷史、發展事業。<ref name="國史館臺灣文獻館"></ref>
