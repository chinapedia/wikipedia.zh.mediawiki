'''臺鹽七股機車庫'''位於臺灣[[臺南市|臺南市]][[七股區|七股區]]的[[七股鹽場|七股鹽場]]內，是停放與維修[[臺灣鹽業鐵路|臺鹽小火車]]的地方{{r|黃}}，於民國九十四年（2005年）3月18日公告為歷史建築{{r|文}}。

== 沿革 ==
七股鹽場在日治時期便已有鐵道運輸，主要是用來將各鹽田所產的鹽集中起來{{r|黃}}。二次大戰之後[[臺灣製鹽總廠|臺灣製鹽總廠]]在美援支助下將舊有系統予以改善，分成東幹線、中幹線、西幹線、西幹支線與臺區1到5號線{{r|黃}}。七股鹽場各個鹽田所產的鹽會先放置於附近的場務所內，再由小火車運走，不過要將鹽運到鹽場之外的話，便需要靠[[臺灣糖業鐵路|臺糖鐵道系統]]的協助{{r|黃}}。但隨著時代演變，公路運輸逐漸取代了鐵路運輸，民國七十六年（1987年）2月七股鹽場鹽鐵因而停駛，之後機車庫便閒置下來{{r|黃}}。<ref>{{Cite news|url=http://nanying.pixnet.net/blog/post/25609859-%E3%80%8A%E5%8D%97%E7%80%9B%E5%AD%B8%E3%80%8B%E9%9B%BB%E5%AD%90%E5%A0%B1%E7%AC%AC104%E6%9C%9F%E3%80%94%E5%AD%B8%E8%A1%93%E6%96%87%E6%91%98%E3%80%95%E6%AD%B7%E5%8F%B2%E5%BB%BA|title=《南瀛學》電子報第104期〔學術文摘〕歷史建築─台鹽七股機車庫 @ 臺南市政府文化局-臺南學部落格 :: 痞客邦 PIXNET ::|last=(NanYing)|first=NanYing|newspaper=臺南市政府文化局-臺南學部落格|accessdate=2016-12-20|language=zh-TW}}</ref>

== 建築 ==
[[File:臺鹽七股機車庫02.JPG|220px]]
機車庫建築為磚造建築，開有許多玻璃窗，從正面來看可分成三個部分{{r|黃}}。最東邊的部分正面階梯山牆上有[[朱玖瑩|朱玖瑩]]所題的機車庫三字，而其屋頂為雙斜坡式，鋪著水泥瓦，而屋架則是木製，但也有使用部分鐵架來支撐{{r|黃}}。在其前後皆開有雙扇式大鐵門，不過正面是兩個大門，後面僅開一個，而室內地上仍留有鐵軌，{{r|黃}}。中間部分與東邊部分之間以水泥牆相隔，但與西邊部分相通，只有用幾根柱子作象徵性的區隔，不過地上也鋪有鐵軌{{r|黃}}。

西邊部分是民國五十一年（1962年）10月9日竣工的七股鹽場木工訓練班，其正立面亦為階梯形山牆，不過跟機車庫不同的是上面還加了四道垂直線條做裝飾{{r|黃}}。屋頂結構與機車庫相似，也使用多組三角型大木架來支撐屋頂{{r|黃}}。
