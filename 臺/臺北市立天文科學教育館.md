'''臺北市立天文科學教育館'''或稱'''台北市立天文館'''、'''台北天文館'''位於[[臺灣|臺灣]][[臺北市|臺北市]][[士林區|士林區]]，為[[臺北市政府教育局|臺北市政府教育局]]所屬之[[社會教育|社會教育]]機構，成立於1996年11月7日，1997年7月20日正式全面開放。其前身為臺灣第一個[[天文|天文]]教育機構「臺北市立天文台」。目前館內設施包括使用[[虛擬實境|虛擬實境]]與[[擴增實境|擴增實境]]技術模擬太空實際場景的展示場、宇宙探險區、[[全天域電影|宇宙劇場]]、[[立體劇場|立體劇場]]、天文教室、[[圖書館|圖書館]]、圓頂天文觀測室等。{{R|臺北星空64|page=14-16}}<ref name = "台北天文館重開幕 柯P和學童大玩機智問答 - 政治 - 自由時報電子報_2018"> {{ cite web | title = 台北天文館重開幕 柯P和學童大玩機智問答 - 政治 - 自由時報電子報 | url = http://news.ltn.com.tw/news/politics/breakingnews/2478980 | publisher = 自由時報 | author = 沈佩瑤 | language = zh-tw | date = 2018-07-05 }} </ref><ref name = "台北天文館重啟 柯文哲：教育任務越來越多_2018"> {{ cite web | title = 台北天文館重啟 柯文哲：教育任務越來越多 | url = http://www.epochtimes.com/b5/18/7/5/n10539054.htm | publisher = 大紀元時報 | author = 責任編輯：王愉悅 | language = zh-tw | date = 2018-07-05 }} </ref>

== 簡介 ==
天文科學教育館的建築主體是以外型穩重的塊體及前衛的採光罩構成。教學區屋頂上有2座圓頂觀測台，放置45[[公分|公分]][[卡塞格林反射鏡|蓋賽格林式反射望遠鏡]]及20公分[[反射望遠鏡#焦平面|庫德焦點]]的[[折射望遠鏡|折射望遠鏡]]。建築主體旁邊的圓頂式全天域劇院[[直徑|直徑]]35[[公尺|公尺]]，表面被覆著象徵宇宙神秘感的金黃色[[鋁|鋁]]錐面板。館區四周放置了[[黃道經緯儀|黃道經緯儀]]、[[天球儀|天球儀]]、[[赤道儀|赤道儀]]和[[日晷|赤道日晷]]，並藉著植栽區隔視線導引來增加空間的變化及私密性。

== 歷史沿革 ==
[[File:Taipeisun01.jpg|thumb]]
1938年，[[台灣新生報|台灣日日新報社]]為紀念創刊40周年，將建在[[台北公會堂|台北公會堂]]屋頂上的4[[吋|吋]][[赤道儀|赤道儀]][[折射望遠鏡|折射望遠鏡]]及圓頂觀測台，捐獻給[[臺北市政府|台北市役所]]，命名為臺北公會堂天文臺。1945年[[二戰結束|二戰結束]]之後，[[臺北市政府|臺北市政府]]接收了公會堂，更名為[[中山堂|中山堂]]，隨即更名為中山堂天文臺，由氣象局管理。1946年改隸臺北市政府教育局，命名臺北市立天文台，此即臺北市立天文科學教育館最早的前身。{{R|臺北星空64|page=12-13}}

1957年，[[蘇聯|蘇聯]]發射了[[人造衛星|人造衛星]]進入[[行星軌道|繞地軌道]]後，引發了社會大眾對天文知識的熱潮。由於中山堂附近夜間[[光害|光害]]嚴重不適合觀測工作，1960年在[[圓山|圓山風景區]]內開始動工興建新的'''圓山天文台'''，1963年3月28日竣工啟用，但初期受限人力不開放民眾參觀。<ref>{{cite news | title = 圓山天文台 昨行落成禮 | date = 1963-03-29 | work = 聯合報 | page=02}}</ref>8月31日臺灣省政府決議每週開放參觀。<ref>{{cite news | title = 圓山天文台 定時開放| date = 1963-08-31 | work = 聯合報 | page = 02}}</ref>

1986年哈雷彗星的回歸，在媒體的推波助攬下掀起了[[台灣地區|台灣地區]]的觀星熱潮。而圓山天文台規模過小，加上臺北市人口急速增加，使圓山天文台不敷使用；加上後方的[[圓山大飯店|圓山大飯店]]和前方的[[中山高速公路|中山高速公路]]興建後造成的[[光害|光害]]問題，遂有了遷建與擴建的計畫。1993年於[[士林區|士林區]]基河路動工興建天文科學教育館，1996年天文台改制為臺北市立天文科學教育館，1997年2月4日舉辦開館典禮，隔日正式開放第一期展區。<ref>{{cite news | title = 市立天文館 今開館典禮 | author = 沈長祿 | work = 聯合報 | date = 1997-02-04 | page = 17版都會掃描 | location = 台北}}</ref><ref>{{cite web | work = 臺北市立天文科學教育館 | url = http://www.tam.gov.tw/ct.asp?xItem=14050515&CtNode=50674&mp=104051 |title = 臺北天文科學教育館-歷史沿革及大事紀 | date = 2014-04-14 | accessdate = 2014-07-04}}</ref>7月20日展館全面開放。<ref>{{cite news | title = 天文展示場明啟用 遊憩增色 | author = 沈長祿 | location = 台北 | work = 聯合報 | date = 1997-07-19 | page = 15版生活資訊}}</ref><ref>{{cite news | title = 台北天文館 全面開放 182件模型、設備、圖片深入淺出呈現宇宙現象 | author = 邱淑玲 | date = 1997-07-21 | work = 民生報 | page=17版戶外焦點}}</ref>

其後圓山天文台建築改為校外教學專區，但因建築老舊等安全問題，於2000年7月25日拆除，但仍留下一個水平式[[日晷|日晷]]供人紀念。{{R|臺北星空64|page=13}}

2017年3月，宣布台北市立天文館將和兒童新樂園及美崙公園合併成一個主題園區。

2018年7月5日，天文館歷經3年耗資5.8億元新台幣經費重整後，新增阿波羅17號等比例的擬真登月車虛擬實境互動設備；新增模擬器可模擬太陽高能粒子打到地球發光光學現象；新增內投影球模擬8大行星的太陽系結構擴增實境設備；另外開設星際迷航主題特展提供體感設備與互動遊戲至2018年12月2日。<ref name = "台北天文館重開幕 柯P和學童大玩機智問答 - 政治 - 自由時報電子報_2018" /><ref name = "台北天文館重啟 柯文哲：教育任務越來越多_2018" /><ref name = "斥資5.8億建擬真設備 北市天文館新展場啟用_2018"> {{ cite web | title = 斥資5.8億建擬真設備 北市天文館新展場啟用 | url = http://www.cna.com.tw/news/ahel/201807050266-1.aspx | publisher = 中央通訊社 | author = 陳妍君 | language = zh-tw | date = 2018-07-05 }} </ref>

==組織架構==
===館本部===
*館長
*秘書

===內部單位===
*研究組：負責天文觀測、記錄、研究、統計、登錄、圖書管理、出版、國際連繫及資料交換等事項。
*推廣組：負責天文教育規劃、輔導、推廣、天文研習、學校天文教育及天文觀測室之開放等事項。
*展示組：負責本館展示主題之規劃與設計製作、展示設施之維護管理及天文營隊活動之辦理等事項。
*視聽組：負責全天域與立體劇場電影放映、天文節目製作、星象解說與儀器操作及維護等事項。
*秘書室：負責文書、檔案、出納、總務、財產之管理，研考及不屬於其他各單位事項。
*會計室：依法辦理歲計、會計及統計事項。
*人事室：依法辦理人事管理事項。
*政風室：依法辦理政風事項。

== 出版品 ==
*月球表面地形名稱：未出刊文字版，可由網路下載說明與附錄(欠主檔)<ref>[http://www.tam.gov.tw/lp.asp?ctNode=50932&CtUnit=7100&BaseDSD=7&mp=104051 臺北市立天文科學教育館-月面地形中文名稱(出版品)]</ref>。
*臺北星空：1998年10月1日創刊<ref>[http://tcgwww.taipei.gov.tw/public/Attachment/9761414610.pdf 第一期館長的話]</ref>，為一年4期的季刊。前身為《天文通訊》<ref>最早是台北市天文同好會的期刊，於1965年(131期)改由天文台出版，至1998年6月1日共出刊287期</ref><ref>天文通訊停刊後先改名為《天文館期刊》，出版了4期，才改名為《台北星空》[http://tcgwww.taipei.gov.tw/public/Attachment/97615305362.pdf 第五期館長的話]</ref>。
*天文年鑑：1988年創刊，每年一期，介紹與說明當年度的主要天象。2018年不再出刊紙本，改為電子版，民眾可自行下載參考。
*太陽黑子年報：1983年創刊，每年一期，統計上一年度的黑子數量與分類，並繪製歷年的平滑曲線。
*天文館學報：與[[中華民國天文學會|中華民國天文學會]]合作出版，於2003年開始出刊，每年一期。
*天文館電子報：2008年1月5日發行第一期，提供天文館最新訊息、天文新知、天象速報等資訊。前身為1975年4月24日開始不定期發行的天文快報，迄1996年3月27日已出刊400期，並彙整每百期集結印製合訂本，共印製四輯。其後仍繼續發行，迄2008年改為電子版，並接受各界訂閱<ref>[http://epaper.taipei.gov.tw/MP_999.html 電子報訂閱中心]</ref>。
