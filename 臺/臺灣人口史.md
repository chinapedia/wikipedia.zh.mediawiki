== 明鄭時期 ==
[[鄭成功|鄭成功]]率軍驅逐了[[荷蘭人|荷蘭人]]，又和[[平埔族|平埔族]]的侵襲發生多起爭鬥，同時，帶來的軍民使臺灣臺南人口激增到3萬多人。

鄭氏經營台灣20年，期間{{citation needed|不少福建南部沿海居民移居台灣}}，據[[臺灣通史|臺灣通史]]之推測，台灣漢人人口已近20萬人。<ref> 《[[臺灣通史|臺灣通史]]》－〈戶役志〉：清人得臺之時，志稱舊額戶一萬二千七百二十七，口一萬六千八百二十人，歲徵銀八千零六兩零三錢二分。是必有所謬誤；不然，何其鮮耶！考施琅疏陳海上情形，謂：『查自故明時，原住澎湖百姓有五、六千人，原住臺灣者有二、三萬人，俱係耕漁為生。至順治十八年，鄭成功挈去水陸官兵眷口三萬有奇。康熙三年，鄭經復挈去六、七千人。』以此計之，則臺灣之人殆十萬。何以僅為一萬六千餘人？且琅之疏亦有未確者。鄭氏陸師七十有二鎮，使鎮為千人，則有七萬二千。加之以四民，應倍其數。是臺灣之民，此時已近二十萬。不然，以一萬六千餘人，僅不過一鄉，而奏設三縣，何其夸耶？蓋志之所載，僅舉丁稅而言爾。</ref>

== 清治時期 ==
滿清統治初期，為免明鄭死灰復燃，曾長期採取海禁政策。[[清朝|清朝]]後期才增加閩粵移民到295萬多人。而據[[魏源_(清朝)|魏源]]《[[聖武記|聖武記]]》所述，[[乾隆|乾隆]]五十一年（1786年）漢人和西部平埔族原住民的比例為20比1。<ref>《重修臺灣省通志》，卷三，第40頁，住民志語言篇</ref>清治台灣兩百多年，閩粵移民大量移居台灣。1895年清廷因中日甲午戰爭戰敗根據馬關條約割讓台灣予日本，此時台灣人口近300萬人。

== 日治時期 ==
日治初期，有一些台灣人移居中國大陸。台灣日治時期共有七次人口普查，台灣人和遷至台灣的日本人（包括[[琉球|琉球]]人）一併記入。1905年10月1日，台灣人口約298萬人。<ref>{{cite book |author=臺灣總督府官房臨時臺灣戶口調查部編 |title=臺灣臨時戶口調查結果表 |year=1908 |publisher=臺灣總督官房臨時戶口調查部 |location=臺北 |isbn= }}</ref>1940年10月1日，台灣人口約581萬人。<ref>{{cite book |author=臺灣省政府主計處編 |title=臺灣第七次人口普查結果表 |year=1953 |publisher=臺灣省政府主計處 |location= |isbn= }}</ref>1945年10月底，[[臺灣戰後時期|中華民國接管臺灣]]，此時的台灣人加上遷至[[台灣|台灣]]的日本人，共約600萬人。

== 戰後初期(1945至1949年) ==
[[臺灣戰後時期|中華民國接管臺灣後]]，遣返了至少49萬日本人（包括[[琉球|琉球]]人）。台灣人口略為減少。

== 中華民國遷台後(1949年至今) ==
{{main|臺灣人口}}
至少121萬戰後移民<ref>[http://203.64.42.21/TG/KhinhoaN/social/%E5%8F%B0%E7%81%A3%E5%90%84%E6%97%8F%E7%BE%A4%E7%9A%84%E4%BA%BA%E5%8F%A3%E8%88%87%E6%94%BF%E7%B6%93.ppt 台灣各族群的人口與政經力量 （陳淑芬）]{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>隨中華民國政府遷入台灣，加上[[戰後嬰兒潮|戰後嬰兒潮]]，台灣人口激增。1956年9月16日，台灣人口達到922万5426人。<ref>{{cite book |author=臺灣省戶口普查處編 |title=中華民國戶口普查報告 |year=1959 |publisher=臺灣省戶口普查處 |location=臺北市 |isbn= }}</ref>2008年7月台灣人口正式突破2300萬人。目前總人口'''23,590,744'''人（2019年1月底官方統計）<ref>[https://www.moi.gov.tw/chi/chi_news/news_detail.aspx?sn=15513&type_code=01&search_unit=025&pages=0&src=news   民國108年1月戶口統計資料分析]行政院內政部戶政司</ref>。目前正面臨[[少子化|少子化]]危機。
