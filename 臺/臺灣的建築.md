《'''臺灣的建築'''》（[[日文|日文]]原名：'''{{lang|ja|臺灣の建築}}'''）為一本關於[[臺灣建築|臺灣建築]]的[[日文|日文]]研究書籍。本書的作者為{{link-ja|藤島亥治郎|藤島亥治郎}}，共計222頁，於1948年2月10日由[[日本|日本]][[彰國社|彰國社]]出版。

== 背景 ==
本書作者[[藤島亥治郎|藤島亥治郎]]為日本著名的建築史學家，曾於[[朝鮮日治時期|朝鮮日治時期]]擔任[[首爾大學|京城工業學校]]的助理教授，並任職於[[朝鮮總督府|朝鮮總督府]]。藤島於1929年返回日本[[內地|內地]]，並在1933年取得[[東京帝國大學|東京帝國大學]]工學博士學位，留任該校教職。由於曾長期旅居朝鮮，藤島因此對[[朝鮮建築|朝鮮建築]]有著相當豐富的研究，並曾於《[[建築雑誌|建築雑誌]]》刊登連載一系列的〈朝鮮建築史論〉文章、於1976年出版《{{lang|ja|韓の建築文化}}》一書。<ref name="國史館">[http://www.th.gov.tw/epaper/view2.php?ID=34&AID=242 《臺灣の建築》 :: 國史館臺灣文獻館-電子報] {{webarchive|url=https://web.archive.org/web/20141021083331/http://www.th.gov.tw/epaper/view2.php?ID=34&AID=242 |date=2014-10-21 }}。</ref>

1936年，藤島接受[[日本學術振興會|日本學術振興會]]昭和十一年度的挹注經費，進行編纂對於[[臺灣建築|臺灣建築]]的研究成果。由於藤島之研究過於專業與深入，使得刊行時有些困難，因此改以簡要的版本刊行，旨在向[[中國|中國]]與日本兩國及世界各國等人士介紹臺灣建築之特色。藤島於該年4月4日抵達[[台灣|台灣]]，並於4月24日離臺。在此期間，藤島由[[臺灣總督府|臺灣總督府]]民政部土木局營繕課長兼臺灣建築會長[[井手薰|井手薰]]陪同訪視台灣各地。由於參訪時間過於短暫，因此藤島於本書中援引了[[台灣清治時期|清治時期]]的[[地方志|地方志]]及總督府的各項調查報告，並得到許多日本研究人士的協助。例如[[台灣原住民|原住民]]建築與西洋建築方面的資料與研究，便大多接受了[[台灣大學|臺北工業學校]]教諭[[千千岩助太郎|千千岩助太郎]]以及臺灣總督府技師[[栗山俊一|栗山俊一]]的提供。<ref name="國史館"></ref>

== 內容 ==
《臺灣的建築》全書共222頁，另外附有照片28禎及插圖50幅。內容分為五大部份：總說、第一篇、第二篇、第三篇、結說。第一章為總說。第二章則偏向分析地文與氣候對[[台灣建築|台灣建築]]的影響，例如：高溫、潮濕、[[颱風|颱風]]及[[地震|地震]]等因素；並且舉出「[[騎樓|亭仔腳]]」作為台灣與中國[[閩南|閩南]]順應氣候的建築的典型例子。此外，亦分析了台灣建築所使用的各種[[建材|建築材料]]，如：[[木材|木材]]、[[竹材|竹材]]、[[石材|石材]]、[[粘土|粘土材]]、[[瓦片|煉瓦]]、[[磚|瓦甎]]等等。

根據藤島的分類，可將台灣的建築分為三大種類：南洋系、大陸系以及西洋系建築。其中，南洋系即是介紹[[台灣原住民族|台灣原住民族]]的家屋、聚落等建築構法。大陸系則是介紹[[閩南人|閩南裔]]與[[客家人|客家裔]]族群的各項建築，其內容較為豐富，分為住宅、庭園、城堡、[[郊商|會館]]、[[儒教|儒]][[佛教|釋]][[道教|道]]三教建築、[[牌樓|牌樓]]、[[石碑|石碑]]等等。西洋系則介紹城堡及[[基督教|基督教]]建築等。

== 語文 ==
由於台灣為以[[閩南人|河老人]]為主體民族，因此在提及建築相關的專有名詞時，除了書寫[[漢字|漢字]]之外，還會以[[片假名|片假名]]附記於旁以提供相關的[[台語|台語]]讀音。例如：「{{lang|ja|灶腳間}}（{{lang|ja|ツアウカーキユン}}）」、「{{lang|ja|食飯間}}（{{lang|ja|チヤブウンキユン}}）」及「{{lang|ja|磚仔}}（{{lang|ja|ツンア}}）」等。
