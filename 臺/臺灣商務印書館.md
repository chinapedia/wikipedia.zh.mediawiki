'''臺灣商務印書館'''，簡稱'''臺灣商務'''或'''商務印書'''，是一家臺灣的[[出版社|出版]][[零售商|零售]]業者，其前身是1947年10月在[[臺灣|臺灣]][[臺北市|臺北市]]設立的「'''[[商務印書館|商務印書館]]臺灣分館'''」。

== 沿革 ==
[[File:Taiwan_Branch,_The_Commercial_Press_1948.jpg|缩略图]]
[[File:CPTW_Yun-wu_Building_20150830.jpg|right]]
1945年[[第二次世界大戰|第二次世界大戰]]後，[[臺灣日治時期|臺灣日治時期]]結束，原在中國大陆的大出版社紛紛來臺設立分店。

1947年9月，商務印書館遣[[福州|福州]]分館經理葉有楳來臺籌設分支。1947年10月，商務印書館購得位於臺北市[[重慶南路_(台北市)|重慶南路一段]]37號的一棟三層磚木造房屋，並予以營修，作為臺灣分館預定地。

1948年初，「商務印書館臺灣分館」開業，販售由商務印書館[[上海|上海]]本部運來的圖書、[[文具|文具]][[儀器|儀器]]等，然而數量與種類皆不多，且無自己的出版業務。

1949年，[[中華人民共和國|中華人民共和國]]成立，上海商務旋即停止對臺灣分館發貨。面臨無書可賣的情況，臺灣分館經理趙叔誠決定開展編輯、印刷業務，從一個販售處轉型成為獨立編印的出版社。商務印書館臺灣分館在[[中華民國政府|中華民國政府]]要求下，脫離總館獨立，改名「臺灣商務印書館」迄今，[[英文|英文]]名稱與總部地址均未改。1950年10月10日，臺灣商務印書館完成公司登記。

商務印書館以編譯[[教科書|教科書]]起家，來臺初期十餘年，在臺灣分館經理趙叔誠規劃下，以再版大陸時期的出版品與教科書為主。

1964年，原商務印書館總經理[[王雲五|王雲五]]出任臺灣商務印書館董事長，並訂定新出版藍圖，重視巨部[[古籍|古籍]]叢書的印行、重版書籍以及新舊書籍參錯的《人人文庫》。古籍印行無[[著作權|著作權]]限制，可免去許多[[成本|成本]]，同時呼應[[中華文化復興運動|中華文化復興運動]]，故成為當時出版主流。

1968年，臺灣商務印書館總部改建為[[鋼筋水泥|鋼筋水泥]]四層樓，稱為「雲五大樓」<ref name="cntimes1">{{cite web |url = http://chincenter.fg.tp.edu.tw/~writing/news.php?id=1224831030&PHPSESSID=2b45d433329b6e7e5f8edf1e877d2065 |title = 重慶南路地標 臺灣商務一甲子 雲五大樓改建 |author = 林欣誼 |publisher = [[中國時報|中國時報]] |date = 2011-08-26 |accessdate = 2013-05-25 |language = zh-tw }}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

1979年8月14日，王雲五病逝於臺北市，享年92歲。1979年至2002年，[[劉發克|劉發克]]出任臺灣商務印書館董事長，帶領臺灣商務印書館開始轉型。2002年5月，劉發克退休，王雲五之子[[王學哲|王學哲]]出任臺灣商務印書館董事長<ref>{{cite web |url = http://news.chinatimes.com/Chinatimes/Philology/Philology-artnews/0,3409,112002112400163+110513+20021124+news,00.html |title = 王雲五之子王學哲掌舵 商務印書館重新出發 |author = 陳文芬 |publisher = [[中時電子報|中時電子報]] |date = 2002-11-24 |accessdate = 2014-02-04 |language = zh-tw |deadurl = yes |archiveurl = https://web.archive.org/web/20140222065142/http://news.chinatimes.com/Chinatimes/Philology/Philology-artnews/0,3409,112002112400163+110513+20021124+news,00.html |archivedate = 2014-02-22 }}</ref>。2011年，[[施嘉明|施嘉明]]出任臺灣商務印書館董事長，王雲五長孫[[王春申|王春申]]接任臺灣商務印書館副董事長。

臺灣商務印書館出版大量學術書，影印出版全部[[文淵閣|文淵閣]]《[[四庫全書|四庫全書]]》，影印出版[[中華民國教育部|中華民國教育部]]1947年重版《[[教育部國語辭典|國語辭典]]》，1981年為中華民國教育部排印出版《重編國語辭典》。

[[File:CPTW_Chongnan_Store_20130913.jpg|right]]
2010年1月14日，臺灣商務印書館第二門市於臺北市重慶南路一段143號1樓開幕。2010年8月8日，位於雲五大樓1樓的臺灣商務印書館門市部結束營業，臺灣商務印書館門市部搬遷至第二門市。<ref>[http://www.cptw.com.tw/event/shop/index.html 臺灣商務印書館-門市訊息] {{webarchive|url=https://web.archive.org/web/20110829031254/http://www.cptw.com.tw/event/shop/index.html |date=2011-08-29 }}</ref>

2011年，《[[科學月刊|科學月刊]]》與臺灣商務印書館合作出版《商務科普館》系列叢書。2011年8月，臺灣商務印書館與[[聯合新聞網|聯合新聞網]]合作發行的[[電子報|電子報]]《重南參柒電子報》創刊<ref>{{cite web |url = http://chincenter.fg.tp.edu.tw/~writing/news.php?id=1224831034 |title = 重南參柒電子報 揭探書籍背後的精彩故事 |author = 台北訊 |publisher = 聯合新聞網 |date = 2011-08-29 |accessdate = 2014-08-23 |language = zh-tw }}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。2011年8月25日，臺灣商務印書館總編輯[[方鵬程|方鵬程]]證實，雲五大樓改建案兩年多前便談妥，拆除後將改建為地上十五層、地下四層的住商大樓，但目前還在建築執照申請程序中，拆遷時間未定，外牆拆下的「臺灣商務印書館」及「雲五大樓」鑄字招牌將收藏在臺北市新生南路[[王雲五紀念館|王雲五紀念館]]<ref name="cntimes1"/><ref name="tvbs1">{{cite web |url = http://news.tvbs.com.tw/old-news.html?nid=64857 |title = 出版「未央歌」聞名 「台灣商務」大樓將拆 |author = 史靜芬 |publisher = TVBS新聞 |date = 2011-08-27 |accessdate = 2015-12-09 |language = zh-tw}}</ref>。2012年6月28日，《重南參柒電子報》停刊。2012年12月28日，臺灣商務印書館第二門市結束營業，重南門市在雲五大樓2樓開幕。

[[File:CPTW_XinNan_Store_20131130.jpg|thumb]]
2013年3月，臺灣商務印書館新南門市在臺北市大安區新生南路三段19巷3號1樓開幕，名為「商務十九人文讀境」，曾經是王雲五的藏書室，斜對面即是王雲五紀念館<ref>{{cite web |url = http://cci.culture.tw/cci/cci/news2_detail.php?p=263&sn=5925 |title = 台灣商務印書館 新生南路開新店  |author = 陳宛茜 |publisher = [[聯合報|聯合報]] |date = 2013-03-20 |accessdate = 2014-05-23 |language = zh-tw}}</ref><ref>{{cite web |url = http://mol.mcu.edu.tw/show_2009.php?nid=154017 |title = 台灣商務印書館 打造王雲五文創基地 |author = 陳岱安 |publisher = 銘報即時新聞 |date = 2013-03-26 |accessdate = 2014-02-02 |language = zh-tw |deadurl = yes |archiveurl = https://web.archive.org/web/20140219080406/http://mol.mcu.edu.tw/show_2009.php?nid=154017 |archivedate = 2014年2月19日 |df = }}</ref>。

2014年6月，王春申接任臺灣商務印書館董事長。2014年12月3日是臺灣商務印書館在雲五大樓營業的最後一天，臺灣商務印書館舉行搬遷特賣，雲五大樓外牆的「臺灣商務印書館」及「雲五大樓」鑄字招牌原地保留。2014年12月8日，臺灣商務印書館總部及門市改在[[新北市|新北市]][[新店區|新店區]][[復興路_(新店區)|復興路]]43號8樓繼續營業。2018年1月3日，臺灣商務印書館復興路門市結束營業。2018年1月15日，臺灣商務印書館總部及門市搬遷至新北市新店區民權路108之3號5樓。<ref>{{Cite web|url=https://www.cptw.com.tw/upload/banner/20180103134040_pic-pc.jpg|title=臺灣商務印書館——官網訊息|author=臺灣商務印書館|date=2018-01-03|accessdate=2018-03-17|publisher=臺灣商務印書館}}</ref>
