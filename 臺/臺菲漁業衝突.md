'''臺菲漁業衝突'''是指[[中華民國|中華民國]]（[[臺灣|臺灣]]）與[[菲律賓|菲律賓]]兩國數十年來在各自主張之[[專屬經濟海域|專屬經濟海域]]重疊區域上多次發生的[[漁業|漁業]]糾紛<ref>[http://udn.com/NEWS/NATIONAL/NATS4/7893036.shtml 曾被押老船長：菲國惡質30年沒變]</ref>，甚至造成[[漁民|漁民]]傷亡。[[漁業署|漁業署]]統計從1990年至2012年，菲律賓就扣留了108艘台灣漁船<ref>[http://www.udn.com/2013/5/15/NEWS/NATIONAL/NATS4/7898808.shtml 扣我漁船 菲國最多 許多漁民賠錢私了，聯合新聞網，2013-05-15]{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。有漁民指到[[南海|南海]]捕魚時必須帶大筆現金在身上，以應付菲律賓海巡署公務船的「登船安檢」。這個習慣被形容為漁民之間的「潛規則」<ref>{{cite news|url=http://www.ettoday.net/news/20130518/209565.htm|title=隨身帶1萬美金保命　南海捕魚「潛規則」|publisher=ETtoday東森新聞雲|date=2013-05-18|language=zh-tw}}</ref><ref>{{cite news|url=http://news.ifeng.com/taiwan/special/taiwanyuchuan/content-3/detail_2013_05/19/25463556_0.shtml|title=臺漁民南海捕魚“潛規則”：隨身帶1萬美金交菲保命|publisher=臺海網|date=2013-05-19}}</ref>。

===1984年2月===
*[[琉球嶼|琉球]]籍的海漁成號在[[北緯|北緯]]12度、[[東經|東經]]116度的[[南中國海|南中國海]]上作業，被編號28號的菲律賓軍艦開砲追擊及捕獲，結果漁獲及物品充公，並被菲律賓以索賠開砲所使用的軍費為由勒索，船員每人賠償10萬[[美元|美元]]後獲釋。5名船員被菲律賓扣押4個多月後駕船返回臺灣。<ref name=ShinChuan />

===1984年5月===                                                                                
*[[高雄市|高雄市]]籍五川三號為拖救故障的東港籍進吉春號，雙雙被菲律賓軍艦扣押，包括全數18名船員。另外一艘東港籍德財興號亦因為機器故障漂流，被菲律賓扣押全數船員11人。東港籍成豐六號被菲律賓扣押5個月後獲釋，5名船員返回臺灣。西嶼鄉豐振興號遇難發出求救訊號，隨即失蹤，5名船員生死不明。<ref name=ShinChuan />
                                                                                
*赫連號5名船員失蹤。敏利號5名船員失蹤。瑞昇號5名船員失蹤。東港籍忠和興號和一艘[[小琉球|小琉球]]漁船被不明國籍艦艇追逐8小時，一直追到[[蘭嶼|蘭嶼]]海面才脫險。<ref name=ShinChuan />

===1984年6月===                                                                                
*[[行政院|行政院]]表示[[中華民國國軍|中華民國國軍]]全力護魚收效。不久，被菲律賓扣押的進吉春號和五川三號被勒索38,000美元軍費後獲釋。<ref name=ShinChuan />
                                                                                
*[[高雄市市長|高雄市市長]][[許水德|許水德]]在[[高雄市議會|高雄市議會]]上說：高雄市籍漁船被外國扣留的共有80艘，人船均獲釋的有37艘，人已回來、船仍然被扣留的有22艘，還有一艘人船均被扣留。<ref name=ShinChuan />

===1984年9月===
*吉進興號被菲律賓扣押，有菲律賓[[華僑|華僑]]傳話獲得60萬元後會放船。聯豐發號在被菲律賓扣押後亦遭菲律賓華僑[[敲詐|敲詐]]。<ref name=ShinChuan />

===1984年10月===
*聯豐號遭菲國劫持，劫持所用武裝船，疑為台灣海難漁船改裝。<ref name=ShinChuan />

===1984年11月===
*被扣的吉興號5名船員在菲律賓斷炊，寫信回國救援。<ref name=ShinChuan />
                                                                                
===1984年12月===
*高雄市15艘漁船、200餘船員被扣海外，其中大部分為[[臺灣原住民|臺灣原住民]]，船公司遲不援救船員，亦無發配家屬安家費用。<ref name=ShinChuan />

===2002年4月===
*[[屏東縣|屏東縣]]籍漁船滿吉財三號船長被菲律賓囚禁1年6個月，繳付巨額罰金後獲釋。<ref name = shot1>{{cite news|title=菲國惡意攻擊台漁船 10年30起|url=http://news.pchome.com.tw/politics/ftv/20130511/index-13682660968711419001.html|publisher=PChome新聞|date=2013-05-10}}</ref>

===2004年===
*[[東港鎮_(台灣)|東港]]籍漁船成福利號在菲律賓北方[[巴丹群島|巴丹群島]]海域被一艘菲律賓武裝艦艇以[[機關槍|機關槍]]掃射，導致船長王保生因為身中數槍，當場斃命。<ref name = shot2>{{cite news|title=菲國武裝船艦惡名昭彰 台菲重疊經濟海域 危機四伏|url=http://www.libertytimes.com.tw/2013/new/may/10/today-fo4.htm|publisher=[[自由電子報|自由電子報]]|date=2013-05-10|accessdate=2013-05-10}}</ref>

===2004年4月===
{{Main|興億福號事件}}
*[[琉球嶼|琉球]]籍興億福號漁船因為機件故障漂流至菲律賓巴丹群島海域擱淺，[[行政院海岸巡防署|海岸巡防署]]接獲船長陳隆泰求救訊息後，派出[[花蓮艦|花蓮艦]]馳援。結果遭到島上武裝份子開槍攻擊，興億福號被洗劫一空，最終船上6名船員成功獲救。<ref name = shot1 /><ref>[http://www.libertytimes.com.tw/2013/new/may/12/today-p1.htm 巡護艦護漁 下月再添生力軍] [[自由電子報|自由電子報]] [2013.05.12]</ref>

===2006年1月===
*1月5日，臺東縣籍漁船滿春億號在巴丹群島東南方約500公尺處海域作業時，遭到[[菲律賓警察|菲律賓警察]]朝著駕駛艙以[[M14|M14]]和[[M16|M16]]步槍連續掃射，導致船長陳安老右大腿動脈中彈，失血過多死亡，胞弟陳明德臀部中彈受傷。<ref>[http://www.cna.com.tw/Topic/Popular/3692-2/201305140064-1.aspx 7年前殺我漁民 台續通緝菲警] [[中央社|中央社]] [2013.05.14]</ref>

===2009年5月===
*屏東縣琉球籍漁船紫福添號於5月12日，因躲避颱風駛近菲律賓海域，等到風雨過後準備返航，卻被當地海巡隊以越界捕魚為由遭扣押。當持船上成員包括船長洪勝福等兩名台籍幹部及10名印尼漁工全數遭軟禁。由於當地漁業單位多，漁船被扣後多方單位以罰款為由多重索錢，期間家屬多次到菲律賓斡旋，前後付款150萬元新台幣仍不放行。至8月3日船長趁天候不佳，斷纜逃出菲國港口，於8月4日安全返回屏東東港。<ref>{{cite news|url=http://news.cts.com.tw/cts/society/200908/200908050297683.html|title=遭菲國扣3個月 紫福添號脫逃歸來|date=2009-08-05|author=陳傳霆|work=台灣華視新聞}}</ref>

===2013年5月===
{{Main|廣大興28號事件}}
*5月9日，台灣[[屏東縣|屏東縣]][[琉球鄉|琉球鄉]]籍漁船廣大興28號於[[巴士海峽|巴士海峽]]裏中華民國（台灣）主張之經濟海域上捕魚時，被[[菲律賓海岸防衞隊|菲律賓海岸防衞隊]]的漁政公務船（編號MCS-3001）開槍攻擊與追逐。事件當中廣大興28號船員洪石成中彈身亡。此事使得[[台菲關係|台菲關係]]越見緊張<ref>[http://www.bbc.co.uk/news/world-asia-china-22520618 China media: Taipei-Manila deadlock，BBC，2013-05-14]</ref>。另外，事件發生當時，另艘台灣漁船「明正福12號」亦在附近，但未注意到廣大興28號已被攻擊，之後同艘公務船轉向明正福12號，整整追逐3個半小時。
