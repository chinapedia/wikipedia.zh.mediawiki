'''臺鐵機班'''指的是於[[臺灣鐵路管理局|臺灣鐵路管理局]]駕駛[[動力車|動力車]]的列車司機。就職稱來說，實際於[[臺鐵|臺鐵]]擔任動力車乘務員包含機車長、司機員、機車助理等約1,200人。連同整備員、學習司機員、未實際擔任運轉乘務或尚未正式取得駕駛證照的其他職稱人員，全部共約1,500人。

== 機班 ==
[[File:TRA_E200_train_driver_20110719.jpg|180px]]
於實際運作方面，整體動力車乘務員可稱為機班，其每人值乘工作稱為工作班。機班整合時刻表來擔任列車行駛工作，並以每100－250位員額，配置於臺鐵7個機務段。七堵、高雄及花蓮機務段，則各轄宜蘭、左營及台東機務分段。其中，配置於高雄機務段的人數為最多。
== 乘務時間 ==
在工作時間方面；如不跨班，臺鐵機班每日工作時間約為6小時－6小時30分鐘；其中不含強制休息之管制時間。另外於薪資方面，最基層機班連同「乘務旅費」、「誤餐費」等，平均月薪可達新台幣5萬元以上，不過實際薪資視服務年資及職位高低而有所不同。至於工作項目，除了駕駛列車外，台鐵司機亦需具備動力車的簡易維修能力，也因為如此，臺鐵招考機班時，常以「機檢工程」為錄取項目名稱。

== 工作 ==
[[File:TRA_EMU300_left_drivers_room.jpg|180px]]的駕駛室]]
就同樣軌道運輸的[[高鐵|高鐵]]及[[捷運|捷運]]來做比較，臺鐵機班工作環境如過小駕駛室、過長乘務距離、配置[[平交道|平交道]]的[[B型路權|B型路權]]行駛等，是臺灣軌道中工作條件較不理想，惟近年來主管單位已有相當幅度改善。這些機班週遭環境的軟硬體改進，亦吸引了更多高學歷的大學及碩士畢業生進入臺鐵機班的行列。

另外，隨著《[[性別工作平等法|性別工作平等法]]》通過，以往男性才能報考臺鐵乘務員的限制已被廢除；女性只要能通過稍為不利的握力35公斤門檻後，就有公平與男性競爭職務的條件。2012年[[邱千芳|邱千芳]]通過合格審查，成為台鐵123年來首位具備司機員資格的女性，之後再現第二位女司機員陳淑盈，預估未來再添四名女駕駛，分別是張櫻環、詹素瑜、蔡宜樺及陳彩霞。
