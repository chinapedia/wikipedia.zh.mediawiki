'''臺南市鹽水區歡雅國民小學'''位於[[中華民國|中華民國]][[臺南市|臺南市]][[鹽水區|鹽水區]]，其創校歷史可追溯至1919年，是鹽水區歷史第二久的國小{{r|鄭}}。該校建於1950年代的大禮堂與時鐘座已在2004年9月22日被公告為歷史建築{{r|文}}。該校在1958年時曾共設19個班，學生1800人，但自1970年代後便迅速減少，1987年全校只剩12班300人，現在六個年級只有1班{{r|鄭}}。目前歡雅國小的學區只有鹽水區的歡雅、大莊、孫厝、桐寮、田寮等里，而早期曾包含13個里{{r|鄭}}。

== 沿革 ==
歡雅國小創立於日大正八年（1919年）3月27日，最初是今鹽水國小的分校，稱為「鹽水港公學校番子厝分校」，由相良義人擔任分校校長，隔年9月20日首任校長廣懶情士才上任{{r|鄭}}。後來學校在大正十一年（1922年）3月31日獨立成「番子厝公學校」，同年4月30日第二任校長木原操上任{{r|史}}，昭和十六年4月1日因制度改革而改為「番子厝國民學校」{{r|鄭}}。

二次大戰後，陳大碑於民國三十四年（1945年）10月29日擔任戰後首任校長，隔年2月1日校名改為「鹽水鎮第二國民學校」，不久同年8月1日改稱「鹽水鎮歡雅國民學校」，後來在實行[[台灣九年國民義務教育|九年國民義務教育]]時於民國五十七年（1968年）8月1日改為「鹽水鎮歡雅國民小學」{{r|鄭}}，而2010年行政區劃改制後改為今名。

== 分校 ==
歡雅國小以「番子厝公學校」的名義獨立後，曾設有過5個分校，分別是昭和九年（1934年）4月1日所設的「番子厝公學校坔頭港分教場」、民國三十六年（1947年）9月所設的大豐分校、民國四十二年（1953年）8月所設竹埔分班、民國五十一年（1962年）8月所設的仁光分班與民國五十五年（1966年）8月所設的後宅分班{{r|鄭}}{{r|史}}。上述該校後來分別獨立成坔頭港國小{{NoteTag|1=日昭和十六年（1941年）改稱「番子厝公學校河南分教場」，戰後變為河南國小{{r|史}}，於2008年更名為「坔頭港國小」<ref>[http://www.libertytimes.com.tw/2008/new/may/15/today-education3.htm 自由電子報——鹽水鎮河南國小將正名] {{webarchive|url=https://web.archive.org/web/20121104150939/http://www.libertytimes.com.tw/2008/new/may/15/today-education3.htm |date=2012-11-04 }}</ref>}}、大豐國小、竹埔國小、仁光國小與文昌國小，但大豐國小因少子化等因素，於民國九十一年（2002年）8月1日變回「歡雅國小大豐分校」，而在民國九十五年（2006年）8月1日時被裁校{{r|鄭}}{{r|史}}。

== 文化資產 ==

{{main|鹽水歡雅國小原大禮堂及時鐘座}}
歡雅國小的大禮堂為一層樓的磚造建築，屋頂則是木構屋架，興建於1956年{{r|鄭}}。時鐘座則建於1952年，是戰後第七屆校友所捐贈{{r|鄭}}。
