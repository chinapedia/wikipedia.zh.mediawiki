'''[[臺北市|臺北市]][[文山區|文山區]]辛亥國民小學'''（英語：Taipei Municipal Xinhai Elementary School），簡稱'''辛亥國小'''，亦稱「辛亥樂園」（英語：Xinhai Paradise）或「辛亥學習樂園」，民國81年正式設校，位於台灣臺北市文山區辛亥路四段103號，由[[臺北市政府教育局|臺北市政府教育局]]管轄。

== 校史沿革 ==

* 1986年9月，陳文欽校長奉派為籌備處主任
* 1992年6月，歷經三次變更設計、徵圖，經過6年的興建，校舍終於完工。
* 1992年8月，陳文欽校長奉派為第一任校長，正式招收新生，並由萬芳國小與興隆國小撥入二至五年級學生，全校共13班，學生共286人。
* 1995年8月，董素貞校長奉派為第二任校長。
* 1996年8月，成立「辛亥國小附設幼兒園」，招收兩班幼生。
* 1996年9月，向市府停管處爭取回饋款施作「操場整體景觀工程」。
* 1998年6月，操場工程完工，舉行「教學環境落成啟用典禮」，市長陳水扁應邀觀禮、剪綵。
* 2000年8月，邱春堂校長遴選為第三任校長。
* 2002年9月，建立學校願景：愛、學習、傳承。
* 2002年11月，舉行創校十週年校慶系列活動，包十週年紀念校徽徵選、才藝表演晚會、出版紀念專刊等活動。
* 2003年8月，發展本校特色課程：藝術與人文、深耕閱讀、武術教學。
* 2004年8月，邱春堂校長連任第四任校長。
* 2008年3月，操場設施改善工程完工，跑道更換新式橡膠毯。
* 2008年8月，修金莒校長遴選為第五任校長。
* 2009年4月，結合廣達文教基金會辦理「辛亥藝想世界」藝術展。
* 2009年8月，完成東北棟廁所改善工程提供完善舒適的如廁環境。
* 2010年1月，參加教育部教師專業發展評鑑計畫。
* 2010年3月，「友善校園，亮麗圍籬」專案整修圍牆，工程融入視覺藝術元素，榮獲銀質獎。
* 2011年1月，武術隊參與中華民國建國百年新版國歌拍攝。
* 2011年5月，榮獲WHO CCCSP臺灣國際安全學校認證。
* 2012年3月，圖書館整修落成，成立五星級書福館，優化閱讀環境。
* 2012年8月，修金莒校長連任第六任校長。
* 2014年8月，李三煌校長遴選為第七任校長。
* 2014年12月，與苗栗縣銅鑼鄉銅鑼國小進行校際交流。
* 2015年1月，參加教育部推動行動學習計畫。
* 2015年8月，申辦教學輔導教師設置方案、參加臺北市E酷幣群組學校、續辦多年期教師專業發展評鑑。
* 2015年9月，學校廊道優質化工程竣工。
* 2015年10月，完成校園環境整修工程，操場更新為壓克力球場。
* 2015年12月，臺北市教育111標竿學校認證。
* 2016年4月，2016臺北世界設計之都-設計家聚落示範學校揭牌營運。
* 2016年5月，與花蓮縣萬榮鄉西林國小進行校際交流。
* 2016年10月，全校電源改善工程第一期(西棟與北棟)竣工。
* 2017年3月，建置資訊系統主機房及普通教室安裝冷氣。
* 2017年8月，申辦臺北市行動學習智慧教學學校、申辦創新實驗教育，英語二班三組。
* 2017年10月，全校電源改善工程第二期(東棟與南棟)竣工
* 2017年12月，日本福岡縣大木町師生蒞校進行校際交流活動。
* 2018年7月，申辦2018海外華裔青年英語服務營。
* 2018年8月，陳弘偉校長遴選為第八任校長。 附設幼兒園增設一班二歲專班。
* 2018年11月，雲林縣重光國小蒞校進行校際交流活動。
* 2018年12月，日本福岡縣大木町師生再次蒞校進行校際交流活動。

== 學校願景 ==

* 愛(To Love)、學習(To Learn)、傳承(To leave a legacy)

== 校歌 ==

* 巍巍辛亥，源遠流長。  青山環抱，鳥與花香。  同窗友好，師長慈祥。  晝夜砌磋，晨昏歡暢。  能歌善舞，絃歌攸揚。  五育兼修，志氣高昂。  精誠勤儉，愛國愛鄉。  辛亥精神，日月同光。

== 校徽 ==

* 緣起：校徽設計於創校10周年校慶，所辦理校徽徵選活動的首獎,原始的設計者為:第11屆畢業生何書楷，並由美勞老師修改後，始成現今的版本。
* 設計理念：小學是充滿著創意與童趣的學 習環境，辛亥的小朋友，就如同Q版卡通造型的紅色愛心，長著天使般的翅膀，充滿著「熱情」 與「活力」。他們喜歡嘗試新的事物，善用資訊科技與團隊合作的精神來學習。校徽設計的主體，就是兼具造型感與創造力的愛心，而且是「擁有翅膀的愛心」！辛亥小朋友的圖像，就是有愛心的天使，積極快樂的遨翔在辛亥學習樂園中。
