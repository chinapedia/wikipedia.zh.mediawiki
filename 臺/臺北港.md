'''臺北港'''是位於[[台灣|台灣]]北部[[新北市|新北市]][[八里區|八里區]]的一個國際[[商港|商港]]，其定位為[[基隆港|基隆港]]的輔助港，但面積比基隆港還大。其中的[[貨櫃|貨櫃]]碼頭是由[[台北港貨櫃碼頭公司|台北港貨櫃碼頭股份有限公司]]於2003年8月28日與政府簽訂50年經營權的[[BOT|BOT]]案。

== 歷史 ==

[[File:Taiwan_Provincial_Government_notice_about_Port_of_Taipei_19990521.jpg|thumb]]
因位於八里地區，故俗稱'''八里港'''。規劃階段曾稱為'''淡水新港'''，以有別與[[淡水港|淡水港]]。此港為[[劉兆玄|劉兆玄]]於1993年任[[中華民國交通部|中華民國交通部]]部長時所規劃，初期僅計畫興建兩座[[碼頭|碼頭]]供東砂北運使用。

劉兆玄於1999年任[[行政院副院長|行政院副院長]]時，將淡水新港變更為綜合性商港並更名為'''臺北港'''<ref>[http://subtpg.tpg.gov.tw/og/image2.asp?f=0880168820001J016 公告淡水港自即日起更名為「臺北港」]，臺灣省政府公報</ref>。原由中華民國交通部基隆港務局管理，2012年3月1起由[[臺灣港務公司|臺灣港務公司]][[基隆港務分公司|基隆港務分公司]]管理。

===發展沿革===
*1993年淡水港第一期工程開工，於民國87年（1998年）底完工。
*1995年政府推動「[[亞太營運中心|亞太營運中心]]計畫」之「海運中心計畫」，同意將淡水港發展定位為基隆國際商港之輔助港。
*1999年「淡水港整體規劃及未來發展計畫」及第二期工程第一個五年計畫（1996年～2001年），經[[行政院|行政院]]核定實施。
** 5月21日，[[臺灣省政府|臺灣省政府]]公告淡水港更名為「臺北港」。
*2002年，臺北港第二期工程第二個五年計畫（2002年～2006年）經行政院核定實施。
*2004年，交通部公告指定臺北港為基隆國際商港之輔助港。
*2005年5月25日，臺北港獲行政院核定籌設自由貿易港區，申請設置範圍為79[[公頃|公頃]]土地（包括散雜貨儲運中心與油品儲運中心），規劃經營業務型態以物流產業為主。
*2009年2月18日，臺北港貨櫃儲運中心開始營運。
*臺北港第二期工程第三個五年計畫（2007年～2011年），目標年定為2011年。
*臺北港第三期工程計畫（2012年～2021年）及遠期發展計畫（2022年以後），並視未來海運發展需求及民間投資意願，全面進行檢討及推動。

== 地理與環境影響 ==
[[File:台北港.jpg|thumb]]
=== 位置 ===
臺北港位於[[淡水河|淡水河]]河口西側，臨[[台灣海峽|台灣海峽]]，南有[[觀音山_(新北市)|觀音山]]為屏障，港區南側並與遊樂區[[八仙海岸|八仙海岸]]相鄰。東距基隆港34[[浬|浬]]，南距[[台中港|台中港]]87浬，西距[[福州|福州港]]134浬。
=== 海岸線影響 ===
==== 淡水河口 ====
淡水河是北台灣第一大河，而1927年到1948年左右，淡水河口及八里地區之海岸線呈向外擴張，淡水河之沈積物在河口地區偏淤積，致使海岸線往外擴張，且飛砂形成高約18公尺長3公里平行海岸沙丘。<ref name="海岸輸砂">{{cite news|url= http://www.ihmt.gov.tw/periodical/pdf/S962240.pdf   |title=林柏青等《95年淡水河河口及附近海岸輸砂之研究》交通部運輸研究所&中央大學}}</ref> 由林雪美1996年之研究可知，淡水河口海岸屬風浪營力型或風浪、逕流及潮汐三個營力之中混合型，淡水河口地形及輸砂機制之分析應考慮風浪、逕流及潮汐之影響。

但於1964年石門水庫完工後，大量河源泥沙被攬阻，河口沈積物之供需平衡遭到破壞，致使八里區海岸線向內陸退縮。比對1948年與1978年的航空圖，可發現30年間八里區的海岸線普遍退縮達180公尺，河口的挖子尾附近最大退縮量則達320公尺。1986年翡翠水庫完工，淡水河口沖積平衡更加受到衝擊，比對1978年與1992年，八里污水處理廠附近海岸退縮達150公尺，挖子尾附近海岸也退縮達200公尺。<ref name="海岸輸砂"/>

1993年淡水河南岸的臺北港開始興建，臺北港的北防波堤凸出海岸1.6 km,幾與淡水河口北側之突岬齊平。所造成突堤效應阻斷淡水河口沙源之往南移行，而臺北港南側防波堤海岸形成嚴重侵蝕，淡水河口與北防波堤間沙灘浮現，河砂淤積且海床趨於淺化。淡水河口之河道左岸挖子尾沙嘴附近，淤積日漸嚴重， 若強颱加上與滿潮時間一致時，颱風暴潮可能導致洪水無法宣洩，淡水河沿岸及大臺北地區形成水患。<ref name="海岸輸砂"/>
==== 臺北港港址海岸 ====
1986年建港以前，此處為侵蝕最為嚴重之海岸，建港後1996年至2001年間呈淤積性海域，年平均淤積量約155.8萬立方公尺。
==== 臺北港以南至林口電廠海岸 ====
築港前相較於其他區域穩定海岸侵淤量不大，築港期間呈侵蝕現象，防波堤完工後亦呈淤積。

== 臺北港客輪碼頭 ==
海峽兩岸自開放直航以來，經貿往來與觀光交流日益密切，商務、旅遊人次逐年增加，推升雙邊運輸的需求性及必要性，因此奉交通部指示臺北港從2013年起開闢「臺北港-平潭」航線。
* 2013年10月9日，「海峽號」首發通航。<ref>[http://www.chinatimes.com/newspapers/20131010000749-260108 福建台北港 海峽號開航《中時電子報》]</ref>
* 2014年5月27日，「麗娜輪」加入營運。
* 2014年8月27日，旅客入出境總人數已達到5萬2,432人。<ref>[http://www.chinatimes.com/realtimenews/20140909005134-260405 台北港平潭航線 累積破5萬人《中時電子報》]</ref>

== 台北港貨櫃碼頭 ==
*台北港目前是台灣第一個、也是投資金額最大的港埠BOT建設，2009年3月正式啟用。<ref>{{Cite web |url=http://crazeroost.pixnet.net/blog/post/24483418 |title=台北港正式啟用，航商年省40億 |accessdate=2017-05-27 |archiveurl=https://web.archive.org/web/20170107014432/http://crazeroost.pixnet.net/blog/post/24483418 |archivedate=2017-01-07 |deadurl=yes }}</ref>
*目前台北港貨櫃碼頭是由台北港貨櫃碼頭股份有限公司經營。
*台北港貨櫃碼頭公司是由[[長榮海運|長榮海運]]、[[萬海航運|萬海航運]]與[[陽明海運|陽明海運]]合資籌組，其持股比例分別為：
**長榮海運 50%
**萬海航運 40%
**陽明海運 10%

*當年臺北港BOT案原本定調是國際競標，而且外資可持有51%的股份，於是各國廠商齊聚台灣付出大量經費請人研究並評估此專案，找出最佳標金後投標。但在開標前一週，政府突然宣布外資僅能持股49%，引發外資不滿而撤資。最後該BOT案由台北港貨櫃碼頭公司拿下。
