'''臺華輪'''是[[台灣|台灣]]本島與[[澎湖|澎湖]]之間的交通船，固定航線是[[高雄港|高雄港]]和[[馬公港|馬公港]]，營運者是[[台灣航業|台灣航業]]<ref>{{cite news|url=http://dailynews.sina.com/bg/tw/twlocal/cna/20090620/2143373342.html|title=蓮花颱風來襲 臺華輪停航班機也受影響|publisher=中央社}}</ref>。除了固定航班以外，但近年來臺華輪也行駛[[高雄市|高雄]]再經由[[馬公市|馬公]]通往[[廈門|廈門]]和[[小三通|小三通]]以及其他包船的航行路線<ref>{{cite news|url=http://www.nownews.com/2009/04/20/11534-2439222.htm|title=臺華輪首航廈門　滿載近千名陸客|publisher=NowNews}}</ref><ref>{{cite news|url=http://www.haixiainfo.com.tw/44603.html|title=臺華輪轉型 包船走高澎廈|publisher=聯合報}}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

==臺澎海運簡史及臺華輪簡介==
臺澎間公營海運:

# 民國42年3月「第三海盛輪」於東吉海域觸礁沉沒，20餘人罹難。民國42年4月同事故公司「聯盛輪」於東吉海域沉沒，48人罹難失蹤。民國45年11月「第二光盛輪」於西吉海域沉沒。
# 政府鑒於民營客輪事故頻仍，指派台灣航業調派C型輪「宜蘭」、「恆春」2艘進行逐日對開，並禁止民營載客。
# 民國48年新造「澎湖輪」進行營運，專行駛高雄馬公的客輪，820.76總噸，載重427.17噸，速率14節，1959年於日本完工，載客314人。
# 民國60年新造2千噸級客船“臺澎輪”，經營船速19節，載客數620。
# 民國78年新造8134噸臺華輪，總載客數1150人，經濟航速約20節，並具備駛上駛下功能，屬客貨車載運船（Roll-on/Roll-off Ship，RoRo Ship），可載運[[遊覽車|遊覽車]]、雜貨及些許貨櫃。
# 1998年台灣航業[[民營化|民營化]]後，泛公部門持股共約4成，目前為政策性補貼台灣航業經營高雄至澎湖航線營運。

==台華輪==
民國七十八年九月（1989年）於日本長崎建造竣工，耗資新台幣七億二千餘萬元。船長120公尺，船寬19.30公尺，總噸位8134噸，航速22浬，載客量1150人，載車能量小客車約100部（可兼載高層大客車、大貨車），全船中央系統冷氣、音響，配備當時之最新科技導航及避碰航儀，並裝置遠洋航行之穩定平衡翼。<ref>{{Cite web |url=http://www.tnc-kao.com.tw/about.aspx| title =台華輪簡介}}</ref>
==新台華輪==
民國92年（2003年）開始倡議更新台華輪，原議以舊船代航、台航易產置產等方式進行改善，後於民國95年10月開始由交通部前高雄港務局辦理「臺華輪汰舊換新計畫」委外研究案，至行政院於103年11月14日同意全額由中央預算補助。但經多次設計案討論（三體輪船、單體輪船、承載量、經濟性等）<ref>{{Cite web |url=https://event.penghu.gov.tw/taihwa/home.jsp?id=3&act=view&dataserno=201511050007|title=新臺華輪規劃設計案成果報告書}}</ref><ref>{{Cite web |url=https://ladopost.com/newsDetail1.php?ntId=1&nId=1255|title=澎湖人看膩了每逢選舉年就會出現的「新臺華輪」}}</ref>，於2019年1月20日，賴峰偉縣長裁示新台華輪船體將回歸單胴體船型設計。規畫建造1萬噸級客貨兩用新船，單胴體鋼船、船長136米、船寬18.5米、最高航速27節、載重量1250噸、載客數720人、載運轎車84輛、摩托車44部、自行車20部、10呎貨櫃32只、兼載高層大客車、大貨車。建造總經費約17.3億元，預定2022年完成建造並正式營運，屆時高雄至馬公航程將縮短為3小時。<ref>{{cite news|url=https://www.chinatimes.com/newspapers/20190120000567-260107|title=台華輪加速更新 4年內正式營運|publisher=中時電子報}}</ref>

==新臺華輪爭議風波==
原本在2018年12月新任縣長賴峰偉先生裁定指示新臺華輪船體回歸最原本簡單的單胴體的客貨雙用途船型設計，但是經過議會的議員審議質疑以及各方意見爭歧「安全，快速，舒適，便捷」各縣議員糾結前述的新船標的到底該以何項為主而和縣府爭執不下陷入多年的膠著。

原本由中央主管交通部於2006-2010年間2度規劃，經濟營運航速20節、總噸位數6000噸，但不符地方各界期待，遂改由地方政府接手，卻歷經2任縣長王乾發9年、陳光復4年，糾結在快速與安全需求外，還有營運可行性，遲遲未能實質新建，導致多年下來仍是「紙上談兵論兵」不見實際建造的階段。

而新任縣長賴峰偉上任後決定以王前縣長當初所規劃的規格略作更改並且宣稱新船可以更舒適更具便捷快速性，由馬公往返高雄只需要3小時即可；原本於2014年議會核定18億經費因為新船的建造費用粗估可能到23億左右而拜會總統府秘書長-陳菊女士，請其幫忙協助爭取經費。

而由於新船的設計及經費上的評估暴漲讓議會議員對此十分不能理解且各個議員的想法每個都不同，因此議會曾要求縣府應該另外「專案報告」，但是顯然的議會對於縣長及縣府團隊仍有諸多疑問及質疑，議會主席副議長-陳雙全先生當場裁示『縣府應該要和中央盡全力協調由中央負責接手建造規劃營運，若是中央不同意則於5月份再進行審議』

於似乎全案又一切回歸到原點，新臺華輪能否於2022如期竣工下水營運仍然是個未知數！<ref>[https://tw.appledaily.com/new/realtime/20190223/1522621/  新台華輪建造案「擱淺」 澎議會決議退回中央]</ref>
