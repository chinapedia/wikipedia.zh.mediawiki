'''台南孔子廟明倫堂'''，為[[臺灣|臺灣]][[明鄭時期|明鄭時期]]時代的最高學府，制度地位基本上與[[中國|中國]]的[[國子監|國子監]]、[[太學|太學]]或[[朝鮮半島|朝鮮半島]]的[[成均館|成均館]]相當。位於[[臺灣|臺灣]]的[[臺南市|臺南市]]，即今日[[國定古蹟|國定古蹟]][[臺南孔子廟|臺南孔子廟]]左側的「明倫堂」。

== 建築 ==
* [[明倫堂|明倫堂]]：建於[[台南孔子廟|文廟]]東側，為教授經史學問的講堂。

== 成立歷史 ==
西元1665年，[[台灣|台灣]]處於[[鄭經|鄭經]]主政時期，大臣[[陳永華_(明朝將領)|陳永華]]以「十年成長，十年教養，十年成聚」的復國計畫，於[[承天府|承天府]][[寧南坊|寧南坊]]興建[[大成殿|大成殿]]、祭祀[[孔子|孔子]]，又於其左側建造了[[泮宮|泮宮]]「明倫堂」。1683年，台灣明鄭滅亡後隨之停擺並消失。1685年，[[臺灣清治時期|清朝]]官員[[周昌|周昌]]與[[蔣毓英|蔣毓英]]將國子監改為[[臺灣府學|臺灣府學]]。

== 當時制度 ==
明鄭時期的[[科舉制度|科舉制度]]由上而下可分為[[院試|院試]]、[[府試|府試]]以及[[州試|州試]]三級考試。男童滿八歲入[[社學|社學]]接受初等的[[經學|經]][[史學|史]]教育。陸續通過州試、府試以及院試者，則取得[[廩膳生|廩膳生]]的資格，並可進入國子監接受高等經史教育。國子監每月授課、考試一次，並依照成績優劣發放[[俸祿|廩膳]]（零用金）。每三年舉行[[大試|大試]]，選拔優秀的國子監儒生進入朝廷，填補[[六官都事|六官都事]]的職缺。<ref>《[[臺灣通史|臺灣通史]]》，卷11，〈教育志〉。</ref>

國子監的官職分為「[[禮部尚書|禮官]]」、「學院」以及「國子監助教」，後二者分別由陳永華以及[[葉亨|葉亨]]擔任。其中，禮官為主辦全國教育的行政機關；學院為掌握教學與考選的主事官。一掃[[明朝|明朝]]末年的[[八股|八股]]取士，實行真正的取才，使得立國不久的台灣也能培養出有真才實學的學者；如[[清朝|清朝]]初年全國性的[[進士|進士]][[陳夢球|陳夢球]]以及[[舉人|舉人]][[蘇峨|蘇峨]]皆為所培育出的人才。<ref>《台灣教育400年》，39頁</ref>
