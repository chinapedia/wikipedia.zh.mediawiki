'''臺鐵爆炸案'''，又稱'''臺鐵1258次區間車爆炸事故'''（簡稱'''1258次區間車爆炸案'''、'''松山車站爆炸案'''，官方名稱'''臺灣鐵路管理局新竹開往基隆1258次區間車爆炸案'''）是一起[[臺灣鐵路|臺灣鐵路]][[公共安全|公共安全]]事故，發生於2016年7月7日晚上約10點時，於即將進入[[松山車站_(台灣)|松山車站]]的[[臺灣鐵路管理局|臺灣鐵路管理局]]第1258次[[臺鐵區間車|區間車]]第六節車廂遭人放置[[爆裂物|爆裂物]]發生爆炸<ref>{{cite web|url=http://news.ltn.com.tw/news/society/breakingnews/1755636|title=台鐵車廂爆炸案 警政署長陳國恩：爆竹引起|publisher=自由時報即時新聞}}</ref>，一共造成25人輕重傷<ref>{{cite web|url=http://www.appledaily.com.tw/realtimenews/article/new/20160707/903486/|title=【更新有片】台鐵松山車站列車爆炸 21名旅客受傷|publisher=蘋果日報即時新聞}}</ref><ref>{{Cite news|url =http://www.taiwandaily.net/gp2.aspx?_p=kSF1c9zU9HR9PhKlUxzC8FWEXNpHOLAs|title =松山車站列車爆炸 2重傷23輕傷|author =|date =2016年7月8日|publisher =美洲台灣日報|language =zh-Hant|accessdate =2016年7月8日|archive-url =https://web.archive.org/web/20161011011810/http://www.taiwandaily.net/gp2.aspx?_p=kSF1c9zU9HR9PhKlUxzC8FWEXNpHOLAs|archive-date =2016-10-11|dead-url =yes}}</ref>，之後查出是炸彈犯林英昌所為。

== 事件經過 ==
事故列車為[[臺鐵|臺鐵]]1258次[[新竹車站|新竹]]開往[[基隆車站|基隆]]區間車(編組EP509+EP504)，21:52駛離[[臺北車站|臺北車站]]，表訂於21:59抵達松山車站<ref>{{Cite web|url=http://163.29.3.98/FullTimeTable/20160701/%E6%96%B0%E7%AB%B9%E2%86%92%E5%9F%BA%E9%9A%86.pdf|title=新竹→基隆|last=|first=|date=|author =|publisher=交通部臺灣鐵路管理局|accessdate=2016年7月8日}}</ref>。在列車行駛至[[營業里程|營業里程]]24K~25K間，第六節車廂EP509行李架上一個黑色行李突然爆炸<ref>{{Cite news |url =http://www.chinatimes.com/newspapers/20160708000450-260102|title =台鐵火車爆炸 25傷|author =胡欣男、陳鴻偉、林郁平|date = 2016年7月8日 |publisher = 中國時報|language = zh-Hant |accessdate = 2016年7月8日 }}</ref><ref>{{Cite news |url =http://www.appledaily.com.tw/realtimenews/article/new/20160708/903614/|title =台鐵爆炸前一刻　目擊者看到奇怪的一幕|date = 2016年7月8日 |publisher = [[台灣蘋果日報|台灣蘋果日報]]|language = zh-Hant |accessdate = 2016年7月8日 }}</ref>，造成25名乘客受傷，多人傷勢嚴重；抵達松山車站後，火勢立即被撲滅<ref>{{Cite news |url =http://news.tvb.com/instant/577e6d166db28c4a170afe34/|title =台北松山車站火車爆炸 多達20人傷|author =|date = 2016年7月8日 |publisher = 無綫新聞|language = zh-Hant |accessdate = 2016年7月8日 }}</ref>，警消隨即派出多台水箱車及多台救護車抵達現場搶救<ref>{{Cite news|url=https://video.udn.com/news/522304|title=台鐵車廂爆炸 逾20名乘客受傷|last=|first=|date=|work=聯合影音|accessdate=2016年7月8日}}</ref><ref>{{Cite news |url =http://news.ifeng.com/a/20160708/49315420_0.shtml|title =台北列车爆炸20余名旅客受伤 警方称发现长形爆竹|author =|date = 2016年7月8日 |publisher = 鳳凰網|language = zh-cn |accessdate = 2016-07-08 }}</ref>。

== 傷亡情況 ==
根據[[臺北市政府消防局|臺北市政府消防局]]統計，傷患人數達到25人，共13男12女，共計有25人受傷，分別護送至[[臺北市立聯合醫院|臺北市立聯合醫院忠孝院區]]6人、[[臺北醫學大學附設醫院|臺北醫學大學附設醫院]]5人、[[國泰醫院|國泰醫院]]2人、[[三軍總醫院|三軍總醫院內湖總院]]4人、[[三軍總醫院松山分院|三軍總醫院松山分院]]5人、[[長庚醫療財團法人|臺北長庚醫院]]3人<ref>{{Cite news|url=http://www.appledaily.com.tw/realtimenews/article/new/20160707/903539/|title=松山車站爆炸案　傷者名單出爐|last=|first=|date=|work=蘋果日報|accessdate=}}</ref>。25名傷患中，包含1名38歲越南籍女子輕傷，另有1名17歲呂姓男子和24歲余姓男子，涉案重大的55歲林姓嫌犯重傷<ref>{{Cite news|url=http://www.cna.com.tw/news/firstnews/201607085004-1.aspx|title=台鐵松山車站列車爆炸 2重傷23輕傷[更新]|last=|first=|date=|work=中央社|accessdate=2016-07-08}}</ref>。

== 調查 ==
針對這起爆炸案，[[內政部警政署刑事警察局|內政部警政署刑事警察局]]、[[內政部警政署鐵路警察局|內政部警政署鐵路警察局]]以及相關單位組成「1258」專案小組<ref>{{Cite news |url =http://udn.com/news/story/10013/1815023-%E6%9D%BE%E5%B1%B1%E7%81%AB%E8%BB%8A%E5%BB%82%E7%88%86%E7%82%B8-%E8%B3%80%E9%99%B3%E6%97%A6%E5%87%8C%E6%99%A8%E5%87%BA%E9%9D%A2%E8%AA%AA%E6%98%8E|title =松山火車廂爆炸 賀陳旦凌晨出面說明|author =許家瑜|date = 2016年7月8日 |publisher = 聯合報|language = zh-Hant |accessdate = 2016年7月8日 }}</ref>。檢警在現場發現到一個長約20公分的鞭炮碎片，並根據遺留的白色粉末，懷疑可能是人為引爆[[鞭炮|鞭炮]][[火藥|火藥]]所致開<ref>{{Cite news |url =http://dailynews.sina.com/bg/tw/twpolitics/phoenixtv/20160707/09597409379.html|title =台鐵松山火車站一列車發生爆炸 警方：不排除恐襲|author =|date = 2016年7月8日 |publisher = 鳳凰網|language = zh-Hant |accessdate = 2016年7月8日 }}</ref>，且不排除是[[恐怖攻擊|恐怖攻擊]]。目擊者稱，看到1名年約40歲男子持長條物進入車廂後，隨即把不知名物體放在車廂後離開；在現場的乘客也提及，有看到1名男子推著黑色行李箱進入車廂，隨即離開<ref>{{Cite news |url =http://www.cna.com.tw/news/firstnews/201607070609-1.aspx|title =松山火車站列車爆炸 疑1男涉案|author =謝佳璋|date = 2016年7月8日 |publisher = 中央社|language = zh-Hant |accessdate = 2016年7月8日 }}</ref>。刑事警察局偵查第五大隊第一隊隊長李子文現場受訪表示，在列車車廂上找到破裂鋼管爆裂物及黑色帆布背包，初步判定就是爆炸點。

李子文說，爆炸意外發生後，相關人員進入案發的第六車廂檢視，在座位上發現寬6公分、長15公分破裂鋼管，地上也有鋼管碎裂痕跡，由於找不到任何開關，初步研判應是點火式爆裂物；而一旁則有黑色帆布背包，根據背包損壞情況判定判斷，破裂鋼管可能是放在背包裡爆炸。[[臺灣臺北地方檢察署|臺灣臺北地方檢察署]]則指派重大刑案專組主任檢察官[[騰治平|騰治平]]，率組內[[檢察官|檢察官]]前往現場進行了解<ref>{{Cite news |url =http://news.ltn.com.tw/news/society/breakingnews/1755689|title =台鐵車廂爆炸 林全證實：此為惡意犯案|author =陳薏云|date = 2016年7月8日 |publisher = 自由時報|language = zh-Hant |accessdate = 2016年7月8日 }}</ref><ref>{{Cite news |url =http://news.ltn.com.tw/news/society/breakingnews/1755688|title =就是它！台鐵爆炸案肇禍鋼管 警籲民眾指認|author =陳恩惠|date = 2016年7月8日 |publisher = 自由時報|language = zh-Hant |accessdate = 2016年7月8日 }}</ref>。

警方於8日下午表示於列車廁所採集之[[DNA|DNA]]與一重傷病患林英昌吻合，涉有重嫌，而該男子早前亦受到多名目擊者指認<ref>{{Cite news |url =http://udn.com/news/story/1/1815909|title =台鐵爆炸案採到重傷林姓男DNA 檢警：他涉有重嫌|date = 2016年7月8日 |publisher = [[聯合新聞網|聯合新聞網]]|language = zh-tw |accessdate = 2016年7月8日 }}</ref><ref>{{Cite news |url =http://www.cna.com.tw/news/firstnews/201607085019-1.aspx|title =台鐵火車爆炸案 55歲林姓傷者涉重嫌|date = 2016年7月8日 |publisher = [[中央通訊社|中央通訊社]]|language = zh-tw |accessdate = 2016年7月8日 }}</ref>。

根據警方15小時的抽絲剝繭後，認定已重傷昏迷的55歲乘客林英昌有重大嫌疑。由於該人居無定所，8日晚間，檢警終於在[[南投縣|南投縣]][[仁愛鄉|仁愛鄉]]山上尋獲廂型車，車內發現林英昌的遺書，內容表明因罹患[[扁桃腺癌|扁桃腺癌]]多年，久病厭世想尋短<ref>{{Cite news |url =https://tw.news.yahoo.com/%E5%8F%B0%E9%90%B5%E7%88%86%E7%82%B8%E6%A1%88-15%E5%B0%8F%E6%99%82%E9%8E%96%E5%AE%9A%E4%B8%BB%E5%AB%8C-55%E6%AD%B2%E7%94%B7%E8%87%AA%E6%AE%BA%E5%BC%8F%E7%8A%AF%E6%A1%88-215005021.html|title =台鐵爆炸案 15小時鎖定主嫌 55歲男自殺式犯案|date = 2016年7月9日 |publisher = [[中國時報|中國時報]]|language = zh-tw |accessdate = 2016年7月9日 }}</ref>。林英昌經法官於2016年7月19日晚間6時40分裁定羈押不禁見，他在法庭上聲稱，當天把陪伴的狗留給早餐店老闆娘後，就從台中北上，原本是決定在新竹站人多時引爆炸彈，但一時下不了手，輾轉搭上往台北的通勤電聯車才到北部犯案。<ref>[http://news.ltn.com.tw/news/society/breakingnews/1768167 台鐵爆炸案 林英昌羈押禁見]</ref><ref>[http://udn.com/news/story/7315/1833919-%E5%8F%B0%E9%90%B5%E7%82%B8%E5%BD%88%E5%AE%A2%EF%BC%9A%E5%8E%9F%E9%8E%96%E5%AE%9A%E6%96%B0%E7%AB%B9%E7%AB%99-%E7%9C%8B%E5%88%B0%E5%B0%8F%E5%A5%B3%E5%AD%A9%E4%BD%9C%E7%BD%B7 台鐵炸彈客：原鎖定新竹站 看到小女孩作罷]</ref>

==處理==
[[行政院院長|行政院院長]][[林全|林全]]凌晨前往各家醫院一一探視傷患，並表示將提高大眾交通系統的維安層級，請民眾不必過度恐慌<ref>{{Cite news |url =http://www.chinatimes.com/realtimenews/20160708000156-260401|title =松山爆炸 林全：提高運輸系統維安民眾勿擔憂|author =呂雪彗|date = 2016年7月8日 |publisher = 中國時報|language = zh-Hant |accessdate = 2016年7月8日 }}</ref>。[[臺北市政府社會局|臺北市政府社會局]]也派人前往各家醫院探視傷者，並代表市府發放每位傷者慰問金5000元<ref>{{Cite news |url =https://tw.news.yahoo.com/5000-163108930.html|title =區間車爆炸/柯文哲：傷者每人5000慰問金|author =政治中心|date = 2016年7月8日 |publisher = 民報 |language = zh-Hant |accessdate = 2016年7月8日 }}</ref>。
