{{台灣歷史}}

'''臺灣學生運動'''為發生在[[臺灣|臺灣]]的[[學生運動|學生運動]]，最早可以追溯到[[台灣日治時期|日治時期]]。日治時期出現如今現代意義的學校，才有學生對於校園內事務或者是社會現況不滿而出聲抗議。通常學生運動中參與的學生多為大學生，不過也有一些學生運動以中學生為主體。

== 日治時期 ==
日治時期，台灣學生運動在島內及海外皆有發端。在海外的部分，1901年開始，已有台灣學生至[[日本|日本]]留學的風潮。其中一群[[東京|東京]]的留學生，受到[[美國|美國]]總統[[伍德羅·威爾遜|威爾遜]]提倡的[[民族自決|民族自決]]影響，又有[[辛亥革命|辛亥革命]]、[[朝鮮獨立運動|朝鮮獨立運動]]、[[俄國十月革命|俄國十月革命]]和[[愛爾蘭獨立運動|愛爾蘭獨立運動]]的刺激，可能變成[[台灣民族主義|台灣民族運動]]的先驅<ref>{{cite book|author=林仁傑|title=一段跨時代的故事：台灣學生運動史研究（1920-1994）|year=2004}}</ref>。

日治初期，台灣缺乏高等教育學校，台灣僅有士紳有能力留學日本，而留學生受到[[大正民主|大正民主]]影響獲得啟蒙，爾後回台留學生成立[[台灣文化協會|台灣文化協會]]，1919年，[[明石元二郎|明石總督]]公布[[台灣教育令|台灣教育令]]，改制師專為師院，台灣學生在1920年代得以進入高教體系，[[田健治郎|田健治郎]]上任總督後積極推行[[內地延長主義|內地延長主義]]，期望造就台灣高等教育人才，然而對台灣人政治上的壓抑，使得這些青年結合文化協會的運動，成為日後走向反抗日本威權的種子。

*'''[[台北師範學校第一次騷亂事件|台北師範學校第一次騷亂事件]]'''：1922年2月台北師範學生因為靠右行走遭日警警告，日警巡佐並到師院要求校方訓誡，導致學生緊閉校門聚集三百多人抗議。官方旋即逮捕45名學生，指為台灣文化協會[[蔣渭水|蔣渭水]]煽動之暴亂，之後，雖經台人及輿論強烈抗議而未起訴，卻使日方將矛頭指向台灣文化協會。<ref name="movement">{{cite book|title=日據時期台灣學生運動1913-1945 |publisher=時報文化|author=藍博洲|date=1993-04-10}}</ref>

*'''[[台北師範學校第二次騷亂事件|台北師範學校第二次騷亂事件]]'''：因為畢旅地點，台灣學生不滿日本學生以少數決，最後在畢旅當天集體罷課，由於文協介入協助，許多學生最後遭到學校退學，甚至演變成為兩百多學生絕食抗議，當局雖虛以尾蛇，後仍有三十多名學生遭到秋後算帳，退學處分。遭退學學生部分經文協協助資助，得以留學東京、[[廈門|廈門]]、[[上海|上海]]等地，並為日後海外[[反日|反日]]思潮之中堅。<ref name="movement"/>

*'''[[台中師範學校騷亂事件|台中師範學校騷亂事件]]'''：1928年，新任舍監小山重郎在宿舍聽到學生以台灣話交談，因校規中禁止說台語，因此犯規學生當場被予以責罰與毆打，次日朝會時小山大發狂言指:「台灣話是清國奴話」；更說:「誰愛說台語就滾回支那好了」，甚至對日本學生說，聽到誰講台語就揍他。引發學生抗議，並於日後成為[[反日|反日]]支持的重要基石。<ref name="movement"/>

== 1945年～1950年 ==
*[[四六事件|四六事件]]發生在1949年4月6日，起因是警察在3月20日取締學生單車雙載，引發以[[國立臺灣師範大學|臺灣省立師範學院]]為主的學生，串聯[[國立臺灣大學|國立臺灣大學]]抗議，至4月6日時軍警包圍師院以及台大宿舍，逮捕當時學生運動的領袖，其中[[鄭鴻溪|鄭鴻溪]]為已加入共產黨之學生領袖僥倖逃脫，後擔任中共統戰部局長，此事件造成日後[[台灣省工委|台灣省工委]]組織得以進一步擴大，如[[葉城松|葉城松]]、[[周慎源|周慎源]]、[[張璧坤|張璧坤]]、[[鄭文峰|鄭文峰]]等皆先後加入台灣省工委。兩校校長處理的態度深深影響各自日後的發展，時任師院校長[[謝東閔|謝東閔]]高度配合軍警，整肅學生名單，重新登入學籍，造就日後保守校風；時任台大校長[[傅斯年|傅斯年]]則極力保全學生，更表示「若有學生流血，我要跟你拚命！」，使得台大受創較輕，並維持自由校風。有觀點認為：四六事件是台灣[[白色恐怖|白色恐怖]]的濫觴。

== 1951年～1979年 ==

*[[臺大哲學系事件|臺大哲學系事件]]發生在1972年12月到1975年6月之間，由[[中國國民黨|中國國民黨]]特工系統以「反共」之名，對[[國立台灣大學|國立台灣大學]][[哲學系|哲學系]]內自由派學者進行整肅的一連串-{行動}-，並導致台大哲學系教職員包括[[趙天儀|趙天儀]]、[[陳鼓應|陳鼓應]]、[[王曉波|王曉波]]、[[陽斐華|陽斐華]]、[[胡基峻|胡基峻]]、[[李日章|李日章]]、[[陳明玉|陳明玉]]、[[梁振生|梁振生]]、[[黃天成|黃天成]]、[[郭實瑜|郭實瑜]]、[[鍾友聯|鍾友聯]]、[[黃慶明|黃慶明]]及美國籍客座教授[[Robert_Martin|Robert Martin]]遭解聘，台大哲學研究所更破天荒停止招生一年。

*'''[[保釣運動|保釣運動]]中的學生運動:'''1970年11月17日，[[美國|美國]][[普林斯頓大學|普林斯頓大學]]的台灣留學生於開始組成「保衛釣魚台行動委員會」，表示「反對美日私相受授」、「外抗強權，內爭主權」，一方面抨擊美國與日本，另一方面也要求[[中華民國政府|中華民國政府]]應該力爭主權。1971年1月29日，大批台灣留美學生在[[聯合國|聯合國]]總部外面示威，高呼「保衛釣魚台」。

== 1990年代 ==

*'''[[野百合學運|野百合學運]]'''，發生時間自1990年3月16日至3月22日，又稱「'''三月學運'''」或「'''台北學運'''」。在該次運動中，人數最多時曾經有將近6000名來自[[台灣|台灣]]南北各地的[[大學生|大學生]]，集結在[[中正紀念堂|中正紀念堂]]廣場上[[靜坐_(示威)|靜坐]][[示威|抗議]]，他們提出「解散[[國民大會|國民大會]]」、「廢除[[臨時條款|臨時條款]]」、「召開[[國是會議|國是會議]]」、以及「政經改革時間表」等四大訴求。這不但是[[中華民國政府|中華民國政府]]遷台以來規模最大的一次[[學生運動|學生抗議行動]]，同時也對台灣的[[民主政治|民主政治]]有著相當程度的影響。在該次學生運動後，時任[[中華民國總統|總統]][[李登輝|李登輝]]一方面依照其對學生的承諾，在不久後召開國是會議，另一方面也在1991年廢除《[[動員戡亂時期臨時條款|動員戡亂時期臨時條款]]》，並結束所謂「[[萬年國會|萬年國會]]」的運作，台灣的民主化工程從此進入另一嶄新紀元。

== 2000年代 ==

* 2004年[[新野百合學運|新野百合學運]]
* 2008年[[野草莓學運|野草莓學運]]
**2008年中共海協會長[[陳雲林|陳雲林]]訪台期間，有許多社團或公民個人欲對其直接表達政治意見；然而警方維安過當，侵害到抗議團體或個人的[[人身自由|人身自由]]、[[言論自由|言論自由]]與[[集會遊行|集會遊行]]等公民權益，引起學生及[[人民團體|公民團體]]抗議，呼籲相關官員道歉或下台，並提出修改或廢除[[集會遊行法|集會遊行法]]之訴求。運動參與者一開始於[[行政院|行政院]]前靜坐表達抗議，被警方驅離之後，抗議活動轉移至[[自由廣場|自由廣場]]。野草莓運動參與者日後仍保有聯繫與行動網絡，延續至2014年[[太陽花學運|太陽花學運]]。
