'''彭三源'''，本名'''彭名燕'''，中国大陸女性作家、影视作品编剧，[[中国作家协会|中国作家协会]]会员。

== 生平 ==
1992年毕业于[[首都师范大学|首都师范大学]]教育心理系，2002年毕业于[[北京电影学院|北京电影学院]]导演进修班。

2011年3月她签约[[华谊兄弟|华谊兄弟]]并成立了自己的影视制作工作室，正式成为一名制片人。<ref>{{cite web |url=http://newspaper.jfdaily.com/xwwb/html/2011-03/09/content_527287.htm |language= |title=彭三源加盟华谊成立工作室当制片 |publisher=新闻晚报 |date=2011年3月9日 |accessdate= |deadurl=yes |archiveurl=https://web.archive.org/web/20140301055215/http://newspaper.jfdaily.com/xwwb/html/2011-03/09/content_527287.htm |archivedate=2014年3月1日 }}</ref>

她是一名[[佛教徒|佛教徒]]。

== 作品 ==
著有[[散文|散文]]集《苦水玫瑰》（[[四川文艺出版社|四川文艺出版社]]，1996年）、[[长篇小说|长篇小说]]《[[北京生活|北京生活]]》（[[作家出版社|作家出版社]]，1996年）。

=== 影视作品 ===
编剧及制作电视剧有《海棠依旧》、《九九归一》、《都市丽人行》、《青鸟的天空》、《今生是亲人》、《手心手背》、《靠近你，温暖我》、《半路夫妻》和《亲兄热弟》等。

2014年她编剧并执导首部电影《[[失孤|失孤]]》，这是[[华谊兄弟|华谊兄弟]]出品、[[刘德华|刘德华]]主演的聚焦“打拐”题材的华语电影。<ref name="聚焦">{{cite web|url=http://ent.sina.com.cn/m/c/2014-02-23/20084101198.shtml | language= | title=《失孤》聚焦打拐 刘德华踏上艰难寻子路 |publisher=新浪娱乐 |date=2014年2月23日 |accessdate= }}</ref><ref>{{cite web |url=http://www.fj.xinhuanet.com/dszx/qz/2014-03/11/c_119710359.htm |language= |title=电影《失孤》昨起泉州取景 刘德华泉州拍戏 |publisher=泉州晚报 |date=2014年3月11日 |accessdate= |deadurl=yes |archiveurl=https://web.archive.org/web/20140311075711/http://www.fj.xinhuanet.com/dszx/qz/2014-03/11/c_119710359.htm |archivedate=2014年3月11日 }}</ref>中国于2015年3月20日上映<ref >{{cite web|url=http://ent.qq.com/a/20150307/006720.htm | language= | title=《失孤》将映 刘德华演农民、遭掌掴、唱主题曲 |publisher=腾讯娱乐 |date=2015-03-07 |accessdate= }}</ref>。由她创作的同名小说《失孤》也于2015年3月出版发行。《失孤》获得[[第16届华表奖|第16届华表奖]]优秀故事片奖，她本人获得第13届[[长春电影节|长春电影节]]最佳编剧奖。
