'''彭佳慧'''（{{lang-en|'''Julia Peng'''}}，{{bd|1972年|4月20日|CatIdx=P彭}}），[[台灣|台灣]]屏東[[六堆|六堆]]之[[客家|客家]]人，[[台灣|台灣]][[女歌手|女歌手]]、作曲人及舞台劇演員。其表哥[[黃中原|黃中原]]為台灣著名音樂人兼創作歌手。其妹[[彭佳霓|彭佳霓]]為台灣[[客家電視台|客家電視台]]《客家新樂園 Hot Music》主持人，與音樂人[[謝宇威|謝宇威]]搭擋主持。彭佳慧1996年以專輯《說真心話》正式出道，低音渾厚高音充滿爆發力，是第八屆[[金曲獎|金曲獎]]最佳新人獎得主，2016年更以《大齡女子》專輯一舉奪下[[第27屆金曲獎|第27屆金曲獎]][[最佳國語女歌手獎_(金曲獎)|最佳國語女歌手獎]]。
擁有〈舊夢〉、〈相見恨晚〉、〈死心眼〉、〈聽到你一聲再會〉、〈敲敲我的頭〉、〈走在紅毯那一天〉、〈鏡子裡的我〉、〈大齡女子〉等多首膾炙人口的歌曲。由於出身自隔代教養的家庭，因此彭佳慧對社會上的弱勢家庭非常關心，成名後多次參與各項公益活動，2014年擔任家扶基金會「無窮世代」公益大使，並且認養5位貧困兒童。

== 出道過程 == 
小時候就熱愛音樂和表演的彭佳慧，在小學時期就加入了合唱團。高中畢業於[[屏東縣|屏東縣]][[國立潮州高級中學|國立潮州高中]]。高三畢業那年獨自北上參加了[[華視|華視]]舉辦的「金曲龍虎榜巨星之聲模仿大賽」，當中模仿[[江淑娜|江淑娜]]而獲得評委們的一致青睞獲得冠軍。而愛唱歌的她回到充滿熱情的南部後便在民生餐廳及PUB演唱， 當時同事包括[[倪子鈞|小馬]]和[[小鐘|小鐘]]<ref>[http://www.youtube.com/watch?v=Eck3_Nhgoqc 2013-10-05天才衝衝衝-2 ]</ref>。1995年彭佳慧（Julia）前往台北EZ5 PUB駐唱演出時，因為獨特的唱腔和舞台魅力得到[[鍾鎮濤|鍾鎮濤]]、[[趙詠華|趙詠華]]、[[李正帆|李正帆]]及巨石唱片老闆鄭柏秋的賞識，並與唱片公司簽約。

==個人生活==
2018年6月6日，彭佳慧被媒體拍到當街與精壯男擁吻，隨後在臉書宣布3月已與老公王丕仁簽字離婚，並po出2人手寫聲明稿，語重心長地說感情的事，尤其夫妻之間，更難以清楚解釋，這是在兩人溝通後所做的決定，彼此也沒有任何離婚條件。<ref>{{cite web |title=彭佳慧親筆信認了今年3月已離婚 |url=http://www.chinatimes.com/realtimenews/20180606001162-260404 |publisher=中時電子報}}</ref>

== 初長期（1996年－1998年） ==
1996年8月，Julia推出首張專輯《說真心話》，銷售達40萬張，其中主打歌〈舊夢〉更是傳唱多時。翌年在[[第8屆金曲獎|第8屆金曲獎]]上擊敗dMDM、黃名偉和黃嘉千等人獲得「[[最佳新人獎_(金曲獎)|最佳新人獎]]」，並以〈舊夢〉拿下「[[最佳作曲人獎_(金曲獎)|最佳作曲人獎]]」（李正帆作曲）。同年11月，與[[劉德華|劉德華]]合作合唱歌〈願牽你的手〉，收錄於劉德華國語專輯《因為愛》當中。

1997年4月，推出第二張專輯《彭佳慧》，賣座依舊，其中主打歌〈愛到無路可退〉獲得馬來西亞「1997中文音樂金曲獎」之「年度金曲」及「實力派女歌手」等大獎。 

1998年1月，推出第三張專輯《看穿》，以反璞歸真之自然風格呈現，創下三白金唱片之銷售佳績，其中主打歌〈聽到你一聲再會〉、〈心酸的情歌〉獲得一致的好評，同時舉辦個人10場大型校園巡迴演唱會。同年12月，推出第四張專輯《敲敲我的頭》，主打歌〈敲敲我的頭〉、〈死心眼〉、〈相見恨晚〉反應不俗，同時並首次為[[柯以敏|柯以敏]]創作〈想你想瘋了〉一曲收錄於[[柯以敏|柯以敏]]個人專輯《想你想瘋了》當中。

== 發展期（1999年－2002年） ==
1999年6月，推出首張新歌加精選輯《過程》，收錄2首新歌及18首精選舊作，其中第一主打歌〈快樂的人請舉手〉獲得[[FOX娛樂台|Channel[V]]]華語榜中榜「最具爆發力女歌手」獎項，而第二主打歌〈走在紅毯的那一天〉獲得飛碟電台「1999年度15大點播金曲」及「1999年度最佳新歌加精選輯」銀獎、Hit FM電台「1999年度十大金曲」獎項。並於同年8月在台北國際會議中心舉行「呼彭喚友」大型演唱會。

2000年1月，推出第五張專輯《First Time》，並首次與陳昇合作主打歌〈愛瘋了〉，以及首次收錄個人創作〈風流女生〉，其中〈甘願〉一曲在2001年獲得中國「喜之郎首屆音樂風雲榜頒獎盛典」之「年度十大金曲」，同時Julia在此專輯還翻唱林憶蓮的國語舊作〈聽說愛情回來過〉。同年5月，受中國大陸邀約，在北京化工大學舉辦個人首場內地校園演唱會。同年12月，推出第六張專輯《說真的》，其中第一主打歌〈一九九幾的他〉是當時還未出道的[[新加坡|新加坡]]創作歌手[[黃義達|黃義達]]的第一首創作，而第二主打歌〈回味〉是與她合作無間的音樂人[[陳國華|陳國華]]的創作，第三主打歌〈說真的〉是Julia的個人創作。

2001年，彭佳慧在中國大陸受到相當多的邀約，其中在同年3月至4月間受湖南經視台之邀赴湖南長沙舉辦「彭佳慧藝能歌友會」，以及在[[上海理工大學|上海理工大學]]舉辦個人第一場校園演唱會。同年7月，擔任[[林志炫|林志炫]]「至情至炫」中國演唱會兩場之嘉賓。同年12月，推出第七張專輯《情歌手》，其中第一主打歌〈亮晶晶〉是彭佳慧與陳國華兩人聯手作曲，第二主打歌〈情歌手〉是陳珊妮首次為她詞曲一手包辦的歌，第三主打歌〈別問我愛誰〉是電視劇《[[多桑與紅玫瑰|多桑與紅玫瑰]]》的主題曲。

2002年3月，受邀至馬來西亞雲頂星河劇場，舉辦一場三小時的演唱會。同年4月，前往深圳參加音樂風雲榜頒獎典禮，獲得2001年度「最佳女飛躍獎」。同年11月，獲頒馬來西亞第二屆金曲紅人頒獎禮之「最受歡迎現場舞台表現女歌手獎」。同年12月，推出第二張新歌加精選輯《暢情錄》，收錄4首新歌及20精選舊作，其中有2首主打歌〈自從〉、〈留住〉是[[黃韻玲|黃韻玲]]首次為她作曲的創作，另2首主打歌〈快樂群島〉、〈紅的黃的橙的灰的〉分別由新加坡音樂人[[李偲菘|李偲菘]]及[[李偉菘|李偉菘]]兩人創作，此精選輯推出後彭佳慧與[[博德曼音樂_(台灣)|BMG唱片]]合約正式約滿。

== 隱沒期（2003年－2006年） ==
與[[博德曼音樂_(台灣)|BMG唱片]]約滿之後，因為[[博德曼音樂_(台灣)|BMG唱片]]人事改組，彭佳慧就從此在歌壇銷聲匿跡，這段期間她除了在[[PUB|PUB]]駐唱及中國、新馬地區做商業演出之外，亦曾參與舞台劇的演出，其中在2004年參與音樂舞台劇《但願人長久》的演出，飾演年輕時期的[[鄧麗君|鄧麗君]]一角。

2006年2月，受邀擔任新加坡音樂人李偲菘、李偉菘之「親親菘菘音樂會」表演嘉賓。同年4月21日，宣布與從事大陸、韓國進口服裝貿易的圈外人王丕仁（Steven）結婚。

== 復出期（2007年－2012年） ==
2007年，彭佳慧與多利安娛樂簽約，並於同年9月在[[游鴻明|游鴻明]]個人創作專輯《倦鳥餘花》當中與[[游鴻明|游鴻明]]合唱〈夫妻臉〉一曲；原定進行新專輯的籌備錄製，但因簽約後發現懷孕而暫時擱置計畫。隨後於11月產下一子（王子仲，Brandon，小名：小王子）。

2008年彭佳慧簽約由[[齊秦|齊秦]]創立的虹之美夢成真經紀，6月與[[游鴻明|游鴻明]]合辦大型售票演唱會《呼彭喚游》，在台北（三場）、台中、高雄三地舉辦，並同時在演唱會中發表親自創作的新歌〈從此以後〉，2009年12月推出第八張專輯《因為…女人說》。

2011年11月28日，彭佳慧再產下雙胞女， Beverly 和 Bella，但剖腹生女後八小時，因為失血過多而癲癇症發作，一度在加護病房。

== 成熟期（2013年－至今） ==
2013年，彭佳慧参加湖南卫视《[[我是歌手|我是歌手]]》节目，進入總決賽。第一次參賽於3月15日，主持人以台灣三大Live唱將介紹彭佳慧出場，彭佳慧於現場演唱個人代表作品之一〈走在红毯那一天〉，以第三名成績順利晉級。同年8月，自組風流女生工作室，與[[金牌大風|金牌大風]]簽約，推出第九張全新專輯《醉佳時分》再戰歌壇，其中專輯首波主打〈鏡子裡的我〉宛如彭佳慧的自傳體歌，呈現彭佳慧一路走來的星路歷程。

2015年6月15日，彭佳慧再度­推出睽違五年半後的全新專輯《大齡女子》。歷經18個月的製作期，聯合了製作人[[陳建騏|陳建騏]]、[[蔡健雅|蔡健雅]]、Jim Lee與[[阿沁|阿沁]]等人全力製作，並由曾獲[[金馬獎|金馬獎]][[電影配樂|電影配樂]]提名的林尚德負責統籌。得到創作人[[羅大佑|羅大佑]]與[[林夕|林夕]]的量身創作，創作陣容包括[[蔡健雅|蔡健雅]]與[[小寒|小寒]]、[[陳小霞|陳小霞]]、[[徐佳瑩|徐佳瑩]]、[[蕭煌奇|蕭煌奇]]、[[阿沁|阿沁]]、[[徐世珍|徐世珍]]、易家揚、[[陳宏宇|陳宏宇]]以及[[伍家輝|伍家輝]]、佳旺、饒彗冰、吳振豪等。2016年憑此張專輯於出道20年後首次入圍並獲得[[第27屆金曲獎|第27屆金曲獎]]「[[最佳國語女歌手獎_(金曲獎)|最佳國語女歌手獎]]」殊榮。

2017年，加盟新公司[[台灣索尼音樂娛樂|台灣索尼音樂娛樂]]，並於年底發行第十一張新專輯《我想念我自己》，延續前一張專輯的風格走向，探討女人的幸福與成長故事。

2018年，憑藉《我想念我自己》入圍[[第29屆金曲獎|第29屆金曲獎]]「[[最佳國語女歌手獎_(金曲獎)|最佳國語女歌手獎]]」，這是她二度入圍本獎項。
