''' 彭罡原'''（[[英文|英文]]：Byron Pang，{{bd|1981年|10月2日|catIdx=P彭}}），生於[[英屬香港|英屬香港]]，籍貫[[廣東|廣東]][[揭西|揭西]]。彭罡原原名'''彭志鴻'''，1993年改為'''彭冠期'''，2011年再易名彭罡原。

彭罡原曾是一位游泳教練和救生員，也曾當過電影副導演、替身及西厨師。參選2005年度第一屆《[[香港先生選舉|香港先生選舉]]》。是香港前無綫電視藝員。目前是一位[[電影演員|電影演員]]。屬於[[藝行者電影公司|藝行者電影公司]]旗下。2009年彭罡原离开电视圈，跃进大银幕，参演拍摄描写香港[[同性恋|同性恋]]、毒品的奇情电影《[[安非他命_(電影)|安非他命]]》，片中毫无保留的全裸演出，完全投入的演技大获好评。彭罡原因此成为第一位在[[电影|电影]]中全裸演出的《[[香港先生|香港先生]]》而轰动全城。

2011年彭罡原凭着艺术电影《安非他命》提名角逐[[2011年香港電影金像獎|2011年香港電影金像獎]]最佳新演员奖项，并成为大热门。除幕前演出外，更自组班子，创作多部[[独立电影|独立电影]]。2010年彭罡原更修读[[心理学|心理学]]导师课程，拓展心灵辅导事业，并尝试将心灵辅导与电影相结合。

== 早年生涯 ==
彭罡原出身於一個破碎家庭，童年時期讀書不成，其叛逆的個性曾使他誤入歧途，經常過着打架、吸毒及破壞社會的日子，12歲的他被關進兒童教導所，及後被安排入讀紀律學校，三年的紀律生活令他浪子回頭。「改過自身」成为彭罡原及后的人生目标，全因為当年其校監的一句說話：「被安排到這裡入讀的每一個人都是有問題，進來就是要你們改過，自己做過的事，應由自己去承擔」。

離開紀律學校後，彭罡原辗转進入工業學院IVE進修汽車維修課程，之後想籍其技術找尋工作，那時正值1997年[[亞洲金融風暴|亞洲金融風暴]]事件後，沒有車行雇佣他，及後彭輾轉做過[[西廚|西廚]]学徒、[[救生員|救生員]]等等，由于覺得自己學歷不足。於是彭罡原決心讀夜校，由小學開始讀起，當時其将所有正職兼職的收入都用來交學費。最终，彭罡原憑個人毅力，多年半工讀下，終讀上[[港大|港大]]校外課程[[HKUSPACE|HKUSPACE]]，后来彭罡原报名参加[[无线电视|无线电视]]举办的《香港先生选举》，因為參加《香港先生》和晉身TVB工作的关系，彭罡原最終也沒有畢業。<ref>[http://ol.mingpao.com/cfm/style5.cfm?File=20080503/sta02/wca2.txt 明報OL網（2008年5日3日）]</ref>

彭罡原在拍摄电影、电视期間也考取了不少的專業技術資格。

== 習武生涯 ==
童年時的彭罡原很不屑那些街頭惡霸恃強凌弱欺負弱小，於是激發自己要段練成一個強壯和可以保護自己及身邊重要的人。因此他苦練中國功夫、格鬥技、自衛術、拳擊等。因為他的師傅是一位電影武術指導，讓他有機會接觸拍攝，當上動作副導演和替身，更被歐洲權威武術雜誌MARTIAL ARTS Magazine訪問。

2004年他被[[香港電台|香港電台]]選中在一齣Y2K青春武俠劇《天下無敵》裏擔一重角，開始踏上演藝舞台。

== 加入藝行者 ==
2008年彭罡原離開[[TVB|TVB]]後，加入了由導演[[雲翔|雲翔]]投資的藝行者電影公司，並成為了旗下[[演員|演員]]，首齣電影作品為《安非他命》，彭罡原是該片男主角，並在2010年，憑着《安非他命》一片，被[[德國|德國]][[柏林影展|柏林影展]]選片委員會負責人評價為「一位極具演藝潛質的年輕實力派演員」，並入選第60屆柏林影展Panorama全景競賽組別，提名競逐「[[太迪熊獎|太迪熊獎]]」，《Amphetamine》更被柏林影展影評人，評為最火熱觸目電影之一。

彭罡原在電影《安非他命》中的演繹風格獨特達致專業水準，由於在戲內飾演一名長期吸毒和精神失常的人（Kafuka） ，劇情有多場玩命鏡頭，需要在跟香港[[青馬大橋|青馬大橋]]齊名的[[昂船州大橋|昂船州大橋]]親身上陣拍攝高空彈跳和全裸跳樓以及被同性强暴[[鸡奸|鸡奸]]的鏡頭，角色極具難度及挑戰性，其尺度具掙議性，其後被[[香港電檢署|香港電檢署]]下令刪剪某些場合，成為各大傳媒重點報道的新聞，使他自從離開TVB以來沉寂下去的知名度和關注度又再度活躍回來。

== 圈中緋聞 ==
=== 林保怡 ===
2007年，有個别傳媒傳出彭罡原與無綫藝人[[林保怡|林保怡]]有[[同性戀|同性戀]]關係，不過彭罡原否認。他表示認識林保怡只是因為彼此是同事後來成為較相熟的普通朋友，他表示他和林保怡在一起時被人拍的合照被人惡意利用和散播謠言，他強調他和林保怡皆不是[[同性戀者|同性戀者]]，彼此皆有自己的女朋友。<ref>[http://paper.wenweipo.com/2007/06/30/EN0706300005.htm 香港文滙報（2007年6月30日）]</ref><ref>[http://ol.mingpao.com/cfm/Archive1.cfm?File=20070630/saa01/mab1.txt 明報OL網（2007年6月30日）]</ref>

=== 雲翔===
2009年，有個别傳媒傳出彭罡原被導演雲翔包養，不過彭罡原和雲翔皆否認。他們共同表示緋聞的理據不成立，因為如果雲翔包養彭罡原，他就可以躲在雲翔的家享受生活而不是為他拍戲，並且每天辛苦地拍攝那些跳樓、被打又被強姦的危險鏡頭。彭罡原再三強調自己不是同性戀者。他演繹有同性戀者行為的角色也只是劇情上和藝術上的需要。
