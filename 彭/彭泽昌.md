'''彭泽昌'''（{{bd|1963年|2月||}}）[[苗族|苗族]]，[[贵州省|贵州省]][[天柱县|天柱县]]人，[[中华人民共和国|中华人民共和国]]官员。

==生平==
1985年7月参加工作，1985年5月加入[[中国共产党|中国共产党]]，[[在职研究生|在职研究生]]。1985年7月至1988年10月，任民族团结杂志社任编辑、记者。1988年10月起，在[[国家民委|国家民委]]经济发展司工作，历任扶贫办、体改处副处长，民贸处、综合处处长。2009年8月起，任国家民委经济发展司副司长。2014年2月，挂职担任中共[[湘西土家族苗族自治州|湘西土家族苗族自治州]]州委常委。2014年4月，挂职担任中共湘西土家族苗族自治州州委常委、副州长<ref>{{Cite web |url = http://district.ce.cn/newarea/sddy/201405/09/t20140509_2792949.shtml |title = 彭泽昌任湘西州委常委、副州长 |author =  |date =2014-05-09  |publisher = 中国经济网 |language =  |accessdate =  |quote =  }}</ref>。2016年4月26日湘西土家族苗族自治州第十三届人民代表大会常务委员会第二十六次会议通过，决定免去彭泽昌的州人民政府副州长职务<ref>{{Cite web |url =http://www.hnxxrd.gov.cn/rsrm/201604/t20160428_208293.html  |title = 湘西州人民代表大会常务委员会决定任免名单 |author =  |date = 2016-04-28|publisher = 湘西土家族苗族自治州人民代表大会常务委员会 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url =http://www.cq.xinhuanet.com/2017-06/15/c_1121147860.htm  |title =  国家民委经济发展司副司长彭泽昌同志讲话|author =  |date = 2017-06-15 |publisher =新华网  |language =  |accessdate =  |quote =  }}</ref>。
