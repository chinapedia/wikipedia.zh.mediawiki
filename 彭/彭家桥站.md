== 历史 ==
* 2009年6月2日《南昌市轨道交通1号线一期工程环境影响报告书（简本）》中本站名为'''北京东路站'''位于北京东路与上坊路交叉口<ref>{{Cite web|url=http://bbs.railcn.net/thread-275044-1-1.html|title=《南昌市轨道交通1号线一期工程环境影响报告书（简本）》|accessdate=2016-07-07|last=中铁第四勘察设计院集团有限公司|first=|work=中铁第四勘察设计院集团有限公司}}</ref><ref>{{Cite web|url=http://www.crfsdi.com.cn/hpgs.asp|title=《南昌市轨道交通1号线一期工程环境影响报告书（简本）》|accessdate=2016-07-07|author=中铁第四勘察设计院集团有限公司|date=2009-6|publisher=铁四院新闻网|archiveurl=https://web.archive.org/web/20091227062216/http://www.crfsdi.com.cn/hpgs.asp|archivedate=2009-12-27}}</ref>。
* 2010年8月30日的1号线一期24个站点暂用名定为'''北京东路站'''<ref name=":0">{{Cite web|url=http://news.jxgdw.com/jszg/1388252.html|title=南昌地铁1号线24个站点具体位置及规划暂用名出炉(组图)|accessdate=|author=|date=2010-08-29|publisher=今视网|deadurl=yes|archiveurl=https://web.archive.org/web/20141211073448/http://news.jxgdw.com/jszg/1388252.html|archivedate=2014年12月11日|df=}}</ref>。

* 2014年1月23日3、4号线优化调整规划批前公示，4号线不再经过本站<ref>{{Cite web |url = http://jiangxi.jxnews.com.cn/system/2014/01/23/012919119.shtml |title = 南昌地铁3、4号线规划调整 4号线调整较大 线路增加8公里 |author =  |date = 2014-01-23 |publisher = 中国江西网-信息日报 |language = |accessdate = |archiveurl= |archivedate=}}</ref><ref>{{Cite web |url = http://ghj.nc.gov.cn/qtpqgg/6540.jhtml |title = 南昌市轨道交通3、4号线优化调整规划批前公示 |author =  |date = 2014-01-21 |publisher = 南昌市城乡规划局 |language =  |accessdate =  |archiveurl = https://web.archive.org/web/20161014013008/http://ghj.nc.gov.cn/qtpqgg/6540.jhtml |archivedate = 2016-10-14 |dead-url = yes }}</ref>。
