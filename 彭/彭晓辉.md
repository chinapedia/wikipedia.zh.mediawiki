'''彭晓辉'''（{{Bd|1957年|6月11日}}）中国大陆著名性学家，[[华中师范大学|华中师范大学]]生命科学院教授和硕士生导师。

==生平==
彭晓辉，生于1957年，祖籍[[湖南|湖南]]，出生于[[湖北|湖北]]武汉，父亲在“文革”中划分为右派，他从小就辗转于舅舅、姨妈等表亲家，过着寄人篱下的生活。

大学开始之后，一直研究性学，已经20年。他被学生和社会各界人士评为“性教授”、“性工作者”、“老师爸爸”、“彭霸天”等称号。<ref name="multiple1">{{cite web |url=http://news.sina.com.cn/s/sd/2011-05-23/103022513379.shtml|title=《彭晓辉：一位性学导师的20年》|accessdate=2011年5月23日|work=中国新闻周刊|publisher=新浪网|language= zh-hans}}</ref>

1992年，作为华中师范大学生物系讲师的彭晓辉，开出一门专业选修课《性生物学》，主讲课程为《人体组织解剖学》《动物及人体生理学》等。

2000年，经华师生命科学学院学术委员会审核批准，彭晓辉获得了性学研究方向硕士生导师资格。

2011年，在彭晓辉54岁的时候，他开通了微博，用140字内的文句来开讲“微性学”，他的学生则在对他的QQ印象评价为“潮男”。<ref name="multiple1">{{cite web |url=http://news.sina.com.cn/s/sd/2011-05-23/103022513379.shtml|title=《彭晓辉：一位性学导师的20年》|accessdate=2011年5月23日|work=中国新闻周刊|publisher=新浪网|language= zh-hans}}</ref>

2012年2月25日，彭晓辉即将邀请日本女优、现台湾红丝带基金会防艾大使[[红音萤|红音萤]]到其任职的大学与大学生进行交流，但是遭到社会人士“上纲上线”的文化大革命时期人与人斗争的对待，活动被迫取消。<ref>{{cite web |url=http://news.xinhuanet.com/yuqing/2012-02/27/c_122759073.htm |title= 专家邀请女优进大学课堂 网友激辩：合适否 |quote =可以用舆论大哗来形容“AV女优进课堂”的新闻热度，对华中师大、对彭晓辉的批评之声遍布网络的每一个空隙——理由就是很多人普遍认为红音萤曾经的AV女优经历，有辱国人传统思维，其身份之“贱”无资格担当“教育”国人之任。但窃以为，不必对“AV女优进课堂”上纲上线。 |accessdate= 2012年2月27日 |publisher=北京晨报 |work= |language=zh-hans}}</ref>

2014年11月7日，[[广州市|广州市]][[第十二届全国性文化节|第十二届全国性文化节]]开幕，彭晓辉演讲“性学”时，一名受到封建糟粕文化毒害的大妈向其泼了“秽物”，反对其讲授性学知识。<ref>{{cite web |url=http://ent.sina.com.cn/s/m/2014-11-11/17084238893.shtml |title=性学家亮相性文化节上演讲 被大妈泼粪 |accessdate= 2014年11月11日|publisher= 新浪网|work= |language=zh-hans}}</ref>

== 言论观点 ==
=== 《3D肉蒲团》看看也无妨 ===

2011年4月22日，在彭晓辉的《人类性学》课程上，一个女生问：“彭老师，你会去看香港上映的《[[3D肉蒲团|3D肉蒲团]]》吗？”彭晓辉回答：“从学术的角度，看看无妨。”并且接着阐释，“古代性小说其实也是性学的研究范围，而拍成电影的《3D肉蒲团》则纯粹属于性消费。底层大众在生活资料的获得方面处于弱势，在性资源的获得上也会因为贫富差距而在一定程度上被剥夺。那么，“挂眼科”消费这种虚拟的性资源是一个弥补，能起到稳定社会的作用。归根结底，“性”不是一个道德问题，而是一个政治和经济问题。”<ref name="multiple2">{{cite web |url=http://news.cntv.cn/society/20110512/101621.shtml|title=《性学导师讲座认可同性婚姻 观点遭学生当场反驳》|accessdate=2011年5月12日|work=扬子晚报|publisher=中国网络电视台|language= zh-hans}}</ref>

=== 遭遇性侵犯女性应主动递上避孕套 ===

2011年5月10日晚，彭晓辉在《性与人际交往》的讲座上回答一个女生关于“男性处女情结”时，他说：“有一方面的原因是在面临性侵犯的时候，能及时递上[[避孕套|避孕套]]，那是保护女性免受[[艾滋病|艾滋病]]等性传播疾病的最后一道屏障。”引起讲座现场和网络媒体的口诛笔伐，唇枪舌战。<ref name="multiple2">{{cite web |url=http://news.cntv.cn/society/20110512/101621.shtml|title=《性学导师讲座认可同性婚姻 观点遭学生当场反驳》|accessdate=2011年5月12日|work=扬子晚报|publisher=中国网络电视台|language= zh-hans}}</ref>
