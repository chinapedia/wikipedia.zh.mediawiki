'''彭建源卡拉OK縱火案'''是在2012年3月12日於[[台湾|台湾]][[新竹市|新竹市]][[東區_(新竹市)|東區]]境內發生的一起5死縱火案<ref name="蘋果20120313">{{cite news|url=http://www.appledaily.com.tw/appledaily/article/headline/20120313/34085952/applesearch/恨友不相挺縱火殺5人|title=恨友不相挺 縱火殺5人|publisher=[[蘋果日報_(台灣)|蘋果日報 (台灣)]]|date=2012-03-13|author=楊勝裕、楊永盛、談曉泉}}</ref>。凶嫌[[彭建源KTV縱火案|彭建源]]（案發時34歲）因不滿好友周志強（案發時34歲）不願出面相挺，故購買300元的汽油至其位於[[新竹市|新竹市]]明湖路的[[KTV|KTV]]縱火，造成5死4傷。
本案在2014年9月2日由[[中華民國最高法院|中華民國最高法院]]判決[[死刑|死刑]]定讞，彭建源目前羈押於[[新北市|新北市]][[土城看守所|土城看守所]]等待[[槍決|槍決]]。<ref name="死刑定讞">{{cite news|url=http://www.appledaily.com.tw/appledaily/article/headline/20140903/36061178/applesearch/縱火殺5命男死刑定讞|title=縱火殺5命 男死刑定讞|publisher=蘋果日報|date=2014-09-03|author=劉志原、楊勝裕}} </ref>

== 凶嫌背景 ==
彭建源自幼被過度寵溺，退伍後結交重度吸毒者，每次細故，例如索討金錢買毒品不成、行車被按喇叭等，就暴怒犯罪，無論是對於父母、舊識或素不相識之人，均採取暴力手段回應，曾有預謀砍殺父親，有毒品、妨害自由及家暴前科。<ref name="蘋果0313">{{cite news|author=楊勝裕、楊永盛、談曉泉|url=http://www.appledaily.com.tw/appledaily/article/headline/20120313/34085967/放火後狂男飆車去|title=放火後 狂男飆車去|date=2012-03-13|publisher=蘋果日報}}</ref><ref name="自由20120607"/><ref name="自由20121130"/>

== 案發過程 ==
彭建源在2012年3月12日凌晨在新竹市，因不滿好友周志強不願幫忙助陣，和王姓女子同車去加油站購買10公升汽油，至周志強位於明湖路的KTV店縱火<ref name="蘋果0313"/>。周一開門，彭就潑油縱火，還將裝汽油的鐵桶把小門頂住不讓人逃生，但是裡面3人從安全門逃生成功。
縱火後行跡詭異，被巡邏警網攔下。隨後並將他戒護就醫。

警方循線查出5死縱火案的凶嫌即是彭建源，訊後依殺人罪及公共危險罪移送[[臺灣新竹地方法院檢察署|新竹地檢署]]偵辦，並聲押獲准。

== 傷亡情況 ==
五位死者分別是田德鎮（33歲、男）、邱文富（32歲、男）、徐宗永（42歲、男）、林美菁（46歲、女）、閻桂芬（42歲、女）<ref name="自由20120607"/>，其中邱文富是[[新竹城隍廟|新竹城隍廟]]知名店家「阿富魯肉飯」老闆<ref>{{cite news|url=http://www.appledaily.com.tw/appledaily/article/headline/20120313/34085986/名店老闆遇死劫4子失依|author=談曉泉 |title=名店老闆遇死劫 4子失依|publisher=蘋果日報|date=2012-03-13}}</ref><ref>{{youtube|jcE-ihoqKOs|新竹知名滷肉飯老闆被燒死　前妻帶4子喊冤|link=no}},[[中天電視台|中天電視台]],2012-03-13</ref>。四位傷者分別是屋主周志強（34歲、男），傷勢四肢[[灼傷|灼傷]]；周妻周昱蓉（26歲、懷孕7月），傷勢[[嗆傷|嗆傷]]；徐郁臻（37歲、女），傷勢嗆傷；縱火嫌犯彭建源（34歲、男），傷勢雙手灼傷。<ref name="蘋果20120313"/>

== 起訴與審判 ==
2012年6月6日，到案後僅關心自己是否符合自首要件，對於犯行毫無悔意，[[臺灣新竹地方法院檢察署|新竹地檢署]]檢察官認為彭男惡性重大，有與社會隔絕的必要，向法官具體求處死刑<ref name="自由20120607">{{cite news|author=王錦義|url=http://news.ltn.com.tw/news/local/paper/589889|title=縱火燒死5人 求處死刑|date=2012-06-07|publisher=[[自由時報|自由時報]]}}</ref>；11月28日，[[新竹地方法院|新竹地方法院]]認為沒有自首減刑的適用，一審依故意殺人、不確定故意殺人等罪判處死刑，褫奪公權終身<ref name="自由20121130">{{cite news|author=王錦義|url=http://news.ltn.com.tw/news/local/paper/634542|title=縱火害5命 惡男判死刑|date=2012-11-30|publisher=自由時報}}</ref><ref name="蘋果20130509">{{cite news|url=http://www.appledaily.com.tw/realtimenews/article/local/20130509/178946/applesearch/縱火燒卡拉OK釀5死男判死刑|title=縱火燒卡拉OK釀5死　男判死刑|date=2013-05-09|publisher=蘋果日報}}</ref>；12月12日，彭建源的友人周志強因在鐵皮屋違法經營[[卡拉OK|卡拉OK]]，未能提供消費者安全設備，造成五人燒死，依業務過失致死及《建築法》起訴<ref>{{cite news|url=http://www.appledaily.com.tw/appledaily/article/headline/20121213/34703215/applesearch/違法卡拉OK害5人燒死遭訴 |title=違法卡拉OK害5人燒死 遭訴|author=楊勝裕|date=2012-12-13|publisher=蘋果日報}}</ref>。
{{Wikisourceshas|臺灣新竹地方法院101年度重訴字第6號刑事判決|臺灣高等法院101年度上重訴字第61號刑事判決|最高法院102年度台上字第5091號刑事判決}}
2013年5月9日，[[台灣高等法院|台灣高等法院]]二審駁回上訴，判處死刑<ref name="蘋果20130509"/>；12月12日，[[中華民國最高法院|最高法院]]認為證據不足，發回高院更審<ref>{{cite news|author=項程鎮|url=http://news.ltn.com.tw/news/society/paper/738182|title=縱火害5命 死囚求死 仍發回更審|date=2013-12-13|publisher=自由時報}}</ref>。
{{Wikisourceshas|臺灣高等法院102年度上重更(一)字第11號刑事判決|最高法院103年度台上字第3062號刑事判決}}
2014年3月11日，高院更一審仍維持死刑判決，法官認為彭嫌罪刑重大，已無法教化，因此維持死刑<ref>{{cite news|url=http://www.ettoday.net/news/20140311/333379.htm|title=新竹KTV縱火害5命　凶嫌彭建源更一審仍判死刑|date=2014-03-11|publisher=ETtoday}}</ref>；9月2日，最高法院三審判處彭建源死刑確定。現羈押在[[台北看守所|台北看守所]]等待槍決，拘置中<ref name="死刑定讞"/>。

==不同見解==
彭建源有[[精神病|精神病]]的就醫紀錄，歷任審判的罪名都是「不確定故意殺人」，依據最高法院在陳昆明殺人案、林基雄縱火燒死１３人案中所釋示的〈[[國際人權公約|兩公約]]〉要求：不得判精神障礙者死刑、非蓄意殺人不得判死刑<ref name=sss>{{cite news|author=黃錦嵐|url=http://hk.on.cc/tw/bkn/cnt/commentary/20140903/bkntw-20140903000414934-0903_04411_001.html|title=彭建源死刑定讞案的〈兩公約〉爭議|date=2014-09-03|publisher=東網}}</ref>。

對此，[[廢死聯盟|廢死聯盟]]執行長[[林欣怡|林欣怡]]認為最高法院對於[[精神障礙|精神障礙]]得否判死的見解不一。對此，最高法院澄清，已做成明確性的法律解釋，其判決理由及要旨，均已明確說明，並無不妥<ref name=ssss>{{cite news|author=林偉信|url=http://www.chinatimes.com/newspapers/20140903000459-260102|title=廢死聯盟轟見解不一 網友批只判死不執行|date=2014-09-03|publisher=中國時報}}</ref>。

最高法院刑四庭[[審判長|審判長]]賴忠星與４位法官認為，所謂不得對任何精神或智能障礙者判處或執行死刑，旨在「敦促」<ref name=ss>{{cite news|author=|url=http://www.ksdreamba.artcom.tw/ap/news_view.aspx?bid=23&sn=8c0f0e2a-9e7f-4521-b4cd-c35e459f0498|title=最高院認定 精障犯案可判死|date=2014-09-03|publisher=社團法人高雄市築夢關懷協會}}</ref>，並非〈兩公約〉的強制要求，並不具有[[國內法|國內法]]的效力<ref name=sss/>。

[[民間司改會|民間司改會]]執行長高榮志則認為最高法院為迎合「判死」民意，召開生死辯後，架空兩公約，做出合理化法律解釋，把彭建源判死<ref name=ssss/>。
