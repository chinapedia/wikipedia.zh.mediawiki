'''張海'''（{{bd|1974年|5月|||}}）<ref>[http://www.apdnews.com/news/71186.html 广东健力宝集团原董事长张海] {{webarchive|url=https://web.archive.org/web/20140301232228/http://www.apdnews.com/news/71186.html |date=2014-03-01 }}，亚太日报，2014年2月25日</ref>，[[河南|河南]][[開封|開封]]人。其父為當地公務員，母親為中學教師。1988年，張海[[初中|初中]][[畢業|畢業]]後未直接升入[[高中|高中]]，而是考入[[河南大學|河南大學]][[武術|武術]]專修班。後因畢業成績不好，僅獲專科[[結業證書|結業證書]]。

== 宣稱擁有特異功能 ==
1988年前後，張海已傳出身懷「[[特異功能|特異功能]]」，號稱能把樹葉「由大變小」、能用意念「讓時間倒流」、讓絕症患者「手到病除」<ref>[http://finance.sina.com.cn/leadership/crz/20050514/01071586965.shtml CCTV经济半小时：健力宝张海诡秘半生(多图)_财富人物_新浪财经_新浪网]</ref>。在就讀武術專修班期間，張海結識了[[藏密|藏密]][[氣功|氣功]]及[[藏傳佛教|藏傳佛教]]人士，並於1992年以藏傳佛教夏日東[[活佛|活佛]]弟子身份在各地廣開藏密健身中心，傳授「密法」，招攬學徒。同年，張海結識藏密信徒、[[河南省社科院|河南省社科院]]哲學所辦公室主任[[孫嶺|孫嶺]]，經後者幫助，在該院設藏密[[瑜伽|瑜伽]]文化研究所，任所長。

== 入主[[健力寶|健力寶]] ==
1995年，張海舉家南遷[[廣州|廣州]]，開始轉向[[資本市場|資本市場]]<ref>[http://finance.takungpao.com.hk/gscy/q/2014/0120/2195183.html 健力寶原董事長張海外逃獲證實 其立功減刑造假_大公財經_大公網]</ref>。2000年開始，張海突然轉身資本市場，入主由國內36所名校發起成立的[[上市公司|上市公司]][[中国高科集團股份有限公司|中国高科集團股份有限公司]]並出任[[董事長|董事長]]，一度是中國「最年輕的上市公司董事長」。兩年後，張海再度出手收購[[飲料|飲料]]企業「健力寶」，出任董事長兼[[總裁|總裁]]，2004年被[[富布斯|富布斯]]雜誌評為中國第137位富翁，但兩年後張海就被公司[[股東|股東]]聯手逐出。

张海入主健力宝后，从平安保险党委书记[[马明哲|马明哲]]手中接手了5100万的平安保险股份<ref>{{cite news |title=争端再起健力宝：5100万平安股权争夺 |url=http://finance.sina.com.cn/roll/20081212/23522570510.shtml |work=中国经营报 |date=2008年12月12日}}</ref>。这部分股份原本是在江南实业公司名下持有，[[马明哲|马明哲]]通过江南实业公司违规持有国有平安保险的股份。低价转让给张海的健力宝后，期望得到张海背后的权力背景保护。

2018年9月12日，大陆流亡富豪[[郭文贵|郭文贵]]在视频里也曾透露平安保险的股份代持问题。<ref>{{cite web|url=https://www.youtube.com/watch?v=O8Cl2OyVFXg | title=郭文贵2018年9月12日视频直播 | url=https://www.youtube.com/watch?v=O8Cl2OyVFXg}}</ref>

== 被捕與入獄 ==
2005年3月24日夜間，張海在一家酒店吃宵夜時，被[[廣東|廣東]][[佛山|佛山]][[公安局|公安局]][[刑事拘留|刑事拘留]]，罪名是涉嫌[[挪用資金|挪用資金]]，還涉嫌[[做假賬|做假賬]]、[[虛假投資|虛假投資]]、[[虛增銷售|虛增銷售]]等罪名，緣於2004年9月下旬，健力寶集團對張海任期內集團財務狀況進行[[審計|審計]]，發現張海等人挪用、抽走、轉移和侵吞集團資金超過7億元[[人民幣|人民幣]]<ref>[http://www.voachinese.com/articleprintview/1042167.html 健力寶集團前總裁張海被拘留-美國之音中文網 熱點新聞]</ref>。2007年2月，佛山市[[中級人民法院|中級人民法院]]以[[職務侵佔|職務侵佔]]罪、挪用資金罪兩項罪名一審判處張海[[有期徒刑|有期徒刑]]15年。2008年9月，[[廣東省高院|廣東省高院]]二審認定張海[[檢舉他人犯罪立功|檢舉他人犯罪立功]]，把原審判決的有期徒刑15年改判為10年。

== 減刑作假與外逃 ==
後來張海在獄中因有立功行為和表現良好等又獲得了兩次[[減刑|減刑]]機會，第一次從10年減為8年，第二次減到6年，故最終在2011年年初就提前出獄<ref>[http://epaper.ynet.com/html/2014-01/19/content_36695.htm?div=-1 北京青年报电子版-健力宝原董事长张海造假立功提前出狱后失踪] {{webarchive|url=https://web.archive.org/web/20140201180022/http://epaper.ynet.com/html/2014-01/19/content_36695.htm?div=-1 |date=2014-02-01 }}</ref>。2014年1月8日，廣州市中級法院通報稱，張海獲得減刑案存在弄虛作假問題。1月10日，廣東省高院在官方微博通報稱，張海立功減刑造假屬實，案件已經立案審查。2014年1月17日，廣東省[[檢察院|檢察院]]副[[檢察長|檢察長]][[黃武|黃武]]在[[廣東省|廣東省]][[政協|政協]]十一屆二次會議分組討論上透露，張海已失蹤，有可能外逃。1月20日，廣東省檢察院對外證實，張海已經逃往境外，在張海服刑的6年內，曾涉嫌造假「立功」材料而獲減刑5年。黃武在接受採訪時披露，張海從進看守所開始就涉嫌作假，在服刑期間的立功表現屬造假。現在回過頭去查，一審判決前就已有作假嫌疑，如果不是作假「立功」，可能就不止15年刑期，廣東省司法機關對此立案審查<ref>[http://finance.ce.cn/rolling/201401/21/t20140121_2168269.shtml 张海外逃获官方证实 健力宝难复当年“魔力” _中国经济网——国家经济门户]</ref><ref>{{cite news|title=中国启动程序追捕引渡健力宝前总裁张海(图)|url=http://finance.ifeng.com/a/20140225/11736863_0.shtml|accessdate=2017-11-14|agency=南方都市报|publisher=凤凰财经|date=2014-02-25}}</ref>。
