'''張簡姓'''是華人的一个[[双姓|双姓]]<ref>{{cite web |url=http://www.nccu.edu.tw/news/detail.php?news_id=557 |title=物理學家跨界歷史 錢致榕研究台灣雙姓家族 |publisher=國立政治大學 |date=2008-08-23 |accessdate=2009年7月9日 |deadurl=yes |archiveurl=https://web.archive.org/web/20091223192427/http://www.nccu.edu.tw/news/detail.php?news_id=557 |archivedate=2009年12月23日 }}</ref>，源自[[张姓|张姓]]与[[简姓|简姓]]，多见于[[闽南|闽南]]、[[台灣|台灣]]等地。根據台灣户政司民國107年（2018年）6月的數據，張簡姓為臺灣第一大雙字姓，人口有9,059人，為全國姓氏人口排名第122名。為台灣單字姓氏外人口中數最高者。<ref>內政部戶政司編，《全國姓名統計分析》，臺北：內政部出版。2018年10月初版一刷</ref>

== 起源 ==
张简姓起源于一次[[联姻|联姻]]，[[明朝|明朝]][[洪武|洪武]]四年（1371年），[[漳州府|漳州府]][[南靖县|南靖县]]永丰里梅林村的[[士人|士人]]'''簡德潤'''，在南靖的[[書院|書院]]教書，[[入贅|入贅]]於當地[[門閥|門閥]]張進興家，被張進興視為[[義子|義子]]，簡德潤娶張進興寡居的儿媳劉氏為妻，生了三子。後又娶[[側室|側室]]盧氏<ref>[http://familyliao.myweb.hinet.net/index.html 張簡源流故事]</ref>，生了五子。后简德润希望张、简两姓均有[[香火|香火]]可传，在和张家商量后，遂以其中四名[[過繼|過繼]]張姓，四名留於簡家，後來才出現了張簡複姓<ref>{{cite web |url=http://210.38.160.81/kejia/difang/html/?389.html |title=族谱资料与客家文化研究 |publisher=嘉应学院客家研究院 |date=2008年6月21日 |accessdate=2009年7月5日 |deadurl=yes |archiveurl=https://web.archive.org/web/20160304080219/http://210.38.160.81/kejia/difang/html/?389.html |archivedate=2016年3月4日 }}</ref>。

[[清|清]]中叶，十二世祖张简成祯渡海来到臺湾南部定居，在[[高雄市|高雄市]][[大寮區|大寮區]]有其众多子孙<ref name="cha"/>，是台湾张简姓的发源地<ref>{{cite journal|author=侯俊榮|title=〈大寮地區張簡宗族之研究〉|journal=國立臺南大學台灣文化研究所碩士論文|publisher=[[國立臺南大學|國立臺南大學]]|location=[[臺南市|臺南市]]|year=2006|language=zh-tw|accessdate=2011-07-14}}</ref>。

[[清|清]][[咸豐|咸豐]]年间，张简家族宗祠追远堂建成，许多简姓改姓张简<ref>[http://www.zzn.ks.edu.tw/dalouw/wd-1.htm 臺灣張簡氏的由來與傳說]</ref>。

[[台灣日治時期|日治时代]]，为避免麻烦，一些张简姓改姓张或姓简。[[臺灣戰後時期|中華民國接管台灣後]]，有部分人恢复张简姓，也有一部分人没有恢复，但其子女为张简姓<ref name="cha">{{cite web|url=http://xing.911cha.com/OTJv.html | title=百家姓：张简姓 |publisher=911查 |date= |accessdate=2009年7月5日 }}</ref>。

在[[臺湾|臺湾]]，还有这样一个传说：有位四处流浪简姓的孤儿，有一天梦到一个仙人让他一直向南走直到绳子断掉。后来他来到一位姓张的员外家时绳子断了，于是留下来帮员外干活，后被员外收为义子，娶了张员外独子的遗孀刘氏，继承了张员外的财产。为了纪念张公，简姓孤儿将自己子女改姓张简<ref>{{cite web|url=http://k.zmp.ks.edu.tw/~lzz/firstname/origin.html | title=張簡姓的由來 |publisher=高雄縣大寮鄉昭明國小 |date= |accessdate=2009年7月6日 }}</ref>。

== 分布 ==
張簡姓多见于闽南、台湾等地<ref>{{cite web |url=http://www.town-all.org.tw/viewweb/stu079/changgin/place.htm |title=台灣張簡氏人口之祖籍與地域分佈 |publisher=小神腦童窩網 |date= |accessdate=2009年7月5日 |deadurl=yes |archiveurl=https://archive.is/20120917212726/http://www.town-all.org.tw/viewweb/stu079/changgin/place.htm |archivedate=2012年9月17日 }}</ref>。

在台灣，張簡姓大多數位於[[高雄市|高雄市]]。在2018年的9,059人張簡姓人口中，高雄市即占有7,197人。高雄市主要集中在[[大寮區|大寮區]]和[[鳳山郡|鳳山地區]]，有張簡姓「五脉31派下」的說法。在高雄市以外，[[屏東縣|屏東縣]]、[[臺南市|臺南市]]亦有分布。
