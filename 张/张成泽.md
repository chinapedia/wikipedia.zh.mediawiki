'''张成泽'''（{{lang-ko|장성택}}，{{bd|1946年|1月22日|2013年|12月12日|catIdx=Jang, Sung-taek}}），出生于[[蘇聯民政廳|蘇聯民政廳]][[江原道|江原道]][[川内郡|川内郡]]，[[朝鲜|朝鲜]]政治及军事人物，前党和国家领导干部之一，曾任[[朝鲜劳动党中央政治局|朝鲜劳动党中央政治局]]委员、[[朝鲜国防委员会|朝鲜国防委员会]]副委员长、[[朝鲜劳动党中央行政部|中央行政部]]部长、[[朝鲜劳动党中央军事委员会|中央军事委员会]]委员、[[国家体育指导委员会|国家体育指导委员会]]委员长，[[朝鲜人民军|朝鲜人民军]][[大将|大将]]。他是[[金正日|金正日]]的妹夫、[[金日成|金日成]]长女[[金敬姬|金敬姬]]的丈夫，[[金正恩|金正恩]]的姑父，被处决前两天金敬姬被金正恩指令离婚。在金正恩掌权初期，一度被视为是朝鲜实际上的“二号人物”，也是[[朝鲜經濟|經濟]]改革和創新領域的標誌性人物，在朝鲜属温和派及知华派。

2013年12月8日，在[[朝鲜劳动党|朝鲜劳动党]][[朝鲜劳动党中央政治局2013年12月扩大会议|中央政治局扩大会议]]上张成泽被现场逮捕，分析认为金正恩选择现场抓获张成泽的原因是要给现场的与会者造成心理恐惧，警告潜在的反对者，起到杀鸡儆猴的作用，后公布张成泽罪名，處理經濟問題失當、貪瀆、結黨營私、亂搞男女關係、受資本主義影響吸毒放縱等5大罪狀遭到撤免<ref>[http://www.apdnews.com/news/52695.html 朝鲜宣佈解除张成泽一切职务并开除出党] {{webarchive|url=https://web.archive.org/web/20131213083401/http://www.apdnews.com/news/52695.html |date=2013-12-13 }}，亚太日报，2013年12月11日</ref><ref>[http://www.cna.com.tw/news/firstnews/201312090010-1.aspx 祭出叛國罪名 北韓處決張成澤]，中央社，2013年12月13日</ref><ref>[https://www.youtube.com/watch?v=5uSvX89_gYk 혁명활동소식 조선로동당 중앙위원회 정치국 확대회의에 관한 보도]</ref>，而外界有媒体认为张成泽是因政治斗争而下台，<ref name="朝鲜权力斗争让北京紧张">{{cite web |url = http://cn.nytimes.com/asia-pacific/20131210/c10nkorea/ |title = 朝鲜权力斗争让北京紧张 |publisher=纽约时报中文网 |accessdate = 2013-12-11}}</ref>据香港TVB新闻及亚洲电视新闻分析张成泽与金正恩在核问题，经济政策问题意见不合导致张成泽下台。随后张成泽被押往军事法庭受审，在庭上承认图谋发动政变并出任[[朝鲜总理|内阁总理]]<ref>{{Cite web |url=http://news.21cn.com/world/guojisaomiao/a/2013/1213/14/25513050.shtml |title=张成泽承认想发动政变：时间未定 自己任总理 |access-date=2013-12-13 |archive-url=https://web.archive.org/web/20131215070607/http://news.21cn.com/world/guojisaomiao/a/2013/1213/14/25513050.shtml |archive-date=2013-12-15 |dead-url=yes }}</ref>。12月12日，张成泽被判处[[死刑|死刑]]并立即执行<ref>[http://news.xinhuanet.com/world/2013-12/13/c_118537153.htm 朝鲜１２日判处张成泽死刑并立即执行，新华网，2013年12月13日]</ref>，这是朝鲜近16年来首次处决政治局委员级的高官，备受外界瞩目<ref>[http://finance.huanqiu.com/data/2013-12/4658290.html 朝鲜判处张成泽死刑并立即执行 法庭现场图曝光，环球网，2013-12-13]</ref>。

2018年2月13日，日本[[NHK|NHK]]采访中国政府相关人士，称2012年8月，张成泽访华，与[[胡锦涛|胡锦涛]]会面时表明想要让[[金正男|金正男]]担任朝鲜最高领导人。周永康派部下窃听到相关内容，并在2013年初将此转告[[金正恩|金正恩]]，直接导致张成泽及金正男先後被處決、殺害。

== 生平 ==
=== 金日成时期 ===
张成泽生于[[咸镜北道|咸镜北道]]，毕业于[[金日成综合大学|金日成综合大学]][[政治经济学|政治经济学]]系和[[金日成高级党校|金日成高级党校]]后，1968年赴[[苏联|苏联]][[莫斯科|莫斯科]]留学<ref name="Wilson">{{cite web|url=https://digitalarchive.wilsoncenter.org/resource/modern-korean-history-portal/jang-seong-taek|title=Jang Seong-taek, Woodrow Wilson International Center for Scholars|accessdate=2017-06-07|website=Wilson Center Digital Archive|author=James E. Hoare}}</ref>，1972年回国并于4月与金敬姬结婚<ref name="Unikorea">{{cite web|url=http://unibook.unikorea.go.kr/new2/tongiljeongbo/p_1_detail_view.jsp?code=305|script-title=ko:장성택(張成澤)|publisher=Information Center on North Korea, Ministry of Unification, Republic of Korea|accessdate=2007-08-20|language=ko|deadurl=yes|archiveurl=https://web.archive.org/web/20070927041859/http://unibook.unikorea.go.kr/new2/tongiljeongbo/p_1_detail_view.jsp?code=305|archivedate=2007-09-27}}</ref>。此后，他在[[朝鲜劳动党|朝鲜劳动党]]担任一连串的职位<ref name="Wilson"/>。一开始他是劳动党平壤市委员会指导员<ref>{{cite web|work=Associated Press|url=http://www.npr.org/templates/story/story.php?storyId=250566496|title=Leader's Uncle Rose To No. 2 In North Korea|date=2013-12-13|accessdate=2017-06-07}}</ref>。70年代后期，仕途阻滞，张成泽被党中央下放到[[南浦特别市|南浦]]一家铸铁厂当工人<ref name="Wilson"/>。一次[[千里马郡|千里马郡]]/[[江西郡|江西郡]]一座工厂发生{{le|工业意外|Work accident}}，张严重烧伤<ref name="NKLW">{{cite web|url=http://www.nkleadershipwatch.org/jang-song-thaek/#_edn3|title=Jang Song Taek|website=North Korea Leadership Watch|date=2011-01-28|accessdate=2017-06-07}}</ref>。1982年10月，他回归仕途，出任[[朝鲜劳动党|朝鲜劳动党]]中央青少年事业部副部长，并于1985年7月升任中央青少年事业部第一副部长，1988年12月再升任部长。1986年，张首次入选[[最高人民会议|最高人民会议]]<ref name="Wilson"/>。

1989年4月被授予“[[朝鲜民主主义人民共和国劳动英雄|劳动英雄]]”称号。1989年6月，他补选为[[朝鲜劳动党中央委员会|朝鲜劳动党中央委员会]]候补委员，并于同年7月任朝鲜劳动党中央青年及三大革命塑造部部长<ref name="Wilson"/>。1992年4月，他荣获[[金日成勋章|金日成勋章]]。同年12月，他正式补选为党中央委员<ref name="Wilson"/>。

=== 金正日时期 ===
1994年，他任金日成治丧委员会委员。1995年11月，他获任为中央组织指导部第一副部长。1997年-2000年協助金正日進行「[[深化組事件|深化組事件]]」肅清運動，令2萬人遭到迫害，數千人死亡。2005年，在權力鬥爭中擊敗宣導鼓動部的核心人物[[鄭夏哲|鄭夏哲]]和[[崔春晃|崔春晃]]，因为被金正日怀疑搞宗派活动而被撤职。2006年1月复出任朝鲜劳动党中央群众团体及首都建设部第一副部长，次年10月晋升朝鲜劳动党中央行政部部长，重新回到权力中心。2009年4月出任[[朝鲜民主主义人民共和国国防委员会|国防委员会]]委员，次年晋升到副委员长，从而晋身最高领导层。2010年9月，在[[朝鮮勞動黨第三次代表會議|朝鮮勞動黨第三次代表會議]]上，他當選為朝鮮勞動黨中央委員會委員；[[朝鲜劳动党中央委员会2010年9月全体会议|朝鲜劳动党中央委员会2010年9月全体会议]]上，當選為[[朝鮮勞動黨中央政治局|中央政治局]]候補委員、[[朝鮮勞動黨中央軍事委员会|中央軍事委员会]]委員<ref name=wtimes>{{cite news|author=Ashish Kumar Sen|title=Kim's heir apparent set for debut in Pyongyang|url=http://www.washingtontimes.com/news/2010/sep/26/kims-heir-apparent-set-for-debut-in-pyongyang/|accessdate=2013-12-13|newspaper=The Washington Times|date=2010-09-26}}</ref>。

=== 金正恩执政初期 ===
2011年12月，张成泽获授[[朝鲜人民军|朝鲜人民军]]大将军衔。<ref name=大将>{{cite news|title = 金正恩姑父张成泽首次身穿大将军服亮相|url = http://news.21cn.com/junshi/gj/2011/12/26/10253811.shtml|accessdate = 2011-12-26|newspaper = 凤凰网|deadurl = yes|archiveurl = https://web.archive.org/web/20120719080656/http://news.21cn.com/junshi/gj/2011/12/26/10253811.shtml|archivedate = 2012-07-19}}</ref>2012年4月[[朝鲜劳动党第四次代表会议|朝鲜劳动党第四次代表会议]]上，张成泽当选为[[朝鲜劳动党中央政治局|朝鲜劳动党中央政治局]]委员。2012年11月4日举行的[[2012年11月朝鲜劳动党中央政治局扩大会议|朝鲜劳动党中央政治局扩大会议]]决定成立国家体育指导委员会，张成泽被任命为委员长。<ref>{{Cite web |url=http://news.163.com/12/1105/08/8FHIL5SG00014AED.html |title=朝鲜成立国家体育指导委员会，网易，2012-11-05 |access-date=2013-12-13 |archive-url=https://web.archive.org/web/20130412014625/http://news.163.com/12/1105/08/8FHIL5SG00014AED.html |archive-date=2013-04-12 |dead-url=yes }}</ref>

2011年[[金正日之死|金正日逝世]]後，在[[平壤|平壤]][[錦繡山紀念宮|錦繡山紀念宮]]舉行了金正日的告別式，張成澤陪同內定接班人[[金正恩|金正恩]]與其他黨政軍高層參加，他是七位扶靈高官之一<ref>[http://www.epochtimes.com/b5/13/12/14/n4034455.htm    處決張成澤 金正恩七「顧命大臣」幾全軍覆沒]</ref>。張成澤站在金正恩身旁，身穿大將軍服；後來金正日的靈車隊伍繞行平壤時，金正恩站在靈車右前方，張成澤站在金正恩的身後，同時中央軍事委员会副委員長兼總參謀長[[李英浩|李英浩]]站在靈車左前方<ref>[http://www.apdnews.com/news/51380.html 护送灵车的7人中军人全部下台，亚太日报，2013年12月9日] {{webarchive|url=https://web.archive.org/web/20131213074800/http://www.apdnews.com/news/51380.html |date=2013年12月13日 }}</ref><ref>{{cite web |url = http://chinese.joins.com/gb/article.do?method=detail&art_id=112575 |language =  |title = 护送灵车的7人中军人全部下台 只剩两名党内官员 |publisher=韩国中央日报中文网 |date=2013-12-04 |accessdate =  }}</ref>。金正恩的領導地位正式確立後，張成澤、[[李英浩|李英浩]]、金敬姬等人成為「輔佐」金正恩的團隊，其中張成澤的地位最為崇高，因此張成澤被一些媒体視為金正恩時代的「攝政王」<ref>{{cite web |url =  http://cn.nytimes.com/asia-pacific/20131204/c04nkorea/| language =  |title = 情报显示朝鲜二号人物被免职 |publisher=纽约时报中文网 |date=2013-12-04 |accessdate =  }}</ref>。此外，张成泽也执掌了金正恩近卫部队——护卫总局的权力<ref>{{cite web |url = http://www.zaobao.com/wencui/2012/09/chosun120910a.shtml |language =  |title = 张成泽穿深棕色军服 证实其执掌护卫总局 |publisher=朝鲜日报 |date=2012-09-10 |accessdate =  }}</ref>。
