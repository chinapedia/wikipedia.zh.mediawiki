《'''张士超你到底把我家钥匙放哪里了？'''》，全称《'''张士超你昨天晚上到底把我家钥匙放在哪里了？'''》，是由[[上海彩虹室内合唱团|上海彩虹室内合唱团]]于2016年1月在“双城记 - 崔薇金承志作品专场”上演唱的一首[[安可|返场曲目]]。因调皮的歌词与严肃的曲风形成强烈的对比而走红网络，被网友称作“2016年第一首神曲”。<ref>[http://www.digitaling.com/articles/21960.html 2016第一首神曲《张士超你到底把我家钥匙放在哪里了》是什么鬼？].数英Digitaling.2016-01-21.[2016-01-23].</ref>

== 梗概 ==
《张士超你到底把我家钥匙放哪里了》由上海彩虹室内合唱团的指挥金承志一人编曲作词，耗时三个小时左右，是金承志于2015年11月<ref>{{cite news|author=顾欣悦|url=http://hzdaily.hangzhou.com.cn/dskb/html/2016-01/19/content_2178527.htm|title=作者金承志靠“黑”朋友一夜成名 每天在杭州排练的他说要多写写杭州|publisher=[[都市快报|都市快报]]|date=2016-01-19|accessdate=2016-01-31}}</ref>为“双城记 - 崔薇金承志作品专场”专门写的一个“不正经”的返场曲、小品曲目、调剂品。<ref name="shenquzhangshichao"/><ref>{{Citeweb|title=“神曲 张士超”背后的合唱团 |url=http://tv.sohu.com/20160118/n434873066.shtml |date=2016-01-18 |publisher=乐视}}</ref>全曲采用了男女八[[声部|声部]][[合唱|合唱]]<ref>{{Citeweb|title=合唱曲火了张士超 能否继《小苹果》后成为又一首神曲 |url=http://www.takefoto.cn/viewnews-656001.html |date=2016-01-19 |publisher=北晚新视觉}}</ref>，在表演中主要伴奏乐器是钢琴、{{le|非洲鼓|Djembe}}、[[手鼓|手鼓]]以及由每名合唱者演奏的[[卡祖笛|卡祖笛]]。在本歌曲的[[副歌|副歌]]“不用麻烦了”部分致敬了[[周杰伦|周杰伦]]的《[[牛仔很忙|牛仔很忙]]》。曲谱上记载的英文歌名《'''Where did you sleep last night mother-fucker?'''》也参照了[[涅槃乐队|涅槃乐队]]1993年演唱过的《{{link-en|昨夜你睡在哪里|In the Pines}}》。<ref>{{cite news|url=http://www.thepaper.cn/newsDetail_forward_1421155|title=华师大的姑娘，云师大的土豆，各个师大的肖秀荣们|publisher=[[澎湃新闻|澎湃新闻]]|date=2016-01-15|accessdate=2016-01-30}}</ref>

该曲讲述的是在2012年的一个秋天雨夜，自己回家时发现忘带钥匙，而室友张士超因为陪朋友，即曲中提到的“[[华东师范大学|华师大]]的姑娘”，而拒绝回来给自己开门，使得自己在雨夜中挨冻。歌曲中的主角张士超在现实中确有其人，他是作者金承志大学时期的挚友，而这首歌曲中发生的事情据张士超回忆其实是两件事。<ref>[http://www.weibo.com/1245412852/DdhzbDXqZ @张士超composer 的微博].新浪微博.2016-01-15.[2016-01-23].</ref>

== 演出与走红 ==
2016年1月9日，上海彩虹室内合唱团在[[上海音乐学院|上海音乐学院]]的[[贺绿汀音乐厅|贺绿汀音乐厅]]举办了“双城记 - 崔薇金承志作品专场”，《张士超你到底把我家钥匙放哪里了》作为全场的返场曲目由合唱团表演。此外，该音乐会上同由金承志创作的部分曲目使用[[温州话|温州话]]演唱，此举在国内的合唱作品中尚属首次。<ref name="oddend">{{cite web|author=钱睿荪|url=http://roll.sohu.com/20160118/n434945100.shtml|title=关于神曲《张士超你到底把我家钥匙放在哪里了》的二三事|publisher=界面|date=2016-01-18|accessdate=2016-01-30}}</ref>

歌曲在演出后被合唱团放到了[[微信|微信]]上，并在一周之内开始受到热捧，后被传到[[新浪微博|新浪微博]]上受到了更广泛的传播；而之后在[[网易云音乐|网易云音乐]]、微博音乐等网络音乐平台上迅速蹿红。<ref>[http://www.qdaily.com/articles/21112.html 谁创造了《张士超你到底把我家钥匙放在哪里了》和那“绝赞の优越感”].好奇心日报.2016-01-22.[2016-01-23].</ref>而作为作品中实名出现的张士超在该歌曲意外走红后其生活被影响和干扰，作为创作者的金承志在各个采访中都表示了歉意。<ref>[http://media.people.com.cn/n1/2016/0116/c40606-28059611.html 2016开年神曲 张士超被菜场阿姨问"钥匙放哪儿啦"].人民网.2016-01-16.[2016-01-23].</ref>合唱团也希望大家在关注这首歌之余也多关注更严肃的作品。2016年4月9日，彩虹室内合唱团在[[中山公园音乐堂|中山公园音乐堂]]演出，观众在合唱团演唱《张士超你昨天晚上到底把我家钥匙放在哪里了》期间表现出强烈反应，而在之前的其他作品演出期间反应平淡。<ref>{{cite news|url=http://bjrb.bjd.com.cn/html/2016-04/11/content_24574.htm|title=阳春白雪不敌搞笑的“张士超”|publisher=[[北京日报|北京日报]]|date=2016-04-11|accessdate=2016-04-15|deadurl=yes|archiveurl=https://web.archive.org/web/20160508150053/http://bjrb.bjd.com.cn/html/2016-04/11/content_24574.htm|archivedate=2016-05-08}}</ref>彩虹团2016年6月4日在[[东方艺术中心|东方艺术中心]]的演出也以“张士超你到底把我家钥匙放在哪里了”为题。

此外，该歌曲在爆红后也出现了一些重新填词的改编版本，如基于[[VOCALOID|VOCALOID]]改编、调侃2016年央视春晚节目组不邀请[[六小龄童|六小龄童]]一事的《[[2016年中国中央电视台春节联欢晚会|某晚会]]你到底把我们大圣放哪儿了》。

== 评论 ==
乐评人顾超评论该歌曲称，该曲内容不用于在一般合唱作品居多的古典、严肃内容，在题材、形式、技法上都接近大众审美，通俗、易打动人。[[国家大剧院|国家大剧院]]合唱团助理指挥焦淼认为，该曲是一首很有个性的作品，打破了大众对中国合唱的固有印象，拉近大众与合唱艺术的距离。音乐美学博士喻宇则称该曲的尝试有益于古典艺术的推广。<ref>{{Citeweb|title=一夜之间，一首《张士超你到底把我家钥匙放在哪里了》火速刷屏——“我们是在一本正经地胡说八道”|url=http://bjrb.bjd.com.cn/html/2016-01/18/content_345864.htm|date=2016-01-18|publisher=北京日报|deadurl=yes|archiveurl=https://web.archive.org/web/20160131231535/http://bjrb.bjd.com.cn/html/2016-01/18/content_345864.htm|archivedate=2016-01-31}}</ref>

此外，歌中提到的[[江湾五角场|五角场]]与[[华东师范大学闵行校区|华东师范大学闵行校区]]距离甚远，《[[中国青年报|中国青年报]]》在对大学新校区过于偏远这一问题的讨论中也提到了这首歌曲。<ref>{{cite news|author=沈杰群|url=http://zqb.cyol.com/html/2016-01/29/nw.D110000zgqnb_20160129_6-02.htm|title=张士超，你真乐意带姑娘去闵行吗|publisher=[[中国青年报|中国青年报]]|date=2016-01-29|accessdate=2016-01-29}}</ref>2016年1月26日《[[人民日报|人民日报]]》也曾评价过这首歌曲，认为这个组合虽然搞笑，但表演却不恶搞，对此曲予以肯定评价<ref>{{cite news|author1=《人民日报》|title=党报刊文评《张士超》：组合很搞笑表演却不恶搞，轻松而快乐|accessdate=2016-01-31|publisher=澎湃新闻网|date=2016-01-26}}</ref>。

在接受[[新京报|新京报]]采访时金承志曾谈到“虽然（流行）是一阵一阵的，但如果有人能通过这首歌喜欢古典音乐，对我们都是一件好事。我们也不是在做流行乐，我们主要还是在做古典音乐，只是偶尔写了一首比较流行的作品”<ref>{{cite news|author1=陈然|title=2016第一神曲诞生：《张士超你到底把我家钥匙放在哪里了》 张士超是谁？|url=http://sd.youth.cn/2016/0119/3661161.shtml|accessdate=2016-04-10|agency=新京报|publisher=中国网|date=2016-01-19|deadurl=yes|archiveurl=https://web.archive.org/web/20160421225712/http://sd.youth.cn/2016/0119/3661161.shtml|archivedate=2016-04-21}}</ref>。
