'''张延'''（{{bd|1970年|3月4日|||Z张}}），曾是第二批[[華南理工|華南理工]]派往香港[[無線電視|無線電視]]實習學生，现为电视剧演员及主持人，其配偶为香港艺员[[张锦程|张锦程]]。

== 早期生活 ==
张延1970年出生在西安，其父母为西安仪表厂工人，家中还有一个姐姐和一个哥哥<ref>{{cite web
|url= http://ent.sina.com.cn/s/h/2002-09-02/132898512.html
|title= 张延新浪访谈实录--不喜化妆 朴素亦新潮
|accessdate= 2008-08-24
|author= 新浪娱乐
|date= 2002-09-02
}}</ref>。小时候性格内向，从小因为长得高而学习跳舞，小学及中学都在西安仪表厂职工子弟学校求学，中考时考入陕西艺术团<ref>{{Cite web |url=http://et.21cn.com/tv/zongyi/2008/08/16/5076756.shtml |title=“男人婆”张延做客《家庭演播室》 |access-date=2008-08-25 |archive-url=https://web.archive.org/web/20130308065613/http://et.21cn.com/tv/zongyi/2008/08/16/5076756.shtml |archive-date=2013-03-08 |dead-url=yes }}</ref>，直到1987年9月进入广东舞蹈学校现代舞大专班学习现代舞<ref>[http://old.metropolis-mag.com/Article/biba/biba_jclm/biba_mlr/200505/Article_590_2.htm 张延，从男人婆到小女人]{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}之前张延曾前往北京舞蹈学院投考芭蕾舞科教育系，经老师劝说之后转考广东舞蹈学校</ref>，成为当年这一现代舞学习班首批学员，而张延也成了中国大陆第一代受过四年正规舞蹈学校的现代舞表演者<ref>{{cite web
|url= http://blog.sina.com.cn/s/blog_4c7b0fe5010009q9.html
|title= 中国第一代专业现代舞者
|accessdate= 2008-08-24
|author= 曹诚渊
|date= 2007-09-10
}}</ref>。当时班上的同学之后大都成为当今中国现代舞表演界的重要人物，包括了[[金星_(舞者)|金星]]、乔杨、马守则、[[沈偉|沈偉]]等<ref>{{cite web
|url= http://blog.sina.com.cn/s/blog_4c7b0fe5010009qm.html
|title= 中国第一代专业现代舞者
|accessdate= 2008-08-24
|author= 曹诚渊
|date= 2007-09-11
}}</ref>。

1990年张延参加了第二届广州“美在花城”选美大赛获得冠军。1991年6月张延从广东舞蹈学校毕业之后加入广东实验现代舞团<ref>{{cite web
|url= http://blog.sina.com.cn/s/blog_4c7b0fe5010009r3.html

== 个人生活 ==

张延的母亲是意外怀上张延的，曾经打算将其打掉，最后张延却生了下来，张延小时候还从火灾中幸存<ref name="born" />。张延原先去广州是学习芭蕾舞的，由于其身高较难找到舞伴因而转去学习现代舞<ref name="born" />。在张延广州上学期间，其父亲去世<ref name="born">{{cite web
 |url        = http://www.dhbc.net/datalib/qikan/2005/2005_06/qikan.2005-05-23.8626095190
 |title      = “男人婆”张延很女人，很幸福
 |accessdate = 2008-08-25
}}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，其父亲是一名军人曾参与[[朝鲜战争|朝鲜战争]]<ref>{{cite web
 |url         = http://www.zhaoni.cn/szyouth/read_6_640.html
 |title       = 柔情似火：“涩女郎”文火慢炖煲出好姻缘
 |accessdate  = 2008-08-25
 |deadurl     = yes
 |archiveurl  = https://web.archive.org/web/20080109233619/http://www.zhaoni.cn/szyouth/read_6_640.html
 |archivedate = 2008-01-09
}}</ref>。

1996年在上海拍摄《绕着神州走》节目时与共同主持节目的张锦程相识，之后于2001年两人重逢相恋，更在张延拍摄《粉红女郎》后成为其经纪人。2004年2月1日与张锦程于[[香港|香港]][[红棉路|红棉路]]婚姻注册处登记结婚<ref>[http://ent.qianlong.com/4543/2004/02/24/233@1901185.htm 张延谈老公:期望和他一起慢慢变老(组图)] {{Wayback|url=http://ent.qianlong.com/4543/2004/02/24/233@1901185.htm |date=20090809030957 }} 张延与张锦程在注册时，[[张家辉|张家辉]]、[[关咏荷|关咏荷]]、[[陈锦鸿|陈锦鸿]]和[[林家栋|林家栋]]为其证婚人。</ref>，2005年7月15日张延在香港剖腹生下2.89公斤的女儿<ref>[http://news.xinhuanet.com/ent/2005-07/17/content_3228925.htm 2005年7月9日张延在香港生下一女，体重2.89kg（6磅半）]</ref>。
