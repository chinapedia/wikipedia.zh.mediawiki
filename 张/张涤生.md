'''张涤生'''（{{bd|1916年|6月|2015年|8月19日|catIdx=Zhang張}}<ref name="sjtu">{{cite web |url=http://www.sjtu.edu.cn/info/1599/28051.htm |title=张涤生 |work=上海交通大学 |accessdate=2015-08-20 |archiveurl=https://www.webcitation.org/6autYxZFZ?url=http://www.sjtu.edu.cn/info/1599/28051.htm |archivedate=2015-08-20 |deadurl=no }}</ref><ref name="xmwb">{{cite web|url=http://xmwb.news365.com.cn/zh/201508/t20150820_2153525.html|title=我国整复外科奠基人张涤生院士逝世|work=新民晚报|date=2015-08-20|deadurl=yes|archiveurl=https://web.archive.org/web/20150821060345/http://xmwb.news365.com.cn/zh/201508/t20150820_2153525.html|archivedate=2015-08-21}}</ref>），中国医学家，[[中国工程院院士|中国工程院院士]]，中国[[整复外科|整复外科]]的重要创始人，被称为“中国[[整形外科|整形外科]]之父”。曾是[[上海交通大学医学院|上海交通大学医学院]]终身教授、[[博士生导师|博士生导师]]，[[上海市第九人民医院|上海市第九人民医院]]院长、整复外科主任，[[上海市整复外科研究所|上海市整复外科研究所]]所长、名誉所长。<ref>{{cite web|url=http://news.sjtu.edu.cn/info/1003/703554.htm|title=中国工程院院士张涤生逝世|work=上海交通大学新闻网|date=2015-08-20|access-date=2015-08-20|archive-url=https://web.archive.org/web/20160304185721/http://news.sjtu.edu.cn/info/1003/703554.htm|archive-date=2016-03-04|dead-url=yes}}</ref>

== 生平 ==
1916年6月12日生于[[江苏省|江苏省]][[无锡县|无锡县]]，父亲是银行[[会计|会计]]，幼年时随父在[[长春|长春]]、[[北京|北京]]、[[天津|天津]]等地生活。后回到无锡念初中，1932年就读于私立无锡中学。

1935年考入[[國立中央大學_(南京)|国立中央大学]]医学院（今[[第四军医大学|第四军医大学]]）{{NoteTag|1949年（民國38年），中华人民共和國成立前夕，國立中央大學被更名為國立南京大學，次年徑稱[[南京大學|南京大學]]。民國51年，[[國立中央大學|國立中央大學]]在臺灣復校。國立中央大學在大陸即為南京大學。南京大學医学院后改为第四军医大学。}}。1937年[[抗日战争|抗日战争]]爆发，随中大医学院西迁[[四川|四川]][[成都|成都]][[华西坝|华西坝]]。1941年毕业于国立中央大学医学院，获[[医学学士|医学学士]]学位。毕业后，在[[贵阳|贵阳]]任“[[中国红十字会|中国红十字会]]救护总队部”医师，从事治疗抗战伤员的工作。<ref name="sjtu" />

抗战胜利后，1946年至1948年，在[[美国|美国]][[宾夕法尼亚大学|宾夕法尼亚大学]]学习整形外科，师从现代整形外科先驱{{link-en|罗伯特·H·艾维|Robert H. Ivy}}<ref name="sjtu" />。

1948年回国，在美国教授来华举办的最早的整形外科进修班担任教员。1949年，在[[上海|上海]][[华中科技大学同济医学院|国立同济大学医学院]]任教。1950年，参加朝鲜战争中国医疗手术队，在长春成立了中国第一个[[烧伤整形|烧伤整形]]中心。

1956年，任上海[[广慈医院|广慈医院]]（后更名为瑞金医院）[[口腔颌面外科|口腔颌面外科]]主任，参与[[抢救丘财康|抢救丘财康]]，1964年担任[[整形外科|整形外科]]主任。1966年随科迁到上海[[上海交通大学医学院附属第九人民医院|第九人民医院]]，先后担任[[上海第二医科大学|上海第二医科大学]]附属[[上海交通大学医学院附属第九人民医院|第九人民医院]]整复外科主任、第九人民医院院长；创办[[上海市整复外科研究所|上海市整复外科研究所]]，任所长。他主持的上海整复外科研究所，曾被国际权威整形外科专家称为“可同世界上任何一个同类专业中心媲美”<ref name="sjtu2">{{cite web |url = http://news.sjtu.edu.cn/info/1006/110658.htm |title = 张涤生：平生志业皆在其中 |website = 上海交通大学 |date = 2011-12-31 |access-date = 2015-08-20 |archive-url = https://web.archive.org/web/20150801110352/http://news.sjtu.edu.cn/info/1006/110658.htm |archive-date = 2015-08-01 |dead-url = yes }}</ref>。

1996年，当选为[[中国工程院|中国工程院]]医药卫生学部院士<ref name="sjtu2" />。

== 学术成就 ==
张涤生创立了“烘绑疗法”概念，开创了中国淋巴学科先河。他是中国第一个应用显微外科技术进行科学实验的人，1964年开始做吻合小血管游离皮瓣的动物实验，1970年代初期逐步将显微外科应用于临床，是中国显微外科的创始人之一。<ref name="sjtu"/>

张涤生将[[显微外科|显微外科]]与[[整形外科|整形外科]]有机的结合，突破传统“整形外科”的观念，倡导把外形的修整与功能的恢复有机地统一起来，形成了以外形修整与功能恢复有机统一的“整复外科”新概念，从而使传统的整形外科得到空前的发展。 他认为，“整”是指对人体组织器官缺损[[畸形|畸形]]在形态上进行修整；“复”则是使病人在生理功能上进行最大限度的恢复。在张涤生的倡导下，整复外科已突破了以往“颌面整形”和“美容”的范畴，扩大了治疗范围，处理范围涉及各种先天或创伤的四肢畸形、颅颌面畸形以及烧伤畸形。<ref name="dxy">{{cite web|url=http://surg.dxy.cn/article/138888|title=沉痛哀悼：中国整复外科创始人张涤生院士逝世|work=丁香园|date=2015-08-20|accessdate=2015-08-20}}</ref>
