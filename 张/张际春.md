'''张际春'''（{{bd|1900年|12月20日|1968年|9月12日|catIdx=Zhang张}}），[[湖南|湖南]][[宜章|宜章]]人，中华人民共和国政治人物。

张际春1926年加入[[中国共产党|中国共产党]]。曾任[[红五军团|红五军团]]宣传部长、[[红五军|红五军]]宣传部长、[[红军大学|红军大学]]政治部主任兼政治主任教员等，参加[[长征|长征]]。[[抗日战争|抗日战争]]期间于1937年任[[八路军|八路军]]后方政治部副主任、[[抗日军政大学|抗日军政大学]]政治部主任、校党务委员会书记、校政治委员。[[第二次国共内战|第二次国共内战]]期间任[[晋冀鲁豫军区|晋冀鲁豫军区]]及野战军副政委兼政治部主任、[[第二野战军|第二野战军]]副政委兼政治部主任。[[中华人民共和国|中华人民共和国]]成立后，担任过[[中共中央宣傳部|中宣部]]常务副部长、[[中華人民共和國國務院|国务院]]文教办公室主任等职。[[文化大革命|文化大革命]]期间受到迫害，1968年在北京逝世。

== 生平 ==
=== 早年经历 ===
张际春早年曾在永福初级小学、县立阁邑高等小学就读1920年到[[衡阳|衡阳]][[湖南省立第三师范学校|湖南省立第三师范学校]]读书。期间常因经济困难时而辍学。1922年后受革命思想影响，参加学生运动和农民运动。1925年毕业后，在本乡小学任教，兼办农民协会。1926年加入[[中国国民党|中国国民党]]，任区党部常务委员。1926年11月加入[[中国共产党|中国共产党]]，1927年初到[[宜章县立女子初级师范学校|宜章县立女子初级师范学校]]任教，同时任国民党县党部常务委员<ref>{{cite book |title=中国二十世纪纪事本末|volume = 附卷·人物 |editor = 周鸿、朱汉国 主编 |location = 济南 |publisher=山东人民出版社 |year=2000 |page = 381 |ISBN = 7-209-02403-4}}</ref>。

1928年1月，张际春参加[[湘南起义|湘南起义]]，任中共宜章县委农民运动委员会书记和县农民协会秘书长。4月随军上[[井冈山|井冈山]]，先后在[[红四军_(红一方面军)|红四军]]第三十二师党委会、第十师第二十八团党委会工作<ref>{{cite book|title=井冈山红军人物志|author=饶道良|publisher=122|location=南昌|publisher=江西人民出版社|year=2010|isbn=978-7-210-04457-4}}</ref>。1929年初，调任红四军第二纵队政治部宣传科科长，随[[朱德|朱德]]、[[毛泽东|毛泽东]]进军赣南闽西<ref>{{cite book|title=星火燎原 Volume6|pages=1-6|location=北京|publisher=解放军出版社|year=1979|isbn=7-5065-3276-X}}</ref>。1930年6月任红四军秘书长，同年8月任红一方面军政治部宣传处处长。1930年12月调任[[中央军委|中央军委]]总政治部组织部干事兼机关党支部书记。

1931年春，张际春调任[[红三军|红三军]]秘书长。5月任红四军第十一师政治委员兼政治部主任，同年9月任红四军第十三师政治委员兼政治部主任。1932年3月任第四十五师政治委员兼政治部主任，月任红一军团第三十一师政治委员兼政治部主任。12月调任红一军团政治部宣传部部长，主办《[[战士报|战士报]]》。1933年兼任红一军团军事裁判所所长。参加中央苏区[[第一次国共内战|历次反围剿战争]]。1934年初任[[红军第二步兵学校|红军第二步兵学校]]政治委员兼政治处主任。同年夏因抵制[[博古|博古]]等人的指挥，受到排斥和打击，改任校训育处副处长。不久调任红五军团政治部宣传部部长<ref name="a1"/>。

1934年10月，随[[红五军团|红五军团]]第十三师第三十七团[[长征|长征]]。1935年10月，任[[红四方面军|红四方面军]][[红军大学|红军大学]]政治主任教员。1936年6月任红军大学政治部主任。1937年1月，张际春入[[抗日军政大学|抗日军政大学]]第二期第二队学习，后任附设步兵学校政治部主任<ref name="a1"/>。
[[File:1939ChenLuoZhang.jpg|thumb]]、[[罗瑞卿|罗瑞卿]]、张际春（左起）合影]]
=== 抗日战争时期 ===
1937年8月，张际春任[[八路军|八路军]]后方政治部副主任。11月调任[[绥德|绥德]]河防警备区政治部主任，兼中共绥德特别区委委员。1938年1月，任抗大政治部主任，同[[罗瑞卿|罗瑞卿]]一起负责抗大的实际领导工作<ref name="a1">{{cite book|title=湘籍开国人物传略 Volume1|editor=夏远生|pages=1043-1051|location=长沙|publisher=湖南人民出版社|year=2009|isbn=7-5438-5788-X}}</ref>。1939年7月，和罗瑞卿等率抗大总校向太行地区迁移<ref>{{cite book|language=zh-hans|author=《罗瑞卿传》编写组|title=罗瑞卿传|pages=71-75|year=2007|location=北京|publisher=当代中国出版社|isbn=978-7-8009-2501-6}}</ref>。1942年9月起，张际春任抗大代理政治委员兼军政委员会主席。1943年1月，任[[中共中央北方局|中共中央北方局]]宣传部部长，同年9月起任八路军野战政治部副主任，兼做北方局党校的整风审干工作。1945年4月，返回[[延安|延安]]出席[[中共七大|中共七大]]，并当选为[[中国共产党第七届中央委员会候补委员列表|中央候补委员]]<ref>{{cite web|accessdate=2015-07-08|title=第七届中央委员会（1945年6月-1956年9月）|publisher=中国共产党新闻网|url=http://cpc.people.com.cn/GB/64162/139962/8437280.html}}</ref>。

=== 第二次国共内战时期 ===
[[File:1949LiuDengZhang.jpg|thumb]]（右）、[[邓小平|邓小平]]（左）、张际春（中）在第二野战军指挥部。]]
1945年8月，张际春任晋冀鲁豫军区副政治委员兼任政治部主任<ref>{{cite book |author=张明金、刘立勤 |title=中国人民解放军历史上的200个军区|year=2010 |publisher=解放军文艺出版社 |location=北京 |page = 221 |ISBN = 978-7-5033-2252-5}}</ref>。参与领导[[上党战役|上党]]、[[邯郸战役|邯郸]]等战役，总结推广[[王克勤|王克勤]]班团结互助运动等经验。1947年6月，随[[刘伯承|刘伯承]]、[[邓小平|邓小平]]率部进军[[大别山|大别山]]。1948年5月，张际春任中原军区、[[中原野战军|中原野战军]]第二副政治委员兼政治部主任，参加[[淮海战役|淮海战役]]<ref>{{cite book|title=淮海战役 第2卷 将军谱|editor=傅继俊主编|pages=40|location=南京|publisher=江苏人民出版社|year=2009|isbn=978-7-214-05576-7}}</ref>。1949年2月，任[[第二野战军|第二野战军]]副政治委员兼政治部主任，参加[[渡江战役|渡江战役]]<ref>{{cite book|author=《中国人民解放军军史》编写组|title=中国人民解放军军史：第三卷|year=2010|publisher=军事科学出版社|pages=324|isbn=978-7-80237-381-5}}</ref>。攻占南京后，任中共南京市委常务委员兼宣传部部长，9月随部队向西南进军。协助刘伯承、邓小平制定作战方案，组织指挥部队。

=== 中华人民共和国成立后 ===
1949年12月，张际春兼任[[重庆市|重庆市]]军事管制委员会主任<ref>{{cite book|title=重庆解放|author=唐润明，艾新全著|pages=218-219|location=北京|publisher=中国档案出版社|year=2009|isbn=978-7-5105-0108-1}}</ref>。1950年2月，任[[中共中央西南局|中共中央西南局]]常委兼西南局办公厅主任、组织部部长、西南军区副政治委员兼政治部主任兼干部管理部部长。1952年9月，任中共中央西南局第二副书记兼西南局农村工作委员会书记、农村工作部部长。1954年1月，张际春调入中央，任[[中共中央宣传部|中共中央宣传部]]副部长。1957年6月，兼任[[国务院第二办公室|国务院第二办公室]]副主任<ref>{{cite book |title=中国革命史手册|author=陈荣华 |year=1986 |publisher = 华中师范大学出版社 |page = 359}}</ref>。1958年6月起任[[中共中央文教小组|中共中央文教小组]]成员。1959年6月兼任国务院文教办公室主任<ref name="a1"/>。

张际春是第二、第三届[[全国政协|全国政协]]常务委员，[[中共第八届中央委员|中共第八届中央委员]]。[[文化大革命|文化大革命]]期间，张际春被诬为 “黑帮”、 “活阎王”、 “反革命修正主义分子”等等，并于1966年7月30日在[[人民大会堂|人民大会堂]]召开万人大会，对他进行批判斗争。1968年9月12日在北京逝世。1979年1月24日，中共中央在全国政协礼堂为他举行了平反昭雪追悼大会<ref>{{cite web |url = http://www.czxww.cn/2011/06/19/210528278595.html |title = 张际春 |publisher = 彬州新闻网 |date = 2011-06-19|accessdate=2015-07-08}}</ref>。
