'''张玉娘'''（{{bd|1250年||1277年}}），字'''若琼'''，自号'''贞一居士'''（一说号一贞居士），[[浙江|浙江]][[松阳|松阳]]人，宋末女诗人，擅长诗词。與[[李清照|李清照]]、[[朱淑貞|朱淑貞]]、[[吳淑姬|吳淑姬]]並稱「宋代四大女詞人」。

==生平==
宋淳祐十年(1250年)生于书香门第，父张懋，仕宋为提举官，母刘氏。及笄，与状元[[沈晦|沈晦]]七世孙[[沈佺_(宋代)|沈佺]]订立婚约，后沈佺病卒于京师，玉娘矢志守节，不食而死，与沈佺合葬于县南官塘桥外枫树林。玉娘所养鹦鹉及侍婢紫娥、霜娥皆悲恸而死，为其殉葬，二娥葬于墓左，鹦鹉葬于墓右，其墓遂名鹦鹉冢<ref>清·支恒春等纂修，《光绪松阳县志·卷十·烈女》：张玉娘，字若琼，父授以《孝经》、《女训》，过目成诵。盖张为宦族，积书遗艺，玉娘窃玩益深，至其所作词文，不下汉之班姫，遂号贞一居士。父母择配为状元沈晦七世孙、榜眼沈佺之妻。未婚而沈生宦游京师，病羸不起。时玉娘年二十四矣，矢志守节，临帷哀恸，恨不同死。忽夜梦沈生驾车相迎，即披衣起坐，谓侍儿曰：“吾事定矣”。未逾月，竟不食而殒。父母痛之，与生并葬于附郭枫林。玉娘旧玩能言鹦鹉及侍儿紫娥、霜娥饮恨悲鸣，相继死，并遗以殉葬，名其冢曰鹦鹉冢。玉娘遗稿曰《兰雪集》，或有携上京师者，学士虞伯生、欧阳元读其诗至“山之高，月出小。月之小，何皎皎！我有所思在远道。一日不见兮，忧心悄悄”，抚几叹曰：“可与《国风·草虫》并称，岂妇人女子之所能及耶？”</ref><ref>{{cite book |author=松阳县志编纂委员会|title=《松阳县志》|year=1996|publisher=浙江人民出版社|location=杭州|isbn=7-213-01211-8|pages=584}}</ref>，明代即其墓建贞文祠（又名鹦鹉庙）<ref>民国·吕耀钤等修、高焕然等纂，《民国松阳县志·卷二·建置》：贞文祠旧名鹦鹉庙，在城南官塘桥外，关岳庙右。……明处士沈承意拨田立祀，建造祠宇，内曰贞文祠，外曰准提庵，招僧承管。年久湮没，清顺治间训导孟称舜复捐田亩，废庵建祠，修改墓道，赖存名迹。……贞女张玉娘鹦鹉冢在县南官塘桥外。</ref>，今松阳环城西路县人民医院南侧尚存鹦鹉冢断碑及兰雪井。
==著作==
著有《[[兰雪集|兰雪集]]》收诗117首、词16阙，由其族孙张献集录。其诗词后传入京师，元诗四大家之一的[[虞集|虞集]]称其《山之高》篇“可与《国风·草虫》并称，岂妇人女子之所能及”。文学研究家[[谭正壁|谭正壁]]在《中国女性的文学生活》一文中将其与[[李清照|李清照]]、[[朱淑真|朱淑真]]、[[吴淑姬|吴淑姬]]并称为“宋代四大女词家”<ref>{{cite web |url=http://www.zjww.gov.cn/magazine/747348.shtml|title =《南宋女诗人张玉娘》|accessdate=2016-07-13|author=潘贤达|publisher=浙江省文物局}}</ref>。《蘭雪集》原本和清初孟稱舜刊本均已佚失，現今保存在松陽縣葉村鄉葉村沈氏後裔沈宗雷家的《蘭雪集》版本，清光緒八年（西元1882年）松陽縣署刊印的，《跋》是沈作霖于道光二十六年（西元1846年）寫的。沈作霖是《蘭雪集》作者張玉娘未婚夫家的後代，也是這個刊本的重刊者。同時列入編輯名單的是饒慶霖編次，葉維蕃校訂。饒、葉均當地的名人。此本更在扉頁注明：「版存松州修凝堂沈宅」，此版本歷三百年而後顯於世，可見它的價值。《蘭雪集》刊本，至今較好是陶湘於民國十四年(1925年)的刊本。編在《托跋廛叢刻》，分上、下卷，有詩詞133首（闋）。後附有沈佺贈張詩一首、王詔《張玉娘傳》。

==傳記==
明嘉靖十二年（1533），松阳学者王诏撰《张玉娘传》附于《兰雪集》后，明末清初，戏剧家[[孟称舜|孟称舜]]任[[松阳文庙|松阳儒学]]训导时重建贞文祠，重修墓道，复刊《兰雪集》，并于崇祯十六年（1643）作传奇剧本《张玉娘闺房三清鹦鹉墓贞文记》（简称《[[贞文记|贞文记]]》）2卷35出，使其传说广为流传<ref>{{cite web |url=http://zjrb.zjol.com.cn/html/2014-12/02/content_2829815.htm?div=-1|title =《一个深埋了800年的凄美故事——“松阳梁祝”鹦鹉冢》|accessdate=2016-07-12|author=金春华|publisher=浙江日报}}</ref>，今松阳张玉娘传奇已被列为丽水市第六批市级非物质文化遗产名录<ref>{{cite web |url=http://www.zjfeiyi.cn/news/detail/31-9584.html|title =《松阳县新增4项市级非物质文化遗产名录》|accessdate=2016-07-12|author= |publisher=浙江非物质文化遗产网}}</ref>。
