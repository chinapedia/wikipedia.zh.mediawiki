'''張克輝'''（{{bd|1928年|2月|catIdx=Z张}}），原名'''張有義'''，[[臺灣|臺灣]][[彰化市|彰化]]人，[[中华人民共和国|中华人民共和国]]政治人物，是至今为止在[[中國大陆|中國大陆]]政治地位最高的台灣人，曾任[[全國政協副主席|全國政協副主席]]，[[臺灣民主自治同盟|臺灣民主自治同盟]]主席<ref>{{Cite book |title=一九四六被遺忘的台籍青年| author=鐘明宏|publisher=沐風文化|date=2014-11-03}}</ref>。

==生平==
張克輝1928年生於[[臺灣日治時期|日治時期]]的臺灣。1942年就讀[[國立彰化高級商業職業學校|彰化商業學校]]，1947年就讀[[臺灣師範大學|臺灣省立師範學院]]。1948年至1949年，在[[福建|福建]][[廈門大學|廈門大學]]經濟系就讀，後任[[中国人民解放军|中国人民解放军]]閩粵贛纵队八支四团十四独立连连长、副指导员<ref name=jianli>{{Cite web |url =  http://www.people.com.cn/GB/shizheng/252/9667/9685/20021129/877688.html|title =张克辉简历  |author =  |date =  |publisher =人民网  |language =  |accessdate =  2017-10-13|quote =  }}</ref>。1948年加入[[中国共产党|中国共产党]]<ref>{{Cite web |url = http://cpc.people.com.cn/GB/7301897.html |title =张克辉（１９２８—　　）  |author =  |date =  |publisher = 中国共产党新闻网 |language =  |accessdate = 2017-10-13 |quote =  }}</ref>。

1949年至1951年任[[福建省|福建省]][[安溪县|安溪县]]人民公安队指导员、县公安局股长。1951年至1952年在[[中共福建省委|中共福建省委]]干校学习，后任中共福建省委[[土改|土改]]工作队组长，后又到中共福建省委党校学习。1952年至1969年任中共福建省委统战部干事、组长。1969年至1973年下放福建省[[宁化县|宁化县]]劳动，后任宁化县革命委员会宣传组副组长。1973年至1978年任[[福建省革命委员会|福建省革命委员会]]外事组翻译<ref name=cppcc/><ref name=jianli/>。

1978年至1982年任[[福建省政协|福建省政协]]副秘书长，[[福建省人民政府侨务办公室|福建省人民政府侨务办公室]]、[[福建省人民政府外事办公室|福建省人民政府外事办公室]]副主任、福建省[[台盟|台盟]]主委、福建省台湾同胞联谊会副会长。1982年至1991年任中共福建省委常委兼统战部部长，福建省对台工作办公室主任，福建省政协副主席、福建省社会主义学院院长，[[国务院台湾事务办公室|国务院台湾事务办公室]]副主任<ref name=cppcc/><ref name=jianli/>。

1991年至1997年任[[中华全国台湾同胞联谊会|中华全国台湾同胞联谊会]]会长、[[海峡两岸交流协会|海峡两岸交流协会]]顾问、台盟中央副主席。1997年任[[台盟|台盟]]中央主席，[[全国台联|全国台联]]名誉会长。1998年3月当选为第九届[[全国政协副主席|全国政协副主席]]。同年任[[中国和平统一促进会|中国和平统一促进会]]会长。2003年3月連任第十届全国政协副主席。2005年12月卸任台盟中央主席，改任台盟中央名譽主席，由[[林文漪|林文漪]]接任台盟中央主席<ref name=cppcc/><ref name=jianli/>。

他是第五届、七届全国政协委员，第九届、第十届全国政协副主席。第八届全国人大常委会委员、[[全国人大内务司法委员会|全国人大内务司法委员会]]副主任委员<ref name=cppcc>{{cite web|url=http://www.cppcc.gov.cn/2011/09/28/ARTI1317196721328135.shtml|title=张克辉|publisher=中国政协网|date=2011-09-28}}</ref><ref name=jianli/>。

他还是位剧作家，曾写出多种电影文学剧本。其中电影文学剧本《寻找》被改编为电影《[[云水谣|云水谣]]》<ref>{{Cite web |url = http://cppcc.people.com.cn/GB/34952/3954882.html |title =全国政协副主席张克辉讲述跨越时空的爱情史诗  |author = 赵文娟 |date =  2005-12-19|publisher =  中国政协新闻网|language =  |accessdate =  |quote =  }}</ref>。

1993年，他到台湾为父亲奔丧，在离别45年后第一次回到家乡。2009年5月，他携夫人第二次回台湾，此行张克辉以中华[[妈祖|妈祖]]文化交流协会会长的身分，应[[大甲镇澜宫|大甲镇澜宫]]邀请率团来台参访，受到大甲镇澜宫董事长[[颜清标|颜清标]]迎接，同[[王金平|王金平]]晚宴，还在彰商校友会举办的“六十年同学会”上见到了前[[中华民国财政部|中华民国财政部]]部长[[李述德|李述德]]、前[[民进党|民进党]]主席[[姚嘉文|姚嘉文]]、自称无党派的新光金控副董事长[[吴家录|吴家录]]、李登辉之友会长[[蔡焜灿|蔡焜灿]]等老同学、老校友<ref name=fantai/><ref>{{Cite web |url =  http://www.taihainet.com/news/twnews/bilateral/2009-05-30/412708.html|title = 红蓝绿大和解！同学宴请张克辉 姚嘉文参加 |author =  |date = 2009-05-30 |publisher =台海网  |language =  |accessdate =  |quote =  }}</ref>。

==家庭==
张克辉到大陆后，由于两岸因政治对立长期分隔，很长时间无法见到父母。1979年冬，张克辉在[[日本|日本]][[东京|东京]]与父母首次秘密会面。此后，张克辉每次到日本时，父母都会来日本和他见面。他们之间还经常通过电话联系。1992年，张克辉的母亲逝世，张克辉作为长子提出回台湾奔丧，但台湾当局因张克辉的政治身份而加以拒绝，张克辉乃让妻子代其回台奔丧，其妻奔丧后回到大陆时，将张克辉的父亲也接到大陆。父亲和张克辉共同度过了近半年，直到冬天才回台湾过冬。原定次年即1993年5月父亲还会再来大陆，但父亲突然逝世。这时张克辉又提出回台湾奔丧，台湾当局仍不同意。后经张克辉的亲属、台湾各界人士及社会舆论努力，台湾当局最终同意张克辉回台湾奔丧。此事当时是台湾岛内轰动一时的大新闻。张克辉终于踏上了离别四十五年的台湾故乡的土地<ref>{{Cite web |url = http://www.tuanjiebao.com/tjej/2015-12/15/content_41352.htm |title =  张克辉最后的孝道|author =  |date =2015-12-15  |publisher = 团结网 |language =  |accessdate =  |quote =  }}</ref>。
