'''张秀萍'''{{BD|1965年|2月|||catIdx=Z张}}女性，[[山西人|山西人]]，毕业于[[山西大学|山西大学]]科技哲学研究所科技哲学专业，曾担任曾任山西省纪委副秘书长、监察综合室主任、常委，中共晋中市委副书记，2014年8月，她被[[中纪委|中纪委]]调查，澎湃新闻称其或与[[金道铭|金道铭]]有关系。<ref>{{cite web |url= http://www.thepaper.cn/newsDetail_forward_1261906|title= 涉金道铭案，晋中市委副书记张秀萍从市委领导名单中“消失” |accessdate=2014 |publisher= 澎湃新闻|work= |language=zh-hans}}</ref>2014年11月，因“利用职务便利，为他人谋取利益，非法索取、收受他人财物，且数额巨大，情节严重；收受礼金；与他人[[通奸|通奸]]”等罪被移送司法机关处理。<ref>{{cite web |url=http://news.163.com/14/1127/10/AC27CAD600014JB5.html#f=dupdate |title=山西晋中女副书记张秀萍涉受贿"通奸"被逮捕(图) |accessdate= 2014|publisher= 网易|work= |language=zh-hans}}</ref>

== 生平 ==
1965年12月，张秀萍出生在[[山西省|山西省]][[山阴县|山阴县]]。张秀萍毕业于[[山西大学|山西大学]]科技哲学研究所科技哲学专业，拥有哲学博士学位。

1985年6月，张秀萍加入[[中国共产党|中国共产党]]，1989年7月参加工作。

1989年7月到1995年1月，张秀萍在[[朔州市|朔州市]]委组织部工作。1998年4月，张秀萍升任朔州市委办公室副主任。

2000年3月，张秀萍调入中共山西省纪委、监委工作，2002年8月担任中共山西省纪委、监委监察综合室主任。

2006年10月，张秀萍担任中共山西省纪委常委。

2013年4月到2014年8月，张秀萍升任中共[[晋中市|晋中市]]委副书记。

2014年8月，她被[[中纪委|中纪委]]调查。2014年11月，因“利用职务便利，为他人谋取利益，非法索取、收受他人财物，且数额巨大，情节严重；收受礼金；与他人[[通奸|通奸]]”等罪被[[双开|双开]]，然后移送司法机关处理。<ref>{{cite web |url=http://news.xinhuanet.com/legal/ttgg/2014-11/26/c_127253901.htm |title=山西晋中原女副书记张秀萍被双开 与他人通奸  |accessdate=2014 |publisher=新华网 |work= |language=zh-hans}}</ref>
