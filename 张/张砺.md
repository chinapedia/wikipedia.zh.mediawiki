'''张砺'''（{{bd|9世纪||947年|6月21日|catIdx=Z}}<ref name=ZZTJ287>[[:s:資治通鑑/卷287|《资治通鉴》卷二百八十七]]</ref>），字'''梦臣'''，[[中国|中国]][[五代十国|五代十国]][[政权|政权]][[后唐|后唐]]和[[辽朝|辽朝]][[官员|官员]]。

== 家世 ==
张砺生年不详，[[滏阳|滏阳]]人。祖父张庆，父张宝，世代为农。张砺年幼时好学有文采。他还是百姓时，目睹百姓争执，就亲自去官府，像官员一样辨别曲直。<ref name=HFD98>[[:s:舊五代史/卷98|《旧五代史》卷九十八]]</ref>

== 后梁年间 ==
张砺起初依附[[后梁|后梁]]大臣[[李愚|李愚]]。[[贞明_(后梁)|贞明]]年间（915年－921年），从[[河阳|河阳]]北上投靠晋王[[李存勖|李存勖]]，得补授[[太原|太原]]府掾，并称赞李愚的节操和文章，晋国于是称道李愚。<ref>[[:s:舊五代史/卷67|《旧五代史》卷六十七]]</ref>

== 后唐年间 ==
李存勖于[[同光|同光]]元年建立后唐，即[[后唐庄宗|后唐庄宗]]，同年灭梁。同光年间（923年－926年），张砺中[[进士|进士]]，不久拜左拾遗，在史馆修史。<ref name=HFD98/>三年（925年），皇子魏王[[李继岌|李继岌]]为名义统帅讨伐西南邻国[[前蜀|前蜀]]，实际主事的[[枢密使|枢密使]][[郭崇韬|郭崇韬]]<ref>[[:s:資治通鑑/卷273|《资治通鉴》卷二百七十三]]</ref>奏请张砺掌军中文书。<ref name=HFD98/>唐军灭蜀后，李继岌奉母[[刘玉娘|刘皇后]]教令杀郭崇韬，郭崇韬亲信惧祸奔逃，只有张砺去魏王府第为郭崇韬而哭。<ref name=ZZTJ274>[[:s:資治通鑑/卷274|《资治通鉴》卷二百七十四]]</ref>

四年（926年）正月，李继岌开始率军班师都城[[洛阳|洛阳]]，二月，排陈斩斫使[[李绍琛|李绍琛]]反叛。李继岌派善于军事的工部尚书[[任圜|任圜]]讨伐之。张砺建议任圜把精兵埋伏在后，以弱兵诱敌，任圜从之，败李绍琛，迫其奔[[汉州|汉州]]。任圜随后破城生擒之。<ref name=ZZTJ274/>李存勖下诏令处死李绍琛，四月诏到时，洛阳已生[[兵变|兵变]]，监军[[李从袭|李从袭]]为了与任圜争功，想留下李绍琛的性命。任圜犹豫，张砺说：“此贼构乱导致班师延迟，且明公血战擒贼，怎能违诏养祸，这是破槛放虎，自贻其咎啊。公若不决，我亲自杀此贼。”任圜不得已而杀了李绍琛。<ref name=HFD98/>

先前，郭崇韬与大将[[李继麟|李继麟]]被冤杀导致后唐多起兵变。<ref name=ZZTJ274/>四月，李存勖死于[[兴教门之变|洛阳兵变]]，其中一起兵变的头领李存勖养兄[[李嗣源|李嗣源]]接管洛阳，先称[[监国|监国]]，再称帝。李继岌想和李嗣源交战，但士兵开始离散，他自杀。<ref>[[:s:資治通鑑/卷275|《资治通鉴》卷二百七十五]]</ref>李嗣源[[天成_(后唐)|天成]]年间（926年－930年），李嗣源闻张砺名，召为[[翰林学士|翰林学士]]。张砺因父母过世服丧，期满，再入为学士，历充礼部、兵部员外郎、知制诰。不久，张宝妾去世。她在世时，因侍奉张宝多年，张砺很敬重她，他的儿子们也称她为祖母。张砺不知道该如何服丧，问同僚，无人能给他建议，于是他请假回滏阳，不穿丧服而闲居三年，为有识之士所称许。<ref name=HFD98/>当时常侍[[张文宝|张文宝]]主管[[科举|科举]]，[[中书省|中书省]]奏落榜进士数人，请诏翰林学士院作一诗一赋下达礼部给[[举人|举人]]作为范文。张砺与学士窦梦徵等撰格诗格赋各一送给中书省，宰相认为不可，窦梦徵等请户部侍郎右丞充承旨[[李怿_(五代)|李怿]]来写，李怿笑而辞之。<ref>[[:s:舊五代史/卷92|《旧五代史》卷九十二]]</ref><ref>[[:s:新五代史/卷55|《新五代史》卷五十五]]</ref>

李嗣源养子[[李从珂|李从珂]][[清泰|清泰]]年间（934年－936年），张砺被授尚书比部郎中、知制诰，依前充学士。<ref name=HFD98/>元年（934年），与翰林学士[[程逊|程逊]]、学士[[和凝|和凝]]等上书十三事。<ref>[[:s:覆奏程逊等陈时务奏|:s:覆奏程逊等陈时务奏]]</ref>三年（936年），李嗣源婿[[河东节度使|河东节度使]][[石敬瑭|石敬瑭]]在后唐北邻敌国[[辽朝|契丹]]支持下反叛，李从珂派李嗣源另一女婿[[枢密使|枢密使]][[赵延寿|赵延寿]]等人讨伐，<ref name=ZZTJ280>[[:s:資治通鑑/卷280|《资治通鉴》卷二百八十]]</ref>命翰林学士和凝与赵延寿同行。张砺素来轻视和凝，请求自己去，李从珂慰劳他，并同意了，于是以赵延寿为河东道南面行营招讨使，张砺为判官。<ref name=HFD98/>后赵延寿与其养父[[赵德钧_(五代)|赵德钧]]为[[辽太宗|契丹太宗]]所败，张砺与赵氏父子都被俘虏到契丹。李从珂绝望自杀，唐亡，其地被石敬瑭建立的[[后晋|后晋]]接管。<ref name=ZZTJ280/><ref name=NHFD72>[[:s:新五代史/卷72|《新五代史》卷七十二]]</ref>

== 辽朝年间 ==
契丹太宗见张砺刚直有文采，擢翰林学士。<ref name=ZZTJ280/>他遇事必尽言，无所避，太宗愈发器重他。<ref name=HL76>[[:s:遼史/卷76|《辽史》卷七十六]]</ref>[[天显|天显]]十一年（937年），张砺逃跑，被追兵所获，太宗斥责他：“为什么舍我而去！”张砺答：“臣是中原人，饮食衣服都与这里不一样，生不如死，愿早就戮。”太宗斥责负责善待中原俘虏的通事高彦英（或作高唐英），鞭打了他一百下，向张砺谢罪，待之如初。<ref name=NHFD72/><ref name=ZZTJ281>[[:s:資治通鑑/卷281|《资治通鉴》卷二百八十一]]</ref>[[会同|会同]]年间（938年－947年），升翰林承旨，兼吏部尚书。<ref name=HL76/>

石敬瑭死后，其侄[[石重贵|石重贵]]继位，当时契丹已改国号辽，石重贵不再依附辽，与辽对抗。<ref name=ZZTJ283>[[:s:資治通鑑/卷283|《资治通鉴》卷二百八十三]]</ref>会同九年（946年），[[辽太宗|辽太宗]]大举南下，打败石重贵姑父后晋大将[[杜重威|杜威]]并迫其投降，逼近晋都[[开封|开封]]，石重贵绝望投降，晋亡，领地被辽所占。辽军逼近开封时，故[[皇后|皇后]][[萧温|萧温]]兄[[萧翰|萧翰]]、[[耶律郎五|耶律郎五]]、太宗从弟[[耶律麻荅|耶律麻荅]]等将肆意杀掠，随征的张砺对太宗说：“今大辽已得天下，应该用中原人为中原将相，不宜用北人（契丹人）及左右近臣。一旦政令有失，则人心不服，即使现在得到了，也将失去。”但太宗不从。<ref name=HL76/><ref name=ZZTJ285>[[:s:資治通鑑/卷285|《资治通鉴》卷二百八十五]]</ref>十年（947年）二月太宗入开封后，称中原皇帝，张砺和赵延寿推荐后晋[[宰相|宰相]][[李崧_(五代)|李崧]]，李崧被任为[[太子太师|太子太师]]，充枢密使。<ref>[[:s:舊五代史/卷108|《旧五代史》卷一百零八]]</ref>张砺又荐后晋屯田员外郎、知制诰[[鱼崇谅|鱼崇谅]]为翰林学士。<ref>[[:s:宋史/卷269|《宋史》卷二百六十九]]</ref>赵延寿被辽封为燕王，想做中原皇帝，未得太宗支持，又请求为[[皇太子|皇太子]]，太宗认为皇太子只有皇子才能做，命迁赵延寿官。张砺奏拟赵延寿为中京（[[恒州_(宣政)|恒州]]）留守、大丞相、[[录尚书事|录尚书事]]、都督中外诸军事，枢密使如故，太宗取笔涂去“录尚书事都督中外诸军事”而执行。以张砺为[[右仆射|右仆射]]兼[[门下侍郎|门下侍郎]]、[[同平章事|同平章事]]。[[左仆射|左仆射]]和凝也被任为兼[[中书侍郎|中书侍郎]]、同平章事。<ref name=HFD98/><ref name=ZZTJ286>[[:s:資治通鑑/卷286|《资治通鉴》卷二百八十六]]</ref><ref>[[:s:舊五代史/卷137|《旧五代史》卷一百三十七]]</ref><ref name=HL4>[[:s:辽史/卷4|《辽史》卷四]]</ref>有司给赵延寿貂蝉冠、张砺三品冠服，二人都不肯穿戴，张砺说：“我在上国时，晋派[[冯道|冯道]]奉册北上来朝，带了二个貂冠，一个给了宰相[[韩延徽|韩延徽]]，一个给了我。现在可以降服吗！”于是着貂蝉冠上朝。<ref name=NHFD72/>

辽兵虐待中原汉人，不久全国叛乱蔓延。太宗疲于应对，决定撤回本土，留[[宣武军节度使|宣武军节度使]]萧翰守开封。<ref name=ZZTJ286/>张砺反对萧翰居此任，未被采纳。<ref name=HL76/>太宗见路过的城池都化为丘墟，就说：“中原如此，都是燕王（赵延寿）的罪过。”又看着张砺说：“你也有责任。”<ref name=NHFD72/><ref name=ZZTJ286/>

太宗没有回到辽本土就在途经恒州附近时病故。<ref name=ZZTJ286/><ref name=HL4/>太宗亡兄[[耶律倍|耶律倍]]子永康王[[耶律阮|耶律阮]]召赵延寿、张砺、和凝、李崧、冯道于所馆饮酒，将意欲自称中原皇帝的赵延寿擒拿，告知张砺等人“燕王谋反，被锁拿，诸公不必担忧。”又至待贤馆受蕃、汉官谒贺，笑着对张砺等说：“燕王如果行此礼，我以铁骑围之，诸公不能免祸。”耶律阮自称受太宗遗命知南朝军国事，遣散张砺等，又宣太宗遗诏称帝，即[[辽世宗|辽世宗]]。考虑到祖母[[皇太后|皇太后]][[述律平|述律平]]将反对自己继位，他夺取辽军主力继续班师，留张砺等汉族官员屯恒州。后来萧翰与[[安国节度使|安国节度使]]中京留守耶律麻荅也从中原撤退到恒州，仍记恨张砺建议以汉治汉，以铁骑围张砺家。张砺正卧病，出来相见，萧翰骂道：“你何故对先帝说胡人不可以为[[节度使|节度使]]？我是宣武节度使，又是国舅，你却在中书省对我下令！先帝留我守汴州，令我处宫中，你又以为不可。又在先帝面前诋毁我和解里（耶律麻荅别名），说解里喜欢掠人财，我喜欢掠人子女。今天我一定要杀你！”下令锁拿张砺。张砺不屈回答：“这都是国家大体，我确实这么说了。想杀就杀，何必要锁！”耶律麻荅认为不可擅杀大臣，不让萧翰杀张砺，萧翰离去。耶律麻荅给张砺解锁。当晚，张砺愤恚而死。<ref name=ZZTJ287/><ref>[[:s:新五代史/卷73|《新五代史》卷七十三]]</ref><ref>[[:s:舊五代史/卷100|《旧五代史》卷一百]]</ref>张砺为人耿直仗义，爱惜人才，急于奖励提拔，赞扬他人善举，以己财救济穷人，中原士大夫闻其过世，都叹惜。家人焚烧了他的骸骨，归葬滏阳。<ref name=HFD98/>
