'''张斌桥'''是[[浙江省|浙江省]][[宁波市|宁波市]]古桥名，原址位于今[[鄞州区|鄞州区]]彩虹北路和中山东路路口，原[[后塘河_(鄞东南)|后塘河]]上，为条石拱桥。始建于[[北宋|北宋]][[元丰_(宋神宗)|元丰]]五年（公元1082年），1914年扩建，1988年因市政建设被拆除。

== 得名传说 ==
张斌桥得名来源于一个传说。据传，阴府有一个判官娶陈姓肉贩的女儿为妻，但由于陈姓肉贩卖肉短斤缺两，仍然使其患病。陈姓肉贩的儿子找到判官请求解救父亲，判官要求肉贩改邪归正，并买下河边张斌老人卖的草鞋并烧掉便可赎罪。肉贩的儿子照做之后，第二天，父亲的疾病痊愈。起初，儿子想要将一半家产赠与张斌，但是，善良的老人坚决不受。考虑到老人一直有造桥连通河两岸的愿望，肉贩的儿子便出资建造了一座桥连接河流两岸，起名为张斌桥<ref>{{cite web|url=http://www.nbdm.gov.cn/html/dimingshengjing/qiao/2016/0324/1350.html|title=张斌桥|publisher=宁波市地名委员会|date=2016-03-24|accessdate=2018-02-27|archive-url=https://web.archive.org/web/20180227001134/http://www.nbdm.gov.cn/html/dimingshengjing/qiao/2016/0324/1350.html|archive-date=2018-02-27|dead-url=yes}}</ref>。

== 桥梁形制 ==
张斌桥为条石拱桥。1914年扩建时，采用鄞县[[东钱湖|东钱湖]]一带的梅园石修筑。桥长30米，高7米，跨度10米。<ref name="ref2">{{cite web|url=http://news.cnnb.com.cn/system/2014/03/09/008006267.shtml|title=回忆老宁波时期的状元房、张斌桥与彩虹坊|publisher=宁波晚报|date=2014-03-09|accessdate=2014-03-30}}</ref>桥面刻有莲花图案，栏杆柱头雕刻有石象、石狮和莲花，象征“祥事连连”<ref name="ref1">{{cite web|url=http://www.nbdm.gov.cn/homepage/view.aspx?id=1449&catid=155&cat_parent=124|title=从江东看宁波的知名桥梁|publisher=四明周刊|date=2008-07-08|accessdate=2014-03-30|deadurl=yes|archiveurl=https://web.archive.org/web/20160304105740/http://www.nbdm.gov.cn/homepage/view.aspx?id=1449&catid=155&cat_parent=124|archivedate=2016-03-04}}</ref>。桥下建有一条纤道，使得旧时纤夫拉纤时不至于到桥上绕行。同时，纤道也常常作为临时码头，曾经接待大批日本僧人前往[[天童寺|天童寺]]或[[阿育王寺|阿育王寺]]<ref name="ref2" />。

== 张斌桥与鄞县史氏家族 ==
[[北宋|北宋]]末年，史氏家族从[[江苏|江苏]][[溧阳|溧阳]]迁入明州，聚居在宁波（时名明州），其中，居住在张斌桥附近的[[史渐|史渐]]在参政知事任上辞职，在明州地方从事教育，形成[[鄞县史氏家族|鄞县史氏家族]]“一门三宰相，四世两封王，五尚书，七十二进士”的名门望族，一时传为佳话，称“张斌桥贤良巷史家”。[[乾隆帝|乾隆帝]]曾赐“贤良方正”匾于张斌桥史家，至今存“贤良巷”地名<ref name="ref2" />。

== 变迁与现状 ==
1914年，张斌桥重建，规模较先前有了较大的扩大。1988年，由于城市改造，张斌桥被拆除，其石料首先被放置于[[东钱湖|东钱湖]]平水堰，后又移至南宋石刻公园的荒山上。2014年，利用原有石料重建的张斌桥在[[鄞州区|鄞州区]][[下应街道|下应街道]]湾底村落成<ref name="ref2" /><ref>{{cite web|url=http://news.cnnb.com.cn/system/2014/03/30/008024160.shtml|title=东边张斌桥 西边鄞江廊桥 宁波两座千年古桥昨重现|publisher=东南商报|date=2014-03-30|accessdate=2014-03-30}}</ref>。

== 影响 ==
张斌桥尽管已被拆除，但是由于其曾经扮演的重要角色，成为宁波一处知名度颇高的地名<ref name="ref1" />。彩虹北路由于张斌桥形似彩虹而得名。鄞州区张斌社区和附近众多商铺和设施也以张斌桥命名<ref>{{cite web|url=http://www.dsjd.gov.cn/list_article.jsp?cid=002001|title=东胜概况|publisher=宁波市江东东胜街道办事处|date=2012-05-23|accessdate=2014-03-30|archiveurl=https://archive.today/20140330020300/http://www.dsjd.gov.cn/list_article.jsp?cid=002001|archivedate=2014年3月30日|deadurl=yes|df=}}</ref>。
