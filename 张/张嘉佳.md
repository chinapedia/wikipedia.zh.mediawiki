'''张嘉佳'''（{{bd|1980年|6月22日|}}），出生于[[江苏省|江苏]][[南通市|南通]]，毕业于[[南京大学|南京大学]]，擔任過雜志主筆，電視編導等。中国当代[[作家|作家]]、[[电影|电影]][[編劇|编剧]]和[[導演|導演]]。2014年，《[[从你的全世界路过|从你的全世界路过]]》出版，在中国大陆销量超过400万册<ref>{{Cite web|url = http://news.xinhuanet.com/book/2014-12/24/c_127329816.htm|title = 张嘉佳成2014作家榜首富|accessdate = 2015-08-02|author = |date = 2014年12月24日|publisher = 新华网|language = zh-cn}}</ref>，同年，荣登第九届[[中國作家富豪榜|中国作家富豪榜]]榜首。因长期在[[新浪微博|新浪微博]]中发表“睡前故事”系列微博，被誉为“最会讲故事的人”<ref>{{Cite web|url = http://www.chinatimes.com/cn/newspapers/20141220001146-260306|title = 张嘉佳大器晚成 睡前故事爆红|accessdate = 2015-08-02|author = |date = 2014年12月20日|publisher = 中时电子报|language = zh-tw}}</ref>。

==生平==
2005年，张嘉佳出版处女作《几乎成了英雄》，首印达10万册<ref>{{Cite web|url = http://news.sina.com.cn/c/2005-11-11/07117412161s.shtml|title = 张嘉佳新作首印10万册|accessdate = 2015-08-02|author = |date = 2005年11月11日|publisher = 新浪网|language = zh-cn}}</ref>。

2011年，张嘉佳首次担任电影编剧，凭借电影《[[刀见笑|刀见笑]]》获得[[第48届金马奖|第48届金马奖]]最佳改编剧本提名<ref>{{Cite web|url = http://ent.163.com/11/1127/09/7JRT380R000300B1.html|title = 《刀见笑》得金马两奖 最佳新导演乌尔善成黑马|accessdate = 2015-08-03|author = |date = 2011-11-27|publisher = 网易娱乐|language = zh-cn}}</ref>。

2013年7月起，张嘉佳在微博上发表33篇睡前小故事，被转发超200万次，超4亿人次阅读。同年11月，这些故事结集成书籍《从你的全世界路过》出版<ref>{{Cite web|url = http://cul.qq.com/a/20150108/052717.htm|title = 专访张嘉佳：爱情太现实只因没有遇到对的人|accessdate = 2015-08-03|author = |date = 2015-01-08|publisher = 腾讯文化|language = zh-cn}}</ref>。

2014年7月-8月，担任中国大陆[[江苏卫视|江苏卫视]]《[[非诚勿扰_(节目)|非诚勿扰]]》的代班嘉宾<ref>{{Cite web|url = http://media.sohu.com/20140716/n402329681.shtml|title = 《非诚勿扰》代班嘉宾张嘉佳表现获赞|accessdate = 2015-08-03|author = |date = 2014-07-16|publisher = 搜狐传媒|language = zh-cn}}</ref>。

2014年，《让我留在你身边》出版。该书通过其宠物狗“梅茜”的视角，讲述狗狗眼中的人世故事<ref>{{Cite web|url = http://news.xinhuanet.com/book/2014-07/30/c_126814379.htm|title = 张嘉佳出新作《让我留在你身边》 以宠物狗为视角|accessdate = 2015-08-03|author = |date = 2014年7月30日|publisher = 新华网|language = zh-cn}}</ref>。

2015年1月，张嘉佳宣布担任电影《[[擺渡人_(2015年電影)|摆渡人]]》的编剧和导演，为其导演处女作<ref>{{Cite web|url = http://news.163.com/15/0112/14/AFP374Q200014AED.html|title = 主角是梁朝伟，导演是张嘉佳|accessdate = 2015-08-03|author = |date = 2015-01-12|publisher = 网易新闻|language = zh-cn}}</ref>。
