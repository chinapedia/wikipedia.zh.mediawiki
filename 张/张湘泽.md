'''张湘泽'''{{BD|1905年||1986年||CatIdx=Z张}}，[[中华民国|中华民国]]大陆时期的[[国军|国军]]将领，[[台湾|台湾]][[基督教|基督教]][[地方教会|地方教会]]的重要人物。生于安徽[[寿县|寿县]]。
== 军旅生涯 ==
张湘泽系日本陆军士官学校中国留学生队第二十七期毕业。历任中央军校南宁分校教官、第13军教导团团长、第5战区战斗干部团教务主任、第4游击纵队副司令、司令、安徽省保安副司令兼保安第2团团长。1949年5月，国民革命军第126军在安徽成立，张湘泽任军长。1949年12月2日，126军在广西博白遭受[[解放军|解放军]]重创，张湘泽率一部分部队退到越南，被越南解除武装，送往[[富国岛|富国岛]]（復岡島），1951年去台湾。詳見：[[富台部队|富台部队]]

== 投身基督教 ==
1944年，张湘泽在大陆时已经相信耶稣，但没有参加聚会<ref> 《历史与启示>第十七篇:台湾众召会的试炼（二）</ref>。1951年到台湾后，[[台北市召会|台北教会]]正经历大复兴，他也开始参加聚会。1953年，张湘泽在台湾参加[[李常受|李常受]]的训练后，成为地方教会里的一位全时间同工。

张湘泽晚年中风，眼睛失明。<!-- 但是他仍然。 -->1986年7月，在参加美国欧文的夏季训练时去世。在安息聚会上，[[李常受|李常受]]在会中痛哭，并且评价张湘泽“的确是个得胜者”，称赞他身上的四个特点：“第一，他是绝对的寻求并跟随主以及祂的恢复；第二，他在主恢复的职事中是有用和忠信的；第三，他不变的、持续不断的与职事是一；第四，他完全不顾自己的性命。”。<ref>帖撒罗尼迦前后书、雅歌七至八章结晶读经，第十二篇  盼望被提</ref>

== 家庭 ==
张湘泽的配偶张王帅信（1915年—2000年）生于南京一个基督徒家庭，18岁进入[[金陵女子大学|金陵女子大学]]，不久进入地方教会，受[[李渊如|李渊如]]帮助，24岁参加倪柝声在上海[[友华村|友华村]]的训练，成为当时最年轻的同工。31岁时与张湘泽在杭州结婚，1951年随其来台，同样投身于基督教会。
