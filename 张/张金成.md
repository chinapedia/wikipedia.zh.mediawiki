'''张金成'''（{{bd|1959年||}}），[[山东|山东]][[安丘|安丘]]人，[[中国人民解放军少将|中国人民解放军专业技术少将]]<ref name=qgrd/>。

==生平==
毕业于[[华中科技大学|华中科技大学]]动力工程与工程热物理专业。加入[[中国共产党|中国共产党]]<ref name=qgrd/>。是[[中国人民解放军第二炮兵指挥学院|中国人民解放军第二炮兵指挥学院]]教授<ref name=yishi>{{Cite web |url = http://www.thepaper.cn/newsDetail_forward_1360146 |title =第二炮兵部队举行晋衔仪式，唐国庆陈平华晋升中将  |author =  |date = 2015-08-04 |publisher = 澎湃新闻 |language =  |accessdate =  |quote =  }}</ref>（2016年改为[[中国人民解放军火箭军指挥学院|中国人民解放军火箭军指挥学院]]教授<ref>{{Cite web |url =http://jz.chinamil.com.cn/zhuanti/content/2016-03/09/content_6949899.htm  |title = 张金成代表：让联合作战指挥人才喷涌而出 |author =  |date =2016-03-09  |publisher = 军报记者 |language =  |accessdate =  |quote =  }}</ref>）。2013年，担任[[第十二届全国人民代表大会|第十二届全国人民代表大会]]解放军代表<ref name=qgrd>{{cite news|title=全国人大代表信息-张金成|url=http://www.npc.gov.cn/delegate/viewDelegate.action?dbid=122869|accessdate=2013-04-05|newspaper=中国人大网}}</ref>。

2015年7月6日，[[第二炮兵|第二炮兵]]宣布中央军委命令大会暨晋升将官军衔仪式在[[北京|北京]]举行，张金成由专业技术大校军衔晋升为专业技术少将军衔<ref name=yishi/><ref>{{Cite web |url = http://www.chinanews.com/mil/2015/08-03/7444056.shtml |title = 二炮2人升中将5人升少将 女导弹专家成新晋少将 |author =  |date =2015-08-03  |publisher = 中国新闻网 |language =  |accessdate =  |quote =  }}</ref>。2018年1月，当选[[中国人民政治协商会议第十三届全国委员会委员名单|第十三届全国政协委员]]<ref>{{Cite web|url=http://sports.163.com/18/0126/06/D92BRKK400051C89.html|title=中国人民政治协商会议第十三届全国委员会委员名单|accessdate=2018-01-26|last=|work=网易}}</ref>。
