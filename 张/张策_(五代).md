'''张策'''（{{bd|9世纪||912年|catIdx=Z}}），[[表字|字]]'''少逸'''，[[中国|中国]][[唐朝|唐朝]]及[[五代十国|五代十国]]时期[[后梁|后梁]][[官员|官员]]，在后梁开国皇帝[[梁太祖|梁太祖]][[朱全忠|朱全忠]]年间为宰相。

== 家世 ==
张策生年不详，祖籍[[敦煌|敦煌]]。父[[张同|张同]]，唐朝官员，官至[[容管经略使|容管经略使]]。<ref name=HFD18>[[:s:舊五代史/卷18|《旧五代史》卷一十八]]</ref><ref name=NHFD35>[[:s:新五代史/卷35|《新五代史》卷三十五]]</ref>

张策年轻时聪警好学，尤其喜爱章句。当时他家住[[洛阳|洛阳]]敦化里，并且从甘泉井挖出古鼎，鼎耳有篆字：“[[曹魏|魏]][[黄初|黄初]]元年春二月，匠吉千”，且制作奇巧，张同很珍重它。但时年才一十三岁的张策却从容地对父亲说：“[[建安_(东汉)|建安]]二十五年，[[曹操|曹公]]去世，改年号为[[延康_(东汉)|延康]]，同年十月，[[魏文帝|魏文帝]]接受[[汉朝|汉帝]][[禅让|禅让]]，才[[改元|改元]]为黄初，因此黄初元年显然是没有二月的。鼎文是多么地荒谬啊！”张同大惊，去书房取《[[三国志|三国志]]·魏志》来读，证实了张策的说法。<ref>又见[[:s:資治通鑑/卷069|《资治通鉴》卷六十九]]。</ref>宗族都为张策的学识称奇。但张策没有进学及准备科考做官，而是信奉[[佛教|佛教]]，未及[[弱冠|弱冠]]就在京城[[长安|长安]]的[[大慈恩寺|慈恩寺]]落发为僧。<ref name=HFD18/><ref name=NHFD35/>

[[唐僖宗|唐僖宗]][[广明_(唐朝)|广明]]末年（880年—881年），长安地区被大规模农民军[[黄巢|黄巢]]攻打，张策还俗奉父母逃难，为君子称道。不久张同去世，张策服丧，以孝闻名。丧期满后，他也不求出仕，在乡间隐居十余年。<ref name=HFD18/><ref name=NHFD35/>

== 入仕 ==
=== 唐朝年间 ===
张策结束隐居后出为广文馆[[博士|博士]]，改[[秘书郎|秘书郎]]。[[军阀|军阀]][[王行瑜|王行瑜]]为[[静难节度使|静难节度使]]，召张策为观察支使，带水曹员外郎，赐穿绯色官服。王行瑜与朝廷冲突，被[[河东节度使|河东节度使]][[李克用|李克用]]攻败，张策与婢女以肩舆载母从静难军部[[邠州_(唐朝)|邠州]]南境冒积雪出逃。<ref>李克用败王行瑜于[[乾宁|乾宁]]二年（895年）。见[[:s:資治通鑑/卷260|《资治通鉴》卷二百六十]]。</ref>路人都同情他。李克用大敌[[宣武军节度使|宣武军节度使]]朱全忠当时兼领四镇，闻讯，奏张策为[[宣义军|宣义军]]支使，不久张策又因母丧去职。<ref name=HFD18/><ref name=NHFD35/>

丧期满后，张策被除国子博士，迁膳部员外郎。不到一年，军阀[[镇国军节度使|镇国军节度使]][[韩建|韩建]]召他为判官，韩建为[[忠武军节度使|忠武军节度使]]后又以他为掌记。<ref>朱全忠迫韩建弃镇国军并调他于忠武军在天复元年（901年），为张策效力他划定了时间范围。见[[:s:資治通鑑/卷262|《资治通鉴》卷二百六十二]]。</ref>[[唐昭宗|唐昭宗]][[天复_(唐朝)|天复]]（901年—904年）年间，张策奉韩建书信和礼物拜访朱全忠，朱全忠见了他，高兴地说：“张夫子到了。”奏为掌记，兼赐金紫。[[天祐_(唐朝)|天祐]]（904年—907年）初年，朱全忠表荐张策之才，拜为职方郎中，二年（905年）五月兼充史馆修撰，不久召入为[[翰林学士|翰林学士]]，<ref>《旧唐书·哀帝纪》作以翰林学士兼充史馆修撰。</ref>转兵部郎中，知制诰，仍修史。不久，迁中书舍人，<ref name=NHFD35/>但职务如前。<ref name=HFD18/>四年（907年），朱全忠迫[[唐哀帝|唐哀帝]][[禅位|禅位]]，代唐建后梁为太祖皇帝。宰相摄[[侍中|侍中]][[杨涉|杨涉]]为押传国宝使，张策为副使，<ref name=BT20-2>[[:s:旧唐书/卷20下|《旧唐书》卷二十下]]</ref>将[[传国玉玺|传国玉玺]]送到[[宣武军|宣武军]]军部[[大梁|大梁]]以行[[受禅|受禅]]仪式，并奉传国玉玺升殿。<ref name=HFD3>[[:s:旧五代史/卷3|《旧五代史》卷三]]</ref><ref name=ZZTJ266>[[:s:資治通鑑/卷266|《资治通鉴》卷二百六十六]]</ref>

=== 后梁年间 ===
太祖登基后，十月，将文武百官遣回岗位，但将张策等留在身边。<ref name=HFD3/>改张策为工部侍郎，加承旨。同年冬，转礼部侍郎。次年，张策随太祖讨伐李克用之子和继承人[[李存勖|李存勖]]治下的[[晋_(五代)|晋国]]，随驾到[[泽州|泽州]]。四月，被任为刑部侍郎，与新拜[[中书侍郎|中书侍郎]][[于兢|于兢]]一同加[[同中书门下平章事|同中书门下平章事]]，为宰相，<ref name=HFD4>[[:s:旧五代史/卷4|《旧五代史》卷四]]</ref>以代杨涉，<ref name=ZZTJ266/><ref name=NHFD2>[[:s:新五代史/卷2|《旧五代史》卷二]]</ref>仍判户部，不久迁中书侍郎。<ref name=HFD18/><ref name=NHFD35/>

== 致仕及过世 ==
[[开平_(后梁)|开平]]二年（908年）十一月，张策因染风恙，上表请辞。<ref name=NHFD2/><ref>[[:s:資治通鑑/卷267|《资治通鉴》卷二百六十七]]；时任[[左仆射|左仆射]]杨涉取代了他，重归相位。</ref>以刑部尚书[[致仕|致仕]]。<ref name=HFD4/>即日乘肩舆归洛阳，住在福善里，以园艺、读书、听琴、饮酒自娱。四年（910年）四月，太祖追赠其先人官一等。<ref name=HFD5>[[:s:旧五代史/卷5|《旧五代史》卷五]]</ref><ref>[[:s:追赠张策等官诏|:s:追赠张策等官诏]]</ref>[[乾化|乾化]]二年（912年）秋卒于洛阳。<ref name=NHFD35/>著《典议》三卷、词制歌诗二十卷、笺表三十卷，存放家中。<ref name=HFD18/>

== 轶闻 ==
《[[唐摭言|唐摭言]]》及《[[北梦琐言|北梦琐言]]》载：广明年间，少师[[赵崇|赵崇]]提倡文学，张策刚还俗，言说时事变化，希望入贡士之列，但赵崇鄙视他，虽然有人提出[[刘轲|刘轲]]、[[蔡京|蔡京]]也曾为僧，但赵崇却说：“刘、蔡辈虽然曾为僧，但不为人所知，转而科举有何不可？张策是衣冠子弟，无故出家，不能参禅访道，抗迹尘外，却在御帘前进诗希求恩泽。如此行止，岂能掩人口实。我十次掌管科举，就斥退他十次。”于是当庭谴责张策；张策只得改习博学宏辞，而赵崇在吏部任职，又罢黜他，并宣扬他的过失。天祐年间，张策成为朱全忠谋士后，进言诬陷，致使赵崇遇害。
