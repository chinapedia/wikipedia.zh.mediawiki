'''张佩芬'''{{BD|1933年||||catIdx=Z}}[[上海人|上海人]]，籍贯[[浙江省|浙江省]][[上虞县|上虞县]]，[[复旦大学|复旦大学]]西语系德语专业，[[中国作家协会|中国作家协会]]成员，中国大陆翻译家。<ref>{{cite web |url= http://beijingww.qianlong.com/1470/2013/07/01/553@194968.htm|title=李文俊、张佩芬：书不能看太多了，就看看古董 |accessdate=2013 |publisher=千龙网 |work= |language=zh-hans}}</ref><ref>{{cite web |url= http://book.ifeng.com/yeneizixun/special/fanyijia/detail_2012_08/09/16681104_0.shtml|title= 张佩芬对与杨绛先生交往过程的深情追忆|accessdate= 2012|publisher=凤凰网 |work= |language=zh-hans}}</ref>

==生平==
1933年，张佩芬出生在[[上海市|上海市]]，籍贯是[[浙江省|浙江省]][[上虞县|上虞县]]。

1953年，张佩芬毕业于[[复旦大学|复旦大学]]西语系德语专业，后来在《译文》、《世界文学》担任编辑，也曾在[[中国社会科学院|中国社会科学院]]担任研究员。

1984年，张佩芬加入[[中国作家协会|中国作家协会]]。

==家庭==
张佩芬的丈夫是[[李文俊_(翻译家)|李文俊]]，中国大陆著名翻译家。<ref>{{cite web |url= http://culture.people.com.cn/n/2013/1028/c172318-23344831.html|title= 门罗获奖并未让译者“发达”：当时只拿了18000元|accessdate=2013 |publisher= 人民网|work= |language=zh-hans}}</ref>
