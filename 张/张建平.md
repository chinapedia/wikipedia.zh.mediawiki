'''张建平'''（{{bd|1956年|4月|||catIdx=Z}}），[[福建|福建]][[沙县|沙县]]人，[[中国人民解放军中将|中国人民解放军空军中将]]，[[中国共产党第十八届中央委员会|中国共产党第十八届中央委员会]]候补委员。

==生平==
1974年入伍。27岁时任空三师团长，成为中国人民解放军空军最年轻的团长。1988年率部进驻[[西藏|西藏]]，完成了高原条件下的科目训练。历任空三师副师长、师长，36岁便成为中国人民解放军空军最年轻的航空兵师长。2009年任“和平使命——2009”中俄联合反恐军演中方副总导演。历任[[南京军区空军|南京军区空军]]副参谋长，空九军参谋长、军长，[[沈阳军区空军|沈阳军区空军]]参谋长，[[北京军区空军|北京军区空军]]副司令员，空军副参谋长<ref>{{Cite web |url =  http://news.shm.com.cn/2011-02/01/content_3365092.htm|title = 解放军三大军区空军岁末换帅 凸显大交流理念 |author =  |date =2011-02-01  |publisher = 水母网 |language =  |accessdate =  |quote =  }}</ref>。2011年1月任[[济南军区空军|济南军区空军]]司令员，同年7月任[[广州军区空军|广州军区空军]]司令员，2013年4月任空军副司令员。2017年12月，调[[军委战略规划办公室|军委战略规划办公室]]任职。

2003年晋升[[中国人民解放军少将|空军少将]]军衔，2012年晋升[[中国人民解放军中将|空军中将]]军衔<ref>{{cite news|title=中国解放军和武警部队今年共晋升22中将|url=http://www.usqiaobao.com/2012-08/20/content_1532950.htm|accessdate=2012-11-05|newspaper=香港大公报|deadurl=yes|archiveurl=https://archive.is/20121204175656/http://www.usqiaobao.com/2012-08/20/content_1532950.htm|archivedate=2012-12-04}}</ref>。
