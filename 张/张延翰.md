'''张延翰'''（{{bd|884年||941年|1月5日|catIdx=Z}}<ref name=SGCQ21>[[:s:十國春秋/卷021|《十国春秋》卷二十一]]</ref><ref name=ZZTJ282>[[:s:資治通鑑/卷282|《资治通鉴》卷二百八十二]]</ref>），[[表字|字]]'''德华'''，[[中国|中国]][[五代十国|五代十国]]时期[[政权|政权]][[吴_(十国)|吴国]]和[[南唐|南唐]]官员，在吴国末年和南唐初年为[[宰相|宰相]]。

== 家世 ==
张延翰生于[[唐僖宗|唐僖宗]][[中和_(唐朝)|中和]]四年（884年），[[睢阳|睢阳]]人，<ref>[[马令|马令]]《[[南唐书_(马令)|南唐书]]》作[[砀山|砀山]]人。</ref>几无家世记载。年轻时游历[[唐朝|唐朝]]都城[[长安|长安]]，唐朝末年任[[陕州|陕州]]司马，一次去看从父权[[感化军|感化军]][[留后|留后]][[张慎思|张慎思]]，告以北方将乱，说想要避居江、淮即大[[军阀|军阀]][[淮南节度使|淮南节度使]][[杨行密|杨行密]]治下，以延续家族世系。张慎思认可他的判断，慨然送他上路。<ref name=SGCQ21/><ref name=LBST6>[[:s:陸氏南唐書/卷06|陆游《南唐书》卷六]]</ref><ref name=MBST10>马令《南唐书》卷一十</ref>

== 吴国年间 ==
[[天祐_(唐朝)|天祐]]四年（907年），唐亡，分裂成几个政权，杨行密辖区当时由其子和继承人[[杨渥|杨渥]]统治，为弘农国（后为吴国）。<ref>[[:s:資治通鑑/卷266|《资治通鉴》卷二百六十六]]</ref>张延翰初任[[盐城|盐城]]令，有治绩。<ref name=LBST6/>迁[[楚州|楚州]]行军司马。<ref name=SGCQ21/><ref name=MBST10/>

十五年（918年），吴国[[摄政|摄政]][[徐温|徐温]]屯[[昇州|昇州]]，由养子[[徐知诰|徐知诰]]在吴国都城[[广陵|广陵]]为少摄政，封徐知诰[[浔阳|浔阳]]公，<ref>陆游《南唐书·张延翰传》作浔阳侯。</ref>授[[江州|江州]][[团练使|团练使]]，<ref>[[:s:資治通鑑/卷270|《资治通鉴》卷二百七十]]</ref>徐知诰显然并不从广陵前去就职，而是表张延翰为江州观察巡官通判军府事、工部郎中，前去江州代为理政。<ref name=SGCQ21/><ref name=LBST6/><ref name=MBST10/>

后来，<ref>《[[十国春秋|十国春秋]]》引[[南宋|南宋]]史学家[[陆游|陆游]]《[[南唐书_(陆游)|南唐书]]》称此事发生在徐知诰于升元元年（937年）登基时，但因与《十国春秋》及其他更多确定的史料对张延翰其余仕途的记载完全不连贯而不取。见[[:s:陸氏南唐書/卷06|陆游《南唐书》卷六]]。此事当在[[顺义_(南吴)|顺义]]七年（927年）徐温过世或当时吴国统治者杨溥称帝、徐知诰的摄政权力更大时。见[[:s:資治通鑑/卷276|《资治通鉴》卷二百七十六]]。</ref>张延翰回到吴国朝廷，为[[殿中侍御史|殿中侍御史]]，知杂事，判台事。左街使张宣恃功骄暴，张延翰当廷弹劾之，其他悍将因而收敛。<ref name=SGCQ21/><ref name=LBST6/><ref name=MBST10/>

张延翰后进礼部侍郎，因之前从未成为摄政的近臣，很感激当时已是全权摄政的徐知诰。当时，吴国还没有沿用唐朝为朝廷选官的[[科举|科举]]制度，而是对献书论事者评定优劣选官。徐知诰以张延翰兼知选事，张延翰表现称职，公平详审，注重提拔孤贫之人，不附权势，官吏敬畏他如神明，不敢作奸犯科。[[大和_(吴)|大和]]二年（930年），徐知诰子[[李璟|徐景通]]为少摄政，<ref>[[:s:資治通鑑/卷277|《资治通鉴》卷二百七十七]]</ref>对人们说：“张君议论公正，处事有条理，经手的簿册和文书都很明晰，我要倾心听之。”张延翰奉命综领六司，为时人所重。五十岁时，拜[[中书侍郎|中书侍郎]]、[[同中书门下平章事|同中书门下平章事]]，为宰相。当时作为宰相他并不算老，但人们仍然认为他拜相晚了。<ref name=SGCQ21/><ref name=LBST6/><ref name=MBST10/>

== 南唐年间 ==
[[天祚|天祚]]三年（937年）十月，徐知诰迫吴帝[[杨溥_(吴)|杨溥]][[禅位|禅位]]，代吴称帝，建立南唐。以张延翰及[[门下侍郎|门下侍郎]][[张居咏|张居咏]]、中书侍郎[[李建勋|李建勋]]三位吴国宰相为同平章事，仍为南唐宰相。<ref>[[:s:新五代史/卷62|《新五代史》卷六十二]]</ref>次年，徐知诰改李姓，改名[[李昪|李昪]]。<ref name=ZZTJ281>[[:s:資治通鑑/卷281|《资治通鉴》卷二百八十一]]</ref>

不久，张延翰病重不能理事。李昪仍希望他理事，不许其辞职，派使者问疾赐良药不绝。[[升元|升元]]四年（940年），张延翰在道路上卒于右仆射<ref>[[马令|马令]]《[[南唐书_(马令)|南唐书]]》作左仆射。</ref>兼门下侍郎同平章事任上。<ref name=ZZTJ282/><ref>[[:s:十國春秋/卷015|《十国春秋》卷一十五]]</ref>士大夫叹惜之。赠[[太师|太师]]。<ref name=SGCQ21/><ref name=LBST6/><ref name=MBST10/><ref>马令、陆游《南唐书》皆作[[太傅|太傅]]。</ref>

== 评价 ==
*《十国春秋》论曰：（张）延翰综理六司，卓然奏绩，晚登揆席，未竟厥用，惜哉！
