'''张晓梅'''（{{bd|1963年|12月|catIdx=Z张}}），女，生于[[北京|北京]]<ref name="b1"/>，[[第十届全国政协委员|第十]]<ref name="a1">{{Cite web |url=http://www.cppcc.gov.cn/2011/12/16/ARTI1513309118001390.shtml |title=张晓梅：穷人也有美丽的权利|accessdate=2018-12-18 |author=|date=2007-03-06 |work= |publisher=中国人民政治协商会议全国委员会网站，来源：人民网|language=Zh-hans }}</ref>、[[第十一届全国政协委员|十一届全国政协委员]]、[[全国工商联|全国工商联]]执委，高级经济师，[[全国三八红旗手|全国三八红旗手]]<ref name="a2">{{Cite web |url=http://www.ce.cn/cysc/ztpd/09/zxm/ |title=中经在线访谈>>嘉宾风采|accessdate=2018-12-18 |author= |date=2010-03-15 |work= |publisher=|language=Zh-hans }}</ref><ref name="a3">{{Cite web |url=http://www.cnr.cn/fortune/guest/200411/t20041111_504044608.html|title=张晓梅：魅力来源于自信|accessdate=2018-12-18 |author= |date=2009-09-17 |work= |publisher=中广网 |language=Zh-hans }}</ref>。

她因在全国政协所提多个[[提案|提案]]，在网络引起广泛争议而出名，张晓梅被网友称为“最[[雷人|雷人]]提案手”<ref name="b2">{{Cite web |url=http://www.zhiyin.cn/tegao/2010/0322/article_1513_2.html |title=“话题雷女”张晓梅的“红与黑”（组图） (2)|accessdate=2018-12-18 |author=|date=2010-03-22|work= |publisher=知音网，来源：南方都市报|language=Zh-hans }}</ref>。另一方面，她在2010年的政协提案因涉及“承认[[家務|家务]]劳动的价值”，而被评论者认为是当年[[两会|两会]]最有价值的提案之一<ref name="a4">{{Cite web |url=http://news.163.com/10/0315/09/61QC1PRK00012Q9L.html |title=长平：家庭主妇劳动的价值真需要重估|accessdate=2018-12-18 |author=长平 |date=2010-03-15 |work=责任编辑：王晓易_NE0011 |publisher=网易，来源：东方早报(上海)|language=Zh-hans }}</ref>。同时，在委员人均提案不足两个的情况下，她每年的提案数几乎在两位数，至2010年时，已过百<ref name="b2"/>。

==经历==
*出生于北京<ref name="b1">{{Cite web |url=http://www.zhiyin.cn/tegao/2010/0322/article_1513_8.html |title=“话题雷女”张晓梅的“红与黑”（组图） (8)|accessdate=2018-12-18 |author=|date=2010-03-22 |work= |publisher=知音网，来源：南方都市报|language=Zh-hans }}</ref>的军人家庭，在四川山区部队的环境中长大<ref name="b3">{{Cite web |url=http://www.zhiyin.cn/tegao/2010/0322/article_1513_6.html |title=“话题雷女”张晓梅的“红与黑”（组图） (6)|accessdate=2018-12-18 |author=|date=2010-03-22 |work= |publisher=知音网，来源：南方都市报|language=Zh-hans }}</ref>。
* 1977年，不到15岁的她入伍。至副营级。<ref> [http://blog.sina.com.cn/s/blog_48f2ee82010003x7.html 中国式美女掌门人张晓梅前身暴光]</ref>
* 转业后，先成都一家杂志社做记者。
* 1989年，开始创业，开小美容店。
* 1990年，创办了西南美容学校
* 1999年，正式对全国发行《中国美容时尚报》。
* 2003年，《美容时装报》报社由成都迁至北京，正式改名《中国美容时尚报》<ref name="b3"/>。
* 2003年<ref name="a1"/>，当选为[[第十届全国政协委员|第十届全国政协委员]]。
*[[四川大学|四川大学]]研究生毕业<ref name="a3"/>，或说四川大学[[MBA|MBA]]<ref name="b1"/>，曾在美国[[加州大学|加州大学]]进修<ref name="a2"/><ref name="a3"/>。

==全国政协提案举例==

*2008年：建议美妆行业提倡小费。女性经期应有法定休假。
*2009年：将‘[[三八妇女节|三八妇女节]]’更名为‘三八女人节’。她认为“妇女”源自“[[家庭妇女|家庭妇女]]”，暗示年龄偏大和没有文化的人，不受欢迎。从人本主义出发改名，体现人文关怀<ref name="a1"/>。
*2010年：“家务劳动工资化，切实保障女性权益”的提案<ref>{{Cite web |url=http://www.chinanews.com/cul/news/2010/03-03/2148053.shtml |title=政协委员张晓梅回应“女性家务劳动工资化”质疑|accessdate=2018-12-18 |author=记者：姜妍、天蓝，实习生：徐娴 |date=2010-03-03 |work=编辑：张中江 |publisher=中国新闻网，来源：新京报|language=Zh-hans }}</ref><ref>{{Cite web |url=http://www.chinanews.com.cn/gn/news/2010/03-03/2149482.shtml |title=张晓梅称没人愿雷人 提案提交时可能会改词|accessdate=2018-12-18 |author=孙颖、阎彤(摄) |date=2010-03-03 |work=编辑：李季 |publisher=中国新闻网，来源：北京晚报|language=Zh-hans }}</ref>。有报道称之为“老婆做家务，老公须发工资”。张晓梅希望通过这种方式，承认和体现女性，承认[[家務|家务]]劳动的价值，特别是那些[[全职太太|全职太太]]在家庭中付出的劳动<ref name="a5">{{Cite web |url=http://news.enorth.com.cn/system/2010/03/02/004519758.shtml |title=新闻热评：老婆做家务，老公须发工资？|accessdate=2018-12-18 |author=周明华|date=2010-03-02 |work=编辑：覃贻花 |publisher=北方网，稿源：每日新报，《华西都市报》|language=Zh-hans }}</ref>。虽然提案因“承认家务劳动的价值”的论题获得了正面的评价，称为2010年[[两会|两会]]最有价值的提案之一，但舆论更多的是嘲讽和挖苦<ref name="a4"/>，亦有评论称其为“无聊也无用的小儿科提案，最后别再出现在公众眼前，因为这是对参政议政的稀缺公共资源的一种浪费<ref name="a5"/>”。
