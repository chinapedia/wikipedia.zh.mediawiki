'''张皓宸'''（{{bd|1990年|4月15日||}}），出生于[[四川省|四川省]][[成都市|成都市]]，毕业于[[四川大学锦城学院|四川大学锦城学院]]。[[中国|中国]]当代[[作家|作家]]、[[插画师|插画师]]、[[编剧|编剧]]。2010年，出版个人小说集《回忆是眼睛里的海》。2014年出版短篇小说合集《你是最好的自己》。2015年4月，出版小说集《我与世界只差一个你》，2015年11月，出版短篇小说合辑《谢谢自己够勇敢》<ref>{{cite web |title=张皓宸拒做鲜肉 《谢谢自己够勇敢》电子书当当独家首发 |url=http://news.sina.com.cn/o/2015-11-09/doc-ifxknivr4380786.shtml |website=齐鲁晚报}}</ref>。

==人生经历==
*2010年出版个人第一本小说《回忆是眼睛的海》。
*2013年与电台主持人杨杨共同创作主题为“那些你很冒险的梦”的卫生纸漫画而受到人们的关注。
*2014年，出版《你是最好的自己》，该书获得当当网新书榜、京东全网青春文学类第一名，销量超过100万册。4月，与[[林俊杰|林俊杰]]共同参加综艺节目《[[我爱记歌词|我爱记歌词]]》。同年12月，凭借短篇小说合集《你是最好的自己》获得第六届报喜鸟新锐艺术人物盛典文学领域新锐奖。
*2015年，出版短篇故事合集《我与和世界只差一个你》，4月24日，参加[[湖南卫视|湖南卫视]]综艺节目《[[天天向上|天天向上]]》。同年与[[杨杨|杨杨]]举办“你是最好的自己”插画展11月出版书籍《谢谢自己够勇敢》。
*2016年，荣登中国90后作家排行榜榜首<ref>{{cite web |title=“90后”百万畅销书作家张皓宸新书《后来时间都与你有关》首发 |url=http://news.sina.com.cn/o/2017-07-28/doc-ifyinwmp0555904.shtml |website=中国新闻网}}</ref>。3月，宣布电影《我与世界，只差一个你》立项，并且担任编剧。随后参加[[中央电视台|中央电视台]][[艺术人生|《艺术人生》]]暑期档特别节目《人生课堂》，与百岁老人[[马识途|马识途]]对话<ref>{{cite web |title=张皓宸拜访马识途 跨越76年对话 |url=http://news.sina.com.cn/o/2016-08-04/doc-ifxutfyw0538376.shtml |website=成都商报}}</ref>。2016年年末，与[[白举纲|白举纲]]，[[吴镇宇|吴镇宇]]，[[张晓龙|张晓龙]]共同参加的综艺节目《[[闪亮的爸爸|闪亮的爸爸]]》第二季在[[泸西县|泸西县]]正式开拍。
