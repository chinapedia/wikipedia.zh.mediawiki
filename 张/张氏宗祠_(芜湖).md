'''张氏宗祠'''或称'''南陵丫山张氏宗祠'''<ref name="a1"/>，位于[[安徽省|安徽省]][[芜湖市|芜湖市]][[南陵县|南陵县]][[何湾镇|何湾镇]]龙山村<ref name="a2"/>，地处南陵县与[[铜陵市|铜陵市]]交界的低山丛岭中<ref name="a1"/>。始建于明代，重建于清代中晚期<ref name="a2"/>。视为南陵县境内“硕果仅存”的[[宗祠|宗祠]]<ref name="a3"/>。

因丫山张氏是外来家族，为避免与当地大族孙氏冲突，故将宗祠选址于山区。[[宁国府|宁国府]]赐匾额“本立堂”以示表扬<ref name="a1">{{Cite web |url=http://mt.sohu.com/20160425/n446022124.shtml |title=让他三尺又何妨 |accessdate=2017-03-10 |author=王陵华 |date= |work= |publisher=搜狐网，来源：安徽商报|language=Zh-hans }}</ref>。[[抗日战争|抗日战争]]时期曾做为临时军事指挥部。1949年后作为住宅居住<ref name="a2">{{Cite web |url=http://www.nlx.gov.cn/content/38_125612.html |title=省级文物保护单位：南陵县张氏宗祠装上“电子眼” |accessdate=2017-03-10 |author=南陵县文物管理所：王东明 |date=2016-09-20 |work= |publisher=中国·南陵——南陵县人民政府网站，来源：文广新局|language=Zh-hans }}</ref>。[[文化大革命|文化大革命]]期间，丫山张氏族人集资两千元，从地方政府手中购回张氏宗祠，使之得以留存<ref name="a3">{{Cite web |url=http://www.ahnlbwzx.com/display.asp?id=751 |title=绳其祖武 宗风不灭——南陵世族的家风 |accessdate=2017-03-10 |author=魏青平 |date=2016-05-19 |work= |publisher=来源：芜湖日报·南陵周刊 |language=Zh-hans }}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。至2000年代，宗祠已年久失修，几近废弃。2008年，在[[第三次全国文物普查|第三次全国文物普查]]中被发现。次年，列入第三次全国文物普查重要发现。后公布为南陵县文物保护单位。2012年12月，列入第七批[[安徽省文物保护单位|安徽省文物保护单位]]。2014年，南陵县人民政府对张氏宗祠进行修缮，次年完工<ref name="a1"/>。
