'''张斌'''（{{bd|1969年|1月3日}}），[[北京|北京]]人，[[中国|中国]]的[[体育|体育]]电视节目著名[[主持人|主持人]]，1991年毕业于[[中国人民大学|中国人民大学]]新闻系本科，取得学士学位，张斌因主持节目《[[足球之夜|足球之夜]]》而成名，现为[[中央电视台|中央电视台]]体育部副主任。

== 个人生活 ==
张斌的妻子叫[[胡紫薇|胡紫薇]]，曾为[[北京电视台|北京电视台]]节目主持人，现为[[湖北卫视|湖北卫视]]财经节目《微观视界》主持人。<ref name="lion " >{{cite web |url=http://roll.sohu.com/20120308/n337080196.shtml |title=央视主持人妻子大揭秘 |publisher=搜狐网 |accessdate=2012-03-08 |work= |language =zh-cn}}</ref><ref name="shark " >{{cite web |url=http://ent.163.com/special/00032H0C/cctv1229.html |title=央视主持人的另一半 |publisher=网易 |accessdate=2010年 |work= |language =zh-cn}}</ref><ref name="duck " >{{cite web |url=http://photo.gmw.cn/2012-03/09/content_3733396.htm |title=央视主持人妻子大揭秘 |publisher=光明网 |accessdate= 2012-03-09|work= |language =zh-cn}}</ref>

== 外遇事件 ==
2007年12月28日下午3时，在[[中央电视台|中央电视台]]体育频道改名为奥运频道新闻发布会上，其妻子[[胡紫薇|胡紫薇]]走到台上哭诉其夫与另一女子发生[[婚外情|婚外情]]。当天晚上，[[网易|网易]]等[[BBS|BBS]]出现“胡紫薇大闹央视发布会，踢爆体育节目主持人张斌婚外情”的帖子，一些[[博客|博客]]也纷纷评论；但次日上午，一些大型网站删除该帖子，提及此事的博客文章也纷纷被撤<ref>{{Cite web |url=http://www.ycwb.com/xkb/2007-12/30/content_1738532.htm |title=央视名嘴妻子大闹CCTV　张斌昨已向央视辞职 |access-date=2007-12-30 |archive-url=https://web.archive.org/web/20080101224455/http://www.ycwb.com/xkb/2007-12/30/content_1738532.htm |archive-date=2008-01-01 |dead-url=yes }}</ref>。但是在[[YouTube|YouTube]]上已经保留了当时的视频，并且在国际范围内造成影响，一些国外主流媒体都有报道并评论（如[[NHK|NHK]]）。

目前对于第三者的身份的猜想多与该台几位女记者和主持有关，但具体之人尚未有任何切实的证据可以证明。

中央电视台天气预报主持人[[裴新华|裴新华]]就《[[羊城晚报|羊城晚报]]》报道其为张斌首任妻子提出诉讼，认为《羊城晚报》侵犯其名誉权，要求《羊城晚报》公开道歉并赔偿精神损害抚慰金100万元人民币<ref>[http://ent.people.com.cn/GB/42075/81374/6779148.html 裴新华称媒体误报自己为张斌前妻 索赔百万] [[新华网|新华网]]</ref>。但是据查证，最初公开表示裴新华是张斌第一任妻子的媒体是'''新华网'''<ref>[http://www.ln.xinhuanet.com/xwzx/2007-12/29/content_12080962.htm 张斌的前妻《天气预报》主持人裴新华(图文)] {{webarchive|url=https://web.archive.org/web/20080112112257/http://www.ln.xinhuanet.com/xwzx/2007-12/29/content_12080962.htm |date=2008-01-12 }} [[新华网|新华网]]</ref>。

在被央视雪藏半年后，张斌于欧洲杯专题节目《足球欧洲》复出。
