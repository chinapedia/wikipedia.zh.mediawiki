'''张俭'''（{{bd|594年||653年|catIdx=Z}}），字'''师约'''，[[雍州|雍州]][[新豐縣_(漢朝)|新丰]]人，祖父皖城郡公[[张威_(隋朝)|张威]]曾當過[[隋朝|隋朝]][[相州|相州]][[刺史|刺史]]。父親张植曾經當過[[车骑将军|车骑将军]]、连城县公。<ref name=BT83>[[:s:旧唐书/卷083|《旧唐书》卷八十三]]</ref><ref name=NBT111>[[:s:新唐书/卷111|《新唐书》卷一百一十一]]</ref>

== 邊境事蹟 ==
张俭是[[唐高祖|唐高祖]][[李渊|李渊]]的从甥（据《旧唐书》，《新唐书》作从外孙）。跟随李渊一起打天下，立下戰功，被除授右卫郎将，[[贞观_(唐朝)|贞观]]初年時，也因军功當了[[朔州|朔州]]刺史。當时[[突厥|突厥]][[颉利可汗|颉利可汗]]覺得自己很強大，每當有要求時，都將自己的[[書信|書信]]稱為[[诏敕|诏敕]]，边境許多官吏為了奉承他，都不敢拒绝。张俭到任時，便拒絕颉利可汗的要求，[[唐太宗|唐太宗]]知道後便嘉奖他。张俭在其任內广营屯田，每年得到穀物十万斛，使得边粮非常丰富。遇到寒害或是旱災时，他便劝百姓互相救助，許多人于是免于饥饿，當時邊疆地帶只有他治下的州境平安无事。后來张俭又當了检校[[胜州|胜州]]都督，但是因為母亲去世需要守丧而離開職位。<ref name=BT83/><ref name=NBT111/><ref name=ZZTJ193>[[:s:资治通鉴/卷193|《资治通鉴》卷一百九十三]]</ref>

张俭之前在朔州，[[李靖|李靖]]平突厥之后，一個貧窮名叫思结的部落归顺张俭，张俭招慰安置他们。还有一些没有来归顺的部落成员在碛北，因為與思结有亲属，時常與他們互相往来，张俭不禁止此事，只维持纲纪，[[羁縻|羁縻]]管治。张俭離開此職位後，贞观四年（630年）九月，新到任的官員不知此事，告訴朝廷思结想要叛乱。朝廷商议想要出兵討伐，於是任用還在守喪的的张俭为使者，观察那些部落的动静，並进行抚纳。张俭一個人進入部落，召诸首领慰谕，推心置腹，他们都匍匐叩头而至，迁徙到[[代州|代州]]。朝廷於是立即命令张俭當检校代州都督。思结部终于没有叛乱者，张俭于是劝他们經营田產，使得每年都丰收。张俭又認為有錢會帶來奢侈的風氣與想反叛的可能，於是請求朝廷和籴（古时官府以议价交易为名向民间强制征购粮食），充实官府储备，蕃人喜悦，营田愈发出力。边军也大收其利。<ref name=BT83/><ref name=NBT111/><ref name=ZZTJ193/>

== 戰功 ==
张俭搬到[[柳城郡|营州]]當都督，並兼任护东夷校尉。後來因為某事被免職，朝廷便叫他以平民的身份领职事。营州部這個地方与[[契丹|契丹]]、[[奚族|奚]]、[[霫|霫]]、[[靺鞨|靺鞨]]等外族接壤，[[高句丽|高句丽]]引他们入侵大唐帝國，於是张俭率兵打敗他們，又得到营州都督這個官位。<ref name=BT83/><ref name=NBT111/>

西元641年十一月，[[薛延陀|薛延陀]]攻打突厥，突厥[[李思摩|俟利苾可汗]]無法抵擋攻勢，派遣大使到唐朝告急。唐太宗命令张俭率領軍隊到奚、霫、契丹压薛延陀东境，兵部尚书[[李勣|李勣]]为[[朔方|朔方]]行军总管，右卫大将军[[李大亮|李大亮]]为[[灵州|灵州]]道行军总管，[[庆州|庆州]]道行军总管[[张士贵|张士贵]]出[[云中|云中]]，[[凉州|凉州]]都督[[李袭誉|李袭誉]]为凉州道行军总管，分道以御之。十二月，唐军打败薛延陀。<ref>[[:s:旧唐书/卷003|《旧唐书》卷三]]</ref><ref>[[:s:新唐书/卷217下|《新唐书》卷二百一十七下]]</ref><ref name=ZZTJ196>[[:s:资治通鉴/卷196|《资治通鉴》卷一百九十六]]</ref>

西元642年十一月，张俭報告朝廷說[[高句丽|高句丽]]东部大人[[渊盖苏文|渊盖苏文]]殺掉他的國王[[荣留王|高武]]。當時[[渊盖苏文|渊盖苏文]]自称[[莫离支|莫离支]]。唐太宗決定遠征辽东，西元644年七月，派遣张俭、守左宗卫率[[高履行|高履行]]等率領幽、营二都督兵及契丹、奚、靺鞨军先击辽东抄掠，以观其势。<ref>[[:s:新唐书/卷002|《新唐书》卷二]]</ref><ref>[[:s:命张俭等征高丽诏|:s:命张俭等征高丽诏]]</ref>张俭率領唐軍到了辽西，但是因為辽河涨潮，一直無法渡河，十一月，太宗卻認為他畏敌懦弱，叫他回來<ref>[[:s:新唐书/卷078|《新唐书》卷七十八]]</ref>。张俭到[[洛阳|洛阳]]見到唐太宗，並当面告訴皇帝戰爭情況及當地的[[地形|地形]]與[[氣候|氣候]]以及一直不出兵的理由，太宗聽了很高兴，仍任命他为行军总管，並帶領各族的軍隊，成為六军[[前锋|前锋]]。当时抓获的高句丽候者称莫离支将要到辽东，诏张俭率兵从新城路邀击之，莫离支终究不敢出战。太宗以李勣为辽东道行军大总管，江夏王[[李道宗|李道宗]]副之，张士贵、张俭、[[执失思力|执失思力]]、[[契苾何力|契苾何力]]、[[阿史那弥射|阿史那弥射]]、[[姜行本|姜行本]]、[[麹智盛|麹智盛]]、[[吴黑闼|吴黑闼]]为行军总管隶之，率骑士六万趋辽东。<ref>[[:s:新唐书/卷220|《新唐书》卷二百二十]]</ref>西元645年四月，张俭帶領胡人军队为前锋进兵渡辽，並趨近建安城，高句丽大敗，被殺了數千人。<ref>[[:s:资治通鉴/卷197|《资治通鉴》卷一百九十七]]</ref><ref>[[:s:克高丽辽东城诏|:s:克高丽辽东城诏]]</ref>张俭因為此戰功被封為'''皖城郡公'''，赏赐甚厚<ref name=BT83/><ref name=NBT111/>。

西元646年六月，唐太宗任命李道宗、左卫大将军[[阿史那社尔|阿史那社尔]]为瀚海安抚大使，並命令右领卫大将军[[执失思力|执失思力]]率突厥兵、右骁卫大将军[[契苾何力|契苾何力]]率凉州及胡兵、代州都督[[薛万彻|薛万彻]]、张俭各率所部兵分道并进，以攻擊刚被[[回纥|回纥]]擊敗的薛延陀，太宗也亲自到[[灵州|灵州]]为大唐軍隊声援<ref>[[:s:旧唐书/卷199下|《旧唐书》卷一百九十九下]]</ref>。薛延陀因此大敗，[[多弥可汗|多弥可汗]]逃走被回纥攻杀<ref>[[:s:资治通鉴/卷198|《资治通鉴》卷一百九十八]]</ref>。戰爭勝利後，把东夷校尉改成东夷都护，並任命张俭擔任此職務<ref name=BT83/><ref name=NBT111/>。

== 过世 ==
[[唐高宗|唐高宗]][[永徽|永徽]]初年，加封為[[金紫光禄大夫|金紫光禄大夫]]。永徽四年，在其任內去世，享壽六十歲，谥'''密'''。<ref name=BT83/><ref name=NBT111/>

女儿张氏年幼时，张俭夫妇生病，女儿服侍父母親，一步也不離開，晝夜皆問候父母親，十分早熟與孝順。稍微年長時，更加恭敬與孝順。张氏後來嫁给延寿公[[于钦明|于钦明]]的兒子[[于敏直|于敏直]]。當女兒张氏一聽到父親得病時，非常的傷心，甚至想與父親同去。张俭死后，張氏聽到消息，便号恸而亡。高宗下诏赐物百段，並且命令史官記錄此事。<ref>[[:s:旧唐书/卷193|《旧唐书》卷一百九十三]]</ref>

张俭的哥哥[[张大师|张大师]]，累積軍功當到[[太仆卿|太仆卿]]、[[华州|华州]]刺史、武功县男。张俭的弟弟[[张延师|张延师]]，在永徽初年時，當到左卫大将军，並封為范阳郡公。根據唐朝的制度，三品以上官员，齊家门前會有棨戟，张俭三兄弟的门前皆立戟，當時人認為是非常光榮的事，便稱為“三戟张家”。<ref name=BT83/><ref name=NBT111/>另一弟张立德，秦城都尉。<ref>《[[张去奢|张去奢]]墓志》</ref>
