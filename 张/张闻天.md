'''张闻天'''（{{bd|1899年|8月30日|1976年|7月1日|catIdx=Z張}}），原名'''张应皋'''（也作'''张荫皋'''），字'''闻天'''，曾經长期使用[[化名|化名]]'''洛甫'''{{NoteTag|“洛甫”源自张闻天在苏联读书时的俄文名“伊思美洛夫”（Измайлов）{{r|程中原|page=71}}}}，[[江苏省|江苏省]][[南汇县|南汇县]]（今属[[上海市|上海市]][[浦东新区|浦东新区]]）人。

张闻天1925年加入[[中国共产党|中国共产党]]，同年前往[[苏联|苏联]]，先后在[[莫斯科中山大学|莫斯科中山大学]]、[[红色教授学院|红色教授学院]]学习、任教，是当时在莫斯科中山大学[[二十八个半布尔什维克|二十八个半布尔什维克]]之一。张闻天1930年回国，1933年1月到达中央革命根据地[[瑞金|瑞金]]，担任[[中共中央政治局委员|中共中央政治局委员]]、[[中共中央书记处书记|中共中央书记处书记]]。[[遵义会议|遵义会议]]后，张闻天被推举为[[中国共产党中央委员会总书记|中国共产党中央委员会总书记]]<ref>[http://history.people.com.cn/GB/205396/15024510.html 遵义会议后张闻天究竟是不是总书记？]</ref>。到达[[陕北|陕北]]之后，随着[[毛泽东|毛泽东]]权力的扩大，张闻天的地位逐渐下降。[[中国抗日战争|中国抗日战争]]期间，张闻天任[[中共中央宣传部|中共中央宣传部]]部长、[[马列学院|马列学院]]院长。[[第二次国共内战|第二次国共内战]]时期，张闻天任中共[[合江省|合江省]]委书记、[[中共中央东北局|中共中央东北局]]宣传部部长、中共[[辽东省|辽东省]]委书记。[[中华人民共和国|中华人民共和国]]建国后，张闻天主要从事[[外交|外交]]工作，曾任[[中国驻俄罗斯/苏联大使列表|中华人民共和国驻苏联大使]]、[[中华人民共和国外交部|中华人民共和国外交部]]第一副部长等职务。1959年，张闻天被划为“[[彭德怀反党集团|彭德怀反党集团]]”成员，受到批判，改而从事[[社会主义|社会主义]]经济建设理论的研究。张闻天于“[[文化大革命|文化大革命]]”中被批斗，“文革”结束前病逝<ref name="min">{{cite web |url = http://dangshi.people.com.cn/GB/85039/14623272.html |title = 张闻天向毛泽东移交权力始末 |accessdate = 2011 |publisher = 中共新闻网}}</ref>。

== 生平 ==
=== 早年生涯 ===
{{See also|莫斯科中山大学|二十八个半布尔什维克}}

[[清|清]][[光绪|光绪]]二十五年（1899年）7月25日<ref>{{Cite web |url=http://www.shtong.gov.cn/newsite/node2/node2245/node4467/node20332/node20410/node63793/userobject1ai15806.html |title=《上海文物博物馆志》>>第一编 文物古迹>>第五章 名人故居、祠、墓、纪念碑>>第一节 故居 |access-date=2014-11-17 |archive-url=https://web.archive.org/web/20141129053746/http://www.shtong.gov.cn/newsite/node2/node2245/node4467/node20332/node20410/node63793/userobject1ai15806.html |archive-date=2014-11-29 |dead-url=yes }}</ref>，张闻天出生于[[江苏|江苏]][[南汇|南汇]]祝桥镇邓三村张家宅<ref>{{cite journal |author=张勤龙 |title=张闻天的老房子 |journal=新民晚报 |issue= |pages= |url = http://news.163.com/14/1023/15/A98I1A6I00014AED.html |date=2014年10月}}</ref>。张闻天的先祖在[[北宋|北宋]]末年为躲避战乱从[[清河郡|清河郡]]迁徙至浦东。[[明朝|明朝]][[永乐_(明朝)|永乐]]年间，[[明成祖|明成祖]]实行驱富民填[[燕京|燕京]]的政策，张氏族人大多迁往[[北京|北京]]，只有[[张仲清|张仲清]]在利造桥隐居下来，并成为当地张姓家族的始祖{{r|程中原|page=1}}。张闻天的祖父张祥福、父亲张芹梅皆为农民，母亲金甜花曾进过私塾，张闻天是家中长子。给张起名的是同村长辈张桂唐，取《[[诗经|诗经]]》中“鹤鸣于九皋，声闻于天”之意，为其起名“应皋”（也作“荫皋”），字“闻天”{{r|程中原|page=2}}。张闻天早年先后在族侗私塾、康家宅小学和惠南小学读书。1915年，考入[[江苏省立水产学校|江苏省立水产学校]]（吴淞水产学校），学习两年后，因身体无法适应学校环境而转出{{r|程中原|page=9}}。1917年，入南京[[河海工程专门学校|河海工程专门学校]]。此时，张闻天接触到了《新青年》，思想开始倾向革命{{r|程中原|page=13}}。

[[五四运动|五四运动]]爆发后，张闻天参加学生运动{{r|程中原|page=13}}，并开始从事文艺创作和翻译，后在南京加入[[少年中国学会|少年中国学会]]{{r|程中原|page=20}}。1920年7月，张闻天与[[沈泽民|沈泽民]]一同留学日本，学习日语并自学哲学及其他社会科学书籍{{r|程中原|page=23-24}}。1921年1月，张闻天回国，8月任[[中华书局|中华书局]][[编辑|编辑]]，与[[左舜生|左舜生]]、[[田汉|田汉]]、[[李达_(哲学家)|李达]]、[[何叔衡|何叔衡]]为同事{{r|程中原|page=28}}。1922年1月5日，张闻天发表《中国底乱源及其解决》{{r|程中原|page=30}}。文章说：“我们对于这种不合理的社会，早感到不安，不过用什么方法去改造呢?应该改造成什么样呢？无抵抗主义呢？反抗主义呢？无政府主义呢？社会主义呢？如江河流水，不绝地引起我的烦闷。但永久不决是不能生活的。那么，取其长，舍其短，自然不能不走社会主义一条路了<ref>{{cite book |author=张闻天选集传记组|title=张闻天早期文集（1919.7—1925.6）|year=2010|publisher=中共党史出版社|isbn=9787801363190|page=98-107}}</ref>。”8月，张闻天自费赴美国[[加利福尼亚大学伯克利分校|加利福尼亚大学伯克利分校]]留学，曾选过几门课程，但不是正式学生{{r|程中原|page=40}}。

张闻天于1924年1月回国，继续任中华书局编辑{{r|程中原|page=45}}。1924年10月，张闻天受[[少年中国学会|少年中国学会]]成都分会之邀，入川担任重庆第二女子师范学校英文教员，与[[萧楚女|萧楚女]]、[[楊闇公|楊闇公]]为同事{{r|程中原|page=53}}。1925年5月，张闻天因宣扬革命思想，与萧楚女、[[廖划平|廖划平]]一起被重庆卫戍司令[[王陵基|王陵基]]勒令离川，返上海{{r|程中原|page=62}}。6月，张闻天经沈泽民、[[董亦湘|董亦湘]]介绍加入中国共产党{{r|程中原|page=64}}。10月，赴[[莫斯科中山大学|莫斯科中山大学]]学习{{r|程中原|page=69}}。

1928年初，中山大学少共支部局一些人对学校支部局工作不满<ref name="陈清泉2006">{{cite book|author=陈清泉|title=在中共高层50年 陆定一传奇人生|url=http://books.google.com/books?id=JA9lAAAAIAAJ|year=2006|publisher=人民出版社|isbn=978-7-01-005363-9|page=47}}</ref>。校支部局以无视党的领导为名，给他们扣上了“先锋主义”帽子，并重新改组少共支部<ref name="孙耀文1996">{{cite book|author=孙耀文|title=风雨五载 莫斯科中山大学始末|url=http://books.google.com/books?id=iuVlAAAAIAAJ|year=1996|publisher=中央编译出版社|isbn=978-7-80109-140-6|page=276}}</ref>。当时苏联内部的两派斗争在1928年夏形成高峰，因此召开了十天会议去讨论学校支部局的工作，当时苏共区委书记[[芬可夫斯基|芬可夫斯基]]压阵支持学校支部局{{r|博古|page=60}}，虽然遭到多数学生反对，但在博古等少数派支持下，仍然通过表决，支持学校支部局工作。会后，因拥护支部局的一派骄傲起来，反对派起名绰号“[[二十八个半布尔什维克|二十八个半布尔什维克]]”{{r|博古|page=62}}，张闻天为其中之一。

1928年9月，张闻天进入红色教授学院学习{{r|程中原|page=78}}。1930年5月19日，张闻天发表《论两条路线的斗争》，文章指出，各国共产党不仅要反对党内的[[右倾|右倾]]，也要反对党内的[[左倾|左倾]]{{r|程中原|page=83}}。这篇文章被认为是针对[[李立三|李立三]]与“[[立三路线|立三路线]]”{{r|程中原|page=84}}。

=== 第一次国共内战时期 ===
{{see also|中华苏维埃共和国|博古|毛泽东|王稼祥|罗明路线|长征|遵义会议|西安事变}}

1931年2月，张闻天回到[[上海|上海]]，不久担任[[中共中央宣传部|中共中央宣传部]]部长{{r|程中原|page=88}}<ref>{{cite journal|url=http://history.news.163.com/09/0619/14/5C67M1JF00011247.html|title=杨尚昆1986年谈张闻天与毛泽东的关系演变|journal=炎黄春秋|year=2009|issue=3|author=张培森整理}}</ref>。同年9月，成立临时中央，临时中央政治局由[[博古|博古]]、张闻天、[[卢福坦|卢福坦]]、[[李竹生|李竹生]]、[[康生|康生]]、[[陈云|陈云]]、[[黄平_(中共中央候补委员)|黄平]]、[[刘少奇|刘少奇]]、[[王云程|王云程]]等九人组成；博古、张闻天、卢福坦为常委{{r|博古|page=85}}。1932年1月，[[一二八事变|一二八事变]]发生后，张闻天受阶级斗争思维模式干扰，指斥[[十九路军|十九路军]]将领是反动的、反革命的“抗日军阀”<ref>张闻天《上海事变与中国的统治阶级》，《斗争》第3期，1932年2月9日</ref>，提出党的策略是“拥护士兵自动抗日的斗争，而反对他们的长官，利用一切机会揭破这些长官的欺骗”；号召革命的士兵与民众联合起来，把军阀的武装变为民众的武装，甚至认为“要作战胜利必须杀掉他们反动的长官”{{r|程中原|page=98}}<ref>张闻天《士兵的反日战争与我们的策略》，《斗争》第2期，1932年1月31日</ref>。

1932年12月，上海地下党再次遭到破坏{{r|程中原|page=106}}。政治局常委卢福坦等被捕变节，致使张闻天、陈云等险些被捕。博古在请示[[共产国际|共产国际]]后，决定整体迁移中央首脑机关进入江西中央苏区{{r|博古|page=99-100}}。1933年1月，张闻天随临时中央迁入[[中央苏区|中央苏区]]，任[[苏区中央局|苏区中央局]]宣传部长{{r|程中原|page=107}}、瑞金工农民主政府主席等。

当时恰逢国民政府准备[[第四次围剿战争|第四次围剿战争]]，[[蒋介石|蒋介石]]亲自挂帅进攻，博古同时召开中共中央局会议，主张坚守阵地。这一主张遭到了[[中共福建省委|中共福建省委]]代理书记[[罗明_(广东)|罗明]]的上书反对，认为民众没有实力抵御长期的阵地战、应当继续游击战{{r|郭廷以|page=431}}，且直言中央局的决议“空洞说教”{{r|博古|page=110-111}}。博古读后勃然大怒并起草《中央局关于闽粤赣省委的决定》，指责罗明为主的“机会主义”（罗明主义），并随后开展反罗明路线运动，大量基层和前线干部被牵连撤职。1933年2月18日，张闻天在中共中央机关刊物《斗争》上发表《什么是罗明同志的机会主义路线》一文<ref>{{cite book | title=张闻天文集·第一卷 | publisher=中共党史出版社 | author=张闻天，责任编辑：萧淮苏 | year=1995| location=北京 | pages=325| isbn=9787800231872}}</ref>，被认为是具有定调子的作用{{r|叶永烈|page=250-251}}。此后，博古批判“江西的罗明路线”，张闻天又发表《罗明路线在江西》{{r|叶永烈|page=253}}、《火力向着右倾机会主义》{{r|叶永烈|page=258}}等文章，支持博古的行动。在这些运动中，包括[[张鼎丞|张鼎丞]]、[[刘晓_(大使)|刘晓]]、[[邓小平|邓小平]]、[[毛泽覃|毛泽覃]]、[[谢维俊|谢维俊]]、[[古柏_(人名)|古柏]]、[[何叔衡|何叔衡]]等人均被牵连{{r|程中原|page=111}}。

[[File:张闻天结婚.jpg|thumb]]
1934年1月，在[[中共六届五中全会|中共六届五中全会]]上当选为中央政治局委员、[[中央书记处书记|中央书记处书记]]。2月，在中华苏维埃二大上当选为中央政府人民委员会主席{{r|程中原|page=124}}。[[广昌县|广昌]]失守后，张闻天在中革军委一次会议上批评博古、[[李德|李德]]等人的军事指挥，开始与他们产生隔阂{{r|程中原|page=122}}。1934年8月，中央苏区机关驻地[[沙洲坝|沙洲坝]]遭国民党轰炸，中央机关被迫搬迁至[[瑞金|瑞金]]云石山。此后，张闻天开始与[[毛泽东|毛泽东]]交流、接近{{r|程中原|page=126}}<ref>{{cite journal|author=刘良|title=长征前夕的毛泽东|journal=党史文苑|year=2006|month=|issue=11|pages=| url=http://www.qingdaonews.com/content/2006-07/21/content_7318350.htm}}</ref>。

1934年10月，张闻天参加[[长征|长征]]，与毛泽东、[[王稼祥|王稼祥]]等同编入军委第二纵队{{r|程中原|page=130}}。三人在行军途中多次交流，结成同盟，共同反对博古、李德{{r|程中原|page=131}}。1935年1月15日至17日，[[红一方面军|红一方面军]]长征途中占领了[[贵州|贵州]][[遵義市|遵义]]，於是順道於當地举行會議，地點在遵义城琵琶桥（后改名子尹路，又改称红旗路）东侧87号[[柏辉章|柏辉章]]公馆。会议上[[博古|博古]]率先作报告，检讨[[第五次反围剿战役|第五次反围剿战役]]的军事指挥错误，但主要是客观条件的困难{{r|程中原|page=135}}。[[周恩来|周恩来]]随后做副报告，称是军事领导战略战术错误，并主动承担责任{{r|程中原|page=137}}。紧接着，张闻天在会上作反对中央单纯防御军事路线的报告<ref>{{cite book |author=Lawrence R. Sullivan |title=Historical Dictionary of the Chinese Communist Party |url = http://books.google.com/books?id=g0PjFe5i0iEC&pg=PA318 |date=16 December 2011|publisher=Scarecrow Press|isbn=978-0-8108-7225-7|page=318}}</ref>。随后[[毛泽东|毛泽东]]作长篇发言，指出导致第五次反围剿战役失利和大转移严重损失的原因是军事上的单纯防御路线，表现为进攻时的冒险主义、防御时的保守主义和突围时的保守主义{{r|程中原|page=136}}。这个发言的提纲是毛泽东与王稼祥、张闻天在先前拟好的。随即，[[王稼祥|王稼祥]]、[[朱德|朱德]]、[[刘少奇|刘少奇]]纷纷表示赞同。会议召开三日，最终决定取消原来的三人团（[[博古|博古]]、[[李德|李德]]、[[周恩来|周恩来]]）领导{{r|程中原|page=139}}，由朱德和周恩来为军事指挥者，增补[[毛泽东|毛泽东]]为[[中共中央政治局常委|中共中央政治局常委]]，毛泽东恢复了对红军的指挥权{{r|程中原|page=138}}。

2月5日，红军进抵贵州[[毕节|毕节]]林口镇鸡鸣三省村，在这里召开政治局会议{{r|程中原|page=135}}。毛泽东此时对周恩来称“应该让洛甫做一个时期”<ref>{{cite book|author=王树增|title=《长征》|year=2006|publisher=人民文学出版社|isbn=978-7-02-005798-6|page=261}}</ref>，于是张闻天取代博古担任中共中央总书记<ref>Snow, Edgar. Red Star Over China, Pelican edition 1972, page 553.</ref><ref>{{cite book |author = [美]徐中约 |others = 计秋枫、朱庆葆 译，茅家琦、钱乘旦 校 |title = 中国近代史（下）| year = 2005| publisher = 香港中文大学|page =564 |isbn=9622019889}}</ref><ref>{{cite journal |author=何方 |url = http://history.people.com.cn/GB/205396/15024507.html|journal=炎黄春秋|year=2011|issue=1 |title=遵义会议后张闻天是不是总书记？}}</ref><ref>{{cite web |url = http://news.ifeng.com/history/2/200712/1217_336_333899_9.shtml |title=张闻天的“总书记问题”：一桩未了公案|publisher=凤凰网|date=2007-12-17 |accessdate=2015-05-10}}</ref>{{r|郭廷以|page=433}}，在常委分工中“负总的责任”{{NoteTag|1938年4月12日武汉《[[新华日报|新华日报]]》刊载《张闻天启事》：“中共中央设有由少数同志组织之书记处，但并无所谓总书记。”}}<ref>{{cite journal |author=梁俊英|title=张闻天：历史功绩载千秋|journal=党史文汇|year=2014|month=|issue=4|pages= |url = http://dangshi.people.com.cn/n/2014/0603/c85037-25097848.html}}</ref><ref>{{cite book |author=Michael Dillon|title=Deng Xiaoping A Political Biography |url = http://books.google.com/books?id=OqjeBAAAQBAJ&pg=PA62 |date=30 October 2014|publisher=I.B.Tauris|isbn=978-1-78076-895-3|page=62}}</ref>。5月12日，张闻天主持[[会理会议|会理会议]]{{r|程中原|page=151}}。1935年6月红一、四方面军会师后，张闻天支持毛泽东的北上路线，反对[[张国焘|张国焘]]的南下路线，随军到达陕北{{r|程中原|page=169}}。为此，张国焘在另立中央时，宣布“开除”毛周洛博的党籍{{r|程中原|page=202}}。1935年12月，张闻天主持[[瓦窑堡会议|瓦窑堡会议]]<ref>{{cite book |authors = 周天度、郑则民、齐福霖、李义彬 等 |editors = 李新 总主编，中国社会科学院近代史研究所中华民国史研究室编 |title=中华民国史|volume=第八卷|location=北京|publisher=中华书局|year=2011 |page = 424}}</ref>。

1936年，红军进行东征、西征，在此期间，张闻天与毛泽东、周恩来等合作，将推进抗日民族统一战线置于中共的工作重心{{r|程中原|page=186}}<ref name="传略">张闻天选集传记组. 张闻天传略. 1990年2月14日</ref>。12月[[西安事變|西安事變]]发生后，張聞天在[[志丹县|保安]][[窯洞|窯洞]]主持召開應對事變的會議，在會上毛澤東提出要“審蔣”、“除蔣”<ref>{{cite journal|author=张培森|title=三大历史关口的张闻天|journal=炎黄春秋|year=2010|month=|issue=6|pages=|url=http://www.21ccom.net/articles/rwcq/article_2010083117298.html|archiveurl=https://web.archive.org/web/20160305114754/http://21ccom.net/articles/rwcq/article_2010083117298.html|archivedate=2016-03-05|deadurl=yes}}</ref><ref>{{cite journal|author=卢荻|title=西安事变：杀蒋与放蒋的艰难抉择|journal=同舟共进|year=2011|month=|issue=1|pages=| url=http://history.people.com.cn/GB/205396/14987043.html}}</ref>，而張聞天主張“不採取與南京對立方針，不組織與南京對立方式”、“盡量爭取南京政府正統，聯合非蔣係隊伍”<ref name="zps" /><ref name="time">{{cite web|url= http://cul.sohu.com/20120105/n331233203_1.shtml|title= 毛泽东成为领袖后张闻天的悲惨命运|accessdate= 2012|publisher= 搜狐网|work= |archive-url= https://web.archive.org/web/20141021191857/http://cul.sohu.com/20120105/n331233203_1.shtml|archive-date= 2014-10-21|dead-url= yes}}</ref>。最後在[[斯大林|斯大林]]的壓力下，[[西安事變|西安事變]]和平解決<ref>{{cite journal|author=杨奎松|title=张学良与西安事变之解决|journal=中国社会科学|year=1996|month=|issue=5|pages=186－199| url=http://www.people.com.cn/GB/198221/198974/199958/12496628.html}}</ref><ref>{{cite journal|author=李义彬|title=中共和平解决西安事变方针的制定|journal=中共党史研究|year=2007|month=|issue=1|pages=|url=http://jds.cass.cn/UploadFiles/ztsjk/2010/11/201011251535549426.pdf|deadurl=yes|archiveurl=https://web.archive.org/web/20140811001306/http://jds.cass.cn/UploadFiles/ztsjk/2010/11/201011251535549426.pdf|archivedate=2014-08-11}}</ref><ref>{{cite journal|title=洛甫、毛泽东致周恩来、博古电(1937年1月5日)|journal=文獻和研究|year=1986|issue=6|pages=37}}</ref>。1937年1月，张闻天秘密前往西安，与周恩来、博古等商议办法<ref>{{cite web |url = http://sn.people.com.cn/n/2014/0716/c354836-21683833.html |title = 张闻天在西安事变中：“一定要争取和平调解”|accessdate = 2014-07-16 |date = |publisher = 人民网 }}</ref>。

=== 抗日战争时期 ===
{{see also|第二次国共合作|洛川会议|中共六届六中全会|中共中央宣传部|延安整风}}
[[File:1938_group_photo.jpg|thumb]]、[[彭德懷|彭德懷]]、[[王稼祥|王稼祥]]、張聞天、[[朱德|朱德]]、[[博古|博古]]、[[王明|王明]]、[[康生|康生]]、[[項英|項英]]、[[劉少奇|劉少奇]]、[[陳雲|陳雲]]、[[周恩來|周恩來]]]]
[[File:Zhang_Wentian-2.jpg|thumb]]

中日全面战争爆发后，1937年8月，中共中央在[[陕西|陕西]][[洛川|洛川]]冯家村召开政治局扩大会议，张闻天主持会议，并作国内外政治形势的分析报告，提出坚持抗日十大纲领，反对民族失败主义{{r|程中原|page=245-246}}<ref>{{cite book | title=张闻天文集·第二卷 | publisher=中共党史出版社 | author=张闻天，责任编辑：萧淮苏 | year=1992 | location=北京 | pages=343-349| isbn=9787800235719}}</ref>。1937年11月底，[[王明|王明]]、[[陈云|陈云]]、[[康生|康生]]由莫斯科回到延安{{r|程中原|page=254}}。张闻天主持12月政治局会议，会上，张闻天在会上作了《目前的政治形势与党的任务》的报告。<ref>{{cite web |url = http://dangshi.people.com.cn/GB/151935/176588/176593/10555467.html |title = 中央政治局会议(1937年12月9-14日) |accessdate = 2012-11-28 |date = |publisher = 中国共产党新闻网 }}</ref>该会议按[[共产国际|共产国际]]还有[[斯大林|斯大林]]的临行交代，改组书记处，增补王明、陈云和康生为书记，实行[[中国共产党集体领导制度|集体领导]]。之后，中央决定，由张闻天主党，毛泽东主军，王明主统战，形成三驾马车式格局{{r|程中原|page=257}}。1938年，张闻天曾反对毛泽东与[[江青|江青]]结婚<ref>{{cite web |url = http://news.ifeng.com/history/phtv/wdzgx/detail_2011_12/27/11597160_0.shtml |title = 张闻天劝阻毛泽东与江青结婚：你得为你威信考虑 |accessdate = 2011-12-27 |date = |publisher = 凤凰网 }}</ref>。

[[中共六届六中全会|中共六届六中全会]]后，张闻天实际上从最高领导人的地位上退下{{r|程中原|page=268}}，改为负责宣传工作和干部教育{{r|叶永烈|page=429}}，任中共中央书记处书记兼[[中国共产党中央宣传部|中央宣传部]]部长、[[马列学院|马列学院]]院长{{r|程中原|page=272}}、西北工作委员会主任、《解放》周刊主要负责人、《共产党人》编辑等职务。1941年[[皖南事变|皖南事变]]后，张闻天多次检讨过去的左倾错误{{r|程中原|page=300}}。

1942年2月，毛泽东发动[[延安整风|延安整风]]运动，反对[[主观主义|主观主义]]以整顿学风，反对[[宗派主义|宗派主义]]以整顿党风，反对[[党八股|党八股]]以整顿文风，打击以[[王明|王明]]为首的国际派，使[[马克思主义|马克思主义]]中国化，学习[[毛泽东思想|毛泽东思想]]{{r|郭廷以|page=478}}。其中，“党八股”一词为张闻天首创{{r|叶永烈|page=434}}。期间，张闻天响应毛泽东的号召，到陕北和晋西北农村作了一年多的农村调查，写出了《出发归来记》{{r|程中原|page=302-310}}。1943年3月，张闻天调查结束归来后出席中央政治局会议，在会上中共中央改组，设立中央书记处主席一职，由毛泽东担任，由此确立毛的一元化领导。此后，张闻天任[[中央组织委员会|中央组织委员会]]委员{{r|叶永烈|page=431}}，并从事陕北地区经济发展状况调查、研究国内外形势和主编党内刊物《[[参考资料_(出版物)|参考资料]]》<ref name="传略"/>。1945年6月，在延安举行的[[中共七大|中共七大]]上当选为中共中央政治局委员{{r|程中原|page=319}}，并担任[[中央政治研究室|中央政治研究室]]主任<ref>{{cite book|author=中共中央文献研究室|title=任弼时年谱|year=2004|publisher=中央文献出版社|isbn=9787507315967|pages=487}}</ref>。

=== 第二次国共内战时期 ===
{{See also|合江省 (解放区)|土地改革运动|新民主主义}}
[[File:Dongbei_Ju_in_Harbin.png|thumb]]、[[高岗|高岗]]、[[陈云|陈云]]、张闻天、[[吕正操|吕正操]]]]

抗日战争胜利后，张闻天主动要求到东北从事地方工作<ref name="传略"/>。1945年10月22日，张闻天与[[高岗|高岗]]、[[李富春|李富春]]、[[王鹤寿|王鹤寿]]、[[凯丰|凯丰]]、[[陈光_(军事家)|陈光]]、[[朱瑞_(1905年)|朱瑞]]等乘坐美军调停小组的飞机前往[[邯郸|邯郸]]，于11月22日到[[沈阳|沈阳]]<ref>{{cite book |authors=戴茂林、赵晓光 |title=高岗传|year=2011|publisher=陕西人民出版社|isbn=978-7-224-09634-7|page=143-144}}</ref>。此后，张闻天等被派到[[北满洲|北满]]{{NoteTag|指满洲（中国东北）北部，即当时的[[松江省_(中華民國)|松江]]、[[合江省|合江]]、[[黑龙江省_(中华民国)|黑龙江]]、[[嫩江省|嫩江]]、牡丹江（[[綏寧省|绥宁]]）五省和[[哈尔滨市|哈尔滨市]]}}。在[[陈云|陈云]]主持下，他参与起草并与陈云、高岗联名向中央发出电报，建议东北工作方针应是及早将武装力量和干部分散到广大农村和中小城市，建立巩固的根据地，而不应将注意力过多集中于大城市<ref>{{cite book |author=叶永烈|title=他影响了中国：陈云全传|year=2013|publisher=四川人民出版社、华夏出版社|isbn=978-7-220-08775-2|pages=126}}</ref>。12月，张闻天出任中共[[合江省_(解放区)|合江省]][[省委书记|省委书记]]。由于当时合江省[[土匪|土匪]]众多，阻碍交通要道，故张闻天先在[[宁安县|宁安县]]从事社会调查{{r|程中原|page=324-327}}，合江军区司令员[[方强|方强]]则化装前往[[佳木斯|佳木斯]]{{r|刘统|page=191}}。

1946年5月，张闻天到佳木斯正式上任{{r|程中原|page=327}}。此后，主持合江地方政权建设、剿匪、[[土地改革|土地改革]]等工作，后又兼任[[合江军区|合江军区]]政治委员<ref>{{cite book |author=张明金、刘立勤 |title=中国人民解放军历史上的200个军区|year=2010|publisher=解放军文艺出版社 |location=北京市西城区地安门西大街40号 |pages=344|isbn=9787503322525}}</ref>。1946年7月3-11日，出席在[[哈尔滨|哈尔滨]]召开的东北局扩大会议，这个会议通过了《[[七七决议|七七决议]]》<ref>{{cite web |url = http://dangshi.people.com.cn/GB/151935/176588/176594/10555486.html |title = 东北局扩大会议(1946年7月3-11日) |accessdate = |date = |publisher = 中国共产党新闻网 }}</ref>。1947年，在东北[[土地改革运动|土地改革运动]]中，出现了“左”的倾向，发生“扫堂子{{NoteTag|方言。本是一种迷信活动谓请跳大神的驱除家宅中的野鬼。土改中亦喻指斗争地主扫除封建势力。}}”等过激行为{{r|刘统|page=360-363}}。对此，张闻天下令合江地区的报纸禁止转载“扫堂子”的经验{{r|程中原|page=336}}{{r|刘统|page=364}}，并保护合江工商业，起草《在平分土地运动中保护工商业的指示》{{r|刘统|page=365}}。

1948年5月，张闻天任[[中共中央东北局|中共中央东北局]]常委兼组织部长，并任[[东北财政经济委员会|东北财政经济委员会]]副主任{{r|程中原|page=341}}。在此期间，张闻天起草《关于东北经济构成及经济建设基本方针的提纲》，提出“五种经济成分”思想{{r|程中原|page=342-345}}。《提纲》受到毛泽东肯定，并将其转发全党，作为各解放区经济建设的方针{{r|程中原|page=346}}。1949年3月，出席[[中共七届二中全会|中共七届二中全会]]{{r|程中原|page=347}}。4月，调任[[辽东省|辽东省]]省委书记、[[辽东军区|辽东军区]]政治委员<ref>{{cite book |author=《中国人民解放军军史》编写组|title=中国人民解放军军史：第三卷|year=2010|publisher=军事科学出版社|page = 333 |isbn=978-7-80237-381-5}}</ref>，组织发展生产、改善人民生活等工作{{r|程中原|page=348}}。

1950年1月20日，张闻天任中华人民共和国出席[[联合国安理会|联合国安理会]]首席代表{{r|程中原|page=361}}，因中华人民共和国未取得席位而未赴任{{r|何方|page=105}}。1951年4月，张闻天任[[中国驻苏联大使|中国驻苏联大使]]{{r|何方|page=106}}<ref>{{cite web |url = http://www.mfa.gov.cn/chn/pds/ziliao/wjrw/2167/2171/t9237.htm |title = 驻前苏联及俄罗斯联邦历任大使 |publisher = 中华人民共和国外交部}}</ref>。张闻天注意研究國際問題，積極向中央提出形勢分析和政策建議，如1951年提出[[朝鲜战争|朝鲜战争]]第一次停戰談判問題報告，受到中央重視<ref>{{Cite web |url = http://www.fmprc.gov.cn/chn//pds/ziliao/wjrw/lrfbzjbzzl/t9115.htm |title = 张闻天 |publisher=[[中華人民共和國外交部|中華人民共和國外交部]]}}</ref>。1954年，与[[周恩来|周恩来]]等出席[[日內瓦會議_(1954年)|日内瓦会议]]{{r|程中原|page=378}}{{r|何方|page=117}}。9月，出席[[第一届全国人民代表大会第一次会议|第一届全国人民代表大会第一次会议]]，当选[[第一届全国人大常委会|第一届全国人大常委会]]委员{{r|程中原|page=499}}。1955年1月回国，任[[中华人民共和国外交部|中华人民共和国外交部]]第一副部长，协助周恩来领导外交工作。1956年9月[[中共八大|中共八大]]上当选为[[中共中央政治局|中共中央政治局]]候补委员{{r|程中原|page=500}}。

1959年，张闻天参加[[庐山会议_(1959年)|庐山会议]]{{r|程中原|page=401}}。7月14日，[[中共中央军委|中共中央军委]]副主席、[[国务院副总理|国务院副总理]]兼[[中华人民共和国国防部部长|国防部部长]][[彭德怀|彭德怀]]给[[中国共产党中央委员会主席|中国共产党中央委员会主席]][[毛泽东|毛泽东]]写信，信中首先肯定1958年“[[大跃进|大跃进]]”的方針是正確的，但接著彭德怀指出了“大跃进”的问题：“1958年的基本建設，現在看來有些項目是過急過多了一些，分散了一部分資金，推遲了一部分必成項目，這是一個缺點”，“1959年就不僅沒有把步伐放慢一點，加以适當控制，而且繼續大躍進，這就使不平衡現象沒有得到及時調整，增加了新的暫時困難”；他指出：“浮誇風、小高爐等等，都不過是表面現象；缺乏民主、個人崇拜，才是這一切弊病的根源{{r|彭德怀|page=358-361}}。”7月19日，[[中国人民解放军总参谋长|中国人民解放军总参谋长]][[黄克诚|黄克诚]]和[[中共湖南省委|中共湖南省委]]第一书记[[周小舟|周小舟]]发言支持彭德怀{{r|彭德怀|page=363-364}}<ref>{{cite web |publisher = 人民网 |title = 黄克诚为真理十次直言 |url = http://www.people.com.cn/GB/shizheng/252/5252/5253/20010521/469756.html |author = 徐焰 |date = 2001年5月21日 |accessdate = 2010-06-20 }}</ref>。7月21日，张闻天作了三个小时发言，用了更精确的理论术语批评[[大跃进|大跃进]]、[[人民公社|人民公社化]]的左倾错误，主张发扬党内民主，支持[[彭德怀|彭德怀]]{{r|程中原|page=413-419}}，张闻天强调应多从思想观点、方法、作风上探讨产生缺点错误的原因，反对不讲条件、不合乎实际的“主观主义、片面性”，“一定要按经济规律办事，不能光凭主观愿望，光凭政治上的要求”，在党内民主作风上，“要造成一种生龙活虎、心情舒畅的局面，才会有战斗力”，为此领导上要“造成一种风气、环境，使得下面敢于提意见”<ref>{{cite book|title=中国当代史 ||last1= [[郭大钧|郭大钧]]  |publisher= 北京师范大学出版社 |year=2011 |page=111}}</ref>。随后，原本主张“纠左”的毛澤東转而批判“[[彭德怀反党集团|彭德怀反党集团]]”{{r|李锐|page=424}}<ref>{{cite book |author=[英]理查德·伊文思 |others = 田山 译 |title=邓小平传|year=2013|publisher=国际文化出版公司|pages=183|isbn=978-7-5125-0483-7}}</ref>，张闻天被毛泽东归类为“右倾机会主义”分子、“[[彭德怀反党集团|彭德怀反党集团]]”的重要成员<ref>[https://zh.wikisource.org/wiki/%E4%B8%AD%E5%9B%BD%E5%85%B1%E4%BA%A7%E5%85%9A%E5%85%AB%E5%B1%8A%E5%85%AB%E4%B8%AD%E5%85%A8%E4%BC%9A%E5%85%B3%E4%BA%8E%E4%BB%A5%E5%BD%AD%E5%BE%B7%E6%80%80%E5%90%8C%E5%BF%97%E4%B8%BA%E9%A6%96%E7%9A%84%E5%8F%8D%E5%85%9A%E9%9B%86%E5%9B%A2%E7%9A%84%E9%94%99%E8%AF%AF%E7%9A%84%E5%86%B3%E8%AE%AE 《中国共产党八届八中全会关于以彭德怀同志为首的反党集团的错误的决议》]</ref>。会议后，“[[反右倾运动|反右倾运动]]”再次展开<ref>[[薄一波|薄一波]]，《若干重大决策与事件的回顾》，1997年，人民出版社，ISBN 978-7-01-002687-9。[http://cpc.people.com.cn/GB/64162/82819/122120/122135/7210831.html 第三十章“庐山会议的‘反右倾’”]</ref>，张闻天在[[全国外事工作会议|全国外事工作会议]]上被进一步批判，被免去外交部副部长职务<ref>{{cite book|title=中国共产党历史：第二卷（1949-1978）|volume=下|year=2011|location=北京|publisher=中共党史出版社|pages=551|author=中共中央党史研究室 |isbn=9787509809501}}</ref>。

1960年11月12日，到[[中国科学院|中国科学院]]哲学社会科学部经济研究所担任特约研究员，从事社会主义经济建设理论的研究{{r|程中原|page=430}}。1962年1月，出席[[七千人大会|七千人大会]]，担任大会主席团成员{{r|程中原|page=440}}。此后，前往江苏、浙江、上海、湖南等地进行社会调查{{r|程中原|page=443-449}}。调查结束后，写出调查报告“集市贸易意见书”{{r|程中原|page=450}}。[[中共八届十中全会|中共八届十中全会]]上，毛泽东又批判[[彭德怀|彭德怀]]、张闻天、[[习仲勋|习仲勋]]等{{r|程中原|page=453}}。会后，张闻天被剥夺了参加中央一切会议、阅读中央一切文件的权利{{r|程中原|page=454}}。此后，又提出“生产关系两重性”理论{{r|程中原|page=457}}。1966年10月1日，张闻天参加国庆庆典活动，登上天安门城楼与毛泽东交谈<ref>{{cite book |title = 共和国震撼瞬间 |date=2012 |publisher = [[人民文学出版社|人民文学出版社]]|author = [[孟昭瑞|孟昭瑞]] |pages = 256 |isbn = 978-7-0200-9124-9}}</ref>。

[[文化大革命|文化大革命]]时，张闻天受到迫害，与[[彭德怀|彭德怀]]同遭批斗<ref>{{cite web |url = http://news.qq.com/a/20120518/000368.htm |title = 文革中张闻天、彭德怀被红卫兵批斗和殴打|accessdate = |date = 2012-05-19 |publisher = 腾讯 }}</ref>。1968年5月17日起，张闻天在寓所内被“监护”{{r|程中原|page=469}}。期间，张闻天与刘英被隔开。[[康生|康生]]为获得对[[刘少奇|刘少奇]]不利的材料，派人要求张闻天对刘少奇负责“[[六十一人叛徒集团|六十一人叛徒集团]]”案一事作证，但张闻天顶住了压力<ref name="传略"/>。1969年10月24日，张闻天化名张普，与其家人被下放到[[广东|广东]][[肇庆|肇庆]]{{r|程中原|page=472}}<ref>{{cite web |url = http://news.qtv.com.cn/system/2014/09/02/011690217.shtml |title = 儿子忆开国元勋张闻天：晚年遭软禁后被迫改名 |accessdate = 2014-09-02 |date = |publisher = 青岛网络电视台 }}</ref>。[[九一三事件|九一三事件]]后，开始撰写一组后来被称为“[[肇庆文稿|肇庆文稿]]”的理论文章，批评[[极左|极左]]路线和“文革”的错误，阐述对中国社会主义建设基本问题的看法。在文稿中写道，[[社会主义社会|社会主义社会]]的任务，首先就是要发展[[生产力理论|生产力]]，“荒唐到把为人民群众物质利益而斗争的原则，也当作修正主义来咒骂，那就是不可容许的糊涂和欺骗！”针对“文革”中打倒一切的做法，他认为，把人民内部矛盾夸大为敌我矛盾，“给党造成极大的危害”。把持不同意见的同志诬陷为“党内资产阶级代理人”，“那更是混淆敌我，是对党的严重破坏”<ref name="zps">{{cite book |title = 张闻天选集 |accessdate = 2011年4月27日 |date=1993 |publisher = [[人民出版社|人民出版社]]|editor = 张培森 |pages = 590-599}}</ref>。1972年5月，中共中央决定恢复张闻天夫妇的工资待遇{{r|程中原|page=484}}。此后，张闻天多次请求返回北京，但未被批准{{r|程中原|page=486}}。1975年8月，张闻天迁居无锡{{r|程中原|page=487}}。

1976年7月1日，张闻天在江苏无锡因[[心脏病|心脏病]]猝发去世，終年76歲。1979年8月25日，中共中央在北京[[人民大会堂|人民大会堂]]召开追悼张闻天大会<ref>{{cite web |url = http://news.china.com/history/today/20140825/index.html |title = 历史上的今天：1979年8月25日 |accessdate = 2014-08-25 |date =  |publisher = 中华网 |deadurl = yes |archiveurl = https://web.archive.org/web/20150101152548/http://news.china.com/history/today/20140825/index.html |archivedate = 2015年1月1日 }}</ref>。

== 评价 ==
在中国共产党主流史学中，认为他作为中共中央的领导人，为中共立下了汗马功劳，因此对他的评价基本持正。[[毛泽东|毛泽东]]在延安时曾经对中央苏区时期的张闻天有过这样的评论：“张闻天在中央苏区是颇有影响的人物。这不只是因为他的地位和身份，还有他本身的因素。”1979年8月25日，中共中央在北京[[人民大会堂|人民大会堂]]召开追悼张闻天大会，[[邓小平|邓小平]]致悼词，称张闻天“是我党的优秀党员，是我党在一个相当长时期的重要领导人”。次日[[人民日报|人民日报]]发布的消息称张闻天是“中国共产党的优秀党员、老一辈无产阶级革命家”<ref>1979年8月26日《人民日报》</ref>。[[江泽民|江泽民]]1990年8月在给张闻天夫人刘英的信中说：“他对共产主义矢志不移的坚定信念，他的政治家的宽阔胸怀和学问家的谨严风范，他为人民利益而坚持真理、修正错误的崇高品德，他深入实际、实事求是、谦虚谨慎、艰苦朴素的优良作风，永远值得我们大家学习<ref>{{cite news|title=张闻天在中共党史上的十大贡献|url=http://news.xinhuanet.com/politics/2011-02/14/c_121073066_5.htm|newspaper=新华网|date=2011-02-14}}</ref>。”

曾任张闻天秘书的[[何方|何方]]则将张闻天与陈独秀、胡耀邦相提并论，称赞三人能够“坚持五四运动传统、弘扬科学民主精神”、“能够成为中国人民的先觉和前驱”，并分析三人在党内的惨淡结局均为“中国封建传统的强大”所造成<ref>{{cite journal|author=[[何方|何方]]|title=从陈独秀张闻天到胡耀邦|journal=《[[炎黄春秋|炎黄春秋]]》|year=2006|issue=06|pages=1-3|url=http://www.cnki.com.cn/Article/CJFDTOTAL-YHCQ200606001.htm|accessdate=2015-01-12}}</ref>。

对于张闻天的负面评价，除庐山会议后受政治因素的影响外，主要是在[[延安整风|延安整风]]时期。毛泽东在批判[[王明|王明]]在十年内战时期的左倾错误时，认为张有责任，且张对承认错误有抵触情绪。毛认为张闻天一事不懂，偏要人家依，不依则打；说张不顾全大局，无自我批评精神。张闻天对[[刘少奇|刘少奇]]在1937年批评中央的路线错误相当不满，这也让毛泽东很不高兴。毛认为，在中央仍然存在一些历史遗留问题需要清理，因此有延安整风运动的发动<ref>{{cite book|author=曹英|title=中共选择了毛泽东|year=2013|publisher=华文出版社|isbn=978-7-5075-3944-8|page=470-472}}</ref>。

== 家庭 ==
张闻天18岁时，在父母的包办下与卫月莲结婚。二人共有两个女儿：大女儿张维英和二女儿张引娣<ref name="a">{{cite web |url = http://dangshi.people.com.cn/GB/85038/11583851.html|title =外孙女讲述晚年张闻天：红色教授"不争权" 被囚523天|accessdate = 2010-05-13 |date = |publisher = 中国共产党新闻网 }}</ref>。1928年，张闻天在苏联认识了安娜·列昂尼多夫娜·古尔宾斯卡娅，并与之相爱{{r|程中原|page=80}}，二人育有一子。1931年张闻天回国后，古尔宾斯卡娅和她的儿子留在苏联，这个孩子于1941年死于战乱{{r|程中原|page=87}}。

张闻天后在中央苏区结识了时任少共中央局巡视员的刘英。红军长征到达陕北后，张闻天与[[刘英_(张闻天夫人)|刘英]]结婚，二人有一子张虹生<ref>{{cite web |url = http://news.eastday.com/epublish/gb/paper81/2/class008100002/hwz167500.htm|title =刘英眼中的张闻天|publisher = 文苑 }}</ref>。刘英曾任[[中华人民共和国外交部|中华人民共和国外交部]]部长助理、驻联合国经社理事会副代表，驻苏联大使馆参赞，外交部党委委员、部长助理兼人事司长，部监委书记等职务。文化大革命结束后，刘英先后当选全国政协委员、常委，中纪委委员等职务。2002年8月26日，刘英在北京逝世<ref>{{cite web |url = http://www.chinanews.com/2002-09-03/26/218688.html|title =张闻天夫人刘英因病在北京逝世 享年97岁|publisher = 中国新闻网|date=2002-09-03 }}</ref>。
*长女[[张维英|张维英]]<ref name="a"/>
*次女[[张引娣|张引娣]]：曾在[[外交部|外交部]]当打字员，后在上海一家自行车厂做普通职员<ref name="a"/>。
*独生子[[张虹生|张虹生]]：曾供职于[[新疆生产建设兵团|新疆生产建设兵团]]，1997年从[[南京大学|南京大学]]图书馆退休<ref name="a"/>。
*1961年，张闻天和刘英收养一孤儿，这个孤儿现居[[北京|北京]]<ref name="a"/>。
