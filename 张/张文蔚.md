'''张文蔚'''（{{bd|9世纪||908年|4月25日|catIdx=Z}}<ref name=ZZTJ266>[[:s:資治通鑑/卷266|《资治通鉴》卷二百六十六]]</ref>），字'''右华'''，[[唐朝|唐朝]]及其后继者[[后梁|后梁]][[官员|官员]]，唐朝末代皇帝[[唐哀帝|唐哀帝]]及后梁开国皇帝[[后梁太祖|后梁太祖]][[朱全忠|朱全忠]]统治时期为[[宰相|宰相]]。

== 唐朝年间 ==
=== 家世 ===
张文蔚生年不详。出身河间张氏，汉常山王[[张耳|张耳]]之后。祖父张君卿，为[[集贤院|集贤院]]正字。父[[张裼|张裼]]历任显职，官终[[天平军节度使|天平军节度使]]。张文蔚至少有四弟张济美、张贻宪、<ref name=NBT72>《新唐书》卷七十二{{cite web |url=http://www.sidneyluo.net/a/a17/072.htm |title=存档副本 |accessdate=2008-10-03 |0=2008-11-20 |deadurl=yes |archiveurl=https://web.archive.org/web/20081120085821/http://www.sidneyluo.net/a/a17/072.htm |archivedate=2008-11-20 }}{{cite web |url=http://www.sidneyluo.net/a/a17/table/form60.htm |title=存档副本 |accessdate=2008-09-01 |0=2010-06-20 |deadurl=yes |archiveurl=https://web.archive.org/web/20100620180412/http://sidneyluo.net/a/a17/table/form60.htm |archivedate=2010-06-20 }}</ref>张仁龟及其余一人，<ref>[[:s:舊五代史/卷18|《旧五代史》卷一十八]]注引《北梦琐言》</ref>与张济美、张贻宪、张仁龟都中了[[进士|进士]]，但只有张贻宪继续在朝为官；<ref name=BT178>[[:s:舊唐書/卷178|《旧唐书》卷一百七十八]]</ref>张济美早早得了精神病，张文蔚在继续仕途的同时照顾弟弟三十余年，为士人君子称道。张文蔚少年时就善作文，好学善交友，有佳士之称。<ref name=HFD18>[[:s:舊五代史/卷18|《旧五代史》卷一十八]]</ref>

=== 早期仕途 ===
张文蔚于[[唐僖宗|唐僖宗]][[乾符|乾符]]二年（875年）中进士。后任判盐铁巡官。<ref>《旧五代史·张文蔚传》称他在宰相[[裴坦|裴坦]]手下任巡官。但裴坦只在874年短暂拜相，在张文蔚中进士前，这不可能。见[[:s:資治通鑑/卷252|《资治通鉴》卷二百五十二]]。</ref>不久，在畿尉任上入馆为学士。<ref name=HFD18/>四年（877年），其父张裼去世，遂弃官归家守丧。<ref name=BT178/>

唐僖宗[[中和_(唐朝)|中和]]年间（881年—885年），因被由攻占[[长安|长安]]、自称大齐皇帝的[[黄巢|黄巢]]率领的农民大军所迫，僖宗被迫逃到[[成都|成都]]，<ref name=ZZTJ254>[[:s:資治通鑑/卷254|《资治通鉴》卷二百五十四]]</ref>为筹集对抗齐军的军费，将盐铁官署移到[[扬州|扬州]]，命[[李都|李都]]判盐铁。李都推荐张文蔚为转运巡官。[[光启|光启]]元年（885年），黄巢败，僖宗回京，<ref name=ZZTJ256>[[:s:資治通鑑/卷256|《资治通鉴》卷二百五十六]]</ref>张文蔚除[[监察御史|监察御史]]，后历任左补阙、殿中侍御史、起居舍人、司勋吏部员外郎，升司勋郎中、祠部郎中、知制诰，任上其母苏氏由扶风郡太夫人进封冯翊郡太夫人。<ref>[[:s:中书舍人苗深母琅琊郡太君王氏封琅琊郡太夫人祠部郎中知制诰张文蔚母扶风郡太夫人苏氏封冯翊郡太夫人等制|刘崇望《中书舍人苗深母琅琊郡太君王氏封琅琊郡太夫人祠部郎中知制诰张文蔚母扶风郡太夫人苏氏封冯翊郡太夫人等制》]]</ref>任满一年，授[[中书舍人|中书舍人]]。张文蔚与兄弟们在母亲膝下尽孝，不因官职而有别。其母死后，离职守丧，<ref name=HFD18/>退居洛阳，哀毁过人。<ref>《册府元龟》</ref>

张文蔚守丧完毕后，复拜中书舍人，不久召入翰林，为承旨学士。当时[[唐昭宗|唐昭宗]]刚回长安，<ref>唐昭宗在任内三次离开长安：[[乾宁|乾宁]]二年（895年）逃避军阀[[李茂贞|李茂贞]]和[[王行瑜|王行瑜]]的进攻威胁，同年返回；三年（896年）躲避李茂贞攻打，[[光化|光化]]元年（898年）返回；[[天复_(唐朝)|天复]]元年（901年）被宦官劫持到李茂贞的辖区凤翔军，三年（903年）返回。见[[:s:資治通鑑/卷260|《资治通鉴》卷二百六十]]、[[:s:資治通鑑/卷261|卷二百六十一]]、[[:s:資治通鑑/卷262|卷二百六十二]]、[[:s:資治通鑑/卷263|卷二百六十三]]。因柳璨于天祐元年（904年）被拜相时，张文蔚被称为翰林学士承旨，据《旧唐书·柳璨传》，此处指的是唐昭宗于天复三年（903年）从凤翔返回。见[[:s:舊唐書/卷179|《旧唐书》卷一百七十九]]。</ref>皇室衰微，张文蔚起草的诏令不失中正，受到称赞。昭宗称赞翰林学士[[柳璨|柳璨]]可任，想用他为宰相，问应该任以何官职。张文蔚认为选贤不需要拘泥资格、级别，但也不宜太快，并同意唐昭宗提拔柳璨为[[谏议大夫|谏议大夫]]，于是柳璨被任为谏议大夫、同平章事。<ref>《旧唐书》卷一百七十九</ref><ref>[[:s:新唐书/卷223下|《新唐书》卷二百二十三下]]</ref>张文蔚转户部侍郎，仍兼翰林学士承旨，不久出翰林院，为礼部侍郎。<ref name=HFD18/>

=== 拜相 ===
[[天祐_(唐朝)|天祐]]元年（904年），权势军阀[[宣武军节度使|宣武军节度使]][[朱全忠|朱全忠]]迫使昭宗从长安迁都[[洛阳|洛阳]]，<ref name=ZZTJ264>[[:s:資治通鑑/卷264|《资治通鉴》卷二百六十四]]</ref>同年弑昭宗，昭宗子[[唐哀帝|唐哀帝]]继立。二年（905年）三月，时任宰相的柳璨依附朱全忠，挑拨朱全忠和其他宰相[[裴枢|裴枢]]、[[独孤损|独孤损]]、[[崔远|崔远]]的关系；三人因而被罢相，时为正议大夫、尚书吏部侍郎（一作礼部侍郎）、[[上柱国|上柱国]]、赐紫金鱼袋的张文蔚和[[杨涉|杨涉]]被授[[同中书门下平章事|同中书门下平章事]]，拜为宰相。<ref>[[:s:新唐书/卷10|《新唐书》卷十]]</ref><ref>《册府元龟》作二月拜相。</ref>张文蔚还被任为[[中书侍郎|中书侍郎]]、监修国史、判度支。五月，有司定于十月行郊礼，张文蔚与杨涉奉命分判祭器，但郊礼最终未举行。<ref name=BT20-2>[[:s:旧唐书/卷20下|《旧唐书》卷二十下]]</ref>当月，朱全忠在柳璨和[[李振|李振]]煽动下杀死了包括裴枢、独孤损、崔远和前宰相[[陆扆|陆扆]]、[[王溥_(唐朝)|王溥]]在内的很多唐朝显官，柳璨起初还准备扩大清洗范围再处死十多名官员，但在张文蔚劝说下未遂。十二月，柳璨自己也被朱全忠所杀，<ref name=ZZTJ265>[[:s:資治通鑑/卷265|《资治通鉴》卷二百六十五]]</ref>张文蔚兼度支盐铁使。<ref name=HFD18/>

四年（907年）正月，张文蔚率百官向朱全忠劝进。<ref name=NHFD2>[[:s:新五代史/卷2|《新五代史》卷二]]</ref>三月，哀帝准备禅位于朱全忠，并下诏。张文蔚以持节[[银青光禄大夫|银青光禄大夫]]、守中书侍郎、同中书门下平章事摄[[中书令|中书令]]，充册礼使，押送传国宝、玉册、金宝，乘辂车与押传国宝使杨涉、押金宝使御史大夫[[薛贻矩|薛贻矩]]率百官备法驾由洛阳经[[郑州|郑州]]前往朱全忠所在的宣武军军部[[大梁|大梁]]以举行典礼。<ref>[[:s:禅位梁王诏|:s:禅位梁王诏]]</ref><ref>[[:s:禅位册文|:s:禅位册文]]</ref>四月，到大梁，从上源驿至金祥殿。朱全忠接受帝位，结束唐朝，建立了新的后梁朝，他就是太祖皇帝。<ref name=BT20-2/>张文蔚与册礼副使[[礼部尚书|礼部尚书]][[苏循|苏循]]奉册升殿进读。<ref name=NHFD35>[[:s:新五代史/卷35|《新五代史》卷三十五]]</ref>新帝为张文蔚等唐朝官员设宴于玄德殿，说自己辅政不久，登基都是诸公的功劳，包括张文蔚在内的绝大多数唐朝官员惭惧俯伏不能对，<ref name=ZZTJ266/><ref name=HFD3>[[:s:旧五代史/卷3|《旧五代史》卷三]]</ref>只是致谢。<ref>[[:s:旧五代史/卷60|《旧五代史》卷六十]]</ref>

== 后梁年间 ==
后梁代立后，五月，张文蔚仍以门下侍郎、同平章事居相位。后梁朝廷的规章制度多出其手。<ref name=NHFD35/>十月，太祖命张文蔚以下文武百官回[[长安|长安]]。<ref name=HFD3/>[[开平_(后梁)|开平]]二年（908年），太祖离开洛阳与拒绝承认后梁的晋王[[李存勖|李存勖]]作战，留张文蔚留守。三月，张文蔚暴疾卒于任上，<ref name=NHFD2/>赠右仆射。<ref name=ZZTJ266/>

== 评价 ==
*《旧五代史》史臣曰：'''文蔚'''、贻矩，皆唐朝之旧臣，遇梁室之强禅，奉君命以来使，狎神器以授之，逢时若斯，亦为臣者之不幸也。抑不为其相，不亦善乎！<ref name=HFD18/>
*《册府元龟》：沉遂重厚，有大臣之风。
*《新五代史》：呜呼！唐之亡也，贤人君子既与之共尽，其馀在者皆庸懦不肖、倾险狯猾、趋利卖国之徒也。不然，安能蒙耻忍辱于梁庭如此哉！……又读《梁实录》，见'''文蔚'''等所为如此，未尝不为之流涕也。……及唐之亡也，又先以朋党尽杀朝廷之士，而其馀存者，皆庸懦不肖倾险之人也，然后唐从而亡。<ref name=NHFD35/>
