'''冬瓜'''（学名：{{lang|la|''Benincasa pruriens''}}），又稱'''東瓜'''、'''白瓜'''、'''白東瓜皮'''、'''白冬瓜'''、'''白瓜皮'''、'''白瓜子'''、'''地芝'''，属[[葫芦目|葫芦目]][[葫芦科|葫芦科]]，是'''冬瓜属'''<ref>{{cite web|url=http://www.cvh.org.cn/dict/ |title=中国种子植物科属词典 |work=中国数字植物标本馆 |deadurl=yes |archiveurl=https://web.archive.org/web/20120411205135/http://www.cvh.org.cn/dict/ |archivedate=2012-04-11 }}</ref>的唯一物种{{r|theplantlist Benincasa}}。是一年生蔓性[[草本植物|草本植物]]。冬瓜原产中国南部及[[印度|印度]]，现在[[東亚|東亚]]和[[南亚|南亚]]地区广泛有栽培。

== 物產志 ==
冬瓜種植起源於中國大陸及東印度，並分佈于亞洲的熱帶、亞熱帶及溫帶地區。中國從秦漢時的《神農本草經》就有栽培記載，西元三世紀初張揖撰《廣雅•釋草》也有冬瓜的記載。《齊民要術》中記述了冬瓜的栽培及醬漬方法。日本在九世紀已有記錄。印度十六世紀有冬瓜記載。歐洲於十六世紀開始栽培。臺灣於十七世紀地方志已有栽種記載。美國於十九世紀由法國傳入。非洲於二十世紀七十年代以後由中國傳入。至二十一世紀冬瓜栽培仍以中國大陸、東南亞和印度等地為主。

== 形态 ==
冬瓜[[茎|茎]]上有茸毛；[[叶|叶]]子稍圆掌状浅裂，表面有毛；黄色花；圆形、扁圆或长圆形果实，大小因品种而不同，可以数斤到数十斤；冬瓜皮为绿色，多数品种成熟的果实有白粉，厚并且疏松多汁白色果肉，味淡；白色扁平[[种子|种子]]，有些具狭翼状边缘。公冬瓜的果實中間的空間較緊密，母的則比較鬆散。<ref name="性別">{{cite news|title=蔬菜性別大揭密！青椒、冬瓜、高麗菜分「公母」 |url=https://tube.chinatimes.com/20131116002320-261402 |language=zh-tw |author=梁嘉文、黃琲茹 |publisher=中時電子報 |date=2013-11-16 |accessdate=2018-12-10 }}</ref>

== 习性 ==
為什麼夏天所產的瓜，卻取名為冬瓜呢？這是因為瓜熟之際，表面上有一層白粉狀的東西，就像冬天的白霜，因此，冬瓜又稱白瓜。冬瓜虽然性喜温暖和濕潤的[[氣候|氣候]]，但亦可像[[笋瓜|笋瓜]]（winter squash）一樣過冬。冬瓜有一定抗旱能力。晚春播种，夏秋采收。

另有說法，冬瓜之名由來，是由於其保存期長，不切開可以保存至冬天而得名。<ref>{{ja}}{{cite web |url=http://kakaku.com/tv/channel=10/programID=55208/episodeID=1184982/ |title=#113 日本人の3割しか知らないこと_くりぃむしちゅーのハナタカ!優越館 2018年8月2日|author=<!--Staff writer(s); no by-line.--> |date=2018-08-02|website=|publisher=[[朝日電視台|朝日電視台]]|access-date=2018-08-09|quote=}}</ref>

== 用途 ==
冬瓜一般可以貯存十二個月之久。果實可生吃、炒吃或做湯。[[廣東|廣東]]名菜“[[冬瓜盅|冬瓜盅]]”，是把冬瓜掏空後，放入配料燉熬而成的菜。

它的[[果實|果實]]作[[蔬菜|蔬菜]]食用。未成熟的冬瓜果實表面有毛，因而稱爲'''毛瓜'''。果實成熟時表皮變得具有[[蠟|蠟質]]，因而利于久存。

除果實外，冬瓜的嫰芽、捲鬚和葉皆可作爲[[葉菜|葉菜]]食用。

[[File:Wintermelonsoup.jpg|thumb]]
'''冬瓜子'''（又名白瓜子、冬瓜仁、瓜子、瓜瓣、瓜犀）是冬瓜的种子，可入药，有清[[肺|肺]]化[[痰|痰]]的功效，具有[[免疫|免疫]]促进作用，但性微寒，因此[[脾胃|脾胃]]虚寒者禁用。<ref>[http://www.tcmlib.com/zy/html10/showdetail-3935392ce586ace7939ce5ad902c7a79.html 冬瓜子] {{webarchive|url=https://web.archive.org/web/20111212025843/http://www.tcmlib.com/zy/html10/showdetail-3935392ce586ace7939ce5ad902c7a79.html |date=2011-12-12 }}，意古中医·中医在线知识库</ref>

据《[[本草綱目|本草綱目]]》記載，冬瓜可治療[[消渴|消渴]]不止、[[浮腫|浮腫]]喘滿、[[痔瘡|痔瘡]]腫痛、[[熱毒|熱毒]]、[[痱子|痱子]]。

另外，冬瓜有一變種，稱為[[節瓜|節瓜]]，亦稱毛瓜，外形與未成熟的冬瓜相似。

==其他==
(1)台灣早期的結婚辦桌也會使用冬瓜盅，為冬瓜[[火鍋|火鍋]] ，當作鍋具，宴請喜客。<ref>[http://blog.xuite.net/cisters/wretch/102422395-%E7%A6%8F%E8%87%A8%E9%96%80,+%E7%A7%81%E6%88%BF%E8%8F%9C%E6%BB%BF%E6%BC%A2%E5%85%A8%E5%B8%AD 冬瓜盅] </ref>

(2)冬瓜可以當作食品原料，冬瓜果肉加糖熬煮可以做成台灣[[冬瓜茶|冬瓜茶]]的飲料與冬瓜糕餅餡料的台灣冬瓜[[鳳梨酥|鳳梨酥]]。 <ref>[http://blog.udn.com/jchen50/5468848 冬瓜食品] </ref>

(3)冬瓜的果實作蔬菜食用。冬天或夏天，甚至一年四季都可以入菜，或做成日式關東煮的食材，更可以將冬瓜切成冬瓜塊，當作湯底食材與火鍋食材。<ref>[http://yukikaku.pixnet.net/blog/post/107326351-%E3%80%90%E6%97%A5%E6%9C%AC%E2%80%A7%E9%9D%9C%E5%B2%A1%E3%80%91%E5%9C%A8%E9%9D%92%E8%91%89%E6%A9%AB%E4%B8%81%E5%93%81%E5%9A%90%E5%BA%B6%E6%B0%91%E7%BE%8E%E9%A3%9F%E3%80%8C%E9%9D%9C  日本冬瓜關東煮] </ref>

(4) 冬瓜有「好年冬」與甜蜜蜜的含意所以冬瓜糖條常出現在台灣結婚喜慶的喜糖與喜餅中，帶有吉利興旺之意。<ref>[http://www.twwiki.com/wiki/%E5%96%9C%E7%B3%96 台灣傳統結婚] </ref>
