'''冬季會議'''（{{lang-en|Winter Meetings}}）是一個於每年12月舉行的活動。[[美國職棒大聯盟|美國職棒大聯盟]]30支球隊和其[[美國職棒小聯盟|小聯盟]]所屬球隊的代表集會四天，討論聯盟事務和休賽期交易，也有其他棒球相關活動如商業展覽和就業諮詢等。包括聯盟行政人員、各隊老闆、總經理、球隊球探、來自他國的參觀者、參加展覽者和希望進入美國職棒組織工作的求職者等人參與。{{sfn|Peden|2011|p=195}}<ref>{{cite web |url=http://www.milb.com/milb/events/wintermeetings.jsp|title=2014 Baseball Winter Meetings returns to San Diego after three decades|publisher=Minor League Baseball|year=2014|accessdate=18 January 2015}}</ref><ref>{{cite web |url=http://www.utsandiego.com/news/2014/dec/09/major-league-baseball-winter-meetings-job-fair/|title=Baseball Jobs: Will work for peanuts|first=Jonathan|last=Horn|date=December 9, 2014|accessdate=18 January 2015|work=U-T San Diego}}</ref>會議最後一天，沒有被列入球隊40人名單的球員可在[[規則五選秀|規則五選秀]]中被其他球隊選中。<ref>{{cite web |url=http://www.mlbtraderumors.com/2014/12/2014-rule-5-draft-results.html|title=2014 Rule 5 Draft Results|first=Zach|last=Links|date=11 December 2014|accessdate=11 December 2014|work=MLB Trade Rumors}}</ref>

2015年，第114屆冬季會議於12月7至11日在[[納許維爾|納許維爾]]進行。<ref name=sun>{{cite web |url=http://www.torontosun.com/2015/12/10/blue-jays-snag-giants-pitcher-in-rule-v-draft|title=Blue Jays snag Giants pitcher in Rule V draft|first=Bob|last=Elliott|work=Toronto Sun|date=10 December 2015|accessdate=12 December 2015}}</ref>2016年冬季會議則選在12月4至8日、[[馬里蘭州|馬里蘭州]]的國家港灣。<ref>{{cite web |url=https://www.washingtonpost.com/news/sports/wp/2015/11/23/baseballs-winter-meetings-coming-to-washington/|title=Baseball's winter meetings coming to — Washington?|first=Barry|last=Svriuga|date=23 November 2015|accessdate=12 December 2015|work=[[The_Washington_Post|The Washington Post]]}}</ref>

==歷史==
在休賽期間舉辦會議的傳統可追溯至[[國家聯盟|國家聯盟]]成立初年——1876年。當次會議，{{link-en|威廉·赫伯特|William Hulbert}}被選為國家聯盟主席，而[[紐約同好|紐約同好]]和費城運動家因為無法完成預訂賽程而從聯盟除名。<ref>{{cite web |url=http://sabr.org/bioproj/topic/league-alliance|title=The League Alliance|first=Brock|last=Helander|accessdate=8 February 2017|work=SABR}}</ref>1901年，冬季會議成為年度活動。1967年，由於[[墨西哥聯盟|墨西哥聯盟]]從2A層級升至3A，冬季會議選在[[墨西哥城|墨西哥城]]進行，為首次於美國境外舉辦的冬季會議。<ref>{{cite web |url=https://sports.vice.com/en_us/article/a-brief-history-of-baseball-winter-meeting-madness|title=A Brief History of Baseball Winter Meeting Madness|first=Jack|last=Moore|date=12 December 2014|accessdate=9 February 2017|work=Vice Sports}}</ref>

==出席者==
冬季會議吸引數以千計的出席者。官方評估大約有3000人參與2014年冬季會議，這包含球隊老闆、總經理、總教練、球探、球員經紀人、律師、會計師還有各國觀光客。{{sfn|Peden|2011|p=195}}{{sfn|Solomon|Freeman|2012|p=185}}{{sfn|Gorman|2007|pp=7-8}} 一般而言，球員不會出席，但是自由球員通常會前來向各隊介紹自己。{{sfn|Solomon|Freeman|2012|p=185}}在2014年的聖地牙哥冬季會議，約有750位媒體人領有[[記者證|記者證]]。<ref name=wsj/>

晚上會有大聯盟球隊為其小聯盟附屬球隊所舉辦的歡迎會。{{sfn|Solomon|Freeman|2012|p=182}} 亦有為總教練和棒球記者準備的午餐會。<ref name=nyt/>

==球員交易和簽約==
[[File:Bill_Veeck_1964.JPG|thumb]]
[[File:BarryBonds1993.jpg|150px]]的[[貝瑞·邦茲|貝瑞·邦茲]]]]
在球隊高層齊聚一堂之下，冬季會議成為典型的球員交易和自由球員簽約的場所。{{sfn|Carroll|2007|p=78}}不過，直到20世紀末期在飯店大廳進行的非正式會議已經被簡訊和電子郵件取代。因為眾多的媒體人員和球迷的聚集在現場，大部分的高層交流改至較隱蔽的套房內。<ref name=wsj>{{cite web |url=http://www.wsj.com/articles/mlb-winter-meetings-with-fewer-meetings-1418139567|title=Baseball's Winter Meetings—Minus the Meetings|first=Brian|last=Costa|date=9 December 2014|accessdate=18 January 2015|work=[[The_Wall_Street_Journal|The Wall Street Journal]]}}</ref><ref name=nyt/> 

下列是在過去的冬季會議中較值得注意的交易和合約:
<!--請添加較為重要的項目，而不要任意列舉-->
*在1975年羅德岱堡的冬季會議，[[芝加哥白襪|芝加哥白襪隊]]的新老闆{{link-en|比爾·維克|Bill Veeck}}坐在一張飯店大廳內的桌子旁，前方有標示寫著「開放交易」(Open for Business)。會議期間他一共談成了6筆交易案，牽涉22名球員。<ref>{{cite web |url=http://m.mlb.com/news/article/158887650/the-best-winter-meetings-trades-of-all-time/|title=The biggest Winter Meetings trades of all time|first=AJ|last=Cassavell|year=2016|accessdate=28 January 2017|work=MLB.com}}</ref>
*於1988年亞特蘭大的冬季會議，[[德克薩斯遊騎兵_(棒球)|德州遊騎兵]]商定了牽涉15名球員的3筆交易，並簽下自由球員投手[[諾蘭·萊恩|諾蘭·萊恩]]。{{sfn|Sypher|1990|p=100}}
*在1992年路易維爾的冬季會議，首度成為自由球員的[[貝瑞·邦茲|貝瑞·邦茲]]和[[舊金山巨人|舊金山巨人]]談妥一紙6年4300萬美元的合約。其中由邦茲親自洽談有關客場比賽期間能自由處置飯店套房的部分。<ref name=nyt>{{cite web |url=http://www.nytimes.com/2014/12/08/sports/baseball/where-baseball-is-all-the-buzz-general-managers-just-want-peace-and-quiet.html?_r=0|title=Baseball's Annual Winter Meetings Have It All, Except Quietude|first=David|last=Waldstein|date=7 December 2014|accessdate=18 January 2015|work=[[The_New_York_Times|The New York Times]]}}</ref>
*[[艾力士·羅德里奎茲|艾力士·羅德里奎茲]]在2000年冬季會議和[[西雅圖水手|西雅圖水手]]談成一筆價值2億5200萬美元、為期10年的合約。<ref>{{cite web |url=http://m.mlb.com/news/article/210165898/the-biggest-deals-in-winter-meetings-history/|title=Biggest free-agent deals in Winter Meetings history
|first=John|last=Schlegel|date=3 December 2016|accessdate=10 February 2017|work=[[USA_Today|USA Today]]}}</ref>
*2011年達拉斯冬季會議的最後一日，當年秋天隨[[聖路易紅雀|聖路易紅雀]]贏下[[世界大賽|世界大賽]]戒指的[[亞伯特·普荷斯|亞伯特·普荷斯]]與[[洛杉磯天使|洛杉磯天使]]敲定一份10年2億5000萬美元的合約。<ref>{{cite web |url=http://espn.go.com/mlb/hotstove11/story/_/id/7330869/los-angeles-angels-shock-world-land-albert-pujols-cj-wilson|title=Angels Shock the Baseball World|first=Jayson|last=Stark|date=9 December 2011|accessdate=18 January 2015|work=[[ESPN|ESPN]]}}</ref>
*在2014年聖地牙哥的冬季會議，經過24小時內許多的談判，[[洛杉磯道奇|洛杉磯道奇]]完成了和4支隊伍的6筆交易，關係到19位選手和1位自由球員。<ref>{{cite web |url=http://www.usatoday.com/story/sports/mlb/2014/12/11/mlb-winter-meetings-trades-dodgers-max-scherzer/20278411/|title='Aggressive' winter meetings end with many winners|first=Bob|last=Nightengale|date=12 December 2014|accessdate=18 January 2015|work=[[USA_Today|USA Today]]}}</ref><ref>{{cite web |url=http://www.ocregister.com/articles/dodgers-644986-team-year.html|title=Kemp trade is latest piece of Dodgers' reconstruction|date=11 December 2014|accessdate=18 January 2015|first=Bill|last=Plunkett|work=Orange County Register}}</ref>

==其他活動==
[[File:Pants-rowland.jpg|150px]]
一場大約300個攤位的商業展覽亦在冬季會議期間進行，涵蓋球具、棒球推廣和其他服務。{{sfn|Solomon|Freeman|2012|p=185}}另外一個年度活動則是職業棒球人才招募會（Professional Baseball Employment Opportunities Job Fair, PBEO Job Fair），剛投入職場的求職者選擇在此尋找工作，接受小聯盟組織的現場面試。<ref>{{cite web |url=http://www.huffingtonpost.com/chad-barr/class-is-in-session-at-th_b_2244114.html|title=Class is in Session at the Baseball Winter Meetings|first1=Chad|last1=Barr|first2=Ted|last2=Curtis|date=12 May 2012|accessdate=18 January 2015|work=[[The_Huffington_Post|The Huffington Post]]}}</ref>每年大概有400至500人獲得工作，因此12月被形容為棒球界僱用的高峰。<ref>{{cite web |url=http://www.kpbs.org/news/2014/dec/08/job-fair-baseballs-winter-meetings/|title=Baseball Executives, Jobseekers In San Diego For Winter Meetings|first=Dwane|last=Brown|date=8 December 2014|work=KPBS}}</ref>

自1951年起，「{{link-en|棒球之王|King of Baseball}}」頭銜在冬季會議的晚宴會被頒予一位小聯盟老將。<ref>{{cite web |url=http://www.baseball-almanac.com/awards/kingofbaseballaward.shtml|title=King of Baseball Award by Minor League Baseball|publisher=Baseball Almanac|year=2014|accessdate=18 January 2015}}</ref>

另有幾個和[[美國國家棒球名人堂暨博物館|名人堂]]有關的活動：
* 取代{{link-en|老將委員會|Veterans Committee}}（現在唯一一個選出退休已久的球員和非現役人員的選舉機關）的選舉團體召集、投票。<ref name=nyt/>
* 頒予為棒球廣播有傑出貢獻人士的{{link-en|福特·弗立克獎|Ford C. Frick Award}}得主公布。<ref>{{cite web |url=http://baseballhall.org/awards/ford-c-frick|title=Ford C. Frick Award|publisher=National Baseball Hall of Fame|accessdate=11 December 2014}}</ref>
* [[全美棒球記者協會|全美棒球記者協會]]宣布優秀棒球新聞工作者的{{link-en|泰勒·史賓克獎|J. G. Taylor Spink Award}}受獎人{{sfn|Geltner|2012|p=203}}和舉行其年度會議。<ref name=nyt/>弗立克獎與史賓克獎皆為名人堂年度入會儀式的一環。
* 年度球探獎亦於此進行。<ref name=sun/>{{sfn|Sandoval|Nowlin|2011|p=165}}
