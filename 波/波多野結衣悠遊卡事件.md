'''波多野結衣悠遊卡事件'''，簡稱'''波卡事件'''，是指2015年8月至9月間，[[臺灣|臺灣]]電子票證業者[[悠遊卡公司|悠遊卡公司]]发行以[[日本|日本]][[AV女優|AV女優]][[波多野結衣|波多野結衣]]為[[悠遊卡|悠遊卡]]封面的悠遊卡，由此造成的一系列[[社会|社会]]和[[政治|政治]]事件。此款悠遊卡分別推出波多野結衣身穿黑衣的「[[惡魔|惡魔]]造型」、以及身穿白衣的「[[天使|天使]]造型」兩種版本，而悠遊卡銷售的收入將會捐贈給[[慈善機構|慈善機構]]。然而悠遊卡公司決定使用波多野結衣形像的消息傳出後，立刻引起[[臺灣|臺灣]][[輿論|輿論]]及[[新聞媒體|新聞媒體]]的熱烈討論。主要意見在於考慮到波多野結衣在日本從事[[AV女優|AV女優]]這一特殊行業，使得將其照片作為悠遊卡封面有許多爭論之處。

在面臨許多反對意見後，悠遊卡公司曾經一度考慮撤回這批具有爭議的[[悠遊卡|悠遊卡]]。但是在8月28日經過公司內部開會討論後，悠遊卡公司決定繼續推出印有波多野結衣照片的悠遊卡。不過由於原本的「[[天使|天使]]造型」版本悠遊卡被民眾質疑直接使用[[日本成人影片|AV]]的封面照片，使得悠遊卡公司決定另外推出新的「天使造型」版本，並延後至9月中旬發行。而原本的「惡魔造型」以及遭到替換的「天使造型」悠遊卡，則仍然按照計畫於9月1日推出。

== 背景與發想 ==
[[File:波多野結衣_(1).jpg|thumb]][[波多野結衣|波多野結衣]]。]]

[[臺北市|臺北市]][[悠遊卡公司|悠遊卡公司]]過去為了推廣可以在[[臺北捷運|臺北捷運]]和[[臺北市市區公車|臺北市市區公車]]等處使用的[[悠遊卡|悠遊卡]]，每年便會與[[運動員|運動員]]、[[藝人|藝人]]和[[政客|政治人物]]合作推出約60款肖像悠遊卡促銷。不過儘管悠遊卡的發卡量已經超過5,400萬張，但不少民眾基於收藏理由仍然會購買紀念套卡<ref>{{Cite web |url = http://www.cna.com.tw/news/aloc/201508260163-1.aspx |title = 波多野結衣拍攝名人悠遊卡 走清新風 |author = 游凱翔 |date = 2015年8月26日 |publisher = [[中央通訊社|中央通訊社]] |language = zh-hant |accessdate = 2015年8月29日 |quote =  }}</ref>。而在2009年至2014年期間，悠遊卡公司還與許多[[公益團體|公益團體]]合作推出公益悠遊卡，藉由[[孩童|孩童]]的畫作與圖片為卡片封面，以呼應公益主題。隨著獲得政府政策上的支持，悠遊卡公司在2014年的營業收入超過新臺幣15億元，稅後盈餘則有近1.5億元。

2015年3月，[[臺北市市長|臺北市市長]][[柯文哲|柯文哲]]要求悠遊卡公司董事長[[戴季全|戴季全]]在1年內，將悠遊卡的交易金額從原先的499億翻倍到1000億。不過與此同時，[[一卡通票證公司|一卡通票證公司]]發行的[[一卡通_(臺灣)|一卡通]]計畫在9月1日可以在臺北捷運與[[臺北市公共自行車租賃系統|臺北市公共自行車租賃系統]]平臺上使用，[[高雄捷運|高雄捷運]]則是計畫在2015年7月全數設置成多卡通單一閘門<ref>{{Cite web |url = http://www.cna.com.tw/news/ahel/201508250205-1.aspx |title = 一卡通暢行北捷　藍白拖展台味 |author = 王淑芬 |date = 2015年8月25日 |publisher = [[中央通訊社|中央通訊社]] |language = zh-hant |accessdate = 2015年8月29日 |quote =  }}</ref>。

其中一卡通票證公司亦在7月時採取[[產學合作|產學合作]]的方式，由學生設計融合一卡通的經營理念的虛擬代言人物[[魔法少女小帕|魔法少女小帕]]<ref>{{Cite web |url = http://www.cna.com.tw/news/ahel/201508260394-1.aspx |title = 小帕戰女優 魔法少女代言一卡通 |author = 王淑芬 |date = 2015年8月26日 |publisher = [[中央通訊社|中央通訊社]] |language = zh-hant |accessdate = 2015年8月29日 |quote =  }}</ref>。

在此背景下，[[悠遊卡公司|悠遊卡公司]]員工則在8月時提案邀請[[日本|日本]][[AV女優|AV女優]][[波多野結衣|波多野結衣]]製作名人悠遊卡，希望藉此能夠增加悠遊卡銷售量。在這之前，波多野結衣剛出道不久便因為相貌與著名模特兒[[林志玲|林志玲]]相似，而被許多[[臺灣|臺灣]]民眾熟知並且將其稱呼為「暗黑林志玲」。而代理[[波多野結衣|波多野結衣]]的商品[[東櫻創意行銷有限公司|東櫻創意行銷有限公司]]已經在[[中華民國|中華民國]]經營長達4年，該[[公司|公司]]也訂定希望能夠去除[[波多野結衣|波多野結衣]][[AV女優|AV女優]]標籤，而逐漸轉型成為具有清新形象的[[藝人|藝人]]、[[歌星|歌星]]和[[演員|演員]]作為目標。對此東櫻創意行銷有限公司認為波多野結衣是個性單純且有才幹的人員，而她也的確因為這一特點而能夠吸引臺灣民眾注意，對於悠遊卡公司提出的公益悠遊卡合作計畫則是感到感謝。

[[波多野結衣|波多野結衣]]過去在[[日本|日本]]是不聞名的二線[[AV女優|AV女優]]，2011年3月發生[[2011年日本東北地方太平洋近海地震|日本東北地方太平洋近海地震]]時，當地[[媒體|媒體]]則一度大幅報導她在海邊拍[[色情片|色情片]]時失蹤而溺斃<ref name = "一劍浣春秋這麼看">{{Cite web |url = http://www.cna.com.tw/news/firstnews/201508265026-1.aspx |title = 波多野結衣悠遊卡 一劍浣春秋這麼看 |author = 游凱翔 |date = 2015年8月26日 |publisher = [[中央通訊社|中央通訊社]] |language = zh-hant |accessdate = 2015年8月29日 |quote =  }}</ref>。2011年9月29日，波多野結衣與[[大槻響|大槻響]]受邀前來臺灣舉辦感謝祭，表達感謝臺灣對於[[臺灣對東日本大震災之援助及各界反應|東日本大震災]]的幫助，在華文媒體報導下使得她很快獲得極高人氣。之後在相貌美麗以及宣傳效應推廣，使得波多野結衣的人氣在亞洲地區不斷增加。不少粉絲對於其個人形象抱持正面看法，而其在臺灣的知名度效益更超過許多重要明星，她還曾經為臺灣遊戲公司[[艾玩天地|艾玩天地]]代言<ref>{{Cite web |url = http://www.nownews.com/n/2015/08/27/1795750 |title = 波多野結衣悠遊卡惹爭議　台灣最具知名度傳奇AV女優！ |author =  |date = 2015年8月27日 |publisher = [[今日新聞網|今日新聞網]] |language = zh-hant |accessdate = 2015年8月29日 |quote =  }}</ref>。之後波多野結衣一方面在日本陸陸續續拍攝了1,200多部色情片，並且贏得2014年[[成人廣播獎|成人廣播獎]]年度最佳女優獎；另一方面在臺灣則積極轉型成藝人，並與[[李康生|李康生]]共同主演電影《[[沙西米_(電影)|沙西米]]》<ref>{{Cite web |url = http://www.nownews.com/n/2015/08/27/1795901 |title = 除了爆乳外，你所不知道波多野結衣私下的日系甜美穿搭風 |author = 賴鵬翔 |date = 2015年8月27日 |publisher = [[今日新聞網|今日新聞網]] |language = zh-hant |accessdate = 2015年8月29日 |quote =  }}</ref>。波多野結衣本人在得知這份工作機會後，則表示對於自己有機會回報臺灣的朋友感到期待與開心<ref>{{Cite web |url = http://www.nownews.com/n/2015/08/27/1795742 |title = 悠遊卡惹議　波多野發聲明：AV女優就不能參與公益嗎？ |author = 黃靜宜 |date = 2015年8月27日 |publisher = [[今日新聞網|今日新聞網]] |language = zh-hant |accessdate = 2015年8月29日 |quote =  }}</ref>。

== 照片爭議 ==
===[[肖像|肖像]]授權===

[[戴季全|戴季全]]表示一開始並不認識[[波多野結衣|波多野結衣]]，直到注意到「暗黑[[林志玲|林志玲]]」稱號後才知道兩者為同一人。而關於此案戴季全則表示其底線為符合[[臺灣|臺灣]][[法律|法律]]，不過他本人考量到波多野結衣在[[臺灣|臺灣]]以及[[網際網路|網際網路]]上有著極高人氣和廣大知名度，以及其人形象清新且唯美，因此認為這樣的悠遊卡商品能夠獲得社會大眾的關注<ref>{{Cite web |url = http://www.cna.com.tw/news/firstnews/201508275006-1.aspx |title = 波多野結衣悠遊卡惹議 擬限購買年齡 |author = 游凱翔 |date = 2015年8月27日 |publisher = [[中央通訊社|中央通訊社]] |language = zh-hant |accessdate = 2015年8月29日 |quote =  }}</ref>。

因此[[悠遊卡公司|悠遊卡公司]]主動向代理[[波多野結衣|波多野結衣]]的商品[[東櫻|東櫻]]創意行銷有限公司邀約，在與波多野結衣[[經紀人|經紀人]][[呂季樺|呂季樺]]商談公益企畫案時，悠遊卡公司主要是朝向經紀公司授權[[肖像|肖像]]照片給悠遊卡公司銷售發行的方向進行討論。最後[[經紀公司|經紀公司]]安排波多野結衣在6月前往[[臺灣|臺灣]]，以拍攝悠遊卡所要使用的照片。

其中由臺灣[[攝影師|攝影師]]拍攝4個系列主題，這之中便包括有讓[[波多野結衣|波多野結衣]]穿著[[白色|白色]]和[[黑色|黑色]]的低胸與露背[[洋裝|洋裝]]的[[天使|天使]]和[[魔鬼|魔鬼]]主題造型之照片，並且把[[毛片|毛片]]提供給悠遊卡公司挑選。但由於[[悠遊卡公司|悠遊卡公司]]對於在[[臺灣|臺灣]]拍攝的[[天使系列|天使系列]]作品中並沒有青睞的照片，而只在[[惡魔|惡魔]]造型中找到理想版本的照片，因此東櫻創意行銷有限公司則向[[日本|日本]][[經紀公司|經紀公司]]請求重新授權。日本[[經紀公司|經紀公司]]方面則依照全新且未曝光的要求，決定提供不曾在[[日本|日本]]以外的[[亞洲|亞洲]]地區授權發表過的圖像<ref>{{Cite web |url = http://www.cna.com.tw/news/firstnews/201508275004-1.aspx |title = 波多野結衣悠遊卡爭議 經紀人鞠躬道歉 |author = 游凱翔 |date = 2015年8月27日 |publisher = [[中央通訊社|中央通訊社]] |language = zh-hant |accessdate = 2015年8月29日 |quote =  }}</ref>。最後選中廣受歡迎的系列[[色情片|色情片]]作品《甜蜜中出同居生活·波多野結衣》封面照片，作為為悠遊卡的使用圖像挑選。

[[東櫻|東櫻]]創意行銷[[有限公司|有限公司]]在認定該張照片可以使用後，便將檔案提供給[[悠遊卡公司|悠遊卡公司]]。[[悠遊卡公司|悠遊卡公司]]以此照片設計成2款悠遊卡，並且個別推出1萬5千張卡片，計畫在9月1日透過[[網際網路|網際網路]]、[[便利商店|便利商店]]等[[通路|通路]]販售。

不過在過去[[名人|名人]][[悠遊卡|悠遊卡]]通常一款不會發行超過5,000張，而[[悠遊卡公司|悠遊卡公司]]內部評估能賣到上萬張者僅有樂團[[五月天|五月天]]或歌手[[周杰倫|周杰倫]]等少數[[藝人|藝人]]；而在這系列悠遊卡發行前，只有[[王建民_(棒球選手)|王建民]]在最著名的時候曾發行過1萬5千套[[悠遊卡|悠遊卡]]，但是當時採取分成三個時期、而個別發行5,000套不同版本的做法<ref>{{Cite web |url = http://udn.com/news/story/9/1149189-%E6%88%B4%E5%AD%A3%E5%85%A8%E7%A8%B1%E4%B8%8D%E8%AD%98%E6%B3%A2%E5%A4%9A%E9%87%8E%E7%B5%90%E8%A1%A3-%E8%AD%B0%E5%93%A1%E6%89%B9%E5%94%AC%E7%88%9B |title = 戴季全稱不識波多野結衣 議員批唬爛  |date = 2015年8月27日 |publisher = [[中央通訊社|中央通訊社]] |language = zh-hant |accessdate = 2015年8月27日 |quote =  }}</ref>。

同時[[悠遊卡公司|悠遊卡公司]]還定調此次販售的悠遊卡為[[公益|公益]]性質<ref>{{Cite web |url = http://www.cna.com.tw/news/amov/201508270365-1.aspx |title = AV女優參與公益 日本已行之有年 |author = 楊明珠 |date = 2015年8月27日 |publisher = [[中央通訊社|中央通訊社]] |language = zh-hant |accessdate = 2015年8月29日 |quote =  }}</ref>，卡片的販售盈餘在扣除必要的管銷費用後會捐給[[台灣兒童暨家庭扶助基金會|台灣兒童暨家庭扶助基金會]]、[[靖娟兒童安全文教基金會|靖娟兒童安全文教基金會]]以及[[行無礙資源推廣協會|行無礙資源推廣協會]]3個公益團體<ref>{{Cite web |url = http://www.nownews.com/n/2015/08/26/1795053 |title = 找波多野結衣代言　悠遊卡公司：看中她的形象清新、唯美 |author = 黃靜宜 |date = 2015年8月26日 |publisher = [[今日新聞網|今日新聞網]] |language = zh-hant |accessdate = 2015年8月29日 |quote =  }}</ref>。

=== 照片爭議 ===
[[File:柯文哲_(15972701052).jpg|thumb]]在接受採訪時，表示此事全部交由悠遊卡公司處理。]]
