'''[[波蘭|波蘭]]'''是位於[[中歐|中歐]]的一個國家，地處[[德國|德國]]以東。波蘭地勢平坦，國土大部都是平原。波蘭[[波羅的海|波羅的海]]沿岸地區除[[格但斯克|格但斯克]]外，缺乏天然良港。位於西北部的[[什切青|什切青]]亦是波蘭重要的港口。波蘭東北部的湖區林木茂密，人口較少。在南部，[[喀爾巴阡山脈|喀爾巴阡山脈]]構成了波蘭同[[捷克|捷克]]、[[斯洛伐克|斯洛伐克]]和[[烏克蘭|烏克蘭]]的國界。

== 地形 ==
波蘭國土的平均標高僅有173米。最高峰是位於波蘭同斯洛伐克邊境上[[塔特拉山脈|塔特拉山脈]]中的[[雷瑟峰|雷瑟峰]]，標高2,499米。[[格但斯克灣|格但斯克灣]]沿岸約60平方公里的陸地在海平面以下。

== 水系 ==
波蘭最大的兩條河流是[[維斯瓦河|維斯瓦河]]和[[奧得河|奧得河]]。維斯瓦河的流域面積占去波蘭一半的國土。

== 氣候 ==
波蘭全國的年均降水量大約在600毫米左右。但一些山區中的觀測站測得的年降水量可達1300毫米。總的來說，南部的山區地帶降水量較北部平原為高。
