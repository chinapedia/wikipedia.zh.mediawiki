'''波旁王朝'''（{{lang-fr|Maison de Bourbon}}）是一個在[[歐洲|歐洲]]歷史上曾斷斷續續地統治[[納瓦拉|納瓦拉]]（1555－1848）、[[法國|法國]]（1589－1792、1814－1830）、[[西班牙|西班牙]]（1700－1808、1813－1868、1875－1931、1975年至今）、[[那不勒斯|那不勒斯]]與[[西西里|西西里]]（1734-1816）、[[盧森堡|盧森堡]]（1964年至今）等國和[[意大利|意大利]]若干[[公國|公國]]的跨國王朝。由於其父系祖先為[[卡佩王朝|卡佩王室]]成員，因此亦稱'''卡佩王朝波旁分支'''。

值得一提的是，波旁王室的近代成員都以[[保守|保守]]和[[反動|反動]]著稱，因此在[[美国英语|美式英語]]中，'''波旁'''一字成為對極端[[保守主義者|保守主義者]]的稱呼<ref>《牛津當代大辭典》，旺文社，1991年版 ISBN 957-508-009-2</ref>。波旁王朝的君主堪稱是历史上最为多灾多难的群体：[[亨利四世_(法兰西)|亨利四世]]遭暗杀身亡，[[路易十三|路易十三]]42岁便英年早逝，[[路易十六|路易十六]]最终在一場大革命中被送上断头台。

{{法国历史}}

== 起源 ==
波旁王室，顧名思義便是起緣於[[法國|法國]]中部的[[波旁_(行省)|波旁]]地區。而這個[[采邑|采邑]]最早出現於13世紀初，是法王一位[[家臣|家臣]]，也即是今天波旁王室成員的母系祖先的私人封地。而其成員的父系祖先可追溯到來自[[卡佩王朝|卡佩王室]]（House of Capet, 987年－1328年）的法王[[路易九世_(法蘭西)|路易九世]]。路易九世之子克莱蒙伯爵[[法蘭西的羅貝爾|羅貝爾]]與其子孫通過他和波旁领地的女继承人[[勃艮第的比阿特丽斯|勃艮第的比阿特丽斯]]的婚姻，獲得了對波旁公國的統治權。他們二人的長子[[路易一世·德·波旁|路易]]在1327年獲封為[[波旁公爵|波旁公爵]]。自此以後，他的子孫以封國波旁為姓，因此這次受封被視為波旁王室的起源。

雖然波旁的主系家族在1523年，因其公爵[[夏爾三世·德·波旁|查理三世]]涉嫌叛國，而被褫奪爵位，失去領地。但其波旁家族分支經過二百多年的繁衍，亦已獲得不少其他爵位。而當中的拉馬爾什-旺多姆分支（La Marche-Vendôme），成為了日後波旁王室的主軸。

== 起家於纳瓦爾 ==
自1523年起，由於出現繼承人問題的爭議，波旁公國的爵位一直空缺，其資產暫時歸屬法國王室。1527年，法王[[弗朗索瓦一世|弗朗索瓦一世]]将波旁公國以及公爵爵位授予旺多姆公爵查理。自此，拉馬爾什-旺多姆分支獲得了波旁公國，而這個公國也是波旁家族日後奪得法國王位的重要資本。

查理的兒子，第9代波旁公爵[[安托萬·德·波旁|安托萬·德·波旁]]於1548年10月12日與納瓦拉女王儲[[胡安娜三世_(纳瓦拉)|让娜·达布雷特]]結婚。他們的長子[[亨利四世_(法兰西)|亨利·德·波旁]]在1572年母親去世後繼承了納瓦拉王國，是為[[納瓦拉|納瓦拉]]國王[[恩里克三世|恩里克三世]]，自此波旁家族入主了這個王國。雖然此時的納瓦拉的疆域只是全盛時期的一小半，但足以成為日後波旁家族爭奪法國王位的重要基地。

== 入主法蘭西 ==
[[File:Augustins_-_Henri_IV,_roi_de_France_et_de_Navarre_-_Jacques_Boulbène.jpg|thumb]]
[[File:Ivryrubens.jpg|300px]]的画作：《伊夫里之战中的[[亨利四世_(法兰西)|亨利四世]]》。[[伊夫里之战|伊夫里之战]]是[[法国宗教战争|法国宗教战争]]中的一个重要战役；此役的胜利将信新教的亨利四世推上了王位，波旁王朝随即开始。]]

亨利·德·波旁跟隨身為[[納瓦拉王國|納瓦拉王國]]女王的母親信奉[[新教|新教]][[克爾文宗|克爾文宗]]的[[胡格諾派|胡格諾派]]，母親過世後，他繼位為[[納瓦拉|納瓦拉]]国王，由於他同時為法國貴族的身分，就成為了法國新教胡格諾派的領袖，而波旁家族的其他分支和主導法國政壇的[[吉斯家族|吉斯家族]]信奉[[天主教|天主教]]，亨利無可避免地被捲入了這場残酷的[[法国宗教战争|法国宗教战争]]。為求自保，亨利數次宣稱[[改宗|改宗]][[天主教|天主教]]，但私底下依舊保持自己對胡格諾派的忠誠。

宗教戰爭不但造成大量法國平民傷亡，也間接導致了[[昂古莱姆王朝|瓦卢瓦-昂古莱姆家族]]的人丁凋零。[[昂古莱姆王朝|昂古莱姆王朝]]的第二代君主[[亨利二世_(法兰西)|亨利二世]]的三个兒子都成為了法国国王，而當中[[弗朗索瓦二世|弗朗索瓦二世]]和[[查理九世_(法兰西)|查理九世]]皆在青年早逝。他們的弟弟[[亨利三世_(法兰西)|亨利三世]]繼承了王位，而他們最小的弟弟，[[阿朗松|阿朗松]]公爵[[弗朗索瓦|弗朗索瓦]]成為王储。1584年，弗朗索瓦王儲年少病逝，按照[[萨利克继承法|萨利克继承法]]法國王位只可傳予[[雨果·卡佩|雨果·卡佩]]男性系后裔中的男子，而波旁家族就是第一顺位家族。

波旁家族的另一個分支，布賽特分支（Bourbon-Busset）與亨利所屬的拉馬爾什-旺多姆分支相比，更親近於當權的瓦卢瓦-昂古莱姆家族；只是布賽特家族的祖先，因婚姻未獲法王[[路易十一|路易十一]]的批准，而被取消王位繼承資格。亨利·德·波旁由此成為法国王位的合法继承人，但是他具有的[[胡格諾派|胡格諾派]]背景，導致很多法國天主教的不滿，將他软禁在法国宫廷，接受王室的庇护。

1589年，[[亨利三世|亨利三世]]遇刺身亡后，亨利·德·波旁順利即位为法国国王，為[[亨利四世|亨利四世]]，納瓦拉王國亦同時併入法國。此後，納瓦拉國王成了續任法王眾多頭銜的其中一個。

[[亨利四世|亨利四世]]成了法王之後，主動结束了困扰法国多年的宗教战争，令法國的經濟得以復甦。1598年，亨利四世颁布了[[南特敕令|南特敕令]]，施行宗教寬容政策，令很多信奉[[基督新教|基督新教]]的手工業者留在法國，促進了法國經濟的繁榮。但亨利四世的[[宗教寬容|宗教寬容]]政策，隨著他的駕崩而被終止施行。

== 爭奪西班牙 ==
[[File:Felipe_V;_Rey_de_España.jpg|thumb]]
{{See also|西班牙波旁王朝}}
1700年，[[西班牙哈布斯堡王朝|西班牙哈布斯堡王朝]]最後的男嗣[[卡洛斯二世|卡洛斯二世]]的去世，结束了[[哈布斯堡王室|哈布斯堡王室]]對西班牙歷5代的統治。卡洛斯二世生前在群臣逼迫之下，立下遺詔，把[[王位|王位]]传给了他的外甥[[腓力五世_(西班牙)|安茹公爵腓力]]，腓力是[[波旁王室|波旁王室]]的成員，也是法國國王[[路易十四|路易十四]]之次孙，而非[[奧地利哈布斯堡王朝|奧地利哈布斯堡王朝]]的[[查理六世_(神聖羅馬帝國)|查理大公]]。

路易十四隨即將次孫推上西班牙王位，是為西班牙王腓力五世。而路易十四捍衛孫兒繼承權的強硬立場，導致[[西班牙王位繼承戰爭|西班牙王位繼承戰爭]]於1702年爆發。

戰事持續到1711年，此時波旁王室已在西班牙站穩，且表示願意為[[英國|英]]、[[荷蘭|荷]]兩國商人提供貿易優惠和保障，加上[[神聖羅馬帝國|神聖羅馬帝國]][[約瑟夫一世_(神圣罗马帝国)|約瑟夫一世]]去世，查理大公登基為[[神聖羅馬皇帝|神聖羅馬皇帝]]，[[列強|列強]]為了[[歐洲|歐洲]]的均衡發展，倒戈支持腓力，哈布斯堡聯軍瓦解。

1714年，西奧兩國簽署和約，[[奧地利哈布斯堡王室|奧地利哈布斯堡王室]]承認波旁王室入主西班牙，但腓力五世及其後代必須放棄對法國的[[王位|王位]]繼承權，[[西班牙帝國|西班牙帝國]]與[[法蘭西帝國|法蘭西帝國]]永遠不可以擁戴同一個[[君主|君主]]。自此，西法兩國波旁王室正式分家，也揭開了波旁王室對西班牙斷斷續續統治三百餘載的首頁。

== 法國大革命 ==
{{See also|法国大革命}}
法國的宗教寬容時期隨著[[亨利四世|亨利四世]]的駕崩而逝去，隨後的波旁君主恢復了對基督新教徒的壓迫，導致大量信奉[[胡格諾派|胡格諾派]]的手工業者離開法國，加上王室生活揮霍，國家連年用兵，導致法國經濟逐漸走下坡。而隨經濟衰退而來的還有社會矛盾，稅制不公、政府腐敗和[[羅馬天主教|羅馬天主教]]會特權等因素都增加了法國社會不穩定的因素。而[[美國獨立戰爭|美國獨立戰爭]]引起民主思潮和法國國內的[[啟蒙運動|啟蒙運動]]思潮，成為了法國大革命的催化劑。

1789年，由於法國王室的財政問題，法王[[路易十六|路易十六]]被迫重開三級議會，但增稅的要求遭到否決，導致法資產階級和王室的武裝鬥爭開始。初時[[資產階級|資產階級]]代表僅是將國家體制由[[君主專制|君主專制]]轉為[[君主立憲制|君主立憲制]]，但亦遭到歐洲列強的反對，繼而藉機入侵法國。而此時路易十六的王后[[玛丽·安托瓦內特|玛丽·安托瓦內特]]涉嫌向娘家[[奧地利|奧地利]]通風報信，惹來極大民憤。波旁王朝終於在1792年8月10日被巴黎民眾的起義推翻。最終，路易十六在1793年1月21日成為法國歷史上唯一一位被送上断头台的君主。

在路易十六被處決後，部分保王黨人士宣稱其長子，王太子路易‧查理為法王[[路易十七|路易十七]]。但那時法國已為共和國，這宣稱也無實際效用。而路易十七也沒熬過法國大革命，於1795年在獄中死去，得年10歲。此後，路易十六的弟弟，流亡[[俄羅斯帝國|俄羅斯帝國]]的[[路易十八|普羅旺斯伯爵路易]]成為王位繼承人。

== 王朝復辟時期 ==
{{main|波旁復辟}}
[[File:Pavillon_royal_de_France.svg|thumb]]
法國大革命期間不健全的政府制度，使[[拿破崙|拿破崙]]這個野心家奪權稱帝。但拿破崙的霸業並不能維持很久。1814年，拿破崙倒台，在[[反法同盟|反法同盟]]和法國國內拿破崙軍中的新貴的奉迎下，流亡于[[英國|英國]]的普羅旺斯伯爵返國即位為法王[[路易十八|路易十八]]。不過在軍隊和資產階級的壓力下，路易十八被迫接受一部傾向自由主義的憲法。

但封建王朝的好景不長，拿破崙於1815年3月殺回巴黎，重建帝國，立[[百日皇朝|百日皇朝]]，[[路易十八|路易十八]]落荒而逃。在[[滑鐵盧戰役|滑鐵盧戰役]]過後，路易十八才得以復位。而他和他的弟弟[[查理十世_(法兰西)|查理十世]]一直致力維護保王黨在國會的勢力。而一些在法國大革命期間的政策，亦因被認為傾向於[[自由主義|自由主義]]而被兩人廢除。然而當時的共和革命觀念深入人心，這些舉動令波旁王朝漸失民心，最終導致1830年[[七月革命|七月革命]]的爆發。

== 波旁-奧爾良王朝 ==
{{See also|七月王朝}}
[[查理十世|查理十世]]保守的政策在1830年終於成為了革命的導火線。在那一年法王[[查理十世|查理十世]]试图推行镇压法令，因此使得國民叛變，史稱「七月革命」。革命過後，[[法國國會|法國國會]]讓[[路易·菲利普|奧爾良公爵路易·菲利普三世]]以法王[[路易十三|路易十三]]的次子[[菲利普一世_(奥尔良公爵)|菲利普]]第八代孫的身分繼承王位，正式稱號為「法蘭西人的國王」[[路易·菲利普|路易·菲利普]]，稱為[[七月王朝|七月王朝]]。由於他以{{link-en|波旁-奧爾良|House of Orléans}}為姓氏，因此他的[[七月王朝|七月王朝]]亦被稱為[[波旁-奧爾良王朝|波旁-奧爾良王朝]]。

[[路易·菲利普|路易·菲利普]]采取中间路线，但依然無法解決法國經濟持續衰退的問題。1846年的工农业萧条，引致發生[[1848年二月革命|1848年二月革命]]，令他被迫退位。自此以後，沒有任何一位[[波旁王室|波旁王室]]或其分支成員能再次統治[[法國|法國]]，法國波旁王朝自始正式落下帷幕。

== 家族現況 ==
1830年革命後，查理十世被迫遜位。但擁護他的[[保王黨|保王黨]]人拒絕承認這個決定，繼續奉他和他的子孫為法王。而路易·菲利普在1848年退位後，他的支持者也繼續擁護他和他的子孫為法王。因此，法國的保王黨分裂兩個支持不同覦位王子的派系，一個是支持查理十世和其後代的正统派（Légitimisme）和支持路易·菲利普一系的奧爾良派（Orléanisme）。

而查理十世的孫子[[尚博伯爵|尚博伯爵]][[亨利_(尚博伯爵)|亨利]]去世，根據法國基於[[薩利克繼承法|薩利克繼承法]]的傳位法則，王位繼承權應首先給予与尚博伯爵血缘最近的同姓堂亲，諷刺的是这个角色却是當時奧爾良派的首領[[巴黎伯爵|巴黎伯爵]][[路易-菲利普·艾伯特·德·波旁|菲利普]]。正统派支持者並不想和奧爾良派合流，因此不惜否認當年西班牙国王腓力五世放棄對法國王位繼承權決定的合法性，擁立伯爵在西班牙的堂親，西班牙王位卡洛斯派首領[[莫提遜伯爵|莫提遜伯爵]][[胡安·卡洛斯·玛利亚·伊斯杜拉|胡安·卡洛斯·玛利亚·伊斯杜拉]]為他們的首領。

由於莫提遜伯爵對法国王位的繼承合法性備受質疑，不少正统派的保王黨人轉而支持奧爾良派，再加上莫提遜伯爵的曾孫[[圣海梅公爵|圣海梅公爵]][[阿方索·卡洛斯|阿方索·卡洛斯]]在1936年無嗣而亡，不受歡迎的西班牙国王[[阿方索十三世|阿方索十三世]]成為正统派領袖，令奧爾良派成為波旁保王黨的主流。

正统派今日的首領是[[安茹公爵|安茹公爵]][[路易·阿方索_(安茹公爵)|路易斯·阿方索]]，他是西班牙波旁王朝的[[阿方索十三世|阿方索十三世]]的次子[[塞戈维亚公爵|塞戈维亚公爵]][[海梅·德·波旁|海梅·德·波旁]]的长孙。而奧爾良派的首領巴黎伯爵[[昂利·菲利普|昂利·菲利普]]，是法王路易·菲利普的第六代子孫。
