'''波斯湾'''（{{lang-fa|خلیج فارس}}），[[印度洋|印度洋]][[阿拉伯海|阿拉伯海]]西北海湾。位于[[阿拉伯半岛|阿拉伯半岛]]與[[伊朗高原|伊朗高原]]之間，简称'''海湾'''。

南通过[[霍尔木兹海峡|霍尔木兹海峡]]与[[阿曼湾|阿曼湾]]相连，总面积约23.3万平方公里，長990公里，寬58-338公里；平均深約30米，最深約100米；盐度38-40‰。它是[[底格里斯河|底格-{里}-斯河]]与[[幼发拉底河|幼发拉底河]]出海的地方。北至東北至東方與[[伊朗|伊朗]]相鄰，西北為[[伊拉克|伊拉克]]和[[科威特|科威特]]，西到西南方為[[沙烏地阿拉伯|沙烏地阿拉伯]]、[[巴林|巴林]]、[[卡塔爾|卡塔爾]]、[[阿拉伯聯合大公國|阿拉伯聯合大公國]]、[[阿曼|阿曼]]。

==名称之争==
长期以来，关于对海湾的叫法[[波斯灣名稱爭議|一直存在争议]]，伊朗把海湾称为“波斯湾”，而阿拉伯国家将其称为“阿拉伯湾”（{{lang-ar|الخليج العربي}}）。<ref> [http://www.chinadaily.com.cn/hqgj/2010-02/24/content_9493805.htm  “海湾”称谓之争再度升级 伊朗发出禁飞威胁] </ref><ref> [http://news.163.com/41224/0/18C4IV300001121Q.html   波斯湾不是阿拉伯湾 海湾称呼之争又起伊朗敏感] </ref><ref> [http://news.xinhuanet.com/world/2007-01/03/content_5561238.htm   “阿拉伯湾”还是“波斯湾”？] </ref>

== 历史 ==
波斯湾的名稱源于[[波斯|波斯]]，最先由1世纪的地理学家[[斯特拉波|斯特拉波]]命名。

== 气候 ==
波斯湾属[[热带沙漠气候|热带沙漠气候]]，气温较高，常年在20度左右，夏季可达32度以上。西北風強，將沙漠地區的沙土吹入灣中，海水渾濁。夏季有[[沙尘暴|沙尘暴]]和[[霾|霾]]，秋季有[[颮|颮]]和[[龍捲風|龍捲風]]。

== 经济 ==
该地方拥有全世界最丰富的[[石油|石油]]资源，儲量約占全球63%。

波斯湾內珊瑚礁遍佈，漁產豐富；但因臨近國家逐漸工業化，環境污染日益嚴重。
