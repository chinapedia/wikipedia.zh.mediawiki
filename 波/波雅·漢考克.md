'''波雅·漢考克'''是[[日本動漫|日本動漫]]作品《[[ONE_PIECE|ONE PIECE]]》裡的其中一個角色，外號「'''蛇姬'''」、「'''海賊女帝'''」，是[[亞馬遜百合_(ONE_PIECE)|亞馬遜百合]]的皇帝和九蛇海賊團的船長，亦為[[世界政府_(ONE_PIECE)|世界政府]]公認的海賊「[[王下七武海|王下七武海]]」當中唯一的女性，因為不易受週遭事物所影響而擁有「不動的女帝」之稱號。由於女帝每次遠征會搶奪大量財寶及[[石化_(虚构作品)|石化]]大量海賊，因此被世界政府視為「七武海中的關鍵人物」；她常常發動無差別攻擊，[[海軍_(ONE_PIECE)|海軍]]視被石化的海兵只是冰山一角，而放任女帝的所作所為。

== 設計 ==
波雅的日文是ボア。ボア就是BOA（[[蚺|蚺]]），是一種無毒的大型蛇類。漢考克這個名字很有可能是來自「[[橡膠|橡膠]]工業之父」Thomas Hancock。

最初設定時的漢考克臉上有[[刺青|刺青]]，稱號為「妖姬」。

== 人物 ==
波雅·漢考克外貌特色是[[姬髮式|姬髮式]]的黑色長髮、深藍色的雙眼、高挑身材和J罩杯的[[巨乳|巨乳]]，佩戴蛇型[[耳環|耳環]]，經常身穿九蛇民族的花紋長裙裝（偶爾會穿著附有絨毛衣領的披風），代表「飛天龍之蹄」的烙印痕跡位於背部，寵物是條名叫薩洛梅的蟒蛇。習慣以「妾身」自稱，九蛇島人民稱她「蛇姬大人」。

漢考克平時性格高傲自負，鄙視他人過頭；就連待在她行經路上的小動物，也被她直接用腳踢開。但她對感情相當直率，面對魯夫直接說中意對方，會仰首插腰指著對方，[[口頭禪|口頭禪]]「是的，因為妾身太美了」。因為她的美貌，屬下也被迷得神魂顛倒（但在[[魯夫|魯夫]]面前無效），不管做什麼都會被原諒。戀上魯夫後，更常顯現柔性姿態。擁有霸王色的[[霸氣_(ONE_PIECE)|霸氣]]。另外，波雅·漢考克在第一部劇情的實際年齡為29歲<small><ref name="ONEPIECE_58_SBS"/></small>，為當時最年輕的七武海成員，後由於[[托拉法爾加·羅|托拉法爾加·羅]]於頂點戰爭結束兩年後成為新任七武海，遂變成第2年輕的七武海成員，但多雷斯羅薩事件結束後，托拉法爾加·羅的七武海稱號被褫奪，故現在回復為最年輕的七武海成員。

== 劇中表現 ==
前傳第零話中，22年前的漢考克與兩位妹妹曾經短暫出場。當時年幼的漢考克認為在人人都可以成為海賊的「大海賊時代」，不變成更強的戰士的話，是進不了九蛇海賊團的<ref>《ONE PIECE》漫畫第0話。</ref>。

漢考克自稱曾在一次冒險的最後，和另外兩個妹妹打倒了中樞之海的怪物「蛇髮魔女」，但三人也因此受到詛咒，背上長出了巨大的眼睛，一旦脫下衣服露出眼睛，附近的人都會被石化。所以在漢考克洗澡的兩個時辰之內都會擺下護陣，任何人禁止入城。實際上這只是為了掩飾背後[[奴隸|奴隸]][[烙印|烙印]]的說詞。

後來她告訴[[魯夫|魯夫]]，12歲的時候漢考克和兩個妹妹桑塔索妮雅、瑪莉哥德被[[人口販子|人口販子]]從九蛇海賊團綁架，當作奴隸賣給世界貴族天龍人，自此以後背部被烙下奴隸的印記，在淪為奴隸的期間，漢考克因為天龍人的一時興起而被迫吃下惡魔果實，成為「迷戀果實」能力者<ref>《ONE PIECE》漫畫第53卷第521話。</ref>。

4年後一位名叫費雪·泰格的冒險家徒手攀登上紅土大陸，為了解放被當作奴隸而慘遭虐待的魚人們，來到天龍人的住地聖地·馬力喬亞。他不顧任何種族的隔閡，解放了所有奴隸，而當時漢考克就是被他拯救的奴隸之一。後來泰格與[[世界政府_(ONE_PIECE)|世界政府]]為敵，並成立了太陽海賊團，讓之前身為奴隸的魚人們出航至外海，由於奴隸的烙印尚在，因此他將太陽海賊團的象徵覆印在其上作為掩飾。對於漢考克來說，費雪·泰格就如她們三姊妹的恩人。

之後當時的九蛇島女皇嘉蘭百合協助漢考克三姊妹回國，並將三姊妹視如己出<ref>《ONE PIECE》漫畫第521話。</ref>，當時[[席爾巴斯·雷利|席爾巴斯·雷利]]與夏克雅克也曾參與了漢考克三姐妹的救援行動<ref>《ONE PIECE》漫畫第591話。</ref>。她於11年前（當時年齡判斷為18歲）僅僅一場遠征中，以最小年紀被懸賞8,000萬貝里，加上九蛇歷代的海賊團都惡名昭彰、唯恐天下，所以中樞馬上就提防她，並提議徵召她成為「[[王下七武海|王下七武海]]」<ref>《ONE PIECE》漫畫第522話。</ref>。

由於長久以來被天龍人支配，漢考克為飾掩自己的過去，女帝把自己的心冰封，以冷傲的姿態面對別人並時時放出輕微的霸王色霸氣，讓人不敢接近，以保護自己。在漫畫516話初登場時便以冷傲的姿態，向[[海軍_(ONE_PIECE)|海軍]]中將飛鼠表態「既不想被剝奪七武海名號，又討厭世界政府不想聽令」，除將海軍軍艦上的所有士兵（飛鼠利用小刀刺傷自己的手以集中注意力因而避免中招）石化外，還掠奪了軍艦上的貨物。

當她回到九蛇宮殿的時候，見到金雀花將孩童們特地依照她的形象、所塑造的人形雕偶帶進宮殿，在收到那時露出一絲暖笑，然後馬上收起這種情感，回復讓人難以接近的姿態（一抺讓人心寒的冷笑），一手將雕像推倒在地上以至粉碎，並且在嘉蘭百合勸諫她參與海軍徵招令時，以「即使造成國家滅亡，也會得到大家的原諒，因為妾身美若天仙」為由，將後者直接拋出窗外，這是因為女帝的恐懼（想起過往的痛苦陰影），再也不想給任何人支配。

=== 第一部 ===
當漢考克在亞馬遜百合的宮殿入浴時，[[魯夫|魯夫]]為了逃避護國戰士們的追趕，不慎踏破屋簷瓦片，墜落在浴池當中，並且見到她背上的印記。由於亞馬遜百合的男人禁令與漢考克三姊妹對印記的忌諱，她們將魯夫帶到決鬥場執行「武武」極刑。

魯夫先是戰勝了處刑黑豹，隨後3位護國戰士雛菊、雅菲蘭朵拉、香豌豆為了幫魯夫辯護，遭到漢考克石化，魯夫在震驚之虞，與漢考克的兩位妹妹展開決鬥並取得勝利。其中漢考克的妹妹波雅·桑塔索妮雅落敗後懸掛在決鬥場邊緣，由於戰鬥中被小妹波雅·瑪莉哥德身上的火炎波及，使她衣服被燒毀之虞差點露出了印記，魯夫卻在此刻攀爬到她背上，以身軀掩蓋印記，此舉讓漢考克與九蛇島人民對魯夫徹底改觀。

事後漢考克在魯夫的求情下，解除了3位護國戰士的石化，並且在召見魯夫同時坦言三姊妹的過去，魯夫誠懇的心打動了漢考克，她也在此時戀上魯夫，魯夫也因此得以接受九蛇島人民的款待。她因為戀上魯夫而得了[[相思病|相思病]]，在魯夫的懇求下，終於答應協助魯夫，同時也和[[海軍_(ONE_PIECE)|海軍]]要求必須先前往海底監獄「[[推進城|推進城]]」，才肯接受[[王下七武海|王下七武海]]召集。

隨後她帶著[[魯夫|魯夫]]藏在她的大衣內，偷偷地乘上了飛鼠的海軍軍艦前往海底監獄「推進城」拯救被囚禁的「火拳」[[波特卡斯·D·艾斯|艾斯]]。在接受「推進城」副看守所長多米諾檢查時發動果實能力，讓魯夫趁機進入LEVEL 1「紅蓮地獄」（動畫版搭乘推進城電梯時，還曾經因故動手教訓時任副署長般若拔），後來到達艾斯被監禁的樓層，向他傳達魯夫即將到來的消息，之後便搭乘軍艦前往海軍總部「馬林福特」，並在艾斯處刑現場與其他五位「[[王下七武海|王下七武海]]」協同[[海軍_(ONE_PIECE)|海軍]]以三位上將為首的所有戰力與[[白鬍子海賊團|白鬍子海賊團]]開戰，但卻對戰場上的所有男人不分敵我一視同仁的全部視為敵人，並還特地提到「那個人」（魯夫）除外。

在魯夫遭「白獵人」[[斯摩格_(ONE_PIECE)|斯摩格]]攻擊之際出手相救，並且使出了[[霸氣_(ONE_PIECE)|武裝色霸氣]]。甚至偷偷地將艾斯的手銬鑰匙交與魯夫，魯夫感激得擁抱她，使她因為開心而全身癱軟，被海軍誤以為是被魯夫以「折腰」擊倒了她。之後持續阻止斯摩格，讓魯夫前往處刑台，並且攔在和平主義者面前援護魯夫。

在魯夫被「死亡外科醫師」[[托拉法爾加·羅|托拉法爾加·羅]]帶離現場後，使用迷戀果實的能力擄獲一艘海軍艦，將艦上海兵全數變成石像，並命令所飼養的蟒蛇薩洛梅追蹤潛水艇追上羅等人，並將魯夫帶至無風地帶的九蛇海賊大本營「女人島」療養。魯夫昏迷的兩星期間，漢考克顯得十分焦急，甚至沒吃過任何東西；得知魯夫甦醒時，她竟不理會眾人制止，直命下人準備食物讓她親自送過去。結果意外的和當年的救命恩人「冥王」[[席爾巴斯·雷利|席爾巴斯·雷利]]再次相會。

得知魯夫必須在魯斯卡伊那島展開修練時，漢考克本希望每天親自替魯夫送飯菜，但雷利認為太過安逸的環境會讓魯夫無法成長，因而回拒她的請求。漢考克一開始對於雷利設下的條件感到相當不滿，但後來得知雷利是為了幫助魯夫成長時，她告訴雷利她會為了體諒魯夫，願意在魯夫修行的這段期間內忍耐。

=== 第二部 ===
兩年後的漢考克依然迷戀著魯夫，她跟著雛菊、桑塔索妮雅、瑪莉哥德以及嘉蘭百合替魯夫送行，還趁魯夫離開以前提供大量的食物和好幾年份的日常用品，讓魯夫可以在旅途中享用，並希望自己能成為理想中的好妻子。魯夫雖然很感激漢考克為他準備食物，但對於要結婚一事則是明確的拒絕了（雖說如此但她依然深愛著魯夫，而且也沒打消結婚的念頭）。

之後漢考克告訴魯夫，由於草帽海賊團在這兩年之間消聲匿跡的緣故，因此大多數的人都認為魯夫已經死了。於是她特別為魯夫準備一件毛皮斗篷與假鼻子，還提醒魯夫千萬不能引發騷動，否則將會無法出航（動畫版漢考克為了避免外人對她和魯夫之間的關聯產生質疑，與九蛇海賊團則將魯夫送到夏波地諸島附近的海上，之後魯夫便自行搭著九蛇海賊團準備的小船抵達夏波帝諸島）。

當海軍艦隊準備捉拿魯夫等人的時候，漢考克率領九蛇海賊團為[[草帽海賊團|草帽海賊團]]阻擋海軍的攻勢。事後海軍雖然對漢考克掩護草帽海賊團的行為感到不滿，卻還是被漢考克迷得神魂顛倒。

多雷斯羅薩事件落幕後，漢考克閱報獲知魯夫等人賞金再度提升，[[唐吉訶德·多佛朗明哥|唐吉訶德·多佛朗明哥]]被魯夫和羅的聯盟擊潰的消息，還特地將魯夫的照片放大貼在九蛇島的宮殿牆外以茲紀念<ref>扉頁連載《來自世界的甲板，五億的男人篇》 VOL.22</ref>。

== 能力 ==
=== 惡魔果實 ===
漢考克為超人系「[[惡魔果實|迷戀果實]]」（{{lang|ja|メロメロの実}}）的能力者。能夠射出粉紅色的光線或是心型物質石化目標。能力者在使用能力時必須要讓身體擺出一些特別的動作才能發射出光線或是粉紅色的心型物質，例如讓雙手擺出愛心的姿勢的「迷戀甘風」以及輕點嘴唇發出飛吻的「手槍之吻」和「虜之箭」。能力甚至能夠讓碰到自己的對手石化<ref>《ONE PIECE》漫畫556話</ref>，用強力的腿技石化踢碎敵人，例如「芳香腳」，因此推測只要是能力者看見的目標就能夠讓目標石化。

順帶一提，由於漢考克發出的粉紅色光線跟「遲緩果實」所發出的桃紅色「遲緩光線」非常類似，因此還曾經一度被魯夫認錯<ref>《ONE PIECE》漫畫518話</ref>。<br>能力者能夠自由控制粉紅色射線以及心型物質所發射出去的範圍大小，可以一次石化整艘船上的人，也能夠集中威力射擊一人。<br>除此之外，能力者的美色和姿態也會影響能力的發動，敵人越被能力者的美貌打動，影響注意力以及感官力，產生的石化威力便會越佳。

但一般的石化能力對於不好美色之人<ref>《ONE PIECE》漫畫518話</ref>（例如魯夫）、注意力不在自己身上的人無效<ref>《ONE PIECE》漫畫516話</ref>（海軍中將飛鼠曾以小刀刺傷自己的手，讓痛覺覆蓋住對於美色的注意力才能避免中招）；另外，盲人的眼睛看不見，所以不會被石化（海賊王特別篇3D X 2Y）。雖然以上的三種對手能夠讓迷戀果實的一般石化能力無效，但仍可以讓其進行「強制石化」或是「部分石化」，強制石化所能夠生效的目標包括了人類以及其他具有生命的物質，以及無生命的物質（例如石頭、砲彈或是機械）。<br>被石化的目標其外表顏色會變成猶如石頭般的闇灰色，且在石化的狀態內會暫時性失去知覺，但不影響其生命機能。

而且只有迷戀果實能力者能夠讓被石化的對手恢復原狀，解除石化的方式為迷戀果實的能力者向被石化的目標吹出自己的飛吻心型物質。如果被石化的目標處於「石化」的期間不受到任何損傷（一旦石化的身體碎裂該生物即會死亡），在石化狀態解除後，被石化的目標身體便不會有任何損傷，但會失去在石化期間的所有記憶。
