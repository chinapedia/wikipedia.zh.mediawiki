'''波浪號'''（tilde、'''~'''）是一個有許多用途的[[標點符號|標點符號]]。原本它作為[[縮寫|縮寫]]符號的一個字母，但亦有作為[[變音符號|變音符號]]或單一文字的用途。在[[數學|數學]]上，它是代表[[等價關係|等價關係]]的[[數學符號|數學符號]]。在最後一個用途裡（尤其是在[[辭書學|辭書學]]裡），它有時會被當做'''代字號'''。

== 變音符號 ==
在語言裡，波浪號是放在[[字母|字母]]上的一變音符音，用來表示發音的改變，如[[鼻音化|鼻音化]]。

在[[希臘語|希臘語]]的[[多聲調拼寫法|多聲調拼寫法]]裡，波浪號為[[抑揚符號|抑揚符號]]的變體。

它被當做縮寫來使用於中古[[拉丁語|拉丁語]]的文書上。當「n」或「m」跟在一母音後時，它常常會被忽略掉，而將波浪號放在前一母音上面來表示此一忽略字母。這是波浪號被用來表示鼻音化的起源。直到17世紀為止，用以忽略「n」或「m」的母音上之波浪號持續被用在[[法語|法語]]出版書籍上以縮短文章的長度。波浪號偶爾亦被使用在其他的縮寫上，如字母「q」上來強調單字''que''（那）。

在[[葡萄牙語|葡萄牙語]]和一些[[美洲原住民語言|美洲原住民語言]]{{le|圖皮瓜拉尼語言|Tupi–Guarani languages}}如[[瓜拉尼語|瓜拉尼語]]、{{le|恩嘎圖語|Nheengatu}}等語言內，波浪號標示一[[鼻化元音|鼻化元音]]。另外，图皮语诸语言的[[瓜拉尼語|瓜拉尼語]]還有使用[[G̃|G̃]]這個字母。

在西班牙語和其他如[[加里西亞語|加里西亞語]]、[[巴斯克語|巴斯克語]]和[[巴拉圭語|巴拉圭語]]等文化上被西班牙影響的語言裡，''ñ''是一單獨字母，表示一[[硬顎鼻音|硬顎鼻音]]（[[國際音標|IPA]] {{IPA|[ɲ]}}）。在西班牙語裡，它原本寫做''nn''。

在[[愛沙尼亞語|愛沙尼亞語]]裡，''[[õ|õ]]''是一單獨字母，表示一單獨母音。

在[[越南語|越南語]]裡，母音上的波浪號表示一下沉[[聲調|聲調]]——'''跌聲'''。

在[[IPA|IPA]]裡，在一符號上的波浪號表示其[[鼻音化|鼻音化]]，在一符号下的则表示吱嘎音，而在一符號中的則表示其{{le|軟顎音化|Velarization}}。例如，法國城市Lyon的發音為{{IPA|[ljɔ̃]}}，及澳式英語中「cool」的發音為{{IPA|[kʊ:ɫ]}}。

== 數學符號 ==
在數學上，它可以代表：
*等價關係；
*渐近相等，如<math>f(x) \sim g(x)</math>表示<math>\lim_{x \to \infty} \frac{f(x)}{g(x)} = 1</math>；
*“服从某概率分布”，如<math>\ X \sim N(\mu, \sigma^2)</math>表示“X服从[[期望值|期望值]]为<math>\mu</math>、[[方差|方差]]为<math>\sigma^2</math>的[[正态分布|正态分布]]”；
*[[傅里叶变换|傅里叶变换]]。
*幾何學的[[相似|相似]]

==非正规用途==
1.波浪号（{{Unicode|~}}，{{Unicode|～}}）的一个非正规（错误）用途就是用来代替当连接号的[[连接号#类似符号|浪纹线]]（{{Unicode|〜}}，{{Lang|zh|〜}}）。<br>
2.社群聊天中，多數人喜歡在語句後方加上『～』用以讓語句更加柔順。
