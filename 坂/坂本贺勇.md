'''坂本賀勇'''（{{bd|1959年|7月23日|||catIdx=}}）是一名日本的电子游戏设计师，目前担任[[任天堂|任天堂]]的[[任天堂企划制作本部|企划制作本部]]总管，此前为[[任天堂企划开发本部|企划开发本部]]总管之一。<ref>{{Cite web|date=2016-07-01|url=http://www.1101.com/nintendo/miitomo2016/index.html |title=Miitomoのポテンシャル！坂本賀勇×糸井重里 |publisher=1101.com |accessdate=2016-07-01}}</ref><ref>{{Cite web |date=2012-12-07 |url=http://www.nintendo.co.jp/wiiu/interview/hardware/vol10/index2.html |title=社長が訊く『Wii U』 Nintendo Land 篇 2. 機能からのデザイン |publisher=任天堂 |accessdate=2015-08-09}}</ref>他因为与[[宫本茂|宫本茂]]等人共同创造了《[[银河战士系列|银河战士]]》系列而初露头角，并在之后负责了系列后续作品；这一系列在赢得优秀的口碑的同时也让他获得了一定声誉。

== 简历 ==

坂本贺勇于1959年7月23日出生在日本[[奈良县|奈良县]]，1982年他从[[大阪藝術大學|大阪藝術大學]]设计系毕业并进入[[任天堂|任天堂]]工作，加入了当时的[[任天堂開發第一部|任天堂開發第一部]]。他在任天堂的第一份工作是为[[Game_&_Watch|Game & Watch]]版的《大金刚》和街机版的《大金刚Jr》设计像素美术。<ref name="wired">{{cite web |url=http://www.wired.com/gamelife/2010/04/sakamoto/ |archiveurl=https://web.archive.org/web/20100425001558/http://www.wired.com/gamelife/2010/04/sakamoto/ |archivedate=2010-04-25 |title=Q&A: Metroid Creator’s Early 8-Bit Days at Nintendo |date=2010-04-07 |author=Kohler, Chris |work=[[连线|Wired: GameLife]] |publisher=[[Condé_Nast_Digital|Condé Nast Digital]] |accessdate=2015-08-09 |dead-url=no }}</ref> 之后他的工作转向了[[FC游戏机|FC游戏机]]，他参与了《[[拆屋工|拆屋工]]》、《[[气球大战|气球大战]]》和《{{le|Gumshoe|Gumshoe (video game)}}》等游戏的设计工作。<ref name="wired" />在这之后他以“Yamamoto”和“Shikamoto”的假名与与[[宫本茂|宫本茂]]等人共同创造了《[[银河战士系列|银河战士]]》系列，并作为角色设计师设计出了[[萨姆斯·艾仁|萨姆斯·艾仁]]等角色。<ref>{{cite web |url=http://press.nintendo.com/GDC2010/downloads/bio_yoshiosakamoto.zip |archiveurl=https://www.webcitation.org/5rmqi4tPr?url=http://press.nintendo.com/GDC2010/downloads/bio_yoshiosakamoto.zip |archivedate=2010-08-06 |title=GDC 2010 Online Press Kit – Yoshio Sakamoto Bio |date=March 2010 |publisher=[[Nintendo|Nintendo of America Inc]] |accessdate=2010-08-06 |dead-url=yes }}</ref><ref>{{cite web |url=http://www.mobygames.com/game/nes/metroid/credits |title=Metroid (1986) NES credits |work=MobyGames |accessdate=2012-05-06}}</ref>在这之后，坂本贺勇还在《{{le|光神话|Kid Icarus}}》中担任游戏设计工作。<ref>{{cite journal |journal=[[Nintendo_Dream|Nintendo Dream]] |script-title=ja:やればやるほどディスクシステムインタビュー(前編) |pages=96–103 |date=2004-08-06 |issue=118 |language=ja |publisher=Mainichi Communications Inc.}}</ref> 

令坂本贺勇受到业界更多关注的是他从《[[超级银河战士|超级银河战士]]》开始负责这一系列的2D风格作品，他担当了《[[超级银河战士|超级银河战士]]》、《[[银河战士_融合|银河战士 融合]]》、《[[银河战士_零点任务|银河战士 零点任务]]》以及《[[银河战士_另一个M|银河战士 另一个M]]》等作品的编剧和制作人。<ref>{{cite web |url=http://www.nintendo.co.uk/NOE/en_GB/news/iwata/iwata_asks_-_metroid_other_m_18705_19405.html |archiveurl=https://www.webcitation.org/69y9Up454?url=http://www.nintendo.co.uk/NOE/en_GB/news/iwata/iwata_asks_-_metroid_other_m_18705_19405.html |archivedate=2012-08-16 |deadurl=no |title=Iwata Asks – Metroid: Other M – Nintendo |date=2010-08-26 |publisher=[[Nintendo|Nintendo of Europe GmbH]] |accessdate=2012-08-16 }}</ref><ref>{{cite web |url=http://www.g4tv.com/thefeed/blog/post/716536/team-ninjas-yosuke-hayashi-talks-ninja-gaiden-3/ |archiveurl=https://www.webcitation.org/69yKxFiz6?url=http://www.g4tv.com/thefeed/blog/post/716536/team-ninjas-yosuke-hayashi-talks-ninja-gaiden-3/ |archivedate=2012-08-17 |deadurl=no |title=Team Ninja's Yosuke Hayashi Talks Ninja Gaiden 3 |date=2011-09-21 |publisher=[[G4_Media|G4 Media, Inc]] |accessdate=2012-08-16 }}</ref>在第一人称视角的《[[银河战士Prime|银河战士Prime]]》系列分支作品诞生以后，《银河战士》系列逐渐把重心从传统的2D动作类转移到了这一新分支上，而一直负责这一系列2D风格作品的坂本贺勇也将工作重心转到了其他游戏系列上；不过由于他对这个系列的巨大影响，在每部《Prime》系列作品中，他都以“特别感谢”的身份出现。<ref>[http://www.nintendoworldreport.com/translation/23982/yoshio-sakamoto-interview Yoshio Sakamoto Interview].NintendoWorldReport.2010-09-05.[2015-08-10].</ref>
