'''坂口理子'''（{{lang-ja|坂口理子}}，{{bd|1972年|5月16日|||Sakaguchi Riko}}）是日本的女性電視、電影、舞台劇本家。

== 經歷 ==
出生位於[[神奈川縣|神奈川縣]][[橫濱市|橫濱市]]一帶的家庭，父親是曾翻譯多部外語文學、以及撰寫兒童讀物的作家[[三田村信行|三田村信行]]。在學生時期坂口理子投入[[歌舞伎|歌舞伎]]方面題材的研究，之後於[[早稻田大學|早稻田大學]]就讀時有研修相關演劇的科目。

從早稻田大學畢業後，坂口理子曾在[[NHK|NHK]]集團裡的NHK ENTERPRISES下工作。之後2000年坂口理子自NHK ENTERPRISES請離，改前往東京的Scenario Center；研修學習創作劇本方法。<ref>{{cite web |url=https://www.scenario.co.jp/?p=5958 |script-title=ja:学校で学ぶということ・・・坂口理子さん |work=Scenario Center |date=2014年4月2日 |accessdate=2016年2月18日}}</ref>

2006年坂口理子以{{nihongo|《祝有個好年》|良いお年を}}的作品投稿，獲得當年[[朝日電視|朝日電視]]挖掘新血劇本家活動的優秀獎。<ref>{{cite web |url=http://news.walkerplus.com/article/60057/ |script-title=ja:「テレビ朝日21世紀新人シナリオ大賞」受賞作品決定 |publisher=[[角川書店|角川書店]] |date=2015年6月15日 |accessdate=2016年2月18日}}</ref>之後坂口理子開始有編寫電視劇的機會，其中在2008年她以[[大鹿歌舞伎|大鹿歌舞伎]]為題材；於[[NHK綜合頻道|NHK綜合頻道]]上播出的《{{lang|ja|おシャシャのシャン!}}》有獲得第31屆日本創作電視劇本大獎裡的最優秀獎，<ref>{{cite web |url=http://www.hosakkyo2012.jp/%E5%89%B5%E4%BD%9C%E3%83%89%E3%83%A9%E3%83%9E%E5%A4%A7%E8%B3%9E-new/%E9%81%8E%E5%8E%BB%E3%81%AE%E5%8F%97%E8%B3%9E%E8%80%85-%E3%83%86%E3%83%AC%E3%83%93/ |script-title=ja:創作テレビドラマ大賞　歴代受賞者一覧  |work={{lang|ja|日本放送作家協会}} |accessdate=2016年2月18日}}</ref>而同一年另編寫的作品{{nihongo|《Freude！～用歡喜的歌說再見～》|フロイデ!〜歓喜の歌でサヨナラを〜}}則得到WOWOW劇本大獎的優秀獎；並在2012年被改寫為小說。<ref>{{cite web |url=http://www.wowow.co.jp/drama/sc/second/ |script-title=ja:WOWOWシナリオ大賞 |work=WOWOW INC. |accessdate=2016年2月18日}}</ref>

之後2009年[[吉卜力工作室|吉卜力工作室]]的導演[[高畑勳|高畑勳]]看了《{{lang|ja|おシャシャのシャン!}}》的電視劇內容後，便邀請坂口理子與他一同編寫改編自日本古典文學《[[竹取物語|竹取物語]]》的動畫《[[輝耀姬物語|輝耀姬物語]]》劇本，<ref>{{cite book |script-title=ja:かぐや姫の物語 ビジュアルガイド |publisher=[[角川書店|角川書店]] |isbn=978-404-110568-9 |date=2013年11月22日 |page=86頁 }}</ref><ref>{{cite web |url=http://magazine.kakaku.com/mag/hobby/id=1413/ |script-title=ja:高畑勲監督の超大作『かぐや姫の物語』完成報告会見レポート |work=Kakaku |date=2013年11月10日 |accessdate=2016年2月18日}}</ref>而坂口理子在2013年《輝耀姬物語》上映後，也將其作品故事改寫為小說。往後2015年坂口理子則擔任了《[[醫師們的戀愛處方箋|醫師們的戀愛處方箋]]》、《[[絲島森林之家|絲島森林之家]]》等多部電視影集的劇本內容。<ref>{{cite web |url=http://www.cinra.net/news/20151129-itonomori |script-title=ja:樹木希林が死刑囚と交流する謎の老婦に、永作博美主演の福岡発ドラマ |work=CINRA |date=2015年11月29日 |accessdate=2016年2月18日}}</ref>
