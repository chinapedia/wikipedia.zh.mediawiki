'''魏章'''，生卒年不详，[[战国|战国]]时期[[秦国|秦国]]将领。

== 生平 ==
魏章本为[[魏国_(战国)|魏国]]人，后投奔秦国，与[[张仪|张仪]]一同侍奉[[秦惠文王|秦惠文王]]。<ref>《史记索隐·魏世家》：章为魏将，後又相秦。</ref>

前313年，[[楚怀王|楚怀王]]派[[屈匄|屈匄]]等三名[[大夫|大夫]]联合[[齐国|齐国]]夺取了秦国的[[曲沃|曲沃]]（今[[河南省|河南省]][[陕县|陕县]]西南），继而包围-{[[於中|於中]]}-（今[[河南省|河南省]][[西峡县|西峡县]]东）。秦惠文王畏惧楚、齐联军，于是派[[张仪|张仪]]出使[[楚国|楚国]]，以割让-{[[商於|商於]]}-（今河南省[[淅川县|淅川县]]西）之地六百里为诱饵，引诱楚怀王解除楚、齐联盟。楚怀王贪图商於之地的富庶，不听[[陈轸|陈轸]]的劝告，果然中计。楚怀王一面派使者与齐国解除盟约，一面派使者随张仪到秦国接收土地。<ref>《战国策·卷四·秦策二·齐助楚攻秦》：齐助楚攻秦，取曲沃。其后，秦欲伐齐，齐、楚之交善，惠王患之。</ref><ref>《史记·卷四十一·越王句践世家》：楚三大夫张九军，北围曲沃、於中。</ref><ref>《史记·卷四十·楚世家》：（楚怀王）十六年，秦欲伐齐，而楚与齐从亲，秦惠王患之，乃宣言张仪免相，使张仪南见楚王，谓楚王曰：“…王为仪闭关而绝齐，今使使者从仪西取故秦所分楚商於之地方六百里，如是则齐弱矣。”怀王大悦…因使一将军西受封地。</ref>

张仪回到秦国后假装从车上掉下来受伤，一连三个月没上朝，不见楚国使者。楚怀王以为这是与齐国断交不彻底的原因，于是派勇士[[宋遗|宋遗]]到齐国辱骂[[齐宣王|齐宣王]]，齐宣王大怒转而与秦国缔结盟约。张仪见秦、齐两国结盟后才上朝面见楚国使者，告诉楚国使者并非割让商於之地六百里，而是自己的封地六里。楚国使者回国后将此事转告给楚怀王，楚怀王得知被骗后大怒，转而与齐国联合，一面命屈匄猛攻秦国的於中；<ref>《史记·卷四十·楚世家》：张仪至秦，详醉坠车，称病不出三月，地不可得。楚王曰：“仪以吾绝齐为尚薄邪？”乃使勇士宋遗北辱齐王。齐王大怒，折楚符而合于秦。秦齐交合，张仪乃起朝，谓楚将军曰：“子何不受地？从某至某，广袤六里。”楚将军曰：“臣之所以见命者六百里，不闻六里。”即以归报怀王。怀王大怒，兴师将伐秦。</ref>一面任命[[柱国|柱国]][[景翠|景翠]]为主将，包围秦国的盟国[[韩国_(战国)|韩国]]的[[雍氏|雍氏]]（今河南省[[禹县|禹县]]东北）。<ref name="秦本纪"/><ref name="年表"/>而齐国也趁机出兵，联合[[宋国|宋国]]包围了秦国的盟国魏国的[[煮枣|煮枣]]（今[[山东省|山东省]][[东明县|东明县]]南）。<ref>《史记集解·韩世家》引徐广曰：齐、宋围煮枣。</ref>秦惠文王发兵三路，一路由[[庶长|庶长]]魏章率领救援商於，<ref name="秦本纪"/>一路由甘茂率领攻占楚国的[[汉中郡|汉中]]，<ref>《史记·卷七十一·樗里子甘茂列传》：王见（甘茂）而说之，使将，而佐魏章略定汉中地。</ref>一路由庶长[[樗里疾|樗里疾]]率领联合韩、魏两国反击楚、齐。<ref name="秦本纪"/>

前312年，樗里疾所率领的秦军与韩军在雍氏击败景翠的军队，继而与魏章的部队会合。<ref>《史记·卷七十一·樗里子甘茂列传》：明年，（樗里疾）助魏章攻楚，败楚将屈丐，取汉中地。</ref><ref name="年表">《史记·卷十五·六国年表》：（韩宣惠王）二十一年，秦助我攻楚，围景座（应为翠）。</ref>秦、韩联军与楚军在[[丹阳_(楚国)|丹阳]]（今河南省淅川县[[丹水|丹水]]和[[淅水|淅水]]交汇处一带）交战，楚军大败，被斩首八万，楚国大将军屈匄及裨将军[[逢侯丑|逢侯丑]]等七十余名将领被俘，屈匄被俘后遭处决，秦军乘胜夺取楚国汉中之地六百里，设立[[汉中郡|汉中郡]]。秦军同时联合魏军在[[濮水|濮水]]击败了包围煮枣的齐军，俘虏了齐将[[声子|声子]]，齐将[[匡章|匡章]]败逃，秦、魏联军乘胜攻打了[[燕国|燕国]]。<ref name="秦本纪">《史记·卷五·秦本纪》：（秦惠文王）（更年）十三年，庶长章击楚于丹阳，虏其将屈丐，斩首八万；又攻楚汉中，取地六百里，置汉中郡。楚围雍氏，秦使庶长疾助韩而东攻齐，到满（应为濮）助魏攻燕。</ref><ref>《战国策·卷十三·齐策六·濮上之事》：濮上之事，赘子死，章子走。</ref>

楚怀王得知战败的消息后大怒，增调全国的军队深入秦国境内，秦、楚两国战于[[蓝田|蓝田]]（今[[陕西省|陕西省]][[蓝田县|蓝田县]]），楚军大败。韩、魏两国也趁机出兵，一直打到邓（今[[湖北省|湖北省]][[襄阳市|襄阳市]]北），迫使楚国撤军，楚怀王被迫割让两座城池与秦国议和。<ref>《史记·卷四十·楚世家》：（楚怀王）十七年春，与秦战丹阳，秦大败我军，斩甲士八万，虏我大将军屈丐、裨将军逢侯丑等七十余人，遂取汉中之郡。楚怀王大怒，乃悉国兵复袭秦，战于蓝田，大败楚军。韩、魏闻楚之困，乃南袭楚，至于邓。楚闻，乃引兵归。</ref><ref>《史记·卷七十·张仪列传》：楚又复益发兵而袭秦，至蓝田，大战，楚大败，于是楚割两城以与秦平。</ref>

秦惠文王死后，其子[[秦武王|秦武王]]继位，张仪、魏章失宠。前310年，张仪、魏章被秦武王驱逐至魏国，此后魏章事迹再无记载。<ref>《史记·卷五·秦本纪》：武王元年，张仪、魏章皆东出之魏。</ref>

== 观点 ==
现代学者[[马非百|马非百]]所著《[[秦集史|秦集史]]》认为秦国国内有朝内执政大臣推荐亲信将领在外统兵的传统，如张仪与魏章、[[魏冉|魏冉]]与[[白起|白起]]、[[范雎|范雎]]与[[郑安平|郑安平]]就属于这种关系。执政大臣与亲信将领是相辅相成的关系，如张仪失势后魏章也被驱逐，魏冉被罢免职务后白起随后被赐死，郑安平投降[[赵国|赵国]]后范雎也辞职归乡。<ref>{{cite book|author=马非百|coauthors= |title=《秦集史》人物传四之二|year=1982年8月|publisher=中华书局|location= |isbn=}}</ref>

== 文学形象 ==
长篇历史小说《[[东周列国志|东周列国志]]》中，魏章于第九十一回《学让国燕哙召兵 伪献地张仪欺楚》中登场。张仪欺楚后，楚怀王大怒，派屈匄为大将、逢侯丑为副将带兵十万由[[天柱山|天柱山]]西北攻打秦国的蓝田。秦惠文王任命魏章为主将、甘茂为副将迎战楚军，[[齐湣王|齐湣王]]也派匡章率军助战，秦、齐大败楚军于丹阳。屈匄聚集残兵再战，被甘茂斩杀。此战楚军被斩首八万，逢侯丑等七十余名楚国将领战死。<ref>见《东周列国志·第九十一回·学让国燕哙召兵 伪献地张仪欺楚》。</ref>在第九十二回《赛举鼎秦武王绝胫 莽赴会楚怀王陷秦》中，秦武王继位后设置[[丞相|丞相]]一职，任命甘茂为左丞相，樗里疾为右丞相，魏章因没有得到相位，一怒之下投奔魏国。<ref>见《东周列国志·第九十二回·赛举鼎秦武王绝胫 莽赴会楚怀王陷秦》。</ref>
