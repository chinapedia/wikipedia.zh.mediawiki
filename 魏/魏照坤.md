'''魏照坤'''{{BD|1995年|1月25日|||W}}，出生于[[新疆|新疆]][[库尔勒|库尔勒]]，是一名中国[[足球|足球]]运动员，司职[[中场|中场]]。目前效力于[[中甲|中甲]]球队[[武汉卓尔|武汉卓尔]]。

==俱乐部生涯==
魏照坤从小在库尔勒市第五小学读书。2007，年仅12岁的魏照坤被召入[[新疆体育职业技术学院|新疆体育职业技术学院]]足球队。2008年代表新疆在四川乐山参加全国U15锦标赛后被招进[[浙江绿城|浙江绿城]]俱乐部梯队作为主力效力至今，曾代表浙江参加[[第十二届全运会|第十二届全运会]]18岁以下年龄男子组足球比赛，并获得亚军。2014年入选U19国家集训队，备战2014年亚洲青年足球锦标赛决赛。

2016年2月，中甲球队武汉卓尔宣布租借魏照坤一年。<ref>{{cite web |url=http://sports.sohu.com/20160227/n438665657.shtml |title=卓尔官方宣布租借绿城两将 程进魏照坤正式加盟 |author= |date=2016年2月27日 |publisher=搜狐体育  |accessdate=2017年2月23日}}</ref>2017年1月，魏照坤正式转会至武汉卓尔，征战当年中甲联赛。<ref>{{cite web |url=http://sports.sina.com.cn/china/b/2017-02-15/doc-ifyamkzq1335739.shtml |title=卓尔官宣95后小将加盟 期待成为攻击线上的“手枪” |author= |date=2016年2月15日 |publisher=新浪体育  |accessdate=2017年2月27日}}</ref>
