'''魏應行'''{{BD|1959年|1月||}}<ref name="食品是老实人做的买卖"/>，[[彰化縣|彰化縣]][[福佬客家人|福佬客家人]]<ref name="魏家Ｆ４">{{Cite news|url=http://cwef.cw.com.tw/2012/article/article47-3.aspx|title=魏家Ｆ４ 合體打造頂新霸業─ 謹守客家人的家庭價值|language=zh-tw|publisher=《天下雜誌》487期|author=熊毅晰|date=2011-12-14|accessdate=2014-10-14}}</ref>，是台灣出身的企業家，[[頂新國際集團|頂新國際集團]]魏家四兄弟之老四，前味全董事長<ref name="支撐頂新集團的男子漢"/>，身兼頂新控股、上海福滿家、[[廣州福滿家|廣州福滿家]]董事長、上海頂鶴、[[德克士|德克士]]、布列德麵包等董事長<ref name="頂新集團組織圖">{{cite web| url =http://thotel.thu.edu.tw/userfiles/files/%E9%A0%82%E6%96%B0%E9%9B%86%E5%9C%98-%E5%85%AC%E5%8F%B8%E5%8F%8A%E8%81%B7%E6%B6%AF%E7%99%BC%E5%B1%95%E7%B0%A1%E4%BB%8B20140422.pdf| date =2014-04-22| accessdate =2014-10-16| title =頂新集團組織圖| author =| publisher =東海大學餐旅管理學系| language =zh-tw| deadurl =yes| archiveurl =https://web.archive.org/web/20160304114826/http://thotel.thu.edu.tw/userfiles/files/%E9%A0%82%E6%96%B0%E9%9B%86%E5%9C%98-%E5%85%AC%E5%8F%B8%E5%8F%8A%E8%81%B7%E6%B6%AF%E7%99%BC%E5%B1%95%E7%B0%A1%E4%BB%8B20140422.pdf| archivedate =2016-03-04}}</ref>。

==生平==
在魏家兄弟中，魏應行善於交際，有人以“[[孔雀|孔雀]]”來形容他<ref name="支撐頂新集團的男子漢"/>。與三哥[[魏應充|魏應充]]都為[[彰化高中|彰化高中]]畢業，皆獲得該校的傑出校友<ref name="彰中">{{cite news| url =http://www.appledaily.com.tw/realtimenews/article/life/20141014/487595/%E3%80%90%E6%9B%B4%E6%96%B0%E3%80%91%E9%AD%8F%E6%87%89%E5%85%85%E8%B3%A3%E9%BB%91%E5%BF%83%E6%B2%B9%E3%80%80%E5%BD%B0%E4%B8%AD%E5%AD%B8%E5%BC%9F%EF%BC%9A%E6%87%89%E9%80%80%E5%82%91%E5%87%BA%E6%A0%A1%E5%8F%8B|date=2014-10-14|accessdate=2014-10-16|title=非魏應充賣黑心油　彰中學弟：應退傑出校友|author=| publisher =《蘋果日報》| language =zh-tw}}</ref>。年輕時在父親的油厂忙活，晚上則去摆摊卖从菲律宾倒腾回来的扇子。後來，魏家兄弟决定派魏应行开拓中國大陆市场。那年僅28歲的他<ref name="回馈社会"/>，就带着500万美元和數桶自家出产的“顶好清香油”單獨到[[北京|北京]]，但三年來生意依舊不见起色，很快就花光多數的钱。决定再试一次的他坐上往[[内蒙古|内蒙古]][[通辽|通辽]]的火车。那時，他端着一碗热气腾腾的[[泡麵|泡麵]]穿过乘客，吸引了群眾目光。隔日一早，当他拉出卧铺下的箱子想要拿泡麵，却意外地发现少了幾包。他立刻意识到这是一个新的商机，因此開始了魏家的富貴之路。他曾表示食品是老实人做的买卖。<ref name="食品是老实人做的买卖">{{cite news| url =http://www.globalpeople.com.cn/index.php/news/finance/5241| date =2014-07-14| accessdate =2014-10-16| title =顶新魏应行：食品是老实人做的买卖| author =黄滢| publisher =《环球人物》256期| language =zh-cn| deadurl =yes| archiveurl =https://web.archive.org/web/20141019214843/http://www.globalpeople.com.cn/index.php/news/finance/5241| archivedate =2014-10-19}}</ref>

1996年，顶新集团在北京亮马河大厦举行捐资助学“育苗行动”新闻发布会。副董事长魏应行宣布将以长期的回馈行动捐资助学，初期投入1000万元人民币捐赠北京市、青海省、福建省的13所中小学，用作教室、学生宿舍楼的建设和教学设备的更新。他表示企业要盈利也要服务社会、回馈社会。<ref name="回馈社会">{{cite news| url =http://www.china.com.cn/book/zhuanti/wktb/2007-06/12/content_8375144.htm| date=2007-06-12|accessdate=2014-10-16|title=回馈社会“康师傅”| author=|publisher =中国网| language =zh-cn}}</ref>

1999年，甫接手味全董事長的魏應行以虧損為由暗自想解散[[味全龍|味全龍]]，沒想到龍隊當年拿下三連霸，再加上其他球團的關切，使公佈解散日期一延再延。龍迷透過網路及媒體發起連署、遊行、抵制味全活動，在同年12月12日發起「棒球迷大遊行」，在[[中正紀念堂|中正紀念堂]]大中至正門前集合表演行動劇、呼口號，並在下午二時出發前往[[立法院|立法院]]及[[監察院|監察院]]向立法院長[[王金平|王金平]]及行政院長[[蕭萬長|蕭萬長]]遞交「呼籲政府出面挽救職棒公開信」。當時部分的味全龍球員甚至自願減薪四成以保住球隊。魏應行依舊堅持解散，並對記者表示棒球比賽是打假的。龍隊總教練[[徐生明|徐生明]]受訪時難過說：「來了魏家才知黃家的好。」[[2014年臺灣劣質豬油事件|2014年臺灣劣質豬油事件]]爆發後，有球迷將這兩事件聯想在一起。<ref>{{cite news| url =http://ndap.dils.tku.edu.tw:8080/ndap/querynews2.jsp?id=116901|date=1999-12-13|accessdate=2014-10-16|title=味全龍最遲明天宣佈解散|author=李一中、李國彥| publisher =《聯合報》| language =zh-tw}}</ref><ref>{{cite news| url =http://news.ltn.com.tw/news/life/breakingnews/1130773|date=2014-10-14|accessdate=2014-10-16|title=非首次遭抵制 15年前魏家不顧球迷硬解散味全龍|author=| publisher =《自由時報》| language =zh-tw}}</ref><ref>{{cite news|url=https://tw.news.yahoo.com/%E4%BE%86%E4%BA%86%E9%AD%8F%E5%AE%B6%E6%89%8D%E7%9F%A5%E9%BB%83%E5%AE%B6%E7%9A%84%E5%A5%BD-%E5%B7%B2%E6%95%85%E5%BE%90%E7%B8%BD%E5%90%90%E6%A7%BD-221052395--mlb.html|date=2014-10-15|accessdate=2014-10-16|title=「來了魏家才知黃家的好」已故徐總吐槽|author=|publisher=《自由時報》|language=zh-tw|deadurl=yes|archiveurl=https://web.archive.org/web/20141023163157/https://tw.news.yahoo.com/%E4%BE%86%E4%BA%86%E9%AD%8F%E5%AE%B6%E6%89%8D%E7%9F%A5%E9%BB%83%E5%AE%B6%E7%9A%84%E5%A5%BD-%E5%B7%B2%E6%95%85%E5%BE%90%E7%B8%BD%E5%90%90%E6%A7%BD-221052395--mlb.html|archivedate=2014-10-23}}</ref>

2001年1月，獲得國家生技醫療品質獎的魏應行，與其他得獎人在總統府獲得[[陳水扁|陳水扁]]接見<ref>{{cite news| url =http://www.president.gov.tw/Default.aspx?tabid=131&itemid=3314&rmid=514|title=總統接見「國家生技醫療品質獎」得獎人|author=|date=2001-01-19|accessdate=2014-10-28| publisher =中華民國總統府| language =zh-tw}}</ref>。隨著味全股價攔腰再折半，魏應行轉為低調，2002年卸下味全董事長一職，轉戰大陸發展加盟連鎖次集團<ref name="支撐頂新集團的男子漢">{{cite news| url =http://big5.huaxia.com/sw/qyfc/00045260.html| date=2003-07-18|accessdate=2014-10-15|title=嘿，支撐頂新集團的男子漢| author=陳彥淳|publisher =華夏經緯網| language =zh}}</ref>。

2011年5月，魏應行與多位政商大老出席[[連戰|連戰]]、[[連勝文|連勝文]]、[[連勝武|連勝武]]一家在台北[[國賓飯店|國賓飯店]]宴請四川省長[[蔣巨峰|蔣巨峰]]的晚宴<ref>{{cite news| url= http://mag.chinareviewnews.com/crn-webapp/search/siteDetail.jsp?id=101703174&sw=%E7%8E%8B%E5%BF%97%E5%88%9A|date=2011-05-24|accessdate=2014-10-16|title=連戰會蔣巨峰　我愛四川獲全場掌聲|author=倪鴻祥| publisher=中評社| language =zh-tw}}</ref>。同年7月，魏應行長子魏士傑與羅心宜22日完婚<ref name="魏士傑">{{cite news| url =http://www.nownews.com/n/2011/07/22/473872|date=2011-07-22|accessdate=2014-10-16|title=頂新集團添囍事　魏應行長子魏士傑完婚|author=楊伶雯| publisher =NOWnews| language =zh-tw}}</ref>，由連戰擔任證婚人<ref>{{cite news| url =http://news.ltn.com.tw/news/politics/paper/821061|date=2014-10-13|accessdate=2014-10-16|title=《昔帝寶為鄰 圈購頂新ＴＤＲ還獲退佣》連被質疑與魏家交好|author=郭安家、涂鉅旻、俞肇福、林嘉東| publisher =《自由時報》| language =zh-tw}}</ref>，[[吳敦義|吳敦義]]、王金平、[[江丙坤|江丙坤]]等政要皆參與<ref>{{Cite news|url=http://www.gvm.com.tw/Boardcontent_18735.html|title=不僅是董事長，魏應充 更扮演集團文化長|author=高宜凡|language=zh-tw|publisher=《遠見雜誌》|date=2011-09|accessdate=2014-10-13}}</ref>。2012年8月，魏應行的女兒魏佳怡與從事五金和紡織品進出口貿易的周志皇企業聯姻，兩家還是帝寶鄰居，由副總統[[蕭萬長|蕭萬長]]當證婚人<ref name="魏應行今嫁女兒">{{cite news| url =https://tw.news.yahoo.com/%E9%A0%82%E6%96%B0%E5%9B%9B%E8%91%A3-%E9%AD%8F%E6%87%89%E8%A1%8C%E4%BB%8A%E5%AB%81%E5%A5%B3%E5%85%92-050655248.html|date=2012-08-04|accessdate=2014-10-16|title=頂新四董 魏應行今嫁女兒|author=| publisher=民視| language =zh-tw}}</ref>。

2013年魏應行至[[總統府_(臺灣政府建築)|總統府]]對總統[[馬英九|馬英九]]簡報兩岸農業，2天後陪同聽取簡報的[[羅智強|羅智強]]受魏應交之邀至魏家彰化祖厝成美堂向魏家四兄弟拜年，該年羅智強與魏家有4次互動<ref>{{cite news| url =http://newtalk.tw/news/2015/01/07/55571.html|date=2015-01-07|accessdate=2015-01-08|title=魏應行向馬簡報後2天 羅智強赴魏家拜年|author=謝莉慧| publisher=新頭殼newtalk| language =zh-tw}}</ref>。
