'''达尔文奖'''（'''Darwin Awards'''）是一个带半[[开玩笑|开玩笑]]性质的奖项，由[[美国|美国]][[斯坦福大学|斯坦福大学]][[神经学|神经学]]研究实验室工作人员温蒂·诺斯卡特（Wendy Northcutt）创立，透過網友投票，以平均得分最高的事件為該年度得獎者。该奖项以著名[[演化论|演化论]]学者[[查尔斯·达尔文|查尔斯·达尔文]]命名，是因为奖项的得主均已逝世或永久失去[[生育能力|生育能力]]，“让自己愚蠢的基因無法再传播出去”，对人类物種的進化作出偉大贡献。亦有人认为，达尔文奖是用来记录“那些在演化过程中走得最慢的人们”。

== 起源 ==

自1985年以来，达尔文奖就在[[电子邮件|电子邮件]]和[[Usenet|Usenet]]的[[新闻组|新闻组]]中广为流传。据称，自1991年以来，每年都有颁发新的达尔文奖。目前，有数个网站在记录着新出现的达尔文奖项，其中较为著名的一个是由[[美国|美国]]人温蒂·诺斯卡特自1994年管理维护的''darwinawards.com''。诺斯卡特自己也写过数本关于达尔文奖的书。

== 获奖要求 ==
诺斯卡特列出了获奖的5项基本要求：
# 必須自我移除於人類基因庫之外（Out of the gene pool: dead or sterile）
#: 即為无繁殖能力、死亡或者已[[绝育|绝育]]。这一项常常引起争论：被提名者往往可能很老，早已失去繁殖能力，或者他们早已育有儿女；或被提名者早已將[[精子|精子]]或[[卵子|卵子]]冷凍儲存；或以生物複製技術將其本複製等。在诺斯卡特自己写的''Darwin Awards''一书提出“荒岛定律”：被提名者已經缺乏在無人荒島上與伴侶繁衍子孫之能力，即合乎資格。
# 必須展現驚人的判斷誤差（Astounding misapplication of judgment）
#: 當事人須嚴重缺乏[[知識|知識]]與判斷力以致[[死亡|死亡]]或[[絕育|絕育]]。
# 必須為自找的、主動而為的、為其死亡之導因（Cause one's own demise）
# 必須具備充分的心智與判斷能力，而且已[[成年|成年]]並擁有為自己行為負責的能力（Capable of sound judgment）
#: 排除[[兒童|兒童]]、[[阿兹海默症|阿兹海默症]]患者、[[唐氏綜合症|唐氏綜合症]]患者，但仍有少量具代表性事件囊括在內。
# 必須為獲得證實的事件（The event must be true）
#: 只接受具公信力的報刊、出版的文章、經證實的電視報導、可信目擊證人的證詞為消息來源。

== 四大類別 ==
* 達爾文獎（Darwin Awards）
: 唯一符合達爾文獎的事件，以表揚那些愚蠢地[[自戕|自戕]]或[[自宮|自宮]]以達致將自己從基因庫移除的人士。
* 榮譽表揚（At-Risk Survivors）
: 雖被證為真實事件，可惜未完全符合獲獎資格，但其愚蠢和創新精神依然值得嘉許、表揚。
* 都市傳奇（Urban Legends）
: 有杜撰可能的故事，來源多不可考，故事中人的真實性亦無從考證，幸甚！
* 私房故事（Personal Accounts）
: 故事多被提供者證實，但無從考證，而提供者與故事不一定有直接關係。

== 砸鍋事件 ==
* 於1995發生的噴氣起飛助推器事件，已被列為1995年的達爾文獎得主，當多數人都認為事實屬真，而後經由[[亞利桑納|亞利桑納]]州公安部門證實，那個事件僅能當做都市傳奇而已。至今，主辦單位只能曲法遷就，使該事件維持1995年的達爾文獎紀錄。
