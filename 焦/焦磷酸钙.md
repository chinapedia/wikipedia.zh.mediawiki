'''焦磷酸钙'''，化学式Ca<sub>2</sub>P<sub>2</sub>O<sub>7</sub>。

== 性质 ==
焦磷酸钙是一种白色粉末，不溶于水和[[醇|醇]]，溶于[[稀盐酸|稀盐酸]]和[[硝酸|硝酸]]，水悬浮液呈酸性。根据温度不同，可有三种不同的晶型：α型（1210 °C）、β型（750～900 °C）和γ型（530～750 °C）。<ref name="zghg">中国化工产品大全 上卷，Bc281 焦磷酸钙，页255</ref>

== 制备 ==
1、由[[磷酸氢钙|磷酸氢钙]]在高温下进行煅烧脱水再经粉碎制得。<ref name="zghg" />
:<math>\rm \ 2CaHPO_4\rightarrow Ca_2P_2O_7+H_2O</math>

2、[[焦磷酸钠|焦磷酸钠]]与[[氯化钙|氯化钙]]溶液在90 °C发生复分解，生成无水焦磷酸钙。经水洗、离心脱水、水洗、干燥、煅烧和粉碎，制得焦磷酸钙成品。<ref name="zghg" />
:<math>\rm \ Na_4P_2O_7+2CaCl_2\rightarrow Ca_2P_2O_7+4NaCl</math>

== 用途 ==
焦磷酸钙用作牙膏磨料、涂料填料、电材萤光体。也在食品工业中用作缓冲剂、中和剂、营养增补剂和酵母养料。<ref name="zghg" />
