'''焦油色素'''，又稱'''苯胺色素'''、'''有機合成色素'''、'''人工色素'''、'''煤焦色素'''、'''煤渣色料'''，是一種人工合成色素，與天然色素相對。常用於食品、[[飲料|飲料]]、[[藥物|藥物]]、[[口紅|口紅]]與[[化妝品|化妝品]]的[[染色|染色]]上，尤其是作為[[食品添加劑|食品添加劑]]。其主要是由[[煤焦油|煤焦油]]經過[[蒸餾|蒸餾]]、[[硫化|硫化]]、[[硝化|硝化]]等程序加工而成<ref name="cichb">{{cite web| title = 食品添加物知多少－嘉義市政府衛生局| url = http://www.cichb.gov.tw/health/health12.htm| accessdate = 08-02-2014| deadurl = yes| archiveurl = https://web.archive.org/web/20131117095855/http://www.cichb.gov.tw/health/health12.htm| archivedate = 2013-11-17}}</ref>。由於攝取焦油色素或會危害健康，例如引致[[過動|過動]]及注意力不集中<ref name="isu">{{cite web| title = 食品添加物與孩童注意力不集中及過動關係| url = http://www2.isu.edu.tw/interface/shownews.php?dept_mno=27&dept_id=25&id=67377| accessdate = 08-02-2014}}</ref>，很多國家地區對它的使用有所規管。人工色素例子包括[[檸檬黃|檸檬黃]]、[[日落黃|日落黃]]等等。

== 偶氮染料 ==
[[偶氮染料|偶氮染料]]是應用最廣的合成染料，常用於各種[[纖維|纖維]]、[[皮革|皮革]]、[[木材|木材]]、[[紙張|紙張]]等的染色，或用於[[油漆|油漆]]、[[塑料|塑料]]、[[橡膠|橡膠]]的著色<ref>{{cite web| title = 偶氮染料－商品安全展示中心| url = http://csec.bsmi.gov.tw/index1.asp?u=3&sub=64&nID=19| accessdate = 09-02-2014| deadurl = yes| archiveurl = https://web.archive.org/web/20140222153511/http://csec.bsmi.gov.tw/index1.asp?u=3&sub=64&nID=19| archivedate = 2014-02-22}}</ref>。

== 食品添加劑 ==
由於人工[[食用色素|食用色素]]在不同的溫度及[[酸鹼度|酸鹼度]]之下穩定性較高，色澤鮮豔、容易調校顏色，效果明顯及成本低，它被廣泛用於[[食品工業|食品工業]]中<ref name="mo">{{cite web| title = 色素的功用－澳門消費者委員會 | url = http://www.consumer.gov.mo/c/active/208-c6.pdf| accessdate = 08-02-2014}}</ref>。
