'''焦国标'''（{{bd|1963年|11月21日|catIdx=J焦}}）出生于[[中华人民共和国|中华人民共和国]][[河南省|河南省]][[开封市|开封市]][[杞县|杞县]]，是[[新闻学|新闻学]]研究者、作家和[[中华人民共和国持不同政见者列表|异见学者]]。

== 简历 ==

焦国标1963年11月21日（农历十月初六）出生于河南开封杞县。焦国标先后取得[[河南大学|河南大学]]中国文学学士、古代汉语硕士、[[中国人民大学|中国人民大学]]新闻学博士。后任[[北京大学|北京大学]]新闻学院副教授。

2003年4月发表诗《致美国兵》，支持美国发动[[美伊战争|美伊战争]]，内有“假如今生注定死于战火， 就作美国精确制导炸弹下的亡灵”的诗句。<ref>{{cite web|author1=子日|title=子日：焦国标《致美国兵》诗解读|url=http://www.epochtimes.com/gb/4/5/11/n536305.htm|website=http://www.epochtimes.com/gb/4/5/11/n536305.htm}}</ref>

2004年3月发表《讨伐中宣部》一文，指责[[中宣部|中宣部]]实行[[愚民政策|愚民政策]]，主张新闻自由。该文最初发表在《开放》杂志上，后来被互联网上广泛转载。<br>2005年3月应美国[[国家民主基金会|国家民主基金会]]邀到[[华盛顿|华盛顿]]做访问学者，计划6个月，同时被北京大学当作“自动离职”单方面除名。校方认为他未经批准擅自出国访问，焦认为是对他写作《讨伐中宣部》的报复。

2005年7月，发表《朱成虎就是中国的[[拉登|本·拉登]]》，批评[[中国人民解放军|中国人民解放军]]少将[[朱成虎|朱成虎]]发动核大战的駭人言论。

2005年11月，和[[高智晟|高智晟]]调查中国政府迫害[[法轮功|法轮功练习者]]的问题。

2009年11月，[[新加坡|新加坡]]内阁资政[[李光耀|李光耀]]在[[亚太经合会|亚太经合会]]（APEC）中概述中国领导体制的好处时，[[埃利奥特|埃利奥特]]评论说，“它不是民主体制。”随后，[[李光耀|李光耀]]反驳说，“中国人对民主体制不感兴趣”。焦国标则发表文章称“看了这篇报道，我顿时火冒三丈。中国人固然希望更好的生活，但更希望选票。”<ref>[http://www.dwnews.com/gb/Mainland/2009-11/2009-11-19-04-41-04-353.html 焦国标：讨伐李光耀侮辱中国人] {{webarchive|url=https://web.archive.org/web/20100221214708/http://www.dwnews.com/gb/Mainland/2009-11/2009-11-19-04-41-04-353.html |date=2010-02-21 }}</ref>[[香港中文大学|香港中文大学]]政治与行政学系高级导师[[蔡子强|蔡子强]]也质疑说：“[[李光耀|李光耀]]没有资格代表中国人。”<ref>[http://www.seehua.com/index.php/international/27-international/6361-%E3%80%8C%E4%B8%AD%E5%9B%BD%E4%BA%BA%E6%B2%A1%E5%85%B4%E8%B6%A3%E6%B0%91%E4%B8%BB%E3%80%8D-%E6%9D%8E%E5%85%89%E8%80%80%E6%8C%A8%E8%BD%B0 中国人没兴趣民主 李光耀挨轰] {{webarchive|url=https://web.archive.org/web/20110504200804/http://www.seehua.com/index.php/international/27-international/6361-%E3%80%8C%E4%B8%AD%E5%9B%BD%E4%BA%BA%E6%B2%A1%E5%85%B4%E8%B6%A3%E6%B0%91%E4%B8%BB%E3%80%8D-%E6%9D%8E%E5%85%89%E8%80%80%E6%8C%A8%E8%BD%B0 |date=2011-05-04 }}</ref>

2012年9月12号，焦被警方从家中带走，后被以涉嫌“煽动颠覆国家政权罪”刑事拘留<ref>{{Cite web |url= http://www.rfa.org/mandarin/Xinwen/jiaoguobiao-09182012134005.html |title= 敢言学者焦国标涉嫌“煽动颠覆国家政权”被拘|date=2012-09-18 |publisher= [[自由亚洲电台|自由亚洲电台]] |language= zh-cn |accessdate= 2013-10-21 }}</ref>。但未見後續審訊新聞。

== 出版作品 ==
杂文集《窝囊中国》、《奉献与义务的边际》、《新闻之外的敏感》、《独立的悲伤》、《你根本吓不住人家》，新闻史《名士风流——文化名人的报刊生涯》。
