'''协和语'''是曾经使用于日本影响下的中国东北地区的一种日语和汉语混杂的[[皮钦语|皮钦语]]（混杂语言），其主要存在时期自清末[[日俄战争|日俄战争]]后至1945年[[满洲国|满洲国]]灭亡。{{r|樱井2012|page1=1}}

指代“通行于满洲地区的日中混杂语”这一概念的名称很多，例如'''日满混淆语'''、'''沿线官话'''等。现在较常使用的“协和语”一名的可考用法最早出现在1948年的中文报纸，在满洲国存在的时代并未发现中文或日文材料如此称呼。

协和语的产生与日本人在满洲地区的日语使用、中文报纸广播等媒体中的日语借词有关，然而协和语本身并未得到[[关东军|关东军]]、满洲国政府的推行。

协和语作为一种皮钦语，产生于日语母语者和汉语官话母语者的接触，不过当时在日本人之间等场合也有所使用。它的语序接近日语，语法较日语简单，常省略日语的[[日语助词|助词]]，词汇来自日中两种语言，也有一些词汇是在协和语中创造出来的。

== 名称 ==
虽然现在对1945年以前满洲地区日中混杂语通常叫做协和语，然而在1945年以前的日文材料中，并不见“协和语”的提法；根据日本学者{{link-ja|安田敏朗}}的研究，同时代中文材料中也未见“协和语”这一叫法。“协和语”最早的出处可能是1948年的中文材料：发表于《生活报》的文章。应是一种讽刺的叫法。此后日本材料逐渐采用此名称。下表列出在当时日文材料中可见的日中混杂语的名称。{{r|樱井2012|page1=1-8}}

== 背景 ==
=== 满洲的日语使用情况 ===
日本人在满洲的存在主要始于1905年日俄战争之后。日本依据《[[朴茨茅斯和约|朴茨茅斯和约]]》，租借[[关东州|关东州]]，获取[[南满洲铁道|南满洲铁道]]权益{{r|樱井2012|page1=1}}。[[九一八事变|九一八事变]]之前日语教育主要发生在[[关东州|关东州]]和各处[[满铁附属地|满铁附属地]]。满洲国建立之后，出现了日本[[满蒙开拓移民|农业移民]]等各种移民，1945年满洲国灭亡时当地日本居民约有166万人。{{r|张2012|page1=52}}

根据日本学者1942年的记述，[[大连|大连]]、[[旅顺|旅顺]]的中国人基本都能说日语，[[奉天市|奉天]]较大连旅顺略少，而[[新京|新京]]更少，到[[哈尔滨|哈尔滨]]又明显减少，在北满的农村则40岁以上的中国人几乎没有能说日语的。{{r|张2012|page1=53}}

==== 教育 ====
控制满洲国的关东军为了避免国际社会的非议，在1932年建国伊始没有采用满铁附属地、关东州时期所使用的日文教材，而是大体沿用了中华民国的教育体制，教育中同时采用日语和汉语。1934年起，开始将日语课程加入小学教育内容，3年级以上的学生每周课有2课时的日语课。因而这一时期在教育体制上来说，日语的地位尚不及汉语和英语。{{r|张2012|page1=52}}

1938年1月，随着新学制改革，汉语{{NoteTag|满洲国官方称汉语为“满语”“满洲语”，而将满族的传统语言称为“固有满洲语”。{{r|张2012|page1=52}}}}、日语、蒙古语被定为满洲国的国语。三语国语时期，日语成为制度上的优势语言。学校教育要求学生必修日语，而在汉语和蒙古语之中选择一种。日语课程自小学一年级开始，每周8到10课时，时间是汉语课程的2倍。{{r|张2012|page1=52}}

==== 法律、行政 ====
满洲国的法律多为日本人起草，故而虽然以中文书写，有疑义时以日文解释为准。满洲国政府公文在建国时仅以中文发行，但由总务厅秘书处发行《满洲国政府公报日译》（公报1-3期时名为《满洲国政府公报邦译》）。1934年4月14日颁布的《政府公报发行规程》中规定，由秘书处发行公报日译版，并且政府公报的规程也适用于日译版。1935年12月24日，随着《政府公报发行规程》的修订，“日文翻译”的说法被“日文版”替代，从此中日双语都是公报的正式语言。此后中央政府的发行物均等地使用中日双文。而各地方政府则仍以中文为主。{{r|安田1996|page1=35-37}}

==== 奖励措施 ====
[[File:Language_Level_Test,_Chinese(Manshugo),_Level_3,_Manchukuo,_1938.jpg|thumb]]
1938年，满洲国民生部为了普及日语，开始组织语言能力考试“语学检定试验”，分为特等、一等、二等、三等这4个等级{{r|安田1996|page=41}}。第1次考试（1936年）时参与人数5,492人，报考日语的有3,607人；第6次考试（1941年）时参与人数39,769人，报考日语的有31,369人。对于考试合格者，政府给予奖励。如果企业招聘了拥有日语等级的人员，政府会给企业以补贴。而[[满铁|满铁]]等大企业还有公司内部的日语能力奖励机制。而自1938年起实施的官吏选拔制度中，参与选拔考试的人必须在自己母语之外选择日语、汉语、[[蒙古语|蒙古语]]、[[俄语|俄语]]之一接受考试。{{r|张2012|page1=52,53}}

=== 满洲国的汉语标准 ===
在满洲国成立之前，“[[满语|满语]]”和“满洲语”指的是满族的传统语言。而满洲国成立后，用“满语”指代汉语，偶尔也称“汉语”。而“支那语”“中华语”“华语”等名称几乎不见使用。这是因为满洲国作为新的国家，理应有对应的国语，所以尽管原本满族的传统语言满语并未灭绝，也要强行改变名字。{{r|安田1996|page1=46}}

为了与原本的汉语尽量区隔，满洲国的汉语在科学、文化、制度、组织等方面有意识地借用日语词汇，乃至在语法上也尽量追求与标准汉语有所区别。满洲国政府也试图确定“满洲语”的标准。1944年，满洲国决定以松花江流域地区的“满洲语”为标准口语。其理由是“此地虽然较偏远，然而发音却比南满地区好，与北京音接近”。{{r|安田1996|page1=46,47}}

=== 汉语新文字的尝试——满洲假名 ===
日语和汉语两种语言在书写上都是用汉字，导致汉字词汇有不同读法。满洲国{{r|曾1934}}和日本的学者为了减少这种混淆，尝试从日文中借用[[片假名|片假名]]来书写“满洲语”即满洲国的汉语，这就是[[满洲假名|满洲假名]]（{{lang|ja|満州カナ}}）。在同一时代，汉语有[[注音符号|注音符号]]和罗马字拼音来标记语音，满洲假名的创制也有与之对抗的意义。如果使用满洲假名，则汉语与日语可以共享一种文字，哪怕具体发音不同，词汇的借用也将变得更容易。此外，推行假名书写汉语，也有降低识字难度，提高识字率的效果。{{r|安田1996|page1=47,48}}

满洲国国语调查委员会在1941年10月，决定废止注音符号的使用。1943年，满洲假名的标准（《{{lang|ja|「満州カナ」趣意書並に解説書}}》和附表《{{lang|ja|満州カナ拼音表}}》）完成。此后，满洲国在继续使用汉字的前提下，要求以满洲假名作为辅助。{{r|安田1996|page1=49,50}}

日本学者之中也有对满洲假名批评。除了担心废除汉字会引起一些人的反抗情绪，也在技术上指出了满洲假名的问题：无法准确反映出汉语的[[双声|双声]]、[[叠韵|叠韵]]；以原本表示日语[[清浊|清浊]]音的假名标记汉语的[[送气音|送气音]]、[[不送气音|不送气音]]{{NoteTag|[[汉语拼音|汉语拼音]]也有这个问题：通常b/p、d/t、g/k、z/s在拉丁字母中表示浊音和清音，但汉语拼音中也用来表示送气、不送气音。}}；汉语中某些音节以满洲假名书写时差别难以理解，例如佛（{{lang|ja|フォー}}）与否（{{lang|ja|フォウ}}）。有学者建议使用满洲假名以外的其他方案作为汉语的表音文字。{{r|安田1996|page1=50,51}}

== 特征 ==
=== 多样的交流形式 ===
作为一种日中混杂语，协和语首先用于日本人和中国人的对话，然而它也被应用于日本人之间、中国人之间。由于使用者语言背景不同，所使用的混杂语中的日语、汉语成分也不同。例如，两个日本人之间的协和语可能只是“{{lang|ja|帆をかけた<u>{{ruby|小車|シァオチェー}}</u>は日本では見られないなァ}}”（在日本可没有[[独轮车#中国的风帆独轮车|挂着帆的独轮车]]），整句是正常日语，但直接借用了汉语词汇“小车”（[[独轮车|独轮车]]）。而日本人和中国人之间的协和语就明显具有混杂语言的特征，语法简单。此外协和语也应用于日本人和非汉语母语者之间。{{r|张2012|page1=58-60}}

=== 语法 ===
==== 语序 ====
协和语常使用日语式的语序，即[[主賓動語序|主语-宾语-动词的顺序]]。例如“他的姑娘的朋友大大的<u>有</u>”（他有很多女性朋友），“有”位于句末。{{r|四宫2017|page1=61,62}}

==== “的”字的用法 ====
汉语中，“你的”“我的”通常表示所属关系，并且在一些情况下可以省略“的”字。然而在协和语中，“{{ruby|你的|{{lang|ja|ニデー}}}}{{NoteTag|发音也记作多一个[[音拍|音拍]]的{{lang|ja|ニーデー}}}}”“{{ruby|我的|{{lang|ja|オデー}}}}”常被用作人称代词。例如“<u>{{ruby|你的|{{lang|ja|ニデー}}}}</u>{{lang|ja|どこへ行くんだ？}}”（你去哪里？）“<u>{{ruby|我的|{{lang|ja|オデー}}}}</u>{{lang|ja|活動でも}}{{ruby|看看|{{lang|ja|カンカン}}}}{{lang|ja|しようと思ふ。}}”（我去看看活动。）将“你的”“我的”当作人称代词借用到日语时，后边可能接续日语表示所属的助词“{{lang|ja|の}}”，产生形如“{{ruby|我的|{{lang|ja|オデー}}}}{{lang|ja|のは～}}”这种语义重复。这种将“你的”“我的”借用作人称代词的做法，可能是因为日语的人称代词多为2、3音节的词汇，而汉语“你”“我”为单音节。日语母语者借用汉语时，出于语音节奏上的习惯，下意识地加入“的”字。{{r|张2012|page1=62,63}}

口语中并不区分“的”与“地”。在协和语中也经常见到在副词、形容词中使用“地”、“的”的用法。例如“{{ruby|大大的|{{lang|ja|タァータァーデー}}}}要”（很需要）、“{{lang|ja|お前は}}{{ruby|聪明的|{{lang|ja|ツォンミンデ}}}}{{lang|ja|だな}}”（你很聪明）。{{r|张2012|page1=63}}

“的”字的过度使用被认知为协和语的特征，至今东北当地人也会在开玩笑时模仿协和语的这种用法。{{r|张2012|page1=63}}

==== 助动词“{{lang|ja|ある}}” ====
[[File:An_advertisement_in_a_Japanese_magazine_using_Kyowago_in_1937_(cropped).jpg|thumb]]
“{{jpn|hg=ある|rm=aru|link=no}}”本来是日语的一个常用[[五段动词|五段动词]]，但在协和语中可以代替标准日语里“{{lang|ja|である／です／だ}}”这三个助动词，表示断定。尽管在日语里这是一个语法错误，在协和语环境中日本人之间也会这样使用。该现象并非协和语所独有，在150年前[[横滨港|横滨港]]开设的时代，伴随着日本人与外国人的接触，也有类似的现象发生。{{r|张2012|page1=63,64}}

该现象在今日的日本也被看作代表性的中国式日语，在模仿中国人时会使用。{{r|张2012|page1=63,64}}

==== 日语助词、形式名词等省略 ====
日语的助词和形式名词在协和语中经常被省略。最常被省略的是助词“{{lang|ja|の}}”和“{{lang|ja|は}}”。例如“{{lang|ja|日本兵隊サン…ワタシ（の）バリカン（は）ヨク切レマシェン}}”（日本兵先生，我的{{link-ja|推剪|バリカン}}不太好用）。又如“{{lang|ja|余り走る（と）お客様車から落ちる（のでは）ないか。}}”（车太快了的话客人不会从车上掉下去吗？）一句省略了表示假设条件的{{lang|ja|と}}和准体言助词{{lang|ja|の}}及{{lang|ja|では}}。{{r|张2012|page1=65,66}}

=== 词汇 ===
==== 借词与句子级别的借用 ====
在日语母语者之间使用的协和语，主要是词汇级别的借用，例如：“{{lang|ja|おい、<u>{{ruby|啤酒|ピーチユ}}</u>を飲めよ}}”（喝啤酒吧）一句中，仅借用了汉语“啤酒”一词。而日语母语者和汉语母语者之间则更可能发生句子结构级别的借用。如（日本人问当地小孩）：“{{ruby|[[邮便局|邮便局]]在哪儿|{{lang|ja|ユービンチューツァイナール}}}}？”{{r|张2012|page1=60,61}}

在借用时可能忽略被借用词汇本身已有的语法意义，导致意思的重复，例如：“{{lang|ja|おやッ…}}<u>{{ruby|落了马了|{{lang|ja|ラオラマーラ}}}}</u>{{lang|ja|したな}}”（哎呀，从马上掉下来了）一句以日语为基础，借用了汉语“落了马了”。汉语落了马了本身已经表明事情已经完成，但日语借用后仍然使用动词过去形“{{lang|ja|した}}”，实质上是意义重复。{{r|张2012|page1=60,61}}

==== 词汇来源 ====
借用自日语的例如“{{jpn|j=進上|rm=shinjō|link=no}}”，表示给予；“{{jpn|j=飯々|rm=meshimeshi|link=no}}”，表示饭或吃；“{{jpn|j=帰ろう|rm=kaerō|link=no}}”，表示回去或走。{{r|四宫2017|page1=65}}

借用自汉语的例如“坏了”，表示破损、受伤、变质腐烂等；“大大的”，表示非常、大量；“一样的”，表示相同。{{r|四宫2017|page1=66}}

日汉混杂的词语例如“还少少”，“少少”为日语，意思为还稍微；“不行{{jpn|hg=じゃないか|link=no|rm=janaika}}”，用于主张否定意见。{{r|四宫2017|page1=67}}

还有一部分为造词，例如“{{jpn|hg=[[不够本|ぽこぺん]]|link=no|rm=pokopen}}”{{NoteTag|有材料认为ぽこぺん来自汉语“不够本”，这里采用论文里的说法，认为属于造词。}}，表示不好、不行；“干活计”，表示工作、做事、做买卖。{{r|四宫2017|page1=68}}

=== 语音 ===
当时对协和语的记录主要是文字，不过从文字记录也能够了解一些上的特征。可以看到借用的汉语词汇尽量接近汉语发音，而不是日语汉字的音读。然而因为使用的仍然是日文假名标音，能够记录的发音仍然和汉语有差距，并且无法记录声调，可以推测当时的协和语在语音上对汉语借词的发音也不够准确。{{r|张2012|page1=61,62}}

== 对协和语的看法 ==
满洲国时期，政府与日本学者对作为混杂语言的协和语持批判态度。{{r|樱井2012|page1=10,11}}

“协和语”这一概念在中文语境中一直带有负面色彩。“协和语”的存在常与日本的侵略行为相联系{{r|王1990|page=13}}。例如1950年代有人批评协和语是一种“怪话”，是破碎的中国话，在话剧舞台上过多地使用协和语是一种畸形语言爱好{{r|黎1959}}。

== 皮钦语以外“协和语”一词的用法 ==
=== 指代书面中文 ===
满洲国时期，报纸等书面中文里也使用很多日语借词，被一些中国研究者称为“协和语”。例如“[[锦县|锦县]]无线电台<u>放送</u>开始”的“放送”意为广播。“电话使用<u>料</u>改正”的“料”意为费用{{r|王1990}}。有一些中文则在语法上也受日文影响，例如“优秀大型货物船热田丸大连着”（先进的大型运输船热田丸抵达大连），“日小铁工业满洲移驻”（日本的小型铁制部件工厂向满洲迁移）{{r|内田2017|page1=148}}。

语言学家陈原用“协和语的思想”形容满洲国时期受日文影响，在标点符号的用法、引语的用法等方面照搬日文标准的书面中文。{{r|樱井2012|page1=11,12|陈2001|page2=76}}

=== 指代汉语之外的语言 ===
[[周恩来|周恩来]]在1963年接见[[朝鲜科学院|朝鲜科学院]]代表团时，曾用“协和语”指代[[延边|延边]]地区受日语影响的[[朝鲜语|朝鲜语]]。{{r|周恩来1963}}

=== 对日本人所使用汉语的讽刺 ===
满洲国政府官吏中的中国人对日本人的汉语发音和日本本位的态度不满，讽刺地称之为协和语。{{r|樱井2012|page1=11}}

满洲国执政、皇帝[[溥仪|溥仪]]在回忆录《[[我的前半生|我的前半生]]》中把被迫使用的日本式汉语词称作协和语，例如“{{link-ja|御真影}}”（皇帝的照片，本应称为“御容”），也把报纸上的带有宣传意味的“赫赫战果”“堂堂入城”称为协和语。{{r|溥仪2013}}
