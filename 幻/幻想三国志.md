《'''幻想三国誌'''》是[[宇峻科技|宇峻科技]]开发的[[电子角色扮演游戏|电子角色扮演游戏]]，在[[中国大陆|中国大陆]]由[[寰宇之星|寰宇之星]]代理發行简体中文版，分别於2003年6月24日和27日在[[台湾|台湾]]和[[中国大陆|中国大陆]]发布。<ref name="Baha" /><ref name="Huanyu" />

本作主打的歷史事件為《[[赤壁之戰|赤壁之戰]]》。

本作標語：『前世的因，今生的果；永恆、無悔、痴心、愛戀，打破時空浪漫戀情』

== 故事情节 ==

游戏以[[三国时代|三国时代]]和远古时代的[[黄帝|黄帝]]、[[炎帝|炎帝]]和[[蚩尤|蚩尤]]之间的抗衡为背景。主角[[姬轩|姬轩]]的父母[[姬風|姬風]]、[[姜秀|姜秀]]在八年前神秘失踪，于是由[[徐庶|徐庶]]与其妹妹[[徐虹|徐虹]]收养，照顾[[姬轩|姬轩]]与[[姬霜|姬霜]]。两姐弟原本生活纯朴，但有一天突然发生一场莫名的追杀，从此他们就卷入了三国乱世以及远古时代的宿命纠葛。

== 角色介紹 ==

=== 主要角色 ===

; [[姬轩|姬轩]]
: 字伯雅，故事男主角。為[[姬風|姬風]]與[[姜秀|姜秀]]之子，夫婦倆於[[蚩尤|蚩尤]]復活事件失蹤後，[[徐庶|徐庶]]兄妹扶養成人。个性温良聪颖，喜讀[[兵書|兵書]]，能文能武。前生为[[黄帝|黄帝]]的部下-{[[風后|風后]]}-。
; [[姬霜|姬霜]]
: 字羽鳳，故事女主角，与姬轩是无血缘关系的姐姐，肩负照顾和保护姬轩的责任。真实身份是[[曹操|曹操]]庶女，因其[[太陰|太陰]]之女的身分，被視為不祥，而於出生之時遭拋棄。前生为[[炎帝|炎帝]]的女儿[[凤曦|凤曦]]。
; [[貂芷|貂芷]]
: 字芸真，[[貂蝉|貂蝉]]与[[吕布|吕布]]的女儿。呂布死後與其母隱居於[[隆中|隆中]]，於[[亂世英雄會|亂世英雄會]]中敗給姬軒，因誇下敗戰即嫁之豪語，為履行婚約而同行。視[[劉備|劉備]]與[[曹操|曹操]]為殺父仇人。
; [[糜香|糜香]]
: 因战乱自小成为孤儿，被[[諸葛亮|諸葛亮]]收养成为其贴身[[婢女|婢女]]。為[[天女|天女]]一族之後。
; [[燕起|燕起]]
: 漂泊各国、四海为家的流浪[[刀客|刀客]]。未婚妻[[蓮兒|蓮兒]]死于[[流民|流民]]之手，对长相与未婚妻相似的虞冰一见钟情。曾受過[[周瑜|周瑜]]之恩，與[[神意派|神意派]][[掌門|掌門]][[狂酒笑十八|狂酒笑十八]]為酒友。擊殺蚩尤後，與虞冰兩人共同重組天若宮。
; [[虞冰|虞冰]]
: [[天若宫|天若宫]]高层弟子，代[[執法|執法]]。與燕起已故之未婚妻蓮兒長相神似。擊殺蚩尤後，與燕起兩人共同重組天若宮。
; [[百玥|百玥]]
: 千年[[白狐精|白狐精]]所化，[[愛心筆三人組|愛心筆三人組]]的大姊，与糜香情同姐妹。與姬軒產有一子，於[[蚩尤|蚩尤]]一戰中重傷，經[[葛玄|葛玄]]之助，以[[九命魂轉術|九命魂轉術]]，以自身性命換取麋香之性命，自此魂飛魄散。
; [[温青|温青]]
: 饱读诗书、个性温文的长江富商之子。看似柔弱的他实为轩辕黄帝的转生，深爱着凤曦的转生姬霜。

== 繭印獸 ==

為[[幻想三國志系列|幻想三國志系列]]所特有之系統，類同於[[軒轅劍參外傳_天之痕|軒轅劍參外傳 天之痕]]之[[符鬼|符鬼]]系統。共分為『赤印』、『闇印』及『藏印』三種。

=== 赤印 ===

主司[[力量|力量]]及[[飛翔|飛翔]]，以攻擊為主體。

* [[文鸞|文鸞]]，[[仙界|仙界]]的[[珍鳥|珍鳥]]，[[吉祥|吉祥]]的[[象徵|象徵]]。傳聞雙翼全張時全長三十尺。於飛行時會流曳出五色彩光，促進萬物生長而得豐收。<ref name="UJ">{{cite web| language =zh-tw| publisher =宇峻奧汀| title =繭印獸| url =http://www.unistar.cn/company/history.html| author =| date =| accessdate =| deadurl =yes| archiveurl =https://web.archive.org/web/20140118083332/http://www.unistar.cn/company/history.html| archivedate =2014年1月18日| df =}}</ref>
* [[狙如|狙如]]，[[惡魔|惡魔]][[翅膀|翅膀]]、血紅雙眼，個性殘暴擁有強大破壞力。所到之處無不盡毀，人人避之唯恐不及。<ref name="UJ"/>
* [[九尾狐|九狐]]，身體優美輕盈，擁有象徵不死與魅惑的九條尾巴，行事難以預料，好惡分明，一旦動怒，強悍魔力難以匹敵。<ref name="UJ"/>

=== 藏印 ===

主司[[法力|法力]]及[[跳躍|跳躍]]，以術法為主體。

* [[光朐|光朐]]，以光為[[皮毛|皮毛]]覆蓋全身，行動迅速，雙腿強而有力，擅長高速打擊。額上的[[如意眼|如意眼]]，能使牠見到一般所看不到之事物。<ref name="UJ"/>
* [[冥蟾|冥蟾]]，混身透出冰冷的死亡氣息，擅長間接攻擊及高等[[妖術|妖術]]，[[百姓|百姓]]視之為不祥的死亡象徵。頭上之角能匯聚[[冥間|冥間]]的[[邪氣|邪氣]]，為其主要的力量來源。<ref name="UJ"/>
* [[邪軍|邪軍]]，外型如[[豹|豹]]、具三尾，彷彿來自[[地獄|地獄]]的暗紫光芒籠罩全身、邪氣逼人，移動迅如鬼魅，頭上金色[[單瞳|單瞳]]勾人[[魂魄|魂魄]]，凡見者輕者失明、精神瘋癲，重者七孔流血、立下[[黃泉|黃泉]]。<ref name="UJ"/>

=== 闇印 ===

主司[[防護|防護]]及[[水性|水性]]，以輔助為主體。

* [[啻羅大蛟|啻羅大蛟]]，[[民間|民間]]傳說自天上下凡的深海大[[蛟|蛟]]，通體碧青，一翻身便足以令風雲變色，浪捲萬丈。<ref name="UJ"/>
* [[延維|延維]]，半[[人|人]]半[[蛇|蛇]]，為[[女媧|女媧]]一族之後裔，性好和平不善武力，然若引其發怒，則具有撼天之威。<ref name="UJ"/>
* [[殤矍|殤矍]]，生長於深海中的巨大神[[龜|龜]]，因浮於海面上百年不動，其[[背甲|背甲]]部份形成了一座[[山島|山島]]，更有人築群居於其[[上背|上背]]。<ref name="UJ"/>

== 鳳凰誓 ==
{{main|鳳凰誓}}
為《幻想三國志》之[[資料片|資料片]]，劇情自[[許都|許都]]姬軒姊弟失散時開始，以姬霜為主角，講述在姬霜脫隊的期間所發生之事件。

=== 主要角色 ===

; [[雷卞|雷卞]]
: 姬霜之[[師傅|師傅]]，傳授姬霜[[招鬼術|招鬼術]]。明為蚩尤之助手，實為[[黃帝|黃帝]]手下大將[[力牧|力牧]]之轉世。
; [[紅夜|紅夜]]
: 半[[人|人]]半[[鬼|鬼]]，百年前為某大官之次女，因意外而身亡，白樺以雙眼為代價，使之得以續命。
; [[白樺|白樺]]
: [[白樺樹|白樺樹]][[樹精|樹精]]，因紅夜之[[要求|要求]]，而已人形行走於世。百年前以雙目為代價換取紅夜之性命。因曾受[[曹仁|曹仁]]之恩，而接下保護姬霜之[[任務|任務]]，與之同行。

=== 繭印獸 ===

* [[朱離|朱離]]，脾氣暴躁易怒，常為仙術道流所飼養，身上之毛銳利如刀劍，生氣時會以突刺展開攻擊。
* [[烏妣|烏妣]]，千年[[靈芝|靈芝]]精，受經年累月的滋養下幻化成人形，喜用[[幻術|幻術]]喬裝成替身模樣來嚇唬人。
* [[邑樞|邑樞]]，[[泥土|泥土]]所製成的[[土偶|土偶]][[機關人|機關人]]，外表溫和而呆楞。

== 游戏系统 ==

=== 画面 ===

游戏使用[[卡通|卡通]]与[[山水画|山水画]]相结合的美术风格。

=== 操作 ===

游戏的战斗模式可自由设置为回合制或半即时制。战斗时敌我双方各站在一个5乘5的布阵图上，玩家可以学习和使用各种阵型。

* 片尾曲：〈尽头〉，演唱：[[蔡順鵬|蔡順鵬]] 作詞：盧彥蓉 作曲：吳菁菁
