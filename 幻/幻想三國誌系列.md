'''幻想三國誌'''系列游戏，是由[[宇峻奧汀|宇峻奧汀]]公司出品的一款單機[[角色扮演遊戲|角色扮演遊戲]] (RPG)，其背景為[[三国|三国]]時期。本作於2003年6月24日發行第一部作品，並於2008年7月2日
<ref name="《幻想三國誌 4 外傳》全新官網開張 限量精裝版開放預購">{{cite web |url=http://gnn.gamer.com.tw/6/30706.html |title=《幻想三國誌 4 外傳》全新官網開張 限量精裝版開放預購|publisher=巴哈姆特電玩資訊站 新聞稿 |date=2008-05-30|accessdate=2015-08-13}}</ref>發行四代外傳，為本系列之最終作。

2015年11月19日[[宇峻奧汀|宇峻奧汀]]宣佈將於與广州叶游合作推出《幻想三國誌5》<ref name="手遊幻想三國誌5 兩岸攜手明年推出">{{cite web |url=http://www.cna.com.tw/news/afe/201511200073-1.aspx |title=手遊幻想三國誌5 兩岸攜手明年推出|publisher=中央社即時新聞 CNA NEWS |date=2015-11-19|accessdate=2016-05-13}}</ref>，並將以單機手遊方式推出。而後，幻想三國誌5手機版更名為《幻想三國誌：天環緣》<ref name="幻三5預計2016年推出！？">{{cite web |url=http://www.chinatimes.com/realtimenews/20151120002917-260410 |title=《科技》宇峻幻想三國誌5，授權廣州葉游營運、開發|publisher=中時電子報 |date=2015-11-20|accessdate=2015-11-29}}</ref><ref name="手游定名">{{cite web |url=http://shouyou.gamersky.com/news/201612/853610.shtml |title=《幻想三国志5》手游定名《幻想三国志：天環缘》 定档2017下半年|publisher=游民星空 |date=2016-12-30|accessdate=2017-01-24}}</ref>，另外，[[宇峻奧汀|宇峻奧汀]]宣布將與鳳凰遊戲合作，推出正統[[個人電腦|PC]]版續作「幻想三國誌5」<ref name="幻三5PC">{{cite web |url=http://www.gameapps.hk/news/20569 |title=PC單機正統新作《幻想三國志5》宣布開發！|publisher=GameApps.hk |date=2016-12-27|accessdate=2017-01-24}}</ref>。


本作的遊戲背景順序為：
[[幻想三國誌3|幻想三國誌3]] → [[幻想三国志2|幻想三國誌貳]] → [[幻想三国志|幻想三國誌]] → [[幻想三國誌4|幻想三國誌4]] ＆ [[幻想三國誌4外傳|幻想三國誌4外傳]]
