《'''幻想三國誌4外傳'''》是[[宇峻奧汀|宇峻奧汀]]開發的[[電子角色扮演遊戲|電子角色扮演遊戲]]，在中國大陸由[[寰宇之星|寰宇之星]]代理發行簡體中文版<ref name="大陸發行商">{{cite web |url=http://www.amazon.cn/mn/detailApp/ref=sr_1_2?_encoding=UTF8&s=videogames&qid=1258005261&asin=B002WGI0Y8&sr=8-2 |title= 幻想三国志4外传(DVD-ROM)|publisher=亞馬遜 amazon.cn |date=2009-11-25|accessdate=2015-08-16}}</ref>，分別於2008年7月2日<ref name="台灣發行日">{{cite web |url=http://acg.gamer.com.tw/acgDetail.php?s=12353 |title= 幻想三國誌4 外傳|publisher=巴哈姆特資料庫 |date=2008-07-02|accessdate=2015-08-16}}</ref>和2009年1月16日<ref name="大陸發行日">{{cite web |url=http://www.3dmgame.com/news/200911/12686.html |title= 《幻想三国志4外传》11月25日上市|publisher=3DM GAME |date=2009-11-25|accessdate=2015-08-16}}</ref>在台灣和中國大陸發布。

本作於2009獲得當年度巴哈姆特遊戲大賞的「人氣國產單機遊戲」金賞和「人氣電腦單機遊戲」銀賞<ref name="巴哈得獎">{{cite web |url=http://prj.gamer.com.tw/2009bhga/vote_open.html |title= 2009 巴哈姆特遊戲大賞|publisher=巴哈姆特電玩資訊站 |date=2009-06-18|accessdate=2015-08-16}}</ref>。並於2013年榮獲第十屆中國遊戲產業年會(CGIAC) 的「十大最受歡迎的單機遊戲」<ref name="幻想三國誌4外傳榮獲第十屆中國遊戲產業年會">{{cite web|url=http://www.userjoy.com.tw/005_awards.htm|title=幻想三國誌4外傳》榮獲第十屆中國遊戲產業年會(CGIAC)|publisher=宇峻奧汀 得獎紀錄|date=2013-12-31|accessdate=2015-08-16|archive-url=https://web.archive.org/web/20160304090623/http://www.userjoy.com.tw/005_awards.htm|archive-date=2016-03-04|dead-url=yes}}</ref>。

==資料片 ==
*本作資料片為名稱為《三界秘聞錄》。
*本資料片為繁體版攻略本之贈品，但簡體版產品內即包含本資料片。
*本資料片故事為眾主角小時候的故事，但宸晏無論於資料片或外傳中，皆以童身現身。

== 音樂 ==
遊戲主題曲《彩虹》和片尾曲《思念》皆由洪一平演唱。

歌曲並未在遊戲中出現，而是收錄於精裝版所附贈的《幻想三國誌 4 外傳全新製作典藏音樂 CD》。
