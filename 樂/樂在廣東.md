《'''樂在廣東'''》（{{lang-en|'''Map of Happiness GuangDong'''}}）是[[香港|香港]][[亞洲電視|亞洲電視]]新聞部資訊科製作的飲食旅遊節目，由[[劉子葱|劉子葱]]、[[吳夢婷|吳夢婷]]、[[涂諾|涂 諾]]擔任主持<ref>{{cite web |url=http://www.hkatv.com/events/guangdong/ |title=存档副本 |accessdate=2015-05-12 |deadurl=yes |archiveurl=https://web.archive.org/web/20150518084221/http://www.hkatv.com/events/guangdong/ |archivedate=2015-05-18 }}</ref>。本節目於2015年5月17日星期日、2015年5月18日至5月29日逢星期一至五20:00-20:30在[[本港台|本港台]]播映，以及在亞洲電視網頁上提供節目重溫<ref>{{cite web |url=http://www.hkatv.com/events/guangdong/?mod=promo |title=存档副本 |accessdate=2015-05-12 |deadurl=yes |archiveurl=https://web.archive.org/web/20150518084316/http://www.hkatv.com/events/guangdong/?mod=promo |archivedate=2015-05-18 }}</ref><ref>http://news.qoos.com/%E4%BA%9E%E8%A6%96%E7%B8%BD%E7%B5%90%E5%85%89%E8%BC%9D%E6%AD%B2%E6%9C%88-1899888.html</ref>。

==內容==
廣東是一個可以給人夢想、給予人幸福的地方。《樂在廣東》遊玩陌生而又親切的地方，看未曾見過的風景，賞最具粵韻的藝術文化，品最地道而豐富的美食，收藏最具特色的粵式之物，認識志同道合的人，開啟一段最輕鬆的旅程。<ref>{{cite web |url=http://forum.hkatv.com/v5/viewthread.php?tid=27106&extra=page%3D1 |title=存档副本 |accessdate=2015年5月12日 |deadurl=yes |archiveurl=https://web.archive.org/web/20150526132428/http://forum.hkatv.com/v5/viewthread.php?tid=27106&extra=page%3D1 |archivedate=2015年5月26日 |df= }}</ref><ref>http://www.on.cc/hk/bkn/cnt/entertainment/20150516/bkn-20150516042102369-0516_00862_001.html</ref>
