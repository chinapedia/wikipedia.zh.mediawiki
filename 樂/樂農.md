'''樂農'''（'''{{lang-en|Happy Veggies}}'''）是[[香港|香港]]第一家[[社會企業|社會企業]][[素食|素食]][[餐廳|餐廳]]<ref>{{cite web|language=zh-hk|url=http://www.hkdailynews.com.hk/news.php?id=132757|title=幫襯社企素食餐廳 特首辦推銷素菜|publisher=[[新報|新報]]|date=|accessdate=2011-06-12}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，其有兩家分店，分別在[[灣仔|灣仔]]及[[荃灣|荃灣]]，總店位於[[香港島|香港島]][[灣仔|灣仔]][[軒尼詩道|軒尼詩道]]99號[[彰顯大廈|彰顯大廈]]1字樓。

==成立由來==
[[電影演員|影視紅星]][[曾志偉|曾志偉]]和[[戚美珍|戚美珍]]為了協助殘疾人士就業<ref>{{cite web|language=zh-hk|url=http://www.hkdailynews.com.hk/enews.php?id=125196&scid=28|title=曾志偉開心投資素食館|publisher=[[新報|新報]]|date=|accessdate=2011-12-03|deadurl=yes|archiveurl=https://archive.is/20130107154432/http://www.hkdailynews.com.hk/enews.php?id=125196&scid=28|archivedate=2013-01-07}}</ref><ref>{{cite web |language=zh-hk |url=http://paper.wenweipo.com/2010/10/05/EN1010050038.htm |title=戚美珍助失聰人士就業 |publisher=[[文匯報_(香港)|文匯報 (香港)]] |date=2010-10-05 |accessdate=2011-06-12 |archive-url=https://web.archive.org/web/20111228092734/http://paper.wenweipo.com/2010/10/05/EN1010050038.htm |archive-date=2011-12-28 |dead-url=yes }}</ref>，透過[[香港影視明星體育協會慈善基金|香港影視明星體育協會慈善基金]]參加[[民政事務總署|民政事務總署]]推行之「[[伙伴倡自強|伙伴倡自強]]」社區協作計劃<ref>{{cite web |language=zh-hk |url=http://hk.apple.nextmedia.com/template/apple/art_main.php?iss_id=20101005&sec_id=462&subsec=830&art_id=14519178 | title=馮仁昭四圍超：志偉擬辦老人健身室 |publisher=[[蘋果日報_(香港)|蘋果日報 (香港)]] |date=2010-10-05 |accessdate=2012-01-07}}</ref>，申請開設社會企業素食餐廳<ref>{{cite web|language=zh-hk|url=http://www.singpao.com/newsarticle.aspx?NewsID=130059&Lang=tc|title=周末茶座：「樂農」素食館顯自強　半數員工聽障人士|publisher=[[成報|成報]]|date=|accessdate=2011-06-12}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，獲得到[[社聯－滙豐社會企業商務中心|社聯－滙豐社會企業商務中心]]的協助<ref>{{cite web |language=zh-hk |url=http://hk.news.yahoo.com/%E6%BB%99%E8%B1%90%E5%B9%AB%E7%A4%BE%E4%BC%81%E5%81%9A%E7%94%9F%E6%84%8F-223000719.html | title=滙豐幫社企做生意 |publisher=[[星島日報|星島日報]] |date= 2011-09-08 | accessdate=2011-09-08}}</ref><ref>{{cite web |language=zh-hk |url=http://www.facebook.com/video/video.php?v=10150264193529145&oid=143545595698458&comments | title=社會企業--樂農 |publisher=[[滙豐銀行慈善基金|滙豐銀行慈善基金]] |date=  |accessdate=2011-07-31}}</ref>。在[[2010年10月|2010年10月]]，便成功在灣仔成立香港第一家社會企業素食餐廳──樂農。

餐廳的名字，「樂」意指開開心心，「農」則取自[[聾|聾]]的[[諧音|諧音]]<ref>{{cite web |language=zh-hk |url=http://www.hkcd.com.hk/content/2010-12/06/content_2647831.htm |title=社企齊鋪樂農傷健共融 |publisher=[[香港商報|香港商報]] |date=2010-12-06 |accessdate=2011-06-12 }}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

==聘用聾人==
樂農聘用不少聾人<ref>{{cite web |language=zh-hk |url=http://hk.apple.nextmedia.com/template/apple_sub/art_main.php?iss_id=20110319&sec_id=12187363&subsec_id=38329&art_id=15088477 | title=佢有佢講：不用憐憫　只要機會 |publisher=[[蘋果日報_(香港)|蘋果日報 (香港)]] |date=2011-03-19 |accessdate=2011-12-03}}</ref><ref>{{cite web |language=zh-hk |url=http://www.takungpao.com.hk/news/11/03/13/GW-1353670.htm |title=無聲企堂招來爆棚生意 |publisher=[[大公網|大公網]] |date=2011-03-13 |accessdate=2011-12-03 }}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，他們都是透過慈善機構「[[龍耳|龍耳]]」聘請<ref>{{cite web |language=zh-hk |url=http://www1.hk.apple.nextmedia.com/template/apple/art_main.php?iss_id=20101219&sec_id=4104&art_id=14780972 | title=生活存關愛：首間素食餐廳聘用聽障人士 |publisher=[[蘋果日報_(香港)|蘋果日報 (香港)]] |date=2010-12-19 |accessdate=2011-06-12}}</ref><ref>{{cite web|language=zh-hk|url=http://www.singpao.com/newsarticle.aspx?NewsID=129007&Lang=tc|title=非牟利社企素食餐廳|publisher=[[成報|成報]]|date=|accessdate=2011-06-12}}{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。聾人員工均身穿綠色制服，而每張餐桌上都放了紙牌給食客溫馨提示，期望給予他們更多包容和耐性。如果食客遇到餐牌上不明白的事情，可以向身穿紅色制服的健聽員工求助或查詢<ref>{{cite web |language=zh-hk |url=http://www.mingpaohealth.com/cfm/health3.cfm?File=20101103/diet/vea1h.txt |title=專題飲食︰溫馨社企 無聲美味 全港首家社企素食餐廳 |publisher=[[明報|明報]] |date=2010-11-03 |accessdate=2011-06-12 }}{{Dead link|date=2018年7月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>。

由於曾志偉積極協助聾人就業<ref>{{cite web |language=zh-hk |url=http://orientaldaily.on.cc/cnt/entertainment/20101120/00282_023.html?pubdate=20101120 | title=曾志偉開餐飲聘聽障人士 |publisher=[[東方日報_(香港)|東方日報 (香港)]] |date=2010-11-20 |accessdate=2012-01-07}}</ref>，於2011年4月3日，獲得[[平等機會委員會|平等機會委員會]]及[[香港失明人互聯會|香港失明人互聯會]]聯合頒發「無障礙英雄」的榮譽<ref>{{cite web |language=zh-hk |url=http://the-sun.on.cc/cnt/news/20110404/00408_002.html?pubdate=20110404 | title=政情：金管局獲殘疾人士嘉許 |publisher=[[太陽報_(香港)|太陽報 (香港)]] |date=2011-04-04 |accessdate=2011-06-12}}</ref>。

==特色==
菜式方面以家庭素食為主，不加味精。沒有齋鴨、齋雞、齋燒鵝，卻有素餃子、花菇及素焗飯等食物<ref>{{cite web |language=zh-hk |url=http://news.hkheadline.com/dailynews/content_su/2010/11/27/130925.asp | title=上樓乍現 超巨型花菇現身 |publisher=[[頭條日報|頭條日報]] |date=2010-11-27 |accessdate=2011-06-12}}</ref><ref>{{cite web |language=zh-hk |url=http://news.sina.com.hk/news/23/1/1/2505029/1.html |title=少油少鹽無味精 貝安琪洪天明學手語食素 |publisher=[[星島日報|星島日報]] |date=2011-11-27 |accessdate=2011-12-03 }}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。此外，[[灣仔區議會|灣仔區議會]]文化及康體事務委員及[[聖雅各福群會|聖雅各福群會]]合辦「灣仔最喜愛食肆選舉2011」，該餐廳獲得票數最高的11間食肆之一<ref>{{cite web |language=zh-hk |url=http://www.hktdc.com/info/vp/a/emk/tc/2/6/1/1X07KHH4/%E6%96%B0%E8%88%88%E5%B8%82%E5%A0%B4/%E7%81%A3%E4%BB%94%E9%A3%B2%E9%A3%9F%E6%96%87%E5%8C%96%E5%9C%B0%E5%9C%96%E5%B8%B6%E8%B7%AF%E9%96%8B%E5%B1%95%E7%BE%8E%E9%A3%9F%E4%B9%8B%E6%97%85.htm | title=灣仔飲食文化地圖帶路開展美食之旅  |publisher=[[香港貿易發展局|香港貿易發展局]] |date= |accessdate=2011-08-18}}</ref>。
