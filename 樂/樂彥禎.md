'''樂彥禎'''（{{bd||？|888年|catIdx=L樂}}），本名'''樂行達'''，晚唐[[藩鎮|藩鎮]][[军阀|军阀]]，883年－888年取代了[[韩简_(唐朝)|韓簡]]，成為[[魏博節度使|魏博節度使]]，因治軍無方，[[士卒|士卒]]醞釀[[兵變|兵變]]，只好[[出家|出家]]為僧。其子[[相州|相州]][[刺史|刺史]][[乐从训|乐从训]]得知，憤而聯合[[宣武军节度使|宣武军节度使]][[朱全忠|朱全忠]]攻擊[[魏博|魏博]]，卻被[[留後|留後]][[罗弘信|罗弘信]]率兵[[斬殺|斬殺]]，樂彥禎也被連累而[[處死|處死]]。

== 背景 ==
乐行达是魏博首府[[魏州|魏州]]人，生年不详。父亲[[樂少寂|樂少寂]]是魏博的[[军官|军官]]，历任魏博下属的[[澶州|澶州]]、[[博州|博州]]、[[贝州|贝州]]三州刺史，赠[[工部尚书|工部尚书]]。乐行达年轻时也当上了魏博的军校。[[韓簡_(唐朝)|韩简]]任魏博节度使后，乐行达被提升为马步军都虞候，后转任博州刺史。[[中和_(唐朝)|中和]]二年（882年），韩简一度攻占[[河阳|河阳]]，迫使节度使[[诸葛爽|诸葛爽]]逃走，<ref name=ZZTJ255>[[:s:资治通鉴/卷255|《资治通鉴》卷二百五十五]]</ref>乐行达因在此战中有功，迁任澶州刺史。<ref name=BT181>[[:s:旧唐书/卷181|《旧唐书》卷一百八十一]]</ref><ref name=NBT210>[[:s:新唐书/卷210|《新唐书》卷二百一十]]</ref>

三年（883年）二月，韩简攻打[[天平|天平]]时，诸葛爽趁机突袭并收复了河阳。韩简闻讯，和天平留后[[朱瑄|朱瑄]]讲和，再次攻打河阳，被诸葛爽部将[[李罕之|李罕之]]所败。乐行达趁韩简战败，众心动摇，以言辞激牙军，抢先回师占领了魏州。<ref>[[:s:新唐书/卷187|《新唐书》卷一百八十七]]</ref>魏博士兵支持他为留后取代韩简，韩简也被部下所杀。<ref name=BT19-2>[[:s:旧唐书/卷19下|《旧唐书》卷一十九下]]</ref><ref>[[:s:旧唐书/卷182|《旧唐书》卷一百八十二]]作中和二年正月事。</ref><ref name=NBT9>[[:s:新唐书/卷9|《新唐书》卷九]]</ref>[[唐僖宗|唐僖宗]]认可乐行达为留后，授检校工部尚书，七月又加户部尚书，充节度观察处置等使。<ref name=ZZTJ255/><ref name=NBT210/>

== 任节度使 ==
四年（884年）正月，僖宗为乐行达赐名为樂彥禎，<ref name=ZZTJ255/>授检校尚书左僕射和[[同中書門下平章事|同中書門下平章事]]的[[宰相|宰相]]荣衔。<ref name=BT181/><ref name=NBT210/>

乐彦祯好儒，聘[[公乘亿|公乘亿]]、[[李山甫|李山甫]]入幕。其子[[樂從訓|樂從訓]]生性暴厉贪婪。十二月，前宰相[[王铎_(唐朝)|王鐸]]被任为[[义昌军节度使|义昌军节度使]]，途径魏博。时任义昌军节度使[[杨全玫|杨全玫]]闻王铎来代自己，告诉乐彦祯。乐彦祯迎谒王铎，宴劳很周到。樂從訓看上了王铎携带的大量侍妾和财物，纠集甘陵州卒数百人在[[漳南县|漳南县]]高鸡泊伏杀王铎及家属吏佐三百餘人，<ref name=BT19-2/>夺取了他的侍妾和财物。<ref>[[:s:旧唐书/卷164|《旧唐书》卷一百六十四]]</ref>乐彦祯报告朝廷称王铎被盗匪所杀，朝廷因深陷黃巢內亂，无力深究。<ref name=ZZTJ256>[[:s:资治通鉴/卷256|《资治通鉴》卷二百五十六]]</ref>魏博百姓素知王铎名望，为他的死而惋惜，<ref>[[:s:新唐书/卷185|《新唐书》卷一百八十五]]</ref>因此痛恨乐从训。<ref name=NBT210/>尽管如此，朝廷仍然授予乐彦祯[[開府儀同三司|開府儀同三司]]和[[司徒|司徒]]的荣衔。<ref name=BT181/>

[[光启|光启]]元年（885年）七月，邻镇[[昭义|昭义]][[洺州|洺州]]刺史[[馬爽|馬爽]]反叛节度使[[孟方立|孟方立]]，想迫使后者杀死与自己不合的行军司马[[奚忠信|奚忠信]]，不久兵败，逃往魏博。奚忠信向乐彦祯行贿，马爽被杀。<ref name=ZZTJ256/>

三年（887年）正月，宗室[[李煴|李煴]]稱帝，乐彦祯为魏博节度使、[[开府仪同三司|开府仪同三司]]、检校[[司空|司空]]、同中书门下平章事，派使[[李山甫|李山甫]]往见[[成德军节度使|成德军节度使]][[王镕|王镕]]，想合幽州、邢州、沧州诸镇同盟拒敌之，<ref>[[:s:致太原汴州两镇书|乐彦祯《致太原汴州两镇书》]]</ref>王镕以厚礼致谢，此事却没成功。<ref name=BT181/>彦祯见皇室衰微，有骄纵之心，他征召百姓修建魏州内城、重建[[黄河|黄河]]旧堤，限月内完工，引发民怨。<ref name=NBT210/>

== 隐退和去世 ==
[[文德|文德]]元年（888年）二月，魏博百姓不满乐彦祯的统治；乐从训聚集五百士兵作为心腹，称之为“子將”，也引起魏博的老兵的怨恨。乐从训察觉后，感到不安，逃离魏州。乐彦祯便任他为六州指挥使、[[相州|相州]][[刺史|刺史]]。乐从训又时常来信索取[[武器|武器]]、[[銀錢|銀錢]]、[[布疋|布疋]]等，越发激怒了魏博的老兵。乐彦祯曾梦见脱去朝服、带子、鞋子走路，醒后觉得是神告诉他将有下变。二月，老兵果然兵变，囚禁了乐彦祯，逼他在[[龍興寺|龍興寺]]出家为僧，<ref name=BT19-2/><ref name=NHFD1>[[:s:新五代史/卷01|《新五代史》卷一]]</ref>不久又杀之。<ref name=NBT210/><ref name=NBT9/><ref>[[:s:旧五代史/卷014|《旧五代史》卷一十四]]</ref><ref name=NHFD39>[[:s:新五代史/卷39|《新五代史》卷三十九]]</ref><ref>《旧唐书·乐彦祯传》作兵变之前乐彦祯已危愤而卒；《资治通鉴》作乐从训死后乐彦祯才被杀。此从《旧唐书·僖宗本纪》<昭宗本纪>《新唐书·僖宗本纪》《新唐书·乐彦祯传》。</ref>混乱中，邻镇的[[宣武节度使|宣武节度使]][[朱全忠|朱全忠]]派来魏博和乐彦祯商议粮食供给问题的使者押牙[[雷鄴|雷鄴]]被杀。<ref name=ZZTJ257>[[:s:资治通鉴/卷257|《资治通鉴》卷二百五十七]]</ref>

魏博士兵支持都将[[趙文㺹|趙文㺹]]为留后。乐从训得知父亲被迫隐退，率军三万进军魏州。趙文㺹拒绝和乐从训交战。魏博士兵因为不知趙文㺹意欲何为而杀了他，另推小校马牧监[[罗弘信|罗弘信]]为帅。<ref name=NBT9/>罗弘信击败乐从训，迫使他撤往[[內黃|內黃]]。魏博军包围內黃。乐从训希望藉魏博军杀死雷鄴一事向朱全忠求援。三月，朱全忠得讯，派都指挥使[[朱珍|朱珍]]等分兵北上，<ref name=BT19-2/>占领魏博[[黎阳|黎阳]]、[[临河|临河]]、[[李固|李固]]三镇，进军[[内黄|内黄]]，起初击败了魏博军，分遣[[聂金|聂金]]、[[范居实|范居实]]等掠[[澶州|澶州]]，在临黄杀魏博豹子军二千人。但四月当乐从训试图突围时，被罗弘信部将程公信（一作程公佐）袭杀。乐彦祯父子都被[[梟首|梟首]]魏博[[轅門|轅門]]示众。<ref name=NBT210/>罗弘信遣使向朱全忠道歉并送了礼物，朱全忠撤军，使罗弘信得以顺利接管了魏博。<ref name=NHFD1/><ref name=NHFD39/><ref name=ZZTJ257/><ref>[[:s:新五代史/卷021|《新五代史》卷二十一]]</ref><ref>[[:s:旧五代史/卷1|《旧五代史》卷一]]作乐彦祯失位、乐从训出奔相州求救皆在四月，朱珍攻取黎阳、临河等地后罗弘信才被推为帅。</ref>

乐彦祯生前封爵'''南阳郡王'''。<ref>《崔令尹颂德记》</ref>
