'''那曲站'''（{{bo|z=ནག་ཆུ།་འབབ་ཚུགས་|w=nag chu 'bab tshugs}}）位于[[中国|中国]][[西藏自治区|西藏自治区]][[那曲市|那曲市]][[色尼区|色尼区]]西面，距[[门地乡|门地乡]][[俄玛迪格村|俄玛迪格村]]约两公里，紧邻[[青藏公路|青藏公路]]，[[海拔|海拔]]4513米，于2006年7月1日启用<ref name="r1">中华建筑资讯网：〈[http://www.rpccn.com/news/33/200673162447.htm 藏域风格建筑群 璀璨“新天路”] {{webarchive|url=https://web.archive.org/web/20070930190113/http://www.rpccn.com/news/33/200673162447.htm |date=2007-09-30 }}〉，于2007年6月25日存取</ref>。该站为[[青藏铁路|青藏铁路]]的中转站和补给站，由[[青藏铁路公司|青藏铁路公司]]营运<ref name="r2">鼎言商旅网：〈[http://xz.d0086.com/SKM/sljt/xxlcz.htm 安多火车站]〉，于2007年6月25日存取</ref>。

== 车站设施 ==
那曲站车站广场占地约1万[[平方米|平方米]]<ref name="r3">美景旅游网：〈[http://blog.mjjq.com/archives/1260.html 青藏铁路沿途风光最美的站点：错那湖]〉，于2007年6月26日存取</ref>，站房总建筑面积7412平方米，分为站房综合楼、货运[[仓库|仓库]]等。其中站房综合楼建筑面积2540平方米。根据鼎言商旅网介绍，车站主入口以[[雨篷|雨篷]]造型设计，反映[[羌塘|羌塘]]地区毡房和[[雪山|雪山]]的形象<ref name="r2"/>。轨道两旁建有四十多米长的[[雨棚|雨棚]]，站台上铺上红黄相间的[[水泥砖|水泥砖]]<ref name="r3"/>。

== 使用情况 ==
那曲站是[[青藏铁路|青藏铁路]]的主要车站，全部14列旅客列车均停靠此站，主要为[[拉萨站|拉萨]]到发的各类旅客列车。

== 始发车次 ==
目前那曲站未有任何始发车次。

== 历史 ==
*2006年7月1日：那曲站建成启用。
