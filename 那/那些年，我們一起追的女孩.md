《'''那些年，我們一起追的女孩'''》（{{lang-en|''You Are the Apple of My Eye''}}）是一部於2011年上映的[[臺灣|臺灣]]青春[[愛情片|愛情片]]，劇情改編自臺灣作家[[九把刀|九把刀]]撰寫的半自傳同名小說《[[那些年，我們一起追的女孩_(小說)|那些年，我們一起追的女孩]]》<ref name = "Ho Yi">{{Cite web |url = http://www.taipeitimes.com/News/feat/archives/2011/08/19/2003511096 |title = Movie review: You are the Apple of My Eye 那些年，我們一起追的女孩 |author = Ho Yi |date = 2011-08-19 |publisher = 《[[台北時報|台北時報]]》 |language = en |accessdate = 2015-07-18 |quote =  }}</ref>，這部片也是九把刀首次執導的[[台灣電影|電影作品]]<ref>{{Cite web |url = http://twitchfilm.com/2011/10/china-beat-apple-of-my-eye.html |title = CHINA BEAT: How d'you like them apples? 這部電影在臺灣獲得許多電影評論人的歡迎，並在臺灣、香港和新加坡創下電影票房紀錄<ref name = "海峽時報"/><ref name = "王英敏"/>。這部電影獲選為2011年香港[[夏日國際電影節|夏日國際電影節]]的開幕電影，並成為香港華語電影史上最賣座的電影作品（2016年被港產片《[[寒戰II|寒戰II]]》取代）。在同一年，柯震東也因電影中的角色而獲頒第48屆金馬獎最佳新演員獎<ref name = "利小刀">{{Cite web |url = http://news.mtime.com/2011/11/26/1476390.html |title = 第48届金马奖完全获奖名单 |author = 利小刀 |date = 2011-11-26 |publisher = [[時光網|時光網]] |language = zh-hans |accessdate = 2015-07-18 |quote =  }}</ref>。在2012年，《那些年，我們一起追的女孩》則在[[第31屆香港電影金像獎|第31屆香港電影金像獎]]中獲選為最佳兩岸華語電影。

== 故事 ==
故事設定在1994年位於[[彰化縣|彰化縣]]的[[彰化縣私立精誠高級中學|彰化縣私立精誠高級中學]]，該校亦是導演兼編劇[[九把刀|九把刀]]的母校。故事圍繞著調皮的男主角柯景騰（綽號「柯騰」）發展，他在這間學校裡還有幾個好友，包括喜歡耍帥但在情場常常失意的曹國勝（綽號「老曹」）、想用搞笑致勝卻總是失敗的廖英宏（綽號「該邊」）、體型偏胖但成績好的謝明和（綽號「阿和」）等人。他們的共同點是喜歡班上成績優秀、相貌甜美、並廣受男學生和老師喜愛的女孩沈佳宜，在沈佳宜身旁則有好友胡家瑋（綽號「彎彎」）跟隨。

儘管柯景騰和沈佳宜在[[高中|高中]]時是同學，但柯景騰因成績糟糕、而被視為「不受控制的問題學生」，對沈佳宜也沒有興趣。有一天柯景騰在課堂上[[自慰|自慰]]被發現，結果被老師懲罰更換坐位，改坐到沈佳宜前面。後來[[英文|英文]]課時，柯景騰在發現沈佳宜沒帶英文課本後，決定把自己的課本偷偷交給她，並站起來告訴老師忘了攜帶課本，這讓他被老師痛罵並遭到[[體罰|體罰]]。沈佳宜因此被柯景騰感動，她為了表達感謝還特地準備「愛心考卷」以作回報。沈佳宜除了鼓勵柯景騰認真學習外，還說服他在[[課程|課程]]結束後一同留下來複習功課。在輔導柯景騰的過程中，兩人感情慢慢地增溫，柯景騰的成績也逐漸拉高。

畢業後，柯景騰在[[大學聯合招生考試|大學聯合招生考試]]考上[[國立交通大學|國立交通大學]]管理科學系，而朋友謝明和考上[[國立清華大學|國立清華大學]]經濟學系、廖英宏考上[[逢甲大學|逢甲大學]]資訊工程學系、曹國勝考上[[國立成功大學|國立成功大學]]化學工程學系；沈佳宜在大學聯合招生考試則因身體不適而表現失準，最後成績只能錄取[[國立臺北教育大學|國立臺北師範學院]]，為此她感到十分難過和沮喪。儘管兩人身隔兩地，柯景騰每天晚上都會排隊打[[公共電話|公共電話]]關心失落的沈佳宜。同年聖誕節假期，柯景騰前往臺北縣探望沈佳宜，兩人展開他們第一次「約會」。會面期間，柯景騰詢問沈佳宜是否喜歡著他；儘管沈佳宜已經想要給予正面答覆，但擔心被拒絕的柯景騰卻說不願意聽見答案。

後來柯景騰認為男生都要在女生面前表現自己最強的一面，便在學校舉辦了「自由格鬥賽」；他還邀請沈佳宜觀賽，希望藉此展現實力並打動她。但在比賽結束後，沈佳宜指責柯景騰行為幼稚，無法理解為何要毫無來由地舉辦把自己弄傷的比賽；柯景騰則認為沈佳宜不理解他想在她面前表現勇敢的一面，兩人因而大吵一架。最後在大雨中，腦袋混亂的柯景騰痛苦地放棄追求沈佳宜，兩人結束關係。分手後的2年內，柯景騰和沈佳宜彼此都沒有接觸，期間沈佳宜曾短暫和謝明和交往5個月。直到[[921大地震|921大地震]]那天，柯景騰立刻打電話給沈佳宜詢問狀況。在長時間的對談中，他們兩人感嘆沒有緣分能夠成為一對夫婦。

之後兩人恢復友誼，儘管兩人未能夠成為戀人，不過友情卻進一步昇華而成為永遠的好友。柯景騰大學畢業後考上[[東海大學_(台灣)|東海大學]]研究所，並開始在網際網路上創作小說。2005年，沈佳宜突然透過電話告訴柯景騰說她要結婚了，在婚禮上所有的老朋友聚集在一起，並開玩笑地計畫讓沈佳宜的丈夫難堪，不過他們發現過去的情感已轉化成深厚的友誼和祝福。後來眾人聚在一起祝賀新郎和新娘，並開玩笑地說應該要能親吻新娘；沈佳宜的丈夫則說任何想親新娘的人，都要先和他親吻。柯景騰立刻抓住新郎熱烈地親吻，甚至把他推倒在桌子上，如同會如此親吻沈佳宜般；在親吻的過程，過去成長和懊惱的回憶也逐一浮現。故事的最後，柯景騰與朋友們到打擊場打棒球，揮桿的同時，他們問道柯景騰什麼時候要開始寫小說，柯景騰說自己還沒想到題材。同時，畫面帶到咖啡店裡的柯景騰，對著筆記型電腦上，一字字地鍵入自己的心情，一部[[那些年，我們一起追的女孩_(書籍)|記錄著自己青春回憶的小說]]就此誕生。

== 製作 ==
=== 發展 ===
最初，香港導演[[馬偉豪|馬偉豪]]在原著問世的2005年，打算買下版權，拿來拍成電影。在那之前，馬導演拍的《[[記得...香蕉成熟時|記得...香蕉成熟時]]》[[記得...香蕉成熟時II初戀情人|續集]]，不光有描述少年成長歷程，還是九把刀在看過的香港電影中，欣賞的其中一部<ref name = "小野和陽光衛視">{{Cite book | author = [[小野|小野]]和[[陽光衛視|陽光衛視]] | title = 《翻滾吧，台灣電影》 | location = 臺灣臺北 | publisher = [[麥田出版|麥田出版]] | date = 2011-09-27 | pages =115-118, 120-123  | ISBN = 978-9861736662 | accessdate = 2011-09-27 | language = zh-hant}}</ref><ref>{{Cite web |url = http://hk.apple.nextmedia.com/entertainment/art/20111231/15939660 |title = 九把刀最愛港產片 |author =  |date = 2011-12-31 |publisher = 蘋果日報 |language = zh-hant |accessdate = 2017-08-25 |quote =  }}</ref><ref name = "萌特斯和洛安吉">{{Cite web |url = http://news.mtime.com/2011/11/10/1475048.html |title = 专访《那些年》导演九把刀：人生就是不停地战斗 |author = 萌特斯和洛安吉 |date = 2011-11-29 |publisher = [[時光網|時光網]] |language = zh-hans |accessdate = 2015-07-18 |quote =  }}</ref>。剛好，九把刀喜歡馬偉豪，也希望可以得到合作，而拍好小說上講的故事<ref name = "小野和陽光衛視"/>。原先，在寫這小說時，認為這會變成一部電影，使他變成這電影的編劇。所以，事先規劃好，與小說不同的電影版最後十分鐘結局<ref name = "小野和陽光衛視"/><ref name = "萌特斯和洛安吉"/>。接著，九把刀同經紀人柴智屏，跟導演討論小說授權。過程中，儘管對方開的授權價格是合理，但九把刀要求價錢再提高一點點，又加上雙方沒達到共識，致使對方無法取得版權，更造成九把刀在馬偉豪沒簽下合約之下，無法分享對於電影版結局的構想。經過五年的歲月，確定要跟柴智屏合作拍片後，才把那電影版結局概念，講給柴智屏了解<ref name = "小野和陽光衛視"/>。

=== 籌拍 ===
[[File:2008TIBE_Day3_Hall2_SigningStage_Giddens_in_Signing.jpg|thumb]]。]]

過去九把刀曾執導短片電影《愛到底》，後來想嘗試投入製作長片電影<ref name = "曝新预告青春无敌">{{Cite web |url = http://news.mtime.com/2011/07/18/1465097.html |title = 九把刀《我们一起追女孩》曝新预告青春无敌 |author = 9 |date = 2011-07-18 |publisher = [[時光網|時光網]] |language = zh-hans |accessdate = 2015-07-18 |quote =  }}</ref>。九把刀決定由他同時擔任編劇和導演，並找過去曾推出多部著名[[台灣偶像劇列表|偶像劇]]的電視製作人[[柴智屏|柴智屏]]擔任製片。起初《那些年，我們一起追的女孩》的預算極為吃緊<ref name = "萌特斯和洛安吉"/>，原本就是暢銷網路小說作家的九把刀也必須投入全部的積蓄，並抵押自己的房子籌錢<ref name = "萌特斯和洛安吉"/>。不過他說這樣做的目的，是為了打動提供這部電影靈感的前女友暨真實故事女主角沈佳儀<ref name = "萌特斯和洛安吉"/>。在籌措資金的過程中，擔任執行製片人的柴智屏也扮演關鍵角色<ref name = "萌特斯和洛安吉"/>。

《那些年，我們一起追的女孩》故事基於九把刀的同名自傳小說《[[那些年，我們一起追的女孩_(書籍)|那些年，我們一起追的女孩]]》而成，內容同樣描述九把刀年少時實際發生的浪漫故事<ref name = "Maggie Lee">{{Cite web |url = http://www.hollywoodreporter.com/review/you-are-apple-my-eye-238703 |title = You Are the Apple of My Eye: Film Review |author = Maggie Lee |date = 2011-09-11 |publisher = 《[[好萊塢報導|好萊塢報導]]》 |language = en |accessdate = 2015-07-18 |quote =  }}</ref>。九把刀曾在部落格上表示網友過去便欣賞他不愛說謊、有話直說的個性，他強調電影絕對會寫實並下流，但不會變成限制級電影<ref name = "曝新预告青春无敌"/>。在將原本故事改寫成電影劇本時，九把刀決定改變一些故事的內容，讓電影更具張力<ref>{{cite news |language = zh-hans |author = 郑净友 |coauthors =  |url =  |title = 为戏剧效果戏剧效果虚构情节 |publisher = 《[[我報|我報]]》 |pages = 第9頁 |date = 2011-11-02 |accessdate = 2015-07-18 }}</ref>。例如柯景騰和沈佳宜原本是在電話中爭吵，不過電影中則改成在大雨中發生<ref name = "陈妍希追求完美">{{cite news |language = zh-hans |author = 郑净友 |coauthors =  |url =  |title = 陈妍希追求完美 |publisher = 《[[我報|我報]]》 |pages = 第9頁 |date = 2011-11-02 |accessdate = 2015-07-18 }}</ref>。對此九把刀表示：「雖然有些原因使得電影中的事件有所變化，主要的故事情節仍保持不變。<ref name = "陈妍希追求完美"/>」當被問及是否受到台灣電影近年來票房成功的壓力影響，九把刀則說：「不，我比較有壓力的是這部電影是否好看、在票房上是否會獲得成功，以及它是否讓我變得更尷尬。此外如果這部電影並不好看，這對於一直努力工作的柯震東和陳妍希來說想必感到十分失望。<ref>{{cite news |language = zh-hans |author = 郑净友 |coauthors =  |url =  |title = 九把刀为青春"加油"加油加醋" |publisher = 《[[我報|我報]]》 |pages = 第9頁 |date = 2011-11-02 |accessdate = 2015-07-18 }}</ref>」

=== 角色遴選 ===
[[File:Michelle_Chen.jpg|thumb]]為中心，挑選其他參與電影拍攝的演員。]]

為了遴選飾演電影角色的劇組成員，製作團隊曾舉辦多次演員試鏡，其中陳妍希是第一位由導演九把刀確認的演員。九把刀在第一次會議上便被她吸引，認為陳妍希很像真正的沈佳宜；甚至願意自己投資製作，以確保陳妍希能在電影中演出<ref name = "陈妍希追求完美"/>。九把刀認為由於《那些年，我們一起追的女孩》可能是自己拍攝的唯一一部電影作品，內容又講述自己過去的故事，因此希望能和女主角有足夠默契並喜愛她<ref name="關於選角">{{Cite web|url =http://app.atmovies.com.tw/movie/movie.cfm?action=extend&exid=fytw9139039103|title =關於選角|author =|date =|publisher =[[開眼電影網|開眼電影網]]|language =zh-hant|accessdate =2015-07-18|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20120125060222/http://app.atmovies.com.tw/movie/movie.cfm?action=extend&exid=fytw9139039103|archivedate =2012-01-25}}</ref>。陳妍希過去曾在《[[換換愛|換換愛]]》、《[[不良笑花|不良笑花]]》等台灣電視連續劇主演<ref name = "陈妍希追求完美"/>，後來在2009年時與[[彭于晏|彭于晏]]、[[陳意涵|陳意涵]]合作拍攝電影《[[聽說|聽說]]》，並入圍金馬獎最佳新人提名。柴智屏本人也認為她可塑性極高，有意將她安排成為[[徐熙媛|徐熙媛]]的接班人<ref name = "曝新预告青春无敌"/>。陳妍希為了這部片特別節食減肥<ref name = "陈妍希追求完美"/>，希望能夠不辜負導演的期待<ref name = "陈妍希追求完美"/>。

後來九把刀以陳妍希為中心選擇其他劇組成員，希望其他角色能與女主角搭襯，而甄選男主角的過程花費了最多時間<ref name = "關於選角"/>。拍攝團隊舉辦一系列試鏡並吸引許多名人參與，不過九把刀最後選擇第一次登臺演出的演員柯震東。過去柯震東儘管外型高挑清秀，主要參與演出的作品仍以廣告為主<ref name = "曝新预告青春无敌"/>。九把刀挑選柯震東的原因在於後者每次連續試鏡時，在演技上都有極大的改善，並且不會怯場；同時看到柯震東試鏡前躲在角落研究劇本的舉動，使得九把刀很喜歡柯震東的演出態度<ref name = "關於選角"/>。儘管柯震東的演技仍未能夠證實，但是柴智屏亦表達支持後，喜歡柯震東為人的九把刀決定嘗試由他擔任男主角。

另外九把刀還選擇了[[莊濠全|莊濠全]]、[[鄢勝宇|鄢勝宇]]、[[郝劭文|郝劭文]]和[[蔡昌憲|蔡昌憲]]飾演他的高中同學<ref name = "演員介紹"/>，他形容過去曾和[[釋小龍|釋小龍]]合作的郝劭文是口才優秀的講者、蔡昌憲是容易相處的搞笑人才，莊濠全則對於演出有著自信<ref name = "關於選角"/>。九把刀還邀請[[彎彎|彎彎]]飾演自己10幾歲的角色，他形容兩人如同「出版界的周杰倫和蔡依林」<ref name = "關於選角"/><ref name="亞洲電影資訊">{{Cite web |url = http://www.filmbiz.asia/reviews/you-are-the-apple-of-my-eye |title = You Are the Apple of My Eye 那些年，我們一起追的女孩。 |author =  |date = 2011年 |publisher = {{Link-en|亞洲電影資訊|Film Business Asia}} |language = zh-hant |accessdate = 2015-07-18 |quote =  |deadurl = yes |archiveurl = https://web.archive.org/web/20111127133129/http://www.filmbiz.asia/reviews/you-are-the-apple-of-my-eye |archivedate = 2011-11-27 }}</ref>。九把刀的母親則希望電影中扮演自己的演員由[[王彩樺|王彩樺]]或者是[[黃嘉千|黃嘉千]]負責，最後九把刀邀請當時沒有接洽其他工作的王彩樺飾演母親的角色。柯震東真實的父親柯義浤則在電影中扮演柯景騰的父親，為了消除兒子的演出壓力，他還陪兒子一同脫光演出<ref name = "關於選角"/>，只是在九把刀的勸說下僅裸背出鏡<ref name = "自由時報-柯義浤裸背出鏡">{{cite news |language = zh-tw |author = 鄒念祖 |coauthors =  |url = http://ent.ltn.com.tw/news/paper/564895 |title = 星爸挺愛子 陪裸入鏡那些年 |publisher = 《[[自由時報|自由時報]]》 |pages =  |date = 2012-03-02 |accessdate = 2018-01-23 }}</ref>。

=== 電影拍攝 ===

《那些年，我們一起追的女孩》由[[周宜賢|周宜賢]]擔任攝影師，這也是他首次擔任電影長片攝影師，並成功打造出青春唯美的新清風格畫面<ref name = "曝新预告青春无敌"/>。電影主要是在九把刀和沈佳儀就讀的[[彰化縣私立精誠高級中學|彰化縣私立精誠高級中學]]拍攝<ref name="關於場景">{{Cite web|url =http://app.atmovies.com.tw/movie/movie.cfm?action=extend&exid=fytw9139039104|title =關於場景|author =|date =|publisher =[[開眼電影網|開眼電影網]]|language =zh-hant|accessdate =2015-07-18|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20120123174708/http://app.atmovies.com.tw/movie/movie.cfm?action=extend&exid=fytw9139039104|archivedate =2012-01-23}}</ref>，整個拍攝過程投入新台幣5,000萬元的預算<ref name = "海峽時報"/>。擔任導演的九把刀表示之所以選擇過去就讀的學校拍攝，因為「太想看到柯景騰與沈佳宜穿上我記憶中制服的模樣了」<ref name = "關於場景"/>。由於主要場景設定在學校校園內，讓許多畫面的拍攝只能在高級中學放假期間進行<ref name = "關於場景"/>。

製作團隊也決定把電影中許多次要場景，安排至九把刀過去成長的[[彰化縣|彰化縣]][[彰化市|彰化市]]各處拍攝<ref name = "關於場景"/>，包括有[[永樂觀光夜市商圈|永樂觀光夜市商圈]]、[[老擔阿璋肉圓|老擔阿璋肉圓]]、[[大佛風景區|大佛風景區]]、中華陸橋等場景<ref>{{Cite web |url = http://news.mtime.com/pix/2011/06/23/65996.html |title = 光影自由行” 40部电影走遍宝岛台湾 |author = menteuse |date = 2011-06-28 |publisher = [[時光網|時光網]] |language = zh-hans |accessdate = 2015-07-18 |quote =  }}</ref>。但因為交通和經費問題，使得拍攝團隊放棄前往[[新竹市|新竹市]]取景，選擇在臺北地區的學校並將場景布置成[[國立交通大學|國立交通大學]]的氛圍拍攝，包括在[[輔仁大學|輔仁大學]]、[[國立臺灣科技大學|國立臺灣科技大學]]、[[國立臺北教育大學|國立臺北教育大學]]和[[德明財經科技大學|德明財經科技大學]]等校園內<ref>{{Cite web |url = http://www.giddens.tw/blog/post/33681439 |title = 那些年的電影實記（2）遺憾捨新竹，集中攻台北！ |author = [[九把刀|九把刀]] |date = 2010-07-20 |publisher = 再度參見，九把刀 |language = zh-hant |accessdate = 2015-07-18 |quote =  |deadurl = yes |archiveurl = https://web.archive.org/web/20111227052434/http://www.giddens.tw/blog/post/33681439 |archivedate = 2011年12月27日 |df =  }}</ref><ref>{{Cite web |url = http://www.giddens.tw/blog/post/35452305 |title = <那些年，我們一起追的女孩>電影創作特輯（12）無差別自由格鬥賽 |author = [[九把刀|九把刀]] |date = 2011-08-01 |publisher = 再度參見，九把刀 |language = zh-hant |accessdate = 2015-07-18 |quote =  |deadurl = yes |archiveurl = https://web.archive.org/web/20120124135513/http://www.giddens.tw/blog/post/35452305 |archivedate = 2012年1月24日 |df =  }}</ref>。而為了電影中出現的一些場景，劇組團隊還前往[[新北市|新北市]]的[[平溪老街|平溪老街]]和[[菁桐車站|菁桐車站]]、[[臺北市|臺北市]]的[[西門町|西門町]]、[[臺中市|臺中市]]的[[成功車站|成功車站]]等地取景。

=== 主題曲 ===
在電影製作的過程中，九把刀還參與主題音樂的創作，包括後來廣受民眾歡迎的主題曲《[[那些年|那些年]]》<ref name = "南方都市報"/>。最初九把刀無法選出心目中作為電影結局的片尾曲，在聽到日本作曲家[[木村充利|木村充利]]長達6分12秒的新作品後，他選擇該曲並親自創作歌曲的歌詞，之後由[[超級星光大道_(第六屆)|第六屆]]《[[超級星光大道|超級星光大道]]》冠軍歌手[[胡夏|胡夏]]演唱<ref name = "南方都市報"/>。《寂寞的咖啡因》則是九把刀與沈佳儀兩人仍然關係密切時，特別為後者作詞和作曲的原創歌曲<ref name="關於電影歌曲">{{Cite web|url =http://app.atmovies.com.tw/movie/movie.cfm?action=extend&exid=fytw9139039105|title =關於電影歌曲|author =|date =|publisher =[[開眼電影網|開眼電影網]]|language =zh-hant|accessdate =2015-07-18|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20120125060227/http://app.atmovies.com.tw/movie/movie.cfm?action=extend&exid=fytw9139039105|archivedate =2012-01-25}}</ref>。後來他詢問飾演男主角的柯震東能否在電影中演唱這首歌，因為覺得能夠傳達歌曲原本的涵義<ref name = "關於電影歌曲"/>。另外電影中使用的歌曲《孩子氣》是由陳妍希創作歌曲和歌詞，並由她親自歌唱。九把刀表示第一次聽到這首歌後，當場感動地流下眼淚，並特別喜歡這首歌的歌詞。他還稱讚陳妍希完全為自己的角色付出心力，表示：「我認為她這麼做的原因是設法得到靈感，因為她喜歡她（在這部電影中）的角色而寫下這首歌。」

電影主題曲《那些年》發表後不久便大受歡迎<ref name = "南方都市報"/>，在[[YouTube|YouTube]]上的音樂影片於2011年11月11日的點擊率突破1,000萬人<ref name = "南方都市報"/>；對於歌曲廣泛獲得民眾的注意，九把刀表示：「（這次是）破了所有華人音樂史上在YouTube上所有可以被記錄的數字。<ref name = "南方都市報"/>」在臺灣地區的[[KKBOX|KKBOX]]單曲每日排行榜上，《那些年》從2011年8月22日至10月22日，持續64天維持在第一名的位置<ref name = "南方都市報"/>，打破原先[[嚴爵|嚴爵]]連續45天奪冠的紀錄<ref name = "南方都市報"/>。在香港[[903專業推介|903專業推介]]上，歌曲獲得了第一名的成績。<ref>{{cite web|url=http://www.881903.com/Page/ZH-TW/index.aspx|title=881903.com 主页|publisher=[[903專業推介|903專業推介]]|archiveurl=https://web.archive.org/web/20111029082423/http://www.881903.com/Page/ZH-TW/index.aspx|archivedate=2011-10-29|accessdate=2017-02-22}}</ref>《那些年》在第48屆金馬獎被提名為最佳原創電影音樂<ref name = "蔡世伟">{{Cite web |url = http://news.mtime.com/2011/07/01/1463649.html |title = 台北电影节青年导演竞赛 九把刀获观众票选奖 |author = 蔡世伟 |date = 2011-07-01 |publisher = [[時光網|時光網]] |accessdate = 2017-02-22 |quote =  }}</ref>，在[[第23屆金曲獎|第23屆金曲獎]]上被提名為最佳年度歌曲獎。<ref name="金曲奖1">{{Cite web |url = http://www.bamid.gov.tw/files/15-1000-565,c138-1.php |title = 第23屆金曲獎流行音樂類入圍名單 |publisher = 文化部影視及流行音樂產業局 |accessdate = 2017-02-22 |quote =  |deadurl = yes |archiveurl = https://web.archive.org/web/20170222195022/http://www.bamid.gov.tw/files/15-1000-565,c138-1.php |archivedate = 2017-02-22 }}</ref>歌曲也贏得許多獎項，其在第23屆金曲獎上使歌曲的制作人[[薛忠銘|薛忠銘]]获得了最佳單曲製作人獎，<ref name="金曲奖">{{Cite web |url = http://www.bamid.gov.tw/files/15-1000-567,c138-1.php |title = 第23屆金曲獎流行音樂類得獎名單 |publisher = 文化部影視及流行音樂產業局 |accessdate = 2017-02-22 |quote =  |deadurl = yes |archiveurl = https://web.archive.org/web/20160304055424/http://www.bamid.gov.tw/files/15-1000-567,c138-1.php |archivedate = 2016-03-04 }}</ref>在香港商業電台舉辦的2011年度[[叱咤樂壇流行榜頒獎典禮|叱咤樂壇流行榜頒獎典禮]]中，歌曲贏得專業推介第二名和[[2011年度叱咤樂壇流行榜頒獎典禮得獎名單|我最喜愛的歌曲大獎]]。<ref>{{Cite web |url = https://www.kkbox.com/hk/tc/column/features-24-631-1.html |title = 2011年度叱咤樂壇流行榜頒獎典禮  |date = 2012-01-03 |publisher = [[KKBOX|KKBOX]] |accessdate = 2017-02-22 }}</ref>

== 發行 ==
=== 電影審查 ===
《那些年，我們一起追的女孩》由於部分的爭議內容，使得整部電影經過多次剪接，並發表多個不同版本。在[[臺灣|臺灣]]方面，最初[[行政院新聞局|行政院新聞局]]以電影涉及未成年的性行為，認定內容實際上屬於電影分級的「限制級」<ref name = "KingMedia影音台">{{Cite web |url = http://movie.kingmedia.com.tw/media_news/?act=movie_news&r=1310383699 |title = 《那些年，我們一起追的女孩》以輔導級過關 |author =  |date = 2011-07-11 |publisher = KingMedia影音台 |language = zh-hant |accessdate = 2015-07-18 |quote =  }}</ref>。九把刀則感到十分懊惱，甚至親​​自前往行政院新聞局辦公室提出上訴<ref name = "KingMedia影音台"/>。之後為了降低年齡分級而重新剪輯4次<ref name = "KingMedia影音台"/>，在經過審查後電影列入允許12歲以上兒童都能觀賞的「輔導級」類別<ref name = "KingMedia影音台"/>。[[香港|香港]]和[[澳門|澳門]]地區將電影列入「[[香港電影分級制度|IIB]]」分級，儘管實際上對於電影並無年齡限制，但認為該片不適合青少年和兒童、建議家長予以指導。{{Link-en|馬來西亞電影審查委員會|Film Censorship Board of Malaysia}}則將《那些年，我們一起追的女孩》列為限制級電影，要求觀眾超過18歲才能進入電影院觀賞。[[馬來西亞|馬來西亞]]上映的版本內容僅刪減約1分鐘，許多敏感的場景仍遵照原始版本予以保留，只有課堂[[自慰|自慰]]的部分場景遭到刪除<ref name="简芳">{{Cite web|url =http://gcontent.oeeee.com/c/91/c913303f392ffc64/Blog/958/bde858.html|title =去影院看《那些年》吧，虽然可能——大陆版107分钟=删减6处+添加素材|author =简芳|date =2011-12-17|publisher =《[[南方都市報|南方都市報]]》|language =zh-hans|accessdate =2015-07-18|quote =|deadurl =yes|archiveurl =https://www.webcitation.org/6C6kJueLL?url=http://gcontent.oeeee.com/c/91/c913303f392ffc64/Blog/958/bde858.html|archivedate =2012-11-12}}</ref>。《那些年，我們一起追的女孩》在新加坡上映則沒有經剪輯處理，電影被列為16歲以下觀眾禁止觀看電影的「NC-16」評級<ref name = "简芳"/>。

由於[[中國大陸|中國大陸]]沒有[[電影分級制度|電影分級制度]]，這讓電影為了能在中國大陸發行而要對忌諱片段做大量剪輯<ref name = "简芳"/>。總共有6個場景因為涉及色情、暴力和髒話等負面內容，和有關親近臺灣等常規政治問題而遭刪減或者修改<ref name = "简芳"/><ref name = "负面能量被剪掉了">{{cite news |language = zh-hans |author =  |coauthors =  |url =  |title = 九把刀：负面能量被剪掉了 |publisher = 《[[聯合早報|聯合早報]]》 |pages = 第5頁 |date = 2012-01-11 |accessdate = 2015-07-18 }}</ref>。包括高中校園升旗場景的[[中華民國國旗|中華民國國旗]]、柯景騰在高中上課時自慰、柯景騰與父親在自家全裸和擁抱、柯景騰在家中自慰的暗示內容、大學室友觀看色情片、男同性戀在大學浴室裡性愛等畫面遭到刪減，柯震東親吻新郎的段落也經剪輯<ref name = "简芳"/>。內容修改的部分，包括柯景騰原先說的「老曹是個很『雞巴』的人」台詞被改為 「老曹是個很『機車』的人」、柯震東與鄰居對罵的「Fuck You」中改成「Love You」、把「勃起」外號完全用「許博淳」名字代替，以及中國大陸偷渡客更改設定為越南人等。

這些劇情內容遭到剪輯後，擔任導演的九把刀不得不增加新的場景段落，以讓電影能流暢觀賞<ref name = "简芳"/>。最後在中國大陸上映的《那些年，我們一起追的女孩》版本，比台灣上映的原始版本刪減約3分鐘，不過在中國大陸的網站上亦有未刪減版流傳。對於中國大陸對電影審查，並以「負面性愛和親台內容」為由要求刪減許多內容，九把刀則是相當不滿<ref name = "负面能量被剪掉了"/>。他後來向中國大陸的觀眾道歉並指責自己，表示暫時無法兌現自己原本能呈現所有故事的諾言<ref name = "负面能量被剪掉了"/><ref>{{Cite web |url = http://news.mtime.com/2012/01/12/1479762.html |title = 《那些年》小说发内地版 九把刀借电影说抱歉 |author = 田超 |date = 2012-01-12 |publisher = [[時光網|時光網]] |language = zh-hans |accessdate = 2015-07-18 |quote =  }}</ref>，同時補充說：「不認為中國大陸版本（比起其他海外版本）更好。<ref name = "负面能量被剪掉了"/>」而在2012年1月7日，九把刀在電影網站上表示自己沒有看過被刪減後的最終上映版本。之後他對自己的失信再度道歉：

=== 作品推廣 ===
《那些年，我們一起追的女孩》在電影正式上映前，於2011年6月25日至7月2日的第13屆[[台北電影節|台北電影節]]首次播出。在更早之前的5月4日，《那些年，我們一起追的女孩》被台北電影節列入44部入選片單中，其他入選的電影還有《[[當愛來的時候|當愛來的時候]]》、《[[翻滾吧！阿信|翻滾吧！阿信]]》、《[[熊熊愛上你|熊熊愛上你]]》、《[[我的拼湊家庭|我的拼湊家庭]]》等<ref>{{Cite web |url = http://news.mtime.com/2011/05/05/1458356.html |title = 台北电影节公布入选名单 44部电影逐百万首奖 |author =  |date = 2011-05-05 |publisher = [[時光網|時光網]] |language = zh-hans |accessdate = 2015-07-18 |quote =  }}</ref>。在6月30日舉辦的國際青年導演競賽上，九把刀邀請柯震東和陳妍希共同上臺接受觀眾票選獎。隨後九把刀表示：「太高興了，平常我很臭屁，但得這座獎很難，所以我很開心，而且一定要和重要的兩個人一起上台，這是我第一部劇情長片，不敢說拍片是夢想，但拍這部片真的是我的夢想，這是一部充滿愛的電影，感謝兩位[[執行導演|執行導演]]，和我一起拍出這麼棒的電影，我要與他們一起分享這座獎的榮耀。<ref name = "蔡世伟"/>」另一方面，電影正式上映前便先行報名參加2012年的[[第48屆金馬獎|第48屆金馬獎]]<ref name = "利小刀"/>。

=== 評價 ===
《[[好萊塢報導|好萊塢報導]]》記者Maggie Lee描述電影為「打鬧、懷舊的成年甜點」<ref name = "Maggie Lee"/>，稱讚這部電影「從不斷自嘲的男孩角度，注入新鮮、酸甜的特點至風格中」<ref name = "Maggie Lee"/>；同時她表示這部電影不同於過去亞洲的青少年電影往往「甜膩」，她提到：「年輕的演員儘管只有有限的表達能力，但已經能夠自覺地呈現出不在乎的態度。<ref name = "Maggie Lee"/>」另外她也認為電影的風格略為明亮<ref name = "Maggie Lee"/>。《[[綜藝_(雜誌)|綜藝]]》評論人羅素·愛德華（Russell Edwards）則批評電影的第二段落，認為「無法維持第一個小時那種男性觀點的古怪幽默」<ref name = "羅素·愛德華">{{Cite web |url = http://variety.com/2011/film/reviews/you-are-the-apple-of-my-eye-1117946448/ |title = Review: ‘You Are the Apple of My Eye’ |author = 羅素·愛德華（Russell Edwards） |date = 2011-10-26 |publisher = 《[[綜藝_(雜誌)|綜藝]]》 |language = en |accessdate = 2015-07-18 |quote =  }}</ref>；同時愛德華還批評電影的最後段落，認為「九把刀高估了自身故事的魅力」<ref name = "羅素·愛德華"/>。不過愛德華稱讚電影的演員陣容，並認為這是電影「最重要的資產」<ref name = "羅素·愛德華"/>，他還稱讚這部電影是「臺灣過去10幾年來，許多同類型之虛構青年故事中最為成熟的一部」<ref name = "羅素·愛德華"/>。

新加坡報紙《{{Link-en|今天 (新加玻報紙)|Today (Singapore newspaper)|今天}}》評論人林詩琳（Serene Lim）則認為電影是「十幾歲的浪漫溫柔故事」，她願意在滿分5分中、對於整部電影給予3分的成績，表示：「九把刀作為小說家的才能是顯而易見的。<ref name="林詩琳" />」不過注重細節部分必須予以寬容<ref name = "林詩琳">{{cite news |language = en |author = 林詩琳（Serene Lim） |coauthors =  |url =  |title = Movie Review: You Are The Apple Of My Eye / NC16, 110min |publisher = 《{{Link-en|今天 (新加玻報紙)|Today (Singapore newspaper)|今天}}》 |pages =  |date = 2011-11-09 |accessdate = 2015-07-18 }}</ref>。林詩琳對於柯震東的演出則提出極多好評，她認為柯震東足以在金馬獎中被提名為最佳新人，並成功演出浮躁且叛逆的角色轉變的故事<ref name = "林詩琳"/>。新加坡《[[我報|我報]]》作家徐湧雄（Yong Shu Hoong）同樣願意在滿分5分的情況下給予3分，他認為電影中穿插[[倒敘_(敘述)|回顧過去]]的安排過於多愁善感，但在評論中補充說：「高中戀情的驚險刺激、激烈競爭與心碎痛苦都以懷舊和幽默手法好好地描繪出來。<ref>{{Cite web |url = http://news.asiaone.com/News/Latest+News/Showbiz/Story/A1Story20111110-309688.html |title = Movie review: You are the Apple of my Eye |author = 徐湧雄（Yong Shu Hoong） |date = 2011-11-10 |publisher = {{Link-en|AsiaOne|AsiaOne}} |language = en |accessdate = 2015-07-18 |quote =  }}</ref>」

《{{Link-en|亞洲電影資訊|Film Business Asia}}》在滿分10中願意給予7分的電影評級<ref name = "亞洲電影資訊"/>。其中著名影評人{{Link-en|艾德禮 (影評人)|Derek Elley|艾德禮}}描述電影有著「大膽的劇情」，但巧妙地安排每個環節，尤其後者更讓「容易被忽略的事實是原來這裡並沒有什麼東西」<ref name = "亞洲電影資訊"/>。他稱讚劇組人員是「精心地個別挑選，又能輕鬆地共同合演」，不過也補充說：「除了下半部分略為拖戲外，在共同導演廖明毅剪輯下整部影片在近2個小時維持一定節奏。<ref name = "亞洲電影資訊"/>」最後他總結電影的情節為「簡單的十幾歲愛情喜劇，在兩人會在一起、或不會在一起這兩個對立選項間遊擺，但最後是透過簡潔的結局予以致敬，這確實帶來一些真實的情感」<ref name = "亞洲電影資訊"/>。

中國大陸則方面，本片成為最流行​​的台灣電影，票房紀錄超越2008年時由《海角七號》創下的紀錄<ref name = "海盗王">{{Cite web |url = http://news.mtime.com/2012/01/10/1479657.html |title = 《金陵十三钗》内地四连冠 《那些年》获季军 |author = 海盗王 |date = 2012-01-10 |publisher = [[時光網|時光網]] |language = zh-hans |accessdate = 2015-07-18 |quote =  }}</ref>。儘管面臨相較於其他電影而晚半年上映、網際網路上盜版影片盛行，以及恰逢《[[金陵十三釵|金陵十三釵]]》、《[[龍門飛甲_(電影)|龍門飛甲]]》、《[[不可能的任務：鬼影行動|不可能的任務：鬼影行動]]》等大投資電影安排在同檔期的影響，電影首次上映地當周仍成為票房收入第三高的電影，總票房收入約[[人民幣|人民幣]]2700萬元<ref name = "海盗王"/>，最终在中國大陸地區創下人民幣7650萬元的紀錄<ref>{{Cite web |url = http://www.cbooo.cn/m/595206 |title = 那些年，我们一起追的女孩  |publisher =中国票房 |accessdate = 2015-07-18 |quote =  }}</ref>。
