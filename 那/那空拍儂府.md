'''那空拍儂府'''（{{thai|t=จังหวัดนครพนม|r=Changwat Nakhon Phanom}}，{{IPA-th|t͡ɕāŋ.wàt ná(ʔ).kʰɔ̄ːn pʰā.nōm}}）是[[泰國|泰國]][[依善地區|東北區]][[泰國行政區劃|府]]之一。鄰府從南起順時針依序為[[穆達漢府|穆達漢府]]、[[沙功那空府|沙功那空府]]和[[汶干府|汶干府]]，該縣東北方與[[寮國|寮國]][[甘蒙省|甘蒙省]]交界。

== 地理 ==

本府座落在[[湄公河|湄公河]]谷。重要水源除[[湄公河|湄公河]]外，還有[[松琴河|松琴河]]。

== 歷史 ==
那空拍儂意指“山城”，由[[拉瑪一世|拉瑪一世]]命名之。不過事實上那空拍儂境內並沒有山，石灰山都集中在位於[[湄公河|湄公河]]另一邊的[[寮國|寮國]][[他曲|他曲]]。那空拍儂的涵義應該是可以觀賞山景的城市。
=== 早期历史 ===
此地長期以來是[[老挝|老挝]]人民居住地，隸屬[[瀾滄王國|瀾滄王國]]。越南历史文献称之为“乐凡”（La Khon），或者“六凡”。

约18世纪，乙牙陇自立为酋长，其子乙升归附暹罗[[阿瑜陀耶王朝|阿瑜陀耶王朝]]，暹王封之为布林马罗。乙升与西山朝暗中来往，为暹人禁锢而死，其婿马炽昌为酋长。马炽昌死后，又立乙升之子为佋芒为酋长。阮朝建立之后，乐凡向顺化进贡。明命九年，阮朝设乐边府，以酋长逋林为宣慰同知，管府事。十四年，逋林投入暹罗，阮朝立土目佋芃代其职，肥蛇嫩为土知州。

=== 20世纪以后 ===
[[越戰|越戰]]期間，北越叛軍與美軍之間幾場最慘烈的戰爭發生於此。1960年代，那空拍儂府分成131個區（[[tambon|tambon]]）。其中73個區據信有越南共產黨滲入，因此這裡被稱為“叛軍集中地”。

1967年，大部分共產黨都已投降，[[越戰|越戰]]也接近尾聲，美軍從[[呵叻府|呵叻府]]撤退，印度支那軍團全數移師至那空拍儂。

1928－1931年間，[[胡志明|胡志明]]曾在此停留，人們可以參觀他在這裡的故居。

== 府徽 ==
府徽是Phra That Phanom 在[[塔帕儂|塔帕儂]]的寶塔。這座崇高的佛塔原本應建於西元前535年左右。
府樹和府花都是[[灰莉|灰莉]]。

== 行政區 ==

本府分成11縣（''Amphoe''）、1次級縣（''King Amphoe''）；然後再進一步分出97個區（''tambon''）和1040個村。
