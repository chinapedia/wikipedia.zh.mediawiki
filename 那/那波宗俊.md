'''那波宗俊'''（生年不詳—卒年不詳）是[[日本戰國時代|日本戰國時代]]的武將。

== 生平 ==
生年不詳。[[山內上杉家|山內上杉家]]配下的[[國人|國人]]領主。[[天文_(後奈良天皇)|天文]]10年（1541年），與近鄰諸氏（[[深谷上杉家|深谷上杉家]]、[[長尾氏|足利長尾氏]]、厩橋{{tsl|ja|長野氏}}、{{tsl|ja|成田氏}}、{{tsl|ja|桐生氏|桐生佐野氏}}）聯合，攻撃[[新田金山城|新田金山城]]的橫瀨氏（{{tsl|ja|由良氏}}），不過最終失敗。此後，投向[[後北條氏|後北條氏]]並攻擊山內上杉家，受到周邊諸氏和橫瀨氏一同攻擊，不過因為{{tsl|ja|小幡氏}}等西[[上野國|上野國]]諸氏離反，結果[[關東管領|關東管領]][[上杉憲政|上杉憲政]]被流放到[[越後國|越後國]]。

[[永祿|永祿]]3年（1560年），居城[[赤石城|赤石城]]被擁立憲政的[[長尾景虎|長尾景虎]]（上杉謙信）攻撃。12月，赤石城、{{tsl|ja|那波城}}被攻陷，在降伏後被視為後北條方的有力國人，在所領被奪去後，不久死去。

== 死後 ==
死後，{{tsl|ja|那波氏}}的所領被賜予橫瀨氏，不過後來橫瀨氏投向後北條氏，於是謙信轉而計畫令成為人質的[[那波顯宗|那波顯宗]]（宗俊的嫡男）復歸舊領。

== 家系 ==
在系圖中記載，妻室是[[長尾景長|長尾景長]]的女兒，不過亦有說法指景長的女兒是母親，妻室則是[[長尾景誠|長尾景誠]]的女兒。
