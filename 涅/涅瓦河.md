'''涅瓦河'''（{{lang-ru|Нева}}）是[[俄羅斯|俄羅斯]]西北部的一條[[河流|河流]]，源出[[拉多加湖|拉多加湖]]，自东向西流，流經[[聖彼德堡|聖彼德堡]]与[[涅瓦湾|涅瓦湾]]，注入[[芬蘭灣|芬蘭灣]]。
<ref>{{Cite web |url= http://www.nevariver.ru/geographic.php
|title= Общие сведения о реке Неве |language= ru}}</ref>

==地理==
涅瓦河長74公里，28公里位於[[聖彼德堡|聖彼德堡]]的範圍內，其餘在[[列寧格勒州|列寧格勒州]]境內。流域面积5000平方公里。

涅瓦河水量较丰富，全年入海水量82立方公里，以流量來計算，涅瓦河是[[歐洲|歐洲]]第三大河流（居[[伏爾加河|伏爾加河]]和[[多瑙河|多瑙河]]之後）。冰期长约5个月。

由[[拉多加湖|拉多加湖]]向西南流至其最南端－和其與[[托斯諾河|托斯諾河]]之交匯處相近，然後再轉向西北流至[[芬蘭灣|芬蘭灣]]。平均寬度400至600公尺，最大寬度1200公尺，最大深度24公尺。涅瓦河流域包括拉多加湖和[[奧湼加湖|奧湼加湖]]，分佈於[[西北部聯邦管區|俄羅斯西北部]]和[[芬蘭|芬蘭]]南部等廣大的面積上。

==分支==
涅瓦河在流经三角洲中最大的分流有[[大涅瓦河|大涅瓦河]]、[[小涅瓦河|小涅瓦河]]、[[大小涅瓦河|大小涅瓦河]]、[[中小涅瓦河|中小涅瓦河]]、[[小小涅瓦河|小小涅瓦河]]

最重要的水道和小河流有：Morskoy Canal、Obvodny Canal、Griboedov Canal、Kryukov Canal、Fontanka、Moika River

著名的島嶼有：Dekabristov Island、Kamenny Island、Krestovsky Island、Petrogradsky Island、Vasilievsky Island、Yelagin Island

== 支流 ==
涅瓦河有26條支流。

== 事件 ==
{{main|1963年苏联民航图-124涅瓦河迫降事故}}
*1963年8月21日，[[苏联民用航空总局|苏联民用航空总局]]的一架[[图-124|图-124]]客机在由[[塔林|塔林]]飞往[[莫斯科|莫斯科]]途中发生故障，因燃油耗尽迫降在涅瓦河河面，机上52人全部获救。
