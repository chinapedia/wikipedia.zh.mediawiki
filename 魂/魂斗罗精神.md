{{Vgname|魂斗羅 Spirits|ja=魂斗羅スピリッツ|na=Contra III: The Alien Wars|eu=Super Probotector: Alien Rebels}}是[[KONAMI|KONAMI]]公司在1992年2月28日[[超級任天堂|超級任天堂]]平台上發售的動作遊戲，《[[魂斗罗|魂斗罗]]》系列的第三作。主要角色也還是Bill Rizer和Lance Bean，歐版則是機器人。

== 游戏系统 ==
[[File:Con3k3-6.png|right]]
游戏共有6关，其中1、3、6为常规横版卷轴；4为强制横版卷轴；2、5为俯视视角画面，值得一提的是这两关充分运用了SFC主机的回转放缩机能，玩家可以用L和R键改变主角所面对的方向。
主角能够同时拥有两种武器，通过手柄上的X键可以切换，死亡时只丢失当前使用的武器；增加了“定位射击”的动作，按住R键攻击时可以站在原地固定不动任意向八个方向射击；L、R键同时按住时再按攻击可以同时使用手上的两种武器乱射。<br />武器方面：机枪M和增加子弹速度的R被取消，加入了爆发弹C和[[魂斗罗_(Game_Boy)|GB版的魂斗罗]]中出现的新武器——追踪弹H（但每次发射的子弹数量变为一颗）。霰彈S也出于武器平衡性的考虑降低了攻击力。火焰弹F变为按住不放即可在一定范围内持续喷射火焰的火焰喷射器。以前的瞬间无敌被改为街机版《超级魂斗罗》的俯视画面中出现的可由玩家自由选择时间发动的炸弹。每条命自带一个，在游戏中还可以得到更多，损失一条生命后炸弹数量也重新变成一个。主角在第一关还可以乘坐坦克。游戏加入了难度选择系统，只有NORMAL和HARD难度下才可能与真正的最终BOSS交手。

== 故事 ==
公元2636年，不甘心数次失败的异形终于发动了对地球的大规模侵略战争，在异形压倒性的科技面前，人类一时束手无策，面对敌人的攻击节节败退，两度打败异形的魂斗罗小队的传奇战士比尔和兰斯再度临危受命，前去直接破坏敌人的中枢系统，以协助人类展开反击。

== 移植作品 ==
;{{Vgname|魂斗羅 Spirits|ja=魂斗羅スピリッツ|na=Contra: The Alien Wars|eu=Probotector 2}}
:由Factor 5公司開發製作在1994年9月23日發售的[[Game_Boy|Game Boy]]遊戲，由于GB机能所限，遊戲關卡數從六關縮減為五關。歐版的主角更改為機器人。

;{{Vgname|魂斗羅 Hard Spirits|ja=魂斗羅ハードスピリッツ|na=Contra Advance: The Alien Wars EX}}
:北美在2002年11月3日發售的Game Boy Advance遊戲，遊戲中混合魂斗羅 Spirits和[[魂斗羅_鐵血兵團|魂斗羅 鐵血兵團]]的關卡。

== 秘技 ==
*在标题画面迅速输入下、右下、右、Start，如听到主角死亡时的效果音则成功，生命数增加至30
*在标题画面迅速输入下、左下、左、Start，如果听到异形的叫声则成功，可以选关
*在标题画面输入下、右、右下、X，进入音乐试听模式
;GB版密码
*B32R，生命无限
*H2F2，游戏中任意跳关
*F5F5，7条生命、5颗炸弹，使用火焰喷射器开始最后一关
