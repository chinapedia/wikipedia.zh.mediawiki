{{Distinguish|水沙連之役}} 
'''骨宗事件'''，又作'''水沙連事件'''或'''水沙連之亂'''。[[雍正|雍正]]年間，[[漢人|漢人]]大量前往台湾拓墾，水沙連諸社因生活空間遭擠壓，遂以[[邵族|邵族]]水社頭目骨宗和其子拔思弄、水-{R|里}-萬為首，對於平地漢人進行一系列[[出草|出草]][[獵首|獵首]]等行動。{{r|citing}}{{r|puli}}出山殺人活動始於雍正三年（1725）年農曆3月起，迄於翌年11月，死亡人數為62人、耕牛140頭、焚燒房屋近百間。<ref name=":0">{{Cite journal|title=〈水沙連之役及其相關問題〉|author=陳哲三|url=http://www.cohss.fcu.edu.tw/wSite/public/Attachment/f1378095906031.pdf|journal=《逢甲人文社會學報》|issue=18|doi=|others=|year=2009|volume=|pmid=|access-date=2018-08-20|archive-url=https://web.archive.org/web/20180820234833/http://www.cohss.fcu.edu.tw/wSite/public/Attachment/f1378095906031.pdf|archive-date=2018-08-20|dead-url=no}}</ref>

== 背景 ==
[[File:Er_nd_3014_Thao.jpg|缩略图]]
水沙連諸社（分布約今[[南投縣|南投縣]][[日月潭|日月潭]]一帶，此區域原住民族群以[[邵族|邵族]]為主）原本[[投降|歸順]]清廷；康熙60年（1721年）三月，[[朱一貴事件|朱一貴事件]]爆發，水沙連諸社趁亂殺[[通事舍人|通事]]、自此拒不納餉。<ref>{{Cite book|title=《臺灣原住民史．政策篇（清治時期）》|last=溫|first=振華|publisher=國史館臺灣文獻館|year=2007|isbn=9789860115611|location=南投市|pages=|language=zh-tw}}</ref>{{Rp|31}}

同一時期，亦參與朱一貴事件的[[潮州市|潮籍]]領袖[[杜君英|杜君英]]、杜會三父子率軍往諸羅縣以北地區逃竄，清軍追緝多時，才於該年九月落網。<ref name=":1">{{Cite book|title=《臺灣通史》|last=連|first=橫|publisher=五南|year=2017|isbn=9789571191300|location=台北市|pages=|language=zh-tw}}</ref>{{Rp|567-572}}朱一貴事平之後，清廷鑒於諸羅縣幅員過廣，不易管理，便於雍正元年（1723年）於[[虎尾溪|虎尾溪]]以北、[[大甲溪|大甲溪]]以南設置「[[彰化縣_(清朝)|彰化縣]]」，縣城設於半線-{庄}-（今[[彰化市|彰化市]]）<ref name=":1" />{{Rp|85}}，並將原在諸羅縣（今嘉義市）之[[臺灣北路協|北路協鎮]]移駐到彰化縣，便於北控淡水、南馭諸羅，大群漢人隨之徙來，由是迫近中部山區原住民的生活圈<ref name=":1" />{{RP|216}}<ref name=":2">{{Cite book|title=《臺灣原住民史．邵族史篇》|last=鄧|first=相揚|last2=許|first2=木柱|publisher=臺灣省文獻委員會|year=2000|isbn=9570269278|location=南投市|pages=|language=}}</ref>{{Rp|26}}。

雍正二年（1724年），因平叛來台的[[福建水师提督|福建水師提督]][[藍廷珍|藍廷珍]]在舊[[張鎮庄|張鎮庄]]處設立藍張興-{r|庄}-（約今[[臺中盆地|臺中盆地]]南緣），報墾立戶陞科，申辦[[墾號|墾照]]「[[藍張興|藍張興]]」<ref>[[禪濟布|禪濟布]]〈奏報嚴禁私墾摺（雍正三年十月十六）〉：藍張興莊，舊名張鎮莊，逼近生番鹿場，兇番不時出入，不令民人開墾者也。...（略）...自雍正二年改屬彰化縣，而提督臣藍廷珍復委管事蔡克俊赴該地方招墾，自立莊戶名為藍張興莊。</ref>，引渡[[漳州裔臺灣人|漳州移民]]偷渡來台開墾農田、闢建水圳<ref>{{Cite journal|title=〈漳州人開發台中地方的歷史意象〉|author=陳炎正|url=|journal=《臺灣源流》|issue=|doi=|others=|year=|volume=|page=48-50|pmid=}}</ref>，拓墾地區迫近水沙連總隘口（今[[南投縣|南投縣]][[名間鄉|名間鄉]]）附近區域，造成水沙連地區各社原住民的傳統領域與生活空間受到壓迫{{r|puli}}<ref name=":2" />{{Rp|26}}。

雍正三年（1725年）起，清廷在臺灣設立[[戰船|戰船]]建造廠，故於臺灣內山處廣設「軍工寮」（或作「軍功寮」）以便伐木，中部地區匠寮多設於今彰化、南投及臺中交界的丘陵地區，隨著匠寮朝內山區域推進，漢人亦競相往山區墾殖，益加侵擾水沙連諸社原住民生活環境。<ref>{{Cite book|title=《臺灣原住民史．平埔族史篇（中）－中臺灣平埔族群史》|last=梁|first=志輝|last2=鍾|first2=幼蘭|publisher=臺灣省文獻委員會|year=2001|isbn=9570288884|location=南投市|pages=|language=zh-tw}}</ref>{{Rp|97}}

此外，彰化縣首任知縣[[談經正|談經正]]於雍正二年（1724年）上任，翌年八月因「打廉-{r|庄}-案」<ref group="註">詳見「事件」列表</ref>革職<ref>{{Cite web|url=http://museum02.digitalarchives.tw/dmp/2000/pingpu/library/fulltext/npmdatabase/yun016.htm|title=〈請嚴禁私墾番界以杜生番擾害摺〉|accessdate=2018-12-24|author=禪濟布、景考祥|date=雍正參年拾月拾陸日|publisher=宮中檔奏摺中臺灣原住民史料|language=zh-tw}}</ref>，到繼任知縣[[張縞|張縞]]於雍正四年（1726年）上任期間，彰化縣事務由[[諸羅縣|諸羅縣]]知縣[[孫魯|孫魯]]代為署理<ref>{{Cite book|title=《重修台灣省通志》|last=劉|first=寧顏|publisher=台灣省文獻委員會|year=|isbn=|location=|pages=}}</ref>，而諸羅縣治遠在今[[嘉義縣|嘉義]]<ref>{{Cite book|title=《臺灣歷史地圖》|last=黃|first=驗|publisher=遠流|year=2015|isbn=9789860475227|location=台北市|pages=54-55|last2=黃|language=zh-tw|first2=裕元}}</ref>，並無法有效第一時間處置或通報骨宗出草狀況，又為一事件拖延之導因。

== 出草 ==

=== 水沙連之役 ===
{{Main|水沙連之役}}
雍正四年（1726年），內山原住民出草不斷，清廷決意興兵鎮壓，年中開始徵召軍隊及壯丁，並結合[[平埔族|平埔族人]]組成數千人的討伐軍隊，採南北夾擊為討伐策略，史稱「水沙連之役」。<ref name=":0" />{{Rp|89-92}}

12月3日兩路軍同時出發。[[巡台御史|巡台御史]][[索琳|索琳]]、[[臺廈道|臺廈道]][[吳昌祚|吳昌祚]]率領由南港竹腳寮（今[[社寮地區|社寮]]）進攻水社，北港由[[臺灣府淡水撫民同知|淡水同知]][[王汧|王汧]]、[[參將|參將]][[何勉|何勉]]從[[南投市|南投社]]出發進攻哈裏難社{{#tag:ref|哈裏難社又稱蛤美蘭社，皆指[[埔里社|埔里社]]{{r|citing}}{{r|tchcc}}。|group=註}}。12月11日，南路軍[[守備|守備]]鍾日昇、[[千總|千總]]蔡彬在骨宗一社內（即水社）找到首級八十五顆{{#tag:ref|《埔里開發簡史》記載為八十五顆{{r|puli}}。|group=註}}。12月13日，以何勉為首的北路軍順利擒獲水社首領骨宗、麻思弄。嗣後，清軍持續對出草涉事者展開清查逮捕，包括骨宗父子在內，貓里眉社（[[泰雅族|泰雅族]]）亦有人犯參與其中。12月16日清軍順利逮捕二十餘名涉案生番，12月17日[[監軍|監軍]][[索琳|索琳]]先行率役下山，[[元帥|總統]]吳昌祚逮捕殘兵，12月27日才率同何勉、王汧循[[埔里鎮|埔里]]下山，並在半線社與索琳會合。<ref name=":0" />{{Rp|89-92}}

雍正五年（1727年）正月9日，吳昌祚率軍押解涉案生番前往[[臺灣府城|臺灣府城]]發禁，喧騰中台灣一時的「骨宗事件」至此落幕。<ref name=":3">{{Cite web|url=http://nrch.culture.tw/twpedia.aspx?id=5593|title=骨宗事件|accessdate=2018-12-24|author=簡史朗|date=2009-09-09|publisher=臺灣大百科全書|language=zh-tw}}</ref><ref name=":0" />{{Rp|89-92}}

== 後續 ==
水沙連之役一平，水沙連周遭諸社（含邵族、[[布農族|布農]]、[[泰雅族|泰雅]]、[[賽德克族|賽德克族]]）沿路被招撫，表達願意納番餉者各社人口共4445名。<ref name=":0" />{{Rp|104}}首犯骨宗與貓著被判「[[斬立決|斬立決]]」，二犯被押解至北路番子山口，梟首示眾。拔思弄等十三人則判「[[斬監候|斬監候]]」，又水-{r|里}-萬等五人照為從例枷責，發落於[[台灣省城|省城]]之中永久監禁。<ref name=":0" />{{Rp|104}}民風原本強悍的邵族，獵首習俗乃絕，邵族也因為勢力大幅衰退，無力抵禦漢人[[拓墾|拓墾]]腳步，從此清政府勢力滲透內山區域，間接導致[[嘉庆年间|嘉慶年間]]的[[郭百年事件|郭百年事件]]<ref name=":3" />{{r|ianthro}}。
