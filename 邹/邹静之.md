'''邹静之'''（{{bd|1952年||catIdx=Zou,邹}}），中国大陆编剧，诗人。他是[[北京作家协会|北京作家协会]]驻会作家<ref name="poemlife" />。由于参与熱播電視劇《[[康熙微服私访记|康熙微服私访记]]》和《[[铁齿铜牙纪晓岚|铁齿铜牙纪晓岚]]》的编剧工作而知名。

他的作品获得过不少奖项。2014年，他由于在电影《一代宗师》中的编剧工作，與徐皓峰、王家卫一起获得[[第33届香港电影金像奖|第33届香港电影金像奖]]最佳编剧奖。

== 生平 ==
*1952年出生於江西南昌<ref name="poemlife" />
*一岁来到北京﻿，喜欢动物，儿时最初的理想是当一名动物饲养员<ref name="golden">{{Cite web |url=http://news.sina.com.cn/o/2006-02-18/10368239164s.shtml|title=节后探访“金牌编剧”邹静之 邹静之：是生活成全了我
 |accessdate=2014-09-08 |author=乌力斯 |date= 2006年2月18日|work= |publisher=南方日报}}</ref>。
*1969年～1977年在北大荒和河南插队。此期间，他爱上了音乐，天天练声。
*回京后想考进文艺团体，堅持十年学习声乐，且两次差点成为歌剧演员。
*考入[[中央广播电视大学|中央广播电视大学]]中文系，讀了不少文學作品<ref name="golden" />。
*畢業後分配到在煤炭研究院做体力活儿。
*1982年被提拔为办公室主任。后在《[[北京晚报|北京晚报]]》发了“一分钟小说”《盖侠其人》，並在《[[诗刊|诗刊]]》发表詩歌
*1987年，成为《诗刊》诗歌编辑<ref name="zhuangjia">{{Cite web|url=http://news.xinhuanet.com/book/2004-05/19/content_1477614.htm|title=庄稼地里“长”出邹静之|accessdate=2014-09-08|author=李冰|date=2004-05-19|work=|publisher=新华网|deadurl=yes|archiveurl=https://web.archive.org/web/20150221111149/http://news.xinhuanet.com/book/2004-05/19/content_1477614.htm|archivedate=2015-02-21}}</ref>。

== 写作 ==
邹静之受到了田壮壮和唐大年的鼓勵，開始當編劇<ref name="zhuangjia" />。

邹静之的作品被认为台词写得精致凝练，他说由于写诗“感到语言也像音乐一样有美妙的节奏，后来创作剧本的时候，就会自然而然地把诗歌的韵律移植到电视剧中。”

邹静之喜歡使用[[圓珠筆|圓珠筆]]寫作，認為自己属于“圆珠笔时代”。他說：“我写的大多数作品，现在从底稿看，除少部分是钢笔之外，百分之九十是用圆珠笔划拉出来的，我就不用蘸着写，我可以一根一根把笔抛掉，所以写的速度和讲究的程度，我认为就没有30年代的那种文风好，跟古人就更没有法比”<ref name="poemlife">{{Cite web |url=http://www.poemlife.com/showart-55950-dawei.htm|title=邹静之：诗歌就是我的心灵 |accessdate=2014-09-08 |author=大卫 |date= |work= |publisher=}}</ref>。

為了“通过追求变化来求得写作新鲜感”，邹静之写过多種表現形式的劇本<ref name="qinghuai" />。

他認為藝術品應該“情怀至上”，他說：“艺术也是娱乐，但我认为电视剧分三个层次，讲故事只是第一个层次，然后是出人物，最高一级是出味道，让观众会心了，这就是最高的境界。我希望能出人物和出味道，出人物和出味道的戏过了几年看还一样，艺术的成分很高，这主要就是文学性的东西在里面了”<ref name="qinghuai">{{Cite web |url=http://news.163.com/14/0105/02/9HPS092U00014AED.html|title= 邹静之：故事为王，更要情怀至上|accessdate=2014-09-08 |author= |date=2014-01-05 |work= |publisher=中国青年报}}</ref>。

由于认为“从小就被灌输（[[写实主义|写实主义]]）是表达情感的唯一正确手段”，扼杀了文学想象力，所以邹静之曾反感写实主义。于是在作品《康熙微服私访记》的时候决定“走反讽路线，让它天马行空，无拘无束”。他說，“创作《康》的宗旨是：亦庄亦谐，借古说今。《康》中所述之事或多或少都在现实中国的经济、政治、文化生活中存在着，所以就借康熙这个筐，将现实往里装”<ref>{{Cite web |url=http://www.ifeng.com/fcd/200804/0424_3040_507145.shtml|title= “康熙”是个筐现实往里装——邹静之访谈
|accessdate=2014-09-08 |author= |date=1999年4月 |work= |publisher=}}</ref>。
邹静之将自己心目中经典的剧本概括为：“纵向能穿透时间，横向能走向世界。像莎士比亚和汤显祖都有四百多年了，现在在世界的舞台上还在演着他们的戏。一纵一横这个坐标交叉点上的作品，就是经典”<ref>{{Cite web |url=http://news.ifeng.com/gundong/detail_2014_04/17/35849028_0.shtml|title=邹静之很庆幸自己接了《归来》 |accessdate=2014-09-08 |author= |date=2014年4月17日 |work= |publisher=北京晚报}}</ref>。

但在閱讀《[[贫嘴张大民的幸福生活|贫嘴张大民的幸福生活]]》后认识到了写实主义的魅力，他在《[[五月槐花香|五月槐花香]]》的剧本中体现了写实主义<ref>{{Cite web|url=http://book.sina.com.cn/news/a/2004-06-22/3/78123.shtml|title=诗人邹静之：《五月槐花香》中重拾写实主义|accessdate=2014-09-08|author=|date=|work=|publisher=北京晨报|deadurl=yes|archiveurl=https://web.archive.org/web/20140906202419/http://book.sina.com.cn/news/a/2004-06-22/3/78123.shtml|archivedate=2014-09-06}}</ref>。
