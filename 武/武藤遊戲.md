'''武藤遊戲'''（香港漫畫版中又叫作'''武藤游憩'''）是[[日本漫畫|日本漫畫]]《[[遊戲王|遊戲王]]》中的主角。他是一名普通的日本高中生，他在解開了爺爺送給他的千年積木後，千年積木內的古埃及[[法老王|法老王]]靈魂進入了他的體內，因此武藤遊戲這個身體擁有兩個靈魂。原本的靈魂是武藤遊戲，通稱'''表遊戲'''，法老王靈魂的本名為'''亞圖姆'''、通稱'''闇遊戲'''。遊戲可以透過千年積木來切換靈魂。

由於兩個遊戲皆具有相當的魅力和人氣，因此在愛好者之間、如果只說「遊戲」的話可能會讓人無法判斷是指哪一個遊戲。

== 武藤遊戲 ==
=== 原作 ===
: 故事的兩名主角之一，一名普通的日本高中生，他具有獨特的海星髮型、而且由紅色、黑色和金色構成。
: 精通包括電子遊戲、桌上遊戲、卡牌遊戲、格鬥遊戲等所有的遊戲以及玩具，而且無論是運氣類型或技術類型的遊戲都非常擅長，但他的身材矮小、不善運動、性格有些懦弱和自卑，經常遭受同班同學[[城之內克也|城之內克也]]和本田廣的[[霸凌|霸凌]]，此外、他暗戀同班同學兼青梅竹馬真崎杏子。
: 他從經營玩具屋的爺爺手中得到了千年積木，並花了八年的時間將其組裝完成，這是三千年來從沒有人能組好的神秘拼圖積木，寄宿於積木中的法老王靈魂因此進入他的體內。
: 同時，他許下了一個願望：想要「一個無論如何都不會背叛自己的朋友」，後來積木再度被砸碎時，許下了希望能再見到另一個我的願望。
: 他在漫畫第一話遭受風紀委員牛尾哲的勒索之後，體內的法老王靈魂出現並懲罰了牛尾，此後也和城之內產生了友情。
: 在此之後，遊戲、城之內和本田就經常一起行動。
: 在故事初期，他並沒有意識到這個法老王靈魂的存在，只覺得自己有時會失去意識，原作中DEATH-T因本田並未逃出積木關卡，差點讓闇遊戲出來，於此時坦承自己有時失去意識是因為心中有另一個不認識的〝自己〞，決鬥基本都是這個〝自己〞代替他，後來在黑暗RPG中兩人才算正式相見(表遊戲被封印在棋子中)，動畫版是在與貘良的黑暗決鬥。
: 直到決鬥者王國篇後，兩個靈魂才在心中正式地對談並合作，共同擊敗貝卡斯，後稱呼闇遊戲為「另外一個我」。
: 火災後察覺到闇遊戲並非他心中的人格，而是寄宿在千年積木的人格，詢問闇遊戲的事，得到的答案是連闇遊戲自己都不知道自己是誰，唯一擁有的只有從遊戲那裡得到的記憶與其他人共處的時光。
: 戰鬥城市篇不解為什麼闇遊戲會參加賭卡比賽，後來想起闇遊戲本身完全沒有任何的記憶，連自己的本名都想不起來，後在與操縱潘多拉的馬利克對談後，確認了闇遊戲之所以會參加戰鬥城市，是為了尋找自己的記憶。
: 雖然在故事中的決鬥通常是由闇遊戲進行，但表遊戲本身也具有不俗的實力，兩人在戰鬥之儀中進行最終的對決，在表遊戲打敗闇遊戲後，闇遊戲表示他有「名為溫柔的堅強」。
: 在記憶篇裡最先察覺到這是一場TPG遊戲的人，在挑戰闇貘良時知道自己必須要獲勝才能得到另外一個自己的記憶關鍵，因此發揮全副實力打敗闇貘良。
: 打倒亞圖姆時，哭著說其實他真正的願望是不希望一直在守護他、陪伴他的亞圖姆離開，坦言自身其實十分仰慕亞圖姆的那份強大與堅強，但最後依然坦然目送亞圖姆離開。
: 即使神之卡在手也不曾使用。

=== 衍生作品 ===
: 在《遊戲王R》裡，使用自己的牌組挑戰獵人，獲勝，不過在苦戰中興起放棄之意，被闇遊戲鼓勵，並說一定會獲勝。
: 在《遊戲王R》的番外篇裡，龜遊戲屋的決鬥者資格認證比賽中，被雙六當成最終勝利者有權向他挑戰的獎品，因本身是最高等級的八級，故無法參加比賽。
: 在《[[10th週年劇場版_遊戲王_～超融合！跨越時空的羈絆～|10th週年劇場版 遊戲王 ～超融合！跨越時空的羈絆～]]》裡，差點因帕拉杜克斯的攻擊而喪命，被紅龍所救，並回到事發前半小時，與十代和遊星製造小騷動驅離其他人。
: 在《[[遊戲王：次元的黑暗面|遊戲王：次元的黑暗面]]》中，變得比較成熟，目標是成為遊戲設計師，也已設計好遊戲並報名參加德國的設計比賽，受海馬邀請參加測試比賽，對於海馬執著於讓亞圖姆回到現世是抱著無奈但能理解，即使自身也希望能再見到亞圖姆卻知道亞圖姆是早已逝去之人而無法相見。後與海馬聯手對抗被千年智慧輪裡的邪念控制的藍神，但是到最後因體力不支而快倒下時，再次見到帶著瑪哈特降臨的亞圖姆。

== 亞圖姆 ==

=== 原作 ===
: 故事的兩名主角之一，是寄宿於千年積木中的靈魂，因為武藤遊戲組合了千年積木，他的靈魂得以出現在遊戲體內。
: 在單行本人物介紹與愛好者中，在本名尚未公布時，一般都是以闇遊戲來指稱他，原作中有說闇遊戲是黑暗守門人。
: 真實身分是三千年前、[[古埃及|古埃及]]時期十八王朝的法老王，他在與大魔王佐克‧內洛法的戰鬥中，以自己的名字封印了佐克，卻也因為失去名字而無法進入冥府，直到武藤遊戲組合好千年積木才得以現身。
: 在故事初期現身時，他的表情邪惡、態度狂妄傲慢，他透過主持黑暗遊戲來彰顯正義，且毫不留情地給違規、使詐的惡人玩家進行殘酷的懲罰，輕則皮肉傷、重則精神崩潰。
: 根據文庫版的追加設定，這是由於他的心中尚殘有佐克的意志所導致的。
: 在故事後期，他的個性雖然同樣高傲、沉穩、不愛開玩笑，但感情已經變得豐富許多，對不熟的人也能保持一定的溫柔。
: 和表遊戲相比，他的肢體能力極佳，常在發生肢體衝突時替換上場，在杜馬篇有出現他騎馬的畫面，在王之記憶篇中，也曾舉劍與佐克的亡靈大軍對打。
: 一開始就知道表遊戲的存在，並擁有他的所有記憶，以及對現代社會、現代遊戲規則的認知，因此他對新遊戲也能很快上手，並且同樣擁有頂尖實力。
: 直到決鬥者王國篇，兩個靈魂才在心中正式地對談並合作，共同擊敗貝卡斯，他稱呼表遊戲為「夥伴」。
: 後來在大火中提示城之內如何解開千年積木的鍊子，因此事讓遊戲與杏子得知他是棲宿於千年積木的靈魂，而非遊戲的另一人格。
: 對遊戲坦承自己完全沒有任何與過去相關的記憶，連自己的本名都不知道，唯一知道的事就只有因為遊戲拚好了千年積木，所以他才存在，遊戲因此說要將他所有的記憶全送給他。
: 後來被遊戲設計與杏子約會，因杏子挑戰強尼獲勝而下定決心前往童實野美術館，於地下展覽室看見了石板，確知了自己的身分是遙遠的古埃及法老王，並從伊西絲口中得知將會一場大規模的戰鬥，也就是戰鬥城市。
: 拜託杏子不要告訴遊戲他今天來過美術館的事，連同石板與伊西絲說的話都不要說。
: 動畫版裡為了去營救被擄的城之內與杏子，與海馬一起搭乘直升機，途中海馬要求情報，並拒不承認貝卡斯的力量時，坦言自身根本就不是〝武藤遊戲〞，只是一抹寄宿在千年神器裡，沒有記憶也沒有名字，只是因為武藤遊戲完成了千年積木，才得以復甦的幽魂。
: 在戰鬥城市中的淘汰賽裡，主要對手都是古魯斯的成員，取得歐西里斯的天空龍後，直到被海馬指出在淘汰賽中根本就沒把那張卡放入牌組，後由遊戲親手將歐西里斯的天空龍放入牌組中。
: 因為自身僅是一抹幽魂，所以清楚的知道，保護宿主對他來說是一種本能，但被表遊戲否決，因為表遊戲自身也想幫助亞圖姆尋回自己的一切。
: 在記憶篇裡因為自身沒有記憶鬧了一些笑話，在貘良帶著亡父的木乃伊挑釁時，暴怒直接親自下場，召喚歐貝利斯克的巨神兵對抗貘良的精靈獸迪爾邦多。
: 動畫版有追加關於父親之間相處的回憶，十分尊敬父親，但也對父親為何要製造千年神器感到困惑。
: 杏子送他的王名項鍊是讓他得以召喚出光之創造神‧赫爾阿克蒂的關鍵。
: 在故事最後與表遊戲進行名為戰鬥儀式的最終對決，在動畫版中，以驚人的實力於兩回合內讓三幻神在場上齊聚一堂，讓眾人見識到壯觀的景象，但由於他的最後王牌被表遊戲料中，最終仍輸給了表遊戲。
: 坦然的接受敗北，笑著安慰哭個不停的表遊戲，給予勸慰後，心無罣礙地回到了古埃及，靈魂得以安息。

=== 衍生作品 ===
: 在《遊戲王R》裡，對於獵人均不曾手下留情，在排名中是懸賞十萬美金的最高等級目標，從誤以為是夜行的月行口中得知夜行的事，最後在與夜行對決時，以被夜行稱讚為完美而崇高的實力打倒夜行。
: 在《遊戲王R》的番外篇裡，對於使用搶奪來的卡組成的屬性相剋牌組的決鬥者三人組，以三幻神為賭注進行決鬥，對於對手以光屬性的牌組是感到吃力，但是認為對方的光無法與自己持有的黑暗相比擬，甚至連葬送自己前往冥府都做不到，在表遊戲過來恭賀時，認為送自己前往冥府的人是表遊戲，呼應正文裡的戰鬥之儀篇。
: 在《怪獸膠囊》中，是被選中的人，在這個黑暗遊戲裡，勝者將得到可以統治一切的王之力，可以用穿在身上的盔甲自由與怪獸融合，不過與日版相比，十分饒舌，結尾與征服並統治他的國家的亞歷山大見面，亞歷山大說亞圖姆是他所仰慕的法老王。
: 《[[10th週年劇場版_遊戲王_～超融合！跨越時空的羈絆～|10th週年劇場版 遊戲王 ～超融合！跨越時空的羈絆～]]》裡登場時宣言絕不原諒帕拉杜克斯以人命作為交換來改變破滅的未來，決鬥中不斷的鼓勵遊星，並幫遊星搶回星塵龍。
: 在《[[遊戲王：次元的黑暗面|遊戲王：次元的黑暗面]]》中已經不存在於現代的日本，被海馬為測試次世代決鬥盤而做成AI，靈魂雖然在冥界，但似乎可以自由的往來冥界與現世，曾出手救了城之內一命，亦幫了遊戲打贏被千年智慧輪裡的邪念控制的藍神，在最後帶著千年積木與遊戲相視一笑後消失。最後與乘坐次元穿梭器，回到三千年前的海馬決鬥。

== 持有牌組 ==
初期，亞圖姆的牌組有五張'''被封印的黑暗大法師'''卡牌，是一旦湊齊就無條件獲勝的究極之牌，但這些卡牌被羽蛾丟入大海，從此再也沒有人能召喚黑暗大法師。決鬥者王國篇時，他誘導對手踏入陷阱靈活地運用怪獸卡和魔法卡，組合出意想不到的戰術並致勝，例如讓岩石巨兵攻擊月亮、用小精靈堵住千眼犧牲祭品的眼睛等。

中期開始、亞圖姆的牌組以'''黑魔導'''為王牌，以及少量搭配黑魔導的魔法卡如死亡魔術箱、魔術帽子等，吉祥物是'''小精靈'''，並搭配惡魔族、戰士族、魔法使族的怪獸，有惡魔的召喚、破壞劍士等強力怪獸、渾沌戰士、渾沌黑魔導等儀式怪獸，以及[[磁石|磁石]]戰士和[[撲克牌|撲克牌]]三劍士等快速湊齊祭品的手段。此外、他還擅長使用強力的魔法卡和陷阱卡，如'''死者甦醒'''、'''神聖彗星反射力量'''等。在戰鬥城市篇，亞圖姆得到了三幻神之一的歐西里斯的天空龍，不過並沒有放入牌組中，後來又陸續得到歐貝利斯克的巨神兵和太陽神的翼神龍，但鮮少使用。

武藤遊戲的牌組來自武藤雙六，並成為遊戲和亞圖姆共用的牌組。直到王之記憶篇，遊戲才擁有自己特有的牌組，而這些牌組的內容是亞圖姆並不清楚的。

遊戲的牌組中有會隨著時間經過而強化的'''沉默劍士'''、'''沉默魔法師'''，以及強力的怪獸'''破壞龍剛多拉'''。另外、還有三色[[齒輪|齒輪]]這項快速召喚下級怪獸的手段，以及陷阱怪獸卡機動碉堡。
