'''武满彻'''（{{bd|1930年|10月8日|1996年|2月20日|}}），[[日本|日本]]20世纪[[现代音乐|现代音乐]][[作曲家|作曲家]]。

== 早年 ==
武满彻1930年生于[[东京|东京]]，一个月后随家人移居中国[[大连|大连]]，后又返回日本。年幼时因为被[[征兵|征兵]]而没有接受完整的教育，他在军中对西方[[古典音乐|古典音乐]]开始发生兴趣。日本战败后他为[[美军|美军]]工作，这又使他有机会聆听古典音乐。他几乎没有接受过正规的[[音乐教育|音乐教育]]，而几乎全靠自学作曲。1951年创建“[[实验工房|实验工房]]”。

== 成名 ==
1959年，作曲家[[斯特拉文斯基|斯特拉文斯基]]访问日本，当时[[NHK|NHK]]电台选择了一些日本作品播放给他听，工作人员由于失误，把原本不在计划之内的武满彻的弦乐《安魂曲》播了出来，斯特拉文斯基听见后坚持要听完全曲，并认为这是一首杰出的作品，更把武满彻请去一起用餐。<ref>http://www.cso.org/main.taf?p=5,5,5,93</ref>自此，武满彻开始成为日本最著名的作曲家之一。

== 音乐创作 ==
武满彻的音乐受到了多方面的影响，例如[[具体音乐|具体音乐]]和[[电子音乐|电子音乐]]，[[德彪西|德彪西]]和[[梅西安|梅西安]]的风格，[[爵士乐|爵士乐]]，以及[[音色|音色]]音乐等，他也深受日本传统音乐的影响，有部分作品使用了民族乐器。他的作品基本上都是慢速度的，充满了东方哲学的韵味。他的创作以器乐曲为主，管弦乐作品效果相当独特，也有一些作品涉及到电子音乐的技术，同时他也作有大量的[[电影配乐|电影配乐]]，著名的包括[[黑泽明|黑泽明]]的《乱》。
