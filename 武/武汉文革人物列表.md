本条目收录湖北省武汉市文化大革命中有重要影响的人物。

== 造反派 ==
* [[董明会|董明会]]（1934—1999）：武汉钢铁公司耐火厂工人，中共党员。后参加“[[钢九一三|九一三兵团]]”，成为中共第九届、第十届中央委员，任湖北省委委员。后任武汉市委书记、武汉市民兵指挥部指挥长、武汉市总工会主任。文革后开除党籍、开除公职、撤销党内外一切职务，关押6年多，免予起诉。
* [[夏邦银|夏邦银]]（1935—2001）：汉阳轧钢厂工人，“工总”勤务组成员，省革委会常委，中共九届、十届中央委员。后开除党籍，判处有期徒刑十三年、剥夺政治权利三年。
* [[谢望春|谢望春]]（1930—2011）：女，1956年4月加入中国共产党。曾当选市劳动模范。文革前为武汉国棉三厂工人，文革中是“[[工造总司|工造总司]]”成员，中共第九届、十届中央候补委员，后任武汉市纺织局党委副书记、湖北省妇联副主任。文革后定为“犯严重政治错误”，1982年开除党籍，撤销党内外一切职务。
* 朱鸿霞：武汉重型机床厂工人，“[[工总|工总]]”一号勤务员，后任湖北省革委会副主任。
* 胡厚民（1937—1990）：武昌造船厂工人，1961年加入中国共产党，“工总”二号勤务员，后任湖北省革委会常委。1970年6月被监护审查，1974年3月解除监禁。1977年11月被逮捕。1982年7月28日判处有期徒刑20年，剥夺政治权利5年。
* 张立国：“[[新华工|新华工]]”副司令，后任新华工革委会主任、湖北省革委会副主任。
* 聂年生（1943—1993）：华中工学院学生，是武汉最早的老造反派，新华工“毛泽东思想红卫兵红色造反团”负责人，后任“新华工”第一副司令、新华工革委会第一副主任、武汉市革委会委员。帮助解放了湖北省、武汉市的许多革命干部。68年底毕业分配到湖北恩施来凤县。在两清运动中被打成“北决扬”的“幕后总后台”。1973年在张体学的关照下，调回武汉无线电元件厂。此后积极活动，为造反派平反。1974年被补为武汉市革委会常委。1976年底被关押，4年后释放。抓“三种人”时，据说由于一些老干部的力保，聂年生没有坐牢。1993年死于脑溢血。
* 郭保安（1944—）：文革前为华中工学院学生、中共党员。文革中为新华工一号勤务员、武汉市革委会常委。文革后开除党籍、留党察看两年，定为“犯严重政治错误”。
* 龙铭鑫（1942—）：文革前为湖北大学学生、中共党员。文革中为新湖大临委会一号勤务员、湖北省革委会常委。文革后被开除党籍、定为“犯严重政治错误”。
* 高玉泽（1945—）：华中农学院学生，中共党员。文革中为“[[新华农|新华农]]”一号勤务员。武汉市革委会常委、团省委副书记。文革后因生活作风问题判刑5年。
* 杨道远：原名“杨树贵”，参加毛泽东接见后改名“杨道远”。河南省镇平县人，武汉测绘学院1966届毕业生，于1966年8月31日参加毛泽东接见群体，回到武汉就组织了“毛泽东主义红卫兵”。 “[[二司|二司]]”成立时被选为“二司”一号勤务员，后任湖北省革委会副主任。1970年6月，被打成“湖北、武汉地区‘五·一六’的总头目”，被关押。1983年1月，被判刑12年。1983年3月，刑满出狱，经朋友帮忙，给别人介绍业务，从中拿了3万元的好处费，被以“贪污罪”判刑15年。
* 吴焱金（1944—）：文革前为武汉市无线电元件厂工人，文革时为“[[工造总司|工造总司]]”一号勤务员，后任武汉市革委会副主任，兼任武汉防汛指挥部副指挥长。1983年被判刑8年。
* 李想玉，也有资料写为李湘玉，武钢机械总厂工人，“九一三”一号勤务员、武汉市革委会副主任。
* 彭祖龙（1937—）：文革前为武汉橡胶厂技术干部，文革中为工造总司勤务组成员、武汉市总工会副主任，1983年被判刑6年。
* 鲁礼安（1946—）：文革前为华中工学院学生。文革中为新华工敢死队头头、北决扬主要撰稿人。因北决扬案遭12年监禁，1979年底出狱，免予刑事处分。2006年4月离家出走，下落不明。
* 顾建棠（1932—）：文革前为长江流域规划办公室团委副书记、中共党员。文革中为长办联司一号勤务员、长办革委会副主任、湖北省革委会常委。1983年被判刑7年。
* 谢华之（1946—）：文革前为武汉大学学生。文革中为三司革联一号勤务员、湖北省革委会常委。文革后受审查，定为“犯严重政治错误”。
* 张维荣，文革前为湖北大学教师，中共党员，文革中为新湖大勤务组成员之一、武汉市革委会常委，1974年后精神失常，1984年在珠江溺水身亡。
* 王彩珠（1945—1980年代），华中师范学院学生，武汉钢二司勤务组成员、华中师范学院革委会主任、武汉市革委会常委，后任湖北省妇联副主任。
* 聂年生（1943—1993）：华中工学院学生，是新华工“毛泽东思想红卫兵红色造反团”负责人，后任“新华工”第一副司令、新华工革委会第一副主任、武汉市革委会委员。帮助解放了湖北省、武汉市的许多革命干部。1968年底毕业分配到湖北恩施来凤县。在两清运动中被打成“北决扬”的“幕后总后台”。1973年在张体学关照下调回武汉无线电元件厂。此后积极活动，为造反派平反。1974年被补为武汉市革委会常委。1976年底被关押，4年后释放。清理“三种人”时，据说由于一些老干部的力保，聂年生没有坐牢。1993年死于脑溢血。

== 保守派 ==
* 俞文斌（1933-）：文革前为武汉市机械工业局武装部副部长、中共党员。1967年5月起是“[[武汉地区无产阶级革命派百万雄师联络站|百万雄师]]”一号头头，1967年七·二〇事件后被关押4年。1972年3月被释放，1975年3月到当阳县挂职锻炼，任当阳县革命委员会副主任、官垱公社党委书记。文革后任武汉市经委副主任、湖北省暨武汉市驻香港办事处主任、湖北省委常委、1983年任武汉市驻香港办事处主任。
* 杨道安（1929-）：武汉汽车修配厂职工，“百万雄师”二号头头，78年时曾被选为第五届全国人大代表。
* 沈邦安：曾任“百万雄师”联络部副部长，文革后任武汉市二轻工业局副局长。
* 章迪杰（1936-）：“文革”前为武汉市硚口区房产公司第五房管所工会主席，“百万雄师”联络部长，平反后任硚口建筑集团公司三产办主任。写有回忆录《峥嵘岁月——回忆“百万雄师”》。
* 汪士奇，“文革”前为武汉市公安局干部，“文革”中为“武汉公检法”一号头头，“七二○”事件后被关押囚禁4年零8个月，平反后任武汉市公安局正科级干部。

== 干部 ==
* [[陈再道|陈再道]]（1909—1993）：文革时任武汉军区司令员，因[[七·二〇事件|七·二〇事件]]倒台。1972年任[[福州军区|福州军区]]副司令员。
* [[钟汉华|钟汉华]]：1963年起任武汉军区第二政治委员。1972年起任广州军区副政治委员，装甲兵政治委员。
* [[曾思玉|曾思玉]]（1911—2012）：1959年任沈阳军区副司令员，七二○事件后调任武汉军区司令员，后任湖北省革委会主任、湖北省委书记。1973年12月八大军区司令员对调时任济南军区司令员。
* [[刘丰|刘丰]]：文革爆发时是武汉军区空军副司令员，七·二〇事件后任武汉军区政委，1968年2月至1972年2月任湖北省革委会副主任，1970年3月至1971年3月任湖北省革委会党的核心小组副组长。1971年3月任中共湖北省委第二书记。1971年11月，被撤销武汉军区第一政委职务。1972年2月，被撤销省委第二书记、省革委会副主任职务。
* [[李迎希|李迎希]]（1902—1981）：1955年被授予少将军衔，武汉军区副司令员。文革中被陈再道、曾思玉先后打成反党乱军的野心家，文革后平反。
* [[张体学|张体学]]（1915—1973）：文革爆发时是湖北省省长、省委书记处书记、省委第二书记，1966年8月任湖北省委代理第一书记。1968年2月任湖北省革委会副主任。是中共第九、十届中央委员。1973年9月3日在北京病逝。
* [[赵辛初|赵辛初]]（1915－1991）：1956年任中共湖北省委书记处书记，两年后兼任湖北省副省长。1965年任[[中华人民共和国文化部|中华人民共和国文化部]]副部长。1973年任湖北省委书记，兼任省革委会副主任、省军区第一政委、武汉军区政委。1975年5月22日任省革委会主任、省委第一书记。1978年兼任湖北省政协主席，同年改任国家计委副主任，次年兼任国家农业委员会副主任。1980年任中华人民共和国粮食部部长。
* [[赵修|赵修]]（1921—1992）：文革前任湖北省副省长、湖北省委书记处书记。1972年3月任湖北省委常委，1972年12月至1977年4月任湖北省委书记。1975年7月至1979年6月任武钢党委第二书记。1979年6月至1982年5月任农业部副部长。
* [[王克文|王克文]]（1917—1993）：文革前任武汉市副市长、中共武汉市委书记处书记，后在五七干校劳动两年。1972年12月任湖北省委书记。1973年1月至1977年11月任武汉市委第一书记、武汉市革委会主任。1978年调任山西省委书记、山西省革委会副主任。
* [[姜一|姜一]]（1919—1998）：文革前为湖北省委候补书记。文革中为湖北省革委会常委、省委书记。
* [[任爱生|任爱生]]：文革前为原省委委员、湖北省委农村工作部主任。文革中是湖北省最早支持造反派的领导干部，1968年2月任湖北省革委会副主任。在“两清一批”中被软禁，文革后期被判刑。
* [[薛朴若|薛朴若]]（1909—2000）：1938年2月加入中国共产党，文革前任武汉市副市长。文革中发表声明支持造反派，成为武汉市革委会常委，1974年任武汉市革委会副主任。1983年离休返乡，致力于家乡教育事业。
