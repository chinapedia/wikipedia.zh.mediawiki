'''武藏野台站'''（{{jpn|j=武蔵野台駅|hg=むさしのだいえき|rm=Musashinodai eki|f=y}}）是一個位於[[東京都|東京都]][[府中市_(東京都)|府中市]]白糸台四丁目，屬於[[京王電鐵|京王電鐵]][[京王線|京王線]]的[[鐵路車站|鐵路車站]]。[[車站編號|車站編號]]是'''KO21'''。

== 年表 ==
* 1916年10月31日，作為京王電氣軌道的車站啟用，最初名為'''車返站'''。
* 1944年5月31日，京王與[[東京急行電鐵|東京急行電鐵]]合併後，車返站作為京王線的車站運行。
* 1948年6月1日，東急與京王帝都電鐵分拆，車返站成為京王的車站。
* 1959年6月1日，改稱'''武藏野台站'''。
* 1979年10月3日，列車在車站東面的飛田給11號與重型電單車相撞，造成1人死亡。
* 2010年12月5日，新建的[[跨站式站房|跨站式站房]]開始運作。
* 2011年9月，車站的改善工程完工。
* 2013年2月22日，[[快速列車_(日本)|快速]]開始在武藏野台站停站，同日以'''KO21'''作為編號。

=== 站名由來 ===
來自於所在地一帶的「[[武藏野台地|武藏野台地]]」地形。

== 車站構造 ==
本站為[[相對式月台|相對式月台]]2面2線地上車站。剪票口位於橋上站房2樓。月台有設置空調的候車室。

廁所位於二樓付費區內。

剪票口往各月台、北口、南口設有[[電梯|電梯]]。

車站南口有巴士站、計程車乘車處。

== 車站周邊 ==
2011年3月站房改建，站體內的{{link-ja|京王零售服務|京王リテールサービス|K-Shop}}與{{link-ja|京王書籍販售|京王書籍販売|啓文堂書店}}開店<ref>[http://ryutsuu.biz/backnumber/accounts/14838.html 流通ニュース] {{Wayback|url=http://ryutsuu.biz/backnumber/accounts/14838.html |date=20150119145136 }}</ref><ref>[http://www.keibundo.co.jp/company/index.html 啓文堂書店　会社概要]</ref>。

南口的小型圓環設有巴士站。

車站東邊與鐵路相交的道路，以及南口圓環周邊為[[商店街|商店街]]。該道路往南可至[[都市再生機構|都市再生機構]]車返團地。往北可至{{link-ja|東京都道229號府中調布線|東京都道229号府中調布線}}（舊[[甲州街道|甲州街道]]）。

車站西側有[[西武鐵道|西武鐵道]][[西武多摩川線|多摩川線]][[白糸台站|白糸台站]]。
