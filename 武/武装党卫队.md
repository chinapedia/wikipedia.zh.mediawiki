'''武裝黨衛隊'''（{{lang-de|'''Die Waffen Schutzstaffel'''}}，简称{{lang-de|'''Waffen-SS'''}}）是[[納粹德國|納粹德國]][[親衛隊|親衛隊]]领导下的一支[[准军事部队|准军事部队]]，由親衛隊特别机动部队（{{lang-de|SS-Verfügungstruppe}}）发展而来，于1939年及1940年冬天开始使用“武裝親衛隊”这一名称。战争中武裝親衛隊在各级武装部隊中定位为“親衛隊全國领袖”[[海因里希·希姆莱|海因里希·希姆莱]]的精锐部队，因此它在二战初期是[[德军|德军]]装备最先进的部队，俗稱[[黑衫軍|黑衫軍]]。

== 前身 ==
=== 阿道夫·希特勒近衛旗队 ===
1933年3月17日納粹黨取得政權後，[[希特勒|希特勒]]在[[柏林|柏林]]親自下令成立'''柏林參謀卫队'''（Stabswache Berlin），主要是負責總理官邸的內部警衛和正式儀式接待的儀仗隊，並指派親衛隊集團的領袖[[約瑟夫·迪特里希|約瑟夫·迪特里希]]為指揮官，由他在親衛隊中嚴格挑選120人組成一隻直屬於希特勒本人的武裝部隊。而德國陸軍所提供的元首護衛隊則負責官邸外圍的安全。這隻部隊在同年9月的纳粹黨大會後正式定名'''阿道夫·希特勒近衛旗队'''（'''L'''eibstandarte-'''SS A'''dolf '''H'''itler，缩写LSSAH），共2,600人編制。

=== 特别机动部隊 ===
柏林儀仗警卫隊成立的同時，纳粹黨在[[慕尼黑|慕尼黑]]、[[汉堡|漢堡]]及[[德勒斯登|德勒斯登]]等大都市成立了政治警备隊（politische Bereitschaft），作為其在國內執行政治任務的武裝部隊。

1934年希特勒發動[[長刀之夜|長刀之夜]]，以[[海因里希·希姆莱|海因里希·希姆莱]]領導下的黨衛隊及正規警察為主力，清算了[[納粹衝鋒隊|納粹衝鋒隊]]的勢力。

1935年3月15日希特勒在柏林宣佈撕毀[[凡爾賽條約|凡爾賽條約]]。3月16日德國恢復义务徵兵制，希特勒下令成立完全軍事化的親衛隊組織，也就是'''親衛隊特别机动部队'''（{{lang|de|SS-Verfügungstruppe}}，缩写：SSVT）。

當時的特别机动部队總共有11個營，包括一個工兵營和一個通訊營，總計有8,459人。1936年前陸軍中將[[保羅·豪賽爾|保羅·豪賽爾]]被任命为特别机动部队總監，他將原本散駐各地的各個營集中起來在[[慕尼黑|慕尼黑]]成立了親衛隊德意志旗队（SS-Standarte Deutschland），在[[汉堡|漢堡]]成立了親衛隊日爾曼旗队（SS-Standarte Germania）。

1937年為解決武裝親衛隊領導的軍事質素不足問題，保羅·豪賽爾特定開設兩個親衛隊獨自的'''巴特特尔茨[[容克|容克]]學校'''（{{lang|de|Junkerschule Bad Tölz}}）和'''布伦瑞克容克學校'''（SS-Junkerschule Braunschweig），專門負責教育培訓親衛隊軍官，兩個學校共759人。

1938年德奧合併後，在維也納的特别机动部队也成立了一個新的親衛隊－元首旗隊（{{lang|de|SS-Standarte der Führer}}）。

德波戰爭後的1939年10月，親衛隊將德意志旗队、日爾曼旗队、元首旗隊3支特别机动部队旗隊合組成'''親衛隊特别機動師'''（{{lang|de|SS-Verfügungs Division}}）參加接下來的德法戰役。

=== 骷髏師 ===
在1933年由德國國內主要负责监狱特别是集中营的看守及管理的警卫部隊所組成的另一個武裝部隊就是所謂的親衛隊[[骷髏总队|骷髏总队]]，其指揮官為[[西奧多·艾克|西奧多·艾克]]。

由於特别机动部队的擴張使得希姆萊的可用兵力愈來愈少，因此他就將原本負責集中營工作的親衛隊骷髏总队中的精英份子獨立出來在[[达豪|达豪]]（Dachau）改編為一個分队，由原先的指揮官艾凱指揮。到了1939年10月16日親衛隊跍髏总隊已經有3個旗队約6,500人的兵力，就以此3個旗队為核心成立了一個親衛隊骷髏師（SS-Division Totenkopf）。

=== 警察師 ===
1939年為了接下來的波蘭戰役，希姆萊開始向擔任德國內一般警察勤務的秩序警察（Ordnungspolizei）招幕專門的戰備部隊，9月18日由警察組成的'''警察師'''（Polizei-Division）正式組成，並立即投入法國戰役。

== 武裝親衛隊成立 ==
1940年4月，為管制親衛隊眾多的不同武裝部隊，親衛隊成立了統一指揮旗下武裝部隊的指揮部'''親衛隊作戰本部'''（SS Führungshauptamt）。

由於親衛隊10月在法國戰役表現傑出，國防軍對其大幅改觀而不再貶稱親衛隊為「瀝青士兵」，將其視為貨真價實的「野戰部隊」。

為此親衛隊作戰本部將旗下各武裝部隊統一整編為單一體制編制組織，並正式命名為'''武裝親衛隊'''。

此後武裝親衛隊各師團開始有正式編號序列（[[武装親衛隊师团|武装親衛隊师团]]），並且配屬裝甲單位。

1943年親衛隊第1裝甲師、親衛隊第12裝甲師、親衛隊第17裝甲掷弹兵師，被整編為'''親衛隊第一裝甲軍'''（I.SS-Panzerkorps），首任軍長為[[約瑟夫·迪特里希|約瑟夫·迪特里希]]。

親衛隊第2裝甲師、親衛隊第9裝甲師（1943年含親衛隊第3裝甲師、1944年含親衛隊第10裝甲師、1945年含親衛隊第23裝甲師），被整編為'''親衛隊第二裝甲軍'''（II.SS-Panzerkorps），首任軍長為[[保羅·豪塞爾|保羅·豪塞爾]]。

1944年親衛隊第11志願裝甲師、親衛隊第20裝甲擲彈兵師、親衛隊第23志願裝甲擲彈兵師、親衛隊第27志願兵師、親衛隊第28志願裝甲擲彈兵師（5月含親衛隊第4志願裝甲擲彈兵師），被整編為'''親衛隊第三裝甲軍'''（III.SS-Panzerkorps）。

親衛隊第3裝甲師、親衛隊第5裝甲師，被整編為'''親衛隊第四裝甲軍'''（IV.SS-Panzerkorps）。

== 結束 ==
1945年5月7日時任德國聯邦大總統的[[卡爾·鄧尼茲|卡爾·鄧尼茲]]宣布德國簽署無條件投降書，並命令德國所有武裝部隊立即停止抵抗後，武裝親衛隊多數部隊在5月8日向英美聯軍與蘇軍投降。

而後武裝親衛隊因在東歐的[[别动队|别动队]]所積極进行的种族灭绝，违反了多项当时的[[国际战争法|战争法]]，其中包括处决战俘、平民或儿童，加上有管理集中營部隊組成的骷髏師參與其中，因此雖然武裝親衛隊的成員不斷聲稱除了[[別動隊|別動隊]]外，武裝親衛隊士如同國防軍一樣是的單純軍事部隊，'''武裝親衛隊'''與執行惡名昭彰的大屠殺與集中營的一般[[親衛隊|親衛隊]]是不同關係的組織，但事實上武裝親衛隊尤其是第2裝甲師（[[帝國師|帝國師]]）在許多德國佔領區都有屠殺大量平民的記錄，尤以[[格拉訥河畔奧拉杜爾|格拉訥河畔奧拉杜爾]]的屠殺事件為甚，因此在纽伦堡审判時被定義為犯罪組織。

== 武裝SS的軍階 ==
{{德国军事历史}}
{{seealso|武裝親衛隊師列表}}
早期SS的軍階與準軍事組織的命名是承襲其對手SA（[[纳粹冲锋队|纳粹冲锋队]]）的軍階加以變化，軍階的名稱是隨著準軍事組織的名稱而來，因此必須先瞭解準軍事組織的名稱，分為下列幾種：
* 集團Gruppe，相當於陸軍的[[師|師]]Division
* 標旗隊Standarte，相當於陸軍的[[團|團]]Regiment
* 突擊區隊Sturmbann，相當於陸軍的[[營|營]]Batailon
* 突擊隊Sturm，相當於陸軍的[[連|連]]Kompanie
* 下級突擊隊Untersturm，相當於陸軍的[[排|排]]Zug
* 群隊Schar，相當於陸軍的[[班|班]]Gruppe

基本上武裝SS軍階是承襲其對手SA的軍階加以變化。不過武裝SS軍階的肩章與陸軍的肩章大同小異，因此可以由陸軍軍階加以比對。因此按照陸軍軍階比對翻譯後在前面冠上SS是比較合理的。

在將官方面武裝SS比陸軍多了一個軍階{{lang|de|SS-Oberführer}}，可以比照翻譯為SS[[准將|准將]]。在校級及尉級軍官則是相同。在士官方面得先瞭解德國陸軍士官的軍階。陸軍士官在肩章上掛有劍結（sword-knot）的都算是資深[[士官|士官]]，包括Feldwebel、Oberfeldwebel及Stabsfeldwebel。按照德文字典翻譯Feldwebel為上士，Oberfeldwebel可翻譯成一級上士或上級上士，而Stabsfeldwebel可翻譯成本部上士或參謀上士。在德國陸軍Stabsfeldwebel是個榮譽職，是頒發給重新服役滿12年的士官，在早期只頒發給曾參加一戰的老士官。德國陸軍並不像英美陸軍有營團級[[士官長|士官長]]的軍階，德國陸軍是有Hauptfeldwebel這個類似連[[士官長|士官長]]的職位，不過並不是軍階，一般由Oberfeldwebel出任，而一旦Oberfeldwebel被任命為Hauptfeldwebel他的軍階將高於Stabsfeldwebel。Unterfeldwebel翻譯為[[中士|中士]]，Unteroffizier翻譯為[[下士|下士]]。按照陸軍軍階比對，{{lang|de|SS-Oberscharführer}}、{{lang|de|SS-Hauptscharführer}}及{{lang|de|SS-Sturmscharführer}}皆為資深SS士官，可依序翻譯成SS上士、SS一級上士或上級[[上士|上士]]、SS本部上士或參謀上士。在士兵軍階方面SS-Mann和SS-Obermann都可稱為SS二兵，其差異在於SS-Obermann服役超過6個月，SS-Sturmann可翻譯為SS一兵，而{{lang|de|SS-Rottenführer}}可翻譯為SS上兵。

== 武裝SS的部隊分類 ==
希姆萊為了擴張武裝親衛隊部隊可說是使盡渾身解數。早期向親衛隊骷髏部隊招兵買馬，後來徵召警察成立警察師，之後招募佔領國的德裔外籍士兵，最後則招募佔領國的志願兵，使得終戰前的武裝親衛隊队擴充成接近600,000人的兵力，而整個党卫队組織則有近100萬人。這600,000人中只有50%是德國人、25%是德裔外國人、25%是外國人。
