'''武汉轨道交通7号线'''（又称'''武汉地铁7号线'''，英语：'''Wuhan Metro Line 7'''）是[[武汉市|武汉市]]第四条穿越[[长江|长江]]的[[轨道交通|轨道交通]]线路。全线北起武汉市[[黄陂区|黄陂]]前川，南至[[江夏|江夏]]纸坊。<ref name="我市轨道交通7号线一期工程获批">[http://cjrb.cjn.cn/html/2013-09/03/content_5212501.htm 郭文雅，孙巍，杨升. 我市轨道交通7号线一期工程获批<nowiki>[N]</nowiki>. 长江日报，2013年9月3日(2).]</ref>不同于之前的[[武汉地铁|武汉地铁]]线路，7号线采用了更大站距、更大编组和更快时速的设计。<ref name="首列“武汉造”100公里A型地铁车辆下线(图)"/>一期工程和南延线工程分别已于2018年10月1日和2018年12月28日开通运营。<ref name="武汉三大工程明天9时开通"/><ref name="181228cnr"/>

根据目前的规划，7号线由三部分组成：'''一期工程'''、'''纸坊线'''和'''前川线'''。

===一期工程===
一期工程南起[[洪山区|洪山区]][[野芷湖站|野芷湖站]]，北至[[东西湖区|东西湖区]][[园博园北站|园博园北站]]，线路全长31公里，共设19座车站，全部为地下车站。线路跨越[[汉口|汉口]]与[[武昌地区|武昌]]两个地区，途径[[武汉市|武汉市]]的[[洪山区|洪山区]]、[[武昌区|武昌区]]、[[江岸区|江岸区]]、[[江汉区|江汉区]]、[[硚口区|硚口区]]和[[东西湖区|东西湖区]]。线路大致走向为金南一路、园博园东路、淮海路、建设大道、澳门路、三阳路、秦园路、友谊大道、中山路、恒安路和南李路。一期工程设有两个车辆基地，分别是[[长丰停车场|长丰停车场]]和[[野芷湖车辆段|野芷湖车辆段]]，其中[[野芷湖车辆段|野芷湖车辆段]]是武汉地铁线网中的[[A型车|A型车]]基地之一，除7、8号线例常的检修外，还负责[[武汉地铁5号线|地铁5号线]]、[[武汉地铁6号线|6号线]]、7号线和[[武汉地铁8号线|8号线]]的厂架修任务。在[[王家墩东站|王家墩东站]]、[[园博园北站|园博园北站]]和[[野芷湖车辆段|野芷湖车辆段]]内分别设有通往[[武汉地铁2号线|2号线]]、[[武汉地铁6号线|6号线]]和[[武汉地铁8号线|8号线]]的联络线。一期工程设有2座主变电所供电，其中王家墩东主变电所已建成，并且与[[武汉地铁2号线|2号线]]共用；新建秦园路主变电所，未来与[[武汉地铁5号线|5号线]]共用。<ref>湖北招标信息网. [http://www.hbzbw.com/news1/2015-3-26/389896.html 武汉轨道交通5号线工程设计总体总包及单项设计中标公告]. 2015-3-26.</ref>包括越江隧道在内，地铁7号线一期工程总投资达321亿余元，是武汉投资最高的市政交通项目。<ref>[http://www.hb.xinhuanet.com/2012-12/25/c_114117502.htm 武汉321亿元建地铁7号线 系投资最高市政交通项目] {{webarchive|url=https://web.archive.org/web/20130614153800/http://www.hb.xinhuanet.com/2012-12/25/c_114117502.htm |date=2013-06-14 }}2012-12-25 荆楚网</ref>

====越江隧道====
{{main|武汉长江公铁隧道}}
为集约利用过江通道资源，7号线三阳路越江通道采用公铁合建双洞隧道方案，在道路行车道下层布设轨道交通。本项目对于支持重点区域发展、缓解过江交通压力、完善轨道交通网络、优化城市空间布局具有重要的意义。<ref>[http://www.ndrc.gov.cn/fzgggz/nyjt/zdxm/201309/t20130925_560483.html 武汉市轨道交通7号线一期工程可行性研究报告通过批准]. [[中华人民共和国国家发展和改革委员会|中华人民共和国国家发展和改革委员会]]. 2013-08-30.</ref>三阳路越江隧道是世界首条公铁合建的盾构法隧道。<ref>作者不明. 世界首条公铁合建三层超大型盾构隧道开建. 建筑机械化, 2016, 37 (5): 9.</ref><ref>{{Cite web |url = http://www.sh.xinhuanet.com/2017-06/29/c_136404001.htm |title =上海隧道股份：不畏多重难度叠加 全力推进首条公铁合建盾构法隧道|author =李硕 |date =2017年06月29日|publisher =新华网}}</ref><ref name="我市轨道交通7号线一期工程获批"/>武汉三阳路长江隧道工程为武汉轨道交通7号线一期工程过江段，距离长江二桥1.3公里、距离青岛路长江隧道1.9公里。<ref name="2018年武汉地铁将开通7号线和纸坊线">{{Cite web |url = http://mp.weixin.qq.com/s/5i8FnwB4rLWlhIteA6myEw|title =2018年武汉地铁将开通7号线和纸坊线 |date =2018-01-10|publisher = 武汉地铁}}</ref>隧道外径15.2m，内径13.9m，隧道规模位列世界第三，仅次于[[屯门至赤鱲角连接路|香港屯门]]（直径17.6米）、美国西雅图（直径17.4米）地下隧道，其深度大于[[武汉长江隧道|武汉长江隧道]]及[[武汉轨道交通2号线|武汉轨道交通2号线]]越江隧道。三阳路越江隧道分为三层：最上层为公路隧道排烟道；中间层是宽为13.6米的双向6车道，其中行车道10.5米；最下层是轨道交通7号线轨道及电缆廊道、排烟道和逃生通道等；公路高4.5米，具备二层公交、大型巴士通行能力。<ref>{{Cite journal|author=陈馈|author2=冯欢欢|title= 武汉三阳路公铁合建超大直径盾构隧道设计方案研究|journal=现代隧道技术|year=2014|volume=51|issue= 4|pages= 168-177}}</ref><ref>{{Cite journal|author=肖明清|author2=凌汉东|author3=孙文昊|title=武汉三阳路公铁合建长江隧道总体设计关键技术研究|journal=现代隧道技术|year=2014|volume=51|issue= 4|pages=161-167}}</ref>

轨道交通7号线一期工程原计划于2017年底开通运营，但由于越江隧道掘进受阻，全线不得不推迟至2018年底投入运营。<ref name="武汉地铁7号线开通时间">{{cite web |url=http://liuyan.cjn.cn/threads/content?tid=52320|title=武汉地铁7号线开通时间|author=武汉地铁集团|date=2017-10-17|publisher=武汉城市留言板}}</ref>

[[三阳路长江隧道|三阳路长江隧道]]位于7号线[[徐家棚站|徐家棚站]]和[[三阳路站|三阳路站]]之间，左右两线各长约2.6公里，分别已于2016年7月和4月底从[[武昌地区|武昌]]徐家棚始发盾构。<ref>{{Cite web |url =http://news.cnhubei.com/xw/wuhan/201610/t3710002.shtml|title =地铁7号线三阳路越江隧道开建 建成后上跑汽车下跑地铁|author =张晟|date =2016-10-01|publisher =长江日报}}</ref>本隧道是公铁合建的超大断面越江隧道工程，施工风险和技术难度均极高。盾构机在长江底不仅要克服高水压施工的巨大风险还要解决开挖面最高达12.6米岩层（约5层楼高）、石英含量达70%的复合地质等严重损耗刀具的硬骨头，同时还要解决复合地层中夹杂的粘性极强、极易造成刀盘被糊的软黏土层，这些软、硬、粘地层及高水压对盾构施工的的四重考验导致工程建设难度更是叠加式增长，盾构掘进难度大。<ref name="2018年武汉地铁将开通7号线和纸坊线"/><ref>{{Cite web |url =http://liuyan.cjn.cn/threads/content?tid=41457|title =地铁7号线三阳路隧道施工进度及预计完工时间|author =武汉地铁集团|date =2017-09-12|publisher = 武汉城市留言板}}</ref><ref>{{Cite web |url =http://news.cjn.cn/sywh/201702/t2955268.htm|title =春节期间 地铁7号线300人不休息啃“硬骨头”|author =张希为|date =2017-02-03|publisher = 武汉晚报}}</ref>此前官方一直声称7号线一期会在2017年开通运营。然而在“武汉市第十四届人民代表大会第一次会议”上，市长[[万勇|万勇]]所作的《政府工作报告》指出2017年武汉将开通两条地铁线路：[[武汉地铁8号线|8号线一期]]和[[武汉地铁21号线|21号线]]。其中，7号线一期并未在列，官方给出的解释是其工期较紧，尤其是越江盾构“卡脖子”，故7号线未被列入2017年地铁开通目标。<ref>{{cite news|author =汪洋 |url =http://hbrb.cnhubei.com/HTML/hbrb/20170217/hbrb3064740.html |title = 武汉地铁阳逻线将提前1年通车 |publisher = 湖北日报|pages = 3|date = 2017-01-17}}</ref>

虽然过江段遭遇障碍，但是一期工程的其他区间却进展较为顺利，市民期望7号线能分段运行，然而地铁集团否决了这一可能，具体的官方回复如下：
 {{Cquote|轨道交通7号线控制中心设置在[[汉口|汉口]]的[[三金潭车辆段|三金潭车辆段]]内，车辆检修及试车基地却位于[[武昌|武昌]]的[[野芷湖车辆段|野芷湖车辆段]]，目前所有设备系统均按照全线通车的要求进行设计，并已开始安装。临时分段开通运营相当于7号线一期工程分成两个独立运营的线路，将存在大量控制系统重新调整设计，增加大量临时控制设备及临时检修设施等问题，后期贯通运营前，需中断运营，对相关控制系统软件进行升级倒切，全线重新完成联调联试、安全认证、验收等程序。综合以上因素，临时分段开通运营既增加了工程投资，又延长了全线通车的工期，同时，反复调整控制系统软件设计也不利于系统的安全稳定。
综上所述，轨道交通7号线一期工程现阶段不宜采取临时分段开通运营方案。我公司（[[武汉地铁集团|武汉地铁集团]]）将加快施工进度，在确保质量安全的前提下力争尽早开通轨道交通7号线一期工程。<ref>{{Cite web |url =http://liuyan.cjn.cn/threads/content?tid=5728|title =地铁7号线2017年底分段运营吗？|author =武汉地铁集团|date =2017-05-17|publisher =武汉城市留言板}}</ref><ref>{{Cite web|url =http://www.ctdsb.net/html/2017/0520/hubei94567.html|title =地铁7号线不宜临时一分为二，明年上半年将全线贯通|author =潘锡珩, 郭徽|date =2017-05-20|publisher =楚天都市报|deadurl =yes|archiveurl =https://web.archive.org/web/20171002071055/http://www.ctdsb.net/html/2017/0520/hubei94567.html|archivedate =2017年10月2日|df =}}</ref>}}

===纸坊线===
'''纸坊线'''，又被称为'''轨道交通7号线南延线'''，由一期工程终点站[[野芷湖站|野芷湖站]]出发继续向南延伸，直至[[江夏区|江夏区]]青龙山地铁小镇站。线路大致沿文化大道、文化路、纸坊大街走行，先后途径洪山区、江夏大桥新区和纸坊老城区。线路长约17km，设站7座，均为地下车站。<ref>[http://www.whrt.gov.cn/public_forward.aspx??url=public_content_page.aspx?newid=3A953CA6C2E6E08248257FA40030B08A 27号线一期工程岩溶处理项目资格预审公告]计划合约部 2016年4月29日</ref><ref>[http://hb.qq.com/a/20141228/002741.htm 地铁21和27号线昨同日开建 2018年通车试运营]荆楚网-楚天都市报 2014-12-28</ref><ref>[http://www.hb.xinhuanet.com/2014-12/28/c_1113800036.htm 武汉地铁21和27号线昨同日开建 连黄陂新洲通江夏] {{webarchive|url=https://web.archive.org/web/20150924025456/http://www.hb.xinhuanet.com/2014-12/28/c_1113800036.htm |date=2015-09-24 }}</ref>本段于2018年12月28日开通，与一期工程贯通运营<ref name="181228cnr"/>。

===前川线===
'''前川线'''，又被称为'''轨道交通7号线北延线'''，起于[[黄陂|黄陂]]前川黄陂广场站，沿黄陂大道、双凤大道、川龙大道、盘龙大道、汤云海路、环湖中路南北向走行，先后经过黄陂前川新城区、武汉临空经济区、黄陂盘龙城区、东西湖区，止于7号线一期工程起点[[园博园北站|园博园北站]]。线路全长36.47km，设站11座，分别是黄陂广场站、百泰路站、北车基地站、余彭塆站（预留）、横店站、临空北路站、天阳路站、腾龙大道站、巨龙大道站、汤云海站、马池站。结合线路长度及车辆段（场）用地条件，7号线北延线设置横店停车场（预留）和前川车辆段各一处。本段已于2017年12月18日开工<ref>郑汝可. [http://cjrb.cjn.cn/html/2017-12/19/content_43129.htm 武汉首条地铁环线 连接所有中心城区]. 长江日报. 2017年12月18日: 1.</ref>，预计2019年建成通车。<ref name="7号线北延线（前川线）工程设计总体总包及单项设计招标公告"/><ref>{{Cite web |url =http://liuyan.cjn.cn/threads/content?tid=71634|title =地铁7号线北延长线有哪些站点|author =武汉地铁集团|date =2017-11-29|publisher = 武汉城市留言板|language = 中文}}</ref><ref>{{Cite web|url =http://www.huangpi.gov.cn/zwgk/xwzs/zxyw/201702/t20170210_75056.html|title =前川开启地铁元年 掀开黄陂发展新篇章 轨道交通进前川计划今年开工建设|author =胡思洁|date =2017年2月10日|publisher =武汉市黄陂区人民政府|language =中文}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

==影响==
===为配合[[园博园|园博园]]顺利开园，两站提前施工===
2015年9月第十届中国（武汉）园林博览会在[[武汉|武汉]]开幕，应园博开园期间周边无工地的要求，地铁7号线{{站|园博园||武汉地铁}}（原长丰）和{{站|园博园北}}（原东方马城）两站提前施工。2014年8月，地铁7号线首台[[盾构机|盾构机]]在[[园博园北站|园博园北站]]-[[园博园站|园博园站]]区间开掘<ref>[http://news.cnhubei.com/xw/wuhan/201408/t3026704.shtml 地铁7号线首台盾构机开掘 速度为武汉在建地铁中最快]2014-08-27 楚天金报</ref>，2015年3月2日，该区间[[隧道|隧道]]双向贯通<ref>[http://news.cnhubei.com/xw/wuhan/201503/t3195959.shtml 武汉地铁7号线首个区间隧道实现双线贯通]2015-03-04 荆楚网</ref>。2014年11月[[园博园站|园博园站]]主体结构顺利封顶，成为全线首座封顶的车站。车站主体的完工，为园博园2015年9月开园、修建园博园东路创造了条件。<ref>[http://house.cnhubei.com/2015/0305/207406.shtml 武汉地铁7号线长丰站 主体率先完工]2015-03-05 长江日报</ref>施工方仅用5个月，就完成了[[地铁站|地铁站]]主体结构的施工，成为武汉在建地铁施工最快的车站。2014年6月1日，园博园站开始浇筑第一块主体结构板，到最后主体封顶，刚好用了5个月。正常情况下，一座标准地铁站主体结构施工需要10-12个月工期才能完成，为配合园博园建设，园博园站仅用了5个月就完工，创下武汉在建地铁车站施工的新纪录。<ref>[http://whwb.cjn.cn/html/2014-11/04/content_5385065.htm 地铁7号线首座车站主体封顶] {{webarchive|url=https://web.archive.org/web/20160817114103/http://whwb.cjn.cn/html/2014-11/04/content_5385065.htm |date=2016-08-17 }}武汉晚报 2014年11月04日</ref>2015年8月，园博园北站完成主体结构施工，这也是7号线一期第二个建成的站点。<ref>[http://hb.sina.com.cn/news/b/2015-08-08/detail-ifxftkpv6915158.shtml?pos=108 武汉地铁7号线东方马城站主体结构完工]2015-08-08 荆楚网-楚天金报</ref>

===缓解[[武汉南湖|南湖]]地区居民的出行难问题===
位于[[武汉市|武汉市]]武昌南部的南湖片区是武汉最大的“睡城”之一，三面被[[铁路|铁路]]设施（[[武昌车辆段|武昌车辆段]]、[[武昌南编组站|武昌南编组站]]、[[京广铁路|京广铁路]]、[[武九铁路|武九铁路]]和[[武咸城际铁路|武咸城际铁路]]等）阻隔，导致南湖片区的北、西、南3个方向都有铁路线，将南湖包围。目前，南湖片区居民的进出主要需经过3个铁路[[涵洞|涵洞]]：南湖路上的长虹桥涵洞、晒湖附近的平安路铁路涵洞、张黄新村岭南路铁路涵洞。仅南湖向东方向不需过涵洞，可通过文治街上书城路、珞狮路（二环线）进入[[市中心|市中心]]。<ref>[http://www.ctdsb.net/html/2017/0327/hubei87652.html 武汉市长：痛定思痛开展南湖大会战，尽可能少扰民] {{Wayback|url=http://www.ctdsb.net/html/2017/0327/hubei87652.html |date=20170327170236 }}2017-03-27 长江日报</ref><ref>[http://www.cnhan.com/html/cjrb/20160414/309060.htm 三面被铁路线包围 出行基本靠3个涵洞]2016-04-14 长江日报</ref>所以长期以来出入南湖片区的[[交通|交通]]都十分拥堵，居住在此片区的几十万居民更是将其戏称为：“早上出不去，晚上进不来。<ref>[http://www.changjiangtimes.com/2014/09/487471.html 南湖“早上出不去 晚上进不来”或更严重]2014-09-12 长江商报</ref>”虽然近年进出南湖片区的市政道路的修建一直在跟进，但情况依然不容乐观。<ref>[http://hb.qq.com/a/20150709/009808.htm 40万南湖居民出行难 6条规划路搁置了20年(图)]荆楚网 2015-07-09</ref>轨道交通7号线一期工程在南湖片区共设有5座车站：[[瑞安街站|瑞安街站]]、[[建安街站|建安街站]]、[[湖工大站|湖工大站]]、[[板桥站|板桥站]]和[[野芷湖站|野芷湖站]]，南湖片区的居民们寄希望于轨道交通7号线能改变他们的出行窘境。<ref>[http://news.fdc.com.cn/cjgh/683602_3.shtml 城建实探：南湖？难湖！地铁封闭施工南湖花园出行雪上加霜]亿房网 2014年10月04日</ref>
