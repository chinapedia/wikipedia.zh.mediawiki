'''武汉王家墩机场'''（{{airport codes|WJD|ZHWT}}），即'''中国人民解放军空军汉口机场'''，位於[[中华人民共和国|中华人民共和国]][[湖北省|湖北省]][[武汉市|武汉市]][[江汉区|江汉区]]王家墩，是军民两用[[机场|机场]]。2007年停用。

== 历史 ==
王家墩机场始建于1931年，因位于[[汉口|汉口]]王家墩，称王家墩机场<ref name=whdq>{{Cite web |url = http://www.whfz.gov.cn:8080/pub/dqwx/delsz/csjsz/DQP/DSZ3/200801/t20080101_19466.shtml |title =  武汉市志(1980-2000) 第一卷：总类 城市建设志 第七篇 交通设施 第四章 民航设施 第二节　机场 |author =  |date =  |publisher = 武汉市地情文献 |language =  |accessdate = 2017-07-13 |quote =  }}</ref>。

[[抗日战争|抗日战争]]爆发后，1937年10月，[[苏联空军志愿队|苏联空军志愿队]]开始援华。[[日军|日军]]攻占[[中华民国|中华民国]]首都[[南京|南京]]后，中国政府的抗战中心迁往武汉，日军随即加强空袭武汉。当时，[[中国空军|中国空军]]基地已迁往汉口、[[南昌|南昌]]、[[襄樊|襄樊]]等地，苏联空军志愿队以汉口、南昌、[[兰州|兰州]]等地作为基地。1938年2月至5月武汉空战、1938年6月至10月[[武汉会战|武汉会战]]期间，中国空军及苏联空军志愿队以汉口王家墩机场等为基地，联合抗击来犯的日军飞机，掌握了武汉制空权<ref name=banjing>{{Cite web |url = http://roll.sohu.com/20130815/n384196963.shtml |title = 1938年 从王家墩划出“正义半径” |author =  |date = 2013-08-15 |publisher = 搜狐 |language =  |accessdate =  |quote =  }}</ref>。1938年2月18日，在武汉空战中，苏联空军志愿队飞行员击落12架日本飞机<ref name=banjing/>，中国空军自王家墩机场出发与日军首次交战<ref name=jueqi/>。1938年2月23日是[[苏联红军|苏联红军]]节20周年，苏联空军志愿队分两批自王家墩机场出动，轰炸了[[台湾|台湾]][[日本空军|日本空军]][[松山机场|松山机场]]。1938年3月12日，中国空军飞行员[[汤卜生|汤卜生]]驾驶[[单翼普式侦察机|单翼普式侦察机]]，从王家墩机场出发，进入已被日军占领的南京上空，飞谒[[中山陵|中山陵]]。1938年4月29日是[[日本|日本]][[裕仁天皇|裕仁天皇]]生日“[[天长节|天长节]]”，侵华日军海军航空兵第十二航空队派36架九六式驱逐机、18架九六式重轰炸机空袭武汉，中国空军自王家墩机场起飞、苏联空军志愿队自孝感机场起飞，击落日机21架，中苏方面损失12架，中国空军飞行员[[陈怀民|陈怀民]]驾机撞向敌机阵亡，成为世界空战史上与敌机对撞的第一人。1938年5月19日，由中国空军飞行员[[徐焕升|徐焕升]]领队的两架[[B-10轰炸机|马丁-139WC轰炸机]]从王家墩机场起飞，前往[[日本|日本]][[九州_(日本)|九州]]散发传单，成为[[二战|二战]]开战後首次飞入[[日本|日本]]领空的外国军机<ref name=banjing/><ref name=jueqi>{{Cite web |url =http://www.whfz.gov.cn/mcjg/201308/t20130801_42862.shtml  |title = 王家墩：飞机场上崛起的商务区 |author =  |date = 2013-08-01 |publisher =  武汉市情网|language =  |accessdate =  |quote =  }}</ref><ref name=ying>{{Cite web |url =http://news.xinhuanet.com/mil/2005-06/28/content_3146293.htm  |title =怀念武汉上空的鹰－－踏访武汉王家墩机场  |author =  |date = 2005-06-28 |publisher = 新华网 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://www.people.com.cn/GB/guoji/14549/1934530.html |title =  史海回眸：1938，中国空军到日本撒传单|author =  |date = 2003-06-25 |publisher =人民网  |language =  |accessdate =  |quote =  }}</ref>。

1950年，[[中国人民解放军空军|中国人民解放军空军]]接管王家墩机场<ref name=whdq/>。此后在王家墩机场设'''中国人民解放军空军王家墩场站'''。1960年代初，升格组建'''中国人民解放军空军王家墩基地'''。1970年代，空军王家墩基地撤销，恢复为'''中国人民解放军空军王家墩场站'''<ref>{{Cite web |url =http://www.hbjc.gov.cn/jcyry/jcyld/201503/t20150319_629828.shtml  |title = 何畏 |author =  |date = 2015-03-19 |publisher =  湖北省人民检察院|language =  |accessdate =  |quote =  }}</ref>。1993年，改称'''[[中国人民解放军空军汉口场站|中国人民解放军空军汉口场站]]'''<ref name=whdq/>。空军王家墩机场（1993年起为空军汉口机场）位于东经114度14分30秒，北纬30度35分30秒，距[[武汉长江大桥|武汉长江大桥]]6.4公里，可用起飞跑道距离2400米，机场总面积502.8万平方米。1987年前，空军王家墩机场主要供空军歼击机使用，1987年后改为主要供空军运输机使用，1991年对跑道局部翻修加固，以便适应部队运输机技术要求<ref name=whdq/>。

1976年以及1983年10月，[[武汉南湖机场|武汉南湖机场]]两次翻修跑道时，民航湖北省管理局均曾临时使用空军王家墩机场。1979年6月到1984年5月，经停武汉的民航大型飞机均在空军王家墩机场起降，民航湖北省管理局设有航站<ref name=whdq/>。

1985年12月9日到2000年12月31日，武汉航空公司以空军汉口机场为营运基地，主要使用其中两个机坪。1987年5月前，旅客待检、安检、候机、到达、现场办公等设施集中在跑道东侧中部驻场部队外场指挥中心，设施总面积1200平方米（其中自建450平方米、租用750平方米）。1997年5月后，除了航管中心、指挥塔台部分租用驻场部队设施外，外场自建设施10000平方米，集中在机场东北侧，自成独立的保障体系。主要有框架彩色浪板结构的候机楼2700平方米，到达厅400平方米，快餐厅400平方米，食品仓库、货运仓库、外场办公用房等6400平方米<ref name=whdq/>。1999年时，该机场客运量1608319人次，排在国内第19名<ref name=jueqi/>。

另外，武汉直升机通用航空有限公司在空军汉口机场设飞行基地，占地面积2000平方米，建有机库、维修工作间、办公室等设施。武汉市公安局飞行大队在驻空军汉口机场的空军十八厂设飞行基地，由驻场部队负责航空管制<ref name=whdq/>

1995年后，因武汉市城市发展，机场所在地已成为武汉市中心城区，国家及中央军委将机场搬迁列入计划<ref name=whdq/>。1997年，开始筹备建设空军阳逻新机场<ref name=touru/>。武汉市决定通过市场经济手段，筹资为空军还建空军阳逻新机场，作为空军汉口机场的新址，再以空军移交的占地面积4000亩的王家墩机场为中心，重新规划周边11.36平方公里范围内的建设，建成[[武汉中央商务区|武汉中央商务区]]。2001年7月，国务院和中央军委批复同意王家墩机场迁建<ref>{{Cite web |url =  http://news.carnoc.com/list/10/10558.html|title = 武汉王家墩机场迁建顺利　新机场年内动工 |author =  |date = 2001-08-28 |publisher = 民航资源网 |language =  |accessdate =  |quote =  }}</ref>。2001年8月，王家墩机场迁建军地联合领导小组成立，军方组长是[[广州军区空军|广州军区空军]]副司令员[[高守维|高守维]]，地方组长是武汉市市长[[王守海|王守海]]<ref>{{Cite web |url =  http://news.carnoc.com/list/8/8545.html|title = 武汉王家墩机场迁建领导小组成立 |author =  |date = 2001-08-28  |publisher = 民航资源网 |language =  |accessdate =  |quote =  }}</ref>。2003年1月，武汉市市长[[李宪生|李宪生]]在武汉市第十一届人民代表大会第一次会议所作的《政府工作报告》中，正式提出在汉口王家墩机场原址建设“武汉王家墩商务区”<ref name=jueqi/>。汉口王家墩机场迁建后，武汉市100余平方公里的城区实现无净空限制。2007年6月6日，空军阳逻新机场竣工典礼举行，空军副司令员[[杨东明|杨东明]]等领导剪彩。空军汉口场站迁至空军阳逻新机场，位于[[武汉市|武汉市]][[新洲区|新洲区]][[阳逻街道|阳逻街道]]毛集村<ref name=touru>{{Cite web |url = http://www.yangluo.cn/article-12144.html |title =  空军阳逻新机场正式投入使用|author =  |date =  2007-06-07 |publisher = 阳逻在线 |language =  |accessdate =  |quote =  }}</ref>。2007年12月25日，王家墩机场停止使用。其军用功能由空军阳逻新机场取代，民用客运功能由[[武汉天河国际机场|武汉天河国际机场]]取代。王家墩机场原址成为[[武汉中央商务区|武汉中央商务区]]<ref name=jueqi/>。

== 事故 ==
*2000年6月22日，由恩施飞往武汉的[[武汉航空343号班机空难|武汉航空343号班机]]试图降落在王家墩机场时遭遇雷击坠毁，造成49人死亡。
