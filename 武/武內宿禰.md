'''武內宿禰'''（{{jpn|j = 武内宿禰|hg = たけのうち の すくね/たけうち の すくね/たけち の すくね|rm = Takenouti no tsukune/Takeuti no tsukune/Taketi no tsukune|kyu = |hgt = }}，84年？－367年？）也作'''建內宿禰'''，根據《[[古事記|古事記]]》和《[[日本書紀|日本書紀]]》的說法，是[[倭朝廷|倭朝廷]]初期的一位大臣。他曆仕[[景行天皇|景行]]、[[成務天皇|成務]]、[[仲哀天皇|仲哀]]、[[應神天皇|應神]]、[[仁德天皇|仁德]]五朝，為[[倭朝廷|倭朝廷]]棟樑之臣。他是後來日本中央朝廷[[紀氏|紀氏]]、[[巨勢氏|巨勢氏]]、[[平群氏|平群氏]]、[[葛城氏|葛城氏]]、[[蘇我氏|蘇我氏]]等[[豪族|豪族]]共同的祖先，後來被尊為神。但其生平事蹟有很大部分屬於傳說類型。
==事蹟==
相傳他是[[孝元天皇|孝元天皇]]的玄孫，與稚足彥尊（後來的[[成務天皇|成務天皇]]）在同年同月同日出生。[[景行天皇|景行天皇]]廿五年（95年）七月，曾奉命前往[[北陸道|北陸]]、[[東國|東國]]視察，並建議天皇征討[[蝦夷|蝦夷]]。因此，在平定南方的[[熊襲|熊襲]]之後，景行天皇於40年派[[日本武尊|日本武尊]]征討蝦夷，佔領[[陸奧國|陸奧]]地區。

[[仲哀天皇|仲哀天皇]]駕崩後，[[神功皇后|神功皇后]]自[[三韓|三韓]]戰場歸來，擁立自己的兒子[[應神天皇|應神天皇]]即位。應神天皇的異母兄[[麝坂皇子|麝坂皇子]]和[[忍熊皇子|忍熊皇子]]起兵欲奪位，神功皇后將年幼的天皇託付給武內宿禰照顧，自己則親自討伐二皇子。在武內宿禰和[[武振熊命|武振熊命]]的幫助下，神功皇后平定了這場叛亂，鞏固了天皇的皇位。

應神天皇在位期間，武內宿禰率[[渡來人|渡來人]]造[[韓人池|韓人池]]。異母弟[[甘美內宿禰|甘美內宿禰]]讒其謀反，武內宿禰以[[盟神探湯|探湯]]的方式，向天皇表示自己是忠誠的。

武內宿禰在《[[日本書紀|日本書紀]]》中最後一次登場的時間是[[仁德天皇|仁德天皇]]五十年（362年）。《[[公卿補任|公卿補任]]》和《[[水鏡|水鏡]]》二書稱武內宿禰於仁德五十五年（368年）死去，《[[帝王編年記|帝王編年記]]》則稱是七十八年（390年）。其逝世時的年齡有280歲、295歲、306歲、312歲、360歲等不同的說法。但如此高齡逝世的記載引起了後世[[疑古|疑古]]的日本學者的懷疑，認為其真實性不可靠，甚至武內宿禰本人就是虛構人物。

相傳在今[[奈良縣|奈良縣]][[御所市|御所市]]的[[室宮山古墳|室宮山古墳]]是武內宿禰和兒子[[葛城襲津彥|葛城襲津彥]]的墳墓。

==後世的尊崇==
[[File:Takeuchi1JPY.JPG|thumb]]
日本人將武內宿禰尊為神，祭祀於[[神社|神社]]之中。在[[鳥取縣|鳥取縣]][[鳥取市|鳥取市]]的[[宇倍神社|宇倍神社]]以及[[福井縣|福井縣]][[敦賀市|敦賀市]]的[[氣比神宮|氣比神宮]]都對他進行了祭祀。[[福岡縣|福岡縣]][[久留米市|久留米市]][[高良大社|高良大社]]的祭神'''高良玉垂神'''可能也是武內宿禰。此外，日本人也將武內宿禰與[[應神天皇|應神天皇]]、[[神功皇后|神功皇后]]一同供奉於全國各個[[八幡宮|八幡宮]]（[[八幡社|八幡社]]）之中。

昔日，在[[皇國史觀|皇國史觀]]的影響下，日本學者把武內宿禰當作忠臣，大肆頌揚他。在1889年（[[明治|明治]]廿二年）發行的壹圓[[日本銀行券|日本銀行券]]中，正面人物頭像為武內宿禰。此後的1889年的甲五圓券、1916年丙五圓券、1942年的丁貳百圓券、1943年的壹圓券，其正面人物頭像也是武內宿禰。
