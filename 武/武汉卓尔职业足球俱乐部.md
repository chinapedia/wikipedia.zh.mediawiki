'''武汉卓尔足球俱乐部'''，是一家位于[[湖北省|湖北省]][[武汉市|武汉市]]的足球俱乐部，主场为[[五环体育中心|五环体育中心]]。俱乐部成立于2009年2月17日，以2008赛季退出[[中超联赛|中超联赛]]的'''[[湖北武汉职业足球俱乐部|武汉光谷]]'''剩余资源为基础组建，目前参加[[中国足球超级联赛|中国足球超级联赛]]。

== 俱乐部历史 ==
2008年10月，[[湖北省|湖北省]]原有的职业俱乐部[[湖北武汉职业足球俱乐部|武汉光谷]]因为抗议[[中国足协|中国足协]]处罚不公而在[[中国足球超级联赛|中超]]退赛，被中国足协取消联赛注册资格，不能再参加中国足协举办的各级联赛。武汉队的退赛使湖北再沒有参加职业联赛的球队，同时武汉光谷的大部分主力也因此转会或租借到其他球队。武汉队要想重新参加联赛，必须借壳购买一个联赛资格或者重新注册新的俱乐部从乙级联赛起步。在多方寻求壳资源无果的情况下，原俱乐部股东之一的湖北省足协为了延续湖北足球的火种，接收未能转会一线队员和梯队球员，以原有的武汉俱乐部U-19梯队（即湖北十一运队）为基础，成立新足球俱乐部。<ref name="foundation" />2009年2月27日，以原湖北武汉职业足球俱乐部留守武汉的部分一线队员和U19梯队为班底重组的'''湖北绿茵'''在[[中国足协|中国足协]]注册成功，从[[中国足球乙级联赛|乙级联赛]]开始肩负着湖北足球重返职业联赛的使命。<ref>[http://www.hbfootball.com/html/2009/0227/1644.shtml 快讯：中国足协领导人终盖章 湖北绿茵队注册成功]</ref>

在2009赛季的中国足球乙级联赛中，凭借原中超球队武汉光谷球员的班底，湖北绿茵展示出了非同一般的实力，10场比赛绿茵取得7胜3平是分区赛中唯一一支保持不败的球队，获得分赛区小组第一直进入半决赛。半决赛，湖北绿茵在兩回合的比赛合计以1-0的比分将[[杭州三超|杭州三超]]淘汰出局，获得2010年[[中甲联赛|中甲联赛]]的参赛资格，从而也完成湖北足球重返职业联赛的使命。<ref>{{cite news|title=绿茵1-0小胜杭州三超 湖北足球宣告重回到中甲 |url=http://sports.sohu.com/20091126/n268494574.shtml |author= |publisher=搜狐体育 |date=2009年11月26日 |accessdate=2017年1月19日 }}</ref>总决赛冠军争夺战中，湖北绿茵在[[点球|点球]]大战告负，不敌[[湖南湘涛|湖南湘涛]]获得亚军。<ref>{{cite news|title=点球大战惜败于湖南湘涛 绿茵无缘问鼎中乙冠军 |url=http://sports.sohu.com/20091129/n268546362.shtml |author= |publisher=搜狐体育 |date=2009年11月29日 |accessdate=2017年1月19日 }}</ref>

2010年湖北绿茵在[[中国足球甲级联赛|中甲联赛]]取得了第五名。

2011年1月10日，武汉中博置业集团联合湖北东方国际旅行社与鲁巷广场共同出资2300万元收购湖北绿茵，俱乐部正式更名为'''湖北武汉中博职业足球俱乐部'''。<ref>{{cite web |url=http://sports.sina.com.cn/b/2011-01-10/17175403982.shtml |title=湖北绿茵股权转让更名中博 新赛季投入2300万打中甲 |author= |date=2011年1月10日 |publisher=新浪体育  |accessdate=2018年8月14日}}</ref>

2011年12月俱乐部再次易主，原本亨达睿投资管理有限公司接手球队，但由于武汉亨达睿未能如约履行先期支付600万股权转让款的义务，因此湖北中博正式解除与武汉亨达睿签署的《股权转让协议》和《补充协议》并保留追究权利，事件最终以“闹剧”收场<ref>[http://roll.sohu.com/20111213/n328776751.shtml 亨达睿中博互相解除“合同”]</ref>。但一日后，武汉中博置业集团与[[卓爾集團|卓爾集團]]签订转让协议，俱乐部更名为'''武汉卓尔职业足球俱乐部'''<ref>[http://news.cnhubei.com/ctjb/ctjbsgk/ctjb01/201112/t1921487.shtml 湖北首富出手鄂足起死回生 卓尔接盘湖北中博]</ref>。

2012年赛季结束，以亚军身份升级到中超。<ref>[http://sports.163.com/12/1110/16/8FVBKET900051C8L.html 回首武汉四年冲超路]</ref> 2013年4月21日，武汉卓尔在联赛前六轮仅以1平5负的成绩排名联赛倒数第二位，主教练[[郑雄_(足球教练)|郑雄]]宣布辞职。第二天，[[留比萨·图巴科维奇|留比萨·图巴科维奇]]成为球队的新任主教练。<ref>[http://sports.sina.com.cn/j/2013-04-22/14466532620.shtml 武汉卓尔董事长宣布图拔任主帅 曾执教鲁能夺五冠]</ref>2013年8月18日，武汉卓尔客场以1-5慘败给广州富力，同時遭遇客场8连败，赛后武汉卓尔俱乐部管理层在广州随即召开会议，並透过官方微博宣佈解除主教练图拔科维奇的职务。<ref>{{cite web |url=http://sports.sina.com.cn/j/2013-08-18/23196730876.shtml |title=武汉卓尔宣布与主帅图拔科维奇解约 同时撤换队长 |author= |date=2017年8月18日 |publisher=新浪体育  |accessdate=2017年1月16日}}</ref>2013年10月5日，中超联赛第27轮，武汉卓尔客场0-0战平青岛中能，从而提前三轮从[[中国足球超级联赛|中超]]降级。<ref>[http://sports.sina.com.cn/j/2013-10-05/17346815167.shtml  中超-乔尔中柱斯托杨失良机 卓尔0-0青岛提前降级]</ref>

2015年[[卓爾集團|卓爾集團]]出售俱樂部與卓尔控股有限公司，為主席[[閻志|閻志]]、[[閻格|閻格]]全資擁有，代價為人民幣20,630,000 元。<ref>{{cite web |url=http://stock.qq.com/a/20151215/016017.htm?stockcode=hk02098&version=2 |title=卓尔发展出售卓尔足球俱乐部权益予控股股东 |author= |date=2015年12月15日 |publisher=腾讯证券  |accessdate=2017年1月16日}}</ref>

2016年7月，武汉卓尔宣布前意甲[[尤文图斯|尤文图斯]]主教练[[奇罗·费拉拉|费拉拉]]担任球队主帅，带队征战本赛季剩余的中甲联赛。<ref>{{cite web |url=http://sports.sina.com.cn/china/b/2016-07-05/doc-ifxtsatn8153828.shtml |title=武汉卓尔官宣费拉拉上任 中甲上演"尤文德比" |author= |date=2016年7月15日 |publisher=新浪体育  |accessdate=2017年1月16日}}</ref>武汉卓尔最后五轮取得全胜，以第六名的成绩结束2016赛季征程。

2017赛季武汉卓尔承办了2017赛季[[中甲|中甲]]开幕式。<ref>{{cite|url=http://sports.sina.com.cn/china/b/2017-02-04/doc-ifyafcyw0195119.shtml |title=新赛季中甲开幕式花落武汉 卓尔豪掷四亿欲冲超 |date=2017年2月4日 |publisher=新浪体育}}</ref>由于上赛季使用的新华路体育场将进行改建，本赛季武汉卓尔主场移至[[武汉体育中心|武汉体育中心]]。<ref>{{cite|url=http://www.sohu.com/a/125704104_501200 |title=费拉拉官宣！2017卓尔移师国足魔鬼主场---武汉体育中心 |date=2017年2月7日 |publisher=搜狐}}</ref> 3月20日，联赛仅仅进行两轮后，俱乐部宣布由于战绩欠佳，经过双方的友好协商，主教练[[奇罗·费拉拉|费拉拉]]正式离开武汉卓尔足球俱乐部。<ref>{{cite|url=http://sports.sina.com.cn/china/b/2017-03-20/doc-ifycnpit2436287.shtml |title=下课第一人!卓尔主帅费拉拉下课 1平1负冲超挖坑 |date=2017年3月20日 |publisher=新浪体育}}</ref>3月30日，[[唐尧东|唐尧东]]成为武汉卓尔新任主教练。<ref>{{cite|url=http://sports.sohu.com/20170330/n485614914.shtml |title=卓尔公告唐尧东任球队主教练 前辽足主帅任助教 |date=2017年3月30日 |publisher=搜狐体育}}</ref> 7月8日，武汉卓尔官方宣布主帅唐尧东下课，助教[[陈洋|陈洋]]接任主帅。<ref>{{cite|url=http://sports.sina.com.cn/china/b/2017-07-09/doc-ifyhwehx5407672.shtml |title=赛季第二次换帅!卓尔官宣唐尧东下课 陈洋接任 |date=2017年7月8日 |publisher= 新浪体育}}</ref> 9月23日，武汉卓尔客场1：1战平[[杭州绿城|杭州绿城]]，提前三轮冲超失败。

2017年10月10日，俱乐部宣布进行人事调整，[[王小松|王小松]]出任俱乐部董事长兼行政总裁，中国国家队领队兼助教[[李铁|李铁]]出任俱乐部总经理兼体育总监。<ref>{{cite|url=http://news.cnhubei.com/xw/ty/201710/t4004698.shtml |title=李铁出任卓尔俱乐部总经理 2018年武汉队将继续冲超 |date=2017年10月11日 |publisher= 湖北日报网}}</ref>11月16日，李铁成为武汉卓尔主教练。<ref>[http://sports.sina.com.cn/china/b/2017-11-16/doc-ifynvxeh5252523.shtml 武汉卓尔官宣李铁成为新帅 新赛季全力冲击中超]</ref>

2018年10月6日，尽管主场2-2战平[[梅县铁汉|梅县铁汉]]，但由于同时进行的比赛中[[深圳佳兆业|深圳佳兆业]]1-2败给[[内蒙古中优|内蒙古中优]]，再加上次日进行的比赛中[[浙江绿城|浙江绿城]]0-0战平[[石家庄永昌|石家庄永昌]]，武汉卓尔提前三轮夺得中甲冠军，并重返中超<ref>{{cite|url=http://sports.ifeng.com/a/20181006/60096965_0.shtml|title=中甲-冲超！汉军归来！卓尔时隔五年重返中超舞台|date=2018年10月6日|publisher=凤凰网体育}}</ref>。
