'''武汉交通职业学院'''，（{{lang-en|'''Wuhan Technical College of Communications'''}}，缩写：{{lang|en|'''WHTCC'''}}）简称'''武交院'''，武汉交通职业学院原名[[武汉交通管理干部学院|武汉交通管理干部学院]]（由[[中华人民共和国交通部|中华人民共和国交通部]]主管）。因学院由[[中华人民共和国交通部|交通部]]划转[[湖北省教育厅|湖北省教育厅]]管理，校名变更为武汉交通职业学院。

==办学状况==
学校有专任教师480人，教授、副教授154人。校园占地面积1025亩，建筑面积近30万平方米，在校学生1.6万余人。固定资产总值9.04亿元，教学仪器总值6456.71万元，馆藏各类书刊（含电子图书）160余万册。学校建有实验实训中心92个，校外实训基地152个，国家级实训基地4个，省级实训基地5个。<ref>{{cite book |url = http://www.whtcc.edu.cn/xxgy/xxjj.htm |title = 学校简介简介|date = 2014年3月7日 |accessdate = 2014年10月6日}}</ref>

==院系设置==

武汉交通职业学院设有航海学院：[[船舶|船舶]]与海洋工程学院、[[航海|航海]]学院、[[物流|物流]]学院、[[交通工程|交通工程]]学院、[[汽车工程|汽车工程]]学院、[[机电工程|机电工程]]学院、[[电子与信息工程|电子与信息工程]]学院、[[经济管理|经济管理]]学院、[[旅游与商务|旅游与商务]]学院、[[艺术|艺术]]学院、[[思想政治|思想政治]]理论课部、公共课部、[[实践教学|实践教学]]中心、现代教育信息中心。<ref> {{Cite book |accessdate = 2014年10月6日 |url = http://www.whtcc.edu.cn/main/index5/ |title = 院系设置 |deadurl = yes |archiveurl = https://web.archive.org/web/20141006150536/http://www.whtcc.edu.cn/main/index5/ |archivedate = 2014年10月6日 }} </ref>

==学校领导==

党委书记 [[盛建龙|盛建龙]]，主持学校党委全面工作，分管学校办公室（党委办公室、校长办公室、校友办公室），联系交通工程学院。
	
校长、党委副书记 [[王进思|王进思]]，主持学校行政全面工作，分管质量管理与评估处（教学督导办公室、评估办公室）、人事处（外事处），联系机电工程学院。
	
党委委员、副校长 [[梁世翔|梁世翔]]，分管质量管理与评估处（质量管理办公室）、校企合作交流处（培训管理处、就业工作处、就业指导中心）、科研处（学报编辑部、高职教育研究所）、资产与后勤管理处（招投标管理办公室）、船员培训中心、图书馆（档案馆），联系航海学院、物流学院。
	
纪委书记、党委副书记 [[张心宇|张心宇]]，分管党委工作部（党委组织部、党委宣传部、统战部、党校）、纪委办公室（监察处、审计处）、学生工作部（党委学生工作部、学生工作处、招生办公室）、党委武装部（保卫处）、离退休人员工作处、工会、团委，联系船舶与海洋工程学院、汽车工程学院。
	
党委委员、副校长 [[杨道远|杨道远]]，分管财务处（会计核算中心）、基建处、后勤集团，联系经济管理学院、旅游与商务学院、艺术学院。
	
党委委员、副校长 余小三，分管教务处、实践教学中心（现代教育信息中心）、继续教育学院（中专部），联系电子与信息工程学院、思想政治理论课部、公共课部。
	
正校级干部 魏太原，协助校领导进行专题调研工作。
