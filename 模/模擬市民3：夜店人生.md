{{Vgname|模擬市民3：夜店人生|en=The Sims 3: Late Night}}是[[電腦遊戲|電腦遊戲]]《[[模擬市民3|模擬市民3]]》推出的第三部[[資料片|資料片]]。《模擬市民3》官方網站於2010年8月宣佈，於2010年10月26日推出。<ref>{{cite web |url=http://pc.ign.com/articles/110/1107301p1.html |title=The Sims 3 Late Night Expansion Announced |last=Reilly |first=Jim |work=[[IGN|IGN]] |date=July 20, 2010 |accessdate=August 9, 2010}}</ref>

本資料片加入了新的公眾地方，包括[[酒吧|酒吧]]、高級[[舞廳|舞廳]]，成為一個[[社區|社區]][[名人|名人]]、[[搖滾樂隊|搖滾樂隊]]的一份子，甚至一位[[吸血鬼|吸血鬼]]。本作亦增加了高層複式豪宅、浴泉及名牌[[房車|房車]]等等，可讓模擬-{市民}-的生活大大豐富，增加遊戲的可玩性。<ref name="simcookie">[http://www.simcookie.com/dossiers-sims/preview-the-sims-3-late-nigth/ SimCookie - Preview : The Sims 3 Late Night] {{webarchive|url=https://web.archive.org/web/20110303222158/http://www.simcookie.com/dossiers-sims/preview-the-sims-3-late-nigth/ |date=2011-03-03 }}</ref>

== 游戏玩法 ==
在“模拟市民3：夜店人生”中，模拟市民可以前往到崭新的城市世界：桥港。游戏中的桥港有着诸多新的地段，与此同时有些地段要求必须为会员、吸血鬼和名人才可访问，但也可以通过[[贿赂|贿赂]]等手段进行访问。在本作中，玩家可建筑或将模拟市民安排至高楼大厦中居住。

本作新添加两种类型的模拟市民：[[吸血鬼|吸血鬼]]与[[名人|名人]]。
=== 吸血鬼 ===
若玩家控制的模拟市民是一名吸血鬼，其市民会得到一些特殊的能力（例如：加快学习技能的速度或看穿其他模拟市民的心理想法，以便正常交往。在[[月圆之夜|月圆之夜]]里，吸血鬼的行动速度将会加快。）但如果想让模拟市民要成为一名吸血鬼，模拟市民必须与吸血鬼达到朋友关系，然后再要求进行授予。在安装“[[模拟市民3：异能新世纪|模拟市民3：异能新世纪]]”资料片后，玩家可以直接从模拟市民编辑器中直接创建一名吸血鬼人物。
=== 名人 ===
若玩家模拟市民是一位名人，那么他们将获得商店的[[折扣|折扣]]活动或完全[[免费|免费]]的消费，同时也可以收到[[粉丝|粉丝]][[邮件|邮件]]或一些[[狂热粉丝|狂热粉丝]]的[[礼物|礼物]]。此外，名人一般可以直接进入高级[[俱乐部|俱乐部]]。
=== 技能 ===
模拟人物也可以学习[[调酒|调酒]]（Mixology）技能，学习此技能可以使模拟市民在酒吧或在家里调配混合[[饮料|饮料]]。

== 桥港 ==
桥港为“[[模拟人生3|模拟人生3]]”系列游戏中的第一个[[城市|城市]]世界地图，其余游戏世界地图大都是[[城镇|城镇]]或[[郊区|郊区]]。
桥港在许多方面类似的市中心，与它的父辈资料片[[模拟人生：火热约会|模拟人生：火热约会]]和[[模拟人生2：美丽夜未眠|模拟人生2：美丽夜未眠]]资料片有诸多相似的地方。但区别在于桥港不是一所城镇，而是一个繁华的城市，其城市中的布局灵感源自于[[美国|美国]][[纽约|纽约]]等其它繁华城市。
=== 灵感 ===
桥港有着众多高层建筑以及大型房屋，并由桥梁将多个岛屿连接而组成。模拟市民可以通过驾驶和步行等其他方式穿过桥梁抵达新的地区。桥梁的灵感源自于[[美国|美国]][[旧金山|旧金山]]、[[洛杉矶|洛杉矶]]以及[[好莱坞|好莱坞]]山顶上的桥梁建筑。<ref name="gamesrader preview3">[http://www.gamesradar.com/exclusive-what-its-like-to-be-a-celebrity-in-the-sims-3-late-night/ GamesRadar - EXCLUSIVE: What it’s like to be a celebrity The Sims 3: Late Night] Retrieved Julu 9, 2016.</ref>
