{{vgname|模擬市民3：島嶼天堂|en=The Sims 3: Island Paradise|cn=模拟人生3：岛屿天堂|tw=模擬市民3：島嶼天堂}}是《[[模擬市民3|模擬市民3]]》的第十款[[資料片|資料片]]，于2013年6月25日推出。<ref>{{cite web |url=http://www.ign.com/articles/2013/02/14/whats-next-for-the-sims-3 |title=What’s Next for The Sims 3? |first=Anthony |last=Gallegos |date=February 13, 2013 |work= |publisher=[[IGN|IGN]] |accessdate=July 6, 2013}}</ref><ref>{{Cite web |url=http://store.steampowered.com/app/223598/?snr=1_7_7_230_150_1 |title=Steam 上的 The Sims 3: Island Paradise |date=2013年6月26日 |publisher=Steam |accessdate=2016-10-18}}</ref>游戏以虚构岛屿生活为主题，首次加入了模拟市民在海水游泳、海上冲浪、开船、潜水等元素，其中还有[[美人鱼|人鱼]]等生物。其中游戏的新城镇“天堂岛”是基于大西洋西部、南北美洲之间的[[加勒比海|加勒比海]]岛屿而设计。<ref>[http://www.beyondsims.com/47780/island-paradise-coverage-preview-6-isla-paradiso-underwater/ Island Paradise Coverage – Preview #6: Isla Paradiso & Scuba Diving]</ref>除此之外，该游戏繼承了[[模擬市民|模擬市民]]二代游戏机版《[[荒岛余生|荒岛余生]]》及《[[模擬市民趴趴走：荒島歷險記|孤岛物语]]》的遊戲特色。
