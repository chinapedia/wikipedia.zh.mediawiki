== 剧情 ==
1951 年[[英國|英國]]數學家[[圖靈|圖靈]]的住所被非法闖入，警察趕到後，圖靈卻拒絕警方的幫助，諾克和史達探員認為事件可疑，著手調查圖靈，甚至懷疑圖靈是[[蘇聯|蘇聯]]派來的間諜，圖靈只好在偵訊時道出自己在[[布萊切利園|布萊切利園]]時的工作。

早在 1927 年，年輕的圖靈就因為沈默寡言，而在[[寄宿學校|寄宿學校]]遭到[[霸凌|霸凌]]，心靈受創。他與另一位天才同學克里斯多夫成為密友，並在其啟發下對[[密碼學|密碼學]]深感興趣。圖靈是個[[同性戀|同性戀]]者，對克里斯多夫漸生情愫，可惜在表明心意前，克里斯多夫突然死於[[肺結核|肺結核]]。

1939 年[[英國|英國]]向[[德國|德國]]宣戰，在丹尼斯頓司令安排下，圖靈赴[[布萊切利園|布萊切利園]]加入解密組，試圖破解當時無人能破的[[德軍|德軍]]密碼機器「Enigma」。圖靈自視高人一等，只顧獨力埋首設計儀器以破解德軍密碼，由於性格孤僻，同事與他合作困難，紛紛投訴圖靈的惡行惡狀。丹尼斯頓司令拒絕提供圖靈製造儀器所需的經費，圖靈竟然直接致函[[唐寧街十號|唐寧街十號]]，還真的獲得了[[首相|首相]][[邱吉爾|邱吉爾]]供給的預算——[[英鎊|英鎊]]十萬鎊的鉅款。圖靈也在首相賞識下成為解密組的新任領導人，甫登大寶，圖靈立刻解僱組內的兩名[[德語|德語]]高手，因為他認為沒有科學頭腦而只會德語的[[語言學|語言學]]專家，對於解密小組是沒有幫助的。

圖靈還在報章刊登招聘廣告，以艱深的[[填字遊戲|填字遊戲]]選拔人才。此時來了一位劍橋畢業的美女瓊恩，成功通過圖靈的考試，圖靈假意要帶瓊恩到自己的「[[無線電|無線電]]公司」上班，事實上卻是要讓她加入解密組，但瓊恩的父母認為「無線電公司」的工作伙伴都是男性，對於單身女子來說，實在太危險，而反對她離鄉。圖靈貼心地安排瓊恩與截聽密碼的女性組員一起住宿、工作，並與她分享自己建造解密機的計劃。圖靈也在瓊恩的幫助與鼓勵下，逐漸打破了他與同事間的隔閡及緊張關係。

圖靈廢寢忘食地努力，終於建成解密機，命名為「克里斯多夫」（紀念自己的昔日密友），但因機器執行緩慢，仍未能破解[[德軍|德軍]]每日改動的加密設定。丹尼斯頓司令見工作未有任何進展，徒耗大量[[經费|經费]]，一怒之下，帶著[[憲兵|憲兵]]衝入了解密組，下令銷毀機器和撤除圖靈的職務，但解密組的成員們此時卻力挺性格古怪的圖靈，並集體以辭職作脅，丹尼斯頓只好答應再寬限一個月時間後離去。

工作了一段時日，瓊恩因父母要求打算離職回鄉，[[相親|相親]]結婚，圖靈為了挽留瓊恩，只好邀其[[訂婚|訂婚]]，瓊恩即刻首肯，並成為圖靈的[[未婚妻|未婚妻]]。在宴會上，圖靈與同事克羅斯聊天，透露了自己的同性戀傾向，克羅斯表示他本有所懷疑，還善意地警告圖靈，萬萬不能向他人透露這個[[性傾向|性傾向]]，因為當時的[[同性性行為|同性性行為]]被視為[[刑事|刑事]]犯罪，必然入獄。

此時圖靈無意聽到一名女截聽員的工作心得，靈光乍現，意會到可以根據已知訊息中的文字來改良「克里斯多夫」，大幅提昇解密執行效率。他重新調校這台機器後，果然在短時間內計算得出解密機當天的設定，也確實破解了一條當天的德軍訊息，解密機證實有效，組員皆喜出望外。但此時也面臨了一個新難題，圖靈明白英國不能對每個訊息都採取反制行動，否則[[納粹德國|納粹德國]]將很快就意識到「Enigma」已遭破解，也必然會立即中斷所有無線電通訊，徹底修改「Enigma」的設計，那麼破解密碼的成果將馬上付之一炬。所以他們只能保密而不去揭露[[納粹|納粹]]的各個攻擊行動，而這也導致了組員彼得的兄長遭到攻擊而喪命。

圖靈與瓊恩尋求軍情六處處長孟席斯的支持，表示對於[[納粹|納粹]]攻擊的反制行動只能選擇性地，以最低程度的方式進行，其他皆需保密。在統計上不能讓德軍產生懷疑，情報來源也必須有其他的說辭與解釋，絕對不能透露是因為「Enigma」已遭破解。

偶然間圖靈發現了克羅斯有本《[[聖經|聖經]]》，竟然就是[[蘇聯|蘇聯]][[間諜|間諜]]使用的密碼本，克羅斯辯稱蘇聯與英國乃是有著共同目標的盟友，又要脅圖靈若泄露半點風聲，便以牙還牙揭發其[[性傾向|性傾向]]。當孟席斯向圖靈質疑瓊恩是蘇聯間諜時，圖靈只好說出克羅斯才是叛徒。豈料孟席斯指出自己早已知悉，為了盡快結束戰爭，克羅斯被特意安插在組織中，以便向[[同盟國|同盟國]]的[[蘇聯|蘇聯]]泄露必要且受控制的情報，他並要脅圖靈協助監視克羅斯，圖靈只能答應。

此時圖靈對孟席斯已有戒心，也擔心瓊恩的安危，因此向她坦承自己是[[同性戀|同性戀]]者，要求解除婚約，也要瓊恩儘快回鄉，甚至訛稱自己從不關心對方。瓊恩表示她
始終有所懷疑，但她覺得雙方仍然可以在一起，這樣的婚姻就算不尋常，其實是適合他們的。瓊恩雖然非常傷心，不過她還是堅持留下不肯離開。戰後，孟席斯要求解密組人員將一切機密文件燒毀，對此計畫完全保密，大家以後須斷絕往來，圖靈與瓊恩也被迫分離。

圖靈的住所被非法闖入證實是他的同性伴侶及其同夥行竊所為，但圖靈的同性戀傾向也因而揭發，被法院判處[[猥褻|猥褻]]罪，他為求能繼續研究工作，選擇接受[[化學閹割|化學閹割]]來代替入獄。瓊恩探望圖靈時，發覺他身心狀態均因為[[荷爾蒙|荷爾蒙]]藥物而大為衰退，瓊恩於是安撫圖靈，稱讚圖靈的成就拯救了英國無數蒼生，又以克里斯多夫向圖靈說過、圖靈亦曾告訴瓊恩的一句話鼓勵他——「有時候，被世人遺棄的人，才能成就讓人想像不到的大事。」

圖靈接受荷爾蒙療法持續一年後自殺身亡，享年 41 歲。

== 制作 ==
影片的剧本在2011年被评为好莱坞最佳未被制作的剧本之一<ref>{{cite web |url= http://www.theguardian.com/film/2011/dec/13/hollywood-black-list-unproduced-films |title=Hollywood's 'Black List' of best unproduced scripts of 2011 revealed  |first=Ben |last=Child |work=The Guardian |date=2011-12-13 |accessdate=2014-09-10| language=en}}</ref>。最初[[华纳兄弟|华纳兄弟]]有意投拍<ref>{{cite web |url= http://deadline.com/2013/02/benedict-cumberbatch-in-talks-to-play-alan-turing-in-the-imitation-game-418902/ |title=Benedict Cumberbatch In Talks To Play Alan Turing In ‘The Imitation Game’ |first=Mike |last=Fleming |work=Deadline |date=2013-02-01 |accessdate=2014-09-10| language=en}}</ref><ref>{{cite web |url= http://www.slashfilm.com/the-imitation-game-begins-filming-with-benedict-cumberbatch-and-keira-knightley/ |title='The Imitation Game' Begins Filming With Benedict Cumberbatch and Keira Knightley |first=Germain |last=Lussier |work=Slash Film |date=2013-09-16 |accessdate=2014-09-10| language=en}}</ref><ref>{{cite web |url= http://deadline.com/2011/10/warner-bros-buys-spec-script-about-math-genius-biopic-because-leonardo-di-caprio-chasing-lead-role-181822/ |title=Warner Bros Buys Spec Script About Math Genius Alan Turing For Leonardo DiCaprio |first=Nikki |last=Finke |work=Deadline |date=2011-10-11 |accessdate=2014-09-10| language=en}}</ref>，后来Black Bear Pictures公司获得剧本拍摄权<ref name=autogenerated1 /><ref>{{cite web |url=http://collider.com/imitation-game-leonardo-dicaprio/ |title=Leonardo DiCaprio Exits THE IMITATION GAME; Warner Bros. Backs Out of the Alan Turing Project |first=Dave |last=Trumbore |work=Collider |date=August 2012 |accessdate=2014-09-10| language=en}}</ref>。2012年12月，挪威导演[[莫腾·泰杜姆|莫腾·泰杜姆]]确认执导，这也成为他导演的首部英语电影<ref>{{cite web |url= http://deadline.com/2012/12/headhunters-helmer-morten-tyldum-to-direct-the-imitation-game-382599/ |title=‘Headhunters’ Helmer Morten Tyldum To Direct ‘The Imitation Game’ |first=Mike |last=Fleming |work=Deadline |date=2012-12-04 |accessdate=2014-09-10| language=en}}</ref>。2013年9月15日在[[英格兰|英格兰]]开拍，拍摄地包括图灵曾经工作生活过的Sherborne School和[[布萊切利園|布萊切利園]]，牛津郡的Nettlebed，白金汉郡的Chesham，Bicester Airfield等地<ref>{{cite web|url=http://oxfordshire.county-homesearch.co.uk/2013/09/18/wartime-spy-movie-to-be-filmed-in-bicester/ |title=Wartime Spy Movie to be Filmed in Bicester |publisher=Oxfordshire |language=en |archiveurl=https://archive.is/20131004130658/http://oxfordshire.county-homesearch.co.uk/2013/09/18/wartime-spy-movie-to-be-filmed-in-bicester/ |archivedate=2013-10-04 |date=2013-09-18 |deadurl=bot: unknown }}</ref><ref>{{cite web |url= http://www.oxfordtimes.co.uk/news/10713158.Sherlock___s_on_the_trail_of_a_new_movie_blockbuster/ |title=Sherlock’s on the trail of a new movie blockbuster |work=The Oxford Times |date=2013-10-03 |accessdate=2014-09-10| language=en}}</ref>。同年11月11日杀青。

[[韦恩斯坦公司|韦恩斯坦公司]]于2014年2月以700万美元的价格取得了影片的美国发行权，这创造了欧洲电影在美国发行的记录<ref>{{cite web |url= http://deadline.com/2014/02/berlin-record-deal-harvey-weinstein-pays-7-million-for-alan-turing-wwii-tale-the-imitation-game-679482/ |title=Harvey Weinstein Pays Record $7 Million For ‘Imitation Game’ Movie |first=Mike |last=Fleming |work=Deadline |date=2014-02-07 |accessdate=2014-09-10| language=en}}</ref>。2014年6月报道法国作曲家[[亚历山大·德斯普拉|亚历山大·德斯普拉]]负责影片的配乐工作。

== 評價 ==
[[爛番茄|爛番茄]]基於242條評論持有90％的新鮮度，平均得分7.7／10<ref>{{cite web |url= http://www.rottentomatoes.com/m/the_imitation_game/ |title=The Imitation Game (2014)|publisher=[[Rotten_Tomatoes|Rotten Tomatoes]] |date= September 2014 |accessdate=2014-09-17}}</ref>。在[[Metacritic|Metacritic]]，根據31篇影评獲得73的高分（滿分100）<ref>{{cite web|url=http://www.metacritic.com/movie/the-imitation-game|title=The Imitation Game |publisher=[[MetaCritic|MetaCritic]]|date=2014-11-28|accessdate=2015-12-13}}</ref>。《模仿遊戲》獲得了良好的口碑，多數評論家稱讚康柏拜區飾演[[艾倫·圖靈|艾倫·圖靈]]的表現相當好<ref>{{cite web|url=http://www.firstshowing.net/2014/telluride-2014-Cumberbatch-is-remarkable-in-the-imitation-game/|title=Telluride 2014: Cumberbatch is Remarkable in 'The Imitation Game'|author=Alex Billington	|publisher=First Showing|date=2014-08-30|accessdate=2015-12-13}}</ref>。

== 電影之外 ==
在[[BBC|BBC]]電視劇《[[新世紀福爾摩斯|新世紀福爾摩斯]]》（由[[班奈狄克·康柏拜區|班奈狄克·康柏拜區]]主演）第二季（2012年）的第一集中，由康柏拜區飾演的角色，福爾摩斯，在劇中跟另一個角色說了二戰時[[考文垂|考文垂]]事件的故事。劇中康柏拜區說道：「二戰時，同盟國解開了德軍當時的加密密碼，因此得知德軍即將轟炸考文垂。但是他們為了不讓德軍知道他們解開了密碼，決定不阻止他們轟炸考文垂。」故事講的就是圖靈等人的事蹟，而後來康柏拜區也正好被找去在此片當中飾演圖靈<ref>{{cite web|url=http://www.imdb.com/title/tt2084970/trivia?item=tr2573872|title=The Imitation Game  (2014)|accessdate=2015-12-13|language=en}}</ref>。
