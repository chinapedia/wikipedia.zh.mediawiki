'''模擬聯合國'''（{{lang-en|'''Model United Nations'''}}，[[缩写|缩写]]{{lang|en|'''MUN'''}}）是一種學術性質活動，藉倣傚精簡後的[[聯合國|聯合國]]議規舉行模擬會議，使與會者瞭解[[多邊外交|多邊外交]]的過程，培養分析[[公民|公民]]議題的能力，促進[[世界|世界]]各地學生的交流，增進[[演講|演講]]和[[辯論|辯論]]能力，提高组织、[[策划|策划]]、[[管理|管理]]、[[研究|研究]]和[[写作|写作]]、解决冲突、求同存异的能力<ref>{{cite web | url=https://www.un.org/zh/cyberschoolbus/modelun/index.shtml | title=欢迎参加模拟联合国！ | accessdate=2014-02-24}}</ref>，訓練[[批判性思考|批判性思考]]、[[團隊精神|團隊精神]]和[[領導|領導]]才能，同時認識不同文化，拓展國際視野<ref name="What MUN2">{{cite web |author=Ryan |url=http://bestdelegate.com/what-is-model-united-nations/ |title=What is Model United Nations? |publisher=BEST DELEGATE |date=November 4, 2007 |accessdate=2013-11-23}}</ref>，瞭解各國在歷史上或現實中的立場與處境<ref name="What MUN"/>。

模拟联合国活动广泛开展于世界各地的[[中学生|中学生]]和[[大学生|大学生]]间，每年全球都會舉辦數十場國際性的模擬聯合國會議<ref name="What MUN" />，参与人数超过四百万人<ref name="What MUN" /><ref name="un" />。

==歷史==
歷史上最早出現的模擬聯合國雛形是在1927年，由[[哈佛大學|哈佛大學]]舉辦的模擬[[國際聯盟|國際聯盟]]<ref name="MUN History" /><ref name="Crisis NYT">{{cite news |last=Parrin |first=Anjli |url=http://www.nytimes.com/2013/08/04/education/edlife/a-new-student-run-breed-crisis-oriented-and-fiercely-competitive.html?_r=0 |title=The Dog-Eat-Dog World of Model U.N. |publisher=[[紐約時報|紐約時報]] |date=2013-08-02 |accessdate=2013-11-23}}</ref>。1946年，國際聯盟被聯合國取代<ref>{{cite web |url=http://www.history.com/this-day-in-history/league-of-nations-instituted |title=League of Nations instituted |publisher=A&E Television Networks, LLC |accessdate=2013-11-23}}</ref>，模擬國際聯盟亦在翌年更改為模擬聯合國。1980年，{{link-en|美國聯合國聯會|United Nations Association of the United States of America}}（UNA-USA）成立，並隨後開始協調世界各地所舉辦的模擬聯合國會議<ref name="MUN History" />

截至2012年，世界各地每年有近400个国际模拟联合国大会在50多个国家召开，每年参与大会的师生来自100多个国家，总人数超过四百万人<ref name="un"/><ref name="What MUN"/>。

在會議當中，每個國家均可以派出一定數量的代表參加會議，並嘗試在會議中將扮演世界各國[[外交官|外交官]]，在搜集資料後依現實或虛構的國際狀況來模擬進行聯合國中的各種會議，按照类似联合国会议的规则和程序进行发言、[[游说|游说]]、辩论、[[谈判|谈判]]，国家集团间达成共识，产生决议草案，并按照联合国的表决程序进行投票从而形成决议<ref name="What MUN">{{cite web |last=Yossinger |first=Nili Sarit |url=http://iipdigital.usembassy.gov/st/english/publication/2012/07/2012073142614.html#axzz2lSH4DnI0 |title=What is Model UN? (And Why Should You Care?) |publisher=U.S. Department of State |date=2012-08-23 |accessdate=2013-11-23}}</ref>。因此，參與者需要在会前的4到9个月里对所代表国家的国情、外交政策和会间将讨论的议题进行深入研究和充分准备，并完成立场文件的写作，同時出席所有正式會議前的工作会议<ref name="un">{{cite web | url=https://www.un.org/zh/cyberschoolbus/modelun/ | title=联合国青少年天地 | accessdate=2014-02-24}}</ref>。

===組織===
會議中模擬的委員會包括聯合國旗下的國際組織<ref name="MUN History">{{cite web |last=Brennan |first=Mary Beth |url=http://students.jccc.edu/orgs/un/valueMUN.htm |title=The Importance of the Model United Nations Experience |publisher=Johnson Country Community College |year=1996 |accessdate=2013-11-23 |deadurl=yes |archiveurl=https://archive.is/20131123103217/http://students.jccc.edu/orgs/un/valueMUN.htm |archivedate=2013-11-23 }}</ref>，通常圍繞[[聯合國大會|聯合國大會]]屬下的六個部門，但並不是所有會議均局限在其之中<ref name="MUN History" />，例如[[聯合國安全理事會|聯合國安全理事會]]、[[聯合國經濟及社會理事會|聯合國經濟及社會理事會]]、[[世界衛生組織|世界衛生組織]]、[[聯合國難民署|聯合國難民署]]、[[歐盟理事會|歐盟理事會]]、[[伊斯蘭會議組織|伊斯蘭會議組織]]和[[亞太經合會|亞太經合會]]等全部都屬於模擬聯合國會議的範疇內<ref>{{cite web |url=http://www.unausa.org/global-classrooms-model-un/model-un-conferences/international-model-un-conference/committees-and-topics |title=Committees and Topics |publisher=United Nations Foundation |accessdate=2013-11-23}}</ref>。

=== 規則 ===
会议具有议事规则，必須遵從[[罗伯特议事规则|罗伯特议事规则]]、欧洲议事规则、<span class="plainlinks">[https://www.un.org/zh/cyberschoolbus/modelun/rules.shtml 模擬聯合國議事規則]</span>或<span class="plainlinks">聯合國大會的官方議事規則</span><ref name="MUN Rules">{{cite web |url=http://www.unausa.org/global-classrooms-model-un/how-to-participate/model-un-preparation/rules-of-procedure |title=Rules of Procedure |publisher=United Nations Foundation |accessdate=2013-11-23}}</ref>，不得飲食或抽烟<ref name="csb">{{cite web | url=https://www.un.org/zh/cyberschoolbus/modelun/code.shtml | title=全球模拟联合国大会行为守则 | accessdate=2014-02-24}}</ref>。在服裝方面，代表必须在会议期间穿着正装，但也可以選擇穿着传统民族服饰或西装，不得戴帽子<ref name="csb"/>。

代表可以在會議當中提出程序性动议、辯論議題、提案、[[修正案|修正案]]和就議案表决等<ref name="un"/><ref>{{cite web | url=https://www.un.org/zh/cyberschoolbus/modelun/rules.shtml | title=全球模拟联合国大会的议事规则 | accessdate=2014-02-24}}</ref><ref>{{cite web |url=http://www.unausa.org/global-classrooms-model-un/how-to-participate/getting-started/model-un-glossary |title=Model UN Glossary |publisher=United Nations Foundation |accessdate=2013-11-23}}</ref>。如果代表在任何時候需要针对其他代表在发言中涉及自己国家的語句或批評作出反驳或评论，大会主席可以在所有代表发言结束后允许代表行使答辩权<ref>{{cite web | url=https://www.un.org/zh/cyberschoolbus/modelun/rule1.shtml | title=会议的掌握 | accessdate=2014-02-24}}</ref>。

=== 议事章程 ===
不同的模拟联合国会议不总会使用相同的议事规程，有时根据会议需要，会议设计者会推出一些特殊的或与现实情况并不完全吻合的规程。议事规程中被最广泛使用的是罗伯特议事规程。根据罗伯特议事规程，会议可以簡單分為以下數個部分。
==== 開始 ====
大会在点名確認三分之二会员国出席时，主席会问场下有无问题或动议，代表需要动议讨论各項议题。如果动议议题超过1个时，代表们通过辩论、投票，确定首先讨论的议题。确定议题后，圍繞議题的正式辩论开始。<ref>{{cite web | url=https://www.un.org/zh/cyberschoolbus/modelun/process.shtml | title=点名、确定议题和产生主发言表 | accessdate=2014-02-24}}</ref>

==== 辯論 ==== 
辯論主要以三種方式進行，分別是正式辩论、有主持核心磋商和自由磋商，而各國在過程中可以透過互相傳遞意向条來私下协商。<ref name="MUN Rules" />

*在正式辩论中（{{lang|en|formal debate}}），各國代表透過舉起國家名牌（{{lang|en|placard}}）來在发言名单上登記，並按议事规则在限時內（一般为两分钟）輪流发言，發言次數不限。如果发言时间剩余时间超过10秒，可有四种选择：一是将时间让渡给主席，相当于放弃时间；二是让渡给其他代表發言（其他代表发言完毕后，若时间还剩余大于10秒，不可继续让渡）；三是让渡给各國代表提问問題，由大会主席决定发言国家；四是让渡给各國代表评论，同樣由大会主席决定发言国家。在每個代表發言後，其他代表可以向主席提出问题或动议，例如詢問大會流程、申請进行非正式辩论、动议结束辩论等，但所有动议均需經過多數同意方可通過。如需追加发言机会，可向主席台传意向条要求在发言名单上添加其代表国家，主席会将该国家名加在发言名单最后。一旦发言名单上所有国家已发言，而又没有任何代表追加发言，会议將直接进入投票表决決議阶段。<ref>{{cite web | url=https://www.un.org/zh/cyberschoolbus/modelun/process1.shtml | title=正式辩论 | accessdate=2014-02-24}}</ref>

*在有主持核心磋商中（{{lang|en|moderated caucus}}），大会由主席主持对某一问题进行固定主题的磋商，代表们在提出此类动议時必须说明主题、总时长以及每位代表发言时长，需要投票經多數同意通過才可以進行。主席将在固定时间内点出举牌的代表輪流发言，剩余时间不能让渡。由於在過程中將暂停所有议事规则，此方式灵活性較强。<ref name="no"/>

*在自由磋商中（{{lang|en|unmoderated caucus}}），大会将暂时中止，代表们可在会场内离开座位自由地相互磋商讨论、游说、撰写文件。代表们在提出此类动议時必须说明总时长，同樣需要多數同意才可以進行。<ref name="no">{{cite web | url=https://www.un.org/zh/cyberschoolbus/modelun/process2.shtml | title=非正式辩论 | accessdate=2014-02-24}}</ref><ref name="MUN Manifesto">{{cite web |last =Shen |first=Sunny |coauthors=Ryan Bae, Amanda Chen, Geneva Nam, Sarah Wang, Marco Wong, Lance Zhou |title=The MUN Manifesto |publisher=Connect Global Youth Association |year=2012 |url=http://project-connect.ca/resources/MUNManifesto.pdf |accessdate=2013-11-25}}</ref>

==== 投票表决 ====
模拟联合国会议中存在兩種投票，分別是对程序的表决和对會議文件的表决。

程序表决包括动议更改发言时间、进行非正式辩论，以及结束辩论。在程序表决中，所有代表都需投票且不能投弃权票，而且在场代表数必須多于三分之二。一般动议需要简单多数（50%+1票）通过，结束辩论的动议需要三分之二多数通过。<ref name="p3"/>

會議文件表决包括修正案和决议草案的投票。在决议草案的表决中，在场代表数也必須多于三分之二，而且任何文件都必需得到有效票中的三分之二赞成票才可以通过<ref name="p3">{{cite web | url=https://www.un.org/zh/cyberschoolbus/modelun/process3.shtml | title=结束辩论与投票表决 | accessdate=2014-02-24}}</ref>。在安全理事會中，[[美國|美國]]、[[俄羅斯|俄羅斯]]、[[中國|中國]]、[[英國|英國]]和[[法國|法國]]任何一個持有[[否決權|否決權]]的[[常任理事國|常任理事國]]一旦投反對票，不管票數如何該文件也會即時被否決。

==== 決議 ====
[[File:模拟联合国流程.png|thumb]]
{{link-en|決議案|Resolution (law)}}是會議上所有辯論的基礎<ref>{{cite web |url=http://web.utk.edu/~modelun/resolutions.htm |title=What is a Resolution? |publisher=University of Tennessee |accessdate=2013-11-25}}</ref>，草案必須先獲主席接納辯論，並在一系列的修正案通過後才能對此作出正式表决<ref name="Resolution">{{cite web |url=http://www.unausa.org/global-classrooms-model-un/how-to-participate/model-un-preparation/resolutions |title=Resolutions |publisher=United Nations Foundation |accessdate=2013-11-25}}</ref>。

当代表们认为决议草案及修正案已讨论成熟，可动议结束辩论进入投票。在投票阶段中，主席將进行点名唱票，按照国家英文字母顺序點名，被点到的代表回答“赞成”、“反对”或“弃权”。一份决议草案表决通过后将不再表决其余决议草案，通过的决议草案将成为大会的决议案进行公告。一旦草案通过决议，议题讨论即宣告结束，隨即进入下一议题的讨论，直至所有議題討論完畢，會議方為完結。<ref>{{cite web | url=https://www.un.org/zh/cyberschoolbus/modelun/process3.shtml | title=结束辩论与投票表决 | accessdate=2014-02-24}}</ref>
