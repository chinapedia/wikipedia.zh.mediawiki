'''黄石镇'''是[[中华人民共和国|中国]][[福建省|福建省]][[莆田市|莆田市]][[荔城区|荔城区]]下辖的一个[[镇|镇]]。

== 行政区划 ==
黄石镇原属莆田县管辖。1984年12月，公社改镇。2002年2月1日[[莆田县|莆田县]]撤销，黄石镇归荔城区管辖。

黄石镇共辖2个居委会、36个村委会，分别是：

黄石居委会、常溪居委会、沙坂村、水南村、登云村、井后村、和平村、横塘村、七境村、瑶台村、凤山村、定庄村、沙堤村、天马村、惠上村、惠下村、屏山村、斗南村、东山村、东洋村、东埭村、金山村、徐厝村、东甲村、遮浪村、海边村、江东村、华东村、华中村、华堤村、桥兜村、下埭村、下江头村、西利村、西洪村、清中村、清后村、清前村。

== 旅游 ==
位于黄石镇江东村的浦口宫是一座纪念[[梅妃|梅妃]]的宫宇，江东村也被认为是梅妃故里<ref> {{cite web |url=http://www.ptqz.cn/zjpt/showart.asp?type=2&id=55 |title=省级文物保护单位浦口宫 |accessdate=2008-10-10 |date=2006-10-12 |publisher=莆田华侨中学 |language=zh-hans }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。2001年1月，浦口宫被福建省人民政府列为第五批省级文物保护单位。
