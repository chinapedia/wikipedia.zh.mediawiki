'''黃興'''（{{bd|1874年|10月25日|1916年|10月31日|catIdx=h}}），[[表字|字]]'''克強'''，[[湖南|湖南]][[長沙|長沙]]人，中國近代民主革命家<ref name="辭海">{{cite book|editor=辭海編輯委員會|title=《辭海》|edition=1989年版|publisher=上海辭書出版社|location=上海|isbn=7532600831|date=1989年9月}}</ref>{{rp|5362}}，[[中華民國|中華民國]]開國元勳，著名的革命家。[[辛亥革命|辛亥革命]]时期，與[[孫中山|孫中山]]常被時人以“'''孫黃'''”並稱<ref name="multiple1">{{cite web |url = http://www.people.com.cn/GB/shizheng/252/6534/6564/20010928/572301.html |title=黄兴|work=|publisher=人民网}}</ref>。1916年10月31日，黃興於[[上海|上海]]去世<ref>{{cite web |url=http://news.ifeng.com/history/special/wusiyundong/juewangdeniandai/200904/0424_6259_1123244.shtml |title=黄兴之死|accessdate=2009-04-24|publisher=凤凰网}}</ref>。

1917年4月15日，黃興受民國元老尊以[[國葬|國葬]]於湖南长沙[[岳麓山|岳麓山]]。著有《黄興集》<ref name="辭海"/>{{rp|5363}}、《黃克強先生全集》、《黄興未刊電稿》及《黃克強先生書翰墨績》刊行。

==名字與稱號==
'''黃興'''原名為'''軫'''，[[表字|字]]'''克強'''，改名'''興'''，一字廑午，號慶午、競武。革命時期[[化名|化名]]'''李有庆'''、'''张愚誠'''、'''张守正'''、'''冈本義一'''、'''今村长藏等'''<ref name="辭海"/>{{rp|5362}}。革命時斷兩指，稱「'''八指將軍'''」。

== 早年經歷 ==
[[清|清]][[同治|同治]]十三年（1874年）9月16日，黃興在[[长沙|长沙]]市郊[[善化县|善化县]]的一个[[地主|地主]]家庭诞生。父亲黃筱村是晚清[[秀才|秀才]]，母亲罗氏出生名门，养育了2男4女。黄兴在6个兄弟姐们中排第六<ref name="multiple2">{{cite web|url=http://book.sina.com.cn/nzt/his/xiandaimingren/15.shtml|title=民国二号人物黄兴|accessdate=|work=吴东平|publisher=新浪网}}</ref>。黃興幼年時思想受湖南的明末大儒[[王夫之|王夫之]]的影响很深<ref name="multiple1"/>。黃興22歲中秀才，在讀書之餘亦喜愛[[武術|武術]]，曾拜拳師李永球，學習巫家拳。

1898年，黃興受時任[[湖廣總督|湖廣總督]][[張之洞|張之洞]]推薦，入[[武昌|武昌]][[两湖书院|两湖书院]]读书，開始同情[[维新运动|维新运动]]、認同变法主张<ref>{{cite web|url=http://hunan.ifeng.com/hunanspecial/hnzhongguomeng/detail_2014_08/13/2759377_0.shtml|title=〈黄兴简介〉|accessdate=2014-08-13|publisher=凤凰网}}</ref>。
1902年，湖南官紳開始選派公費生東渡，並將留日自費生之貧苦者亦悉化為公費<ref name="吳相湘">{{cite book|author=[[吳相湘|吳相湘]]|title=《[[宋教仁|宋教仁]]：中國民主憲政的先驅》|edition=新版|date=1985-09-15|publisher=[[傳記文學_(雜誌)|傳記文學]]出版社|location=[[台北|台北]]|isbn=7532600831}}</ref>{{rp|8}}。黃興赴[[日本|日本]]留学<ref name="辭海"/>{{rp|5362}}，入[[东京弘文学院|东京弘文学院]]师范科学习。黃興、[[楊度|楊度]]等既至[[東京|東京]]，時與日本師友接觸，問學研討之餘，深感救國之道，必先有理論，而後有事實，有[[學術|學術]]而後有[[政治|政治]]；尤須破除過去定於一尊之學風，始能盡除保守濡滯之積習；而介紹東西洋自由之學說，尤為開啟新機運之前提<ref name="吳相湘"/>{{rp|9}}。10月，黃興抵達日本後不久，即和[[杨守仁|杨笃生]]等因之相與集議创刊《游学译编》<ref name="吳相湘"/>{{rp|9}}。黃興並组织“湖南编辑社”，介绍西方科学與文化。

== 中年經歷 ==
1903年，[[俄罗斯帝国|沙俄]]向清朝提出7項要求，企圖侵佔[[中國東北地區|中國東北地區]]，此舉觸及日本，東京《[[朝日新聞|朝日新聞]]》首先刊載，留日學生遂於[[千代田區|神田]]錦輝館召開學生大會<ref name="教育與傳承">{{cite book|author=梁操雅、羅天佑|title=《教育與承傳：歷史文化的視角》|edition=新版|date=2011-01-20|publisher=[[商務印書館_(香港)|香港教育圖書公司]]|location=[[香港|香港]]}}</ref>{{rp|21}}。4月29日，留日學生決定組織義勇隊抗俄以為[[國民|國民]]倡導<ref name="吳相湘"/>{{rp|16}}。據統計，各省留學生到會者500餘人，通過成立拒俄義勇隊，推陸軍士官學校[[藍天蔚|藍天蔚]]為隊長，成員有黃興、[[楊守仁|楊守仁]]、[[陳天華|陳天華]]等，日日操練，意欲隨時開赴前線<ref name="教育與傳承"/>{{rp|21}}。當中，尤以青年會成員最為積極，他們大都參加「拒俄義勇隊」，130人表示願赴戰場者，50人留東京辦事者，有女留學生願擔任看護<ref name="教育與傳承"/>{{rp|21-22}}。留學生正欲改名學生軍，在清朝壓力下，日本政府勒令解散<ref name="教育與傳承"/>{{rp|22}}。5月10日，留日學生惟有在學生軍集會上，議決改名為「军国民教育会」，以便跟進事態發展<ref>{{cite book|author=[[張玉法|張玉法]]|title=《清季的革命團體》|year=1982|publisher=[[中央研究院|中央研究院]]近代史研究所|location=台北}}</ref>{{rp|257-273}}<ref>〈軍國民教育會會約〉，1903年，刊〈紀事〉，《蘇報》1903年第二期</ref>。其精神則不變，且公推代表二人回國，請[[袁世凱|袁世凱]]主戰<ref name="吳相湘"/>{{rp|16}}。北京大學堂及上海教育界均覆電贊成，並紛紛集會，擴大宣傳力量<ref name="吳相湘"/>{{rp|16}}。5月，黃興於弘文学院毕业。

是年6月初，黃興自日本回國，在湖南、湖北醞釀新行動<ref name="吳相湘"/>{{rp|17}}。黃興回到[[上海|上海]]，途经武昌时，黄兴返回母校两湖书院发表演说，武昌府知府兼院长[[梁鼎芬|梁鼎芬]]奉上令，将黄兴驱逐出湖北省境。6月21日，清朝命令嚴厲緝拿倡言革命之學生，意圖以高壓手段鎮壓風潮，然而大勢所趨已不可遏阻，留日學生各種刊物均廢光緒年號，直書[[黃帝紀元|黃帝紀元]]<ref name="吳相湘"/>{{rp|17}}。黄兴於是将随身所携带4千餘冊《[[革命军|革命军]]》和《[[猛回头|猛回头]]》分发给军学各界，然后登轮返回湖南。黃興至[[長沙|長沙]]應聘明德學堂主講席，課餘則秘密進行革命活動<ref>{{cite book|author=劉揆一|title=《黃興傳記》|year=1952|publisher=帕米爾書局|location=台北}}</ref>{{rp|1-2}}。黃興在长沙创办日语学校“东文讲习所”，并翻印[[邹容|邹容]]的《革命军》、[[陈天华|陈天华]]的《猛回头》、《[[警世钟|警世钟]]》等大量革命人士的书籍。

=== 反清革命 ===
{{main|華興會}}
黃興於1903年12月在[[湖南|湖南]]成立之[[華興會|華興會]]以及1904年在[[上海|上海]]創立之[[光復會|光復會]]等組織<ref name=白 >{{cite book |author=[[白吉爾|白吉爾]]（Marie-Claire Bergère）著、溫哈溢譯（第九章協力林添貴，人物小傳協力楊詩韻） |title=《孫逸仙》 |location=台北市 |publisher=[[時報出版|時報出版]] |date=2010-06-21 |isbn=978-957-13-5208-4 }}</ref>{{rp|118}}。1903年11月4日，他以其虚岁三十大寿为由，卖掉自家庄园36亩土地<ref name="multiple4">{{cite web|url=http://ent.sina.com.cn/2004-02-16/1125303813.html|title=《他和孙中山并称为开国二杰 气壮山河·黄兴》|accessdate=2004-02-16|work=|publisher=新浪网}}</ref>，邀集[[刘揆一|刘揆一]]、[[章士钊|章士钊]]、[[宋教仁|宋教仁]]、[[周震鳞|周震鳞]]、[[胡瑛|胡瑛]]、[[張繼_(中華民國)|张继]]等十二人，在长沙坡子街附近的保甲巷彭渊恂宅集会，以兴办矿业为名成立[[华兴会|華興公司]]，口号是：“同心扑满，当面算清”<ref name="multiple4"/>。[[華興會|華興會]]遂於長沙創立<ref name="吳相湘"/>{{rp|18}}。1904年2月15日，华兴会在长沙龙璋的西园寓所正式成立。華興會之創立，以湖南、湖北留日學生及各學堂學生為主體，醞釀已久，《遊學譯編》之所論列，即其宣傳理論及溝通意見，而今可以行動<ref name="吳相湘"/>{{rp|18}}。黃興處事素來沉毅慎重，故此華興會創立之日，黃被公推舉为会长後，即首先提議：望諸同志，對於本省外省各界與有機緣者分途運動，俟有成效，再議發難與應援之策<ref name="吳相湘"/>{{rp|18}}。於是會眾分門別頪，各認職務，積極進行。<ref>錢旡咎編纂：《明德校史》，長沙：明德中學，1947年，刊《中国近代学制史料》第一辑下册，上海：[[華東師範大學|華東師範大學]]出版社，1986年</ref>{{rp|1-2}}以“驱逐鞑虏，复兴中华”为革命口号。科學補習所既成立，適值黃興自上海經武昌回湖南，科學補習所開會歡迎，黃提出華興會預定是年11月16日，乘慈禧太后七十生辰起義計劃，當經一致表示贊同，並約定湖南發難，湖北響應，從此工作進行，益趨積極。<ref>《湖北革命知之錄，第55頁；《武昌革命真史前編自叙》</ref>{{rp|2-3}}自華興會成立，黃即思會眾多屬學界，恐與[[哥老會|哥老會]]黨眾或多隔閡，以[[劉揆一|劉揆一]]曾獲交哥老會大龍頭[[馬福益|馬福益]]，因以聯絡會黨之責，另設一同仇會以支配，使哥老會與華興會總部不相直接，藉防機密偶然洩露，有礙大計。<ref name="吳相湘"/>{{rp|20}}哥老會眾繼加入同仇會者以萬計，黃因仿日本將佐尉各級軍制編組為革命旅，自任[[大將|大將]]兼會長職權，劉揆一任[[中將|中將]]掌理陸軍事，馬福益任[[少將|少將]]主會黨事務；旋復商訂乘慈禧壽辰起義計劃，並擬定長沙省城以武備學堂學生聯絡新舊各軍為主<ref>{{cite book|author=[[馮自由|馮自由]]|title=《革命逸史》第二集|location=上海|publisher=[[商務印書館|商務印書館]]|year=1945}}</ref>{{rp|126}}。随後约半年时间，華興會大力招募会员，並在外地設立分会，同时联络其他同性質的组织，议定于农历十月十日[[慈禧太后|慈禧太后]]70岁生日大宴时在[[长沙起义|长沙起义]]。不久，黄兴变卖长沙的[[黄兴故居|祖屋、田產多處]]筹集革命资金，并在长沙小吴门正街创办东文讲习所作为活动据点。

1904年[[夏天|夏天]]，楊守仁即偕兩同志設機關於[[天津|天津]]，潛居京城數月，無隙可乘，乃失意南歸，得知黃興在湖南起義計劃，楊守仁乃留守上海，與章行嚴等密設愛國協會，以為接濟，並結合留日和省同志，遙為聲援<ref name="吳相湘"/>{{rp|20}}。10月24日，由于起義的风声走漏，官府派兵查封華興會，策划长沙起义未成<ref name="辭海"/>{{rp|5362}}。黄兴住宅被围，幸而黄兴外出，躲于龙璋家中，再秘密隐居[[圣公会|圣公会]]三一堂（今[[基督教北正街教堂|基督教北正街教堂]]）近一月后，从长沙转移到上海，再逃往[[日本|日本]]<ref>{{cite web|url=http://www.gospeltimes.cn/news/33847/%E9%95%BF%E6%B2%99%E5%8C%97%E6%AD%A3%E8%A1%97%E6%95%99%E5%A0%82%E5%A4%8D%E5%A0%8210%E5%91%A8%E5%B9%B4-%E6%9B%BE%E8%8E%B7%E9%BB%84%E5%85%B4%E6%84%9F%E6%81%A9%E9%A2%98%E8%AF%8D|title=长沙北正街教堂复堂10周年-曾获黄兴感恩题词|work=《福音时报》|date=2014-12-30}}</ref>。次年黄兴再次回国策劃起義，失敗後再次折回[[日本|日本]]。

=== 同盟会时期 ===
{{See also|中国同盟会|黄花岗起义|辛亥革命}}
1905年7月，他结识[[孙中山|孙中山]]，在日本拥护孙中山组成[[中国同盟会|中国同盟会]]<ref name="辭海"/>{{rp|5362}}。8月20日，中国同盟会在[[东京|东京]]正式成立。黄兴任执行部庶务（行政庶务組長兼任副會長），居協理地位<ref name="辭海"/>{{rp|5362}}。成为同盟会中仅次于孙中山的领袖，當時號稱孫黃，之后主要发展革命分子、组织武装起义。

1907年夏，到[[越南|越南]][[河內|河内]]，与孙中山一起策划在南方起义。9月，黄兴先后参与或指挥钦廉防城起义；12月发动[[镇南关|镇南关]]（今[[友谊关|友谊关]]）起义<ref name="辭海"/>{{rp|5362}}。

1908年3月，黄兴等人由[[越南|安南]]再次进入[[广西壮族自治区|广西]]，率领200餘人在[[钦州|钦]][[廉州|廉]][[上思|上思]]起义<ref name="辭海"/>{{rp|5362}}，与2万清军周旋月餘，并激战数十回，大破清军。后弹尽粮绝，以四人而御六百敌军，机智从容退回河内。之后[[云南|云南]][[河口|河口]]起义<ref name="辭海"/>{{rp|5362}}，都遭失败。

宣统元年（1909年）秋受孙中山之託，在[[香港|香港]]成立同盟会南方支部，策划次年春广州新军起义<ref name="辭海"/>{{rp|5362}}，再次失败。

1910年11月3日，与孙中山等在[[南洋|南洋]][[檳榔屿|檳榔屿]]（[[马来西亚|马来西亚]]）集会，决定在广州再举行起义，与清军决一死战。

宣统三年三月（1911年4月23日），副總指揮黃興與總指揮[[趙聲|趙聲]]領導第三次廣州起義（[[黃花崗之役|黃花崗之役]]），黃興率領敢死隊進攻督署<ref name="辭海"/>{{rp|5362}}。他首先發難，連發三彈，率敢死队百余人，攻入[[两广总督|两广总督]]轅門，发现总督[[張堅白|張堅白]]已逃跑。此次起义多人牺牲，事后收斂殉难者遗体，有八十餘具，察實身分者七十二具，史称“[[黄花岗七十二烈士|黄花岗七十二烈士]]”。黄兴指挥队伍杀敌，战至最终只剩他一人。右手负伤，断兩指，逃到广州河南女同志[[徐宗汉|徐宗汉]]家，由她护送至香港就医。到了香港，黄兴入雅丽氏医院做手术，按医院的要求，做手术需要家属签名同意，最后徐宗汉以黄兴夫人的名义签字。黃興於辛亥三月二十九廣州一役失敗之後痛心多數青年同道之損失，及海外華僑捐款之虛擲<ref>{{cite book|author=[[馮自由|馮自由]]|title=《革命逸史》第一集|edition=臺一版|publisher=[[臺灣商務印書館|臺灣商務印書館]]|location=台北|page=343|quote=（黃興）欲親至廣州實行暗殺一二滿清大員，以振作全國之民氣。|date=1969年4月}}</ref>。

[[File:黄兴骑马像，辛亥革命博物馆.jpg|250px]]的黃興騎馬雕像]]
1911年10月10日，[[武昌起义|武昌起义]]爆发。10月28日，黄兴由上海到达[[汉口|汉口]]，被推為中华民国军政府战时总司令，指挥民军和清军的战斗。革命军总司令黄兴在汉口、[[汉阳|汉阳]]对清军作战<ref name="辭海"/>{{rp|5362}}。苦战20余日后，民军退至[[武昌|武昌]]；此时，黄兴认为应转攻[[南京市|南京]]，遭当地革命党人反对，乃辞职去上海。黄兴于12月赶往南京指挥战事，并于12月2日一举攻下南京城。

在上海、[[苏州|苏州]]、[[杭州|杭州]]等地相继光复后，黄兴到上海<ref name="辭海"/>{{rp|5362}}。12月4日，[[宋教仁|宋教仁]]、[[陈其美|陈其美]]等却召集留在上海的各省代表在上海江苏省教育总会举行独立各省都督府代表会议，先後推举黄兴为假定[[大元帅|大元帅]]和副元帅代行大元帅职权，均未赴任<ref name="辭海"/>{{rp|5362-5363}}。同时议决公电孙中山归国主持大政。黄兴参与[[南北议和|南北议和]]谈判<ref name="辭海"/>{{rp|5363}}。后改推举[[黎元洪|黎元洪]]为大元帅。

=== 民国时期 ===
{{See also|二次革命|护国战争}}
1912年1月1日，[[中华民国|中华民国]][[南京临时政府|南京临时政府]]成立，黄兴任陆军总长兼任[[参谋总长|参谋总长]]<ref name="辭海"/>{{rp|5363}}，授[[大元帥|大元帥]]軍銜。16日，中華民國自由黨舉黃興為副主裁，黃辭謝<ref>{{cite news|author=中華民國重要史事}}</ref>。2月，黃興在南京辛亥三月二十九廣州革命先烈追悼會演說<ref>{{cite book|author=馮自由|title=《革命逸史》第一集|edition=臺一版|publisher=臺灣商務印書館|location=台北|pages=311-312|quote=黃克強在民元南京黃花岡先烈追悼會演說辭：「今日為黃花岡諸烈士在廣州死義之紀念日。是役也，去年海上各報均有記載，但語焉不詳。近十年來堂堂正正可稱為革命軍者，首推庚尔惠州之役，次為大通之役，此後一二年間寂寂無聞。後二年孫中山先生由美歸，而廣東而日本。乙已年組織同盟會，苦心經營；旋有潮州黃岡之役，萍醴之役，欽廉之役，鎮南關之役，河口之役。河口一役感動軍界，以致復有安慶之役。前年正月有廣東新軍之役。此役敗後，海外各同志更加憤激，即各軍隊中之同志亦非常熱烈，無間於南北。眾多欲利用此時機，尅日起義，可收全功。是時孫中山先生由美至日本，轉而抵南洋，與各同志集議。此時會議者，東南各省多有代表。議會之地點在南洋[[檳榔嶼|檳榔嶼]]。與趙伯先、胡漢民兩先生日日商議。當時所缺者惟餉械二項。幸南洋各志士擔任籌款者極形踴躍，得十餘萬元。乃議決由孫中山先生赴美繼續籌餉，趙君與興（黃興）來內地運動。本擬去年正月即當起義於廣東，後因種種事件均未辦理完善，故遲遲未發。地點議定廣東者，因運有機關鎗四十五支在彼。又廣西軍隊中亦有預備援應。至於內地之佈置，長江一帶譚人鳳先生任之。……」|date=1969年4月}}</ref>。

[[袁世凯|袁世凯]]接任[[臨時大總統|臨時大總統]]。3月30日，袁世凱任黃興為參謀總長<ref>{{cite news|author=全權組織臨時共和政府袁世凱|title=《[[s:臨時公報|臨時公報]]》|location=北京|date=1912-03-31|accessdate=2015-03-19|quote=「兩江一帶軍隊軍隊眾多，事體繁重，參謀總長黃興威望卓著，情形熟悉，特任命統轄布置，現值建設伊始，時艱孔殷，該總長素顧大局，務當力任其難，為國民謀幸福，本大總統有厚望焉。」}}</ref>。临时政府北迁[[北京|北京]]，3月31日，袁世凱任黃興為[[南京|南京]][[留守_(官職)|留守]]<ref name="辭海"/>{{rp|5363}}，仍統轄南洋各軍<ref>{{cite news|author=全權組織臨時共和政府袁世凱|title=《臨時公報》|location=北京|date=1912-04-01|accessdate=2015-03-19}}</ref>，維持整理南方各军及南京地面之責；凡南方陸軍水師要塞，均歸留守處理，隨時通報陸、海軍部及參謀部；關於留守轄範圍內所需款項由留守咨商理財部籌解；南京留守府，俟南方軍隊整理就緒，即行裁撤<ref>{{cite news|title=《南京留守府條例》|date=1912-04-15}}</ref>。袁世凱另任[[徐紹楨|徐紹楨]]為參謀總長<ref name="吳相湘"/>{{rp|206}}。后来因没有经费，军队哗变。6月14日，黃興辭卸南京留守<ref name="吳相湘"/>{{rp|199}}。黃興退居上海。8月13日，孫中山、黃興聯名通電各支部報告同盟会擴大改组为[[國民黨_(1912年－1913年)|國民黨]]<ref name="吳相湘"/>{{rp|208}}。黄兴任理事。8月17日，孫、黃已同登海輪，忽接北京同志來電以[[張振武|張振武]]案發生，暫緩啟行，黃即捨舟登岸並電詰真相，惟孫決心踐約，仍鼓輪北行<ref>{{cite news|title=《[[豎三民|民立報]]》|location=上海|date=1912-08-19|accessdate=2015-03-19|quote=「十八日午後孫中山先生乘安平商輪北上，有以張振武案，多勸勿往者，孫甚決心，謂無論如何不失信於袁總統，且我極以袁總統為可靠。故必欲一試目光。黃克強則偶患喉疾，重將行李運岸，並電京詰問張案。」}}</ref>{{rp|10}}。8月24日，孫行抵北京<ref name="吳相湘"/>{{rp|208}}，英國[[路透社|路透社]]因而大力讚其度量超人萬萬，中國黨爭自能無形消弭<ref>{{cite news|title=《民立報》|location=上海|date=1912-08-25|accessdate=2015-03-19}}</ref>。

1913年3月20日，国民党代理理事长宋教仁即遭預伏奸徒狙擊，腰際中彈，搶救無效，3月22日不治身亡<ref name="吳相湘"/>{{rp|223}}。7月，孙兴师讨袁，发动[[二次革命|二次革命]]。7月14日，黄兴由上海至南京，次日强迫江苏都督[[程德全|程德全]]宣布独立，任江苏讨袁军总司令<ref name="辭海"/>{{rp|5363}}。9月，南京被北洋军攻陷。失败後黄兴流亡日本<ref name="辭海"/>{{rp|5363}}。

1914年7月，孙中山在日本组织[[中华革命党|中华革命党]]，黄兴未参加<ref name="辭海"/>{{rp|5363}}。于7月8日乘船离开日本赴美国（15日到达）考察，宣传反袁思想。

[[File:Huangxinglingtang.jpg|thumb]]
1915年12月15日，袁世凯[[洪憲称帝|称帝]]。黄兴仍积极策动讨袁，与孙中山呼应<ref name="辭海"/>{{rp|5363}}。黄兴派人促进云南[[护国军|护国军]]起义讨伐袁世凯，并在美洲为[[蔡锷|蔡锷]]率领的云南讨袁护国军筹措军饷。1916年6月，因孙中山、蔡锷多次电催，于是他由美国赶往日本，为国内反袁斗争筹款购买军火。当月袁世凯忧愤而死。7月4日，黄兴返回上海<ref name="辭海"/>{{rp|5363}}。

由于长期为革命事业而奔波，积劳成疾，1916年10月10日，黄兴因胃出血入院，10月31日，最终因食道與胃靜脈屈張破裂出血（[[肝門靜脈|肝門靜脈]]高壓所致；根據其死前多次大量吐血以及肝腫大所推測，為[[肝硬化|肝硬化]]常見之併發症）在上海去世，时年僅42岁。孙中山亲自主持治丧活动<ref>{{cite news|title=黄兴：中山先生的“左右膀”|url=http://sztqb.sznews.com/html/2011-03/31/content_1504202.htm|work=《[[深圳特区报|深圳特区报]]》|publisher=深圳报业集团|location=[[深圳|深圳]]|date=2011-03-31|accessdate=2015-03-19|deadurl=yes|archiveurl=https://web.archive.org/web/20150402194935/http://sztqb.sznews.com/html/2011-03/31/content_1504202.htm|archivedate=2015-04-02}}</ref>。

次年4月15日国葬黃興於长沙[[岳麓山|岳麓山]]云麓峰下小月亮坪<ref>{{cite web|url=http://big5.huaxia.com/zhwh/whrw/2013/04/3289990.html|title=黃興蔡鍔魂歸岳麓山 開啟中國近現代國葬之禮|accessdate=2015-03-19|author=任大猛|date=2013-04-12|work=《[[長沙晚報|長沙晚報]]》|publisher=華夏經緯網}}</ref>。

清末的重要革命团体和革命活动，多为孙中山与黄兴所謀劃组织。

1905年，孙中山所建立的兴中会与黄兴所建立的華興會等团体，在日本东京合并成立中国同盟会。在同盟会成立会上，黄兴提议：“公推孙中山先生为本会总理，不必经选举手续”，孙中山被推为总理<ref name="storybase">{{cite web|url=http://www.storybase.cc/11171.html|title=黄兴打下的江山为何老实交给孙中山|accessdate=2015-12-04|work=傅国涌|publisher=詭事基地|deadurl=yes|archiveurl=https://web.archive.org/web/20151208142812/http://www.storybase.cc/11171.html|archivedate=2015-12-08}}</ref>。

1906年，在讨论[[中华民国国旗|中华民国国旗]]时，孙中山主张[[青天白日旗|青天白日旗]]。黃興主张用井字旗，以示“示平均地权意”即[[中国歷史|中国歷史]][[井田制|井田制]]社会意义，认为青天白日旗旗的形式不美，並认为与[[日本國旗|日本國旗]]與軍旗相近，有日本併华之嫌。孙中山嫌井字旗既不美又有复古思想，沒有革命意義。双方争执不下，孙中山坚持己见，黃興力争不得，甚至大怒，誓言要脱离同盟会籍。[[章太炎|章太炎]]、[[刘揆一|刘揆一]]则从中调解，暂时搁置议案。这是辛亥革命以前孙、黄最大的一次分歧，后黄兴“为党与大局，勉强从先生意”，放弃自己主张，孫文也改造了青天白日旗為[[青天白日滿地红旗|青天白日滿地红旗]]<ref>{{cite web|url=http://news.ifeng.com/history/zhongguojindaishi/detail_2012_08/11/16729280_0.shtml|title=黄兴为何反对青天白日旗？觉得是效法日本国旗|publisher=凤凰网|date=2012-08-11}}</ref>。

1907年，[[光复会|光复会]]章太炎、[[陶成章|陶成章]]等人以潮州起义失败为由，要求罢免孙中山总理职务，另举黃興担任，黃興推辞。最终光复会退出中国同盟会<ref>{{cite web|url=http://www.nwp.cn/shouye/n/555.shtml|title=光复会与同盟会的决裂|accessdate=2015-03-19|date=2011-06-28|publisher=新世界数字出版网|deadurl=yes|archiveurl=https://web.archive.org/web/20150403052048/http://www.nwp.cn/shouye/n/555.shtml|archivedate=2015-04-03}}</ref>。

1909年[[秋天|秋天]]，陶成章等起草《孙文罪状》，再次对孙中山发难，要求改选同盟会总理。黄兴极力抵制，在给孙中山的信中，表示“陶等虽悍，弟当以身力拒之”。爾後陶成章被陳其美遣人刺殺於上海[[廣慈醫院|廣慈醫院]]，殺手為蔣介石<ref>{{cite web|url=http://news.zgkqw.com/zhuanti/system/2011/07/12/010168127.shtml|title= 陶成章遇刺之谜|accessdate=2015-03-19|date=2011-07-12|publisher=中国柯桥网}}</ref>。

在辛亥革命以前的几次“倒孙”风波中，黃興素以「成事不必在我」的信念，坚定地拒绝名利的誘惑，其大公無私和忍讓顧全的態度，贏得了多數民國元老乃至於後世人的敬重<ref>{{cite web|url=http://www.chinanews.com/cul/2011/12-19/3540988.shtml|title=黄兴一生戎马屡败屡战 “倒孙”风波中拒任总理|publisher=《湖南日报》|date=2011-12-19}}</ref>。

往後，孫黃在1913年宋教仁案发生后，针对讨袁的方式时再次有分歧，孙中山认为应该武装反抗，而黃興认为应该寻求司法，走和平路线。孙中山发动二次革命时，黃興響應起义，不久失败。孙中山把这次失败原因归结于党员不听话，特别是黃興不听他的領導，要另组秘密的[[中华革命党|中华革命党]]，以服从他为第一要旨。最后黃興並未加入孙中山当年7月所成立的中华革命党，而离开日本前往美国。在[[旧金山|旧金山]]时他接到一些拥护者的来信，希望他另行组党。黃興回曰：“领袖惟有孙中山，其他不知也！” <ref name="multiple6">{{cite web|url=http://news.sina.com.cn/cul/2004-11-11/509.html|title=〈人物风流：黄兴与孙中山生死与共的情谊〉|accessdate=2004-11-11 |work=《长沙晚报》|publisher=新浪网}}</ref>

1916年黄兴逝世后，孙中山闻讯后悲痛欲绝，第二天即发函海内外哀告黄兴逝世消息。过去讣告多由死者亲属发布，而黄兴逝世讣告则由孙中山单独署名发布通令全國全黨，以示創立之義舉。1917年4月15日国葬黄兴于长沙岳麓山云麓峰下小月亮坪。<ref name="multiple3">{{cite web |url=http://news.ifeng.com/history/special/wusiyundong/juewangdeniandai/200904/0424_6259_1123244.shtml|title=《黄兴之死》|accessdate=2009年4月24日|work=凤凰网历史|publisher=凤凰网}}</ref>

== 评价 ==
[[File:湖南省长沙市黄兴墓.jpg|235px]]
一般认为，孙中山是思想家、先行者，黃興是实干家。许多重要的革命活动由黃興所发动并亲自参与。他出生入死多回，最终积劳成疾而中年逝世，是推翻清朝统治、開創民国的元勳。革命时虽然与孙中山有所分歧，但并未夺权而是甘心居次席，鼎力相助之。由于黃興的派系人马在蔣介石北伐后備受忽略，其生前的重大贡献、理念和計劃也长期被擱置。1927年南京政府成立后，“孙黄并称”被“孙陈（其美）并称”所取代。<ref>{{cite web|url=http://www.hn.xinhuanet.com/2012-06/15/c_112224571.htm|title=黄兴被贬内幕：蒋介石编造“正统史观”|publisher=新华网|date=2012-06-15|deadurl=yes|archiveurl=https://web.archive.org/web/20150402101808/http://www.hn.xinhuanet.com/2012-06/15/c_112224571.htm|archivedate=2015-04-02}}</ref>直到1990年代，[[海峡两岸|海峡两岸]]才开始重新重视黃興的功業。 

建立民国后，他初被推举为临时大总统，但并未接受，而是推举即将回国的孙中山<ref>{{cite book|author=[[陈叔通|陈叔通]]|title=《百梅书屋诗存》|location=北京|publisher=[[中华书局|中华书局]]|year=1959}}</ref>。

[[袁世凯|袁世凯]]在亲信面前曾自信地評比孙中山、黃興：“孙氏志气高尚，见解亦超卓，但非实行家，徒居发起人之列而已。黄氏性质直，果于行事，然不免胆小识短，易受小人之欺”。 <ref>{{cite web|url=http://www.nbweekly.com/culture/books/201111/27956.aspx|title=【辛亥逸事】民国了|work=《南都周刊》|publisher=|date=2011-11-03|deadurl=yes|archiveurl=https://web.archive.org/web/20150402114115/http://www.nbweekly.com/culture/books/201111/27956.aspx|archivedate=2015-04-02}}</ref>

黄兴逝世时，在日本疗病的同乡[[蔡锷|蔡锷]]万分悲伤，写下挽联：“以勇健开国，而宁静持身，贯彻实行，是能创作一生者；曾送我海上，忽哭君天涯，惊起挥泪，难为卧病九州人。”不久蔡锷也因病逝世。两人同葬岳麓山。<ref>{{cite web |url=http://www.bjnews.com.cn/ent/2011/04/20/118820.html|title=黄兴 勇健开国 宁静持身|work=《[[新京报|新京报]]》|publisher=新京报社|date=2011-04-20}}</ref>

据说章太炎的外号“章疯子”乃出自於盛怒中的黃興之口。章太炎很敬重黃興，曾愿拥戴黃興为同盟会领袖。在黃興的追悼会上，章太炎送的挽联写道：“无公则无民国，有史必有斯人。”<ref>{{cite news|author=吐鲁番日报记者 阿部都秀库尔·斯马义|title=“无公则无民国，有史必有斯人”|url=http://hnrb.voc.com.cn/hnrb_epaper/html/2014-10/26/content_894406.htm?div=-1|work=《湖南日报》|publisher=湖南日报报业集团|location=长沙|page=2|date=2014-10-26|accessdate=2015-03-19}}</ref>

[[冯自由|冯自由]]评价黄兴说：“世称孙、黄为开国二杰，克强诚当之无愧矣。”<ref name="storybase" />

汪兆銘詩（1938年）:黃花嶽麓兩聯綿，此日相望倍惕然。百戰山河仍破碎，千章林木已風煙。國殤為鬼無新舊，世運因人有轉旋。少壯相從今白髮，可堪攬涕墓門前。

同盟会元老[[胡汉民|胡汉民]]曾评价黄兴说：“黄兴是个标准的‘湖南骡子’。更隐藏‘老子不信邪’的脾气，其雄健不可一世，处世接物则虚衷缜密，转为流辈所弗逮。先生使人，事无大小，辄日慢慢细细。传闻耳熟是语，以为即先生生平治己之格言。”<ref name="multiple5">{{cite web|url=http://www.changsha.cn/gb/content/2002-10/25/content_182766.htm|title=湖南入中的第一奇人——黄兴|accessdate=2002-10-25|work=|publisher=星辰在线|deadurl=yes|archiveurl=https://web.archive.org/web/20131211151702/http://www.changsha.cn/gb/content/2002-10/25/content_182766.htm|archivedate=2013-12-11}}</ref>

黄兴的幕僚[[李书城|李书城]]，评价黄兴：“克强总是个最平实的人，做事有功不居，光明磊落，作战身先士卒，爱护袍泽，做人推诚务实，容忍谦恭，受谤不言诠，受害不怨尤，不道人之短，不说己之长。”<ref name="multiple5"/>

[[章士钊|章士钊]]评价黄兴：“吾持以论交之武器，在‘无争’二字，然持此以御克强，则顿失凭依，手无寸铁。何以言之？我以无争往，而彼之无争尤先于我，大于我……天下最易交之友，莫如黄克强！”<ref name="multiple5"/>

[[宋教仁|宋教仁]]：「優柔寡斷<ref>{{cite web |url = http://news.ifeng.com/history/zhongguojindaishi/special/xinhaigeming100/detail_2011_09/30/9593692_0.shtml |title=宋教仁为何与孙中山过不去？|publisher=凤凰网|date=2011-09-30}}</ref>」

[[楊度|楊度]]：「公谊不妨私，平日政见纷驰，肝胆至今推挚友；一身能敌万，可惜霸才无命，死生自古困英雄。<ref>{{cite web |url=http://www.gmw.cn/content/2005-01/19/content_166391.htm |title=杨度和他的朋友|publisher=光明网|date=2005-01-19}}</ref>」

==家庭==
第一任妻子為廖淡如（1873-1939）。长子[[黄一欧|黄一欧]]，[[中国同盟会|中国同盟会]]会员；次子[[黄一中|黄一中]]，曾任[[国民政府铁道部|国民政府铁道部]][[参事|参事]]、[[中華民國內政部|内务部]]统计司司长；五子黄一寰，后改名[[黄乃|黄乃]]，为遗腹子，日本问题专家，拼音盲文体系创建人，被誉为“中国盲文之父”；长女[[黄振华|黄振华]]，[[立法委员|立法委员]]，其夫[[陈维纶|陈维纶]]為[[台湾师范大学|台湾师范大学]][[师范学校|教育学院]]院长；三女黄德华，其夫[[薛君度|薛君度]]為[[历史学家|历史学家]]，[[斯坦福大学|斯坦福大学]]政治系教授，以辛亥革命研究见长。

次女[[黄文华|黄文华]]，来历不详，生于日本，[[马来亚共产党|马来亚共产党]]创建人之一，其夫為[[黄文山|黄文山]]。

第二任妻子（重婚）為[[徐宗汉|徐宗汉]]（1877-1944）。三子黄一美，其妻張瑛為[[張繼_(中華民國)|张继]]之長女；四子黄一球，[[中华航空公司|中华航空公司]]机械总工程师，其妻程博德為[[程潜|程潜]]之女。<ref>{{cite web |url = http://www.hercity.com/s/201110/18139.html |author=吴东平|title=辛亥人物黄兴后人的传奇命运|work=《走近现代名人的后代》|publisher=倾城网|date=2011-10-19}}</ref>

== 纪念 ==
[[File:PeopleHuangXing.jpg|thumb]]的黄兴雕像]]
1911年11月3日，[[湖北军政府|湖北军政府]]在武昌阅马场用木料搭台举行拜将仪式，[[黎元洪|黎元洪]]向黄兴授战时总司令印、旗和剑。今天的拜将台遗址被公布为湖北省文物保护单位。<ref>{{cite web|url=http://roll.sohu.com/20110916/n319576457.shtml|title=寻访黄兴拜将台遗址|publisher=[[新华网|新华网]]|date=2011-09-16}}</ref>

1917年4月15日，在蔡锷国葬岳麓山之后3天，黄兴也国葬在岳麓山，并修有[[黄兴墓|黄兴墓庐]]。1918年10月，在日本[[横滨市|横滨市]][[鹤见区_(橫濱市)|鶴見區]]的日本著名寺院“[http://www.sojiji.jp/ 曹洞宗大本山总持寺]”内的“手枕坂”边上的山岗上，立有一相当高大的 “黄君克强之碑”。具体当时的立碑事宜不明。碑文由前[[日本首相|日本首相]][[犬养毅|犬养毅]]手书。虽经88年岁月碑体碑文相当完好。<ref>{{cite web|url=http://www.tokyo-kurenaidan.com/sun-yat-sen-yokohama1.htm|title=孫文（孫中山）を歩く　横浜編（鶴見 總持寺） ＜黄君克強之碑＞|work=孫文を歩く|publisher=東京紅團|date=2014-04-05}}</ref>

1933年，在武汉市武昌[[蛇山|蛇山]]设立黄兴铜像。1955年因建设[[武汉长江大桥|武汉长江大桥]]，将铜像搬至黄鹤楼剧场东侧。1985年10月迁至[[龟山_(武漢)|龟山]]东麓。<ref>{{cite web|url=http://www.docin.com/p-764657658.html|author=郑自来|title=黄兴铜像迁至汉阳龟山始末|work=武汉文物博物馆协会|publisher=豆丁网|year=2011}}</ref>

1934年，长沙市把最繁华的街道南正街改名[[黄兴路_(长沙)|黄兴路]]。在武汉、上海等城市也有黄兴路。在臺灣[[臺北市|臺北市]][[士林區|士林區]]和[[高雄市|高雄市]]三民區也有克強路、黃興路之路名。1981年，黄兴故居被列为湖南省省重点文物保护单位，加以修葺并对外开放。1986年10月10日[[中华人民共和国邮电部|中国邮电部]]发行了志号为J.132的《辛亥革命著名领导人物》纪念邮票1套3枚。其中第二枚为黄兴像。另两枚为孙中山和章太炎像。1988年1月13日，黄兴故居和[[黄兴墓|黄兴墓]]一起被列为[[全国重点文物保护单位|全国重点文物保护单位]]。<ref>{{cite web |url=http://www.gov.cn/guoqing/2014-07/21/content_2721163.htm|title=国务院关于公布第三批全国重点文物保护单位的通知|publisher=中国政府网|date=2014-07-21}}</ref>

2004年，黄兴诞辰130周年，湖南省各界人士以及黄兴等革命爱国人士的亲属代表出席矗立在长沙黄兴南路步行商业街北街口的黄兴铜像揭幕仪式。铜像高7.8米，表现黄兴一次回到故乡长沙的情景。基座4个立面浮雕，分别刻画他组织华兴会、创办中国同盟会、指挥武昌保卫战、与孙中山携手合作等重大历史事件。同时在上海黄兴公园，上海市领导也举行黄兴铜像揭幕仪式。上海黄兴铜像高2.8米，人物双手背后握剑，显示黄兴这位辛亥革命卓越领导人的军人形象。<ref>{{cite web |url=http://www.people.com.cn/GB/shizheng/14562/2939421.html|title=纪念黄兴诞生130周年 铜像揭幕仪式在长沙举行|publisher=[[人民网|人民网]]|date=2004-10-24}}</ref>[[长沙县|长沙县]][[星沙镇|星沙镇]]等地也立有黄兴铜像。

2011年9月，在[[辛亥革命|辛亥革命]]一百周年之际，湖南省委决定修建湖南辛亥革命人物纪念馆。纪念馆新建项目初步选址于黄兴故居纪念馆后花园。<ref>{{cite web |url=http://www.xzqh.org/html/show/hn/15911.html|title=1997年长沙县行政区划|publisher=行政区划网|date=2010-11-29}}</ref>黄兴故居所在地高塘乡被更名为黄兴镇。
