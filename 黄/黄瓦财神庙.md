'''黄瓦财神庙'''，位于[[北京市|北京市]][[东城区_(北京市)|东城区]][[鼓楼东大街|鼓楼东大街]]，是[[北京市东城区普查登记文物|北京市东城区普查登记文物]]。

==简介==
该庙本名“增福财神庙”，建于[[明朝|明朝]]晚期，庙顶原为灰筒瓦顶。[[清朝|清朝]][[雍正|雍正]]年间重修财神庙时，将灰筒瓦顶改成黄琉璃瓦顶，故俗称“黄瓦财神庙”<ref name=xz>{{Cite web |url =  http://news.sina.com.cn/s/2006-04-26/02118789185s.shtml|title = 财神庙洗澡“闪金光” |author =  |date = 2006-04-26 |publisher = 新浪 |language =  |accessdate =  |quote =  }}</ref>。

北京民间传说，[[雍正帝|雍正帝]]当雍亲王时住在雍亲王府（今[[雍和宫|雍和宫]]），到[[地安门|地安门]]进[[紫禁城|紫禁城]]路过此处，曾进该财神庙祈求保佑自己登基。雍正帝登基后，乃下令为财神庙换上皇家特许使用的黄琉璃瓦<ref>{{Cite web |url =  http://www.cnepaper.com/jyrbszb/html/2017-02/18/content_4_1.htm|title = 财神庙与匾额书法 |author =曹鹏  |date =2017-02-18  |publisher = 江阴日报 |language =  |accessdate =  |quote =  }}</ref>。

该庙主祀的“增福财神”，应是[[李诡祖|李诡祖]]。1920年出版的《三续淄川县志》载:“[[北魏|北魏]]李诡祖，[[北魏孝文帝|孝文帝]]时任曲梁令。当南北纷争，民苦兵戈，独能抚楫流亡，敦行教化，与民休息，卒于官。民怀其德，立庙尸祝之，至今享祀不衰，明晋祀名宦祠。”<ref>{{Cite web |url =  http://news.sina.com.cn/o/2013-02-08/104826239992.shtml|title = 探寻“文财神”李诡祖 |author =  |date =  2013-02-08|publisher = 新浪 |language =  |accessdate =  |quote =  }}</ref>

[[改革开放|改革开放]]后，黄瓦财神庙曾被空调修理所租用。2005年底，在鼓楼东大街风貌保护与环境整治工程中，拆除了黄瓦财神庙周围的临时建筑，文物部门修复了黄瓦财神庙<ref name=xz/><ref>{{Cite web |url = http://news.qq.com/a/20051106/000100.htm |title =新鼓楼东大街亮相  |author =  |date = 2005-11-06 |publisher =  腾讯|language =  |accessdate =  |quote =  }}</ref><ref name=xisu>{{Cite web |url = http://sh.people.com.cn/n/2014/0117/c359431-20417857.html |title =  腊月到正月·过年习俗知多少之正月初二|author =  |date = 2014-01-17 |publisher = 人民网 |language =  |accessdate =  |quote =  }}</ref>。

2012年，“黄瓦财神庙建筑遗存”被列为[[北京市东城区普查登记文物|北京市东城区普查登记文物]]。
