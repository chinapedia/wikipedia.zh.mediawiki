==生平==
2008年，黄雪平出任中华人民共和国国防部新闻事务局副局长、国防部新闻发言人。他和局长[[胡昌明|胡昌明]]是最早的国防部新闻发言人<ref name=bayi>{{Cite web |url = http://www.thepaper.cn/newsDetail_forward_1515194 |title = 中国驻联合国军事参谋团团长黄雪平“八一”前夕晋升少将军衔 |author =  |date =2016-08-17 |publisher = 澎湃新闻 |language =  |accessdate =  |quote =  }}</ref>。2010年4月，黄雪平卸任，调任国防部外事办公室美洲大洋洲局局长<ref name=bayi/><ref>{{Cite web |url = http://news.cntv.cn/china/20110515/102110.shtml |title =国防部：陈炳德总长访美行程涵多个军事设施  |author =  |date = 2011-05-15 |publisher = 中国网络电视台 |language =  |accessdate =  |quote =  }}</ref>。2016年6月之前，黄雪平接替[[周明|周明]]少将担任[[中国常驻联合国代表团|中国常驻联合国代表团]]军事参谋团团长。中国常驻联合国代表团机构设置共12个部门，其中军事参谋团的职责是“参与联合国军事参谋团会议和活动，出席联合国维和出兵国会议及活动”。2016年[[八一建军节|八一建军节]]前，黄雪平晋升少将军衔<ref name=bayi/><ref>{{Cite web|url =http://military.china.com/news/568/20160616/22879998.html|title =国防部首任新闻发言人任驻联合国军事参谋团团长|author =|date =2016-06-16|publisher =中华网|language =|accessdate =|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20170921064142/http://military.china.com/news/568/20160616/22879998.html|archivedate =2017-09-21}}</ref>。
