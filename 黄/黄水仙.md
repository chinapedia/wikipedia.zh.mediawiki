'''黄水仙'''（[[学名|学名]]：{{lang|la|''Narcissus pseudonarcissus''}}），又名'''洋水仙'''<ref>华北经济植物志要</ref>、'''喇叭水仙'''<ref>上海植物名录</ref>，是[[石蒜科|石蒜科]][[水仙属|水仙属]]的植物。分布在[[欧洲|欧洲]]等地，目前已由人工引种栽培。多年生草本，鳞茎最大者长0.45米，宽0.1米，花瓣淡黄色，中心副冠是深黄色的[[喇叭|喇叭]]形，边缘不规则褶皱，叶绿色，略带灰色，基生，宽线形，先端钝，顶生1花，花期3－4月。

== 分布 ==
原产於[[欧洲|欧洲]]西部，分布於[[西班牙|西班牙]]、[[葡萄牙|葡萄牙]]、[[德国|德国]]、[[英格兰|英格兰]]和[[威尔士|威尔士]]等地。在欧洲的很多花园中都有生长，野外生长於森林、草地和岩石地面上。自19世纪起，英国的黄水仙大量减少，这是由於农业耕地增加、森林砍伐以及园艺用[[鳞茎|鳞茎]]的大量挖掘。

== 亚种 ==
黄水仙有很多[[亚种|亚种]]，但是根据不同的学者观点亚种的数目不同，而其大量的[[栽培种|栽培种]]对亚种的分类造成了困难。在这些亚种中，喇叭水仙（''N. p.'' ssp. ''obvallaris''，有时被认作一个独立种）最初可能是源於栽培种，但现在已野外生长於威尔士西南部。

== 象征 ==
黄水仙是[[威尔士|威尔士]]的[[国花|国花]]，也是[[英国|英国]][[格洛斯特郡|格洛斯特郡]]的郡花。
