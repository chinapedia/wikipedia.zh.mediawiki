{{道教}}
'''黄庭經'''指《'''太上黄庭内景玉經'''》和《'''太上黄庭外景玉經'''》，約出於魏晉之際，收入《[[正統道藏|正統道藏]]》洞玄部本文類，大旨在於服氣以養精神。主張固守精神，則身形可以長存<ref name=hua>{{cite journal|author1=龔鵬程|title=「黃庭經」論要|journal=中國書目季刊|date=1997|url=http://www.ctcwri.idv.tw/INDEXA3/A302/A3005/A3-05001.htm}}</ref>。《内景經》乃[[道教|道教]][[上清派|上清]]一脈修習要典，據傳為上清仙真降授於南嶽[[魏華存|魏華存]]，後來成為宋元[[内丹|内丹]]家修炼内丹的重要典籍。有學者認為《外景經》則與早期[[天師道|天師道]]關係密切，較內景早出，而為內景經的源流<ref>{{cite book|author1=張超然|title=系譜、教法及其整合：東晉南朝道教上清經派的基礎研究|date=2007|pages=201-202|url=http://nccur.lib.nccu.edu.tw/bitstream/140.119/35567/10/51502110.pdf}}</ref><ref name=hua1>{{cite journal|author1=虞萬里|title=黃庭經用韻時代新考|journal=聲韻論叢|date=2001|volume=10|pages=209-241|url=http://www.historyshanghai.com/admin/WebEdit/UploadFile/EpochWL.pdf|deadurl=yes|archiveurl=https://web.archive.org/web/20160309195539/http://www.historyshanghai.com/admin/WebEdit/UploadFile/EpochWL.pdf|archivedate=2016年3月9日|df=}}</ref>，也有相反意見主張內景經較為早出<ref>[[李涵虛|李涵虛]]《太上十三經注解》：黃庭皆太上化身所作。《女仙傳》云：「錄圖作《黃庭經》五十卷，先傳顓頊，繼授藐姑，約五十卷為一卷，名曰《外景》。東華傳魏夫人，夫人並傳於後」，故云皆化身作也。或問：「《外景經》既因《內景》約編，如何《外景》一卷又稱堯時務成子著？《內景》一篇反在晉時始出？」紫霞曰：「《內景》在顓頊後傳藐姑，《外景》繼此而作。當時兩經並隱，至漢而《外景》始大顯，至晉而《內景》始大顯，亦因緣之適然耳。不得以《外經》早出，疑其為先《內景》而作也」。......或謂：「玉宸乃靈寶天尊號。《內景經》不書老君，而書玉宸君何也？」曰：「老君著《內景》，化身玉宸君，其居無垢天宮，玉即無塵之義，宸乃帝居通稱......」。</ref>。

黃庭外、内二經，《外景經》一百九十六句，分為上中下三經或二十四章，《内景經》四百三十七句，分爲三十六章，均是規整的七言韻語，各版本異文甚多，句之多少、有無及次序先後，各有不同<ref name=hua1/>。

是書以韻語描述人體五官、五臟、六腑以及全身八景諸神二十四真形象與作用，旨在存神內觀，固精練氣，填滿黃庭，久而行之，則五臟生華、色返孩童，最後臻至長生久視之境。

==題解==
黃庭一詞在[[東漢|東漢]]後期已出見，如[[邊韶|邊韶]]《老子銘》：「出入丹廬，上下黃庭」。《[[靈寶五符經|靈寶五符序]]》：「奉請降中央含樞紐黃帝君、太帝真人子丹、黃庭戊己、天倉玉女，中央諸靈官」。《老子中經》：「中央黃氣，黃庭高仙，服食黃氣，飲之醴泉」。《[[抱朴子|抱朴子]]·至理》：「堅玉鑰於命門，結北極於黃庭」。

據[[白履忠|梁丘子]]《黃庭內景玉經註》的解釋，黃庭之義喻指中央，「黃者，中央之色也；庭者，四方之中也」。但黃庭所在之身中部位，眾說紛紜。早期的注解者，認為黃庭是「身中神」所居處之一<ref>《夢溪筆談·象數一》：「又曰：黃庭有神人守之。」</ref>，而在：

# 頭兩眉之中（卻入一寸至三寸）。<ref>[[白履忠|梁丘子]]《黃庭外景玉經註》：「黃庭者在'''頭中'''明堂、洞房、丹田，此三處是也。兩眉卻入一寸為明堂，二寸為洞房，三寸為丹田。黃庭者，兩半（陰陽二氣）於洞房中共生赤子，則為真人也，常思之，慎無失，赤子化為真人，在明堂中，字子丹，故知一者在明堂一處也。」</ref>
# [[眼睛|眼睛]]。<ref>務成子注《黃庭外景經·上部經》「黃庭者，'''目'''也。道之父母供養赤子，左為陵陽，字英明；右為太陰，字玄光。」</ref>
# [[脾_(臟腑)|脾]]。<ref>[[白履忠|梁丘子]]《黃庭外景玉經註》說：「一解-{云}-，黃庭者，'''脾'''也，長尺餘，在太倉之上，臍上三寸。脾為中主黃老君，中央神也，治在其中矣。」<br/>務成子注《黃庭內景經·脾長章》：「脾，黃庭之宮也，黃老君之所治。上應明堂，眉間入一寸是也。」<br/>《黃庭內景經·務成子注敘》：「脾為黃庭命門。明堂中部，老君居之，所以-{云}-黃庭內人服錦衣也。」<br/>《[[老子中經|老子中經]]》：「中極黃老者，真人之府中斗君也。......人亦有之，黃庭真人，道之父母也，赤子之所生也，己吾身也。......正在'''脾上'''，中斗中也。......祝曰：黃裳子，黃裳子，黃庭真人在於已。為我致藥酒，松脯，粳糧，黍臛諸可食飲者，令立至！......次念'''脾上'''正黃，中有日，日中有三人，道母玄光玉女在中央，左有黃裳子，右有中黃直......經曰：常以六戊之日雞鳴時、日西黃昏時，拊心祝曰：天道天道，願得不老，壽比中黃，升天常早，願延某命，與道長久！因瞑目念'''脾中黃氣'''來上至口中，上念天精黃氣來在目前，入口中咽之，三五而止。思行黃氣週遍一身中，百五十息止。呼其神曰：玄光玉女、養子赤子、真人子丹、服食元氣，飲宴醴泉！故言一身神有父母凡三人。至己日復存之，呼其神曰：真人黃庭玉女，與己為友，留為己使。王甲欲求'''真人黃庭神仙之道'''。」</ref>（《[[老子中經|老子中經]]》主此說）
# 下[[丹田|丹田]]（臍下三寸）。<ref>《金闕帝君三元真一經》：「'''臍下三寸'''號命門丹田宮，'''下元嬰兒'''居其中。......下元嬰兒諱始明精，字元陽昌，位為'''黃庭元王'''」<br/>《黃庭內景經·務成子注敘》：「自臍後三寸，皆號黃庭命門，故下一-{云}-命門中有黃庭元王、玄闕大君。又-{云}-坐當命門。猶如頭中亦呼為泥丸洞房中也。」</ref>（《[[金闕帝君三元真一經|金闕帝君三元真一經]]》主此說）
# 腦中、[[心臟|心]]中、脾中。<ref>[[白履忠|梁丘子]]《黃庭內景玉經註》「黃庭內指事，即'''腦中、心中、脾中'''」<br/>[[白履忠|梁丘子]]《黃庭內景玉經註》：「上元老君居'''上黄庭宫'''，與泥丸君、倉華君、青城君，及明堂中君臣、洞房中父母、及天庭眞人等共爲朋也。又中元老君居'''中黄庭宫'''，與赤城童子、丹田君、皓華君、含明君、英玄君、丹元眞人等共爲朋也。又下元老君居'''下黄庭宫'''，與太乙君、魂停君、靈元君、太倉君、丹田眞人等共爲朋也。」</ref>（通常採此解釋，謂為上中下三黃庭）

後世的[[内丹术|內丹]]家，則說黃庭在：

# 五臟之正中（心之下去三寸六分，臍腎之上三寸六分）。<ref>《金丹大成集》：「黃庭正在何處？答日：在膀胱之上，脾之下，腎之前，肝之左，肺之右。」<br/>《鐘呂傳道集·論鉛汞》：「黃庭者，脾胃之下，膀胱之上，心之北而腎之南，肝之西而肺之東。」<br/>《至遊子·百問篇》：「何謂黃庭?曰：在膀胱之上，脾腎之前，臍之後，肝之右，肺之左，如雞子，如權石。」<br/>《養生祕錄》：「中言其位，黃言其色，故謂中黃。八極者，是八方總會要處，又只是中宮，即黃庭，即玄牝，即先天一氣，即'''玄關一竅'''，即至善之所，即黃極之道，即兌執厥中。......天地相去八萬四千里，人之心腎即一身之天地，相去八寸四分，以中指節文為則，自臍上至鳩尾骨尖，只有八寸四分。今-{云}-臍者，蓋與腎對也，故'''心之下去三寸六分，臍腎之上三寸六分，惟中間一寸二分為黃庭'''，主我身命。所謂至聖之道。祕之！祕之！......問黃庭正在何處？答曰：在膀胱之上，脾之下，腎之前，肝之左，肺之右也。」<br/>《[[丘祖秘傳大丹直指|丘祖秘傳大丹直指]]》（民國本）：「'''心下三寸六分'''，名曰土釜，黃庭宮也，乃'''中丹田'''，方圓一寸二分，亦'''虛間一穴'''，乃藏氣之所，煉丹之鼎」<br/>《夢溪筆談·象數一》：「或者又謂黃庭在二腎之間。又曰在'''心之下'''。又曰：黃庭有神人守之。」<br/>[[劉一明|劉一明]]《黃庭經解》：「'''心下腎上黃庭之處'''。」<br/>《黃庭外景經·石和陽注》：「'''命門之上'''，有玄關二竅，左玄右牝，'''中虛一處'''，名曰黃庭。」<br/>《中華道教大辭典》解釋此句為：「'''一身內臟之正中'''，約在臍上三寸通向後背之連線中點，胃下空隙處。」</ref>（《[[鐘呂傳道集|鐘呂傳道集]]》等丹書主此說）
# [[臍|臍]]內空處或臍中一寸三分。<ref>[[陳攖寧|陳攖寧]]《黃庭經講義》則說：「'''臍內空處'''，即黃庭也」<br/>[[王重陽|王重陽]]《重陽真人金關玉鎖訣》：「又咽神水到脾，脾為土，土得水者能生黃芽。腹為大小腸九曲，至'''臍中一寸三分'''，方圓一寸，左青右白，前赤後黑，'''中黃戊己，名為丹田'''。田內一座宮，宮中名曰黃庭宮。」<br/>《[[大丹直指|大丹直指]]》（《正統道藏》本）：但'''臍在人身之中'''，名曰中宮命府、混沌神室、黃庭丹田、神氣穴、歸根竅、覆命關、鴻蒙竅、百會穴、生門、太乙神爐、本來面目，異名甚多。此處包藏精髓，貫通百脈，滋養一身，淨躶躶，赤灑灑，無可把蓋。」<br/>[[劉一明|劉一明]]《黃庭經解》：「腎前臍後之中。」</ref>（[[陳攖寧|陳攖寧]]、《重陽真人金關玉鎖訣》主此說）
# [[三焦|三焦]]。<ref>[[俞琰|俞琰]]《席上腐談》：-{「蘇子由云：古說左腎，其腑膀胱；右命門，其腑三焦，丈夫以藏精，女子以系胞。以理推之，三焦皆如膀胱，有形質可見...愚謂聞之隱者云：'''三焦即黃庭'''也。《丹書》以心火、腎火、膀胱火聚於此，以猛烹極煆，故曰「三焦」，焦乃武火之謂也。」}-</ref>
# 脾胃夾中處。<ref>[[清朝|清朝]]唐道宗（三復子）《聖賢實學》：-{「黃庭在何處？在脾胃夾中處也。脾屬土，因土之色黃，故曰『黃』；此穴是神明之舍，虛靈之府，元神之家庭，故曰『庭』。此穴乃脾胃夾中虛懸一穴，故儒曰腔子，又曰衷。因脾為土，故儒曰心地。此穴約廣寸餘，故儒曰方寸，又曰寸衷。此穴其形似田，故儒曰心田，道曰中丹田。此穴上通心，下通腎，此穴居中，故儒曰中，又曰中心。此黃庭諸般異名之義也。...後天父母五行之氣，散於周身，而統於黃庭；先天五行之氣，聚於黃庭，至虛至靈，主宰一身。黃庭一穴，一寸三分寬，在脾胃夾中處，離外肉皮三寸三分。黃庭為心田，心居黃庭。未發為中，發而中節為和，此中和之根源也。」}-</ref>

或認為黃庭並不實指某個身體部位，而是：

# 有名無所，萬有皆空之境，至虛至無，不可見、不可聞，沖氣之所在。<ref>[[沈括|沈括]]《夢溪筆談·象數一》：「黃庭有名而無所，'''沖氣之所在'''也」<br/>[[劉一明|劉一明]]《黃庭經解》：「非腎前臍後之中，非心下腎上之中，亦非頭頂天谷之中，乃在'''四大不著之處，萬有皆空之境'''...黃庭之神不在心下腎上黃庭之處，不在泥丸黃庭之處，不在脾部黃庭之處。經中雖言三黃庭，皆言黃庭之景，非言黃庭之真也。黃庭之真，乃一大包羅。因其為一大包羅，其神無處不在，無時不有」<br/>[[李涵虛|李涵虛]]《太上十三經註解》：「黃庭者，靜養-{谷}-神。靜則言定，'''非真有黃庭'''也。」<br/>[[李涵虚|李涵虚]]《圓嶠內篇》：「玄關一竅，自虛無中生。不居於五臟六腑，肢體間無論也。今以其名而言：此關為玄妙機關，故曰玄關。此竅為萬法歸一之地，有獨無對，故曰一竅。一言以備之曰：中是也。中在上下之中，亦不在上下之中，有死、有活故也。何謂死？以黃庭、[[足少陰腎經|炁穴]]、[[丹田|丹田]]為此中，就是死的。何謂活？'''以凝神聚炁，現出此中'''，就是活的。以死的論，就叫做黃庭、炁穴、丹田。以活的論，乃算做玄關一竅。故曰：自虛無中生。真機直露。得者秘之。」</ref>（[[李涵虚|李涵虚]]、全真道士[[劉一明|劉一明]]主此說）

[[龔鵬程|龔鵬程]]認為黃庭養生之術源自[[中國傳統醫學|中國傳統醫學]]，主張「黃庭」應當是指[[任脈|任脈]]中庭穴<ref name=hua/>。

內景，指心觀一體之象，內視身中之神。[[白履忠|梁丘子]]《黃庭內景玉經註》：「內者心也，景者象也。外象喻即日月星辰雲霞之象也。內象喻即血肉筋骨臟腑之象也。心居身內，存觀一體之象色，故曰內景」。《內景經·務成子注敘》說：「景者，神也。其經有十三神，皆身中之內景名字」。與內景相對為外景，《黃庭外景經》原名《黃庭經》，唐人爲區別於《黃庭內景經》而題作「外景」<ref>{{cite book|editor=周谷城|title=中國學術名著提要宗教卷|date=1997|pages=743-744}}</ref>。

==內容==
[[File:Wang_Xizhi_Huang_Ting_Jing.jpg|thumb]]
全書以道教思神[[守一|守一]]，寶精愛氣之說與古代醫家[[臟腑|臟腑]]理論相結合，闡述修煉長生成仙之術<ref>{{cite book|editor=胡孚琛|title=中華道教大辭典|date=1995|publisher=中國社會科學出版社|pages=341-342}}</ref>。

經中宣稱人首腦室及面部五官、胸腹內五臟六腑及腸胃等器官，皆有真人神仙居住其中。修道者若能常誦此經書，默念神名，存思身神形狀、服色、居處及其職司等等，便能通神感靈，使臟腑安和，形神相守。延年卻老，成仙升天。

經文尤其強調存思上中下三[[丹田|丹田]]之神，即腦部泥丸、胸部絳宮（心臟）及臍下命門之神。認為三丹田乃積精行氣之要穴，常思其神則自然長生不死。除誦經思神之外，經中又言及漱津咽液、吐納元氣、房中固精、服食五芽、飛奔日月等方術，特別重視積精累氣。

《黄庭经》所述的一些内修养生之术，与《[[周易参同契|周易参同契]]》的炼丹之道相结合，在[[唐|唐]][[宋|宋]]时期流变为[[内丹|内丹]]道，成为中唐以後道教养生方术的主流。

==影響==
《黃庭經》不僅被道教尊為重要經典，而且也很受知識分子的歡迎。晉[[王羲之|王羲之]]曾書寫《黃庭經》全文，是書法界的珍寶。[[歐陽修|歐陽修]]刪正過黃庭經，[[蘇軾|蘇軾]]、[[米芾|米芾]]、[[黃庭堅|黃庭堅]]曾書寫《黃庭經》，蘇軾又仿其文體作讚詞。[[陸游|陸游]]對《黃庭經》也很喜愛，曾作詩曰：「白髮始悟頤生妙，盡在《黃庭》兩卷中」。
