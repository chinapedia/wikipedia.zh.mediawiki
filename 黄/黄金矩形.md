'''黄金矩形'''是一个長和寬的比為[[黄金比例|黄金比例]] <math>\varphi</math> 的矩形。

:<math>\varphi = \frac{\sqrt{5} + 1}{2} \approx 1.6180339887...</math>

==特性==
在黄金矩形中以短邊為邊長劃一正方形，矩形剩下的部份是一個較小的黄金矩形，說明如下：

[[File:Golden_rectangles.png|File:Golden rectangles.png]]

假設一黄金矩形的短邊為 <math> b </math>，長邊為　<math> a= \varphi b</math><br />
若在黄金矩形中以短邊為邊長劃一正方形，則長邊剩下的長度為

<math> a - b = (\varphi - 1)b = \left(\frac{\sqrt{5} + 1}{2} - 1\right)b = \left(\frac{\sqrt{5} - 1}{2}\right)b = \frac{2}{\sqrt{5} + 1} b = \frac{1}{\varphi} b</math>

a /b和 b/a-b 的比值均為 <math>\varphi</math>，因此所組成的矩形仍為一黄金矩形。

===原理===
因為[[黃金比例|黃金比例]]等於
:<math>\frac{ 1 + \sqrt{5}}{2} : 1</math>
約等於 162：100

同乘2等於
:<math>1 + \sqrt{5} : 2</math>

約等於 162：100

將正方形的一邊看作2<br>
由中點到對角長的長度即是<math>\sqrt{5}</math>(由[[勾股定理|勾股定理]]求出)，故所求出的長邊即是<math>1 + \sqrt{5}</math><br>
