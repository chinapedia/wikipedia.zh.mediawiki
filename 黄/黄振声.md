'''黄振声'''（{{bd|1925年|9月18日|2014年|6月30日}}）原名'''黄振兴'''，籍贯[[广东省|广东省]][[中山市|中山市]]（原中山县），生于[[上海|上海]]，[[中国共产党|中国共产党]]及[[中华人民共和国|中华人民共和国]]记者、官员<ref name=cpc>{{Cite web |url =  http://cpc.people.com.cn/n/2014/0704/c87393-25240754.html|title =香港中旅集团原董事长黄振声同志逝世 享年89岁  |author =  |date =2014-07-04  |publisher = 中国共产党新闻网 |language =  |accessdate =  |quote =  }}</ref><ref name=qq>{{Cite web |url = http://news.qq.com/a/20140704/050744.htm |title =  香港中旅集团原董事长黄振声同志逝世 享年89岁|author =  |date = 2014-07-04 |publisher =腾讯  |language =  |accessdate =  |quote =  }}</ref>。

==生平==
1943年4月，黄振声加入[[中国共产党|中国共产党]]。1946年8月，任上海市学生联合会[[党组|党组]]书记。1947年6月，负责[[全国学联|全国学联]]及上海学联的工作。1949年，参加[[新政治协商会议筹备会|新政治协商会议筹备会]]并任代表<ref name=cpc/><ref name=qq/>。

1949年，黄振声曾任全国学联国外部部长。1955年6月到1959年6月，任[[哈尔滨工业大学|哈尔滨工业大学]]工程经济系主任兼党总支书记。1959年6月起，任[[新华社|新华社]]驻[[布拉格|布拉格]]首席记者。1979年，任[[香港|香港]][[中国通讯社|中国通讯社]]社长。1985年2月，任[[香港中旅集团|香港中旅集团]]董事长。1992年5月离休<ref name=cpc/><ref name=qq/>。

2014年6月30日，黄振声在北京逝世，享年89岁<ref name=cpc/><ref name=qq/>。
