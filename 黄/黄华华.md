'''黄华华'''（{{bd|1946年|10月|catIdx=Huang黃華華}}），[[广东省|广东省]][[兴宁县|兴宁县]]（今为[[兴宁市|兴宁市]]，属[[梅州市|梅州市]]）人，[[中共中央|中共中央]]委员，现任[[全国人大华侨委员会|全国人大华侨委员会]]副主任委员、粵港澳合作促進會榮譽會長，原[[中共广东省委|中共广东省委]]副書記兼省长；他長期於廣東工作，於2003年起出任[[廣東省省長|廣東省省長]]至2011年。

== 簡歷 ==
黃華華於1964年起就讀於[[中山大學|中山大學]]數力系；1969年9月參加工作並留校有待分配。翌年8月起分別擔任廣東省煤礦機械廠干部、基建連副連長、車間黨支部副書記及廠革委會副主任；並於1971年加入[[中國共產黨|中國共產黨]]。黃華華1978年出任[[中國共產主義青年團|共青團]][[韶關市|韶關市]]委書記；1982年出任共青團[[廣東省|廣東省]]委副書記；而三年後升任為共青團廣東省委書記並出任黨組書記。1983年9月起就讀於中央党校二年制中青年干部研究生班<ref name="CV">{{cite web |language=zh-cn |publisher=[[新華網|新華網]] |accessdate=2011年11月4日 |title=黄华华简历 |url=http://news.xinhuanet.com/ziliao/2003-02/20/content_737815.htm |deadurl=yes |archiveurl=https://web.archive.org/web/20060412095755/http://news.xinhuanet.com/ziliao/2003-02/20/content_737815.htm |archivedate=2006年4月12日 }}</ref>。

除[[中共十四大|中共十四大]]外，黃華華自1987年的[[中國共產黨第十三次全國代表大會|中共十三大]]起為代表至[[中共十七大|中共十七大]]；而他更於中共十五大上獲推為[[中国共产党第十五届中央委员会候补委员|中央候補委員]]，[[中国共产党第十六届中央委员会委员|十六]]、[[中国共产党第十七届中央委员会委员|十七大中央委員]]。黃華華於1987年12月起出任梅縣地委副書記；翌年梅縣升格為[[梅州市|梅州市]]而改任梅州市委副書記並出任市長一職<ref>{{cite web |language=zh-hant |publisher=[[香港蘋果日報|蘋果日報]] |date=2011年11月5日 |title=黃華華突辭職疑雲密佈　汪洋同日大談反腐　朱小丹代理廣東省長  }}</ref>；1992年4月起出任[[中共廣東省委|中共廣東省委]]常委、秘書長；1993年5月當選為廣東省委副書記、秘書長；1996年後任中共廣東省委副書記至2011年11月。

黃華華於1998年當選為廣州市委書記至2002年；2001年當選為[[廣州市人大常委會|廣州市人大常委會]]主任；2002年選為廣東省常務副省長，翌年1月於廣東省第十屆人大第一次會議中當選為[[廣東省省長|廣東省省長]]並於2008年1月24日獲得通過連任<ref name="CV" /><ref>{{cite web |language=zh-cn |publisher=[[新華網|新華網]] |date=2008年1月24日 |accessdate=2011年11月4日 |title=欧广源当选广东省人大常委会主任 黄华华当选广东省省长 |url=http://news.xinhuanet.com/newscenter/2008-01/24/content_7489503.htm |deadurl=yes |archiveurl=https://web.archive.org/web/20111025181703/http://news.xinhuanet.com/newscenter/2008-01/24/content_7489503.htm |archivedate=2011年10月25日 }}</ref>。黃華華從2003年3月起為[[中華人民共和國第十屆全國人民代表大會|第十屆全國人大代表]]及[[中華人民共和國第十一屆全國人民代表大會|第十一屆全國人大代表]]<ref name="CV" />。2007年5月25日於中共廣東省第十屆委員会第一次全體會議上獲選為廣東省委副書記<ref>{{cite web |language=zh-cn |publisher=[[新華網|新華網]] |date=2007年5月25日 |accessdate=2011年11月4日 |title=张德江当选广东省委书记 黄华华刘玉浦为副书记 |url=http://news.xinhuanet.com/local/2007-05/25/content_6151400.htm |deadurl=yes |archiveurl=https://web.archive.org/web/20111025214702/http://news.xinhuanet.com/local/2007-05/25/content_6151400.htm |archivedate=2011年10月25日 }}</ref>。

黃華華於2011年11月4日在廣州召開的廣東省第十一届人大常委会第二十九次會議上請求辭任省長一職，並獲通過接受；其省長一職由原廣東省常務副省長[[朱小丹|朱小丹]]出任代理省長<ref>{{cite news |language=zh-hant |publisher=[[南方日報|南方日報]] |date=2011年11月4日 |accessdate=2011年11月4日 |title=黃華華辭去廣東省省長職務朱小丹代理省長 全国人大常委会2011年12月31日经表决决定任命黄华华为全国人大华侨委员会副主任委员 |url=http://gcontent.oeeee.com/d/ba/dba132f6ab6a3e3d/Blog/47c/faf730.html |deadurl=yes |archiveurl=https://web.archive.org/web/20111105104908/http://gcontent.oeeee.com/d/ba/dba132f6ab6a3e3d/Blog/47c/faf730.html |archivedate=2011年11月5日 }}</ref>。在黃華華主政省長期間成功主辦[[廣州亞運|廣州亞運]]以及[[深圳大運會|深圳大運會]]外，還積極推動[[廣東|粵]][[香港|港]]、粵[[澳門|澳]]以及[[泛珠三角|泛珠三角]]之間的區域合作、落實[[CEPA|CEPA]]政策、改善[[珠江|珠江]]水質問題、放寬[[港澳個人遊|自由行]]到港澳政策、[[八達通|八達通]]與[[嶺南通|嶺南通]]互聯互通使用<ref>{{cite web |language=zh-hant |publisher=[[香港蘋果日報|蘋果日報]] |date=2011年11月5日 |title=港議員憂影響粵港合作 }}</ref>；但在他任內出現多個群眾事件，勞工事件如[[2010年深圳富士康員工墜樓事件|富士康員工自殺事件]]等。

2011年12月31日獲任命為[[全國人大華僑委員會|全國人大華僑委員會]]副主任委員。
