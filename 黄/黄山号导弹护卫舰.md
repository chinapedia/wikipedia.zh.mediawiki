'''“黄山”号导弹护卫舰'''（[[舷号|舷号]]570）简称'''“黄山”舰'''，是[[中国人民解放军海军|中国人民解放军海军]]第三艘[[054A型导弹护卫舰|054A型导弹护卫舰]]，可单独或协同海军其他兵力攻击敌[[水面舰艇|水面舰艇]]、[[潜艇|潜艇]]，具有较强的远程警戒和防空作战能力。黄山舰由[[黄埔造船厂|黄埔造船厂]]于2005年4月28日开工建造，于2007年3月18日下水，2008年5月13日入列，现服役于[[南海舰队|南海舰队]][[驱逐舰第二支队|驱逐舰第二支队]]。<ref name="FFG-570">{{cite web|url=http://www.xinhs.cn/article.asp?id=9057 |title=黄山舰：大海中黄山的流动窗口 |publisher=黄山网 |date=2009-07-27 }}</ref>黄山舰舰名取自[[安徽省|安徽省]][[黄山市|黄山市]]，其入列命名授旗仪式于2008年5月13日在[[广东省|广东省]][[湛江市|湛江市]]举行。<ref name="FFG-570"/>

== 历史 ==
2009年4月2日，“黄山”号导弹护卫舰和[[“深圳”号导弹驱逐舰|“深圳”号导弹驱逐舰]]等组成中国海军第二批护航编队，从广东省湛江市起航，赴[[亚丁湾|亚丁湾]]、[[索马里|索马里]]海域执行护航任务。该次护航标志着中国海军护航行动进入有序接替、常态化运行的新阶段。<ref>{{cite web |url=http://www.chinamil.com.cn/site1/xwpdxw/2009-04/03/content_1713241.htm |title=中国海军第二批护航编队起航 |publisher=中国军网 |date=2009-04-03 |deadurl=yes |archiveurl=https://web.archive.org/web/20140826161907/http://www.chinamil.com.cn/site1/xwpdxw/2009-04/03/content_1713241.htm |archivedate=2014-08-26 }}</ref>该次护航历时142天，黄山舰等为85艘船舶提供区域掩护，成功解救4艘遇袭外国船只，查证、驱离可疑船只129艘。编队于8月21日返回湛江市某军港。<ref>{{cite web|url=http://gb.cri.cn/27824/2009/08/21/3785s2599363.htm |title=中国海军第二批护航编队圆满完成任务返回湛江 |publisher=国际在线 |date=2009-08-21 }}</ref>

2010年7月26日，“黄山”号导弹护卫舰、[[“海口”号导弹驱逐舰|“海口”号导弹驱逐舰]]及[[“宁波”号导弹驱逐舰|“宁波”号导弹驱逐舰]]等舰艇在南海舰队组织下在南海某海域展开了海军多兵种合同实兵实弹演练。<ref>{{cite web|url=http://www.mod.gov.cn/photo/2010-07/30/content_4179360.htm |title=海军多兵种合同实兵实弹演练在南海举行 |publisher=中华人民共和国国防部网 |date=2010-07-30 }}</ref>该次演练在3个海区、18000多平方公里的海域展开，发射导弹16型71枚，被评价为当时解放军海军历史上参训要素最全、实射导弹最多、信息化程度最高、电磁环境最复杂的一次联合对抗实战化演练。<ref>{{cite web |url=http://tp.chinamil.com.cn/news/2010-07/30/content_4269718.htm |title=南海直击中国海军史上最高难度联合演练 |publisher=中国军事图片中心 |date=2010-07-30 |deadurl=yes |archiveurl=https://web.archive.org/web/20140826161053/http://tp.chinamil.com.cn/news/2010-07/30/content_4269718.htm |archivedate=2014-08-26 }}</ref>

2012年11月9日，“黄山”号导弹护卫舰、[[“衡阳”号导弹护卫舰|“衡阳”号导弹护卫舰]]及[[“青海湖”号综合补给舰|“青海湖”号综合补给舰]]等组成中国海军第十三批护航编队，从广东省湛江港起航，赴亚丁湾、索马里海域执行护航任务。<ref>{{cite web|url=http://paper.people.com.cn/rmrbhwb/html/2012-11/10/content_1140294.htm?div=-1 |title=海军第13批护航编队起航 赴亚丁湾索马里海域 |publisher=人民日报海外版 |date=2012-11-10 }}</ref>该次护航历时196个日夜，黄山舰等完成37批、166艘次中外船舶的护航任务，查证、驱离海上可疑目标68批、110艘次。完成护航任务后，编队访问了[[马耳他|马耳他]]、[[阿尔及利亚|阿尔及利亚]]、[[摩洛哥|摩洛哥]]、[[葡萄牙|葡萄牙]]和[[法国|法国]]等5国，并于2013年5月23日回到湛江某军港。<ref>{{cite web |url=http://news.xinhuanet.com/mil/2013-05/23/c_115885966.htm |title=中国海军第十三批护航编队完成任务回国 |publisher=新华网 |date=2013-05-13 |deadurl=yes |archiveurl=https://web.archive.org/web/20140826115645/http://news.xinhuanet.com/mil/2013-05/23/c_115885966.htm |archivedate=2014-08-26 }}</ref>

2013年10月18日，“黄山”号导弹护卫舰和[[“广州”号导弹驱逐舰|“广州”号导弹驱逐舰]]起航参加海军“[[机动-5号|机动-5号]]”远海实兵演习，奔赴西太平洋海域执行海空立体攻防、舰机联合反潜等演练任务。<ref>{{cite web|url= http://www.chinanews.com/mil/2013/11-05/5462698.shtml|title=直击中国海军机动-5号演习：真正像打仗一样演习 |publisher=中新网| date=2013-11-05 }}</ref>该次演习前后共持续15天，于2013年11月1日结束，各参演兵力陆续返回驻地。<ref>{{cite web|url= http://news.xinhuanet.com/2013-11/01/c_117973459.htm|title=海军“机动－5号”实兵演习落下帷幕|publisher=新华网| date=2013-11-01 }}</ref>

2013年12月19日，由“黄山”号导弹护卫舰、[[“运城”号导弹护卫舰|“运城”号导弹护卫舰]]及[[“微山湖”号综合补给舰|“微山湖”号综合补给舰]]等组成海军南海舰队舰艇编队从湛江市启航，赴[[西太平洋|西太平洋]]等海域进行海上持续戒备、反恐反海盗、联合搜救等科目的演练。<ref>{{cite web|url= http://navy.81.cn/content/2013-12/19/content_5698832.htm|title=南海舰队舰艇编队从湛江启航启动新一轮远航训练|publisher=中国海军网| date=2013-12-19 }}</ref>

2014年2月17日，“黄山”号导弹护卫舰等执行[[叙利亚|叙利亚]][[化学武器|化学武器]]海运护航任务接替兵力从广东省湛江市某军港起航，赴[[地中海|地中海]]接替已完成3批叙化武海运护航任务的[[“盐城”号导弹护卫舰|“盐城”号导弹护卫舰]]。<ref>{{cite web |url=http://www.chinamil.com.cn/jfjbmap/content/2014-02/18/content_67685.htm |title=黄山舰起航赴地中海 接替盐城舰执行叙利亚化武海运护航任务 |publisher=中国军网 |date=2014-02-18 |deadurl=yes |archiveurl=https://web.archive.org/web/20140826113528/http://www.chinamil.com.cn/jfjbmap/content/2014-02/18/content_67685.htm |archivedate=2014-08-26 }}</ref>黄山舰与[[俄罗斯|俄罗斯]]、[[丹麦|丹麦]]、[[挪威|挪威]]等国舰艇联合实施了13批护航任务完成叙利亚化学武器海运护航任务后，于7月24日返回湛江市某军港。<ref>{{cite web |url=http://news.xinhuanet.com/mil/2014-07/24/c_126793653.htm |title=南海舰队黄山舰完成叙化武海运护航任务返回湛江 |publisher=新华网 |date=2014-07-24 |deadurl=yes |archiveurl=https://web.archive.org/web/20140826115729/http://news.xinhuanet.com/mil/2014-07/24/c_126793653.htm |archivedate=2014-08-26 }}</ref>

2014年12月29日，“黄山”号导弹护卫舰驶往[[印度尼西亚|印度尼西亚]]海域，参与[[印尼亚洲航空8501号班机空难|印尼亚洲航空8501号班机空难]]的搜救行动。<ref name="12-29">[http://news.sohu.com/20141230/n407407443.shtml 中国海军黄山舰出发搜寻亚航失联航班] 2014-12-30 搜狐</ref>
