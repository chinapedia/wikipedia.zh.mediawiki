'''黃尾嶼'''又被稱作'''黃麻嶼'''、'''黃毛山'''、'''黃毛嶼'''或者是'''黃嶼'''，在[[日本|日本]]則稱呼為'''久場島'''（{{jpn|j= 久場島 |hg= くばじま |rm= Kuba-shima }}），是一座位於[[臺灣|臺灣]]東北方[[島鏈|島鏈]][[釣魚臺列嶼|釣魚臺列嶼]]東側的[[荒島|荒島]]<ref name="日本大學">{{Cite web |url= http://www.geo.chs.nihon-u.ac.jp/tchiba/volcano/kobetsu/id-611.htm |script-title= ja:黄尾嶼 |author= {{lang|ja|小林哲夫}}和{{lang|ja|藤沢康弘}} |date=  |publisher= [[日本大學|日本大學]] |language= ja |accessdate= 2012年2月17日 |quote=  }}{{dead link|date=十一月 2017 |bot=InternetArchiveBot }}</ref><ref>{{Cite web|url=http://twgeoref.moeacgs.gov.tw/star/2007/20072011/ap01.pdf|title=附件一 富貴角海域地質試測圖說明書|author=[[經濟部中央地質調查所|經濟部中央地質調查所]]|date=2007年12月|publisher=[[中華民國經濟部|中華民國經濟部]]|language=zh-hant|accessdate=2012年2月17日|quote=|deadurl=yes|archiveurl=https://web.archive.org/web/20141016074003/http://twgeoref.moeacgs.gov.tw/star/2007/20072011/ap01.pdf|archivedate=2014年10月16日}}</ref>。黃尾嶼在一些古代文獻亦被稱為「黄尾山」、「黃尾島」以及「烏港」，並且於明清時期被視為[[琉球冊封使|琉球冊封使]]的航路指標島嶼之一<ref>{{Cite web |url= http://military.china.com/history4/62/20131118/18154140.html |title= 明代中國與琉球國界的界定：確定釣魚島歸屬明朝 |author= 萬明 |date= 2012年9月15日 |publisher= [[中華網|中華網]] |language= zh-hant |accessdate= 2013年11月27日 |quote=  |archive-url= https://web.archive.org/web/20160809114733/http://military.china.com/history4/62/20131118/18154140.html |archive-date= 2016年8月9日 |dead-url= yes }}</ref>；而在[[琉球國|琉球國]][[第二尚氏|第二尚氏]]當時的文獻資料中也以「黃尾嶼」記載之，而由於[[島嶼|島嶼]]上棲息著大量鳥隻使得該島又被稱作「鳥島」<ref>{{Cite web |url= http://www.ioc.u-tokyo.ac.jp/~worldjpn/documents/texts/JPUS/19700831.O1J.html |script-title= ja:尖閣列島に関する琉球立法院決議および琉球政府声明 |author= [[田中明彥|田中明彥]] |date= 1971年8月31日 |publisher= {{Link-ja|東京大學東亞文化研究所|東京大学東洋文化研究所}} |language= ja |accessdate= 2012年2月17日 |quote=  }}</ref>。而日語之中所使用的漢字「久場」一詞，則是將[[琉球語|琉球語]][[首里－那霸中央方言|首里－那霸中央方言]]中原本指[[蒲葵屬|蒲葵]]的「{{lang|ryu|クバ}}」借用而成，而沖繩地區便常以「{{lang|ryu|クバ島}}」來指稱外觀像蒲葵的黃尾嶼。此外[[八重山群島|八重山群島]]地區則由於島嶼土壤呈現黃色緣故，因此常另外以「黃色之島」（{{lang|rys|チールージマ}}）形容之。

== 歷史 ==
1895年4月17日，於[[甲午戰爭|甲午戰爭]]中戰敗的[[大清帝國|大清帝國]]和[[大日本帝國|大日本帝國]]簽訂《[[馬關條約|馬關條約]]》，清朝割讓臺灣以及附屬島嶼給與日本。不過早在1月14日時，日本政府便以[[無主地|無主地]]先佔原則宣布正式將[[釣魚臺列嶼|釣魚臺列嶼]]納入[[沖繩縣|沖繩縣]]管理範圍，而這也包括作為釣魚臺列嶼之一的黃尾嶼<ref>{{Cite web |url= http://www.economist.com/news/christmas/21568696-behind-row-over-bunch-pacific-rocks-lies-sad-magical-history-okinawa-narrative |title= The Senkaku or Diaoyu Islands Narrative of an empty space |author=  |date= 2012年12月22日 |publisher= 《[[經濟學人|經濟學人]]》 |language= en |accessdate= 2013年2月11日 |quote=  }}</ref>。1896年時[[古賀辰四郎|古賀辰四郎]]向日本政府免費租借釣魚臺、黃尾嶼、[[北小島|北小島]]和[[南小島|南小島]]，並且雙方同意在租借30年後改以付費的方式繼續出借土地。不過到了1932年5月20日，古賀辰四郎的兒子古賀善次便向日本政府提議購買釣魚臺和黃尾嶼，同一年還付費買下了包括北小島和南小島等土地。

1945年[[第二次世界大戰|第二次世界大戰]]正式宣告結束後，[[美國|美國]]於1946年2月2日開始對北緯30度以下的日本領土實施軍事統治<ref name = "共同通訊社">{{Cite web |url= http://www.japantimes.co.jp/text/nn20101208f2.html |title= Ishigaki fishermen fret over Senkaku encroachment |author= [[共同通訊社|共同通訊社]] |date= 2010年2月17日 |publisher= 《[[日本時報|日本時報]]》 |language= en |accessdate= 2012年2月17日 |quote=  }}</ref>。而在1952年4月28日《[[舊金山和約|舊金山和約]]》正式生效以後，黃尾嶼則改歸為[[琉球政府|琉球政府]]進行管理。而在1970年7月時，琉球政府開始於釣魚臺、黃尾嶼、[[赤尾嶼|赤尾嶼]]、北小島和南小島等釣魚臺列嶼的島嶼上架設禁止非法登陸的警告牌<ref>{{Cite web |url= http://www.y-mainichi.co.jp/news/14231/ |script-title= ja:尖閣諸島「警告板の設置作業者を」　与那国の関係者探す |author=  |date= 2009年8月15日 |publisher= 《{{Link-ja|八重山每日新聞|八重山毎日新聞}}》 |language= ja |accessdate= 2012年2月17日 |quote=  }}</ref>。1972年5月15日時，美國決定連同琉球群島以及釣魚臺列嶼都[[沖繩返還|歸還給]]日本政府進行管理<ref>{{Cite web |url= http://select.nytimes.com/gst/abstract.html?res=F20714FB3D591A7493C3A8178AD95F458785F9&scp=44&sq=1972+okinawa&st=p |title= SENATE ENDORSES OKINAWA TREATY; Votes 84 to 6 for Island's Return to Japan |author= John W. Finney |date= 1971年11月11日 |publisher= 《[[紐約時報|紐約時報]]》 |language= en |accessdate= 2012年2月17日 |quote=  }}</ref>。與此同時，美國政府和日本政府也同意將黃尾嶼和赤尾嶼列為[[駐日美軍|駐日美軍]]的轟炸練習處<ref name="琉球新報">{{Cite web |url= http://ryukyushimpo.jp/news/storyid-169158-storytopic-3.html |script-title= ja:尖閣射爆撃場、米軍３０年余不使用　政府「必要」と認識 |author=  |date= 2010年10月23日 |publisher= 《[[琉球新報|琉球新報]]》 |language= ja |accessdate= 2012年2月17日 |quote=  |deadurl= yes |archiveurl= https://web.archive.org/web/20121204062828/http://ryukyushimpo.jp/news/storyid-169158-storytopic-3.html |archivedate= 2012年12月4日 }}</ref>。

=== 生態環境 ===
日本企業家[[古賀辰四郎|古賀辰四郎]]過去曾於黄尾嶼西南地區興建[[柴魚片|柴魚片]]工廠以及鳥羽加工廠，然而隨著太平洋戰爭爆發以及工廠結束營運後島嶼也成為一座荒島。而在1972年5月15日到1978年6月為止，黃尾嶼和赤尾嶼則一直被美軍作為轟炸練習基地使用之<ref name = "琉球新報"/>。今日黃尾嶼島上棲息著大量的[[短尾信天翁|短尾信天翁]]，並且有報告指出亦有信天翁開始於此結巢繁殖<ref>{{Cite web |url= http://ryukyushimpo.jp/news/storyid-111447-storytopic-86.html |script-title= ja:アホウドリの繁殖確認/尖閣・南小島 |author=  |date= 2001年4月4日 |publisher= 《[[琉球新報|琉球新報]]》 |language= ja |accessdate= 2012年2月17日 |quote=  }}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。此外在島嶼周遭還有包括[[黄尾嶼龍蝦|黄尾嶼龍蝦]]在內的豐富海產，另外島嶼上還有棲息著作為特有種之的[[黄尾嶼蜈蚣|黄尾嶼蜈蚣]]。
