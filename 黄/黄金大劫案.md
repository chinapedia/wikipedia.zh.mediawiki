{{engname|《黄金大劫案》|''Guns N'Roses''}}是一部[[小马奔腾|小马奔腾]]出品、[[宁浩|宁浩]]执导的中国喜剧片，中国于2012年4月24日公映<ref>{{cite web|url=http://news.mtime.com/2012/02/21/1482503.html | language= | title=宁浩新片预告神秘亮相 英文片名致敬"枪花" |publisher=Mtime时光网 |date=2012-02-21 |accessdate= }}</ref>。

== 剧情简介 ==
故事发生于1930年代[[满洲国|满洲国]]时期的[[中国东北地区|东北]]<ref name="开机">{{cite web|url=http://news.mtime.com/2011/08/05/1466800.html | language= | title=宁浩拍新片《黄金大劫案》 偷完翡翠劫黄金 |publisher=东方早报 |date=2011-08-05 |accessdate= }}</ref>，主要讲述一个小混混在乱世中的成长史。

“小东北”是个依靠偷抢拐騙为生的小[[流氓|流氓]]，在金钱的驱使下，他阴错阳差地卷入进了一起劫金大案——与一个以[[抗日|抗日]]为理想的「救國會」成员一起合作，企图劫下运往金库的8[[吨|吨]][[黄金|黄金]]，以阻止[[日本人|日本人]]用这笔财富来购买[[意大利|意大利]][[军火|军火]]。

== 创作 ==
2011年5月在[[上海|上海]][[松江影视基地|松江影视基地]]开拍<ref name= "开机"/>。[[陈思勤|陈思勤]]任美术设计，[[梁吉泳|梁吉泳]]任动作指导。

== 评价 ==
本片与宁浩之前的《[[疯狂的石头|疯狂的石头]]》和《[[疯狂的赛车|疯狂的赛车]]》这两部喜剧作品相比，风格有很大转变，即本片除了有不少笑点外，还有以颂扬[[愛國主義|愛國思想]]、[[民族主义|民族主义]]为出发的煽情[[桥段|桥段]]和泪点。

== 票房 ==
中国大陆票房为1.54亿人民币。
