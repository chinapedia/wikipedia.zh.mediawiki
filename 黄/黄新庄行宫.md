'''黄新庄行宫'''，位于[[北京市|北京市]][[房山区|房山区]][[良乡地区|良乡地区]]黄辛庄村，是[[清朝|清朝]]所建的皇家行宫。

==简介==
黄新庄行宫建于[[乾隆|乾隆]]十三年（1748年），是清朝皇帝谒西陵时的驻跸之所。清朝自北京到西陵沿途共建有4座行宫，即黄新庄行宫、[[半壁店行宫|半壁店行宫]]、[[秋澜行宫|秋澜行宫]]、[[梁格庄行宫|梁格庄行宫]]。梁格庄行宫是其中唯一保存至今的行宫。<ref name=xhw>[http://news.xinhuanet.com/ziliao/2003-09/24/content_1097477_2.htm 世界文化遗产——明清皇家陵寝，新华网，2003-09-24]</ref>

《日下旧闻考》卷一百三十三《京畿良乡县》记载：<ref>于敏中，日下旧闻考，北京古籍出版社，2001年</ref>

《乾隆南巡图》第一卷《启跸京师》以“御制辛未孟春奉皇太后南巡启跸京师近体言志”为题，描绘了乾隆十六年正月十三日，乾隆帝奉太后自北京紫禁城内的[[乾清门|乾清门]]启銮，经过[[大清门|大清门]]出[[正阳门|正阳门]]，沿着西河沿大街向西行，转出[[广安门|广宁（安）门]]，过[[宛平城|宛平县拱极城]]、[[卢沟桥|卢沟桥]]、[[长辛店镇|长新(辛)店]]，驻跸于黄新庄行宫的情景。<ref>[http://www.qunxianart.com/news/zonghe/2014-12-29/70649_4.html “国博典藏《乾隆南巡图》长卷数字展示”正式对外开放，群贤艺术网，2014-12-29]</ref>

黄新庄行宫位于黄辛庄村内的公路西侧临街处，据说行宫最后几间建筑在1980年代被拆除。旧址现为一片仿古建筑。仿古建筑院中有片空地，空地上有一株古[[柏树|柏树]]、一座西湖石假山，旁有座小土包。这里应是黄新庄行宫的后花园。<ref>[http://news.sohu.com/20070715/n251062028.shtml 良乡古韵，搜狐，2007-07-15]</ref>

2012年，黄新庄行宫遗址以“黄辛庄行宫遗址”之名被列为[[北京市房山区普查登记文物|北京市房山区普查登记文物]]，2013年1月挂牌。<ref name=buke>{{Cite web |url = http://whw.bjfsh.gov.cn/whycbh/wwbh/19078.htm |title = 北京市房山区第三次全国文物普查不可移动文物名录 |author =  |date = 2015-08-19 |publisher = 北京市房山区文化委员会 |language =  |accessdate =  |quote =  |archive-url = https://web.archive.org/web/20161228032506/http://whw.bjfsh.gov.cn/whycbh/wwbh/19078.htm# |archive-date = 2016-12-28 |dead-url = yes }}</ref>
