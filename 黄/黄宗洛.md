'''黄宗洛'''（{{bd|1926年|9月26日|2012年|6月30日|catIdx=Huang黄宗洛}}），[[浙江|浙江]][[瑞安|瑞安]]人。[[中国|中国]][[北京人民艺术剧院|北京人民艺术剧院]][[话剧|话剧]][[演员|演员]]。就读於[[中央戏剧学院|中央戏剧学院]]，他的哥哥[[黄宗江|黄宗江]]、姐姐[[黄宗英|黄宗英]]都是著名的演艺人士。他的话剧代表作品《[[茶馆_(话剧)|茶馆]]》、《[[三块钱国币|三块钱国币]]》，電視代表作品《[[西游记_(1999年电视剧)|西游记]]》的铜台知府、《[[水浒传_(电视剧)|水浒传]]》的郓城知县、《[[笑傲江湖_(2001年电视剧)|笑傲江湖]]》的平一指、《新天仙配之七仙女正传》的槐树精，有“小草”、“龙套大师”的美称。<ref name="bull ">{{cite web|url=http://roll.sohu.com/20120630/n346955641.shtml|title=图文：黄宗洛陈强张瑞芳 盘点那些绝尘而去的大师们|accessdate=2012年6月30日|publisher=搜狐网|work=|language=zh-cn|deadurl=yes|archiveurl=https://web.archive.org/web/20120702033451/http://roll.sohu.com/20120630/n346955641.shtml|archivedate=2012年7月2日}}</ref>

== 生平 ==
1926年，黄宗洛出生在北京。<ref name=" fish" >{{cite web |url= http://ent.163.com/12/0630/16/858U19M300032DGD.html|title=著名表演艺术家黄宗洛去世 享年85岁 |accessdate=2012-06-30 |publisher=网易 |work= |language =zh-cn}}</ref>

1944，黄宗洛初中毕业于[[天津市耀华中学|天津市耀华中学]]，毕业后回到瑞安老家。1946年，黄宗洛在瑞安中学毕业，后来陆续辗转于[[北平|北平]][[燕京大学|燕京大学]]心理系、[[解放区|解放区]][[华北大学|华北大学]]、[[晋察冀|晋察冀]][[华北联合大学|华北联合大学]]、[[中央戏剧学院|中央戏剧学院]]。<ref name=" fish" />

2012年6月30日，黄宗洛因为[[肺|肺部]]感染而病逝。<ref name="man ">{{cite web |url=http://et.21cn.com/gundong/etscroll/2012/06/30/12257967.shtml |title=黄宗洛自述话剧舞台生涯：人物附我心 |accessdate=2012-06-30 |publisher=21CN |work= |language=zh-cn |archive-url=https://web.archive.org/web/20150603032407/http://et.21cn.com/gundong/etscroll/2012/06/30/12257967.shtml |archive-date=2015-06-03 |dead-url=yes }}</ref>
