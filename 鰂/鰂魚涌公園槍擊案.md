'''鰂魚涌公園槍擊案'''於2018年6月26日在[[香港|香港]][[香港島|香港島]][[鰂魚涌公園|鰂魚涌公園]]發生，共造成2死2傷<ref>{{Cite news |url=http://www.takungpao.com.hk/hongkong/text/2018/0627/180525.html |title=香港鰂魚涌槍擊案增至2死 |date=2018-06-27 |work=大公報}}</ref>，為香港近年來其次極少數涉及[[槍械|槍械]]的[[罪案|罪案]]，對上一次涉及真槍的槍擊案是2014年發生的[[啟晴邨槍擊案|啟晴邨槍擊案]]。

== 事發經過 ==
44歲的詹心桀與魏姓哥哥及分別各兩名舅父及阿姨，相約於[[太古城|太古城]]一間酒樓討論遺產分配問題時未有結論。至下午3時許，事主們移步至[[鰂魚涌公園|鰂魚涌公園]]再次討論。其間疑一言不合，詹女由懷中掏出一柄[[半自動手槍|半自動手槍]]，向長輩們連轟最少四槍，傳出「嘭嘭嘭嘭」巨響。四人全部受傷，其中兩人頭傷倒地不起，另兩人手傷清醒。而魏男目擊後逃跑及報警。

詹女行兇後，將手槍拾起，逃向[[太古城中心|太古城中心]]四期。有市民尾隨追截，與保安員在二樓大堂合力將她截停。不久，警方反恐隊員荷槍實彈趕至將她拘捕。詹女被捕時尚未反抗，神情冷靜。警員在其懷中檢獲一支手槍及彈匣內五粒子彈，質疑與案有關。涉案手槍型號為意大利{{tsl|en|Beretta 950|Beretta 950 Jetfire}}半自動手槍，彈匣裝彈數量為8枚。

== 疑犯 ==
疑犯原姓魏，後改隨母姓詹，時年44歲，居於[[將軍澳|將軍澳]][[健明邨|健明邨]]健晴樓。報道指，詹心桀沒有精神病紀錄，亦沒有案底。

她是一名保鑣以及保鑣公司負責人，但沒有槍械牌照及保安牌照。在2017年時接受《[[蘋果日報_(香港)|蘋果日報]]》專訪時易名李小姐。

== 受害者 ==
4名受害者分別為事主二姨詹少芬（80歲）、三舅父詹前駒（72歲）、四舅父詹鎮基（62歲）及五姨詹小慧（60歲）。

詹少芬送院後傷重不治，詹鎮基送院搶救至事發後第二天傷重不治<ref>{{Cite web |url=https://news.mingpao.com/ins/instantnews/web_tc/article/20180627/s00001/1530084243170 |title=槍擊案再增一死　女槍手四舅父不治 |date=2018-06-27 |publisher=明報}}</ref>，詹前駒傷勢嚴重，詹小慧受輕傷。

== 法庭審訊 ==
2018年6月28日，詹心桀被押送往[[東區裁判法院|東區裁判法院]]出庭應訊，詹被控兩項謀殺罪和有意圖射擊罪。審訊後，被告還押至9月20日再訊。（案件編號：ESCC 1558/2018）

再訊後，女被告毋須答辯。控方指部分文件尚未完成，故再將案件押後至11月29日以待警方進一步查核；辯方並無反對押後及保釋申請，繼續還押監所看管。
