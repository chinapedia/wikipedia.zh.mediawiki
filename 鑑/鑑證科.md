'''鑑證科'''（[[英文|英文]]：'''I'''dentification '''B'''ureau，[[縮寫|縮寫]]：'''IB'''）隸屬於[[香港警務處|香港警務處]][[刑事及保安處|刑事及保安處]][[刑事部|刑事部]]，主要責任為[[指紋識別|指紋識別]]、[[法證|法證]][[攝影|攝影]]及處理[[脫氧核糖核酸|脫氧核糖核酸]]事務<ref>[http://www.police.gov.hk/info/review/2010/tc/pdf/TC_E_organisation.pdf 鑑證科]《香港警察年報》2010</ref>。該科的任務是從[[罪案現場|罪案現場]]尋找[[指紋|指紋]]從而進行指紋識別，及向[[香港法院|法庭]]就鑑定身份事宜提供[[專家|專家]][[證據|證據]]。<ref>[http://www.police.gov.hk/info/review/2009/tc/pdf/TC_E_p68.pdf 鑑證科] 《香港警察年報》2009</ref>

==歷史==
2008年3月，第三代的[[電腦輔助掌紋及指紋鑑證系統|電腦輔助掌紋及指紋鑑證系統]]的裝置程序正式投入服務新型系統除了能夠搜集偵查案件發生現場所搜集到的指紋及[[掌紋|掌紋]]外，相比起上兩代的系統，可以大大地提高圖紋印證搜集偵查的[[效率|效率]]。同時，準確性亦不斷增加，由第一代的70%、第二代的80%，至現在的90%。以往鑑證科每月經過電腦系統成功辨別證明45宗案件的指紋，現在多達80宗。這項最先進及精密的科技推出後，使到香港警務處成為世界上擁有最完善指紋識別設備的警隊之一。<ref>[http://www.police.gov.hk/info/review/2007/big5/operation/07.htm 法證支援服務] 《香港警察年報》2007</ref><ref>[http://www.icac.org.hk/newsl/issue5big5/pe.html 香港警隊－如何在執法中運用資訊科技]</ref>

2010年代，[[軍械法證科|軍械法證科]]併入鑑證科。

2011年，於[[保安局|保安局]]領導下，鑑證科於年內推行了[[性罪行定罪紀錄查核機制|性罪行定罪紀錄查核機制]]，為香港[[僱主|僱主]]提供可靠渠道，讓他們在聘請[[僱員|僱員]]從事與[[兒童|兒童]]或精神上無行為能力人士有關的[[工作|工作]]時，能夠了解申請人是否曾經干犯過[[性罪行|性罪行]]的定罪[[紀錄|紀錄]]<ref>[http://www.police.gov.hk/info/review/2011/ 法證支援] 《香港警察年報》2011</ref>。

== 組織 ==
*鑑證科：由一名[[高級警司|高級警司]]出任主管，由一名[[警司|警司]]出任副主管。
**'''支援課'''：由一名[[總督察|總督察]]領導。
**'''電腦輔助指紋鑒證系統課'''（[[英文|英文]]：'''Computer Assisted Fingerprint Identification System Section'''）：由一名總督察領導，下設一名[[高級督察|高級督察]]。
**'''先進科技課'''（[[英文|英文]]：'''Advanced Technology Section'''）：由一名總督察領導，下設兩名高級督察。
**'''罪案現場課'''（[[英文|英文]]：'''Scenes of Crime Section'''）：由一名總督察領導，下設3名高級督察/督察。
**'''攝影課'''（[[英文|英文]]：'''Photographic Section'''）：由一名總督察領導，下設一名警署警長、一名'''高級特別攝影師'''（[[英文|英文]]：'''Senior Special Photographer'''）和一名'''一級特別攝影師'''（[[英文|英文]]：'''Special Photographer I'''）。
**無犯罪紀錄證明書及犯罪紀錄資料辦事處：由一名一級行政主任領導。
**性罪行定罪紀錄查核辦事處：由一名一級行政主任領導。

== 遴選訓練 ==
投考加入鑑證科的人員必須先通過傾向測試及[[面試|面試]]，通過後會經過挑選，參與為期8星期的入門課程，於考試合格後，會被調動派遣到鑑證科的不同部門，及接受培育訓練。於鑑證科服務滿3年後，可以參與中班課程。視乎表現及才能，合資格者會於稍後階段被挑選參與高級指紋課程，再經過進一步的考核後，才會被考慮挑選為[[指紋專家|指紋專家]]。此外，鑑證科一直為人員提供持續培育訓練，經常邀請[[海外|海外]][[專家|專家]]到訪舉辦專題講學，人員亦會出席不同[[講座|講座]]，研究討論會及海外課程等等<ref>[http://www.police.gov.hk/offbeat/956/chi/n07.htm 退休高級警司盧桂朗 見證警隊鑑證技術一日千里] 《警聲》 第956期</ref>。

== 大事記 ==
* [[德福花園五屍命案|德福花園五屍命案]]，鑑證科使用當時最先進的[[多波段光源機|多波段光源機]]在受害人府上蒐集證據，尋找到及成功辨別確認[[疑犯|疑犯]]的指模，成功破案及將疑犯繩之於法。
* [[性工作者連環兇殺案|性工作者連環兇殺案]]，鑑證科在其中一處案件發生現場的[[門框|門框]]尋找到一副掌紋，使用當時新購入的電腦輔助掌紋及指紋鑑證系統進行辨別認證，於12小時識破疑犯[[身份|身份]]，並且將其繩之於法。
