'''桓姓'''，为[[中文姓氏|中文姓氏]]之一，百家姓排行第407。

== 起源 ==
其來源有四：
*據《姓氏考略》載， [[黃帝|黃帝]]有一個大臣名為[[桓常|桓常]]，其子孫以“桓”字為姓。
*出自[[姜姓|姜姓]][[齐桓公|齐桓公]]後裔，以諡號為氏。
*出自[[子姓|子姓]][[宋桓公|宋桓公]]後裔，以諡號為氏。
*出自[[鮮卑|鮮卑]]族[[烏丸氏|烏丸氏]]所改。

== 遷徙分佈 ==

== 郡望堂號 ==
*[[譙郡|譙郡]]：東漢建元年間的時候，將沛郡分出了一部分設置了譙郡，相當於今天的安徽、河南兩省之間的地區。
*匡晉堂：晉朝桓氏有十一個將軍（桓石秀、桓石虔、桓伊、桓沖、桓雄、桓雲、桓濤、桓溫、桓霍、桓謙、桓振）曾匡扶晉室。

== 歷史名人 ==
*[[桓齮|桓齮]]，[[戰國_(中國)|戰國]]時代軍事人物
*[[桓寬|桓寬]]，字次公，[[西漢|西漢]]時[[河南|河南]][[上蔡|上蔡]]人。曾任[[廬江|廬江]]太守丞，著有《[[鹽鐵論|鹽鐵論]]》。
*[[桓少君|桓少君]]，西漢賢媛。
*[[桓譚|桓譚]]，字君山，兩漢時期學者，文筆佳，好古文，著有《新論》。
*[[桓榮|桓榮]]，字春卿，[[東漢|東漢]]名臣。[[齊桓公|齊桓公]]之後。名族[[譙國|譙國]]桓氏的遠祖。
*[[桓焉|桓焉]]，字叔元，桓榮之孫，東漢名臣。
*[[桓彬|桓彬]]，字彥林，東漢著名文學家，與[[蔡邕|蔡邕]]齊名。
*[[桓階|桓階]]，字伯緒，[[三國|三國]]時[[曹魏|曹魏]]官員，官至[[尚書令|尚書令]]。
*[[桓嘉|桓嘉]]，桓階之子，曹魏將領，參與東興之役陣亡。
*[[桓範|桓範]]，字元則，[[三國|三國]]時[[曹魏|曹魏]]官員，有「[[智囊|智囊]]」之稱，官至[[大司農|大司農]]。
*[[桓宣|桓宣]]，[[東晉|東晉]]時期重要將領。
*[[桓伊|桓伊]]，桓宣族子，東晉著名將領及音樂家，有「笛聖」之稱，名曲《[[梅花三弄|梅花三弄]]》的作者。
*[[桓彝|桓彝]]，字茂倫，桓溫之父，東晉大臣。
*[[桓溫|桓溫]]，字元子，桓彝之子，桓玄之父，是東晉權傾朝野的一代權臣。
*[[桓雲|桓雲]]，字雲子，桓溫之弟，東晉大臣。
*[[桓豁|桓豁]]，字朗子，桓溫之弟，東晉大臣。
*[[桓楚|桓楚]]，中國東晉時期由將領桓玄所建立的一個短期政權
*[[桓沖|桓沖]]，字幼子，桓溫之弟，東晉大臣。
*[[桓玄|桓玄]]，字敬道，桓溫之子，與父親一樣極具野心，[[東晉|東晉]]末期實行篡晉，成為[[桓楚|桓楚]]政權建立者。
*[[桓石虔|桓石虔]]，桓豁之子，以勇猛著稱。
*[[桓石綏|桓石綏]]，桓玄堂弟，桓楚政權將領，《[[晉書|晉書]]》以他的死亡標誌桓楚滅亡。
*[[桓振|桓振]]，桓石虔之子。
*[[桓謙|桓謙]]，桓沖之子，桓楚政權將領。
*[[桓胤|桓胤]]，桓沖之孫，東晉官員。
*[[桓彥範|桓彥範]]，字士則，唐代潤州丹陽（今江蘇鎮江）人。宰相[[狄仁傑|狄仁傑]]賞識其才，推薦給[[武則天|武則天]]，累遷官為御史中丞。
