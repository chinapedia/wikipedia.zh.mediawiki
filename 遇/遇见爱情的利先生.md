== 剧情概述 ==
刘欣桐，一个成长在贫民之家的女孩，可是却始终不改阳光坚强的本性。刘欣桐的生活原本平淡无奇，但是自从她和她的母亲搬进了富商纪百均家里之后，她的生活发生了翻天覆地的变化。进入纪家后，刘欣桐结识了刁蛮任性的纪家大小姐纪智珍、帅气温柔的纪家大公子纪振宇以及她命中注定的爱人利耀南。利耀南，纪智珍名义上的未婚夫和刘欣桐产生了情感，也因此发生了种种矛盾。几个人在爱情、友情、亲情的多重考验下，经历了欺骗、背叛和别离，终于有了救赎和收获。刘欣桐的坚强和善良打动了所有人包括纪智珍和纪振宇，她平凡却珍贵的美丽，散发出夺目的光彩，最终收获爱情的同时，也和失散多年的亲生父母相认。

== 制作与发行 ==
本剧于2014年3月23日在上海开机拍摄<ref>{{cite news|title=《玻璃鞋》开机 盘点周冬雨的时尚进化之路|url=http://pic.yule.sohu.com/group-548266.shtml#0|accessdate=2014-05-30|newspaper=搜狐|date=2014-03-31}}</ref>，5月30日剧方正式公开首款预告和先导海报<ref>{{cite news|title=《遇见爱情的利先生》首曝预告 陈晓周冬雨搭档|url=http://ent.163.com/14/0530/10/9TG4T8MV00031GVS.html|date=2014-05-30|newspaper=网易|accessdate=2014-05-30}}</ref>。

6月4日在上海举行了媒体探班暨新闻发布会，陈晓、周冬雨、叶青、贾景晖、-{于}-效辰，陈晓苹、黄丽娅等主创出席<ref>{{cite news|title=《遇见爱情的利先生》探班 周冬雨称陈晓"爱卖萌"|url=http://news.xinhuanet.com/ent/2014-06/05/c_126582776.htm|date=2014-06-05|newspaper=新华网|accessdate=2016-12-25|deadurl=yes|archiveurl=https://web.archive.org/web/20161226054954/http://news.xinhuanet.com/ent/2014-06/05/c_126582776.htm|archivedate=2016-12-26}}</ref>。

同年6月11日殺青，后推出七夕版预告，12月取得发行许可证<ref>{{cite web|title=发行许可证查询|url=http://dsj.sarft.gov.cn/tims/site/views/include/quickSearch/checkIssue/result.shanty?ids=0144bfd532e939254028e4a044874425|publisher=国家广电总局电视剧电子政务平台|accessdate=2015-01-09}}</ref>，但之后本剧长达近两年时间无宣传和播出消息。

2016年12月17日，[[浙江卫视|浙江卫视]]官方微信号宣布本剧將于2017年中国蓝剧场开年播出。12月23日发布“等待爱情”版定档海报与首支“遇见爱情”版预告片<ref>{{cite news|title=等了三年！陈晓周冬雨《利先生》终定档|url=http://ent.sina.com.cn/v/m/2016-12-23/doc-ifxyxury8169254.shtml|date=2016-12-23|newspaper=新浪|accessdate=2016-12-25}}</ref>。

12月27日发布终极海报，七大主演首次公开同框<ref>{{cite news|title=《利先生》陈晓周冬雨反套路上演治愈爱情|url=http://ent.sina.com.cn/v/m/2016-12-27/doc-ifxyxusa5536974.shtml|date=2016-12-27|newspaper=新浪|accessdate=2016-12-27}}</ref>。

12月28日在北京举行开播发布会，周冬雨、贾景晖、肖涵出席，陈晓也从外地进行了现场互动。发布会现场公布了五分钟终极片花<ref>{{cite news|title=《利先生》暖心开启发布会 周冬雨搞怪贺新年|url=http://ent.qq.com/a/20161229/009999.htm|date=2016-12-29|newspaper=腾讯|accessdate=2016-12-29}}</ref>。周冬雨表示，“戏是很久之前拍的，有些地方或许大家会觉得有点过时，但是我觉得情感是共通的，很有年代感。”

== 幕后花絮 ==
本劇是陈晓首部參演的時裝劇，也是周冬雨首度参演电视剧。

首次试水电视剧的周冬雨表示“现在一部电视剧相当于七八部电影的量，有的时候需要你‘一条过’，不能像电影拍那么多遍，非常锻炼人，必须在一条里面把自己想发挥的想表达的全都表达出来，容不得半点精神不集中，要不然以后就是遗憾。”<ref>{{cite news|title=周同学的灵气在发酵|url=http://news.163.com/14/0606/09/9U21T80400014AED.html|date=2014-06-06|newspaper=网易|accessdate=2016-05-15}}</ref>

周冬雨：“这部戏教会了我演哭戏，导演让我每天都换着法哭，能有一百种哭法。拍电视剧特别锻炼人，拍电影的时候一天可能就拍3-5场戏，但拍摄这部戏时最多的时候一天拍了35场戏。”<ref>{{cite news|title=金马影后周冬雨首部偶像剧将播：哭戏是拍电视剧练出来的|url=http://news.163.com/16/1229/10/C9EQT2U500018AOP.html|date=2016-12-29|newspaper=网易|accessdate=2016-12-29}}</ref>
