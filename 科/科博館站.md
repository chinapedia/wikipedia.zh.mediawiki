'''科博館站'''是位於[[臺灣|臺灣]][[臺中市|臺中市]][[西區_(臺中市)|西區]]的臺灣大道幹線公車站。2014年8月10日啟用時，為[[臺中市快捷巴士藍線|台中BRT藍線]]車站，2015年7月8日轉型為臺灣大道幹線公車站。

== 沿革 ==
*2014年8月10日：BRT藍線科博館站正式啟用。
*2015年7月8日：[[臺中市快捷巴士|臺中市快捷巴士]]系統廢除後，原藍線車站改為[[臺中市公車|臺中市公車]]臺灣大道幹線的公車站，有[[臺中市公車300路|臺中市公車300路]]、[[台中市公車301路|301路]]、[[台中市公車302路|302路]]、[[台中市公車303路|303路]]、[[台中市公車304路|304路]]、[[台中市公車305路|305路]]、[[台中市公車306路|306路]]、[[台中市公車307路|307路]]、[[台中市公車308路|308路]]停靠本站。<ref>[http://www.traffic.taichung.gov.tw/news/index-1.asp?Parser=9,4,20,,,,3897 公共運輸處於6月18日公佈優化公車專用道9條公車路線圖]，[[臺中市交通局|臺中市交通局]]，2015-06-22</ref>
*2017年5月26日：增停機場接駁公車[[台中市公車A2路|A2路]]。<ref>{{cite web|url=https://news.ltn.com.tw/news/local/paper/1105714|title=〈中部〉A2、A3台中機場快線公車上路 採預約搭乘 - 地方 - 自由時報電子報|website=news.ltn.com.tw}}</ref>
*2017年7月7日：增停[[台中市公車309路|309路]]公車。<ref>{{cite web|url=https://www.chinatimes.com/realtimenews/20170707005109-260415|title=台中火車站直達高美濕地 309公車上路|first=|last=中時電子報|website=中時電子報}}</ref> <ref>{{cite web|url=https://news.ltn.com.tw/news/life/breakingnews/2124347|title=遊高美濕地 309公車天天從台中火車站出發 - 生活 - 自由時報電子報|website=news.ltn.com.tw}}</ref>

== 車站概要 ==
車站位於[[臺灣大道|臺灣大道]]二段與[[美村路|美村路]]與[[健行路|健行路]]口，BRT車站代碼為A07<ref name="技師好康報"/>。
站名取自鄰近的[[國立自然科學博物館|國立自然科學博物館]]。

;車站出入口
'''科博館站'''為兩座地面車站，單向共2處出口，雙向共4處。
*東行站（往臺中火車站方向）出口設置位置
**1號出口：臺灣大道二段美村路口
**2號出口：臺灣大道二段中興街口
*西行站（往靜宜大學方向）出口設置位置
**1號出口：臺灣大道二段健行路口
**2號出口：臺灣大道二段中興街口
