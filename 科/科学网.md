'''科学网'''（英文：'''ScienceNet'''），是由[[中国科学院|中国科学院]]、[[中国工程院|中国工程院]]和[[国家自然科学基金委|国家自然科学基金委]]支持的中文综合性科学网站，于2007年1月18日正式上线<ref name="科学网 历史">{{cite web|url=http://blog.sciencenet.cn/home.php?mod=space&uid=45&do=blog&id=405400&from=space |title=科学网四周岁生日：搬进新居！|publisher='''科学网''' 博客 发布时间：2011-1-18 08:42 |author=科学网总编辑 赵彦 |accessdate=2011年1月10日 |language=zh}}</ref>，目标是构建全球华人科学社区。<ref name="编辑部">{{cite web | url = http://www.sciencenet.cn/m/user_content.aspx?id=290199| title = 科学网编辑部博客| accessdate =2010-06-30}}</ref>旗下的科学网博客为实名博客，属[[科学|科学]][[技术|技术]]类[[博客|博客]]群之一，截止到2010年6月30日20时，开通博客的博主有5553位。<ref name="博客">{{cite web | url = http://www.sciencenet.cn/blog/blogtop.aspx| title = 科学网博客用户列表| accessdate =2010-06-30}}</ref>

科学网称其在全球科学类中文网站中排名首位。<ref name="排名">{{cite web | url = http://www.sciencenet.cn/aboutus/default.aspx?id=8&type=1| title = 关于我们--科学网网站排名| accessdate =2010-07-02}}</ref>除了科教界以外，社会上发生的大事也经常会在科学网引起广泛讨论。

很多博客文章探讨最新科技动态、[[中国|中国]]的科研体制、[[诺贝尔奖|诺贝尔奖]]、[[海归|海归]]科技人员的人生职业规划、科技人员创业、[[论文|论文]]发表等等热门话题。科学网博客还发起有“CN域名杯全国青年科学博客大赛”，“年度人物推荐”等活动。

==历史==
*2007年1月18日，科学网正式上线
*2008年4月，出版第一本科学网博文精选集《智者不惑》
*2008年9月-12月，举办首届全国青年科学博客大赛
*2009年7月，出版第二本科学网博文精选集《流动的科学》
*2009年11月，著名博主何毓琦博文精选集《科学人生纵横》出版
*2010年4月-2010年6月，举办第二届全国青年科学博客大赛
*2010年4月12日，开通新浪微博
*2010年7月7日，科学网成立“顾问博主委员会”
*2011年12月30日，评价网站流量的全球权威机构[[Alexa|Alexa]]网站显示：科学网全球排名三个月平均为8220，中国网站排名为1184

==影响与评价==
科学网作为一个较为规范、专业的科学社区，除了对研究人员之间思想交流以及科学知识普及等起到一定促进作用外，对学术界的争鸣以及学术打假亦有影响。<ref name="影响">{{cite web | url = http://www.sciencenet.cn/m/user_content.aspx?id=290199| title = 《科学网博客三周年记》| accessdate =2010-06-30}}</ref>比较有影响的事件如：

'''饶崔之争'''：北大生命科学院院长饶毅与北大退休教授崔克明分别在科学网实名博客为自己的观点辩护，《自然》杂志对此发表社论进行了报道；

'''西安交大造假事件'''：陈永江等西安交大教授在科学网上撰写博文揭露陈连生造假一事，后被焦点访谈曝光而使该造假事件最终得到处罚。<ref name="陈永江科学网博客">{{cite web | url = http://www.sciencenet.cn/m/user_index1.aspx?userid=226527| title = 陈永江科学网博客| accessdate =2010-06-30}}</ref><ref name="焦点">{{cite web | url = http://space.tv.cctv.com/video/VIDE1269086853877888| title = 焦点访谈视频| accessdate =2010-06-30}}</ref><ref name="处罚">{{cite web | url = http://xjtunews.xjtu.edu.cn/zhxw/2010-03/1269152023d26158.shtml| title = 取消李连生教授职务解除其教师聘用合同| accessdate =2010-06-30}}</ref>

尽管科学网博客及博文数量还在持续增加中，但目前实名注册博客也仅5千多，与中国上百万的科研人员相对，比例仍然很低。此外，由于科学网三条红线的实施<ref name="三红线">{{cite web | url = http://www.sciencenet.cn/m/user_content.aspx?id=2641| title =科学网三条红线| accessdate =2010-06-30}}</ref>、政治上等敏感问题讨论的限制及科学网娱乐化等问题，导致流失了不少有影响力的博主，如李淼、李亚辉等。
